# --explanation--

`Was there` 是 `There was` 的疑問句形式，用來詢問某事物是否在過去存在或發生。它將主詞和動詞顛倒以形成疑問句。例如：

- `There was a problem with the layout.`

- `Was there a problem with the layout?`

這有助於詢問過去某事的存在或發生。
