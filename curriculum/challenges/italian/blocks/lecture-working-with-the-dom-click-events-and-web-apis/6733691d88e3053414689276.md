# --interactive--

Nelle lezioni precedenti, hai imparato a lavorare con gli eventi usando il metodo `addEventListener()`. Ma c’è un altro modo, non raccomandato, per lavorare con gli eventi in JavaScript.

I gestori di eventi in linea sono attributi speciali su un elemento HTML usati per eseguire codice JavaScript quando si verifica un evento.

Ecco un esempio di un elemento `button` con un gestore di eventi click in linea:

:::interactive_editor

```html
<button onclick="alert('Hello World!')">Show alert</button>
```

:::

Quando l’utente clicca sul pulsante, viene chiamata la funzione `alert` e viene mostrata una finestra di avviso con il messaggio `Hello World!`.

Un altro modo per usare i gestori di eventi in linea è chiamare una funzione definita in un tag `script` nel documento HTML.

Ecco un esempio di definizione di una funzione chiamata `changeBgColor` e della sua chiamata da un gestore di eventi click in linea:

:::interactive_editor

```html
<script>
  function changeBgColor() {
    document.body.style.backgroundColor = "lightblue";
  }
</script>

<button onclick="changeBgColor()">Change background color</button>
```

:::

Quando l’utente clicca sul pulsante, viene chiamata la funzione `changeBgColor` e il colore di sfondo della pagina cambia in azzurro chiaro. Anche se è possibile usare i gestori di eventi in linea in questo modo, non è considerata una buona pratica.

Per prima cosa, i gestori di eventi in linea possono allegare solo un ascoltatore di eventi a un elemento. Se vuoi allegare più ascoltatori allo stesso elemento, devi usare `addEventListener()`. Un altro motivo è che i gestori di eventi in linea mescolano codice HTML e JavaScript, rendendo il codice più difficile da leggere e mantenere. È meglio tenere separati il codice HTML e JavaScript usando `addEventListener()` per allegare gli ascoltatori di eventi agli elementi.

I gestori di eventi in linea non sono raccomandati nell’uso con JavaScript moderno. Quindi, è buona pratica attenersi al metodo `addEventListener()` quando lavori con gli eventi in JavaScript.
