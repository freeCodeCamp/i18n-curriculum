# --description--

길이 $n$인 수열 $x_1, x_2, \ldots, x_n$이 다음 조건을 만족한다고 하겠습니다:

- $x_1 = 2$
- 모든 $1 &lt; i ≤ n$에 대해: $x_{i - 1} &lt; x_i$
- 모든 $i$와 $j$에 대해 $1 ≤ i, j ≤ n$: ${(x_i)}^j &lt; {(x_j + 1)}^i$

길이 2인 그러한 수열은 단 5개뿐이며, 각각 {2,4}, {2,5}, {2,6}, {2,7}, {2,8}입니다. 길이 5인 그러한 수열은 293개가 있으며, 세 가지 예시는 다음과 같습니다: {2,5,11,25,55}, {2,6,14,36,88}, {2,8,22,64,181}.

$t(n)$을 길이 $n$인 그러한 수열의 개수라고 하겠습니다. $t(10) = 86195$이고 $t(20) = 5227991891$임이 주어졌습니다.

$t({10}^{10})$을 구하고, 답을 $10^9$로 나눈 나머지로 제출하세요.
