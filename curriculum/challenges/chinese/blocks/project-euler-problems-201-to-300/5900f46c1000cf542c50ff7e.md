# --description--

榻榻米是长方形垫子，用于完全覆盖房间的地板，没有重叠。

假设唯一可用的榻榻米尺寸为1×2，那么可以覆盖的房间的形状和大小显然存在一些限制。

对于此问题，我们仅考虑具有整数维度 `a`、`b` 且大小为偶数的矩形房间，大小定义为 $s = a \times b$。我们使用“大小”一词表示房间的地面面积，并且——在不失一般性的情况下——添加条件 $a ≤ b$。

铺设榻榻米时要遵循一条规则：四个不同垫子的角落必须相遇。 例如，考虑下面两个4×4房间的安排：

<img alt="two arrangements of mats in 4x4 room" src="https://cdn.freecodecamp.org/curriculum/project-euler/tatami-free-rooms.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

左侧的排列是可以接受的，而右侧的排列则不行：中间的红色<strong><span style="color: red;">X</span></strong>标记了四个榻榻米相交的点。

由于这个规则，某些偶数大小的房间无法用榻榻米覆盖：我们称它们为无榻榻米房间。进一步地，我们定义 $T(s)$ 为大小为 $s$ 的无榻榻米房间的数量。

最小的不含榻榻米的房间大小为 $s = 70$，维度为 7×10。所有其他大小为 $s = 70$ 的房间都可以用榻榻米覆盖；它们是：1×70、2×35 和 5×14。因此，$T(70) = 1$。

同样，我们可以验证 $T(1320) = 5$，因为恰好有 5 个无榻榻米房间大小为 $s = 1320$：20×66、22×60、24×55、30×44 和 33×40。事实上，$s = 1320$ 是使得 $T(s) = 5$ 的最小房间大小 $s$。

查找满足 $T(s) = 200$ 的最小房间大小 $s$。
