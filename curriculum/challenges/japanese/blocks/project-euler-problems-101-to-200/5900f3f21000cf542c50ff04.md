# --description--

すべての桁が1で構成される数はレプユニットと呼ばれます。$R(k)$ を長さ $k$ のレプユニットと定義します。例えば、$R(6) = 111111$ です。

$R({10}^n)$ の形のレプユニットを考えましょう。

$R(10)$、$R(100)$、$R(1000)$ は17で割り切れませんが、$R(10000)$ は17で割り切れます。しかし、$R({10}^n)$ が19で割り切れる $n$ の値は存在しません。驚くべきことに、11、17、41、73 は100未満の素数の中で $R({10}^n)$ の因数になりうる唯一の4つの素数です。

$R({10}^n)$ の因数にならない100,000未満のすべての素数の和を求めてください。
