# --explanation--

`Third Conditional` は過去の実際には起こらなかった状況と、その可能な結果について話すために使われます。構造は `If` + `past perfect`、モーダル動詞 (`could/might/should`) + `have` + `past participle` です。この条件文を使うことで、`if` 節の行動は起こらず、結果も起こらなかったが、起こり得た結果を想像しています。例えば：

`If we had tested the system earlier, we might have prevented the failure.` – これはテストが行われず、失敗が起きたが、もしテストが行われていれば失敗は避けられたかもしれないという意味です。
