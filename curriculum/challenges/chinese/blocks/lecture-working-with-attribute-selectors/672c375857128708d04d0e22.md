# --interactive--

在 HTML 中处理有序列表时，`type` 属性允许你指定所使用的编号样式，如数字、字母或罗马数字。 

可在 `ol`（有序列表）元素上设置 `type` 属性，以控制编号格式：

- `1` 用于数字列表（如 1、2、3......），

- `A` 用于大写字母列表（如 A、B、C...），

- `a` 用于小写字母列表（如 a、b、c...），

- `I` 用于大写罗马数字（如 I、II、III...），

- `i` 用于小写罗马数字（如 i、ii、iii...）。

以下是如何使用属性选择器来使用特定的 `type` 属性定位有序列表：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="A"] {
  color: purple;
  font-weight: bold;
}
```

:::

在此示例中，CSS 规则将对任何使用大写字母编号的有序列表应用紫色文本和粗体字重，该列表由 `type="A"` 指定。

同样，你也可以通过 `type="i"` 来定位使用小写罗马数字的有序列表：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="i">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="i"] {
  color: green;
}
```

:::

对于任何使用小写罗马数字的有序列表，该规则会将文本颜色更改为绿色。

根据 `type` 属性对有序列表进行样式设置的功能，可让你更好地控制文档中列表的外观，并根据内容的结构进行自定义。
