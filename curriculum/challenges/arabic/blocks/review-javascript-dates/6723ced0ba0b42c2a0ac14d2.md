## كائن `date()` والطرق الشائعة

- **التعريف**: يُستخدم كائن `date()` لإنشاء التواريخ والأوقات والتعامل معها وتنسيقها في JavaScript. في المثال التالي، تُستخدم الكلمة المفتاحية `new` لإنشاء نسخة جديدة من كائن `Date`، ثم يُعيَّن كائن `Date` إلى المتغير `now`. إذا سجلت قيمة `now` في وحدة التحكم، سترى التاريخ والوقت الحاليين بناءً على ساعة النظام في الكمبيوتر الذي يشغل الشفرة.

:::interactive_editor

```js
const now = new Date();
console.log(now); // prints the current date and time
```

:::

- **طريقة `Date.now()`**: تُستخدم هذه الطريقة للحصول على التاريخ والوقت الحاليين. تُرجع `Date.now()` عدد المللي ثانية منذ 1 يناير 1970، 00:00:00 بالتوقيت العالمي المنسق (UTC). يُعرف هذا باسم وقت العصر Unix. وقت العصر Unix هو طريقة شائعة لتمثيل التواريخ والأوقات في أنظمة الحاسوب لأنه عدد صحيح يمكن تخزينه والتعامل معه بسهولة. UTC تعني التوقيت العالمي المنسق، وهو المعيار الزمني الأساسي الذي ينظم العالم من خلاله الساعات والوقت.
- **طريقة `getDate()`**: تُستخدم هذه الطريقة للحصول على يوم من الشهر بناءً على التاريخ الحالي. تُرجع `getDate()` قيمة صحيحة بين 1 و31، حسب يوم الشهر. إذا كان التاريخ غير صالح، سترجع `NaN` (ليس رقمًا).

:::interactive_editor

```js
const now = new Date("2014-10-15");
const date = now.getDate();
console.log(date); // 15
```

:::

- **طريقة `getMonth()`**: تُستخدم هذه الطريقة للحصول على الشهر. الشهر يبدأ من الصفر، إذ يناير هو 0، فبراير 1، وهكذا. في هذا المثال، الناتج هو 9، وهو ما يمثل أكتوبر. إذا كان الشهر غير صالح، سترجع `NaN`.

:::interactive_editor

```js
const now = new Date("2014-10-15");
const month = now.getMonth();
console.log(month); // 9
```

:::

- **طريقة `getFullYear()`**: تُستخدم هذه الطريقة للحصول على السنة كاملة. إذا كانت السنة غير صالحة، سترجع `NaN`.

:::interactive_editor

```js
const now = new Date("2014-10-15");
const year = now.getFullYear();
console.log(year); // 2014
```

:::

## طرق مختلفة لتنسيق التواريخ

- **طريقة `toISOString()`**: تُستخدم هذه الطريقة لتنسيق التاريخ بصيغة `ISO` الممتدة (ISO 8601). ISO 8601 هو معيار دولي لتمثيل التواريخ والأوقات. الصيغة هي `YYYY-MM-DDTHH:mm:ss.sssZ`.

:::interactive_editor

```js
const date = new Date("2014-10-15");
console.log(date.toISOString()); // "2014-10-15T00:00:00.000Z"
```

:::

- **طريقة `toLocaleDateString()`**: تُستخدم هذه الطريقة لتنسيق التاريخ بناءً على لغة المستخدم المحلية.

:::interactive_editor

```js
const date = new Date("2014-10-15");
console.log(date.toLocaleDateString("en-US"));  // "10/15/2014"
```

:::

تقبل طريقة `toLocaleDateString()` معلمتين اختياريتين: locales و options.

معلمة locales هي سلسلة نصية تمثل اللغة المحلية التي تُستخدم. على سبيل المثال، يمكنك تمرير `"en-US"` للإنجليزية (الولايات المتحدة) أو `"fr-FR"` للفرنسية (فرنسا). إذا لم تمرر معلمة locales، تُستخدم اللغة المحلية الافتراضية. المعلمة الاختيارية الثانية هي معلمة options. هذه المعلمة كائن يسمح لك بتحديد تنسيق سلسلة التاريخ.

:::interactive_editor

```js
const date = new Date("2014-10-15");
const options = {
  weekday: "long",
  year: "numeric",
  month: "long",
  day: "numeric",
};
console.log(date.toLocaleDateString("en-GB", options)); // "Wednesday, 15 October 2014"
```

:::
