## 字串基礎

- **定義**：字串是一連串用單引號、雙引號或反引號包住的字元。字串屬於原始資料型別，且具有不可變性。不變性代表字串一旦建立就無法更改。
- **從字串存取字元**：你可以用中括號標記法並傳入索引數字來存取字串中的字元。索引是字元在字串中的位置，從 0 開始。

:::interactive_editor

```js
const developer = "Jessica";
console.log(developer[0]); // J
```

:::

- **`\n`（換行字元）**：你可以用 `\n` 換行字元在字串中產生換行。

:::interactive_editor

```js
const poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

- **跳脫字串**：你可以在字串中的引號前放置反斜線（`\`）來跳脫字元。

:::interactive_editor

```js
const statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

## 樣板字面值（樣板字串）與字串插值

- **定義**：樣板字面值用反引號（`）定義。它們讓字串操作更容易，包括直接在字串中嵌入變數，這個功能稱為字串插值。

:::interactive_editor

```js
const name = "Jessica";
const greeting = `Hello, ${name}!`; 
console.log(greeting); // "Hello, Jessica!"
```

:::

## ASCII、`charCodeAt()` 方法與 `fromCharCode()` 方法

- **ASCII**：ASCII（American Standard Code for Information Interchange）是一種字元編碼標準，用數值來表述基本英文字元。早期課程介紹了使用 ASCII 範例的 `charCodeAt()` 和 `fromCharCode()`。
- **Unicode**：JavaScript 字串在內部使用 Unicode，特別是 UTF-16 編碼。對於前 128 個字元（基本拉丁字母、數字和常用符號），Unicode 值與 ASCII 程式碼相符。這就是為什麼基於 ASCII 的範例在 JavaScript 中仍然有效。
- **`charCodeAt()` 方法**：此方法會傳回指定索引處字元的 UTF-16 編碼單元。對於基本拉丁字元，此值與 ASCII 碼相符。

:::interactive_editor

```js
const letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

:::

- **`fromCharCode()` 方法**：這個方法會將 ASCII 代碼轉換成對應的字元。

:::interactive_editor

```js
const char = String.fromCharCode(65);
console.log(char);  // A
```

:::

## 其他常見字串方法

- **`indexOf` 方法**：這個方法用來在字串中搜尋子字串。如果找到子字串，`indexOf` 會傳回該子字串第一次出現的索引（位置）；如果找不到，`indexOf` 則會傳回 -1，表示搜尋失敗。

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.indexOf("fox")); // 16
console.log(text.indexOf("cat")); // -1
```
  
:::  

- **`includes()` 方法**：這個方法用來檢查字串是否包含特定子字串。如果找到，方法會傳回 true，否則傳回 false。

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.includes("fox")); // true
console.log(text.includes("cat")); // false
```

:::

- **`slice()` 方法**：此方法擷取字串的一部分並傳回新的字串，而不修改原始字串。它接受兩個參數：起始索引和選用的結束索引。

:::interactive_editor

```js
const text = "freeCodeCamp";
console.log(text.slice(0, 4));  // "free"
console.log(text.slice(4, 8));  // "Code"
console.log(text.slice(8, 12)); // "Camp"
```

:::

- **`toUpperCase()` 方法**：這個方法會將所有字元轉換為大寫字母，並傳回一個全為大寫的新字串。

:::interactive_editor

```js
const text = "Hello, world!";
console.log(text.toUpperCase()); // "HELLO, WORLD!"
```

:::

- **`toLowerCase()` 方法**：這個方法會將字串中的所有字元轉換為小寫。

:::interactive_editor

```js
const text = "HELLO, WORLD!"
console.log(text.toLowerCase()); // "hello, world!"
```

:::

- **`replace()` 方法**：這個方法讓你在字串中尋找特定值（如單字或字元），並將其取代為其他值。該方法會傳回一個帶有替換內容的新字串，原始字串不會被改變，因為 JavaScript 字串是不可變的。

:::interactive_editor

```js
const text = "I like cats";
console.log(text.replace("cats", "dogs")); // "I like dogs"
```

:::

- **`replaceAll()` 方法**：此方法允許你在字串中尋找指定值（字、字元或樣式）的所有出現，並將它們替換為另一個值。它的運作方式類似 `replace()`，但不會在找到第一個符合項目後停止，而是更新字串中找到的每一個符合項目。

:::interactive_editor

```js
const text = "I love cats and cats are so much fun!";
console.log(text.replaceAll("cats", "dogs")); // "I love dogs and dogs are so much fun!"
```

:::

- **`repeat()` 方法**：這個方法用來將字串重複指定次數。

:::interactive_editor

```js
const text = "Hello";
console.log(text.repeat(3)); // "HelloHelloHello"
```

:::

- **`trim()` 方法**：這個方法用來移除字串開頭和結尾的空白字元。

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trim()); // "Hello, world!"
```

:::

- **`trimStart()` 方法**：這個方法會移除字串開頭（或「起始」）的空白字元。

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trimStart()); // "Hello, world!  "
```

:::

- **`trimEnd()` 方法**：這個方法會移除字串結尾的空白字元。

:::interactive_editor

```js
const text = " Hello, world! ";
console.log(text.trimEnd()); // "  Hello, world!"
```

:::

- **`prompt()` 方法**：這個 `window` 的方法用來透過對話盒向使用者取得資訊。這個方法接受兩個引數。第一個引數是訊息，會顯示在對話盒中，通常用來提示使用者輸入資訊。第二個引數是預設值（選填），會預先填入輸入欄位。

```js
const answer = window.prompt("What's your favorite animal?"); // This will change depending on what the user answers
```
