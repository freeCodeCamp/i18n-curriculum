# --explanation--

`Consistent`는 변함없이 규칙적으로 일어나는 일을 의미합니다. 예를 들어:

`The website traffic is consistent.` – 이는 사이트 방문자 수가 갑작스러운 변화 없이 꾸준히 유지된다는 뜻입니다.

`Happens enough`는 어떤 일이 눈에 띌 만큼 자주 발생하지만 항상 그런 것은 아니라는 뜻입니다. 예를 들어:

`The error happens enough that it's becoming a problem.` – 이는 오류가 항상 발생하는 것은 아니지만, 문제를 일으킬 만큼 자주 발생한다는 뜻입니다.

`Annoying`는 짜증나거나 불쾌한 것을 의미합니다. 예를 들어:

`The sound is really annoying.` – 이는 소리가 불쾌하다는 뜻입니다.
