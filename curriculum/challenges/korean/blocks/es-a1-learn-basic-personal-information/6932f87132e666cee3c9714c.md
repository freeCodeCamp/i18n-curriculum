# --description--

Ángela는 `¿Cuál es tu` + 명사 구조를 사용해 질문합니다.

그녀는 거기서 멈출 수도 있지만, 대신 연결사 `y`와 두 번째 명사를 사용해 추가 정보를 덧붙입니다.

`y`는 두 명사를 연결하는 접속사입니다. 그녀가 사용하는 구조는 다음과 같습니다:

`¿Cuál es tu` + 명사 + `y` + 명사`?`

이 구조를 통해 화자는 한 질문에서 두 가지 이상의 정보를 요청할 수 있습니다.

# --explanation--

문장 `¿Cuál es tu nombre y apellido?`에서 두 명사가 `y`로 연결되어 있습니다:

- `Nombre`

- `Apellido`

단어 `y`는 두 정보를 연결합니다.

Ángela는 먼저 상대방의 `nombre`를 묻고, 그다음 `apellido`를 덧붙입니다.

이 구조는 한 질문에서 여러 정보를 요청할 수 있게 합니다:

`¿Cuál es tu correo y número?`

여기서 Ángela가 묻는 두 번째 정보는 `apellido`입니다.
