# --description--

Ein Typselektor (oder Elementselektor) wählt alle Elemente des angegebenen Elementtyps aus, und die Syntax ist einfach der Name des Elements:

```html
<!-- index.html -->

<div>Hello, World!</div>
<div>Hello again!</div>
<p>Hi...</p>
<div>Okay, bye.</div>
```

```css
/* styles.css */

div {
  color: white;
}
```

Hier würden alle drei `<div>`-Elemente ausgewählt, während das `<p>`-Element nicht ausgewählt würde.

# --questions--    

## --text--

Welche der folgenden Aussagen beschreibt den oben angegebenen CSS-Code am besten?

## --answers--

Der Code wendet eine `white`-Farbe auf alle Elemente in der HTML-Datei an.

---

Der Code wendet eine `white`-Farbe auf alle `div`-Elemente in der HTML-Datei an.

---

Der Code wendet eine `white`-Farbe auf alle `p`-Elemente in der HTML-Datei an.
