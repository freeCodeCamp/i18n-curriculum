# --description--

La letra `r` en español se llama `erre`.

Esta consonante tiene dos pronunciaciones diferentes.

Una sola `r` tiene un sonido suave. Por ejemplo, `cero` ("zero").

Una doble `r` (`rr`) o una `r` al principio de una palabra produce un sonido más fuerte. Por ejemplo, `perro` ("dog") o `ruido` ("noise").
