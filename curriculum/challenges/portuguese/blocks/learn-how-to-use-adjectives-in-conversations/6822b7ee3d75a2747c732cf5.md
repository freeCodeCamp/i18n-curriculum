# --explanation--

`How they look` é uma oração que começa com `how`, que introduz um tópico ou sujeito. Por exemplo:

`He didn't like how the icons looked on the screen.` – Isso significa que ele não gostou da aparência dos ícones.

Orações como `how they look` são chamadas de **noun clauses** e podem atuar como o objeto de uma frase.

Neste contexto, `Let's start with how they look` significa que a primeira coisa a examinar é a forma como as duas versões aparecem visualmente.
