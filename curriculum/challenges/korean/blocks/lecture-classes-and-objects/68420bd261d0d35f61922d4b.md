# --description--

Python에서 클래스와 객체는 데이터를 조직하고 관리하는 데 함께 작동합니다. 공통 동작을 정의하기 위해 클래스를 만들고, 그 동작을 사용하는 객체를 생성합니다.

다시 말해, 클래스는 객체를 만들 때 사용하는 청사진이나 템플릿과 같습니다.

클래스가 무엇인지, 그리고 클래스를 사용해 객체를 만드는 방법을 살펴보겠습니다.

클래스를 만들려면 `class` 키워드 다음에 클래스 이름과 콜론을 씁니다. 그런 다음 클래스 내부에 초기화자와 속성, 메서드를 추가할 수 있습니다.

속성은 클래스 내 변수와 같으며 데이터를 저장하는 데 사용합니다. 메서드는 클래스 내에 정의된 함수로, 클래스에서 생성된 객체가 수행할 수 있는 동작입니다.

클래스의 기본 구문은 다음과 같습니다:

```python
class ClassName:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def sample_method(self):               
        print(self.name.upper())
```

* `class ClassName`은 클래스를 만들기 위한 `class` 키워드와 클래스 이름인 `ClassName`으로 구성됩니다. Python에서는 클래스 이름을 지을 때 **PascalCase** 관례를 사용하는 것이 일반적입니다.
    
* `def __init__(self, name, age)`는 새 객체가 생성될 때 자동으로 호출되는 특별한 메서드입니다. 이 메서드는 클래스에서 생성될 객체의 속성을 초기화합니다.
    
    또한 `__init__`의 첫 번째 매개변수는 항상 생성되거나 사용되는 특정 객체에 대한 참조입니다. 관례상 이 매개변수 이름은 `self`이지만, 기술적으로는 어떤 이름을 써도 됩니다. `self`을 사용하면 객체 자신의 속성과 메서드에 접근할 수 있습니다.
    
* `self.name = name`과 `self.age = age`는 객체가 가질 속성입니다.
    
* `def sample_method(self):`은 생성된 각 객체가 호출할 수 있는 메서드입니다.
    
* `print(self.name.upper())`은 `sample_method` 메서드가 수행할 작업으로, 이 경우 이름을 대문자로 출력합니다.
    

이 모든 내용이 많게 느껴진다면 걱정하지 마세요. 비슷한 `Dog` 클래스 예제를 보고, 그 클래스로부터 객체를 만드는 방법을 살펴보겠습니다:

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name.upper()} says woof woof!")
```

이 `Dog` 클래스를 사용해 객체를 만들 수 있습니다. 클래스에서 객체를 만드는 기본 구문은 다음과 같습니다:

```python
object_1 = ClassName(attribute_1, attribute_2)
object_2 = ClassName(attribute_1, attribute_2)
```

또한 각 객체에서 클래스에 정의된 메서드를 호출할 수 있습니다:

```python
object_1.method_name()
object_2.method_name()
```

이제 `Dog` 클래스를 청사진으로 사용해 두 마리 개를 만들어 보겠습니다:

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name.upper()} says woof woof! I'm {self.age} years old!")

dog_1 = Dog("Jack", 3)
dog_2 = Dog("Thatcher", 5)

# Call the bark method
dog_1.bark()  # JACK says woof woof! I'm 3 years old!
dog_2.bark()  # THATCHER says woof woof! I'm 5 years old!
```

보시다시피 `Dog` 클래스를 사용해 두 개의 개 객체를 만듭니다. `dog_1`를 초기화할 때 문자열 `Jack`과 숫자 `3`가 전달되어 해당 인스턴스의 `name`과 `age` 속성이 설정됩니다. `dog_2`는 문자열 `Thatcher`과 숫자 `5`를 각각 `name`와 `age`으로 초기화합니다.

그런 다음 `.bark()`과 `dog_1`에서 `dog_2` 메서드를 호출하면, 두 출력이 어떻게 다른지 확인할 수 있고, 각 객체를 만들 때 전달한 고유한 `name`과 `age` 속성을 사용할 수 있습니다.

요약하면, 클래스와 객체의 차이는 클래스가 템플릿 또는 청사진이고, 객체는 그 템플릿을 사용해 만들어진 것입니다.

또한 클래스는 객체가 가져야 할 데이터와 동작을 정의하고, 객체는 실제 데이터를 보유하며 그 동작을 사용합니다. 클래스를 한 번 작성하면, 서로 다른 데이터를 가진 여러 객체를 만들 수 있습니다.
