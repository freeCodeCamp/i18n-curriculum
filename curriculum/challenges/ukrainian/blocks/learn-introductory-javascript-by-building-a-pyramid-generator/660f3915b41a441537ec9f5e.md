# --description--

Оскільки ви лише збільшуєте `i` на `1`, ви можете використати <dfn>оператор інкременту</dfn> `++`. Цей оператор збільшує значення змінної на 1, оновлюючи присвоєння для цієї змінної. Наприклад, `test` тут стане `8`:

```js
let test = 7;
test++;
```

Замініть своє присвоєння додаванням на оператор інкременту для ітерації циклу.
