# --description--

CSSで成功するために理解する必要がある最初の重要な概念はボックスモデルです。複雑ではありませんが、今これを飛ばすと後で大きなフラストレーションになります。

ウェブページ上のすべてのものは長方形のボックスです。これらのボックスは他のボックスを内包したり、隣り合って配置されたりします。ページ上のすべての要素にボーダーを付けてみると、この仕組みがざっくりとわかります。

```css
* {
  border: 1px solid red;
}
```

# --assignment--

ページ上のすべての要素にボーダーを追加して、ボックスの配置を確認してください。
