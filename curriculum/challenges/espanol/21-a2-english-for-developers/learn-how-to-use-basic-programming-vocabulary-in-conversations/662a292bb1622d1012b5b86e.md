# --description--

Aprende a usar `how much` y `how many` para preguntar sobre cantidades. `How much` se usa con sustantivos incontables para preguntar sobre el volumen o la extensión de algo, como `time`, `water`, o `money`. Por ejemplo, `How much money do you have?` pregunta sobre la cantidad de dinero, la cual no puedes contar individualmente.

`How many` se usa con sustantivos contables cuando puedes contar los elementos, como `books`, `apples`, o `people`. Por ejemplo, `How many books do you own?` pregunta por un número contable de libros.
