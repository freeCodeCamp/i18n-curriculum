# --description--

Computerbildschirme und Gerätescreens erzeugen verschiedene Farben, indem sie Mengen von rotem, grünem und blauem Licht kombinieren. Dies ist als das additive RGB-Farbmodell in der modernen Farbtheorie bekannt. Rot (R), Grün (G) und Blau (B) werden Primärfarben genannt. Das Mischen von zwei Primärfarben erzeugt die Sekundärfarben Cyan (G + B), Magenta (R + B) und Gelb (R + G). Diese Farben haben Sie in der Aufgabe Komplementärfarben gesehen. Diese Sekundärfarben sind jeweils die Komplementärfarbe zu der Primärfarbe, die bei ihrer Entstehung nicht verwendet wurde, und liegen auf dem Farbrad gegenüber dieser Primärfarbe. Zum Beispiel wird Magenta aus Rot und Blau hergestellt und ist die Komplementärfarbe zu Grün.

Tertiärfarben entstehen durch die Kombination einer Primärfarbe mit einer ihrer benachbarten Sekundärfarben. Zum Beispiel ergeben im RGB-Farbmodell Rot (Primär) und Gelb (Sekundär) Orange (Tertiär). Dadurch werden sechs weitere Farben zu einem einfachen Farbrad hinzugefügt, sodass es insgesamt zwölf Farben gibt.

Es gibt verschiedene Methoden, um unterschiedliche Farben auszuwählen, die in einem Design eine harmonische Kombination ergeben. Ein Beispiel, das tertiäre Farben verwenden kann, heißt das teilkomplementäre Farbschema. Dieses Schema beginnt mit einer Grundfarbe und kombiniert sie dann mit den zwei Farben, die an ihre Komplementärfarbe angrenzen. Die drei Farben bieten einen starken visuellen Kontrast in einem Design, sind aber subtiler als die Verwendung von zwei komplementären Farben.

# --instructions--

Ändern Sie die `background-color`-Eigenschaft der `orange`-, `cyan`- und `raspberry`-Klassen in ihre jeweiligen Farben. Achten Sie darauf, die Hex-Codes und nicht die Farbnamen zu verwenden.

# --hints--

Das `div`-Element mit der Klasse `orange` sollte eine `background-color` von orange haben.

```js
const orangeElement = document.querySelector('.orange');
const orangeStyle = window.getComputedStyle(orangeElement);
assert.equal(orangeStyle?.backgroundColor, 'rgb(255, 127, 0)');
```

Das `div`-Element mit der Klasse `cyan` sollte eine `background-color` von cyan haben.

```js
const cyanElement = document.querySelector('.cyan');
const cyanStyle = window.getComputedStyle(cyanElement);
assert.equal(cyanStyle?.backgroundColor, 'rgb(0, 255, 255)');
```

Das `div`-Element mit der Klasse `raspberry` sollte eine `background-color` von himbeere haben.

```js
const raspberryElement = document.querySelector('.raspberry');
const raspberryStyle = window.getComputedStyle(raspberryElement);
assert.equal(raspberryStyle?.backgroundColor, 'rgb(255, 0, 127)');
```

Alle `background-color`-Werte für die Farbklassen sollten Hex-Codes und keine Farbnamen sein.

```js
assert.notMatch(code, /background-color:\s(orange|cyan|raspberry)/);
```
