# --interactive--

CSS 負責網頁的樣式。所有這些樣式都是由各種 CSS 規則組成。

CSS 規則由兩個主要零件組成：**selector** 和 **declaration** 區塊。

讓我們來看看基本語法：

```css
selector {
  property: value;
}
```

選擇器是在 CSS 中用來識別並標的特定 HTML 元素以進行樣式設定的樣式。

選擇器的範例包括型別選擇器、類別選擇器和 ID 選擇器。

基本語法中提供的大括號稱為宣告區塊。宣告區塊會對指定的選擇器或選擇器群組套用一組樣式。

在宣告區塊內，你會有一系列的宣告。每個宣告由一個屬性和一個值組成。

屬性是指定要套用樣式的 CSS 識別符。屬性的範例是 `background-color` 屬性。

該值將是套用到該屬性的特定設定。例如，如果該屬性是 `background-color`，則值可以是 `purple`，這會將背景顏色設定為紫色。

在每個屬性名稱後，你需要放置冒號，在每個值後，你應該有分號。

既然你已經知道 CSS 規則的語法，讓我們來看看一個範例。啟用互動式編輯器並點擊 `styles.css` 標籤以查看 CSS 程式碼。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Learning about CSS</h1>

<p>Example paragraph element</p>
<p>Another example paragraph element</p>
```

```css
p {
  color: blue;
}
```

:::

在這個 CSS 規則中，型別選擇器會標的頁面上的所有段落元素。

在宣告區塊中，有一個宣告。

該宣告包含 `color` 屬性，其值設定為 `blue`。這將把所有段落元素的文字顏色更改為 `blue`。

如果你想將相同的一組樣式套用到多個選擇器，你可以創建一個選擇器列表，並用逗號分隔每個選擇器。

這裡是一個多個選擇器樣式的範例（要與範例互動，請啟用互動式編輯器）： 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1 id="title">Example heading</h1>
<h2 class="subheading">Example subheading</h2>
<p>This paragraph is not affected by the selector.</p>
```

```css
#title,
.subheading {
  color: navy;
}
```

:::

在這個選擇器列表中，有一個 `id` 選擇器標的 HTML 元素，其 `id` 值為 `title`。所有 `id` 選擇器必須以井字號 `#` 符號開頭。

接著是一個逗號，後面跟著一個 `class` 選擇器，該選擇器標的所有具有 `class` 值為 `subheading` 的 HTML 元素。所有 `class` 選擇器必須以點號 `.` 開頭。

整個選擇器列表將會接收相同的樣式，文字顏色設定為 `navy`。
