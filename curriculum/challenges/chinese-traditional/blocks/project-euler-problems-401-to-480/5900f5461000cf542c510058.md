# --description--

令 $\varphi$ 為黃金比例：$\varphi = \frac{1+\sqrt{5}}{2}.$

值得注意的是，即使我們要求每個 `\varphi` 的冪次在此和式中最多只能使用一次，仍然可以將每個正整數寫成 `\varphi` 的冪次和。

即使如此，這種表現也不是唯一的。

我們可以透過要求不使用連續指數的冪次且表述為有限的方式，使其具有唯一性。

例如：

$2 = \varphi + \varphi^{-2}$ 且 $3 = \varphi^{2} + \varphi^{-2}$

為了表述這個 $\varphi$ 次方的和，我們使用一串由 0 和 1 組成的字串，並用一個點來顯示負指數開始的位置。我們稱這種表示法為 phigital 數字系統中的表示法。

所以 $1 = 1_{\varphi}$、$2 = 10.01_{\varphi}$、$3 = 100.01_{\varphi}$ 和 $14 = 100100.001001_{\varphi}$。表述 1、2 和 14 的字串在 phigital 數字系統中是回文的，而表述 3 的字串則不是（phigital 小數點不是中間的字元）。

不超過 1000 的正整數中，其 phigital 表述為回文的數字總和是 4345。

找出不超過 $10^{10}$ 的正整數中，其 phigital 表述為回文的數字總和。
