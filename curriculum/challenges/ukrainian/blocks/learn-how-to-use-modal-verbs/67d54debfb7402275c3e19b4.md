# --explanation--

`Sensitive data` означає конфіденційну інформацію, яку потрібно захищати. Наприклад:

`Sensitive data, like passwords, should never be shared.` — приватна інформація, така як паролі, має залишатися таємницею.

`Encryption keys` використовуються для блокування та розблокування зашифрованих даних. Наприклад:

`Encryption keys help keep online payments secure.` — ці спеціальні коди захищають фінансові транзакції.

Настанови Джессіки гарантують, що обидва елементи обробляються безпечно.
