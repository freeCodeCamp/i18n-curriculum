# --explanation--

`Made sure`는 무언가가 사실이거나 제대로 작동하는지 확인하기 위해 조치를 취하는 것을 의미합니다. 예를 들어:  

`They made sure the software was ready before launch.` – 그들은 모든 것이 제대로 작동하는지 확인했습니다.

`As`는 다른 동작과 동시에 일어나는 일을 설명할 때 사용합니다. 예를 들어:  

`The system adds more resources as demand increases.` – 더 많은 사용자가 시작할 때 조정합니다.
