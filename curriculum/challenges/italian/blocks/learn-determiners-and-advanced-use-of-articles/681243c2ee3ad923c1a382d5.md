# --explanation--

`Payment integration` indica quando un sistema di pagamento (come PayPal, Stripe o i processori di carte di credito) viene collegato a un'app o a un sito web per permettere ai clienti di pagare. Per esempio:

- `We need payment integration on our website to accept credit cards.` - Il sito web dovrebbe permettere agli utenti di pagare online.

- `The developer worked on payment integration for the mobile app.` - Hanno aggiunto un modo per gli utenti di pagare tramite l'app.
