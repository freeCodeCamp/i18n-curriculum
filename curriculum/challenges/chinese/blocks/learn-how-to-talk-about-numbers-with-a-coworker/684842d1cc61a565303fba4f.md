# --explanation--

`Make sure to` 用于表达责任或承诺，意味着说话者一定会做某事。例如：

`Make sure to save your work before closing.` — 关闭程序前别忘了储存。

`As needed` 意味着只有在变得必要时才去做某事。例如：

`We can add more team members as needed.` — 他们只有在必要时才会添加成员。

在 Alice 的句子中，她说如果 `project` 的进展需要，她会定期通知 David 并更改预算计划。
