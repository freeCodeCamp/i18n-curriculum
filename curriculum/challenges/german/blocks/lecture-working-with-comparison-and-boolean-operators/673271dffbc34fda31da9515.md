# --interactive--

Vergleichsoperatoren ermöglichen es Ihnen, zwei Werte zu vergleichen und ein `true` oder `false` Ergebnis zurückzugeben. Sie können dieses Ergebnis dann verwenden, um eine Entscheidung zu treffen oder den Ablauf Ihres Programms zu steuern. Sie verwenden Vergleiche in `if`-Anweisungen, Schleifen und vielen anderen Situationen, in denen Sie Entscheidungen basierend auf bestimmten Bedingungen treffen müssen. Lassen Sie uns die gebräuchlichsten Vergleichsoperatoren ansehen und sehen, wie sie funktionieren.

Der Größer-als-Operator, dargestellt durch ein spitzes Klammerzeichen nach rechts (`>`), prüft, ob der Wert links größer ist als der Wert rechts:

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a > b); // false
console.log(b > a); // true
```

:::

Der Größer-als-oder-gleich-Operator, dargestellt durch ein spitzes Klammerzeichen nach rechts und das Gleichheitszeichen (`>=`), prüft, ob der Wert links entweder größer oder gleich dem Wert rechts ist:

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a >= b); // false
console.log(b >= a); // true
console.log(a >= c); // true
```

:::

Der Kleiner-als-Operator, dargestellt durch ein spitzes Klammerzeichen nach links (`<`), funktioniert ähnlich wie `>`, aber umgekehrt. Er prüft, ob der Wert links kleiner ist als der Wert rechts:

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a < b); // true
console.log(b < a); // false
```

:::

Der Kleiner-als-oder-gleich-Operator, dargestellt durch ein spitzes Klammerzeichen nach links und das Gleichheitszeichen (`<=`), prüft, ob der Wert links kleiner oder gleich dem Wert rechts ist:

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a <= b); // true
console.log(b <= a); // false
console.log(a <= c); // true
```

:::
