# --interactive--

المصفوفة في JavaScript هي مجموعة مرتبة من القيم، يُحدد كل منها بواسطة فهرس رقمي. يمكن أن تكون القيم في مصفوفة JavaScript من أنواع بيانات مختلفة، بما في ذلك الأعداد، والسلاسل النصية، والقيم المنطقية، والكائنات، وحتى مصفوفات أخرى.

لإنشاء مصفوفة في JavaScript، يمكنك استخدام الأقواس المربعة، `[]`، وفصل القيم بفواصل. إليك مثالًا:

```js
let fruits = ["apple", "banana", "orange"];
```

في هذا المثال، نُعلن عن متغير `fruits` ونعين له مصفوفة تحتوي على ثلاث قيم نصية: `apple`، و`banana`، و`orange`.

إحدى الخصائص الرئيسية للمصفوفات هي أنها تبدأ فهارسها من الصفر، مما يعني أن العنصر الأول في المصفوفة له فهرس `0`، والعنصر الثاني له فهرس `1`، وهكذا. يمكنك الوصول إلى عناصر فردية في المصفوفة باستخدام فهرسها. على سبيل المثال:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits[0]); // "apple"
console.log(fruits[2]); // "orange"
```

:::

في هذا المثال، نستخدم الفهرس `0` للوصول إلى العنصر الأول (`apple`) والفهرس `2` للوصول إلى العنصر الثالث (`orange`).

للمصفوفات في JavaScript خاصية خاصة تسمى `length` تُرجع عدد العناصر في المصفوفة. يمكنك الوصول إلى هذه الخاصية باستخدام `length`. على سبيل المثال:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits.length); // 3
```

:::

خاصية أخرى مهمة للمصفوفات في JavaScript هي أنها ديناميكية، مما يعني أن حجمها يمكن أن يتغير بعد إنشائها. يمكنك إضافة أو إزالة عناصر من المصفوفة باستخدام طرق مختلفة للمصفوفات، مثل `push()`، و`pop()`، و`shift()`، و`unshift()`، و`splice()`، وغيرها. ستُدرس هذه الطرق في الدروس القادمة.

مصفوفات JavaScript متعددة الاستخدامات ومفيدة جدًا عند تخزين البيانات داخل برامجك. خلال هذه الوحدة، سترى بنفسك كيف يساعدك العمل مع المصفوفات على إدارة مجموعات البيانات والتعامل معها بكفاءة.
