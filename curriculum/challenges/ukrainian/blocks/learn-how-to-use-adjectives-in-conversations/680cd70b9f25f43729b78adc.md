# --explanation--

`We'd prefer` — це ввічливий спосіб сказати, що ви хочете більше. Наприклад:

`We'd prefer faster processors for the new computers.` – це показує сильніший інтерес до кращих варіантів.

`Since` використовується, щоб пояснити причину рішення або ситуації. Наприклад:

`We upgraded the servers since the old ones were too slow.` – це показує причину для оновлення.
