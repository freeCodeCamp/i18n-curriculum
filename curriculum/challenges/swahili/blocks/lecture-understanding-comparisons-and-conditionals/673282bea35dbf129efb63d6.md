# --interactive--

`switch` statements na `if/else if/else` chains zote ni miundo ya mtiririko wa udhibiti katika programu inayoturuhusu kutekeleza sehemu tofauti za msimbo kulingana na masharti fulani. Hata hivyo, zina sifa na matumizi tofauti.

Taarifa ya `switch` hutathmini usemi wa kihesabu na kulinganisha thamani yake na mfululizo wa vifungu vya `case`. Wakati patakapopatikana mlingano, sehemu ya msimbo inayohusiana na `case` hiyo hutatuliwa. Hapa kuna muundo wa msingi wa taarifa ya `switch`:

```js
switch (expression) {
  case value1:
    // code to be executed if expression === value1
    break;
  case value2:
    // code to be executed if expression === value2
    break;
  default:
    // code to be executed if expression doesn't match any case
}
```

Taarifa ya `break` mwishoni mwa kila `case` ni muhimu sana. Inamwambia programu kuondoka kwenye sehemu ya switch mara tu `case` inayolingana itakapotekelezwa. Bila hiyo, programu itaendelea kutekeleza kesi zinazofuata, tabia inayojulikana kama "fall-through".

Taarifa za `switch` kawaida hutumika wakati unalinganisha kigezo kimoja dhidi ya thamani nyingi zinazowezekana. Zinatumika sana wakati una masharti mengi ya kuangalia dhidi ya kigezo kimoja. Hapa kuna mfano wa kutumia taarifa ya `switch` kwa siku za wiki:

:::interactive_editor

```js
let dayOfWeek = 3; 

switch (dayOfWeek) {
    case 1:
        console.log("It's Monday! Time to start the week strong.");
        break;
    case 2:
        console.log("It's Tuesday! Keep the momentum going.");
        break;
    case 3:
        console.log("It's Wednesday! We're halfway there.");
        break;
    case 4:
        console.log("It's Thursday! Almost the weekend.");
        break;
    case 5:
        console.log("It's Friday! The weekend is near.");
        break;
    case 6:
        console.log("It's Saturday! Enjoy your weekend.");
        break;
    case 7:
        console.log("It's Sunday! Rest and recharge.");
        break;
    default:
        console.log("Invalid day! Please enter a number between 1 and 7.");
}
```

:::

Taarifa za `switch` zinaweza kusomeka kwa urahisi zaidi na kuwa fupi wakati zinashughulikia thamani nyingi zinazowezekana kwa kigezo kimoja.

Taarifa za `if/else if` kwa upande mwingine ni za kubadilika zaidi. Zinaweza kutathmini masharti magumu na vigezo tofauti katika kila kifungu. Hii inazifanya zifae kwa aina mbalimbali za hali. Hapa kuna mfano wa wakati unaweza kutumia taarifa ya `if/else` badala ya taarifa ya `switch`:

:::interactive_editor

```js
let creditScore = 720; 
let annualIncome = 60000; 
let loanAmount = 200000; 

let eligibilityStatus;

if (creditScore >= 750 && annualIncome >= 80000) {
    eligibilityStatus = "Eligible for premium loan rates.";
} else if (creditScore >= 700 && annualIncome >= 50000) {
    eligibilityStatus = "Eligible for standard loan rates.";
} else if (creditScore >= 650 && annualIncome >= 40000) {
    eligibilityStatus = "Eligible for subprime loan rates.";
} else if (creditScore < 650) {
    eligibilityStatus = "Not eligible due to low credit score.";
} else {
    eligibilityStatus = "Not eligible due to insufficient income.";
}

console.log(eligibilityStatus);
```

:::

Katika mfano huu, tuna kipato cha mtu kwa mwaka na alama ya mkopo na tunakagua aina gani za mikopo angeweza kustahiki. Kwa kuwa tunashughulikia tathmini za mantiki ngumu na vigezo vingi, ni bora kutumia taarifa ya `if/else` hapa badala ya taarifa ya `switch`.

Inafaa kutambua kuwa taarifa za `switch` katika JavaScript hutumia kulinganisha kwa ukali (`===`), ambayo ina maana hazifanyi kulazimisha aina. Hii inaweza kuwa faida kwa upande wa utabiri na kuepuka hitilafu ndogo ndogo.

Kwa muhtasari, ingawa taarifa za `switch` na minyororo ya `if/else if` zote huruhusu mantiki ya matawi mengi katika msimbo wako, zina nguvu tofauti. Taarifa za `switch` zina ufanisi katika kushughulikia thamani nyingi zinazowezekana kwa kigezo kimoja, wakati minyororo ya `if/else if` inatoa kubadilika zaidi kwa masharti magumu. Uchaguzi kati yao mara nyingi hutegemea mahitaji maalum ya msimbo wako na mapendeleo ya mtindo wa uandishi wa msimbo binafsi au wa timu.
