# --description--

「そして彼は谷へ向かいました。その谷を川が流れており、谷のボーダーは森林に覆われていて、川の両側には平らな牧草地がありました。川の一方には白い羊の群れが、もう一方には黒い羊の群れがいました。白い羊のどれかが鳴くと、黒い羊のどれかが川を渡って白くなり、黒い羊のどれかが鳴くと、白い羊のどれかが川を渡って黒くなりました。」 - Evrawcの息子Peredur

最初は、それぞれの群れに$n$匹の羊がいます。どの羊（色に関係なく）が次に鳴くかは等確率です。羊が鳴いて他方の群れから羊が渡った後、Peredurは白い羊の数を調整して、最終的な黒い羊の期待値を最大化できます。$E(n)$をPeredurが最適戦略を使った場合の黒い羊の最終的な期待値とします。

$E(5) = 6.871346$（小数点以下6桁に四捨五入済み）であることが与えられています。

$E(10\,000)$を求め、小数点以下6桁に四捨五入してください。
