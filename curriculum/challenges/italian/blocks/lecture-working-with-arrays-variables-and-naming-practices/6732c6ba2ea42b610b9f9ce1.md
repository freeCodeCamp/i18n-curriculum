# --description--

Nelle lezioni precedenti ti è stata presentata per la prima volta la proprietà `length`, questa proprietà restituisce il numero di elementi in un array. Ecco un rapido promemoria su come funziona:

```js
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits.length); // 3
```

È possibile avere array con slot vuoti. Gli slot vuoti sono definiti come slot senza nulla al loro interno. Questo è diverso da un array con il valore `undefined`. Questi tipi di array sono noti come array sparsi. Ecco un esempio:

```js
const sparseArray = [1, , , 4];
console.log(sparseArray.length); // 4
```

In questo caso, anche se abbiamo solo due elementi definiti, `1` e `4`, la lunghezza è `4` perché l'indice più alto (`3`) più `1` ci dà una lunghezza di `4`.

Ora discutiamo come creare un array vuoto di lunghezza fissa. Ci sono diversi modi per farlo in JavaScript ma un metodo comune è usare il costruttore `Array()` con un argomento numerico. Il costruttore `Array()` può essere usato con la parola chiave `new` per creare un nuovo array. Ecco un esempio:

```js
const emptyArray = new Array(5);
console.log(emptyArray.length); // 5
console.log(emptyArray); // [ , , , , ]
```

In questo esempio, creiamo un nuovo array usando `Array(5)`. Questo crea un array sparso con una lunghezza di `5` dove tutti gli slot sono vuoti.

Un altro modo per creare un array vuoto di lunghezza fissa è usare il metodo `Array.from()` con un argomento di lunghezza. A differenza di `new Array(n)`, questo metodo crea un array della lunghezza specificata dove tutti gli elementi esistono e hanno un valore `undefined`:

```js
const fixedLengthArray = Array.from({ length: 5 });
console.log(fixedLengthArray.length); // 5
console.log(fixedLengthArray); // [undefined, undefined, undefined, undefined, undefined]
```

Se vuoi creare un array di lunghezza specifica e riempirlo con un valore predefinito, puoi usare il metodo `Array.fill()`:

```js
const filledArray = new Array(3).fill(0);
console.log(filledArray); // [0, 0, 0]
```

Questo crea un array di lunghezza tre e riempie tutti gli elementi con il valore `0`. Nota: quando si riempie con oggetti, tutti gli slot fanno riferimento allo stesso oggetto; se hai bisogno di copie indipendenti, usa una callback o `Array.from()` invece.

Capire come ottenere la lunghezza di un array e creare array di lunghezza fissa è importante per molte attività di programmazione, specialmente quando devi inizializzare array per algoritmi o strutture dati specifiche.
