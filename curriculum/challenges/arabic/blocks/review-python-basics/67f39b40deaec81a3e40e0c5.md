## ما هو Python؟

- **مقدمة**: Python هي لغة برمجة عامة الغرض معروفة ببساطتها وسهولة استخدامها. تُستخدم Python في مجالات عديدة مثل علم البيانات وتعلم الآلة، تطوير الويب، البرمجة النصية والأتمتة، الأنظمة المدمجة وإنترنت الأشياء، وأكثر من ذلك.
- **حالات الاستخدام الشائعة**: تُستخدم Python في علم البيانات، تعلم الآلة، تطوير الويب، الأمن السيبراني، الأتمتة والميكروكمبيوترات مثل Raspberry Pi ولوحات متوافقة مع MicroPython.

## المتغيرات

- **إعلان المتغيرات**: لإعلان متغير، تبدأ باسم المتغير متبوعًا بمعامل التعيين (`=`) ثم القيمة. يمكن أن تكون القيمة رقمًا، سلسلة نصية، قيمة منطقية، إلخ. إليك بعض الأمثلة:

```py
name = 'John Doe'
age = 25
```

- **اصطلاحات تسمية المتغيرات**: إليك الاصطلاحات التي يجب اتباعها لتسمية المتغيرات:

- أسماء المتغيرات يمكن أن تبدأ فقط بحرف أو بشرطة سفلية (_) وليس برقم.
  - أسماء المتغيرات يمكن أن تحتوي فقط على أحرف أبجدية رقمية (a-z, A-Z, 0-9) والشرطة السفلية (_).
  - أسماء المتغيرات حساسة لحالة الأحرف — `age`، `Age`، و`AGE` تعتبر أسماء مختلفة.
  - لا يمكن أن تكون أسماء المتغيرات من الكلمات المفتاحية المحجوزة في Python مثل `if`، `class`، أو `def`.
  - أسماء المتغيرات متعددة الكلمات تُفصل بالشرطة السفلية. مثال: `snake_case`.


## التعليقات

- **تعليقات السطر الواحد**: تُستخدم هذه التعليقات لكتابة ملاحظات قصيرة تريد تركها في الشفرة.

```py
# This is a single line comment
```

- **السلاسل النصية متعددة الأسطر**: يمكن استخدام هذه السلاسل لترك ملاحظات أكبر أو لإضافة تعليق على أجزاء من الشفرة.

```py
"""
This is a multi-line string.
Here is some code commented out.

name = 'John Doe'
age = 25
"""
```

- **الدالة `print()`**: لطباعة البيانات في وحدة التحكم، يمكنك استخدام الدالة `print()` بهذا الشكل:

```py
print('Hello world!') # Hello world!
```

## أنواع البيانات الشائعة في Python

- **مقدمة**: Python لغة ذات نوع ديناميكي مثل JavaScript، مما يعني أنك لا تحتاج إلى إعلان أنواع المتغيرات صراحة. اللغة تعرف نوع المتغير بناءً على القيمة التي تعينها له.
- **عدد صحيح**: رقم كامل بدون كسور عشرية:

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **عدد عشري**: رقم يحتوي على كسور عشرية:

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **سلسلة نصية**: متتالية من الأحرف محاطة بعلامات اقتباس:

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **قيمة منطقية**: قيمة تمثل إما `True` أو `False`:

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **مجموعة**: مجموعة غير مرتبة من العناصر الفريدة:

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **قاموس**: مجموعة من أزواج المفتاح-القيمة، محاطة بأقواس متعرجة:

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **صف مرتب**: مجموعة مرتبة غير قابلة للتغيير، محاطة بأقواس عادية:

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **متتالية أعداد**: متتالية من الأعداد، تُستخدم غالبًا في الحلقات:

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **قائمة**: مجموعة مرتبة من العناصر تدعم أنواع بيانات مختلفة:

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**: قيمة خاصة تمثل غياب القيمة:

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## الأنواع غير القابلة للتغيير والقابلة للتغيير

- **الأنواع غير القابلة للتغيير**: هذه الأنواع لا تتغير بعد إعلانها، رغم أنه يمكنك إعادة تعيين متغيراتها إلى شيء جديد. تشمل الأعداد الصحيحة، العشرية، المركبة، القيمة المنطقية، السلسلة النصية، الصف المرتب، المتتالية، و`None`.
- **الأنواع القابلة للتغيير**: هذه الأنواع يمكن تغييرها بعد إعلانها. يمكنك إضافة أو إزالة أو تحديث عناصرها. تشمل أنواع المجموعات مثل القائمة، المجموعة، والقاموس.
- **الدالة `type()`**: لرؤية نوع متغير، يمكنك استخدام الدالة `type()` بهذا الشكل:

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **الدالة `isinstance()`**: تُستخدم للتحقق مما إذا كان المتغير يطابق نوع بيانات معين:

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## العمل مع السلاسل النصية

- **تعريف**: كما تتذكر من JavaScript، السلاسل النصية غير قابلة للتغيير، مما يعني أنه لا يمكنك تعديلها بعد إنشائها. في Python، يمكنك استخدام علامات اقتباس مفردة أو مزدوجة. يُفضل اختيار قاعدة والالتزام بها:

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **الوصول إلى الأحرف في السلاسل النصية**: يمكنك الوصول إلى الأحرف باستخدام ترميز الأقواس بهذا الشكل:

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **الهروب من الأحرف في السلاسل النصية**: يمكنك استخدام الشرطة المائلة العكسية (`\`) إذا كانت السلسلة تحتوي على علامات اقتباس بهذا الشكل:

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **ربط السلاسل النصية**: لربط السلاسل النصية، يمكنك استخدام معامل `+` بهذا الشكل:

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

طريقة أخرى لربط السلاسل النصية هي باستخدام معامل `+=`. يُستخدم هذا لإجراء الربط والتعيين في خطوة واحدة بهذا الشكل:

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**: هذه اختصار لـ formatted string literals. تسمح بالتعامل مع الاستيفاء وأيضًا إجراء بعض الربط بصيغة مختصرة وقابلة للقراءة:

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **تقطيع السلاسل النصية**: هو استخراج أجزاء من السلسلة النصية. إليك الصيغة الأساسية:

```py
str[start:stop:step]
```

تمثل نقطة البداية الفهرس الذي يبدأ منه الاستخراج. نقطة النهاية هي حيث ينتهي التقطيع، وهذه النقطة غير شاملة. تمثل خطوة التقطيع الفاصل الزمني للزيادة. إليك بعض الأمثلة:

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **الحصول على طول السلسلة النصية**: تُستخدم الدالة `len()` لإرجاع عدد الأحرف في السلسلة:

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## العمل مع معامل `in`

- **معامل `in`**: يُرجع قيمة منطقية تحدد ما إذا كان الحرف أو الأحرف موجودة في السلسلة النصية أم لا:

```py
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

## طرق السلاسل النصية الشائعة

- **`str.upper()`**: تُرجع سلسلة جديدة مع تحويل جميع الأحرف إلى أحرف كبيرة:

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`str.lower()`**: تُرجع سلسلة جديدة مع تحويل جميع الأحرف إلى أحرف صغيرة:

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`str.strip()`**: تُرجع نسخة من السلسلة مع إزالة الأحرف المحددة من البداية والنهاية (إذا لم تُمرر معلمة، تزيل الفراغات من البداية والنهاية).

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**: تُرجع سلسلة جديدة مع استبدال كل تكرارات السلسلة القديمة بسلسلة جديدة.

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**: تُستخدم لتقسيم سلسلة إلى قائمة باستخدام فاصل محدد. الفاصل هو سلسلة تحدد مكان التقسيم.

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**: تُستخدم لضم عناصر قابلة للتكرار إلى سلسلة نصية بفاصل. القابل للتكرار هو مجموعة من العناصر يمكن التكرار عليها مثل القائمة، السلسلة النصية أو الصف المرتب.

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`str.startswith(prefix)`**: تُرجع قيمة منطقية تشير إلى ما إذا كانت السلسلة تبدأ بالبادئة المحددة:

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`str.endswith(suffix)`**: تُرجع قيمة منطقية تشير إلى ما إذا كانت السلسلة تنتهي باللاحقة المحددة:

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`str.find()`**: تُرجع الفهرس لأول ظهور لسلسلة فرعية. إذا لم تُعثر عليها، تُرجع `-1`:

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`str.count(substring)`**: تحصي عدد مرات ظهور سلسلة فرعية في السلسلة:

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`str.capitalize()`**: تُرجع سلسلة جديدة مع تحويل الحرف الأول إلى كبير وباقي الأحرف إلى صغيرة:

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`str.isupper()`**: تُرجع `True` إذا كانت كل الأحرف في السلسلة كبيرة و`False` خلاف ذلك:

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`str.islower()`**: تُرجع `True` إذا كانت كل الأحرف في السلسلة صغيرة و`False` خلاف ذلك:

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`str.title()`**: تُرجع سلسلة جديدة مع تحويل الحرف الأول من كل كلمة إلى كبير:

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`str.maketrans()`**: تُستخدم هذه الطريقة لإنشاء جدول تحويل 1 إلى 1 للأحرف للترجمة. غالبًا ما تُستخدم مع طريقة `translate()` التي تطبق هذا الجدول على السلسلة وتُرجع النتيجة المترجمة.

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## العمليات الشائعة المستخدمة مع الأعداد الصحيحة والعشرية

- **العمليات الحسابية الأساسية**: في Python، يمكنك إجراء العمليات الحسابية الأساسية مع الأعداد الصحيحة والعشرية مثل الجمع، الطرح، الضرب والقسمة:

```py
int_1 = 56
int_2 = 12
float_1 = 5.4
float_2 = 12.0

# Addition

print('Integer Addition:', int_1 + int_2) # Integer Addition: 68
print('Float Addition:', float_1 + float_2) # Float Addition: 17.4

# Subtraction

print('Int Subtraction:', int_1 - int_2) # Int Subtraction: 44
print('Float Subtraction:',  float_2 - float_1) # Float Subtraction: 6.6

# Multiplication

print('Int Multiplication:', int_1 * int_2) # Int Multiplication: 672
print('Float Multiplication:', float_2 * float_1) # Float Multiplication: 64.80000000000001

# Division

print('Division:', int_1 / int_2) # Division: 4.666666666666667
print('Float Division:', float_2 / float_1) # Float Division: 2.222222222222222
```

عند جمع عدد عشري وعدد صحيح، تتحول النتيجة إلى عدد عشري بهذا الشكل:

```py
int_1 = 56
float_1 = 5.4

print(int_1 + float_1) # 61.4
```

- **معامل باقي القسمة (`%`)**: يُرجع الباقي عند قسمة رقم على رقم آخر:

```py
int_1 = 56
int_2 = 12

print(int_1 % int_2) # 8
```

- **القسمة الأرضية (`//`)**: يُستخدم هذا المعامل لقسمة رقمين مع تقريب النتيجة إلى أقرب عدد صحيح أقل أو يساوي الناتج:

```py
int_1 = 56
int_2 = 12

print(int_1 // int_2) # 4
```

- **معامل الأس (`**`)**: يُستخدم لرفع رقم إلى قوة رقم آخر:

```py
int_1 = 4
int_2 = 2

print(int_1 ** int_2) # 16
```

- **الدالة `float()`**: يمكنك استخدام هذه الدالة لتحويل عدد صحيح إلى عدد عشري.

```py
num = 4

print(float(num)) # 4.0
```

- **الدالة `int()`**: يمكنك استخدام هذه الدالة لتحويل عدد عشري إلى عدد صحيح.

```py
num = 4.0

print(int(num)) # 4
```

- **الدالة `round()`**: تُستخدم لتقريب رقم إلى أقرب عدد صحيح:

```py
num_1 = 3.4
num_2 = 7.7

print(round(num_1)) # 3
print(round(num_2)) # 8
```

- **الدالة `abs()`**: تُستخدم لإرجاع القيمة المطلقة لرقم:

```py
num = -13

print(abs(num)) # 13
```

- **الدالة `pow()`**: تُستخدم لرفع رقم إلى قوة رقم آخر:

```py
result = pow(2, 3) 
print(result)  # 8
```

## التعيينات المعززة

- **تعريف**: التعيين المعزز يجمع بين عملية ثنائية مع التعيين في خطوة واحدة. يأخذ متغيرًا، يطبق عليه عملية مع قيمة أخرى، ويخزن النتيجة في نفس المتغير.

```py
# Addition assignment 
my_var = 10
my_var += 5

print(my_var) # 15

# Subtraction assignment
count = 14
count -= 3

print(count) # 11

# Multiplication assignment 
product = 65
product *= 7

print(product) # 455

# Division assignment 
price = 100
price /= 4

print(price) # 25.0

# Floor Division assignment 
total_pages = 23
total_pages //= 5

print(total_pages) # 4

# Modulo assignment 
bits = 35
bits %= 2

print(bits) # 1

# Exponentiation assignment 
power = 2
power **= 3

print(power) # 8
```

هناك معاملات تعيين معززة أخرى، مثل تلك الخاصة بالعمليات على مستوى البت. تشمل `&=`، `^=`، `>>=`، و`<<=`.

## العمل مع الدوال

- **تعريف**: الدوال هي قطع قابلة لإعادة الاستخدام من الشفرة تأخذ مُدخلات (معلمات) وتُرجع مخرجات. لاستدعاء دالة، تحتاج إلى ذكر اسم الدالة متبوعًا بمجموعة من الأقواس العادية:

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call
print(result) # 7
```

إذا لم تُرجع الدالة قيمة صراحة، فإن قيمة الإرجاع الافتراضية هي `None`:

```py
def greet():
    print('hello') 

result = greet() # hello
print(result) # None
```

يمكنك أيضًا تزويد المعلمات بقيم افتراضية بهذا الشكل:

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

إذا استدعيت الدالة بعدد غير صحيح من المعلمات، ستحصل على `TypeError`:

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

## الدوال المدمجة الشائعة

- **الدالة `input()`**: تُستخدم لطلب مُدخل من المستخدم:

```py
name = input('What is your name?') # User types 'Kolade' and presses Enter  
print('Hello', name) # Hello Kolade
```

- **الدالة `int()`**: تُستخدم لتحويل رقم أو قيمة منطقية أو سلسلة رقمية إلى عدد صحيح:

```py
print(int(3.14)) # 3
print(int('42')) # 42
print(int(True)) # 1
print(int(False)) # 0 
```

## النطاق في Python

- **النطاق المحلي**: هو عندما يُعلن متغير داخل دالة أو صنف ولا يمكن الوصول إليه إلا داخل تلك الدالة أو الصنف.

```py
def my_func():
    num = 10
    print(num)
```

- **النطاق المحيط**: هو عندما تكون دالة متداخلة داخل دالة أخرى ويمكنها الوصول إلى متغيرات الدالة التي تحتويها.

```py
def outer_func():
    msg = 'Hello there!'
    
    def inner_func():
        print(msg)
    inner_func()

print(outer_func()) # Hello there!
```

- **النطاق العالمي**: يشير إلى المتغيرات المُعلنة خارج أي دوال أو أصناف ويمكن الوصول إليها من أي مكان في البرنامج.

```py
tax = 0.70 

def get_total(subtotal):
    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **النطاق المدمج**: أسماء محجوزة في Python للدوال والوحدات والكلمات المفتاحية والكائنات المعرفة مسبقًا.

```py
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```

print(3 < 4) # True

## العمل مع عبارات `if`، `elif` و`else`

- **عبارات `if`**: هي شروط تُستخدم لتحديد ما إذا كان شيء ما صحيحًا أم لا. إذا قُيّمت الشرط إلى `True`، يتم تنفيذ ذلك الجزء من الشفرة.

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **بند `elif`**: هي شروط تأتي بعد عبارة `if`. يُنفذ بند `elif` فقط إذا كانت كل الشروط السابقة `False` وشرطه الخاص `True`.

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **بند `else`**: يُنفذ إذا لم تُحقق أي من الشروط الأخرى قيمة `True`.

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

يمكنك أيضًا استخدام عبارات `if` متداخلة بهذا الشكل:

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## القيم الصحيحة والخاطئة

- **تعريف**: في Python، لكل قيمة قيمة منطقية ضمنية، أو إحساس مدمج بما إذا كان يجب معاملتها كـ `True` أو `False` في سياق منطقي. العديد من القيم تُعتبر صحيحة، أي تُقيّم إلى `True` في السياق المنطقي. وأخرى تُعتبر خاطئة، أي تُقيّم إلى `False`. إليك بعض الأمثلة على القيم الخاطئة:

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

القيم الأخرى مثل الأعداد غير الصفرية والسلاسل غير الفارغة تُعتبر صحيحة.

## العمل مع الدالة `bool()`

- **تعريف**: إذا أردت التحقق مما إذا كانت القيمة صحيحة أو خاطئة، يمكنك استخدام الدالة المدمجة `bool()`. تقوم بتحويل القيمة صراحة إلى مكافئها المنطقي وتُرجع `True` للقيم الصحيحة و`False` للقيم الخاطئة. إليك بعض الأمثلة:

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## معاملات القيمة المنطقية والقطع المبكر

- **تعريف**: هذه معاملات خاصة تسمح لك بدمج عدة تعبيرات لإنشاء منطق اتخاذ قرار أكثر تعقيدًا في الشفرة. هناك ثلاثة معاملات منطقية في Python: `and`، `or`، و`not`.
- **معامل `and`**: يأخذ معاملين ويُرجع الأول إذا كان خاطئًا، وإلا يُرجع الثاني. يجب أن يكون كلا المعاملين صحيحين لكي تكون النتيجة صحيحة.

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

يمكنك أيضًا استخدام معامل `and` في الشروط بهذا الشكل:

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **عامل `or`**: يعيد هذا العامل المعامل الأول إذا كانت قيمته صحيحة، وإلا يعيد المعامل الثاني. تعبير `or` ينتج قيمة صحيحة إذا كان أحد المعاملين على الأقل قيمة صحيحة. إليك مثالًا:

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

تمامًا كما في عامل `and`، يمكنك استخدام عامل `or` في العبارات الشرطية هكذا:

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **التوقفات (Short-circuiting)**: يُعرف عاملَي `and` و`or` بأنهما عوامل توقفات. التوقفات تعني أن بايثون يتحقق من القيم من اليسار إلى اليمين ويتوقف بمجرد أن يحدد النتيجة النهائية.
- **عامل `not`**: يأخذ هذا العامل معاملًا واحدًا ويعكس قيمته المنطقية. يحول القيم الصحيحة إلى `False` والقيم الخاطئة إلى `True`. على عكس العوامل السابقة، يعيد `not` دائمًا `True` أو `False`. إليك بعض الأمثلة:

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

إليك مثالًا على استخدام عامل `not` في عبارة شرطية:

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```
