# --description--

Tamko la `flex` ni sifa ya ufupisho kwa kweli kwa sifa tatu ambazo unaweza kuweka kwenye kipande cha kubadilika. Sifa hizi huathiri jinsi vipande vya kubadilika vinavyojipima ndani ya kontena lao. Umeona baadhi ya sifa za ufupisho hapo awali, lakini bado hujazitaja rasmi.

> Sifa za ufupisho ni sifa za CSS zinazokuruhusu kuweka thamani za sifa nyingine nyingi za CSS kwa wakati mmoja. Kwa kutumia sifa ya ufupisho, unaweza kuandika majedwali ya mitindo yenye ufupi zaidi (na mara nyingi yenye kusomeka zaidi), ukihifadhi muda na nguvu.

Katika kesi hii, `flex` ni sifa ya ufupisho kwa `flex-grow`, `flex-shrink` na `flex-basis`.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/flex-box/flexbox-04.png" alt="CSS code setting the flex property to 1 for a div element.">

Katika picha hapo juu, `flex: 1` inalingana na: `flex-grow: 1`, `flex-shrink: 1`, `flex-basis: 0`.

Mara nyingi sana, unaona sifa ya ufupisho ya flex ikielezwa na thamani moja tu. Katika hali hiyo, thamani hiyo hutumika kwa `flex-grow`. Hivyo wakati uliweka `flex: 1` kwenye div zetu, kwa kweli ulikuwa unataja sifa ya ufupisho ya `flex: 1 1 0`.
