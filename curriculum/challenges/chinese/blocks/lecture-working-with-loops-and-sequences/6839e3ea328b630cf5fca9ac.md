# --description--

正如你在之前的模块中学到的，循环用于重复执行一段代码块指定的次数。在本课中，你将学习如何使用 Python 中的不同类型的循环。

我们将首先讲解的是 `for` 循环。下面是一个使用 `for` 循环迭代列表并将每个项打印到控制台的示例：

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)
```

结果将是：

```md
Rust
Java
Python
C++
```

注意 `print(language)` 在 `loop` 内部有缩排。没有那个缩排，你会得到一个 `IndentationError`：

```python
"""
Traceback (most recent call last):
  File "<stdin>", line 4, in <module>
IndentationError: expected an indented block after 'for' statement on line 3
"""
```

你也可以使用 `for` 循环来迭代其他可迭代对象，比如字串。下面是一个使用 `for` 循环遍历字串 `code` 并打印出每个字符的示例：

```py
for char in 'code':
    print(char)
```

结果将是：

```md
c
o
d
e
```

就像在 JavaScript 中一样，你也可以在 Python 中嵌套 `for` 循环。下面是使用嵌套 `for` 循环的示例：

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)
```

外层循环将迭代 `categories` 列表中的每个 `category`。对于每个 `category`，内层循环将迭代 `foods` 列表中的每个 `food`。以下是将打印到控制台的结果：

```md
Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
```

你可以在 Python 中使用的另一种循环类型是 `while` 循环。这种循环类型会重复执行一段代码块，直到条件为 `False`。下面是使用 `while` 循环进行猜数字游戏的示例：

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')
```

在这个例子中，我们有一个值为 `3` 的 `secret_number` 变量和一个初始猜测值为 `0`。然后我们使用 `input` 函数从用户获取输入，再用 `int()` 函数将输入的字串转换为整数，并赋值给 `guess` 变量。如果用户通过输入 `3` 猜对了，`while` 循环将被跳出，并打印消息 `You got it!` 到控制台。否则，打印消息 `Wrong! Try again.` 到控制台，循环重复，提示用户再次猜测。

这是一个示例结果：

```md
Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
```

就像在 JavaScript 中一样，Python 支持 `break` 和 `continue` 语句。

`break` 语句用于停止循环的执行。以下是使用 `break` 语句处理 `developer_names` 列表的示例：

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

在这个例子中，我们迭代一个 `developer_names` 列表并将每个名字打印到控制台。如果名字与 `Naomi` 相等，那么我们就跳出循环。这样只会将名字 `Jess` 打印到控制台。

`continue` 语句用于跳过循环的当前迭代并进入下一次迭代。让我们修改之前的示例，使用 `continue` 语句代替 `break`：

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

现在控制台中的结果将不同。因为当 `developer` 与 `Naomi` 相等时，`continue` 语句跳过了循环的第二次迭代，所以打印了 `Jess` 和 `Tom` 这两个名字，而没有将该名字打印到控制台。

`for` 和 `while` 循环都可以与 `else` 子句结合使用，只有当循环没有被 `break` 语句终止时才会执行该子句。下面是使用多个 `for` 循环的示例：

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

在这个例子中，我们有一个随机单词的列表，使用 `for` 循环遍历每个单词。在外层的 `for` 循环内部，我们有另一个 `for` 循环遍历每个单词的每个字母。如果字母的小写版本是元音，我们打印该单词及其包含的元音，然后跳出内层循环。如果单词不包含元音，则打印一条指示该情况的消息。

以下是在控制台中显示的结果：

```md
'sky' has no vowels
'apple' contains the vowel 'a'
'rhythm' has no vowels
'fly' has no vowels
'orange' contains the vowel 'o'
```

循环在 Python 中非常常见，因此熟悉它们非常重要。在接下来的几节课中，你将学习如何在循环中使用 `enumerate()` 和 `range()` 函数。
