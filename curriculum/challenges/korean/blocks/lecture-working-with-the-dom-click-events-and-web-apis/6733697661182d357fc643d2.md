# --interactive--

`Canvas` API는 JavaScript 파일 안에서 그래픽을 조작할 수 있게 해주는 강력한 도구입니다. 모든 것은 HTML의 `canvas` 요소에서 시작합니다. 이 요소는 `Canvas` API의 인스턴스 메서드와 속성을 사용해 조작할 수 있는 그리기 표면 역할을 합니다.

`Canvas` API는 도형, 텍스트, 애니메이션, 심지어 복잡한 게임까지 만들 수 있는 모든 것을 제공합니다. `HTMLCanvasElement`, `CanvasRenderingContext2D`, `CanvasGradient`, `CanvasPattern`, `TextMetrics` 같은 인터페이스가 있어 JavaScript 파일에서 그래픽을 만들 때 사용할 수 있는 메서드와 속성을 제공합니다.

`Canvas` API가 어떻게 작동하는지 살펴보고 JavaScript로 그래픽을 만드는 방법을 시작해 봅시다.

먼저 HTML 파일에 `canvas` 요소를 만들어야 합니다:

```html
<canvas id="my-canvas"></canvas>
```

`canvas` 요소는 `HTMLCanvasElement` 인터페이스로 나타내며, 이 인터페이스는 조작을 위한 메서드와 속성을 제공합니다. 또한 `Canvas` API의 다른 인터페이스에서 메서드와 속성을 사용할 수 있습니다.

HTML 안에서 `canvas`에 `width`와 `height`을 줄 수 있습니다:

```html
<canvas id="my-canvas" width="400" height="400"></canvas>
```

또는 `width` 인터페이스의 `height`와 `HTMLCanvasElement` 속성을 사용할 수 있습니다:

```js
const canvas = document.getElementById("my-canvas");
canvas.width = 400;
canvas.height = 400;
```

지금은 화면에 아무것도 보이지 않습니다. `canvas` 요소를 만든 후 다음으로 해야 할 일은 `getContext()` 인터페이스의 `HTMLCanvasElement` 메서드를 사용해 캔버스의 그리기 컨텍스트에 접근하는 것입니다.

가장 일반적인 컨텍스트는 2차원으로 그릴 수 있는 `2d`입니다:

```js
const canvas = document.getElementById("my-canvas");
const ctx = canvas.getContext('2d');
```

`ctx` 변수를 콘솔에 출력하면 도형, 색상, 선 등을 만들 때 사용할 수 있는 `CanvasRenderingContext2D`의 메서드와 속성, 그리고 기본값을 볼 수 있습니다:

```js
console.log(ctx);
```

2D 컨텍스트를 얻으면 캔버스에 그리기를 시작할 수 있습니다.

`Canvas` API는 도형, 선, 텍스트를 그리기 위한 여러 메서드와 속성을 제공합니다. 그중 하나가 `fillStyle` 속성으로, `fillRect()` 메서드와 결합해 사각형이나 정사각형을 그릴 수 있습니다:

:::interactive_editor

```html
<html>
  <head>
  </head>
  <body>
    <canvas id="my-canvas" width="400" height="400"></canvas>
    <script src="index.js"></script>
  </body>
</html>
```

```js
const canvas = document.getElementById("my-canvas");

const ctx = canvas.getContext("2d");

// Set the background color
ctx.fillStyle = "crimson";

// Draw a rectangle
ctx.fillRect(1, 1, 150, 100);
```

:::

`fillRect`는 각각 x축, y축, 너비, 높이를 나타내는 4개의 숫자 값을 받습니다.

이제 화면에 무언가 나타났습니다. 텍스트를 그리거나 애니메이션을 만들 수도 있습니다. 텍스트를 나타내는 캔버스는 다음과 같습니다:

```html
<canvas id="my-text-canvas" width="300" height="70"></canvas>
```

마지막으로 텍스트를 그리려면 텍스트를 첫 번째 인수로 `fillText()` 메서드에 전달하고, 그다음에 x축과 y축 값을 넣으세요:

:::interactive_editor

```html
<html>
  <head>
  </head>
  <body>
    <canvas id="my-text-canvas" width="300" height="70"></canvas>
    <script src="index.js"></script>
  </body>
</html>
```

```js
const textCanvas = document.getElementById("my-text-canvas");

const textCanvasCtx = textCanvas.getContext("2d");

// Set font family and size
textCanvasCtx.font = "30px Arial";

// Set text color
textCanvasCtx.fillStyle = "crimson";

// Draw the text
textCanvasCtx.fillText("Hello HTML Canvas!", 1, 50);
```

:::

브라우저 결과는 빨간색 텍스트 `Hello HTML Canvas!`가 될 것입니다.

`Canvas` API로 할 수 있는 일이 훨씬 더 많습니다. 예를 들어 `requestAnimationFrame`와 결합해 맞춤 애니메이션, 시각화, 게임 등을 만들 수 있습니다.
