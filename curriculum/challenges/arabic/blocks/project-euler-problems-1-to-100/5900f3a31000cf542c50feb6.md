# --description--

إذا أخذنا 47، وعكسناه ثم جمعناه، 47 + 74 = 121، وهو عدد متناظر.

ليست كل الأعداد تُنتج أعدادًا متناظرة بهذه السرعة. على سبيل المثال،

<div style="margin-left: 4em;">
  349 + 943 = 1292,<br>
  1292 + 2921 = 4213<br>
  4213 + 3124 = 7337<br>
</div>

أي أن 349 استغرق ثلاث تكرارات ليصل إلى عدد متناظر.

رغم أن أحدًا لم يثبت ذلك بعد، يُعتقد أن بعض الأعداد، مثل 196، لا تُنتج عددًا متناظرًا أبدًا. العدد الذي لا يُكوّن عددًا متناظرًا عبر عملية العكس والجمع يُسمى عدد ليشريل. وبسبب الطبيعة النظرية لهذه الأعداد، ومن أجل هذه المشكلة، سنفترض أن العدد ليشريل حتى يثبت العكس. بالإضافة إلى ذلك، يُعطى أنه لكل عدد أقل من عشرة آلاف، إما (1) أن يصبح عددًا متناظرًا في أقل من خمسين تكرارًا، أو (2) لم يتمكن أحد، بكل قوة الحوسبة الموجودة، من تحويله إلى عدد متناظر حتى الآن. في الواقع، 10677 هو أول عدد ثبت أنه يحتاج لأكثر من خمسين تكرارًا قبل أن يُنتج عددًا متناظرًا: 4668731596684224866951378664 (53 تكرارًا، 28 رقمًا).

من المدهش أن هناك أعدادًا متناظرة هي نفسها أعداد ليشريل؛ وأول مثال هو 4994.

كم عدد أعداد ليشريل الموجودة تحت `num`؟

**ملاحظة:** تم تعديل الصياغة قليلاً في 24 أبريل 2007 لتأكيد الطبيعة النظرية لأعداد ليشريل.

```js
const countLychrelNumbers = (size) => {
  const numReverse = (num) => {
    return Number(num.toString().split('').reverse().join(''));
  };
  const isPalin = (num) => {
    if (numReverse(num) === num) {
      return true;
    }
    return false;
  };
  let total = 0;
  for (let i = 1; i < size; i++) {
    let loopCount = 1;
    let sum = i;
    while (loopCount < 50) {
      sum = sum + numReverse(sum);
      if (isPalin(sum)) {
        break;
      } else {
        loopCount++;
      }
    }
    if (loopCount === 50) {
      total++;
    }
  }
  return total;
}
```
