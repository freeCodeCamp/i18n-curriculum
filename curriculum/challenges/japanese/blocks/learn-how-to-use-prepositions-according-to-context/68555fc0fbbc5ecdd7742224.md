# --explanation--

`On`は`planning on`というフレーズで使われており、何かをするつもりや期待していることを意味します。例えば：

`We're planning on launching next week.` – これはチームがそれをするつもりや期待していることを意味します。

`Bringing in`はチームやグループに人を加えることを意味します。例えば：

`The manager is bringing in a new developer.` – これは誰かが加えられていることを意味します。

`Additional`はすでに存在するものよりも多いことを意味します。例えば：

`We need additional help to finish the project.` – これは追加の助けが必要であることを意味します。
