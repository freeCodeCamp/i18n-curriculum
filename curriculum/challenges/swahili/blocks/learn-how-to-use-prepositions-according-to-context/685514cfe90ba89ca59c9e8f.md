# --explanation--

Katika `by next month`, `by` inaonyesha tarehe ya mwisho, ambayo inamaanisha jambo lazima lifanyike kabla au si baadaye ya wakati huo.

Sentensi `We need to finish the report by Friday` inatumia `by` kwa njia ile ile, ikipanga kikomo cha muda wazi.
