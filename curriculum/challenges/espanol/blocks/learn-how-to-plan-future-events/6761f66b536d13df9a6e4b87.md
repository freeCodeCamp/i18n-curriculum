`parameterized`

### --feedback--

Esta palabra describe consultas que usan valores predefinidos para mejorar la seguridad. Es una palabra larga que se usa frecuentemente en contextos de bases de datos.

# --explanation--

`Parameterized` proviene de la palabra raíz `parameter`, que es un valor o marcador de posición usado para controlar cómo funciona algo.  

Cuando la gente dice que una consulta está `parameterized`, significa que usa valores predefinidos (parámetros) en lugar de insertar directamente la entrada del usuario. 

Esto ayuda a mejorar la seguridad de la base de datos al prevenir ciertos ataques como la inyección SQL. Por ejemplo:  

- `The developer used parameterized queries to secure the database.` - Esto significa que usaron marcadores de posición para asegurar que no se pudiera insertar código dañino.
