# --description--

Ora che conosci la sintassi base di HTML e CSS, è il momento di fare sul serio. Le competenze più importanti da padroneggiare con CSS sono il posizionamento e il layout. Cambiare caratteri tipografici e colori è una capacità fondamentale, ma riuscire a mettere gli elementi esattamente dove vuoi su una pagina web è ancora più importante. Dopotutto, quante pagine web puoi trovare in cui ogni elemento è semplicemente impilato uno sopra l’altro?

Imparare a posizionare gli elementi su una pagina web non è così difficile una volta che comprendi alcuni concetti chiave. Purtroppo, molti studenti corrono attraverso l’apprendimento di HTML e CSS per arrivare a JavaScript e finiscono per perdere questi concetti fondamentali. Questo porta a frustrazione, difficoltà (e gif divertenti) perché tutte le competenze di JavaScript del mondo sono inutili se non riesci a mettere gli elementi sulla pagina dove devono stare. Detto questo, iniziamo.
