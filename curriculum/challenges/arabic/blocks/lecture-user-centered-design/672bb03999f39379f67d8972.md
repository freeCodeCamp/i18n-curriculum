# --interactive--

**ملاحظة**: قد تستخدم بعض الأمثلة التفاعلية CSS لم تتعلمها بعد. لا تقلق بشأن محاولة فهم كل الشفرة. الهدف من الأمثلة هو عرض معاينات لهذه المفاهيم التصميمية لتفهم كيف تعمل الأمور بشكل أفضل. لعرض المعاينات، يجب تفعيل المحرر التفاعلي.

الكشف التدريجي هو نمط تصميم يُستخدم لعرض المحتوى ذي الصلة فقط للمستخدمين بناءً على نشاطهم الحالي وإخفاء الباقي. يُجرى ذلك لتقليل الحمل الذهني وجعل تجربة المستخدم أكثر بديهية.

:::interactive_editor

```js
const btn = document.getElementById('show-details-btn');
const content = document.getElementById('extra-content');

btn.addEventListener('click', () => {
  const isHidden = content.classList.toggle('hidden');
  btn.setAttribute('aria-expanded', !isHidden);
  if (!isHidden) {
    content.focus();
  }
});

```

على سبيل المثال، عند استخدام صفحة البحث في Google، ستكون معظم عمليات البحث بسيطة. لذا من المنطقي وجود واجهة مستخدم بسيطة مع خيارات قليلة جدًا. لكن في الحالات القليلة التي تحتاج فيها إلى بحث متقدم، توفر Google خيار بحث متقدم يكشف عن المزيد من الخيارات.

يمكنك العثور على خيار البحث المتقدم في قائمة الإعدادات على الصفحة الرئيسية لـ Google. أو يمكنك الذهاب إلى `google.com/advanced_search`.

مثال آخر على الكشف التدريجي هو زر "المزيد من التفاصيل" على المنتجات في موقع تجارة إلكترونية. عند زيارة موقع مثل Amazon، سترى قائمة بصور المنتجات في فئات مختلفة. هذا يسمح للمستخدمين بالتسوق بسهولة دون أن يغمرهم الكثير من المعلومات.

إذا كان المستخدم مهتمًا بمنتج معين، يمكنه النقر على صورة المنتج لكشف المزيد من التفاصيل عنه. عرض كل المعلومات دفعة واحدة قد يكون مربكًا للمستخدم. لذا باستخدام الكشف التدريجي، يمكن للمستخدم اختيار رؤية المزيد من المعلومات عندما يكون جاهزًا.

```js
document.querySelectorAll('.details-btn').forEach(button => {
  button.addEventListener('click', () => {
    const product = button.closest('.product');
    const details = product.querySelector('.product-details');
    const isHidden = details.classList.toggle('hidden');
    product.setAttribute('aria-expanded', !isHidden);
    if (!isHidden) {
      details.focus();
      button.textContent = 'Hide details';
    } else {
      button.textContent = 'More details';
    }
  });
});
```
