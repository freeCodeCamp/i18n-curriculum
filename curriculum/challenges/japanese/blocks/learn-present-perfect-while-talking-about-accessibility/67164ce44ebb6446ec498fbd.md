# --explanation--

`Present Perfect Continuous` の疑問文は `Have/Has + subject + been + verb(-ing)` を使って作られます。これは過去に始まり、今も続いている動作について尋ねるために使われます。例えば：

`Have they been testing the new software features?` - 誰かがテストが続いているかどうかを確認しています。
