# --description--

Zahlen sind die Bausteine der Programmierlogik! Tatsächlich ist es schwer, sich eine nützliche Programmieraufgabe vorzustellen, die nicht zumindest ein wenig Grundrechenarten beinhaltet… daher ist es natürlich sehr wichtig, zu wissen, wie Zahlen funktionieren. Zum Glück ist das auch ziemlich einfach.

| Operator | Beschreibung                | Beispiel           |
|----------|----------------------------|--------------------|
| +        | Addition                   | 5 + 3 = 8          |
| -        | Subtraktion                | 5 - 3 = 2          |
| *        | Multiplikation             | 5 * 3 = 15         |
| **       | Exponentiation             | 5 ** 3 = 125       |
| /        | Division                   | 6 / 3 = 2          |
| %        | Modulus                    | 5 % 3 = 2          |
