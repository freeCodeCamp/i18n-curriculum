## CSS Flexboxとフレックスモデルの紹介

- **定義**: CSS flexboxは1次元のレイアウトモデルで、コンテナ内の要素を行や列に配置できます。
- **フレックスモデル**: フレックスコンテナ内でフレックスアイテムがどのように配置されるかを定義します。すべてのフレックスコンテナには主軸と交差軸の2つの軸があります。

## `flex-direction` プロパティ

- **定義**: このプロパティは主軸の方向を設定します。`flex-direction`のデフォルト値は`row`で、すべてのフレックスアイテムをブラウザのデフォルト言語方向（左から右または右から左）の同じ行に配置します。
- **`flex-direction: row-reverse;`**: 行内のアイテムの順序を逆にします。
- **`flex-direction: column;`**: フレックスアイテムを水平方向ではなく垂直方向に整列させます。
- **`flex-direction: column-reverse;`**: フレックスアイテムの垂直方向の順序を逆にします。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: row;
  gap: 10px;
}

.box {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}
```

:::

## `flex-wrap` プロパティ

- **定義**: このプロパティはフレックスコンテナ内でフレックスアイテムが利用可能なスペースに収まるように折り返すかどうかを決定します。`flex-wrap`は`nowrap`、`wrap`、`wrap-reverse`の3つの値を取れます。
- **`flex-wrap: nowrap;`**: デフォルト値です。フレックスアイテムはコンテナの幅を超えても折り返しません。
- **`flex-wrap: wrap;`**: アイテムがコンテナの幅を超えた場合に折り返します。
- **`flex-wrap: wrap-reverse;`**: アイテムを逆順で折り返します。
- **`flex-flow` プロパティ**: `flex-direction`と`flex-wrap`の省略形プロパティです。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
  <div class="box">4</div>
  <div class="box">5</div>
</div>
```

```css
.container {
  display: flex;
  flex-wrap: wrap;
  width: 150px;
  background: #f0f0f0;
}

.box {
  width: 60px;
  padding: 10px;
  margin: 5px;
  background: skyblue;
  text-align: center;
}
```

:::

## `justify-content` プロパティ

- **定義**: このプロパティはフレックスコンテナの主軸に沿って子要素を揃えます。
- **`justify-content: flex-start;`**: この場合、フレックスアイテムは主軸の開始位置に揃えられます。これは水平方向または垂直方向の場合があります。
- **`justify-content: flex-end;`**: この場合、フレックスアイテムは主軸の終了位置に揃えられます（水平方向または垂直方向）。
- **`justify-content: center;`**: これはフレックスアイテムを主軸に沿って中央に配置します。
- **`justify-content: space-between;`**: これは主軸に沿って要素を均等に分布させます。
- **`justify-content: space-around;`**: これは主軸内でフレックスアイテムを均等に分布させ、最初のアイテムの前と最後のアイテムの後にスペースを追加します。
- **`justify-content: space-evenly;`**: これは主軸に沿ってアイテムを均等に分布させます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box">A</div>
  <div class="box">B</div>
  <div class="box">C</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
  background: #eee;
}

.box {
  padding: 20px;
  background: salmon;
}
```

:::
  
## `align-items` プロパティ

- **定義**: このプロパティは交差軸に沿ってアイテムを分布させます。交差軸は主軸に垂直です。
- **`align-items: center;`**: アイテムを交差軸の中央に配置します。
- **`align-items: flex-start;`**: アイテムを交差軸の開始位置に整列させます。
- **`align-items: stretch;`**: フレックスアイテムを交差軸に沿って伸ばします。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box tall">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  align-items: center;
  height: 150px;
  background: #ddd;
}

.box {
  background: lightgreen;
  padding: 10px;
}

.tall {
  height: 100px;
}
```
