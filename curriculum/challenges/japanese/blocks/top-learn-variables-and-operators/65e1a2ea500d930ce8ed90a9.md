# --description--

学校の算数を超えたJavaScriptの演算子の機能を見ていきましょう。

通常、プラス演算子`+`は数値を加算します。

しかし、2項`+`が文字列に適用されると、それらをマージ（連結）します：

```js
let s = "my" + "string";
console.log(s); // mystring
```

どちらかのオペランドが文字列の場合、もう一方も文字列に変換されることに注意してください。

例えば：

```js
console.log(1 + '2'); // '12'
console.log('1' + 2); // '12'
```

最初のオペランドが文字列でも2番目のオペランドが文字列でも関係ありません。

もっと複雑な例を見てみましょう：

```js
console.log(2 + 2 + '1') // "41" and not "221"
```

ここでは、演算子が順番に動作します。最初の`+`は2つの数値を加算するので`4`を返し、次の`+`は文字列`1`をそれに加えるので、`4 + '1' = '41'`のようになります。

```js
console.log('1' + 2 + 2) // "122" and not "14"
```

ここでは、最初の`+`が文字列と数値を連結するので`'12'`を返し、次の`+`が`2`をそれに加えるので、`'12' + 2 = '122'`のようになります。

2項`+`は文字列をこのように扱う唯一の演算子です。他の算術演算子は数値でのみ動作し、常にオペランドを数値に変換します。

減算と除算のデモはこちらです：

```js
console.log( 6 - '2' ); // 4, converts '2' to a number
console.log( '6' / '2' ); // 3, converts both operands to numbers
```
