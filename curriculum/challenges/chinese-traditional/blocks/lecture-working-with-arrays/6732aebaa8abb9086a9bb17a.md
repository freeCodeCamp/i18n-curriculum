# --interactive--

在上一課中，首先為你介紹了如何操作陣列以及存取陣列中的不同元素。這裡提醒你如何存取陣列中的第二個元素：

:::interactive_editor

```js
const fruits = ["apple", "banana", "cherry"];

console.log(fruits[1]); // "banana"
```

:::

由於陣列是從零開始編號，第一個元素會在索引 `0`，第二個元素會在索引 `1`，依此類推。重要的是要注意，如果你嘗試存取陣列中不存在的索引，JavaScript 會傳回 `undefined`。 

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
console.log(fruits[3]); // undefined
```

:::

在這個範例中，`fruits` 陣列沒有索引 `3`。所以日誌會顯示 `undefined`。現在，讓我們來看看如何更新陣列中的元素。你可以透過為特定索引指定新值來更新元素。 

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
fruits[1] = "blueberry";
console.log(fruits); // ["apple", "blueberry", "cherry"]
```

:::

在這個範例中，我們已經在索引 `1` 將 `banana` 取代為 `blueberry`。這個方法允許你更改陣列中任何元素，只要你知道它的索引。你也可以透過為尚不存在的索引指定值，將新元素添加到陣列中：

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
fruits[3] = "date";
console.log(fruits); // ["apple", "blueberry", "cherry", "date"]
```

:::

但是，在這樣做時要小心。如果你為一個遠大於陣列當前長度的索引指定值，你將會為中間的索引創建未定義的元素，這可能導致意外的行為。當你繼續使用 JavaScript 時，你會發現這些存取和更新陣列元素的方法是許多程式設計任務的基礎。無論你是在建置一個簡單的待辦事項列表，還是在處理複雜的資料結構，這些技能都將非常寶貴。
