# --description--

バーバラは数学者でありバスケットボール選手です。彼女は、距離 $x$ からシュートしたときに得点する確率が正確に ($1 - \frac{x}{q}$) であることを見つけました。ここで $q$ は 50 より大きい実数定数です。

練習のたびに、彼女は距離 $x = 1, x = 2, \ldots, x = 50$ からシュートを行い、記録によると、合計でちょうど 20 点を取る確率は正確に 2 % です。

$q$ を求め、答えを小数点以下10桁で四捨五入してください。
