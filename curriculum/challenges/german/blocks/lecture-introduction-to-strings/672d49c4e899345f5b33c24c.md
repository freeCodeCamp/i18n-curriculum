# --interactive--

Die vorherigen Lektionen haben Ihnen `console.log()` vorgestellt, aber in dieser Lektion gehen wir tiefer auf dessen Zweck und Verwendung ein.

In JavaScript ist `console.log()` ein einfaches, aber leistungsstarkes Werkzeug, mit dem Nachrichten oder Informationen in der Konsole des Browsers angezeigt werden können. Es wird hauptsächlich von Entwicklern verwendet, um Code zu debuggen und zu überprüfen, während sie an ihren Programmen arbeiten.

Sie können `console.log()` verwenden, um Text oder Variablen in der Konsole zu protokollieren und sicherzustellen, dass Ihr Code korrekt ausgeführt wird.

Um `console.log()` zu verwenden, rufen Sie die Methode mit dem Wert oder der Nachricht, die Sie ausgeben möchten, innerhalb der runden Klammern auf. Hier sind einige Beispiele:

:::interactive_editor

```js
console.log("Hello, world!");

let num = 5;
console.log(num); // 5
```

:::

Das erste Beispiel gibt `Hello, world!` in der Konsole des Browsers aus, während das zweite Beispiel den Wert `5` ausgibt.

Hier ist ein weiteres Beispiel für die Arbeit mit `console.log()`:

:::interactive_editor

```js
let name = "Alice";
console.log("Hello, " + name + "!"); // Hello, Alice!
```

:::

Sie können auch mehrere Werte an `console.log()` übergeben, getrennt durch Kommas. Zum Beispiel:

:::interactive_editor

```js
let name = "Alice";
let age = 25;
console.log("Name:", name, "Age:", age); // Name: Alice Age: 25
```

:::

Das ist hilfreich, um mehrere Informationen auf einmal zu protokollieren.

Die `console.log()`-Methode hilft Ihnen, Ihren Code während der Ausführung zu überwachen, was es einfacher macht, Fehler zu erkennen und zu verstehen, wie Ihr Programm sich verhält.
