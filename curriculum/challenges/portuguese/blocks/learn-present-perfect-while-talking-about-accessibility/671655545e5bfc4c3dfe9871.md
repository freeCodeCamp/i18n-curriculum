Tratar de `address` significa tomar uma ação para resolver um problema ou responder a algo.Por exemplo:

`The developer is addressing the bug in the software.` - Isso mostra que o desenvolvedor está trabalhando ativamente para corrigir o problema.

`Survey` é uma ferramenta ou método para coletar informações ou opiniões dos usuários, tipicamente por meio de questionários ou entrevistas.
