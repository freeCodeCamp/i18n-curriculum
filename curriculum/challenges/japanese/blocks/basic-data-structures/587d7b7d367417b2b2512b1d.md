# --description--

オブジェクト内のすべてのキーを繰り返し処理する必要がある場合があります。これには <dfn>for...in</dfn> ループを使うことができます。for...in ループは次のようになります。

```javascript
const refrigerator = {
  'milk': 1,
  'eggs': 12,
};

for (const food in refrigerator) {
  console.log(food, refrigerator[food]);
}
```

このコードは `milk 1` と `eggs 12` をログに出力し、各キーと値のペアをそれぞれ別の行に表示します。

ループのヘッドで変数 `food` を定義しており、この変数は各反復でオブジェクトのキーのそれぞれに設定されるため、各食品の名前がコンソールに出力されます。

**注意:** オブジェクトは配列のようにキーの順序を保持しないため、オブジェクト内のキーの位置や表示される順序は、そのキーを参照またはアクセスする際には関係ありません。

```js
  },
  Jeff: {
    online: true
  },
  Sarah: {
    online: false
  }
}
```
