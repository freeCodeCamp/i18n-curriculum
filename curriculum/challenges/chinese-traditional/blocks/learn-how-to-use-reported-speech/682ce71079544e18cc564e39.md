# --explanation--

`To prevent` 是指採取行動以避免某事發生。例如：

`They installed new software to prevent viruses.`－這表示該軟體有助於在病毒造成問題之前阻止它們。

`incident` 是一個意外的事件，通常是負面或有問題的。 例如：

`There was a security incident last night that shut down the server。`－這表示發生了某些事情，導致系統出現問題。
