# --description--

Para un entero $M$, definimos $R(M)$ como la suma de $\frac{1}{p·q}$ para todos los pares de enteros $p$ y $q$ que satisfacen todas estas condiciones:

- $1 ≤ p &lt; q ≤ M$
- $p + q ≥ M$
- $p$ y $q$ son coprimos.

También definimos $S(N)$ como la suma de $R(i)$ para $2 ≤ i ≤ N$.

Podemos verificar que $S(2) = R(2) = \frac{1}{2}$, $S(10) ≈ 6.9147$ y $S(100) ≈ 58.2962$.

Encuentra $S({10}^7)$. Da tu respuesta redondeada a cuatro decimales.
