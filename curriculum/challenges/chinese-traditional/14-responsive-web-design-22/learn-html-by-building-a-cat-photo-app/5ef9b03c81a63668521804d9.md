# --description--

佔位符文本用於提示人們在輸入框中輸入什麼樣的信息。

這是一個 `input` 元素的示例，將佔位符設置爲 `Ex. Jane Doe`：

```html
<input type="text" placeholder="Ex. Jane Doe">
```

將佔位符文本 `cat photo URL` 添加到你的 `input` 元素中。
