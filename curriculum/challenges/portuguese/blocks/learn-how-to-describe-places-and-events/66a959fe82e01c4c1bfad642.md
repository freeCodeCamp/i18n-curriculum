# --explanation--

`Might` is used to express possibility.Significa que algo é possível mas não certo.Por exemplo:

- `I might go to the party if I finish my work.` - É possível que eu vá, mas não tenho certeza.

- `She might call you later.` - É possível que ela ligue, mas não é certo.
