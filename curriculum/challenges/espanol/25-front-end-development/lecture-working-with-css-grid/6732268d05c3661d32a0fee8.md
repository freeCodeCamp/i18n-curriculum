# --transcript--

¿Cómo puedes repetir listados de pistas en un diseño de cuadrícula?

En los videos de las clases anteriores, hemos estado trabajando con la propiedad `grid-template-columns` y estableciendo el valor en algunas unidades fraccionarias.

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}
```

Aunque el siguiente código es completamente válido, hay una forma más sencilla de repetir una sección o todos los listados de tus pistas.

La función `repeat()` se usa para repetir una sección o todos los elementos de las columnas o filas. Esta función toma una cuenta de repetición y los elementos que deseas repetir.

Aquí tienes una versión revisada del ejemplo anterior usando la función de repetición:

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  column-gap: 10px;
}
```

No habrá ningún cambio en los estilos en el navegador, pero esta es una manera más concisa de escribir valores repetidos para las columnas.

La función `repeat()` aceptará cualquier patrón válido que puedas usar para filas o columnas.

En este ejemplo, tenemos marcado HTML para un contenedor de cuadrícula:

```html
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

Luego estamos usando la función `repeat()` para establecer la primera y tercera columnas a 20 píxeles y la segunda y cuarta columnas a una unidad fraccionaria.

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(2, 20px 1fr);
  column-gap: 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

A veces, puedes optar por escribir cada valor individual en lugar de usar la función `repeat()`. Pero hay ocasiones en las que esta función resulta útil, especialmente cuando deseas repetir un patrón particular para un listado de pistas.
