<div style='text-align: center;'>123، 124، 125، 134، 135، 145، 234، 235، 245، و345</div>

في علم التوافيق، نستخدم التدوين، $\\displaystyle \\binom 5 3 = 10$

بشكل عام، $\\displaystyle \\binom n r = \\dfrac{n!}{r!(n-r)!}$، حيث $r \\le n$، و$n! = n \\times (n-1) \\times ... \\times 3 \\times 2 \\times 1$، و$0! = 1$.

لا يتجاوز أي قيمة المليون إلا عندما يكون $n = 23$: $\\displaystyle \\binom {23} {10} = 1144066$.

كم عدد القيم، التي ليست بالضرورة مميزة، لـ $\\displaystyle \\binom n r$ حيث $1 \\le n \\le 100$، التي تزيد عن مليون؟

```js
function combinatoricSelections(limit) {
    const factorial = n =>
        Array.apply(null, { length: n })
            .map((_, i) => i + 1)
            .reduce((p, c) => p * c, 1);

    let result = 0;
    const nMax = 100;

    for (let n = 1; n <= nMax; n++) {
        for (let r = 0; r <= n; r++) {
            if (factorial(n) / (factorial(r) * factorial(n - r)) >= limit)
                result++;
        }
    }

    return result;
}
```
