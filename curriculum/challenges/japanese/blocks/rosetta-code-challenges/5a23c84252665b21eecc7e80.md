# --description--

グレイコードは、連続する数値間の変化が1ビットだけ異なる2進数の符号化形式です。

これは、値が急速に変化したり、遅いハードウェアに入力として接続されたりする場合のハードウェアデータハザードを減らすのに役立つ符号化です。

また、カルノー図の入力を左から右、または上から下の順に生成するのにも便利です。

# --instructions--

数値をグレイコードにエンコードし、グレイコードからデコードする関数を作成してください。関数は2つのパラメータを持ちます。

1つ目はブール値です。trueの場合はエンコードし、falseの場合はデコードします。2つ目のパラメータはエンコード/デコードする数値です。

5ビットの2進数（0から31まで、先頭の0は不要）の通常の2進表現、グレイコード表現、およびデコードされたグレイコード値をすべて表示してください。

グレイコードには多くの種類がありますが、以下は「二進反射グレイコード」と呼ばれるものをエンコードします。

エンコード（MSBはビット0、bは2進数、gはグレイコード）：

<pre>if b[i-1] = 1
  g[i] = not b[i]
else
  g[i] = b[i]
</pre>

または：

<pre>g = b xor (b logically right shifted 1 time)
</pre>

デコード（MSBはビット0、bは2進数、gはグレイコード）：

<pre>b[0] = g[0]<br>
for other bits:
b[i] = g[i] xor b[i-1]
</pre>

function gray(enc, number){
