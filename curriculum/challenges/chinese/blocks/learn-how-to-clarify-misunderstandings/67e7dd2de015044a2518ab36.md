# --explanation--

`Think of it as` 用于通过将某个概念与他们已经熟悉的事物进行比较，帮助某人理解该概念。它鼓励你以一种更容易理解的方式想象该概念。例如：

`Think of a function as a recipe.` —— 这有助于你将函数想象成一组指令，类似于食谱提供制作菜肴的步骤。

`Wrapping code` 是指以一种控件访问的方式封装或打包代码。这将 `limit` 对代码的访问。例如：

`We're wrapping the database access code inside a function to hide the complexity from the rest of the program.` —— 这意味着数据库交互对外部是隐藏的，函数提供了一个简化的接口来使用。
