### --feedback--

Essas três palavras juntas são usadas para verificar se algo que acabou de ser dito leva a um resultado ou conclusão específica. A primeira letra da primeira palavra é maiúscula.

`launch`

### --feedback--

Esta palavra se refere ao lançamento oficial de um produto, como um app.

---

`delayed`

### --feedback--

Esta palavra significa que algo acontecerá mais tarde do que o planejado ou esperado. Termina com `-ed`.

# --explanation--

`Does that mean` é usado para confirmar ou esclarecer o resultado do que alguém acabou de dizer. Por exemplo:

`Does that mean we need to start over?` — Isso está verificando se essa é a conclusão correta.
