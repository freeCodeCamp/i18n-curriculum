# --explanation--

Mark 提到 `incompatible drivers` 和 `outdated software` 是软件崩溃的原因。

`不兼容的驱动程序` 是指与硬件不兼容的驱动程序，而 `过时的软件` 是指软件未更新到最新版本。两者都可能导致崩溃。例如：

- `The graphics card isn't working because of incompatible drivers.` - 错误的驱动程序导致显卡故障。

- `Outdated software can lead to security vulnerabilities.` － 旧程序由于缺乏更新可能不安全。
