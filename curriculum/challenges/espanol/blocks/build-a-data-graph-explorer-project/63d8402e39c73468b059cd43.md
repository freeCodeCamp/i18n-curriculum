# --instructions--

Para este desafío, necesitas crear un explorador de gráficos de datos usando Python que tome entrada y haga lo siguiente:

- Obtén un archivo .csv de tres maneras
  - subiéndolo desde la computadora local
  - obteniendo una url desde la entrada del usuario
  - poniendo la url en el código
- Usa la biblioteca Pandas para guardar el .csv como un dataframe
- Imprime los encabezados y las dos primeras filas
- Almacena los nombres de las columnas como una lista
- Elige una o dos columnas y convierte los datos en arrays de Numpy
- Muestra datos como un diagrama de dispersión o un gráfico de líneas
- Ser capaz de hacer esto para diferentes combinaciones de columnas e interpretar los gráficos
