# --description--

質問 `¿Quieres que te lo deletree?` では、いくつかの要素が組み合わさって申し出を表現しています。

`Quieres que` は、誰かが何かを欲しがっているかどうかを尋ねるために使われます。

`Te` は話しかけられている人を指します。この場合はカミラです。

`Lo` は情報を指し、ここではマテオのメールアドレスです。

`Deletree` は `deletrear` に由来し、単語を文字ごとに綴ることを意味します。

これらが組み合わさって、カミラにメールアドレスを綴ってあげるという申し出を表しています。
