# --description--

Конструктори `Bird` і `Dog` з останнього завдання працювали добре. Однак зверніть увагу, що всі `Birds`, створені за допомогою конструктора `Bird`, автоматично мають ім’я Albert, синій колір і дві ноги. Що, якщо ви хочете птахів із різними значеннями для name і color? Можна змінювати властивості кожного птаха вручну, але це було б дуже трудомістко:

```js
let swan = new Bird();
swan.name = "Carlos";
swan.color = "white";
```

Припустімо, ви пишете програму для обліку сотень або навіть тисяч різних птахів у вольєрі. Створити всіх птахів, а потім змінювати властивості на різні значення для кожного — це займе багато часу. Щоб легше створювати різні об’єкти `Bird`, ви можете спроектувати конструктор Bird так, щоб він приймав параметри:

```js
function Bird(name, color) {
  this.name = name;
  this.color = color;
  this.numLegs = 2;
}
```

Потім передайте значення як аргументи, щоб визначити кожного унікального птаха в конструкторі `Bird`: `let cardinal = new Bird("Bruce", "red");` Це створює новий екземпляр `Bird` із властивостями `name` і `color`, встановленими відповідно на `Bruce` і `red`. Властивість `numLegs` все ще встановлена на 2. `cardinal` має такі властивості:

```js
cardinal.name
cardinal.color
cardinal.numLegs
```

Конструктор став гнучкішим. Тепер можливо визначати властивості для кожного `Bird` під час його створення, що є одним із способів, якими конструктори JavaScript дуже корисні. Вони групують об’єкти за спільними характеристиками та поведінкою і визначають шаблон, який автоматизує їх створення.

# --instructions--

Створіть ще один конструктор `Dog`. Цього разу налаштуйте його так, щоб він приймав параметри `name` і `color`, а властивість `numLegs` була фіксованою зі значенням 4. Потім створіть новий `Dog`, збережений у змінній `terrier`. Передайте йому два рядки як аргументи для властивостей `name` і `color`.

# --hints--

`Dog` має приймати аргумент для `name`.

```js
assert(new Dog('Clifford').name === 'Clifford');
```

```js
assert(new Dog('Clifford', 'yellow').color === 'yellow');
```

`Dog` має мати властивість `numLegs`, встановлену на 4.

```js
assert(new Dog('Clifford').numLegs === 4);
```

```js
assert(terrier instanceof Dog);
```
