# --interactive--

外部フォントとは、プロジェクトのファイルに直接含まれていないフォントファイルのことです。通常は別のサーバーでホストされています。サーバーとは、ネットワークを通じて他のコンピューターにデータやサービスを提供するコンピューターのことです。サーバーについては今後のレッスンで詳しく学びます。外部フォントを使うと、ユーザーのデバイスにインストールされていないカスタムフォントも利用できるため、より柔軟にフォントを扱えます。

Google FontsとFont Squirrelは、ウェブ開発で使える無料フォントを見つけて利用できる人気のオンラインリソースです。これらはプロジェクトに簡単に組み込めます。

まずはGoogle Fontsから始めましょう。これはGoogleが提供するサービスで、ウェブ開発向けに設計されたフォントを多く含むコレクションを提供しています。Google Fontsのユーザーインターフェースにはさまざまな要素があります。ひとつずつ見ていきましょう。左のサイドバーには以下があります：

- フォントを見たり、探したり、フィルターをかけたりできるセクション
- Notoに関する特別なセクション。これは1000以上の言語と150以上の文字体系でのコミュニケーションに最適な、さまざまなウェイト、幅、スタイルを持つ高品質フォントのコレクションです
- ウェブプロジェクト用のアイコンを見つけてダウンロードできるセクション
- フォントやベストプラクティスについて学べるセクションとよくある質問

Google Fontsをウェブサイトに追加するには、最初のセクション（Fonts）に行きます。左のサイドバーの「Preview」と書かれたところでプレビュー用のテキストをカスタマイズできます。見たいテキストを入力すると自動で更新されます。これはウェブサイトで使いたいテキストでフォントをプレビューするのに便利です。フォントサイズの調整やフォントの特徴によるフィルターもできます。

フォントの詳細を見たい場合は、そのフォントをクリックしてください。デザイナー情報、プレビュー用テキスト、ライト、レギュラー、ミディアム、ボールド、ブラックなどの異なるスタイルが表示されます。上部のオプションをクリックすると、タイプテスターに行ったり、個別のグリフを見たり、フォントやライセンスについて詳しく学べます。

プレビュー用テキストのフォントサイズは、テキスト入力欄の隣にある青いスライダーで調整できます。フォントをプロジェクトに追加する準備ができたら、上部の青い「Get font」ボタンをクリックしてください。現在選択しているフォントの一覧が表示されるサマリーページに移動します。複数のフォントを同時に選択できます。

次に、フォントファイルをダウンロードしてローカルファイルとしてプロジェクトに追加するか、ユーザーがウェブサイトにアクセスしたときにGoogleのサーバーから外部フォントとして読み込むかを選びます。ダウンロードしたい場合は「Download all」をクリックし、Googleのサーバーから外部フォントとして使いたい場合は「Get embed code」をクリックしてください。「Get embed code」をクリックすると、外部フォントをプロジェクトに追加するための手順が表示されます。

ウェブ開発プロジェクトでは、`link`要素か`@import`のどちらかを使う選択肢があります。`link`要素を使う場合は、HTMLのスニペットとCSSルールをコピーしてプロジェクトに貼り付けます。HTMLファイルの`head`要素にコードを埋め込み、必要に応じてCSSルールを追加してください。

各フォントスタイルに対応するCSSルールがあります。各ルールは、カスタムフォントが正しく読み込まれなかった場合に備えてフォールバックフォントを代入しています。

`@import`オプションを選ぶ場合は、そのルールをCSSファイルに追加する必要があります。例は以下の通りです：

```css
@import url('https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap');
```

CSSルールはまったく同じです。

特定のフォントスタイルだけを含めたい場合は、これらのスニペットから不要なスタイルを削除することもできます。HTMLとCSSのスニペットは自動で更新されます。

これがGoogle Fontsの使い方です。ウェブ開発プロジェクトで無料かつオープンソースのフォントを見つける最も人気のある選択肢のひとつです。

もうひとつの優れた選択肢はFont Squirrelです。ここではデザインに合ったカスタムフォントを見つけてダウンロードできます。フォントを検索して結果をクリックすると、フォントの詳細が表示されます。サンプル、テストドライブ、グリフ、Webfontキットなどが見られます。細字、ライト、ミディアム、ボールド、ブラックなどの異なるスタイルやバリエーションのサンプルも確認できます。

使いたいフォントが決まったら、「Webfont Kit」タブに移動してください。ここで、そのフォントのライセンスが`@font-face`のCSS埋め込みでの使用を許可しているか確認できます。部分集合やフォーマットも選べます。

「Download @font-face Kit」ボタンをクリックすると圧縮（zip）ファイルがダウンロードされます。解凍すると、「web fonts」フォルダー、テキストファイルのライセンス、ウェブフォントの使い方を説明したHTMLファイルが含まれています。web fontsフォルダーにはプロジェクトに必要なすべてのフォントファイルが入っており、異なるフォントスタイルごとにフォルダー分けされています。

HTMLファイルをブラウザーで開くと、フォントをプロジェクトに追加する方法、`@font-face`宣言を使ってスタイルシートに含める方法、CSSルールで使う方法が説明されています。

これがFont Squirrelのカスタムフォントをプロジェクトに追加する方法です。別のサーバーでホストして外部フォントとして使うこともできます。外部フォントはウェブサイトの全体的な見た目やデザインをカスタマイズするのに非常に役立ちます。ウェブサイトを本当にユニークにできます。

ただし、複数の外部フォントを使うとウェブサイトの読み込み時間が長くなり、全体のパフォーマンスに影響を与えることも覚えておいてください。スタイルと効率のバランスを常に考えることが大切です。
