# --description--

無限に続く箱の列を考えます。いくつかの箱にはボールが入っています。例えば、2つ連続してボールが入った箱があり、その後に2つ空の箱、さらに2つボールが入った箱、1つ空の箱、そして2つボールが入った箱が続く初期配置は、連続したボール入りと空の箱の数が交互に現れるシーケンス (2, 2, 2, 1, 2) と表せます。

1ターンは、以下のルールに従って各ボールをちょうど1回移動させることから成ります：まだ移動していない左端のボールを、その右側で最も近い空の箱に移動させます。

1ターン後、シーケンス (2, 2, 2, 1, 2) は (2, 2, 1, 2, 3) になります。下の図のように、新しいシーケンスは最初のボールが入った箱から始まることに注意してください。

<img alt="animation showing one complete turn from (2, 2, 2, 1, 2) to (2, 2, 1, 2, 3)" src="https://cdn.freecodecamp.org/curriculum/project-euler/box-ball-system-1.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

このようなシステムはボックスボールシステム、略してBBSと呼ばれます。

十分なターン数の後、このシステムは連続したボール入り箱の数が変わらなくなる状態に進化することが示されています。下の例では、連続したボール入り箱の数が [1, 2, 3] に進化します。これを最終状態と呼びます。

<img alt="four turns from occupied boxes [2, 2, 2] to final state [1, 2, 3]" src="https://cdn.freecodecamp.org/curriculum/project-euler/box-ball-system-2.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

シーケンス $\\{t_i\\}$ を次のように定義します：

$$\begin{align}
  & s_0 = 290\\,797 \\\\
  & s_{k + 1} = {s_k}^2\bmod 50\\,515\\,093 \\\\
  & t_k = (s_k\bmod 64) + 1
\end{align}$$

初期配置を $(t_0, t_1, \ldots, t_{10})$ とすると、最終状態は [1, 3, 10, 24, 51, 75] になります。

初期配置を $(t_0, t_1, \ldots, t_{10\\,000\\,000})$ としたときの最終状態を求めてください。

答えとして、最終状態の要素の二乗の和を出してください。例えば、最終状態が [1, 2, 3] なら $14 (= 1^2 + 2^2 + 3^2)$ が答えです。
