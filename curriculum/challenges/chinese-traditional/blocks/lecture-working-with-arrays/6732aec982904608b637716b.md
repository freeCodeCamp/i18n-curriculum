# --interactive--

JavaScript 中的陣列是動態的，這表示你可以輕鬆地為它們添加或移除元素。為你的陣列開頭和結尾添加及移除元素有四個主要的行為：`push()`、`pop()`、`shift()` 和 `unshift()`。讓我們詳細探討這些行為。

`push()` 方法用於將一個或多個元素添加到陣列的末端。`push()` 方法的返回值是陣列的新長度。以下是將一種新水果添加到現有 `fruits` 陣列的範例：

:::interactive_editor

```js
const fruits = ["apple", "banana"];
const newLength = fruits.push("orange");
console.log(newLength); // 3
console.log(fruits); // ["apple", "banana", "orange"]
```

:::

在這個範例中，我們從一個名為 `fruits` 的陣列開始，該陣列包含兩個元素。接著，我們使用 `push()` 方法將字串 `orange` 添加到陣列的末端。

你可能已經注意到，我們在宣告 `fruits` 陣列時使用了 `const`。但是，為什麼當 `fruits` 是常數時，仍然可以為這個 `fruits` 陣列添加更多元素？這是因為使用 `const` 關鍵字宣告陣列會創建對該陣列的引用。雖然陣列本身是可變的並且可以被修改，但你不能像這樣重新指定一個新值給 `fruits` 常數：

```js
const fruits = ["apple", "banana"];
fruits = ["This", "will", "not", "work"];
console.log(fruits); // Uncaught TypeError: Assignment to constant variable. 
```

下一個我們將要看的方法是 `pop()` 方法。`pop()` 方法會從陣列中移除最後一個元素並傳回該元素。它也會修改原始陣列。以下是它的運作方式：

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let lastFruit = fruits.pop();
console.log(fruits); // ["apple", "banana"]
console.log(lastFruit); // "orange"
```

:::

在這個範例中，我們從一個包含三種水果的陣列開始。`pop()` 方法會移除陣列中的最後一個元素（`orange`）並傳回它。原本的 `fruits` 陣列會被修改，現在只包含兩個元素。

`unshift()` 方法會將一個或多個元素添加到陣列的開頭，並傳回其新的長度。它的運作方式類似於 `push()`，但會修改陣列的開頭而非結尾。以下是一個範例：

:::interactive_editor

```js
let numbers = [2, 3];
let newLength = numbers.unshift(1);
console.log(numbers); // [1, 2, 3]
console.log(newLength); // 3
```

:::

在此範例中，我們使用 `unshift()` 為 `numbers` 陣列的開頭添加數字 `1`。該方法傳回陣列的新長度，為 `3`。

最後，`shift()` 方法會從陣列中移除第一個元素並傳回該元素。它類似於 `pop()`，但作用於陣列的開頭而非結尾。以下是它的運作方式：

:::interactive_editor

```js
let colors = ["red", "green", "blue"];
let firstColor = colors.shift();
console.log(colors); // ["green", "blue"]
console.log(firstColor); // "red"
```

:::

在這個範例中，我們從一個包含三個顏色的陣列開始。`shift()` 方法會移除陣列中的第一個元素（`red`）並傳回它。原本的 `colors` 陣列會被修改為只包含兩個元素。

請注意，雖然 `push()` 和 `unshift()` 可以一次添加多個元素，`pop()` 和 `shift()` 則一次只移除一個元素。
