# --interactive--

페이지를 디자인할 때 너비, 높이, 패딩, 여백 등 다양한 프로퍼티를 다루게 됩니다. 이 프로퍼티들을 정의할 때 사용할 길이 단위를 지정해야 합니다.

이 프로퍼티들을 정의할 때 사용할 수 있는 단위는 상대 단위와 절대 단위 두 가지가 있습니다. 이번 수업에서는 절대 단위에만 집중하겠습니다.

절대 길이 단위는 고정된 길이로, 다른 것에 상대적이지 않습니다. 상대적이라는 것은 길이가 화면 크기나 부모 요소 크기 같은 다른 것에 따라 달라진다는 뜻입니다. 상대 단위에 대해서는 다음 수업들에서 더 자세히 다루겠습니다.

가장 흔한 절대 단위는 픽셀(`px`)입니다. 픽셀은 CSS에서 고정 크기 단위로, 치수를 정확하게 제어할 수 있습니다. 즉, 1px는 항상 1/96인치와 같습니다.

CSS 레이아웃 목적상 1px가 1/96인치로 표준화되어 있지만, 실제 픽셀의 물리적 크기는 디스플레이에 따라 다를 수 있다는 점을 기억하세요.

픽셀을 사용해 박스의 너비와 높이를 설정하는 예제를 살펴보겠습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="box"></div>
```

```css
.box {
  width: 100px;
  height: 100px;
  background-color: red;
}
```

:::

이 예제에서는 너비와 높이가 `100px`로 설정된 빨간 박스가 있습니다. 이 박스의 크기는 화면 크기가 바뀌어도 변하지 않습니다.

즉, 작은 화면 기기에서도 박스 너비는 여전히 `100px`입니다. 큰 데스크톱 화면에서도 박스 너비는 여전히 `100px`입니다.

그렇다면 언제 픽셀 같은 절대 단위를 사용해야 할까요? 상황에 따라 다르지만, 일반적으로 요소 크기, 간격, 레이아웃을 정확하게 제어해야 할 때 픽셀을 사용합니다. 때로는 여백, 패딩, 테두리를 픽셀로 설정하기도 합니다.

두 개의 박스에 대해 픽셀로 여백을 설정하는 예제도 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="box"></div>
<div class="box"></div>
```

```css
.box {
  width: 100px;
  height: 100px;
  background-color: red;
  margin: 10px;
}
```

:::

여백은 박스 바깥의 공간이라는 점을 기억하세요. 이 예제에서는 박스가 모든 면에 `10px`의 여백을 갖습니다.

다른 절대 단위 종류는 다음과 같습니다:

- `in` (인치) 단위, 96px와 같습니다
- `cm` (센티미터) 단위, 25.2/64인치와 같습니다
- `mm` (밀리미터) 단위, 1센티미터의 1/10에 해당합니다
- `q` (쿼터 밀리미터) 단위, 1센티미터의 1/40에 해당합니다
- `pc` (피카) 단위, 1/6인치와 같습니다
- `pt` (포인트) 단위, 1/72인치와 같습니다

이 단위들은 대부분 인쇄용으로 사용되며 화면용은 아닙니다.

가장 흔히 사용하는 단위는 픽셀이지만, 다른 절대 단위가 존재한다는 점도 알아두세요.
