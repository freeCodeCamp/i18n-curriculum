## CSS 플렉스박스와 플렉스 모델 소개

- **정의**: CSS 플렉스박스는 컨테이너 안에서 요소들을 행과 열로 배치할 수 있는 1차원 레이아웃 모델입니다. 
- **플렉스 모델**: 이 모델은 플렉스 컨테이너 안에서 플렉스 항목들이 어떻게 배치되는지 정의합니다. 모든 플렉스 컨테이너는 두 개의 축을 가집니다: 주축과 교차 축.

## `flex-direction` 속성

- **정의**: 이 속성은 주축의 방향을 설정합니다. `flex-direction`의 기본값은 `row`로, 모든 플렉스 항목을 브라우저 기본 언어 방향(왼쪽에서 오른쪽 또는 오른쪽에서 왼쪽)으로 같은 행에 배치합니다. 
- **`flex-direction: row-reverse;`**: 이 값은 행 안의 항목 순서를 반대로 뒤집습니다.
- **`flex-direction: column;`**: 이 값은 플렉스 항목을 수평이 아닌 수직으로 정렬합니다.
- **`flex-direction: column-reverse;`**: 이 값은 플렉스 항목의 수직 순서를 반대로 뒤집습니다.

:::interactive_editor

```html
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: row;
  gap: 10px;
}

.box {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}
```

:::

## `flex-wrap` 속성

- **정의**: 이 속성은 플렉스 항목들이 플렉스 컨테이너 안에서 가용 공간에 맞게 어떻게 감싸지는지 결정합니다. `flex-wrap`은 세 가지 값 중 하나를 가질 수 있습니다: `nowrap`, `wrap`, `wrap-reverse`.
- **`flex-wrap: nowrap;`**: 기본값입니다. 플렉스 항목이 컨테이너 너비를 초과해도 새 줄로 감싸지 않습니다.
- **`flex-wrap: wrap;`**: 이 속성은 항목이 컨테이너 너비를 초과할 때 줄 바꿈을 합니다.
- **`flex-wrap: wrap-reverse;`**: 이 속성은 플렉스 항목을 역순으로 감쌉니다.
- **`flex-flow` 속성**: 이 속성은 `flex-direction`와 `flex-wrap`의 단축 속성입니다.

:::interactive_editor

```html
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
  <div class="box">4</div>
  <div class="box">5</div>
</div>
```

```css
.container {
  display: flex;
  flex-wrap: wrap;
  width: 150px;
  background: #f0f0f0;
}

.box {
  width: 60px;
  padding: 10px;
  margin: 5px;
  background: skyblue;
  text-align: center;
}
```

:::

## `justify-content` 속성

- **정의**: 이 속성은 플렉스 컨테이너의 주축을 따라 자식 요소들을 정렬합니다.
- **`justify-content: flex-start;`**: 이 경우 플렉스 항목은 주축의 시작 부분에 정렬됩니다. 주축은 수평 또는 수직일 수 있습니다.
- **`justify-content: flex-end;`**: 이 경우 플렉스 항목은 주축의 끝 부분에 정렬됩니다, 수평 또는 수직.
- **`justify-content: center;`**: 이 값은 플렉스 항목을 주축을 따라 가운데 정렬합니다.
- **`justify-content: space-between;`**: 이 값은 주축을 따라 요소들을 고르게 분배합니다.
- **`justify-content: space-around;`**: 이 값은 첫 번째 항목 앞과 마지막 항목 뒤에 공간을 추가하여 주축 내에서 플렉스 항목을 고르게 분배합니다.
- **`justify-content: space-evenly;`**: 이 값은 주축을 따라 항목을 고르게 분배합니다.

:::interactive_editor

```html
<div class="container">
  <div class="box">A</div>
  <div class="box">B</div>
  <div class="box">C</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
  background: #eee;
}

.box {
  padding: 20px;
  background: salmon;
}
```

:::
  
## `align-items` 속성

- **정의**: 이 속성은 교차 축을 따라 항목을 분배하는 데 사용됩니다. 교차 축은 주축에 수직임을 기억하세요.
- **`align-items: center;`**: 이 값은 교차 축을 따라 항목을 가운데 정렬하는 데 사용됩니다.
- **`align-items: flex-start;`**: 이 값은 교차 축의 시작 부분에 항목을 정렬합니다.
- **`align-items: stretch;`**: 이 값은 교차 축을 따라 플렉스 항목을 늘립니다.

:::interactive_editor

```html
<div class="container">
  <div class="box tall">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  align-items: center;
  height: 150px;
  background: #ddd;
}

.box {
  background: lightgreen;
  padding: 10px;
}

.tall {
  height: 100px;
}
```
