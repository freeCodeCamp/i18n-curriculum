# --description--

У попередньому уроці ви дізналися про загальні типи даних у Python, зокрема рядок, ціле число та число з плаваючою комою. Під час створення програм вам потрібно буде навчитися визначати тип змінної.

Ось приклад змінної:

```py
developer = 'Devin'
```

Щоб побачити, який тип має `developer`, ви можете використати функцію `type()` так:

```py
developer = 'Devin'

print(type(developer)) # <class 'str'>
```

Вихідні дані `<class 'str'>` означають, що `developer` є рядком.

Якщо ви не передасте жодних аргументів у функцію `type()`, ви отримаєте таке повідомлення про помилку:

```md
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: type() takes 1 or 3 arguments
```

Ось усі типи даних, які ви вивчили досі, разом із їхніми типами у терміналі:

```python
my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 'hello', 8.5}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 'hello', 8.5)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>
```

Іноді у вашій програмі потрібно буде перевірити, чи є певна змінна конкретного типу, перш ніж виконувати над нею операції. Саме тут стане в пригоді функція `isinstance()`.

Ось приклад змінної, якій призначено рядок:

```py
account_balance = '12'
```

Якщо ви спробуєте виконати математичні вирази, наприклад ділення, використовуючи змінну `account_balance`, ви отримаєте повідомлення про помилку.

```py
account_balance = '12'

account_balance / 2

# Traceback (most recent call last):
#   File "<stdin>", line 1, in <module>
# TypeError: unsupported operand type(s) for /: 'str' and 'int'
```

Щоб перевірити, чи є `account_balance` цілим числом, можна скористатися функцією `isinstance()` так:

```py
account_balance = '12'

isinstance(account_balance, int) # False
```

Вбудована функція `isinstance()` дозволяє перевірити, чи відповідає змінна певному типу даних. Вона приймає об’єкт і тип, з яким потрібно порівняти, а потім повертає булеве значення. У цьому випадку, оскільки `account_balance` є рядком, вона поверне `False`.

Функція `isinstance()` також дозволяє перевіряти кілька типів одночасно.

Ось приклад перевірки, чи є `account_balance` `int` або `float`:

```py
account_balance = 12
isinstance(account_balance, (int, float)) # True
```

У цьому прикладі `account_balance` — ціле число, тому `isinstance()` повертає `True`. Якби `account_balance` було `12.0`, `isinstance()` все одно повернуло б `True`, бо ви перевіряєте на цілі числа або числа з плаваючою комою.

У майбутніх практичних заняттях і лабораторних роботах ви будете використовувати функції `type()` і `isinstance()`, щоб переконатися, що ваші змінні містять правильні типи даних перед виконанням операцій.
