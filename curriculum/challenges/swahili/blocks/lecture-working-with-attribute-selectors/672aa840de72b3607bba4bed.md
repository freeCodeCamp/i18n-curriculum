# --interactive--

Kichaguzi cha sifa katika CSS ni chombo chenye nguvu kinachokuwezesha kulenga vipengele vya HTML kulingana na sifa zao. Hii inamaanisha unaweza kutumia mitindo kwa vipengele vyenye sifa maalum au hata thamani fulani za sifa hizo.

Ni muhimu hasa unapotaka kupamba vipengele kwa njia ya mabadiliko au wakati majina ya darasa la vitu pekee hayatoi ubainifu wa kutosha.

Kwa mfano, unaweza kutumia kichaguzi cha sifa kulenga viungo vyote vyenye sifa ya `href`. Hii ni rahisi sana unapotaka kutumia mtindo mmoja kwa viungo vyote kwenye ukurasa.

Hapa ni jinsi unavyoweza kutumia kichaguzi cha sifa kulenga viungo vyenye sifa ya `href`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Example link with an href attribute</a>
<a>Example link without an href attribute</a>
```

```css
a {
  display: block;
}

a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

Sheria hii ya CSS itatumia rangi ya buluu na mstari wa chini kwa kiungo chochote chenye sifa ya `href`, kuhakikisha viungo vyote vinavyoweza kubofywa vimepambwa kwa mtindo sawa.

Lakini wachaguzi wa sifa wanaweza kuwa wa kina zaidi. Fikiria unataka kulenga viungo vyenye sifa ya `title` pekee. Sifa ya `title` mara nyingi hutoa taarifa za ziada kuhusu kiungo, na unaweza kutaka kupamba viungo hivi tofauti ili kuonyesha kuwa vina taarifa za ziada zinazopatikana. Hapa ni jinsi unavyoweza kufanya hivyo:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

Sheria hii inatumia maandishi yenye herufi nzito na kuondoa mstari wa chini kwa kiungo chochote chenye sifa ya `title`. Ni njia nzuri ya kutofautisha viungo hivi kwa macho kutoka kwa vingine kwenye ukurasa.

Pia unaweza kuunganisha wachaguzi wa sifa kupata usahihi zaidi. Kwa mfano, ikiwa unataka kupamba viungo vyenye sifa za `href` na `title` zote mbili, unaweza kufanya hivi:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[href][title] {
  display:block;
  color: green;
}
```

:::

Katika kesi hii, viungo pekee vyenye sifa za `href` na `title` zote mbili ndivyo vitapambwa kwa maandishi ya kijani. Kiwango hiki cha udhibiti ndicho kinachofanya wachaguzi wa sifa kuwa na nguvu sana katika CSS.

Mfano mwingine wa kichaguzi cha sifa ni kulinganisha thamani moja ndani ya orodha ya thamani zilizoachwa kwa nafasi katika sifa.

Hapa ni mfano wa kipengele cha nanga chenye madarasa mengi:

```html
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

Ili kulenga kipengele hiki maalum cha nanga, unaweza kutumia kichaguzi kifuatacho:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

Sintaksia ya `[attr~=value]` inatumiwa hapa kulenga vipengele vyote vya nanga ambavyo sifa ya darasa la vitu ina neno `"primary"`.

Ikiwa unahitaji kulenga kipengele ambacho thamani ya sifa ina kiambishi awali cha thamani fulani, basi unaweza kutumia sintaksia ya `[attr^=value]`. Hapa ni mfano:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

Katika mfano huu, kichaguzi cha `a[href^="https://"]` kitalenga vipengele vyote vya nanga ambavyo thamani ya sifa ya `href` inaanza na `"https://"`.

Ili kulenga vipengele ambavyo thamani ya sifa inamalizika na thamani fulani, unaweza kutumia sintaksia ya `[attr$=value]`. Hapa ni mfano:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Visit Example Site</a>
```

```css
a[href$=".com"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

Katika mfano huu, kichaguzi cha `a[href$=".com"]` kitalenga vipengele vyote vya nanga ambavyo thamani ya sifa ya `href` inamalizika na `.com`.

Kutumia wachaguzi wa sifa si tu kunaboresha upambaji wa ukurasa wako wa mtandao bali pia kunaboresha Ufikikaji kwa kufanya vipengele vinavyoweza kuingiliana kama viungo viweze kutofautishwa kulingana na sifa zao.
