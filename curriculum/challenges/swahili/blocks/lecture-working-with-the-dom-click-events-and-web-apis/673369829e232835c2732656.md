# --interactive--

Dialogi hukuruhusu kuonyesha taarifa muhimu au vitendo kwa watumizi. Kwa kutumia kipengele cha `dialog` kilichojengwa ndani ya HTML, unaweza kwa urahisi kuunda dialogi hizi (za modal na zisizo za modal) katika programu zako za mtandao.

Dialogi ya modal ni aina ya dialogi inayomlazimisha mtumizi kuingiliana nayo kabla ya kupata upatikanaji wa sehemu nyingine za programu au ukurasa wa mtandao. Inazuia kabisa mwingiliano na maudhui mengine hadi mtumizi atakapokamilisha kitendo, kama kufunga dialogi au kuwasilisha fomu.

Kinyume chake, dialogi isiyo ya modal inaruhusu mtumizi kuendelea kuingiliana na sehemu nyingine za ukurasa au programu hata wakati dialogi iko wazi. Haiizuizi upatikanaji wa maudhui mengine.

Katika funzo hili, utajifunza jinsi ya kufungua na kufunga modal kwa kutumia njia za `showModal()` na `close()`.

Unapotaka kuhakikisha mtumizi anazingatia kitendo au ujumbe maalum wa modal, unaweza kufungua dialogi ya modal kwa kutumia njia ya `showModal()`. Hii itaongeza nyuma ya rangi kwenye vitu vingine vya ukurasa na kuvitenganisha. Hii ni bora kwa modal zinazonyesha fomu, uthibitisho, na taarifa muhimu zinazohitaji kitendo cha mtumizi.

Hapa kuna HTML ya dialogi ya modal:

:::interactive_editor

```html
<dialog id="my-modal">
  <p>This is a modal dialog.</p>
</dialog>
```

:::

Kwa sasa, huwezi kuona chochote kwenye ukurasa kwa sababu modal imefungwa wakati wa onyesho la awali. Unaweza kufungua modal moja kwa moja kwa kutumia njia ya `showModal()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
dialog.showModal();
```

:::

Matokeo kwenye kivinjari yataonyesha modal yenye maandishi ya `This is a modal dialog.`

Ni bora kumpa mtumizi udhibiti. Ili kufanikisha hili, unaweza kuongeza msikilizaji wa tukio la bofya kwenye kitufe na kutumia njia ya `showModal()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.showModal();
});
```

:::

Ikiwa unahitaji kuonyesha dialogi huku ukiruhusu mwingiliano na maudhui nje ya dialogi, basi unaweza kutumia njia ya `show()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});
```

:::

Ili kufunga modal, unaweza kuongeza kitufe ndani ya kipengele cha `dialog` cha modal na kutumia njia ya `close()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
  <button id="close-modal-btn">Close Modal</button>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");
const closeButton = document.getElementById("close-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});

closeButton.addEventListener("click", () => {
  dialog.close();
});
```
