# --description--


JavaScript ist die Programmiersprache, die das Web antreibt. Im Gegensatz zu HTML und CSS, die Sie zuvor gelernt haben, wird JavaScript am häufigsten verwendet, um Logik zu schreiben, nicht Auszeichnung.

In diesem Projekt lernen Sie die Grundlagen von JavaScript und wenden diese Konzepte an, um einen Pyramiden-Generator zu erstellen.

Ein Pyramiden-Generator ist ein Programm, bei dem Sie den Zeichentyp, die Anzahl für die Pyramide und die Richtung der Pyramide festlegen können. Das Programm erzeugt dann basierend auf diesen Eingaben eine Pyramide.

Klicken Sie auf die Schaltfläche „Check your code“, um zum nächsten Schritt zu gelangen und das Projekt zu starten.
