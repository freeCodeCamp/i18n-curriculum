# --interactive--

Katika mafundisho yaliyopita, ulijifunza jinsi ya kufanya kazi na matukio kwa kutumia njia ya `addEventListener()`. Lakini kuna njia nyingine, isiyopendekezwa, ya kufanya kazi na matukio katika JavaScript.

Mshughulikiaji wa tukio ndani ya mstari ni sifa maalum ya kipengele cha HTML zinazotumika kutekeleza msimbo wa JavaScript wakati tukio linapotokea.

Hapa kuna mfano wa kipengele cha `button` chenye mshughulikiaji wa tukio la bofya ndani ya mstari:

:::interactive_editor

```html
<button onclick="alert('Hello World!')">Show alert</button>
```

:::

Wakati mtumizi anapobofya kitufe, kitendakazi cha `alert` kinaitwa na dirisha la tahadhari linaonyeshwa lenye ujumbe wa `Hello World!`.

Njia nyingine ya kutumia mishughulikiaji wa tukio ndani ya mstari ni kuita kitendakazi kilichofafanuliwa katika lebo ya `script` katika hati ya HTML.

Hapa kuna mfano wa kufafanua kitendakazi kinachoitwa `changeBgColor` na kukiita kutoka kwa mshughulikiaji wa tukio la bofya ndani ya mstari:

:::interactive_editor

```html
<script>
  function changeBgColor() {
    document.body.style.backgroundColor = "lightblue";
  }
</script>

<button onclick="changeBgColor()">Change background color</button>
```

:::

Wakati mtumizi anapobofya kitufe, kitendakazi cha `changeBgColor` kinaitwa na rangi ya nyuma ya ukurasa hubadilishwa kuwa buluu nyepesi. Ingawa inawezekana kutumia mishughulikiaji wa tukio ndani ya mstari kama hii, haizingatiwi kuwa mbinu bora.

Kwa sababu moja, mishughulikiaji wa tukio ndani ya mstari inaweza kutumika tu kuambatanisha mshughulikiaji mmoja wa tukio kwa kipengele. Ikiwa unataka kuambatanisha mishughulikiaji mingi ya tukio kwa kipengele kimoja, utahitaji kutumia `addEventListener()`. Sababu nyingine ni kwamba mishughulikiaji wa tukio ndani ya mstari huchanganya msimbo wa HTML na JavaScript pamoja, jambo ambalo linaweza kufanya msimbo wako kuwa mgumu kusoma na kudumisha. Ni bora kuweka msimbo wako wa HTML na msimbo wa JavaScript tofauti kwa kutumia `addEventListener()` kuambatanisha mishughulikiaji ya matukio kwa vipengele.

Mishughulikiaji wa tukio ndani ya mstari haipendekezwi kwa matumizi katika JavaScript ya kisasa. Kwa hivyo, ni mbinu bora kuendelea kutumia njia ya `addEventListener()` wakati wa kufanya kazi na matukio katika JavaScript.
