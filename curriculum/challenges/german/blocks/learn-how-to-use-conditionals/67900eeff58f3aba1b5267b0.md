# --explanation--

Lassen Sie uns das aufschlüsseln:

Was hat Mark gesagt? Er sagte, das BIOS `prevents the computer from starting.` Das bedeutet, er spricht darüber, was das BIOS **tun kann**, wenn es beschädigt ist.

Wenn Sarah zustimmt, würde sie dann darüber sprechen, was das BIOS **ist** oder was das BIOS **tun kann**? Natürlich das Zweite, oder? Deshalb ist `Yeah, it can` die richtige Wahl.

Warum ist `yeah, it is` falsch? Weil `it is` über den Zustand des BIOS spricht, nicht darüber, was es tun kann. Hier ein Beispiel:

**Richtig**: `Yeah, it can.` – Das stimmt mit Marks Aussage darüber überein, was das BIOS tun kann.

**Falsch**: `Yeah, it is.` – Das passt nicht, weil Mark nicht beschreibt, was das BIOS ist, sondern über seine Auswirkungen spricht.
