# --description--

Eine Reihe mit einer Länge von `n` Einheiten hat rote Blöcke mit einer Mindestlänge von `m` Einheiten, die darauf platziert sind, sodass zwei rote Blöcke (die unterschiedliche Längen haben dürfen) durch mindestens ein schwarzes Quadrat getrennt sind.

Die Fill-Count-Funktion, $F(m, n)$, stellt die Anzahl der Möglichkeiten dar, wie eine Reihe gefüllt werden kann.

Zum Beispiel gilt $F(3, 29) = 673135$ und $F(3, 30) = 1089155$.

Das heißt, für m = 3 sieht man, dass n = 30 der kleinste Wert ist, für den die Fill-Count-Funktion erstmals eine Million überschreitet.

Auf die gleiche Weise kann für m = 10 überprüft werden, dass $F(10, 56) = 880711$ und $F(10, 57) = 1148904$ gilt, sodass n = 57 der kleinste Wert ist, für den die Fill-Count-Funktion erstmals eine Million überschreitet.

Für m = 50 finden Sie den kleinsten Wert von `n`, für den die Fill-Count-Funktion erstmals eine Million überschreitet.

**Hinweis:** Dies ist eine schwierigere Version von Aufgabe 114.
