# --explanation--

`Startups`는 보통 작고 초기 개발 단계에 있는 새로운 사업체입니다. 예를 들어:

`Many startups focus on innovative technology.` - 이것은 새로운 회사들이 종종 새로운 기술 아이디어를 만든다는 뜻입니다.

`Robust`는 강하고 어려운 상황에서도 잘 견디거나 성능을 발휘할 수 있다는 뜻입니다. 예를 들어:

`They built a robust system that rarely breaks down.` - 이것은 시스템이 강하고 신뢰할 수 있다는 뜻입니다.

`Cost-effective`는 많은 비용이 들지 않고 좋은 결과를 준다는 뜻입니다. 예를 들어:

`Buying in bulk is often more cost-effective.` - 이것은 돈을 절약한다는 뜻입니다.
