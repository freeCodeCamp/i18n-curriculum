# --description--

하나의 마커를 색상과 함께 가운데에 배치했으니, 이제 다른 마커들을 추가할 차례입니다.

`container` `div` 안에 `div` 요소 두 개를 더 추가하고 각각에 `marker` 클래스를 지정하세요.

# --hints--

첫 번째 새 `div` 요소는 여는 태그가 있어야 합니다.

```js
assert([...code.matchAll(/<div.*?>/gi)][2]);
```

첫 번째 새 `div` 요소는 닫는 태그가 있어야 합니다.

```js
assert([...code.matchAll(/<\/div\s*>/gi)][2]);
```

두 번째 새 `div` 요소는 여는 태그가 있어야 합니다.

```js
assert([...code.matchAll(/<div.*?>/gi)][3]);
```

두 번째 새 `div` 요소는 닫는 태그가 있어야 합니다.

```js
assert([...code.matchAll(/<\/div\s*>/gi)][3]);
```

새 `div` 요소들은 `div` 클래스가 `container`인 요소 안에 있어야 합니다.

```js
const containerChildren = [...document.querySelector('.container')?.children];
assert(containerChildren.every(child => child?.localName === 'div') && containerChildren.length === 3);
```

새 `div` 요소들은 둘 다 `marker` 클래스를 가져야 합니다.

```js
const containerChildren = [...document.querySelector('.container')?.children];
assert(containerChildren.every(child => child?.classList?.contains('marker')));
```
