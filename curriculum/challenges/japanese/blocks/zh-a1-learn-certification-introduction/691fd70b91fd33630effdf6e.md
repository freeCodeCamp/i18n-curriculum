# --description--

中国語は多くの地域変種で話されています。このカリキュラムでは、最も広く使われている標準的な話し言葉である**普通話（マンダリン）**を学びます。

中国語は主に簡体字と繁体字の2つの書き方があります。このカリキュラムでは**簡体字**に焦点を当てます。

中国語の漢字は音声を表すものではないため、発音を表すために**ピンイン**というシステムが使われます。ピンインはラテンアルファベットを使って中国語の発音を示すローマ字表記システムです。初心者が発音を学び、デジタル機器で中国語を入力するための重要なツールです。

これから、2文字ずつの中国語の単語を2つ入力します。信じられないかもしれませんね？freeCodeCampはこれを簡単にするために**ピンイン声調入力**と**ピンインから漢字への入力**を開発しました。

この課題では、ピンイン声調入力を使ってピンインを書く練習をします。

音声では、ワン・ファが中国語で「こんにちは、世界」と言っています。`ni3 hao3` と `shi4 jie4` を入力してみてください。間違えたら、間違った部分を削除してもう一度入力してください。

# --explanation--
  
ほとんどの場合、各中国語の漢字は1つのピンイン音節で表されます。各音節は、頭子音（子音）、韻母（必ず母音を含む）、そして声調を示す声調記号で構成されます。

コンピューターやモバイル機器のほとんどの組み込み中国語入力方式やサードパーティのアプリでは、通常声調記号は入力しません。頭子音と韻母だけを入力し、リストから正しい漢字を選びます。

しかし、声調記号を入力して見ることは、初心者が漢字の記憶を作り強化するのに役立ちます。

ピンイン声調入力では、文字と声調番号を入力すると、自動的に声調記号付きのピンインに変換されます。例えば、ここでは `ni3 hao3` が `nǐ hǎo` に、`shi4 jie4` が `shì jiè` に変わります。

この入力方法は、このカリキュラムの第2章でピンインを学ぶ際に頻繁に使います。
