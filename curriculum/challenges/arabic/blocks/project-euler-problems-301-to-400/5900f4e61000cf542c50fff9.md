# --description--

لتكن $T(n)$ هي العدد المثلثي رقم $n$، إذًا $T(n) = \frac{n(n + 1)}{2}$.

لتكن $dT(n)$ عدد القواسم للعدد $T(n)$. مثلاً: $T(7) = 28$ و $dT(7) = 6$.

لتكن $Tr(n)$ عدد الثلاثيات ($i$، $j$، $k$) بحيث $1 ≤ i < j < k ≤ n$ و $dT(i) > dT(j) > dT(k)$. حيث $Tr(20) = 14$، $Tr(100) = 5\\,772$ و $Tr(1000) = 11\\,174\\,776$.

احسب $Tr(60\\,000\\,000)$. أعطِ آخر 18 رقمًا من إجابتك.
