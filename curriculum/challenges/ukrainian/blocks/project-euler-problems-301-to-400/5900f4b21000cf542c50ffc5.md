# --description--

Нехай $a_n$ — послідовність, рекурсивно визначена так: $a_1 = 1$, $\displaystyle a_n = \left(\sum_{k = 1}^{n - 1} k \times a_k\right)\bmod n$.

Отже, перші 10 елементів $a_n$ такі: 1, 1, 0, 3, 0, 3, 5, 4, 1, 9.

Нехай $f(N, M)$ представляє кількість пар $(p, q)$ таких, що:

$$ 1 \le p \le q \le N \\; \text{та} \\; \left(\sum_{i = p}^q a_i\right)\bmod M = 0$$

Можна побачити, що $f(10, 10) = 4$ з парами (3,3), (5,5), (7,9) та (9,10).

Також вам дано, що $f({10}^4, {10}^3) = 97\\,158$.

Знайдіть $f({10}^{12}, {10}^6)$.
