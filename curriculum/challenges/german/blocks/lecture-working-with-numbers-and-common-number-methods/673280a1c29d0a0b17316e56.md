# --interactive--

Die `.toFixed()`-Methode ist eine eingebaute JavaScript-Funktion, die eine Zahl im Festkommaformat darstellt. Sie ist besonders nützlich, wenn Sie die Anzahl der Dezimalstellen einer Zahl steuern müssen, insbesondere bei der Anzeige von Währungswerten oder bei der Arbeit mit genauen Messungen.

Die `.toFixed()`-Methode wird auf eine Zahl aufgerufen und nimmt ein optionales Argument entgegen, das die Anzahl der Dezimalstellen angibt. Sie gibt eine String-Darstellung der Zahl mit der angegebenen Anzahl von Dezimalstellen zurück. Hier ist ein einfaches Beispiel, wie `.toFixed()` funktioniert:

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed(2)); // "3.14"
```

:::

In diesem Fall begrenzen wir die Anzahl der Dezimalstellen auf zwei. Also wird aus `3.14159` `3.14`. Es ist wichtig zu beachten, dass `.toFixed()` einen String zurückgibt, keine Zahl. Das liegt daran, dass die Methode hauptsächlich zum Formatieren von Zahlen für die Anzeige gedacht ist, nicht für weitere Berechnungen.

Die `.toFixed()`-Methode rundet die Zahl auf den nächstgelegenen Wert, der mit der angegebenen Anzahl von Dezimalstellen dargestellt werden kann. Dieses Rundungsverhalten ist wichtig zu verstehen:

:::interactive_editor

```js
console.log((3.14159).toFixed(3));  // "3.142"
console.log((3.14449).toFixed(3));  // "3.144"
console.log((3.14550).toFixed(3));  // "3.146"
```

:::

Wie Sie sehen, rundet `.toFixed()` auf, wenn die nächste Dezimalziffer `5` oder größer ist, und rundet ab, wenn sie kleiner ist. Wenn Sie `.toFixed()` ohne Argumente aufrufen, wird standardmäßig auf `0` Dezimalstellen gerundet:

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed()); // "3"
```

:::

Die `.toFixed()`-Methode kann besonders nützlich sein, wenn Sie mit finanziellen Berechnungen arbeiten oder Preise anzeigen:

:::interactive_editor

```js
let price = 19.99;
let taxRate = 0.08;
let total = price + (price * taxRate);

console.log("Total: $" + total.toFixed(2)); // "Total: $21.59"
```

:::

In diesem Beispiel sorgt `.toFixed(2)` dafür, dass der Gesamtbetrag immer mit zwei Dezimalstellen angezeigt wird, was in vielen Ländern der Standard für Währungen ist.

Zusammenfassend ist die `.toFixed()`-Methode ein leistungsfähiges Werkzeug zum Formatieren von Zahlen in JavaScript, besonders wenn Sie die Anzeige der Dezimalstellen steuern müssen. Obwohl sie hauptsächlich zur Formatierung der Ausgabe verwendet wird, sollten Sie ihr Verhalten im Auge behalten, insbesondere wenn präzise Berechnungen erforderlich sind.
