# --explanation--

`It seems to be` + 동사 `-ing` 형태는 겉으로 보기에 어떤 일이 일어나고 있는 것 같지만 확실하지 않을 때 사용합니다. 예를 들어:

`The app seems to be running slower today.` – 이것은 앱이 평소보다 느려 보인다는 뜻입니다.

`To cause something to` + 동사는 어떤 일이 일어나게 하거나 동작을 유발한다는 뜻입니다. 예를 들어:

`Low memory can cause the program to crash.` – 이것은 메모리가 부족해서 프로그램이 정지한다는 뜻입니다.
