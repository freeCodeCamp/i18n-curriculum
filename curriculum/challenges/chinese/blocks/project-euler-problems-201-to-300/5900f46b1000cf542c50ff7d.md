# --description--

定义 $f(n)$ 为 $n$ 的各位数字阶乘之和。例如，$f(342) = 3! + 4! + 2! = 32$。

定义 $sf(n)$ 为 $f(n)$ 各位数字之和。所以 $sf(342) = 3 + 2 = 5$。

定义 $g(i)$ 为满足 $sf(n) = i$ 的最小正整数 $n$。虽然 $sf(342)$ 是 5，$sf(25)$ 也是 5，并且可以验证 $g(5)$ 是 25。

将 $sg(i)$ 定义为 $g(i)$ 的数字之和。所以 $sg(5) = 2 + 5 = 7$。

此外，可以验证 $g(20)$ 是 267 且 $\sum sg(i)$ 对于 $1 ≤ i ≤ 20$ 是 156。

对于 $1 ≤ i ≤ 150$，$\sum sg(i)$ 是多少？
