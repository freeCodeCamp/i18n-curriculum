# --explanation--

この文で、マリアは`they`を使って**チームのメンバー**を指しています。会話の前半では、彼女は`the team`を使い、`I need to understand how the team feels about the new project deadline`という単数動詞を使っていました。今は個々の人を指しているので、`they`に切り替えています。別の例を挙げると：

- `The committee decides on the rules.` - グループ全体を指すときに単数動詞が使われていることに注目してください。

- `They discuss the rules before making a decision.` - 委員会のメンバーという個々の人を強調するために複数形に切り替わっています。
