En el desarrollo de software, un `branch` es como una línea de desarrollo separada. Piénsalo como una forma de trabajar en diferentes versiones de un proyecto al mismo tiempo.

Por ejemplo, un `branch` puede ser para nuevas características, mientras que otro es para corregir errores.
