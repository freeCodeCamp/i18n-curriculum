# --description--

`Touch` はしばしば何かに物理的に触れることを意味します。また、プログラミングのような比喩的な文脈で、何かを変更したり影響を与えたりすることを意味する場合もあります。

`These` は話者に近い、または現在話題に上っている特定のものを指すために使われます。

このチャレンジでは、`can` と `can't` の違いを復習し、人の能力や不可能性を表現する方法を学びます。

トム の発言を聞いて、空欄を埋めてください。
