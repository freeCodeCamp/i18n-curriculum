# --explanation--

`No one mentioned anything` significa que ninguém relatou nada incomum.

`Ask around` significa conversar com diferentes pessoas para reunir informações. Por exemplo:

- `I'll ask around to see if anyone changed the settings.` - Você vai conversar com as pessoas para descobrir quem mudou as configurações.

- `He asked around about the new software tool.` - Ele conversou com outras pessoas para obter mais informações sobre a ferramenta.
