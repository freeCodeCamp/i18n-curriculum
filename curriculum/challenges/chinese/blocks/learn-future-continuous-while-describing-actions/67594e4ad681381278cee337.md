# --explanation--

附加疑问句是添加到句子末尾的简短疑问句。它们用于确认信息或寻求认同。附加疑问句包括一个助动词和一个与主句主语匹配的代词。附加疑问句中的动词时态必须与主句的动词时态匹配。

`Haven't` 和 `hasn't` 是 `have` 和 `has` 的否定缩写， 用于反义疑问句中以确认或查看信息。 

当主语是复数（如 `we` 或 `they`）或主语是 `I` 时，使用 `haven't`。当主语是单数（如 `he`、`she` 或 `it`）时，使用 `hasn't`。例如：

- `We've been working hard, haven't we?`——主语句使用了 `we've`（`we have`），因此附加疑问句使用 `haven't we?` 来确认该语句。

- `She's finished her report, hasn't she?`——主句使用了 `she's`（`she has`），因此反义疑问句使用 `hasn't she?` 来确认该语句。
