# --description--

`There is` e `there are` sono usati per indicare l'esistenza o la presenza di qualcosa. `There is` si usa con nomi singolari o non numerabili, mentre `there are` si usa con nomi plurali.

Esempi:

- `There is a book on the table.` (nome singolare)

- `There is some water in the glass.` (nome non numerabile)

- `There are three apples in the basket.` (nome plurale)
