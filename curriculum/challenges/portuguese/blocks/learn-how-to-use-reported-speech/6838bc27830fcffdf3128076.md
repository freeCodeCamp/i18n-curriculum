# --explanation--

`Exactly` é um advérbio usado para mostrar total concordância ou correção com o que outra pessoa disse. `Exact`, por outro lado, é um adjetivo que significa preciso ou correto em detalhes.Por exemplo:

- `You are exactly right about the cause of the problem.` – Isso mostra total concordância. Expressa o quão certo (adjetivo) a pessoa estava.

- `We need the exact measurements to build the new server rack.` – Isso significa medidas muito precisas. Qualifica um substantivo em vez de um verbo ou adjetivo.
