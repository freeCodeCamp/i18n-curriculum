# --description--

Позитивне ціле число $A$ називатимемо «александрійським цілим числом», якщо існують цілі числа $p$, $q$, $r$ такі, що:

$$A = p \times q \times r$$

та

$$\frac{1}{A} = \frac{1}{p} + \frac{1}{q} + \frac{1}{r}$$


Наприклад, 630 є александрійським цілим числом ($p = 5$, $q = −7$, $r = −18$). Насправді, 630 — це шосте александрійське ціле число, перші шість александрійських цілих чисел: 6, 42, 120, 156, 420 і 630.

Знайдіть 150000-те александрійське ціле число.
