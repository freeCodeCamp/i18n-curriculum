# --interactive--

## Arbeiten mit Floats

- **Definition**: Floats werden verwendet, um ein Element aus dem normalen Fluss der Seite zu entfernen und es entweder links oder rechts in seinem Container zu positionieren. Dabei umfließt der Text den gefloateten Inhalt.

```css
float: left;
float: right;
```

- **Floats löschen**: Die `clear`-Eigenschaft wird verwendet, um zu bestimmen, ob ein Element unter dem gefloateten Inhalt verschoben werden muss. Wenn Sie mehrere gefloatete Elemente nebeneinander stapeln, kann es zu Überlappungen und Zusammenfallen im Layout kommen. Deshalb wurde ein `clearfix`-Hack entwickelt, um dieses Problem zu beheben.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="clearfix">
  <div class="box left">Left</div>
  <div class="box right">Right</div>
</div>
```

```css
.box {
  width: 100px;
  height: 60px;
  color: white;
  text-align: center;
  line-height: 60px;
}

.left {
  float: left;
  background: teal;
}

.right {
  float: right;
  background: purple;
}

.clearfix::after {
  content: "";
  display: block;
  clear: both;
}
```

:::

## Statische, relative und absolute Positionierung

- **Statische Positionierung**: Dies ist der normale Fluss des Dokuments. Elemente werden von oben nach unten und von links nach rechts nacheinander positioniert.
- **Relative Positionierung**: Damit können Sie die `top`, `left`, `right` und `bottom`-Eigenschaften verwenden, um das Element innerhalb des normalen Dokumentflusses zu positionieren. Sie können relative Positionierung auch nutzen, um Elemente mit anderen Elementen auf der Seite überlappen zu lassen.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="relative">Relative Element</div>
```

```css
.relative {
    position: relative;
    top: 30px;
    left: 30px;
}
```

:::

- **Absolute Positionierung**: Damit können Sie ein Element aus dem normalen Dokumentfluss herausnehmen, sodass es unabhängig von anderen Elementen agiert.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  Parent
  <div class="positioned">Absolute</div>
</div>
```

```css
.parent {
  position: relative;
  height: 120px;
  background: #eee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

## Feste und klebende Positionierung

- **Feste Positionierung**: Wenn ein Element mit `position: fixed` positioniert wird, wird es aus dem normalen Dokumentfluss entfernt und relativ zum Viewport platziert. Das bedeutet, es bleibt an derselben Stelle, auch wenn der Benutzer scrollt. Dies wird oft für Elemente wie Kopfzeilen oder Navigationsleisten verwendet, die jederzeit sichtbar bleiben sollen.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="navbar">Fixed Navbar</div>
<div style="height: 600px;"></div>
```

```css
.navbar {
  position: fixed; 
  top: 0; 
  width: 100%;
  background: #0077ff;
  color: white;
  padding: 8px; 
}
```

:::

- **Klebende Positionierung**: Diese Positionierungsart ist eine Mischung aus relativer und fester Positionierung. Anfangs verhält sich das Element wie bei relativer Positionierung und bleibt im Fluss des Dokuments. Sobald der Benutzer jedoch scrollt und das Element einen bestimmten Punkt überschreitet, „klebt“ es am Viewport (meist oben) und verhält sich wie ein fest positioniertes Element. Das ist ideal für Elemente wie klebende Navigationsleisten, die erst beim Scrollen an einer bestimmten Position fixiert werden.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Scroll down</p>
<div class="positioned">Sticky Element</div>
<div style="height: 500px;"></div>
```

```css
.positioned {
  position: sticky;
  top: 30px;
  left: 30px;
  background: #4caf50;
  color: white;
  padding: 6px;
}
```

:::

## Arbeiten mit der `z-index`-Eigenschaft

- **Definition**: Die `z-index`-Eigenschaft in CSS wird verwendet, um die vertikale Stapelreihenfolge von positionierten Elementen zu steuern, die sich auf der Seite überlappen.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box box1">1</div>
  <div class="box box2">2</div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

.box {
  position: absolute;
  width: 100px;
  height: 100px;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

.box1 {
  background: lightcoral;
  top: 20px;
  left: 20px;
  z-index: 1;
}

.box2 {
  background: steelblue;
  top: 40px;
  left: 40px;
  z-index: 2;
}
```

:::
