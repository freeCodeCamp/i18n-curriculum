# --interactive--

Unapojenga tovuti zenye lugha nyingi au kushughulikia sifa za data maalum, mara nyingi unahitaji kupangilia vipengele kulingana na lugha wanayozungumza au thamani maalum za data.

Sifa za `lang` na `data-lang` hutumika sana kwa madhumuni haya, na kichaguzi cha sifa ya kipengele katika CSS hukuwezesha kutumia mitindo kulingana na sifa hizi kwa ufanisi.

Sifa ya `lang` hutumika katika HTML kubainisha lugha ya yaliyomo ndani ya kipengele. Unaweza kutaka kupangilia vipengele tofauti kulingana na lugha zilizoandikwa, hasa kwenye tovuti yenye lugha nyingi.

Hapa kuna mfano wa jinsi unavyoweza kutumia kichaguzi cha sifa ya kipengele kulenga vipengele vyenye sifa maalum ya `lang`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">This is an English paragraph.</p>
<p lang="fr">Ceci est un paragraphe en français.</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

Sheria hii ya CSS inatumia mtindo wa kuandika kwa herufi za mwinuko kwa kipengele chochote cha aya ambacho sifa ya `lang` imewekwa kuwa Kiingereza (`en`). Hii inaweza kuwa na manufaa kuonyesha maandishi ya Kiingereza katika hati inayojumuisha lugha nyingi.

Vivyo hivyo, unaweza kutumia kichaguzi cha sifa ya kipengele kulenga vipengele vyenye sifa ya `data-lang`.

Sifa za data maalum kama `data-lang` hutumika sana kuhifadhi taarifa za ziada katika vipengele, kama vile kubainisha lugha inayotumika ndani ya sehemu maalum ya maandishi.

Hapa ni jinsi unavyoweza kutumia CSS kulenga na kupangilia vipengele kulingana na sifa ya `data-lang`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p data-lang="fr">Ceci est un paragraphe en français.</p>
<p data-lang="en">This is a paragraph in English.</p>
```

```css
p[data-lang="fr"] {
  color: blue;
}
```

:::

Katika kesi hii, kipengele chochote cha `p` chenye sifa ya `data-lang` iliyowekwa kuwa Kifaransa (`fr`) kitawekwa rangi ya maandishi kuwa buluu. Hii inakuwezesha kutambua na kupangilia haraka sehemu za yaliyomo kulingana na taarifa za lugha zilizohifadhiwa katika sifa ya `data-lang`.

Vichaguzi vya sifa kama hivi vinatoa njia yenye nguvu ya kutumia mitindo ya masharti kulingana na metadata iliyojumuishwa ndani ya HTML yako, na kufanya kurasa zako za mtandao kuwa za muktadha zaidi na zenye mabadiliko.
