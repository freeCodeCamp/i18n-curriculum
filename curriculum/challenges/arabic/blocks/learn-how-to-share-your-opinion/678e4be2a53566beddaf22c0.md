# --explanation--

`Build tools` هي برامج تساعد في أتمتة وإدارة المهام المتكررة مثل الترجمة، والاختبار، وتغليف مشاريع البرمجيات. على سبيل المثال:

`Maven is a popular build tool for Java projects, helping automate the compilation process.` - هذا يعني أن Maven يبسط عملية بناء تطبيقات Java من خلال أتمتة مهام مثل ترجمة الشفرة وإنشاء الحزم.

`Built-in` تشير إلى ميزات هي بالفعل جزء من البرنامج ولا تتطلب تثبيتًا أو إعدادًا إضافيًا. على سبيل المثال:

`Visual Studio has built-in tools for debugging code.` - هذا يعني أن أدوات تصحيح الأخطاء مدمجة بالفعل في Visual Studio، ولا يحتاج المستخدمون إلى تثبيت أي أدوات إضافية لبدء تصحيح الأخطاء.

`Version control` هو نظام يتتبع ويدير التغييرات على الشفرة، مما يسمح للفرق بالعمل على نفس المشروع دون فقدان أي عمل أو حدوث تعارضات. على سبيل المثال:

`Git is a widely-used version control system for tracking code changes and collaborating with teammates.` - هذا يعني أن Git يساعد المطورين في إدارة إصدارات الشفرة المختلفة والعمل معًا على نفس المشروع.
