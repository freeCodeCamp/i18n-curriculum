# --description--

Fare `work on` significa dedicare tempo e impegno a svolgere un'attività o migliorare qualcosa. Per esempio:

`She is working on a new app.` - Questo significa che sta sviluppando o migliorando l'app.

Fare `take time off` significa smettere di lavorare per un breve periodo, di solito per riposare o per motivi personali. Per esempio:

`I'm taking time off tomorrow because I have a doctor's appointment.` - Questo significa non lavorare per tutta la giornata.
