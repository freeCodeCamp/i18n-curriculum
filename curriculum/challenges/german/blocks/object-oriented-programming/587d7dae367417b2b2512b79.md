# --description--

Die `Bird` und `Dog` Konstruktoren aus der letzten Aufgabe funktionierten gut. Beachten Sie jedoch, dass alle `Birds`, die mit dem `Bird` Konstruktor erstellt werden, automatisch Albert heißen, blau gefärbt sind und zwei Beine haben. Was ist, wenn Sie Vögel mit unterschiedlichen Werten für name und color möchten? Es ist möglich, die Eigenschaften jedes Vogels manuell zu ändern, aber das wäre sehr aufwendig:

```js
let swan = new Bird();
swan.name = "Carlos";
swan.color = "white";
```

Angenommen, Sie schreiben ein Programm, um Hunderte oder sogar Tausende verschiedener Vögel in einer Voliere zu verwalten. Es würde viel Zeit kosten, alle Vögel zu erstellen und dann die Eigenschaften für jeden einzelnen zu ändern. Um verschiedene `Bird` Objekte leichter zu erstellen, können Sie Ihren Bird-Konstruktor so gestalten, dass er Parameter akzeptiert:

```js
function Bird(name, color) {
  this.name = name;
  this.color = color;
  this.numLegs = 2;
}
```

Dann übergeben Sie die Werte als Argumente, um jeden einzigartigen Vogel im `Bird` Konstruktor zu definieren: `let cardinal = new Bird("Bruce", "red");` Dies erzeugt eine neue Instanz von `Bird` mit den `name` und `color` Eigenschaften, die auf `Bruce` bzw. `red` gesetzt sind. Die `numLegs` Eigenschaft ist weiterhin auf 2 gesetzt. Der `cardinal` hat diese Eigenschaften:

```js
cardinal.name
cardinal.color
cardinal.numLegs
```

Der Konstruktor ist flexibler. Es ist jetzt möglich, die Eigenschaften für jeden `Bird` zum Zeitpunkt seiner Erstellung zu definieren, was eine Möglichkeit ist, wie JavaScript-Konstruktoren so nützlich sind. Sie gruppieren Objekte basierend auf gemeinsamen Merkmalen und Verhalten und definieren eine Vorlage, die deren Erstellung automatisiert.

# --instructions--

Erstellen Sie einen weiteren `Dog` Konstruktor. Richten Sie ihn diesmal so ein, dass er die Parameter `name` und `color` annimmt und die Eigenschaft `numLegs` fest auf 4 setzt. Erstellen Sie dann einen neuen `Dog`, der in einer Variablen `terrier` gespeichert wird. Übergeben Sie ihm zwei Strings als Argumente für die `name` und `color` Eigenschaften.

# --hints--

`Dog` sollte ein Argument für `name` erhalten.

```js
assert(new Dog('Clifford').name === 'Clifford');
```

`Dog` sollte ein Argument für `color` erhalten.

```js
assert(new Dog('Clifford', 'yellow').color === 'yellow');
```

`Dog` sollte die Eigenschaft `numLegs` auf 4 gesetzt haben.

```js
assert(new Dog('Clifford').numLegs === 4);
```

`terrier` sollte mit dem `Dog` Konstruktor erstellt werden.

```js
assert(terrier instanceof Dog);
```
