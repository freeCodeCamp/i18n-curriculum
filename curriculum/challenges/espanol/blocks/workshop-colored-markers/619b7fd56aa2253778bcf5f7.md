# --description--

El modelo de color <dfn>HSL</dfn>, o tono, saturación y luminosidad, es otra manera de representar colores.

La función CSS `hsl` acepta 3 valores: un número del 0 al 360 para el tono, un porcentaje del 0 al 100 para la saturación y un porcentaje del 0 al 100 para la luminosidad.

Si imaginas una rueda de colores, el tono rojo está a 0 grados, el verde a 120 grados y el azul a 240 grados.

La saturación es la intensidad de un color desde un 0%, o gris, hasta un 100% para color puro. Debes añadir el signo de porcentaje `%` a los valores de saturación y luminosidad.

La luminosidad es cuánta luz aparenta tener un color, desde un 0%, o negro total, hasta un 100%, blanco total, siendo 50% neutral.

En la regla CSS `.blue`, utiliza la función `hsl` para cambiar la propiedad `background-color` a azul puro. Configura el tono a `240`, la saturación a `100%` y la luminosidad a `50%`.
