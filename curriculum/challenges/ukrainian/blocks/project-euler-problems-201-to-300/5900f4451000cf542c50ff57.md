# --description--

Розглянемо числа $t(n)$ форми $t(n) = 2n^2 - 1$ при $n > 1$.

Перші такі числа: 7, 17, 31, 49, 71, 97, 127 і 161.

Виявляється, що лише $49 = 7 \times 7$ та $161 = 7 \times 23$ не є простими.

Для $n ≤ 10000$ існує 2202 простих чисел $t(n)$.

Скільки чисел $t(n)$ є простими для $n ≤ 50\\,000\\,000$?
