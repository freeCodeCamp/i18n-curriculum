# --explanation--

`I'll` è la contrazione di `I will`. In questo contesto, esprime una decisione presa nel momento in cui si parla. È comune quando qualcuno reagisce a una situazione o accetta di fare qualcosa prontamente.

`Now` si usa per indicare qualcosa che sta accadendo nel momento presente. Per esempio:

`I'll send the report now.` - Questo mostra la decisione di agire immediatamente.
