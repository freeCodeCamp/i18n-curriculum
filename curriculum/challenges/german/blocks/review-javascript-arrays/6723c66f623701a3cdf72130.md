## Grundlagen zu JavaScript-Arrays

- **Definition**: Ein JavaScript-Array ist eine geordnete Sammlung von Werten, die jeweils durch einen numerischen Index identifiziert werden. Die Werte in einem JavaScript-Array können unterschiedliche Datentypen haben, darunter Zahlen, Strings, Booleans, Objekte und sogar andere Arrays. Arrays sind im Speicher zusammenhängend, das heißt, alle Elemente werden in einem einzigen, durchgehenden Speicherblock abgelegt, was effizientes Indizieren und schnellen Zugriff auf Elemente über ihren Index ermöglicht.

```js
const developers = ["Jessica", "Naomi", "Tom"];
```

- **Zugriff auf Elemente in Arrays**: Um auf Elemente eines Arrays zuzugreifen, müssen Sie das Array gefolgt von seiner Indexnummer in eckigen Klammern referenzieren. JavaScript-Arrays sind nullbasiert indiziert, das heißt, das erste Element befindet sich am Index 0, das zweite am Index 1 usw. Wenn Sie versuchen, auf einen Index zuzugreifen, der im Array nicht existiert, gibt JavaScript `undefined` zurück.

:::interactive_editor

```js
const developers = ["Jessica", "Naomi", "Tom"];
console.log(developers[0]) // "Jessica"
console.log(developers[1]) // "Naomi"

console.log(developers[10]) // undefined
```

:::

- **`length`-Eigenschaft**: Diese Eigenschaft wird verwendet, um die Anzahl der Elemente in einem Array zurückzugeben.

:::interactive_editor

```js
const developers = ["Jessica", "Naomi", "Tom"];
console.log(developers.length) // 3
```

:::

- **Elemente in einem Array aktualisieren**: Um ein Element in einem Array zu aktualisieren, verwenden Sie den Zuweisungsoperator (`=`), um dem Element an einem bestimmten Index einen neuen Wert zuzuweisen.

:::interactive_editor

```js
const fruits = ['apple', 'banana', 'cherry'];
fruits[1] = 'blueberry';

console.log(fruits); // ['apple', 'blueberry', 'cherry']
```

:::

## Zweidimensionale Arrays  

- **Definition**: Ein zweidimensionales Array ist im Grunde ein Array von Arrays. Es wird verwendet, um Daten darzustellen, die eine natürliche rasterartige Struktur haben, wie ein Schachbrett, eine Tabelle oder Pixel in einem Bild. Um auf ein Element in einem zweidimensionalen Array zuzugreifen, benötigen Sie zwei Indizes: einen für die Zeile und einen für die Spalte.

:::interactive_editor

```js
const chessboard = [
    ['R', 'N', 'B', 'Q', 'K', 'B', 'N', 'R'],
    ['P', 'P', 'P', 'P', 'P', 'P', 'P', 'P'],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    ['p', 'p', 'p', 'p', 'p', 'p', 'p', 'p'],
    ['r', 'n', 'b', 'q', 'k', 'b', 'n', 'r']
];

console.log(chessboard[0][3]); // "Q"
```

:::

## Array-Destrukturierung  

- **Definition**: Die Array-Destrukturierung ist eine Funktion in JavaScript, die es Ihnen erlaubt, Werte aus Arrays zu extrahieren und sie auf eine prägnantere und lesbarere Weise Variablen zuzuweisen. Sie bietet eine bequeme Syntax, um Array-Elemente in einzelne Variablen zu entpacken.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange"];

const [first, second, third] = fruits;

console.log(first); // "apple"
console.log(second); // "banana"
console.log(third); // "orange"
```

:::

- **Rest-Syntax**: Diese ermöglicht es Ihnen, die verbleibenden Elemente eines Arrays, die nicht destrukturiert wurden, in einem neuen Array zu erfassen.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange", "mango", "kiwi"];
const [first, second, ...rest] = fruits;

console.log(first); // "apple"
console.log(second); // "banana"
console.log(rest); // ["orange", "mango", "kiwi"]
```

:::

## Häufige Array-Methoden 

- **`push()`-Methode**: Diese Methode wird verwendet, um Elemente am Ende des Arrays hinzuzufügen und gibt die neue Länge zurück.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.push("ice cream");

console.log(desserts); // ["cake", "cookies", "pie", "ice cream"];
```

:::

- **`pop()`-Methode**: Diese Methode wird verwendet, um das letzte Element aus einem Array zu entfernen und gibt dieses entfernte Element zurück. Wenn das Array leer ist, ist der Rückgabewert `undefined`.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.pop();

console.log(desserts); // ["cake", "cookies"];
```

:::

- **`shift()`-Methode**: Diese Methode wird verwendet, um das erste Element aus einem Array zu entfernen und gibt dieses entfernte Element zurück. Wenn das Array leer ist, ist der Rückgabewert `undefined`.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.shift();

console.log(desserts); // ["cookies", "pie"];
```

:::

- **`unshift()`-Methode**: Diese Methode wird verwendet, um Elemente am Anfang des Arrays hinzuzufügen und gibt die neue Länge zurück.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.unshift("ice cream");

console.log(desserts); // ["ice cream", "cake", "cookies", "pie"];
```

:::

- **`indexOf()`-Methode**: Diese Methode ist nützlich, um den ersten Index eines bestimmten Elements innerhalb eines Arrays zu finden. Wenn das Element nicht gefunden werden kann, gibt sie `-1` zurück.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange", "banana"];
const index = fruits.indexOf("banana");

console.log(index); // 1
console.log(fruits.indexOf("not found")); // -1
```

:::

- **`splice()`-Methode**: Diese Methode wird verwendet, um Elemente an beliebiger Position in einem Array hinzuzufügen oder zu entfernen. Der Rückgabewert der `splice()`-Methode ist ein Array der aus dem Array entfernten Elemente. Wenn nichts entfernt wird, wird ein leeres Array zurückgegeben. Diese Methode verändert das ursprüngliche Array, indem sie es an Ort und Stelle modifiziert, anstatt ein neues Array zu erstellen. Das erste Argument gibt den Index an, an dem die Modifikation beginnen soll. Das zweite Argument ist die Anzahl der Elemente, die entfernt werden sollen. Die folgenden Argumente sind die Elemente, die hinzugefügt werden sollen.

:::interactive_editor

```js
const colors = ["red", "green", "blue"];
colors.splice(1, 0, "yellow", "purple");

console.log(colors); // ["red", "yellow", "purple", "green", "blue"]
```

:::

- **`includes()`-Methode**: Diese Methode wird verwendet, um zu überprüfen, ob ein Array einen bestimmten Wert enthält. Diese Methode gibt `true` zurück, wenn das Array das angegebene Element enthält, und `false` sonst.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];

console.log(programmingLanguages.includes("Python")); // true
console.log(programmingLanguages.includes("Perl")); // false
```

:::

- **`concat()`-Methode**: Diese Methode erstellt ein neues Array, indem zwei oder mehr Arrays zusammengeführt werden.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];
const newList = programmingLanguages.concat("Perl");

console.log(newList); // ["JavaScript", "Python", "C++", "Perl"]
```

:::

- **`slice()`-Methode**: Diese Methode gibt ein neues Array zurück, das eine flache Kopie eines Abschnitts des ursprünglichen Arrays enthält, definiert durch Start- und Endindizes. Das neue Array enthält Referenzen auf dieselben Elemente wie das ursprüngliche Array (keine Duplikate). Das bedeutet, wenn die Elemente primitive Werte (wie Zahlen oder Strings) sind, werden die Werte kopiert; wenn die Elemente Objekte oder Arrays sind, werden die Referenzen kopiert, nicht die Objekte selbst.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];
const newList = programmingLanguages.slice(1);

console.log(newList); // ["Python", "C++"]
```

:::

- **Spread-Syntax**: Die Spread-Syntax wird verwendet, um flache Kopien eines Arrays zu erstellen.

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const shallowCopiedArray = [...originalArray];

shallowCopiedArray.push(4);

console.log(originalArray); // [1, 2, 3]
console.log(shallowCopiedArray); // [1, 2, 3, 4]
```

:::

- **`split()`-Methode**: Diese Methode teilt einen String in ein Array von Teilstrings auf und gibt an, wo die Aufteilung anhand eines angegebenen Trennzeichens erfolgen soll. Wenn kein Trennzeichen angegeben wird, gibt die Methode ein Array zurück, das den ursprünglichen String als einzelnes Element enthält.

:::interactive_editor

```js
const str = "hello";
const charArray = str.split("");

console.log(charArray); // ["h", "e", "l", "l", "o"]
```

:::

- **`reverse()`-Methode**: Diese Methode kehrt ein Array an Ort und Stelle um.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
console.log(desserts.reverse()); // ["pie", "cookies", "cake"]
```

:::

- **`join()`-Methode**: Diese Methode verbindet alle Elemente eines Arrays zu einem einzigen String, wobei jedes Element durch ein angegebenes Trennzeichen getrennt wird. Wenn kein Trennzeichen angegeben wird oder ein leerer String (`""`) verwendet wird, werden die Elemente ohne Trennzeichen verbunden.

:::interactive_editor

```js
const reversedArray = ["o", "l", "l", "e", "h"];
const reversedString = reversedArray.join("");

console.log(reversedString); // "olleh"
```

:::
