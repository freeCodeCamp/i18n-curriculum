# --description--

`There is` 和 `there are` 用于表示某物的存在或出现。`There is` 用于单数或不可数名词，而 `there are` 用于复数名词。

示例：

- `There is a book on the table.` (单数名词)

- `There is some water in the glass.`（不可数名词）

- `There are three apples in the basket.`（复数名词）
