# --explanation--

`Lined up`は「正しい順序や位置に整理または準備されている」という意味です。例えば：

`We have all the speakers lined up for the conference.` – これはすべて予定されて確認済みであることを意味します。

`On time`は「遅れておらず、正しいまたは予想された時間にある」という意味です。例えば：

`The train arrived on time despite the heavy rain.` – これは予定通りに到着したことを意味します。
