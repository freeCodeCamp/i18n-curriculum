# --explanation--

`Hidden` significa algo que está oculto o no es inmediatamente visible, a menudo requiriendo un esfuerzo extra para encontrar. 
 
`Malware` se refiere a software malicioso diseñado para dañar, interrumpir o explotar un sistema informático.  

Cuando se combinan, `hidden malware` describe software dañino que es difícil de detectar porque está oculto dentro del sistema. Por ejemplo:

`The scan detected hidden malware that had bypassed the antivirus software.` - Esto significa que el malware pudo evitar la detección por parte del programa antivirus hasta que un escaneo lo encontró.
