# --explanation--

`first`, `then`, `after that`, `finally` 같은 순차 연결사는 행동이나 이벤트의 순서를 설명할 때 사용합니다.

- `First`은 첫 번째 행동을 설명할 때 씁니다.

- `Then`은 바로 다음 단계를 가리킵니다.

- `After that`은 앞선 행동과 구분을 두고 다음 단계를 소개할 때 씁니다.

- `Finally`은 수열의 마지막 단계를 표시합니다.
