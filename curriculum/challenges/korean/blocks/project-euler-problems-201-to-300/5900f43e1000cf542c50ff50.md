# --description--

정수 좌표 ($x$,$y$)를 가지며 $|x| + |y| ≤ r$를 만족하는 점들의 집합 $S(r)$을 고려하세요.

점 $O$는 (0,0)이고 점 $C$는 ($\frac{r}{4}$,$\frac{r}{4}$)입니다.

삼각형 $OBC$가 둔각을 가지도록 하는 $S(r)$ 내의 점 $B$의 개수를 $N(r)$이라 합시다. 즉, 가장 큰 각 $α$가 $90° < α < 180°$를 만족합니다.

예를 들어, $N(4)=24$이고 $N(8)=100$입니다.

$N(1\\,000\\,000\\,000)$은 얼마인가요?
