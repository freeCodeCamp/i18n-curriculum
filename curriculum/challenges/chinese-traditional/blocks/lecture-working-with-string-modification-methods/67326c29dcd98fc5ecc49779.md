# --interactive--

在 JavaScript 中，有許多情況你可能需要用另一個字串取代字串的一部分。

例如，你可能需要在 URL 中更新使用者資訊、變更日期格式，或修正使用者產生內容中的錯誤。

JavaScript 中的 `replace()` 方法允許你在字串中尋找指定的值（例如字或字元）並將其取代為另一個值。該方法傳回一個包含取代結果的新字串，並且保留原始字串不變，因為 JavaScript 字串是不可變的。

這是基本語法：

```js
string.replace(searchValue, newValue);
```

`searchValue` 是你想要在字串中搜尋的值。它可以是字串或描述文字樣式的正規表達式（regex）。這讓你能以靈活且強大的方式搜尋和操作字串。你將在未來的課程中學習更多關於正規表達式的內容。

`newValue` 是將取代 `searchValue` 的值。以下是一個簡單的範例：

:::interactive_editor

```js
let text = "I love JavaScript!";
console.log(text); // "I love JavaScript!"
let newText = text.replace("JavaScript", "coding");
console.log(newText);  // "I love coding!"
```

:::

在此範例中，字 `JavaScript` 位於字串中並被取代為 `coding`。

`replace()` 方法區分大小寫，這表示它只會找到與 `searchValue` 完全相符的項目。例如：

:::interactive_editor

```js
let sentence = "I enjoy working with JavaScript.";
console.log(sentence);  // "I enjoy working with JavaScript."
let updatedSentence = sentence.replace("javascript", "coding");
console.log(updatedSentence);  // "I enjoy working with JavaScript."
```

:::

這裡，由於 `javascript`（小寫的 `j`）與 `JavaScript`（大寫的 `J`）不相符，因此不會進行取代。

預設情況下，`replace()` 方法只會替換 `searchValue` 的第一個出現。如果該值在字串中出現多次，只有第一個會被替換：

:::interactive_editor

```js
let phrase = "Hello, world! Welcome to the world of coding.";
console.log(phrase);  // "Hello, world! Welcome to the world of coding."
let updatedPhrase = phrase.replace("world", "universe");
console.log(updatedPhrase);  // "Hello, universe! Welcome to the world of coding."
```

:::

請注意，只有第一次出現的 `world` 被取代為 `universe`。

JavaScript 中的 `replace()` 方法是字串操作中強大且靈活的工具。

它讓你取代字串中特定的零件，無論你是在處理單一字元、字，或是使用正規表達式的複雜樣式。

雖然它適合用於簡單的替換，但了解它的大小寫敏感性和預設行為（例如只替換第一個出現的項目）可以幫助你更有效地使用它。
