# --interactive--

Ви, можливо, знайомі з роботою з атрибутами у ваших HTML і CSS проєктах. Але чи знали ви, що можна додавати атрибути до HTML-елементів у вашому JavaScript файлі?

У цьому уроці ви навчитеся працювати з методом `setAttribute()`, щоб додавати атрибути до ваших HTML-елементів.

Ось базовий синтаксис:

```js
setAttribute(attribute, value);
```

Давайте розглянемо кілька прикладів, щоб краще зрозуміти, як використовувати метод `setAttribute()`.

У цьому першому прикладі у нас є елемент `p` всередині HTML:

```html
<p id="para">I am a paragraph</p>
```

Щоб додати атрибут `class`, спочатку потрібно отримати доступ до цього елемента `p` за допомогою методу `getElementById()`. Потім ми можемо використати метод `setAttribute()`, щоб додати атрибут `class` і призначити йому значення `my-class`.

Ми використовуємо `outerHTML`, щоб вивести в консоль увесь HTML-елемент разом із його атрибутами.

:::interactive_editor

```html
<p id="para">I am a paragraph</p>
<script src="index.js"></script>
```

```js
const para = document.getElementById("para");
para.setAttribute("class", "my-class");

console.log(`${para.outerHTML}`);
```

:::

Якщо у HTML-елемента вже є атрибут, ви можете оновити його значення за допомогою методу `setAttribute()`.

У цьому прикладі у нас є елемент `div` з атрибутом `class`, встановленим у `my-class`:

```html
<div class="my-class"></div>
```

Щоб оновити значення атрибута `class` на example, ми знову можемо використати метод `setAttribute()`:

:::interactive_editor

```html
<div class="my-class"></div>
<script src="index.js"></script>
```

```js
const divEl = document.querySelector(".my-class");
divEl.setAttribute("class", "example");

console.log(`${divEl.outerHTML}`);
```

:::

Отже, які реальні приклади використання методу `setAttribute()`?

Якщо ви створюєте динамічну галерею зображень, можливо, вам потрібно буде оновити атрибут `src` елемента зображення, коли користувач натискає на мініатюру.

Інший приклад — якщо ви працюєте з валідацією форм і потрібно додати певні атрибути, як `required` або `minlength`, до елемента `input`.
