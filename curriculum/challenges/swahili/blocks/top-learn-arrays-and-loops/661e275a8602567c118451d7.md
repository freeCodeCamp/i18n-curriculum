# --description--

Sasa unajua kuhusu njia maarufu zaidi za kurudia juu ya thamani na safu za data, kuna njia chache zaidi za kurudia juu ya safu za data. Safu za data zina njia iliyo jengwa ndani iitwayo `map()`, ambayo hutumika kuunda safu mpya kwa kutumia kitendakazi kwa kila kipengele cha safu ya awali. Njia ya `map()` haibadilishi safu ya awali. Sintaksia ya njia ya `map()` ni kama ifuatavyo:

```javascript
const array = [1, 2, 3, 4, 5];

const newArray = array.map((arrayValue) => {
  return arrayValue * 2;
});

console.log(newArray); // Output: [2, 4, 6, 8, 10]
```

Njia ya `map()` huunda safu mpya kwa kutumia kitendakazi `(arrayValue) => { return arrayValue * 2; }` kwa kila kipengele cha safu ya awali. Hii ni muhimu hasa unapotaka kubadilisha vipengele vya safu bila kubadilisha safu ya awali.
