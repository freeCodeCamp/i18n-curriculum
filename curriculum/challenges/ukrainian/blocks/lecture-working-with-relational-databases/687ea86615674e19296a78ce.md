# --description--

Реляційні бази даних мають широкий спектр застосувань, включно з веб-розробкою, системами управління запасами, медичними системами, застосунками електронної комерції та іншими.

Деякі з найпоширеніших реляційних баз даних — це MySQL, PostgreSQL, SQLite і Microsoft SQL Server. Давайте дізнаємося трохи про кожну з них.

MySQL — це система управління реляційними базами даних з відкритим кодом. Вона дуже популярна для веб-розробки, бо проста у використанні, надійна та ефективна для створення реальних застосунків. Також має активну спільноту розробників, які підтримують і розвивають її.

PostgreSQL офіційно описується як «найпотужніша у світі реляційна база даних з відкритим кодом». Це безкоштовна, відкрита об’єктно-реляційна система баз даних, відома своєю надійністю, цілісністю даних і розширюваністю. Наприклад, ви можете визначати власні типи даних, кастомні функції та писати код різними мовами програмування без необхідності перекомпіляції бази даних.

SQLite — це безкоштовний, відкритий і легкий SQL-двигун бази даних, що зберігається у файлі. Це один із найпоширеніших двигунів баз даних у світі. SQLite є автономним і безсерверним. Це база даних з нульовою конфігурацією, тобто не потребує початкового налаштування, що чудово підходить для швидкого старту.

Ці три бази даних є безкоштовними та з відкритим кодом. Існують також інші популярні пропрієтарні варіанти, зокрема Microsoft SQL Server і Oracle database.

Microsoft SQL Server — популярна система управління реляційними базами даних, розроблена Microsoft і використовується в різних застосунках.

Oracle database — масштабована система управління реляційними базами даних, часто застосовується в корпоративних застосунках, які потребують високої продуктивності.

Чудово. Тепер, коли ви знаєте більше про різні доступні варіанти, давайте подивимося, як встановити PostgreSQL.

Спершу перейдіть на офіційний вебсайт: <a href="https://www.postgresql.org/" target="_blank">https://www.postgresql.org/</a>.

Коли ви там, натисніть на кнопку «Download».

Це приведе вас на сторінку, де можна вибрати операційну систему. Якщо натиснути на одну з них, ви перейдете до інструкцій для цієї ОС.

Ми пройдемо процес встановлення для операційної системи Windows, але процес схожий для інших ОС.

На цій сторінці можна скачати інсталятор, натиснувши на посилання «Download the installer».

Це приведе вас на сторінку, де можна вибрати версію PostgreSQL, яку хочете встановити. Якщо натиснути на відповідну іконку, завантаження має початися.

Потім у вашій папці Downloads або тій папці, куди ви за замовчуванням зберігаєте завантаження, має з’явитися інсталятор.

Якщо двічі натиснути на нього, почнеться процес налаштування. Натисніть Next.

Виберіть директорію, куди буде встановлено PostgreSQL. За замовчуванням буде запропоновано одну, але ви можете її змінити.

Потім виберіть компоненти, які хочете встановити. Існує чотири можливі компоненти:

- PostgreSQL Server — сам двигун бази даних.
- pgAdmin 4 — зручний інструмент для управління базою даних.
- Stack Builder — інструмент для скачування та встановлення додаткового програмного забезпечення та розширень, пов’язаних з PostgreSQL.
- Command Line Tools — інструменти командного рядка для взаємодії з сервером PostgreSQL безпосередньо з терміналу. Сюди входить psql, інтерактивний термінал для виконання SQL-запитів.

Коли будете готові, натисніть Next.

Потім виберіть директорію для зберігання даних. За замовчуванням буде запропоновано шлях, але ви можете його змінити. Коли будете готові, натисніть Next.

Тепер потрібно ввести пароль для суперкористувача бази даних — користувача з необмеженим доступом до всіх баз даних і об’єктів. Введіть пароль, повторіть його і натисніть Next.

Потім виберіть номер порту, на якому сервер має слухати. За замовчуванням це 5432. Потім натисніть Next.

Вам потрібно вибрати локаль для нового кластера баз даних. Це важливо, щоб база даних правильно інтерпретувала типи даних, чутливі до локалі. Потім натисніть Next.

Нарешті, ви побачите підсумок перед встановленням. Приділіть час, щоб його переглянути, і натисніть Next.

Потрібно підтвердити, що хочете почати процес встановлення. Якщо натиснути Next, процес розпочнеться. Це займе лише кілька хвилин.

Коли процес завершиться, залежно від встановлених компонентів, можливо, вам запропонують вибрати, чи хочете ви запустити Stack Builder для скачування та встановлення додаткових інструментів, драйверів і застосунків.

Якщо вирішите запустити Stack Builder, побачите екран, де можна почати встановлення. Або натисніть Cancel, щоб зробити це пізніше.

Вітаємо! Тепер у вас встановлено PostgreSQL на комп’ютері. Процес має бути схожим для інших операційних систем. Після цього можна починати використовувати базу даних.

Перейдіть до своїх застосунків і знайдіть pgAdmin — платформу для адміністрування та розробки PostgreSQL з відкритим кодом, яка мала встановитися, якщо ви вибрали її серед компонентів. Натисніть на pgAdmin 4, щоб відкрити.

Під час запуску застосунку ви побачите початковий екран завантаження. Потім має з’явитися панель керування, де можна почати новий сервер, перевірити бази даних, налаштувати pgAdmin і багато іншого. Там також будуть корисні посилання на документацію.

Якщо натиснути на «Servers» у верхньому лівому куті, потрібно буде ввести пароль, який ви обрали під час встановлення.

Після введення пароля ви побачите екран із доступом до ваших баз даних, включно з їхніми таблицями.

Також ви знайдете ще один дуже корисний інструмент — psql, інтерактивний термінал для SQL-запитів. Щоб відкрити його, знайдіть psql у системі. Натисніть на нього.

Відкриється SQL Shell, де можна починати виконувати команди.

Ось як можна встановити PostgreSQL на Windows. Процес дуже схожий для macOS і Linux.

Якщо ж ви встановлюєте на macOS, можна скористатися Homebrew — менеджером пакетів. Коли Homebrew встановлено, виконайте цю команду, щоб встановити PostgreSQL:

```bash
brew install postgresql@<version>
```

Ви можете вказати версію, яку хочете встановити, написавши число після символу at (`@`).
