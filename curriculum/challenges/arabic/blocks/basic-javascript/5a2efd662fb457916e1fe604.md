# --description--

نوع الحلقة التالي الذي ستتعلمه يُسمى حلقة `do...while`. تُسمى حلقة `do...while` لأنها تبدأ أولًا بـ `do` تنفيذ مرة واحدة للشفرة داخل الحلقة مهما كان، ثم تستمر في تشغيل الحلقة `while` تحقق الشرط المحدد ويكون نتيجته `true`.

```js
const ourArray = [];
let i = 0;

do {
  ourArray.push(i);
  i++;
} while (i < 5);
```

المثال أعلاه يتصرف بشكل مشابه لأنواع الحلقات الأخرى، وستبدو المصفوفة الناتجة كـ `[0, 1, 2, 3, 4]`. لكن ما يميز حلقة `do...while` عن الحلقات الأخرى هو كيف تتصرف عندما يفشل الشرط في أول تحقق. لنرَ هذا عمليًا. هنا حلقة `while` عادية ستشغل الشفرة داخل الحلقة طالما أن `i < 5`:

```js
const ourArray = []; 
let i = 5;

while (i < 5) {
  ourArray.push(i);
  i++;
}
```

في هذا المثال، نُهيئ قيمة `ourArray` إلى مصفوفة فارغة وقيمة `i` إلى 5. عند تنفيذ حلقة `while`، يُقيّم الشرط إلى `false` لأن `i` ليست أقل من 5، لذلك لا ننفذ الشفرة داخل الحلقة. النتيجة أن `ourArray` ستظل بدون قيم مضافة وستبدو كـ `[]` بعد انتهاء تنفيذ كل الشفرة في المثال أعلاه. الآن، ألقِ نظرة على حلقة `do...while`:

```js
const ourArray = []; 
let i = 5;

do {
  ourArray.push(i);
  i++;
} while (i < 5);
```

في هذه الحالة، نُهيئ قيمة `i` إلى 5، تمامًا كما فعلنا مع حلقة `while`. عندما نصل إلى السطر التالي، لا يوجد شرط للتقييم، لذلك ننتقل إلى الشفرة داخل الأقواس المتعرجة وننفذها. سنضيف عنصرًا واحدًا إلى المصفوفة ثم نزيد `i` قبل التحقق من الشرط. عندما نُقيّم الشرط `i < 5` في السطر الأخير، نرى أن `i` أصبحت 6، وهو ما يفشل التحقق الشرطي، لذا نخرج من الحلقة وننتهي. في نهاية المثال أعلاه، قيمة `ourArray` هي `[5]`. باختصار، تضمن حلقة `do...while` أن الشفرة داخل الحلقة ستُنفذ مرة واحدة على الأقل. دعنا نجرب تشغيل حلقة `do...while` بدفع قيم إلى مصفوفة.

# --instructions--

غيّر حلقة `while` في الشفرة إلى حلقة `do...while` بحيث تدفع الحلقة الرقم `10` فقط إلى `myArray`، وتصبح قيمة `i` مساوية لـ `11` عند انتهاء تنفيذ الشفرة.
