# --description--

$p_n$ を $n$ 番目の素数（2、3、5、7、11、...）とし、$r$ を ${(p_n−1)}^n + {(p_n+1)}^n$ を ${p_n}^2$ で割ったときの余りとします。

例えば、$n = 3$ のとき、$p_3 = 5$ で、$4^3 + 6^3 = 280 ≡ 5\\ mod\\ 25$ となります。

余りが初めて $10^9$ を超える最小の $n$ は 7037 です。

余りが初めて $10^{10}$ を超える最小の $n$ を求めてください。
