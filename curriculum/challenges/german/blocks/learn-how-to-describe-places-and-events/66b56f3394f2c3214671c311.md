# --explanation--

`To roll out` bedeutet, etwas Neues zu starten oder einzuführen, wie ein Produkt, eine Dienstleistung oder ein Projekt. Es wird oft im Geschäfts- und Technologiebereich verwendet. Zum Beispiel:

- `The company plans to roll out a new software update next month.` – Das Unternehmen wird das Update nächsten Monat einführen.

`By the end of` wird verwendet, um eine Frist festzulegen oder einen Zeitpunkt anzugeben, bis zu dem etwas abgeschlossen sein soll. Zum Beispiel:

- `I need to finish this report by the end of the day.` – Der Bericht sollte vor Tagesende fertiggestellt sein.
