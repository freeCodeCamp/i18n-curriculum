# --description--

제목은 다른 HTML 텍스트 요소와 다릅니다: 제목임을 나타내기 위해 다른 텍스트보다 더 크고 굵게 표시됩니다.

제목에는 `<h1>`부터 `<h6>`까지 6가지 다른 수준이 있습니다. 제목 태그 안의 숫자는 해당 제목의 수준을 나타냅니다. 가장 크고 중요한 제목은 `h1`이며, `h6`은 가장 작고 낮은 수준의 제목입니다.

제목은 단락과 비슷하게 정의합니다. 예를 들어, `h1` 제목을 만들려면 제목 텍스트를 `<h1>` 태그로 감싸면 됩니다.

적절한 수준의 제목을 사용하는 것이 중요한데, 수준이 콘텐츠에 계층 구조를 제공하기 때문입니다. 전체 페이지의 제목에는 항상 `h1` 제목을 사용해야 하며, 페이지의 작은 섹션에 대한 제목에는 더 낮은 수준의 제목을 사용해야 합니다.
