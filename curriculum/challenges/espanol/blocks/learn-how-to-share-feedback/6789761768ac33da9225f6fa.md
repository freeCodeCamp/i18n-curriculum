# --explanation--

`However` es una palabra de transición que se usa al comienzo de una oración para introducir una idea contrastante. Por ejemplo:

`The task seemed easy. However, it turned out to be quite challenging.` - Aquí, `however` contrasta la expectativa inicial con la realidad.

`Documentation` se refiere a registros escritos o digitales que explican o detallan un proceso, producto o sistema específico. Por ejemplo:

El equipo actualizó la `documentation` del proyecto para incluir las nuevas funciones.
