# --interactive--

In una lezione precedente, hai imparato come iniziare a lavorare con i gradienti lineari e radiali. In questa lezione, approfondiremo come usare le proprietà CSS per creare design più visivamente accattivanti e complessi.

I gradienti in CSS ti permettono di creare transizioni morbide tra due o più colori specificati. Sono spesso usati per aggiungere profondità, texture e interesse visivo ai design web senza bisogno di file immagine.

CSS supporta due tipi principali di gradienti: gradienti lineari e gradienti radiali.

I gradienti lineari creano una fusione graduale tra i colori lungo una linea retta. Puoi controllare la direzione di questa linea e i colori usati. Ecco un esempio base di un gradiente lineare:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

Questo crea un gradiente che passa dal rosso a sinistra al blu a destra. Il parametro `to right` specifica la direzione del gradiente. Puoi usare altre direzioni come `to left`, `to top`, `to bottom` o anche specificare angoli come `45deg`.

Puoi anche usare più fermate di colore in un gradiente lineare:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="multi-color-gradient"></div>
```

```css
.multi-color-gradient {
  background: linear-gradient(to right, red, yellow, blue);
  height: 40vh;
}
```

:::

Questo crea un gradiente che passa dal rosso al giallo al blu, distribuiti uniformemente sull’elemento.

I gradienti radiali, invece, creano gradienti circolari o ellittici che si irradiano da un punto centrale. Ecco un esempio:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::

Questo crea un gradiente circolare che inizia con il rosso al centro e sfuma verso il blu ai bordi. Puoi anche specificare la forma e la dimensione dei gradienti radiali:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="custom-radial-gradient"></div>
```

```css
.custom-radial-gradient {
  background: radial-gradient(ellipse at top left, red, blue);
  height: 40vh;
}
```

:::

Questo crea un gradiente ellittico che parte dall’angolo in alto a sinistra dell’elemento. Sia i gradienti lineari che quelli radiali ti permettono di specificare fermate di colore in posizioni precise. Per esempio:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="precise-gradient"></div>
```

```css
.precise-gradient {
  background: linear-gradient(to right, red 20%, yellow 40%, blue 80%);
  height: 40vh;
}
```

:::

Questo crea un gradiente dove il rosso occupa il primo `20%`, poi sfuma al giallo a `40%` e infine al blu a `80%`.

I gradienti possono essere usati ovunque useresti un’immagine in CSS. Questo li rende versatili per creare sfondi, pulsanti o anche effetti di testo quando combinati con altre proprietà CSS.

Ricorda, anche se i gradienti possono migliorare l’aspetto visivo del tuo sito, è importante garantire un contrasto sufficiente per l’accessibilità, specialmente quando usi gradienti dietro al testo.
