# --interactive--

程式設計中的迴圈用於重複執行一個區塊的程式碼多次。

迴圈的一個範例是當你設計一個需要列印項目列表的程式時。你可以使用迴圈來列印列表中的每一個項目。

另一個範例是當你在設計一個遊戲並且想要將一個字元移動到螢幕上。你可以使用一個迴圈，在每次迴圈執行時將字元移動一定數量的像素。 

在 JavaScript 中，有幾種你可以使用的迴圈型別。在本課程中，我們將涵蓋 `for` 迴圈。以下是 `for` 迴圈的基本語法：

```js
for (initialization; condition; increment or decrement) {
  // code block to be executed
}
```

初始化述語會在迴圈開始前執行。它通常用來初始化計數器變數。計數器變數是用來追蹤迴圈已執行多少次的變數。

條件述語會在每次迴圈的迭代之前被評估。一次迭代是迴圈的一次單獨執行。

如果條件為真，則執行迴圈內的程式碼區塊。如果條件為假，迴圈停止，並繼續執行下一個程式碼區塊。

迴圈的最後一部分是遞增／遞減述語。此述語會在每次迴圈執行後執行。它通常用於遞增或遞減計數器變數。

:::interactive_editor

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

:::

在上述範例的第一部分，我們將計數變數 `i` 初始化為 `0`。在 `for` 迴圈中使用 `i` 作為計數變數是常見的慣例。

下一個部分是檢查條件。在這個案例中，條件是檢查 `i` 是否小於 `5`。因為 `i` 是 `0`，條件為真，迴圈內的程式碼區塊會被執行。

迴圈內的程式碼區塊是將 `i` 的值記錄到主控台。`i` 的值是 `0`，所以主控台會顯示 `0` 的值。

然後執行遞增述語。在這種情況下，我們將 `i` 遞增 `1`。所以 `i` 現在是 `1`。

然後我們再次檢查條件，也就是檢查 `i` 是否小於 `5`。由於 `i` 現在是 `1`，條件仍然為真，迴圈內的程式碼區塊再次被執行。

我們持續重複這個處理程序，直到條件為假。在這個例子中，當 `i` 是 `5` 時，條件為假，迴圈停止。

當你使用迴圈時，應該小心不要產生一個永遠為真的條件。如果你這麼做，迴圈將會無限執行，而你的程式將會當機。這稱為無窮迴圈。

可以創建巢狀的 `for` 迴圈。巢狀迴圈是指你將一個迴圈放在另一個迴圈內。我們稍後會看到你可能想這麼做的範例。

當你需要重複執行某個區塊的程式碼特定次數時，迴圈在程式設計中會很有幫助。即使一開始使用 `for` 迴圈可能會有點困難，但經過練習你會漸漸掌握它。
