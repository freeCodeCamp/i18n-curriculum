# --description--

У цьому розділі потрібно вказати контактну інформацію автора блогу. Для цього ви використовуватимете елемент <dfn>address</dfn>.

Елемент `address` використовують, щоб вказати контактну інформацію особи чи організації.

Ось приклад використання елемента `address` для фізичної адреси. Елемент `br` використано, щоб розірвати рядки тексту.

```html
<address>
  1234 Make Believe Lane <br />
  Pretend City, USA
</address>
```

Додайте елемент `address` під елементом `h2`.

# --hints--

Код повинен містити початковий тег `address`.

```js
assert.match(code, /<address>/i);
```

Код повинен містити кінцевий тег `address`.

```js
assert.match(code, /<\/address>/i);
```

Елемент `address` має бути в межах елемента `section`.

```js
const addressElement = document.querySelector('footer #contact address');
assert.strictEqual(addressElement?.parentElement.tagName, 'SECTION');
```
