# --description--

Il primo concetto importante che devi capire per avere successo con il CSS è il modello a riquadro. Non è complicato, ma saltarlo ora ti causerà molta frustrazione in futuro.

Ogni singola cosa su una pagina web è un riquadro rettangolare. Questi riquadri possono contenere altri riquadri e possono stare uno accanto all’altro. Puoi farti un’idea approssimativa di come funziona mettendo un bordo su ogni elemento della pagina così:

```css
* {
  border: 1px solid red;
}
```

# --assignment--

Aggiungi un bordo a ogni elemento della pagina e osserva come sono disposti i riquadri.
