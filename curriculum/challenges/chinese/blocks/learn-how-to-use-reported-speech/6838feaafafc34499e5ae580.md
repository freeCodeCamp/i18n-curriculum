# --explanation--

`Past Continuous` 时态（`was/were` + 动词的 `-ing` 形式）用于描述过去一段时间内发生的动作或想法，通常与另一个事件同时发生。它可以表示某人已经在思考同一个想法。例如：

`I was reading the report when you called.` – 这意味着在调用发生时，读取动作正在进行中。

在这个分支中，`I was thinking the same thing` 意味着 Mark 的想法发生在 Lisa 的评论期间或之前，并表示同意。
