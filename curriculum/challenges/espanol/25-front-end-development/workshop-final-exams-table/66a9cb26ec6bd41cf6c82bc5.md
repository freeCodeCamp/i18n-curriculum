# --description--

La última sección a añadir a la tabla sería el elemento <dfn>table foot</dfn>, `tfoot`. El elemento de pie de tabla se utilizará para mostrar la calificación promedio de todos los estudiantes en la tabla.

Añadir un elemento `tfoot` a la tabla.

Dentro del elemento `tfoot`, añadir un elemento `tr`.

Dentro del elemento `tr`, añadir dos elementos `td`.

El primer elemento `td` debe contener el texto `Average Grade`. El segundo elemento `td` debe contener el texto `78.8`.
