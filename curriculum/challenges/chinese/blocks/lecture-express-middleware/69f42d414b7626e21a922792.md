# --description--

应用级中间件是绑定到 Express 应用实例的中间件。它通常用于处理需要跨多个路由处理的请求。它可以全局应用，也可以应用于特定路由。

这种中间件通过 `app.use()` 或特定路由方法如 `app.get()`、`app.post()` 等注册。除非限制在特定路由，否则它会对每个传入请求执行。

```js
const express = require('express')
const app = express()

// Application-level middleware that logs all incoming requests
app.use((req, res, next) => {
  console.log('Request received at:', new Date())
  next(); // Passes control to the next middleware
})

app.get('/', (req, res) => {
  res.send('Hello World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* `app.use()`：这是应用级中间件。它记录对服务器发出的每个请求，无论路由如何。
* 该中间件对每个传入请求运行，向服务器提供当前时间戳等详细信息。

应用级中间件适用于需要在应用中许多或所有路由上执行的任务。这可能包括日志记录、认证和误差处理。
