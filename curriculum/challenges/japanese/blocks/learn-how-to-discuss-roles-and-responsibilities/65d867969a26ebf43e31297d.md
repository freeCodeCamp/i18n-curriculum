# --description--

`monitor` とは、何かが正しく機能しているか、安全が保たれているかを確認するために、一定期間にわたって見守りチェックすることを意味します。例えば、`The security team monitors the cameras to keep the building safe.`

`Access control` は、場所への出入りや資源の使用を制限・管理する方法を指し、許可された人だけが場所に入ったり何かを使ったりできるようにします。例えば、`The company uses access controls to restrict entry to the server room.`
