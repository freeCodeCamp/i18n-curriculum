# --description--

To `automate` means to make a process operate by itself without needing a person to help. En programación, puede implicar el uso de software para escribir código, probar aplicaciones o encontrar y corregir errores. For example, a coffee machine that turns on by itself in the morning to make coffee is `automated`.

La palabra `certain` se utiliza para hablar de cosas o personas específicas, pero no de todas las cosas o personas. For instance, if you say `certain books are on sale`, it means only some books are on sale, not all of them.
