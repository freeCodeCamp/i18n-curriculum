# --description--

Una fracción decimal irracional se crea concatenando los enteros positivos:

0.12345678910**1**112131415161718192021...

Se puede ver que el 12<sup>th</sup> dígito de la parte fraccionaria es 1.

Si *d<sub>n</sub>* representa el *n*<sup>ésimo</sup> dígito de la parte fraccionaria, encuentra el valor de la siguiente expresión.

d<sub>1</sub> × d<sub>10</sub> × d<sub>100</sub> × d<sub>1000</sub> × d<sub>10000</sub> × d<sub>100000</sub> × d<sub>1000000</sub>

```js
function champernownesConstant(n) {
  let fractionalPart = '';
  for (let i = 0; fractionalPart.length <= n; i++) {
    fractionalPart += i.toString();
  }

  let product = 1;
  for (let i = 0; i < n.toString().length; i++) {
    const index = 10 ** i;
    product *= parseInt(fractionalPart[index], 10);
  }

  return product;
}
```
