# --interactive--

Sifa ya CSS `filter` ni chombo chenye nguvu kinachokuwezesha kutumia athari za picha kwa vipengele kwenye ukurasa wa mtandao. Ni muhimu hasa kwa marekebisho ya uwasilishaji wa picha, mandhari, na hata maandishi bila kubadilisha mali halisi.

Sifa ya `filter` inaweza kutumika kuunda athari mbalimbali, kama vile kufifisha, kubadilisha rangi, na marekebisho ya mng’ao. Hebu tujadili jinsi sifa ya filter inavyofanya kazi na kuchunguza baadhi ya mifano ya kawaida. Sintaksia ya msingi kwa sifa ya `filter` ni rahisi:

```css
selector {
  filter: function(amount);
}
```

Hapa, `function` inawakilisha athari maalum ya filter unayotaka kutumia, na `amount` kawaida ni thamani inayobainisha ukali wa athari hiyo. Sasa tuchunguze baadhi ya kitendakazi cha filter na matumizi yake ya kawaida.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: blur(2px);
}
```

:::

Kitendakazi cha `blur` hufanya blur ya Gaussian kwa kipengele, kiasi chake kimebainishwa kwa pikseli na kinawakilisha mduara wa blur. Sheria hii ya CSS itatumia blur ya pikseli 2 kwa picha zote kwenye ukurasa. Athari ya `blur` inaweza kuwa ya msaada kuunda kina katika muundo wako au kuficha sehemu za picha.

Kitendakazi cha `brightness` kinarekebisha mng’ao wa kipengele. Thamani ya `0%` itafanya kipengele kuwa giza kabisa, wakati thamani zaidi ya `100%` itaongeza mng’ao.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: brightness(150%);
}
```

:::

Sheria hii ya CSS inaongeza mng’ao wa kipengele cha picha kwa `50%`. Marekebisho ya mng’ao yanaweza kutumika kufanya picha zionekane zaidi au kuunda athari ya kung’aa kidogo.

Kitendakazi cha `grayscale` kinabadilisha kipengele kuwa rangi za kijivu (grayscale). Kiasi chake kimebainishwa kwa asilimia, ambapo `100%` ni rangi za kijivu kabisa na `0%` hutoa picha isiyobadilika.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: grayscale(100%);
}
```

:::

Sheria hii itabadilisha kipengele cha picha kuwa rangi za kijivu kabisa. `grayscale` inaweza kutumika kuunda muonekano wa zamani au kupunguza umuhimu wa vipengele fulani kwenye ukurasa.

Kitendakazi cha `sepia` hutoa athari ya rangi ya sepia kwa kipengele. Kama grayscale, hutumia thamani ya asilimia:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: sepia(80%);
}
```

:::

Sheria hii inatumia athari ya sepia ya `80%` kwa kipengele cha picha. Athari ya sepia ni nzuri kwa kuunda muonekano wa zamani au wa kale.

Kitendakazi cha `hue-rotate` hutoa mzunguko wa rangi (hue rotation) kwa kipengele. Thamani yake ni kwa digrii, na inawakilisha mzunguko kuzunguka mzunguko wa rangi.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: hue-rotate(90deg);
}
```

:::

Sheria hii inazungusha rangi ya kipengele cha picha kwa digrii `90`. Mzunguko wa rangi unaweza kutumika kuunda athari za psychedelic au kurekebisha mpangilio wa jumla wa rangi za picha.

Moja ya vipengele vyenye nguvu zaidi vya sifa ya `filter` ni uwezo wa kuunganisha athari nyingi. Unaweza kutumia filter nyingi kwa kipengele kimoja kwa kuzitenganisha kwa nafasi:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {  
  filter: contrast(150%) brightness(110%) sepia(30%);  
}
```

:::

Sheria hii inatumia ongezeko la mng’ao, ongezeko kidogo la mng’ao, na athari ya sepia nyepesi kwa kipengele cha picha.

Kwa kuunganisha filter, unaweza kuunda athari za kipekee na tata za kuona zinazolingana na mahitaji ya muundo wako. Sifa ya CSS filter ni chombo chenye matumizi mengi kinachoruhusu udhibiti wa ubunifu wa vipengele vya mtandao.

Ingawa tumeshughulikia baadhi ya kitendakazi maarufu za filter, kuna zingine zinazopatikana, kama `contrast`, `invert`, na `saturate`. Kama kipengele chochote chenye nguvu, ni muhimu kuwa makini jinsi unavyotumia filter ili kuboresha muundo wako bila kumzidi mtumizi au kuathiri ufikikaji.
