# --explanation--

`On vacation` は、通常は休息や旅行のために一定期間働いていないことを意味します。例えば：

`She's on vacation until next Monday.` – これは彼女が今働いていないことを意味します。

`In July` は、1年の7番目の月に起こることを指します。前置詞 `in` は、何かがいつ起こるかを示すために月の前に使われます。例えば：

`We have a big release scheduled in July.` – これはリリースがその月のいつかに行われることを意味します。
