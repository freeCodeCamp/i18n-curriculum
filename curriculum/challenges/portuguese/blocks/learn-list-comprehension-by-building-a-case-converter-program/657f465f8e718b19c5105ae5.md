# --description--

List comprehensions aceitam declarações condicionais, para avaliar a expressão fornecida somente se certas condições forem atendidas:

```py
spam = [i * 2 for i in iterable if i > 0]
```

Como você pode ver na saída, a lista de caracteres gerada a partir de `pascal_or_camel_cased_string` foi unida.Como a expressão dentro da list comprehension é avaliada para cada caractere, o resultado é uma string em minúsculas com todos os caracteres separados por um underscore.

Siga o exemplo acima para adicionar uma cláusula `if` à sua list comprehension para que a expressão seja executada somente se o caractere for maiúsculo.
