# --description--

사용자 개인정보는 개인 데이터가 온라인에서 어떻게 수집되고 사용되며 공유되는지를 통제할 권리에 관한 것입니다. 현대 개인정보 프레임워크는 디지털 흔적을 보호하기 위해 여러 핵심 요소에 의존합니다.

개인정보 보호의 핵심은 **개인 식별 정보(PII)**입니다. PII는 직접적이거나 간접적으로 신원을 특정할 수 있는 모든 데이터를 의미합니다.

직접 식별자는 전체 이름, 집 주소, 주민등록번호 등이 포함되며, 간접 식별자는 IP 주소, 기기 일련번호, 정확한 위치 기록 등이 포함됩니다.

이 데이터를 안전하게 유지하려면 기업은 **기밀성**을 유지해야 합니다. 기밀성은 강력한 암호화와 접근 통제를 통해 무단 접근으로부터 개인 데이터를 안전하게 보호합니다.

무단 제3자가 기록을 읽거나 PII를 유출하면 기밀성이 깨진 것입니다.

개인정보 프레임워크는 이를 **데이터 최소화**와 **목적 제한**을 통해 시행합니다. 데이터 최소화는 기업이 특정 작업을 완료하는 데 필요한 최소한의 PII만 수집해야 한다고 규정합니다.

목적 제한은 기업이 특정 이유로 데이터를 수집한 후에는 광고 네트워크에 연락처 정보를 판매하는 등 다른 용도로 법적으로 사용할 수 없다고 규정합니다.

이 원칙들은 침해적인 디지털 추적에 직접 대응합니다. **추적**은 인터넷 전반에서 온라인 움직임을 지속적으로 감시하는 것입니다.

광고 네트워크는 스크립트, 기기 지문 인식, 브라우저 기록 로그를 사용해 사용자의 행동을 추적하고 상세한 프로필을 작성합니다.

이 추적을 막으려면 사용자 동의가 개인 통제의 기반이 됩니다. 진정한 개인정보 보호는 정보에 기반한 명시적 동의를 요구하며, 사용자가 강제로 하는 것이 아니라 적극적으로 데이터 추적에 참여해야 합니다.

마지막으로, **개인정보 보호법은 잊힐 권리**를 보장합니다. 이는 계정을 닫으면 조직에 개인 기록과 추적 기록을 데이터베이스에서 영구 삭제해 달라고 법적으로 요청할 권리를 의미합니다.

다음 강의에서 이러한 개인정보 보호법과 관련 기관에 대해 배우게 됩니다.
