# --description--

Jumla ya mraba za nambari asilia kumi za kwanza ni,

<div style='text-align: center;'>1<sup>2</sup> + 2<sup>2</sup> + ... + 10<sup>2</sup> = 385</div>

Mraba wa jumla ya nambari asilia kumi za kwanza ni,

<div style='text-align: center;'>(1 + 2 + ... + 10)<sup>2</sup> = 55<sup>2</sup> = 3025</div>

Kwa hiyo tofauti kati ya jumla ya mraba za nambari asilia kumi za kwanza na mraba wa jumla ni 3025 − 385 = 2640.

Tafuta tofauti kati ya jumla ya mraba za nambari asilia `n` za kwanza na mraba wa jumla.

const sumSquareDifference = (number)=>{
  let squareOfSum = Math.pow(sumOfArithmeticSeries(1,1,number),2);
  let sumOfSquare = sumOfSquareOfNumbers(number);
 return squareOfSum - sumOfSquare;
}

function sumOfArithmeticSeries(a,d,n){
  return (n/2)*(2*a+(n-1)*d);
}

function sumOfSquareOfNumbers(n){
 return (n*(n+1)*(2*n+1))/6;
}
