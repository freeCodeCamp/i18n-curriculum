## --text--

Wann wird eine `ArgumentOutOfRangeException`-Ausnahme geworfen?

## --answers--

Eine `ArgumentOutOfRangeException`-Ausnahme wird geworfen, wenn versucht wird, auf einen Index eines Arrays außerhalb der Grenzen des Arrays zuzugreifen.

---

Eine `ArgumentOutOfRangeException`-Ausnahme wird geworfen, wenn der Wert eines Arguments außerhalb des zulässigen Wertebereichs liegt, wie von der Methode definiert.

---

Eine `ArgumentOutOfRangeException`-Ausnahme wird geworfen, wenn versucht wird, einen Wert eines Typs in einem Array eines anderen Typs zu speichern.
