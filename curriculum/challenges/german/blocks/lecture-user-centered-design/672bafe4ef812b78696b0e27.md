# --interactive--

**HINWEIS**: Einige der interaktiven Beispiele verwenden möglicherweise CSS, das Sie noch nicht gelernt haben. Machen Sie sich keine Sorgen, wenn Sie nicht den gesamten Code verstehen. Das Ziel der Beispiele ist es, Ihnen Vorschauen zu diesen Designkonzepten zu zeigen, damit Sie besser verstehen, wie alles funktioniert.

Dark Mode ist eine besondere Funktion in Web-Anwendungen, bei der Sie das standardmäßige helle Farbschema in ein dunkles Farbschema ändern können. Dies hilft, die Augenbelastung zu reduzieren und die Lesbarkeit bei schlechten Lichtverhältnissen zu verbessern. Beim Gestalten Ihrer Dark-Mode-Funktionen ist es wichtig, bewährte Vorgehensweisen zu kennen, um sicherzustellen, dass Ihre Dark-Mode-Funktion effektiv und benutzerfreundlich ist.

Aktivieren Sie den interaktiven Editor und klicken Sie auf die `Toggle Dark Mode`-Schaltfläche im folgenden Beispiel, um zu sehen, wie Dark Mode funktioniert.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<button id="theme-toggle">Toggle Dark Mode</button>

<div class="content">
  <p>This is a simple dark mode example.</p>
</div>

<script src="index.js"></script>
```

```css
body {
  background: #f5f5f5;
  color: #222;
  font-family: sans-serif;
  transition: background 0.3s, color 0.3s;
}

button {
  margin: 1rem;
  padding: 0.5rem 1rem;
  cursor: pointer;
}

body.dark {
  background: #121212;
  color: #e0e0e0;
}
```

```js
document.getElementById('theme-toggle').addEventListener('click', () => {
  document.body.classList.toggle('dark');
});
```

:::

Die erste Überlegung ist die Vermeidung von gesättigten Farben im Dark Mode. Gesättigte Farben sind Farben, die hell und intensiv sind. Zum Beispiel kann ein helles Magenta-Button vor einem dunkelgrauen Hintergrund zu intensiv sein und die Augen belasten. Stattdessen sollten Sie im Dark Mode entsättigte Farben verwenden. Entsättigte Farben sind Farben, die weniger intensiv sind, einen niedrigeren Sättigungsgrad haben und im Dark Mode angenehmer anzusehen sind. Um die Vorschauen zu sehen, müssen Sie den interaktiven Editor aktivieren.

Eine weitere Überlegung beim Dark Mode ist die Verwendung von rein schwarzen Hintergründen mit weißem Text. Obwohl dieser hohe Kontrast effektiv sein kann, kann er auch zu hart für die Augen sein. Stattdessen sollten Sie einen dunkelgrauen Hintergrund mit hellgrauem Text für einen weicheren Kontrast in Betracht ziehen. Der Text wird so augenfreundlicher und angenehmer zu lesen sein im Dark Mode. Um die Vorschauen zu sehen, müssen Sie den interaktiven Editor aktivieren.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<button id="theme-toggle">Toggle Dark Mode</button>

<div class="content">
  <h1>Dark Mode Contrast Comparison</h1>
  <p>Compare the two sections below. One uses pure black and white, the other uses dark gray and light gray for better readability.</p>

  <div class="dark-mode-examples">
    <div class="box harsh-dark">
      <h2>High Contrast</h2>
      <p>Pure black background with white text. While readable, it can be harsh on the eyes.</p>
    </div>

    <div class="box soft-dark">
      <h2>Soft Contrast</h2>
      <p>Dark gray background with light gray text. Easier to read and more comfortable for long periods.</p>
    </div>
  </div>
</div>

<script src="index.js"></script>
```

```css
}

  background-color: #ffffff;
  color: #111;

}

  background-color: #000000; 
  color: #ffffff;           
}

body.dark .box.soft-dark {
  background-color: #1e1e1e; 
  color: #cccccc;           
}

  background-color: #121212;
  color: #e0e0e0;
}
```

Eine weitere Überlegung ist die Verwendung des Dark Mode in Verbindung mit der Markenidentität der Seite. Eine Markenidentität ist eine Reihe visueller Elemente, die eine Marke darstellen, wie das Logo, Farben und Typografie. Beim Implementieren des Dark Mode sollten Sie berücksichtigen, wie die Dark-Mode-Funktion mit den Farben und dem Stil Ihrer Marke übereinstimmt. Es ist in Ordnung, das Marken-Icon und die Buttons in voller Sättigung zu haben, während die umliegenden Elemente entsättigt sind.

Generell sollten Sie beim Design immer die Benutzererfahrung und die Kontrastwerte im Blick behalten. Dark Mode bildet da keine Ausnahme, und wenn Sie diese bewährten Vorgehensweisen befolgen, können Sie eine Dark-Mode-Funktion erstellen, die effektiv und benutzerfreundlich ist.
