# --explanation--

`Miss` 意味着未能出席或参与某事。例如： 

`I missed the meeting because I was stuck in traffic.`——这意味着你在交通中花费了太多时间，导致上班迟到。 

`design sprint` 是一个有时间限制的协作进程，团队在短时间内（通常为几天）致力于解决设计问题或开发原型。例如： 

团队完成了一个设计冲刺以测试 `new user interface concept`。这意味着团队参与了这个协作进程来测试该概念。 

在这个上下文中，Maria 无法参加网络安全性研讨会，因为她正在积极参与设计冲刺研讨会。
