# --description--

$k$자리(십진수)인 양의 정수를 균형 숫자라고 합니다. 균형 숫자는 처음 $⌈\frac{k}{2}⌉$ 자리 숫자의 합이 마지막 $⌈\frac{k}{2}⌉$ 자리 숫자의 합과 같을 때를 말합니다. 여기서 $⌈x⌉$는 천장 함수로, $x$ 이상인 가장 작은 정수를 의미합니다. 예를 들어 $⌈π⌉ = 4$이고 $⌈5⌉ = 5$입니다.

예를 들어, 모든 팰린드롬 수는 균형 숫자이며, 13722도 균형 숫자입니다.

$T(n)$을 $10^n$보다 작은 모든 균형 숫자의 합이라고 합시다.

따라서: $T(1) = 45$, $T(2) = 540$, 그리고 $T(5) = 334\\,795\\,890$입니다.

$T(47)\\,mod\\,3^{15}$ 값을 구하세요.
