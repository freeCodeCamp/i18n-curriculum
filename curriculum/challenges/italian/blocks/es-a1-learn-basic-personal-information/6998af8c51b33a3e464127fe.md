# --explanation--

`Navarro con dos erres.` segue la struttura: `con` + numero + sostantivo plurale.

`Con` aggiunge informazioni sull'ortografia.

`Dos` indica la quantità.

`Erres` si riferisce a due lettere `r`.

Questa struttura è comune quando si chiarisce come si scrive una parola.
