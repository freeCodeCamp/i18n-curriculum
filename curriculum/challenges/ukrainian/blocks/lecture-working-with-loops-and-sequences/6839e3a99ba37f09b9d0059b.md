# --description--

Кортеж — це тип даних у Python, який використовується для створення впорядкованої послідовності значень. Кортежі можуть містити змішаний набір типів даних, як-от:

```py
developer = ('Alice', 34, 'Rust Developer')
```

Кортежі схожі на списки, але якщо списки є змінним типом даних, то кортежі — незмінні. Це означає, що елементи в кортежі не можна змінити після його створення.

Якщо ви спробуєте оновити один із елементів кортежу, ви отримаєте `TypeError`:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: 'tuple' object does not support item assignment
"""
```

Щоб отримати доступ до елемента кортежу, ви можете використати дужкову нотацію та номер індексу:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

Якщо потрібно отримати доступ до елементів, починаючи з кінця кортежу, можна використати від’ємну індексацію. Ось приклад використання від’ємного індексу для доступу до передостаннього елемента кортежу:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

Якщо ви спробуєте передати номер індексу, який перевищує або дорівнює довжині кортежу, ви отримаєте `IndexError` такого вигляду:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list index out of range
"""
```

Інший спосіб створити кортеж — використати конструктор `tuple()` ось так:

```py
developer = 'Jessica'
tuple(developer) # ('J', 'e', 's', 's', 'i', 'c', 'a')
```

Для конструктора `tuple()` можна передавати різні ітеровані об’єкти, такі як рядки, списки і навіть інші кортежі.

Щоб перевірити, чи є елемент у кортежі, можна використати ключове слово `in` ось так:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

Ви також можете розпакувати елементи з кортежу так само, як і зі списків:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

У цьому прикладі `name` має значення `'Alice'`, `age` має значення `34`, а `job` має значення `'Rust Developer'`.

Якщо потрібно зібрати всі залишкові елементи з кортежу, можна використати оператор зірочки (`*`) ось так:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

Тут `name` має значення `'Alice'`, а `rest` — це список, що складається з числа `34` і рядка `'Rust Developer'`.

Як і зі списком, ви можете використати оператор зрізу для кортежу, щоб витягти його частину. Ось приклад витягування елементів `'pie'` і `'cookies'` у окремий кортеж:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

Пам’ятайте, що перше число позначає початковий індекс для витягування, а друге — кінцевий індекс. Але зверніть увагу, що елемент із кінцевим індексом не включається до витягнутого кортежу.

Якщо потрібно видалити елемент із кортежу, це неможливо, бо кортежі незмінні. Тож цей приклад викличе помилку:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

Отже, коли варто використовувати кортеж замість списку?

Якщо вам потрібна динамічна колекція елементів, де можна додавати, видаляти та оновлювати елементи, слід використовувати список. Якщо ви знаєте, що працюєте з фіксованою та незмінною колекцією даних, тоді слід використовувати кортеж.

У наступному уроці ми розглянемо деякі загальні методи, які ви будете використовувати при роботі з кортежами.
