# --description--

Ein HTML-Server hat normalerweise ein oder mehrere Verzeichnisse, auf die der Benutzer zugreifen kann. Dort können Sie die statischen Assets ablegen, die Ihre Anwendung benötigt (Stylesheets, Skripte, Bilder).

In Express können Sie diese Funktionalität mit der Middleware `express.static(path)` einrichten, wobei der `path`-Parameter der absolute Pfad des Ordners ist, der die Assets enthält.

Wenn Sie nicht wissen, was Middleware ist... keine Sorge, wir werden das später ausführlich besprechen. Grundsätzlich sind Middleware Funktionen, die Route-Handler abfangen und zusätzliche Informationen hinzufügen. Eine Middleware muss mit der Methode `app.use(path, middlewareFunction)` eingebunden werden. Das erste `path`-Argument ist optional. Wenn Sie es nicht angeben, wird die Middleware für alle Anfragen ausgeführt.

# --instructions--

Binden Sie die Middleware `express.static()` an den Pfad `/public` mit `app.use()` ein. Der absolute Pfad zum Assets-Ordner ist `__dirname + /public`.

Jetzt sollte Ihre Anwendung in der Lage sein, ein CSS-Stylesheet bereitzustellen. Beachten Sie, dass die `/public/style.css`-Datei im `/views/index.html` im Boilerplate-Code des Projekts referenziert wird. Ihre Startseite sollte jetzt etwas besser aussehen!

# --hints--

Ihre Anwendung sollte Asset-Dateien aus dem Verzeichnis `/public` unter dem Pfad `/public` bereitstellen

```js
  const response = await fetch(code + '/public/style.css');
  if (!response.ok) {
    throw new Error(await response.text());
  }
  const data = await response.text();
  assert.match(
    data,
    /body\s*\{[^\}]*\}/,
    'Your app does not serve static assets'
  );
```

Ihre Anwendung sollte keine Dateien aus anderen Ordnern als dem Verzeichnis `/public` bereitstellen

```js
  const response = await fetch(code + '/server.js');
  if (response.ok) {
    const data = await response.text();
    assert.equal(
      response.status + '',
      404 + '',
      'Your app must serve files only from "public" directory'
    );
  } else {
    assert.equal(
      response.status + '',
      404 + '',
      'Your app must serve files only from "public" directory'
    );
  }
```
