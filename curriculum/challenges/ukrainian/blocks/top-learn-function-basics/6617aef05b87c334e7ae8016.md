# --description--

Як ви вже бачили раніше, функції можуть повертати значення за допомогою ключового слова `return`. Ключове слово `return` використовується для повернення значення з функції. Коли використовується ключове слово `return`, функція припиняє виконання і повертає значення, вказане після ключового слова `return`.

```js
function add(a, b) {
  return a + b
}

console.log(add(2, 3)); // Output: 5
```

Але що станеться, якщо ключове слово `return` буде використане до кінця функції?
Щоб відповісти на це питання, розгляньте наступний приклад:

```js
function add(a, b) {
  if(a > 2){
    return b;
  }

  return a + b;
}

console.log(add(3, 7)); // Output: 7
``` 

У наведеному прикладі функція `add` має умовну інструкцію, яка перевіряє, чи є значення `a` більшим за `2`. Якщо умова виконується, функція поверне значення `b` і припинить виконання. Якщо умова не виконується, функція поверне суму `a` і `b`.
