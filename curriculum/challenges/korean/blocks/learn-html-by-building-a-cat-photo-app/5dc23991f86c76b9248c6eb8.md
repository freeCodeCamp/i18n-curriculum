# --description--

이전 단계에서 `h1`, `h2`, 주석, 그리고 `p` 요소를 `main` 요소 안에 넣었습니다. 이것을 *중첩*이라고 합니다. 중첩된 요소는 중첩된 요소가 속한 요소보다 오른쪽으로 두 칸 더 들여쓰기를 해야 합니다. 이 들여쓰기를 인덴테이션이라고 하며, HTML을 더 읽기 쉽게 만드는 데 사용합니다.

다음은 중첩과 인덴테이션의 예입니다:

```html
<main>
  <h1>Most important content of the document</h1>
  <p>Some more important content...</p>
</main>
```

아래 코드에서 `h1` 요소, `h2` 요소, 그리고 주석은 `main` 요소보다 두 칸 더 들여쓰기 되어 있습니다. 키보드의 스페이스 바를 사용해 `p` 요소 앞에 두 칸을 더 추가하여 올바르게 들여쓰기되도록 하세요.
