# --description--

인터넷에서 정보를 검색하는 일은 벅찰 수 있습니다. 너무 많은 웹사이트와 방대한 데이터가 있어서 원하는 결과를 얻는 것이 불가능해 보일 수도 있습니다.

하지만 불리언 검색 구문 같은 것을 활용하면 필요한 것만 정확히 좁혀서 검색할 수 있습니다.

불리언 검색 구문은 온라인에서 검색할 때 원하는 것을 정확히 찾도록 도와주는 특별한 단어를 사용하는 방법입니다.

여기 `freecodecamp curriculum`에 대한 기본 검색 쿼리 예시가 있습니다.

`freecodecamp curriculum`을 검색하면, 검색 엔진은 이 키워드들 중 *하나라도* 일치하는 결과를 반환합니다 – 보통 관련성(쿼리와 얼마나 일치하거나 관련 있는지)에 따라 정렬됩니다.

그런데 만약 두 키워드 모두 일치하는 결과만 원한다면 어떻게 해야 할까요? 쿼리를 따옴표로 감싸면 된다는 말을 들어봤을 수도 있습니다, 예를 들어 `"freecodecamp curriculum"`처럼요.

하지만 이 쿼리는 *구문* `freecodecamp curriculum`과 정확히 일치하는 결과만 찾습니다. 즉, 결과는 `freecodecamp curriculum`가 함께 나오는 사이트로 제한되며, `freecodecamp`와 `curriculum`가 각각 포함된 사이트를 찾는 것이 아닙니다.

두 용어가 모두 포함된 사이트를 찾고 싶지만 반드시 구문처럼 붙어 있을 필요는 없다면, 각 용어 앞에 더하기 기호를 붙일 수 있습니다: `+freecodecamp +curriculum`

또는 freeCodeCamp의 커리큘럼 외 활동에 관심이 있을 수도 있습니다. 이 경우 `curriculum` 앞에 빼기 기호를 붙여서 이렇게 검색할 수 있습니다: `+freecodecamp -curriculum`.

이렇게 하면 `freecodecamp`을 언급하는 결과는 나오지만 `curriculum`도 언급하는 결과는 제외됩니다.

특정 사이트에서만 검색하고 싶다면, 예를 들어 우리 뉴스 플랫폼에서라면 `site:` 다음에 검색할 사이트 URL을 붙여서 이렇게 검색할 수 있습니다: `site:freecodecamp.org/news curriculum`.

이 쿼리는 `freecodecamp.org/news`에서 `curriculum`라는 단어가 포함된 모든 결과를 반환합니다.

이 도구들을 조합하면 아주 구체적인 쿼리를 만들어 정확히 필요한 정보를 얻을 수 있습니다.

예를 들어, 쿼리 `site:freecodecamp.org/news +python -curriculum`는 우리 뉴스 사이트에서 Python 관련 기사를 검색하되, 커리큘럼 관련 기사는 제외합니다.

검색 연산자는 다를 수 있으니, 선호하는 검색 엔진의 문서를 항상 확인하세요.

하지만 이제 웹 검색에서 원하는 정확한 결과를 얻을 수 있는 능력을 갖추셨습니다.
