# --description--

Wenn Sie über Gegenstände oder Dinge an einem bestimmten Ort sprechen, verwenden Sie oft das Wort `that`, um auf sie hinzuweisen. Zum Beispiel bedeutet `that shelf` das Regal, über das Sie gerade sprechen oder auf das Sie sich gerade konzentrieren.

Das Wort `shelf` ist ein Substantiv, das sich auf eine flache, horizontale Fläche bezieht, die zum Halten von Gegenständen verwendet wird. Wenn Sie über mehr als ein `shelf` sprechen, sollten Sie die Endung ändern, um den Plural zu bilden, indem Sie `-f` durch `-ves` ersetzen, sodass es `shelves` wird. Zum Beispiel `one shelf`, aber `two shelves`.
