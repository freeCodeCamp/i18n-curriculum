# --description--

このラボでは、特定のテスト関数に基づいて許容される要素が見つかるまで配列の要素をスキップする関数を作成します。

例えば、`[1, 1, 1, 2, 1, 1, 1]` のような配列と `function(n) {return n === 2}` のテスト関数がある場合、この関数で許容される最初の要素はインデックス3の要素なので、それ以前のすべての要素は破棄され、出力は残りの要素 `[2, 1, 1, 1]` になります。

**目的**：以下のユーザーストーリーを満たし、すべてのテストに合格してラボを完了してください。

**ユーザーストーリー**

1. `dropElements` という関数があり、配列（`arr`）と関数（`func`）を引数として受け取る必要があります。
1. `dropElements` 関数は配列を繰り返し処理し、`func` が要素に対して `true` を返すまで最初の要素から順に要素を削除する必要があります。
1. `dropElements` 関数は条件が満たされた場合、配列の残りの要素を返す必要があります。
1. 条件が一度も満たされなかった場合は、空の配列を返す必要があります。

# --solutions--

```js
function dropElements(arr, func) {
  while (arr.length > 0 && !func(arr[0])) {
    arr.shift();
  }
  return arr;
}
```
