# --description--

Наступна гра є класичним прикладом комбінаторної теорії ігор:

Двоє гравців починають зі стрічки з $n$ білих квадратів і ходять по черзі. Кожного ходу гравець вибирає два суміжні білі квадрати і фарбує їх у чорний колір. Перший гравець, який не може зробити хід, програє.

- $n = 1$: немає дійсних ходів, тому перший гравець автоматично програє.
- $n = 2$: лише один дійсний хід, після якого другий гравець програє.
- $n = 3$: два дійсні ходи, але обидва залишають ситуацію, в якій другий гравець програє.
- $n = 4$: перший гравець має три дійсні ходи; він може виграти, зафарбувавши два середні квадрати.
- $n = 5$: перший гравець має чотири дійсні ходи (показані нижче червоним кольором); але незалежно від вибору гравця, другий гравець (синій) виграє.

<img alt="valid starting moves for strip with 5 squares" src="https://cdn.freecodecamp.org/curriculum/project-euler/paper-strip-game.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Отже, для $1 ≤ n ≤ 5$ існує 3 значення $n$, для яких перший гравець може забезпечити перемогу.

Аналогічно, для $1 ≤ n ≤ 50$ існує 40 значень $n$, для яких перший гравець може забезпечити перемогу.

Для $1 ≤ n ≤ 1\\,000\\,000$, скільки існує значень $n$, для яких перший гравець може забезпечити перемогу?
