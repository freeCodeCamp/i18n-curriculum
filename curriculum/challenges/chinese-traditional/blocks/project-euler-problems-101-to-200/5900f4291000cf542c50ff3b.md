# --description--

數字 $a$ 以正整數 $b$ 為指數的超指數運算或四次方運算，記作 $a↑↑b$ 或 ${}^ba$，其遞迴定義為：

$a↑↑1 = a$，

$a↑↑(k+1) = a^{(a↑↑k)}$。

因此我們有例如 $3↑↑2 = 3^3 = 27$，因此 $3↑↑3 = 3^{27} = 7625597484987$，而 $3↑↑4$ 大約是 ${10}^{3.6383346400240996 \times {10}^{12}}$。找出 $1777↑↑1855$ 的最後 8 位數字。

# --hints--
