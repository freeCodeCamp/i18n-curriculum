# --description--

في الدرس السابق، تعلّمت عن التذكر وكيف يعمل الخطاف `useMemo`.

في هذا الدرس، ستتعلم كيف يعمل كل من الخطاف `useCallback` و `React.memo`.

في الدرس السابق، ذكرنا أيضًا أن `useCallback` مخصص لتذكر مراجع الدوال.

بالنسبة لـ `React.memo`، فهو يتيح لك تذكر مكوِّن لمنعه من إعادة العرض غير الضرورية عندما لا تتغير خاصيته.

إليك الصيغة النحوية الأساسية للخطاف `useCallback`:

```js
const handleClick = useCallback(() => {
  // code goes here
}, [dependency]);
```

وهنا الصيغة النحوية الأساسية لـ `React.memo`:

```jsx
const MemoizedComponent = React.memo(({ prop }) => {
  return (
    <>
      {/* Presentation */}
    </>
  )
});
```

لننظر إلى مثال على الخطاف `useCallback`:

```jsx
import { useState, useEffect } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  const handleClick = () => {
    setCount((prevCount) => prevCount + 1);
  };

  useEffect(() => {
    console.log("useEffect runs");
  }, [handleClick]);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={handleClick}>Increment</button>
    </div>
  );
}

export default Counter;
```

في المكوِّن، يعمل التأثير في كل مرة يتغير فيها `handleClick` لأن دالة `handleClick` تُعاد إنشاؤها في كل عرض.

لإصلاح هذا، عليك أن تخبر React بالتعامل مع دالة `handleClick` كشيء واحد عبر العروض من خلال تذكرها باستخدام الخطاف `useCallback`، حتى لا تُعاد إنشاؤها:

```jsx
import { useState, useEffect, useCallback } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  // Memoize the handleClick function with useCallback
  const handleClick = useCallback(() => {
    setCount((prevCount) => prevCount + 1);
  }, []);

  useEffect(() => {
    console.log("useEffect runs");
  }, [handleClick]);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={handleClick}>Increment</button>
    </div>
  );
}

export default Counter;
```

الآن، دالة `handleClick` لا تُعاد إنشاؤها في كل عرض.

لأريك كيف يعمل كل من دالة الرتبة العليا `React.memo` (أو `memo`) والخطاف `useCallback` معًا، إليك مكوِّن `Counter` مع دالة `handleClick` تحتاج إلى `useCallback` لكنها لا تستخدمه حاليًا:

```jsx
import { useState, useEffect, useCallback } from "react";
import CounterChild from "./CounterChild";

function Counter() {
  const [count, setCount] = useState(0);
  const [timer, setTimer] = useState(new Date().toLocaleTimeString());

  const handleClick = () => {
    setCount(count + 1);
  };

  useEffect(() => {
    const interval = setInterval(() => {
      setTimer(new Date().toLocaleTimeString());
    }, 1000);

    return () => clearInterval(interval);
  }, []);

  return (
    <div>
      <h1>Time: {timer}</h1>
      <p>Count: {count}</p>
      <button onClick={handleClick}>Increment</button>
      <CounterChild onClick={handleClick} />
    </div>
  );
}

export default Counter;
```

تحتوي هذه الدالة أيضًا على مؤقت في الحالة يتحدث كل ثانية. هذا يجعل المكوِّن يُعاد عرضه في كل مرة يتغير فيها `timer`، مما يجعل دالة `handleClick` تُعاد إنشاؤها في كل عرض.

لهذا السبب، يجب تذكر دالة `handleClick` باستخدام `useCallback`.

إليك مكوِّن `CounterChild`:

```jsx
const CounterChild = ({ onClick }) => {
  console.log("CounterChild component rendered");
  return <button onClick={onClick}>Increment from Child</button>;
};

export default CounterChild;
```

يأخذ مكوِّن `CounterChild` هذا خاصية `onClick`، مما يتيح لك زيادة العداد منه أيضًا.

بما أن مكوِّن `CounterChild` هو طفل لمكوِّن `Counter`، فإنه سيُعاد عرضه أيضًا في كل مرة يُعاد فيها عرض `Counter` بسبب تغير المؤقت. لذلك، يجب تذكر `CounterChild` أيضًا.

بدون التذكر، وبما أن المكوِّن يُعاد عرضه بسبب تحديث المؤقت كل ثانية، فإن مكوِّن `CounterChild` يُعاد عرضه أيضًا.

لمنع ذلك، عليك تذكر مكوِّن `CounterChild` باستخدام `React.memo`:

```jsx
import React from "react";

const CounterChild = React.memo(({ onClick }) => {
  console.log("CounterChild component rendered");
  return <button onClick={onClick}>Increment from Child</button>;
});

export default CounterChild;
```

الأمور لا تعمل بشكل مثالي حتى بعد تذكر `CounterChild` باستخدام `React.memo`.

يحدث هذا لأن دالة `handleClick` تُعاد إنشاؤها في كل عرض، لذا يجب تذكرها أيضًا باستخدام `useCallback`، لإخبار React بأنك تريد بقاء الدالة نفسها عبر العروض:

```js
const handleClick = useCallback(() => {
  setCount((prevCount) => prevCount + 1);
}, [count]);
```

الآن، يُعاد عرض المكوِّن فقط عندما تتغير حالة `count`.
