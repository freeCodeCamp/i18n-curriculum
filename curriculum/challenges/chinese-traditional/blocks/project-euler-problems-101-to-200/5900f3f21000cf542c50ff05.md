# --description--

考慮連續的質數 $p_1 = 19$ 和 $p_2 = 23$。可以驗證 1219 是最小的數字，其末位數字由 $p_1$ 組成，同時可被 $p_2$ 整除。

事實上，除了 $p_1 = 3$ 和 $p_2 = 5$ 這兩個例外，對於每一對連續的質數 $p_2 > p_1$，存在某些 $n$ 的值，使得最後的數字由 $p_1$ 組成且 $n$ 可被 $p_2$ 整除。令 $S$ 為這些 $n$ 值中最小的。

找出每一組連續質數且 $5 ≤ p_1 ≤ 1000000$ 的 $\sum{S}$。
