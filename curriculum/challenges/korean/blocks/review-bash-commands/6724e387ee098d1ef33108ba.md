## 터미널, 셸, 그리고 명령줄 기본

- **명령줄**: 사용자가 명령어를 입력하는 텍스트 인터페이스입니다.  
- **터미널**: 명령줄에 접근할 수 있게 해주는 애플리케이션입니다.  
- **터미널 에뮬레이터**: 터미널에 추가 기능을 더합니다.  
- **셸**: 터미널에 입력된 명령어를 해석합니다(예: Bash).  
- **PowerShell / 명령 프롬프트 / Microsoft Terminal**: Windows에서 명령줄에 접근하는 옵션입니다.  
- **터미널(macOS)**: macOS에 내장된 옵션이며, iTerm이나 Ghostty 같은 서드파티 대안도 있습니다.  
- **터미널(Linux)**: 배포판마다 다르며, kitty 같은 서드파티 에뮬레이터가 많습니다.  
- **용어**: "터미널", "셸", "명령줄"은 종종 혼용되지만 각각 구체적인 의미가 있습니다.  


## 명령줄 단축키

- **위/아래 화살표**: 이전/다음 명령어 기록을 순환합니다.
- **Tab**: 명령어 자동 완성합니다.
- **`Control+L`** (Linux/macOS) 또는 `cls` 입력 (Windows): 터미널 화면을 지웁니다.
- **`Control+C`**: 실행 중인 명령어를 중단합니다(선택된 텍스트가 있으면 PowerShell에서 복사에도 사용).
- **`Control+Z`** (Linux/macOS 전용): 작업을 백그라운드로 일시 중단합니다; 다시 시작하려면 `fg`를 사용하세요.
- **`!!`**: 마지막 실행한 명령어를 즉시 다시 실행합니다.

## Bash 기본

- **Bash** (Bourne Again Shell): 널리 쓰이는 유닉스 계열 셸입니다.  
  주요 명령어:

  - **`pwd`**: 현재 디렉터리를 보여줍니다.
  - **`cd`**: 디렉터리를 변경합니다.
    - **`..`**는 상위 디렉터리(한 단계 위)를 가리킵니다.
    - **`.`**는 현재 디렉터리를 가리킵니다.
  - **`ls`**: 파일과 폴더를 나열합니다.
    - **`-a`**: 숨김 파일을 포함해 모두 보여줍니다.
    - **`-l`**: 파일에 대한 자세한 정보를 보여줍니다.
  - **`less`**: 파일 내용을 한 페이지씩 보고, 뒤로 스크롤하거나 검색할 수 있습니다.
  - **`more`**: 파일 내용을 한 화면씩 보여주며, 제한된 뒤로 스크롤과 기본 탐색 기능이 있습니다.
  - **`cat`**: 스크롤이나 탐색 없이 전체 파일 내용을 한 번에 보여줍니다. 작은 파일에 유용합니다.
  - **`mkdir`**: 새 디렉터리를 만듭니다.
  - **`rmdir`**: 빈 디렉터리를 삭제합니다.
  - **`touch`**: 새 파일을 만듭니다.
  - **`mv`**: 파일을 이동하거나 이름을 바꿉니다.
    - 이름 변경: `mv oldname.txt newname.txt`
    - 이동: `mv filename.txt /path/to/target/`
  - **`cp`**: 파일을 복사합니다.
    - **`-r`**: 디렉터리와 그 내용을 재귀적으로 복사합니다.
  - **`rm`**: 파일을 삭제합니다.
    - **`-r`**: 디렉터리와 그 내용을 재귀적으로 삭제합니다.
  - **`echo`**: 텍스트 한 줄이나 변수 값을 출력합니다.
    - 기존 내용을 덮어쓰려면 `>`를 사용하세요. (예: `echo "text" > file.txt`)
    - 기존 내용을 덮어쓰지 않고 출력 내용을 파일에 추가하려면 `>>`를 사용하세요. (예: `echo "text" >> file.txt`)
  - **`exit`**: 터미널 세션을 종료합니다.
  - **`clear`**: 터미널 화면을 지웁니다.
  - **`find`**: 파일과 디렉터리를 검색합니다.
    - **`-name`**: 이름 패턴으로 파일을 검색합니다(예: `find . -name "*.txt"`).
  - **`man`** 다음에 명령어를 입력하면(예: `man ls`) 자세한 매뉴얼/도움말 페이지에 접근할 수 있습니다.

## 명령어 옵션과 플래그

- **옵션** 또는 **플래그**: 명령어 동작을 수정하며 보통 하이픈으로 시작합니다:
  - **긴 형식(하이픈 두 개)**:  
    - 예: `--help`, `--version`
    - 값은 등호로 붙여서 전달합니다, 예: `--width=50`.
  - **짧은 형식(하이픈 한 개)**:  
    - 예: `-a`, `-l`
    - 값은 공백으로 전달합니다, 예: `-w 50`.
    - 여러 짧은 옵션을 한꺼번에 묶을 수 있습니다, 예: `ls -alh`.
- **`--help`**: 어떤 명령어든 이 플래그와 함께 사용하면 사용 가능한 옵션을 확인할 수 있습니다.
