# --description--

Є $N$ місць у ряду. $N$ людей приходять один за одним, щоб зайняти місця за такими правилами:

1. Жодна людина не сідає поруч з іншою.
1. Перша людина обирає будь-яке місце.
1. Кожна наступна людина обирає місце, яке знаходиться найдале від усіх вже зайнятих, за умови, що це не порушує правило 1. Якщо існує кілька варіантів, що задовольняють цю умову, людина обирає найлівіший варіант.

Зверніть увагу, що через правило 1 деякі місця обов’язково залишаться вільними, і максимальна кількість людей, які можуть сісти, менша за $N$ (для $N > 1$).

Ось можливі варіанти розсадки для $N = 15$:

<img alt="seating arrangements for N = 15" src="https://cdn.freecodecamp.org/curriculum/project-euler/comfortable-distance-ii.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Ми бачимо, що якщо перша людина обирає правильно, 15 місць можуть вмістити до 7 людей. Також видно, що у першої людини є 9 варіантів вибору, щоб максимізувати кількість людей, які можуть сісти.

Нехай $f(N)$ — це кількість варіантів вибору першої людини, щоб максимізувати кількість зайнятих місць для $N$ місць у ряду. Тоді $f(1) = 1$, $f(15) = 9$, $f(20) = 6$, і $f(500) = 16$.

Також $\sum f(N) = 83$ для $1 ≤ N ≤ 20$ і $\sum f(N) = 13\,343$ для $1 ≤ N ≤ 500$.

Знайдіть $\sum f(N)$ для $1 ≤ N ≤ {10}^{12}$. Наведіть останні 8 цифр вашої відповіді.
