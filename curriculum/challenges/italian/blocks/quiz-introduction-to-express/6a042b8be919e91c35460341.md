### --question--

#### --text--

#### --distractors--

---

Genera e registra automaticamente endpoint REST API basati sui nomi dei file di rotta.

#### --answer--

Separa le responsabilità e mantiene il file principale pulito spostando le rotte correlate nei propri moduli.
