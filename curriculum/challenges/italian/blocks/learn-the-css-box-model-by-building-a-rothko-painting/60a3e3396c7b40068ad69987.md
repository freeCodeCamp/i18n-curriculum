# --description--

Non devi sempre usare i pixel per dimensionare un elemento.

Crea una nuova regola, `.three`, e imposta la sua `width` su `91%`.

# --hints--

Dovresti usare il selettore `.three`.

```js
const hasThree = new __helpers.CSSHelp(document).getStyle('.three');
assert(hasThree);
```

Dovresti impostare la proprietà `width` su `91%`.

```js
const hasWidth = new __helpers.CSSHelp(document).getCSSRules().some(x => x.style.width === '91%');
assert(hasWidth);
```

Il tuo elemento `.three` dovrebbe avere un valore `width` di `91%`.

```js
const threeWidth = new __helpers.CSSHelp(document).getStyle('.three')?.getPropertyValue('width');
assert(threeWidth === '91%');
```
