# --interactive--

Властивості flex — це властивості, які можна застосувати до контейнерів flex, щоб визначити розподіл дочірніх елементів. Ми розглянемо деякі з найпоширеніших: `flex-wrap`, `justify-content` і `align-items`.

Почнемо з `flex-wrap`. Ця властивість визначає, як гнучкі елементи обгортаються всередині контейнера flex, щоб поміститися у доступний простір. `flex-wrap` може приймати три можливі значення: `nowrap`, `wrap` і `wrap-reverse`. `nowrap` — це значення за замовчуванням: гнучкі елементи не будуть переноситися на новий рядок, навіть якщо їхня ширина перевищує ширину контейнера.

У наведеному нижче коді у нас є три `div` елементи. Зосередимося на `width`. Контейнер `main`, обведений червоним, має `width` зі значенням `200px`, тоді як його три дочірні `div` елементи разом мають `width` зі значенням `240px` (по `80px` кожен):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::


Ширина `div` елементів перевищує ширину їхнього контейнера, але за замовчуванням вони будуть стиснені, щоб поміститися у доступний простір. Якщо ви хочете, щоб вони переносилися, коли перевищують ширину контейнера, ви можете встановити `flex-wrap: wrap` на контейнері flex:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  flex-wrap: wrap;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Гнучкі елементи `div` будуть розташовані в рядках, коли вони перевищують ширину контейнера. Ви можете обгорнути гнучкі елементи у зворотному порядку за допомогою `flex-wrap: wrap-reverse`. Властивість `flex-flow` є скороченою властивістю для `flex-direction` і `flex-wrap`. У цьому прикладі ми встановили `flex-direction` у `column` і `flex-wrap` у `wrap-reverse`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  flex-flow: column wrap-reverse;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Чудово. Тепер поговоримо про `justify-content`. `justify-content` вирівнює дочірні елементи вздовж головної осі контейнера flex. Якщо ви призначите значення `flex-start` властивості `justify-content`, гнучкі елементи будуть вирівняні на початку головної осі. Це може бути горизонтально або вертикально:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: flex-start;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

За допомогою `justify-content: flex-end` гнучкі елементи вирівнюються в кінці головної осі, горизонтально або вертикально.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: flex-end;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Щоб центровано розташувати гнучкі елементи вздовж головної осі, можна використати `justify-content: center`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: center;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Щоб рівномірно розподілити елементи вздовж головної осі, можна використати `justify-content: space-between`. Це додасть пробіли між гнучкими елементами, якщо це потрібно.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: space-between;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

`justify-content: space-around` рівномірно розподіляє гнучкі елементи всередині головної осі, додаючи пробіл перед першим елементом і після останнього. Цей додатковий пробіл становить половину простору між кожною парою сусідніх елементів. Якщо є лише один елемент для розподілу, він буде центрований.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: space-around;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::


І нарешті, `justify-content: space-evenly` рівномірно розподіляє елементи вздовж головної осі. Пробіл між елементами та пробіли перед першим і після останнього елементів однакові:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: space-evenly;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Чудово. Тепер ви знаєте, як розподіляти гнучкі елементи вздовж головної осі. Але ви також можете захотіти розподілити їх вздовж поперечної осі. Пам’ятайте, що поперечна вісь перпендикулярна до головної осі. Це можна зробити за допомогою властивості `align-items`. Щоб центровано розташувати елементи вздовж поперечної осі, просто додайте `align-items: center` до контейнера flex:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: center;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

У цьому прикладі гнучкі елементи центровані вздовж поперечної осі, яка за замовчуванням вертикальна. Якщо поперечна вісь горизонтальна, вони будуть центровані горизонтально. Натомість `align-items: flex-start` вирівнює елементи на початку поперечної осі:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: flex-start;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Протилежністю є `align-items: flex-end`. Вона вирівнює гнучкі елементи в кінці поперечної осі, вертикально або горизонтально.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: flex-end;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Щоб розтягнути гнучкі елементи вздовж поперечної осі, можна використати `align-items: stretch`. Це також працює з елементами, які автоматично підбирають розмір, наприклад, без встановленої `width` або `height`, або з мінімальним `width` чи `height`. Гнучкі елементи розтягнуться, щоб заповнити контейнер у напрямку поперечної осі.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box">Three</div>
</div>

```

```css
.container {
  display: flex;
  align-items: stretch; 
  border: 2px solid #444;
  height: 200px; 
}

.box {
  width: 100px;           
  border: 1px solid #888;
  background-color: lightblue;
}
```

:::

І нарешті, ви можете використати властивість `align-self`, щоб призначити інше вирівнювання по поперечній осі для окремого гнучкого елемента. Наприклад, ви можете розтягнути його за допомогою `align-self: stretch`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box special">Three</div>
</div>

```

```css
.container {
  display: flex;
  align-items: flex-start; 
  border: 2px solid #444;
  height: 200px; 
}

.box {
  width: 100px;
  border: 1px solid #888;
  background-color: lightblue;
  margin: 4px;
}

.special {
  align-self: stretch; 
  background-color: lightcoral;
}
```

:::

Ви можете центровано розташувати його за допомогою `align-self: center`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

  <div class="box">Two</div>
  <div class="box special">Three</div>
</div>

```

```css

  margin: 4px;

  background-color: lightcoral;
}
```

:::

Ви можете вирівняти його на початку поперечної осі за допомогою `align-self: flex-start`.

:::interactive_editor


:::

Або вирівняти в кінці поперечної осі за допомогою `align-self: flex-end`.

:::interactive_editor


Існують інші властивості flex і значення, які ви можете обрати, щоб створити адаптивний макет, який ви уявляєте, але це найпоширеніші. З цими CSS-властивостями flex і вашими новими знаннями про модель CSS flex ви можете почати створювати адаптивні макети для плавного та інклюзивного користувацького досвіду на різних пристроях.
