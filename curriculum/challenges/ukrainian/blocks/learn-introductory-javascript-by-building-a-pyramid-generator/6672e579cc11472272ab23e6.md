JavaScript — це мова програмування, яка живить всемережжя. На відміну від HTML та CSS, які ви вивчали раніше, JavaScript найчастіше використовується для написання логіки замість розмітки.

У цьому проєкті ви дізнаєтесь про основи Javascript та застосуєте ці концепції для будування генератора пірамід.

Генератор пірамід — це програма, у якої можна встановити тип символ, число для піраміди та напрямок піраміди. Програма потім створить піраміду на основі цих вхідних даних.
