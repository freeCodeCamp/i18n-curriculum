# --explanation--

The phrase `That's a good point` is used to acknowledge that someone has made a valid or important argument. For example:

- **Employee:** `We should consider remote work options to improve productivity.`

- **Manager:** `That's a good point. It could help employees stay more focused.` - This person thinks remote work is a good way to improve workers' productivity by making them keep focus. 

Neste diálogo, Lisa está concordando com a declaração de Brian sobre o impacto de não valorizar as opiniões das pessoas.
