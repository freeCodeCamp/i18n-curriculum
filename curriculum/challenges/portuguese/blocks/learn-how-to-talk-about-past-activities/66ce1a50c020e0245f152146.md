### --feedback--

A forma plural de `fix`, esta palavra se refere a soluções ou reparos para um problema. 

# --explanation--

`Proposed` significa algo que foi sugerido ou apresentado para consideração. Por exemplo:

 `The proposed plan includes a new timeline.` - Isso significa que o plano, que inclui uma nova linha do tempo, foi sugerido mas ainda não foi finalizado ou aceito.

`Fixes` refere-se a soluções ou reparos para um problema. Por exemplo:

 `The team implemented several fixes to address the software bugs.` - Isso significa que a equipe aplicou soluções para resolver os problemas no software.
