# --description--

Hadi sasa, umekuwa ukionyesha sehemu za React kwa mteja. Kawaida, hili ndilo utakalofanya kila wakati. Hata hivyo, kuna baadhi ya matumizi ambapo inafaa kuonyesha sehemu ya React kwenye seva. Kwa kuwa React ni maktaba ya mtazamo ya JavaScript na unaweza kuendesha JavaScript kwenye seva kwa kutumia Node, hili linawezekana. Kwa kweli, React hutoa njia ya `renderToString()` ambayo unaweza kutumia kwa madhumuni haya.

Kuna sababu kuu mbili kwanini kuonyesha kwenye seva kunaweza kutumika katika programu halisi. Kwanza, bila kufanya hivi, programu zako za React zingekuwa na faili la HTML lenye upungufu na kundi kubwa la JavaScript linapopakuliwa kwa mara ya kwanza kwenye kivinjari. Hii inaweza kuwa si nzuri kwa mashine za utafutaji zinazojaribu kuorodhesha yaliyomo kwenye kurasa zako ili watu waweze kukuona. Ikiwa utaonyesha alama za awali za HTML kwenye seva na kuzipeleka kwa mteja, upakuaji wa ukurasa wa awali utakuwa na alama zote za ukurasa ambazo mashine za utafutaji zinaweza kuzichambua. Pili, hii huunda uzoefu wa upakuaji wa ukurasa wa awali kwa kasi zaidi kwa sababu HTML iliyoonyeshwa ni ndogo kuliko msimbo wa JavaScript wa programu nzima. React bado itakuwa na uwezo wa kutambua programu yako na kuisimamia baada ya upakuaji wa awali.

# --instructions--

Njia ya `renderToString()` inapatikana kwenye `ReactDOMServer`, ambayo ipo hapa kama kitu cha ulimwengu mzima. Njia hii inachukua hoja moja ambayo ni kipengele cha React. Tumia hii kuonyesha `App` kuwa mfuatano wa herufi.

# --hints--
