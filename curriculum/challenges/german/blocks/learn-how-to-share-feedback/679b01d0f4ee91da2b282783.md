# --explanation--

Der Ausdruck `That's a good point` wird verwendet, um anzuerkennen, dass jemand ein gültiges oder wichtiges Argument vorgebracht hat. Zum Beispiel:

- **Mitarbeitende:** `We should consider remote work options to improve productivity.`

- **Manager:** `That's a good point. It could help employees stay more focused.` – Diese Person hält Remote-Arbeit für eine gute Möglichkeit, die Produktivität der Mitarbeitenden zu steigern, indem sie den Fokus bewahren.

In diesem Dialog stimmt Lisa der Aussage von Brian über die Auswirkungen zu, wenn die Meinungen von Menschen nicht wertgeschätzt werden.
