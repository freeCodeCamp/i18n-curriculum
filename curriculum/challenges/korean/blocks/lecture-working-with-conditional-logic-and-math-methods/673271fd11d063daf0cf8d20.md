# --interactive--

조건문은 JavaScript 코드에서 결정을 내릴 수 있게 해줍니다. 특정 조건에 따라 프로그램의 흐름을 특정 방식으로 제어할 수 있습니다. `if`, `else if`, `else`, 그리고 삼항 연산자가 코드 흐름을 제어하는 방법을 살펴보겠습니다.

`if` 문은 조건을 받아 그 조건이 참 같은 값일 때 코드 블록을 실행합니다. 참 같은 값은 `true` 문과 같은 불리언 문맥에서 평가할 때 `if`가 되는 모든 값입니다. 다음은 참 같은 값의 예입니다:

- 비어 있지 않은 문자열, 예를 들어 `hello`

- `0`와 `-0`가 아닌 모든 숫자, 예를 들어 `4`, `-5` 등

- 배열

- 객체

- 불리언 `true`

반면, 거짓 같은 값은 불리언 문맥에서 `false`로 평가되는 값입니다. JavaScript에는 몇 가지 거짓 같은 값이 있어 기억하기 쉽습니다. 다음은 몇 가지 거짓 같은 값입니다:

- 불리언 `false`

- `0` (0)

- `""` (빈 문자열)

- `null`

- `undefined`

- `NaN` (숫자가 아님)

이제 참 같은 값과 거짓 같은 값에 대한 기본 이해가 되었으니 `if` 문과 함께 어떻게 작동하는지 살펴보겠습니다. 첫 번째 예에서는 두 개의 `if` 문을 사용해 참 같은 값과 거짓 같은 값을 검사합니다:

:::interactive_editor

```js
if (null) {
  console.log("This will not run.");
}

if ("freeCodeCamp") {
  console.log("This will run.");
}
```

:::

`null`가 거짓 같은 값이므로 블록 안의 메시지는 콘솔에 절대 기록되지 않습니다. 하지만 두 번째 `if` 문에서는 문자열 `freeCodeCamp`가 참 같은 값이며 `true` 문의 불리언 문맥에서 `if`로 간주됩니다. 결과적으로 메시지 `This will run.`가 콘솔에 기록됩니다.

다음으로 `if` 문이 다양한 비교 연산자와 함께 어떻게 작동하는지 몇 가지 예를 더 살펴보겠습니다. 다음은 사용자가 투표 자격이 있는지 확인하는 `if` 문 예입니다:

:::interactive_editor

```js
const age = 22;

if (age >= 18) {
 console.log("You're eligible to vote"); // You're eligible to vote
}
```

:::

이 예에서는 `age`가 현재 `22`이므로 조건은 `true`로 평가됩니다. `22`가 `18` 이상이기 때문입니다. 따라서 메시지 `You're eligible to vote`가 콘솔에 기록됩니다. 만약 `age`가 `15`으로 변경되면 조건은 `false`로 평가되어 메시지가 콘솔에 기록되지 않습니다:

:::interactive_editor

```js
const age = 15;

if (age >= 18) {
 console.log("You're eligible to vote"); // Code not running because age is less than 18
}
```

:::

조건이 `false`일 때 `else` 절을 사용할 수 있습니다:

:::interactive_editor

```js
const age = 15;

if (age >= 18) {
 console.log("You're eligible to vote");
} else {
 console.log("You're not eligible to vote"); // You're not eligible to vote
}
```

:::

이 예에서 `15`는 `18` 이상이 아니므로 조건은 `false`가 됩니다. 이 경우 `else` 블록 안의 코드가 실행됩니다.

여러 조건을 확인하려면 `else if` 블록을 사용할 수 있습니다. 이렇게 하면 프로그램이 두 가지 이상의 경로 중에서 선택할 수 있습니다.

:::interactive_editor

```js
const score = 87;

if (score >= 90) {
 console.log('You got an A'); 
} else if (score >= 80) {
 console.log('You got a B'); // You got a B
} else if (score >= 70) {
 console.log('You got a C');
} else {
 console.log('You failed! You need to study more!');
}
```

:::

`score`가 현재 `87`이므로 메시지 `You got a B`가 콘솔에 기록됩니다.

삼항 연산자는 간단한 `if/else` 문을 작성하는 간결한 방법입니다. 세 부분으로 구성됩니다: 조건, 조건이 참일 때 결과, 조건이 거짓일 때 결과. 기본 구문은 다음과 같습니다:

```js
condition ? expressionIfTrue : expressionIfFalse;
```

섭씨 온도와 관련된 예는 다음과 같습니다:

:::interactive_editor

```js
const temperature = 30;
const weather = temperature > 25 ? 'sunny' : 'cool';

console.log(`It's a ${weather} day!`);
```

:::

`temperature`가 `25`보다 크면 위 코드는 `It's a sunny day!`를 기록합니다. `temperature`가 `25` 이하일 경우 `It's a cool day!`를 기록합니다.

그렇다면 if 문과 삼항 연산자 중 어느 것을 사용해야 할까요? 단일 조건이나 단일 표현식을 다룰 때, 또는 간단한 논리를 간결한 구문으로 작성하고 싶을 때는 삼항 연산자를 사용하세요. 복잡한 조건과 여러 문장을 다룰 때는 `if/else` 문을 사용하세요. 삼항 연산자를 중첩하면 읽기 어려워집니다.
