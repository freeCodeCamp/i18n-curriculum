# --interactive--

Kipengele cha `button` hutumika kutekeleza kitendo fulani wakati kinapoanzishwa. Hapa kuna mfano wa kipengele cha `button` chenye maandishi ya kitufe ya `Start Game`.

```html
<button>Start Game</button>
```

Mifano mingine ya kutumia kipengele cha `button` ni pamoja na kuwasilisha fomu, kuonyesha modal, au kufungua na kufunga menyu ya pembeni. Kipengele cha `button` kina sifa ya `type` inayodhibiti tabia ya kitufe wakati kinapoanzishwa. Thamani ya kwanza inayowezekana kwa sifa ya `type` itakuwa aina ya `button`. Hapa kuna mfano wa kutumia kipengele cha `button` chenye aina ya `button` na maandishi ya `Show Alert`:

```html
<button type="button">Show Alert</button>
```

Kwa chaguo-msingi, kitufe hakitafanya chochote kinapoanzishwa. Hata hivyo, unaweza kuongeza msimbo wa JavaScript kufanya kitufe kiwe cha kuingiliana, kama vile kuonyesha onyo katika kesi hii.

Bofya kitufe cha `Show Alert` kuona onyo likitokea kwenye skrini. Ili kuingiliana na mfano huu, utahitaji kuwezesha mhariri wa kuingiliana.


**KUMBUKA**: Mfano huu wa kuingiliana unatumia JavaScript lakini huna haja ya wasiwasi kuhusu kuelewa msimbo wa JavaScript. Utajifunza kuhusu JavaScript katika moduli zijazo.

:::interactive_editor

```html
<button type="button">Show Alert</button>
<script src="index.js"></script>
```

```js
const btn = document.querySelector("button");
btn.addEventListener("click", () => alert("You clicked on the alert button"));
```

:::

Thamani nyingine inayowezekana kwa sifa ya `type` ni thamani ya `submit`. Hapa kuna mfano wa kutumia kipengele cha `button` chenye aina ya `submit`.

:::interactive_editor

```html
<form action="">
  <label for="email">Email address:</label>
  <input type="email" id="email" name="email" />
  <button type="submit">Submit form</button>
</form>
```

:::

Ndani ya kipengele cha `form`, kuna kipengele cha `label` na `input` kwa ajili ya anwani ya barua pepe ya mtumizi. Mtumiaji anapobofya kitufe cha kuwasilisha, data zao zitatumwa kwa seva na kutatuliwa. Thamani ya tatu inayowezekana kwa sifa ya `type` ni thamani ya `reset`. Hapa kuna mfano wa kipengele cha `form` chenye vitufe vya reset na kuwasilisha.

Wezesha mhariri wa kuingiliana na uingiliane na ingizo la barua pepe katika dirisha la onyesho kwa kutoa anwani ya barua pepe ya uongo. Kisha bofya kitufe cha reset kuona barua pepe yako ikifutika kutoka kwenye sehemu hiyo.

:::interactive_editor

```html
<form action="">
  <label for="email">Email address:</label>
  <input type="email" id="email" name="email" />
  <button type="reset">Reset form</button>
  <button type="submit">Submit form</button>
</form>
```

:::

Katika mfano huu uliobadilishwa, vipengele vya `label` na `input` vinatumika kukusanya anwani ya barua pepe ya mtumizi. Mtumiaji anapobofya kitufe cha reset, kitafuta data zote walizoweka. Ni muhimu kutambua kuwa vitufe vya reset kawaida si wazo zuri kwa sababu vinaweza kusababisha watumizi kufuta data zao kwa bahati mbaya. Pia, vitufe vingi kwenye fomu yako vinaweza kuleta msongamano kwenye kiolesura cha mtumizi.

Njia nyingine ya kuunda vitufe katika HTML ni kutumia kipengele cha `input`. Kipengele cha `input` pia kina sifa ya `type` yenye thamani zinazowezekana za `submit`, `reset`, na `button`. Hapa kuna mfano wa kutumia kipengele cha `input` chenye `type` imewekwa kuwa `button`:

:::interactive_editor

```html
<input class="start-btn" type="button" value="Start Game" />
<script src="index.js"></script>
```

```js
document.addEventListener("DOMContentLoaded", () => {
  const btn = document.querySelector(".start-btn");
  btn.addEventListener("click", () => {
    const paraEl = document.createElement("p");
    const bodyEl = document.querySelector("body");

    bodyEl.appendChild(paraEl);
    paraEl.textContent = "The game has started!!!";
  });
});
```

:::

Sifa ya `value` hutumika kuonyesha maandishi ya kitufe. Basi, ni tofauti gani kati ya kutumia vipengele vya `input` na `button`? Vipengele vya `input` ni vipengele visivyo na maudhui, maana yake haviwezi kuwa na nodi za mtoto, kama maandishi, na vinaweza kuwa na lebo ya kuanzisha tu. Kwa upande mwingine, kipengele cha `button` kinatoa uhuru zaidi kwa sababu unaweza kupanga ndani maandishi, picha, na alama ndani yake.
