# --description--

フィボナッチ数列は次の漸化式で定義されます：

$F_n = F_{n − 1} + F_{n − 2}$、ただし $F_1 = 1$ および $F_2 = 1$ です。

$F_{541}$ は113桁の数字を含み、最後の9桁が1から9までのパンデジタル（1から9の数字をすべて含むが順序は問わない）になる最初のフィボナッチ数であることがわかっています。また、575桁の $F_{2749}$ は最初の9桁が1から9までのパンデジタルになる最初のフィボナッチ数です。

最初の9桁と最後の9桁の両方が1から9までのパンデジタルになる最初のフィボナッチ数 $F_k$ が与えられたとき、`k` を求めてください。
