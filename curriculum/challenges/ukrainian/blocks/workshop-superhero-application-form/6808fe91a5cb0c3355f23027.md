# --description--

У цьому практичному занятті ви навчитеся працювати з формами в React, створюючи форму застосунку для супергероя.

Дано шаблонний код, вам потрібно лише створити залишок React-компонента.

Почніть з повернення елемента `div` з `className` рівним `form-wrap`. Всередині `div` створіть елемент `h2` з текстом `Superhero Application Form`. Нижче створіть елемент `p` з текстом `Please complete all fields`.

# --hints--

Ви повинні повернути елемент `div` з `className` рівним `form-wrap`.

```js
assert.exists(document?.querySelector("div.form-wrap"));
```

Ви повинні створити елемент `h2` всередині вашого елемента `div`.

```js
assert.exists(document?.querySelector(".form-wrap > h2"));
```

Ваш елемент `h2` повинен містити текст `Superhero Application Form`.

```js
const h2El = document?.querySelector("h2")
assert.equal(h2El?.textContent, "Superhero Application Form");
```

Ви повинні створити елемент `p` всередині вашого елемента `div`. Переконайтеся, що він йде після `h2`.

```js
assert.exists(document?.querySelector(".form-wrap > h2 + p"));
```

Ваш елемент `p` повинен містити текст `Please complete all fields`.

```js
const pEl = document?.querySelector("p")
assert.equal(pEl?.textContent, "Please complete all fields");
```
