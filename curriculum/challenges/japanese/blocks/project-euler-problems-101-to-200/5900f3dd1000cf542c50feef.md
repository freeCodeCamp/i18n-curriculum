# --description--

左から右へ見ていって、左側の数字に負けない数字だけで構成されている場合、それは増加数と呼ばれます。例えば、134468です。

同様に、右側の数字に負けない数字だけで構成されている場合、それは減少数と呼ばれます。例えば、66420です。

増加数でも減少数でもない正の整数を「バウンシー数」と呼びます。例えば、155349です。

100未満にはバウンシー数は存在しませんが、1000未満の数のうち半分以上（525個）がバウンシー数です。実際、バウンシー数の割合が初めて50%に達する最小の数は538です。

驚くことに、バウンシー数はどんどん増えていき、21780に達する頃にはバウンシー数の割合が90%になります。

バウンシー数の割合がちょうど99%になる最小の数を見つけてください。

# --hints--

`bouncyNumbers()` は `1587000` を返すべきです。

```js
assert.strictEqual(bouncyNumbers(), 1587000);
```
