# --description--

Python は文字列操作を簡単にする多くの組み込みメソッドを提供しています。これらには以下のものが含まれますが、これらに限定されません：

- `upper()`: すべての文字を大文字に変換した新しい文字列を返します。
    

```python
my_str = 'hello world'

uppercase_my_str = my_str.upper()
print(uppercase_my_str)  # HELLO WORLD
```

- `lower()`: すべての文字を小文字に変換した新しい文字列を返します。
    

```python
my_str = 'Hello World'

lowercase_my_str = my_str.lower()
print(lowercase_my_str)  # hello world
```

- `strip()`: 指定した先頭と末尾の文字を削除した新しい文字列を返します。引数が渡されない場合は、先頭と末尾の空白を削除します。
    

```python
my_str = '  hello world  '

trimmed_my_str = my_str.strip()
print(trimmed_my_str)  # "hello world"
```

- `replace(old, new)`: `old` のすべての出現箇所を `new` に置き換えた新しい文字列を返します。
    

```python
my_str = 'hello world'

replaced_my_str = my_str.replace('hello', 'hi')
print(replaced_my_str)  # hi world
```

- `split(separator)`: 指定した区切り文字で文字列を分割し、文字列のリストを返します。区切り文字が指定されない場合は空白で分割します。
    

```python
my_str = 'hello world'

split_words = my_str.split()
print(split_words)  # ['hello', 'world']
```

- `join(iterable)`: イテラブルの要素を区切り文字で結合して文字列を返します。
    

```python
my_list = ['hello', 'world']

joined_my_str = ' '.join(my_list)
print(joined_my_str)  # hello world
```

- `startswith(prefix)`: 文字列が指定した接頭辞で始まるかどうかを示すブール値を返します。
    

```python
my_str = 'hello world'

starts_with_hello = my_str.startswith('hello')
print(starts_with_hello)  # True
```

- `endswith(suffix)`: 文字列が指定した接尾辞で終わるかどうかを示すブール値を返します。
    

```python
my_str = 'hello world'

ends_with_world = my_str.endswith('world')
print(ends_with_world)  # True
```

- `find(substring)`: `substring` の最初の出現箇所のインデックスを返します。見つからなければ `-1` を返します。
    

```python
my_str = 'hello world'

world_index = my_str.find('world')
print(world_index)  # 6
```

- `count(substring)`: 文字列内に部分文字列が出現する回数を返します。
    

```python
my_str = 'hello world'

o_count = my_str.count('o')
print(o_count)  # 2
```

- `capitalize()`: 最初の文字を大文字にし、他の文字を小文字にした新しい文字列を返します。
    

```python
my_str = 'hello world'

capitalized_my_str = my_str.capitalize()
print(capitalized_my_str)  # Hello world
```

- `isupper()`: 文字列内のすべての文字が大文字なら `True` を、そうでなければ `False` を返します。
    

```python
my_str = 'hello world'

is_all_upper = my_str.isupper()
print(is_all_upper)  # False
```

- `islower()`: 文字列内のすべての文字が小文字なら `True` を、そうでなければ `False` を返します。
    

```python
my_str = 'hello world'

is_all_lower = my_str.islower()
print(is_all_lower)  # True
```

- `title()`: 各単語の最初の文字を大文字にした新しい文字列を返します。
    

```python
my_str = 'hello world'

title_case_my_str = my_str.title()
print(title_case_my_str)  # Hello World
```
