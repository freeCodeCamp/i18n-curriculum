# --interactive--

여백과 line-height는 목록 항목 간격을 조절하여 가독성과 시각적 매력을 높이는 데 필수적입니다.

먼저, 여백을 사용해 목록 항목 간격을 조절하는 방법부터 시작해 보겠습니다!

여백은 `li` 요소에 margin 속성을 적용해 목록 항목 사이에 공간을 만들 수 있습니다. 이 방법은 각 목록 항목 외부의 간격을 조절하여 항목 간 간격을 효과적으로 늘리거나 줄일 수 있습니다.

세 개의 목록 항목이 있는 순서 없는 목록 예제를 살펴보겠습니다.

:::interactive_editor

```html
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

기본적으로 HTML은 목록 항목 사이에 많은 간격을 적용하지 않습니다.

각 목록 항목 아래쪽에 간격을 적용하려면 다음과 같이 `margin-bottom` 속성을 사용할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  margin-bottom: 40px;
}
```

:::

이 예제에서는 순서 없는 목록 안의 각 목록 항목 아래쪽에 `40px`의 margin이 적용됩니다.

목록 항목 간격을 조절하는 또 다른 방법은 `line-height` 속성을 사용하는 것입니다.

`line-height` 속성은 단일 목록 항목 내 텍스트 줄 사이의 수직 간격을 조절합니다.

주로 각 항목 내 텍스트 줄 사이 간격에 영향을 주지만, 항목이 한 줄 텍스트만 포함할 경우 목록 항목 간 전체 간격에도 간접적으로 영향을 줄 수 있습니다.

목록 항목에 여러 줄 텍스트가 있을 경우 `line-height`는 그 줄들 사이 간격에 영향을 주지만, 개별 목록 항목 간 간격을 직접 조절하지는 않습니다.

개별 목록 항목 간 간격을 조절하려면 대신 `margin` 또는 `padding` 속성을 사용해야 합니다.

앞서 사용한 동일한 순서 없는 목록에 `line-height`을 적용한 예는 다음과 같습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  line-height: 2; 
}
```

:::

이 예제에서는 `line-height: 2;`가 폰트 크기의 두 배로 줄 높이를 설정하여 각 목록 항목 내에 더 많은 수직 공간을 만듭니다.

# --questions--

## --text--

각 `li` 요소 외부의 공간을 조절하여 목록 항목 사이에 간격을 만들려면 어떤 CSS 속성을 사용해야 하나요?

## --answers--

`padding`

### --feedback--

이 속성은 요소의 `border` 내부 공간에 영향을 줍니다.

---

`line-height`

### --feedback--

이 속성은 텍스트 줄 사이 간격에 영향을 주며, 요소 외부 공간에는 영향을 주지 않습니다.

---

`margin`

---

`border`

### --feedback--

이 속성은 요소의 `border`에 영향을 주며, 외부 공간에는 영향을 주지 않습니다.

## --text--

CSS 규칙 `li { margin-bottom: 15px; }`이 목록 항목에 어떤 영향을 미치나요?

## --answers--

각 목록 항목 내부에 `15px`의 공간을 추가합니다.

### --feedback--

이 규칙은 목록 항목 내부가 아니라 외부 공간에 영향을 줍니다.

---

각 목록 항목의 내용과 `15px` 사이에 `border`의 공간을 추가합니다.

### --feedback--

이 규칙은 각 목록 항목의 `border` 외부 공간에 영향을 줍니다.

---

각 목록 항목 아래에 `15px`의 공간을 추가합니다.

---

각 목록 항목 위에 `15px`의 공간을 추가합니다.

### --feedback--

이 규칙은 각 목록 항목 아래에 공간을 추가하며, 위쪽에는 추가하지 않습니다.

CSS에서 `line-height` 속성은 무엇을 조절하나요?

내용의 `width`.

### --feedback--

이는 `width` 속성으로 조절하며, `line-height`이 아닙니다.

요소의 `border` 내부 공간.

요소의 `border` 외부 공간.

이는 `margin` 같은 속성으로 조절하며, `line-height`이 아닙니다.
