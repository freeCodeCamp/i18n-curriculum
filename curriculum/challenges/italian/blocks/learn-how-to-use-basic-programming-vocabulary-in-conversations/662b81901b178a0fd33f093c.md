## --text--

Come si dichiara una variabile in Python secondo Sophie?

## --answers--

Dichiari una variabile dandole il nome di un operatore.

### --feedback--

Questa scelta è errata, perché la variabile non prende il nome dall'operatore, ma viene scritta prima di esso.

---

Le variabili si dichiarano usando un comando chiamato `declare`.

### --feedback--

Python non richiede un comando speciale come `declare` per creare variabili; usa direttamente l'assegnazione.

---

Usi il nome della variabile seguito dall'operatore di assegnazione e da un valore.

---

Le variabili devono essere dichiarate scrivendo `variable` prima del nome.

### --feedback--

Python non usa parole chiave come `variable` per dichiarare variabili. Usa semplicemente il nome della variabile, l'operatore di assegnazione e il valore.
