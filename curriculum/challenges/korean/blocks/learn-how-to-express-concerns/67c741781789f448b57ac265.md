# --explanation--

`There's always`는 무언가 부정적인 일이 반복되거나 절대 멈추지 않는 것처럼 느껴질 때 자주 사용됩니다 - 무엇을 해도 부정적인 일이 계속 일어나는 것처럼 보입니다. 예를 들어:

`There's always another problem to fix.` - 문제들이 계속 나타나 상황이 끝이 없는 것처럼 느껴진다는 의미입니다.
