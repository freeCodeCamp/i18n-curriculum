# --description--

**目標：** 建置一個功能上類似於此的應用程式：<a href="https://p2p-video-chat-application.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://p2p-video-chat-application.freecodecamp.rocks/</a>。

完成以下使用者故事並通過所有測試。使用你需要的任何函式庫或 API。展現你自己的風格。

**使用者故事：** 到達時，瀏覽器會提示我存取我的相機和麥克風。

**使用者故事：** 在我授權後，系統會提示我輸入一個房間名稱。

**使用者故事：** 一旦我輸入房間名稱，如果之前沒有該名稱的房間，將會創建一個房間。

**使用者故事：** 我的朋友隨後可以前往相同的網站，輸入我進入的相同房間，並加入該房間，然後與我進行視訊聊天。

**使用者故事：** 如果我輸入一個房間名稱，且該房間已經有兩個人，我會收到房間已滿的通知。

**使用者故事：**任何人都可以創建或加入任何房間。房間數量不限，但所有房間名稱必須唯一。

**使用者故事：** 我可以選擇不允許網站存取我的 `microphone` 和 `webcam`。如果我選擇不這麼做，或是發生其他驅動程式問題，我會看到一則錯誤訊息，說明這些是必須的。

**使用者故事：** 當我選擇取消房間名稱輸入步驟，或是我沒有輸入名稱，或只輸入空白時，系統應該再次要求我輸入有效的房間名稱。

**使用者故事：** 如果房間內的兩個人其中一人斷線，他們可以重新連接到同一個房間並繼續聊天。

完成這些使用者故事的實作後，輸入你的線上應用程式的 URL，並選填你的 GitHub 儲存庫。然後點擊「我已完成此挑戰」按鈕。
