# --explanation--

Слово `Encantada` — це альтернативний, загальний спосіб для жінки сказати "Рада зустрічі".

Воно означає "зачарована" або "в захваті".

| Форма | Рід | Говорить | Значення |
| :--- | :--- | :--- | :--- |
| `Encantado` | Чоловічий (`-o`) | Чоловік | Радий зустрічі. |
| `Encantada` | Жіночий (`-a`) | Жінка | Рада зустрічі. |
