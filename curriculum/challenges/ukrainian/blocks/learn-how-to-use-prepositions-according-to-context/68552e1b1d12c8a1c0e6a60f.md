# --explanation--

`Enough` означає мати стільки, скільки потрібно або вимагається. Наприклад:

`Do we have enough time to finish the project?` – Це означає запит, чи є доступний час достатнім.

`For` можна використовувати, щоб показати мету або ціль, з якою щось пов’язане. Наприклад:

`This budget is for the upcoming product launch.` – Це означає, що бюджет призначений для підтримки тієї події.
