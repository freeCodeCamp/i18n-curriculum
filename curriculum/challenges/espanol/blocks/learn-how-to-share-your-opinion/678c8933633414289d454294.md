# --questions--

## --text--

¿Qué opción describe correctamente `frontend` o `backend`?

## --answers--

El diseño y la estructura de un sitio web son frontend.

---

El almacenamiento de datos y las operaciones del lado del servidor son frontend.

### --feedback--

El frontend se encarga de lo que los usuarios ven e interactúan, no del almacenamiento de datos ni de las tareas del lado del servidor.

---

Las características orientadas al usuario como botones, formularios y menús de navegación son backend.

### --feedback--

El backend maneja datos y procesos del servidor, no funciones visibles para el usuario como botones y formularios.

---

Las bases de datos, las APIs y el mantenimiento del servidor son frontend.

### --feedback--

El frontend se centra en los aspectos visuales e interactivos de un sitio web, no en bases de datos o mantenimiento del servidor.

# --explanation--

El `frontend` es todo con lo que los usuarios interactúan en un sitio web, como el diseño, la estructura y los botones. El `backend` gestiona tareas del lado del servidor como el almacenamiento de datos, las APIs y el mantenimiento del servidor.
