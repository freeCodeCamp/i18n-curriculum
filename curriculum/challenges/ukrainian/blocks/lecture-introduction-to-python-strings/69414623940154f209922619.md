# --description--

При роботі з рядками часто доводиться об’єднувати різні частини тексту.

У Python ви можете об’єднати кілька рядків за допомогою оператора плюс (`+`). Цей процес називається **конкатенацією рядків**. Ось як об’єднати два рядки за допомогою оператора плюс:

```python
my_str_1 = 'Hello'
my_str_2 = "World"

str_plus_str = my_str_1 + ' ' + my_str_2
print(str_plus_str) # Hello World
```

Але зауважте, що це працює лише з рядками. Якщо спробувати конкатенувати рядок із числом, отримаєте `TypeError`:

```python
name = 'John Doe'
age = 26

name_and_age = name + age
print(name_and_age) # TypeError: can only concatenate str (not "int") to str
```

Це відбувається тому, що Python не перетворює автоматично інші типи даних, як-от цілі числа, у рядки під час конкатенації. Python вимагає, щоб усі елементи були рядками перед об’єднанням. Щоб це виправити, можна перетворити число на рядок за допомогою вбудованої функції `str()`, яка повертає рядкове представлення заданого об’єкта без зміни самого об’єкта:

```python
name = 'John Doe'
age = 26

name_and_age = name + str(age)
print(name_and_age) # John Doe26
```

Також можна використати оператор розширеного присвоєння для конкатенації. Він позначається знаком плюс і знаком рівності (`+=`) і виконує одночасно конкатенацію та присвоєння. Ось як це працює:

```py
name = 'John Doe'
age = 26

name_and_age = name  # Start with the name
name_and_age += str(age)  # Append the age as string

print(name_and_age)  # John Doe26
```

Процес вставлення змінних і виразів у рядок називається **інтерполяцією рядків**. У Python є тип рядків, який називається **f-рядки** (скорочено від форматованих рядкових літералів), що дозволяє виконувати інтерполяцію з компактним і зрозумілим синтаксисом.

F-рядки починаються з `f` (нижній або верхній регістр) перед лапками і дозволяють вставляти змінні або вирази всередину замінних полів, позначених фігурними дужками (`{}`). Ось приклад:

```python
name = 'John Doe'
age = 26
name_and_age = f'My name is {name} and I am {age} years old'
print(name_and_age) # My name is John Doe and I am 26 years old

num1 = 5
num2 = 10
print(f'The sum of {num1} and {num2} is {num1 + num2}') # The sum of 5 and 10 is 15
```

Зверніть увагу, що не потрібно перетворювати типи, які не є рядками, за допомогою функції `str()`. У наведеному прикладі значення змінних `age`, `num1` і `num2` під час інтерполяції автоматично перетворюються на рядок.
