# --description--

تحسين "الوقت حتى الاستخدام"، أو بعبارة أخرى الفترة من طلب المستخدم لصفحة حتى تمكنه من التفاعل معها بشكل فعّال، أمر حاسم لتعزيز تجربة المستخدم.

لنستعرض استراتيجيات فعّالة لتحقيق وقت أسرع حتى الاستخدام.

ابدأ بالتركيز على المحتوى الذي يراه المستخدم أولًا. هذا يعني تحميل العناصر الأساسية فورًا وتأجيل المكوّنات غير الحرجة.

على سبيل المثال، يمكنك تجربة تنفيذ التحميل الكسول.

لقد غطينا التحميل الكسول سابقًا، لكن هنا مثال على تحميل الصور كسولًا كتذكير:

```html
<img src="image.jpg" loading="lazy" alt="Description">
```

كل ما عليك فعله هو إضافة الخاصية `loading="lazy"` إلى وسم الصورة. هذا يؤجل تحميل الصور حتى الحاجة إليها، عادةً عندما تدخل إطار العرض الخاص بالمستخدم. بتحميل الصور والفيديوهات فقط عند دخولها إطار العرض، نوفر عرض النطاق الترددي ونعجّل عرض الصفحة الأولي.

عادةً، تقوم صفحة الويب بتحميل كل الصور دفعة واحدة، حتى تلك غير المرئية على الشاشة. مع التحميل الكسول، تُحمّل الصور الموجودة أسفل الصفحة، أو بمعنى آخر خارج الشاشة، فقط عندما ينزل المستخدم إليها بالتمرير.

وهنا مثال على تحميل كسول لـ iframes:

```html
<iframe src="video.html" loading="lazy"></iframe>
```

يمكنك تطبيق نفس الخاصية التي استخدمتها سابقًا، لكن هذه المرة على iframes، وسينطبق نفس المفهوم.

يمكنك أيضًا استخدام الخاصية `defer` لتأجيل جافاسكريبت غير الحرجة حتى بعد تحميل الصفحة الأولي. هنا مثال على ذلك:

```html
<script src="non-critical.js" defer></script>
```

يمكنك أيضًا التفكير في تقليل الموارد التي تعيق العرض.

الموارد التي تعيق العرض تؤخر تفاعل الصفحة.

لتجنب ذلك، يمكنك محاولة تحميل CSS بشكل غير متزامن. بالنسبة لـ CSS غير الحرجة، استخدم الخاصية `media` لتحميل أوراق الأنماط بشكل مشروط. هنا مثال على ذلك:

```html
<link rel="stylesheet" href="print.css" media="print">
```

بتحديد قيمة media كـ `print`، سيتم تحميل `print.css` فقط عند طباعة الصفحة أو معاينتها على الشاشة. في المثال التالي، بتمرير `min-width: 800px` كقيمة للخاصية `media`، نحدد تحميل `desktop.css` فقط للشاشات التي عرضها أكبر من 800 بكسل:

```html
<!-- This stylesheet is only loaded on screens wider than 800px -->
<link rel="stylesheet" href="desktop.css" media="(min-width: 800px)">
```

هذا يوفر عرض النطاق الترددي بتخطي الأنماط غير الضرورية.

في الختام، من خلال تنفيذ هذه الاستراتيجيات، يمكنك تقليل "الوقت حتى الاستخدام" لموقعك بشكل كبير، مما يضمن تمكّن المستخدمين من التفاعل مع محتواك بسرعة.
