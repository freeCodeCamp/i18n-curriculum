# --description--

Ci sono due limitazioni sui nomi delle variabili in JavaScript:

1. Il nome deve contenere solo lettere, cifre o i simboli `$` e `_`.
1. Il primo carattere non deve essere una cifra.

Esempi di nomi validi:

```js
let userName;
let test123;
```

Quello che è interessante – il simbolo del dollaro `'$'` e il trattino basso `'_'` possono essere usati anche nei nomi. Sono simboli normali, proprio come le lettere, senza alcun significato speciale.

Questi nomi sono validi:

```js
let $ = 1; // declared a variable with the name "$"
let _ = 2; // and now a variable with the name "_"

console.log($ + _); // 3
```

Esempi di nomi di variabili non corretti:

```js
let 1a; // cannot start with a digit

let my-name; // hyphens '-' aren't allowed in the name
```
