# --description--

小于等于 100 且仅能被素数 2 和 3 整除的最大整数是 96，因为 $96 = 32 \times 3 = 2^5 \times 3$。

对于两个不同的素数 $p$ 和 $q$，令 $M(p, q, N)$ 为不大于 $N$ 且仅能被 $p$ 和 $q$ 同时整除的最大正整数；如果不存在这样的正整数，则 $M(p, q, N)=0$。

例如 $M(2, 3, 100) = 96$。

$M(3, 5, 100) = 75$ 而不是 90，因为 90 可被 2、3 和 5 整除。同样，$M(2, 73, 100) = 0$，因为不存在可被 2 和 73 同时整除且小于等于 100 的正整数。

设 $S(N)$ 为所有不同 $M(p, q, N)$ 的和。$S(100)=2262$。

查找 $S(10\,000\,000)$。
