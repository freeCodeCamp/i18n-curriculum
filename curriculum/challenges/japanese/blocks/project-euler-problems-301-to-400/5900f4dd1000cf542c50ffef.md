# --description--

調和級数 $1 + \dfrac{1}{2} + \dfrac{1}{3} + \dfrac{1}{4} + \ldots$ は発散することでよく知られています。

しかし、この級数から分母に9が含まれる項をすべて省くと、この修正された調和級数は驚くべきことに約22.9206766193に収束します。この修正調和級数はケンプナー級数と呼ばれます。

ここで、調和級数から分母に3つ以上連続した同じ数字が含まれる項をすべて省く別の修正調和級数を考えます。調和級数の最初の1200項のうち、わずか20項だけが省かれることを確認できます。

これらの20項は以下の通りです：

$$\dfrac{1}{111}, \dfrac{1}{222}, \dfrac{1}{333}, \dfrac{1}{444}, \dfrac{1}{555}, \dfrac{1}{666}, \dfrac{1}{777}, \dfrac{1}{888}, \dfrac{1}{999}, \dfrac{1}{1000}, \dfrac{1}{1110}, \\\\
\dfrac{1}{1111}, \dfrac{1}{1112}, \dfrac{1}{1113}, \dfrac{1}{1114}, \dfrac{1}{1115}, \dfrac{1}{1116}, \dfrac{1}{1117}, \dfrac{1}{1118}, \dfrac{1}{1119}$$

この級数もまた収束します。

この級数が収束する値を求めてください。答えは小数点以下10桁に丸めてください。
