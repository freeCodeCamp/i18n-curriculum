# --description--

`interview`는 누군가가 정보를 얻기 위해 질문하는 대화입니다. 예를 들어:

`He had an interview for a job at the bank.` - 여기서 `interview`는 그가 직무에 대해 질문을 받은 공식적인 면접을 의미합니다.

`video interview`는 Zoom, Teams, Skype 같은 영상 도구를 사용해 대면하지 않고 온라인으로 진행하는 면접 유형입니다. 사람들이 같은 장소에 있을 수 없을 때 직장, 학교, 연구에서 흔히 사용됩니다.
