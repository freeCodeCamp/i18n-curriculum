# --explanation--

`Then` al final de una oración se usa para mostrar una conclusión, sugerencia o resultado que sigue de algo que se acaba de mencionar. A menudo da la idea de un resultado o consecuencia. Por ejemplo:

`If it's too expensive, we'll wait until next month then.` – Esto significa que el hablante ha tomado una decisión basada en lo que se dijo antes.
