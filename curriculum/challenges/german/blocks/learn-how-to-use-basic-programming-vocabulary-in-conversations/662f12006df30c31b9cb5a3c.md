# --description--

Ein `concept` bezieht sich im Allgemeinen auf eine Idee oder ein Prinzip, das verwendet wird, um Aktionen und Abläufe zu planen oder zu definieren. In verschiedenen Fächern, einschließlich der Programmierung, sind `concepts` die grundlegenden Ideen oder Theorien.

Zum Beispiel `Object-oriented programming is a concept in computer programming.`
