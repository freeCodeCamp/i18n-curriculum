# --interactive--

واجهة برمجة تطبيقات `Canvas` هي أداة قوية تتيح لك التلاعب بالرسومات داخل ملف JavaScript الخاص بك. يبدأ كل شيء بعنصر `canvas` في HTML. هذا العنصر يعمل كسطح رسم يمكنك التلاعب به باستخدام الطرق والخصائص الخاصة بواجهة برمجة تطبيقات `Canvas`.

توفر واجهة برمجة تطبيقات `Canvas` كل ما تحتاجه لإنشاء رسومات مذهلة، بما في ذلك الأشكال، النصوص، الرسوم المتحركة، وحتى الألعاب المعقدة. تحتوي على واجهات مثل `HTMLCanvasElement`، `CanvasRenderingContext2D`، `CanvasGradient`، `CanvasPattern`، و`TextMetrics` التي توفر طرقًا وخصائص يمكنك استخدامها لإنشاء الرسومات في ملف JavaScript الخاص بك.

لنلق نظرة على كيفية عمل واجهة برمجة تطبيقات `Canvas` حتى تبدأ في إنشاء الرسومات باستخدام JavaScript.

أولًا، تحتاج إلى إنشاء عنصر `canvas` في ملف HTML الخاص بك:

```html
<canvas id="my-canvas"></canvas>
```

يمثل عنصر `canvas` الواجهة `HTMLCanvasElement`، التي توفر طرقًا وخصائص للتلاعب به. بالإضافة إلى ذلك، يمكنك استخدام الطرق والخصائص من واجهات أخرى في واجهة برمجة تطبيقات `Canvas`.

يمكنك تحديد `width` و`height` للـ `canvas` داخل HTML:

```html
<canvas id="my-canvas" width="400" height="400"></canvas>
```

أو يمكنك استخدام خصائص `width` و`height` من واجهة `HTMLCanvasElement`:

```js
const canvas = document.getElementById("my-canvas");
canvas.width = 400;
canvas.height = 400;
```

حتى الآن، لا يمكنك رؤية أي شيء على الشاشة. بعد إنشاء عنصر `canvas`، الخطوة التالية هي الحصول على وصول إلى سياق الرسم للوحة باستخدام طريقة `getContext()` من واجهة `HTMLCanvasElement`.

السياق الأكثر شيوعًا هو `2d`، الذي يسمح لك بالرسم في بعدين:

```js
const canvas = document.getElementById("my-canvas");
const ctx = canvas.getContext('2d');
```

إذا سجلت المتغير `ctx` في وحدة التحكم، سترى الطرق والخصائص الخاصة بـ `CanvasRenderingContext2D` التي يمكنك استخدامها لإنشاء الأشكال، الألوان، الخطوط، والمزيد، مع القيم الافتراضية الخاصة بها:

```js
console.log(ctx);
```

بمجرد حصولك على سياق 2D، يمكنك البدء في الرسم على اللوحة.

توفر واجهة برمجة تطبيقات `Canvas` عدة طرق وخصائص لرسم الأشكال، الخطوط، والنصوص. واحدة منها هي الخاصية `fillStyle`، التي يمكنك دمجها مع الطريقة `fillRect()` لرسم مستطيل أو مربع:

:::interactive_editor

```html
<html>
  <head>
  </head>
  <body>
    <canvas id="my-canvas" width="400" height="400"></canvas>
    <script src="index.js"></script>
  </body>
</html>
```

```js
const canvas = document.getElementById("my-canvas");

const ctx = canvas.getContext("2d");

// Set the background color
ctx.fillStyle = "crimson";

// Draw a rectangle
ctx.fillRect(1, 1, 150, 100);
```

:::

تأخذ `fillRect` أربع قيم رقمية تمثل محور x، محور y، العرض، والارتفاع، على التوالي.

هناك شيء ظاهر على الشاشة الآن. يمكنك أيضًا رسم نص أو حتى إنشاء رسوم متحركة. إليك لوحة لتمثيل النص:

```html
<canvas id="my-text-canvas" width="300" height="70"></canvas>
```

لرسم النص أخيرًا، مرر النص إلى الطريقة `fillText()` كالمعلمة الأولى، تليها القيم لمحوري x وy:

:::interactive_editor

```html
<html>
  <head>
  </head>
  <body>
    <canvas id="my-text-canvas" width="300" height="70"></canvas>
    <script src="index.js"></script>
  </body>
</html>
```

```js
const textCanvas = document.getElementById("my-text-canvas");

const textCanvasCtx = textCanvas.getContext("2d");

// Set font family and size
textCanvasCtx.font = "30px Arial";

// Set text color
textCanvasCtx.fillStyle = "crimson";

// Draw the text
textCanvasCtx.fillText("Hello HTML Canvas!", 1, 50);
```

:::

النتيجة في المتصفح ستكون النص الأحمر `Hello HTML Canvas!`.

هناك المزيد مما يمكنك فعله مع واجهة برمجة تطبيقات `Canvas`. على سبيل المثال، يمكنك دمجها مع `requestAnimationFrame` لإنشاء رسوم متحركة مخصصة، تصورات، ألعاب، والمزيد.
