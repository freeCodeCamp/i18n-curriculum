# --description--

俄勒冈车牌由三个字母后跟三位数字组成（每位数字可以是 [0...9] 中的任意一个）。

在开车去上班的路上，Seth 玩以下游戏：

每当他旅途中看到的两个车牌号码相加等于 1000 时，那就是一次胜利。

例如，`MIC-012` 和 `HAN-988` 是一次胜利，`RYU-500` 和 `SET-500` 也是（只要他在同一次行程中看到它们）。

找到他想要获胜的预期牌照数量。 将您的答案四舍五入到小数点后面的小数点后8位。

**注意：** 我们假设每个看到的车牌上的三位数字出现的概率是相等的。
