# --interactive--

المصفوفات في JavaScript ديناميكية، مما يعني أنه يمكنك بسهولة إضافة أو حذف عناصر منها. هناك أربع طرق رئيسية لإضافة وحذف العناصر من بداية ونهاية المصفوفة: `push()`, `pop()`, `shift()`, و `unshift()`. دعنا نستعرض كل طريقة من هذه الطرق بالتفصيل.

تُستخدم الطريقة `push()` لإضافة عنصر أو أكثر إلى نهاية المصفوفة. قيمة الإرجاع لطريقة `push()` هي الطول الجديد للمصفوفة. إليك مثالًا على إضافة ثمرة جديدة إلى مصفوفة `fruits` الموجودة:

:::interactive_editor

```js
const fruits = ["apple", "banana"];
const newLength = fruits.push("orange");
console.log(newLength); // 3
console.log(fruits); // ["apple", "banana", "orange"]
```

:::

في هذا المثال، نبدأ بمصفوفة تسمى `fruits` تحتوي على عنصرين. ثم نستخدم الطريقة `push()` لإضافة السلسلة النصية `orange` إلى نهاية المصفوفة.

ربما لاحظت أننا نستخدم `const` عند إعلان مصفوفة `fruits`. لكن لماذا يمكن إضافة عناصر جديدة إلى هذه المصفوفة `fruits` رغم أن `fruits` ثابتة؟ هذا ممكن لأن إعلان مصفوفة باستخدام الكلمة المفتاحية `const` ينشئ مرجعًا للمصفوفة. بينما المصفوفة نفسها قابلة للتغيير ويمكن تعديلها، لا يمكنك إعادة تعيين قيمة جديدة للثابت `fruits`، مثل هذا:

```js
const fruits = ["apple", "banana"];
fruits = ["This", "will", "not", "work"];
console.log(fruits); // Uncaught TypeError: Assignment to constant variable. 
```

الطريقة التالية التي سننظر إليها هي `pop()`. تزيل طريقة `pop()` العنصر الأخير من المصفوفة وتُرجع هذا العنصر. كما أنها تعدل المصفوفة الأصلية. إليك كيف تعمل:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let lastFruit = fruits.pop();
console.log(fruits); // ["apple", "banana"]
console.log(lastFruit); // "orange"
```

:::

في هذا المثال، نبدأ بمصفوفة تحتوي على ثلاث ثمرات. تزيل طريقة `pop()` العنصر الأخير (`orange`) من المصفوفة وتُرجعه. تم تعديل مصفوفة `fruits` الأصلية لتحتوي على عنصرين فقط.

تضيف طريقة `unshift()` عنصرًا أو أكثر إلى بداية المصفوفة وتُرجع طولها الجديد. تعمل بطريقة مشابهة لـ `push()`، لكنها تعدل بداية المصفوفة بدلًا من نهايتها. إليك مثالًا:

:::interactive_editor

```js
let numbers = [2, 3];
let newLength = numbers.unshift(1);
console.log(numbers); // [1, 2, 3]
console.log(newLength); // 3
```

:::

في هذا المثال، نستخدم `unshift()` لإضافة الرقم `1` إلى بداية مصفوفة `numbers`. تُرجع الطريقة الطول الجديد للمصفوفة، وهو `3`.

أخيرًا، تزيل طريقة `shift()` العنصر الأول من المصفوفة وتُرجع هذا العنصر. تشبه `pop()` لكنها تعمل على بداية المصفوفة بدلًا من نهايتها. إليك كيف تعمل:

:::interactive_editor

```js
let colors = ["red", "green", "blue"];
let firstColor = colors.shift();
console.log(colors); // ["green", "blue"]
console.log(firstColor); // "red"
```

:::

في هذا المثال، نبدأ بمصفوفة تحتوي على ثلاث ألوان. تزيل طريقة `shift()` العنصر الأول (`red`) من المصفوفة وتُرجعه. تم تعديل مصفوفة `colors` الأصلية لتحتوي على عنصرين فقط.

لاحظ أنه بينما يمكن لـ `push()` و `unshift()` إضافة عدة عناصر دفعة واحدة، فإن `pop()` و `shift()` تزيلان عنصرًا واحدًا فقط في كل مرة.
