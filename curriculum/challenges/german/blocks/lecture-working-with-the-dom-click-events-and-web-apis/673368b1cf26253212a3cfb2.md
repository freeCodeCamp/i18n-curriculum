# --description--

Lernen wir die `querySelectorAll()`-Methode kennen. Sie können diese Methode verwenden, um eine Liste aller DOM-Elemente zu erhalten, die einem bestimmten CSS-Selektor entsprechen.

So können Sie die `querySelectorAll()`-Methode aufrufen:

```js
document.querySelectorAll(selectors);
```

Rufen Sie sie auf dem `document`-Objekt auf und übergeben Sie einen String mit den CSS-Selektoren als Argument. Das Argument muss ein gültiger CSS-Selektor-String sein. Andernfalls wird eine `SyntaxError`-Ausnahme geworfen.

`querySelectorAll()` gibt ein `NodeList`-Objekt zurück, eine Sammlung von Knoten, die dem angegebenen CSS-Selektor entsprechen.

Die Liste enthält für jedes Element, das den CSS-Selektoren entspricht, ein `Element`-Objekt. Wenn keine Übereinstimmungen gefunden werden, ist die Liste leer.

Die Elemente sind in der Reihenfolge, in der sie im HTML-Dokument erscheinen. Zum Beispiel können Sie alle Elemente eines bestimmten Typs auswählen:

```js
document.querySelectorAll("div");
```

Alle Elemente mit einer bestimmten Klasse:

```js
document.querySelectorAll(".rounded");
```

Alle Elemente mit einer bestimmten ID:

```js
document.querySelectorAll("#logo");
```

Oder alle Elemente mit einem bestimmten Attribut, wie alle Links, die Benutzer zu einer bestimmten URL führen:

```js
document.querySelectorAll("a[href='https://www.freecodecamp.org/']");
```

Sie können auch komplexere CSS-Selektoren verwenden, wie diesen, der alle Listenelemente innerhalb eines ungeordneten Listen-Elements mit der Klasse `ingredients` auswählt:

```js
document.querySelectorAll("ul.ingredients li");
```

Da die Methode eine Sammlung von Knoten zurückgibt, können Sie diese Sammlung einer Variablen zuweisen und in Ihren Programmen verwenden, um mit den einzelnen Elementen zu arbeiten, wie Sie in diesem Beispiel mit der allgemeinen Syntax sehen:

```js
const matches = document.querySelectorAll(selectors); 
```

Wenn Sie zum Beispiel diese ungeordnete Liste mit drei Listenelementen haben:

```html
<ul class="ingredients">
  <li>Flour</li>
  <li>Cheese</li>
  <li>Water</li>
</ul>
```

Können Sie die `querySelectorAll()`-Methode aufrufen, um alle Listenelemente innerhalb der ungeordneten Liste mit der Klasse `ingredients` auszuwählen und den Rückgabewert der Konstanten `matches` zuzuweisen:

```js
const matches = document.querySelectorAll("ul.ingredients li");
```

Dies gibt eine Sammlung von drei Knoten zurück. Jeder Knoten stellt ein Listenelement dar:

```js
// NodeList(3)
{
  0: `<li>Flour</li>`,
  1: `<li>Cheese</li>`,
  2: `<li>Water</li>`,
  length: 3,
}
```

Sie können mit dieser Sammlung genauso arbeiten wie mit jedem anderen JavaScript-Array.

Zum Beispiel können Sie sie mit `console.log()` in der Konsole ausgeben:

```js
console.log(matches);
```

Sie können auch die Länge der Sammlung mit der Eigenschaft `length` überprüfen:

```js
console.log(matches.length);
```

Die Ausgabe ist eine ganze Zahl, die die Anzahl der Knoten in der Sammlung darstellt:

```md
3
```

Sie können auch auf die einzelnen Elemente der Sammlung mit ihren entsprechenden Indizes zugreifen. Das erste Element befindet sich am Index `0`:

```js
console.log(matches[0]);
console.log(matches[1]);
console.log(matches[2]);
```

Dies ist die Ausgabe:

```html
<li>Flour</li>
<li>Cheese</li>
<li>Water</li>
```

Die Elemente werden einzeln in der Konsole ausgegeben.

Sie können die Elemente auch mit einer `for`-Schleife durchlaufen:

```js
for (let i = 0; i < matches.length; i++) {
  console.log(matches[i]);    
}
```

Die Ausgabe ist in diesem Fall genau dieselbe. Aber das kann sehr nützlich sein, um mit allen Elementen der Sammlung zu arbeiten.

Die `querySelectorAll()`-Methode in JavaScript ist ein mächtiges Werkzeug, um mehrere Elemente basierend auf ihren CSS-Selektoren auszuwählen. Mit dieser Methode können Sie bestimmte Elemente innerhalb Ihrer Webseiten auswählen und bei Bedarf manipulieren.
