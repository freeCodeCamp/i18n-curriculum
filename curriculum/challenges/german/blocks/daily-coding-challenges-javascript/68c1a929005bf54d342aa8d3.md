# --description--

Für den zweiten Tag der Space Week erhalten Sie einen String, bei dem jedes Zeichen die Helligkeitsmessung eines Sterns darstellt. Bestimmen Sie, ob die Messungen mit der Transit-Methode einen Exoplaneten entdeckt haben. Die Transit-Methode ist, wenn ein Planet vor einem Stern vorbeizieht und dadurch dessen beobachtete Helligkeit verringert.

- Die Helligkeitsmessungen bestehen nur aus den Zeichen `0-9` und `A-Z`, wobei jede Messung den folgenden numerischen Werten entspricht:
- Die Zeichen `0-9` entsprechen Helligkeitsstufen `0-9`.
- Die Zeichen `A-Z` entsprechen Helligkeitsstufen `10-35`.

Ein Stern gilt als mit einem Exoplaneten versehen, wenn eine einzelne Messung kleiner oder gleich 80 % des Durchschnitts aller Messungen ist. Zum Beispiel gilt ein Stern mit einer durchschnittlichen Helligkeit von 10 als mit einem Exoplaneten versehen, wenn eine einzelne Messung 8 oder weniger beträgt.

```js
function hasExoplanet(readings) {
  let total = 0;
  const values = readings.split('').map(c => parseInt(c, 36));
  
  total = values.reduce((sum, v) => sum + v, 0);
  const average = total / values.length;
  const threshold = average * 0.8;

  return values.some(v => v <= threshold);
}
```
