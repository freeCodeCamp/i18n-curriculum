# --description--

Нехай $f(n) = n^2 - 3n - 1$.

Нехай $p$ — просте число.

Нехай $R(p)$ — найменше додатне ціле число $n$, для якого $f(n)\bmod p^2 = 0$, якщо таке число $n$ існує, інакше $R(p) = 0$.

Нехай $SR(L)$ — це $\sum R(p)$ для всіх простих чисел, що не перевищують $L$.

Знайдіть $SR({10}^7)$.
