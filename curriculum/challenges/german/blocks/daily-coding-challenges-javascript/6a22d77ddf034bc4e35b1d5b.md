# --description--

Gegeben sind zwei Arrays, die RGB-Werte darstellen, und ein Boolean, der angibt, ob der Text groß ist. Geben Sie die WCAG-Kontrastbewertung mit der folgenden Methode zurück:

Konvertieren Sie zuerst jeden RGB-Wert in die relative Leuchtdichte:

- Teilen Sie jeden Kanal `[R, G, B]` durch 255, um einen Wert zwischen `0` und `1` zu erhalten
- Wenden Sie die Gamma-Korrekturformel auf jeden Kanal an:
  - Wenn der Kanalwert kleiner oder gleich `0.04045` ist: `channel / 12.92`
  - Andernfalls: `((channel + 0.055) / 1.055) ^ 2.4`
- Berechnen Sie die Leuchtdichte: `0.2126 * R + 0.7152 * G + 0.0722 * B`

Berechnen Sie dann das Kontrastverhältnis, indem Sie zu jedem Leuchtdichtewert 0,05 addieren und anschließend den helleren Wert durch den dunkleren teilen. Der hellere Wert ist immer das erste Argument.

Geben Sie die Bewertung basierend auf dem Kontrastverhältnis mit der folgenden Tabelle zurück:

| Bewertung | Normaler Text | Großer Text |
|--------|-------------|------------|
| `"AAA"` | 7,0+ | 4,5+ |
| `"AA"` | 4,5+ | 3,0+ |
| `"Fail"` | unter 4,5 | unter 3,0 |

```js
function getContrastRating(rgb1, rgb2, isLargeText) {
  function toLuminance([r, g, b]) {
    return [r, g, b].map(channel => {
      channel = channel / 255;
      return channel <= 0.04045
        ? channel / 12.92
        : ((channel + 0.055) / 1.055) ** 2.4;
    }).reduce((sum, c, i) => sum + c * [0.2126, 0.7152, 0.0722][i], 0);
  }

  const l1 = toLuminance(rgb1);
  const l2 = toLuminance(rgb2);
  const ratio = (l1 + 0.05) / (l2 + 0.05);

  if (isLargeText) {
    if (ratio >= 4.5) return "AAA";
    if (ratio >= 3.0) return "AA";
  } else {
    if (ratio >= 7.0) return "AAA";
    if (ratio >= 4.5) return "AA";
  }
  return "Fail";
}
```
