# --description--

我们用 `do` 询问日常事务。

使用组助动词 `do` 构成疑问句时，主要动词应保持一般形式不变。 例如：

- 语句：`She works as a full-stack dev.`

- 问题：`Does she work as a full-stack dev?`

请注意，上述问题中即使是询问第三人称单数主语如 `he`、`she` 或 `it`，`work` 也不会变成 `works`。

# --questions--

## --text--

选择正确使用 `do` 询问日常活动疑问句。

## --answers--

`Do you write code every day?`

---

`Do you writes code every day?`

### --feedback--

请记住，`do` 之后，主要动词不应以 `-s` 结尾。

---

`Writes you code every day?`

### --feedback--

请记住，对于这类问题你需要一个辅助动词。同时，注意词序。

---

`You do write code every day?`

### --feedback--

此选项误在陈述句中使用 `do`，应在疑问句中使用。
