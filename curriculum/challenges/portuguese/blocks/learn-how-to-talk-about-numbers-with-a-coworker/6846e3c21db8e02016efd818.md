`third phase`

### --feedback--

A primeira palavra é um número ordinal e a segunda se refere a uma etapa no projeto.

# --explanation--

`By when...` é comumente usado para perguntar quando algo será concluído. Esta é uma maneira educada de pedir uma estimativa de tempo ou prazo. Por exemplo:

`By when do you need the report?` — Qual é o prazo máximo para você querer o relatório?

`Third` é um número ordinal, que indica a posição em uma sequência. `Phase` significa uma etapa de um projeto, então `third phase` se refere à terceira grande parte do plano do projeto deles.
