# --description--

Um eine Datei unter Windows mit dem Explorer zu erstellen, können Sie auf die Option „Neu“ oben links klicken. Dadurch wird Ihnen eine Liste der verschiedenen Dateitypen angezeigt, die Sie erstellen können.

Sie können Dateien auch in den entsprechenden Anwendungen erstellen. Lassen Sie uns eine Textdatei erstellen, um Ihnen den Vorgang zu zeigen.

Wenn Sie auf „Neu“ klicken und den Dateityp auswählen, sehen Sie eine neue Textdatei in Ihrem aktuellen Ordner mit einem Standardnamen. Sie können sie bei Bedarf umbenennen.

Der Vorgang zum Erstellen von Ordnern ist genau derselbe. Klicken Sie oben links auf die Schaltfläche „Neu“ und wählen Sie dann die Option „Ordner“ oder klicken Sie mit der rechten Maustaste auf eine freie Stelle und gehen Sie zu „Neu“. Wählen Sie dann die Option „Ordner“. Sie sehen einen neuen leeren Ordner an Ihrem aktuellen Speicherort.

Sie können Dateien und Ordner umbenennen, indem Sie sie auswählen und auf die Option „Umbenennen“ oben klicken. Wenn der aktuelle Name der Datei markiert ist, können Sie den neuen Namen der Datei eingeben. Drücken Sie Enter, um die Änderungen zu bestätigen.

Alternativ können Sie mit der rechten Maustaste auf die Datei oder den Ordner klicken und die Option „Umbenennen“ auswählen oder die Tastenkombination F2 verwenden. Schreiben Sie den neuen Namen und drücken Sie Enter, um die Änderungen zu bestätigen.

Sie können eine Datei oder einen Ordner in einen anderen Ordner verschieben, indem Sie sie in den Zielordner ziehen und dort ablegen, wenn Sie diesen als Tab geöffnet haben.

Sie können auch die Datei oder den Ordner auswählen und oben die Option „Ausschneiden“ wählen.

Gehen Sie dann zum Zielordner und fügen Sie sie ein, indem Sie oben auf die Schaltfläche „Einfügen“ klicken. Wenn Sie die Datei zuvor kopiert oder ausgeschnitten haben, ist diese Schaltfläche aktiviert.

Sie können dies auch tun, indem Sie mit der rechten Maustaste auf eine freie Stelle klicken und dann auf das Einfügesymbol klicken. Dadurch wird die Datei oder der Ordner aus dem ursprünglichen Speicherort entfernt.

Um eine Datei oder einen Ordner zu löschen, klicken Sie darauf und dann auf das Papierkorb-Symbol oben. Wenn Sie mit der rechten Maustaste darauf klicken, finden Sie auch die Option „Löschen“.

Gut. Schauen wir uns nun an, wie Sie dies auf macOS mit Finder machen können.

Finder ist der Standard-Dateimanager auf macOS.

Um eine Datei auf macOS zu erstellen, müssen Sie eine Anwendung öffnen, mit der Sie den Typ des Dokuments erstellen können, den Sie benötigen. Derzeit gibt es keine eingebaute Möglichkeit, eine Datei direkt im Finder zu erstellen.

Zum Beispiel können Sie, um eine einfache Textdatei zu erstellen, TextEdit öffnen und Ihre Datei speichern. Der Vorgang funktioniert für jeden Dateityp genau gleich.

Es gibt viele Möglichkeiten, TextEdit zu öffnen. Eine davon ist, zum Launchpad zu gehen. Suchen Sie dann nach einer App, mit der Sie den Dateityp erstellen können, der Sie interessiert. In diesem Fall wäre das „TextEdit“. Klicken Sie auf das Symbol, um die App zu öffnen.

Sobald Sie in der App sind, sollten Sie im Menü „Datei“ eine Option zum Speichern Ihrer Datei sehen. Dieses Menü befindet sich oben links, neben dem Apple-Menü-Symbol.

In einem der App-Menüoptionen sollten Sie einen Befehl zum Speichern der Datei sehen. Dieser befindet sich normalerweise im Menü „Datei“.

Daneben sehen Sie die Tastenkombination zum Speichern Ihrer Datei. Diese ist auf macOS normalerweise `Command + S`.

Wenn Sie die Datei zum ersten Mal speichern, sollten Sie ein Dialogfenster sehen, in dem Sie den Namen der Datei eingeben und einen Speicherort auswählen können.

Normalerweise wird die Dateiendung von der Anwendung automatisch hinzugefügt, aber Sie können sie vor dem Speichern ändern.

Nachdem Sie die Datei gespeichert haben, können Sie den Finder öffnen, indem Sie auf das Finder-Symbol im Dock klicken. Sie sollten Ihre neue Datei in dem Ordner sehen, in dem Sie sie gespeichert haben.

Um einen neuen Ordner zu erstellen, gehen Sie an den Ort, an dem Sie den neuen Ordner erstellen möchten, und klicken Sie mit der rechten Maustaste auf eine freie Stelle. Sie sehen eine Liste von Optionen. Die erste Option ist „Neuer Ordner“.

Wenn Sie darauf klicken, können Sie Ihrem neuen Ordner einen Namen zuweisen. Schreiben Sie den Namen und drücken Sie Enter, um zu bestätigen.

Es gibt mehrere Möglichkeiten, Dateien und Ordner im Finder zu verschieben. Wenn Sie sie in einen Ordner verschieben müssen, der im Bereich „Favoriten“ Ihres Finders aufgeführt ist, können Sie sie einfach per Drag & Drop verschieben.

Wenn Sie sie in einen anderen Ordner verschieben müssen, haben Sie zwei Optionen.

Sie können entweder den Zielordner als Tab öffnen und die Datei oder den Ordner in diesen Tab ziehen und dort ablegen.

Oder Sie verwenden Tastenkombinationen. Kopieren Sie zuerst die Datei mit `Command + C`, gehen Sie zum neuen Ordner und verwenden Sie dann `Command + Option + V`, um die Datei oder den Ordner an diesem Ort einzufügen. Dadurch wird sie auch aus dem ursprünglichen Ordner entfernt.

Um eine Datei oder einen Ordner zu löschen, können Sie mit der rechten Maustaste darauf klicken und „In den Papierkorb legen“ auswählen.

Sie können sie auch per Drag & Drop in das Papierkorb-Symbol im Dock ziehen. Das ist gleichwertig.

Wenn sich die Datei oder der Ordner im Papierkorb befindet, können Sie mit der rechten Maustaste darauf klicken, um sie sofort zu löschen, oder Sie können den Papierkorb leeren, um alle gelöschten Dateien und Ordner dauerhaft zu entfernen.

Zu wissen, wie man Dateien und Ordner unter Windows und macOS erstellt, verschiebt und löscht, ist sehr wichtig. Wenn Sie diese Fähigkeiten beherrschen, können Sie Ihre Dateien organisieren, sie schnell finden und Speicherplatz auf Ihrem Computer freigeben, indem Sie unnötige Dateien löschen.
