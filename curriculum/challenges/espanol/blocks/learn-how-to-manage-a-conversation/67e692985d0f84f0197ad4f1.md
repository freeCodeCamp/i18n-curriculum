# --explanation--

`To address` significa tomar acción para tratar un problema o situación. Por ejemplo:

`We need to address the login error before users report it.` – Esto significa que se necesita tomar acción para solucionar el problema.

`Quickly` expresa qué tan rápido se hace algo. `Quick`, por otro lado, simplemente describe algo como rápido. Por ejemplo:

- `She responded quickly to the email.` – Usas `quickly` para describir qué tan rápido respondió al correo electrónico.

- `It was a quick meeting.` – Aquí, estás describiendo la reunión como que tomó un tiempo corto y razonable.
