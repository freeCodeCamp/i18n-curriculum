# --description--

포물선 $y = \frac{x^2}{k}$ 위에 세 점 $A(a, \frac{a^2}{k})$, $B(b, \frac{b^2}{k})$, $C(c, \frac{c^2}{k})$가 선택되었습니다.

$F(K, X)$를 삼각형 $ABC$의 적어도 한 각이 45°인 정수 네 쌍 $(k, a, b, c)$의 개수로 정의하며, 여기서 $1 ≤ k ≤ K$이고 $-X ≤ a < b < c ≤ X$입니다.

예를 들어, $F(1, 10) = 41$이고 $F(10, 100) = 12\\,492$입니다.

$F({10}^6, {10}^9)$를 구하세요.
