# --description--

1列に $N$ 席あります。$N$ 人が次々に以下のルールに従って席に座ります。

1. どの人も隣り合って座りません。
1. 最初の人は任意の席を選びます。
1. それ以降の人は、すでに座っている人から最も遠い席を選びます。ただしルール1を破らない範囲で選びます。条件を満たす席が複数ある場合は、左端の席を選びます。

ルール1のため、必ず座らない席ができ、$N > 1$ の場合、座れる最大人数は $N$ より少なくなります。

$N = 15$ の場合の座り方の例を示します。

<img alt="seating arrangements for N = 15" src="https://cdn.freecodecamp.org/curriculum/project-euler/comfortable-distance-ii.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

最初の人が正しく選べば、15席に最大7人座れます。また、最初の人が座る席の選択肢は最大人数を実現するために9通りあります。

$f(N)$ を、$N$ 席の列で最大人数を実現するために最初の人が選べる席の数とします。つまり、$f(1) = 1$、$f(15) = 9$、$f(20) = 6$、$f(500) = 16$ です。

また、$1 ≤ N ≤ 20$ の範囲で $\sum f(N) = 83$、$1 ≤ N ≤ 500$ の範囲で $\sum f(N) = 13\,343$ となります。

$1 ≤ N ≤ {10}^{12}$ の範囲での $\sum f(N)$ を求めてください。答えの下8桁を出力してください。
