# --interactive--

In CSS ermöglicht die Positionierung, wie Elemente auf einer Seite angeordnet werden. Zwei gängige Arten der Positionierung sind statische Positionierung und relative Positionierung. Standardmäßig sind Elemente statisch positioniert. Das bedeutet, sie folgen dem normalen Fluss des Dokuments, nacheinander von oben nach unten, von links nach rechts.

Die statische Positionierung ist die Standardpositionierung für alle Elemente und benötigt keine spezielle Deklaration in CSS. Sie werden keinen Unterschied bemerken, wenn Sie statische Positionierung verwenden, da sie die Elemente einfach dort belässt, wo sie natürlich im Dokument vorkommen. Hier ist ein Beispiel für einen Absatz mit standardmäßiger statischer Positionierung:

:::interactive_editor

```html
<p>This paragraph is statically positioned.</p>
```

:::

Wenn dieser Absatz im Browser gerendert wird, erscheint er an seiner natürlichen Position und folgt dem normalen Dokumentfluss. Es wird keine spezielle Formatierung angewendet, um ihn zu verschieben, und er bleibt dort, wo er im Layout natürlich erscheinen würde.

Relative Positionierung hingegen erlaubt es, ein Element von seiner normalen Position zu verschieben, ohne den Fluss des Dokuments zu stören. Man kann sich das vorstellen wie das Verschieben des Elements von seiner standardmäßigen statischen Position, indem man ihm neue Koordinaten gibt. So könnten Sie relative Positionierung anwenden:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="relative">This paragraph is positioned relatively.</p>
```

```css
body {
  border: solid 1px black;
}

.relative {
  position: relative;
  top: 30px;
  left: 30px;
}
```

:::

In diesem Beispiel erscheint der Absatz `30px` nach unten und `30px` nach rechts von seiner ursprünglichen Position. Der Platz, den er im normalen Fluss eingenommen hätte, bleibt jedoch erhalten. Relative Positionierung ist sehr nützlich, wenn Sie ein Element leicht verschieben möchten, ohne den Rest des Layouts zu beeinflussen.
