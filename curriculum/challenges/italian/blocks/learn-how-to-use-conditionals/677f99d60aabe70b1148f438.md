# --explanation--

Il tempo `Present Simple` viene spesso usato per descrivere stati, fatti o condizioni che sono generalmente veri. Questo tempo è usato perché la corruzione del BIOS è considerata una condizione stabile o immutabile quando il computer tenta di avviarsi.

`Is corrupted` significa danneggiato o modificato in modo che smetta di funzionare correttamente. Per esempio:

`The file is corrupted and won't open.` - Il file è danneggiato e non può essere consultato.
