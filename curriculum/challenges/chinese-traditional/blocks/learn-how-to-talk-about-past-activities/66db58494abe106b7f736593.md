# --explanation--

`Across` 是一個介系詞，表示從一側到另一側或遍及特定區域或群組。 例如：

- 消息迅速傳遍全國。

- `She looked across the room.`－她從房間的一側看向另一側。

在句子中，`Are all updated elements performing well across devices?`，`across` 意思是在所有或不同裝置中。
