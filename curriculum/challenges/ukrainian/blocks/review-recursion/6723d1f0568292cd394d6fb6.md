# --description--

- Рекурсія — це концепція програмування, яка дозволяє викликати функцію повторно, доки не буде досягнуто базового випадку.

Ось приклад рекурсивної функції, яка обчислює факторіал числа:

```js
function findFactorial(n) {
  if (n === 0) {
    return 1;
  }
  return n * findFactorial(n - 1);
}
```

У наведеному прикладі функція `findFactorial` викликається рекурсивно, доки `n` не досягне `0`. Коли `n` дорівнює `0`, досягається базовий випадок, і функція повертає `1`. Потім функція повертає добуток `n` та результату рекурсивного виклику `findFactorial(n - 1)`.

- Рекурсія дозволяє опрацьовувати структури з невідомою глибиною, наприклад, глибоко вкладені об’єкти/масиви або дерево файлів.
- Для відстеження викликів функцій у рекурсивній функції використовується стек викликів. Кожного разу, коли функція викликається, вона додається до стека викликів. Коли досягається базовий випадок, виклики функцій видаляються зі стека.
- Ви повинні ретельно визначити базовий випадок, оскільки безкінечний виклик може призвести до збою коду. Це відбувається тому, що рекурсія накопичує все більше викликів функцій, доки система не вичерпає пам’ять.
- Рекурсія застосовується для розв’язання математичних задач, таких як факторіал і послідовність Фібоначчі, обходу дерев і графів, генерації перестановок і комбінацій та багато іншого.
