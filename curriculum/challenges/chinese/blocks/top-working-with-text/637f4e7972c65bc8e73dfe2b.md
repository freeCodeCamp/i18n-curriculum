# --description--

你可能已经注意到，在本课的所有示例中，你都会对位于其他元素内的任何元素进行缩排。这被称为嵌套元素。

当你在其他元素内嵌套元素时，你创建了它们之间的父子关系。嵌套的元素是子元素，而它们所嵌套的元素是父元素。

在以下示例中，`body` 元素是父元素，`p` 是子元素：

```html
<html>
  <head>
  </head>
  <body>
    <p>Lorem ipsum dolor sit amet.</p>
  </body>
 </html>
```

就像在人际关系中一样，超文本标记语言父元素可以有许多子元素。处于相同嵌套级别的元素被视为兄弟元素。

例如，下面代码中的两个 `p` 元素是兄弟元素，因为它们都是 `body` 标签的子元素，并且处于相同的嵌套层级：

```html
<html>
  <head>
  </head>
  <body>
    <p>Lorem ipsum dolor sit amet.</p>
    <p>Ut enim ad minim veniam.</p>
  </body>
 </html>
```

你使用缩排来使嵌套层级对你自己和将来处理你 `HTML` 的其他开发者来说清晰且易读。建议将任何子元素缩排两个空格。

元素之间的父元素、子元素和兄弟元素关系在你开始用 CSS 样式化你的 HTML 并用 JavaScript 添加行为时将变得更加重要。然而，目前只需了解元素之间关系的区别以及描述这些关系所用的术语即可。
