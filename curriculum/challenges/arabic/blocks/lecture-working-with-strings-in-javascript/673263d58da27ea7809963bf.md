# --interactive--

في JavaScript، تُعد القوالب الحرفية طريقة قوية ومرنة للعمل مع السلاسل النصية. على عكس السلاسل النصية العادية التي تستخدم علامات اقتباس مفردة (`'`) أو علامات اقتباس مزدوجة (`"`)، تُعرف القوالب الحرفية باستخدام علامات دقيقة (`` ` ``).

تسمح بتسهيل التعامل مع السلاسل النصية، بما في ذلك تضمين المتغيرات مباشرة داخل السلسلة النصية، وهي ميزة تُعرف باسم استيفاء السلاسل النصية.

تُسهل القوالب الحرفية إنشاء سلاسل نصية تمتد عبر عدة أسطر أو تضم تعبيرات (مثل المتغيرات أو حتى شفرة JavaScript) مباشرة داخل السلسلة.

إليك مثالًا على قالب حرفي:

:::interactive_editor

```js
const name = "Alice";
const greeting = `Hello, ${name}!`;

console.log(greeting);
```

:::

لاحظ استخدام العلامات الدقيقة بدلًا من علامات الاقتباس المفردة أو المزدوجة. صيغة `${name}` هي مثال على استيفاء السلاسل النصية، حيث تُدرج قيمة المتغير `name` مباشرة داخل السلسلة.

يسمح استيفاء السلاسل النصية بتضمين المتغيرات والتعبيرات داخل السلسلة النصية. هذا تحسن كبير مقارنة بالطريقة القديمة التي كنت تجمع فيها السلاسل النصية والمتغيرات باستخدام العامل `+`.

إليك مثالًا يستخدم جمع السلاسل النصية والعامل زائد (`+`):

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = "My name is " + name + " and I am " + age + " years old.";
console.log(message); 
```

:::

وهنا مثال يستخدم القوالب الحرفية واستيفاء السلاسل النصية:

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = `My name is ${name} and I am ${age} years old.`;
console.log(message); 
```

:::

كما ترى، استيفاء السلاسل النصية مع القوالب الحرفية أنظف وأسهل قراءة، خاصة عند التعامل مع عدة متغيرات.

ميزة رائعة أخرى للقوالب الحرفية هي دعمها للسلاسل النصية متعددة الأسطر. مع السلاسل النصية العادية، تحتاج لاستخدام أحرف الهروب (`\n`) لإنشاء أسطر جديدة. مع القوالب الحرفية، يمكنك ببساطة كتابة السلسلة عبر عدة أسطر، ويُحفظ التنسيق:

:::interactive_editor

```js
let poem = `Roses are red,
Violets are blue,
JavaScript is fun,
And so are you.`;

console.log(poem);
```

:::

ميزة أخرى للقوالب الحرفية هي أنها تسمح بتضمين تعبيرات JavaScript مباشرة داخل السلسلة، كما في هذا المثال:

:::interactive_editor

```js
const song = "Bohemian Rhapsody";
const score = 9.5;
const highestScore = 10;
const output = `One of my favorite songs is "${song}". I rated it ${
  (score / highestScore) * 100
}%.`;
console.log(output); 
```

:::

تُعد القوالب الحرفية مفيدة بشكل خاص عندما تحتاج إلى تضمين متغيرات أو تعبيرات في السلاسل النصية، أو تنسيق سلاسل نصية معقدة، أو العمل مع نص متعدد الأسطر. هي أكثر إيجازًا وأسهل قراءة مقارنة بجمع السلاسل النصية التقليدي.
