# --description--

`To be into...` bedeutet, an etwas interessiert zu sein oder etwas zu mögen.

`These` ist ein Demonstrativpronomen, das sich auf bestimmte Gegenstände oder Ideen bezieht, die zuvor im Gespräch erwähnt wurden.

Wenn Sarah fragt, `Are you into these activities?`, verwendet sie `these`, um auf etwas zurückzuverweisen, das Tom gerade erwähnt hat. So kann sie genau angeben, welche Aktivitäten sie meint, ohne den ganzen Ausdruck zu wiederholen.
