# --description--

每个行业，从音乐到医疗，都处理大量且通常复杂的 `datasets`。这些可以包括患者医疗 `records`、随时间变化的播放列表趋势，或社交媒体参与度指标。

为了理解这些信息，企业使用各种工具来帮助他们的团队分析数据，并就他们的产品和服务做出明智的决策。

理解和传达这些数据最强大的工具之一是**数据可视化**。

数据可视化将复杂数据转换为清晰的视觉形式。这些视觉形式的示例包括饼图、散点图、条形图等。良好的可视化帮助企业节省时间，识别数据中的关键信息、趋势和关系。

在商业中使用许多流行的数据可视化工具，包括 Microsoft Excel、Power BI、Google Charts 和 Tableau。

在接下来的课程和研讨会中，你将学习如何使用 D3.js JavaScript `library` 创建交互式可视化。

这个模块的目标是向你介绍数据可视化的基础知识，并指导你使用 JavaScript 创建你自己的可视化。
