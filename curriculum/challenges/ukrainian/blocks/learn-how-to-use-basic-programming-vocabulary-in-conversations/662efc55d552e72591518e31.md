# --description--

A `function` — це множина інструкцій, які виконують конкретне завдання. `Functions` можуть зробити код більш організованим, багаторазовим і легшим для читання. Наприклад, `function` може обчислювати суму двох чисел або відображати повідомлення користувачу.

Уявіть `function` як маленьку машину на фабриці: ви даєте їй деякі вхідні дані, вона щось робить із цими даними, а потім повертає вам вихідні дані.
