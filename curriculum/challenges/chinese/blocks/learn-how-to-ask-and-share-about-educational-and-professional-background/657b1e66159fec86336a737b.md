# --description--

通常，`Past Simple` 时态的形式是为规则动词添加 `-ed`。例如，动词 `play` 变成 `played`。

`play around` 意味着以随意或实验的方式尝试某事，通常只是为了看看会发生什么。它通常后接 `with`。例如：

`He played around with the new software to see how it works.`——这意味着他没有认真计划，只是为了探索而尝试使用该软件。
