# --description--

Barrierefreiheit ist ein entscheidender, aber oft übersehener Aspekt digitaler Inhalte. Wenn Sie Ihre digitalen Inhalte barrierefrei gestalten, ist es wichtig sicherzustellen, dass sie den Barrierefreiheitsstandards entsprechen.

Ein Tool zur Barrierefreiheitsprüfung ist eine Anwendung, die Ihnen hilft, die Barrierefreiheit Ihrer digitalen Inhalte zu verbessern, indem es Barrierefreiheitsprobleme meldet, die durch automatisierte Tests leicht gefunden werden können. Diese Inhalte umfassen Websites, Webanwendungen und mobile Apps.

Es ist wichtig zu beachten, dass automatisierte Barrierefreiheitstools zwar eine Rolle bei der Verbesserung der Barrierefreiheit spielen, sie jedoch in der Regel nur etwa ein Drittel aller möglichen Barrierefreiheitsprobleme finden. Daher ist es wichtig, sich nicht ausschließlich auf sie zu verlassen, um die Barrierefreiheit Ihrer Inhalte zu bewerten. Manuelle Tests, vorzugsweise von Menschen mit Behinderungen, sind immer erforderlich, um sicherzustellen, dass Ihre Inhalte so barrierefrei wie möglich sind.

Schauen wir uns einige kostenlose Tools an, die Ihnen helfen können, die Barrierefreiheit Ihrer digitalen Inhalte zu verbessern.

Google Lighthouse ist ein beliebter Web-Metrik-Checker, den Sie direkt in den Chrome Entwicklerwerkzeugen oder online verwenden können. Das bedeutet, Sie können nicht nur Live-Websites, sondern auch lokal entwickelte Websites überprüfen.

Die Metriken, die Sie überprüfen können, umfassen Barrierefreiheit, SEO, bewährte Vorgehensweisen und Leistung.

Um Lighthouse zu verwenden, öffnen Sie Ihre Entwicklerwerkzeuge, indem Sie `F12` drücken und zum Tab „Lighthouse“ wechseln.
Wählen Sie die Metriken aus, die Sie überprüfen möchten, wählen Sie das Gerät aus, auf dem Sie testen wollen, und klicken Sie auf die Schaltfläche „Seitenladezeit analysieren“.

Nach Abschluss der Prüfung erscheint eine Barrierefreiheitsbewertung zusammen mit einer Liste aller Probleme, die behoben werden müssen.

Wenn Sie zuverlässigere Metriken wünschen, sollten Sie die Webversion verwenden. Der Nachteil ist, dass sie keine lokalen Websites testen kann. Sie können auf die Webversion unter `pagespeed.web.dev` zugreifen.

WAVE ist ein weiteres zuverlässiges Tool zur Barrierefreiheitsprüfung, das Sie als Chrome-Erweiterung oder im Web verwenden können. Alles, was Sie tun müssen, ist die URL Ihrer Website einzugeben, und es wird ein umfassender Barrierefreiheitsbericht für Sie erstellt. Dieser Bericht enthält implementierte Barrierefreiheitsfunktionen, ARIA und Kontraste.

Der IBM Equal Access Accessibility Checker ist ein weiteres leistungsfähiges Tool zur Verbesserung der Barrierefreiheit digitaler Inhalte. Damit können Sie Ihre Websites auf Barrierefreiheitsprobleme scannen und einen detaillierten Bericht erstellen.

Sie können ihn als Chrome-Erweiterung oder Firefox-Add-on verwenden.

Um den IBM Accessibility Checker als Chrome-Erweiterung zu verwenden, laden Sie ihn aus dem Chrome Web Store herunter. Öffnen Sie Ihre Entwicklerwerkzeuge, indem Sie `F12` drücken und den Tab „Accessibility Checker“ im Elements-Panel auswählen. Klicken Sie auf die Scan-Schaltfläche, um die Prüfung zu starten, und es wird ein Bericht für Sie erstellt. Sie können den Bericht als Tabellenkalkulation und HTML-Datei exportieren, indem Sie auf die Schaltfläche „Export XLS“ klicken.

Bitte beachten Sie, dass diese automatisierten Tools zwar helfen, Ihre Inhalte barrierefreier zu machen, eine perfekte Bewertung bei keinem von ihnen bedeutet, dass Ihre Inhalte vollständig barrierefrei sind. Der Umfang der von diesen Tools getesteten Probleme ist begrenzt, und manuelle Tests sind immer erforderlich, um eine barrierefreiere Erfahrung für alle zu gewährleisten.
