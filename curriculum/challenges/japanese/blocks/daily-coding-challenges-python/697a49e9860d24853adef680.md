# --description--

カーリングのラウンド終了時の「ハウス」を表す5x5の行列が与えられたとき、どのチームが得点し、何点獲得するかを判定してください。

The layout:

- 中央のセル（インデックス `[2, 2]`）が「ボタン」です。
- ボタンを直接囲む8つのセルがリング1を表します。
- ハウスの外周の16セルがリング2を表します。

与えられた行列では:

- `"."` は空きスペースを表します。
- `"R"` は赤いストーンがあるスペースを表します。
- `"Y"` は黄色いストーンがあるスペースを表します。

Scoring rules:

- 1ラウンドで得点できるのは1チームだけです。
- ボタンに最も近いストーンを持つチームが得点します。
- 得点チームは、相手チームの最も近いストーンよりもボタンに近い自分のストーン1つにつき1点獲得します。
- リング番号が小さいほど中心に近いことを表します。
- 両チームの最も近いストーンが同じ距離の場合は、どちらのチームも得点しません。

Return:

- `"team: number_of_points"` の形式の文字列を返してください。例: `"R: 2"`。
- どちらのチームも得点しなかった場合は `"No points awarded"` を返してください。

例えば、次のような場合：

```js
[
  [".", ".", "R", ".", "."],
  [".", "R", ".", ".", "."],
  ["Y", ".", ".", ".", "."],
  [".", "R", ".", ".", "."],
  [".", ".", ".", ".", "."]
]
```

`"R: 2"` を返してください。リング1の赤いストーン2つが最も近く、黄色の最も近いストーンよりも近いため、この2つだけが得点対象です。

```py
def score_curling(house):
  stones = {'R': [], 'Y': []}
  
  for i in range(5):
    for j in range(5):
      cell = house[i][j]
      if cell == 'R' or cell == 'Y':
        distance = max(abs(i - 2), abs(j - 2))
        stones[cell].append(distance)
  
  red_closest = min(stones['R']) if stones['R'] else float('inf')
  yellow_closest = min(stones['Y']) if stones['Y'] else float('inf')
  
  if red_closest == float('inf') and yellow_closest == float('inf'):
    return "No points awarded"
  
  if red_closest == yellow_closest:
    return "No points awarded"
  
  if red_closest < yellow_closest:
    winner = 'R'
    opponent_closest = yellow_closest
    winning_stones = stones['R']
  else:
    winner = 'Y'
    opponent_closest = red_closest
    winning_stones = stones['Y']
  
  points = sum(1 for d in winning_stones if d < opponent_closest)
  
  return "No points awarded" if points == 0 else f"{winner}: {points}"
```
