# --description--

الفعل `encounter` يعني مقابلة أو مواجهة شيء، خاصة شيء غير متوقع. في البرمجة، يعني `encountering` مشكلة أو خطأ برمجي مواجهة مشكلة أثناء تشغيل الشفرة.

على سبيل المثال، `You might encounter an error if the user inputs the wrong type of data.` تعني هذه الجملة أنك قد تواجه خطأً بشكل غير متوقع أو تضطر للتعامل معه.
