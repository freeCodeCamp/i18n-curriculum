# --description--

L’iperesponenziazione o tetration di un numero $a$ per un intero positivo $b$, indicata con $a↑↑b$ o ${}^ba$, è definita ricorsivamente da:

$a↑↑1 = a$,

$a↑↑(k+1) = a^{(a↑↑k)}$.

Quindi, ad esempio, $3↑↑2 = 3^3 = 27$, perciò $3↑↑3 = 3^{27} = 7625597484987$ e $3↑↑4$ è circa ${10}^{3.6383346400240996 \times {10}^{12}}$. Trova le ultime 8 cifre di $1777↑↑1855$.

# --hints--

`hyperexponentation()` dovrebbe restituire `95962097`.

```js
assert.strictEqual(hyperexponentation(), 95962097);
```
