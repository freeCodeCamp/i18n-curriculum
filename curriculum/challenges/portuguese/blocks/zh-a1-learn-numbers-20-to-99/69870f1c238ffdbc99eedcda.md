| Número | Chinês |
| :--- | :--- |
| 0 | `零 (líng)` |
| 1 | `一 (yī)` |

| 3 | `三 (sān)` |
| 4 | `四 (sì)` |

| 6 | `六 (liù)` |
| 7 | `七 (qī)` |
| 8 | `八 (bā)` |
| 9 | `九 (jiǔ)` |
