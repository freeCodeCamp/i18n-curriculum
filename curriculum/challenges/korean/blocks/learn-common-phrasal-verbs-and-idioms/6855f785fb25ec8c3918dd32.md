# --explanation--

`To prompt`는 누군가가 무언가를 하거나 응답하도록 유도하거나 격려하는 것을 의미합니다. 예를 들어:

`The app prompts users to update their settings.` – 이 말은 앱이 사용자에게 행동을 취하도록 상기시키거나 요청한다는 뜻입니다.

무언가가 `unique`라는 것은 독특하거나 다른 어떤 것과도 다르다는 뜻입니다. 예를 들어:

`Each user gets a unique code for account activation.` – 이 말은 코드가 사람마다 다르다는 뜻입니다.
