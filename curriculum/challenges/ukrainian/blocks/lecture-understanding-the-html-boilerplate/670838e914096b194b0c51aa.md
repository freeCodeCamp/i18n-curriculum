# --description--

Давайте дізнаємося про шаблонний код HTML.

Що таке шаблонний код HTML, запитаєте ви? Це ніби готовий шаблон для ваших вебсторінок. Уявіть його як фундамент будинку. Шаблонний код містить базову структуру та необхідні елементи, які потрібні кожному HTML-документу. Він економить ваш час і допомагає правильно налаштувати сторінки. Ось приклад:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>freeCodeCamp</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
  </body>
</html>
```

Розглянемо ключові частини цього шаблону. Спершу йде оголошення `DOCTYPE`:

```html
<!DOCTYPE html>
```

Воно повідомляє браузерам, яку версію HTML ви використовуєте. Далі йде тег `html`:

```html
<!DOCTYPE html>
<html lang="en">
  <!--All other elements go inside here-->
</html>
```

Він охоплює весь ваш вміст і може вказувати мову вашої сторінки. Всередині тегу `html` ви знайдете два основні розділи — `head` і `body`:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <!--Important metadata goes here-->
  </head>
  <body>
    <!--Headings, paragraphs, images, etc. go inside here-->
  </body>
</html>
```

Розділ `head` містить важливу інформацію, що працює за лаштунками:

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Document Title Goes Here</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

Метадані вашого сайту, які містяться в елементах `meta`, містять деталі про кодування символів і про те, як такі сайти, як Twitter, мають попередньо переглядати посилання на вашу сторінку. Назва вашого сайту, що знаходиться в елементі `title`, визначає текст, який відображається на вкладці або вікні браузера. Нарешті, зазвичай ви підключаєте зовнішні таблиці стилів вашої сторінки в розділі `head` за допомогою елементів `link`.

Розділ `body` — це місце, куди поміщається весь ваш вміст:

```html
<body>
  <h1>I am a main title</h1>
  <p>Example paragraph text</p>
</body>
```

Чому ж шаблонний код важливий? Він гарантує, що ваші сторінки мають правильну структуру і добре працюють у різних браузерах. Використання шаблонного коду допомагає уникнути загальних помилок і дотримуватися найкращих практик. Це чудова відправна точка для будь-якого вебпроєкту. Пам’ятайте, ви можете кастомізувати свій власний шаблонний код під свої потреби. З набуттям досвіду ви можете додавати власні улюблені елементи або теги `meta`. Продовжуючи вдосконалювати свій особистий шаблонний код, ви помітите, що він економить вам час при початку нових проєктів.

Наступного разу, коли починатимете новий HTML-файл, подумайте про використання шаблонного коду. Це точно дасть вам міцний фундамент для подальшої роботи.
