# --description--

現在你已經熟悉了次級色，你將學習如何創建<dfn>三級色</dfn>。三級色是透過將一種原色與相鄰的次級色組合而創建的。

為了創建第三色橙色，請更新 `.one` CSS 規則中的 `rgb` 函式（程式），使紅色達到最大值，並將綠色設為 `127`。

# --hints--

你的 `.one` CSS 規則應該有一個 `background-color` 屬性設定為 `rgb(255, 127, 0)`。

```js
assert(new __helpers.CSSHelp(document).getStyle('.one')?.backgroundColor === 'rgb(255, 127, 0)');
```
