# --interactive--

Beim Gestalten von Websites ist eine der vielseitigsten und intuitivsten Methoden, mit Farben zu arbeiten, die Verwendung des HSL-Farbmodells. HSL steht für Farbton, Sättigung und Helligkeit – drei wichtige Komponenten, die eine Farbe definieren.

Das HSL-Farbmodell stellt Farben auf eine Weise dar, die besser mit der menschlichen Farbwahrnehmung übereinstimmt. In dieser Lektion werden wir untersuchen, was das HSL-Farbmodell ist und wie die `hsl()`-Funktion in CSS verwendet wird.

Der Farbton ist die Art der Farbe, dargestellt als Winkel auf dem Farbrad, der von `0` bis `360` Grad reicht. Zum Beispiel steht `0` Grad für Rot, `120` Grad für Grün und `240` Grad für Blau. Durch Anpassen des Farbtons wechseln Sie zwischen verschiedenen Farben im Spektrum.

Die Sättigung bezieht sich auf die Intensität oder Reinheit der Farbe. Sie wird als Prozentsatz von `0%` (eine vollständig entsättigte, graustufige Farbe) bis `100%` (die lebhafteste Form dieser Farbe) gemessen. Eine Farbe mit `100%` Sättigung ist vollständig lebendig, während eine Farbe mit `0%` Sättigung als Grauton erscheint.

Die Helligkeit bestimmt, wie hell oder dunkel die Farbe ist, ebenfalls als Prozentsatz gemessen. Ein Helligkeitswert von `0%` ergibt Schwarz, `50%` gibt den normalen Ton des Farbtons wieder, und `100%` führt zu Weiß.

In CSS wird die `hsl()`-Funktion verwendet, um Farben mit dem HSL-Farbmodell zu definieren. Hier ist die grundlegende Syntax:

```css
element {
  color: hsl(hue, saturation, lightness);
}
```

Lassen Sie uns das mit einem Beispiel aufschlüsseln:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
body {
  background-color: hsla(0, 0%, 1%, 1.00);
}

p {
  color: hsl(120, 100%, 50%);
}
```

:::

In diesem Fall beträgt der Farbton `120` Grad, was Grün entspricht. Die Sättigung ist `100%`, sodass das Grün vollständig lebendig ist. Die Helligkeit beträgt `50%`, also den normalen Ton – weder zu dunkel noch zu hell. Das Ergebnis ist, dass die Textfarbe des Absatzes ein leuchtendes Grün ist.

Einer der Hauptvorteile des HSL-Farbmodells ist seine intuitive Natur. Es macht es einfach, die Lebendigkeit oder Helligkeit einer Farbe anzupassen, indem Sie die Werte für Sättigung und Helligkeit ändern, ohne den Kernfarbton (Farbton) zu verändern.

Wenn Sie zum Beispiel verschiedene Schattierungen oder Tönungen derselben Farbe erstellen möchten, können Sie einfach den Helligkeitswert anpassen.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="light">This is a light blue div.</div>
<div class="dark">This is a dark blue div.</div>
```

```css
div.light {
  background-color: hsl(240, 100%, 80%);
}

div.dark {
  background-color: hsl(240, 100%, 20%);
  color: hsl(0, 0%, 100%);
}
```

:::

Hier verwenden beide `div`-Elemente denselben Farbton (`240` Grad, was Blau ist), aber eines hat eine Helligkeit von `80%` (ein hellerer Blauton), und das andere eine Helligkeit von `20%` (ein dunklerer Blauton).

So wie das RGB-Modell eine `rgba()`-Funktion zur Einbeziehung von Transparenz hat, verfügt das HSL-Modell über eine `hsla()`-Funktion. Der vierte Parameter in dieser Funktion stellt den Alpha-Wert dar, der die Deckkraft der Farbe steuert. Hier ist die grundlegende Syntax:

```css
element {
  background-color: hsla(hue, saturation, lightness, alpha);
}
```

Sehen wir uns ein Beispiel an:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

Dieser Code würde dem `div` einen halbtransparenten roten Hintergrund geben, wobei der Farbton auf `0` Grad (Rot) eingestellt ist, die Sättigung `100%`, die Helligkeit `50%` und der Alpha-Wert `0.5` (50 % Deckkraft). Sie können den Alpha-Wert auch als Prozentsatz angeben – zum Beispiel entspricht `hsla(0, 100%, 50%, 50%)` `hsla(0, 100%, 50%, 0.5)`.

Das HSL-Farbmodell ist besonders nützlich, wenn Sie Farbschemata erstellen und Schattierungen oder Tönungen einfach anpassen müssen.

Da HSL den Farbton von Sättigung und Helligkeit trennt, können Sie ein konsistentes Farbthema beibehalten und gleichzeitig die Helligkeit für Schatten oder Highlights anpassen. Diese Flexibilität macht HSL sowohl für Designer als auch für Entwickler zu einer beliebten Wahl.

Außerdem macht HSL Farbwerte lesbarer und verständlicher im Vergleich zu anderen Methoden wie Hex-Codes oder RGB. Sie können die Absicht hinter der Farbe klarer erkennen und entsprechend anpassen, besonders beim Experimentieren mit verschiedenen Designs.

Mit dem HSL-Farbmodell gewinnen Sie mehr Flexibilität bei der Steuerung von Farbton, Sättigung und Helligkeit, was es zu einem leistungsstarken Werkzeug für die Erstellung reichhaltiger und dynamischer Designs in CSS macht.
