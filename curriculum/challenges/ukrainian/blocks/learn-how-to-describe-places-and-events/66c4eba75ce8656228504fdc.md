# --explanation--

`Should` використовується, щоб висловити, що щось очікується статися або бути правдою. Коли Сара каже `It should`, вона має на увазі, що живий перегляд очікується працювати у Visual Studio Code. Наприклад:

- `The app should start when you press the button.` - Це означає, що ви очікуєте, що застосунок запуститься при натисканні кнопки.
