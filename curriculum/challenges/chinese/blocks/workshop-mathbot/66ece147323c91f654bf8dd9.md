# --description--

In the previous lectures, you learned how to get the maximum and minimum values from a range of numbers like this:

```js
Math.max(1, 2, 3, 4, 5); // 5
Math.min(1, 2, 3, 4, 5); // 1
```

创建一个名为 `maxNum` 的变量，将 `3, 125, 55, 24` 中最大数字赋值给它。 然后，将 `maxNum` 的值记录到控制台。

在下面创建一个名为 `minNum` 的变量，将 `6, 90, 14, 90, 2` 中的最小数赋给它。 然后，将 `minNum` 的值记录到控制台。
