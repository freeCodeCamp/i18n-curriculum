# --description--

`why don't` 구문은 제안이나 권유를 할 때 자주 사용합니다. 그 뒤에는 주어(`we`, `you`, 또는 `they` 같은)와 기본 동사가 옵니다. 구조는 `why don't + subject + base verb`입니다. 누군가에게 어떤 행동을 하자고 정중하게 제안하거나 요청하는 표현입니다.

예시:

`Why don't we go to the cinema tonight?` 화자가 영화관에 가자고 제안합니다.

`Why don't you try restarting the computer?` 화자가 해결책으로 컴퓨터를 재시작하자고 제안합니다.

`check out` 구문은 무언가를 더 자세히 살펴보거나 확인하자는 뜻으로, 무언가를 탐색하거나 배우자고 제안할 때 흔히 사용합니다. 예를 들어 `Let's check out the new restaurant`는 식당을 방문해 보자는 의미입니다.
