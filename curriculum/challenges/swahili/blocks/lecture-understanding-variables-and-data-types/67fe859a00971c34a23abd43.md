# --description--

Kabla ya kufanya kazi na variable za Python, ni muhimu kuelewa aina za data. Aina ya data inaelezea aina ya thamani ambayo variable inahifadhi. Kwa mfano, nambari, kipande cha maandishi, au orodha ya vitu. Lugha za programu hutumia aina za data ili zijue jinsi ya kuhifadhi na kufanya kazi na aina tofauti za taarifa.

Python ni lugha yenye aina zinazobadilika (dynamically-typed) kama JavaScript, ikimaanisha huna haja ya kutangaza aina za variable kwa wazi. Lugha inajua aina ya data ya variable kulingana na unachomweka ndani yake.

Hapa kuna mifano kadhaa:

```python
name = 'John Doe' # Python knows this is a string
age = 25 # Python knows this is an integer
```

Hii ni tofauti na baadhi ya lugha zenye aina zilizotangazwa kabla kama C#, Java, na C++, ambapo unapaswa kutangaza aina za variable, kama ifuatavyo:

```csharp
string name = 'John Doe'
int age = 25
```

Asili ya aina zinazobadilika ya Python hufanya uandishi wa msimbo kuwa wa haraka na rahisi kubadilika, lakini inaweza kusababisha hitilafu zisizotarajiwa kwa sababu makosa ya aina hutambuliwa tu wakati programu inapoendeshwa, si wakati programu inapotengenezwa (compile).

Kwa kuwa Python huchagua aina za data wakati programu yako **inapoendeshwa**, makosa yanayohusiana na aina hutambuliwa wakati huo tu. Programu inapokuwa inafanya kazi, Python hufanya msimbo wako mstari kwa mstari. Ikiwa inafikia mstari ambapo kitu fulani kinapaswa kufanya jambo fulani lakini hakiwezi, Python itasimama na kuonyesha hitilafu.

Kinyume chake, baadhi ya lugha **hutengeneza programu kabla ya kuendesha**. Kutengeneza (compile) kunamaanisha kompyuta inakagua msimbo wako mapema na kuutayarisha kuendeshwa. Katika hatua hii, lugha hizo zinaweza kugundua makosa ya aina kabla hata programu haijaanza.

Huna haja ya kujifunza lugha hizo bado. Wazo muhimu ni hili tu:

- **Katika Python makosa ya aina yanaweza kujitokeza wakati wa utekelezaji**, wakati programu inapoendesha na kutumia msimbo wako.

- **Lugha zinazotengenezwa (compiled) hugundua makosa ya aina wakati wa hatua ya kutengeneza**, kabla programu kupewa ruhusa kuendeshwa.

Kwa sababu hii, huenda usijue kuhusu kosa la aina katika Python hadi programu ifikie mstari huo maalum wa msimbo wakati wa kuendesha.

Hapa ni aina za kawaida za data utakazotumia katika Python:

- Integer: Nambari kamili isiyo na sehemu za desimali, kwa mfano, `10` au `-5`.

```python
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- Float: Nambari yenye sehemu ya desimali, kama `4.41` au `-0.4`.

```python
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- String: Mfuatano wa herufi uliowekwa ndani ya alama za nukuu moja au mbili kama `'Hello world!'`.

```python
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- Boolean: Thamani ya kweli au siyo kweli, imeandikwa kama `True` au `False`.

```python
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- Set: Mkusanyiko usio na mpangilio wa vitu tofauti, kama `{0.5, 4, 'apple'}`.
 
```python
my_set_var = {7, 'hello', 8.5}
print('Set:', my_set_var) # Set: {7, 'hello', 8.5}
```

- Dictionary: Mkusanyiko wa jozi za ufunguo-thamani zilizowekwa ndani ya mabano ya mviringo, kama `{'name': 'John Doe', 'age': 28}`. 

```python
my_dictionary_var = {'name': 'Alice', 'age': 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- Tuple: Mkusanyiko wa vitu uliopangwa na usiobadilika, uliowekwa ndani ya mabano ya kawaida, kama `('apple', 4.5, 7)`.

```python
my_tuple_var = (7, 'hello', 8.5)
print('Tuple:', my_tuple_var) # Tuple: (7, 'hello', 8.5)
```

- Range: Mfululizo wa nambari, mara nyingi hutumika katika mizunguko, kwa mfano, `range(5)`.

```python
my_range_var = range(5)
print('Range:', my_range_var) # Range: range(0, 5)
```

- List: Mkusanyiko uliopangwa wa vitu unaounga mkono aina tofauti za data.
    
```python
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- None: Thamani maalum inayowakilisha kutokuwepo kwa thamani.

```python
my_none_var = None
print('None:', my_none_var) # None: None
```

Ili kupata aina ya data ya variable, unaweza kutumia kitendakazi cha `type()`:

```python
my_var_1 = 'Hello world'
my_var_2 = 21

print(type(my_var_1)) # <class 'str'>
print(type (my_var_2)) # <class 'int'>
```

Na hapa ni aina zote za data zilizojadiliwa katika funzo hili, pamoja na aina zao kwenye terminali:

```python
my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 'hello', 8.5}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 'hello', 8.5)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>
```

Kitendakazi kilicho ndani ya Python cha `isinstance()` hukuruhusu kuangalia kama variable inalingana na aina fulani ya data. Kinapokea kitu na aina unayotaka kuikagua, kisha hurudisha thamani ya kweli au siyo kweli. Hapa kuna mifano:

```python
isinstance('Hello world', str) # True
isinstance(True, bool) # True
isinstance(42, int) # True
isinstance('John Doe', int) # False
```
