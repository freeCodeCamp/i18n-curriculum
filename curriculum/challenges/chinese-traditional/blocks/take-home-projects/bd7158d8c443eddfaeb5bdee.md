# --description--

**目標：** 建置一個功能上類似於此的應用程式：<a href="https://build-a-pinterest-clone.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://build-a-pinterest-clone.freecodecamp.rocks/</a>。

完成以下使用者故事並通過所有測試。使用你需要的任何函式庫或 API。展現你自己的風格。

**使用者故事：** 作為未經身份驗證的使用者，我可以使用 GitHub 登入。

**使用者故事：** 作為已驗證的使用者，我可以連結到影像。

**使用者故事：** 作為已驗證的使用者，我可以刪除我已連結的影像。

**使用者故事：** 作為一個已驗證的使用者，我可以看到一個 Pinterest 風格的牆，展示我所連結的所有影像。

**使用者故事：** 作為未經身份驗證的使用者，我可以瀏覽其他使用者的影像牆。

**使用者故事：** 作為一個已驗證的使用者，如果我上傳的影像損壞，將會被一個佔位影像取代。（可以使用 jQuery broken image detection）

**提示：** Masonry.js 是一個允許建立 Pinterest 風格影像網格的函式庫。

完成這些使用者故事的實作後，輸入你的線上應用程式的 URL，並選填你的 GitHub 儲存庫。然後點擊「我已完成此挑戰」按鈕。
