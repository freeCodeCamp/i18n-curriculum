# --explanation--

`scope`는 무언가의 범위, 경계 또는 정도를 나타냅니다. `Full scope`는 조치를 취하기 전에 문제의 모든 측면을 이해하는 것을 강조합니다. 예를 들어:

`Before launching the new system, we need to analyze the full scope of its impact.` - 이는 진행하기 전에 모든 잠재적 영향을 고려한다는 뜻입니다.

`To make changes`는 무언가를 수정하거나 조정하는 것을 의미합니다. 예를 들어:

`We decided to make changes to the design after receiving user feedback.` - 이는 사용자 피드백을 바탕으로 조정이 이루어졌다는 뜻입니다.
