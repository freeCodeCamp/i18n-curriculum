# --description--

`Top condition` означає бути в найкращому можливому стані або якості, що особливо важливо для обладнання, яке виконує критично важливі функції безпеки. Наприклад, `The fire extinguisher must be kept in top condition`.

`Emergencies` — це серйозні, несподівані ситуації, які потребують негайних дій, як-от пожежа або медична криза. `Emergencies` — це множина від `emergency`.
