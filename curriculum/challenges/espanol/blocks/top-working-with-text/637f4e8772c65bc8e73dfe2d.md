# --description--

Los comentarios en HTML no son visibles para el navegador; nos permiten comentar tu código para que otros desarrolladores o tú en el futuro puedan leerlos y obtener contexto sobre algo que podría no estar claro en el código.

Escribir un comentario en HTML es simple: solo tienes que encerrar el comentario con las etiquetas `<!--` y `-->`. Por ejemplo:

```html
<h1> View the html to see the hidden comments </h1>

<!-- I am a html comment -->

<p>Some paragraph text</p>

<!-- I am another html comment -->
```

# --assignment--

Para practicar trabajando con texto en HTML, crea una página de artículo de blog sencilla que use diferentes encabezados, use párrafos y tenga algo de texto en los párrafos en negrita y cursiva. Puedes usar [Lorem Ipsum](https://loremipsum.io) para generar texto de relleno, en lugar de texto real mientras construyes tus sitios.
