# --description--

En esta tarea, aprenderás la diferencia entre `simple` y `simply`, y entenderás qué significan `double` y `slashes`.

`Simple` es un adjetivo que significa fácil o no complicado, como `a simple task`. `Simply` es un adverbio, que significa de manera simple, como `doing something simply`.

`Double` significa dos de algo. Y `slashes` son los símbolos `/` usados en la escritura. Agregas `es` para hacer plural a `slash` porque las palabras que terminan en `sh` forman su plural agregando `es,` como en `dishes` o `brushes`.
