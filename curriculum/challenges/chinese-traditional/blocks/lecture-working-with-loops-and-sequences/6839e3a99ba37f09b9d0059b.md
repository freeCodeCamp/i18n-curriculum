# --description--

元組是 Python 的一種資料型別，用於創建有序的值序列。元組可以包含像這樣混合的資料型別：

```py
developer = ('Alice', 34, 'Rust Developer')
```

元組類似於列表，但列表是可變的資料型別，而元組是不可變的。這表示元組中的元素在創建後無法更改。

如果你嘗試更新元組中的其中一個項目，將會得到 `TypeError`：

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: 'tuple' object does not support item assignment
"""
```

要存取元組中的元素，你可以使用括號表示法和索引號碼：

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

如果你需要從元組的末端開始存取元素，則你可以使用負索引。以下是使用負索引存取元組中倒數第二個元素的範例：

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

如果你嘗試傳入一個大於或相等於 tuple 長度的索引數字，則你會收到像這樣的 `IndexError`：

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list index out of range
"""
```

另一種建立 tuple 的方式是使用 `tuple()` 建構式，如下所示：

```py
developer = 'Jessica'
tuple(developer) # ('J', 'e', 's', 's', 'i', 'c', 'a')
```

對於 `tuple()` 建構式，你可以傳入不同的可迭代物件，例如字串、列表甚至其他元組。 

要檢查一個項目是否在一個 tuple 中，你可以像這樣使用 `in` 關鍵字：

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

你也可以像對列表一樣從元組中解包項目：

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

在這個範例中，`name` 的值是 `'Alice'`，`age` 的值是 `34`，而 `job` 的值是 `'Rust Developer'`。

如果你需要收集元組中任何剩餘的元素，你可以像這樣使用星號（`*`）運算子：

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

這裡，`name` 的值是 `'Alice'`，而 `rest` 是一個由數字 `34` 和字串 `'Rust Developer'` 組成的列表。

就像對列表一樣，你可以對元組使用切片運算子來擷取其中的一部分。以下是一個將項目 `'pie'` 和 `'cookies'` 擷取到一個獨立元組的範例：

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

請記住，第一個數字表述擷取的起始索引，而第二個數字表述結束索引。但請注意，結束索引的項目不包含在擷取的元組中。

如果你需要從 tuple 中移除一個項目，這是不可能的，因為 tuple 是不可變的。所以這個範例會產生錯誤：

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

那麼，什麼時候你會使用 tuple 而不是 列表？

如果你需要一個可以新增、移除和更新元素的動態群集，則你應該使用列表。如果你知道你正在處理一個固定且不可變的資料群集，則你應該使用元組。

在下一課中，我們將看看你在處理元組時會使用的一些常見方法。
