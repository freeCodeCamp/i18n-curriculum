# --description--

O método `.extend()`, permite que você adicione elementos de um iterável ao final de uma lista:

```py
my_list = ['larch', 'birch']
tree_list = ['fir', 'redwood', 'pine']
my_list.extend(tree_list)
print(my_list) # Output: ['larch', 'birch', 'fir', 'redwood', 'pine']
```

Crie uma cláusula `else` e use o método `.extend()` para adicionar o caminho do nó atual ao caminho do nó vizinho.
