# --explanation--

`Overlap` se usa cuando una cosa cubre o se extiende sobre parte de otra. En este contexto, significa que los elementos en la pantalla están posicionados incorrectamente, cubriendo partes unos de otros.

`On` se usa para indicar el dispositivo donde ocurre la superposición. Muestra la relación entre los elementos y los dispositivos. Por ejemplo:

`The images overlap` en la página web, lo que dificulta ver todo el contenido claramente. Aquí, `overlap` describe el problema de posicionamiento y `on` especifica dónde ocurre.
