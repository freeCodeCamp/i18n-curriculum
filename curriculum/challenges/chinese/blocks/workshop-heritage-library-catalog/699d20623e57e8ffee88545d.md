# --description--

在 `"Unknown"` 查看之后，添加针对带有数字年份的书籍的十年分组逻辑。声明 `const decade = Math.floor(book.year / 10) * 10`。例如，`Math.floor(1977 / 10)` 得到 `197`，乘以 `10` 得到 `1970`。

然后声明 `const decadeKey =` `` `${decade}s` `` to produce a string like `"1970s"`。

使用方括弧表示法对书籍进行分组：检查 `grouped[decadeKey]` 是否尚不存在，如果不存在则初始化为空数组，然后将 `book` 推入其中。这里必须使用方括弧表示法，因为密钥是保存在变量中的。
