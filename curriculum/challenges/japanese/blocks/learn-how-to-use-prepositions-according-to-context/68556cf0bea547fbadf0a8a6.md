# --explanation--

`Around` は時間や場所が「およそ」や「近く」という意味です。例えば：

`We plan to deploy the update around Friday.` – これはその日付の近くを意味し、正確ではありません。

`Mid-quarter` は3か月の営業期間の中間点を指します。例えば：

`The review will happen in mid-quarter.` – これは現在の四半期の途中で行われることを意味します。
