# --instructions--

Kwa changamoto hii, unahitaji kuunda mpelelezi wa grafu ya data kwa kutumia Python ambao unachukua ingizo na kufanya yafuatayo:

- Pata faili la .csv kwa njia tatu
  - kulipakia kutoka kwenye kompyuta ya eneo lako
  - kupata URL kutoka kwa ingizo la mtumizi
  - kuweka URL ndani ya msimbo
- Tumia maktaba ya Pandas kuhifadhi .csv kama fremu ya data
- Chapisha vichwa vya sehemu na mistari miwili ya kwanza
- Hifadhi majina ya safu kama orodha
- Chagua safu moja au mbili na ubadilishe data kuwa safu za data za Numpy
- Onyesha data kama mchoro wa alama au grafu ya mstari
- Kuwa na uwezo wa kufanya hivi kwa mchanganyiko tofauti wa safu, na tafsiri grafu hizo

Mara utakapomaliza, wasilisha URL ya daftari la Colab la umma kwenye hifadhi yako ya Google.
