# --description--

`run code` という用語は、コンピューターがプログラムやスクリプトを実行または操作する行為を指します。この文脈で `run` とは、計算やデータの並べ替えなど、プログラムが設計された通りに動作するまで処理を開始し続けることを意味します。

`run code without managing servers` というフレーズは、コードが実行される物理的または仮想的なサーバーを扱う必要なくコンピュータープログラムやアプリケーションを実行することを指します。これはサーバーレスコンピューティングの重要な側面であり、クラウドプロバイダーがサーバーを管理するため、開発者はサーバー管理よりもコードにより集中できます。
