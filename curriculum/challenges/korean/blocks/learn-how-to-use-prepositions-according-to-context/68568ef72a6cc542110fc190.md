# --explanation--

`Any`는 특정하지 않고 하나 이상을 가리킬 때 사용합니다. 예를 들어:

`Let me know if you have any questions.` – 이는 하나 이상 가능한 질문을 의미합니다.

`Might`는 어떤 일이 가능하지만 확실하지 않음을 나타냅니다. 예를 들어:

`We might need more time to finish.` – 이는 추가 시간이 필요할 가능성이 있음을 의미합니다.
