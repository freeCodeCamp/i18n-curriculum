# --description--

Si estás usando tu dispositivo móvil para navegar por el currículo de español, necesitarás asegurarte de que tu teléfono o tableta estén listos para escribir marcas de acento y caracteres especiales como `á`, `é` o `ñ`.

## El truco rápido de presionar y mantener

La forma más sencilla de escribir letras acentuadas es presionando y manteniendo pulsada la letra que necesitas.  

1. Abre un campo de texto en cualquier aplicación, como mensajes o notas.

2. Mantén presionada la tecla de letra, por ejemplo `a` o `n`.

3. Aparecerá un menú emergente con versiones acentuadas como `á` o `ñ`.

4. Desliza tu dedo hasta el carácter que quieres y levántalo para seleccionarlo.

Esto es perfecto cuando solo necesitas acentos ocasionalmente.  


## Agregar teclados de idioma

Puede resultar un poco complicado encontrar el acento correcto en español, por lo que agregar una distribución de teclado en español ayuda. Esto te da acceso más rápido a los acentos y otros caracteres.


### En Android

1. Abre **Settings**.

2. Ve a **System > Keyboard > On-screen keyboard**.

3. Elige tu aplicación de teclado, como Gboard.

4. Toca **Languages**.

5. Agregar español.

6. Toca **Done** para terminar.


### En iPhone

1. Abre **Settings**.

2. Ve a **General > Keyboard**.

3. Toca **Keyboards**, luego **Add New Keyboard**.

4. Seleccionar español.

5. Usa el ícono del globo en tu teclado para cambiar entre distribuciones de idioma.


## Probando teclados de terceros

También puedes instalar diferentes aplicaciones para tu teléfono o tableta para usar el teclado en español con facilidad. Estas aplicaciones a menudo ofrecen funciones útiles como sugerencias inteligentes, soporte para varios idiomas y formas más rápidas de escribir las tildes.

Ahora estás listo para seguir el currículo de español y completar las tareas, especialmente aquellas que requieren tildes.
