# --description--

`So far` は「これまでのところ」や「今までずっと」という意味です。過去に始まり現在まで続いている時間について話すときに使います。誰かが `Are you happy with the workplace so far?` と尋ねた場合、それは最初の日から今日まで職場が気に入っているかどうかを知りたいということです。

別の例として、もし `The project is good so far` と言ったら、それはプロジェクトの始まりから今まで、すべてが順調だという意味です。

この会話では、サラがトムに「今までのところ新しい仕事に満足しているか」尋ねています。彼女は仕事を始めてから会話の時点までの時間について聞いています。
