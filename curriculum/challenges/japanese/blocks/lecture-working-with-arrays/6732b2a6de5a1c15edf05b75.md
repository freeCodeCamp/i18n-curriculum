# --interactive--

文字列を逆順にするのはよくあるプログラミングの作業で、JavaScriptでは文字列と配列のメソッドを組み合わせて実現できます。この処理は主に3つのステップで行います。

- 文字列を文字の配列に分割する。
- 配列を逆順にする。
- 文字を再び文字列に結合する。

これらのステップを`split()`、`reverse()`、`join()`メソッドを使って見ていきましょう。

文字列を逆順にする最初のステップは、文字列を個々の文字の配列に変換することです。これは`split()`メソッドを使って行います。`split()`メソッドは文字列を区切り文字に基づいて配列に分割します。区切り文字が指定されない場合は、元の文字列を1つの要素とする配列を返します。よく使われる区切り文字の例は以下の通りです。

- 空文字列（`""`）は文字列を1文字ずつに分割します。

- 半角スペース（`" "`）はスペースのあるところで分割します。

- ダッシュ（`"-"`）はダッシュの位置で分割します。

以下はsplitメソッドを使って文字の配列を作る例です。

:::interactive_editor

```js
let str = "hello";
let charArray = str.split("");
console.log(charArray); // ["h", "e", "l", "l", "o"]
```

:::

この例では、`split("")`（空文字列を渡して）を使い、文字列`hello`を個々の文字の配列に変換しています。文字の配列ができたら、`reverse()`メソッドを使って配列の要素の順序を逆にできます。

`reverse()`メソッドは配列の順序をその場で逆にする配列のメソッドです。つまり、新しい配列を作るのではなく元の配列を変更します。使い方は以下の通りです。

:::interactive_editor

```js
let charArray = ["h", "e", "l", "l", "o"];
charArray.reverse();
console.log(charArray); // ["o", "l", "l", "e", "h"]
```

:::

この例では、`reverse()`が`charArray`の要素の順序を変え、`["h", "e", "l", "l", "o"]`から`["o", "l", "l", "e", "h"]`に逆順にしています。

最後のステップは、逆順にした文字の配列を再び文字列に変換することです。これは`join()`メソッドで行います。`join()`メソッドは配列のすべての要素を指定した区切り文字で連結し、新しい文字列を作って返します。区切り文字なしで結合したい場合は空文字列を引数に渡します。例は以下の通りです。

:::interactive_editor

```js
let reversedArray = ["o", "l", "l", "e", "h"];
let reversedString = reversedArray.join("");
console.log(reversedString); // "olleh"
```

:::

この例では、`join("")`（空文字列を引数に渡して）配列のすべての文字を区切りなしで1つの文字列にまとめています。

JavaScriptの文字列は不変なので、文字列を直接逆順に変更することはできません。だからこそ、配列に変換して逆順にし、再び文字列に戻す必要があります。この文字列と配列のメソッドの組み合わせは、JavaScriptで文字列を柔軟に操作する強力な方法です。
