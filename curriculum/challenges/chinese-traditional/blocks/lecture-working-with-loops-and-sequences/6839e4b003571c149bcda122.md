# --description--

在之前的課程中，你學會了如何使用 `for` 迴圈，該迴圈用於重複執行一個區塊的程式碼指定次數。以下是一個使用 `for` 迴圈將 `languages` 列表中的每個語言列印到主控台的範例：

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for language in languages:
    print(language)
```

但是如果你想追蹤每個元素的索引呢？其中一個選項是創建一個 `index` 變數，並在每次迴圈迭代時將它遞增 `1`，像這樣：

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

index = 0

for language in languages:
    print(f'Index {index} and language {language}')
    index += 1
```

雖然那樣可行，但更簡單的方法是使用 `enumerate()` 函式。`enumerate()` 函式會追蹤可迭代物件的索引並傳回一個列舉物件。

如果我們將 `languages` 列表傳遞給 `enumerate()` 函式，並使用 `list()` 函式將其傳回值轉換成列表，它看起來像這樣：

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

list(enumerate(languages))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

列舉物件（現在為列表）中的每個條目都是一個元組，包含一個計數，後面跟著傳遞給 `enumerate()` 函式的可迭代物件中的一個值。

現在，讓我們重構先前的範例以使用 `enumerate()` 函式（程式）：

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')
```

我們將 `enumerate` 物件中每個元組的計數和值分別解包到名為 `index` 和 `language` 的變數中。最後，這兩個變數都用於一個 f-string，該字串在每次迴圈中列印到主控台。

```md
Index 0 and language Spanish
Index 1 and language English
Index 2 and language Russian
Index 3 and language Chinese
```

這消除了手動建立和更新 `index` 變數的需求。

`enumerate()` 函式也接受一個選用的 `start` 引數，用來指定計數的起始值。如果省略此引數，則計數將從 `0` 開始。以下是使用選用 `start` 引數的範例：

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages, 1):
    print(f'Index {index} and language {language}')
```

以下是在主控台中結果的顯示方式：

```md
Index 1 and language Spanish
Index 2 and language English
Index 3 and language Russian
Index 4 and language Chinese
```

到目前為止，我們只在一個列表上進行迭代。但如果你需要平行迭代多個可迭代物件怎麼辦？你可以使用 `zip()` 函式（程式）來達成，它會將列表組合成元素的配對，並傳回一個包含元組的迭代器。

如果我們將 `developers` 和 `ids` 的列表傳遞給 `zip()` 函式，並使用 `list()` 函式將其傳回的值轉換成列表，結果如下：

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

list(zip(developers, ids))
# [('Naomi', 1), ('Dario', 2), ('Jessica', 3), ('Tom', 4)]
```

以下是使用 `zip()` 函式（程式）搭配 `for` 迴圈來迭代 `developers` 和 `ids` 的範例：

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')
```

在這個範例中，`zip()` 將兩個列表組合成元素的配對，並傳回一個包含元組的迭代器。接著，`for` 迴圈將每個元組拆解成 `name` 和 `id`。最後，對於每個列印述語，我們分別從 `ids` 和 `developers` 列表列印每個 `name` 和 `id`。以下是在主控台中顯示的結果：

```md
Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
```

`enumerate()` 和 `zip()` 函式非常強大，當與迴圈結合時，可以讓你的程式碼更加簡潔。
