# --explanation--

Una `code submission` es el conjunto de archivos de código que un programador envía para revisión. Por ejemplo:

- `Make sure your code submission follows the guidelines.` - El código debe cumplir las reglas antes de que lo envíes.

- `The professor reviewed my code submission and gave feedback.` - El profesor revisó tu código y compartió comentarios.

Una `loop structure` ejecuta un bloque de código repetidamente hasta que se cumple una condición determinada, ahorrando tiempo al automatizar tareas repetitivas, como los `for` loops o los `while` loops`. Por ejemplo:

- La estructura de bucle en mi código se ejecuta cinco veces. - Tu bucle se repite cinco veces en el programa.

- `A loop structure allows a program to repeat actions.` - Un bucle ayuda a un programa a hacer lo mismo varias veces.
