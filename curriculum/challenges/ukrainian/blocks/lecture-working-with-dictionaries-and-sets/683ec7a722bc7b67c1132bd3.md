# --description--

У Python словники — це вбудовані структури даних, які зберігають колекції пар ключ-значення. Вони працюють дуже схоже на справжні словники, де ви шукаєте слово, щоб знайти його відповідне значення.

У словниках Python ви використовуєте ключ, щоб знайти відповідне значення. Словники варто використовувати, коли потрібно пов’язати значення з унікальними ключами. Це корисно, коли потрібно швидко знайти значення за ключем і коли потрібно представити структуровані дані.

Ось загальний синтаксис словника Python:

```python
dictionary = {
    key1: value1,
    key2: value2
}
```

Спочатку ми знаходимо змінну, яка містить словник. Не обов’язково призначати словник змінній, але це дуже поширено, щоб зберегти його в пам’яті і використовувати пізніше в коді.

Потім ідуть фігурні дужки, які іноді називають фігурними скобками. Усередині фігурних дужок розташовані пари ключ-значення.

Кожен ключ пов’язаний із значенням, тож ви можете використовувати ключ, щоб отримати це значення. Після кожного значення, крім останнього, стоїть кома, щоб розділити різні пари ключ-значення. Ключі мають бути унікальними в словнику і повинні бути незмінним типом даних. Однак значення можуть повторюватися і бути будь-якого типу.

Ось приклад словника, який зберігає інформацію про рецепт піци Маргарита:

```python
pizza = {
    'name': 'Margherita Pizza',
    'price': 8.9,
    'calories_per_slice': 250,
    'toppings': ['mozzarella', 'basil']
}
```

Словник призначений змінній `pizza`. Він містить чотири пари ключ-значення: `name`, `price`, `calories_per_slice` і `toppings`.

Іншим варіантом є використання конструктора `dict()`, який створює словник із послідовності пар ключ-значення.

Ось еквівалентний синтаксис для нашого прикладу з піцою. Ми передаємо список кортежів як аргумент конструктору `dict()`. Ці кортежі містять ключ як перший елемент і значення як другий.

```python
pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
```

Щоб отримати значення пари ключ-значення, можна використати цей синтаксис, відомий як дужкова нотація. Це ім’я змінної, яка містить словник, за яким ідуть квадратні дужки, а всередині них — ключ, до якого ви хочете отримати доступ:

```python
dictionary[key]
```

У нашому прикладі з піцою, якщо ви хочете отримати значення `name`, ви напишете ім’я змінної `pizza`, потім квадратні дужки, а всередині них ключ `name` у лапках:

```python
pizza['name']
```

Це оціниться як:

```python
'Margherita Pizza'
```

Щоб оновити значення, потрібно просто додати оператор присвоєння, а потім нове значення.

Якщо ключ не існує в словнику, буде створена нова пара ключ-значення. У новіших версіях Python словники зберігають порядок вставлення. Це корисно, коли потрібно ітерувати над словником:

```python
pizza['name'] = 'Margherita'
```

Тепер значення ключа `name` — це `'Margherita'`:

```python
print(pizza['name']) # 'Margherita'
```

У словників також є корисні методи для виконання загальних операцій.

Метод `.get()` отримує значення, пов’язане з ключем. Це схоже на дужкову нотацію, яку ми щойно використали, але його перевага в тому, що можна встановити значення за замовчуванням, щоб не отримати помилку, якщо ключ не існує:

```python
dictionary.get(key, default)
```

У цьому прикладі, якщо ключ `toppings` не існує, буде повернуто порожній список — це значення за замовчуванням, яке ми передаємо як другий аргумент. Але якщо `toppings` існує, буде повернено це значення:

```python
pizza.get('toppings', []) # ['mozzarella', 'basil']
```

Методи `.keys()` і `.values()` повертають об’єкт-подання з усіма ключами та значеннями словника відповідно:

```python
pizza.keys()
# dict_keys(['name', 'price', 'calories_per_slice'])

pizza.values()
# dict_values(['Margherita Pizza', 8.9, 250])
```

Об’єкт-подання — це просто спосіб побачити вміст словника без створення окремої копії даних.

Метод `.items()` повертає об’єкт-подання з усіма парами ключ-значення в словнику, включно з ключами і значеннями:

```python
pizza.items()
# dict_items([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250)])
```

Метод `.clear()` видаляє всі пари ключ-значення зі словника:

```python
pizza.clear()
```

Метод `.pop()` видаляє пару ключ-значення з ключем, який ви вказуєте як перший аргумент, і повертає її значення. Якщо ключ не існує, він повертає значення за замовчуванням, яке ви вказуєте як другий аргумент. Якщо ключ не існує і ви не передали значення за замовчуванням, виникає `KeyError`:

```python
pizza.pop('price', 10)
pizza.pop('total_price') # KeyError
```

У Python 3.7 і новіших версіях метод `.popitem()` видаляє останній доданий елемент:

```python
pizza.popitem()
```

І нарешті, метод `.update()` оновлює пари ключ-значення парами іншого словника. Якщо ключі збігаються, їхні значення замінюються.

У цьому прикладі ми оновлюємо словник `pizza`. Ключ `price` існує в обох словниках, тому його значення буде замінене на `15`.

Але `total_time` — новий ключ, тому він буде доданий до словника `pizza` як нова пара ключ-значення:

```python
pizza.update({ 'price': 15, 'total_time': 25 })
```

Ось новий словник з оновленим `price` і новим `total_time`. Зверніть увагу, що ціна тепер `15`, а `total_time` — це нова пара ключ-значення:

```python
{
    'name': 'Margherita Pizza', 
    'price': 15, 
    'calories_per_slice': 250, 
    'toppings': ['mozzarella', 'basil'], 
    'total_time': 25
}
```

Це деякі з найпоширеніших методів словників, але їх набагато більше. Вибір правильного методу допоможе ефективно виконувати складні операції.
