# --description--

Як останній крок цього практичного заняття, додайте кнопку відправки до форми.

Пригадайте, що ви навчилися працювати з кнопками відправки в попередніх відеолекціях.

Додайте елемент `button` з атрибутом `type` зі значенням `"submit"` і текстом `Submit`.

На цьому форма зворотного зв’язку готова!

# --hints--

Ви повинні мати `button` в межах форми.

```js
assert(document.querySelector('form button'));
```

Елемент `button` повинен мати атрибут `type` зі значенням `"submit"`.

```js
assert(document.querySelector('button[type="submit"]'));
```

Елемент `button` повинен мати текст `Submit`.

```js
assert.strictEqual(document.querySelector('button')?.textContent, 'Submit');
```
