# --explanation--

`To prevent` significa tomar uma ação para que algo não aconteça.Por exemplo:

`They installed new software to prevent viruses.` – Isso significa que o software ajuda a impedir vírus antes que eles causem problemas.

Um `incident` é um evento inesperado, frequentemente negativo ou problemático.Por exemplo:

`There was a security incident last night that shut down the server.` – Isso significa que algo aconteceu que causou problemas no sistema.
