# --interactive--

在 JavaScript 中，處理文字是程式設計的重要部分，且你經常需要將多個文字零件合併或串接在一起。這個處理程序稱為字串串接。

在本課程中，我們將專注於字串串接的運作方式，特別是使用 `+` 運算子、`+=` 運算子和 `concat()` 方法。

`+` 運算子是最簡單且最常用來串接字串的方法之一。它允許你將多個字串連接起來，或將字串與包含文字的變數結合。

這裡有一個範例：

:::interactive_editor

```js
let firstName = "John";
let lastName = "Doe";

let fullName = firstName + " " + lastName; 
console.log(fullName); // John Doe
```

:::

在此範例中，我們使用 `+` 運算子將 `firstName` 和 `lastName` 變數連接起來，並加上一個空格（`" "`）以產生完整名稱。

使用 `+` 運算子進行字串串接的一個缺點是，如果你沒有仔細管理串接字串之間的間距，可能會導致間距問題。

這裡是一個缺少空格的範例：

:::interactive_editor

```js
let firstName = "John";
let lastName = "Doe";

let fullName = firstName + lastName; 
console.log(fullName); // "JohnDoe"
```

:::

每當你使用 `+` 運算子來串接字串時，務必仔細檢查是否有任何潛在的空格問題。

如果你需要為現有的字串添加或附加內容，則你可以使用 `+=` 運算子。當你想隨著時間為字串添加更多文字以進行建置時，這非常有用。

以下是使用 `+=` 運算子將一個字串附加到另一個字串的範例：

:::interactive_editor

```js
let greeting = 'Hello';
greeting += ', John!';

console.log(greeting); // "Hello, John!"
```

:::

重要的是要記住，字串是不可變的，這表示一旦字串被創建，你就無法更改它。

在這種情況下，原始的 `Hello` 字串未被修改。相反地，greeting 現在取用新的 `Hello, John!` 字串。

另一種串接字串的方法是使用 `concat()` 方法。

在我們開始學習 `concat()` 方法之前，首先了解方法和函式在較高層次上的意義是很重要的。

在程式設計中，函式（程式）是一個可重複使用的程式碼區段，用來執行特定任務，並且可以用各種輸入來呼叫。另一方面，方法（函式）是一種與物件相關的函式（程式），表示它會操作該物件內所包含的資料。

在未來的課程中，我們將更深入探討函式（程式）、物件和方法在 JavaScript 中的運作方式。但目前，理解 JavaScript 有數十種你可以使用的方法很重要，例如 `concat()` 方法。

以下是使用 `concat()` 方法將兩個字串串接在一起的範例：

:::interactive_editor

```js
let str1 = 'Hello';
let str2 = 'World';

let result = str1.concat(' ', str2); 
console.log(result); // Hello World
```

:::

在此範例中，我們使用 `concat()` 方法將 `str1`、一個空格（`' '`）和 `str2` 連接成一個字串。

總結來說，`+` 運算子最適合簡單的串接，特別是當你需要結合少數幾個字串或變數時。

當逐步建置字串或將新內容附加到現有字串變數時，`+=` 運算子非常有用。

最後，當你需要將多個字串串接在一起時，`concat()` 方法非常有用。
