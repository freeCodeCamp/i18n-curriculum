# --explanation--

`Should` — це модальне дієслово, яке виражає пораду, рекомендацію або найкращий варіант дії — але не зобов’язання.

- `You should update your password regularly.` — це хороша ідея часто змінювати пароль для безпеки.

- `You should check for errors in the code before running it.` — це хороша практика знаходити помилки перед виконанням коду.

Оскільки Джейк каже `we should focus on the critical systems first`, він дає сильну рекомендацію, але команда не має суворого зобов’язання це робити.
