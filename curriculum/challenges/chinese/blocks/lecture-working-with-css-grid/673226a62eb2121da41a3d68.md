# --interactive--

`minmax()` 函数定义网格轨道大小的范围，指定行或列可以占用多少空间。

记住，你可以使用这些单位设置轨道大小：`px`（像素）、`rem`，甚至 `em`，以及分数单位（`fr`）。

`minmax()` 函数更进一步，允许你设置网格轨道的最小尺寸和最大尺寸。

这是 `minmax()` 函数的语法：

```css
minmax(min, max)
```

`min` 是网格轨道的最小尺寸，可以使用像素、百分比或 `auto` 来设置。 而 `max` 是网格轨道的最大尺寸，你可以使用相同的单位设置。

这两个值是这样一起工作的：

- `min` 值确保网格轨道永远不会缩小到设置的尺寸以下。

- `max` 值限制了网格轨道可以增长到的尺寸。

网格轨道大小根据内容和容器大小在 `min` 和 `max` 值之间动态调整。

让我们来看一个实际的例子：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div>
    <h2>Item 1</h2>
  </div>
  <div>
    <h2>Item 2</h2>
  </div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: minmax(150px, 300px) 1fr;
  gap: 20px;
}

.grid-container > div {
  background: crimson;
  padding: 20px;
  text-align: center;
}
```

:::

这里发生了什么？ 

第一列，`minmax(150px, 300px)`，将始终至少为 `150px`，最多为 `300px`，具体取决于可用空间。

另一方面，第二列 `1fr` 将占用网格容器中任何可用的剩余空间，因为没有其他列来共享该空间。

相较于固定大小和单位 `fr`， `minmax()` 函数的优点是更灵活，使其成为适应性和响应性的理想选择。
