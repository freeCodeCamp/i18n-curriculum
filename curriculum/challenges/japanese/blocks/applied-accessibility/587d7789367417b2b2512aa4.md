# --description--

HTML5の`audio`要素は、マークアップ内で音声やオーディオストリームのコンテンツを囲むことで意味論的な意味を与えます。オーディオコンテンツは、聴覚障害のある人がアクセスできるようにテキストの代替も必要です。これはページ上の近くのテキストやトランスクリプトへのリンクで実現できます。

`audio`タグは`controls`属性をサポートしています。これによりブラウザのデフォルトの再生、一時停止、その他のコントロールが表示され、キーボード操作もサポートされます。これはブール値の属性で、値は不要で、タグに存在するだけで設定が有効になります。

例を示します：

```html
<audio id="meowClip" controls>
  <source src="audio/meow.mp3" type="audio/mpeg">
  <source src="audio/meow.ogg" type="audio/ogg">
</audio>
```

**注意：** マルチメディアコンテンツは通常、視覚的および聴覚的な要素を含みます。視覚や聴覚に障害のあるユーザーがアクセスできるように、同期したキャプションとトランスクリプトが必要です。一般的に、ウェブ開発者はキャプションやトランスクリプトの作成を担当しませんが、それらを含める必要があることを知っておく必要があります。
