# --explanation--

La frase creata usando `get` seguita da un aggettivo descrive un cambiamento di stato o il diventare in un certo modo. Per esempio:

- `They always get excited when a new Python update is released.` - `Get excited` significa diventare entusiasti o emozionati. Questo vuol dire che sono sempre pieni di entusiasmo quando arriva un nuovo aggiornamento di Python.

- `He got tired after working on the project all night.` - Questo significa che prima non era stanco, ma ora lo è, dopo un'intera notte passata a lavorare al progetto.

Nella frase di Maria, sta chiedendo se Brian ricorda l'entusiasmo che provavano per le nuove tecnologie mentre lavoravano insieme.
