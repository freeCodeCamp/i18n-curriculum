# --description--

`IDE` تعني بيئة العمل المتكاملة. هو تطبيق برمجي يوفر تسهيلات شاملة لمبرمجي الكمبيوتر من أجل تطوير البرمجيات.

عادةً ما يتضمن `IDE` محرر شفرة، ومترجم أو مفسر، وأداة تصحيح الأخطاء. من أمثلة `IDEs` برنامج Visual Studio Code وEclipse وIntelliJ IDEA.
