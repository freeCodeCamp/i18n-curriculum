# --description--

Sea $N(i)$ el entero más pequeño $n$ tal que $n!$ sea divisible por $(i!)^{1234567890}$

Sea $S(u) = \sum N(i)$ para $10 ≤ i ≤ u$.

$S(1000)=614\,538\,266\,565\,663$.

Encuentra $S(1\,000\,000)\bmod {10}^{18}$.
