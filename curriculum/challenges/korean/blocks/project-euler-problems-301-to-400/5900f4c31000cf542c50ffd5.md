# --description--

숫자 50을 생각해 보세요.

${50}^2 = 2500 = 2^2 × 5^4$이므로, $φ(2500) = 2 × 4 × 5^3 = 8 × 5^3 = 2^3 × 5^3$입니다. 여기서 $φ$는 오일러 피 함수를 나타냅니다.

따라서 2500은 제곱수이고 $φ(2500)$는 세제곱수입니다.

$φ(n^2)$가 세제곱수인 모든 $1 < n < {10}^{10}$인 수 $n$의 합을 구하세요.
