# --interactive--

Працюючи з рядками в JavaScript, часто можна зустріти небажані пробіли на початку або в кінці рядка. Пробіли можуть заважати таким операціям, як порівняння, збереження або відображення, тому важливо знати, як ефективно їх видаляти.

У цьому уроці ми розглянемо, як можна обрізати пробіли за допомогою методів `trim()`, `trimStart()` та `trimEnd()` у JavaScript.

Пробіли — це пробіли, табуляції або розриви рядків, які зустрічаються в рядку, але не є видимими символами. Наприклад:

```js
let greeting = "   Hello, world!   ";
```

У цьому випадку перед і після видимого тексту є пробіли, `Hello, world!`.

Метод `trim()` — це найпоширеніший спосіб видалити пробіли з початку та кінця рядка. Ось приклад:

:::interactive_editor

```js
let message = "   Hello!   ";
console.log(message); // "   Hello!   "
let trimmedMessage = message.trim();
console.log(trimmedMessage);  // "Hello!"
```

:::

У цьому випадку метод `trim()` видаляє всі провідні та кінцеві пробіли, залишаючи лише `Hello!`. Зверніть увагу, що будь-які пробіли всередині рядка (наприклад, між словами) залишаються недоторканими методом `trim()`.

Іноді потрібно видалити пробіли лише з початку або лише з кінця рядка, але не з обох сторін. Саме для цього існують `trimStart()` та `trimEnd()`.

`trimStart()` видаляє пробіли з початку (або старту) рядка.

:::interactive_editor

```js
let greeting = "   Hello!   ";
console.log(greeting);  // "   Hello!   "
let trimmedStart = greeting.trimStart();
console.log(trimmedStart);  // "Hello!   "
```

:::

`trimEnd()` видаляє пробіли з кінця рядка.

:::interactive_editor

```js
let greeting = "   Hello!   ";
console.log(greeting);  // "   Hello!   "
let trimmedEnd = greeting.trimEnd();
console.log(trimmedEnd);  // "   Hello!"
```

:::

Ці методи дають змогу точніше контролювати, яку частину рядка ви хочете очистити.

Підсумовуючи, обрізання пробілів — це важлива частина роботи з рядками в JavaScript. Якщо ви хочете очистити вхідні дані або забезпечити послідовність порівнянь рядків, ви можете використати `trim()` для видалення пробілів з обох сторін рядка або `trimStart()` та `trimEnd()` для цільового видалення з певної сторони.
