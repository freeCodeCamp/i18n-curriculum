# --description--

O caso genitivo (`'s`) é usado para mostrar posse ou uma relação entre duas coisas.

Para substantivos no singular, você adiciona `'s` ao final do substantivo para indicar posse.Por exemplo, `the cat's toy` significa o brinquedo que pertence ao gato.

Para substantivos no plural que já terminam em `-s`, você só adiciona um `'` no final.Por exemplo, `the teachers' lounge` significa o lounge para professores.

Uma `facility` é um lugar onde uma atividade específica acontece, como uma fábrica, escritório ou escola.
