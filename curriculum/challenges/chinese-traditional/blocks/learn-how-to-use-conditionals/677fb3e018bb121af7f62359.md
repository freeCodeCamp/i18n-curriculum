# --explanation--

`First Conditional` 用來描述未來真實且可能發生的情況。它由 `if` + `Present Simple`（表示條件）和 `will` + 動詞原形（表示結果）組成。例如：

`If you debug the code`，應用程式將會順利執行。修正程式碼將使應用程式正常運作。

在正確的句子中，條件是 `If we don't back up regularly`，結果是 `we could lose important information`。這表示一個真實的可能性。
