# --description--

Ein `mix-up` ist ein Fehler, bei dem Dinge durcheinandergebracht oder in die falsche Reihenfolge gebracht werden. Zum Beispiel, wenn zwei Freunde sich an unterschiedlichen Orten treffen, weil sie verschiedene Orte verstanden haben, ist das ein `mix-up`. Das bedeutet, dass es ein Missverständnis gab und sie an den falschen Orten gelandet sind.

Hören Sie sich den Dialog an und füllen Sie die Lücke aus.
