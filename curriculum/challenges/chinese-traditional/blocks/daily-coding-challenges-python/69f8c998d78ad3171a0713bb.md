# --description--

給定一個秘密數字和一個猜測，判斷猜測是否正確。

Return:

- `"higher"` 如果秘密數字比猜測大。
- `"lower"` 如果秘密數字比猜測小。
- `"you got it!"` 如果猜測正確。
