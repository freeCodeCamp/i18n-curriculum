명사절은 문장 내에서 명사 역할을 하는 절입니다. 문장의 목적어나 주어 역할을 할 수 있습니다. 보통 `that`, `what`, `whether`, `who`, `when`, `why`, `how` 같은 단어로 시작합니다.

명사절은 일반적인 평서문 어순을 사용합니다(의문문 어순이 아님). 예를 들어:

- `The team is checking why the system crashed during the update.` - `Why the system crashed during the update`는 `checking`의 목적어 역할을 하는 명사절입니다.
   
- `What the function returns depends on the input data.` - `What the function returns`는 `depends on`의 주어 역할을 하는 명사절입니다.
