## 排版简介

- **定义**：排版是选择合适字体和形式，使文本在视觉上吸引人且易于读取的艺术。 “Type” 指的是单个字符的设计和排列方式。
- **字型定义**：字型是一组字符、数字和符号的整体设计和风格。 它就像一个字体家族的蓝图。
- **字体定义**：字体是具有如大小、重量、风格和宽度等特定特征的字型的具体变体。
- **衬线体**：衬线体具有古典风格，字符末端有小线条。 衬线体通常用于书籍等印刷品。
- **无衬线体**：无衬线体的外观更为现代，字符末端没有小线条。 无衬线体通常用于数字设计，因为它们在屏幕上易于阅读。 例如，Helvetica、Arial 和 Roboto。
- **字体字重**：字符的粗细，包括细体、常规体、粗体和特粗体。
- **字体样式**：字符的倾斜方向与形态，如斜体和伪斜体。
- **字体宽度**：字符所占的水平空间，包括压缩和扩展。
- **基线**：是大多数字符所依附的一条假想线。
- **大写字母高度**：是指大写字母从基线到顶部的垂直高度。
- **X 高度**：是指小写字母的平均高度，不包括升部笔画和降部笔画。
- **升部**：是小写字母中超出 x 高度的部分，例如字母“h”、“b”、“d”和“f”的顶部。
- **降部**：是小写字母中向下延伸至基线以下的部分，例如字母“y”、“g”、“p”和“q”的尾部。
- **字距调整**：是指调整特定字符对之间的间距，以提升其可读性和美观度。 例如，减少字母 A 和 V 之间的间距。
- **字距微调**：是指调整整段文本中所有字符之间的间距。 本质上就是字符之间的距离。 它会影响文本的疏密程度。
- **行距**：是指文本行与行之间的垂直间距。 通常是从一行的基线到下一行基线的距离。
- **排版最佳实践**：应选择简洁明了的字体，使设计易于理解。 这对于网站正文尤为重要。 如果字体易于阅读，用户更可能与你内容互动。 为了创造视觉一致性，你应该最多使用两到三种字体。 使用过多字体会使文本更难阅读，并削弱你的品牌识别度。 这还可能因增加网站加载时间而影响用户体验。 你可以利用字体大小，为标题、副标题、段落和其他元素创建视觉层次。 例如，网页上的主标题应使用更大的字体，然后是较小字体的副标题。 这将为层次结构中的每个元素赋予特定的字体大小，帮助用户通过视觉层次浏览内容。

  
## CSS `font-family` 属性

- **定义**：字体族是一组共享通用设计的字体。 属于同一字体族的所有字体均基于同一核心字型，但它们的风格、粗细和宽度会有所变化。 你可以通过逗号分隔，按优先级从高到低指定多个字体族。 这些备用字体将作为回退选项。 你应始终在字体族列表末尾包含一个通用字体族。

```css
font-family: Arial, Lato;
```

- **常见字体族**：以下是 CSS 中使用的一些常见字体族：serif、sans-serif、monospace、cursive、fantasy。

## 网页安全字体

- **定义**：网页安全字体是极有可能已安装在计算机或设备上的一类字体子集。 当浏览器需要渲染字体时，它会尝试在用户系统中查找字体文件。 但若未找到该字体，通常会回退到默认系统字体。

- **网页安全字体**：
  - 无衬线字体因字符末端无“小脚”或细线，常用于网页开发，易于屏幕阅读。 网页安全无衬线字体的一些例子有 Arial、Verdana 和 Trebuchet MS。
  - 相比之下，衬线字体因字符末端有“小脚”，常用于传统印刷。 网页安全衬线字体包括：Times New Roman 和 Georgia。

## @ 规则和 `@font-face` 规则

- **定义**：@规则是向浏览器提供指令的语句。 你可以将其用于定义样式表的各个方面，例如媒体查询、关键帧、字体面等。
- **`@font-face`**：此规则能让你通过指定字体文件、格式以及其他重要属性来定义自定义字体，如粗细和样式。 要使 `@font-face` 规则有效，你还需要指定 `src` 属性。 该属性包含对字体资源的引用。

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2"),
       url("path/to/font.woff"),
       url("path/to/font.otf");
}
```

- **字体格式**：你还可以为每个字体资源指定格式。 这是可选项。 这是浏览器对字体格式的提示。 如果省略格式，将下载资源，并在下载后检测格式。 如果格式无效，将无法下载资源。 可能的字体格式包括 `collection`、`embedded-opentype`、`opentype`、`svg`、`truetype`、`woff` 和 `woff2`。

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
       url("path/to/font.otf") format("opentype"),
       url("path/to/font.woff") format("woff");
}
```

- **`woff` 和 `woff2`**：`woff` 代表 "Web Open Font Format"。它是一种由 Mozilla 与 Type Supply、LettError 及其他组织合作开发的广泛使用的字体格式。`woff` 和 `woff2` 之间的区别在于用于压缩数据的算法。
- **OpenType**：这是微软和 Adobe 开发的一种可扩展计算机字体格式，允许用户访问字体中的其他功能。 它广泛应用于各大操作系统。
- **`tech()`**：用于指定字体资源的技术。 这是可选项。

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
       url("path/to/font.otf") format("opentype") tech(color-COLRv1),
       url("path/to/font.woff") format("woff");
}
```

## 使用外部字体

- **Definition**: An external font is a font file that is not included directly within your project files. 它们通常托管在单独的服务器上。 要在项目中包含这些外部字体，可以在 CSS 中使用 `link` 元素或 `@import` 语句。
- **Google 字体**：这是一项 Google 服务，提供一系列字体，其中许多字体是专门为网页开发而设计的。

下面是一个使用 `link` 元素的示例：

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap" rel="stylesheet">
```

```css
.roboto-thin {
  font-family: "Roboto", sans-serif;
  font-weight: 100;
  font-style: normal;
}
```

下面是一个使用 `@import` 语句的示例：

```css
@import url('https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap');
```

- **Font Squirrel**：这是另一种流行的资源，可用于为项目添加自定义外部字体。

## `text-shadow` 属性

- **定义**：此属性用于对文本应用阴影。 你需要指定 X 和 Y 偏移，它们分别代表阴影与文本的水平和垂直距离。 这些值是必需的。 X 偏移和 Y 偏移的正值将分别使阴影向右和向下移动，而负值将使阴影向左和向上移动。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Typography Shadow</p>
```

```css
  p {
    text-shadow: 3px 2px;
  }
```

:::

- **阴影颜色**：你还可以在偏移之前或之后指定此值，从而自定义阴影的颜色。 如果未指定颜色，浏览器会自动确定颜色，因此通常最好明确设置颜色。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Colored Shadow</p>
```

```css
p {
  text-shadow: 3px 2px #00ffc3;
}
```

:::

- **模糊半径**：模糊半径是可选项，会使阴影看起来更平滑、更细腻。 模糊半径的默认值为零。 数值越大，模糊程度越高，这意味着阴影会变浅。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Blurred Shadow</p>
```

```css
p {
  text-shadow: 3px 2px 3px #00ffc3;
}
```

:::

- **应用多个文本阴影**：文本可以有多个阴影。 这些阴影将从前到后分层应用，第一个阴影位于顶层。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Multiple Shadows</p>
```

```css
p {
  text-shadow:
    3px 2px 3px #00ffc3,
    -3px -2px 3px #0077ff,
    5px 4px 3px #dee7e5;
}
```

:::
