# --description--

在這個任務中，你將學習 `simple` 和 `simply` 之間的差異，並了解 `double` 和 `slashes` 的意義。

`Simple` 是形容詞，意思是簡單或不複雜，例如 `a simple task`。`Simply` 是副詞，意思是以簡單的方式，例如 `doing something simply`。

`Double` 意味著兩個某物。而 `slashes` 是用於書寫的 `/` 符號。你為了使 `slash` 變成複數而添加 `es`，因為以 `sh` 結尾的字會透過添加 `es` 來形成複數，就像 `dishes` 或 `brushes` 一樣。
