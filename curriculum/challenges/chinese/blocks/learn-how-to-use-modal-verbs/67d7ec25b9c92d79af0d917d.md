# --explanation--

`Must have happened` 用于对过去的某事做出逻辑推断。  

`Must` 表示强烈的确定性。`Have` + `Past Participle`（如 `happened`）指的是过去发生的事情。例如：

- `He must have forgotten to update the system.`－他很可能没记得去做更新。

- `They must have made some changes to the app.` - 很可能该应用程序已经被更新或改进。

Jake 正在使用证据（`unusual activity on the servers`）来推断该漏洞 `must have happened` 昨晚。此结构对于进行逻辑推断非常有用。
