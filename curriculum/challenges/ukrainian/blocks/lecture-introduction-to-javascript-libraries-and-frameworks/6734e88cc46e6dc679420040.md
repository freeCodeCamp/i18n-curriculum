# --description--

На відміну від роботи з меншими проєктами на чистому HTML, CSS і JavaScript, початок нового проєкту React включає кілька додаткових кроків, щоб усе працювало правильно. Замість того, щоб намагатися налаштувати всі необхідні конфігурації самостійно, існують інструменти, які зроблять це за вас.

Один із найпопулярніших інструментів для налаштування проєктів — це Vite. Vite, що означає «швидкий» французькою, — це інструмент збірки, який має на меті забезпечити швидший процес розробки сучасних вебпроєктів. Vite можна використовувати з React, а також з іншими бібліотеками та фреймворками, такими як Vue або Svelte. Ви навіть можете використовувати його з проєктами на чистому JavaScript.

Щоб створити новий проєкт за допомогою Vite, потрібно скористатися командним рядком. Якщо ви використовуєте Windows, то можете скористатися Command Prompt або Windows PowerShell. Якщо ви користуєтеся Mac, то можна використовувати додаток Terminal.

Якщо у вас не встановлено Node.js, який потрібен для роботи npm, ось посилання на офіційну сторінку Node.js: https://nodejs.org. Дотримуйтесь інструкцій з встановлення, а потім поверніться сюди, щоб продовжити встановлення React.

Коли командний рядок відкрито, ви можете використати таку команду:

```bash
npm create vite@latest my-react-app -- --template react
```

Ця команда створює новий проєкт React з іменем `my-react-app` за допомогою шаблону React від Vite.

Потім ви можете відкрити новий проєкт і побачити шаблонний код React, який Vite підготував для вас.

Головна перевага Vite в тому, що він надає лише ті файли, які абсолютно необхідні для початку роботи з вашим проєктом React.

Щоб фактично запустити проєкт у поточному вигляді, потрібно встановити залежності за допомогою таких команд у командному рядку:

```bash
cd my-react-app
npm install
```

Команда `cd my-react-app` гарантує, що ви перейшли до правильної папки, де знаходиться ваш проєкт.

Команда `npm install` використовується для встановлення залежностей, необхідних для правильного запуску вашого проєкту React. Залежності — це бібліотеки або пакунки, які потрібні вашому проєкту React для коректної роботи, наприклад, сам React, ReactDOM або інші сторонні пакунки, що додають додатковий функціонал.

Виконання `npm install` прочитає файл `package.json` у папці вашого проєкту та встановить усі залежності, які там перераховані, що дозволить вам збирати та запускати ваш застосунок React без відсутніх компонентів.

Файл `package.json` — це ключовий конфігураційний файл у проєктах, який містить метадані про ваш проєкт, включно з його назвою, версією та залежностями. Він також визначає скрипти, інформацію про ліцензію та інші налаштування, які допомагають керувати проєктом і його залежностями.

Після встановлення залежностей ви повинні побачити нову папку у вашому проєкті з назвою `node_modules`.

Папка `node_modules` — це місце, де зберігаються всі пакунки та бібліотеки, необхідні вашому проєкту. Ця папка містить фактичний код залежностей, перерахованих у вашому файлі `package.json`, включно з прямими залежностями вашого проєкту та залежностями цих залежностей.

Щоб запустити проєкт, виконайте команду `npm run dev` і відкрийте нову вкладку браузера за адресою `http://localhost:5173/`.

Ви побачите стартовий шаблон, який Vite підготував для вас.

Якщо потрібно зупинити локальний сервер, натисніть `CTRL + C` у командному рядку.

Щоб побачити код цього стартового шаблону, зайдіть у ваш проєкт у папці `src`, і ви побачите файл `App.jsx`.

Звідти ви можете почати змінювати файл, зберігати зміни і бачити оновлення у браузері.

І з цим ви готові почати створювати свій застосунок React!
