# --interactive--

**ПРИМІТКА**: Деякі інтерактивні приклади можуть використовувати CSS, який ви ще не вивчали. Не хвилюйтеся, якщо не розумієте весь код. Мета прикладів — показати вам попередній перегляд цих концепцій дизайну, щоб ви краще розуміли, як усе працює.

Коли ви створюєте вебсторінки, важливо робити цілісні та візуально привабливі дизайни. Один зі способів досягти цього — використання вирівнювання.

Вирівнювання — це процес розташування тексту та зображень таким чином, щоб створити візуальний зв’язок між елементами.

Воно допомагає створити відчуття порядку та організації на сторінці, що полегшує користувачам навігацію та розуміння вмісту.

Існує кілька типів вирівнювання, які ви можете використовувати, але основні з них:

- ліве вирівнювання
- центроване вирівнювання
- праве вирівнювання
- вирівнювання за шириною
- вертикальне вирівнювання

Ліве, праве та центроване вирівнювання — це підтипи горизонтального вирівнювання, тоді як вертикальне вирівнювання використовується для вирівнювання елементів уздовж вертикальної осі.

Давайте детальніше розглянемо кожен тип вирівнювання і як ви можете їх використовувати у своїх дизайнах.

Ліве вирівнювання зазвичай використовується з текстом, коли кожен елемент вирівнюється по лівому полю. Вирівнювання всіх заголовків і абзаців на вебсторінці по лівому полю полегшує користувачеві читання та сприйняття вмісту.

:::interactive_editor

```html
<style>
  body {
    font-family: Arial, sans-serif;
    padding: 20px;
    background-color: #fff;
    color: #222;
    max-width: 700px;
    margin: 0 auto;
  }

  h1, h2, h3, p {
    text-align: left;
    margin: 10px 0;
  }

  p {
    line-height: 1.5;
  }
</style>

<h1>Why Left Alignment Matters</h1>
<p>Left alignment is one of the most common and effective ways to present text on web pages.</p>

<h2>Consistency</h2>
<p>Aligning all headings and paragraphs to the left margin creates a clean and consistent reading flow.</p>

<h3>Easy to Follow</h3>
<p>Users can easily scan and follow content without confusion or misalignment.</p>
```

:::

Протилежністю лівого вирівнювання є праве вирівнювання, коли кожен елемент вирівнюється по правому полю. Це часто використовується на сайтах для відображення додаткового вмісту, наприклад рекламних банерів або оголошень.

:::interactive_editor

```html
<style>
  body {
    font-family: Arial, sans-serif;
    padding: 20px;
    background-color: #fff;
    color: #222;
    max-width: 700px;
    margin: 0 auto;
  }

  h1, h2, h3, p {
    text-align: right;
    margin: 10px 0;
  }

  p {
    font-size: 16px;
    line-height: 1.5;
  }
</style>

<h1>Right Alignment in Web Design</h1>
<p>Right alignment is commonly used to display additional or promotional content on websites.</p>

<h2>Secondary Content</h2>
<p>Aligning text to the right margin can help separate it visually from the main content.</p>

<h3>Promotional Use</h3>
<p>This alignment is often chosen for banners, advertisements, or sidebar messages.</p>
```

:::

Наприклад, реклама, вирівняна по правому полю, виділяється серед іншого вмісту на сторінці, але не відволікає користувача від основного вмісту.

Центроване вирівнювання — це коли елементи розташовані по центру сторінки. Його часто використовують для заголовків, логотипів та інших важливих елементів, на які хочуть звернути увагу.

:::interactive_editor

```html
<style>
  body {
    font-family: Arial, sans-serif;
    padding: 20px;
    background-color: #fff;
    color: #222;
    max-width: 700px;
    margin: 0 auto;
  }

  h1, h2, h3, p {
    text-align: center;
    margin: 10px 0;
  }

  p {
    font-size: 16px;
    line-height: 1.5;
    max-width: 600px;
    margin-left: auto;
    margin-right: auto;
  }
</style>

<h1>Center Alignment for Impact</h1>
<p>Center alignment is perfect for drawing attention to headings and important elements.</p>

<h2>Eye-Catching Layout</h2>
<p>By centering key content, you make it stand out and create a balanced visual appeal.</p>

<h3>Common Uses</h3>
<p>This technique is often used for logos, hero headings, and call-to-action messages.</p>
```

:::

Вирівнювання за шириною — це коли текст вирівнюється одночасно по лівому та правому полям. Зазвичай його застосовують для описових текстів або статей, що створює чистий і професійний вигляд.

:::interactive_editor

```html
<style>
  body {
    font-family: Georgia, serif;
    padding: 20px;
    background-color: #fff;
    color: #222;
    max-width: 700px;
    margin: 0 auto;
  }

  h1 {
    text-align: center;
    margin-bottom: 20px;
  }

  p {
    text-align: justify;
    line-height: 1.7;
    margin-bottom: 20px;
  }
</style>

<h1>Justified Alignment for Clean Text Blocks</h1>

<p>
  Justified alignment ensures that each line of text stretches evenly between the left and right margins. 
  This creates a smooth, block-like appearance that looks very polished and professional. 
  It’s commonly used in newspapers, magazines, and formal documents where a uniform look is desired.
</p>

<p>
  However, care should be taken with justified text to avoid uneven spacing or "rivers" of white space, 
  especially on narrow columns or screens. Proper hyphenation and responsive design can help maintain readability.
</p>
```

:::

Останній тип вирівнювання — вертикальне вирівнювання, яке використовується для вирівнювання елементів уздовж вертикальної осі.

Вертикальне вирівнювання можна застосувати, наприклад, для контактної форми на сайті. Вирівнювання всіх вхідних даних форми, таких як ім’я, е-пошта та поля повідомлення, уздовж вертикальної осі полегшує користувачеві заповнення форми.

:::interactive_editor

```html
<style>
  body {
    font-family: Arial, sans-serif;
    padding: 20px;
    background-color: #fff;
    color: #222;
    max-width: 400px;
    margin: 0 auto;
  }

  form {
    display: flex;
    flex-direction: column;
    gap: 15px;
  }

  label {
    font-weight: bold;
    margin-bottom: 5px;
  }

  input, textarea {
    padding: 8px;
    border: 1px solid #ccc;
    border-radius: 4px;
    width: 100%;
    box-sizing: border-box;
  }

  button {
    padding: 10px;
    background-color: #007bff;
    border: none;
    border-radius: 4px;
    color: white;
    cursor: pointer;
  }

  button:hover {
    background-color: #0056b3;
  }
</style>

<form>
  <label for="name">Name</label>
  <input type="text" id="name" name="name" placeholder="e.g., Jane Doe">

  <label for="email">Email</label>
  <input type="email" id="email" name="email" placeholder="janedoe@example.com">

  <label for="message">Message</label>
  <textarea id="message" name="message" rows="4" placeholder="Write your message here"></textarea>

  <button type="submit">Submit</button>
</form>
```

:::

Використовуючи різні типи вирівнювання, ви можете створити відчуття порядку та організації на сторінці, що полегшує користувачам навігацію та розуміння вмісту.
