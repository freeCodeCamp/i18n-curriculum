# --description--

本任务侧重于复习使用 `can` 表达能力以及过去进行时态中动词 `try` 的用法。

`Can` 用来表示某人有能力做某事，比如 `I can solve this problem.`

`Try` 的过去进行时（`was`/`were` + 动词的 `-ing` 形式）表示过去正在进行的动作，例如 `I was trying to fix the issue.`
