# --description--

`transform` 속성에 사용할 수 있는 또 다른 값은 `skewX`이며, 이 함수는 요소를 수평으로 기울입니다.

```css
div {
  transform: skewX(7deg);
}
```

호버할 때 `section` 요소를 대상으로 하는 선택자를 추가하세요. `transform` 속성을 `skewX(10deg)`로 설정하세요.

# --hints--

`section:hover` 선택자가 있어야 합니다.

```js
assert.exists(new __helpers.CSSHelp(document).getStyle("section:hover"));
```

`section:hover` 선택자는 `transform` 속성을 `skewX(10deg)`로 설정해야 합니다.

```js
assert.strictEqual(
  new __helpers.CSSHelp(document).getStyle("section:hover")?.getPropertyValue("transform"),
  "skewX(10deg)"
);
```
