# --description--

في الدروس القليلة القادمة، سنتعلم عن القوائم، الصفوف المرتبة، والنطاقات، وهي ثلاثة أنواع أساسية من المتتاليات المستخدمة في Python.

نوع بيانات القائمة هو متتالية مرتبة من العناصر التي يمكن أن تتكون من سلاسل نصية، أرقام، أو حتى قوائم أخرى. القوائم قابلة للتغيير وتستخدم فهرسة تبدأ من الصفر، مما يعني أن العنصر الأول في القائمة يقع عند الفهرس صفر.

إليك الصيغة النحوية الأساسية للقائمة:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
```

للوصول إلى عنصر من قائمة `cities`، يمكنك الإشارة إلى رقم فهرسه في المتتالية. إليك مثالًا على الوصول إلى العنصر الأول في قائمة `cities`:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # 'Los Angeles'
```

تُستخدم الفهرسة السالبة للوصول إلى العناصر بدءًا من نهاية القائمة بدلاً من البداية عند الفهرس `0`. للوصول إلى العنصر الأخير في أي قائمة، يمكنك استخدام `-1` هكذا:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # 'Tokyo'
```

طريقة أخرى لإنشاء قائمة هي استخدام دالة البناء `list()`. تُستخدم دالة البناء `list()` لتحويل كائن قابل للتكرار إلى قائمة هكذا:

```py
developer = 'Jessica'
list(developer) # ['J', 'e', 's', 's', 'i', 'c', 'a']
```

الكائن القابل للتكرار هو نوع خاص من الكائنات يمكن التكرار عليه عنصرًا واحدًا في كل مرة. ستتعلم المزيد عن الحلقات في Python لاحقًا.

للحصول على العدد الإجمالي للعناصر في القائمة، يمكنك استخدام الدالة `len()` هكذا:

```py
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

إذا أردت تحديث قيمة عند فهرس معين، يمكنك فعل شيء مثل هذا:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

بما أن القوائم قابلة للتغيير، يمكنك تحديث أي عنصر في القائمة طالما تمرر رقم فهرس صالح. إذا مررت بفهرس (سواء كان موجبًا أو سالبًا) خارج حدود القائمة، فستتلقى خطأ `IndexError`:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list assignment index out of range
"""
```

إذا أردت إزالة عنصر من القائمة، يمكنك استخدام الكلمة المفتاحية `del` هكذا:

```py
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

أحيانًا يكون من المفيد التحقق مما إذا كان عنصر موجودًا داخل القائمة. للقيام بذلك، يمكنك استخدام الكلمة المفتاحية `in` هكذا:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

أحيانًا يكون من الشائع وجود قوائم مسكنة داخل قوائم أخرى هكذا:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

في هذا المثال، لدينا قائمة مسكنة واحدة تحتوي على ثلاث لغات برمجة شائعة. للوصول إلى القائمة المسكنة، ستحتاج إلى الوصول إليها باستخدام الفهرس `2` لأن القوائم تستخدم فهرسة تبدأ من الصفر:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

ثم للوصول إلى اللغة الثانية من تلك القائمة المسكنة، ستحتاج إلى الوصول إليها باستخدام الفهرس `1` هكذا:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # 'Rust'
```

تقنية شائعة أخرى تستخدم مع القوائم هي تفكيك القيم.

تفكيك القيم من قائمة هو تقنية تُستخدم لتعيين قيم من القائمة إلى متغيرات جديدة. إليك مثالًا على تفكيك قائمة `developer` إلى متغيرات جديدة تسمى `name`، `age` و `job`.

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

هنا، المتغير `name` يحمل القيمة `'Alice'`، و`age` يحمل القيمة `34`، و`job` يحمل القيمة `'Rust Developer'`.

إذا كنت بحاجة إلى جمع أي عناصر متبقية من القائمة، يمكنك استخدام معامل النجمة (`*`) هكذا:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

في هذا المثال، سيظل المتغير `name` يحمل القيمة `'Alice'`، و`rest` هي قائمة تحتوي على عنصرين: الرقم `34` والسلسلة النصية `'Rust Developer'`.

إذا لم يتطابق عدد المتغيرات على الجانب الأيسر من معامل التعيين مع العدد الإجمالي للعناصر في القائمة، فستتلقى خطأ `ValueError`:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job, city = developer

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: not enough values to unpack (expected 4, got 3)
"""
```

المفهوم الأخير الذي سننظر إليه هو معامل القطع (`:`). مثل السلاسل النصية، يمكنك الوصول إلى أجزاء من القائمة باستخدام معامل القطع هكذا:

```py
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie', 'Brownies']
desserts[1:4] # ['Cookies', 'Ice Cream', 'Pie']
```

في هذا المثال، يبدأ الفهرس عند `1` لأنه يشير إلى العنصر الثاني في القائمة. ثم نستخدم معامل القطع متبوعًا بفهرس نهاية `4`، والذي يشمل كل شيء حتى (ولكن لا يشمل) العنصر عند ذلك الفهرس.

شيء آخر يمكنك فعله باستخدام معامل القطع `:` هو تحديد فترة خطوة تحدد مقدار الزيادة بين الفهارس. لنفترض أن لديك قائمة أرقام هكذا:

```py
numbers = [1, 2, 3, 4, 5, 6]
```

إذا أردت استخراج قائمة من الأرقام الزوجية فقط، يمكنك استخدام معامل القطع هكذا:

```py
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

أول رقم زوجي يقع عند الفهرس `1`، لذا سيكون هذا هو فهرس البداية. بما أننا نريد المرور حتى نهاية القائمة، فإننا نحذف فهرس النهاية. أخيرًا، نحدد `2` لفترة الخطوة الاختيارية بحيث تزيد بمقدار `2` فقط بدلاً من القيمة الافتراضية `1`.

القوائم هي هيكل بيانات مفيد ومرن ستستخدمه كثيرًا في برامج Python الخاصة بك. في الدرس التالي، ستتعلم عن الطرق الشائعة التي يمكنك استخدامها مع القوائم.
