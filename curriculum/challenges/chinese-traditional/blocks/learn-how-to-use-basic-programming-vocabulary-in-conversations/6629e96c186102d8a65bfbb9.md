# --description--

`There is` 和 `there are` 用來顯示某物的存在或出現。`There is` 用於單數或不可數名詞，而 `there are` 用於複數名詞。

範例：

- 桌上有一本書。

- 玻璃杯裡有一些水。

- `There are three apples in the basket.` (複數名詞)
