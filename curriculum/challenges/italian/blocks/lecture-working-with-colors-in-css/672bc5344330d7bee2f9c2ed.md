# --interactive--

Quando lavori con il CSS per definire lo stile di una pagina web, uno dei metodi più comuni per definire i colori è usare i valori di colore esadecimali. I valori di colore esadecimali, o codici esadecimali, sono un modo conciso per rappresentare i colori nel modello di colore RGB. In questa lezione, esploreremo cosa sono i codici esadecimali, come funzionano in CSS e quando usarli.

Un codice esadecimale (abbreviazione di codice esadecimale) è una stringa di sei caratteri usata per rappresentare i colori nel modello di colore RGB. "Esadecimale" si riferisce al sistema di numerazione in base 16, che usa le cifre `0` a `9` e le lettere `A` a `F`.

Nel contesto dei colori, i codici esadecimali specificano le quantità di rosso, verde e blu (RGB) che compongono un colore particolare. Ogni codice esadecimale inizia con un simbolo cancelletto (`#`), seguito da sei caratteri che definiscono l’intensità di rosso, verde e blu nel colore.

Per esempio, il codice esadecimale per il nero è `#000000`, e il codice esadecimale per il bianco è `#FFFFFF`. I primi due caratteri rappresentano il valore del rosso, i successivi due il verde e gli ultimi due il blu.

In CSS, puoi applicare colori agli elementi usando i codici esadecimali. Ecco la sintassi di base:

```css
element {
  color: #RRGGBB;
}
```

Qui, `#RRGGBB` è un segnaposto per il codice esadecimale effettivo. Le coppie `RR`, `GG` e `BB` rappresentano ciascuna l’intensità di rosso, verde e blu, rispettivamente.

Queste coppie possono variare da `00` (l’intensità più bassa) a `FF` (l’intensità più alta). Più alto è il numero, maggiore sarà la quantità di quel colore presente nella miscela finale. Per esempio, applichiamo alcuni colori esadecimali a pochi elementi:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h1>This is a heading.</h1>
<p>This is a paragraph.</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}
p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

Nel primo caso, `#FF5733` è un colore arancione-rossastro. Ecco la suddivisione:

- `FF` è il valore più alto per il rosso (completamente saturo).
- `57` è una quantità moderata di verde.
- `33` è una piccola quantità di blu.

Nel secondo caso, `#4CAF50` è una tonalità di verde:

- `4C` è un valore basso per il rosso.
- `AF` è un valore forte per il verde.
- `50` è un valore moderato per il blu.

I codici esadecimali sono popolari perché sono precisi e facili da usare. Ti permettono di regolare finemente i colori, rendendoli ideali per design in cui la coerenza è importante.

La maggior parte dei software di design, come Adobe Photoshop o Figma, fornisce i codici esadecimali per i colori che selezioni, il che rende facile copiare e incollare questi valori direttamente nei tuoi file CSS.

Nello sviluppo web, i codici esadecimali sono spesso usati per definire i colori del testo, gli sfondi, i bordi e altri elementi visivi. Per esempio:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example</button>
```

```css
body {
  background-color: #F0F0F0; /* Light gray background */
}

button {
  background-color: #007BFF; /* A shade of blue */
  color: #FFFFFF; /* White text */
}
```

:::

Questo garantisce che i tuoi colori rimangano coerenti in tutto il sito o nell’applicazione.

In alcuni casi, potresti notare codici esadecimali scritti in formato abbreviato, usando solo tre caratteri invece di sei. Questo è possibile quando entrambi i caratteri in ogni coppia di colore sono uguali. Per esempio, `#FF5733` non può essere abbreviato, ma `#FFFFFF` (bianco) può essere scritto come `#FFF`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h2>This is a heading.</h2>
```

```css
body {
  background-color: #000;
}

h2 {
  color: #FFF; /* White in shorthand */
}
```

:::

L’abbreviazione `#FFF` è equivalente a `#FFFFFF`. Allo stesso modo, `#000` rappresenterebbe il nero che è `#000000` nella forma completa.

I codici esadecimali sono popolari per la loro precisione e compatibilità, permettendo oltre 16 milioni di colori su tutti i browser. Sono ideali quando hai bisogno di colori specifici per il branding, vuoi controllare tonalità esatte che non sono facilmente definite da colori nominati o HSL, o quando copi colori da software di design che forniscono codici esadecimali per impostazione predefinita.
