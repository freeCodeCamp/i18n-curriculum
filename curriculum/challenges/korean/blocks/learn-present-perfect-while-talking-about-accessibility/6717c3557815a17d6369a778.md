# --explanation--

`Haven't`는 최근에 일어나지 않은 행동을 나타내기 위해 `have not` 시제로 사용되는 `Present Perfect`의 축약형입니다.

`Negative`는 부정적이거나 비판적인 것을 의미합니다. 예를 들어:

`The user left a negative review about the app's performance.` - 여기서 `negative`은 검토의 부정적인 성격을 설명합니다.

`Comment`는 누군가가 제공한 의견이나 피드백입니다. 복수형은 `comments`입니다. 예를 들어:

`I received a comment suggesting more features in the app.` - 이 문맥에서 `comment`는 앱에 대해 사용자가 제공한 피드백을 의미합니다.
