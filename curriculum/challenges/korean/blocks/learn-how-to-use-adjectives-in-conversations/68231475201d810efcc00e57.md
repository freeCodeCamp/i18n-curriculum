# --explanation--

`Adjusting to`는 새로운 상황이나 조건에 맞게 적합해지거나 변경하는 것을 의미합니다. 예를 들어:

`The layout adjusts to different browsers.` – 브라우저 설정에 맞게 자동으로 변경됩니다.

`Screen sizes`는 작은 휴대폰 화면이나 큰 데스크톱 모니터 같은 디스플레이의 물리적 크기를 뜻합니다. 예를 들어:

`The app looks good on all screen sizes.` – 디스플레이 크기에 상관없이 어떤 기기에서든 잘 작동합니다.
