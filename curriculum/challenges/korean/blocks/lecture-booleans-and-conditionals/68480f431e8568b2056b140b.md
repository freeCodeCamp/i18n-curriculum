# --description--

이전 수업에서는 비교 연산자와 조건문을 사용해 프로그램의 흐름을 제어하는 방법을 배웠습니다.

이 방법들은 매우 강력하지만, 여러 값을 한 번에 비교해야 하는 상황이 자주 발생합니다. 이럴 때는 중첩된 조건문이 생길 수 있습니다. 예를 들어:

```python
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

위 예제는 먼저 `is_citizen`가 `True`인지 확인합니다. 그렇다면 중첩된 `if` 문으로 이동해 `age`가 `18` 이상인지 검사합니다. `age`가 `18` 이상이므로 터미널에 출력되는 메시지는 `You are eligible to vote`입니다. 만약 `is_citizen`가 `False`였다면 터미널에 출력되는 메시지는 `You are not eligible to vote`이었을 것입니다.

더 복잡한 조건문을 다룰 때는 Python의 `and`, `or`, `not` 논리 연산자를 사용할 수 있습니다.

하지만 그 연산자들을 살펴보기 전에, 참 같은 값과 거짓 같은 값이 무엇인지 먼저 알아봅시다.

Python에서 모든 값은 고유한 불리언 값을 가지며, 논리적 맥락에서 `True` 또는 `False`로 취급되어야 하는 내장 감각이 있습니다. 많은 값은 **참 같은 값**으로 간주되어 논리적 맥락에서 `True`로 평가됩니다. 반면에 **거짓 같은 값**은 `False`로 평가됩니다.

몇 가지 거짓 같은 값은 다음과 같습니다:

- `None`
- `False`
- 정수 `0`
- 부동소수점 `0.0`
- 빈 문자열 `""`

0이 아닌 숫자나 빈 문자열이 아닌 값들은 참 같은 값입니다.

값이 참 같은지 거짓 같은지 확인하려면 내장 `bool()` 함수를 사용할 수 있습니다. 이 함수는 값을 명시적으로 불리언 등가물로 변환하며, 참 같은 값에 대해 `True`를, 거짓 같은 값에 대해 `False`를 반환합니다. 몇 가지 예는 다음과 같습니다:

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

참 같은 값과 거짓 같은 값을 이해했으니, 이제 불리언 연산자, 즉 논리 연산자에 대해 알아봅시다. 이 연산자들은 여러 표현식을 결합해 코드에서 더 복잡한 의사결정 논리를 만들 수 있게 해줍니다.

Python에는 세 가지 불리언 연산자가 있습니다: `and`, `or`, `not`.

먼저 `and` 연산자를 살펴봅시다.

`and` 연산자는 두 피연산자를 받아 첫 번째 피연산자가 거짓 같은 값이면 그 값을 반환하고, 그렇지 않으면 두 번째 피연산자를 반환합니다. 두 피연산자 모두 참 같은 값이어야 표현식이 참 같은 값이 됩니다.

예를 들어:

```python
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

위 예제에서 숫자 25가 터미널에 출력되는 이유는 `and` 연산자가 첫 번째 피연산자가 `True`일 때 두 번째 피연산자를 평가하기 때문입니다. `and` 연산자는 단락 평가 연산자로 알려져 있습니다. 단락 평가는 Python이 왼쪽에서 오른쪽으로 값을 검사하며 최종 결과가 결정되면 즉시 정지하는 것을 의미합니다.

`and`는 `if` 문 내에서 여러 조건이 모두 충족되는지 확인할 때 자주 사용됩니다. 앞서 예제를 중첩된 `and` 문 대신 `if` 연산자를 사용하도록 리팩터링하는 방법은 다음과 같습니다:

```python
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

위 예제에서 `is_citizen`는 `True`이고, `age >= 18`는 `True`로 평가됩니다. `and` 연산자의 두 피연산자가 모두 참 같은 값이므로 조건 `is_citizen and age >= 18`는 `True`로 평가되고, `print` 호출이 `if` 블록에서 실행됩니다.

이제 `or` 연산자를 살펴봅시다. 이 연산자는 첫 번째 피연산자가 참 같은 값이면 그 값을 반환하고, 그렇지 않으면 두 번째 피연산자를 반환합니다. `or` 표현식은 적어도 하나의 피연산자가 참 같은 값이면 참 같은 값이 됩니다. `or` 연산자도 단락 평가 연산자로 알려져 있습니다. 예를 들어:

```python
age = 19
is_employed = False

print(age or is_employed) # 19
```

다음 코드는 첫 번째 피연산자 `age`가 `True`이므로 숫자 19를 출력합니다.

하나 이상의 표현식이 `True`인지 확인해야 할 때는 다음과 같이 조건문에서 `or` 연산자를 사용할 수 있습니다:

```python
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

이 경우 `age < 18`는 `False`이지만 `is_student`는 `True`입니다. 적어도 하나의 조건이 참이므로 전체 `or` 표현식은 `True`로 평가되고, `if` 블록의 할인 메시지가 출력됩니다.

마지막으로 살펴볼 연산자는 `not` 연산자로, 단일 피연산자를 받아 그 불리언 값을 반전시킵니다. 참 같은 값은 `False`로, 거짓 같은 값은 `True`로 변환합니다. 이전에 살펴본 연산자들과 달리 `not`는 항상 `True` 또는 `False`를 반환합니다.

몇 가지 예는 다음과 같습니다:

```python
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

`not` 연산자는 조건문에서 어떤 것이 `True` 또는 `False`가 아닌지 확인할 때 흔히 사용됩니다. 예를 들어:

```python
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```

`is_admin`가 `False`이므로 `not is_admin`는 `False`가 아닌 것을 의미하며, 이는 `True`입니다. 따라서 메시지 `Access denied for non-administrators.`가 출력됩니다.

참 같은 값과 거짓 같은 값, `and`, `or`, `not` 연산자, 그리고 단락 평가를 이해했으니, 더 유연하고 읽기 쉬운 조건문을 작성할 수 있습니다.
