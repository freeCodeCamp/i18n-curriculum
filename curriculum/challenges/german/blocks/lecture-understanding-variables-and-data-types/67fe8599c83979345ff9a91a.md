# --description--

Jede Programmiersprache hat eine eingebaute Methode, Funktion, Eigenschaft oder ein Schlüsselwort, um Daten im Terminal auszugeben. In Python können Sie die `print`-Funktion verwenden, um Daten im Terminal auszugeben. Schauen wir uns die `print`-Funktion genauer an, damit Sie sie mit Vertrauen verwenden können.

Eines der ersten Dinge, die Sie tun, wenn Sie eine Programmiersprache lernen, ist, ein einfaches `Hello world!`-Programm zu schreiben. Das können Sie in Python ganz einfach mit nur der `print`-Funktion machen.

Dazu müssen Sie nur den String `Hello world!` zwischen die öffnenden und schließenden runden Klammern setzen, mit denen Sie die `print`-Funktion aufrufen:

```python
print('Hello world!') # Hello world!
```

Sie werden in zukünftigen Lektionen mehr über Strings und Funktionen in Python lernen. Für den Moment betrachten Sie Strings einfach als eine Folge von Zeichen, die entweder von einfachen (`'`) oder doppelten (`"`) Anführungszeichen umgeben sind.

Im `print('Hello world!')`-Beispiel ist der String `'Hello world!'` ein **Argument**, das an die `print`-Funktion übergeben wird. Sie können auch die `print`-Funktion verwenden, um mehrere Werte oder Argumente gleichzeitig anzuzeigen, indem Sie sie mit Kommas trennen. Zum Beispiel:

```python
print('My favorite colors are', 'blue', 'green', 'red')

# Output: My favorite colors are blue green red
```

Python fügt standardmäßig ein Leerzeichen zwischen jedem Element ein, wenn Sie sie mit Kommas trennen. Das ist hilfreich, wenn Sie mehrere Informationen zusammen ausgeben möchten.
