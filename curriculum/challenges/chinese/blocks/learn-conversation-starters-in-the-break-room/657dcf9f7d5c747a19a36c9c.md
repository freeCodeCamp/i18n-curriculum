# --description--

当你想询问当前位置是否存在特定地点或事物时，你可以使用疑问句结构 `Are there any ... around here?`。例如：

`Are there any Wi-Fi spots around here?` - 你在询问是否能在这里附近查找 Wi-Fi。

短语 `around here` 参考附近区域。
