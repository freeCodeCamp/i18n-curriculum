# --description--

Біноміальний коефіцієнт $\displaystyle\binom{{10}^{18}}{{10}^9}$ — це число з більш ніж 9 мільярдами ($9 × {10}^9$) цифр.

Нехай $M(n, k, m)$ позначає біноміальний коефіцієнт $\displaystyle\binom{n}{k}$ за модулем $m$.

Обчисліть $\sum M({10}^{18}, {10}^9, p \times q \times r)$ для $1000 < p < q < r < 5000$ та $p$, $q$, $r$ — прості числа.
