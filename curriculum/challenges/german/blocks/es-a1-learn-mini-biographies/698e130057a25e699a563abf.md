# --description--

Um auf Spanisch auszudrücken, was jemand mag, können Sie diese Struktur verwenden:

**`Le gusta` („Er/Sie mag“) + Was die Person mag.**

Das Pronomen `le` bezieht sich auf „ihn“ oder „sie“. Es macht deutlich, dass Sie über eine andere Person sprechen.

Um zu sagen, dass jemand etwas **nicht** mag, fügen Sie das Wort `no` vor `le` hinzu.

**`No le gusta` („Er/Sie mag nicht“) + Was die Person nicht mag.**

Sebastián spricht darüber, was Marisol in ihrer täglichen Arbeit gerne macht.
