# --description--

在以下方程式中，$x$、$y$ 和 $n$ 是正整數。

$$\frac{1}{x} + \frac{1}{y} = \frac{1}{n}$$

對於一個極限 $L$，我們定義 $F(L)$ 為滿足 $x &lt; y ≤ L$ 的解法數量。

我們可以驗證 $F(15) = 4$ 和 $F(1000) = 1069$。

尋找 $F({10}^{12})$。
