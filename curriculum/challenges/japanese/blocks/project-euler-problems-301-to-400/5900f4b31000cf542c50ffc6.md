# --description--

3つの部屋が自動ドアで互いに接続されています。

<img alt="自動ドアで互いに接続された3つの部屋の連続" src="https://cdn.freecodecamp.org/curriculum/project-euler/rooms-of-doom.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

各ドアはセキュリティカードで操作されます。部屋に入るとドアは自動で閉まり、そのセキュリティカードは再度使えなくなります。開始時に無制限のカードが配布されますが、各部屋（開始部屋も含む）にはスキャナーがあります。スキャナーが3枚を超えるカードを持っていることや、床に放置されたカードを検出すると、すべてのドアが永久にロックされます。ただし、各部屋にはカードを安全に保管できる箱があり、後で使うために任意の枚数のカードを預けられます。

もし単純に部屋を1つずつ通過しようとすると、3枚のカードをすべて使い切ってしまい、3番目の部屋に入った時点で永遠に閉じ込められてしまいます！

しかし、保管箱を活用すれば脱出は可能です。例えば、最初のカードで1番目の部屋に入り、カードを1枚箱に預け、3枚目のカードでスタート地点に戻ることができます。その後、配布機から3枚のカードを再び受け取り、1枚を使って1番目の部屋に入り、先ほど預けたカードを回収します。これで再び3枚のカードを持ち、残りの3つのドアを通過できます。この方法で合計6枚のカードを使い、3つの部屋すべてを通過できます。

最大3枚のカードを持ちながら、合計123枚のカードで6つの部屋を通過することが可能です。

ここで、$C$ を同時に持てるカードの最大枚数とします。

$R$ を通過する部屋の数とします。

$M(C, R)$ を、最大で$C$枚のカードを持ちながら$R$部屋を通過するために配布機から必要な最小枚数のカードとします。

例えば、$M(3, 6) = 123$、$M(4, 6) = 23$ です。

また、$3 ≤ C ≤ 4$ のとき、$\sum M(C, 6) = 146$ となります。

さらに、$3 ≤ C ≤ 10$ のとき、$\sum M(C, 10) = 10382$ が与えられています。

$3 ≤ C ≤ 40$ のときの $\sum M(C, 30)$ を求めてください。
