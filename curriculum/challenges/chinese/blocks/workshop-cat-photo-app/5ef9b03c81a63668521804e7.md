# --description--

`footer` 元素用于定义文档或章节的页脚。 页脚通常包含文档作者信息、版权数据、使用条款链接、联系信息等。

在 `main` 元素之后，添加一个 `footer` 元素。

# --hints--

你要么删除了 `main` 元素，要么它缺少开始标签或结束标签。

```js
assert.exists(document.querySelector('main'));
assert.match(code, /<\/main>/);
```

你的 `footer` 元素应该有一个开始标签。 开始标签的语法为：`<elementName>`。

```js
assert.exists(document.querySelector('footer'));
```

你的 `footer` 元素应该有一个结束标签。 结束标签在 `<` 字符之后有一个 `/`。

```js
assert.match(code, /<\/footer\>/);
```

你的 `footer` 元素应位于 `main` 元素结束标签的下方。 你把它放在别的地方了。

```js
assert.equal(document.querySelector('main')?.nextElementSibling.nodeName, 'FOOTER');
```
