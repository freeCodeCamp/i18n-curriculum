# --description--

HTML Canvas API と Socket.io を使って、このサイトと機能的に似た2Dリアルタイムマルチプレイヤーゲームを開発してください：<a href="https://secure-real-time-multiplayer-game.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://secure-real-time-multiplayer-game.freecodecamp.rocks/</a>。このプロジェクトに取り組むには、以下のいずれかの方法でコードを書いてください。

- <a href="https://github.com/freeCodeCamp/boilerplate-project-secure-real-time-multiplayer-game/" target="_blank" rel="noopener noreferrer nofollow">このGitHubリポジトリ</a>をクローンして、ローカルでプロジェクトを完成させる。
- お好きなサイトビルダーを使ってプロジェクトを完成させる。必ずGitHubリポジトリのすべてのファイルを取り込んでください。

# --instructions--

各プレイヤーが自分のアバターを動かせるセキュアなマルチプレイヤーゲームを作成してください。少なくとも1つの収集可能なアイテムがあり、プレイヤーの順位はスコアに基づいて計算されます。

詳細は以下のテストを参照してください。

ゲームがセキュアであることを必ず確認してください。以下のセキュリティ対策を含めてください。

- クライアントがMIMEタイプを推測・解析できないようにする
- XSS攻撃を防ぐ
- クライアントにウェブサイトの内容をキャッシュさせない
- ヘッダーにサイトが`PHP 7.4.3`で動作していると表示される

**注意**：`helmet@^3.21.3`はユーザーストーリーに必要です。これにより、ユーザーストーリーを達成する方法についてはHelmetの以前のバージョンのドキュメントを参照する必要があります。
