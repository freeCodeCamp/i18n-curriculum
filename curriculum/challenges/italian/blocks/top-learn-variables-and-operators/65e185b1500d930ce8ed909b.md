# --description--
Il modo più semplice per iniziare è creare un file HTML con il codice JavaScript al suo interno. Digita lo scheletro HTML di base in un file da qualche parte sul tuo computer:

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>
  <script>
    // Your JavaScript goes here!
    console.log("Hello, World!")
  </script>
</body>
</html>
```

Salva e apri questo file in un browser web (puoi usare <a href="https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer" target="_blank" >"Live Server" su Visual Studio Code</a> per farlo!) e poi apri la console del browser facendo clic con il tasto destro sulla pagina vuota e selezionando "Ispeziona" o "Ispeziona elemento". Nel pannello ‘Strumenti per sviluppatori’ trova e seleziona la scheda ‘Console’, dove dovresti vedere l’output della nostra istruzione `console.log`.

> `console.log()` è il comando per stampare qualcosa nella console per sviluppatori del tuo browser. Puoi usarlo per stampare i risultati di qualsiasi articolo ed esercizio successivo nella console. Ti incoraggiamo a scrivere codice insieme a tutti gli esempi di questa e delle prossime lezioni.

Un altro modo per includere JavaScript in una pagina web è tramite uno script esterno. Questo è molto simile al collegamento di file CSS esterni al tuo sito.

```html
  <script src="javascript.js"></script>
```

I file JavaScript hanno l’estensione `.js` simile a `.css` per i fogli di stile. I file JavaScript esterni si usano per script più complessi.

# --questions--

## --text--

Quale affermazione descrive correttamente come includere JavaScript in un documento HTML?

## --answers--

JavaScript può essere incluso solo internamente all’interno di un tag `<script>` nel corpo HTML.

---

Il codice JavaScript deve essere scritto nella sezione `<head>` di un documento HTML per essere eseguito correttamente.

---

JavaScript può essere incluso direttamente in un file HTML usando un tag `<script>` o collegato esternamente tramite un tag `<script src="javascript.js">`.

---

I file JavaScript esterni richiedono un attributo speciale nel tag `<script>` per essere riconosciuti dai browser web.
