# --description--

`Much of` використовується, щоб виразити велику частину чогось, особливо з незлічуваними іменниками. Наприклад, `He spent much of the day reading.` означає, що він витратив значну частину дня на читання.

`Free time` стосується часу, коли ви не працюєте і не виконуєте важливих завдань, і можете обирати, чим зайнятися. Наприклад, `She uses her free time to learn guitar.` показує, що вона обирає робити, коли не має інших зобов’язань.
