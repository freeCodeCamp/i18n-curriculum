# --description--

Eine Menge von Gitterpunkten $S$ wird als Titanic-Menge bezeichnet, wenn es eine Gerade gibt, die genau durch zwei Punkte in $S$ verläuft.

Ein Beispiel für eine Titanic-Menge ist $S = \\{(0, 0), (0, 1), (0, 2), (1, 1), (2, 0), (1, 0)\\}$, wobei die Gerade, die durch (0, 1) und (2, 0) verläuft, keinen weiteren Punkt in $S$ enthält.

Andererseits ist die Menge {(0, 0), (1, 1), (2, 2), (4, 4)} keine Titanic-Menge, da die Gerade, die durch zwei beliebige Punkte der Menge verläuft, auch durch die anderen beiden Punkte geht.

Für jede positive ganze Zahl $N$ sei $T(N)$ die Anzahl der Titanic-Mengen $S$, deren jeder Punkt ($x$, $y$) die Bedingung $0 ≤ x$, $y ≤ N$ erfüllt. Es lässt sich überprüfen, dass $T(1) = 11$, $T(2) = 494$, $T(4) = 33\\,554\\,178$, $T(111)\bmod {10}^8 = 13\\,500\\,401$ und $T({10}^5)\bmod {10}^8 = 63\\,259\\,062$ gilt.

Bestimmen Sie $T({10}^{11})\bmod {10}^8$.
