# --description--

このラボでは、引数として渡された情報を使って文の検索と置換を行い、新しい文を返す関数を作成します。

**注意:** 置換する際は、元の単語の最初の文字の大文字・小文字を保持してください。例えば、`Book` を `dog` に置き換える場合、`Dog` のように置き換えられるべきです。

**目的**: 以下のユーザーストーリーを満たし、すべてのテストに合格してラボを完了してください。

**ユーザーストーリー:**

1. `myReplace` という名前の関数を作成してください。
1. `myReplace` 関数は3つの引数を取る必要があります：文字列、置き換える単語、置き換える単語の代わりの単語。
1. `myReplace` 関数は置換後の新しい文字列を返す必要があります。
1. 置換する際は、元の単語の最初の文字の大文字・小文字を保持してください。

function myReplace(str, before, after) {
  if (before.charAt(0) === before.charAt(0).toUpperCase()) {
    after = after.charAt(0).toUpperCase() + after.substring(1);
  } else {
    after = after.charAt(0).toLowerCase() + after.substring(1);
  }
  return str.replace(before, after);
}
