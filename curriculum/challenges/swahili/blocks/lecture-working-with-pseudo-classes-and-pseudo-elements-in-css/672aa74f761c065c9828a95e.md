# --interactive--

Pseudo-class ni maneno muhimu maalum ya CSS yanayokuwezesha kuchagua kipengele kulingana na hali yake maalum au nafasi yake. Hali au nafasi ya kipengele inaweza kujumuisha:

- Linapokuwa hai (active).
- Linapobofyozwa na panya.
- Linapokuwa mtoto wa kwanza wa mzazi.
- Linapokuwa mtoto wa mwisho wa mzazi.
- Linapokuwa kiungo kilichotembelewa.
- Linapokuwa limezimwa.

Ili kutumia pseudo-class, unaiongeza kwenye kichaguzi kwa kutumia nukta mbili (`:`) ikifuatiwa na jina la pseudo-class:

```css
selector:pseudo-class {
  /* CSS properties */
}
```

Tuchunguze jinsi unavyoweza kutumia pseudo-class kuwakilisha kila moja ya hali na nafasi tulizotaja.

Pseudo-class ya `:active` inakuwezesha kuchagua hali ya kipengele kinachofanya kazi, kama kubofya kitufe:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example Button</button>
```

```css
button:active {
  background: greenyellow;
}
```

:::

Pseudo-class ya `:hover` inaelezea hali ya hover ya kipengele. Mfano ni pale unapobofyoza kitufe au kiungo:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="#">Hover over me!</a>  
```

```css
a:hover {
  text-decoration: none;
  color: white;
  background: crimson;
}
```

:::

Pseudo-class ya `:first-child` huchagua kipengele ambacho ni mtoto wa kwanza wa mzazi wake. Hapa kuna mfano wa kulenga aya ya kwanza ndani ya kontena la `div`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:first-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

Aya ya kwanza ndani ya `div` itapokea rangi ya nyuma ya `lightcoral` na `padding` ya `0.4rem` pande zote nne.

Pseudo-class ya `:last-child` inalenga kipengele cha mwisho ambacho ni mtoto wa mzazi wake. Hapa kuna mfano wa kulenga aya ya mwisho ndani ya kontena la `div`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:last-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

Pseudo-class ya `:visited` inakuwezesha kupangilia kiungo ambacho mtumizi tayari amekitembelea:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="https://www.example.com" target="_blank">Visit Example.com</a>
```

```css
a:visited {
  color: purple;
}
```

:::

Pseudo-class ya `:disabled` inakuwezesha kupangilia kipengele kinachoshirikiana kilichozimwa:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button disabled>Disabled Button</button>
```

```css
button:disabled {
  background-color: lightgray;  
}
```

:::

Kama unavyoona, pseudo-class zinakuwezesha kupangilia vipengele kulingana na mwingiliano wa mtumizi, kama vile hover au kutembelea kiungo. Hii hufanya tovuti yako ihisi kuwa na mwingiliano zaidi.

Mbali na pseudo-class zilizotajwa, kuna nyingine kama:

- `:focus`
- `:first-of-type`
- `:last-of-type`
- `:nth-of-type`
- `:modal`
- `:enabled`
- `:checked`
- `:required`, na zaidi.
