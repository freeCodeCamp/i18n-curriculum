# --description--

`class`는 특히 `object-oriented programming`에서 기본 개념입니다. 객체를 생성하기 위한 청사진 역할을 합니다. `class`는 `methods`에서 생성된 객체가 가질 수 있는 속성(속성)과 동작(`class`)을 정의합니다.

예를 들어, `Person` 클래스는 `name`와 `age` 같은 속성과 `talk()`와 `walk()` 같은 동작을 가질 수 있습니다.

`method`는 객체와 연관된 함수입니다. `Methods`는 `class`에서 생성된 객체가 수행할 수 있는 동작이나 행동을 정의합니다. 객체에게 무엇을 해야 하는지 알려주는 지침과 같습니다.

예를 들어, `Calculator class`에서 `method`는 두 수를 더하는 `add(number1, number2)`일 수 있습니다.
