# --explanation--

`Might` se usa para expresar posibilidad. Significa que algo es posible pero no seguro. Por ejemplo:

- `I might go to the party if I finish my work.` - Es posible que vaya, pero no estoy seguro.

- `She might call you later.` - Es posible que llame, pero no es seguro.
