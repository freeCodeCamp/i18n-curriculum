# --interactive--

Dialoge ermöglichen es Ihnen, wichtige Informationen oder Aktionen für Benutzer anzuzeigen. Mit dem eingebauten HTML-Element `dialog` können Sie diese Dialoge (sowohl modale als auch nicht-modale Dialoge) in Ihren Webanwendungen ganz einfach erstellen.

Ein modaler Dialog ist eine Art Dialog, der den Benutzer zwingt, mit ihm zu interagieren, bevor er auf den Rest der Anwendung oder Webseite zugreifen kann. Er blockiert effektiv die Interaktion mit anderem Inhalt, bis der Benutzer eine Aktion abschließt, wie das Schließen des Dialogs oder das Absenden eines Formulars.

Im Gegensatz dazu erlaubt ein nicht-modaler Dialog dem Benutzer, weiterhin mit anderen Teilen der Seite oder Anwendung zu interagieren, auch wenn der Dialog geöffnet ist. Er verhindert nicht den Zugriff auf den restlichen Inhalt.

In dieser Lektion lernen Sie, wie Sie modale Dialoge mit den Methoden `showModal()` und `close()` öffnen und schließen.

Wenn Sie sicherstellen möchten, dass der Benutzer sich auf eine bestimmte Aktion oder Nachricht eines modalen Dialogs konzentriert, können Sie den modalen Dialog mit der Methode `showModal()` öffnen. Dadurch wird ein Hintergrund für die anderen Elemente auf der Seite hinzugefügt und diese werden deaktiviert. Das ist ideal für modale Dialoge, die Formulare, Bestätigungen und wichtige Informationen anzeigen, die eine Benutzeraktion erfordern.

Hier ist das HTML für den modalen Dialog:

:::interactive_editor

```html
<dialog id="my-modal">
  <p>This is a modal dialog.</p>
</dialog>
```

:::

Im Moment sehen Sie nichts auf der Seite, weil der modale Dialog beim ersten Rendern geschlossen ist. Sie können den modalen Dialog automatisch mit der Methode `showModal()` öffnen:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
dialog.showModal();
```

:::

Das Ergebnis im Browser zeigt einen modalen Dialog mit dem Text `This is a modal dialog.`

Es ist am besten, die Kontrolle dem Benutzer zu überlassen. Um dies zu erreichen, können Sie einen Klick-Ereignis-Listener zu einem Button hinzufügen und die Methode `showModal()` verwenden:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.showModal();
});
```

:::

Wenn Sie einen Dialog anzeigen möchten, während die Interaktion mit Inhalten außerhalb des Dialogs weiterhin möglich ist, können Sie die Methode `show()` verwenden:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});
```

:::

Um einen modalen Dialog zu schließen, können Sie einen Button innerhalb des `dialog`-Elements im Dialog hinzufügen und die Methode `close()` verwenden:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
  <button id="close-modal-btn">Close Modal</button>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");
const closeButton = document.getElementById("close-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});

closeButton.addEventListener("click", () => {
  dialog.close();
});
```

:::

Was ist der Unterschied zwischen einem modalen Dialog und einem nicht-modalen Dialog?

## --answers--
