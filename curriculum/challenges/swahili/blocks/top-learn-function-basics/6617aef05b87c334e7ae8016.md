# --description--

Kama ulivyoona hapo awali, vitendakazi vinaweza kurudisha thamani kwa kutumia neno kuu la `return`. Neno kuu la `return` hutumika kurudisha thamani kutoka kwenye kitendakazi. Wakati neno kuu la `return` linapotumika, kitendakazi kitaacha kutekelezwa na kurudisha thamani iliyobainishwa baada ya neno kuu la `return`.

```js
function add(a, b) {
  return a + b
}

console.log(add(2, 3)); // Output: 5
```

Lakini nini hutokea ikiwa neno kuu la `return` litumika kabla ya mwisho wa kitendakazi?
Ili kujibu swali hili, fikiria mfano ufuatao:

```js
function add(a, b) {
  if(a > 2){
    return b;
  }

  return a + b;
}

console.log(add(3, 7)); // Output: 7
``` 

Katika mfano hapo juu, kitendakazi `add` kina tamko la masharti linalokagua kama thamani ya `a` ni kubwa kuliko `2`. Ikiwa sharti litatimizwa, kitendakazi kitarudisha thamani ya `b` na kuacha kutekelezwa. Ikiwa sharti halitatimizwa, kitendakazi kitarudisha jumla ya `a` na `b`.
