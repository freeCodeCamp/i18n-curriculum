# --description--

Die Suche nach Informationen im Internet kann eine überwältigende Aufgabe sein. Es gibt so viele Websites mit so vielen Daten, dass es unmöglich erscheinen mag, die gewünschten Ergebnisse zu erhalten.

Dank Dingen wie der Boolean-Suchsyntax können Sie Ihre Suche jedoch genau auf das eingrenzen, was Sie brauchen.

Die Boolean-Suchsyntax ist eine Methode, spezielle Wörter zu verwenden, die Ihnen helfen, genau das zu finden, wonach Sie online suchen.

Hier ist ein Beispiel für eine einfache Suchabfrage für `freecodecamp curriculum`.

Wenn Sie nach `freecodecamp curriculum` suchen, liefert die Suchmaschine Ergebnisse, die *irgendeines* dieser Schlüsselwörter enthalten – typischerweise sortiert nach Relevanz (wie wahrscheinlich es ist, dass sie mehr Ihrer Abfrage entsprechen oder damit zusammenhängen).

Aber was, wenn Sie nur Ergebnisse möchten, die beide Schlüsselwörter enthalten? Nun, Sie haben vielleicht gehört, dass Sie eine Abfrage in Anführungszeichen setzen können, wie `"freecodecamp curriculum"`.

Diese Abfrage entspricht jedoch speziell der *Phrase* `freecodecamp curriculum`. Das bedeutet, dass die Ergebnisse auf Seiten beschränkt sind, die `freecodecamp curriculum` zusammen sagen, nicht auf Seiten, die `freecodecamp` UND `curriculum` enthalten.

Wenn Sie Seiten finden möchten, die beide Begriffe enthalten, aber nicht unbedingt nebeneinander als Phrase, können Sie jeden Begriff mit einem Pluszeichen als Präfix versehen, so: `+freecodecamp +curriculum`

Alternativ interessieren Sie sich vielleicht für die Initiativen von freeCodeCamp außerhalb des Curriculums. In diesem Fall können Sie den Begriff `curriculum` mit einem Minuszeichen als Präfix versehen, so: `+freecodecamp -curriculum`.

Dies liefert Ergebnisse, die `freecodecamp` erwähnen, aber Ergebnisse ausschließen, die auch `curriculum` erwähnen.

Oder vielleicht haben Sie eine bestimmte Website, die Sie durchsuchen möchten, wie unsere Nachrichtenplattform? Sie können `site:` gefolgt von der URL der Website verwenden, die Sie durchsuchen möchten: `site:freecodecamp.org/news curriculum`.

Diese Abfrage liefert alle Ergebnisse von `freecodecamp.org/news`, die das Wort `curriculum` enthalten.

Indem Sie diese Werkzeuge kombinieren, können Sie sehr spezifische Abfragen erstellen, um genau die Informationen zu erhalten, die Sie benötigen.

Zum Beispiel würde die Abfrage `site:freecodecamp.org/news +python -curriculum` nach Python-Artikeln auf unserer Nachrichtenseite suchen und Artikel ausschließen, die sich mit unserem Curriculum befassen.

Suchoperatoren können variieren, daher ist es immer wichtig, die Dokumentation Ihrer bevorzugten Suchmaschine zu überprüfen.

Aber jetzt sollten Sie in der Lage sein, genau die Ergebnisse zu erhalten, die Sie bei Ihren Websuchen wünschen.
