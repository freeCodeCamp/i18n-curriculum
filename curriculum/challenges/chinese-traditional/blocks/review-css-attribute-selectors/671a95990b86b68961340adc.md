## 使用不同屬性選擇器和聯結

- **定義**：`attribute` 選擇器允許你根據 HTML 元素的屬性（例如 `href` 或 `title` 屬性）來標的元素。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Link with href</a>
<a>No href</a>
```

```css
a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

- **`title` 屬性**：此屬性提供關於元素的額外資訊。以下是你如何以帶有 `title` 屬性的聯結為標的：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Tooltip">Link with title</a>
<a href="#">Normal link</a>
```

```css
a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

- **組合選擇器以匹配同時具有 `href` 和 `title` 屬性的聯結**：組合多個屬性選擇器。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Info">Href + Title</a>
<a href="#">Only href</a>
```

```css
a[href][title] {
  color: green;
}
```

:::

- **在以空格分隔的列表中匹配單一字**：標的具有特定類別字的聯結。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a class="link primary">Primary link</a>
<a class="link secondary">Secondary link</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

- **比對以特定前置式開頭的值**：標的聯結以 `https://` 開頭。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">HTTPS link</a>
<a href="http://www.freecodecamp.org">HTTP link</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

- **比對以特定字尾結尾的值**：標的聯結以 `.jpg` 結尾。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="photo.jpg">Image link</a>
<a href="index.html">HTML link</a>
```

```css
a[href$=".jpg"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

- **比對包含子字串的值（任意位置）**：標的為值中任意位置包含 `https` 的聯結。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Secure link</a>
<a href="page.html">Local link</a>
```

```css
a[href*="https"] {
  color: teal;
}
```

:::

- **摘要**：這些樣式讓你能輕鬆根據聯結的屬性和值一致地套用樣式。

## 使用 `lang` 和 `data-lang` 屬性來標的元素

- **`lang` 屬性**：此屬性用於 HTML 中，以指定元素內內容的語言。你可能會根據內容所使用的語言，對元素進行不同的樣式設定，特別是在多語言網站上。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">English text</p>
<p lang="fr">French text</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

- **`data-lang` 屬性**：像 `data-lang` 屬性這樣的訂製資料屬性通常用於在元素中儲存額外的資訊，例如指定特定文字區段中使用的語言。以下是如何根據 `data-lang` 屬性來設定元素的樣式：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div data-lang="fr">French content</div>
<div data-lang="en">English content</div>
```

```css
div[data-lang="fr"] {
  color: blue;
}
```

:::

## 使用屬性選擇器、有序列表元素和 `type` 屬性

- **`type` 屬性**：在處理 HTML 中的有序列表時，`type` 屬性允許你指定使用的編號樣式，例如數字、字母或羅馬數字。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Alpha</li>
  <li>Beta</li>
</ol>

<ol type="i">
  <li>One</li>
  <li>Two</li>
</ol>
```

```css
/*Example targeting uppercase alphabetical numbering*/
ol[type="A"] {
  color: purple;
  font-weight: bold;
}

/*Example targeting lowercase Roman numerals*/
ol[type="i"] {
  color: green;
}
```
