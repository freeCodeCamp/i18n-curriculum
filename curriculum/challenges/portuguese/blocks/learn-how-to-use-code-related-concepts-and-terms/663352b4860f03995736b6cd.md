# --description--

`Store information` significa manter ou salvar dados em um local específico, geralmente em uma variável.Variáveis são como recipientes no seu código que armazenam informações, como números, texto ou outros tipos de dados.Você pode pensar neles como locais de armazenamento nomeados onde você pode acessar a `stored information` mais tarde no seu código.

Por exemplo, `int age = 30;` Aqui, a variável `age` armazena o número 30.

A palavra `like` é usada como preposição para dar exemplos ou para comparar coisas semelhantes.

Por exemplo, se você disser, `I love fruits like apples and oranges.` Você está usando `like` para dar exemplos do tipo de frutas que você gosta, sugerindo que pode haver mais frutas nessa lista.
