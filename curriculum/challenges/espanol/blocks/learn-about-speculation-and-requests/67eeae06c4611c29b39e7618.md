### --feedback--

Estas tres palabras se usan para hablar de algo que habría sido una buena idea en el pasado pero no se hizo. La tercera palabra es el `Past Participle` de `run`.

# --explanation--

`Should have` + `Past Participle` se usa para expresar arrepentimiento o consejo sobre el pasado. Por ejemplo:

- `You should have run the program on a different machine to check for compatibility.` - Habría sido útil probar en una máquina diferente, pero no se hizo.

- `I should have run the tests before deploying the app.` - Ejecutar las pruebas era lo correcto, pero no se hizo en ese momento.

En este caso, Sophie le está diciendo a Mark que debería haber ejecutado la suite de pruebas, lo que significa que era importante, pero él no lo hizo.
