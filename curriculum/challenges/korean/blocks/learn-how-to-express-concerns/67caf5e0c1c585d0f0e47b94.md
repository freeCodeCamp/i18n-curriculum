# --explanation--

가정법 문장을 설명할 때 `Second Conditional` 구조를 사용합니다 (Maria의 말에서는 `It'd be helpful if we had...`). 두 번째 조건문의 구조는 `It would` + 기본 동사 ... `if` + 주어 + `Past Simple` 시제 동사입니다. 현재나 미래의 비현실적이거나 가상의 상황을 설명할 때 사용합니다. 예를 들어:

`We would finish earlier if we had more team members.` - 이는 팀원이 더 많지 않지만, 만약 더 많다면 더 빨리 끝낼 수 있다는 뜻입니다. 가정을 말하는 것입니다.
