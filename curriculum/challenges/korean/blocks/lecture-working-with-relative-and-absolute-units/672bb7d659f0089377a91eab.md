# --interactive--

CSS에서 백분율은 부모 요소에 대한 비율로 크기, 치수 및 기타 속성을 정의할 수 있는 상대 단위입니다. 백분율 값을 사용하면 본질적으로 "이것을 컨테이너의 X% 크기로 만들어라"라고 말하는 것과 같습니다.

예를 들어, 요소에 `width: 50%`를 설정하면 부모 컨테이너 너비의 절반을 차지하게 됩니다. 이 때문에 백분율은 다양한 화면 크기에 맞춰 조정되는 반응형 디자인을 만들 때 매우 유용합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <div class="box"></div>
</div>
```

```css
.container {
  width: 400px;
  height: 200px;
  background-color: lightgray;
}

.box {
  width: 50%;
  height: 100%;
  background-color: red;
}
```

:::

백분율은 다양한 화면 크기에 맞춰 조정되는 유동적인 레이아웃을 만드는 데 이상적입니다. 예를 들어, 컨테이너에 `width: 80%`를 설정하면 기기에 상관없이 부모 너비의 80%를 차지하게 됩니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="parent">
  <div class="child"></div>
</div>
```

```css
.parent {
  width: 100%;
  height: 300px;
  background-color: lightblue;
}

.child {
  width: 80%;
  height: 100%;
  background-color: red;
}
```

:::

유연한 이미지에 백분율을 사용하는 것도 흔한 방법입니다. 이미지에 `max-width: 100%`를 적용하면 작은 화면에서 크기가 줄어들면서도 종횡비를 유지할 수 있습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<img src="https://placehold.co/150x150" alt="Example Product Image" />
```

```css
img {
  max-width: 100%;
  height: auto;
}
```

:::

덜 흔하지만, 백분율은 폰트 크기에도 사용하여 확장 가능한 타이포그래피를 만들 수 있습니다. 예를 들어, `font-size: 120%`는 텍스트를 부모 폰트 크기보다 20% 더 크게 만듭니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="text-container">
  Parent text.
  <p class="text">This is some example text.</p>
</div>
```

```css
.text-container {
  font-size: 16px;
}

.text {
  font-size: 120%;
}
```

:::

백분율은 수직 중앙 정렬에도 특히 유용할 수 있습니다. 다음 예시는 `transform` 속성과 함께 백분율을 사용해 요소를 수직으로 중앙에 배치하는 방법입니다.

이 예시는 요소를 컨테이너 상단에서 50% 위치에 두고, `transform`를 사용해 요소 높이의 절반만큼 다시 위로 이동시켜 수직 중앙 정렬을 효과적으로 구현합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="centered"></div>
```

```css
.centered {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
  width: 300px;
  height: 300px;
  background-color: red;
}
```

:::

절대 위치 지정과 transform 속성에 대해서는 앞으로 수업에서 더 자세히 배우게 됩니다.

백분율은 항상 어떤 기준에 상대적이라는 점을 기억하세요. `width` 같은 수평 속성은 부모 너비에 상대적입니다. `height` 같은 수직 속성은 보통 부모 높이에 상대적입니다(부모 높이가 지정된 경우).

하지만 백분율 기반 치수를 가진 요소를 중첩할 때는 예상치 못한 결과가 발생할 수 있으니 주의하세요. 또한 부모에 높이가 정의되어 있지 않으면 백분율 기반 높이는 다루기 까다로울 수 있습니다.

요약하면, CSS에서 백분율은 유연하고 반응형 디자인을 만드는 강력한 도구입니다. 요소가 컨테이너에 비례해 크기 조절되도록 하거나 다양한 화면 크기에 매끄럽게 적응하는 레이아웃이 필요할 때 사용하세요.
