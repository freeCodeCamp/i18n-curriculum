# --description--

`<strong>` 요소는 텍스트를 굵게 만듭니다. 또한 텍스트를 중요하다고 의미론적으로 표시하는데, 이는 시각 장애가 있는 사용자가 웹사이트를 이용할 때 의존하는 화면 읽기 프로그램 같은 도구에 영향을 줍니다. 일부 화면 읽기 프로그램에서는 `strong` 요소 안의 텍스트 중요성을 전달하기 위해 음성 톤이 바뀝니다. `strong` 요소를 정의하려면 텍스트 내용을 `<strong>` 태그로 감싸면 됩니다.
