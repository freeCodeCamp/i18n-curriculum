# --description--

마커가 페이지 중앙에 위치하면 더 보기 좋습니다. 이를 쉽게 하는 방법은 `margin` <dfn>단축 속성</dfn>을 사용하는 것입니다.

`margin-top`과 `margin-left` 같은 속성으로 요소의 여백 영역을 개별적으로 설정할 수 있지만, `margin` 단축 속성은 여러 여백 영역을 한꺼번에 설정하기 쉽게 만듭니다.

마커를 페이지 중앙에 맞추려면 `margin` 속성을 `auto`로 설정하세요. 그러면 `margin-top`, `margin-right`, `margin-bottom`, `margin-left`가 모두 `auto`로 설정됩니다.

# --hints--

`.marker` CSS 규칙에 `margin` 속성이 `auto`로 설정되어 있어야 합니다.

```js
assert.strictEqual(new __helpers.CSSHelp(document).getStyle('.marker')?.margin, 'auto');
```
