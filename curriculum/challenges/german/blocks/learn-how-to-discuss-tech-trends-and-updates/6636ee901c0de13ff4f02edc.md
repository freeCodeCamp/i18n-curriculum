# --description--

Der Begriff `run code` bezeichnet die Aktion, ein Programm oder Skript von einem Computer auszuführen oder zu betreiben. `run` bedeutet in diesem Zusammenhang, den Prozess zu starten und so lange durchzuführen, bis das Programm das tut, wofür es entwickelt wurde, sei es Zahlen zu berechnen, Daten zu sortieren oder etwas anderes.

Der Ausdruck `run code without managing servers` bezieht sich auf das Ausführen von Computerprogrammen oder Anwendungen, ohne sich um die physischen oder virtuellen Server kümmern zu müssen, auf denen der Code läuft. Dies ist ein zentraler Aspekt des serverlosen Rechnens, bei dem der Cloud-Anbieter die Server verwaltet, sodass Entwickler sich mehr auf das Programmieren als auf die Serververwaltung konzentrieren können.
