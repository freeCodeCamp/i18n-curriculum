# --description--

Якщо функцію викликають, але аргумент не надають, тоді параметру буде призначене значення `undefined`. Це може призвести до несподіваних результатів, якщо функція не розроблена для обробки такого випадку.

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal()); // Output: undefined is my favorite animal!
```

У наведеному вище прикладі функція `favoriteAnimal` викликається без аргументу. Параметру `animal` призначається значення `undefined`, яке потім конкатенується з рядком `" is my favorite animal!"`.

Якщо значення не надається, можна використовувати параметри за замовчуванням. Параметри за замовчуванням дозволяють призначити параметру стандартне значення, якщо при виклику функції значення не передається.

```js
function favoriteAnimal(animal = 'Giant Panda') {
  return animal + " is my favorite animal!"
}
```

У наведеному вище прикладі стандартне значення параметра `animal` встановлено як `'Giant Panda'`. Якщо функцію `favoriteAnimal` викликають без аргументу, параметру `animal` буде призначене значення `'Giant Panda'`.
