# --explanation--

`Stronger` — це форма ступеня порівняння від `strong`, що означає, що щось має більшу потужність або ефективність. Наприклад:

`This password is stronger than the previous one because it has special characters.` — це означає, що поточний пароль важче зламати порівняно з попереднім паролем.

`Encryption` стосується перетворення даних у захищений формат, щоб запобігти несанкціонованому доступу. Наприклад:

`End-to-end encryption keeps messages private by encoding them so only the sender and receiver can read them.` — тут `encryption` означає процес захисту інформації шляхом перетворення її у закодовану форму.
