# --description--

標準的な52枚のトランプのデッキにおいて、4枚のカードのセットがバドゥギであるとは、ペアがなく、かつ同じスートのカードが2枚以上含まれていない場合を指します。

$f(n)$ を、4枚の部分集合がバドゥギであるような $n$ 枚のカードの選び方の数とします。例えば、標準の52枚のデッキから5枚を選ぶ方法は $2\,598\,960$ 通りあり、そのうち4枚の部分集合がバドゥギであるものは $514\,800$ 通りなので、$f(5) = 514800$ となります。

$4 ≤ n ≤ 13$ の範囲で $\sum f(n)$ を求めてください。
