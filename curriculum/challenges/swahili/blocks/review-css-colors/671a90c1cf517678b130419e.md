## Nadharia ya rangi

- **Maelezo ya nadharia ya rangi**: Hii ni somo la jinsi rangi zinavyoshirikiana na jinsi zinavyoathiri mtazamo wetu. Inahusisha uhusiano wa rangi, muafaka wa rangi, na athari za kisaikolojia za rangi.
- **Rangi za msingi**: Rangi hizi ambazo ni njano, bluu, na nyekundu, ni rangi za msingi ambazo rangi nyingine zote hutokana nazo.
- **Rangi za sekondari**: Rangi hizi hutokana na kuchanganya kiasi sawa cha rangi mbili za msingi. Kijani, machungwa, na zambarau ni mifano ya rangi za sekondari.
- **Rangi za tersiari**: Rangi hizi hutokana na kuchanganya rangi ya msingi na rangi ya sekondari inayopakana nayo. Njano-Kijani, Bluu-Kijani, na Bluu-Zambarau ni mifano ya rangi za tersiari.
- **Rangi za joto**: Rangi hizi ambazo ni nyekundu, machungwa, na njano, huleta hisia za faraja, joto, na utulivu.
- **Rangi za baridi**: Rangi hizi ambazo ni bluu, kijani, na zambarau, huleta hisia za utulivu, amani, na taaluma.
- **Gurudumu la rangi**: Gurudumu la rangi ni mchoro wa mduara unaoonyesha jinsi rangi zinavyohusiana. Ni chombo muhimu kwa wabunifu kwa sababu huwasaidia kuchagua mchanganyiko wa rangi.
- **Mipangilio ya rangi inayofanana (analogous)**: Mipangilio hii ya rangi huunda uzoefu wa muafaka na utulivu. Ina rangi zinazopakana kwenye gurudumu la rangi.
- **Mipangilio ya rangi ya nyongeza (complementary)**: Mipangilio hii ya rangi huleta tofauti kubwa na athari za kuona. Rangi zao ziko kwenye pande tofauti kabisa za gurudumu la rangi, zikilinganishwa na kila moja.
- **Mpangilio wa rangi wa triadic**: Mpangilio huu wa rangi una rangi zenye nguvu. Hutokana na rangi zinazokaribiana kwa umbali wa takriban sawa. Ikiunganishwa, huunda pembetatu sawa upande kwenye gurudumu la rangi.
- **Mpangilio wa rangi wa monochromatic**: Kwa mpangilio huu wa rangi, rangi zote hutokana na rangi moja ya msingi kwa kurekebisha uangavu, giza, na msongamano wake. Hii huleta hisia ya umoja na muafaka huku ikizalisha tofauti.

## Njia tofauti za kufanya kazi na rangi katika CSS

- **Rangi zilizopewa majina**: Rangi hizi ni majina ya rangi yaliyotangazwa awali yanayotambulika na vivinjari. Mifano ni `blue`, `darkred`, `lightgreen`.
- **Kitendakazi cha `rgb()`**: RGB inasimama kwa Red, Green, na Blue — rangi za msingi za mwanga. Rangi hizi tatu huunganishwa kwa nguvu tofauti kuunda rangi nyingi. Kitendakazi cha `rgb()` hukuruhusu kufafanua rangi kwa kutumia mfano wa rangi wa RGB.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>RGB Color</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

- **Kitendakazi cha `rgba()`**: Kitendakazi hiki huongeza thamani ya nne, alpha, inayodhibiti uwazi wa rangi. Ikiwa haijatolewa, thamani ya alpha ni `1` kwa chaguo-msingi.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>RGBA Background</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

- **Kitendakazi cha `hsl()`**: HSL inasimama kwa Hue, Saturation, na Lightness — sehemu kuu tatu zinazofafanua rangi.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>HSL Color</p>
```

```css
p {
  color: hsl(120, 100%, 50%);
}
```

:::

- **Kitendakazi cha `hsla()`**: Kitendakazi hiki huongeza thamani ya nne, alpha, inayodhibiti uwazi wa rangi.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>HSLA Background</div>
```


```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

- **Hexadecimal**: Msimbo wa hex (mfupi wa msimbo wa hexadecimal) ni mfuatano wa herufi sita unaotumika kuwakilisha rangi katika mfano wa rangi wa RGB. "Hex" inahusu mfumo wa nambari wa msingi-16, unaotumia tarakimu 0 hadi 9 na herufi A hadi F.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1 class="hex-text">Hex Text</h1>
<p class="hex-bg">Hex Background</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}

p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

## Sifa ya `box-shadow`

- **Maelezo**: Sifa ya `box-shadow` huweka kivuli kimoja au zaidi kuzunguka kipengee.
- **Thamani za mbali**: Lazima uweke thamani za usawa (`offset-x`) na wima (`offset-y`). Thamani chanya ya `offset-x` husogeza kivuli kulia, wakati thamani hasi husogeza kushoto. Thamani chanya ya `offset-y` husogeza kivuli chini, wakati thamani hasi husogeza juu. Ikiwa thamani ni `0`, huna haja ya kuweka kitengo.
- **Mzunguko wa blur**: Thamani hii ya hiari hudhibiti jinsi kivuli kinavyoonekana kuwa na ukungu. Ikiwa haijajumuishwa, ni `0` kwa chaguo-msingi, ambayo huunda makali makali. Thamani kubwa huleta kivuli laini zaidi.
- **Mzunguko wa spread**: Thamani hii ya hiari hudhibiti kiasi ambacho kivuli kinaenea au kupungua. Ikiwa haijajumuishwa, ni `0` kwa chaguo-msingi.
- **Rangi ya kivuli**: Unaweza kuweka rangi kwa kutumia rangi zilizopewa majina, thamani za hex, `rgb()`, `rgba()`, `hsl()`, au kitendakazi cha `hsla()`.
- **Neno muhimu la `inset`**: Kuongeza neno muhimu la `inset` huweka kivuli ndani ya kipengee badala ya nje.
- **Kuweka vivuli vingi vya boksi**: Unaweza kuweka vivuli vingi kwa kuvitenganisha kwa koma. Vivuli huwekwa kwa tabaka kutoka mbele kwenda nyuma.

### Sintaksia

```css
box-shadow: offset-x offset-y blur-radius spread-radius color;
```

## Mwinuko wa mstari na mwinuko wa mviringo

- **Mwinuko wa mstari**: Mwinuko huu huunda mchanganyiko wa polepole kati ya rangi kwenye mstari wa moja kwa moja. Unaweza kudhibiti mwelekeo wa mstari huu kwa kutumia maneno muhimu kama `to top`, `to right`, `to bottom right`, au pembe kama `45deg`. Unaweza kutumia rangi yoyote halali ya CSS na vituo vya rangi vingi kama unavyotaka.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient">Linear Gradient</div>
```

```css
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

:::

- **Mwinuko wa mviringo**: Mwinuko huu huunda mwinuko wa mduara au mviringo unaotoka katikati.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient">Radial Gradient</div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```
