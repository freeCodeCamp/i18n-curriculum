# --hints--

Ваш код має містити `link` елемент.

```js
assert.match(code, /<link/);
```

У вас має бути один елемент `link`.

```js
assert.lengthOf(document.querySelectorAll('link'), 1);
```

Ваш елемент `link` має бути всередині елемента `head`.

```js
assert.exists(document.querySelector('head > link'));
```

Ваш `link` елемент має мати атрибут `rel` зі значенням `stylesheet`.

```js
const linkElement = document.querySelector('link');
const rel = linkElement?.getAttribute("rel");
assert.strictEqual(rel, "stylesheet");
```

Ваш `link` елемент має мати атрибут `href` зі значенням `styles.css`.

```js
const linkElement = document.querySelector('link');
assert.strictEqual(linkElement?.dataset?.href, "styles.css");
```
