# --description--

多項式 $n^4 + 4n^3 + 2n^2 + 5n$ はすべての整数 $n$ に対して6の倍数になることが示せます。また、この性質を満たす最大の整数は6であることも示せます。

$M(a, b, c)$ を、すべての整数 $n$ に対して $n^4 + an^3 + bn^2 + cn$ が $m$ の倍数となる最大の $m$ と定義します。例えば、$M(4, 2, 5) = 6$ です。

また、$S(N)$ を $0 < a, b, c \leq N$ すべての $M(a, b, c)$ の和と定義します。

$S(10) = 1\,972$ および $S(10\,000) = 2\,024\,258\,331\,114$ であることを確認できます。

フィボナッチ数列 $F_k$ を次のように定義します：

- $F_0 = 0$, $F_1 = 1$ かつ
- $k \geq 2$ のとき $F_k = F_{k - 1} + F_{k - 2}$。

$2 \leq k \leq 1\,234\,567\,890\,123$ の範囲での $\sum S(F_k)$ の下9桁を求めてください。
