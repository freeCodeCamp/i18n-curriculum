# --description--

Per `depend on` si usa per dire che qualcosa può cambiare in base a cose diverse. Per esempio:

- `The project depends on good teamwork.` - Il progetto avrà successo solo se il team lavora bene insieme.

- `Developers depend on tests to find errors.` - I programmatori hanno bisogno di test per vedere se il codice ha errori.

Nota che dopo `depend` c’è sempre la preposizione `on`.
