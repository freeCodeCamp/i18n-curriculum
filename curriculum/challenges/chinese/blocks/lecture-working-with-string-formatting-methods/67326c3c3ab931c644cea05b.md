# --interactive--

在 JavaScript 中处理字符串时，经常会在字符串的开头或结尾遇到不需要的空白。 空白会影响比较、存储或显示等操作，因此了解如何有效清除空白非常重要。

在本课中，我们将探讨如何使用 JavaScript 的 `trim()`、`trimStart()` 和 `trimEnd()` 方法去除空白。

空白包括出现在字符串中不可见的空格符、制表符或换行符。 例如：

```js
let greeting = "   Hello, world!   ";
```

本例中，可见文本 `Hello, world!` 前后都有空白。

`trim()` 方法是删除字符串首尾空白的最常用方法。 这是一个示例：

:::interactive_editor

```js
let message = "   Hello!   ";
console.log(message); // "   Hello!   "
let trimmedMessage = message.trim();
console.log(trimmedMessage);  // "Hello!"
```

:::

本例中，`trim()` 方法会删除所有前导空格和尾部空格，只留下 `Hello!`。 请注意，`trim()` 不会触及字符串内的任何空白（例如单词之间的）。

有时，你可能只想删除字符串开头或结尾的空白，而不是两者都删除。 这就该用到 `trimStart()` 和 `trimEnd()` 了。

`trimStart()` 删除字符串开头（或起始）的空白。

:::interactive_editor

```js
let greeting = "   Hello!   ";
console.log(greeting);  // "   Hello!   "
let trimmedStart = greeting.trimStart();
console.log(trimmedStart);  // "Hello!   "
```

:::

`trimEnd()` 删除字符串末尾的空白。

:::interactive_editor

```js
let greeting = "   Hello!   ";
console.log(greeting);  // "   Hello!   "
let trimmedEnd = greeting.trimEnd();
console.log(trimmedEnd);  // "   Hello!"
```

:::

这些方法可以让你更精确地控制要清理字符串的哪一部分。

总之，修剪空白是在 JavaScript 中处理字符串的重要部分。 无论你是要清理输入数据还是要确保字符串比较的一致性，都可以使用 `trim()` 删除字符串两边的空白，或使用 `trimStart()` 和 `trimEnd()` 针对特定端进行处理。
