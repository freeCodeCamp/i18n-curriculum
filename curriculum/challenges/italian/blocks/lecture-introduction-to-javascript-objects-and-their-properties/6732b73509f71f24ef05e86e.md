# --interactive--

Quando lavori con JavaScript, ti capiterà spesso di incontrare strutture dati complesse che coinvolgono oggetti annidati e array all'interno di oggetti. Queste strutture possono rappresentare dati ricchi e gerarchici, ma richiedono anche una chiara comprensione di come consultare e manipolare i dati al loro interno. Esploriamo come navigare efficacemente queste strutture annidate.

Consultare le proprietà di oggetti annidati implica usare la notazione a punto o la notazione a parentesi, proprio come per consultare proprietà di oggetti semplici. Tuttavia, dovrai concatenare questi accessi per scendere nella struttura annidata.

Ad esempio, considera un oggetto annidato che rappresenta una persona con informazioni di contatto:

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};
```

Per consultare il numero di telefono del lavoro di `Alice`, concatenare gli accessi alle proprietà in questo modo:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person.contact.phone.work); // "098-765-4321"
```

:::

Puoi anche usare la notazione a parentesi, particolarmente utile quando i nomi delle proprietà includono spazi o caratteri speciali, o quando usi variabili per consultare le proprietà:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person['contact']['phone']['work']); // "098-765-4321"
```

:::

Ora, diamo un’occhiata a come possiamo consultare dati in cui una delle proprietà dell’oggetto ha come valore un array. Ecco un oggetto `person` modificato che include un array di indirizzi:

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};
```

Ecco un esempio di come consultare la città dell’indirizzo di lavoro di `Alice`:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};

console.log(person.addresses[1].city); // "Workville"
```

:::

In questo esempio, `person.addresses` si riferisce all’array di indirizzi. Per consultare il secondo indirizzo in quell’array, usiamo la notazione a parentesi con l’indice `1`. Poi, usiamo la notazione a punto per consultare la `city` da quell’oggetto indirizzo.

Capire come consultare le proprietà in oggetti annidati e array è essenziale quando si lavora con strutture dati complesse. In futuri workshop e laboratori, avrai l’opportunità di esercitarti con questi tipi di strutture dati.
