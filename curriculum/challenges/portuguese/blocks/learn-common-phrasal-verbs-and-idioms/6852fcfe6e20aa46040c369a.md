# --explanation--

`Perhaps` significa "talvez" ou "possivelmente". É usado quando alguém não tem certeza completa mas quer sugerir algo. Por exemplo:

`Perhaps we can try a new tool for project tracking.` – Isso significa que o falante está sugerindo a ideia como uma possibilidade.

O prefixo `un-` é adicionado a adjetivos para criar o significado oposto. Em `unnecessary`, a palavra base é `necessary` (que significa necessário) e `un-` faz com que signifique "não necessário". Outro exemplo:

`It was an unfair decision.` – `Unfair` é o oposto de `fair`, significando não justo ou não igual.
