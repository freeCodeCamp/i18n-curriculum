# --interactive--

المحدِّد الشامل (`*`) هو نوع خاص من محدِّدات CSS يطابق أي عنصر في المستند.

يُستخدم غالبًا لتطبيق نمط على جميع العناصر في الصفحة، وهو مفيد لإعادة تعيين أو تطبيع الأنماط عبر متصفحات مختلفة.

يمكن استخدام المحدِّد الشامل لاختيار جميع العناصر ضمن سياق معين أو على مستوى المستند بأكمله.

إليك مثالًا على استخدام المحدِّد الشامل لضبط `margin` و`padding` لكامل مستند HTML:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1>Heading element</h1>
<p>example paragraph element</p>
```

```css
* {
  margin: 0;
  padding: 0;
}
```

:::

في هذا المثال من الشفرة، يقوم المحدِّد `*` بإعادة تعيين الهامش والحشو لجميع العناصر إلى صفر، وهي تقنية شائعة في إعادة تعيين CSS.

للمحدِّد الشامل أقل قيمة خصوصية بين جميع المحدِّدات. يساهم بقيمة 0 في جميع أجزاء قيمة الخصوصية `(0, 0, 0, 0)`.

هذا يعني أن أي محدِّد آخر، بما في ذلك محدِّدات النوع، ومحدِّدات الصنف، ومحدِّدات المعرف، والأنماط المضمَّنة، ستتجاوز الأنماط التي يحددها المحدِّد الشامل.

لنلق نظرة على المثال التالي في HTML وCSS:

:::interactive_editor

```html
<head>
  <style>
    * {
      color: blue;
    }
    p {
      color: red;
    }
    .highlight {
      color: green;
    }
    #unique {
      color: purple;
    }
  </style>
</head>
<body>
  <p id="unique" class="highlight">This text has multiple styles applied.</p>
</body>
```

:::

يقوم المحدِّد الشامل `*` بتعيين لون النص إلى `blue` لجميع العناصر.

لكن محدِّد النوع `p` يتجاوز ذلك بتعيين لون النص إلى `red` خصيصًا لعناصر `p`.

إذا كان العنصر يحمل الصنف `highlight`، فإن محدِّد الصنف يأخذ الأسبقية، مغيرًا لون النص إلى `green`.

وأخيرًا، محدِّد المعرف `#unique`، الذي له أعلى خصوصية، سيتجاوز الجميع، ويضبط لون النص إلى `purple`.
