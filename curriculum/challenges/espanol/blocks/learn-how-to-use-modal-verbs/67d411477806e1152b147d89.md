Estas dos palabras expresan necesidad y siempre van seguidas de un verbo en forma base.

# --explanation--

`Have to` se usa para expresar obligación y siempre va seguido de la forma base del verbo (`set up`). Por ejemplo:

- Tienes que actualizar el software para usar las nuevas características. - Es necesario actualizar el software para acceder a las nuevas características.

- Tienen que preparar la sala de reuniones para la presentación.
