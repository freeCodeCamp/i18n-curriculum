# --description--

このゲームは、2つの石の山と2人のプレイヤーで行われます。各プレイヤーのターンでは、大きい方の山から石を取り除くことができます。取り除く石の数は、小さい方の山の石の数の正の倍数でなければなりません。

例えば、(6,14) という順序対が、小さい方の山に6個、大きい方の山に14個の石がある状態を表すとすると、最初のプレイヤーは大きい方の山から6個または12個の石を取り除くことができます。

ある山からすべての石を取ったプレイヤーがゲームに勝ちます。

勝ちの状態とは、最初のプレイヤーが必ず勝てる状態のことです。例えば、(1,5)、(2,6)、(3,12) は勝ちの状態で、最初のプレイヤーはすぐに2番目の山の石をすべて取り除くことができます。

負けの状態とは、最初のプレイヤーがどんな手を打っても2番目のプレイヤーが必ず勝てる状態のことです。例えば、(2,3) と (3,4) は負けの状態で、どんな合法的な手を打っても2番目のプレイヤーにとって勝ちの状態を残してしまいます。

$S(N)$ を、負けの状態であるすべての組み合わせ ($x_i$, $y_i$) で $0 < x_i < y_i \leq N$ のときの $(x_i + y_i)$ の合計と定義します。$S(10) = 211$ および $S({10}^4) = 230\\,312\\,207\\,313$ であることが確認できます。

$S({10}^{16}) \bmod 7^{10}$ を求めてください。
