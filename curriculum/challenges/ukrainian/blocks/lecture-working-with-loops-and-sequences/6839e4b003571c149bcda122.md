# --description--

У попередніх уроках ви навчились працювати з циклом `for`, який використовується для повторення блоку коду певну кількість разів. Ось приклад використання циклу `for` для виведення кожної мови зі списку `languages` у консоль:

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for language in languages:
    print(language)
```

Але що, якщо ви хочете відслідковувати індекс для кожного елемента? Один із варіантів — створити змінну `index` і збільшувати її на `1` для кожної ітерації циклу, ось так:

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

index = 0

for language in languages:
    print(f'Index {index} and language {language}')
    index += 1
```

Хоча це працює, простіший спосіб зробити це — використати функцію `enumerate()`. Функція `enumerate()` відслідковує індекс для ітерованого об’єкта і повертає об’єкт enumerate.

Якщо передати список `languages` у функцію `enumerate()` і перетворити її повернене значення у список за допомогою функції `list()`, це виглядає так:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

list(enumerate(languages))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

Кожен запис в об’єкті enumerate (тепер у вигляді списку) — це кортеж, що містить лічильник, а потім значення з ітерованого об’єкта, переданого у функцію `enumerate()`.

Тепер давайте переробимо приклад з початку, щоб використати функцію `enumerate()`:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')
```

Ми розпаковуємо лічильник і значення для кожного кортежу в об’єкті enumerate у змінні з іменами `index` і `language` відповідно. Нарешті, обидві ці змінні використовуються у f-рядку, який виводиться у консоль під час кожної ітерації циклу.

```md
Index 0 and language Spanish
Index 1 and language English
Index 2 and language Russian
Index 3 and language Chinese
```

Це усуває потребу вручну створювати і оновлювати змінну `index`.

Функція `enumerate()` також приймає необов’язковий аргумент `start`, який задає початкове значення для лічильника. Якщо цей аргумент пропущений, лічильник починається з `0`. Ось приклад використання необов’язкового аргументу `start`:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages, 1):
    print(f'Index {index} and language {language}')
```

А ось як виглядає результат у консолі:

```md
Index 1 and language Spanish
Index 2 and language English
Index 3 and language Russian
Index 4 and language Chinese
```

Поки що ми ітерували лише один список. Але що, якщо потрібно ітерувати над кількома ітерованими об’єктами паралельно? Для цього можна використати функцію `zip()`, яка об’єднує списки у пари елементів і повертає ітератор кортежів.

Якщо передати список `developers` і `ids` у функцію `zip()` і перетворити її повернене значення у список за допомогою функції `list()`, це виглядає так:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

list(zip(developers, ids))
# [('Naomi', 1), ('Dario', 2), ('Jessica', 3), ('Tom', 4)]
```

А ось приклад використання функції `zip()` з циклом `for` для ітерації над `developers` і `ids`:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')
```

У цьому прикладі `zip()` об’єднує два списки у пари елементів і повертає ітератор кортежів. Цикл `for` розпаковує кожен кортеж у `name` і `id`. Нарешті, для кожної інструкції print ми виводимо відповідно кожен `name` і `id` зі списків `ids` і `developers`. Ось як виглядає результат у консолі:

```md
Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
```

Функції `enumerate()` і `zip()` дуже потужні, і в поєднанні з циклами можуть зробити ваш код значно лаконічнішим.
