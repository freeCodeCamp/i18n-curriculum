# --description--

Im Bereich „Elements“ können Sie die gesamte HTML-Struktur Ihrer Seite sehen. Sie können auf eines der Elemente in diesem Bereich klicken, um genau dieses Element auszuwählen. Alternativ können Sie auf das unten links blau hervorgehobene Symbol klicken und mit der Maus über ein beliebiges Element auf der Seite fahren.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/the-box-model/inspector-icon.png" alt="Eine Webentwicklungsumgebung mit geöffneten Entwicklerwerkzeugen, die ein blaues Inspektor-Symbol oben links hervorhebt, um Webseiten-Elemente auszuwählen und HTML-Struktur sowie Styles anzuzeigen.">

Wenn ein Element ausgewählt ist, zeigt der Reiter „Styles“ alle aktuell angewendeten Styles sowie alle Styles, die überschrieben werden (gekennzeichnet durch einen durchgestrichenen Text). Wenn Sie zum Beispiel mit dem Inspektor auf die Überschrift „Your Career in Web Development Starts Here“ auf der <a href="https://www.theodinproject.com/" target="_blank">Startseite von The Odin Project</a> klicken, sehen Sie auf der rechten Seite alle Styles, die das Element gerade beeinflussen, wie unten dargestellt:

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/the-box-model/overwritten-style.png" alt="CSS-Code-Ausschnitt in der Entwicklerkonsole, der .hero_main-heading mit angewendetem margin-bottom: 100px, padding-bottom: 1.875rem, .accent color #ce973e zeigt. h1 Style-Regeln umfassen Schriftgröße, Gewicht, Buchstabenabstand, wobei margin-bottom: 2rem überschrieben wird.">
