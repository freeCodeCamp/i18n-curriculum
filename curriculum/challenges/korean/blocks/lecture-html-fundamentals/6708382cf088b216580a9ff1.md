# --interactive--

`id` 속성은 HTML 요소에 고유 식별자를 추가합니다.

다음은 `h1`인 `id`를 가진 `title` 요소의 예입니다.

`h1` 요소 아래에 `h2`로 설정된 `id` 속성을 가진 `"subtitle"` 요소를 추가하세요. `h2`에 원하는 텍스트를 작성할 수 있으며, 미리보기 창에서 변경 사항을 확인할 수 있습니다. 예제와 상호작용하려면 인터랙티브 편집기를 활성화하세요.

:::interactive_editor

```html
<h1 id="title">Movie Review Page</h1>

```

:::

JavaScript나 CSS 내에서 `id`의 `title` 이름을 참조할 수 있습니다. 다음은 `id`의 `title`를 참조하여 텍스트 `color`를 `red`로 변경하는 CSS 예제입니다.

**참고**: 이 인터랙티브 예제에는 일부 CSS가 제공되어 있습니다. CSS 코드를 이해하려고 걱정하지 마세요. 앞으로 수업에서 더 배우게 됩니다. 하지만 텍스트 색상을 파란색으로 바꾸고 싶다면 인터랙티브 편집기를 활성화하고 `styles.css` 탭을 클릭한 다음 `color: red;`를 `color: blue;`로 변경하세요.

:::interactive_editor

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>Review page Example</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
    <h1 id="title">Movie Review Page</h1>
  </body>
</html>
```

```css
#title {
  color: red;
}
```

:::

`#` 앞의 해시 기호(`title`)는 컴퓨터에 해당 값을 가진 `id`를 대상으로 한다는 것을 알려줍니다. `id` 이름은 한 번만 사용해야 하며 항상 고유해야 합니다. `id` 값에 대해 주의할 점은 공백을 포함할 수 없다는 것입니다. 다음은 `main`와 `heading`라는 단어를 `id` 속성 값에 적용한 예입니다:

```html
<h1 id="main heading">Main heading</h1>
```

브라우저는 이 공백을 `id`의 일부로 인식하여 스타일링과 스크립팅 시 원치 않는 문제가 발생할 수 있습니다. `id` 속성 값에는 문자, 숫자, 밑줄, 대시만 포함해야 합니다.

`id` 속성과 달리 `class` 속성 값은 고유할 필요 없으며 공백을 포함할 수 있습니다.

다음은 `box`라는 클래스를 `div` 요소에 적용한 예입니다.

```html
<div class="box"></div>
```

요소에 여러 클래스 이름을 추가하려면 이름을 공백으로 구분하면 됩니다. 다음은 여러 클래스를 `div` 요소에 적용한 업데이트된 예입니다.

```html
<div class="box red-box"></div>
```

다음은 여러 `div` 요소에 여러 클래스를 적용한 또 다른 예입니다.

**참고**: 이 인터랙티브 예제에는 일부 CSS가 제공되어 있습니다. CSS 코드를 이해하려고 걱정하지 마세요. 앞으로 수업에서 더 배우게 됩니다. 하지만 첫 번째와 세 번째 상자의 색상을 바꾸고 싶다면 인터랙티브 편집기를 활성화하고 `styles.css` 탭을 클릭한 다음 `background-color: red;`를 `background-color: black;`로 변경하세요.

:::interactive_editor

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>Colored boxes example</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
    <div class="box red-box"></div>
    <div class="box blue-box"></div>
    <div class="box red-box"></div>
    <div class="box blue-box"></div>
  </body>
</html>
```

```css
.box {
  width: 100px;
  height: 100px;
}

.red-box {
  background-color: red;
}

.blue-box {
  background-color: blue;
}
```

:::

요약하자면, 언제 `id`를 사용하고 언제 `class`를 사용해야 할까요? 여러 요소에 스타일을 적용하려면 클래스를 사용하는 것이 가장 좋습니다. 특정 요소를 대상으로 하려면 고유해야 하는 `id`를 사용하는 것이 가장 좋습니다.
