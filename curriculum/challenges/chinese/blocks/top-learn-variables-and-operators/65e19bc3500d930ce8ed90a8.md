# --description--

数字是编程逻辑的创建块！事实上，很难想到有任何有用的编程任务不涉及至少一点基础数学……所以了解数字的工作原理显然非常重要。幸运的是，这也相当简单明了。

| Operator | Description                  | Example            |
|----------|------------------------------|--------------------|
| +        | Addition                     | 5 + 3 = 8          |
| -        | Subtraction                  | 5 - 3 = 2          |
| *        | Multiplication               | 5 * 3 = 15         |
| **       | Exponentiation               | 5 ** 3 = 125       |
| /        | Division                     | 6 / 3 = 2          |
| %        | Modulus                      | 5 % 3 = 2          |
