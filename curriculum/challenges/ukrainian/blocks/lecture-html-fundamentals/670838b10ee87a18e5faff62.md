# --interactive--

Елемент `script` використовується для вставлення виконуваного коду. Більшість розробників використовують його для виконання коду JavaScript. JavaScript застосовується, щоб додати інтерактивність вашим вебсторінкам. Загальні приклади використання JavaScript — інтерактивні ігри, слайдери зображень і динамічні форми, які в реальному часі перевіряють вхідні дані користувача.

Ось приклад використання елемента `script` у HTML-документі. Видаліть `//` перед `alert("Welcome to freeCodeCamp");`, і ви побачите спливаюче повідомлення alert. Щоб побачити попередній перегляд, потрібно увімкнути інтерактивний редактор.

:::interactive_editor

```html
<body>
  <script>
    // alert("Welcome to freeCodeCamp");
  </script>
</body>
```

:::

Хоча технічно можна писати весь JavaScript-код всередині тегів `script`, вважається найкращою практикою направляти на зовнішній файл JavaScript. Ось приклад використання елемента `script` для направлення на зовнішній файл JavaScript:

```html
<script src="path-to-javascript-file.js"></script>
```

Атрибут `src` тут використовується, щоб вказати розташування зовнішнього файлу JavaScript. `src` означає "source" (джерело). Причина, чому не рекомендується розміщувати весь JavaScript всередині HTML-документа, полягає в принципі розділення обов’язків. Розділення обов’язків — це принцип проєктування, за яким програми розбивають на окремі розділи, і кожен розділ відповідає за окрему задачу. У цьому випадку ми хочемо відокремити JavaScript-код від HTML-коду.
