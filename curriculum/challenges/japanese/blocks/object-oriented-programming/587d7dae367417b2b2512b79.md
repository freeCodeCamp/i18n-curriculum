# --description--

前回の`Bird`と`Dog`のコンストラクタはうまく動作しました。しかし、`Birds`コンストラクタで作成されるすべての`Bird`は自動的に名前がAlbertで、色は青、脚は2本になっていることに気づきましたか？名前や色の異なる鳥を作りたい場合はどうでしょう？各鳥のプロパティを手動で変更することもできますが、それは非常に手間がかかります。

```js
let swan = new Bird();
swan.name = "Carlos";
swan.color = "white";
```

もし鳥類園で何百、何千もの異なる鳥を管理するプログラムを書いているとしたら、すべての鳥を作成してからそれぞれのプロパティを異なる値に変更するのは非常に時間がかかります。異なる`Bird`オブジェクトをより簡単に作成するために、Birdコンストラクタをパラメータを受け取るように設計できます。

```js
function Bird(name, color) {
  this.name = name;
  this.color = color;
  this.numLegs = 2;
}
```

そして、`Bird`コンストラクタに引数として値を渡して、それぞれのユニークな鳥を定義します：`let cardinal = new Bird("Bruce", "red");` これにより、`Bird`の新しいインスタンスが作成され、`name`と`color`のプロパティがそれぞれ`Bruce`と`red`に設定されます。`numLegs`プロパティはまだ2に設定されています。`cardinal`はこれらのプロパティを持っています。

```js
cardinal.name
cardinal.color
cardinal.numLegs
```

このコンストラクタはより柔軟になりました。作成時に各`Bird`のプロパティを定義できるようになり、これがJavaScriptのコンストラクタが非常に便利な理由の一つです。コンストラクタは共通の特徴や動作に基づいてオブジェクトをまとめ、その作成を自動化する設計図を定義します。

# --instructions--

別の`Dog`コンストラクタを作成してください。今回はパラメータ`name`と`color`を受け取り、プロパティ`numLegs`は4に固定してください。次に、`Dog`を新しく作成し、変数`terrier`に保存します。`name`と`color`のプロパティに対して2つの文字列を引数として渡してください。

# --hints--

`Dog`は`name`の引数を受け取る必要があります。

```js
assert(new Dog('Clifford').name === 'Clifford');
```

`Dog`は`color`の引数を受け取る必要があります。

```js
assert(new Dog('Clifford', 'yellow').color === 'yellow');
```

`Dog`はプロパティ`numLegs`を4に設定してください。

```js
assert(new Dog('Clifford').numLegs === 4);
```

`terrier`は`Dog`コンストラクタを使って作成してください。

```js
assert(terrier instanceof Dog);
```
