# --description--

Ikiwa umepewa mfuatano wa herufi, rudisha mfuatano mpya uliopunguzwa ili jumla ya upana wa herufi isiizidi vitengo 50.

Kila herufi ina upana maalum:

| Herufi | Upana |
| - | - |
| `"ilI"` | 1 |
| `"fjrt"` | 2 |
| `"abcdeghkmnopqrstuvwxyzJL"` | 3 |
| `"ABCDEFGHKMNOPQRSTUVWXYZ"` | 4 |

Jedwali lililo juu linajumuisha herufi zote kubwa na ndogo. Zaidi ya hayo:

- Nafasi (`" "`) zina upana wa 2
- Nukta (`"."`) zina upana wa 1

- Ikiwa mfuatano uliotolewa ni vitengo 50 au chini, rudisha mfuatano huo kama ulivyo, vinginevyo
- Punguza mfuatano na ongeza nukta tatu mwishoni (`"..."`) ili jumla ya upana wake, ikijumuisha nukta tatu, iwe karibu iwezekanavyo na vitengo 50 bila kuzidi.

```js
function truncateText(str) {
  const MAX_WIDTH = 50;
  const ELLIPSIS = "...";
  const ELLIPSIS_WIDTH = 3;
  const TRUNCATE_LIMIT = MAX_WIDTH - ELLIPSIS_WIDTH;

  const charGroups = {
    'ilI.': 1,
    'fjrt ': 2,
    'abcdeghkmnopqrstuvwxyzJL': 3,
    'ABCDEFGHKMNOPQRSTUVWXYZ': 4
  };

  function getCharWidth(char) {
    for (const key in charGroups) {
      if (key.includes(char)) return charGroups[key];
    }
    return 3;
  }

  function stringWidth(str) {
    let totalWidth = 0;
    for (const char of str) {
      totalWidth += getCharWidth(char);
    }
    return totalWidth;
  }

  if (stringWidth(str) <= MAX_WIDTH) return str;

  let result = "";
  let totalWidth = 0;

  for (const char of str) {
    const charWidth = getCharWidth(char);

    if (totalWidth + charWidth > TRUNCATE_LIMIT) break;

    result += char;
    totalWidth += charWidth;
  }

  return result + ELLIPSIS;
}
```
