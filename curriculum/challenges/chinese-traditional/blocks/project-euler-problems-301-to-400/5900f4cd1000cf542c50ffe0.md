# --description--

一個月亮可以用以中心 (0, 0, 0) 和半徑 $r$ 的球體 $C(r)$ 來描述。

月球表面上位於 $C(r)$ 上座標為整數的點有基地。位於 (0, 0, $r$) 的基地稱為北極基地，位於 (0, 0, $-r$) 的基地稱為南極基地。

所有車站皆透過經過這些車站的大圓弧上最短的道路彼此相連。兩個車站之間的旅程具有風險。如果 $d$ 是兩個車站之間道路的長度，$\{\left(\frac{d}{πr}\right)}^2$ 是該旅程風險的衡量標準（我們稱之為道路風險）。如果旅程包含超過兩個車站，旅程的風險即為所使用道路風險的總和。

從北極站到南極站的直接旅程長度為 $πr$，風險為 1。經由 (0, $r$, 0) 從北極站到南極站的旅程長度相同，但風險較低：

$${\left(\frac{\frac{1}{2}πr}{πr}\right)}^2+{\left(\frac{\frac{1}{2}πr}{πr}\right)}^2 = 0.5$$

從北極站到南極站在 $C(r)$ 上的最小風險是 $M(r)$。

你已知 $M(7) = 0.178\\,494\\,399\\,8$ 四捨五入到小數點後 10 位數。

求 $\displaystyle\sum_{n = 1}^{15} M(2^n - 1)$。

請將你的答案四捨五入到小數點後 10 位，以 a.bcdefghijk 的形式表示。
