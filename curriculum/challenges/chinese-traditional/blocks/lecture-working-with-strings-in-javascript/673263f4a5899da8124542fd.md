# --interactive--

`prompt()` 方法是 JavaScript 與使用者互動的重要零件之一。它是透過一個小型彈出對話盒從使用者取得輸入的最簡單方法之一。

你經常會看到它用於網頁需要從使用者取得一段資訊的情況，例如姓名或其他形式的文字輸入。

那麼，`prompt()` 方法到底做什麼？它會開啟一個對話盒，要求使用者輸入一些資料，然後將使用者輸入的文字以字串形式傳回。

`prompt()` 方法接受兩個引數：第一個是會出現在對話盒內的訊息，通常用來提示使用者輸入資訊。第二個是選用的預設值，會在輸入欄位中預先填入。

```js
prompt(message, default);
```

這是一個它如何運作的範例。

**注意**：此範例包含你尚未學習的程式碼。不要擔心試著理解程式碼中的所有內容。這只是為了說明 `prompt()` 方法的運作方式，並確保提示不會在頁面載入時立即出現，因為這可能會被視為干擾。如果你啟用了互動式編輯器，你可以自己試試看。

:::interactive_editor

```html
<button id="prompt-btn">Show Prompt</button>
<p id="output"></p>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("prompt-btn");
const output = document.getElementById("output");
btn.addEventListener("click", () => {
  const userName = prompt("What is your name?", "Guest");
  output.textContent = "Hello, " + userName + "!";
});
```

:::

在此範例中，當使用者點擊按鈕時，`prompt()` 方法會顯示一個對話盒，內含訊息 `What is your name?` 以及一個初始值為 `Guest` 的輸入欄。

如果使用者輸入他們的名字並按下「確定」，`userName` 變數將會儲存輸入的值。如果使用者按下「取消」，`userName` 變數將會被設定為 `null`。`null` 表示使用者沒有提供任何輸入。輸出段落將會使用提供的名字或在使用者取消時顯示 `null` 來顯示問候訊息。你將在未來的課程中學習避免在使用者取消提示時顯示 `null` 的技巧。

請記住，`prompt()` 方法會暫停腳本的執行，直到使用者與對話盒交談為止。

這表示你的其餘 JavaScript 程式碼不會執行，直到使用者提供輸入並點擊「OK」，或取消提示。

另一個需要考慮的點是，雖然 `prompt()` 對於快速測試或小型應用程式很有用，但由於其干擾性質及在不同瀏覽器中行為不一致，現代複雜的網頁應用程式通常會避免使用它。

透過理解 `prompt()` 方法，你可以透過瀏覽器與使用者互動並直接取得資訊，儘管它在現代網頁應用程式中可能不常被使用。
