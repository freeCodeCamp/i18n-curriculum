# --explanation--

`Not necessarily`는 부분적으로 동의하지 않을 때 정중하게 표현하는 방법입니다. `no`라고 직접 말하는 대신, 다른 고려할 요소가 있을 수 있음을 암시합니다. 대화에서 이 표현은 토론을 열어두고 존중하는 분위기를 유지하는 데 도움이 됩니다. 예를 들어:

- **Sarah:** `I think upgrading the computer's processor will fix the issue.`

- **Finn:** `Not necessarily. The problem could be with the software, not the hardware.` - 이 사람은 프로세서를 업그레이드하는 것이 문제를 해결하지 못할 수도 있다고 제안하는데, 그 이유는 오래된 소프트웨어 같은 다른 원인이 있을 수 있기 때문입니다.
