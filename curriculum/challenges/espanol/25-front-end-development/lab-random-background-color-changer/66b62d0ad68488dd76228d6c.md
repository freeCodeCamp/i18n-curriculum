# --description--

Camperbot está aprendiendo JavaScript y ha intentado construir su propio Cambiador de Color de Fondo Aleatorio. Sin embargo, ha cometido algunos errores en el proceso.

**Objective:** Fulfill the user stories below and get all the tests to pass so the lab is functioning properly.

**Historia de usuario:**

1. Debes corregir los errores de sintaxis en el array `darkColorsArr`.
1. Debes corregir el `ReferenceError` que se lanza cuando se hace referencia al objeto `math` dentro de la función `getRandomIndex`.
1. Debes redondear `randomIndex` al número entero más cercano.
1. Debes actualizar el valor de la variable `body` para usar el nombre de método correcto y corregir el `TypeError` relacionado.
1. Debes corregir `bgHexCodeSpanElement` para que seleccione correctamente el elemento con el `id` de `bg-hex-code`.
1. Debes modificar la variable `color` dentro de `changeBackgroundColor` para que se asigne a un color aleatorio del array `darkColorsArr`.
1. Debes actualizar la variable `btn` para que apunte al elemento botón. Abre el `index.html` para ver el nombre correcto del `id` para ese elemento botón.
1. Debes corregir la línea que añade un event listener para el evento `click` a `btn` pasando una referencia a la función `changeBackgroundColor` en el método.
1. Tu fondo debería cambiar aleatoriamente a uno de los colores en `darkColorsArr` cada vez que se haga clic en el `#btn`.

```js
const darkColorsArr = [
  "#2C3E50",
  "#34495E",
  "#2C2C2C",
  "#616A6B",
  "#4A235A",
  "#2F4F4F",
  "#0E4B5A",
  "#36454F",
  "#2C3E50",
  "#800020",
];

function getRandomIndex() {
  const randomIndex = Math.floor(darkColorsArr.length * Math.random());
  return randomIndex;
}

const body = document.querySelector("body");
const bgHexCodeSpanElement = document.querySelector("#bg-hex-code");

function changeBackgroundColor() {
  const color = darkColorsArr[getRandomIndex()];

  bgHexCodeSpanElement.innerText = color;
  body.style.backgroundColor = color;
}
const btn = document.querySelector("#btn");

btn.addEventListener("click", changeBackgroundColor);
```
