# --description--

在标准的52张扑克牌中，一套4张牌是Badugi，如果它包含4张没有成对的牌而没有两张相同牌的牌。

设 $f(n)$ 为选择 $n$ 张牌且包含一个 4 张牌的子集为 Badugi 的方式数。例如，从标准 52 张牌的牌堆中选择五张牌有 $2\,598\,960$ 种方式，其中有 $514\,800$ 种方式包含一个 4 张牌的子集为 Badugi，因此 $f(5) = 514800$。

查找 $\sum f(n)$，其中 $4 ≤ n ≤ 13$。
