# --description--

Sei $S(n, m) = \sum φ(n × i)$ für $1 ≤ i ≤ m$. ($φ$ ist die Euler-Totient-Funktion)

Es ist gegeben, dass $S(510\\,510, {10}^6) = 45\\,480\\,596\\,821\\,125\\,120$ gilt.

Finden Sie $S(510\\,510, {10}^{11})$. Geben Sie die letzten 9 Ziffern Ihrer Antwort an.
