# --description--

У розмовах, особливо про правила чи процедури на роботі, `every` та `any` використовуються для обговорення різних ситуацій. `Every` стосується всіх елементів у групі, тоді як `any` часто вживається для одного або кількох елементів у питаннях або запереченнях.
