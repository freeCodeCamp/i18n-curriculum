# --description--

ユニークな投資の機会が与えられています。

1ポンドの資本から始めて、資本の一定割合$f$を選び、公正なコイントスに対して1000回繰り返し賭けることができます。

表が出れば賭け金の2倍のリターンがあり、裏が出れば賭け金を失います。

例えば、$f = \frac{1}{4}$の場合、最初のトスで0.25ポンドを賭け、表が出れば0.5ポンドの勝ちとなり合計1.5ポンドになります。次に0.375ポンドを賭けて2回目のトスが裏なら、1.125ポンドになります。

1000回のコイントスの後に少なくとも10億ポンドを持つ確率を最大化するように$f$を選んだ場合、ビリオネアになる確率はどれくらいですか？

すべての計算は正確に行われるものとし（丸めなし）、答えは小数点以下12桁まで、0.abcdefghijklの形式で四捨五入して示してください。
