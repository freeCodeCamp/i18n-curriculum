# --description--

在编程中，`int` 代表 "integer"。它是一种用于保存整数的 数据类型，整数是没有小数部分或小数点的数字。整数可以是正数、负数或零。

在处理整数时，选择 `int` 数据类型是必不可少的，比如计数、数组中的索引，或任何不需要小数的场景。

示例：

计数项：`int itemCount = 5;`

存储年龄：`int age = 30;`
