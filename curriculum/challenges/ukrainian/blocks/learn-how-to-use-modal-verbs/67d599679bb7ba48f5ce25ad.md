# --explanation--

`It's hard to say` означає, що щось важко передбачити або воно невизначене. Наприклад:

- **Jack:** `Do you think the project will be finished on time?`

- **Tom:** `It's hard to say. It depends on how fast we solve the issues.` — Час завершення невизначений через можливі затримки.

Джеймс використовує цю фразу, бо не знає напевне, що станеться, але припускає можливі варіанти розвитку подій.
