# --explanation--

Dies ist ein `First Conditional`-Satz. Er beschreibt eine reale oder mögliche Situation:

Bedingung: `If the application has memory leaks`.

Ergebnis: `It can lead to crashes`.

`Can` drückt eine Möglichkeit aus und zeigt, dass Abstürze wahrscheinlich, aber nicht garantiert sind, wenn Speicherlecks vorhanden sind. Zum Beispiel:

`If you install the update, the app can run faster.` – Das Installieren des Updates wird die Anwendung effizienter machen.
