# --description--

Una de las formas más comunes de agregar un nuevo elemento a un arreglo es usando el método `push()`. El método `push()` añade uno o más elementos al final de un arreglo y devuelve la nueva longitud del arreglo.

Por ejemplo, para agregar un nuevo elemento al arreglo `pet`, puedes usar el siguiente código:

```javascript
const pet = ['cat', 'dog', 'bunny'];
pet.push('parrot');
console.log(pet); // Output: ['cat', 'dog', 'bunny', 'parrot']
```

Para eliminar el último elemento de un arreglo, puedes usar el método `pop()`. El método `pop()` elimina el último elemento de un arreglo y devuelve ese elemento.

Por ejemplo, para eliminar el último elemento del arreglo `pet`, puedes usar el siguiente código:

```javascript
const pet = ['cat', 'dog', 'tiger'];
pet.pop();
console.log(pet); // Output: ['cat', 'dog']
```
