# --description--

숫자를 1씩 증가시키거나 감소시키는 것은 가장 흔한 수치 연산 중 하나입니다.

그래서 이를 위한 특별한 연산자가 있습니다:

- 증가 연산자 `++`는 변수를 1만큼 증가시킵니다:

```js
let counter = 2;

// works the same as counter = counter + 1, but is shorter
counter++;      

console.log(counter); // 3
```

- 감소 연산자 `--`는 변수를 1만큼 감소시킵니다:

```js
let counter = 2;

// works the same as counter = counter - 1, but is shorter
counter--;

console.log(counter); // 1
```

연산자 `++`와 `--`는 변수 앞이나 뒤에 올 수 있습니다.

- 연산자가 변수 뒤에 올 때는 "후위형"입니다: `counter++`.

- "전위형"은 연산자가 변수 앞에 올 때입니다: `++counter`.

이 두 문장은 같은 일을 합니다: `counter`를 `1`만큼 증가시킵니다.

차이가 있을까요? 네, 있지만 `++/--`의 반환값을 사용할 때만 볼 수 있습니다.

명확히 하겠습니다. 아시다시피 모든 연산자는 값을 반환합니다. 증가/감소 연산자도 예외가 아닙니다. 전위형은 새 값을 반환하고 후위형은 증가/감소 전의 이전 값을 반환합니다.

차이를 보기 위해 예제를 보겠습니다:

```js
let counter = 1;
let a = ++counter; // (*)

console.log(a); // 2
```

요약하자면:

- 증가/감소 결과를 사용하지 않는다면 어떤 형태를 써도 차이가 없습니다:

```js
let counter = 0;
counter++;
++counter;

console.log( counter ); // 2, the lines above did the same
```

- 값을 증가시키고 즉시 연산자의 결과를 사용하려면 전위형을 써야 합니다:

```js
let counter = 0;
console.log( ++counter ); // 1
```

- 값을 증가시키되 이전 값을 사용하려면 후위형을 써야 합니다:

```js 
let counter = 0;
console.log( counter++ ); // 0
```
