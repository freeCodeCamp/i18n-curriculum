# --description--

可數名詞是你可以計數的事物。它們可以是一、二、三或更多。你可以使用數字，以及像 `many`、`a few` 或 `several` 這樣的量詞來描述可數名詞。 例如，apple：你可以有 `one apple`、`two apples` 或 `many apples`。

不可數名詞是指你無法一一計數的事物。它們通常不會變化以表示多於一個。你不會直接使用數字，而是使用像 `much`、`a little`、`some` 這樣的量詞來描述不可數名詞。<dfn>例如</dfn>，水：你不能說 `one water` 或 `two waters`。你可以說 `much water`、`some water`。

# --questions--

## --text--

選擇正確使用可數名詞和不可數名詞的句子。

## --answers--

`I need some informations about the program.`

### --feedback--

`Information` 是不可數名詞，不應該變成複數。只要說 `some information`。

---

`I'll buy some water.`

---

`Can you give me an advice on this issue?`

### --feedback--

`Advice` 是不可數名詞。你不會說 `an advice`。你可以說 `some advice`。

---

`We have much apples.`

### --feedback--

`Apple` 是可數名詞，應該用 `many` 來描述，而不是 `much`。
