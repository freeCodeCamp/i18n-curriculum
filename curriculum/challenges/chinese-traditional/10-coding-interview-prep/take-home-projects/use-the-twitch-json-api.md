# --description--

**Objective:** Build an app that is functionally similar to this: <a href="https://codepen.io/freeCodeCamp/full/Myvqmo/" target="_blank" rel="noopener noreferrer nofollow">https://codepen.io/freeCodeCamp/full/Myvqmo/</a>.

Twitch API 是一個 RESTful API，開發者可以通過它爲 Twitch 社區創建有意思的插件。

實現以下的用戶故事並且通過所有的測試用例。 使用你需要的任何庫和第三方API。 使用你自己的個人風格樣式。

**用戶故事：** 可以看到 freeCodeCamp 當前是否正在 Twitch.tv上直播。

**用戶故事：** 點擊狀態條可以直接跳轉到 freeCodeCamp 的 Twitch.tv 頻道。

**用戶故事：** 如果 Twitch 用戶當前正在直播，可以看到播放內容的詳細信息。

**提示：** 這裏有一組定期直播的人的 Twitch.tv 用戶名：`["ESL_SC2", "OgamingSC2", "cretetion", "freecodecamp", "storbeck", "habathcx", "RobotCaleb", "noobs2ninjas"]`

**更新：** 由於 Twitch.tv 更新了 API 使用條件，現在調用 API 需要提供 API 密鑰，別擔心，我們提供了一個解決方案。 可以使用 <a href="https://twitch-proxy.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://twitch-proxy.freecodecamp.rocks/</a> 而不是 Twitch 的 API 提供的 URL，這樣獲取賬戶信息就不需要 API 密鑰了。
