# --explanation--

As perguntas de confirmação são negativas se a frase principal for positiva e positivas se a frase principal for negativa.

A frase `are they?` é uma pergunta-tag adicionada ao final de uma frase negativa para confirmar ou verificar informações.Por exemplo:

`The updates aren't applied automatically, are they?` - Aqui, `are they?` verifica a confirmação de que as atualizações não estão sendo aplicadas automaticamente.A frase principal é negativa, então a pergunta confirmatória é positiva.
