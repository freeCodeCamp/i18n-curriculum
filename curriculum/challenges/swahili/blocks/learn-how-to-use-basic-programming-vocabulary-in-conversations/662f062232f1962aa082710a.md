# --description--

`plugin` ni sehemu inayoongeza kipengele maalum katika programu ya kompyuta iliyopo. Programu inapounga mkono `plugins`, inaruhusu kubinafsisha. 
Katika muktadha wa IDE, `plugins` inaweza kutoa utendaji zaidi, kama zana mpya za kuandika msimbo, kurekebisha makosa, au kuboresha muonekano wa mtumizi.

Kwa mfano, mpangaji msimbo katika IDE mara nyingi ni `plugin`.
