# --explanation--

Ein `server` ist ein Computer oder ein System, das Informationen verwaltet und speichert sowie anderen Computern oder Geräten üblicherweise über ein Netzwerk Dienste, Daten oder Ressourcen bereitstellt. Zum Beispiel:

`The company's website is hosted on secure servers.` – Das bedeutet, dass die Daten der Website auf Computern gespeichert sind, die dafür ausgelegt sind, diese Daten zu verwalten und bereitzustellen.
