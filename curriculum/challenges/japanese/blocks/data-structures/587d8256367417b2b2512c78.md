# --description--

グラフを表すもう一つの方法は、<dfn>隣接行列</dfn>にすることです。<dfn>隣接行列</dfn>は二次元（2D）の配列で、各ネストされた配列は外側の配列と同じ数の要素を持ちます。つまり、数字のマトリックスやグリッドで、数字が辺を表しています。

**注意**：マトリックスの上部と左側の数字はノードのラベルです。マトリックスの中の1は、行と列で表される頂点（ノード）の間に辺が存在することを意味します。0は辺や関係がないことを意味します。

<pre>
    1 2 3
  \------
1 | 0 1 1
2 | 1 0 0
3 | 1 0 0
</pre>

上は非常にシンプルな無向グラフで、3つのノードがあり、最初のノードが2番目と3番目のノードに接続されています。以下は同じものをJavaScriptで実装した例です。

```js
var adjMat = [
  [0, 1, 1],
  [1, 0, 0],
  [1, 0, 0]
];
```

隣接リストとは異なり、マトリックスの各「行」はグラフのノード数と同じ数の要素を持つ必要があります。ここでは3×3のマトリックスなので、グラフには3つのノードがあります。有向グラフも似たような形になります。以下は、最初のノードから2番目のノードに向かう辺があり、2番目のノードから3番目のノードに向かう辺があるグラフです。

```js
var adjMatDirected = [
  [0, 1, 0],
  [0, 0, 1],
  [0, 0, 0]
];
```

グラフの辺には<dfn>重み</dfn>を持たせることもできます。これまでのところ、辺の有無が二値（`0`か`1`）の<dfn>重みなし</dfn>の辺でした。用途に応じて異なる重みを持たせることも可能です。

# --instructions--

5つのノードを持つ無向グラフの隣接行列を作成してください。このマトリックスは多次元配列で表します。5つのノードの間には、1番目と4番目のノード、1番目と3番目のノード、3番目と5番目のノード、4番目と5番目のノードの間に関係があります。すべての辺の重みは1です。
