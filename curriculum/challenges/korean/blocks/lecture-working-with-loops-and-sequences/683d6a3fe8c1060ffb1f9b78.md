# --description--

이전 수업에서 튜플 데이터 타입을 다루는 방법을 배웠습니다. 이번 수업에서는 튜플을 다룰 때 자주 사용하는 몇 가지 메서드에 대해 알아보겠습니다.

먼저 살펴볼 메서드는 `count()`입니다. 이 메서드는 튜플에서 특정 항목이 몇 번 나타나는지 확인할 때 사용합니다. 다음은 `Rust`라는 튜플에서 문자열 `programming_languages`가 몇 번 나타나는지 확인하는 예시입니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

`Rust`가 튜플에 두 번 나타나므로 `count()` 메서드는 숫자 `2`를 반환합니다. 만약 `count()` 함수에 지정한 항목이 튜플에 전혀 없다면 반환값은 `0`가 됩니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

`count()` 함수에 인수를 전달하지 않으면 Python은 `TypeError`를 발생시킵니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count()

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: tuple.count() takes exactly one argument (0 given)
"""
```

다음으로 살펴볼 메서드는 `index()` 메서드입니다. 이 메서드는 튜플에서 특정 항목이 위치한 인덱스를 찾을 때 사용합니다. 다음은 `index()` 메서드를 사용해 문자열 `Java`의 인덱스를 찾는 예시입니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

지정한 항목을 찾을 수 없으면 Python은 `ValueError`를 발생시킵니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: tuple.index(x): x not in tuple
"""
```

`index()` 메서드에 선택적으로 시작 인덱스와 종료 인덱스 인수를 전달할 수도 있습니다. 다음은 선택적 시작 인덱스를 전달하는 예시입니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

이 예시에서는 문자열 `Python`를 어디서부터 검색할지 지정하고 있습니다. `3` 함수의 두 번째 인수로 숫자 `index()`를 전달해 인덱스 `3`부터 검색을 시작하도록 지정했습니다. `Python`가 튜플에 두 번 나타나므로 선택적 시작 인덱스 인수를 사용했기 때문에 `index()` 함수는 인덱스 `5`를 반환하며, 인덱스 `2`가 반환되지 않습니다.

선택적 종료 인덱스도 전달할 수 있습니다. 다음은 시작 인덱스와 종료 인덱스를 지정한 수정된 예시입니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

이제 인덱스 `2`가 결과로 나오는데, 이는 인덱스 `2`부터 검색을 시작해 인덱스 `5` 전까지 검색하기 때문입니다.

튜플과 함께 자주 사용하는 또 다른 함수는 `sorted()` 함수입니다. 이전 수업에서 리스트에 사용하는 `sort()` 메서드를 배웠습니다. `sorted()` 함수는 튜플을 포함한 모든 반복 가능한 객체에 사용할 수 있습니다.

다음은 `sorted()` 함수를 사용해 숫자 리스트를 새로 만드는 예시입니다:

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

`sorted()` 함수는 항상 정렬된 값을 가진 새 리스트를 만듭니다. 이는 리스트 요소를 제자리에서 정렬하고 새 리스트를 반환하지 않는 `sort()` 메서드와 다릅니다.

반복 가능한 객체의 정렬 동작을 사용자 지정해야 할 경우 선택적 `reverse`와 `key` 인수를 사용할 수 있습니다. 다음은 `key` 인수를 사용해 튜플 항목을 길이 기준으로 정렬하는 예시입니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

값을 역순으로 정렬한 새 리스트를 만들고 싶다면 다음과 같이 `reverse` 인수를 사용할 수 있습니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

튜플은 Python에서 자주 사용하는 데이터 타입입니다. 튜플을 다루는 방법과 유용한 메서드 및 함수를 이해하면 더 효율적인 코드를 작성할 수 있습니다.
