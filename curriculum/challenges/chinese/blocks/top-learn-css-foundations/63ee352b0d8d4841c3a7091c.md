# --description--

如果你有两组元素共享它们的一些样式声明，该怎么办？

```css
.read {
  color: white;
  background-color: black;
  /* several unique declarations */
}

.unread {
  color: white;
  background-color: black;
  /* several unique declarations */
}
```

我们的 `.read` 和 `.unread` 选择器都共享 `color: white;` 和 `background-color: black;` 声明，但除此之外各自还有几个独特的声明。为了减少重复，你可以将这两个选择器作为一个以逗号分隔的列表组合在一起：

```css
.read,
.unread {
  color: white;
  background-color: black;
}

.read {
  /* several unique declarations */
}

.unread {
  /* several unique declarations */
}
```

上面两个示例（有分组和无分组）都会得到相同的结果，但第二个示例减少了声明的重复，使得同时编辑两个 `class` 的 `color` 或 `background-color` 更加容易。

## --answers--

.red-box,
.yellow-box {
  width: 25px;
  height: 25px;
}
