# --explanation--

`To maintain` significa manter algo funcionando corretamente ou em boas condições.Por exemplo:

`We maintain the server by checking it regularly.` – Isso significa que cuidamos dele para que continue funcionando bem.

Uma `vulnerability` é uma fraqueza em um sistema que pode ser atacada ou usada para causar danos.Por exemplo:

`The old version of the app had a security vulnerability.` – Isso significa que tinha um ponto fraco que hackers poderiam usar.
