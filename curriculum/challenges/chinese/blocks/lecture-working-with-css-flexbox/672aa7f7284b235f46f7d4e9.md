# --interactive--

CSS flexbox 是一种一维版型模型，允许你在容器内按行和列排列元素。你还可以控制它们的顺序和方向。网页开发者使用它来创建响应式网站和适应不同屏幕尺寸及方向的网页应用。我们称 flexbox 为一维版型模型，因为它专注于一次沿单一轴线排列元素。该轴线可以是水平或垂直方向。 

在开始使用 flexbox 之前，你应该了解两个关键概念：flex 容器 和 flex 项。

弹性容器是具有弹性版型的超文本标记语言元素。你可以在弹性容器内以多种方式排列和对齐元素。要使超文本标记语言元素成为弹性容器，你需要为它的 CSS 样式添加 `display: flex`。

Flex 项是 flex 容器的直接子元素。这些元素可以根据 flex 容器的属性进行排列和对齐。它们还可以缩小或扩展以适应可用空间。

这是一个包含 `main` 容器和三个子 `div` 元素的示例：

```html
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

如果你只在 CSS 样式表中设置这些 `div` 元素的 `width`、`height` 和 `background-color`，每个子元素都会被放置在自己的行上，因为容器默认不是 flex。 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

但是如果你为 `main` 容器添加 `display: flex`，`div` 元素将被重新排列以适应同一行，并且如果有必要它们会缩小（要查看预览，请启用交互式编辑器）：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

默认情况下，`flex container` 将是一个块级元素，因此该容器本身相对于其他元素和容器将位于独立的一行。

既然你已经了解了更多关于弹性容器和弹性项的内容，你还应该了解弹性属性。这些属性决定了弹性项在弹性容器内如何排列、调整大小和分布。其中一些最常用的属性是 `flex-direction`、`justify-content`、`align-items` 和 `flex-wrap`。

很好。现在让我们稍微谈谈 flex 模型。该模型定义了 flex 项如何在 flex 容器内排列。每个 flex 容器都有两个轴：

- 主轴。
- 交叉轴。

这些轴的方向决定了不同属性如何影响弹性项的版型和发行（版本）。默认情况下，弹性容器的主轴是水平的，交叉轴是垂直的。弹性项沿主轴方向排列。交叉轴垂直于主轴。

`flex-direction` 属性设置主轴的方向。`flex-direction` 的默认值是 `row`，它将所有的弹性项放置在同一行，方向为浏览器默认语言的方向（从左到右或从右到左）：

```css
flex-direction: row; /* Default */
```

要反转行中的项，可以使用 `flex-direction: row-reverse`：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: row-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

这将反转 flex 项的顺序。如果你想改为垂直对齐 flex 项，只需在 flex 容器中将 `flex-direction` 设置为 `column`。这将改变主轴的方向：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

现在 `div` 元素将垂直对齐，因为主轴将是垂直的，交叉轴将是水平的。 

你也可以使用 `flex-direction: column-reverse` 垂直反转 flex 项的顺序：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

CSS flexbox 是一种强大的版型模型，提供了一种灵活且高效的方式来排列容器内的元素。通过理解 flex 容器、flex 项以及各种 flex 属性的关键概念，你可以创建适应不同屏幕尺寸和方向的动态响应式网站。
