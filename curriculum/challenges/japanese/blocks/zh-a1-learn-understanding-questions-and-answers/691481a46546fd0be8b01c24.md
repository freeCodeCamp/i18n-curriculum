# --explanation--

`UI` は `user interface` を指し、これはユーザーが操作する製品の視覚的要素を意味します。`设计师 (shè jì shī)` は「デザイナー」を意味します。

合わせて、`UI 设计师 (shè jì shī)` は製品の見た目やレイアウトをデザインし、使いやすく快適にすることにフォーカスした職業を指します。これはテック企業で非常に一般的な用語で、人々は直接 `UI` を中国語に訳さずに使います。例えば：

`我是 (wǒ shì) UI 设计师 (shè jì shī)。` – 私は UI デザイナーです。
