# --explanation--

`Code reuse` refere-se à prática de usar código existente em novas partes de um programa. Por exemplo:

A classe `Login` foi usada em múltiplas aplicações para gerenciar o acesso do usuário, o que é um bom exemplo de reutilização de código. - Em vez de escrever a lógica de login para cada aplicativo, o mesmo código é reutilizado, melhorando a eficiência.

`Extending` significa adicionar novos recursos a algo que já existe. Por exemplo:

A classe "AdminUser" estende a classe básica "User" adicionando um método "approveUser()". - O `AdminUser` mantém tudo do `User` e constrói sobre isso com novas capacidades.
