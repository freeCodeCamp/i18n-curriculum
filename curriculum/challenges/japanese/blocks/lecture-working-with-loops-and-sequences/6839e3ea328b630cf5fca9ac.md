# --description--

前のモジュールで学んだように、ループはコードのブロックを決まった回数繰り返すために使われます。このレッスンでは、Pythonでさまざまな種類のループを扱う方法を学びます。

最初に説明するループは`for`ループです。以下は`for`ループを使ってリストを繰り返し処理し、各アイテムをコンソールに出力する例です。

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)
```

結果は次のようになります。

```md
Rust
Java
Python
C++
```

`print(language)`がループ内でインデントされていることに注目してください。そのインデントがなければ、`IndentationError`が発生します。

```python
"""
Traceback (most recent call last):
  File "<stdin>", line 4, in <module>
IndentationError: expected an indented block after 'for' statement on line 3
"""
```

また、`for`ループを使って文字列のような他のイテラブルを繰り返し処理することもできます。以下は`for`ループを使って文字列`code`を繰り返し処理し、各文字を出力する例です。

```py
for char in 'code':
    print(char)
```

結果は次のようになります。

```md
c
o
d
e
```

JavaScriptと同様に、Pythonでも`for`ループをネストすることができます。以下はネストされた`for`ループの例です。

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)
```

外側のループは`category`リストの各`categories`を繰り返し処理します。各`category`に対して、内側のループは`food`リストの各`foods`を繰り返し処理します。コンソールに出力される結果は次の通りです。

```md
Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
```

Pythonで使えるもう一つのループの種類は`while`ループです。このタイプのループは条件が`False`になるまでコードのブロックを繰り返します。以下は`while`ループを使った推測ゲームの例です。

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')
```

この例では、値が`secret_number`の`3`変数と初期推測値`0`があります。次に`input`関数でユーザーからの入力を受け取り、`int()`関数で入力文字列を整数に変換し、`guess`変数に代入します。ユーザーが`3`を入力して正解すると、`while`ループから抜け出し、メッセージ`You got it!`がコンソールに出力されます。そうでなければ、メッセージ`Wrong! Try again.`がコンソールに出力され、ループが繰り返されて再度推測を促します。

結果の例は次の通りです。

```md
Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
```

JavaScriptと同様に、Pythonは`break`および`continue`文をサポートしています。

`break`文はループの実行を停止するために使われます。以下は`break`文を使った`developer_names`のリストの例です。

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

この例では、`developer_names`のリストを繰り返し処理し、各名前をコンソールに出力します。名前が`Naomi`と等しい場合、ループを抜けます。これにより、コンソールには`Jess`の名前だけが出力されます。

`continue`文はループの現在の繰り返しをスキップして次の繰り返しに進むために使われます。先ほどの例を`continue`の代わりに`break`文を使うように変更してみましょう。

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

これでコンソールの結果は変わります。名前`Jess`と`Tom`が出力されます。これは`continue`文が`developer`が`Naomi`と等しい2回目の繰り返しをスキップし、その名前をコンソールに出力しないためです。

`for`ループと`while`ループの両方は`else`節と組み合わせることができ、`break`文でループが終了しなかった場合にのみ実行されます。以下は複数の`for`ループを使った例です。

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

この例ではランダムな単語のリストがあり、`for`ループで各単語を繰り返し処理します。外側の`for`ループの中に、各単語の各文字を繰り返し処理する`for`ループがあります。文字の小文字が母音であれば、単語と含まれる母音を出力し、内側のループを抜けます。単語に母音が含まれない場合は、その旨のメッセージを出力します。

コンソールの結果は次のようになります。

```md
'sky' has no vowels
'apple' contains the vowel 'a'
'rhythm' has no vowels
'fly' has no vowels
'orange' contains the vowel 'o'
```
