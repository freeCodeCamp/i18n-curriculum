# --description--

Kusimamia utegemezi wako kwa usahihi—na kujua jinsi ya kusakinisha toleo maalum—ni muhimu kwa kudumisha utulivu wa mradi wako. Hii inakusaidia kuepuka hitilafu zisizotarajiwa na inakupa udhibiti zaidi wakati matoleo mapya yanapotolewa.

Ili kusakinisha toleo maalum la kifurushi, tumia muundo huu:

```bash
npm install package@version-number
```

Kwa mfano, ikiwa mradi wako unahitaji toleo la zamani la Express, sema `4.21.2`, unaweza kulisakinisha hivi:

```bash
npm install express@4.21.2
```

Kama hutajumuishi nambari ya toleo, npm itasakinisha toleo la hivi karibuni linalopatikana moja kwa moja.

Kama huna uhakika matoleo gani yapo, unaweza kuyaangalia kwa amri hii:

```bash
npm view package-name versions
```

Kwa mfano:

```bash
npm view express versions
```

Ili kusasisha vifurushi, unaweza kuvisasisha kimoja kimoja au vyote kwa pamoja. Kabla ya kufanya hivyo, ni vyema kwanza kuona ni vipi vilivyochelewa:

```bash
npm outdated
```

Hii itaonyesha utegemezi gani una matoleo mapya yanayopatikana, na unapaswa kuona matokeo yanayofanana na haya:

```bash
user@Kolade ~/Desktop/fCC/script-code/node/npm/install-spec-version  % npm outdated
Package  Current  Wanted  Latest  Location              Depended by
chalk      5.5.0   5.5.0   5.6.2  node_modules/chalk    install-spec-version
cleave     1.0.1   1.0.1   1.0.2  node_modules/cleave   install-spec-version
express   4.21.2  4.21.2   5.2.1  node_modules/express  install-spec-version
```

Mara baada ya kuangalia utegemezi uliochelewa, unaweza kuanza kuvisasisha.

Ili kusasisha utegemezi mmoja, tumia ifuatayo:

```bash
npm update package-name
```

Hii inasasisha kifurushi ndani ya anuwai ya toleo inayoruhusiwa na faili lako la `package.json`. Kwa mfano, kusasisha Express, unaweza kutumia ifuatayo:

```bash
npm update express
```

Kama unataka kuruka anuwai ya toleo na kusakinisha toleo la hivi karibuni la kifurushi, tumia amri ifuatayo:

```bash
npm install package-name@latest
```

Ili kusasisha utegemezi wote unaoangukia ndani ya anuwai za matoleo zinazoruhusiwa, endesha:

```bash
npm update
```

Hata hivyo, kumbuka kwamba `npm update` haisasishi kifurushi hadi toleo kuu jipya kama `package.json` haikiruhusu. Matoleo makuu mara nyingi huwa na mabadiliko yanayovunja, hivyo npm haitayatumia moja kwa moja.

Kama unataka toleo kuu jipya zaidi, lazima uendeshe:

```bash
npm install package-name@latest
```

Hii inalazimisha npm kusakinisha toleo jipya kabisa, bila kujali anuwai ya toleo iliyo katika `package.json`.
