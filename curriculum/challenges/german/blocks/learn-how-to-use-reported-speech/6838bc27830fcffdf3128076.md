# --explanation--

`Exactly` ist ein Adverb, das verwendet wird, um vollständige Zustimmung oder Richtigkeit dessen auszudrücken, was jemand anderes gesagt hat. `Exact` hingegen ist ein Adjektiv, das präzise oder im Detail korrekt bedeutet. Zum Beispiel:

- `You are exactly right about the cause of the problem.` – Das zeigt vollständige Zustimmung. Es drückt aus, wie richtig (Adjektiv) die Person war.

- `We need the exact measurements to build the new server rack.` – Das bedeutet sehr genaue Messungen. Es beschreibt ein Substantiv und nicht ein Verb oder Adjektiv.
