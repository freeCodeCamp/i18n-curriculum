# --explanation--

`Complexity` 是指某事物的複雜或困難程度，特別是當它有許多零件或步驟時。 例如：

`The back-end logic added a lot of complexity to the app.`－這表示後端程式碼使應用程式更難理解或管理。

「`Keeping the details hidden`」意指不顯示所有內部步驟或運作方式，只顯示必要的部分。例如：

`The user interface hides the complexity of the system by keeping the details hidden.`－這表示使用者只會看到他們需要的部分，而不會看到技術背景。
