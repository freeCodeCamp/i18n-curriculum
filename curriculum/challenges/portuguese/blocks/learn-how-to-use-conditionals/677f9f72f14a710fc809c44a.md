# --explanation--

Mark menciona `incompatible drivers` e `outdated software` como causas para uma falha de software.

`Incompatible drivers` são drivers que não funcionam corretamente com o hardware, enquanto `outdated software` significa que o software não foi atualizado para sua versão mais recente. Ambos podem causar falhas. Por exemplo:

- `The graphics card isn't working because of incompatible drivers.` - O driver errado está causando o mau funcionamento da placa de vídeo.

- `Outdated software can lead to security vulnerabilities.` - Programas antigos podem não ser seguros para usar devido à falta de atualizações.
