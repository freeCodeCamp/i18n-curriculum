# --explanation--

`Might` hutumika kuonyesha uwezekano. Inamaanisha kwamba jambo linawezekana lakini halijathibitishwa. Kwa mfano:

- `I might go to the party if I finish my work.` - Inawezekana nitaenda, lakini sijathibitisha.

- `She might call you later.` - Inawezekana yeye atawito, lakini si hakika.
