# --description--

الدوال هي قطع قابلة لإعادة الاستخدام من الشفرة تُنفذ عند استدعائها. تأتي العديد من لغات البرمجة مع دوال مدمجة تسهل عليك البدء. بايثون ليست استثناء، وقد غطينا بالفعل بعض الدوال المدمجة مثل `print()` في الدروس السابقة.

دالة مدمجة مفيدة أخرى هي `input()`، التي تتيح لك طلب مُدخَل من المستخدم:

```python
name = input('What is your name?') # User types "Kolade" and presses Enter  
print('Hello', name) # Output: Hello Kolade
```

من ناحية أخرى، تقوم `int()` بتحويل رقم أو قيمة منطقية أو سلسلة نصية رقمية إلى عدد صحيح:

```python
print(int(3.14)) # 3
print(int('42')) # 42
print(int(True)) # 1
print(int(False)) # 0 
```

يمكنك أيضًا كتابة دوال مخصصة خاصة بك. للقيام بذلك، استخدم الكلمة المفتاحية `def`، تليها الاسم الذي تريد إعطاؤه للدالة، ثم زوج من الأقواس العادية، ونقطتين. بعد ذلك، في سطر جديد، اكتب الشفرة التي يجب أن تنفذها الدالة. تُسمى الشفرة التي تنفذها الدالة أيضًا جسم الدالة.

إليك مثال على دالة مخصصة اسمها `hello` تطبع السلسلة النصية `Hello World` في الطرفية:

```python
def hello():
    print('Hello World')
```

لتشغيل الدالة، عليك استدعاؤها باسمها متبوعًا بزوج من الأقواس العادية:

```python
hello() # Hello World
```

لاحظ المسافة البادئة قبل `print('Hello World')`. كما تتذكر من الدروس السابقة، تعتمد بايثون على المسافات البادئة لتحديد مجموعات التعليمات التي تنتمي معًا. تُسمى هذه المجموعات كتل الشفرة.

إليك دالة بسيطة أخرى تطبع مجموع رقمين في الطرفية:

```python
def calculate_sum(a, b):
    print(a + b)
```

يمكنك أن ترى أن دالتنا، `calculate_sum`، تحتوي على `a` و `b` داخل الأقواس العادية، مفصولتين بفاصلة. تُسمى هذه المعلمات. فكر في المعلمات كمتغيرات عنصر نائب تعمل كـ "فتحات" للقيم التي تمررها إلى الدوال عند استدعائها.

لاستخدام المعلمات، عليك تمرير "المعلمات". المعلمات هي القيم التي تمررها إلى الدالة عند استدعائها.

إليك كيفية استدعاء دالة `calculate_sum` لجمع الرقمين `3` و `1`:

```python
calculate_sum(3, 1) # 4
```

إذا استدعيت الدالة بدون العدد الصحيح من المعلمات، ستحصل على `TypeError`:

```python
calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

تستخدم الدوال أيضًا الكلمة المفتاحية الخاصة `return` للخروج من الدالة وإرجاع قيمة. إذا لم تستخدم `return` صراحة، ستُرجع بايثون `None` افتراضيًا.

إليك مثال:

```python
def calculate_sum(a, b):
    print(a + b)

my_sum = calculate_sum(3, 1) # 4
print(my_sum) # None
```

يمكنك أن ترى أن دالة `calculate_sum` تطبع مجموع `a` و `b`، لكنها لا تُرجع شيئًا صراحة. لذلك عندما نُعيّن نتيجتها إلى `my_sum`، تكون القيمة فعليًا `None`. لإصلاح ذلك، يمكنك استخدام الكلمة المفتاحية `return` لإرسال النتيجة:

```python
def calculate_sum(a, b):
    return a + b

my_sum = calculate_sum(3, 1)
print(my_sum) # 4
```

الآن، تُرجع `calculate_sum` مجموع `a` و `b`، الذي يُخزن في `my_sum`.
