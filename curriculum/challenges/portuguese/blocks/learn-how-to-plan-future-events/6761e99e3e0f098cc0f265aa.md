# --explanation--

`SQL injection attacks` são um tipo de ataque cibernético onde hackers inserem código malicioso em uma consulta de banco de dados para acessar ou danificar dados. Por exemplo:

`The company added security measures to prevent SQL injection attacks on their website.` - Isso significa que eles tomaram medidas para impedir que hackers inserissem códigos maliciosos.
