# --description--

異なる整数を使って、数値 $\frac{1}{2}$ を逆数の二乗の和として表す方法はいくつかあります。

例えば、{2,3,4,5,7,12,15,20,28,35} の数字を使うと次のようになります：

$$\frac{1}{2} = \frac{1}{2^2} + \frac{1}{3^2} + \frac{1}{4^2} + \frac{1}{5^2} + \frac{1}{7^2} + \frac{1}{{12}^2} + \frac{1}{{15}^2} + \frac{1}{{20}^2} + \frac{1}{{28}^2} + \frac{1}{{35}^2}$$

実際には、2から45までの整数だけを使う場合、ちょうど3通りの方法があり、残りの2つは {2,3,4,6,7,9,10,20,28,35,36,45} と {2,3,4,6,7,9,12,15,28,30,35,36,45} です。

2から80までの異なる整数を使って、数値 $\frac{1}{2}$ を逆数の二乗の和として表す方法は何通りありますか？
