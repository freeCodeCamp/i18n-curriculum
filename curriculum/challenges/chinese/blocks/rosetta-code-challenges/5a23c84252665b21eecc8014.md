# --description--

当按特定列或字段对表格中的记录进行排序时，<a href="https://www.freecodecamp.org/news/stability-in-sorting-algorithms-a-treatment-of-equality-fa3140a5a539/" target="_blank" rel="noopener noreferrer nofollow">稳定排序</a>将始终保留具有相同密钥的记录的相对顺序。

例如，在这个国家和城市的表格中，对**第二**列城市进行稳定排序，会保持美国伯明翰在英国伯明翰之上。（尽管不稳定排序*可能*在这种情况下将美国伯明翰排在英国伯明翰之上，稳定排序例程会*保证*这一点）。

<pre>UK  London
US  New York
US  Birmingham
UK  Birmingham
</pre>

类似地，仅对第一列进行稳定排序将生成“UK London”作为第一项，“US Birmingham”作为最后一项（因为具有相同第一个单词的元素的顺序——“UK”或“US”——将保持）。

```js
function stableSort(arr) {
  arr.sort(function(a, b) {
    return a[1] < b[1] ? -1 : a[1] > b[1] ? 1 : 0;
  });
  return arr;
}
```
