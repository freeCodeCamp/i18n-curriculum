# --description--

Un `security breach` è quando qualcuno ottiene un accesso non autorizzato a un sistema o a dei dati. Questo può essere un grosso problema nei luoghi di lavoro, soprattutto quando riguarda informazioni private. Per esempio:

`A security breach occurred when an outsider accessed confidential files.` - Questo significa che c'è stato un problema di sicurezza perché qualcuno dall'esterno ha consultato file privati.

# --questions--

## --text--

Cos'è un `security breach`?

## --answers--

Un nuovo sistema di sicurezza.

### --feedback--

Un `breach` riguarda una violazione della sicurezza, non la sua configurazione.

---

### --feedback--

Un `breach` è un incidente, non una riunione.

---

Accesso non autorizzato a sistemi o dati.

---

Una ricompensa per una buona sicurezza.

### --feedback--

Un `breach` è un problema, non una ricompensa.
