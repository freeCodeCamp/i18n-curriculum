# --interactive--

JavaScript에서 문자열을 다룰 때, 더 큰 문자열 안에서 특정 부분 문자열의 위치를 찾아야 할 때가 있습니다.

부분 문자열은 더 큰 문자열 안에 나타나는 문자 수열입니다. 예를 들어, 문자열 `hello world`에서 `hello`과 `world`는 부분 문자열입니다.

문자열 안에서 부분 문자열의 위치를 찾으려면 `indexOf()` 메서드를 사용할 수 있습니다.
JavaScript의 `indexOf()` 메서드는 문자열 안에서 부분 문자열을 검색할 수 있게 해줍니다.

부분 문자열이 발견되면 `indexOf()`는 그 부분 문자열이 처음 나타나는 인덱스(또는 위치)를 반환합니다. 부분 문자열이 발견되지 않으면 `indexOf()`는 `-1`를 반환하는데, 이는 검색이 실패했음을 나타냅니다.

`indexOf()` 메서드는 두 개의 인수를 받습니다: 첫 번째는 더 큰 문자열 안에서 찾고자 하는 부분 문자열이고, 두 번째는 검색을 시작할 선택적 위치입니다. 시작 위치를 제공하지 않으면 검색은 문자열의 처음부터 시작합니다.

여기서 **인수**란 함수를 호출할 때 함수가 특정 정보를 사용해 작업을 수행할 수 있도록 함수나 메서드에 전달하는 값입니다. 인수에 대해서는 앞으로 수업에서 더 자세히 배우게 됩니다.

다음은 `indexOf()` 메서드를 사용해 문자열 `awesome`의 위치를 찾는 예입니다:

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("awesome!");
console.log(position); // 14
```

:::

이 예에서, 단어 `awesome`는 문자열 `14`에서 인덱스 `JavaScript is awesome!`부터 시작하므로 `indexOf()` 메서드는 `14`를 반환합니다.

이제 부분 문자열이 발견되지 않을 때 어떻게 되는지 살펴보겠습니다:

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("fantastic");
console.log(position); // -1
```

:::

단어 `fantastic`가 문자열에 나타나지 않으므로 메서드는 `-1`를 반환합니다.

또한 `indexOf()`에 두 번째 인수를 제공해 문자열 내에서 검색을 시작할 위치를 지정할 수 있습니다. 예를 들어:

:::interactive_editor

```js
let sentence = "JavaScript is awesome, and JavaScript is powerful!";
let position = sentence.indexOf("JavaScript", 10);
console.log(position); // 27
```

:::

이 경우 `JavaScript`에 대한 검색은 10번째 문자 이후부터 시작하므로 `JavaScript`의 두 번째 발생이 인덱스 `27`에서 발견됩니다.

`indexOf()` 메서드는 대소문자를 구분한다는 점을 기억하세요.

이 예에서는 대문자 `-1`가 문자열 `F`에 없기 때문에 다음은 `freeCodeCamp`를 반환합니다.

:::interactive_editor

```js
console.log("freeCodeCamp".indexOf("F")) // -1
```

:::

`indexOf()`를 사용하면 부분 문자열이 문자열에 존재하는지 확인하고, 이후 작업을 위해 그 위치를 알아내는 데 매우 유용합니다.
