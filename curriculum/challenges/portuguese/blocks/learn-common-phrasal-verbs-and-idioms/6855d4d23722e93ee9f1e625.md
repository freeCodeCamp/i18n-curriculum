# --explanation--

`Security questions` são perguntas usadas para verificar a identidade de um usuário, geralmente ao fazer login ou recuperar uma conta.Por exemplo:

`One of the security questions asked for my mother's maiden name.` – Isso significa que o usuário é perguntado pelo nome da mãe antes de ela se casar como uma forma de verificar a identidade do usuário.

`Password recovery` é o processo de recuperar o acesso a uma conta quando a senha é esquecida ou perdida.Por exemplo:

`The app sent me a reset link for password recovery.` – Isso significa que o usuário recebeu ajuda para redefinir sua senha esquecida.
