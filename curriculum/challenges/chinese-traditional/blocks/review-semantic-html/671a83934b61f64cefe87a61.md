## 語義化 HTML 的重要性

- **標題元素的結構層次**：使用正確的標題元素來保持內容的結構層次非常重要。 `h1` 元素是最高級別的標題，`h6` 元素是最低級別的標題。
- **表示性 HTML 元素**：定義內容外觀的元素。 例如：已廢棄的 `center`、`big` 和 `font` 元素。
- **語義化 HTML 元素**：具有意義和結構的元素。 例如： `header`、`nav`、`figure`。

## 語義化 HTML 元素

- **header 元素**：用於定義文檔或某個部分的頭部內容。
- **main 元素**：用於包含網頁的主要內容。
- **section 元素**：用於將內容劃分爲較小的部分。
- **導航（`nav`）元素**：代表帶有導航鏈接的部分。
- **figure 元素**：用於包含插圖和圖表。
- **強調（`em`）元素**：用於標記需要強調的文本。

```html
<p>
  Never give up on <em>your</em> dreams.
</p>
```

- **慣用文本元素（`i`）**：用於突出顯示另一種語調或語境下的詞語，比如外語術語、技術術語和思想。

```html
<p>
  There is a certain <i lang="fr">je ne sais quoi</i> in the air.
</p>
```

`i` 開始標籤中的 `lang` 屬性用於指定內容的語言。 本例中，語言就是法語。 `i` 元素並不表示文本是否重要，它只是表示文本與周圍的文本有某種不同。

- **強烈強調（`strong`）元素**：用於標記非常重要的文本。

```html
<p>
  <strong>Warning:</strong> This product may cause allergic reactions.
</p>
```

- **引起注意（`b`）元素**：用於突出顯示對內容含義不重要但需要引起注意的文本。 它常用於總結中的關鍵詞或評論中的產品名。

```html
<p>
  We tested several products, including the <b>SuperSound 3000</b> for audio quality, the <b>QuickCharge Pro</b> for fast charging, and the <b>Ecoclean Vacuum</b> for cleaning. The first two performed well, but the <b>Ecoclean Vacuum</b> did not meet expectations.
</p>
```

- **描述列表（`dl`）元素**：用於表示術語-描述的配對列表。
- **描述術語（`dt`）元素**：表示要定義的術語。
- **描述細節（`dd`）元素**：表示該術語的解釋。

```html
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
```

- **塊級引用（`blockquote`）元素**：用於表示引用自其他來源的較長內容。 該元素有一個 `cite` 屬性。 `cite` 屬性的值是源文件的 URL。

```html
<blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
  "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
</blockquote>
```

- **引用（`cite`）元素**：用於可視化地標記引用來源。 通常用於標記作品的標題。

```html
<div>
  <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
    "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
  </blockquote>
  <p>
    -Quincy Larson, <cite>How to learn to Code and Get a Developer Job [Full Book].</cite>
  </p>
</div>
```

- **內聯引用（`q`）元素**：用於表示短的內聯引用。

```html
<p>
  As Quincy Larson said,
  <q cite="https://www.freecodecamp.org/news/learn-to-code-book/">
    Momentum is everything.
  </q>
</p>
```

- **縮寫（`abbr`）元素**：用於表示縮寫或首字母縮略詞。 爲幫助用戶理解縮寫或首字母縮略詞的含義，可以使用 `title` 屬性顯示其完整形式，即人類可讀的描述。

```html
<p>
  <abbr title="HyperText Markup Language">HTML</abbr> is the foundation of the web.
</p>
```

- **聯繫地址（`address`）元素**：用於表示聯繫信息。
- **（日期）時間（`time`）元素**：用於表示日期或時間。 `datetime` 屬性用於將日期和時間轉換爲機器可讀格式。

```html
<p>
  The reservations are for the <time datetime="20:00">20:00 </time>
</p>
```

- **ISO 8601 日期（`datetime`）屬性**：用於以機器可讀格式表示日期和時間。 標準格式是 `YYYY-MM-DDThh:mm:ss`，其中大寫 `T` 用於分隔日期和時間。
- **上標（`sup`）元素**：用於表示上標文本。 `sup` 元素的常見用例包括指數、上位字母和序數詞。

```html
<p>
  2<sup>2</sup> (2 squared) is 4.
</p>
```

- **下標（`sub`）元素**：用於表示下標文本。 下標元素的常見用例包括化學式、腳註和變量下標。

```html
<p>
  CO<sub>2</sub>
</p>
```

- **內聯代碼（`code`）元素**：用於表示一段代碼。
- **預格式化文本（`pre`）元素**：用於表示預先格式化的文本。

```html
<pre>
  <code>
    body {
      color: red;
    }
  </code>
</pre>
```

- **非文本註釋（`u`）元素**：用於表示一段內聯文本，該文本以表明其具有非文本註釋的方式呈現。

```html
<p>
  You can use the unarticulated annotation element to highlight
  <u>inccccort</u> <u>spling</u> <u>issses</u>.
</p>
```

- **Ruby 註釋（`ruby`）元素**：用於註釋帶有發音或含義解釋的文本。 一個示例用法是東亞文字注音排版。
- **Ruby 回退括號（`rp`）元素**：用作不支持顯示 ruby 註釋的瀏覽器的回退顯示。
- **Ruby 文本（`rt`）元素**：用於表示 ruby 註釋的文本。 通常用於東亞排版中的發音或翻譯細節。

```html
<ruby>
  明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp>
</ruby>
```

- **刪除線（`s`）元素**：用於表示不再準確或相關的內容。

```html
<p>
  <s>Tomorrow's hike will be meeting at noon.</s>
</p>
<p>
  Due to unforeseen weather conditions, the hike has been canceled.
</p>
```
