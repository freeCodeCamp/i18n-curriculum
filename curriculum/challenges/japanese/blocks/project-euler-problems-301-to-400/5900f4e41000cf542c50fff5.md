# --description--

$S_n$ を次の疑似乱数生成器で作られる整数のシーケンスとします：

$$\begin{align}
        S_0 & = 290\\,797 \\\\
  S_{n + 1} & = {S_n}^2\bmod 50\\,515\\,093
\end{align}$$

$A(i, j)$ を $i ≤ j$ のときの数 $S_i, S_{i + 1}, \ldots, S_j$ の最小値とします。$M(N) = \sum A(i, j)$ で $1 ≤ i ≤ j ≤ N$ とします。

$M(10) = 432\\,256\\,955$ および $M(10\\,000) = 3\\,264\\,567\\,774\\,119$ であることが確認できます。

$M(2\\,000\\,000\\,000)$ を求めてください。
