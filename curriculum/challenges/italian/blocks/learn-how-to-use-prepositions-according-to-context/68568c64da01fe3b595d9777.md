# --explanation--

`By tomorrow` indica che qualcosa accadrà prima o al più tardi il giorno seguente. Imposta una scadenza chiara. Per esempio:

`Please finish the report by tomorrow.` – Questo significa che il rapporto dovrebbe essere pronto prima della fine del giorno seguente.
