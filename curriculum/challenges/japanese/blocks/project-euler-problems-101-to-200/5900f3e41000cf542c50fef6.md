# --description--

数値512は、その桁の数字をあるべき乗にしたものの和に等しいため興味深いです：$5 + 1 + 2 = 8$、そして $8^3 = 512$。この性質を持つ別の例は $614656 = 28^4$ です。

ここで、$a_n$ をこのシーケンスの第$n$項と定義し、数値は少なくとも2桁を含む必要があるとします。

$a_2 = 512$ および $a_{10} = 614656$ が与えられています。

$a_{30}$ を求めてください。
