### --feedback--

Queste due parole insieme indicano una situazione in cui un programma o un'applicazione smette di funzionare improvvisamente. La prima parola si riferisce ai programmi per computer, la seconda descrive un guasto improvviso.

# --explanation--

Un `software crash` si verifica quando un programma smette di funzionare all'improvviso, spesso a causa di bug, errori o incompatibilità. Per esempio:

- `A bug in the code led to the software crash.` - Un errore nel programma ha causato l'arresto del funzionamento.

- `We need to fix the issue that caused the software crash.` - Dobbiamo risolvere il problema che ha fatto smettere di funzionare il programma.
