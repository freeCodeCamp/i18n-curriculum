# --description--

Os comentários em HTML não são visíveis para o navegador. Eles nos permitem comentar o código para que outros desenvolvedores ou você mesmo no futuro possam lê-los e ter algum contexto sobre algo que pode não estar claro no código.

Escrever um comentário HTML é simples: basta envolver o comentário com as tags `<!--` e `-->`. Por exemplo:

```html
<h1> View the html to see the hidden comments </h1>

<!-- I am a html comment -->

<p>Some paragraph text</p>

<!-- I am another html comment -->
```

# --assignment--

Para praticar o trabalho com texto em HTML, crie uma página simples de artigo de blog que use diferentes títulos, use parágrafos e tenha algum texto nos parágrafos em negrito e itálico. Você pode usar [Lorem Ipsum](https://loremipsum.io) para gerar texto fictício, no lugar de texto real enquanto constrói seus sites.
