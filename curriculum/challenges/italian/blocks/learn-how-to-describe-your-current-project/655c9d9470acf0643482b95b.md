# --description--

Ricorda, `their` è un pronome possessivo. Indica che qualcosa appartiene a un gruppo di persone. Per esempio:

`Their opinions are important.` - Questo significa che le opinioni di un gruppo di persone sono importanti.

`Thoughts` indica ciò che qualcuno pensa o le sue idee. Quando chiedi il `thoughts` di qualcuno, vuoi sapere cosa pensa o prova riguardo a qualcosa. Per esempio:

`What are their thoughts on this book?` - Questa domanda chiede l’opinione del gruppo sul libro.
