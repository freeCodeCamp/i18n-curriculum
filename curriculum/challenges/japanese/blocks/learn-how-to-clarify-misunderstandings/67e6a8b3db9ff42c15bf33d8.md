# --explanation--

`Not just ... but` は対比を示したり強調を加えたりする一般的な構造です。何かが一つのこと以上であることを意味します。2つ目の部分はさらに情報や重要性を加えます。例えば：

`Debugging is not just about finding errors, but about understanding the code better.` - これはデバッグが単にミスを直すだけでなく、より深い理解にも役立つことを意味します。
