# --explanation--

`Downtime` — це період, коли система або сервіс недоступні або не працюють. Зазвичай він планується для проведення обслуговування, оновлень або ремонту.

`Server downtime` конкретно позначає час, коли сервер — комп’ютерна система, що надає дані, послуги або ресурси іншим пристроям — тимчасово недоступний. Наприклад:

- `The server downtime was scheduled at midnight to minimize the impact on users.` — це означає, що сервер зробили недоступним опівночі, щоб зменшити незручності для користувачів.

- `Unexpected downtime occurred due to a power failure.` — це означає, що система стала недоступною без попереднього планування через відключення електроенергії.
