# --description--

In einer vorherigen Lektion haben Sie gelernt, wie jedes Zeichen in einem String durch seinen Index (beginnend bei null) identifiziert und mit der Klammernotation aufgerufen werden kann:

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
print(my_str[-1]) # d
```

**String-Slicing** ermöglicht es Ihnen, einen Teil eines Strings zu extrahieren oder nur mit einem bestimmten Abschnitt davon zu arbeiten. Hier ist die grundlegende Syntax:

```python
string[start:stop]
```

Wenn Sie Zeichen von einem bestimmten Index bis zu einem anderen extrahieren möchten, trennen Sie die `start` und `stop` Indizes einfach mit einem Doppelpunkt:

```python
my_str = 'Hello world'
print(my_str[1:4]) # ell
```

Beachten Sie, dass der `stop` Index nicht eingeschlossen ist, sodass `[1:4]` gerade die Zeichen vom Index `1` bis, aber nicht einschließlich, des Zeichens am Index `4` extrahiert hat.

Sie können auch die `start` und `stop` Indizes weglassen, und Python verwendet standardmäßig `0` bzw. das Ende des Strings. Zum Beispiel passiert Folgendes, wenn Sie den `start` Index weglassen:

```python
my_str = 'Hello world'
print(my_str[:7])  # Hello w
```

Dies extrahiert alles vom Index `0` bis (aber nicht einschließlich) des Zeichens am Index `7`. Und so sieht es aus, wenn Sie den `stop` Index weglassen:

```python
my_str = 'Hello world'
print(my_str[8:])  # rld
```

Dies extrahiert alles vom Zeichen am Index `8` bis zum Ende des Strings.

Beachten Sie, dass das Slicen eines Strings den ursprünglichen String nicht verändert:

```py
my_str = 'Hello world'
print(my_str[8:])  # rld
print(my_str)  # Hello world
```

Sie können auch beide Indizes `start` und `stop` weglassen, was den gesamten String extrahiert:

```python
my_str = 'Hello world'
print(my_str[:])  # Hello world
```

Neben den Indizes `start` und `stop` gibt es auch einen optionalen `step` Parameter, der verwendet wird, um die Schrittweite zwischen jedem Index im Slice anzugeben.

Hier ist die Syntax dafür:

```python
string[start:stop:step]
```

Im folgenden Beispiel beginnt das Slicing bei Index `0`, stoppt vor `11` und extrahiert jedes zweite Zeichen:

```python
my_str = 'Hello world'
print(my_str[0:11:2])  # Hlowrd
```

Ein hilfreicher Trick, den Sie mit dem `step` Parameter machen können, ist, einen String umzukehren, indem Sie den Schritt auf `-1` setzen und `start` sowie `stop` leer lassen:

```python
my_str = 'Hello world'
print(my_str[::-1]) # dlrow olleH
```
