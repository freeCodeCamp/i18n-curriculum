# --explanation--

`data breach` відбувається, коли конфіденційна інформація потрапляє до неавторизованих осіб. Наприклад:

- `A data breach can happen if hackers get access to private information.` — якщо кіберзлочинці викрадають інформацію, це називається витоком даних.

- `Companies must act quickly if there is a data breach.` — коли трапляється витік даних, бізнес має швидко реагувати, щоб усунути проблему.

Джессіка каже, що шифрування даних і надійне зберігання ключів шифрування є необхідними для запобігання цьому ризику.
