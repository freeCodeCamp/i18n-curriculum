No desenvolvimento de software, um `branch` é como uma linha separada de desenvolvimento. Pense nisso como uma forma de trabalhar em diferentes versões de um projeto ao mesmo tempo. 

Por exemplo, um `branch` pode ser para novas funcionalidades enquanto outro é para corrigir bugs.
