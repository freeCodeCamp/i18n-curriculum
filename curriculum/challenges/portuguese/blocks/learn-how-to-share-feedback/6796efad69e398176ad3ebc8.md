# --explanation--

O `First Conditional` é usado para falar sobre situações reais ou possíveis no futuro e seus resultados prováveis.Segue a estrutura `if` + tempo `Present Simple` para a condição e `will` ou `can` + verbo base para o resultado.Por exemplo:

- `If you study hard, you will pass the exam.` - Isso descreve uma situação possível e seu resultado.  

- `If it rains tomorrow, we can cancel the picnic.` - Isso explica o que pode acontecer se uma condição (chuva) ocorrer.
