# --explanation--

`Get to the bottom of` è una frase comune che significa trovare la vera causa di un problema. Suggerisce di agire per capire qualcosa in profondità. Per esempio:  

`We need to get to the bottom of these login errors before the next release.` – Questo significa che il team vuole scoprire cosa sta davvero causando gli errori per poterli correggere.
