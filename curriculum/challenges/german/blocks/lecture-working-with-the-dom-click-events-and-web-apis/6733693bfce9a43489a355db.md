# --interactive--

Das `DOMContentLoaded`-Ereignis wird ausgelöst, wenn alles im HTML-Dokument geladen und geparst wurde. Wenn Sie externe Stylesheets oder Bilder haben, wartet das `DOMContentLoaded`-Ereignis nicht darauf, dass diese geladen sind. Es wartet nur darauf, dass das HTML geladen ist.

Dies unterscheidet sich vom `load`-Ereignis, das darauf wartet, dass wirklich alles geladen ist, einschließlich externer Stylesheets, Bilder und so weiter.

Hier ist die Beispielsyntax für die Verwendung des `DOMContentLoaded`-Ereignisses:

```js
document.addEventListener("DOMContentLoaded", function () {
  console.log("DOM is loaded.");
});
```

Sobald das DOM geladen ist, wird die Funktion ausgeführt und die Nachricht `DOM is loaded.` wird in der Konsole protokolliert.

Schauen wir uns nun ein weiteres Beispiel mit dem `DOMContentLoaded`-Ereignis an. In diesem Beispiel haben wir ein Bild im HTML. Um das Bild zu aktualisieren, können wir eine Funktion erstellen, die die `src`- und `alt`-Attribute für das Bild ändert:

:::interactive_editor

```html
<h1>Image Change on DOM Loaded</h1>
<img
  id="example-img"
  src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg"
  alt="Cat relaxing"
/>

<script src="index.js"></script>
```

```js
function changeImg() {
  const img = document.getElementById("example-img");
  img.src =
    "https://cdn.freecodecamp.org/curriculum/responsive-web-design-principles/FCCStickers-CamperBot200x200.jpg";
  img.alt = "CamperBot sticker";
  console.log("image was just changed");
}

changeImg();
```

:::

Wir können dann prüfen, ob das DOM noch lädt, und einen Ereignis-Listener für das `DOMContentLoaded`-Ereignis hinzufügen. Wenn das `DOMContentLoaded`-Ereignis bereits ausgelöst wurde, können wir die `changeImg`-Funktion direkt aufrufen:

```js
if (document.readyState === "loading") {
  document.addEventListener("DOMContentLoaded", changeImg);
} else {
  console.log("DOMContentLoaded has already fired");
  changeImg();
}
```

Wenn Sie diesen Code ausführen und die Seite aktualisieren, sehen Sie, dass sich das Bild nach dem Laden des DOMs in das neue Bild ändert. Es passiert ziemlich schnell, aber Sie werden ein leichtes Flackern sehen, wenn das Bild wechselt.

Das `DOMContentLoaded`-Ereignis ist nützlich, wenn Sie JavaScript-Code ausführen möchten, sobald das DOM geladen ist, aber bevor alle externen Ressourcen wie Bilder und Stylesheets geladen sind.
