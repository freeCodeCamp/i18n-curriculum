# --interactive--

JavaScriptで文字列を扱うとき、文字列の一部である特定の部分文字列が含まれているかどうかを確認する必要がある場合がよくあります。

たとえば、ユーザーの入力に特定の単語や文字が含まれているかどうかをチェックしてから何らかの処理を行いたい場合があります。その方法の一つが`includes()`メソッドを使うことです。

`includes()`メソッドは、文字列に特定の部分文字列が含まれているかどうかを確認するために使われます。部分文字列が文字列内に見つかると、このメソッドは`true`を返し、そうでなければ`false`を返します。

基本的な構文は次の通りです。

```js
string.includes(searchValue);
```

構文では、`searchValue`が文字列内で探したい部分文字列です。例を見てみましょう。

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("awesome");

console.log(result);  // true
```

:::

この例では、文字列`awesome`の中に単語`JavaScript is awesome!`が見つかるので、`includes()`メソッドは`true`を返します。

`includes()`メソッドは大文字と小文字を区別することに注意してください。つまり、文字の大文字・小文字も含めて完全に一致する必要があります。

たとえば：

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("Awesome");

console.log(result);  // false
```

:::

大文字の`Awesome`を含む`A`は、小文字の`awesome`を含む`a`と一致しないため、結果は`false`になります。

また、2番目のパラメータを指定して文字列の特定のインデックスから部分文字列を探すために`includes()`メソッドを使うこともできます。

:::interactive_editor

```js
let text = "Hello, JavaScript world!";
let result = text.includes("JavaScript", 7);

console.log(result);  // true
```

:::

ここでは、文字列の7番目の位置から部分文字列`JavaScript`の検索が始まるため、それ以前の文字はスキップされます。

`includes()`メソッドは`true`か`false`の結果しか返しません。部分文字列が文字列のどこにあるかや何回出現するかの情報は提供しません。そのような詳細が必要な場合は、`indexOf()`メソッドなど他のメソッドの方が適しています。
