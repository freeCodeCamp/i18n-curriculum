# --description--

Unaweza kuongeza picha kwenye tovuti yako kwa kutumia kipengele cha `img`. Vipengele vya `img` vina lebo ya kufungua bila lebo ya kufunga. Kipengele kisicho na lebo ya kufunga kinajulikana kama <dfn>kipengele tupu</dfn>.

Ongeza kipengele cha `img` chini ya kipengele cha `p`. Hadi sasa, picha haitajitokeza kwenye kivinjari.

# --hints--

Kipengele chako cha `img` kinapaswa kuwa na lebo ya kufungua. Lebo za kufungua zina sintaksia hii: `<elementName>`.

```js
assert(document.querySelector('img'));
```

Kipengele chako cha `img` hakipaswi kuwa na lebo ya kufunga `</img>`.

```js
assert(!code.match(/<\/img\>/));
```

Unapaswa kuwa na kipengele kimoja tu cha `img`. Ondoa ziada zozote.

```js
assert(document.querySelectorAll('img').length === 1);
```

Kipengele chako cha `img` kinapaswa kuwa chini ya kipengele cha `p`. Umeviweka kwa mpangilio usio sahihi.

```js
const collection = [...document.querySelectorAll('p,img')].map(
  (node) => node.nodeName
);
assert(collection.indexOf('P') < collection.indexOf('IMG'));
```
