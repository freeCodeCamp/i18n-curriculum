# --description--

Eine Folge ganzer Zahlen $S = \\{s_i\\}$ heißt eine $n$-Folge, wenn sie $n$ Elemente hat und jedes Element $s_i$ die Bedingung $1 ≤ s_i ≤ n$ erfüllt. Somit gibt es insgesamt $n^n$ verschiedene $n$-Folgen.

Zum Beispiel ist die Folge $S = \\{1, 5, 5, 10, 7, 7, 7, 2, 3, 7\\}$ eine 10-Folge.

Für jede Folge $S$ sei $L(S)$ die Länge der längsten zusammenhängenden Teilfolge von $S$, die denselben Wert hat. Für die oben gegebene Folge $S$ ist zum Beispiel $L(S) = 3$, wegen der drei aufeinanderfolgenden 7er.

Sei $f(n) = \sum L(S)$ für alle $n$-Folgen $S$.

Zum Beispiel gilt $f(3) = 45$, $f(7) = 1\\,403\\,689$ und $f(11) = 481\\,496\\,895\\,121$.

Finden Sie $f(7\\,500\\,000)\bmod 1\\,000\\,000\\,009$.
