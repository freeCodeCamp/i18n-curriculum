# --description--

讓我們在以下範例函式的背景關係中討論參數和引數：

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

在 JavaScript 中，參數是函式宣告中列在小括弧 `()` 之間的條款。函式引數是你決定傳遞給函式的實際值。

在上述範例中，函式定義寫在第一行：`function favoriteAnimal(animal)`。參數 `animal` 位於小括弧內。你也可以輕易地將 `animal` 替換成 `pet`、`x` 或 `blah`。但在這個例子中，將參數命名為 `animal`，能讓閱讀你程式碼的人獲得一些背景關係，避免他們必須猜測 `animal` 最終可能包含什麼。
