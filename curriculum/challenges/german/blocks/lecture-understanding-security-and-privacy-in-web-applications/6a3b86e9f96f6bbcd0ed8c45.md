# --description--

HTTPS steht für Hypertext Transfer Protocol Secure. Es ist die sichere Version von HTTP.

Wie Sie in einigen vorherigen Vorlesungen gelernt haben, überträgt HTTP Daten zwischen Ihrem Browser und einer Webseite. Standard-HTTP sendet diese Daten im Klartext, sodass jeder diesen Klartext abfangen kann.

Aus diesem Grund verwendet HTTPS ein kryptografisches Protokoll namens Transport Layer Security (TLS), um diese Datenübertragung zu sichern. TLS verschlüsselt den Kommunikationskanal zwischen zwei Punkten. Ältere Systeme nutzten Secure Sockets Layer (SSL), das viele Webhosting-Plattformen noch heute verwenden.

Die Verschlüsselung verwandelt diese Daten in unlesbaren Code, den nur der Browser des Nutzers und der Webserver mit den Schlüsseln wieder entschlüsseln können.

Zum Beispiel, wenn Sie ein Passwort auf einer Webseite eingeben, verschlüsselt HTTPS es sofort und schützt Kreditkarteninformationen, persönliche Adressen, Anmeldedaten und andere sensible Informationen. Hacker, die die Daten abfangen, sehen nur zufällige Zeichen und können die gestohlenen Informationen nicht verwenden.

Dieser Schutz ist auf öffentlichen Wi-Fi-Netzwerken entscheidend. Offene Wi-Fi-Netzwerke in Cafés, Flughäfen und anderen öffentlichen Orten verfügen über keine grundlegende Sicherheit, was Cyberkriminellen ermöglicht, unverschlüsselten Datenverkehr in diesen Netzwerken zu überwachen. HTTPS schafft einen privaten Tunnel durch das öffentliche Netzwerk, um Ihren Browserverlauf vor nahegelegenen Angreifern zu schützen.

HTTPS bietet auch Authentifizierung, die beweist, dass eine Webseite legitim ist. Um HTTPS zu verwenden, muss ein Webseitenbesitzer ein digitales Zertifikat erhalten, das von einer vertrauenswürdigen Organisation ausgestellt wird. Das Zertifikat bestätigt die wahre Identität des Webseitenbesitzers.

Wenn Sie eine Webseite besuchen und die Adresse nur mit „http“ beginnt, ist das ein Hinweis darauf, dass der Webseitenbesitzer dieses Zertifikat nicht erhalten hat, was die Webseite unsicher macht. Moderne Browser warnen Sie ebenfalls vor solchen Webseiten.

Schließlich ist Datenintegrität ein weiterer wichtiger Vorteil von HTTPS. Das bedeutet, dass niemand Ihre Daten während der Übertragung verändern kann. Ein Hacker kann keinen schädlichen Code in die Webseitendateien einschleusen, während sie zu Ihrem Gerät reisen. Sie erhalten genau die Daten, die der Webseitenbesitzer gesendet hat.

Suchmaschinen wie Google bevorzugen HTTPS-Webseiten, da sichere Webseiten in den Suchergebnissen höher eingestuft werden. Moderne Browser verlangen auch HTTPS für erweiterte Funktionen wie Geolokalisierung, Push-Benachrichtigungen und Webkameras.
