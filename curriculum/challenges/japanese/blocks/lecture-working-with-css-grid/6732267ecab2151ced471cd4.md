# --interactive--

前のレッスンでは、グリッドアイテム間にスペースを作る方法について少し話しました。しかし、このレッスンでは、グリッドレイアウトで`row-gap`、`column-gap`、`gap`のプロパティを使う方法について詳しく掘り下げます。

その前に、CSSグリッドにおけるトラックとは何かを復習しましょう。トラックとは、隣接する2つのグリッドラインの間のスペースのことです。これらのラインはCSS Gridを使うと自動的に作成されます。この文脈では、トラックは一般的にグリッドレイアウトを構成する行と列を指します。

CSS Gridで列間にギャップを作るには、`column-gap`プロパティを使います。このプロパティに指定できる値は、ピクセル、`em`単位、パーセンテージ、または`normal`キーワードです。

`normal`プロパティに`column-gap`の値を使うと、グリッドレイアウトでは`0`が、複数列レイアウトでは`1em`が結果として得られます。

以下は4列のグリッドレイアウトのマークアップ例です：

```html
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

CSSでは、displayをgridに設定し、`column-gap`プロパティを10ピクセルに設定しています：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  height: 100px;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.grid-container div {
  background-color: darkblue;
}
```

:::

この例を、青いボックスが2行になるように変更し、行間のスペースを増やしたい場合は、`row-gap`プロパティを使います：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  column-gap: 10px;
  row-gap: 30px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

この修正例では、`row-gap`プロパティを30ピクセルに設定し、`grid-template-columns`を4つの`1fr`単位から2つに変更して2行のボックスを作っています。

`column-gap`プロパティと同様に、`row-gap`プロパティに指定できる値はパーセンテージ、`em`、ピクセルなどがあります。

行と列の間のギャップを省略形で作りたい場合は、`gap`プロパティを使えます。基本的な構文は以下の通りです：

```css
gap: row-value optional-column-value;
```

`gap`プロパティに1つの値を指定すると、その値が行と列の両方に適用されます。2つの値を指定すると、最初の値が行に、2番目の値が列に適用されます：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 30px 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

`gap`省略形プロパティに指定できる値は、パーセンテージ、ピクセル、`em`、さらには`calc()`の値も含まれます。ただし、`fr`単位はここでは使えません。

`row-gap`、`column-gap`、`gap`のプロパティは、CSS Gridレイアウト内のアイテム間のスペースを柔軟に制御する方法を提供します。これらのプロパティを使うことで、行と列の間に一貫性があり調整可能なギャップを持つ、見た目に美しいグリッドを簡単に作成できます。
