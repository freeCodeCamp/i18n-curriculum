# --interactive--

التطبيق الجزئي هو تقنية نحول فيها دالة تأخذ عدة معلمات إلى متتالية من الدوال، كل واحدة تأخذ معلمة واحدة فقط.

لنبدأ بمثال بسيط. تخيل أن لدينا دالة تجمع رقمين:

:::interactive_editor

```js
function add(a, b) {
  return a + b;
}

console.log(add(3, 4)); // 7
```

:::

هذه دالة تأخذ معلمتين وتُرجع مجموعهما. الآن، لنرَ كيف يمكننا تطبيق التقنية الجزئية على هذه الدالة:

:::interactive_editor

```js
function curriedAdd(a) {
  return function(b) {
    return a + b;
  }
}

console.log(curriedAdd(3)(4)); // 7
```

:::

في هذا الكود المحول إلى تطبيق جزئي، بدلاً من أخذ معلمتين مرة واحدة، لدينا دالة تأخذ المعلمة الأولى وتُرجع دالة أخرى. هذه الدالة المُرجعة تأخذ المعلمة الثانية وتُجري الجمع. نستدعيها مثل `curriedAdd(3)(4)`، حيث تمثل كل زوج من الأقواس العادية استدعاء دالة.

لكن لماذا نريد فعل ذلك؟

التطبيق الجزئي يسمح لنا بإنشاء دوال خاصة بسهولة. على سبيل المثال، يمكننا إنشاء دالة تضيف دائمًا خمسة إلى أي رقم:

:::interactive_editor

```js
function curriedAdd(a) {
  return function(b) {
    return a + b;
  }
}

const addFive = curriedAdd(5);
console.log(addFive(10)); // 15
console.log(addFive(20)); // 25
```

:::

هنا، `addFive` هي دالة جاهزة دائمًا لإضافة خمسة إلى أي رقم نعطيها إياه. هذا مثال بسيط على التطبيق الجزئي، حيث نثبت عددًا معينًا من المعلمات لدالة، مما ينتج دالة أخرى تأخذ معلمات أقل.

بينما ركزت أمثلتنا على الدوال ذات المعلمتين، يمكن تطبيق التطبيق الجزئي على الدوال التي لها أي عدد من المعلمات.

مع تقدمك في تعلم JavaScript، ستجد العديد من الحالات التي يجعل فيها التطبيق الجزئي شفرتك أنظف وأكثر تعبيرًا.

## --text--

أي من التالي هو مثال على دالة مطبقة جزئيًا في JavaScript؟

## --answers--

```js
function add(a, b, c) {
  return a + b + c;
}
```

### --feedback--

هذه دالة عادية تأخذ جميع المعلمات مرة واحدة.

---

```js
function add(a) {
  return function (b) {
    return function (c) {
      return a + b + c;
    };
  };
}
```

---

```js
function add(a, b, c) {
  return a() + b() + c();
}
```

### --feedback--

هذه الدالة تقبل جميع المعلمات مرة واحدة وتستدعيها كدوال، وهذا لا يتوافق مع تعريف الدالة المطبقة جزئيًا.

---

```js
function add(a) {
  function inner(b) {
    return a + b;
  }
}
```

### --feedback--

الدالة الداخلية لا تُرجع أبدًا، لذا `add(2)` سترجع `undefined` و`add(2)(3)` ستُطلق خطأ.
