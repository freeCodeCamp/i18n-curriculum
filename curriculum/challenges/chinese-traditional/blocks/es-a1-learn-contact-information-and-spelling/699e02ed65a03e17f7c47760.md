# --description--

如你所見，`repetir` 意思是再說一次某事。當你添加代名詞 `lo`，你是在取用已經提過的東西。該代名詞取代特定的東西（字、數字、電子郵件位址等）。所以：

- `Repetir` = 重複

- `Repetirlo`＝重複它

`Repetirlo` 用於「那件事」已從背景關係中得知時。相同的樣式也適用於其他動詞：

- `¿Puedes escribirlo?` - 你可以寫出來嗎？

- `¿Puedes decirlo otra vez?` - 你可以再說一次嗎？

# --explanation--

`Lo` 是一個直接受詞代名詞，用來取代對話中已知的事物。當附加在不定詞上時，會形成一個字：

- `repetir` + `lo` → `repetirlo` － 重複它

- `escribir` + `lo` → `escribirlo` - 寫它

- `decir` + `lo` → `decirlo`－說它

當監聽器已經了解「it」所指為何時，此結構很常見。
