# --explanation--

`Would` drückt in diesem Kontext eine gewohnheitsmäßige Handlung oder ein wiederholtes Verhalten in der Vergangenheit aus. Es wird oft verwendet, um Handlungen zu beschreiben, die jemand regelmäßig ausführte, ähnlich wie `used to`.

Allerdings konzentriert sich `would` mehr auf die Handlung selbst, während `used to` Zustände oder Gewohnheiten beschreiben kann. Zum Beispiel:

- `She would always read a book before going to bed.` – Ein Buch vor dem Schlafengehen zu lesen war eine häufige Handlung für sie.

`She used to live in New York before moving here.` – hier wird `used to` verwendet, weil das Leben an einem Ort einen Zustand ausdrückt, keine Handlung oder etwas, das man gewohnheitsmäßig tut.

In diesem Satz verwendet Maria `would`, um Brians wiederholte Gewohnheit zu beschreiben, lange zu bleiben, um den Code zu verfeinern, was seine Hingabe betont.
