# --explanation--

تُستخدم `Might` للتعبير عن الاحتمال أو عدم اليقين. تشير إلى أن شيئًا ما ممكن، لكنه غير مؤكد. على سبيل المثال:

- `The system might crash if too many users log in at the same time.` - هناك احتمال تعطل النظام، لكنه غير مؤكد.

- `He might finish the coding task by the end of the day.` - قد ينهي مهمة الترميز بحلول نهاية اليوم، لكنه ليس مؤكدًا.
