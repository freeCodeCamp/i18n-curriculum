# --interactive--

Quando lavori con le stringhe in JavaScript, ci sono momenti in cui devi includere caratteri speciali che altrimenti il motore JavaScript potrebbe interpretare in modo errato.

Due attività comuni riguardano la creazione di una nuova riga all’interno di una stringa e l’escape di certi caratteri (come le virgolette) per assicurarti che appaiano correttamente.

In molti linguaggi di programmazione, incluso JavaScript, puoi creare una nuova riga in una stringa usando un carattere speciale chiamato sequenza di escape. La sequenza di escape più comune per le nuove righe è `\n`.

Per esempio, se vuoi spezzare una stringa in più righe, useresti `\n` dove vuoi che inizi la nuova riga:

:::interactive_editor

```js
let poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

La sequenza di escape `\n` dice a JavaScript di inserire un’interruzione di riga in quel punto, il che fa sì che la stringa venga mostrata su più righe.

Un altro concetto importante quando lavori con le stringhe è fare l’escape dei caratteri. A volte devi includere nella tua stringa caratteri che JavaScript normalmente usa per qualcos’altro, come le virgolette.

Se usi semplicemente le virgolette dentro una stringa senza fare l’escape, può causare un errore perché JavaScript penserà che tu stia cercando di terminare la stringa.

Per esempio, questo causerà un errore:

```js
let statement = "She said, "Hello!"";
```

JavaScript si confonde perché pensa che la stringa finisca dopo la parola `"said,"` ma tu vuoi che le virgolette intorno a `"Hello!"` facciano parte della stringa.

Per risolvere questo, puoi fare l’escape delle virgolette interne mettendo una barra rovesciata (`\`) prima di esse:

:::interactive_editor

```js
let statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

La barra rovesciata dice a JavaScript di trattare le virgolette come caratteri letterali, così appaiono correttamente nell’output.

Puoi anche fare l’escape di altri caratteri speciali, come la barra rovesciata stessa (`\\`), o le virgolette singole all’interno di una stringa racchiusa da virgolette singole (`\'`).

Ecco un altro esempio usando le virgolette singole:

:::interactive_editor

```js
let quote = 'It\'s a beautiful day!';
console.log(quote); // It's a beautiful day!
```

:::

Facendo l’escape della virgoletta singola con `\'`, JavaScript sa di includerla come parte della stringa invece di terminare la stringa prematuramente.

Fare l’escape e creare nuove righe sono essenziali quando formatti l’output o gestisci caratteri speciali nelle stringhe. Queste tecniche ti aiutano a evitare errori e a garantire che il testo appaia esattamente come desideri.
