# --description--

**Ziel:** Erstellen Sie eine App, die funktional ähnlich ist wie diese: <a href="https://choropleth-map.freecodecamp.rocks" target="_blank" rel="noopener noreferrer nofollow">https://choropleth-map.freecodecamp.rocks</a>.

Erfüllen Sie die untenstehenden User Stories und bestehen Sie alle Tests. Verwenden Sie beliebige Bibliotheken oder APIs, die Sie benötigen. Verleihen Sie ihr Ihren eigenen Stil.

Sie können HTML, JavaScript, CSS und die auf SVG basierende Visualisierungsbibliothek D3 verwenden. Die erforderlichen DOM-Elemente werden zum Zeitpunkt jedes Tests abgefragt. Wenn Sie ein Frontend-Framework (wie zum Beispiel Vue) verwenden, können die Testergebnisse für dynamische Inhalte ungenau sein. Wir hoffen, diese in Zukunft zu unterstützen, aber derzeit werden diese Frameworks für D3-Projekte nicht unterstützt.

**User Story #1:** Meine Choroplethenkarte sollte einen Titel mit einer entsprechenden `id="title"` haben.

**User Story #2:** Meine Choroplethenkarte sollte ein Beschreibungselement mit einer entsprechenden `id="description"` haben.

**User Story #3:** Meine Choroplethenkarte sollte Landkreise mit einer entsprechenden `class="county"` haben, die die Daten darstellen.

**User Story #4:** Es sollten mindestens 4 verschiedene Füllfarben für die Landkreise verwendet werden.

**User Story #5:** Meine Landkreise sollten jeweils `data-fips` und `data-education` Eigenschaften haben, die ihre entsprechenden `fips` und `education` Werte enthalten.

**User Story #6:** Meine Choroplethenkarte sollte für jeden bereitgestellten Datenpunkt einen Landkreis haben.

**User Story #7:** Die Landkreise sollten `data-fips` und `data-education` Werte haben, die mit den Beispieldaten übereinstimmen.

**User Story #8:** Meine Choroplethenkarte sollte eine Legende mit einer entsprechenden `id="legend"` haben.

**User Story #9:** Für die Legende sollten mindestens 4 verschiedene Füllfarben verwendet werden.

**User Story #10:** Ich kann mit der Maus über einen Bereich fahren und sehe einen Tooltip mit einer entsprechenden `id="tooltip"`, der weitere Informationen über den Bereich anzeigt.

**User Story #11:** Mein Tooltip sollte eine `data-education` Eigenschaft haben, die dem `data-education` des aktiven Bereichs entspricht.

Hier sind die Datensätze, die Sie für dieses Projekt benötigen:

- **US-Bildungsdaten:**`https://cdn.freecodecamp.org/testable-projects-fcc/data/choropleth_map/for_user_education.json`
- **US-Landkreisdaten:**`https://cdn.freecodecamp.org/testable-projects-fcc/data/choropleth_map/counties.json`

Sie können Ihr Projekt erstellen, indem Sie <a href='https://codepen.io/pen?template=MJjpwO' target="_blank" rel="noopener noreferrer nofollow">diese CodePen-Vorlage verwenden</a> und auf `Save` klicken, um Ihren eigenen Pen zu erstellen. Oder Sie können diesen CDN-Link verwenden, um die Tests in jeder Umgebung auszuführen, die Sie möchten: `https://cdn.freecodecamp.org/testable-projects-fcc/v1/bundle.js`

Wenn Sie fertig sind, senden Sie die URL Ihres funktionierenden Projekts mit allen bestandenen Tests ein.
