# --questions--

## --text--

Ось повідомлення, яке Джейк надіслав клієнтам після фішингової атаки:

`We're currently addressing the issue and taking all necessary steps to secure the system.`  

`We've run a full system scan to identify hidden malware, and the IT support team is actively monitoring the situation.`  

`I'll provide you with regular updates to ensure transparency.`  

`Please let us know if you have further concerns.`  

Клієнти відповіли:  

`We're very concerned about the security of our data. Can you confirm when this issue will be fully resolved?`

Який наступний крок, про який згадує Джейк у своєму повідомленні?

# --explanation--

Щоб знайти наступний крок, про який згадує Джейк, зверніть увагу на конкретні дії, описані в його повідомленні:  

`We've run a full system scan` показує, що цей крок уже виконано.  

`The IT support team is actively monitoring the situation` — це наступний крок, який зараз виконується для захисту системи.  

Джейк ще не згадує про вирішення проблеми або надання додаткових деталей про фішингову атаку.
