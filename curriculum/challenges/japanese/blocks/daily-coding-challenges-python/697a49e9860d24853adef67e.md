# --description--

山の積雪深と斜面の傾斜から、雪崩のリスクがあるかどうかを判定してください。

- 積雪深の値は `"Shallow"`、`"Moderate"`、または `"Deep"` です。
- 傾斜の値は `"Gentle"`、`"Steep"`、または `"Very Steep"` です。

このテーブルに基づいて `"Safe"` または `"Risky"` を返してください：

| |`"Shallow"`|`"Moderate"`|`"Deep"`|
|-|-|-|-|
|`"Gentle"`|`"Safe"`|`"Safe"`|`"Safe"`|
|`"Steep"`|`"Safe"`|`"Risky"`|`"Risky"`|
|`"Very Steep"`|`"Safe"`| `"Risky"`|`"Risky"`|

        },
    }
