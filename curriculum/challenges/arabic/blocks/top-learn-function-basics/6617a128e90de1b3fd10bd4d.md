# --description--

لنناقش المعلمات والمعاملات في سياق دالة المثال التالية:

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

في JavaScript، المعلمات هي العناصر المدرجة بين الأقواس العادية `()` في إعلان الدالة. معاملات الدالة هي القيم الفعلية التي تقرر تمريرها إلى الدالة.

في المثال أعلاه، يُكتب تعريف الدالة في السطر الأول: `function favoriteAnimal(animal)`. المعلمة، `animal`، موجودة داخل الأقواس العادية. يمكنك بسهولة استبدال `animal` بـ `pet` أو `x` أو `blah`. لكن في هذه الحالة، تسمية المعلمة بـ `animal` تعطي من يقرأ شفرتك بعض السياق حتى لا يضطر إلى التخمين بما قد تحتويه `animal` في النهاية.
