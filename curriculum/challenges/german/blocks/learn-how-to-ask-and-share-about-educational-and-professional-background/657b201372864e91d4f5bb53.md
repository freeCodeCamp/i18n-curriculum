# --description--

Die Regel zur Konjugation von Verben im Präteritum, die mit `-y` enden, hängt vom Buchstaben vor dem `-y` ab. So funktioniert es:

Wenn das Verb mit einem **Vokal gefolgt von `-y`** endet, fügen Sie einfach `-ed` an das Ende des Verbs an. Zum Beispiel wird aus `stay` `stayed`.

Wenn das Verb mit einem **Konsonanten gefolgt von `-y`** endet, ändern Sie das `-y` zu `-i` und fügen `-ed` hinzu. Zum Beispiel wird aus `study` `studied`.
