# --interactive--

범용 선택자(`*`)는 문서 내 모든 요소와 일치하는 특별한 유형의 CSS 선택자입니다.

이는 페이지의 모든 요소에 스타일을 적용할 때 자주 사용되며, 서로 다른 브라우저 간 스타일을 재설정하거나 표준화하는 데 유용합니다.

범용 선택자는 특정 컨텍스트 내의 모든 요소를 선택하거나 문서 전체에 전역적으로 사용할 수 있습니다.

다음은 범용 선택자를 사용해 전체 HTML 문서의 `margin`와 `padding`를 설정하는 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1>Heading element</h1>
<p>example paragraph element</p>
```

```css
* {
  margin: 0;
  padding: 0;
}
```

:::

이 코드 예제에서 `*` 선택자는 모든 요소의 여백과 패딩을 0으로 재설정하는데, 이는 CSS 리셋에서 흔히 사용하는 기법입니다.

범용 선택자는 모든 선택자 중에서 가장 낮은 특이성 값을 가집니다. 특이성 값의 모든 부분에 0을 기여합니다 `(0, 0, 0, 0)`.

즉, 타입 선택자, 클래스 선택자, ID 선택자, 인라인 스타일을 포함한 다른 모든 선택자가 범용 선택자가 설정한 스타일을 재정의합니다.

다음 HTML과 CSS 예제를 살펴보겠습니다:

:::interactive_editor

```html
<head>
  <style>
    * {
      color: blue;
    }
    p {
      color: red;
    }
    .highlight {
      color: green;
    }
    #unique {
      color: purple;
    }
  </style>
</head>
<body>
  <p id="unique" class="highlight">This text has multiple styles applied.</p>
</body>
```

:::

범용 `*` 선택자는 모든 요소의 텍스트 색상을 `blue`로 설정합니다.

하지만 `p` 타입 선택자가 `red` 요소에 대해 텍스트 색상을 `p`로 설정하여 이를 재정의합니다.

요소에 `highlight` 클래스가 있으면 클래스 선택자가 우선하여 텍스트 색상을 `green`로 변경합니다.

마지막으로, 가장 높은 특이성을 가진 `#unique` ID 선택자가 모든 다른 선택자를 재정의하여 텍스트 색상을 `purple`로 설정합니다.
