# --description--

雖然紅色和藍色筆看起來相同，但綠色筆比之前淺得多。 這是因爲 `green` 顏色關鍵字實際上是一個較暗的色調，大約介於黑色和綠色最大值之間。

在 `.two` CSS 規則中，將 `rgb` 函數中的綠色值設置爲 `127` 以降低其強度。

# --hints--

你的 `.two` CSS 規則應該有一個 `background-color` 屬性，設置爲 `rgb(0, 127, 0)`。

```js
assert.strictEqual(new __helpers.CSSHelp(document).getStyle('.two')?.backgroundColor, 'rgb(0, 127, 0)');
```
