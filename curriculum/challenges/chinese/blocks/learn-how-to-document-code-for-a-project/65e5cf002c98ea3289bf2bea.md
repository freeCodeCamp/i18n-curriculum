# --description--

当以 `y` 结尾的名词变为复数时，`y` 通常会变为 `ies`。此规则在某些术语经常以 `y` 结尾的技术上下文中特别有用。

# --questions--

## --text--

以 `y` 结尾的名词复数形式如何构成？
