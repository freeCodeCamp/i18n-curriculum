# --description--

Ili kutangaza kigezo kisibadilike (kisicho badilika), tumia `const` badala ya `let`:

```javascript
const myBirthday = '16-09-2003';
```

Vigezo vinavyotangazwa kwa kutumia `const` huitwa "constants" (visibadiliki). Haviwezi kupewa thamani tena. Jaribio la kufanya hivyo litasababisha kosa:

```javascript
const myBirthday = '16-09-2003';

myBirthday = '01-01-2001'; // error, can't reassign the constant!
```

Wakati mpogramu anapohakika kuwa kigezo hakitabadilika kamwe, anaweza kukitangaza kwa `const` ili kuhakikisha na kuwasilisha ukweli huo kwa kila mtu.
