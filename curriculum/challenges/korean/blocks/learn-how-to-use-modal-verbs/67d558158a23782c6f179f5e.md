# --explanation--

Jessica는 Mark에게 안전을 보장하기 위해 `regular security audits`를 실행하라고 말하고 있습니다.

`security audit`는 시스템의 취약점을 점검하는 과정입니다. 예를 들어:

`After the security audit, the team made improvements to the system.` - 검토 후 안전을 향상시키기 위해 변경이 이루어졌습니다.

`Regular`는 시간이 지나도 계속되는 것을 의미합니다. 예를 들어:

`Regular testing helps find problems before they get worse.` - 자주 테스트하면 문제를 조기에 발견하는 데 도움이 됩니다.
