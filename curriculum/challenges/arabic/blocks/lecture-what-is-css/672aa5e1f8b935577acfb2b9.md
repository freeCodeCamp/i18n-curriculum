# --interactive--

CSS، التي تعني أوراق الأنماط المتتالية (Cascading Style Sheets)، هي مكوِّن أساسي في تطوير الويب الحديث. إنها لغة أوراق أنماط تُستخدم لتطبيق الأنماط على HTML. ببساطة، إذا كان HTML هو هيكل صفحة الويب، فإن CSS هو ما يجعلها تبدو جميلة.

إليك مثالًا على تنسيق شريط التنقل.

**ملاحظة**: لا تقلق بشأن محاولة فهم شفرة CSS الآن. ستتعلم كيف يعمل كل هذا في الدروس وورش العمل القادمة. هذا فقط ليعطيك فكرة عما يمكنك فعله بقليل من CSS.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<nav class="navbar">
    <ul class="nav-links">
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Services</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>
```

```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: 'Segoe UI', sans-serif;
  background-color: #f4f4f4;
}

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  background-color: #333;
  padding: 1rem 2rem;
  color: white;
}

.navbar .logo {
  font-size: 1.5rem;
  font-weight: bold;
}

.nav-links {
  list-style: none;
  display: flex;
  gap: 1.5rem;
}

.nav-links a {
  color: white;
  text-decoration: none;
  font-weight: 500;
  transition: color 0.3s ease;
}

.nav-links a:hover {
  color: #ff9800;
}
```

:::

الدور الأساسي لـ CSS هو فصل محتوى صفحة الويب عن عرضها البصري. هذا الفصل يسمح لمطوِّري الويب بإنشاء مواقع أكثر مرونة وسهولة في الصيانة.

مع CSS، يمكنك التحكم في التخطيط، والألوان، والخطوط، والمظهر البصري العام لصفحات الويب دون تغيير هيكل HTML.

لنفترض تشبيهًا بسيطًا. إذا اعتبرت الموقع كمنزل، فـ HTML سيكون الأساس والإطار، بينما CSS سيكون الطلاء، وورق الحائط، والديكورات التي تجعل المنزل جذابًا وفريدًا بصريًا.

يعمل CSS عن طريق اختيار عناصر HTML وتطبيق الأنماط عليها.

إليك مثالًا يوضح عنصر `button` غير منسَّق وآخر منسَّق. للتفاعل مع هذا المثال، تحتاج إلى تفعيل المحرر التفاعلي.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<button class="regular-btn">Unstyled Button</button>
<button class="round-btn">Round Button</button>
```

```css

.round-btn {
  font-size: 1rem;
  font-family: 'Segoe UI', sans-serif;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease, transform 0.2s ease;
  background-color: #2ecc71;
  color: white;
  border-radius: 50px;
  padding: 0.6rem 1.6rem;
}

.round-btn:hover {
  background-color: #27ae60;
  transform: translateY(-2px);
}
```

:::

يمكن أن تشمل هذه الأنماط خصائص مثل اللون، وحجم الخط، وغيرها الكثير. بتغيير هذه الخصائص، يمكنك تغيير مظهر صفحة الويب بشكل كبير دون تغيير محتواها.

واحدة من أقوى ميزات CSS هي قدرتها على إنشاء تصاميم متجاوبة.

هذا يعني أنه مع CSS، يمكنك جعل موقعك يبدو رائعًا على أي جهاز، سواء كان حاسوبًا مكتبيًا، أو جهاز لوحي، أو هاتفًا ذكيًا.

يسمح CSS بتعديل التخطيطات، وأحجام الخطوط، والعناصر البصرية الأخرى بناءً على حجم شاشة الجهاز الذي يعرض الموقع.

جرّب تفعيل المحرر التفاعلي وضبط حجم نافذة المعاينة لترى كيف يتكيف التخطيط مع حجم الشاشة.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<header class="header">
    <h1>Welcome to My Responsive Site</h1>
    <p>This layout adapts to your screen size!</p>
</header>

<main class="content">
    <section class="box">Box 1</section>
    <section class="box">Box 2</section>
    <section class="box">Box 3</section>
</main>
```

```css
body {
  margin: 0;
  font-family: 'Segoe UI', sans-serif;
  background-color: #f9f9f9;
  color: #333;
}

.header {
  background-color: #4CAF50;
  color: white;
  padding: 2rem;
  text-align: center;
}

.content {
  display: flex;
  gap: 1rem;
  padding: 1rem;
  justify-content: center;
}

.box {
  flex: 1;
  min-width: 200px;
  background-color: #ddd;
  padding: 2rem;
  text-align: center;
  font-size: 1.2rem;
  border-radius: 8px;
  transition: background-color 0.3s;
}

.box:hover {
  background-color: #ccc;
}

/* Responsive: for tablets and below (≤ 768px) */
@media (max-width: 768px) {
  .content {
    flex-direction: column;
    align-items: center;
  }

  .box {
    width: 90%;
    font-size: 1.1rem;
  }
}

/* Responsive: for phones (≤ 480px) */
@media (max-width: 480px) {
  .header {
    padding: 1.5rem 1rem;
  }

  .header h1 {
    font-size: 1.5rem;
  }

  .header p {
    font-size: 1rem;
  }

  .box {
    font-size: 1rem;
    padding: 1.5rem;
  }
}
```

:::

ميزة أخرى مهمة في CSS هي طبيعتها المتتالية، وهي مصدر كلمة "cascading" في اسمها.

هذا يعني أن الأنماط يمكن أن تُورث وتُلغى، مما يسمح ببنية هرمية للأنماط. ستتعلم المزيد عن كيفية عمل هذا في الدروس القادمة.

يدعم CSS أيضًا استخدام أوراق أنماط خارجية.

هذا يعني أنه يمكنك الاحتفاظ بكل قواعد التنسيق في ملف منفصل، يمكن ربطه بعد ذلك بعدة صفحات HTML.

تعزز هذه الميزة بشكل كبير من سهولة صيانة المواقع، خاصة الكبيرة منها. بدلًا من تغيير الأنماط في كل صفحة على حدة، يمكنك تعديل ملف CSS واحد يؤثر على جميع الصفحات المرتبطة.

في عالم تطوير الويب، يلعب CSS دورًا حيويًا في إنشاء مواقع جذابة بصريًا، ومتجاوبة، وسهلة الاستخدام.

يسمح للمطوِّرين بتحويل مستندات HTML البسيطة إلى تجارب ويب مشوقة تجذب انتباه المستخدمين وتعزز تفاعلهم مع محتوى الويب.
