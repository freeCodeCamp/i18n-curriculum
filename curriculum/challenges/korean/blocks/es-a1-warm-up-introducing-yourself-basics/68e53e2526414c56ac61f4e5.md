# --description--

스페인어에서 숫자 20은 `veinte`입니다.

21부터 29까지 숫자를 만들 때 단어가 약간 변합니다. `Veinte`은 `veinti-`가 되며(마지막 `e`가 제거됨), 그다음 숫자와 합쳐집니다. 예를 들어:

- `Veinti` + `dos` (둘) = `veintidós` (22)

- `Veinti` + `cinco` (다섯) = `veinticinco` (25)

- `Veinti` + `ocho` (여덟) = `veintiocho` (28)
