# --description--

一场比赛由三堆石头和两名球员组成。

在每个玩家的回合，玩家从堆中移除一个或多个石头。但是，如果玩家从多个堆中取石头，则必须从每个选定的堆中移除相同数量的石头。

换句话说，玩家选择某个 $N > 0$ 并移除：

- 从任意单堆中取出 $N$ 个石头；或者
- 从任意两个堆中各取 $N$ 个石头（共 $2N$ 个）；或者
- 每个堆中各取 $N$ 个石头（共计 $3N$ 个）。

拿最后一块石头的玩家赢了比赛。

获胜配置是第一个玩家可以强制获胜的配置。

例如，（0,0,13），（0,11,11）和（5,5,5）是获胜配置，因为第一个玩家可以立即移除所有宝石。

失败的配置是第二个玩家可以强制获胜的配置，无论第一个玩家做什么。

例如，（0,1,2）和（1,3,3）正在失去配置：任何合法移动都会为第二个玩家留下获胜配置。

考虑所有满足 $x_i ≤ y_i ≤ z_i ≤ 100$ 的失败配置 ($x_i$,$y_i$,$z_i$)。我们可以验证这些配置的 $\sum (x_i + y_i + z_i) = 173\\,895$。

查找 $\sum (x_i + y_i + z_i)$，其中 ($x_i$,$y_i$,$z_i$) 在满足 $x_i ≤ y_i ≤ z_i ≤ 1000$ 的失败配置范围内。
