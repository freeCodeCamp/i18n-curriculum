`Same` significa invariato o identico. Per esempio:

- `We have the same task to complete this week.` - Questo significa che l'attività è identica per tutti.

- `They have the same opinion about the new software update.` - Questo significa che entrambe le persone sono d'accordo sull'aggiornamento.

Bob dice, `we still have the same issue`, che significa che il problema non è cambiato.
