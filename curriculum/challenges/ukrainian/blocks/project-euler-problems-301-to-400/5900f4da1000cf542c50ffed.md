# --description--

Двоє гравців, Антон і Бернгард, грають у таку гру.

Є одна купа з $n$ каменів.

Перший гравець може забрати будь-яку додатну кількість каменів, але не всю купу.

Після цього кожен гравець може забрати не більше ніж удвічі більше каменів, ніж його опонент забрав у попередньому ході.

Гравець, який забирає останній камінь, виграє.

Наприклад, $n = 5$

Якщо перший гравець забере більше ніж один камінь, наступний гравець зможе забрати всі решту каменів.

Якщо перший гравець забере один камінь, залишивши чотири, його опонент також забере один камінь, залишивши три.

Перший гравець не може забрати всі три, бо може взяти не більше ніж $2 \times 1 = 2$ камені. Тож припустимо, він також забирає один камінь, залишаючи 2.

Другий гравець може забрати два решту камені і виграє.

Отже, 5 — програшна позиція для першого гравця.

Для деяких виграшних позицій існує більше ніж один можливий хід для першого гравця.

Наприклад, коли $n = 17$, перший гравець може забрати один або чотири камені.

Нехай $M(n)$ — максимальна кількість каменів, яку перший гравець може забрати у виграшній позиції на своєму першому ході, а $M(n) = 0$ для будь-якої іншої позиції.

$\sum M(n)$ для $n ≤ 100$ дорівнює 728.

Знайдіть $\sum M(n)$ для $n ≤ {10}^{18}$. Відповідь подайте за модулем ${10}^8$.
