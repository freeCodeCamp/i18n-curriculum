# --description--

Kwa nambari kamili chanya $n$, fafanua $f(n)$ kama marudio ya chini kabisa chanya ya $n$ ambayo, ikiolekezwa katika msingi wa 10, inatumia tu nambari $≤ 2$.

Hivyo basi $f(2) = 2$, $f(3) = 12$, $f(7) = 21$, $f(42) = 210$, $f(89) = 1\\,121\\,222$.

Pia, $\displaystyle\sum_{n = 1}^{100} \frac{f(n)}{n} = 11\\,363\\,107$.

Tafuta $\displaystyle\sum_{n = 1}^{10\\,000} \frac{f(n)}{n}$.
