# --interactive--

L’evento `DOMContentLoaded` viene attivato quando tutto il documento HTML è stato caricato e analizzato. Se hai fogli di stile esterni o immagini, l’evento `DOMContentLoaded` non aspetterà che questi vengano caricati. Aspetterà solo che l’HTML sia stato caricato.

Questo è diverso dall’evento `load`, che aspetta che tutto sia caricato, inclusi fogli di stile esterni, immagini e così via.

Ecco la sintassi di esempio per usare l’evento `DOMContentLoaded`:

```js
document.addEventListener("DOMContentLoaded", function () {
  console.log("DOM is loaded.");
});
```

Una volta che il DOM è caricato, la funzione verrà eseguita e il messaggio `DOM is loaded.` sarà registrato nella console.

Ora, diamo un’occhiata a un altro esempio che usa l’evento `DOMContentLoaded`. In questo esempio, abbiamo un’immagine dentro l’HTML. Per aggiornare l’immagine, possiamo creare una funzione che cambia gli attributi `src` e `alt` dell’immagine:

:::interactive_editor

```html
<h1>Image Change on DOM Loaded</h1>
<img
  id="example-img"
  src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg"
  alt="Cat relaxing"
/>

<script src="index.js"></script>
```

```js
function changeImg() {
  const img = document.getElementById("example-img");
  img.src =
    "https://cdn.freecodecamp.org/curriculum/responsive-web-design-principles/FCCStickers-CamperBot200x200.jpg";
  img.alt = "CamperBot sticker";
  console.log("image was just changed");
}

changeImg();
```

:::

Possiamo quindi verificare se il DOM è ancora in caricamento e aggiungere un event listener per l’evento `DOMContentLoaded`. Se l’evento `DOMContentLoaded` è già stato attivato, possiamo chiamare direttamente la funzione `changeImg`:

```js
if (document.readyState === "loading") {
  document.addEventListener("DOMContentLoaded", changeImg);
} else {
  console.log("DOMContentLoaded has already fired");
  changeImg();
}
```

Se esegui questo codice e aggiorni la pagina, vedrai che l’immagine cambia in quella nuova dopo che il DOM è stato caricato. Succede abbastanza rapidamente, ma noterai un leggero lampeggio mentre l’immagine cambia.

L’evento `DOMContentLoaded` è utile quando vuoi eseguire del codice JavaScript non appena il DOM è caricato, ma prima che tutte le risorse esterne come immagini e fogli di stile siano caricate.
