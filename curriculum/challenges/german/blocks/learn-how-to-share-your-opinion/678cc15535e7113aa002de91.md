## --text--

Welcher Satz verwendet `I'd rather` korrekt?

## --answers--

`I'd rather use Python for data analysis than JavaScript.`

---

`I'd rather to use Python for data analysis than JavaScript.`

### --feedback--

`Rather` wird vom Infinitiv des Verbs gefolgt, nicht von `to`.

---

`I'd rather using Python for data analysis than JavaScript.`

### --feedback--

`Rather` wird vom Infinitiv des Verbs gefolgt, nicht von `using`.

---

`I'd rather use Python to data analysis than JavaScript.`

### --feedback--

Die Phrase `to data analysis` sollte `for data analysis` sein.

## --video-solution--

1

# --explanation--

`I'd rather` ist eine Phrase, die verwendet wird, um eine Präferenz zwischen zwei Handlungen oder Optionen auszudrücken. Sie wird vom Infinitiv des Verbs gefolgt.
