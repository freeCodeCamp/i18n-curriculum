# --explanation--

Presta atención a las palabras `will affect` y `deadlines` en el audio. Estas son clave para entender la pregunta.

Una `project deadline` es la fecha final para completar un proyecto.

`Will affect` se usa para preguntar si algo causará un cambio o impacto. Por ejemplo:

`The new policy will affect the project deadline.` - Esto significa que la política cambiará el cronograma del proyecto.
