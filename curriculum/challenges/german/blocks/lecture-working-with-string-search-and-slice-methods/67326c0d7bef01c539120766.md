# --interactive--

Wenn Sie mit Strings in JavaScript arbeiten, gibt es viele Fälle, in denen Sie prüfen müssen, ob ein String eine bestimmte Teilzeichenfolge enthält, also einen kleineren Teil dieses Strings.

Zum Beispiel möchten Sie vielleicht überprüfen, ob die Eingabe eines Benutzers ein bestimmtes Wort oder Zeichen enthält, bevor Sie eine Aktion ausführen. Eine Möglichkeit, dies zu erreichen, ist die Verwendung der `includes()`-Methode.

Die `includes()`-Methode wird verwendet, um zu prüfen, ob ein String eine bestimmte Teilzeichenfolge enthält. Wenn die Teilzeichenfolge im String gefunden wird, gibt die Methode `true` zurück, andernfalls `false`.

Hier ist die grundlegende Syntax:

```js
string.includes(searchValue);
```

Bei der Syntax ist der `searchValue` die Teilzeichenfolge, nach der Sie im String suchen möchten. Und hier ein Beispiel:

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("awesome");

console.log(result);  // true
```

:::

In diesem Beispiel wird das Wort `awesome` im String `JavaScript is awesome!` gefunden, daher gibt die `includes()`-Methode `true` zurück.

Es ist wichtig zu beachten, dass die `includes()`-Methode zwischen Groß- und Kleinbuchstaben unterscheidet. Das bedeutet, dass die genaue Übereinstimmung der Zeichen erforderlich ist, einschließlich ihrer Groß- und Kleinschreibung.

Zum Beispiel:

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("Awesome");

console.log(result);  // false
```

:::

Da `Awesome` (mit einem Großbuchstaben `A`) nicht mit `awesome` (mit Kleinbuchstaben `a`) übereinstimmt, ist das Ergebnis `false`.

Sie können auch die `includes()`-Methode verwenden, um nach einer Teilzeichenfolge ab einem bestimmten Index im String zu suchen, indem Sie einen zweiten Parameter angeben:

:::interactive_editor

```js
let text = "Hello, JavaScript world!";
let result = text.includes("JavaScript", 7);

console.log(result);  // true
```

:::

Hier beginnt die Suche nach der Teilzeichenfolge `JavaScript` ab der 7. Position im String, sodass alle Zeichen davor übersprungen werden.

Die `includes()`-Methode gibt nur ein `true`- oder `false`-Ergebnis zurück. Sie liefert keine Informationen darüber, wo sich die Teilzeichenfolge im String befindet oder wie oft sie vorkommt. Wenn Sie diese Details benötigen, sind andere Methoden, wie die `indexOf()`-Methode, möglicherweise besser geeignet.
