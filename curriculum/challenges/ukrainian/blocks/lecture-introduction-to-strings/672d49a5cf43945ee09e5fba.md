# --interactive--

У JavaScript рядок — це послідовність символів, яка використовується для представлення текстових даних. Рядки є одним із примітивних типів даних у мові, поряд із числами, булевими значеннями, `null` та `undefined`.

Щоб створити рядок у JavaScript, ви можете використовувати одинарні лапки (`'`) або подвійні лапки (`"`).

Ось приклад створення двох змінних, які містять рядкові значення:

:::interactive_editor

```js
let singleQuotes = 'This is a string';
console.log(singleQuotes);
let doubleQuotes = "This is also a string";
console.log(doubleQuotes);
```

:::

Хоча ви можете використовувати одинарні або подвійні лапки для створення рядків, важливо дотримуватися послідовності. Якщо рядок починається з одинарної лапки, він також повинен закінчуватися одинарною лапкою.

Те саме стосується подвійних лапок. Наступний приклад викличе помилку, бо починається з подвійної лапки, а закінчується одинарною:

```js
const improperStr = "Do not do this';
```

Ще один момент, який слід враховувати, — рядки є незмінними. В програмуванні незмінність означає, що після створення щось не можна змінити. Тож коли ви створюєте рядок, ви не можете безпосередньо змінити його символи. Замість цього потрібно створити новий рядок, якщо хочете внести зміни.

Ось приклад призначення нового рядка змінній `developer`:

:::interactive_editor

```js
let developer = "Jessica";
console.log(developer);
developer = "Quincy";
console.log(developer);
```

:::

Оскільки рядки незмінні, ми не можемо безпосередньо оновити перший рядок. Саме тому ми призначаємо новий рядок змінній `developer`.

Рядки — важлива частина програмування, і в майбутніх уроках ви вивчите просунуті техніки для їх маніпулювання та використання їх повного потенціалу для створення динамічних і інтерактивних застосунків.
