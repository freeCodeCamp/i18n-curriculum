# --description--

`Currently` 来源于 `current`，意思是正在发生。添加 `-ly` 会将形容词变成副词。例如：

`She is currently reading a book.` - 她现在正在读取一本书。

`Trends` 是许多人正在讨论或采用的流行风格或想法。单数形式的 `trend` 变为复数形式的 `trends`。例如：

- `One current trend is using eco-friendly products.` - 这里，`trend`（单数）指一个流行的想法。

- `She writes about fashion trends.` - 这里，`trends`（复数）指许多不同的流行风格。

`Tech trends` 指技术领域的热门发展。
