# --interactive--

En las lecciones anteriores, hemos estado trabajando con la propiedad `grid-template-columns` y estableciendo el valor en algunas unidades fraccionales.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

Aunque el siguiente código es completamente válido, hay una forma más sencilla de repetir una sección o todos los listados de tus pistas.

La función `repeat()` se usa para repetir una sección o todos los elementos de las columnas o filas. Esta función toma una cuenta de repetición y los elementos que deseas repetir.

Aquí tienes una versión revisada del ejemplo anterior usando la función de repetición:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

No habrá ningún cambio en los estilos en el navegador, pero esta es una manera más concisa de escribir valores repetidos para las columnas.

La función `repeat()` aceptará cualquier patrón válido que puedas usar para filas o columnas.

Aquí tienes un ejemplo usando la función `repeat()` para establecer la primera y tercera columnas en 20 píxeles y la segunda y cuarta columnas en una unidad fraccional.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(2, 20px 1fr);
  column-gap: 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

A veces, puedes optar por escribir cada valor individual en lugar de usar la función `repeat()`. Pero hay ocasiones en las que esta función resulta útil, especialmente cuando deseas repetir un patrón particular para un listado de pistas.
