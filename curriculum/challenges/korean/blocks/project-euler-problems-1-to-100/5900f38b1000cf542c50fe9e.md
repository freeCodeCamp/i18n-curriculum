# --description--

영국에서는 통화 단위가 파운드(£)와 펜스(p)로 구성되며, 일반적으로 유통되는 동전은 여덟 종류입니다:

<div style='margin-left: 4em;'>1p, 2p, 5p, 10p, 20p, 50p, £1 (100p) 그리고 £2 (200p).</div>

£2를 다음과 같이 만들 수 있습니다:

<div style='margin-left: 4em;'>1×£1 + 1×50p + 2×20p + 1×5p + 1×2p + 3×1p</div>

`n` 펜스를 동전을 아무 개수나 사용해서 만들 수 있는 서로 다른 방법은 몇 가지일까요?

# --solutions--

```js
const coinSums = (n) => {
  const getWays = (n, m=8, c=[1, 2, 5, 10, 20, 50, 100, 200]) => {
    if (n === 0) return 1;
    if (m === 0 || n < 0) return 0;
    return getWays(n - c[m - 1], m, c) + getWays(n, m - 1, c);
  };
  return getWays(n);
};
```
