# --description--

Die folgende Gleichung stellt die kontinuierliche Topografie einer Gebirgsregion dar und gibt die Höhe $h$ an jedem Punkt ($x$,$y$) an:

$$h = \left(5000 - \frac{x^2 + y^2 + xy}{200} + \frac{25(x + y)}{2}\right) \times e^{-\left|\frac{x^2 + y^2}{1\\,000\\,000} - \frac{3(x + y)}{2000} + \frac{7}{10}\right|}$$

Eine Mücke beabsichtigt, von A(200,200) nach B(1400,1400) zu fliegen, ohne den Bereich $0 ≤ x$, $y ≤ 1600$ zu verlassen.

Aufgrund der dazwischenliegenden Berge steigt sie zunächst senkrecht nach oben zu einem Punkt A' mit der Höhe $f$. Dann fliegt sie, während sie auf der gleichen Höhe $f$ bleibt, um alle Hindernisse herum, bis sie an einem Punkt B' direkt über B ankommt.

Bestimmen Sie zuerst $f_{min}$, das Minimum der konstanten Höhe, das eine solche Reise von A nach B ermöglicht, während Sie sich im angegebenen Bereich aufhalten. Finden Sie dann die Länge des kürzesten Wegs zwischen A' und B', während Sie auf dieser konstanten Höhe $f_{min}$ fliegen.

Geben Sie diese Länge als Ihre Antwort an, gerundet auf drei Dezimalstellen.

**Hinweis:** Der oben gezeigte Höhenfunktion wird der Komfort halber unten in einer Form wiederholt, die für die meisten Programmiersprachen geeignet ist: `h=( 5000-0.005*(x*x+y*y+x*y)+12.5*(x+y) )* exp( -abs(0.000001*(x*x+y*y)-0.0015*(x+y)+0.7) )`.
