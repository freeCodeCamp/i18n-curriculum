## 터미널, 셸, 명령줄 기본

- **명령줄**: 사용자가 명령어를 입력하는 텍스트 인터페이스입니다.  
- **터미널**: 명령줄에 접근할 수 있게 해주는 애플리케이션입니다.  
- **터미널 에뮬레이터**: 터미널에 추가 기능을 더한 프로그램입니다.  
- **셸**: 터미널에 입력된 명령어를 해석하는 프로그램입니다(예: Bash).  
- **PowerShell / 명령 프롬프트 / Microsoft Terminal**: Windows에서 명령줄에 접근하는 옵션입니다.  
- **터미널(macOS)**: macOS에 내장된 옵션이며, iTerm이나 Ghostty 같은 서드파티 대안도 있습니다.  
- **터미널(Linux)**: 배포판마다 다르며, kitty 같은 다양한 서드파티 에뮬레이터가 있습니다.  
- **용어**: "터미널", "셸", "명령줄"은 종종 혼용되지만 각각 명확한 의미가 있습니다.


## 명령줄 단축키

- **위/아래 화살표**: 이전/다음 명령어를 순환합니다.
- **Tab**: 명령어 자동 완성.
- **`Control+L`** (Linux/macOS) 또는 `cls` 입력 (Windows): 터미널 화면을 지웁니다.
- **`Control+C`**: 실행 중인 명령어를 중단합니다(PowerShell에서는 텍스트 선택 시 복사에도 사용).
- **`Control+Z`** (Linux/macOS 전용): 작업을 백그라운드로 일시 중단; `fg`로 다시 실행합니다.
- **`!!`**: 마지막 실행 명령어를 즉시 다시 실행합니다.

## Bash 기본

- **Bash** (Bourne Again Shell): 널리 쓰이는 유닉스 계열 셸입니다.  
  주요 명령어:

- **`pwd`**: 현재 디렉터리 표시.
  - **`cd`**: 디렉터리 변경.
    - **`..`**는 상위 디렉터리(한 단계 위)를 가리킵니다.
    - **`.`**는 현재 디렉터리를 가리킵니다.
  - **`ls`**: 파일과 폴더 목록 표시.
    - **`-a`**: 숨김 파일 포함 모든 파일 표시.
    - **`-l`**: 파일에 대한 상세 정보 표시.
  - **`less`**: 파일 내용을 한 페이지씩 보고 뒤로 스크롤 및 검색 가능.
  - **`more`**: 파일 내용을 한 화면씩 표시하며 제한된 뒤로 스크롤과 기본 탐색 가능.
  - **`cat`**: 스크롤이나 탐색 없이 전체 파일 내용을 한 번에 표시, 작은 파일에 유용.
  - **`mkdir`**: 새 디렉터리 생성.
  - **`rmdir`**: 빈 디렉터리 삭제.
  - **`touch`**: 새 파일 생성.
  - **`mv`**: 파일 이동 또는 이름 변경.
    - 이름 변경: `mv oldname.txt newname.txt`
    - 이동: `mv filename.txt /path/to/target/`
  - **`cp`**: 파일 복사.
    - **`-r`**: 디렉터리와 내용 재귀 복사.
  - **`rm`**: 파일 삭제.
    - **`-r`**: 디렉터리와 내용 재귀 삭제.
  - **`echo`**: 텍스트 한 줄 또는 변수 값 출력.
    - `>`로 파일 내용을 덮어쓸 수 있습니다. (예: `echo "text" > file.txt`)
    - `>>`로 기존 내용을 덮어쓰지 않고 출력 내용을 파일에 추가할 수 있습니다 (예: `echo "text" >> file.txt`).
  - **`exit`**: 터미널 세션 종료.
  - **`clear`**: 터미널 화면 지우기.
  - **`find`**: 파일 및 디렉터리 검색.
    - **`-name`**: 이름 패턴으로 파일 검색 (예: `find . -name "*.txt"`).
  - `man` 뒤에 명령어를 붙여 (예: `man ls`) 상세 매뉴얼/도움말 페이지에 접근.

## 명령 옵션과 플래그

- **옵션** 또는 **플래그**: 명령어의 동작을 수정하며 보통 하이픈으로 시작합니다:
  - **긴 형식(하이픈 두 개)**:  
    - 예: `--help`, `--version`
    - 값은 등호(=)로 붙여서 전달합니다, 예: `--width=50`.
  - **짧은 형식(하이픈 한 개)**:  
    - 예: `-a`, `-l`
    - 값은 공백으로 구분하여 전달합니다, 예: `-w 50`.
    - 여러 짧은 옵션을 한꺼번에 연결할 수 있습니다, 예: `ls -alh`.
- **`--help`**: 언제든지 이 플래그를 사용해 명령어의 사용 가능한 옵션을 확인할 수 있습니다.
