# --description--

حسنًا، لقد استعرضت الكثير حتى الآن. الشيء الوحيد المتبقي هو شرح كيفية إضافة كل هذا CSS إلى ملف HTML الخاص بك. هناك ثلاث طرق للقيام بذلك.

CSS الخارجي هو الطريقة الأكثر شيوعًا التي ستصادفها، ويتضمن إنشاء ملف منفصل لـ CSS وربطه داخل وسمَي `<head>` المفتوح والمغلق في HTML باستخدام عنصر `<link>` (وهو عنصر فارغ ولا يحتاج إلى وسم إغلاق):

أولًا، أضف وسمًا فارغًا `<link>` داخل وسمَي `<head>` المفتوح والمغلق في ملف HTML. الخاصية `href` تحدد موقع ملف CSS، سواء كان عنوان URL مطلقًا أو، وهو ما ستستخدمه، عنوان URL نسبيًا لموقع ملف HTML. في المثال أعلاه، تفترض أن كلا الملفين موجودان في نفس المجلد. الخاصية `rel` مطلوبة، وتحدد العلاقة بين ملف HTML والملف المرتبط.

ثم داخل ملف `styles.css` الذي أنشأته حديثًا، لديك المحدد (الـ `div` و `p`)، يتبعه زوج من الأقواس المتعرجة المفتوحة والمغلقة التي تُنشئ "كتلة إعلان". أخيرًا، تضع أي إعلانات داخل كتلة الإعلان. `color: white; ` هو إعلان واحد، حيث تكون `color` الخاصية و `white` هي القيمة، و `background-color: black;` هو إعلان آخر.

ملاحظة حول أسماء الملفات: `styles.css` هو الاسم الذي اخترته هنا للملف. يمكنك تسمية الملف بأي اسم تريده طالما أن نوع الملف هو `.css`، رغم أن "style" أو "styles" هما الأكثر استخدامًا.

بعض مزايا هذه الطريقة هي:

1. تحافظ على فصل HTML وCSS، مما يجعل ملف HTML أصغر ويبدو أكثر ترتيبًا.
2. تحتاج فقط لتعديل CSS في مكان واحد، وهذا مفيد جدًا للمواقع التي تحتوي على صفحات كثيرة تشترك في أنماط متشابهة.
