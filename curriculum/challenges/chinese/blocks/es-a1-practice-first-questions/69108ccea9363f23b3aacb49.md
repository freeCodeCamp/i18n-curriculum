# --explanation--

这个 Mateo 和 Camila 之间的对话框集中讨论职业。

`¿A qué te dedicas?` 意思是“你的职业是什么？”或“你是做什么工作的？”

**`Soy` + [profession]** 意思是“我是一个 [profession]”。

在西班牙语中，大多数职业名称会根据性别变化。然而，一些以 `-ista` 或 `-nte` 结尾的职业名称通常对两种性别保持完全相同：

| Profession | Masculine | Feminine |
| :---: | :---: | :---: |
| Analyst | `analista` | `analista` |
| Journalist | `periodista` | `periodista` |
| Receptionist | `recepcionista` | `recepcionista` |
| Singer | `cantante` | `cantante` |
