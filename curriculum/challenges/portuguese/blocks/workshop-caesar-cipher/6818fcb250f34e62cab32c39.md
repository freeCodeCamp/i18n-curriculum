# --description--

Você pode ter notado que, embora a mensagem esteja criptografada, os caracteres maiúsculos foram deixados intactos.Isso acontece porque a tabela de tradução não inclui letras maiúsculas.

Para corrigir, você precisará modificar as strings passadas para sua chamada `str.maketrans()` usando o método `upper()`:

```py
greet = "Hello"
print(greet.upper()) # HELLO
```

Atualize sua chamada `str.maketrans()` concatenando a cada argumento a versão maiúscula do próprio argumento.
