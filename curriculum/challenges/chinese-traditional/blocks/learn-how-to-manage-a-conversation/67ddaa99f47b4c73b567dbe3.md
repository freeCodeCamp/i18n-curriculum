# --explanation--

`Sure thing.` 是一種非正式且友善的說法，用來表示 `Of course` 或 `I'll do it`。它表示說話者毫不猶豫地同意該請求。 例如：

- 人物 1：`Can you send me the file this afternoon?`－此人正在索取一個檔案。

- 第二人：`Sure thing.`－這表示說話者同意並會依要求傳送該檔案。
