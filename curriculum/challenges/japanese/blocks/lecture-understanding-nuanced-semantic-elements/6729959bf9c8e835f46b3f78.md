# --interactive--

これらの要素は、プレゼンテーション用HTMLと意味論的HTMLの概念に非常に密接に関連しています。イディオマティックテキスト要素である`i`は、もともとテキストをイタリック体で表示するプレゼンテーション目的で使われていました。しかし現在では、別の声やムードの強調、他言語の慣用句、専門用語、思考の表現などに頻繁に使われています。

以下は公式HTML仕様からの例で、`i`要素を使ってフランス語の慣用句を示しています。

:::interactive_editor

```html
<p>There is a certain <i lang="fr">je ne sais quoi</i> in the air.</p>
```

:::

開く`lang`タグ内の`<i>`属性は、コンテンツの言語を指定するために使われます。この場合、言語はフランス語です。`i`要素はテキストが重要かどうかを示すものではなく、周囲のテキストとは何らかの違いがあることを示しています。

もしテキストの重要性を強調する必要がある場合は、意味論的に似た要素である強調要素`em`を使うことができます。これはより多くの文脈を提供する必要がある場合に推奨されます。周囲のテキストと比べて特別な強調が必要な部分にこの要素を使うべきです。文の意味を変える可能性があるため、通常は数語に限定されます。

以下は段落内での強調要素の例です。

:::interactive_editor

```html
<p>
  Never give up on <em>your</em> dreams.
</p>
```

:::

文`Never give up on your dreams`をご覧ください。`your`という単語がこの要素内にあるため強調されます。ブラウザでは`your`という単語がイタリック体で表示され、読者にとってこの単語が文中で重要であることを示します。

イディオマティックテキスト要素内にある場合と見た目は同じでも、意味論的な強調要素は背後でその意味と重要性を伝えています。

これらの要素はプレゼンテーション目的だけで使うべきではないことを知っておくことが重要です。テキストをイタリック体で表示したいだけで、そのテキストに段落内で特別な目的やスタイル、意味がない場合は、代わりにCSSを使うべきです。
