# --interactive--

За допомогою функції `calc()` ви можете виконувати обчислення безпосередньо у своїх таблицях стилів, щоб динамічно визначати значення властивостей. Це означає, що ви можете створювати гнучкі та адаптивні інтерфейси, обчислюючи розміри на основі розміру вікна перегляду або інших елементів.

`calc()` — це CSS-функція. Ви дізнаєтеся більше про функції, коли почнете вивчати JavaScript, але в цьому уроці ви вивчите основи, які потрібно знати, щоб зрозуміти, як працює `calc()`.

Функція — це блок коду, який виконує певне завдання. Деякі функції вже визначені в CSS, тому ви можете використовувати їх безпосередньо та передавати будь-які необхідні значення, щоб налаштувати, як саме виконуватиметься їхнє завдання.

У світі програмування, коли ми запускаємо завдання, яке виконує функція, ми кажемо, що ми "викликаємо" функцію. Значення, які ми передаємо у функцію, називаються аргументами.

Як ви бачите в коді нижче, щоб викликати функцію, ви пишете її ім’я, за яким слідують аргументи в круглих дужках, розділені комами. Між ім’ям функції та відкриваючою дужкою не повинно бути пробілу:

```css
function(argument1, argument2, argument3)
```

Функції може знадобитися лише одне значення, щоб знати, що робити. У такому випадку вона приймає лише один аргумент. Саме так працює функція `calc()`. Вона приймає один аргумент, бо їй потрібно знати, що обчислювати.

Для цього ви передаєте так званий вираз як аргумент. Вираз — це комбінація значень і операторів, яка дає результат.

Ось як ви можете викликати функцію `calc()`. Ви пишете ім’я calc, за яким ідуть круглі дужки, а всередині дужок пишете вираз:

```css
calc(expression)
```

Вираз обчислюється, щоб отримати фінальний результат. "Обчислюється" означає, що значення та оператори за лаштунками перетворюються на одне значення. Результат призначається властивості CSS, де виконується обчислення.

Ви можете виконувати обчислення зі значеннями, які представляють довжину, кут, час, відсотки, числа та кольори. Також можна комбінувати різні одиниці, як пікселі, відсотки та em.

Для чисел усі значення у виразі, які також називають операндами, мають мати відповідні одиниці, як `px`, `em` і відсотки (`%`). Залежно від оператора різні операнди можуть мати різні одиниці.

У виразі можна використовувати оператори додавання (`+`), віднімання (`-`), множення (`*`) і ділення (`/`).

Якщо є кілька операндів і операторів, `calc()` дотримується стандартного правила пріоритету операторів. Також можна додавати круглі дужки, щоб встановити порядок операцій, якщо це потрібно.

У прикладі нижче ви бачите `div` з текстом `Hello, World!`.

Використовуючи селектор типу CSS для вибору `div`, ви можете стилізувати його білим текстом і темно-синім фоном:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>Hello, World!</div>
```

```css
div {
  color: white;
  background-color: #1b1b32;
  width: calc(50% - 20px);
}
```

:::

Новинкою тут є те, що ширина обчислюється динамічно. Зверніть увагу, як ми викликаємо функцію `calc()` і передаємо вираз як аргумент. Вираз має два операнди з різними одиницями та один оператор — оператор віднімання.

Відсоток — це відносна одиниця. Значення (`50%`) визначатиметься шириною батьківського контейнера. Потім від цього значення віднімається `20px`. Результат цього виразу визначить ширину `div`.

Ширина `div` приблизно дорівнює половині загальної ширини його контейнера, і якщо ви зміните розмір батьківського контейнера, ширина буде автоматично перерахована.

Ось ключова перевага використання `calc()`. Ви можете динамічно визначати значення властивості CSS на основі різних аспектів застосунку або вікна перегляду.

У виразі також можуть бути CSS-функції та змінні, якщо вам потрібно їх використовувати в обчисленнях. Ви дізнаєтеся більше про CSS-змінні в наступних уроках.

Чудово. Тепер, коли ви знаєте основи функції `calc()`, розглянемо деякі найкращі практики.

По-перше, потрібно оточувати оператори додавання (`+`) і віднімання (`-`) пробілами.

Наприклад, вираз нижче не буде дійсним, бо оператор віднімання стоїть безпосередньо перед другим операндом.

```css
calc(100% -30px)
```

Оператор віднімання (`-`) має бути оточений пробілами, як тут. Додавання пробілів зробить вираз дійсним.

```css
calc(100% - 30px)
```

Це не обов’язково для операторів множення і ділення, але дуже рекомендується.

Ви також можете вкладати виклики функції `calc()`, якщо потрібно виконувати обчислення і використовувати їхні результати в інших обчисленнях.

Також, якщо ви використовуєте значення нуль для представлення довжини у виразі, який передаєте у функцію `calc()`, потрібно вказувати одиниці. Наприклад, цей вираз не буде дійсним:

```css
calc(100% - 0)
```

Потрібно додати одиниці, наприклад px.

```css
calc(100% - 0px)
```

Вам також слід знати, що наразі, якщо ви використовуєте оператори множення або ділення, один із операндів має бути без одиниць. Для оператора ділення, зокрема, правий операнд має бути без одиниць. Цей вираз не буде дійсним, бо обидва операнди мають одиниці (пікселі). Один із операндів, 5 або 50, має бути без одиниць:

```css
calc(5px * 50px)
```

Потрібно опустити одиниці в одному з них. Обидва ці варіанти будуть дійсними:

```css
calc(5 * 50px)
calc(5px * 50)
```

А це приклад з оператором ділення. Він не буде дійсним, бо обидва мають одиниці:

```css
calc(50% / 5%)
```

Потрібно прибрати одиницю з правого операнда, коли використовується оператор ділення:

```css
calc(50% / 5)
```

Функція `calc()` може бути дуже корисною для вас як для веб-розробника. За допомогою цієї функції ви можете динамічно задавати значення властивостей, щоб створювати гнучкі та адаптивні дизайни.
