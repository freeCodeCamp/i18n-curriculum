# --explanation--

`Logs` son registros del sistema que rastrean la actividad en una red. Por ejemplo:

`The logs recorded a failed login attempt.` - Esto significa que los registros de actividad del sistema muestran que alguien intentó iniciar sesión pero no tuvo éxito.  

`IP address` es un identificador único para un dispositivo en una red. Por ejemplo:

`The attacker used a different IP address each time.` - El atacante usó un identificador de red único para cada intento.
