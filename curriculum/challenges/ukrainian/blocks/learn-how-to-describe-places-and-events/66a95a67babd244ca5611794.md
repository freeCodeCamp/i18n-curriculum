# --explanation--

`Does` використовується для утворення питання у часі `Present Simple` для третіх осіб однини.

`Yet` часто використовується у питаннях, щоб дізнатися, чи щось очікуване вже сталося. Це означає, що дію або подію очікують, але вона ще не відбулася на цей момент. Наприклад:

- `Have you finished your homework yet?` (Ви очікуєте, що домашнє завдання скоро буде виконане, якщо ще ні).
- `Has the movie started yet?` (Ви очікуєте, що фільм скоро почнеться, якщо ще ні).
