# --description--

لتكن $R(a, b, c)$ هي أقصى مساحة تغطيها ثلاث دوائر غير متداخلة داخل مثلث بأطوال أضلاع $a$ و $b$ و $c$.

لتكن $S(n)$ هي القيمة المتوسطة لـ $R(a, b, c)$ عبر جميع الثلاثيات الصحيحة $(a, b, c)$ بحيث $1 ≤ a ≤ b ≤ c < a + b ≤ n$.

معطى لديك $S(2) = R(1, 1, 1) ≈ 0.31998$، و $S(5) ≈ 1.25899$.

احسب $S(1803)$ مقربًا إلى 5 منازل عشرية بعد الفاصلة.
