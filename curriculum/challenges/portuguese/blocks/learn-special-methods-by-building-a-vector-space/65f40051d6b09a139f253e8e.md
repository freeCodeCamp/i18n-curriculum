# --description--

Herança permite que você defina uma classe a partir de uma já existente.A nova classe, chamada child, herda todos os métodos e propriedades da classe existente, chamada parent.

```py
class Tree:
    def sprout(self):
        print('Making new leaves!')

class Oak(Tree):
    pass
    
Oak().sprout() # Output: Making new leaves!
```

No exemplo acima, a classe filha `Oak` herda de `Tree` e herda o método `sprout` da classe pai `Tree`.

Crie uma nova classe chamada `R3Vector` e siga o exemplo acima para fazê-la herdar da classe `R2Vector`.
