# --description--

表達式 `only if` 用來設定某件事情發生的條件。這就像是在說「這件事會發生，如果那件事發生」。這是一種讓一個動作依賴另一個動作的約定方式。

例如，你的朋友可能會說，`I'll go out only if it doesn't rain`。這表示他們會出門，但天氣必須好－不能下雨。
