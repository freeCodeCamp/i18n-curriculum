# --description--

`run code`라는 용어는 컴퓨터가 프로그램이나 스크립트를 실행하거나 작동하는 행위를 가리킵니다. 이 문맥에서 `run`는 프로세스를 시작하고 프로그램이 설계된 대로 숫자를 계산하거나 데이터를 정렬하는 등 무엇이든 수행할 때까지 완료하는 것을 의미합니다.

`run code without managing servers`라는 표현은 코드가 실행되는 물리적 또는 가상 서버를 직접 다루지 않고 컴퓨터 프로그램이나 애플리케이션을 실행하는 것을 뜻합니다. 이는 서버를 클라우드 제공자가 관리하여 개발자가 서버 관리보다 코딩에 더 집중할 수 있게 하는 서버리스 컴퓨팅의 핵심 요소입니다.
