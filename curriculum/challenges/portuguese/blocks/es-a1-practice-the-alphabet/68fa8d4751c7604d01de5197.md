## --text--

É verdadeiro ou falso: A letra `n` está no áudio.

## --answers--

True

### --feedback--

Uma letra muito semelhante é pronunciada no áudio, mas não é a letra `n`.

---

Falso

## --video-solution--

2

# --explanation--

Estas são as três letras pronunciadas no áudio: `g`, `m` e `t`.

A letra `m` soa muito semelhante à letra `n`, mas elas são letras diferentes em espanhol.

Portanto, a resposta é falsa porque a letra `n` não é pronunciada no áudio.
