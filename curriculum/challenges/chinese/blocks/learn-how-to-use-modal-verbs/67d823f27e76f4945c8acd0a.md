# --explanation--

`Should` 是一个情态动词，用于表达建议、推荐或最佳行动方案——但不表示义务。

- `You should update your password regularly。`－为了安全性，定期更改密码是个好主意。

- 你应该在运行代码之前查看代码中的误差。——在执行代码之前查找错误是一个好习惯。

既然 Jake 说 `we should focus on the critical systems first`，他是在给出强烈建议，但团队没有严格义务去执行。
