# --description--

완벽한 원형 피자가 $m·n$개의 동일한 조각으로 잘려 있고, 각 조각에 정확히 하나의 토핑을 올리고 싶습니다.

$m$가지 서로 다른 토핑($m ≥ 2$)이 있을 때, 각 토핑을 정확히 $n$조각에 사용하여($n ≥ 1$) 피자에 토핑을 올리는 방법의 수를 $f(m,n)$이라 합시다. 반사는 서로 다르게 간주하고, 회전은 같다고 봅니다.

예를 들어, $f(2,1) = 1$, $f(2,2) = f(3,1) = 2$, 그리고 $f(3,2) = 16$입니다. 아래는 $f(3,2)$의 예시입니다:

<img alt="animation with 16 ways to have 3 different toppings on 2 slices each" src="https://cdn.freecodecamp.org/curriculum/project-euler/pizza-toppings.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$f(m,n) ≤ {10}^{15}$인 모든 $f(m,n)$의 합을 구하세요.
