## Falando sobre Origin

Para dizer de onde você é, você deve usar: **`Soy` + `de` + [country]`**. Por exemplo:

- `Soy de El Salvador.`

- `Soy de Guatemala.`

Para saber de onde alguém é, pergunte `¿De dónde eres?`.

<br>

## Falando Sobre Nacionalidade

Para dizer sua nacionalidade, você deve usar: **`Soy` + [nationality]**.

- `Soy salvadoreña.`

- `Soy guatemalteco.`

Estas são algumas das nacionalidades que você aprendeu até agora:

| Country | Masculine Form | Feminine Form |
| :--- | :--- | :--- |
| **`El Salvador`** | `salvadoreño` | `salvadoreña` |
| **`Guatemala`** | `guatemalteco` | `guatemalteca` |
| **`Colombia`** | `colombiano` | `colombiana` |
| **`Argentina`** | `argentino` | `argentina` |
| **`Uruguay`** | `uruguayo` | `uruguaya` |
| **`Puerto Rico`** | `puertorriqueño` | `puertorriqueña` |
| **`Chile`** | `chileno` | `chilena` |
| **`Perú`** | `peruano` | `peruana` |

Em espanhol, as nacionalidades normalmente mudam com o gênero. Se uma nacionalidade termina com `-o`, essa última letra geralmente é substituída por um `-a` para criar a forma feminina.

No entanto, nacionalidades que terminam com uma consoante frequentemente adicionam um `-a` para criar a forma feminina. Por exemplo, a forma masculina de alguém de `España` ("Spain") é `español`, enquanto a forma feminina é `española`.

Nacionalidades que terminam em `-e` ou `-í` geralmente são as mesmas para ambos os gêneros. Por exemplo, a forma masculina de alguém dos `Estados Unidos` ("United States") é `estadounidense`, enquanto a forma feminina também é `estadounidense`.

<br>

## Falando Sobre Profissão

Para dizer sua profissão, você deve usar: **`Soy` + [profession]**. Por exemplo:

- `Soy representante de recursos humanos.`

- `Soy analista junior.`

Use esta pergunta para perguntar a alguém sobre sua profissão: `¿A qué te dedicas?`.

<br>

## Falando Sobre Idade

Para dizer sua idade em espanhol, você deve usar: **`Tengo` + [number] + `años`**. Por exemplo:

- `Tengo veintiún años.`

- `Tengo treinta y cinco años.`

Você pode usar esta pergunta para perguntar a idade de alguém: `¿Cuántos años tienes?`.

<br>

## Sendo Educado Durante Apresentações

Existem três expressões de cortesia importantes em espanhol.

**`Mucho gusto`** significa "Prazer em conhecê-lo". É uma forma educada de dizer que você está feliz em conhecer alguém. Esta frase é neutra em termos de gênero.

**`Encantada`** também significa "Prazer em conhecê-lo", mas é uma forma mais expressiva de dizer que você está encantado em conhecer alguém. Deve concordar com o gênero do falante. Sua forma feminina é `Encantada`, enquanto sua forma masculina é `Encantado`.

**`Igualmente`** significa "Likewise". É usado para retribuir educadamente o que a outra pessoa expressou, sem repeti-lo.
