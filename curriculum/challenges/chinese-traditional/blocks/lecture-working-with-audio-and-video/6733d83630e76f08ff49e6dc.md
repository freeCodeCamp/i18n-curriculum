# --description--

`HTMLMediaElement` API 是一個強大的工具，用來控制你頁面上音訊和影片元素的行為。它擴充了基底的 `HTMLElement` 介面，因此你可以存取基底屬性以及這些有用的方法。

你已經在先前關於 `Audio` 元素的課程中探索過一些方法，例如 `play()` 和 `pause()`。這些相同的方法也適用於影片元素。

還有一些其他有用的 `method`，例如 `addTextTrack()` `method`。這個 `method` 允許你指定一個文字軌道與媒體元素關聯，這對於為 `video` 添加字幕特別有幫助。

或者使用 `fastSeek()` 方法，該方法允許你將播放位置移動到媒體中的特定時間。

`HTMLMediaElement` API 也提供了標準元素所沒有的一些新事件的存取。

當媒體開始和停止播放時，會觸發 `play` 和 `pause` 事件。當媒體播放結束時，會觸發 `ended` 事件。

當播放因資料緩衝而自動暫停時，會觸發 `waiting` 事件。

當媒體可以部分播放或完整播放時，會觸發 `canplay` 和 `canplaythrough` 事件。

`HTMLMediaElement` 提供了對你的互動媒體的高度控制元件，而我們才剛剛觸及表面。我鼓勵你探索並嘗試這個 API，以找到將它整合到你的應用程式中的最佳方式。
