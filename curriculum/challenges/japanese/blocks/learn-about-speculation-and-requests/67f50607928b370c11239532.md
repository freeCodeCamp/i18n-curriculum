# --explanation--

`Evidence of` は何かが起きたという兆候や証拠を意味します。例えば：

`The report showed evidence of user error.` - レポートは誰かがミスをしたことを示しました。

`brute force attack` は誰かが正しいパスワードが見つかるまで多くの異なるパスワードを試すことです。

`SQL injection` は誰かが検索ボックスやフォームに悪意のあるコードを入れてシステムに侵入しようとすることです。

ジェイクはこれらの種類の攻撃の兆候を確認することについて話しています。
