# --description--

`Poder` ist ein sehr gebräuchliches Verb für höfliche Anfragen. Es bedeutet, etwas tun zu können.

Verben im Spanischen ändern ihre Form je nachdem, mit wem Sie sprechen. Wenn Sie jemanden fragen, ob er oder sie etwas tun kann, müssen Sie das Hauptverb konjugieren.

Angenommen, Sie möchten eine Person fragen, ob sie Ihnen helfen kann, könnten Sie sagen:

`¿Puedes hablar más despacio?` – Können Sie langsamer sprechen?

Beachten Sie, dass `poder` in der zweiten Person Singular (`tú`) konjugiert wird: `puedes`, während das folgende Verb im Infinitiv bleibt. Das übliche Muster lautet also:

`¿Puedes` + Infinitiv + `...?`
