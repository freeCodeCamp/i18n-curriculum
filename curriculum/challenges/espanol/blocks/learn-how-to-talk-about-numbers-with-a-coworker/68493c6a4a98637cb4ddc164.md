# --explanation--

Tom dice `in the Backup Licenses section`, lo que significa que hay un área o carpeta específica dentro del almacenamiento en la nube donde se guardan esos archivos.

Aquí, `in` se refiere a estar dentro de esa sección. Otro ejemplo:

`The answer is in the FAQ section.` — Puedes encontrar la respuesta en la parte de FAQ.
