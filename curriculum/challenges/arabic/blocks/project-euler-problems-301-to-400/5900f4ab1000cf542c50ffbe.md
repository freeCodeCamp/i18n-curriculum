# --description--

لتكن $x_1, x_2, \ldots, x_n$ متتالية طولها $n$ بحيث:

- $x_1 = 2$
- لكل $1 &lt; i ≤ n : x_{i - 1} &lt; x_i$
- لكل $i$ و $j$ حيث $1 ≤ i, j ≤ n : {(x_i)}^j &lt; {(x_j + 1)}^i$

هناك خمس متتاليات فقط من هذا النوع طولها 2، وهي: {2,4}، {2,5}، {2,6}، {2,7} و{2,8}. هناك 293 متتالية من هذا النوع طولها 5؛ ثلاثة أمثلة منها هي: {2,5,11,25,55}، {2,6,14,36,88}، {2,8,22,64,181}.

لتكن $t(n)$ تمثل عدد هذه المتتاليات بطول $n$. معطى أن $t(10) = 86195$ و $t(20) = 5227991891$.

احسب $t({10}^{10})$ وقدم إجابتك بباقي القسمة على $10^9$.
