# --interactive--

Floats katika CSS ni mbinu iliyoundwa awali kuruhusu maandishi kuzunguka kipengele, kama picha. Hata hivyo, kwa muda, wasanidi programu waligundua njia mpya za kutumia floats, wakitumia kwa ubunifu katika muundo wa mpangilio. Ingawa mbinu za kisasa kama Flexbox na Grid sasa zinatumika zaidi, kuelewa floats bado ni muhimu, hasa unapotumia msimbo wa zamani au unahitaji kufanikisha athari maalum za mpangilio.

Unapofanya kipengele kiwe floated, huondolewa katika mtiririko wa kawaida wa hati na kusukumwa kushoto au kulia ndani ya kontena yake. Yaliyomo yanayofuata yatazunguka kipengele kilichofloat, yakijaza nafasi iliyobaki. Matumizi ya kawaida ni kuzunguka maandishi karibu na picha, ambapo picha huwekwa upande mmoja na maandishi kuzunguka. Mbinu hii bado inatumika sana, hasa katika makala na blogu ambapo picha zinahitaji kuwekwa kando na maandishi. Hapa kuna mfano wa msimbo:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <img src="https://placehold.co/150x150" alt="Placeholder Image">
</div>
```

```css
.container {
  border: 1px solid black;
  padding: 10px;
}

img {
  float: left;
  margin-right: 20px;
}
```

:::

Katika mfano huu, kontena haizunguki picha iliyofloat. Picha iko nje ya mtiririko wa kawaida wa hati, na kontena limeanguka hadi urefu wa sifuri kwa sababu halioni vipengele vilivyofloat vilivyo ndani yake.

Floats pia zilikuwa maarufu kwa kuunda mpangilio wa safu nyingi kabla ya Flexbox na Grid kusambaa. Kwa kuweka vipengele vikiwa kando kando, wasanidi programu waliweza kuunda safu zilizo sawa kwa usawa wa usawa. Hata hivyo, unapotumia floats, ni muhimu kushughulikia tatizo la kuanguka kwa vipengele vya mzazi wakati vipengele vya watoto vimefloat. Suluhisho la mbinu ya clearfix linatumika kwa kipengele cha darasa la `container` kurekebisha tatizo hili.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <img src="https://placehold.co/150x150" alt="Placeholder Image">
  <p> This is an example of text flowing around a floated image.</p>
</div>
```

```css
.container {
  border: 1px solid black;
}

/* Clearfix CSS */
.container::after {
  content: "";  
  display: block;
  clear: both;
}

img {
  float: left;
  margin-right: 20px;
}
```

:::

- `::after` ni kipengele cha bandia kinachoongeza kipande kisichoonekana baada ya yaliyomo ya kontena.
- `content: ""` huhakikisha kipengele cha bandia kiko lakini hakionyeshi maudhui yoyote.
- `display: block` hufanya kipengele cha bandia kuwa kipengele cha ngazi ya block.
- `clear: both` huhakikisha kipengele cha bandia hufuta upande zote mbili za vipengele vyovyote vilivyofloat juu yake.

Katika mfano huu, tumeongeza kipengele kipya cha aya ili kuonyesha kuanguka kwa kontena kwa uwazi zaidi. Kwa sababu aya bado iko katika mtiririko wa kawaida wa hati, kontena hupanuka kiasi cha kuizunguka. Kisha, tunatumia mbinu ya clearfix kurekebisha kuanguka na kufanya mpaka la kontena lionyeshwe ipasavyo. Unaweza kujaribu kufuta sheria ya clearfix kuona kontena likianguka tena.

Mbinu ya clearfix huhakikisha kipengele mzazi kinazunguka vipengele vyake vilivyofloat ipasavyo. Clearfix inalazimisha kontena mzazi "kuona" vipengele vilivyofloat kwa kuongeza sifa ya clear baada ya yaliyomo yaliyofloat.

Ingawa floats si mbinu ya kawaida tena kwa mipangilio tata kutokana na mbinu za kisasa kama Flexbox na Grid, bado zina jukumu muhimu katika hali fulani. Iwe unazunguka maandishi karibu na picha au unafanya kazi kwenye miradi ya zamani, kuelewa jinsi floats zinavyofanya kazi na jinsi ya kuzifuta ipasavyo ni muhimu kwa kudumisha mipangilio iliyopangwa vizuri na inayojibadilisha kulingana na kifaa.
