# --description--

`function` は特定のタスクを実行する命令の集合です。`Functions` はコードをより整理され、再利用可能で読みやすくします。例えば、`function` は2つの数字の合計を計算したり、ユーザーにメッセージを表示したりするかもしれません。

`function` を工場の小さな機械のように考えてください。何らかの入力を与えると、その入力に対して何か処理を行い、出力を返します。
