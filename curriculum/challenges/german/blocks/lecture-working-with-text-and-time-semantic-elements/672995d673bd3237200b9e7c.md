# --interactive--

Eine Abkürzung ist eine verkürzte Form eines Wortes oder Ausdrucks. Zum Beispiel ist „Dr“ gefolgt von einem Punkt eine Abkürzung für das Wort „Doktor“.

Es gibt zwei gebräuchliche Formen von Abkürzungen: Akronyme und Initialismen.

Ein Akronym ist ein Wort, das aus den Anfangsbuchstaben eines Ausdrucks gebildet wird, wobei jeder Buchstabe den ersten Buchstaben eines Wortes in diesem Ausdruck darstellt.

GUI ist ein Beispiel für ein Akronym. Es steht für Graphical User Interface. Indem man die Anfangsbuchstaben jedes Wortes G, U und I nimmt, erhält man das Akronym GUI.

Ein Initialismus wird aus den Anfangsbuchstaben eines Ausdrucks gebildet, wobei jeder Buchstabe den ersten Buchstaben eines Wortes in diesem Ausdruck darstellt.

Zum Beispiel ist HTML ein Initialismus; es steht für HyperText Markup Language und wird buchstabiert ausgesprochen: H, T, M, L.

Sowohl Akronyme als auch Initialismen sind Arten von Abkürzungen. Der Unterschied besteht darin, dass Akronyme als Wörter ausgesprochen werden, während Initialismen als einzelne Buchstaben ausgesprochen werden.

Sie sind sehr hilfreich, um Texte prägnanter zu schreiben, besonders wenn sie in einem bestimmten Kontext gut bekannt und leicht verständlich sind.

Wenn Sie Abkürzungen wie Akronyme oder Initialismen in HTML anzeigen möchten, ist das `abbreviation`-Element genau das Richtige für Sie. Sie sollten deren vollständige Bedeutung immer erklären, wenn Sie sie zum ersten Mal verwenden. Dann können Sie das `abbreviation`-Element nutzen, um sie hervorzuheben und weitere Details bereitzustellen.

Hier ist ein Beispiel, in dem Sie einen Absatz mit dem Satz `HTML is the foundation of the web` sehen können:

:::interactive_editor

```html
<p><abbr>HTML</abbr> is the foundation of the web.</p>
```

:::

Der Initialismus HTML befindet sich innerhalb eines `abbreviation`-Elements. Im Browser werden Sie sehen, dass sich optisch nichts geändert hat. Es sieht weiterhin wie normaler Text aus. Das `abbreviation`-Element liefert hilfreichen Kontext im Hintergrund, aber Benutzer sehen den Initialismus weiterhin als normalen Text.

Wenn Sie den Benutzern helfen möchten, zu verstehen, was dieser Initialismus bedeutet, können Sie seine vollständige Form mit dem `title`-Attribut anzeigen.

Das `title`-Attribut ist optional, aber wenn Sie es hinzufügen, muss es eine für Menschen lesbare Beschreibung der Abkürzung, des Akronyms oder Initialismus sein.

Nehmen wir dasselbe Beispiel wie zuvor, fügen aber das `title`-Attribut hinzu. Es wird `HyperText Markup Language` sein, die ausgeschriebene Form des Initialismus:

:::interactive_editor

```html
<p><abbr title="HyperText Markup Language">HTML</abbr> is the foundation of the web.</p>
```

:::

Normalerweise ändert sich der Stil des `abbreviation`-Elements, wenn Sie dieses Attribut hinzufügen. Der genaue Stil hängt vom Browser ab. Einige Browser zeigen eine gepunktete Unterstreichung an, während andere den Inhalt in Kapitälchen umwandeln. Wenn der Benutzer mit der Maus über die Abkürzung fährt, wird die vollständige Form als Tooltip angezeigt.

Sie müssen das `abbreviation`-Element nicht unbedingt für jede Abkürzung auf Ihrer Webseite verwenden, aber es wird für solche empfohlen, die unklar sein könnten oder zusätzliche Kontextinformationen benötigen.

Verwenden Sie Ihr bestes Urteilsvermögen, um die richtige Balance zwischen Information und Darstellung zu finden, damit der Text nicht überladen wirkt und dennoch klar und prägnant bleibt.
