# --description--

プログラミングにおける `statement` は、特定の動作を実行するコードの一行です。例えば、`print("Hello World")` はテキストを表示する文です。

`condition` は、何かが真か偽かをチェックするコード内のテストです。たとえば、`if x > 5` は `x` が5より大きいかどうかを確認します。

`conditional` は一般的に、`condition` または `conditions` に依存するものを指す形容詞です。プログラミングでは、論理的な `conditions` を含む任意の式や機能を説明するために使われます。これは明示的な「if-then」構造に限定されず、`conditions` に基づいて動作を変える設定全般を指し、ループや三項演算子も含まれます。

`conditional statement` は、特定の `condition` が真の場合にのみ実行される文です。たとえば、`if x > 5: print("Large")` は `conditional statement` です。`Large` が5より大きい場合にのみ `x` を出力します。
