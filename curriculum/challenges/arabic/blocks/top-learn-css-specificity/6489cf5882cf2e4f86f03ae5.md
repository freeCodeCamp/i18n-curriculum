# --description--

الآن، دعنا نغيّر الأمور قليلاً:

```html
<!-- index.html -->

<div class="main">
  <div class="list" id="subsection"></div>
</div>
```

```css
/* rule 1 */
#subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

في المثال أعلاه، رغم أن القاعدة 2 تحتوي على محددات صنف أكثر من محددات معرف، إلا أن القاعدة 1 أكثر تحديدًا لأن المعرف يتفوق على الصنف.
