# --description--

`Object-oriented programming` (OOP) — це парадигма програмування, заснована на концепції `objects`, які можуть містити дані та код: дані у вигляді полів (часто відомих як атрибути або пропи) та код у вигляді процедур (часто відомих як методи).

Загальним прикладом `OOP` є створення класу в мові програмування, як-от Java або Python. Цей клас визначає структуру об’єктів, включно з їхніми атрибутами та методами. Наприклад, клас `Car` може містити атрибути, такі як `color` і `model`, а також методи, як `drive()` і `brake()`.

Термін `oriented` у цьому контексті означає спрямований або орієнтований на певну мету чи принцип.

Наприклад, навчальна програма freeCodeCamp з адаптивного вебдизайну описується як `project-oriented`, бо вона структурована навколо виконання проєктів, що означає, що метод навчання зосереджений на практичному застосуванні, а не лише на теоретичних знаннях.
