# --description--

字串和数字可能是我们的创建块，但随着你的脚本变得更加复杂，你将需要一种处理大量它们的方法。幸运的是，JavaScript 有几种专门用于此的数据类型。`array` 是一个有序的项集合（字串、数字或其他东西）。

数组是一种在单个变量中保存多值的方式。它们是一种特殊类型的对象，具有一个 length 属性和一系列编号属性。每个编号属性称为一个元素，每个元素可以保存任意类型的值。

数组的一个例子是：

```javascript
const fruits = ['apple', 'banana', 'orange'];
```
