# --description--

`join` メソッドは、配列の要素を結合して文字列を作成するために使われます。文字列内で配列の要素を区切るための区切り文字を引数として受け取ります。

例を示します：

```js
const arr = ["Hello", "World"];
const str = arr.join(" ");
```

`str` は文字列 `Hello World` の値を持ちます。
# --instructions--

`join` メソッド（その他のメソッドも可）を `sentensify` 関数の中で使い、文字列 `str` の単語から文を作ってください。関数は文字列を返す必要があります。例えば、`I-like-Star-Wars` は `I like Star Wars` に変換されます。このチャレンジでは、`replace` メソッドは使わないでください。
