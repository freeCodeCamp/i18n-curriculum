# --description--

ブランマンジュ曲線は、$0 ≤ x ≤ 1$ かつ $\displaystyle y = \sum_{n = 0}^{\infty} \frac{s(2^nx)}{2^n}$ を満たす点 ($x$,$y$) の集合で、ここで $s(x)$ は $x$ から最も近い整数までの距離です。

ブランマンジュ曲線の下のエリアは $\frac{1}{2}$ に等しく、下の図のピンク色で示されています。

<img alt="diagram of blancmange curve, with circle C shown on diagram" src="https://cdn.freecodecamp.org/curriculum/project-euler/a-scoop-of-blancmange.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$C$ を中心が ($\frac{1}{4}$,$\frac{1}{2}$)、半径が $\frac{1}{4}$ の円とし、図では黒で示されています。

$C$ に囲まれたブランマンジュ曲線の下のエリアはいくらですか？答えは小数点以下8桁まで、0.abcdefgh の形式で答えてください。
