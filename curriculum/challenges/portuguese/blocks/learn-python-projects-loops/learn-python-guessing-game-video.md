## --text--

O que significa a condição `guess != secret_word` neste código?

```python
secret_word = "giraffe"
guess = ""

while guess != secret_word:
    guess = input("Enter guess: ")
```

## --answers--

Se `guess` não for igual a `secret_word`.

---

Se `guess` for igual a `secret_word`.

---

Se `guess` for maior que `secret_word`.

---

Se `guess` for menor que `secret_word`.
