# --description--

`Encounter`, especialmente em sua forma passada `encountered`, significa encontrar algo, geralmente de forma inesperada.Por exemplo, `I encountered an error while testing the software.` Isso implica encontrar algo sem procurá-lo.

`Pinpoint` significa identificar algo com muita precisão.Por exemplo, `After reviewing the code, I was able to pinpoint the error.` Isso significa encontrar exatamente onde e qual é o problema.

Em programação, encontrar problemas muitas vezes pode ser sobre `encountering` problemas inesperados e tentar `pinpoint` suas causas.
