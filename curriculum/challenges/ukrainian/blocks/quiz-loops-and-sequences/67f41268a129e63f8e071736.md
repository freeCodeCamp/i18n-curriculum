#### --text--

Який із наведених варіантів є правильним синтаксисом для списку?

#### --distractors--

```py
cities = //'Los Angeles', 'London', 'Tokyo'\\
```

---

```py
cities = {'Los Angeles', 'London', 'Tokyo'}
```

---

```py
cities = <'Los Angeles', 'London', 'Tokyo'>
```

#### --answer--

```py
cities = ['Los Angeles', 'London', 'Tokyo']
```
