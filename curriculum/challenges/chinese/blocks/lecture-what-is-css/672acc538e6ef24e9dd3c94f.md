# --interactive--

边距和内边距是 CSS 中创建结构良好、易读且视觉上吸引人的网页的基本属性。

边距控件元素外部的空间，有助于将其与其他元素分开并定义版型结构，而内边距控件元素内部的空间，提升内容的可读性和美观度。

为了更好地理解 `margin` 和 `padding` 之间的区别，让我们来看一些示例。

这是页面上三个段落元素的超文本标记语言示例：

:::interactive_editor

```html
<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

:::

要为每个段落元素的顶部应用间距，你可以像这样使用 `margin-top` 属性：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  margin-top: 20px;
  border: 2px solid black;
}
```

:::

在此示例中，我们仅为每个段落元素的顶部应用 `20px` 的外边距。

我们在四个边都加了实心黑色边框，这样你可以更清楚地看到边距。

四个不同的 `margin` 属性是 `margin-top`、`margin-right`、`margin-bottom` 和 `margin-left`。

下面是使用所有四个属性的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 30px;
  margin-left: 40px;
  border: 2px solid black;
}
```

:::

在此示例中，我们为段落元素的四个边都赋值了间距。还添加了一个实心黑色边框，以便你更好地看到边距。

使用 `margin` 属性的另一种方法是使用简写表示法。你可以一次指定一、二、三或四个值。让我们一起探索这些选项。

当在 `margin` 简写中使用单个值时，该值将应用于目标元素的四个边。

下面是使用单个值作为 `margin` 简写的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px;
}
```

:::

此代码示例将为段落元素的四个边均匀应用 `10px` 的 `margin`。

当使用两个值时，第一个值应用于元素的 `top` 和 `bottom`，而第二个值应用于元素的 `left` 和 `right` 侧。

下面是使用两个值作为 `margin` 简写的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px;
}
```

:::

这将段落元素的 `top` 和 `bottom` 边距设置为 `10px`，`left` 和 `right` 边距设置为 `20px`。

如果提供了三个值，第一个值应用于 `top` 边距，第二个值应用于 `left` 和 `right` 边距，第三个值应用于 `bottom` 边距。

这是一个更好理解的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px;
}
```

:::

这将 `top` 设置为 `10px` 的边距，`left` 和 `right` 设置为 `20px`，`bottom` 设置为 `30px`。

当使用四个值时，这为你提供了更多控件，因为你可以为目标元素的每一侧独立指定每个边距值。

第一个值目标是 `top`，第二个值目标是 `right`，第三个值目标是 `bottom`，第四个值目标是 `left`。

这是使用四个值的 margin 简写的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px 40px;
}
```

:::

这将 `top` 设置为 `10px`，`right` 设置为 `20px`，`bottom` 设置为 `30px`，`left` 设置为 `40px`。

`padding` 属性用于在元素内部、内容与其边框之间应用空间。

像 `margin` 属性一样，四个 `padding` 属性是 `padding-top`、`padding-right`、`padding-bottom` 和 `padding-left`。

以下是如何为段落元素设置 `padding` 的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 30px;
  padding-left: 40px;
  border: 2px solid black;
}
```

:::

这将 `top` 设置为 `10px` 的内边距，`right` 设置为 `20px`，`bottom` 设置为 `30px`，`left` 设置为 `40px`。

正如你所见，`padding` 应用于边框内的内容，不同于应用于边框外的 `margin`。

就像 `margin` 属性一样，你也可以选择使用 `padding` 属性的简写。

你也可以在 `padding` 简写属性上指定一、二、三或四个值。

下面是之前段落元素使用 `padding` 简写的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding: 10px 20px 30px 40px;
  border: 2px solid black;
}
```

:::

在示例中，使用简写代码将为段落元素的 `top` 设置 `10px` 的内边距，为 `right` 设置 `20px`，为 `bottom` 设置 `30px`，为 `left` 设置 `40px`。
