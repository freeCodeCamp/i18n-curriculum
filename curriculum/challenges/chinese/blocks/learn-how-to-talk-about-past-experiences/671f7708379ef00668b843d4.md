# --explanation--

`Would` 在此上下文中表示过去的习惯性动作或重复行为。它通常用于描述某人经常执行的动作，类似于 `used to`。 

然而，`would` 更侧重于动作本身，而 `used to` 可以描述状态或习惯。例如：

- `She would always read a book before going to bed.` —— 睡觉前阅读一本书是她的常见行为。

`She used to live in New York before moving here.` —— 这里使用 `used to` 是因为居住在某地表示一种状态，而不是一个动作或你习惯性做的事情。

在这个句子中，Maria 使用 `would` 来描述 Brian 反复熬夜微调代码的习惯，这强调了他的奉献精神。
