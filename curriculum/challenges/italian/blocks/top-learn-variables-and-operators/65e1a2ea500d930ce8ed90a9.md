# --description--

Conosciamo le funzionalità degli operatori JavaScript che vanno oltre l'aritmetica scolastica.

Di solito, l'operatore più `+` somma i numeri.

Ma, se l'operatore binario `+` viene applicato a stringhe, le unisce (concatena):

```js
let s = "my" + "string";
console.log(s); // mystring
```

Nota che se uno degli operandi è una stringa, anche l'altro viene convertito in stringa.

Per esempio:

```js
console.log(1 + '2'); // '12'
console.log('1' + 2); // '12'
```

Vedi, non importa se il primo operando è una stringa o il secondo.

Ecco un esempio più complesso:

```js
console.log(2 + 2 + '1') // "41" and not "221"
```

Qui, gli operatori agiscono uno dopo l'altro. Il primo `+` somma due numeri, quindi restituisce `4`, poi il successivo `+` aggiunge la stringa `1`, quindi è come `4 + '1' = '41'`.

```js
console.log('1' + 2 + 2) // "122" and not "14"
```

Qui, il primo `+` concatena la stringa e il numero, quindi restituisce `'12'`, poi il successivo `+` aggiunge `2`, quindi è come `'12' + 2 = '122'`.

L'operatore binario `+` è l'unico che supporta le stringhe in questo modo. Gli altri operatori aritmetici funzionano solo con numeri e convertono sempre i loro operandi in numeri.

Ecco la demo per sottrazione e divisione:

```js
console.log( 6 - '2' ); // 4, converts '2' to a number
console.log( '6' / '2' ); // 3, converts both operands to numbers
```
