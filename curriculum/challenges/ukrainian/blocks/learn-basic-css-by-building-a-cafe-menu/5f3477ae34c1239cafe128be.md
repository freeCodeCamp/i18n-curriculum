# --description--

Тепер у вас є три селектори типу з однаковим стилем. Ви можете додати одну й ту саму групу стилів до багатьох елементів, створивши список селекторів. Кожен селектор розділяється комами ось так:

```css
selector1, selector2 {
  property: value;
}
```

Видаліть три існуючі селектори типу і замініть їх одним списком селекторів, який вирівнює текст по центру для елементів `h1`, `h2` і `p`.

# --hints--

Ви повинні використати один селектор типу для всіх трьох елементів, `h1, h2, p`. Обов’язково дотримуйтеся цього порядку.

```js
const hasSelector = new __helpers.CSSHelp(document).getStyle('h1, h2, p');
assert(hasSelector);
```

У вашому елементі `style` має бути лише один селектор.

```js
const selectors = new __helpers.CSSHelp(document).getCSSRules().filter(x => x.selectorText)
assert(selectors.length === 1);
```
