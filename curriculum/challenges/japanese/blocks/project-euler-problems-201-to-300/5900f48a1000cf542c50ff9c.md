# --description--

アルバートは正の整数 $k$ を選びます。次に、区間 [0,1] から一様分布で実数 $a$、$b$ がランダムに選ばれます。

その後、${(ka + 1)}^2 + {(kb + 1)}^2$ の平方根が計算され、小数点以下を四捨五入して最も近い整数にします。結果が $k$ と等しければ、$k$ 点を獲得します。そうでなければ得点はありません。

例えば、$k = 6$、$a = 0.2$、$b = 0.85$ の場合、${(ka + 1)}^2 + {(kb + 1)}^2 = 42.05$ となります。42.05 の平方根は 6.484... で、四捨五入すると 6 になります。これは $k$ と等しいので、6 点を獲得します。

$k = 1, k = 2, \ldots, k = 10$ の 10 回のターンをプレイした場合、合計得点の期待値は小数点以下5桁で 10.20914 であることが示されています。

$k = 1, k = 2, k = 3, \ldots, k = {10}^5$ の ${10}^5$ 回のターンをプレイした場合、合計得点の期待値は小数点以下5桁でいくつになりますか？
