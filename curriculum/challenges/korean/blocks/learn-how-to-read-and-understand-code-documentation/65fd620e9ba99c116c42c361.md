# --description--

개발자는 작업 흐름 내에서 작업의 시점을 나타내기 위해 `once`를 자주 사용합니다. 이는 `as soon as` 또는 `when`를 의미합니다.

예를 들어, `Once you receive the client's feedback, update the project documentation.` 이는 문서 업데이트가 피드백을 받은 직후에 시작되어야 하며, 프로젝트 수명 주기에서 작업의 순서를 보여줍니다.
