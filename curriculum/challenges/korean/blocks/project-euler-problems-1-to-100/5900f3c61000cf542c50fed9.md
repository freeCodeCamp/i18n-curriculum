# --description--

한 정육면체의 여섯 면에는 서로 다른 숫자(0부터 9까지)가 하나씩 적혀 있습니다. 두 번째 정육면체에도 같은 방식으로 숫자가 적혀 있습니다. 두 정육면체를 나란히 놓고 위치를 바꾸어 다양한 두 자리 숫자를 만들 수 있습니다.

예를 들어, 제곱수 64를 만들 수 있습니다:

<img alt="two cubes, one with the number 6 and the other with number 4" src="https://cdn-media-1.freecodecamp.org/project-euler/cube-digit-pairs.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

사실, 두 정육면체에 적힌 숫자를 신중하게 선택하면 100 미만의 모든 제곱수인 01, 04, 09, 16, 25, 36, 49, 64, 81을 모두 표시할 수 있습니다.

예를 들어, 한 정육면체에 {0, 5, 6, 7, 8, 9}를, 다른 정육면체에 {1, 2, 3, 4, 8, 9}를 배치하는 방법이 있습니다.

하지만 이 문제에서는 6과 9를 뒤집어서 사용할 수 있도록 허용합니다. 따라서 {0, 5, 6, 7, 8, 9}와 {1, 2, 3, 4, 6, 7}의 조합으로도 모든 아홉 개 제곱수를 표시할 수 있습니다. 그렇지 않으면 09를 만들 수 없기 때문입니다.

서로 다른 배치를 결정할 때는 각 정육면체에 적힌 숫자만 고려하며, 순서는 중요하지 않습니다.

<div style="margin-left: 4em;">
  {1, 2, 3, 4, 5, 6}는 {3, 6, 4, 1, 2, 5}와 같습니다.<br>
  {1, 2, 3, 4, 5, 6}는 {1, 2, 3, 4, 5, 9}와 다릅니다.
</div>

하지만 6과 9를 뒤집을 수 있으므로, 마지막 예시의 두 서로 다른 집합은 모두 2자리 숫자를 만들 때 확장된 집합 {1, 2, 3, 4, 5, 6, 9}를 나타냅니다.

두 정육면체의 서로 다른 배치 중에서 모든 제곱수를 표시할 수 있는 경우는 몇 가지일까요?
