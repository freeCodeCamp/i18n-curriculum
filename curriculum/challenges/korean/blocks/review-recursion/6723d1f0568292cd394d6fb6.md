# --description--

- 재귀는 기본 사례에 도달할 때까지 함수를 반복해서 호출할 수 있게 하는 프로그래밍 개념입니다.

다음은 숫자의 팩토리얼을 계산하는 재귀 함수 예시입니다:

```js
function findFactorial(n) {
  if (n === 0) {
    return 1;
  }
  return n * findFactorial(n - 1);
}
```

위 예시에서 `findFactorial` 함수는 `n`이 `0`에 도달할 때까지 재귀적으로 호출됩니다. `n`이 `0`일 때 기본 사례에 도달하여 함수가 `1`를 반환합니다. 그 후 함수는 `n`과 `findFactorial(n - 1)`에 대한 재귀 호출 결과의 곱을 반환합니다.

- 재귀는 깊이가 알 수 없는 중첩 객체/배열이나 파일 트리 같은 구조를 처리할 때 유용합니다.
- 재귀 함수에서는 호출 스택이 함수 호출을 추적하는 데 사용됩니다. 함수가 호출될 때마다 호출 스택에 추가되고, 기본 사례에 도달하면 호출들이 스택에서 제거됩니다.
- 기본 사례를 신중하게 정의해야 합니다. 무한히 호출하면 재귀가 계속 쌓여 시스템 메모리가 부족해져 코드가 중단될 수 있습니다.
- 재귀는 팩토리얼과 피보나치 같은 수학 문제 해결, 트리와 그래프 탐색, 순열과 조합 생성 등 다양한 분야에서 활용됩니다.
