# --description--

コンピューターサイエンスにおける分割統治法は、問題を再帰的により小さな部分問題に分解していく手法です。この手法の重要なポイントの一つは再帰であり、これは関数が基本ケースに達するまで自分自身を繰り返し呼び出すことを指します。このレッスンでは、マージソートアルゴリズムを見て、分割統治法がどのように機能するかを理解していきます。

次の数字のリストがあるとしましょう：

```md
42 37 53 17
```

このリストをマージソートアルゴリズムを使って、小さい順に並べ替えることが目標です。最初のステップはリストを半分に分割することです：

```md
42 37 | 53 17
```

次にリストの左側を見てみます：

```md
42 37
```

そのサブリストをさらに半分に分割し、各サブリストが1つの要素だけになるまで繰り返します：

```md
42 | 37
```

1つの要素だけのリストはデフォルトではソート済みです。次に、それらの1要素のサブリストをマージしてソートされたリストにします：

```md
37 42
```

同じプロセスを元のリストの右側にも適用します：

```py
# right side of original list
53 17

# divide the list in half
53 | 17

# merge the lists in sorted order
17 53
```

元のリストの両方の半分がソートされたので、それら2つの半分をマージして要素をソートします：

```markdown
17 37 42 53
```

アルゴリズムのコードは次のようになります：

```py
def merge_sort(arr):
    if len(arr) <= 1:
        return arr

    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])

    sorted_list = []
    i = 0
    j = 0

    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            sorted_list.append(left[i])
            i += 1
        else:
            sorted_list.append(right[j])
            j += 1

    sorted_list.extend(left[i:])
    sorted_list.extend(right[j:])

    return sorted_list
```

マージソートの時間計算量は`O(n log n)`です。なぜならリストが継続的に半分に分割され（`log n`）、その後マージされるからです（`O(n)`）。バブルソートのような他のソートアルゴリズムとは異なり、マージソートはインプレースでソートされず、空間計算量は`O(n)`です。
