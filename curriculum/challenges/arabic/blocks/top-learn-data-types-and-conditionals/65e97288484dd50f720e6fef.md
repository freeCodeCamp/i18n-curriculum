# --description--

قد تتضمن جمل الشرط الأكثر تعقيدًا شروطًا متعددة. لهذا السبب تُستخدم المعاملات المنطقية لدمج شروط متعددة. المعاملات المنطقية هي `&&` و `||` و `!` التي تُستخدم لتمثيل `and` و `or` و `not` على التوالي.

يُستخدم المعامل المنطقي `||` لدمج شرطين من نوع قيمة منطقية. يُرجع `true` إذا كان أحد الشرطين على الأقل `true`. وإلا فإنه يُرجع `false`:

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a > b || a > c) {
  console.log("At least one of the conditions is true");
} else {
  console.log("Both of the conditions are false");
}
```

يُستخدم المعامل المنطقي `&&` لدمج شرطين من نوع قيمة منطقية. يُرجع `true` فقط إذا كان كلا الشرطين `true`. وإلا فإنه يُرجع `false`:

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a < b && a < c) {
  console.log("Both of the conditions are true");
} else {
  console.log("At least one of the conditions is false");
}
```

يُستخدم المعامل المنطقي `!` لنفي شرط من نوع قيمة منطقية. يُرجع `true` إذا كان الشرط `false`. وإلا فإنه يُرجع `false`:

```javascript
let a = 5;
let b = 10;

if (!(a > b)) {
  console.log("The condition is false");
} else {
  console.log("The condition is true");
}
```

في المثال أعلاه، تُستخدم زوج إضافي من الأقواس العادية لجعل الشفرة أكثر قابلية للقراءة. لا يلزم استخدامها.

## --text--

مهمتك كتابة جملة `if` الشرطية التي تتحقق من الشروط التالية في تطبيق ويب لعرض `Welcome!` على المستخدم:

1. يجب أن يكون لدى المستخدم إما حساب بريميوم (`isPremium`) أو أن يكون عضوًا لأكثر من عام (`membershipDuration` > 12 شهرًا).

1. يجب ألا يكون المستخدم محظورًا حاليًا من الخدمة (`!isBlocked`).

أي جملة if الشرطية تتحقق من هذه الشروط بشكل صحيح؟

## --answers--

```javascript
if (isPremium && membershipDuration > 12 && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (isPremium || (membershipDuration > 12 && !isBlocked)) {
  console.log("Welcome!");
}
```

---

```javascript
if ((isPremium || membershipDuration > 12) && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (!isPremium || membershipDuration <= 12 || isBlocked) {
  console.log("Welcome!");
}
```
