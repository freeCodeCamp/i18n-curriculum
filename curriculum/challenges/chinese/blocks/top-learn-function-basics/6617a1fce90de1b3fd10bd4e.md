# --description--

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

通过将 `animal` 放入 `favoriteAnimal()` 函数的括号内，你是在告诉 JavaScript 你将向你的 `favoriteAnimal` 函数发送某个值。这意味着 animal 只是某个未来值的占位符。但你发送的是什么值？

最后一行，`favoriteAnimal('Goat')`，是你调用 `favoriteAnimal` 函数并传递值 `'Goat'` 到该函数中。这里，`'Goat'` 是你的参数。你告诉 `favoriteAnimal` 函数，“请将 `'Goat'` 发送到 `favoriteAnimal` 函数，并在 `'animal'` 占位符处使用 `'Goat'`。”由于使用参数提供的灵活性，你可以声明任何动物为你最喜欢的。

这里有一个图表，帮助你形象化参数如何传递给函数，以及值如何从函数中返回。

<img src="https://cdn.statically.io/gh/TheOdinProject/curriculum/c53dd9a12f0c9afde0d9229f82a176170f12e120/foundations/javascript_basics/function_basics/imgs/00.png" alt="A description of a function with arrows explaining that the values between the parentheses of the function itself are called parameters, and the values which are issued when the function is used are called arguments" style="width:95%;height:auto;">

请注意，通过在 `console.log()` 内调用带有参数 `'Goat'` 的 `favoriteAnimal()`，你会得到该函数的返回值，即字串 `"Goat is my favorite animal!"`，并打印到控制台。你将函数调用 `favoriteAnimal('Goat')` 作为参数传递给另一个函数调用——`log()`。

## --answers--

`"Goat is my favorite animal!"`
