# --explanation--

`That's a good idea` é uma forma natural de responder a uma pergunta `Have you thought about...`.Ele reconhece a sugestão e mostra que o falante a considera uma ideia útil.Por exemplo:

- Pessoa 1: `Have you thought about using a different strategy?` - Esta pessoa está sugerindo uma abordagem diferente para um problema.

- Pessoa 2: `That's a good idea! We should try it.` - Esta pessoa reconhece a sugestão e quer fazer o que a outra pessoa pediu.
