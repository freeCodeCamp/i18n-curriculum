# --description--

数量に関する質問と回答をする際は、`much` と `many` を一貫して使うことが重要です。`Much` は不可算名詞に使い、`many` は可算名詞に使います。会話では、質問と回答で `much` または `many` のどちらかを使って一致させる必要があります。

例：

質問: `How much time do you have?` 回答: `Not much.`

質問: `How many books do you own?` 回答: `Not many.`
