# --description--

Gli sviluppatori spesso usano `once` per indicare la tempistica delle attività all'interno del loro flusso di lavoro. Significa `as soon as` o `when`.

Per esempio, `Once you receive the client's feedback, update the project documentation.` Questo significa che l'aggiornamento della documentazione dovrebbe iniziare subito dopo aver ricevuto il feedback, mostrando la sequenza delle azioni nel ciclo di vita di un progetto.
