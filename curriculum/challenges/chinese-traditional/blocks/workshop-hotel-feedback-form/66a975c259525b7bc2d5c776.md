# --description--

你的飯店回授表單也應該讓使用者能夠評分餐點。

首先為你的表單添加一個 `label` 元素，該元素的文字為 `How was the food?`。該 `label` 元素應該有一個 `for` 屬性，設置為 `"food"`。

在你的 `label` 元素下方，添加一個 `select` 元素，其 `id` 和 `name` 設為 `"food"`。

# --hints--

你應該有一個 `label` 元素，其 `for` 屬性設為 `"food"`。

```js
assert.exists(document.querySelector('label[for="food"]'));
```

你的 `label` 應該有文字 `How was the food?`。

```js
assert.strictEqual(document.querySelector('label[for="food"]')?.textContent.trim(), 'How was the food?');
```

你應該在你的 `label` 下方有一個 `select` 元素，其 `id` 設為 `"food"`。

```js
assert.exists(document.querySelector('label + select#food'));
```

你的 `select` 元素應該將 `name` 屬性設為 `"food"`。

```js
assert.exists(document.querySelector('label + select[name="food"]'));
```
