# --interactive--

Beim Gestalten Ihrer Seiten arbeiten Sie mit verschiedenen Eigenschaften wie Breiten, Höhen, Innenabständen, Außenabständen und mehr. Wenn Sie diese Eigenschaften definieren, müssen Sie die Längeneinheiten angeben, die Sie verwenden möchten.

Es gibt zwei Arten von Einheiten, die Sie zur Definition dieser Eigenschaften verwenden können: relative Einheiten und absolute Einheiten. In dieser Lektion konzentrieren wir uns nur auf absolute Einheiten.

Absolute Längeneinheiten haben eine feste Länge und sind nicht relativ zu etwas anderem. Relativ bedeutet, dass die Länge relativ zu etwas anderem ist, wie der Größe des Bildschirms oder der Größe des übergeordneten Elements. Über relative Einheiten sprechen wir in den nächsten Lektionen.

Die gebräuchlichste absolute Einheit ist das Pixel (`px`). Pixel sind eine feste Maßeinheit in CSS, die eine präzise Kontrolle über Abmessungen ermöglicht. Das bedeutet, dass 1px immer 1/96 Zoll entspricht.

Es ist wichtig zu beachten, dass 1px zwar für CSS-Layouts standardisiert als 1/96 Zoll gilt, die tatsächliche physische Größe eines Pixels je nach Anzeige variieren kann.

Sehen wir uns ein Beispiel an, wie man Pixel verwendet, um die Breite und Höhe einer Box festzulegen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="box"></div>
```

```css
.box {
  width: 100px;
  height: 100px;
  background-color: red;
}
```

:::

In diesem Beispiel haben wir eine rote Box mit einer Breite und Höhe von `100px`. Die Abmessungen dieser Box bleiben unabhängig von Änderungen der Bildschirmgröße gleich.

Wenn Sie also ein Gerät mit kleinem Bildschirm verwenden, ist die Box immer noch `100px` breit. Wenn Sie einen großen Desktop-Bildschirm verwenden, ist die Box immer noch `100px` breit.

Wann sollten Sie also absolute Einheiten wie Pixel verwenden? Das hängt von der Situation ab. Im Allgemeinen verwenden Sie Pixel, wenn Sie eine präzise Kontrolle über die Abmessungen, Abstände und das Layout von Elementen benötigen. Manchmal verwenden Sie Pixel auch für Außenabstände, Innenabstände und Rahmen.

Hier ist ein Beispiel, wie man Pixel verwendet, um den Außenabstand für zwei Boxen festzulegen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="box"></div>
<div class="box"></div>
```

```css
.box {
  width: 100px;
  height: 100px;
  background-color: red;
  margin: 10px;
}
```

:::

Denken Sie daran, dass der Außenabstand der Raum außerhalb der Box ist. In diesem Beispiel hat die Box auf allen Seiten einen Außenabstand von `10px`.

Weitere Arten von absoluten Einheiten sind:

- Die `in` (Zoll) Einheit, die 96px entspricht
- Die `cm` (Zentimeter) Einheit, die 25,2/64 Zoll entspricht
- Die `mm` (Millimeter) Einheit, die 1/10 Zentimeter entspricht
- Die `q` (Viertelmillimeter) Einheit, die 1/40 Zentimeter entspricht
- Die `pc` (Picas) Einheit, die 1/6 Zoll entspricht
- Die `pt` (Punkte) Einheit, die 1/72 Zoll entspricht

Die meisten dieser Einheiten werden für den Druck verwendet und nicht für Bildschirme.

Obwohl die gebräuchlichste Einheit Pixel sind, ist es wichtig zu wissen, dass es auch andere absolute Einheiten gibt.
