# --explanation--

`Want` expressa um desejo de fazer algo.No passado, você diz `wanted` para mostrar que o desejo existia em um momento anterior.É seguido pelo infinitivo de um verbo (com `to`).Por exemplo:

- `She wants to install the new software.` - Ela deseja configurar ou adicionar o novo software ao sistema.

- `They wanted to upgrade their system last week.` - No passado, eles planejavam fazer o upgrade.
