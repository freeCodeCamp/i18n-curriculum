# --description--

合成数とは、少なくとも2つの素因数を含む数のことです。例えば、$15 = 3 × 5$、$9 = 3 × 3$、$12 = 2 × 2 × 3$です。

30未満の合成数のうち、正確に2つの素因数（必ずしも異なるとは限らない）を持つものは10個あります：4、6、9、10、14、15、21、22、25、26。

$10^8$未満の合成整数で、正確に2つの素因数（必ずしも異なるとは限らない）を持つものはいくつありますか？
