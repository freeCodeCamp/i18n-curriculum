# --description--

このような機能を持つフルスタックJavaScriptアプリを作成してください：<a href="https://stock-price-checker.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://stock-price-checker.freecodecamp.rocks/</a>。

信頼できる株価APIはすべてAPIキーが必要なため、回避策を用意しました。<a href="https://stock-price-checker-proxy.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://stock-price-checker-proxy.freecodecamp.rocks/</a> を使えば、自分でキーを登録しなくても最新の株価情報を取得できます。

このプロジェクトに取り組むには、以下のいずれかの方法でコードを書いてください：

- <a href="https://github.com/freeCodeCamp/boilerplate-project-stockchecker/" target="_blank" rel="noopener noreferrer nofollow">このGitHubリポジトリ</a>をクローンして、ローカルでプロジェクトを完成させる。
- お好きなサイトビルダーを使ってプロジェクトを完成させる。ただし、GitHubリポジトリのすべてのファイルを組み込むことを忘れないでください。

# --instructions--

1. `NODE_ENV` 環境変数を `test` に設定してください（引用符なし）
2. `routes/api.js` でプロジェクトを完成させるか、ハンドラ/コントローラを作成してください
3. セキュリティ機能はすべて `server.js` に追加してください
4. すべての機能テストは `tests/2_functional-tests.js` に作成してください

**注意** プライバシーに関する配慮：IPごとに1回のいいねのみ受け付ける必要があるため、IPアドレスを保存する必要があります。一般データ保護規則（GDPR）などのデータプライバシー法に準拠することが重要です。ユーザーのデータ保存許可を得る方法もありますが、匿名化するほうが簡単です。このチャレンジでは、IPアドレスをデータベースに保存する前に匿名化してください。匿名化の方法としては、ハッシュ化、切り捨て、IPアドレスの一部を0に設定するなどがあります。

以下のテストを `tests/2_functional-tests.js` に書いてください：

- 1つの株を見る：`/api/stock-prices/` へのGETリクエスト
- 1つの株を見ていいねする：`/api/stock-prices/` へのGETリクエスト
- 同じ株を見て再度いいねする：`/api/stock-prices/` へのGETリクエスト
- 2つの株を見る：`/api/stock-prices/` へのGETリクエスト
- 2つの株を見ていいねする：`/api/stock-prices/` へのGETリクエスト

`GET` リクエストを `/api/stock-prices` に送信し、NASDAQの株式シンボルを `stock` クエリパラメータとして渡せます。返されるオブジェクトには `stockData` というプロパティが含まれます。

```js
  const data = await fetch(
    code + '/api/stock-prices?stock=GOOG'
  );
  const parsed = await data.json();
  assert.property(parsed, 'stockData');
```

`stockData` プロパティには、文字列としての `stock` シンボル、数値としての `price`、および数値としての `likes` が含まれます。

```js
  const data = await fetch(
    code + '/api/stock-prices?stock=GOOG'
  );
  const parsed = await data.json();
  const ticker = parsed.stockData;
  assert.typeOf(ticker.price, 'number');
  assert.typeOf(ticker.likes, 'number');
  assert.typeOf(ticker.stock, 'string');
```

2つの株を渡すと、返される値は両方の株の情報を含む配列になります。`likes` の代わりに、両方の `rel_likes` オブジェクトに対していいねの差分である `stockData` が表示されます。

```js
  const data = await fetch(
    code + '/api/stock-prices?stock=GOOG&stock=MSFT'
  );
  const parsed = await data.json();
  const ticker = parsed.stockData;
  assert.typeOf(ticker, 'array');
  assert.property(ticker[0], 'rel_likes');
  assert.property(ticker[1], 'rel_likes');
```
