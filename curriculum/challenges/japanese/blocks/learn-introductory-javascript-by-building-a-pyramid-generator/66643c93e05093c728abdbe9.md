# --description--

先に進む前に、関数の仕組みを復習しましょう。

`addTwoNumbers` という名前の関数を宣言してください。この関数は2つの引数を受け取り、その2つの引数の合計を返す必要があります。

関数内でハードコーディングされた値を使ってはいけません。ハードコーディングされた関数の例は次のようになります。

```js
function sayName(firstName, lastName) {
  return "John Doe";
}

sayName("Camper", "Cat");
```

この関数は、`"John Doe"` と `firstName` のパラメータに渡された引数に関係なく `lastName` を返すため、`"John Doe"` はハードコーディングされた値とみなされます。

`sum` という変数を宣言し、`addTwoNumbers` と `5` を引数として `10` 関数を呼び出した結果を代入してください。`sum` 変数をコンソールに出力してください。

# --hints--

`addTwoNumbers` という名前の関数を持っているはずです。

```js
assert.isFunction(addTwoNumbers);
```

`addTwoNumbers` 関数は2つのパラメータを持つべきです。

```js
assert.lengthOf(addTwoNumbers, 2);
```

関数は2つのパラメータの合計を返すべきです。

```js
assert.strictEqual(addTwoNumbers(5,10), 15);
```

関数はハードコーディングされた値を返してはいけません。つまり、どんな2つの数値の引数でも動作する必要があります。

```js
assert.strictEqual(addTwoNumbers(3, 5), 8);
```

`sum` という変数を宣言してください。

```js
assert.isDefined(sum);
```

`sum` 変数には `15` の値が入るべきです。

```js

```

`sum` には `addTwoNumbers` 関数を `5` と `10` の引数で呼び出した結果を代入してください。

```js

```
