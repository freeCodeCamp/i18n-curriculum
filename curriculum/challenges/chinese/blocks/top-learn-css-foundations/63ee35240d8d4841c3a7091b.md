# --description--

类选择器将选择所有具有给定 `class` 的元素，`class` 只是你添加到超文本标记语言元素上的一个属性。以下是如何为超文本标记语言标签添加类并在 CSS 中选择它：

```html
<!-- index.html -->

<div class="alert-text">
  Please agree to our terms of service.
</div>
```

```css
/* styles.css */

.alert-text {
  color: red;
}
```

注意 `class` 选择器的语法：一个点紧跟着 `class` 属性区分大小写的值。`class` 不要求唯一，因此你可以在任意多个元素上使用相同的 `class`。

你可以对 `class` 属性做的另一件事是为单个元素添加多个类，作为以空格分隔的列表，例如 `class="alert-text severe-alert"`。由于空白字符用于分隔 `class` 名称，因此你绝不应该在多词名称中使用空格，而应使用连字符。

## ID 选择器
ID 选择器类似于 `class` 选择器。它们选择具有给定 `id` 的元素，`id` 是你放置在超文本标记语言元素上的另一种属性：

```html
<!-- index.html -->

<div id="title">My Awesome 90's Page</div>
```

```css
/* styles.css */

#title {
  background-color: red;
}
```

你不是用句号，而是用紧跟着 `id` 属性区分大小写的值的井号。一个常见的误区是人们过度使用 `id` 属性，而实际上并不一定需要，使用类就足够了。虽然在某些情况下使用 `id` 是有意义或必要的，比如利用特异性或让链接重定向到当前页面的某个部分，但你应该尽量少用 `id`（如果使用的话）。

类和 ID 之间的主要区别是一个元素只能有一个 `id`。单个页面上不能重复 `id`，且 `id` 属性中不应包含任何空白字符。
