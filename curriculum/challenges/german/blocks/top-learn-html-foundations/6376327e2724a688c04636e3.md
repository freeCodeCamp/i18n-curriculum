# --description--

HTML und CSS sind zwei Sprachen, die zusammenarbeiten, um alles zu erstellen, was Sie sehen, wenn Sie etwas im Internet betrachten. HTML ist die Rohdatenbasis, aus der eine Webseite aufgebaut ist. Alle Texte, Links, Karten, Listen und Schaltflächen werden in HTML erstellt. CSS fügt diesen einfachen Elementen Stil hinzu. HTML platziert Informationen auf einer Webseite, und CSS positioniert diese Informationen, verleiht ihnen Farbe, ändert die Schriftart und sorgt dafür, dass alles großartig aussieht!

# --questions--
    
## --text--

Was sind die Hauptaufgaben von HTML und CSS bei der Erstellung einer Webseite?

## --answers--

HTML strukturiert den Inhalt, während CSS die Elemente gestaltet.

---

HTML erstellt das visuelle Design, während CSS den Inhalt organisiert.

---

CSS steuert die Funktionalität, während HTML das Layout kontrolliert.
