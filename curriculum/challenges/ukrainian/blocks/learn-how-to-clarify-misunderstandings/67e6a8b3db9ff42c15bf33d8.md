# --explanation--

`Not just ... but` — це загальна конструкція, яка використовується для показу протиставлення або додавання акценту. Вона означає, що щось є більше, ніж одна річ. Друга частина додає більше інформації або важливості. Наприклад:

`Debugging is not just about finding errors, but about understanding the code better.` — це означає, що налагодження — це не просто виправлення помилок; воно також допомагає глибше зрозуміти.
