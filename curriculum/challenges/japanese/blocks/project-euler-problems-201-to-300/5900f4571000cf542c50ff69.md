# --description--

整数 $n ≥ 4$ に対して、$n$ の下側の素数平方根を $lps(n)$ と定義します。これは $\sqrt{n}$ 以下の最大の素数です。また、$n$ の上側の素数平方根を $ups(n)$ とし、$\sqrt{n}$ 以上の最小の素数とします。

例えば、$lps(4) = 2 = ups(4)$、$lps(1000) = 31$、$ups(1000) = 37$ です。

整数 $n ≥ 4$ が半分割数であるとは、$lps(n)$ と $ups(n)$ のうちどちらか一方だけが $n$ を割り切る場合を指します。ただし両方が割り切る場合は含みません。

15 以下の半分割数の合計は 30 で、その数は 8、10、12 です。15 は $lps(15) = 3$ と $ups(15) = 5$ の両方の倍数なので半分割数ではありません。さらに例を挙げると、1000 以下の 92 個の半分割数の合計は 34825 です。

999966663333 以下のすべての半分割数の合計を求めてください。
