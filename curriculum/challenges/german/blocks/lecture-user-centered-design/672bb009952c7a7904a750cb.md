Infinite Scrolling ist ein Designmuster, das mehr Inhalte lädt, während der Benutzer die Seite nach unten scrollt. Oft wird dies auf Social-Media-Seiten wie Twitter verwendet. Zum Beispiel, wenn Sie eingeloggt sind und mehr Tweets sehen möchten, können Sie nach unten scrollen und es werden mehr Tweets geladen. Das ist ein Beispiel für Infinite Scrolling.

Infinite Scrolling wird auch als Ersatz für die Paginierung verwendet. Paginierung ist ein Designmuster, das Inhalte in Seiten aufteilt. Dies wird oft verwendet, wenn viele Inhalte angezeigt werden müssen. Ein Beispiel für Paginierung ist, wenn Sie etwas bei Google suchen und die Suchergebnisse auf mehreren Seiten sehen. Bei der Paginierung müssen Sie auf eine Schaltfläche klicken, um zur nächsten Seite zu gelangen. Beim Infinite Scrolling scrollen Sie einfach weiter nach unten und es werden mehr Inhalte geladen.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="pagination">
  <div class="post">Post 1</div>
  <div class="post">Post 2</div>
  <div class="post">Post 3</div>
  <!-- More posts will be on the next pages -->
</div>
<div class="pagination-controls">
  <button class="prev" disabled>Previous</button>
  <button class="next">Next</button>
</div>
<script src="index.js"></script>
```

```css
.pagination {
  display: flex;
  flex-direction: column;
  gap: 12px;
}

.post {
  padding: 12px;
  border: 1px solid #ccc;
  border-radius: 4px;
}

.pagination-controls {
  margin-top: 12px;
}

.pagination-controls button {
  padding: 8px 16px;
  margin-right: 8px;
}
```

```js
let currentPage = 1;
const postsPerPage = 3;
const totalPosts = 9; 
const totalPages = Math.ceil(totalPosts / postsPerPage);
const container = document.querySelector('.pagination');
const prevButton = document.querySelector('.prev');
const nextButton = document.querySelector('.next'); 

function renderPosts() {
  container.innerHTML = '';
  const start = (currentPage - 1) * postsPerPage;
  const end = start + postsPerPage;
  for (let i = start; i < end && i < totalPosts; i++) {
    const post = document.createElement('div');
    post.className = 'post';
    post.textContent = `Post ${i + 1}`;
    container.appendChild(post);
  }
  prevButton.disabled = currentPage === 1;
  nextButton.disabled = currentPage === totalPages;
}

prevButton.addEventListener('click', () => {
  if (currentPage > 1) {
    currentPage--;
    renderPosts();
  }
});

nextButton.addEventListener('click', () => {
  if (currentPage < totalPages) {
    currentPage++;
    renderPosts();
  }
});

renderPosts();
```

Wenn Sie Infinite Scrolling in Ihr Design integrieren, gibt es einige bewährte Vorgehensweisen, die Sie beachten sollten. Die erste Überlegung ist, eine „Mehr laden“-Schaltfläche bereitzustellen, die den nächsten Satz von Ergebnissen lädt, wenn der Benutzer darauf klickt. Dies ist eine gute Möglichkeit, dem Benutzer die Kontrolle darüber zu geben, wann er mehr Inhalte sehen möchte.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

  <div class="post">Post 1</div>
  <div class="post">Post 2</div>
  <div class="post">Post 3</div>
  <!-- More posts will load when the user clicks "Load More" -->
</div>
<button class="load-more">Load More</button>
<script src="index.js"></script>
```

```css

  flex-direction: column;
  gap: 12px;
}

  padding: 12px;
  border: 1px solid #ccc;
  border-radius: 4px;
}

  margin-top: 12px;

}
```

```js
const loadMoreButton = document.querySelector('.load-more');
```

Eine weitere Überlegung wäre, eine „Zurück“-Schaltfläche hinzuzufügen. Diese gibt den Nutzenden die Möglichkeit, zur vorherigen Seite zurückzukehren, ohne ganz nach oben scrollen zu müssen. Das schafft eine bessere Benutzererfahrung und gibt ihnen mehr Kontrolle über ihr Surferlebnis.

Manchmal sehen Sie Designs mit einer „Zurück zum Anfang“-Schaltfläche, die die Nutzenden zurück zum Seitenanfang der Ergebnisse führt. Eine weitere Überlegung ist, einen Ladeindikator bereitzustellen. Nutzende sollten eine klare Anzeige haben, dass mehr Inhalte geladen werden; andernfalls könnten sie denken, die Seite sei defekt.

:::interactive_editor

```js
const container = document.querySelector('.infinite-scroll');
const loadingIndicator = document.querySelector('.loading-indicator');
const backToTopBtn = document.getElementById('back-to-top');

window.addEventListener('scroll', () => {
  if (window.scrollY > 400) {
    backToTopBtn.style.display = 'block';
  } else {
    backToTopBtn.style.display = 'none';
  }

  if (window.innerHeight + window.scrollY >= document.body.offsetHeight) {
    loadMorePosts();
  }
});

backToTopBtn.addEventListener('click', () => {
  window.scrollTo({ top: 0, behavior: 'smooth' });
});

function loadMorePosts() {
  loadingIndicator.style.display = 'block';
  setTimeout(() => {
    for (let i = 0; i < 3; i++) {
      const post = document.createElement('div');
      post.className = 'post';
      post.textContent = `Post ${container.children.length + 1}`;
      container.appendChild(post);
    }
    loadingIndicator.style.display = 'none';
  }, 1000);
}
```

Eine der letzten Überlegungen ist, die Fußzeile für den Nutzer zugänglich zu halten. Wenn die Fußzeile wichtige Informationen enthält, sollte sie jederzeit zugänglich sein.

Zusammenfassend ist Infinite Scrolling eine großartige Möglichkeit, Inhalte auf Ihrer Website anzuzeigen. Sie sollten jedoch die bewährten Vorgehensweisen beim Design Ihres Infinite Scrolls beachten, damit Sie die bestmögliche Benutzererfahrung bieten können.
