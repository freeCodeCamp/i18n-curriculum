# --explanation--

`What do you think about...` é frequentemente usado para fazer uma sugestão educada, não apenas para pedir uma opinião.Ele convida a outra pessoa a responder ou dar uma contribuição sobre um possível plano.Por exemplo:

`What do you think about moving the meeting to Friday?` – Esta é uma forma educada de sugerir a mudança do dia da reunião.Em vez de perguntar se a outra pessoa gosta ou não do plano, você está na verdade sugerindo mover a reunião para outra data.
