# --explanation--

`Maybe` significa `possibly` ou `there's a chance`. É usado para expressar incerteza ou falta de confiança total.

`Hard to know` significa que é difícil determinar ou fazer um julgamento devido à falta de informação ou clareza.

`Maybe, but it's hard to know`, combina essas ideias para indicar que resolver problemas financeiros é possível, mas o resultado é incerto.
