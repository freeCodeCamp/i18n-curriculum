# --explanation--

`As I was saying` wird verwendet, um nach einer Unterbrechung zu einem vorherigen Thema zurückzukehren. Es signalisiert, dass der Sprecher seinen vorherigen Punkt fortsetzt. Zum Beispiel:  

`As I was saying, we should finalize the report by Friday.` – Das bedeutet, dass der Sprecher bereits über den Bericht gesprochen hat, bevor er unterbrochen wurde.
