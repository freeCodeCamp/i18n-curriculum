# --instructions--

Для цього завдання потрібно створити дослідник графіків даних за допомогою Python, який прийматиме вхідні дані та виконуватиме наступне:

- Отримати .csv файл трьома способами
  - завантаженням з локального комп’ютера
  - отриманням URL від користувача
  - вставленням URL у код
- Використовувати бібліотеку Pandas для збереження .csv як таблиці даних
- Вивести заголовки та перші два рядки
- Зберегти назви стовпців у вигляді списку
- Вибрати один або два стовпці та конвертувати дані у масиви Numpy
- Відобразити дані у вигляді точкової діаграми або лінійного графіка
- Мати можливість робити це для різних комбінацій стовпців і інтерпретувати графіки

Після завершення відправте URL публічного записника Colab у вашому Google диску.
