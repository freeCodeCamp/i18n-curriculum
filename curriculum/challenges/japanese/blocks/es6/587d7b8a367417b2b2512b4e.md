# --description--

ES6の新しい機能のひとつに<dfn>テンプレートリテラル</dfn>があります。これは複雑な文字列を簡単に作成できる特別な種類の文字列です。

テンプレートリテラルを使うと、複数行の文字列を作成したり、文字列の補間機能を使って文字列を作成したりできます。

以下のコードを見てください。

```js
const person = {
  name: "Zodiac Hasbro",
  age: 56
};

const greeting = `Hello, my name is ${person.name}!
I am ${person.age} years old.`;

console.log(greeting);
```

コンソールには文字列`Hello, my name is Zodiac Hasbro!`と`I am 56 years old.`が表示されます。

ここでは多くのことが起きています。まず、例ではバックティック（`` ` ``), not quotes (`'` or `"`), to wrap the string. Secondly, notice that the string is multi-line, both in the code and the output. This saves inserting `\n` within strings. The `${variable}` syntax used above is a placeholder. Basically, you won't have to use concatenation with the `+` operator anymore. To add variables to strings, you just drop the variable in a template string and wrap it with `${` and `}`. Similarly, you can include other expressions in your string literal, for example `${a + b}`）を使っています。この新しい文字列の作成方法は、より柔軟に堅牢な文字列を作成できるようにします。

# --instructions--

バックティックを使ったテンプレートリテラル構文で、リスト要素（`li`）の文字列の配列を作成してください。各リスト要素のテキストは`failure`オブジェクトの`result`プロパティの配列の要素のいずれかであり、`class`属性に`text-warning`の値を持つようにしてください。`makeList`関数はリストアイテムの文字列の配列を返す必要があります。

目的の出力（以下に示す）を得るためにイテレーターのメソッド（任意のループ）を使ってください。

```js
[
  '<li class="text-warning">no-var</li>',
  '<li class="text-warning">var-on-top</li>',
  '<li class="text-warning">linebreak</li>'
]
```
