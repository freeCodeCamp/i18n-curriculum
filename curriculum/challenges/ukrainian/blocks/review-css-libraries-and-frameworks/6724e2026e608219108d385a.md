## Фреймворки CSS

- **Фреймворки CSS**: фреймворки CSS можуть прискорити ваш потік роботи, створити єдиний візуальний стиль на сайті, забезпечити послідовний дизайн у різних браузерах і допомогти краще організувати ваш CSS-код.
- **Популярні фреймворки CSS**: серед популярних фреймворків CSS — Tailwind CSS, Bootstrap, Materialize і Foundation.
- **Можливі недоліки**:
  - CSS, який надає фреймворк, може конфліктувати з вашим кастомним CSS.
  - Ваш сайт може виглядати схожим на інші сайти, що використовують той самий фреймворк.
  - Великі фреймворки можуть спричиняти проблеми з продуктивністю.

## Два типи фреймворків CSS

- **Фреймворки CSS, орієнтовані на утиліти**: ці фреймворки мають невеликі класи з конкретними призначеннями, наприклад, для встановлення поля, відступу або кольору фону. Ви можете призначати ці невеликі класи безпосередньо HTML-елементам за потреби. Tailwind CSS належить до фреймворків CSS, орієнтованих на утиліти.

Ось приклад використання Tailwind CSS для стилізації кнопки.

```html
<button class="bg-blue-500 text-white font-bold py-2 px-4 rounded-full hover:bg-blue-700">
  Button
</button>
```

- **Фреймворки CSS, орієнтовані на компоненти**: ці фреймворки мають готові компоненти з попередньо визначеними стилями, які ви можете додати на свій сайт. Компоненти доступні в офіційній документації фреймворку CSS, і ви можете скопіювати їх і вставити у свій проєкт. Bootstrap належить до фреймворків CSS, орієнтованих на компоненти.

Ось приклад використання Bootstrap для створення групи списку. Замість того, щоб призначати невеликі класи HTML-елементам, ви додасте весь компонент разом із HTML-структурою.

```html
<div class="card" style="width: 25rem;">
  <ul class="list-group list-group-flush">
    <li class="list-group-item">HTML</li>
    <li class="list-group-item">CSS</li>
    <li class="list-group-item">JavaScript</li>
  </ul>
</div>
```

## Tailwind CSS

Tailwind — це фреймворк CSS, орієнтований на утиліти. Замість написання кастомних правил CSS ви будуєте дизайн, комбінуючи невеликі утилітарні класи безпосередньо в HTML.

### Утиліти для адаптивного дизайну

Tailwind використовує префікси, такі як `sm:`, `md:` і `lg:`, щоб застосовувати стилі на різних розмірах екрана.

```html
<div class="w-full md:w-1/2 lg:flex-row">Responsive layout</div>
```

### Утиліти для гнучкого контейнера

Класи на кшталт `flex`, `flex-col`, `justify-around` і `items-center` полегшують створення гнучких макетів.

```html
<div class="flex flex-col md:flex-row justify-around items-center">
  <p>Column on small screens</p>
  <p>Row on medium and larger screens</p>
</div>
```

### Утиліти для сітки

Tailwind включає утиліти для CSS-сітки, такі як `grid`, `grid-cols-1` і `md:grid-cols-3`.

```html
<div class="grid grid-cols-1 md:grid-cols-3 gap-8">
  <div class="bg-gray-100 p-4">Column 1</div>
  <div class="bg-gray-100 p-4">Column 2</div>
  <div class="bg-gray-100 p-4">Column 3</div>
</div>
```

### Утиліти для відступів

Утиліти на кшталт `mt-8`, `mx-auto`, `p-4` і `gap-4` допомагають створювати послідовні відступи без написання CSS.

```html
<div class="mt-8 p-4 bg-indigo-600 text-white">Spaced content</div>
```

### Утиліти для типографіки

Утиліти на кшталт `uppercase`, `font-bold`, `font-semibold` і `text-4xl` контролюють вигляд тексту.

Ви можете встановлювати розміри шрифтів, які підлаштовуються під точки зупинки, наприклад `text-3xl` `md:text-5xl`.

```html
<h1 class="text-3xl md:text-5xl font-semibold text-center">Responsive Heading
</h1>
```

### Кольори та стани наведення

Tailwind пропонує широку палітру кольорів, таких як `text-red-700`, `bg-indigo-600` і `bg-gray-100`.

Класи на кшталт `hover:bg-pink-600` роблять інтерактивні ефекти простими.

```html
<a href="#" class="bg-pink-500 hover:bg-pink-600 text-white px-4 py-2 rounded-md">
  Hover Me
</a>
```

### Кордони, кільця та ефекти

- **Кордон**: `border-2 border-red-300` додає кордони з вказаною товщиною та кольорами.
- **Кільця**: `ring-1 ring-gray-300` створює ефекти, схожі на обриси, які часто використовують для фокусу або карток.
- **Закруглені кути та масштабування**: класи на кшталт `rounded-md`, `rounded-xl` і `scale-105` додають завершеності.

```html
<div class="p-6 rounded-xl ring-2 ring-fuchsia-500 scale-105">
  Highlighted card
</div>
```

### Градієнти

Tailwind підтримує утиліти для градієнтів, такі як `bg-gradient-to-r from-fuchsia-500 to-indigo-600`.

```html
<div class="p-4 text-white bg-gradient-to-r from-fuchsia-500 to-indigo-600">
  Gradient background
</div>
```

## CSS-препроцесори

- **CSS-препроцесор**: це інструмент, який розширює стандартний CSS. Він компілює код із розширеним синтаксисом у нативний CSS-файл. Це корисно для написання чистішого, повторно використовуваного, менш повторюваного та масштабованого CSS для складних проєктів.
- **Функції**: деякі функції, які можуть надавати CSS-препроцесори, — це змінні, міксини, вкладеність і успадкування селекторів.
- **Популярні CSS-препроцесори**: серед популярних препроцесорів — Sass, Less і Stylus.  
- **Можливі недоліки**:
  - Компіляція правил CSS у стандартний CSS може спричиняти додаткове навантаження.
  - Скомпільований код може бути складним для налагодження.

## Sass

- **Sass**: один із найпопулярніших CSS-препроцесорів. Sass означає "Syntactically Awesome Style Sheets".
- **Функції, які підтримує Sass**: Sass підтримує змінні, вкладені CSS-правила, модулі, міксини, успадкування та оператори для базових математичних операцій.

## Два синтаксиси, які підтримує Sass

- **Синтаксис SCSS**: SCSS (Sassy CSS) розширює базовий синтаксис CSS. Це найпоширеніший синтаксис для Sass. Файли SCSS мають розширення `.scss`.

Ось приклад визначення та використання змінної в SCSS.

```scss
$primary-color: #3498eb;

header {
  background-color: $primary-color;
}
```

- **Відступний синтаксис**: відступний синтаксис був оригінальним синтаксисом Sass і також відомий як "Sass syntax".

Ось приклад визначення та використання змінної у відступному синтаксисі.

```sass
$primary-color: #3498eb

header
  background-color: $primary-color
```

### Міксини

- **Міксини**: міксини дозволяють групувати кілька CSS-властивостей і їхніх значень під одним іменем і повторно використовувати цей блок CSS-коду по всьому стилю.

Ось приклад визначення міксина в синтаксисі SCSS. У цьому випадку міксин називається `center-flex`. Він містить три CSS-властивості для центрування елементів за допомогою гнучкого контейнера.

```scss
@mixin center-flex {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

Ось приклад використання визначеного міксина.

```scss
section {
  @include center-flex;
  height: 500px;
  background-color: #3289a8;
}
```
