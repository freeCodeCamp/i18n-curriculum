# --description--

Нехай $T(n)$ — $n$-те трикутне число, тобто $T(n) = \frac{n(n + 1)}{2}$.

Нехай $dT(n)$ — кількість дільників числа $T(n)$. Наприклад: $T(7) = 28$ і $dT(7) = 6$.

Нехай $Tr(n)$ — кількість трійок ($i$, $j$, $k$), таких що $1 ≤ i < j < k ≤ n$ і $dT(i) > dT(j) > dT(k)$. $Tr(20) = 14$, $Tr(100) = 5\,772$ і $Tr(1000) = 11\,174\,776$.

Знайдіть $Tr(60\,000\,000)$. Вкажіть останні 18 цифр вашої відповіді.
