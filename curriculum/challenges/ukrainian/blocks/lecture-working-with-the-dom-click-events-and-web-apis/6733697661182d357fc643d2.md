# --interactive--

`Canvas` — це потужний інструмент, який дозволяє маніпулювати графікою безпосередньо у вашому JavaScript-файлі. Все починається з елемента `canvas` в HTML. Цей елемент слугує поверхнею для малювання, якою ви можете керувати за допомогою методів і властивостей `Canvas`.

API `Canvas` надає все необхідне для створення вражаючих візуалізацій, включно з формами, текстом, анімаціями та навіть складними іграми. Він має інтерфейси, такі як `HTMLCanvasElement`, `CanvasRenderingContext2D`, `CanvasGradient`, `CanvasPattern`, `TextMetrics`, які надають методи і властивості для створення графіки у вашому JavaScript-файлі.

Давайте розглянемо, як працює `Canvas` API, щоб ви могли почати створювати графіку за допомогою JavaScript.

Спочатку потрібно створити елемент `canvas` у вашому HTML-файлі:

```html
<canvas id="my-canvas"></canvas>
```

Елемент `canvas` представлений інтерфейсом `HTMLCanvasElement`, який надає методи і властивості для його маніпуляції. Крім того, ви можете використовувати методи і властивості інших інтерфейсів API `Canvas`.

Ви можете задати вашому `canvas` `width` і `height` всередині HTML:

```html
<canvas id="my-canvas" width="400" height="400"></canvas>
```

Або ви можете використати властивості `width` і `height` інтерфейсу `HTMLCanvasElement`:

```js
const canvas = document.getElementById("my-canvas");
canvas.width = 400;
canvas.height = 400;
```

Поки що на екрані нічого не видно. Після створення елемента `canvas` наступним кроком є отримання доступу до контексту малювання полотна за допомогою методу `getContext` інтерфейсу `HTMLCanvasElement`.

Найпоширенішим контекстом є `2d`, який дозволяє малювати у двох вимірах:

```js
const canvas = document.getElementById("my-canvas");
const ctx = canvas.getContext('2d');
```

Якщо вивести змінну `ctx` у консоль, ви побачите методи і властивості `CanvasRenderingContext2D`, які можна використовувати для створення форм, кольорів, ліній та іншого, разом із їхніми значеннями за замовчуванням:

```js
console.log(ctx);
```

Отримавши 2D-контекст, ви можете починати малювати на полотні.

API `Canvas` надає кілька методів і властивостей для малювання форм, ліній і тексту. Однією з таких є властивість `fillStyle`, яку можна поєднати з методом `fillRect()` для малювання прямокутника або квадрата:

:::interactive_editor

```html
<html>
  <head>
  </head>
  <body>
    <canvas id="my-canvas" width="400" height="400"></canvas>
    <script src="index.js"></script>
  </body>
</html>
```

```js
const canvas = document.getElementById("my-canvas");

const ctx = canvas.getContext("2d");

// Set the background color
ctx.fillStyle = "crimson";

// Draw a rectangle
ctx.fillRect(1, 1, 150, 100);
```

:::

Метод `fillRect` приймає 4 числові значення, які відповідно представляють вісь x, вісь y, ширину і висоту.

Тепер на екрані щось є. Ви також можете малювати текст або навіть створювати анімацію. Ось полотно для відображення тексту:

```html
<canvas id="my-text-canvas" width="300" height="70"></canvas>
```

Щоб нарешті намалювати текст, передайте текст у метод `fillText` як перший аргумент, а потім значення для осі x і осі y:

:::interactive_editor

```html
<html>
  <head>
  </head>
  <body>
    <canvas id="my-text-canvas" width="300" height="70"></canvas>
    <script src="index.js"></script>
  </body>
</html>
```

```js
const textCanvas = document.getElementById("my-text-canvas");

const textCanvasCtx = textCanvas.getContext("2d");

// Set font family and size
textCanvasCtx.font = "30px Arial";

// Set text color
textCanvasCtx.fillStyle = "crimson";

// Draw the text
textCanvasCtx.fillText("Hello HTML Canvas!", 1, 50);
```

:::

Результатом у браузері буде червоний текст `Hello HTML Canvas!`.

З API `Canvas` можна робити набагато більше. Наприклад, ви можете поєднувати його з `requestAnimationFrame` для створення кастомних анімацій, візуалізацій, ігор і не тільки.
