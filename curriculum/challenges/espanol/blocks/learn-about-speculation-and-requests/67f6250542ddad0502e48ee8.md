## --text--

¿Cuál oración es gramaticalmente correcta?

## --answers--

`They might has missed the deadline because of a system error.`

### --feedback--

La forma correcta es `might have`, no `might has`.

---

`They might missed the deadline due to a system error.`

### --feedback--

El verbo después de `might` debe ser la forma base de un verbo.

---

`They might have missed the deadline because of a system error.`

---

`They might had missed the deadline if they weren't reminded.`

### --feedback--

Después de `might`, usamos `have`, no `had`.

# --explanation--

El verbo modal `might` va seguido de `have` + `Past Participle` para expresar una posibilidad en el pasado. Esta estructura se usa cuando no estás seguro si algo ocurrió, pero crees que es posible.

- Correcto: `They might have missed the deadline.`

- Incorrecto: `They might has missed...`, `They might missed...`, o `They might had missed...`
