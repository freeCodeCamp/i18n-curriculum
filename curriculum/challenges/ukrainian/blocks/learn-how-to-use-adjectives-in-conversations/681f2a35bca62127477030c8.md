# --explanation--

`Plug-and-play` означає, що щось готове до використання одразу, без потреби в складній інсталяції. Наприклад:

`The printer was plug-and-play. We just connected it and it worked.` – Це означає, що не було потрібних додаткових кроків.

`User-friendly` означає, що щось спроєктоване так, щоб бути простим і зручним для користувачів. Наприклад:

`The app has a user-friendly interface.` – Це означає, що ним легко користуватися, навіть для початківців.
