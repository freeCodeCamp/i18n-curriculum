# --interactive--

JavaScript ist eine leistungsstarke Programmiersprache, die Interaktivität und dynamisches Verhalten auf Webseiten ermöglicht.

Während HTML und CSS verwendet werden, um Inhalte zu strukturieren und Elemente auf einer Seite zu gestalten, geht JavaScript darüber hinaus, indem es komplexere Funktionen ermöglicht, wie das Verarbeiten von Benutzereingaben, das Animieren von Elementen und sogar das Erstellen kompletter Webanwendungen.

Zum Beispiel bestimmt JavaScript, wie sich die Seite verhält, wenn Sie auf einen Button klicken, ein Formular absenden oder mit der Maus über ein Menü fahren.

Hier ist ein Beispiel, wie diese drei zusammenarbeiten:

:::interactive_editor

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: green;
    }
  </style>
</head>
<body>
  <h1>Hello, World!</h1>
  <button onclick="alert('Button clicked!')">Click me</button>
</body>
</html>
```

:::

In diesem Beispiel wird HTML verwendet, um den Inhalt zu definieren: eine Überschrift (ein `h1`-Element) und einen Button (das `button`-Element).

CSS wird verwendet, um der Überschrift Stil zu verleihen, zum Beispiel den Text grün zu machen. JavaScript wird verwendet, um eine Warnmeldung anzuzeigen, wenn der Button angeklickt wird.

Zusammengefasst liefert HTML die Struktur, CSS fügt das Styling hinzu, und JavaScript ermöglicht das Verhalten, wodurch ein vollständiges und interaktives Web-Erlebnis entsteht.
