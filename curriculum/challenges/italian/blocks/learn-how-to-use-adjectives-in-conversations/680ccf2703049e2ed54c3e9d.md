# --explanation--

`Longer` significa avere più distanza da un'estremità all'altra. Per esempio:

`We need longer cables to reach the new server racks.` – Questo indica cavi con più lunghezza.

`Twice` significa il doppio della quantità. Per esempio:

`We ordered twice the number of processors this time.` – Questo indica il doppio della quantità.

`As long as` si usa per un confronto diretto in lunghezza. Per esempio:

`The new cable is as long as the room.` – Questo mostra che la lunghezza del cavo corrisponde alla dimensione della stanza.
