## --text--

أي (method) يمتلكها `Cursor` كـ instance ماذا تفعل؟

## --answers--

`Cursor` كـ instance تمتلك (method) تُسمَي `.run()` التي تسمح بتشغيل (SQL queries).

---

`Cursor` تمتلك `.select()` كـ method التي تسمح لك باختيار الصفوف(records).

---

`Cursor` تمتلك method تسمي `.execute()` التي تستلم مدخل SQL لتشغيلها ضد قاعدة البيانات.
