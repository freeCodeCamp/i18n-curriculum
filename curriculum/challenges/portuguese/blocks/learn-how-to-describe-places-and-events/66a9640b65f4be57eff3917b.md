# --explanation--

As pessoas sugerem horários para se encontrar e confirmam se esses horários funcionam para os outros. 

Ouça as palavras que Bob diz depois de `How about...`, ele diz um dia e hora específicos: `tomorrow during lunch`. 

Lisa responde com `works for me`, que é uma forma informal de dizer que o horário sugerido é conveniente ou aceitável.Por exemplo:

- Bob: `How about meeting at 3 PM?`, Lisa: `Works for me!` - Isso significa que Lisa está disponível para se encontrar às 15h.
