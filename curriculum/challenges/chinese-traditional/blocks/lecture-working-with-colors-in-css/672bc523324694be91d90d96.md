# --interactive--

在設計網站樣式時，使用 HSL 色彩模型是處理顏色最萬用且直覺的方法之一。HSL 代表 Hue、Saturation 和 Lightness——定義顏色的三個關鍵組件。 

HSL 色彩模型以更符合人類感知色彩的方式表述顏色。在本課程中，我們將探討什麼是 HSL 色彩模型以及如何在 CSS 中使用 `hsl()` 函式。

色相是顏色的型別，以色輪上的角度表述，範圍為 `0` 到 `360` 度。 例如，`0` 度代表紅色，`120` 度是綠色，`240` 度是藍色。透過調整色相，你可以在光譜上移動不同的顏色。

飽和度是指顏色的強度或純度。它以百分比來衡量，範圍從 `0%`（完全去飽和的灰階顏色）到 `100%`（該顏色最鮮豔的形式）。飽和度為 `100%` 的顏色是完全鮮明的，而飽和度為 `0%` 的顏色則呈現灰階色調。

明度決定顏色的明暗程度，同樣以百分比來衡量。明度值為 `0%` 時會產生黑色，`50%` 則會呈現色相的正常色調，`100%` 則會變成白色。

在 CSS 中，`hsl()` 函式（程式）用於使用 HSL 色彩模型定義顏色。以下是基本語法：

```css
element {
  color: hsl(hue, saturation, lightness);
}
```

讓我們用一個範例來拆解這個問題：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
body {
  background-color: hsla(0, 0%, 1%, 1.00);
}

p {
  color: hsl(120, 100%, 50%);
}
```

:::

在這種情況下，色相是 `120` 度，對應綠色。飽和度是 `100%`，所以綠色將會是完全鮮豔的。亮度是 `50%`，因此它處於正常色調——既不會太暗也不會太亮。結果，段落的文字顏色將會是明亮的綠色。

HSL 色彩模型的主要優點之一是其直覺性。它讓你可以輕鬆調整色彩的鮮豔度或明度，只需微調飽和度和明度數值，而不必改變核心色相（hue）。 

例如，如果你想創建相同顏色的不同色調或色階，你只需調整亮度值。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="light">This is a light blue div.</div>
<div class="dark">This is a dark blue div.</div>
```

```css
div.light {
  background-color: hsl(240, 100%, 80%);
}

div.dark {
  background-color: hsl(240, 100%, 20%);
  color: hsl(0, 0%, 100%);
}
```

:::

這裡，兩個 `div` 元素都使用相同的色相（`240` 度，即藍色），但一個的明度為 `80%`（較淺的藍色調），另一個的明度為 `20%`（較深的藍色調）。

就像 RGB 模型有一個 `rgba()` 函式（程式）來包含透明度，HSL 模型也有一個 `hsla()` 函式（程式）。這個函式（程式）中的第四個參數表述 alpha 值，該值控制顏色的不透明度。以下是基本語法：

```css
element {
  background-color: hsla(hue, saturation, lightness, alpha);
}
```

讓我們來看一個範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

這段程式碼會給 `div` 一個半透通的紅色背景，其中色相設定為 `0` 度（紅色）、飽和度為 `100%`、亮度為 `50%`，以及 alpha 為 `0.5`（50% 不透明度）。你也可以將 alpha 指定為百分比 — 例如，`hsla(0, 100%, 50%, 50%)` 等價於 `hsla(0, 100%, 50%, 0.5)`。

HSL 色彩模型在你需要創建色彩結構綱目並輕鬆調整色調或色階時特別有用。 

因為 HSL 將色相與飽和度和明度分開，你可以在調整陰影或凸顯標示的明度時，保持一致的色彩主題。這種層次的彈性使 HSL 成為設計師和開發者的最愛。

此外，與十六進位色碼或 RGB 等其他方法相比，HSL 使顏色值更易讀且更易理解。你可以更清楚地看到顏色背後的意圖，並相應地進行調整，特別是在嘗試不同設計時。

使用 HSL 色彩模型，你將獲得更多控制色調、飽和度和亮度的彈性，使其成為在 CSS 中創造豐富且動態設計的強大工具。
