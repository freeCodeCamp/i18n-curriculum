# --description--

`Encourage` significa dar a alguém apoio ou confiança para fazer algo. Por exemplo, quando seu amigo diz, `You can solve this coding problem!` eles estão te encorajando. 

`Discourage` é o oposto; significa fazer alguém se sentir menos confiante ou menos propenso a fazer algo. Se aquele amigo disser, `That programming language is too hard for beginners`, isso pode desencorajá-lo a aprendê-lo.

`Discouraging` descreve algo que faz você se sentir desanimado. Como, `Finding a bug in your code after hours of work can be discouraging.`

`Encouraging` é quando algo lhe dá coragem ou esperança. Por exemplo, `Getting your first program to run is very encouraging.`
