# --description--

Komponenten sind das Herzstück von React. Alles in React ist eine Komponente, und hier lernen Sie, wie man eine erstellt.

Es gibt zwei Möglichkeiten, eine React-Komponente zu erstellen. Die erste Möglichkeit ist die Verwendung einer JavaScript-Funktion. Eine Komponente auf diese Weise zu definieren, erzeugt eine *zustandslose funktionale Komponente*. Das Konzept des Zustands in einer Anwendung wird in späteren Aufgaben behandelt. Für den Moment können Sie sich eine zustandslose Komponente als eine vorstellen, die Daten empfangen und darstellen kann, aber keine Änderungen an diesen Daten verwaltet oder verfolgt. (Die zweite Möglichkeit, eine React-Komponente zu erstellen, behandeln wir in der nächsten Aufgabe.)

Um eine Komponente mit einer Funktion zu erstellen, schreiben Sie einfach eine JavaScript-Funktion, die entweder JSX oder `null` zurückgibt. Wichtig ist, dass React verlangt, dass der Name Ihrer Funktion mit einem Großbuchstaben beginnt. Hier ist ein Beispiel für eine zustandslose funktionale Komponente, die eine HTML-Klasse in JSX zuweist:

```jsx
const DemoComponent = function() {
  return (
    <div className='customClass' />
  );
};
```

Nach der Transpilierung wird das `<div>` eine CSS-Klasse von `customClass` haben.

Da eine JSX-Komponente HTML darstellt, können Sie mehrere Komponenten zusammenfügen, um eine komplexere HTML-Seite zu erstellen. Dies ist einer der wichtigsten Vorteile der von React bereitgestellten Komponentenarchitektur. Sie ermöglicht es Ihnen, Ihre Benutzeroberfläche aus vielen separaten, isolierten Komponenten zusammenzusetzen. Das macht es einfacher, komplexe Benutzeroberflächen zu erstellen und zu pflegen.

# --instructions--

Der Code-Editor enthält eine Funktion namens `MyComponent`. Vervollständigen Sie diese Funktion so, dass sie ein einzelnes `div`-Element zurückgibt, das einen String mit Text enthält.

**Hinweis:** Der Text gilt als Kind des `div`-Elements, daher können Sie kein selbstschließendes Tag verwenden.

# --hints--

`MyComponent` sollte JSX zurückgeben.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.length === 1;
  })()
);
```

`MyComponent` sollte ein `div`-Element zurückgeben.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.children().type() === 'div';
  })()
);
```

Das `div`-Element sollte einen String mit Text enthalten.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.find('div').text() !== '';
  })()
);
```
