# --description--

令 $T(n)$ 为第 $n^{\text{个}}$ 三角形数，因此 $T(n) = \frac{n(n + 1)}{2}$。

设 $dT(n)$ 为 $T(n)$ 的约数个数。例如：$T(7) = 28$ 且 $dT(7) = 6$。

设 $Tr(n)$ 为满足 $1 ≤ i &lt; j &lt; k ≤ n$ 且 $dT(i) > dT(j) > dT(k)$ 的三元组 ($i$, $j$, $k$) 的数量。$Tr(20) = 14$，$Tr(100) = 5\\,772$，且 $Tr(1000) = 11\\,174\\,776$。

查找 $Tr(60\,000\,000)$。给出你答案的最后 18 位数字。
