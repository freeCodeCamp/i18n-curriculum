### --question--

#### --text--

How can you use CSS variables to dynamically change the color of multiple elements?

#### --distractors--

```css

  primary-color: blue;
}

.element1, .element2 {
  color: --primary-color;
}
```

---

```css

  --color: blue;
}

.element1, .element2 {
  color: var(color);
}
```

---

```css
:root {
  --primary: blue;
}

.element1, .element2 {
  color: var(--primary);
}
```

#### --answer--

```css
:root {
  --primary-color: blue;
}

.element1, .element2 {
  color: var(--primary-color);
}
```
