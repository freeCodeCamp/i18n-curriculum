# --explanation--

有时，当你对某事不完全确定，但你相信它应该是真的时，你可以使用短语 `It's supposed to` + 动词。 

这将表示某事预计会以某种方式存在，即使你尚未确认。举例来说：

- `The update is supposed to fix the bug。`——这意味着你期望更新修复该错误，但你尚未确认。
