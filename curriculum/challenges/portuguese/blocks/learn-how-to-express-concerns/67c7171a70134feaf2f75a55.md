# --explanation--

Nesta frase, Maria usa `they` para se referir aos **membros da equipe**.Mais cedo na conversa, ela mencionou `the team` e usou um verbo no singular quando disse `I need to understand how the team feels about the new project deadline`.Agora, ela se refere às pessoas individualmente, então ela muda para `they`.Outro exemplo:

- O comitê decide as regras. - Observe o uso do verbo no singular para se referir ao grupo como um todo. 

- `They discuss the rules before making a decision.` - Há uma mudança para enfatizar os indivíduos no grupo (os membros do comitê), então o plural é usado.
