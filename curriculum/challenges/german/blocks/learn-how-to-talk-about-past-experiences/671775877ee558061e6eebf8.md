# --explanation--

Der Ausdruck `in real time` bezeichnet etwas, das unmittelbar im Moment des Geschehens passiert, ohne Verzögerung. Er wird oft im Zusammenhang mit Überwachung, Kommunikation oder Datenverarbeitung verwendet. Zum Beispiel:

`The system updates the dashboard in real time, so you can see changes instantly.` – Das bedeutet, dass sich das Dashboard sofort ändert, nachdem die Updates angewendet wurden.

In Jakes Satz meint er, dass er zwar nach Sicherheitslücken gesucht hat, dies aber nicht immer sofort im Moment der Updates geschah.
