# --interactive--

La griglia implicita si riferisce alle righe e colonne create automaticamente dal browser quando posizioni gli elementi in un layout a griglia – quelle non definite esplicitamente usando `grid-template-rows` o `grid-template-columns`.

Le proprietà che controllano le colonne e le righe create implicitamente dal browser sono `grid-auto-columns` e `grid-auto-rows`.

La griglia implicita si riferisce anche alle righe e colonne aggiuntive che il browser genera automaticamente quando posizioni un elemento al di fuori delle righe e colonne definite esplicitamente.

Per esempio, supponiamo che tu definisca solo due colonne esplicite in un layout a griglia in questo modo:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
  <div class="grid-item">Item 4</div>
  <div class="grid-item">Item 5</div>
  <div class="grid-item">Item 6</div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 100px 100px; /* Only 2 explicit columns */
}

.grid-item {
  background-color: burlywood;
  border: 1px solid orangered;
  padding: 0.5rem;
  margin: 0.5rem;
}
```

:::

Due elementi riempiono la prima riga usando le due colonne esplicite – `Item 1` nella prima colonna e `Item 2` nella seconda colonna.

Gli elementi successivi iniziano una nuova riga con `Item 3` che va nella prima colonna della seconda riga e `Item 4` nella seconda colonna della seconda riga, e così via.

Come hai già visto, le griglie esplicite sono le aree della griglia che imposti intenzionalmente. Cioè, le righe e colonne che definisci esplicitamente per un layout a griglia usando le proprietà `grid-template-rows` e `grid-template-columns`.

Ecco una tabella che riassume le differenze tra griglie implicite ed esplicite:

| **Funzionalità**          | **Griglia esplicita**                                                                  | **Griglia implicita**                                                                          |
|---------------------------|---------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| **Controllo delle dimensioni**     | Completamente personalizzabile usando ` grid-template-rows ` e ` grid-template-columns `. | Controllata da ` grid-auto-rows ` e ` grid-auto-columns `, o predefinita a ` auto `. |
| **Comportamento predefinito** | Non cambia a meno che non sia definita esplicitamente.                                         | Si adatta automaticamente agli elementi posizionati fuori dalla griglia esplicita.                            |
| **Complessità**       | Richiede più pianificazione per la struttura del layout.                                       | Più facile da implementare per contenuti non strutturati o variabili.                                  |
| **Flessibilità**      | Struttura rigida con righe e colonne definite.                                     | Flessibile e si adatta ai contenuti posizionati dinamicamente.                                         |
| **Prestazioni**      | Potenzialmente più performante poiché il layout è predefinito.                           | Può richiedere calcoli aggiuntivi del browser per la generazione della griglia implicita.                  |
| **Caso d’uso**         | Utile quando la struttura della griglia è prevedibile e definita in anticipo.                 | Utile per layout dinamici dove il contenuto è sconosciuto o cambia frequentemente.                 |
