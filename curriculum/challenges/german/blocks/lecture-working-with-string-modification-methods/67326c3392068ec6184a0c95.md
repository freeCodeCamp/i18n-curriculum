# --interactive--

Wenn Sie mit JavaScript arbeiten, können Sie auf Situationen stoßen, in denen Sie einen String eine bestimmte Anzahl von Malen wiederholen müssen.

Egal, ob Sie wiederholte Muster erzeugen oder einfach Text duplizieren möchten, die `repeat()`-Methode bietet eine einfache und effektive Möglichkeit, dies zu erreichen.

Die `repeat()`-Methode ist eine eingebaute Funktion in JavaScript, mit der Sie einen String eine festgelegte Anzahl von Malen wiederholen können. Hier ist die grundlegende Syntax:

```js
string.repeat(count);
```

`string` ist der String, den Sie wiederholen möchten, und `count` ist die Anzahl der Wiederholungen. Hier ein Beispiel:

:::interactive_editor

```js
let word = "Hello!";
let repeatedWord = word.repeat(3);
console.log(repeatedWord);  // "Hello!Hello!Hello!"
```

:::

In diesem Fall wird der String `Hello!` dreimal wiederholt, was zu `Hello!Hello!Hello!` führt.

Obwohl die `repeat()`-Methode nützlich ist, gibt es einige Ausnahmen und Einschränkungen, die Sie beachten sollten.

Der `count`-Parameter muss eine nicht-negative Zahl sein. Wenn Sie eine negative Zahl übergeben, wirft JavaScript einen `RangeError`.

```js
let word = "Test";
console.log(word.repeat(-1));  // Throws RangeError: Invalid count value
```

Der `count`-Parameter muss eine endliche Zahl sein. Wenn Sie versuchen, einen String unendlich oft zu wiederholen oder `Infinity` als Anzahl verwenden, erhalten Sie ebenfalls einen `RangeError`.

In JavaScript ist `Infinity` ein spezieller Wert, der eine unendliche Menge darstellt. Er wird verwendet, um Zahlen zu kennzeichnen, die größer sind als jede endliche Zahl.

```js
let word = "Test";
console.log(word.repeat(Infinity));  // Throws RangeError: Invalid count value
```

Wenn die Anzahl keine ganze Zahl ist (zum Beispiel eine Dezimalzahl wie `2.5`), rundet die `repeat()`-Methode sie auf die nächstkleinere ganze Zahl ab.

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(2.5));  // "TestTest"
```

:::

Wenn Sie `0` als Anzahl übergeben, gibt die `repeat()`-Methode einen leeren String zurück.

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(0));  // ""
```

:::

Die `repeat()`-Methode kann Aufgaben vereinfachen, die String-Duplikation beinhalten, und macht Ihren Code dadurch kürzer und besser lesbar.

Egal, ob Sie wiederholte Textmuster erzeugen oder einen Bereich mit Zeichen füllen möchten, `repeat()` erspart Ihnen das Schreiben von Schleifen oder komplexerem Code.

Sie sind nicht darauf beschränkt, eine Zahl direkt an die `repeat()`-Methode zu übergeben. Sie können auch eine Variable übergeben, die einen Zahlenwert speichert.

:::interactive_editor

```js
let count = 4;
let word = "Test";
let repeatedWord = word.repeat(count);
console.log(repeatedWord); // TestTestTestTest
```

:::

In diesem Beispiel speichert die Variable `count` die Anzahl der Wiederholungen. Das kann nützlich sein, wenn die Anzahl der Wiederholungen von Benutzereingaben oder anderen dynamischen Werten in Ihrem Programm abhängt.
