# --description--

У кожної мови програмування є спосіб виводити дані в термінал за допомогою вбудованого методу, функції, властивості або ключового слова. У Python ви можете використовувати функцію `print` для виведення даних у термінал. Давайте детальніше розглянемо функцію `print`, щоб ви могли впевнено почати її використовувати.

Одне з перших, що ви робите, коли вивчаєте будь-яку мову програмування — це написати просту програму `Hello world!`. У Python це дуже легко зробити, використовуючи лише функцію `print`.

Для цього потрібно просто помістити рядок `Hello world!` між відкриваючими та закриваючими круглими дужками, які ви використовуєте для виклику функції `print`:

```python
print('Hello world!') # Hello world!
```

Ви дізнаєтеся більше про рядки та функції в Python у майбутніх уроках. А поки просто вважайте рядки послідовністю символів, оточених або одинарними (`'`), або подвійними (`"`) лапками.

У прикладі `print('Hello world!')` рядок `'Hello world!'` є **аргументом**, який передається функції `print`. Ви також можете використовувати функцію `print` для одночасного показу кількох значень, або аргументів, розділяючи їх комами. Наприклад:

```python
print('My favorite colors are', 'blue', 'green', 'red')

# Output: My favorite colors are blue green red
```

Python автоматично додає пробіл між кожним елементом, коли ви розділяєте їх комами. Це зручно, коли потрібно вивести кілька частин інформації разом.
