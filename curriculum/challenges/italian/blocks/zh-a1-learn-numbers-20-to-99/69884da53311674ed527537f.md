# --description--

Esercitiamoci a scrivere alcuni numeri tra 70 e 79. Per completare questa attività, tieni presente che in cinese il numero 70 si chiama `七十 (qī shí)`.

I numeri da 0 a 9 sono:

| Numero | Cinese |
| :--- | :--- |
| 0 | `零 (líng)` |
| 1 | `一 (yī)` |
| 2 | `二 (èr)` |
| 3 | `三 (sān)` |
| 4 | `四 (sì)` |
| 5 | `五 (wǔ)` |
| 6 | `六 (liù)` |
| 7 | `七 (qī)` |
| 8 | `八 (bā)` |
| 9 | `九 (jiǔ)` |
