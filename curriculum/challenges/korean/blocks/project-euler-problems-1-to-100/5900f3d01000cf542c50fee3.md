# --description--

상자에 파란색 원판 15개와 빨간색 원판 6개로 구성된 총 21개의 색 원판이 들어 있고, 무작위로 두 개의 원판을 뽑았을 때 두 개 모두 파란색 원판일 확률을 구할 수 있습니다.

$${P(BB)} = \frac{15}{21}×\frac{14}{20} = \frac{1}{2}$$

두 개의 파란색 원판을 뽑을 확률이 정확히 50%인 다음 배열은 파란색 원판 85개와 빨간색 원판 35개가 들어 있는 상자입니다.

총 원판 수가 `limit`개를 초과하는 첫 번째 배열을 찾아, 그 상자에 들어 있는 파란색 원판의 개수를 구하세요.

```js
function arrangedProbability(limit) {
  // Based on https://www.mathblog.dk/project-euler-100-blue-discs-two-blue/
  let blue = 15;
  let discs = 21;

  while (discs < limit) {
    const nextBlue = 3 * blue + 2 * discs - 2;
    const nextDiscs = 4 * blue + 3 * discs - 3;

    blue = nextBlue;
    discs = nextDiscs;
  }
  return blue;
}
```
