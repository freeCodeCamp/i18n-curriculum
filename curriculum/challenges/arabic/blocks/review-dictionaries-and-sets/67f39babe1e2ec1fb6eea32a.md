## القواميس

- **القواميس**: القواميس هي هياكل بيانات مدمجة تخزن مجموعات من أزواج المفتاح-القيمة. يجب أن تكون المفاتيح من أنواع بيانات غير قابلة للتغيير. هذه هي الصيغة النحوية العامة لقاموس في Python:

```python
dictionary = {
    key1: value1,
    key2: value2
}
```

- **دالة البناء `dict()`**: دالة البناء `dict()` هي طريقة بديلة لبناء القاموس. تمرر قائمة من الصفوف المرتبة كمعلمة إلى دالة البناء `dict()`. تحتوي هذه الصفوف على المفتاح كالعنصر الأول والقيمة كالعنصر الثاني.

```python
pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
```

- **ترميز الأقواس**: للوصول إلى قيمة زوج المفتاح-القيمة، يمكنك استخدام الصيغة المعروفة بترميز الأقواس.

```python
dictionary[key]
```

## طرق القاموس الشائعة

- **طريقة `get()`**: تسترجع طريقة `get()` القيمة المرتبطة بمفتاح. تشبه ترميز الأقواس، لكنها تتيح لك تعيين قيمة افتراضية، مما يمنع حدوث أخطاء إذا لم يكن المفتاح موجودًا.

```python
dictionary.get(key, default)
```

- **طرائق `keys()` و `values()`**: تُرجع طرائق `keys()` و `values()` كائن عرض يحتوي على جميع المفاتيح والقيم في القاموس، على التوالي. كائن العرض هو طريقة لرؤية محتوى القاموس دون إنشاء نسخة منفصلة من البيانات.

```python
pizza = {
    'name': 'Margherita Pizza',
    'price': 8.9,
    'calories_per_slice': 250
}

pizza.keys()
# dict_keys(['name', 'price', 'calories_per_slice'])

pizza.values()
# dict_values(['Margherita Pizza', 8.9, 250])
```

- **طريقة `items()`**: تُرجع طريقة `items()` كائن عرض يحتوي على جميع أزواج المفتاح-القيمة في القاموس، بما في ذلك المفاتيح والقيم معًا.

```python
pizza.items()
# dict_items([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250)])
```

- **طريقة `clear()`**: تزيل طريقة `clear()` جميع أزواج المفتاح-القيمة من القاموس.

```python
pizza.clear()
```

- **طريقة `pop()`**: تزيل طريقة `pop()` زوج المفتاح-القيمة الذي يحدد مفتاحه كالمعلمة الأولى وتُرجع قيمته. إذا لم يكن المفتاح موجودًا، تُرجع القيمة الافتراضية المحددة كالمعلمة الثانية. إذا لم يكن المفتاح موجودًا ولم تُحدد قيمة افتراضية، يُثار خطأ `KeyError`.

```python
pizza.pop('price', 10)
pizza.pop('total_price') # KeyError
```

- **طريقة `popitem()`**: في Python 3.7 وما بعدها، تزيل طريقة `popitem()` آخر عنصر تم إدخاله.

```python
pizza.popitem()
```

- **طريقة `update()`**: تحدّث طريقة `update()` أزواج المفتاح-القيمة بأزواج مفتاح-قيمة من قاموس آخر. إذا كان هناك مفاتيح مشتركة، تُستبدل قيمها. تُضاف المفاتيح الجديدة إلى القاموس كأزواج مفتاح-قيمة جديدة.

```python
pizza.update({ 'price': 15, 'total_time': 25 })
```

## التكرار على قاموس

- **التكرار على القيم**: إذا كنت بحاجة إلى التكرار على القيم في قاموس، يمكنك كتابة حلقة `for` مع `values()` للحصول على جميع قيم القاموس.

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,
    'Headphones': 70,
}

for price in products.values():
    print(price)
```

المخرجات:

```md
990
600
250
70
```

- **التكرار على المفاتيح**: إذا كنت بحاجة إلى التكرار على المفاتيح في قاموس `products` أعلاه، يمكنك كتابة `products.keys()` أو `products` مباشرة.

```python
for product in products.keys():
    print(product)

# Or

for product in products:
    print(product)
```

المخرجات:

```md
Laptop
Smartphone
Tablet
Headphones
```

- **التكرار على أزواج المفتاح-القيمة**: إذا كنت بحاجة إلى التكرار على المفاتيح وقيمها المقابلة في نفس الوقت، يمكنك التكرار على `products.items()`. تحصل على صفوف مرتبة فردية تحتوي على المفاتيح وقيمها المقابلة.

```python
for product in products.items():
    print(product)
```

المخرجات:

```md
('Laptop', 990)
('Smartphone', 600)
('Tablet', 250)
('Headphones', 70)
```

لتخزين المفتاح والقيمة في متغيري حلقة منفصلين، يجب فصلهما بفاصلة. يخزن المتغير الأول المفتاح، والمتغير الثاني يخزن القيمة.

```python
for product, price in products.items():
    print(product, price)
```

المخرجات:

```md
Laptop 990
Smartphone 600
Tablet 250
Headphones 70
```

- **دالة `enumerate()`**: إذا كنت بحاجة إلى التكرار على قاموس مع تتبع عداد، يمكنك استدعاء دالة `enumerate()`. تُرجع الدالة كائن `enumerate` يعين عددًا صحيحًا لكل عنصر، مثل عداد. يمكنك بدء العداد من أي رقم، لكنه يبدأ افتراضيًا من 0.

تعيين الفهرس والعنصر إلى متغيري حلقة منفصلين هو الطريقة الشائعة لاستخدام `enumerate()`. على سبيل المثال، مع `products.items()`، يمكنك الحصول على زوج المفتاح-القيمة كاملًا بالإضافة إلى الفهرس:

```python
for index, product in enumerate(products.items()):
    print(index, product)
```

المخرجات:

```md
0 ('Laptop', 990)
1 ('Smartphone', 600)
2 ('Tablet', 250)
3 ('Headphones', 70)
```

لتخصيص القيمة الابتدائية للعداد، يمكنك تمرير معلمة ثانية إلى `enumerate()`. على سبيل المثال، هنا نبدأ العد من 1.

```python
for index, product in enumerate(products.items(), 1):
    print(index, product)
```

المخرجات:

```md
1 ('Laptop', 990)
2 ('Smartphone', 600)
3 ('Tablet', 250)
4 ('Headphones', 70)
```

## المجموعات

- **المجموعات**: المجموعات هي هياكل بيانات مدمجة في Python لا تسمح بالقيم المكررة. المجموعات قابلة للتغيير وغير مرتبة، مما يعني أن عناصرها لا تُخزن بترتيب محدد، لذلك لا يمكنك استخدام الفهارس أو المفاتيح للوصول إليها. أيضًا، يمكن للمجموعات أن تحتوي فقط على قيم من أنواع بيانات غير قابلة للتغيير، مثل الأعداد والسلاسل النصية والصفوف المرتبة.

- **تعريف مجموعة**: لتعريف مجموعة، اكتب عناصرها داخل أقواس متعرجة وافصل بينها بفواصل.

```python
my_set = {1, 2, 3, 4, 5}
```

- **تعريف مجموعة فارغة**: إذا كنت بحاجة إلى تعريف مجموعة فارغة، يجب استخدام دالة `set()`. كتابة أقواس متعرجة فارغة فقط ستنشئ قاموسًا تلقائيًا.

```python
set() # Set
{}    # Dictionary
```

## طرق المجموعات الشائعة

- **طريقة `add()`**: يمكنك إضافة عنصر إلى مجموعة باستخدام طريقة `add()`، مع تمرير العنصر الجديد كمعلمة.

```python
my_set.add(6)
```

- **طرائق `remove()` و `discard()`**: لإزالة عنصر من مجموعة، يمكنك استخدام طريقة `remove()` أو طريقة `discard()`، مع تمرير العنصر الذي تريد إزالته كمعلمة. تثير طريقة `remove()` خطأ `KeyError` إذا لم يُعثر على العنصر، بينما لا تفعل طريقة `discard()` ذلك.

```python
my_set.remove(4)
my_set.discard(4)
```

- **طريقة `clear()`**: تزيل طريقة `clear()` جميع العناصر من المجموعة.

```python
my_set.clear()
```

## عمليات المجموعات الرياضية

- **طرائق `issubset()` و `issuperset()`**: تتحقق طرائق `issubset()` و `issuperset()` مما إذا كانت مجموعة ما مجموعة جزئية أو مجموعة شاملة لمجموعة أخرى، على التوالي.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 5}

print(your_set.issubset(my_set)) # True
print(my_set.issuperset(your_set)) # True
```

- **طريقة `isdisjoint()`**: تتحقق طريقة `isdisjoint()` مما إذا كانت مجموعتان منفصلتين، أي لا تشتركان في عناصر.

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

print(my_set.isdisjoint(your_set)) # True
```

- **عامل الاتحاد (`|`)**: يعيد عامل الاتحاد `|` مجموعة جديدة تحتوي على جميع العناصر من المجموعتين.

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

my_set | your_set # {1, 2, 3, 4, 5, 6}
```

- **عامل التقاطع (`&`)**: يعيد عامل التقاطع `&` مجموعة جديدة تحتوي فقط على العناصر المشتركة بين المجموعتين.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set & your_set # {2, 3, 4}
```

- **عامل الفرق (`-`)**: يعيد عامل الفرق `-` مجموعة جديدة تحتوي على عناصر المجموعة الأولى التي ليست في المجموعات الأخرى.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set - your_set # {1, 5}
```

- **عامل الفرق المتماثل (`^`)**: يعيد عامل الفرق المتماثل `^` مجموعة جديدة تحتوي على العناصر الموجودة في المجموعة الأولى أو الثانية، ولكن ليس في كلتيهما.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set ^ your_set # {1, 5, 6}
```

- **عامل `in`**: يمكنك التحقق مما إذا كان عنصر موجودًا في مجموعة باستخدام عامل `in`.

```python
print(5 in my_set) # True
```

## مكتبة Python القياسية

- **مكتبة Python القياسية**: المكتبة توفر لك شفرة مكتوبة مسبقًا وقابلة لإعادة الاستخدام، مثل الدوال، الأصناف، وهياكل البيانات، التي يمكنك إعادة استخدامها في مشاريعك. لدى Python مكتبة قياسية واسعة تحتوي على وحدات مدمجة تنفذ حلولًا معيارية للعديد من المشاكل والمهام. بعض الأمثلة على الوحدات المدمجة الشهيرة هي `math`، `random`، `re` (اختصار لـ "التعبيرات النمطية"), و `datetime`.

## جملة الاستيراد

- **جملة الاستيراد**: للوصول إلى العناصر المعرفة في الوحدات المدمجة، تستخدم جملة استيراد. تُكتب جمل الاستيراد عادة في أعلى الملف. تعمل جمل الاستيراد بنفس الطريقة مع الدوال، الأصناف، الثوابت، المتغيرات، وأي عناصر أخرى معرفة في الوحدة.

- **جملة الاستيراد الأساسية**: يمكنك استخدام الكلمة المفتاحية `import` متبوعة باسم الوحدة:

```python
import module_name
```

بعدها، إذا كنت بحاجة إلى استدعاء دالة من تلك الوحدة، تستخدم ترميز النقطة، مع اسم الوحدة متبوعًا باسم الدالة.

```python
module_name.function_name()
```

على سبيل المثال، تكتب التالي في شفرتك لاستيراد وحدة `math` والحصول على الجذر التربيعي للعدد 36:

```python
import math

math.sqrt(36)
```

- **استيراد وحدة باسم مختلف**: إذا كنت بحاجة إلى استيراد الوحدة باسم مختلف (المعروف أيضًا باسم "اسم مستعار"), يمكنك استخدام `as` متبوعة بالاسم المستعار في نهاية جملة الاستيراد. يُستخدم هذا غالبًا لأسماء الوحدات الطويلة أو لتجنب تعارض الأسماء.

```python
import module_name as module_alias
```

على سبيل المثال، للإشارة إلى وحدة `math` باسم `m` في شفرتك، يمكنك تعيين اسم مستعار هكذا:

```python
import math as m
```

بعدها، يمكنك الوصول إلى عناصر الوحدة باستخدام الاسم المستعار:

```python
m.sqrt(36)
```

- **استيراد عناصر محددة**: إذا لم تكن بحاجة إلى كل شيء من الوحدة، يمكنك استيراد عناصر محددة باستخدام `from`. في هذه الحالة، تبدأ جملة الاستيراد بـ `from`، متبوعة باسم الوحدة، ثم الكلمة المفتاحية `import`، وأخيرًا أسماء العناصر التي تريد استيرادها.

```python
from module_name import name1, name2
```

بعدها، يمكنك استخدام هذه الأسماء بدون بادئة اسم الوحدة في شفرتك. على سبيل المثال:

```python
from math import radians, sin, cos

angle_degrees = 40
angle_radians = radians(angle_degrees)

sine_value = sin(angle_radians)
cos_value = cos(angle_radians)

print(sine_value) # 0.6427876096865393
print(cos_value)  # 0.766044443118978
```
 
هذا مفيد، لكنه قد يؤدي إلى تعارضات في الأسماء إذا كان لديك دوال أو متغيرات بنفس الاسم. ضع هذا في اعتبارك عند اختيار نوع جملة الاستيراد التي تريد استخدامها.

إذا كنت بحاجة إلى تعيين أسماء مستعارة لهذه الأسماء، يمكنك فعل ذلك أيضًا باستخدام الكلمة المفتاحية `as` متبوعة بالاسم المستعار.

```python
from module_name import name1 as alias1, name2 as alias2
```

- **جملة الاستيراد مع النجمة (`*`)**: النجمة تخبر Python أنك تريد استيراد كل شيء في تلك الوحدة، لكنك تريد استيراده بحيث لا تحتاج إلى استخدام اسم الوحدة كبادئة.

```python
from module_name import *
```

على سبيل المثال، إذا استخدمت هذا لاستيراد وحدة `math`، ستتمكن من استدعاء أي دالة معرفة في تلك الوحدة بدون تحديد اسم الوحدة كبادئة.

```python
from math import *
print(sqrt(36))  # 6.0
```

ومع ذلك، يُنصح عمومًا بتجنب هذا لأنه قد يؤدي إلى تصادم في أسماء النطاقات ويجعل من الصعب معرفة مصدر الأسماء.

## `if __name__ == '__main__'`

- **المتغير `__name__`**: `__name__` هو متغير مدمج خاص في Python. عندما يُنفذ ملف Python مباشرة، يعيّن Python قيمة هذا المتغير إلى السلسلة `"__main__"`. لكن إذا تم استيراد ملف Python كوحدة في سكربت Python آخر، تُعيّن قيمة المتغير `__name__` إلى اسم تلك الوحدة.

لهذا السبب غالبًا ما تجد هذا الشرط في سكربتات Python. يحتوي على الشفرة التي تريد تشغيلها **فقط** إذا كان سكربت Python يعمل كبرنامج رئيسي.

```python
if __name__ == '__main__': 
    # Code
```
