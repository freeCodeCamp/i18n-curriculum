# --description--

当你谈论别人时，可以用这个问题来询问他们的名字：

`¿Cómo se llama?` - 他/她叫什么名字？

单词 `cómo` 意思是“如何”。当这个词用来开头一个问题时，在字母 `o` 上写重音符号非常重要。

单词 `se` 表示你在谈论第三者。

Elena 正在和 Esteban 谈论他的新同事 Diego。
