# --description--

El radical de $n$, $rad(n)$, es el producto de los factores primos distintos de $n$. Por ejemplo, $504 = 2^3 × 3^2 × 7$, entonces $rad(504) = 2 × 3 × 7 = 42$.

Definiremos el triplete de enteros positivos (a, b, c) como un abc-hit si:

1. $GCD(a, b) = GCD(a, c) = GCD(b, c) = 1$
2. $a &lt; b$
3. $a + b = c$
4. $rad(abc) &lt; c$

Por ejemplo, (5, 27, 32) es un abc-hit, porque:

1. $GCD(5, 27) = GCD(5, 32) = GCD(27, 32) = 1$
2. $5 &lt; 27$
3. $5 + 27 = 32$
4. $rad(4320) = 30 &lt; 32$

It turns out that abc-hits are quite rare and there are only thirty-one abc-hits for $c &lt; 1000$, with $\sum{c} = 12523$.

Find $\sum{c}$ for $c &lt; 120000$.
