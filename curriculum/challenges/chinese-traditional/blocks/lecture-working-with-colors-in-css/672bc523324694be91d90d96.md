# --interactive--

在設計網站樣式時，使用 HSL 色彩模型是最具彈性且直覺的色彩處理方式之一。HSL 代表色相（Hue）、飽和度（Saturation）和明度（Lightness）——這三個關鍵元素定義了一種顏色。

HSL 色彩模型以更符合人類感知色彩的方式來表述顏色。在本課程中，我們將探討 HSL 色彩模型是什麼，以及 `hsl()` 函式在 CSS 中的使用方式。

色相是顏色的類型，以色輪上的角度表示，範圍從 `0` 到 `360` 度。舉例來說，`0` 度代表紅色，`120` 度是綠色，`240` 度是藍色。調整色相會讓你在色譜中切換不同的顏色。

飽和度指的是顏色的強度或純度。它以百分比表示，從 `0%`（完全去飽和的灰階色）到 `100%`（該顏色最鮮豔的狀態）。飽和度為 `100%` 的顏色是完全鮮豔的，而飽和度為 `0%` 的顏色則呈現灰色調。

明度決定顏色的明暗程度，同樣以百分比表示。明度值為 `0%` 時會產生黑色，`50%` 則是該色相的正常色調，`100%` 則會變成白色。

在 CSS 中，`hsl()` 函式用來使用 HSL 色彩模型定義顏色。現代語法使用空格分隔的數值：

```css
element {
  color: hsl(hue saturation lightness);
}
```

你也可能會在現有程式碼中看到舊版的逗號分隔語法，瀏覽器仍然支援：

```css
element {
  color: hsl(hue, saturation, lightness);
}
```

讓我們用一個範例來解析：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
body {
  background-color: hsl(0 0% 1% / 1);
}

p {
  color: hsl(120 100% 50%);
}
```

:::

在這個例子中，色相是 `120` 度，對應綠色。飽和度是 `100%`，所以綠色會是完全鮮豔的。明度是 `50%`，處於正常色調——既不太暗也不太亮。因此，段落的文字顏色會是明亮的綠色。

HSL 色彩模型的主要優點之一是它的直覺性。你可以輕鬆調整顏色的鮮豔度或明暗度，只需調整飽和度和明度的數值，而不必改變核心色相。

舉例來說，如果你想創造同一顏色的不同色階或色調，只要調整明度數值即可。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="light">This is a light blue div.</div>
<div class="dark">This is a dark blue div.</div>
```

```css
div.light {
  background-color: hsl(240 100% 80%);
}

div.dark {
  background-color: hsl(240 100% 20%);
  color: hsl(0 0% 100%);
}
```

:::

這裡，兩個 `div` 元素使用相同的色相（`240` 度，即藍色），但一個明度是 `80%`（較淺的藍色），另一個明度是 `20%`（較深的藍色）。

就像 RGB 模型一樣，`hsl()` 函式也支援選用的 alpha 值來控制透明度。你可以在 `/` 分隔符後面加上第四個參數。基本語法如下：

```css
element {
  background-color: hsl(hue saturation lightness / alpha);
}
```

我們來看一個範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: hsl(0 100% 50% / 0.5);
}
```

:::

這段程式碼會給 `div` 一個半透明的紅色背景，其中色相設定為 `0` 度（紅色），飽和度為 `100%`，明度為 `50%`，alpha 為 `0.5`（50% 不透明度）。你也可以用百分比指定 alpha——例如 `hsl(0 100% 50% / 50%)` 等同於 `hsl(0 100% 50% / 0.5)`。

你也常會在現有 CSS 中看到 `hsla()` 函式。這是最初用來在 HSL 顏色中包含透明度的方式，使用逗號分隔數值：

```css
element {
  background-color: hsla(hue, saturation, lightness, alpha);
}
```

舉例來說，`hsla(0, 100%, 50%, 0.5)` 是 `hsl(0 100% 50% / 0.5)` 的舊版等價寫法。兩者都被瀏覽器支援，但現代的 `hsl()` 語法搭配 `/` 分隔符現在是首選。

HSL 色彩模型特別適合在需要創建色彩結構綱目並輕鬆調整色階或色調時使用。

因為 HSL 將色相與飽和度和明度分開，你可以在調整陰影或高光的明度時，保持一致的色彩主題。這種彈性使 HSL 成為設計師和開發者的最愛。

此外，與十六進位碼或 RGB 等其他方法相比，HSL 讓色彩數值更易讀且更容易理解。你可以更清楚看出顏色背後的意圖，並根據需要調整，特別是在嘗試不同設計時。

使用 HSL 色彩模型，你可以更靈活地控制色調、飽和度和明度，使其成為在 CSS 中創造豐富且動態設計的強大工具。
