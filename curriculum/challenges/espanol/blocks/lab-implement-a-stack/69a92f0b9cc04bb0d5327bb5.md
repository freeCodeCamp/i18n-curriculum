# --description--

En este laboratorio, implementarás una estructura de datos pila usando funciones. Una pila es una estructura de datos LIFO (último en entrar, primero en salir) donde los elementos se agregan y se eliminan desde la parte superior.

**Objetivo:** Cumple las historias de usuario a continuación y haz que todas las pruebas pasen para completar el laboratorio.

**Historias de usuario:**

1. Debes tener una función `initStack` que devuelva un objeto con una propiedad `collection` establecida como un arreglo vacío.
1. Debes tener una función `push` que agregue un elemento a la parte superior de la pila.
1. Debes tener una función `pop` que elimine y devuelva el elemento superior de la pila, o `undefined` si no hay ninguno.
1. Debes tener una función `peek` que devuelva el elemento superior de la pila sin eliminarlo, o `undefined` si no hay ninguno.
1. Debes tener una función `isEmpty` que devuelva `true` si la pila no contiene elementos, y `false` en caso contrario.
1. Debes tener una función `clear` que elimine todos los elementos de la pila.

**Nota:** La mayoría de las pruebas dependen de las funciones `initStack` y `push`. Implementa primero estas funciones, ya que las pruebas para `pop`, `peek`, `isEmpty` y `clear` requieren agregar elementos a la pila.

# --solutions--

```js
function initStack() {
  return {
    collection: []
  };
}

function print(stack) {
  console.log(stack.collection);
}

function push(stack, element) {
  stack.collection.push(element);
}

function pop(stack) {
  return stack.collection.pop();
}

function peek(stack) {
  return stack.collection[stack.collection.length - 1];
}

function isEmpty(stack) {
  return stack.collection.length === 0;
}

function clear(stack) {
  stack.collection = [];
}
```
