# --description--

これは認定を取得するために必要なプロジェクトの1つです。このプロジェクトでは、ワールドカップの試合情報をPostgreSQLに入力するBashスクリプトを作成し、その後データベースから有用な統計情報をクエリします。

# --instructions--

**重要：** すべてのプロジェクトテストに合格した後、データベースのダンプを`worldcup.sql`ファイルに保存し、さらに`insert_data.sh`と`queries.sh`のファイルも保存してください。これによりステップ2を完了できます。仮想マシン内でその方法の指示があります。
