# --description--

У D3 є кілька методів, які дозволяють додавати та змінювати елементи у вашому документі.

Метод `select()` вибирає один елемент із документа. Він приймає аргумент — назву елемента, який ви хочете, і повертає HTML-вузол для першого елемента в документі, що відповідає цій назві. Ось приклад:

```js
const anchor = d3.select('a');
```

У наведеному прикладі знаходиться перший тег посилання на сторінці і зберігається HTML-вузол для нього у змінній `anchor`. Ви можете використовувати вибір з іншими методами. Частина `d3` у прикладі — це референс на об’єкт D3, через який ви отримуєте доступ до методів D3.

Ще два корисні методи — `append()` і `text()`.

Метод `append()` приймає аргумент — елемент, який ви хочете додати до документа. Він додає HTML-вузол до вибраного елемента і повертає посилання на цей вузол.

Метод `text()` або встановлює текст вибраного вузла, або отримує поточний текст. Щоб встановити значення, передайте рядок як аргумент у круглих дужках методу.

Ось приклад, який вибирає невпорядкований список, додає елемент списку і додає текст:

```js
d3.select('ul')?.append('li').text('Very important item');
```

D3 дозволяє ланцюжком викликати кілька методів через крапки, щоб виконати низку дій підряд.
