# --description--

`use`는 무언가를 특정한 목적에 맞게 사용하는 것을 의미합니다. 예를 들어:

`She uses a notebook to take notes in class.` - 이것은 그녀가 노트북을 글쓰기에 어떻게 활용하는지 보여줍니다.

`tool`는 일을 하는 데 도움이 되는 것입니다. 업무에서는 도구가 종종 소프트웨어나 컴퓨터 상의 무언가를 뜻합니다. 예를 들어:

`Zoom is a tool for online meetings.` - 이 경우 도구는 사람들이 연결하고 대화하는 데 도움을 주는 소프트웨어입니다.
