# --description--

令 $a_k$、$b_k$ 和 $c_k$ 表述該表達式 $\frac{1}{x} = {\left(\frac{k}{x} \right)}^2 (k + x^2) - kx$ 的三個解法（實數或複數）。

例如，對於 $k = 5$，我們看到 $\\{a_5, b_5, c_5\\}$ 約為 $\\{5.727244, -0.363622 + 2.057397i, -0.363622 - 2.057397i\\}$。

令 $S(n) = \displaystyle\sum_{p = 1}^n \sum_{k = 1}^n {(a_k + b_k)}^p {(b_k + c_k)}^p {(c_k + a_k)}^p$，其中所有整數 $p$、$k$ 滿足 $1 ≤ p, k ≤ n$。

有趣的是，$S(n)$ 總是整數。 例如，$S(4) = 51\\,160$。

尋找 $S({10}^6) \text{ 對 } 1\\,000\\,000\\,007$ 取模。
