# --description--

考虑所有满足 $0 ≤ a, b, c ≤ N$ 的格点 ($a$, $b$, $c$)。

从原点 $O(0, 0, 0)$ 向其他格点绘制所有直线。令 $D(N)$ 为这些不同直线的数量。

已知 $D(1\,000\,000) = 831\,909\,254\,469\,114\,121$。

查找 $D({10}^{10})$。将你的答案写成前 9 位数字，后跟后 9 位数字。
