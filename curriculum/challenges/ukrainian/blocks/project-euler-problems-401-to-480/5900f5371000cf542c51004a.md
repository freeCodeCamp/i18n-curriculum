# --description--

Гра з перевертанням — це гра для двох гравців, яка проходить на квадратній дошці розміром $N$ на $N$.

Кожна клітинка містить диск, у якого одна сторона біла, а інша — чорна.

Гра починається з того, що всі диски показують білу сторону.

Хід полягає у перевертанні всіх дисків у прямокутнику з такими властивостями:

- верхній правий кут прямокутника містить білий диск
- ширина прямокутника є ідеальним квадратом (1, 4, 9, 16, ...)
- висота прямокутника є трикутним числом (1, 3, 6, 10, ...)

<img alt="перевернути всі диски у прямокутнику 4x3 на дошці 5x5" src="https://cdn.freecodecamp.org/curriculum/project-euler/flipping-game-1.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Гравці ходять по черзі. Гравець виграє, якщо зробить усю сітку чорною.

Нехай $W(N)$ — це кількість виграшних ходів для першого гравця на дошці розміром $N$ на $N$, коли всі диски білі, за умови ідеальної гри.

$W(1) = 1$, $W(2) = 0$, $W(5) = 8$ і $W({10}^2) = 31\\,395$.

Для $N = 5$ вісім виграшних перших ходів першого гравця такі:

<img alt="вісім виграшних перших ходів для N = 5" src="https://cdn.freecodecamp.org/curriculum/project-euler/flipping-game-2.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Знайдіть $W({10}^6)$.
