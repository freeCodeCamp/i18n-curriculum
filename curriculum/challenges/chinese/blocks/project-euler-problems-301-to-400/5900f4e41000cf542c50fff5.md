# --description--

令 $S_n$ 为使用以下伪随机数生成器生成的整数序列：

$$\begin{align}
        S_0 & = 290\\,797 \\\\
  S_{n + 1} & = {S_n}^2\bmod 50\\,515\\,093
\end{align}$$

令 $A(i, j)$ 为数字 $S_i, S_{i + 1}, \ldots, S_j$ 中的最小值，条件是 $i ≤ j$。令 $M(N) = \sum A(i, j)$，其中 $1 ≤ i ≤ j ≤ N$。

我们可以验证 $M(10) = 432\,256\,955$ 和 $M(10\,000) = 3\,264\,567\,774\,119$。

查找 $M(2\,000\,000\,000)$。
