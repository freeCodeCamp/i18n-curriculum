# --explanation--

`Logs` sono registri di sistema che tracciano l'attività in una rete. Per esempio:

`The logs recorded a failed login attempt.` - Questo significa che i registri di sistema mostrano che qualcuno ha provato a effettuare l'accesso ma non ci è riuscito.  

`IP address` è un identificatore unico per un dispositivo su una rete. Per esempio:

`The attacker used a different IP address each time.` - L'attaccante ha usato un identificatore di rete unico per ogni tentativo.
