# --instructions--

Für diese Aufgabe müssen Sie einen Datengraph-Explorer mit Python erstellen, der Eingaben entgegennimmt und Folgendes ausführt:

- Eine .csv-Datei auf drei Arten erhalten
  - Hochladen von Ihrem lokalen Computer
  - Eingabe einer URL durch den Benutzer
  - Einfügen der URL direkt im Code
- Die Pandas-Bibliothek verwenden, um die .csv als DataFrame zu speichern
- Überschriften und die ersten zwei Zeilen ausgeben
- Die Spaltennamen als Liste speichern
- Eine oder zwei Spalten auswählen und die Daten in Numpy-Arrays umwandeln
- Die Daten als Streudiagramm oder Liniendiagramm darstellen
- Dies für verschiedene Spaltenkombinationen durchführen und die Graphen interpretieren können

Wenn Sie fertig sind, senden Sie die URL zum öffentlichen Colab-Notizbuch in Ihrem Google Drive ab.
