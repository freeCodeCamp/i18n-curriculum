# --description--

`Active` と `actively` はどちらも何かをすることに関係していますが、少し異なる方法です。`Active` は状態を示す形容詞で、今まさに起こっていることや、いつも忙しく活発であることを表します。例えば、誰かが active である場合、それは彼らがいつも何かをしていてじっとしていないことを意味します。`Actively` は対応する副詞で、エネルギーや努力をもって何かをすることを表します。これは動作の仕方を示します。

音声を聞いて、空欄に正しい形 `active` または `actively` を入れてください。
