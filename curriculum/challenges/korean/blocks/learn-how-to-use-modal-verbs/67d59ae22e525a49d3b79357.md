# --explanation--

`Could`와 `might`은 가능성을 설명할 때 쓰이는 조동사입니다.

`Could`는 어떤 일이 가능하지만 확실하지 않음을 나타냅니다. 예를 들어:

`If we don't fix this bug, it could cause a system crash.` - 시스템이 작동을 멈출 가능성이 있습니다.

`Might`는 `could`보다 가능성이 낮지만 여전히 가능함을 나타냅니다. 예를 들어:

`We might need to rewrite some parts of the code.` - 변경이 필요할 수도 있습니다.
