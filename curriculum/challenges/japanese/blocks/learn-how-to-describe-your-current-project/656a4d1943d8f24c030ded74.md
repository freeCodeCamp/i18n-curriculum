# --description--

誰かが否定文の最後に`either`を使うと、その状況が自分にも当てはまることを意味します。例えば：

`I don't like carrots. My sister doesn't like them either.` - ここで`either`は、話者とその妹の両方がニンジンが好きではないことを意味します。
