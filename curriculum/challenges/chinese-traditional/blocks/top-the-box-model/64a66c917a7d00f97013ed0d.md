# --description--

你需要理解的第一個重要概念是盒模型。它並不複雜，但如果現在跳過它，將來會讓你非常挫折。

網頁上的每一個東西都是一個矩形框。這些框可以包含其他框，並且可以並排放置。你可以透過在頁面上的每個條款上加上邊框來大致了解這是如何運作的，如下所示：

```css
* {
  border: 1px solid red;
}
```

# --assignment--

為你的頁面上的每個元素添加邊框，並查看這些方塊的佈局方式。
