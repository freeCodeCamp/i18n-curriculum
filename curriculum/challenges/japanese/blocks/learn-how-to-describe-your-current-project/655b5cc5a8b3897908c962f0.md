# --description--

`Cybersecurity` はコンピューターやオンライン情報を安全に保つことに関するものです。悪意のある人がコンピューターやインターネット上の情報を盗んだり、破壊したりするのを防ぎます。

例えば、メールを守るためにパスワードを使ったり、ウェブサイトがクレジットカード情報を安全に保護したりするのが `cybersecurity` です。コンピューターやインターネットを多くの人が使うため、とても重要です。
