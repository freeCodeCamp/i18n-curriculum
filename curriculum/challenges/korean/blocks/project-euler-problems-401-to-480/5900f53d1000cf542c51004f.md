# --description--

뫼비우스 함수는 $μ(n)$로 표기하며 다음과 같이 정의합니다:

- $n$이 제곱 인수 없는 경우(여기서 $ω(n)$은 $n$의 서로 다른 소인수 개수) $μ(n) = (-1)^{ω(n)}$
- $n$이 제곱 인수 없는 수가 아니면 $μ(n) = 0$

$P(a, b)$를 구간 $[a, b]$ 내에서 $μ(n) = 1$인 정수 $n$의 개수라고 합시다.

$N(a, b)$를 구간 $[a, b]$ 내에서 $μ(n) = -1$인 정수 $n$의 개수라고 합시다.

예를 들어, $P(2, 10) = 2$이고 $N(2, 10) = 4$입니다.

$C(n)$을 다음 조건을 만족하는 정수 쌍 $(a, b)$의 개수라고 합시다:

- $1 ≤ a ≤ b ≤ n$,
- $99 \times N(a, b) ≤ 100 \times P(a, b)$, 그리고
- $99 \times P(a, b) ≤ 100 \times N(a, b)$.

예를 들어, $C(10) = 13$, $C(500) = 16\,676$, 그리고 $C(10\,000) = 20\,155\,319$입니다.

$C(20\,000\,000)$을 구하세요.
