# --interactive--

웹페이지 스타일링을 위해 CSS를 사용할 때, 색상을 정의하는 가장 일반적인 방법 중 하나는 16진수 색상 값을 사용하는 것입니다. 16진수 색상 값, 즉 16진수 코드는 RGB 색상 모델에서 색상을 간결하게 나타내는 방법입니다. 이 수업에서는 16진수 코드가 무엇인지, CSS에서 어떻게 작동하는지, 그리고 언제 사용하는지 살펴보겠습니다.

16진수 코드(16진 코드의 줄임말)는 RGB 색상 모델에서 색상을 나타내는 데 사용되는 6자리 문자열입니다. "16진수"는 `0`부터 `9`까지의 숫자와 `A`부터 `F`까지의 문자를 사용하는 16진법 숫자 체계를 의미합니다.

색상의 맥락에서 16진수 코드는 특정 색상을 구성하는 빨강, 초록, 파랑(RGB)의 양을 지정합니다. 각 16진수 코드는 해시 기호(`#`)로 시작하며, 그 뒤에 빨강, 초록, 파랑의 강도를 정의하는 6개의 문자가 옵니다.

예를 들어, 검정색의 16진수 코드는 `#000000`이고, 흰색의 16진수 코드는 `#FFFFFF`입니다. 처음 두 문자는 빨강 값을 나타내고, 다음 두 문자는 초록, 마지막 두 문자는 파랑을 나타냅니다.

CSS에서는 16진수 코드를 사용해 요소에 색상을 적용할 수 있습니다. 기본 구문은 다음과 같습니다:

```css
element {
  color: #RRGGBB;
}
```

여기서 `#RRGGBB`은 실제 16진수 코드 자리 표시자입니다. `RR`, `GG`, `BB` 쌍은 각각 빨강, 초록, 파랑의 강도를 나타냅니다.

이 쌍은 `00` (가장 낮은 강도)부터 `FF` (가장 높은 강도)까지 범위를 가질 수 있습니다. 숫자가 클수록 최종 색상에 해당 색상이 더 많이 포함됩니다. 예를 들어, 몇 가지 16진수 색상을 요소에 적용해 보겠습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h1>This is a heading.</h1>
<p>This is a paragraph.</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}
p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

첫 번째 경우, `#FF5733`은 붉은 주황색입니다. 세부 내용은 다음과 같습니다:

- `FF`는 빨강의 최대값(완전 포화)입니다.
- `57`는 중간 정도의 초록입니다.
- `33`는 적은 양의 파랑입니다.

두 번째 경우, `#4CAF50`은 녹색 계열입니다:

- `4C`는 낮은 빨강 값입니다.
- `AF`는 강한 초록 값입니다.
- `50`는 중간 정도의 파랑 값입니다.

16진수 코드는 정밀하고 사용하기 쉬워 인기가 많습니다. 색상을 세밀하게 조정할 수 있어 일관성이 중요한 디자인에 적합합니다.

Adobe Photoshop이나 Figma 같은 대부분의 디자인 소프트웨어는 선택한 색상의 16진수 코드를 제공하므로, 이 값을 CSS 파일에 쉽게 복사해 붙여넣을 수 있습니다.

웹 개발에서는 텍스트 색상, 배경, 테두리 및 기타 시각적 요소를 정의할 때 16진수 코드를 자주 사용합니다. 예를 들어:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example</button>
```

```css
body {
  background-color: #F0F0F0; /* Light gray background */
}

button {
  background-color: #007BFF; /* A shade of blue */
  color: #FFFFFF; /* White text */
}
```

:::

이렇게 하면 웹사이트나 애플리케이션 전체에서 색상이 일관되게 유지됩니다.

경우에 따라 16진수 코드가 6자리 대신 3자리 축약형으로 쓰이는 것을 볼 수 있습니다. 이는 각 색상 쌍의 두 문자가 동일할 때 가능합니다. 예를 들어, `#FF5733`은 축약할 수 없지만, `#FFFFFF` (흰색)는 `#FFF`으로 쓸 수 있습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h2>This is a heading.</h2>
```

```css
body {
  background-color: #000;
}

h2 {
  color: #FFF; /* White in shorthand */
}
```

:::

축약형 `#FFF`는 `#FFFFFF`와 같습니다. 마찬가지로 `#000`는 검정을 나타내며, 전체 형식은 `#000000`입니다.

16진수 코드는 정밀도와 호환성 덕분에 인기가 많으며, 모든 브라우저에서 1,600만 가지 이상의 색상을 지원합니다. 특정 브랜드 색상을 정확히 지정하거나, 이름이 있는 색상이나 HSL로 쉽게 정의할 수 없는 정확한 색조를 제어해야 할 때, 또는 디자인 소프트웨어에서 기본적으로 제공하는 16진수 코드를 복사할 때 이상적입니다.
