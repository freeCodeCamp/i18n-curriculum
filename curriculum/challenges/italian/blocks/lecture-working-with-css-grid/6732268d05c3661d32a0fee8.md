# --interactive--

Nelle lezioni precedenti abbiamo lavorato con la proprietà `grid-template-columns` impostando il valore su alcune unità frazionarie.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

Anche se il codice seguente è completamente valido, esiste un modo più semplice per ripetere una sezione o tutte le tracce.

La funzione `repeat()` serve a ripetere una sezione o tutte le tracce per colonne o righe. Questa funzione prende in input un numero di ripetizioni e le tracce che vuoi ripetere.

Ecco una versione rivista dell’esempio precedente usando la funzione repeat:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

Non ci sarà alcun cambiamento nello stile nel browser, ma questo è un modo più conciso per scrivere valori ripetuti per le colonne.

La funzione `repeat()` accetta qualsiasi pattern valido che puoi usare per righe o colonne.

Ecco un esempio che usa la funzione `repeat()` per impostare la prima e la terza colonna a 20 pixel e la seconda e la quarta colonna a un’unità frazionaria.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(2, 20px 1fr);
  column-gap: 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

A volte potresti preferire scrivere ogni singolo valore invece di usare la funzione `repeat()`. Ma ci sono momenti in cui questa funzione è molto utile, specialmente quando vuoi ripetere un pattern particolare per una lista di tracce.
