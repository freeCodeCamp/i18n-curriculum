# --description--

A palavra `took` é o que você usa em vez de `take` quando está falando sobre algo que aconteceu no passado, como ontem ou no ano passado.

A maioria dos verbos recebe `-ed` adicionado para o tempo passado, como `play` que se torna `played`. Mas `take` é diferente; muda para `took`.
