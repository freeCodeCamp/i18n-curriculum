# --description--

Fetch APIはウェブアプリがネットワークリクエストを行い、通常はサーバーからデータを取得したり送信したりするための仕組みです。このAPIは`fetch()`メソッドを提供しており、これを使ってリクエストを行えます。`fetch`の基本的な使い方を見てみましょう。

```js
fetch('https://api.example.com/data')
```

この例では、`GET`に対して`https://api.example.com/data`リクエストを行っています。これにより、JSON形式に変換してどこでも使えるデータが返されます。

Fetch APIはデフォルトでは`GET`メソッドを使ってデータを取得します。これは次のレッスンで、他の一般的なHTTPリクエストメソッドとともに説明します。

では、ネットワークから取得される一般的なリソースの種類について話しましょう。

ウェブアプリでは、天気データ、職業リストデータ、国名リスト、国コードや国旗のアイコン・画像などの共通データが必要です。これらのデータを使うことで、アプリをより情報豊かでインタラクティブにできます。Fetch APIのおかげで、これらのリソースをネットワークから取得できます。

画像はよく取得されるリソースの一つです。fetchを使って画像を静的に、またはユーザーの操作に応じて動的に読み込み、ウェブアプリに表示することがあります。

テキストファイルもよく取得されるリソースの一種です。設定ファイルやログファイル、あるいはウェブページに表示したい文書全体などが含まれます。

場合によっては、音声や動画ファイルを取得することもあります。Fetch APIはこれらのリソースも扱えるため、多様なデータタイプを扱うことが可能です。
