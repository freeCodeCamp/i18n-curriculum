# --description--

D3 有數個方法讓你為你的文件添加和更改元素。

`select()` 方法從文件中選擇一個元素。它接受一個用於你想要的元素名稱的引數，並傳回文件中第一個符合該名稱的 HTML 節點。以下是一個範例：

```js
const anchor = d3.select('a');
```

上述範例會尋找頁面上的第一個錨點標籤，並將其 HTML 節點存檔到變數 `anchor`。你可以將此選取與其他方法一起使用。範例中的 `d3` 部分是對 D3 物件的引用，這是你存取 D3 方法的方式。

另外兩個有用的 `append()` 和 `text()` 方法。

`append()` 方法接受一個引數，該引數是你想要添加到文件中的元素。它會將一個 HTML 節點附加到選取的項目，並傳回該節點的控制代碼。

`text()` 方法會設定所選節點的文字，或取得目前的文字。若要設定值，請在方法的小括弧內傳入字串作為引數。

這是一個範例，選取一個無序列表，附加一個列表項目，並添加文字：

```js
d3.select('ul')?.append('li').text('Very important item');
```

D3 允許你用句點串鏈多個方法，以連續執行多個動作。

# --instructions--

使用 `select` 方法選取文件中的 `body` 標記。然後對它 `append` 一個 `h1` 標記，並將文字 `Learning D3` 添加到 `h1` 元素中。

# --hints--

`body` 應該有一個 `h1` 元素。

```js
const body = document.querySelector('body');
const headers = body?.querySelectorAll('h1');
assert.lengthOf(headers, 1);
```

`h1` 元素中應該包含文字 `Learning D3`。

```js
assert.strictEqual(document.querySelector('h1')?.textContent, 'Learning D3');
```

你的程式碼應該存取 `d3` 物件。

```js
assert.match(code, /d3/g);
```

你的程式碼應該使用 `select` 方法。

```js
assert.match(code, /\.select/g);
```

你的程式碼應該使用 `append` 方法。

```js
assert.match(code, /\.append/g);
```

你的程式碼應該使用 `text` 方法。

```js
assert.match(code, /\.text/g);
```
