# --description--

Se você estiver usando seu dispositivo móvel para navegar pelo currículo de chinês, pode digitar Pinyin e caracteres chineses diretamente usando os inputs personalizados do freeCodeCamp. No entanto, se quiser usar o teclado chinês no seu dispositivo, aqui está como configurá-lo.

## Adicionando Teclados de Idiomas

### No Android

**NOTA:** O caminho do Android não é universal e pode variar dependendo do dispositivo.

1. Abra Configurações.

2. Vá para Sistema (ou Gerenciamento Geral, dependendo do dispositivo).

3. Toque em Configurações do Teclado (ou Idiomas e entrada).

4. Toque em Languages and types (ou Add keyboard).
 
5. Ative o Chinês (Simplificado ou CN) ou faça o download se necessário.

6. Salve ou toque em Done, se solicitado.

### No iPhone

1. Abra Configurações.

2. Vá para Geral > Teclado.

3. Teclados de Toque.

4. Toque em Adicionar Novo Teclado.

5. Selecione Chinês (simplificado).

6. Use o ícone de globo no teclado para alternar idiomas.

## Tentando Teclados de Terceiros

Você também pode instalar aplicativos de teclado de terceiros no seu telefone ou tablet para facilitar a digitação em chinês. Esses teclados geralmente vêm com recursos úteis como previsões inteligentes de caracteres e suporte para vários métodos de entrada, como Pinyin ou escrita manual.
