# --interactive--

محددات الصنف جزء أساسي من CSS، تتيح للمطوِّرين استهداف عدة عناصر تحمل نفس خاصية الصنف وتطبيق تنسيق متناسق.

هذا يجعلها متعددة الاستخدامات وفعالة جدًا لتطبيق التنسيقات عبر موقع الويب.

تُعرَّف محددات الصنف بنقطة (`.`) تليها اسم الصنف. يمكن تطبيقها على أي عنصر في وثيقة HTML.

إليك مثالًا يستخدم محدد صنف:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="highlight">Example paragraph</p>
```

```css
.highlight {
  color: green;
}
```

:::

في هذا المثال، أي عنصر يحمل الصنف `highlight` سيُضبط لون نصه على `green`.

لمحدِّدات الصنف خصوصية أعلى من محدِّدات النوع لكنها أقل من محدِّدات المعرف والأنماط المضمَّنة.

قيمة التخصيص لمحدِّد الصنف هي `(0, 0, 1, 0)`. هذا يعني أن محددات الصنف يمكنها تجاوز محددات النوع، لكنها يمكن أن تُتجاوز بواسطة محددات المعرف والأنماط المضمَّنة.

يمكن دمج محددات الصنف مع محددات أخرى لإنشاء قواعد أكثر تحديدًا.

إليك مثالًا يجمع بين محدد نوع الفقرة ومحدد صنف:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="bold-text">Example paragraph</p>
<p class="bold-text">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p.bold-text {
  font-weight: bold;
}
```

:::

تنطبق هذه القاعدة فقط على عناصر `p` التي تحمل أيضًا الصنف `bold-text`، مما يجعل نصها عريضًا.
