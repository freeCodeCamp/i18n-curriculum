# --description--

Wie zuvor beginnen Sie mit der Validierung der Eingabe innerhalb der `swapCrewMembers` Funktion. Insbesondere müssen Sie die Indizes der auszutauschenden Astronauten validieren.

Wenn eine der folgenden Bedingungen zutrifft:

- `fromIndex` ist negativ

- `fromIndex` ist größer oder gleich `crew.length`

- `toIndex` ist negativ

- `toIndex` ist größer oder gleich `crew.length`

dann sollten Sie `"Invalid crew indices"` in der Konsole ausgeben und `return` aufrufen, um die Funktion zu beenden.

Sie können den ODER-Operator `||` verwenden, um mehrere Bedingungen gleichzeitig zu prüfen:

```js
if (condition1 || condition2 || condition3 || condition4) {
  console.log("Message");
  return;
}
```
