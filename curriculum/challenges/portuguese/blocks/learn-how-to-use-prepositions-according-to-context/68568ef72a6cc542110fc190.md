### --feedback--

Esta palavra mostra possibilidade — algo não é certo mas pode acontecer ou ser necessário.

# --explanation--

`Any` é usado para se referir a um ou mais de algo, sem ser específico. Por exemplo:

`Let me know if you have any questions.` – Isso significa uma ou mais possíveis perguntas.

`Might` indica que algo é possível, mas não definitivo. Por exemplo:

`We might need more time to finish.` – Isso significa que pode ser necessário tempo extra.
