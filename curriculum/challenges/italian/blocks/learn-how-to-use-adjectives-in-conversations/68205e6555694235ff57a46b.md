# --explanation--

`Minimal` significa solo una quantità molto piccola. Appena sufficiente per soddisfare un bisogno. Per esempio:

`The task required minimal effort.` – Questo significa che è stato molto facile e non ha richiesto molta energia.

`Setup time` è il tempo necessario per preparare qualcosa all'uso. Per esempio:

`The software has a short setup time.` – Questo significa che non ci vuole molto tempo per renderlo pronto all'uso.
