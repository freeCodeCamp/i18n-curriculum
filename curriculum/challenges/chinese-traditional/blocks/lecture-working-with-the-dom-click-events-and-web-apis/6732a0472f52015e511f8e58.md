# --description--

讓我們來了解 API 以及它們為何對網頁開發至關重要。API 代表 Application Programming Interface。API 建立一套規則和通訊協定，使軟體應用程式能夠相互溝通並高效交換資料。

你可以將它們視為構件，允許開發者基於已實作的較簡單建置區塊來創建更複雜的功能。

API 有各種不同的型別。Web API 是專門為網頁應用程式設計的。針對客戶端 JavaScript 開發，有不同型別的 Web API。它們並非 JavaScript 本身的一部分。

這些型別的 API 通常分為兩大類別：瀏覽器 API 和第三方 API。

瀏覽器 API 揭露來自瀏覽器的資料。作為網頁開發者，你可以使用 JavaScript 存取並操作這些資料。

它們也提供存取各種功能的能力，例如操作網站的結構、處理事件、使用儲存空間，以及與網路通訊。

一些常用的瀏覽器 API 範例包括：

DOM API，可用來操作 HTML 元素、它們的樣式和屬性。你將在接下來的課程中學到更多關於 DOM 的知識。它是網頁開發中的核心概念。

Storage API，用於在使用者的裝置上區域地儲存資料。

還有更多。 

有各種瀏覽器 API 可供你用來控制代碼使用者互動，並從瀏覽器取得更多資訊，例如使用者的當前位置。 

你也可以處理其他類型的元素，例如影片和音訊。

瀏覽器 API 是建置強大網頁應用程式的有用工具。它們已經內建於瀏覽器中。

但你也可以使用第三方 API，這些 API 並非瀏覽器預設內建。要使用它們，你需要以某種方式取得它們的程式碼。通常，它們會附帶詳細的文件，說明如何使用它們的服務。

一個範例是 Google Maps API，你可以用它在你的網站上顯示互動式地圖。

有各種第三方網路 API 型別，包括天氣 API、社群媒體 API、支付 API、資料 API、翻譯 API 等。你只需要找到符合你需求的 API。

Web API 是建置現代網頁應用程式不可或缺的工具，能與各種服務和資料來源互動。透過學習如何善用它們的強大功能，你將擴展用於創建動態且引人入勝網站的工具集。
