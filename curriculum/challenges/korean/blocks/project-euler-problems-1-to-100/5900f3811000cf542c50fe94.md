# --description--

d(`n`)를 `n`의 진약수 합으로 정의합니다 (`n`보다 작은 수 중 `n`을 나누어떨어지게 하는 수).

만약 d(`a`) = `b`이고 d(`b`) = `a`이며, `a` ≠ `b`라면 `a`과 `b`은 친구수 쌍이고 `a`와 `b` 각각을 친구수라고 합니다.

예를 들어, 220의 진약수는 1, 2, 4, 5, 10, 11, 20, 22, 44, 55, 110입니다; 따라서 d(220) = 284입니다. 284의 진약수는 1, 2, 4, 71, 142이므로 d(284) = 220입니다.

`n` 미만의 모든 친구수의 합을 구하세요.

```js
const sumAmicableNum = (n) => {
  const fsum = (n) => {
    let sum = 1;
    for (let i = 2; i <= Math.floor(Math.sqrt(n)); i++)
      if (Math.floor(n % i) === 0)
        sum += i + Math.floor(n / i);
    return sum;
  };
  let d = [];
  let amicableSum = 0;
  for (let i=2; i<n; i++) d[i] = fsum(i);
  for (let i=2; i<n; i++) {
    let dsum = d[i];
    if (d[dsum]===i && i!==dsum) amicableSum += i+dsum;
  }
  return amicableSum/2;
};
```
