# --description--

في Express، يُستخدم كائن `res` (الاستجابة) لإرسال الاستجابات إلى العميل بعد معالجة الطلبات الواردة. يوفر Express عدة طرق مدمجة لإرسال أنواع مختلفة من الاستجابات. على سبيل المثال، يمكنك إرسال نص عادي، بيانات JSON، إعادة توجيه المستخدم إلى عنوان URL آخر، أو عرض قالب.

واحدة من أكثر طرق الاستجابة استخدامًا هي `res.send()`:

```javascript
app.get("/text", (req, res) => {
  res.send("Hello, World!");
});
```

تُرسل طريقة `res.send()` استجابة إلى العميل. يمكنها إرسال نص عادي، HTML، JSON، أو حتى بيانات ثنائية. في هذا المثال، عندما يزور العميل `/text`، يرد الخادم بالنص `Hello, World!`.

طريقة استجابة أخرى شائعة الاستخدام هي `res.json()`.

ترسل هذه الطريقة استجابة JSON إلى العميل. هي مفيدة بشكل خاص عند بناء واجهات برمجة التطبيقات (APIs)، حيث تحتاج غالبًا إلى إرجاع بيانات منظمة مثل الكائنات أو المصفوفات. يقوم Express تلقائيًا بتحويل كائن JavaScript إلى سلسلة نصية بصيغة JSON:

```javascript
app.get("/user", (req, res) => {
  res.json({ name: "John Doe", age: 30 });
});
```

إذا أرسل العميل طلبًا إلى `/user`، يرد الخادم بما يلي:

```json
{
  "name": "John Doe",
  "age": 30
}
```

عندما تحتاج إلى التحكم في رمز حالة HTTP للاستجابة، يمكنك استخدام `res.status()`. تحدد هذه الطريقة رمز الحالة وعادةً ما تُسلسل مع طريقة استجابة أخرى مثل `res.send()` أو `res.json()`:

```javascript
app.get("/notfound", (req, res) => {
  res.status(404).send("Not Found");
});
```

هنا، يحدد الخادم رمز الحالة إلى `404` ويرسل الرسالة `Not Found` إلى العميل.

إذا أردت إرسال المستخدم إلى عنوان URL مختلف، يمكنك استخدام `res.redirect()`:

```javascript
app.get("/oldpage", (req, res) => {
  res.redirect("/newpage");
});
```

عندما يزور العميل `/oldpage`، يتم إعادة توجيهه تلقائيًا إلى `/newpage`.

أخيرًا، تُستخدم `res.render()` عند العمل مع محركات القوالب مثل Pug أو EJS أو Handlebars. تقوم بعرض قالب وإرسال HTML الناتج إلى العميل:

```javascript
app.get("/profile", (req, res) => {
  res.render("profile", { username: "john_doe" });
});
```

في هذا المثال، يعرض الخادم قالب `profile` ويمرر إليه متغير `username`، الذي يمكن عرضه لاحقًا في HTML.

تتيح لك هذه الطرق في الاستجابة إرجاع أنواع مختلفة من البيانات بناءً على طبيعة الطلب. سواء كنت ترسل نصًا، بيانات JSON، أو تعيد توجيه العميل، لدى Express مجموعة مرنة من الأدوات للتعامل مع ذلك.
