# --description--

`So far` significa até este momento ou até agora. É usado quando se fala de um tempo que começou no passado e continua até o presente. Se alguém perguntar, `Are you happy with the workplace so far?`, quer saber se você gostou do seu local de trabalho desde o primeiro dia até hoje.

Outro exemplo: se você disser, `The project is good so far`, significa que desde o início do projeto até agora, tudo está bom.

No diálogo, Sarah pergunta a Tom se ele está satisfeito com seu novo emprego até agora. Ela está perguntando sobre o tempo dele no emprego desde o início até a conversa deles.
