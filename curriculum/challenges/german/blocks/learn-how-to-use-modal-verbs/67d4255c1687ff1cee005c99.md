## --text--

Welcher Satz ist grammatikalisch korrekt?

## --answers--

`Everyone must to follow the security guidelines.`

### --feedback--

`Must` wird vom Grundverb ohne `to` gefolgt.

---

`Everyone must following the security guidelines.`

### --feedback--

`Must` sollte vom Grundverb gefolgt werden, nicht von der `-ing`-Form.

---

`Everyone must followed the security guidelines.`

### --feedback--

`Must` wird vom Grundverb gefolgt, nicht von der Vergangenheitsform.

---

`Everyone must follow the security guidelines.`

## --video-solution--

4

# --explanation--

Ein Modalverb wird immer vom Grundform des Verbs gefolgt. Das Modalverb nimmt kein `to`, `-ing` oder Vergangenheitsformen an. Hier sind Beispiele für die falschen Verbformen:

- `Must to follow` ist falsch, weil `must` `to` verlangt.

- `Must following` ist falsch, weil `must` von einer `-ing`-Form gefolgt wird.

- `Must followed` ist falsch, weil `followed` in der Vergangenheitsform steht.
