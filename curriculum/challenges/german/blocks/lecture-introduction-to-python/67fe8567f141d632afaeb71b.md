# --description--

In der letzten Lektion haben Sie gelernt, was Python ist und was Sie damit machen können. Jetzt schauen wir uns an, wie Sie Python auf Ihrem lokalen Rechner einrichten.

Der einfachste Weg, Python auf Windows und Mac zu installieren, ist, den Installer von der offiziellen Python-Website herunterzuladen. Später in dieser Lektion gehen wir auch darauf ein, wie man Python unter Linux ausführt.

Gehen Sie zu `https://www.python.org/` und fahren Sie mit der Maus über „Downloads“. Ein Modalfenster erscheint, das die aktuelle Version von Python für Ihr Betriebssystem (OS) anzeigt.

Zuerst erklären wir, wie Sie Python auf einem Computer mit macOS installieren:

- Klicken Sie auf die Schaltfläche mit der aktuellen Python-Version (aus dem vorherigen Modal), und der Download einer `.pkg`-Installationsdatei startet automatisch.
- Sobald der `.pkg`-Installer heruntergeladen ist, öffnen Sie ihn und klicken im sich öffnenden Fenster auf „Weiter“.
- Klicken Sie weiter auf die Schaltfläche „Weiter“, bis Sie zum Abschnitt „Installationstyp“ gelangen. Dort klicken Sie auf die Schaltfläche „Installieren“.
- Geben Sie bei Bedarf Ihr Passwort ein und starten Sie die Installation.
- Danach sollten Sie eine Glückwunschnachricht erhalten, dass Python erfolgreich installiert wurde.
- Klicken Sie auf die Schaltfläche „Schließen“, und Sie sind fertig!

Sie können die Installation überprüfen, indem Sie Ihr Terminal öffnen und `python --version` oder `python3 --version` ausführen.

Sie können den Python-Interpreter auch öffnen, indem Sie `python` oder `python3` im Terminal ausführen.

Ein <dfn>Terminal</dfn> ist eine textbasierte Schnittstelle, mit der Sie durch Eingabe von Befehlen mit Ihrem Computer interagieren können. Jedes Betriebssystem hat eine Standard-Terminal-App. Unter macOS können Sie die Terminal-App verwenden. Unter Windows können Sie die Eingabeaufforderung oder PowerShell nutzen. Unter Linux hat jede Desktop-Umgebung ihre eigene Standard-Terminal-App, wie GNOME Terminal oder Konsole.

Beachten Sie, dass auf einigen älteren macOS- und Linux-Systemen `python` für Python 2 reserviert sein kann, während `python3` speziell für Python 3 ist. Wenn Sie `python --version` ausführen und eine Python-2-Version wie `Python 2.7.18` sehen, ist es möglich, dass Ihr Betriebssystem auf Software angewiesen ist, die in der älteren Python-Version geschrieben wurde. In diesem Fall sollten Sie `python3` verwenden, um Ihren Python-Code künftig auszuführen. Python 2 ist am Ende seines Lebenszyklus und sollte für keine neue Entwicklung mehr verwendet werden.

Um Python unter Windows zu installieren, gehen Sie wie folgt vor:

- Gehen Sie zu `https://www.python.org/` und fahren Sie mit der Maus über „Downloads“. Es sollte ein Modal erscheinen, das „Download für Windows“ anzeigt, sowie eine Download-Schaltfläche mit der aktuellen Python-Version.
- Klicken Sie auf die Versionsnummer, und der Download einer Windows-Executable (`.exe`) startet automatisch.
- Sobald Sie den Python-Installer für Windows heruntergeladen haben, doppelklicken Sie darauf und folgen den Anweisungen.
- Wenn Sie die Option `Add python.exe to Path` sehen, aktivieren Sie diese Option und klicken dann auf `Install Now`. Das macht es Ihnen später leichter.


Sie können die Installation überprüfen, indem Sie eine Befehlszeilenschnittstelle wie PowerShell öffnen und `python --version` ausführen. Sie können den Python-Interpreter auch öffnen, indem Sie `python` ausführen.

Für Python unter Linux sind die meisten großen Distributionen wie Ubuntu, Debian und Fedora bereits mit Python ausgestattet.

Öffnen Sie einfach ein Terminal und führen Sie `python --version` oder `python3 --version` aus:

Wenn keiner der Befehle eine Python-Version anzeigt, können Sie nach einem Installationspaket für Ihre Linux-Variante bei `https://www.python.org` suchen oder online nach der empfohlenen Methode zur Installation von Python für Ihre Distribution suchen.
