# --description--

يُستخدم زمن **المضارع التام المستمر** للحديث عن أفعال أو حالات بدأت في الماضي وما زالت مستمرة حتى الآن. هذا الزمن مفيد جدًا عندما تريد التأكيد على مدة النشاط أو أنه عملية مستمرة.

إليك كيفية صياغة **زمن المضارع التام المستمر (Present Perfect Continuous)**: الفاعل + `has/have been` + صيغة الفعل مع `-ing`. غالبًا ما يُستخدم هذا الزمن مع تعبيرات الزمن مثل `for` و`since`. تُستخدم `For` مع مدة زمنية مثل سنتين أو ستة أشهر، بينما تُستخدم `since` مع نقطة زمنية محددة مثل 2018 أو هذا الصباح.

أمثلة:

`I have been studying English for two years.` هذا يعني أنني بدأت دراسة اللغة الإنجليزية منذ سنتين وما زلت أدرسها حتى الآن.

`She has been working here since 2018.` هذا يعني أنها بدأت العمل هنا في عام 2018 وما زالت تعمل هنا.

استمع إلى براين واملأ الفراغات.
