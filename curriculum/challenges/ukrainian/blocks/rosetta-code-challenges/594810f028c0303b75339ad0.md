# --description--

Дано масив із багатьох ліній, де поля в межах лінії розділені одним символом `$`. Напишіть програму, яка вирівнює кожну колонку полів, забезпечуючи, щоб слова в кожній колонці були розділені принаймні одним пробілом. Крім того, дозвольте кожному слову в колонці бути вирівняним ліворуч, праворуч або по центру в межах своєї колонки.

# --instructions--

Використайте наступний текст для тестування своїх програм:

```js
const testText = [
  'Given$a$text$file$of$many$lines',
  'where$fields$within$a$line$',
  'are$delineated$by$a$single$"dollar"$character',
  'write$a$program',
  'that$aligns$each$column$of$fields',
  'by$ensuring$that$words$in$each$',
  'column$are$separated$by$at$least$one$space.',
  'Further,$allow$for$each$word$in$a$column$to$be$either$left$',
  'justified,$right$justified',
  'or$center$justified$within$its$column.'
];
```

**Зверніть увагу, що:**

- Лінії вхідного тексту прикладу можуть мати або не мати кінцеві символи долара.
- Всі колонки повинні мати однакове вирівнювання.
- Послідовні пробіли, які з’являються поруч із кінцем ліній, не мають значення для виконання завдання.
- Вихідний текст буде переглядатися у моноширинному шрифті в простому текстовому редакторі або базовому терміналі. Лінії в ньому повинні бути з’єднані символом нового рядка (`\n`).
- Мінімальна кількість пробілів між колонками має обчислюватися з тексту, а не бути жорстко закодованою.
- Не обов’язково додавати розділові символи між або навколо колонок.

Наприклад, одна з ліній із `testText`, після вирівнювання праворуч, ліворуч і по центру відповідно:

```js
'    column        are separated     by     at    least       one space.\n'
'column     are        separated by     at     least    one       space.\n'
'  column      are     separated   by     at    least      one    space.\n'
```

```js
String.prototype.repeat = function (n) { return new Array(1 + parseInt(n)).join(this); };

function formatText(input, justification) {
  let x, y, max, cols = 0, diff, left, right;
  for (x = 0; x < input.length; x++) {
    input[x] = input[x].split('$');
    if (input[x].length > cols) {
      cols = input[x].length;
    }
  }
  for (x = 0; x < cols; x++) {
    max = 0;
    for (y = 0; y < input.length; y++) {
      if (input[y][x] && max < input[y][x].length) {
        max = input[y][x].length;
      }
    }
    for (y = 0; y < input.length; y++) {
      if (input[y][x]) {
        diff = (max - input[y][x].length) / 2;
        left = ' '.repeat(Math.floor(diff));
        right = ' '.repeat(Math.ceil(diff));
        if (justification === 'left') {
          right += left; left = '';
        }
        if (justification === 'right') {
          left += right; right = '';
        }
        input[y][x] = left + input[y][x] + right;
      }
    }
  }
  for (x = 0; x < input.length; x++) {
    input[x] = input[x].join(' ');
  }
  input = input.join('\n');
  return input;
}
```
