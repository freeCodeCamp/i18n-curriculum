# --description--

`infographic` はチャートや図のような視覚的なイメージで、情報を表すために使われます。例えば：

`The infographic shows the survey results clearly.` - ここで、`infographic` はデータをわかりやすく説明する画像を意味します。

`interactive infographic` は、クリックやホバー、スクロールで情報を探れるタイプのインフォグラフィックです。例えば：

`The company made an interactive infographic where readers can click to see details about each product.` - ここで、`interactive infographic` は情報を表示するだけでなく、読者が操作できる視覚的なものを意味します。
