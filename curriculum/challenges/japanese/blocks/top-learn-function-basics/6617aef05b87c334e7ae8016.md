# --description--

これまでに見てきたように、関数は`return`キーワードを使って値を返すことができます。`return`キーワードは関数から値を返すために使われます。`return`キーワードが使われると、関数の実行は停止し、`return`キーワードの後に指定された値が返されます。

```js
function add(a, b) {
  return a + b
}

console.log(add(2, 3)); // Output: 5
```

しかし、関数の最後の前に`return`キーワードが使われたらどうなるでしょうか？
この疑問に答えるために、次の例を考えてみましょう。

```js
function add(a, b) {
  if(a > 2){
    return b;
  }

  return a + b;
}

console.log(add(3, 7)); // Output: 7
```

上の例では、関数`add`に条件文があり、`a`の値が`2`より大きいかどうかをチェックしています。条件が満たされると、関数は`b`の値を返して実行を停止します。条件が満たされない場合は、関数は`a`と`b`の合計を返します。
