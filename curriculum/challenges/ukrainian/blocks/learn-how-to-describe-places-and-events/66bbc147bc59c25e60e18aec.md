# --explanation--

Час `Present Continuous` використовується для опису дій, які відбуваються зараз. Щоб висловити, що щось **не** відбувається, використовується заперечна форма.

У реченні `The live preview isn't showing` `isn't` — це скорочення від `is not`, що робить речення заперечним.

Це вказує на те, що живий перегляд не працює так, як очікувалося. Отже, Том повідомляє, що живий перегляд наразі не видно у Visual Studio Code.
