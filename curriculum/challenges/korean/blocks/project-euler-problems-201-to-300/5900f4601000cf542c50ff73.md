# --description--

분자가 분모보다 작은 양의 분수를 진분수라고 합니다.

어떤 분모 $d$에 대해서도, 진분수는 $d−1$개가 있습니다. 예를 들어, $d = 12$일 때:

$$\frac{1}{12}, \frac{2}{12}, \frac{3}{12}, \frac{4}{12}, \frac{5}{12}, \frac{6}{12}, \frac{7}{12}, \frac{8}{12}, \frac{9}{12}, \frac{10}{12}, \frac{11}{12}$$

약분할 수 없는 분수를 탄력 분수라고 부르겠습니다.

또한 분모 $d$의 탄력성 $R(d)$를 탄력 분수의 진분수 비율로 정의하겠습니다. 예를 들어, $R(12) = \frac{4}{11}$입니다.

사실, $d = 12$는 탄력성 $R(d) < \frac{4}{10}$을 가지는 가장 작은 분모입니다.

탄력성 $R(d) < \frac{15\,499}{94\,744}$을 가지는 가장 작은 분모 $d$를 찾으세요.
