# --description--

Le istruzioni condizionali più complesse possono includere condizioni multiple. Per questo si usano gli operatori logici per combinare più condizioni. Gli operatori logici sono `&&`, `||` e `!` che servono a rappresentare rispettivamente `and`, `or` e `not`.

L'operatore logico `||` serve a combinare due condizioni booleane. Restituisce `true` se almeno una delle condizioni è `true`. Altrimenti, restituisce `false`:

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a > b || a > c) {
  console.log("At least one of the conditions is true");
} else {
  console.log("Both of the conditions are false");
}
```

L'operatore logico `&&` serve a combinare due condizioni booleane. Restituisce `true` solo se entrambe le condizioni sono `true`. Altrimenti, restituisce `false`:

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a < b && a < c) {
  console.log("Both of the conditions are true");
} else {
  console.log("At least one of the conditions is false");
}
```

L'operatore logico `!` serve a negare una condizione booleana. Restituisce `true` se la condizione è `false`. Altrimenti, restituisce `false`:

```javascript
let a = 5;
let b = 10;

if (!(a > b)) {
  console.log("The condition is false");
} else {
  console.log("The condition is true");
}
```

Nell'esempio sopra, la coppia extra di parentesi tonde serve a rendere il codice più leggibile. Non è necessario usarle.

# --questions--

## --text--

Devi scrivere un'istruzione `if` che controlli le seguenti condizioni in una app web per mostrare `Welcome!` all'utente:

1. L'utente deve avere un account premium (`isPremium`) oppure essere membro da più di un anno (`membershipDuration` > 12 mesi).

1. L'utente non deve essere attualmente bloccato dal servizio (`!isBlocked`).

Quale istruzione if controlla correttamente queste condizioni?

## --answers--

```javascript
if (isPremium && membershipDuration > 12 && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (isPremium || (membershipDuration > 12 && !isBlocked)) {
  console.log("Welcome!");
}
```

---

```javascript
if ((isPremium || membershipDuration > 12) && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (!isPremium || membershipDuration <= 12 || isBlocked) {
  console.log("Welcome!");
}
```
