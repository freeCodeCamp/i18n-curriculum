# --description--

以下方程式表述山區的連續地形，給出任意點 ($x$,$y$) 的海拔高度 $h$：

$$h = \left(5000 - \frac{x^2 + y^2 + xy}{200} + \frac{25(x + y)}{2}\right) \times e^{-\left|\frac{x^2 + y^2}{1\\,000\\,000} - \frac{3(x + y)}{2000} + \frac{7}{10}\right|}$$

一隻蚊子打算從 A(200,200) 飛到 B(1400,1400)，且不離開由 $0 ≤ x$、$y ≤ 1600$ 所限定的區域。

由於中間有山脈，它首先垂直上升到一個高程為 $f$ 的點 A'。然後，在保持相同高程 $f$ 的情況下，它繞過任何障礙物，直到到達位於 B 正上方的點 B'。

首先，確定 $f_{min}$，即允許從 A 到 B 進行此類旅行且保持在指定區域內的最小恆定高度。然後，在以該恆定高度 $f_{min}$ 飛行時，找出 A' 和 B' 之間最短路徑的長度。

將該長度作為你的答案，並四捨五入到小數點後 3 位。

**注意：** 為方便起見，上述顯示的高程函式（function）在下方重複，以適用於大多數程式設計語言的表單：`h=( 5000-0.005*(x*x+y*y+x*y)+12.5*(x+y) )* exp( -abs(0.000001*(x*x+y*y)-0.0015*(x+y)+0.7) )`。
