# --description--

لكل عدد صحيح موجب $n$ تُعيد الدالة $\text{next_prime}(n)$ أصغر عدد أولي $p$ بحيث $p > n$.

تُعرّف المتتالية $a(n)$ كما يلي: $a(1) = \text{next_prime}({10}^{14})$ و $a(n) = \text{next_prime}(a(n - 1))$ لكل $n > 1$.

تُعرّف متتالية فيبوناتشي $f(n)$ كما يلي: $f(0) = 0$، $f(1) = 1$ و $f(n) = f(n - 1) + f(n - 2)$ لكل $n > 1$.

تُعرّف المتتالية $b(n)$ كـ $f(a(n))$.

احسب $\sum b(n)$ لـ $1≤n≤100\\,000$. قدّم إجابتك $\bmod 1\\,234\\,567\\,891\\,011$.
