# --interactive--

## 목록 스타일링

- **`line-height` 속성**: 이 속성은 텍스트 줄 사이에 공간을 만들 때 사용합니다. 허용되는 `line-height` 값에는 키워드 `normal`, 숫자, 백분율, `em` 단위 같은 길이 단위가 포함됩니다.
- **`list-style-type` 속성**: 이 속성은 목록 항목의 마커를 지정할 때 사용합니다. 허용 가능한 값에는 원(circle), 점(disc), 또는 십진수(decimal)가 포함될 수 있습니다.

다음은 `list-style-type`를 사용해 글머리 기호 스타일을 변경한 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
ul {
  list-style-type: square;
}
```

:::

- **`list-style-position` 속성**: 이 속성은 목록 마커의 위치를 설정할 때 사용합니다. 허용되는 값은 inside와 outside 두 가지뿐입니다.

다음은 `inside`과 `outside`의 차이를 보여주는 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="inside-list">
  <li>Item with inside position</li>
  <li>Item with inside position</li>
</ul>
<ul class="outside-list">
  <li>Item with outside position</li>
  <li>Item with outside position</li>
</ul>
```

```css
.inside-list {
  list-style-position: inside;
}

.outside-list {
  list-style-position: outside;
}
```

:::

- **`list-style-image` 속성**: 이 속성은 목록 항목 마커로 이미지를 사용할 때 사용합니다. 일반적인 사용 사례는 유효한 이미지 위치를 값으로 설정한 `url` 함수를 사용하는 것입니다.

## `margin`을 사용한 목록 항목 간격 조절

- `line-height` 외에도, CSS에서 여백을 사용해 목록 항목 간격과 가독성을 향상할 수 있습니다.
- 여백은 각 `li` 요소 바깥에 공간을 만들어 목록 항목 사이의 간격을 조절할 수 있게 합니다.
- `margin-bottom`는 각 목록 항목 아래에 공간을 만듭니다. 예를 들어, `margin-bottom: 10px;`는 각 목록 항목 아래에 10픽셀 간격을 만듭니다.

다음은 `margin-bottom`을 사용해 목록 항목 간격을 조절한 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="list">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
.list li {
  margin-bottom: 20px;
}
```

:::

## 링크 스타일링

- **`pseudo-class`**: 이 키워드는 특정 상태에 따라 요소를 선택할 수 있게 선택자에 추가합니다. 일반적인 상태에는 `:hover`, `:visited`, `:focus` 상태가 포함됩니다.
- **`:link pseudo-class`**: 이 가상 클래스는 사용자가 방문하지 않은 링크를 스타일링할 때 사용합니다.

다음은 `:link` 가상 클래스를 사용한 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:link {
  color: red;
}
```

:::

- **`:visited pseudo-class`**: 이 가상 클래스는 사용자가 이미 방문한 링크를 스타일링할 때 사용합니다.
- **`:hover pseudo-class`**: 이 가상 클래스는 사용자가 요소 위에 마우스를 올려놓았을 때 스타일링할 때 사용합니다.

다음은 `:hover` 가상 클래스를 사용한 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Hover over me!</a>
```

```css
a:hover {
  color: green;
  text-decoration: underline;
}
```

:::

- **`:focus pseudo-class`**: 이 가상 클래스는 요소가 포커스를 받을 때 스타일링할 때 사용합니다. 예를 들어, `input` 또는 `select` 요소에 클릭하거나 탭하여 포커스를 줄 때입니다.

다음은 `:focus` 가상 클래스를 사용한 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:focus {
  outline: 2px solid orange;
}
```

:::

- **`:active pseudo-class`**: 이 가상 클래스는 사용자가 요소를 활성화했을 때 스타일링할 때 사용합니다. 일반적인 예는 사용자가 버튼을 클릭했을 때입니다.

다음은 `:active` 가상 클래스를 사용한 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Click me!</a>
```

```css
a:active {
  color: pink;
}
```

:::

## 배경과 테두리 작업

- **`background-size` 속성**: 이 속성은 요소의 배경 크기를 설정할 때 사용합니다. 일반적인 값으로는 배경 이미지가 요소 전체를 덮도록 하는 `cover`와 배경 이미지가 요소 내에 맞도록 하는 `contain`가 있습니다.

다음은 `background-size: cover`를 사용한 예입니다:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

다음은 `background-size: contain`를 사용한 예입니다:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **`background-repeat` 속성**: 이 속성은 배경 이미지가 수평 및 수직 축을 따라 어떻게 반복될지 결정할 때 사용합니다. `background-repeat`의 기본값은 `repeat`로, 이미지가 수평과 수직 모두에서 반복됨을 의미합니다. `no-repeat` 속성을 사용해 반복하지 않도록 지정할 수도 있습니다.

다음은 `background-repeat: no-repeat`를 사용한 예입니다:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **`background-position` 속성**: 이 속성은 배경 이미지의 위치를 지정할 때 사용합니다. 특정 길이, 백분율 또는 `top`, `bottom`, `left`, `right`, `center` 같은 키워드 값으로 설정할 수 있습니다.

다음은 `background-position`를 사용한 예입니다:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

- **`background-attachment` 속성**: 이 속성은 배경 이미지가 콘텐츠와 함께 스크롤할지 아니면 화면에 고정될지 지정할 때 사용합니다. 주요 값은 기본값인 `scroll`으로, 배경 이미지가 콘텐츠와 함께 스크롤되고, `fixed`은 배경 이미지가 화면에서 같은 위치에 고정됩니다.
- **`background-image` 속성**: 이 속성은 요소의 배경 이미지를 설정할 때 사용합니다. 여러 배경 이미지를 동시에 설정할 수 있으며, 값으로 `url`, `radial-gradient`, `linear-gradient` 함수 중 하나를 사용할 수 있습니다.

다음은 `background-image`를 사용한 예입니다:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **`background` 속성**: 이 속성은 모든 배경 속성을 한 선언에서 설정하는 단축 속성입니다. 다음은 배경 이미지를 설정하고 반복하지 않도록 설정한 예입니다: `background: no-repeat url("example-url-goes-here");`

다음은 `background` 단축 속성을 사용한 예입니다:

:::interactive_editor

```html
<style>
  body {
    background: center top no-repeat url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **배경과 전경 색상의 좋은 명암 대비**: 텍스트 가독성을 위해 배경과 전경 색상 간에 좋은 명암 대비를 유지하는 것이 중요합니다. 웹 콘텐츠 접근성 지침(WCAG)은 일반 텍스트에 대해 최소 명암 대비율 4.5:1, 큰 텍스트에 대해 3:1을 권장합니다.

## 테두리

- **`border-top` 속성**: 이 속성은 요소의 상단 테두리 스타일을 설정할 때 사용합니다. `border-top: 3px solid blue;`는 요소 상단에 3픽셀 너비의 실선 파란색 테두리를 설정합니다.
- **`border-right` 속성**: 이 속성은 요소의 오른쪽 테두리 스타일을 설정할 때 사용합니다. `border-right: 2px solid red;`는 요소 오른쪽에 2픽셀 너비의 실선 빨간색 테두리를 설정합니다.
- **`border-bottom` 속성**: 이 속성은 요소의 하단 테두리 스타일을 설정할 때 사용합니다. `border-bottom: 1px dashed green;`는 요소 하단에 1픽셀 너비의 점선 초록색 테두리를 설정합니다.
- **`border-left` 속성**: 이 속성은 요소의 왼쪽 테두리 스타일을 설정할 때 사용합니다. `border-left: 4px dotted orange;`는 요소 왼쪽에 4픽셀 너비의 점선 주황색 테두리를 설정합니다.

다음은 개별 테두리 속성을 사용한 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="bordered-box">Box with different borders</div>
```

```css
.bordered-box {
  border-top: 3px solid blue;
  border-right: 2px solid red;
  border-bottom: 1px dashed green;
  border-left: 4px dotted orange;
  padding: 20px;
}
```

:::

- **`border` 속성**: 이 속성은 요소 테두리의 너비, 스타일, 색상을 한꺼번에 설정하는 단축 속성입니다. `border: 1px solid black;`는 1픽셀 너비의 실선 검은색 테두리를 설정합니다.

다음은 `border` 단축 속성을 사용한 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

- **`border-radius` 속성**: 이 속성은 요소 테두리에 둥근 모서리를 만들 때 사용합니다.

다음은 `border-radius`를 사용한 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

- **`border-style` 속성**: 이 속성은 요소 테두리 스타일을 설정할 때 사용합니다. 허용되는 값에는 `solid`, `dashed`, `dotted`, `double` 등이 포함됩니다.

다음은 다양한 `border-style` 값을 사용한 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="solid-border">Solid border</div>
<div class="dashed-border">Dashed border</div>
<div class="dotted-border">Dotted border</div>
<div class="double-border">Double border</div>
```

```css
.solid-border {
  border: 3px solid blue;
  margin-bottom: 10px;
  padding: 10px;
}

.dashed-border {
  border: 3px dashed red;
  margin-bottom: 10px;
  padding: 10px;
}

.dotted-border {
  border: 3px dotted green;
  padding: 10px;
}

.double-border {
  border: 3px double blueviolet;
  padding: 10px;
}
```

:::

## 그라디언트

- **`linear-gradient()` 함수**: 이 CSS 함수는 여러 색상 사이를 직선으로 전환하는 그라디언트를 만듭니다.

다음은 `linear-gradient()`를 사용한 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

- **`radial-gradient()` 함수**: 이 CSS 함수는 원이나 타원처럼 특정 지점에서 방사형으로 퍼지는 이미지를 만들며, 여러 색상 사이를 점진적으로 전환합니다.

다음은 `radial-gradient()`를 사용한 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::
