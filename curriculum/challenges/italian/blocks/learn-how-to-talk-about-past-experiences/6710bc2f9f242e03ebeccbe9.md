# --explanation--

Quando vuoi esprimere accordo con l’idea di qualcuno, puoi dirlo brevemente usando `Sure`. Per esempio:

- `Would you like to visit another country some day? Sure.` - Questo significa che vuoi farlo.

- `How about going to the movies this week? Sure.` - Questo significa che accetti l’invito.
