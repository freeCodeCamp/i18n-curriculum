# --description--

Для додатного числа $n$ визначимо $C(n)$ як кількість цілих чисел $x$, для яких $1 < x < n$ та $x^3 \equiv 1\bmod n$.

Коли $n = 91$, існує 8 можливих значень для $x$, а саме: 9, 16, 22, 29, 53, 74, 79, 81. Отже, $C(91) = 8$.

Знайдіть суму додатних чисел $n ≤ {10}^{11}$, для яких $C(n)=242$.
