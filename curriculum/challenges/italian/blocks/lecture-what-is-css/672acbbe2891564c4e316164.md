# --interactive--

Il CSS è responsabile degli stili di una pagina web. Tutti questi stili sono composti da varie regole CSS.

Una regola CSS è composta da due parti principali: un **selettore** e un blocco di **dichiarazione**.

Diamo un'occhiata alla sintassi di base:

```css
selector {
  property: value;
}
```

Un selettore è un pattern usato in CSS per identificare e prendere di mira specifici elementi HTML da stilizzare.

Esempi di selettori includono selettori di tipo, selettori di classe e selettori ID.

Le parentesi graffe presenti nella sintassi di base sono conosciute come blocco di dichiarazione. Un blocco di dichiarazione applica un insieme di stili a un dato selettore, o selettori.

All'interno del blocco di dichiarazione, troverai una serie di dichiarazioni. Ogni dichiarazione consiste in una proprietà e un valore.

La proprietà è l'identificatore CSS che specifica quale funzionalità viene stilizzata. Un esempio di proprietà sarebbe la proprietà `background-color`.

Il valore sarebbe l'impostazione specifica applicata a quella proprietà. Per esempio, se la proprietà è `background-color`, un valore potrebbe essere `purple`, che imposta il colore di sfondo su viola.

Dopo ogni nome di proprietà, devi mettere due punti, e dopo ogni valore, devi mettere un punto e virgola.

Ora che conosci la sintassi di una regola CSS, diamo un'occhiata a un esempio. Attiva l'editor interattivo e clicca sulla scheda `styles.css` per vedere il codice CSS.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Learning about CSS</h1>

<p>Example paragraph element</p>
<p>Another example paragraph element</p>
```

```css
p {
  color: blue;
}
```

:::

In questa regola CSS, un selettore di tipo prende di mira tutti gli elementi paragrafo nella pagina.

All'interno del blocco di dichiarazione, c'è una dichiarazione.

La dichiarazione consiste nella proprietà `color` con un valore impostato su `blue`. Questo cambierà il colore del testo di tutti gli elementi paragrafo in `blue`.

Se vuoi applicare lo stesso insieme di stili a più selettori, puoi creare una lista di selettori separando ogni selettore con una virgola.

Ecco un esempio di stilizzazione di selettori multipli (per interagire con l'esempio, attiva l'editor interattivo):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1 id="title">Example heading</h1>
<h2 class="subheading">Example subheading</h2>
<p>This paragraph is not affected by the selector.</p>
```

```css
#title,
.subheading {
  color: navy;
}
```

:::

In questa lista di selettori, c'è un selettore `id` che prende di mira l'elemento HTML con il valore `id` di `title`. Tutti i selettori `id` devono iniziare con il simbolo hash `#`.

Poi c'è una virgola seguita da un selettore `class` che prende di mira tutti gli elementi HTML con il valore `class` di `subheading`. Tutti i selettori di classe devono iniziare con un punto `.`.

L'intera lista di selettori riceverà la stessa stilizzazione, con il colore del testo impostato su `navy`.
