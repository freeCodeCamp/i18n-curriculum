# --explanation--

`To prevent` bedeutet, Maßnahmen zu ergreifen, die verhindern, dass etwas Schlechtes oder Unerwünschtes passiert. Zum Beispiel:

`We installed a firewall to prevent unauthorized access.` – Das bedeutet, dass die Firewall unerwünschte Benutzer am Eindringen ins System hindert.

Eine `overload` ist, wenn ein System oder Gerät mehr Arbeit, Daten oder Energie erhält, als es bewältigen kann, was Probleme verursacht. Zum Beispiel:

`The power outage was caused by an overload in the electrical grid.` – Das bedeutet, dass zu viel Strom einen Ausfall verursacht hat.
