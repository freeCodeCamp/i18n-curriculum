# --explanation--

`True, but…`는 누군가의 의견에 동의하면서도 다른 관점을 덧붙일 수 있게 해줍니다. 이 표현은 유효한 의견을 인정하면서 자신의 인수를 제시하고 싶을 때 토론에서 유용합니다. 예를 들어:

- **Quinn:** `AI tools can speed up coding.`

- **Sem:** `True, but they still make mistakes that need human review.` - 이 사람은 AI가 코딩에 도움이 된다는 점에 동의하지만 완벽하지 않고 여전히 인간의 감독이 필요하다는 점을 지적합니다.
