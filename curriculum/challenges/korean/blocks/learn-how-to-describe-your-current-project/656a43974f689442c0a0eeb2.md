# --description--

`Currently`는 현재 진행 중임을 뜻하는 `current`에서 왔습니다. `-ly`를 붙이면 형용사를 부사로 만듭니다. 예를 들어:

`She is currently reading a book.` - 그녀는 지금 책을 읽고 있습니다.

`Trends`는 많은 사람이 이야기하거나 하고 있는 인기 있는 스타일이나 아이디어입니다. 단수형 `trend`는 복수형 `trends`가 됩니다. 예를 들어:

- `One current trend is using eco-friendly products.` - 여기서 단수형 `trend`는 한 가지 인기 있는 아이디어를 가리킵니다.

- `She writes about fashion trends.` - 여기서 복수형 `trends`는 여러 가지 인기 있는 스타일을 가리킵니다.

`Tech trends`는 기술 분야에서 인기 있는 개발 동향을 뜻합니다.
