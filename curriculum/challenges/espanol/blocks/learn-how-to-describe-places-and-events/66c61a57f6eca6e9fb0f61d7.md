### --feedback--

# --explanation--

`Built-in` se refiere a características o herramientas que están incluidas como parte de un programa de software o dispositivo desde el inicio, sin necesidad de ser añadidas o instaladas por separado. 

Cuando algo es `built-in`, ya está ahí y listo para usar. Por ejemplo:

- `This laptop has a built-in camera.` - La cámara está incluida como parte de la laptop.

- `The software comes with built-in security features.` - Las funciones de seguridad ya están incluidas en el software.
