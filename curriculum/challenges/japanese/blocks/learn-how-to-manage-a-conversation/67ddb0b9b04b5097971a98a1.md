# --explanation--

日常会話では、意味が明確な場合に言葉を省略することがよくあります。この場合、`Anything else we should discuss?` は `Is there anything else we should discuss?` の短縮形です。このような省略は会話でよく見られます。別の例として：

`Need any help?` は `Do you need any help?` を短く言ったもので、主語と助動詞が省略されていますが、意味は同じです。
