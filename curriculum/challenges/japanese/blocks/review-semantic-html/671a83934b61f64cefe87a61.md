# --interactive--

## セマンティックHTMLの重要性

- **見出し要素の構造的階層**：コンテンツの構造的階層を維持するために、正しい見出し要素を使うことが重要です。`h1` 要素は最上位の見出しで、`h6` 要素は最下位の見出しです。
- **プレゼンテーション用HTML要素**：コンテンツの見た目を定義する要素です。例：廃止された `center`、`big`、`font` 要素。
- **セマンティックHTML要素**：意味と構造を持つ要素です。例：`header`、`nav`、`figure`。

## セマンティックHTML要素

- **ヘッダー要素**：ドキュメントやセクションのヘッダーを定義するために使います。

:::interactive_editor

  ```html
  <header>
    <h1>CatPhotoApp</h1>
    <p>Welcome to our cat gallery.</p>
  </header>
  ```

:::

- **メイン要素**：ウェブページの主要なコンテンツを含むために使います。

:::interactive_editor

  ```html
  <main>
    <section>
      <h2>Cat Photos</h2>
      <p>Browse adorable cat pictures.</p>
    </section>
  </main>
  ```

:::

- **セクション要素**：コンテンツを小さなセクションに分割するために使います。

:::interactive_editor

  ```html
  <section>
    <h2>About Me</h2>
    <p>Hi, I am Jane Doe and I am a web developer.</p>
  </section>
  ```

:::

- **ナビゲーションセクション（`nav`）要素**：ナビゲーションリンクを含むセクションを表します。

:::interactive_editor

  ```html
  <nav>
    <ul>
      <li><a href="#photos">Photos</a></li>
      <li><a href="#videos">Videos</a></li>
    </ul>
  </nav>
  ```

:::

- **図要素**：イラストや図を含むために使います。

:::interactive_editor

  ```html
  <figure>
    <img
      src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"
      alt="Two tabby kittens sleeping together on a couch."
    />
    <figcaption>Cats <strong>hate</strong> other cats.</figcaption>
  </figure>
  ```

:::

- **強調（`em`）要素**：強調したいテキストをマークします。

:::interactive_editor

  ```html
  <p>
    Never give up on <em>your</em> dreams.
  </p>
  ```

:::

- **慣用表現テキスト（`i`）要素**：別の言語の慣用句や技術用語、思考などを強調するために使います。

:::interactive_editor

  ```html
  <p>
    There is a certain <i lang="fr">je ne sais quoi</i> in the air.
  </p>
  ```

:::

開く `lang` タグ内の `i` 属性はコンテンツの言語を指定するために使います。この場合、言語はフランス語です。`i` 要素はテキストが重要かどうかを示すものではなく、周囲のテキストと何らかの違いがあることを示しています。

- **強い重要性（`strong`）要素**：強い重要性を持つテキストをマークします。

:::interactive_editor

  ```html
  <p>
    <strong>Warning:</strong> This product may cause allergic reactions.
  </p>
  ```

:::

- **注意を引く（`b`）要素**：コンテンツの意味には重要でないが注意を引きたいテキストに使います。要約のキーワードやレビューの製品名の強調に一般的に使われます。

:::interactive_editor

  ```html
  <p>
    We tested several products, including the <b>SuperSound 3000</b> for audio quality, the <b>QuickCharge Pro</b> for fast charging, and the <b>Ecoclean Vacuum</b> for cleaning. The first two performed well, but the <b>Ecoclean Vacuum</b> did not meet expectations.
  </p>
  ```

:::

- **説明リスト（`dl`）要素**：用語と説明のグループを表すリストに使います。
- **説明用語（`dt`）要素**：定義される用語を表します。
- **説明詳細（`dd`）要素**：用語の説明を表します。

:::interactive_editor

  ```html
  <dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
  </dl>
  ```

:::

- **ブロック引用（`blockquote`）要素**：別の情報源からの引用部分を表します。この要素には `cite` 属性があります。`cite` 属性の値は情報源のURLです。

:::interactive_editor

  ```html
  <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
    "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
  </blockquote>
  ```

:::

- **引用（`cite`）要素**：参照元の作品の出典を視覚的に示すために使います。参照のタイトルをマークアップします。

:::interactive_editor

  ```html
  <div>
    <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
      "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
    </blockquote>
    <p>
      -Quincy Larson, <cite>How to learn to Code and Get a Developer Job [Full Book].</cite>
    </p>
  </div>
  ```

:::

- **インライン引用（`q`）要素**：短いインラインの引用を表します。

:::interactive_editor

  ```html
  <p>
    As Quincy Larson said,
    <q cite="https://www.freecodecamp.org/news/learn-to-code-book/">
      Momentum is everything.
    </q>
  </p>
  ```

:::

- **略語（`abbr`）要素**：略語や頭字語を表します。ユーザーが略語の意味を理解できるように、`title` 属性で完全な形や説明を表示できます。

:::interactive_editor

  ```html
  <p>
    <abbr title="HyperText Markup Language">HTML</abbr> is the foundation of the web.
  </p>
  ```

:::

- **連絡先住所（`address`）要素**：連絡先情報を表すために使います。
- **（日付）時間（`time`）要素**：日付や時間を表します。`datetime` 属性は日付や時間を機械判読可能な形式に変換するために使います。

:::interactive_editor

  ```html
  <p>
    The reservations are for the <time datetime="20:00">20:00 </time>
  </p>
  ```

:::

- **ISO 8601 日付（`datetime`）属性**：日付や時間を機械判読可能な形式で表します。標準形式は `YYYY-MM-DDThh:mm:ss` で、大文字の `T` が日付と時間の区切りです。
- **上付き文字（`sup`）要素**：上付き文字を表します。`sup` 要素の一般的な用途は指数、上位文字、序数です。

:::interactive_editor

  ```html
  <p>
    2<sup>2</sup> (2 squared) is 4.
  </p>
  ```

:::

- **下付き文字（`sub`）要素**：下付き文字を表します。一般的な用途は化学式、脚注、変数の添え字です。

:::interactive_editor

  ```html
  <p>
    CO<sub>2</sub>
  </p>
  ```

:::

- **インラインコード（`code`）要素**：コンピューターコードの断片を表します。
- **整形済みテキスト（`pre`）要素**：整形済みテキストを表します。

:::interactive_editor

  ```html
  <pre>
    <code>
      body {
        color: red;
      }
    </code>
  </pre>
  ```

:::

- **非明示的注釈（`u`）要素**：非テキスト注釈があることを示すインラインテキストの範囲を表します。

:::interactive_editor

  ```html
  <p>
    You can use the unarticulated annotation element to highlight
    <u>inccccort</u> <u>spling</u> <u>issses</u>.
  </p>
  ```

:::

- **ルビ注釈（`ruby`）要素**：発音や意味の説明を注釈するために使います。東アジアの組版でよく使われます。
- **ルビフォールバック括弧（`rp`）要素**：ルビ注釈表示に対応していないブラウザのフォールバックとして使います。
- **ルビテキスト（`rt`）要素**：ルビ注釈のテキストを示します。通常は発音や翻訳の詳細に使います。

:::interactive_editor

  ```html
  <ruby>
    明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp>
  </ruby>
  ```

:::

- **取り消し線（`s`）要素**：もはや正確でない、または関連性のないコンテンツを表します。

:::interactive_editor

  ```html
  <p>
    <s>Tomorrow's hike will be meeting at noon.</s>
  </p>
  <p>
    Due to unforeseen weather conditions, the hike has been canceled.
  </p>
  ```

:::

- **内部リンク**：`href="#id"` 要素に `a` を使い、リンク先の要素に同じ `id` を付けることでページ内の別のセクションにリンクします。スキップリンクや目次、複数セクションの長いページでよく使われます。

:::interactive_editor

```html
<a href="#about-section">Go to About Section</a>

<section id="about-section">
  <h2>About</h2>
  <p>This is the about section of the page.</p>
</section>
```

:::
