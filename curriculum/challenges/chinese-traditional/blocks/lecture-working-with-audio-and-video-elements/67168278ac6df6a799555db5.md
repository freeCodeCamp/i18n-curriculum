# --interactive--

`audio` 和 `video` 元素允許你為你的 HTML 文件添加音訊和影片內容。`audio` 元素支援像是 mp3、wav 和 ogg 等流行的音訊格式。`video` 元素支援 mp4、ogg 和 webm 格式。

要在你的網頁中包含音訊內容，你可以使用帶有指向你的音訊檔案位置的 `src` 屬性的 `audio` 元素。 

如你在預覽視窗中所見，頁面上沒有任何顯示。若要看到預覽，你需要啟用互動式編輯器。

:::interactive_editor

```html
<audio src="https://cdn.freecodecamp.org/curriculum/js-music-player/cruising-for-a-musing.mp3"></audio>
```

:::

如果你想在頁面上看到音訊播放器，則可以為你的 `audio` 元素添加 `controls` 屬性。

在預覽視窗中按播放以聽取 Quincy Larson 的其中一首歌曲。若要聽取不同的歌曲，請將 `src` 值更改為 `"https://cdn.freecodecamp.org/curriculum/js-music-player/never-not-favored.mp3"`。若要查看預覽，你需要啟用互動式編輯器。

:::interactive_editor

```html
<audio src="https://cdn.freecodecamp.org/curriculum/js-music-player/cruising-for-a-musing.mp3" controls></audio>
```

:::

`controls` 屬性啟用使用者管理音訊播放，包括暫停或恢復播放。`controls` 屬性是一個布林值屬性，可以添加到元素以啟用內建的播放控制元件。如果省略，將不會顯示任何控制元件。

注意：某些瀏覽器，例如 Safari，即使存在 `controls` 屬性，也可能不會預設顯示音量控制元件。

除了 `src` 和 `controls` 屬性之外，還有其他幾個屬性能增強 `audio` 元素的功能。`loop` 屬性是一個布林值屬性，可讓音訊持續重播。 

這裡有一個使用 `loop` 屬性來播放 Quincy Larson 的歌曲「Can't stay down」的範例。若要看到迴圈效果，請啟用互動式編輯器，將播放頭拖曳至歌曲接近結尾處，歌曲結束後便會重新開始。 

:::interactive_editor

```html
<audio
  src="https://cdn.freecodecamp.org/curriculum/js-music-player/can't-stay-down.mp3"
  loop
  controls
></audio>
```

:::

你可以使用的另一個屬性是 `muted` 屬性。當它出現在 `audio` 元素中時，這個布林值屬性會讓音訊以靜音狀態開始。以下是使用 `muted` 屬性的範例。

要聽音樂，請啟用互動式編輯器並點擊音訊播放器中的音量圖示。 

:::interactive_editor

```html
<audio
  src="https://cdn.freecodecamp.org/curriculum/js-music-player/can't-stay-down.mp3"
  loop
  controls
  muted
></audio>
```

:::

當談到音訊檔案型別時，不同瀏覽器對於支援的型別有所差異。為了因應這點，你可以在 `audio` 元素內使用多個 `source` 元素，瀏覽器會選擇它能理解的第一個來源。以下是為 `audio` 元素使用多個 `source` 元素的範例：

```html
<audio controls>
  <source src="audio.ogg" type="audio/ogg" />
  <source src="audio.wav" type="audio/wav" />
  <source src="audio.mp3" type="audio/mpeg" />
</audio>
```

瀏覽器會先從 ogg 型別開始，如果無法播放該音訊，則會移至列表中的下一個型別。

我們目前學過的所有屬性也都支援在 `video` 元素中。以下是一個使用 `video` 元素並帶有 `loop`、`controls` 和 `muted` 屬性的範例。 

為你的開啟 `video` 頁籤添加 `autoplay` 屬性，使影片自動播放。若要與範例互動，你需要啟用互動式編輯器。

**注意**：這裡使用了 `width` 屬性來讓影片變小並更適合預覽視窗。你將在未來的課程中學習更多關於 `width` 屬性。

:::interactive_editor

```html
<video
  src="https://archive.org/download/BigBuckBunny_124/Content/big_buck_bunny_720p_surround.mp4"
  loop
  controls
  muted
  width="400"
></video>
```

:::

對於 `src`，或原始碼屬性，我們使用來自 archive.org 的名為「Big Buck Bunny」的影片。如果你想在影片下載時顯示影像，可以使用 `poster` 屬性。此屬性不適用於 `audio` 元素，且是 `video` 元素特有的。以下是使用 `poster` 屬性並由 peach.blender.org 提供內容的範例。

```html
<video
  src="https://archive.org/download/BigBuckBunny_124/Content/big_buck_bunny_720p_surround.mp4"
  loop
  controls
  muted
  poster="https://peach.blender.org/wp-content/uploads/title_anouncement.jpg?x11217"
  width="400"
></video>
```

你也可以在 `video` 元素內使用 `source` 元素，就像你對 `audio` 元素所做的那樣。這讓你能夠提供多種格式的相同影片，瀏覽器會選擇它能播放的第一個格式。

```html
<video
  controls
  width="400"
  poster="https://peach.blender.org/wp-content/uploads/title_anouncement.jpg?x11217"
>
  <source
    src="https://archive.org/download/BigBuckBunny_124/Content/big_buck_bunny_720p_surround.mp4"
    type="video/mp4"
  />
  <source
    src="https://archive.org/download/BigBuckBunny_124/Content/big_buck_bunny_720p_surround.webm"
    type="video/webm"
  />
  Your browser does not support the video tag.
</video>
```
