이번 도전 과제에서는 회귀 알고리즘을 사용해 의료비용을 예측할 것입니다.

여러 사람에 대한 정보와 의료비용이 포함된 데이터셋이 주어집니다. 이 데이터를 사용해 새로운 데이터에 기반한 의료비용을 예측하세요.

이 노트북의 처음 두 셀은 라이브러리와 데이터를 가져옵니다.

범주형 데이터를 숫자로 변환해야 합니다. 데이터의 80%를 `train_dataset`로, 20%를 `test_dataset`로 사용하세요.

이 데이터셋들에서 "expenses" 열을 `pop`하여 `train_labels`와 `test_labels`라는 새 데이터셋을 만드세요. 모델을 훈련할 때 이 레이블을 사용하세요.

모델을 만들고 `train_dataset`로 훈련하세요. 이 노트북의 마지막 셀을 실행해 모델을 확인하세요. 마지막 셀은 보지 않은 `test_dataset`를 사용해 모델의 일반화 능력을 평가합니다.

도전 과제를 통과하려면 `model.evaluate`가 평균 절대 오차가 3500 미만이어야 합니다. 이는 의료비용을 3500달러 이내로 정확히 예측한다는 뜻입니다.

마지막 셀은 `test_dataset`를 사용해 비용을 예측하고 결과를 그래프로 표시합니다.
