# --explanation--

Em espanhol, o verbo `gustar` ("gostar") é usado para expressar que alguém gosta de algo.

`Me gusta crear` - Eu gosto de criar.

Para dizer que alguém **não** gosta de algo, você deve escrever a palavra `no` no início. Por exemplo:

`No me gusta crear` - Eu não gosto de criar.

Esteban diz `Le gusta crear ideas visuales`, então Diego gosta de criar ideias visuais.

A versão negativa seria `No le gusta crear ideas visuales.`
