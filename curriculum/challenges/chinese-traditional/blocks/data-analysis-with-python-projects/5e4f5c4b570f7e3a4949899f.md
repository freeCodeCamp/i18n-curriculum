# --instructions--

你將分析自 1880 年以來全域的平均海平面變化的資料集。你將使用這些資料來預測直到 2050 年的海平面變化。

使用資料完成以下任務：

- 使用 Pandas 從 `epa-sea-level.csv` 匯入資料。
- 使用 matplotlib 以 `Year` 行作為 x 軸，並以 `CSIRO Adjusted Sea Level` 行作為 y 軸來創建散佈圖。
- 使用 `scipy.stats` 中的 `linregress` 函式來取得最佳擬合線的斜率和 y 截距。在散佈圖上繪製最佳擬合線。讓該線通過 2050 年，以預測 2050 年的海平面上升。
- 繪製一條新的最佳擬合線，僅使用從 2000 年到資料集中最近年份的資料。若自 2000 年以來的上升速率持續，則使該線也通過 2050 年，以預測 2050 年的海平面上升。
- x 標籤應該是 `Year`，y 標籤應該是 `Sea Level (inches)`，標題應該是 `Rise in Sea Level`。

樣板程式碼也包含用於存檔和傳回影像的指令。
