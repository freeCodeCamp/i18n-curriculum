# --explanation--

Der Ausdruck `stuck on` wird verwendet, wenn jemand nicht in der Lage ist, mit einer Aufgabe voranzukommen oder ein Problem zu lösen. Er deutet darauf hin, dass man feststeckt oder keinen Fortschritt macht. Zum Beispiel:

- `We were stuck on this code error for hours.` – Wir konnten nicht weitermachen, bis der Fehler im Code gefunden wurde.

- `She got stuck on the final question of the exam.` – Sie hat lange an der letzten Frage gearbeitet.

Im Dialog benutzt Lisa `stuck on`, um zu beschreiben, wie sie bei der Lösung eines Bugs in ihrer Arbeit nicht vorankamen.
