# --explanation--

`It's like` wird verwendet, um einen Vergleich einzuführen oder eine Situation so zu beschreiben, dass sie leichter verständlich wird. Es deutet darauf hin, dass sich etwas **ähnlich anfühlt wie** eine andere Situation. Zum Beispiel:

`It's like he's angry at me.` – Das bedeutet, die Situation **fühlt sich so an, als ob** jemand anders wütend auf Sie ist.
