# --description--

In diesem Labor implementieren Sie den Sortieralgorithmus Quicksort.

Quicksort ist ein effizienter, rekursiver Divide-and-Conquer-Ansatz zum Sortieren eines Arrays. Bei dieser Methode wird ein Pivot-Wert im ursprünglichen Array ausgewählt. Das Array wird dann in zwei Teilarrays aufgeteilt, die Werte enthalten, die kleiner bzw. größer als der Pivot-Wert sind. Anschließend wird das Ergebnis der rekursiven Aufrufe des Quicksort-Algorithmus auf beiden Teilarrays kombiniert. Dies wird fortgesetzt, bis der Basisfall eines leeren oder ein Element enthaltenden Arrays erreicht ist, das zurückgegeben wird. Das Zurückgeben der rekursiven Aufrufe liefert das sortierte Array.

Obwohl die Wahl des Pivot-Werts wichtig ist, genügt für unsere Zwecke hier jeder Pivot. Zur Vereinfachung kann das erste oder letzte Element verwendet werden.

Quicksort ist eine sehr effiziente Sortiermethode, die im Durchschnitt eine Laufzeit von *O(nlog(n))* bietet. Außerdem ist sie relativ einfach zu implementieren. Diese Eigenschaften machen sie zu einer beliebten und nützlichen Sortiermethode.

**Ziel:** Erfüllen Sie die untenstehenden User Stories und bestehen Sie alle Tests, um das Labor abzuschließen.

**User Stories:**

1. Sie sollten eine `quicksort`-Funktion haben, die ein Array ganzer Zahlen als Eingabe erhält.
1. Die `quicksort`-Funktion sollte ein Array zurückgeben, das dieselben ganzen Zahlen wie die Eingabe enthält, aber von klein nach groß sortiert ist.
1. Die `quicksort`-Funktion sollte sich selbst rekursiv aufrufen, um das Array zu sortieren.

```js
function quicksort(array) {
  if (array.length === 0) {
    return [];
  } else {
    const pivotValue = array[0];

    // Sort elements into three piles
    let lesser = [];
    let equal = [];
    let greater = [];
    for (let e of array) {
      if (e < pivotValue) {
        lesser.push(e);
      } else if (e > pivotValue) {
        greater.push(e);
      } else {
        equal.push(e);
      }
    }

    return [...quicksort(lesser), ...equal, ...quicksort(greater)];
  }
}
```
