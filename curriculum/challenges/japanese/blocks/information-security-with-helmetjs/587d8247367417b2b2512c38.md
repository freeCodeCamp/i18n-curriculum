# --description--

このプロジェクトは、<a href="https://github.com/freeCodeCamp/boilerplate-infosec/" target="_blank" rel="noopener noreferrer nofollow">GitHub</a> からクローンした以下のスタータープロジェクトを基に構築されています。

あなたのページは、あなたの同意なしに `<frame>` や `<iframe>` に埋め込まれる可能性があります。これにより、クリックジャッキング攻撃などが発生することがあります。クリックジャッキングとは、ユーザーが認識しているページとは異なるページと誤認させて操作させる手法です。これは、iframeを使って悪意のあるコンテキストであなたのページを実行することで実現されます。そのコンテキスト内で、ハッカーはあなたのページの上に隠れたレイヤーを置くことができます。隠れたボタンを使って悪意のあるスクリプトを実行させることも可能です。このミドルウェアは X-Frame-Options ヘッダーを設定します。誰があなたのサイトをフレームに入れられるかを制限します。モードは DENY、SAMEORIGIN、ALLOW-FROM の3つがあります。

私たちのアプリはフレームに入れられる必要はありません。

# --instructions--

設定オブジェクト `helmet.frameguard()` を渡して `{action: 'deny'}` を使ってください。

# --hints--

helmet.frameguard() ミドルウェアが正しくマウントされていること

```js
const response = await fetch(code + '/_api/app-info');
if (!response.ok) {
  throw Error(await response.text());
}
const data = await response.json();
assert.include(
  data.appStack,
  'frameguard',
  'helmet.frameguard() middleware is not mounted correctly'
);
```

helmet.frameguard() の 'action' は 'DENY' に設定してください

```js
const response = await fetch(code + '/_api/app-info');
if (!response.ok) {
  throw Error(await response.text());
}
const data = await response.json();
assert.property(data.headers, 'x-frame-options');
assert.equal(data.headers['x-frame-options'], 'DENY');
```
