## CSS 프레임워크

- **CSS 프레임워크**: CSS 프레임워크는 작업 속도를 높이고, 웹사이트 전반에 걸쳐 일관된 시각적 스타일을 만들며, 여러 브라우저에서 디자인이 일관되게 보이도록 하고, CSS 코드를 더 체계적으로 관리할 수 있게 합니다.
- **인기 있는 CSS 프레임워크**: 인기 있는 CSS 프레임워크로는 Tailwind CSS, Bootstrap, Materialize, Foundation 등이 있습니다.
- **잠재적 단점**:
  - 프레임워크에서 제공하는 CSS가 사용자 정의 CSS와 충돌할 수 있습니다.
  - 같은 프레임워크를 사용하는 다른 웹사이트와 비슷하게 보일 수 있습니다.
  - 큰 프레임워크는 성능 문제를 일으킬 수 있습니다.

## 두 가지 유형의 CSS 프레임워크

- **유틸리티 우선 CSS 프레임워크**: 이 프레임워크는 여백, 패딩, 배경색 설정 같은 특정 목적의 작은 클래스를 제공합니다. 필요에 따라 이 작은 클래스를 HTML 요소에 직접 할당할 수 있습니다. Tailwind CSS가 유틸리티 우선 CSS 프레임워크에 해당합니다.

다음은 Tailwind CSS를 사용해 버튼을 스타일링하는 예시입니다.

```html
<button class="bg-blue-500 text-white font-bold py-2 px-4 rounded-full hover:bg-blue-700">
  Button
</button>
```

- **컴포넌트 기반 CSS 프레임워크**: 이 프레임워크는 미리 정의된 스타일을 가진 미리 만들어진 컴포넌트를 제공하며, 이를 웹사이트에 추가할 수 있습니다. 컴포넌트는 CSS 프레임워크 공식 문서에서 확인할 수 있고, 프로젝트에 복사해 붙여넣으면 됩니다. Bootstrap이 컴포넌트 기반 CSS 프레임워크에 해당합니다.

다음은 Bootstrap을 사용해 리스트 그룹을 만드는 예시입니다. HTML 요소에 작은 클래스를 적용하는 대신, HTML 구조를 포함한 전체 컴포넌트를 추가합니다.

```html
<div class="card" style="width: 25rem;">
  <ul class="list-group list-group-flush">
    <li class="list-group-item">HTML</li>
    <li class="list-group-item">CSS</li>
    <li class="list-group-item">JavaScript</li>
  </ul>
</div>
```

## Tailwind CSS

Tailwind는 유틸리티 우선 CSS 프레임워크입니다. 사용자 정의 CSS 규칙을 작성하는 대신, HTML 내에서 작은 유틸리티 클래스를 조합해 디자인을 만듭니다.

### 반응형 디자인 유틸리티

Tailwind는 `sm:`, `md:`, `lg:` 같은 접두사를 사용해 다양한 화면 크기에서 스타일을 적용합니다.

```html
<div class="w-full md:w-1/2 lg:flex-row">Responsive layout</div>
```

### 플렉스박스 유틸리티

`flex`, `flex-col`, `justify-around`, `items-center` 같은 클래스는 유연한 레이아웃을 쉽게 만듭니다.

```html
<div class="flex flex-col md:flex-row justify-around items-center">
  <p>Column on small screens</p>
  <p>Row on medium and larger screens</p>
</div>
```

### 그리드 유틸리티

Tailwind는 `grid`, `grid-cols-1`, `md:grid-cols-3` 같은 CSS Grid 유틸리티를 포함합니다.

```html
<div class="grid grid-cols-1 md:grid-cols-3 gap-8">
  <div class="bg-gray-100 p-4">Column 1</div>
  <div class="bg-gray-100 p-4">Column 2</div>
  <div class="bg-gray-100 p-4">Column 3</div>
</div>
```

### 간격 유틸리티

`mt-8`, `mx-auto`, `p-4`, `gap-4` 같은 유틸리티는 CSS를 작성하지 않고도 일관된 간격을 만듭니다.

```html
<div class="mt-8 p-4 bg-indigo-600 text-white">Spaced content</div>
```

### 타이포그래피 유틸리티

`uppercase`, `font-bold`, `font-semibold`, `text-4xl` 같은 유틸리티는 텍스트 모양을 제어합니다.

`text-3xl` `md:text-5xl` 같은 브레이크포인트에 맞춰 폰트 크기를 설정할 수 있습니다.

```html
<h1 class="text-3xl md:text-5xl font-semibold text-center">Responsive Heading
</h1>
```

### 색상과 호버 상태

Tailwind는 `text-red-700`, `bg-indigo-600`, `bg-gray-100` 같은 다양한 색상 팔레트를 제공합니다.

`hover:bg-pink-600` 같은 클래스는 인터랙티브 효과를 간단하게 만듭니다.

```html
<a href="#" class="bg-pink-500 hover:bg-pink-600 text-white px-4 py-2 rounded-md">
  Hover Me
</a>
```

### 테두리, 링, 효과

- **테두리**: `border-2 border-red-300`는 지정한 두께와 색상의 테두리를 추가합니다.
- **링**: `ring-1 ring-gray-300`는 포커스나 카드에 자주 쓰이는 윤곽선 같은 효과를 만듭니다.
- **둥근 모서리와 확대/축소**: `rounded-md`, `rounded-xl`, `scale-105` 같은 클래스는 세련된 효과를 더합니다.

```html
<div class="p-6 rounded-xl ring-2 ring-fuchsia-500 scale-105">
  Highlighted card
</div>
```

### 그라디언트

Tailwind는 `bg-gradient-to-r from-fuchsia-500 to-indigo-600` 같은 그라디언트 유틸리티를 지원합니다.

```html
<div class="p-4 text-white bg-gradient-to-r from-fuchsia-500 to-indigo-600">
  Gradient background
</div>
```

## CSS 전처리기

- **CSS 전처리기**: CSS 전처리기는 표준 CSS를 확장하는 도구입니다. 확장된 구문으로 작성한 코드를 네이티브 CSS 파일로 컴파일합니다. 복잡한 프로젝트에서 더 깔끔하고 재사용 가능하며 반복을 줄이고 확장 가능한 CSS를 작성하는 데 도움이 됩니다.
- **기능**: CSS 전처리기가 제공하는 기능에는 변수, 믹스인, 중첩, 선택자 상속 등이 있습니다.
- **인기 있는 CSS 전처리기**: 인기 있는 CSS 전처리기로는 Sass, Less, Stylus가 있습니다.  
- **잠재적 단점**:
  - CSS 규칙을 표준 CSS로 컴파일하는 데 오버헤드가 발생할 수 있습니다.
  - 컴파일된 코드는 디버그하기 어려울 수 있습니다.

## Sass

- **Sass**: 가장 인기 있는 CSS 전처리기 중 하나입니다. Sass는 "Syntactically Awesome Style Sheets"의 약자입니다.
- **Sass가 지원하는 기능**: 변수, 중첩된 CSS 규칙, 모듈, 믹스인, 상속, 기본 수학 연산자를 지원합니다.

## Sass가 지원하는 두 가지 구문

- **SCSS 구문**: SCSS(Sassy CSS)는 CSS 기본 구문을 확장한 것입니다. Sass에서 가장 널리 사용되는 구문이며, SCSS 파일은 `.scss` 확장자를 가집니다.

다음은 SCSS에서 변수를 정의하고 사용하는 예시입니다.

```scss
$primary-color: #3498eb;

header {
  background-color: $primary-color;
}
```

- **들여쓰기 구문**: 들여쓰기 구문은 Sass의 원래 구문이며 "Sass 구문"이라고도 합니다.

다음은 들여쓰기 구문에서 변수를 정의하고 사용하는 예시입니다.

```sass
$primary-color: #3498eb

header
  background-color: $primary-color
```

### 믹스인

- **믹스인**: 믹스인은 여러 CSS 속성과 값을 하나의 이름 아래 묶어 스타일시트 전체에서 재사용할 수 있게 합니다.

다음은 SCSS 구문으로 믹스인을 정의하는 예시입니다. 이 믹스인은 `center-flex`라는 이름이며, 플렉스박스를 사용해 요소를 가운데 정렬하는 세 가지 CSS 속성을 포함합니다.

```scss
@mixin center-flex {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

다음은 정의한 믹스인을 사용하는 예시입니다.

```scss
section {
  @include center-flex;
  height: 500px;
  background-color: #3289a8;
}
```
