# --description--

방 안에 원형 테이블 주위로 $N$개의 의자가 놓여 있습니다.

기사들이 한 명씩 방에 들어와서 빈 의자 중 무작위로 하나를 선택합니다.

충분한 공간을 확보하기 위해 기사들은 항상 서로 사이에 최소한 한 개의 빈 의자를 남겨 둡니다.

적합한 의자가 더 이상 없을 때, 빈 의자의 분수 $C$가 결정됩니다. 또한 $E(N)$을 $C$의 기댓값으로 정의합니다.

$E(4) = \frac{1}{2}$이고 $E(6) = \frac{5}{9}$임을 확인할 수 있습니다.

$E({10}^{18})$을 구하세요. 답은 소수점 아래 열네 자리까지 반올림하여 0.abcdefghijklmn 형식으로 제출하세요.
