# --interactive--

طريقة `map` هي دالة قوية وشائعة الاستخدام في JavaScript تعمل على المصفوفات. صُممت لإنشاء مصفوفة جديدة بتطبيق دالة معينة على كل عنصر من عناصر المصفوفة الأصلية.

هذه الطريقة لا تُعدل المصفوفة الأصلية، بل تُرجع مصفوفة جديدة تحتوي على نتائج تطبيق الدالة على كل عنصر.

إليك مثالًا على استخدام طريقة `map` على مصفوفة من الأرقام:

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map((num) => num * 2);
console.log(numbers); // [1, 2, 3, 4, 5]
console.log(doubled); // [2, 4, 6, 8, 10]
```

:::

لإنشاء مصفوفة جديدة حيث يتضاعف كل رقم، نستخدم طريقة `map`. تقبل طريقة `map` دالة استدعاء راجع تُستدعى على كل عنصر في المصفوفة.

في هذه الحالة، سيُضرب كل رقم في المصفوفة بـ `2`. النتيجة ستكون مصفوفة جديدة من الأرقام `2,4,6,8,10`.

يمكن لدالة الاستدعاء الراجع أن تقبل حتى ثلاث معلمات.

المعلمة الأولى هي العنصر الحالي الذي تتم معالجته.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element) => {
  console.log("Element:", element);
  return element * 2;
});
```

:::

المعلمة الثانية هي فهرس العنصر الحالي الذي تتم معالجته.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index) => {
  console.log("Element:", element);
  console.log("Index:", index);
  return element * 2;
});
```

:::

المعلمة الثالثة هي المصفوفة التي تُستدعى عليها طريقة `map`.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index, array) => {
  console.log("Element:", element);
  console.log("Index:", index);
  console.log("Array:", array);
  return element * 2;
});
```

:::

فهم واستخدام طريقة `map` بفعالية يمكن أن يحسن كثيرًا قدرتك على التعامل مع المصفوفات في JavaScript. في الدروس القادمة، سنتعمق أكثر في استخدامات متقدمة لـ `map` ونستكشف كيف يمكن أن تكون أداة قوية لبناء برامج ديناميكية وفعالة.
