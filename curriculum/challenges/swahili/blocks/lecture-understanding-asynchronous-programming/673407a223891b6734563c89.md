# --description--

API ya Fetch inaruhusu programu za mtandao kufanya maombi ya mtandao, kawaida ili kupata au kutuma data kwa seva. API hii hutoa njia ya `fetch()` ambayo unaweza kutumia kufanya maombi haya. Tuchunguze mfano wa msingi wa jinsi ya kutumia `fetch`:

```js
fetch('https://api.example.com/data')
```

Katika mfano huu, tunafanya ombi la `GET` kwa `https://api.example.com/data`. Hii itaturudishia data ambayo lazima ibadilishwe kuwa muundo wa JSON kabla haijatumika.

Kwa chaguo-msingi, API ya Fetch hutumia njia ya `GET` kupata data. Hii itajadiliwa katika funzo lijalo, pamoja na njia nyingine za kawaida za maombi ya HTTP.

Sasa, tuchambue aina za kawaida za rasilimali zinazopatikana kutoka mtandao.

Katika programu zetu za mtandao, tunahitaji data za kawaida kama data ya hali ya hewa, orodha ya taaluma, orodha ya majina ya nchi, nambari za nchi au picha za bendera za nchi. Kwa kutumia data hizi tunaweza kufanya programu yetu kuwa na taarifa zaidi na kuingiliana zaidi. Shukrani kwa API ya Fetch, tunaweza kupata rasilimali hizi kutoka mtandao.

Picha ni baadhi ya rasilimali zinazopatikana mara kwa mara. Unaweza kutumia fetch kupakia picha kwa njia ya kimya au kwa nguvu kulingana na vitendo vya mtumizi, na kuonyesha kwenye programu yako ya mtandao.

Faili za maandishi ni aina nyingine ya rasilimali zinazopatikana mara kwa mara. Hii inaweza kujumuisha faili za usanidi, faili za kumbukumbu, au hata nyaraka nzima unazotaka kuonyesha kwenye ukurasa wako wa mtandao.

Katika baadhi ya matukio, unaweza kupata faili za sauti au video. API ya Fetch inaweza kushughulikia aina hizi za rasilimali pia, ikikuruhusu kufanya kazi na aina mbalimbali za data.
