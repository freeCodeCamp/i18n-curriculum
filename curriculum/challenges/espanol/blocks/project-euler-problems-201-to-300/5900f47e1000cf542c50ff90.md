# --description--

Considere ecuaciones de la forma: $a^2 + b^2 = N$, $0 ► b$, $a$, $b$ y $N$ entero.

Para $N = 65$ hay dos soluciones:

$a = 1, b = 8$ y $a = 4, b = 7$.

Llamamos a $S(N)$ la suma de los valores de $a$ de todas las soluciones de $a^2 + b^2 = N$, $0 ≤ a ≤ b$, $a$, $b$ y $N$ entero.

Así que $S(65) = 1 + 4 = 5$.

Find $\sum S(N)$, for all squarefree $N$ only divisible by primes of the form $4k + 1$ with $4k + 1 &lt; 150$.
