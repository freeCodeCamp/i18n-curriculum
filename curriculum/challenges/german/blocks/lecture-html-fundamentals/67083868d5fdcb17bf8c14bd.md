# --interactive--

Eine HTML-Entity oder Zeichenreferenz ist eine Zeichenfolge, die verwendet wird, um ein reserviertes Zeichen in HTML darzustellen.

Angenommen, Sie möchten den Text `This is an <img/> element` auf dem Bildschirm anzeigen. Wenn Sie den Code verwenden, der sich derzeit im Editor befindet, wird das gewünschte Ergebnis nicht angezeigt. Selbst wenn Sie `src` und `alt` Attribute zum Beispiel hinzufügen, würde in der Mitte des Absatzes ein Bild angezeigt werden. Das ist nicht das gewünschte Ergebnis. Um mit dem Beispiel zu interagieren, müssen Sie den interaktiven Editor aktivieren.


:::interactive_editor

```html
<p>This is an <img /> element</p>
```

:::

Wenn der HTML-Parser das Kleiner-als-Symbol (`<`) gefolgt von einem HTML-Tag-Namen sieht, interpretiert er das als ein HTML-Element. Deshalb erhalten Sie nicht das gewünschte Ergebnis von `This is an <img/> element` auf dem Bildschirm.

Um dieses Problem zu beheben, können Sie HTML-Entities verwenden. Hier ist ein aktualisiertes Beispiel mit den korrekten HTML-Entities für die Kleiner-als- (`<`) und Größer-als- (`>`) Zeichen. Jetzt sollten Sie `This is an <img/> element` auf dem Bildschirm sehen.

Aktivieren Sie den interaktiven Editor und versuchen Sie, ein `&lt;p&gt;learning is fun&lt;/p&gt;` unter dem Absatz-Element hinzuzufügen. Sie sollten `<p>learning is fun</p>` auf dem Bildschirm sehen.

:::interactive_editor

```html
<p>This is an &lt;img /&gt; element</p>

```

:::

Diese Art von Zeichenreferenzen werden als benannte Zeichenreferenzen bezeichnet. Benannte Referenzen beginnen mit einem kaufmännischen Und-Zeichen (`&`) und enden mit einem Semikolon (`;`). Durch die Verwendung einer benannten Zeichenreferenz verwechselt der HTML-Parser diese nicht mit einem tatsächlichen HTML-Element.

Eine andere Art von Zeichenreferenz ist die dezimale numerische Referenz. Diese Zeichenreferenz beginnt mit einem kaufmännischen Und-Zeichen und einem Rautezeichen (`#`), gefolgt von einer oder mehreren Dezimalziffern, und endet mit einem Semikolon.

Hier ist ein Beispiel für die Verwendung der dezimalen numerischen Referenz für das Kleiner-als-Symbol.

Aktivieren Sie den interaktiven Editor und ändern Sie den Code, um verschiedene Symbole zu sehen. Sie können `&#169;` für das Copyright-Symbol und `&#174;` für das eingetragene Markenzeichen verwenden.

:::interactive_editor

```html
&#60;
```

:::

Die letzte Art von Zeichenreferenz ist die hexadezimale numerische Referenz. Diese Zeichenreferenz beginnt mit einem kaufmännischen Und-Zeichen, einem Rautezeichen und dem Buchstaben `x`. Danach folgen eine oder mehrere ASCII-Hexadezimalziffern, und sie endet mit einem Semikolon.

Hier ist ein Beispiel für die Verwendung der hexadezimalen numerischen Referenz für das Kleiner-als-Symbol.

Aktivieren Sie den interaktiven Editor und ändern Sie den Code, um verschiedene Symbole zu sehen. Sie können `&#x20AC;` für das Euro-Symbol und `&#x03A9;` für das griechische Großbuchstaben-Omega-Symbol verwenden.

:::interactive_editor

```html
&#x3C;
```

:::
