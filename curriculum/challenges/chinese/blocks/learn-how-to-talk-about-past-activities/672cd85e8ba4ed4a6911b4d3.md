# --explanation--

`We've` 是 `we have` 的缩写。

`Made` 是不规则动词 `make` 的过去分词。`Have made` 描述过去创建或完成且对现在有影响的事物。

在这个句子中，`have had` 是现在完成时。这个结构 `have` + `had` 表示某事从过去开始并且仍然影响现在。 

- `Have` 是一个用于现在完成时的辅助动词。

- `Had` 是 `have` 的过去分词，意思是经历或拥有。

合起来，`have had` 意味着这些变化较早开始，并且现在仍在产生影响。

总之，这些不规则动词（`make` 的过去式为 `made`，`have` 的过去式为 `had`）构成现在完成时，表示过去开始且现在仍然相关的动作。
