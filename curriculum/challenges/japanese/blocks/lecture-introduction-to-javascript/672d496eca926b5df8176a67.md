# --interactive--

JavaScriptでは、データ型とは数値やテキストのように保存する値の種類のことです。

変数は特定のデータ型の値を保存する名前付きのコンテナで、コード全体で参照したり操作したりできます。

数学の授業で次のような変数を扱ったことを覚えているかもしれません：

```md
x = 2
y = 4

x + y
```

`x`や`y`のような変数を作成し、それらをプログラム全体で参照して加算などの数学的操作を行うことができました。プログラミングでも同じような考え方です。自分で変数名を作成し、値を代入できます。これらの値は異なるデータ型になります。

データ型は、プログラムが扱うデータの種類を理解するのに役立ちます。数値なのかテキストなのか、あるいは別のものなのかを判別します。

JavaScriptにはプログラムで使う基本的なデータ型がいくつかあります。これらのデータ型については今後のレッスンで詳しく学びます。ここではJavaScriptのさまざまなデータ型の簡単な紹介をします。

最初に見るデータ型は`Number`型です。

`Number`は整数と浮動小数点の両方の値を表します。整数の例には`7`、`19`、`90`があります。

**注意**：`console.log()`はコンソールに情報を出力する関数で、これはコードのデバッグに使うウェブブラウザの一部です。`console.log()`については今後のレッスンで詳しく学びます。また、`//`の記号はコード内にコメントを追加するために使います。コメントはコード実行時に無視される、自分や他のプログラマーのためのメモです。

インタラクティブエディターを有効にして、いくつかの整数を変更してコンソールの更新を確認してください。

:::interactive_editor

```js
// Examples of integers
console.log(3);
console.log(5);
console.log(-67);
```

:::

浮動小数点数は小数点を含む数値です。浮動小数点数の例には`3.14`や`5.2`があります。

:::interactive_editor

```js
// Examples of floating point numbers
console.log(3.14);
console.log(7.2);
console.log(-14.5);
```

:::

次のデータ型は`String`です。

`String`は引用符で囲まれた文字のシーケンス、つまりテキストです。二重引用符を使った文字列の例を示します：

:::interactive_editor

```js
console.log("I love to code!");
```

:::

こちらは単一引用符を使った例です：

:::interactive_editor

```js
console.log('I love to code!');
```

:::

文字列は名前やラベル、アラートメッセージなどを表すのによく使います。

JavaScriptで使われるもう一つのデータ型は`Boolean`型です。

`Boolean`は`true`か`false`のどちらかの値を表します。例えば、プログラムがユーザーがログインしているか（`true`）していないか（`false`）をチェックし、それに応じてページを変更する場合があります。ユーザーがログインしていればダッシュボードページを表示し、そうでなければログインページを表示したいでしょう。

JavaScriptで使われる次の2つのデータ型は`undefined`と`null`です。

`undefined`は変数が宣言されたがまだ値が代入されていないことを意味します。これは次のレッスンで詳しく学びます。

`null`は変数が意図的に「何もない」状態に設定されていて、値を持っていないことを意味します。これについても今後のレッスンで詳しく学びます。

最後の3つのデータ型はより複雑です。これらはオブジェクト、シンボル、そして`BigInt`です。

`Object`はキーと値のペアの集合です。

```js
{
  name: "Alice",
  age: 30
};
```

オブジェクトは関連する情報をまとめるのに適しています。オブジェクトの扱い方は今後のモジュールで詳しく学びます。

`Symbol`はJavaScriptで常にユニークで変更できない特別な値の型です。プロパティのユニークなラベルや識別子を作るのに使われます：

```js
Symbol('mySymbol');
```

`BigInt`は`Number`型の制限を超える非常に大きな数値に使われます：

```js
1234567890123456789012345678901234567890n;
```

この例では、非常に大きな数値の末尾に`BigInt`を付けて`n`を作成しています。

`Symbol`と`BigInt`はあまり一般的ではない2つの型ですが、知っておくことは重要です。

これらのデータ型を理解することで、プログラム内のさまざまな種類のデータを適切に扱うことができます。各型にはそれぞれ特徴や動作があります。
