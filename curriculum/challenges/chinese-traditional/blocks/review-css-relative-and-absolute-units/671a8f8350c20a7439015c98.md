## 絕對單位

- **`px` (像素)**：這個絕對單位是在 CSS 中固定大小的測量單位。它是最常見的絕對單位，並提供對尺寸的精確控制。`1px` 永遠相當於 1/96 英吋。
- **`in` (Inch)**：此絕對單位相當於 96px。
- **`cm` (公分)**：此絕對單位相當於 25.2/64 英吋。
- **`mm` (毫米)**：此絕對單位相當於 1/10 公分。
- **`q` (四分之一毫米)**：這個絕對單位相當於 1/40 公分。
- **`pc` (Picas)**：這個絕對單位相當於 1/6 英吋。
- **`pt` (Points)**：此絕對單位相當於 1/72 英吋。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="units">
  <div class="unit px">px</div>
  <div class="unit inch">in</div>
  <div class="unit cm">cm</div>
  <div class="unit mm">mm</div>
  <div class="unit q">q</div>
  <div class="unit pc">pc</div>
  <div class="unit pt">pt</div>
</div>
```

```css
.units {
  display: flex;

}

  width: 40px;
  height: 40px;

}

  height: 1cm;
}

  width: 10mm;
  height: 10mm;
}

}

.pc {
  width: 6pc;
  height: 6pc;
}

.pt {
  width: 36pt;
  height: 36pt;
}
```

## 相對單位

- **百分比**：這些相對單位允許你將大小、尺寸和其他屬性定義為其父元素的比例。<dfn>例如</dfn>，如果你在元素上設定 `width: 50%;`，它將佔據其父容器寬度的一半。
- **`em` 單位**：這些單位是相對於元素的字型大小。如果你在 `font-size` 屬性中使用 `ems`，文字的大小將相對於父元素的字型大小。
- **`rem` 單位**：這些單位是相對於根元素的字型大小，也就是 `html` 元素。
- **`vh` 單位**：`vh` 代表「viewport height」，`1vh` 相當於視窗高度的 1%。
- **`vw` 單位**：`vw` 代表「viewport width」，`1vw` 相當於視窗寬度的 1%。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  <div class="box percent">50%</div>
  <div class="box em">2em</div>
  <div class="box rem">2rem</div>
  <div class="box vh">10vh</div>
  <div class="box vw">10vw</div>
</div>
```

```css
html {
  font-size: 16px;
}

.parent {
  width: 200px;
  font-size: 20px;
  border: 2px dashed #555;
  padding: 10px;
  display: flex;
  flex-direction: column;
  gap: 12px;
}

.box {
  background: seagreen;
  color: white;
  text-align: center;
  padding: 6px;
}

.percent {
  width: 50%;
}

.em {
  font-size: 2em;
}

.rem {
  font-size: 2rem;
}

.vh {
  height: 10vh;
}

.vw {
  width: 10vw;
}
```

## `calc` 函式（程式）

- **`calc()` 函式（程式）**：使用 `calc()` 函式（程式），你可以直接在你的樣式表中執行計算，以動態決定屬性值。這表示你可以透過根據視窗大小或其他元素計算尺寸，來創建彈性且響應式的使用者介面。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="calc-box">calc()</div>
```

```css
.calc-box {
  width: calc(100% - 40px);
  padding: 10px;
  background: steelblue;
  color: white;
  text-align: center;
  border: 2px solid black;
}
```
