# --interactive--

Kutakuwa na nyakati ambapo utahitaji kuongeza au kuondoa nodi kutoka DOM na kuna baadhi ya API za Mtandao unazoweza kutumia.

Katika funzo hili, tutazungumzia njia za `appendChild()` na `removeChild()`.

Njia ya `appendChild()` hutumika kuongeza nodi mwishoni mwa orodha ya watoto wa nodi ya mzazi iliyotajwa.

Hapa kuna sintaksia ya msingi kwa njia ya `appendChild()`:

```js
parentNode.appendChild(newNode);
```

Tuchunguze mfano ili kuelewa vizuri jinsi ya kutumia njia ya `appendChild()`.

Katika mfano huu, tuna kipengele cha orodha isiyopangwa yenye kitambulisho cha `desserts` ndani ya HTML. Orodha hii ina vitu viwili vya orodha vya `Cake` na `Pie`:

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
```

Ili kupata orodha hiyo ndani ya faili la JavaScript, tunaweza kutumia njia ya `getElementById()`:

```js
const dessertsList = document.getElementById("desserts");
```

Tuna kigezo kinachoitwa `dessertsList` kinachohifadhi rejea kwa kipengele cha `ul` chenye kitambulisho cha `desserts`.

Kisha tunahitaji kuunda kipengele kipya cha kipengele cha orodha kwa kutumia njia ya `createElement()`:

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");
```

Msimbo ufuatao utaunda kipengele kipya cha kipengele cha orodha na kuuhifadhi katika kigezo kinachoitwa `listItem`.

Ili kuongeza nodi hiyo mwishoni mwa orodha ya watoto wa kipengele cha `ul`, tunaweza kutumia njia ya `appendChild()`:

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

dessertsList.appendChild(listItem);
```

Ukikimbiza msimbo huu, utaona kwamba kipengele kipya cha kipengele cha orodha kimeongezwa mwishoni mwa orodha ya watoto wa kipengele cha `ul`.

:::interactive_editor

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
<script src="index.js"></script>
```

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

dessertsList.appendChild(listItem);
```

:::

Tatizo ni kwamba kipengele kipya cha kipengele cha orodha ni tupu. Ili kuongeza maudhui ya maandishi kwenye kipengele kipya cha kipengele cha orodha, unaweza kutumia sifa ya `textContent`:

:::interactive_editor

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
<script src="index.js"></script>
```

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

listItem.textContent = "Cookies";
dessertsList.appendChild(listItem);
```

:::

Sasa orodha itaonyesha `Cake`, `Pie`, na `Cookies`.

Ili kuondoa nodi kutoka DOM, unaweza kutumia njia ya `removeChild()`.

Hapa kuna sintaksia ya msingi kwa njia ya `removeChild()`:

```js
parentNode.removeChild(childNode);
```

Tuchunguze mfano ambapo tunataka kuondoa kipengele cha aya cha mwisho kutoka kwa kipengele cha `section`:

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
```

Tunaweza kupata kipengele cha `section` ndani ya faili la JavaScript kwa kutumia njia ya `getElementById()`:

```js
const sectionEl = document.getElementById("example-section");
```

Mara tunapopata rejea kwa kipengele cha `section`, tunaweza kisha kupata aya ya mwisho kwa kutumia njia ya `querySelector()`:

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");
```

Tunatumia darasa la bandia la `:last-of-type` kuchagua kipengele cha aya cha mwisho ndani ya kipengele cha `section`.

Sasa tunapokuwa na nodi ya mzazi na nodi ya mtoto, tunaweza kuondoa kipengele cha aya cha mwisho kutoka kwa kipengele cha `section` kwa kutumia njia ya `removeChild()`:

:::interactive_editor

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
<script src="index.js"></script>
```

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");

sectionEl.removeChild(lastParagraph);
```

:::

Basi, ni lini unaweza kutaka kuongeza au kuondoa nodi kutoka DOM?

Ikiwa unafanya kazi na maudhui yanayobadilika, unaweza kuhitaji kuongeza au kuondoa nodi kutoka DOM. Mifano halisi ni pamoja na vikapu vya ununuzi, orodha za kazi za kufanya, na vyanzo vya habari vya mitandao ya kijamii.
