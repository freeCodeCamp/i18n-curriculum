# --description--

У попередньому уроці ви дізналися, як працювати з типом даних кортеж. У цьому уроці ви ознайомитеся з деякими загальними методами, які використовують при роботі з кортежами.

Перший метод, який ми розглянемо, — це `count()`. Цей метод використовується, щоб визначити, скільки разів елемент зустрічається в кортежі. Ось приклад перевірки, скільки разів рядок `Rust` з’являється в кортежі з назвою `programming_languages`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

Оскільки `Rust` зустрічається двічі в кортежі, метод `count()` повертає число `2`. Якщо вказаний елемент у функції `count()` зовсім відсутній у кортежі, тоді повернене значення буде `0`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

Якщо в функцію `count()` не передати аргументів, Python викликає `TypeError`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count()

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: tuple.count() takes exactly one argument (0 given)
"""
```

Наступний метод, який ми розглянемо, — це метод `index()`. Цей метод використовується, щоб знайти індекс, де певний елемент присутній у кортежі. Ось приклад використання методу `index()` для пошуку індексу рядка `Java`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

Якщо вказаний елемент не можна знайти, Python викликає `ValueError`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: tuple.index(x): x not in tuple
"""
```

Ще одна можливість методу `index()` — передати необов’язкові аргументи початкового та кінцевого індексів. Ось приклад передачі необов’язкового початкового індексу:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

У цьому прикладі ми вказуємо, з якого місця починати пошук рядка `Python`. Передаючи число `3` як другий аргумент у функцію `index()`, ми вказуємо почати пошук з індексу `3`. Оскільки `Python` зустрічається двічі в кортежі, функція `index()` поверне індекс `5` замість індексу `2` через використання необов’язкового аргументу початкового індексу.

Ви також можете передати необов’язковий кінцевий індекс. Ось змінений приклад із вказанням початкового та кінцевого індексів:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

Тепер результатом буде індекс `2`, оскільки ми починаємо пошук з індексу `2` і шукаємо до, але не включно, індексу `5`.

Ще одна часто використовувана функція з кортежами — це функція `sorted()`. У попередньому уроці ви дізналися про метод `sort()` для списків. Функцію `sorted()` можна застосовувати до будь-якого ітерованого об’єкта, включно з кортежами.

Ось приклад створення нового списку чисел за допомогою функції `sorted()`:

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

Функція `sorted()` завжди створює новий список відсортованих значень. Це відрізняється від методу `sort()`, який сортує елементи списку на місці і не повертає новий список.

Якщо потрібно налаштувати поведінку сортування для ітерованого об’єкта, можна використати необов’язкові аргументи `reverse` і `key`. Ось приклад використання аргументу `key` для сортування елементів кортежу за довжиною:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

Якщо ви хочете створити новий список значень у зворотному порядку, можна використати аргумент `reverse` ось так:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

Кортежі — це загальний тип даних у Python. Розуміння того, як з ними працювати, а також деякі корисні методи й функції допоможуть вам писати більш ефективний код.
