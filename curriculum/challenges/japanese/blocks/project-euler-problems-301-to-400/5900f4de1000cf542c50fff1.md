# --description--

幾何学的三角形を、辺の長さが整数であり、辺の長さが $a ≤ b ≤ c$ を満たし、辺が等比数列を形成する三角形、すなわち $b^2 = a \times c$ を満たす三角形と定義します。

このような幾何学的三角形の例として、辺の長さが $a = 144$、$b = 156$、$c = 169$ の三角形があります。

$\text{周囲の長さ} ≤ {10}^6$ の幾何学的三角形は $861\,805$ 個存在します。

$\text{周囲の長さ} ≤ 2.5 \times {10}^{13}$ の幾何学的三角形は何個存在するでしょうか？
