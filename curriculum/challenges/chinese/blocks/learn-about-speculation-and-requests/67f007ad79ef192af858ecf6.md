# --explanation--

Jake 可以说 `It's possible`，因为他认为可疑邮件有可能是钓鱼攻击的一部分。

`Impossible` 是通过为基础词 `possible` 添加前缀 `im-`（意思是“不”或“相反的”）形成的。例如：

- `It's possible.` - 事情可能发生。

- `It's impossible.` - 它不可能发生。
