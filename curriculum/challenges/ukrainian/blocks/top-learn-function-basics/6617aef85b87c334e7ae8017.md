# --description--

Стрілкові функції — це більш сучасний спосіб написання функцій у JavaScript. Вони забезпечують більш компактний синтаксис у порівнянні з традиційними функціональними виразами. Стрілкові функції визначаються за допомогою синтаксису `=>`.

```js
const add = (a, b) => {
  return a + b;
}
```

У наведеному вище прикладі стрілкова функція `add` приймає два параметри `a` і `b` та повертає їх суму. Ключове слово `return` використовується для повернення результату операції додавання.

Якщо стрілкова функція має лише один вираз, фігурні дужки `{}` і ключове слово `return` можна опустити. Вираз буде повернений неявно.

```js
const add = (a, b) => a + b;
```

У наведеному вище прикладі стрілкова функція `add` приймає два параметри `a` і `b` та повертає їх суму. Ключове слово `return` і фігурні дужки `{}` опущені, і результат операції додавання повертається неявно.
