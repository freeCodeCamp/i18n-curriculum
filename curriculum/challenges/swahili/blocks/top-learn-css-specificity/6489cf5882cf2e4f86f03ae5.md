# --description--

Sasa, tubadilishe mambo kidogo:

```html
<!-- index.html -->

<div class="main">
  <div class="list" id="subsection"></div>
</div>
```

```css
/* rule 1 */
#subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

Katika mfano hapo juu, licha ya kanuni ya 2 kuwa na vichaguzi vya darasa la vitu zaidi kuliko vichaguzi vya ID, kanuni ya 1 ni maalum zaidi kwa sababu ID inashinda darasa la vitu.
