# --description--

중국어로 자신의 이름을 말하고 싶을 때, 아주 흔하고 자연스러운 구조를 사용할 수 있습니다:

`​我是 (wǒ shì)​​` + 이름.

이는 "나는..." 또는 "내 이름은..."이라는 뜻입니다. `我 (wǒ)`은 "나"를 의미합니다. `是 (shì)`는 영어의 동사 "to be"에 해당한다고 생각하세요.

# --explanation--

`我是 (wǒ shì)​​`은 "나는..." 또는 "내 이름은..."이라는 뜻입니다. "나"를 의미하는 대명사 `是 (shì)` 다음에 동사 `我 (wǒ)`를 사용하여 정체성을 나타냅니다. 예를 들어:

`我是刘明 (wǒ shì liú míng)。` — 나는 리우밍입니다.
