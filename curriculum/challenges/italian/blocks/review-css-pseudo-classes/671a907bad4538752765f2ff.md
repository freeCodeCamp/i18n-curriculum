## Pseudo-classi per azioni dell’utente

- **Definizione di pseudo-classi**: Sono parole chiave speciali di CSS che ti permettono di selezionare un elemento in base al suo stato o posizione specifica.
- **Pseudo-classi per azioni dell’utente**: Sono parole chiave speciali che ti consentono di modificare l’aspetto degli elementi in base alle interazioni dell’utente, migliorando l’esperienza complessiva.
- **Pseudo-classe `:active`**: Questa pseudo-classe ti permette di selezionare lo stato attivo di un elemento, come quando si clicca su un pulsante.
- **Pseudo-classe `:hover`**: Questa pseudo-classe definisce lo stato hover di un elemento.
- **Pseudo-classe `:focus`**: Questa pseudo-classe applica stili quando un elemento riceve il focus, tipicamente tramite la navigazione da tastiera o quando un utente clicca in un campo di input di un modulo.
- **Pseudo-classe `:focus-within`**: Questa pseudo-classe si usa per applicare stili a un elemento quando esso o uno qualsiasi dei suoi discendenti ha il focus.

## Pseudo-classi per input

- **Pseudo-classi per input**: Queste pseudo-classi si usano per selezionare elementi HTML `input` in base allo stato in cui si trovano prima e dopo l’interazione dell’utente.
- **Pseudo-classe `:enabled`**: Questa pseudo-classe si usa per selezionare pulsanti di modulo o altri elementi attualmente abilitati.
- **Pseudo-classe `:disabled`**: Questa pseudo-classe ti permette di definire lo stile di un elemento interattivo in modalità disabilitata.
- **Pseudo-classe `:checked`**: Questa pseudo-classe indica all’utente che un elemento è selezionato.
- **Pseudo-classe `:valid`**: Questa pseudo-classe seleziona i campi di input che soddisfano i criteri di validazione.
- **Pseudo-classe `:invalid`**: Questa pseudo-classe seleziona i campi di input che non soddisfano i criteri di validazione.
- **Pseudo-classi `:in-range` e `:out-of-range`**: Queste pseudo-classi applicano stili agli elementi in base al fatto che i loro valori siano entro o fuori dai limiti di intervallo specificati.
- **Pseudo-classe `:required`**: Questa pseudo-classe seleziona gli elementi `input` che hanno l’attributo `required`. Indica all’utente che deve compilare il campo per inviare il modulo.
- **Pseudo-classe `:optional`**: Questa pseudo-classe applica stili agli elementi di input che non sono obbligatori e possono essere lasciati vuoti.
- **Pseudo-classe `:autofill`**: Questa pseudo-classe applica stili ai campi di input che il browser compila automaticamente con dati salvati.

## Pseudo-classi di posizione

- **Pseudo-classi di posizione**: Queste pseudo-classi si usano per definire lo stile di collegamenti e elementi che sono obiettivi all’interno del documento corrente.
- **Pseudo-classe `:any-link`**: Questa pseudo-classe è una combinazione delle pseudo-classi `:link` e `:visited`. Quindi, corrisponde a qualsiasi elemento àncora con un attributo href, indipendentemente dal fatto che sia stato visitato o meno.
- **Pseudo-classe `:link`**: Questa pseudo-classe ti permette di selezionare tutti i collegamenti non visitati su una pagina web. Puoi usarla per definire uno stile diverso prima che l’utente ci clicchi sopra.
- **Pseudo-classe `:local-link`**: Questa pseudo-classe seleziona i collegamenti che puntano allo stesso documento. È utile quando vuoi distinguere i collegamenti interni da quelli esterni.
- **Pseudo-classe `:visited`**: Questa pseudo-classe seleziona un collegamento che l’utente ha già visitato.
- **Pseudo-classe `:target`**: Questa pseudo-classe si usa per applicare stili a un elemento che è l’obiettivo di un frammento URL.

## Pseudo-classi strutturali dell’albero

- **Pseudo-classi strutturali dell’albero**: Queste pseudo-classi ti permettono di selezionare e definire lo stile degli elementi in base alla loro posizione nell’albero del documento.
- **Pseudo-classe `:root`**: Questa pseudo-classe è solitamente l’elemento radice `html`. Ti aiuta a selezionare il livello più alto nel documento per applicare uno stile comune all’intero documento.
- **Pseudo-classe `:empty`**: Gli elementi vuoti, cioè quelli senza figli diversi dallo spazio bianco, sono inclusi nell’albero del documento. Per questo esiste la pseudo-classe `:empty` per selezionare gli elementi vuoti.
- **Pseudo-classe `:nth-child(n)`**: Questa pseudo-classe ti permette di selezionare elementi in base alla loro posizione all’interno di un genitore.
- **Pseudo-classe `:nth-last-child(n)`**: Questa pseudo-classe ti consente di selezionare elementi contando dalla fine.
- **Pseudo-classe `:first-child`**: Questa pseudo-classe seleziona il primo elemento in un genitore o nel documento.
- **Pseudo-classe `:last-child`**: Questa pseudo-classe seleziona l’ultimo elemento in un genitore o nel documento.
- **Pseudo-classe `:only-child`**: Questa pseudo-classe seleziona l’unico elemento in un genitore o nel documento.
- **Pseudo-classe `:first-of-type`**: Questa pseudo-classe seleziona la prima occorrenza di un tipo specifico di elemento all’interno del suo genitore.
- **Pseudo-classe `:last-of-type`**: Questa pseudo-classe seleziona l’ultima occorrenza di un tipo specifico di elemento all’interno del suo genitore.
- **Pseudo-classe `:nth-of-type(n)`**: Questa pseudo-classe ti permette di selezionare un elemento specifico all’interno del suo genitore in base alla sua posizione tra fratelli dello stesso tipo.
- **Pseudo-classe `:only-of-type`**: Questa pseudo-classe seleziona un elemento se è l’unico del suo tipo all’interno del suo genitore.

## Pseudo-classi funzionali

- **Pseudo-classi funzionali**: Le pseudo-classi funzionali ti permettono di selezionare elementi basandoti su condizioni o relazioni più complesse. A differenza delle pseudo-classi normali che selezionano elementi in base a uno stato (per esempio `:hover`, `:focus`), le pseudo-classi funzionali accettano argomenti.
- **Pseudo-classe `:is()`**: Questa pseudo-classe prende una lista di selettori (es. `ol`, `ul`) e seleziona un elemento che corrisponde a uno dei selettori nella lista.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<p class="example">This text will change color.</p>
<p>This text will not change color.</p>
<p>This text will not change color.</p>
<p class="this-works-too">This text will change color.</p>
```

```css
p:is(.example, .this-works-too) {
    color: red;
}
```

:::

- **Pseudo-classe `:where()`**: Questa pseudo-classe prende una lista di selettori (es. `ol`, `ul`) e seleziona un elemento che corrisponde a uno dei selettori nella lista. La differenza tra `:is` e `:where` è che quest’ultima avrà una specificità pari a 0.

```css
:where(h1, h2, h3) {
    margin: 0;
    padding: 0;
}
```

- **Pseudo-classe `:has()`**: Questa pseudo-classe è spesso chiamata selettore `"parent"` perché ti permette di definire lo stile degli elementi che contengono figli specificati nella lista di selettori.

```css
article:has(h2) {
    border: 2px solid hotpink;
}
```

- **Pseudo-classe `:not()`**: Questa pseudo-classe si usa per selezionare elementi che non corrispondono al selettore fornito.

```css
p:not(.example) {
  color: blue;
}
```

## Pseudo-elementi

- **Pseudo-elemento `::before`**: Questo pseudo-elemento usa la proprietà `content` per inserire contenuti cosmetici come icone subito prima dell’elemento.
- **Pseudo-elemento `::after`**: Questo pseudo-elemento usa la proprietà `content` per inserire contenuti cosmetici come icone subito dopo l’elemento.
- **Pseudo-elemento `::first-letter`**: Questo pseudo-elemento seleziona la prima lettera del contenuto di un elemento, permettendoti di definirne lo stile.
- **Pseudo-elemento `::marker`**: Questo pseudo-elemento ti permette di selezionare il marcatore (punto elenco o numerazione) degli elementi di una lista per definirne lo stile.
