# --interactive--

HTMLとCSSでは、要素は**インライン要素**か**ブロックレベル要素**のどちらかに分類され、この分類によってドキュメントのレイアウトでの振る舞いが決まります。

この違いを理解することは、ウェブページ上でコンテンツの表示をコントロールするために非常に重要です。

ブロックレベル要素は、デフォルトでは利用可能な幅いっぱいに広がり、コンテナの幅全体を占めます。

これらの要素は常に新しい行から始まり、他のコンテンツを次の行に押し出して「ブロック」のコンテンツを作ります。

ブロックレベル要素にはデフォルトでCSSプロパティ`display: block;`が適用されています。このプロパティによって、要素はコンテナの幅いっぱいに広がり、新しい行に表示されます。

よく使われるブロックレベル要素には、`div`要素、段落、見出し、番号付きリスト、番号なしリスト、セクション要素などがあります。

以下はブロックレベル要素のコード例です：

:::interactive_editor

```html
<p style="border: 2px solid red;">
  First paragraph
</p>
<p>Second paragraph</p>
```

:::

この例では、2つの段落要素があり、最初の段落には赤いボーダーが付いています。

2つの段落要素はデフォルトでブロックレベル要素なので、同じ行には並びません。

そのため、両方の段落要素はコンテナの幅いっぱいを占めます。この場合、コンテナは`body`要素です。

ブロックレベル要素は、段落やセクション、大きなコンテンツのブロックのように、縦に積み重ねたいコンテンツに最適です。ウェブページのレイアウトや構造を定義するためによく使われます。

一方、インライン要素はブロックレベル要素とは異なり、必要な幅だけを占め、新しい行から始まりません。これらの要素はコンテンツの流れの中にあり、テキストや他のインライン要素と並んで表示されます。

インライン要素にはデフォルトでCSSプロパティ`display: inline;`が適用されています。このプロパティによって、要素はコンテンツの流れの中にとどまり、新しい行に分断されません。

よく使われるインライン要素には、`span`、`anchor`、`img`要素があります。

インライン要素をよりよく理解するための例を示します：

:::interactive_editor

```html
<p>This is a
  <span style="color: red; border: 2px solid green;">red</span>
  word inside a paragraph.
</p>
```

:::

この例では、段落要素の中に`span`要素がネストされています。`span`要素は`red`のテキストカラーと`green`のボーダーが付いていて、強調された単語が見やすくなっています。

ご覧の通り、`span`要素は「red」という単語の幅だけを占めており、後続のコンテンツを新しい行に押し出していません。

インライン要素は、行内のテキストやコンテンツの一部をスタイリングするのに最適です。例えば、単語の強調、ハイパーリンクの作成、段落の一部に特定のスタイルを適用するときに使います。
