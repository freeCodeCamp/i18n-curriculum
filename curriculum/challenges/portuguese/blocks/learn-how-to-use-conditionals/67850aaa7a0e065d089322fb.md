### --feedback--

Essas duas palavras são usadas no `Second Conditional` para descrever o impacto ou resultado da condição hipotética.

# --explanation--

A frase usa o `Second Conditional`, que é estruturado como:

`If` + `Past Simple` ou `Past Continuous` tense, `might / would / could` + base verb. Neste exemplo:

`If they closed` é a condição (imaginando um desligamento).

`It would affect` é o resultado (o que aconteceria como consequência).

Este tipo de condicional é usado para falar sobre situações irreais ou hipotéticas.
