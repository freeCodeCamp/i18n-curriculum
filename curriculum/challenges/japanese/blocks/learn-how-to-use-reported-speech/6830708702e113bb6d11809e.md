# --explanation--

この文は**受動態**を使っており、行為者ではなく行為を受ける対象に焦点を当てています。例えば：

`The navigation menu was appreciated.` – これは「人々がそれを気に入った」という意味ですが、誰が気に入ったかは示していません。

**能動態**では、`The users appreciated the navigation menu.` のように言い、誰が行動をしたかをはっきり示します。

**受動態**は、行為者が不明または重要でない場合によく使われます。別の例：

`The report was completed on time.` – レポートが完了したことはわかりますが、誰が完了したかはわからないし、言う必要もありません。
