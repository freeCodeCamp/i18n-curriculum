# --explanation--

`Unusual` означає щось, що не є загальним, нормальним або очікуваним. Воно описує щось поза звичайним. Наприклад:

`It's unusual for him to be late for a meeting.` — це означає, що для людини не є нормальним або очікуваним запізнюватися. Це натякає, що така поведінка є незвичайною.

`Isn't` використовується в питальних тегах для підтвердження або перевірки інформації. Його застосовують після стверджувального речення з `is`.
