# --description--

`Le` — це займенник, який представляє "йому", "їй" або "вам".

`Le gusta` означає "йому подобається", коли йдеться про Дієго, який є чоловіком. Те саме стосується, коли йдеться про жінку. Наприклад:

- `Le gusta el fútbol.` — Йому подобається футбол (про чоловіка).

- `Le gusta el fútbol.` — Їй подобається футбол (про жінку).
