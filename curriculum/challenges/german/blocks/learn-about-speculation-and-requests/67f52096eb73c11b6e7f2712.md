## --blanks--

`physical access`

### --feedback--

Dieser zweiteilige Ausdruck bezeichnet jemanden, der physisch einen gesperrten Bereich betritt, anstatt sich aus der Ferne in ein System zu hacken.

# --explanation--

`Physical access` bedeutet, Zugang zu einem Ort zu erhalten, wie zum Beispiel einem Büro oder einem Serverraum.

Ein `physical access breach` tritt auf, wenn jemand ohne Erlaubnis einen gesicherten Bereich betritt. Zum Beispiel:

- `To avoid a physical access breach, we should require ID checks at the entrance.` – Ein Unternehmen kann unbefugten Zutritt verhindern, indem es Ausweise an der Tür kontrolliert.

- `A physical access breach can happen if someone enters the server room without permission.` – Ein physisches Sicherheitsproblem kann entstehen, wenn jemand ohne Erlaubnis in den Serverraum geht.
