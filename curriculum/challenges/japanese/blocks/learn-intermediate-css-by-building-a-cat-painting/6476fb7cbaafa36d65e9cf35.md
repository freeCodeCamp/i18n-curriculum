# --description--

`fixed` は、ユーザーがページをどこまでスクロールしても要素をページに固定できる `position` プロパティ値です。

`fixed` の位置指定がどのように機能するかを見るために、もう少しマークアップを行う必要があります。HTML内で、クラス `div` を持つ `box` 要素を作成してください。

# --hints--

`div` 要素があるはずです。

```js
assert(document.querySelectorAll('div').length === 2);
```

`div` 要素にはクラス `box` を付けてください。

```js
assert(document.querySelectorAll('div')[1]?.getAttribute('class') === 'box');
```
