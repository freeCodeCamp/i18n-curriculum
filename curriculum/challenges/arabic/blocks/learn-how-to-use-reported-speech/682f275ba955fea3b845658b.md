# --explanation--

في **الخطاب المنقول**، يُستخدم `to explain` عندما يوضح شخص سببًا أو يشرح أمرًا. على سبيل المثال:

- الخطاب المباشر: `The server went down because of a power issue.`

- الخطاب المنقول: `She explained that the server went down due to a power issue.`

الـ `system component` هو جزء أو قطعة من نظام أكبر، عادة في الحواسيب أو الشبكات. على سبيل المثال:

`The login module is a key system component for user access.` – هذا يعني أنه أحد الأجزاء الأساسية للنظام.
