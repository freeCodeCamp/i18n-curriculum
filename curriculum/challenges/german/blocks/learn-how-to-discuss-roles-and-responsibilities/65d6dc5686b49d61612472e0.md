# --description--

Der Ausdruck `sound like` wird verwendet, wenn etwas aufgrund dessen, was Sie hören, so zu sein scheint. Zum Beispiel bedeutet `It sounds like she had fun at the party`, dass es aufgrund des Gehörten so aussieht, als hätte sie die Party genossen.

Ebenso wird `look like` basierend auf dem, was Sie sehen, verwendet. Zum Beispiel bedeutet `He looks like a runner`, dass er aufgrund seines Aussehens so wirkt, als wäre er jemand, der läuft.
