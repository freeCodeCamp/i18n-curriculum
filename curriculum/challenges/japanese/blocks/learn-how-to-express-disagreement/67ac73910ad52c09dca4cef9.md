# --explanation--

`Not necessarily` は部分的な反対を丁寧に表現する方法です。`no` と直接言う代わりに、他に考慮すべき要素があるかもしれないことを示唆しています。会話では、このフレーズが議論を開かれたものにし、敬意を保つのに役立ちます。例えば：

- **サラ:** `I think upgrading the computer's processor will fix the issue.`

- **フィン:** `Not necessarily. The problem could be with the software, not the hardware.` - この人は、プロセッサーのアップグレードが問題を解決しないかもしれないと示唆しています。なぜなら、問題は古いソフトウェアのような他の原因による可能性があるからです。
