## Теорія кольору

- **Визначення теорії кольору**: Це вивчення того, як кольори взаємодіють між собою і як вони впливають на наше сприйняття. Воно охоплює відносини кольорів, колірну гармонію та психологічний вплив кольору.
- **Основні кольори**: Ці кольори, а саме жовтий, синій і червоний, є фундаментальними відтінками, з яких походять усі інші кольори.
- **Вторинні кольори**: Ці кольори утворюються шляхом змішування рівних частин двох основних кольорів. Прикладами вторинних кольорів є зелений, помаранчевий і фіолетовий.
- **Третинні кольори**: Ці кольори утворюються шляхом поєднання основного кольору з сусіднім вторинним кольором. Прикладами третинних кольорів є жовто-зелений, синьо-зелений і синьо-фіолетовий.
- **Теплі кольори**: Ці кольори, до яких належать червоні, помаранчеві та жовті, викликають відчуття комфорту, тепла та затишку.
- **Холодні кольори**: Ці кольори, до яких належать сині, зелені та фіолетові, викликають відчуття спокою, умиротворення та професіоналізму.
- **Колірне коло**: Колірне коло — це кругова діаграма, яка показує, як кольори пов’язані між собою. Це важливий інструмент для дизайнерів, оскільки допомагає їм вибирати колірні поєднання.
- **Аналогові колірні схеми**: Ці схеми створюють цілісні та заспокійливі враження. Вони містять аналогові кольори, які розташовані поруч один з одним на колірному колі.
- **Доповнюючі колірні схеми**: Ці схеми створюють високий контраст і візуальний ефект. Їхні кольори розташовані на протилежних кінцях колірного кола відносно один одного.
- **Тріадична колірна схема**: Ця схема має яскраві кольори. Вона складається з кольорів, які приблизно рівновіддалені один від одного. Якщо їх з’єднати, вони утворять рівносторонній трикутник на колірному колі.
- **Монохроматична колірна схема**: Для цієї схеми всі кольори походять від одного базового кольору шляхом регулювання його світлості, темряви та насиченості. Це викликає відчуття єдності та гармонії, одночасно створюючи контраст.

## Різні способи роботи з кольорами в CSS

- **Іменовані кольори**: Це кольори з попередньо визначеними назвами, які розпізнають браузери. Прикладами є `blue`, `darkred`, `lightgreen`.
- **Функція `rgb()`**: RGB означає червоний, зелений і синій — основні кольори світла. Ці три кольори комбінуються з різною інтенсивністю, щоб створити широкий спектр кольорів. Функція `rgb()` дозволяє визначати кольори за допомогою моделі RGB.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>RGB Color</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

- **Функція `rgba()`**: Ця функція додає четверте значення — альфа, яке контролює прозорість кольору. Якщо його не вказано, значення альфа за замовчуванням дорівнює `1`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>RGBA Background</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

- **Функція `hsl()`**: HSL означає відтінок, насиченість і світлість — три ключові компоненти, що визначають колір. Сучасний синтаксис `hsl()` також підтримує необов’язкове значення альфа через роздільник `/`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>HSL Color</p>
<p class="alpha">HSL Color with Alpha</p>
```

```css
p {
  color: hsl(120 100% 50%);
}

.alpha {
  color: hsl(120 100% 50% / 0.8);
}
```

:::

- **Функція `hsla()`**: Функція `hsla()` — це застарілий спосіб додати прозорість до кольорів HSL. Сучасний CSS віддає перевагу `hsl()` з роздільником `/` для значень альфа.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>HSLA Background</div>
```


```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

- **Шістнадцятковий код**: Шістнадцятковий код (скорочено hex) використовується для представлення кольорів у моделі RGB. "Hex" означає систему числення з основою 16, яка використовує цифри `0`–`9` і літери `A`–`F`. Шістнадцяткові коди можуть бути довжиною 6 символів, наприклад, `#F498A2`, або скороченою 3-символьною формою, яка працює, коли кожна пара у повній формі складається з однакових символів двічі. Наприклад, `#2FB` еквівалентний `#22FFBB`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1 class="hex-text">Hex Text</h1>
<p class="hex-bg">Hex Background</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}

p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

## Властивість `box-shadow`

- **Визначення**: властивість `box-shadow` застосовує одну або кілька тіней навколо елемента.
- **Значення зміщення**: ви повинні вказати горизонтальні (`offset-x`) та вертикальні (`offset-y`) значення. Додатне `offset-x` зміщує тінь вправо, тоді як від’ємні значення зміщують її вліво. Додатні `offset-y` зміщує тінь вниз, а від’ємні — вгору. Якщо значення `0`, не потрібно вказувати одиниці вимірювання.
- **Радіус розмиття**: це необов’язкове значення контролює, наскільки розмитою виглядає тінь. Якщо його не вказано, за замовчуванням використовується `0`, що створює чіткі краї. Чим більше значення, тим м’якша тінь.
- **Радіус розповсюдження**: це необов’язкове значення контролює, наскільки тінь розширюється або звужується. Якщо його не вказано, за замовчуванням використовується `0`.
- **Колір тіні**: ви можете вказати колір за допомогою іменованих кольорів, шістнадцяткових значень, `rgb()`, `rgba()`, `hsl()` або `hsla()`.
- **Ключове слово `inset`**: додавання ключового слова `inset` розміщує тінь всередині елемента замість поза межами.
- **Застосування кількох тіней блоку**: ви можете застосувати кілька тіней, розділяючи їх комами. Тіні накладаються шарами від переднього плану до заднього.

### Синтаксис

```css
box-shadow: offset-x offset-y blur-radius spread-radius color;
```

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="shadow-box">Shadow Color Example</div>
```

```css
.shadow-box {
  width: 200px;
  padding: 20px;
  background-color: lightblue;
  box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}
```

:::

## Лінійні та радіальні градієнти

- **Лінійні градієнти**: Ці градієнти створюють поступове змішування кольорів уздовж прямої лінії. Ви можете контролювати напрямок цієї лінії за допомогою ключових слів, таких як `to top`, `to right`, `to bottom right`, або кутів, як `45deg`. Ви можете використовувати будь-який дійсний CSS колір і стільки кольорових зупинок, скільки забажаєте.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient">Linear Gradient</div>
```

```css
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

:::

- **Радіальні градієнти**: Ці градієнти створюють кругові або еліптичні градієнти, що розходяться від центральної точки.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient">Radial Gradient</div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::
