# --explanation--

`Reasonable` significa justo, prático ou lógico com base no contexto. É usado para perguntar se algo faz sentido ou é aceitável. Por exemplo:  

- `The timeline for completing the project seems reasonable.` - Isso significa que o cronograma é justo e alcançável.  

- `It's reasonable to expect a response within 24 hours.` - Isso significa que é prático e justo esperar uma resposta dentro desse prazo.

Usar `reasonable` em uma conversa mostra uma maneira educada e profissional **de confirmar o acordo** ou verificar se uma sugestão é aceitável.
