# --description--

تستخدم جداول HTML عنصر `caption` لوصف محتوى الجدول. يجب أن يكون عنصر `caption` دائمًا هو الطفل الأول لـ `table`، لكن يمكن وضعه باستخدام خاصية CSS المسماة `caption-side`.

أضف عنصر `caption` إلى أول `table` لديك، وأعطه النص `Assets`.

# --hints--

يجب أن يحتوي أول عنصر `table` لديك على عنصر `caption`.

```js
assert(document.querySelector('table')?.children?.[0]?.localName === 'caption');
```

يجب أن يحتوي عنصر `caption` الخاص بك على النص `Assets`.

```js
assert(document.querySelector('caption')?.textContent === 'Assets');
```
