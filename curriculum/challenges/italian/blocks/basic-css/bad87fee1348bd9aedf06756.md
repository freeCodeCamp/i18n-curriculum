# --description--

Abbiamo dimostrato che le dichiarazioni di id sovrascrivono quelle di classe, indipendentemente da dove sono dichiarate nel CSS del tuo elemento `style`.

Ci sono altri modi per sovrascrivere il CSS. Ti ricordi gli stili in linea?

# --instructions--

Usa uno stile in linea per cercare di rendere bianco il nostro elemento `h1`. Ricorda, gli stili in linea si presentano così:

```html
<h1 style="color: green;">
```

Lascia le classi `blue-text` e `pink-text` sul tuo elemento `h1`.

# --hints--

Il tuo elemento `h1` dovrebbe avere la classe `pink-text`.

```js
assert.isTrue(document.querySelector('h1').classList.contains('pink-text'));
```

Il tuo elemento `h1` dovrebbe avere la classe `blue-text`.

```js
assert.isTrue(document.querySelector('h1').classList.contains('blue-text'));
```

Il tuo elemento `h1` dovrebbe avere l'id `orange-text`.

```js
assert.strictEqual(document.querySelector('h1').getAttribute('id'), 'orange-text');
```

Il tuo elemento `h1` dovrebbe avere uno stile in linea.

```js
assert.exists(document.querySelector('h1[style]'));
```

Il tuo elemento `h1` dovrebbe essere bianco.

```js
const h1Element = document.querySelector('h1');
const color = window.getComputedStyle(h1Element)['color']; 
assert.strictEqual(color, 'rgb(255, 255, 255)');
```
