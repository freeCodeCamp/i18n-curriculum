# --description--

`Probably` は、何かが起こりそうだと思うけれど、100％確信が持てないときに使います。

例えば、`The code will probably run without errors after the update` と言うことができます。これはそう信じているけれど、まだ少し問題が起こる可能性があるという意味です。
