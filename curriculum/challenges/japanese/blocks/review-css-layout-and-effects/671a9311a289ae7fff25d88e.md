## CSSのオーバーフロープロパティ

- **定義**: オーバーフローとは、要素の中身がそのコンテナのサイズを超えてはみ出す場合の処理方法を指します。オーバーフローは2次元的な概念です。
- **`overflow-x`**: x軸は水平方向のオーバーフローを決定します。
- **`overflow-y`**: y軸は垂直方向のオーバーフローを決定します。
- **`overflow`**: `overflow-x` と `overflow-y` の省略形プロパティです。1つの値を指定すると両方のオーバーフローに適用されます。2つの値を指定すると、`overflow-x` が最初の値を使い、`overflow-y` が2番目の値を使います。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="overflow-box">
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
</div>
```

```css
.overflow-box {
  width: 140px;
  height: 70px;
  border: 2px solid black;
  overflow-x: auto;
  overflow-y: auto;
  white-space:nowrap;
}
```

## CSSのトランスフォームプロパティ

- **定義**: このプロパティは、要素に対して回転、拡大縮小、傾斜、移動（2Dまたは3D空間で）などのさまざまな変形を適用できます。
- **`translate()` 関数**: 要素を現在の位置から移動させるために使います。
- **`scale()` 関数**: 要素のサイズを変更できます。
- **`rotate()` 関数**: 要素を回転させることができます。
- **`skew()` 関数**: 要素を傾斜させることができます。
- **変形とアクセシビリティ**: transformを使ってコンテンツを隠したり表示したりする場合、スクリーンリーダーやキーボード操作でコンテンツにアクセスできることを確認してください。隠すコンテンツは、単に画面外に視覚的に移動させるのではなく、`display: none` や `visibility: hidden` のように本当に隠す方法を使うべきです。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="transform-box translate">Translate</div>
  <div class="transform-box scale">Scale</div>
  <div class="transform-box rotate">Rotate</div>
  <div class="transform-box skew">Skew</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: column;
  align-items: center;
  gap: 16px;
  flex-wrap: wrap;
}

.transform-box {
  width: 100px;
  height: 100px;
  background: coral;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

/* translate() */
.translate {
  transform: translateX(20px);
}

/* scale() */
.scale {
  transform: scale(1.2);
}

/* rotate() */
.rotate {
  transform: rotate(15deg);
}

/* skew() */
.skew {
  transform: skew(10deg, 5deg);
}
```

## ボックスモデル

- **定義**: CSSのボックスモデルでは、すべての要素はボックスに囲まれています。このボックスは4つの要素で構成されます：コンテンツエリア、`padding`、`border`、`margin`。
- **コンテンツエリア**: コンテンツエリアはボックスの最も内側の部分で、テキストや画像など要素の実際の内容が入るスペースです。
- **`padding`**: パディングはコンテンツエリアのすぐ外側のエリアで、コンテンツエリアと要素のボーダーの間のスペースです。
- **`border`**: ボーダーはCSSボックスモデルにおける要素の外枠や輪郭です。要素の視覚的な境界線です。
- **`margin`**: マージンは要素のボーダーの外側のスペースで、要素と周囲の他の要素との距離を決めます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="box-model">Content</div>
```

```css
.box-model {
  width: 120px;
  background: lightblue;
  padding: 16px;
  border: 4px solid teal;
  margin: 20px;
  text-align: center;
}
```

:::

## マージンの重なり

- **定義**: この挙動は隣接する要素の垂直方向のマージンが重なり合い、2つのうち大きい方のマージン1つ分だけになる現象です。この挙動は垂直方向のマージン（上下）にのみ適用され、水平方向のマージン（左右）には適用されません。

## `content-box` と `border-box` のプロパティ値

- **`box-sizing` プロパティ**: HTML要素の最終的な`width` と `height` の計算方法を決定するために使います。
- **`content-box` 値**: `content-box` モデルでは、要素に設定した`width` と `height` がコンテンツエリアの寸法を決めますが、`padding`、`border`、`margin` は含みません。
- **`border-box` 値**: `border-box` では、要素の`width` と `height` にコンテンツエリア、`padding`、`border` が含まれますが、`margin` は含まれません。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="sizing-box content-box">Content-box</div>
<div class="sizing-box border-box">Border-box</div>
```

```css
.sizing-box {
  width: 150px;
  padding: 10px;
  border: 4px solid teal;
  margin-bottom: 10px;
  background: peachpuff;
  text-align: center;
}

.content-box {
  box-sizing: content-box;  
}

.border-box {
  box-sizing: border-box;
  background: lightgreen;
}
```

## CSSリセット

- **定義**: CSSリセットは、ウェブブラウザがHTML要素に適用するデフォルトのスタイルの一部または全部を取り除くスタイルシートです。サードパーティのCSSリセットには`sanitize.css` や `normalize.css` があります。

## CSSフィルタープロパティ

- **定義**: このプロパティは、ぼかし、色調の変更、コントラスト調整などさまざまなエフェクトを作成するために使えます。
- **`blur()` 関数**: 要素にガウスぼかしを適用します。量はピクセル単位で、ぼかしの半径を表します。
- **`brightness()` 関数**: 要素の明るさを調整します。0%は完全に黒くなり、100%を超える値は明るさを増加させます。
- **`contrast()` 関数**: 要素のコントラストを調整します。0%は完全にグレーにし、100%は通常の見た目、100%を超える値はコントラストを強めます。
- **`grayscale()` 関数**: 要素をグレースケールに変換します。量はパーセンテージで指定し、100%は完全なグレースケール、0%は変更なしです。
- **`sepia()` 関数**: 要素にセピア調を適用します。グレースケールと同様にパーセンテージ値を使います。
- **`hue-rotate()` 関数**: 要素に色相回転を適用します。値は度数で、色相環を回転させる角度を表します。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <img class="filter blur" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Blur" />
  <img class="filter brightness" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Brightness" />
  <img class="filter contrast" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Contrast" />
  <img class="filter grayscale" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Grayscale" />
  <img class="filter sepia" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Sepia" />
  <img class="filter hue" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Hue Rotate" />
</div>
```

```css
.container {
  display: flex;
  gap: 12px;
  flex-wrap: wrap;
}

.filter {
  width: 120px;
}

/* blur() */
.blur {
  filter: blur(3px);
}

/* brightness() */
.brightness {
  filter: brightness(130%);
}

/* contrast() */
.contrast {
  filter: contrast(150%);
}

/* grayscale() */
.grayscale {
  filter: grayscale(100%);
}

/* sepia() */
.sepia {
  filter: sepia(100%);
}

/* hue-rotate() */
.hue {
  filter: hue-rotate(90deg);
}
```
