# --description--

```css
/* rule 1 */
.class.second-class {
  font-size: 12px;
}

/* rule 2 */
.class .second-class {
  font-size: 24px;
}
```

這裡規則 1 和規則 2 具有相同的特異性。規則 1 使用串鏈選擇器（無空格），規則 2 使用後代組合符號（空格）。但兩個規則都有兩個類別，且組合符號本身不會增加特異性。

```css
/* rule 1 */
.class.second-class {
  font-size: 12px;
}
/* rule 2 */
.class > .second-class {
  font-size: 24px;
}
```

這個範例顯示相同的結果。即使規則 2 使用子代選擇器（`>`），這並不改變特異性值。兩個規則仍然都有兩個類別，因此它們具有相同的特異性值。

*注意：* 並非所有內容都會增加特異性

在比較選擇器時，你可能會遇到通用選擇器（`*`）以及組合符（`+`、`~`、`>` 和空白）的特殊符號。這些符號本身並不增加任何特異性。
