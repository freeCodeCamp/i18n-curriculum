# --description--

Quando você quer `make sure` que algo funciona, você tenta fazer isso com cuidado especial para confirmar o resultado no final.Por exemplo:

- `We have to make sure the code is running.` - Isso significa que você terá um cuidado especial com o código para confirmar que ele roda bem.

- `Please make sure that you order the correct color.` - Aqui, está sendo solicitado que você confirme a cor desejada antes de fazer o pedido.
