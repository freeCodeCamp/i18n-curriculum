# --description--

`Uncertain` 意味著對某事不確定，就像對決定感到不確定一樣。相反的是 `certain`，表示確定。 例如，`I am certain this is the right way` 意味著對方向感到確定。

`Achieve` 意味著經過努力成功達成目標，例如 `achieve a good result in a test`。在程式碼中，`achieve` 通常指程式設計師想要用他們的程式碼完成的目標。
