# --description--

`constant` katika programu ni sawa na variable, lakini mara tu inapopewa thamani, thamani hiyo haiwezi kubadilishwa. `Constants` hutumika wakati una thamani ambazo unajua hazitahitaji kubadilishwa katika programu nzima. Mara nyingi huandikwa kwa herufi kubwa ili kuzitofautisha na variable.

Kwa mfano, `const PI = 3.14;` Katika lugha nyingi za programu, `const` hutumika kutangaza constant. Hapa, `PI` ni `constant` yenye thamani imepewa `3.14`.
