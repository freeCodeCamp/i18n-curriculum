# --description--

점진적 향상은 모든 사용자가 브라우저나 기기에 상관없이 애플리케이션의 핵심 콘텐츠와 기능에 접근할 수 있도록 보장하는 디자인 접근법입니다.

모든 사용자에게 작동하는 기본 경험을 제공하는 데 초점을 맞추면서, 더 발전된 브라우저나 더 나은 인터넷 연결을 가진 사용자에게는 추가 기능과 향상을 제공합니다.

점진적 향상 접근법은 다음과 같은 핵심 원칙을 따릅니다:

- 모든 핵심 콘텐츠와 기본 기능은 모든 브라우저에서 접근 가능해야 합니다
- 모든 고급 레이아웃은 외부 CSS 스타일시트를 통해 제공되어야 합니다
- 모든 고급 기능은 외부 JavaScript 파일을 통해 제공되어야 합니다
- 사용자의 브라우저 환경 설정을 존중해야 합니다

점진적 향상 접근법을 사용하면 지원하지 않는 환경에서 핵심 콘텐츠와 기능이 차단되지 않기 때문에 애플리케이션의 접근성이 향상됩니다.

속도 측면에서도 점진적 향상은 애플리케이션 성능 향상에 도움이 될 수 있습니다.

인터넷 연결 속도가 느린 사용자도 브라우저가 필요한 리소스를 먼저 다운로드하기 때문에 콘텐츠에 접근할 수 있습니다.

SEO 측면에서도 점진적 향상은 애플리케이션의 가시성 향상에 도움이 됩니다.

검색 엔진이 애플리케이션의 콘텐츠를 크롤링할 수 있는데, 이는 핵심 콘텐츠가 초기 HTML 응답에 포함되어 있기 때문입니다.

일부에서는 이 접근법이 기능에 JavaScript를 많이 사용하는 애플리케이션에는 항상 현실적이지 않다고 비판하지만, 애플리케이션을 구축할 때 따라야 할 좋은 관행입니다.
