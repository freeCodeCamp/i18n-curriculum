# --description--

能夠檢查和偵錯你的 HTML 和 CSS 對前端開發至關重要。本課程將帶領我們了解 Chrome Dev Tools，它讓你能查看關於你的元素和 CSS 規則的詳細資訊，並協助你在程式碼中找到並修正問題。

要開啟檢查器，你可以在網頁的任何元素上按右鍵並點擊「Inspect」或按下 F12。現在就這麼做，查看此頁面使用的 HTML 和 CSS。

不要被你現在看到的所有工具淹沒！在本課程中，我們想專注於 Elements 和 Styles 面板。
