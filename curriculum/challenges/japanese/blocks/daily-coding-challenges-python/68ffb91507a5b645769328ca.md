# --description--

整数 (`n`) が与えられたとき、`1` から `n` までの整数の配列を返してください（両端を含む）。ただし、次の条件に当てはまる数字はそれぞれ置き換えます：

- 3の倍数は `"Fizz"` に置き換えます。
- 5の倍数は `"Buzz"` に置き換えます。
- 3と5の両方の倍数は `"FizzBuzz"` に置き換えます。

def fizz_buzz(n):
    result = []
    for i in range(1, n+1):
        if i % 15 == 0:
            result.append("FizzBuzz")
        elif i % 3 == 0:
            result.append("Fizz")
        elif i % 5 == 0:
            result.append("Buzz")
        else:
            result.append(i)
    return result
