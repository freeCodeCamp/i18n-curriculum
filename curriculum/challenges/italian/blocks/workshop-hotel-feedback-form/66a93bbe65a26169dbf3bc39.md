# --description--

Il modulo di feedback del tuo hotel dovrebbe anche raccogliere un indirizzo email dall'utente.

Inizia aggiungendo un nuovo elemento `label` con il testo `Email address (required):` al modulo. Il tuo elemento `label` dovrebbe avere un attributo `for` impostato sul valore `"email"`.

```js
assert.lengthOf(document.querySelectorAll('fieldset label'), 2);
```

Il tuo elemento `label` dovrebbe avere il testo `Email address (required):`.

```js
assert.strictEqual(document.querySelector('fieldset label:nth-of-type(2)')?.innerText.trim(), 'Email address (required):');
```

