## 使用数字数据类型

- **定义**：JavaScript 的 `Number` 类型包括整数、浮点数、`Infinity` 和 `NaN`。浮点数是带小数点的数字。正 `Infinity` 是比任何其他数字都大的数字，而 `-Infinity` 是比任何其他数字都小的数字。`NaN`（`Not a Number`）表示无效的数字值，如字符串 `"Jessica"`。

## 常见算术操作

- **加法操作符**：此操作符（`+`）用于计算两个或多个数字的和。
- **减法操作符**：此操作符（`-`）用于计算两个数字的差。
- **乘法操作符**：此操作符（`*`）用于计算两个或多个数字的乘积。
- **除法操作符**：此操作符（`/`）用于计算两个数字的商。
- **除以零**：如果你尝试除以零，JavaScript 会返回 `Infinity`。
- **取余操作符**：此操作符（`%`）返回除法的余数。
- **指数操作符**：此操作符（`**`）将一个数字提升到另一个数字的幂。

## 数字和字串的计算

- **解释**：当你使用 `+` 操作符连接数字和字串时，JavaScript 会将数字强制转换为字串并连接两个值。当你使用 `-`、`*` 或 `/` 操作符连接字串和数字时，JavaScript 会将字串强制转换为数字，结果是一个数字。对于 `null` 和 `undefined`，JavaScript 在数学运算中将 `null` 视为 0，将 `undefined` 视为 `NaN`。

:::interactive_editor

```js
const result = 5 + '10';

console.log(result); // "510"
console.log(typeof result); // string

const subtractionResult = '10' - 5;
console.log(subtractionResult); // 5
console.log(typeof subtractionResult); // number

const multiplicationResult = '10' * 2;
console.log(multiplicationResult); // 20
console.log(typeof multiplicationResult); // number

const divisionResult = '20' / 2;
console.log(divisionResult); // 10
console.log(typeof divisionResult); // number

const result1 = null + 5;
console.log(result1); // 5
console.log(typeof result1); // number

const result2 = undefined + 5;
console.log(result2); // NaN
console.log(typeof result2); // number
```

:::

## 操作符优先级

- **定义**：运算符优先级决定了表达式中运算的求值顺序。 优先级较高的运算符先于优先级较低的运算符进行运算。 括号内的值将优先计算，乘法/除法的优先级高于加法/减法。 如果运算符具有相同的优先级，那么 JavaScript 将使用结合性。

:::interactive_editor

```js
const result = (2 + 3) * 4;

console.log(result); // 20

const result2 = 10 - 2 + 3;

console.log(result2); // 11

const result3 = 2 ** 3 ** 2;

console.log(result3); // 512
```

:::

- **定义**： 结合性告诉我们存在多个相同类型的运算符时表达式的求值方向。 它定义了表达式应该从左到右（`left-associative`）还是从右到左（`right-associative`）进行求值。 例如，指数运算符也是从右向左结合的：

:::interactive_editor

```js
const result4 = 5 ** 4 ** 1; 

console.log(result4); // 625
```

:::

## 自增和自减操作符

- **自增操作符**：此操作符用于将值增加 1。前缀表示法 `++num` 先增加变量值，然后返回新值。后缀表示法 `num++` 先返回变量当前值，然后增加它。

:::interactive_editor

```js
let x = 5;

console.log(++x); // 6
console.log(x); // 6


let y = 5;

console.log(y++); // 5
console.log(y); // 6
```

:::

- **自减操作符**：此操作符用于将值减少 1。前缀和后缀表示法与自增操作符的工作方式相同。

:::interactive_editor

```js
let num = 5;

console.log(--num); // 4
console.log(num--); // 4
console.log(num); // 3
```

:::

## 复合赋值操作符

- **加法赋值（`+=`）操作符**：此操作符对值执行加法，并将结果赋给变量。
- **减法赋值（`-=`）操作符**：此操作符对值执行减法，并将结果赋给变量。
- **乘法赋值（`*=`）操作符**：此操作符对值执行乘法，并将结果赋给变量。
- **除法赋值（`/=`）操作符**：此操作符对值执行除法，并将结果赋给变量。
- **取余赋值（`%=`）操作符**：此操作符将变量除以指定数字，并将余数赋给变量。
- **指数赋值（`**=`）操作符**：此操作符将变量提升到指定数字的幂，并重新赋值给变量。

## 布尔值和相等性

- **布尔定义**：布尔是一种只能有两个值的数据类型：`true` 或 `false`。
- **相等（`==`）操作符**：此操作符在检查值是否相等之前使用类型强制转换。

:::interactive_editor

```js
console.log(5 == '5'); // true
```

:::

- **严格相等（`===`）操作符**：此操作符不执行类型强制转换，检查类型和值是否都相等。

:::interactive_editor

```js
console.log(5 === '5'); // false
```

:::

- **不等（`!=`）运算符**：该操作符在检查值是否不相等之前使用类型强制。
- **严格不等（`!==`）运算符**：该运算符不执行类型强制，并检查类型和值是否不相等。

## 比较操作符

- **大于（`>`）操作符**：此操作符检查左侧值是否大于右侧值。
- **大于等于（`>=`）操作符**：此操作符检查左侧值是否大于或等于右侧值。
- **小于（`<`）操作符**：此操作符检查左侧值是否小于右侧值。
- **小于等于（`<=`）操作符**：此操作符检查左侧值是否小于或等于右侧值。

## 一元操作符

- **一元加操作符**：此操作符将其操作对象转换为数字。如果操作对象已经是数字，则保持不变。

:::interactive_editor

```js
const str = '42';
const num = +str;

console.log(num); // 42
console.log(typeof num); // number
```

:::

- **一元取反（`-`）操作符**：此操作符对操作对象取反。

:::interactive_editor

```js
const num = 4;
console.log(-num); // -4
```

:::

- **逻辑非（`!`）操作符**：此操作符翻转操作对象的布尔值。如果操作对象是 `true`，则变为 `false`；如果是 `false`，则变为 `true`。

## 位操作符

- **按位与（`&`）操作符**：此操作符在两个操作数的对应位都为 1 时返回 1。
- **按位与赋值（`&=`）操作符**：此操作符对指定数字执行 `bitwise AND` 操作，并将结果重新赋值给变量。
- **按位或（`|`）操作符**：此操作符在两个操作数的对应位任一为 1 时返回 1。
- **按位或赋值（`|=`）操作符**：此操作符对指定数字执行 `bitwise OR` 操作，并将结果重新赋值给变量。
- **按位异或（`^`）操作符**：此操作符在两个操作数的对应位只有一个为 1 时返回 1。
- **按位非（`~`）操作符**：此操作符对数字的二进制表示取反。
- **左移（`<<`）操作符**：此操作符将所有位向左移动指定位置数。
- **右移（`>>`）操作符**：此操作符将所有位向右移动。

## 条件语句、真值、假值和三元操作符

- **`if/else if/else`**：`if` 语句接受一个条件，如果该条件为 `truthy`，则执行一段代码块。如果条件为 `false`，则执行 `else if` 块。如果这些条件都不为 `true`，则执行 `else` 子句。`Truthy` 值是在布尔上下文（如 `if` 语句）中计算结果为 `true` 的任何值。`Falsy` 值是在布尔上下文中计算结果为 `false` 的值。

:::interactive_editor

```js
const score = 87;

if (score >= 90) {
 console.log('You got an A'); 
} else if (score >= 80) {
 console.log('You got a B'); // You got a B
} else if (score >= 70) {
 console.log('You got a C');
} else {
 console.log('You failed! You need to study more!');
}
```

:::

- **三元操作符**：此操作符常用作编写 `if else` 语句的简写。

:::interactive_editor

```js
const temperature = 30;
const weather = temperature > 25 ? 'sunny' : 'cool';

console.log(`It's a ${weather} day!`); // It's a sunny day!
```

:::

## 二元逻辑操作符

- **Logical AND (`&&`) Operator**: This operator checks if both operands are `true`. 如果两个值都是 `true` ，那么它将返回第二个值。 如果任一操作数为 `falsy`，则会返回 `falsy` 值。 如果两个操作数都是 `falsy`，则返回第一个 `falsy` 值。

:::interactive_editor

```js
const result = true && 'hello';

console.log(result); // hello
```

:::

- **逻辑或 (`||`) 操作符**：此操作符检查是否至少有一个操作对象为真值。
- **空值合并（`??`）运算符**：只有当第一个值为 `null` 或 `undefined` 时，该运算符才会返回一个值。

:::interactive_editor

```js
const userSettings = {
 theme: null,
 volume: 0,
 notifications: false,
};

let theme = userSettings.theme ?? 'light';
console.log(theme); // light
```

:::

## `Math` 对象

- **`Math.random()` 方法**：此方法生成一个介于 0（含）和 1（不含）之间的随机浮点数。这意味着输出可能是 0，但永远不会达到 1。
- **`Math.max()` 方法**：此方法接受一组数字并返回最大值。
- **`Math.min()` 方法**：此方法接受一组数字并返回最小值。
- **`Math.ceil()` 方法**：此方法将值向上舍入到最接近的整数。
- **`Math.floor()` 方法**：此方法将值向下舍入到最接近的整数。
- **`Math.round()` 方法**：此方法将值舍入到最接近的整数。

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

- **`Math.trunc()` 方法**：此方法去除数字的小数部分，仅返回整数部分，不进行四舍五入。
- **`Math.sqrt()` 方法**：此方法返回数字的平方根。
- **`Math.cbrt()` 方法**：此方法返回数字的立方根。
- **`Math.abs()` 方法**：此方法返回数字的绝对值。
- **`Math.pow()` 方法**：此方法接受两个数字，将第一个数字提升到第二个数字的幂。

## 常用数字方法

- **`isNaN()`**：`NaN` 代表“非数字”（Not-a-Number）。它是一个特殊值，表示无法表示或未定义的数值结果。`isNaN()` 函数属性用于判断一个值是否是 `NaN`。`Number.isNaN()` 提供了一种更可靠的方式来检查 `NaN` 值，特别是在类型强制转换可能导致全局 `isNaN()` 函数出现意外结果的情况下。

:::interactive_editor

```js
console.log(isNaN(NaN));       // true
console.log(isNaN(undefined)); // true
console.log(isNaN({}));        // true

console.log(isNaN(true));      // false
console.log(isNaN(null));      // false
console.log(isNaN(37));        // false


console.log(Number.isNaN(NaN));        // true
console.log(Number.isNaN(Number.NaN)); // true
console.log(Number.isNaN(0 / 0));      // true

console.log(Number.isNaN("NaN"));      // false
console.log(Number.isNaN(undefined));  // false
```

:::

- **`parseFloat()` 方法**：该方法解析字符串参数并返回一个浮点数。它设计用于从字符串开头提取数字，即使字符串后面包含非数字字符。
- **`parseInt()` 方法**：该方法解析字符串参数并返回一个整数。`parseInt()` 在遇到第一个非数字字符时停止解析。对于浮点数，它只返回整数部分。如果在字符串开头找不到有效整数，则返回 `NaN`。
- **`toFixed()` 方法**：该方法在数字上调用，接受一个可选参数，表示小数点后显示的位数。它返回指定小数位数的数字字符串表示。
