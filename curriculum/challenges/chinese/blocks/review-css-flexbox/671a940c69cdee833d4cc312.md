## CSS Flexbox 和 Flex 模型简介

- **定义**：CSS flexbox 是一种一维版型模型，允许你在容器内按行和列排列元素。
- **Flex Model**：该模型定义了弹性项在弹性容器内的排列方式。每个弹性容器都有两个轴：主轴和交叉轴。

## `flex-direction` 属性

- **定义**：此属性设置主轴的方向。`flex-direction` 的默认值是 `row`，它将所有的弹性项放置在同一行，方向为浏览器默认语言的方向（从左到右或从右到左）。
- **`flex-direction: row-reverse;`**：这会反转行中的项。
- **`flex-direction: column;`**：这将使弹性项垂直排列，而不是水平排列。
- **`flex-direction: column-reverse;`**：这会垂直反转弹性项的顺序。

:::interactive_editor

```html
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: row;
  gap: 10px;
}

.box {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}
```

:::

## `flex-wrap` 属性

- **定义**：此属性决定了弹性项如何在弹性容器内换行以适应可用空间。`flex-wrap` 可以取三个可能的值：`nowrap`、`wrap` 和 `wrap-reverse`。
- **`flex-wrap: nowrap;`**：这是默认值。弹性项不会换行，即使它们的宽度超过容器的宽度。
- **`flex-wrap: wrap;`**：当项超出其容器宽度时，此属性将自动换行。
- **`flex-wrap: wrap-reverse;`**：此属性将以相反顺序换行 flex 项。
- **`flex-flow` 属性**：此属性是 `flex-direction` 和 `flex-wrap` 的简写属性。

:::interactive_editor

```html
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
  <div class="box">4</div>
  <div class="box">5</div>
</div>
```

```css
.container {
  display: flex;
  flex-wrap: wrap;
  width: 150px;
  background: #f0f0f0;
}

.box {
  width: 60px;
  padding: 10px;
  margin: 5px;
  background: skyblue;
  text-align: center;
}
```

:::

## `justify-content` 属性

- **定义**：此属性沿着弹性容器的主轴对齐子元素。
- **`justify-content: flex-start;`**：在这种分支中，弹性项将对齐到主轴的起点。这可以是水平或垂直方向。
- **`justify-content: flex-end;`**：在这种情况下，弹性项沿主轴的末端对齐，水平或垂直方向。
- **`justify-content: center;`**：这会将弹性项沿主轴居中。
- **`justify-content: space-between;`**：这将沿主轴均匀分布元素。
- **`justify-content: space-around;`**：这将在主轴内均匀分布弹性项，在第一个项之前和最后一个项之后添加空白。
- **`justify-content: space-evenly;`**：这将沿主轴均匀分布这些项。

:::interactive_editor

```html
<div class="container">
  <div class="box">A</div>
  <div class="box">B</div>
  <div class="box">C</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
  background: #eee;
}

.box {
  padding: 20px;
  background: salmon;
}
```

:::
  
## `align-items` 属性

- **定义**：此属性用于沿交叉轴分布项。请记住，交叉轴垂直于主轴。
- **`align-items: center;`**：用于沿交叉轴居中这些项。
- **`align-items: flex-start;`**：这将项对齐到交叉轴的起点。
- **`align-items: stretch;`**：用于沿交叉轴拉伸弹性项。

:::interactive_editor

```html
<div class="container">
  <div class="box tall">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  align-items: center;
  height: 150px;
  background: #ddd;
}

.box {
  background: lightgreen;
  padding: 10px;
}

.tall {
  height: 100px;
}
```
