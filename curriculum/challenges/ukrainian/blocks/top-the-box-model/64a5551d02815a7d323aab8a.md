# --description--

У панелі елементів ви можете побачити всю HTML-структуру вашої сторінки. Ви можете натиснути на будь-який елемент у цій панелі, щоб вибрати саме цей елемент. Або ж можна натиснути на синій підсвічений значок зліва, показаний нижче, і навести курсор на будь-який елемент на сторінці.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/the-box-model/inspector-icon.png" alt="A web development environment with developer tools open, highlighting a blue inspector icon in the top-left for selecting webpage elements to view HTML structure and styles.">

Коли елемент вибрано, вкладка стилів покаже всі застосовані стилі, а також ті, які були перекриті (позначені закресленням тексту). Наприклад, якщо ви за допомогою інспектора натиснете на заголовок “Your Career in Web Development Starts Here” на <a href="https://www.theodinproject.com/" target="_blank">головній сторінці The Odin Project</a>, праворуч ви побачите всі стилі, які зараз впливають на цей елемент, як показано нижче:
