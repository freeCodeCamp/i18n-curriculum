# --interactive--

При стилізації вебсайтів одним із найуніверсальніших і найінтуїтивніших способів роботи з кольором є використання моделі кольору HSL. HSL означає відтінок, насиченість і світлість — три ключові компоненти, які визначають колір.

Модель кольору HSL представляє кольори так, як їх сприймає людина. У цьому уроці ми розглянемо, що таке модель кольору HSL і як у CSS використовується функція `hsl()`.

Відтінок — це тип кольору, представлений як кут на колірному колі, який варіюється від `0` до `360` градусів. Наприклад, `0` градусів відповідає червоному, `120` градусів — зеленому, а `240` градусів — синьому. Змінюючи відтінок, ви переходите між різними кольорами спектра.

Насиченість означає інтенсивність або чистоту кольору. Вона вимірюється у відсотках від `0%` (повністю десатурований, сірий колір) до `100%` (найяскравіша форма цього кольору). Колір із `100%` насиченістю є повністю яскравим, а колір із `0%` насиченістю виглядає як відтінок сірого.

Світлість визначає, наскільки світлим або темним є колір, також вимірюється у відсотках. Значення світлості `0%` дасть чорний, `50%` — нормальний тон відтінку, а `100%` — білий.

У CSS функція `hsl()` використовується для визначення кольорів за моделлю HSL. Ось базовий синтаксис:

```css
element {
  color: hsl(hue, saturation, lightness);
}
```

Розглянемо це на прикладі:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
body {
  background-color: hsla(0, 0%, 1%, 1.00);
}

p {
  color: hsl(120, 100%, 50%);
}
```

:::

У цьому випадку відтінок становить `120` градусів, що відповідає зеленому. Насиченість — `100%`, тому зелений буде повністю яскравим. Світлість — `50%`, тобто нормальний тон — не надто темний і не надто світлий. В результаті колір тексту абзацу буде яскраво-зеленим.

Однією з головних переваг моделі кольору HSL є її інтуїтивність. Вона дозволяє легко регулювати яскравість або світлість кольору, змінюючи значення насиченості та світлості, не змінюючи основний колір (відтінок).

Наприклад, якщо ви хочете створити різні відтінки або тони одного й того ж кольору, просто змініть значення світлості.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="light">This is a light blue div.</div>
<div class="dark">This is a dark blue div.</div>
```

```css
div.light {
  background-color: hsl(240, 100%, 80%);
}

div.dark {
  background-color: hsl(240, 100%, 20%);
  color: hsl(0, 0%, 100%);
}
```

:::

Тут обидва `div` елементи використовують однаковий відтінок (`240` градусів, що є синім), але один має світлість `80%` (світліший відтінок синього), а інший — світлість `20%` (темніший відтінок синього).

Як модель RGB має функцію `rgba()` для прозорості, так і модель HSL має функцію `hsla()`. Четвертий параметр цієї функції відповідає за альфа-значення, яке контролює прозорість кольору. Ось базовий синтаксис:

```css
element {
  background-color: hsla(hue, saturation, lightness, alpha);
}
```

Розглянемо приклад:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

Цей код задасть `div` напівпрозорий червоний фон, де відтінок встановлено на `0` градуси (червоний), насиченість — `100%`, світлість — `50%`, а альфа — `0.5` (50% прозорості). Альфу також можна вказати у відсотках — наприклад, `hsla(0, 100%, 50%, 50%)` еквівалентне `hsla(0, 100%, 50%, 0.5)`.

Модель кольору HSL особливо корисна, коли потрібно створювати кольорові схеми та легко регулювати відтінки або тони.

Оскільки HSL розділяє відтінок від насиченості та світлості, ви можете зберігати послідовну кольорову тему, змінюючи світлість для тіней або підсвітлень. Такий рівень гнучкості робить HSL улюбленим інструментом дизайнерів і розробників.

Крім того, HSL робить значення кольорів більш читабельними та зрозумілими порівняно з іншими методами, як-от шістнадцяткові коди або RGB. Ви чіткіше бачите задум кольору і можете відповідно налаштовувати його, особливо під час експериментів із різними дизайнами.

З моделлю кольору HSL ви отримуєте більше гнучкості у контролі тонів, насиченості та світлості, що робить її потужним інструментом для створення насичених і динамічних дизайнів у CSS.
