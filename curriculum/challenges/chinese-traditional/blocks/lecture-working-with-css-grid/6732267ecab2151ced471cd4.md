# --interactive--

在之前的課程中，我們稍微談到如何在網格項目之間創建間距。但在本課程中，我們將更詳細地探討如何在網格佈局中使用 `row-gap`、`column-gap` 和 `gap` 屬性。

但首先我們需要回顧 CSS grid 中的 track 是什麼。track 是兩條相鄰的 grid 線之間的空間。當你使用 CSS Grid 時，這些線會自動生成。在這個背景關係中，track 通常指組成 grid 佈局的列和行。

要在 CSS Grid 的行之間創建間隙，你可以使用 `column-gap` 屬性。此屬性可接受的值包括像素、`em` 單位、百分比或 `normal` 關鍵字。

如果你為 `column-gap` 屬性使用 `normal` 值，則網格配置的結果將是 `0`，多欄配置的結果將是 `1em`。

以下是四行網格佈局標記的範例：

```html
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

對於 CSS，我們將 display 設為 grid 並將 `column-gap` 屬性設為 10 像素：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  height: 100px;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.grid-container div {
  background-color: darkblue;
}
```

:::

如果我們想將範例改為有兩列藍色方塊，並在列之間創建更多空間，我們可以使用 `row-gap` 屬性：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  column-gap: 10px;
  row-gap: 30px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

在此修訂範例中，我們將 `row-gap` 屬性設定為 30 像素，並將 `grid-template-columns` 更改為只使用兩個 `1fr` 單位而非四個，以產生兩列的方塊。

就像 `column-gap` 屬性一樣，`row-gap` 屬性可接受的值可以包括百分比、`em` 和像素。

如果你想使用簡寫來建立列與行之間的間隙，你可以使用 `gap` 屬性。以下是基本語法：

```css
gap: row-value optional-column-value;
```

如果你為 `gap` 屬性指定一個值，則該值將同時應用於列和行。如果你指定兩個值，則第一個值將用於列，第二個值將應用於行：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 30px 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

`gap` 簡寫屬性可接受的值包括百分比、像素、`em` 甚至 `calc()` 值。但你不能在這裡使用 `fr` 單位。

`row-gap`、`column-gap` 和 `gap` 屬性提供靈活的方式來控制 CSS Grid 佈局中項目之間的間距。透過使用這些屬性，你可以輕鬆創建具有一致且可調整的列與行間隙的視覺上吸引人的網格。
