# --interactive--

في CSS، توجد عدة طرق لإضافة حدود حول الصور، كل منها يقدم خيارات تصميم مختلفة ومستويات تحكم متنوعة.

لنستعرض بعض الطرق الأكثر شيوعًا ومرونة.

أسهل طريقة لإضافة حدود إلى صورة هي باستخدام خاصية `border`. هذه الخاصية هي اختصار يسمح لك بتعيين العرض، والنمط، واللون للحدود دفعة واحدة.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

تضيف قاعدة CSS هذه حدودًا حمراء صلبة بعرض 2 بكسل حول جميع عناصر `img`. يمكنك تعديل العرض، والنمط (مثل `dashed`، `dotted`، أو `double`)، واللون بما يتناسب مع تصميمك.

إذا كنت تحتاج إلى مزيد من التحكم في جوانب الحدود الفردية، يمكنك استخدام خصائص الحدود المحددة لكل جانب:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border-top: 10px solid red;
  border-right: 10px dashed green;
  border-bottom: 10px dotted blue;
  border-left: 10px double purple;
}
```

:::

هذا يسمح لك بإنشاء أنماط حدود فريدة لكل جانب من جوانب الصورة.

طريقة أخرى لإنشاء تأثير الحدود هي استخدام خاصية `outline`. على الرغم من تشابهها مع الحدود، إلا أن outline لا تؤثر على أبعاد العنصر أو تخطيطه:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  outline: 3px solid gold;
}
```

:::

هذا ينشئ إطارًا ذهبيًا حول الصورة دون تغيير حجمها أو موضعها.

إذا أردت إنشاء زوايا مستديرة على الحدود، يمكنك استخدام خاصية `border-radius` مع الحدود:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

تذكر، يمكن دمج هذه التقنيات وتخصيصها لإنشاء تأثيرات حدود فريدة. يعتمد اختيار الطريقة على متطلبات التصميم الخاصة بك ومستوى التعقيد الذي تحتاجه.
