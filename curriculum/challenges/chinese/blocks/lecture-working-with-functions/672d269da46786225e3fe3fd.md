# --interactive--

函数是可重复使用的代码片段，可执行特定任务或计算数值。 把函数想象成一台机器，它接受一些输入，对其进行一些运算，然后产生输出。 下面是一个声明函数的示例：

```js
function greet() {
  console.log("Hello, Jessica!");
}
```

In this example, we have declared a function called `greet`.  Inside that function, we have a `console.log` that logs the message `Hello, Jessica!`. 如果我们尝试运行这段代码，并不会在控制台中看到消息。 这是因为我们需要调用函数。

函数调用，或称调用，是指我们实际使用或执行函数。要调用函数，你需要引用函数名，后跟一对括号：

:::interactive_editor

```js
function greet() {
  console.log("Hello, Jessica!");
}

greet(); // "Hello, Jessica!"
```

:::

现在，`Hello, Jessica!` 消息将被记录到控制台。 但是，如果我们想让消息显示 `Hello, Nick!` 或 `Hello, Anna!` 呢？ 我们不想每次迎接不同的用户时都写一个新函数。 相反，我们可以创建一个使用函数形参和实参的可重用函数。 

形参是函数调用时将传递给函数的值的占位符。 它们允许函数接受并使用输入值。 实参是调用函数时传递给函数的实际值。 以下是使用参数的 `greet` 函数的更新版本：

:::interactive_editor

```js
function greet(name) {
  console.log("Hello, " + name + "!");
}

greet("Alice"); // Hello, Alice!
greet("Nick"); // Hello, Nick!
```

:::

`name` 是形参，而字符串 `Alice` 和 `Nick` 是实参。 现在，我们有了一个可复用的函数，可以在代码中使用不同的参数多次复用。 

函数执行完毕后，总会返回一个值。 默认情况下，返回值为 `undefined` 。 这是一个示例： 

:::interactive_editor

```js
function doSomething() {
  console.log("Doing something...");
}

let result = doSomething();
console.log(result); // undefined
```

:::

如果你需要函数返回一个特定值，则需要使用 `return` 语句。 下面是一个使用 `return` 语句返回两个值之和的示例：

:::interactive_editor

```js
function calculateSum(num1, num2) {
  return num1 + num2;
}

console.log(calculateSum(3, 4)); // 7
```

:::

通常你会使用 `return` 语句，因为你可以在以后的代码中使用函数输出的值。

到目前为止，我们一直在使用命名函数，你也可以创建匿名函数。 匿名函数是一个没有名称的函数，可以像这样分配给一个变量：

:::interactive_editor

```js
const sum = function (num1, num2) {
  return num1 + num2;
};

console.log(sum(3, 4)); // 7
```

:::

在本例中，我们有一个名为 `sum` 的 `const` 变量，我们为它指定一个匿名函数，返回 `num1` 与 `num2` 的和。 然后，我们就可以调用 `sum`，并输入数字 `3` 和 `4`，得到结果 `7`。 

函数支持默认参数，允许你为参数设置默认值。 如果调用函数时该形参没有传入实参，则使用这些默认值。 这是一个示例：

:::interactive_editor

```js
function greetings(name = "Guest") {
  console.log("Hello, " + name + "!");
}

greetings(); // Hello, Guest!
greetings("Anna"); // Hello, Anna!
```

:::

在此示例中，如果没有为 `name` 传入参数值，则默认为 `Guest`。

总之，函数可以编写可复用且有条理的代码。 它们可以接受输入（参数）、执行操作并返回输出。
