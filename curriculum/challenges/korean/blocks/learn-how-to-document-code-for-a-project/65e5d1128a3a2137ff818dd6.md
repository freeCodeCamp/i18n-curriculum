# --description--

`Anyone who`는 특정 조건이나 요구 사항을 충족하는 사람을 지칭할 때 사용합니다.

기술 분야에서는 특정 기술이나 지식을 가진 사람을 설명할 때 자주 사용합니다. 예를 들어, `Anyone who understands JavaScript can modify this script.`

# --questions--

## --text--

기술 맥락에서 `Anyone who`를 올바르게 사용한 문장은 무엇인가요?

## --answers--

`Anyone who has access to the server can upload the files.`

`Anyone who please fix this bug.`

### --feedback--

문법적으로 올바른 사용은 `Anyone who knows the solution, please fix this bug.`입니다.

`Anyone who the programmer is busy now.`

### --feedback--

이 문장은 문법적으로 올바르지 않습니다. 더 나은 문장은 `Anyone who is a programmer is busy now.`입니다.

`Who anyone can join the meeting is not clear.`

### --feedback--

이 문장은 구조가 잘못되었습니다. 올바른 형태는 `It is not clear who can join the meeting.`입니다.
