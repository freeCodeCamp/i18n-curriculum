# --description--

JavaScript에서 `throw` 문은 사용자가 정의한 예외를 발생시키는 데 사용됩니다. 프로그래밍에서 예외란 예상치 못한 이벤트가 발생하여 프로그램의 정상 흐름을 방해하는 경우를 말합니다.

프로그래머로서 이러한 예외를 처리하는 것이 중요합니다. 그래야 오류가 발생했을 때 프로그램이 예기치 않게 중단되지 않습니다. `throw` 문의 기본 구문은 다음과 같습니다:

```js
throw expression;
```

이 경우 `expression` 는 발생시키려는 예외를 나타내는 객체나 값입니다. 예를 들어 내장된 예외 클래스인 `Error`, `TypeError`, 또는 `RangeError` 클래스가 있습니다. `throw` 문을 사용해 `TypeError` 을 발생시키는 예는 다음과 같습니다:

```js
function validateNumber(input) {
  if (typeof input !== "number") {
    throw new TypeError("Expected a number, but received " + typeof input);
  }
  return input * 2;
}
```

이 예제에서는 `input` 의 타입이 `number` 가 아닌지 확인합니다. 그렇지 않으면 사용자 정의 메시지와 함께 `TypeError` 을 발생시킵니다. 그렇지 않으면 함수는 `input` 과 `2` 를 곱한 결과를 반환합니다.

더 일반적인 오류 메시지를 발생시키고 싶다면 다음과 같이 `Error` 생성자를 참조할 수 있습니다:

```js
function divide(numerator, denominator) {
  if (denominator === 0) {
    throw new Error("Cannot divide by zero");
  }
  return numerator / denominator;
}
```

다음은 분모가 `0` 인지 확인하는 함수 예제입니다. 그런 경우 `Cannot divide by zero` 라는 사용자 정의 오류 메시지를 발생시킵니다.

다음 수업에서는 JavaScript에서 예외를 우아하게 처리하는 데 사용되는 `try`/`catch` 블록 내에서 오류 메시지를 발생시키는 방법을 살펴보겠습니다.
