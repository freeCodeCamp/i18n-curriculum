# --explanation--

Maria 說，`at the end of each day`，這明確顯示備份是每日排程的。這表示該任務計劃每天執行一次。 例如：

- `At the end of each day, the system runs a data backup。` 系統每晚會存檔所有資料以防止遺失。

- `We ensure the servers are updated at the end of each day。`－更新會在每晚套用到伺服器，以避免工作時間中斷。
