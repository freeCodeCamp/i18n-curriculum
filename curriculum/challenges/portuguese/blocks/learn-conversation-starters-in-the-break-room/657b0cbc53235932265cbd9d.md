# --description--

`How about...?` é frequentemente usado para pedir informações sobre um tópico ou para fazer uma sugestão. Por exemplo:

- `How about the coffee here?` - Isso pergunta a opinião de alguém sobre o café.

- `How about the meeting tomorrow?` - Isso pergunta se haverá uma reunião.
