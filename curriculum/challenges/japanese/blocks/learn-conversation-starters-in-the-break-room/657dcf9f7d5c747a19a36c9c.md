# --description--

現在地の特定の場所や物の存在について尋ねたいときは、質問の構造 `Are there any ... around here?` を使うことができます。例えば：

`Are there any Wi-Fi spots around here?` - この場所の近くにWi-Fiがあるかどうかを尋ねています。

フレーズ `around here` は近くのエリアを指します。
