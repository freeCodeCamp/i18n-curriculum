# --description--

Zu `make sure` bedeutet, etwas sorgfältig zu überprüfen, um sicherzugehen, dass es korrekt ist oder dass etwas erledigt wurde. Zum Beispiel können Sie sagen `I wanted to make sure the door is locked`, wenn Sie das Schloss überprüfen, um zu bestätigen, dass es sicher ist.

`Choices` sind die verschiedenen Optionen, aus denen Sie wählen können. Wenn Sie zum Beispiel eine Eissorte auswählen, könnten die `choices` Schokolade, Vanille oder Erdbeere sein. Das bedeutet, dass Sie mehr als eine Möglichkeit zur Auswahl haben.

Hören Sie Bob zu und füllen Sie die Lücken aus.
