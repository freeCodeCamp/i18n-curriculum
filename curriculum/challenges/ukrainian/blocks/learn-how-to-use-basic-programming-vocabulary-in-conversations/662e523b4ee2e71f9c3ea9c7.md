# --description--

`How much` використовується, щоб дізнатися про кількість або обсяг чогось, що зазвичай не можна порахувати або що не має множини.

У цьому контексті `code` стосується обсягу програмних команд, над якими хтось працює, який вимірюється не підрахунком окремих елементів, а врахуванням його обсягу чи складності.
