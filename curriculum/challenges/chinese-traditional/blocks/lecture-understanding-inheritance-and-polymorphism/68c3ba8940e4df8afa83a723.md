# --description--

繼承是我們接下來要涵蓋的物件導向程式設計（OOP）中的關鍵概念。

讓我們更深入地探討這個概念以及它如何讓你撰寫可重複使用的程式碼。

透過繼承，子類別可以使用基礎類別的屬性和方法。這讓你能夠重複使用程式碼、建立清晰的類別階層，並且在不重寫所有內容的情況下自訂行為。你可以透過擴充現有的方法或在子類別中覆寫它們來自訂。

以下是繼承的基本語法：

```py
class Parent:
    # Parent attributes and methods

class Child(Parent):
    # Child inherits, extends, and/or overrides where necessary
```

為了讓 `Child` 類別繼承自 `Parent` 類別，你必須將 `Parent` 傳遞給 `Child`。

這種風格稱為單一繼承，因為子類別只從一個父類別繼承。

這裡有一個範例：

```py
class Animal:
    def __init__(self, name):
        self.name = name

    def sound(self):
        return f'{self.name} makes a sound'

class Dog(Animal):
    bark = 'woof! woof!! woof!!!'

jack = Dog('Jack')
print(jack.sound())  # Jack makes a sound
print(jack.bark)  # woof! woof!! woof!!!
```

你可以看到我們能夠在子類別 `Dog` 中重複使用父類別 `Animal` 的 `self.name` 屬性和 `sound()` 方法。

讓我們在子類別 `Dog` 中改寫父類別 `Animal` 的 `sound()` 方法，這樣我們就可以讓 `sound()` 使用 `bark` 類別變數：

```py
class Animal:
    def __init__(self, name):
        self.name = name

    def sound(self):
        return f'{self.name} makes a sound.'

class Dog(Animal):
    bark = 'woof! woof!! woof!!!'

    # Override sound() to use bark class variable
    def sound(self):
        return f'{self.name} barks {self.bark}'

jack = Dog('Jack')
print(jack.sound())  # Jack barks woof! woof!! woof!!!
```

如果你想保留 `sound()` 的返回值，並稍後為 `bark` 類別變數添加，你可以使用 `super()` 函式來擴充 `sound()`：

```py
class Animal:
    def __init__(self, name):
        self.name = name

    def sound(self):
        return f'{self.name} makes a sound'

class Dog(Animal):
    bark = 'woof! woof!! woof!!!'

    # Call Animal.sound(), then append bark
    def sound(self):
        base = super().sound()
        return f'{base}, then {self.name} barks {self.bark}'

jack = Dog('Jack')
print(jack.sound())  # Jack makes a sound, then Jack barks woof! woof!! woof!!!
```

在此範例中，`base` 是從 `Animal` 類別呼叫 `sound()` 方法的結果，然後我們將 `Dog` 類別特定的聲音附加到它。這樣，你可以擴充父類別 `Animal` 的功能，同時仍保留其原始行為。

還有多重繼承，子類別可以從多個父類別繼承。

以下是多重繼承的基本語法：

```py
class Parent:
    # Attributes and methods for Parent

class Child:
    # Attributes and methods for Child

class GrandChild(Parent, Child):
    # GrandChild inherits from both Parent and Child
    # GrandChild can combine or override behavior from each
```

一個簡單展示多重繼承的方法是使用青蛙，它既能在陸地上行走，也能在水中游泳：

```py
class Walker:
    def walk(self):
        return 'I can walk on land'

class Swimmer:
    def swim(self):
        return 'I can swim in water'

# Amphibian inherits from both Walker and Swimmer
class Amphibian(Walker, Swimmer):
    def __init__(self, name):
        self.name = name

    def introduce(self):
        return f"I'm {self.name} the frog. {self.walk()} and {self.swim()}."

frog = Amphibian('Freddy')
print(frog.introduce())
# Output: I'm Freddy the frog. I can walk on land and I can swim in water.
```
