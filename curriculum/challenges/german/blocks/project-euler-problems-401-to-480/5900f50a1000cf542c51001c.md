# --description--

Wir sagen, dass eine $d$-stellige positive Zahl (ohne führende Nullen) eine Zahl mit genau einem teilbaren Teilstring ist, wenn genau einer ihrer Teilstrings durch $d$ teilbar ist.

Zum Beispiel ist 5671 eine 4-stellige Zahl mit genau einem teilbaren Teilstring. Unter allen ihren Teilstrings 5, 6, 7, 1, 56, 67, 71, 567, 671 und 5671 ist nur 56 durch 4 teilbar.

Ebenso ist 104 eine 3-stellige Zahl mit genau einem teilbaren Teilstring, weil nur 0 durch 3 teilbar ist. 1132451 ist eine 7-stellige Zahl mit genau einem teilbaren Teilstring, weil nur 245 durch 7 teilbar ist.

Sei $F(N)$ die Anzahl der Zahlen mit genau einem teilbaren Teilstring, die kleiner als $N$ sind. Wir können überprüfen, dass $F(10) = 9$, $F({10}^3) = 389$ und $F({10}^7) = 277\\,674$ gilt.

Finden Sie $F({10}^{19})$.
