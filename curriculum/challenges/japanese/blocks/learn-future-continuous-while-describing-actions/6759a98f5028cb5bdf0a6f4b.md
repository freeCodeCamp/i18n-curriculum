# --explanation--

タグ質問は、主文が肯定文なら否定形になり、主文が否定文なら肯定形になります。

フレーズ`are they?`は、否定文の最後に付け加えられるタグ質問で、情報の確認やチェックを行います。例えば：

`The updates aren't applied automatically, are they?` - ここでは`are they?`が、更新が自動的に適用されていないことを確認しています。主文が否定文なので、タグ質問は肯定形です。
