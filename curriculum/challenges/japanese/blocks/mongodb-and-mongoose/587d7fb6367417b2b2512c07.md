# --description--

**C**RUD パート I - CREATE

まずはスキーマが必要です。各スキーマは MongoDB のコレクションに対応し、そのコレクション内のドキュメントの形状を定義します。スキーマはモデルの構成要素です。複雑なモデルを作るためにネストすることもできますが、ここではシンプルにします。モデルはドキュメントと呼ばれるオブジェクトのインスタンスを作成できるようにします。

サーバーでは、データベースとのやり取りはハンドラ関数内で行われます。これらの関数は何らかのイベント（例：APIのエンドポイントにアクセスされたとき）が発生した際に実行されます。この演習でも同じ方法を使います。`done()` 関数は、挿入、検索、更新、削除などの非同期操作が完了した後に処理を続行できることを知らせるコールバックです。Node の慣例に従っており、成功時は `done(null, data)`、エラー時は `done(err)` として呼び出されるべきです。

注意 - リモートサービスとやり取りする際はエラーが発生する場合があります！

```js
/* Example */

const someFunc = function(done) {
  //... do something (risky) ...
  if (error) return done(error);
  done(null, result);
};
```

# --instructions--

次の形状を持つ `personSchema` という名前の person スキーマを作成してください：

* `name` という必須の `String` 型フィールド
* `age` という `Number` 型フィールド
* `favoriteFoods` という `[String]` 型フィールド

Mongoose の基本的なスキーマ型を使ってください。必要に応じて他のフィールドを追加したり、required や unique のようなシンプルなバリデータを使ったり、デフォルト値を設定してもかまいません。詳しくは <a href="https://www.freecodecamp.org/news/introduction-to-mongoose-for-mongodb-d2a7aa593c57/" target="_blank" rel="noopener noreferrer nofollow">Mongoose 記事</a> を参照してください。

次に、`personSchema` からモデルを作成し、既存の変数 `Person` に代入してください。

# --hints--

mongoose スキーマからインスタンスを作成することが成功するはずです

```js
  const response = await fetch(code + '/_api/mongoose-model', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      name: 'Mike',
      age: 28,
      favoriteFoods: ['pizza', 'cheese']
    })
  });
  if (!response.ok) {
    throw new Error(await response.text());
  }
  const data = await response.json();
  assert.equal(data.name, 'Mike', '"model.name" is not what expected');
  assert.equal(data.age, '28', '"model.age" is not what expected');
  assert.isArray(
    data.favoriteFoods,
    '"model.favoriteFoods" is not an Array'
  );
  assert.include(
    data.favoriteFoods,
    'pizza',
    '"model.favoriteFoods" does not include the expected items'
  );
  assert.include(
    data.favoriteFoods,
    'cheese',
    '"model.favoriteFoods" does not include the expected items'
  );
```
