# --description--

Sie können das `hr`-Tag verwenden, um eine horizontale Linie über die Breite des umgebenden Elements hinzuzufügen. Dies kann genutzt werden, um einen Themenwechsel zu kennzeichnen oder Gruppen von Inhalten optisch zu trennen.

# --instructions--

Fügen Sie ein `hr`-Tag unterhalb des `h4` hinzu, das den Kartentitel enthält.

**Hinweis:** In HTML ist `hr` ein Void-Element und benötigt daher kein separates schließendes Tag.

# --hints--

Ihr Code sollte ein `hr`-Tag in die Auszeichnung einfügen.

```js
assert.lengthOf(document.querySelectorAll('hr'),1);
```

Das `hr`-Tag sollte zwischen dem Titel und dem Absatz stehen.

```js
assert.match(code,/<\/h4>\s*?<hr(>|\s*?\/>)\s*?<p>/gi);
```
