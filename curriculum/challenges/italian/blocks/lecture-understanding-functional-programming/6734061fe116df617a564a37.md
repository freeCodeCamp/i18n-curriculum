# --interactive--

Il currying è una tecnica che trasforma una funzione che prende più argomenti in una sequenza di funzioni, ognuna delle quali prende un solo argomento.

Iniziamo con un esempio base. Immagina di avere una funzione che somma due numeri:

:::interactive_editor

```js
function add(a, b) {
  return a + b;
}

console.log(add(3, 4)); // 7
```

:::

Questa è una funzione che prende due argomenti e restituisce la loro somma. Ora vediamo come possiamo applicare il currying a questa funzione:

:::interactive_editor

```js
function curriedAdd(a) {
  return function(b) {
    return a + b;
  }
}

console.log(curriedAdd(3)(4)); // 7
```

:::

In questo codice convertito con currying, invece di prendere due argomenti contemporaneamente, abbiamo una funzione che prende il primo argomento e restituisce un’altra funzione. Questa funzione restituita prende il secondo argomento ed esegue l’addizione. La chiamiamo come `curriedAdd(3)(4)`, dove ogni coppia di parentesi tonde rappresenta una chiamata di funzione.

Ma perché dovremmo farlo?

Il currying ci permette di creare facilmente alcune funzioni speciali. Per esempio, potremmo creare una funzione che aggiunge sempre cinque a qualsiasi numero:

:::interactive_editor

```js
function curriedAdd(a) {
  return function(b) {
    return a + b;
  }
}

const addFive = curriedAdd(5);
console.log(addFive(10)); // 15
console.log(addFive(20)); // 25
```

:::

Qui, `addFive` è una funzione sempre pronta ad aggiungere cinque a qualunque numero le passiamo. Questo è un esempio semplice di applicazione parziale, dove fissiamo un certo numero di argomenti a una funzione, producendo un’altra funzione che prende meno argomenti.

Anche se i nostri esempi si sono concentrati su funzioni con due argomenti, il currying può essere applicato a funzioni con qualsiasi numero di argomenti.

Man mano che prosegui nel tuo percorso con JavaScript, probabilmente incontrerai molte situazioni in cui il currying può rendere il tuo codice più pulito e più espressivo.

## --text--

Quale delle seguenti è un esempio di funzione curried in JavaScript?

## --answers--

```js
function add(a, b, c) {
  return a + b + c;
}
```

### --feedback--

Questa è una funzione normale che prende tutti i parametri contemporaneamente.

---

```js
function add(a) {
  return function (b) {
    return function (c) {
      return a + b + c;
    };
  };
}
```

---

```js
function add(a, b, c) {
  return a() + b() + c();
}
```

### --feedback--

Questa funzione accetta tutti i parametri contemporaneamente e li chiama come funzioni, cosa che non corrisponde alla definizione di funzione curried.

---

```js
function add(a) {
  function inner(b) {
    return a + b;
  }
}
```

### --feedback--

La funzione interna non viene mai restituita, quindi `add(2)` restituirà `undefined` e `add(2)(3)` lancerà un errore.
