## --text--

哪个句子语法正确？

## --answers--

`If we lose critical data, it do set us back weeks.`

### --feedback--

该句中错误地用 `do` 代替了 `could`。

---

`If we lose critical data, it could set us back weeks.`

---

`If we lose critical data, it could sets us back weeks.`

### --feedback--

动词 `sets` 在此句中形式不正确。

---

`If we losing critical data, it could set us back weeks.`

### --feedback--

在这个句子中，单词 `losing` 被错误地用来代替 `lose`。

## --video-solution--

2

# --explanation--

正确的句子使用 `First Conditional` 结构，该结构由以下部分组成：

`if` + `Present Simple`：`If we lose critical data`。

情态动词（`could`）+ 基本动词：`it could set us back weeks`。


`First Conditional` 用于描述可能发生的未来事件及其结果。
