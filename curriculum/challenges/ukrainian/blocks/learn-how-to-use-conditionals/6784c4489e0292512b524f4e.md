# --explanation--

Ліза використовує `Second Conditional` для обговорення гіпотетичної ситуації.

`Second Conditional` використовується для розмов про нереальні або малоймовірні ситуації в теперішньому або майбутньому часі. Він має структуру `if` + `Present Simple` або `Past Continuous` (для умови) та `might / would / could` + інфінітив (для результату). Наприклад:

- `If I had more time, I would learn a new programming language.` — описує гіпотетичну ситуацію, коли більше часу призвело б до навчання.

- `If the server were faster, it could handle more users.` — сервер зараз не швидкий, але якби він був, міг би підтримувати більше користувачів.
