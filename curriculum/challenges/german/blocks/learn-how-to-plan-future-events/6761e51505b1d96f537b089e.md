# --explanation--

`Going to` wird verwendet, um geplante Handlungen oder Absichten für die Zukunft auszudrücken. Zum Beispiel:

- `We are going to upgrade the system next month.` – Das bedeutet, dass das Upgrade geplant ist und bald stattfinden wird.
  
- `She is going to learn SQL next year.` – Das bedeutet, dass sie beabsichtigt, in Zukunft SQL zu lernen.

In diesem Satz:

`We're` ist die Kontraktion von `we are`.

`Going to` zeigt an, dass die Handlung in naher Zukunft geplant ist.

`Need` ist das Grundverb, das etwas Notwendiges oder Erforderliches anzeigt.
