# --interactive--

Beim Arbeiten mit geordneten Listen in HTML ermöglicht Ihnen das `type`-Attribut, den Stil der Nummerierung festzulegen, zum Beispiel numerisch, alphabetisch oder römisch.

Das `type`-Attribut kann auf einem `ol`-Element (geordnete Liste) gesetzt werden, um das Nummerierungsformat zu steuern:

- `1` für numerische Listen (wie 1, 2, 3, ...),

- `A` für Großbuchstaben-Listen (wie A, B, C, ...),

- `a` für Kleinbuchstaben-Listen (wie a, b, c, ...),

- `I` für Großbuchstaben in römischen Zahlen (wie I, II, III, ...),

- `i` für Kleinbuchstaben in römischen Zahlen (wie i, ii, iii, ...).

So können Sie den Attributselektor verwenden, um eine geordnete Liste mit einem bestimmten `type`-Attribut anzusprechen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="A"] {
  color: purple;
  font-weight: bold;
}
```

:::

In diesem Beispiel wendet die CSS-Regel lila Textfarbe und fettgedruckte Schrift auf jede geordnete Liste an, die Großbuchstaben als Nummerierung verwendet, was durch `type="A"` angegeben wird.

Ebenso können Sie geordnete Listen ansprechen, die Kleinbuchstaben in römischen Zahlen verwenden, indem Sie diejenigen mit `type="i"` auswählen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="i">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="i"] {
  color: green;
}
```

:::

Diese Regel ändert die Textfarbe auf grün für jede geordnete Liste, die Kleinbuchstaben in römischen Zahlen verwendet.

Die Möglichkeit, geordnete Listen basierend auf ihrem `type`-Attribut zu gestalten, gibt Ihnen mehr Kontrolle über das Aussehen von Listen in Ihrem Dokument und erlaubt eine Anpassung je nach Struktur des Inhalts.
