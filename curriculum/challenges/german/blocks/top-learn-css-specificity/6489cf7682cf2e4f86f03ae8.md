# --description--

Vererbung bezieht sich auf bestimmte CSS-Eigenschaften, die, wenn sie auf ein Element angewendet werden, von den Nachfahren dieses Elements übernommen werden, selbst wenn wir keine Regel explizit für diese Nachfahren schreiben. Typografie-basierte Eigenschaften (`color`, `font-size`, `font-family` usw.) werden normalerweise vererbt, während die meisten anderen Eigenschaften dies nicht tun.

Die Ausnahme ist, wenn ein Element direkt angesprochen wird, da dies immer die Vererbung übertrumpft:

```html
<!-- index.html -->

<div id="parent">
  <div class="child"></div>
</div>
```

```css
/* styles.css */

#parent {
  color: red;
}

.child {
  color: blue;
}
```

Obwohl das `parent`-Element eine höhere Spezifität mit einer ID hat, würde das `child`-Element den `color: blue`-Stil erhalten, da diese Deklaration es direkt anspricht, während `color: red` vom übergeordneten Element nur vererbt wird.
