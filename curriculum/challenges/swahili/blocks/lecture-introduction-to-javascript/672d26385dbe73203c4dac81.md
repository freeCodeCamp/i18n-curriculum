# --interactive--

JavaScript ni lugha yenye nguvu ya programu inayoleta mwingiliano na tabia ya mabadiliko kwenye tovuti.

Wakati HTML na CSS zinatumika kuunda muundo wa maudhui na kupamba vipengele kwenye ukurasa, JavaScript huenda zaidi ya hivyo kwa kuwezesha utendaji mgumu zaidi, kama kushughulikia ingizo la mtumizi, kuhuisha vipengele, na hata kujenga programu kamili za mtandao.

Kwa mfano, unapobofya kitufe, kuwasilisha fomu, au kupeleka kidole juu ya menyu, JavaScript huamua jinsi ukurasa unavyotenda.

Hapa kuna mfano wa jinsi hizi tatu zinavyofanya kazi pamoja:

:::interactive_editor

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: green;
    }
  </style>
</head>
<body>
  <h1>Hello, World!</h1>
  <button onclick="alert('Button clicked!')">Click me</button>
</body>
</html>
```

:::

Katika mfano huu, HTML inatumika kufafanua maudhui: kichwa cha sehemu (kipengele cha `h1`) na kitufe (kipengele cha `button`).

CSS inatumika kuweka mitindo kwenye kichwa cha sehemu, kama kufanya maandishi kuwa ya kijani. JavaScript inatumika kuonyesha ujumbe wa tahadhari wakati kitufe kinapobofwa.

Kwa muhtasari, HTML hutoa muundo, CSS inaongeza mitindo, na JavaScript huwezesha tabia, kuunda uzoefu kamili na mwingiliano wa mtandao.
