# --description--

数値 50 を考えます。

${50}^2 = 2500 = 2^2 × 5^4$ なので、$φ(2500) = 2 × 4 × 5^3 = 8 × 5^3 = 2^3 × 5^3$ です。ここで $φ$ はオイラーのトーシェント関数を表します。

つまり、2500 は平方数であり、$φ(2500)$ は立方数です。

$1 < n < {10}^{10}$ を満たし、かつ $φ(n^2)$ が立方数になるすべての数 $n$ の合計を求めてください。
