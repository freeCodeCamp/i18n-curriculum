# --description--

في JavaScript، الوحدة هي وحدة شفرة مستقلة تحتوي على دوال أو أصناف أو متغيرات مرتبطة ببعضها.

فكر في الوحدة كقطعة بناء لتطبيقك، مثل فصل في كتاب. تركز كل وحدة على وظيفة محددة، مما يجعل شفرتك أكثر تنظيمًا وقابلية للصيانة وإعادة الاستخدام. تساعد الوحدات في منع تعارض الأسماء وتسمح لك بتنظيم تطبيقك إلى أجزاء منفصلة ومترابطة.

تطور مفهوم الوحدات في JavaScript مع الوقت، لكن النظام الأكثر استخدامًا ودعمًا هو نظام وحدات ES6 (ECMAScript 2015). يوفر هذا النظام طريقة موحدة لتعريف واستخدام الوحدات عبر بيئات JavaScript المختلفة.

لإنشاء وحدة، اكتب شفرة JavaScript في ملف منفصل. يجب تصدير أي متغيرات أو دوال أو أصناف تريد إتاحتها لأجزاء أخرى من التطبيق صراحةً. يمكنك فعل ذلك باستخدام الكلمة المفتاحية `export`.

على سبيل المثال، لنفترض أن لديك ملفًا يسمى `math.js` يحتوي على بعض دوال الرياضيات:

```js
export function add(a, b) {
    return a + b;
}

export function subtract(a, b) {
    return a - b;
}

const PI = 3.14159;
export { PI };
```

في هذا المثال، نصدر دالتين (`add` و `subtract`) وثابتًا (`PI`). يمكنك تصدير عدد غير محدود من العناصر من وحدة واحدة.

لاستخدام هذه العناصر المصدرة في جزء آخر من التطبيق، تحتاج إلى استيرادها. يتم ذلك باستخدام الكلمة المفتاحية `import`.

لنفترض أنك تريد استخدام دوال الرياضيات هذه في ملف يسمى `app.js`:

```js
import { add, subtract, PI } from './math.js';

console.log(add(5, 3));        // Outputs: 8
console.log(subtract(10, 4));  // Outputs: 6
console.log(PI);               // Outputs: 3.14159
```

هنا، نستورد الدوال والثابت المحددين الذين نحتاجهم من وحدة `math.js`. الجزء `'./math.js'` يخبر JavaScript بمكان العثور على ملف الوحدة نسبةً إلى الملف الحالي.

أحيانًا، قد ترغب في استيراد كل ما تصدره الوحدة. يمكنك فعل ذلك باستخدام صيغة النجمة (`*`):

```js
import * as Math from './math.js';

console.log(Math.add(5, 3));        // Outputs: 8
console.log(Math.subtract(10, 4));  // Outputs: 6
console.log(Math.PI);               // Outputs: 3.14159
```

في هذه الحالة، تُستورد كل الصادرات من `math.js` كخصائص لكائن يسمى `Math`.

نمط شائع آخر هو وجود تصدير افتراضي في الوحدة. يُستخدم هذا عادةً عندما تصدر الوحدة دالة واحدة بشكل رئيسي. يمكنك أن تملك تصديرًا افتراضيًا واحدًا فقط لكل وحدة.

إليك كيف يبدو ذلك:

```js
// In math.js
export default function multiply(a, b) {
    return a * b;
}

// In app.js
import multiply from './math.js';

console.log(multiply(4, 5));  // Outputs: 20
```

لاحظ أنه عند استيراد تصدير افتراضي، لا تحتاج إلى استخدام الأقواس المتعرجة، ويمكنك تسمية الاستيراد بأي اسم تريده.

من المهم أن تعرف أنه لاستخدام وحدات ES6 في المتصفح، يجب تحديد `type` كـ `module` في وسم `script` الخاص بك:

```html
<script type="module" src="app.js"></script>
```

توفر الوحدات طريقة قوية لتنظيم وبناء شفرة JavaScript الخاصة بك. تسمح لك بتقسيم تطبيقك إلى أجزاء أصغر يمكن إدارتها، وتعزز إعادة استخدام الشفرة، وتساعد في الحفاظ على فصل واضح للمسؤوليات. مع بناء تطبيقات أكبر، ستجد أن الوحدات تصبح أداة أساسية في مجموعة أدوات تطوير JavaScript الخاصة بك.
