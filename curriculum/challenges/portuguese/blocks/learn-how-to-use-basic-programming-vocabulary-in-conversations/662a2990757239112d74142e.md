# --description--

Vamos revisar o tempo present perfect nesta tarefa. É usado para descrever ações ou situações que ocorreram em um momento não especificado antes de agora. É formado usando `have/has` + particípio passado. Por exemplo, `She has visited Paris.`

A frase `spend time on` é usada para descrever o uso de uma certa quantidade de tempo em uma atividade, tarefa ou assunto específico. Um exemplo é `She spends a lot of time on her homework.` O particípio passado de `spend` é `spent`.
