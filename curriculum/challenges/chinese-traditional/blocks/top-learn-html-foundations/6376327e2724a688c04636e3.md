# --description--

HTML 和 CSS 是兩種協同運作的語言，用來創建你在網際網路上看到的所有內容。HTML 是構成網頁的未經處理的資料。所有的文字、連結、卡片、列表和按鈕都是用 HTML 產生的。CSS 則是為那些簡單元素添加樣式。HTML 將資訊放在網頁上，而 CSS 則定位那些資訊、賦予顏色、改變字型，並讓它看起來很棒！

# --questions--
    
## --text--

HTML 和 CSS 在建立網頁時的主要責任是什麼？

## --answers--

HTML 結構化內容，而 CSS 為元素設計樣式。

---

CSS 管理功能，HTML 控制版面配置。
