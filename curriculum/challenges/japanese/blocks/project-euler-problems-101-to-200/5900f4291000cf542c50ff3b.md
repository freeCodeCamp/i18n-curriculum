# --description--

数 $a$ の正の整数 $b$ による超指数演算（テトレーション）は、$a↑↑b$ または ${}^ba$ と表され、次のように再帰的に定義されます。

$a↑↑1 = a$、

$a↑↑(k+1) = a^{(a↑↑k)}$。

例えば、$3↑↑2 = 3^3 = 27$ であり、したがって $3↑↑3 = 3^{27} = 7625597484987$、$3↑↑4$ はおよそ ${10}^{3.6383346400240996 \times {10}^{12}}$ となります。$1777↑↑1855$ の下8桁を求めてください。
