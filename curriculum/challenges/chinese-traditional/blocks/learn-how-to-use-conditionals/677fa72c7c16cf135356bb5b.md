# --explanation--

Mark 說，`If the application has memory leaks, it can lead to crashes`。

這裡的關鍵詞是 `memory leaks`。當一個程式為臨時任務配置記憶體，但在任務完成後未能釋放它時，就會發生這種情況。 例如：

- `If the application uses too much memory, it could crash.`－這表示記憶體使用量是導致當機的問題。

- `Memory leaks` 可能會導致效率、性能問題和當機。— 這說明了 `memory leaks` 是造成當機的因素。
