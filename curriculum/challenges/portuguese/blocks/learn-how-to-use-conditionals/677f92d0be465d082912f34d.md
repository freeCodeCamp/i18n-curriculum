# --explanation--

A pergunta `Did you check that?` está perguntando se alguém verificou ou confirmou algo. 

Neste contexto, Mark quer saber se Sarah verificou a conexão do cabo de energia para garantir que está segura. Outros exemplos:

- `Did you check that the windows are closed?` - Isso significa perguntar se alguém garantiu que as janelas estão devidamente fechadas.

- `Did you check that the light is off?` - Isso significa perguntar se alguém confirmou que a luz está apagada.
