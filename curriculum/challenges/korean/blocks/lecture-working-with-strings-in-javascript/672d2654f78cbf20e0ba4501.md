# --interactive--

JavaScript에서 문자열은 문자들의 수열로 취급되며, 문자열 내 각 문자는 대괄호 표기법을 사용해 접근할 수 있습니다. 이를 통해 문자열에서 특정 위치에 있는 문자를 인덱스를 기준으로 가져올 수 있습니다.

인덱스는 문자열 내 문자의 위치를 나타내며, 0부터 시작합니다. 즉, 문자열의 첫 번째 문자는 인덱스 `0`를 가지며, 두 번째 문자는 인덱스 `1`, 이런 식으로 이어집니다.

예를 들어, 문자열 `hello`에서 문자 `h`는 인덱스 `0`에 있고, `e`는 인덱스 `1`, `l`는 인덱스 `2`에 있습니다.

대괄호 표기법은 대괄호(`[]`)와 접근하려는 문자의 인덱스를 사용합니다. 예제를 살펴보겠습니다:

:::interactive_editor

```js
let greeting = "hello";
console.log(greeting[1]); // "e"
```

:::

이 예제에서 인덱스 `1`에 있는 문자 `e`에 접근할 수 있습니다.

문자열의 마지막 문자를 얻으려면 문자열 길이에서 1을 뺀 값을 사용할 수 있습니다.
문자열의 `length` 속성은 문자열에 포함된 문자 수를 알려주므로, 마지막 문자에 접근하려면 길이에서 1을 빼면 됩니다:

:::interactive_editor

```js
let greeting = "hello";
console.log(greeting[greeting.length - 1]); // "o"
```

:::

이 경우 `length`의 `hello`은 `5`이고, 마지막 문자(`o`)는 인덱스 `4`에 있으며, 그 값은 `5 - 1`입니다.

여러 문자를 얻고 싶다면 다음과 같이 대괄호 표기법을 사용할 수 있습니다:

:::interactive_editor

```js
let greeting = "hello";
let firstTwo = greeting[0] + greeting[1]; // "he"
console.log(firstTwo);
```

:::

이 예제에서는 대괄호 표기법을 사용해 첫 번째와 두 번째 문자를 연결하여 문자열 `he`를 만듭니다.

대괄호 표기법은 이름에서 이니셜을 추출하거나 특정 문자를 유효성 검사할 때처럼 문자열 내 특정 문자에 접근해야 할 때 유용합니다.
