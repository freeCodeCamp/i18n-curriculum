# --explanation--

覚えておきましょう：

`May` は可能性や許可を表します。例えば：

`May I use your phone?`

`Might` は `may` よりも可能性が低く、不確かな状況でよく使われます。例えば：

`She might come to the meeting.`

`Can` は能力や許可を表します。例えば：

`Can I leave early?`

`Could` は過去の能力、丁寧な依頼、または `can` よりも可能性が低い場合に使われます。例えば：

`Could you help me?`
