# --interactive--

## Kufanya kazi na mizunguko

- **`for` mzunguko**: Aina hii ya mzunguko hutumika kurudia kipande cha msimbo mara fulani. Mzunguko huu umegawanywa katika sehemu tatu: tamko la uanzishaji, hali, na tamko la kuongeza/kupunguza. Tamko la uanzishaji hufanyika kabla mzunguko haujaanza. Kwa kawaida hutumika kuanzisha kigezo cha kuhesabu. Hali hupimwa kabla ya kila kurudia kwa mzunguko. Kurudia ni kupita mara moja kupitia mzunguko. Ikiwa hali ni `true`, kipande cha msimbo ndani ya mzunguko hufanyika. Ikiwa hali ni `false`, mzunguko unasimama na unaendelea na kipande kingine cha msimbo. Tamko la kuongeza/kupunguza hufanyika baada ya kila kurudia kwa mzunguko. Kwa kawaida hutumika kuongeza au kupunguza kigezo cha kuhesabu.

:::interactive_editor

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

:::

- **`for...of` mzunguko**: Aina hii ya mzunguko hutumika wakati unahitaji kurudia juu ya thamani kutoka kwa kitu kinachoweza kurudiwa. Mifano ya vitu vinavyoweza kurudiwa ni safu za data na mfuatano wa herufi.

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

:::

- **`for...in` mzunguko**: Aina hii ya mzunguko ni bora kutumika wakati unahitaji kurudia juu ya vigezo vya kitu. Mzunguko huu utapita juu ya vigezo vyote vinavyohesabiwa vya kitu, ikiwa ni pamoja na vigezo vilivyorithiwa na vigezo visivyo vya nambari.

:::interactive_editor

```js
const fruit = {
  name: 'apple',
  color: 'red',
  price: 0.99
};

for (const prop in fruit) {
  console.log(fruit[prop]);
}
```

:::

- **`while` mzunguko**: Aina hii ya mzunguko itaendesha kipande cha msimbo mradi tu hali ni `true`.

:::interactive_editor

```js
let i = 5;

while (i > 0) {
  console.log(i);
  i--;
}
```

:::

- **`do...while` mzunguko**: Aina hii ya mzunguko itatekeleza kipande cha msimbo angalau mara moja kabla ya kuangalia hali.

```js
let userInput;

do {
  userInput = prompt("Please enter a number between 1 and 10");
} while (Number(userInput) < 1 || Number(userInput) > 10);

alert("You entered a valid number!");
```

## Tamko za `break` na `continue`

- **Ufafanuzi**: Tamko la `break` hutumika kutoka katika mzunguko mapema, wakati tamko la `continue` hutumika kuruka kurudia kwa sasa na kuendelea na kurudia inayofuata.

:::interactive_editor

```js
// Example of break statement
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, and 4

// Example of continue statement 
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, 4, 6, 7, 8, and 9
```
