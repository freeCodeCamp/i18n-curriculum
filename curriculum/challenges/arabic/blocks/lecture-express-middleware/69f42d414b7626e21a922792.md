# --description--

برمجيات وسيطة على مستوى التطبيق هي برمجيات وسيطة مرتبطة بنسخة من تطبيق Express. تُستخدم عادةً لمعالجة الطلبات التي تتطلب تنفيذ عبر عدة مسارات. يمكن تطبيقها على مستوى عام أو على مسارات محددة.

يُسجل هذا النوع من البرمجيات الوسيطة باستخدام `app.use()` أو طرق المسار المحددة مثل `app.get()`، `app.post()`، وغيرها. تُنفذ مع كل طلب وارد ما لم تُقيد لمسارات معينة.

```js
const express = require('express')
const app = express()

// Application-level middleware that logs all incoming requests
app.use((req, res, next) => {
  console.log('Request received at:', new Date())
  next(); // Passes control to the next middleware
})

app.get('/', (req, res) => {
  res.send('Hello World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* `app.use()`: هذه برمجيات وسيطة على مستوى التطبيق. تسجل كل طلب يُرسل إلى الخادم، بغض النظر عن المسار.
* تعمل البرمجيات الوسيطة مع كل طلب وارد، وتوفر للخادم تفاصيل مثل الطابع الزمني الحالي.

تُفيد برمجيات وسيطة على مستوى التطبيق في المهام التي يجب تطبيقها عبر العديد من المسارات أو كلها في التطبيق. قد تشمل ذلك التسجيل، التوثيق، ومعالجة الأخطاء.
