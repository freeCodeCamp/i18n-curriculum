# --questions--    

## --text--

當你在 Chrome Dev Tools 中檢查一個元素時，如果一個樣式屬性顯示為刪除線，這表示什麼？

## --answers--

該樣式屬性目前已被停用，且未套用到該元素。

---

樣式屬性會被更具體的 CSS 規則覆寫。

---

`style` 屬性已被棄用，不應該使用。

---

該 style 屬性是實驗性的，可能並非所有瀏覽器都支援。
