# --explanation--

`Several` 意味着多于几个但不一定是大量。它表示一个未指定的数量，超过两个。例如：

`We've received several requests for a new feature.` - 这意味着有多个请求，但不是特定的大数量。

`Present Perfect` 时态出现在 `which has really helped the team` 中，是因为发现问题的积极影响始于过去并且仍然影响着团队。例如：

`她在软件中的评论使新员工更容易理解代码。` - 这意味着她的评论至今仍然有益。
