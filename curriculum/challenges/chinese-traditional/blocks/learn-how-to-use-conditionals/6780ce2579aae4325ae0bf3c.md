# --explanation--

`Critical` 意味著非常重要或不可或缺的東西，而 `critical data` 指的是團隊依賴以有效運作的重要資訊。遺失它會導致延誤或問題。例如：

- `The backup includes all critical data in case of a failure.` - 儲存的副本包含從問題中復原所需的關鍵資訊。

- `You must encrypt critical data to ensure it's secure。`－使用加密保護重要資訊是確保安全所必須的。
