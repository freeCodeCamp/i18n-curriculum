# --description--

패딩은 정의된 테두리 안쪽, 요소의 콘텐츠 주변에 공간을 만들 때 사용합니다. 인라인 요소에서는 패딩이 콘텐츠의 왼쪽, 오른쪽, 위, 아래에 공간을 추가합니다. 하지만 패딩이 콘텐츠 주변 공간에 영향을 주긴 해도 인라인 요소 주변의 다른 요소 배치에는 영향을 주지 않는다는 점이 중요합니다. 즉, 수평 패딩(왼쪽과 오른쪽)은 반영되지만, 수직 패딩(위와 아래)은 다른 요소가 멀어지게 하지 않습니다.
