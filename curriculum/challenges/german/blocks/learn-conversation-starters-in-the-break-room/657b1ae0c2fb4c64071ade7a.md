# --description--

Wenn Sie nach etwas Bestimmtem fragen möchten, können Sie das Fragewort `what` verwenden. `What` wird häufig verwendet, um über Dinge zu sprechen.

Normalerweise stellen Sie `what` an den Anfang einer Frage. Dann folgt das Verb `to be` (wie `is` oder `are`) oder andere Verben, und danach das Subjekt (wie `you` oder `the team`). Zum Beispiel:

- `What is your favorite color?` – Jemand möchte wissen, welche Farbe Sie am meisten mögen.

- `What does this code do?` – Jemand möchte den Beruf oder das Ergebnis des Codes wissen.
