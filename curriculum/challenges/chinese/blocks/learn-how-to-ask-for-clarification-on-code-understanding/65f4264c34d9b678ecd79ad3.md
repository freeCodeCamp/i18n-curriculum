# --description--

`Uncertain` 意味着对某事不确定，比如对一个决定感到不确定。相反，`certain` 意味着确定。例如，`I am certain this is the right way` 意味着对方向有把握。

`Achieve` 意味着经过努力成功达到目标，例如 `achieve a good result in a test`。在编程中，`achieve` 通常指程序员希望通过他们的代码完成的目标。
