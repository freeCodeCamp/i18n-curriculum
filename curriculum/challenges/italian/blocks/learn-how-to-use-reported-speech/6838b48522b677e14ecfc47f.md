# --explanation--

Il tempo `Past Continuous` (`was/were` + verbo nella forma `-ing`) descrive un'azione che stava accadendo per un periodo di tempo nel passato. Viene spesso usato insieme a un evento `Simple Past` per mostrare che l'azione continua è stata interrotta o è avvenuta prima di un altro evento. Per esempio:

`She was working on the report when the computer crashed.` – L'azione di lavorare era in corso quando è avvenuto il crash (`Simple Past`).
