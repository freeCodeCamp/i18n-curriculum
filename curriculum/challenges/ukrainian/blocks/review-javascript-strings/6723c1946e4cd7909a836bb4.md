# --interactive--

## Основи рядків

- **Визначення**: Рядок — це послідовність символів, укладена в одинарні лапки, подвійні лапки або поділки. Рядки — це примітивні типи даних, і вони незмінні. Незмінність означає, що після створення рядка його не можна змінити. 
- **Доступ до символів рядка**: Щоб отримати доступ до символу рядка, можна використати дужкову нотацію і передати індекс символу. Індекс — це позиція символу в рядку, і він починається з нуля.

:::interactive_editor

```js
const developer = "Jessica";
console.log(developer[0]); // J
```

:::

- **`\n` (символ нового рядка)**: Ви можете створити новий рядок у рядку, використовуючи символ нового рядка `\n`.

:::interactive_editor

```js
const poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

- **Екранування рядків**: Ви можете екранувати символи в рядку, поставивши перед лапками зворотні слеші (`\`).

:::interactive_editor

```js
const statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

## Шаблонні літерали (шаблонні рядки) та інтерполяція рядків

- **Визначення**: Шаблонні літерали визначаються за допомогою поділок (`). Вони дозволяють легше маніпулювати рядками, зокрема вставляти змінні безпосередньо в рядок — це функція, відома як інтерполяція рядків.

:::interactive_editor

```js
const name = "Jessica";
const greeting = `Hello, ${name}!`; 
console.log(greeting); // "Hello, Jessica!"
```

:::

## ASCII, метод `charCodeAt()` та метод `fromCharCode()`

- **ASCII**: ASCII (Американський стандартний код для обміну інформацією) — це стандарт кодування символів, який використовується для представлення базових англійських символів за допомогою числових значень. Раніші уроки знайомлять з `charCodeAt()` і `fromCharCode()` на прикладах ASCII.
- **Unicode**: Рядки JavaScript внутрішньо використовують Unicode, зокрема кодування UTF-16. Для перших 128 символів (базові латинські літери, цифри та загальні символи) значення Unicode співпадають з кодами ASCII. Саме тому приклади на основі ASCII продовжують працювати в JavaScript.
- **Метод `charCodeAt()`**: Цей метод повертає одиницю коду UTF-16 символу за вказаним індексом. Для базових латинських символів це значення співпадає з кодом ASCII.

:::interactive_editor

```js
const letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

:::

- **Метод `fromCharCode()`**: Цей метод перетворює код ASCII у відповідний символ.

:::interactive_editor

```js
const char = String.fromCharCode(65);
console.log(char);  // A
```

:::

## Інші загальні методи рядків

- **Метод `indexOf`**: Цей метод використовується для пошуку підрядка в рядку. Якщо підрядок знайдено, `indexOf` повертає індекс (або позицію) першого входження цього підрядка. Якщо підрядок не знайдено, `indexOf` повертає -1, що означає, що пошук не вдався.

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.indexOf("fox")); // 16
console.log(text.indexOf("cat")); // -1
```
  
:::  

- **Метод `includes()`**: Цей метод перевіряє, чи містить рядок певний підрядок. Якщо підрядок знайдено, метод повертає true. Інакше — хиба.

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.includes("fox")); // true
console.log(text.includes("cat")); // false
```

:::

- **Метод `slice()`**: Цей метод витягує частину рядка і повертає новий рядок, не змінюючи оригінал. Він приймає два параметри: початковий індекс і необов’язковий кінцевий індекс.

:::interactive_editor

```js
const text = "freeCodeCamp";
console.log(text.slice(0, 4));  // "free"
console.log(text.slice(4, 8));  // "Code"
console.log(text.slice(8, 12)); // "Camp"
```

:::

- **Метод `toUpperCase()`**: Цей метод перетворює всі символи рядка на великі літери і повертає новий рядок з усіма символами у верхньому регістрі.

:::interactive_editor

```js
const text = "Hello, world!";
console.log(text.toUpperCase()); // "HELLO, WORLD!"
```

:::

- **Метод `toLowerCase()`**: Цей метод перетворює всі символи рядка на нижній регістр.

:::interactive_editor

```js
const text = "HELLO, WORLD!"
console.log(text.toLowerCase()); // "hello, world!"
```

:::

- **Метод `replace()`**: Цей метод дозволяє знайти в рядку певне значення (наприклад, слово або символ) і замінити його іншим значенням. Метод повертає новий рядок із заміною і залишає оригінал без змін, оскільки рядки JavaScript незмінні.

:::interactive_editor

```js
const text = "I like cats";
console.log(text.replace("cats", "dogs")); // "I like dogs"
```

:::

- **Метод `replaceAll()`**: Цей метод дозволяє знайти всі входження певного значення (слова, символу або шаблону) в рядку і замінити їх іншим значенням. Він працює як `replace()`, але замість того, щоб зупинитися після першого збігу, оновлює кожен знайдений збіг у рядку.

:::interactive_editor

```js
const text = "I love cats and cats are so much fun!";
console.log(text.replaceAll("cats", "dogs")); // "I love dogs and dogs are so much fun!"
```

:::

- **Метод `repeat()`**: Цей метод використовується для повторення рядка певну кількість разів.

:::interactive_editor

```js
const text = "Hello";
console.log(text.repeat(3)); // "HelloHelloHello"
```

:::

- **Метод `trim()`**: Цей метод видаляє пробіли з початку і кінця рядка.

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trim()); // "Hello, world!"
```

:::

- **Метод `trimStart()`**: Цей метод видаляє пробіли з початку (або «старту») рядка.

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trimStart()); // "Hello, world!  "
```

:::

- **Метод `trimEnd()`**: Цей метод видаляє пробіли з кінця рядка.

:::interactive_editor

```js
const text = " Hello, world! ";
console.log(text.trimEnd()); // "  Hello, world!"
```

:::

- **Метод `prompt()`**: Цей метод `window` використовується для отримання інформації від користувача через форму діалогового вікна. Цей метод приймає два аргументи. Перший аргумент — це повідомлення, яке з’явиться у вікні діалогу, зазвичай з проханням ввести інформацію. Другий — значення за замовчуванням, яке є необов’язковим і спочатку заповнює поле введення.

```js
const answer = window.prompt("What's your favorite animal?"); // This will change depending on what the user answers
```
