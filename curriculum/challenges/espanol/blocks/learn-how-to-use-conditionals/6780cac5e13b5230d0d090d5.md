# --explanation--

La oración correcta usa una estructura `First Conditional`, que se forma con:

`if` + `Present Simple`: `If we lose critical data`.

Verbo modal (`could`) + verbo base: `it could set us back weeks`.


El `First Conditional` se usa para describir un evento futuro posible y su resultado.
