# --description--

Express надає низку вбудованих функцій підпрограмного забезпечення, які допомагають виконувати загальні завдання, такі як обслуговування статичних файлів, аналіз JSON-тіл запитів та інше. Ви також можете використовувати стороннє підпрограмне забезпечення для додавання функціональності.

**Ось кілька прикладів вбудованого підпрограмного забезпечення**:

1. `express.json()`: Вбудоване підпрограмне забезпечення, яке аналізує вхідні JSON-корисні навантаження.

```js
app.use(express.json())
```

1. `express.static()`: Підпрограмне забезпечення для обслуговування статичних файлів, таких як зображення, CSS і JavaScript-файли.

```js
app.use(express.static('public'))
```

1. `express.urlencoded()`: Підпрограмне забезпечення, яке аналізує вхідні запити з URL-кодованими корисними навантаженнями, форматом, який зазвичай надсилають HTML-форми. Воно робить проаналізовані дані доступними на `req.body`.

```js
app.use(express.urlencoded({ extended: true }))
```

**Тепер ось кілька прикладів стороннього підпрограмного забезпечення:**

1. `cors`: Популярне стороннє підпрограмне забезпечення для увімкнення Cross-Origin Resource Sharing.

```bash
npm install cors
```

```js
const cors = require('cors')
app.use(cors())
```

1. `morgan`: Підпрограмне забезпечення для логування HTTP-запитів.

```bash
npm install morgan
```

```js
const morgan = require('morgan')
app.use(morgan('tiny'))
```

Тепер розглянемо приклад використання як вбудованого, так і стороннього підпрограмного забезпечення:

```js
const express = require('express')
const cors = require('cors')
const morgan = require('morgan')
const app = express()

app.use(express.json()) // Built-in middleware to parse JSON
app.use(cors())       // Third-party middleware for CORS
app.use(morgan('tiny')) // Third-party middleware for logging

app.get('/', (req, res) => {
  res.send('Hello, World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* Вбудоване підпрограмне забезпечення `express.json()` аналізує вхідні JSON-корисні навантаження.
* Стороннє підпрограмне забезпечення `cors` дозволяє крос-доманні запити, а `morgan` веде логування вхідних HTTP-запитів.

Важливо зазначити, що функції підпрограмного забезпечення додаються в тому порядку, в якому їх викликають за допомогою `app.use()`.

Express надає необхідне вбудоване підпрограмне забезпечення для обробки загальних завдань, таких як аналіз JSON і обслуговування статичних файлів. Ви також можете розширити функціональність за допомогою стороннього підпрограмного забезпечення для додаткових можливостей.
