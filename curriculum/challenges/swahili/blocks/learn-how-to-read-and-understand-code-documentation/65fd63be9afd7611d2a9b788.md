# --description--

Sehemu ya `introductory` ni mwanzo wa hati inayotambulisha mada kuu. Mfano: `The introductory chapter of the book outlines the key concepts.`

`paragraph` ni sehemu ya maandishi kuhusu mada fulani. Kama vile, `Each paragraph in the blog post focuses on a different technology trend.`

`To cover` inamaanisha kushughulikia au kujumuisha maudhui maalum. Kwa mfano, katika mwongozo wa teknolojia, `This section covers the steps to install software.`
