# --explanation--

`On vacation` bedeutet, für eine Zeit nicht zu arbeiten, normalerweise zur Erholung oder zum Reisen. Zum Beispiel:

`She's on vacation until next Monday.` – Das bedeutet, dass sie gerade nicht arbeitet.

`In July` bezieht sich auf etwas, das im siebten Monat des Jahres passiert. Die Präposition `in` wird vor Monatsnamen verwendet, um anzugeben, wann etwas stattfindet. Zum Beispiel:

`We have a big release scheduled in July.` – Das bedeutet, dass die Veröffentlichung irgendwann in diesem Monat stattfinden wird.
