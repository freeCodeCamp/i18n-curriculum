# --description--

若要讓 `option` 預設被選取，你可以將 `selected` 屬性添加到你想要被選取的 `option` 元素。

以下是使用 `selected` 屬性的範例：

```html
<option selected value="amazing">Amazing</option>
```

在你的 `select` 元素內，為值為 `"excellent"` 的 `option` 元素添加 `selected` 屬性。

# --hints--

你的 `option` 元素，其值為 `"excellent"`，應該具有 `selected` 屬性。

```js
assert.exists(document.querySelector('option[value="excellent"][selected]'));
```
