# --explanation--

`On another note`는 대화에서 새로운 주제를 소개할 때 사용합니다. 예를 들어:

`On another note, have you seen the new project updates?` – 화자가 주제를 바꾸고 있다는 뜻입니다.

`To touch base on something`는 연락하거나 주제에 대해 잠깐 이야기하는 것을 의미합니다. 예를 들어:

`We should touch base on the client's feedback before the meeting.` – 클라이언트가 말한 내용에 대해 짧게 이야기한다는 뜻입니다.
