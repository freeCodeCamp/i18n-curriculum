# --description--

$S$ を、10進法で書かれた連続する正の整数（1から始まる）をつなげて作られた（無限の）文字列と呼びます。

つまり、$S = 1234567891011121314151617181920212223242\ldots$ です。

任意の数が $S$ に無限回現れることは簡単にわかります。

$n$ 番目に $S$ に現れる $n$ の開始位置を $f(n)$ と呼びます。例えば、$f(1) = 1$、$f(5) = 81$、$f(12) = 271$、$f(7780) = 111\,111\,365$ です。

$1 \leq k \leq 13$ のときの $\sum f(3^k)$ を求めてください。
