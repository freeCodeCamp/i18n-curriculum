Esta palabra se usa cuando se pide información específica, como el apellido de alguien. No olvides el acento en la `a`.

### --feedback--

Esta palabra se usa cuando preguntas si la persona puede hacer algo.

# --explanation--

Cada palabra interrogativa se usa para un propósito comunicativo diferente en español.

`¿Cómo te llamas?` se usa para preguntar el nombre de alguien. Aquí, `cómo` introduce la pregunta usando el verbo `llamarse`.

`¿Cuál es tu apellido?` se usa para pedir información específica. `cuál` se usa comúnmente cuando se solicitan detalles identificativos como nombres, números o correos electrónicos.

`¿Puedes repetirlo?` es una solicitud cortés. `puedes` viene del verbo `poder` y se usa en la estructura:

`¿Puedes` + infinitivo + `...?`

Este patrón se usa comúnmente para pedirle a alguien que haga algo de manera educada.
