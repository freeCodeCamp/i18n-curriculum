# --description--

一組正整數三元組（$a$、$b$、$c$）若符合以下條件，則稱為 Cardano 三元組：

$$\sqrt[3]{a + b \sqrt{c}} + \sqrt[3]{a - b \sqrt{c}} = 1$$

例如，(2,1,5) 是一個 Cardano 三元組。

存在 149 組 Cardano Triplets，且 $a + b + c ≤ 1000$。

找出有多少個 Cardano Triplets 滿足 $a + b + c ≤ 110\\,000\\,000$。
