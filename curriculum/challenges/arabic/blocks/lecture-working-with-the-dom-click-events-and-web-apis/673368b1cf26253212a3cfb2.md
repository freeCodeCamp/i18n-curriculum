# --description--

دعنا نتعلم عن طريقة `querySelectorAll()`. يمكنك استخدام هذه الطريقة للحصول على قائمة بكل عناصر DOM التي تطابق محدد CSS معين.

هكذا يمكنك استدعاء طريقة `querySelectorAll()`:

```js
document.querySelectorAll(selectors);
```

استدعيها على كائن `document` ومرر سلسلة نصية تحتوي على محددات CSS كمعلمة. يجب أن تكون المعلمة سلسلة محدد CSS صحيحة. وإلا، سيتم رمي استثناء `SyntaxError`.

ترجع `querySelectorAll()` كائن `NodeList`، وهو مجموعة من العقد التي تطابق محدد CSS المحدد.

ستحتوي القائمة على كائن `Element` واحد لكل عنصر يطابق محددات CSS. إذا لم يتم العثور على تطابقات، ستكون القائمة فارغة.

ستكون العناصر بالترتيب الذي تظهر به في مستند HTML. على سبيل المثال، يمكنك مطابقة كل العناصر من نوع معين:

```js
document.querySelectorAll("div");
```

كل العناصر التي تحمل صنفًا معينًا:

```js
document.querySelectorAll(".rounded");
```

كل العناصر التي تحمل معرفًا معينًا:

```js
document.querySelectorAll("#logo");
```

أو كل العناصر التي تحمل خاصية معينة، مثل كل الروابط التي توجه المستخدمين إلى عنوان URL معين:

```js
document.querySelectorAll("a[href='https://www.freecodecamp.org/']");
```

يمكنك أيضًا استخدام محددات CSS أكثر تعقيدًا، مثل هذا الذي يطابق كل عناصر القائمة داخل عنصر قائمة غير مرتبة يحمل الصنف `ingredients`:

```js
document.querySelectorAll("ul.ingredients li");
```

بما أن الطريقة سترجع مجموعة من العقد، يمكنك تعيين هذه المجموعة إلى متغير واستخدامها في برامجك للعمل مع العناصر الفردية، كما ترى في هذا المثال مع الصيغة العامة:

```js
const matches = document.querySelectorAll(selectors); 
```

على سبيل المثال، إذا كان لديك هذه القائمة غير المرتبة التي تحتوي على ثلاثة عناصر قائمة:

```html
<ul class="ingredients">
  <li>Flour</li>
  <li>Cheese</li>
  <li>Water</li>
</ul>
```

يمكنك استدعاء طريقة `querySelectorAll()` لمطابقة كل عناصر القائمة داخل القائمة غير المرتبة التي تحمل الصنف `ingredients` وتعيين قيمة الإرجاع إلى الثابت `matches`:

```js
const matches = document.querySelectorAll("ul.ingredients li");
```

سيُرجع هذا مجموعة من ثلاث عقد. كل عقدة تمثل عنصر قائمة:

```js
// NodeList(3)
{
  0: `<li>Flour</li>`,
  1: `<li>Cheese</li>`,
  2: `<li>Water</li>`,
  length: 3,
}
```

يمكنك العمل مع هذه المجموعة تمامًا كما تعمل مع أي مصفوفة JavaScript أخرى.

على سبيل المثال، يمكنك عرضها في وحدة التحكم باستخدام `console.log()`:

```js
console.log(matches);
```

يمكنك أيضًا التحقق من طول المجموعة باستخدام الخاصية `length`:

```js
console.log(matches.length);
```

سيكون الناتج عددًا صحيحًا يمثل عدد العقد في المجموعة:

```md
3
```

يمكنك أيضًا الوصول إلى العناصر الفردية في المجموعة باستخدام الفهارس المقابلة لها. سيكون العنصر الأول عند الفهرس `0`:

```js
console.log(matches[0]);
console.log(matches[1]);
console.log(matches[2]);
```

هذا هو الناتج:

```html
<li>Flour</li>
<li>Cheese</li>
<li>Water</li>
```

تُطبع العناصر واحدة تلو الأخرى في وحدة التحكم.

يمكنك أيضًا التكرار على العناصر باستخدام حلقة `for`:

```js
for (let i = 0; i < matches.length; i++) {
  console.log(matches[i]);    
}
```

سيكون الناتج هو نفسه تمامًا في هذه الحالة. لكن هذا يمكن أن يكون قويًا جدًا للعمل مع كل عناصر المجموعة.

طريقة `querySelectorAll()` في JavaScript هي أداة قوية لاختيار عدة عناصر بناءً على محددات CSS الخاصة بها. باستخدام هذه الطريقة، يمكنك اختيار عناصر محددة داخل صفحات الويب الخاصة بك والتعامل معها حسب الحاجة.
