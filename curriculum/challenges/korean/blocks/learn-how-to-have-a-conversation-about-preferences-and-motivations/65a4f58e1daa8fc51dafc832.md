# --description--

`Which`는 알려진 옵션 집합에서 하나 이상의 항목에 대해 질문할 때 사용합니다. 예를 들어, `Which computer do you prefer?`은 알려진 컴퓨터 중에서 선택을 요구합니다.

`What`는 더 개방적이며, `What do you like to do?`처럼 어떤 답변도 가능할 때 사용합니다.

`Absolute`는 완전하거나 전부를 의미합니다. 무언가가 최대한임을 강조할 때 쓰입니다. 예를 들어, `My absolute favorite game is chess`는 체스가 의심할 여지 없이 가장 좋아하는 것임을 뜻합니다.
