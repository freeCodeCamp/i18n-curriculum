# --description--

聲明一個 `average` 函數，該函數以數字數組作爲 `nums` 參數。 它應該返回數組中所有數字的平均值。

可以通過將數組中所有數字的總和除以數組的長度來計算平均值。 請記住，你有一個可以使用的 `sum` 函數。
