# --explanation--

`Lightweight` descreve algo que não requer muitos recursos ou energia, tornando-o fácil de executar, rápido e eficiente.Por exemplo:

`The app is lightweight, so it runs smoothly even on older smartphones.` - Isso significa que o app usa muito pouca memória ou poder de processamento, permitindo que ele funcione bem em dispositivos com menos capacidade.

`Features` são funções ou ferramentas específicas que um produto ou software oferece aos usuários.Por exemplo:

`The software has many useful features like file sharing and cloud storage.` - Isso significa que o software inclui várias funções úteis que o tornam mais valioso e eficiente para os usuários.
