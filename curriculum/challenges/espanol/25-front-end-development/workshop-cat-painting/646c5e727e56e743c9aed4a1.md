# --description--

La propiedad de posicionamiento en CSS te permite establecer cómo deseas que un elemento se posicione en el navegador. Tiene una propiedad `position` que se puede establecer en `static`, `absolute`, `relative`, `sticky` o `fixed`.

Una vez que estableces la propiedad `position` del elemento, puedes mover el elemento ajustando un valor en píxeles o un porcentaje para una o más de las propiedades `top`, `right`, `left` o `bottom`.

`static` es el posicionamiento predeterminado para todos los elementos. Si lo asignas a un elemento, no podrás moverlo con `top`, `right`, `left` o `bottom`.

Asigna a `.cat-head` una propiedad `position` de `static`, luego establece las propiedades `top` y `left` en `100px` cada una.
