Dieses Wort bezeichnet Fehler oder Probleme in einem Programm, die Schwierigkeiten verursachen. Verwenden Sie die Mehrzahlform.

# --explanation--

Ein `critical bug` ist ein schwerwiegender Fehler in der Software, der die Funktionalität beeinträchtigt und sofort behoben werden muss. Zum Beispiel:

- `We need to fix the critical bugs before launching the app.` – Die schwerwiegendsten Probleme müssen vor der Veröffentlichung gelöst werden.

- `A critical bug caused the website to crash.` – Ein schwerwiegender Fehler ließ die Website nicht mehr funktionieren.
