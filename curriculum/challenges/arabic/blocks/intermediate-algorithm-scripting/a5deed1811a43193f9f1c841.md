# --description--

بالنظر إلى المصفوفة `arr`، قم بالتكرار خلال عناصرها واحذف كل عنصر بدءًا من العنصر الأول (الفهرس 0) حتى تُرجع الدالة `func` القيمة `true` عند تمرير العنصر الجاري تكراره من خلالها.

ثم أعد بقية المصفوفة بمجرد تحقق الشرط، وإلا يجب إرجاع `arr` كمصفوفة فارغة.

# --solutions--

```js
function dropElements(arr, func) {
  while (arr.length && !func(arr[0])) {
    arr.shift();
  }
  return arr;
}
```
