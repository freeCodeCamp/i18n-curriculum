# --description--

`Identity verification` è come dimostrare chi sei. È un modo per confermare che la persona che usa un sistema o un servizio è davvero chi dice di essere. Questo può comportare mostrare un documento d'identità, usare una password o utilizzare dati biometrici come il riconoscimento facciale o le impronte digitali.
