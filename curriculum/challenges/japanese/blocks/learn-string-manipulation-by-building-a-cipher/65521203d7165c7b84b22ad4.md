# --description--

ループを使うと、シーケンスの要素を順番に繰り返し処理して、それぞれに対してアクションを実行できます。

ここでは、`for` ループを使います。`text` を繰り返す方法は次の通りです。

```py
for i in text:
```

`for` はループの種類を示すキーワードです。`i` は `text` の要素の値を順に受け取る変数です。文の最後はコロン `:` で終わります。

`alphabet` を宣言した行の下に、`for` を繰り返す `text` ループを書いてください。ループ変数には `i` を使います。

こうすると、ターミナルにエラーが出ます。次のステップでその内容を学びます。

# --hints--

ループを作るには `for` キーワードを使ってください。行の先頭に `for` キーワードを置き、その後に半角スペースを入れることを忘れないでください。

```js
const commentless_code = __helpers.python.removeComments(code);
assert.match(commentless_code, /^for\s+/m)
```

`i` キーワードの後に `for` 変数を書く必要があります。

```js
const commentless_code = __helpers.python.removeComments(code);
assert.match(commentless_code, /^for\s+i/m)
```

`in` の後に `for i ` キーワードを書いてください。`in` キーワードの前後にスペースを入れることを忘れないでください。

```js
const commentless_code = __helpers.python.removeComments(code);
assert.match(commentless_code, /^for\s+i\s+in\s+/m)
```

`text` の後に `for i in ` を書いてください。最後に `:` を付けるのを忘れないでください。

```js
const commentless_code = __helpers.python.removeComments(code);
assert.match(commentless_code, /^for\s+i\s+in\s+text\s*:\s*$/m)
```

`for` のコード行の下に `alphabet = 'abcdefghijklmnopqrstuvwxyz'` ループを置いてください。

```js
const commentless_code = __helpers.python.removeComments(code);
assert.match(commentless_code, /^alphabet\s*=\s*("|')abcdefghijklmnopqrstuvwxyz\1\s*^for\s+i\s+in\s+text\s*:\s*$/m)
```
