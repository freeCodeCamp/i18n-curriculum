# --description--

`Overwhelming` bedeutet, dass sich etwas zu viel anfühlt, um es zu bewältigen.

Zum Beispiel `The amount of information was overwhelming.` (was bedeutet, dass es viel war!)

`Effective` ist ein Adjektiv, das etwas beschreibt, das ein gewünschtes Ergebnis erzielt. Zum Beispiel `An effective method`.

`Effectively` ist ein Adverb und bedeutet auf eine `effective` Weise. Zum Beispiel `He completed the task effectively.`
