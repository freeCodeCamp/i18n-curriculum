# --description--

`Serverless computing` 是一种 `cloud computing` 执行模型，其中云提供商运行服务器并动态管理机器资源的分配。定价基于应用实际消耗的资源量，而不是预先购买的容量单位。之所以称为 `serverless`，是因为开发者无需处理服务器——他们只需上传他们的代码，云提供商负责其余部分。

例如，在 `serverless computing` 中，你可能会将一个函数上传到云服务，并且它只在需要时运行。
