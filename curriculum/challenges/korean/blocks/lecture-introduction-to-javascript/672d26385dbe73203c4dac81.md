# --interactive--

JavaScript는 웹사이트에 상호작용성과 동적 동작을 더하는 강력한 프로그래밍 언어입니다.

HTML과 CSS가 페이지의 콘텐츠 구조와 요소 스타일을 정의하는 마크업 언어인 반면, JavaScript는 사용자 입력 처리, 요소 애니메이션, 심지어 완전한 웹 애플리케이션 구축 같은 더 복잡한 기능을 가능하게 합니다.

예를 들어, 버튼을 클릭하거나, 폼을 제출하거나, 메뉴 위에 마우스를 올리면 JavaScript가 페이지가 어떻게 동작할지 결정합니다.

세 가지가 함께 작동하는 예시는 다음과 같습니다:

:::interactive_editor

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: green;
    }
  </style>
</head>
<body>
  <h1>Hello, World!</h1>
  <button onclick="alert('Button clicked!')">Click me</button>
</body>
</html>
```

:::

이 예시에서 HTML은 콘텐츠를 정의하는 데 사용됩니다: 제목(`h1` 요소)과 버튼(`button` 요소).

CSS는 제목에 녹색 텍스트 같은 스타일을 적용하는 데 사용됩니다. JavaScript는 버튼을 클릭할 때 경고 메시지를 표시하는 데 사용됩니다.

요약하자면, HTML은 구조를 제공하고, CSS는 스타일을 추가하며, JavaScript는 동작을 가능하게 하여 완전하고 상호작용적인 웹 경험을 만듭니다.
