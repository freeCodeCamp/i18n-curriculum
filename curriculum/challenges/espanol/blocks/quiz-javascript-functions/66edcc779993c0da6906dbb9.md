¿Cuál de las siguientes es la manera correcta de declarar una función en JavaScript?

#### --distractors--

```js
func greet() {
  console.log("Hello there!");
}
```

---

```js
defineFunction greet() {
  console.log("Hello there!");
}
```

---

```js
def greet() {
  console.log("Hello there!");
}
```

```js
  console.log("Hello there!");
}
```
