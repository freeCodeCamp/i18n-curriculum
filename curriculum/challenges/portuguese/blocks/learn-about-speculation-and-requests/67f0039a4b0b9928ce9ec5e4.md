# --explanation--

`Report of` é usado para descrever o assunto de um relatório.A preposição correta após `report` neste caso é `of` porque indica o que foi reportado.Por exemplo:

`We received a report of an accident.` - O relatório era sobre um acidente.

`Report` também pode ser usado com `to` quando se fala sobre para quem alguém dá atualizações ou respostas.Por exemplo:

`I report to the team leader every day.` - Você dá atualizações para seu chefe todos os dias.
