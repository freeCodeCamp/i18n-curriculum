# --description--

많은 분들이 컴퓨터를 오랫동안 사용해왔습니다. 하지만 컴퓨터에 로그인하는 가장 안전하고 보안이 강한 방법에 대해 생각해본 적은 없을 수 있습니다.

Mac이든 PC든 컴퓨터에 안전하게 로그인하는 방법은 여러 가지가 있습니다.

가장 먼저 해야 할 일은 컴퓨터에 비밀번호가 설정되어 있는지 확인하는 것입니다.

Windows 사용자는 시작 메뉴를 열고 설정을 선택하세요. 거기서 계정으로 이동한 다음 로그인 옵션으로 들어가세요. 그 후 컴퓨터 비밀번호를 설정할 수 있습니다.

*참고: 문제가 발생하거나 다른 버전의 Windows를 사용 중이라면, 해당 버전에 맞춘 자세한 안내를 위해 공식 Microsoft 문서나 지원 자료를 참조하세요.*

비밀번호를 만들 때는 길고 복잡한 비밀번호를 설정하세요. 문자, 숫자, 특수 문자를 조합해 보안을 강화할 수 있습니다.

`12345` 또는 `password` 같은 쉬운 비밀번호를 선택하는 것은 좋지 않습니다. 이런 비밀번호는 쉽게 추측할 수 있고 해킹당하기 쉽습니다.

또한 이름, 생일, 주소 같은 개인 정보를 기반으로 한 비밀번호도 사용하지 마세요. 해커가 쉽게 추측할 수 있습니다.

비밀번호를 만들 때 이중 인증(2FA)을 설정하는 것도 좋은 방법입니다. 2FA는 추가 보안 계층 역할을 하며, 계정에 접근하려는 사람이 실제 권한이 있는 사용자임을 확인하기 위해 두 번째 인증 수단을 요구합니다.

Mac 사용자라면 Apple 메뉴를 클릭한 후 시스템 설정으로 이동하세요. 거기서 사용자 및 그룹으로 들어가 컴퓨터 비밀번호를 설정할 수 있습니다.

일부 Mac 컴퓨터에는 터치 ID 기능이 있어 일반 비밀번호보다 더 안전하다고 여겨집니다. 이 기능은 지문을 사용해 컴퓨터에 로그인할 수 있게 해줍니다.

Windows 사용자는 Windows Hello 기능을 통해 더 안전하게 로그인할 수 있습니다. 이 기능은 얼굴 인식이나 지문 같은 생체 인식 방법을 사용해 인증하며, 전통적인 비밀번호 대신 사용할 수 있습니다.

이 수업을 마친 후에는 이러한 추가 보안 기능들을 살펴보고 컴퓨터에 설정해 컴퓨터를 안전하게 보호하세요.
