# --description--

Você pode usar diferentes frases ou palavras para falar sobre raciocínio, tais como:

`Justifying` significa dar razões ou explicações para suas opiniões ou ações.Um exemplo é, `I chose to study at the library instead of at home.`Justificando minha decisão, acho que é mais silencioso e menos distrativo, o que me ajuda a focar melhor no meu trabalho.`

`Because` é usado para explicar a razão de algo.Por exemplo, `We're updating the website because it's outdated.`

`That is why` é outra forma de dar uma razão.É como dizer, `The server is down.É por isso que o site não está acessível.`

`Given` é usado para introduzir um fato que é a base para a decisão.Por exemplo, `Given the high demand, we need more people on the team.` ou `We need more people on the team given the high demand.`
