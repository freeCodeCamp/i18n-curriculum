# --description--

字 `might` 是用來表達可能性或潛在性的情態動詞。片語 `be able to` 顯示能力。

「`might be able to`」表示有可能達成某事，但並不確定。 

例如，`She might be able to finish the work by tomorrow.` 這表示她有可能在明天之前完成工作，但並不保證。
