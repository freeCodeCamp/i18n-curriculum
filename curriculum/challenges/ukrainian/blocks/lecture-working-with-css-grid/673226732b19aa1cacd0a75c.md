# --interactive--

У попередньому уроці ви ознайомилися з CSS сіткою, яку можна використовувати для створення складних і плавних макетів на ваших вебсторінках. У цьому уроці ми розглянемо, як створювати гнучкі макети сітки за допомогою одиниці `fr`.

Почнемо з цієї HTML розмітки, яка буде представляти наш контейнер сітки:

```html
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

У CSS ми встановлюємо властивість display зі значенням grid для контейнера.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 25% 25% 25% 25%;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

`grid-template-columns` використовується для встановлення розміру кожного стовпця. У цьому випадку розмір кожного стовпця буде `25%` від контейнера. Потім властивість gap використовується для створення пробілу навколо кожного стовпця.

До цього часу ми використовували відсотки для розміру стовпців, але також можемо використовувати одиницю `fr`.

Одиниця `fr` — це дробова одиниця, яка представляє частку простору контейнера сітки.

Ось як виглядатиме код, коли його перероблять для використання одиниць `fr` замість відсотків.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

Кожен стовпець займатиме одну частку доступного простору. Оскільки є чотири стовпці, кожен матиме рівну частку простору, доступного в контейнері сітки.

Коли ви почнете створювати свої макети сітки, ви помітите, що все частіше хочете використовувати одиниці `fr`, тому що вони забезпечують гнучкий, пропорційний спосіб розподілу простору, дозволяючи створювати адаптивні макети, які підлаштовуються під різні розміри екранів без необхідності вручну регулювати піксельні значення.
