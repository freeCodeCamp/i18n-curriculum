# --explanation--

`Mentioned`は、何かに簡単に言及したり話題にすることを意味します。会話の中で誰かが話題を取り上げるときによく使われます。例えば：

`She mentioned the meeting time in her email.` – 彼女は会議の時間に言及しました。

`Polymorphism`はプログラミングの概念です。1つの関数、メソッド、または演算子が異なる種類のデータやオブジェクトで使えることを意味します。例えば：

`In object-oriented programming, polymorphism allows a single method to behave differently based on the object that calls it.` - ここで、`polymorphism`は、どのオブジェクトが使うかによって異なる動作をするメソッドを指します。例えば、同じメソッド`draw()`が`Circle`オブジェクトや`Rectangle`オブジェクトから呼び出されたときに異なる動作をする場合があります。
