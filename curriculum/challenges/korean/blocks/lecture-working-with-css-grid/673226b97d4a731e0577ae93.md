# --interactive--

`grid-template-areas` 속성은 이름이 지정된 레이블을 사용해 시각적 그리드 레이아웃을 설계할 수 있게 해줍니다.

그다음 grid-area 속성을 사용해 특정 그리드 항목에 레이블을 할당합니다. 다시 말해, 이 이름이 지정된 레이블을 "그리드 영역 이름"이라고도 합니다.

`grid-template-areas` 속성의 기본 구문은 다음과 같습니다:

```css
grid-template-areas:
 'header header header'
 'left-sidebar main right-sidebar'
 'footer footer footer';
```

기본 구문에서 기억할 점은 다음과 같습니다:

- `header`와 `main` 같은 값은 그리드 영역의 이름입니다.

- 문자열 내 공백으로 구분된 각 값은 열 하나를 나타냅니다.

- 각 문자열은 그리드의 한 행을 나타냅니다.

따라서 구문에서는 3x3 그리드 컨테이너가 있습니다.

템플릿을 정의한 후에는 `grid-area` 속성과 이름이 지정된 레이블(또는 영역)을 값으로 사용해 특정 요소가 그리드의 해당 영역에 속한다는 것을 CSS에 알려야 합니다.

`grid-area` 속성은 그리드 항목을 `grid-template-areas`에서 정의한 이름이 지정된 영역과 연결합니다.

`grid-template-areas` 속성의 기능을 보여주는 인기 있는 방법은 고전적인 홀리 그레일 레이아웃을 만드는 것입니다.

모른다면, 홀리 그레일 레이아웃은 헤더, 푸터, 두 개의 사이드바, 그리고 메인 콘텐츠 영역이 있는 웹 디자인 패턴입니다. 메인 콘텐츠가 우선순위를 가지면서 사이드바와 다른 섹션들이 반응형으로 레이아웃 내에서 조정되도록 합니다.

홀리 그레일 레이아웃을 구현하는 다양한 방법이 있지만, `grid-template-areas`와 `grid-area` 속성을 사용하는 것이 가장 간단한 방법입니다.

홀리 그레일 레이아웃 예시는 다음과 같습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="header">
    <h2>Header</h2>
  </div>
  <div class="sidebar-left">
    <h2>Left Sidebar</h2>
  </div>
  <div class="main"><h2>Main Content</h2></div>
  <div class="sidebar-right">
    <h2>Right Sidebar</h2>
  </div>
  <div class="footer">
    <h2>Footer</h2>
  </div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-areas:
    'header header header'
    'sidebar-left main sidebar-right'
    'footer footer footer';
  grid-gap: 10px;
  background-color: #2196F3;
  padding: 10px;
}

.header {
  grid-area: header;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.sidebar-left {
  grid-area: sidebar-left;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.main {
  grid-area: main;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.sidebar-right {
  grid-area: sidebar-right;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.footer {
  grid-area: footer;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}
```

:::

`grid-template-areas`과 `grid-area` 속성은 서로 독립적으로 사용할 수 있다는 점을 참고하세요.

`grid-template-areas` 속성은 그리드 컨테이너 내에서 이름이 지정된 그리드 영역을 매핑하여 시각적 레이아웃을 정의하는 데 특별히 사용됩니다.

반면에 `grid-area` 속성은 개별 그리드 항목을 배치하는 데 사용되며, 행과 열 위치를 지정하거나 `grid-template-areas` 속성으로 정의된 이름이 지정된 영역을 참조합니다.
