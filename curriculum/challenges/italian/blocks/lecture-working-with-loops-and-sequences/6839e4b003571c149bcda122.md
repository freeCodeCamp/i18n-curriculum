# --description--

Nelle lezioni precedenti hai imparato a lavorare con il loop `for`, che serve a ripetere un blocco di codice un numero prestabilito di volte. Ecco un esempio di utilizzo di un loop `for` per stampare ogni lingua dalla lista `languages` nella console:

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for language in languages:
    print(language)
```

Ma cosa succede se vuoi tenere traccia dell’indice di ogni elemento? Una possibilità è creare una variabile `index` e incrementarla di `1` a ogni iterazione del loop, così:

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

index = 0

for language in languages:
    print(f'Index {index} and language {language}')
    index += 1
```

Anche se funziona, un modo più semplice per farlo è usare la funzione `enumerate()`. La funzione `enumerate()` tiene traccia dell’indice per un iterabile e restituisce un oggetto enumerate.

Se passiamo la lista `languages` alla funzione `enumerate()` e convertiamo il valore restituito in una lista con la funzione `list()`, otteniamo questo:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

list(enumerate(languages))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

Ogni voce nell’oggetto enumerate (ora una lista) è una tupla contenente un conteggio, seguito da un valore dell’iterabile passato alla funzione `enumerate()`.

Ora, rifattorizziamo l’esempio precedente per usare la funzione `enumerate()`:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')
```

Assembliamo il conteggio e il valore di ogni tupla nell’oggetto enumerate in variabili chiamate rispettivamente `index` e `language`. Infine, entrambe le variabili sono usate in una f-string che viene stampata nella console a ogni iterazione del loop.

```md
Index 0 and language Spanish
Index 1 and language English
Index 2 and language Russian
Index 3 and language Chinese
```

Questo elimina la necessità di creare e aggiornare manualmente una variabile `index`.

La funzione `enumerate()` accetta anche un argomento `start` opzionale che specifica il valore iniziale del conteggio. Se questo argomento viene omesso, il conteggio parte da `0`. Ecco un esempio di utilizzo dell’argomento opzionale `start`:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages, 1):
    print(f'Index {index} and language {language}')
```

Ecco come apparirà il risultato nella console:

```md
Index 1 and language Spanish
Index 2 and language English
Index 3 and language Russian
Index 4 and language Chinese
```

Finora abbiamo iterato solo su una lista. Ma cosa succede se devi iterare su più iterabili in parallelo? Puoi usare la funzione `zip()` per questo, che combina le liste in coppie di elementi e restituisce un iteratore di tuple.

Se passiamo una lista di `developers` e `ids` alla funzione `zip()` e convertiamo il valore restituito in una lista con la funzione `list()`, ecco cosa otteniamo:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

list(zip(developers, ids))
# [('Naomi', 1), ('Dario', 2), ('Jessica', 3), ('Tom', 4)]
```

Ecco un esempio di utilizzo della funzione `zip()` con un loop `for` per iterare su `developers` e `ids`:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')
```

In questo esempio, `zip()` combina le due liste in coppie di elementi e restituisce un iteratore di tuple. Il loop `for` poi assembla ogni tupla in `name` e `id`. Infine, per ogni istruzione print, stampiamo rispettivamente ogni `name` e `id` dalle liste `ids` e `developers`. Ecco come appare il risultato nella console:

```md
Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
```

Le funzioni `enumerate()` e `zip()` sono molto potenti e, combinate con i loop, possono rendere il tuo codice molto più conciso.
