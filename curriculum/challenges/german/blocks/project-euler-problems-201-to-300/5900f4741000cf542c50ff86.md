# --description--

Betrachten Sie die Zahl 6. Die Teiler von 6 sind: 1, 2, 3 und 6.

Jede Zahl von 1 bis einschließlich 6 kann als Summe verschiedener Teiler von 6 geschrieben werden:

$1 = 1$, $2 = 2$, $3 = 1 + 2$, $4 = 1 + 3$, $5 = 2 + 3$, $6 = 6$.

Eine Zahl $n$ heißt praktische Zahl, wenn jede Zahl von 1 bis einschließlich $n$ als Summe verschiedener Teiler von $n$ ausgedrückt werden kann.

Ein Paar aufeinanderfolgender Primzahlen mit einem Abstand von sechs wird sexy Paar genannt (da „sex“ das lateinische Wort für „sechs“ ist). Das erste sexy Paar ist (23, 29).

Gelegentlich finden wir ein Triple-Paar, das bedeutet drei aufeinanderfolgende sexy Primzahlpaare, sodass das zweite Element jedes Paares das erste Element des nächsten Paares ist.

Wir nennen eine Zahl $n$ ein Ingenieursparadies, wenn:

- ($n - 9$, $n - 3$), ($n - 3$, $n + 3$), ($n + 3$, $n + 9$) ein Triple-Paar bilden, und

- die Zahlen $n - 8$, $n - 4$, $n$, $n + 4$ und $n + 8$ alle praktisch sind,

finden Sie die Summe der ersten vier Ingenieursparadiese.
