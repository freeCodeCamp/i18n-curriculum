# --description--

`Seem to be` 用於當你不確定某件事並且想要猜測或提出一種可能性時。 例如，如果你說 `It seems to be working`，你是在猜測某件事正在運作，但你並不完全確定。

範例：

肯定：`It seems to be working fine.`

負面：`It doesn't seem to be a major issue.`

問題：`Does it seem to be related to the network?`
