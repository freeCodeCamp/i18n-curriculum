# --description--

الدوال السهمية هي طريقة أكثر حداثة لكتابة الدوال في JavaScript. توفر صيغة نحوية أكثر اختصارًا مقارنة بتعابير الدوال التقليدية. تُعرّف الدوال السهمية باستخدام الصيغة `=>`.

```js
const add = (a, b) => {
  return a + b;
}
```

في المثال أعلاه، تأخذ الدالة السهمية `add` معلمتين `a` و `b` وتُرجع مجموعهما. تُستخدم الكلمة المفتاحية `return` لإرجاع نتيجة عملية الجمع.

إذا كانت الدالة السهمية تحتوي على تعبير واحد فقط، يمكن حذف الأقواس المتعرجة `{}` والكلمة المفتاحية `return`. سيتم إرجاع التعبير ضمنيًا.

```js
const add = (a, b) => a + b;
```

في المثال أعلاه، تأخذ الدالة السهمية `add` معلمتين `a` و `b` وتُرجع مجموعهما. تم حذف الكلمة المفتاحية `return` والأقواس المتعرجة `{}`، وتُرجع نتيجة عملية الجمع ضمنيًا.
