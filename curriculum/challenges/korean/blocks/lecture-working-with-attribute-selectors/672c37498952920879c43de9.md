# --interactive--

다국어 웹사이트를 만들거나 사용자 정의 데이터 속성을 다룰 때, 포함된 언어나 특정 데이터 값에 따라 요소를 스타일링해야 할 때가 많습니다.

`lang`와 `data-lang` 속성은 이러한 목적에 자주 사용되며, CSS의 속성 선택자를 사용하면 이러한 속성을 기반으로 효과적으로 스타일을 적용할 수 있습니다.

`lang` 속성은 HTML에서 요소 내 콘텐츠의 언어를 지정하는 데 사용됩니다. 특히 다국어 웹사이트에서는 작성된 언어에 따라 요소를 다르게 스타일링하고 싶을 수 있습니다.

다음은 특정 `lang` 속성을 가진 요소를 선택자 속성으로 타겟팅하는 방법의 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">This is an English paragraph.</p>
<p lang="fr">Ceci est un paragraphe en français.</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

이 CSS 규칙은 `lang` 속성이 영어(`en`)로 설정된 모든 문단 요소에 이탤릭체 스타일을 적용합니다. 이는 여러 언어가 포함된 문서에서 영어 텍스트를 강조하는 데 유용할 수 있습니다.

마찬가지로, `data-lang` 속성을 가진 요소를 선택자 속성으로 타겟팅할 수도 있습니다.

`data-lang` 같은 사용자 정의 데이터 속성은 특정 텍스트 섹션 내에서 사용된 언어를 지정하는 등 요소에 추가 정보를 저장하는 데 자주 사용됩니다.

다음은 `data-lang` 속성을 기반으로 요소를 타겟팅하고 스타일링하는 CSS 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p data-lang="fr">Ceci est un paragraphe en français.</p>
<p data-lang="en">This is a paragraph in English.</p>
```

```css
p[data-lang="fr"] {
  color: blue;
}
```

:::

이 경우, `p` 요소 중 `data-lang` 속성이 프랑스어(`fr`)로 설정된 모든 요소는 텍스트 색상이 파란색으로 변경됩니다. 이를 통해 `data-lang` 속성에 저장된 언어 정보를 기반으로 콘텐츠 섹션을 빠르게 식별하고 스타일링할 수 있습니다.

이와 같은 속성 선택자는 HTML에 임베드된 메타데이터를 기반으로 조건부 스타일링을 적용하는 강력한 방법을 제공하여 웹 페이지를 더 동적이고 상황에 맞게 만듭니다.
