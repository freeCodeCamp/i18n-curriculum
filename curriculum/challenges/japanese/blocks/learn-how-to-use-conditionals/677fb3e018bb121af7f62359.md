# --explanation--

`First Conditional` は未来の現実的かつ可能な状況を説明するために使われます。これは `if` + `Present Simple`（条件節）と `will` + 動詞の原形（結果節）で構成されます。例えば：

`If you debug the code, the app will run smoothly.` - コードを修正すれば、アプリが正しく動作します。

正しい文では、条件が `If we don't back up regularly`、結果が `we could lose important information` です。これは現実的な可能性を示しています。
