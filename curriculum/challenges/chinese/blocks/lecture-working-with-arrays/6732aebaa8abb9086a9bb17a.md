# --interactive--

在上一课中，你首次接触了如何操作数组以及访问数组中的不同元素。这里提醒你如何访问数组中的第二个元素：

:::interactive_editor

```js
const fruits = ["apple", "banana", "cherry"];

console.log(fruits[1]); // "banana"
```

:::

由于数组是从零开始索引的，第一个元素的索引是 `0`，第二个元素的索引是 `1`，依此类推。 需要注意的是，如果你尝试访问数组中不存在的索引，JavaScript 会返回 `undefined`。 

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
console.log(fruits[3]); // undefined
```

:::

在这个示例中，`fruits` 数组并没有索引 `3`。 因此，日志输出将显示 `undefined`。 现在，让我们看看如何更新数组中的元素。 你可以通过为特定索引重新赋值来更新元素。 

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
fruits[1] = "blueberry";
console.log(fruits); // ["apple", "blueberry", "cherry"]
```

:::

在这个例子中，我们把索引 `1` 处的 `banana` 替换成了 `blueberry`。 只要你知道索引值，这种方法允许你修改数组中的任意元素。 你还可以通过给不存在的索引赋值来向数组中添加新元素：

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
fruits[3] = "date";
console.log(fruits); // ["apple", "blueberry", "cherry", "date"]
```

:::

不过，这样做时要小心谨慎。 如果你把值赋给一个远大于当前数组长度的索引，就会在中间创建 undefined 元素，这可能会导致意外的行为。 随着你继续学习 JavaScript，你会发现这些访问和更新数组元素的方法是许多编程任务的基础。 无论是构建一个简单的待办事项列表，还是处理复杂的数据结构，这些技能都非常有用。
