# --description--

두 개의 돌 더미와 두 명의 플레이어가 참여하는 게임입니다. 각 플레이어의 차례에, 플레이어는 더 큰 더미에서 돌을 제거할 수 있습니다. 제거하는 돌의 수는 더 작은 더미에 있는 돌 수의 양의 배수여야 합니다.

예를 들어, (6,14)라는 순서쌍이 더 작은 더미에 6개, 더 큰 더미에 14개의 돌이 있는 상태를 나타낸다면, 첫 번째 플레이어는 더 큰 더미에서 6개 또는 12개의 돌을 제거할 수 있습니다.

한 더미에서 모든 돌을 가져가는 플레이어가 게임에서 승리합니다.

첫 번째 플레이어가 반드시 이길 수 있는 상태를 승리 상태라고 합니다. 예를 들어, (1,5), (2,6), (3,12)는 첫 번째 플레이어가 두 번째 더미의 모든 돌을 즉시 제거할 수 있기 때문에 승리 상태입니다.

두 번째 플레이어가 어떤 첫 번째 플레이어의 행동에도 반드시 이길 수 있는 상태를 패배 상태라고 합니다. 예를 들어, (2,3)과 (3,4)는 어떤 합법적인 수를 두어도 두 번째 플레이어에게 승리 상태를 남기므로 패배 상태입니다.

$S(N)$을 $0 < x_i < y_i \le N$인 모든 패배 상태 ($x_i$, $y_i$)에 대해 ($x_i + y_i$)의 합으로 정의합니다. $S(10) = 211$이고 $S({10}^4) = 230\,312\,207\,313$임을 확인할 수 있습니다.

$S({10}^{16}) \bmod 7^{10}$의 값을 구하세요.
