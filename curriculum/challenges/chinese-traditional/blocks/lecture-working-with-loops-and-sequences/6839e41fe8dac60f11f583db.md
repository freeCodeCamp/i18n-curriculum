# --description--

`range()` 函式用於產生一個整數序列。以下是 `range()` 函式的基本語法：

```py
range(start, stop, step)
```

必要的 `stop` 引數是一個整數，用來表述正在產生的數字序列的終點。以下是使用 `range()` 函式的範例：

```py
for num in range(3):
    print(num)
```

以下程式碼會產生介於 `0` 和 `2` 之間的數字序列。整數 `3` 不包含在內，因為 `stop` 引數是不包含結尾的。

如果沒有指定 `start` 引數，則預設值為 `0`。否則，你可以使用選用的 `start` 引數，從非 `0` 的整數開始整數序列。以下是產生介於 `1` 和 `4` 之間整數序列的範例：

```py
for num in range(1, 5):
    print(num)
```

預設情況下，整數序列會以 `1` 遞增。但如果你想更改該預設值，則可以使用選用的 `step` 引數。以下是產生介於 `2` 和 `10` 之間偶數序列的範例：

```py
for num in range(2, 11, 2):
    print(num)
```

如前所述，`range()` 函式只有一個必須的引數。如果你沒有為 `range()` 提供任何引數，則會得到 `TypeError`：

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: range expected at least 1 argument, got 0
```

重要的是要注意，`range()` 函式只接受整數作為引數，而非浮點數。請記得浮點數是帶有小數點的數字，例如 `3.4`。如果你嘗試將浮點數作為引數傳入，將會得到 `TypeError`：

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

如果你想產生一個遞減順序的整數序列，那麼你可以為 `step` 引數使用負整數，如下所示：

```py
for num in range(40, 0, -10):
    print(num)
```

以下程式碼依序在主控台列印數字 `40`、`30`、`20` 和 `10`。

你還可以使用 `range()` 函式搭配 `list` 建構式來創建整數列表。`list` 建構式用於將可迭代物件轉換成列表。以下是生成介於 `2` 和 `10` 之間偶數列表的範例：

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

`range()` 函式（程式）是在 Python 中產生整數序列的非常方便的方法。一旦你掌握了它，你很可能會在你的 Python 程式中經常使用它。
