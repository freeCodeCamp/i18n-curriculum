# --instructions--

In dieser Aufgabe sagen Sie Gesundheitskosten mit einem Regressionsalgorithmus voraus.

Sie erhalten einen Datensatz mit Informationen über verschiedene Personen, einschließlich ihrer Gesundheitskosten. Verwenden Sie die Daten, um Gesundheitskosten basierend auf neuen Daten vorherzusagen.

Die ersten beiden Zellen dieses Notebooks importieren Bibliotheken und die Daten.

Stellen Sie sicher, dass Sie kategoriale Daten in Zahlen umwandeln. Verwenden Sie 80 % der Daten als `train_dataset` und 20 % der Daten als `test_dataset`.

`pop` der Spalte „expenses“ aus diesen Datensätzen, um neue Datensätze namens `train_labels` und `test_labels` zu erstellen. Verwenden Sie diese Bezeichnungen beim Trainieren Ihres Modells.

Erstellen Sie ein Modell und trainieren Sie es mit den `train_dataset`. Führen Sie die letzte Zelle in diesem Notebook aus, um Ihr Modell zu überprüfen. Die letzte Zelle verwendet die ungesehenen `test_dataset`, um zu prüfen, wie gut das Modell generalisiert.

Um die Aufgabe zu bestehen, muss `model.evaluate` einen mittleren absoluten Fehler von unter 3500 zurückgeben. Das bedeutet, dass es die Gesundheitskosten korrekt innerhalb von 3500 $ vorhersagt.

Die letzte Zelle wird außerdem die Ausgaben mit den `test_dataset` vorhersagen und die Ergebnisse grafisch darstellen.
