# --explanation--

`Simple Past`の肯定文は特定のルールに従います。  

動詞`to be`の場合、形は`was`（単数）と`were`（複数）です。これらの形は助動詞なしで使われます。例えば： 

- `The project was successful.` - これはプロジェクトが良い結果をもたらしたことを意味します。 

- `They were happy with the results.` - これはプロジェクトの結果に対する`they`の気持ちを表しています。 

規則動詞の過去形は、動詞の基本形に`-ed`を付けて作られます。例えば： 

- `I worked on the project yesterday.` - 動詞`-ed`の基本形に`work`を付けて、過去の一瞬の出来事を表します。
