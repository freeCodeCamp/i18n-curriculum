# --interactive--

CSS 애니메이션은 JavaScript나 복잡한 프로그래밍 없이도 웹 페이지에서 동적이고 시각적으로 매력적인 효과를 만들 수 있게 해줍니다. 지정된 기간 동안 요소가 서로 다른 스타일 사이를 부드럽게 전환하도록 하는 방법을 제공합니다.

기본적으로 CSS 애니메이션은 두 가지 주요 요소로 구성됩니다: `@keyframes` 규칙과 animation 속성입니다.

`@keyframes` 규칙은 애니메이션의 단계와 스타일을 정의합니다. 애니메이션 진행 중 요소가 가져야 할 스타일을 각 시점별로 지정합니다.

예를 들어 보겠습니다:

```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(0);
  }
}
```

이 `@keyframes` 규칙, 이름은 `slide-in`이며, 요소를 왼쪽에서 오른쪽으로 이동시키는 애니메이션을 정의합니다. 백분율은 애니메이션 진행 정도를 나타내며 `0%`가 시작, `100%`가 끝을 의미합니다.

`translateX` 함수는 `@keyframes` 애니메이션에서 요소가 화면에 나타날 때 수평 위치를 제어합니다.

이 애니메이션을 요소에 적용하려면 `animation` 속성을 사용합니다. 이 예제는 애니메이션을 무한 반복하도록 설정해 실제 동작을 볼 수 있습니다(미리 보려면 인터랙티브 편집기를 활성화하세요):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="sliding-element">Hello, I slide in!</div>
```

```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(0);
  }
}

.sliding-element {
  animation: slide-in 2s ease-out infinite;
}
```

:::

이 코드는 `slide-in` 애니메이션을 요소에 2초 동안, `ease-out` 타이밍 함수와 함께 적용합니다.

`animation` 속성은 실제로 여러 개별 속성을 축약한 것입니다:

`animation-name`는 사용할 `@keyframes` 규칙을 지정합니다.

`animation-duration`는 애니메이션이 완료되는 데 걸리는 시간을 설정합니다.

`animation-timing-function`는 애니메이션이 시간에 따라 어떻게 진행되는지 정의합니다 — 예를 들어 `ease`, `linear`, `ease-in-out` 등이 있습니다.

`animation-delay`는 애니메이션 시작 전에 지연 시간을 지정합니다.

`animation-iteration-count`는 애니메이션이 몇 번 반복될지 설정합니다.

`animation-direction`는 애니메이션이 앞으로, 뒤로, 또는 교대로 재생될지 결정합니다.

`animation-fill-mode`는 애니메이션 전후에 요소가 어떻게 스타일링될지 지정합니다.

`animation-play-state`는 애니메이션을 일시 중지하거나 다시 시작할 수 있게 합니다.

더 세밀한 제어를 위해 이 속성들을 개별적으로 사용할 수도 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="complex-animation">Watch my colors change!</div>
```

```css
.complex-animation {
  animation-name: color-change;
  animation-duration: 3s;
  animation-timing-function: linear;
  animation-delay: 1s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}

@keyframes color-change {
  0% {
    background-color: red;
  }
  50% {
    background-color: blue;
  }
  100% {
    background-color: green;
  }
}
```

:::

이 코드는 요소의 배경색이 빨강, 파랑, 초록 사이를 계속해서 전환하는 애니메이션을 만듭니다.

CSS 애니메이션은 요소 위에 마우스를 올리는 등 다양한 이벤트로 트리거할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Hover over me!</button>
```

```css
.button {
  background-color: blue;
  transition: background-color 0.3s;
}

.button:hover {
  background-color: red;
}
```

:::

이 예제는 기본 효과에 더 간단한 `transition` 속성을 사용하지만, CSS로 인터랙티브하고 애니메이션이 적용된 요소를 만드는 방법을 보여줍니다.

CSS 애니메이션은 강력하지만 적절히 사용해야 한다는 점을 기억하세요. 애니메이션을 과도하게 사용하면 성능 저하가 발생할 수 있고, 특정 접근성 요구가 있는 사용자에게 방해가 될 수 있습니다. 움직임을 줄이거나 끌 수 있는 옵션을 제공하는 것을 항상 고려하세요.

CSS 애니메이션은 JavaScript에 의존하지 않고도 매력적이고 인터랙티브한 웹 경험을 만드는 방법을 제공합니다. `@keyframes`와 `animation` 속성의 원칙을 이해하면 성능과 접근성을 고려한 웹 디자인에 생명을 불어넣을 수 있습니다.
