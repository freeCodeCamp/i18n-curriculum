# --description--

Il CSS interno (o CSS incorporato) consiste nell'aggiungere il CSS direttamente all'interno del file HTML invece di creare un file completamente separato. Con il metodo interno, inserisci tutte le regole all'interno di una coppia di tag `<style>` di apertura e chiusura, che vengono poi posizionati all'interno dei tag `<head>` di apertura e chiusura del tuo file HTML. Poiché gli stili vengono inseriti direttamente all'interno dei tag `<head>`, non serve più un elemento `<link>` che invece richiede il metodo esterno.

A parte queste differenze, la sintassi è esattamente la stessa del metodo esterno (selettore, parentesi graffe, dichiarazioni):

```html
<head>
  <style>
    div {
      color: white;
      background-color: black;
    }

    p {
      color: red;
    }
  </style>
</head>
<body>...</body>
```

Questo metodo può essere utile per definire stili unici a una singola pagina di un sito web, ma non mantiene le cose separate come il metodo esterno e, a seconda di quante regole e dichiarazioni ci sono, può far diventare il file HTML piuttosto grande.

# --questions--    

## --text--

Qual è una differenza tra i metodi CSS interno ed esterno?

## --answers--

Il metodo esterno posiziona le regole CSS in un file separato, mentre il metodo interno posiziona le regole CSS all'interno del file HTML stesso.

---

Il metodo interno mantiene il CSS separato dall'HTML, mentre il metodo esterno incorpora il CSS direttamente nell'HTML.

---

Il metodo interno usa l'elemento `<link>` per collegare il CSS all'HTML, mentre il metodo esterno incorpora il CSS direttamente nell'HTML.
