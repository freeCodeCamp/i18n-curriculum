# --explanation--

Lisa는 `Second Conditional`를 사용해 가상의 상황을 이야기하고 있습니다.

`Second Conditional`는 현재나 미래에 사실이 아니거나 가능성이 낮은 상황을 말할 때 사용합니다. 구조는 `if` + `Present Simple` 또는 조건절에 `Past Continuous`와 결과절에 `might / would / could` + 기본 동사 형태를 따릅니다. 예를 들어:

- `If I had more time, I would learn a new programming language.` - 시간이 더 많다면 배우게 될 가상의 상황을 설명합니다.

- `If the server were faster, it could handle more users.` - 서버가 지금은 빠르지 않지만, 만약 그렇다면 더 많은 사용자를 지원할 수 있다는 뜻입니다.
