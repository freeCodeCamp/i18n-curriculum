# --description--

すべての三角形には3つの頂点を通る外接円があります。外接円の半径も整数であるような整数辺の三角形をすべて考えます。

$S(n)$ を、外接円の半径が $n$ 以下であるすべてのそのような三角形の外接円の半径の和とします。

$S(100) = 4\,950$、$S(1\,200) = 1\,653\,605$ です。

$S({10}^7)$ を求めてください。
