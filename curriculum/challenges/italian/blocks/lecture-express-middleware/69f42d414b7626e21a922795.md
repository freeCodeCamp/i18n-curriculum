# --description--

Express offre una serie di funzioni middleware integrate che aiutano a svolgere attività comuni, come servire file statici, analizzare i corpi delle richieste JSON e altro ancora. Puoi anche usare middleware di terze parti per aggiungere funzionalità.

**Ecco alcuni esempi di middleware integrato**:

1. `express.json()`: Un middleware integrato che analizza i payload JSON in arrivo.

```js
app.use(express.json())
```

1. `express.static()`: Un middleware per servire file statici come immagini, CSS e file JavaScript.

```js
app.use(express.static('public'))
```

1. `express.urlencoded()`: Un middleware che analizza le richieste in arrivo con payload codificati in URL, il formato tipicamente inviato dai moduli HTML. Rende i dati analizzati disponibili su `req.body`.

```js
app.use(express.urlencoded({ extended: true }))
```

**Ora, ecco alcuni esempi di middleware di terze parti:**

1. `cors`: Un middleware di terze parti molto usato per abilitare il Cross-Origin Resource Sharing.

```bash
npm install cors
```

```js
const cors = require('cors')
app.use(cors())
```

1. `morgan`: Un middleware per il logging delle richieste HTTP.

```bash
npm install morgan
```

```js
const morgan = require('morgan')
app.use(morgan('tiny'))
```

Ora, vediamo un esempio di utilizzo sia di middleware integrato sia di terze parti:

```js
const express = require('express')
const cors = require('cors')
const morgan = require('morgan')
const app = express()

app.use(express.json()) // Built-in middleware to parse JSON
app.use(cors())       // Third-party middleware for CORS
app.use(morgan('tiny')) // Third-party middleware for logging

app.get('/', (req, res) => {
  res.send('Hello, World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* Il middleware integrato `express.json()` analizza i payload JSON in arrivo.
* Il middleware di terze parti `cors` permette le richieste cross-origin, e `morgan` registra le richieste HTTP in arrivo.

È importante notare che le funzioni middleware vengono aggiunte nell’ordine in cui sono chiamate con `app.use()`.

Express fornisce middleware integrato essenziale per gestire attività comuni come l’analisi JSON e il servizio di file statici. Puoi anche estendere le funzionalità con middleware di terze parti per capacità aggiuntive.
