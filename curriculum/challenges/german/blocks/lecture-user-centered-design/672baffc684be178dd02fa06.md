# --interactive--

**HINWEIS**: Einige der interaktiven Beispiele verwenden möglicherweise CSS, das Sie noch nicht gelernt haben. Machen Sie sich keine Sorgen, wenn Sie nicht versuchen, den gesamten Code zu verstehen. Das Ziel der Beispiele ist es, Ihnen Vorschauen für diese Designkonzepte zu zeigen, damit Sie besser verstehen, wie die Dinge funktionieren. Um die Vorschauen zu sehen, müssen Sie den interaktiven Editor aktivieren.

Karten-Komponenten sind sehr häufig in E-Commerce-, Social-Media- und Nachrichtenseiten anzutreffen. Sie werden verwendet, um Informationen strukturiert darzustellen. Wenn Sie Ihre Karten gestalten, ist es wichtig, bewährte Vorgehensweisen zu verstehen, damit Ihre Nutzer die Informationen, die Sie vermitteln möchten, leicht erfassen können.

Die erste Überlegung beim Kartendesign sollte Einfachheit sein. Sie möchten nicht, dass Ihre Karten visuell überladen sind oder zu viele Informationen anzeigen. Wenn ein Kartendesign zum Beispiel visuell überladen ist, gibt es zu viele Informationen, die der Nutzer auf einmal verarbeiten muss.

Hier ist ein Beispiel für ein überladenes Kartendesign:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2><strong><em>Product Name: Ultimate Edition</em></strong></h2>
  <p>
    This is a great product that you will love! It has many features and benefits that will make your life easier and more enjoyable. Whether you're at home or on the go, this product is perfect for all situations. Don’t miss out on this incredible offer, buy now and experience the difference!
  </p>
  <button>Buy Now</button>
  <button>Learn More</button>
  <button>Add to Wishlist</button>
  <p><strong>Only 3 left in stock!</strong></p>
  <p><em>Rated 4.8 stars by 2,391 customers</em></p>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}
.card p {
  font-size: 1em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

Weniger Informationen und guter Abstand zwischen den Elementen auf der Karte erleichtern es dem Nutzer, die Informationen zu verarbeiten, und ermöglichen mehrere Karten auf der Seite.

Hier ist ein Beispiel für ein einfaches Kartendesign:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2>Product Name</h2>
  <button>Buy Now</button>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 200px;
  text-align: center;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

Ein weiterer Punkt ist, wo der Nutzer auf der Karte klicken kann. Manche Kartendesigns haben eine einzelne Schaltfläche, wodurch klar wird, wo der Nutzer klicken kann. Andere Kartendesigns machen die gesamte Karte anklickbar. Wenn der Nutzer mit der Maus über einen Teil der Karte fährt, ändert die Karte ihre Farbe oder erhält einen Schatteneffekt, um anzuzeigen, dass die Karte anklickbar ist. Welches Design Sie auch wählen, es muss auf Ihrer gesamten Website konsistent sein und für den Nutzer leicht verständlich.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <a href="#">
    <img src="https://placehold.co/150x150" alt="Example Product Image">
    <h2>Product Name</h2>
    <button>Buy Now</button>
  </a>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
  cursor: pointer;
}

.card:hover {
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
}

.card a {
  text-decoration: none;
  color: inherit;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

Eine weitere Überlegung ist die Verwendung von Medien auf Ihren Karten. Die Wahl hochwertiger Medien kann das Nutzererlebnis erheblich verbessern. Wenn Sie zum Beispiel Bilder oder Videos für eine Produktkarte verwenden, gilt: Je höher die Qualität, desto mehr wird der Nutzer an diesem Produkt interessiert sein. Wenn Sie jedoch Medien von schlechter Qualität verwenden, könnte der Nutzer der Qualität Ihrer Produkte und Dienstleistungen misstrauen.

Einer der letzten Punkte ist die Verwendung einer Farb-Hierarchie. Sie sollten sicherstellen, dass die wichtigsten Informationen auf der Karte am auffälligsten sind. Sie können helle Farben für wichtige Elemente wie eine Call-to-Action-Schaltfläche verwenden und helle Farben für weniger wichtige Elemente auf der Karte.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2>Product Name</h2>
  <button>Buy Now</button>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
  text-align: center;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #28a745; 
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::
