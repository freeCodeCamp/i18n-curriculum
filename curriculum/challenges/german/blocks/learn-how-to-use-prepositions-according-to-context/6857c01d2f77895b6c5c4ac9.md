# --explanation--

`Opposite` bedeutet direkt gegenüber von oder zugewandt zu etwas. Zum Beispiel:

`The meeting room is opposite the elevator.` – Das bedeutet, dass sich die beiden auf gegenüberliegenden Seiten befinden.

`Mail room` ist ein Ort in einem Büro, an dem die Post bearbeitet wird. Zum Beispiel:

`Check the mail room for the delivery.` – Das bedeutet, dass das Paket dort sein könnte.
