# --description--

私たちの`pink-text`クラスは`body`要素のCSS宣言をオーバーライドしました！

クラスが`body`要素のCSSをオーバーライドすることを証明しました。では次に論理的な疑問として、`pink-text`クラスをオーバーライドするにはどうすればよいでしょうか？

# --instructions--

`blue-text`という名前の追加のCSSクラスを作成し、その要素の色を青にしてください。`pink-text`クラスの宣言の下に書くことを忘れないでください。

`blue-text`クラスを`h1`要素に、`pink-text`クラスと一緒に適用して、どちらが優先されるか見てみましょう。

HTML要素に複数のクラス属性を適用するには、クラス名の間にスペースを入れて次のようにします：

```html
class="class1 class2"
```

**注意：** HTML要素内でクラスの順番は関係ありません。

しかし、`class`セクション内の`<style>`宣言の順番が重要です。2番目の宣言が常に1番目より優先されます。`.blue-text`が2番目に宣言されているため、`.pink-text`の属性をオーバーライドします。

# --hints--

`h1`要素には`pink-text`クラスを付けてください。

```js
assert.isTrue(document.querySelector('h1').classList.contains('pink-text'));
```

`h1`要素には`blue-text`クラスを付けてください。

```js
assert.isTrue(document.querySelector('h1').classList.contains('blue-text'));
```

`blue-text`と`pink-text`は同じ`h1`要素に属している必要があります。

```js
assert.isTrue(document.querySelector('.pink-text').classList.contains('blue-text'));
```

`h1`要素は青色にしてください。

```js
const h1Element = document.querySelector('h1');
const color = window.getComputedStyle(h1Element)['color']; 
assert.strictEqual(color, 'rgb(0, 0, 255)');
```

# --solutions--

```html
<style>
  body {
    background-color: black;
    font-family: monospace;
    color: green;
  }
  .pink-text {
    color: pink;
  }

  .blue-text {
    color: blue;
  }  
</style>
<h1 class="pink-text blue-text">Hello World!</h1>
```
