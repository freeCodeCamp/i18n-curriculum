# --instructions--

このチャレンジでは、回帰アルゴリズムを使って医療費を予測します。

異なる人々の情報と医療費を含むデータセットが与えられます。このデータを使って、新しいデータに基づき医療費を予測してください。

このノートブックの最初の2つのセルはライブラリとデータをインポートします。

カテゴリカルデータは数値に変換してください。データの80％を`train_dataset`、20％を`test_dataset`として使います。

これらのデータセットから「expenses」列を`pop`して、`train_labels`と`test_labels`という新しいデータセットを作成してください。モデルの学習時にはこれらのラベルを使います。

モデルを作成し、`train_dataset`で学習させてください。このノートブックの最後のセルを実行してモデルを確認します。最後のセルは未使用の`test_dataset`を使ってモデルの汎化性能をチェックします。

チャレンジをクリアするには、`model.evaluate`が平均絶対誤差（Mean Absolute Error）を3500未満にする必要があります。これは医療費を3500ドル以内の誤差で正しく予測していることを意味します。

最後のセルは`test_dataset`を使って費用を予測し、結果をグラフ化します。
