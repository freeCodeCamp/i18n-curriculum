# --description--

`Seem to be` 用于当你不确定某事并且想要猜测或提出一种可能性时。例如，如果你说 `It seems to be working`，你是在猜测某事正在工作，但你并不百分之百确定。

示例：

肯定：`It seems to be working fine.`

负面：`It doesn't seem to be a major issue.`

问题：`Does it seem to be related to the network?`
