# --interactive--

CSS flexboxは、コンテナ内の要素を行や列に並べたり、その順序や向きを制御したりできる一次元のレイアウトモデルです。ウェブ開発者は、異なる画面サイズや向きに対応するレスポンシブなウェブサイトやウェブアプリケーションを作るためにこれを使います。flexboxを一次元のレイアウトモデルと呼ぶのは、一度に1つの軸に沿って要素を配置することに焦点を当てているからです。その軸は水平または垂直のいずれかになります。

flexboxを使い始める前に知っておくべき2つの重要な概念があります：フレックスコンテナとフレックスアイテムです。

フレックスコンテナとは、フレックスレイアウトを持つHTML要素のことです。フレックスコンテナ内で要素をさまざまな方法で配置・整列できます。HTML要素をフレックスコンテナにするには、そのCSSスタイルに`display: flex`を追加する必要があります。

フレックスアイテムはフレックスコンテナの直接の子要素です。これらの要素は、フレックスコンテナのプロパティに基づいて配置・整列されます。また、利用可能なスペースに合わせて縮小または拡大することもできます。

これは`main`コンテナと3つの子`div`要素の例です：

```html
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

これらの`width`要素の`height`、`background-color`、`div`だけをCSSスタイルシートで設定すると、コンテナはデフォルトではフレックスではないため、すべての子要素がそれぞれの行に配置されます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

しかし、`display: flex`コンテナに`main`を追加すると、`div`要素は同じ行に収まるように並べ替えられ、必要に応じて縮小されます（プレビューを見るにはインタラクティブエディターを有効にしてください）：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

デフォルトでは、フレックスコンテナはブロックレベル要素になるため、コンテナ自体は他の要素やコンテナに対して独立した行に配置されます。

フレックスコンテナとフレックスアイテムについて理解したので、次はフレックスプロパティについても知っておくべきです。これらのプロパティは、フレックスアイテムがフレックスコンテナ内でどのように配置、サイズ変更、分配されるかを決定します。よく使われるものには`flex-direction`、`justify-content`、`align-items`、`flex-wrap`があります。

では、フレックスモデルについて少し説明しましょう。このモデルはフレックスアイテムがフレックスコンテナ内でどのように配置されるかを定義します。すべてのフレックスコンテナには2つの軸があります：

- 主軸
- 交差軸

これらの軸の向きによって、さまざまなプロパティがレイアウトやフレックスアイテムの分配にどのように影響するかが決まります。デフォルトでは、フレックスコンテナの主軸は水平で、交差軸は垂直です。フレックスアイテムは主軸の方向に沿って配置されます。交差軸は主軸に対して垂直です。

`flex-direction`プロパティは主軸の方向を設定します。`flex-direction`のデフォルト値は`row`で、これはすべてのフレックスアイテムをブラウザのデフォルト言語の方向（左から右、または右から左）に沿って同じ行に配置します：

```css
flex-direction: row; /* Default */
```

アイテムの順序を逆にしたい場合は、`flex-direction: row-reverse`を使えます：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: row-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

これによりフレックスアイテムの順序が逆になります。フレックスアイテムを縦方向に整列させたい場合は、フレックスコンテナに`flex-direction`を`column`に設定するだけです。これにより主軸の方向が変わります：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

これで`div`要素は縦に整列されます。なぜなら主軸が垂直になり、交差軸が水平になるからです。

`flex-direction: column-reverse`を使えば、縦方向のフレックスアイテムの順序も逆にできます：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

CSS flexboxは、コンテナ内の要素を柔軟かつ効率的に配置する強力なレイアウトモデルです。フレックスコンテナ、フレックスアイテム、さまざまなフレックスプロパティの基本を理解すれば、異なる画面サイズや向きに対応する動的でレスポンシブなウェブサイトを作成できます。
