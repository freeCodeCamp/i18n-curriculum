# --description--

Ein Tuple ist ein Python-Datentyp, der verwendet wird, um eine geordnete Folge von Werten zu erstellen. Tuples können eine gemischte Menge von Datentypen enthalten, wie hier:

```py
developer = ('Alice', 34, 'Rust Developer')
```

Tuples sind Listen ähnlich, aber während Listen ein veränderbarer Datentyp sind, sind Tuples unveränderlich. Das bedeutet, dass die Elemente in einem Tuple nicht geändert werden können, sobald es erstellt wurde.

Wenn Sie versuchen, eines der Elemente im Tuple zu aktualisieren, erhalten Sie einen `TypeError`:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: 'tuple' object does not support item assignment
"""
```

Um auf ein Element eines Tuples zuzugreifen, können Sie die Klammernotation und die Indexnummer verwenden:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

Wenn Sie Elemente vom Ende eines Tuples aus zugreifen müssen, können Sie negative Indizes verwenden. Hier ist ein Beispiel, wie Sie mit einem negativen Index auf das vorletzte Element eines Tuples zugreifen:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

Wenn Sie versuchen, eine Indexnummer zu verwenden, die größer oder gleich der Länge des Tuples ist, erhalten Sie eine `IndexError` wie diese:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list index out of range
"""
```

Eine andere Möglichkeit, ein Tuple zu erstellen, ist die Verwendung des `tuple()`-Konstruktors, so:

```py
developer = 'Jessica'
tuple(developer) # ('J', 'e', 's', 's', 'i', 'c', 'a')
```

Für den `tuple()`-Konstruktor können Sie verschiedene iterierbare Objekte wie Strings, Listen und sogar andere Tuples übergeben.

Um zu überprüfen, ob ein Element in einem Tuple enthalten ist, können Sie das Schlüsselwort `in` verwenden, so:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

Sie können Elemente aus einem Tuple auch entpacken, genau wie Sie es bei Listen gemacht haben:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

In diesem Beispiel hat `name` den Wert `'Alice'`, `age` hat den Wert `34` und `job` hat den Wert `'Rust Developer'`.

Wenn Sie alle verbleibenden Elemente aus einem Tuple sammeln müssen, können Sie den Stern-Operator (`*`) verwenden, so:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

Hier hat `name` den Wert `'Alice'`, und `rest` ist eine Liste, die aus der Zahl `34` und dem String `'Rust Developer'` besteht.

Genau wie bei einer Liste können Sie den Slice-Operator auf ein Tuple anwenden, um einen Teil davon zu extrahieren. Hier ist ein Beispiel, wie die Elemente `'pie'` und `'cookies'` in ein separates Tuple extrahiert werden:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

Denken Sie daran, dass die erste Zahl den Startindex für die Extraktion darstellt, während die zweite Zahl den Endindex angibt. Beachten Sie aber, dass das Element am Endindex nicht im extrahierten Tuple enthalten ist.

Wenn Sie ein Element aus einem Tuple entfernen möchten, ist das nicht möglich, da Tuples unveränderlich sind. Dieses Beispiel führt also zu einem Fehler:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

Wann sollten Sie also ein Tuple einer Liste vorziehen?

Wenn Sie eine dynamische Sammlung von Elementen benötigen, bei der Sie Elemente hinzufügen, entfernen und aktualisieren können, sollten Sie eine Liste verwenden. Wenn Sie wissen, dass Sie mit einer festen und unveränderlichen Datensammlung arbeiten, sollten Sie ein Tuple verwenden.

In der nächsten Lektion schauen wir uns einige gängige Methoden an, die Sie beim Arbeiten mit Tuples verwenden werden.
