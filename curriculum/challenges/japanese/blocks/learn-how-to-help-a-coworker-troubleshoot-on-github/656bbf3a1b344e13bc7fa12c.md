# --description--

`Strange` は、普通ではない、または予期しないことを意味します。普段見ているものとは違う何かを見つけるようなものです。例えば：`A strange sound in a car might mean a problem.`

`error` は、特に何かが正しく動作しなくなる原因となる間違いです。例えば：`An error in a code means there is a mistake that needs fixing.`
