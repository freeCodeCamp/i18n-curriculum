# --description--

Це один із необхідних проєктів для отримання вашої сертифікації. У цьому проєкті ви створите Bash-скрипт, який вводитиме інформацію про матчі чемпіонату світу у PostgreSQL, а потім виконуватиме запити до бази даних для отримання корисної статистики.

# --instructions--

**Важливо:** Після того, як ви пройдете всі тести проєкту, збережіть дамп вашої бази даних у файл `worldcup.sql`, а також ваші файли `insert_data.sh` і `queries.sh`, щоб ви могли виконати крок 2. Інструкції, як це зробити, будуть у віртуальній машині.

# --notes--
