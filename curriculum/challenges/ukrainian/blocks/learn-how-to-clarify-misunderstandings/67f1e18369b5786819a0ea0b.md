# --explanation--

Джессіка каже, `There seems to be some misunderstanding about the testing phase`, і запитує, `Can we go over it?`

`Go over` означає ретельно переглянути або обговорити щось.  

Це показує, що вона хоче прояснити ситуацію, переглянувши деталі — а не починати, завершувати чи призначати щось.
