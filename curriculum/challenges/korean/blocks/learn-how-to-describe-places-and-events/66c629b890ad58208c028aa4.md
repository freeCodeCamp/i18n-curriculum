# --explanation--

`Walkthrough`는 두 부분으로 이루어져 있습니다:

- `Walk`는 보통 걸어서 한 걸음씩 이동하는 것을 의미합니다.

- `Through`는 한쪽에서 다른 쪽으로 또는 처음부터 끝까지 이동하는 것을 뜻합니다.

이 단어들이 `walkthrough`로 결합되면 누군가가 어떻게 하는지 이해하도록 돕는 단계별 설명이나 안내를 의미합니다. 예를 들어:

- `The software includes a walkthrough to help new users get started.` - 소프트웨어 사용법을 각 단계별로 설명하는 안내서가 있습니다.

- `Let me give you a walkthrough of the process.` - 과정의 각 단계를 자세히 설명할 것입니다.
