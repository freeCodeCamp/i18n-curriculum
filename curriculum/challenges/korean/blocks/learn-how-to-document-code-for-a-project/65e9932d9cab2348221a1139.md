# --description--

`Changes`는 이 문맥에서 코드에 가해진 수정이나 편집을 의미합니다.

`Descriptive`는 무언가의 모습, 느낌, 소리, 냄새, 맛에 대해 자세히 설명하는 것입니다. 명사를 더 생생하고 명확하게 설명하기 위해 사용하는 형용사입니다. 예를 들어, 문장 `she wears a beautiful dress`에서 단어 `beautiful`는 드레스에 대해 더 많은 정보를 전달하기 때문에 `descriptive`입니다.

`message`는 누군가가 다른 사람에게 보내는 정보나 소통의 한 조각입니다. 글로 쓰거나, 말하거나, 이메일이나 문자로 보낼 수도 있습니다.
