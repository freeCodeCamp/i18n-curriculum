# --description--

لنلقِ نظرة على بعض الأمثلة السريعة لتصور كيفية عمل التحديدية. انظر إلى شفرة HTML وCSS التالية:

```html
<!-- index.html -->

<div class="main">
  <div class="list subsection"></div>
</div>
```

```css
/* rule 1 */
.subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

في المثال أعلاه، كلا القاعدتين تستخدمان محددات صنف فقط، لكن القاعدة الثانية أكثر تحديدًا لأنها تستخدم عددًا أكبر من محددات الصنف.
