## --text--

Welcher Satz verwendet `since` korrekt im `Present Perfect Continuous`-Tempus?

## --answers--

`I have worked on the project since last week.`

### --feedback--

Dies ist das `Present Perfect Simple`-Tempus. Die Verbform sollte im `Present Perfect Continuous`-Tempus (`have been working`) stehen, um eine andauernde Handlung anzuzeigen.

---

`I was testing the program since yesterday.`

### --feedback--

Dies ist das `Past Simple`-Tempus. Das `Present Perfect Continuous`-Tempus sollte `have been` anstelle von `was` verwenden.

---

`I have been testing them since the latest software update.`

---

`I will have been working on the project since next month.`

### --feedback--

Das `Future Perfect`-Tempus ist hier nicht passend; `since` wird mit dem `Present Perfect Continuous`-Tempus verwendet.
