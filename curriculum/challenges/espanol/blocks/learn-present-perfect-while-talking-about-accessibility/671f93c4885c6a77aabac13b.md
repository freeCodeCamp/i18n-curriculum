# --explanation--

`Concerning` se usa para describir algo que causa preocupación o ansiedad. Por ejemplo:

`The drop in performance is quite concerning.` - Significa que la caída en el rendimiento es motivo de preocupación.

`Found out` es el tiempo pasado de `find out`. Significa descubrir o aprender algo que no se conocía previamente. Por ejemplo,

`She found out the reason for the delay.` - Se refiere a aprender nueva información sobre la causa del retraso.
