# --explanation--

完整的句子是 `Mi correo es mateo.delgado@sanjuan.pr.`

這裡，`correo` 取用的是電子郵件位址。

結構 `Mi` + 名詞 + `es...` 用於提供個人資訊。 例如：

- `Mi nombre es Ana.` - 我的名字是 Ana。

- `Mi número es 555-1234.` - 我的電話號碼是 555-1234。
