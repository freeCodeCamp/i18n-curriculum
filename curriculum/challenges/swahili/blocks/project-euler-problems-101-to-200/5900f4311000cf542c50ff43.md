# --description--

Kuna pembetatu 1234 za 60° ambazo $r ≤ 100$.

Acha $T(n)$ iwe idadi ya pembetatu za 60° ambazo $r ≤ n$, kwa hiyo $T(100) = 1234$, $T(1000) = 22767$, na $T(10000) = 359912$.

Tafuta $T(1053779)$.
