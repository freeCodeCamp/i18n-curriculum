# --explanation--

`user interface`（UI）指系统或软件中用户直接交互的部分，例如按钮、菜单和屏幕。例如： 

`The user interface was redesigned to be more intuitive.` —— 意思是用户交互的软件部分得到了改进。

`User feedback` 指用户关于他们对产品或服务体验的意见、建议或报告的问题。例如： 

`The company improved the app based on user feedback。` —— 意思是读取并使用了来自用户的评论和建议，以改进该应用程序。

在这个上下文中，Linda 解释说团队依靠用户反馈不断改进用户界面，确保它满足用户的需求。
