En esta oración, `first` se usa para mostrar la acción inicial que James tomó para depurar el código. Indica el inicio de una secuencia de pasos. Otros ejemplos de este uso incluyen:

- `First, we need to set up the database.` - Esto significa que configurar la base de datos es el paso inicial.
