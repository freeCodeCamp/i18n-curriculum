# --explanation--

En el lenguaje hablado informal, a menudo omites palabras cuando el significado sigue siendo claro. En este caso, `Anything else we should discuss?` es una forma abreviada de `Is there anything else we should discuss?`. Este tipo de omisión es común en la conversación. Otro ejemplo:

`Need any help?` es una forma más corta de decir `Do you need any help?` – El sujeto y el verbo auxiliar se omiten, pero el significado permanece igual.
