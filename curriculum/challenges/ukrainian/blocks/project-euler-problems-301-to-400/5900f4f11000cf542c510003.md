# --description--

Число Харшада або Нівена — це число, яке ділиться на суму своїх цифр.

201 — число Харшада, бо воно ділиться на 3 (суму своїх цифр).

Якщо відкинути останню цифру у 201, отримаємо 20, яке також є числом Харшада.

Якщо відкинути останню цифру у 20, отримаємо 2, що теж є числом Харшада.

Назвемо число Харшада, яке при рекурсивному відкиданні останньої цифри завжди дає число Харшада, правостороннім усіченим числом Харшада.

Також:

$\frac{201}{3} = 67$, що є простим числом.

Назвемо число Харшада, яке при діленні на суму своїх цифр дає просте число, сильним числом Харшада.

Тепер візьмемо число 2011, яке є простим. Якщо відкинути в ньому останню цифру, отримаємо 201 — сильне число Харшада, яке також є правостороннім усіченим. Назвемо такі прості числа сильними, правосторонніми усіченими простими числами Харшада.

Відомо, що сума сильних, правосторонніх усічених простих чисел Харшада менших за 10000 дорівнює 90619.

Знайдіть суму сильних, правосторонніх усічених простих чисел Харшада менших за ${10}^{14}$.
