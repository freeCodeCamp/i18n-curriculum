# --explanation--

`Trying to`는 무언가를 하려고 노력하는 것을 의미합니다. 예를 들어:

`She's trying to finish the report before noon.` – 이것은 그녀가 그것을 하려고 열심히 노력하고 있음을 보여줍니다.

`Monitor`는 이미지를 표시하는 장치로 컴퓨터와 함께 자주 사용됩니다. 예를 들어:

`He connected a second monitor to his laptop.` – 이것은 그가 추가 화면을 설치했다는 뜻입니다.

`They've`는 `they have`의 축약형입니다. 현재완료 시제로, 지금 이전에 완료된 동작이나 과거에 시작되어 지금까지 계속되는 상황을 나타냅니다. 예를 들어:

`They've moved the files to a new folder.` – 이것은 그들이 파일을 옮겼다는 뜻입니다.
