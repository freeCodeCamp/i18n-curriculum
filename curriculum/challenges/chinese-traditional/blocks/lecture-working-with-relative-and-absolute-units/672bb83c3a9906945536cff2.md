# --interactive--

在 CSS 中，`vh` 和 `vw` 是相對於視窗的單位，允許你根據瀏覽器視窗的尺寸來調整元素大小。這些單位特別適合用於建立能夠適應不同螢幕尺寸的響應式設計。

`vh` 代表「視窗高度」，而 `1vh` 相當於視窗高度的 1%。

同樣地，`vw` 代表「視窗寬度」，而 `1vw` 相當於視窗寬度的 1%。 

這表示如果你將元素的高度設為 `100vh`，它將佔據視窗的全部高度，而不管裝置的實際像素尺寸。

這些單位在你想要創建全螢幕佈局或維護螢幕特定比例的元素時特別方便。 

例如，你可能想要使用它們來創建一個永遠填滿整個螢幕的主視覺區塊（要與範例互動，你需要啟用互動式編輯器）。 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<section class="hero">
  <h1>100vh / 100vw Example</h1>
  <p>This section fills the entire browser window.</p>
</section>
```

```css
body {
  margin: 0;
  font-family: sans-serif;
  border: 5px dashed #333;
}

.hero {
  height: 100vh;
  width: 100vw;
  background-color: #add8e6; 
  padding: 2em;
}

.hero h1 {
  font-size: 2em;
  margin-bottom: 0.5em;
}

.hero p {
  font-size: 1em;
}
```

:::

這個 CSS 確保 hero 區塊將始終精確符合視窗大小，無論裝置的螢幕尺寸為何。

`vh` 和 `vw` 單位也可以用於排版，以創建響應式文字大小。

啟用互動式編輯器，並嘗試調整預覽視窗的大小，以查看文字如何隨視口大小縮放：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h1>Responsive Heading</h1>
<p>The h1 heading scales with the viewport width.</p>
```

```css
h1 {
  font-size: 5vw;
}
```

:::

這將把 `h1` 元素的字型大小設定為視窗寬度的 5％，讓文字能隨瀏覽器視窗大小平滑縮放。

`vh` 和 `vw` 單位的優點之一是它們能即時回應視口大小的變化。這表示如果使用者調整瀏覽器視窗大小，使用這些單位設定大小的元素會相應調整，而不需要重新載入頁面。然而，謹慎使用這些單位非常重要。例如，僅使用 `vw` 單位設定字型大小，可能會導致在窄螢幕上文字變得過小，或在寬螢幕上文字變得過大。 

另一個需要考慮的是，在行動裝置上，當瀏覽器的位址列出現或消失時，viewport 的高度可能會改變，如果你大量使用 `vh` 單位，這可能會導致意外的版面配置位移。

總結來說，`vh` 和 `vw` 單位是用於建立響應式版面配置和適應視窗大小的元素的強大工具。它們特別適用於全螢幕區塊、維護長寬比以及創建平滑縮放的設計。然而，應該謹慎使用，並且經常與其他 CSS 技術結合，以確保在所有裝置上提供最佳的使用者體驗。
