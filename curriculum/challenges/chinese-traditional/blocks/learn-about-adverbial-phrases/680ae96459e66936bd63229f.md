# --explanation--

`What about...` 用來詢問與剛剛討論內容相關的資訊或引發新的主題。它經常用來轉移焦點或檢查某事的不同面向。例如：

- `We finished the presentation. What about the handouts?`－這表示講者在詢問講義是否已經準備好或是否已經考慮過。

- `That plan sounds good. What about the timeline?`－這表示講話者同意這個想法，但想詢問需要花費多長時間。
