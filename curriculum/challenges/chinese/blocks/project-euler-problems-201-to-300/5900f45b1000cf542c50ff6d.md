# --description--

使用 "Blum Blum Shub" 伪随机数生成器创建数字序列：

$$
s_0 = 14025256 \\\\
s_{n + 1} = {s_n}^2 \\; mod \\; 20\\,300\\,713
$$

将这些数字 $s_0s_1s_2\ldots$ 连接起来以创建一个无限长度的字串 $w$。然后，$w = 14025256741014958470038053646\ldots$

对于正整数 $k$，如果不存在 $w$ 的子串，其数字和等于 $k$，则 $p(k)$ 被定义为零。如果存在至少一个 $w$ 的子串，其数字和等于 $k$，我们定义 $p(k) = z$，其中 $z$ 是最早出现的此类子串的起始位置。

例如：

子串 1、14、1402、…… 的各位数字之和分别相等于 1、5、7、……，它们从位置 1 开始，因此 $p(1) = p(5) = p(7) = \ldots = 1$。

子串 4、402、4025、… 的各自数字和分别相等于 4、6、11、…，它们从位置 2 开始，因此 $p(4) = p(6) = p(11) = \ldots = 2$。

子串 02、0252、… 的各位数字和分别相等于 2、9、…，它们从位置 3 开始，因此 $p(2) = p(9) = \ldots = 3$。

请注意，从位置 3 开始的子串 025 的数字和相等于 7，但之前有一个子串（从位置 1 开始）的数字和也相等于 7，所以 $p(7) = 1$，而不是 3。

我们可以验证，对于 $0 &lt; k ≤ {10}^3$，$\sum p(k) = 4742$。

查找 $\sum p(k)$，对于 $0 &lt; k ≤ 2 \times {10}^{15}$。
