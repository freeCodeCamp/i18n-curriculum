# --description--

如果你使用移动设备浏览中文课程，你可以使用 freeCodeCamp 的自定义输入直接输入拼音和汉字。但是，如果你想使用设备上的中文键盘，以下是设置方法。

## 添加语言键盘

### 在 Android 上

**注意：** Android `path` 不是通用的，可能因设备而异。

1. 打开设置。

2. 进入系统（或通用管理，取决于设备）。

3. 触碰 键盘 设置（或 语言 与 输入）。

4. 触碰 Languages 和 types（或添加 keyboard）。
 
5. 开启中文（简体或 CN），如有需要请下载。

6. 如果出现提示，储存或触碰 Done。

### 在 iPhone 上

1. 打开设置。

2. 转到通用的 > Keyboard。

3. 触碰键盘。

4. 触碰 添加 新建 键盘。

5. 选择中文（简体）。

6. 使用键盘上的地球图标切换语言。

## 尝试第三方键盘

你也可以在你的手机或平板上安装第三方键盘应用，以便更轻松地输入中文。这些键盘通常随附智能字符预测等实用特色，并支持多种输入方法，例如拼音或手写。

现在你已全部准备就绪。
