# --explanation--

**分數** 用來描述整體的零件。

`One-fourth` (¼) 是四個相等零件中的一個零件。其他範例：

- ½ = `one-half`

- ⅓ = `one-third`

- ¾ = `three-fourths`

在 Alice 的案例中，四分之一的開發預算花費在後端開發上，這顯示了資金在一個類別內的分配方式。
