## 使用者動作偽類

- **偽類別定義**：這些是特殊的 CSS 關鍵字，允許你根據元素的特定狀態或位置來選取該元素。
- **使用者操作偽類**：這些是特殊關鍵字，允許你根據使用者互動改變元素的外觀，提升整體使用者體驗。
- **`:active` 偽類別**：此偽類別讓你選取元素的作用中狀態，例如點擊按鈕時。
- **`:hover` 偽類**：此偽類定義元素的暫留狀態。
- **`:focus` 偽類**：當元素獲得焦點時，此偽類會套用樣式，通常是透過鍵盤導覽或使用者點擊進入表單輸入時。
- **`:focus-within` 偽類別**：當元素本身或其任何子孫擁有焦點時，使用此偽類別來套用樣式。

## 輸入偽類別

- **輸入偽類**：這些偽類用於根據使用者互動前後的狀態來標的 HTML `input` 元素。
- **`:enabled` 偽類**：此偽類用於標的目前已啟用的表單按鈕或其他元素。
- **`:disabled` 偽類別**：此偽類別讓你為作用中的互動元素設定停用模式的樣式。
- **`:checked` 偽類**：此偽類用於向使用者顯示它已被選取。
- **`:valid` 偽類**：此偽類標的符合驗證條件的輸入欄。
- **`:invalid` 偽類別**：此偽類別標的未符合驗證條件的輸入欄。
- **`:in-range` 和 `:out-of-range` 偽類別**：這些偽類別根據元素的值是否在指定的範圍條件內或外，套用樣式。
- **`:required` 偽類別**：此偽類別標的具有 `required` 屬性的 `input` 元素。它向使用者表示必須填寫該欄位才能提交表單。
- **`:optional` 偽類**：此偽類將樣式套用到非必填且可以留空的輸入元素。
- **`:autofill` 偽類別**：此偽類別會將樣式套用到瀏覽器自動填入已儲存資料的輸入欄位。

## 位置偽類

- **位置偽類**：這些偽類用於為目前文件中被標的的聯結和元素設定樣式。
- **`:any-link` 偽類**：此偽類是 `:link` 和 `:visited` 偽類的組合。因此，它會匹配任何具有 href 屬性的錨點元素，無論是否已造訪。
- **`:link` 偽類別**：此偽類別允許你標的所有尚未造訪的連結。你可以用它在使用者點擊之前，為連結設定不同的樣式。
- **`:local-link` 偽類別**：此偽類別標的指向相同文件的連結。當你想區分內部連結與外部連結時，它會很有用。
- **`:visited` 偽類別**：此偽類別標的使用者已造訪的聯結。
- **`:target` 偽類別**：此偽類別用於將樣式套用到作為 URL 片段標的的元素。

## 樹狀結構偽類別

- **樹狀結構偽類**：這些偽類允許你根據元素在文件樹中的位置來標的並設定樣式。
- **`:root` 偽類別**：此偽類別通常是根 `html` 元素。它可幫助你標的文件中的最高層級，從而為整個文件套用共用樣式。
- **`:empty` 偽類別**：空元素，也就是除了空白字元外沒有其他子元素的元素，也包含在文件樹中。這就是為什麼有 `:empty` 偽類別來標的空元素。
- **`:nth-child(n)` 偽類**：此偽類允許你根據元素在父元素中的位置來選取元素。
- **`:nth-last-child(n)` 偽類別**：此偽類別可讓你從結尾開始計數以選取元素。
- **`:first-child` 偽類**：此偽類會選取父元素或文件中的第一個元素。
- **`:last-child` 偽類別**：此偽類別會選取父元素或文件中的最後一個元素。
- **`:only-child` 偽類**：此偽類會選取父元素或文件中唯一的元素。
- **`:first-of-type` 偽類別**：此偽類別會選取其父元素中指定元素型別的第一個出現。
- **`:last-of-type` 偽類別**：此偽類別會選取其父元素中指定元素型別的最後一個出現。
- **`:nth-of-type(n)` 偽類別**：此偽類別允許你根據同類型兄弟元素中的位置，選取其父元素內的特定元素。
- **`:only-of-type` 偽類別**：如果元素是在其父元素中唯一的該型別，則此偽類別會選取該元素。

## 功能性偽類別

- **功能性偽類別**：功能性偽類別允許你根據更複雜的條件或關係來選取元素。與根據狀態（例如，`:hover`、`:focus`）來標的元素的一般偽類別不同，功能性偽類別接受引數。
- **`:is()` 偽類**：此偽類接受一個選擇器列表（例如 `ol`、`ul`），並選取符合列表中任一選擇器的元素。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<p class="example">This text will change color.</p>
<p>This text will not change color.</p>
<p>This text will not change color.</p>
<p class="this-works-too">This text will change color.</p>
```

```css
p:is(.example, .this-works-too) {
    color: red;
}
```

:::

- **`:where()` 偽類別**：此偽類別接受一個選擇器列表（例如 `ol`、`ul`），並選擇符合列表中任一選擇器的元素。` :is` 與 `:where` 的差別在於後者的特異性為 0。

```css
:where(h1, h2, h3) {
    margin: 0;
    padding: 0;
}
```

- **`:has()` 偽類別**：這個偽類別常被稱為「`parent`」選擇器，因為它允許你為包含選擇器列表中指定子元素的元素設定樣式。

```css
article:has(h2) {
    border: 2px solid hotpink;
}
```

- **`:not()` 偽類別**：此偽類別用於選取不符合所提供選擇器的元素。

```css
p:not(.example) {
  color: blue;
}
```

## 偽元素

- **`::before` 偽元素**：此偽元素使用 `content` 屬性在元素之前插入像圖示這樣的裝飾性內容。
- **`::after` 偽元素**：此偽元素使用 `content` 屬性在元素之後插入像圖示這樣的裝飾性內容。
- **`::first-letter` 偽元素**：此偽元素會標的元素內容的第一個字母，讓你可以為它設定樣式。
- **`::marker` 偽元素**：此偽元素讓你選取列表項目的標記（項目符號或編號）以進行樣式設定。
