# --description--

Infine, per la proprietà `box-shadow` del marcatore blu, sostituisci il colore nominato con la funzione `hsla`. Usa i valori `223` per la tonalità, `59%` per la saturazione, `31%` per la luminosità e `0.8` per il canale alfa.

E con questo, il tuo insieme di marcatori colorati è completo! Ben fatto.

# --hints--

La tua regola CSS `.blue` dovrebbe avere una proprietà abbreviata `box-shadow` con il valore `0 0 20px 0 hsla(223, 59%, 31%, 0.8)`.

```js
assert.strictEqual(new __helpers.CSSHelp(document).getStyle('.blue')?.boxShadow, 'rgba(32, 59, 126, 0.8) 0px 0px 20px 0px');
```
