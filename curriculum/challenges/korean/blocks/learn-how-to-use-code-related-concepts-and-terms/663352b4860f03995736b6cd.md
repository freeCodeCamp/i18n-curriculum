# --description--

`Store information`는 특정 위치, 보통 변수에 데이터를 저장하거나 보관하는 것을 의미합니다. 변수는 숫자, 텍스트 또는 다른 유형의 데이터를 담는 코드 내 컨테이너와 같습니다. 변수는 나중에 코드에서 `stored information`에 접근할 수 있는 이름 붙은 저장 공간이라고 생각할 수 있습니다.

예를 들어, `int age = 30;` 여기서 변수 `age`는 숫자 30을 저장합니다.

단어 `like`는 예시를 들거나 비슷한 것들을 비교할 때 전치사로 사용됩니다.

예를 들어, `I love fruits like apples and oranges.`라고 말하면, 당신이 좋아하는 과일 종류의 예시를 들기 위해 `like`를 사용하고 있으며, 그 목록에 더 많은 과일이 있을 수 있음을 암시합니다.
