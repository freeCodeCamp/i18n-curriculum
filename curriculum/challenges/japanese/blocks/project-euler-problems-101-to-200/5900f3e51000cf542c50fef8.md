# --description--

袋の中に赤いディスクが1枚と青いディスクが1枚入っています。ゲームでは、プレイヤーがランダムにディスクを1枚取り、その色が記録されます。各ターンの後、ディスクは袋に戻され、赤いディスクが1枚追加され、再びランダムにディスクが1枚取られます。

プレイヤーはゲームをプレイするのに1ポンド支払い、ゲーム終了時に青いディスクを赤いディスクより多く取っていれば勝ちとなります。

ゲームが4ターン行われた場合、プレイヤーが勝つ確率は正確に11/120であり、このゲームで勝利に対して銀行家が割り当てるべき最大の賞金基金は10ポンドとなります。これは、銀行家が損失を被ることを期待する前の金額です。支払いは必ず整数ポンドで、ゲームをプレイするために支払った最初の1ポンドも含まれるため、例の場合、プレイヤーの実際の勝ち金は9ポンドとなります。

15ターン行われる単一のゲームに割り当てるべき最大の賞金基金を求めてください。
