# --interactive--

تسمح لك معاملات المقارنة بمقارنة قيمتين وإرجاع نتيجة `true` أو `false`. يمكنك بعد ذلك استخدام النتيجة لاتخاذ قرار أو التحكم في تدفق برنامجك. تستخدم المقارنات في جمل `if` الشرطية، والحلقات، والعديد من الحالات الأخرى التي تحتاج فيها لاتخاذ قرارات بناءً على شروط معينة. لنغص في أكثر معاملات المقارنة شيوعًا ونرى كيف تعمل.

معامل أكبر من، المُمثل بواسطة قوس زاوية يمين (`>`)، يتحقق مما إذا كانت القيمة على اليسار أكبر من القيمة على اليمين:

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a > b); // false
console.log(b > a); // true
```

:::

معامل أكبر من أو يساوي، المُمثل بواسطة قوس زاوية يمين وعلامة يساوي (`>=`)، يتحقق مما إذا كانت القيمة على اليسار إما أكبر من أو تساوي القيمة على اليمين:

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a >= b); // false
console.log(b >= a); // true
console.log(a >= c); // true
```

:::

معامل أصغر من، المُمثل بواسطة قوس زاوية يسار (`<`) يعمل بطريقة مشابهة لـ `>`، لكن بالعكس. يتحقق مما إذا كانت القيمة على اليسار أصغر من القيمة على اليمين:

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a < b); // true
console.log(b < a); // false
```

:::

معامل أصغر من أو يساوي، المُمثل بواسطة قوس زاوية يسار وعلامة يساوي (`<=`) يتحقق مما إذا كانت القيمة على اليسار أصغر من أو تساوي القيمة على اليمين:

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a <= b); // true
console.log(b <= a); // false
console.log(a <= c); // true
```

:::
