# --interactive--

## 시맨틱 HTML의 중요성

- **제목 요소의 구조적 계층**: 콘텐츠의 구조적 계층을 유지하려면 올바른 제목 요소를 사용하는 것이 중요합니다. `h1` 요소는 가장 높은 수준의 제목이고 `h6` 요소는 가장 낮은 수준의 제목입니다.
- **표현용 HTML 요소**: 콘텐츠의 외형을 정의하는 요소입니다. 예: 더 이상 사용되지 않는 `center`, `big`, `font` 요소.
- **시맨틱 HTML 요소**: 의미와 구조를 담고 있는 요소입니다. 예: `header`, `nav`, `figure`.

## 시맨틱 HTML 요소

- **헤더 요소**: 문서나 섹션의 헤더를 정의하는 데 사용합니다.

:::interactive_editor

  ```html
  <header>
    <h1>CatPhotoApp</h1>
    <p>Welcome to our cat gallery.</p>
  </header>
  ```

:::

- **메인 요소**: 웹 페이지의 주요 콘텐츠를 담는 데 사용합니다.

:::interactive_editor

  ```html
  <main>
    <section>
      <h2>Cat Photos</h2>
      <p>Browse adorable cat pictures.</p>
    </section>
  </main>
  ```

:::

- **섹션 요소**: 콘텐츠를 더 작은 섹션으로 나누는 데 사용합니다.

:::interactive_editor

  ```html
  <section>
    <h2>About Me</h2>
    <p>Hi, I am Jane Doe and I am a web developer.</p>
  </section>
  ```

:::

- **내비게이션 섹션 (`nav`) 요소**: 내비게이션 링크가 있는 섹션을 나타냅니다.

:::interactive_editor

  ```html
  <nav>
    <ul>
      <li><a href="#photos">Photos</a></li>
      <li><a href="#videos">Videos</a></li>
    </ul>
  </nav>
  ```

:::

- **그림 요소**: 삽화와 도표를 담는 데 사용합니다.

:::interactive_editor

  ```html
  <figure>
    <img
      src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"
      alt="Two tabby kittens sleeping together on a couch."
    />
    <figcaption>Cats <strong>hate</strong> other cats.</figcaption>
  </figure>
  ```

:::

- **강조 (`em`) 요소**: 강조된 텍스트를 표시합니다.

:::interactive_editor

  ```html
  <p>
    Never give up on <em>your</em> dreams.
  </p>
  ```

:::

- **관용구 텍스트 (`i`) 요소**: 대체 어조나 기분, 다른 언어의 관용구, 기술 용어, 생각을 강조하는 데 사용합니다.

:::interactive_editor

  ```html
  <p>
    There is a certain <i lang="fr">je ne sais quoi</i> in the air.
  </p>
  ```

:::

열린 `lang` 태그 안의 `i` 속성은 콘텐츠의 언어를 지정하는 데 사용합니다. 이 경우 언어는 프랑스어입니다. `i` 요소는 텍스트가 중요한지 여부를 나타내지 않고, 단지 주변 텍스트와 다르다는 것을 보여줍니다.

- **강한 중요성 (`strong`) 요소**: 강한 중요성을 가진 텍스트를 표시합니다.

:::interactive_editor

  ```html
  <p>
    <strong>Warning:</strong> This product may cause allergic reactions.
  </p>
  ```

:::

- **주의 환기 (`b`) 요소**: 콘텐츠 의미에 중요하지 않은 텍스트에 주의를 끌 때 사용합니다. 요약에서 키워드를 강조하거나 리뷰에서 제품명을 강조할 때 흔히 사용합니다.

:::interactive_editor

  ```html
  <p>
    We tested several products, including the <b>SuperSound 3000</b> for audio quality, the <b>QuickCharge Pro</b> for fast charging, and the <b>Ecoclean Vacuum</b> for cleaning. The first two performed well, but the <b>Ecoclean Vacuum</b> did not meet expectations.
  </p>
  ```

:::

- **설명 목록 (`dl`) 요소**: 용어-설명 그룹 목록을 나타내는 데 사용합니다.
- **설명 용어 (`dt`) 요소**: 정의되는 용어를 나타내는 데 사용합니다.
- **설명 세부사항 (`dd`) 요소**: 용어의 설명을 나타내는 데 사용합니다.

:::interactive_editor

  ```html
  <dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
  </dl>
  ```

:::

- **블록 인용 (`blockquote`) 요소**: 다른 출처에서 인용한 섹션을 나타내는 데 사용합니다. 이 요소는 `cite` 속성을 가집니다. `cite` 속성 값은 출처 URL입니다.

:::interactive_editor

  ```html
  <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
    "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
  </blockquote>
  ```

:::

- **인용 (`cite`) 요소**: 참조된 작업의 출처를 시각적으로 표시하는 데 사용합니다. 참조 제목을 마크업합니다.

:::interactive_editor

  ```html
  <div>
    <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
      "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
    </blockquote>
    <p>
      -Quincy Larson, <cite>How to learn to Code and Get a Developer Job [Full Book].</cite>
    </p>
  </div>
  ```

:::

- **인라인 인용 (`q`) 요소**: 짧은 인라인 인용을 나타내는 데 사용합니다.

:::interactive_editor

  ```html
  <p>
    As Quincy Larson said,
    <q cite="https://www.freecodecamp.org/news/learn-to-code-book/">
      Momentum is everything.
    </q>
  </p>
  ```

:::

- **약어 (`abbr`) 요소**: 약어나 두문자어를 나타내는 데 사용합니다. 사용자가 약어나 두문자어를 이해할 수 있도록 `title` 속성으로 전체 형태나 사람이 읽을 수 있는 설명을 보여줄 수 있습니다.

:::interactive_editor

  ```html
  <p>
    <abbr title="HyperText Markup Language">HTML</abbr> is the foundation of the web.
  </p>
  ```

:::

- **연락처 주소 (`address`) 요소**: 연락처 정보를 나타내는 데 사용합니다.
- **(날짜) 시간 (`time`) 요소**: 날짜와/또는 시간을 나타내는 데 사용합니다. `datetime` 속성은 날짜와 시간을 기계가 읽을 수 있는 형식으로 변환하는 데 사용합니다.

:::interactive_editor

  ```html
  <p>
    The reservations are for the <time datetime="20:00">20:00 </time>
  </p>
  ```

:::

- **ISO 8601 날짜 (`datetime`) 속성**: 날짜와 시간을 기계가 읽을 수 있는 형식으로 나타내는 데 사용합니다. 표준 형식은 `YYYY-MM-DDThh:mm:ss`이며, 대문자 `T`는 날짜와 시간 사이의 구분자입니다.
- **위 첨자 (`sup`) 요소**: 위 첨자 텍스트를 나타내는 데 사용합니다. `sup` 요소의 일반적인 용도는 지수, 윗글자, 서수입니다.

:::interactive_editor

  ```html
  <p>
    2<sup>2</sup> (2 squared) is 4.
  </p>
  ```

:::

- **아래 첨자 (`sub`) 요소**: 아래 첨자 텍스트를 나타내는 데 사용합니다. 일반적인 용도는 화학식, 각주, 변수 아래 첨자입니다.

:::interactive_editor

  ```html
  <p>
    CO<sub>2</sub>
  </p>
  ```

:::

- **인라인 코드 (`code`) 요소**: 컴퓨터 코드 조각을 나타내는 데 사용합니다.
- **서식 있는 텍스트 (`pre`) 요소**: 서식이 지정된 텍스트를 나타냅니다.

:::interactive_editor

  ```html
  <pre>
    <code>
      body {
        color: red;
      }
    </code>
  </pre>
  ```

:::

- **비표현 주석 (`u`) 요소**: 비텍스트 주석이 있음을 나타내는 인라인 텍스트 범위를 렌더링하는 데 사용합니다.

:::interactive_editor

  ```html
  <p>
    You can use the unarticulated annotation element to highlight
    <u>inccccort</u> <u>spling</u> <u>issses</u>.
  </p>
  ```

:::

- **루비 주석 (`ruby`) 요소**: 발음이나 의미 설명을 주석으로 다는 데 사용합니다. 동아시아 타이포그래피에 주로 사용됩니다.
- **루비 대체 괄호 (`rp`) 요소**: 루비 주석을 표시하지 못하는 브라우저를 위한 대체 수단입니다.
- **루비 텍스트 (`rt`) 요소**: 루비 주석 텍스트를 나타냅니다. 보통 발음이나 번역 세부 정보를 위해 동아시아 타이포그래피에서 사용됩니다.

:::interactive_editor

  ```html
  <ruby>
    明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp>
  </ruby>
  ```

:::

- **취소선 (`s`) 요소**: 더 이상 정확하지 않거나 관련 없는 콘텐츠를 나타내는 데 사용합니다.

:::interactive_editor

  ```html
  <p>
    <s>Tomorrow's hike will be meeting at noon.</s>
  </p>
  <p>
    Due to unforeseen weather conditions, the hike has been canceled.
  </p>
  ```

:::

- **내부 링크**: `href="#id"`를 `a` 요소에 사용하고 대상 요소에 같은 `id`를 부여하여 페이지 내 다른 섹션으로 링크하는 데 사용합니다. 주로 건너뛰기 링크, 목차, 여러 섹션이 있는 긴 페이지에 사용됩니다.

:::interactive_editor

```html
<a href="#about-section">Go to About Section</a>

<section id="about-section">
  <h2>About</h2>
  <p>This is the about section of the page.</p>
</section>
```

:::
