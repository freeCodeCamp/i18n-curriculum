# --description--

ليكن $n$ عددًا صحيحًا موجبًا. يُسمى الثلاثي الصحيح ($a$، $b$، $c$) ثلاثيًا للتحليل إلى عوامل لـ $n$ إذا:

- $1 ≤ a ≤ b ≤ c$
- $a \times b \times c = n$.

عرف $f(n)$ بأنه $a + b + c$ للثلاثي للتحليل إلى عوامل ($a$، $b$، $c$) لـ $n$ الذي يُصغر قيمة $\frac{c}{a}$. يمكن إثبات أن هذا الثلاثي فريد.

على سبيل المثال، $f(165) = 19$، $f(100\\,100) = 142$ و $f(20!) = 4\\,034\\,872$.

احسب $f(43!)$.
