# --description--

有幾種方法可以用不同整數的倒平方和來表示數字 $\frac{1}{2}$。

例如，數字 {2,3,4,5,7,12,15,20,28,35} 可以使用：

$$\frac{1}{2} = \frac{1}{2^2} + \frac{1}{3^2} + \frac{1}{4^2} + \frac{1}{5^2} + \frac{1}{7^2} + \frac{1}{{12}^2} + \frac{1}{{15}^2} + \frac{1}{{20}^2} + \frac{1}{{28}^2} + \frac{1}{{35}^2}$$

事實上，只使用介於 2 和 45（含）之間的整數，恰好有三種方法做到，剩下的兩種為：{2,3,4,6,7,9,10,20,28,35,36,45} 和 {2,3,4,6,7,9,12,15,28,30,35,36,45}。

有多少種方法可以將數字 $\frac{1}{2}$ 寫成介於 2 和 80（含）之間不同整數的倒數平方和？
