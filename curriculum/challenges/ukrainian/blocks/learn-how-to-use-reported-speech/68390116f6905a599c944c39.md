# --explanation--

`If only` використовується на початку речення, щоб висловити бажання або жаль щодо чогось, що не сталося в минулому. Це показує, що мовник хотів би, щоб ситуація була іншою. Наприклад:

`If only we had checked the system earlier, we could have avoided the crash.` – Це означає, що вони не перевірили достатньо рано і тепер шкодують про це.
