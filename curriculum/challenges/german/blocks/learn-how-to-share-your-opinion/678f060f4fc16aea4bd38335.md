# --explanation--

`Thinking of` wird verwendet, wenn Sie eine Idee, Handlung oder einen Plan in Erwägung ziehen. Zum Beispiel:

`I'm thinking of buying a new laptop.` – Das bedeutet, dass Sie die Option in Betracht ziehen, einen neuen Laptop zu kaufen, aber noch keine Entscheidung getroffen haben.

`Cloud` bezeichnet ein System, bei dem Rechenressourcen wie Speicher über das Internet bereitgestellt werden, anstatt lokal auf einem Gerät gespeichert zu sein. Zum Beispiel:

`We use a cloud system to back up our files online.` – Das bedeutet, dass die Dateien in einem Online-Dienst gespeichert sind, anstatt auf einer physischen Festplatte oder einem Server.
