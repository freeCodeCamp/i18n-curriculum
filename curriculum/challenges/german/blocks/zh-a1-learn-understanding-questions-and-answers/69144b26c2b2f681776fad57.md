# --explanation--

`吗 (ma)` ist ein Fragepartikel. Es wird ans Ende eines Aussagesatzes gesetzt, um daraus eine Frage zu machen, ähnlich wie ein Fragezeichen (?) am Satzende.

Wang Hua bildet eine Frage, indem sie `吗 (ma)` ans Ende eines Aussagesatzes `你是设计师 (nǐ shì shè jì shī)` setzt. Das zeigt, dass sie eine Bestätigung über Liu Mings Beruf als Designerin erfragt.
