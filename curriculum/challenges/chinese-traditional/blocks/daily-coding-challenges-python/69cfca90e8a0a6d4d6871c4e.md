# --description--

給定一個字陣列，傳回一個二維陣列，將字依字謎分組。

- 如果字包含相同字母且順序可不同，則它們是字謎。
- 每個字只屬於一個群組。
- 傳回的順序不重要。

例如，給定 `["listen", "silent", "hello", "enlist", "world"]`，傳回 `[["listen", "silent", "enlist"], ["hello"], ["world"]]`。

```py
def group_anagrams(words):
    groups = {}
    for word in words:
        key = ''.join(sorted(word))
        if key not in groups:
            groups[key] = []
        groups[key].append(word)
    return list(groups.values())
```
