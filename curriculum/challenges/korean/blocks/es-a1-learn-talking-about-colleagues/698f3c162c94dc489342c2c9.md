# --explanation--

카밀라는 `¿Es de Colombia?`라고 묻습니다. 이는 "그녀는 콜롬비아 출신인가요?"라는 뜻입니다.

따라서 카밀라는 마리솔이 어디 출신인지 묻고 있습니다.
