# --description--

El término `reusable` se refiere a algo que puede ser utilizado más de una vez, especialmente en diferentes situaciones o para varios propósitos. En programación, `reusable` a menudo describe código que está diseñado para ser usado múltiples veces, ya sea dentro del mismo programa o en programas diferentes, sin necesidad de ser reescrito.
