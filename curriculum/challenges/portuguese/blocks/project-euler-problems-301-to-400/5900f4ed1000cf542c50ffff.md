# --description--

Considere $f_5(n)$ como o maior número inteiro $x$ para o qual $5^x$ divide $n$.

Por exemplo, $f_5(625\,000) = 7$.

Let $T_5(n)$ be the number of integers $i$ which satisfy $f_5((2 \times i - 1)!) &lt; 2 \times f_5(i!)$ and $1 ≤ i ≤ n$.

Pode-se verificar que $T_5({10}^3) = 68$ e $T_5({10}^9) = 2\,408\,210$.

Encontre $T_5({10}^{18})$.
