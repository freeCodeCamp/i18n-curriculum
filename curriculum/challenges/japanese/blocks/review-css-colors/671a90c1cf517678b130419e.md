## カラー理論

- **カラー理論の定義**：これは色同士の相互作用や、色が私たちの認識にどのように影響するかを研究するものです。色の関係性、色の調和、色の心理的影響を扱います。
- **原色**：黄色、青、赤のこれらの色は、すべての他の色の元となる基本的な色相です。
- **二次色**：これらの色は、2つの原色を等量混ぜることで生まれます。緑、オレンジ、紫が二次色の例です。
- **三次色**：これらの色は、原色と隣接する二次色を組み合わせることで生まれます。イエローグリーン、ブルーグリーン、ブルーバイオレットが三次色の例です。
- **暖色**：赤、オレンジ、黄色を含むこれらの色は、快適さ、暖かさ、居心地の良さを感じさせます。
- **寒色**：青、緑、紫を含むこれらの色は、落ち着き、静けさ、プロフェッショナルな印象を与えます。
- **カラーホイール**：カラーホイールは色同士の関係を示す円形の図です。デザイナーにとって重要なツールで、色の組み合わせを選ぶのに役立ちます。
- **類似色配色**：これらの配色はまとまりがあり落ち着いた印象を作り出します。カラーホイール上で隣接する類似色を使います。
- **補色配色**：これらの配色は高いコントラストと視覚的なインパクトを生みます。カラーホイールの反対側に位置する色同士を使います。
- **三色配色**：この配色は鮮やかな色合いを持ちます。ほぼ等間隔に配置された色を使い、カラーホイール上で正三角形を形成します。
- **単色配色**：この配色は同じベースカラーから明度、暗度、彩度を調整して作られます。統一感と調和を感じさせつつ、コントラストも生み出します。

## CSSで色を扱うさまざまな方法

- **名前付きカラー**：これらはブラウザが認識するあらかじめ定義された色名です。例には`blue`、`darkred`、`lightgreen`があります。
- **`rgb()`関数**：RGBは光の三原色である赤、緑、青を表します。これら3色を異なる強度で組み合わせて幅広い色を作ります。`rgb()`関数はRGBカラーモデルを使って色を定義できます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>RGB Color</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

- **`rgba()`関数**：この関数は4番目の値であるアルファを追加し、色の透明度を制御します。指定しない場合、アルファ値は`1`がデフォルトです。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>RGBA Background</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

- **`hsl()`関数**：HSLは色相、彩度、明度の3つの主要な要素を表します。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>HSL Color</p>
```

```css
p {
  color: hsl(120, 100%, 50%);
}
```

:::

- **`hsla()`関数**：この関数は4番目の値であるアルファを追加し、色の不透明度を制御します。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>HSLA Background</div>
```


```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

- **16進数表記**：16進数コード（hexコード）はRGBカラーモデルの色を表す6文字の文字列です。「16進数」は0から9の数字とAからFの文字を使う16進法のことを指します。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1 class="hex-text">Hex Text</h1>
<p class="hex-bg">Hex Background</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}

p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

## `box-shadow`プロパティ

- **定義**：`box-shadow`プロパティは要素の周囲に1つ以上の影を適用します。
- **オフセット値**：水平（`offset-x`）と垂直（`offset-y`）の値を指定する必要があります。正の`offset-x`は影を右に移動し、負の値は左に移動します。正の`offset-y`は影を下に移動し、負の値は上に移動します。値が`0`の場合、単位は不要です。
- **ぼかし半径**：このオプションの値は影のぼかし具合を制御します。指定しない場合は`0`がデフォルトで、シャープなエッジを作ります。値が大きいほど影は柔らかくなります。
- **広がり半径**：このオプションの値は影の拡大・縮小を制御します。指定しない場合は`0`がデフォルトです。
- **影の色**：名前付きカラー、16進数値、`rgb()`、`rgba()`、`hsl()`、`hsla()`関数で色を指定できます。
- **`inset`キーワード**：`inset`キーワードを追加すると、影が要素の内側に配置されます。
- **複数のボックスシャドウの適用**：複数の影はカンマで区切って適用できます。影は前面から背面へと重なります。

### 構文

```css
box-shadow: offset-x offset-y blur-radius spread-radius color;
```

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="shadow-box">Shadow Color Example</div>
```

```css
.shadow-box {
  width: 200px;
  padding: 20px;
  background-color: lightblue;
  box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}
```

:::

## 線形グラデーションと放射状グラデーション

- **線形グラデーション**：これらのグラデーションは直線に沿って色が徐々に混ざり合います。`to top`、`to right`、`to bottom right`のようなキーワードや`45deg`のような角度で方向を制御できます。任意の有効なCSSカラーと好きな数のストップを使えます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient">Linear Gradient</div>
```

```css
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

:::

- **放射状グラデーション**：これらのグラデーションは中心点から放射状に広がる円形または楕円形のグラデーションを作ります。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient">Radial Gradient</div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::
