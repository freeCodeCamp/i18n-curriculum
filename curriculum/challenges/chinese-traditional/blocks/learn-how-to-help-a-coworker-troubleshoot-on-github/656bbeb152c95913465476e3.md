# --description--

在程式設計中，`Compile` 是指將人們撰寫的程式碼轉換成電腦能理解和使用的形式。 

這就像將用一種語言寫成的食譜翻譯成另一種語言，讓別人能夠烹飪這道菜。 

例如，`When we compile Java code, it changes into a form that the computer can run.`
