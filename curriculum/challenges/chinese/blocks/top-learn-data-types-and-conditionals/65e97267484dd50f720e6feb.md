# --description--

你知道 JavaScript 有多种数据类型，包括字串、数字和布尔。你可以使用比较操作符来比较这些数据类型，并根据结果做出决策。

JavaScript 中可用的以下比较操作符：

| Operator | Description                              | Example      |
|----------|------------------------------------------|--------------|
| ==       | Equal (value comparison)                 | x == y       |
| !=       | Not equal                                | x != y       |
| >        | Greater than                             | x > y        |
| <        | Less than                                | x < y        |
| >=       | Greater than or equal to                 | x >= y       |
| <=       | Less than or equal to                    | x <= y       |

要在变量赋值中使用比较，可以使用 `==` 操作符。例如：

```javascript
let x = 5;
let y = 5;

let result = x == y;

console.log(result); // true
```
