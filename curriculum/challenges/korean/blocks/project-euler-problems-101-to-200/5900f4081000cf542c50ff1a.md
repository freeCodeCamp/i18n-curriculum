# --description--

전기 회로는 모두 동일한 값 C를 가진 동일한 커패시터만 사용합니다.

커패시터는 직렬 또는 병렬로 연결하여 하위 단위를 만들 수 있고, 이 하위 단위들은 다시 다른 커패시터나 다른 하위 단위와 직렬 또는 병렬로 연결하여 더 큰 하위 단위를 만들 수 있으며, 이렇게 계속해서 최종 회로까지 확장할 수 있습니다.

이 간단한 절차를 사용하여 최대 n개의 동일한 커패시터로 다양한 총 커패시턴스 값을 가진 회로를 만들 수 있습니다. 예를 들어, 각각 60 μF인 커패시터 3개까지 사용하면 다음과 같이 7가지 서로 다른 총 커패시턴스 값을 얻을 수 있습니다:

<img alt="example circuits having up to three capacitors, each of 60 μF" src="https://cdn.freecodecamp.org/curriculum/project-euler/counting-capacitor-circuits.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$n$개의 동일한 커패시터를 최대한 사용하여 위 절차로 만들 수 있는 서로 다른 총 커패시턴스 값의 수를 $D(n)$이라 할 때, $D(1) = 1, D(2) = 3, D(3) = 7, \ldots$ 입니다.

$D(18)$의 값을 구하세요.

참고: 커패시터 $C_1$, $C_2$ 등을 병렬로 연결할 때 총 커패시턴스는 $C_T = C_1 + C_2 + \cdots$이고, 직렬로 연결할 때는 전체 커패시턴스가 $\frac{1}{C_T} = \frac{1}{C_1} + \frac{1}{C_2} + \cdots$로 계산됩니다.
