# --description--

`flex` 선언은 실제로 플렉스 항목에 설정할 수 있는 3가지 속성의 단축 속성입니다. 이 속성들은 플렉스 항목이 컨테이너 내에서 크기를 조절하는 방식에 영향을 줍니다. 이전에 몇 가지 단축 속성을 본 적은 있지만, 아직 공식적으로 정의하지는 않았습니다.

> 단축 속성은 여러 다른 CSS 속성의 값을 동시에 설정할 수 있게 해주는 CSS 속성입니다. 단축 속성을 사용하면 더 간결하고(종종 더 읽기 쉬운) 스타일시트를 작성할 수 있어 시간과 노력을 절약할 수 있습니다.

이 경우, `flex`은 실제로 `flex-grow`, `flex-shrink`, `flex-basis`의 단축 속성입니다.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/flex-box/flexbox-04.png" alt="CSS code setting the flex property to 1 for a div element.">

위 스크린샷에서 `flex: 1`는 `flex-grow: 1`, `flex-shrink: 1`, `flex-basis: 0`와 같습니다.

대부분의 경우, flex 단축 속성은 하나의 값만으로 정의됩니다. 이 경우 그 값은 `flex-grow`에 적용됩니다. 그래서 div에 `flex: 1`을 넣었을 때, 실제로는 `flex: 1 1 0`의 단축 속성을 지정한 것입니다.
