# --interactive--

Beim Erstellen mehrsprachiger Webseiten oder beim Umgang mit benutzerdefinierten Datenattributen müssen Sie häufig Elemente basierend auf der Sprache, die sie enthalten, oder bestimmten Datenwerten gestalten.

Die `lang` und `data-lang` Attribute werden häufig für diese Zwecke verwendet, und der Attributselektor in CSS ermöglicht es Ihnen, Stile effektiv basierend auf diesen Attributen anzuwenden.

Das `lang` Attribut wird in HTML verwendet, um die Sprache des Inhalts innerhalb eines Elements anzugeben. Sie möchten Elemente möglicherweise unterschiedlich gestalten, je nachdem, in welcher Sprache sie geschrieben sind, insbesondere auf einer mehrsprachigen Webseite.

Hier ist ein Beispiel, wie Sie den Attributselektor verwenden können, um Elemente mit einem bestimmten `lang` Attribut anzusprechen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">This is an English paragraph.</p>
<p lang="fr">Ceci est un paragraphe en français.</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

Diese CSS-Regel wendet kursiven Stil auf jedes Absatz-Element an, bei dem das `lang` Attribut auf Englisch (`en`) gesetzt ist. Dies kann nützlich sein, um englischen Text in einem Dokument mit mehreren Sprachen hervorzuheben.

Ebenso können Sie den Attributselektor verwenden, um Elemente mit einem `data-lang` Attribut anzusprechen.

Benutzerdefinierte Datenattribute wie `data-lang` werden häufig verwendet, um zusätzliche Informationen in Elementen zu speichern, zum Beispiel um die innerhalb eines bestimmten Textabschnitts verwendete Sprache anzugeben.

So können Sie CSS verwenden, um Elemente basierend auf dem `data-lang` Attribut anzusprechen und zu gestalten:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p data-lang="fr">Ceci est un paragraphe en français.</p>
<p data-lang="en">This is a paragraph in English.</p>
```

```css
p[data-lang="fr"] {
  color: blue;
}
```

:::

In diesem Fall wird bei jedem `p` Element mit einem `data-lang` Attribut, das auf Französisch (`fr`) gesetzt ist, die Textfarbe auf Blau geändert. So können Sie schnell Abschnitte von Inhalten basierend auf den in dem `data-lang` Attribut gespeicherten Sprachinformationen identifizieren und gestalten.

Attributselektoren wie diese bieten eine leistungsstarke Möglichkeit, bedingte Gestaltung basierend auf den in Ihrem HTML eingebetteten Metadaten anzuwenden, wodurch Ihre Webseiten dynamischer und kontextbewusster werden.
