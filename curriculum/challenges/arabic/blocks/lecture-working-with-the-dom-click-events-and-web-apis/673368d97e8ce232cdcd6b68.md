# --interactive--

ستحتاج أحيانًا إلى إضافة أو حذف عقد من DOM وهناك بعض واجهات برمجة التطبيقات للويب التي يمكنك استخدامها.

في هذا الدرس، سنغطي طريقتي `appendChild()` و `removeChild()`.

تُستخدم طريقة `appendChild()` لإضافة عقدة إلى نهاية قائمة العقد الفرعية لعقدة الأصل المحددة.

إليك الصيغة النحوية الأساسية لطريقة `appendChild()`:

```js
parentNode.appendChild(newNode);
```

لنلقِ نظرة على مثال لفهم أفضل لكيفية استخدام طريقة `appendChild()`.

في هذا المثال، لدينا عنصر قائمة غير مرتبة له معرف `desserts` داخل HTML. تحتوي هذه القائمة على عنصرين للقائمة هما `Cake` و `Pie`:

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
```

للوصول إلى تلك القائمة داخل ملف JavaScript، يمكننا استخدام طريقة `getElementById()`:

```js
const dessertsList = document.getElementById("desserts");
```

لدينا متغير يسمى `dessertsList` يخزن المرجع إلى عنصر `ul` الذي له المعرف `desserts`.

بعد ذلك نحتاج إلى إنشاء عنصر عنصر قائمة جديد باستخدام طريقة `createElement()`:

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");
```

الكود التالي سينشئ عنصر عنصر قائمة جديد ويخزنه في متغير يسمى `listItem`.

لإضافة تلك العقدة إلى نهاية قائمة العقد الفرعية لعنصر `ul`، يمكننا استخدام طريقة `appendChild()`:

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

dessertsList.appendChild(listItem);
```

إذا شغلت هذا الكود، سترى أن عنصر عنصر القائمة الجديد أُضيف إلى نهاية قائمة العقد الفرعية لعنصر `ul`.

:::interactive_editor

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
<script src="index.js"></script>
```

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

dessertsList.appendChild(listItem);
```

:::

المشكلة أن عنصر عنصر القائمة الجديد فارغ. لإضافة محتوى نصي إلى عنصر عنصر القائمة الجديد، يمكنك استخدام خاصية `textContent`:

:::interactive_editor

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
<script src="index.js"></script>
```

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

listItem.textContent = "Cookies";
dessertsList.appendChild(listItem);
```

:::

الآن ستعرض القائمة `Cake` و `Pie` و `Cookies`.

لحذف عقدة من DOM، يمكنك استخدام طريقة `removeChild()`.

إليك الصيغة النحوية الأساسية لطريقة `removeChild()`:

```js
parentNode.removeChild(childNode);
```

لننظر إلى مثال حيث نريد حذف آخر عنصر فقرة من هذا العنصر `section`:

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
```

يمكننا الوصول إلى عنصر `section` داخل ملف JavaScript باستخدام طريقة `getElementById()`:

```js
const sectionEl = document.getElementById("example-section");
```

بمجرد أن نحصل على المرجع إلى عنصر `section`، يمكننا بعد ذلك الوصول إلى آخر فقرة باستخدام طريقة `querySelector()`:

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");
```

نستخدم فئة مزيفة `:last-of-type` لاختيار آخر عنصر فقرة داخل عنصر `section`.

الآن بعد أن أصبح لدينا عقدة الأصل والعقدة الفرعية، يمكننا حذف آخر عنصر فقرة من عنصر `section` باستخدام طريقة `removeChild()`:

:::interactive_editor

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
<script src="index.js"></script>
```

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");

sectionEl.removeChild(lastParagraph);
```

:::

فمتى قد تحتاج إلى إضافة أو حذف عقد من DOM؟

إذا كنت تعمل مع محتوى ديناميكي، قد تحتاج إلى إضافة أو حذف عقد من DOM. أمثلة من الواقع تشمل عربات التسوق، قوائم المهام، وخلاصات وسائل التواصل الاجتماعي.
