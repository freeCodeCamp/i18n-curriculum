# --explanation--

`Have to` は、何かが必要であることや必須であることを表すために使われます。これは、何かをしなければならないという選択肢がないことを示します。例えば：

- `You have to save your changes before closing the program.` - これは、プログラムを閉じる前に変更を保存する必要があることを意味します。

トムが `I'll have to check again` と言うとき、彼は設定をもう一度確認する必要があると述べており、確信が持てていないことを示しています。
