# --description--

在使用 Python 變數之前，了解資料型別非常重要。資料型別描述變數所持有的值的種類。例如，一個數字、一段文字或一個項目列表。程式語言使用資料型別，以便知道如何儲存和處理不同種類的資訊。

Python 是一種動態型別語言，像 JavaScript 一樣，這表示你不需要明確地宣告變數的型別。語言會根據你指定給變數的資料來判斷該變數的型別。

以下是一些範例：

```python
name = 'John Doe' # Python knows this is a string
age = 25 # Python knows this is an integer
```

這與某些靜態型別語言如 C#、Java 和 C++ 不同，在那些語言中，你必須像這樣為變數宣告型別：

```csharp
string name = 'John Doe'
int age = 25
```

Python 的動態型別特性使得編碼非常快速且更具彈性，但因為型別錯誤僅在程式執行時被偵測，而非在程式編譯時，因此可能導致意外的錯誤。

由於 Python 在你的程式**執行**時決定資料型別，與型別相關的錯誤只有在那時才會被發現。當程式執行時，Python 會逐行執行你的程式碼。如果它遇到某一行期望某個物件以特定方式運作，但該物件無法如此運作，Python 就會停止並顯示錯誤。

相反地，有些語言會在你的程式執行之前**編譯**你的程式。編譯表示電腦事先檢查你的程式碼並準備執行。在這個步驟中，這些語言可以在程式開始之前就捕捉到型別錯誤。

你還不需要知道那些語言。重要的概念很簡單：

- **在 Python 中，型別錯誤會在執行期間顯現出來**，當程式實際執行並使用你的程式碼時。

- **編譯型語言會在編譯階段捕捉型別錯誤**，在程式被允許執行之前。

因此，你可能要等到程式執行到該特定程式碼行時，才會發現 Python 中的型別錯誤。

以下是你在 Python 中最常使用的資料型別：

- 整數：沒有小數的整數，例如 `10` 或 `-5`。

```python
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- 浮點數：帶有小數點的數字，例如 `4.41` 或 `-0.4`。

```python
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- 字串：一連串被單引號或雙引號包圍的字元，例如 `'Hello world!'`。

```python
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- 布林值：一種真或假的型別，以 `True` 或 `False` 表示。

```python
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- 集合：一個無序且唯一元素的群集，例如 `{4, 2, 0}`。
 
```python
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- 字典：一個由大括號包住的鍵值對組成的群集，例如 `{'name': 'John Doe', 'age': 28}`。

```python
my_dictionary_var = {'name': 'Alice', 'age': 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- Tuple：一個不可變的有序群集，括在括號中，如 `(7, 8, 4)`。

```python
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- 範圍：一連串的數字，常用於迴圈，例如 `range(5)`。

```python
my_range_var = range(5)
print('Range:', my_range_var) # Range: range(0, 5)
```

- 列表：一個有序的群集，支援不同的資料型別。
    
```python
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- None：一個特殊值，用來表述沒有值的狀態。

```python
my_none_var = None
print('None:', my_none_var) # None: None
```

要取得變數的資料型別，你可以使用 `type()` 函式：

```python
my_var_1 = 'Hello world'
my_var_2 = 21

print(type(my_var_1)) # <class 'str'>
print(type (my_var_2)) # <class 'int'>
```

以下是本課程涵蓋的所有資料型別，以及它們在終端機中的型別：

```python
my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 5, 8}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 5, 8)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>
```

內建的 `isinstance()` 函式讓你檢查一個變數是否符合特定的資料型別。它接受一個物件和你想要檢查的型別，然後傳回一個布林值。以下是一些範例：

```python
isinstance('Hello world', str) # True
isinstance(True, bool) # True
isinstance(42, int) # True
isinstance('John Doe', int) # False
```
