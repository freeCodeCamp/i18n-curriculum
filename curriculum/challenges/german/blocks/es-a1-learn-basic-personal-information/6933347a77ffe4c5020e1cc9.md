# --description--

In dieser Aufgabe hören Sie Ángelas Reaktion, als sie Estebans Herkunftsland erfährt. Achten Sie auf diese Ausdrücke:

- `Ah` ist ein natürlicher Reaktionslaut. Er zeigt Verständnis oder Erkenntnis.

- `de` wird verwendet, um über Herkunft zu sprechen.

- `Muy bien` ist eine positive Reaktion. Sie zeigt Zustimmung oder Zufriedenheit.
