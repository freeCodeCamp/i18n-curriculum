# --description--

Njia nyingine ya kutumia vichaguzi ni kuviunganisha kama orodha bila kutenganishwa na kitu chochote. Tuseme una HTML ifuatayo:

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection preview">This is where a preview for a post might go.</p>
</div>
```

Una vipengele viwili vyenye darasa la `subsection` ambavyo vina mitindo ya kipekee, lakini vipi kama unataka tu kutumia sheria tofauti kwa kipengele ambacho pia kina `header` kama darasa la pili? Naam, unaweza kuunganisha vichaguzi vyote vya `class` pamoja katika CSS yako kama ifuatavyo:

```css
.subsection.header {
  color: red;
}
```

Kile `.subsection.header` kinachofanya ni kuchagua kipengele chochote chenye darasa la `subsection` na `header` kwa pamoja. Angalia jinsi hakuna nafasi kati ya vichaguzi vya `.subsection` na `.header` `class`. Sintaksia hii inafanya kazi kwa kuunganisha mchanganyiko wowote wa vichaguzi, isipokuwa kuunganisha zaidi ya kichaguzi cha aina moja.

Hii pia inaweza kutumika kuunganisha darasa la vitu na kitambulisho (ID), kama inavyoonyeshwa hapa chini:

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection" id="preview">This is where a preview for a post might go.</p>
</div>
```

Unaweza kuchanganya vipengele viwili hapo juu na ifuatayo:

```css
.subsection.header {
  color: red;
}

.subsection#preview {
  color: blue;
}
```

Kwa ujumla, huwezi kuunganisha zaidi ya kichaguzi cha aina moja kwa sababu kipengele hakiwezi kuwa aina mbili tofauti kwa wakati mmoja. Kwa mfano, kuunganisha vichaguzi viwili vya aina kama `div` na `p` kungeunda kichaguzi `divp`, ambacho hakingefanya kazi kwa sababu kichaguzi kingetafuta kipengele halisi cha `<divp>`, ambacho hakipo.
