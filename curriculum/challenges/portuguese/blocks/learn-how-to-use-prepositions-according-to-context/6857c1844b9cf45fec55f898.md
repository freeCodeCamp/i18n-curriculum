# --explanation--

`Lastly` introduz o item ou ponto final.Por exemplo:

`Lastly, please remember to lock the door.` – Isso mostra que é a instrução final.

`We need more Ethernet cables for the new desks.` – Isso significa que mais cabos de rede são necessários.

`Ended up` significa onde alguém ou algo finalmente estava.Por exemplo:

`The documents ended up on my desk.` – Isso significa que foi onde eles foram encontrados finalmente.
