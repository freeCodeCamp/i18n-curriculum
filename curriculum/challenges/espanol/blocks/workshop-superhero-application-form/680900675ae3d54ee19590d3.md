# --description--

¿Notaste que puedes enviar el formulario sin completar los campos?

Para evitarlo, añade el atributo `disabled` al botón de envío si `heroName`, `realName` o `powerSource` son falsos, o si la longitud de `powers` es `0`.

Ahora, el botón se desactivará si falta algo por completar. Completa el formulario con la información de tu superhéroe y envíalo para unirte a la liga.

Con eso, tu formulario de aplicación de superhéroe está completo.

const { useState } = React;

export const SuperheroForm = () => {

  const powerSourceOptions = [
    'Bitten by a strange creature',
