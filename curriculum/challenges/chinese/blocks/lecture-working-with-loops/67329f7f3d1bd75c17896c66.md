# --interactive--

编程中的循环用于多次重复一个代码块。

循环的一个例子是当你设计一个需要打印项列表的程序时。你可以使用循环来打印列表中的每一项。

另一个例子是当你设计一个游戏并且想要让一个字符在屏幕上移动时。你可以使用一个循环，在每次循环运行时将字符移动一定数量的像素。 

在 JavaScript 中，有几种你可以使用的循环类型。在本课中，我们将介绍 `for` 循环。以下是 `for` 循环的基本语法：

```js
for (initialization; condition; increment or decrement) {
  // code block to be executed
}
```

初始化语句在循环开始前执行。它通常用于初始化计数变量。计数变量是用于跟踪循环运行次数的变量。

条件语句在每次循环迭代之前进行评估。一次迭代是循环的一次单独执行。

如果条件为真，循环内的代码块将被执行。如果条件为假，循环停止，你将继续执行下一个代码块。

循环的最后部分是增量/减量语句。该语句在每次循环迭代后执行。它通常用于增加或减少计数器变量。

:::interactive_editor

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

:::

在上面示例的第一部分，我们初始化计数变量 `i` 为 `0`。在 `for` 循环中使用 `i` 作为计数变量是常见的惯例。

下一部分是查看条件。在这个分支中，条件是查看 `i` 是否小于 `5`。由于 `i` 是 `0`，条件为真，循环内的代码块被执行。

循环内的代码块用于将 `i` 的值记录到控制台。`i` 的值是 `0`，所以控制台将显示值 `0`。

然后执行递增语句。在这个分支中，我们将 `i` 增加了 `1`。所以 `i` 现在是 `1`。

然后我们再次查看条件，即查看 `i` 是否小于 `5`。由于 `i` 现在是 `1`，条件仍然为真，循环内的代码块再次执行。

我们不断重复这个进程，直到条件为假。在这种分支中，当 `i` 是 `5` 时，条件为假，循环停止。

当你使用 `loop` 时，应当小心不要创建一个始终为真的条件。如果你这样做，`loop` 将永远 `Run`，你的 `Program` 会崩溃。这被称为无限循环。

可以创建嵌套的 `for` 循环。嵌套循环是指将一个循环放在另一个循环内部。稍后我们将看到你可能想这样做的示例。

当你需要重复执行某个代码块若干次时，循环在编程中非常有用。尽管刚开始使用 `for` 循环可能会有些棘手，但通过练习你会掌握它。
