# --interactive--

Починаючи працювати з JavaScript, ви швидко зрозумієте, що виконання математичних операцій — це загальне завдання. Хоча базові арифметичні оператори можуть впоратися з простими обчисленнями, JavaScript пропонує вбудований `Math` об’єкт для розв’язання складніших математичних задач.

Цей зручний інструмент надає різноманітні методи, які полегшують виконання складних обчислень і маніпуляції з числами. Давайте розглянемо ці методи і побачимо, як вони можуть спростити ваше програмування.

Метод `Math.random()` генерує випадкове число з плаваючою комою між `0` (включно) і `1` (виключно). Це означає, що можливий вихід може бути `0`, але він ніколи не досягне `1`. Ось приклад роботи з методом `Math.random()`:

:::interactive_editor

```js
const randomNum = Math.random();

console.log(randomNum);
// any number between 0 and 1 – 0 inclusive and 1 exclusive
```

:::

`Math.min()` і `Math.max()` приймають множину чисел і повертають мінімальне та максимальне значення відповідно. Ось приклад роботи з обома цими методами:

:::interactive_editor

```js
const smallest = Math.min(1, 5, 3, 9);
console.log(smallest); // 1

const largest = Math.max(1, 5, 3, 9);
console.log(largest); // 9
```

:::

Перший `console.log()` виведе число `1`, оскільки `1` є найменшим у цьому списку чисел. А другий `console.log()` виведе число `9`, оскільки `9` є найбільшим числом у цьому списку.

Якщо ви хочете округлити числа вгору або вниз до найближчого цілого числа, можна використати методи `Math.ceil()` і `Math.floor()`. Ось приклад роботи з `Math.ceil()`:

:::interactive_editor

```js
console.log(Math.ceil(4.3)); // 5
```

:::

`Math.ceil()` округлить `4.3` вгору до найближчого цілого числа, що в цьому випадку дорівнює `5`. Тепер подивимося, як округлити число вниз:

:::interactive_editor

```js
console.log(Math.floor(4.7)); // 4
```

:::

`Math.floor()` округлить `4.7` вниз до найближчого цілого числа, що в цьому випадку дорівнює `4`. `Math.round()` є гібридом `Math.ceil()` і `Math.floor()`. Він округлює число до найближчого цілого, враховуючи десяткову частину:

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

Отже, якщо десяткова частина менша за `5`, число округлюється вниз. А якщо десяткова частина дорівнює або більша за `5`, число округлюється вгору. Практичне застосування `Math.floor()` і `Math.random()` — це генерація випадкового числа між двома цілими числами. Ось синтаксис для цього:

:::interactive_editor

```js
const max = 10;
const min = 5;
const randomNum = Math.floor(Math.random() * (max - min + 1)) + min;
console.log(randomNum);
```

:::

Генерація випадкового числа між `20` і `1` виглядає так:

:::interactive_editor

```js
const randomNumBtw1And20 = Math.floor(Math.random() * 20) + 1;
console.log(randomNumBtw1And20);
```

:::

Ще одним корисним методом `Math` є метод `Math.trunc()`. `Math.trunc()` видаляє десяткову частину числа, повертаючи лише цілу частину без округлення:

:::interactive_editor

```js
console.log(Math.trunc(2.9)); // 2
console.log(Math.trunc(9.1)); // 9
```

:::

Якщо потрібно отримати квадратний корінь або кубічний корінь числа, можна використати методи `Math.sqrt()` і `Math.cbrt()` відповідно:

:::interactive_editor

```js
console.log(Math.sqrt(81)); // 9
console.log(Math.cbrt(27)); // 3
```

:::

Перша інструкція виведе `9`, бо квадратний корінь з `81` дорівнює `9`, а друга інструкція виведе `3`, бо кубічний корінь з `27` дорівнює `3`. Якщо потрібно отримати абсолютне значення числа, можна використати метод `Math.abs()`:

:::interactive_editor

```js
console.log(Math.abs(-5)); // 5
console.log(Math.abs(5)); // 5
```

:::

`Math.abs()` повертає абсолютне значення числа, перетворюючи від’ємні на додатні. Останнім методом, який ми розглянемо, буде метод `Math.pow()`:

:::interactive_editor

```js
console.log(Math.pow(2, 3)); // 8
console.log(Math.pow(8, 2)); // 64
```

:::

`Math.pow()` приймає два числа і підносить перше до степеня другого. Існує багато інших методів, що належать до об’єкта `Math`, які ви можете дослідити самостійно. Однак це лише кілька з найпоширеніших, які зустрічаються у кодах JavaScript.
