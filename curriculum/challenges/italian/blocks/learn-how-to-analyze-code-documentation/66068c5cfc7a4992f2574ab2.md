# --description--

`Make sense` significa essere comprensibile o logico.

Per esempio, se stai imparando un nuovo linguaggio di programmazione e finalmente capisci come funziona una funzione, potresti dire `Oh, now it makes sense!`. Qui `makes sense` viene usato per esprimere che il concetto di come funziona una funzione nel linguaggio di programmazione è ora chiaro e comprensibile.
