# --description--

연속하는 소수 $p_1 = 19$와 $p_2 = 23$를 생각해보세요. 1219가 $p_1$로 끝나는 숫자 중에서 가장 작으면서 $p_2$로 나누어떨어지는 수임을 확인할 수 있습니다.

사실, $p_1 = 3$과 $p_2 = 5$를 제외하고, 모든 연속하는 소수 쌍 $p_2 > p_1$에 대해, 끝자리가 $p_1$로 이루어지고 $n$이 $p_2$로 나누어떨어지는 $n$ 값이 존재합니다. 이 중 가장 작은 값을 $S$라고 합시다.

$5 ≤ p_1 ≤ 1000000$인 모든 연속하는 소수 쌍에 대해 $\sum{S}$를 구하세요.
