# --explanation--

`user interface`（通常簡稱為 `UI`）是數位產品中視覺和互動的零件——使用者點擊、看到或觸摸的部分。例如：

`The user interface includes the navigation bar and search field。` — 應用程式中使用者看到的部分包含像是選單和搜尋框。

`consume` 意味著使用完某些東西，例如時間、能量或預算。例如：

`Testing often consumes a large part of the schedule.` — 測試通常會佔用專案排程中大量時間。
