# --description--

الجذر لـ $n$، $rad(n)$، هو حاصل ضرب العوامل الأولية المميزة لـ $n$. على سبيل المثال، $504 = 2^3 × 3^2 × 7$، إذن $rad(504) = 2 × 3 × 7 = 42$.

إذا حسبنا $rad(n)$ لكل $1 ≤ n ≤ 10$، ثم رتبناها حسب $rad(n)$، وإذا تساوت قيم الجذر رتبنا حسب $n$، نحصل على:
