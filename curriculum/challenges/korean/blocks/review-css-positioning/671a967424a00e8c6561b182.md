## 플로트 다루기

- **정의**: 플로트는 요소를 페이지의 일반적인 흐름에서 제거하고 컨테이너의 왼쪽 또는 오른쪽에 위치시키는 데 사용됩니다. 이렇게 하면 텍스트가 플로트된 콘텐츠 주위를 감싸게 됩니다.

```css
float: left;
float: right;
```

- **플로트 지우기**: `clear` 속성은 요소가 플로트된 콘텐츠 아래로 이동해야 하는지 결정하는 데 사용됩니다. 여러 개의 플로트 요소가 나란히 쌓여 있을 때 레이아웃에서 겹침이나 붕괴 문제가 발생할 수 있습니다. 이를 해결하기 위해 `clearfix` 해킹 기법이 만들어졌습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="clearfix">
  <div class="box left">Left</div>
  <div class="box right">Right</div>
</div>
```

```css
.box {
  width: 100px;
  height: 60px;
  color: white;
  text-align: center;
  line-height: 60px;
}

.left {
  float: left;
  background: teal;
}

.right {
  float: right;
  background: purple;
}

.clearfix::after {
  content: "";
  display: block;
  clear: both;
}
```

:::

## 정적, 상대 및 절대 위치 지정

- **정적 위치 지정**: 문서의 일반적인 흐름입니다. 요소들은 위에서 아래로, 왼쪽에서 오른쪽으로 차례대로 배치됩니다.
- **상대 위치 지정**: `top`, `left`, `right`, `bottom` 속성을 사용해 요소를 일반 문서 흐름 내에서 위치시킬 수 있습니다. 상대 위치 지정은 요소가 페이지 내 다른 요소와 겹치도록 만들 때도 사용됩니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="relative">Relative Element</div>
```

```css
.relative {
    position: relative;
    top: 30px;
    left: 30px;
}
```

:::

- **절대 위치 지정**: 요소를 일반 문서 흐름에서 제거하여 다른 요소와 독립적으로 동작하게 만듭니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  Parent
  <div class="positioned">Absolute</div>
</div>
```

```css
.parent {
  position: relative;
  height: 120px;
  background: #eee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

## 고정 및 스티키 위치 지정

- **고정 위치 지정**: 요소가 `position: fixed`으로 위치 지정되면 일반 문서 흐름에서 제거되고 뷰포트를 기준으로 배치되어 사용자가 스크롤해도 같은 위치에 고정됩니다. 헤더나 내비게이션 바처럼 항상 보여야 하는 요소에 자주 사용됩니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="navbar">Fixed Navbar</div>
<div style="height: 600px;"></div>
```

```css
.navbar {
  position: fixed; 
  top: 0; 
  width: 100%;
  background: #0077ff;
  color: white;
  padding: 8px; 
}
```

:::

- **스티키 위치 지정**: 상대 위치 지정과 고정 위치 지정의 혼합 형태입니다. 처음에는 요소가 상대적으로 위치하여 문서 흐름 내에 머무르지만, 사용자가 특정 지점까지 스크롤하면 요소가 뷰포트(보통 상단)에 "붙어서" 고정된 것처럼 동작합니다. 스크롤 위치에 따라 고정되는 스티키 내비게이션 바 같은 요소를 만들 때 유용합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Scroll down</p>
<div class="positioned">Sticky Element</div>
<div style="height: 500px;"></div>
```

```css
.positioned {
  position: sticky;
  top: 30px;
  left: 30px;
  background: #4caf50;
  color: white;
  padding: 6px;
}
```

:::

## `z-index` 속성 다루기

- **정의**: CSS의 `z-index` 속성은 페이지에서 겹치는 위치 지정 요소들의 수직 쌓임 순서를 제어하는 데 사용됩니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box box1">1</div>
  <div class="box box2">2</div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

.box {
  position: absolute;
  width: 100px;
  height: 100px;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

.box1 {
  background: lightcoral;
  top: 20px;
  left: 20px;
  z-index: 1;
}

.box2 {
  background: steelblue;
  top: 40px;
  left: 40px;
  z-index: 2;
}
```
