# --description--

セットはPythonの組み込みデータ構造の一つです。セットの核心的な特徴の一つは、重複した値を保存しないことです。もしセットに重複した値を追加しようとすると、そのうちの一つだけが保存されます。

セットはミュータブルで順序付けられていません。つまり、要素は特定の順序で保存されていないため、インデックスやキーを使ってアクセスすることはできません。セットには数値、文字列、タプルのようなイミュータブルなデータ型の値だけを含めることができます。そして、和集合、積集合、差集合、対称差集合などの数学的な集合演算をサポートしています。

セットを定義するには、波括弧の中に要素を書き、カンマで区切るだけです。これは数値のセットの例です：

```python
my_set = {1, 2, 3, 4, 5} 
```

セットを扱う際の一つの特徴は、空のセットを定義する必要がある場合、`set()`関数を使わなければならないことです。もし空の波括弧だけを書いた場合、`{}`のように、Pythonは自動的に辞書を作成します。

```python
set() # Set
{}    # Dictionary
```

セットに要素を追加するには、`.add()`メソッドを使い、新しい要素を引数として渡します：

```python
my_set.add(6)
```

この例では、新しいセットは次のようになります：

```python
{1, 2, 3, 4, 5, 6}
```

すでにセットにある要素を追加しようとすると、一つだけが保持されます。この場合、セットにはすでに数字の5が含まれています：

```python
my_set.add(5)
```

したがって、セットは変わりません：

```python
{1, 2, 3, 4, 5, 6}
```

セットから要素を削除するには、2つの方法があります。`.remove()`メソッドか`.discard()`メソッドを使い、削除したい要素を引数として渡します。

`.remove()`メソッドは要素が見つからない場合に`KeyError`を発生させますが、`.discard()`メソッドは発生させません：

```python
my_set.remove(4)
my_set.discard(4)
```

`.clear()`メソッドはセットのすべての要素を削除します：

```python
my_set.clear()
```

Pythonのセットには、一般的な数学的集合演算を行う強力なメソッドもあります。

`.issubset()`メソッドと`.issuperset()`メソッドは、それぞれあるセットが別のセットの部分集合か超集合かをチェックします。

ここでは、`your_set`が`my_set`の部分集合かどうかをチェックしていますが、`False`です。なぜなら`your_set`のすべての要素が`my_set`に含まれているわけではないからです。

また、`my_set`が`your_set`の超集合かどうかもチェックしています。これも`False`です。なぜなら`my_set`は`your_set`のすべての要素を持っていないからです：

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

print(your_set.issubset(my_set)) # False
print(my_set.issuperset(your_set)) # False
```

`.isdisjoint()`メソッドは2つのセットが互いに素かどうかをチェックします。つまり、共通の要素がないかどうかです。この場合は`False`です。なぜなら`my_set`と`your_set`は共通の要素（2、3、4）を持っているからです：

```python
print(my_set.isdisjoint(your_set)) # False
```

和集合演算子`|`は、両方のセットのすべての要素を含む新しいセットを返します：

```python
my_set | your_set # {1, 2, 3, 4, 5, 6}
```

積集合演算子`&`は、両方のセットに共通する要素だけを含む新しいセットを返します：

```python
my_set & your_set # {2, 3, 4}
```

差集合演算子`-`は、最初のセットにあって他のセットにはない要素を含む新しいセットを返します。この例では、数字の1と5は`my_set`にありますが`your_set`にはありません：

```python
my_set - your_set # {1, 5}
```

対称差集合演算子`^`は、最初のセットか2番目のセットのどちらか一方にだけある要素を含む新しいセットを返します。この場合、1と5は`my_set`にあって`your_set`にはないので含まれます。そして数字の6は`your_set`にあって`my_set`にはないので、これも含まれます：

```python
my_set ^ your_set # {1, 5, 6}
```

これらの演算子には、それぞれ等号を付けた複合代入演算子もあります。これらの演算子は、結果のセットを式の最初のセットに自動的に代入します：

```python
|= &= -= ^=
```

例えば、`-=`演算子はセットの差を見つけ、その結果で最初のセットを更新します：

```python
my_set -= your_set
```

これにより、`my_set`は`{1, 5}`に更新されます：

```python
print(my_set) # {1, 5}
```

セットに要素が含まれているかどうかは、`in`演算子でチェックできます。ここでは、5が`my_set`にあるかどうかをチェックしています。結果はブール値の`True`か`False`になります：

```python
print(5 in my_set)
```

これがセットの基本です。セットは、値を特定の順序で保存する必要がなく、かつ一意の値だけを保存したい場合に非常に役立ちます。
