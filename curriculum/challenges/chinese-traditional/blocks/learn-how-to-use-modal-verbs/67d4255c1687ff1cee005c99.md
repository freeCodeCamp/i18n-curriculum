## --text--

哪一句語法正確？

## --answers--

`Everyone must to follow the security guidelines.`

### --feedback--

`Must` 後面接不帶 `to` 的原形動詞。

---

`Everyone must following the security guidelines.`

### --feedback--

`Must` 應該接原形動詞，而非 `-ing` 形式。

---

`Everyone must followed the security guidelines.`

### --feedback--

`Must` 後面接的是動詞原形，而非過去式。

---

`Everyone must follow the security guidelines.`

## --video-solution--

4

# --explanation--

情態動詞後面總是接動詞原形。情態動詞不會接 `to`、`-ing` 或過去式形式。以下是錯誤動詞形式的範例： 

- `Must to follow` 是錯誤的，因為 `must` 接 `to`。

- `Must following` 是錯誤的，因為 `must` 後面接的是 `-ing` 形式。

- `Must followed` 是錯誤的，因為 `followed` 是過去式。
