# --description--

단어 `everyone` 는 그룹 내 모든 사람을 의미합니다. `Every-` 은 그룹 내 모든 사물이나 사람을 하나씩 이야기할 때 사용합니다.

`everyone` 는 많은 사람을 말하는 것처럼 들리지만 단수 동사와 함께 사용한다는 점이 중요합니다. 이는 `every-` 로 시작하는 다른 단어들, 예를 들어 `everything` 와 `everybody` 와 비슷합니다.

예를 들어, `Everyone goes to the party`, `Everything is right` 또는 `Everybody works hard` 라고 말합니다.

`Make sure` 는 무언가가 올바른지 확인하거나 보장하는 것을 의미합니다. 예를 들어, `You make sure the code runs` 는 코드에 오류가 없고 제대로 실행될 수 있는지 확인하는 것을 뜻합니다.

`Follow the rules` 는 규칙에 따라 행동하는 것을 의미합니다. 만약 `use camelCase for variable names` 라는 규칙이 있다면, `follow the rules` 는 변수 이름을 `myVariable` 대신 `myBLANKvariable` 처럼 짓는 것을 뜻합니다.
