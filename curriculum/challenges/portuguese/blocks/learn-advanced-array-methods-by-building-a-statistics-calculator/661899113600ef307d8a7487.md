# --description--

Para obter a mediana de um array com um número ímpar de elementos, você precisará encontrar e retornar o número do meio.

Aqui está como encontrar o número do meio de um array com um número ímpar de elementos:

```js
arr[Math.floor(arr.length / 2)];
```

Aqui está um exemplo mais longo encontrando o número do meio de um array com `5` elementos:

```js
const numbers = [1, 2, 3, 4, 5];
const middleNumber = numbers[Math.floor(numbers.length / 2)];
console.log(middleNumber); // 3
```

A razão pela qual você usa `Math.floor` é porque o resultado de dividir um número ímpar por `2` será um decimal. `Math.floor` arredondará para baixo até o número inteiro mais próximo.

Declare uma variável `oddListMedian` e atribua a ela o resultado de encontrar o número do meio de `testArr1`. Então registre a variável `oddListMedian` no console.

Abra o console para ver o resultado.
