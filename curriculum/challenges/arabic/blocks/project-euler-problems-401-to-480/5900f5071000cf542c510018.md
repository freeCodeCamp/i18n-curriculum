# --description--

لتكن $C$ الدائرة ذات نصف القطر $r$، حيث $x^2 + y^2 = r^2$. نختار نقطتين $P(a, b)$ و $Q(-a, c)$ بحيث يكون الخط المار عبر $P$ و $Q$ مماسًا لـ $C$.

على سبيل المثال، الرباعية $(r, a, b, c) = (2, 6, 2, -7)$ تحقق هذه الخاصية.

لتكن $F(R, X)$ عدد الرباعيات الصحيحة $(r, a, b, c)$ التي تحقق هذه الخاصية، مع $0 < r ≤ R$ و $0 < a ≤ X$.

يمكننا التحقق من أن $F(1, 5) = 10$، و $F(2, 10) = 52$، و $F(10, 100) = 3384$.

احسب $F({10}^8, {10}^9) + F({10}^9, {10}^8)$.
