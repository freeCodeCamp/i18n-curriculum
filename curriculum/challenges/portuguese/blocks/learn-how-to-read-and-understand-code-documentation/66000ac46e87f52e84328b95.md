# --description--

`Interpret` significa tentar entender ou encontrar o significado de algo, como descobrir o que uma palavra difícil significa ou o que alguém está tentando dizer.Por exemplo, `He tried to interpret the instructions as best as he could.`

`Focus` é quando você presta atenção extra a uma coisa específica, ignorando as outras.É como olhar através de uma câmera e ajustá-la para que apenas um objeto esteja claro, enquanto todo o resto está borrado.Por exemplo, `She decided to focus on her math homework to finish it quickly.`
