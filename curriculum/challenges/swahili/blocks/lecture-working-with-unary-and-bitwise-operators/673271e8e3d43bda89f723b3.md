# --interactive--

Operators za unary hufanya kazi kwa operand moja tu ili kutekeleza operesheni kama vile kubadilisha aina, kubadilisha thamani, au kukagua masharti fulani. Tuchunguze baadhi ya operators za unary zinazojulikana na jinsi zinavyofanya kazi.

Operator ya unary plus hubadilisha operand yake kuwa nambari. Ikiwa operand tayari ni nambari, haibadiliki.

:::interactive_editor

```js
const str = '42';
const strToNum = +str;

console.log(strToNum); // 42
console.log(typeof str); // string
console.log(typeof strToNum); // number
```

:::

Unary plus ni muhimu unapotaka kuhakikisha unafanya kazi na thamani ya nambari. Kama unavyoweza kubashiri, kuna operator ya unary negation. Inageuza thamani ya operand. Inafanya kazi kama unary plus, isipokuwa hubadilisha alama.

:::interactive_editor

```js
const str = '42';
const strToNegativeNum = -str;

console.log(strToNegativeNum); // -42
console.log(typeof str); // string
console.log(typeof strToNegativeNum); // number
```

:::

Operator ya logical NOT, inayowakilishwa na alama ya mshangao (`!`), ni operator nyingine ya unary. Hubadilisha thamani ya boolean ya operand yake. Hivyo, ikiwa operand ni `true`, hubadilika kuwa `false`, na ikiwa ni `false`, hubadilika kuwa `true`. 

:::interactive_editor

```js
let isOnline = true;
console.log(!isOnline); // false

let isOffline = false;
console.log(!isOffline); // true
```

:::

Operator ya bitwise NOT ni operator ya unary isiyotumika sana. Inawakilishwa na tilde, `~`, hubadilisha uwakilishi wa binary wa nambari. Kompyuta huhifadhi nambari kwa mfumo wa binary (1s na 0s). Operator ya `~` hubadilisha kila bit, ikimaanisha hubadilisha 1 zote kuwa 0 na 0 zote kuwa 1. Utajifunza zaidi kuhusu binary na bits katika funzo lijalo.

:::interactive_editor

```js
const num = 5; // The binary for 5 is 00000101

console.log(~num); // -6
```

:::

Katika mfano huu, `5` ikawa `-6` kwa sababu kwa kutumia operator ya `~` kwa `5`, unapata `- (5 + 1)`, ambayo ni sawa na `-6` kutokana na uwakilishi wa two's complement. Two's complement ni njia ambayo kompyuta hutumia kuwakilisha nambari hasi katika binary. Huenda usitumie operator ya bitwise NOT mara nyingi isipokuwa ukiwa unafanya kazi za programu za ngazi ya chini kama vile kubadilisha bits moja kwa moja.

Neno kuu la `void` ni operator ya unary inayochambua usemi wa kihesabu na kurudisha `undefined`.

:::interactive_editor

```js
const result = void (2 + 2);

console.log(result); // undefined
```

:::

`void` pia hutumika sana katika viungo vya mtandao ili kuzuia uelekezaji:

```js
<a href="javascript:void(0);">Click Me</a>
```

Mwishowe, kuna operator ya `typeof` ambayo ulijifunza katika mafundisho ya awali. Hii hurudisha aina ya operand yake kama mfuatano wa herufi.

:::interactive_editor

```js
const value = 'Hello world';

console.log(typeof value); // string
```

:::
