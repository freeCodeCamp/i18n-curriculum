# --description--

Шаблон `be`, за яким слідує `-ing` форма дієслова, використовується для опису дій, що відбуваються в цей момент.

Фраза `have trouble with` використовується, щоб висловити складність у роботі з певною проблемою або об’єктом. Наприклад, `She is having trouble with her computer` означає, що вона має складнощі з використанням свого комп’ютера.
