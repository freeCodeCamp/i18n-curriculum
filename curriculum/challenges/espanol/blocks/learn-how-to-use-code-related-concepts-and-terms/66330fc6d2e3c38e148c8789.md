# --description--

Ya has aprendido sobre pronombres demostrativos como `this`, `that`, `these` y `those`. Recuerda usarlos para referirte a cosas o personas específicas, considerando su proximidad en el tiempo o espacio.

El caso genitivo, a menudo indicado por un apóstrofe y `s` (como `function's`), se usa para mostrar posesión o pertenencia, típicamente para personas u objetos. Le indica a la gente que algo pertenece a o es parte de algo más.

La palabra `purpose` se refiere a la razón por la que algo se hace o utiliza, o por qué algo existe. Por ejemplo, `The purpose of a clock is to tell time.`
