# --description--

Корінь або нуль полінома $P(x)$ — це розв’язок рівняння $P(x) = 0$.

Визначимо $P_n$ як поліном, коефіцієнтами якого є цифри числа $n$.

Наприклад, $P_{5703}(x) = 5x^3 + 7x^2 + 3$.

Можемо побачити, що:

- $P_n(0)$ — це остання цифра числа $n$,
- $P_n(1)$ — сума цифр числа $n$,
- $P_n(10)$ — саме число $n$.

Визначимо $Z(k)$ як кількість натуральних чисел $n$, що не перевищують $k$, для яких поліном $P_n$ має принаймні один цілий корінь.

Можна перевірити, що $Z(100\\,000)$ дорівнює 14696.

Яке значення $Z({10}^{16})$?
