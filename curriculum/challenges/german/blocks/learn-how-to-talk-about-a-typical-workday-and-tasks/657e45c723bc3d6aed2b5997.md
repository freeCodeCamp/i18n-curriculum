# --description--

`Essential` bedeutet etwas, das absolut notwendig oder äußerst wichtig ist. Zum Beispiel `Regular training is essential for team development`.

Andererseits bezeichnet `optional` etwas, das Sie wählen können. Zum Beispiel impliziert `attending the social event is optional`, dass es eine Wahl und keine Verpflichtung ist.
