# --description--

لكل عدد صحيح $n > 1$، تُعرّف عائلة الدوال $f_{n, a, b}$ كما يلي:

$f_{n, a, b}(x) ≡ ax + b\bmod n$ حيث $a, b, x$ أعداد صحيحة و $0 \lt a \lt n$، $0 \le b \lt n$، $0 \le x \lt n$.

نسمي $f_{n, a, b}$ انسحابًا إذا تحقق $f_{n, a, b}(f_{n, a, b}(x)) \equiv f_{n, a, b}(x)\bmod n$ لكل $0 \le x \lt n$.

لتكن $R(n)$ عدد الانسحابات لـ $n$.

$F(N) = \displaystyle\sum_{n = 1}^N R(n^4 + 4)$.

$F(1024) = 77\\,532\\,377\\,300\\,600$.

احسب $F({10}^7)$. أجب عن نتيجتك بباقي القسمة على $1\\,000\\,000\\,007$.
