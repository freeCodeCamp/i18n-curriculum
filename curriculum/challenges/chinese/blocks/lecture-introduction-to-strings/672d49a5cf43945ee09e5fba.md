# --interactive--

在 JavaScript 中，字符串是用来表示文本数据的一串字符。 字符串与数值、布尔、`null` 和 `undefined` 一样，是编程语言中的原始数据类型之一。

要在 JavaScript 中创建字符串，可以使用单引号（`'`）或双引号（`"`）。

下面是创建两个保存字符串值的变量的示例：

:::interactive_editor

```js
let singleQuotes = 'This is a string';
console.log(singleQuotes);
let doubleQuotes = "This is also a string";
console.log(doubleQuotes);
```

:::

尽管你既可以使用单引号也可以使用双引号来创建字符串，但重要的是要保持一致。 如果字符串以单引号开头，也必须以单引号结尾。

双引号也是如此。 下面的示例会报错，因为它以双引号开始，以单引号结束：

```js
const improperStr = "Do not do this';
```

另一个需要考虑的因素是字符串是不可变的。 在程序设计中，不可变性意味着一旦创建了某种东西，就无法更改。 因此，创建字符串后，不能直接更改其字符。 相反，如果要进行更改，你可以创建一个新字符串。

下面是一个将新字符串赋值给 `developer` 变量的示例：

:::interactive_editor

```js
let developer = "Jessica";
console.log(developer);
developer = "Quincy";
console.log(developer);
```

:::

由于字符串是不可变的，所以我们不能直接更新第一个字符串。 这就是为什么我们要为 `developer` 变量分配一个新字符串。

字串是编程的重要组成部分，在后续课程中，你将学习高级的操作技巧，充分利用它们的潜力来创建动态且交互式的应用。
