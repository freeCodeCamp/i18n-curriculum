`Real-time alerts`는 어떤 일이 발생하자마자 즉시 나타나는 메시지나 경고입니다. 예를 들어:

`The app sends real-time alerts if someone logs in from a new location.` – 이는 누군가 다른 장소에서 로그인할 때마다 사용자가 즉시 경고 메시지를 받는다는 뜻입니다.

`Graphics`는 정보를 전달하는 데 사용되는 이미지나 시각적 요소입니다. 예를 들어:

`The dashboard uses graphics to show project progress.` – 이는 차트나 아이콘 같은 시각 도구를 사용해 데이터를 더 쉽게 이해하도록 만든다는 뜻입니다.
