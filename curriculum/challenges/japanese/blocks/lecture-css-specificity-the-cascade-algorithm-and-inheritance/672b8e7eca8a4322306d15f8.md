# --interactive--

ユニバーサルセレクター（`*`）は、ドキュメント内のあらゆる要素にマッチする特別な種類のCSSセレクターです。

これは、ページ上のすべての要素にスタイルを適用するためによく使われ、異なるブラウザー間でスタイルをリセットまたは正規化するのに役立ちます。

ユニバーサルセレクターは、特定のコンテキスト内のすべての要素や、ドキュメント全体に対してグローバルに使用できます。

以下は、ユニバーサルセレクターを使ってHTMLドキュメント全体の`margin`と`padding`を設定する例です：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1>Heading element</h1>
<p>example paragraph element</p>
```

```css
* {
  margin: 0;
  padding: 0;
}
```

:::

このコード例では、`*`セレクターがすべての要素のマージンとパディングをゼロにリセットしており、これはCSSリセットでよく使われる手法です。

ユニバーサルセレクターは、どのセレクターよりも特異性の値が最も低く、特異性のすべての部分に0を寄与します `(0, 0, 0, 0)`。

つまり、型セレクター、クラスセレクター、IDセレクター、インラインスタイルを含む他のどのセレクターも、ユニバーサルセレクターで設定されたスタイルをオーバーライドします。

次のHTMLとCSSの例を見てみましょう：

:::interactive_editor

```html
<head>
  <style>
    * {
      color: blue;
    }
    p {
      color: red;
    }
    .highlight {
      color: green;
    }
    #unique {
      color: purple;
    }
  </style>
</head>
<body>
  <p id="unique" class="highlight">This text has multiple styles applied.</p>
</body>
```

:::

ユニバーサル`*`セレクターは、すべての要素のテキスト色を`blue`に設定します。

しかし、`p`型セレクターは、`red`要素に対してテキスト色を`p`に設定することでこれをオーバーライドします。

要素に`highlight`クラスがある場合、クラスセレクターが優先され、テキスト色を`green`に変更します。

最後に、最も特異性が高い`#unique`IDセレクターが他のすべてをオーバーライドし、テキスト色を`purple`に設定します。
