# --description--

$R(M, N)$ を、$M \lt x \le N$、$M \lt y \le N$ を満たし、かつ $\left\lfloor\frac{y^2}{x^2}\right\rfloor$ が奇数である格子点 ($x$, $y$) の数とします。

$R(0, 100) = 3\,019$ および $R(100, 10\,000) = 29\,750\,422$ であることが確認できます。

$R(2 \times {10}^6, {10}^9)$ を求めてください。

**注意:** $\lfloor x\rfloor$ は床関数を表します。
