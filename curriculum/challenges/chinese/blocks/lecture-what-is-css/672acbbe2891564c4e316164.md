# --interactive--

CSS 负责网页的样式。所有这些样式都是由各种 CSS 规则组成的。

一个 CSS 规则由两个主要部分组成：一个 **选择器** 和一个 **声明** 块。

让我们来看一下基本语法：

```css
selector {
  property: value;
}
```

选择器是 CSS 中使用的一种模式，用于识别和针对特定 HTML 元素设计样式。

选择器的示例包括类型选择器、类选择器和 ID 选择器。

基本语法中提供的括弧状的花括弧称为声明块。声明块为给定的选择器或选择器集合应用一组样式。

在声明块内，你将有一系列声明。每个声明由一个属性和一个值组成。

该属性是指定正在设置样式的特色的 CSS 标识符。属性的一个例子是 `background-color` 属性。

该值是应用于该属性的具体设置。例如，如果属性是 `background-color`，值可以是 `purple`，这将把背景颜色设置为紫色。

在每个属性名称后，你需要放置一个冒号，在每个值后，你应该有一个分号。

既然你已经了解了 CSS 规则的语法，让我们来看一个示例。启用交互式编辑器并点击 `styles.css` 标签（页）以查看 CSS 代码。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Learning about CSS</h1>

<p>Example paragraph element</p>
<p>Another example paragraph element</p>
```

```css
p {
  color: blue;
}
```

:::

在此 CSS 规则中，类型选择器定位页面上的所有段落元素。

在声明块内，有一个声明。

该声明由 `color` 属性组成，值设置为 `blue`。这将把所有段落元素的文本颜色更改为 `blue`。

如果你想将同一组样式应用到多个选择器，可以创建一个选择器列表，每个选择器之间用逗号分隔。

这是一个为多个选择器设置样式的示例（要与示例交互，请启用交互式编辑器）： 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1 id="title">Example heading</h1>
<h2 class="subheading">Example subheading</h2>
<p>This paragraph is not affected by the selector.</p>
```

```css
#title,
.subheading {
  color: navy;
}
```

:::

在这个选择器列表中，有一个 `id` 选择器针对 `id` 值为 `title` 的超文本标记语言元素。所有 `id` 选择器必须以井号 `#` 符号开头。

然后是一个逗号，后面跟着一个 `class` 选择器，该选择器定位所有 `class` 值为 `subheading` 的超文本标记语言元素。所有 `class` 选择器必须以点号 `.` 开头。

整个选择器列表将应用相同的样式，文本颜色设置为 `navy`。
