# --questions--

## --text--

Welcher Satz verwendet die Komparativform korrekt?

## --answers--

`This task is more easy than the previous one.`  

### --feedback--

Das Wort `more` ist überflüssig, weil `easy` sich im Komparativ zu `easier` ändert.

---

`This code runs more faster than before.` 

### --feedback--

`More faster` ist falsch, weil `faster` bereits eine Komparativform ist.

---

`This algorithm is more efficient than the old one.` 

---

`The new update is gooder than the last one.`

### --feedback--

Die korrekte Komparativform von `good` ist `better`.

## --video-solution--

3

# --explanation--

Um den Komparativ zu bilden, fügen Sie `-er` an das Ende von Adjektiven oder Adverbien mit einer Silbe an, während Sie bei längeren am Anfang `more` hinzufügen.

`More efficient` ist korrekt, weil `efficient` ein längeres Adjektiv ist.

Sie werden bemerken, dass jede Option das magische Wort `than` enthält. Es ist eine Konjunktion, die für Vergleiche verwendet wird. Sie folgt typischerweise Adjektiven oder Adverbien im Komparativ und leitet das zweite Element eines Vergleichs ein, um einen Unterschied im Grad, der Menge oder der Präferenz zu zeigen.
