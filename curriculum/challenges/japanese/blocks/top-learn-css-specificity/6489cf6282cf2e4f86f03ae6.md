# --description--

```css
/* rule 1 */
#subsection .list {
  background-color: yellow;
  color: blue;
}

/* rule 2 */
#subsection .main .list {
  color: red;
}
```

この最後の例では、両方のルールがIDセレクターとクラスセレクターを使っているため、どちらのルールももう一方よりも具体的なセレクターを使っていません。カスケードは次に各セレクタータイプの数をチェックします。両方のルールはIDセレクターが1つずつですが、ルール2の方がクラスセレクターが多いため、ルール2の方が特異性が高くなります！

`color: red`の宣言が優先されますが、`background-color: yellow`の宣言も競合する宣言がないため適用されます。
