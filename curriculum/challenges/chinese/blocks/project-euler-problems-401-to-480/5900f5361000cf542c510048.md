# --description--

令 $f(n) = n^2 - 3n - 1$。

设 $p$ 为一个素数。

令 $R(p)$ 为最小的正整数 $n$，使得 $f(n)\bmod p^2 = 0$，如果存在这样的整数 $n$，否则 $R(p) = 0$。

令 $SR(L)$ 为所有不超过 $L$ 的素数的 $\sum R(p)$。

查找 $SR({10}^7)$。
