# --description--

クライアントがチャットメッセージをサーバーに送信し、すべてのクライアントに配信できるようにしましょう！`client.js`ファイルには、メッセージフォームが送信されたときに処理するコードのブロックがすでにあります。

```js
$('form').submit(function() {
  /*logic*/
});
```

フォーム送信のコード内で、`messageToSend`を定義した後、テキストボックスをクリアする`#m`の前にイベントを発火させてください。イベント名は`'chat message'`で、データは`messageToSend`だけにしてください。

```js
socket.emit('chat message', messageToSend);
```

次に、サーバー側では、ソケットで`'chat message'`イベントをリッスンし、データは`message`という名前で受け取るようにしてください。イベントを受信したら、`'chat message'`イベントを`io.emit`を使ってすべてのソケットに送信し、`username`と`message`を含むデータオブジェクトを送ってください。

`client.js`では、`'chat message'`イベントをリッスンし、受信したらユーザー名、コロン、メッセージを含むリストアイテムを`#messages`に追加してください！

これでチャットは完全に機能し、すべてのクライアント間でメッセージが送信されるはずです！

正しくできたと思ったらページを提出してください。エラーが出る場合は、<a href="https://forum.freecodecamp.org/t/advanced-node-and-express/567135#send-and-display-chat-messages-11" target="_blank" rel="noopener noreferrer nofollow">ここまで完成したプロジェクトを確認できます</a>。

# --hints--

サーバーは`'chat message'`をリッスンし、正しく発火させる必要があります。

```js
  const url = new URL("/_api/server.js", code);
  const res = await fetch(url);
  const data = await res.text();
  assert.match(
    data,
    /socket.on.*('|")chat message('|")[^]*io.emit.*('|")chat message('|").*username.*message/s,
    'Your server should listen to the socket for "chat message" then emit to all users "chat message" with name and message in the data object'
  );
```

クライアントは`'chat message'`イベントからの新しいデータを正しく処理し表示してください。

```js
  const url = new URL("/public/client.js", code);
  const res = await fetch(url);
  const data = await res.text();
  assert.match(
    data,
    /socket.on.*('|")chat message('|")[^]*messages.*li/s,
    'You should append a list item to #messages on your client within the "chat message" event listener to display the new message'
  );
```
