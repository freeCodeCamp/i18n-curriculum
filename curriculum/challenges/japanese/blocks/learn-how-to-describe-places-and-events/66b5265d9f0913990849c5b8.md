# --explanation--

`To touch on` は、話題に簡単に触れたり扱ったりすることを意味します。詳細に踏み込まずに話題を扱う場合によく使われます。例えば：

- `The speaker touched on the new software update during the meeting.` - 話者はソフトウェアのアップデートに簡単に触れましたが、詳しくは説明しませんでした。
