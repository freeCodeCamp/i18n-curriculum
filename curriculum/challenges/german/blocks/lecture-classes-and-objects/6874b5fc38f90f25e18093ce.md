# --description--

In der letzten Lektion haben Sie etwas über Klassen gelernt und wie sie als Blaupausen für die Erstellung von Objekten dienen.

Hier tauchen wir tiefer in Attribute und Methoden ein.

Schauen wir uns zuerst Attribute genauer an, dann Methoden.

Attribute sind Variablen, die zu einem Objekt gehören, also Daten speichern. Es gibt zwei Arten von Attributen: Instanzattribute und Klassenattribute.

Instanzattribute sind für jedes aus einer Klasse erstellte Objekt einzigartig, und Sie setzen sie normalerweise mit der `__init__`-Methode. Klassenattribute hingegen gehören zur Klasse selbst und werden von allen Instanzen dieser Klasse geteilt.

Um auf ein Attribut zuzugreifen, verwenden Sie die Punktnotation.

Hier sind Beispiele für Instanz- und Klassenattribute und wie Sie von Objekten aus auf sie zugreifen:

```python
class Dog:
    species = "French Bulldog" # Class attribute

    def __init__(self, name):
        self.name = name # Instance attribute

print(Dog.species) # French Bulldog

dog1 = Dog("Jack")
print(dog1.name)    # Jack
print(dog1.species) # French Bulldog

dog2 = Dog("Tom")
print(dog2.name)    # Tom
print(dog2.species) # French Bulldog
```

Beachten Sie, dass Sie Klassenattribute direkt von der Klasse aus aufrufen können, aber Sie müssen zuerst ein Objekt erstellen und ihm Daten übergeben, bevor Sie auf Instanzattribute zugreifen können.

Autos sind ein weiteres gutes Beispiel, da alle Autos ein Modell und eine Farbe haben:

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

car_1 = Car("red", "Toyota Corolla")
car_2 = Car("green", "Lamborghini Revuelto")

print(car_1.model) # Toyota Corolla
print(car_2.model) # Lamborghini Revuelto

print(car_1.color) # red
print(car_2.color) # green
```

Methoden sind Funktionen, die innerhalb einer Klasse definiert sind. Mit ihnen kann jedes aus einer Klasse definierte Objekt Aktionen ausführen, die auf seine eigenen Daten wirken oder diese verändern. Sie greifen ebenfalls mit der Punktnotation auf eine Methode zu.

Zum Beispiel können Hunde bellen. Also können wir eine `bark`-Methode in der `Dog`-Klasse haben, wie Sie in einer vorherigen Lektion gesehen haben:

```python
class Dog:
   species = "French Bulldog"

   def __init__(self, name):
     self.name = name

   def bark(self):
       return f"{self.name} says woof woof!"

jack = Dog("Jack")
jill = Dog("Jill")

print(jack.bark()) # Jack says woof woof!
print(jill.bark()) # Jill says woof woof!
```

Eine `Car`-Klasse kann auch eine `describe`-Methode haben:

```python
class Car:
    def __init__(self, color, model):
        self.color = color  # Instance attribute
        self.model = model  # Instance attribute

    def describe(self):
        return f"This car is a {self.color} {self.model}"

car_1 = Car("red", "Toyota Corolla")
car_2 = Car("green", "Lamborghini Revuelto")

print(car_1.describe()) # This car is a red Toyota Corolla
print(car_2.describe()) # This car is a green Lamborghini Revuelto
```
