# --explanation--

`data breach`는 민감한 정보에 권한 없는 사람이 접근할 때 발생합니다. 예를 들어:

- `A data breach can happen if hackers get access to private information.` - 사이버 범죄자가 정보를 훔치면, 이를 데이터 유출이라고 합니다.

- `Companies must act quickly if there is a data breach.` - 데이터 유출이 발생하면, 기업은 신속하게 대응해 문제를 해결해야 합니다.

Jessica는 이 위험을 방지하려면 데이터를 암호화하고 암호화 키를 안전하게 저장하는 것이 필수적이라고 말합니다.
