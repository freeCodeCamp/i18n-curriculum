# --description--

Sema una mfuatano wa herufi mbili ambao ungependa kuziunganisha kuunda ujumbe mmoja, kama ifuatavyo:

```js
const str1 = "Hello";
const str2 = "World!";
const message = str1.push(str2);
```

<img src="https://cdn.statically.io/gh/TheOdinProject/curriculum/4ed59981b4ce2c60b5b83bf7415d3127b61821f5/foundations/javascript_basics/understanding_errors/imgs/03.png" style="width:100%" alt="mfano wa kosa la aina katika konsoli ya msanidi programu wa Chrome">

Hapa, utapata `TypeError` yenye ujumbe unaosema kuwa `str1.push is not a function`. Huu ni ujumbe wa kawaida wa kosa unaomchanganya msanidi programu kwa sababu unaweza kujua kuwa `.push()` hakika ni kitendakazi (kwa mfano, ikiwa umeitumia kuongeza vitu kwenye safu za data hapo awali).

Lakini hapo ndipo panapokuwa tofauti - `.push()` si njia ya String, ni njia ya Safu ya data. Kwa hiyo, si “kitendakazi” unachoweza kupata kama njia ya String. Ukibadilisha `.push()` kuwa `.concat()`, njia sahihi ya String, msimbo wetu utaendeshwa kama ilivyokusudiwa!

Kumbuka vizuri unapotokea na `TypeError` ni kuzingatia aina ya data unayojaribu kuendesha njia au operesheni dhidi yake. Huenda ukagundua kuwa si kile unachodhani, au operesheni au njia hiyo hailingani na aina hiyo.

# --questions-- 

## --text--

Kwa nini msimbo ufuatao wa JavaScript unasababisha `TypeError`?

```js
const str1 = "Hello";
const str2 = "World!";
const message = str1.push(str2);
```

## --answers--

Njia ya `.push()` inaweza kutumika tu kwa vitu, si mfuatano wa herufi.

---

Njia ya `.push()` haipatikani kwa mfuatano wa herufi kwa sababu ni njia ya Safu ya data.

---

Njia ya `.push()` imeandikwa vibaya na inapaswa kuwa .pusch().

---

`TypeError` ni matokeo ya kosa la sintaksia katika msimbo wa JavaScript.
