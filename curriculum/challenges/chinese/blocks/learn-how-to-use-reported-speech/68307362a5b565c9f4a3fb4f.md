# --explanation--

`It was mentioned...` 也是一种 **被动语态** 结构，重点在于所传达的观点，而不是谁说的。例如：

提到该更新提升了安全性。

结构 `it's` + 比较级形容词 + `to` + 动词 用于表示某事物与另一事物相比是如何完成的。例如：

使用新建工具查找更快。——这意味着与旧工具相比，新建工具在查找时节省了时间。
