## さまざまな属性セレクターとリンクの操作

- **定義**: `attribute` セレクターは、`href` や `title` のような属性に基づいてHTML要素をターゲットにできます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Link with href</a>
<a>No href</a>
```

```css
a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

- **`title` 属性**: この属性は要素に関する追加情報を提供します。`title` 属性を持つリンクをターゲットにする方法は次の通りです。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Tooltip">Link with title</a>
<a href="#">Normal link</a>
```

```css
a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

- **`href` と `title` の両方の属性を持つリンクをマッチさせるためにセレクターを組み合わせる**: 複数の属性セレクターを組み合わせます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Info">Href + Title</a>
<a href="#">Only href</a>
```

```css
a[href][title] {
  color: green;
}
```

:::

- **空白で区切られたリスト内の単語をマッチさせる**: 特定のクラス単語を持つリンクをターゲットにします。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a class="link primary">Primary link</a>
<a class="link secondary">Secondary link</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

- **特定の接頭辞で始まる値をマッチさせる**: `https://` で始まるリンクをターゲットにします。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">HTTPS link</a>
<a href="http://www.freecodecamp.org">HTTP link</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

- **特定の接尾辞で終わる値をマッチさせる**: `.jpg` で終わるリンクをターゲットにします。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="photo.jpg">Image link</a>
<a href="index.html">HTML link</a>
```

```css
a[href$=".jpg"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

- **値のどこかに部分文字列を含むものをマッチさせる**: `https` を含むリンクをターゲットにします。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Secure link</a>
<a href="page.html">Local link</a>
```

```css
a[href*="https"] {
  color: teal;
}
```

:::

- **まとめ**: これらのパターンを使うと、属性や値に基づいてリンクを一貫してスタイル設定しやすくなります。

## `lang` と `data-lang` 属性を持つ要素のターゲット

- **`lang` 属性**: この属性はHTMLで要素内のコンテンツの言語を指定するために使われます。多言語サイトでは、書かれている言語に応じて要素のスタイルを変えたい場合があります。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">English text</p>
<p lang="fr">French text</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

- **`data-lang` 属性**: `data-lang` のようなカスタムデータ属性は、要素内に追加情報を保存するためによく使われます。例えば、特定のテキストセクションで使われている言語を指定する場合などです。`data-lang` 属性に基づいて要素をスタイル設定する方法は次の通りです。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div data-lang="fr">French content</div>
<div data-lang="en">English content</div>
```

```css
div[data-lang="fr"] {
  color: blue;
}
```

:::

## 属性セレクター、番号付きリスト要素、`type` 属性の操作

- **`type` 属性**: HTMLの番号付きリストを扱う際、`type` 属性は数字、アルファベット、ローマ数字などの番号付けスタイルを指定できます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Alpha</li>
  <li>Beta</li>
</ol>

<ol type="i">
  <li>One</li>
  <li>Two</li>
</ol>
```

```css
/*Example targeting uppercase alphabetical numbering*/
ol[type="A"] {
  color: purple;
  font-weight: bold;
}

/*Example targeting lowercase Roman numerals*/
ol[type="i"] {
  color: green;
}
```
