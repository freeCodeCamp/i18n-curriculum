# --explanation--

동사 `load`는 소프트웨어나 시스템 같은 것을 시작하거나 작동시키는 과정을 의미합니다. 이 문맥에서 Sarah는 하드 드라이브가 고장 나면 운영 체계가 시작되거나 접근할 수 없게 된다고 설명합니다. 예를 들어:

- `The program won't load because there's an error`는 프로그램이 시작할 수 없다는 뜻입니다.

- `The page took a long time to load`는 화면에 내용이 나타나는 데 걸리는 시간을 의미합니다.
