# --interactive--

在设计网站风格时，使用 HSL 色彩模型是最通用、最直观的色彩处理方法之一。 HSL 代表色相、饱和度和亮度——是定义颜色的三个关键要素。 

HSL 颜色模型以更符合人类感知颜色的方式表现颜色。在本课中，我们将探讨什么是 HSL 颜色模型以及如何在 CSS 中使用 `hsl()` 函数。

色相是颜色类型，用色轮上的角度表示，范围从 `0` 到 `360` 度。 例如，`0` 度代表红色、`120` 度为绿色、而 `240` 度为蓝色。 通过调整色相，你可以在光谱上变换不同的颜色。

饱和度指颜色的强度或纯度。 它是以从 `0%`（完全不饱和的灰度颜色）到 `100%`（该颜色最鲜艳的形式）的百分比来衡量的。 饱和度为 `100%` 的颜色完全饱和，而饱和度为 `0%` 的颜色则显示为灰色。

明度决定颜色的明暗程度，同样以百分比来衡量。 明度值为 `0%` 将产生黑色效果，`50%` 将显示正常色调，`100%` 将显示为白色。

在 CSS 中，`hsl()` 函数用于使用 HSL 颜色模型定义颜色。 这是基本语法：

```css
element {
  color: hsl(hue, saturation, lightness);
}
```

让我们举例说明：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
body {
  background-color: hsla(0, 0%, 1%, 1.00);
}

p {
  color: hsl(120, 100%, 50%);
}
```

:::

在本例中，色相为 `120` 度，绿色。 饱和度为 `100%`，因此绿色完全饱和。 明度为 `50%`，因此它处于正常色调——既不会太暗也不会太亮。 这样，该段的文字颜色就会变成明亮的绿色。

HSL 色彩模型的主要优势之一是其直观性。 通过调整饱和度和明度值，它可以轻松调整颜色的鲜艳度或明度，而无需改变核心颜色（色相）。 

例如，如果你想为同一种颜色创建不同的色调，只需调整明度值即可。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="light">This is a light blue div.</div>
<div class="dark">This is a dark blue div.</div>
```

```css
div.light {
  background-color: hsl(240, 100%, 80%);
}

div.dark {
  background-color: hsl(240, 100%, 20%);
  color: hsl(0, 0%, 100%);
}
```

:::

在这里，两个 `div` 元素使用了相同的色相（`240` 度，即蓝色），但其中一个的明度为 `80%`（较亮的蓝色），而另一个的明度为 `20%`（较暗的蓝色）。

就像 RGB 模型有一个 `rgba()` 函数用于包含透明度一样，HSL 模型也有一个 `hsla()` 函数。 该函数中的第四个参数表示 alpha 值，它控制着颜色的不透明度。 这是基本语法：

```css
element {
  background-color: hsla(hue, saturation, lightness, alpha);
}
```

让我们来看一个例子：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

这段代码会给 `div` 一个半透明的红色背景，其中色相设置为 `0` 度（红色），饱和度为 `100%`，亮度为 `50%`，透明度为 `0.5`（50% 不透明度）。你也可以将透明度指定为百分比——例如，`hsla(0, 100%, 50%, 50%)` 等同于 `hsla(0, 100%, 50%, 0.5)`。

当你需要创建配色方案并轻松调整色调时，HSL 色彩模型尤其有用。 

由于 HSL 将色相与饱和度和明度分开，因此你可以在保持色彩主题一致的同时，通过调整明度来生成阴影或高光效果。 这种灵活性使 HSL 成为设计人员和开发人员的最爱。

此外，与十六进制代码或 RGB 等其他方法相比，HSL 使颜色值更易读、易懂。 你可以更清楚地看到颜色背后的意图，并做出相应的调整，尤其是在尝试不同的设计时。

有了 HSL 色彩模型，你就可以更灵活地控制色相、饱和度和明度，使其成为在 CSS 中创建丰富动态设计的强大工具。
