# --description--

给定整数值 $1 < a_1 < a_2 < \ldots < a_n$，考虑线性组合 $q_1a_1 + q_2a_2 + \ldots + q_na_n = b$，仅使用整数值 $q_k ≥ 0$。

请注意，对于给定的一组 $a_k$，可能并非所有的 $b$ 值都是可能的。例如，如果 $a_1 = 5$ 且 $a_2 = 7$，则不存在满足 $q_1 ≥ 0$ 和 $q_2 ≥ 0$ 的情况，使得 $b$ 可以是 1、2、3、4、6、8、9、11、13、16、18 或 23。

事实上，23 是当 $a_1 = 5$ 和 $a_2 = 7$ 时最大的不可达值。因此我们称 $f(5, 7) = 23$。同样，可以证明 $f(6, 10, 15)=29$ 和 $f(14, 22, 77) = 195$。

查找 $\sum f(pq,pr,qr)$，其中 $p$、$q$ 和 $r$ 是素数且 $p &lt; q &lt; r &lt; 5000$。
