# --description--

$f_5(n)$을 $5^x$가 $n$을 나누는 가장 큰 정수 $x$라고 하겠습니다.

예를 들어, $f_5(625\\,000) = 7$입니다.

$T_5(n)$을 $f_5((2 \times i - 1)!) < 2 \times f_5(i!)$를 만족하는 정수 $i$의 개수로 정의하며, $1 ≤ i ≤ n$입니다.

$T_5({10}^3) = 68$이고 $T_5({10}^9) = 2\\,408\\,210$임을 확인할 수 있습니다.

$T_5({10}^{18})$을 구하세요.
