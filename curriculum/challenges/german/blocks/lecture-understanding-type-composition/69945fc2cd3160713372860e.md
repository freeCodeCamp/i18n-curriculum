# --description--

In der vorherigen Lektion haben Sie gelernt, wie man mit Union-Typen arbeitet. Eine gängige Methode, um mit Union-Typen zu arbeiten, ist die Verwendung eines Type Alias.

Ein Type Alias ist ein Name für einen Typ. Hier ist ein Beispiel, wie man einem Union-Typ einen Namen gibt:

```ts
type ID = number | string;
```

Oft erstellen Sie in Ihren Projekten Type Aliases und verwenden sie im gesamten Programm so:

```ts
type ID = number | string;

type User = {
  id: ID;
  name: string;
};

function getUserId(user: User): ID {
  return user.id;
}
```

Dieses Beispiel verwendet zwei Typen namens `ID` und `User`. `ID` wäre ein Union-Typ für `number` oder `string`, während `User` ein Objekttyp mit den Eigenschaften `id` und `name` ist. Die `id` wird mit dem Typ `ID` angegeben, während die `name` den Typ `string` hat.

Die `getUserId` Funktion hat einen Parameter namens `user`, dessen Typ `User` ist. Der Rückgabetyp der `getUserId` Funktion wäre `ID`.

Type Aliases sind eine großartige Möglichkeit, die Struktur eines Objekts zu beschreiben. Es ist üblich, Objekttypen zu erstellen und sie in Arrays so zu verwenden:

```ts
type User = {
  id: number | string;
  name: string;
};

const users: User[] = [
  { id: 101, name: "Alice" },
  { id: "A102", name: "Bob" },
  { id: 103, name: "Charlie" }
];
```

In diesem Beispiel muss das `users` Array nur mit `User` Objekten gefüllt werden. Wenn Sie eine Eigenschaft in einem der Objekte vergessen, wirft TypeScript einen Fehler wie diesen:

```ts
type User = {
  id: number | string;
  name: string;
};

const users: User[] = [
  // Property 'name' is missing in type '{ id: number; }' but required in type 'User'.
  { id: 101 },
  { id: "A102", name: "Bob" },
  { id: 103, name: "Charlie" }
];
```

Vielleicht ist Ihnen aufgefallen, dass bei der Benennung der Type Aliases PascalCase verwendet wird. Das liegt daran, dass es eine Konvention ist, mit PascalCase Typen von Variablen zu unterscheiden. Es verbessert auch die Lesbarkeit. Wenn Sie `User[]` sehen, wissen Sie, dass es sich um ein Array von Objekten des Typs `User` handelt.
