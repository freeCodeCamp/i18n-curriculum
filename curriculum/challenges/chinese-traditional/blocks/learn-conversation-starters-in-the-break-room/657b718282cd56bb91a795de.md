# --description--

在英文中，有些動詞會變化詞形。這表示它們會根據你談論的人稱改變形式。對於大多數動詞，當你談論另一個人像是 `he`、`she` 或 `it` 時，你會在動詞後面加上 `-s` 或 `-es`。 

例如，如果你談論 Tom 和攝影，你可以說：`Tom takes photographs`，而不是 `Tom take photographs`。這就是你如何在句子中顯示這個動作是由 Tom 在現在式完成的。

對於否定句，你可以使用 `doesn't` 後接動詞原形。你不需要在主要動詞上加 `-s` 或 `-es`，因為 `does` 已經完成變化（`do+es`）。

例如，如果 Tom 不以錄製影片作為興趣，你可以說：`He doesn't record videos`。

理解這個差異對於準確描述習慣和行為至關重要。現在，讓我們將這個理解應用到 Sophie 在她的空閒時間所做的事情上。
