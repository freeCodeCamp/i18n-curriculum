# --description--

`AM`와 `PM`은 하루 중 시간을 나타낼 때 사용합니다. `AM`는 자정부터 정오까지의 시간을 의미합니다(예: `8:00 AM`는 아침 시간입니다). `PM`는 정오부터 자정까지의 시간을 의미합니다(예: `6:00 PM`는 저녁 시간입니다).

영어에서는 `AM`와 `PM`을 사용해 하루를 두 시기로 나눕니다. `AM`은 "정오 이전"이라는 뜻의 라틴어 `Ante Meridiem`를 나타냅니다. 이는 자정(`12:00 AM`)부터 정오(`12:00 PM`)까지의 시간을 가리킵니다. 따라서 `9:00 AM`라고 하면 아침 9시를 의미합니다.

`PM`은 "정오 이후"라는 뜻의 라틴어 `Post Meridiem`를 나타냅니다. 이는 정오(`12:00 PM`)부터 자정 직전(`11:59 PM`)까지의 시간을 가리킵니다. 따라서 `3:00 PM`라고 하면 오후 3시를 의미합니다.
