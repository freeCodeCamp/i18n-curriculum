# --explanation--

El tiempo `Present Continuous` se usa para describir acciones que están ocurriendo ahora mismo. Para expresar que algo **no está** ocurriendo, se usa la forma negativa. 

En la oración `The live preview isn't showing`, `isn't` es una contracción de `is not`, lo que hace que la oración sea negativa. 

Esto indica que la vista previa en vivo no está funcionando como se esperaba. Entonces, Tom está reportando que la vista previa en vivo actualmente no es visible en Visual Studio Code.
