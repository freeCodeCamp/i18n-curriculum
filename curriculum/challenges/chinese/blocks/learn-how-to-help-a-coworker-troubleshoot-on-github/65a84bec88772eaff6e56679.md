# --description--

想象你正在一个项目中工作，并且一直在尝试不同的方案。为了表达这种持续的努力，你可以使用现在完成时态。这个时态将过去的动作与当前时刻连接起来。

现在完成时由辅助动词 `have` 或 `has` 加上主要动词的过去分词形式构成。例如，`I have been working on this for a long time` 或 `he has worked on this for a long time`。

让我们练习使用现在完成时来谈论从过去开始且现在仍然相关的动作。

# --questions--

## --text--

哪个句子正确使用了现在完成时？

## --answers--

`I has tried a few things to solve the issue.`

### --feedback--

`has` 用于代词 `he`、`she` 或 `it`。

---

`I am trying a few things to solve the issue.`

### --feedback--

这个句子使用现在进行时，表示正在发生的动作。

---

`I will try a few things to solve the issue.`

### --feedback--

这个句子使用将来时，表示稍后将发生的动作。

---

`I have tried a few things to solve the issue.`
