## --text--

哪一句語法正確？

## --answers--

`You can explain the project details to me?`

### --feedback--

這句話應該以問題的結構來表達。

---

`Can you explain the project details to me?`

---

`Can explains you the project details?`

### --feedback--

助動詞 `can` 應該接動詞原形，而不是 `explains`。

---

`Do can you explain the project details?`

### --feedback--

`Can` 已經作為助動詞使用，所以在疑問句中不需要 `do`。

# --explanation--

當使用情態動詞構成疑問句時，結構為：情態動詞 + 主詞 + 原形動詞 + （物件）？例如：

- `Can we complete the project by Friday?` - 詢問是否有可能在星期五前完成專案。

- `Should we update the software to the latest version?`－詢問是否應該更新軟體的建議或提議。

在正確答案中，`can` 放在主詞（`you`）之前，動詞（`explain`）則使用原形。
