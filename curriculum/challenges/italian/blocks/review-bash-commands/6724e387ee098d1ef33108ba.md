## Nozioni di base su terminale, shell e riga di comando

- **Riga di comando**: Un'interfaccia testuale dove gli utenti digitano comandi.  
- **Terminale**: L'applicazione che fornisce accesso alla riga di comando.  
- **Emulatore di terminale**: Aggiunge funzionalità extra a un terminale.  
- **Shell**: Interpreta i comandi inseriti nel terminale (es. Bash).  
- **PowerShell / Prompt dei comandi / Microsoft Terminal**: Opzioni per accedere alla riga di comando su Windows.  
- **Terminale (macOS)**: Opzione integrata su macOS, con alternative di terze parti come iTerm o Ghostty.  
- **Terminale (Linux)**: Le opzioni variano a seconda della distribuzione, con molti emulatori di terze parti come kitty.  
- **Terminologia**: Anche se "terminale", "shell" e "riga di comando" sono spesso usati in modo intercambiabile, hanno significati specifici.


## Scorciatoie da tastiera per la riga di comando

- **Freccia su/giù**: Scorri tra i comandi precedenti/successivi nella cronologia.
- **Tab**: Completamento automatico dei comandi.
- **`Control+L`** (Linux/macOS) o digitando `cls` (Windows): Cancella lo schermo del terminale.
- **`Control+C`**: Interrompe un comando in esecuzione (usato anche per copiare in PowerShell se il testo è selezionato).
- **`Control+Z`** (solo Linux/macOS): Sospende un'attività in background; usa `fg` per riprenderla.
- **`!!`**: Riesegue immediatamente l'ultimo comando eseguito.

## Nozioni di base su Bash

- **Bash** (Bourne Again Shell): Shell Unix-like molto diffusa.  
  Comandi chiave:

- **`pwd`**: Mostra la directory corrente.
  - **`cd`**: Cambia directory.
    - **`..`** indica la directory genitore (un livello sopra).
    - **`.`** indica la directory corrente.
  - **`ls`**: Elenca file e cartelle.
    - **`-a`**: Mostra tutti i file, inclusi quelli nascosti.
    - **`-l`**: Mostra informazioni dettagliate sui file.
  - **`less`**: Visualizza il contenuto di un file una pagina alla volta con opzioni di navigazione, incluso lo scorrimento indietro e la ricerca.
  - **`more`**: Mostra il contenuto di un file uno schermo alla volta, con scorrimento indietro limitato e navigazione base.
  - **`cat`**: Mostra tutto il contenuto del file in una volta senza scorrimento o navigazione, utile per file piccoli.
  - **`mkdir`**: Crea una nuova directory.
  - **`rmdir`**: Rimuove una directory vuota.
  - **`touch`**: Crea un nuovo file.
  - **`mv`**: Sposta o rinomina file.
    - Rinomina: `mv oldname.txt newname.txt`
    - Sposta: `mv filename.txt /path/to/target/`
  - **`cp`**: Copia file.
    - **`-r`**: Copia ricorsivamente directory e contenuti.
  - **`rm`**: Elimina file.
    - **`-r`**: Elimina ricorsivamente directory e contenuti.
  - **`echo`**: Mostra una riga di testo o il valore di una variabile.
    - Usa `>` per sovrascrivere il contenuto esistente in un file (es. `echo "text" > file.txt`).
    - Usa `>>` per aggiungere output a un file **senza sovrascrivere il contenuto esistente** (es. `echo "text" >> file.txt`).
  - **`exit`**: Esce dalla sessione del terminale.
  - **`clear`**: Cancella lo schermo del terminale.
  - **`find`**: Cerca file e directory.
    - **`-name`**: Cerca file per pattern nel nome (es. `find . -name "*.txt"`).
  - Usa **`man`** seguito da un comando (es. `man ls`) per accedere a pagine manuali/d'aiuto dettagliate.

## Opzioni e flag dei comandi

- **Opzioni** o **flag**: modificano il comportamento di un comando e di solito sono preceduti da trattini:
  - **Forma lunga (due trattini)**:  
    - Esempio: `--help`, `--version`
    - I valori sono allegati usando un segno di uguale, ad esempio `--width=50`.
  - **Forma corta (un trattino)**:  
    - Esempio: `-a`, `-l`
    - I valori sono passati con uno spazio, ad esempio `-w 50`.
    - Più opzioni corte possono essere concatenate insieme, ad esempio `ls -alh`.
- **`--help`**: puoi sempre usare un comando con questo flag per capire le opzioni disponibili per qualsiasi comando.
