# --description--

Um `challenge` refere-se a uma tarefa ou problema que requer habilidade e esforço para ser resolvido. Em programação, um `coding challenge` é um problema ou tarefa que requer habilidades de codificação para ser concluído.

`Fresh` neste contexto significa novo ou diferente. Pedir uma `fresh perspective` significa buscar um ponto de vista novo ou diferente sobre um problema ou tarefa.
