# --explanation--

`Would` はこの文脈で過去の習慣的な行動や繰り返された行動を表します。これは誰かが定期的に行っていた行動を説明する際によく使われ、`used to` と似ています。

ただし、`would` は行動そのものに焦点を当てるのに対し、`used to` は状態や習慣を表すことができます。例えば：

- `She would always read a book before going to bed.` - 寝る前に本を読むのは彼女のよくある行動でした。

`She used to live in New York before moving here.` - ここで`used to` が使われているのは、住むことが行動ではなく状態や習慣を表すためです。

この文では、マリアが`would` を使ってブライアンが遅くまで残ってコードを微調整するという繰り返された習慣を表現しており、彼の献身を強調しています。
