# --interactive--

`break` 문은 루프를 조기에 종료할 때 사용하고, `continue` 문은 루프의 현재 반복을 건너뛰고 다음 반복으로 넘어갈 때 사용합니다.

다음은 `break` 루프에서 `for` 문을 사용하는 예입니다:

:::interactive_editor

```js
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}
```

:::

위 예제에서 루프는 `0`부터 카운트를 시작하며 `i`가 `10`보다 작을 동안 루프가 계속 실행됩니다.

루프 내부에서 `i`이 `5`과 같은지 확인합니다. 같으면 `break` 문을 사용해 루프를 조기에 종료합니다. 그렇지 않으면 `i`의 값을 콘솔에 출력합니다. 따라서 코드 출력은 `0`, `1`, `2`, `3`, `4` 숫자를 출력합니다.

`break` 문은 특정 조건에 따라 루프를 조기에 종료하고 싶을 때 유용합니다. 예를 들어 배열에서 특정 값을 검색할 때, 값을 찾으면 `break` 문을 사용해 루프를 종료할 수 있습니다.

때로는 루프를 완전히 종료하지 않고 특정 반복만 건너뛰고 싶을 때가 있습니다. 이럴 때 `continue` 문을 사용합니다. 다음은 `continue` 루프에서 `for` 문을 사용하는 예입니다:

:::interactive_editor

```js
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}
```

:::

앞과 마찬가지로 `i`을 `0`로 초기화했고 `i`이 `10`보다 작을 동안 루프가 실행됩니다.

루프 내부에서 `i`가 `5`과 같을 때 `continue` 문을 사용해 현재 반복을 건너뛰고 다음 반복으로 넘어갑니다.

이 코드의 출력은 `0`, `1`, `2`, `3`, `4`, `6`, `7`, `8`, `9` 숫자를 출력합니다. `5` 숫자는 `continue` 문 때문에 건너뛰어집니다.

`break`와 `continue` 문 모두에서 사용할 수 있는 또 다른 기능은 레이블을 사용해 어떤 루프를 종료하거나 계속할지 지정하는 것입니다.

이는 중첩된 루프가 있을 때 내부 루프에서 외부 루프의 흐름을 제어하고 싶을 때 유용합니다.

다음은 `break` 문과 함께 레이블을 사용하는 예입니다:

:::interactive_editor

```js
outerLoop: for (let i = 0; i < 3; i++) {
  innerLoop: for (let j = 0; j < 3; j++) {
    if (i === 1 && j === 1) {
      break outerLoop;
    }
    console.log(`i: ${i}, j: ${j}`);
  }
}
```

:::

이 예제에서는 `for`라는 레이블이 붙은 외부 `outerLoop` 루프와 `for`라는 레이블이 붙은 내부 `innerLoop` 루프가 있습니다.

`i`가 `1`과 같고 `j`가 `1`과 같을 때 `break` 레이블과 함께 `outerLoop` 문을 사용해 외부 루프를 조기에 종료합니다. 이렇게 하면 내부와 외부 루프 모두 종료됩니다.

이 코드의 출력은 콘솔에 다음과 같이 기록됩니다:

```md
"i: 0, j: 0"
"i: 0, j: 1"
"i: 0, j: 2"
"i: 1, j: 0"
```

대부분의 경우 `break`와 `continue` 문에 레이블을 사용할 필요는 없지만, 필요할 때 사용할 수 있다는 점을 알고 있으면 좋습니다.
