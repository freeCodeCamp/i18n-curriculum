# --description--

Sei $f_5(n)$ die größte ganze Zahl $x$, für die $5^x$ $n$ teilt.

Zum Beispiel gilt $f_5(625\\,000) = 7$.

Sei $T_5(n)$ die Anzahl der ganzen Zahlen $i$, die die Bedingung $f_5((2 \times i - 1)!) < 2 \times f_5(i!)$ erfüllen, wobei $1 ≤ i ≤ n$ gilt.

Es lässt sich überprüfen, dass $T_5({10}^3) = 68$ und $T_5({10}^9) = 2\\,408\\,210$ gilt.

Bestimmen Sie $T_5({10}^{18})$.
