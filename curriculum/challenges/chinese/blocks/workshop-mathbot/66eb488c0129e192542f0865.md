# --description--

In the previous lectures, you learned how to work with the `Math.ceil()` method like this:

```js
const price = 10.01;
Math.ceil(price); // 11
```

`Math.ceil()` 方法将数值向上取整为最接近的整数。

创建一个名为 `numRoundedUp` 的变量，将浮点数 `3.2` 向上取整到最接近的整数赋值给它。

然后，将 `numRoundedUp` 变量记录到控制台以查看结果。
