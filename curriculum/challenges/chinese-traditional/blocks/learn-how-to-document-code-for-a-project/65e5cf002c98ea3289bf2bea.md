# --description--

當以 `y` 結尾的名詞變成複數時，`y` 通常會變成 `ies`。這條規則在某些術語經常以 `y` 結尾的技術背景關係中特別有用。

# --questions--

## --text--

以 `y` 結尾的名詞如何變成複數？

## --answers--

將 `y` 改為 `ys`

### --feedback--

這不是一般的規則。通常，`y` 變成 `ies` 來表示複數，如 `query` 變成 `queries`。

---

透過添加 `s`

### --feedback--

對於大多數以 `y` 結尾的名詞，僅僅添加 `s` 是不正確的。 例如，`directory` 變成 `directories`，而不是 `directorys`。

---

將 `y` 改為 `ies`

---

透過移除 `y`

### --feedback--

移除 `y` 並不是形成複數的正確方式。`y` 通常會在複數中變成 `ies`，例如 `binary` 變成 `binaries`。
