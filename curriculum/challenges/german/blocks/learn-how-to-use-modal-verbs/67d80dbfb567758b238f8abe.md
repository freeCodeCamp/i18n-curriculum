# --explanation--

Ein `targeted attack` ist ein gezielter Cyberangriff auf eine bestimmte Person, ein Unternehmen oder ein System. Zum Beispiel:

`Hackers launched a targeted attack against the financial sector.` – Das bedeutet, die Angreifer hatten ein konkretes Ziel, anstatt Malware zufällig zu verbreiten.

Er vermutet, dass jemand den Angriff speziell geplant hat, anstatt dass es eine zufällige Cyberbedrohung war.
