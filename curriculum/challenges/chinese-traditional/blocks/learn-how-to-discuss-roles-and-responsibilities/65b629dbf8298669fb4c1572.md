# --description--

字 `everyone` 意味著群組中的每一個人。當逐一談論群組中所有事物或人時，會使用 `Every-`。

重要的是要注意 `everyone` 聽起來像是在談論很多人，但你要搭配單數動詞使用。這與其他以 `every-` 開頭的字詞類似，例如 `everything` 和 `everybody`。

例如，你說 `Everyone goes to the party`、`Everything is right` 或 `Everybody works hard`。

`Make sure` 意味著在繼續之前檢查或確保某事是正確的。 例如，`You make sure the code runs` 意味著你檢查程式碼以確定它沒有錯誤且能正常執行。

「`Follow the rules`」意指遵守規則所說的內容。如果有規則說明「`use camelCase for variable names`」，那麼「`follow the rules`」就是指你將變數命名為 `myVariable`，而不是 `myBLANKvariable`。
