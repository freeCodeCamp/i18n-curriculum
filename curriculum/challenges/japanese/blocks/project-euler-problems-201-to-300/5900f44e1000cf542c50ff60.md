# --description--

数列 1, 1, 1, 3, 5, 9, 17, 31, 57, 105, 193, 355, 653, 1201 ...

は $T_1 = T_2 = T_3 = 1$ および $T_n = T_{n - 1} + T_{n - 2} + T_{n - 3}$ によって定義されます。

この数列のどの項も27で割り切れないことが示されています。実際、27はこの性質を持つ最初の奇数です。

上記の数列のどの項も割り切らない ${124}^{\text{番目}}$ の奇数を見つけてください。
