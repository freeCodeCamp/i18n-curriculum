# --description--

Der Kleiner-als-Operator (`<`) vergleicht die Werte von zwei Zahlen. Wenn die Zahl links kleiner ist als die Zahl rechts, gibt er `true` zurück. Andernfalls gibt er `false` zurück. Wie der Gleichheitsoperator wandelt der Kleiner-als-Operator die Datentypen beim Vergleichen um.

**Beispiele**

```js
2   < 5 // true
'3' < 7 // true
5   < 5 // false
3   < 2 // false
'8' < 4 // false
```

# --instructions--

Fügen Sie den Kleiner-als-Operator in die angegebenen Zeilen ein, sodass die return-Anweisungen sinnvoll sind.
