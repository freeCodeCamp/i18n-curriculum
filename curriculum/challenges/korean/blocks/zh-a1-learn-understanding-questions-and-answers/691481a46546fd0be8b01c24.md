# --explanation--

`UI`는 `user interface`를 의미하며, 이는 사용자가 상호작용하는 제품의 시각적 요소를 가리킵니다. `设计师 (shè jì shī)`는 "디자이너"를 뜻합니다.

함께 `UI 设计师 (shè jì shī)`는 제품의 외관과 레이아웃을 디자인하여 사용하기 쉽고 쾌적하게 만드는 데 집중하는 직업을 의미합니다. 기술 회사에서 매우 흔한 용어이며, 사람들은 이를 중국어로 번역하지 않고 직접 `UI`라고 부릅니다. 예를 들어:

`我是 (wǒ shì) UI 设计师 (shè jì shī)。` – 나는 UI 디자이너입니다.
