# --interactive--

HTML과 CSS 프로젝트에서 속성을 다뤄본 적이 있을 거예요. 그런데 JavaScript 파일에서 HTML 요소에 속성을 추가할 수 있다는 사실을 알고 있었나요?

이번 수업에서는 `setAttribute()` 메서드를 사용해 HTML 요소에 속성을 추가하는 방법을 배울 거예요.

기본 구문은 다음과 같습니다:

```js
setAttribute(attribute, value);
```

`setAttribute()` 메서드를 어떻게 사용하는지 더 잘 이해할 수 있도록 몇 가지 예제를 살펴보겠습니다.

첫 번째 예제에서는 HTML 안에 `p` 요소가 있습니다:

```html
<p id="para">I am a paragraph</p>
```

`class` 속성을 추가하려면 먼저 `p` 메서드를 사용해 `getElementById()` 요소에 접근해야 합니다. 그다음 `setAttribute()` 메서드를 사용해 `class` 속성을 추가하고 값을 `my-class`로 설정할 수 있습니다.

전체 HTML 요소와 그 속성을 로그로 출력하기 위해 `outerHTML`를 사용하고 있습니다.

:::interactive_editor

```html
<p id="para">I am a paragraph</p>
<script src="index.js"></script>
```

```js
const para = document.getElementById("para");
para.setAttribute("class", "my-class");

console.log(`${para.outerHTML}`);
```

:::

HTML 요소에 이미 존재하는 속성이 있다면 `setAttribute()` 메서드를 사용해 그 값을 업데이트할 수 있습니다.

이 예제에서는 `div` 요소에 `class` 속성이 `my-class`로 설정되어 있습니다:

```html
<div class="my-class"></div>
```

`class` 속성 값을 example로 업데이트하려면 다시 `setAttribute()` 메서드를 사용할 수 있습니다:

:::interactive_editor

```html
<div class="my-class"></div>
<script src="index.js"></script>
```

```js
const divEl = document.querySelector(".my-class");
divEl.setAttribute("class", "example");

console.log(`${divEl.outerHTML}`);
```

:::

그렇다면 실제 상황에서 `setAttribute()` 메서드를 언제 사용하면 좋을까요?

동적인 이미지 갤러리를 만든다면, 사용자가 썸네일을 클릭할 때 이미지 요소의 `src` 속성을 업데이트해야 할 수 있습니다.

또 다른 예로는 폼 유효성 검사에서 `required` 또는 `minlength` 같은 특정 속성을 `input` 요소에 추가해야 할 때가 있습니다.
