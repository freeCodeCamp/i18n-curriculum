# --explanation--

Die `Present Simple`-Zeitform bezieht sich oft auf Handlungen oder Situationen, die dauerhaft wahr sind, statt auf vorübergehende Ereignisse. Zum Beispiel:

- `The system updates automatically every night.` – Bezieht sich auf eine regelmäßig stattfindende Routine.

- `The app saves data to the cloud.` – Gibt eine allgemeine Tatsache über die Funktionalität der App an.

`Securely` bedeutet, dass etwas befestigt, positioniert oder so ausgeführt ist, dass Sicherheit oder ordnungsgemäße Funktion gewährleistet sind.

In diesem Satz beschreibt `isn't connected securely` einen Zustand, bei dem das Stromkabel nicht richtig angeschlossen ist, was verhindert, dass der Computer eingeschaltet wird.
