# --explanation--

`To take feedback to someone` bedeutet, Kommentare oder Vorschläge an eine andere Person weiterzugeben, normalerweise an jemanden in einer Entscheidungsposition. Zum Beispiel:

`I'll take your feedback to the manager so we can discuss possible changes.` – Das bedeutet, dass die sprechende Person das Feedback mit ihrem Manager teilen wird.
