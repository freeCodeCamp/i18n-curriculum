## --text--

このコードの条件 `guess != secret_word` は何を意味していますか？

```python
secret_word = "giraffe"
guess = ""

while guess != secret_word:
    guess = input("Enter guess: ")
```

## --answers--

`guess` が `secret_word` と等しくない場合。

---

`guess` が `secret_word` と等しい場合。

---

`guess` が `secret_word` より大きい場合。

---

`guess` が `secret_word` より小さい場合。
