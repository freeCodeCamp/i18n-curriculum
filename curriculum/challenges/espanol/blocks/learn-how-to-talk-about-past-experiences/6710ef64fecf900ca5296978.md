# --explanation--

La `user interface` (UI) se refiere a la parte de un sistema o software con la que los usuarios interactúan directamente, como botones, menús y pantallas. Por ejemplo: 

`The user interface was redesigned to be more intuitive.` - Esto significa que la parte del software con la que el usuario interactúa fue mejorada.

`User feedback` se refiere a opiniones, sugerencias o problemas reportados por los usuarios sobre su experiencia con un producto o servicio. Por ejemplo: 

La empresa mejoró la app basándose en los comentarios de los usuarios.

En este contexto, Linda explica que el equipo se basó en la retroalimentación de los usuarios para hacer mejoras constantes en la interfaz de usuario, asegurando que cumpliera con las necesidades de los usuarios.
