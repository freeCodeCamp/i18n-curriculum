# --explanation--

`Phishing` indica un tipo di truffa online in cui qualcuno cerca di rubare informazioni personali, come password o dati di carte di credito. Non è collegato a `fishing`, anche se suonano simili.

Prova ad ascoltare le parole `phishing attack` nella frase di Jake. Chiariscono che si tratta di un evento dannoso.
