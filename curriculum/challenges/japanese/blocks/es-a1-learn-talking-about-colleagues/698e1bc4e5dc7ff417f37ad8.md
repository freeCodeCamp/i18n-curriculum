# --description--

他の人について話すとき、その人の名前を尋ねるにはこの質問を使えます。

`¿Cómo se llama?` - 彼/彼女の名前は何ですか？

`cómo` は「どのように」という意味です。この単語が質問の最初に来るときは、`o` の文字にアクセント記号を付けることがとても重要です。

`se` は三人称について話していることを示します。

エレナ は エステバン に彼の新しい同僚、ディエゴ について話しています。

# --instructions--

音声を聞いて、エレナ が尋ねている質問を選んでください。
