# --description--

타입 선택자(또는 요소 선택자)는 주어진 요소 타입의 모든 요소를 선택하며, 구문은 단순히 요소 이름입니다:

```html
<!-- index.html -->

<div>Hello, World!</div>
<div>Hello again!</div>
<p>Hi...</p>
<div>Okay, bye.</div>
```

```css
/* styles.css */

div {
  color: white;
}
```

여기서 세 개의 `<div>` 요소가 모두 선택되고, `<p>` 요소는 선택되지 않습니다.
