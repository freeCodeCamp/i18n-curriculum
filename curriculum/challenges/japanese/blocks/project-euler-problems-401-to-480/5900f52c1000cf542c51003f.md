# --description--

関数 $lcm(a, b)$ は $a$ と $b$ の最小公倍数を表します。

$A(n)$ を $1 ≤ i ≤ n$ の範囲での $lcm(n, i)$ の値の平均とします。

例：$A(2) = \frac{2 + 2}{2} = 2$ および $A(10) = \frac{10 + 10 + 30 + 20 + 10 + 30 + 70 + 40 + 90 + 10}{10} = 32$ です。

$S(n) = \sum A(k)$ を $1 ≤ k ≤ n$ の範囲での和とします。

$S(100) = 122\\,726$ です。

$S(99\\,999\\,999\\,019)\bmod 999\\,999\\,017$ を求めてください。
