# --description--

Це один із необхідних проєктів для отримання вашої сертифікації. Для цього проєкту ви створите Bash-скрипт, щоб отримувати інформацію про хімічні елементи з бази даних періодичної таблиці.

# --instructions--

**Важливо:** Після того, як ви пройдете всі тести проєкту, збережіть дамп своєї бази даних у файл `periodic_table.sql`, а також свій файл `element.sh`, щоб ви могли виконати крок 2. Інструкції щодо цього будуть у віртуальній машині.

# --notes--

Необхідні файли: `periodic_table.sql`, `element.sh`
