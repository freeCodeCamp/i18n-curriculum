# --description--

`Class` — це фундаментальна концепція, особливо в `object-oriented programming`. Він виступає як шаблон для створення об’єктів. `Class` визначає властивості (атрибути) та поведінку (`methods`), які можуть мати об’єкти, створені з цього `class`.

Наприклад, клас `Person` може мати властивості, як-от `name` і `age`, а також поведінку, як-от `talk()` та `walk()`.

`Method` — це функція, пов’язана з об’єктом. `Methods` визначають поведінку або дії, які може виконувати об’єкт, створений із `class`. Вони подібні до інструкцій, які кажуть об’єкту, що робити.

Наприклад, у `Calculator class` `method` може бути `add(number1, number2)`, який додає два числа разом.
