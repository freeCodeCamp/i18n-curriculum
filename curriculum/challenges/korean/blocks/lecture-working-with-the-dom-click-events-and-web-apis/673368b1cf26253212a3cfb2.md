# --description--

`querySelectorAll()` 메서드에 대해 배워봅시다. 이 메서드를 사용하면 특정 CSS 선택자와 일치하는 모든 DOM 요소 목록을 가져올 수 있습니다.

`querySelectorAll()` 메서드를 호출하는 방법은 다음과 같습니다:

```js
document.querySelectorAll(selectors);
```

`document` 객체에서 호출하고 CSS 선택자가 포함된 문자열을 인수로 전달하세요. 인수는 유효한 CSS 선택자 문자열이어야 합니다. 그렇지 않으면 `SyntaxError` 예외가 발생합니다.

`querySelectorAll()`는 `NodeList` 객체를 반환하는데, 이는 지정한 CSS 선택자와 일치하는 노드들의 컬렉션입니다.

목록에는 선택자와 일치하는 각 요소마다 하나의 `Element` 객체가 포함됩니다. 일치하는 요소가 없으면 목록은 비어 있습니다.

요소들은 HTML 문서에 나타나는 순서대로 정렬됩니다. 예를 들어, 특정 유형의 모든 요소를 선택할 수 있습니다:

```js
document.querySelectorAll("div");
```

특정 클래스를 가진 모든 요소:

```js
document.querySelectorAll(".rounded");
```

특정 ID를 가진 모든 요소:

```js
document.querySelectorAll("#logo");
```

또는 특정 속성을 가진 모든 요소, 예를 들어 사용자를 특정 URL로 안내하는 모든 링크:

```js
document.querySelectorAll("a[href='https://www.freecodecamp.org/']");
```

더 복잡한 CSS 선택자도 사용할 수 있습니다. 예를 들어, 클래스 `ingredients`를 가진 순서 없는 목록 요소 내의 모든 목록 항목을 선택하는 경우:

```js
document.querySelectorAll("ul.ingredients li");
```

이 메서드는 노드 컬렉션을 반환하므로, 이 컬렉션을 변수에 할당하여 프로그램에서 개별 요소를 다룰 수 있습니다. 일반 구문을 사용한 예시는 다음과 같습니다:

```js
const matches = document.querySelectorAll(selectors); 
```

예를 들어, 세 개의 목록 항목이 있는 이 순서 없는 목록이 있다고 가정해봅시다:

```html
<ul class="ingredients">
  <li>Flour</li>
  <li>Cheese</li>
  <li>Water</li>
</ul>
```

클래스 `querySelectorAll()`를 가진 순서 없는 목록 내의 모든 목록 항목을 선택하기 위해 `ingredients` 메서드를 호출하고 반환값을 `matches` 상수에 할당할 수 있습니다:

```js
const matches = document.querySelectorAll("ul.ingredients li");
```

이렇게 하면 세 개의 노드 컬렉션이 반환됩니다. 각 노드는 목록 항목 요소를 나타냅니다:

```js
// NodeList(3)
{
  0: `<li>Flour</li>`,
  1: `<li>Cheese</li>`,
  2: `<li>Water</li>`,
  length: 3,
}
```

이 컬렉션은 다른 JavaScript 배열과 똑같이 다룰 수 있습니다.

예를 들어, `console.log()`를 사용해 콘솔에 출력할 수 있습니다:

```js
console.log(matches);
```

또한 `length` 속성으로 컬렉션의 길이를 확인할 수 있습니다:

```js
console.log(matches.length);
```

출력은 컬렉션 내 노드 수를 나타내는 정수입니다:

```md
3
```

컬렉션의 개별 요소는 해당 인덱스로 접근할 수 있습니다. 첫 번째 요소는 인덱스 `0`에 있습니다:

```js
console.log(matches[0]);
console.log(matches[1]);
console.log(matches[2]);
```

출력 결과는 다음과 같습니다:

```html
<li>Flour</li>
<li>Cheese</li>
<li>Water</li>
```

요소들이 콘솔에 하나씩 출력됩니다.

`for` 반복문을 사용해 요소들을 순회할 수도 있습니다:

```js
for (let i = 0; i < matches.length; i++) {
  console.log(matches[i]);    
}
```

이 경우 출력은 정확히 동일합니다. 하지만 이 방법은 컬렉션 내 모든 요소를 다룰 때 매우 유용합니다.

JavaScript의 `querySelectorAll()` 메서드는 CSS 선택자를 기반으로 여러 요소를 선택하는 강력한 도구입니다. 이 메서드를 사용하면 웹 페이지 내 특정 요소를 선택하고 필요에 따라 조작할 수 있습니다.
