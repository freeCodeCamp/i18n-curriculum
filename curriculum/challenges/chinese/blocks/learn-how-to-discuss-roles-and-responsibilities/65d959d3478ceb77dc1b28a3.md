# --description--

`Work overtime` 意味着工作时间超过通常的工作调度。例如，`I usually work overtime during project deadlines` 意味着为了满足项目截止日期而加班。

`Unless` 用于表示某事的异常情况或引入使前一语句不成立的条件。它的意思是 `if it's not the case`。例如，`I walk to work unless it rains` 意味着如果不下雨，我就步行上班。

`Urgent` 意味着某事非常重要且需要立即关注。例如，`The project is urgent` 意味着该项目需要立即完成。

在否定句中，`don't/doesn't have to` 表示某事不是必须的。例如，`I don't have to work late` 意味着加班不是必须的。
