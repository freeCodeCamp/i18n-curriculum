# --description--

يعتمد تشفير RSA على الإجراء التالي:

أنشئ عددين أوليين مميزين `p` و `q`.
احسب `n=p*q` و `φ=(p-1)(q-1)`.
ابحث عن عدد صحيح `e` بحيث `1 < e < φ` و `gcd(e,φ) = 1`

الرسالة في هذا النظام هي رقم في الفترة `[0,n-1]`.
يُحوّل النص المراد تشفيره بطريقة ما إلى رسائل (أرقام في الفترة `[0,n-1]`).
لتشفير النص، لكل رسالة `m`، يُحسب c=m<sup>e</sup> mod n.

لفك تشفير النص، يلزم الإجراء التالي: احسب `d` بحيث `ed=1 mod φ`، ثم لكل رسالة مشفرة `c`، احسب m=c<sup>d</sup> mod n.

توجد قيم لـ `e` و `m` بحيث m<sup>e</sup> mod n = m.
نسمي الرسائل `m` التي تحقق m<sup>e</sup> mod n=m رسائل غير مخفية.

مشكلة اختيار `e` هي أنه لا يجب أن يكون هناك عدد كبير جدًا من الرسائل غير المخفية.
على سبيل المثال، لنفترض `p=19` و `q=37`.
حينها `n=19*37=703` و `φ=18*36=648`.
إذا اخترنا `e=181`، رغم أن `gcd(181,648)=1`، يتبين أن جميع الرسائل الممكنة
m `(0≤m≤n-1)` غير مخفية عند حساب m<sup>e</sup> mod n.
لكل اختيار صالح لـ `e` توجد بعض الرسائل غير المخفية.
من المهم أن يكون عدد الرسائل غير المخفية في الحد الأدنى.

لأي قيم معطاة لـ `p` و `q`، جد مجموع كل قيم `e`، حيث `1 < e < φ(p,q)` و `gcd(e,φ)=1`، بحيث يكون عدد الرسائل غير المخفية لهذه القيمة من `e` في الحد الأدنى.

```js
function gcd(a, b) {
    if (b)
        return gcd(b, a % b);
    else
        return a;
}

function RSAEncryption(p, q) {
    let phi = (p - 1) * (q - 1);

    let best = Number.MAX_SAFE_INTEGER;
    let sum = 0;

    for (let e = 0; e < phi; ++e) {
        if (!(gcd(e, phi) == 1))
            continue;

        let msg = (gcd(p - 1, e - 1) + 1) * (gcd(q - 1, e - 1) + 1);

        if (best == msg) {
            sum += e;
        } else if (best > msg) {
            best = msg;
            sum = e;
        }
    }

    return sum;
}
```
