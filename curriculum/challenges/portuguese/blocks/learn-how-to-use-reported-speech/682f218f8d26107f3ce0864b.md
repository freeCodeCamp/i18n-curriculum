# --explanation--

`Of course` é uma forma educada de mostrar concordância ou dizer `yes` sem hesitação. Isso significa que o falante está feliz ou disposto a fazer algo. Por exemplo:

`Of course, I'll help you with the report.` – Isso significa que o falante está feliz em ajudar.

`Will` é frequentemente usado para fazer promessas ou expressar uma decisão no momento da fala. Por exemplo:

`I'll email you the file this afternoon.` – Isso mostra uma promessa de fazer algo em breve.
