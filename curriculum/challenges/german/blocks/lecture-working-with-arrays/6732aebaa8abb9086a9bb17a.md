# --interactive--

In der vorherigen Lektion haben Sie erstmals gelernt, wie man mit Arrays arbeitet und auf verschiedene Elemente in Arrays zugreift. Hier eine Erinnerung, wie Sie auf das zweite Element eines Arrays zugreifen:

:::interactive_editor

```js
const fruits = ["apple", "banana", "cherry"];

console.log(fruits[1]); // "banana"
```

:::

Da Arrays nullbasiert indiziert sind, befindet sich das erste Element am Index `0`, das zweite Element am Index `1` usw. Es ist wichtig zu beachten, dass JavaScript `undefined` zurückgibt, wenn Sie versuchen, auf einen Index zuzugreifen, der im Array nicht existiert.

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
console.log(fruits[3]); // undefined
```

:::

In diesem Beispiel gibt es keinen Index `3` für das `fruits`-Array. Daher zeigt die Konsole `undefined` an. Schauen wir uns nun an, wie man Elemente in einem Array aktualisiert. Sie können ein Element aktualisieren, indem Sie einem bestimmten Index einen neuen Wert zuweisen.

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
fruits[1] = "blueberry";
console.log(fruits); // ["apple", "blueberry", "cherry"]
```

:::

In diesem Beispiel haben wir `banana` durch `blueberry` am Index `1` ersetzt. Mit dieser Methode können Sie jedes Element im Array ändern, solange Sie dessen Index kennen. Sie können auch neue Elemente zu einem Array hinzufügen, indem Sie einem Index einen Wert zuweisen, der noch nicht existiert:

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
fruits[3] = "date";
console.log(fruits); // ["apple", "banana", "cherry", "date"]
```

:::

Seien Sie dabei jedoch vorsichtig. Wenn Sie einem Index einen Wert zuweisen, der viel größer als die aktuelle Länge des Arrays ist, werden für die dazwischenliegenden Indizes undefinierte Elemente erstellt, was zu unerwartetem Verhalten führen kann. Während Sie weiterhin mit JavaScript arbeiten, werden Sie feststellen, dass diese Methoden zum Zugreifen auf und Aktualisieren von Array-Elementen grundlegend für viele Programmieraufgaben sind. Egal, ob Sie eine einfache To-do-Liste erstellen oder komplexe Datenstrukturen verarbeiten – diese Fähigkeiten werden Ihnen von großem Nutzen sein.
