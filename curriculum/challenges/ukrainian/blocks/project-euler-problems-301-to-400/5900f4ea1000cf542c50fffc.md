# --description--

Для простого числа $p$ нехай $S(p) = (\sum (p - k)!)\bmod (p)$ для $1 ≤ k ≤ 5$.

Наприклад, якщо $p = 7$,

$$(7 - 1)! + (7 - 2)! + (7 - 3)! + (7 - 4)! + (7 - 5)! = 6! + 5! + 4! + 3! + 2! = 720 + 120 + 24 + 6 + 2 = 872$$

Оскільки $872\bmod (7) = 4$, то $S(7) = 4$.

Можна перевірити, що $\sum S(p) = 480$ для $5 ≤ p &lt; 100$.

Знайдіть $\sum S(p)$ для $5 ≤ p &lt; {10}^8$.
