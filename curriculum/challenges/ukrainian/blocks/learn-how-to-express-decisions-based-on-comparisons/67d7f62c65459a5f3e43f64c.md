# --explanation--  

`Makes sense` означає, що щось є логічним або зрозумілим. Наприклад:

`Using version control makes sense because it helps track changes in the code.` – Це означає, що контроль версій є логічним вибором для керування змінами коду.  

`So` використовується для введення висновку на основі того, що щойно обговорювалося. Наприклад:

`The server is down, so we need to restart it.` – Це означає, що перезапуск сервера є логічним наступним кроком, бо сервер не працює.

`If` використовується для введення умови, яка впливає на результат. Наприклад:

`If we optimize the database, the application will run faster.` – Це означає, що покращення бази даних призведе до кращої продуктивності застосунку.
