# --description--

你現在知道 11 到 19 的數字是由 `十 (shí)`（10）後面加上個位數字（1–9）組成。相同的邏輯也適用於 21 到 29 的數字。

在中文裡，數字 20 稱為 `二十 (èr shí)`。

要組成 21 到 29 的數字，只要將 `二十 (èr shí)`（20）後面直接加上個位數字（1–9）。

| 數字 | 中文 |
| :--- | :--- |
| 20 | `二十 (èr shí)` |
| 21 | `二十一 (èr shí yī)` |
| 22 | `二十二 (èr shí èr)` |
| 23 | `二十三 (èr shí sān)` |
| 24 | `二十四 (èr shí sì)` |
| 25 | `二十五 (èr shí wǔ)` |
| 26 | `二十六 (èr shí liù)` |
| 27 | `二十七 (èr shí qī)` |
| 28 | `二十八 (èr shí bā)` |
| 29 | `二十九 (èr shí jiǔ)` |
