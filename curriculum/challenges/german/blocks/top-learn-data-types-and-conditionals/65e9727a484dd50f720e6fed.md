# --description--

In jeder Programmiersprache müssen Sie Werte vergleichen, um Entscheidungen zu treffen. Zum Beispiel: Wenn das Wetter sonnig ist, gehen Sie nach draußen, andernfalls bleiben Sie zu Hause. Ein weiteres Beispiel wäre zu prüfen, ob ein Benutzer genug Punkte hat, um in einem Spiel aufzusteigen.

Die `if-else` bedingte Anweisung wird in JavaScript verwendet, um Entscheidungen zu treffen. Sie wird oft mit Vergleichsoperatoren kombiniert, um Werte zu vergleichen und basierend auf den Ergebnissen Entscheidungen zu treffen.

Ein Beispiel für eine if-Anweisung ist:

```javascript
let x = 5;

if (x > 3) {
  console.log("x is greater than 3");
} else {
  console.log("x is less than or equal to 3");
}
```

In diesem Beispiel wird die `x` Variable mit der Zahl `3` unter Verwendung des `>` Operators verglichen. Wenn `x` größer als `3` ist, wird der erste Codeblock ausgeführt. Andernfalls wird der zweite Codeblock ausgeführt.
