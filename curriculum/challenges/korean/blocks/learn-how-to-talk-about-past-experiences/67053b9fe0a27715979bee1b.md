# --explanation--

`Logs`는 일반적으로 시간에 따라 이벤트, 동작, 데이터 등을 문서화한 기록이나 보고서를 뜻합니다. `Logs`는 기술 분야에서 시스템 활동을 추적하는 데 자주 사용됩니다. 예를 들어:

`The server logs showed multiple login attempts.` - 서버에 로그인 시도 기록이 있습니다.

`Error logs`는 특히 시스템이나 애플리케이션 오류 기록을 의미합니다. 이 로그는 무엇이 언제 잘못되었는지에 대한 상세 정보를 제공하여 문제를 파악하는 데 도움을 줍니다. 예를 들어:

`The error logs revealed that the application crashed due to a missing file.` - 문제 기록에서 누락된 파일이 충돌 원인임을 확인할 수 있습니다.
