# --description--

Цей тест перевіряє ваше розуміння того, як описувати місця та події, включно з проєктами, проблемами та робочими ситуаціями.

Щоб пройти тест, потрібно правильно відповісти щонайменше на 18 із 20 запитань нижче.

Прочитайте кожне запитання та виберіть правильну відповідь. Для кожного запитання є лише одна правильна відповідь.

Для доступності доступні транскрипти для учнів із порушеннями слуху; рекомендуємо використовувати їх лише за потреби.
