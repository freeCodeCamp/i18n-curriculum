# --description--

Іноді у нас можуть бути правила, які конфліктують одне з одним, і ми отримуємо несподівані результати. «Але я хотів, щоб ці абзаци були синіми, чому вони червоні, як інші абзаци?!» Хоч це й може дратувати, важливо розуміти, що CSS не робить нічого всупереч вашим бажанням. CSS виконує лише те, що ми йому наказуємо. Одне виключення — це стилі за замовчуванням, які надає браузер. Ці стилі за замовчуванням відрізняються в різних браузерах, і саме тому деякі елементи створюють великий горизонтальний розрив між собою та іншими елементами, або чому кнопки виглядають так, як вони виглядають, незважаючи на те, що ми не писали жодних CSS правил для їх стилізації.

Отже, якщо ви отримуєте несподівану поведінку, це або через ці стилі за замовчуванням, або через нерозуміння того, як працює властивість, або через нерозуміння цієї маленької штуки, яка називається каскадом.

Каскад визначає, які правила насправді застосовуються до вашого HTML. Існують різні фактори, які каскад використовує для цього. Ми розглянемо три з цих факторів, які, сподіваємося, допоможуть вам уникнути тих розчарувань «Я ненавиджу CSS».
