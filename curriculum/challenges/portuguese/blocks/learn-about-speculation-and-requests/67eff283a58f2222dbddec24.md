# --explanation--

`Have you checked...?` é usado quando se pergunta se alguém fez algo em algum momento antes de agora.Por exemplo:

- `Have you checked your email?` - Isso está perguntando se a pessoa já olhou o email em algum momento antes de agora.

- `Have you checked if the server is running?` - Alguém está perguntando se verificaram o status do servidor.
