# --description--

Das `method`-Attribut legt fest, wie Formulardaten an die im `action`-Attribut angegebene URL gesendet werden. Die Formulardaten können über eine `GET`-Anfrage als URL-Parameter (mit `method="get"`) oder über eine `POST`-Anfrage als Daten im Anfragekörper (mit `method="post"`) gesendet werden.

Setzen Sie das `method`-Attribut, um Ihre Formulardaten über eine `POST`-Anfrage zu senden.
