`physical access`

### --feedback--

Questa frase di due parole si riferisce a qualcuno che entra fisicamente in un'area riservata invece di hackerare un sistema da remoto.

# --explanation--

`Physical access` significa entrare in un luogo, come un ufficio o una sala server.

Un `physical access breach` accade quando qualcuno entra in una zona sicura senza permesso. Per esempio:

- `To avoid a physical access breach, we should require ID checks at the entrance.` - Un'azienda può prevenire l'ingresso non autorizzato controllando gli id all'ingresso.

- `A physical access breach can happen if someone enters the server room without permission.` - Un problema di sicurezza fisica può verificarsi se qualcuno entra nella sala server senza autorizzazione.
