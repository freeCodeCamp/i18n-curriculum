# --interactive--

オプショナルチェーン演算子（`?.`）は、JavaScriptでオブジェクトのプロパティに安全にアクセスしたり、メソッドを呼び出したりできる便利なツールです。存在するかどうか心配せずに使える安全ネットのようなものです。

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30
};

console.log(person.name); // "Alice"
console.log(person.job); // undefined
```

:::

この例では、`person.name`が存在するので`Alice`がログに表示されます。しかし`person.job`は存在しないため、`undefined`が返されます。

では、存在しないかもしれないオブジェクトのプロパティにアクセスしたい場合を考えてみましょう。

```js
const person = {
  name: "Alice",
  age: 30
};

console.log(person.address.street); // This will throw an error!
```

この例では`Uncaught TypeError`がスローされます。`person.address`が`undefined`なので、`street`プロパティにアクセスできません。ここでオプショナルチェーン演算子が役立ちます。オプショナルチェーン演算子を使った例は以下の通りです。

:::interactive_editor

```ts
const user = {
  name: "John",
  profile: {
    email: "john@example.com",
    address: {
      street: "123 Main St",
      city: "Somewhere"
    }
  }
};

console.log(user?.profile?.address?.street); // "123 Main St"
console.log(user?.profile?.phone?.number);   // undefined
```

:::

オプショナルチェーン演算子を使うことで、オブジェクト（または`?.`の前の値）が存在し、`null`や`undefined`でない場合にのみ処理を続けるようJavaScriptに指示しています。

もし`?.`の前の値が`null`や`undefined`であれば、JavaScriptはエラーをスローせずに`undefined`を返します。

オプショナルチェーン演算子は、プロパティやメソッドが存在するか確信が持てない場合に最も役立ちます。エラーを防ぎ、コードをより堅牢にします。
