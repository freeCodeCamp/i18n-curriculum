# --explanation--

Ein `coding issue` bezeichnet ein Problem im Software-Code, das Fehler oder unerwartetes Verhalten verursacht. Diese Probleme können Bugs, Logikfehler, fehlende Funktionen oder Leistungsprobleme umfassen. Zum Beispiel:

`The team is investigating a coding issue that prevents users from logging in.` – Das bedeutet, dass ein Problem im Code Anmeldefehler verursacht.
