# --explanation--

`Present Simple` 时态通常指的是一直为真的动作或情况，而不是临时发生的事件。例如：

- `The system updates automatically every night.` - 指的是定期发生的调度例程。

- `The app saves data to the cloud。` - 陈述该应用的通用功能性事实。

`Securely` 意味着某物被固定、定位或以确保安全或正常运行的方式完成。

在这句话中，`isn't connected securely` 指的是电源线未正确连接，导致计算机无法开机的状态。
