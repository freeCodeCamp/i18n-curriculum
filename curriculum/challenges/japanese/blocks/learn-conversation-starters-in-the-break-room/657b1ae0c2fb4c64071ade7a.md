# --description--

何か特定のことについて尋ねたいときは、疑問詞 `what` を使うことができます。`What` は物事について話すときによく使われます。

通常、質問の最初に `what` を置きます。その後に動詞 `to be`（`is` や `are` のような）や他の動詞を続け、さらに主語（`you` や `the team` のような）を置きます。例えば：

- `What is your favorite color?` – 誰かがあなたの一番好きな色を知りたいとき。

- `What does this code do?` - 誰かがコードの役割や結果を知りたいとき。
