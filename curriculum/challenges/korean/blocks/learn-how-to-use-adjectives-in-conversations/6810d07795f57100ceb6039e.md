# --explanation--

`Alright`는 말을 시작하고 누군가의 주의를 끌 때 쓰는 일상적인 표현입니다. 예를 들어:

`Alright, let's get started.` - 시작할 준비가 되었음을 나타냅니다.

`Here's`는 `here is`의 축약형입니다. 무언가를 소개하거나 발표할 때 사용할 수 있습니다. 예를 들어:

`Here's the report you asked for.` - 누군가에게 보고서를 보여주고 있습니다.

`Pitch`는 보통 무언가를 홍보하거나 설명하기 위한 짧은 발표를 의미합니다. 예를 들어:

`She gave a pitch to investors.` - 그녀가 사업 발표를 했다는 뜻입니다.
