# --explanation--

`It seems to be` + verbo na forma `-ing` é usado para descrever algo que aparentemente está acontecendo, embora possa não estar confirmado. Por exemplo:

`The app seems to be running slower today.` – Isso significa que parece que o app está mais lento do que o usual.

`To cause something to` + verbo significa fazer algo acontecer ou desencadear uma ação. Por exemplo:

`Low memory can cause the program to crash.` – Isso significa que a baixa memória é a razão pela qual o programa para de funcionar.
