# --explanation--

`How they look` 是一個以 `how` 開頭的語句，用來介紹一個主題或對象。例如：

`He didn't like how the icons looked on the screen。`－這表示他不喜歡圖示的外觀。

像 `how they look` 這樣的語句稱為**名詞語句**，它們可以作為句子的物件。

在這個背景關係中，`Let's start with how they look` 意味著首先要檢查的是兩個版本在視覺上的呈現方式。
