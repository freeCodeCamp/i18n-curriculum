# --description--

Eine Reihe mit einer Länge von sieben Einheiten hat rote Blöcke mit einer Mindestlänge von drei Einheiten, die darauf platziert sind, sodass zwei rote Blöcke (die unterschiedliche Längen haben dürfen) durch mindestens ein schwarzes Quadrat getrennt sind. Es gibt genau siebzehn Möglichkeiten, dies zu tun.

<img alt="Mögliche Arten, einen Block mit einer Mindestlänge von drei Einheiten auf einer Reihe mit einer Länge von sieben Einheiten zu platzieren" src="https://cdn.freecodecamp.org/curriculum/project-euler/counting-block-combinations-i.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Wie viele Möglichkeiten gibt es, eine Reihe mit einer Länge von fünfzig Einheiten zu füllen?

**Hinweis:** Obwohl das obige Beispiel dies nicht zeigt, ist es im Allgemeinen erlaubt, Blockgrößen zu mischen. Zum Beispiel könnten Sie auf einer Reihe mit einer Länge von acht Einheiten Rot (3), Schwarz (1) und Rot (4) verwenden.
