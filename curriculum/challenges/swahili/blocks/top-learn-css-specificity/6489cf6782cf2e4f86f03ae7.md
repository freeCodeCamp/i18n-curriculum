# --description--

```css
/* rule 1 */
.class.second-class {
  font-size: 12px;
}

/* rule 2 */
.class .second-class {
  font-size: 24px;
}
```

Hapa sheria 1 na sheria 2 zote zina umahiri sawa. Sheria 1 inatumia kichaguzi cha uunganishaji wa mfululizo (hakuna nafasi) na sheria 2 inatumia mchanganyiko wa mzao (nafasi tupu). Lakini sheria zote mbili zina madarasa mawili na alama ya mchanganyiko yenyewe haiongezi umahiri wowote.

```css
/* rule 1 */
.class.second-class {
  font-size: 12px;
}
/* rule 2 */
.class > .second-class {
  font-size: 24px;
}
```

Mfano huu unaonyesha jambo lile lile. Ingawa sheria 2 inatumia mchanganyiko wa mtoto (`>`), hii haibadilishi thamani ya umahiri. Sheria zote mbili bado zina madarasa mawili hivyo zina thamani sawa za umahiri.

*Kumbuka:* Sio kila kitu kinaongeza umahiri

Unapolinganishwa vichaguzi, unaweza kukutana na alama maalum za kichaguzi cha ulimwengu (`*`) pamoja na mchanganyiko (`+`, `~`, `>`, na nafasi tupu). Alama hizi haziongezi umahiri wowote kwao wenyewe.
