# --description--

`Changes` neste contexto significa modificações ou edições feitas no código.

`Descriptive` significa fornecer detalhes sobre como algo parece, sente, soa, cheira ou tem gosto. É um adjetivo usado para fornecer mais informações sobre um substantivo, tornando a descrição mais vívida e clara. Por exemplo, na frase `she wears a beautiful dress`, a palavra `beautiful` é `descriptive` porque ela informa aos ouvintes mais sobre o vestido.

Uma `message` é uma informação ou comunicação que alguém envia para outra pessoa. Pode ser escrito, falado ou até enviado por email ou mensagem.
