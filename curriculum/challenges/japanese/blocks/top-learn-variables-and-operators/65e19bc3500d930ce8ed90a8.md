# --description--

数字はプログラミングの論理の基礎です！実際、基本的な数学を少しも使わない有用なプログラミングのタスクを考えるのは難しいので、数字の仕組みを知ることは非常に重要です。幸いにも、それはかなりシンプルです。

| 演算子 | 説明                         | 例                 |
|--------|------------------------------|--------------------|
| +      | 加算                         | 5 + 3 = 8          |
| -      | 減算                         | 5 - 3 = 2          |
| *      | 乗算                         | 5 * 3 = 15         |
| **     | べき乗                       | 5 ** 3 = 125       |
| /      | 除算                         | 6 / 3 = 2          |
| %      | モジュラス                   | 5 % 3 = 2          |
