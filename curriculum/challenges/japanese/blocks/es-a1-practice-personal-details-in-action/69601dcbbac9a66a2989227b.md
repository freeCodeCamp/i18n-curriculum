# --description--

最近、`¿Quieres que lo deletree?`という「つづりを尋ねる」表現を学びました。もう一つよく使われる質問は次の通りです。

`¿Cómo se deletrea?`

この質問は、単語や名前、メールアドレスが一文字ずつどうつづられているかを知りたいときに使います。

連絡先の確認をする会話でとてもよく使われます。

# --explanation--

エステバンは`¿Cómo se deletrea?`と尋ねているので、つづりを尋ねています。正しい答えは一文字ずつつづりを示すものです。

それをしているのは`j-u-l-i-e-t-a punto c-a-c-e-r-e-s arroba a-n-d-e-s punto com.`だけです。

他の選択肢は名前、国籍、職業を述べる完全な文で、つづりを示していません。
