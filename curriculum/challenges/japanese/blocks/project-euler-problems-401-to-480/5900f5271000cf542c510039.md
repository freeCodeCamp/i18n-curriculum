# --description--

整数が11を含まないとは、その10進展開に1を除く11のべき乗を表す部分文字列が含まれていないことを指します。

例えば、2404と13431は11を含まない整数ですが、911や4121331は含みます。

$E(n)$を$n$番目の正の11を含まない整数とします。例えば、$E(3) = 3$、$E(200) = 213$、$E(500\\,000) = 531\\,563$です。

$E({10}^{18})$を求めてください。
