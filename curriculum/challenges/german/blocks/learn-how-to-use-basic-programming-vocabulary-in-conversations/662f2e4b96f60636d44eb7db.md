# --description--

Eine `class` ist ein grundlegendes Konzept, besonders in `object-oriented programming`. Sie fungiert wie ein Bauplan zur Erstellung von Objekten. Eine `class` definiert die Eigenschaften (Attribute) und Verhaltensweisen (`methods`), die die aus der `class` erstellten Objekte haben können.

Zum Beispiel könnte eine `Person`-Klasse Eigenschaften wie `name` und `age` sowie Verhaltensweisen wie `talk()` und `walk()` besitzen.

Eine `method` ist eine Funktion, die mit einem Objekt verbunden ist. `Methods` definieren die Verhaltensweisen oder Aktionen, die ein aus einer `class` erstelltes Objekt ausführen kann. Sie sind wie Anweisungen, die dem Objekt sagen, was es tun soll.

Zum Beispiel könnte in einer `Calculator class` eine `method` `add(number1, number2)` sein, die zwei Zahlen addiert.
