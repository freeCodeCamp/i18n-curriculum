在使用远程 MCP `server` 时需要考虑哪些事项？

## --answers--

你想要处理信任、数据隐私、授权和认证。

你只需要担心服务器的处理速度和内存使用。

你需要确保服务器在本地运行你的 LLM 的最新版本。

你需要确保服务器有足够的盘空间用于临时日志。
