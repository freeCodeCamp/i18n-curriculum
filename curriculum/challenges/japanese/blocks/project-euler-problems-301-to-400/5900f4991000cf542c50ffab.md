# --description--

Nimは石のヒープを使って遊ぶゲームで、2人のプレイヤーが交互に任意のヒープから任意の数の石を取り除き、石がなくなるまで続けます。

ここでは、3つのヒープを使った通常プレイのNimを考えます。ルールは以下の通りです：

- ゲーム開始時に3つの石のヒープがあります。
- プレイヤーは自分のターンに、任意の1つのヒープから正の数の石を取り除きます。
- 石が残っていないために動けなくなった最初のプレイヤーが負けです。

位置を示す3つのヒープのサイズを($n_1$, $n_2$, $n_3$)とすると、$X(n_1,n_2,n_3)$という単純な関数があります。これは調べるか自分で推測してもよいですが、次の値を返します：

- 完璧な戦略で動く場合、次に動くプレイヤーが最終的に負けるなら0を返す
- 完璧な戦略で動く場合、次に動くプレイヤーが最終的に勝つなら0以外の値を返す

例えば、$X(1, 2, 3) = 0$です。なぜなら、現在のプレイヤーがどんな手を打っても、相手は同じサイズの2つのヒープを残す手で応じられ、その後は相手が現在のプレイヤーの手をすべて鏡のように真似できるため、石がなくなるまで続き、現在のプレイヤーが負けるからです。例を示します：

- 現在のプレイヤーが(1,2,1)に動く
- 相手が(1,0,1)に動く
- 現在のプレイヤーが(0,0,1)に動く
- 相手が(0,0,0)に動き、勝利する

$X(n, 2n, 3n) = 0$となる正の整数$n ≤ 2^{30}$はいくつありますか？
