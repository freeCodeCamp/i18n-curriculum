# --explanation--

استخدام `that` بعد `I think` اختياري في جمل مثل هذه. قول `I think that it's vital we make these updates.` صحيح أيضًا. في المحادثة اليومية، من الشائع حذف `that` لجعل الجملة أكثر سلاسة وسرعة. على سبيل المثال:

`I think that we need more time to finish the project.` – هذا التركيب صحيح. لكن معظم الناس يقولون `I think we need more time…`.
