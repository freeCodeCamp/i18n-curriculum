# --description--

アリは、黒または白に塗られた正方形の規則的なグリッド上を移動します。

アリは常に4つの基本方向（左、右、上、下）のいずれかを向いており、次のルールに従って隣接する正方形へ移動します。

- 黒い正方形の上にいる場合、その正方形の色を白に変え、90°反時計回りに回転して1マス前に進みます。
- 白い正方形の上にいる場合、その正方形の色を黒に変え、90°時計回りに回転して1マス前に進みます。

すべて白のグリッドから始めて、アリが${10}^{18}$回移動した後、黒い正方形はいくつありますか？
