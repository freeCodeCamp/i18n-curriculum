# --description--

正の整数の三つ組 ($a$,$b$,$c$) が次の条件を満たすとき、カルダノ三つ組と呼ばれます：

$$\sqrt[3]{a + b \sqrt{c}} + \sqrt[3]{a - b \sqrt{c}} = 1$$

例えば、(2,1,5) はカルダノ三つ組です。

$a + b + c ≤ 1000$ を満たすカルダノ三つ組は149組存在します。

$a + b + c ≤ 110\\,000\\,000$ を満たすカルダノ三つ組がいくつ存在するか求めてください。
