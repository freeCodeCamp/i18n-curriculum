# --description--

A palavra `must` é usada para expressar uma necessidade ou obrigação forte. Ela é seguida por um verbo para mostrar algo que é muito importante ou necessário fazer.

É semelhante a `have to`, mas `must` é frequentemente usado quando a obrigação vem do falante ou é interna, enquanto `have to` é frequentemente usado para obrigações ou regras externas.

Exemplos:

`You must study for the test` significa que o falante acha muito importante que você estude para o teste.

`I have to go to work at 8 AM` significa que o falante precisa trabalhar às 8 AM por causa do horário de trabalho.

`Talk with` significa ter uma conversa ou discussão com alguém.Por exemplo, `I need to talk with my teacher about the homework` significa que eu preciso ter uma conversa com meu professor sobre o dever de casa.
