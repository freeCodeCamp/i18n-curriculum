# --description--

Spinal case ist ein String-Format, bei dem alle Wörter in Kleinbuchstaben geschrieben und durch Bindestriche getrennt sind. `"this-is-spinal-tap"` ist ein Beispiel für einen String im Spinal Case.

In diesem Labor erstellen Sie eine Funktion, die einen gegebenen String in Spinal Case umwandelt.

**Ziel**: Erfüllen Sie die untenstehenden User Stories und lassen Sie alle Tests erfolgreich durchlaufen, um das Labor abzuschließen.

**User Stories:**

1. Sie sollten eine Funktion mit dem Namen `spinalCase` erstellen.
2. Die `spinalCase`-Funktion sollte ein einzelnes Argument, einen String, entgegennehmen.
3. Die `spinalCase`-Funktion sollte den String im Spinal-Case-Format zurückgeben. Zum Beispiel, wenn das Argument `ProductLanding page` ist, sollte die Funktion `product-landing-page` zurückgeben.
