# --description--

考虑一局 Ramvok 游戏：

令 $t$ 表现游戏持续的最大回合数。如果 $t = 0$，则游戏立即结束。否则，在每个回合 $i$，玩家掷骰子。掷完后，如果 $i &lt; t$，玩家可以选择停止游戏并获得等于当前掷骰值的奖品，或者放弃本次掷骰并在下一回合再试一次。如果 $i = t$，则本次掷骰不能被放弃，必须接受奖品。在游戏开始前，$t$ 由玩家选择，玩家必须为某个常量 $c$ 支付预付费用 $ct$。当 $c = 0$ 时，$t$ 可以选择为无限（预付费用为 0）。令 $R(d, c)$ 为玩家在使用公平的 $d$ 面骰和成本常量 $c$ 的情况下，经过最优玩法后从单局 Ramvok 游戏中获得的期望利润（即净收益）。例如，$R(4, 0.2) = 2.65$。假设玩家有足够资金支付任何/所有预付费用。

现在考虑一个 Super Ramvok `game`：

In Super Ramvok, the game of Ramvok is played repeatedly, but with a slight modification.  After each game, the die is altered. 改变过程如下：模具滚动一次，如果得到的面部有可见的点，则该面被改为空白。 如果面部已经是空白，则会将其更改回原始值。 在进行改变之后，可以开始另一场Ramvok游戏（并且在这样的游戏中，在每个回合中，掷骰子被滚动直到出现具有其值的面部）。 玩家知道哪些面是空白的，哪些面不是。 一旦模具的所有面都空白，超级拉姆沃克的游戏结束。

设 $S(d, c)$ 为玩家在公平的 $d$ 面骰子（所有面均可见）和成本常量 $c$ 下，经过最优玩法获得的 Super Ramvok 游戏的期望收益。例如，$S(6, 1) = 208.3$。

令 $F(n) = \sum_{4 ≤ d ≤ n} \sum_{0 ≤ c ≤ n} S(d, c)$。

计算 $F(20)$，四舍五入到最接近的整数。
