## Python-Listen

- **Einführung**: In Python ist der Listentyp eine geordnete Folge von Elementen, die aus Strings, Zahlen oder sogar anderen Listen bestehen kann. Listen sind veränderbar und nullbasiert indiziert.

```python
cities = ['Los Angeles', 'London', 'Tokyo']
```

- **Zugreifen auf Elemente in einer Liste**: Um auf ein Element aus der `cities` Liste zuzugreifen, können Sie dessen Indexnummer in der Folge referenzieren:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # Los Angeles
```

- **Zugreifen auf Elemente mit negativer Indexierung**: Um auf das letzte Element einer beliebigen Liste zuzugreifen, können Sie `-1` als Indexnummer verwenden:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # Tokyo
```

- Die negative Indexierung wird verwendet, um Elemente vom Ende der Liste aus zuzugreifen, anstatt vom Anfang bei Index `0`.

- **Erstellen von Listen mit dem `list()` Konstruktor**: Listen können auch mit dem `list()` Konstruktor erstellt werden. Der `list()` Konstruktor wird verwendet, um ein iterierbares Objekt in eine Liste umzuwandeln:

```python
developer = 'Jessica'

print(list(developer)) 
# Result: ['J', 'e', 's', 's', 'i', 'c', 'a']
```

- **Ermitteln der Länge einer Liste**: Sie können die `len()` Funktion verwenden, um die Länge einer Liste zu erhalten:

```python
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

- **Veränderbarkeit von Listen**: Listen sind veränderbar, das heißt, Sie können jedes Element in der Liste aktualisieren, solange Sie eine gültige Indexnummer angeben. Um Listen an einem bestimmten Index zu aktualisieren, können Sie diesem Index einen neuen Wert zuweisen:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

- **Index außerhalb des Bereichs Fehler**: Wenn Sie einen Index (entweder positiv oder negativ) angeben, der außerhalb des Bereichs der Liste liegt, erhalten Sie eine `IndexError`:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
"""
```

- **Entfernen von Elementen aus einer Liste**: Elemente können mit dem Schlüsselwort `del` aus einer Liste entfernt werden:

```python
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

- **Überprüfen, ob ein Element in einer Liste vorhanden ist**: Das Schlüsselwort `in` kann verwendet werden, um zu prüfen, ob ein Element in einer Liste vorhanden ist:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **Verschachteln von Listen**: Listen können in anderen Listen verschachtelt sein:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

- Um auf die verschachtelte Liste zuzugreifen, müssen Sie sie mit dem Index `2` ansprechen, da Listen nullbasiert indiziert sind.

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

- Um weiter auf die zweite Sprache aus dieser verschachtelten Liste zuzugreifen, müssen Sie den Index `1` verwenden:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # Rust
```

- **Entpacken von Werten aus einer Liste:** Das Entpacken von Werten aus einer Liste ist eine Technik, um Werte aus einer Liste neuen Variablen zuzuweisen. Hier ist ein Beispiel, um die `developer` Liste in neue Variablen namens `name`, `age` und `job` wie folgt zu entpacken:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer
```

- Wenn die Anzahl der Variablen auf der linken Seite des Zuweisungsoperators nicht mit der Gesamtzahl der Elemente in der Liste übereinstimmt, erhalten Sie eine `ValueError`.

- **Sammeln der verbleibenden Elemente aus einer Liste**: Um alle verbleibenden Elemente aus einer Liste zu sammeln, können Sie den Stern (`*`) Operator wie folgt verwenden:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer
```

- **Slicing von Listen**: Slicing ist das Konzept, einen Teil einer Liste mit dem Slice-Operator `:` zuzugreifen. Um eine Liste zu schneiden, die bei Index `1` beginnt und vor Index `3` endet, können Sie folgende Syntax verwenden:

```python
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie']
desserts[1:3] # ['Cookies', 'Ice Cream']
```

- **Schrittintervalle**: Es ist auch möglich, ein Schrittintervall anzugeben, das bestimmt, wie viel zwischen den Indizes inkrementiert wird. Hier ist ein Beispiel, wenn Sie eine Liste nur mit geraden Zahlen durch Slicing extrahieren möchten:

```python
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

## List-Methoden

- **append()**: Wird verwendet, um ein Element am Ende der Liste hinzuzufügen. Hier ist ein Beispiel für die Verwendung der `append()` Methode, um die Zahl `6` zu dieser `numbers` Liste hinzuzufügen:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **Anhängen von Listen**: Die `append()` Methode kann auch verwendet werden, um eine Liste am Ende einer anderen hinzuzufügen:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

- **extend()**: Wird verwendet, um mehrere Elemente am Ende einer Liste hinzuzufügen. Hier ist ein Beispiel, um die Zahlen `6`, `8` und `10` am Ende der `numbers` Liste hinzuzufügen:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

- **insert()**: Wird verwendet, um ein Element an einem bestimmten Index in der Liste einzufügen. Hier ist ein Beispiel für die Verwendung der `insert()` Methode:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

- **remove():** Wird verwendet, um ein Element aus der Liste zu entfernen. Die `remove()` Methode entfernt nur das erste Vorkommen eines Elements in der Liste:

```py
numbers = [1, 2, 3, 4, 5, 5, 5]
numbers.remove(5)

print(numbers) # [1, 2, 3, 4, 5, 5]

```

- **pop()**: Wird verwendet, um ein bestimmtes Element aus der Liste zu entfernen und zurückzugeben:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

- Wenn Sie kein Element für die `pop` Methode angeben, wird das letzte Element entfernt.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

- **clear()**: Wird verwendet, um alle Elemente aus der Liste zu entfernen:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

- **sort()**: Die `sort()` Methode wird verwendet, um die Elemente an Ort und Stelle zu sortieren. Hier ist ein Beispiel, um eine zufällige Liste von `numbers` an Ort und Stelle zu sortieren:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

- **sorted()**: Eine eingebaute Funktion, die eine neue sortierte Liste zurückgibt, anstatt die ursprüngliche Liste zu verändern:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
print(numbers) # [19, 2, 35, 1, 67, 41]
```

- **reverse()**: Wird verwendet, um die Reihenfolge der Elemente in einer Liste umzukehren:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **index()**: Wird verwendet, um den ersten Index zu finden, an dem ein Element in einer Liste gefunden werden kann:

```python
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

- Wenn das Element mit der `index()` Methode nicht gefunden werden kann, ist das Ergebnis eine `ValueError`.

## Tuples in Python

- **Definition**: Ein Tuple ist ein Python-Datentyp, der verwendet wird, um eine geordnete Folge von Werten zu erstellen. Tuples können eine gemischte Menge von Datentypen enthalten:

```py
developer = ('Alice', 34, 'Rust Developer')
```

- Tuples sind unveränderlich, das heißt, die Elemente im Tuple können nach der Erstellung nicht geändert werden. Wenn Sie versuchen, eines der Elemente im Tuple zu aktualisieren, erhalten Sie eine `TypeError`:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **Zugreifen auf Elemente aus einem Tuple**: Um auf ein Element aus einem Tuple zuzugreifen, verwenden Sie die Klammernotation und die Indexnummer:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

- Negative Indexierung kann verwendet werden, um Elemente vom Ende des Tuples aus zuzugreifen:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- Wenn Sie versuchen, eine Indexnummer anzugeben, die die Länge des Tuples überschreitet oder gleich ist, erhalten Sie eine `IndexError`:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- Ein Tuple kann auch mit dem `tuple()` Konstruktor erstellt werden. Innerhalb des Konstruktors können Sie verschiedene iterierbare Objekte wie Strings, Listen und sogar andere Tuples übergeben.

```python
developer = 'Jessica'

print(tuple(developer)) 
# Result: ('J', 'e', 's', 's', 'i', 'c', 'a')
```

- **Überprüfen von Elementen in einem Tuple**: Um zu prüfen, ob ein Element in einem Tuple vorhanden ist, können Sie das Schlüsselwort `in` wie folgt verwenden:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **Entpacken von Tuples**: Elemente können aus einem Tuple wie folgt entpackt werden:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

- Wenn Sie verbleibende Elemente aus einem Tuple sammeln müssen, können Sie den Stern (`*`) Operator wie folgt verwenden:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer
```

- **Slicing von Tuples**: Slicing kann verwendet werden, um einen Teil eines Tuples zu extrahieren. Zum Beispiel können die Elemente `pie` und `cookies` in ein separates Tuple geschnitten werden:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

- **Entfernen von Elementen aus Tuples**: Das Entfernen eines Elements aus einem Tuple führt zu einer `TypeError`, da Tuples unveränderlich sind:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

- **Wann sollte man ein Tuple statt einer Liste verwenden?**: Wenn Sie eine dynamische Sammlung von Elementen benötigen, bei der Sie Elemente hinzufügen, entfernen und aktualisieren können, sollten Sie eine Liste verwenden. Wenn Sie wissen, dass Sie mit einer festen und unveränderlichen Datensammlung arbeiten, sollten Sie ein Tuple verwenden.

## Häufige Tuple-Methoden

- **`count()`**: Wird verwendet, um zu bestimmen, wie oft ein Element in einem Tuple vorkommt. Zum Beispiel können Sie prüfen, wie oft die Sprache `'Rust'` im Tuple vorkommt:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

- Wenn das angegebene Element in der `count()` Methode überhaupt nicht im Tuple vorhanden ist, ist der Rückgabewert `0`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

- Wenn keine Argumente an die `count()` Methode übergeben werden, gibt Python eine `TypeError` zurück.

- **index()**: Wird verwendet, um den Index zu finden, an dem ein bestimmtes Element im Tuple vorhanden ist. Hier ist ein Beispiel für die Verwendung der `index()` Methode, um den Index für die Sprache `'Java'` zu finden:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

- Wenn das angegebene Element nicht gefunden werden kann, gibt Python eine `ValueError` zurück.

- Sie können einen optionalen Startindex an die `index()` Methode übergeben, um anzugeben, wo die Suche nach dem Element im Tuple beginnen soll:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

- Sie können auch einen optionalen Endindex an die `index()` Methode übergeben, um anzugeben, wo die Suche nach dem Element im Tuple enden soll:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

- **`sorted()`**: Wird verwendet, um die Elemente in einem iterierbaren Objekt zu sortieren und eine neue sortierte Liste zurückzugeben. Hier ist ein Beispiel, um eine neue Liste von Zahlen mit der `sorted()` Funktion zu erstellen:

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

- **Anpassen des Sortierverhaltens**: Sie können das Sortierverhalten für ein iterierbares Objekt mit den optionalen Argumenten `reverse` und `key` anpassen. Hier ist ein Beispiel für die Verwendung des Arguments `key`, um Elemente in einem Tuple nach Länge zu sortieren:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

- Sie können eine neue Liste von Werten in umgekehrter Reihenfolge mit dem Argument `reverse` wie folgt erstellen:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

## Schleifen in Python

- **Definition**: Schleifen werden verwendet, um einen Codeblock eine festgelegte Anzahl von Malen zu wiederholen.

- **`for` Schleife**: Wird verwendet, um über eine Folge (wie eine Liste, ein Tuple oder einen String) zu iterieren und einen Codeblock für jedes Element in dieser Folge auszuführen. Hier ist ein Beispiel für die Verwendung einer `for` Schleife, um eine Liste zu durchlaufen und jede Sprache in der Konsole auszugeben:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)

"""
Result 

Rust
Java
Python
C++
"""
```

- Hier ist ein Beispiel für die Verwendung einer `for` Schleife, um den String `code` zu durchlaufen und jedes Zeichen auszugeben:

```py
for char in 'code':
    print(char)

"""
Result 

c
o
d
e
"""
```

- `for` Schleifen können verschachtelt sein. Hier ist ein Beispiel für die Verwendung einer verschachtelten `for` Schleife:

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)

"""
Result

Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
"""

```

- **`while` Schleife**: Wiederholt einen Codeblock, bis die Bedingung `False` ist. Hier ist ein Beispiel für die Verwendung einer `while` Schleife für ein Ratespiel:

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')

"""
Result

Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
"""

```

- **`break` und `continue` Anweisungen**: Werden in Schleifen verwendet, um die Ausführung einer Schleife zu verändern.

- Die `break` Anweisung wird verwendet, um die Schleife sofort zu beenden, wenn eine bestimmte Bedingung erfüllt ist. Hier ist ein Beispiel für die Verwendung der `break` Anweisung für eine Liste von `developer_names`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

- Die `continue` Anweisung wird verwendet, um die aktuelle Iteration zu überspringen und mit der nächsten Iteration der Schleife fortzufahren. Hier ist ein Beispiel, um die `continue` Anweisung anstelle einer `break` Anweisung zu verwenden:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

- Sowohl `for` als auch `while` Schleifen können mit einer `else` Klausel kombiniert werden, die nur ausgeführt wird, wenn die Schleife nicht durch eine `break` beendet wurde:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

## Bereiche und ihre Verwendung in Schleifen

- **Die `range()` Funktion**: Wird verwendet, um eine Folge von ganzen Zahlen zu erzeugen.

```py
range(start, stop, step)
```

- Das erforderliche `stop` Argument ist eine ganze Zahl (nicht inklusive), die den Endpunkt für die erzeugte Zahlenfolge darstellt. Hier ist ein Beispiel für die Verwendung der `range()` Funktion:

```py
for num in range(3):
    print(num)
```

- Wenn kein `start` Argument angegeben wird, ist der Standardwert `0`. Standardmäßig wird die Folge von ganzen Zahlen um `1` inkrementiert. Sie können das optionale `step` Argument verwenden, um den Standardinkrementwert zu ändern. Hier ist ein Beispiel, um eine Folge von geraden ganzen Zahlen von 2 bis einschließlich 10 (also bis, aber nicht einschließlich 11) zu erzeugen:

```py
for num in range(2, 11, 2):
    print(num)
```

- Wenn Sie keine Argumente an die `range()` Funktion übergeben, erhalten Sie eine `TypeError`.

- Die `range()` Funktion akzeptiert nur ganze Zahlen als Argumente und keine Fließkommazahlen. Die Verwendung von Fließkommazahlen führt ebenfalls zu einer `TypeError`:

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- Sie können eine negative ganze Zahl für das `step` Argument verwenden, um eine Folge von ganzen Zahlen in absteigender Reihenfolge zu erzeugen:

```py
for num in range(40, 0, -10):
    print(num)
```

- Die `range()` Funktion kann auch verwendet werden, um eine Liste von ganzen Zahlen zu erstellen, indem sie mit dem `list` Konstruktor verwendet wird. Der `list` Konstruktor wird verwendet, um ein iterierbares Objekt in eine Liste umzuwandeln. Hier ist ein Beispiel, um eine Liste von geraden ganzen Zahlen zwischen 2 und 10 inklusive zu erzeugen:

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

## `enumerate()` und `zip()` Funktionen in Python

- **`enumerate()`**: Wird verwendet, um über eine Folge zu iterieren und den Index für jedes Element in dieser Folge zu verfolgen. Die `enumerate()` Funktion nimmt ein iterierbares Objekt als Argument und gibt ein `enumerate` Objekt zurück, das aus dem Index und dem Wert jedes Elements im iterierbaren Objekt besteht.

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')

# Result
# Index 0 and language Spanish
# Index 1 and language English
# Index 2 and language Russian
# Index 3 and language Chinese

```

- Die `enumerate()` Funktion kann auch außerhalb einer `for` Schleife verwendet werden:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

print(list(enumerate(languages)))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

- Die `enumerate()` Funktion akzeptiert auch ein optionales `start` Argument, das den Startwert für die Zählung angibt. Wenn dieses Argument weggelassen wird, beginnt die Zählung bei `0`.

-  **`zip()`** : Wird verwendet, um parallel über mehrere iterierbare Objekte zu iterieren. Hier ist ein Beispiel für die Verwendung der `zip()` Funktion, um über `developers` und `ids` zu iterieren:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')


"""
Result

Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
"""

```

## List Comprehensions in Python

- **Definition**: List Comprehension ermöglicht es Ihnen, eine neue Liste in einer einzigen Zeile zu erstellen, indem Sie die Schleife und die Bedingung direkt innerhalb von eckigen Klammern kombinieren. Dies macht den Code kürzer und oft leichter lesbar.

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

## Iterable-Methoden

- **`filter()`**: Wird verwendet, um Elemente aus einem iterierbaren Objekt basierend auf einer Bedingung zu filtern. Es gibt einen Iterator zurück, der nur die Elemente enthält, die die Bedingung erfüllen. Hier ist ein Beispiel, um eine neue Liste nur mit Wörtern zu erstellen, die länger als vier Zeichen sind:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**: Wird verwendet, um eine Funktion auf jedes Element in einem iterierbaren Objekt anzuwenden und ein neues iterierbares Objekt mit den Ergebnissen zurückzugeben. Hier ist ein Beispiel für die Verwendung der `map()` Funktion, um eine Liste von Celsius-Temperaturen in Fahrenheit umzuwandeln:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**: Wird verwendet, um die Summe aus einem iterierbaren Objekt wie einer Liste oder einem Tuple zu erhalten. Hier ist ein Beispiel für die Verwendung der `sum()` Funktion:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- Sie können auch ein optionales `start` Argument übergeben, das den Anfangswert für die Summierung festlegt. Hier ist ein aktualisiertes Beispiel, das das `start` Argument als Positionsargument verwendet:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- Sie können stattdessen auch das `start` Argument als Schlüsselwortargument wie folgt verwenden:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## Lambda-Funktionen

- **Definition**: Eine Lambda-Funktion in Python ist eine kurze Möglichkeit, eine Funktion ohne Namen (eine anonyme Funktion) zu erstellen.  
- Lambda-Funktionen werden oft als Argument an eine andere Funktion übergeben. Hier ist ein Beispiel für eine Lambda-Funktion:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

- Bewährte Vorgehensweisen für die Verwendung von Lambda-Funktionen umfassen, sie nicht einer Variablen zuzuweisen, sie einfach und lesbar zu halten und sie für kurze, einmalige Funktionen zu verwenden.
