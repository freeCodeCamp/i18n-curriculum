# --description--

从字母表的26个字母中取三个不同的字母，可以构成长度为3的字符串。

例如“abc”，“hat” 以及“zyx”。

当我们研究这三个例子时，我们看到对于 `abc`，有两个字符在字典序上位于其左侧邻居之后。

对于 `hat`，恰好有一个字符在字典序上位于其左侧邻居之后。对于 `zyx`，没有字符在字典序上位于其左侧邻居之后。

总共有 10400 个长度为 3 的字串，其中恰好有一个字符在字典序上位于其左侧邻居之后。

我们现在考虑由 $n ≤ 26$ 的字母表中的不同字符组成的字符串。

对于每个 $n$，$p(n)$ 是长度为 $n$ 的字串中恰好有一个字符在字典序上位于其左侧邻居之后的字串数量。

求 $p(n)$ 的最大值是多少？
