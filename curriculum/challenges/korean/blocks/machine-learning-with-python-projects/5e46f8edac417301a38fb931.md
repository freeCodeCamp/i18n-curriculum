# --instructions--

이번 도전 과제에서는 SMS 메시지를 "ham" 또는 "spam"으로 분류하는 머신러닝 모델을 만들어야 합니다. "ham" 메시지는 친구가 보낸 일반 메시지이고, "spam" 메시지는 광고나 회사에서 보낸 메시지입니다.

메시지 문자열을 인수로 받고 리스트를 반환하는 `predict_message`라는 함수를 만들어야 합니다. 리스트의 첫 번째 요소는 "ham"(0) 또는 "spam"(1)일 가능성을 나타내는 0과 1 사이의 숫자여야 합니다. 리스트의 두 번째 요소는 가장 가능성이 높은 "ham" 또는 "spam" 단어여야 합니다.

이번 도전 과제에서는 <a href="http://www.dt.fee.unicamp.br/~tiago/smsspamcollection/" target="_blank" rel="noopener noreferrer nofollow">SMS Spam Collection</a> 데이터셋을 사용할 것입니다. 데이터셋은 이미 학습 데이터와 테스트 데이터로 나누어져 있습니다.

처음 두 셀은 라이브러리와 데이터를 가져옵니다. 마지막 셀은 모델과 함수를 테스트합니다. 이 셀들 사이에 코드를 작성하세요.
