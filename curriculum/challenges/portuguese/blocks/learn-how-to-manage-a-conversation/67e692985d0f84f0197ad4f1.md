# --explanation--

`To address` significa tomar uma ação para lidar com um problema ou situação.Por exemplo:

`We need to address the login error before users report it.` – Isso significa que é necessário tomar uma ação para corrigir o problema.

`Quickly` expressa quão rápido algo é feito. `Quick`, por outro lado, simplesmente descreve algo como sendo rápido.Por exemplo:

- `She responded quickly to the email.` – Você usa `quickly` para descrever com que rapidez ela respondeu ao email.  

- `It was a quick meeting.` – Aqui, você está descrevendo a reunião em si como tendo durado um tempo curto e razoável.
