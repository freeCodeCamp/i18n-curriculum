# --questions--

## --text--

Quale frase è grammaticalmente corretta?

## --answers--

`You can explain the project details to me?`

### --feedback--

Questa frase dovrebbe essere strutturata come una domanda.

---

`Can you explain the project details to me?`

---

`Can explains you the project details?`

### --feedback--

Il verbo modale `can` dovrebbe essere seguito dalla forma base del verbo, non da `explains`.

---

`Do can you explain the project details?`

### --feedback--

`Can` agisce già come verbo ausiliare, quindi `do` non è necessario nella domanda.

# --explanation--

Quando si forma una domanda con i verbi modali, la struttura è: verbo modale + soggetto + verbo base + (oggetto)? Per esempio:

- `Can we complete the project by Friday?` - Chiedere se è possibile finire il progetto entro venerdì.

- `Should we update the software to the latest version?` - Chiedere un consiglio o suggerimento sul fatto di aggiornare il software.

Nella risposta corretta, `can` è posizionato prima del soggetto (`you`), e il verbo (`explain`) è nella sua forma base.
