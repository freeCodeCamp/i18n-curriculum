# --explanation--

`Reducing complexity` はシステムを理解しやすく、扱いやすくすることを意味します。
  
`Showing only what's necessary` はユーザーが操作に必要な部分だけを見ることを意味します。
  
`Keeping the details hidden` はユーザーに不要な内部のコードやロジックを隠すことを指します。

これらを合わせると、抽象化は複雑な内部の仕組みを隠しながら、明確でシンプルなインターフェースを作ることを意味します。
