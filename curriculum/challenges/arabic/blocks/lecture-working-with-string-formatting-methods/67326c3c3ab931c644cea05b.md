# --interactive--

عند العمل مع السلاسل النصية في JavaScript، من الشائع مواجهة فراغات غير مرغوب فيها في بداية أو نهاية السلسلة النصية. قد تتداخل الفراغات مع عمليات مثل المقارنة أو التخزين أو العرض، ولهذا من المهم معرفة كيفية إزالتها بكفاءة.

في هذا الدرس، سنستعرض كيف يمكنك إزالة الفراغات باستخدام طرق `trim()` و `trimStart()` و `trimEnd()` في JavaScript.

تشير الفراغات إلى المسافات أو علامات الجدولة أو فواصل الأسطر التي تظهر في السلسلة النصية لكنها ليست حروفًا مرئية. على سبيل المثال:

```js
let greeting = "   Hello, world!   ";
```

في هذه الحالة، توجد مسافات قبل وبعد النص المرئي `Hello, world!`.

طريقة `trim()` هي الأكثر استخدامًا لإزالة الفراغات من بداية ونهاية السلسلة النصية معًا. إليك مثالًا:

:::interactive_editor

```js
let message = "   Hello!   ";
console.log(message); // "   Hello!   "
let trimmedMessage = message.trim();
console.log(trimmedMessage);  // "Hello!"
```

:::

في هذه الحالة، تزيل طريقة `trim()` كل المسافات في البداية والنهاية، تاركة فقط `Hello!`. لاحظ أن أي فراغات داخل السلسلة النصية (بين الكلمات، على سبيل المثال) تبقى كما هي ولا تؤثر عليها `trim()`.

أحيانًا قد ترغب في إزالة الفراغات من بداية السلسلة أو نهايتها فقط، وليس من كلا الجانبين. هنا تأتي أهمية `trimStart()` و `trimEnd()`.

تزيل `trimStart()` الفراغات من بداية (أو بداية) السلسلة النصية.

:::interactive_editor

```js
let greeting = "   Hello!   ";
console.log(greeting);  // "   Hello!   "
let trimmedStart = greeting.trimStart();
console.log(trimmedStart);  // "Hello!   "
```

:::

تزيل `trimEnd()` الفراغات من نهاية السلسلة النصية.

:::interactive_editor

```js
let greeting = "   Hello!   ";
console.log(greeting);  // "   Hello!   "
let trimmedEnd = greeting.trimEnd();
console.log(trimmedEnd);  // "   Hello!"
```

:::

تمنحك هذه الطرق تحكمًا أدق في الجزء الذي تريد تنظيفه من السلسلة النصية.

باختصار، إزالة الفراغات جزء أساسي من التعامل مع السلاسل النصية في JavaScript. سواء أردت تنظيف بيانات المُدخَل أو ضمان مقارنة متسقة للسلاسل النصية، يمكنك استخدام `trim()` لإزالة الفراغات من كلا الجانبين، أو استخدام `trimStart()` و `trimEnd()` لاستهداف جانب معين.
