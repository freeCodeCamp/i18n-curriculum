# --description--

Para agregar datos de estudiantes a la tabla, necesitarás usar los elementos de fila de tabla y de datos de tabla.

El elemento <dfn>table data</dfn>, `td`, se utiliza para crear una celda en la tabla.

Aquí tienes un ejemplo de uso del elemento `td` para una tabla de jugadores de deportes:

```html
<tr>
  <td>1</td>
  <td>John Doe</td>
  <td>USA</td>
</tr>
```

Dentro de tu elemento de cuerpo de tabla, agrega un elemento de fila de tabla, `tr`, con tres elementos de datos de tabla, `td`.

El primer elemento de datos de tabla debe contener el apellido `Davis`.

El segundo elemento de datos de tabla debe contener el nombre `Alex`.

El tercer elemento de datos de tabla debe contener la calificación `54`.
