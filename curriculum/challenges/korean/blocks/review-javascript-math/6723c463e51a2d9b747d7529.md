# --interactive--

## 숫자 데이터 타입 다루기

- **정의**: JavaScript의 `Number` 타입은 정수, 부동소수점 숫자, `Infinity` 및 `NaN`를 포함합니다. 부동소수점 숫자는 소수점이 있는 숫자입니다. 양의 `Infinity`은 다른 어떤 숫자보다 큰 숫자이며 `-Infinity`는 다른 어떤 숫자보다 작은 숫자입니다. `NaN` (`Not a Number`)는 문자열 `"Jessica"`처럼 유효하지 않은 숫자 값을 나타냅니다.

## 일반적인 산술 연산

- **덧셈 연산자**: 이 연산자(`+`)는 두 개 이상의 숫자를 더하는 데 사용됩니다.
- **뺄셈 연산자**: 이 연산자(`-`)는 두 숫자 사이의 차이를 계산하는 데 사용됩니다.
- **곱셈 연산자**: 이 연산자(`*`)는 두 개 이상의 숫자를 곱하는 데 사용됩니다.
- **나눗셈 연산자**: 이 연산자(`/`)는 두 숫자 사이의 몫을 계산하는 데 사용됩니다.
- **0으로 나누기**: 0으로 나누려고 하면 JavaScript는 `Infinity`를 반환합니다.
- **나머지 연산자**: 이 연산자(`%`)는 나눗셈의 나머지를 반환합니다.
- **거듭제곱 연산자**: 이 연산자(`**`)는 한 숫자를 다른 숫자의 거듭제곱으로 올립니다.

## 숫자와 문자열 계산

- **설명**: 숫자와 문자열에 `+` 연산자를 사용하면 JavaScript가 숫자를 문자열로 강제 변환하여 두 값을 연결합니다. 문자열과 숫자에 `-`, `*` 또는 `/` 연산자를 사용하면 JavaScript가 문자열을 숫자로 강제 변환하여 결과가 숫자가 됩니다. `null`과 `undefined`의 경우, JavaScript는 수학 연산에서 `null`을 0으로, `undefined`를 `NaN`으로 처리합니다.

:::interactive_editor

```js
const result = 5 + '10';

console.log(result); // "510"
console.log(typeof result); // string

const subtractionResult = '10' - 5;
console.log(subtractionResult); // 5
console.log(typeof subtractionResult); // number

const multiplicationResult = '10' * 2;
console.log(multiplicationResult); // 20
console.log(typeof multiplicationResult); // number

const divisionResult = '20' / 2;
console.log(divisionResult); // 10
console.log(typeof divisionResult); // number

const result1 = null + 5;
console.log(result1); // 5
console.log(typeof result1); // number

const result2 = undefined + 5;
console.log(result2); // NaN
console.log(typeof result2); // number
```

:::

## 연산자 우선순위

- **정의**: 연산자 우선순위는 표현식에서 연산이 평가되는 순서를 결정합니다. 우선순위가 높은 연산자가 낮은 연산자보다 먼저 평가됩니다. 괄호 안의 값이 먼저 평가되고 곱셈/나눗셈이 덧셈/뺄셈보다 우선순위가 높습니다. 연산자 우선순위가 같으면 JavaScript는 결합법칙을 사용합니다.

:::interactive_editor

```js
const result = (2 + 3) * 4;

console.log(result); // 20

const result2 = 10 - 2 + 3;

console.log(result2); // 11

const result3 = 2 ** 3 ** 2;

console.log(result3); // 512
```

:::

- **정의**: 결합법칙은 동일한 유형의 연산자가 여러 개 있을 때 표현식이 평가되는 방향을 알려줍니다. 왼쪽에서 오른쪽(`left-associative`)인지 오른쪽에서 왼쪽(`right-associative`)인지 정의합니다. 예를 들어, 거듭제곱 연산자는 오른쪽에서 왼쪽으로 결합합니다:

:::interactive_editor

```js
const result4 = 5 ** 4 ** 1; 

console.log(result4); // 625
```

:::

## 증가 연산자와 감소 연산자

- **증가 연산자**: 이 연산자는 값을 1만큼 증가시키는 데 사용합니다. 접두사 표기법 `++num`은 변수의 값을 먼저 증가시키고, 그 다음 새로운 값을 반환합니다. 접미사 표기법 `num++`은 변수의 현재 값을 먼저 반환한 후, 값을 증가시킵니다.

:::interactive_editor

```js
let x = 5;

console.log(++x); // 6
console.log(x); // 6


let y = 5;

console.log(y++); // 5
console.log(y); // 6
```

:::

- **감소 연산자**: 이 연산자는 값을 1만큼 감소시키는 데 사용합니다. 접두사 표기법과 접미사 표기법은 증가 연산자와 마찬가지로 작동합니다.

:::interactive_editor

```js
let num = 5;

console.log(--num); // 4
console.log(num--); // 4
console.log(num); // 3
```

:::

## 복합 대입 연산자

- **덧셈 복합 대입 (`+=`) 연산자**: 이 연산자는 값들을 더한 후 결과를 변수에 할당합니다.
- **뺄셈 복합 대입 (`-=`) 연산자**: 이 연산자는 값들을 빼고 결과를 변수에 할당합니다.
- **곱셈 복합 대입 (`*=`) 연산자**: 이 연산자는 값들을 곱한 후 결과를 변수에 할당합니다.
- **나눗셈 복합 대입 (`/=`) 연산자**: 이 연산자는 값들을 나누고 결과를 변수에 할당합니다.
- **나머지 복합 대입 (`%=`) 연산자**: 이 연산자는 변수를 지정한 수로 나눈 후 나머지를 변수에 할당합니다.
- **거듭제곱 복합 대입 (`**=`) 연산자**: 이 연산자는 변수를 지정한 수의 거듭제곱으로 올리고 결과를 변수에 다시 할당합니다.

## 불리언과 동등성

- **불리언 정의**: 불리언은 두 가지 값만 가질 수 있는 데이터 타입입니다: `true` 또는 `false`.
- **동등 연산자 (`==`)**: 이 연산자는 값을 비교하기 전에 타입 강제 변환을 수행합니다.

:::interactive_editor

```js
console.log(5 == '5'); // true
```

:::

- **엄격 동등 연산자 (`===`)**: 이 연산자는 타입 강제 변환을 수행하지 않으며, 타입과 값이 모두 같은지 확인합니다.

:::interactive_editor

```js
console.log(5 === '5'); // false
```

:::

- **부등 연산자 (`!=`)**: 이 연산자는 값을 비교하기 전에 타입 강제 변환을 수행합니다.
- **엄격 부등 연산자 (`!==`)**: 이 연산자는 타입 강제 변환을 수행하지 않고 타입과 값이 모두 다른지 확인합니다.

## 비교 연산자

- **크다 (`>`) 연산자**: 이 연산자는 왼쪽 값이 오른쪽 값보다 큰지 확인합니다.  
- **크거나 같다 (`>=`) 연산자**: 이 연산자는 왼쪽 값이 오른쪽 값보다 크거나 같은지 확인합니다.  
- **작다 (`<`) 연산자**: 이 연산자는 왼쪽 값이 오른쪽 값보다 작은지 확인합니다.  
- **작거나 같다 (`<=`) 연산자**: 이 연산자는 왼쪽 값이 오른쪽 값보다 작거나 같은지 확인합니다.

## 단항 연산자

- **단항 더하기 연산자**: 이 연산자는 피연산자를 숫자로 변환합니다. 피연산자가 이미 숫자라면 변경되지 않습니다.

:::interactive_editor

```js
const str = '42';
const num = +str;

console.log(num); // 42
console.log(typeof num); // number
```

:::

- **단항 부정 (`-`) 연산자**: 이 연산자는 피연산자를 부정합니다.

:::interactive_editor

```js
const num = 4;
console.log(-num); // -4
```

:::

- **논리 NOT (`!`) 연산자**: 이 연산자는 피연산자의 불리언 값을 반전시킵니다. 따라서 피연산자가 `true`이면 `false`가 되고, `false`이면 `true`가 됩니다.

## 비트 연산자

- **비트 AND (`&`) 연산자**: 이 연산자는 두 피연산자의 해당 비트가 모두 1인 각 비트 위치에 1을 반환합니다.  
- **비트 AND 할당 (`&=`) 연산자**: 이 연산자는 지정된 숫자와 `bitwise AND` 연산을 수행하고 결과를 변수에 다시 할당합니다.  
- **비트 OR (`|`) 연산자**: 이 연산자는 두 피연산자 중 하나 또는 둘 다의 해당 비트가 1인 각 비트 위치에 1을 반환합니다.  
- **비트 OR 할당 (`|=`) 연산자**: 이 연산자는 지정된 숫자와 `bitwise OR` 연산을 수행하고 결과를 변수에 다시 할당합니다.  
- **비트 XOR (`^`) 연산자**: 이 연산자는 두 피연산자 중 하나만 1인 각 비트 위치에 1을 반환합니다.  
- **비트 NOT (`~`) 연산자**: 이 연산자는 숫자의 이진 표현을 반전시킵니다.  
- **왼쪽 시프트 (`<<`) 연산자**: 이 연산자는 모든 비트를 지정된 수만큼 왼쪽으로 이동시킵니다.  
- **오른쪽 시프트 (`>>`) 연산자**: 이 연산자는 모든 비트를 오른쪽으로 이동시킵니다.

## 조건문, 참 같은 값, 거짓 같은 값 그리고 삼항 연산자

- **`if/else if/else`**: `if` 문은 조건을 받아 그 조건이 `truthy`일 때 코드 블록을 실행합니다. 조건이 `false`이면 `else if` 블록으로 넘어갑니다. 그 어떤 조건도 `true`가 아니면 `else` 절을 실행합니다. `Truthy` 값은 `if` 문과 같은 불리언 문맥에서 평가할 때 `true`가 되는 모든 값입니다. `Falsy` 값은 불리언 문맥에서 `false`로 평가되는 값입니다.

:::interactive_editor

```js
const score = 87;

if (score >= 90) {
 console.log('You got an A'); 
} else if (score >= 80) {
 console.log('You got a B'); // You got a B
} else if (score >= 70) {
 console.log('You got a C');
} else {
 console.log('You failed! You need to study more!');
}
```

:::

- **삼항 연산자**: 이 연산자는 `if else` 문을 더 짧게 작성할 때 자주 사용합니다.

:::interactive_editor

```js
const temperature = 30;
const weather = temperature > 25 ? 'sunny' : 'cool';

console.log(`It's a ${weather} day!`); // It's a sunny day!
```

:::

## 이항 논리 연산자

- **논리 AND (`&&`) 연산자**: 이 연산자는 두 피연산자가 모두 `true`인지 확인합니다. 둘 다 `true`면 두 번째 값을 반환합니다. 어느 하나가 `falsy`면 `falsy` 값을 반환합니다. 둘 다 `falsy`면 첫 번째 `falsy` 값을 반환합니다.

:::interactive_editor

```js
const result = true && 'hello';

console.log(result); // hello
```

:::

- **논리 OR (`||`) 연산자**: 이 연산자는 피연산자 중 적어도 하나가 참 같은 값인지 확인합니다.
- **널 병합 (`??`) 연산자**: 이 연산자는 첫 번째 값이 `null` 또는 `undefined`일 때만 값을 반환합니다.

:::interactive_editor

```js
const userSettings = {
 theme: null,
 volume: 0,
 notifications: false,
};

let theme = userSettings.theme ?? 'light';
console.log(theme); // light
```

:::

## `Math` 객체

- **`Math.random()` 메서드**: 이 메서드는 0(포함)과 1(배타적) 사이의 임의의 부동 소수점 숫자를 생성합니다. 즉, 출력값으로 0이 나올 수 있지만 1에 도달하지는 않습니다.
- **`Math.max()` 메서드**: 이 메서드는 숫자 집합을 받아 최대값을 반환합니다.
- **`Math.min()` 메서드**: 이 메서드는 숫자 집합을 받아 최솟값을 반환합니다.
- **`Math.ceil()` 메서드**: 이 메서드는 값을 가장 가까운 정수로 올림합니다.
- **`Math.floor()` 메서드**: 이 메서드는 값을 가장 가까운 정수로 내림합니다.
- **`Math.round()` 메서드**: 이 메서드는 값을 가장 가까운 정수로 반올림합니다.

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

- **`Math.trunc()` 메서드**: 이 메서드는 숫자의 소수 부분을 제거하고 정수 부분만 반환하며 반올림하지 않습니다.
- **`Math.sqrt()` 메서드**: 이 메서드는 숫자의 제곱근을 반환합니다.
- **`Math.cbrt()` 메서드**: 이 메서드는 숫자의 세제곱근을 반환합니다.
- **`Math.abs()` 메서드**: 이 메서드는 숫자의 절댓값을 반환합니다.
- **`Math.pow()` 메서드**: 이 메서드는 두 숫자를 받아 첫 번째 숫자를 두 번째 숫자만큼 거듭제곱합니다.

## 자주 쓰이는 숫자 메서드

- **`isNaN()`**: `NaN`는 "숫자가 아님(Not-a-Number)"을 뜻합니다. 이는 표현할 수 없거나 정의되지 않은 숫자 결과를 나타내는 특별한 값입니다. `isNaN()` 함수 속성은 값이 `NaN`인지 아닌지 판단하는 데 사용됩니다. `Number.isNaN()`은 특히 타입 강제 변환으로 인해 전역 `isNaN()` 함수가 예상치 못한 결과를 낼 수 있는 경우에 `NaN` 값을 더 신뢰성 있게 확인하는 방법을 제공합니다.

:::interactive_editor

```js
console.log(isNaN(NaN));       // true
console.log(isNaN(undefined)); // true
console.log(isNaN({}));        // true

console.log(isNaN(true));      // false
console.log(isNaN(null));      // false
console.log(isNaN(37));        // false


console.log(Number.isNaN(NaN));        // true
console.log(Number.isNaN(Number.NaN)); // true
console.log(Number.isNaN(0 / 0));      // true

console.log(Number.isNaN("NaN"));      // false
console.log(Number.isNaN(undefined));  // false
```

:::

- **`parseFloat()` 메서드**: 이 메서드는 문자열 인수를 파싱하여 부동소수점 숫자를 반환합니다. 문자열에 숫자가 아닌 문자가 뒤에 있어도 문자열 시작 부분에서 숫자를 추출하도록 설계되었습니다.
- **`parseInt()` 메서드**: 이 메서드는 문자열 인수를 파싱하여 정수를 반환합니다. `parseInt()`는 처음 만나는 숫자가 아닌 문자에서 파싱을 멈춥니다. 부동소수점 숫자의 경우 정수 부분만 반환합니다. 문자열 시작 부분에서 유효한 정수를 찾지 못하면 `NaN`을 반환합니다.
- **`toFixed()` 메서드**: 이 메서드는 숫자에서 호출되며, 소수점 아래 표시할 자릿수를 나타내는 선택적 인수를 하나 받습니다. 지정한 소수점 자릿수만큼 숫자를 문자열로 반환합니다.
