# --description--

Оскільки ти програмуєш на JavaScript, ти неминуче стикатимешся з повідомленнями про помилки. Розуміння загальних повідомлень про помилки допоможе тобі ефективніше налагоджувати код і стати кращим програмістом. Чотири загальні типи повідомлень про помилки — це `SyntaxError`, `ReferenceError`, `TypeError` та `RangeError`.

`SyntaxError` виникає, коли ти неправильно пишеш щось у своєму коді, наприклад, пропускаєш дужку або квадратну дужку. Уяви це як граматичну помилку в реченні. Ось поширена помилка, яку роблять розробники при створенні масивів:

```js
const arr = ["Beau", "Quincy" "Tom"]
```

Кожен елемент масиву повинен бути відокремлений комою, інакше виникне повідомлення про помилку.

Другою загальною помилкою JavaScript є `ReferenceError`. Існує кілька типів `ReferenceError`, які виникають різними способами. Перший тип `ReferenceError` — це не визначені змінні.

```js
console.log(price);
```

У цьому прикладі ми намагаємося вивести змінну `price` в консоль, але вона не була визначена. Це призведе до `ReferenceError`.

Ще один приклад `ReferenceError` — це спроба отримати доступ до змінної, оголошеної за допомогою `let` або `const`, до того, як вона була визначена:

```js
console.log(b);
const b = 50;
```

Вищенаведений приклад призведе до помилки `Cannot access 'b' before initialization`. 

Третьою загальною помилкою буде `TypeError`. Ці помилки виникають, коли ти намагаєшся виконати операцію над неправильним типом. Ось приклад спроби використати `map` метод на об’єкті:

```js
const developerObj = {
  name: "Jessica",
  country: "USA",
  isEmployed: true
};

developerObj.map()
```

Цей приклад призведе до помилки `developerObj.map is not a function`, тому що метод `map` використовується для масивів, а не для об’єктів. 

Останньою загальною помилкою, яку ми розглянемо, є `RangeError`.

`RangeError` виникає, коли твій код намагається використати значення, яке знаходиться поза межами діапазону, який JavaScript може обробити. Ось приклад призначення недійсного значення довжині масиву:

```js
const arr = [];
arr.length = -1; 
```

Оскільки `length` масиву має бути невід’ємним цілим числом, встановлення його в `-1` викликає `RangeError`.

Продовжуючи програмувати на JavaScript, просто май на увазі ці різні типи помилок, з якими ти, ймовірно, зіткнешся, і чому вони виникають.
