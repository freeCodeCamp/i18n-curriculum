# --interactive--

CSS 中的属性选择器是一个功能强大的工具，它允许你根据 HTML 元素的属性来定位这些元素。 这意味着你可以将样式应用到具有特定属性的元素，甚至是这些属性的特定值。

当你想对元素进行动态样式化处理，或者仅凭类名无法提供足够的特定性时，它就显得尤为有用。

例如，你可以使用属性选择器来定位具有 `href` 属性的所有链接。 当你想对页面上的所有链接应用统一的样式时，这尤其方便。

以下是如何使用属性选择器来定位带有 `href` 属性的链接：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Example link with an href attribute</a>
<a>Example link without an href attribute</a>
```

```css
a {
  display: block;
}

a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

此 CSS 规则将对包含 `href` 属性的任何链接应用蓝色和下划线，以确保所有可点击链接的样式一致。

但属性选择器可以更加具体。 假设你只想针对具有 `title` 属性的链接。 `title` 属性通常会提供有关链接的附加信息，你可能希望以不同的样式来显示这些链接的附加信息。 具体方法如下：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

此规则应用粗体文本，并删除带有 `title` 属性的任何链接的下划线。 这是一种在视觉上将这些链接与页面上其他链接区分开来的好方法。

你还可以组合属性选择器，以获得更精确的结果。 例如，如果你想为同时具有 `href` 和 `title` 属性的链接设计样式，你可以这样做：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[href][title] {
  display:block;
  color: green;
}
```

:::

在这种情况下，只有同时包含 `href` 和 `title` 属性的链接才会使用绿色文本样式。 这种程度的控制正是属性选择器在 CSS 中如此强大的原因。

另一个属性选择器的示例是匹配属性中以空格分隔的值列表中的单个值。

这是一个具有多个类的锚点元素示例：

```html
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

要定位此特定锚点元素，你可以使用以下选择器：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

这里使用 `[attr~=value]` 语法来定位所有类属性包含单词 `"primary"` 的锚点元素。

如果你需要定位属性值以特定值为前缀的元素，那么你可以使用 `[attr^=value]` 语法。下面是一个示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

在此示例中，`a[href^="https://"]` 选择器将定位所有 `href` 属性值以 `"https://"` 开头的锚点元素。

要定位属性值以特定值结尾的元素，你可以使用 `[attr$=value]` 语法。以下是一个示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Visit Example Site</a>
```

```css
a[href$=".com"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

在此示例中，`a[href$=".com"]` 选择器将定位所有 `href` 属性值以 `.com` 结尾的锚点元素。

使用属性选择器不仅能增强网页的样式，还能使链接等交互式元素更容易根据属性加以区分，从而提高可访问性。
