`Let's work on` 是一種提出合作的方式。它通常後接名詞或 `-ing` 形式。當鼓勵團隊合作並推進任務時，這個片語經常被使用。 例如：

- `Let's work on fixing these security vulnerabilities.` - 提議解決系統中的安全問題。

- `Let's work on the documentation so it's easier to follow.` - 提議改進文件以提高清晰度。
