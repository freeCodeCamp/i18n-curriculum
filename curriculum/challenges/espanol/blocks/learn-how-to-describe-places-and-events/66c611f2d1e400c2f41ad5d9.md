# --explanation--

En el tiempo `Present Simple`, la forma negativa se crea usando `doesn't` o `isn't` para sujetos en tercera persona del singular (`he, she, it`). 

- Usa `doesn't` con verbos de acción para mostrar que algo no está ocurriendo, como `It doesn't work.`

- Usa `isn't` con formas de `to be` para mostrar que algo no es cierto, como `isn't set up.`

En esta oración, necesitarás tanto `doesn't` como `isn't` para describir correctamente por qué la función no está funcionando como se esperaba.
