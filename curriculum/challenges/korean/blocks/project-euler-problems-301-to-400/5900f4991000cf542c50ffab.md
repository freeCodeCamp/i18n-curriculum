# --description--

님은 돌 더미로 하는 게임으로, 두 명의 플레이어가 번갈아 가며 아무 더미에서나 원하는 수의 돌을 제거하다가 돌이 모두 없어지면 게임이 끝납니다.

여기서는 세 개의 더미로 하는 일반 님 게임 버전을 다룹니다. 규칙은 다음과 같습니다:

- 게임 시작 시 세 개의 돌 더미가 있습니다.
- 차례가 된 플레이어는 한 더미에서 양수 개수의 돌을 제거합니다.
- 돌이 남아 있지 않아 움직일 수 없는 첫 번째 플레이어가 패배합니다.

만약 ($n_1$, $n_2$, $n_3$)가 크기가 각각 $n_1$, $n_2$, $n_3$인 더미들로 이루어진 님 위치를 나타낸다면, $X(n_1,n_2,n_3)$라는 간단한 함수가 있습니다 — 직접 찾아보거나 추론해도 좋습니다 — 이 함수는 다음을 반환합니다:

- 완벽한 전략을 사용했을 때, 다음 차례인 플레이어가 결국 패배할 경우 0을 반환합니다.
- 완벽한 전략을 사용했을 때, 다음 차례인 플레이어가 결국 승리할 경우 0이 아닌 값을 반환합니다.

예를 들어 $X(1, 2, 3) = 0$입니다. 현재 플레이어가 어떤 수를 움직여도 상대가 두 더미의 크기를 같게 만드는 수를 두어, 이후 현재 플레이어의 모든 움직임을 상대가 그대로 따라 할 수 있기 때문에 결국 현재 플레이어가 패배합니다. 예시를 들면:

- 현재 플레이어가 (1,2,1)로 이동
- 상대가 (1,0,1)로 이동
- 현재 플레이어가 (0,0,1)로 이동
- 상대가 (0,0,0)으로 이동하여 승리

$X(n, 2n, 3n) = 0$이 되는 양의 정수 $n ≤ 2^{30}$는 몇 개일까요?
