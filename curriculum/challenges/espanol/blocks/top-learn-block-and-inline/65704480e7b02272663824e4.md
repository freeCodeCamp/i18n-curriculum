El concepto de “flujo normal” está implícito en los recursos del modelo de caja, pero no se explica de manera muy específica. Lee <a href="https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Normal_Flow" target="_blank">Normal Flow</a> de MDN para asegurarte de entender cómo los elementos se disponen por defecto.

# --questions-- 

## --text--

¿Qué es el "Normal flow" en los diseños CSS?

## --answers--

El flujo normal se refiere al proceso de animar elementos en un diseño CSS.

---

El flujo normal es una propiedad CSS usada para controlar la visibilidad de elementos en un diseño.

---

El flujo normal es una regla CSS que define el orden de las media queries para diseño responsivo.

---

El flujo normal en los diseños CSS se refiere a la posición predeterminada de los elementos en una página web.
