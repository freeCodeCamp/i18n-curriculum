# --description--

正の整数 $n$ と数字 $d$ に対して、$F(n, d)$ を $n$ の約数のうち末尾の数字が $d$ と等しいものの個数と定義します。

例えば、$F(84, 4) = 3$ です。84 の約数（1、2、3、4、6、7、12、14、21、28、42、84）の中で、末尾の数字が 4 であるものは 3 つ（4、14、84）あります。

また、$F(12!, 12) = 11$ および $F(50!, 123) = 17\,888$ であることも確認できます。

$F({10}^6!, 65\,432) \text{ を } ({10}^{16} + 61) \text{ で割った余り}$ を求めてください。
