# --description--

使用反引号定义的字串称为模板字面值。它们是 JavaScript 中定义字串的一种新方式。它们允许你在字串中嵌入表达式。这是通过将表达式包裹在 `${}` 中完成的。

例如，下面的代码：

```javascript
let name = "John";
let age = 25;
let greeting = `Hello, my name is ${name} and I am ${age} years old.`;
```

将导致 `greeting` 变量包含字串 "Hello, my name is John and I am 25 years old."。
