# --description--

Un anno bisestile è un anno divisibile per `4`, tranne gli anni divisibili per `100` e non divisibili per `400`. Per esempio, `2000` è un anno bisestile, ma `1900` non lo è. Inoltre, un anno bisestile ha un giorno in più a febbraio, che è il 29° giorno del mese.

**Obiettivo:** Completa le storie utente qui sotto e fai passare tutti i test per completare il laboratorio.

**Storie utente:**

1. Definisci una funzione chiamata `isLeapYear` che prende un numero come argomento.
2. All’esterno della funzione, dichiara una variabile `year` che archivia il valore dell’anno che vuoi controllare.
3. All’interno della funzione, usa un’istruzione `if/ else` o un operatore ternario per verificare se l’anno è bisestile.
4. Per verificare se l’anno è bisestile, soddisfa le seguenti condizioni:

   - Se l’anno è divisibile per `4`, allora è bisestile.
   - A meno che l’anno sia anche divisibile per `100`, allora non è bisestile.
   - A meno che l’anno sia anche divisibile per `400`, allora è bisestile.

5. Se l’anno è bisestile, restituisci `[year] is a leap year.`. Altrimenti, restituisci `[year] is not a leap year.`. Sostituirai `[year]` con il parametro definito nella funzione `isLeapYear`.
6. Dovresti chiamare la funzione `isLeapYear` con `year` come argomento e assegnare il risultato a una variabile chiamata `result`.
7. Dovresti mostrare la variabile `result` nella console usando `console.log()`.

function isLeapYear(year) {
  if ((year % 4 === 0 && year % 100 !== 0) || year % 400 === 0) {
    return(year + " is a leap year.");
  } else {
    return(year + " is not a leap year.");
  }
}
