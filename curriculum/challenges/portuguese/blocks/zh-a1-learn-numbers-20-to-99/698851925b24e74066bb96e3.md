# --description--

Vamos praticar a escrita de alguns números entre 80 e 89. Para completar esta tarefa, lembre-se de que, em chinês, o número 80 é chamado `八十 (bā shí)`.

Os números de 0 a 9 são:

| Número | Chinês |
| :--- | :--- |
| 0 | `零 (líng)` |
| 1 | `一 (yī)` |
| 2 | `二 (èr)` |
| 3 | `三 (sān)` |
| 4 | `四 (sì)` |
| 5 | `五 (wǔ)` |
| 6 | `六 (liù)` |
| 7 | `七 (qī)` |
| 8 | `八 (bā)` |
| 9 | `九 (jiǔ)` |

# --instructions--

Esta tarefa não tem áudio. Complete os espaços em branco abaixo com os números correspondentes em chinês.
