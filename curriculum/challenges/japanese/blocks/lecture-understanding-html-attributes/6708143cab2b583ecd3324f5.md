# --interactive--

属性とは、HTML要素の開始タグ内に配置される値のことです。属性は要素に関する追加情報を提供したり、要素の動作方法を指定したりします。属性の基本的な構文は以下の通りです。

```html
<element attribute="value"></element>
```

属性名の後に等号（`=`）が続き、値は引用符で囲まれます。値は属性によって文字列または数値の場合があります。

最初の例では、`href`属性と`target`属性が使われています。`href`属性はリンクのURLを指定し、`target`属性はリンクをどこで開くかを指定します。

**注意：** `a`要素はアンカー要素とも呼ばれ、ハイパーリンクを作成するために使われます。開始タグと終了タグの間のテキストが、ユーザーがクリックして移動する部分です。

インタラクティブエディターを有効にして、`href="https://www.freecodecamp.org/news/"`を`href="https://www.freecodecamp.org"`に変更してください。これでインタラクティブエディター内のリンクをクリックすると、新しいブラウザタブでfreeCodeCampのホームページが表示されます。

:::interactive_editor

```html
<a href="https://www.freecodecamp.org/news/" target="_blank">Visit freeCodeCamp</a>
```

:::

`href`属性がなければ、リンクは機能しません。なぜならリンク先のURLが存在しないからです。したがって、この`href`属性を含めてリンクを機能させる必要があります。`target="_blank"`属性はリンクを新しいブラウザタブで開くことを可能にします。`target`属性については今後のレッスンで詳しく学びます。

他の一般的な属性には、`src`属性と`alt`属性（代替属性とも呼ばれます）があり、それぞれ画像のソースを指定し、画像の代替説明テキストを提供するために使われます。

インタラクティブエディターを有効にして、`src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"`を`src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/running-cats.jpg"`に変更してください。次に`alt="Two tabby kittens sleeping together on a couch."`を`alt="Two cats running in the dirt."`に変更してください。

:::interactive_editor

```html
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" alt="Two tabby kittens sleeping together on a couch." />
```

:::

`href`属性と同様に、`src`属性は表示する画像ファイルを指定するため必須です。`alt`属性は必須ではありませんが、アクセシビリティのために推奨されます。アクセシビリティとは、障害のある方も含めて誰もがウェブサイトやアプリ、物理的な空間を利用し理解できるようにすることです。アクセシビリティについては今後のレッスンでさらに学びます。

一部の属性は`checked`属性のように構文が少し独特です。

インタラクティブエディターを有効にして、プレビューウィンドウのチェックボックスをクリックして、チェック状態と未チェック状態が切り替わるのを確認してください。

:::interactive_editor

```html
<input type="checkbox" checked />
```

:::

次の例では、`input`要素に`type`属性が`checkbox`に設定されています。入力はユーザーからデータを収集するために使われ、`type`属性は入力の種類を指定します。この場合、この入力はチェックボックスです。入力の動作については今後のレッスンで詳しく学びます。

`checked`属性はチェックボックスをデフォルトでチェック済みにするために使われます。`checked`属性は値を必要としません。存在すればチェックボックスはデフォルトでチェックされ、存在しなければ未チェックになります。これはブール値属性として知られています。ブール値についてはJavaScriptのセクションでさらに学びます。

インタラクティブエディターを有効にして、`checked`属性を`input`から削除してみてください。チェックボックスがデフォルトでチェックされなくなるのがわかります。

:::interactive_editor

```html
<input type="checkbox" checked />
```

:::

HTMLには`disabled`、`readonly`、`required`など、よく使われるブール値属性がいくつかあります。これらの属性は、要素が無効か、読み取り専用か、必須かなどの状態を指定するために使われます。

以下はデフォルトで無効になっているテキスト`input`要素の例です。インタラクティブエディターを有効にして、プレビューウィンドウの`input`要素をクリックしてみてください。次に`input`要素から`disabled`属性を削除すると、`input`がデフォルトで無効でなくなり、クリックしてフィールド内に入力できるようになります。

:::interactive_editor

```html
<input type="text" disabled>
```

:::

HTMLには、ウェブページ上の要素の動作や見た目をカスタマイズするために使える多くの属性があります。属性の使い方を理解することは、インタラクティブでアクセシブルなウェブコンテンツを作成するために不可欠です。今後のレッスンでさらに多くのHTML属性と、それらを効果的に使う方法を学びます。
