# --explanation--

O `Zero Conditional` é usado para coisas que são sempre verdadeiras quando uma condição é satisfeita. 

Segue a estrutura `if` + `Present Simple` (para a condição) e `Present Simple` (para o resultado).Por exemplo:

- `If you press the power button, the computer starts.` - Pressionar o botão de energia sempre liga o computador.

- `If the internet is down, the website doesn't load.` - O site não abrirá se não houver conexão com a internet.

Neste diálogo, `if the BIOS is corrupted` é a condição e `it prevents the computer from starting` é o resultado.Esta é uma verdade geral sobre como o BIOS corrompido afeta o computador.
