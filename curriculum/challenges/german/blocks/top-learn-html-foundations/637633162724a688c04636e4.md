# --description--

HTML und CSS befassen sich hauptsächlich mit der Darstellung von Informationen. Sie werden nicht verwendet, um Logik zu programmieren. JavaScript, das Sie im nächsten Abschnitt lernen werden, wird verwendet, um Webseiten zum Handeln zu bringen. Mit nur HTML und CSS können Sie schon eine Menge machen, und Sie werden beide definitiv brauchen. Im Verlauf unseres Curriculums konzentrieren sich die folgenden Lektionen darauf, Ihnen die Werkzeuge zu geben, die Sie benötigen, um erfolgreich zu sein, sobald Sie zu JavaScript-Inhalten kommen.
