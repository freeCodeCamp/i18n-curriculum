# --description--

現在讓我們使用 jQuery 從你的頁面中移除一個 HTML 元素。

jQuery 有一個名為 `.remove()` 的函式（程式），可以完全移除一個 HTML 元素。

使用 `.remove()` 函式（程式）從頁面中移除 `#target4` 元素。

# --hints--

你應該使用 jQuery 從你的頁面中移除 `target4` 元素。

```js
assert(
  $('#target4').length === 0 && code.match(/\$\(["']#target4["']\).remove\(\)/g)
);
```

你應該只使用 jQuery 來移除此元素。

```js
assert(
  code.match(/id="target4/g) &&
    !code.match(/<!--.*id="target4".*-->/g) &&
    $('#right-well').length > 0
);
```
