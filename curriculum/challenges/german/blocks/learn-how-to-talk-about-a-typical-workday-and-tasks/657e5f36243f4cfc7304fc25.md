# --description--

Etwas `report` bedeutet, Informationen über ein Ereignis, ein Problem oder eine Situation an die richtige Person oder Behörde weiterzugeben. Am Arbeitsplatz bedeutet das oft, einem Manager, der Personalabteilung oder dem Sicherheitsteam von einem Problem zu berichten, damit es gelöst werden kann. Zum Beispiel:

`I reported the broken printer to the IT department.` – Das bedeutet, die Person hat die IT über das Problem informiert, damit sie es beheben kann.

Berichten ist wichtig, weil so sichergestellt wird, dass Probleme nicht ignoriert werden und effektiv behandelt werden können.
