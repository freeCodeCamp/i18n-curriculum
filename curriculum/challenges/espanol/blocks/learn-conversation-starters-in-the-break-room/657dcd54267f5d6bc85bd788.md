# --description--

Si necesitas obtener algo de dinero y estás buscando una máquina que dispense efectivo, puedes preguntar, `Is there an ATM nearby?`

Un `ATM`, que significa `Automated Teller Machine`, es una máquina donde puedes usar una tarjeta bancaria para retirar efectivo.

La palabra `nearby` significa que no está lejos de ti, tal vez a un corto paseo de distancia. Es una palabra útil para usar cuando necesitas encontrar algo rápidamente en un lugar nuevo.
