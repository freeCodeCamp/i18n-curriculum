# --explanation--

`Need to be careful` означає звертати увагу, щоб уникнути помилок або проблем. Це відрізняється від `need to care`. У той час як `need to care` означає, що потрібна турбота або увага, `need to be careful` означає, що людина має діяти обережно. Акцент більше на людині, ніж на самому акті піклування. Наприклад:  

- `We need to have patience while waiting for results.` - Тут важливий сам акт терпіння.

- `We need to be patient while waiting for results.` - Тут акцент на людях, які мають бути терплячими - `we`, у цьому випадку.
