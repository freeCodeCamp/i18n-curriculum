# --explanation--  

`That way` wird verwendet, um das Ergebnis oder den Nutzen einer Entscheidung oder Handlung zu erklären. Zum Beispiel:

`We should automate testing. That way, we can catch bugs faster.` – `That way` zeigt, dass Automatisierung dabei hilft, Bugs schneller zu finden.  

`While` wird verwendet, um anzuzeigen, dass zwei Dinge gleichzeitig passieren. Zum Beispiel:  

`The team is fixing bugs while the designers work on the UI.` – Das bedeutet, dass beide Aufgaben gleichzeitig stattfinden.
