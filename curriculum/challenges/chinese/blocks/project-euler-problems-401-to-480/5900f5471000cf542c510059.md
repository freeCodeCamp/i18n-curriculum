# --description--

对于正整数 $n$ 和数字 $d$，我们定义 $F(n, d)$ 为 $n$ 的所有除数中末尾数字等于 $d$ 的数量。

例如，$F(84, 4) = 3$。在 84 的除数中（1、2、3、4、6、7、12、14、21、28、42、84），有三个（4、14、84）的最后一位数字是 4。

我们还可以验证 $F(12!, 12) = 11$ 和 $F(50!, 123) = 17\\,888$。

查找 $F({10}^6!, 65\\,432) <<<LTXT0>>> 模 ${10}^{16} + 61$。
