# --description--

In programmazione, `int` indica "integer". È un tipo di dato usato per archiviare numeri interi, cioè numeri senza parti frazionarie o decimali. I numeri interi possono essere positivi, negativi o zero.

Scegliere il tipo di dato `int` è fondamentale quando si lavora con numeri interi, come conteggi, indici negli array o in qualsiasi situazione in cui non servano numeri frazionari.

Esempi:

Contare elementi: `int itemCount = 5;`

Archiviare l'età: `int age = 30;`
