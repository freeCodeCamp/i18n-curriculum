# --description--

「To `tackle tasks`」意指開始著手處理任務並嘗試完成它們，通常需要付出努力。例如：

- `She tackled the hardest task first.`－她先從最困難的工作開始。

- `They tackled all the tasks before lunch.`－他們在午餐時間前完成了工作。
