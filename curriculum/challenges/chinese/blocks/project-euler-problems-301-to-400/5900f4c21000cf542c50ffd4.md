# --description--

对于固定的整数 $a$、$b$、$c$，定义疯狂函数 $F(n)$ 如下：

$$\begin{align}
  & F(n) = n - c \\;\text{ 对所有 } n > b \\\\
  & F(n) = F(a + F(a + F(a + F(a + n)))) \\;\text{ 对所有 } n ≤ b.
\end{align}$$

另外，定义 $S(a, b, c) = \displaystyle\sum_{n = 0}^b F(n)$。

例如，如果 $a = 50$，$b = 2000$ 且 $c = 40$，那么 $F(0) = 3240$ 且 $F(2000) = 2040$。此外，$S(50, 2000, 40) = 5\\,204\\,240$。

查找 $S({21}^7, 7^{21}, {12}^7)$ 的最后 9 位数字。
