# --explanation--

`Made sure` 是指採取行動以確認某事為真或運作正常。例如：  

`They made sure the software was ready before launch.` – 他們檢查了一切是否正常運作。

`As` 用來描述某件事與另一個動作同時發生。 例如：  

`The system adds more resources as demand increases.` – 它會在更多人開始使用時進行調整。
