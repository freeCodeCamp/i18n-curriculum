# --description--

$ n $ 個のマスが一列に並んでいて、左端のマスにカエルがいます。カエルは連続したジャンプで右端のマスまで行き、そこからまた左端のマスに戻ります。行きの旅では右に1、2、または3マスジャンプし、帰りの旅では同様に左にジャンプします。マスの外にジャンプすることはできません。この往復の旅を $ m $ 回繰り返します。

$F(m, n)$ を、カエルが移動する方法のうち、最大で1つのマスだけが訪問されない場合の数とします。

例えば、$F(1, 3) = 4$、$F(1, 4) = 15$、$F(1, 5) = 46$、$F(2, 3) = 16$、そして $F(2, 100) \bmod {10}^9 = 429\,619\,151$ です。

$F(10, {10}^{12})$ の下9桁を求めてください。
