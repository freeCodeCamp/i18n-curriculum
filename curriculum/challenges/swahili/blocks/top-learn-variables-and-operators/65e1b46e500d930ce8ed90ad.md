# --description--

Kuongeza au kupunguza nambari kwa moja ni mojawapo ya shughuli za kawaida za kihesabu.

Kwa hiyo, kuna waendeshaji maalum kwa ajili yake:

- Kuongeza `++` huongeza thamani ya kigezo kwa 1:

```js
let counter = 2;

// works the same as counter = counter + 1, but is shorter
counter++;      

console.log(counter); // 3
```

- Kupunguza `--` hupunguza thamani ya kigezo kwa 1:

```js
let counter = 2;

// works the same as counter = counter - 1, but is shorter
counter--;

console.log(counter); // 1
```

Waendeshaji `++` na `--` wanaweza kuwekwa kabla au baada ya kigezo.

- Wakati mwendeshaji anafuata kigezo, huitwa "postfix form": `counter++`.

- "Prefix form" ni wakati mwendeshaji anakuwa kabla ya kigezo: `++counter`.

Kauli hizi mbili hufanya jambo moja: kuongeza `counter` kwa `1`.

Je, kuna tofauti yoyote? Ndiyo, lakini unaweza kuiona tu ukitumia thamani inayorejeshwa na `++/--`.

Hebu tuelezee. Kama unavyojua, waendeshaji wote hurudisha thamani. Kuongeza/kupunguza si tofauti. Aina ya prefix hurudisha thamani mpya wakati aina ya postfix hurudisha thamani ya zamani (kabla ya kuongeza/kupunguza).

Ili kuona tofauti, hapa kuna mfano:

```js
let counter = 1;
let a = ++counter; // (*)

console.log(a); // 2
```

Kwa muhtasari:

- Ikiwa matokeo ya kuongeza/kupunguza hayatumiwi, hakuna tofauti ni aina gani ya kutumia:

```js
let counter = 0;
counter++;
++counter;

console.log( counter ); // 2, the lines above did the same
```

- Ikiwa unataka kuongeza thamani na mara moja kutumia matokeo ya mwendeshaji, unahitaji aina ya prefix:

```js
let counter = 0;
console.log( ++counter ); // 1
```

- Ikiwa unataka kuongeza thamani lakini kutumia thamani yake ya awali, unahitaji aina ya postfix:

```js 
let counter = 0;
console.log( counter++ ); // 0
```

## --text--

Ni matokeo gani ya kauli mbili za `console.log` katika msimbo wa JavaScript hapa chini?

```js
let counter = 1;
console.log(2 * ++counter); // Statement A
```

```js
let counter = 1;
console.log(2 * counter++); // Statement B
```

## --answers--

Hakuna tofauti; kauli zote mbili za `console.log` zitaonyesha matokeo sawa.

---

Kauli A itaonyesha `4` kwa sababu `++counter` huongeza thamani kabla ya kuzidisha. Kauli B itaonyesha `2` kwa sababu `counter++` huongeza thamani baada ya kuzidisha.

---

Kauli A itaonyesha `2` kwa sababu `++counter` ni kosa la sintaksia. Kauli B itaonyesha `4` kwa sababu `counter++` ni njia sahihi ya kuongeza kielekezi.

---

Kauli zote mbili zitaonyesha `3` kwa sababu kielekezi kinaongezwa katika kesi zote mbili kabla ya `console.log`.
