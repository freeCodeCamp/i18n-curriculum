# --explanation--

`by next month`에서 `by`는 마감 시간을 나타내며, 이는 어떤 일이 그 시간 이전이나 늦어도 그 시간까지 일어나야 함을 의미합니다.

문장 `We need to finish the report by Friday`은 `by`를 같은 방식으로 사용하여 명확한 시간 제한을 설정합니다.
