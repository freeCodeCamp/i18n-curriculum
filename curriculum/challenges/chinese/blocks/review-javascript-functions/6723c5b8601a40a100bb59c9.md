## JavaScript 函数

- 函数是可重复使用的代码片段，可执行特定任务。
- 可以使用 `function` 关键字定义函数，该关键字后跟一个名称、一个参数列表和一个执行任务的代码块。

```js
function addNumbers(x, y, z) {
  return x + y + z;
}

console.log(addNumbers(5, 3, 8)); // Output: 16
```

- 实参是调用函数时传递给函数的值。
- 函数调用是在程序中执行一个函数的过程，具体方法是在函数名称后加上括号，括号内可选择包含参数。
- 函数执行完毕后，总会返回一个值。
- 默认情况下，函数的返回值为 `undefined` 。
- `return` 关键字用于指定从函数返回的值，并结束函数的执行。
- 默认参数允许函数具有预定义值，如果调用函数时没有提供实参，就会使用这些值。 这使得函数更加灵活，并防止在省略某些实参时出现错误。

```js
const calculateTotal = (amount, taxRate = 0.05) => {
  return amount + (amount * taxRate);
};

console.log(calculateTotal(100)); // Output: 105
```

- 函数表达式是赋值给变量的函数。 这样，你就可以在代码中任何可以访问变量的地方使用该函数。

```js
const multiplyNumbers = function(firstNumber, secondNumber) {
  return firstNumber * secondNumber;
};

console.log(multiplyNumbers(4, 5)); // Output: 20
```

## 箭头函数

- 箭头函数是 JavaScript 中编写函数的一种更简洁的方法。

```js
const calculateArea = (length, width) => {
  const area = length * width;
  return `The area of the rectangle is ${area} square units.`;
};

console.log(calculateArea(5, 10)); // Output: "The area of the rectangle is 50 square units."
```

- 定义箭头函数时，不需要使用 `function` 关键字。
- 如果只使用一个参数，可以省略参数列表中的括号。

```js
const cube = x => {
  return x * x * x;
};

console.log(cube(3)); // Output: 27
```

- 如果函数体只有一个表达式，则可以省略大括号和 `return` 关键字。

```js
const square = number => number * number;

console.log(square(5)); // Output: 25
```

## 编程中的作用域

- **全局作用域**：这是 JavaScript 中最外层的作用域。 在全局作用域中声明的变量可以从代码中的任何地方访问，这些变量被称为全局变量。
- **局部作用域**：指在函数中声明的变量。 这些变量只能在声明它们的函数中访问，称为局部变量。
- **块级作用域**：块是由 `{}` 括起来的一组语句，比如 `if` 语句或循环中的代码块。
- 使用 `let` 和 `const` 的块级作用域可对变量的可访问性进行更精细的控制，有助于防止错误并使代码更具可预测性。
