# --description--

Cuando se hacen y se responden preguntas sobre cantidad, es importante utilizar `much` y `many` consistentemente. `Much` se usa con sustantivos incontables, mientras que `many` se usa con sustantivos contables. En una conversación, la pregunta y la respuesta deben coincidir en el uso de `much` o `many`.

Ejemplos:

Pregunta: `¿Cuánto tiempo tienes?` Respuesta: `No mucho.`

Pregunta: `¿Cuántos libros posees?` Respuesta: `No muchos.`
