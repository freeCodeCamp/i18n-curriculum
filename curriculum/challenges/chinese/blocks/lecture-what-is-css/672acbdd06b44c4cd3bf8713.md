# --interactive--

当你开始使用 `HTML` 和 `CSS` 时，你会注意到即使在你编写任何 `CSS` 之前，一些样式已经应用到你的网页上。这些样式称为“默认浏览器样式”或“用户代理样式”。

这些默认浏览器样式提供了基本格式，以确保 HTML 元素在所有浏览器中以可读的方式显示。

但是，这些样式在不同浏览器之间可能会略有不同，这就是为什么在设计你网站的一致外观时理解它们很重要。

让我们来看一些应用于各种 `HTML` 元素的常见默认浏览器样式。

标题是最常用的超文本标记语言元素之一，默认情况下具有不同的大小和权值样式。

例如，`h1`（最高级别的标题）通常比 `h2`、`h3` 等较低级别的标题更粗且字体大小更大。

这些标题标签有助于定义你的内容的层次结构和结构。

以下示例显示了从 `h1` 到 `h6` 的所有六个标题元素。

:::interactive_editor

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```

:::

另一个具有默认样式的元素是 `hr` 元素，它创建一个水平线，通常用于视觉上分隔内容的各个部分。浏览器通常会为该元素应用简单的线条样式。

为了更好地理解它，让我们看这个代码示例：

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<hr>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

水平线显示为一条细线，文本上下有间距以区分内容部分。

接下来，我们将查看 `blockquote` 元素。

引用块用于表示一段来自其他来源的引用文本。浏览器通常会添加缩排，有时还会将文本斜体显示。

缩排有助于将引用块与其余文本区分开来，清楚地表明该内容是引用自其他来源。

让我们来看这个代码示例：

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<blockquote>I think, therefore I am. (Rene Descartes)</blockquote>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

在上面的示例中，`blockquote` 元素将对文本产生缩排，该缩排由默认浏览器样式应用。

另一个具有默认样式的元素是锚点元素（`<a>`），它使用默认的蓝色和下划线样式，使其可识别为链接。

让我们来看以下超文本标记语言 示例：

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<a href="https://freecodecamp.org/">Visit the freeCodeCamp learn page</a>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

上述代码有四个段落元素，中间有一个锚点元素。锚点元素的样式为蓝色并带下划线，以提示用户访问 freeCodeCamp 学习页面。

最后，我们将查看无序和有序列表元素。

浏览器为列表添加基本格式，包括缩排和项目符号或数字，这取决于你使用的是无序列表（`ul`）还是有序列表（`ol`）。

让我们来看一个例子：

:::interactive_editor

```html
<p>Sample Paragraph</p>
<ol>
  <li>item 1</li>
  <li>item 2</li>
  <li>item 3</li>
</ol>
<ul>
  <li>item</li>
  <li>another item</li>
  <li>yet another item</li>
</ul>
<p>Ending Paragraph</p>
```

:::

在上述 `code` 示例中，我们使用了无序 `list` 和有序 `list`。这两个 `list` 默认都会稍微向右缩排。

对于所有这些示例，你看到了浏览器如何将默认间距、不同的字体大小和权值应用到这些 `HTML` 元素。

在后续课程中，你将学习如何使用 CSS reset 重置其中一些默认样式。
