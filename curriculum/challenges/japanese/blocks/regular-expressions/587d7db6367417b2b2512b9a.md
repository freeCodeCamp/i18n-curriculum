# --description--

前回のチャレンジでは、プラス記号`+`を使って1回以上繰り返される文字を探しました。今回は0回以上繰り返される文字に一致させる方法もあります。

これを行う文字はアスタリスク、またはスター記号です：`*`。

```js
let soccerWord = "gooooooooal!";
let gPhrase = "gut feeling";
let oPhrase = "over the moon";
let goRegex = /go*/;
soccerWord.match(goRegex);
gPhrase.match(goRegex);
oPhrase.match(goRegex);
```

順番に、3つの`match`の呼び出しは`["goooooooo"]`、`["g"]`、`null`の値を返します。

# --instructions--

`chewieRegex`という正規表現を作成してください。この正規表現は`*`文字を使い、大文字の`A`文字に続いて0回以上の小文字の`a`文字に一致させます。`chewieQuote`内で使います。フラグや文字クラスは必要なく、他の引用文には一致させないでください。
