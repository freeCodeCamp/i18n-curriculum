# --description--

En la lección anterior, aprendiste cómo ejecutar scripts de Python localmente. Pero puede haber ocasiones en las que no quieras crear programas completos en Python y solo necesites probar algo de código Python. Aquí es donde entra la consola interactiva de Python.

Pero primero, repasemos qué es un terminal.

Un terminal es una interfaz basada en texto que te permite interactuar con tu computadora escribiendo comandos. Cada sistema operativo viene con una app de terminal predeterminada. En macOS, puedes usar la app Terminal. En Windows, puedes usar Command Prompt o PowerShell. En Linux, cada entorno de escritorio tiene su propia app de terminal predeterminada, como GNOME Terminal o Konsole.

Abre una app de terminal y escribe `python` y presiona `Enter`. Esto iniciará una consola interactiva de Python. Una consola interactiva es un programa que te permite escribir comandos uno a la vez y ver los resultados.

Cuando inicias una nueva sesión, podrías ver este tipo de salida inicialmente:

```bash
Python 3.12.2 (main, Mar 21 2024, 22:48:26) [Clang 14.0.3 (clang-1403.0.22.14.1)] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>> 
```

El símbolo `>>>` significa que Python está esperando que escribas un comando. Intenta imprimir `"Hello, world!"` en el terminal:

```bash
print("Hello, world!")
```

Deberías ver que el texto aparece así:

```bash
>>> print("Hello, world!")
Hello, world!
```

Después de que el texto se imprime, Python vuelve a mostrar lo siguiente:

```bash
>>>
```

Esto significa que puedes escribir otro comando.

El intérprete de Python sigue lo que se conoce como el ciclo Leer, Evaluar, Imprimir, Ciclar. O REPL por sus siglas en inglés. Cada vez que escribes comandos, el intérprete lee la entrada, la evalúa, imprime el resultado y vuelve a mostrar el `>>>` para que puedas escribir más comandos.

¿Qué pasa si intentas escribir un comando inválido como este?

```bash
>>> something random
```

Bueno, el intérprete de Python seguirá el mismo proceso REPL. En este caso, recibirás un mensaje de error:

```bash
>>> something random
  File "<stdin>", line 1
    something random
              ^^^^^^
SyntaxError: invalid syntax
```

Si quieres salir de la consola interactiva, puedes escribir `exit()` o presionar `Ctrl + D` (Mac/Linux) o `Ctrl + Z + Enter` (Windows).

Usar la consola interactiva de Python es ideal para pequeños experimentos con código o exploraciones básicas. Mientras avanzas en las lecciones restantes del curso, puedes escribir algunos de los nuevos métodos que aprendas dentro de una consola interactiva.

Si planeas trabajar con programas más largos o en varios archivos, entonces es mejor trabajar en un editor de código o IDE.
