# --description--

你已经学习了指示代词，如 `this`、`that`、`these` 和 `those`。记住要使用它们来指代特定的事物或人，考虑它们在时间或空间上的接近程度。

所有格分支，通常由撇号和 `s`（如 `function's`）表示，用于显示所有权或归属，通常针对人或对象。它告诉人们某物属于或是另一个事物的一部分。

单词 `purpose` 指的是某事被执行或使用的原因，或某事存在的理由。例如，`The purpose of a clock is to tell time.`
