# --interactive--

La `Web Animations` API (WAAPI) ti permette di creare e controllare animazioni direttamente all’interno di JavaScript. Con WAAPI puoi lavorare con le animazioni in modo più dinamico, rendendo più semplice manipolarle.

Diamo un’occhiata più da vicino a WAAPI, così potrai iniziare a lavorare con le animazioni direttamente nel tuo codice JavaScript. Al centro di WAAPI c’è il costruttore `Animation`, che offre diversi metodi e proprietà di istanza che ti consentono di animare dinamicamente gli elementi.

Un metodo importante nel costruttore `Animation` è `animate()`. Ti permette di creare un’animazione specificando i fotogrammi chiave e opzioni come durata, direzione, easing e iterazioni.

Ecco la sintassi base del metodo `animate()`:

```js
element.animate(keyframes, options);
```

Vediamo un esempio.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="square" id="square"></div>
<script src="index.js"></script>
```

```css
body {
  background: #f1f1f1;
  display: flex;
  align-items: center;
  justify-content: center;
  height: 100vh;
}

.square {
  background: #1b1b32;
  width: 10rem;
  aspect-ratio: 1/1;
}
```

```js
const square = document.querySelector("#square");

const animation = square.animate(
  [{ transform: "translateX(0px)" }, { transform: "translateX(100px)" }],
  {
    duration: 2000, // makes animation lasts 2 seconds
    iterations: Infinity, // loops indefinitely
    direction: "alternate", // moves back and forth
    easing: "ease-in-out" // smooth easing
  }
);
```

:::

Il risultato nel browser sarà un quadrato blu che si muove avanti e indietro orizzontalmente.

I metodi di istanza del costruttore `Animation` includono:

```js
play()
pause()
reverse()
finish()
cancel()
```

E le proprietà di istanza includono:

```md
playbackRate
currentTime
startTime
effect
timeline
playState
finished
onfinish
oncancel
```

Modifichiamo l’esempio per usare i metodi `play()`, `pause()` e la proprietà `onfinish`. Ecco la nuova versione:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="square" id="square"></div>

<button id="playBtn">Play</button>
<button id="pauseBtn">Pause</button>
<script src="index.js"></script>
```

```css
body {
  background: #f1f1f1;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  height: 100vh;
}
.square {
  background: #1b1b32;
  width: 10rem;
  aspect-ratio: 1/1;
  margin-bottom: 20px;
}
button {
  margin: 10px;
  padding: 10px 20px;
  font-size: 16px;
}
```

```js
const square = document.querySelector("#square");
const playBtn = document.querySelector("#playBtn");
const pauseBtn = document.querySelector("#pauseBtn");

const animation = square.animate(
  [{ transform: "translateX(0px)" }, { transform: "translateX(200px)" }],
  {
    duration: 5000, // Animation lasts 5 seconds
    // iterations: Infinity, // Loops indefinitely
    direction: "alternate", // Moves back and forth
    easing: "ease-in-out" // Smooth easing function
  }
);

// Set the onfinish property to log a message when the animation ends
animation.onfinish = () => {
  console.log("Animation finished!");
};

// Play the animation when the "Play" button is clicked
playBtn.addEventListener("click", () => {
  animation.play();
  console.log("You start the animation");
});

// Pause the animation when the "Pause" button is clicked
pauseBtn.addEventListener("click", () => {
  animation.pause();
  console.log("You pause the animation");
});
```

:::

Il risultato nel browser mostrerà il riquadro blu che si muove da sinistra a destra quando si clicca il pulsante `play`.

WAAPI estende le capacità delle animazioni CSS offrendo un controllo più dinamico sulle animazioni direttamente dentro JavaScript.

Con le animazioni CSS definisci le animazioni in modo dichiarativo usando proprietà come `animation-name`, `animation-duration` e `animation-timing-function`. Hai anche la possibilità di fare la stessa cosa usando il metodo `animate()` di WAAPI.

La differenza è che puoi controllare le animazioni create con il metodo `animate()` in modo più diretto e dinamico, mentre con le animazioni CSS devi fare molto di più definendo stili personalizzati e attivandoli dentro il tuo file JavaScript.

L’animazione CSS è ideale per animazioni semplici e dichiarative che si avviano automaticamente. Queste includono effetti al passaggio del mouse, transizioni o animazioni che non richiedono molta interazione una volta attivate. Se la tua animazione deve rispondere a interazioni dell’utente come clic, scroll o vuoi che l’utente possa mettere in pausa, invertire o cambiare la velocità in modo dinamico, WAAPI è la scelta migliore.

Puoi considerare di combinare sia WAAPI che animazioni CSS quando vuoi la semplicità del CSS per la configurazione base ma hai bisogno di controllare le animazioni durante l’esecuzione.
