# --interactive--

Wenn Sie in JavaScript eintauchen, werden Sie schnell feststellen, dass mathematische Operationen eine häufige Aufgabe sind. Während einfache arithmetische Operatoren einfache Berechnungen erledigen können, bietet JavaScript ein eingebautes `Math`-Objekt, um komplexere mathematische Herausforderungen zu bewältigen.

Dieses praktische Werkzeug stellt verschiedene Methoden bereit, die es einfacher machen, fortgeschrittene Berechnungen durchzuführen und Zahlen zu manipulieren. Lassen Sie uns diese Methoden erkunden und sehen, wie sie Ihre Programmiererfahrung vereinfachen können.

Die `Math.random()`-Methode erzeugt eine zufällige Fließkommazahl zwischen `0` (einschließlich) und `1` (ausschließlich). Das bedeutet, dass der mögliche Wert `0` sein kann, aber niemals tatsächlich `1` erreicht. Hier ist ein Beispiel für die Arbeit mit der `Math.random()`-Methode:

:::interactive_editor

```js
const randomNum = Math.random();

console.log(randomNum);
// any number between 0 and 1 – 0 inclusive and 1 exclusive
```

:::

`Math.min()` und `Math.max()` nehmen jeweils eine Menge von Zahlen und geben den Minimum- bzw. Maximumwert zurück. Hier ist ein Beispiel, wie man beide Methoden verwendet:

:::interactive_editor

```js
const smallest = Math.min(1, 5, 3, 9);
console.log(smallest); // 1

const largest = Math.max(1, 5, 3, 9);
console.log(largest); // 9
```

:::

Die erste `console.log()` wird die Zahl `1` ausgeben, da `1` die kleinste Zahl in dieser Liste ist. Und die zweite `console.log()` wird die Zahl `9` ausgeben, da `9` die größte Zahl in dieser Liste ist.

Wenn Sie Zahlen auf- oder abrunden möchten, können Sie die `Math.ceil()`- und `Math.floor()`-Methoden verwenden. Hier ist ein Beispiel für die Arbeit mit `Math.ceil()`:

:::interactive_editor

```js
console.log(Math.ceil(4.3)); // 5
```

:::

`Math.ceil()` rundet `4.3` auf die nächste ganze Zahl auf, was in diesem Fall `5` ist. Schauen wir uns nun das Abrunden einer Zahl an:

:::interactive_editor

```js
console.log(Math.floor(4.7)); // 4
```

:::

`Math.floor()` rundet `4.7` auf die nächste ganze Zahl ab, was in diesem Fall `4` ist. `Math.round()` ist die Kombination aus `Math.ceil()` und `Math.floor()`. Es rundet eine Zahl auf die nächste ganze Zahl, wobei die Nachkommastelle berücksichtigt wird:

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

Wenn die Nachkommastelle kleiner als `5` ist, wird die Zahl abgerundet. Und wenn die Nachkommastelle `5` oder größer ist, wird die Zahl aufgerundet. Eine praktische Anwendung von `Math.floor()` und `Math.random()` ist das Erzeugen einer Zufallszahl zwischen zwei ganzen Zahlen. Hier ist die Syntax dafür:

:::interactive_editor

```js
const max = 10;
const min = 5;
const randomNum = Math.floor(Math.random() * (max - min + 1)) + min;
console.log(randomNum);
```

:::

Das Erzeugen einer Zufallszahl zwischen `20` und `1` würde so aussehen:

:::interactive_editor

```js
const randomNumBtw1And20 = Math.floor(Math.random() * 20) + 1;
console.log(randomNumBtw1And20);
```

:::

Eine weitere hilfreiche `Math`-Methode ist die `Math.trunc()`-Methode. `Math.trunc()` entfernt den Dezimalteil einer Zahl und gibt nur den ganzzahligen Anteil zurück, ohne zu runden:

:::interactive_editor

```js
console.log(Math.trunc(2.9)); // 2
console.log(Math.trunc(9.1)); // 9
```

:::

Wenn Sie die Quadratwurzel oder Kubikwurzel einer Zahl benötigen, können Sie die `Math.sqrt()`- und `Math.cbrt()`-Methoden verwenden:

:::interactive_editor

```js
console.log(Math.sqrt(81)); // 9
console.log(Math.cbrt(27)); // 3
```

:::

Die erste Ausgabe gibt `9` aus, weil die Quadratwurzel von `81` `9` ist, während die zweite Ausgabe `3` ausgibt, weil die Kubikwurzel von `27` `3` ist. Wenn Sie den Absolutwert einer Zahl benötigen, können Sie die `Math.abs()`-Methode verwenden:

:::interactive_editor

```js
console.log(Math.abs(-5)); // 5
console.log(Math.abs(5)); // 5
```

:::

`Math.abs()` gibt den Absolutwert einer Zahl zurück und wandelt negative Zahlen in positive um. Die letzte Methode, die wir uns ansehen, ist die `Math.pow()`-Methode:

:::interactive_editor

```js
console.log(Math.pow(2, 3)); // 8
console.log(Math.pow(8, 2)); // 64
```

:::

`Math.pow()` nimmt zwei Zahlen und hebt die erste auf die Potenz der zweiten. Es gibt viele weitere Methoden, die zum `Math`-Objekt gehören und die Sie selbst erkunden können. Dies sind jedoch nur einige der häufiger verwendeten Methoden, die in JavaScript-Codebasen zu finden sind.
