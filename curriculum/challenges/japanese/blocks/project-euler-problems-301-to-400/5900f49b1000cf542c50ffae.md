# --description--

正の整数 $n$ に対して、$f(n)$ を、10進数で表したときに数字がすべて $≤ 2$ である最小の正の $n$ の倍数と定義します。

したがって、$f(2) = 2$、$f(3) = 12$、$f(7) = 21$、$f(42) = 210$、$f(89) = 1\\,121\\,222$ となります。

また、$\displaystyle\sum_{n = 1}^{100} \frac{f(n)}{n} = 11\\,363\\,107$ です。

$\displaystyle\sum_{n = 1}^{10\\,000} \frac{f(n)}{n}$ を求めてください。
