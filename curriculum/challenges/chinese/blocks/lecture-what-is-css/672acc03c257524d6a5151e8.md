# --interactive--

在 CSS 中，`width` 和 `height` 属性用于控制网页上元素的维度。

这些属性可以用不同的单位定义，例如像素（`px`）、百分比（`%`）、视口单位（`vw`、`vh`）等。

`width` 属性指定元素的宽度。如果你不指定宽度，则默认设置为 `auto`。这让浏览器根据元素的内容、父元素和显示类型来确定元素的宽度。对于 `div` 元素，`width: auto` 使其扩展以填充父容器的整个宽度。

`height` 属性指定元素的高度。同样，默认情况下高度为 `auto`，这意味着它会根据内部内容进行调整。

这是一个使用 `width` 和 `height` 属性的示例：

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 100px;
      height: 100px;
      background-color: skyblue
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

在这个例子中，我们有一个类名为 `box` 的 `div` 元素。该元素的高度和宽度将占用 `100px`，背景颜色将是 `skyblue`。

像素是 CSS 中的固定大小测量单位，提供对维度的精确控件。 

`min-width` 属性指定元素的最小宽度。即使内部内容更小，元素也不会缩小到低于该值。

`min-height` 指定元素的最小高度。它确保元素不会比设置的值更矮。

这是一个示例：

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 50px;
      min-width: 100px;
      height: 50px;
      min-height: 100px;
      background-color: lightcoral;
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

上面的示例演示了 `min-width` 和 `min-height` 的工作原理。

尽管盒子的 `width` 和 `height` 设置为 50px，但它实际上会是 `100px` × `100px`。这是因为 `min-width` 和 `min-height` 设置为 `100px`，它们比指定的 `width` 和 `height` 更大。

请记住，如果 `min-width` 或 `min-height` 大于 `width` 或 `height`，它们将覆写较小的值。这确保了元素不会变得过小，这对于维护一致且可用的设计非常重要。

`max-width` 指定元素可以增长到的最大宽度，即使有足够的空间使其更宽。

`max-height` 指定元素可以增长到的最大高度，无论内容大小如何。

这是一个示例：

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 200px;
      max-width: 150px;
      height: 200px;
      max-height: 150px;
      background-color: lightgreen;
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

上面的示例演示了 `max-width` 和 `max-height` 如何覆写 `width` 和 `height`。即使盒子被设置为 `200px` 乘以 `200px`，它实际上将是 `150px` 乘以 `150px`。这是因为 `max-width` 和 `max-height` 被设置为较小的 `150px`。

请记住，当 `max-width` 或 `max-height` 小于 `width` 或 `height` 时，它们具有优先级。这对于控制你版型中元素的最大尺寸非常重要。

CSS 优先考虑 `min-width` 和 `min-height`，而不是 `width` 和 `height`。如果值超过限制，`max-width` 和 `max-height` 会限制维度。

例如，如果你将 `width` 设置为 `600px`，将 `max-width` 设置为 `500px`，元素的宽度将被限制为 `500px`。`max-width` 覆写了更宽的设置，使元素保持在指定的最大尺寸内。

这确保元素保持在所需的大小范围内，无论标准宽度和高度值如何。
