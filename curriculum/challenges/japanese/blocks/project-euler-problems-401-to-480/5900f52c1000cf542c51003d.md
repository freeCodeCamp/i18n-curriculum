# --description--

すべての整数 $n > 1$ に対して、関数族 $f_{n, a, b}$ は次のように定義されます：

$f_{n, a, b}(x) ≡ ax + b\bmod n$ ただし $a, b, x$ は整数で、$0 \lt a \lt n$、$0 \le b \lt n$、$0 \le x \lt n$。

すべての $0 \le x \lt n$ に対して $f_{n, a, b}(f_{n, a, b}(x)) \equiv f_{n, a, b}(x)\bmod n$ が成り立つ場合、$f_{n, a, b}$ をリトラクションと呼びます。

$R(n)$ を $n$ に対するリトラクションの数とします。

$F(N) = \displaystyle\sum_{n = 1}^N R(n^4 + 4)$。

$F(1024) = 77\\,532\\,377\\,300\\,600$。

$F({10}^7)$ を求めてください。答えは $1\\,000\\,000\\,007$ で割った余りで提出してください。
