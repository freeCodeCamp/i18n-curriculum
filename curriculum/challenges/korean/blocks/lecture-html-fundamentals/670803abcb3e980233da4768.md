# --interactive--

`div` 요소는 다른 요소들을 그룹화하는 컨테이너로 사용됩니다.

다음은 `div` 요소의 예입니다. `div` 요소 안에 또 다른 단락 요소를 추가하고 미리보기 창에서 변화를 확인하세요. 미리보기를 보려면 인터랙티브 편집기를 활성화해야 합니다.

:::interactive_editor

```html
<div>
  <p>Example paragraph element.</p>
</div>
```

:::

주로 CSS 스타일을 공유할 HTML 요소들을 그룹화할 때 `div` 요소를 사용합니다. CSS에 대해서는 앞으로의 수업과 워크숍에서 더 배우게 됩니다.

실제 코드베이스에서 `div` 요소가 흔히 사용되지만, 과도하게 사용하지 않도록 주의해야 합니다. 때로는 다른 요소가 더 적합할 때가 있습니다.

예를 들어, 콘텐츠를 섹션으로 나누고 싶다면 `section` 요소가 `div` 요소보다 더 적합합니다.

첫 번째 `section` 요소 아래에 또 다른 `section` 요소를 추가하세요. 그런 다음 `section` 요소 안에 `h2`와 `p` 요소를 넣으세요. 원하는 텍스트를 사용해도 되고, 미리보기 창에서 변화를 확인할 수 있습니다. 예제와 상호작용하려면 인터랙티브 편집기를 활성화해야 합니다.

:::interactive_editor

```html
<section>
  <h2>Mammals</h2>
  <p>
    Mammals are warm-blooded animals with fur or hair. Most give birth to live
    young.
  </p>
  <ul>
    <li>Lion</li>
    <li>Elephant</li>
    <li>Dolphin</li>
  </ul>
</section>
```

:::

`section` 요소는 의미론적 의미가 있는 반면 `div` 요소는 의미론적이지 않습니다. 의미론은 언어에서 단어나 구의 의미를 뜻합니다. HTML이라는 언어에서도 요소마다 고유한 의미론적 의미가 있습니다. 따라서 `section` 요소를 사용하면 브라우저가 그 의미론적 의미를 인식하여 데스크톱, 모바일 등 어디서든 이를 섹션으로 처리합니다.
