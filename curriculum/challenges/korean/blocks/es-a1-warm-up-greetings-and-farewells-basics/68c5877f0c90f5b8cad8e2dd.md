# --explanation--

`Buenas noches`는 밤에 사용하는 인사입니다. "Good night"로 번역됩니다.

`Buenas noches`는 저녁에 누군가를 맞이할 때 **또는** 밤에 누군가에게 인사하거나 작별 인사를 할 때 모두 사용할 수 있습니다. 예를 들어, 아래 문장은 밤에 인사하거나 작별 인사할 때 모두 사용할 수 있습니다:

`Buenas noches, señora López.` – 좋은 밤 되세요, Mrs. López.
