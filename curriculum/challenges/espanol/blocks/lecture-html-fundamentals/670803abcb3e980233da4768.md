# --interactive--

El elemento `div` se usa como contenedor para agrupar otros elementos.

Aquí tienes un ejemplo de un elemento `div`. Agrega otro elemento párrafo dentro del elemento `div` y observa los cambios en la ventana de vista previa. Para ver las vistas previas, necesitarás activar el editor interactivo.

:::interactive_editor

```html
<div>
  <p>Example paragraph element.</p>
</div>
```

:::

Principalmente usarás el elemento `div` cuando quieras agrupar elementos HTML que compartirán un conjunto de estilos CSS. Aprenderás más sobre CSS en futuras lecciones y talleres.

Aunque el elemento `div` se usa comúnmente en bases de código del mundo real, debes tener cuidado de no usarlo en exceso. Hay ocasiones en las que otro elemento sería más apropiado.

Por ejemplo, si quieres dividir tu contenido en secciones, entonces el elemento `section` sería más apropiado que un elemento `div`.

Agrega otro elemento `section` debajo del primero. Luego, dentro del elemento `section`, agrega elementos `h2` y `p`. Puedes usar el texto que quieras y verás los cambios en la ventana de vista previa. Para interactuar con el ejemplo, necesitarás habilitar el editor interactivo.

:::interactive_editor

```html
<section>
  <h2>Mammals</h2>
  <p>
    Mammals are warm-blooded animals with fur or hair. Most give birth to live
    young.
  </p>
  <ul>
    <li>Lion</li>
    <li>Elephant</li>
    <li>Dolphin</li>
  </ul>
</section>
```

:::

The `section` element has semantic meaning over the `div` element which is not semantic. La semántica es el significado de las palabras o frases en un idioma. En HTML, que es un lenguaje, los elementos también tienen su propio significado semántico. Entonces, esto significa que si usas un elemento `section`, el navegador captará su significado semántico y entenderá que debe tratarlo como una sección - en escritorios, móviles, lo que sea.
