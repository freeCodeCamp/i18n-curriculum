# --interactive--

In CSS, ci sono diversi modi per aggiungere bordi intorno alle immagini, ognuno con opzioni di definizione dello stile e livelli di controllo diversi.

Esploriamo alcune delle tecniche più comuni e versatili.

Il modo più semplice per aggiungere un bordo a un'immagine è usare la proprietà `border`. Questa proprietà è una scorciatoia che ti permette di definire la larghezza, lo stile e il colore del bordo tutto in una volta.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

Questa regola CSS aggiunge un bordo rosso solido di 2 pixel di larghezza intorno a tutti gli elementi `img`. Puoi modificare la larghezza, lo stile (come `dashed`, `dotted` o `double`) e il colore per adattarli al tuo design.

Se ti serve più controllo sui singoli lati del bordo, puoi usare le proprietà specifiche del bordo per ogni lato:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border-top: 10px solid red;
  border-right: 10px dashed green;
  border-bottom: 10px dotted blue;
  border-left: 10px double purple;
}
```

:::

Questo ti permette di creare stili di bordo unici per ogni lato dell'immagine.

Un altro modo per creare un effetto bordo è usare la proprietà `outline`. Pur essendo simile a border, outline non influisce sulle dimensioni o sul layout dell'elemento:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  outline: 3px solid gold;
}
```

:::

Questo crea un contorno dorato intorno all'immagine senza modificarne la dimensione o la posizione.

Se vuoi creare angoli arrotondati sul bordo, puoi usare la proprietà `border-radius` insieme a border:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

Ricorda, queste tecniche possono essere combinate e personalizzate per creare effetti di bordo unici. La scelta del metodo dipende dai requisiti specifici del tuo design e dal livello di complessità di cui hai bisogno.
