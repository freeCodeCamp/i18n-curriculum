# --interactive--

當使用 JavaScript 時，你可能會遇到需要重複某個字串特定次數的情況。

無論你是在產生重複的樣式還是單純複製文字，`repeat()` 方法都提供了一種簡單且有效的方式來達成這個目的。

`repeat()` 方法是 JavaScript 中的內建函式，允許你將字串重複指定的次數。以下是基本語法：

```js
string.repeat(count);
```

`string` 是你想要重複的字串，而 `count` 是你想要重複該字串的次數。以下是一個範例：

:::interactive_editor

```js
let word = "Hello!";
let repeatedWord = word.repeat(3);
console.log(repeatedWord);  // "Hello!Hello!Hello!"
```

:::

在這種情況下，字串 `Hello!` 重複了三次，結果為 `Hello!Hello!Hello!`。

雖然 `repeat()` 方法很有用，但有幾個例外和限制需要注意。

`count` 參數必須是非負數字。如果你傳入負數字，JavaScript 將會丟出 `RangeError`。

```js
let word = "Test";
console.log(word.repeat(-1));  // Throws RangeError: Invalid count value
```

`count` 必須是有限數字。如果你嘗試將字串重複無限次數或使用 `Infinity` 作為 `count`，你也會得到 `RangeError`。

在 JavaScript 中，`Infinity` 是一個特殊值，用來表述無限的數量。它用於表示比任何有限數字都大的數字。

```js
let word = "Test";
console.log(word.repeat(Infinity));  // Throws RangeError: Invalid count value
```

如果計數不是整數（例如像 `2.5` 這樣的小數），`repeat()` 方法會將其向下取整到最接近的整數。

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(2.5));  // "TestTest"
```

:::

如果你傳入 `0` 作為計數，`repeat()` 方法將傳回空字串。

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(0));  // ""
```

:::

`repeat()` 方法可以簡化涉及字串複製的任務，使你的程式碼更簡潔且易讀。

無論你是在產生重複的文字樣式或是用字元填滿空間，`repeat()` 都能讓你免於撰寫迴圈或更複雜的程式碼。
