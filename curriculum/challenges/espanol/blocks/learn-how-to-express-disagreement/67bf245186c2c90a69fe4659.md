# --explanation--

`Same` significa sin cambios o idéntico. Por ejemplo:

- `We have the same task to complete this week.` - Esto significa que la tarea es idéntica para todos.

- `They have the same opinion about the new software update.` - Esto significa que ambas personas están de acuerdo con la actualización.

Bob dice, `we still have the same issue`, lo que significa que el problema no ha cambiado.
