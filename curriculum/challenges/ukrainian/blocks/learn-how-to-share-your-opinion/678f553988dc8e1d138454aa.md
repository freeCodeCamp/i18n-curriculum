# --explanation--

`Channel` використовується для позначення інструменту комунікації, де члени команди можуть обмінюватися повідомленнями або інформацією. Наприклад:

`We discuss project updates on the Slack channel.` — це означає, що ви використовуєте платформу Slack для обговорення оновлень.

`Poll` означає процес збору думок шляхом запитування групи людей вибрати з різних варіантів. Наприклад:

`The team created a poll to decide which platform to use for the project.` — це означає, що команда попросила вхідні дані через опитування, щоб прийняти рішення.

`Votes` — це вибори, зроблені учасниками опитування, які представляють їхню думку. Наприклад:

`There were 15 votes for using Google Cloud.` — це означає, що 15 людей обрали Google Cloud як свій улюблений варіант в опитуванні.
