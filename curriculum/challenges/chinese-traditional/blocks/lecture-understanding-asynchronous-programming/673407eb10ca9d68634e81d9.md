# --description--

Geolocation API 提供網站請求使用者位置的方法。重要的是要注意，出於隱私原因，使用者必須先授權，網站才能存取他們的位置。

今天我們將專注的主要方法是 `getCurrentPosition`。這個方法用於收集裝置的地理位置。以下是你可能如何使用 `getCurrentPosition` 的範例：

```js
navigator.geolocation.getCurrentPosition(
  (position) => {
    console.log("Latitude: " + position.coords.latitude);
    console.log("Longitude: " + position.coords.longitude);
  },
  (error) => {
    console.log("Error: " + error.message);
  }
);
```

在這段程式碼中，我們正在呼叫 `getCurrentPosition` 並傳入一個函式（程式），當成功取得位置時該函式（程式）將被呼叫。這個位置物件包含各種資訊，但我們只關注 `latitude` 和 `longitude`。

如果取得位置時發生問題，錯誤將會記錄到主控台。 

`getCurrentPosition` 方法會根據裝置及其設定，使用 GPS、Wi-Fi 網路或 IP 位址地理定位。定位完成後，會以一個位置物件呼叫成功回呼函式（function）。

位置物件包含各種屬性，其中最常用的是 `latitude` 和 `longitude`，但它也可以包含 `altitude`、`accuracy`、`speed` 和 `heading` 等。

使用地理定位時，一個重要的考量是使用者隱私。向你的使用者說明你為何需要他們的地理位置資料以及你將如何使用這些資料。
