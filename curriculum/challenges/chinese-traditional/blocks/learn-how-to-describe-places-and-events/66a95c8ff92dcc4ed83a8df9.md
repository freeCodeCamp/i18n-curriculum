# --explanation--

`How many` 用來詢問某物的數量。`Usually` 顯示某事定期發生。 

`Present Simple` 時態的疑問句用來詢問習慣、事實或一般真理。 例如：

- `How many books do you read in a month?`－它詢問某人通常每月閱讀多少本書。

- `How many people usually attend the meeting?`－它詢問一般會議通常有多少人參加。
