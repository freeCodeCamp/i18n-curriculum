# --explanation--

`Simple Past`의 긍정문은 특정 규칙을 따릅니다:

동사 `to be`의 형태는 `was` (단수)와 `were` (복수)입니다. 이 형태들은 조동사 없이 사용됩니다. 예를 들어:

- `The project was successful.` - 이 문장은 프로젝트가 좋은 결과를 가져왔다는 뜻입니다.

- `They were happy with the results.` - 이 문장은 프로젝트 결과에 대해 `they`가 어떻게 느꼈는지를 나타냅니다.

규칙 동사의 단순 과거형은 기본 동사에 `-ed`를 추가하여 만듭니다. 예를 들어:

- `I worked on the project yesterday.` - 동사 `-ed`의 기본형에 `work`를 추가하여 과거의 순간에 일어난 동작을 표현합니다.
