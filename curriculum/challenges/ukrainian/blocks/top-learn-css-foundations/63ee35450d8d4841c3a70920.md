# --description--

Внутрішній CSS (або вставлений CSS) передбачає додавання CSS безпосередньо у HTML-файл замість створення окремого файлу. За внутрішнім методом усі правила розміщуються всередині пари відкриваючих і закриваючих `<style>` тегів, які потім розміщуються всередині відкриваючих і закриваючих `<head>` тегів вашого HTML-файлу. Оскільки стилі розміщуються безпосередньо всередині `<head>` тегів, елемент `<link>`, який потрібен для зовнішнього методу, більше не потрібен.

Окрім цих відмінностей, синтаксис точно такий самий, як і у зовнішнього методу (селектор, фігурні дужки, оголошення):

```html
<head>
  <style>
    div {
      color: white;
      background-color: black;
    }

    p {
      color: red;
    }
  </style>
</head>
<body>...</body>
```

Цей метод може бути корисним для додавання унікальних стилів до однієї сторінки сайту, але він не розділяє стилі так, як зовнішній метод, і залежно від кількості правил і оголошень HTML-файл може стати досить великим.

# --questions--    

## --text--

Яка з наведених відмінностей існує між внутрішнім і зовнішнім методами CSS?

## --answers--

Зовнішній метод розміщує CSS-правила у окремому файлі, тоді як внутрішній метод розміщує CSS-правила безпосередньо у HTML-файлі.

---

Внутрішній метод тримає CSS окремо від HTML, тоді як зовнішній метод вставляє CSS безпосередньо в HTML.

---

Внутрішній метод використовує елемент `<link>` для зв’язку CSS з HTML, тоді як зовнішній метод вставляє CSS безпосередньо в HTML.
