# --description--

이전 모듈에서 배운 것처럼, 루프는 코드 블록을 정해진 횟수만큼 반복하는 데 사용됩니다. 이번 수업에서는 Python에서 다양한 유형의 루프를 다루는 방법을 배우세요.

먼저 살펴볼 루프는 `for` 루프입니다. 다음은 `for` 루프를 사용해 리스트를 순차적으로 반복하며 각 항목을 콘솔에 출력하는 예시입니다:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)
```

결과는 다음과 같습니다:

```md
Rust
Java
Python
C++
```

루프 안에 `print(language)`가 들여쓰기 되어 있는 것을 확인하세요. 이 들여쓰기가 없으면 `IndentationError`가 발생합니다:

```python
"""
Traceback (most recent call last):
  File "<stdin>", line 4, in <module>
IndentationError: expected an indented block after 'for' statement on line 3
"""
```

또한 `for` 루프를 사용해 문자열 같은 다른 반복 가능한 객체를 순차적으로 반복할 수 있습니다. 다음은 `for` 루프를 사용해 문자열 `code`를 순차적으로 반복하며 각 문자를 출력하는 예시입니다:

```py
for char in 'code':
    print(char)
```

결과는 다음과 같습니다:

```md
c
o
d
e
```

JavaScript와 마찬가지로 Python에서도 `for` 루프를 중첩할 수 있습니다. 다음은 중첩된 `for` 루프를 사용하는 예시입니다:

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)
```

외부 루프는 `category` 리스트의 각 `categories`를 순차적으로 반복합니다. 각 `category`에 대해 내부 루프는 `food` 리스트의 각 `foods`를 순차적으로 반복합니다. 콘솔에 출력될 결과는 다음과 같습니다:

```md
Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
```

Python에서 사용할 수 있는 또 다른 유형의 루프는 `while` 루프입니다. 이 루프는 조건이 `False`가 될 때까지 코드 블록을 반복합니다. 다음은 추측 게임에 `while` 루프를 사용하는 예시입니다:

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')
```

이 예시에서는 값이 `secret_number`인 `3` 변수가 있고 초기 추측값은 `0`입니다. 그다음 `input` 함수를 사용해 사용자로부터 입력을 받고, 입력된 문자열을 `int()` 함수로 정수로 변환하여 `guess` 변수에 할당합니다. 사용자가 `3`을 입력해 맞히면 `while` 루프를 빠져나가고 메시지 `You got it!`가 콘솔에 출력됩니다. 그렇지 않으면 메시지 `Wrong! Try again.`가 출력되고 루프가 반복되어 다시 추측하도록 요청합니다.

예시 결과는 다음과 같습니다:

```md
Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
```

JavaScript와 마찬가지로 Python은 `break`와 `continue` 문을 지원합니다.

`break` 문은 루프 실행을 중단하는 데 사용됩니다. 다음은 `break` 문을 사용해 `developer_names` 리스트를 처리하는 예시입니다:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

이 예시에서는 `developer_names` 리스트를 순차적으로 반복하며 각 이름을 콘솔에 출력합니다. 이름이 `Naomi`과 같으면 루프를 빠져나갑니다. 따라서 콘솔에는 `Jess` 이름만 출력됩니다.

`continue` 문은 루프의 현재 반복을 건너뛰고 다음 반복으로 넘어가는 데 사용됩니다. 이전 예시를 `continue` 문으로 수정해 `break` 대신 사용해 보겠습니다:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

이제 콘솔 결과가 달라집니다. `Jess`와 `Tom` 이름이 출력되는데, `continue` 문이 `developer`가 `Naomi`일 때 두 번째 반복을 건너뛰어 그 이름을 출력하지 않기 때문입니다.

`for`와 `while` 루프 모두 `else` 절과 함께 사용할 수 있으며, 이 절은 루프가 `break` 문으로 종료되지 않았을 때만 실행됩니다. 다음은 여러 `for` 루프를 사용하는 예시입니다:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

이 예시에서는 무작위 단어 리스트가 있고, `for` 루프를 사용해 각 단어를 순차적으로 반복합니다. 외부 `for` 루프 안에는 각 단어의 각 글자를 순차적으로 반복하는 또 다른 `for` 루프가 있습니다. 글자의 소문자 버전이 모음이면, 단어와 포함된 모음을 출력한 뒤 내부 루프를 빠져나갑니다. 단어에 모음이 없으면 해당 메시지를 출력합니다.

콘솔 결과는 다음과 같습니다:

```md
'sky' has no vowels
'apple' contains the vowel 'a'
'rhythm' has no vowels
'fly' has no vowels
'orange' contains the vowel 'o'
```

루프는 Python에서 매우 흔하므로 익숙해지는 것이 중요합니다. 다음 수업들에서는 루프에서 `enumerate()`와 `range()` 함수를 다루는 방법을 배우세요.
