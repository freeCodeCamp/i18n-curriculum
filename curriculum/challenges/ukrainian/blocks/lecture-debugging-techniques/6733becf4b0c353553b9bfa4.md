# --description--

Інструкція `debugger` — це потужний інструмент JavaScript, який дозволяє призупинити виконання коду на певній лінії, щоб дослідити, що відбувається в програмі. При правильному використанні інструкція `debugger` може заощадити вам багато часу, допомагаючи зрозуміти, чому щось не працює так, як має.

JavaScript виконує ваш код зверху вниз. Коли JavaScript доходить до інструкції `debugger`, він одразу призупиняє виконання на цій лінії. Це дає вам можливість перевірити змінні, функції та загальний потік коду.

Таким чином ви можете побачити, що саме працює правильно, а що — ні. Однак це відбувається лише якщо відкриті інструменти розробника у вашому браузері. Інакше інструкція `debugger` ігнорується, і код продовжує виконуватися як зазвичай.

Крім того, коли ви використовуєте інструкцію `debugger`, сучасні браузери призупиняють виконання коду на вказаній лінії. Вони також дозволяють відновити виконання, натиснувши кнопку відтворення, але сама сторінка не перезавантажується автоматично, якщо це не зробити вручну.

Щоб використати інструкцію `debugger`, розмістіть її на тій лінії, де хочете призупинити виконання коду. Ось приклад:

```js
let firstNumber = 5;
let secondNumber = 10;
debugger; // Code execution pauses here
let sum = firstNumber + secondNumber;
console.log(sum);
```

У цьому прикладі, якщо консоль не відкрита до запуску коду, а потім ви її відкриєте, ви побачите `15` там. Поки консоль відкрита і ви перезавантажуєте сторінку, виконання призупиняється, щоб ви могли дослідити код.

Ось більш складний приклад:

```js
function calculateTotalPrice(price, discountPercentage) {
 debugger
 let discountAmount = (price * discountPercentage) / 100
 let totalPrice = price - discountAmount

 console.log(`Original Price: ${price}`)
 console.log(`Discount Amount: ${discountAmount}`)
 console.log(`Total Price after Discount: ${totalPrice}`)

 return totalPrice
}

let price = 100
let discount = 15

let finalPrice = calculateTotalPrice(price, discount)
console.log(`Final Price: ${finalPrice}`);
```

У цьому прикладі інструкція `debugger` розміщена на початку `calculateTotalPrice`. Коли викликається функція, її виконання призупиняється.

Якщо перезавантажити сторінку з відкритою консоллю, ви побачите, що сторінка продовжує перезавантажуватися, підтверджуючи, що виконання коду лише призупинене, а не зупинене. Якщо хочете продовжити виконання, натисніть кнопку відтворення.

Продовжуючи створювати свої JavaScript-програми, випробовуйте інструкцію `debugger` у своєму коді, щоб краще розуміти потік виконання, досліджувати змінні в реальному часі та швидко знаходити, де щось може йти не так.
