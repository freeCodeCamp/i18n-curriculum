# --description--

Lernen Sie `variables`, `data types` und `conditional statements` in der Programmierung.

Eine `variable` ist wie eine Box, in der Sie Informationen speichern können. Diese Informationen können Zahlen, Text oder andere Datentypen sein. Sie können den Inhalt der Box ändern, weshalb sie `variable` genannt wird – weil sie variieren kann. Zum Beispiel hält `x = 5` in `x` vorübergehend den Wert 5.

Eine `data type` ist eine Klassifikation, die den Datentyp angibt, den eine `variable` speichern kann. Häufige `data types` sind ganze Zahlen, Fließkommazahlen (Zahlen mit Dezimalstellen), Strings (Text) und Booleans (wahr oder falsch). Zum Beispiel ist der `age = 21` in `data type` eine ganze Zahl.

Eine `conditional statement` ist ein Befehl, der dem Computer sagt, etwas nur dann zu tun, wenn eine bestimmte Bedingung wahr ist. Es ist wie eine Entscheidung im echten Leben, die von einer bestimmten Situation abhängt. Häufige `conditional statements` sind `if-else`-Anweisungen. Zum Beispiel ist `if (temperature > 20) {console.log("It's warm outside!");}` eine `conditional statement`, die `console.log("It's warm outside!")` ausführt, wenn die Temperatur über 20 Grad liegt.
