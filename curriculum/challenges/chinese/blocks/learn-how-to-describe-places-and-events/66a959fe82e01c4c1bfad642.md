# --explanation--

`Might` 用于表达可能性。它表示某事是可能的，但不确定。例如：

- `I might go to the party if I finish my work.` - 我可能会去参加聚会，但我不确定。

- `She might call you later.` - 她可能稍后会调用你，但这不确定。
