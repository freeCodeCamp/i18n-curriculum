# --interactive--

ウェブページのスタイルをCSSで設定するとき、色を定義する最も一般的な方法の一つが16進カラー値を使うことです。16進カラー値、または16進コードは、RGBカラーモデルの色を簡潔に表す方法です。このレッスンでは、16進コードとは何か、CSSでどのように機能するのか、そしていつ使うべきかを探ります。

16進コード（hex codeの略）は、RGBカラーモデルの色を表すために使われる6文字の文字列です。「16進」とは、数字の`0`から`9`と文字の`A`から`F`を使う16進数のことを指します。

色の文脈では、16進コードは特定の色を構成する赤、緑、青（RGB）の量を指定します。各16進コードはハッシュ記号（`#`）で始まり、その後に赤、緑、青の強度を定義する6文字が続きます。

例えば、黒の16進コードは`#000000`で、白の16進コードは`#FFFFFF`です。最初の2文字が赤の値、次の2文字が緑、最後の2文字が青を表します。

CSSでは、16進コードを使って要素に色を適用できます。基本的な構文は以下の通りです。

```css
element {
  color: #RRGGBB;
}
```

ここで`#RRGGBB`は実際の16進コードのプレースホルダーです。`RR`、`GG`、`BB`の各ペアはそれぞれ赤、緑、青の強度を表します。

これらのペアは`00`（最も低い強度）から`FF`（最も高い強度）までの範囲です。数値が高いほど、その色の量が多くなります。例えば、いくつかの要素に16進カラーを適用してみましょう。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h1>This is a heading.</h1>
<p>This is a paragraph.</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}
p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

最初の場合、`#FF5733`は赤みがかったオレンジ色です。内訳は以下の通りです。

- `FF`は赤の最大値（完全に飽和）。
- `57`は中程度の緑の量。
- `33`は少量の青。

次の場合、`#4CAF50`は緑の色合いです。

- `4C`は赤の低い値。
- `AF`は強い緑の値。
- `50`は中程度の青の値。

16進コードは精度が高く使いやすいため人気があります。色を細かく調整できるので、一貫性が重要なデザインに最適です。

Adobe PhotoshopやFigmaなどの多くのデザインソフトは、選択した色の16進コードを提供しており、これをCSSファイルに直接コピー＆ペーストできます。

ウェブ開発では、16進コードはテキストの色、背景、ボーダー、その他の視覚要素の定義によく使われます。例えば：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example</button>
```

```css
body {
  background-color: #F0F0F0; /* Light gray background */
}

button {
  background-color: #007BFF; /* A shade of blue */
  color: #FFFFFF; /* White text */
}
```

:::

これにより、ウェブサイトやアプリケーション全体で色の一貫性が保たれます。

場合によっては、6文字ではなく3文字だけで書かれた省略形の16進コードを見かけることがあります。これは各色のペアの両方の文字が同じ場合に可能です。例えば、`#FF5733`は省略できませんが、`#FFFFFF`（白）は`#FFF`と書けます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h2>This is a heading.</h2>
```

```css
body {
  background-color: #000;
}

h2 {
  color: #FFF; /* White in shorthand */
}
```

:::

省略形の`#FFF`は`#FFFFFF`と同じ意味です。同様に、`#000`は黒を表し、完全な形では`#000000`です。

16進コードは精度と互換性に優れ、すべてのブラウザで1600万色以上を表現できます。特定のブランドカラーが必要な場合、名前付きカラーやHSLでは定義しにくい正確な色合いを制御したい場合、またはデザインソフトからデフォルトで16進コードをコピーする場合に最適です。
