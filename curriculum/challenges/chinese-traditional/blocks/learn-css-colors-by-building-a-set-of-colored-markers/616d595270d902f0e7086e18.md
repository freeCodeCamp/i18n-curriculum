# --description--

雖然你有三個獨立的標記 `div` 元素，但它們看起來像是一個大矩形。你應該在它們之間添加一些間距，以便更容易看清每個元素。

當縮寫 `margin` 屬性有兩個值時，它會將 `margin-top` 和 `margin-bottom` 設定為第一個值，並將 `margin-left` 和 `margin-right` 設定為第二個值。

在你的 `.marker` CSS 規則中，將 `margin` 屬性設為 `10px auto`。

# --hints--

你的 `.marker` CSS 規則應該有一個 `margin` 屬性設定為 `10px auto`。

```js
assert(new __helpers.CSSHelp(document).getStyle('.marker')?.margin === '10px auto');
```
