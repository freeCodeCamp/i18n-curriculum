# --description--

一個具有 $k$（十進位）位數的正整數稱為平衡數，當其前 $⌈\frac{k}{2}⌉$ 位數字的總和與其後 $⌈\frac{k}{2}⌉$ 位數字的總和相同，其中 $⌈x⌉$，讀作 $x$ 的天花板，是大於或等於 $x$ 的最小整數，因此 $⌈π⌉ = 4$ 且 $⌈5⌉ = 5$。

所以，例如，所有回文都是平衡的，13722 也是如此。

令 $T(n)$ 為所有小於 $10^n$ 的平衡數之和。

因此：$T(1) = 45$、$T(2) = 540$ 和 $T(5) = 334\\,795\\,890$。

尋找 $T(47)\\,mod\\,3^{15}$
