# --explanation--

`Data backup`는 시스템 오류, 손실 또는 손상 시 복구할 수 있도록 중요한 정보를 복사하는 것을 말합니다:

- `data`는 장치에 저장된 디지털 정보입니다.

- `backup`는 이 데이터를 안전하게 복사하는 과정입니다.

이 둘을 합쳐 `data backup`는 중요한 파일을 보호하고 IT 환경에서 위험을 최소화하는 데 필수적입니다. 예를 들어:

`The IT team performs a data backup every night.` - 팀은 모든 데이터를 매일 저장하여 안전을 보장합니다.
