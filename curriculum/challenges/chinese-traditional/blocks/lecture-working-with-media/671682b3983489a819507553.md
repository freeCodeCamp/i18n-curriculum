# --interactive--

在先前的課程中，你首次接觸到 `iframe` 元素。在本課程中，你將進一步學習如何使用 `iframe` 元素。此元素代表行內訊框。它是一個行內元素，用於將其他 HTML 內容直接嵌入 HTML 頁面中。該 HTML 內容可以是影片、地圖、另一個 HTML 元素，甚至是其他網頁。

這裡是一個嵌入來自 YouTube 的熱門 freeCodeCamp 課程的範例。若要觀看不同的影片，請啟用互動式編輯器並將 `src` 值更改為你選擇的影片。

:::interactive_editor

```html
<iframe
  width="400"
  height="400"
  src="https://www.youtube.com/embed/PkZNo7MFNFg?si=-UBVIUNM3csdeiWF"
  title="Learn JavaScript - Full Course for Beginners (YouTube video)"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
  referrerpolicy="strict-origin-when-cross-origin"
  allowfullscreen
></iframe>
```

:::

`src` 屬性指定你想要嵌入的頁面 URL。`width` 屬性指定 `iframe` 的寬度。`height` 屬性指定 `iframe` 的高度。`allowfullscreen` 屬性允許使用者將 `iframe` 顯示為全螢幕模式。為 `iframe` 指定 `title` 屬性也是良好做法，因為這對於無障礙性非常重要。

另一方面，`allow` 屬性讓你定義 iframe 可以或不可以做什麼。這稱為允許列表。在上述範例中，為它添加 `clipboard-write` 允許嵌入頁面將項目寫入你的剪貼簿。允許列表中的項目可以用分號或空格分隔，兩者也可以一起使用。

請注意，影片可以來自任何地方。不必一定來自像 YouTube 和 Vimeo 這樣的影片服務。

別忘了你也可以在 `iframe` 元素中嵌入地圖、另一個網頁或直接的 HTML。這裡有一個嵌入地圖的範例。

嘗試透過放大和縮小與地圖互動。

:::interactive_editor

```html
<h1>A Map from Openstreetmap.org Embedded with the iframe Element</h1>

<iframe
  width="425"
  height="350"
  src="https://www.openstreetmap.org/export/embed.html?bbox=3.006134033203125%2C6.150112578753815%2C3.6357879638671875%2C6.749850810550778&amp;layer=mapnik"
  title="Map of Lagos area, Nigeria"
  style="border: 1px solid black"
>
</iframe>
<br />
<small>
  <a href="https://www.openstreetmap.org/#map=11/6.4501/3.3210">
    View Larger Map
  </a>
</small>
```

:::

如果你想在 `iframe` 元素中嵌入直接的 HTML，必須使用 `srcdoc` 屬性，而不是 `src`。
