## 문자열 기본

- **정의**: 문자열은 작은따옴표, 큰따옴표 또는 백틱으로 감싼 문자 수열입니다. 문자열은 원시 데이터 유형이며 변경할 수 없습니다. 변경 불가능하다는 것은 문자열이 생성된 후에는 변경할 수 없다는 뜻입니다. 
- **문자열에서 문자 접근하기**: 문자열에서 문자를 접근하려면 대괄호 표기법을 사용하고 인덱스 번호를 전달하세요. 인덱스는 문자열 내 문자의 위치이며 0부터 시작합니다.

:::interactive_editor

```js
const developer = "Jessica";
console.log(developer[0]); // J
```

:::

- **`\n` (줄바꿈 문자)**: 문자열에서 줄바꿈을 만들려면 `\n` 줄바꿈 문자를 사용하세요.

:::interactive_editor

```js
const poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

- **문자열 이스케이프**: 문자열 내에서 문자를 이스케이프하려면 따옴표 앞에 백슬래시(`\`)를 넣으세요.

:::interactive_editor

```js
const statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

## 템플릿 리터럴과 문자열 보간

- **정의**: 템플릿 리터럴은 백틱(`)으로 정의합니다. 변수 등을 문자열 안에 직접 임베드할 수 있어 문자열 조작이 더 쉽습니다. 이를 문자열 보간이라고 합니다.

:::interactive_editor

```js
const name = "Jessica";
const greeting = `Hello, ${name}!`; 
console.log(greeting); // "Hello, Jessica!"
```

:::

## ASCII, `charCodeAt()` 메서드와 `fromCharCode()` 메서드

- **ASCII**: ASCII(미국 표준 정보 교환 코드)는 기본 영어 문자를 숫자 값으로 나타내는 문자 인코딩 표준입니다. 이전 수업에서는 ASCII 예제를 사용해 `charCodeAt()`와 `fromCharCode()`를 소개했습니다.
- **유니코드**: JavaScript 문자열은 내부적으로 유니코드, 특히 UTF-16 인코딩을 사용합니다. 처음 128자(기본 라틴 문자, 숫자, 일반 기호)는 유니코드 값이 ASCII 코드와 일치합니다. 그래서 ASCII 기반 예제가 JavaScript에서 계속 작동합니다.
- **`charCodeAt()` 메서드**: 이 메서드는 지정한 인덱스의 문자에 해당하는 UTF-16 코드 단위를 반환합니다. 기본 라틴 문자에 대해서는 이 값이 ASCII 코드와 일치합니다.

:::interactive_editor

```js
const letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

:::

- **`fromCharCode()` 메서드**: 이 메서드는 ASCII 코드를 해당 문자로 변환합니다.

:::interactive_editor

```js
const char = String.fromCharCode(65);
console.log(char);  // A
```

:::

## 기타 자주 쓰이는 문자열 메서드

- **`indexOf` 메서드**: 이 메서드는 문자열 내에서 부분 문자열을 검색하는 데 사용합니다. 부분 문자열이 발견되면 `indexOf`는 그 부분 문자열이 처음 나타나는 인덱스(위치)를 반환합니다. 찾지 못하면 `indexOf`는 -1을 반환하여 검색 실패를 나타냅니다.

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.indexOf("fox")); // 16
console.log(text.indexOf("cat")); // -1
```
  
:::  

- **`includes()` 메서드**: 이 메서드는 문자열에 특정 부분 문자열이 포함되어 있는지 확인합니다. 포함되어 있으면 true를 반환하고, 그렇지 않으면 false를 반환합니다.

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.includes("fox")); // true
console.log(text.includes("cat")); // false
```

:::

- **`slice()` 메서드**: 이 메서드는 문자열의 일부를 추출해 새 문자열로 반환하며 원본 문자열은 변경하지 않습니다. 시작 인덱스와 선택적 종료 인덱스 두 매개변수를 받습니다.

:::interactive_editor

```js
const text = "freeCodeCamp";
console.log(text.slice(0, 4));  // "free"
console.log(text.slice(4, 8));  // "Code"
console.log(text.slice(8, 12)); // "Camp"
```

:::

- **`toUpperCase()` 메서드**: 이 메서드는 모든 문자를 대문자로 변환해 새 문자열로 반환합니다.

:::interactive_editor

```js
const text = "Hello, world!";
console.log(text.toUpperCase()); // "HELLO, WORLD!"
```

:::

- **`toLowerCase()` 메서드**: 이 메서드는 문자열 내 모든 문자를 소문자로 변환합니다.

:::interactive_editor

```js
const text = "HELLO, WORLD!"
console.log(text.toLowerCase()); // "hello, world!"
```

:::

- **`replace()` 메서드**: 이 메서드는 문자열에서 지정한 값을 찾아 다른 값으로 교체합니다. 이 메서드는 교체된 새 문자열을 반환하며, JavaScript 문자열이 변경 불가능하기 때문에 원본은 그대로 유지됩니다.

:::interactive_editor

```js
const text = "I like cats";
console.log(text.replace("cats", "dogs")); // "I like dogs"
```

:::

- **`replaceAll()` 메서드**: 이 메서드는 문자열 내 지정한 값(단어, 문자, 패턴)의 모든 발생을 찾아 다른 값으로 교체합니다. `replace()`와 비슷하지만 첫 번째 일치 후 멈추지 않고 문자열 내 모든 일치를 업데이트합니다.

:::interactive_editor

```js
const text = "I love cats and cats are so much fun!";
console.log(text.replaceAll("cats", "dogs")); // "I love dogs and dogs are so much fun!"
```

:::

- **`repeat()` 메서드**: 이 메서드는 문자열을 지정한 횟수만큼 반복하는 데 사용합니다.

:::interactive_editor

```js
const text = "Hello";
console.log(text.repeat(3)); // "HelloHelloHello"
```

:::

- **`trim()` 메서드**: 이 메서드는 문자열 양쪽 끝에 있는 공백을 제거합니다.

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trim()); // "Hello, world!"
```

:::

- **`trimStart()` 메서드**: 이 메서드는 문자열 시작 부분에 있는 공백을 제거합니다.

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trimStart()); // "Hello, world!  "
```

:::

- **`trimEnd()` 메서드**: 이 메서드는 문자열 끝 부분에 있는 공백을 제거합니다.

:::interactive_editor

```js
const text = " Hello, world! ";
console.log(text.trimEnd()); // "  Hello, world!"
```

:::

- **`prompt()` 메서드**: `window`의 이 메서드는 대화 상자 형태로 사용자에게 정보를 입력받는 데 사용합니다. 이 메서드는 두 개의 인수를 받습니다. 첫 번째 인수는 대화 상자 안에 표시될 메시지로, 보통 사용자에게 정보를 입력하라고 요청합니다. 두 번째 인수는 선택 사항인 기본값으로, 입력 필드를 처음에 채웁니다.

```js
const answer = window.prompt("What's your favorite animal?"); // This will change depending on what the user answers
```
