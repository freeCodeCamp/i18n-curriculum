**目標：** 建置一個功能上類似於此的應用程式：<a href="https://codepen.io/freeCodeCamp/full/Myvqmo/" target="_blank" rel="noopener noreferrer nofollow">https://codepen.io/freeCodeCamp/full/Myvqmo/</a>。

Twitch API 是一個 RESTful API，讓開發者為更廣泛的 Twitch 社群建置創意整合。

完成以下使用者故事並通過所有測試。使用你需要的任何函式庫或 API。展現你自己的風格。

**使用者故事：** 我可以看到 freeCodeCamp 是否正在 Twitch.tv 上進行串流。

**使用者故事：** 我可以點擊狀態輸出，並直接被導向 freeCodeCamp 的 Twitch.tv 頻道。

**使用者故事：** 如果 Twitch 使用者目前正在串流，我可以看到他們串流內容的更多詳細資訊。

**提示：** 這是一個經常直播的 Twitch.tv 使用者名稱的陣列：`["ESL_SC2", "OgamingSC2", "cretetion", "freecodecamp", "storbeck", "habathcx", "RobotCaleb", "noobs2ninjas"]`

**更新：** 由於 API 使用條件的變更，Twitch.tv 現在需要 API 金鑰，但我們已經建立了解決方法。請使用 <a href="https://twitch-proxy.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://twitch-proxy.freecodecamp.rocks/</a> 取代 Twitch 的 API 基底 URL，這樣你仍然可以取得帳號資訊，而不需要註冊 API 金鑰。
