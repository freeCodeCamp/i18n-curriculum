# --description--

Agora que seu regex corresponde à letra `p` seguida por um ou mais dígitos, a última coisa que você precisa verificar é se nenhum caractere extra foi encontrado na string.

Para fazer isso você pode usar outra função do módulo `re`.A função `fullmatch` retorna um objeto de correspondência quando o padrão regex corresponde à string inteira e `None` caso contrário.

```py
import re

book = "Fahrenheit 451"
print(re.fullmatch('\d+', book)) #None

print(re.fullmatch('Fahrenheit \d+', book))
# <re.Match object; span=(0, 14), match='Fahrenheit 451'>
```

Substitua a chamada `search` por uma chamada `fullmatch` mantendo os mesmos argumentos.
