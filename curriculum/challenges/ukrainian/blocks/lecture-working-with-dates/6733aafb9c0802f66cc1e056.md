# --interactive--

Робота з датами та часом у JavaScript не завжди була простою. У професійному коді ви, ймовірно, використовуватимете бібліотеку, яка вирішила багато таких проблем за вас.

Але будуть випадки, коли вам доведеться працювати з вбудованим об’єктом `Date` у JavaScript.

У цьому уроці ми розглянемо основи об’єкта `Date` та деякі з найпоширеніших методів, які ви використовуватимете при роботі з датами та часом у JavaScript.

Об’єкт `Date` — це вбудований об’єкт у JavaScript, який дозволяє працювати з датами та часом. Ось базовий синтаксис для створення нового об’єкта `Date`:

```js
const now = new Date();
```

Ключове слово `new` використовується для створення нового екземпляра об’єкта `Date`, а об’єкт `Date` потім призначається змінній `now`. Якщо ви виведете значення `now` у консоль, ви побачите поточну дату та час на основі системного годинника комп’ютера, який виконує код.

Ось приклад відповіді, яку ви побачите, якщо виведете значення `now` у консоль:

:::interactive_editor

```js
const now = new Date();
console.log(now);
```

:::

Для часу використовується 24-годинний формат, тому `14:30` — це `2:30 PM`. `GMT-0700` — це зсув часової зони, а `Pacific Daylight Time` — назва часової зони.

Ви також можете передати конкретну дату та час у об’єкт `Date`, вказавши рік, місяць, день, годину, хвилину, секунду та мілісекунду як аргументи.

Ось приклад створення нового об’єкта `Date` з конкретною датою та часом:

:::interactive_editor

```js
const specificDate = new Date("July 4, 1776 12:00:00");

console.log(specificDate);
```

:::

Це корисно, коли потрібно працювати з конкретною датою та часом, а не з поточними. Вхідні дані завжди мають бути рядком, а формат повинен бути розпізнаваним об’єктом `Date`.

Щоб отримати поточну дату та час, можна використати метод `Date.now()`, який повертає кількість мілісекунд, що минули з `January 1, 1970, 00:00:00 UTC`. Це відоме як час Unix epoch.

Час Unix epoch — це загальний спосіб представлення дат і часу в комп’ютерних системах, оскільки це ціле число, яке легко зберігати та обробляти. UTC означає Universal Time Coordinated, що є основним часовим стандартом, за яким світ регулює годинники та час.

Якщо потрібно отримати день місяця на основі поточної дати, можна використати метод `getDate`. Ось приклад використання методу `getDate`:

:::interactive_editor

```js
const now = new Date();
const date = now.getDate();
console.log(date); 
```

:::

У цьому прикладі ми створюємо новий екземпляр дати за допомогою об’єкта `Date` і призначаємо його змінній `now`. Потім викликаємо метод `getDate` на об’єкті `now`, щоб отримати день місяця, і призначаємо його змінній `date`. Нарешті, виводимо значення `date` у консоль, що виведе поточний день місяця.

`getDate` поверне ціле число від `1` до `31`, залежно від дня місяця. Якщо дата недійсна, він поверне `NaN` (Not a Number).

Щоб отримати місяць, можна використати метод `getMonth` так:

:::interactive_editor

```js
const now = new Date();
const month = now.getMonth();
console.log(month);
```

:::

Місяць починається з нуля, тому `January` — це `0`, `February` — це `1` і так далі. Якщо місяць недійсний, він поверне `NaN`.

Якщо потрібно отримати повний рік, можна використати метод `getFullYear` так:

:::interactive_editor

```js
const now = new Date();
const year = now.getFullYear();
console.log(year); 
```

:::

Якщо рік недійсний, він поверне `NaN`.

Існує багато інших методів у об’єкті `Date`, включно з `getHours`, `getMinutes`, `getSeconds` та іншими. Раджу вам самостійно дослідити їх через документацію Mozilla або інші ресурси.
