# --interactive--

تُستخدم جملة `break` للخروج المبكر من الحلقة، بينما تُستخدم جملة `continue` لتخطي التكرار الحالي للحلقة والانتقال إلى التكرار التالي.

إليك مثالًا على استخدام جملة `break` في حلقة `for`:

:::interactive_editor

```js
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}
```

:::

في المثال أعلاه، تبدأ الحلقة العد من `0` وطالما أن `i` أقل من `10`، تستمر الحلقة في التنفيذ.

داخل الحلقة، نتحقق مما إذا كانت قيمة `i` تساوي `5`. إذا كانت كذلك، نستخدم جملة `break` للخروج المبكر من الحلقة. وإذا لم تكن، نطبع قيمة `i` في وحدة التحكم. لذلك، ستطبع الشفرة الأرقام `0`، `1`، `2`، `3`، و`4`.

تُفيد جملة `break` عندما تريد الخروج المبكر من الحلقة بناءً على شرط معين. على سبيل المثال، إذا كنت تبحث عن قيمة محددة في مصفوفة، يمكنك استخدام جملة `break` للخروج من الحلقة بمجرد العثور على القيمة.

أحيانًا قد ترغب في تخطي تكرار معين من الحلقة دون الخروج منها تمامًا. هنا تأتي فائدة جملة `continue`. إليك مثالًا على استخدام جملة `continue` في حلقة `for`:

:::interactive_editor

```js
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}
```

:::

تمامًا كما في السابق، قمنا بتهيئة `i` إلى `0` ولدينا شرط يجعل الحلقة تعمل طالما أن `i` أقل من `10`.

داخل الحلقة، عندما تكون قيمة `i` تساوي `5`، نستخدم جملة `continue` لتخطي التكرار الحالي والانتقال إلى التكرار التالي.

سيطبع ناتج هذه الشفرة الأرقام `0`، `1`، `2`، `3`، `4`، `6`، `7`، `8`، و`9`. الرقم `5` تم تخطيه بسبب جملة `continue`.

شيء آخر يمكنك فعله مع جملتي `break` و `continue` هو استخدام المسميات لتحديد الحلقة التي تريد كسرها أو تخطيها.

هذا مفيد عندما يكون لديك حلقات متداخلة وتريد التحكم في تدفق الحلقة الخارجية من داخل الحلقة الداخلية.

إليك مثالًا على استخدام المسميات مع جملة `break`:

:::interactive_editor

```js
outerLoop: for (let i = 0; i < 3; i++) {
  innerLoop: for (let j = 0; j < 3; j++) {
    if (i === 1 && j === 1) {
      break outerLoop;
    }
    console.log(`i: ${i}, j: ${j}`);
  }
}
```

:::

في هذا المثال، لدينا حلقة `for` خارجية مسماة `outerLoop` وحلقة `for` داخلية مسماة `innerLoop`.

عندما تكون قيمة `i` تساوي `1` و`j` تساوي `1`، نستخدم جملة `break` مع المسمى `outerLoop` للخروج المبكر من الحلقة الخارجية. هذا سيخرج من كل من الحلقات الداخلية والخارجية.

سيطبع ناتج هذه الشفرة التالي في وحدة التحكم:

```md
"i: 0, j: 0"
"i: 0, j: 1"
"i: 0, j: 2"
"i: 1, j: 0"
```

في معظم الأحيان، لن تحتاج إلى استخدام المسميات مع جملتي `break` و `continue`، لكن من الجيد أن تعرف أن لديك هذا الخيار إذا احتجت إليه يومًا.
