# --interactive--

**HINWEIS**: Einige der interaktiven Beispiele verwenden möglicherweise CSS, das Sie noch nicht gelernt haben. Machen Sie sich keine Sorgen, wenn Sie nicht den gesamten Code verstehen. Das Ziel der Beispiele ist es, Ihnen Vorschauen zu diesen Designkonzepten zu zeigen, damit Sie besser verstehen, wie alles funktioniert. Um die Vorschauen zu sehen, müssen Sie den interaktiven Editor aktivieren.

Im Webdesign gibt es viele Arten von Navigationshilfen, die Sie verwenden können. Beispiele sind obere Navigationsleisten, Seitenleisten und Fußzeilen. Wenn Ihre Website jedoch komplexer ist und tiefere Navigationsebenen hat, sollten Sie die Verwendung von Breadcrumbs in Betracht ziehen.

Breadcrumbs sind eine Navigationshilfe, die dem Benutzer zeigt, wo er sich in der Hierarchie der Website befindet. Hier ist ein Beispiel, wie Breadcrumbs für eine fiktive Elektronik-Website aussehen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home / Electronics / Phones / Smartphone XYZ
</div>
```

```css
.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

Auf den meisten Websites werden Breadcrumbs oben auf der Seite angezeigt und zeigen dem Benutzer den Pfad, den er genommen hat, um zur aktuellen Seite zu gelangen. Ausgehend von `Homepage` navigierte der Benutzer zur `Electronics`-Kategorie, dann zur `Phones`-Kategorie und schließlich zum `Smartphone XYZ`-Produkt. Wahrscheinlich sind Sie schon einmal mit Breadcrumbs auf einer Website in Berührung gekommen, während Sie nach einem Produkt oder einer bestimmten Information gesucht haben.

Die Verwendung von Breadcrumbs ist hilfreich, weil sie den Nutzern helfen kann zu verstehen, wo sie sich in der Hierarchie der Website befinden und wie sie zu den vorherigen Seiten zurückkehren können. Das ist besonders nützlich, wenn ein Benutzer von einem Suchergebnis oder einem externen Link kommt und den Kontext der Seite, auf der er sich befindet, verstehen muss.

Beim Design von Breadcrumbs gibt es einige Überlegungen, die Sie beachten sollten. Die erste ist die Entscheidung, welches Trennzeichen verwendet wird. Das Trennzeichen ist das Zeichen, das die verschiedenen Ebenen der Hierarchie voneinander trennt. Übliche Trennzeichen sind das Größer-als-Zeichen (`>`), spitze Anführungszeichen (`»`) und der Schrägstrich (`/`).

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
```

```css
.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

Die zweite Überlegung ist die Platzierung der Breadcrumbs. Breadcrumbs werden typischerweise oben auf der Seite platziert, entweder über oder unter der Hauptnavigationsleiste. Benutzer sollten nicht danach suchen müssen, also stellen Sie sicher, dass sie sichtbar und leicht zu finden sind.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<nav class="main-nav">
  <a href="#">Home</a>
  <a href="#">Electronics</a>
  <a href="#">Phones</a>
</nav>
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
```

```css
.main-nav {
  background-color: #333;
  padding: 1rem;
}

.main-nav a {
  color: white;
  margin-right: 1rem;
  text-decoration: none;
}

.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

Eine weitere Überlegung ist die Größe der Breadcrumbs. Sie sollten groß genug sein, um leicht gelesen zu werden, aber nicht so groß, dass sie zu viel Platz auf der Seite einnehmen. Denken Sie daran, dass Breadcrumbs nicht als primäres Navigationswerkzeug dienen, sondern als sekundäre Navigationshilfe. Auf Websites mit vielen Informationen auf einer Seite können Benutzer mithilfe von Breadcrumbs leicht sehen, wo sie sich in der Hierarchie befinden, und zu vorherigen Seiten zurückkehren.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
<p>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
</p>
```

```css
.breadcrumbs {
  font-size: 1rem;
  color: #555;
  margin: 1rem 0;
}   

p {
  font-size: 1.2rem;
  line-height: 1.6;
}
```

:::

Wann sollten Sie also Breadcrumbs auf Ihrer Website verwenden? Wenn Ihre Website einfach ist und eine flache Hierarchie hat, sind Breadcrumbs möglicherweise nicht notwendig. Eine flache Hierarchie bedeutet, dass die Website nur wenige Navigationsebenen hat, sodass Breadcrumbs nicht viel Mehrwert bieten. Typischerweise finden Sie Breadcrumbs auf E-Commerce-Websites, Nachrichtenseiten und anderen Websites mit einer tiefen Hierarchie wie technischen Dokumentationsseiten.
