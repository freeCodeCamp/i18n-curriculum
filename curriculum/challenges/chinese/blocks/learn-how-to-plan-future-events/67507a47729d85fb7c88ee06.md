# --explanation--

`Is planned` 使用 `Present Simple` 时态来描述已经调度的未来事件。它表示对培训何时发生的确定性，而不是表示培训已经发生。例如：

`The meeting is planned for next Monday。` - 这表明会议已被调度在未来。
