# --description--

تُستخدم عبارة `Anyone who` للإشارة إلى أي شخص يستوفي شرطًا أو متطلبًا معينًا.

في مجال التكنولوجيا، تُستخدم غالبًا لوصف الأشخاص الذين يمتلكون مهارات أو معرفة معينة. على سبيل المثال، `Anyone who understands JavaScript can modify this script.`
