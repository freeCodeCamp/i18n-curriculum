# --description--

$n$-튜플 정수 $t = (a_1, \ldots, a_n)$에 대해, 다항방정식 $x^n + a_1x^{n - 1} + a_2x^{n - 2} + \ldots + a_{n - 1}x + a_n = 0$의 해를 $(x_1, \ldots, x_n)$이라 하겠습니다.

다음 두 조건을 고려하세요:

- $x_1, \ldots, x_n$가 모두 실수입니다.
- $x_1, ..., x_n$를 정렬했을 때, $⌊x_i⌋ = i$가 $1 ≤ i ≤ n$에 대해 성립합니다. ($⌊·⌋:$ 내림 함수)

$n = 4$인 경우, 두 조건을 모두 만족하는 $n$-튜플 정수는 12개입니다.

$t$에 대해 $S(t)$를 $t$에 포함된 정수들의 절댓값 합으로 정의합니다.

$n = 4$일 때, 두 조건을 만족하는 모든 $n$-튜플 $t$에 대해 $\sum S(t) = 2087$임을 확인할 수 있습니다.

$n = 7$일 때 $\sum S(t)$를 구하세요.
