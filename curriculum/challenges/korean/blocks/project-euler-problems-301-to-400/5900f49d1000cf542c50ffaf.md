# --description--

모든 양의 정수 $n$에 대해 함수 $\text{next_prime}(n)$은 $p > n$인 가장 작은 소수 $p$를 반환합니다.

수열 $a(n)$은 다음과 같이 정의됩니다: $a(1) = \text{next_prime}({10}^{14})$이고 $n > 1$일 때 $a(n) = \text{next_prime}(a(n - 1))$입니다.

피보나치 수열 $f(n)$은 $f(0) = 0$, $f(1) = 1$이며 $n > 1$일 때 $f(n) = f(n - 1) + f(n - 2)$로 정의됩니다.

수열 $b(n)$은 $f(a(n))$로 정의됩니다.

$1 \leq n \leq 100\,000$인 모든 $n$에 대해 $\sum b(n)$을 구하세요. 답은 $\bmod 1\,234\,567\,891\,011$로 제출하세요.
