# --description--

다음 방정식에서 $x$, $y$, 그리고 $n$은 양의 정수입니다.

$$\frac{1}{x} + \frac{1}{y} = \frac{1}{n}$$

한계 $L$에 대해 $F(L)$을 $x < y ≤ L$을 만족하는 해의 개수로 정의합니다.

$F(15) = 4$이고 $F(1000) = 1069$임을 확인할 수 있습니다.

$F({10}^{12})$를 구하세요.
