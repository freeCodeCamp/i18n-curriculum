# --interactive--

當使用者點擊按鈕或表單發生變更時，這稱為事件。在你的程式中，你需要有一種方式來監聽這些事件並對其做出回應。

`addEventListener()` 方法用於監聽事件。它接受兩個引數：你想監聽的事件以及事件發生時會被呼叫的函式。

這是基本語法：

```js
element.addEventListener("event", listener);
```

該元素是要監聽 `事件` 的 HTML 元素，而 `event` 指定要監聽的 `事件` 型別，例如 `"click"`。

監聽器是一個物件，當事件發生時會接收通知。大多數情況下，這會是你定義用來控制代碼該事件的函式。

這裡有一個範例：

```js
element.addEventListener("click", () => {
  // code to run when the click event occurs
});
```

你也可以選擇傳入像這樣的函式引用：

```js
function handleClick() {
  // code to run when the click event occurs
}

element.addEventListener("click", handleClick);
```

有時你可能想要創建一個獨立的函式（程式）來控制該事件。這可以讓你的程式碼更容易閱讀和維護。

監聽器引數也可以是 `null`，或者可以是使用 `EventListener` 介面的物件。`EventListener` 介面定義了一個名為 `handleEvent()` 的單一方法。每當你監聽的事件發生時，這個方法會自動被呼叫。使用這個介面可以讓同一個物件在需要時控制多個事件。

大多數時間，你不需要使用這個介面，而是可以傳入一個在事件發生時會被呼叫的函式。

現在，讓我們來看一個範例，以更好地理解這是如何運作的：

在此範例中，我們有一個 id 為 `btn` 的 `button` 元素：

```html
<button id="btn">Show alert</button>
```

我們想監聽這個按鈕上的 `"click"` 事件，並在按鈕被點擊時顯示警示。

我們首先需要在我們的 JavaScript 程式碼中存取那個 `button` 元素。然後我們需要為它添加一個事件監聽器來傾聽點擊事件，並在按鈕被點擊時顯示警示：

:::interactive_editor

```html
<button id="btn">Show alert</button>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("btn");

btn.addEventListener("click", () => alert("You clicked the button"));
```

:::

現在每次按鈕被點擊時，使用者將會在螢幕上看到像這樣顯示的警示訊息：

```md
You clicked the button
```

你可以監聽的另一種事件是 `"input"` 事件。當輸入元素的值改變時，會觸發此事件。

這裡有一個範例：

```html
<input type="text" id="input" placeholder="Type something" />
```

我們想要監聽此輸入元素上的 input 事件，並在使用者每次輸入時將輸入的值記錄到主控台。

我們首先需要在我們的 JavaScript 程式碼中存取那個 `input` 元素。然後我們需要為 `input` 事件添加一個事件監聽器，並在每次使用者輸入時將輸入的值記錄到主控台：

:::interactive_editor

```html
<input type="text" id="input" placeholder="Type something" />
<script src="index.js"></script>
```

```js
const input = document.getElementById("input");

input.addEventListener("input", () => {
  console.log(input.value);
});
```

:::

你可以使用 `addEventListener()` 方法監聽更多的事件。一些常見的事件包括 `mouseover`、`mouseout`、`keydown`、`keyup` 和 `submit`。
