# --interactive--

在 CSS 中，定位允许我们控制元素在页面上的布局。 两种常见的定位方式是静态定位和相对定位。 默认情况下，元素是静态定位的。 这意味着它们遵循文件的正常流程，从上到下，从左到右，一个接一个。 

静态定位是所有元素的默认设置，不需要在 CSS 中做任何特殊声明。 使用静态定位时，你不会注意到任何不同之处，因为它只是将元素保持在文档中自然出现的位置。 下面是一个使用默认静态定位的段落示例：

:::interactive_editor

```html
<p>This paragraph is statically positioned.</p>
```

:::

在浏览器中呈现时，该段将按照正常的文档流程，以自然的位置出现。 移动它时不会应用特殊的样式，它会保持在布局中自然出现的位置。

而相对定位允许元素从其正常位置移动，而不会破坏文档的流程。 可将其视为通过赋予元素新坐标，从而将元素从默认静态位置移动。 以下是应用相对定位的方法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="relative">This paragraph is positioned relatively.</p>
```

```css
body {
  border: solid 1px black;
}

.relative {
  position: relative;
  top: 30px;
  left: 30px;
}
```

:::

在这个示例中，该段落将从其原始位置向下偏移 `30px`，并向右偏移 `30px`。 不过，它在正常流动中所占的空间仍然保留。 当你想在不影响布局其他部分的情况下轻稍微动一个元素时，相对定位非常有用。
