# --explanation--

Vamos a desglosarlo:

¿Qué dijo Mark? Él dijo que el BIOS `prevents the computer from starting.` Eso significa que está hablando de lo que el BIOS **puede hacer** cuando está corrupto. 

Si Sarah está de acuerdo, ¿hablaría sobre qué es el BIOS o qué puede hacer el BIOS? La segunda, ¿verdad? Por eso `Yeah, it can` es la opción correcta.

¿Por qué `yeah, it is` está mal? Porque `it is` habla sobre el estado del BIOS, no sobre lo que puede hacer. Aquí tienes un ejemplo:

**Correcto**: `Yeah, it can.` – Esto coincide con el punto de Mark sobre lo que el BIOS puede hacer.

**Incorrecto**: `Yeah, it is.` – Esto no encaja porque Mark no está describiendo qué es el BIOS, está hablando de sus efectos.
