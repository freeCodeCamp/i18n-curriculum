# --description--

`Every` と `any` は文中でよく使われますが、意味が異なります。

`Every` はグループやカテゴリー内のすべての項目について話すときに使います。例えば、`Every student must complete the assignment` はすべての学生を意味します。

一方で、`any` は疑問文や否定文で、1つ、いくつか、またはすべてを意味するときに使います。例えば、`Can any student answer the question?` は答えられる学生が少なくとも1人いるかどうかを尋ねています。
