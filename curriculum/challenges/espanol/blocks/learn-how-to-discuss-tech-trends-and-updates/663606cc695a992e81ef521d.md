# --description--

La expresión `That/it sounds` se usa para describir cómo algo parece o se presenta, a menudo basado en la información escuchada o en la impresión que da. Por ejemplo, `That sounds like a great idea!` Esto indica que la idea parece buena según lo que se escuchó.

De manera similar, `That/it looks` se puede usar para describir cómo algo se presenta basándose en la apariencia, como en `It looks beautiful.`

La palabra `convenient` es un adjetivo que describe algo que es adecuado para las necesidades de uno, fácil de usar, o que ayuda a hacer las cosas más fácilmente y rápidamente. A menudo se refiere a algo que hace que una tarea sea menos difícil o ahorre tiempo.

Por ejemplo, `Online shopping is convenient because you can do it from home.`
