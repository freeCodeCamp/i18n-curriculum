# --description--

在以下方程式中，x、y 和 n 是正整數。

$$\frac{1}{x} + \frac{1}{y} = \frac{1}{n}$$

可以驗證當 `n` = 1260 時，有 113 個不同的解法，且這是 `n` 的最小值，使得不同解法的總數超過一百。

`n` 的最小值為何，使得不同解法的數量超過四百萬？

**注意：** 這個問題是問題 108 的更困難版本，由於它遠超過暴力破解方法的限制，因此需要巧妙的實作。
