# --description--

واحدة من أكثر الطرق شيوعًا لإضافة عنصر جديد إلى مصفوفة هي استخدام الطريقة `push()`. تضيف الطريقة `push()` عنصرًا أو أكثر إلى نهاية المصفوفة وتُرجع الطول الجديد للمصفوفة.

على سبيل المثال، لإضافة عنصر جديد إلى مصفوفة `pet`، يمكنك استخدام الشفرة التالية:

```javascript
const pet = ['cat', 'dog', 'bunny'];
pet.push('parrot');
console.log(pet); // Output: ['cat', 'dog', 'bunny', 'parrot']
```

لإزالة العنصر الأخير من مصفوفة، يمكنك استخدام الطريقة `pop()`. تزيل الطريقة `pop()` العنصر الأخير من المصفوفة وتُرجع ذلك العنصر.

على سبيل المثال، لإزالة العنصر الأخير من مصفوفة `pet`، يمكنك استخدام الشفرة التالية:

```javascript
const pet = ['cat', 'dog', 'tiger'];
pet.pop();
console.log(pet); // Output: ['cat', 'dog']
```
