# --interactive--

在构建多语言网站或处理自定义数据属性时，你经常需要根据元素所包含的语言或特定数据值来设计元素样式。

`lang` 和 `data-lang` 属性常用于这些目的，CSS 中的属性选择器可让你有效地应用基于这些属性的样式。

HTML 中的 `lang` 属性用于指定元素内容的语言。 特别是在多语言网站上，你可能需要根据元素所使用的语言为其设计不同的样式。

下面的示例说明了如何使用属性选择器来定位具有特定 `lang` 属性的元素：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">This is an English paragraph.</p>
<p lang="fr">Ceci est un paragraphe en français.</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

此 CSS 规则将斜体样式应用于 `lang` 属性设置为英语（`en`）的任何段落元素。 这对于在包含多种语言的文件中强调英文文本非常有用。

同样，你也可以使用属性选择器来定位带有 `data-lang` 属性的元素。 

自定义数据属性如 `data-lang` 通常用于在元素中存储附加信息，例如指定特定文本部分所使用的语言。 

下面是如何使用 CSS 根据 `data-lang` 属性来定位和样式化元素：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p data-lang="fr">Ceci est un paragraphe en français.</p>
<p data-lang="en">This is a paragraph in English.</p>
```

```css
p[data-lang="fr"] {
  color: blue;
}
```

:::

在这种分支中，任何带有 `data-lang` 属性且属性值设置为法语（`fr`）的 `p` 元素，其文本颜色将被更改为蓝色。这允许你根据存储在 `data-lang` 属性中的语言信息快速识别和样式化内容部分。

类似这样的属性选择器提供了一种功能强大的方法，可根据 HTML 中嵌入的元数据应用条件样式，使网页更具动态性和上下文感知能力。
