# --explanation--

`Compiled`는 동사 `compile`의 과거 분사형입니다. `To compile`는 여러 출처에서 정보나 항목을 모아 하나의 모음으로 만드는 것을 의미합니다. 예를 들어:

- `She compiled all the feedback from the customers into a report.` - 그녀는 다양한 출처에서 피드백을 모아 하나의 보고서로 정리했습니다.

- `They compiled data from several studies to write their research paper.` - 그들은 특정 목적을 위해 여러 연구에서 데이터를 수집했습니다.
