# --description--

Para dar colores diferentes a los marcadores, necesitará añadir una clase única a cada uno. Puede añadir varias clases a un elemento listándolas en el atributo `class` y separándolas con un espacio. Por ejemplo, el siguiente código añade tanto la clase `animal` como la clase `dog` a un elemento `div`:

```html
<div class="animal dog">
```

Si añade varias clases a un elemento HTML, los estilos de las primeras clases que liste en la hoja de estilos pueden ser sobrescritos por las clases posteriores.

Para empezar, añada la clase `one` al primer elemento `div` del marcador.
