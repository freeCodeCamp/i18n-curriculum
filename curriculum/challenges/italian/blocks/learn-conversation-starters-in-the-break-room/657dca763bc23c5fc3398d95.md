# --description--

`Probably` si usa quando pensi che qualcosa sia probabile, ma non sei sicuro al 100%.

Per esempio, potresti dire: `The code will probably run without errors after the update`, perché ci credi, ma c’è ancora una piccola possibilità di un problema.
