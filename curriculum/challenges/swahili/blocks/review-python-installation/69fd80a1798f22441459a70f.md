## Python katika Mazingira Yako ya Ndani

- **Usakinishaji**: Njia bora ya kusakinisha Python kwenye Windows, Mac, na Linux ni kupakua msakinishaji kutoka kwenye tovuti rasmi ya Python (`https://www.python.org/`).
- **Terminal**: Kiolesura cha maandishi kinachokuwezesha kuingiliana na kompyuta yako kwa kuandika amri. Kila mfumo wa uendeshaji huja na programu ya terminal ya msingi. Kwenye macOS, unaweza kutumia programu ya Terminal. Kwenye Windows, unaweza kutumia Command Prompt au PowerShell. Kwenye Linux, kila mazingira ya desktop yana programu yake ya terminal ya msingi, kama GNOME Terminal au Konsole.
- **IDE**: IDE inasimama kwa Integrated Development Environment na ina vipengele kama mhariri wa msimbo, zana za majaribio, terminal na zaidi.
- **Mahariri wa msimbo na IDE maarufu kwa Python**: VS Code, PyCharm, na Spyder
- **Kukimbia msimbo ndani ya kompyuta yako**: Ili kukimbia msimbo wa Python, una chaguzi kadhaa. Chaguo la kwanza ni kubofya kitufe cha kuendesha msimbo katika VS Code ambacho kinaonekana kama kitufe cha kucheza kilicho upande wa juu kulia. Hilo litafungua terminal na kukimbia script yako ya Python hapo.

Chaguo jingine ni kufungua terminal na kukimbia programu kwa mikono. Hapa kuna mfano wa kukimbia programu inayoitwa `main.py`:

```bash
python main.py
```

Lazima ukimbie amri hii kutoka kwenye folda ambapo `main.py` imehifadhiwa. Kwa mfano, kama `main.py` iko ndani ya folda inayoitwa `python-projects`, kwanza tumia ifuatayo kwenye terminal:

```bash
cd python-projects
```

Kisha kimbia:

```bash
python main.py
```

## Kutumia Python Interactive Shell

- **Maelezo**: Interactive shell ni programu inayokuwezesha kuandika amri moja kwa moja na kuona matokeo. Fungua programu ya terminal na andika `python` kisha bonyeza `Enter`. Hii itaanzisha interactive shell ya Python. Alama ya `>>>` inaonyesha Python inakungoja uandike amri. Jaribu kuchapisha `"Hello, world!"` kwenye terminal:

```bash
print("Hello, world!")
```

Unapaswa kuona maandishi yakionekana kama haya:

```bash
>>> print("Hello, world!")
Hello, world!
```

Baada ya maandishi kuchapishwa, Python inarudi kwenye ifuatayo:

```bash
>>>
```

Hii ina maana unaweza kuandika amri nyingine.

- **Mzunguko wa Soma, Tathmini, Chapisha, Rudia (REPL)**: Kila unapoiandika amri, interpreter itasoma ingizo, litathmini, lichapishe matokeo na kurudia kuonyesha `>>>` ili uendelee kuandika amri zaidi.
