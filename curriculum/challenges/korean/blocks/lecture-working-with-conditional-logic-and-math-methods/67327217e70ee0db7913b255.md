# --interactive--

JavaScript를 배우다 보면 수학 연산을 수행하는 일이 매우 흔하다는 것을 금방 알게 됩니다. 기본 산술 연산자는 간단한 계산을 처리할 수 있지만, JavaScript는 더 복잡한 수학 문제를 해결할 수 있도록 내장 `Math` 객체를 제공합니다.

이 유용한 도구는 고급 계산을 수행하고 숫자를 조작하는 데 도움이 되는 다양한 메서드를 제공합니다. 이 메서드들을 살펴보고 코딩 경험을 어떻게 더 쉽게 만드는지 알아보겠습니다.

`Math.random()` 메서드는 `0` (포함)와 `1` (미포함) 사이의 임의의 부동 소수점 숫자를 생성합니다. 즉, 가능한 출력값은 `0`일 수 있지만 실제로 `1`에 도달하지는 않습니다. 다음은 `Math.random()` 메서드를 사용하는 예시입니다:

:::interactive_editor

```js
const randomNum = Math.random();

console.log(randomNum);
// any number between 0 and 1 – 0 inclusive and 1 exclusive
```

:::

`Math.min()`과 `Math.max()`은 모두 숫자 집합을 받아 각각 최솟값과 최댓값을 반환합니다. 다음은 두 메서드를 모두 사용하는 예시입니다:

:::interactive_editor

```js
const smallest = Math.min(1, 5, 3, 9);
console.log(smallest); // 1

const largest = Math.max(1, 5, 3, 9);
console.log(largest); // 9
```

:::

첫 번째 `console.log()`는 `1`을 출력할 것입니다. 왜냐하면 `1`이 그 숫자 목록에서 가장 작기 때문입니다. 두 번째 `console.log()`는 `9`을 출력할 것입니다. 왜냐하면 `9`가 그 숫자 목록에서 가장 크기 때문입니다.

숫자를 가장 가까운 정수로 올림하거나 내림하려면 `Math.ceil()`와 `Math.floor()` 메서드를 사용할 수 있습니다. 다음은 `Math.ceil()`을 사용하는 예시입니다:

:::interactive_editor

```js
console.log(Math.ceil(4.3)); // 5
```

:::

`Math.ceil()`는 `4.3`를 가장 가까운 정수인 `5`로 올림합니다. 이제 숫자를 내림하는 예를 살펴보겠습니다:

:::interactive_editor

```js
console.log(Math.floor(4.7)); // 4
```

:::

`Math.floor()`는 `4.7`를 가장 가까운 정수인 `4`로 내림합니다. `Math.round()`는 `Math.ceil()`와 `Math.floor()`의 혼합형입니다. 소수점 값을 고려하여 숫자를 가장 가까운 정수로 반올림합니다:

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

따라서 소수점이 `5`보다 작으면 숫자를 내림하고, 소수점이 `5` 이상이면 숫자를 올림합니다. `Math.floor()`와 `Math.random()`의 실용적인 활용법은 두 정수 사이의 임의의 숫자를 생성하는 것입니다. 구문은 다음과 같습니다:

:::interactive_editor

```js
const max = 10;
const min = 5;
const randomNum = Math.floor(Math.random() * (max - min + 1)) + min;
console.log(randomNum);
```

:::

`20`와 `1` 사이의 임의 숫자를 생성하는 코드는 다음과 같습니다:

:::interactive_editor

```js
const randomNumBtw1And20 = Math.floor(Math.random() * 20) + 1;
console.log(randomNumBtw1And20);
```

:::

또 다른 유용한 `Math` 메서드는 `Math.trunc()` 메서드입니다. `Math.trunc()`는 숫자의 소수 부분을 제거하고 반올림 없이 정수 부분만 반환합니다:

:::interactive_editor

```js
console.log(Math.trunc(2.9)); // 2
console.log(Math.trunc(9.1)); // 9
```

:::

숫자의 제곱근이나 세제곱근을 구해야 한다면 각각 `Math.sqrt()`와 `Math.cbrt()` 메서드를 사용할 수 있습니다:

:::interactive_editor

```js
console.log(Math.sqrt(81)); // 9
console.log(Math.cbrt(27)); // 3
```

:::

첫 번째 로그 문은 `9`을 출력할 것입니다. 왜냐하면 `81`의 제곱근이 `9`이기 때문입니다. 두 번째 로그 문은 `3`을 출력할 것입니다. 왜냐하면 `27`의 세제곱근이 `3`이기 때문입니다. 숫자의 절댓값을 구해야 한다면 `Math.abs()` 메서드를 사용할 수 있습니다:

:::interactive_editor

```js
console.log(Math.abs(-5)); // 5
console.log(Math.abs(5)); // 5
```

:::

`Math.abs()`는 숫자의 절댓값을 반환하여 음수를 양수로 바꿉니다. 마지막으로 살펴볼 메서드는 `Math.pow()` 메서드입니다:

:::interactive_editor

```js
console.log(Math.pow(2, 3)); // 8
console.log(Math.pow(8, 2)); // 64
```

:::

`Math.pow()`는 두 숫자를 받아 첫 번째 숫자를 두 번째 숫자의 거듭제곱으로 올립니다. `Math` 객체에는 이 외에도 탐색할 수 있는 많은 메서드가 있습니다. 하지만 이들은 JavaScript 코드베이스에서 자주 사용되는 몇 가지 메서드일 뿐입니다.
