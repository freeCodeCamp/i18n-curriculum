# --description--

La propiedad `transform` puede transformar la apariencia del elemento. Por ejemplo, darle un valor de `scale(0.9)` haría que el elemento sea un 10% más pequeño.

```css
p {
  transform: scale(0.9);
}
```

Agrega una propiedad `transform` al selector `.card:hover` y establecela en `scale(1.1)`.

# --hints--

El selector `.card:hover` debería tener `transform: scale(1.1)`.

```js
assert.strictEqual(new __helpers.CSSHelp(document).getStyle(".card:hover")?.getPropertyValue("transform"), "scale(1.1)");
```
