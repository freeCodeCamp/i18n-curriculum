# --explanation--

`Probably` — це прислівник, який використовується, щоб висловити, що щось, ймовірно, станеться, але не повністю впевнено. Наприклад:

`We'll probably finish the project by Friday.` — це показує впевненість у завершенні проєкту до п’ятниці, але залишає місце для невизначеності.
