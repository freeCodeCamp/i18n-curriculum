# --interactive--

Псевдокласи — це спеціальні ключові слова CSS, які дозволяють вибрати елемент залежно від його конкретного стану або позиції. Стан або позиція елемента можуть включати:

- Коли він активний.
- Коли на нього наводять курсор миші.
- Коли він є першим дочірнім елементом батьківського елемента.
- Коли він є останнім дочірнім елементом батьківського елемента.
- Коли посилання було відвідане.
- Коли він вимкнений.

Щоб використати псевдоклас, додайте його до селектора, використовуючи двокрапку (`:`) з наступним іменем псевдокласу:

```css
selector:pseudo-class {
  /* CSS properties */
}
```

Давайте подивимося, як можна використати псевдоклас, щоб представити кожен із згаданих станів і позицій.

Псевдоклас `:active` дозволяє вибрати активний стан елемента, наприклад, при натисканні на кнопку:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example Button</button>
```

```css
button:active {
  background: greenyellow;
}
```

:::

Псевдоклас `:hover` визначає стан наведення курсору на елемент. Прикладом буде наведення на кнопку або посилання:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="#">Hover over me!</a>  
```

```css
a:hover {
  text-decoration: none;
  color: white;
  background: crimson;
}
```

:::

Псевдоклас `:first-child` вибирає елемент, який є першим дочірнім елементом свого батьківського елемента. Ось приклад вибору першого абзацу в контейнері `div`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:first-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

Перший абзац у цьому `div` отримає фон `lightcoral` і відступи `padding` по `0.4rem` з усіх чотирьох сторін.

Псевдоклас `:last-child` вибирає останній дочірній елемент свого батьківського елемента. Ось приклад вибору останнього абзацу в контейнері `div`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:last-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

Псевдоклас `:visited` дозволяє стилізувати посилання, яке користувач уже відвідав:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="https://www.example.com" target="_blank">Visit Example.com</a>
```

```css
a:visited {
  color: purple;
}
```

:::

Псевдоклас `:disabled` дозволяє стилізувати інтерактивний елемент у вимкненому стані:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button disabled>Disabled Button</button>
```

```css
button:disabled {
  background-color: lightgray;  
}
```

:::

Як бачите, псевдокласи дозволяють стилізувати елементи залежно від взаємодії користувача, наприклад, наведення курсору або відвідування посилання. Це робить ваш сайт більш інтерактивним.

Окрім уже згаданих псевдокласів, існують інші, такі як:

- `:focus`
- `:first-of-type`
- `:last-of-type`
- `:nth-of-type`
- `:modal`
- `:enabled`
- `:checked`
- `:required` та інші.
