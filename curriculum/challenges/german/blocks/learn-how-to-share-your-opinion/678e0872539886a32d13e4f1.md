# --explanation--

`Lightweight` beschreibt etwas, das wenig Ressourcen oder Leistung benötigt, wodurch es einfach zu bedienen, schnell und effizient ist. Zum Beispiel:

`The app is lightweight, so it runs smoothly even on older smartphones.` – Das bedeutet, die App verwendet sehr wenig Speicher oder Rechenleistung, sodass sie gut auf Geräten mit geringerer Kapazität funktioniert.

`Features` sind spezifische Funktionen oder Werkzeuge, die ein Produkt oder eine Software den Nutzenden bietet. Zum Beispiel:

`The software has many useful features like file sharing and cloud storage.` – Das bedeutet, die Software enthält mehrere hilfreiche Funktionen, die sie für die Nutzenden wertvoller und effizienter machen.
