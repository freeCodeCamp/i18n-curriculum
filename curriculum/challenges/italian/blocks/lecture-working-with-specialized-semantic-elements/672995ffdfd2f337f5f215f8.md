# --interactive--

L'elemento superscript viene usato per mostrare un testo come apice. Un apice è un simbolo o una lettera stampata sopra la linea normale del testo.

Ecco un esempio che usa l'elemento superscript per illustrare gli esponenti (per vedere le anteprime, abilita l'editor interattivo):

:::interactive_editor

```html
<p>2<sup>2</sup> (2 squared) is 4.</p>
```

:::

In questo esempio, il numero 2 è racchiuso tra tag `sup` per rappresentare l'apice all'interno del paragrafo. Nella finestra di anteprima vedrai che il secondo numero 2 è più piccolo e leggermente più in alto rispetto al primo numero 2.

Gli usi comuni dell'elemento superscript includono esponenti, lettere apicali e numeri ordinali. Ecco un esempio che usa l'elemento superscript per lettere apicali (per vedere le anteprime, abilita l'editor interattivo):

:::interactive_editor

```html
<p>
  Monseigneur is often written as <strong>M<sup>gr</sup></strong>.
</p>
```

:::

Le lettere apicali indicano lettere scritte in apice, di solito per indicare abbreviazioni. Le lettere `g` e `r` sono racchiuse all'interno di tag superscript per illustrare l'abbreviazione in questo esempio.

È importante notare che l'elemento superscript dovrebbe essere usato solo per motivi tipografici. Se vuoi definire lo stile di un testo con una linea di base rialzata, allora dovresti usare CSS invece dell'elemento superscript.

Per rappresentare equazioni chimiche in HTML, si usa l'elemento subscript. Questo elemento usa un pedice che mostra una linea di base abbassata con testo più piccolo.

Ecco un esempio che usa l'elemento subscript per mostrare la rappresentazione chimica dell'anidride carbonica (per vedere le anteprime, abilita l'editor interattivo):

:::interactive_editor

```html
<p>CO<sub>2</sub></p>
```

:::

Il numero due è racchiuso tra tag `sub` per illustrare che il carattere dovrebbe essere un pedice.

Gli usi comuni dell'elemento subscript includono formule chimiche, note a piè di pagina e pedici di variabili.
