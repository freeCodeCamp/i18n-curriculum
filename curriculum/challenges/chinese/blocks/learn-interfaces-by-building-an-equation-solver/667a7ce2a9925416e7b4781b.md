# --description--

结构模式匹配使你能够验证主题是否具有特定的结构。 除此之外，它还将表达式中的名称与主题的元素绑定。 例如：

```py
match my_list:
    case [a]:
        print(a)
    case [a, b]:
        print(a, b)
```

修改 `match`/`case` 构造以匹配 `results` 而不是 `len(results)`。 然后，修改每个 `case` 以使用具有适当数量元素的列表。 如果列表包含单个元素，则使用 `x`；如果列表包含两个元素，则使用 `x1` 和 `x2`。

最后，修改 f-string 以使用每个 `case` 中使用的变量名。
