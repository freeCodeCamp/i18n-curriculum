# --explanation--

`Scheduled` 表示某事已提前計劃好，或安排在特定時間。 例如：

`The scheduled meeting will start at 2 PM.`（預定的會議將於下午2點開始）——說明會議已提前確定在指定時間召開。

`Maintenance` 指爲確保某物正常運作或解決問題而採取的措施。 例如：

`The team performed maintenance on the server to ensure it runs smoothly.`（團隊對服務器進行了維護以確保其平穩運行）——指通過修復或更新來保持服務器處於良好狀態。
