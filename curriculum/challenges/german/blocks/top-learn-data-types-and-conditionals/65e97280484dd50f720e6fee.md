# --description--

Im letzten Beispiel hatten Sie zwei Auswahlmöglichkeiten `if` oder `else`. Aber was, wenn Sie mehr als zwei Auswahlmöglichkeiten haben möchten? Sie können die `else if`-Anweisung verwenden, um unserem Code weitere Bedingungen hinzuzufügen. Die `else if`-Anweisung ermöglicht es uns, mehrere Bedingungen zu prüfen und einen Codeblock auszuführen, wenn die erste Bedingung wahr ist. Wenn die erste Bedingung falsch ist, wird die nächste Bedingung geprüft und so weiter. Wenn keine der Bedingungen wahr ist, wird der `else`-Block ausgeführt.

Hier ist ein Beispiel für die `else if`-Anweisung:

```javascript
let x = 5;

if (x > 10) {
  console.log("x is greater than 10");
} else if (x > 5) {
  console.log("x is greater than 5");
} else {
  console.log("x is less than or equal to 5");
}
```

In diesem Beispiel wird die `x`-Variable mit der Zahl `10` unter Verwendung des `>`-Operators verglichen. Wenn `x` größer als `10` ist, wird der erste Codeblock ausgeführt. Andernfalls prüft die `else if`-Anweisung die nächste Bedingung. Wenn `x` größer als `5` ist, wird der zweite Codeblock ausgeführt. Andernfalls wird der `else`-Block ausgeführt.
