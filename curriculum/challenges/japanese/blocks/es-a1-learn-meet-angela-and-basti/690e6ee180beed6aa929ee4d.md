# --description--

自己紹介された後に丁寧に返事をする方法を見てみましょう。

`Mucho gusto` と `Encantada/Encantado` はどちらも「はじめまして」という意味です。

`Mucho gusto` は性別に関係なく使え、ほとんどの状況で使えます。

`Encantada/Encantado` はより感情を込めた表現で、話者の性別に合わせる必要があります。

| 形 | 話者の性別 | 意味 |
| :--- | :--- | :--- |
| `Encantado` | **男性** (`-o`) | はじめまして。 |
| `Encantada` | **女性** (`-a`) | はじめまして。 |

これらのフレーズのどちらかを使うことは、スペイン語の自己紹介で丁寧さを示すために重要です。
