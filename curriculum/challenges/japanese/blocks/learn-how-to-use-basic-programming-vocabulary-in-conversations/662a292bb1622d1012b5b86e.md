# --description--

`how much` と `how many` を使って数量について尋ねる方法を学びましょう。`How much` は、`time`、`water`、`money` のように、数えられない名詞と一緒に使い、量や程度を尋ねるときに使います。例えば、`How much money do you have?` は個別に数えられないお金の量について尋ねます。

`How many` は、`books`、`apples`、`people` のように、数えられる名詞と一緒に使い、アイテムの数を尋ねるときに使います。例えば、`How many books do you own?` は数えられる本の数を尋ねます。
