# --explanation--

`Of course` は丁寧に同意を示したり、ためらわずに `yes` と言う方法です。話者が喜んで何かをする意志があることを意味します。例えば：

`Of course, I'll help you with the report.` – これは話者が喜んで助けることを意味します。

`Will` は約束をしたり、話している瞬間に決断を表すためによく使われます。例えば：

`I'll email you the file this afternoon.` – これはすぐに何かをする約束を示しています。
