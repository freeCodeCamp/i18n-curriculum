# --explanation--

`Weak passwords` 是容易被猜測或破解的密碼。例如：

`123456 and qwerty are weak passwords.` - 強密碼應該很長，並包含字母、數字和符號。

`Shared credentials` 是由多個人使用的登入資訊（使用者名稱和密碼）。例如：

`Using shared credentials increases security risks.`－如果密碼落入錯誤的人手中，共用密碼可能導致未經授權的存取。
