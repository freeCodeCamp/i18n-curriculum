# --description--

Los elementos inline-block se comportan como elementos en línea, pero con relleno y margen al estilo bloque. `display: inline-block` es una herramienta útil para conocer, pero en la práctica, probablemente termines usando flexbox con más frecuencia si intentas alinear un montón de cajas. Flexbox se cubrirá en profundidad en la siguiente lección.

# --questions--    

## --text--

¿Cuál es la diferencia entre un elemento `inline` y un elemento `inline-block`?

## --answers--

Los elementos `inline` se muestran sin saltos de línea, mientras que los elementos `inline-block` se muestran con saltos de línea.

---

Los elementos `inline` no permiten establecer ancho y alto, mientras que los elementos `inline-block` permiten establecer ancho y alto.

---

Los elementos `inline` siempre se posicionan horizontalmente, mientras que los elementos `inline-block` se posicionan verticalmente.

---

Los elementos `inline` pueden contener elementos de nivel bloque, mientras que los elementos `inline-block` no pueden contener otros elementos.
