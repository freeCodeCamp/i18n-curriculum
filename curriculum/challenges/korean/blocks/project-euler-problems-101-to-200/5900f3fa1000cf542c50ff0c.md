# --description--

무한 다항급수 $A_G(x) = xG_1 + x^2G_2 + x^3G_3 + \cdots$를 생각하세요. 여기서 $G_k$는 두 번째 차수 점화식 $G_k = G_{k − 1} + G_{k − 2}, G_1 = 1$, $G_2 = 4$의 $k$번째 항입니다. 즉, 수열은 $1, 4, 5, 9, 14, 23, \ldots$입니다.

이 문제에서는 $A_G(x)$가 양의 정수가 되는 $x$ 값을 다룹니다.

첫 다섯 자연수에 대응하는 $x$ 값은 아래와 같습니다.

| $x$                         | $A_G(x)$ |
|-----------------------------|----------|
| $\frac{\sqrt{5} − 1}{4}$    | $1$      |
| $\frac{2}{5}$               | $2$      |
| $\frac{\sqrt{22} − 2}{6}$   | $3$      |
| $\frac{\sqrt{137} − 5}{14}$ | $4$      |
| $\frac{1}{2}$               | $5$      |

$x$가 유리수일 때 $A_G(x)$를 골든 너겟이라고 부르겠습니다. 골든 너겟은 점점 희귀해지기 때문입니다. 예를 들어, 20번째 골든 너겟은 211345365입니다. 처음 30개의 골든 너겟의 합을 구하세요.
