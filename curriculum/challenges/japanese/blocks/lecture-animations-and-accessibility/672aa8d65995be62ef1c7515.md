# --interactive--

CSSアニメーションは、JavaScriptや複雑なプログラミングを使わずに、ウェブページ上で動的で視覚的に魅力的な効果を作り出すことができます。指定した期間にわたって要素を異なるスタイル間で滑らかに変化させる方法を提供します。

基本的に、CSSアニメーションは2つの主要な要素で構成されています：`@keyframes`ルールとanimationプロパティです。

`@keyframes`ルールはアニメーションの段階とスタイルを定義します。アニメーションのさまざまな時点で要素がどのようなスタイルを持つべきかを指定します。

例を示します：

```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(0);
  }
}
```

この`@keyframes`ルールは、`slide-in`という名前で、要素を左から右へ移動させるアニメーションを定義しています。パーセンテージはアニメーションの進行度を表し、`0%`が開始、`100%`が終了を示します。

`translateX`関数は、`@keyframes`アニメーション内で、要素が表示される際の水平方向の位置を制御しています。

このアニメーションを要素に適用するには、`animation`プロパティを使います。この例ではアニメーションを無限に繰り返すように設定しているので、動作を確認できます（プレビューを見るにはインタラクティブエディターを有効にしてください）：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="sliding-element">Hello, I slide in!</div>
```

```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(0);
  }
}

.sliding-element {
  animation: slide-in 2s ease-out infinite;
}
```

:::

これは要素に`slide-in`アニメーションを2秒の期間で、`ease-out`タイミング関数を使って適用しています。

`animation`プロパティは実際にはいくつかの個別のプロパティのショートハンドです：

`animation-name`は使用する`@keyframes`ルールを指定します。

`animation-duration`はアニメーションが完了するまでの時間を設定します。

`animation-timing-function`はアニメーションの進行の仕方を定義します。例えば`ease`、`linear`、`ease-in-out`などです。

`animation-delay`はアニメーション開始前の遅延を指定します。

`animation-iteration-count`はアニメーションの繰り返し回数を設定します。

`animation-direction`はアニメーションを前方向、後方向、または交互に再生するかを決定します。

`animation-fill-mode`はアニメーションの前後で要素がどのようにスタイル付けされるかを指定します。

`animation-play-state`はアニメーションの一時停止と再開を可能にします。

これらのプロパティは個別に使うことで、より細かい制御ができます：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="complex-animation">Watch my colors change!</div>
```

```css
.complex-animation {
  animation-name: color-change;
  animation-duration: 3s;
  animation-timing-function: linear;
  animation-delay: 1s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}

@keyframes color-change {
  0% {
    background-color: red;
  }
  50% {
    background-color: blue;
  }
  100% {
    background-color: green;
  }
}
```

:::

これは要素の背景色を赤、青、緑の間で連続的に変化させるアニメーションを作成します。

CSSアニメーションは、要素にマウスを乗せるなどのさまざまなイベントでトリガーできます：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Hover over me!</button>
```

```css
.button {
  background-color: blue;
  transition: background-color 0.3s;
}

.button:hover {
  background-color: red;
}
```

:::

この例では基本的な効果に簡単な`transition`プロパティを使っていますが、CSSでインタラクティブなアニメーション要素を作成できることを示しています。

CSSアニメーションは強力ですが、使いすぎるとパフォーマンスが低下したり、アクセシビリティの観点で問題が生じたりする場合があります。アニメーションを控えめに使い、動きを控えたいユーザー向けにアニメーションを減らすか無効にするオプションを提供することが重要です。

CSSアニメーションはJavaScriptに頼らずに魅力的でインタラクティブなウェブ体験を作る方法を提供します。`@keyframes`と`animation`プロパティの原則を理解することで、パフォーマンスとアクセシビリティに優れたウェブデザインを実現できます。
