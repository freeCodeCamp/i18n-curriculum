**Ziel:** Bauen Sie eine App, die funktional ähnlich ist wie diese: <a href="https://drum-machine.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://drum-machine.freecodecamp.rocks/</a>.

Erfüllen Sie die untenstehenden User Stories und bringen Sie alle Tests zum Bestehen. Verwenden Sie beliebige Bibliotheken oder APIs, die Sie benötigen. Verleihen Sie ihr Ihren eigenen Stil.

Sie können jede Kombination aus HTML, JavaScript, CSS, Bootstrap, SASS, React, Redux und jQuery verwenden, um dieses Projekt abzuschließen. Sie sollten ein Frontend-Framework verwenden (zum Beispiel React), da es in diesem Abschnitt um das Lernen von Frontend-Frameworks geht. Weitere Technologien, die nicht oben aufgeführt sind, werden nicht empfohlen und deren Verwendung erfolgt auf eigenes Risiko. Wir planen, weitere Frontend-Frameworks wie Angular und Vue zu unterstützen, aber diese werden derzeit nicht unterstützt. Wir akzeptieren und versuchen, alle Fehlerberichte zu beheben, die den vorgeschlagenen Technologie-Stack für dieses Projekt verwenden. Viel Erfolg beim Programmieren!

**User Story #1:** Ich sollte einen äußeren Container mit einem entsprechenden `id="drum-machine"` sehen können, der alle anderen Elemente enthält.

**User Story #2:** Innerhalb von `#drum-machine` kann ich ein Element mit einem entsprechenden `id="display"` sehen.

**User Story #3:** Innerhalb von `#drum-machine` sehe ich 9 anklickbare Drum-Pad-Elemente, jedes mit einem Klassennamen `drum-pad`, einer eindeutigen ID, die den Audioclip beschreibt, den das Drum-Pad auslösen wird, und einem inneren Text, der einer der folgenden Tasten auf der Tastatur entspricht: `Q`, `W`, `E`, `A`, `S`, `D`, `Z`, `X`, `C`. Die Drum-Pads MÜSSEN in dieser Reihenfolge sein.

**User Story #4:** Innerhalb jedes `.drum-pad` sollte sich ein HTML5 `audio`-Element befinden, das ein `src`-Attribut hat, das auf einen Audioclip zeigt, einen Klassennamen `clip` und eine ID, die dem inneren Text seines übergeordneten `.drum-pad` entspricht (z. B. `id="Q"`, `id="W"`, `id="E"` usw.).

**User Story #5:** Wenn ich auf ein `.drum-pad`-Element klicke, sollte der Audioclip, der in seinem Kind-`audio`-Element enthalten ist, ausgelöst werden.

**User Story #6:** Wenn ich die Auslösetaste drücke, die mit jedem `.drum-pad` verbunden ist, sollte der Audioclip, der in seinem Kind-`audio`-Element enthalten ist, ausgelöst werden (z. B. das Drücken der `Q`-Taste sollte das Drum-Pad auslösen, das den String `Q` enthält, das Drücken der `W`-Taste sollte das Drum-Pad auslösen, das den String `W` enthält usw.).

**User Story #7:** Wenn ein `.drum-pad` ausgelöst wird, wird ein String, der den zugehörigen Audioclip beschreibt, als innerer Text des `#display`-Elements angezeigt (jeder String muss eindeutig sein).
