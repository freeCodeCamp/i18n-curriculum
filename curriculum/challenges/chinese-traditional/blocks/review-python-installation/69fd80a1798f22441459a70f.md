## 你區域環境中的 Python

- **安裝**：在 Windows、Mac 和 Linux 上安裝 Python 的最佳方式是從官方 Python 網站下載安裝程式 (`https://www.python.org/`)。
- **終端機**：一種以文字為基礎的介面，讓你透過輸入指令與你的電腦互動。每個作業系統都附帶一個預設的終端機應用程式。在 macOS 上，你可以使用 Terminal 應用程式。在 Windows 上，你可以使用命令提示字元或 PowerShell。在 Linux 上，每個桌面環境都有自己的預設終端機應用程式，例如 GNOME Terminal 或 Konsole。
- **整合開發環境（IDE）**：IDE 是指整合開發環境，具備包括程式碼編輯器、測試工具、終端機等多種功能。
- **Python 常用的程式碼編輯器和 IDE**：VS Code、PyCharm 和 Spyder
- **在本機執行程式碼**：要執行 Python 程式碼，你有幾種選擇。第一種是點擊 VS Code 右上角看起來像播放按鈕的執行按鈕。這會開啟終端機並在那裡執行你的 Python 腳本。

另一種選擇是開啟終端機並手動執行程式。以下是一個執行名為 `main.py` 的程式範例：

```bash
python main.py
```

你必須從 `main.py` 所儲存的資料夾執行此指令。例如，如果 `main.py` 在一個名為 `python-projects` 的資料夾內，先在終端機使用以下指令：

```bash
cd python-projects
```

然後執行：

```bash
python main.py
```

## 使用 Python 互動式 Shell

- **定義**：互動式 shell 是一個程式，讓你一次輸入一個指令並看到結果。打開終端機應用程式，輸入 `python` 並按下 `Enter`。這會啟動 Python 互動式 shell。`>>>` 符號表示 Python 正在等待你輸入指令。試著在終端機列印 `"Hello, world!"`：

```bash
print("Hello, world!")
```

你應該會看到文字像這樣出現：

```bash
>>> print("Hello, world!")
Hello, world!
```

文字列印完後，Python 會回到以下狀態：

```bash
>>>
```

這表示你可以輸入另一個指令。

- **讀取、求值、列印、迴圈循環（REPL）**：每當你輸入指令時，直譯器會讀取輸入、求值、列印結果，然後迴圈回到顯示 `>>>`，讓你繼續輸入更多指令。
