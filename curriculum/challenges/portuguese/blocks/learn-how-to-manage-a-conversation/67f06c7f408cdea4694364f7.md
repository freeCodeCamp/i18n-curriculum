# --explanation--

`I'll` é a contração de `I will` e é frequentemente usada para tomar uma decisão ou fazer uma promessa no momento da fala. 

Nesta frase, `I'll send you the details` é uma promessa que Brian está fazendo para agir. Outro exemplo: 

`I'll call you later with the results.` – Isso significa que o falante está prometendo ligar em um momento posterior para fornecer mais informações.
