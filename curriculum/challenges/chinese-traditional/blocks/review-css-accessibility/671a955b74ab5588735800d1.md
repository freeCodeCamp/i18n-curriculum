## 色彩對比工具

- **WebAIM 的 Color Contrast Checker**：這個線上工具允許你輸入設計的前景色和背景色，並立即查看它們是否符合 Web Content Accessibility Guidelines（WCAG）標準。
- **TPGi Colour Contrast Analyzer**：這是一款免費的色彩對比檢查工具，讓你檢查你的網站和應用程式是否符合網頁內容無障礙指引（WCAG）標準。此工具還附帶色盲模擬器特性，讓你能看到你的網站或應用程式在色覺障礙者眼中的樣貌。

## 無障礙樹狀結構

無障礙樹是一種結構，用於輔助技術，例如螢幕閱讀器，來解讀和互動網頁上的內容。

## `max()` 函式（程式）

`max()` 函式傳回一組以逗號分隔的值中最大的值：

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <img src="https://cdn.freecodecamp.org/curriculum/css-photo-gallery/9.jpg" alt="A cat sitting on a table">
</body>
```

```css
img {
  width: max(250px, 25vw);
}
```

:::

在上述範例中，如果視窗寬度小於 1000 像素，影像的寬度將會是 250px。如果視窗寬度大於 1000 像素，影像的寬度將會是 25vw。這是因為 25vw 相等於視窗寬度的 25％。

## 使用 CSS 和無障礙性的最佳實踐

- **`display: none;`**：使用 `display: none;` 意味著螢幕閱讀器和其他輔助技術將無法存取此內容，因為它不包含在無障礙樹中。因此，只有當你想要完全從視覺呈現和無障礙中移除內容時，才應該使用此方法。
- **`visibility: hidden;`**：此屬性和值會將內容視覺上隱藏，但仍保留在文件流程中，表示它仍佔據頁面上的空間。這些元素也不會被螢幕閱讀器讀取，因為它們已從存取樹中移除。
- **`.sr-only` CSS 類別**：這是一種常見技術，用來視覺上隱藏內容，同時保持其對螢幕閱讀器的可存取性。

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <span>This span element can be read</span>
  <span class="sr-only">This span element can only be read by screen readers</span>
  <p style="display: none">
    This text is hidden with display: none and is NOT accessible to screen readers.
  </p>
  <p style="visibility: hidden">
    This text is hidden with visibility: hidden and is NOT read by screen readers,
    but it still takes up space.
  </p>
</body>

```

```css
.sr-only {
  position: absolute;
  width: 1px;
  height: 1px;
  padding: 0;
  margin: -1px;
  overflow: hidden;
  clip: rect(0, 0, 0, 0);
  white-space: nowrap;
  border: 0;
}

```

:::

- **`scroll-behavior: smooth;`**：此屬性和值啟用平滑滾動行為。
- **`prefers-reduced-motion` 特性**：這是一個媒體特性，可用來偵測使用者的動畫偏好。

## 使用 HTML 屬性隱藏內容

- **`aria-hidden` 屬性**：用於將元素隱藏於使用輔助技術（例如螢幕閱讀器）的人員。舉例來說，這可用於隱藏不提供任何有意義內容的裝飾性圖片。
- **`hidden` 屬性**：此屬性被大多數現代瀏覽器支援（某功能），並且會在視覺上及無障礙樹中隱藏內容。它可以透過 JavaScript 輕鬆切換。

:::interactive_editor

```html
<p aria-hidden>This paragraph is visible for sighted users, but is hidden from assistive technology.</p>
<p hidden>This paragraph is hidden from both sighted users and assistive technology.</p>
<p>This is a normal paragraph</p>

```

:::

## `placeholder` 屬性的無障礙問題

使用佔位符文字對於存取性並不理想。使用者經常會將佔位符文字誤認為是實際的輸入值——他們以為輸入框中已經有值。
