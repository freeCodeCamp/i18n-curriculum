## Absolute Einheiten

- **`px` (Pixel)**: Diese absolute Einheit ist eine feste Maßeinheit in CSS. Sie ist die gebräuchlichste absolute Einheit und bietet präzise Kontrolle über Dimensionen. `1px` entspricht immer 1/96 Zoll.
- **`in` (Zoll)**: Diese absolute Einheit entspricht 96px.
- **`cm` (Zentimeter)**: Diese absolute Einheit entspricht 25,2/64 Zoll.
- **`mm` (Millimeter)**: Diese absolute Einheit entspricht 1/10 Zentimeter.
- **`q` (Viertelmillimeter)**: Diese absolute Einheit entspricht 1/40 Zentimeter.
- **`pc` (Picas)**: Diese absolute Einheit entspricht 1/6 Zoll.
- **`pt` (Punkte)**: Diese absolute Einheit entspricht 1/72 Zoll.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="units">
  <div class="unit px">px</div>
  <div class="unit inch">in</div>
  <div class="unit cm">cm</div>
  <div class="unit mm">mm</div>
  <div class="unit q">q</div>
  <div class="unit pc">pc</div>
  <div class="unit pt">pt</div>
</div>
```

```css
.units {
  display: flex;
  gap: 12px;
  flex-wrap: wrap;
  align-items: flex-end;
}

.unit {
  background: steelblue;
  color: white;
  text-align: center;
  padding: 4px;
}

.px {
  width: 40px;
  height: 40px;
}

.inch {
  width: 0.5in;
  height: 0.5in;
}

.cm {
  width: 1cm;
  height: 1cm;
}

.mm {
  width: 10mm;
  height: 10mm;
}

.q {
  width: 40q;
  height: 40q;
}

.pc {
  width: 6pc;
  height: 6pc;
}

.pt {
  width: 36pt;
  height: 36pt;
}
```

## Relative Einheiten

- **Prozentangaben**: Diese relativen Einheiten erlauben es Ihnen, Größen, Dimensionen und andere Eigenschaften als Anteil des übergeordneten Elements zu definieren. Wenn Sie beispielsweise `width: 50%;` bei einem Element einstellen, nimmt es die halbe Breite seines übergeordneten Containers ein.
- **`em` Einheit**: Diese Einheiten beziehen sich auf die Schriftgröße des Elements. Wenn Sie `ems` für die `font-size` Eigenschaft verwenden, ist die Größe des Texts relativ zur Schriftgröße des übergeordneten Elements.
- **`rem` Einheit**: Diese Einheiten beziehen sich auf die Schriftgröße des Wurzelelements, also des `html` Elements.
- **`vh` Einheit**: `vh` steht für `"viewport height"` und `1vh` entspricht 1 % der Höhe des Viewports.
- **`vw` Einheit**: `vw` steht für `"viewport width"` und `1vw` entspricht 1 % der Breite des Viewports.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  <div class="box percent">50%</div>
  <div class="box em">2em</div>
  <div class="box rem">2rem</div>
  <div class="box vh">10vh</div>
  <div class="box vw">10vw</div>
</div>
```

```css
html {
  font-size: 16px;
}

.parent {
  width: 200px;
  font-size: 20px;
  border: 2px dashed #555;
  padding: 10px;
  display: flex;
  flex-direction: column;
  gap: 12px;
}

.box {
  background: seagreen;
  color: white;
  text-align: center;
  padding: 6px;
}

.percent {
  width: 50%;
}

.em {
  font-size: 2em;
}

.rem {
  font-size: 2rem;
}

.vh {
  height: 10vh;
}

.vw {
  width: 10vw;
}
```

## `calc` Funktion

- **`calc()` Funktion**: Mit der `calc()` Funktion können Sie Berechnungen direkt in Ihren Stylesheets durchführen, um Eigenschaftswerte dynamisch zu bestimmen. Das bedeutet, dass Sie flexible und responsive Benutzeroberflächen erstellen können, indem Sie Dimensionen basierend auf der Viewport-Größe oder anderen Elementen berechnen.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="calc-box">calc()</div>
```

```css
.calc-box {
  width: calc(100% - 40px);
  padding: 10px;
  background: steelblue;
  color: white;
  text-align: center;
  border: 2px solid black;
}
```
