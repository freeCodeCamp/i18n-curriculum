# --description--

Zoezi hili linazingatia ukaguzi wa matumizi ya `can` kwa ajili ya kudhihirisha uwezo na matumizi ya kitenzi `try` katika wakati uliopita unaoendelea.

`Can` hutumika kuzungumzia uwezo wa mtu kufanya jambo fulani, kama `I can solve this problem.`

`Try` katika wakati uliopita unaoendelea (`was`/`were` + kitenzi katika fomu ya `-ing`) inaonyesha tendo linaloendelea wakati uliopita, kama `I was trying to fix the issue.`
