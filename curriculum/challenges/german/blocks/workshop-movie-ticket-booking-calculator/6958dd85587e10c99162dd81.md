# --description--

Wie Sie in den vorherigen Lektionen gelernt haben, kann in Python eine `if`-Anweisung verwendet werden, um Code auszuführen, wenn eine Bedingung wahr ist.

Eine `if`-Anweisung besteht aus dem Schlüsselwort `if`, gefolgt von einer Bedingung und einem Doppelpunkt. Der Code, der ausgeführt wird, wenn die Bedingung wahr ist und eingerückt sein muss, wird als Körper der `if`-Anweisung bezeichnet.

```py
if condition:
    # Code to execute if condition is True
```

In diesem Schritt prüfen Sie, ob der Benutzer basierend auf seinem Alter berechtigt ist, ein Kinoticket zu buchen.

Erstellen Sie eine `if`-Anweisung, um zu prüfen, ob `age` größer als `17` ist. Innerhalb des Körpers der `if`-Anweisung geben Sie `User is eligible to book a ticket` aus. Dadurch wird die Nachricht nur ausgegeben, wenn das Alter des Benutzers größer als `17` ist.

Denken Sie daran, den Körper der `if`-Anweisung einzurücken und die Nachricht innerhalb des `print()`-Aufrufs mit einfachen oder doppelten Anführungszeichen zu umgeben.
