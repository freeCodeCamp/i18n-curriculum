# --description--

在这个任务中，你将听到 Mateo 在被感谢后礼貌地回应。他的短语是西班牙语中承认别人感谢的标准表达方式。

还有其他表达“你欢迎”的选项。以下是一些例子：

- `De nada`——不客气。（正式和非正式）
- `Por nada` - 没什么。（非正式）
- `No hay de qué` - 不用谢。（更正式或传统）
- `Con gusto` - 很乐意。（拉丁美洲常用）
