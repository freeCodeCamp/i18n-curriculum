# --description--

`UX` steht für User Experience. Es beschreibt, wie sich eine Person fühlt, wenn sie ein Produkt, System oder eine Dienstleistung nutzt. Dazu gehören Websites, Apps und elektronische Geräte. Eine `UX designer` sorgt dafür, dass Dinge auf dem Computer oder Telefon einfach und angenehm zu bedienen sind.

Zum Beispiel: `The UX of this website is excellent because it's easy to navigate and find the information you need.`
