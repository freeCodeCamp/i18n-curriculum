# --description--

假設你有兩個字串，想要將它們組合起來生成一個訊息，如下所示：

```js
const str1 = "Hello";
const str2 = "World!";
const message = str1.push(str2);
```

<img src="https://cdn.statically.io/gh/TheOdinProject/curriculum/4ed59981b4ce2c60b5b83bf7415d3127b61821f5/foundations/javascript_basics/understanding_errors/imgs/03.png" style="width:100%" alt="an example of a type error in the Chrome developer console">

在這裡，你會收到一個 `TypeError`，訊息指出 `str1.push is not a function`。這是一個常見的錯誤訊息，會讓學習者感到困惑，因為你可能知道 `.push()` 確實是一個函式（例如，如果你之前用它來為陣列添加項目）。

但這就是重點－`.push()` 不是字串的方法，而是陣列的方法。因此，你找不到作為字串方法的「不是函式」。如果你將 `.push()` 改成 `.concat()`，一個正確的字串方法，我們的程式碼就會如預期執行！

當遇到 `TypeError` 時，一個值得記住的重點是要考慮你嘗試對其執行方法或操作的資料型別。你很可能會發現它並非你所想，或者該操作或方法與該型別不相容。

為什麼以下 JavaScript 程式碼會導致 `TypeError`？

```js
const str1 = "Hello";
const str2 = "World!";
const message = str1.push(str2);
```

## --answers--

`.push()` 方法只能用於物件，不能用於字串。

---

`.push()` 方法拼寫錯誤，應該是 .pusch()。

---
