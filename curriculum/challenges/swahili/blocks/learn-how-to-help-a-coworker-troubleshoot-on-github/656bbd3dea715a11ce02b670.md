# --description--

Katika uandishi wa msimbo, hasa unapotumia mifumo ya udhibiti wa toleo kama Git, `push` inamaanisha kupakia mabadiliko kutoka hifadhidata ya eneo la karibu kwenda ile ya mbali. 

Kitenzi cha zamani `pushed` kinaonyesha kuwa hatua hii tayari imetekelezwa. 

Kwa mfano, `He pushed his code to the GitHub repository` inamaanisha alipakia mabadiliko yake ya hivi karibuni ya msimbo kwenye GitHub.
