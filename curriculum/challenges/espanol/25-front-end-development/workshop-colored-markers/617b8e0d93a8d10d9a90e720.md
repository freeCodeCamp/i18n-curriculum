# --description--

Tenga en cuenta que el `background-color` de su marcador aún es rojo. Esto se debe a que estableció el valor rojo de la función `rgb` al máximo de `255`, o 100% rojo, y estableció ambos valores de verde y azul en `0`.

Ahora use la función `rgb` para establecer los otros colores.

En la regla CSS `.two`, use la función `rgb` para establecer el `background-color` al valor máximo para el verde, y `0` para los otros valores. Y en la regla CSS `.three`, use la función `rgb` para establecer el `background-color` al valor máximo para el azul, y `0` para los otros valores.
