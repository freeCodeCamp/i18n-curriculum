# --description--

字串是一連串被單引號或雙引號包圍的字元。在某些程式語言中，被單引號包圍的字元與被雙引號包圍的字元會有不同的處理方式，但在 Python 中，它們被視為相等。因此，當你處理字串時，可以使用任一種引號。以下是一些字串的範例：

```python
my_str_1 = 'Hello'
my_str_2 = "World"
```

如果你需要多行字串，可以使用三個雙引號或單引號：

```python
my_str_3 = """Multiline
string"""
my_str_4 = '''Another
multiline
string'''
```

如果你的字串包含單引號或雙引號，則你有兩個選項，可選方案：

- 使用相反種類的引號。也就是說，如果你的字串包含單引號，請使用雙引號將字串包起來，反之亦然：
    

```python
msg = "It's a sunny day"
quote = 'She said, "Hello World!"'
```

- 在字串中使用反斜線（`\`）來跳脫單引號或雙引號。使用這種方法，你可以使用單引號或雙引號來包裹字串本身：
    

```python
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

有時候，你可能需要檢查字串是否包含一個或多個字元。為此，Python 提供了 `in` 運算子，它會傳回一個布林值，指定該字元或字元是否存在於字串中。

以下是一些範例：

```python
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

現在，讓我們來看看你如何取得字串的長度並操作字串中的各個字元，這個過程稱為**索引**。要取得字串的長度，你可以使用內建的 `len()` 函式。以下是一個範例：

```python
my_str = 'Hello world'
print(len(my_str))  # 11
```

字串中的每個字元都有一個稱為索引的位置。索引是從零開始，這表示字串中第一個字元的索引是 `0`，第二個字元的索引是 `1`，依此類推。要透過索引存取字元，你可以使用方括號（`[]`），並在其中放入你想存取的字元索引。以下是一些範例：

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
```

也允許負索引，因此你可以用 `-1` 取得任何字串的最後一個字元，用 `-2` 取得倒數第二個字元，依此類推：

```python
my_str = 'Hello world'
print(my_str[-1])  # d
print(my_str[-2]) # l
```

許多其他程式語言將資料型別大致分為原生類型或引用類型。原生類型簡單且不可變的，表示宣告後無法更改。引用類型可以包含多個值，且可變或不可變。但 Python 並不在這兩個群組之間劃出明確界線。相反地，所有資料都被視為物件，有些物件是不可變的，而其他則是可變的。

不可變的資料型別在宣告後無法被修改或更動。你可以將它們的變數指向新的東西，這稱為重新指派，但你無法透過新增、移除或替換任何元素來改變原本的物件本身。 

字串是在 Python 中的不可變資料型別。這表示你可以將不同的字串指定給變數：

```python
greeting = 'hi'
greeting = 'hello'
print(greeting) # hello
```

但不允許直接修改字串：

```python
greeting = 'hi'
greeting[0] = 'H' # TypeError: 'str' object does not support item assignment
```

Python 中其他不可變的資料型別範例有整數、浮點數、布林值、tuple 和 range。你將在接下來的課程中認識這些型別。
