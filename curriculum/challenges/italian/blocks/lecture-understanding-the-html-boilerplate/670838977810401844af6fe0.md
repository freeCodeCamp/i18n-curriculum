# --description--

Impariamo a conoscere l'elemento `link` e come usarlo per collegare fogli di stile esterni.

L'elemento `link` serve a collegare risorse esterne come fogli di stile e icone del sito. Ecco la sintassi base per usare l'elemento `link` per un file CSS esterno:

```html
<link rel="stylesheet" href="./styles.css" />
```

L'attributo `rel` serve a specificare la relazione tra la risorsa collegata e il documento HTML. In questo caso, dobbiamo indicare che questa risorsa collegata è un `stylesheet`.

È considerata una buona pratica separare HTML e CSS in file diversi. Gli sviluppatori useranno l'elemento `link` per il loro file CSS esterno invece di scrivere tutto nel documento HTML.

L'attributo `href` serve a specificare la posizione dell'URL per la risorsa esterna.

Il `dot` seguito da una barra in avanti nell'esempio indica al computer di cercare nella cartella o directory corrente il file `styles.css`.

L'elemento `link` dovrebbe essere inserito all'interno dell'elemento `head` come mostrato nel seguente esempio:

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Examples of the link element</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

Spesso vedrai più elementi `link` all'interno di un codice professionale che collegano diversi fogli di stile, caratteri tipografici e icone. Ecco un esempio di uso dell'elemento `link` per collegare un carattere Google esterno chiamato *Playwright Cuba*:

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Playwrite+CU:wght@100..400&display=swap"
  rel="stylesheet"
/>
```

Google Fonts sono un insieme di caratteri personalizzati gratuiti e open source che puoi usare in qualsiasi progetto. Puoi scegliere quali caratteri usare e Google ti fornirà il codice HTML e CSS necessario. In questo esempio, il valore `preconnect` per l'attributo `rel` dice al browser di creare una connessione anticipata al valore specificato nell'attributo `href`. Questo serve ad accelerare i tempi di caricamento di queste risorse esterne.

Un altro uso comune dell'elemento `link` è collegare icone. Ecco un esempio di collegamento a un favicon:

```html
<link rel="icon" href="favicon.ico" />
```

Un favicon, abbreviazione di favorite icon, è una piccola icona solitamente mostrata nella scheda del browser accanto al titolo del sito. Molti siti usano un favicon per mostrare la loro icona di brand.
