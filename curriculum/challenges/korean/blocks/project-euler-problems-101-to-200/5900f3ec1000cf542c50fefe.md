# --description--

$n$의 근본(radical) $rad(n)$은 $n$의 서로 다른 소인수들의 곱입니다. 예를 들어, $504 = 2^3 × 3^2 × 7$이므로 $rad(504) = 2 × 3 × 7 = 42$입니다.

양의 정수 삼중항 (a, b, c)를 abc-히트라고 정의합니다. 조건은 다음과 같습니다:

1. $GCD(a, b) = GCD(a, c) = GCD(b, c) = 1$
2. $a &lt; b$
3. $a + b = c$
4. $rad(abc) &lt; c$

예를 들어, (5, 27, 32)는 abc-히트입니다. 이유는 다음과 같습니다:

1. $GCD(5, 27) = GCD(5, 32) = GCD(27, 32) = 1$
2. $5 &lt; 27$
3. $5 + 27 = 32$
4. $rad(4320) = 30 &lt; 32$

abc-히트는 매우 드물며 $c &lt; 1000$인 경우 31개만 존재하고, $\sum{c} = 12523$입니다.

$c &lt; 120000$인 경우의 $\sum{c}$를 구하세요.
