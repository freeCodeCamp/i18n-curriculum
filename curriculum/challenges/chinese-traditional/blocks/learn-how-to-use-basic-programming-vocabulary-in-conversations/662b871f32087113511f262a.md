# --description--

樣式 `be` 後接動詞的 `-ing` 形式用來描述正在發生的動作。

片語 `have trouble with` 用來表達處理特定問題或物件時遇到困難。 例如，`She is having trouble with her computer` 意味著她在使用她的電腦時遇到困難。
