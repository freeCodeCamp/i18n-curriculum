# --description--

La sequenza Q di Hofstadter è definita come:

$Q(1)=Q(2)=1, \\\\ Q(n)=Q\\big(n-Q(n-1)\\big)+Q\\big(n-Q(n-2)), \\quad n>2.$

È definita come la sequenza di Fibonacci, ma mentre il termine successivo nella sequenza di Fibonacci è la somma dei due termini precedenti, nella sequenza Q i due termini precedenti indicano quanto tornare indietro nella sequenza Q per trovare i due numeri da sommare per ottenere il termine successivo della sequenza.

# --instructions--

Implementa l'equazione della sequenza Q di Hofstadter come una funzione. La funzione deve accettare un numero, `n`, e restituire un intero.

```js
function hofstadterQ (n) {
  const memo = [1, 1, 1];
  const Q = function (i) {
    let result = memo[i];
    if (typeof result !== 'number') {
      result = Q(i - Q(i - 1)) + Q(i - Q(i - 2));
      memo[i] = result;
    }
    return result;
  };
  return Q(n);
}
```
