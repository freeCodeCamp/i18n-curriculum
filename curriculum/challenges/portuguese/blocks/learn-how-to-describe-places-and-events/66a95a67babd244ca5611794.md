# --explanation--

`Does` é usado para formar uma pergunta no tempo `Present Simple` para sujeitos na terceira pessoa do singular.

`Yet` é frequentemente usado em perguntas para perguntar se algo que se espera aconteceu.Isso implica que a ação ou evento é antecipado mas não ocorreu até este ponto.Por exemplo:

- `Have you finished your homework yet?` (Você espera que o dever de casa seja concluído em breve, se ainda não estiver).
- `Has the movie started yet?` (Você espera que o filme comece em breve, se ainda não começou).
