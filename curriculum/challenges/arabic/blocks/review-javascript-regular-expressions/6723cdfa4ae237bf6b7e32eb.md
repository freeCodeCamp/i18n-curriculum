## التعبيرات النمطية والطرق الشائعة

- **تعريف**: التعبيرات النمطية، أو Regex، تُستخدم لإنشاء "نمط" يمكنك استخدامه للتحقق من سلسلة نصية، استخراج نص، والمزيد.

```js
const regex = /freeCodeCamp/;
```

- **طريقة `test()`**: تقبل هذه الطريقة سلسلة نصية لاختبار تطابقها مع التعبير النمطي. تُرجع قيمة منطقية إذا تطابقت السلسلة مع التعبير.

:::محرر_تفاعلي

```js
const regex = /freeCodeCamp/;
const test = regex.test("e");
console.log(test); // false
```

:::

- **طريقة `match()`**: تقبل هذه الطريقة تعبيرًا نمطيًا، ويمكنك أيضًا تمرير سلسلة تُحوّل إلى تعبير نمطي. تُرجع طريقة `match` مصفوفة التطابقات للسلسلة.

```js
const regex = /freeCodeCamp/;
const match = "freeCodeCamp".match(regex);
console.log(match); // ['freeCodeCamp', index: 0, input: 'freeCodeCamp', groups: undefined]
```

- **طريقة `replace()`**: تقبل هذه الطريقة معاملين: التعبير النمطي للمطابقة (أو سلسلة)، والسلسلة التي تستبدل بها المطابقة (أو دالة تُنفذ على كل تطابق).

:::محرر_تفاعلي

```js
const regex = /Jessica/;
const str = "Jessica is rly kewl";
const replaced = str.replace(regex, "freeCodeCamp");
console.log(replaced); // "freeCodeCamp is rly kewl"
```

:::

- **طريقة `replaceAll`**: تُستخدم هذه الطريقة لاستبدال كل حالات نمط معين بسلسلة جديدة. سترمي خطأ إذا أعطيتها تعبيرًا نمطيًا بدون المعدل العام (global modifier).

:::محرر_تفاعلي

```js
const text = "I hate JavaScript! I hate programming!";
const newText = text.replaceAll("hate", "love");
console.log(newText);  // "I love JavaScript! I love programming!"
```

:::

- **طريقة `matchAll`**: تُستخدم لاسترجاع كل التطابقات لتعبير نمطي معين في سلسلة نصية، بما في ذلك مجموعات الالتقاط، وتُرجعها كمكرر (iterator). المكرر هو كائن يسمح لك بالتنقل عبر مجموعة من العناصر.

:::محرر_تفاعلي

```js
const str = "JavaScript, Python, JavaScript, Swift, JavaScript";
const regex = /JavaScript/g;

const iterator = str.matchAll(regex);

for (let match of iterator) {
  console.log(match[0]); // "JavaScript" for each match
}
```

:::

## معدلات التعبير النمطي

- **تعريف**: المعدلات، المعروفة أيضًا بـ"الأعلام"، تُغير سلوك التعبير النمطي.
- **العلم `i`**: يجعل التعبير النمطي يتجاهل حالة الأحرف.

:::محرر_تفاعلي

```js
const regex = /freeCodeCamp/i;
console.log(regex.test("freecodecamp")); // true
console.log(regex.test("FREECODECAMP")); // true
```

:::

- **العلم `g`**: هذا العلم، أو المعدل العام، يسمح لتعبيرك النمطي بمطابقة النمط أكثر من مرة.

:::محرر_تفاعلي

```js
const regex = /freeCodeCamp/gi;
console.log(regex.test("freeCodeCamp")); // true
console.log(regex.test("freeCodeCamp is great")); // false
```

:::

- **تعريف الوسم (Anchor)**: الوسم `^` في بداية التعبير النمطي يعني "مطابقة بداية السلسلة". الوسم `$` في نهاية التعبير يعني "مطابقة نهاية السلسلة".

:::محرر_تفاعلي

```js
const start = /^freeCodeCamp/i;
const end = /freeCodeCamp$/i;
console.log(start.test("freecodecamp")); // true
console.log(end.test("freecodecamp")); // true
```

:::

- **العلم `m`**: الوسوم تبحث عن بداية ونهاية السلسلة بأكملها. لكن يمكنك جعل التعبير النمطي يتعامل مع عدة أسطر باستخدام العلم `m`، أو معدل متعدد الأسطر.

:::محرر_تفاعلي

```js
const start = /^freecodecamp/im;
const end = /freecodecamp$/im;
const str = `I love 
freecodecamp
it's my favorite
`;
console.log(start.test(str)); // true
console.log(end.test(str)); // true
```

:::

- **العلم `d`**: هذا العلم يوسع المعلومات التي تحصل عليها في كائن التطابق.

```js
const regex = /freecodecamp/di;
const string = "we love freecodecamp isn't freecodecamp great?";
console.log(string.match(regex));
```

والنتيجة هي:

```js
// [
//   'freecodecamp',
//   index: 8,
//   input: "we love freecodecamp isn't freecodecamp great?",
//   groups: undefined,
//   indices: [
//     0: [8, 20],
//     groups: undefined
//   ]
// ]
```

- **علامة `u`**: توسع هذه العلامة وظيفة التعبير النمطي لتسمح له بمطابقة أحرف يونيكود الخاصة. تمنحك علامة `u` إمكانية الوصول إلى أصناف خاصة مثل `Extended_Pictographic` لمطابقة معظم الرموز التعبيرية. هناك أيضًا علامة `v`، التي توسع وظيفة مطابقة اليونيكود أكثر.
- **علامة `y`**: يعد معدل اللزوجة مشابهًا جدًا لمعدل التعميم، مع بعض الاستثناءات. أهمها أن التعبير النمطي العام يبدأ من lastIndex ويبحث في بقية السلسلة عن تطابق آخر، بينما التعبير اللزج يعيد null ويعيد تعيين lastIndex إلى 0 إذا لم يكن هناك تطابق فوري عند lastIndex السابق.
- **علامة `s`**: يسمح معدل السطر الواحد للحرف البدل، الممثل بـ `.` في التعبير النمطي، بمطابقة فواصل الأسطر - مما يعامل السلسلة النصية كسطر واحد فقط.

## فئات الأحرف

- **الحرف البدل `.`**: فئات الأحرف هي صيغة خاصة يمكنك استخدامها لمطابقة مجموعات أو مجموعات فرعية من الأحرف. أول فئة أحرف يجب أن تتعلمها هي فئة الحرف البدل. يُمثل الحرف البدل بنقطة، ويطابق أي حرف مفرد باستثناء فواصل الأسطر. للسماح للحرف البدل بمطابقة فواصل الأسطر، تذكر أنه يجب استخدام العلم `s`.

```js
const regex = /a./;
```

- **`\d`**: سيطابق كل الأرقام (`0-9`) في السلسلة.

```js
const regex = /\d/;
```

- **`\w`**: يُستخدم لمطابقة أي حرف كلمة (`a-zA-Z0-9_`) في السلسلة. يُعرف حرف الكلمة بأنه أي حرف من a إلى z أو A إلى Z، أو رقم من 0 إلى 9، أو الشرطة السفلية.

```js
const regex = /\w/;
```

- **`\s`**: فئة الفراغ `\s` تمثلها الشرطة المائلة العكسية متبوعة بـ `s`. تطابق هذه الفئة أي فراغ، بما في ذلك الأسطر الجديدة، والمسافات، وعلامات الجدولة، وأحرف الفراغ الخاصة باليونيكود.
- **نفي فئات الأحرف الخاصة**: لنفي إحدى هذه الفئات، بدلاً من استخدام حرف صغير بعد الشرطة المائلة، يمكنك استخدام الحرف الكبير المكافئ. المثال التالي لا يطابق حرفًا رقميًا. بدلاً من ذلك، يطابق أي حرف مفرد ليس رقميًا.

```js
const regex = /\D/;
```

- **فئات الأحرف المخصصة**: يمكنك إنشاء فئات أحرف مخصصة بوضع الأحرف التي تريد مطابقتها داخل مجموعة من الأقواس المربعة.

```js
const regex = /[abcdf]/;
```

## تأكيدات النظر للأمام والنظر للخلف

- **تعريف**: تسمح تأكيدات النظر للأمام والنظر للخلف بمطابقة أنماط محددة بناءً على وجود أو عدم وجود أنماط محيطة.
- **تأكيد النظر للأمام الإيجابي**: يطابق نمطًا عندما يتبعه نمط آخر. لبناء تأكيد نظر للأمام إيجابي، تبدأ بالنمط الذي تريد مطابقته، ثم تستخدم أقواسًا لتغليف النمط الذي تريد استخدامه كشرط. بعد القوس الافتتاحي، استخدم `?=` لتعريف النمط كتأكيد نظر للأمام إيجابي.

```js
const regex = /free(?=code)/i;
```

- **تأكيد النظر للأمام السلبي**: هو نوع من الشروط في التعبيرات النمطية للتحقق من أن نمطًا معينًا لا يحدث أمام السلسلة.

```js
const regex = /free(?!code)/i;
```

- **تأكيد النظر للخلف الإيجابي**: يطابق نمطًا فقط إذا سبق بنمط محدد آخر، دون تضمين النمط السابق في المطابقة.

```js
const regex = /(?<=free)code/i;
```

- **تأكيد النظر للخلف السلبي**: يضمن أن نمطًا لا يسبقه نمط محدد آخر. يطابق فقط إذا لم يسبقه النمط المحدد مباشرة، دون تضمين التسلسل السابق في المطابقة.

```js
const regex = /(?<!free)code/i;
```

## المحددات الكمية في التعبيرات النمطية

- **تعريف**: المحددات الكمية في التعبيرات النمطية تحدد عدد مرات ظهور نمط (أو جزء من نمط). تساعد في التحكم بعدد تكرارات الأحرف أو المجموعات في المطابقة. المثال التالي يُستخدم لمطابقة الحرف السابق بالضبط أربع مرات.

```js
const regex = /^\d{4}$/;
```

- **`*`** : يطابق 0 أو أكثر من تكرارات العنصر السابق.
- **`+`**: يطابق 1 أو أكثر من تكرارات العنصر السابق.
- **`?`**: يطابق 0 أو 1 تكرار من العنصر السابق.
- **`{n}`**: يطابق بالضبط n تكرار من العنصر السابق.
- **`{n,}`**: يطابق n أو أكثر من تكرارات العنصر السابق.
- **`{n,m}`**: يطابق بين n و m تكرارات من العنصر السابق.

## مجموعات الالتقاط والمرجع العكسي

- **مجموعات الالتقاط**: تسمح لك مجموعة الالتقاط بـ"التقاط" جزء من السلسلة المطابقة لاستخدامه كما تريد. تُعرف مجموعات الالتقاط بالأقواس التي تحتوي على النمط المراد التقاطه، بدون أحرف قيادية مثل النظر للأمام.

```js
const regex = /free(code)camp/i;
```

- **المجموعات غير الالتقاطية**: تشبه مجموعة الالتقاط لكنها لا تخزن الجزء المطابق من السلسلة للاستخدام لاحقًا. تُعرف المجموعات غير الالتقاطية بـ `(?:...)`.

```js
const regex = /free(?:code)camp/i;
```

- **المرجع العكسي**: يشير المرجع العكسي في التعبيرات النمطية إلى طريقة لإعادة استخدام جزء من النمط الذي تم مطابقته سابقًا في نفس التعبير. يسمح لك بالإشارة إلى مجموعة ملتقطة (جزء من النمط داخل الأقواس) برقمها. على سبيل المثال، `$1` يشير إلى المجموعة الملتقطة الأولى.

:::محرر_تفاعلي

```js
const regex = /free(co+de)camp/i;
console.log("freecoooooooodecamp".replace(regex, "paid$1world"));
```

:::

- يمكنك استخدام المرجعيات الخلفية داخل التعبير النمطي نفسه لمطابقة نفس النص الذي التقطته مجموعة سابقة باستخدام الشرطة المائلة العكسية ورقم مجموعة الالتقاط. على سبيل المثال:

:::محرر_تفاعلي

```js
const regex = /(hello) \1/i;
console.log(regex.test("hello hello")); // true
console.log(regex.test("hello world")); // false
```

:::
