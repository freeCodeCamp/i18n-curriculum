`The team spread the project tasks evenly among all members.` – Esto significa que el trabajo se dividió de manera justa para que cada persona tuviera una cantidad similar que hacer.

`Funds` se refiere al dinero que está disponible para un propósito específico, como un proyecto, campaña o evento. Por ejemplo:
