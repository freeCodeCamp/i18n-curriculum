# --explanation--

`Just to clarify` é uma frase usada para garantir que o ouvinte entenda algo corretamente.Por exemplo:

`Just to clarify, the meeting is at 3 PM.` — Isso significa que o falante está confirmando os detalhes.

`Core features` são as partes essenciais de um produto ou sistema, aquelas que fazem com que ele funcione conforme o esperado.Por exemplo:

`The core features of the app include messaging and notifications.` — Estas são as principais funções do aplicativo.
