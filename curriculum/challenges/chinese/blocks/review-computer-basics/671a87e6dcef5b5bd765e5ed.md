## 理解计算机、互联网和开发者工具基础知识

- **主板**：包含运行计算机所需的所有内存、连接器和硬盘。它是计算机的主电路板。
- **中央处理器（CPU）**：负责执行指令和进行计算的处理器。
- **随机存取存储器（RAM）**：计算机 CPU 的临时存储位置。
- **硬盘驱动器（HDD）**：用于永久存储数据，即使计算机关机数据也不会丢失。
- **固态硬盘（SSD）**：非易失性闪存，可以替代硬盘使用。
- **电源供应单元（PSU）**：负责将墙壁插座的电力转换成计算机可用的形式。
- **图形处理器（GPU）**：负责在计算机屏幕上渲染视觉内容。
- **不同类型的互联网服务提供商**：互联网服务提供商（ISP）是提供互联网访问的公司。ISP 有多种类型，包括拨号、DSL、有线、光纤和卫星。
- **安全登录计算机的方法**：安全登录计算机的示例包括使用强密码、启用双因素认证和使用密码管理器。
- **集成开发环境（IDE）**：帮助开发者高效编写、测试和调试代码的工具。
- **代码编辑器**：开发者用来编写和调试代码的工具。
- **Git**：一种流行的版本控制系统，允许开发者跟踪代码更改并与他人协作。
- **基于云的代码仓库托管服务**：仓库是存储项目文件和版本历史的位置。流行的云托管服务包括 GitHub、GitLab 和 Bitbucket。
- **包管理器**：帮助开发者简化添加、更新和移除库及项目依赖的工具。示例包括 npm、pip 和 Maven。
- **测试库和框架**：软件测试用于确保代码按预期工作。测试库和框架的示例包括 Jest、PHPUnit 和 JUnit。

## 处理文件、文件系统和媒体格式

- **命名文件的最佳实践**：你需要以易于理解和维护的方式命名文件。例如，`about-us.html` 比 `page1.html` 更具描述性。
- **根目录**：文件系统中的顶层目录。目录是文件夹的另一种称呼。
- **Markdown**：一种常用于文档和 `README` 文件的标记语言。`README` 文件是包含有关项目如何安装和使用信息的文件。
- **`index.html`**：表示用户访问网站时显示的默认页面。
- **使用资源管理器/访达创建、移动和删除文件及文件夹**：资源管理器是 Windows 的文件管理器，访达是 macOS 的文件管理器。你可以使用这些工具创建、移动和删除文件及文件夹。
- **查找文件和文件夹**：你可以使用资源管理器或访达中的搜索功能查找计算机上的文件和文件夹。
- **HTML、CSS 和 JS 文件类型**：`.html` 文件扩展名用于 HTML 文件，`.css` 用于 CSS 文件，`.js` 用于 JavaScript 文件。
- **常见图像和图形格式**：`JPEG` 和 `PNG` 是常见的图像文件格式。`GIF` 是另一种支持动画的常见图像文件格式。`SVG` 是向量图形的文件格式。
- **常见音频和视频格式**：`MP3` 格式常用于音频文件。`MP4` 格式常用于视频文件。`MOV` 格式由 Apple 开发，常用于视频文件。
- **常见字体格式**：`TTF` 格式常用于 TrueType 字体。`WOFF` 格式常用于网页字体。`WOFF` 的继任者是 `WOFF2`，提供更好的压缩效果。
- **ZIP**：一种用于压缩文件和文件夹的文件格式。

## 高效浏览网页

- **什么是网页浏览器？**：允许用户访问和查看互联网网站的软件应用。
- **什么是搜索引擎？**：允许用户在互联网上搜索信息的工具。示例包括 Google、Bing 和 Yahoo。
- **常见网页浏览器**：常见网页浏览器的几个示例包括 Google Chrome、Mozilla Firefox 和 Microsoft Edge。
- **常见搜索策略**：你可以使用 `site:` 后跟网站 URL 来搜索该网站的内容。你可以使用 `filetype:` 后跟文件扩展名来搜索该类型的文件。你可以在搜索词前加减号以排除包含该词的结果。你可以在搜索词前加加号以包含包含该词的结果。
