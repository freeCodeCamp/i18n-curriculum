# --description--

但是如果你想要將陰影擴展得更遠呢？你可以使用選用的 `spreadRadius` 值來做到這一點：

```css
box-shadow: offsetX offsetY blurRadius spreadRadius color;
```

像 `blurRadius` 一樣，如果未包含 `spreadRadius`，則其預設值為 `0`。

透過在藍色標記周圍直接添加 5 像素陰影來練習。

在 `.blue` CSS 規則中，為你的 `box-shadow` 屬性添加值，`offsetX` 為 `0`、`offsetY` 為 `0`、`blurRadius` 為 `0`、`spreadRadius` 為 `5px`，以及 `color` 為 `blue`。

# --hints--

你的 `.blue` CSS 規則應該有一個 `box-shadow` 簡寫屬性，且值為 `0 0 0 5px blue`。

```js
assert(new __helpers.CSSHelp(document).getStyle('.blue')?.boxShadow === 'blue 0px 0px 0px 5px');
```
