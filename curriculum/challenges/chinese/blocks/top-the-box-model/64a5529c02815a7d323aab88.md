# --description--

能够检查和调试你的 `HTML` 和 `CSS` 对于前端开发至关重要。本课将带你了解 Chrome Dev Tools，它允许你查看关于你的元素和 `CSS` 规则的详细信息，并帮助你查找和修复代码中的错误。

要打开检查器，你可以右键点击网页上的任何元素，然后点击“Inspect”或按 F12。现在就去试试，查看此页面使用的超文本标记语言和 CSS。

不要被你现在看到的所有工具弄得不知所措！在本课中，我们要专注于 Elements 和 Styles 窗格。
