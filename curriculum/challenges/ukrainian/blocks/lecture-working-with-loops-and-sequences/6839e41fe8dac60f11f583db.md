# --description--

Функція `range()` використовується для генерації послідовності цілих чисел. Ось базовий синтаксис функції `range()`:

```py
range(start, stop, step)
```

Обов’язковий аргумент `stop` — це ціле число, яке представляє кінцеву точку для послідовності чисел, що генерується. Ось приклад використання функції `range()`:

```py
for num in range(3):
    print(num)
```

Наступний код генерує послідовність чисел між `0` та `2`. Ціле число `3` не включається, оскільки аргумент `stop` є неінклюзивним.

Якщо аргумент `start` не вказано, тоді за замовчуванням використовується `0`. Інакше ви можете використати необов’язковий аргумент `start`, щоб почати послідовність цілих чисел з числа, відмінного від `0`. Ось приклад генерації послідовності цілих чисел між `1` та `4`:

```py
for num in range(1, 5):
    print(num)
```

За замовчуванням послідовність цілих чисел збільшується на `1`. Але якщо ви хочете змінити це значення за замовчуванням, то можете використати необов’язковий аргумент `step`. Ось приклад генерації послідовності парних цілих чисел між `2` та `10`:

```py
for num in range(2, 11, 2):
    print(num)
```

Як уже згадувалося, функція `range()` має лише один обов’язковий аргумент. Якщо ви не передасте жодних аргументів функції `range()`, тоді отримаєте `TypeError`:

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: range expected at least 1 argument, got 0
```

Важливо зазначити, що функція `range()` приймає лише цілі числа як аргументи, а не числа з плаваючою комою. Пам’ятайте, що числа з плаваючою комою — це числа з десятковою крапкою, як `3.4`. Якщо ви спробуєте передати числа з плаваючою комою як аргументи, отримаєте `TypeError`:

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

Якщо ви хочете згенерувати послідовність цілих чисел у спадному порядку, то можете використати від’ємне число для аргументу `step`, ось так:

```py
for num in range(40, 0, -10):
    print(num)
```

Наступний код виводить числа `40`, `30`, `20` і `10` у такому порядку в консоль.

Ще одна річ, яку можна зробити з функцією `range()`, — це створити список цілих чисел, використовуючи її з конструктором `list`. Конструктор `list` використовується для перетворення ітерованого об’єкта на список. Ось приклад генерації списку парних цілих чисел між `2` та `10`:

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

Функція `range()` — це дуже зручний спосіб генерувати послідовність цілих чисел у Python. Як тільки ви освоїте її, ви, ймовірно, будете часто використовувати її у своїх програмах на Python.
