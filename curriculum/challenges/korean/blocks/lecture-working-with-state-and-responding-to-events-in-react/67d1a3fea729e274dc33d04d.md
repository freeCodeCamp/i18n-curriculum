# --description--

이벤트 핸들링은 모든 인터랙티브한 웹사이트에서 필수적인 부분입니다.

React는 이전 수업에서 배운 `click`, `keydown`, `submit` 같은 네이티브 이벤트를 감싸는 Synthetic Event System을 통해 강력하고 일관된 이벤트 처리 방식을 제공합니다.

이 크로스 브라우저 래퍼는 모든 브라우저에서 이벤트가 동일하게 작동하도록 보장하여 불일치가 없게 합니다.

React에서 이벤트가 어떻게 작동하는지 살펴보고 프로젝트에 활용해 보세요.

React에서 이벤트 핸들러는 네이티브 브라우저 이벤트와 비슷하게 작동하지만 약간의 차이가 있습니다.

소문자 이벤트 속성 이름인 `onclick`와 `onsubmit` 대신 React는 `onClick`와 `onSubmit` 같은 카멜 케이스를 사용합니다.

또한, 이벤트 종류를 문자열로 지정하는 대신 React는 이벤트 핸들러로 함수를 기대합니다.

이벤트 핸들러 함수는 프로프 형태로 요소에 전달되고, `onClick` 또는 `onSubmit` 같은 이벤트 타입은 JSX에서 속성으로 사용됩니다.

일반 HTML에서 클릭 이벤트를 다루는 방법을 다시 확인해 보세요:

```html
<button onclick="alert('Button clicked!')">Click Me</button>
```

React에서는 다음과 같이 합니다:

```jsx
function handleClick() {
  console.log("Button clicked!");
}

<button onClick={handleClick}>Click Me</button>;
```

이 예제에서 `handleClick`는 사용자가 버튼을 클릭할 때 콘솔에 메시지를 기록합니다.

`handleClick` 속성에 `onClick` 뒤에 소괄호를 붙이지 않는 점에 주의하세요. 함수 참조를 전달하는 것이지 호출하는 것이 아닙니다.

React에서 이벤트 핸들러 함수는 보통 `handle` 접두사로 시작하여 이벤트를 처리함을 나타냅니다. 예를 들어 `handleClick` 또는 `handleSubmit`가 있습니다.

사용자 동작이 이벤트를 발생시키면 React는 Synthetic Event 객체를 핸들러에 전달합니다. 이 객체는 일반 JavaScript의 네이티브 이벤트 객체처럼 동작하며 `type`, `target`, `currentTarget` 같은 속성을 제공합니다.

핸들러 함수에 `event`를 매개변수로 전달하고 콘솔에 기록하여 구조를 확인할 수 있습니다:

```js
function handleClick(event) {
  console.log(event);
}
```

예를 들어 `onSubmit` 이벤트 중 브라우저 새로 고침 같은 기본 동작을 막으려면 `preventDefault()` 메서드를 호출하세요:

```jsx
function handleSubmit(event) {
  event.preventDefault();
  console.log("Form submitted!");
}

<form onSubmit={handleSubmit}>
  <input type="text" />
  <button>Submit</button>
</form>;
```

또한 `event.stopPropagation()`을 호출해 이벤트가 상위 요소로 버블링되는 것을 막을 수 있습니다.

삭제 및 편집 기능 같은 특별한 경우를 처리할 때 이벤트 핸들러에 추가 데이터를 전달하고 싶으면 인라인 화살표 함수로 핸들러를 감싸면 됩니다:

```jsx
function handleDelete(id) {
  console.log("Deleting item:", id);
}

<button onClick={() => handleDelete(1)}>Delete Item</button>;
```

React에서는 인라인 이벤트 핸들러를 사용하는 것이 괜찮습니다. React가 효율적으로 리렌더링을 관리해 성능 문제를 피하기 때문입니다.

반면 일반 JavaScript에서는 가상 DOM이 없기 때문에 인라인 이벤트 핸들러가 매 렌더링마다 새 함수를 생성해 성능 문제를 일으킬 수 있습니다.
