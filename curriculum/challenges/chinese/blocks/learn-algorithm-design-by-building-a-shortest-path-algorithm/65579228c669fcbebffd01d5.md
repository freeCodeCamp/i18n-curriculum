# --description--

另一个错误则比较隐蔽。 当找到一条距离相邻节点比较短的路径时，`paths[current]` 将被分配给相邻节点路径，即 `paths[node]` 。

这意味着两个变量指向相同的列表。 因为列表是可变的，当你将邻接节点附加到其路径时， `paths[node]` 和 `paths[current]` 都被修改，因为它们是相同的列表。 这导致路径错误，尽管距离是正确的。

你可以通过将一个 `paths[current]` 分配给邻居节点路径来修复这个错误。 为此，你可以使用切片语法：

```py
my_list[:]
```

`my_list` 是你想要复制的列表。 通过切分 `paths[current]` 来修改 `if` 块内的现有 `paths[node] = paths[current]` 赋值。
