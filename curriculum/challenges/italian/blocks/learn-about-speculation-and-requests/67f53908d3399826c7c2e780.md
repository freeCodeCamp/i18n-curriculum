# --explanation--

`Want` esprime il desiderio di fare qualcosa. Al passato, si usa `wanted` per mostrare che il desiderio esisteva in un momento precedente. È seguito dall'infinito di un verbo (con `to`). Per esempio:

- `She wants to install the new software.` - Lei desidera impostare o aggiungere il nuovo software al sistema.

- `They wanted to upgrade their system last week.` - In passato, avevano pianificato di aggiornare.
