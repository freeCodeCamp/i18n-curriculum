# --description--

Матіо каже, скільки йому років іспанською, використовуючи таку структуру з дієсловом `tener` («мати»):

`Tengo` + число словами + `años`.

Пам’ятайте, що числа в іспанській від 21 до 29 починаються з `veinti-`, а потім іде відповідне число. Наприклад:

25 — це `veinticinco`. Це утворюється шляхом видалення `e` і додавання `i` перед числом `cinco`.

Числа від 1 до 9 іспанською:

| Число | Іспанське слово |
| :---: | :---: |
| 1 | `uno` |
| 2 | `dos` |
| 3 | `tres` |
| 4 | `cuatro` |
| 5 | `cinco` |
| 6 | `seis` |
| 7 | `siete` |
| 8 | `ocho` |
| 9 | `nueve` |
