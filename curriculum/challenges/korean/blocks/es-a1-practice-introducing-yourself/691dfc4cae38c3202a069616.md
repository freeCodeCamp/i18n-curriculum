# --explanation--

스페인어에서 21부터 29까지 숫자는 일반적으로 `veinti-`로 시작하는 하나의 단어로 쓰이며 뒤에 단위가 옵니다. 예를 들어:

- `Veinticinco` = 25

- `Veintidós` = 22

- `Veintiocho` = 28

- `Veinte` = 20
