# --explanation--

`Backend` 指軟體的伺服端零件 — 負責管理資料、邏輯和幕後處理程序的部分。在對話窗中，它作為複合形容詞並使用 `-`。另一個範例：

`She's working on the back-end code today.` — 她正在撰寫或修正幕後執行的程式碼零件。

`Scalability` 是指系統對成長的控制能力，例如更多使用者或更大量的資料。例如：

`They tested the system's scalability under high traffic.` — 他們檢查系統在大量使用者使用時是否運作良好。
