# --explanation--

`Bring in`는 팀, 프로젝트, 회사에 누군가를 초대하거나 추가하거나 고용하는 것을 의미하는 구동사입니다. 예를 들어:

- `He brought in a security expert to check the system.` - 그는 보안을 점검할 전문가를 초대했습니다.

- `We might need to bring in outside help if the issue continues.` - 문제가 멈추지 않으면 외부 지원이 필요할 수도 있습니다.
