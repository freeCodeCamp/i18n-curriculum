# --interactive--

In JavaScript, le stringhe sono trattate come sequenze di caratteri, e ogni carattere in una stringa può essere consultato usando la notazione a parentesi. Questo ti permette di recuperare un carattere specifico da una stringa in base alla sua posizione, chiamata indice.

Un indice è la posizione di un carattere all’interno di una stringa, ed è a base zero. Questo significa che il primo carattere di una stringa ha indice `0`, il secondo carattere ha indice `1`, e così via.

Per esempio, nella stringa `hello`, il carattere `h` si trova all’indice `0`, `e` si trova all’indice `1`, `l` si trova all’indice `2`, e così via.

La notazione a parentesi usa le parentesi quadre (`[]`) e l’indice del carattere che vuoi consultare. Vediamo un esempio:

:::interactive_editor

```js
let greeting = "hello";
console.log(greeting[1]); // "e"
```

:::

In questo esempio, possiamo consultare il carattere all’indice `1`, che è `e`.

Per ottenere l’ultimo carattere di una stringa, puoi usare la lunghezza della stringa meno uno.
La proprietà `length` di una stringa ti dice quanti caratteri contiene, quindi per consultare l’ultimo carattere devi sottrarre uno dalla lunghezza:

:::interactive_editor

```js
let greeting = "hello";
console.log(greeting[greeting.length - 1]); // "o"
```

:::

In questo caso, la `length` di `hello` è `5`, e l’ultimo carattere (`o`) si trova all’indice `4` che è `5 - 1`.

Se vuoi ottenere più caratteri, puoi usare la notazione a parentesi così:

:::interactive_editor

```js
let greeting = "hello";
let firstTwo = greeting[0] + greeting[1]; // "he"
console.log(firstTwo);
```

:::

In questo esempio, stiamo concatenando il primo e il secondo carattere usando la notazione a parentesi per formare la stringa `he`.

La notazione a parentesi è utile quando hai bisogno di consultare caratteri specifici in una stringa, come estrarre le iniziali da un nome o controllare una lettera specifica per la validazione.
