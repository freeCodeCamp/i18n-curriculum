# --description--

Kwa kushangaza kuna nambari tatu tu ambazo zinaweza kuandikwa kama jumla ya nguvu za nne za tarakimu zao:

<div style='margin-left: 4em;'>
  1634 = 1<sup>4</sup> + 6<sup>4</sup> + 3<sup>4</sup> + 4<sup>4</sup><br>
  8208 = 8<sup>4</sup> + 2<sup>4</sup> + 0<sup>4</sup> + 8<sup>4</sup><br>
  9474 = 9<sup>4</sup> + 4<sup>4</sup> + 7<sup>4</sup> + 4<sup>4</sup><br>
</div>

Kwa kuwa 1 = 1<sup>4</sup> siyo jumla, haijajumuishwa.

Jumla ya nambari hizi ni 1634 + 8208 + 9474 = 19316.

Tafuta jumla ya nambari zote ambazo zinaweza kuandikwa kama jumla ya `n` nguvu za tarakimu zao.
