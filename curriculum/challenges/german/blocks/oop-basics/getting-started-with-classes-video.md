## --text--

Welche der folgenden Möglichkeiten ist der richtige Weg, um einer Klasse eine Methode hinzuzufügen?

## --answers--

```py
class Dog:
    set bark(self):
        print("Woof!")
```

---

```py
class Dog:
    def bark(self):
        print("Woof!")
```

---

```py
class Dog:
    method bark(self):
        print("Woof!")
```

---

```py
class Dog:
    def bark:
        print("Woof!")
```
