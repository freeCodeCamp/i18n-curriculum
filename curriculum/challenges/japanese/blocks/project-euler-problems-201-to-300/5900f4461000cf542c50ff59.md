# --description--

辺の長さが $a=7$、$b=24$、$c=25$ の直角三角形を考えます。

この三角形のエリアは84で、完全数である6と28で割り切れます。

さらに、$gcd(a,b) = 1$ および $gcd(b,c) = 1$ なので、この三角形は原始的な直角三角形です。

また、$c$ は完全平方数です。

以下の条件を満たす直角三角形を「完全な直角三角形」と呼びます：

- 原始的な直角三角形であること
- 斜辺が完全平方数であること

さらに、以下の条件を満たす直角三角形を「超完全な直角三角形」と呼びます：

- 完全な直角三角形であること
- エリアが完全数6と28の倍数であること

$c ≤ {10}^{16}$ を満たす完全な直角三角形のうち、超完全でないものはいくつ存在しますか？
