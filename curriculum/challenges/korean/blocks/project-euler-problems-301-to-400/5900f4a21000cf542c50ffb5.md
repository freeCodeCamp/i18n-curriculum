# --description--

앨리스와 밥이 님 제곱수 게임을 합니다.

님 제곱수는 일반적인 세 힙 님과 같지만, 플레이어는 힙에서 제곱수만큼의 돌을 제거할 수 있습니다.

세 힙의 돌 개수는 순서쌍 ($a$, $b$, $c$)로 나타냅니다.

$0 ≤ a ≤ b ≤ c ≤ 29$일 때 다음 플레이어가 지는 위치의 개수는 1160입니다.

$0 ≤ a ≤ b ≤ c ≤ 100\,000$일 때 다음 플레이어가 지는 위치의 개수를 구하세요.
