# --description--

`Can` 用于表示某事是可能的或某人有能力做某事，而 `can't` 是 `cannot` 的缩写，表示某事是不可能的或某人没有能力做某事。两个词后面都应跟动词原形。

示例：

`I can swim.` (表示能力)

`I can't swim.`（表示无能或缺乏能力）

# --questions--

## --text--

哪个句子正确使用了 `can` 或 `can't`？

## --answers--

`I can to play the guitar.`

### --feedback--

这个句子不正确，因为 `to` 不应该和 `can` 一起使用。正确的句子是：`I can play the guitar.`

### --feedback--
