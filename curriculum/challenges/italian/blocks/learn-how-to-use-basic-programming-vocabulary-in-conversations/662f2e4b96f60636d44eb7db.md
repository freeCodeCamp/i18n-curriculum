# --description--

Una `class` è un concetto fondamentale, soprattutto in `object-oriented programming`. Funziona come un modello per creare oggetti. Una `class` definisce le proprietà (attributi) e i comportamenti (`methods`) che gli oggetti creati dalla `class` possono avere.

Per esempio, una classe `Person` potrebbe avere proprietà come `name` e `age`, e comportamenti come `talk()` e `walk()`.

Una `method` è una funzione associata a un oggetto. Le `Methods` definiscono i comportamenti o le azioni che un oggetto creato da una `class` può eseguire. Sono come istruzioni che dicono all’oggetto cosa fare.

Per esempio, in una `Calculator class`, una `method` potrebbe essere `add(number1, number2)`, che somma due numeri.
