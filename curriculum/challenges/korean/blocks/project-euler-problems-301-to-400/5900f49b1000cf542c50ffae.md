# --description--

양의 정수 $n$에 대해, $f(n)$을 10진수로 표현했을 때 숫자가 모두 $≤ 2$인 $n$의 가장 작은 양의 배수로 정의합니다.

따라서 $f(2) = 2$, $f(3) = 12$, $f(7) = 21$, $f(42) = 210$, $f(89) = 1\\,121\\,222$입니다.

또한, $\displaystyle\sum_{n = 1}^{100} \frac{f(n)}{n} = 11\\,363\\,107$입니다.

$\displaystyle\sum_{n = 1}^{10\\,000} \frac{f(n)}{n}$을 구하세요.
