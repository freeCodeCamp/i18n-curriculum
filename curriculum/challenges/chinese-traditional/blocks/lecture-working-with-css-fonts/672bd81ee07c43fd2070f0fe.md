# --description--

網頁安全字型是字型的子集，非常可能已安裝在電腦或裝置上。它們在不同作業系統和網頁瀏覽器中廣泛支援，因此很可能會被一致地算繪和呈現。讓我們來看看這是如何運作的。

瀏覽器負責解讀並呈現網站上的字型。當瀏覽器必須呈現字型時，它會嘗試在使用者系統中尋找該字型檔案。但如果找不到字型，通常會回退到預設系統字型。這確保即使網站上應該呈現的特定字型遺失，內容仍然可讀。

然而，瀏覽器所選擇的備用字型可能與原本預期要呈現的字型看起來非常不同。這可能對整體設計和使用者體驗產生關鍵影響。為避免這種情況，你應該盡可能使用網頁安全字型。你有兩個選項。你可以將它們用作你的主要字型，或者你可以使用訂製字型並將網頁安全字型作為備用選項。這樣一來，當找不到訂製字型時，你就能控制網站的外觀。

讓我們來看看一些網頁安全字型的範例。在之前的課程中，你已經學到無襯線字型通常用於網頁開發，因為它們的字元末端沒有小「腳」或線條，所以在螢幕上容易閱讀。一些網頁安全的無襯線字型範例如下：

- Arial
- Verdana
- Trebuchet MS

相較之下，襯線字型的字元末端確實有小「腳」，因此它們通常用於傳統列印。但如果你需要將它們用於網頁開發，網頁安全的襯線字型包括：

- Times New Roman
- 喬治亞

透過使用網頁安全字型，你可以確保你的設計在各種裝置和平台上看起來一致。它們也能提升視覺障礙使用者的可及性，因為這些字型簡單且易於閱讀。最後，網頁安全字型可以減少頁面載入時間，因為如果字型已經安裝，就不需要下載。

既然你已經知道什麼是網頁安全字型，你就可以開始使用它們並將它們指定為備用字型，以在不同瀏覽器和平台上創建可靠的使用者體驗。
