# --explanation--

`To maintain` significa mantener algo funcionando correctamente o en buen estado. Por ejemplo:

`We maintain the server by checking it regularly.` – Esto significa que lo cuidamos para que siga funcionando bien.

Una `vulnerability` es una debilidad en un sistema que podría ser atacada o utilizada para causar daño. Por ejemplo:

`The old version of the app had a security vulnerability.` – Esto significa que tenía un punto débil que los hackers podían aprovechar.
