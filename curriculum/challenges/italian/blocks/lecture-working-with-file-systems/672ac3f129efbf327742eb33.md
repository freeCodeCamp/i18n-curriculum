# --description--

Impariamo come cercare file e cartelle su Windows e macOS.

Iniziamo con Windows. Puoi cercare dalla barra delle applicazioni o dall’Esplora file.

Per cercare dalla barra delle applicazioni, scrivi il nome del file o della cartella che stai cercando. Puoi anche cercare parole chiave. Vedrai i risultati provenienti da tutto il tuo PC e da OneDrive.

Se vuoi vedere solo i documenti, puoi cliccare sulla scheda "Documenti". Hai anche schede per app e risultati web.

Un’altra alternativa è usare la ricerca nell’Esplora file. La trovi in alto a destra nella finestra dell’Esplora, dove vedi "Cerca documenti".

Hai tre opzioni per definire l’ambito della ricerca:

- Puoi cercare da Home per trovare rapidamente file dal tuo PC e dal cloud.

- Puoi cercare da una cartella per trovare i file archiviati al suo interno.

- Puoi cercare da Questo PC per una ricerca lenta ma approfondita.

Ecco come cercare file e cartelle su Windows. Ora vediamo come farlo su macOS.

Per prima cosa, puoi usare Spotlight, uno strumento che ti aiuta a trovare file sul Mac. Per aprire Spotlight, vai alla barra dei menu in alto a destra e trova l’icona della lente di ingrandimento. Se non vedi questa icona nella barra dei menu, puoi aggiungerla nelle impostazioni del Centro di controllo.

Vai nelle impostazioni, filtra per "Centro di controllo", clicca su questa opzione e poi scorri verso il basso finché non trovi l’opzione "Spotlight". Puoi anche usare una scorciatoia da tastiera per aprire Spotlight: `Command + Spacebar`.

Quando apri Spotlight, vedrai la ricerca Spotlight, dove puoi cercare qualsiasi file o cartella digitandone il nome. Quando inizi a digitare, vedrai i risultati raggruppati per categoria, incluse le suggerimenti, cartelle, presentazioni, foto, documenti e ricerche correlate.

Puoi anche espandere una categoria specifica cliccando su "Mostra altro".

Un altro modo per cercare file che hai aperto di recente è aprire Finder cliccando sull’icona Finder nel Dock. Poi vai su "Recenti" nella barra laterale di Finder. Lì troverai tutti i file che hai visualizzato di recente.

Ecco come cercare file e cartelle su macOS. Con questi strumenti di ricerca puoi ottimizzare il tuo flusso di lavoro e trovare molto rapidamente i file e le cartelle di cui hai bisogno.
