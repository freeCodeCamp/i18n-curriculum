# --description--

HTML 페이지의 거의 모든 요소는 여는 태그와 닫는 태그로 감싼 콘텐츠 조각입니다.

여는 태그는 브라우저에 HTML 요소의 시작임을 알려줍니다. 이것은 꺾쇠괄호 `<>` 안에 키워드가 들어간 형태입니다. 예를 들어, 여는 단락 태그는 이렇게 생겼습니다: `<p>`.

닫는 태그는 브라우저에 요소가 끝나는 위치를 알려줍니다. 여는 태그와 거의 같지만, 키워드 앞에 슬래시가 붙는 점이 다릅니다. 예를 들어, 닫는 단락 태그는 이렇게 생겼습니다: `</p>`.

완전한 단락 요소는 다음과 같습니다:
