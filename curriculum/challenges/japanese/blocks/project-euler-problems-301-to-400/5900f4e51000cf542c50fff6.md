# --description--

整数 $n$ の整数分割とは、$n$ を正の整数の和として表す方法です。

加算項の順序が異なるだけの分割は同じものとみなされます。異なる部分からなる $n$ の分割とは、各部分が最大で1回だけ現れる $n$ の分割のことです。

5 の異なる部分からなる分割は次の通りです：

5、4 + 1、3 + 2。

$f(n)$ を、$n$ のそのような異なる部分からなる分割の中で部分の積が最大となる値とし、$m(n)$ をその積を持つ分割の要素数とします。

したがって $f(5) = 6$ で $m(5) = 2$ です。

$n = 10$ の場合、最大積を持つ分割は $10 = 2 + 3 + 5$ で、$f(10) = 30$、$m(10) = 3$ となります。そしてその積は $f(10) \times m(10) = 30 \times 3 = 90$ です。

$1 ≤ n ≤ 100$ に対して $\sum f(n) \times m(n) = 1\,683\,550\,844\,462$ であることが確認できます。

$1 ≤ n ≤ {10}^{14}$ に対して $\sum f(n) \times m(n)$ を求めてください。答えは $982\,451\,653$（5,000万番目の素数）でのモジュラスを取ってください。
