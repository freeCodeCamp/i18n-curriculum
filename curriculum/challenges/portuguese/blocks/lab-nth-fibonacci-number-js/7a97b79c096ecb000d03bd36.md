2. Você deve definir um array chamado `sequence` dentro da função `fibonacci`, e ele deve ser inicializado com os valores `[0, 1]`.

# --solutions--

```js
function fibonacci(n) {
  const sequence = [0, 1];

  if (n < 2) {
    return sequence[n];
  }

  for (let i = 2; i <= n; i++) {
    sequence.push(sequence[i - 1] + sequence[i - 2]);
  }

  return sequence[n];
}
```
