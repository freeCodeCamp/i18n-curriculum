# --explanation--

`UI` steht für `user interface`, was sich auf die visuellen Elemente eines Produkts bezieht, mit denen Benutzer interagieren. `设计师 (shè jì shī)` bedeutet „Designer“.

Zusammen bedeutet `UI 设计师 (shè jì shī)` einen Beruf, der sich darauf konzentriert, das Aussehen und Layout von Produkten so zu gestalten, dass sie einfach und angenehm zu benutzen sind. Es ist ein sehr gebräuchlicher Begriff in Technologieunternehmen, und man verwendet direkt `UI`, ohne ihn ins Chinesische zu übersetzen. Zum Beispiel:

`我是 (wǒ shì) UI 设计师 (shè jì shī)。` – Ich bin UI-Designer.
