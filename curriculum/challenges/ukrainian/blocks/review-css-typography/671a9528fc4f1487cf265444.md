## Вступ до типографіки

- **Визначення**: Типографіка — це мистецтво вибору правильних шрифтів і форматування, щоб текст був візуально привабливим і легким для читання. "Тип" означає, як окремі символи спроектовані та розташовані.
- **Визначення сімейства шрифтів**: Сімейство шрифтів — це загальний дизайн і стиль набору символів, чисел і знаків. Це як креслення для сімейства шрифтів.
- **Визначення шрифту**: Шрифт — це конкретний варіант сімейства шрифтів із певними характеристиками, такими як розмір, товщина, стиль і ширина.
- **Серіфний шрифт**: Це сімейство шрифтів із класичним стилем і маленькими лініями на кінцях символів. Серіфні шрифти зазвичай використовують для друкованих матеріалів, наприклад, книг.
- **Безсерифний шрифт**: Це сімейство шрифтів із більш сучасним виглядом, без маленьких ліній на кінцях символів. Безсерифні шрифти часто використовують у цифровому дизайні, бо їх легко читати на екрані. Прикладами є Helvetica, Arial і Roboto.
- **Товщина шрифту**: Це товщина символів, включно з легким, звичайним, жирним і чорним варіантами.
- **Стиль шрифту**: Це нахил і орієнтація символів, наприклад, курсив і похилий.
- **Ширина шрифту**: Це горизонтальний простір, який займають символи, наприклад, стиснутий і розширений.
- **Базова лінія**: Уявна лінія, на якій стоять більшість символів.
- **Висота капіталів**: Висота великих літер, вимірювана від базової лінії до верхньої межі.
- **Висота x**: Середня висота малих літер, без урахування висхідних і низхідних частин.
- **Висхідні елементи**: Частини малих літер, що виступають над висотою x, наприклад, верхівки літер 'h', 'b', 'd' і 'f'.
- **Низхідні елементи**: Частини малих літер, що виступають нижче базової лінії, наприклад, хвостики 'y', 'g', 'p' і 'q'.
- **Кернінг**: Регулювання простору між конкретними парами символів для покращення читабельності та естетики. Наприклад, зменшення простору між літерами A і V.
- **Трекінг**: Регулювання простору між усіма символами в блоці тексту. Це відстань між символами, що впливає на щільність і відкритість тексту.
- **Інтерліньяж**: Вертикальний простір між рядками тексту. Вимірюється від базової лінії одного рядка до базової лінії наступного.
- **Найкращі практики типографіки**: Обирайте чіткі й прості шрифти, щоб ваші дизайни були легкими для розуміння. Це особливо важливо для основного тексту сайту. Користувачі частіше взаємодіють із вмістом, якщо шрифт легко читати. Використовуйте не більше двох-трьох шрифтів для створення візуальної узгодженості. Занадто багато шрифтів ускладнюють читання і послаблюють ідентичність бренду. Це також може вплинути на досвід користувача, збільшуючи час завантаження сайту. Розмір шрифту можна використовувати для створення візуальної ієрархії заголовків, підзаголовків, абзаців і інших елементів. Наприклад, головний заголовок на сторінці має бути більшим, а підзаголовки — меншими. Це допоможе користувачам візуально орієнтуватися в структурі.

  
## Властивість CSS `font-family`

- **Визначення**: Сімейство шрифтів — це група шрифтів із загальним дизайном. Усі шрифти в сімействі базуються на одному основному типі, але мають варіації стилю, товщини та ширини. Ви можете вказати кілька сімейств шрифтів у порядку пріоритету від найвищого до найнижчого, розділяючи їх комами. Ці альтернативні шрифти будуть запасними варіантами. Завжди слід додавати загальне сімейство шрифтів у кінці списку font-family.

```css
font-family: Arial, Lato;
```

- **Загальні сімейства шрифтів**: Ось деякі загальні сімейства шрифтів, які використовують у CSS: serif, sans-serif, monospace, cursive, fantasy 

## Безпечні для веб шрифти

- **Визначення**: Це підмножина шрифтів, які дуже ймовірно встановлені на комп’ютері або пристрої. Коли браузер має відтворити шрифт, він намагається знайти файл шрифту в системі користувача. Якщо шрифт не знайдено, зазвичай використовується системний шрифт за замовчуванням.

- **Безпечні для веб шрифти**:
  - Безсерифні шрифти часто використовують для веброзробки, бо вони не мають маленьких "ніжок" або ліній на кінцях символів, тому їх легко читати на екрані. Прикладами безпечних для веб безсерифних шрифтів є: Arial, Verdana і Trebuchet MS.
  - Натомість серіфні шрифти мають маленькі "ніжки" на кінцях символів, тому їх зазвичай використовують для традиційного друку. Безпечні для веб серіфні шрифти включають: Times New Roman і Georgia.

## Директиви та директива `@font-face`

- **Визначення**: Директиви — це інструкції для браузера. Їх можна використовувати для визначення різних аспектів таблиці стилів, таких як медіазапити, ключові кадри, шрифтові обличчя тощо.  
- **`@font-face`**: Ця директива дозволяє визначити кастомний шрифт, вказавши файл шрифту, формат і інші важливі властивості, як-от товщину і стиль. Щоб директива `@font-face` була дійсною, потрібно також вказати властивість `src`, яка містить референси на ресурси шрифту.

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2"),
       url("path/to/font.woff"),
       url("path/to/font.otf");
}
```

- **Формати шрифтів**: Для кожного ресурсу шрифту можна також вказати формат. Це необов’язково. Це підказка для браузера про формат шрифту. Якщо формат не вказано, ресурс буде скачано, а формат визначать після завантаження. Якщо формат недійсний, ресурс не буде скачано. Можливі формати шрифтів включають `collection`, `embedded-opentype`, `opentype`, `svg`, `truetype`, `woff` і `woff2`.

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
       url("path/to/font.otf") format("opentype"),
       url("path/to/font.woff") format("woff");
}
```

- **`woff` і `woff2`**: `woff` означає "Web Open Font Format". Це широко використовуваний формат шрифтів, розроблений Mozilla у співпраці з Type Supply, LettError та іншими організаціями. Різниця між `woff` і `woff2` полягає в алгоритмі стиснення даних.
- **OpenType**: Це формат масштабованих комп’ютерних шрифтів, розроблений Microsoft і Adobe, який дозволяє користувачам отримувати додаткові функції шрифту. Широко використовується в основних операційних системах.
- **`tech()`**: Використовується для вказання технології ресурсу шрифту. Це необов’язково.

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
       url("path/to/font.otf") format("opentype") tech(color-COLRv1),
       url("path/to/font.woff") format("woff");
}
```

## Робота із зовнішніми шрифтами

- **Визначення**: Зовнішній шрифт — це файл шрифту, який не включений безпосередньо у файли вашого проєкту. Зазвичай вони розміщені на окремому сервері. Щоб додати ці зовнішні шрифти у проєкт, можна використати елемент `link` або інструкцію `@import` у CSS. 
- **Google Fonts**: Це сервіс Google, який пропонує колекцію шрифтів, багато з яких спеціально розроблені для веброзробки.

Ось приклад із використанням елемента `link`:

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap" rel="stylesheet">
```

```css
.roboto-thin {
  font-family: "Roboto", sans-serif;
  font-weight: 100;
  font-style: normal;
}
```

Ось приклад із використанням інструкції `@import`:

```css
@import url('https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap');
```

- **Font Squirrel**: Це ще один популярний ресурс, який можна використовувати для додавання кастомних зовнішніх шрифтів у ваші проєкти.

## Властивість `text-shadow`

- **Визначення**: Ця властивість використовується для застосування тіней до тексту. Потрібно вказати зміщення по X і Y, які представляють горизонтальну і вертикальну відстань тіні від тексту відповідно. Ці значення обов’язкові. Позитивні значення зміщення по X і Y перемістять тінь вправо і вниз відповідно, а від’ємні — вліво і вгору.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Typography Shadow</p>
```

```css
  p {
    text-shadow: 3px 2px;
  }
```

:::

- **Колір тіні**: Ви також можете налаштувати колір тіні, вказавши це значення перед або після зміщення. Якщо колір не вказано, браузер визначить його автоматично, тому зазвичай краще задати його явно.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Colored Shadow</p>
```

```css
p {
  text-shadow: 3px 2px #00ffc3;
}
```

:::

- **Радіус розмиття**: Радіус розмиття необов’язковий, але робить тінь більш плавною і делікатною. Значення за замовчуванням — нуль. Чим більше значення, тим більший розмиття, а отже тінь стає світлішою.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Blurred Shadow</p>
```

```css
p {
  text-shadow: 3px 2px 3px #00ffc3;
}
```

:::

- **Застосування кількох тіней до тексту**: Текст може мати більше однієї тіні. Тіні накладаються шарами, від переднього до заднього, причому перша тінь буде зверху.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Multiple Shadows</p>
```

```css
p {
  text-shadow:
    3px 2px 3px #00ffc3,
    -3px -2px 3px #0077ff,
    5px 4px 3px #dee7e5;
}
```

:::
