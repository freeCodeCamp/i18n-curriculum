# --description--

時間の前置詞は、いつ何かが起こるかを話すのに欠かせません。

前置詞 `in` は、月、年、世紀、長期間に使われます。例えば、`We will update the software in July`、`I started learning coding in 2022`、`I like to work in the morning` です。

前置詞 `on` は、曜日や日付に使われます。例えば、`Our team meets on Mondays`、`The coding class is on April 10th`、`I finished the task on the last day` です。

前置詞 `at` は、1日の正確な時間を指定するときに使われます。例えば、`Our meeting is at 9 AM`、`I begin coding at 10 AM`、`We have lunch at 12 PM` です。
