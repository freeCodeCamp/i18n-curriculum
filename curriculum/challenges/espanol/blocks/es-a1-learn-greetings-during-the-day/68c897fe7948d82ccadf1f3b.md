# --explanation--

Camila ahora está usando la misma estructura que usó para decir su nombre. Pero esta vez, para decir su profesión. La estructura se ve así:

`Soy` + profesión

De nuevo, no necesitas incluir el pronombre `yo` porque el verbo `soy` ya te indica quién está hablando.

`Soy analista` significa "I am an analyst".

`Analista` es un sustantivo neutral, por lo que es igual para hablantes masculinos y femeninos. Por ejemplo:  

`Soy analista de datos.` – Soy analista de datos.

Esta oración de ejemplo funciona tanto para hombres como para mujeres.
