## Псевдокласи дій користувача

- **Визначення псевдокласів**: Це спеціальні ключові слова CSS, які дозволяють вибирати елемент залежно від його конкретного стану або позиції.
- **Псевдокласи дій користувача**: Це спеціальні ключові слова, які дозволяють змінювати вигляд елементів залежно від взаємодії користувача, покращуючи загальний користувацький досвід.
- **Псевдоклас `:active`**: Цей псевдоклас дозволяє вибрати активний стан елемента, наприклад, при натисканні на кнопку.
- **Псевдоклас `:hover`**: Цей псевдоклас визначає стан наведення на елемент.
- **Псевдоклас `:focus`**: Цей псевдоклас застосовує стилі, коли елемент отримує фокус, зазвичай через навігацію клавіатурою або коли користувач клікає в поле введення форми.
- **Псевдоклас `:focus-within`**: Цей псевдоклас використовується для застосування стилів до елемента, коли він або будь-який з його нащадків має фокус.

## Псевдокласи для input

- **Псевдокласи для input**: Ці псевдокласи використовуються для націлювання на HTML `input` елементи залежно від стану, в якому вони перебувають до та після взаємодії користувача.
- **Псевдоклас `:enabled`**: Цей псевдоклас використовується для націлювання на кнопки форми або інші елементи, які наразі увімкнені.
- **Псевдоклас `:disabled`**: Цей псевдоклас дозволяє стилізувати інтерактивний елемент у вимкненому режимі.
- **Псевдоклас `:checked`**: Цей псевдоклас використовується, щоб вказати користувачу, що елемент позначений як вибраний.
- **Псевдоклас `:valid`**: Цей псевдоклас націлює поля введення, які відповідають критеріям валідації.
- **Псевдоклас `:invalid`**: Цей псевдоклас націлює поля введення, які не відповідають критеріям валідації.
- **Псевдокласи `:in-range` і `:out-of-range`**: Ці псевдокласи застосовують стилі до елементів залежно від того, чи їхні значення знаходяться в межах або поза межами заданих діапазонів.
- **Псевдоклас `:required`**: Цей псевдоклас націлює `input` елементи, які мають атрибут `required`. Він сигналізує користувачу, що поле потрібно заповнити для відправлення форми.
- **Псевдоклас `:optional`**: Цей псевдоклас застосовує стилі до input елементів, які не є обов’язковими і можуть залишатися порожніми.
- **Псевдоклас `:autofill`**: Цей псевдоклас застосовує стилі до полів введення, які браузер автоматично заповнює збереженими даними.

## Псевдокласи розташування

- **Псевдокласи розташування**: Ці псевдокласи використовуються для стилізації посилань і елементів, які є ціллю в межах поточного документа.
- **Псевдоклас `:any-link`**: Цей псевдоклас є комбінацією псевдокласів `:link` і `:visited`. Він відповідає будь-якому елементу посилання з атрибутом href, незалежно від того, чи було його відвідано.
- **Псевдоклас `:link`**: Цей псевдоклас дозволяє націлити всі непереглянуті посилання на вебсторінці. Ви можете використати його, щоб стилізувати посилання інакше до того, як користувач натисне на них.
- **Псевдоклас `:local-link`**: Цей псевдоклас націлює посилання, які ведуть до того самого документа. Це корисно, коли потрібно відрізнити внутрішні посилання від зовнішніх.
- **Псевдоклас `:visited`**: Цей псевдоклас націлює посилання, які користувач відвідав.
- **Псевдоклас `:target`**: Цей псевдоклас використовується для застосування стилів до елемента, який є ціллю фрагмента URL.

## Псевдокласи структурного дерева

- **Псевдокласи структурного дерева**: Ці псевдокласи дозволяють націлювати та стилізувати елементи залежно від їх позиції в дереві документа.
- **Псевдоклас `:root`**: Цей псевдоклас зазвичай відповідає кореневому `html` елементу. Він допомагає націлити найвищий рівень у документі, щоб застосувати загальний стиль до всього документа.
- **Псевдоклас `:empty`**: Порожні елементи, тобто елементи без дочірніх, окрім порожнього простору, також включені в дерево документа. Саме тому існує псевдоклас `:empty` для націлювання порожніх елементів.
- **Псевдоклас `:nth-child(n)`**: Цей псевдоклас дозволяє вибирати елементи залежно від їх позиції в межах батьківського елемента.
- **Псевдоклас `:nth-last-child(n)`**: Цей псевдоклас дозволяє вибирати елементи, рахуючи з кінця.
- **Псевдоклас `:first-child`**: Цей псевдоклас вибирає перший елемент у межах батьківського елемента або документа.
- **Псевдоклас `:last-child`**: Цей псевдоклас вибирає останній елемент у межах батьківського елемента або документа.
- **Псевдоклас `:only-child`**: Цей псевдоклас вибирає єдиний елемент у межах батьківського елемента або документа.
- **Псевдоклас `:first-of-type`**: Цей псевдоклас вибирає перше входження певного типу елемента в межах батьківського елемента.
- **Псевдоклас `:last-of-type`**: Цей псевдоклас вибирає останнє входження певного типу елемента в межах батьківського елемента.
- **Псевдоклас `:nth-of-type(n)`**: Цей псевдоклас дозволяє вибрати конкретний елемент у межах батьківського елемента залежно від його позиції серед братських елементів того самого типу.
- **Псевдоклас `:only-of-type`**: Цей псевдоклас вибирає елемент, якщо він єдиний свого типу в межах батьківського елемента.

## Функціональні псевдокласи

- **Функціональні псевдокласи**: Функціональні псевдокласи дозволяють вибирати елементи на основі складніших умов або відносин. На відміну від звичайних псевдокласів, які націлюють елементи за станом (наприклад, `:hover`, `:focus`), функціональні псевдокласи приймають аргументи.
- **Псевдоклас `:is()`**: Цей псевдоклас приймає список селекторів (наприклад, `ol`, `ul`) і вибирає елемент, який відповідає одному з селекторів у списку.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<p class="example">This text will change color.</p>
<p>This text will not change color.</p>
<p>This text will not change color.</p>
<p class="this-works-too">This text will change color.</p>
```

```css
p:is(.example, .this-works-too) {
    color: red;
}
```

:::

- **Псевдоклас `:where()`**: Цей псевдоклас приймає список селекторів (наприклад, `ol`, `ul`) і вибирає елемент, який відповідає одному з селекторів у списку. Різниця між `:is` і `:where` полягає в тому, що останній матиме специфічність 0.

```css
:where(h1, h2, h3) {
    margin: 0;
    padding: 0;
}
```

- **Псевдоклас `:has()`**: Цей псевдоклас часто називають селектором `"parent"`, оскільки він дозволяє стилізувати елементи, які містять дочірні елементи, вказані у списку селекторів.

```css
article:has(h2) {
    border: 2px solid hotpink;
}
```

- **Псевдоклас `:not()`**: Цей псевдоклас використовується для вибору елементів, які не відповідають заданому селектору.

```css
p:not(.example) {
  color: blue;
}
```

## Псевдоелементи

- **Псевдоелемент `::before`**: Цей псевдоелемент використовує властивість `content` для вставлення косметичного вмісту, наприклад іконок, безпосередньо перед елементом.
- **Псевдоелемент `::after`**: Цей псевдоелемент використовує властивість `content` для вставлення косметичного вмісту, наприклад іконок, безпосередньо після елемента.
- **Псевдоелемент `::first-letter`**: Цей псевдоелемент націлює першу літеру вмісту елемента, дозволяючи її стилізувати.
- **Псевдоелемент `::marker`**: Цей псевдоелемент дозволяє вибрати маркер (крапку або нумерацію) елементів списку для стилізації.
