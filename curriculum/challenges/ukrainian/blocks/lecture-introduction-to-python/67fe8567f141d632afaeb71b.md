# --description--

У попередньому уроці ви дізналися, що таке Python і що з ним можна робити. Тепер давайте розглянемо, як можна налаштувати Python на вашому локальному комп’ютері.

Найпростіший спосіб встановити Python на Windows і Mac — це скачати інсталятор з офіційного сайту Python. Також пізніше в цьому уроці ми розглянемо, як запускати Python на Linux.

Перейдіть на `https://www.python.org/` і наведіть курсор на "Downloads". З’явиться модальне вікно з поточною версією Python для вашої операційної системи (ОС).

Спочатку розглянемо, як встановити Python на комп’ютері з macOS:

- Натисніть на кнопку з поточною версією Python (з попереднього модального вікна), і автоматично почнеться завантаження `.pkg` інсталяційного файлу.
- Коли інсталятор `.pkg` завершить завантаження, відкрийте його, потім натисніть "Continue" у вікні, що з’явиться.
- Продовжуйте натискати кнопку "Continue", доки не дійдете до розділу "Installation Type". Там натисніть кнопку "Install".
- За потреби введіть свій пароль, потім почніть встановлення.
- Після цього ви повинні побачити повідомлення про успішне встановлення Python.
- Натисніть кнопку "Close", і все готово!

Ви можете перевірити встановлення, відкривши термінал і виконавши `python --version` або `python3 --version`.

Також можна відкрити інтерпретатор Python, виконавши `python` або `python3` у терміналі.

<dfn>Термінал</dfn> — це текстовий інтерфейс, який дозволяє взаємодіяти з комп’ютером, вводячи команди. Кожна операційна система має свій стандартний застосунок терміналу. На macOS це застосунок Terminal. На Windows — Command Prompt або PowerShell. На Linux у кожного робочого середовища є свій стандартний термінал, наприклад GNOME Terminal або Konsole.

Зверніть увагу, що на деяких старіших системах macOS і Linux `python` може бути зарезервований для Python 2, тоді як `python3` — для Python 3. Якщо ви виконаєте `python --version` і побачите версію Python 2, наприклад `Python 2.7.18`, це означає, що ваша ОС використовує програмне забезпечення, написане на старішій версії Python. У такому разі слід використовувати `python3` для запуску свого Python-коду надалі. Python 2 більше не підтримується і не повинен використовуватися для нової розробки.

Щоб встановити Python на Windows, виконайте такі кроки:

- Перейдіть на `https://www.python.org/` і наведіть курсор на “Downloads“. З’явиться модальне вікно з написом "Download for Windows" і кнопкою завантаження з поточною версією Python.
- Натисніть на номер версії, і автоматично почнеться завантаження виконуваного файлу Windows (`.exe`).
- Коли завантаження інсталятора Python для Windows завершиться, двічі клацніть на ньому і дотримуйтесь інструкцій.
- Коли побачите опцію `Add python.exe to Path`, поставте там прапорець, потім натисніть `Install Now`. Це полегшить вам роботу пізніше.

Ви можете перевірити встановлення, відкривши командний рядок, наприклад PowerShell, і виконавши `python --version`. Також можна відкрити інтерпретатор Python, виконавши `python`.

Для Python на Linux більшість основних дистрибутивів, таких як Ubuntu, Debian і Fedora, вже мають Python.

Просто відкрийте термінал і виконайте `python --version` або `python3 --version`:

Якщо жодна з команд не показує версію Python, ви можете пошукати пакет для встановлення для вашого дистрибутива Linux на `https://www.python.org` або знайти в інтернеті рекомендований спосіб встановлення Python для вашого дистрибутива.
