# --description--

閏年是可以被 `4` 整除的年份，但可被 `100` 整除且不可被 `400` 整除的年份除外。 例如，`2000` 是閏年，但 `1900` 不是。 此外，閏年的二月多一天，即該月的第 29 天。

**目標：** 完成以下使用者故事並通過所有測試以完成實驗。

**使用者故事：**

1. 定義一個名為 `isLeapYear` 的函式，該函式接受一個數字作為引數。
2. 在函式外宣告一個變數 `year`，用來儲存你想要檢查的年份值。
3. 在函式（程式）內，使用 `if/ else` 述語或三元運算子來檢查年份是否為閏年。
4. 要檢查年份是否為閏年，請符合以下條件：

   - 如果年份能被 `4` 整除，則該年為閏年。
   - 除非該年也能被 `100` 整除，否則它不是閏年。
   - 除非該年份也能被 `400` 整除，否則它是閏年。

5. 如果該年是閏年，傳回 `[year] is a leap year.`。否則，傳回 `[year] is not a leap year.`。你將會用在 `isLeapYear` 函式中定義的參數取代 `[year]`。
6. 你應該以 `year` 作為引數呼叫 `isLeapYear` 函式，並將結果指定到名為 `result` 的變數。
7. 你應該使用 `console.log()` 將 `result` 變數輸出到主控台。

function isLeapYear(year) {
  if ((year % 4 === 0 && year % 100 !== 0) || year % 400 === 0) {
    return(year + " is a leap year.");
  } else {
    return(year + " is not a leap year.");
  }
}

const year = 2024;
const result = isLeapYear(year);
console.log(result);
