# --explanation--

**Las fracciones** se usan para describir partes de un todo.

`One-fourth` (¼) es una parte de cuatro partes iguales. Otros ejemplos:

- ½ = `one-half`

- ⅓ = `one-third`

- ¾ = `three-fourths`

En el caso de Alice, una cuarta parte del presupuesto de desarrollo se gasta en desarrollo back-end, lo que muestra cómo se divide el dinero dentro de una categoría.
