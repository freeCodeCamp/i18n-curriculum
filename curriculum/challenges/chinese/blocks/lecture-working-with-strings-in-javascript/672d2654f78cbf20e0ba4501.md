# --interactive--

在 JavaScript 中，字符串被视为字符序列，每个字符都可以通过方括号表示法访问。 这种方法允许你根据字符的位置即索引，来获取字符串中的特定字符。

索引表示字符在字符串中的位置，从零开始的计数。 也就是说：字符串的第一个字符索引为 `0`，第二个字符索引为 `1`，以此类推。

例如，在字符串 `hello` 中，字符 `h` 的索引是 `0`，`e` 的索引是 `1`，`l` 的索引是 `2`，以此类推。

方括号表示法使用方括号（`[]`）和字符索引来访问特定字符。 让我们看一个示例：

:::interactive_editor

```js
let greeting = "hello";
console.log(greeting[1]); // "e"
```

:::

在这个例子中，我们访问索引为 `1` 的字符，即字母 `e`。

要获取字符串的最后一个字符，可以使用字符串长度减一。 字符串的 `length` 属性表示其包含的字符数量，因此访问最后一个字符的索引应为长度减一：

:::interactive_editor

```js
let greeting = "hello";
console.log(greeting[greeting.length - 1]); // "o"
```

:::

这里 `hello` 的 `length` 是 `5`，最后一个字符（`o`）的索引是 `4` 即 `5 - 1`。

如果需要获取多个字符，可以这样使用方括号表示法：

:::interactive_editor

```js
let greeting = "hello";
let firstTwo = greeting[0] + greeting[1]; // "he"
console.log(firstTwo);
```

:::

这个例子中，我们通过方括号表示法拼接前两个字符，形成了字符串 `he`。

方括号表示法在需要访问字符串中特定字符时非常实用，如从姓名中提取首字母或验证特定字符有效性。
