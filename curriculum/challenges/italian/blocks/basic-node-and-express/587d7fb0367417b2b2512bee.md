# --description--

Nelle prime due righe del file `myApp.js` puoi vedere quanto sia facile creare un oggetto app di Express. Questo oggetto ha diversi metodi, e ne imparerai molti in queste sfide. Un metodo fondamentale è `app.listen(port)`. Dice al tuo server di ascoltare su una determinata porta, mettendolo in stato di esecuzione. Per motivi di test, abbiamo bisogno che l’app sia in esecuzione in background, quindi abbiamo aggiunto questo metodo nel file `server.js` per te.

Serviamo la nostra prima stringa! In Express, le rotte hanno la seguente struttura: `app.METHOD(PATH, HANDLER)`. METHOD è un metodo http in minuscolo. PATH è un percorso relativo sul server (può essere una stringa o anche un’espressione regolare). HANDLER è una funzione che Express chiama quando la rotta corrisponde. I gestori hanno la forma `function(req, res) {...}`, dove req è l’oggetto richiesta e res è l’oggetto risposta. Per esempio, il gestore

```js
function(req, res) {
  res.send('Response String');
}
```

servirà la stringa 'Response String'.

# --instructions--

Usa il metodo `app.get()` per servire la stringa "Hello Express" alle richieste GET che corrispondono al percorso `/` (root). Assicurati che il tuo codice funzioni controllando i log.

**Nota:** Tutto il codice per queste lezioni deve essere aggiunto tra le poche righe di codice con cui abbiamo iniziato per te.
