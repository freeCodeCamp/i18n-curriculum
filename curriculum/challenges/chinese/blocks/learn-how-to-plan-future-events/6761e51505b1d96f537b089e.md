# --explanation--

`Going to` 用于谈论对未来的计划行动或意图。例如：

- `We are going to upgrade the system next month。` - 这意味着升级已计划并将很快进行。
  
- `She is going to learn SQL next year.` - 这意味着她打算在未来学习 SQL。

在这个句子中：

`We're` 是 `we are` 的缩写。

`Going to` 表示该动作计划在不久的将来发生。

`Need` 是表示某事必要或必需的基本动词。
