# --description--

レーザー物理学において、「白セル」とはレーザービームの遅延線として機能する鏡のシステムです。ビームはセルに入り、鏡の間を跳ね返りながら進み、最終的に外に出ていきます。

今回考える特定の白セルは、方程式 $4{x}^2 + y^2 = 100$ の楕円です。

上部の $−0.01 ≤ x ≤ +0.01$ に対応する部分が欠けており、光がその穴から入って出ることができます。

この問題のレーザービームは、白セルのすぐ外側の点 (0.0, 10.1) から始まり、最初に鏡に当たる点は (1.4, -9.6) です。

レーザービームが楕円の表面に当たるたびに、通常の反射の法則「入射角は反射角に等しい」に従います。つまり、入射ビームと反射ビームは、入射点での法線と同じ角度を作ります。

左の図では、赤い線がレーザービームと白セルの壁との最初の2つの接触点を示し、青い線は最初の跳ね返りの入射点での楕円の接線を示しています。

与えられた楕円の任意の点 (x, y) における接線の傾き m は次の通りです：$m = −4 × \frac{x}{y}$

法線は入射点でこの接線に垂直です。

右のアニメーションはビームの最初の10回の反射を示しています。

ビームが白セルの内部表面に当たってから外に出るまでに、何回当たるでしょうか？
