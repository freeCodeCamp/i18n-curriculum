# --explanation--

`Notification`는 중요한 내용을 알리기 위해 보내는 메시지나 알림을 뜻합니다. 예를 들어:  

`We received a notification about the upcoming software update.` - 이는 소프트웨어 업데이트에 대해 사용자에게 알림 메시지를 보냈다는 뜻입니다.

`Maintenance window`는 시스템 업데이트, 수리 또는 테스트를 위해 정해진 예정된 시간을 의미합니다. 예를 들어:  

`The website will be down during the maintenance window from 2 AM to 4 AM.` - 이는 지정된 기간 동안 유지 관리 작업이 진행되는 동안 웹사이트를 사용할 수 없다는 뜻입니다.

`Inform`는 정보를 공유하거나 누군가에게 알리는 것을 의미합니다. 예를 들어:  

`The team will inform the users about the new feature rollout.` - 이는 팀이 새로운 기능 도입에 대해 사용자에게 알릴 것이라는 뜻입니다.
