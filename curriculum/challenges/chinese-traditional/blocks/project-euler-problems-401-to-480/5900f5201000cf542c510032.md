# --description--

費波那契數列 $\\{f_n, n ≥ 0\\}$ 以遞迴方式定義為 $f_n = f_{n - 1} + f_{n - 2}$，基底條件為 $f_0 = 0$ 和 $f_1 = 1$。

定義多項式 $\\{F_n, n ≥ 0\\}$ 為 $F_n(x) = \displaystyle\sum_{i = 0}^n f_ix^i$。

例如，$F_7(x) = x + x^2 + 2x^3 + 3x^4 + 5x^5 + 8x^6 + 13x^7$，且 $F_7(11) = 268\\,357\\,683$。

令 $n = {10}^{15}$。求和 $\displaystyle\sum_{x = 0}^{100} F_n(x)$ 並將你的答案對 $1\\,307\\,674\\,368\\,000 \\, (= 15!)$ 取模。
