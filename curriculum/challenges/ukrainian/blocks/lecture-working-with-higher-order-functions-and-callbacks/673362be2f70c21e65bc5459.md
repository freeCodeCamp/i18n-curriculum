# --interactive--

Метод `reduce` — це функція в JavaScript, яка дозволяє обробити масив і звести його до одного значення. Це значення може бути числом, рядком, об’єктом або навіть іншим масивом.

Він називається `reduce`, бо зменшує масив до одного вихідного значення. Хоча спочатку це може здатися складним, розуміння `reduce` значно спростить ваш код у багатьох випадках.

У своїй основі `reduce` працює, застосовуючи функцію до кожного елемента масиву по черзі, передаючи результат кожного обчислення наступному. Цю функцію часто називають функцією-редуктором.

Функція-редуктор приймає два основні параметри: акумулятор і поточне значення. Акумулятор — це місце, де ви зберігаєте поточний результат операцій, а поточне значення — це елемент масиву, який обробляється.

Давайте розглянемо приклад, щоб проілюструвати, як працює `reduce`:

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];
const sum = numbers.reduce(
  (accumulator, currentValue) => accumulator + currentValue,
  0
);

console.log(sum); // 15
```

:::

У цьому прикладі ми використовуємо `reduce`, щоб отримати суму всіх чисел у масиві.

Функція-редуктор бере акумулятор (який починається з `0`, як вказано другим аргументом у `reduce`) і додає до нього кожне число.

Результат кожного додавання стає новим акумулятором для наступної ітерації.

Метод `reduce` також може приймати початкове значення як другий аргумент. Це значення, з якого починається акумулятор. У наведеному вище прикладі ми встановили його на `0`.

Якщо ви не надасте початкового значення, `reduce` використає перший елемент масиву як початковий акумулятор і почне процес з другого елемента.

Одна з переваг `reduce` — це його гнучкість. Оскільки ви визначаєте функцію-редуктор, ви маєте повний контроль над тим, як обробляється масив і який результат хочете отримати. Це робить `reduce` надзвичайно потужним, але спочатку може бути трохи складним для розуміння.

З практикою ви освоїте роботу з методом `reduce`.
