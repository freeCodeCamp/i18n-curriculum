# --description--

이번 작업에서는 프로그래밍 개념을 이해하기 위해 은유적 표현을 탐구합니다. 은유는 복잡한 아이디어를 일상적인 사물이나 행동에 비유하여 시각화하고 설명하는 강력한 방법입니다.

`Container`는 변수를 다양한 데이터를 담고 저장할 수 있는 것으로 은유적으로 설명하는데, 이는 `container`가 여러 가지 물건을 담을 수 있는 것과 비슷합니다.

행동 `fill and empty`는 변수를 어떻게 데이터를 추가하거나 제거할 수 있는지 설명하는데, 이는 물병에 물을 `fill` 하거나 물을 붓는 것과 유사합니다.

`Locked boxes`는 상수를 안전하고 변경할 수 없는 `containers`로 설명하며, 한 번 설정되면 변경할 수 없음을 강조합니다.

`keep the same contents`라는 표현은 상수 안의 데이터가 변하지 않는다는 점을 강조하는데, 이는 잠긴 상자가 항상 같은 내용을 담고 있는 것과 같습니다.
