# --description--

`So far`는 지금까지 또는 지금 이 순간까지를 의미합니다. 과거에 시작해서 현재까지 이어지는 시간을 말할 때 사용합니다. 누군가 `Are you happy with the workplace so far?`라고 묻는다면, 처음부터 오늘까지 직장이 마음에 들었는지 알고 싶다는 뜻입니다.

또 다른 예로, `The project is good so far`라고 말하면 프로젝트 시작부터 지금까지 모든 것이 좋다는 의미입니다.

대화에서 Sarah는 Tom에게 지금까지 새 직장에 만족하는지 묻고 있습니다. 이는 처음부터 대화 시점까지 직장에서의 시간을 묻는 것입니다.
