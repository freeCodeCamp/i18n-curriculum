# --instructions--

このプロジェクトでは、医療検査データを`matplotlib`、`seaborn`、`pandas`を使って可視化し、計算を行います。データセットの値は医療検査時に収集されました。

## データの説明

データセットの行は患者を表し、列は体の測定値、さまざまな血液検査の結果、生活習慣などの情報を表します。このデータセットを使って、心臓病、体の測定値、血液マーカー、生活習慣の関係を調べます。

ファイル名：medical_examination.csv

| 機能 | 変数タイプ | 変数 | 値のタイプ |
|:-------:|:------------:|:-------------:|:----------:|
| 年齢 | 客観的機能 | `age` | int（日数） |
| 身長 | 客観的機能 | `height` | int（cm） |
| 体重 | 客観的機能 | `weight` | float（kg） |
| 性別 | 客観的機能 | `gender` | カテゴリコード |
| 収縮期血圧 | 検査機能 | `ap_hi` | int |
| 拡張期血圧 | 検査機能 | `ap_lo` | int |
| コレステロール | 検査機能 | `cholesterol` | 1:正常、2:正常より高い、3:かなり高い |
| グルコース | 検査機能 | `gluc` | 1:正常、2:正常より高い、3:かなり高い |
| 喫煙 | 主観的機能 | `smoke` | バイナリ |
| 飲酒 | 主観的機能 | `alco` | バイナリ |
| 運動習慣 | 主観的機能 | `active` | バイナリ |
| 心血管疾患の有無 | 目的変数 | `cardio` | バイナリ |


## 指示

`examples/Figure_1.png`に似たチャートを作成してください。ここでは、`cholesterol`と`gluc`の患者について、`alco`、`active`、`smoke`、`cardio=1`、`cardio=0`の変数の良い結果と悪い結果の数を異なるパネルで示します。

`medical_data_visualizer.py`ファイルの各番号に対して、以下の指示番号に対応するコードを追加してください。

1. `medical_examination.csv`からデータをインポートし、`df`変数に代入してください。
2. データに`overweight`列を追加してください。人が過体重かどうかを判断するには、まず体重（kg）を身長（m）の二乗で割ってBMIを計算します。その値が25を超える場合は過体重です。過体重でない場合は`0`の値を、過体重の場合は`1`の値を使ってください。
3. `0`を常に良い、`1`を常に悪いとするようにデータを正規化してください。`cholesterol`または`gluc`の値が1の場合は`0`に設定し、値が`1`より大きい場合は`1`に設定してください。
4. `draw_cat_plot`関数でカテゴリカルプロットを描画してください。
5. `pd.melt`変数に、`cholesterol`、`gluc`、`smoke`、`alco`、`active`、`overweight`の値を使って`df_cat`のデータフレームを作成してください。
6. `df_cat`のデータを`cardio`で分割してグループ化および再フォーマットしてください。各機能の数を表示します。`catplot`が正しく動作するように列の1つの名前を変更する必要があります。
7. データを`long`形式に変換し、seabornライブラリの`sns.catplot()`メソッドを使ってカテゴリカル機能の値の数を示すチャートを作成してください。
8. 出力の図を取得し、`fig`変数に保存してください。
9. 次の2行は変更しないでください。
10. `draw_heat_map`関数でヒートマップを描画してください。
11. `df_heat`変数のデータを以下の誤ったデータを表す患者セグメントを除外してクリーンアップしてください：
    - 拡張期血圧が収縮期血圧より高い（`(df['ap_lo'] <= df['ap_hi'])`で正しいデータを保持）
    - 身長が2.5パーセンタイル未満（`(df['height'] >= df['height'].quantile(0.025))`で正しいデータを保持）
    - 身長が97.5パーセンタイルより大きい
    - 体重が2.5パーセンタイル未満
    - 体重が97.5パーセンタイルより大きい
12. 相関行列を計算し、`corr`変数に保存してください。
13. 上三角行列のマスクを生成し、`mask`変数に保存してください。
14. `matplotlib`の図を設定してください。
15. `seaborn`ライブラリの`sns.heatmap()`メソッドを使って相関行列をプロットしてください。
16. 次の2行は変更しないでください。
