# --explanation--

`Need to be careful` は、ミスや問題を避けるために注意を払うことを意味します。これは `need to care` とは異なります。`need to care` は懸念や注意が必要であることを意味しますが、`need to be careful` は人が慎重に行動すべきことを意味します。注意を払う行為自体よりも、その人に重点が置かれています。例えば：  

- `We need to have patience while waiting for results.` - ここでは、我慢する行為が重要です。

- `We need to be patient while waiting for results.` - ここでは、我慢すべき人々、つまり `we` に重点が置かれています。
