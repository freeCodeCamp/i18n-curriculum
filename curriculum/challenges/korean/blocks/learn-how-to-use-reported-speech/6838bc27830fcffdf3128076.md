# --explanation--

`Exactly`는 누군가가 한 말에 완전히 동의하거나 옳다는 것을 나타내는 부사입니다. 반면 `Exact`는 세부적으로 정확하거나 올바르다는 뜻의 형용사입니다. 예를 들어:

- `You are exactly right about the cause of the problem.` – 이것은 완전한 동의를 나타냅니다. 그 사람이 얼마나 옳았는지(형용사)를 표현합니다.

- `We need the exact measurements to build the new server rack.` – 이것은 매우 정밀한 측정을 의미합니다. 동사나 형용사 대신 명사를 수식합니다.
