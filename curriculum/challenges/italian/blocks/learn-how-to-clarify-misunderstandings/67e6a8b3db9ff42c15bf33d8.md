# --explanation--

`Not just ... but` è una struttura comune usata per mostrare contrasto o aggiungere enfasi. Significa che qualcosa è più di una cosa sola. La seconda parte aggiunge più informazioni o importanza. Per esempio:

`Debugging is not just about finding errors, but about understanding the code better.` - Questo significa che il debugging è più che correggere errori; aiuta anche a comprendere più a fondo.
