# --description--

整数 $a$ と $b$ に対して、$D(a, b)$ を放物線 $y = x^2$ と直線 $y = ax + b$ に囲まれた領域として定義します：$D(a, b) = \\{ (x, y) | x^2 ≤ y ≤ ax + b \\}$。

$L(a, b)$ は $D(a, b)$ に含まれる格子点の数と定義します。例えば、$L(1, 2) = 8$、$L(2, -1) = 1$ です。

また、$S(N)$ を、$D(a, b)$ のエリアが有理数であり、かつ $|a|,|b| ≤ N$ を満たすすべての組 ($a$, $b$) に対する $L(a, b)$ の総和と定義します。

$S(5) = 344$、$S(100) = 26\\,709\\,528$ であることが確認できます。

$S({10}^{12})$ を求めてください。答えは $\bmod {10}^8$ で提出してください。
