# --explanation--

`Wouldn't` ist eine Kontraktion von `would not`, die verwendet wird, um eine Frage zu bilden, oft um Zweifel auszudrücken oder nach etwas zu fragen. Zum Beispiel:

`The system should be secure, but wouldn't it be safer with extra testing?` – In diesem Fall hinterfragt die sprechende Person, ob mehr Tests nötig sind, um das System sicherer zu machen.

`Vulnerabilities` bezeichnen Schwachstellen in einem System, einer Software oder einem Netzwerk, die von Angreifern ausgenutzt werden können. In diesem Zusammenhang. Zum Beispiel:

`The app had several vulnerabilities that hackers could exploit.` – Hier bezieht es sich auf die Schwachstellen in der App, die Angreifern Schaden zufügen könnten.
