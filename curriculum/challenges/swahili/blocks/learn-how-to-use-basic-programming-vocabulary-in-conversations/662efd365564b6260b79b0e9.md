# --description--

`loop` ni mfululizo wa maagizo unaorudiwa mara kwa mara hadi hali fulani itakapofikiwa. `Loops` hutumika kuendesha msimbo uleule mara nyingi kwa ufanisi. Aina za kawaida za `loops` ni mizunguko ya `for` na mizunguko ya `while`. 

Kwa mfano, `for loop`: `for (int i = 0; i < 5; i++) { /* code to be executed */ }`

Katika mfano huo, msimbo ulioko ndani ya mabano ya `{ }` unatekelezwa tena na tena hadi hali isipotakuwa kweli tena.
