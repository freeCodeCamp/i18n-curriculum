### --feedback--

Esta palavra descreve consultas que usam valores predefinidos para melhorar a segurança.É uma palavra longa frequentemente usada em contextos de banco de dados.

# --explanation--

`Parameterized` vem da palavra raiz `parameter`, que é um valor ou espaço reservado usado para controlar como algo funciona.  

Quando as pessoas dizem que uma consulta é `parameterized`, isso significa que ela usa valores predefinidos (parâmetros) em vez de inserir a entrada do usuário diretamente. 

Isso ajuda a melhorar a segurança do banco de dados prevenindo certos ataques como SQL injection.Por exemplo:  

- `The developer used parameterized queries to secure the database.` - Isso significa que eles usaram espaços reservados para garantir que nenhum código malicioso pudesse ser inserido.
