# --explanation--

`To set goals`는 달성할 구체적인 목표나 대상을 정의하는 것을 의미합니다. 예를 들어:

`The team set goals to improve customer satisfaction this year.` - 이는 팀이 고객 만족과 관련된 목표를 설정했다는 뜻입니다.

`Quarter`는 1년의 4분의 1, 즉 3개월 기간을 의미합니다. 비즈니스 맥락에서 `the next quarter`는 달력 연도나 회계 연도에서 다가오는 3개월 기간을 뜻합니다. 예를 들어:

- `Our sales target for the next quarter is $1 million.` - 이는 목표가 다음 3개월 기간에 적용된다는 뜻입니다.

- `We review our performance at the end of each quarter.` - 이는 3개월마다 진행 상황을 평가하는 것을 의미합니다.
