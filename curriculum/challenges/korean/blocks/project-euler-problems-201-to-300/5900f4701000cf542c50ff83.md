# --description--

세 개의 돌 더미와 두 명의 플레이어가 참여하는 게임입니다.

각 플레이어의 차례에, 플레이어는 더미에서 하나 이상의 돌을 제거합니다. 단, 여러 더미에서 돌을 가져갈 경우, 선택한 각 더미에서 같은 수의 돌을 제거해야 합니다.

즉, 플레이어는 어떤 $N > 0$을 선택하고 다음 중 하나를 수행합니다:

- 한 더미에서 $N$개의 돌을 제거하거나;
- 두 더미 각각에서 $N$개의 돌을 제거하여 총 $2N$개의 돌을 제거하거나;
- 세 더미 각각에서 $N$개의 돌을 제거하여 총 $3N$개의 돌을 제거합니다.

마지막 돌을 가져가는 플레이어가 게임에서 승리합니다.

첫 번째 플레이어가 반드시 이길 수 있는 상태를 승리 상태라고 합니다.

예를 들어, (0,0,13), (0,11,11), (5,5,5)는 첫 번째 플레이어가 즉시 모든 돌을 제거할 수 있으므로 승리 상태입니다.

두 번째 플레이어가 어떤 첫 번째 플레이어의 수에도 반드시 이길 수 있는 상태를 패배 상태라고 합니다.

예를 들어, (0,1,2)와 (1,3,3)은 어떤 합법적인 수를 해도 두 번째 플레이어에게 승리 상태를 남기므로 패배 상태입니다.

$x_i ≤ y_i ≤ z_i ≤ 100$인 모든 패배 상태 ($x_i$,$y_i$,$z_i$)를 고려하면, 이들의 합 $\sum (x_i + y_i + z_i) = 173\,895$임을 확인할 수 있습니다.

$x_i ≤ y_i ≤ z_i ≤ 1000$인 패배 상태 ($x_i$,$y_i$,$z_i$)에 대해 $\sum (x_i + y_i + z_i)$를 구하세요.
