# --description--

`to debug`라는 용어는 프로그래밍에서 매우 중요합니다. 이는 컴퓨터 프로그램에서 오류나 버그를 찾아 고치는 것을 의미합니다. `bug`는 프로그램이 제대로 작동하지 못하게 하는 실수나 문제입니다.

`Debugging`는 프로그래머의 일상적인 업무입니다.

예를 들어, `She spent the whole day debugging the code.`는 그녀가 코드의 실수를 고치고 있었다는 뜻입니다.
