# --description--

In den letzten beiden Aufgaben haben wir den Gleichheitsoperator (`==`) und den strikten Gleichheitsoperator (`===`) kennengelernt. Lassen Sie uns eine kurze Wiederholung machen und die Verwendung dieser Operatoren weiter üben.

Wenn die zu vergleichenden Werte nicht vom gleichen Typ sind, führt der Gleichheitsoperator eine Typumwandlung durch und bewertet dann die Werte. Der strikte Gleichheitsoperator hingegen vergleicht sowohl den Datentyp als auch den Wert unverändert, ohne einen Typ in den anderen umzuwandeln.

**Beispiele**

`3 == '3'` gibt `true` zurück, weil JavaScript eine Typumwandlung von String zu Zahl durchführt. `3 === '3'` gibt `false` zurück, weil die Typen unterschiedlich sind und keine Typumwandlung erfolgt.

**Hinweis:** In JavaScript können Sie den Typ einer Variablen oder eines Werts mit dem `typeof`-Operator bestimmen, wie folgt:

```js
typeof 3
typeof '3'
```

`typeof 3` gibt den String `number` zurück, und `typeof '3'` gibt den String `string` zurück.

# --instructions--

Die `compareEquality`-Funktion im Editor vergleicht zwei Werte mit dem Gleichheitsoperator. Ändern Sie die Funktion so, dass sie den String `Equal` nur dann zurückgibt, wenn die Werte strikt gleich sind.
