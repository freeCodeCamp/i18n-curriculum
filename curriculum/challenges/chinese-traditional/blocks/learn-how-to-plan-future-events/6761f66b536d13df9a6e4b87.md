# --explanation--

`Parameterized` 來自根字 `parameter`，它是一個用來控制某事如何運作的值或佔位符。  

當人們說查詢是 `parameterized`，表示它使用預先定義的值（參數），而不是直接插入使用者輸入。 

這有助於透過防止某些攻擊（例如 SQL 注入）來提升資料庫安全性。例如：  

- `The developer used parameterized queries to secure the database。`－這表示他們使用了佔位符以確保不會插入有害的程式碼。
