# --description--

Was tun Sie, wenn Sie zwei Gruppen von Elementen haben, die einige ihrer Stil-Deklarationen teilen?

```css
.read {
  color: white;
  background-color: black;
  /* several unique declarations */
}

.unread {
  color: white;
  background-color: black;
  /* several unique declarations */
}
```

Sowohl unsere `.read` als auch `.unread` Selektoren teilen die `color: white;` und `background-color: black;` Deklarationen, haben aber ansonsten jeweils mehrere eigene Deklarationen. Um die Wiederholung zu reduzieren, können Sie diese beiden Selektoren als durch Kommas getrennte Liste zusammenfassen:

```css
.read,
.unread {
  color: white;
  background-color: black;
}

.read {
  /* several unique declarations */
}

.unread {
  /* several unique declarations */
}
```

Beide obigen Beispiele (mit und ohne Gruppierung) führen zum gleichen Ergebnis, aber das zweite Beispiel verringert die Wiederholung der Deklarationen und erleichtert es, entweder die `color` oder `background-color` für beide Klassen gleichzeitig zu bearbeiten.
