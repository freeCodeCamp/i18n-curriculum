# --interactive--

CSS 中的 `!important` 关键字用于赋予样式规则最高优先级，使其能够覆写该属性的任何其他声明。

使用时，它会强制浏览器应用指定的样式，而不考虑其他选择器的优先级。

假设你有一个带有内联样式的超文本标记语言段落元素，如下所示：

:::interactive_editor

```html
<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

:::

在此示例中，段落元素的背景颜色设置为 `lightblue`，文本颜色设置为 `black`。

CSS 文件将以下样式应用于段落元素：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black;
  color: white;
}
```

:::

由于内联样式的优先级高于类、ID 和类型选择器，黑色背景色和白色文本色将不会应用到该段落元素。

要覆写那些内联样式，你可以像这样使用 `!important` 关键字：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black !important;
  color: white !important;
}
```

:::

`!important` 关键字用于 CSS 值之后和分号之前。

现在段落元素将应用那些黑白颜色，因为内联样式被使用 `!important` 关键字覆盖。

CSS 中的 `!important` 关键字用于赋予样式规则最高优先级，有效地覆盖其他声明，包括具有更高特异性和内联样式的声明。

但是，`!important` 关键字不会改变 CSS 选择器本身的特异性。它只是确保带有 `!important` 的规则被应用，即使存在其他具有更高特异性的冲突规则。

`!important` 关键字的另一个适用分支是当你无法控制原始 CSS 时，覆写第三方库或框架的样式。

然而，过度使用 `!important` 关键字会导致维护和调试你的 CSS 变得困难，因为它破坏了样式的自然层叠，可能导致意想不到的后果。

因此，最好少用 `!important` 关键字。
