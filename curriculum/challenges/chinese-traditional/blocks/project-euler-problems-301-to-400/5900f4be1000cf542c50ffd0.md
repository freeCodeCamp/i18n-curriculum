# --description--

設 $\\{a_1, a_2, \ldots, a_n\\}$ 為長度為 $n$ 的整數序列，使得：

- $a_1 = 6$
- 對所有 $1 ≤ i &lt; n$ ：$φ(a_i) &lt; φ(a_{i + 1}) &lt; a_i &lt; a_{i + 1}$

$φ$ 表示歐拉函數（Euler's totient function）。

令 $S(N)$ 為滿足 $a_n ≤ N$ 的此類序列數量。

例如，$S(10) = 4$：{6}、{6, 8}、{6, 8, 9} 和 {6, 10}。

我們可以驗證 $S(100) = 482\\,073\\,668$ 且 $S(10\\,000)\bmod {10}^8 = 73\\,808\\,307$。

尋找 $S(20\,000\,000)\bmod {10}^8$。
