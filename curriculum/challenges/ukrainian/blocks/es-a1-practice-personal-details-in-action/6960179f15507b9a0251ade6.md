# --explanation--

Естебан запитує `¿Puedes repetirlo?`. Це питання просить когось сказати щось знову.

Воно відповідає загальному ввічливому шаблону запиту:

`¿Puedes` + інфінітив + `...?`

- `Puedes` — це `tú` форма `poder`.
- `repetir` означає сказати щось знову.
- `lo` стосується вже згаданої речі.

Разом питання запитує, чи здатен слухач повторити інформацію.
