# --description--

Para cualquier entero positivo $n$, la $n$-ésima secuencia débil de Goodstein $\\{g1, g2, g3, \ldots\\}$ se define como:

- $g_1 = n$
- para $k > 1$, $g_k$ se obtiene escribiendo $g_{k - 1}$ en base $k$, interpretándolo como un número en base $k + 1$ y restando 1.

La secuencia termina cuando $g_k$ se vuelve 0.

Por ejemplo, la $6$ª secuencia débil de Goodstein es $\\{6, 11, 17, 25, \ldots\\}$:

- $g_1 = 6$.
- $g_2 = 11$ ya que $6 = 110_2$, $110_3 = 12$, y $12 - 1 = 11$.
- $g_3 = 17$ ya que $11 = 102_3$, $102_4 = 18$, y $18 - 1 = 17$.
- $g_4 = 25$ ya que $17 = 101_4$, $101_5 = 26$, y $26 - 1 = 25$.

y así sucesivamente.

Se puede demostrar que toda secuencia débil de Goodstein termina.

Sea $G(n)$ el número de elementos distintos de cero en la $n$-ésima secuencia débil de Goodstein.

Se puede verificar que $G(2) = 3$, $G(4) = 21$ y $G(6) = 381$.

También se puede verificar que $\sum G(n) = 2517$ para $1 ≤ n &lt; 8$.

Encuentra los últimos 9 dígitos de $\sum G(n)$ para $1 ≤ n &lt; 16$.
