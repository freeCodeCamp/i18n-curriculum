# --interactive--

**ПРИМІТКА**: Деякі інтерактивні приклади можуть використовувати CSS, який ви ще не вивчали. Не хвилюйтеся, якщо не розумієте весь код. Мета прикладів — показати вам попередній перегляд цих концепцій дизайну, щоб ви краще зрозуміли, як усе працює.

Додавання зображень на ваші вебсайти — це чудовий спосіб зацікавити користувачів і підвищити візуальну привабливість сайту. Але є кілька речей, які варто враховувати при роботі із зображеннями у ваших дизайнах.

Перше, що потрібно врахувати — це створення адаптивних зображень. Адаптивні зображення — це зображення, які масштабуються відповідно до розміру екрану, на якому їх переглядають. Це важливо, бо гарантує, що ваші зображення виглядають добре на всіх пристроях — від десктопів до мобільних телефонів.

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    padding: 20px;
    background-color: #fefefe;
    color: #333;
    text-align: center;
  }

  img {
    max-width: 100%;
    height: auto;
    border-radius: 8px;
  }

  p {
    font-size: 16px;
    max-width: 600px;
    margin: 20px auto;
    line-height: 1.6;
  }
</style>

<h1>Responsive Cat Image</h1>

<img 
  src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" 
  alt="Two cats peacefully sleeping together."
/>

<p>
  This image automatically scales based on the screen size. Whether you're viewing on a desktop or a mobile phone,
  it adjusts its size without losing proportions, making the design clean and user-friendly on all devices.
</p>
```

:::

Ще одна річ, яку варто врахувати — це роздільність зображень. Зображення вищої якості з кращою роздільністю мають більше пікселів на дюйм. Пікселі — це маленькі квадратики, з яких складається зображення.

Пікселів на дюйм, або PPI, — це кількість пікселів в одному дюймі зображення. Чим вищий PPI, тим краща якість зображення.

Вам потрібно переконатися, що ваші зображення мають високу якість і добре виглядають на всіх пристроях. Це означає, що слід використовувати зображення з високою роздільністю, оптимізовані для вебу.

Ще один момент — це розмір ваших зображень і те, як вони вписуються у простір макету. Потрібно переконатися, що ваші зображення мають правильний розмір і не надто великі чи надто малі.

Використання великих зображень, призначених для менших просторів у дизайні, може уповільнити роботу вашого сайту і ускладнити завантаження для користувачів. Тож переконайтеся, що ваші зображення мають правильний розмір і оптимізовані для вебу.

Щодо розміщення зображень, подумайте про баланс, ієрархію та вирівнювання, щоб забезпечити оптимізацію зображень для вебу.

Баланс — це розподіл візуальної ваги у дизайні. Потрібно переконатися, що між текстом і зображеннями на сайті є хороший баланс, щоб створити гармонійний дизайн.

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    margin: 0;
    padding: 40px 20px;
    background-color: #f9f9f9;
    color: #333;
  }

  .container {
    display: flex;
    flex-wrap: wrap;
    align-items: center;
    justify-content: space-between;
    gap: 30px;
    max-width: 1000px;
    margin: 0 auto;
  }

  .text {
    flex: 1 1 400px;
  }

  .text h2 {
    font-size: 28px;
    margin-bottom: 10px;
  }

  .text p {
    font-size: 16px;
    line-height: 1.6;
  }

  .image {
    flex: 1 1 400px;
  }

  .image img {
    width: 100%;
    height: auto;
    border-radius: 8px;
  }
</style>

<div class="container">
  <div class="text">
    <h2>Balanced Layout</h2>
    <p>
      Balance is essential in web design. By evenly distributing visual weight—such as pairing this block of text
      with a complementary image—you create a layout that feels calm, structured, and easy to navigate.
    </p>
  </div>

  <div class="image">
    <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" alt="Two cats peacefully sleeping together.">
  </div>
</div>
```

:::

Ієрархія — це порядок, у якому елементи переглядаються на сторінці. Потрібно переконатися, що зображення, які відповідають важливому вмісту, розміщені вище за менш важливі.

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    padding: 30px 20px;
    background-color: #fff;
    color: #222;
    max-width: 800px;
    margin: 0 auto;
  }

  .section {
    margin-bottom: 40px;
  }

  .section img {
    width: 100%;
    height: auto;
    border-radius: 8px;
    margin-bottom: 15px;
  }

  .section h2 {
    font-size: 24px;
    margin-bottom: 10px;
  }

  .section p {
    font-size: 16px;
    line-height: 1.6;
  }
</style>

<div class="section">
  <img 
    src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" 
    alt="Two cats peacefully sleeping together."
  >
  <h2>Our Feline Rescue Mission</h2>
  <p>
    This image supports one of our most important stories — the rescue of 12 stray cats from a storm drain. 
    Placing it at the top ensures visitors connect emotionally with our mission right away.
  </p>
</div>

<div class="section">
  <h2>Upcoming Fundraiser</h2>
  <p>
    Join us for a small fundraising event to support our shelter. It’ll be a fun, casual afternoon with 
    snacks, games, and of course — plenty of cats!
  </p>
  <img 
    src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" 
    alt="Two cats peacefully sleeping together."
  >
</div>
```

:::

Вирівнювання — це розміщення елементів відносно один одного. Потрібно переконатися, що ваші зображення вирівняні з текстом та іншими елементами на сайті, щоб створити цілісний дизайн.

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    padding: 40px 20px;
    background-color: #ffffff;
    color: #222;
    max-width: 900px;
    margin: 0 auto;
  }

  .aligned-content {
    display: flex;
    flex-wrap: wrap;
    align-items: center;
    gap: 20px;
  }

  .aligned-content img {
    width: 300px;
    height: auto;
    border-radius: 8px;
    flex-shrink: 0;
  }

  .aligned-text {
    flex: 1;
    min-width: 250px;
  }

  .aligned-text h2 {
    font-size: 24px;
    margin-bottom: 10px;
  }

  .aligned-text p {
    font-size: 16px;
    line-height: 1.6;
  }
</style>

<div class="aligned-content">
  <img 
    src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" 
    alt="Two cats peacefully sleeping together."
  >

  <div class="aligned-text">
    <h2>Aligned Elements</h2>
    <p>
      In this example, the image is aligned horizontally next to the text block. This creates a visually cohesive layout, 
      where both elements feel like they belong together. Proper alignment like this improves readability and reinforces 
      structure in your design.
    </p>
  </div>
</div>
```

:::

Останнє, що варто врахувати — це доступність зображень. Потрібно переконатися, що ваші зображення доступні для всіх користувачів, включно з тими, хто має порушення зору. Це означає, що слід використовувати alt-текст для зображень, щоб читачі екрана могли озвучувати текст користувачам із порушеннями зору.
