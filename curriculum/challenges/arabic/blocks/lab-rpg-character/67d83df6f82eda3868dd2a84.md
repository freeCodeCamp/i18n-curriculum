# --description--

في هذا المختبر ستتمرن على أساسيات Python ببناء تطبيق صغير ينشئ شخصية لمغامرة لعبة تقمص أدوار.

**الهدف:** نفذ قصص المستخدم أدناه واجعل جميع الاختبارات تنجح لإكمال المختبر.

**قصص المستخدم:**

1. يجب أن يكون لديك دالة باسم `create_character`.
1. يجب أن تقبل الدالة، بالترتيب، اسم الشخصية متبوعًا بثلاث إحصائيات: القوة، الذكاء، والكاريزما.
1. يجب التحقق من صحة اسم الشخصية:
   - إذا لم يكن اسم الشخصية سلسلة نصية، يجب أن تُرجع الدالة `The character name should be a string`.
   - إذا كان اسم الشخصية سلسلة فارغة، يجب أن تُرجع الدالة `The character should have a name`.
   - إذا كان اسم الشخصية أطول من 10 أحرف، يجب أن تُرجع الدالة `The character name is too long`.
   - إذا احتوى اسم الشخصية على مسافات، يجب أن تُرجع الدالة `The character name should not contain spaces`.
1. يجب التحقق من صحة الإحصائيات أيضًا:
   - إذا كانت إحدى الإحصائيات أو أكثر ليست أعداد صحيحة، يجب أن تُرجع الدالة `All stats should be integers`.
   - إذا كانت إحدى الإحصائيات أو أكثر أقل من 1، يجب أن تُرجع الدالة `All stats should be no less than 1`.
   - إذا كانت إحدى الإحصائيات أو أكثر أكثر من 4، يجب أن تُرجع الدالة `All stats should be no more than 4`.
   - إذا كان مجموع كل الإحصائيات مختلفًا عن 7، يجب أن تُرجع الدالة `The character should start with 7 points`.
1. إذا اجتازت كل القيم التحقق، يجب أن تُرجع الدالة سلسلة نصية من أربعة أسطر:
   - السطر الأول يجب أن يحتوي على اسم الشخصية
   - الأسطر من 2 إلى 4 يجب أن تبدأ باختصار الإحصائية، `STR`، `INT` أو `CHA` (بهذا الترتيب)، ثم مسافة، ثم عدد من النقاط الممتلئة (`●`) يساوي قيمة الإحصائية، وعدد من النقاط الفارغة (`○`) ليصل المجموع إلى 10. مثال: إذا كانت قيمة القوة 3 يجب أن تكون هناك 3 نقاط ممتلئة تليها 7 نقاط فارغة. النقاط موجودة في المحرر.

إليك السلسلة التي يجب أن تُرجعها الدالة `create_character('ren', 4, 2, 1)`:

   ```md
   ren
   STR ●●●●○○○○○○
   INT ●●○○○○○○○○
   CHA ●○○○○○○○○○
   ```

ملاحظة: رغم أن `str` و `int` اختصارات شائعة للإحصائيات، تذكر أنها كلمات مفتاحية محجوزة في Python ولا يجب استخدامها كأسماء متغيرات.

```py
full_dot = '●'
empty_dot = '○'

def create_character(character_name, strength, intelligence, charisma):
    if type(character_name) != str:
        return 'The character name should be a string'
    if len(character_name) == 0:
        return 'The character should have a name'
    if len(character_name) > 10:
        return 'The character name is too long'
    if ' ' in character_name:
        return 'The character name should not contain spaces'

    if type(strength) != int or type(intelligence) != int or type(charisma) != int:
       return 'All stats should be integers'
    if strength < 1 or intelligence < 1 or charisma < 1:
        return 'All stats should be no less than 1'
    if strength > 4 or intelligence > 4 or charisma > 4:
        return 'All stats should be no more than 4'
    if strength + intelligence + charisma != 7:
        return 'The character should start with 7 points'

    character_sheet = f'{character_name}\n'
    character_sheet += f'STR {full_dot * strength}{empty_dot * (10 - strength)}\n'
    character_sheet += f'INT {full_dot * intelligence}{empty_dot * (10 - intelligence)}\n'
    character_sheet += f'CHA {full_dot * charisma}{empty_dot * (10 - charisma)}'
    return character_sheet
```
