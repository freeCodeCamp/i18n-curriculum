# --description--

`infographic`는 차트나 도표 같은 시각적 이미지로 정보를 나타내는 데 사용됩니다. 예를 들어:

`The infographic shows the survey results clearly.` - 여기서 `infographic`는 데이터를 쉽게 이해할 수 있도록 설명하는 그림을 의미합니다.

`interactive infographic`는 시청자가 클릭하거나 마우스를 올리거나 스크롤하여 정보를 탐색할 수 있는 유형의 인포그래픽입니다. 예를 들어:

`The company made an interactive infographic where readers can click to see details about each product.` - 여기서 `interactive infographic`는 정보를 보여줄 뿐만 아니라 독자가 상호작용할 수 있게 하는 시각 자료를 의미합니다.
