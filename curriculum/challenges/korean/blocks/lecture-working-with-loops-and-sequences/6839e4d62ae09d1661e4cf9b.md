# --description--

지난 수업들에서 다음과 같은 반복문을 다루는 데 익숙해졌을 것입니다:

```py
even_numbers = []

for num in range(21):
    if num % 2 == 0:
        even_numbers.append(num)

print(even_numbers)
```

이 예제는 `even_numbers`라는 새 빈 리스트를 만들고 `0`부터 `20` 사이의 숫자 수열을 반복합니다. 반복문 안에는 현재 숫자를 `0`로 나누었을 때 나머지가 `2`인지 확인하는 조건문이 있습니다. 이 조건은 숫자가 짝수인지 판단하는 데 사용됩니다. 조건이 `True`일 경우 현재 `num`가 `even_numbers` 리스트 끝에 추가됩니다. 마지막으로 `even_numbers` 리스트를 콘솔에 출력합니다.

이 코드는 작동하지만, 리스트 내포를 사용하면 더 간결하게 작성할 수 있습니다. 리스트 내포는 대괄호 안에 반복문과 조건문을 직접 결합하여 한 줄로 새 리스트를 만들 수 있게 해줍니다. 이렇게 하면 코드가 더 짧아지고 읽기 쉬워집니다.

아래는 앞서 예제를 대괄호를 사용해 리팩터링한 코드입니다:

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

이 리팩터링된 예제에서는 `even_numbers` 리스트가 한 줄의 코드로 생성됩니다. 리스트 내포는 `0`부터 `20`까지 숫자를 반복하며 `2`로 나누어 떨어지는 숫자만 포함합니다. 이 방법은 더 간결하며 별도의 반복문과 조건문 블록이 필요 없습니다.

리스트 내포가 어떻게 작동하는지 더 잘 이해하기 위해 또 다른 예제를 살펴보겠습니다:

```py
numbers = [1, 2, 3, 4, 5]
result = [(num, 'Even') if num % 2 == 0 else (num, 'Odd') for num in numbers]
print(result)
```

이 예제에서는 숫자 리스트가 있고, 짝수인지 홀수인지 표시하는 튜플로 새 리스트를 만들고자 합니다. 리스트 내포의 첫 부분에서는 `if` 문을 사용해 숫자가 `2`로 나누어 떨어지는지 확인합니다. 그렇다면 결과는 해당 숫자와 `Even`라는 단어가 들어간 튜플입니다. 그렇지 않으면 숫자와 `Odd`라는 단어가 들어간 튜플이 결과가 됩니다.

콘솔에 출력된 결과는 다음과 같습니다:

```py
[(1, 'Odd'), (2, 'Even'), (3, 'Odd'), (4, 'Even'), (5, 'Odd')]
```

기존 반복 가능한 객체에서 새 리스트를 만드는 또 다른 방법은 `filter()` 함수입니다. 다음은 네 글자 이상인 단어만 포함하는 새 리스트를 만드는 예제입니다:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

`filter()` 함수는 특정 조건을 만족하는 반복 가능한 객체의 요소를 선택하는 데 사용됩니다. `filter()` 함수는 함수와 반복 가능한 객체를 인수로 받습니다. 이 예제에서는 `is_long_word` 함수를 `filter()` 함수에 전달해 현재 단어 길이가 `4`보다 큰지 확인합니다. 문자 수가 `4`보다 큰 모든 단어가 새 리스트에 추가되어 `long_words` 변수에 할당됩니다.

`filter()` 함수 외에도 리스트 작업에 유용한 몇 가지 함수가 더 있습니다. 또 알아둘 함수는 `map()` 함수로, 반복 가능한 객체의 각 요소에 함수를 적용합니다. 다음은 `map()` 함수를 사용해 섭씨 온도를 화씨로 변환하는 예제입니다:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

`filter()` 함수와 마찬가지로 `map()` 함수도 함수와 반복 가능한 객체를 인수로 받습니다. `to_fahrenheit` 함수는 온도를 받아 섭씨에서 화씨로 변환합니다.

마지막으로 살펴볼 함수는 `sum()` 함수입니다. 이 함수는 리스트나 튜플 같은 반복 가능한 객체의 합계를 구하는 데 사용됩니다. 다음은 `sum()` 함수를 사용하는 예제입니다:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

선택적으로 `start` 인수를 전달해 합계의 초기값을 설정할 수도 있습니다. 다음은 위치 인수로 `start` 인수를 사용하는 업데이트된 예제입니다:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

또는 다음과 같이 키워드 인수로 `start` 인수를 사용할 수도 있습니다:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

두 버전 모두 같은 결과를 내지만, 키워드 인수가 좀 더 명확합니다.

리스트 내포뿐 아니라 `map()`, `filter()`, `sum()` 같은 다른 함수들도 처음에는 다소 혼란스러울 수 있습니다. 하지만 충분한 연습과 시간이 지나면 Python 프로그램에서 이들을 더 편하게 사용할 수 있게 될 것입니다.
