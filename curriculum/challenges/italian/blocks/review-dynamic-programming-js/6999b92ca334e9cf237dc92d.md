## Introduzione alla programmazione dinamica

- **Definizione**: La programmazione dinamica è una tecnica algoritmica che risolve problemi complessi suddividendoli in sottoproblemi più semplici e memorizzando i risultati per evitare calcoli ridondanti. 
- **Sottoproblemi sovrapposti**: Gli stessi sottoproblemi più piccoli si presentano più volte durante la risoluzione del problema più grande. Invece di ricalcolare ripetutamente questi sottoproblemi, memorizziamo le loro soluzioni.
- **Struttura ottimale**: La soluzione ottimale al problema contiene soluzioni ottimali ai suoi sottoproblemi. Questo significa che possiamo costruire la soluzione migliore combinando le soluzioni migliori delle parti più piccole.

## Soluzioni di programmazione dinamica

- **Memoizzazione (approccio top-down)**: La memoizzazione memorizza i risultati di chiamate di funzione costose e restituisce il risultato memorizzato quando si presentano gli stessi input.

```js
function climbStairsMemo(n, memo = {}) {
  // If already calculated, return cached value
  if (memo[n] !== undefined) {
    return memo[n];
  }

  // Base cases: 1 way for 1 step, 2 ways for 2 steps
  if (n <= 2) {
    return n;
  }

  // Calculate once and store in memo for future use
  memo[n] =
    climbStairsMemo(n - 1, memo) +
    climbStairsMemo(n - 2, memo);

  return memo[n];
}
```

- **Tabulazione (approccio bottom-up)**: La tabulazione costruisce la soluzione dal basso verso l’alto, riempiendo una tabella con le soluzioni ai sottoproblemi.

```js
function climbStairsTabulation(n) {
  if (n <= 2) {
    return n;
  }

  // Create an array to store solutions from 0 to n
  const dp = new Array(n + 1).fill(0);

  // Base cases
  dp[1] = 1; // 1 way to reach step 1
  dp[2] = 2; // 2 ways to reach step 2

  // Build up the solution iteratively
  for (let i = 3; i <= n; i++) {
    // Ways to reach step i = ways to reach (i-1) + ways to reach (i-2)
    dp[i] = dp[i - 1] + dp[i - 2];
  }

  return dp[n];
}
```

## Applicazioni reali della programmazione dinamica

- **Ottimizzazione delle rotte**: I sistemi GPS usano algoritmi di programmazione dinamica per trovare i percorsi più brevi tra le località.
- **Elaborazione del testo**: I correttori ortografici e le funzionalità di completamento automatico spesso si basano sulla programmazione dinamica per calcolare le distanze di modifica tra le parole.
- **Modellazione finanziaria**: Le strategie di investimento e l’ottimizzazione del portfolio impiegano frequentemente tecniche di programmazione dinamica.
- **Assegnazione delle risorse**: Il problema dello zaino e le sue varianti compaiono nella pianificazione, nel budgeting e nella gestione delle risorse.

## Quando usare la programmazione dinamica

Dovresti considerare di usare la programmazione dinamica nei seguenti casi:

- Il problema può essere suddiviso in sottoproblemi sovrapposti.
- Il problema presenta una struttura ottimale.
- Una soluzione ricorsiva ingenua comporterebbe calcoli ripetuti.
- Hai bisogno di ottimizzare la complessità temporale a scapito della complessità spaziale.
