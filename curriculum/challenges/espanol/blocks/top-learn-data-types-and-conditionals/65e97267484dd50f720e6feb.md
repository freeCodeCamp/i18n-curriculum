# --description--

Sabes que JavaScript tiene una variedad de tipos de datos, incluyendo strings, numbers y booleans. Puedes usar operadores de comparación para comparar estos tipos de datos y tomar decisiones basadas en los resultados.

Los siguientes operadores de comparación están disponibles en JavaScript:

| Operator | Description                              | Example      |
|----------|------------------------------------------|--------------|
| ==       | Equal (value comparison)                 | x == y       |
| !=       | Not equal                                | x != y       |
| >        | Greater than                             | x > y        |
| <        | Less than                                | x < y        |
| >=       | Greater than or equal to                 | x >= y       |
| <=       | Less than or equal to                    | x <= y       |

Para usar una comparación en una asignación de variable, puedes usar el operador `==`. Por ejemplo:

```javascript
let x = 5;
let y = 5;

let result = x == y;

console.log(result); // true
```
