# --description--

JavaScript에서 변수 이름에는 두 가지 제한이 있습니다:

1. 이름에는 문자, 숫자, 또는 기호 `$`와 `_`만 포함할 수 있습니다.
1. 첫 번째 문자는 숫자가 될 수 없습니다.

유효한 이름의 예:

```js
let userName;
let test123;
```

흥미로운 점은, 달러 기호 `'$'`와 밑줄 `'_'`도 이름에 사용할 수 있다는 것입니다. 이들은 문자처럼 특별한 의미 없이 일반 기호입니다.

다음 이름들은 유효합니다:

```js
let $ = 1; // declared a variable with the name "$"
let _ = 2; // and now a variable with the name "_"

console.log($ + _); // 3
```

잘못된 변수 이름의 예:

```js
let 1a; // cannot start with a digit

let my-name; // hyphens '-' aren't allowed in the name
```
