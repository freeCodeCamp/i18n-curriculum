# --description--

$S(A)$ をサイズ n の集合 A の要素の合計と表します。任意の2つの空でない互いに素な部分集合 B と C に対して、次の性質が成り立つ場合、これを特別な和集合と呼びます。

1. $S(B) ≠ S(C)$；つまり、部分集合の和は等しくなりません。
2. B の要素数が C より多い場合、$S(B) > S(C)$ となります。

与えられた n に対して $S(A)$ が最小化されている場合、それを最適な特別な和集合と呼びます。最初の5つの最適な特別な和集合は以下の通りです。

$$\begin{align}
  & n = 1: \\{1\\} \\\\
  & n = 2: \\{1, 2\\} \\\\
  & n = 3: \\{2, 3, 4\\} \\\\
  & n = 4: \\{3, 5, 6, 7\\} \\\\
  & n = 5: \\{6, 9, 11, 12, 13\\} \\\\
\end{align}$$

ある最適な集合 $A = \\{a_1, a_2, \ldots, a_n\\}$ に対して、次の最適な集合は $B = \\{b, a_1 + b, a_2 + b, \ldots, a_n + b\\}$ の形をしており、ここで b は前の行の「中央」の要素です。

この「ルール」を適用すると、$n = 6$ の最適な集合は $A = \\{11, 17, 20, 22, 23, 24\\}$ で、$S(A) = 117$ となるはずです。しかしこれは最適な集合ではなく、近似的な最適集合を得るためのアルゴリズムを単に適用しただけです。$n = 6$ の最適な集合は $A = \\{11, 18, 19, 20, 22, 25\\}$ で、$S(A) = 115$、対応する集合の文字列は `111819202225` です。

$A$ が $n = 7$ の最適な特別な和集合であるとき、その集合の文字列を求めてください。

**注意：** この問題は問題 105 と問題 106 に関連しています。
