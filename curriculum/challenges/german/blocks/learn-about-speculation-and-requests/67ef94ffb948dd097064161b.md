# --explanation--

Sophie sagt, dass das Ausführen von Tests eine gute Praxis ist, weil `it helps avoid mistakes`. Das bedeutet, es hilft Entwicklerinnen und Entwicklern, Fehler frühzeitig zu erkennen, bevor der Code veröffentlicht wird.

Tests können zeigen, ob etwas im Code nicht stimmt, auch wenn auf den ersten Blick alles in Ordnung scheint. Es ist ein wichtiger Teil beim Schreiben von hochwertiger Software.
