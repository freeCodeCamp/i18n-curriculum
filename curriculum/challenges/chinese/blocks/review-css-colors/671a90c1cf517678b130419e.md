## 色彩理论

- **色彩理论定义**：这是研究颜色如何相互作用以及它们如何影响我们感知的学问。它涵盖了颜色关系、颜色和谐以及颜色的心理影响。
- **原色**：这些颜色是黄色、蓝色和红色，是所有其他颜色派生的基本色调。
- **次级色**：这些颜色是通过混合等量的两种原色得到的。绿色、橙色和紫色是次级色的例子。
- **三级色**：这些颜色是通过将一种原色与相邻的次级色组合而成。黄绿色、蓝绿色和蓝紫色是三级色的例子。
- **暖色**：这些颜色包括红色、橙色和黄色，能唤起舒适、温暖和惬意的感觉。
- **冷色**：这些颜色包括蓝色、绿色和紫色，能唤起平静、宁静和专业的感觉。
- **色轮**：色轮是一个圆形图表，展示颜色之间的关系。它是设计师的重要工具，因为它帮助他们选择颜色组合。
- **类似色配色方案**：这些配色方案创造出协调且舒缓的体验。它们包含色轮上相邻的类似颜色。
- **互补色配色方案**：这些配色方案产生高对比度和视觉冲击力。它们的颜色位于色轮的相对两端。
- **三色配色方案**：这种配色方案色彩鲜艳。它们由色轮上大致等距的颜色组成。如果连接这些颜色，会形成一个等边三角形。
- **单色配色方案**：这种配色方案中，所有颜色都源自同一基色，通过调整其明度、暗度和饱和度来表现。这种方案既表现出统一和谐，又能制造对比。

## 在 CSS 中使用颜色的不同方法

- **命名颜色**：这些颜色是浏览器识别的预定义颜色名称。例子包括 `blue`、`darkred`、`lightgreen`。
- **`rgb()` 函数**：RGB 代表红色、绿色和蓝色——光的三原色。这三种颜色以不同强度组合，创造出广泛的颜色。`rgb()` 函数允许你使用 RGB 颜色模型定义颜色。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>RGB Color</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

- **`rgba()` 函数**：该函数添加了第四个值 alpha，用于控制颜色的透明度。 如果未提供，alpha 值默认为`1`。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>RGBA Background</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

- **`hsl()` 函数**：HSL 代表色相、饱和度和亮度——定义颜色的三个关键组成部分。现代 `hsl()` 语法还支持使用 `/` 分隔符的可选 alpha 值。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>HSL Color</p>
<p class="alpha">HSL Color with Alpha</p>
```

```css
p {
  color: hsl(120 100% 50%);
}

.alpha {
  color: hsl(120 100% 50% / 0.8);
}
```

:::

- **`hsla()` 函数**：`hsla()` 函数是为 HSL 颜色添加透明度的旧方法。现代 CSS 更倾向于使用带 `/` 分隔符的 `hsl()` 来表示 alpha 值。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>HSLA Background</div>
```


```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

- **十六进制**：十六进制代码（简称 hex 代码）用于表现 RGB 颜色模型中的颜色。“hex”指的是基数为 16 的数字系统，使用数字 `0` 到 `9` 和字母 `A` 到 `F`。十六进制代码可以是 6 个字符长，如 `#F498A2`，也可以是 3 个字符的简写形式，当完整形式中每对字符相同时可用简写。例如，`#2FB` 等同于 `#22FFBB`。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1 class="hex-text">Hex Text</h1>
<p class="hex-bg">Hex Background</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}

p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

## `box-shadow` 属性

- **定义**：`box-shadow` 属性在一个元素周围应用一个或多个阴影。
- **偏移值**：你必须指定水平（`offset-x`）和垂直（`offset-y`）值。正的 `offset-x` 会将阴影向右移动，负值则向左移动。正的 `offset-y` 会将阴影向下移动，负值则向上移动。如果值为 `0`，则无需包含单位。
- **模糊半径**：此可选的值控制阴影的模糊程度。如果未包含，则默认为 `0`，这会创建犀利的边缘。数值越大，阴影越柔和。
- **扩散半径**：此可选的值控制阴影的扩展或收缩程度。如果未包含，则默认为 `0`。
- **阴影颜色**：你可以使用命名颜色、十六进制值、`rgb()`、`rgba()`、`hsl()` 或 `hsla()` 函数来指定颜色。
- **`inset` 关键字**：添加 `inset` 关键字会将阴影放置在元素内部，而不是外部。
- **应用多个盒子阴影**：你可以通过逗号分隔来应用多个阴影。阴影从前到后分层。

### 语法

```css
box-shadow: offset-x offset-y blur-radius spread-radius color;
```

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="shadow-box">Shadow Color Example</div>
```

```css
.shadow-box {
  width: 200px;
  padding: 20px;
  background-color: lightblue;
  box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}
```

:::

## 线性和径向渐变

- **线性渐变**：这类渐变沿直线创建颜色之间的渐变混合。 你可以使用关键字来控制渐变的方向，如 `to top`、`to right`、`to bottom right`，或者使用角度，如 `45deg`。 你可以使用任何有效的 CSS 颜色和任意多的颜色停止点。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient">Linear Gradient</div>
```

```css
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

:::

- **径向渐变**：这些渐变可创建从中心点辐射开来的圆形或椭圆形渐变。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient">Radial Gradient</div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::
