# --interactive--

Die `toString()`-Methode ist eine grundlegende Funktion in JavaScript, die einen Wert in seine String-Darstellung umwandelt. Sie ist eine Methode, die Sie für Zahlen, Booleans, Arrays und Objekte verwenden können. Eine der häufigsten Anwendungen von `toString()` ist die Umwandlung einer Zahl in ihre String-Darstellung. Hier ist ein Beispiel:

:::interactive_editor

```js
const num = 10;
console.log(num.toString()); // "10"
```

:::

Diese Methode akzeptiert eine optionale Basis, die eine Zahl von `2` bis `36` ist. Diese Basis stellt das Zahlensystem dar, zum Beispiel Basis 2 für binär oder Basis 8 für oktal. Wenn die Basis nicht angegeben wird, ist standardmäßig Basis 10, also das Dezimalsystem, eingestellt. Hier ist ein Beispiel, bei dem `2` als Argument an die `toString()`-Methode übergeben wird:

:::interactive_editor

```js
const num = 10;
console.log(num.toString(2)); // "1010"
```

:::

Das Ergebnis wird `1010` sein, was die binäre Darstellung der Dezimalzahl `10` ist.

Sie können auch die `toString()`-Methode verwenden, um Arrays und Objekte in Strings umzuwandeln. Arrays haben eine eigene Implementierung von `toString()`, die jedes Element in einen String umwandelt und sie mit Kommas verbindet. Hier ist ein Beispiel:

:::interactive_editor

```js
const arr = [1, 2, 3];
console.log(arr.toString()); // "1,2,3"
```

:::

In diesem Beispiel werden alle Elemente des Arrays zusammengefügt, um den String `1,2,3` zu bilden.

Wenn die `toString()`-Methode bei Objekten verwendet wird, ist das Ergebnis keine stringifizierte Version der Objekteigenschaften.

:::interactive_editor

```js
const person = {
  name: "John",
  age: 30,
  isStudent: true
};

console.log(person.toString()); // "[object Object]"
```

:::

In diesem Beispiel ist das Ergebnis die standardmäßige String-Darstellung des Objekts, nämlich `[object Object]`. Um eine stringifizierte Version der `person`-Objekteigenschaften zu erhalten, müssen Sie `JSON.stringify()` verwenden, worüber Sie in zukünftigen Lektionen mehr erfahren werden.

Zusammenfassend wird die `toString()`-Methode verwendet, um Werte in Strings umzuwandeln. Zu verstehen, wie sie mit verschiedenen Datentypen funktioniert und wie Sie sie für Ihre eigenen Objekte anpassen können, verbessert Ihre Fähigkeit, Daten in Ihren JavaScript-Anwendungen zu verarbeiten und anzuzeigen.
