# --interactive--

Margini e spaziatura interna sono proprietà essenziali in CSS per creare pagine web ben strutturate, leggibili e visivamente attraenti.

I margini controllano lo spazio esterno a un elemento, aiutando a separarlo dagli altri elementi e a definire la struttura del layout, mentre la spaziatura interna controlla lo spazio all’interno di un elemento, migliorando la leggibilità del contenuto e l’aspetto estetico.

Per capire meglio le differenze tra `margin` e `padding`, diamo un’occhiata ad alcuni esempi.

Ecco un esempio HTML di tre elementi paragrafo nella pagina:

:::interactive_editor

```html
<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

:::

Per applicare spaziatura nella parte superiore di ogni elemento paragrafo, puoi usare la proprietà `margin-top` in questo modo:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  margin-top: 20px;
  border: 2px solid black;
}
```

:::

In questo esempio, stiamo applicando `20px` di margine solo nella parte superiore di ogni elemento paragrafo.

Abbiamo anche un bordo nero solido su tutti e quattro i lati così puoi vedere meglio il margine.

Le quattro diverse proprietà `margin` sono `margin-top`, `margin-right`, `margin-bottom` e `margin-left`.

Ecco un esempio di utilizzo di tutte e quattro le proprietà:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 30px;
  margin-left: 40px;
  border: 2px solid black;
}
```

:::

In questo esempio, stiamo assegnando valori di spaziatura su tutti e quattro i lati dell’elemento paragrafo. È stato aggiunto anche un bordo nero solido così puoi vedere meglio i margini.

Un altro modo per usare la proprietà `margin` è con la notazione abbreviata. Puoi specificare uno, due, tre o quattro valori contemporaneamente. Esploriamo insieme queste opzioni.

Quando usi un valore singolo nella proprietà abbreviata `margin`, quel valore esatto verrà applicato a tutti e quattro i lati dell’elemento obiettivo.

Ecco un esempio di utilizzo di un singolo valore nella proprietà abbreviata `margin`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px;
}
```

:::

Questo esempio di codice applicherà `10px` di `margin` in modo uguale su tutti e quattro i lati dell’elemento paragrafo.

Quando usi due valori, il primo valore si applica ai lati `top` e `bottom`, mentre il secondo valore si applica ai lati `left` e `right` dell’elemento.

Ecco un esempio di utilizzo di due valori per la proprietà abbreviata `margin`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px;
}
```

:::

Questo imposta i margini `top` e `bottom` a `10px`, e `20px` per i margini `left` e `right` dell’elemento paragrafo.

Se vengono forniti tre valori, il primo valore si applica al margine `top`, il secondo valore ai margini `left` e `right`, e il terzo valore al margine `bottom`.

Ecco un esempio per capire meglio:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px;
}
```

:::

Questo imposta il margine a `10px` per `top`, `20px` per `left` e `right`, e `30px` per `bottom`.

Quando usi quattro valori, hai più controllo, perché puoi specificare indipendentemente ogni valore del margine per ciascun lato dell’elemento obiettivo.

Il primo valore riguarda `top`, il secondo `right`, il terzo `bottom` e il quarto `left`.

Ecco un esempio di utilizzo della proprietà abbreviata margin con quattro valori:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px 40px;
}
```

:::

Questo imposta il margine a `10px` per `top`, `20px` per `right`, `30px` per `bottom` e `40px` per `left`.

La proprietà `padding` si usa per applicare spazio all’interno dell’elemento, tra il contenuto e il suo bordo.

Come la proprietà `margin`, le quattro proprietà `padding` sono `padding-top`, `padding-right`, `padding-bottom` e `padding-left`.

Ecco un esempio di come impostare la `padding` per un elemento paragrafo:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 30px;
  padding-left: 40px;
  border: 2px solid black;
}
```

:::

Questo imposta la spaziatura interna a `10px` per `top`, `20px` per `right`, `30px` per `bottom` e `40px` per `left`.

Come puoi vedere, `padding` si applica al contenuto che è dentro il bordo, a differenza di `margin` che si applica all’esterno del bordo.

Proprio come la proprietà `margin`, puoi anche scegliere di usare la proprietà abbreviata per `padding`.

Puoi specificare uno, due, tre o quattro valori anche per la proprietà abbreviata `padding`.

Ecco un esempio di utilizzo della proprietà abbreviata `padding` per l’elemento paragrafo visto prima:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding: 10px 20px 30px 40px;
  border: 2px solid black;
}
```

:::

Nell’esempio, usando la proprietà abbreviata il codice imposterà la spaziatura interna a `10px` per `top`, `20px` per `right`, `30px` per `bottom` e `40px` per `left` dell’elemento paragrafo.
