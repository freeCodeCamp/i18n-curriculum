# --explanation--

`That would be great` é usado para expressar entusiasmo e concordância com uma sugestão.O uso de `would` mostra que o falante está imaginando o resultado positivo da ideia.Por exemplo:

`If we could finish early, that would be great.` - Isso significa que terminar cedo é um resultado desejável.
