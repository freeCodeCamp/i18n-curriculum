# --description--

在上一課中，你學會了如何在區域執行 Python 腳本。但有時你可能不想創建完整的 Python 程式，只需要測試一些 Python 程式碼。這時候就會用到 Python 互動式 shell。

但首先，我們來複習一下什麼是終端機。

終端機是一種以文字為基礎的介面，讓你透過輸入指令與你的電腦互動。每個作業系統都附帶一個預設的終端機應用程式。在 macOS 上，你可以使用 Terminal 應用程式。在 Windows 上，你可以使用命令提示字元或 PowerShell。在 Linux 上，每個桌面環境都有自己的預設終端機應用程式，例如 GNOME Terminal 或 Konsole。

打開終端機應用程式，輸入 `python` 並按下 `Enter`。這會啟動 Python 互動式 shell。互動式 shell 是一個程式，讓你一次輸入一個指令並立即看到結果。

當你開始一個新的工作階段時，最初可能會看到這樣的輸出：

```bash
Python 3.12.2 (main, Mar 21 2024, 22:48:26) [Clang 14.0.3 (clang-1403.0.22.14.1)] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>> 
```

`>>>` 符號表示 Python 正在等待你輸入指令。試著在終端機輸入 `"Hello, world!"`：

```bash
print("Hello, world!")
```

你應該會看到文字像這樣出現：

```bash
>>> print("Hello, world!")
Hello, world!
```

文字列印完後，Python 會回到以下狀態：

```bash
>>>
```

這表示你可以輸入另一個指令。

Python 直譯器遵循所謂的讀取、求值、列印、迴圈循環（Read, Evaluate, Print, Loop）週期，簡稱 REPL。每當你輸入指令時，直譯器會讀取輸入、求值、列印結果，然後回到 `>>>`，讓你繼續輸入更多指令。

如果你嘗試輸入像這樣的無效指令會發生什麼事？

```bash
>>> something random
```

Python 直譯器仍會遵循相同的 REPL 流程。在這種情況下，你會收到錯誤訊息：

```bash
>>> something random
  File "<stdin>", line 1
    something random
              ^^^^^^
SyntaxError: invalid syntax
```

如果你想離開互動式 shell，可以輸入 `exit()`，或在 Mac/Linux 上按 `Ctrl + D`，在 Windows 上按 `Ctrl + Z + Enter`。

使用 Python 的互動式 shell 非常適合做小型程式實驗或基本探索。在你完成課程剩餘的課程時，可以在互動式 shell 中輸入你學到的一些新方法。

如果你打算處理較長的程式或跨多個檔案工作，則最好使用程式碼編輯器或整合開發環境。
