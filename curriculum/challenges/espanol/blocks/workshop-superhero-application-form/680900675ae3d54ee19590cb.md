# --description--

En una lección anterior, aprendiste cómo renderizar una lista de opciones usando el método `map()` así:

```jsx
function FruitList() {
  const fruits = ["Apple", "Banana", "Cherry", "Date"];
  return (
    <ul>
      {fruits.map((fruit, index) => (
        <li key={`${fruit}-${index}`}>{fruit}</li>
      ))}
    </ul>
  );
}
```

Recuerda que la `key` siempre debe ser única. Ayuda a React a identificar qué elementos han cambiado, y han sido añadidos o eliminados.

Después del primer `option`, utiliza una función flecha para recorrer el array `powerSourceOptions` usando `source` como parámetro.

Luego, dentro del método `map()`, crea un elemento `option` con una `key` de `source` y un `value` de `source`. Finalmente, usa `{source}` para mostrar el nombre de cada fuente de energía en el menú desplegable.
