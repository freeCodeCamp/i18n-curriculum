# --description--

Ili kupata upatikanaji wa vipengele vya safu ya data, unaweza kutumia nambari ya kielezo. Nambari ya kielezo huanza kutoka 0, kwa hivyo kipengele cha kwanza cha safu ya data kiko kwenye kielezo 0, kipengele cha pili kiko kwenye kielezo 1, na kadhalika.

Kwa mfano, ili kupata kipengele cha kwanza cha safu ya `fruits`, unaweza tumia msimbo ufuatao:

```javascript
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits[0]); // Output: apple
console.log(fruits[2]); // Output: orange
```

Ikiwa kielezo kiko nje ya mipaka, JavaScript itarudisha `undefined`. Hii inamaanisha kwamba ikiwa utajaribu kupata `fruits[3]` katika mfano uliotajwa hapo juu, itarudisha `undefined`.
