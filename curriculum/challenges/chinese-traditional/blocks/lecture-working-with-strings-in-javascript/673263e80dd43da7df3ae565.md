# --interactive--

在 JavaScript 中處理字串時，可能會有需要在較大的字串中定位特定子字串位置的時候。

子字串是一個出現在較大字串中的字元序列。例如，在字串 `hello world` 中，`hello` 和 `world` 是子字串。

要定位字串中子字串的位置，你可以使用 `indexOf()` 方法。  
JavaScript 中的 `indexOf()` 方法允許你在字串中搜尋子字串。

如果找到子字串，`indexOf()` 傳回該子字串第一次出現的索引（或位置）。如果未找到子字串，`indexOf()` 傳回 `-1`，這顯示搜尋失敗。

`indexOf()` 方法接受兩個引數：第一個是你想在較大字串中尋找的子字串，第二個是搜尋的起始位置選項。若你沒有提供起始位置，搜尋將從字串的開頭開始。

在這個背景關係中，**引數** 是你在呼叫函式（程式）或方法時給予的值，使該函式（程式）或方法能夠使用你提供的特定資訊來執行其任務。你將在未來的課程中學習更多關於引數的內容。

這裡是一個使用 `indexOf()` 方法來尋找字串 `awesome` 位置的範例：

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("awesome!");
console.log(position); // 14
```

:::

在此範例中，字串 `JavaScript is awesome!` 中的字 `awesome` 從索引值 `14` 開始，因此 `indexOf()` 方法傳回 `14`。

現在，讓我們看看當找不到子字串時會發生什麼：

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("fantastic");
console.log(position); // -1
```

:::

由於字串中沒有出現字 `fantastic`，該方法傳回 `-1`。

你也可以透過為 `indexOf()` 提供第二個引數來指定從字串中的哪裡開始搜尋。以下是一個範例：

:::interactive_editor

```js
let sentence = "JavaScript is awesome, and JavaScript is powerful!";
let position = sentence.indexOf("JavaScript", 10);
console.log(position); // 27
```

:::

在這種情況下，對 `JavaScript` 的搜尋從第 10 個字元之後開始，因此第二個出現的 `JavaScript` 位於索引 `27`。

重要的是要注意 `indexOf()` 方法區分大小寫。

在此範例中，以下會傳回 `-1`，因為大寫字母 `F` 在字串 `freeCodeCamp` 中找不到。

:::interactive_editor

```js
console.log("freeCodeCamp".indexOf("F")) // -1
```

:::

當你需要檢查字串中是否存在子字串並判斷其位置以進行後續操作時，使用 `indexOf()` 會非常有用。
