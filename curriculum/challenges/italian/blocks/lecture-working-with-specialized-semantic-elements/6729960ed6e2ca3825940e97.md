# --interactive--

L'elemento inline code si usa per rappresentare brevi frammenti di codice all'interno del testo. Gli usi comuni dell'elemento code sono per articoli tecnici e pagine di documentazione.

Ecco un esempio di utilizzo dell'elemento `code` per mostrare un frammento di codice CSS:

:::interactive_editor

```html
<p>
  To set the text color to blue in CSS, use the following code:
  <code>color: blue;</code>
</p>
```

:::

In questo esempio, la proprietà CSS `color` viene usata per impostare il colore del testo su `blue`. Racchiudendo questo frammento di codice all'interno dei tag `<code>`, si comunica al browser che il testo è un pezzo di codice in linea.

Il browser applicherà gli stili predefiniti per il contenuto all'interno dell'elemento `code`. Lo stile predefinito è un carattere tipografico a spaziatura fissa.

L'elemento `code` serve a rappresentare una singola riga di codice. Se vuoi rappresentare più righe di codice, devi inserire un elemento `code` all'interno di un elemento di testo preformattato.

L'elemento di testo preformattato si usa per rappresentare testo preformattato. Ecco un esempio di utilizzo dell'elemento di testo preformattato per mostrare una dichiarazione CSS:

:::interactive_editor

```html
<pre>
  <code>
    body {
      color: red;
    }
  </code>
</pre>
```

:::

Quando usi l'elemento `pre`, devi fare attenzione agli spazi perché verrà mostrato esattamente come scritto nel documento HTML.

Nel browser vedrai che il codice è indentato di diversi spazi verso destra. Se cambi l'indentazione nell'esempio di codice, vedrai una differenza nell'indentazione sullo schermo.

Quando si tratta di includere esempi di codice nel documento HTML, dovresti usare l'elemento `code` per esempi brevi in linea.

Se devi mostrare frammenti di codice più lunghi, allora devi usare gli elementi `pre` e `code`.
