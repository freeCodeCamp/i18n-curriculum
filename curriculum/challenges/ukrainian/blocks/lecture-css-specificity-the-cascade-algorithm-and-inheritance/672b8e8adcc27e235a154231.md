# --interactive--

Селектори типу, також відомі як селектори елементів, націлені на елементи за їхнім тегом.

Ці селектори є базовими в CSS і дозволяють застосовувати стилі до всіх випадків певного HTML-елемента.

Селектори типу дуже прості у використанні і записуються просто як ім’я тегу елемента, який ви хочете стилізувати.

Ось приклад селектора типу, що націлений на всі елементи абзацу на сторінці:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  color: blue;
}
```

:::

У цьому прикладі всі `p` елементи матимуть колір тексту, встановлений у `blue`.

Специфічність селекторів типу відносно низька порівняно з іншими селекторами. Значення специфічності для селектора типу — `(0, 0, 0, 1)`.

Це означає, що селектори типу будуть заміщені селекторами класу, селекторами ідентифікатора та рядковими стилями, але все одно можуть застосовувати стилі, якщо немає правил із вищою специфічністю.

Давайте розглянемо приклад, де селектори класу заміщають стилі селектора типу.

Ось приклад із двома елементами абзацу:

:::interactive_editor

```html
<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

:::

Обидва елементи абзацу мають клас `para`.

У CSS-файлі селектор типу націлений на абзаци, а селектор класу — на елементи з класом `para`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

```css
p {
  color: blue;
}

.para {
  color: red;
}
```

:::

Всі абзаци на сторінці з класом `para` матимуть колір тексту, встановлений у `red`, замість `blue`, тому що селектори класу мають вищу специфічність, ніж селектори типу.
