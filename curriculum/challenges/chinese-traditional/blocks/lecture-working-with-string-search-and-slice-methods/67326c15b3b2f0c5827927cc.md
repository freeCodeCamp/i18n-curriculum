# --interactive--

在 JavaScript 中處理字串時，你經常需要從較大的字串中擷取一部分或子字串。

例如，你可能想要擷取字的一部分、特定的字元序列，或只是句子的一段片段。

JavaScript 提供了數種完成此任務的行為，其中最常用的是 `slice()` 方法。

`slice()` 方法允許你擷取字串的一部分並傳回一個新的字串，而不會修改原始字串。它接受兩個參數：起始索引和選用的結束索引。

以下是基本語法：

```js
string.slice(startIndex, endIndex);
```

`startIndex` 是擷取開始的位置。`endIndex` 是擷取結束的位置。如果未提供，`slice()` 會擷取到字串的結尾。

讓我們來看一個從字串中擷取零件的簡單範例：

:::interactive_editor

```js
let message = "Hello, world!";
let greeting = message.slice(0, 5);

console.log(greeting);  // Hello
```

:::

在此範例中，`slice(0, 5)` 從索引 `0` 開始擷取字元直到但不包含索引 `5`。因此，字 `Hello` 被擷取出來。

如果你省略第二個參數，`slice()` 將會從起始索引擷取到字串的結尾：

:::interactive_editor

```js
let message = "Hello, world!";
let world = message.slice(7);

console.log(world);  // world!
```

:::

這裡，`slice(7)` 從索引 `7` 開始擷取字串直到字串結尾，結果為 `world!`。

你也可以使用負數作為索引。當你使用負數時，它會從字串的末尾向後計數：

:::interactive_editor

```js
let message = "JavaScript is fun!";
let lastWord = message.slice(-4);

console.log(lastWord);  // fun!
```

:::

在這種情況下，`slice(-4)` 從字串中擷取最後 4 個字元，給我們 `fun!`。

假設你想從字串的中間擷取一段。你可以同時提供起始和結束索引，以精確控制你想要的字串部分：

:::interactive_editor

```js
let message = "I love JavaScript!";
let language = message.slice(7, 17);

console.log(language);  // JavaScript
```

:::

這裡，`slice(7, 17)` 擷取從索引 7 開始並在索引 `17` 之前結束的字串，該字串是字 `JavaScript`。

`slice()` 方法是在 JavaScript 中擷取字串零件的強大工具。

你指定起始和結束索引，該方法傳回一個包含擷取部分的新字串。

透過正數、負數和省略的索引選項，可在不更改原始字串的情況下，將它調整為各種情境。
