# --description--

Дано трикутник $ABC$ з цілими сторонами, для якого $BC ≤ AC ≤ AB$. $k$ — кутовий бісектор кута $ACB$. $m$ — дотична в точці $C$ до описаного кола трикутника $ABC$. $n$ — пряма, паралельна $m$, що проходить через точку $B$.

Перетин прямих $n$ і $k$ позначено як $E$.

Скільки існує трикутників $ABC$ з периметром, що не перевищує $100\\,000$, для яких довжина відрізка $BE$ є цілим числом?
