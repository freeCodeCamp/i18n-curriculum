# --interactive--

تُعد محددات الـ ID من أقوى المحددات في CSS، إذ تسمح للمطوِّرين بتطبيق أنماط على عناصر محددة تحمل معرفات فريدة.

هذا يجعلها فعالة جدًا لاستهداف العناصر الفردية التي تحتاج إلى تنسيق فريد.

تُعرَّف محددات الـ ID برمز الهاش (`#`) متبوعًا باسم الـ ID. يجب أن تكون فريدة داخل مستند HTML، بمعنى ألا يشترك عنصران في نفس الـ ID.

إليك مثالًا يستخدم محدد ID:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="unique">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#unique {
  color: purple;
}
```

:::

في هذا المثال، سيكون لون نص العنصر الذي يحمل الـ ID `unique` هو `purple`.

تمتلك محددات الـ ID خصوصية عالية جدًا، أعلى من محددات النوع والصنف، لكنها أقل من الأنماط المضمَّنة. قيمة الخصوصية لمحدد ID هي `(0, 1, 0, 0)`.

هذا يعني أن محددات الـ ID يمكنها تجاوز محددات الصنف والنوع، لكنها يمكن أن تُتجاوز بواسطة الأنماط المضمَّنة.

```html
<head>
  <style>
    #unique {
      color: purple;
    }
    .highlight {
      color: green;
    }
    p {
      color: blue;
    }
  </style>
</head>
<body>
  <p id="unique" class="highlight">This text</p>
</body>
```
