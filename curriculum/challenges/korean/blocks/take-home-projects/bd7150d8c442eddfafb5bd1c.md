**목표:** 다음과 기능적으로 유사한 애플리케이션을 만드세요: <a href="https://p2p-video-chat-application.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://p2p-video-chat-application.freecodecamp.rocks/</a>.

아래 사용자 스토리를 충족하고 모든 테스트를 통과하세요. 필요한 라이브러리나 API는 자유롭게 사용하세요. 자신만의 스타일을 더해 만드세요.

**사용자 스토리:** 접속하면 브라우저가 내 카메라와 마이크에 접근할 수 있도록 허용을 요청합니다.

**사용자 스토리:** 허용하면 방 이름을 입력하라는 메시지가 표시됩니다.

**사용자 스토리:** 방 이름을 입력하면, 해당 이름의 방이 없을 경우 새 방이 생성됩니다.

**사용자 스토리:** 내 친구가 같은 웹사이트에 접속해 내가 입력한 방 이름을 입력하면 같은 방에 들어가 나와 영상 채팅을 할 수 있습니다.

**사용자 스토리:** 방 이름을 입력했는데 이미 그 방에 두 명이 있으면 방이 가득 찼다는 알림을 받습니다.

**사용자 스토리:** 누구나 방을 만들거나 참여할 수 있습니다. 방은 여러 개 있을 수 있지만 모두 고유한 이름이어야 합니다.

**사용자 스토리:** 사이트가 내 마이크와 웹캠에 접근하지 못하도록 선택할 수 있습니다. 그렇게 하거나 다른 드라이버 문제가 발생하면 필수 항목이라는 오류 메시지가 표시됩니다.

**사용자 스토리:** 방 이름 입력 단계를 취소하거나, 이름을 입력하지 않거나 공백만 입력하면 다시 유효한 방 이름을 입력하라는 메시지가 표시됩니다.

**사용자 스토리:** 방에 있는 두 사람 중 한 명이 연결이 끊기면, 같은 방에 다시 연결해 채팅을 계속할 수 있습니다.

이 사용자 스토리를 모두 구현한 후 라이브 앱 URL과 선택적으로 GitHub 저장소 URL을 입력하세요. 그런 다음 "도전 과제 완료" 버튼을 클릭하세요.
