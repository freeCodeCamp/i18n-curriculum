# --interactive--

CSS에서 배경 이미지를 다룰 때, 이러한 이미지가 어떻게 표시되는지 제어할 수 있는 여러 속성이 있습니다.

우리가 집중할 주요 속성은 `background-size`, `background-repeat`, `background-position`, 그리고 `background-attachment`입니다.

먼저 `background-image` 속성을 살펴보겠습니다:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

위 CSS는 `body` 요소에 고양이 배경 이미지를 설정합니다.

배경 이미지의 크기를 설정하려면 `background-size` 속성을 사용할 수 있습니다.

`contain`을 사용하면 이미지를 자르거나 늘리지 않고 가능한 한 크게 조절할 수 있습니다.

다음은 `background-size: contain`을 사용한 예입니다:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    min-height: 100px;
  }
</style>
```

:::

우리는 `min-height`를 `100px`으로 설정하여 배경 이미지가 보이도록 하고, 레이아웃이 기본 높이를 유지하게 하여 내용이 적어도 디자인이 균형 있고 시각적으로 매력적으로 보이게 합니다.

`background-size` 속성을 `cover` 값으로 변경하면, 배경 이미지가 비율을 유지하면서 `body` 요소 전체를 덮도록 크기가 조절됩니다.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    min-height: 100px;
  }
</style>
```

:::

앞선 예제들에서 배경 이미지가 계속 반복되는 것을 아마 눈치챘을 것입니다.

기본적으로 배경 이미지는 요소 전체를 채우기 위해 가로와 세로 모두 반복됩니다. 하지만 이 동작을 제어할 수 있습니다.

`background-repeat` 속성을 `no-repeat` 값으로 설정할 수 있습니다.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

`background-size`를 `contain`으로, `background-repeat`를 `no-repeat`으로 설정하면 이미지가 화면에서 더 이상 반복되지 않습니다.

배경 이미지를 가로 방향으로만 반복하고 싶다면 `repeat-x` 속성에 `background-repeat` 값을 사용할 수 있습니다.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-x;
    min-height: 100px;
  }
</style>
```

:::

배경 이미지를 세로 방향으로만 반복하고 싶다면 `repeat-y` 속성에 `background-repeat` 값을 사용할 수 있습니다.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-y;
    min-height: 100px;
  }
</style>
```

:::

배경 이미지를 화면에서 위치시키려면 `background-position` 속성을 사용할 수 있습니다.

`background-position` 속성은 요소 내에서 배경 이미지가 나타나는 위치를 설정할 수 있게 합니다. `top`, `bottom`, `left`, `right`, `center` 같은 키워드나 특정 픽셀 또는 백분율 값을 사용할 수 있습니다.

다음은 `center`과 `top`를 `background-position`에 사용한 예입니다:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

이 CSS 코드는 배경 이미지를 수평으로는 요소 중앙에, 수직으로는 위쪽에 위치시킵니다.

마지막으로 `background-attachment`는 페이지를 스크롤할 때 배경 이미지가 콘텐츠와 함께 움직일지 아니면 고정될지를 결정합니다.

주요 값은 `scroll` (기본값)으로 배경 이미지가 콘텐츠와 함께 스크롤되고, `fixed`으로 배경 이미지가 화면에서 같은 위치에 고정됩니다.

다음은 `fixed` 값을 `background-attachment` 속성에 사용한 예입니다:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-position: center top;
    background-attachment: fixed;
  }
</style>
```

:::

이 CSS 코드는 페이지를 스크롤해도 배경 이미지가 제자리에 고정되도록 설정합니다.

몇 가지 속성을 한 줄로 결합하고 싶다면 `background` 단축 속성을 사용할 수 있습니다.

예시는 다음과 같습니다:

:::interactive_editor

```html
<style>
  body {
    background: center top fixed
      url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

위 예시는 `background-image`, `background-position`, `background-attachment`를 한 줄로 결합했습니다.

이 속성들을 숙달하면 웹 디자인에서 배경 이미지가 어떻게 표시되는지 훌륭하게 제어할 수 있어, 더 시각적으로 매력적이고 기능적인 레이아웃을 만들 수 있습니다.
