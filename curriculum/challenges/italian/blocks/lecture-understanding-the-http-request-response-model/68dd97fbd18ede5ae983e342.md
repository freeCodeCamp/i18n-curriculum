# --description--

Nell'ultima lezione, hai imparato che il server elabora una richiesta HTTP e invia indietro una risposta. Quella risposta include varie risorse che vedi sul web.

Le risorse restituite da una risposta HTTP includono risorse basate su testo, risorse multimediali, caratteri tipografici e altro ancora. Ognuna di queste risorse arriva con un'intestazione **Content-Type**, che indica al browser come elaborare e mostrare correttamente la risorsa.

Le risorse basate su testo sono sempre state una parte fondamentale del web. Infatti, il web è nato come puro testo. Oggi, le risorse basate su testo includono:

* **HTML (Hypertext Markup Language)**: la struttura di base di qualsiasi pagina web. Senza HTML, una pagina web non può esistere.
    
* **CSS (Cascading Stylesheet)**: la risorsa che controlla l'aspetto di una pagina web. Con il CSS, applichi stili come colori, caratteri tipografici e layout all'HTML.
    
* **JavaScript**: JavaScript aggiunge interattività e contenuti dinamici a HTML e CSS.
    
* **JSON e XML**: questi due formati permettono di trasferire dati strutturati tra server e browser.
    
Le risorse multimediali includono immagini, audio e video. Portano il web alla vita con immagini e suoni.

Molti siti web si affidano alle immagini per comunicare informazioni visive. Le immagini esistono in diversi formati come `JPG`, `JPEG`, `PNG`, `WEBP` e `GIF`.

Siti come YouTube e Vimeo offrono contenuti video in formati come `MP4`, `MOV`, `AVI`, `WebM` e altri.

L'audio è una parte fondamentale di molte piattaforme perché permette agli utenti di ascoltare musica, podcast e altri contenuti sonori. I formati audio comuni includono `MP3`, `AAC`, Ogg e `WAV`. Piattaforme come Spotify e YouTube Music si basano su questi formati per offrire esperienze audio di alta qualità.

Altre risorse restituite in una risposta HTTP sono documenti come `PDF`, `DOCX` e `PAGES`, archivi compressi come `ZIP` e `RAR`, file di caratteri tipografici come `WOFF`, `WOFF2` e `TTF`, ed eseguibili come `EXE`.

Ecco una tabella di queste risorse comuni e i loro tipi di contenuto:

| Tipo di risorsa | Intestazione Content-Type |
| --- | --- |
| HTML | text/html |
| CSS | text/css |
| JavaScript | application/javascript |
| JPEG | image/jpeg |
| PNG | image/png |
| GIF | image/gif |
| SVG | image/svg+xml |
| WebP | image/webp |
| WOFF | font/woff |
| WOFF2 | font/woff2 |
| TTF | font/ttf |
| OTF | font/otf |
| MP4 | video/mp4 |
| WebM | video/webm |
| Ogg (Video) | video/ogg |
| MP3 | audio/mpeg |
| AAC | audio/aac |
| WAV | audio/wav |
| Ogg (Audio) | audio/ogg |
| JSON | application/json |
| XML | application/xml |
| PDF | application/pdf |
| ZIP | application/zip |
