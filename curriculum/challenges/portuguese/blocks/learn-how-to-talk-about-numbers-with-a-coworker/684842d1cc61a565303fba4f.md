`as needed`

### --feedback--

Esta frase de duas palavras significa quando necessário ou somente se a situação exigir.

# --explanation--

`Make sure to` é usado para expressar responsabilidade ou promessa, o que significa que o falante definitivamente fará algo.Por exemplo:

`Make sure to save your work before closing.` — Não esqueça de salvar antes de fechar o programa.

`As needed` significa fazer algo somente quando se torna necessário.Por exemplo:

`We can add more team members as needed.` — Eles adicionarão pessoas somente se isso se tornar necessário.

Na frase de Alice, ela está dizendo que informará regularmente David e mudará o plano orçamentário se o progresso do projeto exigir.
