# --description--

表達式 `a lot of` 可以用於可數和不可數名詞，以顯示大量的數量或數額。

對於可數名詞，例如 `cars`，你可能會說 `a lot of cars`。對於不可數名詞，例如 `water`，你會說 `a lot of water`。
