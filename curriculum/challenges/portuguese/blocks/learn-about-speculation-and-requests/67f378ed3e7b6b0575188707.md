## --blanks--

### --feedback--

`malicious link`

### --feedback--

Esta frase de duas palavras refere-se a um elemento clicável prejudicial que leva os usuários a uma página da web.

# --explanation--

Ataques de phishing frequentemente enganam usuários para clicarem em `malicious links`, levando a `unauthorized access` a contas ou sistemas.

`Unauthorized access` significa que alguém entra em um sistema sem permissão.Por exemplo:

`The system blocked unauthorized access.` - O sistema impediu que usuários não aprovados obtivessem acesso.

`Malicious link` refere-se a um link malicioso que pode roubar informações ou infectar um sistema.Por exemplo:

`The email had a malicious link inside.` - O email continha um link malicioso feito para enganar você.
