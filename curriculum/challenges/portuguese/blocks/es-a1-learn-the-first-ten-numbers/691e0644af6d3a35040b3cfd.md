# --description--

Em espanhol, o número 1 é chamado de `uno`.

É bastante único porque muda com base no gênero do substantivo que vem depois dele.

- Para substantivos masculinos, ele se reduz a `un`, como em `un libro` ("um livro").

- Para substantivos femininos, muda para `una`, como em `una pizza` ("a pizza").

Pratique a pronúncia deste número até se sentir mais confortável com seu som.
