# --explanation--

`Several` は「数人以上だが全員ではない」という意味です。例えば：

`Several users reported issues with the new software.` – これは2人以上のユーザーに問題があったが、全員ではないことを意味します。

`To target` は「誰かや何かを狙う、向ける」という意味です。例えば：

`The hackers targeted customers by sending fake emails.` – これは特に顧客を騙そうとしたことを意味します。
