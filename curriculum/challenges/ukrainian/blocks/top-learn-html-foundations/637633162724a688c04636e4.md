# --description--

HTML і CSS головним чином відповідають за відображення інформації. Вони не використовуються для програмування логіки. JavaScript, який ви вивчатимете в наступному розділі, використовується, щоб змусити вебсторінки виконувати дії. З HTML і CSS можна зробити досить багато, і вони вам обов’язково знадобляться обидва. Упродовж нашої навчальної програми наступні уроки зосереджені на тому, щоб надати вам інструменти, необхідні для успіху, коли ви дійдете до вмісту з JavaScript.
