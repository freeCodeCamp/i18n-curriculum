# --explanation--

الـ `log` هو سجل للأحداث أو الإجراءات أو الرسائل التي يُنشئها نظام حاسوبي أو برنامج. على سبيل المثال:

`The system logs showed several errors before the crash.` – هذا يعني أن السجلات احتوت معلومات عن مشاكل حدثت.

الـ `crash` هو عندما يتوقف نظام حاسوبي أو برنامج فجأة عن العمل بشكل صحيح. على سبيل المثال:

`The application crash caused all unsaved data to be lost.` – هذا يعني أن البرنامج فشل بشكل غير متوقع، ولم تُحفظ البيانات.
