# --explanation--

Структура `it's + adjective + to` використовується для опису характеру дії. Наприклад:

`It's necessary to prepare before an important presentation.` — це означає, що підготовка є необхідною дією перед презентацією.

У цьому реченні `it's healthy to have` означає, що наявність незгоди є корисною.
