# --description--

Sei $S(A)$ die Summe der Elemente in der Menge A mit der Größe n. Wir nennen sie eine spezielle Summenmenge, wenn für zwei nichtleere disjunkte Teilmengen B und C die folgenden Eigenschaften gelten:

1. $S(B) ≠ S(C)$; das heißt, die Summen der Teilmengen dürfen nicht gleich sein.
2. Wenn B mehr Elemente als C enthält, dann gilt $S(B) > S(C)$.

Wenn $S(A)$ für ein gegebenes n minimiert wird, nennen wir sie eine optimale spezielle Summenmenge. Die ersten fünf optimalen speziellen Summenmengen sind unten angegeben.

$$\begin{align}
  & n = 1: \\{1\\} \\\\
  & n = 2: \\{1, 2\\} \\\\
  & n = 3: \\{2, 3, 4\\} \\\\
  & n = 4: \\{3, 5, 6, 7\\} \\\\
  & n = 5: \\{6, 9, 11, 12, 13\\} \\\\
\end{align}$$

Es scheint, dass für eine gegebene optimale Menge $A = \\{a_1, a_2, \ldots, a_n\\}$ die nächste optimale Menge die Form $B = \\{b, a_1 + b, a_2 + b, \ldots, a_n + b\\}$ hat, wobei b das „mittlere“ Element der vorherigen Zeile ist.

Wenn wir diese „Regel“ anwenden, würden wir erwarten, dass die optimale Menge für $n = 6$ $A = \\{11, 17, 20, 22, 23, 24\\}$ ist, mit $S(A) = 117$. Dies ist jedoch nicht die optimale Menge, da wir lediglich einen Algorithmus angewandt haben, um eine nahezu optimale Menge zu erhalten. Die optimale Menge für $n = 6$ ist $A = \\{11, 18, 19, 20, 22, 25\\}$ mit $S(A) = 115$ und dem entsprechenden Mengenstring: `111819202225`.

Gegeben, dass A eine optimale spezielle Summenmenge für $n = 7$ ist, finden Sie ihren Mengenstring.

**Hinweis:** Dieses Problem steht in Zusammenhang mit Problem 105 und Problem 106.
