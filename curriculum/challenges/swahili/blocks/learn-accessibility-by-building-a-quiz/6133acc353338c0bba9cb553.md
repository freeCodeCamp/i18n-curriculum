# --description--

Mwishowe katika `head`, kipengele cha `title` ni muhimu kwa kisomaji cha skrini kuelewa yaliyomo kwenye ukurasa. Zaidi ya hayo, ni sehemu muhimu ya _SEO_.

Toa ukurasa wako `title` ambayo inaelezea na ni fupi.

# --hints--

Unapaswa kuongeza kipengele cha `title` katika `head`.

```js
assert.exists(document.querySelector('head > title'));
```

Haupaswi kufanya `title` kuwa ndefu zaidi ya herufi 60.

```js
assert.isAtMost(document.querySelector('head > title')?.textContent?.length, 60);
```

Jaribu kuwa na maelezo zaidi na kipengele cha `title`. _Kidokezo: Angalau herufi 15_

```js
assert.isAtLeast(document.querySelector('head > title')?.textContent?.length, 15);
```
