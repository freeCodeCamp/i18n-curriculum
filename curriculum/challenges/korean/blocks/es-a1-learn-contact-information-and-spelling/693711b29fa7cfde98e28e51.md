# --description--

전화번호와 같은 개인 정보를 줄 때는 다음 구조를 사용할 수 있습니다:

`Mi` + 명사 + `es ...`

`Mi`는 소유를 나타내며 "나의"를 의미합니다. 예를 들어:

- `Mi nombre es Ana.`

- `Mi correo es ana@ejemplo.com.`

- `Mi número es 5551234.`

이 도전 과제에서는 전화번호가 소리 내어 말해지는 것을 듣게 됩니다.
