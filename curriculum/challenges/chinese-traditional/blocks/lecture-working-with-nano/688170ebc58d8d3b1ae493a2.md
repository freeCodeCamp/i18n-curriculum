# --description--

在終端機中工作時，標準文字編輯器並不總是可用，但有些程式是專門設計用來直接從終端機內編輯檔案的。

Linux 環境通常包含一個文字編輯器。`Vi` 或 `Vim` 通常是最常見的。`Emacs` 和 `Nano` 可能需要手動安裝，但幾乎每個發行版都能輕鬆取得。

Vim 的挑戰在於它有相當陡峭的學習曲線。如果你還不熟悉它，你甚至可能不知道如何關閉該應用程式！Emacs 可以說是稍微對使用者友善一些，但仍然有自己的一大本鍵盤快速鍵。

Nano 明顯更精簡且更友善，特別是對於新手學習者。使用 Nano 可以更快速地開始並編輯一個檔案，且該應用程式甚至提供基本鍵盤快速鍵的螢幕說明。

然而，這樣的取捨是缺乏擴充性。與可以透過外掛程式擴充成完整開發環境的 Vim 或 Emacs 不同，Nano 相當簡潔。

讓我們來看看 Nano 可以做什麼。要開啟一個檔案，你可以在終端機輸入 `nano <filename>`。

一旦開啟檔案，你會在螢幕頂端看到你正在執行的 Nano 版本以及你已開啟的檔案名稱。

在那下面是檔案的內容。你可以使用方向鍵來移動你的游標，並用鍵盤插入或刪除文字。檔案會根據需要捲動以跟隨。

然後，在底部，你會看到一個可用鍵盤快速鍵的列表，看起來像這樣：

```bash
^G Help   ^O Write Out    ^F Where Is   ^K Cut      ^T Execute  ^C Location     M-U Undo    M-A Set Mark    M-] To Bracket    M-B Previous
^X Exit   ^R Read File    ^\ Replace    ^U Paste    ^J Justify  ^/ Go To Line   M-E Redo    M-6 Copy        ^B Where Was      M-F Next
```

以插入符號（`^`）符號開頭的快速鍵顯示你需要按住 Ctrl 鍵並按下顯示的字母。以 `M-` 開頭的快速鍵表示你需要按住「meta key」，在大多數設定中應該是 Alt 鍵，然後按下該字母。

要在 Nano 中編輯檔案，你只需要知道幾個快捷鍵指令。對檔案進行更改後，你可以按 `Ctrl + O` 來「寫出」或儲存檔案。輸入指令後，你會看到底部的選單變成這樣：

```bash
File Name to write : <filename>
^G Get Help                 ^T  To Files
^C Cancel                   TAB Complete
```

按下 Enter 以存檔該檔案並返回編輯該檔案，或按 `Ctrl + C` 取消並返回編輯該檔案。

當你在編輯器中時，可以按 `Ctrl + X` 離開 Nano 並返回終端機。如果你在嘗試離開時有未儲存的變更，你會在底部選單看到這個：

```bash
Save modified buffer (ANSWERING "No" WILL DESTROY CHANGES) ?
                            Y Yes
^C Cancel                   N No
```

按下 `Y` 以存檔你未存檔的變更，或按下 `N` 以捨棄它們。

有了這些知識，當你下次需要在遠端伺服器上快速編輯檔案，而你可能無法存取完整的圖形化環境時，應該會更有準備。
