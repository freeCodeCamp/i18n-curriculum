# --description--

`everyone` という言葉は、グループ内のすべての人を意味します。`Every-` は、グループ内のすべての物や人を一つずつ話すときに使われます。

`everyone` は多くの人について話しているように聞こえますが、単数の動詞と一緒に使うことが重要です。これは、`every-` で始まる他の言葉、例えば `everything` や `everybody` と似ています。

例えば、`Everyone goes to the party`、`Everything is right`、または `Everybody works hard` と言います。

`Make sure` は、先に進む前に何かが正しいかどうかを確認することを意味します。例えば、`You make sure the code runs` は、コードにエラーがなく正しく動作することを確かめることを意味します。

`Follow the rules` は、ルールに従うことを意味します。もし「`use camelCase for variable names`」というルールがあれば、`follow the rules` は変数の名前を `myVariable` のように付けることを意味し、`myBLANKvariable` のようにはしません。
