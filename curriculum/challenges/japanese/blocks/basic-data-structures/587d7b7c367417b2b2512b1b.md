# --description--

オブジェクトとは何か、その基本的な特徴や利点がわかりましたね。簡単に言うと、オブジェクトはキーと値のペアを格納するストアであり、柔軟で直感的にデータを構造化でき、さらに非常に高速な検索時間を提供します。このチャレンジの残りの部分では、オブジェクトに対してよく使われる操作をいくつか紹介し、プログラムでこれらの便利なデータ構造を使いこなせるようにしていきます。

これまでのチャレンジでは、オブジェクトのキーと値のペアを追加したり変更したりしてきました。ここでは、オブジェクトからキーと値のペアを*削除*する方法を見ていきます。

最後にもう一度、`foods` オブジェクトの例を見てみましょう。もし `apples` キーを削除したい場合は、次のように `delete` キーワードを使って削除できます。

```js
delete foods.apples;
```

# --instructions--

`delete` キーワードを使って、`oranges`、`plums`、`strawberries` のキーを `foods` オブジェクトから削除してください。
