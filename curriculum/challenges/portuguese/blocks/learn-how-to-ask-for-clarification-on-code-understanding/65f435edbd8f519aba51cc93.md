# --description--

Uma `function` em programação é um conjunto de instruções dentro de um programa. Ele foi projetado para executar uma tarefa específica. Você define uma `function` uma vez, especificando o que ela deve fazer e depois pode usá-la quantas vezes precisar chamando seu nome.

Se algo está `missing` no código, significa que uma parte necessária não está presente ou foi negligenciada.

Quando o código está `working as expected`, ele funciona exatamente como deveria, sem erros ou problemas.
