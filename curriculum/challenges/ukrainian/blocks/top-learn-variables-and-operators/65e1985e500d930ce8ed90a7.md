# --description--

Існують два обмеження щодо імен змінних у JavaScript:

1. Ім’я має містити лише літери, цифри або символи `$` та `_`.
1. Перший символ не повинен бути цифрою.

Приклади дійсних імен:

```js
let userName;
let test123;
```

Цікаво, що знак долара `'$'` і підкреслення `'_'` також можна використовувати в іменах. Вони є звичайними символами, як і літери, без жодного спеціального значення.

Ці імена є дійсними:

```js
let $ = 1; // declared a variable with the name "$"
let _ = 2; // and now a variable with the name "_"

console.log($ + _); // 3
```

Приклади неправильних імен змінних:

```js
let 1a; // cannot start with a digit

let my-name; // hyphens '-' aren't allowed in the name
```
