# --explanation--

`Software tools` são programas ou plataformas usados para completar tarefas, como codificação, design ou comunicação.

Um `inventory` é uma lista de itens que você possui — geralmente para rastreamento ou revisão.

Então, um `software tools inventory` é uma lista de todos os programas de software que a equipe usa.Por exemplo:

`We updated our software tools inventory last week.` — Eles verificaram e listaram todas as ferramentas de software que possuem.
