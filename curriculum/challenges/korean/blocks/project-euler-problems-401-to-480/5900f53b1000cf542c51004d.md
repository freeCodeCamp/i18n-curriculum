# --description--

3-스무스 수는 3보다 큰 소인수를 가지지 않는 정수입니다. 정수 $N$에 대해, $S(N)$을 $N$ 이하의 3-스무스 수들의 집합으로 정의합니다. 예를 들어, $S(20) = \\{1, 2, 3, 4, 6, 8, 9, 12, 16, 18\\}$입니다.

$F(N)$을 $S(N)$의 순열 중 각 원소가 모든 진약수 뒤에 오는 순열의 개수로 정의합니다.

$N = 20$일 때 가능한 순열 중 하나는 다음과 같습니다.

-   1, 2, 4, 3, 9, 8, 16, 6, 18, 12.

이 순열은 12가 약수 6보다 앞서 있기 때문에 유효하지 않습니다.

-   1, 2, 4, 3, 9, 8, 12, 16, 6, 18.

$F(6) = 5$, $F(8) = 9$, $F(20) = 450$, 그리고 $F(1000) ≈ 8.8521816557e\\,21$임을 확인할 수 있습니다.

$F({10}^{18})$을 구하세요. 답은 소수점 아래 열 자릿수까지 반올림한 과학적 표기법 문자열로 제출하세요. 답을 제출할 때는 가수와 지수를 구분하는 데 소문자 `e`를 사용하세요. 예를 들어 답이 $112\\,233\\,445\\,566\\,778\\,899$이라면 답 형식은 `1.1223344557e17`가 됩니다.
