# --description--

`Work overtime` significa lavorare più ore rispetto all'orario di lavoro abituale. Per esempio, `I usually work overtime during project deadlines` indica lavorare ore extra per rispettare le scadenze del progetto.

`Unless` si usa per parlare di un'eccezione a qualcosa o per introdurre una condizione che rende falsa l'affermazione precedente. Significa `if it's not the case`. Per esempio, `I walk to work unless it rains` significa che vado a piedi al lavoro se non piove.

`Urgent` indica qualcosa di molto importante che richiede attenzione immediata. Per esempio, `The project is urgent` significa che il progetto deve essere completato subito.

In una frase negativa, `don't/doesn't have to` esprime che qualcosa non è un requisito. Per esempio, `I don't have to work late` significa che non è necessario lavorare fino a tardi.
