## --text--

どの文が文法的に正しいですか？

## --answers--

`You must to complete the security training.`

### --feedback--

`Must` は助動詞であり、基本動詞の前に `to` を使いません。

---

`You must completed the security training.`

### --feedback--

`must` の後の動詞は基本形（`complete`）でなければならず、過去形（`completed`）ではありません。

---

`You must complete the security training.`

---

`You must completing the security training.`

### --feedback--

`Must` の後には基本動詞が続き、`-ing` 形は続きません。

# --explanation--

`Must` は助動詞なので、以下のルールに従います：

- 常に動詞の基本形（`to` なし）に続きます。

- 三人称単数形で `-s` を取りません。

- 疑問文や否定文で `do/does` を使いません。
