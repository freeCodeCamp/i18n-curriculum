# --description--

`Patterns` 是某事發生或執行的規律且重複的方式。在資料中，樣式可以顯示趨勢或行為。 

例如，客戶資料中的一個 `pattern` 可能是更多人使用弓作為配件。這是一個 `trend pattern`。

也有 `cycle patterns`，例如咖啡店在早上賣出更多咖啡。 

`Patterns` 可以在數字中找到，例如每個月銷售額增加，或在行為中找到，例如顧客偏好線上購物。
