# --description--

Você precisa saber quantos ticks a simulação deve executar. Lembre-se que:

- Cada portão tem um array `queue` que mostra quantos participantes chegam em cada tick.  
- Como todos os portões têm o mesmo número de ticks, você pode simplesmente usar o comprimento da fila do primeiro portão.  

Crie uma variável chamada `maxTicks` e atribua a ela o valor de `gates[0].queue.length`.

Esse valor será usado para controlar seu loop de simulação, garantindo que cada tick seja processado para todos os portões.
