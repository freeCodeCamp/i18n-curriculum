# --description--

У попередньому уроці ви дізналися, як встановити Python на свій локальний пристрій. У цьому уроці ви навчитеся запускати код Python локально.

Перш ніж почати створювати свої скрипти Python локально, вам потрібно встановити редактор коду або IDE. IDE означає інтегроване середовище розробки, яке має функції, включно з редактором коду, інструментами тестування, терміналом і не тільки.

Одним із популярних редакторів коду, який використовують багато розробників, є VS Code. Відвідайте `https://code.visualstudio.com/download`, щоб скачати VS Code, який підтримує середовища Mac, Windows і Linux. Саме цей редактор буде використаний у цьому уроці.

Інші загальні варіанти для розробки на Python включають:

- PyCharm
- Spyder

Після того, як ви скачали свій редактор, вам потрібно його відкрити. Коли він відкритий, потрібно відкрити папку. Ви можете натиснути на опцію меню `File` у верхньому лівому куті і вибрати `Open Folder`. Звідти ви можете обрати або створити папку для свого проєкту. Для додаткової практики поза межами уроку, можливо, вам знадобиться папка з назвою `python-projects`.

Коли ваша папка налаштована, ви можете відкрити провідник, який знаходиться в лівій бічній панелі VS Code. Натисніть на іконку `New File`, яка виглядає як аркуш паперу з іконкою плюса. Потім створіть файл з назвою `main.py` і натисніть Enter. `.py` — це розширення файлу, що сигналізує, що це файл Python.

Відкрийте свій файл `main.py` і введіть наступний код:

```py
print("Hello, world!")
```

Щоб запустити цей код, у вас є кілька варіантів. Перший варіант — натиснути на кнопку запуску, яка виглядає як кнопка відтворення і розташована у верхньому правому куті. Це відкриє термінал і запустить ваш скрипт Python там. Ви повинні побачити текст `"Hello, world!"`.

Інший варіант — відкрити термінал і вручну запустити програму за допомогою:

```bash
python main.py
```

Цю команду потрібно запускати з папки, де збережено `main.py`. Наприклад, якщо `main.py` знаходиться у папці `python-projects`, спочатку використайте у терміналі:

```bash
cd python-projects
```

Потім запустіть:

```bash
python main.py
```

У деяких середовищах, особливо на macOS і Linux, де встановлені і Python 2, і Python 3, можливо, доведеться використовувати `python3` замість `python`. Це поширено, коли команда `python` або не існує, або вказує на старішу версію Python.

```bash
python3 main.py
```

У міру проходження курсу ви повинні намагатися запускати приклади з уроків локально і дивитися на результати. Також варто пробувати створювати власні приклади, щоб перевірити, що ви засвоїли з уроків.
