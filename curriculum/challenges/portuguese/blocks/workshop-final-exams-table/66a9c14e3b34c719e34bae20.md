# --description--

Para adicionar dados de estudantes à tabela, você precisará usar os elementos table row e table data.

O elemento <dfn>table data</dfn>, `td`, é usado para criar uma célula na tabela.

Aqui está um exemplo de uso do elemento `td` para uma tabela de jogadores:

```html
<tr>
  <td>1</td>
  <td>John Doe</td>
  <td>USA</td>
</tr>
```

Dentro do elemento do corpo da sua tabela, adicione um elemento de linha da tabela, `tr`, com três elementos de dados da tabela, `td`. 

O primeiro elemento de dados da tabela deve conter o sobrenome de `Davis`. 

O segundo elemento de dados da tabela deve conter o primeiro nome de `Alex`. 

O terceiro elemento de dados da tabela deve conter a nota de `54`.
