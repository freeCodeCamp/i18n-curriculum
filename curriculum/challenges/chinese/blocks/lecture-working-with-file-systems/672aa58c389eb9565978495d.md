# --description--

文件管理应用是一种在你的计算机上轻松保存、组织和编辑文件的方式。无论你使用 Mac 还是 Windows 计算机，都有内置的应用可以用来组织你的文件。

如果你有一台 Windows 计算机，默认的文件管理器是 File Explorer。它用于浏览、查找和管理文件及文件夹，以及执行诸如拷贝、移动和删除等文件操作。

要查找文件资源管理器，你可以进入开始菜单或按键盘上的 Windows 徽标键。

要固定一个文件夹，你可以用鼠标右键点击并选择 "Pin"。要移动一个文件或文件夹，你首先需要选择它，然后选择 "Cut"，接着导航到新位置进行粘贴。

如果你有 Mac 计算机，默认的文件管理器是 Finder。它提供对局部的和远程文件的访问，支持文件预览，并提供多种组织特色，如标签和智能文件夹。

要访问你 Mac 上的 Finder，请转到底部的 Dock 并点击 Finder 图标。

你会注意到左侧有一个侧边栏，里面是你收藏的文件夹。

组织你的文件的一种方法是使用标签。要为文件或文件夹添加标签，你可以在 Finder 中选择该项，然后用鼠标右键点击。你会在下拉菜单中看到 `Tags...` 选项，你可以应用自定义颜色标签。

要查找带标签的项，你可以滚动到侧边栏底部并点击相应颜色的标签，或者你可以在查找字段中按颜色名称查找标签。

另一种组织你的文件的方法是使用智能文件夹。智能文件夹是根据你设置的条件（例如文件类型、日期或关键字）自动组织的文件集合。

无论你是在 Windows 上使用文件资源管理器还是在 Mac 上使用 Finder，这些应用都提供了通过标签、查找和智能文件夹等特色来管理你的文件的灵活性。
