# --explanation--

`Let's work on` — це спосіб запропонувати співпрацю. Зазвичай за ним іде іменник або форма `-ing`. Цю фразу часто використовують, щоб заохотити командну роботу та рухатися вперед із завданням. Наприклад:

- `Let's work on fixing these security vulnerabilities.` — пропозиція розглянути проблеми безпеки в системі.

- `Let's work on the documentation so it's easier to follow.` — пропозиція покращити документацію для кращої зрозумілості.
