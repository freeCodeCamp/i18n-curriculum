# --description--

多項式 $g(x) = x^3 - 2^n \times x^2 + n$ の最大の実根を $a_n$ とします。

例えば、$a_2 = 3.86619826\ldots$ です。

$\displaystyle\sum_{i = 1}^{30} \lfloor {a_i}^{987654321}\rfloor$ の下8桁を求めてください。

**注意:** $\lfloor a\rfloor$ は床関数を表します。
