# --explanation--

`Lightweight` описує щось, що не потребує багато ресурсів або потужності, завдяки чому його легко запустити, воно швидке та ефективне. Наприклад:

`The app is lightweight, so it runs smoothly even on older smartphones.` — це означає, що застосунок використовує дуже мало пам’яті або обчислювальної потужності, що дозволяє йому добре працювати на пристроях із меншою потужністю.

`Features` — це конкретні функції або інструменти, які продукт або програмне забезпечення пропонують користувачам. Наприклад:

`The software has many useful features like file sharing and cloud storage.` — це означає, що програмне забезпечення включає кілька корисних функцій, які роблять його більш цінним і ефективним для користувачів.
