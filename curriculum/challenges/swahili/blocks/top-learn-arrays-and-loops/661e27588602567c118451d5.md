# --description--

Sasa baada ya kuwa na uelewa wa msingi kuhusu safu za data, tuzungumzie kuhusu mizunguko. Mizunguko hutumika kutekeleza kipande cha msimbo mara nyingi. Moja ya mizunguko hiyo ni mzunguko wa `while`. Mzunguko wa `while` hutekeleza kipande cha msimbo mradi tu sharti liwe kweli. Sintaksia ya mzunguko wa `while` ni kama ifuatavyo:

```javascript
while (condition) {
  // code block to be executed
}
```

Kwa mfano, kipande cha msimbo kilicho hapa chini kinachapisha nambari kutoka 1 hadi 5:

```javascript
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
```

Kipande cha msimbo kilicho juu kinaanzisha kigezo `i` na thamani `1`. Mzunguko wa `while` hutekeleza kipande cha msimbo mradi tu thamani ya `i` iwe ndogo au sawa na `5`. Thamani ya `i` huongezwa kwa `1` kila mzunguko.
