# --interactive--

在使用 CSS 為網頁設計樣式時，定義顏色最常見的方法之一是使用十六進位顏色值。十六進位顏色值，或稱為 hex code，是在 RGB 顏色模型中表述顏色的一種簡潔方式。在本課程中，我們將探討什麼是 hex code、它們在 CSS 中如何運作，以及何時使用它們。

十六進位碼（hex code 的縮寫）是一個六字元字串，用來表述 RGB 色彩模型中的顏色。「hex」指的是使用數字 `0` 到 `9` 和字母 `A` 到 `F` 的十六進位制系統。 

在顏色的背景關係中，十六進位碼指定組成特定顏色的紅色、綠色和藍色（RGB）數量。每個十六進位碼以井字號符號（`#`）開始，後面跟著六個字元，定義該顏色中紅色、綠色和藍色的強度。 

例如，黑色的十六進位碼是 `#000000`，白色的十六進位碼是 `#FFFFFF`。前兩個字元表述紅色值，接下來兩個表述綠色，最後兩個表述藍色。

在 CSS 中，你可以使用十六進位色碼為元素套用顏色。以下是基本語法：

```css
element {
  color: #RRGGBB;
}
```

這裡，`#RRGGBB` 是實際十六進位碼的佔位符。`RR`、`GG` 和 `BB` 配對分別表述紅色、綠色和藍色的強度。 

這些對組可以從 `00`（最低強度）到 `FF`（最高強度）不等。數字越高，該顏色在最終混合中出現的比例越多。 例如，讓我們將一些十六進位顏色套用到幾個元素上：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h1>This is a heading.</h1>
<p>This is a paragraph.</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}
p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

在第一種情況下，`#FF5733` 是一種紅橙色。以下是詳細說明：

- `FF` 是紅色（完全飽和）的最高值。
- `57` 是適量的綠色。
- `33` 是少量的藍色。

在第二種情況下，`#4CAF50` 是一種綠色調：

- `4C` 是紅色的一個低值。
- `AF` 是綠色的強值。
- `50` 是藍色的中等值。

十六進位碼很受歡迎，因為它們精確且易於使用。它們讓你能微調顏色，非常適合對一致性要求高的設計。 

大多數設計軟體，例如 Adobe Photoshop 或 Figma，都會提供你所選擇顏色的十六進位碼，讓你可以輕鬆地將這些數值直接複製並貼到你的 CSS 檔案中。

在網頁開發中，十六進位代碼經常用於定義文字顏色、背景、邊框和其他視覺元素。 例如：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example</button>
```

```css
body {
  background-color: #F0F0F0; /* Light gray background */
}

button {
  background-color: #007BFF; /* A shade of blue */
  color: #FFFFFF; /* White text */
}
```

:::

這確保你的顏色在整個網站或應用程式中保持一致。

在某些情況下，你可能會注意到十六進位碼以簡寫格式表示，只使用三個字元而非六個字元。當每個顏色配對中的兩個字元相同時，這是可能的。例如，`#FF5733` 無法簡寫，但 `#FFFFFF`（白色）可以寫成 `#FFF`。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h2>This is a heading.</h2>
```

```css
body {
  background-color: #000;
}

h2 {
  color: #FFF; /* White in shorthand */
}
```

:::

簡寫 `#FFF` 等價於 `#FFFFFF`。同樣地，`#000` 表述黑色，完整表單為 `#000000`。

十六進位碼因其精確度和相容性而廣受歡迎，允許所有瀏覽器呈現超過 1,600 萬種顏色。當你需要特定的品牌色彩、想要控制無法輕易用命名顏色或 HSL 定義的精確色調，或是從預設提供十六進位碼的設計軟體複製顏色時，它們是理想的選擇。
