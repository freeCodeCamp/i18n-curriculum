# --interactive--

Safu ya data katika JavaScript ni mkusanyiko uliopangwa wa thamani, kila moja ikitambulika kwa kielezo cha nambari. Thamani katika safu ya data ya JavaScript zinaweza kuwa za aina tofauti za data, ikiwa ni pamoja na nambari, mfuatano wa herufi, thamani ya kweli au siyo kweli, vitu, na hata safu nyingine za data.

Ili kuunda safu ya data katika JavaScript, unaweza kutumia mabano ya mraba, `[]`, na kutenganisha thamani kwa koma. Hapa kuna mfano:

```js
let fruits = ["apple", "banana", "orange"];
```

Katika mfano huu, tunatangaza kigezo `fruits` na kuweka thamani yake kuwa safu ya data yenye mfuatano wa herufi tatu: `apple`, `banana`, na `orange`.

Moja ya sifa kuu za safu za data ni kwamba zina kielezo cha kuanzia sifuri, ikimaanisha kuwa kipengele cha kwanza katika safu kina kielezo cha `0`, kipengele cha pili kina kielezo cha `1`, na kadhalika. Unaweza kupata upatikanaji wa vipengele binafsi katika safu kwa kutumia kielezo chao. Kwa mfano:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits[0]); // "apple"
console.log(fruits[2]); // "orange"
```

:::

Katika mfano huu, tunatumia kielezo `0` kupata kipengele cha kwanza (`apple`) na kielezo `2` kupata kipengele cha tatu (`orange`).

Safu za data katika JavaScript zina sifa maalum ya `length` inayorejesha idadi ya vipengele vilivyomo katika safu. Unaweza kupata sifa hii kwa kutumia sifa ya `length`. Kwa mfano:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits.length); // 3
```

:::

Sifa nyingine kuu ya safu za data katika JavaScript ni kwamba ni za mabadiliko, ikimaanisha kuwa ukubwa wake unaweza kubadilika baada ya kuundwa. Unaweza kuongeza au kuondoa vipengele kutoka katika safu kwa kutumia njia mbalimbali za safu, kama `push()`, `pop()`, `shift()`, `unshift()`, `splice()`, na nyinginezo. Njia hizi zitatufundishwa katika mafundisho yajayo.

Safu za data za JavaScript ni za aina nyingi na zinafaa sana linapokuja suala la uhifadhi wa data ndani ya programu zako. Katika moduli hii, utaona kwa karibu jinsi kufanya kazi na safu za data kutakusaidia kusimamia na kubadilisha makusanyo ya data kwa ufanisi.
