# --description--

Найменше додатне ціле число $n$, для якого числа $n^2 + 1$, $n^2 + 3$, $n^2 + 7$, $n^2 + 9$, $n^2 + 13$ та $n^2 + 27$ є послідовними простими числами, дорівнює 10. Сума всіх таких цілих чисел $n$ менших за мільйон дорівнює 1242490.

Яка сума всіх таких цілих чисел $n$ менших за 150 мільйонів?
