# --interactive--

JavaScriptファイル内でHTML要素のスタイルを操作する必要がある場合があります。例えば、ユーザーがボタンをクリックしたときにメニューを隠したり表示したりする場合です。

このレッスンでは、`Element.style` と `Element.classList` のプロパティを使ってスタイルを操作する方法を学びます。

`Element.style` プロパティは読み取り専用のプロパティで、要素のインラインスタイルを表します。このプロパティを使って要素のスタイルを取得したり設定したりできます。

以下は、IDが `para` の段落要素の例です：

```html
<p id="para">This is a paragraph</p>
```

`style` プロパティを使って、段落要素の色を赤に変更できます：

:::interactive_editor

```html
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```js
const paraEl = document.getElementById("para");
paraEl.style.color = "red";
```

:::

`style` プロパティは、`color`、`background-color`、`font-size`、`font-weight` など、多くのCSSプロパティを設定するのに使えます。

スタイルを操作するもう一つの方法は、`Element.classList` プロパティを使うことです。`classList` プロパティは読み取り専用で、要素のクラスを追加、削除、または切り替えるのに使えます。いくつか例を見てみましょう。

この例では、IDが `highlight` の段落要素に `para` というクラスを追加します。CSSでは、`highlight` クラスを設定して段落要素の背景色を黄色に変えます。JavaScriptでは、`highlight` メソッドを使って段落要素に `classList.add()` クラスを追加します。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```css
.highlight {
  background-color: yellow;
}
```

```js
const paraEl = document.getElementById("para");
paraEl.classList.add("highlight");
```

:::

複数のクラスを要素に追加したい場合は、`classList.add()` メソッドに引数として渡すことができます：

```js
classList.add("class1", "class2", "class3");
```

要素からクラスを削除する必要がある場合は、次のように `classList.remove()` メソッドを使えます：

```js
classList.remove("highlight");
```

要素のクラスを切り替えるには、`classList.toggle()` メソッドを使います。この例では、IDが `button` の `toggle-btn` 要素と、IDが `nav` の `menu` 要素があります。CSS内では、メニュー用のスタイルと、`show` というクラスを追加しています。このクラスは `display` の `.menu` プロパティを `block` に変えます。JavaScriptでは、`button` メソッドを使ってメニュー要素と `getElementById()` 要素にアクセスしています。そして、`button` 要素にイベントリスナーを追加し、ボタンがクリックされると `show` 要素の `.menu` クラスを切り替えます：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button id="toggle-btn">Toggle Menu</button>

<nav id="menu" class="menu">
  <ul>
    <li>Home</li>
    <li>About</li>
    <li>Products</li>
  </ul>
</nav>
<script src="index.js"></script>
```

```css
.menu {
  display: none;
  background-color: lightgray;
  width: 50%;
  padding: 10px;
}

.menu.show {
  display: block;
}
```

```js
const menu = document.getElementById("menu");
const toggleBtn = document.getElementById("toggle-btn");

toggleBtn.addEventListener("click", () => menu.classList.toggle("show"));
```

:::

ユーザーがボタンをクリックすると、`show` 要素に `.menu` クラスが追加され、メニューが表示されます。もう一度ボタンをクリックすると、`show` 要素から `.menu` クラスが削除され、メニューが非表示になります。

`style`、`add()`、`remove()`、`toggle()` といったメソッドやプロパティを使うことで、JavaScriptで要素のスタイルを簡単に操作し、動的でインタラクティブなウェブページを作成できます。
