# --description--

Agora é hora de aprender os números de 70 a 79.

Em espanhol, o número 70 é chamado de `setenta`.

Esses números são formados usando a palavra das dezenas `setenta` (70), seguida da conjunção `y` ("and") e depois o dígito das unidades (1-9).

| Number | Spanish |
| :--- | :--- |
| 70 | `Setenta` |
| 71 | `Setenta y uno` |
| 72 | `Setenta y dos` |
| 73 | `Setenta y tres` |
| 74 | `Setenta y cuatro` |
| 75 | `Setenta y cinco` |
| 76 | `Setenta y seis` |
| 77 | `Setenta y siete` |
| 78 | `Setenta y ocho` |
| 79 | `Setenta y nueve` |

O número `setenta y uno` (71) é único nesta faixa porque se torna `setenta y un` antes de um substantivo masculino. Por exemplo:

`Setenta y un años` - Setenta e um anos.
