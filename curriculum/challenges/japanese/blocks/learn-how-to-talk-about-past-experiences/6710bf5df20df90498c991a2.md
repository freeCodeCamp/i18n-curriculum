# --explanation--

`Past Continuous` 時制は過去のある時点で進行中だった動作の継続や背景を強調することが多いです。過去の特定の時点で進行していた動作を説明するために使われます。構造は **`was`/`were` + 動詞の`-ing`形** です。

ジェイクの文では、`I was reviewing` が昨日レビューしている動作が進行中だったことを示しています。別の例：

`She was testing the new feature when the system crashed.` - ここでは、システムのクラッシュが新機能のテストという長く続く動作の途中で起こりました。テストはより長い進行中の動作です。
