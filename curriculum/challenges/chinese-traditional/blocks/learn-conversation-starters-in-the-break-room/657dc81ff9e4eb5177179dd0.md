# --description--

`Occasionally` 意味著某件事偶爾發生。它不是一直發生，也不是從不發生，而是在兩者之間。 例如：

`I occasionally get a bug in my code.` - 有時你的程式碼中會有錯誤，但並不常見。

看看這些副詞，以及它們通常表述的頻率。

| Adverb       | Frequency | Adverb    | Frequency |
| ------------ | --------- | --------- | --------- |
| Never        | 0%        | Often     | 70%       |
| Rarely       | 5%        | Generally | 80%       |
| Seldom       | 15%       | Usually   | 90%       |
| Occasionally | 30%       | Always    | 100%      |
| Sometimes    | 50%       |           |           |
