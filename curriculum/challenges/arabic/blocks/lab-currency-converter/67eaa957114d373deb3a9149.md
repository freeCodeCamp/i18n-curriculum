# --description--

**الهدف:** نفذ قصص المستخدم أدناه واجعل جميع الاختبارات تجتاز لإكمال المختبر.

**قصص المستخدم:**

1. يجب أن يعرض مكوِّن `CurrencyConverter` عنصر `input` لقبول المبلغ المراد تحويله.
2. يجب أن يقبل عنصر `input` الأرقام.
3. يجب أن يعرض مكوِّن `CurrencyConverter` عنصرين `select` لاختيار العملة للتحويل **من** وإلى.
4. يجب أن يحتوي عنصر `select` على خيارات لـ **على الأقل** `USD`، `EUR`، `GBP`، و`JPY`. يمكنك استخدام أي سعر صرف، بشرط ألا يكون هناك تطابق واحد لواحد بين العملات. إليك بعض الأمثلة على الخرائط الجيدة والسيئة:

   ```js
   const badMapping = {
       USD: 1,
       EUR: 1,
       GBP: 1,
       JPY: 1
   };
      
   const goodMapping = {
       USD: 1,
       EUR: 0.92,
       GBP: 0.78,
       JPY: 156.7
   };
   ```

5. يجب أن يحفظ مكوِّن `CurrencyConverter` حساب المبالغ المحولة للعملة **المراد التحويل منها** بحيث لا يعيد حساب المبالغ عند تغيير خيار `select` الخاص بالعملة **المراد التحويل إليها**.
6. يجب أن يعرض مكوِّن `CurrencyConverter` عنصرًا يظهر المبلغ المحول بالتنسيق `XX.XX CCC`، حيث `XX.XX` هو المبلغ المحول و`CCC` هو رمز العملة.
7. يجب تقريب المبلغ المحول إلى منزلتين عشريتين.

  const body = doc.cloneNode(true);

export function CurrencyConverter() {
  const [amount, setAmount] = React.useState(1);
  const [startCurrency, setStartCurrency] = React.useState("USD");
  const [endCurrency, setEndCurrency] = React.useState("EUR");
  
  const exchangeRates = {
    USD: 1,
    EUR: 0.85,
    GBP: 0.75,
    JPY: 110
  }

  const convertedAmounts = React.useMemo(() => {
    const converted = {};
    Object.keys(exchangeRates).forEach((curr) => {
      converted[curr] = ((amount / exchangeRates[startCurrency]) * exchangeRates[curr]).toFixed(2);
    });
    return converted;
  }, [amount, startCurrency]);
  
  return (
    <main>
      <h1>Currency Converter</h1>
      <p className="conversion-display">{startCurrency} to {endCurrency} Conversion</p>
      <input
        type="number"
        value={amount}
        onChange={(e) => setAmount(Number(e.target.value))}
      />
      <label>
        Start Currency:
        <select value={startCurrency} onChange={(e) => setStartCurrency(e.target.value)}>
          {Object.keys(exchangeRates).map((curr) => (
            <option key={curr} value={curr}>
              {curr}
            </option>
          ))}
        </select>
      </label>
      <label>
        Target Currency:
        <select value={endCurrency} onChange={(e) => setEndCurrency(e.target.value)}>
          {Object.keys(exchangeRates).map((curr) => (
            <option key={curr} value={curr}>
              {curr}
            </option>
          ))}
        </select>
      </label>
      <p>Converted Amount: <span>{convertedAmounts[endCurrency]}</span> {endCurrency}</p>
    </main>
  );
}
