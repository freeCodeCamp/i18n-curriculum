# --description--

Hadi sasa, umekuwa na udhibiti mdogo juu ya uwasilishaji na muonekano wa maudhui yako. Ili kubadilisha hilo, ongeza kipengele cha `style` ndani ya kipengele cha `head`.

# --hints--

Msimbo wako unapaswa kuwa na lebo ya kufungua ya `<style>`.

```js
assert(code.match(/<style\s*>/i));
```

Msimbo wako unapaswa kuwa na lebo ya kufunga ya `</style>`.

```js
assert(code.match(/<\/style\s*>/));
```

Kipengele chako cha `style` kinapaswa kupangwa ndani ya kipengele chako cha `head`.

```js
assert(code.match(/<head\s*>[\w\W\s]*<style\s*>[\w\W\s]*<\/style\s*>[\w\W\s]*<\/head\s*>/i))
```
