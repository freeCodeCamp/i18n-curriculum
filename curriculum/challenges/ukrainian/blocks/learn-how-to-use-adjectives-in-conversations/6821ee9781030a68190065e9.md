# --explanation--

`Need to` використовується, коли щось є необхідним. Наприклад:  

`We need to finish the report by Friday.` – Важливо завершити звіт до п’ятниці.

`Compare` означає досліджувати речі, щоб знайти схожості або відмінності. Наприклад:  

`Let's compare both designs before we choose one.` – Команда хоче побачити, що схоже або відрізняється у дизайнах.

`Versions` означають різні форми або оновлення одного й того самого. Наприклад:  

`The latest version of the app has a new feature.` – Це означає, що новіша версія застосунку містить щось нове.
