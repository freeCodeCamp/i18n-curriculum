# --description--

Nambari asilia, `N`, ambayo inaweza kuandikwa kama jumla na kuzidisha kwa seti fulani ya angalau nambari mbili asilia, $\\{a_1, a_2, \ldots , a_k\\}$ huitwa nambari ya kuzidisha-na-kuzidisha: $N = a_1 + a_2 + \cdots + a_k = a_1 × a_2 × \cdots × a_k$.

Kwa mfano, 6 = 1 + 2 + 3 = 1 × 2 × 3.

Kwa seti ya ukubwa fulani, `k`, tutaita N ndogo kabisa yenye sifa hii nambari ya chini ya kuzidisha-na-kuzidisha. Nambari ndogo za kuzidisha-na-kuzidisha kwa seti za ukubwa, `k` = 2, 3, 4, 5, na 6 ni kama ifuatavyo.

<div style='margin-left: 4em;'>
  <var>k</var>=2: 4 = 2 × 2 = 2 + 2<br>
  <var>k</var>=3: 6 = 1 × 2 × 3 = 1 + 2 + 3<br>
  <var>k</var>=4: 8 = 1 × 1 × 2 × 4 = 1 + 1 + 2 + 4<br>
  <var>k</var>=5: 8 = 1 × 1 × 2 × 2 × 2  = 1 + 1 + 2 + 2 + 2<br>
  <var>k</var>=6: 12 = 1 × 1 × 1 × 1 × 2 × 6 = 1 + 1 + 1 + 1 + 2 + 6
</div><br>

Kwa hivyo kwa 2 ≤ `k` ≤ 6, jumla ya nambari zote ndogo za kuzidisha-na-kuzidisha ni 4 + 6 + 8 + 12 = 30; kumbuka kwamba `8` inahesabiwa mara moja tu katika jumla.

Kwa kweli, kama seti kamili ya nambari ndogo za kuzidisha-na-kuzidisha kwa 2 ≤ `k` ≤ 12 ni $\\{4, 6, 8, 12, 15, 16\\}$, jumla ni `61`.

Ni jumla gani ya nambari zote ndogo za kuzidisha-na-kuzidisha kwa 2 ≤ `k` ≤ `limit`?

function productSumNumbers(limit) {
  function getProductSums(curProduct, curSum, factorsCount, start) {
    const k = curProduct - curSum + factorsCount;
    if (k <= limit) {
      if (curProduct < minimalProductSums[k]) {
        minimalProductSums[k] = curProduct;
      }
      for (let i = start; i < Math.floor((limit / curProduct) * 2) + 1; i++) {
        getProductSums(curProduct * i, curSum + i, factorsCount + 1, i);
      }
    }
  }

  const minimalProductSums = new Array(limit + 1).fill(2 * limit);
  getProductSums(1, 1, 1, 2);

  const uniqueProductSums = [...new Set(minimalProductSums.slice(2))];

  let sum = 0;
  for (let i = 0; i < uniqueProductSums.length; i++) {
    sum += uniqueProductSums[i];
  }

  return sum;
}
