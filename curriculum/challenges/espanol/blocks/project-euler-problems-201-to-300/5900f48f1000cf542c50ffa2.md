# --description--

Un número primo $p$ se llama primo de Panaitopol si $p = \frac{x^4 - y^4}{x^3 + y^3}$ para algunos enteros positivos $x$ e $y$.

Encuentra cuántos primos Panaitopol son menores que $5 × {10}^{15}$.
