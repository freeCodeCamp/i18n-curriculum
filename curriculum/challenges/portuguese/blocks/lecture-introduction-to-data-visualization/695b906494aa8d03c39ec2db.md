# --description--

Cada indústria, da música à saúde, trabalha com conjuntos de dados grandes e frequentemente complexos. Estes podem incluir registros médicos de pacientes, tendências de playlists ao longo do tempo ou métricas de engajamento em redes sociais.

Para entender essas informações, as empresas usam uma variedade de ferramentas para ajudar suas equipes a analisar dados e tomar decisões informadas sobre seus produtos e serviços.

Uma das ferramentas mais poderosas para entender e comunicar esses dados é a **visualização de dados**.

A visualização de dados cria formatos visuais claros a partir de dados complexos. Exemplos desses tipos de visuais incluem gráficos de pizza, gráficos de dispersão, gráficos de barras e mais. Boas visualizações ajudam as empresas a economizar tempo identificando informações-chave, tendências e relacionamentos nos dados.

Existem muitas ferramentas populares de visualização de dados usadas em negócios, incluindo Microsoft Excel, Power BI, Google Charts e Tableau.

Nessas próximas aulas e workshops, você aprenderá como criar visualizações interativas usando a biblioteca JavaScript `D3.js`.

O objetivo deste módulo é apresentar a você os fundamentos da visualização de dados e guiá-lo na construção de suas próprias visualizações com JavaScript.
