# --description--

オブジェクトは基本的にキーと値のペアの集合です。つまり、ユニークな識別子であるプロパティ（キー）に対応するデータ（値）がマッピングされています。例を見てみましょう。

```js
const tekkenCharacter = {
  player: 'Hwoarang',
  fightingStyle: 'Tae Kwon Doe',
  human: true
};
```

上記のコードは、`tekkenCharacter`という鉄拳のビデオゲームキャラクターオブジェクトを定義しています。3つのプロパティがあり、それぞれ特定の値にマッピングされています。もし「origin」という追加のプロパティを加えたい場合は、オブジェクトに`origin`を代入することで可能です。

```js
tekkenCharacter.origin = 'South Korea';
```

これはドット記法を使っています。`tekkenCharacter`オブジェクトを確認すると、`origin`プロパティが追加されているのがわかります。ホワランは特徴的なオレンジ色の髪も持っていました。このプロパティはブラケット記法で次のように追加できます。

```js
tekkenCharacter['hair color'] = 'dyed orange';
```

ブラケット記法は、プロパティ名にスペースが含まれている場合や、変数を使ってプロパティ名を指定したい場合に必要です。上記の例では、プロパティ名が文字列として認識されるように引用符で囲まれており、そのまま追加されます。引用符がなければ変数として評価され、プロパティ名は変数の値になります。変数を使った例を示します。

```js
const eyes = 'eye color';

tekkenCharacter[eyes] = 'brown';
```

すべての例を追加した後、オブジェクトは次のようになります。

```js
{
  player: 'Hwoarang',
  fightingStyle: 'Tae Kwon Doe',
  human: true,
  origin: 'South Korea',
  'hair color': 'dyed orange',
  'eye color': 'brown'
};
```

# --instructions--

`foods`オブジェクトが3つのエントリーで作成されています。お好きな構文を使って、さらに3つのエントリーを追加してください：`bananas`に`13`の値、`grapes`に`35`の値、`strawberries`に`27`の値を代入してください。

# --hints--

```js
assert(typeof foods === 'object');
```

`foods`オブジェクトは、キー`bananas`に値`13`を持つ必要があります。

```js
assert(foods.bananas === 13);
```

`foods`オブジェクトは、キー`grapes`に値`35`を持つ必要があります。

```js
assert(foods.grapes === 35);
```

`foods`オブジェクトは、キー`strawberries`に値`27`を持つ必要があります。

```js
assert(foods.strawberries === 27);
```

`foods`オブジェクトの定義は変更しないでください。
