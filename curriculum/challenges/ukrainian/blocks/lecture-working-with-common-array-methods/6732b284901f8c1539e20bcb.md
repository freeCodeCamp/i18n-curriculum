# --interactive--

Метод `splice()` у JavaScript — це потужний спосіб модифікації масивів. Він дозволяє додавати або видаляти елементи з будь-якої позиції в масиві, включно з серединою. Повернене значення методу `splice()` буде масивом елементів, видалених з масиву. Якщо нічого не було видалено, повернеться порожній масив.

Важливо зазначити, що цей метод змінює оригінальний масив, модифікуючи його на місці, а не створюючи новий масив. Це слід враховувати при роботі з `splice()`. Ось базовий синтаксис:

```js
array.splice(startIndex, itemsToRemove, item1, item2)
```

`startIndex` вказує індекс, з якого починається модифікація масиву, тоді як `itemsToRemove` — це необов’язковий параметр, що вказує, скільки елементів видалити. Якщо `itemsToRemove` опущено, `splice()` видалить усі елементи від `startIndex` до кінця масиву. Наступні параметри (`item1`, `item2` і так далі) — це елементи, які будуть додані до масиву, починаючи з індексу початку.

Почнемо з прикладу видалення елементів із середини масиву:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango", "kiwi"];
let removed = fruits.splice(2, 2);

console.log(fruits);  // ["apple", "banana", "kiwi"]
console.log(removed); // ["orange", "mango"]
```

:::

У цьому прикладі `splice(2, 2)` починається з індексу `2` і видаляє `2` елементів. Змінений масив тепер міститиме лише `apple`, `banana` і `kiwi`. Тепер подивимося, як додати елементи до середини масиву:

:::interactive_editor

```js
let colors = ["red", "green", "blue"];
colors.splice(1, 0, "yellow", "purple");

console.log(colors); // ["red", "yellow", "purple", "green", "blue"]
```

:::

Тут `splice(1, 0, "yellow", "purple")` починається з індексу `1`, видаляє `0` елементів і вставляє `yellow` та `purple`. Другий параметр (`0`) означає, що перед вставкою елементи не видаляються. Ви також можете одночасно видаляти та додавати елементи за допомогою `splice()`:

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];
numbers.splice(1, 2, 6, 7, 8);

console.log(numbers); // [1, 6, 7, 8, 4, 5]
```

:::

У цьому випадку `splice(1, 2, 6, 7, 8)` починається з індексу `1`, видаляє `2` елементів (`2` і `3`) і вставляє `6`, `7` і `8`. Якщо потрібно зберегти оригінальний масив без змін, слід створити копію перед використанням `splice()`:

:::interactive_editor

```js
let original = [1, 2, 3, 4, 5];
let copy = [...original];
copy.splice(2, 1, 6);

console.log(original); // [1, 2, 3, 4, 5]
console.log(copy);     // [1, 2, 6, 4, 5]
```

:::

У цьому прикладі, щоб створити копію масиву `original` без його модифікації, ми використовуємо оператор розширення (`...`). Оператор розширення створить поверхневу копію елементів масиву `original` у новий масив. Ви дізнаєтеся більше про це в майбутніх уроках.

Коли ми використовуємо `copy.splice(2, 1, 6)`, він змінює копію масиву, видаляючи елемент за індексом `2` (який є `3`) і вставляючи новий елемент `6` на цю позицію.

Одне з поширених застосувань `splice()` — видалити один елемент із масиву, якщо відомий його індекс:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango"];
let indexToRemove = fruits.indexOf("orange");
if (indexToRemove !== -1) {
    fruits.splice(indexToRemove, 1);
}

console.log(fruits); // ["apple", "banana", "mango"]
```

:::

У цьому прикладі ми спочатку використовуємо метод `indexOf()`, щоб знайти індекс елемента `orange` у масиві `fruits`. Метод `indexOf()` повертає індекс першої появи заданого елемента або `-1`, якщо елемент не знайдено в масиві.

Потім ми порівнюємо `indexToRemove` з `-1`, щоб переконатися, що елемент існує в масиві, перш ніж намагатися його видалити. Якщо `indexToRemove` не дорівнює `-1` (тобто елемент знайдено), ми використовуємо `splice()`, щоб видалити один елемент, починаючи з позиції `indexToRemove`.

Ви також можете використати `splice()`, щоб очистити масив, видаливши всі елементи:

:::interactive_editor

```js
let array = [1, 2, 3, 4, 5];
array.splice(0);

console.log(array); // []
```

:::

Хоча `splice()` є потужним, варто зазначити, що для дуже великих масивів він може бути менш ефективним за інші методи, особливо при модифікації початку масиву. Це тому, що `splice()` може знадобитися зсунути всі наступні елементи. У таких випадках, якщо ви додаєте або видаляєте елементи лише в кінці масиву, методи на кшталт `push()`, `pop()`, `unshift()` і `shift()` можуть бути більш доречними.

Підсумовуючи, метод `splice()` — це універсальний спосіб модифікації масивів у JavaScript. Він дозволяє точно контролювати додавання та видалення елементів з будь-якої позиції в масиві. Розуміння ефективного використання `splice()` значно покращить вашу здатність маніпулювати масивами у вашому JavaScript-коді.
