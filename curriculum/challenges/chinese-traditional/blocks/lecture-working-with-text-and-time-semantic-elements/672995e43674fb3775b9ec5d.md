# --interactive--

聯絡位址元素用於表述網頁某區段的聯絡資訊。`address` 元素用途多元，可用於商務頁面、作者頁面、個人網站等。

當你在建置你網站的聯絡區塊時，你應該使用語意化的 `address` 元素，而非像 `div` 這類泛型元素。

以下是使用 `address` 元素於公司聯絡頁面的範例：

:::interactive_editor

```html
<address>
  <h2>Company Name</h2>
  <p>
    1234 Elm Street<br />
    Springfield, IL 62701<br />
    United States
  </p>
  <p>Phone: <a href="tel:+15555555555">+1 (555) 555-5555</a></p>
  <p>Email: <a href="mailto:contact@company.com">contact@company.com</a></p>
</address>
```

:::

在此範例中，有公司名稱、實體的位址、電話和電子郵件資訊。對於實體的位址，使用換行元素 `br` 來顯示街道名稱、城市和國家之間的區隔。

對於電話號碼，我們有一個錨點元素，其 `href` 屬性值設定為電話號碼。`href` 屬性中的 `tel:+` 值會在某些支援該功能的裝置上產生可點擊的連結，以啟動電話呼叫。

對於電子郵件位址，使用另一個錨點元素，其 `href` 值設定為 `mailto` 聯結。`mailto` 聯結用於 HTML 文件中，讓使用者能在其偏好的電子郵件客戶端中開啟新的電子郵件。

使用 `mailto` 聯結的一個缺點是使用者經常將它視為垃圾郵件。不幸的是，許多垃圾郵件發送者會使用這個選項來寄送電子郵件給使用者。所以當你使用它時，請記得這一點。
