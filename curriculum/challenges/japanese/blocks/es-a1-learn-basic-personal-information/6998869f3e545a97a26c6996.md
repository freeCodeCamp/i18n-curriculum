# --explanation--

完全な質問は `¿Cuál es tu nombre y apellido?` です。

- `¿Cuál...?` – 特定の情報を尋ねます。

- `es` – 動詞 `ser` から派生し、名詞に一致します。

- `tu` – 聞き手に属する情報を示す所有形容詞です。

この構造は個人情報を尋ねるときによく使われます。例えば：

`¿Cuál es tu número?` - あなたの番号は何ですか？
