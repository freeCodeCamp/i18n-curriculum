# --description--

Gegeben sind die Ausgangsposition eines Snowboarders und eine Drehung in Grad. Bestimmen Sie die Landeposition.

- Die Position eines Snowboarders ist entweder `"Regular"` oder `"Goofy"`.
- Drehungen bei Tricks sind Vielfache von 90 Grad. Positive Werte bedeuten eine Drehung im Uhrzeigersinn, negative eine gegen den Uhrzeigersinn.
- Die Landeposition wechselt alle 180 Grad Drehung.

Zum Beispiel: Bei `"Regular"` und `90` geben Sie `"Regular"` zurück. Bei `"Regular"` und `180` Grad geben Sie `"Goofy"` zurück.

```py
def get_landing_stance(start_stance, rotation):
    flips = abs(rotation) // 180
    if flips % 2 == 0:
        return start_stance
    else:
        return "Goofy" if start_stance == "Regular" else "Regular"
```
