# --description--

$R(a, b, c)$ を、辺の長さが $a$、$b$、$c$ の三角形の内部に重ならずに配置できる3つの円が覆う最大のエリアとします。

$S(n)$ を、$1 ≤ a ≤ b ≤ c < a + b ≤ n$ を満たすすべての整数の組 $(a, b, c)$ に対する $R(a, b, c)$ の平均値とします。

$S(2) = R(1, 1, 1) ≈ 0.31998$、$S(5) ≈ 1.25899$ が与えられています。

$S(1803)$ を小数点以下5桁で四捨五入して求めてください。
