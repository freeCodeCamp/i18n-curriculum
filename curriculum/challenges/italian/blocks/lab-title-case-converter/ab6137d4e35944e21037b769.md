# --description--

In questo laboratorio creerai una funzione che converte una stringa in maiuscolo in stile titolo. Il maiuscolo in stile titolo significa che la prima lettera di ogni parola è maiuscola e il resto della parola è in minuscolo.

`"Web Development Is Awesome"` è un esempio di stringa in maiuscolo in stile titolo.

**Obiettivo**: soddisfa le storie utente qui sotto e fai superare tutti i test per completare il laboratorio.

**Storie utente:**

1. Dovresti avere una funzione `titleCase` che prende una stringa come argomento.
1. La funzione `titleCase` dovrebbe restituire una stringa con la prima lettera di ogni parola maiuscola e il resto della parola in minuscolo.
1. `titleCase("I like to code")` dovrebbe restituire `"I Like To Code"`.
1. `titleCase("javaScript is fun")` dovrebbe restituire `"Javascript Is Fun"`.

# --solutions--

```js
function titleCase(str) {
  return str
    .split(' ')
    .map(word => word.charAt(0).toUpperCase() + word.substring(1).toLowerCase())
    .join(' ');
}
```
