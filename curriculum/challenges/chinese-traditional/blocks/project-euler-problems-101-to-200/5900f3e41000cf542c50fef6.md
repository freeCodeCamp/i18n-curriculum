# --description--

數字 512 很有趣，因為它相等於其數字和的某個次方：$5 + 1 + 2 = 8$，且 $8^3 = 512$。另一個具有此屬性的數字範例是 $614656 = 28^4$。

我們將定義 $a_n$ 為此序列的第 $n$ 項，並堅持數字必須至少包含兩位數才能有總和。

已知 $a_2 = 512$ 和 $a_{10} = 614656$。

尋找 $a_{30}$。
