# --interactive--

Метод `.toFixed()` — це вбудована функція JavaScript, яка форматує число, використовуючи фіксовану десяткову нотацію. Він особливо корисний, коли потрібно контролювати кількість десяткових знаків у числі, зокрема для відображення валютних значень або при роботі з точними вимірами.

Метод `.toFixed()` викликається на числі і приймає один необов’язковий аргумент — кількість знаків після десяткової крапки. Він повертає рядкове представлення числа з вказаною кількістю десяткових знаків. Ось базовий приклад того, як працює `.toFixed()`:

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed(2)); // "3.14"
```

:::

У цьому випадку ми обмежуємо кількість десяткових знаків двома. Тож `3.14159` стає `3.14`. Важливо зауважити, що `.toFixed()` повертає рядок, а не число. Це тому, що метод призначений насамперед для форматування чисел для відображення, а не для подальших обчислень.

Метод `.toFixed()` округлює число до найближчого значення, яке можна представити з вказаною кількістю десяткових знаків. Це округлення важливо розуміти:

:::interactive_editor

```js
console.log((3.14159).toFixed(3));  // "3.142"
console.log((3.14449).toFixed(3));  // "3.144"
console.log((3.14550).toFixed(3));  // "3.146"
```

:::

Як бачите, `.toFixed()` округлює вгору, коли наступна цифра `5` або більша, і округлює вниз в іншому випадку. Якщо викликати `.toFixed()` без аргументів, він за замовчуванням використовує `0` десяткових знаків:

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed()); // "3"
```

:::

Метод `.toFixed()` може бути особливо корисним при роботі з фінансовими обчисленнями або відображенні цін:

:::interactive_editor

```js
let price = 19.99;
let taxRate = 0.08;
let total = price + (price * taxRate);

console.log("Total: $" + total.toFixed(2)); // "Total: $21.59"
```

:::

У цьому прикладі `.toFixed(2)` гарантує, що загальна сума завжди відображається з двома десятковими знаками, що є стандартом для валюти у багатьох країнах.

Підсумовуючи, метод `.toFixed()` — це потужний інструмент для форматування чисел у JavaScript, особливо коли потрібно контролювати відображення десяткових знаків. Хоча він здебільшого використовується для форматування вихідних даних, пам’ятайте про його поведінку, особливо коли потрібні точні обчислення.
