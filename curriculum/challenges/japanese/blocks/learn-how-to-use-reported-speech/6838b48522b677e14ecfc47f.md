# --explanation--

`Past Continuous` 時制（`was/were` + 動詞の `-ing` 形）は、過去のある期間にわたって続いていた動作を表します。これは、継続的な動作が中断されたり、別のイベントの前に起こったことを示すために、`Simple Past` イベントと一緒に使われることが多いです。例えば：

`She was working on the report when the computer crashed.` – クラッシュ（`Simple Past`）が起こったとき、作業の動作が続いていました。
