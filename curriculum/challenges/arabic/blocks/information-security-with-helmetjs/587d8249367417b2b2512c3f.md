يبرز هذا التحدٍّ دفاعًا جديدًا واعدًا يمكنه تقليل المخاطر والتأثيرات للعديد من أنواع الهجمات في المتصفحات الحديثة بشكل كبير. من خلال تعيين وتكوين سياسة أمان المحتوى يمكنك منع حقن أي شيء غير مقصود في صفحتك. هذا يحمي تطبيقك من ثغرات XSS، والتتبع غير المرغوب فيه، والإطارات الخبيثة، وأكثر من ذلك. تعمل سياسة أمان المحتوى عن طريق تحديد قائمة مصادر محتوى مسموح بها وموثوقة. يمكنك تكوينها لكل نوع من الموارد التي قد تحتاجها صفحة الويب (السكريبتات، أوراق الأنماط، الخطوط، الإطارات، الوسائط، وهكذا...). هناك توجيهات متعددة متاحة، بحيث يمكن لمالك الموقع التحكم بدقة. راجع HTML 5 Rocks و KeyCDN لمزيد من التفاصيل. للأسف، سياسة أمان المحتوى غير مدعومة في المتصفحات القديمة.

افتراضيًا، تكون التوجيهات مفتوحة على مصراعيها، لذلك من المهم تعيين توجيه defaultSrc كنظام بديل. يدعم Helmet كلا من أسلوبي التسمية defaultSrc و default-src. ينطبق النظام البديل على معظم التوجيهات غير المحددة.

# --instructions--

في هذا التمرين، استخدم `helmet.contentSecurityPolicy()`. قم بتكوينه بإضافة كائن `directives`. في هذا الكائن، عيّن `defaultSrc` إلى `["'self'"]` (يجب أن تكون قائمة المصادر المسموح بها داخل مصفوفة)، لكي تثق فقط بعنوان موقعك افتراضيًا. كما عيّن توجيه `scriptSrc` بحيث تسمح فقط بتنزيل السكريبتات من موقعك (`'self'`) ومن النطاق `'trusted-cdn.com'`.

تلميح: في الكلمة المفتاحية `'self'`، علامات الاقتباس المفردة جزء من الكلمة نفسها، لذلك يجب أن تُحاط بـ علامات اقتباس مزدوجة لتعمل بشكل صحيح.

# --hints--

يجب تركيب برمجيات وسيطة helmet.contentSecurityPolicy() بشكل صحيح

```js
const response = await fetch(code + '/_api/app-info');
if (!response.ok) {
  throw Error(await response.text());
}
const data = await response.json();
assert.include(data.appStack, 'csp');
```

تكوين سياسة أمان المحتوى الخاص بك غير صحيح. يجب أن يكون defaultSrc هو ["'self'"] و scriptSrc هو ["'self'", 'trusted-cdn.com']

```js
const response = await fetch(code + '/_api/app-info');
if (!response.ok) {
  throw Error(await response.text());
}
const data = await response.json();
var cspHeader = Object.keys(data.headers).filter(function (k) {
  return (
    k === 'content-security-policy' ||
    k === 'x-webkit-csp' ||
    k === 'x-content-security-policy'
  );
})[0];
assert.equal(
  data.headers[cspHeader],
  "default-src 'self'; script-src 'self' trusted-cdn.com"
);
```
