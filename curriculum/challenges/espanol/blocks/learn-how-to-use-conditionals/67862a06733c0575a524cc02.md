# --explanation--

`Definitely` se usa para expresar un acuerdo fuerte o certeza. Significa `without a doubt` o `absolutely`. Por ejemplo:

- `This plan is definitely the best option.` - Muestra completa certeza de que el plan es la mejor opción.

- `I'll definitely be at the meeting tomorrow.` - Expresa confianza en que el hablante asistirá a la reunión.

En este contexto, Tom lo usa para estar completamente de acuerdo con la afirmación de Lisa sobre el impacto significativo de que la empresa cierre.
