# --interactive--

У JavaScript тип даних — це вид значення, яке ви зберігаєте, наприклад число або текст.

Змінна — це іменований контейнер, який зберігає значення певного типу даних, що дозволяє вам посилатися на нього та маніпулювати ним у вашому коді.

Ви, можливо, пам’ятаєте зі шкільного курсу математики роботу зі змінними так:

```md
x = 2
y = 4

x + y
```

Ви могли створювати змінні, як `x` і `y`, а потім посилатися на них у програмі та виконувати математичні операції, наприклад додавання. Це схоже на концепцію в програмуванні. Ви можете створювати власні імена змінних і призначати їм значення. Ці значення будуть різних типів даних.

Типи даних допомагають програмі зрозуміти, з яким видом даних вона працює — числом, текстом чи чимось іншим.

У JavaScript є кілька базових типів даних, які ви будете використовувати у своїх програмах. Ми детальніше розглянемо кожен тип даних у майбутніх уроках. А поки що ось коротке введення в різні типи даних у JavaScript.

Перший тип даних, який ми розглянемо, — це тип `Number`.

`Number` представляє як цілі числа, так і числа з плаваючою комою. Прикладами цілих чисел є `7`, `19` і `90`.

**ПРИМІТКА**: `console.log()` — це функція, яка виводить інформацію в консоль, що є частиною вашого веб-браузера для налагодження коду. Ви дізнаєтеся більше про `console.log()` у майбутніх уроках. Також символи `//` використовуються для додавання коментарів у код. Коментарі — це нотатки для себе або інших програмістів, які ігноруються під час виконання коду.

Увімкніть інтерактивний редактор і спробуйте змінити деякі цілі числа, щоб побачити оновлення в консолі.

:::interactive_editor

```js
// Examples of integers
console.log(3);
console.log(5);
console.log(-67);
```

:::

Число з плаваючою комою — це число з десятковою крапкою. Прикладами таких чисел є `3.14` і `5.2`.

:::interactive_editor

```js
// Examples of floating point numbers
console.log(3.14);
console.log(7.2);
console.log(-14.5);
```

:::

Наступний тип даних — це `String`.

`String` — це послідовність символів або текст, укладений у лапки. Ось приклад рядка з подвійними лапками:

:::interactive_editor

```js
console.log("I love to code!");
```

:::

Ось приклад із одинарними лапками:

:::interactive_editor

```js
console.log('I love to code!');
```

:::

Часто рядки використовують для представлення імен, міток, повідомлень тривоги тощо.

Ще один тип даних у JavaScript — це тип `Boolean`.

`Boolean` представляє одне з двох значень: `true` або `false`. Наприклад, програма може перевіряти, чи користувач увійшов у систему (`true`) або ні (`false`) і змінювати сторінку залежно від цього. Якщо користувач увійшов, ви, ймовірно, хочете показати йому сторінку панелі керування. Інакше ви покажете сторінку входу.

Наступні два типи даних у JavaScript — це `undefined` і `null`.

`undefined` означає, що змінна оголошена, але їй ще не призначено значення. Ви дізнаєтеся більше про це в наступному уроці.

`null` означає, що змінна навмисно встановлена в "нічого" і не містить жодного значення. Ми розглянемо це докладніше у майбутніх уроках.

Останні три типи даних є більш складними за своєю природою. Це об’єкти, символи та `BigInt`.

`Object` — це колекція пар ключ-значення.

```js
{
  name: "Alice",
  age: 30
};
```

Об’єкти чудово підходять для групування пов’язаної інформації. Ви дізнаєтеся більше про роботу з об’єктами в майбутньому модулі.

`Symbol` — це спеціальний тип значення в JavaScript, який завжди унікальний і не може бути змінений. Його часто використовують для створення унікальних міток або ідентифікаторів для властивостей:

```js
Symbol('mySymbol');
```

`BigInt` використовується для дуже великих чисел, які перевищують межі типу `Number`:

```js
1234567890123456789012345678901234567890n;
```

У цьому прикладі ми створюємо `BigInt`, додавши `n` наприкінці дуже великого числа.

`Symbol` і `BigInt` — це два типи, які використовуються рідше, але про них також важливо знати.

Розуміння цих типів даних допомагає вам обробляти та працювати з різними видами даних у ваших програмах, оскільки кожен тип має свої характеристики та поведінку.
