# --description--

تجمع عملية التعيين المعزز بين عملية ثنائية ومع عملية التعيين في خطوة واحدة. تأخذ متغيرًا، وتطبق عليه عملية مع قيمة أخرى، ثم تخزن النتيجة مرة أخرى في نفس المتغير.

إذا كنت تعرف لغة مثل JavaScript، فمن المحتمل أنك سمعت عن معامل التعيين بالإضافة (`+=`) أو معامل التعيين بالطرح (`-=`)، وغيرها. هذه المعاملات موجودة أيضًا في Python. الفرق الوحيد هو أنها تُسمى **عمليات التعيين المعززة**.

الصيغة النحوية الأساسية لعملية التعيين المعزز تبدو هكذا:

```python
variable <operator>= value
```

وهي طريقة أكثر كفاءة للقيام بهذا:

```python
variable = variable <operator> value
```

على سبيل المثال، إليك مثالًا على استخدام التعيين المعزز لإضافة `5` إلى متغير موجود:

```python
my_var = 10
my_var += 5

print(my_var) # 15
```

وهنا نفس الشيء، ولكن بدون استخدام التعيين المعزز:

```python
my_var = 10
my_var = my_var + 5

print(my_var) # 15
```

ميزة التعيين المعزز أنه يوفر طريقة مختصرة وقابلة للقراءة لتحديث قيمة متغير دون تكرار اسم المتغير. وهذا يقلل التكرار والأخطاء المحتملة التي قد تنشأ من خطأ مطبعي أو شيء مشابه.

يمكن لكل معامل استخدام التعيين المعزز. لقد نظرنا إلى معامل التعيين بالإضافة (`+=`)، فلنلق نظرة على الآخرين.

- معامل التعيين بالطرح (`-=`) يطرح المعامل الموجود على اليمين من المتغير الموجود على اليسار ويخزن الفرق في المتغير الأيسر:

```python
count = 14
count -= 3

print(count) # 11
```

- معامل التعيين بالضرب (`*=`) يضرب المتغير الأيسر بالمعامل الموجود على اليمين ويخزن الناتج مرة أخرى في المتغير الأيسر:
    

```python
product = 65
product *= 7

print(product) # 455
```

- معامل التعيين بالقسمة (`/=`) يقسم المتغير الأيسر على المعامل الموجود على اليمين ويخزن النتيجة مرة أخرى في المتغير الأيسر:
    

```python
price = 100
price /= 4

print(price) # 25.0
```

- معامل التعيين بالقسمة الأرضية (`//=`) يقسم المتغير الأيسر قسمة أرضية على المعامل الموجود على اليمين ويخزن النتيجة مرة أخرى في المتغير الأيسر:
    

```python
total_pages = 23
total_pages //= 5

print(total_pages) # 4
```

- معامل التعيين بباقي القسمة (`%=`) يحسب باقي قسمة المتغير الأيسر على المعامل الموجود على اليمين ويخزنه مرة أخرى في المتغير الأيسر:
    

```python
bits = 35
bits %= 2

print(bits) # 1
```

- معامل التعيين بالأس (`**=`) يرفع المتغير الأيسر إلى قوة المعامل الموجود على اليمين ويخزن النتيجة مرة أخرى في المتغير الأيسر:
    

```python
power = 2
power **= 3

print(power) # 8
```

يمكنك استخدام بعض معاملات التعيين المعزز مع السلاسل النصية أيضًا. على سبيل المثال، معامل التعيين بالإضافة يجعل من السهل ربط السلاسل النصية:

```python
greet = 'Hello'
greet += ' World'

print(greet) # Hello World
```

ومعامل التعيين بالضرب يمكن استخدامه لتكرار سلسلة نصية:

```python
greet = 'Hello'
greet *= 3

print(greet) # HelloHelloHello
```

عمليات التعيين المعزز الأخرى ترمي `TypeError` عند استخدامها مع السلاسل النصية:

```python
greet = 'Hello'
greet -= ' World'

print(greet) # TypeError: unsupported operand type(s) for -=: 'str' and 'str'


greet = 'Hello'
greet /= 'World'

print(greet) # TypeError: unsupported operand type(s) for /=: 'str' and 'str' 
```

إذا كنت تتساءل هل يعمل معاملا الزيادة والنقصان (`++` و `--`) في Python، فالجواب لا. ذلك لأن Python تتجنب عمدًا اختصارات الزيادة والنقصان على طريقة C للحفاظ على وضوح اللغة وصراحتها.

بدلًا من `x++`، يمكنك ببساطة كتابة `x += 1`، مما يوضح أنك تزيد قيمة `x` بمقدار `1`.

كتابة `++x` في Python تطبق فقط علامة الجمع الأحادية مرتين، ولا تزيد أي شيء:

```python
my_var = 5

print(+my_var)   # 5
print(++my_var)  # 5
print(+++my_var) # 5

my_var += 1

print(my_var) # 6
```
