# --description--

In questa attività imparerai frasi come `Did you get it?`, `Got it` e `What else?`. Queste frasi sono comuni nelle conversazioni in inglese. `Did you get it?` è una domanda che chiede se qualcuno ha capito qualcosa. `Got it` è un modo breve per dire `I understood.`. `What else?` si usa per chiedere informazioni o elementi aggiuntivi.

Per esempio, in una riunione, qualcuno potrebbe dire `Did you get it?` e tu puoi rispondere `Got it, what else do we need to discuss?`.
