# --interactive--

النسب المئوية في CSS هي وحدات نسبية تتيح لك تحديد الأحجام والأبعاد وخصائص أخرى كنسبة من العنصر الأصل. عند استخدام قيمة نسبة مئوية، فأنت تقول أساسًا: "اجعل هذا العنصر يشغل X% من حاويته."

على سبيل المثال، إذا ضبطت `width: 50%` على عنصر، فسيشغل نصف عرض الحاوية الأصلية له. هذا يجعل النسب المئوية مفيدة جدًا لإنشاء تصميمات متجاوبة تتكيف مع أحجام الشاشات المختلفة.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <div class="box"></div>
</div>
```

```css
.container {
  width: 400px;
  height: 200px;
  background-color: lightgray;
}

.box {
  width: 50%;
  height: 100%;
  background-color: red;
}
```

:::

النسب المئوية مثالية لإنشاء تخطيطات مرنة تتكيف مع أحجام الشاشات المختلفة. على سبيل المثال، ضبط حاوية على `width: 80%` يضمن أن تشغل 80% من عرض الحاوية الأصلية، بغض النظر عن الجهاز.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="parent">
  <div class="child"></div>
</div>
```

```css
.parent {
  width: 100%;
  height: 300px;
  background-color: lightblue;
}

.child {
  width: 80%;
  height: 100%;
  background-color: red;
}
```

:::

استخدام النسب المئوية للصور المرنة هو ممارسة شائعة أخرى. بتطبيق `max-width: 100%` على الصور، تسمح لها بالتقلص على الشاشات الصغيرة مع الحفاظ على نسبة العرض إلى الارتفاع.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<img src="https://placehold.co/150x150" alt="Example Product Image" />
```

```css
img {
  max-width: 100%;
  height: auto;
}
```

:::

رغم أنه أقل شيوعًا، يمكن أيضًا استخدام النسب المئوية لأحجام الخطوط لإنشاء طباعات قابلة للتوسع. على سبيل المثال، `font-size: 120%` يجعل النص أكبر بنسبة 20% من حجم الخط في الحاوية الأصل.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="text-container">
  Parent text.
  <p class="text">This is some example text.</p>
</div>
```

```css
.text-container {
  font-size: 16px;
}

.text {
  font-size: 120%;
}
```

:::

النسب المئوية مفيدة جدًا للتمركز العمودي. إليك مثالًا على كيفية استخدام النسب مع خاصية `transform` لتمركز عنصر عموديًا.

يضع هذا المثال العنصر على بعد 50% من أعلى الحاوية، ثم يستخدم `transform` لتحريكه للأعلى بمقدار نصف ارتفاعه، مما يحقق تمركزًا عموديًا فعّالًا.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="centered"></div>
```

```css
.centered {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
  width: 300px;
  height: 300px;
  background-color: red;
}
```

:::

سوف تتعلم المزيد عن كيفية عمل التمركز المطلق وخصائص التحويل بتفصيل أكبر في الدروس القادمة.

تذكر، النسب المئوية دائمًا نسبية لشيء ما. بالنسبة للخصائص الأفقية مثل `width`، فهي نسبية لعرض الحاوية الأصل. أما الخصائص العمودية مثل `height`، فعادة ما تكون نسبية لارتفاع الحاوية الأصل (إذا كان محددًا).

لكن كن حذرًا عند تسكين العناصر ذات الأبعاد المعتمدة على النسب المئوية، إذ قد يؤدي ذلك إلى نتائج غير متوقعة. أيضًا، ضع في اعتبارك أن ارتفاعات النسب المئوية قد تكون معقدة إذا لم يكن للحاوية الأصل ارتفاع محدد.

باختصار، النسب المئوية في CSS أدوات قوية لإنشاء تصميمات مرنة ومتجاوبة. استخدمها عندما تريد أن تتناسب العناصر تناسبيًا مع حاوياتها أو عندما تحتاج إلى تخطيطات تتكيف بسلاسة مع أحجام الشاشات المختلفة.
