# --description--

前のレッスンでは、Pythonの一般的なデータ型である文字列、整数、浮動小数点数の型について学びました。プログラムを作成していく中で、変数の型を確認する方法を覚える必要があります。

こちらは例の変数です：

```py
developer = 'Devin'
```

`developer` の型を確認するには、`type()` 関数を次のように使います：

```py
developer = 'Devin'

print(type(developer)) # <class 'str'>
```

`<class 'str'>` の出力は、`developer` が文字列型であることを意味します。

`type()` 関数に引数を渡さないと、次のエラーメッセージが表示されます：

```md
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: type() takes 1 or 3 arguments
```

これまでに学んだすべてのデータ型と、それらの型がターミナルに表示される例は以下の通りです：

```python
my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 'hello', 8.5}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 'hello', 8.5)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>
```

プログラムの中で、特定の変数が特定の型であることを確認してから操作を行う必要がある場合があります。ここで役立つのが `isinstance()` 関数です。

こちらは文字列が代入された例の変数です：

```py
account_balance = '12'
```

`account_balance` 変数を使って割り算などの数学的な式を実行しようとすると、エラーメッセージが表示されます。

```py
account_balance = '12'

account_balance / 2

# Traceback (most recent call last):
#   File "<stdin>", line 1, in <module>
# TypeError: unsupported operand type(s) for /: 'str' and 'int'
```

`account_balance` が整数かどうかを確認するには、`isinstance()` 関数を次のように使います：

```py
account_balance = '12'

isinstance(account_balance, int) # False
```

組み込みの `isinstance()` 関数を使うと、変数が特定のデータ型に一致するかどうかを確認できます。この関数はオブジェクトと確認したい型を受け取り、ブール値を返します。この場合、`account_balance` は文字列なので、`False` が返されます。

`isinstance()` 関数は複数の型を一度にチェックすることもできます。

こちらは `account_balance` が `int` または `float` かどうかを確認する例です：

```py
account_balance = 12
isinstance(account_balance, (int, float)) # True
```

この例では、`account_balance` は整数なので `isinstance()` は `True` を返します。もし `account_balance` が `12.0` であっても、整数または浮動小数点数をチェックしているため `isinstance()` は `True` を返します。

今後のワークショップやラボでは、`type()` と `isinstance()` 関数を使って、操作を行う前に変数が正しいデータ型を持っていることを確認します。
