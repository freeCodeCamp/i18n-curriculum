# --description--

在本課程中，你將學習如何使用函式（程式）型別。讓我們從參數型別註解開始。

這是一個基本函式（程式）的範例：

```ts
function circleArea(radius) {
  return Math.PI * radius * radius;
}
```

目前，TypeScript 會顯示一則訊息為 `Parameter 'radius' implicitly has an 'any' type`。`any` 型別顯示該值可以是任何型別。你將在未來的課程中學習更多關於此型別的內容。

為了消除錯誤並明確地為參數指定型別，你可以這樣指定：

```ts
function circleArea(radius: number) {
  return Math.PI * radius * radius;
}
```

如果你嘗試用非數字的不同型別呼叫 `circleArea` 函式，TypeScript 將會顯示類似這樣的錯誤：

```ts
function circleArea(radius: number) {
  return Math.PI * radius * radius;
}

// Argument of type 'string' is not assignable to parameter of type 'number'
circleArea("3");
```

如果你嘗試提供比函式預期更多的引數，TypeScript 將顯示錯誤：

```ts
function circleArea(radius: number) {
  return Math.PI * radius * radius;
}

// Expected 1 arguments, but got 2.
circleArea(3, 4);
```

如果你提供的引數太少，TypeScript 也會顯示錯誤：

```ts
function circleArea(radius: number) {
  return Math.PI * radius * radius;
}

// Expected 1 arguments, but got 0.
// An argument for 'radius' was not provided.
circleArea();
```

你可以透過在參數附近使用 `?` 符號，使參數成為選用。

```ts
function area(radius: number, height?: number): number {
  const baseArea = Math.PI * radius * radius;

  if (height !== undefined) {
    const sideArea = 2 * Math.PI * radius * height;
    return baseArea + sideArea;
  }

  return baseArea;
}
```

函式內的條件檢查是為了防止未提供 `height` 引數。 

如果你想為你的返回值指定型別，你可以像這樣添加返回型別註解：

```ts
function circleArea(radius: number): number {
  return Math.PI * radius * radius;
}
```

返回類型註解放在參數列之後。類似於變數型別註解，大多數時候 TypeScript 都會足夠聰明地推斷返回類型。因此，當你想讓函式的意圖更明確或防止對返回值的非預期更改時，請使用明確的返回類型註解。

下一個範例將會處理非同步函式（asynchronous functions）和 promises。如果你的函式將要傳回一個 promise，你可以像這樣使用 `Promise` 型別：

```ts
async function getCityTemperature(city: string): Promise<number> {
  const response = await fetch(`https://api.example.com/weather?city=${city}`);
  const data = await response.json(); 
  return data.temperature;
}
```

在這種情況下，如果你嘗試移除返回型別，TypeScript 可能會推斷返回為 `Promise<any>`。所以這會是為返回值明確指定型別的好時機。

到目前為止，我們只看過具名函式的範例。但匿名函式的型別是如何運作的呢？

這裡是一個範例，示範在傳遞給 `forEach` 的回呼中使用匿名函式。

```ts
const languages = ["JavaScript", "Python", "TypeScript"];

languages.forEach((lang) => {
  console.log(`${lang.toUpperCase()} has ${lang.length} characters`);
});
```

`lang` 參數沒有型別註解。取而代之的是，TypeScript 使用背景關係型別，這表示它會從出現的上下文中推斷型別。
