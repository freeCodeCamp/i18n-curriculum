# --explanation--

`Must have happened` se usa para hacer una suposición lógica sobre algo en el pasado.  

`Must` expresa una fuerte certeza. `Have` + `Past Participle` (como `happened`) se refiere a algo en el pasado. Por ejemplo:

- `He must have forgotten to update the system.` - Es probable que no haya recordado hacer la actualización.

- `They must have made some changes to the app.` - Es probable que la app haya sido actualizada o mejorada.
