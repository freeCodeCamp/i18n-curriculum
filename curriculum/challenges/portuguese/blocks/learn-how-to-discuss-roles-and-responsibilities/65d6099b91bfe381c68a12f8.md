# --description--

A palavra `seriously` é um advérbio que significa fazer algo com muito pensamento e atenção, mostrando que é importante para você.É a forma adverbial do adjetivo `serious`.

Por exemplo, quando `someone is serious about a project`, eles acham que é muito importante.Se `they work seriously`, significa que eles trabalham com muito esforço e cuidado.

Para formar um advérbio a partir de um adjetivo, você geralmente adiciona `-ly` ao final do adjetivo.Quando um adjetivo termina em `y`, você precisa mudar o `y` para `i` e adicionar `-ly`.Por exemplo, `happy` se torna `happily`.
