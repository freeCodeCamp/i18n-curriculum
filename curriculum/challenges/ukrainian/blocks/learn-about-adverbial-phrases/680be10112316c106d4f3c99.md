# --explanation--

`To spread` у цьому контексті означає розподіляти щось по ширшій області або між різними частинами. Наприклад:

`The team spread the project tasks evenly among all members.` – Це означає, що роботу було розподілено справедливо, щоб кожна людина мала приблизно однакову кількість завдань.

`Funds` стосується грошей, які доступні для конкретної мети, як-от проєкт, кампанія або подія. Наприклад:

`The company set aside special funds for emergency expenses.` – Це означає, що вони зарезервували гроші, які можна використати лише у випадку несподіваних проблем.
