# --description--

Для кожного цілого числа $n > 1$ сімейство функцій $f_{n, a, b}$ визначається так:

$f_{n, a, b}(x) ≡ ax + b\bmod n$ для $a, b, x$ цілих чисел, де $0 \lt a \lt n$, $0 \le b \lt n$, $0 \le x \lt n$.

Ми назвемо $f_{n, a, b}$ ретракцією, якщо $f_{n, a, b}(f_{n, a, b}(x)) \equiv f_{n, a, b}(x)\bmod n$ для кожного $0 \le x \lt n$.

Нехай $R(n)$ — кількість ретракцій для $n$.

$F(N) = \displaystyle\sum_{n = 1}^N R(n^4 + 4)$.

$F(1024) = 77\\,532\\,377\\,300\\,600$.

Знайдіть $F({10}^7)$. Відповідь подайте за модулем $1\\,000\\,000\\,007$.
