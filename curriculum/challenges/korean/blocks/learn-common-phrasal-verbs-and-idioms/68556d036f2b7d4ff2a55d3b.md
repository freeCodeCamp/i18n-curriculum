# --explanation--

`that`를 `I think` 뒤에 사용하는 것은 이런 문장에서 선택 사항입니다. `I think that it's vital we make these updates.`라고 말하는 것도 맞습니다. 하지만 일상 대화에서는 `that`를 생략하는 것이 더 자연스럽고 빠른 문장이 됩니다. 예를 들어:

`I think that we need more time to finish the project.` – 이 구조는 맞습니다. 하지만 대부분 사람들은 `I think we need more time…`라고 말합니다.
