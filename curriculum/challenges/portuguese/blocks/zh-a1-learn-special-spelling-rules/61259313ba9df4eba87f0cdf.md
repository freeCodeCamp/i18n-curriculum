# --description--

`w` também é um símbolo ortográfico. É usado quando `u` é o som vocálico principal em uma sílaba. Ele ajuda a esclarecer os limites das sílabas e torna as sílabas mais fáceis de ler.

- Se `u` for usado como uma sílaba, adicione `w` na frente dele: `u` → `wu`.

- Se uma sílaba começar com `u`, mude para `w`. Exemplos: `ua` → `wa`, `uo` → `wo`, `uai` → `wai`, `uei` → `wei`, `uan` → `wan`, `ueng` → `weng`.

Vamos praticar a pronúncia das sílabas e depois usar `wan` em diferentes tons como exemplo.

| Pinyin   | Character|
|--------|----------|
| `wān`  | `弯`|
| `wán`   | `完`|
| `wǎn`  | `晚`|
| `wàn`  | `万`|
