## 앵커 요소
HTML에서 링크를 만들려면 앵커 요소를 사용합니다. 앵커 요소는 링크로 만들고 싶은 텍스트나 다른 HTML 요소를 `<a>` 태그로 감싸서 정의합니다.

만든 `body` 페이지의 `index.html`에 다음을 추가하고 브라우저에서 열어 보세요:

```html
<a>click me</a>
```

이 링크를 클릭해도 아무 일도 일어나지 않는 것을 눈치챘을 수 있습니다. 이는 앵커 태그만으로는 어디로 링크할지 알 수 없기 때문입니다. 링크할 목적지를 알려줘야 합니다. 이 작업은 HTML 속성을 사용해 합니다.

HTML 속성은 HTML 요소에 추가 정보를 제공하며 항상 요소의 여는 태그 안에 들어갑니다. 속성은 보통 이름과 값 두 부분으로 이루어지지만, 모든 속성이 값을 필요로 하지는 않습니다. 여기서는 여는 앵커 태그에 `href` (하이퍼링크 참조) 속성을 추가해야 합니다. `href` 속성의 값은 링크가 이동할 목적지입니다.

이전에 만든 앵커 요소에 다음 `href` 속성을 추가하고 다시 클릭해 보세요. 새 변경 사항이 적용되도록 브라우저를 새로 고침하는 것도 잊지 마세요.

```html
<a href="https://www.theodinproject.com/about">click me</a>
```

기본적으로 `href` 속성이 없는 앵커 태그로 감싼 텍스트는 일반 텍스트처럼 보입니다. `href` 속성이 있으면 브라우저가 텍스트에 파란색과 밑줄을 표시해 링크임을 나타냅니다.

앵커 태그는 다른 HTML 문서뿐 아니라 인터넷상의 모든 종류 리소스에 링크하는 데 사용할 수 있다는 점도 알아두세요. 동영상, pdf 파일, 이미지 등에도 링크할 수 있지만, 대부분은 다른 HTML 문서에 링크할 것입니다.
