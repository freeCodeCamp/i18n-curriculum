# --description--

このラボでは、メールアドレスのユーザー名部分をアスタリスクでマスクします。マスキングとは、機密情報をアスタリスクや他の文字で隠したり置き換えたりすることを指します。

例えば、メールアドレスが `myEmail@email.com` の場合、マスクされたメールアドレスは `m*****l@email.com` になります。

**目的:** 以下のユーザーストーリーを満たし、すべてのテストに合格してラボを完了してください。

**ユーザーストーリー:**

1. `maskEmail` を引数に取る `email` という名前の関数を作成してください。
2. 関数内で `email` をマスクし、ドメイン名を付け加えてください。`slice`、`repeat`、`indexOf`、または `replace` のようなメソッドを使うことができます。
3. 関数の外で、マスクしたいメールアドレスを格納する `email` という変数を宣言してください。
4. `maskEmail` 変数を使って `email` 関数を呼び出し、その結果をコンソールに出力してください。
5. `maskEmail("apple.pie@example.com")` は `"a*******e@example.com"` を返すべきです。
6. `maskEmail("freecodecamp@example.com")` は `"f**********p@example.com"` を返すべきです。
7. `maskEmail("info@test.dev")` は `"i**o@test.dev"` を返すべきです。
8. `maskEmail("user@domain.org")` は `"u**r@domain.org"` を返すべきです。

# --solutions--

```js
function maskEmail(email) {
  const atIndex = email.indexOf("@");
  const userName = email.slice(0, atIndex);
  const domain = email.slice(atIndex);

  const maskedName =
    userName[0] + "*".repeat(userName.length - 2) + userName[userName.length - 1] + domain;

  return maskedName;
}

const email = "apple.pie@example.com";
console.log(maskEmail(email));
```
