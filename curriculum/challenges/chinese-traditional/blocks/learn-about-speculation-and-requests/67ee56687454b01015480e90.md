# --explanation--

當你知道迴圈應該執行多少次時，`for loop` 很有用。例如：

- `A for loop is useful for going through a list.`－你可以用它來檢查列表中的每個項目。

- `The for loop stops when it reaches the last item。` - 當沒有更多項目時，迴圈結束。

當重複次數取決於條件時，`while loop` 較為適合。例如：

- `A while loop runs until a condition is false。`－只要規則為真，迴圈就會繼續執行。

- `Be careful, a while loop can run forever!`－如果你不停止它，迴圈不會結束。
