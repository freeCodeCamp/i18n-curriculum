# --explanation--

`Present Continuous` 시제는 말하는 순간에 일어나고 있는 행동을 설명할 때 사용합니다. 동사 `to be` (`am, is, are`) + 동사의 기본형 + `-ing`로 구성됩니다. 예를 들어:

- `She is coding a new feature right now.` - 그녀는 바로 이 순간 새로운 기능을 코딩하는 중입니다.

- `They are testing the software.` - 그들은 지금 소프트웨어를 테스트하는 중입니다.
