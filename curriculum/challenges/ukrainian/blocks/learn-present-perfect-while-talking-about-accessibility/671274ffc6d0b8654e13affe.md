# --explanation--

Час `Present Perfect Continuous` використовується для опису дій, які почалися в минулому і тривають досі. Він фокусується на тривалості або продовженні дії. Структура цього часу — `have/has + been + verb(-ing)`. Наприклад:

- `We have been developing a new app for accessibility.` — це означає, що розробка почалася в минулому і досі триває.

- `He has been fixing bugs in the software for several hours.` — це означає, що він почав виправляти помилки раніше і досі працює над цим.
