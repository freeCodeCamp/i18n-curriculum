# --description--

Nambari kamili mbili $N$ na $M$ husemwa kuwa jozi za kirafiki ikiwa $N \\neq M$ na jumla ya vigawanyo halisi vya $N$ ($\\mathrm{sum}(\\mathrm{propDivs}(N))$) ni $M$ pamoja na $\\mathrm{sum}(\\mathrm{propDivs}(M)) = N$.

**Mfano:**

**1184** na **1210** ni jozi za kirafiki, na vigawanyo halisi ni:

<ul>
  <li>1, 2, 4, 8, 16, 32, 37, 74, 148, 296, 592 na</li>
  <li>1, 2, 5, 10, 11, 22, 55, 110, 121, 242, 605 kwa mtiririko.</li>
</ul>

Jumla ya vigawanyo kwa thamani ya kwanza, **1184**, ni **1210** na jumla ya vigawanyo kwa thamani ya pili, **1210**, ni **1184**.

// amicablePairsUpTo :: Int -> [(Int, Int)]
function amicablePairsUpTo(maxNum) {
  return range(1, maxNum)
    .map(x => properDivisors(x)
      .reduce((a, b) => a + b, 0))
    .reduce((a, m, i, lst) => {
      const n = i + 1;

      return (m > n) && lst[m - 1] === n ?
        a.concat([
          [n, m]
        ]) : a;
    }, []);
}

// properDivisors :: Int -> [Int]
function properDivisors(n) {
  if (n < 2) return [];

  const rRoot = Math.sqrt(n);
  const intRoot = Math.floor(rRoot);
  const blnPerfectSquare = rRoot === intRoot;
  const lows = range(1, intRoot)
  .filter(x => (n % x) === 0);

  return lows.concat(lows.slice(1)
    .map(x => n / x)
    .reverse()
    .slice(blnPerfectSquare | 0));
}

// Int -> Int -> Maybe Int -> [Int]
function range(m, n, step) {
  const d = (step || 1) * (n >= m ? 1 : -1);

  return Array.from({
    length: Math.floor((n - m) / d) + 1
  }, (_, i) => m + (i * d));
}
