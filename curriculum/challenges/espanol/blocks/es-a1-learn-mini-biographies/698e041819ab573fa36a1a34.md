# --description--

El rol de un ingeniero es muy común en contextos profesionales. En español, la palabra "ingeniero" cambia según el género. `Ingeniero` es la forma masculina, mientras que `ingeniera` es la forma femenina.

Para especificar el campo, puedes usar esta estructura:

**`Ingeniero(a)` + `de` + campo**.

Algunos campos de la ingeniería omiten la palabra `de`.

Por ejemplo:

- `Ingeniero de software` - Ingeniero de software.

- `Ingeniero civil` - Ingeniero civil.
