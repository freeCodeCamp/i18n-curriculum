# --description--

`tech recruiter` 是一个专注于为技术相关职位招聘候选人的专业角色。例如：

`The tech recruiter is looking for a Python developer.` - 这意味着招聘人员的工作是查找具有 Python 技能的人。

该角色涉及了解技术行业，识别修饰的候选人，并将他们与合适的技术职位匹配。
