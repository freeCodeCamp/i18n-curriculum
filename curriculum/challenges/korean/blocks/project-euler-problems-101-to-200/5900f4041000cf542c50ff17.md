# --description--

서로 다른 정수를 사용하여 $\frac{1}{2}$를 역제곱의 합으로 표현하는 여러 방법이 있습니다.

예를 들어, {2,3,4,5,7,12,15,20,28,35}라는 숫자들을 사용할 수 있습니다:

$$\frac{1}{2} = \frac{1}{2^2} + \frac{1}{3^2} + \frac{1}{4^2} + \frac{1}{5^2} + \frac{1}{7^2} + \frac{1}{{12}^2} + \frac{1}{{15}^2} + \frac{1}{{20}^2} + \frac{1}{{28}^2} + \frac{1}{{35}^2}$$

사실, 2부터 45까지의 정수만 사용했을 때, 정확히 세 가지 방법이 있으며, 나머지 두 가지는 {2,3,4,6,7,9,10,20,28,35,36,45}와 {2,3,4,6,7,9,12,15,28,30,35,36,45}입니다.

2부터 80까지의 서로 다른 정수를 사용하여 $\frac{1}{2}$를 역제곱의 합으로 표현하는 방법은 몇 가지일까요?
