# --interactive--

在 HTML 和 CSS 中，元素被分類為 **行內元素** 或 **區塊層級元素**，而這種分類決定它們在文件佈局中的行為。

了解這個差異對於控制你的內容如何在網頁上顯示至關重要。

區塊層級元素是預設會佔據其容器可用寬度全部空間的元素，橫跨其容器的寬度。

這些元素總是從新的一行開始，並將其他內容推到下一行，形成一個「區塊」內容。

區塊層級元素預設會套用 CSS 屬性 `display: block;`。此屬性確保該元素會延伸以填滿容器的寬度，並出現在新的一行。

一些常見的區塊層次元素有 `div` 元素、段落、標題、有序列表、無序列表和區段元素。

這是一個區塊層次元素的程式碼範例：

:::interactive_editor

```html
<p style="border: 2px solid red;">
  First paragraph
</p>
<p>Second paragraph</p>
```

:::

在這個範例中，我們有兩個段落元素，其中第一個元素周圍有紅色的邊框。

這兩個段落元素不共用同一行，因為它們預設為區塊層級元素。

因此，兩個段落元素都會佔滿其容器的全部寬度，在此情況下該容器是 `body` 元素。

區塊層級元素非常適合當你想讓內容垂直堆疊時使用，例如段落、區段或較大的內容區塊。它們通常用來定義網頁的版面配置和結構。

行內元素與區塊層級元素不同，只佔用它們所需的寬度，且不會另起新行。這些元素會在內容中流動，允許文字和其他行內元素與它們並排出現。

行內元素預設會套用 CSS 屬性 `display: inline;`。此屬性確保元素保持在內容流程中，且不會換行。

常見的行內元素有 `span`、`anchor` 和 `img` 元素。

這裡有一個範例可以更好地理解行內元素：

:::interactive_editor

```html
<p>This is a
  <span style="color: red; border: 2px solid green;">red</span>
  word inside a paragraph.
</p>
```

:::

在此範例中，我們有一個 `span` 元素巢狀在段落元素內。`span` 元素具有 `red` 文字顏色，並且周圍有一個 `green` 邊框，讓你可以更清楚地看到被凸顯標示的字。

如你所見，`span` 元素只佔用「red」這個字的空間，並不會將後續內容推到新的一行。

行內元素最適合用於為行內較小的文字或內容零件進行樣式設定，例如強調字、建立超鏈結，或對段落的部分套用特定樣式。
