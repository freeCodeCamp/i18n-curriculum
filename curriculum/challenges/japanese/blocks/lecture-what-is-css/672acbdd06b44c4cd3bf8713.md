# --interactive--

HTMLとCSSの作業を始めると、CSSを書かなくてもウェブページにいくつかのスタイルが適用されていることに気づきます。これらのスタイルは「デフォルトのブラウザスタイル」または「ユーザーエージェントスタイル」と呼ばれます。

これらのデフォルトのブラウザスタイルは、すべてのブラウザでHTML要素が読みやすく表示されるように基本的なフォーマットを提供します。

ただし、これらのスタイルはブラウザごとにわずかに異なる場合があるため、ウェブサイトの見た目を一貫させるためには理解しておくことが重要です。

それでは、さまざまなHTML要素に適用される一般的なデフォルトのブラウザスタイルを見ていきましょう。

見出しは最もよく使われるHTML要素の一つで、デフォルトでサイズや太さが異なるスタイルが適用されています。

例えば、`h1`（最上位の見出し）は通常、太字でフォントサイズが大きく、`h2`、`h3`などの下位の見出しよりも目立ちます。

これらの見出しタグは、コンテンツの階層と構造を定義するのに役立ちます。

次の例は、`h1`から`h6`までの6つの見出し要素を示しています。

:::interactive_editor

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```

:::

もう一つのデフォルトスタイルが適用される要素は`hr`要素で、これはコンテンツのセクションを視覚的に区切るためによく使われる水平線を作成します。ブラウザは通常、この要素にシンプルな線のスタイルを適用します。

より理解しやすくするために、このコード例を見てみましょう。

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<hr>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

水平線は細い線として表示され、テキストの上下にスペースがあり、コンテンツのセクションを区別しています。

次に、`blockquote`要素を見ていきます。

ブロック引用は、他の情報源からの引用文を示すために使われます。ブラウザは通常、インデントを追加し、場合によってはテキストを斜体にします。

インデントはブロック引用を他のテキストから区別し、この内容が引用であることを明確にします。

このコード例を見てみましょう。

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<blockquote>I think, therefore I am. (Rene Descartes)</blockquote>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

上の例では、`blockquote`要素にデフォルトのブラウザスタイルでインデントが適用されます。

もう一つのデフォルトスタイルが適用される要素はアンカー要素（`<a>`）で、リンクとして認識されやすいようにデフォルトで青色と下線が付けられています。

次のHTML例を見てみましょう。

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<a href="https://freecodecamp.org/">Visit the freeCodeCamp learn page</a>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

上のコードには4つの段落要素があり、その中にアンカー要素があります。アンカー要素は青色で下線が付けられ、ユーザーにfreeCodeCampの学習ページへ移動することを示しています。

最後に、番号なしリストと番号付きリストの要素を見ていきます。

ブラウザはリストに基本的なフォーマットを追加し、番号なしリスト（`ul`）か番号付きリスト（`ol`）かによってインデントや箇条書き、番号を付けます。

例を見てみましょう。

:::interactive_editor

```html
<p>Sample Paragraph</p>
<ol>
  <li>item 1</li>
  <li>item 2</li>
  <li>item 3</li>
</ol>
<ul>
  <li>item</li>
  <li>another item</li>
  <li>yet another item</li>
</ul>
<p>Ending Paragraph</p>
```

:::

上のコード例では、番号なしリストと番号付きリストを使っています。どちらのリストもデフォルトで少し右にインデントされます。

これらの例すべてで、ブラウザがデフォルトのスペース、異なるフォントサイズや太さをHTML要素に適用している様子がわかりました。

後のレッスンで、CSSリセットを使ってこれらのデフォルトスタイルの一部をリセットする方法を学びます。
