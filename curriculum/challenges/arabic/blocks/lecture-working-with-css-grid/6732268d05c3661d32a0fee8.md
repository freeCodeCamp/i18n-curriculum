# --interactive--

في الدروس السابقة، كنا نعمل مع خاصية `grid-template-columns` ونضبط القيمة على بعض الوحدات الكسرية.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

بينما الشفرة التالية صحيحة تمامًا، هناك طريقة أسهل لتكرار قسم أو كل قوائم المسارات الخاصة بك.

تُستخدم الدالة `repeat()` لتكرار قسم أو كل المسارات للأعمدة أو الصفوف. تستقبل هذه الدالة عدد التكرار والمسارات التي تريد تكرارها.

إليك نسخة معدلة من المثال السابق باستخدام دالة repeat:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

لن يكون هناك تغيير في الأنماط في المتصفح، لكنها طريقة أكثر إيجازًا لكتابة القيم المكررة للأعمدة.

تقبل دالة `repeat()` أي نمط صالح يمكنك استخدامه للصفوف أو الأعمدة.

إليك مثالًا يستخدم دالة `repeat()` لضبط العمودين الأول والثالث على 20 بكسل والعمودين الثاني والرابع على وحدة كسرية واحدة.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(2, 20px 1fr);
  column-gap: 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

أحيانًا قد تختار كتابة كل قيمة على حدة بدلًا من استخدام دالة `repeat()`، لكن هناك أوقات تكون فيها هذه الدالة مفيدة، خاصة عندما تريد تكرار نمط معين لقائمة المسارات.
