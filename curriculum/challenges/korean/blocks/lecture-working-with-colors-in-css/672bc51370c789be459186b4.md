# --interactive--

CSS에서 색상을 다룰 때 RGB 색상 모델을 이해하는 것이 중요합니다. RGB는 빨강(Red), 초록(Green), 파랑(Blue)을 의미하며, 빛의 기본 색상입니다. 이 세 가지 색상을 다양한 강도로 조합해 다양한 색상을 만듭니다.

RGB 색상 모델은 가산 색상 모델로, 빛을 서로 다른 강도로 합쳐 색상을 만듭니다.

각 색상의 강도는 `0`(빛이 전혀 없는 상태)부터 `255`(빛이 최대인 상태)까지 범위가 있습니다. 빨강, 초록, 파랑의 강도를 다르게 섞으면 화면에서 볼 수 있는 모든 색상을 만들 수 있습니다. 예를 들어:

- 검정은 `rgb(0, 0, 0)`로 나타내며, 세 가지 색상 모두 빛이 없는 상태입니다.
- 흰색은 `rgb(255, 255, 255)`로 나타내며, 세 가지 색상 모두 빛이 최대 강도입니다.
- 빨강은 `rgb(255, 0, 0)`로 나타내며, 빨강은 최대 강도이고 초록과 파랑은 빛이 없습니다.

CSS에서 `rgb()` 함수는 RGB 색상 모델을 사용해 색상을 정의할 수 있게 해줍니다. 이 함수는 빨강, 초록, 파랑 빛의 강도를 각각 나타내는 세 개의 값을 받습니다. 기본 구문은 다음과 같습니다:

```css
element {
  color: rgb(red, green, blue);
}
```

빨강, 초록, 파랑 값은 `0`부터 `255`까지 범위이며, `0`은 해당 색상이 없는 상태, `255`는 최대 채도를 나타냅니다. CSS에서 `rgb()` 함수를 사용하는 예는 다음과 같습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

이 코드는 빨강 값이 `255`로 설정되고 초록과 파랑이 `0`로 설정되어 있으므로 단락의 텍스트 색상을 빨강으로 변경합니다.

CSS는 또한 투명도를 제어하는 네 번째 값인 알파(alpha)를 추가하는 `rgba()` 함수를 제공합니다. 알파 값은 `0`(완전히 투명)부터 `1`(완전히 불투명)까지 범위입니다. `rgba()` 사용 예는 다음과 같습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

이 코드는 빨강과 초록 값이 `div`로, 파랑 값이 `0`로, 알파 값이 `255`(50% 투명도)로 설정되어 `0.5` 요소에 반투명 파란색 배경을 적용합니다.

RGB 색상 모델은 디지털 미디어에 특히 유용한데, 이는 화면이 색상을 표시하는 방식을 직접 반영하기 때문입니다. 모니터와 디스플레이는 작은 빨강, 초록, 파랑 픽셀을 사용해 화면에 색상을 만듭니다. RGB 값을 통해 이 픽셀들의 강도를 조절하면 웹 디자인에 다양한 색상을 표현할 수 있습니다.

또한 RGB 모델은 동적 디자인 작업에 이상적입니다. 예를 들어, 실시간으로 변하는 애니메이션이나 색상 효과를 만들 때 RGB 함수의 유연성 덕분에 프로그래밍 방식으로 색상을 수정할 수 있습니다.

RGB 색상 모델과 CSS에서 `rgb()` 및 `rgba()` 함수를 사용하는 방법을 이해하면 웹 페이지에서 색상을 더 세밀하게 제어할 수 있습니다. 생생한 디자인, 투명도, 그리고 동적 효과를 위한 색상 조정도 가능해집니다.
