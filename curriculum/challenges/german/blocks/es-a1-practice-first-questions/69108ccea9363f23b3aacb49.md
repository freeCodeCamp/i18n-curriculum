# --explanation--

Dieser Dialog zwischen Mateo und Camila hat den Fokus auf Berufe.

`¿A qué te dedicas?` bedeutet „Was ist Ihr Beruf?“ oder „Womit verdienen Sie Ihren Lebensunterhalt?“

**`Soy` + [Beruf]** bedeutet „Ich bin ein/e [Beruf]“.

Im Spanischen ändern sich die meisten Berufe je nach Geschlecht. Einige Berufe, die auf `-ista` oder `-nte` enden, bleiben jedoch oft für beide Geschlechter genau gleich:

| Beruf | Maskulin | Feminin |
| :---: | :---: | :---: |
| Analyst | `analista` | `analista` |
| Journalist | `periodista` | `periodista` |
| Rezeptionist | `recepcionista` | `recepcionista` |
| Sänger | `cantante` | `cantante` |
