# --description--

Reactでは、状態の配列を更新するのはかなり簡単ですが、特に配列を直接変更できるバニラJavaScriptから来た場合は、間違いやすいです。

Reactでは状態は不変として扱われるため、変更を認識してユーザーインターフェースを適切に更新できます。

Reactで状態に保持された配列をどのように更新できるか見てみましょう。

Reactの状態で配列を更新するときに最もよくある間違いの1つは、`push()`、`pop()`、または`splice()`のようなメソッドを使って配列を直接変更することです。これらのメソッドは元の配列を変更してしまい、Reactではそれが許されません。

Reactは新しい配列の参照を使って変更を検出するため、配列を直接変更するとコンポーネントが期待通りに再レンダリングされないことがあります。

状態の配列に要素を追加するために`push()`メソッドを使った例を見てみましょう。これは動作しません：

```jsx
import { useState } from "react";

function ItemsList() {
  const [items, setItems] = useState([
    { id: 0, name: "Item 1" },
    { id: 1, name: "Item 2" },
    { id: 2, name: "Item 3" },
  ]);

  const addItem = () => {
    const newItem = { id: items.length + 1, name: `Item ${items.length + 1}` };
    items.push(newItem); // This modifies the original array
    setItems(items); // React will not detect this change
  };

  return (
    <div>
      <button onClick={addItem}>Add Item</button>
      <ul>
        {items.map((item) => (
          <li key={item.id}>{item.name}</li>
        ))}
      </ul>
    </div>
  );
}

export default ItemsList;
```

`Add Item`ボタンをクリックしても、ユーザーインターフェースには何も起こりません。

配列から要素を削除するために`pop()`メソッドを使いたくなるかもしれません：

```js
const removeItem = () => {
  items.pop(); // Modifies the original array
  setItems(items); // React will not detect this change, either
};
```

状態の配列を更新するには、新しい配列を作成して操作を行い、それをReactに渡すことが重要です。既存の配列を変更してはいけません。

新しい配列なので、Reactは状態が変更されたことを認識し、再レンダリングをトリガーします。

スプレッド演算子を使って`items`配列に要素を追加する方法は次の通りです：

```js
const addItem = () => {
  const newItem = {
    id: items.length + 1,
    name: `Item ${items.length + 1}`,
  };

  // Creates a new array
  setItems((prevItems) => [...prevItems, newItem]);
};
```

`[...prevItems, newItem]`は状態に保持された既存の`items`配列のすべての要素をコピーして新しい配列を作成し、最後に`newItem`を追加します。これは`id`とアイテム番号をインクリメントします。

配列から何かを削除したい場合は、`filter()`メソッドを使うことができます。これは削除したいものを除外して新しい配列を返します：

```js
const removeItem = (id) => {
  setItems((prevItems) => prevItems.filter((item) => item.id !== id));
};
```

完全なコードは次の通りです：

```jsx
import { useState } from "react";

function ItemsList() {
  const [items, setItems] = useState([
    { id: 0, name: "Item 1" },
    { id: 1, name: "Item 2" },
    { id: 2, name: "Item 3" },
  ]);

  const addItem = () => {
    const newItem = { id: items.length + 1, name: `Item ${items.length + 1}` };
    setItems((prevItems) => [...prevItems, newItem]); // Creates a new array
  };

  const removeItem = (id) => {
    setItems((prevItems) => prevItems.filter((item) => item.id !== id)); // Creates a new array
  };

  return (
    <div>
      <button onClick={addItem}>Add Item</button>
      <ul>
        {items.map((item) => (
          <li key={item.id}>
            {item.name}{" "}
            <button onClick={() => removeItem(item.id)}>Remove</button>
          </li>
        ))}
      </ul>
    </div>
  );
}

export default ItemsList;
```

これらは状態の配列を更新する一般的な方法です。
