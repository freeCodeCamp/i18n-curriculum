# --description--

이전 수업에서 프로젝트 폴더를 만들고 Express를 설치했습니다:

```bash
mkdir my-app
cd my-app
npm init -y
npm install express
```

다시 한 번 말씀드리면, 이 명령어들은 새 프로젝트 폴더를 만들고, `cd`로 프로젝트 폴더에 들어가서, 프로젝트를 초기화하고 기본 `package.json` 파일을 생성하며, Express를 설치합니다.

다음으로 코드 편집기를 열고 `index.js`라는 파일을 만든 후 첫 번째 Express 서버를 작성하세요:

```javascript
const express = require("express");
const app = express();
const port = 3000;

app.get("/", (req, res) => {
  res.send("Hello World!");
});

app.listen(port, () => {
  console.log(`Example app listening on port ${port}`);
});
```

터미널에서 다음 명령어로 코드를 실행할 수 있습니다:

```bash
node index.js
```

브라우저에서 [http://localhost:3000](http://localhost:3000)를 열면 **"Hello World!"** 메시지를 볼 수 있습니다.

Express로 시작하는 속도가 이렇게 빠릅니다! 이제 코드를 단계별로 살펴보겠습니다.

첫 번째 코드 줄에서는 Express 라이브러리를 불러와 `express`라는 변수에 할당합니다:

```javascript
const express = require("express");
```

`express` 변수는 팩토리 함수입니다. 호출하면 새 Express 애플리케이션 인스턴스를 생성하고 반환합니다:

```javascript
const app = express();
```

이제 `app`는 Express가 제공하는 `.get()`, `.post()`, `.listen()` 등 모든 메서드에 접근할 수 있습니다. 이것이 메인 서버 객체입니다.

다음 줄에서는 `port` 변수를 정의합니다. 포트는 애플리케이션이 수신 대기하는 네트워크 엔드포인트이며, `3000`은 로컬 개발에 자주 사용하는 포트입니다:

```javascript
const port = 3000;
```

이제 다음 부분을 살펴보겠습니다:

```javascript
app.get("/", (req, res) => {
  res.send("Hello World!");
});
```

이 코드 블록은 라우트를 정의하고 서버가 요청에 어떻게 응답할지 알려줍니다. `app.get()`은 HTTP GET 요청에 대한 핸들러를 등록합니다. GET 요청은 데이터를 가져오는 데 자주 사용하는 HTTP 메서드입니다.

`"/"` 인수는 `app.get()` 내부에 있으며 라우트 경로를 나타냅니다. 이 경우 애플리케이션의 루트 URL을 가리킵니다.

`(req, res) => { ... }` 부분은 요청이 이 라우트와 일치할 때 실행되는 콜백 함수입니다. `req` 객체는 들어오는 요청에 대한 정보를 담고 있고, `res` 객체는 클라이언트에 응답을 보내는 데 사용됩니다.

마지막으로 `res.send("Hello World!")`은 클라이언트에 응답을 보냅니다. 이 예에서는 응답이 일반 텍스트 `"Hello World!"`입니다.

이제 마지막 코드 부분을 살펴보겠습니다:

```javascript
app.listen(port, () => {
  console.log(`Example app listening on port ${port}`);
});
```

`app.listen()` 메서드는 Express 애플리케이션을 시작하고 Node에게 지정된 포트, 여기서는 `3000`에서 들어오는 HTTP 요청을 수신 대기하라고 지시합니다.

서버가 시작되면 콜백 함수가 실행되고 `Example app listening on port 3000` 메시지가 콘솔에 기록되어 Express 앱이 성공적으로 시작되어 요청을 받을 준비가 되었음을 확인합니다.
