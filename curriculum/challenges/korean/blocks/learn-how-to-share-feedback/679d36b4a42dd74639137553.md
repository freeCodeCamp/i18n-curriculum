# --explanation--

동사가 전치사(`without`) 뒤에 올 때는 `-ing` 형태를 취합니다. 이 경우 `without causing`는 혼란을 초래하는 행동을 피하는 것을 의미합니다. 모든 전치사에 해당하는 규칙입니다. 예를 들어:

- `She left without saying goodbye.` - 떠나기 전에 작별 인사를 하지 않았다는 뜻입니다.

- `We succeeded by working together.` - 전치사 `by`는 그들이 팀으로 일하며 성공한 방식을 나타냅니다.

- `They talked about improving the process.` - 전치사 `about`는 대화 주제인 프로세스 개선 방법 앞에 옵니다.
