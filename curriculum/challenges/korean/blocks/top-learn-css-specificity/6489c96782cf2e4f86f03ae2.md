# --description--

더 구체적인 CSS 선언이 덜 구체적인 선언보다 우선합니다. 이전 수업에서 다룬 인라인 스타일은 선택자에 비해 가장 높은 구체성을 가지며, 각 선택자 유형은 선언의 구체성에 기여하는 고유한 구체성 수준을 가집니다. 다른 선택자들도 구체성에 기여하지만, 지금까지 다룬 다음 선택자에만 집중하세요:

1. ID 선택자 (가장 구체적인 선택자)
2. 클래스 선택자
3. 타입 선택자

구체성은 요소에 여러 개의 상충하는 선언이 적용될 때만 고려되며, 일종의 동점자 결정 기준 역할을 합니다. ID 선택자는 어떤 수의 클래스 선택자보다 항상 우선하며, 클래스 선택자는 어떤 수의 타입 선택자보다 항상 우선하고, 타입 선택자는 자신보다 덜 구체적인 어떤 선택자보다도 항상 우선합니다. 더 높은 구체성을 가진 선택자가 없을 때는, 같은 선택자가 더 많이 포함된 선언이 더 적은 수의 같은 선택자보다 우선합니다.
