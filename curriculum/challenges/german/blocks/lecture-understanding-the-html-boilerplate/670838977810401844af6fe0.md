# --description--

Lernen wir das `link`-Element kennen und wie Sie es verwenden, um auf externe Stylesheets zu verknüpfen.

Das `link`-Element wird verwendet, um auf externe Ressourcen wie Stylesheets und Website-Icons zu verknüpfen. Hier ist die grundlegende Syntax für die Verwendung des `link`-Elements für eine externe CSS-Datei:

```html
<link rel="stylesheet" href="./styles.css" />
```

Das `rel`-Attribut wird verwendet, um die Beziehung zwischen der verknüpften Ressource und dem HTML-Dokument anzugeben. In diesem Fall müssen wir angeben, dass diese verknüpfte Ressource ein `stylesheet` ist.

Es gilt als bewährte Vorgehensweise, HTML und CSS in verschiedenen Dateien zu trennen. Entwickler verwenden das `link`-Element für ihre externe CSS-Datei, anstatt alles im HTML-Dokument zu schreiben.

Das `href`-Attribut wird verwendet, um den Speicherort der URL für die externe Ressource anzugeben.

Der `.` gefolgt von einem Schrägstrich im Beispiel weist den Computer an, im aktuellen Ordner oder Verzeichnis nach der Datei `styles.css` zu suchen.

Das `link`-Element sollte wie im folgenden Beispiel innerhalb des `head`-Elements platziert werden:

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Examples of the link element</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

Oft sehen Sie mehrere `link`-Elemente in einem professionellen Code, die mit verschiedenen Stylesheets, Schriftarten und Icons verknüpft sind. Hier ist ein Beispiel, wie das `link`-Element verwendet wird, um eine externe Google-Schriftart namens *Playwrite Cuba* zu verknüpfen:

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Playwrite+CU:wght@100..400&display=swap"
  rel="stylesheet"
/>
```

Google Fonts sind eine Sammlung kostenloser und Open-Source-Schriftarten, die Sie in jedem Projekt verwenden können. Sie können auswählen, welche Schriftarten Sie verwenden möchten, und Google stellt Ihnen den notwendigen HTML- und CSS-Code bereit. In diesem Beispiel weist der `preconnect`-Wert für das `rel`-Attribut den Browser an, eine frühe Verbindung zu dem im `href`-Attribut angegebenen Wert herzustellen. Dies geschieht, um die Ladezeiten für diese externen Ressourcen zu beschleunigen.

Ein weiterer häufiger Anwendungsfall für das `link`-Element ist das Verknüpfen von Icons. Hier ist ein Beispiel für die Verknüpfung mit einem Favicon:

```html
<link rel="icon" href="favicon.ico" />
```

Ein Favicon, kurz für favorite icon, ist ein kleines Icon, das typischerweise im Browser-Tab neben dem Seitentitel angezeigt wird. Viele Websites verwenden ein Favicon, um ihr Marken-Icon darzustellen.
