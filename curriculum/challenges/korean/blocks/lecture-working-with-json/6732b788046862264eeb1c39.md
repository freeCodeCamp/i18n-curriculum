# --description--

JSON이란 무엇이며 대괄호 표기법과 점 표기법으로 값을 어떻게 접근하나요?

JSON은 JavaScript Object Notation의 약자입니다. 서버와 웹 애플리케이션 간에 데이터를 교환할 때 흔히 사용하는 가볍고 텍스트 기반의 데이터 형식입니다.

JSON이 웹 개발에서 매우 인기 있는 이유 중 하나는 기계가 파싱할 수 있으면서도 사람이 읽기 쉽기 때문입니다.

JSON은 언어에 독립적이어서 Java 애플리케이션에서 Python 애플리케이션으로, 또는 JavaScript 애플리케이션에서 C# 애플리케이션으로 JSON 데이터를 쉽게 보낼 수 있습니다.

JSON은 객체, 배열, 문자열, 불리언, null, 숫자 등 다양한 데이터 타입을 지원합니다.

다음은 JSON 객체의 예입니다:

```js
{
  "name": "Alice",
  "age": 30,
  "isStudent": false,
  "list of courses": ["Mathematics", "Physics", "Computer Science"]
}
```

보시다시피 JSON은 정보를 저장하기 위해 키-값 쌍을 사용하며 각 쌍은 쉼표로 구분됩니다. 각 키는 반드시 큰따옴표로 감싸야 하며, 그렇지 않으면 오류가 발생합니다.

JSON 객체에서 데이터를 접근하려면 점 표기법이나 대괄호 표기법을 사용할 수 있습니다. 이 예에서는 점 표기법을 사용해 JSON 객체에서 `age`에 접근하고 있습니다:

```js
import data from "./example.json" with { type: "json" };

console.log(data.age);
```

이 예제는 `import` 문장으로 알려진 것을 사용하고 있는데, 이는 JSON 객체를 이 파일로 가져와서 접근할 수 있게 합니다. `import` 문장에 대해서는 이후 수업에서 더 배우게 됩니다.

또한 대괄호 표기법을 사용해 JSON 객체에서 정보를 접근할 수도 있습니다. 다음은 `list of courses` 배열에 접근하는 예입니다:

```js
import data from "./example.json" with { type: "json" };

console.log(data["list of courses"]);
```

여기서 대괄호 표기법이 특히 유용한 이유는 키에 공백으로 구분된 여러 단어가 포함되어 있기 때문입니다. 점 표기법을 사용하려 하면 오류가 발생합니다.

요약하자면, JSON은 배열과 중첩 객체를 포함해 다양한 데이터 타입을 저장할 수 있는 다재다능한 형식입니다. 점 표기법이나 대괄호 표기법을 사용해 JSON 객체 내에 저장된 값에 쉽게 접근할 수 있습니다.
