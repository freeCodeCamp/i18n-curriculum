# --explanation--

En el `Simple Past`, las preguntas pueden comenzar con una palabra interrogativa (por ejemplo, `what`, `where`, `how`, `who`) usando la siguiente estructura:

**Palabra interrogativa + `did` + sujeto + verbo en forma base.**

La pregunta de Lisa, `What did you find there?`, usa `what` para pedir información específica sobre el resultado de las acciones de James. Otros ejemplos incluyen:

- `Where did you go yesterday?` - Estás preguntando sobre una ubicación.

- `How did they solve the problem?` - Estás preguntando sobre un método.

- `Who did you call last night?` - Estás preguntando por una persona.

En el diálogo, Lisa está preguntando sobre el descubrimiento de James mientras buscaba el error.
