# --description--

Un `loop` è una sequenza di istruzioni che viene ripetuta continuamente fino a quando non si raggiunge una certa condizione. I `Loops` sono usati per eseguire lo stesso codice più volte in modo efficiente. I tipi più comuni di `loops` includono i loop `for` e i loop `while`. 

Per esempio, `for loop`: `for (int i = 0; i < 5; i++) { /* code to be executed */ }`

Nell'esempio, il codice all'interno delle parentesi graffe `{ }` viene eseguito ripetutamente finché la condizione non è più vera.
