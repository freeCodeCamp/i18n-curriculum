# --explanation--

조동사 `might` 다음에는 `have` + `Past Participle`가 와서 과거의 가능성을 나타냅니다. 이 구조는 어떤 일이 일어났는지 확실하지 않지만 가능성이 있다고 생각할 때 사용합니다.

- 올바른 예: `They might have missed the deadline.`

- 올바르지 않은 예: `They might has missed...`, `They might missed...`, 또는 `They might had missed...`
