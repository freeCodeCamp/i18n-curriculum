# --description--

Quando você está falando sobre outra pessoa, pode usar esta pergunta para saber o nome dela:

`¿Cómo se llama?` - Qual é o nome dele/dela?

A palavra `cómo` significa "como". É muito importante colocar o acento na letra `o` quando essa palavra inicia uma pergunta.

A palavra `se` indica que você está falando de uma terceira pessoa.

Elena está falando com Esteban sobre seu novo colega, Diego.
