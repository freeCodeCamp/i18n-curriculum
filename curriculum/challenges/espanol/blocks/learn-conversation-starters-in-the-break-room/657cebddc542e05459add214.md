# --description--

Para `schedule` significa organizar un evento o actividad para que ocurra en un tiempo específico. Es como hacer una cita o establecer una hora para algo. Por ejemplo:

- `We will schedule a meeting for Monday.` - El equipo planificará la reunión el lunes.

- `They scheduled a call at 3 p.m.` - Planearon hablar por teléfono a las 3 p.m.
