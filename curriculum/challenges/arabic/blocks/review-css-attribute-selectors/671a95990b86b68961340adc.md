# --interactive--

## العمل مع محددات الخاصية المختلفة والروابط

- **التعريف**: يسمح محدد `attribute` باستهداف عناصر HTML بناءً على خصائصها مثل خاصيتي `href` أو `title`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Link with href</a>
<a>No href</a>
```

```css
a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

- **خاصية `title`**: توفر هذه الخاصية معلومات إضافية عن العنصر. إليك كيفية استهداف الروابط التي تحتوي على خاصية `title`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Tooltip">Link with title</a>
<a href="#">Normal link</a>
```

```css
a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

- **دمج المحددات لمطابقة الروابط التي تحتوي على كل من خاصيتي `href` و `title`**: يجمع بين عدة محددات خاصية.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Info">Href + Title</a>
<a href="#">Only href</a>
```

```css
a[href][title] {
  color: green;
}
```

:::

- **مطابقة كلمة واحدة ضمن قائمة مفصولة بمسافات**: يستهدف الروابط التي تحتوي على كلمة صنف محددة.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a class="link primary">Primary link</a>
<a class="link secondary">Secondary link</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

- **مطابقة القيم التي تبدأ ببادئة محددة**: يستهدف الروابط التي تبدأ بـ `https://`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">HTTPS link</a>
<a href="http://www.freecodecamp.org">HTTP link</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

- **مطابقة القيم التي تنتهي بلواحق محددة**: يستهدف الروابط التي تنتهي بـ `.jpg`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="photo.jpg">Image link</a>
<a href="index.html">HTML link</a>
```

```css
a[href$=".jpg"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

- **مطابقة القيم التي تحتوي على سلسلة فرعية في أي مكان**: يستهدف الروابط التي تحتوي على `https` في أي مكان ضمن القيمة.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Secure link</a>
<a href="page.html">Local link</a>
```

```css
a[href*="https"] {
  color: teal;
}
```

:::

- **الملخص**: تسهل هذه الأنماط تنسيق الروابط بشكل متسق بناءً على خصائصها وقيمها.

## استهداف العناصر باستخدام خاصيتي `lang` و `data-lang`

- **خاصية `lang`**: تُستخدم هذه الخاصية في HTML لتحديد لغة المحتوى داخل العنصر. قد ترغب في تنسيق العناصر بشكل مختلف بناءً على اللغة المكتوبة بها، خاصة في موقع ويب متعدد اللغات.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">English text</p>
<p lang="fr">French text</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

- **خاصية `data-lang`**: تُستخدم خصائص البيانات المخصصة مثل `data-lang` عادةً لتخزين معلومات إضافية في العناصر، مثل تحديد اللغة المستخدمة داخل قسم معين من النص. إليك كيفية تنسيق العناصر بناءً على خاصية `data-lang`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div data-lang="fr">French content</div>
<div data-lang="en">English content</div>
```

```css
div[data-lang="fr"] {
  color: blue;
}
```

:::

## العمل مع محدد الخاصية، عناصر القائمة المرتبة، وخاصية `type`

- **خاصية `type`**: عند العمل مع القوائم المرتبة في HTML، تسمح خاصية `type` بتحديد نمط الترقيم المستخدم، مثل الأرقام، الحروف الأبجدية، أو الأرقام الرومانية.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Alpha</li>
  <li>Beta</li>
</ol>

<ol type="i">
  <li>One</li>
  <li>Two</li>
</ol>
```

```css
/* Example targeting uppercase alphabetical numbering */
ol[type="A"] {
  color: purple;
  font-weight: bold;
}

/* Example targeting lowercase Roman numerals */
ol[type="i"] {
  color: green;
}
```
