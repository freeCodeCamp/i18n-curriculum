# --description--

يشير `Compile` في البرمجة إلى تحويل الشفرة التي يكتبها الأشخاص إلى صيغة يمكن لأجهزة الكمبيوتر فهمها واستخدامها.

يشبه ذلك تحويل وصفة مكتوبة بلغة إلى لغة أخرى حتى يتمكن شخص آخر من طهي الوجبة.

على سبيل المثال، `When we compile Java code, it changes into a form that the computer can run.`
