# --description--

Sie haben kürzlich das Muster `¿Quieres que lo deletree?` gelernt, um anzubieten, etwas zu buchstabieren. Eine weitere häufige Frage ist:

`¿Cómo se deletrea?`

Diese Frage wird verwendet, wenn jemand wissen möchte, wie ein Wort, ein Name oder eine E-Mail Buchstabe für Buchstabe buchstabiert wird.

Sie ist sehr gebräuchlich in Gesprächen, in denen Kontaktdaten bestätigt werden.

# --explanation--

Esteban fragt `¿Cómo se deletrea?`, also möchte er wissen, wie etwas buchstabiert wird. Die korrekte Antwort sollte die Information Buchstabe für Buchstabe buchstabieren.

Nur eine Option tut dies: `j-u-l-i-e-t-a punto c-a-c-e-r-e-s arroba a-n-d-e-s punto com.`

Die anderen Optionen sind vollständige Aussagen (Name, Nationalität, Beruf) und geben keine Buchstabierung an.
