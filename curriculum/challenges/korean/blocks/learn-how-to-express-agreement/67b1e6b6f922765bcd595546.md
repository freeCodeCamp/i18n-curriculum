# --explanation--

`Make sure`는 무언가가 정확하거나 완료되었는지 확인하는 행동을 취하는 것을 의미합니다. 예를 들어:  

`Please make sure the report is submitted before the deadline.` – 이것은 보고서가 제시간에 제출되었는지 점검하고 확인하는 것을 의미합니다.  

`Reliable`는 무언가 또는 누군가가 잘 작동하거나 시간이 지나도 일관성을 유지할 수 있음을 의미합니다. 예를 들어:

`The server is reliable; it has been running without any issues for months.` - 이것은 서버가 일관되게 작동하며 문제를 일으키지 않는다는 뜻입니다.

`More reliable`는 무언가가 더욱 신뢰할 수 있고 안정적이게 되었음을 의미합니다. 예를 들어:

`The new software update makes the app more reliable.` - 이것은 업데이트 후 앱이 더 신뢰할 수 있고 안정적이게 되었다는 뜻입니다.
