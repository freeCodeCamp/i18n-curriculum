# --description--

語意是語言中字或片語的意義。在 HTML 這種語言中，元素也有其自身的語意意義。事實上，你可以將你的 HTML 文件想像成一份文字文件。就像文字文件一樣，你可能會有標題、影像、粗體文字和其他格式設定。

元素的語意意義是指該元素所傳達的特殊資訊。`p` 元素的語意意義，例如，是一段文字：

```html
<p>
  Let me tell you about my fantastic holiday in Paris.
  I saw the impressive Eiffel Tower up close!
</p>
```

大多數元素都有語意意義。`div` 元素是少數沒有語意意義的元素之一。但這為什麼很重要？

首先，使用正確的語意化 HTML 將確保使用輔助技術如螢幕閱讀器的使用者獲得最佳體驗。但同時，語意化 HTML 也能提升你的搜尋等級，這稱為搜尋引擎優化，或 SEO。

最後，使用正確的語意元素可以提升你的開發體驗。你不必從一堆開發內容中篩選出你的導覽列，反而可以直接編輯 `nav` 元素並知道你正在更改什麼。在本節中，你將進一步了解這些主題、如何使用語意元素，以及為什麼語意 HTML 如此重要。
