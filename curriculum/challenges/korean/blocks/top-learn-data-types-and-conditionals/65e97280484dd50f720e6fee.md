# --description--

지난 예제에서는 두 가지 선택지 `if` 또는 `else`가 있었습니다. 그런데 선택지를 두 개 이상으로 늘리고 싶다면 어떻게 해야 할까요? `else if` 문을 사용해 코드에 조건을 더 추가할 수 있습니다. `else if` 문은 여러 조건을 확인하고 첫 번째 조건이 참일 때 코드 블록을 실행하게 해줍니다. 첫 번째 조건이 거짓이면 다음 조건을 확인하고 이런 식으로 계속 진행합니다. 조건이 하나도 참이 아니면 `else` 블록이 실행됩니다.

다음은 `else if` 문의 예시입니다:

```javascript
let x = 5;

if (x > 10) {
  console.log("x is greater than 10");
} else if (x > 5) {
  console.log("x is greater than 5");
} else {
  console.log("x is less than or equal to 5");
}
```

이 예제에서 `x` 변수는 `10` 숫자와 `>` 연산자를 사용해 비교됩니다. 만약 `x`가 `10`보다 크면 첫 번째 코드 블록이 실행됩니다. 그렇지 않으면 `else if` 문이 다음 조건을 확인합니다. `x`가 `5`보다 크면 두 번째 코드 블록이 실행됩니다. 그렇지 않으면 `else` 블록이 실행됩니다.
