# --description--

Hemos visto unas cuantas maneras de establecer colores planos en CSS, pero también se pueden aplicar transiciones de color, o <dfn>gradientes</dfn>, sobre un elemento.

Un degradado es cuando un color transiciona a otro. La función de CSS `linear-gradient` te permite controlar la dirección de la transición a lo largo de una línea, y qué colores se utilizan.

Algo a recordar es que la función `linear-gradient` en realidad crea un elemento `image`, y generalmente se combina con la propiedad `background` que puede aceptar una imagen como valor.

En la regla CSS `.red`, cambia la propiedad `background-color` a `background`.
