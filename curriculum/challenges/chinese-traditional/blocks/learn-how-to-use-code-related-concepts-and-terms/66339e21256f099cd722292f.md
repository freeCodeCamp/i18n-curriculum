# --description--

程式設計中的 `constant` 類似於變數，但一旦指定了值，該值就無法更改。當你知道某些值在整個程式中不需要被修改時，會使用 `constants`。它們通常以大寫字母書寫，以區別於變數。

例如，`const PI = 3.14;` 在許多程式語言中，`const` 用來宣告一個常數。這裡，`PI` 是一個被指定值為 `3.14` 的 `constant`。
