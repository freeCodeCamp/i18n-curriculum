# --description--

`Encounter`，特別是在過去式 `encountered` 中，意指遇到某事，通常是意外地。 例如，`I encountered an error while testing the software.` 它暗示在沒有尋找的情況下發現某物。

`Pinpoint` 意味著非常精確地識別某事。例如，`After reviewing the code, I was able to pinpoint the error.` 這表示找到問題的確切位置和內容。

在程式設計中，發現問題通常是關於 `encountering` 意外的問題並嘗試 `pinpoint` 它們的原因。
