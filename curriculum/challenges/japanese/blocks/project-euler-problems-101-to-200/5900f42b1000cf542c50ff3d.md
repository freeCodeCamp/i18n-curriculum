# --description--

正の実数の $m$-タプル $S_m = (x_1, x_2, \ldots, x_m)$ で、$x_1 + x_2 + \cdots + x_m = m$ を満たし、$P_m = x_1 \times {x_2}^2 \times \cdots \times {x_m}^m$ が最大となるものを考えます。

例えば、$[P_{10}] = 4112$ （[ ] は整数部分関数）であることが確認できます。

$2 ≤ m ≤ 15$ の範囲で $\sum {[P_m]}$ を求めてください。
