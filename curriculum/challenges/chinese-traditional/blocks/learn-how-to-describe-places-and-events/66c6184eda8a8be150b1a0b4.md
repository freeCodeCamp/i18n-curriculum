# --explanation--

`Have to` 用來表達某事是必要或必須的。它顯示沒有其他選擇只能做某事。例如：

- `You have to save your changes before closing the program。`－這表示在關閉程式之前必須先儲存變更。

當 Tom 說 `I'll have to check again` 時，他表示他需要再檢查一次設定，暗示他不確定。
