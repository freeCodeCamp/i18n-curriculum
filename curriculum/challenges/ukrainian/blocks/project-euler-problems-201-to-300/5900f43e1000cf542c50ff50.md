# --description--

Розглянемо множину $S(r)$ точок ($x$,$y$) з цілими координатами, які задовольняють нерівність $|x| + |y| ≤ r$.

Нехай $O$ — точка (0,0), а $C$ — точка ($\frac{r}{4}$,$\frac{r}{4}$).

Нехай $N(r)$ — кількість точок $B$ у множині $S(r)$, таких що трикутник $OBC$ має тупий кут, тобто найбільший кут $α$ задовольняє $90°<α<180°$.

Наприклад, $N(4)=24$ та $N(8)=100$.

Яке значення $N(1\\,000\\,000\\,000)$?
