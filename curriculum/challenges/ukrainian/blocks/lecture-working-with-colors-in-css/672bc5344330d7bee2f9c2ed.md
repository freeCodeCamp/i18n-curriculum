# --interactive--

Працюючи з CSS для стилізації вебсторінки, одним із найпоширеніших способів визначення кольорів є використання hex-значень кольорів. Шістнадцяткові значення кольорів, або hex-коди, — це стислий спосіб представлення кольорів у моделі кольорів RGB. У цьому уроці ми розглянемо, що таке hex-коди, як вони працюють у CSS і коли їх варто використовувати.

Hex-код (скорочення від шістнадцяткового коду) — це рядок із шести символів, який використовується для представлення кольорів у моделі кольорів RGB. «Hex» означає систему числення з основою 16, яка використовує цифри `0` до `9` і літери `A` до `F`.

У контексті кольорів hex-коди вказують кількість червоного, зеленого та синього (RGB), які складають певний колір. Кожен hex-код починається з символу решітки (`#`), за яким ідуть шість символів, що визначають інтенсивність червоного, зеленого та синього кольорів.

Наприклад, hex-код для чорного — `#000000`, а для білого — `#FFFFFF`. Перші два символи представляють значення червоного, наступні два — зеленого, а останні два — синього.

У CSS ви можете застосовувати кольори до елементів, використовуючи hex-коди. Ось базовий синтаксис:

```css
element {
  color: #RRGGBB;
}
```

Тут `#RRGGBB` — це заповнювач для фактичного hex-коду. Пари `RR`, `GG` і `BB` кожна відповідає інтенсивності червоного, зеленого та синього відповідно.

Ці пари можуть варіюватися від `00` (найнижча інтенсивність) до `FF` (найвища інтенсивність). Чим більше число, тим більше цього кольору буде у фінальній суміші. Наприклад, давайте застосуємо кілька hex-кодів до різних елементів:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h1>This is a heading.</h1>
<p>This is a paragraph.</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}
p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

У першому випадку `#FF5733` — це червоно-помаранчевий колір. Ось розбір:

- `FF` — найвище значення для червоного (повністю насичений).
- `57` — помірна кількість зеленого.
- `33` — невелика кількість синього.

У другому випадку `#4CAF50` — відтінок зеленого:

- `4C` — низьке значення для червоного.
- `AF` — сильне значення для зеленого.
- `50` — помірне значення для синього.

Hex-коди популярні, бо вони точні й прості у використанні. Вони дозволяють точно налаштовувати кольори, що робить їх ідеальними для дизайнів, де важлива послідовність.

Більшість програм для дизайну, таких як Adobe Photoshop або Figma, надають hex-коди для вибраних кольорів, що полегшує копіювання та вставлення цих значень безпосередньо у ваші CSS-файли.

У веброзробці hex-коди часто використовують для визначення кольорів тексту, фону, кордонів та інших візуальних елементів. Наприклад:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example</button>
```

```css
body {
  background-color: #F0F0F0; /* Light gray background */
}

button {
  background-color: #007BFF; /* A shade of blue */
  color: #FFFFFF; /* White text */
}
```

:::

Це гарантує, що ваші кольори залишатимуться послідовними на всьому сайті або застосунку.

Іноді ви можете побачити hex-коди у скороченому форматі, що використовує лише три символи замість шести. Це можливо, коли обидва символи в кожній парі кольору однакові. Наприклад, `#FF5733` не можна скоротити, але `#FFFFFF` (білий) можна записати як `#FFF`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h2>This is a heading.</h2>
```

```css
body {
  background-color: #000;
}

h2 {
  color: #FFF; /* White in shorthand */
}
```

:::

Скорочений `#FFF` еквівалентний `#FFFFFF`. Аналогічно, `#000` представляє чорний, що у повній формі є `#000000`.

Hex-коди популярні через свою точність і сумісність, дозволяючи використовувати понад 16 мільйонів кольорів у всіх браузерах. Вони ідеальні, коли потрібні конкретні кольори бренду, коли хочеться контролювати точні відтінки, які важко визначити за допомогою іменованих кольорів або HSL, або коли ви копіюєте кольори з програм для дизайну, які за замовчуванням надають hex-коди.
