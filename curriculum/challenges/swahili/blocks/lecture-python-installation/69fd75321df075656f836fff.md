# --description--

Katika funzo lililopita, ulijifunza jinsi ya kuendesha script za Python kwa ndani ya kompyuta yako. Lakini kuna nyakati ambapo hutaki kuunda programu kamili za Python na unahitaji tu kujaribu msimbo fulani wa Python. Hapa ndipo Python interactive shell inakuja.

Lakini kwanza, hebu tukague ni nini terminal.

Terminal ni kiolesura cha maandishi kinachokuwezesha kuingiliana na kompyuta yako kwa kuandika amri. Kila mfumo wa uendeshaji huja na programu ya terminal ya msingi. Kwenye macOS, unaweza kutumia programu ya Terminal. Kwenye Windows, unaweza kutumia Command Prompt au PowerShell. Kwenye Linux, kila mazingira ya desktop yana programu yake ya terminal ya msingi, kama GNOME Terminal au Konsole.

Fungua programu ya terminal na andika `python` kisha bonyeza `Enter`. Hii itaanzisha Python interactive shell. Interactive shell ni programu inayokuwezesha kuandika amri moja kwa moja na kuona matokeo yake.

Unapoanza kikao kipya, unaweza kuona aina hii ya matokeo mwanzoni:

```bash
Python 3.12.2 (main, Mar 21 2024, 22:48:26) [Clang 14.0.3 (clang-1403.0.22.14.1)] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>> 
```

Alama ya `>>>` ina maana Python inasubiri wewe uandike amri. Jaribu kuchapisha `"Hello, world!"` kwenye terminal:

```bash
print("Hello, world!")
```

Unapaswa kuona maandishi yakionekana kama haya:

```bash
>>> print("Hello, world!")
Hello, world!
```

Baada ya maandishi kuchapishwa, Python inarudi kwenye ifuatayo:

```bash
>>>
```

Hii ina maana unaweza kuandika amri nyingine.

Interpreter ya Python inafuata mzunguko unaojulikana kama Soma, Kagua, Chapisha, Rudia (Read, Evaluate, Print, Loop). Au REPL kwa kifupi. Kila unapoandika amri, interpreter husoma ingizo, hulikagua, huchapisha matokeo na kurudia ili kuonyesha `>>>` ili uandike amri zaidi.

Nini hutokea ukijaribu kuandika amri isiyo sahihi kama hii?

```bash
>>> something random
```

Basi, interpreter ya Python bado itafuata mchakato huo wa REPL. Katika kesi hii, utapokea ujumbe wa kosa:

```bash
>>> something random
  File "<stdin>", line 1
    something random
              ^^^^^^
SyntaxError: invalid syntax
```

Ikiwa unataka kutoka kwenye interactive shell unaweza kuandika `exit()` au bonyeza `Ctrl + D` kwa (Mac/Linux) au `Ctrl + Z + Enter` kwa (Windows).

Kutumia Python interactive shell ni nzuri kwa majaribio madogo ya msimbo au uchunguzi wa msingi. Wakati unapoendelea na mafundisho yaliyobaki katika kozi hii, unaweza kuandika baadhi ya njia mpya ulizojifunza ndani ya interactive shell.

Ikiwa unapanga kufanya kazi na programu ndefu au kazi zinazohusisha mafaili mengi, basi ni bora kufanya kazi katika mhariri wa msimbo au IDE.
