# --description--

Existen algunos valores primos, $p$, para los cuales existe un número entero positivo, $n$, tal que la expresión $n^3 + n^{2}p$ es un cubo perfecto.

Por ejemplo, cuando $p = 19,\ 8^3 + 8^2 × 19 = {12}^3$.

Quizás lo más sorprendente es que el valor de $n$ es único para cada primo con esta propiedad, y hay solamente cuatro tales primos por debajo de cien.

¿Cuántos primos por debajo de un millón tienen esta notable propiedad?
