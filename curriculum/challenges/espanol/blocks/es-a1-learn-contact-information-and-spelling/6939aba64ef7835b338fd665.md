# --description--

¿Recuerdas que los verbos se conjugan según la persona a la que se refieren? ¡Si no, ahora sí!

Hace un par de tareas aprendiste cómo usar el verbo `poder` para preguntar si alguien podía hacer algo por ti.

Ahora aprenderás cómo conjugar este mismo verbo para preguntar si **tú** puedes hacer algo. Solo necesitas usar la estructura:

`Puedo` + verbo en infinitivo + `...?`

Veamos algunos ejemplos:

- `¿Puedo usar la computadora?` - ¿Puedo usar la computadora?

- `¿Puedo usar mi apodo?` - ¿Puedo usar mi apodo?

En ambos ejemplos, estás pidiendo permiso. El primero para usar la computadora, el segundo para usar tu apodo.

# --explanation--

Basti dice `¿Puedo usar mi apodo?`.

Aquí, `puedo` es la forma de primera persona del singular de `poder`, usada cuando alguien pide permiso para sí mismo.

La estructura sigue el patrón:

`¿Puedo` + verbo en infinitivo + `...?`

En este caso:

- `Usar` significa usar

- `Mi apodo` se refiere a su apodo

Entonces Basti está preguntando si puede usar su apodo.
