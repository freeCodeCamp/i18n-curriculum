# --description--

طريقة أخرى مفيدة للمصفوفة هي الطريقة `filter()`. تنشئ الطريقة `filter()` مصفوفة جديدة تحتوي على كل العناصر التي تجتاز الاختبار الذي تنفذه الدالة المقدمة. صيغة الطريقة `filter()` النحوية كما يلي:

```javascript
const numbers = [2, 5, 6, 1, 9, -1]

const newNumbers = numbers.filter((number) => {
  return number > 2;
});

console.log(newNumbers); // Output: [5, 6, 9]
```

تنشئ الطريقة `filter()` مصفوفة جديدة تحتوي على كل العناصر الأكبر من 2. لا تغير الطريقة `filter()` المصفوفة الأصلية.
