# --explanation--

무언가에 대해 완전히 확신하지는 못하지만 그렇게 되어야 한다고 믿을 때, `It's supposed to` + 동사 구문을 사용할 수 있습니다.

이 표현은 아직 확인하지 않았더라도 어떤 상태일 것으로 예상된다는 뜻을 나타냅니다. 예를 들어:

- `The update is supposed to fix the bug.` - 이 표현은 업데이트가 버그를 고칠 것으로 예상하지만 아직 확인하지는 않았다는 의미입니다.
