# --interactive--

**注意**：某些交互示例可能使用你尚未学习的 CSS。不要担心尝试理解所有的代码。示例的目的是向你展示这些设计概念的预览，以便你更好地理解其工作原理。要查看预览，你需要启用交互式编辑器。

在网页设计中，你可以使用多种类型的导航辅助工具。示例包括顶部导航栏、侧边栏和页脚。但如果你的网站较为复杂，且导航层级较深，你可能需要考虑使用面包屑导航。

面包屑是一种导航辅助，显示用户在网站层次结构中的位置。以下是一个模拟电子产品网站的面包屑示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home / Electronics / Phones / Smartphone XYZ
</div>
```

```css
.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

在大多数网站中，面包屑导航显示在页面顶部，向用户展示他们到达当前页面所经过的路径。从 `Homepage` 开始，用户导航到 `Electronics` 分类，然后到 `Phones` 分类，最后到 `Smartphone XYZ` 产品。你可能在查找产品或特定信息时与网站上的面包屑导航有过交互。

使用面包屑导航很有帮助，因为它可以帮助用户了解他们在网站层次结构中的位置以及如何返回到之前的页面。当用户来自搜索结果或外部链接并需要了解他们所在页面的上下文时，这尤其有用。

在设计面包屑时，有几个注意事项需要牢记。首先是决定分隔符是什么。分隔符是用于分隔层次结构不同级别的字符。常见的分隔符包括大于号（`>`）、右角引号（`»`）和斜线（`/`）。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
```

```css
.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

第二个考虑因素是面包屑的位置。面包屑通常放置在页面顶部，主导航栏的上方或下方。用户不应该费力去查找面包屑，因此确保它们可见且易于定位。 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<nav class="main-nav">
  <a href="#">Home</a>
  <a href="#">Electronics</a>
  <a href="#">Phones</a>
</nav>
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
```

```css
.main-nav {
  background-color: #333;
  padding: 1rem;
}

.main-nav a {
  color: white;
  margin-right: 1rem;
  text-decoration: none;
}

.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

另一个需要考虑的是面包屑的大小。你要确保面包屑足够大以便于轻松读取，但又不能太大以至于占用页面过多空间。请记住，面包屑并不是作为主要导航工具，而是作为辅助导航工具。在页面信息量较大的网站中，用户可以轻松看到自己在层次结构中的位置，并使用面包屑导航返回到之前的页面。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
<p>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
</p>
```

```css
.breadcrumbs {
  font-size: 1rem;
  color: #555;
  margin: 1rem 0;
}   

p {
  font-size: 1.2rem;
  line-height: 1.6;
}
```

:::

那么你什么时候应该在你的网站上使用面包屑导航？如果你的网站结构简单且层次结构较浅，面包屑导航可能不是必需的。层次结构较浅意味着网站只有少数几个导航层级，因此面包屑导航可能不会带来太大价值。通常，你会在电子商务网站、新闻网站以及其他具有深层层次结构的网站上看到面包屑导航，比如技术文档网站。
