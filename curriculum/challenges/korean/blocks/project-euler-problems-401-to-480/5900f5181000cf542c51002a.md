# --description--

정수 수열 $S = \\{s_i\\}$가 $n$개의 원소를 가지고 각 원소 $s_i$가 $1 ≤ s_i ≤ n$을 만족하면 $n$-수열이라고 합니다. 따라서 총 $n^n$개의 서로 다른 $n$-수열이 존재합니다.

예를 들어, 수열 $S = \\{1, 5, 5, 10, 7, 7, 7, 2, 3, 7\\}$는 10-수열입니다.

임의의 수열 $S$에 대해, $L(S)$를 $S$에서 같은 값이 연속으로 나타나는 가장 긴 부분열의 길이라고 합시다. 위에 주어진 수열 $S$의 경우, 세 개의 연속된 7 때문에 $L(S) = 3$입니다.

모든 $n$-수열 $S$에 대해 $f(n) = \sum L(S)$라고 정의합니다.

예를 들어, $f(3) = 45$, $f(7) = 1\\,403\\,689$, 그리고 $f(11) = 481\\,496\\,895\\,121$입니다.

$f(7\\,500\\,000)\bmod 1\\,000\\,000\\,009$를 구하세요.
