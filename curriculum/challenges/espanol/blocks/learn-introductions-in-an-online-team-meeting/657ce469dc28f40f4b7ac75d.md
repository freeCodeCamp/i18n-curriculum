# --description--

`Is there` y `are there` se usan al comienzo de preguntas para preguntar sobre la existencia o presencia de algo.

- `Is there` se usa para sustantivos singulares, como en `Is there a solution?`

- `Are there` se usa para sustantivos plurales, como en `Are there any questions?`

# --questions--

## --text--

¿Cuál es la forma correcta de preguntar sobre la existencia de algo usando `there is` o `there are`?

## --answers--

`Is there a meeting scheduled for today?`

---

`Are there the new designer in the office?`

### --feedback--

Recuerda usar `is there` para singular y `are there` para plural. La forma correcta debería ser `Is there a new designer in the office?`

---

`Is there any chairs in the meeting room?`

### --feedback--

`Chairs` es plural, por lo que la forma correcta es `Are there any chairs in the meeting room?`

---

`Are there a new designer in the office?`

### --feedback--

Esta pregunta es sobre una persona, por lo que debe usar `is there` en lugar de `are there`. La forma correcta es `Is there a new designer in the office?`
