# --description--

함수가 호출되었지만 인수가 제공되지 않으면 매개변수에 `undefined` 값이 할당됩니다. 함수가 이 경우를 처리하도록 설계되지 않았다면 예상치 못한 결과가 발생할 수 있습니다.

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal()); // Output: undefined is my favorite animal!
```

위 예제에서 함수 `favoriteAnimal`는 인수 없이 호출됩니다. 매개변수 `animal`에 `undefined` 값이 할당되고, 이 값은 문자열 `" is my favorite animal!"`와 연결됩니다.

값이 제공되지 않을 경우 기본 매개변수를 사용할 수 있습니다. 기본 매개변수는 함수가 호출될 때 값이 제공되지 않으면 매개변수에 기본값을 할당할 수 있게 해줍니다.

```js
function favoriteAnimal(animal = 'Giant Panda') {
  return animal + " is my favorite animal!"
}
```

위 예제에서 매개변수 `animal`의 기본값은 `'Giant Panda'`으로 설정되어 있습니다. 함수 `favoriteAnimal`가 인수 없이 호출되면 매개변수 `animal`에 `'Giant Panda'` 값이 할당됩니다.
