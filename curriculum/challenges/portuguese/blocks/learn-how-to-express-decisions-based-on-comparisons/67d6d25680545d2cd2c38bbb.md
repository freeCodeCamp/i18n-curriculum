# --explanation--

`Sure` é uma palavra usada para mostrar concordância ou permissão.Por exemplo: 

`Sure, I can help you with that.` – Isso significa que o falante concorda ou está disposto a ajudar.

`What` é uma palavra interrogativa usada para pedir informações.Por exemplo:  

`What is your name?` – Isso significa que o falante quer saber um detalhe específico.

`Choices` refere-se às diferentes opções disponíveis.Por exemplo:  

`We have three choices for the new logo design.` – Isso significa que há três opções diferentes para escolher.
