# --description--

`challenge` означає завдання або проблему, для розв’язання якої потрібні навички та зусилля. У програмуванні `coding challenge` — це проблема або завдання, яке потрібно виконати за допомогою навичок кодування.

`Fresh` у цьому контексті означає новий або інший. Запит про `fresh perspective` означає пошук нового або іншого погляду на проблему чи завдання.
