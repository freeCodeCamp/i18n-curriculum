# --description--

Mbinu ya kugawanya na kushinda katika sayansi ya kompyuta ni mbinu ya kugawanya changamoto kuwa changamoto ndogo ndogo kwa kutumia mchakato wa kurudia. Moja ya vipengele muhimu vya mbinu hii ni kurudia, ambayo hutokea wakati kitendakazi kinapojitoa wito mara kwa mara hadi kifikie kesi ya msingi. Katika funzo hili, tutaangalia algoriti ya merge sort ili kuelewa vyema jinsi mbinu ya kugawanya na kushinda inavyofanya kazi.

Tuseme tuna orodha hii ya nambari:

```md
42 37 53 17
```

Lengo ni kupanga orodha hiyo kutoka ndogo hadi kubwa kwa kutumia algoriti ya merge sort. Hatua ya kwanza ni kugawanya orodha hiyo nusu:

```md
42 37 | 53 17
```

Kisha tunahitaji kuangalia upande wa kushoto wa orodha:

```md
42 37
```

Tunachukua orodha ndogo hiyo na kugawanya tena nusu hadi kila orodha ndogo iwe na kipengele kimoja tu ndani yake:

```md
42 | 37
```

Orodha yenye kipengele kimoja tu imepangwa kwa chaguo-msingi. Ifuatayo tunahitaji kuunganisha kila orodha ndogo yenye kipengele kimoja kuwa orodha iliyopangwa:

```md
37 42
```

Kisha tunafuata mchakato ule ule kwa upande wa kulia wa orodha ya awali:

```md
// right side of original list
53 17

// divide the list in half
53 | 17

// merge the lists in sorted order
17 53
```

Sasa kwamba nusu zote mbili za orodha ya awali zimepangwa, tunaunganisha nusu hizo mbili pamoja na kupanga vipengele:

```markdown
17 37 42 53
```

Hivi ndivyo algoriti inavyoonekana katika msimbo:

```javascript
function mergeSort(arr) {
    if (arr.length <= 1) {
        return arr;
    }

    const mid = Math.floor(arr.length / 2);
    const left = mergeSort(arr.slice(0, mid));
    const right = mergeSort(arr.slice(mid));

    const sorted = [];
    let i = 0;
    let j = 0;

    while (i < left.length && j < right.length) {
        if (left[i] <= right[j]) {
            sorted.push(left[i]);
            i += 1;
        } else {
            sorted.push(right[j]);
            j += 1;
        }
    }

    return sorted.concat(left.slice(i)).concat(right.slice(j));
}
```

Ugumu wa wakati kwa algoriti ya merge sort ni `O(n log n)` kwa sababu orodha hugawanywa mara kwa mara nusu (`log n`) kisha kuunganishwa pamoja (`O(n)`). Tofauti na algoriti nyingine za kupanga kama bubble sort, merge sort si algoriti ya kupanga mahali pake na ina ugumu wa nafasi wa `O(n)`.
