# --description--

以下方程表现了山区的连续地形，给出任意点 ($x$,$y$) 的海拔 $h$：

$$h = \left(5000 - \frac{x^2 + y^2 + xy}{200} + \frac{25(x + y)}{2}\right) \times e^{-\left|\frac{x^2 + y^2}{1\\,000\\,000} - \frac{3(x + y)}{2000} + \frac{7}{10}\right|}$$

一只蚊子打算从 A(200,200) 飞到 B(1400,1400)，且不离开由 $0 ≤ x$，$y ≤ 1600$ 给定的区域。

由于中间的山脉，它首先垂直上升到点 A'，其海拔为 $f$。然后，在保持相同海拔 $f$ 的情况下，它绕过任何障碍物飞行，直到到达正上方的点 B'。

首先，确定 $f_{min}$，即允许从 A 到 B 进行此类旅行且保持在指定区域内的最小常量高度。然后，查找在该常量高度 $f_{min}$ 飞行时 A' 和 B' 之间的最短路径长度。

将该长度作为答案，四舍五入到小数点后三位。

**注意：** 为方便起见，上述海拔函数在下面重复，以适合大多数编程语言的形式：`h=( 5000-0.005*(x*x+y*y+x*y)+12.5*(x+y) )* exp( -abs(0.000001*(x*x+y*y)-0.0015*(x+y)+0.7) )`。
