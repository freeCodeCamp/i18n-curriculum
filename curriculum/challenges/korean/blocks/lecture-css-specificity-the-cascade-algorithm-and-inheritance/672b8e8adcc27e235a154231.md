# --interactive--

타입 선택자, 또는 요소 선택자는 태그 이름을 기준으로 요소를 선택합니다.

이 선택자는 CSS에서 기본적이며 특정 HTML 요소의 모든 인스턴스에 스타일을 적용할 수 있게 해줍니다.

타입 선택자는 사용하기 간단하며 스타일을 적용하려는 요소의 태그 이름만 적으면 됩니다.

다음은 페이지의 모든 문단 요소를 대상으로 하는 타입 선택자의 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  color: blue;
}
```

:::

이 예제에서 모든 `p` 요소는 텍스트 색상이 `blue`로 설정됩니다.

타입 선택자는 다른 선택자에 비해 특이성이 비교적 낮습니다. 타입 선택자의 특이성 값은 `(0, 0, 0, 1)`입니다.

즉, 타입 선택자는 클래스 선택자, ID 선택자, 인라인 스타일에 의해 재정의되지만, 그런 높은 특이성 규칙이 없으면 스타일을 적용할 수 있습니다.

이제 클래스 선택자가 타입 선택자의 스타일을 재정의하는 예를 살펴보겠습니다.

다음은 두 개의 문단 요소가 있는 예입니다:

:::interactive_editor

```html
<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

:::

두 문단 요소 모두 `para`라는 클래스를 가지고 있습니다.

CSS 파일 안에서 타입 선택자는 문단을 대상으로 하고, 클래스 선택자는 `para` 클래스를 가진 요소를 대상으로 합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

```css
p {
  color: blue;
}

.para {
  color: red;
}
```

:::

페이지의 모든 `para` 클래스를 가진 문단은 텍스트 색상이 `red`로 설정되며, `blue`가 아닌 이유는 클래스 선택자가 타입 선택자보다 특이성이 더 높기 때문입니다.
