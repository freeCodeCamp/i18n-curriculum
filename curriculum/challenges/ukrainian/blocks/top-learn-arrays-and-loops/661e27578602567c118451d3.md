# --description--

Один із найпоширеніших способів додати новий елемент до масиву — це використати метод `push()`. Метод `push()` додає один або кілька елементів у кінець масиву і повертає нову довжину масиву.

Наприклад, щоб додати новий елемент до масиву `pet`, ви можете використати такий код:

```javascript
const pet = ['cat', 'dog', 'bunny'];
pet.push('parrot');
console.log(pet); // Output: ['cat', 'dog', 'bunny', 'parrot']
```

Щоб видалити останній елемент масиву, ви можете використати метод `pop()`. Метод `pop()` видаляє останній елемент із масиву і повертає цей елемент.

Наприклад, щоб видалити останній елемент із масиву `pet`, ви можете використати такий код:

```javascript
const pet = ['cat', 'dog', 'tiger'];
pet.pop();
console.log(pet); // Output: ['cat', 'dog']
```
