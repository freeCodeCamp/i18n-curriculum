# --description--

تُستخدم `There is` و`there are` للدلالة على وجود أو تواجد شيء ما. تُستخدم `There is` مع الأسماء المفردة أو غير المعدودة، بينما تُستخدم `there are` مع الأسماء الجمع.

أمثلة:

- `There is a book on the table.` (اسم مفرد)

- `There is some water in the glass.` (اسم غير معدود)

- `There are three apples in the basket.` (اسم جمع)
