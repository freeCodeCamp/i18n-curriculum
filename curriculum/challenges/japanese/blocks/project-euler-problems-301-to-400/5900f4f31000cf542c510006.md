# --description--

$s_k$ を、0 から $k$ までの数を2進数で書いたときの1の数とします。

例えば、0から5までを2進数で書くと、0、1、10、11、100、101 となります。1は7個あるので、$s_5 = 7$ です。

数列 $S = \\{s_k : k ≥ 0\\}$ は $\\{0, 1, 2, 4, 5, 7, 9, 12, \ldots\\}$ から始まります。

2人のプレイヤーでゲームを行います。ゲーム開始前に数値 $n$ が選ばれます。カウンター $c$ は0から始まります。各ターンで、プレイヤーは1から $n$ までの数（両端含む）を選び、その数だけ $c$ を増やします。増やした結果の $c$ は必ず $S$ のメンバーでなければなりません。もし有効な手がなくなったら、そのプレイヤーの負けです。

例えば、$n = 5$ で $c = 0$ から始めると：

- プレイヤー1は4を選び、$c$ は $0 + 4 = 4$ になります。
- プレイヤー2は5を選び、$c$ は $4 + 5 = 9$ になります。
- プレイヤー1は3を選び、$c$ は $9 + 3 = 12$ になります。
- など。

$c$ は常に $S$ に属し、各プレイヤーは $c$ を最大で $n$ だけ増やせます。

$M(n)$ を、先手プレイヤーが勝つために最初のターンで選べる最大の数とし、そのような手がなければ $M(n) = 0$ とします。例えば、$M(2) = 2$、$M(7) = 1$、$M(20) = 4$ です。

$1 ≤ n ≤ 20$ のとき、$\sum M{(n)}^3 = 8150$ であることが確認できます。

$1 ≤ n ≤ 1000$ のときの $\sum M{(n)}^3$ を求めてください。
