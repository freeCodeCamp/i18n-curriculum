# --description--

밑변 길이 $b = 16$이고 두 변 길이 $L = 17$인 이등변삼각형을 생각해보세요.

<img alt="L로 이름 붙여진 변 두 개가 같은 길이이고, 삼각형 밑변은 b; 삼각형 높이 h는 밑변에서 L 변 사이 각까지의 거리" src="https://cdn.freecodecamp.org/curriculum/project-euler/special-isosceles-triangles.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

피타고라스 정리를 사용하면 삼각형 높이 $h = \sqrt{{17}^2 − 8^2} = 15$임을 알 수 있는데, 이는 밑변 길이보다 1 작습니다.

$b = 272$이고 $L = 305$일 때, $h = 273$이 되어 밑변 길이보다 1 크며, 이것이 $h = b ± 1$ 조건을 만족하는 두 번째로 작은 이등변삼각형입니다.

$h = b ± 1$이고 $b$, $L$이 양의 정수인 12개의 가장 작은 이등변삼각형에 대해 $\sum{L}$ 값을 구하세요.
