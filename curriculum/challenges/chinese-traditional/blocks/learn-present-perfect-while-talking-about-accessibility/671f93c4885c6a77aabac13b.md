# --explanation--

`Concerning` 用來描述引起擔憂或焦慮的事物。 例如：

`The drop in performance is quite concerning.`－這表示效率、性能的下降是令人擔憂的原因。

`Found out` 是 `find out` 的過去式。它表示發現或學習先前未知的事情。 例如，

`She found out the reason for the delay.`－它指的是了解有關延遲原因的新資訊。
