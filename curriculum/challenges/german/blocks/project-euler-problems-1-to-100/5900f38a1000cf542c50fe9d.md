# --description--

Überraschenderweise gibt es nur drei Zahlen, die als Summe der vierten Potenzen ihrer Ziffern geschrieben werden können:

<div style='margin-left: 4em;'>
  1634 = 1<sup>4</sup> + 6<sup>4</sup> + 3<sup>4</sup> + 4<sup>4</sup><br>
  8208 = 8<sup>4</sup> + 2<sup>4</sup> + 0<sup>4</sup> + 8<sup>4</sup><br>
  9474 = 9<sup>4</sup> + 4<sup>4</sup> + 7<sup>4</sup> + 4<sup>4</sup><br>
</div>

Da 1 = 1<sup>4</sup> keine Summe ist, wird es nicht mit einbezogen.

Die Summe dieser Zahlen ist 1634 + 8208 + 9474 = 19316.

Finden Sie die Summe aller Zahlen, die als Summe von `n` Potenzen ihrer Ziffern geschrieben werden können.
