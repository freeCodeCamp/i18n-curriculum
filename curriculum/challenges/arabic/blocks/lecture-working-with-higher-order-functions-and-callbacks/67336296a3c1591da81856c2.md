# --interactive--

الدوال ذات الرتبة العليا هي مفهوم قوي في JavaScript يمكنه تعزيز مهاراتك في البرمجة بشكل كبير وجعل شفرتك أكثر مرونة وقابلية لإعادة الاستخدام.

باختصار، الدالة ذات الرتبة العليا هي دالة تأخذ دالة أو أكثر كمعاملات، أو تُرجع دالة، أو تفعل كلا الأمرين.

لفهم الدوال ذات الرتبة العليا، دعنا أولًا نعتبر الدوال كمواطنين من الدرجة الأولى في JavaScript. هذا يعني أن الدوال يمكن التعامل معها مثل أي قيمة أخرى – يمكن تعيينها إلى متغيرات، وتمريرها كمعاملات إلى دوال أخرى، وإرجاعها من دوال. هذه المرونة هي التي تمكّن من إنشاء واستخدام الدوال ذات الرتبة العليا.

استخدام شائع للدوال ذات الرتبة العليا هو تجريد العمليات المعقدة.

على سبيل المثال، قد يكون لديك دالة تؤدي عملية محددة على كل عنصر في مصفوفة. بدلًا من كتابة دوال منفصلة لعمليات مختلفة، يمكنك إنشاء دالة ذات رتبة عليا تأخذ العملية كمعلمة. هذا يسمح لك بإعادة استخدام نفس هيكل الدالة مع سلوكيات مختلفة.

إليك مثالًا لتوضيح هذا المفهوم:

:::interactive_editor

```js
function operateOnArray(arr, operation) {
  let result = [];
  for (let i = 0; i < arr.length; i++) {
    result.push(operation(arr[i]));
}
  return result;
}

function double(x) {
    return x * 2;
}

let numbers = [1, 2, 3, 4, 5];
let doubledNumbers = operateOnArray(numbers, double);
console.log(doubledNumbers); // [2, 4, 6, 8, 10]
```

:::

في هذا المثال، `operateOnArray` هي دالة ذات رتبة عليا. تأخذ مصفوفة ودالة (`operation`) كمعاملات. ثم تطبق العملية على كل عنصر في المصفوفة. الدالة المزدوجة تُمرر كمعلمة إلى `operateOnArray`، مما يوضح كيف يمكن استخدام الدوال كقيم.

يمكن للدوال ذات الرتبة العليا أيضًا أن تُرجع دوالًا. هذا مفيد بشكل خاص لإنشاء دوال متخصصة بناءً على دوال أكثر عمومية. يُشار إلى هذا المفهوم غالبًا باسم مصانع الدوال. إليك مثالًا:

:::interactive_editor

```js
function multiplyBy(factor) {
  return function(number) {
    return number * factor;
  }
}

let double = multiplyBy(2);
let triple = multiplyBy(3);

console.log(double(5)); // 10
console.log(triple(5)); // 15
```

:::

في هذه الحالة، `multiplyBy` هي دالة ذات رتبة عليا تُرجع دالة جديدة. هذه الدالة الجديدة متخصصة بناءً على العامل الممرر إلى `multiplyBy`. هذا يسمح لنا بإنشاء دوال ضرب مخصصة بسهولة.

الدوال ذات الرتبة العليا ليست مجرد مفهوم نظري – بل تُستخدم على نطاق واسع في JavaScript.

العديد من الطرق المدمجة للمصفوفات في JavaScript، مثل `map()` و `filter()` و `reduce()`، هي دوال ذات رتبة عليا. هذه الطرق تأخذ دالة كمعلمة وتطبقها على عناصر المصفوفة بطرق مختلفة. ستتعلم المزيد عن هذه الطرق في الدروس القادمة.

استخدام الدوال ذات الرتبة العليا يمكن أن يؤدي إلى شفرة أكثر تعبيرًا وأسهل للفهم.

بدلًا من وصف كيفية إنجاز مهمة خطوة بخطوة (البرمجة الإجرائية)، تسمح لك الدوال ذات الرتبة العليا بوصف ما تريد إنجازه (البرمجة التصريحية). هذا يمكن أن يجعل شفرتك أكثر قابلية للقراءة والصيانة.

مع استمرارك في العمل مع JavaScript، ستواجه وتستخدم الدوال ذات الرتبة العليا كثيرًا. إنها جزء أساسي من البرمجة الوظيفية في JavaScript وضرورية لكتابة شفرة نظيفة وفعالة ومرنة.

فهم واستخدام الدوال ذات الرتبة العليا سيعزز بشكل كبير قدرتك على كتابة برامج JavaScript متطورة وأنيقة.
