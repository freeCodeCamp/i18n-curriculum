# --description--

最後的因素、最後的關鍵、決勝的決勝。如果在考慮了所有其他因素後，仍有多個相互衝突的規則作用於同一標的，串接規則如何決定要套用哪一個規則？

其實非常簡單。最後定義的規則即為勝出者。

```css
/* styles.css */

.alert {
  color: red;
}

.warning {
  color: yellow;
}
```

對於同時具有 `alert` 和 `warning` 類別的元素，層疊會通過所有其他因素，包括繼承（此處無）和特異性（兩個規則的特異性皆相同）。由於 `.warning` 規則是最後定義的，且沒有其他因素能決定應用哪個規則，因此會將其套用到該元素。
