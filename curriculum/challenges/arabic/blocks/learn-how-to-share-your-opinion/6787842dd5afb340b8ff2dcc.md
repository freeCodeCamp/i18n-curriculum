# --explanation--

عند سؤالك لماذا اخترت Python كلغتك الأولى، من الأفضل إبراز المزايا المحددة لـ Python.

تُعرف Python بأنها مناسبة للمبتدئين، بصيغة نحوية بسيطة يسهل قراءتها وتعلمها. بالإضافة إلى ذلك، تمتلك Python مجموعة رائعة من المكتبات وإطارات العمل، مما يجعلها خيارًا عمليًا للمبتدئين.

من ناحية أخرى، قول `I've always wanted to be a developer` لا يشرح سبب اختيارك Python. يركز على الرغبة في أن تصبح مطوِّرًا لكنه يفتقر إلى أي سبب محدد يتعلق بـ Python.
