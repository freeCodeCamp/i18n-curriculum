# --description--

`to debug` 這個術語在程式設計中非常重要。它的意思是尋找並修正電腦程式中的錯誤或漏洞。`bug` 是程式中阻止其正常運作的錯誤或問題。

`Debugging` 是程式設計師工作中經常的部分。

例如，`She spent the whole day debugging the code.` 這表示她正在修正程式碼中的錯誤。
