# --explanation--

`Exactly` — це прислівник, який використовується, щоб показати повну згоду або правильність того, що хтось сказав. `Exact`, натомість, — це прикметник, що означає точний або правильний у деталях. Наприклад:

- `You are exactly right about the cause of the problem.` – Це показує повну згоду. Виражає, наскільки людина була права (прикметник).

- `We need the exact measurements to build the new server rack.` – Це означає дуже точні виміри. Визначає іменник, а не дієслово чи прикметник.
