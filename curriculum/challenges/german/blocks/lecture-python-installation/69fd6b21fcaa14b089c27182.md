# --description--

In der vorherigen Lektion haben Sie gelernt, wie Sie Python auf Ihrem lokalen Gerät installieren. In dieser Lektion lernen Sie, wie Sie Python-Code lokal ausführen.

Bevor Sie mit dem Erstellen Ihrer Python-Skripte lokal beginnen können, müssen Sie einen Code-Editor oder eine IDE installieren. IDE steht für Integrated Development Environment und bietet Funktionen wie einen Code-Editor, Testwerkzeuge, ein Terminal und mehr.

Ein beliebter Code-Editor, den viele Entwickler verwenden, ist VS Code. Besuchen Sie `https://code.visualstudio.com/download`, um VS Code herunterzuladen, das Mac-, Windows- und Linux-Umgebungen unterstützt. Dies ist der Editor, der in dieser Lektion verwendet wird.

Weitere gängige Optionen für die Python-Entwicklung sind:

- PyCharm
- Spyder

Nachdem Sie Ihren Editor heruntergeladen haben, müssen Sie ihn öffnen. Sobald er geöffnet ist, müssen Sie einen Ordner öffnen. Sie können auf die Menüoption `File` oben links klicken und dann `Open Folder` auswählen. Von dort aus können Sie einen Ordner für Ihr Projekt auswählen oder erstellen. Für zusätzliches Üben außerhalb der Lektion möchten Sie vielleicht einen Ordner namens `python-projects` anlegen.

Sobald Ihr Ordner eingerichtet ist, können Sie den Explorer öffnen, das ist die linke Seitenleiste in VS Code. Klicken Sie auf das Symbol `New File`, das wie ein Blatt Papier mit einem Pluszeichen aussieht. Erstellen Sie dann eine Datei namens `main.py` und drücken Sie Enter. Die `.py` ist eine Dateiendung, die signalisiert, dass es sich um eine Python-Datei handelt.

Öffnen Sie Ihre `main.py`-Datei und geben Sie folgenden Code ein:

```py
print("Hello, world!")
```

Um diesen Code auszuführen, haben Sie mehrere Möglichkeiten. Die erste Möglichkeit ist, auf die Schaltfläche zum Ausführen zu klicken, die wie ein Play-Button oben rechts aussieht. Dadurch wird ein Terminal geöffnet und Ihr Python-Skript dort ausgeführt. Sie sollten den Text `"Hello, world!"` sehen.

Eine andere Möglichkeit ist, ein Terminal zu öffnen und das Programm manuell mit folgendem Befehl auszuführen:

```bash
python main.py
```

Sie müssen diesen Befehl aus dem Ordner ausführen, in dem `main.py` gespeichert ist. Wenn sich `main.py` beispielsweise in einem Ordner namens `python-projects` befindet, verwenden Sie zuerst Folgendes im Terminal:

```bash
cd python-projects
```

Dann führen Sie aus:

```bash
python main.py
```

In einigen Umgebungen, besonders auf macOS- und Linux-Systemen, auf denen sowohl Python 2 als auch Python 3 installiert sind, müssen Sie möglicherweise `python3` anstelle von `python` verwenden. Das ist üblich, wenn der Befehl `python` entweder nicht existiert oder auf eine ältere Python-Version verweist.

```bash
python3 main.py
```

Während Sie im Kurs voranschreiten, sollten Sie die Beispiele aus den Lektionen lokal ausprobieren und die Ergebnisse ansehen. Versuchen Sie auch, eigene Beispiele zu erstellen, um das Gelernte aus den Lektionen zu testen.
