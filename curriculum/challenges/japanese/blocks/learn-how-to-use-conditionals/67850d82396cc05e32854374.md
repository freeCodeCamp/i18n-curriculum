# --explanation--

- 主文から助動詞（例：`would`、`is`、`has`、`will`）を取る。

- 反対の形を使う：肯定なら否定、否定なら肯定。

- 適切な代名詞を付け加える。

- 主節は`It would affect a lot of tech companies`（肯定形）です。

- タグは`wouldn't it?`（`would`の否定形）です。
