# --explanation--

El `Third Conditional` se usa para hablar de situaciones irreales en el pasado — cosas que no ocurrieron — y sus posibles resultados. Su estructura es `If` + `past perfect`, modal (`could/might/should`) + `have` + `past participle`. Usar este condicional significa que la acción en la cláusula `if` no ocurrió, y el resultado no se dio, pero imaginas que fue un resultado posible. Por ejemplo:

`If we had tested the system earlier, we might have prevented the failure.` – Esto significa que no se realizó la prueba y ocurrió la falla, pero si se hubiera hecho, la falla podría haberse evitado.
