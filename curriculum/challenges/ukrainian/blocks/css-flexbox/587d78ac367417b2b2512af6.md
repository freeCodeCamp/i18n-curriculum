# --description--

Іноді гнучкі елементи всередині гнучкого контейнера не заповнюють весь простір контейнера. Часто хочеться вказати CSS, як вирівняти та розподілити гнучкі елементи певним чином. На щастя, властивість `justify-content` має кілька варіантів для цього. Але спочатку потрібно зрозуміти важливу термінологію перед тим, як розглядати ці варіанти.

<a href="https://www.freecodecamp.org/news/flexbox-the-ultimate-css-flex-cheatsheet/" target="_blank" rel="noopener noreferrer nofollow">Для детальнішої інформації про властивості гнучкого контейнера читайте тут</a>

Нагадаємо, що встановлення гнучкого контейнера як рядка розміщує гнучкі елементи поруч зліва направо. Гнучкий контейнер, встановлений як стовпець, розміщує гнучкі елементи вертикально зверху вниз. Напрямок, у якому розташовані гнучкі елементи, називається **головною віссю**. Для рядка це горизонтальна лінія, що проходить через кожен елемент. Для стовпця головна вісь — це вертикальна лінія через елементи.

Існує кілька варіантів, як розподілити гнучкі елементи вздовж лінії, що є головною віссю. Один із найпоширеніших — `justify-content: center;`, який вирівнює всі гнучкі елементи по центру всередині гнучкого контейнера. Інші варіанти включають:

<ul><li><code>flex-start</code>: вирівнює елементи на початку гнучкого контейнера. Для рядка це штовхає елементи до лівого краю контейнера. Для стовпця — до верхнього краю контейнера. Це вирівнювання за замовчуванням, якщо не вказано <code>justify-content</code>.</li><li><code>flex-end</code>: вирівнює елементи в кінці гнучкого контейнера. Для рядка це штовхає елементи до правого краю контейнера. Для стовпця — до нижнього краю контейнера.</li><li><code>space-between</code>: вирівнює елементи по центру головної осі, розподіляючи додатковий простір між елементами. Перший і останній елементи притиснуті до самого краю гнучкого контейнера. Наприклад, у рядку перший елемент прилягає до лівого краю контейнера, останній — до правого, а решта простору рівномірно розподілена між іншими елементами.</li><li><code>space-around</code>: схоже на <code>space-between</code>, але перший і останній елементи не закріплені за краями контейнера, простір розподіляється навколо всіх елементів із половинним пробілом на кожному кінці гнучкого контейнера.</li><li><code>space-evenly</code>: рівномірно розподіляє простір між гнучкими елементами з повним пробілом на кожному кінці гнучкого контейнера.</li></ul>

# --instructions--

Приклад допоможе показати цю властивість у дії. Додайте CSS-властивість `justify-content` до елемента `#box-container` і задайте їй значення `center`.

**Бонус**  
Спробуйте інші варіанти властивості `justify-content` у редакторі коду, щоб побачити їхні відмінності. Але зверніть увагу, що значення `center` — єдине, що пройде це завдання.

# --hints--

Елемент `#box-container` повинен мати властивість `justify-content` зі значенням `center`.

```js
const boxContainer = document.querySelector('#box-container');
const justifyDirection =
  window.getComputedStyle(boxContainer)['justify-content'];
assert.strictEqual(justifyDirection, 'center');
```
