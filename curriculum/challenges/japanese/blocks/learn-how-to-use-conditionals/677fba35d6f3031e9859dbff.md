# --explanation--

文の最後にある`Right`は付加疑問文です。これは同意を確認したり、聞き手が理解しているかを確かめるために使われます。例えば：

- `It's going to rain today, right?` - 話者が天気について確認を求めています。

- `You finished the project, right?` - 話者が聞き手が課題を終えたかどうかを確認しています。

この会話では、ボブがマリアに自分の発言に同意するか確認しています。
