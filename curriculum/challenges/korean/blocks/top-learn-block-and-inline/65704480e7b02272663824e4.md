## --text--

CSS 레이아웃에서 "Normal flow"란 무엇인가요?

## --answers--

Normal flow는 CSS 레이아웃에서 요소들이 기본적으로 배치되는 방식을 의미합니다.

---

Normal flow는 CSS 레이아웃에서 요소들의 가시성을 제어하는 속성입니다.

---

Normal flow는 반응형 디자인을 위한 미디어 쿼리 순서를 정의하는 CSS 규칙입니다.

---

Normal flow는 CSS 레이아웃에서 요소들을 애니메이션하는 과정을 의미합니다.
