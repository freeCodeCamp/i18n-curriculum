# --description--

У будь-якій мові програмування потрібно порівнювати значення, щоб приймати рішення. Наприклад, якщо погода сонячна, ви вийдете на вулицю, інакше залишитесь вдома. Інший приклад — перевірити, чи має користувач достатньо очок, щоб піднятися на наступний рівень у грі.

Інструкція `if-else` використовується для прийняття рішень у JavaScript. Її часто застосовують разом з операторами порівняння, щоб порівнювати значення і приймати рішення на основі результатів.

Приклад оператора if:

```javascript
let x = 5;

if (x > 3) {
  console.log("x is greater than 3");
} else {
  console.log("x is less than or equal to 3");
}
```

У цьому прикладі змінна `x` порівнюється з числом `3` за допомогою оператора `>`. Якщо `x` більше за `3`, виконається перший блок коду. Інакше виконається другий блок коду.
