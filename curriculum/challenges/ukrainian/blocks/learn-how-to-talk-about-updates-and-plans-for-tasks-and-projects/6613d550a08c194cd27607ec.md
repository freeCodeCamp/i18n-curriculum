# --description--

`As for` — це елемент, який використовується для зміни фокусу на іншу тему або аспект обговорення.

Наприклад, на нараді, після обговорення маркетингової стратегії, ви можете сказати `As for the sales plan, we'll start implementing it next quarter`, щоб змінити тему на план продажів.
