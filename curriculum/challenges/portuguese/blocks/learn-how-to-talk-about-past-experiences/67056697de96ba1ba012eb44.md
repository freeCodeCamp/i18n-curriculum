# --explanation--

Conectores sequenciais como `first`, `then`, `after that` e `finally` são usados para descrever a ordem das ações ou eventos.

- `First` é usado para descrever a ação inicial.

- `Then` refere-se ao próximo passo imediato.

- `After that` é usado para introduzir uma etapa que segue outra ação com alguma distinção entre elas.

- `Finally` marca a última etapa em uma sequência.
