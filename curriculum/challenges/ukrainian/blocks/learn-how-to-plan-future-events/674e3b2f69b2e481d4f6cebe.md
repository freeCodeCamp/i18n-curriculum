# --explanation--

У реченні використано `will` для позначення часу `Future Simple`, що показує, що дія очікується в майбутньому.

Конструкція `will + verb` часто використовується для розмов про майбутні події, плани або передбачення. Наприклад:

- `They will be at the meeting tomorrow.` — це означає, що їхня присутність запланована на майбутнє.

- `We will start the project next week.` — це означає, що проєкт заплановано розпочати в майбутньому.

Використання `will` — це простий спосіб висловити наміри, обіцянки або розклад на майбутнє.
