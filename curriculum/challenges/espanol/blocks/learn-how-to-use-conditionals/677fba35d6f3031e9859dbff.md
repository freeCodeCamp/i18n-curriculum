# --explanation--

`Right` al final de una oración es una coletilla interrogativa. Se usa para confirmar acuerdo o verificar si el oyente entiende. Por ejemplo:

- `It's going to rain today, right?` - El hablante está pidiendo confirmación sobre el clima.

- `You finished the project, right?` - El hablante está verificando si el oyente completó la tarea.

En este diálogo, Bob le está pidiendo a Maria que confirme o esté de acuerdo con su afirmación.
