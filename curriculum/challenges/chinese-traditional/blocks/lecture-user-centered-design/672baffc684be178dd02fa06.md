# --interactive--

**注意**：有些互動範例可能會使用你還沒學過的 CSS。不要擔心試著理解所有的程式碼。這些範例的目標是為了展示這些設計概念的預覽，讓你更了解運作方式。要看到預覽，你需要啟用互動式編輯器。

卡片組件在電子商務、社交媒體和新聞網站中非常常見。它們用於幫助以結構化的方式顯示資訊。當你設計你的卡片時，了解最佳實踐非常重要，這樣你的使用者才能輕鬆理解你想傳達的資訊。

卡片設計的首要考量應該是簡潔。你不希望你的卡片在視覺上過於雜亂或顯示過多資訊。例如，如果卡片設計在視覺上過於雜亂，使用者將會有太多資訊需要一次處理。 

這是一個雜亂卡片設計的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2><strong><em>Product Name: Ultimate Edition</em></strong></h2>
  <p>
    This is a great product that you will love! It has many features and benefits that will make your life easier and more enjoyable. Whether you're at home or on the go, this product is perfect for all situations. Don’t miss out on this incredible offer, buy now and experience the difference!
  </p>
  <button>Buy Now</button>
  <button>Learn More</button>
  <button>Add to Wishlist</button>
  <p><strong>Only 3 left in stock!</strong></p>
  <p><em>Rated 4.8 stars by 2,391 customers</em></p>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}
.card p {
  font-size: 1em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

卡片上的資訊較少且條款之間有良好間距，讓使用者更容易處理資訊，並允許頁面上有多張卡片。

這是一個簡單卡片設計的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2>Product Name</h2>
  <button>Buy Now</button>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 200px;
  text-align: center;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

另一件需要考慮的事情是使用者可以在哪裡點擊卡片。有些卡片設計會有單一按鈕，使使用者可以點擊的位置顯而易見。其他卡片設計則會讓整張卡片都可點擊。當使用者暫留在卡片的任何零件時，卡片會改變顏色或有陰影效果，以顯示卡片是可點擊的。無論你選擇哪種設計，都需要在整個網站中保持一致，並且讓使用者容易理解。 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <a href="#">
    <img src="https://placehold.co/150x150" alt="Example Product Image">
    <h2>Product Name</h2>
    <button>Buy Now</button>
  </a>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
  cursor: pointer;
}

.card:hover {
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
}

.card a {
  text-decoration: none;
  color: inherit;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

另一個需要考慮的是在你的卡片上使用媒體。選擇高品質的媒體可以大幅提升使用者體驗。如果你在產品卡片中使用影像或影片，品質越高，使用者對該產品的興趣就越大。但如果你使用低品質的媒體，使用者可能不會信任你的產品和服務的品質。 

最後要考慮的其中一件事是顏色階層體系的使用。你要確保卡片上最重要的資訊是最顯眼的。你可以為像是呼叫行動按鈕這類重要元素使用鮮豔的顏色，並為卡片上較不重要的項目使用淡色。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2>Product Name</h2>
  <button>Buy Now</button>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
  text-align: center;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #28a745; 
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

當你繼續開發網頁應用程式時，請記住卡片設計的不同最佳實踐。這將幫助你為你的使用者創建更好的使用者體驗。
