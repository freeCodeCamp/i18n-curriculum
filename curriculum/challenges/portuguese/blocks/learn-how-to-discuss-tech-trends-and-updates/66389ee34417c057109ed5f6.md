# --description--

Automatizar significa fazer um processo operar sozinho sem precisar da ajuda de uma pessoa.Em programação, pode envolver o uso de software para escrever código, testar aplicações ou encontrar e corrigir erros.Por exemplo, uma cafeteira que liga sozinha pela manhã para fazer café é `automated`.

A palavra `certain` é usada para falar sobre coisas ou pessoas específicas, mas não todas as coisas ou pessoas.Por exemplo, se você disser `certain books are on sale`, isso significa que apenas alguns livros estão à venda, não todos eles.
