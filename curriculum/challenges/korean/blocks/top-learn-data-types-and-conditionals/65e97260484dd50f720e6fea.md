# --description--

백틱을 사용해 정의한 문자열을 템플릿 리터럴이라고 합니다. 템플릿 리터럴은 JavaScript에서 문자열을 정의하는 새로운 방법입니다. 문자열 안에 표현식을 임베드할 수 있게 해줍니다. 이는 표현식을 `${}`로 감싸서 합니다.

예를 들어, 다음 코드는:

```javascript
let name = "John";
let age = 25;
let greeting = `Hello, my name is ${name} and I am ${age} years old.`;
```

`greeting` 변수에 "Hello, my name is John and I am 25 years old."라는 문자열이 저장됩니다.

다음 중 JavaScript에서 템플릿 리터럴을 정의하는 올바른 방법은 무엇인가요?

## --answers--

``` `Hello, my name is ${name} and I am ${age} years old.` ```

``` `Hello, my name is `name` and I am `age` years old.` ```

---

``` `Hello, my name is {name} and I am {age} years old.` ```

---

``` `Hello, my name is $name and I am $age years old.` ```
