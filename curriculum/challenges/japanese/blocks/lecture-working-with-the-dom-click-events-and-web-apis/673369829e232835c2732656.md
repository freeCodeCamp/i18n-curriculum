# --interactive--

ダイアログはユーザーに重要な情報や操作を表示するためのものです。HTMLの組み込み`dialog`要素を使えば、ウェブアプリでこれらのダイアログ（モーダルおよび非モーダルの両方）を簡単に作成できます。

モーダルダイアログは、ユーザーがそれとやり取りしない限り、アプリケーションやウェブページの他の部分にアクセスできなくするタイプのダイアログです。ユーザーがダイアログを閉じるかフォームを提出するなどの操作を完了するまで、他のコンテンツとのやり取りを実質的にブロックします。

一方、非モーダルダイアログは、ダイアログが開いている間もページやアプリケーションの他の部分とやり取りを続けられます。残りのコンテンツへのアクセスを妨げません。

このレッスンでは、`showModal()`と`close()`メソッドを使ってモーダルを開閉する方法を学びます。

モーダルの特定の操作やメッセージにユーザーのフォーカスを確実に向けたい場合は、`showModal()`メソッドを使ってモーダルダイアログを開くことができます。これにより、ページ上の他の要素にバックドロップが追加され、それらが無効化されます。フォーム、確認、ユーザーの操作が必要な重要な情報を表示するモーダルに最適です。

モーダルダイアログのHTMLは以下の通りです：

:::interactive_editor

```html
<dialog id="my-modal">
  <p>This is a modal dialog.</p>
</dialog>
```

:::

現時点では、初期レンダリング時にモーダルが閉じているため、ページ上に何も表示されません。`showModal()`メソッドを使うとモーダルを自動的に開けます：

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
dialog.showModal();
```

:::

ブラウザの結果は、テキスト`This is a modal dialog.`を含むモーダルが表示されます。

ユーザーに操作を任せるのが最善です。そのために、ボタンにクリックイベントリスナーを追加し、`showModal()`メソッドを使うことができます：

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.showModal();
});
```

:::

ダイアログを表示しつつ、ダイアログ外のコンテンツとのやり取りも許可したい場合は、`show()`メソッドを使えます：

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});
```

:::

モーダルを閉じるには、`dialog`要素内のモーダルにボタンを追加し、`close()`メソッドを使います：

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
  <button id="close-modal-btn">Close Modal</button>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");
const closeButton = document.getElementById("close-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});

closeButton.addEventListener("click", () => {
  dialog.close();
});
```

:::
