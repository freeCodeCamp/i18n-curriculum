# --description--

5개 중 3개를 선택하는 방법은 정확히 열 가지가 있습니다, 12345:

<div style='text-align: center;'>123, 124, 125, 134, 135, 145, 234, 235, 245, 그리고 345</div>

조합론에서는 다음 표기법을 사용합니다, $\\displaystyle \\binom 5 3 = 10$

일반적으로, $\\displaystyle \\binom n r = \\dfrac{n!}{r!(n-r)!}$이며, 여기서 $r \\le n$, $n! = n \\times (n-1) \\times ... \\times 3 \\times 2 \\times 1$, 그리고 $0! = 1$입니다.

값이 백만을 넘는 것은 $n = 23$일 때부터입니다: $\\displaystyle \\binom {23} {10} = 1144066$.

$1 \\le n \\le 100$ 범위에서, 백만보다 큰 $\\displaystyle \\binom n r$ 값은 몇 개일까요? 중복된 값일 수도 있습니다.

```js
function combinatoricSelections(limit) {
    const factorial = n =>
        Array.apply(null, { length: n })
            .map((_, i) => i + 1)
            .reduce((p, c) => p * c, 1);

    let result = 0;
    const nMax = 100;

    for (let n = 1; n <= nMax; n++) {
        for (let r = 0; r <= n; r++) {
            if (factorial(n) / (factorial(r) * factorial(n - r)) >= limit)
                result++;
        }
    }

    return result;
}
```
