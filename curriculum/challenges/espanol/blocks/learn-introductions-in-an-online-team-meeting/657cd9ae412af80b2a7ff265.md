# --description--

`Any` se usa para indicar que no importa cuál, no importa cuándo o no importa qué. Por ejemplo, `You can ask me any question` significa que puedes hacer todo tipo de preguntas.

`Even` se usa para enfatizar algo inesperado o para incluir algo adicional.  This word highlights something extra or beyond what is usual. Por ejemplo, `She can speak many languages, even Tupi` significa que es sorprendente o notable que ella también pueda hablar Tupi, además de otros idiomas.
