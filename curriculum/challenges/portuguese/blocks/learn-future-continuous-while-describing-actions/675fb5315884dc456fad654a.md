# --explanation--

`Every month` significa que algo acontece uma vez por mês, regularmente.Por exemplo:

`We back up our system data every month to prevent data loss.` - Esta frase mostra que fazer backup dos dados do sistema é uma tarefa regular que acontece mensalmente para garantir a segurança dos dados.

`Every two weeks` significa que algo acontece uma vez a cada duas semanas ou duas vezes por mês.Por exemplo:

`The IT team checks for security updates every two weeks to keep systems up to date.` - Esta frase destaca a importância de verificar atualizações duas vezes por mês para garantir que os sistemas permaneçam seguros e funcionando bem.

`Vigilant` significa estar atento e cuidadoso para evitar riscos ou problemas.Por exemplo:

`Employees must stay vigilant when opening emails to avoid phishing scams.` - Esta frase enfatiza a necessidade de os funcionários prestarem atenção e serem cautelosos ao lidar com e-mails para evitar cair em golpes ou ataques.
