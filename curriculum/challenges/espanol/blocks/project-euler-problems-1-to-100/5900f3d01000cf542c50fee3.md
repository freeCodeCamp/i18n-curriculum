# --description--

Si una caja contiene veintiún discos de colores, compuestos por quince discos azules y seis discos rojos, si dos discos son tomados al azar, podemos ver la probabilidad de tomar dos discos azules.

$${P(BB)} = \frac{15}{21}×\frac{14}{20} = \frac{1}{2}$$

La siguiente estructura de este tipo, para la cual hay exactamente un 50% de probabilidad de tomar dos discos azules al azar, es una caja que contiene ochenta y cinco discos azules y treinta y cinco discos rojos.

Al encontrar la primera estructura que contenga más de `limit` discos en total, determina la cantidad de discos azules que tendría la caja.

```js
function arrangedProbability(limit) {
  // Based on https://www.mathblog.dk/project-euler-100-blue-discs-two-blue/
  let blue = 15;
  let discs = 21;

  while (discs < limit) {
    const nextBlue = 3 * blue + 2 * discs - 2;
    const nextDiscs = 4 * blue + 3 * discs - 3;

    blue = nextBlue;
    discs = nextDiscs;
  }
  return blue;
}
```
