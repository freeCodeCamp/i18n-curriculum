# --interactive--

CSSの`z-index`プロパティは、ページ上で重なり合う位置指定要素の垂直方向の重なり順を制御するために使われます。複数の要素が重なって積み重なっている場合、`z-index`の値によってどの要素が最前面に表示されるかが決まります。`z-index`の値が高いほど要素は閲覧者に近くなり、値が低いほど要素はスタックの奥に配置されます。

ただし、`z-index`は位置指定された要素にのみ機能します。つまり、要素は`static`以外のposition値を持っている必要があり、例えば`relative`、`absolute`、または`fixed`などです。デフォルトの`z-index`値は`auto`で、これは要素をデフォルトの重なり順に配置します。3つのボックスを使った例を見てみましょう。

```html
<div class="container">
  <div class="box1">Box 1</div>
  <div class="box2">Box 2</div>
  <div class="box3">Box 3</div>
</div>
```

次に、ボックスが互いに重なるように配置するためのスタイルを適用します。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box1">Box 1</div>
  <div class="box2">Box 2</div>
  <div class="box3">Box 3</div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

.box1 {
  position: absolute;
  z-index: 1;
  background: lightcoral;
  top: 20px;
  left: 20px;
  width: 100px;
  height: 100px;
}

.box2 {
  position: absolute;
  z-index: 3;
  background: gold;
  top: 40px;
  left: 40px;
  width: 100px;
  height: 100px;
}

.box3 {
  position: absolute;
  z-index: 2;
  background: lightgreen;
  top: 60px;
  left: 60px;
  width: 100px;
  height: 100px;
}
```

:::

コンテナには`relative`のpositionが設定され、内部にネストされたすべてのボックスは`absolute`のpositionが設定されます。各ボックスは異なる`z-index`の値を持っており、その結果ボックスが重なり合ってレイヤー化されます。

`z-index`はウェブページ上でレイヤーを作る方法と考えることができ、`z-index`の値が高い要素は値が低い要素の上に配置されます。これはモーダル、ポップアップ、ツールチップなどの複雑なレイアウトで重なり合う要素の挙動を制御するのに特に役立ちます。
