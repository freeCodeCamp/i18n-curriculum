# --explanation--

`Have to` si usa per esprimere obbligo ed è sempre seguito dalla forma base del verbo (`set up`). Per esempio:

- `You have to update the software to use the new features.` - È necessario aggiornare il software per consultare le nuove funzionalità.

- `They have to set up the meeting room for the presentation.` - Devono preparare la sala riunioni per la presentazione.
