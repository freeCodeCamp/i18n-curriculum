# --description--

使用 freeCodeCamp 的自定义输入，你可以直接在任务中输入拼音和汉字。

如果你在你的计算机上使用中文键盘，你将能够输入拼音并从建议中选择汉字。以下是设置方法。

## 在 Windows 11 上设置

1. 从开始菜单打开 **设置**。

2. 进入 **时间和语言**，然后点击 **语言和区域**。

3. 转到 **首选语言** 并选择 **添加语言**。

4. 查找 **Chinese** 并选择 **Chinese (Simplified)**。

5. 点击 **Next** ，然后按照步骤操作。

语言安装完成后：  

- 在你的**首选语言**列表中查找。

- 点击省略号以显示更多选项并选择**语言选项**。

- 进入 **Keyboards**，点击 **Add a keyboard** 并选择中文（简体）版型。

在键盘之间切换：  

- 按 **Windows 密钥 + 空格键**。

- 或者点击任务栏上的语言图标（通常标记为 **ENG**）并选择中文。

## 在 Mac 上设置

**注意：** 根据你运行的操作系统，步骤可能有所不同。

1. 打开 **Apple 菜单** 并选择 **系统偏好设置**。

2. 转到 **Keyboard**。

3. 点击 **Input Sources** 标签（页）。

4. 选择 **plus (+)** 按钮，点击 **Chinese, Simplified** 并从列表中选择 **Pinyin - Simplified**。

在键盘之间切换：

- 按 **Control + Spacebar**。

- 或者点击菜单栏上的输入图标并选择拼音方法。

现在你已全部准备就绪。
