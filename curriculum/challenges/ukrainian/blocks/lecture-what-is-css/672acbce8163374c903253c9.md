# --description--

Елемент meta viewport — це ключовий компонент у адаптивному вебдизайні.

Це спеціальний HTML meta елемент, який дає браузеру інструкції про те, як керувати розмірами сторінки та масштабуванням на різних пристроях, особливо на мобільних телефонах і планшетах.

Давайте розглянемо базовий синтаксис елемента meta viewport:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

Цей елемент зазвичай розміщується в розділі `head` вашого HTML-документа. Але що означає кожна частина цього елемента?

Частина `width=device-width` каже браузеру встановити ширину сторінки відповідно до ширини екрану пристрою. Це необхідно для створення адаптивних макетів, які підлаштовуються під різні розміри екранів.

`initial-scale=1.0` встановлює початковий рівень масштабування при першому завантаженні сторінки. Значення 1.0 означає, що сторінка відображається з початковим масштабом `100%`, без будь-якого збільшення чи зменшення.

Використовуючи елемент meta viewport, ви гарантуєте, що ваші вебсторінки відображаються правильно на мобільних пристроях.

Без нього мобільні браузери зазвичай відтворюють сторінку з шириною екрану для десктопу, а потім масштабують її, що може призвести до поганого користувацького досвіду через дуже дрібний і важкочитаний текст.

Елемент meta viewport також дозволяє контролювати, чи можуть користувачі збільшувати або зменшувати масштаб ваших вебсторінок.

Хоча можна вимкнути масштабування за допомогою атрибута `user-scalable=no`, загалом рекомендується уникати цього з міркувань доступності.

Багато користувачів покладаються на можливість масштабування для кращої читабельності, особливо ті, хто має порушення зору.

Ось приклад того, чого не варто робити:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0, user-scalable=no">
```

Натомість краще створювати ваш сайт адаптивним і читабельним на різних рівнях масштабування, щоб усі користувачі могли комфортно отримувати доступ до вашого вмісту.

Елемент meta viewport відіграє важливу роль у створенні мобільно-дружніх вебсайтів.

Він забезпечує правильне відтворення ваших ретельно продуманих адаптивних дизайнів на різних пристроях, покращуючи користувацький досвід для всіх відвідувачів вашого сайту.
