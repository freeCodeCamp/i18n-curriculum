# --description--

`Software` bezeichnet die Programme oder Anwendungen, die auf einem Computer oder Gerät ausgeführt werden. Es unterscheidet sich von `hardware`, das die physischen Teile eines Computers beschreibt. Zum Beispiel:

`She works in software development.` – Das bedeutet, sie erstellt oder entwirft Computerprogramme.

Ein `software update team` ist eine Gruppe von Personen, die dafür verantwortlich sind, bestehende Software zu verbessern, indem sie Probleme (Bugs) beheben, neue Funktionen hinzufügen oder die Sicherheit erhöhen.
