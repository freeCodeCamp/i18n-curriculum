# --description--

في الدروس السابقة تعلمت كيف تستخدم حلقة `for`، التي تُستخدم لتكرار كتلة شفرة عددًا محددًا من المرات. إليك مثالًا على استخدام حلقة `for` لطباعة كل لغة من قائمة `languages` إلى وحدة التحكم:

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for language in languages:
    print(language)
```

لكن ماذا لو أردت تتبع الفهرس لكل عنصر؟ أحد الخيارات هو إنشاء متغير `index` وزيادته بمقدار `1` في كل تكرار للحلقة، مثل هذا:

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

index = 0

for language in languages:
    print(f'Index {index} and language {language}')
    index += 1
```

بينما هذه الطريقة تعمل، هناك طريقة أسهل باستخدام دالة `enumerate()`. دالة `enumerate()` تتعقب الفهرس لقابل للتكرار وتُرجع كائن enumerate.

إذا مررنا قائمة `languages` إلى دالة `enumerate()` وحولنا القيمة المرجعة إلى قائمة باستخدام دالة `list()`، فستبدو هكذا:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

list(enumerate(languages))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

كل مدخل في كائن enumerate (الذي أصبح الآن قائمة) هو صف مرتب يحتوي على عد، يليه قيمة من القابل للتكرار الذي مررناه إلى دالة `enumerate()`.

الآن، دعنا نعيد صياغة المثال السابق لاستخدام دالة `enumerate()`:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')
```

نُفكك العد والقيمة لكل صف مرتب في كائن enumerate إلى متغيرين اسمهما `index` و `language` على التوالي. وأخيرًا، يُستخدم كلا المتغيرين في سلسلة نصية f تُطبع إلى وحدة التحكم في كل تكرار للحلقة.

```md
Index 0 and language Spanish
Index 1 and language English
Index 2 and language Russian
Index 3 and language Chinese
```

هذا يلغي الحاجة إلى إنشاء وتحديث متغير `index` يدويًا.

تقبل دالة `enumerate()` أيضًا معلمة اختيارية اسمها `start` تحدد القيمة الابتدائية للعد. إذا تم حذف هذه المعلمة، يبدأ العد من `0`. إليك مثالًا على استخدام المعلمة الاختيارية `start`:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages, 1):
    print(f'Index {index} and language {language}')
```

وهذا ما سيبدو عليه الناتج في وحدة التحكم:

```md
Index 1 and language Spanish
Index 2 and language English
Index 3 and language Russian
Index 4 and language Chinese
```

حتى الآن كنا نكرر فقط على قائمة واحدة. لكن ماذا لو احتجت إلى التكرار على عدة قابلات للتكرار بالتوازي؟ يمكنك استخدام دالة `zip()` لذلك، التي تجمع القوائم في أزواج من العناصر وتُرجع مكررًا من صفوف مرتبة.

إذا مررنا قائمة من `developers` و `ids` إلى دالة `zip()` وحولنا القيمة المرجعة إلى قائمة باستخدام دالة `list()`، فستبدو هكذا:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

list(zip(developers, ids))
# [('Naomi', 1), ('Dario', 2), ('Jessica', 3), ('Tom', 4)]
```

وهنا مثال على استخدام دالة `zip()` مع حلقة `for` للتكرار على `developers` و `ids`:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, dev_id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {dev_id}')
```

في هذا المثال، تقوم الدالة `zip()` بدمج القائمتين في أزواج من العناصر وتُرجع مُكرّرًا من الصفوف المرتبة. ثم يقوم حلقة `for` بفك كل صف مرتب إلى المتغيرين `name` و `dev_id`. أخيرًا، في كل أمر طباعة، نطبع كل من `name` و `dev_id` من قائمتي `ids` و `developers` على التوالي. هذا هو شكل النتيجة في وحدة التحكم:

```md
Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
```

دالتي `enumerate()` و `zip()` قويتان جدًا، وعند دمجهما مع الحلقات، يمكن أن تجعل شفرتك أكثر إيجازًا.
