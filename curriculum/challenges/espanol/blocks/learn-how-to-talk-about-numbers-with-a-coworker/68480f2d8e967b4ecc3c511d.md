# --explanation--

`Intuitive` significa que algo es fácil de entender de forma natural, incluso para usuarios primerizos. Por ejemplo:

`The app's intuitive layout made it easy to find everything.` — La app fue diseñada de manera simple y clara, para que los usuarios pudieran encontrar rápidamente lo que necesitaban.

`User-friendly` significa que el diseño es fácil, agradable y eficiente para la persona que lo usa. Por ejemplo:

`User-friendly tools reduce the need for training.` — Las funciones fáciles de usar significan que los usuarios no necesitan mucha ayuda o instrucciones para comenzar a usarlas.
