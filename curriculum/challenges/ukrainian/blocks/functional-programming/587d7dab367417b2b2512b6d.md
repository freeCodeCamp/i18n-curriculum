# --description--

Останні кілька завдань охоплювали низку корисних методів масивів і рядків, які відповідають принципам функціонального програмування. Ми також дізналися про `reduce`, який є потужним методом, що використовується для спрощення задач. Від обчислення середніх значень до сортування — будь-яку операцію з масивом можна виконати, застосувавши його. Нагадаємо, що `map` і `filter` є особливими випадками `reduce`.

Давайте поєднаємо те, що ми вивчили, щоб розв’язати практичну задачу.

Багато сайтів управління контентом (CMS) додають заголовки публікацій до частини URL для зручного збереження посилань. Наприклад, якщо ви пишете публікацію на Medium з назвою `Stop Using Reduce`, ймовірно, URL міститиме деяку форму рядка заголовка (`.../stop-using-reduce`). Ви, можливо, вже помічали це на сайті freeCodeCamp.

# --instructions--

Заповніть функцію `urlSlug`, щоб вона перетворювала рядок `title` і повертала версію через дефіс для URL. Ви можете використовувати будь-які методи, розглянуті в цьому розділі, але не використовуйте `replace`. Ось вимоги:

Вхідні дані — рядок із пробілами та словами з великої літери

Вихідні дані — рядок, у якому пробіли між словами замінені на дефіси (`-`)

Вихідні дані мають бути всіми літерами нижнього регістру

Вихідні дані не повинні містити пробілів
