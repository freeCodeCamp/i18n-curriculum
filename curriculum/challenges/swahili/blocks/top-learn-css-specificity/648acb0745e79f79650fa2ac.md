# --description--

Kipengele cha mwisho, mwisho wa mstari, mkatishaji wa mkatishaji. Tuseme kwamba baada ya kuzingatia kila kipengele kingine, bado kuna sheria nyingi zinazopingana zinazolenga kipengele kimoja. Mfuatano huamua vipi ni sheria gani itumike?

Kwa urahisi kabisa, kweli. Sheria ambayo ilielezwa mwisho ndiyo mshindi.

```css
/* styles.css */

.alert {
  color: red;
}

.warning {
  color: yellow;
}
```

Kwa kipengele chenye darasa la vitu la `alert` na `warning`, mfuatano utapitia kila kipengele kingine, ikiwa ni pamoja na urithi (hakuna hapa) na uhalisia maalum (hakuna sheria inayozidi nyingine kwa uhalisia maalum). Kwa kuwa sheria ya `.warning` ilielezwa mwisho, na hakuna kipengele kingine kilichoweza kuamua ni sheria gani itumike, ndiyo itakayotekelezwa kwa kipengele hicho.
