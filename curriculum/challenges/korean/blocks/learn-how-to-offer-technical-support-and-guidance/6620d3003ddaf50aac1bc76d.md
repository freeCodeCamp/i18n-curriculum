# --description--

프로그래밍에서 `statement`는 특정 작업을 수행하는 코드 한 줄입니다. 예를 들어, `print("Hello World")`는 텍스트를 출력하는 문장입니다.

`condition`는 코드에서 어떤 것이 참인지 거짓인지 확인하는 테스트입니다. 예를 들어, `if x > 5`는 `x`가 5보다 큰지 확인합니다.

`conditional`는 일반적으로 `condition` 또는 `conditions`에 따라 달라지는 것을 가리키는 형용사입니다. 프로그래밍에서는 논리 `conditions`를 포함하는 모든 표현식이나 기능을 설명하는 데 사용됩니다. 명시적인 "if-then" 구조에만 국한되지 않고, `conditions`에 따라 동작을 변경하는 모든 설정, 반복문이나 심지어 삼항 연산자도 포함할 수 있습니다.

`conditional statement`는 특정 `condition`가 참일 때만 실행되는 문장입니다. 예를 들어, `if x > 5: print("Large")`는 `conditional statement`입니다. `Large`가 5보다 클 때만 `x`를 출력합니다.
