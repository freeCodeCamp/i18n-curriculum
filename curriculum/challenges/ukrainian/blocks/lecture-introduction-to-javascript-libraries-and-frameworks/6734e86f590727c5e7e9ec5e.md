# --description--

React — одна з найпопулярніших бібліотек JavaScript для створення користувацьких інтерфейсів і веб-застосунків. Спочатку розроблена та підтримувана Facebook, React здобула велику популярність у веб-розробці завдяки своїй ефективності, гнучкості та функціям.

Основна концепція React — створення багаторазових UI-компонентів. Ці компоненти, такі як кнопки, картки та аватарки, можна легко повторно використовувати по всьому застосунку. Ви можете вкладати ці компоненти один в одного, щоб створювати більш складні, динамічні інтерфейси.

Одна з ключових переваг React полягає в тому, що ці кастомні компоненти можуть оновлюватися та відтворюватися незалежно один від одного при зміні даних.

На відміну від традиційного JavaScript, який вимагає прямої маніпуляції DOM (Document Object Model), React використовує віртуальний DOM, що покращує продуктивність і ефективність. Ви дізнаєтеся більше про віртуальний DOM і як він працює в наступних уроках.

Окрім багаторазових компонентів, React також надає потужний спосіб керування станом вашого застосунку. Стан — це дані, які визначають, як компонент відтворюється і поводиться.

З React ви легко можете відстежувати та оновлювати стан компонентів, забезпечуючи, щоб UI відображав найактуальніші дані. Ви дізнаєтеся більше про роботу зі станом у майбутніх уроках.

Хоча в екосистемі JavaScript існує багато бібліотек, freeCodeCamp зосередиться переважно на навчанні React через його широке використання та попит у галузі.

У наступних кількох уроках ми глибше розглянемо створення кастомних компонентів, керування станом і не тільки.
