# --description--

Necesita confirmar que el usuario ha leído los términos y condiciones.

Agregue un elemento `label`. Dentro del elemento `label` recién creado, agregue un elemento `input` y establezca el atributo `type` en `checkbox`. Haga que este elemento `input` sea `required` para que los usuarios no puedan registrarse sin aceptar los términos y condiciones.

Agregue un atributo `id` y `for` con el valor `terms-and-conditions` a los elementos para mejorar la accesibilidad.
