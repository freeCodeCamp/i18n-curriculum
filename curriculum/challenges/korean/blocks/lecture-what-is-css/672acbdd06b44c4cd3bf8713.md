# --interactive--

HTML과 CSS 작업을 시작하면 CSS를 작성하기 전에도 웹 페이지에 일부 스타일이 적용된 것을 알게 됩니다. 이러한 스타일을 "기본 브라우저 스타일" 또는 "사용자 에이전트 스타일"이라고 합니다.

이 기본 브라우저 스타일은 모든 브라우저에서 HTML 요소가 읽기 쉽게 표시되도록 기본적인 서식을 제공합니다.

하지만 이러한 스타일은 브라우저마다 약간씩 다를 수 있으므로, 웹사이트의 일관된 디자인을 위해 이를 이해하는 것이 중요합니다.

이제 다양한 HTML 요소에 기본적으로 적용되는 일반적인 브라우저 스타일을 살펴보겠습니다.

제목은 가장 자주 사용되는 HTML 요소 중 하나이며, 기본적으로 크기와 굵기가 다르게 스타일이 적용됩니다.

예를 들어, `h1` (가장 높은 수준의 제목)은 보통 굵게 표시되고 `h2`, `h3` 등 하위 제목보다 폰트 크기가 큽니다.

이 제목 태그들은 콘텐츠의 계층 구조와 구성을 정의하는 데 도움을 줍니다.

다음 예제는 `h1`부터 `h6`까지 여섯 개의 제목 요소를 보여줍니다.

:::interactive_editor

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```

:::

또 다른 기본 스타일이 적용되는 요소는 `hr` 요소로, 콘텐츠 섹션을 시각적으로 구분하는 데 자주 사용되는 수평선을 만듭니다. 브라우저는 일반적으로 이 요소에 간단한 선 스타일을 적용합니다.

더 잘 이해하려면 다음 코드 예제를 살펴보세요:

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<hr>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

수평선은 얇은 선으로 나타나며, 텍스트 위아래에 간격이 있어 콘텐츠 섹션을 구분합니다.

다음으로 `blockquote` 요소를 살펴보겠습니다.

블록인용문은 다른 출처에서 인용한 텍스트 부분을 나타내는 데 사용됩니다. 브라우저는 보통 들여쓰기를 추가하고 때로는 텍스트를 이탤릭체로 표시합니다.

들여쓰기는 블록인용문을 나머지 텍스트와 구분해 이 내용이 다른 출처에서 인용된 것임을 명확히 합니다.

다음 코드 예제를 살펴보세요:

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<blockquote>I think, therefore I am. (Rene Descartes)</blockquote>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

위 예제에서 `blockquote` 요소는 기본 브라우저 스타일에 의해 텍스트에 들여쓰기가 적용됩니다.

또 다른 기본 스타일이 적용되는 요소는 앵커 요소(`<a>`)로, 기본적으로 파란색과 밑줄 스타일이 적용되어 링크임을 쉽게 알아볼 수 있습니다.

다음 HTML 예제를 살펴보세요:

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<a href="https://freecodecamp.org/">Visit the freeCodeCamp learn page</a>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

위 코드는 네 개의 문단 요소가 있고 그 중간에 앵커 요소가 있습니다. 앵커 요소는 파란색과 밑줄로 스타일이 적용되어 사용자가 freeCodeCamp 학습 페이지로 이동할 수 있음을 보여줍니다.

마지막으로 순서 없는 목록과 순서 있는 목록 요소를 살펴보겠습니다.

브라우저는 목록에 기본 서식을 추가하는데, 순서 없는 목록(`ul`)인지 순서 있는 목록(`ol`)인지에 따라 들여쓰기와 글머리표 또는 숫자가 적용됩니다.

예제를 살펴보겠습니다:

:::interactive_editor

```html
<p>Sample Paragraph</p>
<ol>
  <li>item 1</li>
  <li>item 2</li>
  <li>item 3</li>
</ol>
<ul>
  <li>item</li>
  <li>another item</li>
  <li>yet another item</li>
</ul>
<p>Ending Paragraph</p>
```

:::

위 코드 예제에서는 순서 없는 목록과 순서 있는 목록을 사용했습니다. 두 목록 모두 기본적으로 오른쪽으로 약간 들여쓰기가 적용됩니다.

이 모든 예제에서 브라우저가 기본적으로 HTML 요소에 간격, 다양한 폰트 크기와 굵기를 어떻게 적용하는지 확인했습니다.
