# --description--

継承とは、ある要素に適用された特定のCSSプロパティが、その要素の子孫要素にも明示的にルールを書かなくても引き継がれることを指します。タイポグラフィに関するプロパティ（`color`、`font-size`、`font-family`など）は通常継承されますが、その他のほとんどのプロパティは継承されません。

これに対する例外は、要素を直接ターゲットにした場合で、これは常に継承より優先されます。

```html
<!-- index.html -->

<div id="parent">
  <div class="child"></div>
</div>
```

```css
/* styles.css */

#parent {
  color: red;
}

.child {
  color: blue;
}
```

`parent`要素はIDによってより高い特異性を持っていますが、`child`要素はその宣言が直接ターゲットにしているため`color: blue`スタイルが適用されます。一方、親からの`color: red`は継承されるだけです。
