# --explanation--

`The problem is random.` – これは、問題がどのようになるか予測できず、予期せず変化することを意味します。

`More random` は、状況が以前よりも予測しにくくなっていることを意味します。より不規則で、次に何が起こるかを推測するのが難しくなっています。
