# --description--

$12x^2 + 7xy - 12y^2 = 625$ で定義される双曲線を $H$ とします。

次に、点 $X$ を (7, 1) と定義します。$X$ は $H$ 上にあることがわかります。

ここで、$H$ 上の点のシーケンス $\\{P_i : i ≥ 1\\}$ を次のように定義します。

- $P_1 = (13, \frac{61}{4})$。
- $P_2 = (\frac{-43}{6}, -4)$。
- $i > 2$ の場合、$P_i$ は $H$ 上で $P_{i - 1}$ と異なり、かつ直線 $P_iP_{i - 1}$ が直線 $P_{i - 2}X$ と平行になる唯一の点です。$P_i$ は一意に定まり、その座標は常に有理数であることが示せます。

<img alt="animation showing defining points P_1 to P_6" src="https://cdn.freecodecamp.org/curriculum/project-euler/sequence-of-points-on-a-hyperbola.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$P_3 = (\frac{-19}{2}, \frac{-229}{24})$、$P_4 = (\frac{1267}{144}, \frac{-37}{12})$、$P_7 = (\frac{17\\,194\\,218\\,091}{143\\,327\\,232}, \frac{274\\,748\\,766\\,781}{1\\,719\\,926\\,784})$ が与えられています。

$n = {11}^{14}$ のときの $P_n$ を次の形式で求めてください。$P_n = (\frac{a}{b}, \frac{c}{d})$ で、分数は既約分数かつ分母は正の数とします。このとき答えは $(a + b + c + d)\bmod 1\\,000\\,000\\,007$ です。

$n = 7$ の場合、答えは $806\\,236\\,837$ となります。
