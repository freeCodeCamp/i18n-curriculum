# --explanation--

这个句子使用了**被动语态**，它关注的是接收动作的对象，而不是执行动作的人。例如：  

`The navigation menu was appreciated.` – 这意味着人们喜欢它，但没有说明是谁喜欢的。  

在**主动语态**中，你会说 `The users appreciated the navigation menu.` —— 这清楚地显示了谁执行了该动作。

**被动语态**通常在执行动作的人未知或不重要时使用。另一个例子：

`The report was completed on time.` – 你知道报告已完成，但你不知道也不需要说明是谁完成的。
