# --interactive--

Tuwe tione jinsi unavyoweza kutengeneza nodi kwa kutumia `innerHTML` na `createElement()`.

Tutaanza na `innerHTML`.

`innerHTML` ni sifa ya vitu vya `Element` ambayo unaweza kutumia kuweka alama za uandishi za HTML zao. Kwa `innerHTML`, unaweza kuweka muundo wa HTML wa kipengele kilichopo kwa kutumia mfuatano wa herufi, ukitengeneza nodi zote zinazohitajika.

Hii ni mfano. Tuna `div` tupu na tutaongeza baadhi ya vipengele ndani yake kwa kutumia sifa ya `innerHTML`:

```html
<div id="container">
  <!-- Add new elements here -->
</div>
```

Kwanza, tunahitaji kuchagua kipengele kwa kutumia ID yake:

```js
const container = document.getElementById("container");
```

Kisha, tunaweka sifa ya `innerHTML` ya kontena kuwa mfuatano wa herufi. Mfuatano huu lazima uwe na alama zote za uandishi zinazohitajika kuwakilisha nodi na muundo unaotaka kuunda. Unaweza kufikiria kama kuandika HTML ndani ya mfuatano wa herufi.

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
container.innerHTML = "<ul><li>Cheese</li><li>Tomato</li></ul>";
```

:::

Tutawaweka `innerHTML` ya kipengele hiki kuwa orodha isiyopangwa ya viungo vya pizza (jibini na nyanya).

Unaweza pia kuandika mfuatano huo kwenye mistari mingi ikiwa inafanya alama za uandishi kuwa rahisi kuelewa. Jambo muhimu ni kwamba alama za uandishi zinawakilisha kwa usahihi muundo wa HTML unaotaka kuunda.

Baada ya kuendesha msimbo huu, utaona muundo wa HTML kama ifuatavyo ukikagua alama zako za uandishi.

```html
<div id="container">
  <ul>
    <li>Cheese</li>
    <li>Tomato</li>
  </ul>
</div>
```

Nodi mpya zilitengenezwa na kuongezwa kwa nguvu kwenye DOM baada ya mfuatano huo kusomwa.

`innerHTML` inaweza kusaidia sana katika hali fulani. Hata hivyo, ina hatari fulani za usalama unazopaswa kufahamu. Hupaswi kuitumia ikiwa huna udhibiti kamili juu ya mfuatano huo.

Kwa mfano, ikiwa mfuatano utaingizwa na mtumizi, hupaswi kutumia `innerHTML` kwa sababu mtumizi anaweza kuingiza maudhui hatari kwenye tovuti yako. Kwa sababu hii, kawaida inashauriwa kutumia `textContent` badala yake, ili kuingiza maandishi ya kawaida.

Njia nyingine ya kutengeneza nodi mpya ni kwa kutumia njia ya `createElement()`. Kwa njia hii mpya, unaweza kutengeneza kipengele kipya kwa kubainisha jina la lebo yake.

Kwa mfano, ikiwa unataka kutengeneza kipengele cha picha, utapita lebo ya `img` kama mfuatano wa herufi unapoita njia hii:

```js
document.createElement("img");
```

Na unaweza kuweka kitu hiki kipya kwenye kigezo:

```js
const img = document.createElement("img");
```

Njia ya `createElement()` inarudisha kitu kipya cha `HTMLElement` ikiwa hati ni `HTMLDocument`. Vinginevyo, inarudisha kitu cha `Element`.

Mara tu unapokuwa na kipengele hiki kipya tayari, unaweza kuiongeza kwenye DOM kama mtoto wa kipengele kingine kilichopo kwa kutumia njia ya `appendChild()`, au unaweza kuingiza mahali maalum kwa kutumia njia nyingine. Chagua ile inayokufaa zaidi.

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
const img = document.createElement("img");
img.src = "https://cdn.freecodecamp.org/curriculum/cat-photo-app/lasagna.jpg";
img.alt = "A slice of lasagna on a plate.";
container.appendChild(img);
```

:::

Kutengeneza nodi kwa kutumia `innerHTML` na `createElement()` kunakuwezesha kudhibiti kwa nguvu muundo na maudhui ya tovuti zako. Kwa kuunganisha mbinu hizi, unaweza kujenga programu za mtandao zenye mwingiliano.
