# --interactive--

`div` 要素は、他の要素をグループ化するためのコンテナとして使われます。

こちらは `div` 要素の例です。`div` 要素の中にもう一つ段落要素を追加して、プレビューウィンドウの変化を確認してください。プレビューを見るには、インタラクティブエディターを有効にする必要があります。

:::interactive_editor

```html
<div>
  <p>Example paragraph element.</p>
</div>
```

:::

主に `div` 要素は、CSSスタイルのセットを共有するHTML要素をグループ化したいときに使います。CSSについては今後のレッスンやワークショップで詳しく学びます。

`div` 要素は実際のコードベースでよく使われますが、使いすぎには注意してください。別の要素の方が適切な場合もあります。

例えば、コンテンツをセクションに分けたい場合は、`section` 要素の方が `div` 要素より適切です。

最初の `section` 要素の下にもう一つ `section` 要素を追加してください。次に `section` 要素の中に `h2` と `p` 要素を入れてください。好きなテキストを使って構いません。プレビューウィンドウの変化を確認できます。例とやり取りするにはインタラクティブエディターを有効にしてください。

:::interactive_editor

```html
<section>
  <h2>Mammals</h2>
  <p>
    Mammals are warm-blooded animals with fur or hair. Most give birth to live
    young.
  </p>
  <ul>
    <li>Lion</li>
    <li>Elephant</li>
    <li>Dolphin</li>
  </ul>
</section>
```

:::

`section` 要素は意味論的な意味を持ちますが、`div` 要素は意味論的な意味を持ちません。意味論とは言語における単語やフレーズの意味のことです。HTMLという言語でも、要素にはそれぞれ意味論的な意味があります。つまり、`section` 要素を使うと、ブラウザーはその意味論的な意味を認識し、デスクトップやモバイルなどあらゆる環境でこれをセクションとして扱います。
