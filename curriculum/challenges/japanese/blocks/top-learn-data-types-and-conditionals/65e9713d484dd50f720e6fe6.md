# --description--

どのような作業をしているかによっては、数字よりもテキストの断片を扱うことが多くなる場合があります。**文字列**はテキストの一部であり、言語の基本的な構成要素です。

HTMLはテキストに構造と意味を与え、CSSは正確にスタイルを指定できるようにし、JavaScriptは文字列を操作するための多くの機能を提供します。これには、カスタムの歓迎メッセージやプロンプトの作成、必要に応じた適切なテキストラベルの表示、用語の希望する順序への並べ替えなどが含まれます。

文字列はJavaScriptの基本的なデータ型です。テキストを表すために使われ、シングルクォート、二重引用符、またはバックティックで囲まれます。

```javascript
let greeting = "Hello World!";

let greeting2 = 'I am learning JavaScript!';

let greeting3 = `with The Odin Project!`;
```

シングルクォートで宣言された文字列と二重引用符で宣言された文字列は同じで、どちらを使うかは個人の好みによります。ただし、一つのスタイルを選んでコード内で一貫して使うことが良い習慣です。
