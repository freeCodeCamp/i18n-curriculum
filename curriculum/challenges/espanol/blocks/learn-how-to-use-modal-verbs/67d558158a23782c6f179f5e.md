# --explanation--

Jessica le dice a Mark que ejecute `regular security audits` para garantizar la seguridad.

Una `security audit` es un proceso de revisar un sistema en busca de vulnerabilidades. Por ejemplo:

`After the security audit, the team made improvements to the system.` - Se hicieron cambios para mejorar la seguridad después de la revisión.

`Regular` significa que ocurre de manera constante a lo largo del tiempo. Por ejemplo:

`Regular testing helps find problems before they get worse.` - Probar cosas con frecuencia ayuda a detectar problemas temprano.
