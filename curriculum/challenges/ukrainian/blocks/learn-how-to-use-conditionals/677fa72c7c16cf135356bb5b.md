# --explanation--

Марк каже, `If the application has memory leaks, it can lead to crashes`.

Ключовий термін тут — `memory leaks`. Витоки пам’яті виникають, коли програма виділяє пам’ять для тимчасових завдань, але не звільняє її після завершення цих завдань. Наприклад:

- `If the application uses too much memory, it could crash.` — це свідчить, що проблема, яка призводить до збоїв, пов’язана з використанням пам’яті.

- `Memory leaks can cause performance issues and crashes.` — це показує, як витоки пам’яті впливають на виникнення збоїв.
