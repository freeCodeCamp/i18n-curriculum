# --description--

في الدروس السابقة، تعرّفت أولًا على خاصية `length`، التي تُرجع عدد العناصر في المصفوفة. إليك تذكيرًا سريعًا بكيفية عملها:

```js
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits.length); // 3
```

من الممكن وجود مصفوفات تحتوي على خانات فارغة. تُعرف الخانات الفارغة بأنها خانات لا تحتوي على أي شيء بداخلها. هذا يختلف عن المصفوفة التي تحتوي على القيمة `undefined`. تُعرف هذه الأنواع من المصفوفات بالمصفوفات المتفرقة. إليك مثالًا:

```js
const sparseArray = [1, , , 4];
console.log(sparseArray.length); // 4
```

في هذه الحالة، رغم أن لدينا عنصرين معرفين فقط، `1` و `4`، فإن الطول هو `4` لأن أعلى فهرس (`3`) زائد `1` يعطينا طولًا يساوي `4`.

الآن دعنا نناقش كيفية إنشاء مصفوفة فارغة بطول ثابت. هناك عدة طرق للقيام بذلك في JavaScript، لكن إحدى الطرق الشائعة هي استخدام دالة البناء `Array()` مع معلمة رقمية. يمكن استخدام دالة البناء `Array()` مع الكلمة المفتاحية `new` لإنشاء مصفوفة جديدة. إليك مثالًا:

```js
const emptyArray = new Array(5);
console.log(emptyArray.length); // 5
console.log(emptyArray); // [ , , , , ]
```

في هذا المثال، ننشئ مصفوفة جديدة باستخدام `Array(5)`. هذا ينشئ مصفوفة متفرقة بطول `5` حيث كل الخانات فارغة.

طريقة أخرى لإنشاء مصفوفة فارغة بطول ثابت هي استخدام الطريقة `Array.from()` مع معلمة الطول. على عكس `new Array(n)`, تنشئ هذه الطريقة مصفوفة بالطول المحدد حيث توجد كل العناصر وتكون قيمتها `undefined`:

```js
const fixedLengthArray = Array.from({ length: 5 });
console.log(fixedLengthArray.length); // 5
console.log(fixedLengthArray); // [undefined, undefined, undefined, undefined, undefined]
```

إذا أردت إنشاء مصفوفة بطول محدد وملؤها بقيمة افتراضية، يمكنك استخدام الطريقة `Array.fill()`:

```js
const filledArray = new Array(3).fill(0);
console.log(filledArray); // [0, 0, 0]
```

هذا ينشئ مصفوفة بطول ثلاثة ويملأ كل العناصر بالقيمة `0`. ملاحظة: عند الملء بالكائنات، تشير كل الخانات إلى نفس الكائن؛ إذا كنت تحتاج نسخًا مستقلة، استخدم استدعاء راجع أو `Array.from()` بدلاً من ذلك.

فهم كيفية الحصول على طول المصفوفة وإنشاء مصفوفات بطول ثابت مهم للعديد من مهام البرمجة، خاصة عندما تحتاج إلى تهيئة مصفوفات لخوارزميات أو هياكل بيانات محددة.
