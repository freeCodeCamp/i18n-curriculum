## --text--

사용자가 문자열 `"random"`을 입력하면 다음 코드의 결과는 무엇일까요?

```python
try:
    number = int(input("Enter a number: "))
    print(number)
except:
    print("Invalid Input")
```

## --answers--

콘솔에 아무것도 출력되지 않습니다.

---

콘솔에 문자열 `"random"`이 출력됩니다.

---

콘솔에 문자열 `"Invalid Input"`이 출력됩니다.

---

프로그램이 충돌합니다.
