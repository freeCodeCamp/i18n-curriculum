# --description--

Acha $a_k$, $b_k$, na $c_k$ wakilishe suluhisho tatu (nambari halisi au ngumu) za usemi wa kihesabu $\frac{1}{x} = {\left(\frac{k}{x} \right)}^2 (k + x^2) - kx$.

Kwa mfano, kwa $k = 5$, tunaona kuwa $\\{a_5, b_5, c_5\\}$ ni takriban $\\{5.727244, -0.363622 + 2.057397i, -0.363622 - 2.057397i\\}$.

Acha $S(n) = \displaystyle\sum_{p = 1}^n \sum_{k = 1}^n {(a_k + b_k)}^p {(b_k + c_k)}^p {(c_k + a_k)}^p$ kwa nambari kamili zote $p$, $k$ zenye $1 ≤ p, k ≤ n$.

Kivutio ni kwamba $S(n)$ daima ni nambari kamili. Kwa mfano, $S(4) = 51\\,160$.

Tafuta $S({10}^6) \text{ modulo } 1\\,000\\,000\\,007$.
