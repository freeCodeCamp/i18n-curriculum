# --description--

長さ $n$ の整数シーケンス $\\{a_1, a_2, \ldots, a_n\\}$ が次の条件を満たすとします：

- $a_1 = 6$
- すべての $1 ≤ i &lt; n$ に対して：$φ(a_i) &lt; φ(a_{i + 1}) &lt; a_i &lt; a_{i + 1}$

ここで $φ$ はオイラーのトーシェント関数を表します。

$S(N)$ を $a_n ≤ N$ となるそのようなシーケンスの個数とします。

例えば、$S(10) = 4$ です：{6}、{6, 8}、{6, 8, 9}、および {6, 10}。

$S(100) = 482\\,073\\,668$ および $S(10\\,000)\bmod {10}^8 = 73\\,808\\,307$ であることが確認されています。

$S(20\\,000\\,000)\bmod {10}^8$ を求めてください。
