# --description--

以下は、最も単純な配列データ構造の実装例です。これは<dfn>一次元配列</dfn>と呼ばれ、1つのレベルしか持たず、他の配列がネストされていないことを意味します。ここには<dfn>ブール値</dfn>、<dfn>文字列</dfn>、<dfn>数値</dfn>など、他の有効なJavaScriptデータ型が含まれていることに注目してください。

```js
let simpleArray = ['one', 2, 'three', true, false, undefined, null];
console.log(simpleArray.length);
```

`console.log`の呼び出しは`7`を表示します。

すべての配列にはlengthプロパティがあり、上記のように`Array.length`の構文で簡単にアクセスできます。より複雑な配列の実装例は以下の通りです。これは<dfn>多次元配列</dfn>と呼ばれ、他の配列を含む配列です。この配列にはJavaScriptの<dfn>オブジェクト</dfn>も含まれていることに注目してください。次のセクションで詳しく見ていきますが、今は配列が複雑なオブジェクトも保存できることだけ覚えておいてください。

```js
let complexArray = [
  [
    {
      one: 1,
      two: 2
    },
    {
      three: 3,
      four: 4
    }
  ],
  [
    {
      a: "a",
      b: "b"
    },
    {
      c: "c",
      d: "d"
    }
  ]
];
```

# --instructions--

`yourArray`という変数が定義されています。`yourArray`変数に、少なくとも5つの要素を持つ配列を代入して文を完成させてください。配列には少なくとも1つの<dfn>文字列</dfn>、1つの<dfn>数値</dfn>、1つの<dfn>ブール値</dfn>を含めてください。

# --hints--

`yourArray`は配列である必要があります。

```js
assert.strictEqual(Array.isArray(yourArray), true);
```

`yourArray`は少なくとも5つの要素を持つ必要があります。

```js
assert.isAtLeast(yourArray.length, 5);
```

`yourArray`は少なくとも1つの`boolean`を含む必要があります。

```js
assert(yourArray.filter((el) => typeof el === 'boolean').length >= 1);
```

`yourArray`は少なくとも1つの`number`を含む必要があります。

```js
assert(yourArray.filter((el) => typeof el === 'number').length >= 1);
```

`yourArray`は少なくとも1つの`string`を含む必要があります。

```js
assert(yourArray.filter((el) => typeof el === 'string').length >= 1);
```
