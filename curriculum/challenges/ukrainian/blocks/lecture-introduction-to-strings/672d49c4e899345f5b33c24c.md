# --interactive--

Попередні уроки познайомили вас із `console.log()`, але цей урок заглибиться у його призначення та використання.

У JavaScript `console.log()` — це простий, але потужний інструмент, який використовується для відображення повідомлень або виведення інформації у консоль браузера. Його здебільшого використовують розробники для налагодження та перевірки коду під час роботи над програмами.

Ви можете використовувати `console.log()`, щоб виводити текст або змінні у консоль і переконатися, що ваш код працює правильно.

Щоб використати `console.log()`, ви викликаєте метод із значенням або повідомленням, яке хочете вивести, всередині круглих дужок. Ось кілька прикладів:

:::interactive_editor

```js
console.log("Hello, world!");

let num = 5;
console.log(num); // 5
```

:::

Перший приклад виводить `Hello, world!` у консолі браузера, а другий приклад виводить значення `5`.

Ось ще один приклад роботи з `console.log()`:

:::interactive_editor

```js
let name = "Alice";
console.log("Hello, " + name + "!"); // Hello, Alice!
```

:::

Ви також можете передавати кілька значень у `console.log()`, розділених комами. Наприклад:

:::interactive_editor

```js
let name = "Alice";
let age = 25;
console.log("Name:", name, "Age:", age); // Name: Alice Age: 25
```

:::

Це корисно для одночасного виведення кількох фрагментів інформації.

Метод `console.log()` допомагає вам контролювати ваш код під час його виконання, що полегшує виявлення помилок і розуміння поведінки програми.
