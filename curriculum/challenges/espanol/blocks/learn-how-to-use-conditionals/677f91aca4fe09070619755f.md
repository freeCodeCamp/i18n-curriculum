El `Power cable` se refiere al cable que suministra electricidad a la computadora:

`Power` significa energía o electricidad, y `cable` es el alambre o cordón usado para transferirla. Por ejemplo:

- `The server shut down because the power cable was disconnected.` - El servidor se apagó cuando el cable de alimentación fue desconectado.
