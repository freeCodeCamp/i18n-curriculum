# --description--

`To roll back` bedeutet, zu einem früheren Zustand oder einer früheren Version von etwas zurückzukehren. Zum Beispiel:

`If the new app is not working, try rolling back to the old version.`

Ein `previous version` ist eine frühere Form von etwas, wie eine ältere Version einer Software. Verwenden wir es im gleichen Satz wie oben:

`If the new app is not working, try rolling back to the previous version.`
