# --description--

HTML e CSS si occupano principalmente di presentare informazioni. Non vengono usati per programmare la logica. JavaScript, che imparerai nella sezione successiva, serve a far fare cose alle pagine web. Con solo HTML e CSS puoi fare già molto, e avrai sicuramente bisogno di entrambi. Nel nostro curriculum, le seguenti lezioni si concentrano sul darti gli strumenti necessari per avere successo quando arriverai ai contenuti di JavaScript.
