# --description--

讓我們來了解 HTML 樣板。

你問什麼是 HTML boilerplate？它就像是為你的網頁準備好的範本。可以把它想像成房子的地基。boilerplate 包含每個 HTML 文件所需的基本結構和必要元素。它能幫你節省時間，並確保你的頁面正確設置。以下是一個範例：

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>freeCodeCamp</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
  </body>
</html>
```

讓我們拆解這個樣板的關鍵零件。首先，是 `DOCTYPE` 宣告：

```html
<!DOCTYPE html>
```

它告訴瀏覽器你正在使用哪個版本的 HTML。接著，是 `html` 頁籤：

```html
<!DOCTYPE html>
<html lang="en">
  <!--All other elements go inside here-->
</html>
```

這會包裹你所有的內容，並且可以指定你頁面的語言。在 `html` 頁籤內，你會找到兩個主要區段－`head` 和 `body`：

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <!--Important metadata goes here-->
  </head>
  <body>
    <!--Headings, paragraphs, images, etc. go inside here-->
  </body>
</html>
```

`head` 區段包含重要的幕後資訊：

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Document Title Goes Here</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

你網站的詮釋資料包含在 `meta` 元素中，裡面有關於字元編碼以及像 Twitter 這類網站應如何預覽你頁面聯結的詳細資訊。你網站的標題位於 `title` 元素中，決定瀏覽器分頁或視窗中顯示的文字。最後，你通常會在 `head` 區段使用 `link` 元素將頁面的外部樣式表聯結起來。

`body` 區段是你所有內容的所在：

```html
<body>
  <h1>I am a main title</h1>
  <p>Example paragraph text</p>
</body>
```

現在，為什麼樣板很重要？它確保你的頁面結構正確，並且能在不同瀏覽器中良好運作。使用樣板可以幫助你避免常見錯誤並遵循最佳實務。它是任何網頁專案的絕佳起點。記住，你可以自訂自己的樣板以符合你的需求。隨著經驗的累積，你可能會為你的樣板添加自己偏好的元素或 `meta` 標記。當你持續改進個人樣板時，你會發現它在開始新專案時能為你節省時間。

下次你開始新的 HTML 檔案時，請考慮使用樣板。它絕對會為你提供一個穩固的基礎來建置。
