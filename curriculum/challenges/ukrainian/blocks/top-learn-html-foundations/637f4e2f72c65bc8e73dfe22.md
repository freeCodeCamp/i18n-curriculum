# --description--

Елемент `<head>` — це місце, де ви розміщуєте важливу метаінформацію про свої вебсторінки та речі, необхідні для правильного відтворення вебсторінок у браузері. Всередині `<head>` не слід використовувати жоден елемент, який відображає вміст на вебсторінці.

## Метаелемент Charset
Ви завжди повинні мати тег `meta` для кодування `charset` вебсторінки у елементі `head`: `<meta charset="utf-8">`.

Встановлення кодування дуже важливе, оскільки це гарантує, що вебсторінка правильно відображатиме спеціальні символи та знаки з різних мов у браузері.

## Елемент title
Ще один елемент, який ви завжди повинні включати в заголовок HTML-документа, — це елемент `title`:

```html
<title>My First Webpage</title>
```

Елемент `title` використовується, щоб надати вебсторінкам зрозумілу людині назву, яка відображається на вкладці браузера вашої вебсторінки.

Якщо ви не додасте елемент `title`, назва вебсторінки за замовчуванням буде іменем файлу. У вашому випадку це буде `index.html`, що не дуже інформативно для користувачів; це ускладнить пошук вашої вебсторінки, якщо у користувача відкрито багато вкладок браузера.

Існує багато інших елементів, які можуть розміщуватися всередині `head` HTML-документа. Однак наразі важливо знати лише про два елементи, які ви тут розглянули. Ви познайомитеся з іншими елементами, що розміщуються в `head`, протягом решти навчальної програми.

Повертаючись до файлу `index.html`, давайте додамо елемент `head` з елементом `charset` `meta` і `title` всередині нього. Елемент head розміщується всередині елемента HTML і завжди має бути першим елементом після відкриваючого тегу `<html>`:


```html
<!DOCTYPE html>

<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Webpage</title>
  </head>
</html>
```

# --questions--
    
## --text--

Яке призначення елемента `head`?

## --answers--

Елемент `head` використовується для відображення всіх елементів, які показуються на вебсторінці.

---

Елемент `head` використовується для відображення важливої інформації про вашу вебсторінку і застосовується для правильного відтворення вебсторінок разом з елементами `meta`.

---

Елемент `head` використовується для відображення вмісту заголовка у верхній частині вебсторінки.
