# --description--

학교 산수 범위를 넘어서는 JavaScript 연산자의 기능을 살펴보겠습니다.

보통, 더하기 연산자 `+`는 숫자를 더합니다.

하지만, 이항 `+`이 문자열에 적용되면, 문자열을 병합(연결)합니다:

```js
let s = "my" + "string";
console.log(s); // mystring
```

피연산자 중 하나가 문자열이면, 다른 피연산자도 문자열로 변환된다는 점을 기억하세요.

예를 들어:

```js
console.log(1 + '2'); // '12'
console.log('1' + 2); // '12'
```

첫 번째 피연산자가 문자열이든 두 번째 피연산자가 문자열이든 상관없습니다.

더 복잡한 예는 다음과 같습니다:

```js
console.log(2 + 2 + '1') // "41" and not "221"
```

여기서 연산자는 차례로 작동합니다. 첫 번째 `+`는 두 숫자를 더해서 `4`를 반환하고, 다음 `+`는 문자열 `1`를 더하므로 `4 + '1' = '41'`와 같습니다.

```js
console.log('1' + 2 + 2) // "122" and not "14"
```

여기서는 첫 번째 `+`가 문자열과 숫자를 연결해서 `'12'`를 반환하고, 다음 `+`가 `2`를 더하므로 `'12' + 2 = '122'`와 같습니다.

이항 `+`는 문자열을 이렇게 지원하는 유일한 연산자입니다. 다른 산술 연산자는 숫자만 다루며 항상 피연산자를 숫자로 변환합니다.

뺄셈과 나눗셈 데모는 다음과 같습니다:

```js
console.log( 6 - '2' ); // 4, converts '2' to a number
console.log( '6' / '2' ); // 3, converts both operands to numbers
```
