# --explanation--

`We'd prefer` é uma forma educada de dizer o que você gostaria mais. Por exemplo:

`We'd prefer faster processors for the new computers.` – Isso demonstra um interesse maior por opções melhores.

`Since` é usado para explicar a razão por trás de uma decisão ou situação. Por exemplo:

`We upgraded the servers since the old ones were too slow.` – Isso mostra a causa para fazer a atualização.
