# --description--

```css
/* rule 1 */
#subsection .list {
  background-color: yellow;
  color: blue;
}

/* rule 2 */
#subsection .main .list {
  color: red;
}
```

在这个最终示例中，两个规则都使用了 ID 和类选择器，因此没有一个规则使用比另一个更具体的选择器。级联随后查看每种选择器类型的数量。两个规则都只有一个 ID 选择器，但规则 2 有更多的类选择器，所以规则 2 具有更高的特异性！

虽然 `color: red` 声明具有优先级，但由于没有冲突的声明，`background-color: yellow` 声明仍会被应用。
