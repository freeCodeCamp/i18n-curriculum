# --explanation--

La oración usa el `Second Conditional`, que está estructurado como:

`If` + `Past Simple` o `Past Continuous` tense, `might / would / could` + base verb. En este ejemplo:

`If they closed` es la condición (imaginando un apagado).

`It would affect` es el resultado (lo que pasaría como consecuencia).

Este tipo de condicional se usa para hablar de situaciones irreales o hipotéticas.
