# --description--

`introductory` 부분은 문서의 시작 부분으로 주요 주제를 소개합니다. 예: `The introductory chapter of the book outlines the key concepts.`

`paragraph` 는 특정 주제에 관한 글의 한 섹션입니다. 예를 들어, `Each paragraph in the blog post focuses on a different technology trend.`

`To cover` 는 특정 내용을 다루거나 포함한다는 뜻입니다. 예를 들어, 기술 안내서에서 `This section covers the steps to install software.`
