# --explanation--

`Make it a habit` означає робити щось регулярно, поки це не стане автоматичним. Наприклад:

- `Try to make it a habit to write clean code.` — ви повинні регулярно писати код, який легко читати.

- `She made it a habit to learn one new programming tip every week.` — вона регулярно навчається як частина своєї рутини.

Марк хоче тестувати свій код щоразу, щоб не забувати робити це в майбутньому.
