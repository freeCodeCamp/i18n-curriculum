# --explanation--

Phishing-Angriffe bringen Nutzer oft dazu, `malicious links` anzuklicken, was zu `unauthorized access` auf Konten oder Systeme führt.

`Unauthorized access` bedeutet, dass jemand ohne Erlaubnis in ein System eindringt. Zum Beispiel:

`The system blocked unauthorized access.` – Das System verhinderte, dass nicht genehmigte Benutzer zugreifen konnten.

`Malicious link` bezeichnet einen schädlichen Link, der Informationen stehlen oder ein System infizieren kann. Zum Beispiel:

`The email had a malicious link inside.` – Die E-Mail enthielt einen schlechten Link, der Sie täuschen sollte.
