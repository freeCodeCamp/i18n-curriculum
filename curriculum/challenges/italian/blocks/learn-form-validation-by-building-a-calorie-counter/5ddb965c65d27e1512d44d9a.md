# --description--

In questo progetto imparerai a creare un modulo contacalorie che permette agli utenti di inserire il loro budget calorico giornaliero e il conteggio delle calorie di vari pasti. Il modulo calcolerà e mostrerà se l'utente è in un deficit o surplus calorico.

Ti sono stati forniti codice boilerplate CSS e HTML. Tuttavia, devi costruire il tuo modulo contacalorie.

Sentiti libero di esplorare l'HTML e il CSS, poi aggiungi un elemento `form` e dagli un insieme `id` impostato su `calorie-counter`.

# --hints--

Dovresti avere un elemento `form`.

```js
assert.exists(document.querySelector('form'));
```

Il tuo elemento `form` dovrebbe avere un attributo `id` impostato su `calorie-counter`.

```js
assert.equal(document.querySelector('form')?.id, 'calorie-counter');
```
