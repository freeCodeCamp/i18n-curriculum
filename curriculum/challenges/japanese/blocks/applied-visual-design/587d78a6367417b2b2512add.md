# --description--

さまざまなセレクターやプロパティを操作することで、面白い形を作ることができます。試しやすい形の一つが三日月の形です。このチャレンジでは、要素の影を設定する`box-shadow`プロパティと、要素の角の丸みを制御する`border-radius`プロパティを使います。

丸くて透明なオブジェクトを作り、影を少し横にずらしてはっきり見えるようにします。この影が実際に見える三日月の形になります。

丸いオブジェクトを作るには、`border-radius`プロパティを50%に設定する必要があります。

以前のチャレンジで、`box-shadow`プロパティが`offset-x`、`offset-y`、`blur-radius`、`spread-radius`、そして`color`の値を順に取ることを覚えているかもしれません。`blur-radius`と`spread-radius`の値はオプションです。

# --instructions--

エディター内の正方形の要素を操作して三日月の形を作ってください。まず、`background-color`を`transparent`に変更し、次に`border-radius`プロパティを50%に設定して円形にします。最後に、`box-shadow`プロパティを変更して、`offset-x`を25px、`offset-y`を10px、`blur-radius`を0、`spread-radius`を0、`color`を`blue`に設定してください。

# --hints--

`background-color`プロパティの値は`transparent`に設定してください。

```js
assert.match(code,/background-color:\s*?transparent;/gi);
```

`border-radius`プロパティの値は`50%`に設定してください。

```js
assert.match(code,/border-radius:\s*?50%;/gi);
```

`box-shadow`プロパティの値は、`offset-x`に25px、`offset-y`に10px、`blur-radius`に0、`spread-radius`に0、最後に`blue`を`color`に設定してください。

```js
assert.match(code,/box-shadow:\s*?25px\s+?10px\s+?0(px)?\s+?0(px)?\s+?blue\s*?;/gi);
```
