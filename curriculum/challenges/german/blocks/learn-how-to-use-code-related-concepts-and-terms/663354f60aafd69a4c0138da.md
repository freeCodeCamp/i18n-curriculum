# --description--

Das `equals symbol` (`=`) wird häufig für Zuweisungen verwendet. Es dient dazu, einer Variablen einen Wert zuzuweisen. Der Variablenname steht links vom Gleichheitszeichen, und der Wert, den Sie in der Variablen speichern möchten, steht rechts davon.

Zum Beispiel weist `name = "Sophie";` den String `"Sophie"` der Variablen `name` zu.

In der Programmierung bedeutet `assign a value` einer Variablen einen bestimmten Wert zu geben. Das `equals symbol` (`=`) wird dafür häufig verwendet. Wenn Sie einer Variablen einen Wert zuweisen, sagen Sie dem Programm, dass es sich diesen Wert merken soll, um ihn später zu verwenden.

Zum Beispiel weist `age = 25` der Variablen `25` den Wert `age` zu.
