# --description--

对于一个整数 $M$，我们定义 $R(M)$ 为所有满足以下所有条件的整数点对 $p$ 和 $q$ 的 $\frac{1}{p·q}$ 之和：

- $1 ≤ p &lt; q ≤ M$
- $p + q ≥ M$
- $p$ 和 $q$ 是互质的。

我们还定义 $S(N)$ 为 $2 ≤ i ≤ N$ 时 $R(i)$ 的和。

我们可以验证 $S(2) = R(2) = \frac{1}{2}$，$S(10) ≈ 6.9147$ 和 $S(100) ≈ 58.2962$。

查找 $S({10}^7)$。将你的答案四舍五入到四位小数。
