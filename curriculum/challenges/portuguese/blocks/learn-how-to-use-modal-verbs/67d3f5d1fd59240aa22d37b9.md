Qual frase está gramaticalmente correta?

## --answers--

`You must to complete the security training.`

### --feedback--

`Must` é um verbo modal e não usa `to` antes do verbo base.

---

`You must completed the security training.`

### --feedback--

O verbo após `must` deve estar em sua forma base (`complete`), não no passado (`completed`).

---

`You must complete the security training.`

---

`You must completing the security training.`

### --feedback--

`Must` é seguido pelo verbo na forma base, não pela forma `-ing`.

# --explanation--

`Must` é um verbo modal, então ele segue estas regras:

- É sempre seguido pela forma base do verbo (sem `to`).

- Não aceita `-s` na terceira pessoa do singular.

- Não usa `do/does` em perguntas ou negativas.
