# --description--

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

Indem Sie `animal` in die runden Klammern der `favoriteAnimal()`-Funktion setzen, sagen Sie JavaScript, dass Sie einen Wert an Ihre `favoriteAnimal`-Funktion übergeben werden. Das bedeutet, dass animal nur ein Platzhalter für einen zukünftigen Wert ist. Aber welchen Wert übergeben Sie?

Die letzte Zeile, `favoriteAnimal('Goat')`, ist der Aufruf Ihrer `favoriteAnimal`-Funktion, bei dem der Wert `'Goat'` an diese Funktion übergeben wird. Hier ist `'Goat'` Ihr Argument. Sie sagen der `favoriteAnimal`-Funktion: „Bitte senden Sie `'Goat'` an die `favoriteAnimal`-Funktion und verwenden Sie `'Goat'` überall dort, wo der `'animal'`-Platzhalter steht.“ Aufgrund der Flexibilität, die ein Parameter bietet, können Sie jedes Tier zu Ihrem Lieblingstier erklären.

Hier ist ein Diagramm, das Ihnen hilft, zu visualisieren, wie Parameter an eine Funktion übergeben werden und wie Werte von ihr zurückgegeben werden.

Beachten Sie, dass Sie durch den Aufruf von `favoriteAnimal()` innerhalb von `console.log()` mit dem Argument `'Goat'` den Rückgabewert der Funktion, den String `"Goat is my favorite animal!"`, in der Konsole ausgeben. Sie übergeben einen Funktionsaufruf `favoriteAnimal('Goat')` als Argument in einem anderen Funktionsaufruf – `log()`.
