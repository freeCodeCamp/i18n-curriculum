# --description--

`interview` は、情報を得るために誰かが質問をする会話のことです。例えば：

`He had an interview for a job at the bank.` - ここでの `interview` は、その仕事について質問された正式な面接を意味します。

`video interview` は、Zoom、Teams、Skype のようなビデオツールを使ってオンラインで行われる面接の一種で、対面で会う代わりに行われます。人々が同じ場所にいられない場合の仕事、学校、研究でよく使われます。
