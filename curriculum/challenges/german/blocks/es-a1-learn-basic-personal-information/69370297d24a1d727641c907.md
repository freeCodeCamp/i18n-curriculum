# --description--

In diesem Austausch wiederholt Camila einen Teil der E-Mail-Adresse, um sie zu bestätigen.

Sie verwendet eine steigende Intonation, um zu zeigen, dass es sich um eine Bestätigungsfrage handelt.

Wenn jemand Informationen mit Frageintonation wiederholt, geschieht dies normalerweise, um zu überprüfen, ob sie korrekt sind.

In dieser Situation kann die andere Person die Information entweder bestätigen oder ablehnen. Zum Beispiel:

- `Sí, correcto.` (Was die Person gesagt hat, ist korrekt).

- `No, es ...` + Korrektur (Was die Person gesagt hat, ist falsch).
