# --explanation--

`Minor bugs` 是系統或程式中的小錯誤或問題。例如：

`There are some minor bugs in the app, but it still runs.` — 這表示問題不嚴重，但需要修復。

`User interface`（通常稱為 `UI`）是人們與軟體互動的視覺零件。 例如： 

`The user interface was updated to include a dark mode。` — 這是指使用者可以直接看到並使用的變更。
