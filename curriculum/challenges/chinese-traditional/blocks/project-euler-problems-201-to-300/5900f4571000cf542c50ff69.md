# --description--

對於整數 $n ≥ 4$，我們定義 $n$ 的下質數平方根，記為 $lps(n)$，為 $\text{小於或等於 \sqrt{n} 的最大質數}$，以及 $n$ 的上質數平方根，$ups(n)$，為 $\text{大於或等於 \sqrt{n} 的最小質數}$。

所以， 例如，$lps(4) = 2 = ups(4)$，$lps(1000) = 31$，$ups(1000) = 37$。

如果一個整數 $n ≥ 4$ 的 $lps(n)$ 和 $ups(n)$ 中有一個能整除 $n$，但不是兩者皆能整除，則稱該整數為半可除數。

不超過 15 的半可除數之和為 30，這些數字是 8、10 和 12。15 不是半可除數，因為它同時是 $lps(15) = 3$ 和 $ups(15) = 5$ 的倍數。作為進一步的例子，不超過 1000 的 92 個半可除數之和為 34825。

不超過 999966663333 的所有半可除數的總和是多少？
