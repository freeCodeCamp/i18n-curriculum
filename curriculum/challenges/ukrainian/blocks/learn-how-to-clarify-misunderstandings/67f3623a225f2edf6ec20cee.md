# --explanation--

`Supposed to be` означає, що щось очікувалося або планувалося статися. Наприклад:

`The new feature was supposed to be ready by last week.` — Це означає, що планували, що це буде готово минулого тижня.

`Coming later` означає, що щось планується статися у майбутньому. Наприклад:

`The update is coming later this month.` — Це означає, що оновлення буде випущене пізніше, а не зараз.
