# --description--

Dire `recommend` significa dire che qualcosa è una buona scelta per qualcuno. È quando dai un consiglio su cosa un’altra persona dovrebbe fare, usare o provare. Per esempio:

`They recommended testing the app on different devices.` - Questo suggerisce la migliore azione per la qualità.

Quando hai azioni che seguono il verbo `recommend`, usale nella forma `-ing`. Per esempio, `I recommend reading this book if you want to understand this topic better`.
