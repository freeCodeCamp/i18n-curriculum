# --description--

`Strange` significa insolito o non previsto. È come trovare qualcosa di diverso da ciò che vedi normalmente. Per esempio: `A strange sound in a car might mean a problem.`

Un `error` è un errore, specialmente uno che impedisce a qualcosa di funzionare correttamente. Per esempio: `An error in a code means there is a mistake that needs fixing.`
