# --description--

請記住，`occasionally` 意味著某事有時會發生，但不是定期或經常發生。它比 `sometimes` 發生頻率低，但比 `rarely` 發生頻率高。

這是上一次挑戰的副詞頻率圖，方便你參考。

| Adverb       | Frequency | Adverb    | Frequency |
| ------------ | --------- | --------- | --------- |
| Never        | 0%        | Often     | 70%       |
| Rarely       | 5%        | Generally | 80%       |
| Seldom       | 15%       | Usually   | 90%       |
| Occasionally | 30%       | Always    | 100%      |
| Sometimes    | 50%       |           |           |
