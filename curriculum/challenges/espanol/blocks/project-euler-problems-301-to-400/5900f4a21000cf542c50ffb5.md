# --description--

Alice y Bob juegan el juego Nim Square.

Nim Square es igual que el Nim normal de tres montones, pero los jugadores solo pueden quitar una cantidad de piedras que sea un número cuadrado de un montón.

El número de piedras en las tres pilas está representado por el triple ordenado ($a$, $b$, $c$).

Si $0 ≤ a ≤ b ≤ c ≤ 29$ entonces el número de posiciones perdedoras para el siguiente jugador es 1160.

Encuentra el número de posiciones perdedoras para el siguiente jugador si $0 ≤ a ≤ b ≤ c ≤ 100\\,000$.
