# --description--

Erstellen Sie eine Zahlenfolge mit dem Pseudozufallszahlengenerator „Blum Blum Shub“:

$$
s_0 = 14025256 \\\\
s_{n + 1} = {s_n}^2 \\; mod \\; 20\\,300\\,713
$$

Verketten Sie diese Zahlen $s_0s_1s_2\ldots$, um einen String $w$ unendlicher Länge zu erzeugen. Dann gilt $w = 14025256741014958470038053646\ldots$

Für eine positive ganze Zahl $k$ gilt: Wenn kein Teilstring von $w$ existiert, dessen Ziffernsumme gleich $k$ ist, wird $p(k)$ als null definiert. Existiert mindestens ein Teilstring von $w$ mit Ziffernsumme $k$, definieren wir $p(k) = z$, wobei $z$ die Startposition des frühesten solchen Teilstrings ist.

Zum Beispiel:

Die Teilstrings 1, 14, 1402, … mit den jeweiligen Ziffernsummen 1, 5, 7, … beginnen an Position 1, daher gilt $p(1) = p(5) = p(7) = \ldots = 1$.

Die Teilstrings 4, 402, 4025, … mit den jeweiligen Ziffernsummen 4, 6, 11, … beginnen an Position 2, daher gilt $p(4) = p(6) = p(11) = \ldots = 2$.

Die Teilstrings 02, 0252, … mit den jeweiligen Ziffernsummen 2, 9, … beginnen an Position 3, daher gilt $p(2) = p(9) = \ldots = 3$.

Beachten Sie, dass der Teilstring 025, der an Position 3 beginnt, eine Ziffernsumme von 7 hat, aber es gab einen früheren Teilstring (beginnend an Position 1) mit Ziffernsumme 7, daher ist $p(7) = 1$ und nicht 3.

Wir können überprüfen, dass für $0 < k ≤ {10}^3$ gilt: $\sum p(k) = 4742$.

Finden Sie $\sum p(k)$ für $0 < k ≤ 2 \times {10}^{15}$.
