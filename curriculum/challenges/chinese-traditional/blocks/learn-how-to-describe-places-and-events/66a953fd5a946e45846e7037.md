# --explanation--

`Real-time` 意味著即時發生，沒有任何延遲。例如：

- `We are having a real-time video chat。`－對話即時進行，回應之間沒有延遲。

`Collaboration` 是指與他人共同合作以達成共同目標。 例如：

- 團隊合作完成了專案。

`Real-time project collaboration` 是指即使成員位於不同地點，也能即時共同合作完成專案。例如：

- `Our team uses a shared platform for real-time project collaboration。`－團隊在專案上即時更新並進行通訊，共同合作。
