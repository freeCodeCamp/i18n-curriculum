# --description--

Programu zinazotumia matawi, yaani zile zinazofanya mambo tofauti ikiwa masharti fulani yatatimizwa, hutegemea tamko za `if`, `else if`, na `else` katika JavaScript. Masharti hayo mara nyingine hujaribu kama matokeo ni sawa na thamani fulani.

Mantiki hii huzungumzwa (kwa Kiingereza, angalau) kama "if x equals y, then ..." ambayo kwa haraka inaweza kutafsiriwa kuwa msimbo ukitumia `=`, au opereta ya mgawo. Hii husababisha mtiririko wa udhibiti usiotarajiwa katika programu yako.

Kama ilivyojadiliwa katika changamoto zilizopita, opereta ya mgawo (`=`) katika JavaScript huweka thamani kwa jina la kigezo. Na opereta za `==` na `===` huchunguza usawa (opereta ya triple `===` huchunguza usawa mkali, yaani thamani na aina zote mbili ni sawa).

Msimbo ulio chini unaweka `x` kuwa 2, ambayo hutathminiwa kama `true`. Karibu kila thamani peke yake katika JavaScript hutathminiwa kuwa `true`, isipokuwa zile zinazojulikana kama thamani "falsy": `false`, `0`, `""` (mfuatano wa herufi tupu), `NaN`, `undefined`, na `null`.

```js
let x = 1;
let y = 2;
if (x = y) {

} else {

}
```

Katika mfano huu, kipande cha msimbo ndani ya tamko la `if` kitaendeshwa kwa thamani yoyote ya `y`, isipokuwa `y` iwe falsy. Kipande cha `else`, ambacho tunatarajia kiendeshwe hapa, hakitafanya kazi kweli.

# --instructions--

Rekebisha masharti ili programu iendeshe tawi sahihi, na thamani inayofaa iwe imepewa `result`.

# --hints--

Msimbo wako unapaswa kurekebisha masharti ili uchunguze usawa, badala ya kutumia mgawo.

```js
assert(result == 'Not equal!');
```

Masharti yanapaswa kutumia `==` au `===` kujaribu usawa.

```js
assert(__helpers.removeJSComments(code).match(/x\s*?===?\s*?y/g));
```
