# --description--

分母の上限 $d$ に対する実数 $x$ の最良近似は、$s ≤ d$ を満たす約分済みの有理数 $\frac{r}{s}$ であり、$\frac{r}{s}$ より $x$ に近い有理数 $\frac{p}{q}$ は $q > d$ となります。

通常、実数の最良近似はすべての分母の上限に対して一意に定まります。しかし例外もあり、例えば分母の上限が $6$ のとき、$\frac{9}{40}$ は $\frac{1}{4}$ と $\frac{1}{5}$ の2つの最良近似を持ちます。実数 $x$ が、少なくとも1つの分母の上限に対して2つの最良近似を持つ場合、$x$ をあいまいな数と呼びます。あいまいな数は必ず有理数であることは明らかです。

分母 $q$ が ${10}^8$ 以下で、$0 < x < \frac{1}{100}$ を満たすあいまいな数 $x = \frac{p}{q}$ はいくつありますか？
