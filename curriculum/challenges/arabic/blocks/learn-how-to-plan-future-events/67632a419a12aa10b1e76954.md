# --explanation--

`Database` تشير إلى نظام يُخزن فيه البيانات ويُنظم.

`Activity` تشير إلى الأفعال أو العمليات التي تحدث في قاعدة البيانات.

- `The database contains customer information and order details.` - هذا يعني أن قاعدة البيانات تخزن بيانات تجارية مهمة.

- `There was unusual activity in the database last night.` - هذا يعني أن شيئًا غير متوقع حدث في قاعدة البيانات.

تشير عبارة `database activity` إلى جميع الإجراءات أو التغييرات أو العمليات التي تحدث داخل نظام قاعدة البيانات.
