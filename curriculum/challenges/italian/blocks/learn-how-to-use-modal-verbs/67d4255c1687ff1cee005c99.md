## --text--

Quale frase è grammaticalmente corretta?

## --answers--

`Everyone must to follow the security guidelines.`

### --feedback--

`Must` è seguito dal verbo base senza `to`.

---

`Everyone must following the security guidelines.`

### --feedback--

`Must` dovrebbe essere seguito dal verbo base, non dalla forma `-ing`.

---

`Everyone must followed the security guidelines.`

### --feedback--

`Must` è seguito dal verbo base, non dalla forma al passato.

---

`Everyone must follow the security guidelines.`

## --video-solution--

4

# --explanation--

Il verbo modale è sempre seguito dalla forma base del verbo. Il verbo modale non prende `to`, `-ing` o forme al passato. Ecco esempi delle forme verbali errate:

- `Must to follow` è errato perché `must` richiede `to`.

- `Must following` è errato perché `must` è seguito da una forma `-ing`.

- `Must followed` è errato perché `followed` è al passato.
