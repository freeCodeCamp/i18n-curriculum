# --interactive--

在 CSS 中，`width` 和 `height` 屬性用於控制網頁元素的尺寸。

這些屬性可以用不同的單位定義，例如像素（`px`）、百分比（`%`）、視窗單位（`vw`、`vh`）等。

`width` 屬性指定元素的寬度。如果你沒有指定寬度，則預設值會設定為 `auto`。這讓瀏覽器根據元素的內容、父容器和顯示型別來決定元素的寬度。對於 `div` 元素，`width: auto` 會使其擴展以填滿父容器的整個寬度。

`height` 屬性指定元素的高度。同樣地，高度的預設值是 `auto`，這表示它會根據內部內容進行調整。

以下是使用 `width` 和 `height` 屬性的範例：

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 100px;
      height: 100px;
      background-color: skyblue
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

在這個範例中，我們有一個 `div` 元素，其類別名稱為 `box`。此元素將佔據高度和寬度各為 `100px`，而背景顏色將是 `skyblue`。

像素是 CSS 中固定大小的測量單位，提供對尺寸的精確控制。 

`min-width` 屬性指定元素可以有的最小寬度。即使內部內容較小，元素也不會縮小到低於此值。

`min-height` 指定元素的最小高度。它確保元素不會比設定的值更矮。

這裡有一個範例：

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 50px;
      min-width: 100px;
      height: 50px;
      min-height: 100px;
      background-color: lightcoral;
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

上述範例示範了 `min-width` 和 `min-height` 的運作方式。

即使盒子的 `width` 和 `height` 設定為 50px，實際上它會是 `100px` × `100px`。這是因為 `min-width` 和 `min-height` 設定為 `100px`，比指定的 `width` 和 `height` 還要大。

請記住，如果 `min-width` 或 `min-height` 大於 `width` 或 `height`，它們將會改寫較小的值。這確保元素不會變得過小，這對於維護一致且可用的設計非常重要。

`max-width` 指定元素可以擴展到的最大寬度，即使有足夠的空間讓它更寬。

`max-height` 指定元素可以增長到的最大高度，無論內容大小為何。

這裡有一個範例：

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 200px;
      max-width: 150px;
      height: 200px;
      max-height: 150px;
      background-color: lightgreen;
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

上述範例示範了 `max-width` 和 `max-height` 如何改寫 `width` 和 `height`。即使盒子設定為 `200px` × `200px`，實際上它會是 `150px` × `150px`。這是因為 `max-width` 和 `max-height` 設定為較小的 `150px`。

請記住，當 `max-width` 或 `max-height` 小於 `width` 或 `height` 時，它們具有優先權。這對於控制佈局中元素的最大尺寸非常重要。

CSS 優先使用 `min-width` 和 `min-height`，而非 `width` 和 `height`。如果值超過限制，`max-width` 和 `max-height` 會限制尺寸。

例如，如果你將 `width` 設定為 `600px` 且 `max-width` 設定為 `500px`，該元素的寬度將被限制在 `500px`。`max-width` 會改寫較寬的設定，使元素保持在指定的最大尺寸內。

這確保元素保持在所需的大小範圍內，無論標準寬度和高度值為何。
