# --interactive--

CSS, das für Cascading Style Sheets steht, ist eine entscheidende Komponente der modernen Webentwicklung. Es ist eine Auszeichnungssprache, die verwendet wird, um Styles für HTML anzuwenden. Einfacher gesagt: Wenn HTML die Struktur einer Webseite ist, dann sorgt CSS dafür, dass sie gut aussieht.

Hier ist ein Beispiel für das Stylen einer Navigationsleiste.

**HINWEIS**: Machen Sie sich noch keine Sorgen, den CSS-Code jetzt schon zu verstehen. Sie werden in zukünftigen Lektionen und Workshops lernen, wie das alles funktioniert. Dies soll Ihnen nur eine Vorstellung davon geben, was Sie mit ein wenig CSS machen können.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<nav class="navbar">
    <ul class="nav-links">
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Services</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>
```

```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: 'Segoe UI', sans-serif;
  background-color: #f4f4f4;
}

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  background-color: #333;
  padding: 1rem 2rem;
  color: white;
}

.navbar .logo {
  font-size: 1.5rem;
  font-weight: bold;
}

.nav-links {
  list-style: none;
  display: flex;
  gap: 1.5rem;
}

.nav-links a {
  color: white;
  text-decoration: none;
  font-weight: 500;
  transition: color 0.3s ease;
}

.nav-links a:hover {
  color: #ff9800;
}
```

:::

Die Hauptaufgabe von CSS ist es, den Inhalt einer Webseite von ihrer visuellen Darstellung zu trennen. Diese Trennung ermöglicht es Webentwicklern, flexiblere und besser wartbare Webseiten zu erstellen.

Mit CSS können Sie das Layout, die Farben, Schriftarten und das gesamte visuelle Erscheinungsbild von Webseiten steuern, ohne die HTML-Struktur zu verändern.

Betrachten wir eine einfache Analogie: Wenn Sie eine Webseite als Haus sehen, wäre HTML das Fundament und das Gerüst, während CSS die Farbe, Tapeten und Dekorationen sind, die das Haus optisch ansprechend und einzigartig machen.

CSS funktioniert, indem es HTML-Elemente auswählt und ihnen Styles zuweist.

Hier ist ein Beispiel, das ein ungestyltes `button`-Element und ein gestyltes zeigt. Um mit diesem Beispiel zu interagieren, müssen Sie den interaktiven Editor aktivieren.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<button class="regular-btn">Unstyled Button</button>
<button class="round-btn">Round Button</button>
```

```css

.round-btn {
  font-size: 1rem;
  font-family: 'Segoe UI', sans-serif;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease, transform 0.2s ease;
  background-color: #2ecc71;
  color: white;
  border-radius: 50px;
  padding: 0.6rem 1.6rem;
}

.round-btn:hover {
  background-color: #27ae60;
  transform: translateY(-2px);
}
```

:::

Diese Styles können Eigenschaften wie Farbe, Schriftgröße und viele weitere umfassen. Durch das Ändern dieser Eigenschaften können Sie das Aussehen einer Webseite drastisch verändern, ohne ihren Inhalt zu ändern.

Eine der stärksten Eigenschaften von CSS ist seine Fähigkeit, responsives Design zu ermöglichen.

Das bedeutet, dass Sie mit CSS Ihre Webseite auf jedem Gerät großartig aussehen lassen können, egal ob Desktop-Computer, Tablet oder Smartphone.

CSS erlaubt es Ihnen, Layouts, Schriftgrößen und andere visuelle Elemente basierend auf der Bildschirmgröße des Geräts, das die Webseite betrachtet, anzupassen.

Versuchen Sie, den interaktiven Editor zu aktivieren und die Größe des Vorschaufensters zu ändern, um zu sehen, wie sich das Layout an die Bildschirmgröße anpasst.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<header class="header">
    <h1>Welcome to My Responsive Site</h1>
    <p>This layout adapts to your screen size!</p>
</header>

<main class="content">
    <section class="box">Box 1</section>
    <section class="box">Box 2</section>
    <section class="box">Box 3</section>
</main>
```

```css
body {
  margin: 0;
  font-family: 'Segoe UI', sans-serif;
  background-color: #f9f9f9;
  color: #333;
}

.header {
  background-color: #4CAF50;
  color: white;
  padding: 2rem;
  text-align: center;
}

.content {
  display: flex;
  gap: 1rem;
  padding: 1rem;
  justify-content: center;
}

.box {
  flex: 1;
  min-width: 200px;
  background-color: #ddd;
  padding: 2rem;
  text-align: center;
  font-size: 1.2rem;
  border-radius: 8px;
  transition: background-color 0.3s;
}

.box:hover {
  background-color: #ccc;
}

/* Responsive: for tablets and below (≤ 768px) */
@media (max-width: 768px) {
  .content {
    flex-direction: column;
    align-items: center;
  }

  .box {
    width: 90%;
    font-size: 1.1rem;
  }
}

/* Responsive: for phones (≤ 480px) */
@media (max-width: 480px) {
  .header {
    padding: 1.5rem 1rem;
  }

  .header h1 {
    font-size: 1.5rem;
  }

  .header p {
    font-size: 1rem;
  }

  .box {
    font-size: 1rem;
    padding: 1.5rem;
  }
}
```

:::

Eine weitere wichtige Funktion von CSS ist seine kaskadierende Natur, wovon auch der Name „Cascading“ herrührt.

Das bedeutet, dass Styles vererbt und überschrieben werden können, was eine hierarchische Struktur des Stylings ermöglicht. Sie werden in zukünftigen Lektionen mehr darüber lernen, wie das funktioniert.

CSS unterstützt auch die Verwendung externer Stylesheets.

Das heißt, Sie können alle Ihre Styling-Regeln in einer separaten Datei speichern, die dann mit mehreren HTML-Seiten verlinkt wird.

Diese Funktion verbessert die Wartbarkeit von Webseiten erheblich, besonders bei größeren Projekten. Anstatt Styles auf jeder einzelnen Seite ändern zu müssen, können Sie Änderungen in einer CSS-Datei vornehmen, die alle verlinkten Seiten beeinflusst.

In der Welt der Webentwicklung spielt CSS eine zentrale Rolle bei der Erstellung optisch ansprechender, responsiver und benutzerfreundlicher Webseiten.

Es ermöglicht Entwicklern, einfache HTML-Dokumente in ansprechende Web-Erlebnisse zu verwandeln, die die Aufmerksamkeit der Nutzer fesseln und ihre Interaktion mit Webinhalten verbessern.
