# --explanation--

`Lastly` 介紹最後一個項目或條款。例如：

`Lastly, please remember to lock the door.` – 這表示這是最後的指示。

`Ethernet cables` 是用來將電腦和其他裝置連接到網路的線材。例如：

`We need more Ethernet cables for the new desks。`－這表示需要更多網路線。

`Ended up` 是指某人或某物最終所在的位置。例如：

`The documents ended up on my desk.`－這表示它們最終是在那裡被找到的。
