# --explanation--

`First off` 用來介紹將要討論或執行的第一件事。它表示列表或重點順序的開始。例如：

- `First off, we need to address the project's scope.` – 這表示首先要處理專案的範圍。

- `First off, let's review the current goals before we start planning.`－這表示在開始規劃之前，首先會檢視目前的目標。
