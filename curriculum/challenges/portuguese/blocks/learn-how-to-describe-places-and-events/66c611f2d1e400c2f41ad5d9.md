# --explanation--

No tempo `Present Simple`, a forma negativa é criada usando `doesn't` ou `isn't` para sujeitos na terceira pessoa do singular (`he, she, it`). 

- Use `doesn't` com verbos de ação para mostrar que algo não está acontecendo, como `It doesn't work.`

- Use `isn't` com formas de `to be` para mostrar que algo não é verdade, como `isn't set up.`

Nesta frase, você precisará tanto de `doesn't` e `isn't` para descrever corretamente por que a funcionalidade não está funcionando como esperado.
