# --interactive--

前のレッスンでは、`"click"` と `"input"` イベントを `addEventListener()` メソッドを使って扱う方法を学びました。このレッスンでは、`removeEventListener()` メソッドを使ってイベントリスナーを削除する方法を学びます。

`removeEventListener()` メソッドは、以前に `addEventListener()` メソッドを使って要素に追加されたイベントリスナーを削除するために使われます。これは、特定の要素のイベントの監視をやめたい場合に便利です。

`removeEventListener()` メソッドの基本的な構文は以下の通りです。

```js
element.removeEventListener("event", listener);
```

`addEventListener()` メソッドと同様に、`removeEventListener()` メソッドは2つの引数を取ります。削除したいイベントと、以前に追加されたリスナー関数です。

また、`removeEventListener()` メソッドには追加のオプションとして3番目の引数を渡すこともできます。この引数は `options` または `useCapture` のいずれかです。

`options` 引数は、イベントリスナーのオプションを指定するオブジェクトで、イベントリスナーをパッシブにするか一度だけにするかなどを設定できます。

`useCapture` 引数は、イベント伝播のフェーズでイベントをキャプチャするかどうかを指定するブール値です。

ほとんどの場合、`removeEventListener()` メソッドにはイベントとリスナーの引数だけを渡せば十分です。

`removeEventListener()` メソッドの動作をよりよく理解するために、例を見てみましょう。

背景色をグレーと青の間で切り替えたい場合、イベントリスナーを使うことができます。

まず、`button` 要素と `body` 要素にアクセスする必要があります。

```js
const bodyEl = document.querySelector("body");
const btn = document.getElementById("btn");
```

次に、グレーと青の色を切り替える関数を作成します。

```js
let isBgColorGrey = true;

function toggleBgColor() {
  bodyEl.style.backgroundColor = isBgColorGrey ? "blue" : "grey";
  isBgColorGrey = !isBgColorGrey;
}
```

現在の背景色を追跡するためにブール変数 `isBgColorGrey` を使っています。背景色がグレーなら青に、青ならグレーに変更します。

次に、ボタンがクリックされたときに `button` 関数を呼び出すために `toggleBgColor` 要素にイベントリスナーを追加します。

```js
btn.addEventListener("click", toggleBgColor);
```


ボタンがクリックされるたびに、body の背景色がグレーと青の間で切り替わります。

次に、段落要素を追加して、ホバーされたときにイベントリスナーを削除するようにHTMLを更新します。

```html
<link rel="stylesheet" href="styles.css" />
<h1>Event Listener examples</h1>
<p id="para">MouseOver this text to remove the event listener</p>
<button id="btn">Change background color</button>
<script src="index.js"></script>
```

段落がホバーされたときにイベントリスナーを削除するには、段落要素にイベントリスナーを追加します。`mouseover` イベントを使い、`button` 要素からイベントリスナーを削除する関数を渡しています。

```js
const para = document.getElementById("para");
const btn = document.getElementById("btn");

para.addEventListener("mouseover", () => {
  btn.removeEventListener("click", toggleBgColor);
});
```

段落がホバーされると、ボタンクリックイベントのイベントリスナーが削除され、ボタンをクリックしても背景色が変わらなくなります。

実際のアプリケーションで `removeEventListener()` メソッドを使う例としては、ウェブアプリでモーダルが閉じられたときやユーザーがログアウトしたときにイベントリスナーを削除する場合があります。
