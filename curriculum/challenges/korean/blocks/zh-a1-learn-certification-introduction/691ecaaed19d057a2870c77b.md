# --description--

freeCodeCamp의 맞춤 입력 기능을 사용하면 작업 중에 바로 병음과 한자를 입력할 수 있습니다.

컴퓨터에서 중국어 키보드를 사용하면 병음을 입력하고 제안 목록에서 한자를 선택할 수 있습니다. 설정 방법은 다음과 같습니다.

## Windows 11에서 설정하기  

1. 시작 메뉴에서 **설정**을 엽니다.

2. **시간 및 언어**로 이동한 후 **언어 및 지역**을 클릭합니다.

3. **선호하는 언어**에서 **언어 추가**를 선택합니다.

4. **중국어**를 검색하고 **중국어(간체)**를 선택합니다.

5. **다음**을 클릭하고 단계를 따라 진행합니다.

언어가 설치되면:  

- **선호하는 언어** 목록에서 해당 언어를 찾습니다.

- 줄임표(…)를 클릭해 추가 옵션을 표시한 후 **언어 옵션**을 선택합니다.

- **키보드**로 이동해 **키보드 추가**를 클릭하고 중국어(간체) 배열을 선택합니다.

키보드 간 전환 방법:  

- **Windows 키 + 스페이스바**를 누릅니다.

- 또는 작업 표시줄의 언어 아이콘(보통 **ENG**로 표시됨)을 클릭하고 중국어를 선택합니다.

## Mac에서 설정하기  

**참고:** 사용하는 운영체제에 따라 단계가 다를 수 있습니다.

1. **애플 메뉴**를 열고 **시스템 환경설정**을 선택합니다.

2. **키보드**로 이동합니다.

3. **입력 소스** 탭을 클릭합니다.

4. **플러스(+)** 버튼을 선택하고 **중국어, 간체**를 클릭한 후 목록에서 **병음 - 간체**를 선택합니다.  

키보드 간 전환 방법:

- **컨트롤 + 스페이스바**를 누릅니다.

- 또는 메뉴 막대의 입력 아이콘을 클릭하고 병음 방식을 선택합니다.

이제 모두 설정되었습니다.
