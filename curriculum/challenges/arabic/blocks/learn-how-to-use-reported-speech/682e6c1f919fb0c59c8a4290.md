# --explanation--

في **الكلام المنقول**، تُستخدم عبارة `said that` لنقل ما قاله شخص ما بطريقة مختلفة، دون استخدام كلماته الحرفية. كلمة `that` اختيارية وغالبًا ما تُحذف في الكلام اليومي. على سبيل المثال:

- الكلام المباشر: `We need to update the software.`

- الكلام المنقول: `He said that we need to update the software.` أو `He said we need to update the software.`

كلا النسختين صحيحتان وتحملان نفس المعنى.
