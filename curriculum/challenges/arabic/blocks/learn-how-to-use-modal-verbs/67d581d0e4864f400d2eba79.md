# --explanation--

`Can` هو فعل مساعد يُستخدم لطلب المساعدة أو الإذن أو القدرة. على سبيل المثال:

- طلب المساعدة: `Can you explain this to me?`

- طلب الإذن: `Can I leave early today?`

- الحديث عن القدرة: `She can swim.`

صوفي تستخدم `can` لطلب المساعدة في فهم الاحتمالات.
