# --explanation--

`Past Continuous` 时态通常强调过去某个动作的持续时间或背景。它用于描述过去某个特定时间正在进行的动作。结构是：**`was`/`were` + 以 `-ing` 结尾的动词**。 

在 Jake 的句子中，`I was reviewing` 表示复习的动作昨天正在进行。另一个例子： 

`She was testing the new feature when the system crashed.`——这里，系统的崩溃发生在她测试新建特色的过程中——测试是一个较长的持续动作。
