# --description--

In den vorherigen Lektionen haben Sie gelernt, wie man mit Funktionsparametern arbeitet.

Ein Funktionsparameter ist eine Variable, die in der Deklaration der Funktion definiert wird und als Platzhalter dient.

Hier ist ein Beispiel für eine Funktion, die einen Parameter hat:

```js
// The parameter is `name`
function greetUser(name) {
  return `Hello, ${name}!`;
}
```

Deklarieren Sie eine Funktion namens `calculateSum`, die zwei Parameter, `num1` und `num2`, entgegennimmt.

assert.isFunction(calculateSum);

const regex = __helpers.functionRegex('calculateSum', ['num1', 'num2']);
assert.match(__helpers.removeJSComments(code), regex);
