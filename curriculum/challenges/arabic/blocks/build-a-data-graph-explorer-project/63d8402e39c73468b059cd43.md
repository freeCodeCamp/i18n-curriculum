# --instructions--

لهذا التحدٍّ، تحتاج إلى إنشاء مستكشف رسم بياني للبيانات باستخدام Python يأخذ مُدخَل وينفذ التالي:

- الحصول على ملف .csv بثلاث طرق
  - رفعه من الكمبيوتر المحلي
  - الحصول على رابط من مُدخَل المستخدم
  - وضع الرابط في الشفرة
- استخدام مكتبة Pandas لحفظ ملف .csv كإطار بيانات
- طباعة العناوين وأول صفين
- تخزين أسماء الأعمدة في قائمة
- اختيار عمود أو عمودين وتحويل البيانات إلى مصفوفات Numpy
- عرض البيانات كرسم مبعثر أو رسم خطي
- القدرة على فعل ذلك لمجموعات أعمدة مختلفة، وتفسير الرسوم البيانية

عند الانتهاء، أرسل عنوان URL لدفتر ملاحظات Colab العام على Google Drive.
