# --explanation--

`To give feedback` significa compartir opiniones, consejos o información sobre algo, generalmente para ayudar a mejorarlo. Por ejemplo:

`She gave feedback on the presentation to help the speaker improve.` - Ella compartió su opinión sobre cómo mejorar la presentación.

`Similar` significa parecido o casi igual. Indica que dos o más cosas comparten características comunes. Por ejemplo:

Tuvimos un problema similar el mes pasado con el sistema de login. – Esto significa que el problema fue como el que está ocurriendo ahora.
