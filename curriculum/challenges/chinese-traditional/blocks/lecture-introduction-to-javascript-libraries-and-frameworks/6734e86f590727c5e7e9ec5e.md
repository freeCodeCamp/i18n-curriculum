# --description--

React 是最受歡迎的 JavaScript 函式庫之一，用於建置使用者介面和網頁應用程式。React 最初由 Facebook 開發並維護，因其效率、彈性和特性，在網頁開發中獲得了極大的人氣。

React 中一個基本的概念是可重複使用的 UI 組件的產生。這些組件，例如按鈕、卡片和頭像組件，可以輕鬆地在你的應用程式中重複使用。你可以將這些組件相互巢狀，以建造更複雜且動態的介面。

React 的一大優勢是這些訂製組件可以隨著資料變更而獨立更新和呈現。

與傳統 JavaScript 需要直接操作 DOM（文件物件模型）不同，React 使用虛擬 DOM，這提升了效率與效率。你將在接下來的課程中學習更多關於虛擬 DOM 及其運作方式。

除了可重複使用的組件之外，React 也提供了一種強大的方式來管理你的應用程式的狀態。狀態指的是決定組件如何呈現和行為的資料。

使用 React，你可以輕鬆追蹤和更新組件的狀態，確保 UI 反映最新的資料。你將在未來的課程中學習更多關於操作狀態的內容。

雖然 JavaScript 生態系中有許多函式庫，但 freeCodeCamp 將主要專注於教學 React，因為它在業界的廣泛使用和需求。

接下來的幾堂課將更深入探討建置訂製組件、管理狀態等內容。
