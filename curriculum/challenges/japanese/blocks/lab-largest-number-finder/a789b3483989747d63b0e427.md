# --description--

このラボでは、与えられた各サブ配列から最大の数を取り出し、それらを集めた配列を返す関数を作成します。

配列はシンプルな`for`ループで繰り返し処理でき、配列構文`arr[i]`で各要素にアクセスできます。

**目的:** 以下のユーザーストーリーを満たし、すべてのテストに合格してラボを完了してください。

**ユーザーストーリー:**

1. 配列の配列を引数に取る`largestOfAll`という関数を作成してください。
2. その関数は、各サブ配列から最大の数を含む配列を返す必要があります。

function largestOfAll(arr) {
  return arr.map(subArr => Math.max.apply(null, subArr));
}

largestOfAll([
  [4, 5, 1, 3],
