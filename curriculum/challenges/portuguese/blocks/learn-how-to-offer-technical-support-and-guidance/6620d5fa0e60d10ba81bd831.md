# --description--

`Gracefully` significa lidar com problemas de uma maneira suave, eficiente e não abrupta. Quando um programa lida com erros `gracefully`, ele gerencia problemas sem travar ou causar grandes interrupções. Por exemplo, em vez de parar de repente, um programa pode mostrar uma mensagem de erro útil.

Lembre-se, `encounter` significa encontrar algo inesperadamente e um `conditional statement` é um código que roda somente se uma certa condição for verdadeira.
