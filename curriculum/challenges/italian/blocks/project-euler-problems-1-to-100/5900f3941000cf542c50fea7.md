# --description--

Una frazione decimale irrazionale viene creata concatenando i numeri interi positivi:

0.12345678910**1**112131415161718192021...

Si può vedere che la dodicesima cifra della parte frazionaria è 1.

Se *d<sub>n</sub>* rappresenta la *n*<sup>esima</sup> cifra della parte frazionaria, trova il valore della seguente espressione.

d<sub>1</sub> × d<sub>10</sub> × d<sub>100</sub> × d<sub>1000</sub> × d<sub>10000</sub> × d<sub>100000</sub> × d<sub>1000000</sub>

```js
function champernownesConstant(n) {
  let fractionalPart = '';
  for (let i = 0; fractionalPart.length <= n; i++) {
    fractionalPart += i.toString();
  }

  let product = 1;
  for (let i = 0; i < n.toString().length; i++) {
    const index = 10 ** i;
    product *= parseInt(fractionalPart[index], 10);
  }

  return product;
}
```
