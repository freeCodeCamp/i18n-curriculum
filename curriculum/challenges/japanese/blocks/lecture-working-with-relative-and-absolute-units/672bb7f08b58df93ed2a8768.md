# --interactive--

前のレッスンでは、ピクセルのような絶対長さ単位について学びました。絶対単位は特定の状況で役立ちますが、相対単位を使いたい場合もあります。

このレッスンでは、2つの相対単位である`em`sと`rem`sについて学びます。

`em`単位は要素のフォントサイズに対して相対的です。`font-size`プロパティにemを使う場合、テキストのサイズは親要素のフォントサイズに対して相対的になります。

これがどのように機能するかを理解するために、例を見てみましょう。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p class="para">I am a paragraph element</p>

<div class="blue-box"></div>
```

```css
.para {
  font-size: 20px;
  margin-bottom: 1.5em;
  border: 2px solid red;
}

.blue-box {
  background-color: blue;
  color: white;
  padding: 10px;
  width: 100px;
  height: 100px;
}
```

:::

HTMLでは、段落と`div`要素があります。段落要素には`para`というクラスがあり、`div`要素には`blue-box`というクラスがあります。

`para`クラスでは、`font-size`を`20px`に、`margin-bottom`を`1.5em`に設定しています。これは、マージンが段落要素のフォントサイズの1.5倍になることを意味します。`1.5em`は段落の下に30ピクセルのマージンをもたらします。マージンが見やすいように`border`を`2px solid red`に設定しています。

`blue-box`クラスでは、背景色を`blue`、テキスト色を`white`、そして4辺すべてに`padding`を`10px`に設定しています。

この例から、段落要素の下と青いボックスの間にクリアなスペースがあることがわかります。

では、`font-size`クラスから`para`プロパティを削除したらどうなるでしょうか？

下のマージンは親要素のフォントサイズに対して相対的になります。この場合、親要素はbody要素で、デフォルトのフォントサイズは`16px`です。

`em`単位の良い使いどころは、ボタンやカードのようなモジュラーコンポーネントを扱うときです。`em`単位を使うことで、パディング、マージン、ボーダーなどコンポーネントのすべての側面がフォントサイズに比例してスケールし、一貫した比率を保てます。

ここまで、要素のフォントサイズをピクセルで設定してきましたが、これはユーザーにとっていくつかの課題があります。

ブラウザの設定内で、デフォルトのフォントサイズを調整できます。

視覚障害のあるユーザーは、読みやすくするためにフォントサイズを大きくする場合があります。しかし、ウェブデザインでフォントサイズをピクセルで設定していると、テキストは他のコンテンツと比例してスケールしません。

この問題を解決する方法の一つが、タイポグラフィに`rem`単位を使うことです。`rem`単位はルート要素のフォントサイズに対して相対的で、そのルート要素は`html`要素です。

デフォルトでは、`html`要素のフォントサイズは`16px`です。ユーザーがブラウザ設定でフォントサイズを大きくすると、`html`要素のフォントサイズも大きくなり、すべてのrem単位が比例してスケールします。

ピクセルの代わりに`rem`単位をフォントサイズに使った例を見てみましょう。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is regular text.</p>
<p class="para">This text is slightly larger.</p>
```

```css
.para {
  font-size: 1.2rem;
  margin-bottom: 1.5em;
  border: 2px solid red;
}
```

:::

フォントサイズを`1.2rem`に設定すると、段落要素のフォントサイズはルート要素のフォントサイズの1.2倍になります。ユーザーがデフォルトのフォントサイズを変更していなければ、段落要素のフォントサイズは`19.2px`になります。これは`16px`の1.2倍だからです。

では、いつ`rem`単位を使うべきでしょうか？`rem`単位は、ユーザーのブラウザ設定に比例してスケールするため、タイポグラフィにはピクセルよりも好まれます。これにより、視覚障害のあるユーザーにとってコンテンツのアクセシビリティが向上します。

`rem`単位は、異なる要素間で一貫した間隔やレイアウトを維持するのにも役立ちます。
