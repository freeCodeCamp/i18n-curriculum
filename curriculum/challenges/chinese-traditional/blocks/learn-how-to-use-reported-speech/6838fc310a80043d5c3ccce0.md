# --explanation--

`Third Conditional` 用來談論過去不真實的情況——未發生的事情——及其可能的結果。它的結構是 `If` + `past perfect`，情態動詞（`could/might/should`）+ `have` + `past participle`。使用這個條件語句表示 `if` 語句中的動作沒有發生，結果也沒有出現，但你想像它是一個可能的結果。<dfn>例如</dfn>：

`If we had tested the system earlier, we might have prevented the failure.`－這表示沒有進行測試，且發生了故障，但如果有進行測試，故障可能就能避免。
