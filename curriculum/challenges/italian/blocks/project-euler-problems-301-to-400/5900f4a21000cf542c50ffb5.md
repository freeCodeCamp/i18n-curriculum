# --description--

Alice e Bob giocano a Nim square.

Nim square è come il Nim normale con tre heap, ma i giocatori possono rimuovere solo un numero quadrato di pietre da un heap.

Il numero di pietre nei tre heap è rappresentato dalla terna ordinata ($a$, $b$, $c$).

Se $0 ≤ a ≤ b ≤ c ≤ 29$ allora il numero di posizioni perdenti per il giocatore successivo è 1160.

Trova il numero di posizioni perdenti per il giocatore successivo se $0 ≤ a ≤ b ≤ c ≤ 100\\,000$.
