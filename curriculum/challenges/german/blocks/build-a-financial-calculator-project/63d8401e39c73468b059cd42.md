# --instructions--

Für diese Aufgabe müssen Sie einen Finanzrechner mit Python erstellen, der Eingaben vom Benutzer entgegennimmt und Folgendes berechnet:

- Annuität mit monatlichem oder kontinuierlichem Wachstum berechnen
- Monatliche Hypothekenzahlung berechnen
- Geschätzten Rentenanlagebetrag ermitteln
- Bestimmen, wie lange es dauert, bis sich ein Betrag bei gegebener Rate verdoppelt
- Logarithmische Gleichungen lösen
- In wissenschaftliche Notation umwandeln und zurück
