# --description--

比起較不具特異性的宣告，更具特異性的 CSS 宣告會擁有較高的優先權。行內樣式（你在之前的課程中已經學過）相較於選擇器擁有最高的特異性，而每種類型的選擇器都有其自身的特異性層級，這會影響宣告的特異性。其他選擇器也會影響特異性，但你目前只專注於已經學過的那些：

1. ID 選擇器（最具特異性的選擇器）
2. 類別選擇器
3. 型別選擇器

只有當一個元素有多個相互衝突的宣告標的時，才會考慮特異性，類似於決勝負的機制。`ID` 選擇器總是會勝過任意數量的 `class` 選擇器，`class` 選擇器總是會勝過任意數量的 `type` 選擇器，而 `type` 選擇器總是會勝過任意數量比它特異性低的選擇器。當沒有宣告的選擇器具有較高的特異性時，較多數量的同一選擇器會勝過較少數量的該選擇器。
