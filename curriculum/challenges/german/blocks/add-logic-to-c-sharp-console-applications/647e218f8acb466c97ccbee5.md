## --text--

Welche der folgenden `for` Anweisungen ist korrekt?

## --answers--

`for (int x = 20: x < 80: x++)`

---

`for (int j = 0; j == 80; j + 1)`

---

`for (int counter = 20; counter < 80; counter++)`
