# --explanation--

`Might` se usa para hablar sobre posibilidad — algo que podría suceder pero no es seguro. Por ejemplo:

- `It might rain tomorrow.` - Hay una posibilidad, pero no es seguro.

- `We might need more time to finish the project.` - Hay una posibilidad, pero no está confirmado.
