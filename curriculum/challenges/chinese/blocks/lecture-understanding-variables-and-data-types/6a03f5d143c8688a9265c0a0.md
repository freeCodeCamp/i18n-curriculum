# --description--

在上一课中，你学习了 Python 中常见的数据类型，包括字串、整数和浮点数类型。当你构建程序时，你需要学会如何查看变量的数据类型。

这是一个示例变量：

```py
developer = 'Devin'
```

要查看 `developer` 的类型，可以像这样使用 `type()` 函数：

```py
developer = 'Devin'

print(type(developer)) # <class 'str'>
```

`<class 'str'>` 的输出表示 `developer` 是字串类型。

如果你调用 `type()` 函数时没有提供任何参数，则会收到以下错误信息：

```md
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: type() takes 1 or 3 arguments
```

以下是你目前学过的所有数据类型及其在终端中的类型显示：

```python
my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 'hello', 8.5}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 'hello', 8.5)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>
```

在程序中，有时你需要确认某个变量是否为特定类型，才能对其执行操作。这时 `isinstance()` 函数非常有用。

这是一个赋值为字串的示例变量：

```py
account_balance = '12'
```

如果你尝试对 `account_balance` 变量执行除法等数学表达式，就会收到错误信息。

```py
account_balance = '12'

account_balance / 2

# Traceback (most recent call last):
#   File "<stdin>", line 1, in <module>
# TypeError: unsupported operand type(s) for /: 'str' and 'int'
```

要查看 `account_balance` 是否为整数，可以使用 `isinstance()` 函数进行检查，如下所示：

```py
account_balance = '12'

isinstance(account_balance, int) # False
```

内置的 `isinstance()` 函数允许你检查变量是否匹配特定的数据类型。它接受一个对象和你想检查的类型，然后返回一个布尔值。在此例中，因为 `account_balance` 是字串，所以它会返回 `False`。

`isinstance()` 函数还允许你一次检查多个类型。

这是一个检查 `account_balance` 是否为 `int` 或 `float` 的示例：

```py
account_balance = 12
isinstance(account_balance, (int, float)) # True
```

在这个例子中，`account_balance` 是整数，所以 `isinstance()` 返回 `True`。如果 `account_balance` 是 `12.0`，`isinstance()` 仍会返回 `True`，因为你正在检查整数或浮点数。

在未来的工作坊和实验中，你将使用 `type()` 和 `isinstance()` 函数，确保变量在执行操作前包含正确的数据类型。
