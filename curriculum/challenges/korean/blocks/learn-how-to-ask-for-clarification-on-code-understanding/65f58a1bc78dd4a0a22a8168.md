# --description--

`Encounter`는 특히 과거형 `encountered`에서 무언가를 우연히 발견하는 것을 의미합니다. 예를 들어 `I encountered an error while testing the software.`는 찾으려 하지 않고 무언가를 발견하는 것을 뜻합니다.

`Pinpoint`는 무언가를 아주 정확하게 찾아내는 것을 의미합니다. 예를 들어 `After reviewing the code, I was able to pinpoint the error.`는 문제의 위치와 원인을 정확히 찾는 것을 뜻합니다.

코딩에서는 문제를 찾는 일이 종종 예상치 못한 문제를 `encountering`하고 그 원인을 `pinpoint`하려는 과정입니다.
