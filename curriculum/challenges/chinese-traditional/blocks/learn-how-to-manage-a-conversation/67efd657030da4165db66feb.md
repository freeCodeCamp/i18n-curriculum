# --explanation--

`It's been a while since...` 通常用於某件事情發生後已經過了相當長的時間。它經常用來指涉某件事情上次發生的時間，但重點在於已經過了多少時間。例如：

`It's been a while since we last met.`－這句話的意思是說話者和對方有一段時間沒見面了。
