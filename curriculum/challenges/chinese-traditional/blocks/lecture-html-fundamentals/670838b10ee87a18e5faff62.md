# --interactive--

`script` 元素用於嵌入可執行的程式碼。大多數開發者會使用它來執行 JavaScript 程式碼。JavaScript 用於為你的網頁添加互動性。使用 JavaScript 的常見範例包括互動式遊戲、影像滾軸，以及即時驗證使用者輸入的動態表單。 

這裡是一個在 HTML 文件中使用 `script` 元素的範例。移除 `alert("Welcome to freeCodeCamp");` 前面的 `//`，你應該會看到一個警示彈出。要查看預覽，你需要啟用互動式編輯器。

:::interactive_editor

```html
<body>
  <script>
    // alert("Welcome to freeCodeCamp");
  </script>
</body>
```

:::

雖然你技術上可以將所有 JavaScript 程式碼寫在 `script` 標記內，但最佳實務是將其聯結到外部 JavaScript 檔案。以下是使用 `script` 元素聯結到外部 JavaScript 檔案的範例：

```html
<script src="path-to-javascript-file.js"></script>
```

這裡使用 `src` 屬性來指定該外部 JavaScript 檔案的位址。`src` 代表「source」。不建議將所有 JavaScript 放在 HTML 文件內的原因是為了關注點分離。關注點分離是一種設計原則，將你的程式分成不同區塊，並讓每個區塊處理不同的關注點。在這個例子中，我們希望將 JavaScript 程式碼與 HTML 程式碼分開。
