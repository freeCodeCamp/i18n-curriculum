# --interactive--

الدرورس السابقة قدّمت لك `console.log()` لكن هذا الدرس سيتعمق أكثر في هدفه واستخدامه.

في JavaScript، `console.log()` أداة بسيطة لكنها قوية تُستخدم لعرض الرسائل أو إخراج المعلومات في وحدة تحكم المتصفح. يستخدمها المطوّرون غالبًا لتصحيح الأخطاء وفحص الشفرة أثناء العمل على برامجهم.

يمكنك استخدام `console.log()` لتسجيل نص أو متغيرات في وحدة التحكم والتأكد من أن الشفرة تعمل بشكل صحيح.

لاستخدام `console.log()`، تستدعي الطريقة مع القيمة أو الرسالة التي تريد إخراجها داخل الأقواس العادية. إليك بعض الأمثلة:

:::interactive_editor

```js
console.log("Hello, world!");

let num = 5;
console.log(num); // 5
```

:::

المثال الأول يطبع `Hello, world!` في وحدة تحكم المتصفح، بينما المثال الثاني يطبع القيمة `5`.

إليك مثال آخر على العمل مع `console.log()`:

:::interactive_editor

```js
let name = "Alice";
console.log("Hello, " + name + "!"); // Hello, Alice!
```

:::

يمكنك أيضًا تمرير قيم متعددة إلى `console.log()` مفصولة بفواصل. على سبيل المثال:

:::interactive_editor

```js
let name = "Alice";
let age = 25;
console.log("Name:", name, "Age:", age); // Name: Alice Age: 25
```

:::

هذا مفيد لتسجيل عدة معلومات دفعة واحدة.

تساعدك طريقة `console.log()` على مراقبة شفرتك أثناء تشغيلها، مما يسهل اكتشاف الأخطاء وفهم كيفية تصرف برنامجك.
