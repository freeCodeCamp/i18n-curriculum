# --instructions--

Створіть сканер портів за допомогою Python.

У файлі `port_scanner.py` створіть функцію з назвою `get_open_ports`, яка приймає аргумент `target` та аргумент `port_range`. `target` може бути URL або IP-адресою. `port_range` — це список із двох чисел, які вказують перше та останнє число діапазону портів для перевірки.

Ось приклади, як може викликатися ця функція:

```py
get_open_ports("209.216.230.240", [440, 445])
get_open_ports("www.stackoverflow.com", [79, 82])
```

Функція має повертати список відкритих портів у заданому діапазоні.

Функція `get_open_ports` також має приймати необов’язковий третій аргумент `True`, щоб вказати режим "Verbose". Якщо він встановлений у true, функція має повертати описовий рядок замість списку портів.

Ось формат рядка, який має повертатися в режимі verbose (текст у `{}` вказує інформацію, яка має з’явитися):

```bash
Open ports for {URL} ({IP address})
PORT     SERVICE
{port}   {service name}
{port}   {service name}
```

Ви можете використовувати словник у `common_ports.py`, щоб отримати правильну назву сервісу для кожного порту.

Наприклад, якщо функцію викликати так:

```py
port_scanner.get_open_ports("scanme.nmap.org", [20, 80], True)
```

Вона має повернути таке:

```bash
Open ports for scanme.nmap.org (45.33.32.156)
PORT     SERVICE
22       ssh
80       http
```

Переконайтеся, що включили правильні пропуски та символи нового рядка.

Якщо URL, переданий у функцію `get_open_ports`, недійсний, функція має повертати рядок: "Error: Invalid hostname".

Якщо IP-адреса, передана у функцію `get_open_ports`, недійсна, функція має повертати рядок: "Error:  Invalid IP address".
