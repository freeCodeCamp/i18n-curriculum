# --description--

このタスクでは、`specific` と `specify` の違いを学びます。`Specific` は、明確に定義されたまたは詳細なものを表す形容詞で、`specify` は何かを詳細に述べるという意味の動詞です。

たとえば、`specific` の指示は `use port 8080` のようなもので、`specify` はその詳細を述べることを意味します：`specify the port number.`

`Setup` は何かの配置や準備を指し、ソフトウェアや機器の準備の文脈でよく使われます。たとえば `setup instructions for the software.` のように。
