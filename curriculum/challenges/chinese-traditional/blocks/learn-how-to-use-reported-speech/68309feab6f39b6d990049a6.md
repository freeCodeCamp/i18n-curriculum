# --explanation--

在回答 `yes`/`no` 問題時，你可以只用主詞和助動詞回覆一個簡短的肯定句——而不必在開頭說 `Yes`。這仍然清楚地確認了該動作。 例如：

- 人物 1：`Was the report submitted on time?`－此人正在詢問報告是否準時繳交。

- 第二人：`It was.`－這是一個完整且禮貌的肯定回答，無需以 `Yes` 開頭。
