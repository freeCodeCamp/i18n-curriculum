# --description--

Hay muchas formas de representar datos en un formato visual. En esta lección, aprenderás sobre algunos tipos de visualizaciones de datos comúnmente usados.

El primer tipo que aprenderás son los gráficos que representan el cambio a lo largo del tiempo, como el rendimiento del precio de acciones o las estadísticas de salud.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-are-common-chart-types-and-when-should-you-use-them-1.png" alt="example area chart">

Otro tipo común de visual sería un gráfico de comparación de categorías. Este tipo de visual muestra comparaciones entre diferentes categorías, como el rendimiento de anuncios en canales de marketing o las ventas por categoría de producto en tiendas en línea.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-are-common-chart-types-and-when-should-you-use-them-2.png" alt="example bubble chart for marketing channel performance leads">

Si necesitas representar la posición de un ítem en una lista ordenada, podrías usar un gráfico de clasificación. Ejemplos de este tipo de visualización incluirían estadísticas de rendimiento o resultados electorales.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-are-common-chart-types-and-when-should-you-use-them-3.png" alt="example ranking chart for election results">

Otro tipo de visualización de datos comúnmente usado sería una parte del todo visual. Esto incluiría gráficos de pastel o treemaps que podrían usarse para representar presupuestos o categorías de productos.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-are-common-chart-types-and-when-should-you-use-them-4.png" alt="example pie chart for budget">

Si estás trabajando con datos que muestran una correlación entre dos o más variables, puedes usar un gráfico de correlación. Estos tipos de gráficos incluirían mapas de calor o diagramas de dispersión que podrían representar datos como ingresos vs. nivel educativo, gasto en publicidad vs. ventas, o temperatura vs. consumo de energía.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-are-common-chart-types-and-when-should-you-use-them-5.png" alt="example scatterplot for income vs. education level">

Hay muchos más tipos de gráficos como diagramas de flujo, gráficos de distribución, gráficos de relaciones y más. Pero esta lección trató de ofrecerte una breve introducción a algunos de los tipos de gráficos que podrías considerar al representar datos complejos.
