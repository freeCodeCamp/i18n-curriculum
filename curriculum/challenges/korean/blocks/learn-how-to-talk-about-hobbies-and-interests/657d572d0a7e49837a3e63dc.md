# --description--

To be `into something`는 어떤 것을 매우 좋아하거나 관심이 많다는 뜻입니다. 취미, 활동, 관심사에 대해 비공식적으로 이야기할 때 쓰입니다. 예를 들어:

- `They are into AI and machine learning.` - 이것은 그들이 AI 공부와 작업을 좋아한다는 뜻입니다.

- `Are you into learning new programming languages?` - 이것은 새로운 프로그래밍 언어를 배우는 것을 즐기거나 좋아하는지 묻는 질문입니다.
