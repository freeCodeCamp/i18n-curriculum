# --explanation--

`Zero Conditional` використовується для речей, які завжди є правдою, коли виконується умова.

Воно слідує структурі `if` + `Present Simple` (для умови) і `Present Simple` (для результату). Наприклад:

- `If you press the power button, the computer starts.` — Натискання кнопки живлення завжди вмикає комп’ютер.

- `If the internet is down, the website doesn't load.` — Вебсайт не відкриється, якщо немає інтернет-з’єднання.

У цьому діалозі `if the BIOS is corrupted` — це умова, а `it prevents the computer from starting` — результат. Це загальна істина про те, як пошкоджений BIOS впливає на комп’ютер.
