# --description--

Da Sie nun die gebräuchlichsten Methoden kennen, um Werte und Arrays zu durchlaufen, gibt es noch einige weitere Möglichkeiten, Arrays zu durchlaufen. Arrays haben eine eingebaute Methode namens `map()`, mit der Sie ein neues Array erstellen können, indem Sie eine Funktion auf jedes Element des ursprünglichen Arrays anwenden. Die `map()`-Methode ändert das ursprüngliche Array nicht. Die Syntax der `map()`-Methode sieht folgendermaßen aus:

```javascript
const array = [1, 2, 3, 4, 5];

const newArray = array.map((arrayValue) => {
  return arrayValue * 2;
});

console.log(newArray); // Output: [2, 4, 6, 8, 10]
```

Die `map()`-Methode erstellt ein neues Array, indem sie die Funktion `(arrayValue) => { return arrayValue * 2; }` auf jedes Element des ursprünglichen Arrays anwendet. Das ist besonders nützlich, wenn Sie die Elemente eines Arrays transformieren möchten, ohne das ursprüngliche Array zu verändern.
