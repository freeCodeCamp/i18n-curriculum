# --description--

在你現有的 `if` 述語之後，為當新薪資低於目前層級的基本薪資時創建另一個 `if` 述語。

在 `if` 述語中，引發一個 `ValueError`，並帶有訊息 `Salary must be higher than minimum salary $`，後面接著目前層級的基本薪資及句號。

# --hints--

你應該在你的 `salary` 設值函式中有第二個 `if` 述語。

當新的薪資低於目前層次的基本薪資時，你應該引發一個 `ValueError`，並帶有訊息 `Salary must be higher than minimum salary $`，後面接著目前層次的基本薪資和句號。

```js
({ test: () => runPython(`
  emp = Employee('Frank', 'trainee')
  built_in_print = print
  print = lambda x: x
  minimum_salary = Employee._base_salaries['trainee']
  try:
    emp.salary = minimum_salary - 1
  except ValueError as e:
    print = built_in_print
    assert str(e) == f"Salary must be higher than minimum salary \${minimum_salary}."
  else:
    assert False, "Expected to raise ValueError with invalid new_salary"
`) })
```

當 `new_salary` 大於目前薪資時，你不應該引發任何例外。

```js
({ test: () => runPython(`
  emp = Employee('Frank', 'trainee')
  built_in_print = print
  print = lambda x: x
  minimum_salary = Employee._base_salaries['trainee']
  try:
    emp.salary = minimum_salary + 1
  except Exception:
    assert False, "Expected not to raise exception with valid new_salary"
  try:
    emp.salary = minimum_salary + 1000
  except Exception:
    assert False, "Expected not to raise exception with valid new_salary"
`) })
```
