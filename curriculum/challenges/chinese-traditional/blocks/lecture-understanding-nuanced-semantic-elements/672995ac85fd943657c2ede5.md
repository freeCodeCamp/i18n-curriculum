# --interactive--

「bring attention to」元素 `b` 通常用於凸顯標示摘要中的關鍵字，或評論中的產品名稱。瀏覽器通常會以粗體顯示這些文字。以下是一個使用 `b` 元素來凸顯標示此評論中產品名稱的範例：

:::interactive_editor

```html
<p>
  We tested several products, including the <b>SuperSound 3000</b> for audio
  quality, the <b>QuickCharge Pro</b> for fast charging, and the
  <b>EcoClean Vacuum</b> for cleaning. The first two performed well, but the
  <b>EcoClean Vacuum</b> did not meet expectations.
</p>
```

:::

瀏覽器將這些文字的零件呈現為粗體文字。這種視覺強調會吸引讀者注意產品名稱。

如果你需要強調文字的重要性，你應該使用 `strong` 元素，而不是 `b` 元素。

`strong` 是一個語意 HTML 元素，用來強調關鍵或緊急的文字。以下是一個使用 `strong` 元素來標示關於顧客可能對產品產生過敏反應的非常重要警告的範例：

:::interactive_editor

```html
<p>
  <strong>Warning:</strong> This product may cause allergic reactions.
</p>
```

:::

`strong` 元素傳達那種緊迫感。

在視覺上兩者非常相似，因為它們預設都是以粗體呈現。但它們的意義卻相當不同。當「bring attention to」元素只是吸引注意力，卻不表示較高的層級重要性時，`strong` 元素則不僅如此。它傳達了重要性或緊急性的感覺。這是它們的主要差異。 

要在它們之間做選擇，請考慮文字的用途及其在周圍內容中的重要性。
