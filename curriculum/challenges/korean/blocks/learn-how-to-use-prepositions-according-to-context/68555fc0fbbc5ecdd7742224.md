# --explanation--

`On`는 `planning on`라는 구에서 사용되며, 이는 어떤 일을 하려고 의도하거나 기대한다는 뜻입니다. 예를 들어:

`We're planning on launching next week.` – 이는 팀이 그렇게 하기를 기대하거나 의도한다는 의미입니다.

`Bringing in`는 팀이나 그룹에 사람을 추가하는 것을 의미합니다. 예를 들어:

`The manager is bringing in a new developer.` – 이는 누군가가 추가되고 있다는 뜻입니다.

`Additional`는 이미 존재하는 것보다 더 많다는 뜻입니다. 예를 들어:

`We need additional help to finish the project.` – 이는 추가적인 도움이 필요하다는 의미입니다.
