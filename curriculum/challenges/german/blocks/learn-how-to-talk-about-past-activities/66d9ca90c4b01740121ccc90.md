# --explanation--

`Deploy` bedeutet, etwas in Betrieb zu nehmen oder verfügbar zu machen. Es bezieht sich oft darauf, Software oder ein Update den Nutzenden zugänglich zu machen. Zum Beispiel:

`The team will deploy the new version next week.` – Die neue Version wird nächste Woche den Nutzenden zur Verfügung gestellt.

Wenn Sie über etwas sprechen, das bereits verfügbar gemacht oder in Betrieb genommen wurde, verwenden Sie die Vergangenheitsform `deployed`. Zum Beispiel:

`They deployed the update yesterday.` – Das Update wurde gestern verfügbar gemacht.
