# --description--

Ein `constant` in der Programmierung ist ähnlich wie eine Variable, aber sobald ihm ein Wert zugewiesen wurde, kann dieser Wert nicht mehr geändert werden. `Constants` werden verwendet, wenn Sie Werte haben, von denen Sie wissen, dass sie im Verlauf des Programms nicht verändert werden müssen. Sie werden oft in Großbuchstaben geschrieben, um sie von Variablen zu unterscheiden.

Zum Beispiel `const PI = 3.14;` In vielen Programmiersprachen wird `const` verwendet, um eine Konstante zu deklarieren. Hier ist `PI` eine `constant` mit dem zugewiesenen Wert `3.14`.
