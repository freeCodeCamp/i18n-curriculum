# --description--

如果两个整数 $N$ 和 $M$ 满足 $N \\neq M$ 且 $N$ 的真因数之和（$\\mathrm{sum}(\\mathrm{propDivs}(N))$）等于 $M$，同时 $M$ 的真因数之和（$\\mathrm{sum}(\\mathrm{propDivs}(M))$）等于 $N$，则称这两个整数为友好点对。

**示例：**

**1184** 和 **1210** 是友好的一对，具有适当的除数：

<ul>
  <li>1, 2, 4, 8, 16, 32, 37, 74, 148, 296, 592 and</li>
  <li>1, 2, 5, 10, 11, 22, 55, 110, 121, 242, 605 respectively.</li>
</ul>

第一个值 **1184** 的除数之和是 **1210**，第二个值 **1210** 的除数之和是 **1184**。

// amicablePairsUpTo :: Int -> [(Int, Int)]
function amicablePairsUpTo(maxNum) {
  return range(1, maxNum)
    .map(x => properDivisors(x)
      .reduce((a, b) => a + b, 0))
    .reduce((a, m, i, lst) => {
      const n = i + 1;

      return (m > n) && lst[m - 1] === n ?
        a.concat([
          [n, m]
        ]) : a;
    }, []);
}

// properDivisors :: Int -> [Int]
function properDivisors(n) {
  if (n < 2) return [];

  const rRoot = Math.sqrt(n);
  const intRoot = Math.floor(rRoot);
  const blnPerfectSquare = rRoot === intRoot;
  const lows = range(1, intRoot)
  .filter(x => (n % x) === 0);

  return lows.concat(lows.slice(1)
    .map(x => n / x)
    .reverse()
    .slice(blnPerfectSquare | 0));
}

// Int -> Int -> Maybe Int -> [Int]
function range(m, n, step) {
  const d = (step || 1) * (n >= m ? 1 : -1);

  return Array.from({
    length: Math.floor((n - m) / d) + 1
  }, (_, i) => m + (i * d));
}
