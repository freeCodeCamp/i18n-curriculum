# --explanation--

`While` можна використовувати для протиставлення двох ідей. Це працює як `even though` або `although`, щоб показати, що обидві речі відбулися одночасно. Наприклад:

`While the design looked good, the app didn't work properly.` – це означає, що дизайн був гарним, але була проблема з тим, як функціонував застосунок.

У цьому випадку були позитивні (швидка реакція команди) та негативні (плутанина щодо відповідальностей) аспекти інциденту. `While` використовується для протиставлення обох.
