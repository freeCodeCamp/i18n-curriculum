# --description--

**現在完成進行式** 用來描述從過去開始且現在仍在發生的動作或情況。當你想強調一個活動的持續時間或它是一個持續進行的處理程序時，它非常有用。

以下是如何構成**現在完成進行式**：主詞 + `has/have been` + 動詞的 `-ing` 形式。這個時態經常與時間表達式如 `for` 和 `since` 一起使用。`For` 用於表示一段時間，例如 2 年或 6 個月，而 `since` 用於表示特定時間點，例如 2018 年或今天早上。

範例：

`I have been studying English for two years.` 這表示我在兩年前開始學習英文，並且現在仍在學習。

`She has been working here since 2018.` 意思是她從 2018 年開始在這裡工作，並且仍然在這裡工作。
