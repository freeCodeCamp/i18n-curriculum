# --description--

学习编程中的 `variables`、`data types` 和 `conditional statements`。

`variable` 就像一个可以保存信息的盒子。这些信息可以是数字、文本或其他类型的 `data`。你可以改变盒子里的内容，这就是为什么它被称为 `variable` —— 因为它可以变化。例如，在 `x = 5` 中，`x` 临时保存了值 5。

`data type` 是一种分类，用于指定 `variable` 可以保存的数据类型。常见的 `data types` 包括整数（整数）、浮点数（带小数的数字）、字串（文本）和布尔（真或假）。例如，如果 `age = 21`，则 `data type` 是整数。

`conditional statement` 是一种命令，告诉计算机只有在特定条件为真时才执行某些操作。这就像在现实生活中根据某种情况做出决定。常见的 `conditional statements` 是 `if-else` 语句。例如，`if (temperature > 20) {console.log("It's warm outside!");}` 是一个 `conditional statement`，当温度高于 20 度时执行 `console.log("It's warm outside!")`。
