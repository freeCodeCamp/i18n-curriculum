# --description--

`input` de número se utiliza para crear un campo de entrada numérica.

Aquí hay un ejemplo de un campo de entrada de número:

```html
<input type="number" id="age" name="age" min="18" max="100">
```

Los atributos `min` y `max` se usan para establecer los valores mínimo y máximo que se pueden ingresar en el campo de entrada.

Debajo de su elemento de `label`, añade un input con el atributo `type` establecido en `"number"` y un `id` de `"age"`.

El atributo `name` debe estar configurado como `"age"`, el atributo `min` debe ser `"3"` y el atributo `max` debe ser `"100"`.
