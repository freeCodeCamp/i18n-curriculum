# --description--

Reduxストアに複数のアクションタイプをどのように処理するか指示できます。たとえば、Reduxストアでユーザー認証を管理しているとします。ユーザーがログインしている状態とログアウトしている状態を表すステートが必要です。これは、`authenticated` プロパティを持つ単一のステートオブジェクトで表します。また、ユーザーログインとユーザーログアウトに対応するアクションを作成するアクションクリエーターと、それに対応するアクションオブジェクトも必要です。

# --instructions--

コードエディターにはストア、アクション、アクションクリエーターが用意されています。複数の認証アクションを処理するために `reducer` 関数を完成させてください。JavaScriptの `switch` 文を `reducer` 内で使い、異なるアクションイベントに対応してください。これはReduxのリデューサーを書く際の標準的なパターンです。switch文は `action.type` を対象にして、適切な認証ステートを返すようにしてください。

**注意:** この時点ではステートのイミュータビリティは気にしなくて大丈夫です。この例は小さく単純なので、各アクションに対して新しいオブジェクトを返せばよいです。たとえば `{authenticated: true}` のようにしてください。また、switch文の中に現在の `default` を返す `state` ケースを書くのを忘れないでください。これは重要です。アプリに複数のリデューサーがある場合、アクションがそのリデューサーに関係なくてもすべてのリデューサーが実行されるためです。その場合、現在の `state` を返す必要があります。

# --hints--

`loginUser` 関数を呼び出すと、type プロパティが文字列 `LOGIN` に設定されたオブジェクトが返されます。

```js
assert(loginUser().type === 'LOGIN');
```

`logoutUser` 関数を呼び出すと、type プロパティが文字列 `LOGOUT` に設定されたオブジェクトが返されます。

```js
assert(logoutUser().type === 'LOGOUT');
```

ストアは `authenticated` プロパティが `false` に設定されたオブジェクトで初期化されるべきです。

```js
assert(store.getState().authenticated === false);
```

`loginUser` をディスパッチすると、ストアのステートの `authenticated` プロパティが `true` に更新されます。

```js
assert(
  (function () {
    const initialState = store.getState();
    store.dispatch(loginUser());
    const afterLogin = store.getState();
    return (
      initialState.authenticated === false && afterLogin.authenticated === true
    );
  })()
);
```

`logoutUser` をディスパッチすると、ストアのステートの `authenticated` プロパティが `false` に更新されます。

```js
assert(
  (function () {
    store.dispatch(loginUser());
    const loggedIn = store.getState();
    store.dispatch(logoutUser());
    const afterLogout = store.getState();
    return (
      loggedIn.authenticated === true && afterLogout.authenticated === false
    );
  })()
);
```

`authReducer` 関数は `switch` 文で複数のアクションタイプを処理するべきです。

```js
assert(
    code.toString().includes('switch') &&
      code.toString().includes('case') &&
      code.toString().includes('default')
  );
```

# --solutions--

```js
const defaultState = {
  authenticated: false
};

const authReducer = (state = defaultState, action) => {

  switch (action.type) {

    case 'LOGIN':
      return {
        authenticated: true
      }

    case 'LOGOUT':
      return {
        authenticated: false
      }

    default:
      return state;

  }

};

const store = Redux.createStore(authReducer);

const loginUser = () => {
  return {
    type: 'LOGIN'
  }
};

const logoutUser = () => {
  return {
    type: 'LOGOUT'
  }
};
```
