# --interactive--

`Number` 타입은 JavaScript와 다른 프로그래밍 언어에서 가장 많이 사용되는 데이터 타입 중 하나입니다. 숫자는 단순해 보일 수 있지만, JavaScript에서 숫자에 대해 탐구할 것이 많습니다. 그럼 좀 더 자세히 살펴보겠습니다. JavaScript에서 `Number` 데이터 타입은 숫자 값을 나타냅니다.

정수와 부동 소수점을 별도의 타입으로 구분하는 다른 많은 프로그래밍 언어와 달리, JavaScript는 숫자를 처리하기 위해 하나의 통합된 `Number` 타입을 사용합니다. 이는 정수, 소수, 심지어 특수 숫자 값도 모두 같은 `Number` 데이터 타입 아래에서 다룰 수 있다는 뜻입니다.

다음은 정수, 부동 소수점 숫자, 음수가 모두 number 타입임을 보여주는 기본 예제입니다:

:::interactive_editor

```js
const wholeNumber = 50;
const decimalNumber = 4.5;
const negativeNumber = -7;

console.log(typeof wholeNumber); // number
console.log(typeof decimalNumber); // number
console.log(typeof negativeNumber); // number
```

:::

JavaScript의 `Number` 타입은 단순 정수와 부동 소수점 숫자부터 `Infinity`와 `NaN` 같은 특수한 경우까지 다양한 숫자 값을 포함합니다. 주요 유형을 살펴보겠습니다. 정수는 분수나 소수 부분이 없는 온전한 숫자입니다. 양수, 음수, 0이 될 수 있습니다. 다음은 몇 가지 예입니다:

:::interactive_editor

```js
const positiveInteger = 100;
const negativeInteger = -25;
const zero = 0;

console.log(typeof positiveInteger); // number
console.log(typeof negativeInteger); // number
console.log(typeof zero); // number
```

:::

부동 소수점 숫자는 소수점이 있는 숫자입니다. JavaScript 개발자들은 이를 흔히 "float"라고 부릅니다. 측정값이나 통화처럼 더 높은 정밀도가 필요할 때 유용합니다. 다음은 몇 가지 예입니다:

:::interactive_editor

```js
const floatingPointNumber = 4.5;
const anotherFloat = 89.56;
const oneMoreFloat = 16.462;

console.log(typeof floatingPointNumber); // number
console.log(typeof anotherFloat); // number
console.log(typeof oneMoreFloat); // number
```

:::

JavaScript는 `Infinity`를 사용해 최대 한도를 초과하는 숫자를 나타낼 수 있습니다. 이는 숫자를 0으로 나누거나 드물게 `Number` 타입의 상한을 넘을 때 발생합니다. 다음은 예제입니다:

:::interactive_editor

```js
const infiniteNumber = 1 / 0;
console.log(infiniteNumber); // Infinity
console.log(typeof infiniteNumber); // number
```

:::

때때로 JavaScript에서 일부 산술 연산은 유효한 숫자를 반환하지 않습니다. 예를 들어, 숫자가 아닌 것에 산술 연산을 시도하면 `NaN`가 나오는데, 이는 "Not a Number"를 의미합니다:

:::interactive_editor

```js
const notANumber = 'hello world' / 2;
console.log(notANumber); // NaN
```

:::

놀랍게도 `NaN`의 타입도 `Number`입니다:

:::interactive_editor

```js
const notANumber = 'hello world' / 2;
console.log(typeof notANumber); // number
```

:::

표준 10진법 외에도 JavaScript는 이진법, 8진법, 16진법 같은 다른 진법의 숫자도 지원합니다. 이진법은 1과 0만 사용하는 2진법 체계입니다. 8진법은 0부터 7까지 숫자를 사용하는 8진법 체계입니다. 16진법은 0부터 9까지 숫자와 a부터 f까지 문자를 사용하는 16진법 체계로, CSS 16진수 색상에서 볼 수 있습니다.
