# --explanation--

`However` ist ein Übergangswort, das am Satzanfang verwendet wird, um eine kontrastierende Idee einzuführen. Zum Beispiel:

`The task seemed easy. However, it turned out to be quite challenging.` – Hier kontrastiert `however` die anfängliche Erwartung mit der Realität.

`Documentation` bezieht sich auf schriftliche oder digitale Aufzeichnungen, die einen bestimmten Prozess, ein Produkt oder ein System erklären oder detaillieren. Zum Beispiel:

`The team updated the project documentation to include the new features.` – In diesem Fall bezieht sich `documentation` auf Aufzeichnungen über das Projekt.
