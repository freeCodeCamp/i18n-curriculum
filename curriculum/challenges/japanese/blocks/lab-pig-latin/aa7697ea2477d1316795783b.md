# --description--

ピッグラテンは、特定の変換ルールに従って英単語を変形する方法です。

**目的：** 以下のストーリーを満たし、すべてのテストに合格してラボを完了してください。

**ストーリー：**

1. 1つの文字列を引数に取る`translatePigLatin`関数を作成してください。
1. 文字列の引数が子音で始まる場合、最初の子音または子音の連続部分を単語の末尾に移動し、`ay`を追加して結果を返してください。
1. 文字列の引数が母音で始まる場合、末尾に`way`を追加して結果を返してください。
1. 最初の母音が単語の途中にある文字列の引数も処理し、適切に変換された文字列を返してください。
1. 母音がまったく含まれていない文字列の引数の場合は、末尾に`ay`を追加して結果を返してください。

注：このラボの文脈では、母音は`a`、`e`、`i`、`o`、および`u`です。文字`y`は母音とはみなされません。

```js
function translatePigLatin(str) {
  if (isVowel(str.charAt(0))) return str + "way";
  let front = [];
  str = str.split('');
  while (str.length && !isVowel(str[0])) {
    front.push(str.shift());
  }
  return [].concat(str, front).join('') + 'ay';
}

function isVowel(c) {
  return ['a', 'e', 'i', 'o', 'u'].indexOf(c.toLowerCase()) !== -1;
}
```
