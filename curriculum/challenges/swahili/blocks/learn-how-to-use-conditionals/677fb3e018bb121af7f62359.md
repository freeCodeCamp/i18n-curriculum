# --explanation--

`First Conditional` hutumika kuelezea hali halisi na zinazowezekana katika siku zijazo. Inaundwa kwa `if` + `Present Simple` (kwa ajili ya sharti) na `will` + kitenzi cha msingi (kwa ajili ya matokeo). Kwa mfano:

`If you debug the code, the app will run smoothly.` - Kutengeneza msimbo kutasababisha programu ifanye kazi ipasavyo.

Katika sentensi sahihi, sharti ni `If we don't back up regularly`, na matokeo ni `we could lose important information`. Hii inaonyesha uwezekano halisi.
