# --explanation--

تخيل أنك تكتب شفرة وتريد إيجاد الدالة أو المتغير الصحيح بسرعة دون كتابة كل شيء.

`Auto-completion` يساعد باقتراح ما تحاول كتابته بناءً على ما كتبته سابقًا. هذا يجعل البرمجة أسرع ويقلل الأخطاء.
