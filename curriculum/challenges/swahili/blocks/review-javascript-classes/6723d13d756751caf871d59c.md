## Misingi ya Kufanya kazi na Madarasa

- **Ufafanuzi**: Madarasa katika JavaScript hutumika kufafanua ramani za kuunda vitu na kufunga data. Madarasa yanajumuisha kijenzi ambacho ni njia maalum inayoitwa moja kwa moja wakati kitu kipya kinapotengenezwa kutoka darasa. Kijenzi hutumika kuanzisha sifa za kitu hicho. Neno kuu la `this` linatumika hapa kurejelea mfano wa sasa wa darasa. Chini ya kijenzi, unaweza kuwa na kinachoitwa njia. Njia ni vitendakazi vilivyofafanuliwa ndani ya darasa vinavyofanya vitendo au shughuli kwenye data au hali ya darasa. Zinatumika kufafanua tabia ambazo mifano ya darasa inaweza kutekeleza.

:::interactive_editor

```js
class Dog {
  constructor(name) {
    this.name = name;
  }

  bark() {
    console.log(`${this.name} says woof!`);
  }
}

const dog = new Dog("Gino");
console.log(dog.name);  // Gino
```

:::

Ili kuunda mfano mpya wa darasa, utatumia neno kuu la `new` ikifuatiwa na jina la darasa:

```js
const dog = new Dog("Gino");
```

Unaweza pia kuunda madarasa kama usemi wa darasa. Hii ni pale darasa linapokuwa lisilo na jina na limepewa thamani kwa kigezo.

:::interactive_editor

```js
const Dog = class {
  constructor(name) {
    this.name = name;
  }

  bark() {
    console.log(`${this.name} says woof!`);
  }
};

const dog = new Dog("Gino");
console.log(dog.name);  // Gino
dog.bark();  // Gino says woof!
```

:::

## Urithi wa Darasa

- **Ufafanuzi**: Katika programu, urithi hukuruhusu kufafanua madarasa yanayorithi sifa na njia kutoka kwa madarasa ya mzazi. Hii huongeza matumizi ya msimbo na kuanzisha uhusiano wa ngazi kati ya madarasa. Darasa la mzazi ni darasa linalotumika kama ramani kwa madarasa mengine. Linafafanua sifa na njia zinazorithiwa na madarasa mengine. Darasa la mtoto ni darasa linalorithi sifa na njia za darasa jingine. Madarasa ya watoto pia yanaweza kuongeza utendaji wa madarasa yao ya mzazi kwa kuongeza sifa na njia mpya. Katika JavaScript, tunatumia neno kuu la `extends` kutekeleza urithi. Neno hili kuu linaonyesha kuwa darasa ni darasa la mtoto wa darasa jingine.

:::interactive_editor

```js
class Vehicle {
  constructor(brand, year) {
    this.brand = brand;
    this.year = year;
  }
}

class Car extends Vehicle {
  honk() {
    console.log("Honk! Honk!");
  }
}

const myCar = new Car("freeCodeCamp Motors", 2019);
console.log(myCar.brand);  // freeCodeCamp Motors
console.log(myCar.year);  // 2019
myCar.honk();  // Honk! Honk!
```

:::

Neno kuu la `super` linatumika kufikia njia, vijenzi, na sehemu za darasa la mzazi.

:::interactive_editor

```js
class Vehicle {
  constructor(brand, year) {
    this.brand = brand;
    this.year = year;
  }
}

class Car extends Vehicle {
  constructor(brand, year, numDoors) {
    super(brand, year);
    this.numDoors = numDoors;
  }
}

const myCar = new Car("freeCodeCamp Motors", 2019, 4);
console.log(myCar.brand);  // freeCodeCamp Motors
console.log(myCar.year);  // 2019
console.log(myCar.numDoors);  // 4
```

:::

## Kufanya kazi na Njia za Static na Sifa za Static

- **Njia za static**: Njia hizi hutumika mara nyingi kwa vitendakazi vya huduma ambavyo havihitaji upatikanaji wa hali maalum ya kitu. Zimefafanuliwa ndani ya madarasa kufunga utendaji unaohusiana.

:::interactive_editor

```js
class Movie {
  constructor(title, rating) {
    this.title = title;
    this.rating = rating;
  }

  static compareMovies(movieA, movieB) {
    if (movieA.rating > movieB.rating) {
      console.log(`${movieA.title} has a higher rating.`);
    } else if (movieA.rating < movieB.rating) {
      console.log(`${movieB.title} has a higher rating.`);
    } else {
      console.log("These movies have the same rating.");
    }
  }
}

let movieA = new Movie("Movie A", 80);
let movieB = new Movie("Movie B", 45);

Movie.compareMovies(movieA, movieB);  // Movie A has a higher rating.
```

:::

Njia za static pia ni msaada katika kutekeleza njia za "kiwanda". Njia ya kiwanda ni njia unayoifafanua pamoja na kijenzi kuunda vitu kulingana na vigezo maalum.

```js
class Pizza {
  constructor(type, price) {
    this.type = type;
    this.price = price;
  }

  static createMargherita() {
    return new this("Margherita", 6.99);
  }
}

let myPizza = Pizza.createMargherita();
console.log(myPizza);  // Pizza { type: "Margherita", price: 6.99 }
console.log(myPizza.type);  // Margherita
```

- **Sifa za static**: Sifa hizi hutumika kufafanua thamani au sifa zinazohusiana na darasa lenyewe, badala ya mifano ya darasa. Sifa za static zinashirikiwa kati ya mifano yote ya darasa na zinaweza kufikiwa bila kuunda mfano wa darasa.

:::interactive_editor

```ts
class Car {
  // Static property
  static numberOfWheels = 4;

  constructor(make, model) {
    this.make = make;
    this.model = model;
  }

  // Instance method
  getCarInfo() {
    return `${this.make} ${this.model}`;
  }

  // Static method
  static getNumberOfWheels() {
    return Car.numberOfWheels;
  }
}

// Accessing static property directly from the class
console.log(Car.numberOfWheels);  // 4
```
