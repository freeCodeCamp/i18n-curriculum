# --description--

ピラミッドが逆でない場合、通常の順序でピラミッドを構築する `else` ブロックが必要になります。

以前のステップで、次のような `else` 文の使い方を学びました。

```js
if (condition) {
  // if condition is true, run this code
} else {
  // if condition is false, run this code
}
```

では、`if` ブロックに `else` ブロックを追加しましょう。
