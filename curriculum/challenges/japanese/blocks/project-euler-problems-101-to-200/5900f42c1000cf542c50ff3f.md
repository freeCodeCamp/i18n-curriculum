# --description--

$x$ を実数とします。

分母の上限を $d$ としたときの $x$ の最良近似とは、約分された形の有理数 $\frac{r}{s}$ で、$s ≤ d$ を満たし、$\frac{r}{s}$ より $x$ に近い有理数はすべて分母が $d$ より大きいものです：

$$|\frac{p}{q} - x| &lt; |\frac{r}{s} - x| ⇒ q > d$$

例えば、分母の上限が $20$ のときの $\sqrt{13}$ の最良近似は $\frac{18}{5}$ であり、分母の上限が $30$ のときの $\sqrt{13}$ の最良近似は $\frac{101}{28}$ です。

$n$ が完全平方数でなく $1 &lt; n ≤ 100000$ のとき、分母の上限を ${10}^{12}$ とした $\sqrt{n}$ の最良近似の分母の総和を求めてください。
