# --description--

`Additional` تعني إضافي أو أكثر، مثل قولك، `She added additional sugar to the cake`.

`Logs` في التكنولوجيا تشير إلى سجلات الأحداث أو العمليات، مثل، `The system logs show when the app was used`.
