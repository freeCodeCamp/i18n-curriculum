# --explanation--

`Could` und `might` sind Modalverben, die verwendet werden, um Möglichkeiten zu beschreiben.

`Could` deutet darauf hin, dass etwas möglich, aber nicht sicher ist. Zum Beispiel:

`If we don't fix this bug, it could cause a system crash.` – Es besteht die Möglichkeit, dass das System nicht mehr funktioniert.

`Might` deutet darauf hin, dass etwas möglich ist, aber weniger sicher als `could`. Zum Beispiel:

`We might need to rewrite some parts of the code.` – Es ist möglich, dass Änderungen notwendig sein werden.
