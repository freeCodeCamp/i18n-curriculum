# --explanation--

`They'll` 是 `they will` 的縮寫。它用來談論預期或預測將在未來發生的事情。 

`Likely` 意味著某事很可能發生或有很高的機率發生。 例如：

- `They'll likely ask for more information.`－這預測了客戶可能會要求更多資訊。

- `The feature will likely need additional testing.`－這顯示測試很可能需要進行。
