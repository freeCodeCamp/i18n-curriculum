# --description--

Wenn Sie etwas ausdrücken müssen, das verboten ist oder eine starke Empfehlung dagegen darstellt, etwas zu tun, können Sie `mustn't` (Kontraktion von `must not`) verwenden, um einen negativen Satz zu bilden.

Zum Beispiel bedeutet `You mustn't touch the wet paint`, dass es sehr wichtig ist, die Farbe nicht zu berühren, weil sie nass ist.

`Anyone` wird verwendet, wenn Sie allgemein über alle Menschen sprechen und normalerweise in Fragen oder negativen Sätzen:

`Don't tell anyone` bedeutet, erzähle keiner Person etwas.

`Someone` wird verwendet, wenn Sie in einem positiven Satz über eine unbestimmte Person sprechen:

`I need someone to help me` bedeutet, dass Sie eine unbestimmte Person brauchen, die hilft.
