# --description--

如果你使用你的移动设备学习西班牙语课程，你需要确保你的设备已准备好输入重音符号和特殊字符，比如 `á`、`é` 或 `ñ`。

## 快速按住技巧

输入并按住你需要的字母是输入带重音字母的最简单方法。

1. 当键盘显示时，按住字母键，例如 `a` 或 `n`。

2. 将弹出一个带有重音版本如 `á` 或 `ñ` 的菜单。

3. 滑动你的手指到你想要的 `character` 上，然后抬起以选择它。

当你只偶尔需要重音符号时，这是完美的选择。 

## 添加语言键盘

另一个备选方案是添加西班牙语键盘版型，这有助于更快地访问重音符号和其他字符。

### 在 Android 上

**注意：** Android `path` 不是通用的，可能因设备而异。

1. 打开设置。

2. 进入系统（或通用管理，取决于设备）。

3. 触碰 键盘 设置（或 语言 与 输入）。

4. 触碰 Languages 和 types（或添加 keyboard）。

5. 选择 Spanish (US)，或触碰 Manage input languages 并选择 Spanish (US)。

6. 如果出现提示，储存或触碰 Done。

### 在 iPhone 上

1. 打开设置。

2. 转到通用的 > Keyboard。

3. 触碰键盘。

4. 触碰 添加 新建 键盘。

5. 选择西班牙语（拉丁美洲）。

6. 使用键盘上的地球图标切换语言。

## 第三方键盘

你还可以为你的手机或平板安装不同的应用程序，以便轻松使用西班牙语键盘。这些应用程序通常提供智能建议、多语言支持以及更快捷输入重音符号的方法等实用特色。
