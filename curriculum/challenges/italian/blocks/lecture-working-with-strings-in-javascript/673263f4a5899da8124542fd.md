# --interactive--

Il metodo `prompt()` è una parte importante dell’interazione di JavaScript con l’utente. È uno dei modi più semplici per ottenere input da un utente tramite un piccolo riquadro di dialogo a comparsa.

Lo vedrai spesso usato nei casi in cui la pagina web ha bisogno di un’informazione dall’utente, come un nome o qualche altro tipo di input testuale.

Allora, cosa fa esattamente il metodo `prompt()`? Apre un riquadro di dialogo che chiede all’utente di inserire un input, e poi restituisce il testo inserito dall’utente come stringa.

Il metodo `prompt()` prende due argomenti: il primo è il messaggio che apparirà all’interno del riquadro di dialogo, solitamente per invitare l’utente a inserire un’informazione. Il secondo è un valore predefinito, che è opzionale e riempirà inizialmente il campo di input.

```js
prompt(message, default);
```

Ecco un esempio di come funziona.

**NOTA**: questo esempio include codice che non hai ancora imparato. Non preoccuparti di cercare di capire tutto il codice. Serve solo a illustrare come funziona il metodo `prompt()` e a garantire che il prompt non appaia immediatamente quando la pagina si carica, cosa che potrebbe risultare invadente. Se hai abilitato l’editor interattivo, puoi provarlo tu stesso.

:::interactive_editor

```html
<button id="prompt-btn">Show Prompt</button>
<p id="output"></p>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("prompt-btn");
const output = document.getElementById("output");
btn.addEventListener("click", () => {
  const userName = prompt("What is your name?", "Guest");
  output.textContent = "Hello, " + userName + "!";
});
```

:::

In questo esempio, quando l’utente clicca sul pulsante, il metodo `prompt()` mostra un riquadro di dialogo con il messaggio `What is your name?` e un campo di input che inizialmente contiene il valore `Guest`.

Se l’utente digita il proprio nome e preme "OK", la variabile `userName` memorizzerà il valore inserito. Se l’utente preme "Annulla", la variabile `userName` sarà impostata su `null`. `null` indica che l’utente non ha fornito alcun input. Il paragrafo di output mostrerà quindi un messaggio di saluto usando il nome fornito o `null` se l’utente ha annullato. Imparerai tecniche per evitare di mostrare `null` quando un utente annulla il prompt nelle lezioni future.

Tieni presente che il metodo `prompt()` bloccherà l’esecuzione dello script finché l’utente non interagisce con il riquadro di dialogo.

Questo significa che il resto del tuo codice JavaScript non verrà eseguito finché l’utente non fornisce un input e clicca su "OK", oppure annulla il prompt.

Un altro punto da considerare è che, sebbene `prompt()` sia utile per test veloci o piccole applicazioni, generalmente viene evitato nelle applicazioni web moderne e complesse a causa della sua natura invasiva e del comportamento incoerente tra diversi browser.

Capendo il metodo `prompt()`, ottieni un modo semplice per interagire con gli utenti e recuperare informazioni direttamente tramite il browser, anche se potrebbe non essere molto usato nelle applicazioni web moderne.
