# --explanation--

`Make sure to`는 화자가 반드시 어떤 일을 하겠다는 책임이나 약속을 표현할 때 사용합니다. 예를 들어:

`Make sure to save your work before closing.` — 프로그램을 종료하기 전에 저장하는 것을 잊지 마세요.

`As needed`는 필요할 때만 어떤 일을 한다는 뜻입니다. 예를 들어:

`We can add more team members as needed.` — 필요할 때만 사람을 추가할 것입니다.
