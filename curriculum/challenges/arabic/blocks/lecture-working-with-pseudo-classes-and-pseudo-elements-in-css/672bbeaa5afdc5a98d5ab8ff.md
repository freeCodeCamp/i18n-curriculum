# --interactive--

تُستخدم الفئات المزيفة للموقع لتنسيق الروابط والعناصر المستهدفة داخل المستند الحالي. توفر طريقة لتطبيق أنماط بناءً على ما إذا تم زيارة الرابط أو ما إذا كان العنصر في حالة تركيز حالياً.

أمثلة على الفئات المزيفة للموقع هي:

- `:link`
- `:visited`
- `:any-link`
- `:local-link`
- `:target`

لنلق نظرة أعمق على كل واحدة من هذه الفئات المزيفة.

تسمح لك الفئة المزيفة `:link` باستهداف جميع الروابط غير المزارة على صفحة الويب. يمكنك استخدامها لتنسيق الروابط بشكل مختلف قبل أن ينقر المستخدم عليها. على سبيل المثال، قد ترغب في جعل جميع الروابط غير المزارة باللون الأزرق أو بلون موقعك الأساسي:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a target="_blank" href="https://www.example.com">Visit Example.com</a>
```

```css
a:link {
  color: magenta;
}
```

:::

في هذه الحالة، سيظهر أي رابط لم ينقر عليه المستخدم بعد باللون الأرجواني. بمجرد نقر المستخدم على الرابط، لن ينطبق نمط `:link` بعد ذلك، وتبدأ الفئة المزيفة `:visited` في العمل. تظهر الفئة المزيفة `:visited` بعد نقر المستخدم على الرابط، لذا يمكنك استخدامها لاستهداف الروابط التي نقر عليها المستخدم بالفعل.

إليك مثالاً على تغيير حالة الرابط المزور إلى اللون `purple`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a target="_blank" href="https://www.example.com">Visit Example.com</a>
```

```css
a:visited {
  color: purple;
}
```

:::

تساعد الفئة المزيفة `:visited` المستخدمين على التمييز بين الروابط التي زاروها وتلك التي لم يزوروها.

الفئة المزيفة `:any-link` هي مزيج من الفئتين المزيفتين `:link` و`:visited`. لذا فهي تطابق أي عنصر وسم الرابط الذي يحتوي على خاصية `href`، بغض النظر عما إذا كان الرابط مزورًا أم لا.

إليك مثالاً على تغيير لون الرابط للفئة المزيفة `:any-link` إلى اللون `crimson`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a target="_blank" href="https://www.example.com">Visit Example.com</a>
```

```css
a:any-link {
  color: crimson;
}
```

:::

تستهدف الفئة المزيفة `:local-link` الروابط التي تشير إلى نفس المستند. يمكن أن تكون مفيدة عندما تريد التمييز بين الروابط الداخلية والخارجية. حالياً، لا يدعم أي متصفح الفئة المزيفة `:local-link`.

تختار الفئة المزيفة `:target` العنصر الذي يطابق معرف جزء عنوان URL الحالي، مثل `#section1`. وهي مفيدة جدًا للصفحات التي تحتوي على تنقل داخل الصفحة.

إليك مثال HTML يمثل تنقلًا داخل الصفحة. يستخدم CSS الفئة المزيفة `:target` لتنسيق القسم الذي يطابق المكان الذي ينتقل إليه المستخدم:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<nav id="table-of-contents">
  <ul>
    <li><a href="#section1">Introduction</a></li>
    <li><a href="#section2">Features</a></li>
  </ul>
</nav>

<section id="section1">
  <h2>Introduction</h2>
  <p>This is the introduction section.</p>
</section>

<section id="section2">
  <h2>Features</h2>
  <p>This section describes the features.</p>
</section>
```

```css
section:target {
  background-color: green;
  border: 2px solid green;
  padding: 10px;
}
```

:::

عندما ينقر المستخدم على أحد روابط التنقل، سيتغير لون خلفية القسم المعني إلى اللون الأخضر.
