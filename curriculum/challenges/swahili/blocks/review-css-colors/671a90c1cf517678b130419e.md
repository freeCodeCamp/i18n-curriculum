## Nadharia ya Rangi

- **Ufafanuzi wa Nadharia ya Rangi**: Hii ni somo la jinsi rangi zinavyoshirikiana na jinsi zinavyoathiri mtazamo wetu. Inajumuisha uhusiano wa rangi, muafaka wa rangi, na athari za kisaikolojia za rangi.
- **Rangi za Msingi**: Rangi hizi, ambazo ni njano, bluu, na nyekundu, ni rangi msingi ambazo rangi zote nyingine zinatokana nazo.
- **Rangi za Sekondari**: Rangi hizi hutokana na kuchanganya kiasi sawa cha rangi mbili za msingi. Kijani, machungwa, na zambarau ni mifano ya rangi za sekondari.
- **Rangi za Tertiary**: Rangi hizi hutokana na kuchanganya rangi ya msingi na rangi ya sekondari inayopakana nayo. Njano-Kijani, Bluu-Kijani, na Bluu-Zambarau ni mifano ya rangi za tertiary.
- **Rangi za Joto**: Rangi hizi, ambazo ni pamoja na nyekundu, machungwa, na njano, huleta hisia za faraja, joto, na utulivu.
- **Rangi za Baridi**: Rangi hizi, ambazo ni pamoja na bluu, kijani, na zambarau, huleta hisia za utulivu, amani, na taaluma.
- **Gurudumu la rangi**: Gurudumu la rangi ni mchoro mduara unaoonyesha jinsi rangi zinavyohusiana. Ni chombo muhimu kwa wabunifu kwa sababu huwasaidia kuchagua mchanganyiko wa rangi.
- **Mipangilio ya Rangi ya Mwandamizi**: Mipangilio hii ya rangi huunda uzoefu wa muafaka na utulivu. Ina rangi zinazopakana, ambazo ziko karibu kwenye gurudumu la rangi.
- **Mipangilio ya Rangi ya ya nyongeza**: Mipangilio hii ya rangi huleta tofauti kubwa na athari za kuona. Rangi zao ziko kwenye pande tofauti kabisa za gurudumu la rangi, zikilinganishwa na kila nyingine.
- **Mpangilio wa Rangi wa Triadic**: Mpangilio huu wa rangi una rangi angavu. Hutengenezwa kwa rangi ambazo ziko takriban umbali sawa kutoka kwa kila mmoja. Ikiwa zitaunganishwa, zitaunda pembetatu sawa kwenye gurudumu la rangi.
- **Mpangilio wa Rangi wa Monochromatic**: Kwa mpangilio huu wa rangi, rangi zote hutokana na rangi msingi moja kwa kurekebisha uangavu, giza, na usafi wake. Hii huleta hisia ya umoja na muafaka huku ikizalisha tofauti pia.

## Njia Tofauti za Kufanya kazi na Rangi katika CSS

- **Rangi Zilizotajwa**: Rangi hizi ni majina ya rangi yaliyotanguliwa ambayo vivinjari vinatambua. Mifano ni pamoja na `blue`, `darkred`, `lightgreen`.
- **Kitendakazi cha `rgb()`**: RGB inasimama kwa Red, Green, na Blue — rangi za msingi za mwanga. Rangi hizi tatu huunganishwa kwa nguvu tofauti ili kuunda rangi nyingi. Kitendakazi cha `rgb()` hukuruhusu kufafanua rangi kwa kutumia mfano wa rangi wa RGB.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>RGB Color</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

- **Kitendakazi cha `rgba()`**: Kitendakazi hiki huongeza thamani ya nne, alpha, inayodhibiti uwazi wa rangi. Ikiwa haijatolewa, thamani ya alpha ni `1` kwa chaguo-msingi.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>RGBA Background</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

- **Kitendakazi cha `hsl()`**: HSL inasimama kwa Hue, Saturation, na Lightness — sehemu kuu tatu zinazofafanua rangi. Sintaksia ya kisasa ya `hsl()` pia inaunga mkono thamani ya hiari ya alpha kwa kutumia kitenzi cha `/`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>HSL Color</p>
<p class="alpha">HSL Color with Alpha</p>
```

```css
p {
  color: hsl(120 100% 50%);
}

.alpha {
  color: hsl(120 100% 50% / 0.8);
}
```

:::

- **Kitendakazi cha `hsla()`**: Kitendakazi cha `hsla()` ni njia ya zamani ya kuongeza uwazi kwa rangi za HSL. CSS ya kisasa inapendelea kutumia `hsl()` yenye kitenzi cha `/` kwa thamani za alpha.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>HSLA Background</div>
```


```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

- **Hexadecimal**: Msimbo wa hex (mfupi wa msimbo wa hexadecimal) hutumika kuwakilisha rangi katika mfano wa rangi wa RGB. "Hex" inamaanisha mfumo wa nambari wa msingi-16, unaotumia tarakimu `0` hadi `9` na herufi `A` hadi `F`. Msimbo wa hex unaweza kuwa na herufi 6, kama `#F498A2`, au mfupi wa herufi 3, unaofanya kazi pale kila jozi katika fomu kamili inapotumia herufi ileile mara mbili. Kwa mfano, `#2FB` ni sawa na `#22FFBB`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1 class="hex-text">Hex Text</h1>
<p class="hex-bg">Hex Background</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}

p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

## Sifa ya `box-shadow`

- **Ufafanuzi**: Sifa ya `box-shadow` inaweka kivuli kimoja au zaidi kuzunguka kipengele.
- **Thamani za Mbali**: Lazima ueleze thamani za usawa (`offset-x`) na wima (`offset-y`). Thamani chanya ya `offset-x` husogeza kivuli kulia, wakati thamani hasi husogeza kushoto. Thamani chanya ya `offset-y` husogeza kivuli chini, wakati thamani hasi husogeza juu. Ikiwa thamani ni `0`, huna haja ya kuweka kipimo.
- **Radius ya Blur**: Thamani hii ya hiari hudhibiti jinsi kivuli kinavyoonekana kuwa na ukungu. Ikiwa haijajumuishwa, chaguo-msingi ni `0`, inayounda mipaka kali. Thamani kubwa zaidi inafanya kivuli kuwa laini zaidi.
- **Radius ya Kuenea**: Thamani hii ya hiari hudhibiti kiasi ambacho kivuli kinaenea au kupungua. Ikiwa haijajumuishwa, chaguo-msingi ni `0`.
- **Rangi ya Kivuli**: Unaweza kuweka rangi kwa kutumia majina ya rangi, thamani za hex, `rgb()`, `rgba()`, `hsl()`, au kazi za `hsla()`.
- **Neno la `inset`**: Kuongeza neno la `inset` huweka kivuli ndani ya kipengele badala ya nje.
- **Kuweka Kivuli Kadhaa cha Sanduku**: Unaweza kuweka vivuli vingi kwa kuvitenganisha kwa koma. Vivuli huwekwa kwa tabaka kutoka mbele kwenda nyuma.

### Sintaksia

```css
box-shadow: offset-x offset-y blur-radius spread-radius color;
```

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="shadow-box">Shadow Color Example</div>
```

```css
.shadow-box {
  width: 200px;
  padding: 20px;
  background-color: lightblue;
  box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}
```

## Mchanganyiko wa Mstari na Mviringo

- **Mwinuko wa mstari**: Mwinuko huu huunda mchanganyiko wa polepole kati ya rangi kwenye mstari wa moja kwa moja. Unaweza kudhibiti mwelekeo wa mstari huu kwa kutumia maneno muhimu kama `to top`, `to right`, `to bottom right`, au pembe kama `45deg`. Unaweza kutumia rangi yoyote halali ya CSS na vituo vya rangi vingi kama unavyotaka.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient">Linear Gradient</div>
```

```css
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

:::

- **Mwinuko wa mviringo**: Mwinuko huu huunda mwinuko wa mduara au mviringo unaotoka katikati.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient">Radial Gradient</div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::
