# --description--

$n > 1$ のとき、$t(n) = 2n^2 - 1$ という形の数 $t(n)$ を考えます。

最初の数は 7、17、31、49、71、97、127、161 です。

$49 = 7 \times 7$ と $161 = 7 \times 23$ のみが素数ではありません。

$n ≤ 10000$ のとき、$t(n)$ が素数である数は 2202 個あります。

$n ≤ 50\,000\,000$ のとき、$t(n)$ が素数である数はいくつありますか？
