# --interactive--

深入学习 JavaScript 后，你会很快发现执行数学运算是一项常见任务。 虽然基本的算术运算符可以处理简单的计算，但 JavaScript 提供了一个内置 `Math` 对象来应对更复杂的数学挑战。

这款便捷的工具提供了多种方法，让你更轻松地执行高级计算和操作数字。 让我们一起来探索这些方法，看看它们如何简化你的编码体验。

`Math.random()`方法生成一个介于 `0`（包含）和 `1`（不包含）之间的随机浮点数。 这意味着输出可以是 `0`，但它永远不会达到 `1` 。 这是一个使用 `Math.random()` 方法的示例：

:::interactive_editor

```js
const randomNum = Math.random();

console.log(randomNum);
// any number between 0 and 1 – 0 inclusive and 1 exclusive
```

:::

`Math.min()` 和 `Math.max()` 都接收一组数字，并分别返回最小值和最大值。 这是一个同时使用这两种方法的示例：

:::interactive_editor

```js
const smallest = Math.min(1, 5, 3, 9);
console.log(smallest); // 1

const largest = Math.max(1, 5, 3, 9);
console.log(largest); // 9
```

:::

第一个 `console.log()` 将记录数字 `1`，因为 `1` 是数字列表中最小的一个。 第二个 `console.log()` 将记录数字 `9`，因为 `9` 是列表中最大的数字。

如果你想将数字向上或向下舍入到最接近的整数，可以使用 `Math.ceil()` 和 `Math.floor()` 方法。下面是使用 `Math.ceil()` 的示例：

:::interactive_editor

```js
console.log(Math.ceil(4.3)); // 5
```

:::

`Math.ceil()` 将把 `4.3` 向上取整为最接近的整数，在这里是 `5`。 现在，让我们来看看向下取整的方法：

:::interactive_editor

```js
console.log(Math.floor(4.7)); // 4
```

:::

`Math.floor()` 会将 `4.7` 向下舍入到最接近的整数，在此情况下为 `4`。`Math.round()` 是 `Math.ceil()` 和 `Math.floor()` 的混合体。它会根据小数点将数字四舍五入到最接近的整数：

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

因此，如果小数位小于 `5`，数字将向下取整。 如果小数位大于等于 `5`，数字将向上取整。 `Math.floor()` 和 `Math.random()` 的一个实际应用是在两个整数之间生成一个随机数。 这是其语法：

:::interactive_editor

```js
const max = 10;
const min = 5;
const randomNum = Math.floor(Math.random() * (max - min + 1)) + min;
console.log(randomNum);
```

:::

生成一个介于 `20` 和 `1` 之间的随机数将如下所示：

:::interactive_editor

```js
const randomNumBtw1And20 = Math.floor(Math.random() * 20) + 1;
console.log(randomNumBtw1And20);
```

:::

另一个有用的 `Math` 方法是 `Math.trunc()` 方法。`Math.trunc()` 会去除数字的小数部分，只返回整数部分，不进行四舍五入：

:::interactive_editor

```js
console.log(Math.trunc(2.9)); // 2
console.log(Math.trunc(9.1)); // 9
```

:::

如果需要获取一个数字的平方根或立方根，可以分别使用 `Math.sqrt()` 和 `Math.cbrt()` 方法：

:::interactive_editor

```js
console.log(Math.sqrt(81)); // 9
console.log(Math.cbrt(27)); // 3
```

:::

第一条日志语句将记录 `9`，因为 `81` 的平方根是 `9`，第二条日志语句将记录 `3`，因为 `27` 的立方根是 `3`。 如果需要获取一个数字的绝对值，可以使用 `Math.abs()` 方法：

:::interactive_editor

```js
console.log(Math.abs(-5)); // 5
console.log(Math.abs(5)); // 5
```

:::

`Math.abs()` 返回一个数字的绝对值，将负数变为正数。 我们将学习的最后一个方法是 `Math.pow()` 方法：

:::interactive_editor

```js
console.log(Math.pow(2, 3)); // 8
console.log(Math.pow(8, 2)); // 64
```

:::

`Math.pow()` 接收两个数字，第二个数字是第一个数字的指数。 属于 `Math` 对象的方法还有很多，你可以自行探索。 这些只是 JavaScript 代码库中比较常用的几种。
