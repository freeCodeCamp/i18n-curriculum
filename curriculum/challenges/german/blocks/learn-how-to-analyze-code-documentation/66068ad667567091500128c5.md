# --description--

Eine `library` ist wie ein großes Bücherregal mit viel Code, den Programmierende verwenden können. Stellen Sie sich das vor wie ein Kochbuch mit vielen Rezepten (Funktionen), die Sie nutzen können, um verschiedene Gerichte (Projekte) zu kochen (programmieren).

Zum Beispiel könnte eine `library` in der Programmierung eine Funktion enthalten, um Diagramme zu erstellen. Anstatt Code von Grund auf neu zu schreiben, können Sie diese Funktion verwenden, so wie Sie einem Rezept folgen, um einen Kuchen zu backen.
