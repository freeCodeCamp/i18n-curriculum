## --text--

Welcher Satz verwendet das Wort `in` auf dieselbe Weise wie in diesem Satz: `The server room is located in the building's basement`?

## --answers--

`The developer worked in the morning to fix the bug.`

### --feedback--

Diese Verwendung von `in` bezieht sich auf Zeit, nicht auf einen Ort wie im Beispielsatz.

`She found an error in the code after testing.`

### --feedback--

---

---

# --explanation--

Im Beispielsatz zeigt `in` einen physischen Ort — etwas innerhalb eines Ortes.

Die richtige Antwort verwendet `in` ebenfalls, um einen physischen Ort zu zeigen: der Drucker befindet sich im Büro.

Andere Sätze verwenden `in` für Zeit, abstrakten Ort oder Interesse, was unterschiedliche Verwendungen sind.
