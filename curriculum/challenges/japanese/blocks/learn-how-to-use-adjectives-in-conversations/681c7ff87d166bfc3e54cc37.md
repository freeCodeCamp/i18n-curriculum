# --explanation--

`Highlights`は、特に重要なものとして注意を引きつけることを意味します。例えば：

`The report highlights key trends in the market.` – これはレポートが最も重要なパターンにフォーカスしていることを意味します。

`Most important`は、グループの中で最も重要なものや価値の高いものを指します。例えば：

`The most important factor is safety.` – これは安全が他よりも重要であることを意味します。

`Right away`は、何かがすぐに起こることを意味します。例えば：

`She answered the email right away.` – これは彼女が待たずに返事をしたことを意味します。
