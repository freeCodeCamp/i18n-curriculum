# --description--

単語 `so` は、原因と結果を示したり、何かの理由を説明したりするための接続詞です。技術やプログラミングの分野では、`so` は問題や状況と解決策や結果をつなぐことがよくあります。

# --questions--

## --text--

技術的な文脈で原因と結果を正しくつなぐために `so` が使われている文を選んでください。

## --answers--

`The program crashed, so we must reboot the computer.`

---

`We wrote the code, so but it didn't compile.`

### --feedback--

ここでの `so` の使い方は間違っています。`But` は `so` の後に使うべきではありません。`The code didn't compile, so we rewrote it` または `We wrote the code, but it didn't compile.` のどちらかを使うべきです。

---

`The server is online, so therefore we can't connect.`

### --feedback--

`so` と `therefore` を一緒に使うのは冗長です。`The server is online, so we can connect` または `The server is online; therefore, we can connect.` のどちらかにすべきです。

---

`We need more data, so we can ignore the results.`

### --feedback--

この文は原因と結果を論理的につなげていません。`so` を使うなら、`We need more data, so we will perform more tests.` のようにするのがよいでしょう。
