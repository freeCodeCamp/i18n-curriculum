## Mti wa Ufikikaji

Mti wa ufikikaji ni muundo unaotumika na teknolojia za kusaidia, kama vile kisomaji cha skrini, kutafsiri na kuingiliana na maudhui kwenye ukurasa wa mtandao.

## Kazi ya `max()`

Kazi ya `max()` inarudisha kubwa zaidi kati ya seti ya thamani zilizoandaliwa kwa koma:

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <img src="https://cdn.freecodecamp.org/curriculum/css-photo-gallery/9.jpg" alt="A cat sitting on a table">
</body>
```

```css
img {
  width: max(250px, 25vw);
}
```

:::

Katika mfano hapo juu, upana wa picha utakuwa 250px ikiwa upana wa eneo la kuona ni chini ya pikseli 1000. Ikiwa upana wa eneo la kuona ni zaidi ya pikseli 1000, upana wa picha utakuwa 25vw. Hii ni kwa sababu 25vw ni sawa na asilimia 25 ya upana wa eneo la kuona.

## Mbinu Bora na CSS na Ufikikaji

- **`display: none;`**: Kutumia `display: none;` kunamaanisha kwamba visomaji vya skrini na teknolojia nyingine za kusaidia hazitaweza kupata upatikanaji wa maudhui haya, kwa sababu hayajumuishwi katika mti wa ufikikaji. Kwa hiyo, ni muhimu kutumia njia hii tu wakati unataka kuondoa kabisa maudhui kutoka kwa uwasilishaji wa kuona na ufikikaji.
- **`visibility: hidden;`**: Sifa hii na thamani yake huficha maudhui kwa njia ya kuona lakini huiacha katika mtiririko wa hati, ikimaanisha bado inachukua nafasi kwenye ukurasa. Vitu hivi pia havitatambuliwa tena na visomaji vya skrini kwa sababu vitakuwa vimeondolewa kutoka kwenye mti wa ufikikaji.
- **`.sr-only` darasa la vitu la CSS**: Hii ni mbinu ya kawaida inayotumika kuficha maudhui kwa njia ya kuona huku ikiyafanya yaweze kupatikana kwa visomaji vya skrini.

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <span>This span element can be read</span>
  <span class="sr-only">This span element can only be read by screen readers</span>
  <p style="display: none">
    This text is hidden with display: none and is NOT accessible to screen readers.
  </p>
  <p style="visibility: hidden">
    This text is hidden with visibility: hidden and is NOT read by screen readers,
    but it still takes up space.
  </p>
</body>

```

```css
.sr-only {
  position: absolute;
  width: 1px;
  height: 1px;
  padding: 0;
  margin: -1px;
  overflow: hidden;
  clip: rect(0, 0, 0, 0);
  white-space: nowrap;
  border: 0;
}

```

:::

- **`scroll-behavior: smooth;`**: Sifa hii na thamani huruhusu tabia ya kusogeza kwa laini.
- **Sifa ya `prefers-reduced-motion`**: Hii ni sifa ya media inayoweza kutumika kugundua upendeleo wa mtumiaji wa michoro ya mzunguko.

## Kuficha Maudhui kwa Sifa za HTML

- **Sifa ya `aria-hidden`**: Inatumika kuficha kipengele kutoka kwa watu wanaotumia teknolojia saidizi kama wasomaji wa skrini. Kwa mfano, hii inaweza kutumika kuficha picha za mapambo ambazo hazitoi maudhui yoyote yenye maana.
- **Sifa ya `hidden`**: Sifa hii inaungwa mkono na vivinjari vingi vya kisasa na huficha maudhui kwa mtazamo wa kuona na kutoka kwenye mti wa ufikikaji. Inaweza kubadilishwa kwa urahisi kwa kutumia JavaScript.

:::interactive_editor

```html
<p aria-hidden="true">This paragraph is visible for sighted users, but is hidden from assistive technology.</p>
<p hidden>This paragraph is hidden from both sighted users and assistive technology.</p>
<p>This is a normal paragraph</p>

```

:::

## Tatizo la Ufikikaji la Sifa ya `placeholder`

Kutumia maandishi ya kiwakilishi si nzuri kwa ajili ya ufikikaji. Mara nyingi sana, watumizi huchanganya maandishi ya kiwakilishi na thamani halisi ya ingizo, na hufikiri tayari kuna thamani katika ingizo hilo.
