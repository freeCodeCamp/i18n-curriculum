# --interactive--

Außenabstand und Innenabstand sind wichtige Eigenschaften in CSS, um gut strukturierte, lesbare und optisch ansprechende Webseiten zu erstellen.

Außenabstände steuern den Raum außerhalb eines Elements, helfen dabei, es von anderen Elementen zu trennen und die Layoutstruktur zu definieren, während Innenabstände den Raum innerhalb eines Elements steuern und so die Lesbarkeit des Inhalts und die ästhetische Wirkung verbessern.

Um die Unterschiede zwischen `margin` und `padding` besser zu verstehen, schauen wir uns einige Beispiele an.

Hier ist ein HTML-Beispiel mit drei Absatz-Elementen auf der Seite:

:::interactive_editor

```html
<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

:::

Um den oberen Abstand jedes Absatz-Elements zu gestalten, können Sie die `margin-top`-Eigenschaft wie folgt verwenden:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  margin-top: 20px;
  border: 2px solid black;
}
```

:::

In diesem Beispiel wenden wir `20px` Außenabstand nur oben auf jedes Absatz-Element an.

Wir haben außerdem einen durchgehenden schwarzen Rahmen an allen vier Seiten, damit Sie den Außenabstand besser sehen können.

Die vier verschiedenen `margin`-Eigenschaften sind `margin-top`, `margin-right`, `margin-bottom` und `margin-left`.

Hier ist ein Beispiel, das alle vier Eigenschaften verwendet:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 30px;
  margin-left: 40px;
  border: 2px solid black;
}
```

:::

In diesem Beispiel weisen wir Abstandswerte an allen vier Seiten des Absatz-Elements zu. Ein durchgehender schwarzer Rahmen wurde ebenfalls hinzugefügt, damit Sie die Außenabstände besser sehen können.

Eine andere Möglichkeit, die `margin`-Eigenschaft zu verwenden, ist die Kurzschreibweise. Sie können ein, zwei, drei oder vier Werte gleichzeitig angeben. Lassen Sie uns diese Optionen gemeinsam erkunden.

Wenn Sie einen einzelnen Wert bei der `margin`-Kurzschreibweise verwenden, wird genau dieser Wert auf alle vier Seiten des Ziel-Elements angewendet.

Hier ist ein Beispiel für die Verwendung eines einzelnen Werts bei der `margin`-Kurzschreibweise:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px;
}
```

:::

Dieses Codebeispiel wendet `10px` von `margin` gleichmäßig auf alle vier Seiten des Absatz-Elements an.

Wenn Sie zwei Werte verwenden, gilt der erste Wert für die `top` und `bottom`, während der zweite Wert für die `left` und `right` Seiten des Elements gilt.

Hier ist ein Beispiel für die Verwendung von zwei Werten bei der `margin`-Kurzschreibweise:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px;
}
```

:::

Dies setzt die `top` und `bottom` Außenabstände auf `10px` und `20px` für die `left` und `right` Außenabstände des Absatz-Elements.

Wenn drei Werte angegeben werden, gilt der erste Wert für den `top` Außenabstand, der zweite Wert für den `left` und `right` Außenabstand, und der dritte Wert für den `bottom` Außenabstand.

Hier ist ein Beispiel zum besseren Verständnis:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px;
}
```

:::

Dies setzt den Außenabstand auf `10px` für den `top`, `20px` für den `left` und `right` sowie `30px` für den `bottom`.

Wenn Sie vier Werte verwenden, haben Sie mehr Kontrolle, da Sie jeden Außenabstandswert für jede Seite des Ziel-Elements unabhängig angeben können.

Der erste Wert gilt für den `top`, der zweite Wert für den `right`, der dritte Wert für den `bottom` und der vierte Wert für den `left`.

Hier ist ein Beispiel für die Verwendung der Kurzschreibweise für Außenabstand mit vier Werten:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px 40px;
}
```

:::

Dies setzt den Außenabstand auf `10px` für den `top`, `20px` für den `right`, `30px` für den `bottom` und `40px` für den `left`.

Die `padding`-Eigenschaft wird verwendet, um den Raum innerhalb des Elements zwischen dem Inhalt und seinem Rahmen zu gestalten.

Wie bei der `margin`-Eigenschaft sind die vier `padding`-Eigenschaften `padding-top`, `padding-right`, `padding-bottom` und `padding-left`.

Hier ist ein Beispiel, wie Sie den `padding` für ein Absatz-Element festlegen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 30px;
  padding-left: 40px;
  border: 2px solid black;
}
```

:::

Dies setzt den Innenabstand auf `10px` für den `top`, `20px` für den `right`, `30px` für den `bottom` und `40px` für den `left`.

Wie Sie sehen, wird `padding` auf den Inhalt angewendet, der sich innerhalb des Rahmens befindet, im Gegensatz zu `margin`, das außerhalb des Rahmens gilt.

Genau wie bei der `margin`-Eigenschaft können Sie auch die Kurzschreibweise für die `padding`-Eigenschaft verwenden.

Sie können auch bei der `padding`-Kurzschreibweise ein, zwei, drei oder vier Werte angeben.

Hier ist ein Beispiel für die Verwendung der `padding`-Kurzschreibweise für das zuvor gezeigte Absatz-Element:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding: 10px 20px 30px 40px;
  border: 2px solid black;
}
```

:::

Im Beispiel setzt der Code mit der Kurzschreibweise den Innenabstand auf `10px` für den `top`, `20px` für den `right`, `30px` für den `bottom` und `40px` für den `left` des Absatz-Elements.
