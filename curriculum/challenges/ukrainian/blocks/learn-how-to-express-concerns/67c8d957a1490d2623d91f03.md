# --explanation--

`To take feedback to someone` означає передати коментарі або пропозиції іншій людині, зазвичай тій, хто приймає рішення. Наприклад:

`I'll take your feedback to the manager so we can discuss possible changes.` - Це означає, що мовник поділиться відгуком зі своїм менеджером.
