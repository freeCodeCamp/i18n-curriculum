## --text--

Wie beschreibt Lisa `serverless computing`?

## --answers--

Es ist eine günstigere Art, Computer zu nutzen.

### --feedback--

Obwohl `serverless computing` kosteneffektiv sein kann, konzentriert sich Lisas Beschreibung auf den operativen Aspekt, nicht direkt auf die Kosten.

---

Es geht darum, mehr Server in der Datenverarbeitung zu verwenden.

### --feedback--

`Serverless computing` bedeutet, dass weniger oder gar keine sichtbaren Server für den Entwickler verwendet werden.

---

Es bedeutet Programmieren ohne Computer.

### --feedback--

`Serverless computing` beinhaltet weiterhin Computer, speziell Server, die vom Anbieter verwaltet werden, nicht vom Programmierer.

---

Es ist wie Computerhelfer zu haben, die Code ausführen, ohne Server verwalten zu müssen.
