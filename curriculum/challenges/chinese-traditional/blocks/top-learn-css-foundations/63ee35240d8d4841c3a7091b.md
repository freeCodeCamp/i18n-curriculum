# --description--

類別選擇器會選取所有具有指定 `class` 的元素，這只是你放在 HTML 元素上的一個屬性。以下是如何為 HTML 頁籤添加類別並在 CSS 中選取它：

```html
<!-- index.html -->

<div class="alert-text">
  Please agree to our terms of service.
</div>
```

```css
/* styles.css */

.alert-text {
  color: red;
}
```

請注意 `class` 選擇器的語法：一個句點緊接著類別屬性的大小寫敏感值。類別不需要是唯一的，所以你可以在任意多個元素上使用相同的 `class`。

你還可以使用 `class` 屬性為單一元素添加多個類別，這些類別以空格分隔的列表形式呈現，例如 `class="alert-text severe-alert"`。由於空白字元用來分隔 `class` 名稱，因此你絕不應該在多字名稱中使用空格，而應該改用連字號。

## ID 選取器
ID 選擇器類似於 `class` 選擇器。它們會選擇具有指定 `id` 的元素，`id` 是你放在 HTML 元素上的另一個屬性：

```html
<!-- index.html -->

<div id="title">My Awesome 90's Page</div>
```

```css
/* styles.css */

#title {
  background-color: red;
}
```

你不是使用句點，而是使用井字號，緊接著大小寫敏感的 `id` 屬性值。常見的陷阱是人們在不必要時過度使用 `id` 屬性，而使用類別就足夠了。雖然有些情況下使用 `id` 是合理或必要的，例如利用特異性或讓連結重新導向到當前頁面的某個區段，但你應該謹慎（如果有的話）使用 `id`。

類別和 ID 之間的主要差異在於一個元素只能有一個 `id`。在單一頁面中，`id` 不可重複，且 `id` 屬性中不應包含任何空白字元。
