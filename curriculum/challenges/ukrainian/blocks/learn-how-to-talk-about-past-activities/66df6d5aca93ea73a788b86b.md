# --explanation--

`Roll out` означає представити або запустити щось нове, наприклад продукт, сервіс або оновлення, для публіки чи користувачів. Наприклад:

`The company will roll out the new software next week.` - They will release the new software next week.

Якщо Лінда думає про оновлення наступного місяця, вона могла б використати `roll out`, щоб говорити про його випуск.
