# --description--

Puoi usare la variabile `__name__` per determinare se uno script Python viene eseguito come programma principale oppure se viene importato come modulo (codice scritto in un altro file Python).

Se il valore di `__name__` è impostato su `'__main__'`, significa che lo script corrente è il programma principale e non un modulo.

In questo progetto, userai lo script corrente come programma principale.

Crea un’istruzione `if` che verifichi se il valore di `__name__` è `'__main__'`.

Usa la parola chiave `pass` nel corpo dell’istruzione `if`.
