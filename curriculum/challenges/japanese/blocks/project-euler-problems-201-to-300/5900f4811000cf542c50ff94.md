# --description--

修正された整数のコラッツシーケンスは、次の方法で開始値 $a_1$ から得られます。

$a_n$ が3で割り切れる場合、$a_{n + 1} = \frac{a_n}{3}$ となります。これは大きな下向きのステップ「D」と表します。

$a_n$ を3で割った余りが1の場合、$a_{n + 1} = \frac{4a_n + 2}{3}$ となります。これは上向きのステップ「U」と表します。

$a_n$ を3で割った余りが2の場合、$a_{n + 1} = \frac{2a_n - 1}{3}$ となります。これは小さな下向きのステップ「d」と表します。

シーケンスは、ある $a_n = 1$ となった時点で終了します。

任意の整数が与えられたとき、そのステップのシーケンスを列挙できます。例えば、$a_1 = 231$ の場合、シーケンス $\\{a_n\\} = \\{231, 77, 51, 17, 11, 7, 10, 14, 9, 3, 1\\}$ はステップ「DdDddUUdDD」に対応します。

もちろん、同じステップシーケンス「DdDddUUdDD...」で始まる他のシーケンスも存在します。

例えば、$a_1 = 1004064$ の場合、シーケンスは DdDddUUdDDDdUDUUUdDdUUDDDUdDD となります。

実際、1004064 は $10^6$ より大きい $a_1$ の中で、シーケンス DdDddUUdDD で始まる最小の値です。

シーケンス "UDDDUdddDDUDDddDdDddDDUDDdUUDd" で始まる $10^{15}$ より大きい最小の $a_1$ は何でしょうか？
