# --description--

Dada uma grade 2D, uma posição inicial (`[row, col]`) e um novo valor, substitua o valor na posição inicial e todas as células conectadas com o mesmo valor pelo novo valor.

- As células estão conectadas se forem adjacentes horizontalmente ou verticalmente (não diagonalmente).

Retorne a grade atualizada.

```py
def bucket_fill(grid, pos, new_value):
    row, col = pos
    target = grid[row][col]
    if target == new_value:
        return grid

    def fill(r, c):
        if r < 0 or r >= len(grid):
            return
        if c < 0 or c >= len(grid[0]):
            return
        if grid[r][c] != target:
            return
        grid[r][c] = new_value
        fill(r - 1, c)
        fill(r + 1, c)
        fill(r, c - 1)
        fill(r, c + 1)

    fill(row, col)
    return grid
```
