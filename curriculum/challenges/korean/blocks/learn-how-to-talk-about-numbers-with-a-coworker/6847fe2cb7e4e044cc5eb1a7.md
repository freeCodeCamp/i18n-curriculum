# --explanation--

⅗를 단어로 올바르게 쓰는 방법은 `three-fifths`입니다. 다음 형식을 사용하세요:

분자(기수 형태) + 분모(분자가 1보다 크면 복수형). 예를 들어:

- ⅖ = `two-fifths`

- ⅘ = `four-fifths`

이는 데이터를 설명하거나 재무 정보를 구두 발표할 때 유용합니다.
