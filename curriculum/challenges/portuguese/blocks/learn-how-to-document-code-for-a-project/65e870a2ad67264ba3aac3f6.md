# --description--

Nesta tarefa, você aprenderá a diferença entre `simple` e `simply` e entenderá o que `double` e `slashes` significam.

`Simple` é um adjetivo que significa fácil ou não complicado, como `a simple task`. `Simply` é um advérbio, que significa de maneira simples, como `doing something simply`.

`Double` significa dois de algo. E `slashes` são os símbolos `/` usados na escrita.  You add `es` to make `slash` plural because words ending in `sh` form their plurals by adding `es`, like in `dishes` or `brushes`.
