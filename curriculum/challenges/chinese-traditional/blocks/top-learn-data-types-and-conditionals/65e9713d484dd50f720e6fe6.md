# --description--

根據你正在進行的工作類型，你可能會更多地處理文字片段而非數字。**string** 是一段文字……並且是語言的基本建造區塊。

HTML 為文字提供結構和意義，CSS 讓我們能精確地為它設計樣式，而 JavaScript 則提供許多用於操作字串的功能。這些功能包括建立訂製的歡迎訊息和提示、在需要時顯示正確的文字標籤、將詞彙排序成所需的順序，以及更多其他功能。

字串是 JavaScript 中的基本資料型別。它們用來表述文字，並且包裹在單引號、雙引號或反引號中：

```javascript
let greeting = "Hello World!";

let greeting2 = 'I am learning JavaScript!';

let greeting3 = `with The Odin Project!`;
```

使用單引號宣告的字串和使用雙引號宣告的字串是相同的，使用哪一種取決於個人偏好——儘管在你的程式碼中選擇一種風格並持續使用是良好的做法。
