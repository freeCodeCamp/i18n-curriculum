# --description--

Para ser reconhecido como um método abstrato, um método deve ser decorado com o decorador `@abstractmethod`.

Um <dfn>decorator</dfn> é usado em Python para modificar o comportamento de uma função.Aqui está um exemplo de como usar um decorator chamado `spam`:

```py
@spam
def foo():
    pass
```

Modifique sua declaração de importação para importar o decorador `abstractmethod` e decorar os métodos `solve` e `analyze` da classe `Equation`.Isso gerará duas exceções.

Uma vez que uma classe que herda de `ABC` tenha um método abstrato, a classe não pode mais ser instanciada.Portanto, exclua a instância `Equation` para se livrar do erro.

O outro erro ocorre porque a classe `LinearEquation` deve implementar todos os métodos abstratos definidos na interface.Certifique-se de defini-los dentro da classe `LinearEquation`, também.Você não deve usar o decorator `abstractmethod` na classe concreta.
