# --description--

このチャレンジでは、以下のいずれかの方法でコードを書いて進めます。

- <a href="https://github.com/freeCodeCamp/boilerplate-mongomongoose/" target="_blank" rel="noopener noreferrer nofollow">このGitHubリポジトリ</a>をクローンして、ローカルでチャレンジを完了する。
- お好みのサイトビルダーを使ってプロジェクトを完成させる。この場合、GitHubリポジトリのすべてのファイルを必ず取り込んでください。

このチャレンジでは、MongoDB Atlasのデータベースをセットアップし、それに接続するために必要なパッケージをインポートします。

MongoDB Atlasでホストされたデータベースをセットアップするには、<a href='https://www.freecodecamp.org/news/get-started-with-mongodb-atlas/' target="_blank" rel="noopener noreferrer nofollow">こちらのチュートリアル</a>に従ってください。

# --instructions--

`mongoose@^5.11.15` はプロジェクトの `package.json` ファイルに追加されています。まず、`mongoose` 内で mongoose を `myApp.js` として require してください。次に、`.env` ファイルを作成し、そこに `MONGO_URI` 変数を追加します。その値はMongoDB AtlasのデータベースURIにしてください。URIはシングルクォートか二重引用符で囲み、環境変数の`=`の前後にスペースを入れないように注意してください。例えば、`MONGO_URI='VALUE'` のようになります。

完了したら、次の構文を使って `connect` ファイル内で `myApp.js` メソッドを呼び出してデータベースに接続してください。

```js
mongoose.connect(<Your URI>, { useNewUrlParser: true, useUnifiedTopology: true });
```
