## Framework CSS

- **Framework CSS**: I framework CSS possono velocizzare il tuo flusso di lavoro, creare uno stile visivo uniforme su un sito web, far apparire il design coerente su più browser e mantenere il tuo codice CSS più organizzato.
- **Framework CSS popolari**: Alcuni dei framework CSS più popolari sono Tailwind CSS, Bootstrap, Materialize e Foundation.
- **Possibili svantaggi**:
  - Il CSS fornito dal framework potrebbe entrare in conflitto con il tuo CSS personalizzato.
  - Il tuo sito web potrebbe sembrare simile ad altri siti che usano lo stesso framework.
  - I framework di grandi dimensioni potrebbero causare problemi di prestazioni.

## Due tipi di framework CSS

- **Framework CSS utility-first**: Questi framework hanno piccole classi con scopi specifici, come impostare margine, spaziatura interna o colore di sfondo. Puoi assegnare queste piccole classi direttamente agli elementi HTML secondo necessità. Tailwind CSS è classificato come un framework CSS utility-first.

Ecco un esempio di utilizzo di Tailwind CSS per definire lo stile di un pulsante.

```html
<button class="bg-blue-500 text-white font-bold py-2 px-4 rounded-full hover:bg-blue-700">
  Button
</button>
```

- **Framework CSS basati su componenti**: Questi framework hanno componenti predefiniti con stili preimpostati che puoi aggiungere al tuo sito web. I componenti sono disponibili nella documentazione ufficiale del framework CSS e puoi copiarli e incollarli nel tuo progetto. Bootstrap è classificato come un framework CSS basato su componenti.

Ecco un esempio di utilizzo di Bootstrap per creare un gruppo di liste. Invece di applicare piccole classi agli elementi HTML, aggiungerai l’intero componente, inclusa la struttura HTML.

```html
<div class="card" style="width: 25rem;">
  <ul class="list-group list-group-flush">
    <li class="list-group-item">HTML</li>
    <li class="list-group-item">CSS</li>
    <li class="list-group-item">JavaScript</li>
  </ul>
</div>
```

## Tailwind CSS

Tailwind è un framework CSS utility-first. Invece di scrivere regole CSS personalizzate, costruisci i tuoi design combinando piccole classi utility direttamente nel tuo HTML.

### Utility per il design responsivo

Tailwind usa prefissi come `sm:`, `md:` e `lg:` per applicare gli stili a diverse dimensioni dello schermo.

```html
<div class="w-full md:w-1/2 lg:flex-row">Responsive layout</div>
```

### Utility Flexbox

Classi come `flex`, `flex-col`, `justify-around` e `items-center` facilitano la creazione di layout flessibili.

```html
<div class="flex flex-col md:flex-row justify-around items-center">
  <p>Column on small screens</p>
  <p>Row on medium and larger screens</p>
</div>
```

### Utility Grid

Tailwind include utility per CSS Grid, come `grid`, `grid-cols-1` e `md:grid-cols-3`.

```html
<div class="grid grid-cols-1 md:grid-cols-3 gap-8">
  <div class="bg-gray-100 p-4">Column 1</div>
  <div class="bg-gray-100 p-4">Column 2</div>
  <div class="bg-gray-100 p-4">Column 3</div>
</div>
```

### Utility per la spaziatura

Utility come `mt-8`, `mx-auto`, `p-4` e `gap-4` aiutano a creare spaziature coerenti senza scrivere CSS.

```html
<div class="mt-8 p-4 bg-indigo-600 text-white">Spaced content</div>
```

### Utility per la tipografia

Utility come `uppercase`, `font-bold`, `font-semibold` e `text-4xl` controllano l’aspetto del testo.

Puoi impostare dimensioni del carattere tipografico che si adattano ai breakpoint, come `text-3xl` `md:text-5xl`.

```html
<h1 class="text-3xl md:text-5xl font-semibold text-center">Responsive Heading
</h1>
```

### Colori e stati hover

Tailwind offre una vasta palette di colori, come `text-red-700`, `bg-indigo-600` e `bg-gray-100`.

Classi come `hover:bg-pink-600` rendono semplici gli effetti interattivi.

```html
<a href="#" class="bg-pink-500 hover:bg-pink-600 text-white px-4 py-2 rounded-md">
  Hover Me
</a>
```

### Bordi, anelli ed effetti

- **Bordi**: `border-2 border-red-300` aggiunge bordi con spessore e colori specificati.
- **Anelli**: `ring-1 ring-gray-300` crea effetti simili a contorni spesso usati per il focus o le card.
- **Angoli arrotondati e scaling**: Classi come `rounded-md`, `rounded-xl` e `scale-105` aggiungono rifiniture.

```html
<div class="p-6 rounded-xl ring-2 ring-fuchsia-500 scale-105">
  Highlighted card
</div>
```

### Gradienti

Tailwind supporta utility per i gradienti come `bg-gradient-to-r from-fuchsia-500 to-indigo-600`.

```html
<div class="p-4 text-white bg-gradient-to-r from-fuchsia-500 to-indigo-600">
  Gradient background
</div>
```

## Preprocessori CSS

- **Preprocessore CSS**: Un preprocessore CSS è uno strumento che estende il CSS standard. Compila il codice con sintassi estesa in un file CSS nativo. Può essere utile per scrivere CSS più pulito, riutilizzabile, meno ripetitivo e scalabile per progetti complessi.
- **Funzionalità**: Alcune delle funzionalità fornite dai preprocessori CSS sono variabili, mixin, annidamento e ereditarietà dei selettori.
- **Preprocessori CSS popolari**: Alcuni preprocessori CSS popolari sono Sass, Less e Stylus.  
- **Possibili svantaggi**:
  - Compilare le regole CSS in CSS standard potrebbe causare un sovraccarico.
  - Il codice compilato potrebbe essere difficile da eseguire il debug.

## Sass

- **Sass**: È uno dei preprocessori CSS più popolari. Sass sta per "Syntactically Awesome Style Sheets".
- **Funzionalità supportate da Sass**: Sass supporta funzionalità come variabili, regole CSS annidate, moduli, mixin, ereditarietà e operatori per operazioni matematiche di base.

## Due sintassi supportate da Sass

- **Sintassi SCSS**: La sintassi SCSS (Sassy CSS) espande la sintassi base del CSS. È la sintassi più usata per Sass. I file SCSS hanno un’estensione `.scss`.

Ecco un esempio di definizione e utilizzo di una variabile in SCSS.

```scss
$primary-color: #3498eb;

header {
  background-color: $primary-color;
}
```

- **Sintassi indentata**: La sintassi indentata era la sintassi originale di Sass ed è anche conosciuta come "sintassi Sass".

Ecco un esempio di definizione e utilizzo di una variabile nella sintassi indentata.

```sass
$primary-color: #3498eb

header
  background-color: $primary-color
```

### Mixin

- **Mixin**: I mixin ti permettono di raggruppare più proprietà CSS e i loro valori sotto un nome e riutilizzare quel blocco di codice CSS in tutto il foglio di stile.

Ecco un esempio di definizione di un mixin in sintassi SCSS. In questo caso, il mixin si chiama `center-flex`. Ha tre proprietà CSS per centrare gli elementi usando flexbox.

```scss
@mixin center-flex {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

Ecco un esempio di utilizzo del mixin che hai definito.

```scss
section {
  @include center-flex;
  height: 500px;
  background-color: #3289a8;
}
```
