# --description--

$k$ 桁の正の整数は、その最初の $⌈\frac{k}{2}⌉$ 桁の合計が最後の $⌈\frac{k}{2}⌉$ 桁の合計と等しい場合、バランスの取れた数と呼ばれます。ここで $⌈x⌉$（シーリング）は $x$ 以上の最小の整数を意味し、例えば $⌈π⌉ = 4$、$⌈5⌉ = 5$ です。

例えば、すべての回文数はバランスの取れた数であり、13722 もそうです。

$T(n)$ を $10^n$ 未満のすべてのバランスの取れた数の合計とします。

したがって、$T(1) = 45$、$T(2) = 540$、$T(5) = 334\,795\,890$ です。

$T(47) \bmod 3^{15}$ を求めてください。

# --hints--

`balancedNumbers()` は `6273134` を返すべきです。
