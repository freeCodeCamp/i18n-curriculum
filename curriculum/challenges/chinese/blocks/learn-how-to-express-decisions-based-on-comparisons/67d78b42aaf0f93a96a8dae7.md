# --explanation--

`That's true` 是一个用来同意某人语句或承认他们所说内容正确的短语。例如：

`That's true, we do need more time to finish the project.` – 这意味着同意需要更多时间。  

`Cost-effective` 意味着某物在花费的资金上提供良好的价值或帮助节省成本，同时仍能取得良好的结果。例如：

`Using cloud services can be more cost-effective than buying expensive servers.` – 这意味着云服务可以在仍然有用的同时节省资金。
