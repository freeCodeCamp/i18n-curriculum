# --interactive--

CSS, che sta per Cascading Style Sheets, è un componente fondamentale dello sviluppo web moderno. È un linguaggio di markup usato per applicare stili all’HTML. In termini più semplici, se l’HTML è la struttura di una pagina web, il CSS è ciò che la rende bella da vedere.

Ecco un esempio di come stilizzare una barra di navigazione.

**NOTA**: Non preoccuparti di capire subito il codice CSS. Imparerai come funziona tutto questo nelle lezioni e nei workshop futuri. Questo serve solo a darti un’idea di cosa puoi fare con un po’ di CSS.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<nav class="navbar">
    <ul class="nav-links">
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Services</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>
```

```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: 'Segoe UI', sans-serif;
  background-color: #f4f4f4;
}

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  background-color: #333;
  padding: 1rem 2rem;
  color: white;
}

.navbar .logo {
  font-size: 1.5rem;
  font-weight: bold;
}

.nav-links {
  list-style: none;
  display: flex;
  gap: 1.5rem;
}

.nav-links a {
  color: white;
  text-decoration: none;
  font-weight: 500;
  transition: color 0.3s ease;
}

.nav-links a:hover {
  color: #ff9800;
}
```

:::

Il ruolo principale del CSS è separare il contenuto di una pagina web dalla sua presentazione visiva. Questa separazione permette agli sviluppatori web di creare siti più flessibili e facili da mantenere.

Con il CSS puoi controllare il layout, i colori, i caratteri tipografici e l’aspetto visivo generale delle pagine web senza modificare la struttura HTML.

Facciamo un’analogia semplice. Se pensi a un sito web come a una casa, l’HTML sarebbe le fondamenta e la struttura, mentre il CSS sarebbe la pittura, la carta da parati e le decorazioni che rendono la casa visivamente attraente e unica.

Il CSS funziona selezionando gli elementi HTML e applicando loro degli stili.

Ecco un esempio che mostra un elemento `button` senza stile e uno stilizzato. Per interagire con questo esempio, devi abilitare l’editor interattivo.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<button class="regular-btn">Unstyled Button</button>
<button class="round-btn">Round Button</button>
```

```css

.round-btn {
  font-size: 1rem;
  font-family: 'Segoe UI', sans-serif;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease, transform 0.2s ease;
  background-color: #2ecc71;
  color: white;
  border-radius: 50px;
  padding: 0.6rem 1.6rem;
}

.round-btn:hover {
  background-color: #27ae60;
  transform: translateY(-2px);
}
```

:::

Questi stili possono includere proprietà come il colore, la dimensione del carattere e molte altre. Cambiando queste proprietà, puoi modificare drasticamente l’aspetto di una pagina web senza cambiare il suo contenuto.

Uno degli aspetti più potenti del CSS è la sua capacità di creare design responsivi.

Questo significa che con il CSS puoi far sì che il tuo sito web abbia un aspetto ottimale su qualsiasi dispositivo, che sia un computer desktop, un tablet o uno smartphone.

Il CSS ti permette di regolare i layout, le dimensioni dei caratteri e altri elementi visivi in base alla dimensione dello schermo del dispositivo che visualizza il sito.

Prova ad abilitare l’editor interattivo e a modificare la dimensione della finestra di anteprima per vedere come il layout si adatta alla dimensione dello schermo.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<header class="header">
    <h1>Welcome to My Responsive Site</h1>
    <p>This layout adapts to your screen size!</p>
</header>

<main class="content">
    <section class="box">Box 1</section>
    <section class="box">Box 2</section>
    <section class="box">Box 3</section>
</main>
```

```css
body {
  margin: 0;
  font-family: 'Segoe UI', sans-serif;
  background-color: #f9f9f9;
  color: #333;
}

.header {
  background-color: #4CAF50;
  color: white;
  padding: 2rem;
  text-align: center;
}

.content {
  display: flex;
  gap: 1rem;
  padding: 1rem;
  justify-content: center;
}

.box {
  flex: 1;
  min-width: 200px;
  background-color: #ddd;
  padding: 2rem;
  text-align: center;
  font-size: 1.2rem;
  border-radius: 8px;
  transition: background-color 0.3s;
}

.box:hover {
  background-color: #ccc;
}

/* Responsive: for tablets and below (≤ 768px) */
@media (max-width: 768px) {
  .content {
    flex-direction: column;
    align-items: center;
  }

  .box {
    width: 90%;
    font-size: 1.1rem;
  }
}

/* Responsive: for phones (≤ 480px) */
@media (max-width: 480px) {
  .header {
    padding: 1.5rem 1rem;
  }

  .header h1 {
    font-size: 1.5rem;
  }

  .header p {
    font-size: 1rem;
  }

  .box {
    font-size: 1rem;
    padding: 1.5rem;
  }
}
```

:::

Un’altra funzionalità importante del CSS è la sua natura a cascata, da cui deriva il termine "cascading" nel nome.

Questo significa che gli stili possono essere ereditati e sovrascritti, permettendo una struttura gerarchica di definizione degli stili. Imparerai di più su come funziona questo nelle lezioni future.

Il CSS supporta anche l’uso di fogli di stile esterni.

Questo significa che puoi mantenere tutte le regole di stile in un file separato, che poi può essere collegato a più pagine HTML.

Questa funzionalità migliora molto la manutenibilità dei siti, specialmente quelli più grandi. Invece di dover cambiare gli stili in ogni singola pagina, puoi fare modifiche in un solo file CSS che influenzeranno tutte le pagine collegate.

Nel mondo dello sviluppo web, il CSS ha un ruolo vitale nel creare siti visivamente attraenti, responsivi e facili da usare.

Permette agli sviluppatori di trasformare semplici documenti HTML in esperienze web coinvolgenti che catturano l’attenzione degli utenti e migliorano la loro interazione con i contenuti web.
