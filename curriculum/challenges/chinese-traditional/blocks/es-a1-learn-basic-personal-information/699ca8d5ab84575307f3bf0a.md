# --description--

人們根據情況以不同方式溝通。

你可以使用 `teléfono` 直接與某人通話，或使用 `mensaje de texto`，這是一種透過電話發送的簡短書面訊息。

你也可以使用 `correo`，它是指傳統郵件，例如寄到家庭位址的信件。如今，許多訊息是線上傳送的。在這些情況下，它被稱為 `correo electrónico`。

`Electrónico` 描述使用科技或電子裝置的事物。
