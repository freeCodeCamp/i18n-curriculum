# --description--

Sie können ein Dictionary durchlaufen, wenn Sie auf seine Schlüssel-Wert-Paare zugreifen und diese verarbeiten müssen. Das ist hilfreich, um deren Werte zu aktualisieren oder eine Logik darauf anzuwenden.

Werfen wir einen Blick auf einige Techniken, die Sie verwenden können.

Angenommen, wir haben ein `products`-Dictionary, das jedes Produkt mit seinem Preis verknüpft:

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,
    'Headphones': 70,
}
```

Wenn wir auf alle unsere Produkte einen Rabatt von 20 % anbieten wollen, können wir alle Schlüssel-Wert-Paare durchlaufen und die Preise ändern.

Die `.values()`-, `.keys()`- und `.items()`-Methoden sind für diese Techniken unerlässlich. Wir haben sie in einer vorherigen Lektion kurz behandelt.

Sie geben ein View-Objekt mit den Werten, Schlüsseln und Schlüssel-Wert-Paaren des Dictionaries zurück. Sie können diese View-Objekte in `for`-Schleifen verwenden, um die Elemente zu durchlaufen.

Zum Beispiel können Sie alle Werte des Dictionaries so durchlaufen.

Sie schreiben `for`, die Schleifenvariable (hier `price`), `products.values()`, um alle Werte des `products`-Dictionaries zu erhalten, einen Doppelpunkt und dann den Schleifenkörper, in dem Sie jede Logik auf die Werte anwenden können. In diesem Fall geben wir sie aus.

Die Schleifenvariable nimmt jeden einzelnen Wert an, jeweils einen pro Iteration:

```python
for price in products.values():
    print(price)
```

Und hier ist die Ausgabe. Wie Sie sehen, wird jeder Wert einzeln in der Konsole ausgegeben:

```md
990
600
250
70
```

Das funktioniert genauso für `.keys()`, wenn Sie über die Schlüssel eines Dictionaries iterieren müssen. Sie müssen nur direkt über `products.keys()` oder `products` iterieren und der Schleifenvariable einen beschreibenden Namen zuweisen:

```python
for product in products.keys():
    print(product)

# Or

for product in products:
    print(product)
```

Das ist die Ausgabe. Jeder Schlüssel wird einzeln in der Konsole ausgegeben:

```md
Laptop
Smartphone
Tablet
Headphones
```

Und das funktioniert genauso für Schlüssel-Wert-Paare, wenn Sie gleichzeitig über die Schlüssel und ihre entsprechenden Werte iterieren müssen. Sie müssen nur über `products.items()` iterieren:

```python
for product in products.items():
    print(product)
```

Das ist die Ausgabe. Jetzt erhalten Sie einzelne Tuple mit den Schlüsseln und ihren entsprechenden Werten:

```md
('Laptop', 990)
('Smartphone', 600)
('Tablet', 250)
('Headphones', 70)
```

Wenn Sie den Schlüssel und den Wert in separaten Schleifenvariablen speichern möchten, müssen Sie sie nur definieren und mit einem Komma trennen. Dann können Sie sie im Schleifenkörper verwenden.

Hier definieren wir eine `product`-Schleifenvariable und eine `price`-Schleifenvariable. Jede hält ihren entsprechenden Wert. Wichtig ist, sie in der Reihenfolge zu definieren – zuerst der Schlüssel, dann der Wert:

```python
for product, price in products.items():
    print(product, price)
```

Das ist die Ausgabe. Wir geben sie nebeneinander aus, aber Sie können diese Werte in Ihrem Code nach Bedarf verwenden.

```md
Laptop 990
Smartphone 600
Tablet 250
Headphones 70
```

Jetzt, wo Sie mehr darüber wissen, können wir zu unserem Anfangsbeispiel zurückkehren. Wenn wir einen Rabatt von 20 % anbieten wollen, würden wir jeden Preis mit `0.8` multiplizieren und ihn als Wert für den entsprechenden Produktschlüssel neu zuweisen.

Wir könnten das Ergebnis auch abrunden, wenn wir mit ganzen Zahlen arbeiten wollen:

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,
    'Headphones': 70,
}

for product, price in products.items():
    products[product] = round(price * 0.8)

print(products)
```

Wenn wir dann das Dictionary ausgeben, erhalten wir diese Schlüssel-Wert-Paare mit den rabattierten Preisen:

```python
{
    'Laptop': 792, 
    'Smartphone': 480, 
    'Tablet': 200, 
    'Headphones': 56
}
```

Und schließlich, wenn Sie über die Schlüssel-Wert-Paare iterieren müssen und dabei einen Zähler führen wollen, können Sie die `enumerate()`-Funktion aufrufen. Dieser Zähler fungiert im Grunde als eine Art „Index“ oder „Zählung“ für das Element innerhalb der Schleife.

Die Funktion gibt ein `enumerate`-Objekt zurück, das jedem Schlüssel-Wert-Paar eine ganze Zahl zuweist, wie einen Zähler. Sie können den Zähler bei jeder Zahl starten, standardmäßig beginnt er bei 0.

Hier iterieren wir über die Schlüssel des `products`-Dictionaries:

```python
for product in enumerate(products):
    print(product)
```

Aber die `enumerate()`-Funktion weist auch jedem Schlüssel eine ganze Zahl zu, sodass wir Tupel mit der Zahl und dem Schlüssel erhalten.

Hier ist die Ausgabe:

```md
(0, 'Laptop')
(1, 'Smartphone')
(2, 'Tablet')
(3, 'Headphones')
```

Wenn Sie möchten, können Sie diese Werte auch auf separate Schleifenvariablen zuweisen. Hier haben wir zwei Schleifenvariablen (`index` und `product`). Das ist das, was Sie häufig sehen und verwenden, wenn Sie mit `enumerate()` arbeiten:

```python
for index, product in enumerate(products):
    print(index, product)
```

Wenn Sie über die Werte iterieren müssen, können Sie `products` durch `products.values()` ersetzen:

```python
for price in enumerate(products.values()):
    print(price)
```

Die Ausgabe enthält den Index und den Preis in jedem Tupel:

```md
(0, 990)
(1, 600)
(2, 250)
(3, 70)
```

Sie können sie auch auf separate Schleifenvariablen zuweisen:

```python
for index, price in enumerate(products.values()):
    print(index, price)
```

Das ist die Ausgabe. Sie können sie in Ihrem Code nach Bedarf verwenden:

```md
0 990
1 600
2 250
3 70
```

Und mit `products.items()` erhalten Sie das gesamte Schlüssel-Wert-Paar zusätzlich zum „Index“ oder „Zähler“:

```python
for index, product in enumerate(products.items()):
    print(index, product)
```

In diesem Beispiel erhalten wir zuerst den Index, gefolgt von einem Tupel, das den Schlüssel und den Wert des entsprechenden Schlüssel-Wert-Paares enthält:

```md
0 ('Laptop', 990)
1 ('Smartphone', 600)
2 ('Tablet', 250)
3 ('Headphones', 70)
```

Um den Anfangswert der Zählung anzupassen, können Sie der `enumerate()`-Funktion ein zweites Argument übergeben. Zum Beispiel starten wir hier die Zählung bei 1:

```python
for index, product in enumerate(products.items(), 1):
    print(index, product)
```

Sie sehen diese Änderung in der Ausgabe. Jetzt ist die erste ganze Zahl 1 statt 0:

```md
1 ('Laptop', 990)
2 ('Smartphone', 600)
3 ('Tablet', 250)
4 ('Headphones', 70)
```

Das funktioniert mit jeder bisher gesehenen Variation. Sie müssen nur die Anfangszahl als zweites Argument übergeben.

Es gibt viele Techniken, um ein Dictionary zu durchlaufen. Dies sind einige gängige Methoden, und Sie müssen die beste für Ihr Projekt auswählen.
