# --explanation--

`Could` y `might` son verbos modales usados para hablar sobre posibilidades.

`Could` significa que algo es posible pero no seguro. Por ejemplo:

`This update could make the app run faster.` - La actualización tiene el potencial de mejorar la velocidad de la app.

`Might` también expresa posibilidad pero a menudo sugiere una probabilidad menor que `could`. Por ejemplo:

`The new feature might improve the user experience.` - Es posible que la característica mejore la experiencia.
