# --description--

令 $a_k$、$b_k$ 和 $c_k$ 表现表达式 $\frac{1}{x} = {\left(\frac{k}{x} \right)}^2 (k + x^2) - kx$ 的三个解（实数或复杂数）。

例如，对于 $k = 5$，我们看到 $\\{a_5, b_5, c_5\\}$ 大约是 $\\{5.727244, -0.363622 + 2.057397i, -0.363622 - 2.057397i\\}$。

令 $S(n) = \displaystyle\sum_{p = 1}^n \sum_{k = 1}^n {(a_k + b_k)}^p {(b_k + c_k)}^p {(c_k + a_k)}^p$，其中所有整数 $p$、$k$ 满足 $1 ≤ p, k ≤ n$。

有趣的是，$S(n)$ 总是一个整数。例如，$S(4) = 51\\,160$。

查找 $S({10}^6)$ \text{ 模数 } 1\\,000\\,000\\,007$。
