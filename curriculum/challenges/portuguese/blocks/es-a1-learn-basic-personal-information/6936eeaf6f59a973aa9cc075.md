- `Mi nombre es...` – Meu nome é...

- `Mi apellido es...` – Meu sobrenome é...

- `Mi edad es...` – Minha idade é...

- `Mi número es...` – Meu número é...

# --explanation--

A frase completa é `Mi correo es mateo.delgado@sanjuan.pr.`

Aqui, `correo` se refere a um endereço de e-mail.

A estrutura `Mi` + substantivo + `es...` é usada para fornecer informações pessoais. Por exemplo:

- `Mi nombre es Ana.` - Meu nome é Ana.

- `Mi número es 555-1234.` - Meu número é 555-1234.
