## 絶対単位

- **`px`（ピクセル）**: この絶対単位はCSSでの固定サイズの単位です。最も一般的な絶対単位で、寸法を正確に制御できます。`1px`は常に1/96インチに相当します。
- **`in`（インチ）**: この絶対単位は96pxに相当します。
- **`cm`（センチメートル）**: この絶対単位は1インチの25.2/64に相当します。
- **`mm`（ミリメートル）**: この絶対単位は1センチメートルの1/10に相当します。
- **`q`（クォーターミリメートル）**: この絶対単位は1センチメートルの1/40に相当します。
- **`pc`（パイカ）**: この絶対単位は1インチの1/6に相当します。
- **`pt`（ポイント）**: この絶対単位は1インチの1/72に相当します。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="units">
  <div class="unit px">px</div>
  <div class="unit inch">in</div>
  <div class="unit cm">cm</div>
  <div class="unit mm">mm</div>
  <div class="unit q">q</div>
  <div class="unit pc">pc</div>
  <div class="unit pt">pt</div>
</div>
```

```css
.units {
  display: flex;
  gap: 12px;
  flex-wrap: wrap;
  align-items: flex-end;
}

.unit {
  background: steelblue;
  color: white;
  text-align: center;
  padding: 4px;
}

.px {
  width: 40px;
  height: 40px;
}

.inch {
  width: 0.5in;
  height: 0.5in;
}

.cm {
  width: 1cm;
  height: 1cm;
}

.mm {
  width: 10mm;
  height: 10mm;
}

.q {
  width: 40q;
  height: 40q;
}

.pc {
  width: 6pc;
  height: 6pc;
}

.pt {
  width: 36pt;
  height: 36pt;
}
```

## 相対単位

- **パーセンテージ**: これらの相対単位は、親要素に対する比率としてサイズや寸法、その他のプロパティを定義できます。例えば、要素に`width: 50%;`を設定すると、その要素は親コンテナの幅の半分を占めます。
- **`em`単位**: これらの単位は要素のフォントサイズに対して相対的です。`ems`を`font-size`プロパティに使う場合、テキストのサイズは親要素のフォントサイズに対して相対的になります。
- **`rem`単位**: これらの単位はルート要素のフォントサイズに対して相対的で、ルート要素は`html`要素です。
- **`vh`単位**: `vh`は`"viewport height"`を表し、`1vh`はビューポートの高さの1%に相当します。
- **`vw`単位**: `vw`は`"viewport width"`を表し、`1vw`はビューポートの幅の1%に相当します。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  <div class="box percent">50%</div>
  <div class="box em">2em</div>
  <div class="box rem">2rem</div>
  <div class="box vh">10vh</div>
  <div class="box vw">10vw</div>
</div>
```

```css
html {
  font-size: 16px;
}

.parent {
  width: 200px;
  font-size: 20px;
  border: 2px dashed #555;
  padding: 10px;
  display: flex;
  flex-direction: column;
  gap: 12px;
}

.box {
  background: seagreen;
  color: white;
  text-align: center;
  padding: 6px;
}

.percent {
  width: 50%;
}

.em {
  font-size: 2em;
}

.rem {
  font-size: 2rem;
}

.vh {
  height: 10vh;
}

.vw {
  width: 10vw;
}
```

## `calc`関数

- **`calc()`関数**: `calc()`関数を使うと、スタイルシート内で直接計算を行い、プロパティの値を動的に決定できます。これにより、ビューポートのサイズや他の要素に基づいて寸法を計算し、柔軟でレスポンシブなユーザーインターフェースを作成できます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="calc-box">calc()</div>
```

```css
.calc-box {
  width: calc(100% - 40px);
  padding: 10px;
  background: steelblue;
  color: white;
  text-align: center;
  border: 2px solid black;
}
```
