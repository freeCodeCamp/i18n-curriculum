# --interactive--

비표현 주석 요소, 줄여서 `u` 요소는 비텍스트 주석이 적용된 인라인 텍스트를 나타내는 데 사용됩니다.

다음은 다양한 철자 오류를 강조하기 위해 `u` 요소를 사용하는 예입니다:

:::interactive_editor

```html
<p>
  You can use the unarticulated annotation element to highlight
  <u>inccccort</u> <u>spling</u> <u>issses</u>.
</p>
```

:::

예제에서 `incorrect`, `spelling`, `issues` 단어들은 철자가 틀렸습니다. `u` 요소의 기본 스타일은 텍스트 아래에 검은색 밑줄입니다.

HTML4에서는 `u` 요소가 스타일링 목적으로 사용되었습니다. 하지만 HTML5에서는 `u` 요소가 비텍스트 주석이 적용된 텍스트임을 나타내는 데만 사용되어야 합니다.

텍스트에 밑줄 스타일을 적용하려면 HTML 대신 CSS를 사용하세요.

취소선 요소, 줄여서 `s` 요소는 텍스트가 더 이상 정확하지 않거나 관련이 없을 때 나타내는 데 사용해야 합니다. 다음은 활동 취소를 보여주기 위해 `s` 요소를 사용하는 예입니다:

:::interactive_editor

```html
<p><s>Tomorrow's hike will be meeting at noon.</s></p>

<p>Due to unforeseen weather conditions, the hike has been canceled.</p>
```

:::

이 예제에서 첫 문장은 날씨 때문에 하이킹이 취소되어 취소선이 그어져 있습니다.

`s` 요소는 문서 변경 사항을 단순히 보여주기 위해 절대 사용해서는 안 됩니다. 그럴 경우 더 적절한 요소는 삭제된 텍스트 요소와 삽입된 텍스트 요소입니다.

`ruby` 요소는 본문 위나 아래에 표시되는 작은 텍스트를 나타냅니다. 주로 동아시아 문자 발음을 보여주는 데 사용됩니다. 다음은 MDN 웹 문서 페이지의 `ruby` 요소 예입니다:

:::interactive_editor

```html
<ruby> 明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp> </ruby>
```

:::

`rp` 요소, 즉 루비 대체 괄호 요소는 루비 주석 표시를 지원하지 않는 브라우저를 위한 대체 수단으로 사용됩니다.

`rt` 요소, 즉 루비 텍스트 요소는 루비 주석 텍스트를 나타내는 데 사용됩니다. 이 텍스트는 보통 동아시아 타이포그래피에서 발음이나 번역 세부 정보를 위해 사용됩니다.

`ruby` 요소는 다른 유형의 주석에도 사용할 수 있지만, 가장 일반적인 용도는 동아시아 타이포그래피입니다.
