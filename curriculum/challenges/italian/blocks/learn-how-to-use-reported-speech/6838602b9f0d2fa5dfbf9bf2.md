# --explanation--

`Unusually` è un avverbio usato per descrivere qualcosa che accade più o meno di quanto sia normale o previsto. D'altra parte, `unusual` è un aggettivo che descrive qualcosa che non è comune o normale. Per esempio:

- `The server experienced an unusually high load today.` – In questa frase, `unusually` riguarda quanto fosse alto il carico (molto oltre il normale).

- `It was unusual for the server to slow down like that.` – In questa frase, `unusual` si riferisce alla situazione. Non modifica un verbo o un aggettivo.

Una `query` è una domanda o richiesta inviata a un database per ottenere informazioni o eseguire un'azione. Per esempio:

`The application sent a query to the database to fetch user details.` – Questo significa che l'app ha chiesto al database informazioni specifiche.
