# --description--

Uma `library` é como uma grande estante com muitos códigos que os programadores podem usar.Pense nisso como um livro de receitas com muitas receitas (funções) que você pode usar para cozinhar (programar) diferentes pratos (projetos).

Por exemplo, uma `library` em programação pode ter uma função para criar gráficos.Em vez de escrever o código do zero, você pode usar essa função, como seguir uma receita para fazer um bolo.
