# --description--

`profile` オブジェクトにこのようにプロパティを追加し続けることは可能ですが、許可されるプロパティの型を制限したほうがよいでしょう。`profile` オブジェクトの形状をあらかじめ定義するほうが望ましいです。

これまでのレッスンでは、次のようなオブジェクト型の扱い方を学びました。

```ts
const composer: { 
  name: string; 
  instrument: string; 
  isActive: boolean; 
} = {
  name: "Ludwig van Beethoven",
  instrument: "Piano",
  isActive: false
};
```

この例では、`composer` はインラインオブジェクト型のオブジェクトです。このオブジェクトには `name`、`instrument`、`isActive` の3つの必須プロパティがあります。最初の2つのプロパティは `string` 型で、`isActive` は `boolean` 型でなければなりません。

既存の `profile` オブジェクトを更新してインラインオブジェクト型を含めてください。`username` は `string` 型、`age` は `number` 型、`isLoggedIn` は `boolean` 型にしてください。
