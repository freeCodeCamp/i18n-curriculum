# --description--

Тепер ви можете почати писати свій JavaScript. Розпочніть зі створення елемента `script`. Цей елемент використовується для завантаження JavaScript у ваш HTML-файл.

```html
<script>
  // JavaScript code goes here
</script>
```

# --hints--

У вас має бути елемент `script`.

```js
assert.isAtLeast(document.querySelectorAll('script').length, 1);
```

Ваш елемент `script` повинен мати початковий тег.

```js
assert.match(code, /<script\s*>/i);
```

Ваш елемент `script` повинен мати кінцевий тег.

```js
assert.match(code, /<\/script\s*>/i);
```
