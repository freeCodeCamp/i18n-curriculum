# --explanation--

`Not necessarily` é uma forma educada de expressar desacordo parcial.Em vez de dizer diretamente `no`, sugere que pode haver outros fatores a considerar.Em conversas, esta frase ajuda a manter as discussões abertas e respeitosas.Por exemplo:

- Pessoa 1: `I think upgrading the computer's processor will fix the issue.`

- Pessoa 2: `Not necessarily. The problem could be with the software, not the hardware.` - Esta pessoa está sugerindo que atualizar o processador pode não resolver o problema porque ele pode ser causado por outra coisa, como software desatualizado.
