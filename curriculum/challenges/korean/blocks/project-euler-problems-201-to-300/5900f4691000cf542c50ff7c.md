# --description--

어린아이가 40개의 퍼즐 조각으로 이루어진 "숫자 애벌레"를 가지고 있습니다. 각 조각에는 숫자 하나가 적혀 있고, 이 조각들을 일렬로 연결하면 1부터 40까지의 숫자가 순서대로 나타납니다.

매일 밤, 아이의 아버지는 놀이방에 흩어진 애벌레 조각들을 주워야 합니다. 아버지는 조각들을 무작위로 집어 올려 올바른 순서로 배치합니다.

이렇게 애벌레가 조립되면서 점차 서로 병합되는 구간들이 형성됩니다. 구간의 수는 처음에는 0개(조각이 하나도 놓이지 않은 상태)에서 시작해 보통 11개나 12개 정도까지 증가한 뒤, 다시 줄어들어 마지막에는 하나의 구간(모든 조각이 놓인 상태)으로 끝납니다.

예를 들어:

| 놓인 조각 | 지금까지 구간 수 |
| --------- | ---------------- |
|    12     |        1         |
|     4     |        2         |
|    29     |        3         |
|     6     |        4         |
|    34     |        5         |
|     5     |        4         |
|    35     |        4         |
|    …      |        …         |

$M$을 애벌레를 무작위로 정리할 때 만나는 최대 구간 수라고 합시다. 10조각 애벌레의 경우, 각 $M$에 대한 가능한 경우의 수는 다음과 같습니다.

| M | 경우의 수    |
|---|--------------|
| 1 |     512      |
| 2 |   250912     |
| 3 |  1815264     |
| 4 |  1418112     |
| 5 |   144000     |

따라서 가장 가능성이 높은 $M$ 값은 3이고, 평균값은 $\frac{385\\,643}{113\\,400} = 3.400732$로 소수점 여섯 자리까지 반올림한 값입니다.

40조각 애벌레의 경우 가장 가능성이 높은 $M$ 값은 11입니다. 그렇다면 $M$의 평균값은 얼마일까요? 소수점 여섯 자리까지 반올림하여 답하세요.
