# --description--

Sei $S_m = (x_1, x_2, \ldots, x_m)$ das $m$-Tuple positiver reeller Zahlen mit $x_1 + x_2 + \cdots + x_m = m$, für das $P_m = x_1 \times {x_2}^2 \times \cdots \times {x_m}^m$ maximiert wird.

Zum Beispiel kann überprüft werden, dass $[P_{10}] = 4112$ gilt (wobei [ ] die Ganzzahlfunktion ist).

Berechnen Sie $\sum {[P_m]}$ für $2 ≤ m ≤ 15$.
