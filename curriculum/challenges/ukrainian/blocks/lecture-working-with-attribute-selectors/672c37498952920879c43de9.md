# --interactive--

При створенні багатомовних вебсайтів або роботі з кастомними атрибутами даних часто потрібно стилізувати елементи залежно від мови, якою вони написані, або конкретних значень даних.

Атрибути `lang` і `data-lang` часто використовують для таких цілей, а селектор атрибутів у CSS дозволяє ефективно застосовувати стилі на основі цих атрибутів.

Атрибут `lang` використовується в HTML для вказівки мови вмісту всередині елемента. Ви можете захотіти стилізувати елементи по-різному залежно від мови, якою вони написані, особливо на багатомовному сайті.

Ось приклад, як можна використати селектор атрибутів для цільового вибору елементів з конкретним атрибутом `lang`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">This is an English paragraph.</p>
<p lang="fr">Ceci est un paragraphe en français.</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

Це CSS-правило застосовує курсив до будь-якого абзацу, у якого атрибут `lang` встановлений на англійську (`en`). Це може бути корисно для виділення англомовного тексту в документі, що містить кілька мов.

Аналогічно, ви можете використати селектор атрибутів для цільового вибору елементів з атрибутом `data-lang`.

Кастомні атрибути даних, як `data-lang`, часто використовують для зберігання додаткової інформації в елементах, наприклад, для вказівки мови, що використовується в певному розділі тексту.

Ось як можна використати CSS для цільового вибору та стилізації елементів на основі атрибута `data-lang`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p data-lang="fr">Ceci est un paragraphe en français.</p>
<p data-lang="en">This is a paragraph in English.</p>
```

```css
p[data-lang="fr"] {
  color: blue;
}
```

:::

У цьому випадку будь-який елемент `p` з атрибутом `data-lang`, встановленим на французьку (`fr`), матиме змінений колір тексту на синій. Це дозволяє швидко ідентифікувати та стилізувати розділи вмісту на основі інформації про мову, що зберігається в атрибуті `data-lang`.

Селектори атрибутів, як ці, надають потужний спосіб застосовувати умовне стилювання на основі метаданих, вбудованих у ваш HTML, роблячи ваші вебсторінки більш динамічними та контекстно-залежними.
