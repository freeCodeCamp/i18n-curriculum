# --explanation--

`No one mentioned anything` означає, що ніхто не повідомляв про щось незвичайне.

`Ask around` означає поговорити з різними людьми, щоб зібрати інформацію. Наприклад:

- `I'll ask around to see if anyone changed the settings.` — ви поговорите з людьми, щоб дізнатися, хто змінив налаштування.

- `He asked around about the new software tool.` — він поспілкувався з іншими, щоб отримати більше інформації про інструмент.
