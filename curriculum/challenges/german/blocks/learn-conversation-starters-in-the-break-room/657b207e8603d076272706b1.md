# --description--

Das Suffix `-ly` kann an Wörter angehängt werden, um die Häufigkeit eines Ereignisses anzugeben. So wird es auf verschiedene Zeiträume angewendet:

- `Daily` bedeutet, dass es einmal täglich stattfindet.
- `Weekly` zeigt an, dass etwas einmal wöchentlich passiert.
- `Monthly` beschreibt etwas, das einmal im Monat passiert.
- `Yearly` (oder `annually`) beschreibt etwas, das einmal im Jahr passiert.
