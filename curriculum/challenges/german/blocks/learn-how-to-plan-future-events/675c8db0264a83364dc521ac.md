# --explanation--

`I'll` ist die Kontraktion von `I will`. In diesem Kontext drückt sie eine Entscheidung aus, die im Moment des Sprechens getroffen wird. Das ist üblich, wenn jemand auf eine Situation reagiert oder zustimmt, etwas umgehend zu tun.

`Now` wird verwendet, um etwas auszudrücken, das gerade jetzt passiert. Zum Beispiel:

`I'll send the report now.` – Dies zeigt eine Entscheidung, sofort zu handeln.
