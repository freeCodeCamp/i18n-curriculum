# --description--

Um ein bestimmtes Wort oder eine Phrase hervorzuheben, können Sie das `em`-Element verwenden.

Heben Sie das Wort `love` im `figcaption`-Element hervor, indem Sie es in ein Hervorhebungs-`em`-Element einschließen.

# --hints--

Ihr Hervorhebungs-`em`-Element sollte ein öffnendes Tag haben. Öffnende Tags haben diese Syntax: `<elementName>`.

```js
assert(document.querySelector('em'));
```

Ihr Hervorhebungs-`em`-Element sollte ein schließendes Tag haben. Schließende Tags haben ein `/` direkt nach dem `<`-Zeichen.

```js
assert(code.match(/<\/em\>/));
```

Sie haben entweder das `figcaption`-Element gelöscht oder es fehlt ein öffnendes oder schließendes Tag.

```js
assert(document.querySelector('figcaption') && code.match(/<\/figcaption\>/));
```

Ihr Hervorhebungs-`em`-Element sollte den Text `love` umschließen. Sie haben entweder den Text ausgelassen oder einen Tippfehler.

```js
assert(
  document.querySelector('figcaption > em').innerText.toLowerCase() === 'love'
);
```

Der Text von `figcaption` sollte `Cats love lasagna` sein. Überprüfen Sie Tippfehler und ob die notwendigen Leerzeichen um die öffnenden und schließenden Tags des `em`-Elements vorhanden sind.

```js
assert(
  document
    .querySelector('figcaption')
    .innerText.replace(/\s+/gi, ' ')
    .match(/cats love lasagna\.?/i)
);
```
