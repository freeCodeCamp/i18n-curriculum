# --explanation--

La forma negativa del tiempo `Present Perfect continuous` se crea con `hasn't/haven't + been + present participle`. Describe una acción en curso que comenzó en el pasado y continúa o sigue siendo relevante, pero no ha ocurrido como se esperaba.
