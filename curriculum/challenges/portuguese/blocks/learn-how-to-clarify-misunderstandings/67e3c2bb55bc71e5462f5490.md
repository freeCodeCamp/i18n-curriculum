`Polymorphism` é um conceito de programação. Isso significa que uma única função, método ou operador pode ser usado com diferentes tipos de dados ou objetos. Por exemplo:

Em programação orientada a objetos, `polymorphism` permite que um único método se comporte de maneira diferente com base no objeto que o chama. - Aqui, `polymorphism` refere-se a um método que pode executar ações diferentes dependendo de qual objeto está usando-o.
