# --explanation--

Sebastián dice `¿Cómo se llama el ingeniero de software senior?`. Questo significa "Come si chiama l'ingegnere software senior?"

Quando la parola `Cómo` inizia una domanda, deve avere l'accento sulla prima lettera `o`.

Inoltre, ricorda che le domande in spagnolo iniziano con un punto interrogativo di apertura, `¿`.
