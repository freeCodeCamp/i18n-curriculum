# --description--

正如你所学，`repetir` 意味着再次说某事。当你添加代词 `lo` 时，你是在指代已经提到的某物。该代词替代具体的事物（一个单词、一个数字、一个电子邮件 `address` 等）。所以：

- `Repetir` = 重复。

- `Repetirlo` = 重复它。

`Repetirlo` 用于“这件事”已经从上下文中已知的情况。相同的模式适用于其他动词：

- `¿Puedes escribirlo?` - 你能写出来吗？

- `¿Puedes decirlo otra vez?` - 你能再说一遍吗？

# --explanation--

`Lo` 是一个直接宾语代词，用于替换对话中已知的内容。当它附加在不定式上时，会形成一个单词：

- `repetir` + `lo` → `repetirlo` － 重复它。

- `escribir` + `lo` → `escribirlo` - 写它。

- `decir` + `lo` → `decirlo` - 说出来。

当监听器已经理解“it”指的是什么时，这种结构很常见。
