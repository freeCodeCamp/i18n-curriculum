# --description--

涵蓋尺寸為 3 x 2 x 1 的長方體上每個可見面的最少立方體數量是二十二。

<img alt="3x2x1 cuboid covered by twenty-two 1x1x1 cubes" src="https://cdn.freecodecamp.org/curriculum/project-euler/cuboid-layers.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

如果我們為這個實心物體添加第二層，將需要四十六個立方體來涵蓋每個可見的面，第三層將需要七十八個立方體，第四層將需要一百一十八個立方體來涵蓋每個可見的面。

然而，尺寸為 5 x 1 x 1 的長方體的第一層也需要二十二個立方體；同樣地，尺寸為 5 x 3 x 1、7 x 2 x 1 和 11 x 1 x 1 的長方體的第一層都包含四十六個立方體。

我們將定義 $C(n)$ 來表述在其某一層包含 $n$ 個立方體的長方體數量。所以 $C(22) = 2$、$C(46) = 4$、$C(78) = 5$，以及 $C(118) = 8$。

結果顯示 154 是使 $C(n) = 10$ 的最小 $n$ 值。

找出使得 $C(n) = 1000$ 的最小 $n$ 值。
