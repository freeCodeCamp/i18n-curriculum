# --description--

У <a href="/learn/javascript-algorithms-and-data-structures/basic-javascript/replace-loops-using-recursion" target="_blank" rel="noopener noreferrer nofollow">попередньому завданні</a> ви навчились використовувати рекурсію замість `for` циклу. Тепер розглянемо складнішу функцію, яка повертає масив послідовних цілих чисел, починаючи з `1` і до числа, переданого у функцію.

Як згадувалося в попередньому завданні, існує <dfn>базовий випадок</dfn>. Базовий випадок повідомляє рекурсивній функції, коли їй більше не потрібно викликати саму себе. Це простий випадок, коли повернене значення вже відоме. Також буде <dfn>рекурсивний виклик</dfn>, який виконує оригінальну функцію з іншими аргументами. Якщо функція написана правильно, зрештою буде досягнуто базового випадку.

Наприклад, уявімо, що ви хочете написати рекурсивну функцію, яка повертає масив, що містить числа від `1` до `n`. Ця функція повинна приймати аргумент `n`, який представляє кінцеве число. Потім вона повинна викликати саму себе з поступово меншими значеннями `n`, доки не досягне `1`. Функцію можна написати так:

```javascript
function countup(n) {
  if (n < 1) {
    return [];
  } else {
    const countArray = countup(n - 1);
    countArray.push(n);
    return countArray;
  }
}
console.log(countup(5));
```

Значення `[1, 2, 3, 4, 5]` буде виведено в консоль.

Спочатку це здається нелогічним, оскільки значення `n` *зменшується*, але значення у фінальному масиві *зростають*. Це відбувається тому, що додавання відбувається в останню чергу, після того, як рекурсивний виклик повернувся. У момент, коли `n` додається до масиву, `countup(n - 1)` вже було обчислено і повернуло `[1, 2, ..., n - 1]`.

# --instructions--

Ми визначили функцію з назвою `countdown` з одним параметром (`n`). Функція повинна використовувати рекурсію, щоб повернути масив, що містить цілі числа від `n` до `1` на основі параметра `n`. Якщо функцію викликати з числом меншим за 1, вона повинна повернути порожній масив. Наприклад, виклик цієї функції з `n = 5` повинен повернути масив `[5, 4, 3, 2, 1]`. Ваша функція має використовувати рекурсію, викликаючи саму себе, і не повинна використовувати цикли будь-якого виду.
