# --explanation--

Ángela diz `¿Y tu profesión?`, uma pergunta curta de acompanhamento. `Y` conecta esta pergunta ao tópico anterior.

Embora ela não diga `¿Cuál es...?`, a pergunta completa é entendida pelo contexto:

`¿Cuál es tu profesión?`

Em conversas, o espanhol frequentemente omite palavras repetidas quando o significado está claro.
