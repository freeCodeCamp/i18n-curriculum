# --interactive--

인라인 코드 요소는 텍스트 안에 짧은 코드 스니펫을 나타내는 데 사용됩니다. code 요소의 일반적인 사용 사례는 기술 문서나 문서 페이지입니다.

다음은 `code` 요소를 사용해 CSS 코드 스니펫을 보여주는 예시입니다:

:::interactive_editor

```html
<p>
  To set the text color to blue in CSS, use the following code:
  <code>color: blue;</code>
</p>
```

:::

이 예시에서 CSS `color` 속성은 텍스트 색상을 `blue`로 설정하는 데 사용됩니다. 이 코드 스니펫을 `<code>` 태그로 감싸면 브라우저에 이 텍스트가 인라인 코드임을 알립니다.

브라우저는 `code` 요소 안의 콘텐츠에 기본 스타일을 적용합니다. 기본 스타일은 고정폭 폰트입니다.

`code` 요소는 한 줄짜리 코드를 나타내는 데 적합합니다. 여러 줄 코드를 나타내려면 `code` 요소를 미리 서식이 지정된 텍스트 요소 안에 넣어야 합니다.

미리 서식이 지정된 텍스트 요소는 미리 서식이 지정된 텍스트를 나타내는 데 사용됩니다. 다음은 미리 서식이 지정된 텍스트 요소를 사용해 CSS 선언을 보여주는 예시입니다:

:::interactive_editor

```html
<pre>
  <code>
    body {
      color: red;
    }
  </code>
</pre>
```

:::

`pre` 요소를 사용할 때는 공백에 주의해야 합니다. HTML 문서 안에 작성된 대로 정확히 표시되기 때문입니다.

브라우저에서는 코드가 오른쪽으로 여러 칸 들여쓰기된 것을 볼 수 있습니다. 코드 예시에서 들여쓰기를 변경하면 화면에 표시되는 들여쓰기에도 차이가 나타납니다.

HTML 문서 안에 코드 예시를 포함할 때는 짧은 인라인 예시에는 `code` 요소를 사용하세요.

더 긴 코드 스니펫을 표시해야 한다면 `pre`와 `code` 요소를 사용해야 합니다.
