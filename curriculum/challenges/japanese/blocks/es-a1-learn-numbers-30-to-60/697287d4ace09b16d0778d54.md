# --description--

次は50台の数字に進みましょう。

スペイン語で50は `cincuenta` と呼ばれます。

51から59までの数字は、10の位の単語 `cincuenta`（50）に、接続詞 `y`（「と」）を続け、最後に1から9の数字を付けて作られます。

| 数字 | スペイン語 |
| :--- | :--- |
| 50 | `Cincuenta` |
| 51 | `Cincuenta y uno` |
| 52 | `Cincuenta y dos` |
| 53 | `Cincuenta y tres` |
| 54 | `Cincuenta y cuatro` |
| 55 | `Cincuenta y cinco` |
| 56 | `Cincuenta y seis` |
| 57 | `Cincuenta y siete` |
| 58 | `Cincuenta y ocho` |
| 59 | `Cincuenta y nueve` |

この範囲で51の数字 `cincuenta y uno` は特別で、男性名詞の前では `cincuenta y un` に変わります。例えば：

`Cincuenta y un años` - 51歳。
