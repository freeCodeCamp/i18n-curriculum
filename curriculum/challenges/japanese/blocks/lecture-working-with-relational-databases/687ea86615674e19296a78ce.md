# --description--

リレーショナルデータベースは、ウェブ開発、在庫管理システム、医療システム、eコマースアプリケーションなど、幅広い用途で使われています。

最も一般的なリレーショナルデータベースには、MySQL、PostgreSQL、SQLite、Microsoft SQL Serverがあります。それぞれについて少し学んでみましょう。

MySQLはオープンソースのリレーショナルデータベース管理システムです。使いやすく、信頼性が高く、実際のアプリケーション構築に効率的なため、ウェブ開発で非常に人気があります。また、活発な開発者コミュニティがサポートとメンテナンスを行っています。

PostgreSQLは公式に「世界で最も高度なオープンソースのリレーショナルデータベース」と説明されています。無料のオープンソースのオブジェクトリレーショナルデータベースシステムで、信頼性、データの整合性、拡張性で知られています。例えば、独自のデータ型やカスタム関数を定義したり、データベースを再コンパイルせずに異なるプログラミング言語でコードを書いたりできます。

SQLiteは無料でオープンソースの軽量なファイルベースのSQLデータベースエンジンです。世界で最も広く使われているデータベースエンジンの一つです。SQLiteは自己完結型でサーバーレスです。ゼロコンフィギュレーションのデータベースで、初期設定が不要なので、すぐに始められるのが特徴です。

これら3つのデータベースは無料でオープンソースです。他にもMicrosoft SQL ServerやOracleデータベースなどの人気のある商用オプションがあります。

Microsoft SQL ServerはMicrosoftが開発した人気のリレーショナルデータベース管理システムで、さまざまなアプリケーションで使われています。

Oracleデータベースは高性能を必要とする企業向けアプリケーションでよく使われるスケーラブルなリレーショナルデータベース管理システムです。

よいですね。これで利用可能なさまざまな選択肢について理解できました。次にPostgreSQLのインストール方法を見てみましょう。

まず、公式ウェブサイトにアクセスしてください：<a href="https://www.postgresql.org/" target="_blank">https://www.postgresql.org/</a>。

そこに着いたら、「Download」ボタンをクリックしてください。

すると、使用しているオペレーティングシステムを選択できるページに移動します。どれかをクリックすると、そのOS向けのインストラクションが表示されます。

ここではWindowsのインストール手順を説明しますが、他のOSでもほぼ同じです。

このページから、「Download the installer」リンクをクリックしてインストーラーをダウンロードできます。

すると、インストールしたいPostgreSQLのバージョンを選べるページに移動します。対応するアイコンをクリックするとダウンロードが始まります。

次に、ダウンロードフォルダーやデフォルトの保存先フォルダーにインストーラーがあるはずです。

それをダブルクリックするとセットアップが始まります。Nextをクリックしてください。

PostgreSQLをインストールするディレクトリを選びます。デフォルトの場所が表示されますが、カスタマイズも可能です。

次にインストールしたいコンポーネントを選択します。4つのコンポーネントがあります：

- PostgreSQL Server、データベースエンジン本体
- pgAdmin 4、データベース管理のための使いやすいツール
- Stack Builder、PostgreSQL関連の追加ソフトや拡張機能をダウンロード・インストールするツール
- Command Line Tools、ターミナルから直接PostgreSQLサーバーとやり取りするためのツール。SQLクエリを実行する対話型ターミナルpsqlも含まれます。

準備ができたらNextをクリックしてください。

次にデータを保存するディレクトリを選びます。デフォルトのパスが表示されますが、カスタマイズ可能です。準備ができたらNextをクリックしてください。

次に、データベースのスーパーユーザー（すべてのデータベースとオブジェクトに無制限にアクセスできるユーザー）のパスワードを入力します。パスワードを入力し、再入力してNextをクリックしてください。

次にサーバーがリッスンするポート番号を選びます。デフォルトは5432です。Nextをクリックしてください。

新しいデータベースクラスタのロケールを選択します。これはロケールに依存するデータ型を正しく解釈するために重要です。Nextをクリックしてください。

最後にインストール前の概要が表示されます。内容を確認してNextをクリックしてください。

インストールを開始するか確認が求められます。Nextをクリックするとインストールが始まります。数分で完了します。

インストールが完了すると、選択したコンポーネントに応じて、Stack Builderを起動して追加のツールやドライバー、アプリケーションをダウンロード・インストールするかどうか選べます。

Stack Builderを起動する場合は、インストールを開始できる画面が表示されます。後で行う場合はCancelをクリックしてください。

おめでとうございます！これでPostgreSQLがコンピューターにインストールされました。他のOSでもほぼ同じ手順です。これでデータベースの使用を開始できます。

アプリを開いて、コンポーネントで選択していればインストールされているはずのPostgreSQL用のオープンソース管理・開発プラットフォームpgAdminを検索してください。pgAdmin 4をクリックして開きます。

アプリ起動時に初期の読み込み画面が表示されます。その後、新しいサーバーの開始、データベースの確認、pgAdminの設定などができるダッシュボードが表示されます。ドキュメントへの便利なリンクもあります。

左上の「Servers」をクリックすると、インストール時に設定したパスワードの入力が求められます。

パスワードを入力すると、テーブルを含むデータベースにアクセスできる画面が表示されます。

また、SQLクエリ用の対話型ターミナルpsqlも非常に便利なツールです。システムでpsqlを検索して開いてください。

SQLシェルが開き、コマンドの実行を開始できます。

これがWindowsでPostgreSQLをインストールする方法です。macOSやLinuxでもほぼ同じです。

macOSにインストールする場合は、パッケージマネージャーのHomebrewを使うこともできます。Homebrewがインストールされていれば、次のコマンドでPostgreSQLをインストールできます：

```bash
brew install postgresql@<version>
```

インストールしたいバージョンは、at記号(`@`)の後に番号を書いて指定できます。
