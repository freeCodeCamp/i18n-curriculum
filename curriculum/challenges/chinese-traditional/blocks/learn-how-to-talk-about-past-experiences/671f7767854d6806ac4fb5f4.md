# --explanation--

`Miss` 意思是未能出席或參與某事。例如： 

`I missed the meeting because I was stuck in traffic.`－這表示你在交通上花了太多時間，導致上班遲到。 

`design sprint` 是一個有時間限制的協作處理程序，團隊在短時間內（通常為幾天）致力於解決設計問題或開發原型。 例如： 

`The team completed a design sprint to test the new user interface concept.` 這表示團隊參與了這個協作處理程序來測試該概念。 

在這個背景關係中，Maria 無法參加資安工作坊，因為她正在積極參與設計衝刺工作坊。
