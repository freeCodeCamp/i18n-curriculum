# --description--

では、少し内容を変えてみましょう：

```html
<!-- index.html -->

<div class="main">
  <div class="list" id="subsection"></div>
</div>
```

```css
/* rule 1 */
#subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

上の例では、ルール2がIDセレクターよりも多くのクラスセレクターを持っているにもかかわらず、IDがクラスよりも優先されるため、ルール1の方がより具体的です。
