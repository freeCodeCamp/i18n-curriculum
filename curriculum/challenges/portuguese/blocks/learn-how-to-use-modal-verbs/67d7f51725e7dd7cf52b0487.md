# --explanation--

A contração `can't` é a forma curta de `cannot`, que significa não ser capaz de fazer algo.  

`Can't` (`cannot`) + verbo base é usado quando algo é impossível ou difícil de entender.Por exemplo:

- `I can't believe this happened!` - É difícil para você acreditar.

- `She can't access the system.` - Ela não consegue acessar o sistema.

Jake está confuso sobre como os atacantes contornaram o firewall, o que significa que ele não consegue descobrir.
