# --description--

Como has visto antes, las funciones pueden devolver un valor usando la palabra clave `return`. La palabra clave `return` se usa para devolver un valor desde una función. Cuando se usa la palabra clave `return`, la función dejará de ejecutarse y devolverá el valor especificado después de la palabra clave `return`.

```js
function add(a, b) {
  return a + b
}

console.log(add(2, 3)); // Output: 5
```

Pero, ¿qué pasa si la palabra clave `return` se usa antes del final de la función?  
Para responder a esta pregunta, considera el siguiente ejemplo:

```js
function add(a, b) {
  if(a > 2){
    return b;
  }

  return a + b;
}

console.log(add(3, 7)); // Output: 7
``` 

En el ejemplo anterior, la función `add` tiene una sentencia condicional que verifica si el valor de `a` es mayor que `2`. Si se cumple la condición, la función devolverá el valor de `b` y dejará de ejecutarse. Si no se cumple la condición, la función devolverá la suma de `a` y `b`.
