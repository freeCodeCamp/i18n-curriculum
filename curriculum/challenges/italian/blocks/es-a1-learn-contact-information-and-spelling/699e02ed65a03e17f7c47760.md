# --description--

Come hai imparato, `repetir` significa dire qualcosa di nuovo. Quando aggiungi il pronome `lo`, ti riferisci a qualcosa già menzionato. Il pronome sostituisce la cosa specifica (una parola, un numero, un indirizzo email, ecc.). Quindi:

- `Repetir` = ripetere.

- `Repetirlo` = ripeterlo.

`Repetirlo` si usa quando "la cosa" è già nota dal contesto. Lo stesso pattern funziona con altri verbi:

- `¿Puedes escribirlo?` - Puoi scriverlo?

- `¿Puedes decirlo otra vez?` - Puoi dirlo di nuovo?

# --explanation--

`Lo` è un pronome oggetto diretto usato per sostituire qualcosa già noto nella conversazione. Quando è allegato a un infinito, forma una sola parola:

- `repetir` + `lo` → `repetirlo`  - Ripeterlo.

- `escribir` + `lo` → `escribirlo` - Scriverlo.

- `decir` + `lo` → `decirlo` - Dirlo.

Questa struttura è comune quando chi ascolta capisce già a cosa si riferisce "lo".
