# --interactive--

При стилізації вебсайтів одним із найуніверсальніших і найінтуїтивніших способів роботи з кольором є використання моделі кольору HSL. HSL означає відтінок, насиченість і світлість — три ключові компоненти, що визначають колір.

Модель кольору HSL представляє кольори так, як їх сприймає людина. У цьому уроці ми розглянемо, що таке модель кольору HSL і як функція `hsl()` використовується в CSS.

Відтінок — це тип кольору, представлений як кут на колірному колі, який варіюється від `0` до `360` градусів. Наприклад, `0` градусів відповідає червоному, `120` градусів — зеленому, а `240` градусів — синьому. Змінюючи відтінок, ви переходите через різні кольори спектра.

Насиченість означає інтенсивність або чистоту кольору. Вона вимірюється у відсотках від `0%` (повністю знебарвлений, сірий колір) до `100%` (найяскравіша форма цього кольору). Колір із `100%` насиченістю є повністю яскравим, а колір із `0%` насиченістю виглядає як відтінок сірого.

Світлість визначає, наскільки світлим або темним є колір, також вимірюється у відсотках. Значення світлості `0%` дасть чорний колір, `50%` — нормальний тон відтінку, а `100%` — білий.

У CSS функція `hsl()` використовується для визначення кольорів за моделлю HSL. Сучасний синтаксис використовує значення, розділені пробілами:

```css
element {
  color: hsl(hue saturation lightness);
}
```

Ви також можете зустріти старіший синтаксис із комами у наявному коді, який досі підтримується браузерами:

```css
element {
  color: hsl(hue, saturation, lightness);
}
```

Розглянемо це на прикладі:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
body {
  background-color: hsl(0 0% 1% / 1);
}

p {
  color: hsl(120 100% 50%);
}
```

:::

У цьому випадку відтінок становить `120` градусів, що відповідає зеленому кольору. Насиченість — `100%`, тому зелений буде повністю яскравим. Світлість — `50%`, тобто нормальний тон — ні занадто темний, ні занадто світлий. В результаті колір тексту абзацу буде яскраво-зеленим.

Однією з головних переваг моделі кольору HSL є її інтуїтивність. Вона дозволяє легко регулювати яскравість або світлість кольору, змінюючи значення насиченості та світлості, не змінюючи основний колір (відтінок).

Наприклад, якщо ви хочете створити різні відтінки або тони одного й того ж кольору, просто змініть значення світлості.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="light">This is a light blue div.</div>
<div class="dark">This is a dark blue div.</div>
```

```css
div.light {
  background-color: hsl(240 100% 80%);
}

div.dark {
  background-color: hsl(240 100% 20%);
  color: hsl(0 0% 100%);
}
```

:::

Тут обидва `div` елементи використовують однаковий відтінок (`240` градусів, що є синім), але один має світлість `80%` (світліший відтінок синього), а інший — світлість `20%` (темніший відтінок синього).

Як і модель RGB, функція `hsl()` також підтримує необов’язкове значення альфа для контролю прозорості. Його додають після роздільника `/` як четвертий параметр. Ось базовий синтаксис:

```css
element {
  background-color: hsl(hue saturation lightness / alpha);
}
```

Розглянемо приклад:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: hsl(0 100% 50% / 0.5);
}
```

:::

Цей код задасть `div` напівпрозорий червоний фон, де відтінок встановлено на `0` градусів (червоний), насиченість — `100%`, світлість — `50%`, а альфа — `0.5` (50% прозорості). Ви також можете вказати альфу у відсотках — наприклад, `hsl(0 100% 50% / 50%)` еквівалентне `hsl(0 100% 50% / 0.5)`.

Ви також часто побачите функцію `hsla()` у наявному CSS. Це був оригінальний спосіб додати прозорість у кольори HSL, використовуючи значення, розділені комами:

```css
element {
  background-color: hsla(hue, saturation, lightness, alpha);
}
```

Наприклад, `hsla(0, 100%, 50%, 0.5)` — це застарілий еквівалент `hsl(0 100% 50% / 0.5)`. Обидва підтримуються браузерами, але сучасний синтаксис `hsl()` із роздільником `/` тепер є пріоритетним.

Модель кольору HSL особливо корисна, коли потрібно створювати кольорові схеми та легко регулювати відтінки або тони.

Оскільки HSL розділяє відтінок від насиченості та світлості, ви можете зберігати послідовну кольорову тему, регулюючи світлість для тіней або підсвітлень. Такий рівень гнучкості робить HSL улюбленим інструментом як для дизайнерів, так і для розробників.

Крім того, HSL робить значення кольорів більш читабельними та зрозумілими порівняно з іншими методами, як-от шістнадцяткові коди або RGB. Ви чіткіше бачите задум кольору і можете відповідно налаштовувати, особливо під час експериментів із різними дизайнами.

З моделлю кольору HSL ви отримуєте більше гнучкості у контролі тонів, насиченості та світлості, що робить її потужним інструментом для створення насичених і динамічних дизайнів у CSS.
