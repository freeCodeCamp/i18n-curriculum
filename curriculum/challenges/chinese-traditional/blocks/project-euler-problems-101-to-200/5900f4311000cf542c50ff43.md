# --description--

我們稱一個整數邊長且恰有一個角為 60° 的三角形為 60° 三角形。

設 $r$ 為此 60° 三角形的內切圓半徑。

有 1234 個 60° 三角形，其 $r ≤ 100$。

令 $T(n)$ 為 $r ≤ n$ 的 60° 三角形數量，因此 $T(100) = 1234$、$T(1000) = 22767$ 和 $T(10000) = 359912$。

尋找 $T(1053779)$。
