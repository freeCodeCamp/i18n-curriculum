# --explanation--

`Could it have been...?` é uma forma de perguntar se algo poderia ter acontecido.

`Could` expressa possibilidade.

`Have been` é usado para falar sobre algo que possivelmente aconteceu no passado.

Jessica está supondo ou sugerindo uma possibilidade. Outro exemplo:

`Could it have been the update that broke the app?` - Alguém está perguntando se a atualização causou o aplicativo a parar de funcionar.
