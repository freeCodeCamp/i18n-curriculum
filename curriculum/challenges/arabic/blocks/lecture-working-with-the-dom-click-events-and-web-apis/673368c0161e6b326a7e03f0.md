# --interactive--

لنرَ كيف يمكنك إنشاء عقد باستخدام `innerHTML` و `createElement()`.

سنبدأ بـ `innerHTML`.

`innerHTML` خاصية من خصائص كائنات `Element` يمكنك استخدامها لتعيين ترميز HTML الخاص بها. باستخدام `innerHTML`، يمكنك تعيين الهيكل HTML لعنصر موجود باستخدام سلسلة نصية، مما ينشئ كل العقد اللازمة.

هذا مثال. لدينا `div` فارغ وسنضيف بعض العناصر بداخله باستخدام خاصية `innerHTML`:

```html
<div id="container">
  <!-- Add new elements here -->
</div>
```

أولًا، نحتاج إلى اختيار العنصر بواسطة معرفه (ID):

```js
const container = document.getElementById("container");
```

بعدها، نعيّن خاصية `innerHTML` للحاوية إلى سلسلة نصية. يجب أن تحتوي هذه السلسلة على كل الترميز اللازم لتمثيل العقد والهيكل الذي تريد إنشاؤه. يمكنك التفكير فيها ككتابة HTML داخل سلسلة نصية.

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
container.innerHTML = "<ul><li>Cheese</li><li>Tomato</li></ul>";
```

:::

سنعيّن `innerHTML` لهذا العنصر إلى قائمة غير مرتبة من مكونات البيتزا (جبن وطماطم).

يمكنك أيضًا كتابة السلسلة على عدة أسطر إذا جعل ذلك الترميز أسهل للفهم. المهم أن الترميز يمثل بدقة الهيكل HTML الذي تريد إنشاؤه.

بعد تشغيل هذا الكود، سترى الهيكل HTML التالي إذا فحصت الترميز الخاص بك.

```html
<div id="container">
  <ul>
    <li>Cheese</li>
    <li>Tomato</li>
  </ul>
</div>
```

تم إنشاء العقد الجديدة وإضافتها ديناميكيًا إلى DOM بعد تحليل السلسلة النصية.

يمكن أن يكون `innerHTML` مفيدًا جدًا في بعض السيناريوهات. مع ذلك، له بعض مخاطر الأمان التي يجب أن تكون على دراية بها. لا يجب استخدامه إذا لم تكن تتحكم بالكامل في السلسلة النصية.

على سبيل المثال، إذا كانت السلسلة ستُدخل بواسطة المستخدم، فلا يجب استخدام `innerHTML` لأن المستخدم قد يُدخل محتوى خبيثًا في موقعك. لذلك، يُنصح عادة باستخدام `textContent` بدلاً منه لإدخال نص عادي.

طريقة أخرى لإنشاء عقدة جديدة هي باستخدام طريقة `createElement()`. باستخدام هذه الطريقة، يمكنك إنشاء عنصر جديد بتحديد اسم الوسم الخاص به.

على سبيل المثال، إذا أردت إنشاء عنصر صورة، تمرر وسم `img` كسلسلة نصية عند استدعاء هذه الطريقة:

```js
document.createElement("img");
```

ويمكنك تعيين هذا الكائن الجديد إلى متغير:

```js
const img = document.createElement("img");
```

ترجع طريقة `createElement()` كائن `HTMLElement` جديد إذا كان المستند من نوع `HTMLDocument`. وإلا، فترجع كائن `Element`.

بمجرد أن يصبح لديك هذا العنصر الجديد جاهزًا، يمكنك إضافته إلى DOM كطفل لعنصر موجود آخر باستخدام طريقة `appendChild()`، أو يمكنك إدخاله في موقع محدد باستخدام طرق أخرى. اختر الطريقة التي تناسب احتياجاتك.

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
const img = document.createElement("img");
img.src = "https://cdn.freecodecamp.org/curriculum/cat-photo-app/lasagna.jpg";
img.alt = "A slice of lasagna on a plate.";
container.appendChild(img);
```

:::

إنشاء العقد باستخدام `innerHTML` و `createElement()` يسمح لك بالتلاعب ديناميكيًا بهيكل ومحتوى مواقعك. بدمج هذه التقنيات، يمكنك بناء تطبيقات ويب تفاعلية.
