# --explanation--

Sophie dice che eseguire i test è una buona pratica perché `it helps avoid mistakes`. Questo significa che aiuta gli sviluppatori a individuare gli errori in anticipo prima che il codice venga rilasciato.

Eseguire i test può mostrare se c'è qualcosa che non va nel codice, anche se a prima vista sembra tutto a posto. È una parte importante per scrivere software di alta qualità.
