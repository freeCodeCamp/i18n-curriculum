# --description--

Розглянемо рівняння виду: $a^2 + b^2 = N$, $0 ≤ a ≤ b$, де $a$, $b$ та $N$ — цілі числа.

Для $N = 65$ існують два розв’язки:

$a = 1, b = 8$ та $a = 4, b = 7$.

Позначимо $S(N)$ суму значень $a$ усіх розв’язків рівняння $a^2 + b^2 = N$, $0 ≤ a ≤ b$, де $a$, $b$ та $N$ — цілі числа.

Отже, $S(65) = 1 + 4 = 5$.

Знайдіть $\sum S(N)$ для всіх безквадратних $N$, які діляться лише на прості числа форми $4k + 1$ з $4k + 1 < 150$.
