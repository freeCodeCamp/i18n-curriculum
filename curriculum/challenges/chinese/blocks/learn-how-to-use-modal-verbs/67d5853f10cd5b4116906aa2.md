`Can you explain the project details to me?`

`Can explains you the project details?`

### --feedback--

情态动词 `can` 后应跟动词原形，而不是 `explains`。

---

`Do can you explain the project details?`

### --feedback--

`Can` 已经作为助动词使用，因此在疑问句中不需要 `do`。

# --explanation--

当使用情态动词构成疑问句时，结构为：情态动词 + 主语 + 动词原形 + （对象）？例如：

- `Can we complete the project by Friday?` - 询问是否有可能在星期五之前完成项目。

- `Should we update the software to the latest version?` - 询问是否应该将软件更新到最新版本的建议。

在正确答案中，`can` 位于主语（`you`）之前，动词（`explain`）使用其基本形式。
