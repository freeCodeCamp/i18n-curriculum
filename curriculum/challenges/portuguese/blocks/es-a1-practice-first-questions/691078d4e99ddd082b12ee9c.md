# --explanation--

Essas perguntas são úteis para conhecer alguém em um contexto informal, usando o pronome `tú` ("você"):

`¿Cómo te llamas?` é usado para perguntar o nome de uma pessoa. É equivalente a "What is your name?"

`¿De dónde eres?` é usado para perguntar sobre a origem ou nacionalidade de uma pessoa. É equivalente a "Where are you from?"

`¿A qué te dedicas?` é usado para perguntar sobre a profissão ou trabalho de uma pessoa. É equivalente a "What is your profession?"

`¿Cuántos años tienes?` é usado para perguntar a idade de uma pessoa. É equivalente a "How old are you?"
