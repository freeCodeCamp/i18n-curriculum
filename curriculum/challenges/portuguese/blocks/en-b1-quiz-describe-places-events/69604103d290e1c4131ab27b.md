# --description--

Este quiz verifica sua compreensão sobre como descrever lugares e eventos, incluindo projetos, issues e situações no local de trabalho.

Para passar no quiz, você deve responder corretamente pelo menos 18 das 20 perguntas abaixo.

Leia cada pergunta e escolha a resposta correta. Há apenas uma resposta correta para cada pergunta.
