# --interactive--

Поля та висота рядка є важливими для створення відступів між елементами списку, щоб покращити читабельність і візуальну привабливість.

Спочатку почнемо створювати відступи між елементами списку за допомогою полів!

Поля можна використовувати для створення простору між елементами списку, застосовуючи властивості margin до `li` елементів. Цей метод дозволяє контролювати відступи поза межами кожного елемента списку, ефективно збільшуючи або зменшуючи горизонтальний розрив між ними.

Давайте розглянемо приклад невпорядкованого списку з трьома елементами.

:::interactive_editor

```html
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

За замовчуванням HTML не створює багато простору між елементами списку.

Щоб додати відступ знизу кожного елемента списку, ви можете використати властивість `margin-bottom` так:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  margin-bottom: 40px;
}
```

:::

У цьому прикладі `40px` поля буде застосовано до нижньої частини кожного елемента списку всередині невпорядкованого списку.

Інший спосіб створити відступи між елементами списку — це використати властивість `line-height`.

Властивість `line-height` регулює вертикальний простір між рядками тексту всередині одного елемента списку.

Хоча вона головним чином впливає на відступи між рядками тексту в кожному елементі, вона також може опосередковано впливати на загальний простір між елементами списку, якщо елементи містять лише один рядок тексту.

Якщо елементи списку мають кілька рядків тексту, `line-height` впливатиме на відступи між цими рядками, але безпосередньо не регулюватиме простір між окремими елементами списку.

Щоб контролювати відступи між окремими елементами списку, слід використовувати властивості `margin` або `padding`.

Використовуючи той самий невпорядкований список з попереднього прикладу, ось приклад застосування `line-height` до елементів списку:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  line-height: 2; 
}
```

:::

У цьому прикладі `line-height: 2;` встановлює висоту рядка вдвічі більшою за розмір шрифту, створюючи більше вертикального простору всередині кожного елемента списку.

---

### --feedback--

Ця властивість впливає на `border` елемента, а не на простір поза межами.
