# --interactive--

ID-Selektoren gehören zu den mächtigsten Selektoren in CSS und ermöglichen es Entwicklern, Stile auf bestimmte Elemente mit eindeutigen Bezeichnern anzuwenden.

Dadurch sind sie sehr effektiv, um einzelne Elemente gezielt zu stylen, die eine einzigartige Gestaltung benötigen.

ID-Selektoren werden durch ein Rautezeichen (`#`) gefolgt vom ID-Namen definiert. Sie sollten innerhalb eines HTML-Dokuments einzigartig sein, das heißt, kein zwei Elemente sollten dieselbe ID haben.

Hier ist ein Beispiel mit einem ID-Selektor:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="unique">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#unique {
  color: purple;
}
```

:::

In diesem Beispiel wird die Textfarbe des Elements mit der ID `unique` auf `purple` gesetzt.

ID-Selektoren haben eine sehr hohe Spezifität, höher als Typselektoren und Klassen-Selektoren, aber niedriger als inline-Stile. Der Spezifitätswert für einen ID-Selektor ist `(0, 1, 0, 0)`.

Das bedeutet, dass ID-Selektoren Klassen- und Typselektoren überschreiben können, aber von inline-Stilen überschrieben werden können.
