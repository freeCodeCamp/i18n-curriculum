# --description--

Wenn Sie eine Liste von Elementen haben möchten, bei der die Reihenfolge keine Rolle spielt, wie eine Einkaufsliste mit Artikeln, die in beliebiger Reihenfolge gekauft werden können, dann können Sie eine ungeordnete Liste verwenden.

Ungeordnete Listen werden mit dem `<ul>`-Element erstellt, und jedes Element innerhalb der Liste wird mit dem Listenelement `<li>` erstellt.

Jedes Listenelement in einer ungeordneten Liste beginnt mit einem Aufzählungspunkt:

Wenn Sie stattdessen eine Liste von Elementen erstellen möchten, bei der die Reihenfolge wichtig ist, wie Schritt-für-Schritt-Anleitungen für ein Rezept oder Ihre Top 10 Lieblingsfernsehserien, dann können Sie eine geordnete Liste verwenden.
