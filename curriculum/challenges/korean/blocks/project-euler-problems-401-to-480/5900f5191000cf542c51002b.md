# --description--

$a$, $b$, $c$를 양수라고 하겠습니다.

$W$, $X$, $Y$, $Z$는 네 개의 공선점이며, $|WX| = a$, $|XY| = b$, $|YZ| = c$, 그리고 $|WZ| = a + b + c$입니다.

$C_{\text{in}}$은 지름이 $XY$인 원입니다.

$C_{\text{out}}$은 지름이 $WZ$인 원입니다.

$(a, b, c)$가 *목걸이 삼중항*이라고 불리는 조건은, 서로 다른 원 $C_1, C_2, \ldots, C_k$를 $k ≥ 3$개 배치할 수 있을 때입니다. 이 원들은 다음 조건을 만족해야 합니다:

- $1 ≤ i, j ≤ k$이고 $i ≠ j$일 때, $C_i$와 $C_j$는 내부 점을 공유하지 않습니다.
- $1 ≤ i ≤ k$일 때, $C_i$는 $C_{\text{in}}$과 $C_{\text{out}}$에 모두 접합니다.
- $1 ≤ i < k$일 때, $C_i$는 $C_{i + 1}$에 접합니다.
- $C_k$는 $C_1$에 접합니다.

예를 들어, (5, 5, 5)와 (4, 3, 21)은 목걸이 삼중항이지만, (2, 2, 5)는 그렇지 않다는 것을 증명할 수 있습니다.

<img alt="목걸이 삼중항의 시각적 표현" src="https://cdn.freecodecamp.org/curriculum/project-euler/necklace-of-circles.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$T(n)$을 $a$, $b$, $c$가 양의 정수이고 $b ≤ n$인 목걸이 삼중항 $(a, b, c)$의 개수라고 하겠습니다. 예를 들어, $T(1) = 9$, $T(20) = 732$, 그리고 $T(3\,000) = 438\,106$입니다.

$T(1\,000\,000\,000)$을 구하세요.
