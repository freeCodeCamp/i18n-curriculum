# --description--

En inglés, puedes usar el verbo `to be` para hacer preguntas sobre sentimientos, estados o condiciones. Cuando quieres hacer tales preguntas, simplemente intercambias las posiciones del sujeto (como `you` o `he`) y el verbo `to be` (como `are` o `is`). Por ejemplo, para convertir esta oración `You are a developer.` en una pregunta, intercambias `you` y `are`:

`Are you a developer?` Observa cómo cambia el orden para hacer una pregunta.

Aquí tienes algunos ejemplos más de cómo convertir oraciones en preguntas usando `to be`:

| Afirmativa        | Interrogativa     |
| ----------------- | ----------------- |
| `He is here.`     | `Is he here?`     |
| `They are ready.` | `Are they ready?` |
