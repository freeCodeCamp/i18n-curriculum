# --explanation--

`However` é usado para reconhecer um ponto de vista oposto antes de apresentar um contra-argumento.Essa estrutura torna os desacordos mais equilibrados e respeitosos nas discussões.Por exemplo:

- `The app is user-friendly. However, it still has some performance issues.` - O aplicativo é fácil de usar, mas há problemas de velocidade.

- `The software update improved security. However, it introduced a few new bugs.` - A atualização tornou o sistema mais seguro, mas também criou alguns problemas.
