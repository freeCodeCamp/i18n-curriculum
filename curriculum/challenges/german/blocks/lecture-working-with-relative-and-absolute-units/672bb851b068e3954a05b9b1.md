# --interactive--

Mit der `calc()`-Funktion können Sie Berechnungen direkt in Ihren Stylesheets durchführen, um Eigenschaftswerte dynamisch zu bestimmen. Das bedeutet, dass Sie flexible und responsive Benutzeroberflächen erstellen können, indem Sie Maße basierend auf der Viewport-Größe oder anderen Elementen berechnen.

`calc()` ist eine CSS-Funktion. Mehr über Funktionen lernen Sie, wenn Sie mit JavaScript beginnen, aber in dieser Lektion lernen Sie die Grundlagen, die Sie benötigen, um zu verstehen, wie `calc()` funktioniert.

Eine Funktion ist ein Codeblock, der eine bestimmte Aufgabe ausführt. Einige Funktionen sind bereits in CSS definiert, sodass Sie sie direkt verwenden und die notwendigen Werte übergeben können, um anzupassen, wie ihre Aufgaben ausgeführt werden.

In der Programmierwelt sagen wir, dass wir eine Funktion „aufrufen“, wenn wir die von ihr ausgeführte Aufgabe starten. Die Werte, die wir an die Funktion übergeben, nennt man Argumente.

Wie Sie im folgenden Code sehen, schreiben Sie zum Aufrufen einer Funktion ihren Namen, gefolgt von den Argumenten in runden Klammern, getrennt durch Kommas. Zwischen dem Funktionsnamen und der öffnenden Klammer darf kein Leerzeichen stehen:

```css
function(argument1, argument2, argument3)
```

Eine Funktion benötigt möglicherweise nur einen Wert, um zu wissen, was zu tun ist. In diesem Fall nimmt sie nur ein Argument. So ist es bei der `calc()`-Funktion. Sie nimmt ein Argument, weil sie wissen muss, was berechnet werden soll.

Dafür übergeben Sie ein sogenanntes Ausdruck als Argument. Ein Ausdruck ist eine Kombination aus Werten und Operatoren, die ein Ergebnis liefert.

So können Sie die `calc()`-Funktion aufrufen. Sie schreiben den Namen calc, gefolgt von runden Klammern, und innerhalb der Klammern schreiben Sie den Ausdruck:

```css
calc(expression)
```

Der Ausdruck wird ausgewertet, um das endgültige Ergebnis zu berechnen. „Ausgewertet“ bedeutet einfach, dass die Werte und Operatoren hinter den Kulissen in einen einzelnen Wert umgewandelt werden. Das Ergebnis wird der CSS-Eigenschaft zugewiesen, bei der die Berechnung erfolgt.

Sie können Berechnungen mit Werten durchführen, die Länge, Winkel, Zeit, Prozentsätze, Zahlen und Farben darstellen. Sie können auch verschiedene Einheiten wie Pixel, Prozentsätze und em kombinieren.

Bei Zahlen müssen alle Werte im Ausdruck, auch Operanden genannt, ihre entsprechenden Einheiten haben, wie `px`, `em` und Prozent (`%`). Je nach Operator können verschiedene Operanden unterschiedliche Einheiten haben.

Sie können die Operatoren Addition (`+`), Subtraktion (`-`), Multiplikation (`*`) und Division (`/`) im Ausdruck verwenden.

Wenn es mehrere Operanden und Operatoren gibt, folgt `calc()` der Standardregel der Operatorpriorität. Sie können auch Klammern hinzufügen, um die Reihenfolge der Operationen festzulegen, falls nötig.

Im folgenden Beispiel sehen Sie ein `div` mit dem Text `Hello, World!`.

Mit dem CSS-Typselektor zur Auswahl des `div` können Sie es mit weißem Text und einem dunkelblauen Hintergrund gestalten:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>Hello, World!</div>
```

```css
div {
  color: white;
  background-color: #1b1b32;
  width: calc(50% - 20px);
}
```

:::

Neu hier ist, dass die Breite dynamisch berechnet wird. Beachten Sie, wie wir die `calc()`-Funktion aufrufen und einen Ausdruck als Argument übergeben. Der Ausdruck hat zwei Operanden mit unterschiedlichen Einheiten und einen Operator, den Subtraktionsoperator.

Prozent ist eine relative Einheit. Der Wert (`50%`) wird durch die Breite des übergeordneten Containers bestimmt. Dann wird `20px` vom Wert subtrahiert. Das Ergebnis dieses Ausdrucks bestimmt die Breite des `div`.

Die Breite des `div` beträgt ungefähr die Hälfte der Gesamtbreite seines Containers, und wenn Sie den übergeordneten Container in der Größe ändern, wird die Breite automatisch neu berechnet.

Das ist der entscheidende Vorteil der Verwendung von `calc()`. Sie können den Wert einer CSS-Eigenschaft dynamisch basierend auf verschiedenen Aspekten der Anwendung oder des Viewports bestimmen.

Der Ausdruck kann auch CSS-Funktionen und Variablen enthalten, wenn Sie diese in Ihren Berechnungen verwenden müssen. Mehr über CSS-Variablen lernen Sie in den nächsten Lektionen.

Prima. Jetzt, wo Sie die Grundlagen der `calc()`-Funktion kennen, besprechen wir einige bewährte Vorgehensweisen.

Zuerst müssen Sie die Operatoren Addition (`+`) und Subtraktion (`-`) mit Leerzeichen umgeben.

Zum Beispiel wäre der folgende Ausdruck kein gültiger Ausdruck, weil der Subtraktionsoperator direkt vor dem zweiten Operanden steht.

```css
calc(100% -30px)
```

Der Subtraktionsoperator (`-`) muss von Leerzeichen umgeben sein, so wie hier. Das Hinzufügen der Leerzeichen macht den Ausdruck gültig.

```css
calc(100% - 30px)
```

Das ist für die Multiplikations- und Divisionsoperatoren nicht notwendig, wird aber dringend empfohlen.

Sie können auch verschachtelte `calc()`-Funktionsaufrufe verwenden, wenn Sie Berechnungen durchführen und diese Ergebnisse in anderen Berechnungen verwenden müssen.

Außerdem müssen Sie, wenn Sie den Wert null zur Darstellung von Länge im Ausdruck verwenden, den Sie an die `calc()`-Funktion übergeben, die Einheiten angeben. Zum Beispiel wäre dieser Ausdruck nicht gültig:

```css
calc(100% - 0)
```

Sie müssten die Einheiten hinzufügen, zum Beispiel px.

```css
calc(100% - 0px)
```

Sie sollten auch wissen, dass aktuell, wenn Sie die Multiplikations- oder Divisionsoperatoren verwenden, einer der Operanden keine Einheit haben darf. Beim Divisionsoperator muss speziell der rechte Operand einheitenlos sein. Dieser Ausdruck wäre nicht gültig, weil beide Operanden Einheiten haben (Pixel). Einer der Operanden, entweder 5 oder 50, muss einheitenlos sein:

```css
calc(5px * 50px)
```

Sie müssten die Einheit bei einem der beiden weglassen. Beide Alternativen wären gültig:

```css
calc(5 * 50px)
calc(5px * 50)
```

Und hier ist ein Beispiel mit dem Divisionsoperator. Dieser Ausdruck wäre nicht gültig, da beide Operanden Einheiten haben:

```css
calc(50% / 5%)
```

Sie sollten die Einheit beim rechten Operanden entfernen, wenn Sie den Divisionsoperator verwenden:

```css
calc(50% / 5)
```

Die `calc()`-Funktion kann für Sie als Webentwickler sehr hilfreich sein. Mit dieser Funktion können Sie Eigenschaftswerte dynamisch festlegen, um flexible und responsive Designs zu erstellen.
