# --description--

No momento, a largura do elemento `div` abrange toda a largura da página.Para aplicar larguras no Tailwind CSS, você tem algumas opções, como usar o padrão `w-number` para larguras fixas ou o padrão `w-fraction` para porcentagens.  

Aqui está um exemplo de aplicação de uma largura definida para `66%` do seu contêiner pai:

```html
<p class="w-2/3">This is a percentage width.</p>
```

Dentro da sua tag de abertura `div`, aplique uma largura definida para `50%` do seu contêiner pai.

# --hints--

Sua tag de abertura `div` deve ter a classe utilitária `w-1/2`.

```js
const divEl = document.querySelector("div");
assert.isTrue(divEl.classList.contains("w-1/2"));
```
