# --explanation--

`Might` é usado para falar sobre possibilidade — algo que pode acontecer mas não é certo. Por exemplo:

- `It might rain tomorrow.` - Há uma chance, mas não é garantido.

- `We might need more time to finish the project.` - Há uma possibilidade, mas não está confirmado.
