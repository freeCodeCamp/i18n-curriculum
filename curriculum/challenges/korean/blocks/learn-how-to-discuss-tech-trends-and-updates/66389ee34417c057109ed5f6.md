# --description--

`automate`는 사람이 도와주지 않아도 스스로 작동하도록 만드는 것을 의미합니다. 프로그래밍에서는 소프트웨어를 사용해 코드를 작성하거나 애플리케이션을 테스트하거나 오류를 찾아 고치는 작업이 포함될 수 있습니다. 예를 들어, 아침에 스스로 켜져서 커피를 만드는 커피 머신은 `automated`입니다.
