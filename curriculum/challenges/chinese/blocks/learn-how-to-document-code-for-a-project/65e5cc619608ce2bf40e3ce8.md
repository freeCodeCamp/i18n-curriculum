# --description--

在这个任务中，你将学习诸如 `Did you get it?`、`Got it` 和 `What else?` 这样的短语。这些短语在英语对话中很常见。`Did you get it?` 是一个询问某人是否理解某事的问题。`Got it` 是 `I understood.` 的简短表达。`What else?` 用于询问额外的信息或项。

例如，在会议中，有人可能会说，`Did you get it?`，你可以回复，`Got it, what else do we need to discuss?`
