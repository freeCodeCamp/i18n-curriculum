# --explanation--

`Past Continuous` 時態（`was/were` + 動詞的 `-ing` 形式）用來描述過去一段時間內發生的動作或想法，通常與另一個事件同時發生。它可以表示某人當時已經在思考相同的想法。<dfn>例如</dfn>：

`I was reading the report when you called.`－這表示在發生呼叫時，閱讀的動作正在進行中。

在這種情況下，`I was thinking the same thing` 意味著 Mark 的想法發生在 Lisa 的評論期間或之前，並表示同意。
