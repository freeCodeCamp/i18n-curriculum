# --description--

El elemento de la cabeza de la tabla consiste en un elemento <dfn>fila de tabla</dfn>, `tr`, que contiene los elementos <dfn>celda de encabezado de tabla</dfn>, `th`.

Aquí hay un ejemplo que utiliza los elementos `tr` y `th` para una tabla de deportes:

```html
<table>
  <caption>Football Scores</caption>
  <thead>
    <tr>
      <th>Team</th>
      <th>Wins</th>
      <th>Losses</th>
    </tr>
  </thead>
</table>
```

Dentro de tu elemento `thead`, añade un elemento `tr`.

Dentro de tu elemento `tr`, añade tres elementos `th`.

El primer elemento `th` debe contener el texto `Last Name`. El segundo elemento `th` debe contener el texto `First Name`. El tercer elemento `th` debe contener el texto `Grade`.
