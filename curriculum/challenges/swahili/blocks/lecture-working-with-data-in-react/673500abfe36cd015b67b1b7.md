# --description--

Onyesho la masharti katika React linakuwezesha kuunda interface za mtumiaji zinazobadilika. Linatumika kuonyesha maudhui tofauti kulingana na masharti au hali fulani ndani ya programu yako.

Njia zinazotumika zaidi za onyesho la masharti ni pamoja na kutumia tamko la `if`, opereta ya kikokotoo cha masharti cha pande tatu (`?:`), na opereta ya mantiki AND (`&&`).

Aina rahisi zaidi ya onyesho la masharti hutumia tamko la `if`. Hapa kuna mfano:

```jsx
function Greeting({ isLoggedIn }) {
  if (isLoggedIn) {
    return <h1>Welcome back!</h1>;
  }
  return <h1>Please sign in</h1>;
}
```

Katika mfano huu, sehemu ya `Greeting` inakagua vigezo vya `isLoggedIn`. Ikiwa ni `true`, inarudisha ujumbe wa kukaribisha, vinginevyo, inahimiza mtumiaji kuingia.

Hapa kuna mfano unaotumia sehemu ya `Greeting` ndani ya sehemu mama ya `App`:

```jsx
function App() {
  return (
    <div className="App">
      <Greeting isLoggedIn={false} />
    </div>
  );
}
```

Kwa masharti rahisi, opereta ya kikokotoo cha masharti cha pande tatu (`?:`) mara nyingi hutumika moja kwa moja ndani ya JSX. Hii inaruhusu onyesho la masharti ndani ya mstari, ambalo linaweza kufanya msimbo wako kuwa mfupi zaidi:

```jsx
function Greeting({ isLoggedIn }) {
  return <h1>{isLoggedIn ? "Welcome back!" : "Please sign in."}</h1>;
}
```

Msimbo huu unafanikisha matokeo sawa na mfano uliotangulia lakini kwa njia iliyoshikana zaidi. Opereta ya ternary inakagua `isLoggedIn` na inaonyesha ujumbe unaofaa.

Mfumo mwingine wa kawaida wa onyesho la masharti ni kutumia opereta ya mantiki AND (`&&`). Hii ni muhimu hasa unapotaka kuonyesha kitu, au kutokuonyesha chochote, kulingana na sharti:

```jsx
function Notification({ message }) {
  return (
    <div>
      {message && <p>{message}</p>}
    </div>
  );
}
```

Katika mfano huu, kipengele cha aya chenye ujumbe kinaonyeshwa tu ikiwa vigezo vya `message` ni kweli. Ikiwa `message` ni thamani ya uongo - maana yake ni mfuatano wa herufi tupu, `null`, au `undefined`, hakuna kinachoonyeshwa kwenye skrini.

Kwa kumudu mbinu hizi za onyesho la masharti, unaweza kujenga programu zinazoshirikisha zaidi na rafiki kwa mtumiaji ambazo zinabadilika kulingana na data na mwingiliano wa mtumiaji.
