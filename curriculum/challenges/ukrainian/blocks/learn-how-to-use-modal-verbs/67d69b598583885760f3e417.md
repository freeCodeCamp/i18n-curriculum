# --explanation--

`Might` використовується, щоб говорити про ймовірність — щось, що може статися, але не є певним. Наприклад:

- `It might rain tomorrow.` — Є шанс, але це не гарантовано.

- `We might need more time to finish the project.` — Існує ймовірність, але це не підтверджено.
