# --description--

In diesem Workshop lernen Sie, wie Sie mit den Grundlagen von JavaScript arbeiten, indem Sie einen Begrüßungs-Bot erstellen.

In diesem ersten Schritt sollen Sie eine Nachricht von dem Begrüßungs-Bot in der Konsole ausgeben.

Denken Sie daran, dass Sie in vorherigen Lektionen etwas über `console.log()` und Strings gelernt haben.

Hier eine Erinnerung, wie man `console.log()` mit Strings verwendet:

```js
console.log("Hello, World!");
```

Fügen Sie eine `console.log()`-Anweisung hinzu, die den String `"Hi there!"` in der Konsole ausgibt. Vergessen Sie nicht die Anführungszeichen um die Nachricht!


# --hints--

Ihr Code sollte eine `console.log()`-Anweisung enthalten.

```js
assert.match(__helpers.removeJSComments(code), /console\.log(.*)/);
```

Ihre `console`-Anweisung sollte die Nachricht `"Hi there!"` enthalten. Vergessen Sie die Anführungszeichen nicht.

```js
assert.match(__helpers.removeJSComments(code), /console\.log\((['"])Hi\s+there!\1\);?/);
```
