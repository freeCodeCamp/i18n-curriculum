# --description--

```css
/* rule 1 */
.class.second-class {
  font-size: 12px;
}

/* rule 2 */
.class .second-class {
  font-size: 24px;
}
```

这里规则 1 和规则 2 具有相同的特异性。规则 1 使用链式选择器（无空格），规则 2 使用后代组合符号（空格）。但两个规则都有两个类，组合符号本身不增加特异性。

```css
/* rule 1 */
.class.second-class {
  font-size: 12px;
}
/* rule 2 */
.class > .second-class {
  font-size: 24px;
}
```

此示例展示了相同的内容。即使规则 2 使用了子组合器（`>`），这也不会改变规格值。两个规则仍然都有两个类，因此它们具有相同的规格值。

*注意：* 并非所有内容都会添加到特异性中

在比较选择器时，你可能会遇到通用选择器（`*`）以及组合符号（`+`、`~`、`>` 和空格）。这些符号本身不会添加任何特异性。
