# --description--

设 $s_k$ 为将数字从 0 写到 $k$ 的二进制中 1 的个数。

例如，将 0 到 5 写成二进制，我们有 0、1、10、11、100、101。有七个 1，因此 $s_5 = 7$。

序列 $S = \{s_k : k ≥ 0\}$ 以 $\{0, 1, 2, 4, 5, 7, 9, 12, \ldots\}$ 开始。

一个游戏由两个玩家进行。在游戏开始前，选择一个数字 $n$。计数器 $c$ 从 0 开始。每回合，玩家选择一个从 1 到 $n$（含）之间的数字，并将 $c$ 增加该数字。$c$ 的结果值必须是集合 $S$ 的成员。如果没有更多有效的移动，玩家则输掉游戏。

例如，$n = 5$ 且从 $c = 0$ 开始：

- 玩家 1 选择 4，因此 $c$ 变为 $0 + 4 = 4$。
- 玩家 2 选择 5，因此 $c$ 变为 $4 + 5 = 9$。
- 玩家 1 选择 3，所以 $c$ 变成 $9 + 3 = 12$。
- 等等。

请注意，$c$ 必须始终属于 $S$，且每个玩家最多可以将 $c$ 增加 $n$。

设 $M(n)$ 为第一位玩家在她的第一回合中可以选择的最高数字以迫使获胜的数字，如果不存在这样的移动，则 $M(n) = 0$。例如，$M(2) = 2$，$M(7) = 1$ 和 $M(20) = 4$。

可以验证 $\sum M{(n)}^3 = 8150$ 对于 $1 ≤ n ≤ 20$。

查找 $\sum M{(n)}^3$，其中 $1 ≤ n ≤ 1000$。
