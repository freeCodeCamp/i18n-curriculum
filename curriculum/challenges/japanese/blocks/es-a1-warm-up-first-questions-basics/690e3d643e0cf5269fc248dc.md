# --explanation--

`Salvadoreña`という言葉は、エルサルバドル出身の人の女性形の国籍です。

国籍は、その人の性別に合わせて変わります。例えば：

| 性別 | 国籍 | 例文 |
| :--- | :--- | :--- |
| **女性形** | `Salvadoreña` | `Ella es salvadoreña.`（「彼女はエルサルバドル人です」。） |
| **男性形** | `Salvadoreño` | `Él es salvadoreño.`（「彼はエルサルバドル人です」。） |
