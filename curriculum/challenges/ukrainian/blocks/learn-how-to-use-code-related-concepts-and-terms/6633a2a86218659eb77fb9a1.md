# --description--

Модальне дієслово `can` використовується, щоб вказати на можливість або здатність. Наприклад, у реченні `You can see the stars at night` `can` показує, що можливо побачити зірки вночі.

`Change` означає зробити або стати іншим. У повсякденному житті ви можете `change` свій одяг після роботи, що означає, що ви переходите з робочого одягу на повсякденний.

`Stay the same` означає залишатися незмінним або постійним. Наприклад, `The rules of the game stay the same every year.` Це означає, що правила не змінюються з року в рік.
