# --explanation--

`True` é um adjetivo que significa que algo está correto, é real ou preciso. É usado para concordar com uma afirmação ou fato. É diferente de `the truth`, que é um substantivo que se refere aos fatos reais ou à realidade sobre algo, frequentemente usado em um sentido mais amplo ou mais abstrato. Por exemplo:

- `It's true that the server crashed yesterday.` – Isso significa que a afirmação sobre a falha está correta.

- `The truth is that we didn't have a backup.` – Isso significa que o fato real é que nenhum backup existia.
