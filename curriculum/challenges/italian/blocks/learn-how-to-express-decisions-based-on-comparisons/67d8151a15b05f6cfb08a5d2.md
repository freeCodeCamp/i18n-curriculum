# --explanation--  

`That way` si usa per spiegare il risultato o il beneficio di una decisione o azione. Per esempio:

`We should automate testing. That way, we can catch bugs faster.` – `That way` indica che l'automazione aiuterà a trovare i bug più velocemente.  

`While` si usa per indicare che due cose stanno succedendo contemporaneamente. Per esempio:  

`The team is fixing bugs while the designers work on the UI.` – Questo significa che entrambe le attività stanno succedendo nello stesso momento.
