# --explanation--

`Code reuse` bezeichnet die Praxis, vorhandenen Code in neuen Teilen eines Programms zu verwenden. Zum Beispiel:

`The "Login" class was used in multiple applications to handle user access, which is a good example of code reuse.` – Anstatt die Login-Logik für jede App neu zu schreiben, wird derselbe Code wiederverwendet, was die Effizienz verbessert.

`Extending` bedeutet, etwas, das bereits existiert, um neue Funktionen zu erweitern. Zum Beispiel:

`The "AdminUser" class extends the basic "User" class by adding an "approveUser()" method.` – Die `AdminUser` übernimmt alles von `User` und baut darauf mit neuen Fähigkeiten auf.
