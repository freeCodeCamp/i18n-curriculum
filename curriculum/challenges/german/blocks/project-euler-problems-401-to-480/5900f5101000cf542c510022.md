# --description--

Die Look-and-say-Folge lautet 1, 11, 21, 1211, 111221, 312211, 13112221, 1113213211, ...

Die Folge beginnt mit 1, und alle weiteren Elemente werden gebildet, indem das vorherige Element in aufeinanderfolgenden Ziffern beschrieben wird.

Es hilft, dies laut auszusprechen:

1 ist „eine Eins“ $→ 11$

11 ist „zwei Einsen“ $→ 21$

21 ist „eine Zwei und eine Eins“ $→ 1211$

1211 ist „eine Eins, eine Zwei und zwei Einsen“ $→ 111221$

111221 ist „drei Einsen, zwei Zweien und eine Eins“ $→ 312211$

...

Definieren Sie $A(n)$, $B(n)$ und $C(n)$ als die Anzahl der Einsen, Zweien und Dreien im $n$-ten Element der Folge. Man kann überprüfen, dass $A(40) = 31\\,254$, $B(40) = 20\\,259$ und $C(40) = 11\\,625$ gilt.

Finden Sie $A(n)$, $B(n)$ und $C(n)$ für $n = {10}^{12}$. Geben Sie Ihre Antwort modulo $2^{30}$ als String an und trennen Sie Ihre Werte für $A$, $B$ und $C$ durch ein Komma. Zum Beispiel wäre die Antwort für $n = 40$ `31254,20259,11625`.
