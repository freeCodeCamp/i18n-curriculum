# --explanation--

`Get used to` significa se adaptar a algo novo. It is usually followed by a noun or the `-ing` form.Por exemplo:

- `New remote employees need time to get used to asynchronous communication.` - Trabalhadores remotos podem ter dificuldades no começo, mas eventualmente eles irão se adaptar a essa forma de trabalho.

- `I can't get used to working without dual monitors.` - O falante acha difícil se adaptar a usar apenas uma tela.

Anna está dizendo que o trabalho remoto assíncrono pode parecer estranho no começo, mas com o tempo as pessoas se adaptam e descobrem que funciona bem.
