# --explanation--

Mateo pregunta `¿Dónde vive Valeria?`, que significa "¿Dónde vive Valeria?"

`¿Dónde...?` significa "¿Dónde...?"

Recuerda incluir el signo de interrogación de apertura `¿` y la tilde en la letra `ó` cuando esta palabra inicia una pregunta.

La palabra `vive` significa "vive", la tercera persona del singular del verbo `vivir`.
