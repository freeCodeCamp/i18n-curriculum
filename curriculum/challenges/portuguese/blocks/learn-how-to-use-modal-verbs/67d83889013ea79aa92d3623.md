# --explanation--

Um `full scan` é uma verificação de segurança abrangente que examina um sistema ou rede inteira em busca de ameaças.Por exemplo:

- `You should run a full scan to check for viruses.` - É uma boa ideia escanear todo o sistema em busca de arquivos nocivos.

- `The IT team performs a full scan of the network every month.` - A equipe técnica verifica todos os sistemas de rede em busca de problemas de segurança uma vez por mês.
