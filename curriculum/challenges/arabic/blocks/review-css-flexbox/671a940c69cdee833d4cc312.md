## مقدمة إلى CSS Flexbox ونموذج المرونة

- **التعريف**: CSS flexbox هو نموذج تخطيط أحادي البعد يسمح لك بترتيب العناصر في صفوف وأعمدة داخل حاوية.
- **نموذج المرونة**: يحدد هذا النموذج كيفية ترتيب العناصر المرنة داخل الحاوية المرنة. لكل حاوية مرنة محوران: المحور الرئيسي والمحور المتقاطع.

## الخاصية `flex-direction`

- **التعريف**: تحدد هذه الخاصية اتجاه المحور الرئيسي. القيمة الافتراضية لـ `flex-direction` هي `row`، التي تضع كل العناصر المرنة في نفس الصف، باتجاه لغة المتصفح الافتراضية (من اليسار إلى اليمين أو من اليمين إلى اليسار).
- **`flex-direction: row-reverse;`**: يعكس ترتيب العناصر في الصف.
- **`flex-direction: column;`**: يصطف العناصر المرنة عموديًا بدلاً من أفقيًا.
- **`flex-direction: column-reverse;`**: يعكس ترتيب العناصر المرنة عموديًا.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: row;
  gap: 10px;
}

.box {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}
```

:::

## الخاصية `flex-wrap`

- **التعريف**: تحدد هذه الخاصية كيفية التفاف العناصر المرنة داخل الحاوية المرنة لتناسب المساحة المتاحة. يمكن أن تأخذ `flex-wrap` ثلاث قيم ممكنة: `nowrap`، `wrap`، و`wrap-reverse`.
- **`flex-wrap: nowrap;`**: هذه هي القيمة الافتراضية. لا تُلف العناصر المرنة إلى سطر جديد حتى لو تجاوز عرضها عرض الحاوية.
- **`flex-wrap: wrap;`**: تلف هذه الخاصية العناصر عندما تتجاوز عرض الحاوية.
- **`flex-wrap: wrap-reverse;`**: تلف هذه الخاصية العناصر المرنة بترتيب عكسي.
- **الخاصية `flex-flow`**: هي خاصية مختصرة لـ `flex-direction` و`flex-wrap`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
  <div class="box">4</div>
  <div class="box">5</div>
</div>
```

```css
.container {
  display: flex;
  flex-wrap: wrap;
  width: 150px;
  background: #f0f0f0;
}

.box {
  width: 60px;
  padding: 10px;
  margin: 5px;
  background: skyblue;
  text-align: center;
}
```

:::

## الخاصية `justify-content`

- **التعريف**: تستخدم هذه الخاصية لمحاذاة العناصر الفرعية على طول المحور الرئيسي للحاوية المرنة.
- **`justify-content: flex-start;`**: تصطف العناصر المرنة عند بداية المحور الرئيسي، سواء كان أفقيًا أو عموديًا.
- **`justify-content: flex-end;`**: تصطف العناصر المرنة عند نهاية المحور الرئيسي، أفقيًا أو عموديًا.
- **`justify-content: center;`**: توسيط العناصر المرنة على طول المحور الرئيسي.
- **`justify-content: space-between;`**: توزع العناصر بالتساوي على طول المحور الرئيسي.
- **`justify-content: space-around;`**: توزع العناصر بالتساوي داخل المحور الرئيسي مع إضافة فراغ قبل العنصر الأول وبعد الأخير.
- **`justify-content: space-evenly;`**: توزع العناصر بالتساوي على طول المحور الرئيسي.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box">A</div>
  <div class="box">B</div>
  <div class="box">C</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
  background: #eee;
}

.box {
  padding: 20px;
  background: salmon;
}
```

:::
  
## الخاصية `align-items`

- **التعريف**: تستخدم هذه الخاصية لتوزيع العناصر على طول المحور المتقاطع. تذكر أن المحور المتقاطع عمودي على المحور الرئيسي.
- **`align-items: center;`**: تستخدم لمركزة العناصر على طول المحور المتقاطع.
- **`align-items: flex-start;`**: تصطف العناصر عند بداية المحور المتقاطع.
- **`align-items: stretch;`**: تستخدم لتمديد العناصر المرنة على طول المحور المتقاطع.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box tall">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  align-items: center;
  height: 150px;
  background: #ddd;
}

.box {
  background: lightgreen;
  padding: 10px;
}

.tall {
  height: 100px;
}
```
