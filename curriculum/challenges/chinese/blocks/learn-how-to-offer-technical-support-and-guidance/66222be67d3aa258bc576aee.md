# --description--

表达式 `first things first` 用于强调重要的事情应该先于其他任何事情完成。它通常在任务开始时说，以便首先关注最重要的步骤。例如，`First things first, let's review the project plan` 意味着在进行其他任务之前先审查项目计划。

在技术上下文中，`environment` 通常指系统运行的一组软件和硬件条件。例如，`development environment` 是程序员编写和测试他们代码的地方。
