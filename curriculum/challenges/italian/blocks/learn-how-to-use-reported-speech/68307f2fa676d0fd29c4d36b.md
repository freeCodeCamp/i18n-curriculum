# --explanation--

`Was there` è la forma interrogativa (domanda) di `There was`, usata per chiedere se qualcosa è esistito o è accaduto nel passato. Inverte il soggetto e il verbo per formare una domanda. Per esempio:

- `There was a problem with the layout.` (affermazione - stai dicendo che è successo un problema)  

- `Was there a problem with the layout?` (domanda - stai chiedendo se è successo un problema)

Questo aiuta a chiedere della presenza o dell'occorrenza di qualcosa nel passato.
