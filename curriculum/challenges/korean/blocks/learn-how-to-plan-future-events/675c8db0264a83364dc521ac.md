# --explanation--

`I'll`는 `I will`의 축약형입니다. 이 문맥에서는 말하는 순간에 내린 결정을 표현합니다. 누군가 상황에 반응하거나 즉시 무언가를 하기로 동의할 때 흔히 사용됩니다.

`Now`는 현재 순간에 일어나는 일을 나타내는 데 사용됩니다. 예를 들어:

`I'll send the report now.` - 즉시 행동하기로 한 결정을 보여줍니다.
