# --explanation--

`Specify` означає чітко вказати або описати щось детально. Наприклад:

`Please specify the time and place of the meeting.` – Це означає сказати точно, коли і де відбудеться зустріч.

`What makes it` — це іменна умова. Група слів, яка виконує роль іменника в реченні. Вона пояснює причину або підставу чогось. Наприклад:

`She described what makes it so effective.` – Це означає, що вона пояснила причину, чому щось ефективне. Тут умова `what makes it so effective` виконує роль додатка до дієслова `described`.
