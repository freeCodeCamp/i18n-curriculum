# --interactive--

CSSのバックグラウンドグラデーションは、要素の背景に適用できる2色以上の色の滑らかな移行です。グラデーションを使うことで、画像を使わずに視覚的に魅力的な背景を作成できます。

CSSには主に2種類のグラデーションがあります：リニアグラデーションとラジアルグラデーションです。

リニアグラデーションは直線に沿って色が変化します。方向や使用する色を指定できます。

基本的な構文は以下の通りです：

```css
background: linear-gradient(direction, color-stop1, color-stop2, ...);
```

この例では、`background` CSSプロパティに`linear gradient`の値を使っています。

方向はグラデーションの向きを指定します。角度（例えば`45deg`）、キーワード（例えば`to right`、`to bottom`）、または辺や角を指定できます。

`color-stop`はグラデーションの色と色の変化位置を指定します。

リニアグラデーションの仕組みをよりよく理解するために、次の例を見てみましょう：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient{
  background: linear-gradient(to right, red, yellow);
  height: 40vh;
}
```

:::

このCSSは、`red`の`left`から`yellow`の`right`へと変化するリニアグラデーションを作成します。グラデーションはビューポートの高さの`40%`の高さを持つ要素に適用されています。`vh`単位については今後のレッスンで詳しく学びます。

`to right`の方向は、グラデーションが左から右へ水平に流れることを意味します。

もう一つのグラデーションの種類は`radial`グラデーションです。

ラジアルグラデーションは、起点（通常は中央）から放射状に色が円形または楕円形に広がります。

基本的な構文は以下の通りです：

```css
background: radial-gradient(shape size at position, color-stop1, color-stop2, ...)
```

構文では、`shape`がグラデーションの形状を指定し、`circle`または`ellipse`のいずれかになります。

`size`はグラデーションの終端形状のサイズを決め、`closest-side`、`closest-corner`、`farthest-side`、または`farthest-corner`のいずれかです。

`position`はグラデーションの中心位置を決め、キーワード（例えば`center`、`top left`、`bottom right`）や正確な値（例えば`50% 50%`、`10px 20px`）で指定できます。

最後に、カラーストップはグラデーションが通過する色のリストです。各カラーストップは、色を配置する位置（パーセンテージや長さ）をオプションで含めることができます。

例は以下の通りです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient{
  background: radial-gradient(circle closest-side at center, red, yellow 50%, green);
  height: 60vh;
}
```

:::

このCSSは、要素の中央に円形のラジアルグラデーションを作成します。中心で`red`から始まり、半径の`yellow`で`50%`に変化し、`green`で終わります。

`closest-side`キーワードはグラデーションの終端形状を要素の最も近い辺に合わせます。グラデーションはビューポートの高さの`60%`の高さを持つ要素に適用されています。

CSSグラデーションの使い方を理解すると、画像を使わずに視覚的に魅力的な背景を提供できるため、デザインの幅が大きく広がります。

リニアグラデーションの滑らかな色の移行やラジアルグラデーションの円形効果などのオプションにより、ウェブデザインに柔軟性と創造性をもたらします。
