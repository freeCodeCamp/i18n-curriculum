# --description--

**Ziel:** Erstellen Sie eine Anwendung, die funktional ähnlich ist wie diese: <a href="https://build-a-pinterest-clone.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://build-a-pinterest-clone.freecodecamp.rocks/</a>.

Erfüllen Sie die untenstehenden User Stories und bestehen Sie alle Tests. Verwenden Sie beliebige Bibliotheken oder APIs, die Sie benötigen. Verleihen Sie der Anwendung Ihren persönlichen Stil.

**User Story:** Als nicht authentifizierter Benutzer kann ich mich mit GitHub anmelden.

**User Story:** Als authentifizierter Benutzer kann ich Bilder verknüpfen.

**User Story:** Als authentifizierter Benutzer kann ich Bilder löschen, die ich verknüpft habe.

**User Story:** Als authentifizierter Benutzer kann ich eine Pinterest-ähnliche Wand mit allen Bildern sehen, die ich verknüpft habe.

**User Story:** Als nicht authentifizierter Benutzer kann ich die Bildwände anderer Benutzer durchsuchen.

**User Story:** Als authentifizierter Benutzer wird ein von mir hochgeladenes defektes Bild durch ein Platzhalterbild ersetzt. (jQuery Broken-Image-Erkennung kann verwendet werden)

**Hinweis:** Masonry.js ist eine Bibliothek, die Pinterest-ähnliche Bildraster ermöglicht.

Sobald Sie diese User Stories implementiert haben, geben Sie die URL Ihrer Live-Anwendung und optional Ihr GitHub-Repository ein. Klicken Sie dann auf die Schaltfläche „Ich habe diese Aufgabe abgeschlossen“.

Sie können Feedback zu Ihrem Projekt erhalten, indem Sie es im <a href="https://forum.freecodecamp.org/c/project-feedback/409" target="_blank" rel="noopener noreferrer nofollow">freeCodeCamp-Forum</a> teilen.
