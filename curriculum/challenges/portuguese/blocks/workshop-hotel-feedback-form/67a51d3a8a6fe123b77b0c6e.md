# --description--

Elementos `input` podem ter um atributo `size`. Este atributo define o número de caracteres que devem estar visíveis enquanto o usuário digita no input. O valor de `size` deve ser um inteiro não negativo maior que zero. Se `size` não for especificado ou for especificado com um valor inválido, a entrada terá a largura padrão definida pelo navegador.

```html
<label for="lastName">Last Name:</label>
<input id="lastName" name="lastName" type="text" size="10" />
```

Dê aos inputs de nome e email um atributo `size` com valor `"20"`.
