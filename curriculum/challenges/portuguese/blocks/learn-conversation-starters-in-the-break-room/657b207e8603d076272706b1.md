# --description--

O sufixo `-ly` pode ser adicionado às palavras para indicar a frequência de um evento.Aqui está como isso se aplica a diferentes períodos de tempo:

- `Daily` significa que ocorre uma vez a cada dia.
- `Weekly` indica algo que acontece uma vez por semana.
- `Monthly` descreve algo que acontece uma vez a cada mês. 
- `Yearly` (ou `annually`) descreve algo que acontece uma vez a cada ano.
