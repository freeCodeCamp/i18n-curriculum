# --description--

이전 수업에서 문자열의 각 문자가 인덱스(0부터 시작)로 식별되고 대괄호 표기법을 사용해 접근할 수 있다는 것을 배웠습니다:

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
print(my_str[-1]) # d
```

**문자열 슬라이싱**은 문자열의 일부를 추출하거나 특정 부분만 다룰 수 있게 해줍니다. 기본 구문은 다음과 같습니다:

```python
string[start:stop]
```

특정 인덱스부터 다른 인덱스까지 문자를 추출하려면 `start`와 `stop` 인덱스를 콜론으로 구분하세요:

```python
my_str = 'Hello world'
print(my_str[1:4]) # ell
```

`stop` 인덱스는 포함되지 않는다는 점에 유의하세요. 그래서 `[1:4]`는 `1` 인덱스부터 `4` 인덱스 바로 전까지의 문자를 추출했습니다.

`start`와 `stop` 인덱스를 생략할 수도 있는데, 그러면 Python이 각각 `0` 또는 문자열 끝까지를 기본값으로 사용합니다. 예를 들어, `start` 인덱스를 생략하면 다음과 같이 됩니다:

```python
my_str = 'Hello world'
print(my_str[:7])  # Hello w
```

이것은 `0` 인덱스부터 `7` 인덱스 바로 전까지 모든 문자를 추출합니다. `stop` 인덱스를 생략하면 다음과 같이 됩니다:

```python
my_str = 'Hello world'
print(my_str[8:])  # rld
```

이것은 `8` 인덱스부터 문자열 끝까지 모든 문자를 추출합니다.

문자열을 슬라이싱해도 원본 문자열은 변경되지 않는다는 점에 유의하세요:

```py
my_str = 'Hello world'
print(my_str[8:])  # rld
print(my_str)  # Hello world
```

`start`와 `stop` 인덱스를 모두 생략하면 전체 문자열을 추출합니다:

```python
my_str = 'Hello world'
print(my_str[:])  # Hello world
```

`start`와 `stop` 인덱스 외에도 선택적인 `step` 매개변수가 있는데, 이는 슬라이스 내 각 인덱스 사이의 증가값을 지정하는 데 사용됩니다.

구문은 다음과 같습니다:

```python
string[start:stop:step]
```

아래 예시에서 슬라이싱은 `0` 인덱스에서 시작해 `11` 인덱스 바로 전에 정지하며, 두 번째 문자마다 추출합니다:

```python
my_str = 'Hello world'
print(my_str[0:11:2])  # Hlowrd
```

`step` 매개변수를 활용하는 유용한 팁은 step을 `-1`로 설정하고 `start`와 `stop`를 비워두어 문자열을 뒤집는 것입니다:

```python
my_str = 'Hello world'
print(my_str[::-1]) # dlrow olleH
```
