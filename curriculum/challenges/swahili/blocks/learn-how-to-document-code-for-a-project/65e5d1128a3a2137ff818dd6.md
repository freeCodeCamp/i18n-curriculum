# --description--

`Anyone who` hutumika kumrejelea mtu yeyote anayekidhi sharti au hitaji fulani.

Katika teknolojia, mara nyingi hutumika kuelezea watu wenye ujuzi au maarifa maalum. Kwa mfano, `Anyone who understands JavaScript can modify this script.`

# --questions--

## --text--

Ni sentensi gani inayotumia `Anyone who` kwa usahihi katika muktadha wa teknolojia?

## --answers--
