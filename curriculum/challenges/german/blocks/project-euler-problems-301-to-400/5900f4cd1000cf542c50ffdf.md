# --description--

Jedes der 25 Schafe in einer Herde muss auf ein seltenes Virus getestet werden, von dem bekannt ist, dass es 2 % der Schafpopulation betrifft.

Es gibt einen genauen und äußerst empfindlichen PCR-Test für Blutproben, der ein klares positives oder negatives Ergebnis liefert, aber sehr zeitaufwendig und teuer ist.

Aufgrund der hohen Kosten schlägt der verantwortliche Tierarzt vor, statt 25 einzelner Tests das folgende Verfahren zu verwenden:

Die Schafe werden in 5 Gruppen mit jeweils 5 Schafen aufgeteilt. Für jede Gruppe werden die 5 Proben zusammen gemischt und ein einziger Test durchgeführt. Dann gilt:

- Ist das Ergebnis negativ, gelten alle Schafe in dieser Gruppe als virusfrei.
- Ist das Ergebnis positiv, werden 5 zusätzliche Tests durchgeführt (ein separater Test für jedes Tier), um die betroffenen Individuen zu bestimmen.

Da die Infektionswahrscheinlichkeit für ein bestimmtes Tier nur 0,02 beträgt, ist der erste Test (an den gepoolten Proben) für jede Gruppe:

- Negativ (und keine weiteren Tests nötig) mit der Wahrscheinlichkeit ${0.98}^5 = 0,9039207968$.
- Positiv (5 zusätzliche Tests nötig) mit der Wahrscheinlichkeit $1 - 0,9039207968 = 0,0960792032$.

Daher ist die erwartete Anzahl der Tests für jede Gruppe $1 + 0,0960792032 × 5 = 1,480396016$.

Folglich können alle 5 Gruppen mit durchschnittlich nur $1,480396016 × 5 = \mathbf{7,40198008}$ Tests untersucht werden, was eine enorme Einsparung von mehr als 70 % bedeutet!

Obwohl das beschriebene Verfahren sehr effizient erscheint, kann es noch erheblich verbessert werden (immer vorausgesetzt, dass der Test ausreichend sensitiv ist und keine negativen Effekte durch das Mischen verschiedener Proben entstehen). Zum Beispiel:

- Wir können mit einem Test an einer Mischung aller 25 Proben beginnen. Es lässt sich nachweisen, dass in etwa 60,35 % der Fälle dieser Test negativ ausfällt, sodass keine weiteren Tests nötig sind. Weitere Tests sind nur für die verbleibenden 39,65 % der Fälle erforderlich.
- Wenn wir wissen, dass mindestens ein Tier in einer Gruppe von 5 infiziert ist und die ersten 4 Einzeltests negativ sind, muss kein Test für das fünfte Tier durchgeführt werden (wir wissen, dass es infiziert sein muss).
- Wir können eine andere Anzahl von Gruppen oder eine andere Anzahl von Tieren pro Gruppe ausprobieren und diese Zahlen auf jeder Ebene so anpassen, dass die insgesamt erwartete Anzahl der Tests minimiert wird.

Um die sehr große Bandbreite an Möglichkeiten zu vereinfachen, gibt es eine Einschränkung bei der Entwicklung des kosteneffizientesten Testverfahrens: Wann immer wir mit einer gemischten Probe beginnen, müssen alle Schafe, die zu dieser Probe beitragen, vollständig untersucht werden (d. h. es muss für alle ein Urteil über infiziert oder virusfrei gefällt werden), bevor wir mit der Untersuchung anderer Tiere beginnen.

Für das aktuelle Beispiel stellt sich heraus, dass das kosteneffizienteste Testverfahren (wir nennen es die optimale Strategie) im Durchschnitt nur <strong>4,155452</strong> Tests benötigt!

Verwenden Sie für die optimale Strategie $T(s, p)$ als die durchschnittliche Anzahl der Tests, die benötigt werden, um eine Herde von $s$ Schafen auf ein Virus mit der Wahrscheinlichkeit $p$ für jedes Individuum zu untersuchen. Gerundet auf sechs Dezimalstellen gilt $T(25, 0.02) = 4,155452$ und $T(25, 0.10) = 12,702124$.

Berechnen Sie $\sum T(10\\,000, p)$ für $p = 0,01, 0,02, 0,03, \ldots 0,50$. Geben Sie Ihre Antwort auf sechs Dezimalstellen gerundet an.
