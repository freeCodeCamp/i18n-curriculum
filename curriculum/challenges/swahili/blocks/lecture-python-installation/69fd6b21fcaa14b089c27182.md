# --description--

Katika funzo lililopita, ulijifunza jinsi ya kusakinisha Python kwenye kifaa chako cha karibu. Katika funzo hili, utajifunza jinsi ya kuendesha msimbo wa Python kwa karibu.

Kabla hujaanza kujenga skripti zako za Python kwa karibu, utahitaji kusakinisha mhariri wa msimbo au IDE. IDE inasimama kwa Integrated Development Environment na ina vipengele kama mhariri wa msimbo, zana za majaribio, terminali na zaidi.

Mhariri maarufu wa msimbo ambao wasanidi programu wengi hutumia ni VS Code. Tembelea `https://code.visualstudio.com/download` kupakua VS Code ambayo inaunga mkono mazingira ya Mac, Windows na Linux. Huu ndio mhariri utakaotumika katika funzo hili.

Chaguzi nyingine za kawaida kwa ukuzaji wa Python ni pamoja na:

- PyCharm
- Spyder

Mara baada ya kupakua mhariri wako, utahitaji kuufungua. Ukifungua, utahitaji kufungua folda. Unaweza bofya kwenye chaguo la menyu la `File` upande wa juu kushoto na bofya `Open Folder`. Kutoka hapo, unaweza kuchagua au kuunda folda kwa ajili ya mradi wako. Kwa mazoezi ya ziada nje ya funzo, unaweza kutaka folda iitwayo `python-projects`.

Mara baada ya kuweka folda yako, unaweza kufungua explorer ambayo ni upau wa upande wa kushoto katika VS Code. Bofya ikoni ya `New File` ambayo inaonekana kama kipande cha karatasi chenye alama ya kuongeza juu yake. Kisha unda faili iitwayo `main.py` na bonyeza enter. `.py` ni kiambatisho cha faili kinachoashiria kuwa hii ni faili ya Python.

Fungua faili yako ya `main.py` na andika msimbo ufuatao:

```py
print("Hello, world!")
```

Ili kuendesha msimbo huu, una chaguzi kadhaa. Chaguo la kwanza ni kubofya kitufe cha kuendesha ambacho kinaonekana kama kitufe cha kucheza kilicho upande wa juu kulia. Hilo litafungua terminali na kuendesha skripti yako ya Python hapo. Unapaswa kuona maandishi `"Hello, world!"`.

Chaguo jingine ni kufungua terminali na kuendesha programu kwa mkono kwa kutumia:

```bash
python main.py
```

Lazima ukimbie amri hii kutoka kwenye folda ambapo `main.py` imehifadhiwa. Kwa mfano, kama `main.py` iko ndani ya folda inayoitwa `python-projects`, kwanza tumia ifuatayo kwenye terminali:

```bash
cd python-projects
```

Kisha kimbia:

```bash
python main.py
```

Katika baadhi ya mazingira, hasa kwenye mifumo ya macOS na Linux ambapo Python 2 na Python 3 zote zimewekwa, unaweza kuhitaji kutumia `python3` badala ya `python`. Hii ni kawaida wakati amri ya `python` haipo au inaelekeza kwenye toleo la zamani la Python.

```bash
python3 main.py
```

Unapoendelea katika kozi hii, unapaswa kujaribu mifano kutoka kwenye mafundisho kwa karibu na kuona matokeo. Pia unapaswa kujaribu kuunda mifano yako mwenyewe ili kupima kile ulichojifunza kutoka kwa mafundisho.
