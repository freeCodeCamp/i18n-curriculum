# --explanation--

`final report`는 프로젝트, 이벤트, 활동의 결과, 진행 상황 또는 발견 사항을 요약한 완전하고 최종적인 문서입니다. 보통 작업이 끝난 후 수행한 일과 배운 점을 검토하기 위해 작성합니다. 예를 들어:

`The final report showed that the training program improved employee performance.` – 이 문서는 교육 프로그램이 끝난 후 결과 요약을 제공했다는 의미입니다.
