# --interactive--

Das `button`-Element wird verwendet, um eine bestimmte Aktion auszuführen, wenn es aktiviert wird. Hier ist ein Beispiel für ein `button`-Element mit dem Schaltflächentext `Start Game`.

```html
<button>Start Game</button>
```

Weitere Beispiele für die Verwendung des `button`-Elements sind das Absenden eines Formulars, das Anzeigen eines Modals oder das Öffnen und Schließen eines Seitenmenüs. Das `button`-Element hat ein `type`-Attribut, das das Verhalten der Schaltfläche bei Aktivierung steuert. Der erste mögliche Wert für das `type`-Attribut wäre der `button`-Typ. Hier ist ein Beispiel für die Verwendung des `button`-Elements mit dem `button`-Typ und einem Text von `Show Alert`:

```html
<button type="button">Show Alert</button>
```

Standardmäßig führt die Schaltfläche bei Aktivierung keine Aktion aus. Sie können jedoch JavaScript-Code hinzufügen, um die Schaltfläche interaktiv zu machen, zum Beispiel um in diesem Fall eine Warnmeldung anzuzeigen.

Klicken Sie auf die `Show Alert`-Schaltfläche, um eine Warnmeldung auf dem Bildschirm anzuzeigen. Um mit dem Beispiel zu interagieren, müssen Sie den interaktiven Editor aktivieren.


**HINWEIS**: Dieses interaktive Beispiel verwendet JavaScript, aber Sie müssen sich keine Sorgen machen, den JavaScript-Code zu verstehen. Sie lernen JavaScript in zukünftigen Modulen kennen.

:::interactive_editor

```html
<button type="button">Show Alert</button>
<script src="index.js"></script>
```

```js
const btn = document.querySelector("button");
btn.addEventListener("click", () => alert("You clicked on the alert button"));
```

:::

Ein weiterer möglicher Wert für das `type`-Attribut ist der `submit`-Wert. Hier ist ein Beispiel für die Verwendung eines `button`-Elements mit dem `submit`-Typ.

:::interactive_editor

```html
<form action="">
  <label for="email">Email address:</label>
  <input type="email" id="email" name="email" />
  <button type="submit">Submit form</button>
</form>
```

:::

Innerhalb dieses `form`-Elements befinden sich ein `label`- und ein `input`-Element für die E-Mail-Adresse des Benutzers. Wenn der Benutzer auf die Absenden-Schaltfläche klickt, werden seine Daten an den Server gesendet und verarbeitet. Der dritte mögliche Wert für das `type`-Attribut ist der `reset`-Wert. Hier ist ein Beispiel für ein `form`-Element mit Reset- und Absenden-Schaltflächen.

Aktivieren Sie den interaktiven Editor und interagieren Sie mit der E-Mail-Eingabe im Vorschaufenster, indem Sie eine gefälschte E-Mail-Adresse eingeben. Klicken Sie dann auf die Reset-Schaltfläche, um zu sehen, wie Ihre E-Mail aus dem Feld verschwindet.

:::interactive_editor

```html
<form action="">
  <label for="email">Email address:</label>
  <input type="email" id="email" name="email" />
  <button type="reset">Reset form</button>
  <button type="submit">Submit form</button>
</form>
```

:::

In diesem modifizierten Beispiel werden ein `label`- und ein `input`-Element verwendet, um die E-Mail-Adresse des Benutzers zu erfassen. Wenn der Benutzer auf die Reset-Schaltfläche klickt, werden alle seine Eingabedaten gelöscht. Es ist wichtig zu beachten, dass Reset-Schaltflächen normalerweise keine gute Idee sind, da sie dazu führen können, dass Benutzer versehentlich ihre Daten zurücksetzen. Außerdem können mehrere Schaltflächen in Ihrem Formular die Benutzeroberfläche überladen.

Eine weitere Möglichkeit, Schaltflächen in HTML zu erstellen, ist die Verwendung des `input`-Elements. Das `input`-Element hat ebenfalls ein `type`-Attribut mit den möglichen Werten `submit`, `reset` und `button`. Hier ist ein Beispiel für die Verwendung des `input`-Elements mit dem `type`, das auf `button` gesetzt ist:

:::interactive_editor

```html
<input class="start-btn" type="button" value="Start Game" />
<script src="index.js"></script>
```

```js
document.addEventListener("DOMContentLoaded", () => {
  const btn = document.querySelector(".start-btn");
  btn.addEventListener("click", () => {
    const paraEl = document.createElement("p");
    const bodyEl = document.querySelector("body");

    bodyEl.appendChild(paraEl);
    paraEl.textContent = "The game has started!!!";
  });
});
```

:::

Das `value`-Attribut wird verwendet, um den Schaltflächentext anzuzeigen. Was ist also der Unterschied zwischen der Verwendung der `input`- und `button`-Elemente? `input`-Elemente sind Void-Elemente, was bedeutet, dass sie keine Kindknoten wie Text haben können und nur einen Start-Tag besitzen. Andererseits bietet das `button`-Element mehr Flexibilität, da Sie Text, Bilder und Symbole darin verschachteln können.
