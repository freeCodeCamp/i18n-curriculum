# --explanation--

`True` è un aggettivo che indica che qualcosa è corretto, reale o preciso. Si usa per concordare con un'affermazione o un fatto. È diverso da `the truth`, che è un sostantivo che si riferisce ai fatti reali o alla realtà di qualcosa, spesso usato in senso più ampio o astratto. Per esempio:

- `It's true that the server crashed yesterday.` – Questo significa che l'istruzione sul crash è corretta.

- `The truth is that we didn't have a backup.` – Questo significa che il fatto reale è che non esisteva alcun backup.
