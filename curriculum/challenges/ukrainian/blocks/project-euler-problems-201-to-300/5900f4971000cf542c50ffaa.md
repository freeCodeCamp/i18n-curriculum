# --description--

Вибрано чотири точки з цілими координатами:

$A(a, 0)$, $B(b, 0)$, $C(0, c)$ та $D(0, d)$, де $0 < a < b$ і $0 < c < d$.

Точку $P$, також з цілими координатами, вибирають на прямій $AC$ так, щоб трикутники $ABP$, $CDP$ та $BDP$ були всі подібні.

<img alt="points A, B, C, D and P creating three triangles: ABP, CDP, and BDP" src="https://cdn.freecodecamp.org/curriculum/project-euler/three-similar-triangles.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Легко довести, що трикутники можуть бути подібними лише якщо $a = c$.

Отже, за умови $a = c$, ми шукаємо трійки ($a$, $b$, $d$), для яких існує принаймні одна точка $P$ (з цілими координатами) на $AC$, що робить трикутники $ABP$, $CDP$ та $BDP$ всі подібними.

Наприклад, якщо $(a, b, d) = (2, 3, 4)$, можна легко перевірити, що точка $P(1, 1)$ задовольняє цю умову. Зверніть увагу, що трійки (2,3,4) та (2,4,3) вважаються різними, хоча точка $P(1, 1)$ спільна для обох.

Якщо $b + d < 100$, існує 92 різні трійки ($a$, $b$, $d$), для яких існує точка $P$.

Якщо $b + d < 100\,000$, існує 320471 різних трійок ($a$, $b$, $d$), для яких існує точка $P$.

Якщо $b + d < 100\,000\,000$, скільки існує різних трійок ($a$, $b$, $d$), для яких існує точка $P$?
