### --feedback--

Правильний порядок слів — `What would happen...`.

`Would what happen if the system crashed?`

### --feedback--

У питанні `would` стоїть після `what`, а не перед ним.

---

`What would happens if the system crashed?`

### --feedback--

`Would` супроводжується інфінітивом дієслова, тому `happens` має бути `happen`.

---

`What would happen if the system crashed?`

4

# --explanation--

Якщо `wh-` питання (`when`, `who`, `why`, `how`) використовує модальне дієслово, модальне дієслово стоїть перед підметом. Основне дієслово залишається у початковій формі. Наприклад:

- `What can you do to fix this?` — запитання про можливості.

- `Why should we start early?` — запитання поради.

- `How must we prepare?` — запитання про вимоги.

- `When will you finish?` — запитання про майбутні дії.

- `Where could I find help?` — запитання про варіанти.

Таким чином ви питаєте про те, що можливо, необхідно або очікується.
