# --interactive--

在之前的课程中，你学习了如何开始使用线性和径向渐变。在本课中，我们将深入探讨如何使用 CSS 属性来创建更具视觉吸引力和复杂的设计。

通过 CSS 中的渐变效果，你可以在两种或多种指定颜色之间创建平滑过渡。 它们通常用于增加网页设计的深度、质感和视觉趣味，而无需图像文件。 

CSS 支持两种主要类型的渐变：线性渐变和径向渐变。

线性渐变沿直线在颜色之间形成渐变。 你可以控制这条线的方向和使用的颜色。 下面是一个线性渐变的基本示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

这样就形成了一个从左边红色过渡到右边蓝色的渐变效果。 `to right` 参数指定渐变的方向。 你可以使用其他方向，如 `to left`、`to top`、`to bottom`，甚至指定角度，如 `45deg`。

你还可以在线性渐变中使用多个色标：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="multi-color-gradient"></div>
```

```css
.multi-color-gradient {
  background: linear-gradient(to right, red, yellow, blue);
  height: 40vh;
}
```

:::

这样就可以在整个元素上创建一个从红色到黄色再到蓝色均匀过渡的渐变效果。

而径向渐变则是从中心点辐射开来的圆形或椭圆形渐变。 这是一个示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::

这样就形成了一个圆形渐变，中间为红色，边缘过渡到蓝色。 你还可以指定径向渐变的形状和大小：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="custom-radial-gradient"></div>
```

```css
.custom-radial-gradient {
  background: radial-gradient(ellipse at top left, red, blue);
  height: 40vh;
}
```

:::

这将从元素的左上角开始创建一个椭圆渐变。 线性渐变和径向渐变都允许你在精确的位置指定色标。 例如：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="precise-gradient"></div>
```

```css
.precise-gradient {
  background: linear-gradient(to right, red 20%, yellow 40%, blue 80%);
  height: 40vh;
}
```

:::

这将创建一个渐变，其中红色占最初的 `20%`，然后在 `40%` 处过渡到黄色，最后在 `80%` 处变为蓝色。

在 CSS 中只要是能使用图片的地方，就可以使用渐变。 因此，当与其他 CSS 属性结合使用时，它们可用于创建背景、按钮甚至文本效果。

请记住，虽然渐变可以增强网站的视觉吸引力，但重要的是要确保足够的对比度，以便于访问，尤其是在文字后面使用渐变时。
