# --description--

Una `extension` es un componente de software que añade capacidades específicas a una aplicación de software más grande. Las `extensions` pueden proporcionar nueva funcionalidad o mejorar características existentes.

Por ejemplo, una `extension` de navegador puede agregar funcionalidad especial a tu navegador web, como bloquear anuncios o gestionar contraseñas.
