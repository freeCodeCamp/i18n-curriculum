# --description--

令 $n$ 为正整数。考虑以下的 nim 位置：

- 有 $n$ 个非空堆。
- 每个堆的大小小于 $2^n$。
- `euler409()`应该返回253223948。

令 $W(n)$ 为满足上述条件的获胜 nim 位置数（如果先手玩家有必胜策略，则该位置为获胜位置）。

例如，$W(1) = 1$，$W(2) = 6$，$W(3) = 168$，$W(5) = 19\\,764\\,360$，以及 $W(100)\bmod 1\\,000\\,000\\,007 = 384\\,777\\,056$。

查找 $W(10\,000\,000)\bmod 1\,000\,000\,007$。
