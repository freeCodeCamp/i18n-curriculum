# --questions--

## --text--

¿Qué oración usa `on` correctamente para mostrar el tema que se está discutiendo?

## --answers--

`The team is working on the new software update.`

---

`She fixed the bug by working on the weekend.`

### --feedback--

Este uso de `on` indica tiempo, no el tema.

---

`He logged in on his computer to check the server.`

### --feedback--

Aquí, `on` indica ubicación, no el tema.

---

`The meeting started on time with all developers present.`

### --feedback--

Este `on` se relaciona con el tiempo, no con el tema discutido.

1

# --explanation--

La preposición `on` puede tener muchos significados, pero en esta tarea, debe usarse para mostrar el tema o asunto que se está discutiendo o trabajando.

`Working on the new software update` significa que el equipo está enfocando su esfuerzo y atención en ese proyecto o tema específico.

Otras oraciones usan `on` para mostrar tiempo (`on the weekend`, `on time`) o lugar/dispositivo (`on his computer`), que son usos diferentes.
