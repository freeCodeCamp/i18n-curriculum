# --description--

`switch` 语句用于根据不同条件执行不同操作。它类似于 `if-else` 语句，但在需要检查多个条件时更易读且更易理解。`switch` 语句用于选择要执行的多个代码块中的一个。

`switch` 语句计算一个表达式并将其与每个分支的值进行比较。如果有匹配，则执行关联的代码块。如果没有匹配，则执行默认的代码块。

```javascript
let day = "Monday";

switch (day) {
  case "Monday":
    console.log("Today is Monday");
    break;
  case "Tuesday":
    console.log("Today is Tuesday");
    break;
  case "Wednesday":
    console.log("Today is Wednesday");
    break;
  case "Thursday":
    console.log("Today is Thursday");
    break;
  case "Friday":
    console.log("Today is Friday");
    break;
  case "Saturday":
    console.log("Today is Saturday");
    break;
  case "Sunday":
    console.log("Today is Sunday");
    break;
  default:
    console.log("Invalid day");
}
```

在上述示例中，`day` 变量的值会与每个分支的值进行比较。如果有匹配，则执行关联的代码块。如果没有匹配，则执行默认的代码块。

## --text--

给定下面的 JavaScript 代码截图，该截图根据星期几修改 `activity` 变量，如果 `day` 变量的值是 `"Tuesday"`，那么 `activity` 变量的值将是多少？

```javascript
let day = "Tuesday";
let activity;

switch (day) {
  case "Monday":
    activity = "Go to the gym";
    break;
  case "Tuesday":
    activity = "Attend coding meetup";
    break;
  case "Wednesday":
    activity = "Watch a movie";
    break;
  case "Thursday":
    activity = "Visit a museum";
    break;
  case "Friday":
    activity = "Dinner with friends";
    break;
  case "Saturday":
    activity = "Hiking in the mountains";
    break;
  case "Sunday":
    activity = "Rest at home";
    break;
  default:
    activity = "Undefined day";
}

```

`activity` 变量的值将是 `"Attend coding meetup"`。
