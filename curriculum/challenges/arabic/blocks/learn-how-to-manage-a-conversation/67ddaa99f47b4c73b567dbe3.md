# --explanation--

`Sure thing` طريقة غير رسمية وودية لقول `Of course` أو `I'll do it`. تُظهر أن المتحدث يوافق على الطلب دون تردد. على سبيل المثال:

- **Ashley:** `Can you send me the file this afternoon?` - هذا الشخص يطلب ملفًا.

- **Sam:** `Sure thing.` - هذا يعني أن المتحدث يوافق وسيُرسل الملف كما طُلب.
