# --description--

當你想詢問目前位置是否存在特定地點或事物時，你可以使用疑問結構 `Are there any ... around here?`。例如：

`Are there any Wi-Fi spots around here?`－你在詢問這附近是否有 Wi-Fi 熱點。

片語 `around here` 是指附近區域。
