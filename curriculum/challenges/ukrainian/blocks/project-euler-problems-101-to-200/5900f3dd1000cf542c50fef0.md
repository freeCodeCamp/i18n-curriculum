# --description--

Рухаючись зліва направо, якщо жодна цифра не перевищується цифрою зліва від неї, таке число називають зростаючим; наприклад, 134468.

Аналогічно, якщо жодна цифра не перевищується цифрою справа від неї, таке число називають спадним; наприклад, 66420.

Позитивне ціле число, яке не є ні зростаючим, ні спадним, ми назвемо «хитким»; наприклад, 155349.

Зі збільшенням n частка хитких чисел менша за n зростає так, що існує лише 12951 число менше за мільйон, яке не є хитким, і лише 277032 не хитких числа менше за ${10}^{10}$.

Скільки чисел менше за гугол (${10}^{100}$) не є хиткими?
