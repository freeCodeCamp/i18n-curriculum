# --description--

Модуль OS — це ще один стандартний модуль, який вбудований у Node.js.

Він дозволяє взаємодіяти з поточною операційною системою, на якій працює Node, щоб ви могли отримувати важливу інформацію, таку як тип ОС, деталі центрального процесора, доступна пам’ять, загальна пам’ять, мережеві інтерфейси та інше.

Щоб використати модуль OS, імпортуйте його таким чином:

```js
const os = require("os");
```

Модуль OS надає кілька методів, які ви можете використовувати, щоб зробити свій застосунок більш обізнаним про систему. Розглянемо деякі загальні.

Метод `platform()` отримує операційну систему, на якій наразі працює Node:

```js
const os = require("os");
console.log(os.platform()); // darwin
```

Це може бути корисним для реалізації кросплатформеного скриптингу:

```js
if (os.platform() === 'win32') {
 // Windows specific code
} else {
 // Non-Windows specific code
}
```

Метод `arch()` показує рядок, що представляє архітектуру центрального процесора, для якої було скомпільовано Node.js:

```js
const os = require("os");
console.log(os.arch()); // arm64
```

Це може бути корисно, якщо ви хочете, щоб користувачі завантажували правильні бінарні файли та залежності для певної архітектури.

Інші можливі значення — `'x64'`, `'arm'`, `'arm64'`, `'ia32'`, `'mips'`, `'ppc'` та інші.

`type()` отримує офіційну назву ОС, щоб ви могли програмно ідентифікувати операційні системи:

```js
const os = require("os");
console.log(os.type()); // Darwin (the core OS for macOS, iOS, and other Apple products)
```

`release()` показує версію ядра ОС системи — основну частину операційної системи, яка керує системними ресурсами та комунікацією між апаратними та програмними компонентами. Цей метод може бути корисним для відстеження сумісності між версіями ядра ОС і вимогами сервера.

Він повертає це у вигляді рядка, наприклад `'20.6.0'`:

```js
const os = require("os");
console.log(os.release()); // 25.0.0
```

`version()` повертає конкретну версію операційної системи з більш детальною інформацією, ніж метод `release()`:

```js
const os = require("os");
console.log(os.version());
// Darwin Kernel Version 25.0.0: Wed Sep 17 21:41:39 PDT 2025;
// root:xnu-12377.1.9~141/RELEASE_ARM64_T8103
```

`cpus()` повертає масив об’єктів з деталями про кожне логічне ядро центрального процесора. Це може допомогти відстежувати навантаження на CPU:

`uptime()` показує час, що минув від запуску системи. Це може допомогти визначити, як довго працюють сервери:

```js
const os = require("os");
console.log(os.uptime()); // 23047
```

`totalmem()` і `freemem()` показують загальну кількість системної пам’яті в байтах і вільну системну пам’ять у байтах відповідно:

```js
const os = require("os");
console.log(os.totalmem()); // 8589934592 (8 GB)
console.log(os.freemem()); // 93585408 (87 MB)
```

`userInfo()` повертає об’єкт, що містить інформацію про поточного користувача системи:

```js
const os = require("os");
console.log(os.userInfo());

/*

*/
```
