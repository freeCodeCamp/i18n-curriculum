# --description--

行內 `CSS` 使得可以直接為 `HTML` 元素添加樣式，儘管這種方法不太建議：

```html
<body>
  <div style="color: white; background-color: black;">...</div>
</body>
```

首先要注意的是這裡沒有任何選擇器，因為樣式是直接添加到開啟的 `<div>` 頁籤本身。接著，你有 `style` 屬性，其值位於一對引號內的是宣告。

如果你需要為單一元素添加獨特的樣式，這個方法可以正常運作。不過，通常來說，基於幾個原因，這並不是將 CSS 添加到 HTML 的推薦方式：

一旦你開始為單一元素添加大量宣告，檔案很快就會變得相當混亂，導致你的 HTML 檔案不必要地膨脹。  
如果你想讓許多元素擁有相同的樣式，你必須將相同的樣式複製並貼上到每個元素，造成大量不必要的重複和更多膨脹。  
任何行內 CSS 都會改寫另外兩種方法，這可能會導致意想不到的結果。（雖然你在這裡不會深入探討，但這其實可以被利用）。
