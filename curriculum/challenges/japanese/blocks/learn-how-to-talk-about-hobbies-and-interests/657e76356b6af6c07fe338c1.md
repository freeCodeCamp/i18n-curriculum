# --description--

`Present Perfect` 時制では、`for` と `since` が時間の継続や開始点について話すために使われます。

`For` は、何かがどれくらいの間続いているかを示します。例えば、`for five years` や `for two months` です。

`Since` は、何かがいつ始まったかを示します。例えば、`since 2020` や `since last week` です。
