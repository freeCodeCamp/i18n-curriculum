# --description--

每種程式語言都有某種方式，透過內建的方法、函式、屬性或關鍵字將資料輸出到終端機。在 Python 中，你可以使用 `print` 函式將資料列印到終端機。讓我們更仔細地看看 `print` 函式，讓你能夠自信地開始使用它。

當你學習任何程式語言時，最先做的事情之一就是撰寫一個簡單的 `Hello world!` 程式。你可以用 Python 只用 `print` 函式輕鬆做到這一點。

為此，你只需要將字串 `Hello world!` 放在你用來呼叫 `print` 函式的小括弧開頭和結尾之間：

```python
print('Hello world!') # Hello world!
```

你將在未來的課程中學習更多關於 Python 中的字串和函式。現在，請將字串視為被單引號（`'`）或雙引號（`"`）包圍的一連串字元。

在 `print('Hello world!')` 範例中，字串 `'Hello world!'` 是傳遞給 `print` 函式的**引數**。你也可以使用 `print` 函式一次顯示多個值或引數，方法是用逗號分隔它們。<dfn>例如</dfn>：

```python
print('My favorite colors are', 'blue', 'green', 'red')

# Output: My favorite colors are blue green red
```

當你用逗號分隔每個項目時，Python 會自動在每個項目之間添加空格。當你想要一起列印多個資訊時，這非常有用。
