`Can you explain the project details to me?`

---

`Can explains you the project details?`

### --feedback--

El verbo modal `can` debe ir seguido de la forma base del verbo, no de `explains`.

---

`Do can you explain the project details?`

### --feedback--

`Can` ya actúa como verbo auxiliar, por lo que no se necesita `do` en la pregunta.

# --explanation--

Al formar una pregunta con verbos modales, la estructura es: verbo modal + sujeto + verbo base + (objeto)? Por ejemplo:

- `Can we complete the project by Friday?` - Preguntando si es posible terminar el proyecto para el viernes.

- `Should we update the software to the latest version?` - Preguntando por un consejo o sugerencia sobre si actualizar el software.

En la respuesta correcta, `can` se coloca antes del sujeto (`you`), y el verbo (`explain`) está en su forma base.
