# --explanation--

`Once` es un sinónimo de `as soon as` o `after`. Se usa para indicar que una acción particular ha sido completada. Por ejemplo:

`Once I've finished reviewing the report, I'll send my feedback.` - Esto significa que el envío de comentarios ocurrirá solo después de que la revisión esté completamente terminada.
