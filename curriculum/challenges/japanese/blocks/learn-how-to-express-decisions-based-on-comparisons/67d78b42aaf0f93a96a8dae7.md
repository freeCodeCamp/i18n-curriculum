# --explanation--

`That's true`は、誰かの発言に同意したり、その内容が正しいことを認めたりするときに使うフレーズです。例えば：

`That's true, we do need more time to finish the project.` – これは、もっと時間が必要だということに同意する意味です。

`Cost-effective`は、支払ったお金に対して良い価値を提供したり、コストを節約しつつ良い結果を出したりすることを意味します。例えば：

`Using cloud services can be more cost-effective than buying expensive servers.` – これは、クラウドサービスが費用を節約しつつ役立つことを意味します。
