# --description--

在歐幾里得平面上，一隻螞蟻從點 $A(0, 1)$ 移動到點 $B(d, 1)$，其中 $d$ 為整數。

在每一步中，位於點 ($x_0$, $y_0$) 的螞蟻會選擇一個滿足 $x_1 ≥ 0$ 且 $y_1 ≥ 1$ 的格點 ($x_1$, $y_1$)，並以恆定速度 $v$ 直接前往 ($x_1$, $y_1$)。速度 $v$ 的值取決於 $y_0$ 和 $y_1$，如下所示：

- 如果 $y_0 = y_1$，則 $v$ 的值與 $y_0$ 相等。
- 如果 $y_0 ≠ y_1$，則 $v$ 的值等於 $\frac{y_1 - y_0}{\ln y_1 - \ln y_0}$。

左邊的影像是 $d = 4$ 的其中一條可能路徑。螞蟻首先以速度 $\frac{3 - 1}{\ln 3 - \ln 1} ≈ 1.8205$ 從 $A(0, 1)$ 移動到 $P_1(1, 3)$。接著所需時間為 $\frac{\sqrt{5}}{1.820} ≈ 1.2283$。

從 $P_1(1, 3)$ 到 $P_2(3, 3)$ 螞蟻以速度 3 移動，因此所需時間為 $\frac{2}{3} ≈ 0.6667$。從 $P_2(3, 3)$ 到 $B(4, 1)$ 螞蟻以速度 $\frac{1 - 3}{\ln 1 - \ln 3} ≈ 1.8205$ 移動，因此所需時間為 $\frac{\sqrt{5}}{1.8205} ≈ 1.2283$。

因此總共所需時間為 $1.2283 + 0.6667 + 1.2283 = 3.1233$。

右邊的影像是另一條路徑。總所需時間計算為 $0.98026 + 1 + 0.98026 = 2.96052$。可以證明這是 $d = 4$ 時最快的路徑。

<img alt="two possible paths for d = 4" src="https://cdn.freecodecamp.org/curriculum/project-euler/an-ant-on-the-move.jpg" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

設 $F(d)$ 為螞蟻選擇最快路徑時所需的總時間。 例如，$F(4) ≈ 2.960\\,516\\,287$。 我們可以驗證 $F(10) ≈ 4.668\\,187\\,834$ 和 $F(100) ≈ 9.217\\,221\\,972$。

求 $F(10\,000)$。將你的答案四捨五入到九位小數。
