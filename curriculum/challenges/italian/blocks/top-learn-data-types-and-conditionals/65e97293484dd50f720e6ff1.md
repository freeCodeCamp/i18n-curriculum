# --description--

Un altro modo per confrontare le istruzioni condizionali è l'operatore ternario. L'operatore ternario è un modo abbreviato di scrivere un'istruzione `if-else`. Viene usato per valutare una condizione e restituire un valore in base al risultato della condizione. La sintassi dell'operatore ternario è la seguente:

```javascript
let result = condition ? value1 : value2;
```

Nell'esempio sopra, se la condizione è vera, il valore di `result` sarà `value1`. Altrimenti, il valore di `result` sarà `value2`.

Gli operatori ternari sono spesso usati per assegnare un valore a una variabile in base a una condizione. Vengono anche usati per restituire un valore in base a una condizione.
