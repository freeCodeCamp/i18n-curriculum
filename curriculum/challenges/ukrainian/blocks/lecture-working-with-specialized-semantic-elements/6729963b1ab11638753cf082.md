# --interactive--

Елемент неартикульованої анотації, або коротко `u`, використовується для представлення рядкового тексту з застосованою не текстовою анотацією.

Ось приклад використання елемента `u` для виділення різних орфографічних помилок:

:::interactive_editor

```html
<p>
  You can use the unarticulated annotation element to highlight
  <u>inccccort</u> <u>spling</u> <u>issses</u>.
</p>
```

:::

У прикладі слова `incorrect`, `spelling` і `issues` написані з помилками. Стиль за замовчуванням для елемента `u` — чорна підкреслення під текстом.

В HTML4 елемент `u` використовувався для стилізації. Але в HTML5 елемент `u` слід використовувати лише для позначення тексту з не текстовою анотацією.

Якщо ви хочете стилізувати текст підкресленням, слід використовувати CSS замість HTML.

Елемент закреслення, або коротко `s`, слід використовувати для позначення тексту, який більше не є точним або актуальним. Ось приклад використання елемента `s` для показу скасування заходу:

:::interactive_editor

```html
<p><s>Tomorrow's hike will be meeting at noon.</s></p>

<p>Due to unforeseen weather conditions, the hike has been canceled.</p>
```

:::

У цьому прикладі перше речення закреслене, бо похід скасували через погодні умови.

Елемент `s` ніколи не слід використовувати лише для показу змін у документі. У такому випадку доречнішими будуть елементи видаленого тексту та вставленого тексту.

Елемент `ruby` представляє дрібний текст, показаний над або під основним текстом. Зазвичай його використовують для показу вимови східноазійських символів. Ось приклад елемента `ruby` зі сторінки MDN web docs:

:::interactive_editor

```html
<ruby> 明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp> </ruby>
```

:::

Елемент `rp`, або елемент запасної дужки ruby, використовується як запасний варіант для браузерів, які не підтримують відображення ruby-анотацій.

Елемент `rt`, або ruby-текст, використовується для позначення тексту ruby-анотації. Цей текст зазвичай служить для вимови або деталей перекладу у східноазійській типографіці.

Хоча елемент `ruby` можна використовувати для інших типів анотацій, найпоширеніше застосування — у східноазійській типографіці.
