# --description--

Die `for`-Schleife ist eine weitere Art von Schleife, die verwendet wird, um einen Codeblock mehrfach auszuführen. Die `for`-Schleife wird verwendet, wenn die Anzahl der Wiederholungen bekannt ist. Die Syntax der `for`-Schleife ist wie folgt:

```javascript
for (initialization; condition; increment/decrement) {
  // code block to be executed
}
```

Zum Beispiel gibt der folgende Code-Ausschnitt die Zahlen von 1 bis 5 aus:

```javascript
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
```

Um ein Array mit einer `for`-Schleife zu durchlaufen, können Sie die Länge des Arrays verwenden. Zum Beispiel gibt der folgende Code-Ausschnitt die Elemente eines Arrays aus:

```javascript
const arr = [1, 2, 3, 4, 5];
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}
```
