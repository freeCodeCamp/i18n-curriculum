## Вступ до основних модулів Node.js

- **Основні модулі Node.js**: Вбудовані модулі, які постачаються разом із Node.js і забезпечують базову функціональність для роботи з файлами, криптографією, взаємодією з операційною системою та інше.
- **Стандартні модулі**: Не потребують окремої інсталяції; доступні за умови встановлення Node.js.
- **Синтаксис імпорту**: Використовуйте `require()` для імпорту основних модулів у свій застосунок.

```javascript
const fs = require("fs");
const crypto = require("crypto");
const os = require("os");
```

## Модуль файлової системи (fs)

- **Призначення**: Надає методи для роботи з файлами та папками, включно з відкриттям, закриттям, читанням, записом і видаленням.
- **Імпорт**: `const fs = require("fs");`
- **Гнучкість**: Методи доступні у трьох формах: функції зворотного виклику, проміси та синхронні.

### Шаблони використання методів

- **Асинхронно з функціями зворотного виклику**: Поведінка за замовчуванням, неблокуюча.

```javascript
fs.writeFile("filePath", "content", "utf8", (err) => {
  if (err) throw err;
  console.log("File written to!");
});
```

- **На основі промісів**: Рекомендовано для сучасних застосунків, використовує `async/await`.

```javascript
const fs = require("fs/promises");

async function writeToFile() {
  try {
    await fs.writeFile("article.md", "## Node fs Module");
    console.log("File written to!");
  } catch (err) {
    console.error(err);
  }
}
```

- **Синхронно**: Блокуюча операція, підходить для простих скриптів.

```javascript
try {
  fs.writeFileSync("article.md", "## Node fs Module", "utf8");
  console.log("File written to!");
} catch (err) {
  console.error(err);
}
```

### Основні файлові операції

- **Метод `writeFile()`**: Створює або перезаписує файл із заданим вмістом.

```javascript
await fs.writeFile("article.md", "## Node fs Module: The Complete Guide");
```

- **Метод `appendFile()`**: Додає вміст у кінець існуючого файлу.

```javascript
await fs.appendFile("article.md", "\n\nIn this article, you will learn...");
```

- **Метод `readFile()`**: Читає вміст файлу.

```javascript
const content = await fs.readFile("article.md", "utf8");
console.log("File content:", content);
```

- **Метод `unlink()`**: Видаляє файл із файлової системи.

```javascript
await fs.unlink("article.md");
console.log("File deleted successfully");
```

### Кодування символів

- **Кодування UTF-8**: Вкажіть `"utf8"`, щоб читати файли як текстові рядки.
- **Формат буфера**: Без вказівки кодування файли читаються як бінарні об’єкти буфера.

```javascript
// Returns string
const textContent = await fs.readFile("file.txt", "utf8");

// Returns Buffer
const bufferContent = await fs.readFile("file.txt");
```

## Модуль буфера

- **Призначення**: Обробляє бінарні дані безпосередньо в пам’яті для файлів, зображень і мережевих потоків.
- **Імпорт**: `const { Buffer } = require("buffer");`
- **Використання**: Вхід/вихід файлів, TCP-потоки, обробка зображень і маніпуляції бінарними даними.

### Створення буфера

- **Метод `Buffer.from()`**: Створює буфер із рядків, масивів або інших даних.

```javascript
// From string
const myStrBuffer = Buffer.from("freeCodeCamp");
console.log(myStrBuffer); // <Buffer 66 72 65 65 43 6f 64 65 43 61 6d 70>

// From array of numbers
const myNumBuffer = Buffer.from([70, 82, 69, 69, 67, 79, 68, 69, 67, 65, 77, 80]);
```

- **Метод `Buffer.alloc()`**: Створює буфер заданого розміру, заповнений нулями.

```javascript
const someBuffer = Buffer.alloc(10);
console.log(someBuffer); // <Buffer 00 00 00 00 00 00 00 00 00 00>
```

### Операції з буфером

- **Доступ як до масиву**: Отримуйте доступ до окремих елементів буфера за індексом.

```javascript
console.log(myStrBuffer[0]); // 102 (byte value for 'f')
```

- **Метод `toString()`**: Конвертує буфер у читабельний рядок.

```javascript
console.log(myStrBuffer.toString()); // "freeCodeCamp"
```

- **Метод `Buffer.write()`**: Записує дані у виділений буфер.

```javascript
someBuffer.write("Hello fCC");
console.log(someBuffer.toString()); // "Hello fCC"
```

- **Метод `Buffer.byteLength()`**: Повертає кількість байтів, необхідних для збереження рядка.

```javascript
console.log(Buffer.byteLength("Hello freeCodeCamp")); // 18
```

### Додаткові методи буфера

- **`Buffer.isBuffer()`**: Перевіряє, чи є об’єкт буфером.
- **`Buffer.compare()`**: Порівнює два буфери та повертає порядок сортування.
- **`Buffer.concat()`**: Об’єднує кілька буферів в один.

## Модуль Crypto

- **Призначення**: Забезпечує криптографічні функції, включно з хешуванням, шифруванням, дешифруванням і цифровими підписами.
- **Імпорт**: `const crypto = require("crypto");`
- **Примітка з безпеки**: Для автентифікації в продакшені використовуйте добре перевірені бібліотеки, такі як `bcrypt` або `jsonwebtoken`.

### Методи хешування

- **Метод `createHash()`**: Створює односторонній хеш за допомогою алгоритмів SHA256, SHA512 або MD5.

```javascript
const hashedPassword = crypto
  .createHash("sha256")
  .update("myPassword123")
  .digest("hex");

console.log("createHash result:", hashedPassword);
```

- **Метод `createHmac()`**: Створює хеш із секретним ключем для автентифікації та цілісності даних.

```javascript
const hashedMessage = crypto
  .createHmac("sha256", "secret-key")
  .update("Hello World")
  .digest("hex");
```

### Шифрування та дешифрування

- **Методи `createCipheriv()` і `createDecipheriv()`**: Шифрують і дешифрують дані за допомогою алгоритмів, ключів і ініціалізаційних векторів.

```javascript
const key = Buffer.from("12345678901234567890123456789012"); // 32 bytes for AES-256
const iv = Buffer.from("1234567890123456"); // 16 bytes for AES

// Encryption
const cipher = crypto.createCipheriv("aes-256-cbc", key, iv);
let encrypted = cipher.update("Hello campers!", "utf8", "hex");
encrypted += cipher.final("hex");

// Decryption
const decipher = crypto.createDecipheriv("aes-256-cbc", key, iv);
let decrypted = decipher.update(encrypted, "hex", "utf8");
decrypted += decipher.final("utf8");
```

### Генерація випадкових значень

- **Метод `randomBytes()`**: Генерує криптографічно безпечні випадкові байти.

```javascript
console.log("Random Bytes:", crypto.randomBytes(16).toString("hex"));
// Output: a6154ef5a296fa176ad0f332bd94d712
```

- **Метод `randomInt()`**: Генерує безпечні випадкові цілі числа в заданому діапазоні.

```javascript
console.log("Random Int:", crypto.randomInt(0, 100)); // Random number 0-99
```

### Управління ключами

- **Метод `createSecretKey()`**: Створює криптографічні ключі з сирих байтів.

```javascript
const secret = crypto.createSecretKey(crypto.randomBytes(32));
console.log(secret.export().toString('hex'));
```

### Цифрові підписи

- **Методи `sign()` і `verify()`**: Створюють і перевіряють цифрові підписи за допомогою пар приватних/публічних ключів.
- **Використання**: Доказ автентичності та цілісності даних.

### Додаткові методи Crypto

- **`createPublicKey()` і `createPrivateKey()`**: Працюють із зовнішньо згенерованими ключами.
- **`createDiffieHellman()`**: Увімкнення безпечного створення спільного секрету.
- **`Certificate()`**: Обробка операцій із HTTPS-сертифікатами.

## Модуль OS

- **Призначення**: Надає утиліти, пов’язані з операційною системою, для отримання інформації про систему та забезпечення кросплатформної сумісності.
- **Імпорт**: `const os = require("os");`
- **Використання**: Моніторинг системи, кросплатформне скриптування, інформація про апаратне забезпечення.

### Методи отримання інформації про систему

- **Метод `platform()`**: Повертає платформу операційної системи.

```javascript
console.log(os.platform()); // 'darwin', 'win32', 'linux', etc.

// Cross-platform usage
if (os.platform() === 'win32') {
  // Windows-specific code
} else {
  // Unix-like systems
}
```

- **Метод `arch()`**: Повертає архітектуру центрального процесора.

```javascript
console.log(os.arch()); // 'arm64', 'x64', 'arm', 'ia32', etc.
```

- **Метод `type()`**: Повертає офіційну назву операційної системи.

```javascript
console.log(os.type()); // 'Darwin', 'Linux', 'Windows_NT'
```

### Інформація про версію

- **Метод `release()`**: Показує версію ядра ОС.

```javascript
console.log(os.release()); // '25.0.0'
```

- **Метод `version()`**: Повертає детальну інформацію про версію ОС.

```javascript
console.log(os.version());
// Darwin Kernel Version 25.0.0: Wed Sep 17 21:41:39 PDT 2025
```

### Інформація про апаратне забезпечення

- **Метод `cpus()`**: Повертає масив інформації про ядра центрального процесора.

```javascript
console.log(os.cpus()); // Array of CPU core objects with model, speed, times
```

- **Метод `uptime()`**: Повертає час роботи системи в секундах.

```javascript
console.log(os.uptime()); // 23047 (seconds since system boot)
```

### Інформація про пам’ять

- **Метод `totalmem()`**: Повертає загальний обсяг системної пам’яті в байтах.
- **Метод `freemem()`**: Повертає доступний обсяг системної пам’яті в байтах.
