# --interactive--

Gli operatori di confronto ti permettono di confrontare due valori e restituire un risultato `true` o `false`. Puoi poi usare il risultato per prendere una decisione o controllare il flusso del tuo programma. Usi i confronti nelle istruzioni `if`, nei loop e in molte altre situazioni in cui devi prendere decisioni basate su certe condizioni. Immergiamoci negli operatori di confronto più comuni e vediamo come funzionano.

L'operatore maggiore, rappresentato da un simbolo di maggiore (`>`), verifica se il valore a sinistra è maggiore di quello a destra:

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a > b); // false
console.log(b > a); // true
```

:::

L'operatore maggiore o uguale, rappresentato da un simbolo di maggiore seguito dal segno di uguale (`>=`), verifica se il valore a sinistra è maggiore o uguale a quello a destra:

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a >= b); // false
console.log(b >= a); // true
console.log(a >= c); // true
```

:::

L'operatore minore, rappresentato da un simbolo di minore (`<`), funziona in modo simile a `>`, ma al contrario. Verifica se il valore a sinistra è minore di quello a destra:

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a < b); // true
console.log(b < a); // false
```

:::

L'operatore minore o uguale, rappresentato da un simbolo di minore seguito dal segno di uguale (`<=`), verifica se il valore a sinistra è minore o uguale a quello a destra:

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a <= b); // true
console.log(b <= a); // false
console.log(a <= c); // true
```

:::
