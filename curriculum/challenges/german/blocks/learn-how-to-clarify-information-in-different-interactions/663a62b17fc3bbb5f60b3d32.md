# --description--

Wenn Sie über die Kombination von zwei Verben sprechen, kann das zweite Verb in verschiedenen Formen auftreten. Im Englischen ist es bei der Verwendung des Verbs `like` üblich, das zweite Verb in der `-ing`-Form zu verwenden. Beispiel: `They like eating chocolate.`, was bedeutet, dass das Essen von Schokolade etwas ist, das sie befriedigt.
