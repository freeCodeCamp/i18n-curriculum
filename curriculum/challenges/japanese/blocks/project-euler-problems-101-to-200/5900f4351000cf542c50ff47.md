# --description--

スクエブを、${p^2}{q^3}$ の形の数と定義します。ここで $p$ と $q$ は異なる素数です。

例えば、$200 = {5^2}{2^3}$ または $120072949 = {{23}^2}{{61}^3}$ です。

最初の5つのスクエブは 72、108、200、392、500 です。

興味深いことに、200 はどの一桁を変えても素数にできない最初の数であり、そのような数を素数証明と呼びます。連続部分文字列 `200` を含む次の素数証明スクエブは 1992008 です。

連続部分文字列 `200` を含む200番目の素数証明スクエブを見つけてください。
