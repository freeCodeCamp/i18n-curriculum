# --description--

2つの値の間でランダムな整数を生成するための式は次の通りです。

```js
Math.floor(Math.random() * (maximum - minimum) + minimum);
```

これは、2つの値の間の整数を生成します。

`randomInt` という変数を作成し、`min` と `max` の値の間で値を生成した結果を代入してください。

その後、結果を確認するために `randomInt` 変数をコンソールに出力してください。ページを更新すると異なる結果が表示されます。


`randomInt` 変数は数値である必要があります。

