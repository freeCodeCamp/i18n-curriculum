# --interactive--

In JavaScript ist ein Konstruktor eine spezielle Art von Funktion, die zum Erstellen und Initialisieren von Objekten verwendet wird. Er wird mit dem `new` Schlüsselwort aufgerufen und kann Eigenschaften und Methoden auf dem neu erstellten Objekt initialisieren.

In dieser Lektion schauen wir uns an, wie man mit dem `Object()` Konstruktor arbeitet. Der `Object()` Konstruktor erstellt ein neues leeres Objekt. Hier ist ein Beispiel:

```js
new Object()
```

Wenn Sie `new Object()` aufrufen, gibt er ein neues Objekt zurück, das zum Speichern von Werten verwendet werden kann.

Der `Object()` Konstruktor kann mit oder ohne das `new` Schlüsselwort verwendet werden. Wenn er als Funktion ohne `new` aufgerufen wird, verhält er sich je nach Typ des übergebenen Werts unterschiedlich. Hier ist ein Beispiel für die Verwendung des `Object()` Konstruktors ohne das `new` Schlüsselwort:

:::interactive_editor

```js
const num = 42;
const numObj = Object(num); // Creates an object wrapper for the number

console.log(numObj);
console.log(typeof numObj); // "object"
```

:::

Wie Sie im zweiten `console.log` sehen können, ist `numObj` ein Objekt. Das passiert, weil wir den `Object()` Konstruktor verwendet haben, um diese Eingabe einer Zahl in ein Objekt umzuwandeln.

Was passiert, wenn wir versuchen, `null` oder `undefined` an den `Object()` Konstruktor zu übergeben?

:::interactive_editor

```js
const newObj = new Object(undefined);
console.log(newObj); // {}
```

:::

Das Ergebnis wird ein leeres Objekt sein. Ein weiterer Anwendungsfall für den `Object()` Konstruktor ist, wenn Sie mit einem Wert unbekannten Typs arbeiten und sicherstellen müssen, dass es sich um ein Objekt handelt. Schauen wir uns folgendes Beispiel an:

:::interactive_editor

```js
function toObject(value) {
  if (value === null || value === undefined) {
    return {};
  }

  if (typeof value === "object") {
    return value;
  }

  return Object(value);
}

console.log(toObject(null));

console.log(toObject(true));

console.log(toObject([1, 2, 3]));
```

:::

In diesem Beispiel haben wir eine Funktion namens `toObject`. Die zweite Bedingung prüft, ob der Wert vom Typ Objekt ist, und gibt den Wert zurück, wenn die Bedingung `true` ist. Diese Bedingung prüft sowohl Objekte als auch Arrays, da Arrays spezielle Objekttypen sind.

Wenn keine der Bedingungen zutrifft, gibt die Funktion `Object(value)` zurück, das die Eingabe in ein Objekt umwandelt. Das funktioniert für Werte wie Zahlen, Strings und Booleans.

Meistens werden Sie den `Object()` Konstruktor nicht verwenden, um neue Objekte zu erstellen, weil Sie stattdessen die Objektliteral-Syntax verwenden (z. B. `const objectLiteral = { name: "Beau" }`). Es ist aber trotzdem gut, die Grundlagen der Arbeit mit dem Object-Konstruktor zu verstehen.
