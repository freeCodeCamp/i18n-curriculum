# --interactive--

JavaScript 中有两个强大的方法用于处理 JSON 数据：`JSON.parse()` 和 `JSON.stringify()`。这些方法通常用于在 JSON 字串和 JavaScript 对象之间转换。

`JSON.stringify()` 用于将 JavaScript 对象转换为 JSON 字串。当你想以一种可以轻松共享或在系统之间传输的形式保存或传输数据时，这非常有用。 

以下是你如何使用 `JSON.stringify()` 方法：

:::interactive_editor

```js
const user = {
  name: "John",
  age: 30,
  isAdmin: true
};

const jsonString = JSON.stringify(user);
console.log(jsonString);
```

:::

`JSON.stringify()` 方法还接受一个名为 replacer 的可选参数，该参数可以是一个函数或数组。以下是使用数组作为可选 replacer 参数的示例：

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

// result: {"firstName":"Jessica","country":"USA"}
console.log(JSON.stringify(developerObj, ["firstName", "country"]));
```

:::

在此示例中，我们有一个包含四个属性的 `developerObj`。当我们使用 `JSON.stringify()` 方法时，我们可以为第二个参数传入一个数组，并指定我们想要序列化的属性。结果将是一个只包含 `firstName` 和 `country` 属性的序列化对象字串。 

`JSON.stringify()` 方法的另一个可选参数是 spacer 参数。它允许你控制字符串化结果的间距：

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

console.log(JSON.stringify(developerObj, null, 2));

/* result
{
  "firstName": "Jessica",
  "isAwesome": true,
  "isMusician": true,
  "country": "USA"
}
*/
```

:::

大多数情况下，你不会使用 `JSON.stringify()` 方法的这两个可选参数，但了解它们仍然很有帮助。 

你在编程中将经常使用的另一个方法是 `JSON.parse()` 方法。`JSON.parse()` 将 JSON 字串转换回 JavaScript 对象。当你从网络服务器或 `localStorage` 获取 JSON 数据并且需要在你的应用中操作这些数据时，这非常有用。你将在未来的课程中学习更多关于 `localStorage` 的内容。 

这是一个关于如何使用 `JSON.parse()` 方法的示例：

:::interactive_editor

```js
const jsonString = '{"name":"John","age":30,"isAdmin":true}';
const userObject = JSON.parse(jsonString);
console.log(userObject);

// Result:
// { name: 'John', age: 30, isAdmin: true }
```

:::

这允许你将你的程序中的数据作为普通的 JavaScript 对象来处理，使其更容易操作和使用。
