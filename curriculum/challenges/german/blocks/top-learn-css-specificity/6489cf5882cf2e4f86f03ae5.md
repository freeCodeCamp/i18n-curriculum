# --description--

Ändern wir jetzt ein wenig die Dinge:

```html
<!-- index.html -->

<div class="main">
  <div class="list" id="subsection"></div>
</div>
```

```css
/* rule 1 */
#subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

Im obigen Beispiel ist trotz der Tatsache, dass Regel 2 mehr Klassen-Selektoren als ID-Selektoren hat, Regel 1 spezifischer, weil die ID die Klasse übertrumpft.
