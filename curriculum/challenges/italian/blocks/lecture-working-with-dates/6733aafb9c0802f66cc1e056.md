# --interactive--

Date e orari in JavaScript non sono sempre stati facili da gestire. In un codice professionale probabilmente userai una libreria che ha risolto molti di questi problemi per te.

Ma ci saranno momenti in cui dovrai lavorare con l'oggetto integrato `Date` di JavaScript.

In questa lezione vedremo le basi dell'oggetto `Date` e alcuni dei metodi piﺣﺗ comuni che userai quando lavori con date e orari in JavaScript.

L'oggetto `Date` ﺣ۷ un oggetto integrato in JavaScript che ti permette di lavorare con date e orari. Ecco la sintassi base per creare un nuovo oggetto `Date`:

```js
const now = new Date();
```

La parola chiave `new` viene usata per creare una nuova istanza dell'oggetto `Date`, e l'oggetto `Date` viene poi assegnato alla variabile `now`. Se stampassi il valore di `now` nella console, vedresti la data e l'ora correnti basate sull'orologio di sistema del computer che esegue il codice.

Ecco un esempio di risposta che vedresti se stampassi il valore di `now` nella console:

:::interactive_editor

```js
const now = new Date();
console.log(now);
```

:::

Per l'orario, usa il formato militare, quindi `14:30` ﺣ۷ `2:30 PM`. `GMT-0700` ﺣ۷ l'offset del fuso orario, e `Pacific Daylight Time` ﺣ۷ il nome del fuso orario.

Puoi anche passare una data e un orario specifici all'oggetto `Date` fornendo come argomenti anno, mese, giorno, ora, minuto, secondo e millisecondo.

Ecco un esempio di creazione di un nuovo oggetto `Date` con una data e un orario specifici:

:::interactive_editor

```js
const specificDate = new Date("July 4, 1776 12:00:00");

console.log(specificDate);
```

:::

Questo ﺣ۷ utile quando devi lavorare con una data e un orario specifici invece che con quelli correnti. L'input deve sempre essere una stringa, e il formato deve essere riconoscibile dall'oggetto `Date`.

Per ottenere la data e l'ora correnti, puoi usare il metodo `Date.now()`, che restituisce il numero di millisecondi trascorsi da `January 1, 1970, 00:00:00 UTC`. Questo ﺣ۷ noto come tempo epoch Unix.

Il tempo epoch Unix ﺣ۷ un modo comune per rappresentare date e orari nei sistemi informatici perchﺣ۸ ﺣ۷ un intero che puﺣﺎ essere facilmente archiviato e manipolato. UTC sta per Universal Time Coordinated, che ﺣ۷ lo standard principale con cui il mondo regola orologi e tempo.

Se hai bisogno di ottenere il giorno del mese basandoti sulla data corrente, puoi usare il metodo `getDate`. Ecco un esempio di utilizzo del metodo `getDate`:

:::interactive_editor

```js
const now = new Date();
const date = now.getDate();
console.log(date); 
```

:::

In questo esempio, creiamo una nuova istanza di data usando l'oggetto `Date` e la assegniamo alla variabile `now`. Poi chiamiamo il metodo `getDate` sull'oggetto `now` per ottenere il giorno del mese e lo assegniamo alla variabile `date`. Infine, stampiamo il valore di `date` nella console, che mostrerﺣ  il giorno corrente del mese.

`getDate` restituirﺣ  un valore intero tra `1` e `31`, a seconda del giorno del mese. Se la data non ﺣ۷ valida, restituirﺣ  `NaN` (Not a Number).

Per ottenere il mese, puoi usare il metodo `getMonth` cosﺣ؛:

:::interactive_editor

```js
const now = new Date();
const month = now.getMonth();
console.log(month);
```

:::

Il mese ﺣ۷ a base zero, quindi `January` ﺣ۷ `0`, `February` ﺣ۷ `1`, e cosﺣ؛ via. Se il mese non ﺣ۷ valido, restituirﺣ  `NaN`.

Se hai bisogno di ottenere l'anno completo, puoi usare il metodo `getFullYear` cosﺣ؛:

:::interactive_editor

```js
const now = new Date();
const year = now.getFullYear();
console.log(year); 
```

:::

Se l'anno non ﺣ۷ valido, restituirﺣ  `NaN`.

Ci sono molti altri metodi disponibili sull'oggetto `Date` inclusi `getHours`, `getMinutes`, `getSeconds` e cosﺣ؛ via. Ti incoraggio a esplorare altri metodi da solo attraverso la documentazione di Mozilla o altre risorse.
