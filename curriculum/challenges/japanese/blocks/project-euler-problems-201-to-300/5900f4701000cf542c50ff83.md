# --description--

3つの石の山と2人のプレイヤーでゲームが行われます。

各プレイヤーのターンでは、山から1つ以上の石を取り除きます。ただし、複数の山から石を取る場合は、選んだすべての山から同じ数の石を取り除く必要があります。

つまり、プレイヤーはある $N > 0$ を選び、次のいずれかを行います：

- 任意の1つの山から $N$ 個の石を取る；または
- 任意の2つの山からそれぞれ $N$ 個の石を取る（合計 $2N$ 個）；または
- 3つの山すべてからそれぞれ $N$ 個の石を取る（合計 $3N$ 個）。

最後の石を取ったプレイヤーが勝ちです。

勝ちの状態とは、先手プレイヤーが必ず勝てる状態を指します。

例えば、(0,0,13)、(0,11,11)、(5,5,5) は先手がすぐにすべての石を取れるため勝ちの状態です。

負けの状態とは、先手がどんな手を打っても後手が必ず勝てる状態を指します。

例えば、(0,1,2) と (1,3,3) は負けの状態です。どの合法的な手を打っても後手に勝ちの状態を残します。

$0 \leq x_i \leq y_i \leq z_i \leq 100$ のすべての負けの状態 ($x_i$,$y_i$,$z_i$) を考えると、$\sum (x_i + y_i + z_i) = 173\,895$ となることが確認できます。

$0 \leq x_i \leq y_i \leq z_i \leq 1000$ のすべての負けの状態 ($x_i$,$y_i$,$z_i$) に対して、$\sum (x_i + y_i + z_i)$ を求めてください。
