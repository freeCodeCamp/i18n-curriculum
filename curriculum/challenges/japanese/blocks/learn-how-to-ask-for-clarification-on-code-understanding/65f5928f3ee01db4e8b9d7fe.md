# --description--

このタスクでは、フレーズ `to be willing` について学びます。

誰かが `willing` to do something と言うとき、それは親切や協力の気持ちから、何かをする準備ができていることを意味します。例えば、`Brian is willing to help Sophie` は ブライアン が ソフィー を助ける準備ができていて、喜んでいることを示します。

このフレーズは、何かをしたり誰かを助けたりすることに対して前向きな態度を示すために、仕事上や私生活の両方でよく使われます。
