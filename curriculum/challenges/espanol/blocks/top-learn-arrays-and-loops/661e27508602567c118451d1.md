# --description--

Las cadenas y los números pueden ser nuestros bloques de construcción, pero a medida que tus scripts se vuelven más complejos, vas a necesitar una forma de manejar grandes cantidades de ellos. Afortunadamente, JavaScript tiene un par de tipos de datos que se usan precisamente para eso. Un array es una colección ordenada de elementos (cadenas, números u otras cosas).

Los arrays son una forma de almacenar múltiples valores en una sola variable. Son un tipo especial de objeto que tiene una propiedad length y una serie de propiedades numeradas. Cada propiedad numerada se llama elemento, y cada elemento puede almacenar un valor de cualquier tipo.

Un ejemplo de un array es:

```javascript
const fruits = ['apple', 'banana', 'orange'];
```
