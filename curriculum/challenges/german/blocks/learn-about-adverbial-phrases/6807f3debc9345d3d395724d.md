# --explanation--

`Central servers` beziehen sich auf leistungsstarke Computer oder Systeme, die Daten für ein Netzwerk verwalten und speichern, sodass mehrere Benutzer oder Geräte darauf zugreifen können. Diese Server befinden sich typischerweise an einem zentralen Ort und übernehmen Aufgaben wie Datenverarbeitung, Speicherung und Kommunikation. Zum Beispiel:

`The company uses central servers to host all their internal applications.` – Das bedeutet, dass das Unternehmen seine Anwendungen auf Servern an einem einzigen zentralen Standort speichert und ausführt.
