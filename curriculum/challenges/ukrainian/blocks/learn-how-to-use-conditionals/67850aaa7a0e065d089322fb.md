# --explanation--

Речення використовує `Second Conditional`, який має структуру:

`If` + `Past Simple` або `Past Continuous` час, `might / would / could` + основний дієслово. У цьому прикладі:

`If they closed` — це умова (уявлення про закриття).

`It would affect` — це результат (що станеться внаслідок).

Цей тип умовного речення використовується для розмови про нереальні або гіпотетичні ситуації.
