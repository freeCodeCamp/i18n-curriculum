# --explanation--

`Actually` se usa para mostrar la verdad real o para corregir un malentendido. Por ejemplo:

`I actually finished the project early.` – Esto significa que terminaste antes de lo esperado.

`Quite a few` significa un número mayor que solo unos pocos. Por ejemplo:

`There were quite a few people at the meeting.` – Esto significa que muchas personas asistieron.

`Those` es una palabra demostrativa usada para referirse a elementos específicos que no están cerca. Por ejemplo:

`Those are my files over there.` – Esto señala archivos que están más lejos.
