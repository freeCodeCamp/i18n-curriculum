# --interactive--

前のレッスンでは、線形グラデーションと放射状グラデーションの基本的な使い方を学びました。このレッスンでは、CSSのプロパティを使ってより視覚的に魅力的で複雑なデザインを作成する方法をさらに深く掘り下げていきます。

CSSのグラデーションは、2色以上の指定した色の間で滑らかな移行を作り出すことができます。画像ファイルを使わずに、ウェブデザインに奥行きや質感、視覚的な興味を加えるためによく使われます。

CSSでは主に2種類のグラデーションがサポートされています：線形グラデーションと放射状グラデーションです。

線形グラデーションは、直線に沿って色が徐々に混ざり合う効果を作ります。この直線の方向や使う色をコントロールできます。以下は線形グラデーションの基本例です：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

これは左から赤、右へ青へと移行するグラデーションを作成します。`to right` パラメータはグラデーションの方向を指定します。`to left`、`to top`、`to bottom`のような他の方向や、`45deg`のような角度も指定できます。

線形グラデーションでは複数の色ストップを使うこともできます：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="multi-color-gradient"></div>
```

```css
.multi-color-gradient {
  background: linear-gradient(to right, red, yellow, blue);
  height: 40vh;
}
```

:::

これは赤から黄色、青へと均等に配置されたグラデーションを作成します。

一方、放射状グラデーションは中心点から放射状に広がる円形または楕円形のグラデーションを作ります。例を示します：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::

これは中心が赤で、端が青に移行する円形のグラデーションを作成します。放射状グラデーションでは形状やサイズも指定できます：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="custom-radial-gradient"></div>
```

```css
.custom-radial-gradient {
  background: radial-gradient(ellipse at top left, red, blue);
  height: 40vh;
}
```

:::

これは要素の左上隅から始まる楕円形のグラデーションを作成します。線形グラデーションと放射状グラデーションの両方で、色ストップを正確な位置に指定できます。例えば：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="precise-gradient"></div>
```

```css
.precise-gradient {
  background: linear-gradient(to right, red 20%, yellow 40%, blue 80%);
  height: 40vh;
}
```

:::

これは赤が最初の`20%`を占め、次に黄色が`40%`で、最後に青が`80%`で移行するグラデーションを作成します。

グラデーションはCSSで画像を使う場所ならどこでも使えます。これにより、背景、ボタン、他のCSSプロパティと組み合わせたテキスト効果など、さまざまな用途に活用できます。

グラデーションはウェブサイトの視覚的な魅力を高めますが、特にテキストの背後にグラデーションを使う場合は、アクセシビリティのために十分なコントラストを確保することが重要です。

## --answers--
