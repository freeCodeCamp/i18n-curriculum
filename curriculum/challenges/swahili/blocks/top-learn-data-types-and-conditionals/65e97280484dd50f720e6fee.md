# --description--

Katika mfano uliopita, ulikuwa na chaguzi mbili `if` au `else`. Lakini vipi kama unataka kuwa na chaguzi zaidi ya mbili? Unaweza kutumia usemi wa `else if` kuongeza masharti zaidi kwenye msimbo wetu. Usemi wa `else if` unaturuhusu kuangalia masharti mengi na kutekeleza kipande cha msimbo wakati sharti la kwanza ni kweli. Ikiwa sharti la kwanza si kweli, litaangalia sharti lifuatalo na kadhalika. Ikiwa hakuna sharti lolote ambalo ni kweli, kipande cha `else` kitatekelezwa.

Hapa kuna mfano wa usemi wa `else if`:

```javascript
let x = 5;

if (x > 10) {
  console.log("x is greater than 10");
} else if (x > 5) {
  console.log("x is greater than 5");
} else {
  console.log("x is less than or equal to 5");
}
```

Katika mfano huu, kigezo cha `x` kinalinganishwa na nambari `10` kwa kutumia kielekezi cha `>`. Ikiwa `x` ni kubwa kuliko `10`, kipande cha kwanza cha msimbo kitaendeshwa. Vinginevyo, usemi wa `else if` utaangalia sharti lifuatalo. Ikiwa `x` ni kubwa kuliko `5`, kipande cha pili cha msimbo kitaendeshwa. Vinginevyo, kipande cha `else` kitaendeshwa.
