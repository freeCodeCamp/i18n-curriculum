## --text--

Quale frase usa correttamente `I'd rather`?

## --answers--

`I'd rather use Python for data analysis than JavaScript.`

---

`I'd rather to use Python for data analysis than JavaScript.`

### --feedback--

`Rather` è seguito dalla forma base del verbo, non da `to`.

---

`I'd rather using Python for data analysis than JavaScript.`

### --feedback--

`Rather` è seguito dalla forma base del verbo, non da `using`.

---

`I'd rather use Python to data analysis than JavaScript.`

### --feedback--

La frase `to data analysis` dovrebbe essere `for data analysis`.

# --explanation--

`I'd rather` è una frase usata per esprimere preferenza tra due azioni o opzioni. È seguita dalla forma base del verbo.
