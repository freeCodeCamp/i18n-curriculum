# --description--

前のチャレンジで `splice()` が最大3つのパラメータを取れると説明しましたね。実は、3つ目のパラメータとして1つ以上の要素を渡すことで、配列に要素を追加できます。これは、ある要素や複数の要素を別のものに素早く入れ替えるのに非常に便利です。

```js
const numbers = [10, 11, 12, 12, 15];
const startIndex = 3;
const amountToDelete = 1;

numbers.splice(startIndex, amountToDelete, 13, 14);
console.log(numbers);
```

`12` の2回目の出現は削除され、同じインデックスに `13` と `14` が追加されます。`numbers` 配列は今や `[ 10, 11, 12, 13, 14, 15 ]` となります。

ここでは、数値の配列から始めます。次に `splice()` に以下を渡します：要素の削除を開始するインデックス（3）、削除する要素の数（1）、そして残りの引数（13、14）は同じインデックスから挿入されます。`amountToDelete` の後にはカンマで区切られた任意の数の要素を続けることができ、それぞれが挿入されます。

# --instructions--

HTMLカラーの配列を引数に取る `htmlColorNames` 関数が定義されています。`splice()` を使って配列の最初の2つの要素を削除し、それぞれの場所に `'DarkSalmon'` と `'BlanchedAlmond'` を追加するように関数を修正してください。
