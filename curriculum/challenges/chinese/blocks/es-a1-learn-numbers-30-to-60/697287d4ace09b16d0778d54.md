# --description--

现在让我们来看 50 多的数字。

在西班牙语中，数字 50 被称为 `cincuenta`。

51 到 59 的数字是通过使用十位词 `cincuenta`（50），后跟连词 `y`（“和”），然后是个位数字（1-9）组成的。

| Number | Spanish |
| :--- | :--- |
| 50 | `Cincuenta` |
| 51 | `Cincuenta y uno` |
| 52 | `Cincuenta y dos` |
| 53 | `Cincuenta y tres` |
| 54 | `Cincuenta y cuatro` |
| 55 | `Cincuenta y cinco` |
| 56 | `Cincuenta y seis` |
| 57 | `Cincuenta y siete` |
| 58 | `Cincuenta y ocho` |
| 59 | `Cincuenta y nueve` |

数字 `cincuenta y uno`（51）在此范围内是独特的，因为它在阳性名词前变为 `cincuenta y un`。例如：

`Cincuenta y un años` - 五十一年。
