# --interactive--

في البرمجة، فهم كيفية تمثيل الأحرف كأرقام أمر أساسي. هنا يأتي دور ASCII. ASCII، اختصار لـ American Standard Code for Information Interchange، هو معيار ترميز الأحرف المستخدم في الحواسيب لتمثيل النصوص. يعين قيمة رقمية لكل حرف، وهذه القيمة معترف بها عالميًا من قبل الأجهزة.

في هذا الدرس، سنستكشف ما هو ASCII، كيف يعمل، وكيف ترتبط طرق JavaScript مثل `charCodeAt()` و `fromCharCode()` بترميز الأحرف. رغم أن سلاسل JavaScript تستخدم Unicode (UTF-16) داخليًا، فإن قيم ASCII تطابق أول 128 حرفًا من Unicode، ولهذا تعمل الأمثلة المبنية على ASCII في JavaScript.

ASCII هو نظام لترميز الأحرف مثل الحروف والأرقام والرموز إلى قيم رقمية. يُربط كل حرف برقم محدد.

على سبيل المثال، الحرف الكبير `A` يُمثل بالرقم `65` في ASCII، بينما الحرف الصغير `a` يُمثل بالرقم `97`. هذا الترميز يسمح للحواسيب بتخزين النصوص والتعامل معها.

معيار ASCII يغطي 128 حرفًا تشمل:

- الحروف الإنجليزية الكبيرة والصغيرة (A-Z, a-z).
- الأرقام (0-9).  
- علامات الترقيم والرموز الشائعة (!, @, #، وهكذا).
- أحرف التحكم (مثل السطر الجديد والمسافة الأفقية).

في JavaScript، يمكنك الوصول إلى الكود الرقمي لحرف باستخدام طريقة `charCodeAt()`. تعيد هذه الطريقة وحدة كود UTF-16 للحرف عند فهرس محدد. بالنسبة لأول 128 حرفًا، تطابق هذه القيمة كود ASCII.

لنلق نظرة على مثال:

:::interactive_editor

```js
let letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

:::

في هذا المثال، `A` هو الحرف الأول في السلسلة، واستدعاء `charCodeAt(0)` يعيد الكود الرقمي له (الذي يطابق قيمة ASCII للحروف اللاتينية الأساسية)، وهو `65`.

يمكنك أيضًا استخدام هذه الطريقة مع أحرف أخرى لمعرفة قيمتها الرقمية:

:::interactive_editor

```js
let symbol = "!";
console.log(symbol.charCodeAt(0));  // 33
```

:::

هنا، الكود الرقمي لعلامة التعجب `!` يُعاد كـ `33` (وهو يطابق قيمة ASCII لها).

بينما تساعدك `charCodeAt()` في استرجاع الكود الرقمي لحرف، تسمح لك طريقة `fromCharCode()` بالعكس: تحويل وحدة كود UTF-16 (التي تطابق ASCII للحروف الأساسية) إلى الحرف المقابل.

لنرَ هذا عمليًا:

:::interactive_editor

```js
let char = String.fromCharCode(65);
console.log(char);  //  A
```

:::

في هذا المثال، `fromCharCode(65)` يحول الكود الرقمي `65` (الذي يطابق قيمة ASCII لـ `A`) مرة أخرى إلى الحرف `A`.

مثال آخر هو تحويل الرقم `97` إلى الحرف الصغير المقابل:

:::interactive_editor

```js
let char = String.fromCharCode(97);
console.log(char);  // a
```

:::

هذه الطرق مفيدة جدًا عندما تحتاج إلى التعامل مع الأحرف أو مقارنتها بناءً على قيمها الرقمية.

على سبيل المثال، قد تستخدم `charCodeAt()` للتحقق مما إذا كان الحرف كبيرًا أو صغيرًا أو رقمًا بمقارنة قيمته في ASCII.

من ناحية أخرى، يمكن استخدام `fromCharCode()` لتوليد الأحرف ديناميكيًا من أكواد ASCII الخاصة بها.
