# --interactive--

Asilimia katika CSS ni vitengo vinavyohusiana ambavyo hukuruhusu kufafanua ukubwa, vipimo, na sifa nyingine kama sehemu ya kipimo cha kontena lake la mzazi. Unapotumia thamani ya asilimia, kwa msingi unamaanisha, "Fanya hii iwe X% ya kontena lake."

Kwa mfano, ukipanga `width: 50%` kwenye kipengele, kitachukua nusu ya upana wa kontena lake la mzazi. Hii inafanya asilimia kuwa muhimu sana kwa kuunda miundo inayojibadilisha kulingana na kifaa tofauti.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <div class="box"></div>
</div>
```

```css
.container {
  width: 400px;
  height: 200px;
  background-color: lightgray;
}

.box {
  width: 50%;
  height: 100%;
  background-color: red;
}
```

:::

Asilimia ni bora kwa kuunda mipangilio inayobadilika kwa urahisi kulingana na ukubwa wa skrini tofauti. Kwa mfano, kuweka kontena kuwa `width: 80%` kunahakikisha kinachukua 80% ya upana wa mzazi wake, bila kujali kifaa kinachotumika.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="parent">
  <div class="child"></div>
</div>
```

```css
.parent {
  width: 100%;
  height: 300px;
  background-color: lightblue;
}

.child {
  width: 80%;
  height: 100%;
  background-color: red;
}
```

:::

Kutumia asilimia kwa picha zinazobadilika ni desturi nyingine ya kawaida. Kwa kutumia `max-width: 100%` kwa picha, unawawezesha kupungua ukubwa kwenye skrini ndogo huku wakidumisha uwiano wa vipimo vyao.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<img src="https://placehold.co/150x150" alt="Example Product Image" />
```

```css
img {
  max-width: 100%;
  height: auto;
}
```

:::

Ingawa si za kawaida sana, asilimia pia zinaweza kutumika kwa ukubwa wa fonti ili kuunda maandishi yanayoweza kubadilika ukubwa. Kwa mfano, `font-size: 120%` itafanya maandishi kuwa 20% kubwa zaidi kuliko ukubwa wa fonti wa mzazi wake.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="text-container">
  Parent text.
  <p class="text">This is some example text.</p>
</div>
```

```css
.text-container {
  font-size: 16px;
}

.text {
  font-size: 120%;
}
```

:::

Asilimia zinaweza kuwa zenye msaada sana kwa kuweka kitu katikati kwa wima. Hapa kuna mfano wa jinsi unavyoweza kutumia asilimia na sifa ya `transform` kuweka kipengele katikati kwa wima.

Mfano huu unaweka kipengele 50% kutoka juu ya kontena lake, kisha hutumia `transform` kukisogeza juu kwa nusu ya urefu wake, kwa ufanisi kikawa katikati kwa wima.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="centered"></div>
```

```css
.centered {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
  width: 300px;
  height: 300px;
  background-color: red;
}
```

:::

Utajifunza zaidi kuhusu jinsi kuweka nafasi ya kudumu (absolute positioning) na sifa za transform zinavyofanya kazi kwa undani zaidi katika mafundisho yajayo.

Kumbuka, asilimia daima ni kwa uhusiano na kitu kingine. Kwa sifa za usawa kama `width`, ni kwa uhusiano na upana wa mzazi. Kwa sifa za wima kama `height`, kawaida ni kwa uhusiano na urefu wa mzazi (ikiwa umefafanuliwa).

Hata hivyo, kuwa makini unapopanga ndani vipengele vyenye vipimo vya asilimia, kwani hii inaweza kusababisha matokeo yasiyotegemewa. Pia, kumbuka kuwa urefu wa asilimia unaweza kuwa mgumu ikiwa mzazi hana urefu uliowekwa.

Kwa muhtasari, asilimia katika CSS ni zana zenye nguvu za kuunda miundo inayobadilika na inayojibadilisha kulingana na kifaa. Zitumiapo unapohitaji vipengele kubadilika kwa uwiano na kontena lao au unapohitaji mipangilio inayobadilika kwa urahisi kwa ukubwa tofauti wa skrini.
