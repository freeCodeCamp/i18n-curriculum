# --explanation--

Jake는 보안 침해를 조사하는 것이 시급하다고 강조합니다.

`Need to` + 동사는 중요한 일이나 필요한 일을 표현할 때 자주 쓰는 구조입니다. 예를 들어:

`We need to improve security.` - 이것은 보안을 향상시키는 것이 필요하다는 뜻입니다.

`Find out`는 정보를 발견하거나 알아내는 것을 의미합니다. 예를 들어:

`I need to find out who accessed the system.` - 이것은 화자가 조사하고 정보를 얻어야 한다는 뜻입니다.
