# --explanation--

`Me too`는 동의를 표현하거나 같은 내용이 자신에게도 해당함을 나타낼 때 사용합니다. 이는 공통된 경험이나 감정을 간결하고 비공식적으로 응답하는 방법입니다. 예를 들어:

- **Jennie:** `I love Python programming.`

- **Dan:** `Me too.` - 이 사람도 Python 프로그래밍을 좋아합니다.

이 상황에서 Maria는 `Yeah, me too`라고 말해 Brian에게 자신도 PyCon에서 정말 흥미로운 세션에 참석했다는 것을 빠르게 알릴 수 있습니다.
