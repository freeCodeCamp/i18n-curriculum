# --description--

可数名词是你可以计数的事物。它们可以是 1 个、2 个、3 个或更多。你可以使用数字和量词，例如 `many`、`a few` 或 `several` 来描述可数名词。例如，apple：你可以有 `one apple`、`two apples` 或 `many apples`。

不可数名词是指你无法一个一个计数的事物。它们通常不会变化以表示复数。你不会直接使用数字，而是使用像 `much`、`a little`、`some` 这样的量词来描述不可数名词。例如，water：你不能说 `one water` 或 `two waters`。你可以说 `much water`、`some water`。
