# --interactive--

`reduce` 메서드는 JavaScript에서 배열을 처리하여 하나의 값으로 압축할 수 있게 해주는 함수입니다. 이 하나의 값은 숫자, 문자열, 객체 또는 또 다른 배열일 수 있습니다.

이 메서드는 배열을 하나의 출력값으로 줄인다는 의미에서 `reduce`라고 불립니다. 처음에는 복잡해 보일 수 있지만 `reduce`를 이해하면 많은 상황에서 코드를 훨씬 간단하게 만들 수 있습니다.

본질적으로 `reduce`는 배열의 각 요소에 순서대로 함수를 적용하고, 각 계산 결과를 다음 계산에 전달하는 방식으로 작동합니다. 이 함수는 보통 리듀서 함수라고 부릅니다.

리듀서 함수는 두 가지 주요 매개변수를 받습니다: 누산기(accumulator)와 현재 값(current value)입니다. 누산기는 연산의 누적 결과를 저장하는 곳이고, 현재 값은 처리 중인 배열 요소입니다.

`reduce`가 어떻게 작동하는지 예제를 통해 살펴보겠습니다:

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];
const sum = numbers.reduce(
  (accumulator, currentValue) => accumulator + currentValue,
  0
);

console.log(sum); // 15
```

:::

이 예제에서는 `reduce`를 사용해 배열에 있는 모든 숫자의 합을 구하고 있습니다.

리듀서 함수는 누산기(두 번째 인수로 `0`에서 시작하며 `reduce`에 지정됨)를 받아 각 숫자를 더합니다.

각 덧셈의 결과가 다음 반복에서 새로운 누산기가 됩니다.

`reduce` 메서드는 두 번째 인수로 초기값을 받을 수도 있습니다. 이 값이 누산기의 시작 값입니다. 위 예제에서는 `0`로 설정했습니다.

초기값을 제공하지 않으면 `reduce`는 배열의 첫 번째 요소를 초기 누산기로 사용하고 두 번째 요소부터 연산을 시작합니다.

`reduce`의 장점 중 하나는 유연성입니다. 리듀서 함수를 직접 정의하기 때문에 배열을 어떻게 처리하고 어떤 결과를 만들지 완전히 제어할 수 있습니다. 이 때문에 `reduce`는 매우 강력하지만 처음에는 이해하기 어려울 수도 있습니다.

연습을 통해 `reduce` 메서드 사용에 익숙해질 것입니다.
