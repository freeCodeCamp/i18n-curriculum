# --interactive--

Ein Array in JavaScript ist eine geordnete Sammlung von Werten, die jeweils durch einen numerischen Index identifiziert werden. Die Werte in einem JavaScript-Array können unterschiedliche Datentypen haben, darunter Zahlen, Strings, Booleans, Objekte und sogar andere Arrays.

Um ein Array in JavaScript zu erstellen, können Sie eckige Klammern verwenden, `[]`, und die Werte mit Kommas trennen. Hier ist ein Beispiel:

```js
let fruits = ["apple", "banana", "orange"];
```

In diesem Beispiel deklarieren wir eine Variable `fruits` und weisen ihr ein Array zu, das drei String-Werte enthält: `apple`, `banana` und `orange`.

Eines der wichtigsten Merkmale von Arrays ist, dass sie nullbasiert indiziert sind, was bedeutet, dass das erste Element in einem Array den Index `0` hat, das zweite Element den Index `1` und so weiter. Sie können auf einzelne Elemente in einem Array über ihren Index zugreifen. Zum Beispiel:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits[0]); // "apple"
console.log(fruits[2]); // "orange"
```

:::

In diesem Beispiel verwenden wir den Index `0`, um auf das erste Element (`apple`) zuzugreifen, und den Index `2`, um auf das dritte Element (`orange`) zuzugreifen.

Arrays in JavaScript haben eine spezielle `length`-Eigenschaft, die die Anzahl der Elemente im Array zurückgibt. Sie können auf diese Eigenschaft mit dem Schlüsselwort `length` zugreifen. Zum Beispiel:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits.length); // 3
```

:::

Ein weiteres wichtiges Merkmal von Arrays in JavaScript ist, dass sie dynamisch sind, was bedeutet, dass sich ihre Größe nach der Erstellung ändern kann. Sie können Elemente zu einem Array hinzufügen oder daraus entfernen, indem Sie verschiedene Array-Methoden verwenden, wie `push()`, `pop()`, `shift()`, `unshift()`, `splice()` und mehr. Diese Methoden werden in den kommenden Lektionen vermittelt.

JavaScript-Arrays sind vielseitig und nützlich, wenn es darum geht, Daten innerhalb Ihrer Programme zu speichern. Im Verlauf dieses Moduls werden Sie aus erster Hand erleben, wie die Arbeit mit Arrays Ihnen hilft, Datenmengen effizient zu verwalten und zu manipulieren.
