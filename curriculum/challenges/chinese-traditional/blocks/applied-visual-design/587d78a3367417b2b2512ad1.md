# --description--

色彩理論及其對設計的影響是一個深奧的主題，以下挑戰僅涵蓋基本內容。在網站上，色彩可以吸引對內容的注意、喚起情感，或創造視覺和諧。使用不同的色彩組合確實可以改變網站的外觀，並且在選擇與你的內容相符的調色盤時會投入大量思考。

色輪是一個有用的工具，用來視覺化顏色之間的關係－它是一個圓圈，類似的色調彼此相鄰，而不同的色調則相距較遠。當兩個顏色在色輪上彼此相對時，它們被稱為互補色。它們具有這樣的特性：如果將它們組合在一起，會彼此「抵銷」並產生灰色。然而，當這些顏色並排放置時，會顯得更加鮮豔並產生強烈的視覺對比。

一些互補色及其十六進位色碼的範例有：

<blockquote>red (#FF0000) and cyan (#00FFFF)<br>green (#00FF00) and magenta (#FF00FF)<br>blue (#0000FF) and yellow (#FFFF00)</blockquote>

這與許多人在學校學到的過時 RYB 色彩模型不同，該模型具有不同的原色和互補色。現代色彩理論使用加法 RGB 模型（如在電腦螢幕上）和減法 CMY(K) 模型（如在列印中）。

線上有許多取色工具，這些工具具有尋找顏色互補色的選項。

**注意：** 使用顏色可以是為頁面添加視覺吸引力的強大方式。然而，僅使用顏色不應該是傳達重要資訊的唯一方式，因為有視覺障礙的使用者可能無法理解該內容。這個問題將在《應用無障礙挑戰》中更詳細地涵蓋。

# --instructions--

將 `blue` 和 `yellow` 類別的 `background-color` 屬性更改為它們各自的顏色。注意這些顏色並排時看起來與在白色背景下比較時不同。

# --hints--

具有類別 `blue` 的 `div` 元素應該有藍色的 `background-color`。

```js
const blueElement = document.querySelector('.blue');
const blueStyle = window.getComputedStyle(blueElement); 
assert.equal(blueStyle?.backgroundColor, 'rgb(0, 0, 255)');
```

具有類別 `yellow` 的 `div` 元素應該有黃色的 `background-color`。

```js
const yellowElement = document.querySelector('.yellow');
const yellowStyle = window.getComputedStyle(yellowElement);
assert.equal(yellowStyle?.backgroundColor, 'rgb(255, 255, 0)');
```
