# --explanation--

`Looks better` 意味著某物看起來更具吸引力或視覺上更悅目。 例如：

`The updated login screen looks better with the new icons。`－設計更佳且更具吸引力。

`Works better` 意味著某物執行得更高效或更容易使用。 例如：

`The search feature works better after the update。`－ 它執行得更快或提供更準確的結果。

結構 `not only ... but also ...` 用於表示兩個相互連結的正向變化。在此背景關係中，新版本在外觀和功能上都有所提升。
