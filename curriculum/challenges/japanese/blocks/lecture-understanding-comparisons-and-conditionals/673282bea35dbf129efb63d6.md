# --interactive--

`switch`文と`if/else if/else`チェーンはどちらもプログラミングにおける制御フロー構造で、特定の条件に基づいて異なるコードブロックを実行できます。しかし、それぞれ特徴や使いどころが異なります。

`switch`文は式を評価し、その値を一連の`case`節と照合します。マッチした場合、その`case`に関連付けられたコードブロックが実行されます。以下は`switch`文の基本構造です。

```js
switch (expression) {
  case value1:
    // code to be executed if expression === value1
    break;
  case value2:
    // code to be executed if expression === value2
    break;
  default:
    // code to be executed if expression doesn't match any case
}
```

各`break`の最後にある`case`文は非常に重要です。これは、マッチした`case`が実行された後にswitchブロックから抜けるようプログラムに指示します。これがないと、プログラムは次のケースも続けて実行してしまい、これを「フォールスルー」と呼びます。

`switch`文は通常、単一の変数を複数の可能な値と比較するときに使われます。特に、1つの変数に対して多くの条件をチェックする場合に便利です。以下は曜日に対して`switch`文を使った例です。

:::interactive_editor

```js
let dayOfWeek = 3; 

switch (dayOfWeek) {
    case 1:
        console.log("It's Monday! Time to start the week strong.");
        break;
    case 2:
        console.log("It's Tuesday! Keep the momentum going.");
        break;
    case 3:
        console.log("It's Wednesday! We're halfway there.");
        break;
    case 4:
        console.log("It's Thursday! Almost the weekend.");
        break;
    case 5:
        console.log("It's Friday! The weekend is near.");
        break;
    case 6:
        console.log("It's Saturday! Enjoy your weekend.");
        break;
    case 7:
        console.log("It's Sunday! Rest and recharge.");
        break;
    default:
        console.log("Invalid day! Please enter a number between 1 and 7.");
}
```

:::

`switch`文は、単一の変数に対して多くの可能な値を扱う場合に、より読みやすく簡潔になることがあります。

一方で`if/else if`文はより柔軟です。各節で複雑な条件や異なる変数を評価できるため、より幅広いシナリオに適しています。以下は`if/else`文を`switch`文より使うべき例です。

:::interactive_editor

```js
let creditScore = 720; 
let annualIncome = 60000; 
let loanAmount = 200000; 

let eligibilityStatus;

if (creditScore >= 750 && annualIncome >= 80000) {
    eligibilityStatus = "Eligible for premium loan rates.";
} else if (creditScore >= 700 && annualIncome >= 50000) {
    eligibilityStatus = "Eligible for standard loan rates.";
} else if (creditScore >= 650 && annualIncome >= 40000) {
    eligibilityStatus = "Eligible for subprime loan rates.";
} else if (creditScore < 650) {
    eligibilityStatus = "Not eligible due to low credit score.";
} else {
    eligibilityStatus = "Not eligible due to insufficient income.";
}

console.log(eligibilityStatus);
```

:::

この例では、ある人の年収とクレジットスコアをもとに、どの種類のローンが利用可能かをチェックしています。複雑な論理評価と複数の変数を扱うため、`if/else`文を使うほうが`switch`文より適しています。

JavaScriptの`switch`文は厳密な比較（`===`）を使うことも覚えておくと良いでしょう。これは型変換を行わないため、予測可能性が高く微妙なバグを避けるのに役立ちます。

まとめると、`switch`文と`if/else if`チェーンはどちらも複数分岐のロジックを可能にしますが、それぞれ強みが異なります。`switch`文は単一の変数に対する複数の可能な値を扱うのに優れ、`if/else if`チェーンは複雑な条件に対してより柔軟です。どちらを使うかはコードの具体的な要件や個人・チームのコーディングスタイルによります。
