# --interactive--

JavaScriptでは、テンプレートリテラルは文字列を扱うための強力で柔軟な方法です。通常の文字列はシングル(`'`)または二重引用符(`"`)を使いますが、テンプレートリテラルはバックティック(`` ` ``)で定義されます。

テンプレートリテラルは、文字列の中に変数を直接埋め込むことができる文字列の補間という機能を含め、文字列操作をより簡単にします。

テンプレートリテラルを使うと、複数行にまたがる文字列や、文字列の中に式（変数やJavaScriptのコードなど）を直接含めることが簡単になります。

以下はテンプレートリテラルの例です：

:::interactive_editor

```js
const name = "Alice";
const greeting = `Hello, ${name}!`;

console.log(greeting);
```

:::

シングルや二重引用符の代わりにバックティックを使っていることに注目してください。`${name}`構文は文字列の補間の例で、変数`name`の値が文字列の中に直接挿入されています。

文字列の補間は、文字列の中に変数や式を埋め込むことを可能にします。これは、以前のように`+`演算子を使って文字列と変数を連結していた方法に比べて大きな改善です。

以下は文字列の連結とプラス(`+`)演算子を使った例です：

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = "My name is " + name + " and I am " + age + " years old.";
console.log(message); 
```

:::

そしてこちらはテンプレートリテラルと文字列の補間を使った例です：

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = `My name is ${name} and I am ${age} years old.`;
console.log(message); 
```

:::

ご覧の通り、テンプレートリテラルの文字列補間は、特に複数の変数を扱う場合に、はるかにすっきりして読みやすいです。

テンプレートリテラルのもう一つの素晴らしい機能は、複数行の文字列をサポートしていることです。通常の文字列では改行を作るためにエスケープ文字(`\n`)を使う必要がありますが、テンプレートリテラルでは単に複数行にわたって文字列を書けば、そのままの書式が保持されます：

:::interactive_editor

```js
let poem = `Roses are red,
Violets are blue,
JavaScript is fun,
And so are you.`;

console.log(poem);
```

:::

テンプレートリテラルのもう一つの機能は、文字列の中にJavaScriptの式を直接埋め込めることです。以下の例をご覧ください：

:::interactive_editor

```js
const song = "Bohemian Rhapsody";
const score = 9.5;
const highestScore = 10;
const output = `One of my favorite songs is "${song}". I rated it ${
  (score / highestScore) * 100
}%.`;
console.log(output); 
```

:::

テンプレートリテラルは、文字列に変数や式を含めたい場合、複雑な文字列をフォーマットしたい場合、または複数行のテキストを扱いたい場合に特に便利です。従来の文字列連結に比べて、より簡潔で読みやすくなっています。
