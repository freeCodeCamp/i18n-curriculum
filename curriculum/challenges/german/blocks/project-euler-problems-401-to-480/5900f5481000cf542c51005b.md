# --description--

Sei $R(a, b, c)$ die maximale Fläche, die von drei sich nicht überlappenden Kreisen innerhalb eines Dreiecks mit den Seitenlängen $a$, $b$ und $c$ bedeckt wird.

Sei $S(n)$ der Durchschnittswert von $R(a, b, c)$ über alle ganzzahligen Tripel $(a, b, c)$, für die gilt: $1 ≤ a ≤ b ≤ c &lt; a + b ≤ n$.

Es ist gegeben, dass $S(2) = R(1, 1, 1) ≈ 0,31998$ und $S(5) ≈ 1,25899$.

Berechnen Sie $S(1803)$, gerundet auf 5 Dezimalstellen hinter dem Komma.
