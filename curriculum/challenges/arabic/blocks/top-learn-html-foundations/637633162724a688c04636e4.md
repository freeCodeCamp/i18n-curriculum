# --description--

HTML و CSS تهتمان بشكل أساسي بعرض المعلومات. لا تُستخدمان لبرمجة المنطق. JavaScript، التي ستتعلمها في القسم التالي، تُستخدم لجعل صفحات الويب تقوم بأفعال. يمكنك فعل الكثير باستخدام HTML و CSS فقط، وستحتاج إليهما بالتأكيد معًا. طوال منهجنا، تركز الدروس التالية على تزويدك بالأدوات التي تحتاجها للنجاح عند وصولك إلى محتوى JavaScript.
