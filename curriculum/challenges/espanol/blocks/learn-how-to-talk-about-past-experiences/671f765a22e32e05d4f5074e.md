# --explanation--

La palabra `Absolutely` se usa para expresar un acuerdo fuerte o afirmación. A menudo se utiliza en respuesta a una declaración o pregunta para enfatizar certeza o entusiasmo. Por ejemplo: 

Persona 1: `Do you think this approach will work?`

Persona 2: `Absolutely!` - Esto significa que estás seguro de que la estructura tendrá éxito. 

En este contexto, Brian podría decir `Absolutely` para confirmar con entusiasmo que recuerda haberse emocionado con la nueva tecnología en el trabajo.
