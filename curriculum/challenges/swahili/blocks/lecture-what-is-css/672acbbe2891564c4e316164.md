# --interactive--

CSS inahusika na mitindo ya ukurasa wa mtandao. Mitindo yote hii inaundwa na sheria mbalimbali za CSS.

Sheria ya CSS ina sehemu kuu mbili: **kichaguzi** na sehemu ya **tamko**.

Tuchunguze sintaksia ya msingi:

```css
selector {
  property: value;
}
```

Kichaguzi ni muundo unaotumika katika CSS kutambua na kulenga vipengele maalum vya HTML kwa ajili ya kupamba.

Mifano ya vichaguzi ni pamoja na vichaguzi vya aina, vichaguzi vya darasa la vitu, na vichaguzi vya ID.

Mabano ya mviringo yaliyopo katika sintaksia ya msingi yanajulikana kama sehemu ya tamko. Sehemu ya tamko hutumia seti ya mitindo kwa kichaguzi fulani, au vichaguzi.

Ndani ya sehemu ya tamko, utakuwa na mfululizo wa tamko. Kila tamko lina sifa na thamani.

Sifa ni kitambulisho cha CSS kinachoeleza kipengele kinachopambwa. Mfano wa sifa ni sifa ya `background-color`.

Thamani ni usanidi maalum unaotumika kwa sifa hiyo. Kwa mfano, kama sifa ni `background-color`, thamani inaweza kuwa `purple`, ambayo huweka rangi ya nyuma kuwa ya zambarau.

Baada ya kila jina la sifa, unahitaji kuweka nukta mbili, na baada ya kila thamani, unapaswa kuweka nukta ya nusu.

Sasa unajua sintaksia ya sheria ya CSS, tuchunguze mfano. Washa mhariri wa kuingiliana na bofya kichupo cha `styles.css` kuona msimbo wa CSS.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Learning about CSS</h1>

<p>Example paragraph element</p>
<p>Another example paragraph element</p>
```

```css
p {
  color: blue;
}
```

:::

Katika sheria hii ya CSS, kichaguzi cha aina kinawalenga vipengele vyote vya aya kwenye ukurasa.

Ndani ya sehemu ya tamko, kuna tamko moja.

Tamko lina sifa ya `color` na thamani iliyowekwa kuwa `blue`. Hii itabadilisha rangi ya maandishi kwa vipengele vyote vya aya kuwa `blue`.

Ikiwa unataka kutumia seti ile ile ya mitindo kwa vichaguzi vingi, unaweza kuunda orodha ya vichaguzi ambapo kila kichaguzi kimegawanywa kwa koma.

Hapa kuna mfano wa kupamba vichaguzi vingi (kwa kuingiliana na mfano, washa mhariri wa kuingiliana): 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1 id="title">Example heading</h1>
<h2 class="subheading">Example subheading</h2>
<p>This paragraph is not affected by the selector.</p>
```

```css
#title,
.subheading {
  color: navy;
}
```

:::

Katika orodha hii ya vichaguzi, kuna kichaguzi cha `id` kinacholenga kipengele cha HTML chenye thamani ya `id` ya `title`. Vichaguzi vyote vya `id` lazima vianze na alama ya hash `#`.

Kisha kuna koma ikifuatiwa na kichaguzi cha `class` kinacholenga vipengele vyote vya HTML vyenye thamani ya `class` ya `subheading`. Vichaguzi vyote vya darasa la vitu lazima vianze na nukta `.`.

Orodha yote ya vichaguzi itapokea mtindo ule ule, na rangi ya maandishi itakuwa `navy`.
