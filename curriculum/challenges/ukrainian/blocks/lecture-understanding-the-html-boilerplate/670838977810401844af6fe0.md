# --description--

Давайте дізнаємося про елемент `link` і як ним користуватися для зв’язку із зовнішніми таблицями стилів.

Елемент `link` використовується для зв’язку із зовнішніми ресурсами, такими як таблиці стилів і іконки сайту. Ось базовий синтаксис використання елемента `link` для зовнішнього CSS-файлу:

```html
<link rel="stylesheet" href="./styles.css" />
```

Атрибут `rel` використовується, щоб вказати зв’язок між пов’язаним ресурсом і HTML-документом. У цьому випадку потрібно вказати, що цей пов’язаний ресурс є `stylesheet`.

Вважається найкращою практикою розділяти HTML і CSS у різних файлах. Розробники використовують елемент `link` для свого зовнішнього CSS-файлу замість того, щоб писати все в HTML-документі.

Атрибут `href` використовується, щоб вказати розташування URL для зовнішнього ресурсу.

`dot`, за яким іде коса риска у прикладі, каже комп’ютеру шукати у поточній папці, або директорії, файл `styles.css`.

Елемент `link` слід розміщувати всередині елемента `head`, як показано в наступному прикладі:

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Examples of the link element</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

Часто ви побачите кілька елементів `link` у професійному коді, які зв’язують різні таблиці стилів, шрифти та іконки. Ось приклад використання елемента `link` для зв’язку із зовнішнім шрифтом Google під назвою *Playwright Cuba*:

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Playwrite+CU:wght@100..400&display=swap"
  rel="stylesheet"
/>
```

Google Fonts — це множина безкоштовних і відкритих кастомних шрифтів, які ви можете використовувати в будь-якому проєкті. Ви можете обрати, які шрифти хочете використати, і Google надасть вам необхідний HTML і CSS код. У цьому прикладі значення `preconnect` для атрибута `rel` каже браузеру створити раннє з’єднання з адресою, вказаною в атрибуті `href`. Це робиться для прискорення часу завантаження цих зовнішніх ресурсів.

Ще одним загальним випадком використання елемента `link` є зв’язок з іконками. Ось приклад зв’язку з favicon:

```html
<link rel="icon" href="favicon.ico" />
```

Favicon, що є скороченням від favorite icon, — це маленька іконка, яка зазвичай відображається на вкладці браузера поруч із назвою сайту. Багато сайтів використовують favicon, щоб показати свою бренд-іконку.
