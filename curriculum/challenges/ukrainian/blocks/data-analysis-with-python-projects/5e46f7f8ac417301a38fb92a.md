## Опис даних

Рядки в наборі даних представляють пацієнтів, а стовпці — інформацію, таку як виміри тіла, результати різних аналізів крові та спосіб життя. Ви використаєте набір даних, щоб дослідити зв’язок між серцево-судинними захворюваннями, вимірами тіла, маркерами крові та способом життя.

Назва файлу: medical_examination.csv

| Функція | Тип змінної | Змінна      | Тип значення |
|:-------:|:------------:|:-------------:|:----------:|
| Age | Об’єктивна функція | `age` | int (дні) |
| Height | Об’єктивна функція | `height` | int (см) |
| Weight | Об’єктивна функція | `weight` | float (кг) |
| Gender | Об’єктивна функція | `gender` | категоріальний код |
| Систолічний артеріальний тиск | Функція огляду | `ap_hi` | int |
| Діастолічний артеріальний тиск | Функція огляду | `ap_lo` | int |
| Холестерин | Функція огляду | `cholesterol` | 1: нормальний, 2: вище норми, 3: значно вище норми |
| Глюкоза | Функція огляду | `gluc` | 1: нормальний, 2: вище норми, 3: значно вище норми |
| Куріння | Суб’єктивна функція | `smoke` | бінарний |
| Вживання алкоголю | Суб’єктивна функція | `alco` | бінарний |
| Фізична активність | Суб’єктивна функція | `active` | бінарний |
| Наявність або відсутність серцево-судинних захворювань | Цільова змінна | `cardio` | бінарний |


## Інструкції

Створіть діаграму, подібну до `examples/Figure_1.png`, де ми показуємо кількість хороших і поганих результатів для змінних `cholesterol`, `gluc`, `alco`, `active` та `smoke` для пацієнтів з `cardio=1` і `cardio=0` у різних панелях.

За кожним числом у файлі `medical_data_visualizer.py` додайте код із відповідного номера інструкції нижче.

1. Імпортуйте дані з `medical_examination.csv` і призначте їх змінній `df`.
2. Додайте стовпець `overweight` до даних. Щоб визначити, чи є людина з надмірною вагою, спочатку обчисліть її ІМТ, поділивши вагу в кілограмах на квадрат зросту в метрах. Якщо це значення > 25, то людина має надмірну вагу. Використовуйте значення `0` для НЕ надмірної ваги і значення `1` для надмірної ваги.
3. Нормалізуйте дані, зробивши `0` завжди хорошим, а `1` завжди поганим. Якщо значення `cholesterol` або `gluc` дорівнює 1, встановіть значення `0`. Якщо значення більше за `1`, встановіть значення `1`.
4. Намалюйте категорійну діаграму у функції `draw_cat_plot`.
5. Створіть таблицю даних для категорійної діаграми, використовуючи `pd.melt` зі значеннями `cholesterol`, `gluc`, `smoke`, `alco`, `active` та `overweight` у змінній `df_cat`.
6. Групуйте та переформатуйте дані у `df_cat`, щоб розбити їх за `cardio`. Покажіть кількість кожної функції. Вам доведеться перейменувати один зі стовпців, щоб `catplot` працював правильно.
7. Перетворіть дані у формат `long` і створіть діаграму, що показує кількість значень категоріальних функцій, використовуючи метод, наданий бібліотекою seaborn: `sns.catplot()`.
8. Отримайте фігуру для вихідних даних і збережіть її у змінній `fig`.
9. Не змінюйте наступні два рядки.
10. Намалюйте теплову карту у функції `draw_heat_map`.
11. Очистіть дані у змінній `df_heat`, відфільтрувавши такі сегменти пацієнтів, які представляють некоректні дані:
    - діастолічний тиск вищий за систолічний (Залиште коректні дані з `(df['ap_lo'] <= df['ap_hi'])`)
    - зріст менший за 2,5-й перцентиль (Залиште коректні дані з `(df['height'] >= df['height'].quantile(0.025))`)
    - зріст більший за 97,5-й перцентиль
    - вага менша за 2,5-й перцентиль
    - вага більша за 97,5-й перцентиль
12. Обчисліть матрицю кореляції і збережіть її у змінній `corr`.
13. Згенеруйте маску для верхнього трикутника і збережіть її у змінній `mask`.
14. Налаштуйте фігуру `matplotlib`.
15. Побудуйте матрицю кореляції, використовуючи метод, наданий бібліотекою `seaborn`: `sns.heatmap()`.
16. Не змінюйте наступні два рядки.
