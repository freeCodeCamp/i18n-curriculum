# --description--

상수(변하지 않는) 변수를 선언하려면 `const` 대신 `let`을 사용하세요:

```javascript
const myBirthday = '16-09-2003';
```

`const`로 선언된 변수는 "상수"라고 합니다. 이 변수는 다시 할당할 수 없습니다. 다시 할당을 시도하면 오류가 발생합니다:

```javascript
const myBirthday = '16-09-2003';

myBirthday = '01-01-2001'; // error, can't reassign the constant!
```

프로그래머가 변수가 절대 변하지 않을 것이라고 확신할 때, `const`로 선언하여 그 사실을 보장하고 모두에게 알릴 수 있습니다.

## --answers--

다시 할당이 오류 없이 성공합니다.

---

`const`로 선언된 변수는 다시 할당할 수 없기 때문에 오류가 발생합니다.

---

JavaScript가 다시 할당을 조용히 무시합니다.

---

변수 타입이 자동으로 `let`로 변경됩니다.
