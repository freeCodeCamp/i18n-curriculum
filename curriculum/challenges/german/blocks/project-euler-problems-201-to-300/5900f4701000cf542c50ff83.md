# --description--

Ein Spiel wird mit drei Steinbergen und zwei Spielern gespielt.

In jedem Zug entfernt der Spieler einen oder mehrere Steine aus den Bergen. Wenn der Spieler jedoch Steine aus mehr als einem Berg nimmt, muss aus jedem der ausgewählten Berge die gleiche Anzahl an Steinen entfernt werden.

Mit anderen Worten wählt der Spieler ein $N > 0$ und entfernt:

- $N$ Steine aus einem einzelnen Berg; oder
- $N$ Steine aus jedem von zwei beliebigen Bergen ($2N$ insgesamt); oder
- $N$ Steine aus jedem der drei Berge ($3N$ insgesamt).

Der Spieler, der die letzten Stein(e) nimmt, gewinnt das Spiel.

Eine Gewinnkonfiguration ist eine, bei der der erste Spieler einen Sieg erzwingen kann.

Zum Beispiel sind (0,0,13), (0,11,11) und (5,5,5) Gewinnkonfigurationen, weil der erste Spieler sofort alle Steine entfernen kann.

Eine Verlustkonfiguration ist eine, bei der der zweite Spieler einen Sieg erzwingen kann, egal was der erste Spieler macht.

Zum Beispiel sind (0,1,2) und (1,3,3) Verlustkonfigurationen: Jeder erlaubte Zug hinterlässt eine Gewinnkonfiguration für den zweiten Spieler.

Betrachten Sie alle Verlustkonfigurationen ($x_i$,$y_i$,$z_i$) mit $x_i ≤ y_i ≤ z_i ≤ 100$. Wir können überprüfen, dass $\sum (x_i + y_i + z_i) = 173\\,895$ für diese gilt.

Finden Sie $\sum (x_i + y_i + z_i)$, wobei ($x_i$,$y_i$,$z_i$) über die Verlustkonfigurationen mit $x_i ≤ y_i ≤ z_i ≤ 1000$ läuft.
