# --description--

`<head>` 元素是你放置关于网页的重要元-信息以及网页在浏览器中正确渲染所需内容的地方。在 `<head>` 内，你不应使用任何在网页上显示内容的元素。

## 字符集 Meta 元素
你应该始终在 `head` 元素中为网页的 `charset` 编码添加 `meta` 标签：`<meta charset="utf-8">`。

设置编码非常重要，因为它确保网页能够在浏览器中正确显示来自不同语言的特殊符号和字符。

## 标题 元素
你应该始终在 HTML 文档的头部包含的另一个元素是 `title` 元素：

```html
<title>My First Webpage</title>
```

`title` 元素用于为网页提供一个人类可读的标题，该标题显示在你网页的浏览器标签（页）中。

如果你没有包含 `title` 元素，网页的标题将默认使用它的文件名。在你的分支中，那将是 `index.html`，这对用户来说意义不大；如果用户打开了许多浏览器标签（页），这将使查找你的网页变得非常困难。

在一个 HTML `document` 的 `head` 中可以包含更多的元素。然而，目前你只需了解这里介绍的两个元素。你将在后续课程中逐步学习更多放入 `head` 的元素。

回到 `index.html` 文件中，让我们添加一个包含 `charset` `meta` 元素和 `title` 的 `head` 元素。`head` 元素位于 `HTML` 元素内，且应始终是打开的 `<html>` 标签下的第一个元素：


```html
<!DOCTYPE html>

<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Webpage</title>
  </head>
</html>
```
