Lo scroll infinito è un pattern di design che carica più contenuti man mano che l’utente scorre verso il basso nella pagina. Spesso viene usato su siti di social media come Twitter. Per esempio, se sei loggato e vuoi vedere più tweet, puoi scorrere verso il basso e verranno caricati altri tweet. Questo è un esempio di scroll infinito.

  }
}

Lo scroll infinito viene anche usato come sostituto della paginazione. La paginazione è un pattern di design che suddivide i contenuti in pagine. Viene spesso usata quando ci sono molti contenuti da mostrare. Un esempio di paginazione è quando cerchi qualcosa su Google e vedi i risultati su più pagine. Con la paginazione devi cliccare un pulsante per andare alla pagina successiva. Con lo scroll infinito, continui semplicemente a scorrere verso il basso e vengono caricati altri contenuti.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="pagination">
  <div class="post">Post 1</div>
  <div class="post">Post 2</div>
  <div class="post">Post 3</div>
  <!-- More posts will be on the next pages -->
</div>
<div class="pagination-controls">
  <button class="prev" disabled>Previous</button>
  <button class="next">Next</button>
</div>
<script src="index.js"></script>
```

```css
.pagination {
  display: flex;
  flex-direction: column;
  gap: 12px;
}

.post {
  padding: 12px;
  border: 1px solid #ccc;
  border-radius: 4px;
}

.pagination-controls {
  margin-top: 12px;
}

.pagination-controls button {
  padding: 8px 16px;
  margin-right: 8px;
}
```

```js
let currentPage = 1;
const postsPerPage = 3;
const totalPosts = 9; 
const totalPages = Math.ceil(totalPosts / postsPerPage);
const container = document.querySelector('.pagination');
const prevButton = document.querySelector('.prev');
const nextButton = document.querySelector('.next'); 

function renderPosts() {
  container.innerHTML = '';
  const start = (currentPage - 1) * postsPerPage;
  const end = start + postsPerPage;
  for (let i = start; i < end && i < totalPosts; i++) {
    const post = document.createElement('div');
    post.className = 'post';
    post.textContent = `Post ${i + 1}`;
    container.appendChild(post);
  }
  prevButton.disabled = currentPage === 1;
  nextButton.disabled = currentPage === totalPages;
}

prevButton.addEventListener('click', () => {
  if (currentPage > 1) {
    currentPage--;
    renderPosts();
  }
});

nextButton.addEventListener('click', () => {
  if (currentPage < totalPages) {
    currentPage++;
    renderPosts();
  }
});

renderPosts();
```

Quando incorpori lo scroll infinito nel tuo design, ci sono alcune buone pratiche da tenere a mente. La prima considerazione è fornire un pulsante "Carica altro" che carica il prossimo insieme di risultati quando l’utente ci clicca sopra. Questo è un buon modo per dare all’utente il controllo su quando vuole vedere più contenuti.

Un’altra considerazione è aggiungere un pulsante "Indietro". Questo dà agli utenti la possibilità di tornare alla pagina precedente senza dover scorrere tutto il percorso verso l’alto. Questo crea una migliore esperienza utente e dà loro più controllo sulla navigazione.

A volte vedrai design con un pulsante "Torna all’inizio" che riporta gli utenti in cima alla pagina dei risultati. Un’altra considerazione è fornire un indicatore di caricamento. Gli utenti dovrebbero avere un’indicazione chiara che altri contenuti stanno venendo caricati; altrimenti potrebbero pensare che la pagina sia rotta.

```js
const container = document.querySelector('.infinite-scroll');
const loadingIndicator = document.querySelector('.loading-indicator');
const backToTopBtn = document.getElementById('back-to-top');

window.addEventListener('scroll', () => {
  if (window.scrollY > 400) {
    backToTopBtn.style.display = 'block';
  } else {
    backToTopBtn.style.display = 'none';
  }

  if (window.innerHeight + window.scrollY >= document.body.offsetHeight) {
    loadMorePosts();
  }
});

backToTopBtn.addEventListener('click', () => {
  window.scrollTo({ top: 0, behavior: 'smooth' });
});

function loadMorePosts() {
  loadingIndicator.style.display = 'block';
  setTimeout(() => {
    for (let i = 0; i < 3; i++) {
      const post = document.createElement('div');
      post.className = 'post';
      post.textContent = `Post ${container.children.length + 1}`;
      container.appendChild(post);
    }
    loadingIndicator.style.display = 'none';
  }, 1000);
}
```

Una delle ultime considerazioni è mantenere il piè di pagina accessibile all’utente. Se il piè di pagina contiene informazioni importanti, allora dovrebbe essere accessibile in ogni momento.

In conclusione, lo scroll infinito è un ottimo modo per mostrare contenuti sul tuo sito web. Tuttavia, dovresti tenere a mente le buone pratiche quando progetti il tuo scroll infinito per offrire la migliore esperienza utente possibile.
