# --description--

`Encourage` significa dar a alguien apoyo o confianza para hacer algo. Por ejemplo, cuando tu amigo dice, `You can solve this coding problem!` te está animando.

`Discourage` es lo opuesto; significa hacer que alguien se sienta menos confiado o menos dispuesto a hacer algo. If that friend says, `That programming language is too hard for beginners`, it might discourage you from learning it.

`Discouraging` describe algo que te hace sentir desanimado. Como, `Finding a bug in your code after hours of work can be discouraging.`

`Encouraging` es cuando algo te da coraje o esperanza. Por ejemplo, `Getting your first program to run is very encouraging.`
