# --description--

Adjektive beschreiben oder liefern mehr Informationen über Nomen oder Pronomen. Sie stehen normalerweise vor dem Nomen, das sie beschreiben. Zum Beispiel:

- `Her older sister is a programmer.` — Das Adjektiv `older` beschreibt das Nomen `sister`. Beachten Sie, wie das Adjektiv `older` vor dem Nomen steht.

- `He joined an international company.` — Das Adjektiv `international` gibt mehr Informationen über das Nomen `company`.

- `He has a quick shower in the morning.` - Das Adjektiv `quick` beschreibt, wie kurz die Dusche ist.

Adjektive können die Sprache auch ausdrucksstärker machen, indem sie Details über eine Person, einen Ort oder eine Sache liefern. Zum Beispiel könnten Sie statt `It seems like you have a lot of energy this morning` sagen `You look energetic this morning`. Hier ist das Adjektiv `energetic` prägnanter und ausdrucksstärker und zeigt, dass jemand auf positive Weise viel Energie hat.
