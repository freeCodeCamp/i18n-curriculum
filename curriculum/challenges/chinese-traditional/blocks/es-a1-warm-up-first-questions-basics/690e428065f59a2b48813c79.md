# --explanation--

字 `Guatemalteco` 是來自瓜地馬拉的男性國籍形式。 

國籍必須與所描述的人稱性別一致。

| Gender | Nationality | Example |
| :--- | :--- | :--- |
| **Feminine** | `Guatemalteca` | `Ella es guatemalteca.` ("She is Guatemalan.") |
| **Masculine** | `Guatemalteco` | `Él es guatemalteco.` ("He is Guatemalan.") |
