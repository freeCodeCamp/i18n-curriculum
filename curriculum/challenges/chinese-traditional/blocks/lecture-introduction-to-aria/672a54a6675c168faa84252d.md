# --interactive--

確保所有使用者，包括身心障礙者，都能順利存取網站非常重要。

對於使用螢幕閱讀器的人來說，`aria-label` 和 `aria-labelledby` 屬性提供了關於頁面元素的重要資訊，這些元素可能不清楚或不可見。

讓我們來看看 `aria-label` 和 `aria-labelledby` 屬性是什麼，以及它們在讓網頁對視覺障礙和相關障礙者可存取方面所扮演的角色。

你會注意到 `aria-label` 和 `aria-labelledby` 都有 `aria-` 前置式。那代表什麼意思呢？ARIA 代表 Accessible Rich Internet Applications（無障礙豐富網路應用）。它是一組以 `aria-` 為前置式的屬性，讓開發者能向輔助技術傳達元素的用途。`aria-label` 屬性是互動元素的隱形標籤，它為元素添加一個螢幕閱讀器可以讀取的文字標籤。

`aria-label` 對於沒有可見文字但仍需由螢幕閱讀器描述的元素特別有用。 例如，只有圖示的按鈕通常需要 `aria-label` 來傳達其用途。

以下是一個範例：

```html
<button aria-label="Search">
  <i class="fa-solid fa-magnifying-glass"></i>
</button>
```

在這種情況下，螢幕閱讀器可能會將此按鈕宣告為 `Search, button`，即使按鈕只包含圖示。`aria-label` 屬性告訴螢幕閱讀器要使用什麼文字來取代圖示。

如果按鈕包含「搜尋」的文字而非圖示，則不需要 `aria-label` 屬性，因為文字會為按鈕提供標籤。

對於輸入元素，如果沒有與輸入相關的可見標籤，`aria-label` 屬性會直接提供標籤。

`aria-labelledby` 屬性的作用與 `aria-label` 屬性完全相同，但它不是直接在屬性中定義文字，而是引用頁面上已經存在的文字。這些已存在的文字必須具有 `id` 屬性，該屬性將用作 `aria-labelledby` 屬性中的引用值。

這裡有一個範例：

:::interactive_editor

```html
<input type="text" aria-labelledby="search-btn">
<button type="button" id="search-btn">Search</button>
```

:::

在這個例子中，按鈕的文字被用作搜尋輸入框的標籤。螢幕閱讀器會將輸入框讀作類似 `Search, edit`。如果你之後決定將按鈕文字改成 `Find`，輸入框的標籤會自動更新為新文字，因為它是參考按鈕文字。

也可以將多個 `id` 值合併成單一的 `aria-labelledby` 屬性值。以下是它的運作方式：

:::interactive_editor

```html
<div>
  <span id="volume-label">Volume</span>
  <span id="volume-details">Adjust the volume level</span>
  <input
    type="range"
    min="0"
    max="100"
    value="30"
    aria-labelledby="volume-label volume-details">
</div>
```

:::

對於滾軸，螢幕閱讀器會注意 `volume-label` 和 `volume-details` 元素的內容，並宣告 `Volume Adjust the volume level`。

你已經看到 `aria-label` 和 `aria-labelledby` 這兩個屬性都能幫助螢幕閱讀器理解元素的用途。那麼，你應該使用哪一個呢？由於它們提供相同的功能，兩者都可以使用，但使用 `aria-labelledby` 相較於 `aria-label` 可能有幾個優勢：

- 如果有人使用翻譯服務來翻譯你頁面上的內容，`aria-label` 屬性中的文字可能不會總是被翻譯。
- 使用 `aria-labelledby` 也可以幫助防止可見標籤文字與螢幕閱讀器使用者看不到的標籤之間的不匹配，因為更新可見文字會自動更新看不到的標籤。
- `aria-labelledby` 可以讓你更輕鬆地以程式方式創建由多個文字來源組成的複雜隱形標籤。

最後一點，請勿同時在一個元素上使用 `aria-label` 和 `aria-labelledby`。在這種情況下，螢幕閱讀器的隱藏標籤將永遠由 `aria-labelledby` 決定，而 `aria-label` 將被完全忽略。
