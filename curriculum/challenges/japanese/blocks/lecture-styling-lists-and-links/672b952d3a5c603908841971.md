# --interactive--

デフォルトのリンクスタイルは、ウェブの使いやすさとアクセシビリティを向上させる上で非常に重要な役割を果たします。

これらのスタイルは、通常、未訪問のリンクが青色、訪問済みのリンクが紫色で表示され、ユーザーがウェブサイトをナビゲートするときに期待し、頼りにする標準となっています。

デフォルトのリンクスタイルの主な目的は、ユーザーがウェブページ上のインタラクティブな要素と非インタラクティブな要素を区別しやすくするための明確な視覚的手がかりを提供することです。

この区別は、直感的で使いやすいブラウジング体験を作るために不可欠です。

基本的なリンクのデフォルトスタイルを見てみましょう：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
a:link {
  color: blue;
  text-decoration: underline;
}

a:visited {
  color: purple;
}
```

:::

これらのスタイルはいくつかの重要な機能を果たしています。

まず、未訪問のリンクの青色はほとんどの背景色やテキストに対して目立ち、リンクを簡単に識別できます。このコントラストは、ユーザーがページを素早くスキャンしてナビゲーション要素や重要な情報を見つけるために非常に重要です。

下線は、テキストがクリック可能であることをさらに強調し、追加の視覚的手がかりを提供します。これは、色覚障害のあるユーザーや色の区別が難しいユーザーに特に役立ちます。

訪問済みのリンクの色（通常は紫色）に変わることで、ユーザーがどこを訪れたかを把握しやすくなります。この機能は、大規模なウェブサイトをナビゲートしたり調査を行ったりするときに非常に役立ち、同じページを誤って再訪問するのを防ぎます。

このHTMLの例を見てみましょう：

:::interactive_editor

```html
<p>Learn more about <a href="https://www.example.com/cats">cats</a> and <a href="https://www.example.com/dogs">dogs</a>.</p>
```

:::

カスタムCSSがない場合、ほとんどのブラウザはこれらのリンクを青色の下線付きで表示します。リンクの1つをクリックすると、その色が紫色に変わり、ユーザーに閲覧履歴のフィードバックを即座に提供します。

デザイナーがこれらのデフォルトスタイルをウェブサイトの美観に合わせて変更することはよくありますが、その背後にある基本的な原則を守ることが重要です。

デフォルトスタイルを変更する場合でも、リンクが通常のテキストと明確に区別できること、訪問済みリンクと未訪問リンクの間に目に見える違いがあること、選んだ色が背景と十分なコントラストを持つことを確認してください。

例えば、次のようなカスタムスタイルを使うことができます：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
a:link {
  color: blue;
  text-decoration: none;
  border-bottom: 1px solid blue;
}

a:visited {
  color: purple;
  border-bottom: 1px solid purple;
}
```

:::

これは青と紫のカラースキームを維持しつつ、下線の代わりにボトムボーダーを使ってよりモダンな見た目にしています。

また、リンクの異なる状態も考慮することが重要です。デフォルトと訪問済みの状態に加えて、リンクには通常ホバー状態とアクティブ状態があります：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
a:hover {
  color: red;
}

a:active {
  color: darkorange;
}
```

:::

これらの状態は、ユーザーがリンクとやり取りするときに即時のフィードバックを提供し、サイト全体の使いやすさを高めます。

まとめると、リンクスタイルをカスタマイズすることは可能ですが、デフォルトスタイルの背後にある原則は守るべきです。

これらは使いやすくアクセシブルなウェブ体験を作る上で重要な役割を果たし、ユーザーがコンテンツを効果的にナビゲートし操作できるようにします。

ウェブサイトのリンクスタイルをデザインするときは、常に明確さとユーザー体験を優先してください。
