# --interactive--

Die CSS-`filter`-Eigenschaft ist ein mächtiges Werkzeug, mit dem Sie grafische Effekte auf Elemente einer Webseite anwenden können. Sie ist besonders nützlich, um die visuelle Darstellung von Bildern, Hintergründen und sogar Text anzupassen, ohne das ursprüngliche Asset zu verändern.

Die `filter`-Eigenschaft kann verwendet werden, um verschiedene Effekte zu erzeugen, wie Unschärfe, Farbverschiebung und Kontrastanpassungen. Lassen Sie uns besprechen, wie die Filter-Eigenschaft funktioniert, und einige gängige Beispiele ansehen. Die grundlegende Syntax für eine `filter`-Eigenschaft ist einfach:

```css
selector {
  filter: function(amount);
}
```

Hierbei steht `function` für den spezifischen Filtereffekt, den Sie anwenden möchten, und `amount` ist typischerweise ein Wert, der die Intensität des Effekts bestimmt. Schauen wir uns nun einige gängige Filterfunktionen und ihre Anwendungen an.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: blur(2px);
}
```

:::

Die `blur`-Funktion wendet eine Gaußsche Unschärfe auf das Element an, die Menge wird in Pixeln angegeben und stellt den Radius der Unschärfe dar. Diese CSS-Regel wendet eine 2-Pixel-Unschärfe auf alle Bilder der Seite an. Der `blur`-Effekt kann nützlich sein, um Tiefe in Ihrem Design zu erzeugen oder Teile eines Bildes zu verschleiern.

Die `brightness`-Funktion passt die Helligkeit des Elements an. Ein Wert von `0%` macht das Element komplett schwarz, während Werte über `100%` die Helligkeit erhöhen.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: brightness(150%);
}
```

:::

Diese CSS-Regel erhöht die Helligkeit des Bildelements um `50%`. Helligkeitsanpassungen können verwendet werden, um Bilder hervorzuheben oder einen ausgewaschenen Effekt zu erzeugen.

Die `grayscale`-Funktion wandelt das Element in Graustufen um. Die Menge wird als Prozentsatz definiert, wobei `100%` komplett graustufig ist und `0%` das Bild unverändert lässt.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: grayscale(100%);
}
```

:::

Diese Regel wandelt das Bildelement vollständig in Graustufen um. `grayscale` kann verwendet werden, um einen Vintage-Look zu erzeugen oder bestimmte Elemente auf einer Seite weniger hervorzuheben.

Die `sepia`-Funktion wendet einen Sepia-Ton auf das Element an. Wie bei Graustufen wird ein Prozentwert verwendet:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: sepia(80%);
}
```

:::

Diese Regel wendet einen `80%`-Sepiaeffekt auf das Bildelement an. Der Sepiaeffekt eignet sich hervorragend, um einen Vintage- oder altmodischen Look zu erzeugen.

Die `hue-rotate`-Funktion dreht den Farbton des Elements. Der Wert wird in Grad angegeben und stellt eine Drehung um den Farbkreis dar.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: hue-rotate(90deg);
}
```

:::

Diese Regel dreht den Farbton des Bildelements um `90` Grad. Die Farbtonrotation kann verwendet werden, um psychedelische Effekte zu erzeugen oder das gesamte Farbschema eines Bildes anzupassen.

Eine der mächtigsten Eigenschaften der `filter`-Eigenschaft ist die Möglichkeit, mehrere Effekte zu kombinieren. Sie können mehrere Filter auf dasselbe Element anwenden, indem Sie sie mit Leerzeichen trennen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {  
  filter: contrast(150%) brightness(110%) sepia(30%);  
}
```

:::

Diese Regel wendet erhöhten Kontrast, leicht erhöhte Helligkeit und einen subtilen Sepiaeffekt auf das Bildelement an.

Durch die Kombination von Filtern können Sie komplexe und einzigartige visuelle Effekte erstellen, die auf Ihre Designanforderungen zugeschnitten sind. Die CSS-Filter-Eigenschaft ist ein vielseitiges Werkzeug, das kreative visuelle Manipulation von Webelementen ermöglicht.

Obwohl wir einige der gängigsten Filterfunktionen behandelt haben, gibt es weitere, wie `contrast`, `invert` und `saturate`. Wie bei jeder mächtigen Funktion ist es wichtig, vorsichtig mit der Verwendung der Filter umzugehen, um Ihr Design zu verbessern, ohne Ihre Nutzer zu überfordern oder die Barrierefreiheit zu beeinträchtigen.
