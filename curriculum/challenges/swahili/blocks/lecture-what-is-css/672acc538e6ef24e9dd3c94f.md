# --interactive--

Margin na nafasi ya ndani ni sifa muhimu katika CSS kwa kuunda kurasa za mtandao zilizo na muundo mzuri, zinazosomeka, na zenye mvuto wa kuona.

Margins hudhibiti nafasi nje ya kipengele, kusaidia kukitenganisha na vipengele vingine na kufafanua muundo wa mpangilio, wakati nafasi ya ndani hudhibiti nafasi ndani ya kipengele, kuboresha usomaji wa maudhui na mvuto wa kuona.

Ili kuelewa tofauti kati ya `margin` na `padding` vizuri zaidi, tuchunguze mifano kadhaa.

Hapa kuna mfano wa HTML wa vipengele vitatu vya aya kwenye ukurasa:

:::interactive_editor

```html
<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

:::

Ili kuweka nafasi juu ya kila kipengele cha aya, unaweza kutumia sifa ya `margin-top` kama ifuatavyo:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  margin-top: 20px;
  border: 2px solid black;
}
```

:::

Katika mfano huu, tunatumia `20px` ya margin tu juu ya kila kipengele cha aya.

Pia tuna mpaka mweusi imara kwenye pande zote nne ili uweze kuona margin vizuri zaidi.

Sifa nne tofauti za `margin` ni `margin-top`, `margin-right`, `margin-bottom` na `margin-left`.

Hapa kuna mfano wa kutumia sifa zote nne:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 30px;
  margin-left: 40px;
  border: 2px solid black;
}
```

:::

Katika mfano huu, tunaweka thamani za nafasi kwenye pande zote nne za kipengele cha aya. Pia kumeongezwa mpaka mweusi imara ili uweze kuona margins vizuri zaidi.

Njia nyingine ya kutumia sifa ya `margin` ni kwa kutumia ufupisho wa sifa. Unaweza kubainisha thamani moja, mbili, tatu, au nne mara moja. Tuchunguze chaguzi hizi pamoja.

Unapotumia thamani moja kwenye ufupisho wa `margin`, thamani hiyo hiyo itatumika kwa pande zote nne za kipengele lengwa.

Hapa kuna mfano wa kutumia thamani moja kwenye ufupisho wa `margin`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px;
}
```

:::

Mfano huu wa msimbo utaweka `10px` ya `margin` kwa pande zote nne za kipengele cha aya kwa usawa.

Unapotumia thamani mbili, thamani ya kwanza hutumika kwa `top` na `bottom`, wakati thamani ya pili hutumika kwa `left` na `right` za kipengele.

Hapa kuna mfano wa kutumia thamani mbili kwa ufupisho wa `margin`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px;
}
```

:::

Hii itaweka margins za `top` na `bottom` kuwa `10px`, na `20px` kwa margins za `left` na `right` za kipengele cha aya.

Ikiwa thamani tatu zitatolewa, thamani ya kwanza itatumika kwa margin ya `top`, thamani ya pili kwa margin za `left` na `right`, na thamani ya tatu kwa margin ya `bottom`.

Hapa kuna mfano wa kuelewa vizuri:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px;
}
```

:::

Hii itaweka margin kuwa `10px` kwa `top`, `20px` kwa `left` na `right`, na `30px` kwa `bottom`.

Unapotumia thamani nne, hii inakupa udhibiti zaidi, kwani unaweza kubainisha kila thamani ya margin kwa kila upande wa kipengele lengwa kwa uhuru.

Thamani ya kwanza inalenga `top`, thamani ya pili inalenga `right`, thamani ya tatu inalenga `bottom`, na thamani ya nne inalenga `left`.

Hapa kuna mfano wa kutumia ufupisho wa margin na thamani nne:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px 40px;
}
```

:::

Hii itaweka margin kuwa `10px` kwa `top`, `20px` kwa `right`, `30px` kwa `bottom`, na `40px` kwa `left`.

Sifa ya `padding` hutumika kuweka nafasi ndani ya kipengele, kati ya maudhui na mpaka wake.

Kama sifa ya `margin`, sifa nne za `padding` ni `padding-top`, `padding-right`, `padding-bottom` na `padding-left`.

Hapa kuna mfano wa jinsi ya kuweka `padding` kwa kipengele cha aya:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 30px;
  padding-left: 40px;
  border: 2px solid black;
}
```

:::

Hii itaweka nafasi ya ndani kuwa `10px` kwa `top`, `20px` kwa `right`, `30px` kwa `bottom`, na `40px` kwa `left`.

Kama unavyoona, `padding` inatumika kwa maudhui ambayo yako ndani ya mpaka, tofauti na `margin` ambayo inatumika nje ya mpaka.

Kama sifa ya `margin`, pia unaweza kuchagua kutumia ufupisho wa sifa ya `padding`.

Unaweza pia kubainisha thamani moja, mbili, tatu, au nne kwenye ufupisho wa sifa ya `padding`.

Hapa kuna mfano wa kutumia ufupisho wa `padding` kwa kipengele cha aya kutoka awali:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding: 10px 20px 30px 40px;
  border: 2px solid black;
}
```

:::

Katika mfano huu, kwa kutumia ufupisho msimbo utaweka nafasi ya ndani kuwa `10px` kwa `top`, `20px` kwa `right`, `30px` kwa `bottom`, na `40px` kwa `left` za kipengele cha aya.
