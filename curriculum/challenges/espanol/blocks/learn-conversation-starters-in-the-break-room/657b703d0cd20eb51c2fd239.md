# --description--

Practiquemos la creación de oraciones negativas con un verbo auxiliar.

# --questions--

## --text--

Elige la oración que usa correctamente `don't` para expresar lo que alguien no hace.

## --answers--

`I don't coding on weekends.`

### --feedback--

Cuando se usa con `don't`, `coding` debe estar en la forma base `code`.

---

`I don't code on weekends.`

---

`I don't codes on weekends.`

### --feedback--

Cuando se usa con `don't`, el verbo no debe tener una `-s` al final.

---

`I does not code on weekends.`

### --feedback--

`Does not` se usa con sujetos en tercera persona del singular (`he`, `she`, `it`); con `I`, debe ser `do not`.
