# --interactive--

Umejifunza awali jinsi alama za uandishi wa kihisia na sifa za ARIA zinavyofanya maudhui yafikike. Katika programu halisi, maudhui mara nyingi hayako kimya, na kurasa huendelea kusasishwa kwa njia ya mabadiliko kupitia JavaScript. Hii inapofanyika, ni muhimu kuhakikisha mabadiliko haya yanaonyeshwa pia katika HTML. Hii inaruhusu visomaji vya skrini kutoa taarifa sahihi kuhusu hali iliyosasishwa kwa watumizi. Bila hili, watu wanaotegemea zana za kusaidia wanaweza wasijue kuwa kitu kimebadilika, au wanaweza kupokea taarifa za zamani au zisizo sahihi.

Tuchunguze hali ya kawaida ambapo JavaScript hubadilisha hali ya kipengele, na jinsi unavyoweza kuweka HTML yako ifikike kwa kusasisha sifa za ARIA.

Fikiria una kitufe kinachofungua na kufunga menyu ya kushuka:

:::interactive_editor

```html
<button id="menuButton" aria-expanded="false" aria-controls="menuList">
  Menu
</button>

<ul id="menuList" hidden>
  <li>List item 1</li>
  <li>List item 2</li>
</ul>
<script src="index.js"></script>
```

```js
const button = document.getElementById("menuButton");
const menu = document.getElementById("menuList");

button.addEventListener("click", () => {
  const expanded = button.getAttribute("aria-expanded") === "true";
  button.setAttribute("aria-expanded", String(!expanded));
  menu.hidden = expanded;
});
```

:::

HTML inaandaa kitufe kinachodhibiti menyu ya kushuka. Kitufe kinatumia `aria-expanded` kuonyesha kama menyu iko wazi, na `aria-controls` kuunganisha na menyu kwa kurejelea `id` ya menyu.

JavaScript husikiliza bofya kwenye kitufe cha menyu. Inapobofya, hubadilisha thamani ya `aria-expanded` na kusasisha sifa ya `hidden` kwenye menyu. Hii huweka uonekano wa menyu na taarifa za ufikikaji ziko sambamba, hivyo watumizi wanaoona na watumizi wa kisomaji cha skrini wanajua kama menyu iko wazi au imefungwa.

Utajifunza zaidi kuhusu sifa hizo za ARIA katika mafundisho yajayo. Kwa sasa, kumbuka tu kwamba unapobadilisha hali ya kipengele kwa JavaScript, unapaswa pia kusasisha sifa husika za ARIA ili teknolojia za kusaidia ziweze kuonyesha kwa usahihi kile kilicho kwenye skrini.
