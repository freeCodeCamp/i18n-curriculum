다음과 기능적으로 유사한 풀스택 JavaScript 앱을 만드세요: <a href="https://yoyo44.herokuapp.com/" target="_blank" rel="noopener noreferrer nofollow">https://yoyo44.herokuapp.com/</a>. 원하는 사이트 빌더를 사용해 프로젝트를 완성하세요.

이 프로젝트에서 구현해야 할 구체적인 사용자 스토리는 다음과 같습니다:

**사용자 스토리:** 인증되지 않은 사용자로서, 내 영역에 있는 모든 바를 볼 수 있습니다.

**사용자 스토리:** 인증된 사용자로서, 오늘 밤 가려는 바에 자신을 추가할 수 있습니다.

**사용자 스토리:** 인증된 사용자로서, 더 이상 가고 싶지 않은 바에서 자신을 제거할 수 있습니다.

**사용자 스토리:** 인증되지 않은 사용자로서, 로그인할 때 다시 검색할 필요가 없어야 합니다.

**힌트:** Yelp API를 사용해 사용자가 검색하는 도시의 장소를 찾아보세요. Yelp API를 사용한다면 앱에 반드시 명시하세요.

완료하면, 프로젝트의 작동하는 데모를 공개된 곳에 호스트했는지 확인하세요. 그런 다음 솔루션 링크 필드에 URL을 제출하세요. 선택적으로 GitHub 링크 필드에 프로젝트 소스 코드 링크도 제출할 수 있습니다.
