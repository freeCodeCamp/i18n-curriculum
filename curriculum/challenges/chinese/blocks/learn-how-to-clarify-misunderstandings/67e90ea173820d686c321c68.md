# --explanation--

一个例子是 `the "Student" class inherits properties and methods from the "Person" class`。

- `Student` 和 `Person` 都是 `类`。一个 `类` 定义了用于创建具有特定数据和行为的对象的蓝图。

- `Student` 继承了 `Person` 的特色，这意味着它不需要重新定义共享的特征——它可以重用它们。

- `Properties` 是类内部的变量或数据，比如描述对象的 `name` 或 `age`。

- `Methods` 是在类中定义的函数，比如 `enroll()` 或 `introduce()`，用来描述对象能做什么。
