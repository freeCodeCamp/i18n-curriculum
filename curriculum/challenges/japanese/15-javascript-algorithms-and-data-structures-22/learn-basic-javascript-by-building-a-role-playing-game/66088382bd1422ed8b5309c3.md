# --description--

オブジェクトは配列に似ていますが、データにアクセスしたり変更したりする際、インデックスの代わりに、プロパティを通じてオブジェクトのデータにアクセスします。

プロパティは「キー」と「値」で構成されます。 キーはプロパティの名前であり、値はプロパティに格納されているデータです。

こちらはプロパティを 1 つ持っているオブジェクトの例です:

```js
const obj = {
  name: "Quincy Larson"
};
```

では、`cat` オブジェクト内に新しいプロパティを追加してください。 キーは `name` で、値は文字列 `"Whiskers"` にしてください。

コンソールを開いて、オブジェクトが変更されたことを確認してください。

# --hints--

`cat` オブジェクトには `name` というキーを持たせましょう。

```js
assert.property(cat, "name");
```

`cat` オブジェクトには、値が `"Whiskers"` の `name` プロパティを持たせましょう。

```js
assert.propertyVal(cat, "name", "Whiskers");
```

`cat` オブジェクトには 1 つのプロパティが必要です。

```js
assert.lengthOf(Object.keys(cat), 1);
```
