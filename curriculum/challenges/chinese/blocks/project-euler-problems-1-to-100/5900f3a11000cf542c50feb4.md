# --description--

有十种方法从五种中选择三种，12345：123,124,125,134,135,145,234,235,245和345在组合学中，我们使用符号，5C3 = 10.一般来说，

<div style='text-align: center;'>123, 124, 125, 134, 135, 145, 234, 235, 245, and 345</div>

在组合数学中，我们使用符号，$\\displaystyle \\binom 5 3 = 10$

通用的，$\\displaystyle \\binom n r = \\dfrac{n!}{r!(n-r)!}$，其中 $r \\le n$，$n! = n \\times (n-1) \\times ... \\times 3 \\times 2 \\times 1$，且 $0! = 1$。

直到 $n = 23$，值才超过一百万：$\\displaystyle \\binom {23} {10} = 1144066$。

在 1 ≤ n ≤ 100 范围内，有多少个（不一定不同的）值的 $\\displaystyle \\binom n r$ 大于一百万？

```js
function combinatoricSelections(limit) {
    const factorial = n =>
        Array.apply(null, { length: n })
            .map((_, i) => i + 1)
            .reduce((p, c) => p * c, 1);

    let result = 0;
    const nMax = 100;

    for (let n = 1; n <= nMax; n++) {
        for (let r = 0; r <= n; r++) {
            if (factorial(n) / (factorial(r) * factorial(n - r)) >= limit)
                result++;
        }
    }

    return result;
}
```
