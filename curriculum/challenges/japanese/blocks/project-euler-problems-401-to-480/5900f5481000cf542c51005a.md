# --description--

$12n$ 人のミュージシャンが音楽フェスティバルに参加します。初日には、$3n$ 組のカルテットを結成し、一日中練習します。

これは大惨事です。一日の終わりに、すべてのミュージシャンは、自分のカルテットのメンバーとは二度と一緒に演奏しないと決めます。

2日目には、$4n$ 組のトリオを結成しますが、各ミュージシャンは前日に一緒だったカルテットのメンバーを避けます。

$12n$ 人のミュージシャンの間でトリオを組む方法の数を $f(12n)$ とします。

$f(12) = 576$ および $f(24)\bmod 1\,000\,000\,007 = 509\,089\,824$ が与えられています。

$f(600)\bmod 1\,000\,000\,007$ を求めてください。
