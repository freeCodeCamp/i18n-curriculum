# --description--

6174 是一個特殊的數字；如果我們將它的數字按遞增順序排序，並從將數字按遞減順序排序得到的數字中減去該數字，我們會得到 $7641 - 1467 = 6174$。

更令人驚訝的是，如果我們從任何 4 位數字開始並重複這個排序和相減的過程，最終會得到 6174，或者如果所有數字都相等，則會立即得到 0。

如果我們用前置零填充數字直到有 4 位數，這對少於 4 位數的數字也適用。

例如，讓我們從數字 0837 開始：

$$\begin{align}
  & 8730 - 0378 = 8352 \\\\
  & 8532 - 2358 = 6174
\end{align}$$

6174 被稱為 Kaprekar 常數。將數字排序、相減並重複此過程直到達到 0 或 Kaprekar 常數的處理程序稱為 Kaprekar 常式。

我們可以考慮其他進位制和位數的 Kaprekar 常式。不幸的是，並不保證在所有情況下都存在 Kaprekar 常數；常式可能會對某些輸入數字進入循環，或者常式所達到的常數對不同的輸入數字可能不同。然而，可以證明對於 5 位數且進位制 $b = 6t + 3 ≠ 9$，存在 Kaprekar 常數。

例如  
base 15：${(10, 4, 14, 9, 5)}\_{15}$  
base 21：$(14, 6, 20, 13, 7)\_{21}$

定義 $C_b$ 為 5 位數制底數 $b$ 的 Kaprekar 常數。定義函式（程式） $sb(i)$ 為：

- 當 $i = C_b$ 或 $i$ 以底數 $b$ 表示時由 5 個相同的數字組成，則為 0
- Kaprekar 常式在底數為 $b$ 時到達 $C_b$ 所需的迭代次數，否則

請注意，我們可以為所有整數 $i &lt; b^5$ 定義 $sb(i)$。如果以底數 $b$ 表示的 $i$ 少於 5 位數，則該數字會以前置零補足至 5 位數，然後再套用 Kaprekar 常式。

定義 $S(b)$ 為 $sb(i)$ 在 $0 &lt; i &lt; b^5$ 範圍內的總和。例如 $S(15) = 5\\,274\\,369$ $S(111) = 400\\,668\\,930\\,299$

求和 $S(6k + 3)$，其中 $2 ≤ k ≤ 300$。將最後 18 位數字作為你的答案。
