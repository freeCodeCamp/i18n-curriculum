# --description--

```css
/* rule 1 */
.class.second-class {
  font-size: 12px;
}

/* rule 2 */
.class .second-class {
  font-size: 24px;
}
```

هنا كلا القاعدتين 1 و 2 لهما نفس القيمة من حيث التحديد. القاعدة 1 تستخدم محدِّد تسلسلي (بدون فراغ) والقاعدة 2 تستخدم جامع عناصر تابع (الفراغ الفارغ). لكن كلا القاعدتين تحتويان على صنفين ورمز الجامع نفسه لا يضيف إلى التحديد.

```css
/* rule 1 */
.class.second-class {
  font-size: 12px;
}
/* rule 2 */
.class > .second-class {
  font-size: 24px;
}
```

هذا المثال يوضح نفس الشيء. رغم أن القاعدة 2 تستخدم جامع عنصر طفل (`>`)، إلا أن هذا لا يغير قيمة التحديد. كلا القاعدتين لا تزالان تحتويان على صنفين لذا لهما نفس قيم التحديد.

*ملاحظة:* ليس كل شيء يضيف إلى التحديد

عند مقارنة المحدِّدات، قد تصادف رموزًا خاصة للمحدِّد العام (`*`) وكذلك الجامعين (`+`، `~`، `>`، والفراغ الفارغ). هذه الرموز لا تضيف أي قيمة للتحديد بحد ذاتها.
