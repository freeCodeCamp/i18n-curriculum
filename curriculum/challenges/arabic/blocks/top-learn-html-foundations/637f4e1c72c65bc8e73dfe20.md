# --description--

لعرض قالب جاهز لـ HTML، تحتاج أولًا إلى ملف HTML للعمل عليه.

أنشئ مجلدًا جديدًا على جهازك وسمّه `html-boilerplate`. داخل ذلك المجلد، أنشئ ملفًا جديدًا وسمّه `index.html`.

من المحتمل أنك تعرف بالفعل العديد من أنواع الملفات المختلفة، مثل ملفات doc وpdf وملفات الصور.

لتخبر الكمبيوتر بأنك تريد إنشاء ملف HTML، يجب أن تضيف لاحقة `.html` لاسم الملف كما فعلت عند إنشاء ملف `index.html`.

من الجدير بالذكر أنك سمّيت ملف HTML الخاص بك index. يجب دائمًا تسمية ملف HTML الذي يحتوي على الصفحة الرئيسية لمواقعك بـ `index.html`. ذلك لأن خوادم الويب تبحث افتراضيًا عن صفحة `index.html` عندما يزور المستخدمون مواقعك - وعدم وجودها يسبب مشاكل كبيرة.

## إعلان DOCTYPE

تبدأ كل صفحة HTML بإعلان doctype. الغرض من doctype هو إخبار المتصفح بإصدار HTML الذي يجب استخدامه لعرض المستند. أحدث إصدار من HTML هو HTML5، وإعلان doctype لهذا الإصدار هو ببساطة `<!DOCTYPE html>`.

كانت إعلانات doctype لإصدارات HTML الأقدم أكثر تعقيدًا قليلًا. على سبيل المثال، هذا هو إعلان doctype لإصدار HTML4:

```html
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
```

مع ذلك، من المحتمل ألا ترغب أبدًا في استخدام إصدار أقدم من HTML، لذلك ستستخدم دائمًا `<!DOCTYPE html>`.

افتح ملف `index.html` الذي أنشأته سابقًا في محرر النصوص وأضف `<!DOCTYPE html>` في السطر الأول تمامًا.
