# --explanation--

**分数**は全体の一部を表すために使います。

`One-fourth`（¼）は4つの等しい部分のうちの1つです。ほかの例は以下の通りです：

- ½ = `one-half`

- ⅓ = `one-third`

- ¾ = `three-fourths`
