# --description--

你已經學過像 `this`、`that`、`these` 和 `those` 這類指示代名詞。請記得使用它們來指涉特定的事物或人物，並考慮它們在時間或空間上的接近程度。

所有格，通常以撇號和 `s`（例如 `function's`）顯示，用來表示擁有或所屬，通常用於人或物件。它告訴人們某物屬於或是另一物的零件。

字 `purpose` 指的是某事被執行或使用的原因，或某事存在的理由。例如，`The purpose of a clock is to tell time.`
