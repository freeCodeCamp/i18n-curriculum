# --description--

문자열을 다룰 때, 여러 텍스트 조각을 합치는 작업은 자주 하게 되는 일반적인 작업입니다.

Python에서는 더하기(`+`) 연산자를 사용해 여러 문자열을 합칠 수 있습니다. 이 과정을 **문자열 연결**이라고 합니다. 더하기 연산자로 두 문자열을 연결하는 방법은 다음과 같습니다:

```python
my_str_1 = 'Hello'
my_str_2 = "World"

str_plus_str = my_str_1 + ' ' + my_str_2
print(str_plus_str) # Hello World
```

하지만 이 방법은 문자열에만 적용됩니다. 문자열과 숫자를 연결하려 하면 `TypeError` 오류가 발생합니다:

```python
name = 'John Doe'
age = 26

name_and_age = name + age
print(name_and_age) # TypeError: can only concatenate str (not "int") to str
```

이는 Python이 문자열을 연결할 때 정수 같은 다른 데이터 타입을 자동으로 문자열로 변환하지 않기 때문입니다. Python은 연결하기 전에 모든 요소가 문자열이어야 합니다. 이를 해결하려면 내장 `str()` 함수를 사용해 숫자를 문자열로 변환할 수 있습니다. 이 함수는 원본 객체를 변경하지 않고 해당 객체의 문자열 표현을 반환합니다:

```python
name = 'John Doe'
age = 26

name_and_age = name + str(age)
print(name_and_age) # John Doe26
```

또한 연결을 위해 증강 대입 연산자를 사용할 수 있습니다. 이는 더하기와 등호 기호(`+=`)로 표현되며, 연결과 할당을 한 번에 수행합니다. 사용 예는 다음과 같습니다:

```py
name = 'John Doe'
age = 26

name_and_age = name  # Start with the name
name_and_age += str(age)  # Append the age as string

print(name_and_age)  # John Doe26
```

변수와 표현식을 문자열 안에 삽입하는 과정을 **문자열 보간**이라고 합니다. Python에는 **f-strings**(형식화된 문자열 리터럴의 약자)라는 문자열 종류가 있어, 간결하고 읽기 쉬운 구문으로 보간을 처리할 수 있습니다.

f-strings는 따옴표 앞에 `f` (소문자 또는 대문자)를 붙여 시작하며, 중괄호(`{}`)로 표시된 대체 필드 안에 변수나 표현식을 임베드할 수 있습니다. 예시는 다음과 같습니다:

```python
name = 'John Doe'
age = 26
name_and_age = f'My name is {name} and I am {age} years old'
print(name_and_age) # My name is John Doe and I am 26 years old

num1 = 5
num2 = 10
print(f'The sum of {num1} and {num2} is {num1 + num2}') # The sum of 5 and 10 is 15
```

위 예시에서 `str()` 함수를 사용해 비문자열 타입을 변환할 필요가 없다는 점에 주목하세요. `age`, `num1`, `num2` 변수의 값은 보간 과정에서 내부적으로 문자열로 변환됩니다.
