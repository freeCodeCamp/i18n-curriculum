# --description--

Katika Kihispania, nambari 20 ni `veinte`. 

Unapotengeneza nambari kutoka 21 hadi 29, maneno hubadilika kidogo. `Veinte` hubadilika kuwa `veinti-` (`e` ya mwisho huondolewa), kisha huungana na nambari inayofuata. Kwa mfano:

- `Veinti` + `dos` (mbili) = `veintidós` (22)

- `Veinti` + `cinco` (tano) = `veinticinco` (25)

- `Veinti` + `ocho` (nane) = `veintiocho` (28)
