# --explanation--

`请问 (qǐng wèn)` significa "com licença". É frequentemente usado no início de uma pergunta para soar educado. Por exemplo:

`请问你是刘明吗 (qǐng wèn nǐ shì Liu Ming ma)？` – Com licença, você é Liu Ming?
