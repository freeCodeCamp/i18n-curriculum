# --description--

我们说一个 $d$ 位正数（无前导零）是一个一子数，如果它的子字串中恰好有一个能被 $d$ 整除。

例如，5671 是一个 4 位数的单子号码。 在它的所有子串 5、6、7、1、56、67、71、567、671 和 5671 中，只有 56 能被 4 整除。

同样，104 是一个 3 位数的单子数，因为只有 0 能被 3 整除。1132451 是一个 7 位数的单子数，因为只有 245 能被 7 整除。

设 $F(N)$ 为小于 $N$ 的单子数的数量。我们可以验证 $F(10) = 9$，$F({10}^3) = 389$ 和 $F({10}^7) = 277\\,674$。

查找 $F({10}^{19})$。
