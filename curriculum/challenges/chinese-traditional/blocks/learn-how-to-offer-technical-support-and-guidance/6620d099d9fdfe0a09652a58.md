# --description--

字 `potential` 指的是未來可能發生或存在的事物，儘管尚未確定。在程式設計中，談論 `potential issue` 意味著程式碼可能在某些條件下出現問題。這不是確定的問題，但可能會成為問題。

例如，`There is a potential issue if the user enters a very large number.` 這表示問題目前並未發生，但如果情況出現，問題可能會發生。
