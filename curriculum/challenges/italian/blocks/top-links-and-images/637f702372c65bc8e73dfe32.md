# --description--

I siti web sarebbero piuttosto noiosi se potessero mostrare solo testo. Per fortuna HTML offre una vasta gamma di elementi per mostrare ogni tipo di media diverso. Il più usato tra questi è l'elemento immagine.

Per mostrare un'immagine in HTML usi l'elemento `<img>`. A differenza degli altri elementi che hai incontrato, `<img>` è un elemento vuoto. Gli elementi HTML vuoti non hanno bisogno di un tag di chiusura.

Invece di racchiudere contenuto con un tag di apertura e uno di chiusura, incorpora un'immagine nella pagina usando un attributo `src` che indica al browser dove si trova il file immagine. L'attributo `src` funziona in modo simile all'attributo `href` per i tag àncora. Può incorporare un'immagine usando sia percorsi assoluti sia relativi.

Per esempio, usando un percorso assoluto puoi mostrare un'immagine situata sul sito The Odin Project:

Per usare immagini che hai sul tuo sito, puoi usare un percorso relativo.

- Crea una nuova directory chiamata `images` all'interno del progetto `odin-links-and-images`.

- Poi, scarica <a href="https://unsplash.com/photos/Mv9hjnEUHR4/download?force=true&w=640" target="_blank">questa immagine</a> e spostala nella directory `images` che hai appena creato.

- Rinomina l'immagine in `dog.jpg`.

Infine aggiungi l'immagine al file `index.html`:

```html
<body>
  <h1>Homepage</h1>
	<a href="https://www.theodinproject.com/about">click me</a>

	<a href="pages/about.html">About</a>

	<img src="images/dog.jpg">
</body>
```

Salva il file `index.html` e aprilo in un browser per vedere Charles in tutto il suo splendore.


## Directory genitrici

E se volessi usare l'immagine del cane nella pagina `about`? Prima dovresti salire di un livello fuori dalla directory pages nella sua directory genitrice per poter poi consultare la directory images.

Per andare nella directory genitrice devi usare due punti nel percorso relativo così: `../.` Vediamo questo in pratica, all'interno del `body` del file `about.html`, aggiungi l'immagine seguente sotto l'intestazione che hai aggiunto prima:

```html
<img src="../images/dog.jpg">
```

Per spiegare:

- Prima, stai andando nella directory genitrice della directory pages che è `odin-links-and-images`.

- Poi, dalla directory genitrice, puoi entrare nella directory `images`.

- Infine, puoi consultare il file `dog.jpg`.

Usando la metafora che abbiamo usato prima, usare `../` in un percorso è un po' come uscire dalla stanza in cui ti trovi per andare nel corridoio principale così da poter andare in un'altra stanza.

## Attributo `Alt`

Oltre all'attributo `src`, ogni elemento immagine dovrebbe avere anche un attributo `alt` (testo alternativo).

L'attributo `alt` serve a descrivere un'immagine. Verrà usato al posto dell'immagine se non può essere caricata. Viene anche usato con i lettori di schermo per descrivere cosa rappresenta l'immagine agli utenti con disabilità visive.
