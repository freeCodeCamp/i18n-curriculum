# --description--

Decir To `depend on` se usa para indicar que algo puede cambiar según diferentes cosas. Por ejemplo:

- `The project depends on good teamwork.` - El proyecto solo tendrá éxito si el equipo trabaja bien en conjunto.

- `Developers depend on tests to find errors.` - Los programadores necesitan pruebas para ver si el código tiene errores.

Nota que después de `depend` siempre está la preposición `on`.
