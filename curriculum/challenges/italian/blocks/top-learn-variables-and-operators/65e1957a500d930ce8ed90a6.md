# --description--

Per dichiarare una variabile costante (immutabile), usa `const` invece di `let`:

```javascript
const myBirthday = '16-09-2003';
```

Le variabili dichiarate usando `const` si chiamano "costanti". Non possono essere riassegnate. Un tentativo di farlo causerebbe un errore:

```javascript
const myBirthday = '16-09-2003';

myBirthday = '01-01-2001'; // error, can't reassign the constant!
```

Quando un programmatore è sicuro che una variabile non cambierà mai, può dichiararla con `const` per garantire e comunicare questo fatto a tutti.
