# --interactive--

CSSのpositionプロパティを使うと、ページ上の要素の配置を制御できます。よく使われる位置指定の種類には、静的位置指定と相対位置指定があります。デフォルトでは、要素は静的に配置されます。これは、要素が文書の通常の流れに従い、上から下、左から右へと順番に並ぶことを意味します。

静的位置指定はすべての要素のデフォルトであり、CSSで特別な宣言は必要ありません。静的位置指定を使っても見た目に変化はありません。なぜなら、要素は文書内で自然に存在する場所にそのまま配置されるからです。以下は、デフォルトの静的位置指定を使った段落の例です。

:::interactive_editor

```html
<p>This paragraph is statically positioned.</p>
```

:::

ブラウザでレンダリングすると、この段落は自然な位置に表示され、文書の通常の流れに従います。特別なスタイルは適用されず、レイアウト内で自然に現れる場所に留まります。

一方、相対位置指定は、要素の通常の位置からずらして配置できますが、文書の流れは乱しません。つまり、要素をデフォルトの静的な位置から新しい座標に動かすイメージです。相対位置指定の例は以下の通りです。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="relative">This paragraph is positioned relatively.</p>
```

```css
body {
  border: solid 1px black;
}

.relative {
  position: relative;
  top: 30px;
  left: 30px;
}
```

:::

この例では、段落は元の位置から`30px`下、`30px`右に移動して表示されます。ただし、通常の流れでその要素が占めていたスペースはそのまま残ります。相対位置指定は、レイアウト全体に影響を与えずに要素を少し動かしたい場合に非常に便利です。
