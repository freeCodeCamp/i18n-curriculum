# --description--

ユーザー名はインターネット上のあらゆる場所で使われています。ユーザーが好きなサイトで一意の識別子を持つためのものです。

データベース内のすべてのユーザー名をチェックする必要があります。ユーザーがユーザー名を作成するときに守らなければならない簡単なルールを以下に示します。

1) ユーザー名は英数字のみ使用できます。

2) ユーザー名に含まれる数字は末尾にのみなければなりません。末尾には0個以上の数字があってもよいです。ユーザー名は数字で始まってはいけません。

3) ユーザー名の文字は小文字および大文字の両方が使えます。

4) ユーザー名は少なくとも2文字以上でなければなりません。2文字のユーザー名はアルファベットの文字のみを使えます。

# --instructions--

上記の制約に合うように正規表現 `userCheck` を変更してください。
