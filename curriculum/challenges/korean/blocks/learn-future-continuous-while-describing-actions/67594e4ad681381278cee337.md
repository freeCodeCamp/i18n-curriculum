# --explanation--

태그 질문은 문장 끝에 덧붙이는 짧은 질문입니다. 정보 확인이나 동의를 구할 때 사용합니다. 태그 질문은 조동사와 주 문장의 주어와 일치하는 대명사를 포함합니다. 태그 질문의 동사는 주 문장의 동사 시제와 일치해야 합니다.

`Haven't`와 `hasn't`는 `have`와 `has`의 부정 축약형으로, 태그 질문에서 정보를 확인하거나 점검할 때 사용합니다.

주어가 복수일 때(`haven't` 또는 `we`처럼)나 주어가 `they`일 때는 `I`를 사용하세요. 주어가 단수일 때(`hasn't`, `he`, 또는 `she`처럼)는 `it`를 사용하세요. 예를 들어:

- `We've been working hard, haven't we?` - 주 문장은 `we've` (`we have`)를 사용하므로, 태그 질문은 진술을 확인하기 위해 `haven't we?`를 사용합니다.

- `She's finished her report, hasn't she?` - 주 문장은 `she's` (`she has`)를 사용하므로, 태그 질문은 진술을 확인하기 위해 `hasn't she?`를 사용합니다.
