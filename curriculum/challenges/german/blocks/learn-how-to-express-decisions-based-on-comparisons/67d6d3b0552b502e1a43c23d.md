# --explanation--

`Build the software` bedeutet, ein Programm oder eine Anwendung von Grund auf zu erstellen und zu entwickeln. Es umfasst das Programmieren, Entwerfen und Testen der Software. Zum Beispiel:

`The company wants to build the software from the ground up using Python.` – Das bedeutet, dass sie die Software vollständig von Anfang an mit Python entwickeln werden.

`In-house` bedeutet, dass die Arbeit innerhalb des Unternehmens erledigt wird, anstatt ein externes Team zu beauftragen oder das Projekt auszulagern. Zum Beispiel:

`Our team decided to do front-end development in-house instead of hiring an external agency.` – Das bedeutet, dass die eigenen Mitarbeiter des Unternehmens die Frontend-Entwicklung übernehmen, anstatt sie auszulagern.
