# --description--

`He's` kann eine Kurzform von `he has` oder `he is` sein. Sie können erkennen, welche es ist, indem Sie den Kontext betrachten:

- Wenn `he's` von einem Verb im Partizip Perfekt gefolgt wird (wie `been`, `done` oder `seen`), bedeutet es `he has`. Zum Beispiel bedeutet `he's been working` `he has been working`.

- Wenn `he's` von einem Adjektiv oder einem Partizip Präsens gefolgt wird (wie `tired`, `going` oder `sleeping`), bedeutet es `he is`. Zum Beispiel bedeutet `he's tired` `he is tired`.

Dieses Verständnis hilft Ihnen, genau zu wissen, was jemand meint, wenn er `he's` sagt.
