# --description--

`work on` significa dedicar tempo e esforço para realizar uma tarefa ou melhorar algo. Por exemplo:

`She is working on a new app.` - Isso significa que ela está desenvolvendo ou melhorando o app.

Parar de `take time off` significa interromper o trabalho por um curto período, geralmente para descansar ou por motivos pessoais. Por exemplo:

`I'm taking time off tomorrow because I have a doctor's appointment.` - Isso significa que não vou trabalhar durante o dia.
