# --interactive--

CSS Grid è un potente sistema di layout che permette agli sviluppatori web di creare layout complessi e responsivi per le pagine web con facilità.

Immagina di sistemare i mobili in una stanza: CSS Grid è come avere una griglia invisibile sul pavimento che ti aiuta a posizionare tutto esattamente dove vuoi.

Quando costruiamo siti web, spesso dobbiamo disporre diversi elementi sulla pagina.

Prima di CSS Grid, questo a volte era complicato, soprattutto per layout complessi. CSS Grid semplifica questo processo dividendo la tua pagina web in righe e colonne, creando una struttura a griglia.

Immaginiamo che tu stia lavorando con un contenitore `div` con diversi elementi annidati all’interno così:

:::interactive_editor

```html
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

:::

Se vuoi definire lo stile di quegli elementi in formato griglia, puoi impostare il `display` su `grid` e applicare le colonne così:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-gap: 20px;
}

.item {
  background-color: lightgray;
  padding: 20px;
  text-align: center;
  border: 1px solid #ccc;
}
```

:::

In questo codice, stiamo dicendo al browser di creare una griglia con tre colonne di larghezza uguale, questo è il significato di `1fr 1fr 1fr`, e stiamo aggiungendo uno spazio di 20 pixel tra ogni elemento griglia.

Ora, potresti chiederti: “E Flexbox? Non si usa anche per i layout?”

Hai ragione! Flexbox è un altro modello di layout CSS, ed è molto utile. Ma ci sono alcune differenze chiave.

Flexbox è unidimensionale, mentre Grid è bidimensionale. Questo significa che Flexbox funziona bene per disporre elementi in una singola riga o colonna, mentre Grid eccelle nel creare layout con righe e colonne.

Flexbox è content-first, cioè adatta il layout in base al contenuto. Grid, invece, è layout-first, permettendoti di creare il layout e poi posizionare gli elementi al suo interno. Grid ti dà un controllo più preciso sul posizionamento. Puoi dire a un elemento esattamente quale riga e colonna occupare.

Ecco un esempio di Flexbox per confronto:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
}
```

:::

Questo crea un contenitore flex dove gli elementi sono distribuiti uniformemente lungo l’asse principale.

Sia Grid che Flexbox hanno i loro punti di forza, e spesso i layout migliori usano una combinazione di entrambi. Potresti usare Grid per il layout generale della pagina e poi Flexbox per allineare gli elementi all’interno di ogni area griglia.

In sintesi, CSS Grid è uno strumento potente che permette layout precisi e bidimensionali. Anche se all’inizio può sembrare complesso, con la pratica diventa uno strumento indispensabile per creare layout web responsivi e complessi.
