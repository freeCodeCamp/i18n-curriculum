# --description--

Nas lições anteriores, você aprendeu como obter os valores máximos e mínimos de um intervalo de números assim:

```js
Math.max(1, 2, 3, 4, 5); // 5
Math.min(1, 2, 3, 4, 5); // 1
```

Crie uma variável chamada `maxNum` e atribua a ela o resultado de encontrar o número máximo entre os números `3, 125, 55, 24`. Então, registre o valor de `maxNum` no console.

Abaixo disso, crie uma variável chamada `minNum` e atribua a ela o resultado de encontrar o número mínimo entre os números `6, 90, 14, 90, 2`. Então, registre o valor de `minNum` no console.
