# --description--

Hai notato come il valore all’interno del tuo array `rows` sia stato modificato direttamente? Questo si chiama <dfn>mutazione</dfn>. Man mano che impari di più sugli array, scoprirai quando è opportuno mutare un array e quando invece non dovresti farlo.

Prima di andare avanti, questa è un’ottima occasione per imparare un uso comune degli array. Attualmente, il tuo codice consulta l’ultimo elemento dell’array con `rows[2]`. Ma potresti non sapere quanti elementi ci sono in un array quando vuoi prendere l’ultimo.

Puoi usare la proprietà `.length` di un array — questa restituisce il numero di elementi nell’array. Per ottenere l’ultimo elemento di qualsiasi array, puoi usare la seguente sintassi:

```js
array[array.length - 1]
```

`array.length` restituisce il numero di elementi nell’array. Sottraendo `1`, ottieni l’indice dell’ultimo elemento nell’array. Puoi applicare lo stesso concetto al tuo array `rows`.

Aggiorna il tuo `rows[2]` per consultare dinamicamente l’ultimo elemento nell’array `rows`. Usa l’esempio sopra per aiutarti.

Non dovresti vedere alcun cambiamento nella tua console.
