# --description--

`Additional` bedeutet zusätzlich oder mehr, so wie wenn man sagt, `She added additional sugar to the cake`.

`Logs` in der Technologie beziehen sich auf Aufzeichnungen von Ereignissen oder Prozessen, wie zum Beispiel `The system logs show when the app was used`.
