# --interactive--

Katika mafundisho yaliyopita, tumekuwa tukifanya kazi na sifa ya `grid-template-columns` na kuweka thamani kwa sehemu chache za sehemu.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

Ingawa msimbo ufuatao ni halali kabisa, kuna njia rahisi ya kurudia sehemu au orodha zote za njia zako.

Kitendakazi cha `repeat()` kinatumika kurudia sehemu au orodha zote za njia kwa safu au nguzo. Kitendakazi hiki kinapokea idadi ya kurudia na njia unazotaka kurudia.

Hapa kuna toleo lililorekebishwa la mfano wa awali ukitumia kitendakazi cha repeat:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

Hakutakuwa na mabadiliko kwa mitindo katika kivinjari, lakini hii ni njia fupi zaidi ya kuandika thamani zinazorudiwa kwa nguzo.

Kitendakazi cha `repeat()` kitakubali muundo wowote halali unaoweza kutumia kwa safu au nguzo.

Hapa kuna mfano ukitumia kitendakazi cha `repeat()` kuweka nguzo ya kwanza na ya tatu kuwa pikseli 20 na nguzo ya pili na ya nne kuwa sehemu moja ya fractional.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(2, 20px 1fr);
  column-gap: 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

Wakati mwingine, unaweza kuchagua kuandika kila thamani binafsi badala ya kutumia kitendakazi cha `repeat()`. Lakini kuna nyakati ambapo kitendakazi hiki kinakuwa cha msaada, hasa unapotaka kurudia muundo fulani kwa orodha ya njia.
