# --interactive--

JavaScript에서 `indexOf()` 메서드는 배열 내 특정 요소의 첫 번째 인덱스를 찾는 데 유용합니다. 요소를 찾지 못하면 `-1`을 반환합니다. 기본 구문은 다음과 같습니다:

```js
array.indexOf(element, fromIndex)
```

`element`는 배열 내에서 검색하려는 값을 나타내고, `fromIndex` 매개변수는 검색을 시작할 위치입니다. `fromIndex` 매개변수는 선택 사항입니다. `fromIndex`가 제공되지 않으면 검색은 배열의 처음부터 시작합니다. 예제를 살펴보겠습니다:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "banana"];
let index = fruits.indexOf("banana");
console.log(index); // 1
```

:::

이 예제에서는 다양한 과일 이름을 포함하는 배열 `fruits`가 있습니다. `indexOf()` 메서드를 사용해 `banana` 배열 내 문자열 `fruits`의 인덱스를 찾습니다. `banana`이 인덱스 `1`에 있으므로 메서드는 `1`를 반환하며, 이는 `index` 변수에 저장되고 콘솔에 출력됩니다.

검색하는 요소가 배열에 없으면 `indexOf()`는 `-1`를 반환합니다. 예를 들어:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let index = fruits.indexOf("grape");
console.log(index); // -1
```

:::

여기서는 `grape`을 사용해 fruits 배열에서 문자열 `indexOf()`를 검색합니다. `grape`이 배열에 없으므로 메서드는 `-1`를 반환하며, 이는 `index` 변수에 저장되고 콘솔에 출력됩니다.

특정 인덱스 번호 이후부터 항목을 찾고 싶으면 다음 예제처럼 두 번째 인수를 전달할 수 있습니다:

:::interactive_editor

```js
let colors = ["red", "green", "blue", "yellow", "green"];
let index = colors.indexOf("green", 3);
console.log(index); // 4
```

:::

이 예제에서는 검색이 배열의 처음부터 시작하지 않고 인덱스 번호 `3`인 `yellow`부터 시작하며 `4` 결과를 얻습니다.
