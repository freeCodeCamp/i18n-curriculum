# --interactive--

HTMLで番号付きリストを扱うとき、`type`属性を使うと、数字、アルファベット、ローマ数字などの番号付けスタイルを指定できます。

`type`属性は`ol`（番号付きリスト）要素に設定でき、番号付けの形式を制御します：

- `1`は数字リスト（1、2、3、...）、

- `A`は大文字アルファベットリスト（A、B、C、...）、

- `a`は小文字アルファベットリスト（a、b、c、...）、

- `I`は大文字ローマ数字リスト（I、II、III、...）、

- `i`は小文字ローマ数字リスト（i、ii、iii、...）です。

属性セレクターを使って特定の`type`属性を持つ番号付きリストを指定する方法は以下の通りです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="A"] {
  color: purple;
  font-weight: bold;
}
```

:::

この例では、CSSルールが大文字アルファベットの番号付けを使う番号付きリストに対して、紫色のテキストと太字のフォントウェイトを適用します。これは`type="A"`で指定されています。

同様に、小文字ローマ数字を使う番号付きリストを、`type="i"`を持つものとして指定できます：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="i">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="i"] {
  color: green;
}
```

:::

このルールは、小文字ローマ数字を使う番号付きリストのテキスト色を緑に変えます。

番号付きリストの`type`属性に基づいてスタイルを設定できることで、ドキュメント内のリストの見た目をより細かく制御でき、コンテンツの構造に応じたカスタマイズが可能になります。
