# --description--

为了让马克笔有不同的颜色，需要给每支笔设置一个不同的类。 通过在 `class` 属性中列出多个类，并用空格隔开，你可以为一个元素添加多个类。 比如，下面的代码给 `div` 元素添加了 `animal` 和 `dog` 类：

```html
<div class="animal dog">
```

如果你给一个 HTML 元素添加多个类，你在样式表中列出的前几个类的样式可能会被之后的类覆盖。

首先，给第一个马克笔 `div` 元素添加类 `one`。
