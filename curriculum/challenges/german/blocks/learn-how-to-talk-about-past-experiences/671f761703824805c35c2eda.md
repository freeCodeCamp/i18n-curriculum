# --explanation--

Der Ausdruck, der durch die Verwendung von `get` gefolgt von einem Adjektiv entsteht, beschreibt einen Zustandswechsel oder das Werden eines bestimmten Zustands. Zum Beispiel:

- `They always get excited when a new Python update is released.` - `Get excited` bedeutet, begeistert oder begeistert zu werden. Das heißt, sie sind immer voller Begeisterung, wenn ein neues Python-Update erscheint.

- `He got tired after working on the project all night.` - Das bedeutet, er war vorher nicht müde, aber jetzt ist er müde, nachdem er die ganze Nacht an dem Projekt gearbeitet hat.

In Marias Satz fragt sie, ob Brian sich an die Begeisterung erinnert, die sie früher für neue Technologien bei der gemeinsamen Arbeit empfanden.
