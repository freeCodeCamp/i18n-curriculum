# --description--

Розширене присвоєння поєднує двочленну операцію з присвоєнням в одному кроці. Воно бере змінну, застосовує до неї операцію з іншим значенням і зберігає результат назад у ту ж змінну.

Якщо ви знайомі з мовою на кшталт JavaScript, ви, мабуть, чули про оператор додавання з присвоєнням (`+=`) або оператор віднімання з присвоєнням (`-=`), а також інші. Вони існують і в Python. Єдина різниця в тому, що їх називають **розширеними операторами присвоєння**.

Основний синтаксис розширеного присвоєння виглядає так:

```python
variable <operator>= value
```

Що є більш ефективним способом зробити це:

```python
variable = variable <operator> value
```

Наприклад, ось приклад використання розширеного присвоєння, щоб додати `5` до існуючої змінної:

```python
my_var = 10
my_var += 5

print(my_var) # 15
```

А ось те саме, але без розширеного присвоєння:

```python
my_var = 10
my_var = my_var + 5

print(my_var) # 15
```

Перевага розширеного присвоєння в тому, що воно забезпечує стислий і зрозумілий спосіб оновити значення змінної без повторення її імені. Це, у свою чергу, зменшує надмірність і потенційні помилки, які можуть виникнути через опечатку чи щось подібне.

Кожен оператор може використовувати розширене присвоєння. Ми розглянули оператор додавання з присвоєнням (`+=`), тож давайте подивимось на інші.

- Оператор віднімання з присвоєнням (`-=`) віднімає правий операнд від лівої змінної і зберігає різницю у лівій змінній:

```python
count = 14
count -= 3

print(count) # 11
```

- Оператор множення з присвоєнням (`*=`) множить ліву змінну на правий операнд і зберігає добуток назад у ліву змінну:
    

```python
product = 65
product *= 7

print(product) # 455
```

- Оператор ділення з присвоєнням (`/=`) ділить ліву змінну на правий операнд і зберігає результат назад у ліву змінну:
    

```python
price = 100
price /= 4

print(price) # 25.0
```

- Оператор цілочисельного ділення з присвоєнням (`//=`) виконує цілочисельне ділення лівої змінної на правий операнд і зберігає результат назад у ліву змінну:
    

```python
total_pages = 23
total_pages //= 5

print(total_pages) # 4
```

- Оператор за модулем з присвоєнням (`%=`) обчислює остачу від ділення лівої змінної на правий операнд і зберігає її назад у ліву змінну:
    

```python
bits = 35
bits %= 2

print(bits) # 1
```

- Оператор піднесення до степеня з присвоєнням (`**=`) підносить ліву змінну до степеня правого операнда і зберігає результат назад у ліву змінну:
    

```python
power = 2
power **= 3

print(power) # 8
```

Деякі розширені оператори присвоєння можна використовувати і з рядками. Наприклад, оператор додавання з присвоєнням полегшує конкатенацію рядків:

```python
greet = 'Hello'
greet += ' World'

print(greet) # Hello World
```

А оператор множення з присвоєнням можна використати для повторення рядка:

```python
greet = 'Hello'
greet *= 3

print(greet) # HelloHelloHello
```

Інші розширені оператори при використанні з рядками викликають `TypeError`:

```python
greet = 'Hello'
greet -= ' World'

print(greet) # TypeError: unsupported operand type(s) for -=: 'str' and 'str'


greet = 'Hello'
greet /= 'World'

print(greet) # TypeError: unsupported operand type(s) for /=: 'str' and 'str' 
```

Якщо ви цікавитесь, чи працюють оператори інкременту і декременту (`++` і `--`) у Python, то ні. Це тому, що Python свідомо уникає скорочень інкременту і декременту в стилі C, щоб зберегти мову чіткою і явною.

Замість `x++` ви можете просто написати `x += 1`, що явно показує, що ви збільшуєте значення `x` на `1`.

Написання `++x` у Python просто застосовує унарний плюс двічі і нічого не інкрементує:

```python
my_var = 5

print(+my_var)   # 5
print(++my_var)  # 5
print(+++my_var) # 5

my_var += 1

print(my_var) # 6
```
