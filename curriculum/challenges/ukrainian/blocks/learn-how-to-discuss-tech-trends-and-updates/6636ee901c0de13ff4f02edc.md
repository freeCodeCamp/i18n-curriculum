# --description--

Термін `run code` означає дію виконання або запуску програми чи скрипта комп’ютером. `run` у цьому контексті означає розпочати процес і довести його до кінця, доки програма не виконає те, для чого вона призначена, чи то обчислення чисел, сортування даних, чи щось інше.

Фраза `run code without managing servers` означає запуск комп’ютерних програм або застосунків без необхідності керувати фізичними чи віртуальними серверами, на яких виконується код. Це ключовий аспект безсерверних обчислень, де провайдер хмари керує серверами, дозволяючи розробникам більше зосереджуватися на кодуванні, а не на управлінні сервером.
