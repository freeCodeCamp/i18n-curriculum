# --explanation--

`Brute force attacks` são quando hackers tentam múltiplas senhas até encontrarem a correta. Por exemplo:

`The system detected a brute force attack on the login page.` - O ataque tentou muitas tentativas de senha para invadir.

Tentativas de `SQL injection` acontecem quando hackers tentam inserir código SQL malicioso para acessar ou modificar um banco de dados. Por exemplo:

`The firewall stopped two SQL injection attempts.` - Alguém tentou invadir o banco de dados usando código malicioso.
