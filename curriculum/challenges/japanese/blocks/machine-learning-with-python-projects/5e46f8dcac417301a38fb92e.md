## Cell 3

ここからあなたの番です！このセル内の各変数を正しく設定してください。（`None`ではなくなっているはずです）

3つの画像データセット（訓練、検証、テスト）それぞれのために画像ジェネレーターを作成します。画像を読み込み/デコードして浮動小数点テンソルに変換するには`ImageDataGenerator`を使います。`rescale`引数（他の引数は今のところ使いません）を使って、テンソルの値を0から255の範囲から0から1の範囲にスケールしてください。

`*_data_gen`変数には`flow_from_directory`メソッドを使います。バッチサイズ、ディレクトリ、ターゲットサイズ（`(IMG_HEIGHT, IMG_WIDTH)`）、クラスモード、その他必要なものを渡してください。`test_data_gen`が最も難しいでしょう。`test_data_gen`には`shuffle=False`を`flow_from_directory`メソッドに渡すことを忘れないでください。これにより、最終的な予測がテストで期待される順序のままになります。`test_data_gen`についてはディレクトリ構造をよく観察することも役立ちます。

コードを実行すると、出力は以下のようになるはずです：

```py
Found 2000 images belonging to 2 classes.
Found 1000 images belonging to 2 classes.
Found 50 images belonging to 1 class.
```
