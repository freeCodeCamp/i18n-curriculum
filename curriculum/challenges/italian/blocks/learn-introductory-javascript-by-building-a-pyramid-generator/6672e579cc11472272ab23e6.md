# --description--


JavaScript è il linguaggio di programmazione che alimenta il web. A differenza di HTML e CSS che hai imparato in precedenza, JavaScript viene usato più comunemente per scrivere la logica invece del markup.

In questo progetto imparerai le basi di JavaScript e applicherai questi concetti per creare un generatore di piramidi.

Un generatore di piramidi è un programma in cui puoi impostare il tipo di carattere, il numero per la piramide e la direzione della piramide. Il programma genererà quindi una piramide basata su questi input.

Clicca sul pulsante "Check your code" per passare al passaggio successivo e iniziare il progetto.
