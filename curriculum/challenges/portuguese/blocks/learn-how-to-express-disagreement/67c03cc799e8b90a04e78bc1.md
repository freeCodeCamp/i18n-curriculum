# --explanation--

`I see your point, but` é uma forma de reconhecer a preocupação de alguém enquanto apresenta uma opinião diferente.Por exemplo:

- Pessoa 1: `Let's add more features to the app.`

- Pessoa 2: `I see your point, but too many features might make it confusing.` - Esta pessoa entende mas está preocupada em tornar o aplicativo mais difícil de usar.
