# --description--

Das Wort `earlier` wird verwendet, um über eine Zeit vor dem aktuellen Moment oder der besprochenen Zeit zu sprechen. Es ist so, als würde man `before now` oder `some time ago` sagen.

Zum Beispiel bedeutet `If you finished your work earlier, you can relax now`, dass wenn Sie Ihre Arbeit vor einiger Zeit erledigt haben, Sie jetzt Zeit zum Entspannen haben.
