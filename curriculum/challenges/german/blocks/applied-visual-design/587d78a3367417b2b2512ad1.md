# --description--

Farbtheorie und ihre Auswirkungen auf das Design sind ein tiefgehendes Thema, und in den folgenden Aufgaben werden nur die Grundlagen behandelt. Auf einer Website kann Farbe Aufmerksamkeit auf Inhalte lenken, Emotionen hervorrufen oder visuelle Harmonie schaffen. Die Verwendung verschiedener Farbkombinationen kann das Aussehen einer Website wirklich verändern, und es kann viel Überlegung in die Auswahl einer Farbpalette fließen, die zu Ihren Inhalten passt.

Das Farbrad ist ein nützliches Werkzeug, um zu visualisieren, wie Farben zueinander in Beziehung stehen – es ist ein Kreis, bei dem ähnliche Farbtöne Nachbarn sind und unterschiedliche Farbtöne weiter voneinander entfernt liegen. Wenn zwei Farben sich gegenüber auf dem Rad befinden, nennt man sie komplementäre Farben. Sie haben die Eigenschaft, dass sie sich bei Kombination „aufheben“ und eine graue Farbe erzeugen. Werden sie jedoch nebeneinander platziert, wirken diese Farben lebendiger und erzeugen einen starken visuellen Kontrast.

Einige Beispiele für komplementäre Farben mit ihren Hex-Codes sind:

<blockquote>rot (#FF0000) und cyan (#00FFFF)<br>grün (#00FF00) und magenta (#FF00FF)<br>blau (#0000FF) und gelb (#FFFF00)</blockquote>

Dies unterscheidet sich vom veralteten RYB-Farbmodell, das viele von uns in der Schule gelernt haben und das andere Primär- und Komplementärfarben verwendet. Die moderne Farbtheorie nutzt das additive RGB-Modell (wie auf einem Computerbildschirm) und das subtraktive CMY(K)-Modell (wie beim Drucken).

Es gibt viele Online-Farbwähler, die eine Option bieten, die Komplementärfarbe einer Farbe zu finden.

**Hinweis:** Farbe kann eine wirkungsvolle Möglichkeit sein, einer Seite visuelles Interesse zu verleihen. Allerdings sollte Farbe nicht die einzige Methode sein, um wichtige Informationen zu vermitteln, da Nutzer mit Sehbeeinträchtigungen diese Inhalte möglicherweise nicht verstehen. Dieses Thema wird in den Applied Accessibility-Aufgaben ausführlicher behandelt.

# --instructions--

Ändern Sie die `background-color`-Eigenschaft der `blue`- und `yellow`-Klassen in ihre jeweiligen Farben. Beachten Sie, wie die Farben nebeneinander anders wirken als im Vergleich zum weißen Hintergrund.

# --hints--

Das `div`-Element mit der Klasse `blue` sollte eine `background-color` in Blau haben.

```js
const blueElement = document.querySelector('.blue');
const blueStyle = window.getComputedStyle(blueElement); 
assert.equal(blueStyle?.backgroundColor, 'rgb(0, 0, 255)');
```

Das `div`-Element mit der Klasse `yellow` sollte eine `background-color` in Gelb haben.

```js
const yellowElement = document.querySelector('.yellow');
const yellowStyle = window.getComputedStyle(yellowElement);
assert.equal(yellowStyle?.backgroundColor, 'rgb(255, 255, 0)');
```
