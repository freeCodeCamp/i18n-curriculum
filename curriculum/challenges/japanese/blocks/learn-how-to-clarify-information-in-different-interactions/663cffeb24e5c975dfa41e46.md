# --description--

`Double check` は、何かが正しいかどうかをもう一度確認することを意味します。たとえば、`I wanted to be sure I locked the door, so I double checked it` と言うことができます。これは、ミスを避けるために注意深く行動し、自分がしたことを再確認していることを意味します。

`schedule` は、物事がいつ起こるかを示す計画です。たとえば、`school schedule` は授業の開始時間と終了時間を示します。これは、一日の異なる時間に何をすべきかを知るのに役立ちます。プロジェクトでは、特定の部分が始まる日や終わる日を見積もるために使うことができます。

`Accurate` は、何かが正しくて間違いがないことを意味します。たとえば、時計が正しい時間を示していれば、それは `accurate` です。これは情報が真実で正確であることを意味します。
