# --interactive--

`toString()` 方法是 JavaScript 中一個基本的特性，用於將一個值轉換為其字串表現。你可以對數字、布林值、陣列和物件使用這個方法。`toString()` 最常見的用途之一是將數字轉換為其字串表現。以下是一個範例：

:::interactive_editor

```js
const num = 10;
console.log(num.toString()); // "10"
```

:::

此方法接受一個選用的基數，該基數是介於 `2` 到 `36` 的數字。此基數表述基底，例如二進位的基底為 2，八進位的基底為 8。如果未指定基數，則預設為基底 10，即十進位。以下是將 `2` 作為引數傳遞給 `toString()` 方法的範例：

:::interactive_editor

```js
const num = 10;
console.log(num.toString(2)); // "1010"
```

:::

結果將會是 `1010`，這是十進位數字 `10` 的二進位表述。

你也可以使用 `toString()` 方法將陣列和物件轉換為字串。陣列有訂製的 `toString()` 實作，會將每個元素轉換為字串並以逗號連接。以下是一個範例：

:::interactive_editor

```js
const arr = [1, 2, 3];
console.log(arr.toString()); // "1,2,3"
```

:::

在這個範例中，陣列的所有元素都被串接在一起形成字串 `1,2,3`。 

當 `toString()` 方法用於物件時，結果不會是物件屬性的字串化版本。 

:::interactive_editor

```js
const person = {
  name: "John",
  age: 30,
  isStudent: true
};

console.log(person.toString()); // "[object Object]"
```

:::

在這個範例中，結果將會是該物件的預設字串表現形式 `[object Object]`。若要取得 `person` 物件屬性的字串化版本，你需要使用 `JSON.stringify()`，你將在未來的課程中學習更多相關內容。

總結來說，`toString()` 方法用於將值轉換為字串。了解它如何與不同資料型別一起運作，以及如何為你的物件自訂它，可以大幅提升你在 JavaScript 應用程式中操作和顯示資料的能力。
