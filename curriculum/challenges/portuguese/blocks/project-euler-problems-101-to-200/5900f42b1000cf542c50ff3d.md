# --description--

Seja $S_m = (x_1, x_2, \ldots, x_m)$ o $m$-tuplo de números reais positivos com $x_1 + x_2 + \cdots + x_m = m$ para o qual $P_m = x_1 \times {x_2}^2 \times \cdots \times {x_m}^m$ é maximizado.

Por exemplo, pode-se verificar que $[P_{10}] = 4112$ ([ ] é a parte inteira da função).

Encontre $\sum {[P_m]}$ para $2 ≤ m ≤ 15$.
