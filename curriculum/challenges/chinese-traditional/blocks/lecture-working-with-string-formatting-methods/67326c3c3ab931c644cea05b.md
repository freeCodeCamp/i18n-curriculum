# --interactive--

在 JavaScript 中處理字串時，常會遇到字串開頭或結尾有不需要的空白字元。空白字元會干擾比較、儲存或顯示等操作行為，因此知道如何高效地將其移除非常重要。

在本課程中，我們將探討如何使用 JavaScript 的 `trim()`、`trimStart()` 和 `trimEnd()` 方法來修剪空白字元。

空白字元是指出現在字串中但不可見的空格、製表符或換行字元。例如：

```js
let greeting = "   Hello, world!   ";
```

在這種情況下，顯示的 `Hello, world!` 文字前後都有空格。

`trim()` 方法是最常用來移除字串開頭和結尾空白的方式。以下是一個範例：

:::interactive_editor

```js
let message = "   Hello!   ";
console.log(message); // "   Hello!   "
let trimmedMessage = message.trim();
console.log(trimmedMessage);  // "Hello!"
```

:::

在這種情況下，`trim()` 方法會移除所有前後的空格，只留下 `Hello!`。請注意，字串內部的任何空白（例如字與字之間）不會被 `trim()` 影響。

有時候，你可能只想從字串的開頭或結尾移除空白字元，但不是兩者同時移除。這時候就會用到 `trimStart()` 和 `trimEnd()`。

`trimStart()` 會移除字串開頭（或起始處）的空白字元。

:::interactive_editor

```js
let greeting = "   Hello!   ";
console.log(greeting);  // "   Hello!   "
let trimmedStart = greeting.trimStart();
console.log(trimmedStart);  // "Hello!   "
```

:::

`trimEnd()` 會移除字串結尾的空白字元。

:::interactive_editor

```js
let greeting = "   Hello!   ";
console.log(greeting);  // "   Hello!   "
let trimmedEnd = greeting.trimEnd();
console.log(trimmedEnd);  // "   Hello!"
```

:::

這些方法讓你能更精確地控制你想要清理的字串零件。

總結來說，修剪空白字元是使用 JavaScript 處理字串的重要部分。無論你是想清理輸入的資料，或是確保字串比較的一致性，你都可以使用 `trim()` 來移除字串兩側的空白字元，或使用 `trimStart()` 和 `trimEnd()` 來針對特定一側進行移除。
