# --description--

Ядро полігона визначається множиною точок, з яких видно всю межу полігона. Полярним полігоном називаємо полігон, для якого початок координат строго міститься всередині його ядра.

Для цієї задачі полігон може мати колінеарні послідовні вершини. Однак полігон не може мати самоперетину і не може мати нульову площу.

Наприклад, лише перший із наведених нижче є полярним полігоном (ядра другого, третього та четвертого не містять строго початок координат, а п’ятий взагалі не має ядра):

<img alt="five example polygons" src="https://cdn.freecodecamp.org/curriculum/project-euler/polar-polygons.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Зверніть увагу, що перший полігон має три послідовні колінеарні вершини.

Нехай $P(n)$ — кількість полярних полігонів, таких що вершини $(x, y)$ мають цілі координати, абсолютні значення яких не більші за $n$.

Зверніть увагу, що полігони слід рахувати різними, якщо вони мають різну множину ребер, навіть якщо вони охоплюють однакову площу. Наприклад, полігон з вершинами [(0,0), (0,3), (1,1), (3,0)] відрізняється від полігона з вершинами [(0,0), (0,3), (1,1), (3,0), (1,0)].

Наприклад, $P(1) = 131$, $P(2) = 1\\,648\\,531$, $P(3) = 1\\,099\\,461\\,296\\,175$ і $P(343)\bmod 1\\,000\\,000\\,007 = 937\\,293\\,740$.

Знайдіть $P(7^{13})\bmod 1\\,000\\,000\\,007$.
