# --explanation--

`Actually` é usado para mostrar a verdade real ou para corrigir um mal-entendido.Por exemplo:

`I actually finished the project early.` – Isso significa que você terminou antes do esperado.

`Quite a few` significa um número maior do que apenas alguns.Por exemplo:

`There were quite a few people at the meeting.` – Isso significa que muitas pessoas participaram.

`Those` é uma palavra demonstrativa usada para se referir a itens específicos que não estão próximos.Por exemplo:

`Those are my files over there.` – Isso aponta para arquivos mais distantes.
