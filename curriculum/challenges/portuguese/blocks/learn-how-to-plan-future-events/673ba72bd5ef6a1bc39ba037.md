# --explanation--

`Downtime` é um período em que um sistema ou serviço está indisponível ou não operacional.Frequentemente é agendado para realizar manutenção, atualizações ou reparos. 

`Server downtime` refere-se especificamente ao tempo em que um servidor — um sistema de computador que fornece dados, serviços ou recursos para outros dispositivos — está temporariamente indisponível.Por exemplo:

- `The server downtime was scheduled at midnight to minimize the impact on users.` - Isso significa que o servidor ficou indisponível à meia-noite para reduzir o transtorno aos usuários.

- `Unexpected downtime occurred due to a power failure.` - Isso significa que o sistema ficou indisponível sem planejamento prévio por causa de uma queda de energia.
