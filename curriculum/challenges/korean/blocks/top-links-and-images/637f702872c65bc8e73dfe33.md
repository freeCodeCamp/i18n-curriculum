# --description--


일반적으로 만들게 될 링크는 두 가지 종류가 있습니다:

- 인터넷상의 다른 웹사이트 페이지로 가는 링크

- 자신의 웹사이트 내에 있는 페이지로 가는 링크


## 절대 링크
인터넷상의 다른 웹사이트 페이지로 가는 링크를 절대 링크라고 합니다. 일반적인 절대 링크는 다음 부분들로 구성됩니다: `protocol://domain/path`. 절대 링크는 항상 목적지의 프로토콜과 도메인을 포함합니다.

이미 절대 링크가 어떻게 작동하는지 본 적이 있습니다. 이전에 The Odin Project의 About 페이지로 만든 링크는 프로토콜과 도메인을 포함하고 있으므로 절대 링크였습니다.

`https://www.theodinproject.com/about`

## 상대 링크
자신의 웹사이트 내 다른 페이지로 가는 링크를 상대 링크라고 합니다. 상대 링크는 도메인 이름을 포함하지 않습니다. 같은 사이트 내 다른 페이지이기 때문에 링크를 만든 페이지와 도메인 이름이 같다고 가정합니다.

상대 링크는 링크를 만드는 페이지를 기준으로 다른 페이지의 파일 경로만 포함합니다. 이 개념은 다소 추상적이니 예제를 통해 살펴보겠습니다.

`odin-links-and-images` 디렉터리 내에 `about.html`라는 이름의 HTML 파일을 새로 만들고 다음 코드를 붙여넣으세요:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Odin Links and Images</title>
  </head>

  <body>
    <h1>About Page</h1>
  </body>
</html>
```

다시 `index` 페이지로 돌아가서 `about` 페이지로 가는 링크를 만들기 위해 다음 앵커 요소를 추가하세요:

```html
<body>
  <h1>Homepage</h1>
	<a href="https://www.theodinproject.com/about">click me</a>

	<a href="about.html">About</a>
</body>
```

브라우저에서 `index.html` 파일을 열고 about 링크를 클릭해 제대로 연결되었는지 확인하세요. 링크를 클릭하면 방금 만든 `about` 페이지로 이동해야 합니다.

이것이 가능한 이유는 `index`와 `about` 페이지가 같은 디렉터리에 있기 때문입니다. 따라서 링크의 `about.html` 값으로 단순히 이름(`href`)만 사용할 수 있습니다.

하지만 보통은 웹사이트 디렉터리를 좀 더 체계적으로 정리하고 싶을 것입니다. 일반적으로 루트 디렉터리에는 `index.html`만 두고 나머지 HTML 파일은 각자의 디렉터리에 둡니다.

`pages` 디렉터리 내에 `odin-links-and-images`라는 디렉터리를 만들고 `about.html` 파일을 이 새 디렉터리로 옮기세요.

브라우저에서 `index` 페이지를 새로 고침한 후 `about` 링크를 클릭하세요. 이제 링크가 깨졌을 것입니다. 이는 `about` 페이지 파일 위치가 변경되었기 때문입니다.

이 문제를 고치려면 `about` 링크의 `href` 값을 `pages/` 디렉터리를 포함하도록 업데이트하면 됩니다. 이 디렉터리가 `about.html` 파일의 새로운 위치이고 `index.html` 파일을 기준으로 한 상대 경로이기 때문입니다.

```html
<body>
  <h1>Homepage</h1>
  <a href="pages/about.html">About</a>
</body>
```

브라우저에서 `index` 페이지를 새로 고침하고 `about` 링크를 다시 클릭해 보세요. 이제 정상 작동할 것입니다.

대부분의 경우 이 방법으로 잘 작동하지만, 예상치 못한 문제가 발생할 수도 있습니다. 링크 앞에 `./`를 붙이면 대부분 이런 문제를 방지할 수 있습니다. `./`를 추가하면 코드가 **현재** 디렉터리를 기준으로 파일이나 디렉터리를 찾기 시작하도록 지정하는 것입니다.

```html
<body>
  <h1>Homepage</h1>
  <a href="./pages/about.html">About</a>
</body>
```
