# --description--

Otra forma de comparar declaraciones condicionales sería el operador ternario. El operador ternario es una forma abreviada de escribir una declaración `if-else`. Se usa para evaluar una condición y devolver un valor basado en el resultado de la condición. La sintaxis del operador ternario es la siguiente:

```javascript
let result = condition ? value1 : value2;
```

En el ejemplo anterior, si la condición es verdadera, el valor de `result` será `value1`. De lo contrario, el valor de `result` será `value2`.

Los operadores ternarios se usan a menudo para asignar un valor a una variable basado en una condición. También se usan para devolver un valor basado en una condición.
