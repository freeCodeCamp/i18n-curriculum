# --explanation--

`Unusually` ist ein Adverb, das verwendet wird, um etwas zu beschreiben, das mehr oder weniger als normal oder erwartet passiert. Andererseits ist `unusual` ein Adjektiv, das etwas beschreibt, das nicht üblich oder normal ist. Zum Beispiel:

- `The server experienced an unusually high load today.` – In diesem Satz bezieht sich `unusually` darauf, wie hoch die Last war (weit über dem Normalen).

- `It was unusual for the server to slow down like that.` – In diesem Satz bezieht sich `unusual` auf die Situation. Es modifiziert kein Verb oder Adjektiv.

Eine `query` ist eine Frage oder Anfrage, die an eine Datenbank gesendet wird, um Informationen zu erhalten oder eine Aktion auszuführen. Zum Beispiel:

`The application sent a query to the database to fetch user details.` – Das bedeutet, dass die App die Datenbank nach bestimmten Informationen gefragt hat.
