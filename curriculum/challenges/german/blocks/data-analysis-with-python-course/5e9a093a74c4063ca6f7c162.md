## --text--

Wie würden Sie die CSV-Datei `data.csv` importieren und mit dem Pandas-Modul in einem DataFrame speichern?

## --answers--

```python
import pandas as pd
df = pd.csv("data.csv")
```

---

```python
import pandas as pd
df = pd.read_csv("data.csv")
```

---

```python
import pandas as pd
pd.read_csv("data.csv")
```

---

```python
import pandas as pd
df = pd.csv_reader("data.csv")
```
