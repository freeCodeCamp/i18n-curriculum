# --description--

配列で使われるより複雑なメソッドの一つに`splice()`と`slice()`メソッドがあります。`splice()`メソッドは、配列の要素を削除または置き換えることで配列の内容を変更します。`slice()`メソッドは、配列の一部を浅くコピーして新しい配列オブジェクトに返します。コピーする範囲は`begin`から`end`まで（`end`は含まれません）です。元の配列は変更されません。

例えば、`characters`配列から2番目の要素を削除するには、以下のコードを使えます。

```javascript
const characters = ['Harry', 'Ron', 'Hermione'];
characters.splice(1, 1);
console.log(characters); // Output: ['Harry', 'Hermione']
```

上記のコードは`characters`配列から2番目の要素を削除します。`splice()`メソッドは2つの引数を取ります：削除する要素のインデックスと削除する要素の数です。

`character`配列から2番目の要素を含む新しい配列を作成するには、以下のコードを使えます。

```javascript
const characters = ['Harry', 'Ron', 'Hermione'];
const newCharacters = characters.slice(1, 2);
console.log(newCharacters); // Output: ['Ron']
```

上記のコードは`newCharacters`配列の2番目の要素を含む新しい配列`characters`を作成します。`slice()`メソッドは2つの引数を取ります：スライスを開始する要素のインデックスとスライスを終了する要素のインデックス（終了インデックスの要素は含まれません）。
