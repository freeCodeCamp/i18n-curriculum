# --description--

Існують різні форми порушення сприйняття кольору. Вони можуть варіюватися від зниженої чутливості до певної довжини хвилі світла до повної нездатності бачити колір. Найпоширенішою формою є знижена чутливість до зелених відтінків.

Наприклад, якщо два схожі зелені кольори є кольорами переднього плану та фону вашого вмісту, користувач з порушенням сприйняття кольору може не розрізнити їх. Близькі кольори можна уявити як сусідів на колірному колі, і такі поєднання слід уникати при передачі важливої інформації.

**Примітка:** Деякі онлайн-інструменти для вибору кольорів включають візуальні симуляції того, як кольори виглядають для різних типів порушення сприйняття кольору. Це чудові ресурси на додаток до онлайн-калькуляторів перевірки коефіцієнта контрастності.

# --instructions--

Кіт Кемпер тестує різні стилі для важливої кнопки, але жовтий (`#FFFF33`) `background-color` і зелений (`#33FF33`) текст `color` — це сусідні відтінки на колірному колі і практично нерозрізнені для деяких користувачів з порушенням сприйняття кольору. (Їхня схожа світлість також не проходить перевірку коефіцієнта контрастності). Змініть текст `color` на темно-синій (`#003366`), щоб вирішити обидві проблеми.

# --hints--

Ваш код має змінити текст `color` для `button` на темно-синій.

```js
const button = document.querySelector('button');
const buttonColor = window.getComputedStyle(button).color; 
assert.equal(buttonColor, 'rgb(0, 51, 102)');
```
