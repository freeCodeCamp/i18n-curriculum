# --explanation--

`Which` kann verwendet werden, um zusätzliche Informationen zur vorherigen Klausel einzufügen. Dies nennt man eine **nicht-definierende Relativklausel** – sie fügt Details hinzu, ist aber für die Bedeutung des Satzes nicht wesentlich. Zum Beispiel:

`The system crashed during the update, which caused a delay.` – Der Teil nach `which` gibt mehr Informationen über das Ergebnis des Absturzes, könnte aber weggelassen werden, ohne den Hauptpunkt zu verändern.
