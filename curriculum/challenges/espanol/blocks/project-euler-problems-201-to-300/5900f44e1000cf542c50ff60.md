# --description--

La secuencia 1, 1, 1, 3, 5, 9, 17, 31, 57, 105, 193, 355, 653, 1201 ...

se define por $T_1 = T_2 = T_3 = 1$ y $T_n = T_{n - 1} + T_{n - 2} + T_{n - 3}$.

Se puede demostrar que 27 no divide a ningún término de esta secuencia. De hecho, 27 es el primer número impar con esta propiedad.

Encuentra el ${124}^{\text{ésimo}}$ número impar que no divide a ningún término de la secuencia anterior.
