# --interactive--

`time` 元素用於表述特定的時間點。

這裡是一個使用 `time` 元素來表述二千小時，或晚上八點的範例。

:::interactive_editor

```html
<p>The reservations are for <time datetime="20:00">20:00 </time></p>
```

:::

`datetime` 屬性用於將日期和時間轉換為機器可讀格式。

這很重要，因為它有助於搜尋引擎結果，並幫助瀏覽器更有效地處理日期和時間資訊。

`datetime` 屬性的值必須是有效的年份、有效的月份、有效的時間、區域日期、全域日期，或有效的持續時間字串。

這裡是另一個使用 `time` 元素來表述特定日期的範例：

:::interactive_editor

```html
<p>
  The graduation will be on <time datetime="2024-06-15T15:00">June 15</time>
</p>
```

:::

`datetime` 屬性的值採用 ISO 8601 格式。ISO 8601 是一個用來表述日期和時間的國際標準。

該值的第一個零件是年、月和日。該值中的大寫 T 是日期和時間之間的分隔符。

十五百時是下午 3 點。

每當你需要表述事件、發佈日期或約會時，最好使用 `time` 元素。
