# --description--

이전 수업에서는 리스트 데이터 타입을 소개하고 리스트에서 요소에 접근하는 방법과 리스트 슬라이싱을 배웠습니다. 이번 수업에서는 리스트와 관련된 몇 가지 자주 사용되는 메서드인 `append()`, `pop()`, `sort()`에 대해 계속 배워보겠습니다.

먼저 살펴볼 메서드는 `append()` 메서드입니다. 이 메서드는 리스트의 끝에 항목을 추가하는 데 사용됩니다. 다음은 `append()` 메서드를 사용해 숫자 `6`를 숫자 리스트에 추가하는 예시입니다:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

한 리스트를 다른 리스트의 끝에 추가하고 싶다면, 다음과 같이 `append()` 메서드를 사용할 수도 있습니다:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

`even_numbers` 리스트 전체가 `numbers` 리스트 안에 중첩된 것을 확인하세요.

하지만 `even_numbers` 리스트의 개별 숫자들을 `numbers` 리스트 끝에 모두 추가하고 싶다면 `extend()` 메서드를 사용할 수 있습니다.

`extend()` 메서드는 `append()` 메서드와 비슷하지만, `extend()`를 사용하면 한 리스트에서 다른 리스트로 여러 요소를 추가할 수 있습니다. 다음은 `6`, `8`, `10` 숫자들을 한 리스트에서 `numbers` 리스트 끝에 추가하는 예시입니다:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

보시다시피 중첩된 리스트가 사라지고 숫자 리스트만 남았습니다.

리스트의 특정 인덱스에 요소를 삽입하려면 `insert()` 메서드를 사용할 수 있습니다. 이 메서드는 두 개의 인수를 받는데, 첫 번째는 새 항목을 삽입할 인덱스이고 두 번째는 삽입할 항목입니다.

다음은 `insert()` 메서드를 사용하는 예시입니다:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

다음 코드는 `2.5` 리스트의 인덱스 `2`에 숫자 `numbers`를 삽입합니다.

리스트에서 요소를 제거하려면 `remove()` 메서드를 사용할 수 있습니다. `remove()` 메서드는 제거할 요소의 값을 인수로 받습니다:

```py
numbers = [10, 20, 30, 40, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50]
```

이 메서드는 항목의 첫 번째 발생만 제거한다는 점을 기억하세요. 모두 제거하지는 않습니다:

```py
numbers = [10, 20, 30, 40, 50, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50, 50]
```

리스트에서 특정 인덱스의 요소를 제거하려면 다음과 같이 `pop()` 메서드를 사용할 수 있습니다:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

`pop` 메서드에 요소를 지정하지 않으면 마지막 요소가 제거됩니다.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

리스트를 비워야 할 경우 다음과 같이 `clear()` 메서드를 사용할 수 있습니다:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

다음으로 살펴볼 메서드는 `sort()` 메서드입니다. 이 메서드는 리스트의 요소들을 제자리에서 정렬하는 데 사용됩니다. 다음은 무작위 숫자 리스트를 제자리에서 정렬하는 예시입니다:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

`sort()` 메서드와 달리, `sorted()` 함수는 모든 반복 가능한 객체에 대해 작동하며 원본 리스트를 수정하지 않고 새로 정렬된 리스트를 반환합니다. 예를 들어:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(numbers) # [19, 2, 35, 1, 67, 41]
print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
```

참고로 반복 가능한 객체는 한 번에 하나씩 각 항목에 접근할 수 있도록 반복할 수 있는 특별한 유형의 객체입니다. 파이썬에서 반복문이 어떻게 작동하는지는 이후 수업에서 더 배우게 됩니다.

`sort()` 메서드와 `sorted()` 함수 모두 선택적인 `key`와 `reverse` 매개변수를 받습니다. 이 선택적 매개변수에 대해서는 튜플에 대해 배우는 이후 수업에서 더 자세히 다룹니다. `sorted()` 같은 다른 내장 함수도 이후 수업에서 배우게 됩니다.

다음으로 살펴볼 메서드는 `reverse()` 메서드입니다. 이 메서드는 리스트의 요소들을 제자리에서 뒤집습니다:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

마지막으로 살펴볼 메서드는 `index` 메서드입니다. 이 메서드는 리스트에서 특정 요소가 처음 발견되는 인덱스를 찾는 데 사용됩니다. 다음은 `index` 메서드를 사용해 `'Java'` 리스트에서 언어 `programming_languages`를 찾는 예시입니다:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

만약 요소를 찾을 수 없으면 파이썬은 `ValueError`를 발생시킵니다:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: 'JavaScript' is not in list
"""
```

파이썬 리스트에는 더 많은 메서드가 있지만, 이 초기 메서드 목록이 시작하기에 좋은 지점입니다.
