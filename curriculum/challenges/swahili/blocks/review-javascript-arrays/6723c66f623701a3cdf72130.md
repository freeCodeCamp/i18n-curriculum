## Misingi ya safu ya data ya JavaScript

- **Maelezo**: Safu ya data ya JavaScript ni mkusanyiko uliopangwa wa thamani, kila moja ikitambulishwa na kielezo cha nambari. Thamani katika safu ya data ya JavaScript zinaweza kuwa za aina tofauti za data, ikiwa ni pamoja na nambari, mfuatano wa herufi, thamani ya kweli au siyo kweli, vitu, na hata safu nyingine. Safu za data ziko mfululizo katika kumbukumbu, ambayo ina maana kwamba vipengele vyote vinahifadhiwa katika sehemu moja mfululizo ya maeneo ya kumbukumbu, kuruhusu upatikanaji wa haraka na ufanisi kwa kutumia kielezo.

```js
const developers = ["Jessica", "Naomi", "Tom"];
```

- **Kupata vipengele kutoka katika safu ya data**: Ili kupata vipengele kutoka katika safu ya data, unahitaji kurejea safu hiyo ikifuatiwa na nambari ya kielezo ndani ya mabano ya mraba. Safu za data za JavaScript zina kielezo cha kuanzia sifuri, ambayo ina maana kipengele cha kwanza kiko kwenye kielezo 0, kipengele cha pili kiko kwenye kielezo 1, n.k. Ikiwa utajaribu kupata kielezo ambacho hakipo katika safu, basi JavaScript itarudisha `undefined`.

:::interactive_editor

```js
const developers = ["Jessica", "Naomi", "Tom"];
console.log(developers[0]) // "Jessica"
console.log(developers[1]) // "Naomi"

console.log(developers[10]) // undefined
```

:::

- **Sifa ya `length`**: Sifa hii hutumika kurudisha idadi ya vitu vilivyo katika safu ya data.

:::interactive_editor

```js
const developers = ["Jessica", "Naomi", "Tom"];
console.log(developers.length) // 3
```

:::

- **Kusahihisha vipengele katika safu ya data**: Ili kusahihisha kipengele katika safu ya data, unatumia opereta ya mgawo (`=`) kuweka thamani mpya kwa kipengele kilicho kwenye kielezo fulani.

:::interactive_editor

```js
const fruits = ['apple', 'banana', 'cherry'];
fruits[1] = 'blueberry';

console.log(fruits); // ['apple', 'blueberry', 'cherry']
```

:::

## Safu za data za vipimo viwili  

- **Maelezo**: Safu ya data ya vipimo viwili ni safu ya safu za data. Inatumika kuwakilisha data yenye muundo wa gridi, kama vile ubao wa chess, karatasi ya hesabu, au pikseli katika picha. Ili kupata kipengele katika safu ya data ya vipimo viwili, unahitaji vielezo viwili: kimoja kwa safu na kingine kwa safu wima.

:::interactive_editor

```js
const chessboard = [
    ['R', 'N', 'B', 'Q', 'K', 'B', 'N', 'R'],
    ['P', 'P', 'P', 'P', 'P', 'P', 'P', 'P'],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    ['p', 'p', 'p', 'p', 'p', 'p', 'p', 'p'],
    ['r', 'n', 'b', 'q', 'k', 'b', 'n', 'r']
];

console.log(chessboard[0][3]); // "Q"
```

:::

## Ufumbuzi wa muundo wa safu ya data  

- **Maelezo**: Ufumbuzi wa muundo wa safu ya data ni kipengele katika JavaScript kinachokuwezesha kutoa thamani kutoka katika safu za data na kuziweka katika vigezo kwa njia fupi na inayoeleweka zaidi. Hutoa sintaksia rahisi ya kufungua vipengele vya safu ya data katika vigezo tofauti.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange"];

const [first, second, third] = fruits;

console.log(first); // "apple"
console.log(second); // "banana"
console.log(third); // "orange"
```

:::

- **Sintaksia ya REST**: Hii inakuwezesha kukamata vipengele vilivyobaki vya safu ya data ambavyo havijafumbuliwa na kuweka katika safu mpya.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange", "mango", "kiwi"];
const [first, second, ...rest] = fruits;

console.log(first); // "apple"
console.log(second); // "banana"
console.log(rest); // ["orange", "mango", "kiwi"]
```

:::

## Njia za kawaida za safu ya data 

- **Njia ya `push()`**: Njia hii hutumika kuongeza vipengele mwishoni mwa safu ya data na itarudisha urefu mpya wa safu.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.push("ice cream");

console.log(desserts); // ["cake", "cookies", "pie", "ice cream"];
```

:::

- **Njia ya `pop()`**: Njia hii hutumika kuondoa kipengele cha mwisho kutoka katika safu ya data na itarudisha kipengele kilichoondolewa. Ikiwa safu ni tupu, basi thamani ya return itakuwa `undefined`.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.pop();

console.log(desserts); // ["cake", "cookies"];
```

:::

- **Njia ya `shift()`**: Njia hii hutumika kuondoa kipengele cha kwanza kutoka katika safu ya data na kurudisha kipengele kilichoondolewa. Ikiwa safu ni tupu, basi thamani ya return itakuwa `undefined`.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.shift();

console.log(desserts); // ["cookies", "pie"];
```

:::

- **Njia ya `unshift()`**: Njia hii hutumika kuongeza vipengele mwanzoni mwa safu ya data na itarudisha urefu mpya wa safu.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.unshift("ice cream");

console.log(desserts); // ["ice cream", "cake", "cookies", "pie"];
```

:::

- **Njia ya `indexOf()`**: Njia hii ni muhimu kwa kupata kielezo cha kwanza cha kipengele fulani ndani ya safu ya data. Ikiwa kipengele hakipatikani, basi itarudisha `-1`.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange", "banana"];
const index = fruits.indexOf("banana");

console.log(index); // 1
console.log(fruits.indexOf("not found")); // -1
```

:::

- **Njia ya `splice()`**: Njia hii hutumika kuongeza au kuondoa vipengele kutoka sehemu yoyote katika safu ya data. Thamani ya return ya njia ya `splice()` itakuwa safu ya vitu vilivyoondolewa kutoka safu. Ikiwa hakuna kitu kilichoondolewa, basi safu tupu itarudishwa. Njia hii itabadilisha safu ya asili, ikibadilisha mahali hapo badala ya kuunda safu mpya. Hoja ya kwanza inaeleza kielezo cha kuanzia cha kubadilisha safu. Hoja ya pili ni idadi ya vipengele unavyotaka kuondoa. Hoja zinazofuata ni vipengele unavyotaka kuongeza.

:::interactive_editor

```js
const colors = ["red", "green", "blue"];
colors.splice(1, 0, "yellow", "purple");

console.log(colors); // ["red", "yellow", "purple", "green", "blue"]
```

:::

- **Njia ya `includes()`**: Njia hii hutumika kuangalia kama safu ina thamani fulani. Njia hii inarudisha `true` ikiwa safu ina kipengele kilichotajwa, na `false` vinginevyo.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];

console.log(programmingLanguages.includes("Python")); // true
console.log(programmingLanguages.includes("Perl")); // false
```

:::

- **Njia ya `concat()`**: Njia hii huunda safu mpya kwa kuunganisha safu mbili au zaidi.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];
const newList = programmingLanguages.concat("Perl");

console.log(newList); // ["JavaScript", "Python", "C++", "Perl"]
```

:::

- **Njia ya `slice()`**: Njia hii inarudisha safu mpya inayojumuisha nakala ya sehemu ya safu ya asili, iliyoainishwa na vielezo vya kuanzia na kumalizika. Safu mpya ina rejea kwa vipengele sawa na safu ya asili (si nakala za vipengele). Hii ina maana kwamba ikiwa vipengele ni vitu vya msingi (kama nambari au mfuatano wa herufi), thamani zinakopiwa; lakini ikiwa vipengele ni vitu au safu, rejea zinakopiwa, si vitu wenyewe.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];
const newList = programmingLanguages.slice(1);

console.log(newList); // ["Python", "C++"]
```

:::

- **Sintaksia ya spread**: Sintaksia ya spread hutumika kuunda nakala za kina za safu ya data.

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const shallowCopiedArray = [...originalArray];

shallowCopiedArray.push(4);

console.log(originalArray); // [1, 2, 3]
console.log(shallowCopiedArray); // [1, 2, 3, 4]
```

:::

- **Njia ya `split()`**: Njia hii hugawanya mfuatano wa herufi kuwa safu ya sehemu ndogo ndogo na inaeleza mahali kila mgawanyo unapaswa kutokea kulingana na kitenzi kilichotolewa. Ikiwa hakuna kitenzi kilichotolewa, njia hii inarudisha safu yenye mfuatano wa herufi wa awali kama kipengele kimoja.

:::interactive_editor

```js
const str = "hello";
const charArray = str.split("");

console.log(charArray); // ["h", "e", "l", "l", "o"]
```

:::

- **Njia ya `reverse()`**: Njia hii inageuza safu ya data mahali hapo.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
console.log(desserts.reverse()); // ["pie", "cookies", "cake"]
```

:::

- **Njia ya `join()`**: Njia hii huunganisha vipengele vyote vya safu ya data kuwa mfuatano wa herufi mmoja, kila kipengele kikiwa kimegawanywa na kitenzi kilichotolewa. Ikiwa hakuna kitenzi kilichotolewa, au mfuatano wa herufi tupu (`""`) umetumika, vipengele vitachanganywa bila kitenzi chochote kati yao.

:::interactive_editor

```js
const reversedArray = ["o", "l", "l", "e", "h"];
const reversedString = reversedArray.join("");

console.log(reversedString); // "olleh"
```
