# --description--

1つ以上の単語を含む文字列が与えられたとき、次の条件に従って単語の頭字語を返してください。

- 頭字語は各単語の最初の文字を大文字にしたものとします。ただし、特に指定がある場合を除きます。
- これらの単語 `a`、`for`、`an`、`and`、`by`、および `of` の最初の文字は、与えられた文字列の最初の単語でない限り無視してください。
- 頭字語の文字は与えられた順序で返してください。
- 頭字語にスペースは含めないでください。

```js
function buildAcronym(str) {
  const smallWords = ["a", "for", "an", "and", "by", "of"];
  const words = str.split(" ");
  let acronym = "";

  for (let i = 0; i < words.length; i++) {
    const word = words[i];
    const lowerWord = word.toLowerCase();

    if (i === 0 || !smallWords.includes(lowerWord)) {
      acronym += word[0].toUpperCase();
    }
  }

  return acronym;
}
```
