# --description--

設 $R(a, b, c)$ 為三個不重疊圓形在邊長為 $a$、$b$ 和 $c$ 的三角形內所涵蓋的最大面積。

令 $S(n)$ 為所有整數三元組 $(a, b, c)$（滿足 $1 ≤ a ≤ b ≤ c &lt; a + b ≤ n$）中 $R(a, b, c)$ 的平均值。

你得到 $S(2) = R(1, 1, 1) ≈ 0.31998$，$S(5) ≈ 1.25899$。

找出 $S(1803)$ 並將小數點後四捨五入至 5 位數。
