# --description--

フレーズ `not that much` は、量や頻度についての質問に対して、少しだけ、わずかにという意味を示すために使われます。例えば：

- **ソフィー:** `Do you travel for work?` - これは仕事のために他の場所や都市に行くかどうかを尋ねています。

- **Dakota:** `Not that much.` - これは少しだけ、あまり頻繁ではないことを意味します。

一方で、`at all times` は常に、継続的にという意味で、中断なく起こることを示します。例えば：

`Security guards must be alert at all times.` - これは警備員が人々の安全を守るために常に注意深く見守らなければならないことを意味します。
