# --description--

Urithi unahusu vigezo fulani vya CSS ambavyo, vinapotumika kwa kipengele, vinarithiwa na mzao wa kipengele hicho, hata kama hatuandiki sheria moja kwa moja kwa mzao hao. Vigezo vinavyohusiana na aina ya herufi (`color`, `font-size`, `font-family`, n.k.) kawaida hurithiwa, wakati vigezo vingine vingi havirithiwi.

Tofauti ni pale unapolenga moja kwa moja kipengele, kwani hii daima hushinda urithi:

```html
<!-- index.html -->

<div id="parent">
  <div class="child"></div>
</div>
```

```css
/* styles.css */

#parent {
  color: red;
}

.child {
  color: blue;
}
```

Licha ya kipengele cha `parent` kuwa na umahiri mkubwa zaidi kwa kutumia ID, kipengele cha `child` kingepata mtindo wa `color: blue` kwa sababu tamko hilo linamlenga moja kwa moja, wakati `color: red` kutoka kwa mzazi ni urithi tu.
