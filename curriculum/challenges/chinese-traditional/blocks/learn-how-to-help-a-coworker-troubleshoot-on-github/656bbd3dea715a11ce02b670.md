# --description--

在程式設計中，特別是在使用像 Git 這樣的版本控制系統時，`push` 意味著將變更從區域的儲存庫上傳到遠端儲存庫。 

過去式 `pushed` 顯示這個動作已經完成。 

例如，`He pushed his code to the GitHub repository` 意味著他將最近的程式碼更改上傳到 GitHub。
