# --description--

令 $x$ 为实数。

对于分母边界 `d`，对 $x$ 的最佳近似是一个最简形式的有理数 $\frac{r}{s}$，其中 $s ≤ d$，且任何比 $\frac{r}{s}$ 更接近 $x$ 的有理数，其分母都大于 $d$：

$$|\frac{p}{q} - x| &lt; |\frac{r}{s} - x| ⇒ q > d$$

例如，对于分母边界 `20`，$\sqrt{13}$ 的最佳近似是 $\frac{18}{5}$，对于分母边界 `30`，$\sqrt{13}$ 的最佳近似是 $\frac{101}{28}$。

查找所有分母边界为 `${10}^{12}` 的对 $\sqrt{n}$ 的最佳近似分母之和，其中 $n$ 不是完全平方数且 $1 < n ≤ 100000$。
