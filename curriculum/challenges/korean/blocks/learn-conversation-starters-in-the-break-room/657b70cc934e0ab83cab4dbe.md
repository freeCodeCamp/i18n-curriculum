# --description--

지금까지 `do` / `does`로 질문하는 방법과 `don't` / `doesn't`로 부정문을 만드는 방법을 연습했습니다. 이를 검토해 보세요. 기억하세요:

- 긍정문: `He does work on weekends.` (그 사람이 주말에 일한다는 의미)

- 부정문: `He doesn't work on weekends.` (그 사람이 주말에 일하지 않는다는 의미)

- 의문문: `Does he work on weekends?` (그 사람이 주말에 일하는지 묻는 문장)

# --instructions--

대화 발췌문을 읽고 `do` 또는 `don't`을 사용해 빈칸을 채우세요.

# --fillInTheBlank--

## --sentence--

`Tom: How about you? BLANK you have any hobbies?`

`Sophie: That's cool. I like photography, but I BLANK think of it as a hobby.`
