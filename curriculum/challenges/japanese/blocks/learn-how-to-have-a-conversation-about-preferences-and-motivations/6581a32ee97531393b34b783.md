# --description--

`Encourage` は、誰かに何かをする自信や支援を与えることを意味します。たとえば、友達が `You can solve this coding problem!` と言ったら、それはあなたを励ましていることになります。

`Discourage` はその反対で、誰かの自信を失わせたり、何かをする気持ちを弱めたりすることを意味します。その友達が `That programming language is too hard for beginners` と言ったら、あなたがそれを学ぶ気持ちをそぐかもしれません。

`Discouraging` は、気持ちが落ち込むようなものを表します。たとえば、`Finding a bug in your code after hours of work can be discouraging.` のようなものです。

`Encouraging` は、勇気や希望を与えるものを指します。たとえば、`Getting your first program to run is very encouraging.` のようなものです。
