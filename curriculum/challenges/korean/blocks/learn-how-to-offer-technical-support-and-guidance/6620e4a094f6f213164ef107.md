# --description--

`To grasp`는 무언가를 이해한다는 의미의 동사입니다. `grasping`나 `is` 같은 단어 뒤에 오면 `am`로 변하여 지금 일어나고 있는 동작을 설명합니다. 예를 들어, `She is grasping the concept`는 그녀가 현재 그 개념을 이해하고 있다는 뜻입니다.

누군가 `great job grasping`라고 말하면, "무언가를 아주 잘 이해했다"는 의미입니다. 특히 어려운 것을 배웠을 때 칭찬하는 표현입니다.

예를 들어, 컴퓨터 사용법을 이해하기 시작했을 때 누군가 `Great job grasping how to use the computer!`라고 말하면, 새로운 것을 잘 배우고 있다는 뜻입니다.
