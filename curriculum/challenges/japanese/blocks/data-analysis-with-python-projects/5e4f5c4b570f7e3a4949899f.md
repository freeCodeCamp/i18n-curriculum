# --instructions--

1880年以降の世界の平均海面変動のデータセットを分析します。このデータを使って2050年までの海面変動を予測します。

以下のタスクをデータを使って完了してください：

- Pandasを使って`epa-sea-level.csv`からデータをインポートしてください。
- matplotlibを使って、`Year`列をx軸、`CSIRO Adjusted Sea Level`列をy軸にした散布図を作成してください。
- `linregress`から`scipy.stats`関数を使って最適な直線の傾きと切片を取得してください。散布図の上に最適な直線を描画してください。2050年まで直線を伸ばして2050年の海面上昇を予測してください。
- 2000年からデータセットの最新年までのデータだけを使って新しい最適な直線を描画してください。この直線も2050年まで伸ばして、2000年以降の上昇率が続いた場合の2050年の海面上昇を予測してください。
- x軸のラベルは`Year`、y軸のラベルは`Sea Level (inches)`、タイトルは`Rise in Sea Level`にしてください。

ボイラープレートには画像を保存して返すコマンドも含まれています。
