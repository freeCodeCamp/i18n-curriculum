# --description--

Angenommen, Sie möchten ein Wort finden, das mehrfach vorkommt, wie unten gezeigt.

```js
let repeatStr = "row row row your boat";
```

Sie könnten `/row row row/` verwenden, aber was, wenn Sie das spezifische wiederholte Wort nicht kennen? <dfn>Capture-Gruppen</dfn> können verwendet werden, um wiederholte Teilstrings zu finden.

Capture-Gruppen werden gebildet, indem das zu erfassende Regex-Muster in runde Klammern eingeschlossen wird. In diesem Fall besteht das Ziel darin, ein Wort aus alphanumerischen Zeichen zu erfassen, daher wird die Capture-Gruppe `\w+` in runde Klammern eingeschlossen: `/(\w+)/`.

Der vom Capture-Gruppe gematchte Teilstring wird in einer temporären „Variablen“ gespeichert, auf die innerhalb desselben Regex mit einem Backslash und der Nummer der Capture-Gruppe zugegriffen werden kann (z. B. `\1`). Capture-Gruppen werden automatisch nach der Position ihrer öffnenden runden Klammern von links nach rechts nummeriert, beginnend bei 1.

Das folgende Beispiel findet ein Wort, das dreimal vorkommt und durch Leerzeichen getrennt ist:

```js
let repeatRegex = /(\w+) \1 \1/;
repeatRegex.test(repeatStr); // Returns true
repeatStr.match(repeatRegex); // Returns ["row row row", "row"]
```

Die Verwendung der `.match()`-Methode auf einem String gibt ein Array mit dem gematchten Teilstring sowie seinen erfassten Gruppen zurück.


# --instructions--

Verwenden Sie Capture-Gruppen in `reRegex`, um einen String zu finden, der nur aus derselben Zahl besteht, die genau dreimal durch einzelne Leerzeichen getrennt wiederholt wird.
