# --description--

Nel corso delle lezioni precedenti, hai imparato a definire funzioni usando la parola chiave `def` in questo modo:

```py
def square(num):
    return num ** 2

print(square(4)) # 16
```

Ma quando lavori con funzioni di ordine superiore come `map()` e `filter()`, puoi usare una funzione anonima in linea. Ed è qui che entrano in gioco le funzioni lambda.

Ecco come appare la funzione `square()` rifattorizzata in una funzione lambda:

```py
lambda num: num ** 2
```

Come detto prima, le funzioni lambda sono anonime, quindi questa funzione non ha più il nome `square` associato. Le funzioni lambda sono ottime quando devi usarle in funzioni di ordine superiore come questa:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

In questo esempio, abbiamo una lista di numeri e vogliamo crearne una nuova contenente solo i numeri pari. Passiamo quindi una funzione lambda come uno degli argomenti alla funzione `filter()` per ottenere una nuova lista con i numeri `2` e `4`.

Quando usi le funzioni lambda è importante seguire le buone pratiche. Per esempio, non è una buona pratica assegnare una funzione lambda a una variabile in questo modo:

```py
numbers = [1, 2, 3, 4, 5]

square = lambda x: x ** 2
squared_numbers = list(map(square, numbers))
print(squared_numbers) # [1, 4, 9, 16, 25]
```

Questo vanifica lo scopo di usare funzioni anonime. In questo caso, dovresti usare una funzione normale, come questa:

```py
numbers = [1, 2, 3, 4, 5]

def square(num):
    return num ** 2

squared_numbers = list(map(square, numbers))
print(squared_numbers) # [1, 4, 9, 16, 25]
```

Inoltre, dovresti evitare di creare funzioni lambda difficili da leggere o inutilmente complicate, come questa:

```py
result = (lambda x: (x**2 + 2*x - 1) if x > 0 else (x**3 - x + 4))(3)
print(result)  # 14
```

Anche se questa funzione funziona correttamente e produce il risultato giusto `14`, non è facile da leggere o da capire. In questo caso, sarebbe meglio creare una funzione separata con un’istruzione `if/else` e poi chiamare quella funzione:

```py
def calculate_expression(x):
    if x > 0:
        return x**2 + 2*x - 1
    else:
        return x**3 - x + 4

print(calculate_expression(3))  # 14
```

Sia le funzioni normali sia le funzioni lambda hanno il loro utilizzo nei programmi Python. Se devi gestire una singola espressione in linea, potresti considerare di usare una funzione lambda. Altrimenti, usare una funzione normale è la scelta migliore.
