# --description--

在上一課中，你學到了 Python 中常見的資料型別，包括字串、整數和浮點數型別。當你撰寫程式時，你需要學會如何查看變數的型別。

這裡有一個範例變數：

```py
developer = 'Devin'
```

要查看 `developer` 是什麼型別，你可以像這樣使用 `type()` 函式：

```py
developer = 'Devin'

print(type(developer)) # <class 'str'>
```

`<class 'str'>` 的輸出表示 `developer` 是字串型別。

如果你沒有提供任何引數給 `type()` 函式，則會收到以下錯誤訊息：

```md
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: type() takes 1 or 3 arguments
```

以下是你目前學過的所有資料型別以及它們在終端機中的型別顯示：

```python
my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 'hello', 8.5}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 'hello', 8.5)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>
```

在你的程式中，有時你需要確認某個變數是否為特定型別，才能對它執行操作。這時 `isinstance()` 函式就非常有用。

這裡有一個被指定為字串的範例變數：

```py
account_balance = '12'
```

如果你嘗試對 `account_balance` 變數執行像除法這樣的數學運算，將會收到錯誤訊息。

```py
account_balance = '12'

account_balance / 2

# Traceback (most recent call last):
#   File "<stdin>", line 1, in <module>
# TypeError: unsupported operand type(s) for /: 'str' and 'int'
```

要檢查 `account_balance` 是否為整數，你可以像這樣使用 `isinstance()` 函式：

```py
account_balance = '12'

isinstance(account_balance, int) # False
```

內建的 `isinstance()` 函式讓你可以檢查變數是否符合特定資料型別。它接受一個物件和你想檢查的型別，然後傳回布林值。在這個例子中，因為 `account_balance` 是字串，所以會傳回 `False`。

`isinstance()` 函式也允許你一次檢查多個型別。

這裡有一個範例，檢查 `account_balance` 是否為 `int` 或 `float`：

```py
account_balance = 12
isinstance(account_balance, (int, float)) # True
```

在這個例子中，`account_balance` 是整數，所以 `isinstance()` 傳回 `True`。如果 `account_balance` 是 `12.0`，`isinstance()` 仍會傳回 `True`，因為你正在檢查是否為整數或浮點數。

在未來的工作坊和實驗中，你會使用 `type()` 和 `isinstance()` 函式，確保你的變數在執行操作前包含正確的資料型別。
