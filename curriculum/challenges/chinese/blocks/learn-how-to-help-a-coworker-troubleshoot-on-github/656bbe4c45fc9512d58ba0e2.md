# --description--

单词 `earlier` 用于表示当前时刻或正在讨论的时间之前的时间。它类似于说 `before now` 或 `some time ago`。

例如，`If you finished your work earlier, you can relax now` 意味着如果你在之前完成了工作，现在有时间放松。
