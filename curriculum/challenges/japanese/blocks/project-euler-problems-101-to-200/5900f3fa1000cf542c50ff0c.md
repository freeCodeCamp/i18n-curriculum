# --description--

無限多項式級数 $A_G(x) = xG_1 + x^2G_2 + x^3G_3 + \cdots$ を考えます。ここで $G_k$ は2次の漸化式 $G_k = G_{k − 1} + G_{k − 2}, G_1 = 1, G_2 = 4$ の第 $k$ 項であり、すなわち $1, 4, 5, 9, 14, 23, \ldots$ となります。

この問題では、$A_G(x)$ が正の整数となる $x$ の値に注目します。

最初の5つの自然数に対応する $x$ の値は以下の通りです。

| $x$                         | $A_G(x)$ |
|-----------------------------|----------|
| $\frac{\sqrt{5} − 1}{4}$    | $1$      |
| $\frac{2}{5}$               | $2$      |
| $\frac{\sqrt{22} − 2}{6}$   | $3$      |
| $\frac{\sqrt{137} − 5}{14}$ | $4$      |
| $\frac{1}{2}$               | $5$      |

$x$ が有理数である場合、$A_G(x)$ をゴールデンナゲットと呼びます。これは有理数の値が次第に希少になるためです。例えば、20番目のゴールデンナゲットは 211345365 です。最初の30個のゴールデンナゲットの合計を求めてください。
