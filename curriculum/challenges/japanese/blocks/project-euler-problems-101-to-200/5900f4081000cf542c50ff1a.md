# --description--

電気回路は、すべて同じ値 C の同一のコンデンサーのみを使用します。

コンデンサーは直列または並列に接続してサブユニットを形成でき、そのサブユニットをさらに他のコンデンサーやサブユニットと直列または並列に接続してより大きなサブユニットを作り、最終的な回路を構成します。

この単純な手順を使い、最大で n 個の同一コンデンサーを用いることで、さまざまな合計静電容量の回路を作ることができます。例えば、$n = 3$ 個のそれぞれ $60 μF$ のコンデンサーを使うと、以下の7つの異なる合計静電容量の値が得られます。

<img alt="example circuits having up to three capacitors, each of 60 μF" src="https://cdn.freecodecamp.org/curriculum/project-euler/counting-capacitor-circuits.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$n$ 個までの同じ値のコンデンサーを使い、上記の単純な手順で得られる異なる合計静電容量の数を $D(n)$ と表すと、$D(1) = 1, D(2) = 3, D(3)=7, \ldots$ となります。

$D(18)$ を求めてください。

補足：コンデンサー $C_1$, $C_2$ などを並列に接続すると合計静電容量は $C_T = C_1 + C_2 + \cdots$ となり、直列に接続すると全体の静電容量は $\frac{1}{C_T} = \frac{1}{C_1} + \frac{1}{C_2} + \cdots$ で与えられます。
