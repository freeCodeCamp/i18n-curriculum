# --description--

人们根据不同的情况以不同的方式交流。

你可以使用 `teléfono` 直接与某人通话，或者使用 `mensaje de texto`，这是一种通过电话发送的简短书面消息。

你也可以使用 `correo`，它指的是传统邮件，例如寄送到家庭地址的信件。如今，许多消息是在线发送的。在这些情况下，它被称为 `correo electrónico`。

`Electrónico` 描述使用技术或电子设备的事物。
