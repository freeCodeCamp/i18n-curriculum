# --explanation--

`To prompt` 意思是促使或鼓勵某人做某事或給出回應。例如：

`The app prompts users to update their settings.`－這表示該應用程式會提醒或要求使用者採取行動。

某些 `unique` 是獨一無二的，或與其他任何東西不同。 例如：

`Each user gets a unique code for account activation.`－這表示每個人都有不同的 code。
