# --description--

Quasi tutti gli elementi in una pagina HTML sono semplicemente pezzi di contenuto racchiusi tra tag HTML di apertura e chiusura.

I tag di apertura indicano al browser l'inizio di un elemento HTML. Sono composti da una parola chiave racchiusa tra parentesi angolari `<>`. Per esempio, un tag di apertura per un paragrafo appare così: `<p>`.

I tag di chiusura indicano al browser dove termina un elemento. Sono quasi identici ai tag di apertura; l’unica differenza è che hanno una barra obliqua prima della parola chiave. Per esempio, un tag di chiusura per un paragrafo appare così: `</p>`.
