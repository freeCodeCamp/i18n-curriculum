# --description--

Sea $S_m = (x_1, x_2, \ldots, x_m)$ la estructura de $m$ números reales positivos con $x_1 + x_2 + \cdots + x_m = m$ para la cual $P_m = x_1 \times {x_2}^2 \times \cdots \times {x_m}^m$ está maximizado.

Por ejemplo, se puede verificar que $[P_{10}] = 4112$ ( [ ] es la función parte entera).

Encuentra $\sum {[P_m]}$ para $2 ≤ m ≤ 15$.
