# --explanation--

`Initial setup`는 프로젝트의 첫 단계, 즉 실제 작업이 시작되기 전에 시스템, 도구, 접근을 준비하는 것을 의미합니다. 예를 들어:

`The initial setup of the system took one day.` — 시스템 설치의 시작 부분은 하루가 걸렸습니다.

`Early development`는 보통 프로토타입이나 기본 기능을 포함하는 제작 과정의 시작을 의미합니다. 예를 들어:

`In early development, we focused on design.` — 처음에는 주로 외관 작업을 했습니다.
