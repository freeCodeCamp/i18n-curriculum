# --description--

Pig Latin هي طريقة لتغيير الكلمات الإنجليزية باتباع قواعد تحويل محددة.

**الهدف:** نفذ قصص المستخدم أدناه واجعل جميع الاختبارات تنجح لإكمال المختبر.

**قصص المستخدم:**

1. يجب أن تنشئ دالة `translatePigLatin` تقبل سلسلة نصية واحدة كمعلمة.
1. إذا بدأت السلسلة النصية بمعرف صوتي، يجب أن تأخذ الدالة أول معرف صوتي أو مجموعة معرفات صوتية، تنقلها إلى نهاية الكلمة، تضيف إليها `ay`، ثم تُرجع النتيجة.
1. إذا بدأت السلسلة النصية بحرف متحرك، يجب أن تضيف الدالة `way` في النهاية وتُرجع النتيجة.
1. يجب أن تتعامل الدالة مع السلاسل النصية التي يظهر فيها أول حرف متحرك في وسط الكلمة وتُرجع السلسلة المحوّلة بشكل مناسب.
1. إذا لم تحتوي السلسلة النصية على أحرف متحركة، يجب أن تضيف الدالة `ay` في النهاية وتُرجع النتيجة.

ملاحظة: في سياق هذا المختبر، الأحرف المتحركة هي `a` و `e` و `i` و `o` و `u`. الحرف `y` لا يُعتبر حرفًا متحركًا.

```js
function translatePigLatin(str) {
  if (isVowel(str.charAt(0))) return str + "way";
  let front = [];
  str = str.split('');
  while (str.length && !isVowel(str[0])) {
    front.push(str.shift());
  }
  return [].concat(str, front).join('') + 'ay';
}

function isVowel(c) {
  return ['a', 'e', 'i', 'o', 'u'].indexOf(c.toLowerCase()) !== -1;
}
```
