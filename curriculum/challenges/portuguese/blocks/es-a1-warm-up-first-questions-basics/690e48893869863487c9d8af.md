# --description--

Ángela está perguntando sobre a idade de alguém. Ela começa sua pergunta com uma palavra que literalmente significa "quantos" em espanhol.

# --instructions--

Ouça o áudio e responda a pergunta abaixo.

# --questions--

## --text--

Qual palavra Ángela está usando para começar sua pergunta?

## --answers--

`¿Qué…?`

### --feedback--

A palavra `Qué` significa "What". Não é usada para perguntar a idade de alguém.

---

`¿Cómo…?`

### --feedback--

A palavra `Cómo` significa "How". Não é usada para perguntar a idade de alguém.

---

`¿Cuántos…?`

---

`¿Cuál…?`

### --feedback--

A palavra `Cuál` significa "Which" ou "What" (quando selecionando entre opções). Não é usada para perguntar a idade de alguém.

## --video-solution--

3

# --explanation--

A palavra `Cuántos` traduz-se como "How many" e é usada para perguntar sobre uma quantidade.

Deve ter um acento na letra `a` quando fizer parte de uma pergunta.
