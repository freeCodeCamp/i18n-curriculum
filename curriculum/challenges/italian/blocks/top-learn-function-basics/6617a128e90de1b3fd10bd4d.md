# --description--

Parliamo di parametri e argomenti nel contesto della seguente funzione di esempio:

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

In JavaScript, i parametri sono gli elementi elencati tra le parentesi tonde `()` nella dichiarazione della funzione. Gli argomenti della funzione sono i valori effettivi che decidi di passare alla funzione.

Nell'esempio sopra, la definizione della funzione è scritta nella prima riga: `function favoriteAnimal(animal)`. Il parametro, `animal`, si trova all'interno delle parentesi tonde. Potresti altrettanto facilmente sostituire `animal` con `pet`, `x` o `blah`. Ma in questo caso, chiamare il parametro `animal` dà a chi legge il tuo codice un po' di contesto, così non deve indovinare cosa `animal` potrebbe contenere alla fine.
