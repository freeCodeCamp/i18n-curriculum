# --explanation--

`Might`는 가능성이나 불확실성을 표현하는 데 사용됩니다. 무언가가 가능하지만 확실하지 않다는 것을 암시합니다. 예를 들어:

- `The system might crash if too many users log in at the same time.` - 시스템 충돌 가능성이 있지만 확실하지 않습니다.

- `He might finish the coding task by the end of the day.` - 그가 작업을 끝낼 수도 있지만 확실하지 않습니다.
