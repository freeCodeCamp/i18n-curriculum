# --explanation--

질문에서 `can`, `could`, `should` 같은 조동사는 주어 앞에 옵니다. 그 뒤에 오는 동사는 기본형이어야 합니다. 그래서 `Could you help?`나 `Do you can help?` 대신 `Can helping?`라고 말합니다.

예를 들어, `Should we report this issue?`와 `Might he know the password?`에서는 조동사가 먼저 나오고, 본동사는 기본형으로 유지됩니다.

조동사와 함께 `do`나 `does`를 사용하는 것은 잘못된 표현입니다. `Do you can reset the system?`라고 말하지 않고 `Can you reset the system?`라고 해야 합니다.

영어 질문에서 어순이 중요합니다. `You can help me?`라고 하면 주어와 조동사의 순서가 맞지 않아 부자연스럽게 들립니다. 올바른 형태는 `Can you help me?`입니다.
