# --description--

フィボナッチ数列 $\\{f_n, n ≥ 0\\}$ は、$f_n = f_{n - 1} + f_{n - 2}$ という再帰的な定義で、初期条件は $f_0 = 0$ と $f_1 = 1$ です。

多項式 $\\{F_n, n ≥ 0\\}$ を $F_n(x) = \displaystyle\sum_{i = 0}^n f_ix^i$ と定義します。

例えば、$F_7(x) = x + x^2 + 2x^3 + 3x^4 + 5x^5 + 8x^6 + 13x^7$ であり、$F_7(11) = 268\\,357\\,683$ です。

$n = {10}^{15}$ とします。$\displaystyle\sum_{x = 0}^{100} F_n(x)$ の合計を求め、その答えを $1\\,307\\,674\\,368\\,000 \\, (= 15!)$ で割った余りとして答えてください。
