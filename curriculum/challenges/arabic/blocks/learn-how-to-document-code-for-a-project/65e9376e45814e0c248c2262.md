# --description--

`Clarify` تعني جعل شيء ما واضحًا أو أسهل للفهم. يمكنك أن `clarify` فكرة معقدة من خلال شرحها بكلمات أبسط. على سبيل المثال: `He clarified the instructions by providing a step-by-step guide.`

`Function` في البرمجة هي مجموعة من التعليمات تنفذ مهمة محددة. على سبيل المثال: `The "calculateTotal" function adds up all the prices.`

`variable` في البرمجة يشبه صندوقًا يمكنك تخزين المعلومات أو البيانات فيه داخل البرنامج. يمكنك تغيير ما بداخل الصندوق، أو `variable`، أثناء تشغيل البرنامج. هذه طريقة لتتبع البيانات واستخدامها في الشفرة. على سبيل المثال: `The variable "score" keeps track of the player's points.`
