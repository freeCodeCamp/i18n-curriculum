# --description--

一個整數 $n$ 的整數分割是將 $n$ 寫成正整數和的一種方式。

僅在加數順序上不同的分割被視為相同。將 $n$ 分割成不同零件是指每個零件最多出現一次的 $n$ 分割。

將 5 分割成不同零件的組合有：

5、4 + 1 和 3 + 2。

令 $f(n)$ 為將 $n$ 分割成不同零件時，任一此類分割的零件乘積的最大值，令 $m(n)$ 為具有該乘積的任一此類分割中零件的數量。

所以 $f(5) = 6$ 且 $m(5) = 2$。

對於 $n = 10$，乘積最大的分割是 $10 = 2 + 3 + 5$，其結果為 $f(10) = 30$ 和 $m(10) = 3$。它們的乘積為 $f(10) \times m(10) = 30 \times 3 = 90$。

可以驗證 $\sum f(n) \times m(n)$ 對於 $1 ≤ n ≤ 100 = 1\\,683\\,550\\,844\\,462$。

求 $\sum f(n) \times m(n)$，其中 $1 ≤ n ≤ {10}^{14}$。請將你的答案對 $982\,451\,653$ 取模，該數為第 5\,000\,0000 個質數。
