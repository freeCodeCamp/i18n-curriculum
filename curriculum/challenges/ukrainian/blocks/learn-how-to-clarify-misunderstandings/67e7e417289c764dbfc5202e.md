# --explanation--

У фразі `limit what other parts can see` `what` вводить іменникову умову, яка виконує роль додатка до `limit`.

Іменникова умова — це умова, яка виконує роль іменника в реченні. Вона може бути додатком або підметом речення. Зазвичай починається зі слів на кшталт `that`, `what`, `whether`, `who`, `when`, `why`, `how` тощо.

Іменникові умови мають звичайний порядок слів, як у стверджувальних реченнях (не питальних). Наприклад:

- `The team is checking why the system crashed during the update.` — `Why the system crashed during the update` є іменниковою умовою, що виконує роль додатка до `checking`.
   
- `What the function returns depends on the input data.` — `What the function returns` є іменниковою умовою, що виконує роль підмета `depends on`.
