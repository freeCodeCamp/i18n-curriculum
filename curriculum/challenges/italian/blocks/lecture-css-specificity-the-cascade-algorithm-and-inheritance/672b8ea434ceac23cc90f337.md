# --interactive--

I selettori id sono tra i selettori più potenti in CSS, permettendo agli sviluppatori di definire lo stile per elementi specifici con identificatori unici.

Questo li rende molto efficaci per mirare a singoli elementi che necessitano di uno stile unico.

I selettori id sono definiti da un simbolo hash (`#`) seguito dal nome dell'id. Dovrebbero essere unici all'interno di un documento HTML, quindi nessun elemento dovrebbe condividere lo stesso id.

Ecco un esempio che usa un selettore id:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="unique">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#unique {
  color: purple;
}
```

:::

In questo esempio, l'elemento con l'id `unique` avrà il colore del testo impostato su `purple`.

I selettori id hanno una specificità molto alta, superiore ai selettori di tipo e ai selettori di classe, ma inferiore agli stili in linea. Il valore di specificità per un selettore id è `(0, 1, 0, 0)`.

Questo significa che i selettori id possono sovrascrivere i selettori di classe e di tipo ma possono essere sovrascritti dagli stili in linea.
