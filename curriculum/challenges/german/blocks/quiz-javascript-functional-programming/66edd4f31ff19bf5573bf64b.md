### --question--

#### --text--

Was ist eine reine Funktion?

#### --distractors--

Eine Funktion, die globale Variablen verändert und entweder `undefined` oder `null` zurückgibt.

---

Eine Funktion, die eine Ausgabe in der Konsole protokolliert.

---

Eine Funktion, die ihr Verhalten basierend auf externen Faktoren ändert und `null` zurückgibt.

#### --answer--

Eine Funktion, die für dieselbe Eingabe immer dieselbe Ausgabe liefert und keine Nebeneffekte erzeugt.

### --question--

#### --text--

Was ist ein Nebeneffekt in der funktionalen Programmierung?

#### --distractors--

Ein unerwartetes `TypeError` in Ihrem Code.

---

Eine Funktion, die zu lange zur Ausführung braucht und das Programm abstürzen lässt.

---

Ein rekursiver Funktionsaufruf, der eine Endlosschleife erzeugt und das Programm abstürzen lässt.

#### --answer--

Eine Änderung des Programmzustands, die außerhalb der Funktion beobachtbar ist.

### --question--

#### --text--

Was ist Currying in der funktionalen Programmierung?

#### --distractors--

Eine Technik zum Entwerfen von Tests, die Ihnen hilft, sicherzustellen, dass Ihr Code zu 100 % fehlerfrei ist.

---

Eine optimierte Methode, um rekursive Funktionen so zu schreiben, dass sie doppelt so schnell laufen wie andere Funktionen.

---

Der Prozess des Erstellens, Entwerfens und Testens Ihres Codes, um sicherzustellen, dass er den von ECMAScript festgelegten Standards entspricht.

#### --answer--

Der Prozess, eine Funktion mit mehreren Argumenten in eine Folge von Funktionen umzuwandeln, von denen jede ein einzelnes Argument hat.

### --question--

#### --text--

Welches der folgenden Beispiele ist ein Beispiel für Currying?

#### --distractors--

```js
const curriedAverage = (a, b, c) => a + b + c / 3
```

---

```js
const curriedAverage = (a, b, c) => a + b + c
```

---

```js
function curriedAverage(a) {
  return a 
}

function curried(c) {
  return c
}
```

#### --answer--

```js
function curriedAverage(a) {
  return function(b) {
    return function(c) {
      return (a + b + c) / 3;
    };
  };
}
```
