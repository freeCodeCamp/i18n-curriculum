# --interactive--

In einer vorherigen Lektion wurden Sie erstmals mit dem `iframe`-Element vertraut gemacht. In dieser Lektion lernen Sie mehr darüber, wie Sie mit dem `iframe`-Element arbeiten. Dieses Element steht für Inline-Frame. Es ist ein Inline-Element, das verwendet wird, um andere HTML-Inhalte direkt innerhalb der HTML-Seite einzubetten. Diese HTML-Inhalte können ein Video, eine Karte, ein anderes HTML-Element oder sogar andere Webseiten sein.

Hier ist ein Beispiel, wie Sie einen beliebten freeCodeCamp-Kurs von YouTube einbetten. Um ein anderes Video zu sehen, aktivieren Sie den interaktiven Editor und ändern Sie den Wert von `src` zu einem Video Ihrer Wahl.

:::interactive_editor

```html
<iframe
  width="400"
  height="400"
  src="https://www.youtube.com/embed/PkZNo7MFNFg?si=-UBVIUNM3csdeiWF"
  title="Learn JavaScript - Full Course for Beginners (YouTube video)"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
  referrerpolicy="strict-origin-when-cross-origin"
  allowfullscreen
></iframe>
```

:::

Das `src`-Attribut gibt die URL der Seite an, die Sie einbetten möchten. Das `width`-Attribut gibt die Breite des `iframe` an. Das `height`-Attribut gibt die Höhe des `iframe` an. Das `allowfullscreen`-Attribut erlaubt dem Benutzer, das `iframe` im Vollbildmodus anzuzeigen. Es ist außerdem eine gute Praxis, ein `title`-Attribut für das `iframe` anzugeben, da dies für die Barrierefreiheit wichtig ist.

Das `allow`-Attribut hingegen ermöglicht es Ihnen, festzulegen, was ein `iframe` darf oder nicht darf. Das nennt man eine Allowlist. Im obigen Beispiel erlaubt das Hinzufügen von `clipboard-write`, dass die eingebettete Seite Elemente in Ihre Zwischenablage schreiben kann. Elemente in einer Allowlist können durch Semikolons oder Leerzeichen getrennt werden, und beides kann zusammen verwendet werden.

Beachten Sie, dass das Video von überallher stammen kann. Es muss nicht von Videodiensten wie YouTube oder Vimeo kommen.

Vergessen Sie nicht, dass Sie auch eine Karte, eine andere Webseite oder direkten HTML-Code innerhalb des `iframe`-Elements einbetten können. Hier ist ein Beispiel für eine eingebettete Karte.

Versuchen Sie, mit der Karte zu interagieren, indem Sie hinein- und herauszoomen.

:::interactive_editor

```html
<h1>A Map from Openstreetmap.org Embedded with the iframe Element</h1>

<iframe
  width="425"
  height="350"
  src="https://www.openstreetmap.org/export/embed.html?bbox=3.006134033203125%2C6.150112578753815%2C3.6357879638671875%2C6.749850810550778&amp;layer=mapnik"
  title="Map of Lagos area, Nigeria"
  style="border: 1px solid black"
>
</iframe>
<br />
<small>
  <a href="https://www.openstreetmap.org/#map=11/6.4501/3.3210">
    View Larger Map
  </a>
</small>
```

:::

Wenn Sie direktes HTML innerhalb des `iframe`-Elements einbetten möchten, müssen Sie das `srcdoc`-Attribut anstelle von `src` verwenden.
