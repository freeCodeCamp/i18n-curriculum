# --explanation--

`Work on` significa dedicar tiempo o esfuerzo a una tarea. Va seguido de un sustantivo o un verbo en forma de `-ing` (gerundio). Por ejemplo:

- `She is working on a new project at the moment.` - En este momento, ella está realizando tareas para un nuevo proyecto.

- `We need to work on improving our website.` - Debes mejorar el sitio web.
