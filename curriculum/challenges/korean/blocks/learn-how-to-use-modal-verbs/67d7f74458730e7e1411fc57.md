# --explanation--

조동사의 부정형을 만들려면 조동사 뒤에 `not`를 추가하면 됩니다. 예를 들어:

- `You shouldn't ignore security updates on your computer.` - 보안 업데이트를 설치하는 것이 중요하니, 건너뛰지 마세요.

- `We might not have enough time to finish the project by the deadline.` - 그들이 프로젝트를 제시간에 끝내지 못할 가능성이 있습니다.

조동사 `can`와 `should`의 경우, 부정형이 보통 축약형(`can't`와 `shouldn't`)으로 나타납니다.
