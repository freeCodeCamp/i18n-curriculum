# --description--

`extension` 是一个软件组件，为更大型的软件应用添加特定的能力。`Extensions` 可以提供新功能性或增强现有特色。

例如，浏览器 `extension` 可以为你的网页浏览器添加特殊功能性，比如屏蔽广告或管理密码。
