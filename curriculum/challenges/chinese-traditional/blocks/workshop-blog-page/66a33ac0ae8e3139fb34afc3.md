# --description--

現在已經完成了所有博文的添加，請嘗試點擊 `Posts` 鏈接，你會看到頁面跳轉到 `Posts` 部分。

添加到博客頁面的最後一個組件是聯繫方式部分。

在 `main` 元素下方，添加一個 `footer` 元素。

# --hints--

你的 `footer` 元素應該有一個開始標籤。

```js
assert.match(code, /<footer\>/);
```

你的 `footer` 元素應該有一個結束標籤。

```js
assert.match(code, /<\/footer\>/);
```

你的 `footer` 元素應該在 `body` 元素內。

```js
const footerElement = document.querySelector('footer');
assert.strictEqual(footerElement?.parentElement.tagName, 'BODY');
```
