# --description--

Albert wählt eine positive ganze Zahl $k$, dann werden zwei reelle Zahlen $a$, $b$ zufällig und gleichverteilt im Intervall [0,1] ausgewählt.

Anschließend wird die Quadratwurzel der Summe ${(ka + 1)}^2 + {(kb + 1)}^2$ berechnet und auf die nächste ganze Zahl gerundet. Wenn das Ergebnis gleich $k$ ist, erhält er $k$ Punkte; andernfalls erhält er keine Punkte.

Zum Beispiel, wenn $k = 6$, $a = 0,2$ und $b = 0,85$ sind, dann ist ${(ka + 1)}^2 + {(kb + 1)}^2 = 42,05$. Die Quadratwurzel von 42,05 ist 6,484... und gerundet auf die nächste ganze Zahl wird daraus 6. Das ist gleich $k$, also erhält er 6 Punkte.

Es lässt sich zeigen, dass wenn er 10 Runden mit $k = 1, k = 2, \ldots, k = 10$ spielt, der Erwartungswert seiner Gesamtpunktzahl, gerundet auf fünf Dezimalstellen, 10,20914 beträgt.

Wenn er ${10}^5$ Runden mit $k = 1, k = 2, k = 3, \ldots, k = {10}^5$ spielt, wie hoch ist der Erwartungswert seiner Gesamtpunktzahl, gerundet auf fünf Dezimalstellen?
