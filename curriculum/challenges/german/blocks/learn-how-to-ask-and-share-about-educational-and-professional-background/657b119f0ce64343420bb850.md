# --description--

Eine `database` ist eine strukturierte Sammlung von Daten, die typischerweise elektronisch in einem Computersystem gespeichert wird. Sie dient dazu, große Mengen an Informationen effizient zu organisieren, zu speichern und abzurufen.

In der Programmierung ist eine `database` unerlässlich, um Daten zu verwalten, die Anwendungen zum Funktionieren benötigen.

# --questions--

## --text--

Was ist eine `database` im Kontext der Programmierung?

## --answers--

Eine strukturierte Sammlung von Daten, die elektronisch gespeichert wird.

---

Ein Werkzeug zur Gestaltung von Websites.

### --feedback--

Obwohl sie in der Webentwicklung wichtig ist, ist eine `database` kein Werkzeug zur Gestaltung von Websites, sondern dient der Datenspeicherung.

---

Eine Programmiersprache zur Erstellung von Anwendungen.

### --feedback--

Eine `database` ist keine Programmiersprache; sie ist ein System zum Speichern und Verwalten von Daten.

---

Eine Art Computervirus.

### --feedback--

Eine `database` ist eine sichere und strukturierte Methode zur Datenspeicherung, kein Computervirus.
