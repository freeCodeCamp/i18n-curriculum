# --description--

数组中使用的更复杂的方法之一是 `splice()` 和 `slice()` 方法。`splice()` 方法通过删除或替换数组中的元素来更改数组的内容。`slice()` 方法返回从 `begin` 到 `end`（不包括 `end`）选取的一部分数组的浅拷贝，作为一个新建的数组对象。原始数组不会被修改。

例如，要从 `characters` 数组中移除第二个元素，你可以使用以下代码：

```javascript
const characters = ['Harry', 'Ron', 'Hermione'];
characters.splice(1, 1);
console.log(characters); // Output: ['Harry', 'Hermione']
```

上述元素从 `characters` 数组中移除第二个元素。`splice()` 方法接受两个参数：要移除的元素索引和要移除的元素数量。


要创建一个包含 `character` 数组中第二个元素的新建数组，你可以使用以下代码：

```javascript
const characters = ['Harry', 'Ron', 'Hermione'];
const newCharacters = characters.slice(1, 2);
console.log(newCharacters); // Output: ['Ron']
```

上述代码使用 `characters` 数组中的第二个元素创建了一个新的数组 `newCharacters`。`slice()` 方法接受两个参数：开始切片的元素索引和结束切片的元素索引（不包括该索引）。
