# --description--

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

`animal`를 `favoriteAnimal()` 함수의 소괄호 안에 넣으면 JavaScript에 `favoriteAnimal` 함수에 어떤 값을 보낼 것임을 알리는 것입니다. 즉, animal은 미래의 어떤 값을 위한 자리 표시자일 뿐입니다. 그런데 어떤 값을 보내는 걸까요?

마지막 줄인 `favoriteAnimal('Goat')`은 `favoriteAnimal` 함수를 호출하면서 `'Goat'` 값을 그 함수 안에 전달하는 부분입니다. 여기서 `'Goat'`는 인수입니다. `favoriteAnimal` 함수에 "`'Goat'`를 `favoriteAnimal` 함수에 보내고 `'Goat'` 자리 표시자가 있는 곳마다 `'animal'`를 사용해 주세요"라고 말하는 것입니다. 매개변수를 사용해 주는 유연성 덕분에 어떤 동물이든 좋아하는 동물로 선언할 수 있습니다.

다음은 매개변수가 함수에 어떻게 전달되고, 함수에서 값이 어떻게 반환되는지 시각화하는 데 도움이 되는 도표입니다.

<img src="https://cdn.statically.io/gh/TheOdinProject/curriculum/c53dd9a12f0c9afde0d9229f82a176170f12e120/foundations/javascript_basics/function_basics/imgs/00.png" alt="함수 자체의 소괄호 사이에 있는 값은 매개변수라고 하고, 함수가 사용될 때 전달되는 값은 인수라고 설명하는 화살표가 있는 함수 설명" style="width:95%;height:auto;">

`favoriteAnimal()` 안에서 `console.log()`를 `'Goat'` 인수와 함께 호출하면 `"Goat is my favorite animal!"` 문자열의 반환값이 콘솔에 출력된다는 점을 기억하세요. 함수 호출 `favoriteAnimal('Goat')`를 다른 함수 호출 `log()`의 인수로 전달하는 것입니다.
