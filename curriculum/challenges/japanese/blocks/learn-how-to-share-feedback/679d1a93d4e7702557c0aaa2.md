# --explanation--

`Past Simple` の疑問文では、助動詞 `did` がすでに過去形を表しているため、主動詞は基本形のままです。例えば：

`What time did you arrive?` （`did you arrived` ではありません）  

`Coding` は形容詞としてソフトウェアを書くことに関連するものを指します。`coding issue` はプログラムの機能に影響を与えるコードの問題を意味します。例えば：

`The development team is working on fixing a coding issue that caused system crashes.` - これはコードの問題がシステムに影響を与えていたことを意味します。
