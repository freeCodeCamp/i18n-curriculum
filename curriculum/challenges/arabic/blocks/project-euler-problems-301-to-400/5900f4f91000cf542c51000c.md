# --description--

على القطع المكافئ $y = \frac{x^2}{k}$، يُختار ثلاث نقاط $A(a, \frac{a^2}{k})$، $B(b, \frac{b^2}{k})$ و $C(c, \frac{c^2}{k})$.

لتكن $F(K, X)$ عدد الرباعيات الصحيحة $(k, a, b, c)$ بحيث يكون على الأقل زاوية واحدة في المثلث $ABC$ مقدارها 45°، مع $1 ≤ k ≤ K$ و $-X ≤ a < b < c ≤ X$.

على سبيل المثال، $F(1, 10) = 41$ و $F(10, 100) = 12\\,492$.

احسب $F({10}^6, {10}^9)$.
