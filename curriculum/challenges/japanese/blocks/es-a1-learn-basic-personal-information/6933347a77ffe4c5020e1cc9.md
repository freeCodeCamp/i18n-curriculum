# --description--

このタスクでは、エステバンの出身国を知ったときのアンヘラの反応を聞きます。以下の表現に注目してください。

- `Ah` は自然な反応音で、理解や気づきを示します。

- `Muy bien` は肯定的な反応で、承認や満足を表します。
