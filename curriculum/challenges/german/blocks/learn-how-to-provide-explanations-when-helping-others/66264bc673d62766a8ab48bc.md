# --description--

Das Wort `which` wird verwendet, um nach spezifischen Informationen aus einer Menge bekannter Optionen zu fragen. Zum Beispiel, wenn jemand `Which book did you read?` fragt. Sie möchten das bestimmte Buch aus mehreren möglichen wissen.

In einem technischen Kontext bedeutet die Frage `Which software are you using?`, dass nach einem bestimmten Software-Namen aus bekannten Optionen gefragt wird.
