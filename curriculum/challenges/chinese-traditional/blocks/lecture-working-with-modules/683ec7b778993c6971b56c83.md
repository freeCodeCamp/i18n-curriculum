# --description--

在軟體開發中，函式庫就像是開發者的工具箱。

函式庫提供你預先撰寫且可重複使用的程式碼，例如函式（程式）、類別和資料結構，讓你不必從頭實作程式碼的每一個零件，並能在你的專案中使用。

Python 擁有龐大的標準程式庫，包含許多不同的內建模組。它們都是針對你作為程式設計師每天會面臨的許多問題和任務所提供的標準化、經過嚴格審核的解決方案，例如：

* 與作業系統互動。
    
* 處理檔案。
    
* 網路。
    
* 處理日期和時間。
    
* 執行數學操作。
    
* 使用正規表達式。
    
* 測試和偵錯你的程式碼。
    
* 還有更多！
    

一些常見的內建模組範例有 `math`、`random`、`re`（「regular expressions」的縮寫）和 `datetime`。

`math` 模組具有用於執行更複雜數學操作的有用函式（程式）。

`random` 模組有助於產生隨機數。

`re` 模組用於處理正規表達式。

而 `datetime` 模組對於在 Python 中處理日期和時間非常有幫助。

但是你如何存取這些內建模組中定義的變數、常數、函式（程式）和類別？

你使用 `import` 述語。這些述語讓你將模組匯入你的 Python 腳本中。匯入述語通常寫在檔案的頂端。此外，你可以根據你的需求自訂它們。首先，你使用 `import` 述語，接著是模組的名稱：

```python
import module_name
```

假設你想要匯入 `math` 模組。在這種情況下，你會在你的檔案頂部寫下這段程式碼：

```python
import math
```

然後，如果你需要在你的 Python 腳本中呼叫該模組的某個方法，你會使用點記法，先寫模組名稱，接著寫方法名稱：

```python
module_name.method_name() 
```

例如，要取得 36 的平方根，你會寫 `math`，接著是一個點，再寫 `sqrt`，平方根的縮寫，並在小括弧內傳入任何必要的引數。在這個例子中，我們只需要傳入我們想要取得平方根的數字：

```python
math.sqrt(36)
```

這是最基本版本的匯入述語，但還有其他替代方案。

如果你需要以不同名稱（也稱為「別名」）匯入模組，可以使用此語法，在匯入述語的結尾使用 `as`，後面接別名：

```python
import module_name as module_alias
```

這通常用於縮短冗長的模組名稱，或避免命名衝突。

例如，若要在你的程式碼中將 `math` 模組指涉為 `m`，你可以為它指定一個別名，如下所示：

```python
import math as m 
```

然後，你可以使用別名來存取該模組的元素：

```python
m.sqrt(36)
```

但有時候你不需要從模組匯入所有東西。也許你只需要一兩個特定的函式（程式）或類別。Python 在這種情況下正好有你需要的功能。

現在的匯入述語以 `from` 開頭，接著是模組名稱，然後是 `import` 關鍵字，接著是你想要匯入的元素名稱：

```python
from module_name import name1, name2
```

然後，你可以在你的 Python 腳本中使用這些名稱，而不需要模組前置式。

如果你想為這些名稱指定別名，可以在每個名稱後使用 `as` 關鍵字，接著是你想使用的別名：

```python
from module_name import name1 as alias1, name2 as alias2
```

假設你只想從 `math` 模組匯入徑度、正弦和餘弦函式（程式）。你會寫：

```python
from math import radians, sin, cos
```

現在你可以直接在你的程式碼中呼叫這些函式，而不需要以 `math` 模組作為前置式。

這裡有一個更詳細的範例：

要找出最初以度數表示的特定角度的正弦和餘弦，我們可以呼叫 `radians` 函式將其轉換為徑度，然後呼叫正弦和餘弦函式，傳入以徑度表示的角度：

```python
from math import radians, sin, cos

angle_degrees = 40
angle_radians = radians(angle_degrees)

sine_value = sin(angle_radians)
cos_value = cos(angle_radians)

print(sine_value) # 0.6427876096865393
print(cos_value)  # 0.766044443118978
```

注意我們是直接呼叫函式（程式），而沒有使用模組名稱作為前置式。這是因為我們使用這種替代語法匯入了函式（程式）。

這很有幫助，但如果你已經在 Python 腳本中定義了相同名稱的函式或變數，可能會導致命名衝突。因此，在選擇你想使用的匯入述語型別時，這是需要注意的事項。

最後，我們找到這個以星號結尾的匯入述語。星號告訴 Python 你想要匯入該模組中的所有內容，但你想要匯入它，使你不需要使用該模組名稱作為前置式。

```python
from module_name import *
```

例如，如果你在匯入 `math` 模組時這麼做，你將能夠呼叫該模組中定義的任何函式，而無需指定模組名稱作為前置式。以下是一些範例：

```python
from math import *
print(sqrt(36))  # 6.0
print(pow(5, 2)) # 25.0
print(exp(1))    # 2.718281828459045
```

然而，這通常不被建議，因為它可能導致命名空間衝突，並且使得難以知道某些名稱的來源。

匯入述語對於函式、類別、常數、變數以及模組中定義的任何其他元素的運作方式完全相同。

這裡是一個來自 `math` 模組的常數範例，數字為 `pi`：

```python
import math
print(math.pi)
```

這裡是一個來自 `datetime` 模組的類別範例。我們創建一個表述 1959 年 7 月 15 日的日期物件。然後，我們將該 `date` 物件指定給一個變數，並使用點記法分別存取日、月和年：

```python
import datetime
birthday = datetime.date(1959, 7, 15)
print(birthday.day)    # 15
print(birthday.month)  # 7
print(birthday.year)   # 1959
```

你可以在該模組的官方 Python 文件中找到更多關於模組內容的資訊。

很好。既然你對模組有更多了解，你也應該知道這個在 Python 腳本中非常重要的慣用法，因為它們彼此密切相關：

```python
if __name__ == '__main__': 
    # Code
```

`__name__` 是 Python 中一個特殊的內建變數。

當 Python 檔案被直接執行時，Python 將此變數的值設定為字串 `"__main__"`。

但是如果 Python 檔案被匯入為另一個 Python 腳本的模組，則 `__name__` 變數的值會被設定為該模組的名稱（通常是沒有 `.py` 副檔名的檔名）。

這就是為什麼你經常會在 Python 腳本中看到這個條件。它包含你只想在 Python 腳本作為主程式執行時才執行的程式碼：

```python
if __name__ == '__main__': 
    # Code
```

但如果腳本是以模組方式匯入，該區塊內的程式碼不會執行。

這很有幫助，因為它允許 Python 腳本有兩種用途。它們可以直接執行以執行主要邏輯，或者可以匯入到另一個模組中而不執行主要邏輯。
