# --instructions--

Erstellen Sie eine Funktion namens `calculate()` in `mean_var_std.py`, die Numpy verwendet, um den Mittelwert, die Varianz, die Standardabweichung, das Maximum, das Minimum und die Summe der Zeilen, Spalten und Elemente in einer 3 x 3 Matrix auszugeben.

Die Eingabe der Funktion sollte eine Liste mit 9 Ziffern sein. Die Funktion soll die Liste in ein 3 x 3 Numpy-Array umwandeln und dann ein Wörterbuch zurückgeben, das den Mittelwert, die Varianz, die Standardabweichung, das Maximum, das Minimum und die Summe entlang beider Achsen und für die abgeflachte Matrix enthält.

Das zurückgegebene Wörterbuch sollte folgendes Format haben:

```py
{
  'mean': [axis1, axis2, flattened],
  'variance': [axis1, axis2, flattened],
  'standard deviation': [axis1, axis2, flattened],
  'max': [axis1, axis2, flattened],
  'min': [axis1, axis2, flattened],
  'sum': [axis1, axis2, flattened]
}
```

Wenn eine Liste mit weniger als 9 Elementen an die Funktion übergeben wird, soll sie eine `ValueError`-Ausnahme mit der Meldung „List must contain nine numbers.“ auslösen. Die Werte im zurückgegebenen Wörterbuch sollten Listen sein und keine Numpy-Arrays.

Zum Beispiel sollte `calculate([0,1,2,3,4,5,6,7,8])` Folgendes zurückgeben:

```py
{
  'mean': [[3.0, 4.0, 5.0], [1.0, 4.0, 7.0], 4.0],
  'variance': [[6.0, 6.0, 6.0], [0.6666666666666666, 0.6666666666666666, 0.6666666666666666], 6.666666666666667],
  'standard deviation': [[2.449489742783178, 2.449489742783178, 2.449489742783178], [0.816496580927726, 0.816496580927726, 0.816496580927726], 2.581988897471611],
  'max': [[6, 7, 8], [2, 5, 8], 8],
  'min': [[0, 1, 2], [0, 3, 6], 0],
  'sum': [[9, 12, 15], [3, 12, 21], 36]
}
```
