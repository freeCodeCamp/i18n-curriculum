# --description--

`Store information` は、特定の場所、通常は変数にデータを保持または保存することを意味します。変数はコード内のコンテナのようなもので、数字、テキスト、その他の種類のデータなどの情報を保持します。変数は名前付きの保存場所と考えられ、コードの後で`stored information` にアクセスできます。

例えば、`int age = 30;` ここで、変数`age` は数字の30を保存しています。

`like` という言葉は、例を示したり、似たものを比較したりするための前置詞として使われます。

例えば、`I love fruits like apples and oranges.` と言う場合、あなたは`like` を使って好きな果物の種類の例を示しており、そのリストにもっと果物がある可能性を示唆しています。
