# --description--

Sie können sich Variablen als „Speichercontainer“ für Daten in Ihrem Code vorstellen.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/variables-and-operators/top-learn-variables-and-operators.png" alt='das Konzept von Variablen im Programmieren als Boxen mit der Bezeichnung „birthday“, die Daten speichern und zugänglich machen, speziell ein Datum „16-09-2003“'>

Eine Variable ist ein „benannter Speicher“ für Daten. Sie können Variablen verwenden, um Werte, Besucher und andere Daten zu speichern.

Um eine Variable in JavaScript zu erstellen, verwenden Sie das Schlüsselwort `let`.

Die folgende Anweisung erstellt (anders gesagt: deklariert) eine Variable mit dem Namen „message“:

```js
let message;
```

Jetzt können Sie mit dem Zuweisungsoperator `=` Daten darin speichern:

```js
let message;

// store the string 'Hello' in the variable named message
message = 'Hello'; 
```

Um es kurz zu machen, können Sie die Variablendeklaration und die Zuweisung in einer einzigen Zeile kombinieren:

```js
let message = 'Hello';
```
