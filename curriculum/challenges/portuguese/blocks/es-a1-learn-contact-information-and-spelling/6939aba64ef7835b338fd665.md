# --description--

Você se lembra que os verbos são conjugados de acordo com a pessoa a que se referem? Se não, agora você sabe!

Algumas tarefas atrás, você aprendeu como usar o verbo `poder` para perguntar se alguém foi capaz de fazer algo para você.

Agora você vai aprender como conjugar esse mesmo verbo para perguntar se **você** pode fazer algo. Você só precisa usar a estrutura:

`Puedo` + verbo no infinitivo + `...?`

Vamos ver alguns exemplos:

- `¿Puedo usar la computadora?` - Posso usar o computador?

- `¿Puedo usar mi apodo?` - Posso usar meu apelido?

Em ambos os exemplos, você está pedindo permissão para usar o computador e seu apelido, respectivamente.

# --explanation--

Basti diz `¿Puedo usar mi apodo?`.

Aqui, `puedo` é a forma de primeira pessoa do singular do verbo `poder`, usada quando alguém pede permissão.

A estrutura segue o padrão:

`Posso` + verbo no infinitivo + `...?`

Neste caso:

- `Usar` significa usar.

- `Mi apodo` refere-se ao apelido dele.

Então Basti está perguntando se ele tem permissão para usar seu apelido.
