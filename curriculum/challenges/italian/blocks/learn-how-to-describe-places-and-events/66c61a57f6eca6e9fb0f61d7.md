# --explanation--

`Built-in` indica funzionalità o strumenti che sono inclusi come parte di un programma software o dispositivo fin dall'inizio, senza bisogno di essere aggiunti o installati separatamente.

Quando qualcosa è `built-in`, è già presente e pronto all'uso. Per esempio:

- `This laptop has a built-in camera.` - La fotocamera è inclusa come parte del portatile.

- `The software comes with built-in security features.` - Le funzionalità di sicurezza sono già incluse nel software.
