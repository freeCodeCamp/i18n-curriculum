# --interactive--

在 JavaScript 中，字串是一連串用來表述文字資料的字元。字串是語言中原始資料型別之一，與數字、布林值、`null` 和 `undefined` 一起。

要在 JavaScript 中創建字串，你可以使用單引號（`'`）或雙引號（`"`）。

以下是建立兩個儲存字串值的變數的範例：

:::interactive_editor

```js
let singleQuotes = 'This is a string';
console.log(singleQuotes);
let doubleQuotes = "This is also a string";
console.log(doubleQuotes);
```

:::

即使你可以使用單引號或雙引號來創建字串，也必須保持一致。如果字串以單引號開始，則也必須以單引號結束。

雙引號也適用相同規則。以下範例會發出錯誤，因為它以雙引號開始，卻以單引號結束：

```js
const improperStr = "Do not do this';
```

另一件需要考慮的事情是字串是不可變的。在程式設計中，不變性表示一旦某物被創建，就無法更改。因此，當你創建一個字串時，你無法直接更改它的字元。相反地，如果你想做更改，你會創建一個新的字串。

以下是將新的字串指定給 `developer` 變數的範例：

:::interactive_editor

```js
let developer = "Jessica";
console.log(developer);
developer = "Quincy";
console.log(developer);
```

:::

由於字串是不可變的，我們無法直接更新第一個字串。這就是為什麼我們要將新的字串指定給 `developer` 變數。

字串是程式設計中重要的部分，在未來的課程中，你將學習操作字串的進階技巧，並發揮其全部潛力，以創建動態且互動的應用程式。
