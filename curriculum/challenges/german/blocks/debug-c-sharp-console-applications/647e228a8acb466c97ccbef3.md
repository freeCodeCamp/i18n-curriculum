## --text--

Wann sollte eine Methode eine Ausnahme werfen?

## --answers--

Eine Methode sollte eine Ausnahme werfen, wenn die Methode ihren vorgesehenen Zweck erfüllen kann.

---

Eine Methode sollte eine Ausnahme werfen, wenn die Methode ihren vorgesehenen Zweck nicht erfüllen kann.

---

Eine Methode sollte eine Ausnahme werfen, wenn ein Ausnahmetyp dem vorgesehenen Zweck der Methode entspricht.
