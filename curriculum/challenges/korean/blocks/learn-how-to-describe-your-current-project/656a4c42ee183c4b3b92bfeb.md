# --description--

`Busy`는 할 일이 많거나 일이나 활동에 적극적으로 참여하고 있다는 뜻입니다. 예를 들어:

`She's busy with her project.` - 이것은 그녀가 프로젝트에 시간과 노력을 쏟고 있어서 자유 시간이 많지 않다는 뜻입니다.

`By the way`는 주제를 바꾸거나 이전에 말한 내용과 직접적으로 연결되지 않은 추가 정보를 덧붙일 때 쓰는 표현입니다. 일상 대화에서 자주 사용됩니다. 예를 들어:

`By the way, have you seen the new blog feature?` - 이것은 화자가 방금 떠오른 새로운 주제를 꺼내고 있다는 뜻입니다.
