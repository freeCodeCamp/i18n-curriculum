# --explanation--

`Database` означає систему, де зберігаються та організовуються дані.

`Activity` означає дії або процеси, що відбуваються в базі даних.

- `The database contains customer information and order details.` — це означає, що база даних зберігає важливі бізнес-дані.

- `There was unusual activity in the database last night.` — це означає, що в базі даних сталася якась несподівана подія.

Разом `database activity` означає всі дії, зміни або процеси, що відбуваються в системі бази даних.
