# --description--

アルファベットの26文字から異なる3文字を取り出して、長さ3の文字列を作ることができます。

例としては 'abc'、'hat'、'zyx' があります。

これら3つの例を調べると、'abc' では左隣の文字より辞書順で後に来る文字が2文字あります。

'hat' では左隣の文字より辞書順で後に来る文字がちょうど1文字あります。'zyx' では左隣の文字より辞書順で後に来る文字は0文字です。

長さ3の文字列のうち、左隣の文字より辞書順で後に来る文字がちょうど1文字であるものは全部で10400個あります。

ここで、アルファベットから異なる $n ≤ 26$ 文字を使った文字列を考えます。

任意の $n$ に対して、$p(n)$ は長さ $n$ の文字列のうち、左隣の文字より辞書順で後に来る文字がちょうど1文字であるものの数です。

$p(n)$ の最大値はいくつでしょうか？
