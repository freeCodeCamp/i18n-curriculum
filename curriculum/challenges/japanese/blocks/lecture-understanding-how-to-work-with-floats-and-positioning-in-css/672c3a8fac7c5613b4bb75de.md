# --interactive--

絶対位置指定を使うと、要素を通常のドキュメントの流れから外し、他の要素とは独立して動作させることができます。要素が絶対位置指定されると、その要素は独自のレイヤーに配置され、レイアウト内の他のすべてのものから完全に分離されます。これにより、モーダル、ツールチップ、ドロップダウンメニューなど、ページ上の他の要素と重なり合う浮動的なUI機能を作成するのに便利です。

デフォルトでは、絶対位置指定された要素は最も近い位置指定された祖先要素を基準に配置されます。位置指定された祖先が見つからない場合、その要素は初期包含ブロック（通常はブラウザのビューポート）を基準に配置されます。`top`、`bottom`、`left`、`right` プロパティを使って、絶対位置指定された要素を包含要素の端からどれだけ離すかを指定して移動できます。

たとえば、`top: 30px` と `left: 30px` を設定すると、要素は包含ブロックの上端と左端から `30px` 離れた位置に移動します。以下はCSSで絶対位置指定を要素に適用する例です。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="positioned">Absolutely Positioned</div>
```

```css
body {
  background-color: #eeeeee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

このコードを適用すると、要素は通常のドキュメントの流れから外され、包含ブロックの上端と左端から `30px` 離れた位置に配置されます。

絶対位置指定は、オーバーレイやモーダルウィンドウのように、他のコンテンツの上に浮かせたい要素を作るのに特に役立ちます。ただし、要素がドキュメントの流れから外れるため、適切に扱わないと隙間ができたり、他の要素が重なり合ってしまうことがあります。
