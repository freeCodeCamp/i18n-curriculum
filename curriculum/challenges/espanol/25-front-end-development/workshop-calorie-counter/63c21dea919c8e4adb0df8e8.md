# --description--

En el proyecto de Juego de Roles, aprendiste cómo establecer el comportamiento de un botón editando su propiedad `onclick`. También puedes editar el comportamiento de un elemento agregando un escucha de eventos.

El siguiente ejemplo utiliza el método `addEventListener` para agregar un evento de clic a un botón. Cuando se hace clic en el botón, se llama a la función `printName`.

```html
<button class="btn">Print name</button>
```

```js
const button = document.querySelector('.btn');
function printName() {
  console.log("Jessica");
}
button.addEventListener('click', printName);
```

El método `addEventListener` recibe dos argumentos. El primero es el evento al que escuchar. (Ej. `'click'`) El segundo es la función de llamada, o la función que se ejecuta cuando el evento es activado.

Llama al método `.addEventListener()` en el `addEntryButton`. Pasa la cadena `"click"` como el primer argumento y la función `addEntry` como el segundo argumento.

Nota que no deberías *llamar* a `addEntry`, sino pasar la variable (o <dfn>referencia de función</dfn>) directamente.
