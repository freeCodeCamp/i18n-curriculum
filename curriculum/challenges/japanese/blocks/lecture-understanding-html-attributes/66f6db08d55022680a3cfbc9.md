# --interactive--

HTMLはHypertext Markup Languageの略で、ウェブページを作成するためのマークアップ言語です。ウェブサイトを訪れて、段落、見出し、リンク、画像、動画などのコンテンツを目にする場合、それはHTMLによるものです。

ここにHTML要素を使った小さな例があります。エディター内のテキストを編集して、プレビューウィンドウで変化を確認してみてください。

:::interactive_editor

```html
<h1>Main heading element</h1>

<p>I am a paragraph element.</p>
```

:::

HTMLは要素を使ってウェブページのコンテンツと構造を表します。ほとんどの要素は開始タグと終了タグを持ちます。これらのタグは開始タグと終了タグとも呼ばれます。2つのタグの間にはコンテンツがあり、このコンテンツはテキストや他のHTML要素で構成されます。

こちらは段落要素の別の例です。エディター内のテキストを`I love coding!`に変更して、プレビューウィンドウで結果を確認してください。

:::interactive_editor

```html
<p>I am a paragraph element.</p>
```

:::

開始タグと終了タグはどちらも左山括弧（`<`）で始まり、右山括弧（`>`）で終わります。タグ名はこれらの山括弧の間に配置されます。HTMLのタグ名は大文字・小文字を区別しませんが、小文字で書くのが広く受け入れられている慣例でありベストプラクティスです。

ここでは開始タグと終了タグの段落タグを詳しく見てみましょう：

```html
<p>
```

```html
</p>
```

開始タグと終了タグを区別するのは、終了タグの左山括弧の直後にあるスラッシュ（`/`）です。終了タグを持たないHTML要素もあります。これらはvoid要素と呼ばれます。

こちらはvoid要素である画像要素の例です：

```html
<img>
```

この画像要素には終了タグがなく、コンテンツもありません。void要素はコンテンツを持たず、開始タグのみを持ちます。

時にはvoid要素で、`/`の前に`>`を使うことがあります：

```html
<img />
```

多くのコードフォーマッター（Prettierなど）はvoid要素に`/`を含めることを選びますが、HTML仕様では`/`の存在は「開始タグを自己終了タグとしてマークするものではなく、不要であり何の効果もありません」とされています。

実際の開発では両方の形式を見ることがあるので、両方に慣れておくことが重要です。

画像を表示したい場合は、画像要素の中にいくつかの属性を含める必要があります。属性とはHTML要素の動作を調整するための特別な値です。

こちらは`src`属性を持つ画像要素の例です。`src`属性の値を`"https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"`に更新すると、2匹の猫が平和に眠っている画像に変わります。

:::interactive_editor

```html
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/running-cats.jpg" />
```

:::

`src`属性は画像の場所を指定するために使われます。画像要素には、`alt`属性という別の属性を含めることがベストプラクティスです。`alt`属性は画像の短く説明的なテキストを提供するために使われます。

こちらは`src`と`alt`属性を持つ画像要素の例です。`src`の値を`"https://.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"`に更新して画像を壊してみてください。画像が消えて`alt`のテキストだけが表示されるのがわかります。

:::interactive_editor

```html
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" alt="Two tabby kittens sleeping together on a couch." />
```

:::

それでは、HTMLだけでウェブサイトを作るのに十分か気になるかもしれません。答えは「場合によります」です。テキストと画像だけを表示する小さな練習プロジェクトなら、HTMLだけで十分かもしれません。しかし、現代的でプロフェッショナルなウェブサイトを作る場合は、HTML、CSS、JavaScriptが必要です。

HTMLはコンテンツと構造のためのものです。CSSはスタイリングのためのものです。JavaScriptはウェブページにインタラクティブ性を加えるためのものです。これらを建物に例えるとわかりやすいです。

HTMLは壁を構成するブロック、コンクリート、鉄筋を表します。建物を強固にする基礎です。CSSは家を美しく見せる内装と外装のデザインを表します。JavaScriptは水道や電気のシステムを表し、水や電気の途切れないアクセスを保証します。
