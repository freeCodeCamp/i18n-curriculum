## --text--

Як імпортувати CSV-файл `data.csv` і зберегти його у таблиці даних за допомогою модуля Pandas?

## --answers--

```python
import pandas as pd
df = pd.csv("data.csv")
```


```python
import pandas as pd

```

```python
import pandas as pd
df = pd.csv_reader("data.csv")
```
