# --explanation--

`Soy` 意思是“我是”，来源于动词 `ser`。当描述你是谁或你做什么时使用。例如：

- `Soy ingeniera.` – 我是工程师。（女性）

- `Soy ingeniero.` – 我是工程师。（男性）

- `Soy profesor.` – 我是老师。（男性）

- `Soy profesora.` – 我是老师。（女性）
