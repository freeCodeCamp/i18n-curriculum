# --interactive--

Метод `map` — це потужна і широко використовувана функція в JavaScript, яка працює з масивами. Вона призначена для створення нового масиву шляхом застосування заданої функції до кожного елемента початкового масиву.

Цей метод не змінює початковий масив, а натомість повертає новий масив, що містить результати застосування функції до кожного елемента.

Ось приклад використання методу `map` на масиві чисел:

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map((num) => num * 2);
console.log(numbers); // [1, 2, 3, 4, 5]
console.log(doubled); // [2, 4, 6, 8, 10]
```

:::

Щоб створити новий масив, де кожне число подвоєне, ми використовуємо метод `map`. Метод `map` приймає функцію зворотного виклику, яка виконується для кожного елемента масиву.

У цьому випадку кожне число в масиві буде помножене на `2`. Результатом буде новий масив чисел `2,4,6,8,10`.

Функція зворотного виклику може приймати до трьох аргументів.

Перший аргумент — це поточний елемент, який обробляється.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element) => {
  console.log("Element:", element);
  return element * 2;
});
```

:::

Другий аргумент — це індекс поточного елемента, який обробляється.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index) => {
  console.log("Element:", element);
  console.log("Index:", index);
  return element * 2;
});
```

:::

Третій аргумент — це масив, на якому викликається `map`.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index, array) => {
  console.log("Element:", element);
  console.log("Index:", index);
  console.log("Array:", array);
  return element * 2;
});
```

:::

Розуміння та ефективне використання методу `map` може значно покращити вашу здатність працювати з масивами в JavaScript. У майбутніх уроках ми детальніше розглянемо більш просунуті способи використання `map` і дослідимо, як він може стати потужним інструментом для створення динамічних і ефективних програм.
