# --explanation--

Jake puede decir `It's possible` porque piensa que hay una posibilidad de que el correo electrónico sospechoso haya sido parte de un ataque de phishing.

`Impossible` se forma añadiendo el prefijo `im-` (que significa "no" o "lo opuesto de") a la palabra base `possible`. Por ejemplo:

- `It's possible.` - Algo puede suceder.

- `It's impossible.` - No puede suceder.
