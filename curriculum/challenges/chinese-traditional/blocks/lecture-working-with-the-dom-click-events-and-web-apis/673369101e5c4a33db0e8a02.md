# --interactive--

在上一課中，你學會了如何使用 `addEventListener()` 方法來處理 `"click"` 和 `"input"` 事件。在本課中，你將學習如何使用 `removeEventListener()` 方法來移除事件監聽器。

`removeEventListener()` 方法用於移除先前使用 `addEventListener()` 方法添加到元素的事件監聽器。當你想停止監聽元素上的特定事件時，這非常有用。

以下是 `removeEventListener()` 方法的基本語法：

```js
element.removeEventListener("event", listener);
```

就像 `addEventListener()` 方法一樣，`removeEventListener()` 方法接受兩個引數：你想要移除的事件以及先前添加的監聽器函式（函式）。

`removeEventListener()` 方法還可以傳入第三個額外的選用引數。這個引數可以是 `options` 或 `useCapture`。

`options` 引數是一個物件，用來指定事件監聽器的選項，例如事件監聽器是否應該是被動的或只執行一次。 

`useCapture` 引數是一個布林值，用來指定是否應該在事件傳播階段捕獲該事件。

大多數時間，你只需要將事件和監聽器引數傳遞給 `removeEventListener()` 方法。

如果我們想在灰色和藍色之間切換背景顏色，則可以為此使用事件監聽器。

我們首先需要存取 `button` 元素以及 `body` 元素：

```js
const bodyEl = document.querySelector("body");
const btn = document.getElementById("btn");
```

然後我們需要創建負責在灰色和藍色之間切換的函式（程式）：

```js
let isBgColorGrey = true;

function toggleBgColor() {
  bodyEl.style.backgroundColor = isBgColorGrey ? "blue" : "grey";
  isBgColorGrey = !isBgColorGrey;
}
```

我們使用布林值變數 `isBgColorGrey` 來追蹤目前的背景顏色。如果背景顏色是灰色，則將它改為藍色，反之亦然。

接著我們需要為 `button` 元素添加事件監聽器，以便在按鈕被點擊時呼叫 `toggleBgColor` 函式（程式）：

```js
btn.addEventListener("click", toggleBgColor);
```

這是目前為止的完整範例：

每次點擊按鈕時，`body` 的背景顏色會在灰色和藍色之間切換。

我們可以更新 HTML 以包含一個段落元素，當暫留時將用來移除事件監聽器：

```html
<link rel="stylesheet" href="styles.css" />
<h1>Event Listener examples</h1>
<p id="para">MouseOver this text to remove the event listener</p>
<button id="btn">Change background color</button>
<script src="index.js"></script>
```

要在段落被暫留時移除事件監聽器，我們可以為段落元素添加事件監聽器。我們使用 `mouseover` 事件並傳入一個函式（程式），該函式會從 `button` 元素移除事件監聽器：

```js
const para = document.getElementById("para");
const btn = document.getElementById("btn");

para.addEventListener("mouseover", () => {
  btn.removeEventListener("click", toggleBgColor);
});
```

當滑鼠暫留在段落上時，按鈕點擊事件的事件監聽器會被移除，按鈕被點擊時背景顏色將不再改變。
