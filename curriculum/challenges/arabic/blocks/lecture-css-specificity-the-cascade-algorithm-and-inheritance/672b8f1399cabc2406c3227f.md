# --interactive--

يُستخدم المفتاح `!important` في CSS لإعطاء قاعدة النمط أعلى أولوية، مما يسمح لها بتجاوز أي إعلانات أخرى لخاصية معينة.

عند استخدامه، يُجبر المتصفح على تطبيق النمط المحدد بغض النظر عن خصوصية المحددات الأخرى.

لنفترض أن لديك عنصر فقرة HTML مع أنماط مضمَّنة مثل هذا:

:::interactive_editor

```html
<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

:::

في هذا المثال، يحتوي عنصر الفقرة على لون خلفية محدد بـ `lightblue` ولون نص محدد بـ `black`.

يطبق ملف CSS الأنماط التالية على عنصر الفقرة:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black;
  color: white;
}
```

:::

بما أن الأنماط المضمَّنة لها أولوية أعلى من محددات الصنف، المعرف، والنوع، فلن يتم تطبيق لون الخلفية الأسود ولون النص الأبيض على عنصر الفقرة هذا.

لتجاوز تلك الأنماط المضمَّنة، يمكنك استخدام المفتاح `!important` هكذا:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black !important;
  color: white !important;
}
```

:::

يُستخدم المفتاح `!important` بعد قيمة CSS وقبل الفاصلة المنقوطة.

الآن سيحصل عنصر الفقرة على ألوان الأسود والأبيض لأن الأنماط المضمَّنة تم تجاوزها باستخدام المفتاح `!important`.

يُستخدم المفتاح `!important` في CSS لإعطاء قاعدة النمط أعلى أولوية، متجاوزًا بفعالية الإعلانات الأخرى، بما في ذلك تلك التي لها خصوصية أعلى والأنماط المضمَّنة.

مع ذلك، لا يغير المفتاح `!important` خصوصية محدد CSS نفسه. بل يضمن فقط تطبيق القاعدة التي تحتوي على `!important` حتى لو كانت هناك قواعد أخرى متضاربة لها خصوصية أعلى.

حالة استخدام مناسبة أخرى للمفتاح `!important` هي تجاوز الأنماط من مكتبات أو أُطُر عمل خارجية عندما لا تملك السيطرة على CSS الأصلي.

لكن الإفراط في استخدام المفتاح `!important` قد يؤدي إلى صعوبات في صيانة وتصحيح الأخطاء في CSS، لأنه يكسر التتابع الطبيعي للأنماط وقد يسبب نتائج غير مقصودة.

لذا، من الأفضل استخدام المفتاح `!important` باعتدال.

    #unique {
      color: purple;
    }
  </style>

  <p id="unique" class="highlight">This text</p>
