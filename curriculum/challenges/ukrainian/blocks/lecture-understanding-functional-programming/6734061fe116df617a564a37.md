# --interactive--

Каррінг — це техніка, коли ми перетворюємо функцію, яка приймає кілька аргументів, у послідовність функцій, кожна з яких приймає один аргумент.

Почнемо з базового прикладу. Уявімо, що у нас є функція, яка додає два числа:

:::interactive_editor

```js
function add(a, b) {
  return a + b;
}

console.log(add(3, 4)); // 7
```

:::

Це функція, яка приймає два аргументи і повертає їх суму. Тепер подивімося, як ми можемо зробити каррінг цієї функції:

:::interactive_editor

```js
function curriedAdd(a) {
  return function(b) {
    return a + b;
  }
}

console.log(curriedAdd(3)(4)); // 7
```

:::

У цьому коді з каррінгом замість того, щоб приймати два аргументи одночасно, ми маємо функцію, яка приймає перший аргумент і повертає іншу функцію. Ця повернена функція потім приймає другий аргумент і виконує додавання. Ми викликаємо її як `curriedAdd(3)(4)`, де кожна пара круглих дужок представляє виклик функції.

Але навіщо це робити?

Каррінг дозволяє нам легко створювати спеціальні функції. Наприклад, ми можемо створити функцію, яка завжди додає п’ять до будь-якого числа:

:::interactive_editor

```js
function curriedAdd(a) {
  return function(b) {
    return a + b;
  }
}

const addFive = curriedAdd(5);
console.log(addFive(10)); // 15
console.log(addFive(20)); // 25
```

:::

Тут `addFive` — це функція, яка завжди готова додати п’ять до будь-якого числа, яке ми їй передамо. Це простий приклад часткового застосування, коли ми фіксуємо певну кількість аргументів функції, отримуючи іншу функцію, яка приймає менше аргументів.

Хоча наші приклади зосереджені на функціях з двома аргументами, каррінг можна застосовувати до функцій з будь-якою кількістю аргументів.

Продовжуючи вивчати JavaScript, ви, ймовірно, знайдете багато випадків, коли каррінг допоможе зробити ваш код чистішим і виразнішим.

## --text--

Який із наведених прикладів є прикладом функції з каррінгом у JavaScript?

## --answers--

```js
function add(a, b, c) {
  return a + b + c;
}
```

### --feedback--

Це звичайна функція, яка приймає всі параметри одночасно.

---

```js
function add(a) {
  return function (b) {
    return function (c) {
      return a + b + c;
    };
  };
}
```

---

```js
function add(a, b, c) {
  return a() + b() + c();
}
```

### --feedback--

Ця функція приймає всі параметри одночасно і викликає їх як функції, що не відповідає визначенню функції з каррінгом.

---

```js
function add(a) {
  function inner(b) {
    return a + b;
  }
}
```

### --feedback--

Внутрішня функція ніколи не повертається, тому `add(2)` поверне `undefined`, а `add(2)(3)` викличе помилку.
