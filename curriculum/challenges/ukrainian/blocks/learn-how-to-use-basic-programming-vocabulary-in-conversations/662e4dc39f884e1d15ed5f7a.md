# --description--

Термін `to debug` є ключовим у програмуванні. Він означає знаходження та виправлення помилок або багів у комп’ютерній програмі. `bug` — це помилка або проблема в програмі, яка заважає їй працювати правильно.

`Debugging` є звичайною частиною роботи програміста.

Наприклад, `She spent the whole day debugging the code.` Це означає, що вона виправляла помилки в коді.
