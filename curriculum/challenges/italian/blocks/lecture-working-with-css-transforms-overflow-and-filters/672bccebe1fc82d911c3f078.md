# --interactive--

La proprietà CSS `filter` è uno strumento potente che ti permette di applicare effetti grafici agli elementi di una pagina web. È particolarmente utile per regolare la presentazione visiva di immagini, sfondi e persino testo senza modificare la risorsa originale.

La proprietà `filter` può essere usata per creare vari effetti, come sfocatura, spostamento del colore e regolazioni del contrasto. Vediamo come funziona la proprietà filter ed esploriamo alcuni esempi comuni. La sintassi base per una proprietà `filter` è semplice:

```css
selector {
  filter: function(amount);
}
```

Qui, `function` rappresenta l’effetto filtro specifico che vuoi applicare, e `amount` è tipicamente un valore che determina l’intensità dell’effetto. Ora vediamo alcune funzioni filtro comuni e i loro usi.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: blur(2px);
}
```

:::

La funzione `blur` applica una sfocatura gaussiana all’elemento, la quantità è specificata in pixel e rappresenta il raggio della sfocatura. Questa regola CSS applicherà una sfocatura di 2 pixel a tutte le immagini nella pagina. L’effetto `blur` può essere utile per creare profondità nel design o per oscurare parti di un’immagine.

La funzione `brightness` regola la luminosità dell’elemento. Un valore di `0%` renderà l’elemento completamente nero, mentre valori superiori a `100%` aumenteranno la luminosità.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: brightness(150%);
}
```

:::

Questa regola CSS aumenta la luminosità dell’elemento immagine di `50%`. Le regolazioni di luminosità possono essere usate per far risaltare le immagini o creare un effetto sbiadito.

La funzione `grayscale` converte l’elemento in scala di grigi. La quantità è definita come percentuale, dove `100%` è completamente in scala di grigi e `0%` lascia l’immagine invariata.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: grayscale(100%);
}
```

:::

Questa regola convertirà l’elemento immagine in scala di grigi completa. `grayscale` può essere usato per creare un aspetto vintage o per de-enfatizzare certi elementi in una pagina.

La funzione `sepia` applica una tonalità seppia all’elemento. Come la scala di grigi, usa un valore percentuale:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: sepia(80%);
}
```

:::

Questa regola applica un effetto seppia `80%` all’elemento immagine. L’effetto seppia è ottimo per creare un aspetto vintage o retrò.

La funzione `hue-rotate` applica una rotazione della tonalità all’elemento. Il valore è definito in gradi e rappresenta una rotazione intorno al cerchio dei colori.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: hue-rotate(90deg);
}
```

:::

Questa regola ruota la tonalità dell’elemento immagine di `90` gradi. La rotazione della tonalità può essere usata per creare effetti psichedelici o per regolare lo schema cromatico complessivo di un’immagine.

Uno degli aspetti più potenti della proprietà `filter` è la possibilità di combinare più effetti. Puoi applicare diversi filtri allo stesso elemento separandoli con spazi:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {  
  filter: contrast(150%) brightness(110%) sepia(30%);  
}
```

:::

Questa regola applica un contrasto aumentato, una luminosità leggermente aumentata e un sottile effetto seppia all’elemento immagine.

Combinando i filtri, puoi creare effetti visivi complessi e unici su misura per le esigenze del tuo design. La proprietà CSS filter è uno strumento versatile che permette una manipolazione visiva creativa degli elementi web.

Anche se abbiamo coperto alcune delle funzioni filtro più comuni, ce ne sono altre disponibili, come `contrast`, `invert` e `saturate`. Come per ogni funzionalità potente, è importante usare i filtri con attenzione per migliorare il design senza sovraccaricare gli utenti o compromettere l’accessibilità.
