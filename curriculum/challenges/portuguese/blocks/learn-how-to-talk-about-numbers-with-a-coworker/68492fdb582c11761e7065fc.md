### --feedback--

Esta frase de duas palavras usa um número cardinal para indicar as permissões legais concedidas para usar, possuir ou realizar algo.A segunda palavra está no plural.

# --explanation--

Lembre-se, **números cardinais** são usados para mostrar quantos de algo existem.Por exemplo, `one license`, `ten computers` e `fifty users`.

No diálogo, `fifty licenses` significa que a equipe atualmente tem 50 permissões para usar várias ferramentas de software.
