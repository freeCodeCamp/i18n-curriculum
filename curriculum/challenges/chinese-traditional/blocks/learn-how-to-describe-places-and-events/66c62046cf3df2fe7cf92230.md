# --explanation--

要找到答案，請注意字 `but`，它表示對比或例外。 

在 `but` 之後，Sarah 提到 `they're not visible`，接著是條件 `if the project isn't linked to Git.` 

這告訴你，除非專案已連接到 Git，否則工具會被隱藏。 

片語 `if the project isn't` 是一個關鍵標記，用來介紹 Git 工具不會出現的條件。

在 Sarah 的句子中，她使用 `they're not` 來說明 Git 工具不可見，並使用 `isn't` 來表示專案未連結到 Git。
