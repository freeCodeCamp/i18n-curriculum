# --interactive--

عندما ينقر المستخدم على زر أو يحدث تغيير في نموذج، يُعرف هذا بحدث. في برامجك، ستحتاج إلى طريقة للاستماع إلى هذه الأحداث والاستجابة لها.

تُستخدم طريقة `addEventListener()` للاستماع إلى الأحداث. تأخذ معلمتين: الحدث الذي تريد الاستماع إليه ودالة تُستدعى عند حدوث الحدث.

إليك الصيغة النحوية الأساسية:

```js
element.addEventListener("event", listener);
```

العنصر هو عنصر HTML الذي تراقب أحداثه، وevent يحدد نوع الحدث للاستماع إليه، مثل `"click"`.

المستمع هو كائن يتلقى الإشعار عند حدوث الحدث. في معظم الأحيان، سيكون دالة تعرفها للتعامل مع الحدث.

إليك مثالاً:

```js
element.addEventListener("click", () => {
  // code to run when the click event occurs
});
```

يمكنك أيضًا اختيار تمرير مرجع دالة هكذا:

```js
function handleClick() {
  // code to run when the click event occurs
}

element.addEventListener("click", handleClick);
```

أحيانًا قد ترغب في إنشاء دالة منفصلة للتعامل مع الحدث. هذا يجعل شفرتك أسهل في القراءة والصيانة.

يمكن أن تكون المعلمة listener أيضًا `null`، أو كائنًا يستخدم واجهة `EventListener`. تحدد واجهة `EventListener` طريقة واحدة تسمى `handleEvent()`. تُستدعى هذه الطريقة تلقائيًا كلما حدث الحدث الذي تستمع إليه. استخدام هذه الواجهة يسمح لنفس الكائن بالتعامل مع عدة أحداث إذا لزم الأمر.

في معظم الأحيان، لن تحتاج إلى استخدام هذه الواجهة، ويمكنك بدلاً من ذلك تمرير دالة تُستدعى عند حدوث الحدث.

الآن، لنلقِ نظرة على مثال لفهم كيفية عمل ذلك بشكل أفضل:

في هذا المثال، لدينا عنصر `button` بمعرف `btn`:

```html
<button id="btn">Show alert</button>
```

نريد الاستماع إلى حدث `"click"` على هذا الزر وعرض تنبيه عند النقر عليه.

أولًا نحتاج إلى الوصول إلى عنصر `button` هذا في شفرة JavaScript الخاصة بنا. ثم نضيف مستمع حدث للاستماع إلى حدث النقر وعرض تنبيه عند النقر على الزر:

:::interactive_editor

```html
<button id="btn">Show alert</button>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("btn");

btn.addEventListener("click", () => alert("You clicked the button"));
```

:::

الآن في كل مرة يُنقر فيها الزر، سيرى المستخدم رسالة تنبيه معروضة على الشاشة هكذا:

```md
You clicked the button
```

نوع آخر من الأحداث التي يمكنك الاستماع إليها هو حدث `"input"`. يُطلق هذا الحدث عندما يتغير قيمة عنصر الإدخال.

إليك مثالاً:

```html
<input type="text" id="input" placeholder="Type something" />
```

نريد الاستماع إلى حدث الإدخال على هذا العنصر وتسجيل قيمة الإدخال في وحدة التحكم في كل مرة يكتب فيها المستخدم شيئًا.

أولًا نحتاج إلى الوصول إلى عنصر `input` هذا في شفرة JavaScript الخاصة بنا. ثم نضيف مستمع حدث للاستماع إلى حدث الإدخال وتسجيل قيمة الإدخال في وحدة التحكم في كل مرة يكتب فيها المستخدم شيئًا:

:::interactive_editor

```html
<input type="text" id="input" placeholder="Type something" />
<script src="index.js"></script>
```

```js
const input = document.getElementById("input");

input.addEventListener("input", () => {
  console.log(input.value);
});
```

:::

هناك العديد من الأحداث الأخرى التي يمكنك الاستماع إليها باستخدام طريقة `addEventListener()`. بعض الأحداث الشائعة تشمل `mouseover`، `mouseout`، `keydown`، `keyup`، و`submit`.

في الدروس القادمة، سنغطي المزيد من الأحداث وكيفية استخدام طريقة `addEventListener()` للاستماع إليها.
