# --description--

To `make sure` se usa para expresar certeza o confirmación sobre algo. Por ejemplo:

`I check the list twice to make sure I have everything.` - Esto significa que miras la lista dos veces para asegurarte de que no falta nada.

Que `work properly` significa funcionar correctamente o como se espera. Por ejemplo:

`It's important that all machines work properly for safety.` - Esto significa que las máquinas deben funcionar de la manera correcta para que las personas estén seguras.
