# --description--

CRUD هو اختصار لـ Create وRead وUpdate وDelete. هذه هي العمليات الأربع للتخزين الدائم.

التخزين الدائم يعني حفظ البيانات بطريقة تجعلها متاحة حتى بعد إيقاف تشغيل الطاقة أو إعادة تشغيل الجهاز.

فهم كيفية عمل العملية الأساسية لـ CRUD أمر ضروري لتطوير الويب لأنه يشكل الأساس للعمل مع قواعد البيانات وبناء التطبيقات التي يمكن للمستخدمين من خلالها إضافة وعرض وتعديل وحذف البيانات.

الآن، دعنا نلقي نظرة أقرب على كل جزء من CRUD

- Create تشير إلى عملية إنشاء بيانات جديدة. على سبيل المثال، في تطبيق ويب، قد يكون هذا عندما يضيف المستخدم منشورًا جديدًا إلى مدونة.

- Read هي العملية التي يتم فيها استرجاع البيانات من قاعدة بيانات. على سبيل المثال، عندما تزور منشور مدونة أو تعرض ملفك الشخصي على موقع ويب، فأنت تقوم بعملية قراءة لجلب وعرض البيانات المخزنة في قاعدة البيانات.

- Update تتضمن تعديل البيانات الموجودة في قاعدة البيانات. مثال على ذلك هو تحرير منشور مدونة أو تحديث معلومات ملفك الشخصي.

- Delete هي العملية التي تزيل البيانات من قاعدة بيانات. على سبيل المثال، عندما تحذف منشور مدونة أو حسابًا، فأنت تقوم بعملية حذف.

تُستخدم CRUD عند العمل مع قواعد البيانات، وواجهة المستخدم، وواجهات برمجة التطبيقات RESTful. واجهات برمجة التطبيقات RESTful هي مجموعة من الاصطلاحات لبناء خدمات الويب التي تسمح للعميل بالتفاعل مع قاعدة بيانات أو نظام خلفي من خلال تنفيذ عمليات CRUD عبر طرق HTTP القياسية.

HTTP تعني بروتوكول نقل النص الفائق وهو الأساس لتواصل البيانات على الويب. هناك طرق HTTP تحدد الإجراءات التي يمكن تنفيذها على الموارد عبر الويب. الطرق الشائعة هي `GET`، `POST`، `PUT`، `PATCH`، `DELETE`.

سوف تتعلم المزيد عن واجهات برمجة التطبيقات RESTful وHTTP في دروس لاحقة، لكن هنا شرح سريع لكيفية ارتباط CRUD بطرق HTTP المختلفة.

- تُستخدم `POST` لإنشاء مورد جديد.

- تُستخدم `GET` لاسترجاع أو قراءة البيانات.

- تُستخدم `PUT` لتحديث مورد عن طريق استبداله بالكامل.

- تُستخدم `PATCH` لتحديث مورد جزئيًا.

- تُستخدم `DELETE` لإزالة مورد.

إليك مثالًا على كيفية تمثيل عمليات CRUD في الشفرة باستخدام مصفوفة بسيطة في JavaScript:

```js
let items = [];

// Create
function createItem(item) {
  items.push(item);
}

// Read
function readItems() {
  return items;
}

// Update
function updateItem(index, newItem) {
  items[index] = newItem;
}

// Delete
function deleteItem(index) {
  items.splice(index, 1);
}

// Example Usage
createItem('Book');
console.log(readItems()); // ['Book']
updateItem(0, 'Magazine');
console.log(readItems()); // ['Magazine']
deleteItem(0);
console.log(readItems()); // []
```

في هذا المثال:

- ننشئ عنصرًا عن طريق دفعه إلى مصفوفة.

- نقرأ العناصر عن طريق إرجاع المصفوفة.

- نُحدّث عنصرًا بتعديل عنصر المصفوفة عند فهرس معين.

- نحذف عنصرًا بإزالته من المصفوفة باستخدام `splice()`.

هذا تمثيل أساسي لكيفية عمل عمليات CRUD على المستوى المفاهيمي.
