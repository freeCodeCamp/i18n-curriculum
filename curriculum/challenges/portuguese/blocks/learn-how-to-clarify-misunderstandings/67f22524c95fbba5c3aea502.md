# --explanation--

`On track` significa que um projeto ou tarefa está progredindo conforme o planejado. Por exemplo:

`We're on track to finish the website next month.` — Isso significa que a equipe provavelmente vai atingir a meta.

`Release` significa lançar ou disponibilizar algo. Por exemplo:

`They plan to release the new version tomorrow.` — Isso significa que será tornado público.

`By the end of` define um período de tempo, significando antes de um ponto específico no tempo. Por exemplo:

`The report is due by the end of the day.` — Isso significa que deve ser concluído antes do final do dia.
