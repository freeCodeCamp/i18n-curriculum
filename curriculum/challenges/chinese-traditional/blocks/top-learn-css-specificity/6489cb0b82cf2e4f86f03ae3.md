# --description--

讓我們來看幾個快速範例，以視覺化了解特異性的運作方式。請參考以下的 HTML 和 CSS 程式碼：

```html
<!-- index.html -->

<div class="main">
  <div class="list subsection"></div>
</div>
```

```css
/* rule 1 */
.subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

在上述範例中，兩個規則都只使用類別選擇器，但規則 2 更具特異性，因為它使用了更多類別選擇器。
