# --description--

Іноді трапляються ситуації, коли ви знаєте більше про тип, ніж TypeScript. Саме тут на допомогу приходять твердження типів.

Ось приклад із використанням методу `querySelector()` для доступу до елемента з ідентифікатором `id` `submit`:

```ts
const submitBtn = document.querySelector("#submit");
```

Зараз TypeScript знає лише, що цей `submitBtn` є якимось `Element` або, можливо, `null`. TypeScript не дивиться на відповідний HTML, щоб це з’ясувати.

Це гарне місце, щоб використати твердження типу і сказати TypeScript, який тип елемента це є. Ось приклад:

```ts
const submitBtn = document.querySelector("#submit") as HTMLButtonElement;
```

Тепер TypeScript буде розглядати це як елемент `button` завдяки цьому твердженню типу.

Існують обмеження на використання ключового слова `as`. Наприклад, ви не можете робити таке без помилки від TypeScript:

```ts
// Conversion of type 'string' to type 'number' may be a mistake because neither type sufficiently overlaps with the other. If this was intentional, convert the expression to 'unknown' first.
const age = "awesome" as number;
```

У цьому прикладі не має великого сенсу намагатися стверджувати, що рядок `"awesome"` слід розглядати як тип number. TypeScript дозволяє твердження типів лише між сумісними або перекривними типами.

Один зі способів позбутися цієї помилки — використати подвійне твердження типу так:

```ts
const age = "awesome" as unknown as number;
```

Хоча це працює в TypeScript, все одно не рекомендується, бо логічно це не має сенсу.

Ще один спосіб записати твердження типів — використати синтаксис із кутовими дужками так:

```ts
const submitBtn = <HTMLButtonElement>document.querySelector("#submit");
```

Потрібно бути обережним із цим синтаксисом у файлах `tsx`, бо TypeScript може подумати, що ви намагаєтеся відтворити компонент, і викине помилку такого типу:

```ts
// file: index.tsx
// This JSX tag requires 'React' to be in scope, but it could not be found.
// JSX element 'HTMLButtonElement' has no corresponding closing tag.
const submitBtn =  <HTMLButtonElement>document.querySelector("#submit");
```

У більшості випадків ви побачите, що перший синтаксис із ключовим словом `as` використовується частіше за другий варіант.
