# --interactive--

In JavaScript, i template literal sono un modo potente e flessibile per lavorare con le stringhe. A differenza delle stringhe normali, che usano virgolette singole (`'`) o virgolette doppie (`"`), i template literal si definiscono con i segni di graduazione (`` ` ``).

Permettono una manipolazione più semplice delle stringhe, incluso incorporare variabili direttamente all’interno di una stringa, una funzionalità nota come interpolazione di stringhe.

I template literal facilitano la creazione di stringhe che si estendono su più righe o che includono espressioni (come variabili o anche codice JavaScript) direttamente all’interno della stringa.

Ecco un esempio di template literal:

:::interactive_editor

```js
const name = "Alice";
const greeting = `Hello, ${name}!`;

console.log(greeting);
```

:::

Nota l’uso dei segni di graduazione invece delle virgolette singole o doppie. La sintassi `${name}` è un esempio di interpolazione di stringhe, dove il valore della variabile `name` viene inserito direttamente nella stringa.

L’interpolazione di stringhe ti permette di incorporare variabili ed espressioni dentro una stringa. Questo è un miglioramento significativo rispetto al metodo più vecchio, dove concatenavi stringhe e variabili usando l’operatore `+`.

Ecco un esempio che usa la concatenazione di stringhe e l’operatore più (`+`):

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = "My name is " + name + " and I am " + age + " years old.";
console.log(message); 
```

:::

E qui un esempio che usa i template literal e l’interpolazione di stringhe:

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = `My name is ${name} and I am ${age} years old.`;
console.log(message); 
```

:::

Come vedi, l’interpolazione di stringhe con i template literal è molto più pulita e facile da leggere, specialmente quando lavori con più variabili.

Un’altra grande funzionalità dei template literal è che supportano stringhe multilinea. Con le stringhe normali, devi usare caratteri di escape (`\n`) per creare nuove righe. Con i template literal, puoi semplicemente scrivere la stringa su più righe e la formattazione viene mantenuta:

:::interactive_editor

```js
let poem = `Roses are red,
Violets are blue,
JavaScript is fun,
And so are you.`;

console.log(poem);
```

:::

Un’altra funzionalità dei template literal è che ti permettono di incorporare espressioni JavaScript direttamente nella stringa, come in questo esempio:

:::interactive_editor

```js
const song = "Bohemian Rhapsody";
const score = 9.5;
const highestScore = 10;
const output = `One of my favorite songs is "${song}". I rated it ${
  (score / highestScore) * 100
}%.`;
console.log(output); 
```

:::

I template literal sono particolarmente utili quando devi includere variabili o espressioni nelle stringhe, formattare stringhe complesse o lavorare con testo multilinea. Sono più concisi e leggibili rispetto alla tradizionale concatenazione di stringhe.
