# --explanation--

`Thorough` bedeutet, dass etwas mit großer Sorgfalt und Liebe zum Detail erledigt wurde. Zum Beispiel:

`She gave the code a thorough review.` – Das bedeutet, sie hat es sehr sorgfältig betrachtet.

`Coverage` bezieht sich darauf, wie viel von etwas einbezogen, getestet oder bearbeitet wurde. Zum Beispiel:

`The testing coverage for the new feature was excellent.` – Das bedeutet, dass viele Teile der Funktion getestet wurden.
