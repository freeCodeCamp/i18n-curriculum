# --description--
Ihr Spiel wird gegen den Computer gespielt. Sie schreiben eine Funktion, die zufällig `"rock"`, `"paper"` oder `"scissors"` zurückgibt.

Sie müssen sich nicht um den Frontend-Teil des Spiels kümmern. Sie schreiben nur die Logik für das Spiel. Öffnen Sie die `script.js` und folgen Sie den Anweisungen.

**User Stories:**

1. Sie sollten eine Funktion mit dem Namen `getComputerChoice` haben.

1. Ihre `getComputerChoice`-Funktion sollte zufällig `"rock"`, `"paper"` oder `"scissors"` zurückgeben.

**Hinweis:** Die [Math.random](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/random)-Methode gibt eine Zufallszahl zurück, die größer oder gleich 0 und kleiner als 1 ist. Überlegen Sie, wie Sie dies verwenden können, um bedingt eine der mehreren Möglichkeiten zurückzugeben.

Ihr Spiel wird von einem menschlichen Spieler gespielt. Sie schreiben eine Funktion, die die Wahl des Benutzers entgegennimmt und zurückgibt.

1. Erstellen Sie eine Funktion mit dem Namen `getHumanChoice`.

1. Schreiben Sie den Code so, dass `getHumanChoice` eine der gültigen Optionen zurückgibt, abhängig davon, was der Benutzer eingibt.

**Hinweis:** Verwenden Sie die [prompt](https://developer.mozilla.org/en-US/docs/Web/API/Window/prompt)-Methode, um die Eingabe des Benutzers zu erhalten.

Ihr Spiel wird den Punktestand des Spielers verfolgen. Sie schreiben Variablen, um den Punktestand des Spielers zu speichern.

1. Erstellen Sie zwei neue Variablen mit den Namen `humanScore` und `computerScore` im globalen Gültigkeitsbereich.

1. Initialisieren Sie diese Variablen mit dem Wert `0`.

Ihr Spiel wird Runde für Runde gespielt. Sie schreiben eine Funktion, die die Wahl des menschlichen und des Computer-Spielers als Argumente entgegennimmt, eine einzelne Runde spielt, den Punktestand des Rundengewinners erhöht und eine Gewinnerankündigung protokolliert.

1. Erstellen Sie eine neue Funktion mit dem Namen `playRound`.

1. Definieren Sie zwei Parameter für `playRound`. Parameter eins `humanChoice` und Parameter zwei `computerChoice`. Verwenden Sie diese beiden Parameter, um die Wahl des Menschen und des Computers als Argumente zu übergeben.

1. Machen Sie den `humanChoice`-Parameter Ihrer Funktion case-insensitive, damit Spieler `"rock"`, `"ROCK"`, `"RocK"` oder andere Varianten eingeben können.

1. Schreiben Sie den Code für Ihre `playRound`-Funktion, die einen String-Wert zurückgibt, der den Gewinner der Runde darstellt.

- Wenn es ein Unentschieden ist, sollte sie `"It's a tie!"` zurückgeben.

- Wenn der Spieler gewinnt, sollte sie `"You win! [player choice] beats [computer choice]"` zurückgeben.

- Wenn der Computer gewinnt, sollte sie `"You lose! [computer choice] beats [player choice]"` zurückgeben.

1. Erhöhen Sie die Variable `humanScore` oder `computerScore` basierend auf dem Gewinner der Runde.

Ihr Spiel wird 3 Runden spielen. Sie schreiben eine Funktion mit dem Namen `playGame`, die `playRound` aufruft, um 3 Runden zu spielen, die Punktestände verfolgt und am Ende einen Gewinner erklärt.

1. Erstellen Sie eine neue Funktion mit dem Namen `playGame`.

1. Erstellen Sie eine Schleife, die 3 Runden spielt und jedes Mal die `playRound`-Funktion mit den Funktionen für die Wahl des Menschen und des Computers als Argumente aufruft.

1. Geben Sie am Ende des Spiels den Gewinner basierend darauf zurück, wer die meisten Runden gewonnen hat.

- Wenn der menschliche Spieler mehr Runden als der Computer gewinnt, geben Sie eine Nachricht zurück, die `"You win the game!"` sagt.

- Wenn der Computer mehr Runden als der menschliche Spieler gewinnt, geben Sie eine Nachricht zurück, die `"You lose the game!"` sagt.

```js
const hand = ['rock', 'paper', 'scissors'];


let computerScore = 0;
let humanScore = 0;

const getComputerChoice = () => {
  return hand[Math.floor(Math.random() * hand.length)];
}

const getHumanChoice = () => {
  return prompt();
}

const playRound = (humanChoice, computerChoice) => {
  humanChoice = humanChoice.toLowerCase()

  const tie = "It's a tie!"
  const win = `You win! ${humanChoice} beats ${computerChoice}`
  const lose = `You lose! ${computerChoice} beats ${humanChoice}`


  if (humanChoice == 'rock') {
    if (computerChoice == 'rock') {
      computerScore++;
      humanScore++;
      return tie;
    } else if (computerChoice == 'paper') {
      computerScore++;
      return lose;
    } else {
      humanScore++;
      return win;
    }
  } else if (humanChoice == 'paper') {
    if (computerChoice == 'rock') {
      humanScore++;
      return win;
    } else if (computerChoice == 'paper') {
      computerScore++;
      humanScore++;
      return tie;
    } else {
      computerScore++;
      return lose;
    }
  } else if (humanChoice == 'scissors') {
    if (computerChoice == 'rock') {
      computerScore++;
      return lose;
    } else if (computerChoice == 'paper') {
      humanScore++;
      return win;
    } else {
      computerScore++;
      humanScore++;
      return tie;
    }
  }
}

const playGame = () => {
  for(let i = 0; i < 3; i++){
    playRound(getHumanChoice(), getComputerChoice())
  }

  return humanScore > computerScore ? "You win the game!" : "You lose the game!"
}
```
