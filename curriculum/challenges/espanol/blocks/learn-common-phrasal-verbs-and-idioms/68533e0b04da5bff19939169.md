# --explanation--

La estructura `help` + `to` + verbo se usa para mostrar que alguien apoya o asiste en hacer algo. También es correcto decir `help` + verbo sin `to` (por ejemplo, `help make`). Otro ejemplo: 

`He helped to fix the code.` o `He helped fix the code.` – Ambos significan que él brindó apoyo para resolver el problema.
