# --description--

En el juego de casino Blackjack, un jugador puede determinar si tiene una ventaja en la siguiente mano sobre la casa llevando la cuenta del número relativo de cartas altas y bajas que quedan en la baraja. Esto se llama cuenta de tarjetas.

Tener más cartas altas en la baraja es una ventaja para el jugador. Cuando el conteo es positivo, el jugador debería apostar alto. Cuando el conteo da 0 o negativo, el jugador debería apostar bajo.

**Objetivo:** Cumplir con las historias de usuario a continuación y pasar todas las pruebas para completar el laboratorio.

**Historias de usuario:**

1. Debes usar `let` para declarar una variable global llamada `count` y asignarle el valor `0`.
1. Deberías tener una función llamada `cc`.
1. The `cc` function should receive a `card` parameter which can either be a number or string.
    - Para valores entre `2` y `10`, el parámetro `card` será un número.
    - Para todos los demás valores, el parámetro `card` será una cadena.
1. La función `cc` debería modificar la variable global `count` según ciertos criterios.
1. La variable global `count` debe incrementarse en `1` para las cartas `2`, `3`, `4`, `5` o `6`
1. La variable global `count` debe permanecer sin cambios para las tarjetas `7`, `8`, `9`.
1. La variable global `count` debe disminuirse en `1` para las cartas `10`, `"J"`, `"Q"`, `"K"`, `"A"`
1. La función `cc` debería devolver una cadena con el conteo actual y la cadena `Bet` si el conteo es positivo.
1. La función `cc` debería devolver una cadena con el conteo actual y la cadena `Hold` si el conteo es menor o igual a `0`.
1. En la salida de la función, el conteo actual y la decisión del jugador (`Bet` o `Hold`) deben estar separados por un espacio. Por ejemplo, `-3 Hold`.

# --solutions--

```js
let count = 0;
function cc(card) {
  switch(card) {
    case 2:
    case 3:
    case 4:
    case 5:
    case 6:
      count++;
      break;
    case 10:
    case 'J':
    case 'Q':
    case 'K':
    case 'A':
      count--;
  }
  if(count > 0) {
    return count + " Bet";
  } else {
    return count + " Hold";
  }
}
```
