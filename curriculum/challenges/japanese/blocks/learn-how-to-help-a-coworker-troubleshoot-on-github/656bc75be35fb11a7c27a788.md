# --description--

フレーズ `Got it` は、誰かが何かを理解したことを示すために会話でよく使われます。これは、情報を受け取り理解したことをカジュアルかつ素早く認める方法です。例えば：

Lucy: `You need to finish the report by tomorrow.`
ボブ: `Got it!` 

`Got it` は、ボブ が指示を理解していることを意味します。
