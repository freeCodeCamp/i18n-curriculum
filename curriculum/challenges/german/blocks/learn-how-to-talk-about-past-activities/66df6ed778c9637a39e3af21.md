# --explanation--

Um zu verstehen, warum jemand glaubt, dass etwas bald passieren kann, suchen Sie nach Gründen, die in der Aussage genannt werden.

Im Satz `We've already started working on the fixes` zeigt `already`, dass sie früher als erwartet mit den Fehlerbehebungen begonnen haben.

Das zeigt, dass sie mit der notwendigen Arbeit begonnen haben, was ihre Überzeugung unterstützt, dass sie bald ein weiteres Update veröffentlichen können.
