# --description--

在上一课中，你学习了如何在本地运行 Python 脚本。但有时你可能不想创建完整的 Python 程序，只需要测试一些 Python 代码。这时就用到了 Python 交互式 shell。

但首先，让我们回顾一下什么是终端。

终端是一个基于文本的接口，允许你通过输入命令与计算机交互。每个操作系统都自带一个默认的终端应用程序。在 macOS 上，你可以使用 Terminal 应用。在 Windows 上，你可以使用命令提示符或 PowerShell。在 Linux 上，每个桌面环境都有自己的默认终端应用，比如 GNOME Terminal 或 Konsole。

打开一个终端应用，输入 `python` 并按下 `Enter`。这将启动 Python 交互式 shell。交互式 shell 是一个程序，允许你一次输入一个命令并看到结果。

当你启动一个新会话时，最初你可能会看到如下输出：

```bash
Python 3.12.2 (main, Mar 21 2024, 22:48:26) [Clang 14.0.3 (clang-1403.0.22.14.1)] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>> 
```

`>>>` 符号表示 Python 正在等待你输入命令。试着向终端打印 `"Hello, world!"`：

```bash
print("Hello, world!")
```

你应该会看到文本像这样显示：

```bash
>>> print("Hello, world!")
Hello, world!
```

文本打印完毕后，Python 会返回到以下状态：

```bash
>>>
```

这意味着你可以输入另一个命令。

Python 解释器遵循被称为读取、求值、打印、循环的周期，简称 REPL。每当你输入命令时，解释器会读取输入，求值，打印结果，然后循环回显示 `>>>`，以便你输入更多命令。

如果你尝试输入一个无效命令，会发生什么？

```bash
>>> something random
```

Python 解释器仍然会遵循相同的 REPL 过程。在这种情况下，你会收到错误信息：

```bash
>>> something random
  File "<stdin>", line 1
    something random
              ^^^^^^
SyntaxError: invalid syntax
```

如果你想退出交互式 shell，可以输入 `exit()`，或者在 Mac/Linux 上按 `Ctrl + D`，在 Windows 上按 `Ctrl + Z + Enter`。

使用 Python 的交互式 shell 非常适合进行小规模的代码实验或基础探索。在你完成本课程剩余的课程时，可以在交互式 shell 中输入你学到的一些新方法。

如果你打算处理更长的程序或跨多个文件工作，那么最好使用代码编辑器或集成开发环境。
