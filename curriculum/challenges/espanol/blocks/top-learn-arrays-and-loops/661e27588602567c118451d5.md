# --description--

Ahora que tienes un entendimiento básico de arreglos, hablemos sobre bucles. Los bucles se usan para ejecutar un bloque de código varias veces. Uno de esos bucles es el bucle `while`. El bucle `while` ejecuta un bloque de código siempre y cuando la condición sea verdadera. La sintaxis del bucle `while` es la siguiente:

```javascript
while (condition) {
  // code block to be executed
}
```

Por ejemplo, el siguiente fragmento de código muestra los números de 1 a 5:

```javascript
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
```

El fragmento de código anterior inicializa una variable `i` con el valor `1`.  The `while` loop executes the block of code as long as the value of `i` is less than or equal to `5`. El valor de `i` se incrementa por `1` en cada iteración.
