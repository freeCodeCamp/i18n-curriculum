# --description--

今日はグラウンドホッグデーです。この日はグラウンドホッグが自分の影を見たかどうかで天気を予測します。

グラウンドホッグの様子を表す値が与えられたら、正しい予測を返してください。

- 与えられた値がブール値の`true`（グラウンドホッグが影を見た）なら、`"Looks like we'll have six more weeks of winter."`を返してください。
- 値がブール値の`false`（グラウンドホッグが影を見なかった）なら、`"It's going to be an early spring."`を返してください。
- それ以外の値（グラウンドホッグが現れなかった）なら、`"No prediction this year."`を返してください。
