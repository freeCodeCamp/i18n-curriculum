# --description--

`Low-code development` é uma forma de criar aplicações de software com codificação manual mínima.Em vez de escrever linhas extensas de código, os desenvolvedores usam uma interface gráfica do usuário com recursos de arrastar e soltar para montar e configurar aplicações.Essa abordagem permite um desenvolvimento mais rápido e é especialmente benéfica para usuários que podem não ter habilidades extensas de programação.
