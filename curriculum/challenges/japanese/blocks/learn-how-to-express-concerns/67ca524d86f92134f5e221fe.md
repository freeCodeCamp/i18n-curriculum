# --explanation--

`How` は、何かを行う方法や手段について尋ねるときに使います。`how` が別の疑問詞（この場合は`Did they say`）に続くと、文の残りは肯定文の構造（`we could fix it`）になります。例えば：

- `Did she explain how we can improve communication?` - これは正しいです。`how` の後に続く部分は肯定文の構造です。ここで`Did she explain how can we improve communication?` を使うのは誤りです。

- `Can you show me how I should submit the report?` - `how`（`I should submit the report`）の後に続く部分は肯定文の構造（`I should` で、`should I` ではありません）になります。
