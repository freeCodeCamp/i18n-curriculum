# --description--

반지름이 $r$인 원 $C$, $x^2 + y^2 = r^2$가 있습니다. $P(a, b)$와 $Q(-a, c)$ 두 점을 선택하여 $P$와 $Q$를 지나는 직선이 $C$에 접하도록 합니다.

예를 들어, 사중쌍 $(r, a, b, c) = (2, 6, 2, -7)$는 이 조건을 만족합니다.

$F(R, X)$를 이 조건을 만족하고 $0 < r ≤ R$ 및 $0 < a ≤ X$인 정수 사중쌍 $(r, a, b, c)$의 개수라고 합시다.

$F(1, 5) = 10$, $F(2, 10) = 52$, $F(10, 100) = 3384$임을 확인할 수 있습니다.

$F({10}^8, {10}^9) + F({10}^9, {10}^8)$의 값을 구하세요.
