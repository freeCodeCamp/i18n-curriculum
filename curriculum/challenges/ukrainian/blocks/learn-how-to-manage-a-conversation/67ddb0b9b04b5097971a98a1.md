# --explanation--

У розмовній мові часто опускають слова, якщо значення залишається зрозумілим. У цьому випадку `Anything else we should discuss?` — це скорочена форма `Is there anything else we should discuss?`. Таке опущення є поширеним у розмовах. Ще один приклад:

`Need any help?` — це коротший спосіб сказати `Do you need any help?` — підмет і допоміжне дієслово опущені, але значення залишається тим самим.
