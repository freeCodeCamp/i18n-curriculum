# --questions--

## --text--

Яке речення правильно використовує `I'd rather`?

## --answers--

`I'd rather use Python for data analysis than JavaScript.`

---

`I'd rather to use Python for data analysis than JavaScript.`

### --feedback--

Після `Rather` має йти інфінітив дієслова, а не `to`.

---

`I'd rather using Python for data analysis than JavaScript.`

### --feedback--

Після `Rather` має йти інфінітив дієслова, а не `using`.

---

`I'd rather use Python to data analysis than JavaScript.`

### --feedback--

Фраза `to data analysis` має бути `for data analysis`.

1

# --explanation--

`I'd rather` — це фраза, яка використовується для вираження переваги між двома діями або варіантами. Після неї має йти інфінітив дієслова.
