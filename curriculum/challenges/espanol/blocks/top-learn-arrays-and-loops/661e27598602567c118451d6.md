# --description--

El `for` loop es otro tipo de ciclo que se usa para ejecutar un bloque de código múltiples veces. El `for` loop se usa cuando se conoce el número de iteraciones. La sintaxis del `for` loop es la siguiente:

```javascript
for (initialization; condition; increment/decrement) {
  // code block to be executed
}
```

Por ejemplo, el siguiente fragmento de código muestra los números de 1 a 5:

```javascript
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
```

Para iterar sobre un arreglo usando un ciclo `for`, puedes usar la propiedad length del arreglo. Por ejemplo, el siguiente fragmento de código imprime los elementos de un arreglo:

```javascript
const arr = [1, 2, 3, 4, 5];
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}
```
