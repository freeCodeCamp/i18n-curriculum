# --interactive--

CSS Gridは、開発者が複雑でレスポンシブなウェブページのレイアウトを簡単に作成できる強力なレイアウトシステムです。

部屋に家具を配置することを想像してください。CSS Gridは、床に見えないグリッドがあって、すべてを正確に配置できるようにするイメージです。

ウェブサイトを作るとき、ページ上のさまざまな要素を配置する必要があります。

CSS Gridが登場する前は、特に複雑なレイアウトの場合、これが難しいことがありました。CSS Gridは、ウェブページを行と列に分割してグリッドのような構造を作ることで、この作業を簡単にします。

例えば、`div`というコンテナの中にいくつかのアイテムがネストされているとします。

:::interactive_editor

```html
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

:::

これらの要素をグリッド形式でスタイル設定したい場合、`display`を`grid`に設定し、次のように列を指定できます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-gap: 20px;
}

.item {
  background-color: lightgray;
  padding: 20px;
  text-align: center;
  border: 1px solid #ccc;
}
```

:::

このコードでは、ブラウザに3つの等幅の列を持つグリッドを作成するよう指示しています。`1fr 1fr 1fr`はその意味で、各グリッドアイテムの間に20ピクセルのギャップを追加しています。

ここで、「Flexboxはどうなの？レイアウトに使われるのでは？」と思うかもしれません。

その通りです！Flexboxも別のCSSレイアウトモデルで、とても便利です。しかし、いくつか重要な違いがあります。

Flexboxは一次元で、Gridは二次元です。つまり、Flexboxは1列または1行のレイアウトに適していて、Gridは行と列の両方を使ったレイアウトに優れています。

Flexboxはコンテンツ優先で、内容に合わせてレイアウトを調整します。一方、Gridはレイアウト優先で、レイアウトを先に作ってからアイテムを配置します。Gridは配置の精密な制御が可能で、アイテムがどの行と列に入るかを正確に指定できます。

比較のためにFlexboxの例を示します。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
}
```

:::

これは、主軸に沿ってアイテムが均等に配置されるフレックスコンテナを作成します。

GridとFlexboxはそれぞれ強みがあり、多くの場合、最適なレイアウトは両方を組み合わせて使います。ページ全体のレイアウトにはGridを使い、各グリッドエリア内のアイテムの整列にはFlexboxを使うことがあります。

まとめると、CSS Gridは精密な二次元レイアウトを可能にする強力なツールです。最初は複雑に感じるかもしれませんが、練習すればレスポンシブで複雑なウェブレイアウトを作るのに欠かせないツールになります。
