# --description--

Компонент `extension` — це програмний компонент, який додає певні можливості до більшого програмного застосунку. `Extensions` може надавати нові функції або покращувати існуючі.

Наприклад, розширення для браузера `extension` може додати спеціальні функції до вашого веббраузера, як-от блокування реклами або керування паролями.

# --questions--

## --text--

Що означає `extension` у контексті програмування?

## --answers--

Це команда, яка використовується для завершення програми.

### --feedback--

`extension` — це не команда, а програмне розширення, яке покращує функціональність.

---

Це програмне розширення, яке покращує можливості існуючої програми.

---

Це означає додатковий час, наданий для завершення проєкту.

### --feedback--

Хоча `extension` у інших контекстах може означати додатковий час, у програмуванні це конкретно програмний компонент.

---

Це тип формату даних, який використовується для зберігання.

### --feedback--

`extension` покращує функціональність програмного забезпечення; він не означає формат зберігання даних.
