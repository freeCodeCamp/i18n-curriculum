# --description--

L'istruzione `debugger` è uno strumento potente di JavaScript che ti permette di mettere in pausa il codice su una riga specifica per indagare cosa sta succedendo nel programma. Usata correttamente, l'istruzione `debugger` può farti risparmiare molto tempo nel cercare di capire perché qualcosa non funziona come dovrebbe.

JavaScript esegue il tuo codice dall'alto verso il basso. Quando JavaScript esegue il codice e incontra un'istruzione `debugger`, mette immediatamente in pausa l'esecuzione su quella riga. Questo ti dà la possibilità di ispezionare variabili, controllare funzioni e il flusso del codice in generale.

Così puoi vedere esattamente cosa sta andando bene o male. Tuttavia, questo succede solo se gli strumenti per sviluppatori del browser sono aperti. Altrimenti, l'istruzione `debugger` viene ignorata e il codice continua a essere eseguito normalmente.

Inoltre, quando usi l'istruzione `debugger`, i browser moderni mettono in pausa l'esecuzione del codice sulla riga specificata. Ti permettono anche di riprendere l'esecuzione cliccando il pulsante play, ma la pagina stessa non si ricarica automaticamente a meno che non venga fatto manualmente.

Per usare l'istruzione `debugger`, posizionala sulla riga in cui vuoi che l'esecuzione del codice si fermi. Ecco un esempio:

```js
let firstNumber = 5;
let secondNumber = 10;
debugger; // Code execution pauses here
let sum = firstNumber + secondNumber;
console.log(sum);
```

Con questo esempio, se non hai la console aperta prima che il codice venga eseguito e la apri in seguito, vedrai `15` lì dentro. Mentre la console è aperta e ricarichi la pagina, l'esecuzione si mette in pausa così puoi ispezionare il codice.

Ecco un esempio più complesso:

```js
function calculateTotalPrice(price, discountPercentage) {
 debugger
 let discountAmount = (price * discountPercentage) / 100
 let totalPrice = price - discountAmount

 console.log(`Original Price: ${price}`)
 console.log(`Discount Amount: ${discountAmount}`)
 console.log(`Total Price after Discount: ${totalPrice}`)

 return totalPrice
}

let price = 100
let discount = 15

let finalPrice = calculateTotalPrice(price, discount)
console.log(`Final Price: ${finalPrice}`);
```

In questo esempio l'istruzione `debugger` è posizionata all'inizio della `calculateTotalPrice`. Quando la funzione viene chiamata, l'esecuzione della funzione si mette in pausa.

Se ricarichi la pagina con la console aperta, vedrai che la pagina continua a ricaricarsi, confermando che l'esecuzione del codice è solo in pausa, non terminata. Se vuoi che l'esecuzione continui, puoi cliccare il pulsante play.

Mentre continui a costruire i tuoi programmi JavaScript, prova l'istruzione `debugger` nel tuo codice per vedere come può aiutarti a capire meglio il flusso di esecuzione, ispezionare le variabili in tempo reale e identificare rapidamente dove qualcosa potrebbe andare storto.
