# --description--

Дивно, але існують лише три числа, які можна записати як суму четвертих степенів своїх цифр:

<div style='margin-left: 4em;'>
  1634 = 1<sup>4</sup> + 6<sup>4</sup> + 3<sup>4</sup> + 4<sup>4</sup><br>
  8208 = 8<sup>4</sup> + 2<sup>4</sup> + 0<sup>4</sup> + 8<sup>4</sup><br>
  9474 = 9<sup>4</sup> + 4<sup>4</sup> + 7<sup>4</sup> + 4<sup>4</sup><br>
</div>

Оскільки 1 = 1<sup>4</sup> не є сумою, воно не включається.

Сума цих чисел дорівнює 1634 + 8208 + 9474 = 19316.

Знайдіть суму всіх чисел, які можна записати як суму `n` степенів своїх цифр.
