# --explanation--

`Me too` 用于表达认同或表示相同的语句也适用于说话者。这是一种简洁且非正式的方式，用来回应共同的经历或感受。例如： 

人物 1：`I love Python programming.` 

人物 2：`Me too.` － 意思是他们也喜欢用 Python 编程。

在这个上下文中，Maria 可以说 `Yeah, me too` 来快速让 Brian 知道她也参加了 PyCon 上非常有趣的会话。
