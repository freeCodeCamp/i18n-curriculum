# --description--

`to debug` という用語はプログラミングで非常に重要です。これはコンピューターのプログラムのエラーやバグを見つけて修正することを意味します。`bug` はプログラムが正しく動作しない原因となる間違いや問題のことです。

`Debugging` はプログラマーの仕事の中で日常的に行われる作業です。

例えば、`She spent the whole day debugging the code.` これは彼女がコードの間違いを修正していたことを意味します。
