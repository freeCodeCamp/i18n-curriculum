# --description--

Існують деякі прості числа, $p$, для яких існує додатне ціле число, $n$, таке що вираз $n^3 + n^{2}p$ є досконалим кубом.

Наприклад, коли $p = 19,\\ 8^3 + 8^2 × 19 = {12}^3$.

Що, можливо, найбільше дивує, це те, що значення $n$ унікальне для кожного простого числа з цією властивістю, і таких простих чисел менше за сто лише чотири.

Скільки простих чисел менше за мільйон мають цю дивовижну властивість?
