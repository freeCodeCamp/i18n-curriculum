# --description--

Cuando hablas de otra persona, puedes usar esta pregunta para preguntar su nombre:

`¿Cómo se llama?` - ¿Cómo se llama?

La palabra `cómo` significa "cómo". Es muy importante escribir la tilde en la letra `o` cuando esta palabra inicia una pregunta.

La palabra `se` implica que estás hablando de una tercera persona.

Elena está hablando con Esteban sobre su nuevo colega, Diego.
