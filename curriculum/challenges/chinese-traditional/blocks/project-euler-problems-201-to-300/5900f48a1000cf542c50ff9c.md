# --description--

Albert 選擇一個正整數 `k`，然後在區間 [0,1] 內以均勻分布隨機選擇兩個實數 `a`、`b`。

接著計算和 ${(ka + 1)}^2 + {(kb + 1)}^2$ 的平方根，並將結果四捨五入到最接近的整數。如果結果與 $k$ 相等，他得 $k$ 分；否則他得零分。

例如，如果 $k = 6$、$a = 0.2$ 和 $b = 0.85$，則 ${(ka + 1)}^2 + {(kb + 1)}^2 = 42.05$。42.05 的平方根是 6.484……，四捨五入到最接近的整數後，變成 6。這與 $k$ 相等，所以他得 6 分。

可以證明，如果他以 $k = 1、k = 2、\ldots、k = 10$ 玩 10 回合，他的總分期望值四捨五入到小數點後五位是 10.20914。

如果他玩 ${10}^5$ 回合，且 $k = 1, k = 2, k = 3, \ldots, k = {10}^5$，他的總分期望值是多少，並四捨五入到小數點後五位？
