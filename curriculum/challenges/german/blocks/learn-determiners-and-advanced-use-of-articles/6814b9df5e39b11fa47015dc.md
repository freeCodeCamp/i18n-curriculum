# --explanation--

`Either` wird verwendet, um eine Wahl zwischen zwei Optionen einzuleiten. In diesem Fall die Nutzung von Auftragnehmern oder die Verteilung der Arbeit.

`Contractors` sind temporäre oder freiberufliche Arbeitskräfte, die oft für eine bestimmte Aufgabe eingestellt werden. Zum Beispiel:

`We hired contractors to help with the new app design.` – Externe Arbeitskräfte helfen bei der Gestaltung der App.

In diesem Dialog wird kein Artikel vor `part-time contractors` verwendet, weil es sich um eine allgemeine Kategorie handelt, nicht um eine spezifische Gruppe.
