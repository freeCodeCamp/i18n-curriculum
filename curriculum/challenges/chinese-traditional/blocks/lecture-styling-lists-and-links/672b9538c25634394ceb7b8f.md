# --interactive--

連結有不同的狀態，包括 `link`、`visited`、`hover`、`focus` 和 `active`。這些狀態對於幫助使用者識別連結並在互動後提供清楚的回授非常重要，這能提升可用性和無障礙性。

為這些不同的 `link` 狀態設計樣式對於可用性和無障礙性至關重要，因為它提供了關於 `link` 當前狀態的視覺提示。這有助於使用者了解他們已經造訪過哪些 `link`、正在互動的 `link` 是哪一個，以及點擊時會發生什麼。

對於有視覺障礙或認知障礙的使用者，這些不同的樣式可以讓導航變得更加簡單且直覺。

此外，清晰的聯結狀態透過提供即時的回授來增強整體使用者體驗，減少混淆並提升網站的可導覽性。

這些狀態可以使用 CSS 中稱為 `pseudo-classes` 的東西來設定樣式。

偽類別是添加到選擇器中的關鍵字，用以指定所選元素的特殊狀態。

例如，` :hover` 可以在使用者的指標暫留在按鈕上時改變按鈕的顏色，而 `:visited` 可以改變已經造訪過的連結的顏色。

偽類別允許你根據元素的狀態或使用者與其互動的情況來設定樣式，而無需在你的 HTML 中添加額外的標記。

`pseudo-class` 的語法看起來像這樣，其中 `A` 是選擇器，`:B` 是 `pseudo-class`：

```css
A:B {
  property: value;
}
```

為了更好地了解如何為不同的 `link` 狀態設計樣式，讓我們來看看一些範例。

`:link` 偽類別會為未造訪的連結套用樣式，表示它們是可點擊的。

以下是標的錨點元素並使用 `:link` 偽類的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Normal state (unvisited link) */
a:link { 
  color: red;
}
```

:::

上述範例會在聯結未被造訪時將其預設藍色改為 `red`。

`:visited` 樣式會套用到已經被造訪或點擊過的連結，幫助使用者追蹤他們之前點擊過的連結。以下是 `:visited` 偽類的範例用法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="https://www.freecodecamp.org/learn/" target="_blank">freeCodeCamp</a>
```

```css
/* Visited link */
a:visited {
  color: green;
}
```

:::

當 `link` 被點擊時，這段程式碼會將其顏色設為 `green`。

`:hover` 會在使用者暫留於聯結上時改變聯結的樣式，提供視覺提示表示該聯結是可互動的。以下是 `:hover` 偽類的範例用法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Hover state */
a:hover {
  color: green;
}
```

:::

當滑鼠暫留在聯結上時，這段程式碼會將聯結的顏色設為 `green`。

`:focus` 在聯結獲得焦點時（例如使用鍵盤導航或增強無障礙功能時）會添加樣式。

以下範例使用 `outline` 屬性，當 `link` 獲得焦點時套用實心橘色外框。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Focus state */
a:focus {
  outline: 2px solid orange;
}
```

:::

`:active` 會在連結被點擊時改變連結的樣式，提供即時回授給使用者，讓他們知道他們的動作已被暫存器。以下是 `:active` 偽類的範例用法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Active state */
a:active {
  color: pink;
}
```

:::

當點擊聯結時，這段程式碼範例會立即將聯結變為 `pink` 顏色。
