# --description--

N×N のディスクが正方形のゲームボードに置かれています。各ディスクは黒い面と白い面を持っています。

各ターンで、1つのディスクを選び、そのディスクと同じ行および同じ列にあるすべてのディスクを裏返すことができます。つまり、$2 × N - 1$ 個のディスクが裏返されます。ゲームはすべてのディスクが白い面を向くと終了します。以下の例は 5×5 のボードでのゲームを示しています。

<img alt="animation showing game on 5x5 board" src="https://cdn.freecodecamp.org/curriculum/project-euler/cross-flips.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

このゲームを終了するための最小ターン数は 3 であることが証明されています。

$N×N$ ボードの左下のディスクの座標は (0, 0)、右下のディスクの座標は ($N - 1$, $0$)、左上のディスクの座標は ($0$, $N - 1$) です。

$C_N$ を $N × N$ のディスクが置かれたボードの次の配置とします：座標 ($x$, $y$) のディスクが $N - 1 \le \sqrt{x^2 + y^2} \lt N$ を満たす場合、そのディスクは黒い面を示し、それ以外は白い面を示します。$C_5$ は上記の図の通りです。

$T(N)$ を、配置 $C_N$ から始めてゲームを終了するための最小ターン数、または配置 $C_N$ が解けない場合は 0 とします。$T(5) = 3$ であることが示されています。また、$T(10) = 29$、$T(1\,000) = 395\,253$ であることも与えられています。

$\displaystyle \sum_{i = 3}^{31} T(2^i - i)$ を求めてください。
