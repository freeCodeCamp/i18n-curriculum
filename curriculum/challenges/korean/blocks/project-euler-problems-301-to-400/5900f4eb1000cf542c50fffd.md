# --description--

다각형은 닫힌 사슬이나 회로를 이루도록 연결된 직선 구간으로 구성된 평면 도형입니다. 다각형은 적어도 세 변으로 이루어지며 자기 자신과 교차하지 않습니다.

양수 집합 $S$가 다각형 $P$를 생성한다고 할 때 다음 조건을 만족합니다:

- $P$의 두 변 길이가 같지 않습니다,
- $P$의 모든 변 길이가 $S$에 포함되어 있습니다,
- $S$에는 다른 값이 없습니다.

예를 들어:

집합 {3, 4, 5}는 변이 3, 4, 5인 다각형(삼각형)을 생성합니다.

집합 {6, 9, 11, 24}는 변이 6, 9, 11, 24인 다각형(사각형)을 생성합니다.

집합 {1, 2, 3}과 {2, 3, 4, 9}는 어떤 다각형도 생성하지 않습니다.

다음과 같이 정의된 수열 $s$를 고려하세요:

- $s_1 = 1$, $s_2 = 2$, $s_3 = 3$
- $s_n = s_{n - 1} + s_{n - 3}$ for $n > 3$.

$U_n$을 집합 $\\{s_1, s_2, \ldots, s_n\\}$로 정의합니다. 예를 들어, $U_{10} = \\{1, 2, 3, 4, 6, 9, 13, 19, 28, 41\\}$입니다.

$f(n)$을 $U_n$의 부분집합 중 적어도 하나의 다각형을 생성하는 부분집합의 개수라고 합시다.

예를 들어, $f(5) = 7$, $f(10) = 501$, $f(25) = 18\\,635\\,853$입니다.

$f({10}^{18})$의 마지막 9자리를 구하세요.
