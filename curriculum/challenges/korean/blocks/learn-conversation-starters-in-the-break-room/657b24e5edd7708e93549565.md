# --description--

`Present Perfect`는 현재와 과거 사이의 연결을 나타내는 시제입니다. 행동의 시점은 지금보다 이전이지만 구체적으로 명시되지 않았고, 이를 사용하는 사람은 보통 행동 자체보다 결과에 더 관심이 있습니다. 예를 들어:

- `I played this game yesterday.` - 이것은 `past simple` 시제입니다. 게임을 한 행동이 어제라는 특정한 과거 시점에 일어났음을 나타냅니다.

- `I have played this game before.` - 이것은 `present perfect` 시제입니다. 게임을 한 행동이 불특정한 과거 시점에 일어났음을 나타냅니다.

- `I've played this game before.` - 이 문장도 `present perfect` 시제이며, `I've`의 축약형인 `I have`을 사용합니다. 이전 예와 마찬가지로 게임을 한 행동이 불특정한 과거 시점에 일어났음을 나타냅니다.
