# --explanation--

Los `Brute force attacks` son cuando los hackers intentan múltiples contraseñas hasta encontrar la correcta. Por ejemplo:

`The system detected a brute force attack on the login page.` - El ataque intentó muchas conjeturas de contraseña para entrar.

Los `SQL injection attempts` ocurren cuando los hackers intentan insertar código SQL malicioso para acceder o modificar una base de datos. Por ejemplo:

`The firewall stopped two SQL injection attempts.` - Alguien intentó hackear la base de datos usando código dañino.
