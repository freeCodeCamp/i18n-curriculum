# --description--

考虑连续的素数 $p_1 = 19$ 和 $p_2 = 23$。 可以验证 1219 是最小的以数字 $p_1$ 形成低位部分，而又能够被 $p_2$ 整除的数字。

事实上，除了 $p_1 = 3$ 和 $p_2 = 5$ 之外，对于每一对连续的素数 $p_2 > p_1$，存在某些 $n$ 的值，使得最后的数字由 $p_1$ 组成且 $n$ 能被 $p_2$ 整除。令 $S$ 为这些 $n$ 值中的最小值。

对连续素数对 $5 ≤ p_1 ≤ 1000000$ 求 $\sum{S}$。
