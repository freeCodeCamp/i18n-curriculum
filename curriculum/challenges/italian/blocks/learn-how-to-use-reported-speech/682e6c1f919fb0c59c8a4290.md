# --explanation--

Nel **discorso indiretto**, la frase `said that` si usa per riferire ciò che qualcuno ha detto in modo diverso, senza usare le sue parole esatte. La parola `that` è opzionale e spesso viene omessa nel linguaggio quotidiano. Per esempio:

- Discorso diretto: `We need to update the software.`

- Discorso indiretto: `He said that we need to update the software.` o `He said we need to update the software.`

Entrambe le versioni sono corrette e hanno lo stesso significato.
