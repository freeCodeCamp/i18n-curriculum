# --description--

A expressão `only if` é usada para definir uma condição para que algo aconteça. É como dizer "isso acontecerá se aquilo acontecer". É uma forma de fazer um acordo onde uma ação depende de outra.

Por exemplo, seu amigo pode dizer, `I'll go out only if it doesn't rain`. Isso significa que ele vai sair mas o tempo deve estar bom - sem chuva.
