## ユーザー操作の疑似クラス

- **疑似クラスの定義**：特定の状態や位置に基づいて要素を選択できる特別なCSSキーワードです。
- **ユーザー操作の疑似クラス**：ユーザーの操作に応じて要素の見た目を変えるための特別なキーワードで、ユーザー体験を向上させます。
- **`:active` 疑似クラス**：ボタンのクリックなど、要素のアクティブ状態を選択できます。
- **`:hover` 疑似クラス**：要素のホバー状態を定義します。
- **`:focus` 疑似クラス**：キーボード操作やフォーム入力へのクリックで要素がフォーカスを得たときにスタイルを適用します。
- **`:focus-within` 疑似クラス**：要素自身またはその子孫要素がフォーカスを持つ場合にスタイルを適用します。

## 入力の疑似クラス

- **入力の疑似クラス**：ユーザー操作の前後で状態に応じてHTML `input` 要素をターゲットにします。
- **`:enabled` 疑似クラス**：現在有効なフォームのボタンやその他の要素をターゲットにします。
- **`:disabled` 疑似クラス**：無効状態のインタラクティブな要素にスタイルを適用します。
- **`:checked` 疑似クラス**：チェックされていることをユーザーに示すために使います。
- **`:valid` 疑似クラス**：バリデーション基準を満たす入力フィールドをターゲットにします。
- **`:invalid` 疑似クラス**：バリデーション基準を満たさない入力フィールドをターゲットにします。
- **`:in-range` と `:out-of-range` 疑似クラス**：値が指定された範囲内か範囲外かに応じてスタイルを適用します。
- **`:required` 疑似クラス**：`input` 要素のうち、`required` 属性を持つものをターゲットにします。フォーム提出のために必須入力であることを示します。
- **`:optional` 疑似クラス**：必須ではなく空欄のままにできる入力要素にスタイルを適用します。
- **`:autofill` 疑似クラス**：ブラウザーが保存データで自動入力した入力フィールドにスタイルを適用します。

## ロケーションの疑似クラス

- **ロケーションの疑似クラス**：現在のドキュメント内でターゲットとなるリンクや要素のスタイルに使います。
- **`:any-link` 疑似クラス**：`:link` と `:visited` 疑似クラスの組み合わせで、訪問済みかどうかに関わらずhref属性を持つすべてのアンカー要素にマッチします。
- **`:link` 疑似クラス**：ウェブページ上の未訪問リンクすべてをターゲットにできます。ユーザーがクリックする前のリンクのスタイルを変えるのに使います。
- **`:local-link` 疑似クラス**：同じドキュメントを指すリンクをターゲットにします。内部リンクと外部リンクを区別したいときに便利です。
- **`:visited` 疑似クラス**：ユーザーが訪問済みのリンクをターゲットにします。
- **`:target` 疑似クラス**：URLフラグメントのターゲットとなっている要素にスタイルを適用します。

## ツリー構造の疑似クラス

- **ツリー構造の疑似クラス**：ドキュメントツリー内の位置に基づいて要素をターゲットにし、スタイルを適用できます。
- **`:root` 疑似クラス**：通常はルートの`html` 要素を指します。ドキュメント全体に共通のスタイルを適用するのに役立ちます。        
- **`:empty` 疑似クラス**：空の要素（空白以外の子要素を持たない要素）もドキュメントツリーに含まれます。そのため空の要素をターゲットにする`:empty` 疑似クラスがあります。
- **`:nth-child(n)` 疑似クラス**：親要素内での位置に基づいて要素を選択できます。
- **`:nth-last-child(n)` 疑似クラス**：末尾から数えて要素を選択できます。
- **`:first-child` 疑似クラス**：親要素またはドキュメント内の最初の要素を選択します。
- **`:last-child` 疑似クラス**：親要素またはドキュメント内の最後の要素を選択します。
- **`:only-child` 疑似クラス**：親要素またはドキュメント内で唯一の要素を選択します。
- **`:first-of-type` 疑似クラス**：親内で特定の要素タイプの最初の出現を選択します。
- **`:last-of-type` 疑似クラス**：親内で特定の要素タイプの最後の出現を選択します。
- **`:nth-of-type(n)` 疑似クラス**：同じタイプの兄弟要素の中での位置に基づいて特定の要素を選択できます。
- **`:only-of-type` 疑似クラス**：親内でそのタイプの唯一の要素であれば選択します。

## 関数型疑似クラス

- **関数型疑似クラス**：より複雑な条件や関係に基づいて要素を選択できます。通常の疑似クラスは状態に基づいて要素をターゲットにしますが（例：`:hover`、`:focus`）、関数型疑似クラスは引数を受け取ります。
- **`:is()` 疑似クラス**：セレクターのリスト（例：`ol`、`ul`）を受け取り、そのリストのいずれかにマッチする要素を選択します。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<p class="example">This text will change color.</p>
<p>This text will not change color.</p>
<p>This text will not change color.</p>
<p class="this-works-too">This text will change color.</p>
```

```css
p:is(.example, .this-works-too) {
    color: red;
}
```

:::

- **`:where()` 疑似クラス**：セレクターのリスト（例：`ol`、`ul`）を受け取り、そのリストのいずれかにマッチする要素を選択します。`:is` と `:where` の違いは、後者は特異性が0になる点です。

```css
:where(h1, h2, h3) {
    margin: 0;
    padding: 0;
}
```

- **`:has()` 疑似クラス**：子要素がセレクターリストに指定された要素を含む場合にスタイルを適用できるため、`"parent"` セレクターと呼ばれることがあります。

```css
article:has(h2) {
    border: 2px solid hotpink;
}
```

- **`:not()` 疑似クラス**：指定されたセレクターにマッチしない要素を選択するために使います。

```css
p:not(.example) {
  color: blue;
}
```

## 疑似要素

- **`::before` 疑似要素**：`content` プロパティを使って、要素の直前にアイコンなどの装飾的なコンテンツを挿入します。
- **`::after` 疑似要素**：`content` プロパティを使って、要素の直後にアイコンなどの装飾的なコンテンツを挿入します。
- **`::first-letter` 疑似要素**：要素の内容の最初の文字をターゲットにしてスタイルを適用できます。
- **`::marker` 疑似要素**：リスト項目のマーカー（箇条書きや番号）を選択してスタイルを適用できます。
