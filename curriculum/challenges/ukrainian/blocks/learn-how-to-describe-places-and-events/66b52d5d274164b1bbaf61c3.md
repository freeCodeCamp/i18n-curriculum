# --explanation--

`Present Continuous` час використовується для опису дій, які відбуваються в момент мовлення. Він утворюється за допомогою дієслова `to be` (`am, is, are`) + початкової форми дієслова + `-ing`. Наприклад:

- `She is coding a new feature right now.` - Вона зараз кодує нову функцію.

- `They are testing the software.` - Вони зараз тестують програмне забезпечення.
