# --interactive--

當在 CSS 中處理背景影像時，你有多個屬性可用來控制這些影像的顯示方式。

我們將專注的主要屬性是 `background-size`、`background-repeat`、`background-position` 和 `background-attachment`。

讓我們先來看看 `background-image` 屬性：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

上述 CSS 為 `body` 元素設定了一個貓咪背景影像。

如果你想設定背景影像的大小，可以使用 `background-size` 屬性。

你可以使用 `contain` 將影像縮放到盡可能大而不裁切或拉伸。

以下是使用 `background-size: contain` 的範例：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    min-height: 100px;
  }
</style>
```

:::

我們將 `min-height` 設定為 `100px`，以便背景影像可見且版面維持基準高度，確保即使內容最少，設計仍顯得平衡且視覺上吸引人。

如果我們將 `background-size` 屬性更改為使用 `cover` 值，則背景影像會縮放以涵蓋整個 `body` 元素，同時維護其長寬比。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    min-height: 100px;
  }
</style>
```

:::

在前面的範例中，你可能注意到背景影像會持續重複。

預設情況下，背景影像會在水平方向和垂直方向重複，以填滿整個元素。然而，你可以控制此行為。

你可以使用 `background-repeat` 屬性並將值設為 `no-repeat`。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

當 `background-size` 設定為 `contain` 且 `background-repeat` 設定為 `no-repeat` 時，影像將不會在螢幕上重複。

如果你想要水平重複背景影像，可以為 `background-repeat` 屬性使用 `repeat-x` 值。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-x;
    min-height: 100px;
  }
</style>
```

:::

如果你想要垂直重複背景影像，可以為 `background-repeat` 屬性使用 `repeat-y` 值。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-y;
    min-height: 100px;
  }
</style>
```

:::

要在螢幕上定位背景影像，可以使用 `background-position` 屬性。

`background-position` 屬性允許你設定背景影像在元素中的顯示位置。你可以使用關鍵字如 `top`、`bottom`、`left`、`right` 和 `center`，或是特定的像素或百分比數值。

以下是使用 `center` 和 `top` 作為 `background-position` 的範例：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

這段 CSS 程式碼將背景影像水平置中並垂直置頂於元素。

最後，`background-attachment` 決定背景影像在頁面滾動時是隨內容一起滾動還是保持固定。

主要的值有 `scroll`（預設值），背景影像會隨內容捲動，以及 `fixed`，背景影像會固定在螢幕上的相同位置。

以下是使用 `background-attachment` 屬性中 `fixed` 值的範例：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-position: center top;
    background-attachment: fixed;
  }
</style>
```

:::

這段 CSS 程式碼設定背景影像在頁面捲動時保持固定位置。

如果你想將幾個屬性合併成一行，可以使用簡寫的 `background` 屬性來達成。

這裡有一個範例：

:::interactive_editor

```html
<style>
  body {
    background: center top fixed
      url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

上述範例將 `background-image`、`background-position` 和 `background-attachment` 結合成一行。

透過精通這些屬性，你將能夠很好地控制背景影像在你的網頁設計中的顯示方式，從而實現更具視覺吸引力和功能性的版面配置。
