# --description--

Wenn Sie das Aussehen von jemandem beschreiben, verwenden Sie oft `with` gefolgt von einer Funktion.

Beispiele:

`with glasses` für jemanden, der eine Brille trägt

`with curly hair` für jemanden mit lockigem Haar oder

`with a blue jacket` für jemanden, der eine blaue Jacke trägt

# --questions--
