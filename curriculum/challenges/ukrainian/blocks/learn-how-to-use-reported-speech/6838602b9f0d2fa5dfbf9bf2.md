# --explanation--

`Unusually` — це прислівник, який описує, що щось відбувається більше або менше, ніж зазвичай чи очікувалося. З іншого боку, `unusual` — це прикметник, який описує щось, що не є загальним або нормальним. Наприклад:

- `The server experienced an unusually high load today.` – У цьому реченні `unusually` стосується того, наскільки велике було навантаження (набагато більше за норму).

- `It was unusual for the server to slow down like that.` – У цьому реченні `unusual` описує ситуацію. Воно не змінює дієслово чи прикметник.

`query` — це питання або запит, надісланий до бази даних для отримання інформації або виконання дії. Наприклад:

`The application sent a query to the database to fetch user details.` – Це означає, що застосунок звернувся до бази даних за конкретною інформацією.
