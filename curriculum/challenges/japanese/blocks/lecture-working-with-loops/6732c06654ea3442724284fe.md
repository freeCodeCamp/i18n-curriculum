# --interactive--

前のレッスンでは、`for`ループ、`for...in`ループ、`for...of`ループの使い方を学びました。このレッスンでは、`while`ループと`do...while`ループについて学びます。

`while`ループは、条件がtrueの間、コードのブロックを実行し続けます。`while`ループの基本的な構文は次の通りです。

```js
while (condition) {
  // code block to be executed
}
```

条件はコードブロックが実行される前にチェックされます。条件がfalseの場合、コードブロックは実行されません。

`while`ループは、コードブロックを何回実行する必要があるかわからない場合に便利です。`while`ループを使った例を示します。

:::interactive_editor

```js
let counter = 0;
while(counter < 5) {
  console.log(counter);
  counter++;
}
```

:::

この例では、`counter`という変数が`0`で初期化されています。`while`ループは、`counter`の値が`5`より小さい間、実行され続けます。ループ内では、`counter`の値をコンソールに出力し、その後`counter`を`1`だけ増やします。

`while`ループに似たもう一つのループは`do...while`ループです。基本的な構文は次の通りです。

```js
do {
  // code block to be executed
} while (condition);
```

`do...while`ループと`while`ループの主な違いは、`do...while`ループは条件をチェックする前に少なくとも一度はコードブロックを実行することです。

条件がtrueの場合、コードブロックは実行を続けます。条件がfalseの場合、コードブロックの実行は停止します。

`do...while`ループを使った例を示します。

:::interactive_editor

```js
let counter = 0;
do {
  console.log(counter);
  counter++;
} while (counter < 5);  
```

:::

この例では、`counter`という変数が`0`で初期化されています。`do...while`ループは`counter`の値をコンソールに出力し、その後`counter`を`1`だけ増やします。コードブロックを実行した後、`counter`の値が`5`より小さいかどうかをチェックします。小さければループは続きます。そうでなければループは停止します。

ほとんどの場合、`while`ループを`do...while`ループよりもよく使うでしょう。しかし、両方のループの種類と使いどころを知っておくことは良いことです。

`while`ループと`do...while`ループの主な違いは何ですか？

## --answers--
