# --description--

考慮邊長為 $\sqrt{5}$、$\sqrt{65}$ 和 $\sqrt{68}$ 的三角形。可以證明此三角形的面積為 9。

$S(n)$ 是所有邊長為 $\sqrt{1 + b^2}$、$\sqrt{1 + c^2}$ 和 $\sqrt{b^2 + c^2}$（其中 $b$ 和 $c$ 為正整數）且面積為整數且不超過 $n$ 的三角形面積總和。

範例三角形有 $b = 2$ 和 $c = 8$。

$S({10}^6) = 18\,018\,206$。

尋找 $S({10}^{10})$。
