# --explanation--

`不是 (bù shì)` significa "no ser". `不 (bù)` significa "no" o "no". Se añade antes de un verbo o un adjetivo para indicar que algo no es así. `是 (shì)` significa "ser".

Esta palabra puede usarse para negar cosas como identidad, existencia y clasificación. Por ejemplo:

`我不是开发者 (wǒ bù shì kāi fā zhě)。` – No soy desarrollador.
