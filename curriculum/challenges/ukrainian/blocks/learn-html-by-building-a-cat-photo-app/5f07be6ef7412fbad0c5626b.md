# --description--

Перед тим як додати будь-який новий вміст, слід використати елемент `section`, щоб відокремити вміст із фотографіями котів від майбутнього вмісту.

Елемент `section` використовується для визначення розділів у документі, таких як глави, заголовки, нижні колонтитули або будь-які інші розділи документа. Це семантичний елемент, який допомагає з SEO та доступністю.

```html
<section>
  <h2>Section Title</h2>
  <p>Section content...</p>
</section>
```

Візьміть свої елементи `h2`, `p` та посилання (`a`) і вкладіть їх у елемент `section`.

# --hints--

Ваш елемент `section` повинен мати початковий тег. Початкові теги мають такий синтаксис: `<elementName>`.

```js
assert(document.querySelector('section'));
```

Ваш елемент `section` повинен мати кінцевий тег. Кінцеві теги мають `/` одразу після символу `<`.

```js
assert(code.match(/<\/section\s*>/i));
```

Весь елемент `section` повинен бути між початковим і кінцевим тегами елемента `main`.

```js
assert(document.querySelector('section').parentNode.nodeName === 'MAIN');
```

Існуючий елемент `h2`, два елементи `p` та посилання (`a`) повинні бути між початковим і кінцевим тегами елемента `section`.

```js
const childrenOfSection = [...document.querySelector('section').childNodes];
const foundElements = childrenOfSection.filter((child) => {
  return ['H2', 'A', 'P'].includes(child.nodeName);
});
assert.lengthOf(foundElements, 4);
```

Елемент `h1` не повинен бути вкладений у елемент `section`.

```js
const childrenOfSection = [...document.querySelector('section').childNodes];
const includesH1 = childrenOfSection.some((child) => child.nodeName === 'H1');
assert.isFalse(includesH1);
```
