# --interactive--

比較演算子は、2つの値を比較して `true` または `false` の結果を返します。その結果を使って判断を下したり、プログラムの流れを制御したりできます。比較は `if` 文やループ、そして特定の条件に基づいて判断が必要な多くの場面で使われます。さあ、最も一般的な比較演算子を見て、その動作を理解しましょう。

大なり演算子は、右向きの山括弧（`>`）で表され、左側の値が右側の値より大きいかどうかをチェックします。

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a > b); // false
console.log(b > a); // true
```

:::

大なりイコール演算子は、右向きの山括弧とイコール記号（`>=`）で表され、左側の値が右側の値より大きいか、または等しいかどうかをチェックします。

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a >= b); // false
console.log(b >= a); // true
console.log(a >= c); // true
```

:::

小なり演算子は、左向きの山括弧（`<`）で表され、`>` と似ていますが逆方向です。左側の値が右側の値より小さいかどうかをチェックします。

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a < b); // true
console.log(b < a); // false
```

:::

小なりイコール演算子は、左向きの山括弧とイコール記号（`<=`）で表され、左側の値が右側の値より小さいか、または等しいかどうかをチェックします。

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a <= b); // true
console.log(b <= a); // false
console.log(a <= c); // true
```

:::
