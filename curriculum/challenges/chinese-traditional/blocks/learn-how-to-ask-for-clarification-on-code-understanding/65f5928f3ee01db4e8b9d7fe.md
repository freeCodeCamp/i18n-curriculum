# --description--

在這個任務中，你將學習片語 `to be willing`。

當有人說他們 `willing` 做某事時，表示他們已經準備好並願意去做，通常是出於善意或合作。 例如，`Brian is willing to help Sophie` 意味著 Brian 已經準備好並樂意協助 Sophie。

這個片語經常用於專業和個人背景關係中，以表達對做某事或幫助某人的積極態度。
