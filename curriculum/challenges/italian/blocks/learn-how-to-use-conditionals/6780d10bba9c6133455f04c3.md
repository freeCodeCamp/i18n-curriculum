# --explanation--

`Set us back` significa ritardare il progresso o causare un passo indietro. Per esempio:

`The power outage set us back several hours.` - Questo significa che l’interruzione di corrente ha causato un ritardo nel loro lavoro.

Nell’audio, Bob usa `set us back` per spiegare che perdere dati critici ritarderebbe il loro progresso di settimane.
