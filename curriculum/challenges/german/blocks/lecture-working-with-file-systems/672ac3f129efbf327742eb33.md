# --description--

Lernen wir, wie Sie Dateien und Ordner unter Windows und macOS suchen können.

Wir beginnen mit Windows. Sie können entweder über die Taskleiste oder den Datei-Explorer suchen.

Um über die Taskleiste zu suchen, geben Sie den Namen der Datei oder des Ordners ein, den Sie suchen. Sie können auch nach Schlüsselwörtern suchen. Sie sehen Ergebnisse von Ihrem gesamten PC und OneDrive.

Wenn Sie nur Dokumente sehen möchten, können Sie auf den Tab „Dokumente“ klicken. Es gibt auch Tabs für Apps und Web-Ergebnisse.

Eine weitere Möglichkeit ist die Suche im Datei-Explorer. Sie finden sie oben rechts im Explorer-Fenster, wo „Dokumente durchsuchen“ steht.

Sie haben drei Optionen, um den Gültigkeitsbereich der Suche festzulegen:

- Sie können von „Start“ aus suchen, um schnell Dateien von Ihrem PC und aus der Cloud zu finden.

- Sie können von einem Ordner aus suchen, um Dateien zu finden, die darin gespeichert sind.

- Sie können von „Dieser PC“ aus suchen, was eine langsame, aber gründliche Suche ist.

So können Sie Dateien und Ordner unter Windows suchen. Nun sehen wir, wie Sie das unter macOS machen können.

Zuerst können Sie Spotlight verwenden, ein Tool, das Ihnen hilft, Dateien auf Ihrem Mac zu finden. Um Spotlight zu öffnen, gehen Sie zur Menüleiste oben rechts und suchen das Lupensymbol. Wenn Sie dieses Symbol nicht in der Menüleiste sehen, können Sie es in den Einstellungen des Kontrollzentrums hinzufügen.

Gehen Sie zu Ihren Einstellungen, filtern Sie nach „Kontrollzentrum“, klicken Sie auf diese Option und scrollen Sie dann nach unten, bis Sie die Option „Spotlight“ finden. Sie können auch eine Tastenkombination verwenden, um Spotlight zu öffnen: `Command + Spacebar`.

Wenn Sie Spotlight öffnen, sehen Sie die Spotlight-Suche, in der Sie jede Datei oder jeden Ordner durch Eingabe des Namens suchen können. Sobald Sie zu tippen beginnen, sehen Sie die Ergebnisse nach Kategorien gruppiert, darunter Vorschläge, Ordner, Präsentationen, Fotos, Dokumente und verwandte Suchanfragen.

Sie können auch eine bestimmte Kategorie erweitern, indem Sie auf „Mehr anzeigen“ klicken.

Eine weitere Möglichkeit, nach Dateien zu suchen, die Sie kürzlich geöffnet haben, ist, den Finder zu öffnen, indem Sie auf das Finder-Symbol im Dock klicken. Gehen Sie dann in der Finder-Seitenleiste zu „Zuletzt“. Dort finden Sie alle Dateien, die Sie kürzlich angesehen haben.

So können Sie Dateien und Ordner unter macOS suchen. Mit diesen Suchwerkzeugen können Sie Ihren Arbeitsablauf optimieren und die benötigten Dateien und Ordner sehr schnell finden.
