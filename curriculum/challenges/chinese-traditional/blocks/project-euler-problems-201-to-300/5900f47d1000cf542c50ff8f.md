# --description--

對於正整數 $n$，定義 $C(n)$ 為滿足 $1 < x < n$ 且 $x^3 \equiv 1\bmod n$ 的整數 $x$ 的數量。

當 $n = 91$ 時，$x$ 有 8 個可能的值，分別是：9、16、22、29、53、74、79、81。因此，$C(91) = 8$。

找出正整數 $n ≤ {10}^{11}$ 中，滿足 $C(n)=242$ 的數字總和。
