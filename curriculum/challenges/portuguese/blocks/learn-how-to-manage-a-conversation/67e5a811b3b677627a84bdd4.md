# --explanation--

`Might` é usado para mostrar que algo é possível, mas não certo.Jessica está dizendo que o bug *poderia* causar um problema, mas isso ainda não foi confirmado.Compare isso com o uso do `Simple Present`, `we've discovered a critical bug that affects the user interface`, o que significa que o bug definitivamente causa problemas.Outros exemplos:

- `The delay might cause problems.` – É possível, mas não certo.

- `The delay causes problems.` – Definitivamente cria problemas.
