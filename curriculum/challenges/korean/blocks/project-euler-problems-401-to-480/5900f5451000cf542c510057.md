# --description--

한 줄에 $N$개의 좌석이 있습니다. $N$명의 사람이 다음 규칙에 따라 차례로 좌석을 채웁니다:

1. 어떤 사람도 다른 사람 옆에 앉지 않습니다.
1. 첫 번째 사람은 아무 좌석이나 선택합니다.
1. 이후 각 사람은 이미 앉아 있는 사람들과 가장 멀리 떨어진 좌석을 선택하는데, 규칙 1을 위반하지 않는 범위 내에서 선택합니다. 조건을 만족하는 좌석이 여러 개라면 가장 왼쪽 좌석을 선택합니다.

규칙 1 때문에 일부 좌석은 반드시 비어 있게 되며, $N > 1$일 때 앉을 수 있는 최대 인원은 $N$보다 적습니다.

$N = 15$일 때 가능한 좌석 배치는 다음과 같습니다:

<img alt="seating arrangements for N = 15" src="https://cdn.freecodecamp.org/curriculum/project-euler/comfortable-distance-ii.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

첫 번째 사람이 올바르게 선택하면 15개의 좌석에 최대 7명이 앉을 수 있음을 알 수 있습니다. 또한 첫 번째 사람은 최대 인원을 앉히기 위해 9가지 선택지가 있습니다.

$f(N)$을 한 줄에 $N$개의 좌석이 있을 때 첫 번째 사람이 최대 인원을 앉히기 위해 선택할 수 있는 좌석 수라고 합시다. 따라서 $f(1) = 1$, $f(15) = 9$, $f(20) = 6$, $f(500) = 16$입니다.

또한 $1 ≤ N ≤ 20$일 때 $\sum f(N) = 83$이고, $1 ≤ N ≤ 500$일 때 $\sum f(N) = 13\,343$입니다.

$1 ≤ N ≤ {10}^{12}$일 때 $\sum f(N)$을 구하세요. 답의 마지막 8자리를 제출하세요.
