# --description--

Ora che hai una conoscenza di base sugli array, parliamo dei loop. I loop si usano per eseguire un blocco di codice più volte. Uno di questi loop è il loop `while`. Il loop `while` esegue un blocco di codice finché la condizione è vera. La sintassi del loop `while` è la seguente:

```javascript
while (condition) {
  // code block to be executed
}
```

Per esempio, il seguente frammento di codice stampa i numeri da 1 a 5:

```javascript
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
```

Il frammento di codice sopra inizializza una variabile `i` con il valore `1`. Il loop `while` esegue il blocco di codice finché il valore di `i` è minore o uguale a `5`. Il valore di `i` viene incrementato di `1` a ogni iterazione.
