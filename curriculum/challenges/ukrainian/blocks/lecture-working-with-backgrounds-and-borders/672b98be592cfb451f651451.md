# --interactive--

Фонова градієнт у CSS — це плавний перехід між двома або більше кольорами, який можна застосувати до фону елемента. Градієнти дозволяють створювати привабливі фони без потреби використовувати зображення.

Існує два основні типи градієнтів у CSS: лінійні градієнти та радіальні градієнти.

Лінійний градієнт переходить між кольорами вздовж прямої лінії. Ви можете визначити напрямок і кольори, які беруть участь.

Ось базовий синтаксис:

```css
background: linear-gradient(direction, color-stop1, color-stop2, ...);
```

У цьому прикладі ми використовуємо властивість `background` CSS зі значенням `linear gradient`.

Напрямок визначає напрям градієнта. Це може бути кут (наприклад, `45deg`), ключове слово (наприклад, `to right`, `to bottom`) або сторона/кут.

`color-stop` визначає кольори та позиції, де відбуваються переходи градієнта.

Щоб краще зрозуміти, як працюють лінійні градієнти, розглянемо наступний приклад:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient{
  background: linear-gradient(to right, red, yellow);
  height: 40vh;
}
```

:::

Цей CSS створює лінійний градієнт, який переходить від `red` на `left` до `yellow` на `right`. Градієнт застосовується до елемента з висотою `40%` від висоти вікна перегляду. Ви дізнаєтеся більше про одиниці `vh` у майбутньому уроці.

Напрямок `to right` означає, що градієнт проходить горизонтально зліва направо.

Іншим типом градієнта є `radial` градієнт.

Радіальний градієнт переходить між кольорами, що розходяться від початкової точки (зазвичай центру) назовні у круглій або еліптичній формі.

Ось базовий синтаксис:

```css
background: radial-gradient(shape size at position, color-stop1, color-stop2, ...)
```

У синтаксисі `shape` визначає форму градієнта, яка може бути `circle` або `ellipse`.

`size` визначає розмір кінцевої форми градієнта, який може бути `closest-side`, `closest-corner`, `farthest-side` або `farthest-corner`.

`position` визначає позицію центру градієнта, яку можна вказати за допомогою ключових слів (наприклад, `center`, `top left`, `bottom right`) або точних значень (наприклад, `50% 50%`, `10px 20px`).

Нарешті, зупинки кольорів — це список кольорів, через які проходить градієнт. Кожна зупинка кольору може необов’язково містити значення позиції (відсоток або довжину), що вказує, де саме має розташовуватися цей колір.

Приклад:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient{
  background: radial-gradient(circle closest-side at center, red, yellow 50%, green);
  height: 60vh;
}
```

:::

Цей CSS створює круговий радіальний градієнт, центрований у елементі. Він починається з `red` у центрі, переходить до `yellow` на `50%` радіуса і завершується `green`.

Ключове слово `closest-side` змушує кінцеву форму градієнта відповідати найближчій стороні елемента. Градієнт застосовується до елемента з висотою `60%` від висоти вікна перегляду.

Розуміння роботи з CSS-градієнтами може значно покращити ваші дизайни, надаючи привабливі фони без потреби у зображеннях.

З такими варіантами, як лінійні градієнти для плавних переходів і радіальні градієнти для кругових ефектів, вони пропонують і гнучкість, і творчість у вебдизайні.
