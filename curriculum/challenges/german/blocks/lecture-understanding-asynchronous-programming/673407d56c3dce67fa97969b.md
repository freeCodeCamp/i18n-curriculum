# --interactive--

In den vorherigen Lektionen haben Sie asynchrones Programmieren kennengelernt, das es ermöglicht, dass anderer Code ausgeführt wird, während wir auf zeitaufwändige Aufgaben warten, wie das Abrufen von Daten von einem Server, das Lesen von Daten aus einer Datei und so weiter.

`async`/`await`, aufgebaut auf Promises, macht das Schreiben und Lesen von asynchronem Code einfacher. Wenn Sie das `async`-Schlüsselwort vor eine Funktion setzen, bedeutet das, dass diese Funktion immer ein `Promise` zurückgibt. Das `await`-Schlüsselwort, mit dem Sie auf die Auflösung eines `Promise` warten können, bevor Sie mit der nächsten Codezeile fortfahren, kann nur innerhalb asynchroner Funktionen oder im obersten Modulbereich verwendet werden. Hier ist ein Beispiel, das zeigt, wie `async`/`await` funktioniert:

:::interactive_editor

```js
async function delayedGreeting(name) {
  console.log("A Messenger entered the chat...");
  await new Promise(resolve => setTimeout(resolve, 2000));
  console.log(`Hello, ${name}!`);
}

delayedGreeting("Alice");
console.log("First Printed Message!");
```

:::

In diesem Code definieren wir eine `async`-Funktion namens `delayedGreeting`. Innerhalb dieser Funktion verwenden wir `await`, um die Ausführung für 2 Sekunden zu pausieren. Nach der Verzögerung wird eine Begrüßung ausgegeben. 

Wenn wir diese Funktion aufrufen, sehen Sie `First Printed Message!` vor der Begrüßung erscheinen. Das liegt daran, dass die Funktion asynchron ist – sie blockiert nicht den Rest des Codes.

Einer der größten Vorteile von `async`/`await` ist die Fehlerbehandlung. Bei Promises mussten wir oft die `.catch()`-Methode verwenden, um Fehler zu behandeln. Mit `async`/`await` können wir `try`/`catch`-Blöcke verwenden. Hier ein Beispiel:

```js
async function fetchUserData() {
  try {
    let response = await fetch(`https://api.example.com/users`);
    let userData = await response.json();
    console.log(userData);
  } catch (error) {
    console.log("Error fetching user data:", error);
  }
}

fetchUserData();
```

In diesem Beispiel verwenden wir `async`/`await`, um Benutzerdaten von einer API abzurufen. Das `await`-Schlüsselwort wird zweimal verwendet: einmal, um auf den Abschluss des Fetch-Vorgangs zu warten, und ein weiteres Mal, um auf das Parsen des JSON zu warten. Wenn während dieses Prozesses ein Fehler auftritt, wird er im `catch`-Block abgefangen.
