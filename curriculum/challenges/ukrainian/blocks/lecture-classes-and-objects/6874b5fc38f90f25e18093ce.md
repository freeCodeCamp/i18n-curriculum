# --description--

У попередньому уроці ви дізналися про класи та як вони слугують шаблонами для створення об’єктів.

Тут ми заглибимося в атрибути та методи.

Спочатку розглянемо атрибути, а потім методи.

Атрибути — це змінні, які належать об’єкту, тому вони зберігають дані. Існують два типи атрибутів: атрибути екземпляра та атрибути класу.

Атрибути екземпляра унікальні для кожного об’єкта, створеного з класу, і зазвичай ви встановлюєте їх за допомогою методу `__init__`. Атрибути класу, навпаки, належать самому класу і спільні для всіх екземплярів цього класу.

Щоб отримати доступ до атрибуту, ви використовуєте точкову нотацію.

Ось приклади як атрибутів екземпляра, так і атрибутів класу, а також як отримати до них доступ з об’єктів:

```python
class Dog:
    species = "French Bulldog" # Class attribute

    def __init__(self, name):
        self.name = name # Instance attribute

print(Dog.species) # French Bulldog

dog1 = Dog("Jack")
print(dog1.name)    # Jack
print(dog1.species) # French Bulldog

dog2 = Dog("Tom")
print(dog2.name)    # Tom
print(dog2.species) # French Bulldog
```

Зверніть увагу, що ви можете отримати доступ до атрибутів класу безпосередньо з самого класу, але для доступу до атрибутів екземпляра потрібно спочатку створити об’єкт і передати йому дані.

Автомобілі — ще один хороший приклад, адже всі автомобілі мають модель і колір:

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

car_1 = Car("red", "Toyota Corolla")
car_2 = Car("green", "Lamborghini Revuelto")

print(car_1.model) # Toyota Corolla
print(car_2.model) # Lamborghini Revuelto

print(car_1.color) # red
print(car_2.color) # green
```

Методи — це функції, визначені всередині класу. За їх допомогою будь-який об’єкт, створений з класу, може виконувати дії, які працюють з його власними даними або змінюють їх. Ви також отримуєте доступ до методу за допомогою точкової нотації.

Наприклад, собаки можуть гавкати. Тож у класі `bark` ми можемо мати метод `Dog`, як ви бачили в попередньому уроці:

```python
class Dog:
   species = "French Bulldog"

   def __init__(self, name):
     self.name = name

   def bark(self):
       return f"{self.name} says woof woof!"

jack = Dog("Jack")
jill = Dog("Jill")

print(jack.bark()) # Jack says woof woof!
print(jill.bark()) # Jill says woof woof!
```

Клас `Car` також може мати метод `describe`:

```python
class Car:
    def __init__(self, color, model):
        self.color = color  # Instance attribute
        self.model = model  # Instance attribute

    def describe(self):
        return f"This car is a {self.color} {self.model}"

car_1 = Car("red", "Toyota Corolla")
car_2 = Car("green", "Lamborghini Revuelto")

print(car_1.describe()) # This car is a red Toyota Corolla
print(car_2.describe()) # This car is a green Lamborghini Revuelto
```
