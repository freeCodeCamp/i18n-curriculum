# --interactive--

多言語ウェブサイトを構築したり、カスタムデータ属性を扱ったりするとき、含まれる言語や特定のデータ値に基づいて要素のスタイルを設定する必要があります。

`lang` と `data-lang` 属性はこれらの目的でよく使われ、CSSの属性セレクターを使うとこれらの属性に基づいて効果的にスタイルを適用できます。

`lang` 属性はHTMLで要素内のコンテンツの言語を指定するために使われます。特に多言語サイトでは、書かれている言語に応じて要素のスタイルを変えたい場合があります。

以下は、特定の`lang`属性を持つ要素を属性セレクターで指定する例です。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">This is an English paragraph.</p>
<p lang="fr">Ceci est un paragraphe en français.</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

このCSSルールは、`lang`属性が英語（`en`）に設定されている段落要素に斜体のスタイルを適用します。複数言語を含む文書で英語のテキストを強調したい場合に便利です。

同様に、`data-lang`属性を持つ要素を属性セレクターで指定できます。

`data-lang`のようなカスタムデータ属性は、特定のテキストセクション内で使われている言語を指定するなど、要素に追加情報を保存するためによく使われます。

以下は、`data-lang`属性に基づいて要素を指定しスタイルを適用するCSSの例です。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p data-lang="fr">Ceci est un paragraphe en français.</p>
<p data-lang="en">This is a paragraph in English.</p>
```

```css
p[data-lang="fr"] {
  color: blue;
}
```

:::

この場合、`p`要素で`data-lang`属性がフランス語（`fr`）に設定されているものは、文字色が青に変わります。これにより、`data-lang`属性に保存された言語情報に基づいてコンテンツのセクションを素早く識別しスタイルを適用できます。

このような属性セレクターは、HTMLに埋め込まれたメタデータに基づいて条件付きのスタイルを適用する強力な方法を提供し、ウェブページをより動的で文脈に応じたものにします。
