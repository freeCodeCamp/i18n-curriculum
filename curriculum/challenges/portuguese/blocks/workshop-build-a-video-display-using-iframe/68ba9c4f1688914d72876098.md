# --description--

O próximo atributo que você vai adicionar é `referrerpolicy`. É a regra que determina quanta informação você compartilha quando sua página se conecta a outra página.

Adicione o atributo `referrerpolicy` e defina-o como `strict-origin-when-cross-origin`. Isso compartilha o endereço completo no mesmo site, apenas o nome do site em outros sites e nada em sites inseguros.

# --hints--

Seu elemento `iframe` deve ter um atributo `referrerpolicy` definido como `strict-origin-when-cross-origin`.

```js
const iframeEl = document.querySelector('iframe')
assert.equal(iframeEl?.getAttribute('referrerpolicy'), 'strict-origin-when-cross-origin')
```
