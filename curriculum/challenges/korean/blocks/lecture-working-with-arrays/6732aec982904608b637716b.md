# --interactive--

JavaScript의 배열은 동적이어서 요소를 쉽게 추가하거나 제거할 수 있습니다. 배열의 시작과 끝에서 요소를 추가하고 제거하는 주요 메서드는 `push()`, `pop()`, `shift()`, `unshift()` 네 가지입니다. 각 메서드를 자세히 살펴보겠습니다.

`push()` 메서드는 배열의 끝에 하나 이상의 요소를 추가하는 데 사용됩니다. `push()` 메서드의 반환값은 배열의 새로운 길이입니다. 다음은 기존 `fruits` 배열에 새로운 과일을 추가하는 예입니다:

:::interactive_editor

```js
const fruits = ["apple", "banana"];
const newLength = fruits.push("orange");
console.log(newLength); // 3
console.log(fruits); // ["apple", "banana", "orange"]
```

:::

이 예제에서는 두 개의 요소를 가진 `fruits` 배열로 시작합니다. 그런 다음 `push()` 메서드를 사용해 문자열 `orange`를 배열 끝에 추가합니다.

`const`를 사용해 `fruits` 배열을 선언한 것을 눈치챘을 것입니다. 그런데 `fruits`가 상수인데도 왜 이 `fruits` 배열에 더 많은 요소를 추가할 수 있을까요? 이는 `const` 키워드로 배열을 선언하면 배열에 대한 참조가 생성되기 때문입니다. 배열 자체는 변경 가능하지만, `fruits` 상수에 새 값을 재할당할 수는 없습니다. 예를 들어 다음과 같습니다:

```js
const fruits = ["apple", "banana"];
fruits = ["This", "will", "not", "work"];
console.log(fruits); // Uncaught TypeError: Assignment to constant variable. 
```

다음으로 살펴볼 메서드는 `pop()` 메서드입니다. `pop()` 메서드는 배열의 마지막 요소를 제거하고 그 요소를 반환합니다. 또한 원래 배열도 수정합니다. 작동 방식은 다음과 같습니다:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let lastFruit = fruits.pop();
console.log(fruits); // ["apple", "banana"]
console.log(lastFruit); // "orange"
```

:::

이 예제에서는 세 개의 과일이 들어 있는 배열로 시작합니다. `pop()` 메서드는 배열에서 마지막 요소(`orange`)를 제거하고 반환합니다. 원래 `fruits` 배열은 수정되어 두 개의 요소만 포함합니다.

`unshift()` 메서드는 배열의 시작 부분에 하나 이상의 요소를 추가하고 새로운 길이를 반환합니다. `push()`와 비슷하지만 배열 끝이 아니라 시작 부분을 수정합니다. 예를 들어:

:::interactive_editor

```js
let numbers = [2, 3];
let newLength = numbers.unshift(1);
console.log(numbers); // [1, 2, 3]
console.log(newLength); // 3
```

:::

이 예제에서는 `unshift()`을 사용해 `1` 배열의 시작에 숫자 `numbers`를 추가합니다. 메서드는 배열의 새로운 길이인 `3`을 반환합니다.

마지막으로 `shift()` 메서드는 배열의 첫 번째 요소를 제거하고 그 요소를 반환합니다. `pop()`와 비슷하지만 배열 끝이 아니라 시작 부분에서 작동합니다. 작동 방식은 다음과 같습니다:

:::interactive_editor

```js
let colors = ["red", "green", "blue"];
let firstColor = colors.shift();
console.log(colors); // ["green", "blue"]
console.log(firstColor); // "red"
```

:::

이 예제에서는 세 가지 색상이 들어 있는 배열로 시작합니다. `shift()` 메서드는 배열에서 첫 번째 요소(`red`)를 제거하고 반환합니다. 원래 `colors` 배열은 수정되어 두 개의 요소만 포함합니다.

`push()`와 `unshift()`는 한 번에 여러 요소를 추가할 수 있지만, `pop()`와 `shift()`는 한 번에 하나의 요소만 제거한다는 점을 기억하세요.
