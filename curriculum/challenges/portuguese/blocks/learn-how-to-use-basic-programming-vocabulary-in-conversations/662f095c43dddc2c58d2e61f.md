# --description--

A palavra `super` é frequentemente usada informalmente antes de um adjetivo para intensificar seu significado, semelhante a `very` ou `extremely`, para realçar o significado do adjetivo que segue.

Por exemplo, dizer `this book is super interesting` significa que o livro é extremamente interessante, mais do que apenas interessante.

`Helpful` descreve alguém ou algo que fornece assistência ou suporte valioso.Juntos, `super helpful` significa fornecer assistência excepcional ou ser extremamente útil.
