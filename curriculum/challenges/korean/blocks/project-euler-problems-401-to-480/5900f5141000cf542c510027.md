# --description--

$n$을 양의 정수라고 하겠습니다.

6면체 주사위를 $n$번 던집니다. $c$는 연속된 던지기 쌍 중에서 같은 값이 나오는 쌍의 개수입니다.

예를 들어, $n = 7$이고 주사위 던지기 값이 (1, 1, 5, 6, 6, 6, 3)이라면, 다음과 같은 연속된 던지기 쌍이 같은 값을 가집니다:

$$\begin{align}
  & (\underline{1}, \underline{1}, 5, 6, 6, 6, 3) \\\\
  & (1, 1, 5, \underline{6}, \underline{6}, 6, 3) \\\\
  & (1, 1, 5, 6, \underline{6}, \underline{6}, 3)
\end{align}$$

따라서 (1, 1, 5, 6, 6, 6, 3)에 대해 $c = 3$입니다.

$C(n)$을 6면체 주사위를 $n$번 던졌을 때 $c$가 $π(n)$을 초과하지 않는 결과의 개수로 정의합니다.<sup>1</sup>

예를 들어, $C(3) = 216$, $C(4) = 1290$, $C(11) = 361\\,912\\,500$, 그리고 $C(24) = 4\\,727\\,547\\,363\\,281\\,250\\,000$입니다.

$S(L)$을 $1 ≤ n ≤ L$인 모든 $n$에 대해 $C(n)$의 합으로 정의합니다.

예를 들어, $S(50)\bmod 1\\,000\\,000\\,007 = 832\\,833\\,871$입니다.

$S(50\\,000\\,000)\bmod 1\\,000\\,000\\,007$을 구하세요.

<sup>1</sup> $π$는 소수 개수 함수로, $π(n)$은 $n$ 이하의 소수 개수입니다.
