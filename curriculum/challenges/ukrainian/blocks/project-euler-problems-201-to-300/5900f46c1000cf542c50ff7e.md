# --description--

Татамі — це прямокутні мати, які використовують для повного покриття підлоги кімнати без накладань.

Припускаючи, що єдиний доступний тип татамі має розміри 1×2, очевидно, що існують певні обмеження щодо форми та розміру кімнат, які можна покрити.

Для цього завдання ми розглядаємо лише прямокутні кімнати з цілими розмірами $a$, $b$ та парним розміром $s = a \times b$. Під терміном «розмір» ми розуміємо площу підлоги кімнати, і — без втрати загальності — додаємо умову $a ≤ b$.

Існує одне правило при укладанні татамі: не повинно бути точок, де зустрічаються кути чотирьох різних матів. Наприклад, розглянемо два варіанти укладання для кімнати 4×4:

<img alt="two arrangements of mats in 4x4 room" src="https://cdn.freecodecamp.org/curriculum/project-euler/tatami-free-rooms.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Укладання зліва прийнятне, а зправа — ні: червона "<strong><span style="color: red;">X</span></strong>" посередині позначає точку, де зустрічаються чотири татамі.

Через це правило деякі кімнати з парною площею не можна покрити татамі: ми називаємо їх кімнатами без татамі. Далі визначаємо $T(s)$ як кількість кімнат без татамі розміру $s$.

Найменша кімната без татамі має розмір $s = 70$ та розміри 7×10. Всі інші кімнати розміру $s = 70$ можна покрити татамі; це: 1×70, 2×35 та 5×14. Отже, $T(70) = 1$.

Аналогічно, можна перевірити, що $T(1320) = 5$, бо існує рівно 5 кімнат без татамі розміру $s = 1320$: 20×66, 22×60, 24×55, 30×44 та 33×40. Насправді, $s = 1320$ — це найменший розмір кімнати $s$, для якого $T(s) = 5$.

Знайдіть найменший розмір кімнати $s$, для якого $T(s) = 200$.
