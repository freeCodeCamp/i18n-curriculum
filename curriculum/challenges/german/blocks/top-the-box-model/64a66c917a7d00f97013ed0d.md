# --description--

Das erste wichtige Konzept, das Sie verstehen müssen, um in CSS erfolgreich zu sein, ist das Box-Modell. Es ist nicht kompliziert, aber wenn Sie es jetzt überspringen, wird es später zu viel Frustration führen.

Jedes einzelne Element auf einer Webseite ist eine rechteckige Box. Diese Boxen können andere Boxen enthalten und nebeneinander stehen. Sie können sich eine grobe Vorstellung davon machen, wie das funktioniert, indem Sie jedem Element auf der Seite einen Rahmen geben, so wie hier:

```css
* {
  border: 1px solid red;
}
```

# --assignment--

Fügen Sie jedem Element auf der Seite einen Rahmen hinzu und sehen Sie, wie die Boxen angeordnet sind.
