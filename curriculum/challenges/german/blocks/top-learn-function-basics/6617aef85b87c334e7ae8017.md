# --description--

Pfeilfunktionen sind eine modernere Art, Funktionen in JavaScript zu schreiben. Sie bieten eine kompaktere Syntax im Vergleich zu traditionellen Funktionsausdrücken. Pfeilfunktionen werden mit der `=>`-Syntax definiert.

```js
const add = (a, b) => {
  return a + b;
}
```

Im obigen Beispiel nimmt die Pfeilfunktion `add` zwei Parameter `a` und `b` entgegen und gibt deren Summe zurück. Das Schlüsselwort `return` wird verwendet, um das Ergebnis der Addition zurückzugeben.

Wenn die Pfeilfunktion nur einen Ausdruck enthält, können die geschweiften Klammern `{}` und das Schlüsselwort `return` weggelassen werden. Der Ausdruck wird implizit zurückgegeben.

```js
const add = (a, b) => a + b;
```

Im obigen Beispiel nimmt die Pfeilfunktion `add` zwei Parameter `a` und `b` entgegen und gibt deren Summe zurück. Das Schlüsselwort `return` und die geschweiften Klammern `{}` werden weggelassen, und das Ergebnis der Addition wird implizit zurückgegeben.
