## Об’єкт `date()` і загальні методи

- **Визначення**: Об’єкт `date()` використовується для створення, маніпулювання та форматування дат і часу в JavaScript. У наступному прикладі ключове слово `new` використовується для створення нового екземпляра об’єкта `Date`, а об’єкт `Date` призначається змінній `now`. Якщо ви виведете значення `now` у консоль, побачите поточну дату і час на основі системного годинника комп’ютера, що виконує код.

:::interactive_editor

```js
const now = new Date();
console.log(now); // prints the current date and time
```

:::

- **Метод `Date.now()`**: Цей метод використовується для отримання поточної дати і часу. `Date.now()` повертає кількість мілісекунд, що минули з 1 січня 1970 року, 00:00:00 UTC. Це відоме як час Unix-епохи. Час Unix-епохи — це загальний спосіб представлення дат і часу в комп’ютерних системах, оскільки це ціле число, яке легко зберігати і обробляти. UTC означає Всесвітній координований час, який є основним стандартом часу, за яким світ регулює годинники і час.
- **Метод `getDate()`**: Цей метод використовується для отримання дня місяця на основі поточної дати. `getDate()` поверне ціле число від 1 до 31, залежно від дня місяця. Якщо дата недійсна, він поверне `NaN` (Not a Number).

:::interactive_editor

```js
const now = new Date("2014-10-15");
const date = now.getDate();
console.log(date); // 15
```

:::

- **Метод `getMonth()`**: Цей метод використовується для отримання місяця. Місяць починається з нуля, тому січень — це 0, лютий — 1 і так далі. У цьому прикладі вихідні дані — 9, що відповідає жовтню. Якщо місяць недійсний, він поверне `NaN`.

:::interactive_editor

```js
const now = new Date("2014-10-15");
const month = now.getMonth();
console.log(month); // 9
```

:::

- **Метод `getFullYear()`**: Цей метод використовується для отримання повного року. Якщо рік недійсний, він поверне `NaN`.

:::interactive_editor

```js
const now = new Date("2014-10-15");
const year = now.getFullYear();
console.log(year); // 2014
```

:::

## Різні способи форматування дат

- **Метод `toISOString()`**: Цей метод використовується для форматування дати у розширеному форматі `ISO` (ISO 8601). ISO 8601 — це міжнародний стандарт для представлення дат і часу. Формат такий: `YYYY-MM-DDTHH:mm:ss.sssZ`.

:::interactive_editor

```js
const date = new Date("2014-10-15");
console.log(date.toISOString()); // "2014-10-15T00:00:00.000Z"
```

:::

- **Метод `toLocaleDateString()`**: Цей метод використовується для форматування дати відповідно до локалі користувача.

:::interactive_editor

```js
const date = new Date("2014-10-15");
console.log(date.toLocaleDateString("en-US"));  // "10/15/2014"
```

:::

Метод `toLocaleDateString()` приймає два необов’язкові параметри: locales і options.

Параметр locales — це рядок, що представляє локаль для використання. Наприклад, ви можете передати `"en-US"` для англійської (США) або `"fr-FR"` для французької (Франція). Якщо ви не передасте параметр locales, використовується локаль за замовчуванням. Другий необов’язковий параметр — options. Цей параметр є об’єктом, який дозволяє вказати формат рядка дати.

:::interactive_editor

```js
const date = new Date("2014-10-15");
const options = {
  weekday: "long",
  year: "numeric",
  month: "long",
  day: "numeric",
};
console.log(date.toLocaleDateString("en-GB", options)); // "Wednesday, 15 October 2014"
```

:::
