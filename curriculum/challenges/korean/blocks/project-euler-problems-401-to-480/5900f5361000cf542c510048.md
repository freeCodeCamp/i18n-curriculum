# --description--

$f(n) = n^2 - 3n - 1$이라고 하겠습니다.

$p$를 소수라고 하겠습니다.

$f(n) \bmod p^2 = 0$이 되는 가장 작은 양의 정수 $n$이 존재하면, 그 값을 $R(p)$라고 하고, 존재하지 않으면 $R(p) = 0$이라고 하겠습니다.

$L$ 이하의 모든 소수에 대해 $R(p)$를 합한 값을 $SR(L)$이라고 하겠습니다.

$SR({10}^7)$을 구하세요.
