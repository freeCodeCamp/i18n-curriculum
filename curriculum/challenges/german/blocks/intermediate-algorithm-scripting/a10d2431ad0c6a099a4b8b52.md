# --description--

Prüfen Sie, ob das Prädikat (zweites Argument) für alle Elemente einer Sammlung (erstes Argument) <dfn>truthy</dfn> ist.

Mit anderen Worten: Sie erhalten ein Array mit Objekten. Das Prädikat `pre` wird eine Objekteigenschaft sein, und Sie müssen `true` zurückgeben, wenn ihr Wert `truthy` ist. Andernfalls geben Sie `false` zurück.

In JavaScript sind `truthy`-Werte Werte, die in einem Boolean-Kontext zu `true` ausgewertet werden.

Denken Sie daran, dass Sie auf Objekteigenschaften entweder über Punktnotation oder `[]`-Notation zugreifen können.

# --solutions--

```js
function truthCheck(collection, pre) {
  return collection.every(function(e) { return e[pre]; });
}
```
