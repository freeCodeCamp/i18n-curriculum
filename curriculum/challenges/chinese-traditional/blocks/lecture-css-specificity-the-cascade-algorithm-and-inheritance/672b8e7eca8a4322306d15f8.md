# --interactive--

通用選擇器（`*`）是一種特殊型別的 CSS 選擇器，可匹配文件中的任何元素。

它通常用於將樣式套用到頁面上的所有元素，這對於重設或標準化不同瀏覽器之間的樣式非常有用。

通用選擇器可用於選取特定背景關係內的所有元素，或全域地跨整個文件選取所有元素。

以下是使用通用選擇器為整個 HTML 文件設定 `margin` 和 `padding` 的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1>Heading element</h1>
<p>example paragraph element</p>
```

```css
* {
  margin: 0;
  padding: 0;
}
```

:::

在這個程式碼範例中，`*` 選擇器將所有元素的外距和內距重設為零，這是 CSS 重設中常用的技巧。

通用選擇器具有所有選擇器中最低的特異性值。它對特異性值的所有零件貢獻為 0 `(0, 0, 0, 0)`。

這表示任何其他選擇器，包括型別選擇器、類別選擇器、ID 選擇器和行內樣式，都會改寫由通用選擇器設定的樣式。

讓我們來看看以下的 HTML 和 CSS 範例：

:::interactive_editor

```html
<head>
  <style>
    * {
      color: blue;
    }
    p {
      color: red;
    }
    .highlight {
      color: green;
    }
    #unique {
      color: purple;
    }
  </style>
</head>
<body>
  <p id="unique" class="highlight">This text has multiple styles applied.</p>
</body>
```

:::

通用的 `*` 選擇器將所有元素的文字顏色設為 `blue`。

然而，`p` 型別選擇器透過將文字顏色專門設定為 `red`，來改寫此設定，僅針對 `p` 元素。

如果一個元素具有 `highlight` 類別，類別選擇器具有優先權，將文字顏色改為 `green`。

最後，具有最高特異性的 `#unique` ID 選擇器會改寫所有其他選擇器，將文字顏色設定為 `purple`。
