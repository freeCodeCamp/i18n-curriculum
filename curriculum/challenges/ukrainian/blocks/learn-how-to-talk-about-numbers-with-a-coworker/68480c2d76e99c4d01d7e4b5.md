# --explanation--

`User experience` (або `UX`) описує, як користувач почувається, користуючись продуктом — чи це легко, приємно та плавно. Наприклад:

`Improving page load time boosts user experience.` — Прискорення завантаження сторінок допомагає користувачам більше насолоджуватися застосунком.

`Enhancements` — це зміни, внесені для покращення продукту чи процесу. Наприклад:

`The latest update includes performance enhancements.` — Нова версія робить застосунок кращим і швидшим.

У реченні Аліси половина бюджету на дизайн витрачається на те, щоб зробити продукт простішим і зручнішим у використанні.
