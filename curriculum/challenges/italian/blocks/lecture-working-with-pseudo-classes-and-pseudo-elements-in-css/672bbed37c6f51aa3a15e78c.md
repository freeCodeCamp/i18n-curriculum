# --interactive--

Uno degli aspetti più interessanti del CSS è l’uso dei pseudo-elementi. In questo contesto, "pseudo" significa "non reale", quindi i pseudo-elementi sono elementi virtuali o sintetici che non corrispondono direttamente a nessun elemento HTML reale. Permettono di definire lo stile di parti specifiche di un elemento o di inserire contenuto senza aggiungere HTML extra.

Per applicare un pseudo-elemento, allegalo al selettore dell’elemento originale usando due punti doppi (`::`). Nota che il selettore può essere di qualsiasi tipo, come un selettore di classe o di ID. Ecco come appare la sintassi base dei pseudo-elementi:

```css
selector::pseudo-element {
  property: value;
}
```

Questi due punti doppi sono ciò che distingue i pseudo-elementi dalle pseudo-classi, che usano un solo punto.

I pseudo-elementi permettono di definire lo stile di parti specifiche del contenuto di un elemento o di inserire contenuto prima o dopo di esso, ma non possono esistere indipendentemente. L’elemento a cui è allegato un pseudo-elemento si chiama elemento originatore.

Iniziamo guardando esempi per i pseudo-elementi `::before` e `::after`. Come suggeriscono i loro nomi, `::before` ti permette di inserire contenuto proprio prima del contenuto dell’elemento mentre `::after` ti permette di inserire contenuto dopo.

Ecco un esempio di un elemento button. Nel CSS useremo il posizionamento assoluto e il pseudo-elemento `::before` per aggiungere una stella prima del testo `Learn More` del pulsante. Imparerai di più sul posizionamento assoluto nelle lezioni future.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: lightseagreen;
  color: white;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
  position: relative;
}

.cta-button::before {
  content: "⭐";
  position: absolute;
  left: 3px;
  top: 8px;
  font-size: 0.75rem;
}
```

:::

La proprietà `content` viene usata per rappresentare il contenuto che vuoi aggiungere prima del testo del pulsante. In questo esempio, stiamo aggiungendo una stella. Noterai che puoi non solo inserire contenuto ma anche definirne lo stile. Ecco un esempio del pseudo-elemento `::after` con lo stesso pulsante:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: '➡️';
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}
```

:::

Nel pseudo-elemento `::after`, la proprietà `transition` viene usata per animare i cambiamenti in 0,3 secondi con un effetto easing, creando una trasformazione fluida e graduale invece che improvvisa. Imparerai di più sulla proprietà `transition` nelle lezioni future.

Puoi anche allegare una pseudo-classe al contenuto che inserisci in un altro contenuto con i pseudo-elementi `::before` e `::after`. Per esempio, uno stato hover per il contenuto. Ecco un esempio:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: "➡️";
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}

.cta-button:hover::after {
  transform: translateX(2px);
}
```

:::

Con `transform: translateX(2px)` nello stato hover, il contenuto viene spostato a destra di `2px` ogni volta che l’utente passa sopra il pulsante. La proprietà transition nel `::after` stesso assicura che il processo duri `0.3s`.

Questo è ciò che fa la proprietà `transform` – permette di ruotare, inclinare, scalare o traslare un elemento in una direzione particolare. Imparerai di più su questo nelle lezioni future.

Nel prossimo esempio, vedremo il pseudo-elemento `::first-letter`. Il pseudo-elemento `::first-letter` prende di mira la prima lettera del contenuto di un elemento, permettendoti di definirne lo stile. Ecco un esempio di testo in un paragrafo. Se vogliamo definire lo stile della prima lettera, possiamo usare il pseudo-elemento `::first-letter` così:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>freeCodeCamp lets you learn to code without having to pay.</p>
```

```css
p::first-letter {
  font-size: 4rem;
}
```

:::

Nell’ultimo esempio, vedremo il pseudo-elemento `::marker`, che ti permette di selezionare il marcatore, il pallino o la numerazione degli elementi di una lista per definirne lo stile. Il pseudo-elemento `::marker` offre un modo per migliorare l’identità del brand del tuo sito personalizzando i marcatori delle liste per abbinarli allo schema di colori.

Ecco un esempio di una lista non ordinata e di una lista ordinata. Per cambiare il colore e la dimensione del marcatore degli elementi della lista, puoi usare il pseudo-elemento `::marker` così:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<ul>
  <li>Unordered list item 1</li>
  <li>Unordered list item 2</li>
  <li>Unordered list item 3</li>
  <li>Unordered list item 4</li>
</ul>

<ol>
  <li>Ordered list item 1</li>
  <li>Ordered list item 2</li>
  <li>Ordered list item 3</li>
  <li>Ordered list item 4</li>
</ol>
```

```css
li::marker {
  color: crimson;
  font-size: 1.5em;
  font-weight: bold;
}
```

:::

In questa lezione abbiamo coperto solo alcuni pseudo-elementi. Ma ce ne sono molti altri come `::placeholder`, `::spelling-error` e `::selection` che ti incoraggio a esplorare da solo.
