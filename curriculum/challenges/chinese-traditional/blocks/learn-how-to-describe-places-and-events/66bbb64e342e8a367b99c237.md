# --explanation--

當有人說他們在 `struggling with` 某事時，表示他們在那方面遇到困難或麻煩。 例如：

- 團隊正在努力趕上專案截止日期。

`IDEs` 是 `Integrated Development Environments` 的縮寫。這些是開發人員用來撰寫、測試和偵錯程式碼的軟體工具。常見的熱門 `IDEs` 範例包括 Visual Studio Code、IntelliJ IDEA 和 Eclipse。

在對話窗中，Tom 正在他的整合開發環境遇到問題，並且想要和 Sarah 談談這件事。
