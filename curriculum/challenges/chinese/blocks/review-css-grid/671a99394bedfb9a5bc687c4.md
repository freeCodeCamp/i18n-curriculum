## CSS Grid 基础

- **定义**： CSS Grid 是一种二维布局系统，用于在网页中创建复杂的布局。 Grid 有行和列，行与列之间有间隙。 要定义网格布局，你需要将 `display` 属性设置为 `grid`。

```css
.container {
  display: grid;
}
```

- ** `fr`（分数）单位**: 该单位代表网格容器内空间的一部分。 你可以使用 `fr` 单元创建弹性的网格。
- **在轨道间创建间隙**：在 CSS 网格中，有三种方法可以在轨道间创建间隙。 你可以使用 `column-gap` 属性在列之间创建间隙。 你可以使用 `row-gap` 属性在行与行之间创建间隙。 你也可以使用 `gap` 简写属性同时在行和列之间创建间隙。
- **`grid-template-columns`**：该属性用于设置网格列轨道的网格线名称和尺寸。

```css
.container {
  display: grid;
  width: 100%;
  grid-template-columns: 30px 1fr;
}
```

- **`grid-template-rows`**：该属性用于设置网格行轨道的网格线名称和尺寸。
- **`grid-auto-flow`**：这决定了自动放置项如何适应网格。

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-flow: column;
}
```

- **`grid-auto-columns`**：该属性用于设置隐式创建的列的尺寸。

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-columns: auto;
}
```

- **`place-items`**：该属性用于同时设置项目在堆叠方向和行内方向的对齐方式。
- **`align-items`**：该属性用于设置网格容器中项目的对齐方式。
- **`repeat()` 函数**: 此函数用于重复轨道列表中的部分。 你可以使用 `repeat()` 函数来代替 `grid-template-columns: 1fr 1fr 1fr;`。

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}
```

- **显式网格**：你可以使用 `grid-template-columns` 或 `grid-template-rows` 属性指定行数或轨道数。
- **隐式网格**：当项目放置在网格外时，会自动为这些外部元素创建行和列。
- **`minmax()` 函数**：此函数用于设置轨道的最小和最大尺寸。

```css
.container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-auto-rows: minmax(150px, auto);
}
```

- **基于网格线的定位**：所有网格都有网格线。 要指定项目的起始网格线，你可以使用 `grid-column-start` 和 `grid-row-start` 属性。 要指定项目的结束网格线，你可以使用 `grid-column-end` 和 `grid-row-end` 属性。 你也可以使用 `grid-column` 和 `grid-row` 简写属性。

下面是一个使用 `grid-column` 属性使一个元素横跨所有列的示例。

```css
.element {
  grid-column: 1 / -1;
}
```

- **`grid-template-areas`**：该属性用于为要在网格上定位的项目提供名称。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="header">Header</div>
  <div class="sidebar">Sidebar</div>
  <div class="main">Main Content</div>
  <div class="footer">Footer</div>
</div>

```

```css
.container {
  display: grid;
  grid-template-columns: 200px 1fr; 
  grid-template-rows: auto 1fr auto; 
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer"; 
  gap: 20px; 
}

.header {
  grid-area: header; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}

.sidebar {
  grid-area: sidebar;
  background-color: #f4f4f4;
  padding: 10px;
}

.main {
  grid-area: main; 
  background-color: #e0e0e0;
  padding: 10px;
}

.footer {
  grid-area: footer; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}
```

:::

## 调试 CSS

- **DevTools（开发者工具）**: DevTools 允许你实时检查和修改 CSS。 样式窗格显示应用于所选元素的所有 CSS 规则，包括继承样式。 你可以在浏览器中直接开关单个属性、编辑值，甚至添加新规则。 这种即时反馈对于在不修改源代码的情况下尝试不同的样式非常有用。
- **CSS 验证器**：验证器用于根据官方规范检查 CSS，并报告任何错误或警告。 W3C CSS 验证器是一种常用的验证器。
- **调试响应式设计**：DevTools 有一个选项，允许你模拟网站在各种屏幕尺寸和设备上的显示效果。 这可以帮助你识别断点问题或识别在不同视口尺寸下不能很好缩放的样式。
