# --description--

العامل النهائي، نهاية السطر، الفاصل الحاسم بين التعادل. لنفترض أنه بعد أخذ كل العوامل الأخرى في الاعتبار، لا تزال هناك قواعد متضاربة متعددة تستهدف عنصرًا واحدًا. كيف يحدد التسلسل أي قاعدة تُطبق؟

ببساطة شديدة، في الواقع. القاعدة التي تم تعريفها أخيرًا هي الفائزة.

```css
/* styles.css */

.alert {
  color: red;
}

.warning {
  color: yellow;
}
```

بالنسبة لعنصر يحمل كلا الصنفين `alert` و `warning`، يمر التسلسل بكل العوامل الأخرى، بما في ذلك الوراثة (لا توجد هنا) والتحديدية (لا توجد قاعدة أكثر تحديدًا من الأخرى). وبما أن قاعدة `.warning` هي الأخيرة التي تم تعريفها، ولم يتمكن أي عامل آخر من تحديد القاعدة التي تُطبق، فهي التي تُطبق على العنصر.
