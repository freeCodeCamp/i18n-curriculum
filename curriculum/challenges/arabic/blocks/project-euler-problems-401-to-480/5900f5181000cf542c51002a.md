# --description--

تُسمى متتالية من الأعداد الصحيحة $S = \{s_i\}$ سلسلة n إذا كانت تحتوي على $n$ عنصرًا وكل عنصر $s_i$ يحقق $1 ≤ s_i ≤ n$. إذًا هناك $n^n$ سلسلة n مميزة إجمالًا.

على سبيل المثال، المتتالية $S = \{1, 5, 5, 10, 7, 7, 7, 2, 3, 7\}$ هي سلسلة n حيث $n=10$.

لكل متتالية $S$، ليكن $L(S)$ طول أطول تسلسل فرعي متجاور في $S$ له نفس القيمة. على سبيل المثال، للمتتالية $S$ المعطاة أعلاه، $L(S) = 3$ بسبب وجود ثلاثة أعداد 7 متتالية.

لتكن $f(n) = \sum L(S)$ لجميع سلاسل n $S$.

على سبيل المثال، $f(3) = 45$، $f(7) = 1\,403\,689$ و$f(11) = 481\,496\,895\,121$.

احسب $f(7\,500\,000)\bmod 1\,000\,000\,009$.
