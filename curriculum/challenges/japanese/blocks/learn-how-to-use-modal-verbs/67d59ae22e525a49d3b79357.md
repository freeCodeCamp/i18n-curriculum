# --explanation--

`Could` と `might` は可能性を表す助動詞です。

`Could` は可能性はあるが確実ではないことを示します。例えば：

`If we don't fix this bug, it could cause a system crash.` - システムが停止する可能性があります。

`Might` は `could` よりも可能性が低いことを示します。例えば：

`We might need to rewrite some parts of the code.` - 変更が必要になる可能性があります。
