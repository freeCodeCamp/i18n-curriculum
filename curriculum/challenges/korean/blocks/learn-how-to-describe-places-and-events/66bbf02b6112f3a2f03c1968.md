# --explanation--

`It`는 이미 언급된 명사를 대신하여 문장에서 반복을 피하는 데 사용됩니다. 예를 들어:

- `I bought a new laptop. It works really well.` - 여기서 `it`는 단어를 반복하지 않기 위해 `laptop`를 대신합니다.

`The Live Preview isn't showing in Visual Studio Code. Isn't it supposed to support that feature?`에서 `it`는 Tom이 프로그램과 그 기능 지원 능력에 대해 이야기하고 있기 때문에 `Visual Studio Code`를 가리킵니다.

`It`는 프로그램 이름을 다시 말하지 않기 위해 `Visual Studio Code`를 대신합니다.

기능을 지원할 것으로 예상되는 대상을 파악함으로써 `it`가 프로그램 자체를 가리킨다는 것을 알 수 있습니다.
