# --description--

Per creare un file su Windows usando Explorer, puoi cliccare sull'opzione "Nuovo" in alto a sinistra. Questo ti mostrerà una lista dei diversi tipi di file che puoi creare.

Puoi anche creare file nelle applicazioni corrispondenti. Creiamo un file di testo per mostrarti il processo.

Quando clicchi su "Nuovo" e selezioni il tipo di file, vedrai un nuovo file di testo nella cartella corrente con un nome predefinito. Puoi rinominarlo se vuoi.

Il processo per creare cartelle è esattamente lo stesso. Clicca sul pulsante "Nuovo" in alto a sinistra e poi seleziona l'opzione "Cartella" oppure fai clic con il tasto destro su uno spazio vuoto e vai su "Nuovo". Poi, seleziona l'opzione "Cartella". Vedrai una nuova cartella vuota nella posizione corrente.

Puoi rinominare file e cartelle selezionandoli e cliccando sull'opzione "Rinomina" in alto. Quando il nome attuale del file è evidenziato, puoi iniziare a scrivere il nuovo nome del file. Premi Invio per confermare le modifiche.

In alternativa, puoi fare clic con il tasto destro sul file o sulla cartella e selezionare l'opzione "Rinomina" oppure usare la scorciatoia da tastiera F2. Scrivi il nuovo nome e premi Invio per confermare le modifiche.

Puoi spostare un file o una cartella in un'altra cartella trascinandolo e rilasciandolo nella cartella di destinazione, se l'hai aperta come scheda.

Puoi anche selezionare il file o la cartella e scegliere l'opzione "Taglia" in alto.

Poi, vai nella cartella di destinazione e incollalo cliccando sul pulsante "Incolla" in alto. Se hai copiato o tagliato il file in precedenza, questo pulsante sarà abilitato.

Puoi farlo anche facendo clic con il tasto destro su uno spazio vuoto e poi cliccando sull'icona incolla. Questo rimuoverà il file o la cartella dalla sua posizione originale.

Per eliminare un file o una cartella, cliccaci sopra e poi clicca sull'icona del Cestino in alto. Troverai anche un'opzione "Elimina" se fai clic con il tasto destro su di esso.

Ottimo. Ora vediamo come fare questo su macOS usando Finder.

Finder è il gestore di file predefinito su macOS.

Per creare un file su macOS, devi aprire un'applicazione che ti permetta di creare il tipo di documento che ti serve. Attualmente, non esiste un modo integrato per creare un file direttamente su Finder.

Per esempio, per creare un semplice file di testo, puoi aprire TextEdit e salvare il file. Il processo funziona esattamente allo stesso modo per qualsiasi tipo di file.

Ci sono molti modi per aprire TextEdit. Uno di questi è andare su Launchpad. Poi, cerca un'app dove puoi creare il tipo di file che ti interessa. In questo caso, sarebbe "TextEdit". Clicca sull'icona per aprire l'app.

Una volta dentro l'app, dovresti vedere un'opzione per salvare il file nel menu File. Questo menu si trova in alto a sinistra, accanto all'icona del menu Apple.

In una delle opzioni del menu dell'app, dovresti vedere un comando per salvare il file. Di solito si trova sotto il menu File.

Accanto a questo, vedrai la scorciatoia da tastiera per salvare il file. Di solito è `Command + S` su macOS.

Quando salvi il file per la prima volta, vedrai una finestra di dialogo dove puoi scrivere il nome del file e scegliere una posizione.

Di solito, l'estensione del file viene aggiunta automaticamente dall'applicazione ma puoi cambiarla prima di salvare il file.

Dopo aver salvato il file, puoi aprire Finder cliccando sull'icona di Finder nel Dock. Dovresti vedere il tuo nuovo file nella cartella dove l'hai salvato.

Per creare una nuova cartella, devi andare nella posizione dove vuoi creare quella nuova cartella e fare clic con il tasto destro su uno spazio vuoto. Vedrai una lista di opzioni. La prima opzione è "Nuova cartella".

Se ci clicchi, puoi assegnare un nome alla tua nuova cartella. Scrivi il nome e premi Invio per confermare.

Ci sono diversi modi per spostare file e cartelle in Finder. Se devi spostarli in una cartella elencata nella sezione "Preferiti" di Finder, puoi semplicemente trascinarli e rilasciarli.

Se devi spostarli in una cartella diversa, hai due opzioni.

Puoi aprire la cartella di destinazione come scheda e trascinare il file o la cartella dentro la scheda.

Oppure puoi usare le scorciatoie da tastiera. Prima, copia il file con `Command + C`, vai nella nuova cartella e poi usa `Command + Option + V` per incollare il file o la cartella in quella posizione. Questo lo rimuoverà anche dalla cartella originale.

Per eliminare un file o una cartella, puoi fare clic con il tasto destro su di esso e selezionare "Sposta nel cestino".

Puoi anche trascinarli e rilasciarli sull'icona del cestino nel Dock. È equivalente.

Quando il file o la cartella è nel cestino, puoi fare clic con il tasto destro su di esso per eliminarlo immediatamente oppure puoi svuotare il cestino per rimuovere definitivamente tutti i file e le cartelle eliminati.

Sapere come creare, spostare ed eliminare file e cartelle su Windows e macOS è molto importante. Padroneggiando queste abilità, puoi organizzare i tuoi file, trovarli rapidamente e liberare spazio sul computer eliminando i file inutili.
