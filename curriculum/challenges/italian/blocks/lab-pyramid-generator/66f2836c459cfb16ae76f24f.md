# --description--

**Obiettivo:** Completa le storie utente qui sotto e fai superare tutti i test per completare il laboratorio.

**Storie utente:**

1. Devi avere una funzione chiamata `pyramid` che prende tre argomenti.
1. Il primo argomento deve essere una stringa che rappresenta il carattere pattern da ripetere nella tua piramide.
1. Il secondo argomento deve essere un intero che rappresenta il numero di righe nella piramide.
1. Il terzo argomento deve essere un valore booleano.
1. La funzione `pyramid` deve restituire una stringa in cui il carattere pattern è ripetuto e disposto per formare una piramide con il vertice rivolto verso l’alto quando il terzo argomento è `false`.
1. Quando il terzo argomento è `true` la piramide deve avere il vertice rivolto verso il basso.
1. La riga del vertice deve avere un solo carattere pattern, e ogni altra riga deve avere due caratteri pattern in più rispetto alla precedente.
1. Ogni riga deve iniziare con un numero di spazi sufficiente a mettere il carattere centrale di ogni riga nella stessa colonna e non devono esserci spazi alla fine di ogni riga.
1. La piramide deve iniziare e finire con un carattere di nuova riga.

Per esempio, chiamare `pyramid("o", 4, false)` dovrebbe dare questo output:

```js

   o
  ooo
 ooooo
ooooooo

```

function pyramid(char, count, isInverted) {
    const rows = []
    for (let i = 1; i <= count; i++) {
        if (isInverted) {
            rows.unshift(" ".repeat(count - i) + char.repeat(2 * i - 1))
        } else {
            rows.push(" ".repeat(count - i) + char.repeat(2 * i - 1))
        }        
    }
    return "\n" + rows.join("\n") + "\n";
}
