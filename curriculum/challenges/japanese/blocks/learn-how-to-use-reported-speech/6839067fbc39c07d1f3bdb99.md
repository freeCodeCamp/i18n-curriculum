# --explanation--

`Forward` は今後の行動や時間について話すときに使います。例えば：

`Going forward, we will hold weekly meetings.` – これは今から始まり、毎週ミーティングが行われるという意味です。

`To happen` はイベントや状況が起こる、発生するという意味です。例えば：

`If problems happen, please report them immediately.` – これは問題が起こった場合、誰かに伝えるべきだという意味です。
