# --interactive--

在 CSS 中處理顏色時，了解 RGB 色彩模型是必須的。RGB 代表紅色、綠色和藍色——光的三原色。這三種顏色以不同強度組合，產生廣泛的顏色範圍。 

RGB 色彩模型是一種加色模型，這表示顏色是透過以不同強度組合光線來創建的。 

每種顏色的強度可以在 `0`（表示無光）到 `255`（表示全光）之間變化。透過混合不同層次的紅色、綠色和藍色，你可以產生螢幕上看到的任何顏色。 例如：

- 黑色由 `rgb(0, 0, 0)` 表述，這表示三種顏色中都沒有光。
- 白色由 `rgb(255, 255, 255)` 表現，這表示三種顏色的光強度皆為最大。
- 紅色由 `rgb(255, 0, 0)` 表述，紅色為全強度，綠色和藍色則為無強度。

在 CSS 中，`rgb()` 函式允許你使用 RGB 色彩模型定義顏色。該函式接受三個數值，分別表述紅色、綠色和藍色光的強度。以下是基本語法：

```css
element {
  color: rgb(red, green, blue);
}
```

紅色、綠色和藍色的數值可以在 `0` 到 `255` 的範圍內，其中 `0` 表示該顏色的缺失，`255` 表示完全飽和。以下是你可能在 CSS 中使用 `rgb()` 函式的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

這段程式碼會將段落的文字顏色改為紅色，因為紅色值設定為 `255`，而綠色和藍色則設定為 `0`。

CSS 也提供了 `rgba()` 函式，該函式添加了第四個值 — alpha — 用來控制顏色的透明度。alpha 值的範圍是從 `0`（完全透通）到 `1`（完全不透明）。以下是使用 `rgba()` 的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

這段程式碼會將半透通的藍色背景套用到 `div` 元素，因為紅色和綠色值設定為 `0`，藍色值設定為 `255`，而 alpha 值設定為 `0.5`（50％ 透明度）。

RGB 色彩模型對數位媒體特別有用，因為它直接對應到螢幕如何顯示顏色。顯示器和螢幕使用微小的紅色、綠色和藍色像素來產生你所看到的顏色。透過控制這些像素的強度，利用 RGB 值，你可以為你的網頁設計達成廣泛的色彩範圍。

此外，當處理動態設計時，RGB 模型是理想的選擇。例如，如果你想創建需要即時變化的動畫或色彩效果，RGB 函式的彈性允許你以程式方式修改顏色。

透過了解 RGB 色彩模型以及如何在 CSS 中使用 `rgb()` 和 `rgba()` 函式（程式），你將能更有效地控制網頁上顏色的顯示。你可以創建鮮豔的設計、透明效果，甚至程式化地調整顏色以達成動態效果。
