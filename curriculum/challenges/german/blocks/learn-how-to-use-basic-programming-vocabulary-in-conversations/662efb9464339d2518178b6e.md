# --description--

Das Wort `issue` kann sich auf ein Problem oder eine Angelegenheit beziehen, die bearbeitet werden muss.

In der Programmierung bezieht sich ein `issue` oft auf einen Bug oder Fehler im Code, der behoben werden muss. Zum Beispiel `We found an issue in the latest software update.`

# --questions--

## --text--

Was bedeutet `issue` im Programmierkontext?

## --answers--

Es bedeutet, dass die Dokumentation unvollständig ist.

### --feedback--

Obwohl Dokumentationsprobleme als `issues` betrachtet werden können, bezieht sich `issue` in der Programmierung direkter auf Probleme im Code selbst.
