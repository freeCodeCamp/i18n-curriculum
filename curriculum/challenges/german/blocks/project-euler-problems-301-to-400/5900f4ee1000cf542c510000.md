# --description--

Für jedes Dreieck $T$ in der Ebene lässt sich zeigen, dass es eine eindeutige Ellipse mit größter Fläche gibt, die vollständig innerhalb von $T$ liegt.

<img alt="ellipse completely inside a triangle" src="https://cdn.freecodecamp.org/curriculum/project-euler/ellipses-inside-triangles.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Betrachten Sie für ein gegebenes $n$ Dreiecke $T$, so dass:

-   die Eckpunkte von $T$ ganzzahlige Koordinaten mit absolutem Wert $≤ n$ haben, und
-   die Brennpunkte<sup>1</sup> der Ellipse mit größter Fläche innerhalb von $T$ $(\sqrt{13}, 0)$ und $(-\sqrt{13}, 0)$ sind.

Sei $A(n)$ die Summe der Flächen aller solchen Dreiecke.

Zum Beispiel gibt es für $n = 8$ zwei solche Dreiecke. Ihre Eckpunkte sind (-4,-3), (-4,3), (8,0) und (4,3), (4,-3), (-8,0), und die Fläche jedes Dreiecks beträgt 36. Somit ist $A(8) = 36 + 36 = 72$.

Es lässt sich überprüfen, dass $A(10) = 252$, $A(100) = 34\\,632$ und $A(1000) = 3\\,529\\,008$ gilt.

Finden Sie $A(1\\,000\\,000\\,000)$.

<sup>1</sup>Die Brennpunkte (Plural von Brennpunkt) einer Ellipse sind zwei Punkte $A$ und $B$, so dass für jeden Punkt $P$ auf dem Rand der Ellipse $AP + PB$ konstant ist.
