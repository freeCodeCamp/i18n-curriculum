### --question--

#### --text--

Який із наведених способів є правильним для оголошення функції в JavaScript?

#### --distractors--

```js
func greet() {
  console.log("Hello there!");
}
```

---

```js
defineFunction greet() {
  console.log("Hello there!");
}
```

---

```js
def greet() {
  console.log("Hello there!");
}
```

#### --answer--

```js
function greet() {
  console.log("Hello there!");
}
```

### --question--

#### --text--

Що відбувається, коли ви викликаєте (або запускаєте) функцію?

#### --distractors--

Код всередині функції виконається, і `null` завжди буде повернено.

---

Нічого не станеться, коли ви викликаєте функцію.

---

Викинеться `TypeError`, і програма аварійно завершиться.

#### --answer--

Код всередині функції виконається, і буде повернено значення.

};
