# --explanation--

Fomu ya kuuliza swali katika `Present Perfect Continuous` huundwa kwa kutumia `Have/Has + subject + been + verb(-ing)`. Inatumika kuuliza kuhusu vitendo vilivyoanza zamani na bado vinaendelea. Kwa mfano:

`Have they been testing the new software features?` - Mtu anahakikisha kama majaribio yanaendelea.
