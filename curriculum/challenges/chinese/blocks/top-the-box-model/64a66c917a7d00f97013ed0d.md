# --description--

你需要理解的第一个重要概念是盒模型。它并不复杂，但现在跳过它将会在以后给你带来很多挫折。

网页上的每一个元素都是一个矩形框。这些框可以包含其他框，也可以并排放置。你可以通过给页面上的每个项添加边框来大致了解这是如何工作的，方法如下：

```css
* {
  border: 1px solid red;
}
```

# --assignment--

为页面上的每个元素添加边框，查看盒子是如何布局的。
