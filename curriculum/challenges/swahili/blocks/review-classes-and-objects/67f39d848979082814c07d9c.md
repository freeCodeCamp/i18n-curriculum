## Madarasa na vitu vya Python

- **Ufafanuzi wa darasa la vitu**: Darasa ni ramani ya kuunda vitu. Linafafanua tabia ambayo kitu kitakuwa nayo kupitia sifa zake za kipengele na njia zake. Hapa kuna mfano wa msingi wa ufafanuzi wa darasa katika Python:

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f'{self.name.upper()} says woof woof!')
```

- **Kuunda vitu**: Vitu ni matukio ya darasa. Vinaundwa kwa kuita darasa na hoja zinazohitajika.

```python
dog1 = Dog('Jack', 3)
dog2 = Dog('Thatcher', 5)

dog1.bark()  # JACK says woof woof!
dog2.bark()  # THATCHER says woof woof!
```

- **Kuita njia kwenye vitu**: Unaweza kuita njia kwenye vitu ili kufanya vitendo au kupata taarifa.

```python
object_name1.method_name()
object_name2.method_name()
```

- **Tofauti kati ya darasa na kitu**: Darasa ni kiolezo kinachoweza kutumika tena, wakati kitu ni mfano maalum wa darasa hilo lenye data halisi.

## Sifa za kipengele

- **Sifa za kipengele za mfano**: Zinafafanuliwa katika `__init__()` kwa kutumia `self`, na ni za kipekee kwa kila kitu.
- **Sifa za kipengele za darasa**: Zinafafanuliwa moja kwa moja ndani ya darasa na zinashirikiwa na matukio yote.

```python
class Dog:
    species = 'French Bulldog'  # Class attribute

    def __init__(self, name):
        self.name = name  # Instance attribute

print(Dog.species) # French Bulldog

jack = Dog('Jack')
print(jack.name)     # Jack
print(jack.species)  # French Bulldog
```

## Njia

- **Njia**: Vitendakazi vilivyofafanuliwa ndani ya darasa vinavyofanya kazi kwenye sifa za kipengele za kitu.

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
print(my_car_1.describe())  # This car is a red Tesla Model S
```

- **Kupata njia**: Ita njia kwenye vitu kwa kutumia uandishi wa nukta. Hapa kuna mfano wa kuita njia ya `describe` kwenye magari mawili tofauti:

```python
class Car:
    def __init__(self, color, model):
        self.color = color  
        self.model = model  

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
my_car_2 = Car('green', 'Lamborghini Revuelto')

print(my_car_1.describe()) # Calling method using the dot notation

print(my_car_2.describe()) # Calling method using the dot notation
```

## Njia za dunder (za kichawi)

- **Ufafanuzi**: Njia maalum zinazaanza na kumalizika na alama mbili za mstari chini (mfano, `__init__`, `__len__`, `__str__`, `__eq__`). Python inazitumia ndani kwa ajili ya operesheni zilizojengwa ndani.

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **Kuita njia za dunder kwa njia isiyo ya moja kwa moja**: Huna haja ya kuita njia za dunder moja kwa moja. Badala yake, Python huziita moja kwa moja wakati vitendo fulani vinafanyika. Operesheni hizi ni pamoja na:

  - **operesheni za kihesabu kama kuongeza, kutoa, kuzidisha, kugawanya, na nyinginezo**. Zaidi ya hayo, `__add__()` huitwa, `__sub__()` kwa kutoa, `__mul__()` kwa kuzidisha, na `__truediv__()` kwa kugawanya.
        
  - **operesheni za mfuatano wa herufi kama kuunganisha, kurudia, kuunda muundo, na kubadilisha kuwa maandishi**. `__add__()` huitwa kwa kuunganisha, `__mul__()` kwa kurudia, `__format__()` kwa kuunda muundo, `__str__()` na `__repr__()` kwa kubadilisha kuwa maandishi, na kadhalika.
        
  - **operesheni za kulinganisha kama usawa, chini ya, juu ya, na nyinginezo**. `__eq__()` huitwa kwa ukaguzi wa usawa, `__lt__()` kwa chini ya, `__gt__()` kwa juu ya, na kadhalika.
        
  - **operesheni za kurudia kama kufanya kitu kiwe kinachoweza kurudiwa na kusonga mbele kupitia vitu**. `__iter__()` huitwa kurudisha kirejeshi na `__next__()` kupata kipengele kinachofuata.

## Mfano halisi wa dunia: Gari la manunuzi

- **Darasa la gari lenye njia za dunder**: Linawezesha kuongeza, kuondoa, kurudia, na kukagua yaliyomo kwa tabia iliyojengwa ndani.

```python
class Cart:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

    def remove(self, item):
        if item in self.items:
            self.items.remove(item)
        else:
            print(f'{item} is not in cart')

    def list_items(self):
        return self.items

    def __len__(self):
        return len(self.items)

    def __getitem__(self, index):
        return self.items[index]

    def __contains__(self, item):
        return item in self.items

    def __iter__(self):
        return iter(self.items)

cart = Cart()
cart.add('Laptop')
print(len(cart))        # 1
print('Laptop' in cart) # True
```
