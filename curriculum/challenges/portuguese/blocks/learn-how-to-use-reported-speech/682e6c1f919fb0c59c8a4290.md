# --explanation--

Em **discurso indireto**, a frase `said that` é usada para compartilhar o que alguém disse de uma forma diferente, não usando suas palavras exatas. A palavra `that` é opcional e frequentemente omitida na fala cotidiana. Por exemplo:

- Discurso direto: `We need to update the software.`

- Discurso indireto: `He said that we need to update the software.` ou `He said we need to update the software.`

Ambas as versões estão corretas e significam a mesma coisa.
