# --explanation--

`To prevent` は、何かが起こらないように行動を起こすことを意味します。例えば：

`They installed new software to prevent viruses.` – これはソフトウェアが問題を引き起こす前にウイルスを防ぐのを助けることを意味します。

`incident` は予期しない出来事で、多くの場合、否定的または問題を引き起こすものです。例えば：

`There was a security incident last night that shut down the server.` – これはシステムに問題を引き起こした何かが起こったことを意味します。
