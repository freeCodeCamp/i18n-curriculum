# --description--

Зазвичай `with` використовують, щоб показати зв’язок або спосіб використання. Наприклад, `I write with a pen`. У деяких реченнях `with` пишуть після слова, якого воно стосується. Так зазвичай буває, коли хочуть підкреслити об’єкт або річ, яку хтось використовує, особливо у запитаннях або висловлюваннях про технології чи роботу.

Наприклад, ви можете сказати `This program I work with is challenging`, де `with` використано після слова, якого воно стосується, щоб підкреслити програму, з якою важко працювати. Ще один приклад: `What else is he working with?`, де `with` запитує про інші інструменти або технології.
