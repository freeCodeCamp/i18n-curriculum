# --interactive--

在使用 JavaScript 时，你可能会遇到需要将一个字符串重复特定次数的情况。

无论是生成重复模式还是简单地复制文本，`repeat()` 方法都提供了一种简单而有效的方法来实现这一目标。

`repeat()`方法是 JavaScript 中的一个内置函数，可让你重复指定次数的字符串。 这是基本语法：

```js
string.repeat(count);
```

`string` 是要重复的字符串，`count` 是你希望字符串重复的次数。 这是一个示例：

:::interactive_editor

```js
let word = "Hello!";
let repeatedWord = word.repeat(3);
console.log(repeatedWord);  // "Hello!Hello!Hello!"
```

:::

在本例中，字符串 `Hello!` 重复了三次，结果是 `Hello!Hello!Hello!`。

虽然 `repeat()` 方法非常有用，但仍有一些例外情况和限制需要注意。

`count` 参数必须是非负数。 如果你传递的是负数，JavaScript 将抛出 `RangeError` 错误。

```js
let word = "Test";
console.log(word.repeat(-1));  // Throws RangeError: Invalid count value
```

`count` 必须是一个有限的数字。 如果你尝试无限次重复一个字符串，或使用 `Infinity` 作为计数，也将得到 `RangeError`。

在 JavaScript 中，`Infinity` 是一个表示无限量的特殊值。 它用来表示比任何有限数都大的数。

```js
let word = "Test";
console.log(word.repeat(Infinity));  // Throws RangeError: Invalid count value
```

如果计数不是整数（例如小数，如 `2.5`），`repeat()` 方法将四舍五入为最接近的整数。

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(2.5));  // "TestTest"
```

:::

如果你传递 `0` 作为计数，`repeat()` 方法将返回空字符串。

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(0));  // ""
```

:::

`repeat()` 方法可以简化涉及字符串重复的任务，使代码更加简洁易读。

无论是生成重复的文本模式还是填充字符空间，`repeat()` 都能让你免于编写循环或更复杂的代码。
