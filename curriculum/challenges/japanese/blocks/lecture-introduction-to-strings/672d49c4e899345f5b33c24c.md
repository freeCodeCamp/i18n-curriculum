# --interactive--

前のレッスンでは`console.log()`を紹介しましたが、このレッスンではその目的と使い方をさらに詳しく見ていきます。

JavaScriptでは、`console.log()`はブラウザのコンソールにメッセージや情報を表示するためのシンプルで強力なツールです。主に開発者がプログラムのコードをデバッグしたり検査したりするときに使います。

`console.log()`を使ってテキストや変数をコンソールにログ出力し、コードが正しく動いているか確認できます。

`console.log()`を使うには、丸括弧の中に出力したい値やメッセージを入れてメソッドを呼び出します。いくつか例を見てみましょう。

:::interactive_editor

```js
console.log("Hello, world!");

let num = 5;
console.log(num); // 5
```

:::

最初の例はブラウザのコンソールに`Hello, world!`を出力し、2つ目の例は値`5`を出力します。

`console.log()`を使った別の例はこちらです。

:::interactive_editor

```js
let name = "Alice";
console.log("Hello, " + name + "!"); // Hello, Alice!
```

:::

また、複数の値をカンマで区切って`console.log()`に渡すこともできます。例えば：

:::interactive_editor

```js
let name = "Alice";
let age = 25;
console.log("Name:", name, "Age:", age); // Name: Alice Age: 25
```

:::

これは複数の情報を一度にログ出力したいときに便利です。

`console.log()`メソッドはコードの実行を監視するのに役立ち、ミスを見つけやすくプログラムの動きを理解しやすくします。
