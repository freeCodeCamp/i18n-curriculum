# --description--

Ми також можемо згенерувати масив, який містить усі ключі, збережені в об’єкті, за допомогою методу `Object.keys()`. Цей метод приймає об’єкт як аргумент і повертає масив рядків, які представляють кожну властивість у об’єкті. Знову ж таки, порядок записів у масиві не буде визначеним.

# --instructions--

Завершіть написання функції `getArrayOfUsers`, щоб вона повертала масив, що містить усі властивості об’єкта, який вона отримує як аргумент.

# --hints--

Об’єкт `users` повинен містити лише ключі `Alan`, `Jeff`, `Sarah` і `Ryan`

```js
assert(
  'Alan' in users &&
    'Jeff' in users &&
    'Sarah' in users &&
    'Ryan' in users &&
    Object.keys(users).length === 4
);
```

Функція `getArrayOfUsers` має повертати масив, який містить усі ключі в об’єкті `users`

```js
assert(
  (function () {
    users.Sam = {};
    users.Lewis = {};
    let R = getArrayOfUsers(users);
    return (
      R.indexOf('Alan') !== -1 &&
      R.indexOf('Jeff') !== -1 &&
      R.indexOf('Sarah') !== -1 &&
      R.indexOf('Ryan') !== -1 &&
      R.indexOf('Sam') !== -1 &&
      R.indexOf('Lewis') !== -1
    );
  })() === true
);
```
