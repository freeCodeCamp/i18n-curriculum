## Nozioni di base sugli array in JavaScript

- **Definizione**: Un array in JavaScript è una raccolta ordinata di valori, ciascuno identificato da un indice numerico. I valori in un array JavaScript possono essere di tipi di dati diversi, inclusi numeri, stringhe, booleani, oggetti e persino altri array. Gli array sono contigui in memoria, il che significa che tutti gli elementi sono archiviati in un unico blocco continuo di posizioni di memoria, permettendo un indicizzazione efficiente e un accesso rapido agli elementi tramite il loro indice.

```js
const developers = ["Jessica", "Naomi", "Tom"];
```

- **Consultare gli elementi di un array**: Per consultare gli elementi di un array, devi fare riferimento all'array seguito dal suo numero indice all'interno di parentesi quadre. Gli array JavaScript hanno indice basato su zero, il che significa che il primo elemento si trova all'indice 0, il secondo all'indice 1, e così via. Se provi a consultare un indice che non esiste nell'array, JavaScript restituirà `undefined`.

:::interactive_editor

```js
const developers = ["Jessica", "Naomi", "Tom"];
console.log(developers[0]) // "Jessica"
console.log(developers[1]) // "Naomi"

console.log(developers[10]) // undefined
```

:::

- **Proprietà `length`**: Questa proprietà viene usata per restituire il numero di elementi in un array.

:::interactive_editor

```js
const developers = ["Jessica", "Naomi", "Tom"];
console.log(developers.length) // 3
```

:::

- **Aggiornare gli elementi in un array**: Per aggiornare un elemento in un array, usa l'operatore di assegnazione (`=`) per assegnare un nuovo valore all'elemento in uno specifico indice.

:::interactive_editor

```js
const fruits = ['apple', 'banana', 'cherry'];
fruits[1] = 'blueberry';

console.log(fruits); // ['apple', 'blueberry', 'cherry']
```

:::

## Array bidimensionali  

- **Definizione**: Un array bidimensionale è essenzialmente un array di array. Viene usato per rappresentare dati che hanno una struttura naturale a griglia, come una scacchiera, un foglio di calcolo o i pixel in un'immagine. Per consultare un elemento in un array bidimensionale, servono due indici: uno per la riga e uno per la colonna.

:::interactive_editor

```js
const chessboard = [
    ['R', 'N', 'B', 'Q', 'K', 'B', 'N', 'R'],
    ['P', 'P', 'P', 'P', 'P', 'P', 'P', 'P'],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    ['p', 'p', 'p', 'p', 'p', 'p', 'p', 'p'],
    ['r', 'n', 'b', 'q', 'k', 'b', 'n', 'r']
];

console.log(chessboard[0][3]); // "Q"
```

:::

## Destrutturazione degli array  

- **Definizione**: La destrutturazione degli array è una funzionalità di JavaScript che ti permette di estrarre valori dagli array e assegnarli a variabili in modo più conciso e leggibile. Fornisce una sintassi comoda per decomprimere gli elementi di un array in variabili distinte.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange"];

const [first, second, third] = fruits;

console.log(first); // "apple"
console.log(second); // "banana"
console.log(third); // "orange"
```

:::

- **Sintassi rest**: Questa ti permette di catturare gli elementi rimanenti di un array che non sono stati destrutturati in un nuovo array.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange", "mango", "kiwi"];
const [first, second, ...rest] = fruits;

console.log(first); // "apple"
console.log(second); // "banana"
console.log(rest); // ["orange", "mango", "kiwi"]
```

:::

## Metodi comuni degli array 

- **Metodo `push()`**: Questo metodo viene usato per aggiungere elementi alla fine dell'array e restituisce la nuova lunghezza.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.push("ice cream");

console.log(desserts); // ["cake", "cookies", "pie", "ice cream"];
```

:::

- **Metodo `pop()`**: Questo metodo viene usato per rimuovere l'ultimo elemento da un array e restituisce quell'elemento rimosso. Se l'array è vuoto, il valore di ritorno sarà `undefined`.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.pop();

console.log(desserts); // ["cake", "cookies"];
```

:::

- **Metodo `shift()`**: Questo metodo viene usato per rimuovere il primo elemento da un array e restituisce quell'elemento rimosso. Se l'array è vuoto, il valore di ritorno sarà `undefined`.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.shift();

console.log(desserts); // ["cookies", "pie"];
```

:::

- **Metodo `unshift()`**: Questo metodo viene usato per aggiungere elementi all'inizio dell'array e restituisce la nuova lunghezza.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.unshift("ice cream");

console.log(desserts); // ["ice cream", "cake", "cookies", "pie"];
```

:::

- **Metodo `indexOf()`**: Questo metodo è utile per trovare il primo indice di un elemento specifico all'interno di un array. Se l'elemento non viene trovato, restituisce `-1`.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange", "banana"];
const index = fruits.indexOf("banana");

console.log(index); // 1
console.log(fruits.indexOf("not found")); // -1
```

:::

- **Metodo `splice()`**: Questo metodo viene usato per aggiungere o rimuovere elementi da qualsiasi posizione in un array. Il valore di ritorno del metodo `splice()` sarà un array degli elementi rimossi dall'array. Se non viene rimosso nulla, verrà restituito un array vuoto. Questo metodo modifica l'array originale, aggiornandolo in loco invece di crearne uno nuovo. Il primo argomento specifica l'indice da cui iniziare a modificare l'array. Il secondo argomento è il numero di elementi che vuoi rimuovere. Gli argomenti successivi sono gli elementi che vuoi aggiungere.

:::interactive_editor

```js
const colors = ["red", "green", "blue"];
colors.splice(1, 0, "yellow", "purple");

console.log(colors); // ["red", "yellow", "purple", "green", "blue"]
```

:::

- **Metodo `includes()`**: Questo metodo viene usato per verificare se un array contiene un valore specifico. Restituisce `true` se l'array contiene l'elemento specificato, e `false` altrimenti.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];

console.log(programmingLanguages.includes("Python")); // true
console.log(programmingLanguages.includes("Perl")); // false
```

:::

- **Metodo `concat()`**: Questo metodo crea un nuovo array unendo due o più array.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];
const newList = programmingLanguages.concat("Perl");

console.log(newList); // ["JavaScript", "Python", "C++", "Perl"]
```

:::

- **Metodo `slice()`**: Questo metodo restituisce un nuovo array contenente una copia superficiale di una porzione dell'array originale, specificata dagli indici di inizio e fine. Il nuovo array contiene riferimenti agli stessi elementi dell'array originale (non duplicati). Questo significa che se gli elementi sono primitivi (come numeri o stringhe), i valori vengono copiati; ma se gli elementi sono oggetti o array, vengono copiati i riferimenti, non gli oggetti stessi.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];
const newList = programmingLanguages.slice(1);

console.log(newList); // ["Python", "C++"]
```

:::

- **Sintassi di espansione**: La sintassi di espansione viene usata per creare copie superficiali di un array.

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const shallowCopiedArray = [...originalArray];

shallowCopiedArray.push(4);

console.log(originalArray); // [1, 2, 3]
console.log(shallowCopiedArray); // [1, 2, 3, 4]
```

:::

- **Metodo `split()`**: Questo metodo divide una stringa in un array di sottostringhe e specifica dove deve avvenire ogni divisione in base a un separatore dato. Se non viene fornito alcun separatore, il metodo restituisce un array contenente la stringa originale come unico elemento.

:::interactive_editor

```js
const str = "hello";
const charArray = str.split("");

console.log(charArray); // ["h", "e", "l", "l", "o"]
```

:::

- **Metodo `reverse()`**: Questo metodo inverte un array in loco.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
console.log(desserts.reverse()); // ["pie", "cookies", "cake"]
```

:::

- **Metodo `join()`**: Questo metodo concatena tutti gli elementi di un array in una singola stringa, con ogni elemento separato da un separatore specificato. Se non viene fornito alcun separatore, o viene usata una stringa vuota (`""`), gli elementi verranno uniti senza alcun separatore.

:::interactive_editor

```js
const reversedArray = ["o", "l", "l", "e", "h"];
const reversedString = reversedArray.join("");

console.log(reversedString); // "olleh"
```

:::
