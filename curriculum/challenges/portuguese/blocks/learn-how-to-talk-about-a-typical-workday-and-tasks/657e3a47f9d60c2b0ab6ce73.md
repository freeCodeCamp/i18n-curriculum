# --description--

To `make sure` é usado para expressar certeza ou confirmação sobre algo. Por exemplo:

`I check the list twice to make sure I have everything.` - Isso significa que você olha a lista duas vezes para ter certeza de que nada está faltando.

Significa `work properly` funcionar corretamente ou como esperado. Por exemplo:

`It's important that all machines work properly for safety.` - Isso significa que as máquinas devem funcionar da maneira correta para que as pessoas fiquem seguras.
