# --explanation--

`Could have` + `Past Participle` se usa para hablar de algo que era posible pero no ocurrió. Por ejemplo:

- `You could have added a comment to explain this function.` - Habría sido útil si la persona hubiera incluido un comentario para explicar qué hace la función.

- `We could have added a search bar to make the website more user-friendly.` - El equipo estaba pensando en agregar una barra de búsqueda, pero aún no se había hecho.
