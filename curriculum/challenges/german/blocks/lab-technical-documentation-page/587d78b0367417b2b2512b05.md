# --description--

**Ziel:** Erfüllen Sie die unten stehenden User Stories und bestehen Sie alle Tests, um das Labor abzuschließen.

**User Stories:**

1. Sie sehen ein `main`-Element mit einem entsprechenden `id="main-doc"`, das den Hauptinhalt der Seite (technische Dokumentation) enthält.
1. Innerhalb des `#main-doc`-Elements sehen Sie mehrere `section`-Elemente, von denen jedes die Klasse `main-section` hat. Es sollten mindestens fünf sein.
1. Das erste Element innerhalb jedes `.main-section` sollte ein `header`-Element sein, das Text enthält, der das Thema dieses Abschnitts beschreibt.
1. Jedes `section`-Element mit der Klasse `main-section` sollte auch ein `id` haben, das mit dem Text jedes `header` darin übereinstimmt. Alle Leerzeichen sollten durch Unterstriche ersetzt werden (z. B. sollte der Abschnitt, der die Kopfzeile „JavaScript and Java“ enthält, ein entsprechendes `id="JavaScript_and_Java"` haben).
1. Die `.main-section`-Elemente sollten insgesamt mindestens zehn `p`-Elemente enthalten (nicht jeweils).
1. Die `.main-section`-Elemente sollten insgesamt mindestens fünf `code`-Elemente enthalten (nicht jeweils).
1. Die `.main-section`-Elemente sollten insgesamt mindestens fünf `li`-Elemente enthalten (nicht jeweils).
1. Sie sehen ein `nav`-Element mit einem entsprechenden `id="navbar"`.
1. Das navbar-Element sollte ein `header`-Element enthalten, das Text enthält, der das Thema der technischen Dokumentation beschreibt.
1. Zusätzlich sollte die Navigationsleiste Link-Elemente (`a`) mit der Klasse `nav-link` enthalten. Es sollte für jedes Element mit der Klasse `main-section` eines geben.
1. Das `header`-Element im `#navbar` muss vor allen Link-Elementen (`a`) in der Navigationsleiste stehen.
1. Jedes Element mit der Klasse `nav-link` sollte Text enthalten, der mit dem `header`-Text innerhalb jedes `section` übereinstimmt (z. B. wenn Sie einen Abschnitt/Kopfzeile „Hello world“ haben, sollte Ihre Navigationsleiste ein Element mit dem Text „Hello world“ enthalten).
1. Wenn Sie auf ein Element in der Navigationsleiste klicken, sollte die Seite zum entsprechenden Abschnitt des `#main-doc`-Elements navigieren (z. B. wenn Sie auf ein `.nav-link`-Element mit dem Text „Hello world“ klicken, navigiert die Seite zu einem `section`-Element mit dieser ID und der entsprechenden Kopfzeile).
1. Auf Geräten mit normaler Bildschirmgröße (Laptops, Desktops) sollte das Element mit `id="navbar"` auf der linken Seite des Bildschirms angezeigt werden und für den Benutzer immer sichtbar sein.
1. Ihre technische Dokumentation sollte mindestens eine Media Query verwenden.

**Hinweis:** Vergewissern Sie sich, dass Sie Ihr Stylesheet in Ihr HTML verlinken und Ihr CSS anwenden.

        <header>while statement</header>
        <article>
          A while statement executes its statements as long as a specified
          condition evaluates to true. A while statement looks as follows:

          <code>while (condition) statement</code> If the condition becomes
          false, statement within the loop stops executing and control passes to
          the statement following the loop.

          <p>
            The condition test occurs before statement in the loop is executed.
            If the condition returns true, statement is executed and the
            condition is tested again. If the condition returns false, execution
            stops and control is passed to the statement following while.
          </p>

          <p>
            To execute multiple statements, use a block statement ({ ... }) to
            group those statements.
          </p>

          Example:

          <p>
            The following while loop iterates as long as n is less than three:
          </p>

          <code>var n = 0; var x = 0; while (n &lt; 3) { n++; x += n; }</code>
          <p>
            With each iteration, the loop increments n and adds that value to x.
            Therefore, x and n take on the following values:
          </p>

          <ul>
            <li>After the first pass: n = 1 and x = 1</li>
            <li>After the second pass: n = 2 and x = 3</li>
            <li>After the third pass: n = 3 and x = 6</li>
          </ul>
          <p>
            After completing the third pass, the condition n &lt; 3 is no longer
            true, so the loop terminates.
          </p>
        </article>
