# --description--

`Fair`는 사람을 공평하게 대하고 올바르고 공정한 결정을 내리는 것을 의미합니다. 예를 들어, `The teacher is fair when grading tests`는 선생님이 학생들의 과제를 채점할 때 모두를 똑같이 대한다는 뜻입니다.

`Supportive`는 도움과 격려를 주는 사람을 묘사하는 형용사입니다. 이는 누군가나 무언가에게 도움이나 지원을 제공한다는 뜻의 동사 `support`에서 왔습니다.

예시:

`My coach is very supportive during training`는 코치가 팀을 많이 돕고 격려한다는 뜻입니다.

`My friends support me when I have problems`는 친구들이 어려운 시기에 당신을 돕고 격려한다는 뜻입니다.

`Be willing to`는 준비가 되어 있고 주저하지 않는 상태를 의미합니다. `be willing to` 다음에는 누군가가 하려는 행동을 설명하기 위해 기본 동사를 사용합니다. 예를 들어, `He is willing to learn new programming languages`는 그가 새로운 언어를 배우기 시작할 준비가 되어 있고 열린 마음임을 뜻합니다.
