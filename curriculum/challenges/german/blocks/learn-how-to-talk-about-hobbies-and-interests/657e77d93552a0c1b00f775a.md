# --description--

`Would you like...?` wird oft verwendet, wenn man eine Aktivität vorschlägt oder empfiehlt. Zum Beispiel:

`Would you like to test the new app with me?` – Es ist eine höfliche Einladung, gemeinsam am Testen zu arbeiten.

Wenn Linda `Would you like to go cycling with me one of these days?` sagt, ist das ähnlich wie „Hey, lass uns zusammen Rad fahren gehen.“
