# --explanation--

`First Conditional`은 미래의 실제 가능성과 가능한 상황을 설명할 때 사용합니다. `if` + `Present Simple` (조건절)과 `will` + 기본 동사 (결과절)로 구성됩니다. 예를 들어:

`If you debug the code, the app will run smoothly.` - 코드를 고치면 앱이 제대로 작동할 것입니다.

올바른 문장에서 조건은 `If we don't back up regularly`이고 결과는 `we could lose important information`입니다. 이는 실제 가능성을 나타냅니다.
