# --description--

الـ`extension` هو مكوِّن برمجي يضيف قدرات محددة إلى تطبيق برمجي أكبر. يمكن لـ`Extensions` توفير وظائف جديدة أو تحسين الميزات الموجودة.

على سبيل المثال، يمكن لـ `extension` في المتصفح إضافة وظائف خاصة إلى متصفح الويب لديك، مثل حجب الإعلانات أو إدارة كلمات المرور.

## --answers--
