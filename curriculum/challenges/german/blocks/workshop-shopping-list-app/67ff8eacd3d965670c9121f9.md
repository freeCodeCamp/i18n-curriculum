# --description--

Der `useCallback()`-Hook nimmt zwei Argumente entgegen: eine Funktion und ein Array von Abhängigkeiten. Die Funktion wird nur neu erstellt, wenn sich eine der Abhängigkeiten ändert.

Hier ist die grundlegende Syntax für `useCallback()`:

```js
const memoizedCallback = useCallback(() => {
  // Your function logic
  return value;
}, [dependency1, dependency2]);
```

Setzen Sie `toggleItem` gleich `useCallback()` und übergeben Sie ihm eine anonyme Funktion, die `item` als Argument annimmt. Verschieben Sie Ihre bestehende `toggleItem()`-Funktionslogik in die anonyme Funktion, wo `setSelectedItems` aufgerufen wird. Fügen Sie schließlich `setSelectedItems` als Abhängigkeit im Abhängigkeitsarray hinzu.

Danach sind Sie mit der Optimierung fertig! Ihre App sollte weiterhin gleich funktionieren, aber Sie sehen Ihre Konsolenprotokolle nur, wenn sich der `query`-Status ändert oder wenn die `toggleItem()`-Funktion neu erstellt wird.
