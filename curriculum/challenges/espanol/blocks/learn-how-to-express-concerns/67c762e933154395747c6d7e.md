# --explanation--

`Hands on deck` significa que se necesitan más personas para ayudar con una tarea o situación. Proviene de la navegación, donde todos los miembros de la tripulación eran necesarios en la cubierta del barco para manejar una situación urgente. Por ejemplo:

`We have a tight deadline, so we need all hands on deck.` - Esto significa que todos deben ayudar para completar el trabajo a tiempo.
