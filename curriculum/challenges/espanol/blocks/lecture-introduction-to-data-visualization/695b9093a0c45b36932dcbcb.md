# --description--

En las lecciones anteriores, se te presentó la visualización de datos, que es el proceso de crear formatos visuales claros para representar datos complejos. En esta lección, aprenderás sobre la biblioteca D3.

La biblioteca D3 es una biblioteca gratuita y de código abierto que puedes usar para crear visualizaciones de datos atractivas y efectivas en JavaScript.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-1.png" alt="A graph with four nodes labeled A, B, C, and D. Node A connects to B, C, and D. Node B connects to A and D. Node C connects to A. Node D connects to A and B. It's the same image as before.">


El nombre D3 es una abreviatura de data-driven documents, refiriéndose a cómo enlaza dinámicamente datos con elementos en el Document Object Model (DOM). Esta biblioteca es flexible y funciona de manera natural con la web. En lugar de tener que aprender una nueva estructura gráfica completa, puedes crear visualizaciones de datos con elementos `svg` y `canvas`.

Por ejemplo, si quieres crear un gráfico de barras usando D3, puedes hacer lo siguiente todo en un solo archivo JS:

* Declara variables que representen las dimensiones y márgenes del gráfico
    
* Declara la escala de posición horizontal
    
* Declara la escala de posición vertical
    
* Crea el contenedor SVG con las barras y etiquetas

Entonces, ¿cuáles son algunas otras razones para aprender D3 en lugar de otras herramientas de visualización como Chart.js, Plotly o Tableau?

A diferencia de otras herramientas, D3 te ofrece mucho control a bajo nivel donde puedes construir visualizaciones personalizables e interactivas con solo HTML, CSS y JavaScript puro. Además, la biblioteca está en JavaScript, así que no necesitas preocuparte por aprender un lenguaje completamente nuevo.

En los próximos talleres y laboratorios, aprenderás los fundamentos de la biblioteca D3, como selecciones, formas y escalas. Aprenderás estos conceptos clave construyendo gráficos de dispersión, gráficos de barras, aplicaciones de panel de control y más.
