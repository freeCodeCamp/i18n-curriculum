# --description--

學習使用 `how much` 和 `how many` 來詢問數量。`How much` 用於不可數名詞，詢問某物的體積或程度，例如 `time`、`water` 或 `money`。例如，`How much money do you have?` 是在詢問你擁有多少錢，錢是無法逐一計數的。

`How many` 用於可數名詞，當你可以計算項目時，例如 `books`、`apples` 或 `people`。例如，`How many books do you own?` 是在詢問可數的書本數量。
