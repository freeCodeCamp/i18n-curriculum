# --description--

JavaScript huzingatiwa kama lugha yenye aina zinazobadilika wakati wa utekelezaji. Hii ina maana kwamba vigezo vinaweza kupokea thamani yoyote wakati wa utekelezaji - kutangaza kigezo kama nambari hakuzuizi kuweka thamani ya mfuatano wa herufi baadaye, na vigezo vya kitendakazi vinaweza kupitishwa thamani yoyote.

Changamoto ya lugha yenye aina zinazobadilika ni kwamba ukosefu wa usalama wa aina unaweza kuleta makosa ikiwa hutakuwa makini. TypeScript inaongeza lugha ya JavaScript kwa kujumuisha aina thabiti, ambayo husaidia kugundua makosa hayo kabla hata hujasambaza msimbo wako. Lakini inafanya kazi vipi?

Fikiria kitendakazi hiki cha JavaScript:

```js
const getRandomValue = (array) => {
  return array[Math.floor(Math.random() * array.length)];
}
```

Kitendakazi chetu cha `getRandomValue` kinachukua safu ya data na kurudisha thamani ya bahati nasibu kutoka kwenye safu hiyo. Lakini kwa sababu JavaScript haithibitishi aina, hakuna kinachozuia kuitaja kitendakazi hicho na nambari:

```js
console.log(getRandomValue(10));
```

Matokeo ya konsoli kwa mfano huu yatarudisha `undefined` kwa sababu kilitarajia safu ya data badala ya nambari.

Lakini kwa TypeScript unaweza kufafanua aina ya kigezo cha safu ya data:

```js
const getRandomValue = (array: string[]) => {
  return array[Math.floor(Math.random() * array.length)];
}
```

Ufafanuzi huu wa aina unaambia TypeScript kwamba hoja ya safu lazima iwe safu ya mfuatano wa herufi. Kisha unapoitaja `getRandomValue` na kuipatia nambari, unapata aina tofauti ya kosa linaloitwa kosa la mkusanyaji msimbo.

Mazingira mengi ya utekelezaji ya JavaScript, kama kivinjari au Node.js, hayawezi kuendesha TypeScript moja kwa moja. Badala yake, unahitaji kwanza kukusanya, au kutafsiri, msimbo wa TypeScript kuwa JavaScript ya kawaida. Unaweza kufanya hivyo kwa mkusanyaji msimbo uliyojengwa ndani unaokuja na lugha ya TypeScript. Unapotekeleza mkusanyaji msimbo, TypeScript itakagua msimbo wako na tupa kosa kwa matatizo yoyote ambapo aina hazilingani - kama vile kupitisha nambari kwenye kitendakazi kinachotarajia safu ya data.

Utajifunza zaidi kuhusu jinsi aina hizi zinavyofanya kazi katika mafundisho yajayo. Lakini ni usalama huu wa ziada unaofanya TypeScript kuwa lugha inayovutia kwa wasanidi programu wengi na mashirika.
