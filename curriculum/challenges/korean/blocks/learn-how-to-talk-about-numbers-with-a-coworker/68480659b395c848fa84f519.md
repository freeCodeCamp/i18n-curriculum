# --explanation--

`front-end`는 소프트웨어의 사용자와 직접 마주하는 부분으로, 사용자가 보고 상호작용하는 모든 것(버튼, 메뉴, 레이아웃 등)을 포함합니다. 대화에서는 복합 형용사로 사용되며 `-`를 사용합니다. 예를 들어:

`The front-end team built the new homepage interface.` — 사용자가 보는 부분을 담당하는 팀이 메인 페이지의 새 디자인을 만들었습니다.

`Design`는 소프트웨어의 외관과 느낌, 즉 색상, 레이아웃, 아이콘, 타이포그래피, 사용 편의성을 의미합니다. 예를 들어:

`Good design improves how users experience the app.` — 앱이 보기 좋고 사용하기 쉬우면 사람들이 더 즐겁게 사용합니다.

David는 프로젝트의 시각적 요소와 상호작용 요소에 얼마나 많은 관심이나 예산이 할당되는지 묻고 있습니다.
