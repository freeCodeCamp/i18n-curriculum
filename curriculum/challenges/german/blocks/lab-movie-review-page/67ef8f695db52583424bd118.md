# --description--

**Ziel:** Erfüllen Sie die unten stehenden User Stories und bestehen Sie alle Tests, um die Aufgabe abzuschließen.

**User Stories:**

1. Sie sollten ein `main`-Element haben.
2. Innerhalb des `main`-Elements sollten Sie ein `h1`-Element für den Filmtitel haben.
3. Unterhalb des `h1`-Elements sollten Sie ein `img`-Element haben, das das Filmcover anzeigt. Ihr `img`-Element sollte einen beschreibenden `alt`-Text enthalten, der das Bild beschreibt. Sie können gerne das folgende Bild verwenden: `https://cdn.freecodecamp.org/curriculum/labs/rise-beyond-2.png`.
4. Sie sollten ein `p`-Element mit einer kurzen Filmbeschreibung haben.
5. Sie sollten ein weiteres `p`-Element haben, um die Filmbewertung anzuzeigen. Darin sollten sich die folgenden Elemente in der angegebenen Reihenfolge befinden:
   - Ein `strong`-Element mit dem Text `Movie Rating:`.
   - Ein `span`-Element mit einem `aria-hidden`-Attribut, das auf `true` gesetzt ist und eine visuelle Darstellung der Bewertung mit Sternen `⭐⭐⭐⭐⭐⭐⭐⭐⭐☆` enthält.
   - Ein numerischer Wert, der die Bewertung darstellt, in runden Klammern (z. B. `9.2/10`) nach dem span.
6. Sie sollten ein `h2`-Element mit dem Text `Cast Members` haben.
7. Sie sollten ein `ul`-Element haben.
8. Innerhalb des `ul`-Elements sollten Sie mehrere `li`-Elemente haben, von denen jedes ein `strong`-Element für den Namen des Schauspielers enthält, gefolgt vom entsprechenden Charakternamen, der vom Text `as` eingeleitet wird. (z. B. `James Holloway as Ethan Carter`).

# --hints--

Sie sollten ein `main`-Element haben.

```js
assert.isNotNull(document.querySelector("main"));
```

Innerhalb des `main`-Elements sollten Sie ein `h1`-Element mit dem Filmtitel haben.

```js
const h1 = document.querySelector("main > h1");
assert.isAbove(h1?.innerText.trim().length, 0);
```

Unterhalb des `h1`-Elements sollten Sie ein `img`-Element mit einem beschreibenden `alt`-Attribut haben.

```js
const img = document.querySelector("main > h1 + img");
assert.isNotNull(img);
assert.isNotEmpty(img.getAttribute("alt"));
```

Sie sollten ein `p`-Element nach dem Bild haben, das die Filmbeschreibung enthält.

```js
const description = document.querySelector("main img + p");
assert.isNotNull(description);
assert.isNotEmpty(description.innerText.trim());
```

Sie sollten ein weiteres `p`-Element haben, das die Bewertung enthält.

```js
const paragraphs = document.querySelectorAll("main p");
assert.isAtLeast(paragraphs.length, 2);
```

Innerhalb des zweiten `p`-Elements sollten Sie ein `strong`-Element mit dem Text `Movie Rating: ` haben.

```js
const strongEl = document.querySelector("main p:nth-of-type(2) strong");
assert.match(strongEl?.innerText.trim(), /^Movie Rating:?$/);
```

Es sollte ein `span`-Element innerhalb des Bewertungsabsatzes geben.

```js
const spanEl = document.querySelector("main p:nth-of-type(2) span");
assert.isNotNull(spanEl);
```

Das `span`-Element innerhalb des Bewertungsabsatzes sollte zehn Sterne enthalten, entweder ausgefüllt (⭐) oder leer (☆), gefolgt von einem numerischen Wert in runden Klammern nach dem `span`.

```js  
const spanEl = document.querySelector("main p:nth-of-type(2) span");
const fullRatingText = document.querySelector("main p:nth-of-type(2)")?.textContent.replace(/\s+/g, ' ').trim();

assert.match(spanEl?.innerText.trim(), /^[⭐☆]{10}$/);
assert.match(fullRatingText, /Movie Rating:?\s*[⭐☆]{10}\s*\(\s*\d+(?:\.\d+)?\s*\/\s*10\s*\)$/);
```

Das `span`-Element innerhalb des Bewertungsabsatzes sollte ein `aria-hidden`-Attribut haben, das auf `true` gesetzt ist.

```js
const spanEl = document.querySelector("main p:nth-of-type(2) span");
assert.strictEqual(spanEl?.getAttribute("aria-hidden"), "true");
```

Sie sollten ein `h2`-Element haben.

```js
assert.exists(document.querySelector("h2"));
```

Ihr `h2` sollte nach Ihrem zweiten `p`-Element kommen.

```js
const allParagraphs = document.querySelectorAll("p");
assert.isAtLeast(allParagraphs.length, 2);

const h2 = document.querySelector("h2");
const main = document.querySelector("main");

assert.exists(h2);
assert.exists(main);

const elements = Array.from(main.querySelectorAll("*"));
const lastPIndex = elements.indexOf(allParagraphs[1]);
const h2Index = elements.indexOf(h2);

assert.isAbove(h2Index, lastPIndex);
```

Ihr `h2` sollte den Text `Cast Members` haben.

```js
const h2 = document.querySelector("h2");
assert.equal(h2?.textContent.trim(), "Cast Members");
```

Sie sollten ein `ul`-Element nach dem `h2` haben.

```js
const ul = document.querySelector("main h2 + ul");
assert.isNotNull(ul);
```

Das `ul` sollte mehrere `li`-Elemente enthalten.

```js
const liItems = document.querySelectorAll("main ul li");
assert.isAbove(liItems.length, 1);
```

Jedes `li` sollte ein `strong`-Element für den Namen des Schauspielers enthalten, gefolgt vom entsprechenden Charakternamen, der vom Text `as` eingeleitet wird. (z. B. `James Holloway as Ethan Carter`).

```js
const liItems = document.querySelectorAll("main ul li");  
assert.isNotEmpty(liItems);
liItems.forEach(li => {
  const strong = li.querySelector("strong");
  assert.isNotNull(strong);
  const fullText = li.textContent.trim();
  const actorName = strong.textContent.trim();
  assert.isTrue(fullText.startsWith(actorName));
  const characterPart = fullText.slice(actorName.length).trim();
  assert.match(characterPart, /^as\s.+/);
});
```
