# --description--

`who`、`whom`、`which`、`that` のような関係代名詞は、関係節を名詞や代名詞に結びつけるために使われます。

`Who` と `whom` は人を指します。違いは、`who` が動詞の主語として使われるのに対し、`whom` は動詞や前置詞の目的語として使われる点です。

`Which` は特定の物や動物に使われ、`that` は定義的節で人、物、動物のいずれにも使えます。

例：

`The programmer who wrote this code is brilliant.`（`who` はプログラマーを指します）

`The artist whom we met at the gallery is internationally recognized.`（`whom` はアーティストを指します）

`The tool that you lent me was useful.`（`that` は道具を指します）

`The book, which I bought yesterday, is already a bestseller.`（`which` は本を指します）

# --questions--

## --text--

どの文が関係代名詞を正しく使っていますか？

## --answers--

`The teacher which teaches us math is very kind.`

### --feedback--

`which` は通常、人ではなく物に使われます。ここで正しい代名詞は `who` です。

---

`The car, that won the race, is very fast.`

### --feedback--

この文では `that` の使い方は文法的に正しいですが、コンマは不要です。正しくは：`The car that won the race is very fast.`

---

`She is the one who made the cake.`

---

`The flowers whom are on the table smell lovely.`

### --feedback--

`Whom` は人を指すときに動詞や前置詞の目的語に使われ、花のような物には使いません。ここで正しい代名詞は `which` または `that` です。
