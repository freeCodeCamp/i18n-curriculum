# --description--

Wenn Sie sagen, dass Sie `run into` eine Person, meinen Sie, dass Sie jemanden treffen, obwohl Sie es nicht erwartet haben. Zum Beispiel könnten Sie `run into` einen Freund im Laden treffen, obwohl Sie nicht wussten, dass er dort sein würde. Manchmal wird es einfach als Synonym für `meet` verwendet.

Hören Sie sich den Satz an und füllen Sie die Lücke aus.

# --fillInTheBlank--

## --sentence--

`Thanks for the description, Tom. I'll make sure to say hi when I BLANK into her.`

## --blanks--

`run`

### --feedback--

In Kombination mit `into` steht dieses Verb für das Treffen von jemandem, meist zufällig.
