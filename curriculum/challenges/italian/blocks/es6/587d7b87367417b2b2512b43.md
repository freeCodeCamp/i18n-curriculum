# --description--

In JavaScript, spesso non serve dare un nome alle funzioni, soprattutto quando si passa una funzione come argomento a un'altra funzione. Invece, creiamo funzioni in linea. Non serve nominare queste funzioni perché non le riutilizziamo altrove.

Per farlo, usiamo spesso questa sintassi:

```js
const myFunc = function() {
  const myVar = "value";
  return myVar;
}
```

ES6 ci offre una sintassi più semplice per non dover scrivere funzioni anonime in questo modo. Puoi invece usare la **sintassi delle funzioni freccia**:

```js
const myFunc = () => {
  const myVar = "value";
  return myVar;
}
```

Quando non c’è un corpo della funzione, ma solo un valore di ritorno, la sintassi delle funzioni freccia ti permette di omettere la parola chiave `return` e anche le parentesi che racchiudono il codice. Questo aiuta a semplificare funzioni più piccole in istruzioni su una sola riga:

```js
const myFunc = () => "value";
```

Questo codice restituirà comunque la stringa `value` per impostazione predefinita.
