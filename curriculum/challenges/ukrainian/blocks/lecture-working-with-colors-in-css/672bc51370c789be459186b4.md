# --interactive--

Працюючи з кольорами в CSS, важливо розуміти модель кольору RGB. RGB означає червоний, зелений і синій — основні кольори світла. Ці три кольори комбінуються з різною інтенсивністю, щоб створити широкий спектр кольорів.

Модель кольору RGB є адитивною, тобто кольори створюються шляхом поєднання світла з різною інтенсивністю.

Інтенсивність кожного кольору може варіюватися від `0` (що означає відсутність світла) до `255` (що означає повну інтенсивність світла). Змішуючи різні рівні червоного, зеленого та синього, ви можете отримати будь-який колір, який бачите на екрані. Наприклад:

- Чорний представлений `rgb(0, 0, 0)`, що означає відсутність світла для всіх трьох кольорів.
- Білий представлений `rgb(255, 255, 255)`, що означає повну інтенсивність світла для всіх трьох кольорів.
- Червоний представлений `rgb(255, 0, 0)`, з повною інтенсивністю червоного і відсутністю зеленого та синього.

У CSS функція `rgb()` дозволяє визначати кольори за допомогою моделі кольору RGB. Функція приймає три значення, кожне з яких представляє інтенсивність червоного, зеленого та синього світла відповідно. Ось базовий синтаксис:

```css
element {
  color: rgb(red, green, blue);
}
```

Значення для червоного, зеленого та синього можуть варіюватися від `0` до `255`, де `0` означає відсутність цього кольору, а `255` — повну насиченість. Ось приклад використання функції `rgb()` у CSS:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

Цей код змінить колір тексту абзацу на червоний, оскільки значення червоного встановлено на `255`, а зеленого і синього — на `0`.

CSS також надає функцію `rgba()`, яка додає четверте значення — альфа — що контролює прозорість кольору. Значення альфа варіюється від `0` (повністю прозорий) до `1` (повністю непрозорий). Ось приклад використання `rgba()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

Цей код застосує напівпрозорий синій фон до елемента `div`, оскільки значення червоного і зеленого встановлені на `0`, синього — на `255`, а альфа — на `0.5` (50% прозорості).

Модель кольору RGB особливо корисна для цифрових медіа, оскільки вона безпосередньо відповідає тому, як екрани відображають кольори. Монітори та дисплеї використовують крихітні червоні, зелені та сині пікселі для створення кольорів, які ви бачите. Контролюючи інтенсивність цих пікселів за допомогою значень RGB, ви можете отримати широкий спектр кольорів для свого вебдизайну.

Крім того, модель RGB ідеально підходить для роботи з динамічними дизайнами. Наприклад, якщо ви хочете створити анімації або колірні ефекти, які потребують змін у реальному часі, гнучкість функції RGB дозволяє програмно змінювати кольори.

Розуміючи модель кольору RGB і як використовувати функції `rgb()` та `rgba()` у CSS, ви отримаєте більший контроль над тим, як кольори відображаються на ваших вебсторінках. Ви зможете створювати яскраві дизайни, прозорість і навіть програмно налаштовувати кольори для динамічних ефектів.
