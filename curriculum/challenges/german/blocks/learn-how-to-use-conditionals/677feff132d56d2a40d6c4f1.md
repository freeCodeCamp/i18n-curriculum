# --explanation--

Maria fragt, `Did you notice something wrong?` Sie möchte wissen, ob es ein bestimmtes Problem gibt. Die Phrase `No. Not really` beruhigt sie, indem sie signalisiert, dass es nichts Ernstes gibt, worüber man sich Sorgen machen müsste. Sie ist höflich, locker und hält das Gespräch leicht. Zum Beispiel:  

Frage: `Is something wrong with the system?`  

Antwort: `No. Not really.` – Das bedeutet, dass es vielleicht kleinere Bedenken gibt, aber nichts Ernstes, das Anlass zur Sorge gibt.  

Denken wir nun darüber nach, warum die andere Option nicht funktioniert:  

`Yes, it's a big problem.` – Das deutet darauf hin, dass etwas Ernstes nicht stimmt, was Maria nicht beruhigt. Stattdessen könnte es sie noch besorgter machen.
