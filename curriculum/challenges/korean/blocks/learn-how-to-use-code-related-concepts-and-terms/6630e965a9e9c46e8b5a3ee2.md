# --description--

프로그래밍에서 `int`는 "정수"를 나타냅니다. 이는 분수나 소수 부분이 없는 정수를 저장하는 데 사용되는 데이터 유형입니다. 정수는 양수, 음수 또는 0일 수 있습니다.

`int` 데이터 유형을 선택하는 것은 개수, 배열의 인덱스 또는 분수 숫자가 필요 없는 모든 상황에서 전체 숫자를 다룰 때 필수적입니다.

예시:

항목 세기: `int itemCount = 5;`

나이 저장: `int age = 30;`

# --questions--

## --text--

프로그래밍에서 `int` 데이터 유형은 무엇을 나타내나요?

## --answers--

소수 숫자

### --feedback--

`int`는 소수 부분이 있는 숫자가 아니라 전체 숫자에 사용됩니다.

### --feedback--

`int`는 텍스트가 아니라 전체 숫자를 나타냅니다.

---

이미지 또는 그래픽

### --feedback--

`int`는 이미지나 그래픽이 아니라 숫자 값을 저장하는 데 사용됩니다.
