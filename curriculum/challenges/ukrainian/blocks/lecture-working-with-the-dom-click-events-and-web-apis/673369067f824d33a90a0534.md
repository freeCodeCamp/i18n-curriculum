# --interactive--

Коли користувач натискає на кнопку або відбувається зміна у формі, це називається подією. У ваших програмах потрібно мати спосіб слухати ці події та реагувати на них.

Метод `addEventListener()` використовується для прослуховування подій. Він приймає два аргументи: подію, яку ви хочете слухати, і функцію, яка буде викликана, коли ця подія станеться.

Ось базовий синтаксис:

```js
element.addEventListener("event", listener);
```

Елемент — це HTML-елемент, за яким слідкують для подій, а event вказує тип події для прослуховування, наприклад `"click"`.

Слухач — це об’єкт, який отримає повідомлення, коли подія станеться. Зазвичай це функція, яку ви визначаєте для обробки події.

Ось приклад:

```js
element.addEventListener("click", () => {
  // code to run when the click event occurs
});
```

Ви також можете передати референс на функцію ось так:

```js
function handleClick() {
  // code to run when the click event occurs
}

element.addEventListener("click", handleClick);
```

Іноді ви можете створити окрему функцію для обробки події. Це робить ваш код легшим для читання та підтримки.

Аргумент слухача також може бути `null`, або це може бути об’єкт, який використовує інтерфейс `EventListener`. Інтерфейс `EventListener` визначає один метод під назвою `handleEvent()`. Цей метод автоматично викликається щоразу, коли відбувається подія, яку ви слухаєте. Використання цього інтерфейсу дозволяє одному об’єкту обробляти кілька подій, якщо це потрібно.

Зазвичай вам не потрібно використовувати цей інтерфейс, і ви можете просто передати функцію, яка буде викликана при настанні події.

Тепер давайте розглянемо приклад, щоб краще зрозуміти, як це працює:

У цьому прикладі у нас є елемент `button` з ідентифікатором `btn`:

```html
<button id="btn">Show alert</button>
```

Ми хочемо слухати подію `"click"` на цій кнопці і показувати сповіщення, коли кнопку натискають.

Спочатку потрібно отримати доступ до цього елемента `button` у нашому JavaScript-коді. Потім потрібно додати слухача події, щоб слухати подію click і показувати сповіщення, коли кнопку натискають:

:::interactive_editor

```html
<button id="btn">Show alert</button>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("btn");

btn.addEventListener("click", () => alert("You clicked the button"));
```

:::

Тепер щоразу, коли кнопку натискають, користувач бачитиме повідомлення сповіщення на екрані ось так:

```md
You clicked the button
```

Ще один тип події, яку можна слухати, — це подія `"input"`. Вона спрацьовує, коли змінюється значення елемента введення.

Ось приклад:

```html
<input type="text" id="input" placeholder="Type something" />
```

Ми хочемо слухати подію input на цьому елементі введення і виводити значення введення в консоль щоразу, коли користувач щось вводить.

Спочатку потрібно отримати доступ до цього елемента `input` у нашому JavaScript-коді. Потім потрібно додати слухача події, щоб слухати подію input і виводити значення введення в консоль щоразу, коли користувач щось вводить:

:::interactive_editor

```html
<input type="text" id="input" placeholder="Type something" />
<script src="index.js"></script>
```

```js
const input = document.getElementById("input");

input.addEventListener("input", () => {
  console.log(input.value);
});
```

:::

Існує багато інших подій, які можна слухати за допомогою методу `addEventListener()`. Деякі загальні події включають `mouseover`, `mouseout`, `keydown`, `keyup` і `submit`.

У майбутніх уроках ми розглянемо більше подій і як використовувати метод `addEventListener()` для їх прослуховування.
