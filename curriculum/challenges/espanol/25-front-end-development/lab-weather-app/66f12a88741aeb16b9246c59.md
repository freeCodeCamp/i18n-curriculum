# --description--

In this lab, you'll build an app that provides weather information from different cities.

Utilizarás una API de clima. Los datos de salida tienen el siguiente formato:

```json
{
  "weather": [
    {
      "main": "Clear",
      "description": "clear sky",
      "icon": "https://cdn.freecodecamp.org/weather-icons/01n.png" // icon representing the weather
    }
  ],
  "main": {
    "temp": 2.62, // temperature in C
    "feels_like": 0.84, // temperature in C
    "temp_min": 1.72, // min temperature of the day in C
    "temp_max": 3.49, // max temperature of the day in C
    "pressure": 1010, // atmospheric pressure in hPa
    "humidity": 81 // humidity in %
  },
  "visibility": 10000, // distance in meters
  "wind": {
    "speed": 1.79, // speed of the wind in m/s
    "deg": 285, // orientation of the wind in degrees
    "gust": 3.13 // gust speed in m/s
  },
  "name": "London"
}
```

**Objetivo:** Cumplir con las historias de usuario a continuación y hacer que todas las pruebas pasen para completar el laboratorio.

**Historias de usuario:**

1. Deberías tener un elemento `button` con un `id` de `get-weather-btn`.

1. Deberías tener un elemento `select` con siete elementos `option` anidados en él. La primera opción debería tener una cadena vacía como su texto y atributo `value`. El resto debería tener lo siguiente para su texto y valores (con el valor en minúsculas):

   - Nueva York
   - Los Ángeles
   - Chicago
   - París
   - Tokio
   - Londres

1. Si no se selecciona ninguna ciudad, presionar el botón no debería hacer nada.

1. Si se selecciona una ciudad, deberían aparecer elementos para mostrar el clima:

   - Deberías tener un elemento `img` con el id `weather-icon` para mostrar el ícono del clima.

   - Deberías tener un elemento con el id `main-temperature` para mostrar la temperatura principal.

   - Deberías tener un elemento con el id `feels-like` para mostrar cómo se siente la temperatura.

   - Deberías tener un elemento con el id `humidity` para mostrar la cantidad de humedad en el aire.

   - Deberías tener un elemento con el id `wind` para mostrar la velocidad del viento.

   - Deberías tener un elemento con el id `wind-gust` para mostrar las ráfagas de viento.

   - Deberías tener un elemento con el id `weather-main` para mostrar el tipo principal de clima.

   - Deberías tener un elemento con el id `location` para mostrar la ubicación actual.

1. Deberías tener una función asincrónica llamada `getWeather` que obtiene la información del clima de la API `https://weather-proxy.freecodecamp.rocks/api/city/<CITY>` y la devuelve. Ten en cuenta que esta API devuelve datos usando el sistema métrico, es decir, m/s para velocidad del viento, y Celsius para la temperatura.

1. La función asincrónica `getWeather` debe aceptar una ciudad como argumento.

1. Deberías manejar cualquier error que ocurra dentro de la función `getWeather` y registrarlos en la consola.

1. Deberías tener una función asincrónica `showWeather` que acepte una ciudad como parámetro.

1. La función `showWeather` debería llamar a la función `getWeather` para recuperar los datos del tiempo para la ciudad seleccionada del desplegable.

1. Si la función `getWeather` tuvo un error, la aplicación solo debería mostrar una alerta que diga `Algo salió mal, por favor inténtalo de nuevo más tarde`.

1. Si los datos de `getWeather` son utilizables, la función `showWeather` debería mostrar los datos del tiempo en los elementos correspondientes. Si un cierto valor de la API es `undefined`, deberías escribir `N/A` en el elemento correspondiente.

NOTA: Las pruebas llevarán tiempo para completarse. Mientras veas `// running tests` en la consola, se están ejecutando.

const getWeatherBtn = document.getElementById('get-weather-btn');
const selectEl = document.getElementById('location-selector');

const weatherInfoEl = document.getElementById('weather-info');
const iconEl = document.getElementById('weather-icon');
const tempEl = document.getElementById('main-temperature');
const feelEl = document.getElementById('feels-like');
const humidityEl = document.getElementById('humidity');
const windEl = document.getElementById('wind');
const gustEl = document.getElementById('wind-gust');
const mainEl = document.getElementById('weather-main');
const locationEl = document.getElementById('location');
const arrowEl = document.getElementById('compass-arrow');

getWeatherBtn.addEventListener(
  'click',
  () => selectEl.value && showWeather(selectEl.value)
);

async function showWeather(city) {
  const json = await getWeather(city);
  const {
    weather,
    main: { temp, feels_like, humidity },
    wind: { speed, gust, deg = 0 },
    name
  } = json;

  const { main, icon } = weather[0];

  weatherInfoEl.style.display = 'block';

  iconEl.src = icon || '';
  tempEl.innerHTML = temp ? `${temp}&deg; C` : 'N/A';
  feelEl.innerHTML = `Feels Like: ${feels_like ? `${feels_like}&deg; C` : 'N/A'}`;
  humidityEl.innerHTML = `Humidity: ${humidity ? `${humidity}%` : 'N/A'}`;
  windEl.innerHTML = `Wind: ${speed ? `${speed} m/s` : 'N/A'}`;
  gustEl.innerHTML = `Gusts: ${gust ? `${gust} m/s` : 'N/A'}`;

  mainEl.innerHTML = main || 'N/A';
  locationEl.innerHTML = name || 'N/A';
  arrowEl.style.transform = `rotate(${deg}deg)`;
}

async function getWeather(city) {
  try {
    const response = await fetch(
      `https://weather-proxy.freecodecamp.rocks/api/city/${city}`
    );

    if (!response.ok) {
      alert('Something went wrong, please try again later');
    }

    const json = await response.json();
    return json;
  } catch (error) {
    console.error(error.message);
  }
}
