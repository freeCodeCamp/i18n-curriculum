# --description--

`Patterns` 是某事发生或完成的规律且重复的方式。在数据中，`patterns` 可以显示趋势或行为。 

例如，客户数据中的一个 `pattern` 可能是更多人使用弓作为配饰。这是一个 `trend pattern`。

还有一些 `cycle patterns`，比如咖啡店早晨卖出更多咖啡。 

`Patterns` 可以出现在数字中，比如每月销售额增加，或者出现在行为中，比如客户更喜欢在线购物。
