**Ziel:** Erstellen Sie eine App, die funktional ähnlich ist wie diese: <https://codepen.io/freeCodeCamp/full/JqdoMV>.

Erfüllen Sie die untenstehenden User Stories und bringen Sie alle Tests zum Bestehen. Verwenden Sie beliebige Bibliotheken oder APIs, die Sie benötigen. Verleihen Sie ihr Ihren eigenen Stil.

**User Story:** Ich kann eine Liste der neuesten Beiträge im freeCodeCamp-Forum sehen.

**User Story:** Für jedes Thema kann ich den Titel und eine Liste von Links zu den Benutzern sehen, die kürzlich darin gepostet haben.

**User Story:** Ich kann die Anzahl der Antworten und Ansichten sehen, die jedes Thema hatte, sowie einen Zeitstempel, wann das Thema zuletzt aktiv war.

**Hinweis:** Um die 30 neuesten Forenbeiträge zu erhalten: <https://forum-proxy.freecodecamp.rocks/latest>.

Wenn Sie fertig sind, fügen Sie einen Link zu Ihrem Projekt auf CodePen ein und klicken Sie auf die Schaltfläche „Ich habe diese Aufgabe abgeschlossen“.
