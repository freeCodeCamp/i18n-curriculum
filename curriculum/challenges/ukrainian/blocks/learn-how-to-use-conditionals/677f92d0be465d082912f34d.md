# --explanation--

Питання `Did you check that?` запитує, чи хтось перевірив або підтвердив щось.

У цьому контексті Марк хоче дізнатися, чи перевірила Сара підключення кабелю живлення, щоб переконатися, що воно надійне. Інші приклади:

- `Did you check that the windows are closed?` — це означає запитати, чи хтось переконався, що вікна надійно зачинені.

- `Did you check that the light is off?` — це означає запитати, чи хтось підтвердив, що світло вимкнене.
