# --explanation--

Die `Present Continuous`-Zeitform wird verwendet, um Handlungen oder Zustände zu beschreiben, die gerade jetzt oder über einen Zeitraum hinweg stattfinden. Zum Beispiel:

- `The system is freezing` – Es bedeutet, dass es aktuell nicht reagiert.
