# --interactive--

関数は特定のタスクを実行したり値を計算したりする再利用可能なコードのかたまりです。関数は、何らかの入力を受け取り、それに対して処理を行い、結果を出力する機械のようなものだと考えてください。以下は関数を宣言する例です。

```js
function greet() {
  console.log("Hello, Jessica!");
}
```

この例では、`greet`という関数を宣言しました。その関数の中で、`console.log`が`Hello, Jessica!`というメッセージをコンソールに出力します。このコードを実行しても、メッセージはコンソールに表示されません。なぜなら、関数を呼び出す必要があるからです。

関数の呼び出し（または実行）とは、実際に関数を使うことを指します。関数を呼び出すには、関数名の後に丸括弧を付けて参照します。

:::interactive_editor

```js
function greet() {
  console.log("Hello, Jessica!");
}

greet(); // "Hello, Jessica!"
```

:::

これで`Hello, Jessica!`というメッセージがコンソールに表示されます。でも、もしメッセージを`Hello, Nick!`や`Hello, Anna!`にしたい場合はどうでしょう？異なるユーザーに挨拶するたびに新しい関数を書くのは避けたいですよね。代わりに、関数のパラメータと引数を使って再利用可能な関数を作成できます。

パラメータは、関数が呼び出されたときに渡される値のプレースホルダーとして機能します。これにより関数は入力を受け取り、その入力を使って処理できます。引数は、関数が呼び出されたときに実際に渡される値です。以下はパラメータと引数を使った`greet`関数の更新版です。

:::interactive_editor

```js
function greet(name) {
  console.log("Hello, " + name + "!");
}

greet("Alice"); // Hello, Alice!
greet("Nick"); // Hello, Nick!
```

:::

`name`がパラメータとして機能し、文字列の`Alice`と`Nick`が引数として使われています。これで、異なる引数を使ってコード内で何度も再利用できる関数ができました。

関数が実行を終えると、必ず戻り値を返します。デフォルトでは戻り値は`undefined`になります。以下はその例です。

:::interactive_editor

```js
function doSomething() {
  console.log("Doing something...");
}

let result = doSomething();
console.log(result); // undefined
```

:::

関数に特定の値を戻り値として返してほしい場合は、`return`文を使う必要があります。以下は2つの値の合計を返す`return`文の例です。

:::interactive_editor

```js
function calculateSum(num1, num2) {
  return num1 + num2;
}

console.log(calculateSum(3, 4)); // 7
```

:::

多くの場合、`return`文を使います。なぜなら、関数から出力された値を後でコード内で使えるからです。

ここまで名前付き関数を扱ってきましたが、無名関数と呼ばれる名前のない関数を作ることもできます。無名関数は変数に代入できます。

:::interactive_editor

```js
const sum = function (num1, num2) {
  return num1 + num2;
};

console.log(sum(3, 4)); // 7
```

:::

この例では、`const`という変数`sum`に、`num1`と`num2`の合計を返す無名関数を代入しています。次に`sum`を呼び出し、`3`と`4`の数字を渡して`7`の結果を得ています。

関数はデフォルトパラメータをサポートしており、パラメータのデフォルト値を設定できます。関数がそのパラメータの引数なしで呼び出された場合にデフォルト値が使われます。以下は例です。

:::interactive_editor

```js
function greetings(name = "Guest") {
  console.log("Hello, " + name + "!");
}

greetings(); // Hello, Guest!
greetings("Anna"); // Hello, Anna!
```

:::

この例では、`name`の引数が指定されなければ、`Guest`がデフォルト値になります。

まとめると、関数は再利用可能で整理されたコードを書くことを可能にします。入力（パラメータ）を受け取り、処理を行い、出力を返します。
