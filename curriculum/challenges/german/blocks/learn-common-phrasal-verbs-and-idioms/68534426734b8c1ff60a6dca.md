# --explanation--

`To do just that` bedeutet, genau so zu handeln, wie zuvor beschrieben oder beabsichtigt. Zum Beispiel:

`She said she would fix the code, and she did just that.` – Das bedeutet, sie hat den Code genau so repariert, wie sie gesagt hat.
