# --description--

Eine `statement` in der Programmierung ist eine Codezeile, die eine bestimmte Aktion ausführt. Zum Beispiel ist `print("Hello World")` eine Anweisung, die Text anzeigt.

Ein `condition` ist ein Test im Code, der überprüft, ob etwas wahr oder falsch ist. Zum Beispiel prüft `if x > 5`, ob `x` größer als 5 ist.

Ein `conditional` ist im Allgemeinen ein Adjektiv, das sich auf etwas bezieht, das von einem `condition` oder `conditions` abhängt. In der Programmierung wird es verwendet, um jeden Ausdruck oder jede Funktion zu beschreiben, die logische `conditions` beinhaltet. Es ist nicht auf eine explizite „if-then“-Struktur beschränkt, sondern kann auch jede Einrichtung bezeichnen, die das Verhalten basierend auf `conditions` ändert, einschließlich Schleifen oder sogar ternärer Operatoren.

Eine `conditional statement` ist eine Anweisung, die nur ausgeführt wird, wenn eine bestimmte `condition` wahr ist. Zum Beispiel ist `if x > 5: print("Large")` eine `conditional statement`. Sie gibt `Large` nur aus, wenn `x` größer als 5 ist.

### --feedback--

Eine `conditional statement` beinhaltet das Überprüfen von `conditions`; sie ist keine Codezeile, die ohne Überprüfungen ausgeführt wird.

### --feedback--

Obwohl eine `conditional statement` helfen kann, zu überprüfen, wie ein Programm funktioniert, wird sie hauptsächlich verwendet, um Code basierend auf `conditions` auszuführen, nicht um das Programm zu testen.

### --feedback--
