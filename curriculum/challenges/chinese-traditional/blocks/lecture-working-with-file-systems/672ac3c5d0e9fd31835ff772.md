# --description--

要在 Windows 使用 Explorer 創建檔案，你可以點擊左上角的「New」選項。這將顯示你可以創建的不同檔案型別的列表。 

你也可以在對應的應用程式中創建檔案。讓我們創建一個程式本文檔來向你展示這個處理程序。

當你點擊「New」並選擇檔案型別時，你會在目前資料夾中看到一個具有預設名稱的新程式本文檔。你可以根據需要重新命名它。

建立資料夾的處理程序完全相同。點擊左上角的「New」按鈕，然後選擇「Folder」選項，或是在空白處按右鍵並前往「New」，接著選擇「Folder」選項。你會在目前位置看到一個新的空資料夾。

你可以透過選取檔案和資料夾，然後點擊頂端的「重新命名」選項，來重新命名它們。當檔案的當前名稱被凸顯標示時，你可以開始輸入檔案的新名稱。按下 Enter 鍵以確認變更。

或者，你可以在檔案或資料夾上按右鍵並選擇「重新命名」選項，或使用 F2 鍵盤快速鍵。輸入新名稱並按 Enter 鍵以確認變更。 

如果你已將目標資料夾以分頁方式開啟，你可以透過拖放將檔案或資料夾移動到另一個資料夾中。 

你也可以選擇該檔案或資料夾，然後在頂部選擇「剪下」選項。

然後，前往目標資料夾，並點擊上方的「貼上」按鈕進行貼上。如果你先前複製或剪下了該檔案，這個按鈕將會被啟用。

你也可以透過在空白處按右鍵，然後點擊貼上圖示來完成此操作。這將會將檔案或資料夾從其原始位置移除。

要刪除一個檔案或資料夾，請點擊它，然後點擊頂部的垃圾桶圖示。若你對它按右鍵，也會找到「刪除」選項。

太好了。現在讓我們看看你如何在 macOS 上使用 Finder 來做到這一點。 

Finder 是 macOS 上的預設檔案管理員。

要在 macOS 上創建檔案，你需要打開一個允許你創建所需文件型別的應用程式。目前，Finder 中沒有內建的方式可以直接創建檔案。

例如，要創建一個簡單的純文字程式本文檔，你可以打開 TextEdit 並存檔你的檔案。這個處理程序對任何類型的檔案都完全相同。

有許多方法可以開啟 TextEdit。其中一種是前往 Launchpad。然後，搜尋一個你可以創建你感興趣的檔案類型的應用程式。在這個例子中，那就是「TextEdit」。點擊圖示以開啟該應用程式。

一旦你進入應用程式，你應該會在 `File` 選單中看到一個存檔你的檔案的選項。此選單位於左上角，緊鄰 Apple 圖示旁邊。

在其中一個 App 選單選項中，你應該會看到一個將檔案存檔的指令。這通常位於檔案選單下。

在旁邊，你會看到儲存你的檔案的鍵盤快速鍵。這通常是在 macOS 上的 `Command + S`。

當你第一次儲存檔案時，你應該會看到一個對話窗，在那裡你可以輸入檔案名稱並選擇位置。

通常，副檔名是由應用程式自動添加，但你可以在儲存檔案之前更改它。

在存檔該檔案後，你可以點擊 Dock 中的 Finder 圖示來開啟 Finder。你應該會在你存檔的資料夾中看到你的新檔案。

要創建新資料夾，你應該前往你想要創建該新資料夾的位置，然後在空白處按右鍵。你會看到一個選項列表。第一個選項是「New Folder」。

如果你點擊它，你可以為你的新資料夾指定名稱。輸入名稱並按 Enter 鍵確認。

在 Finder 中有多種方法可以移動檔案和資料夾。如果你需要將它們移動到列在 Finder「最愛」區段的資料夾，你只需拖放即可。

如果你需要將它移動到不同的資料夾，你有兩個選項。 

你可以將目標資料夾以分頁方式開啟，然後將檔案或資料夾拖放到該分頁中。

或者你可以使用鍵盤快速鍵。首先，使用 `Command + C` 複製檔案，前往新資料夾，然後使用 `Command + Option + V` 將檔案或資料夾貼到該位置。這也會將它從原始資料夾中移除。

要刪除一個檔案或資料夾，你可以對它按右鍵並選擇「移動到垃圾桶」。

你也可以將它們拖放到 Dock 上的垃圾桶圖示。這是等價的。

當檔案或資料夾在垃圾桶中時，你可以右鍵點擊它立即刪除，或者你可以清空垃圾桶以永久移除所有已刪除的檔案和資料夾。

了解如何在 Windows 和 macOS 上創建、移動及刪除檔案和資料夾非常重要。掌握這些技能後，你可以整理你的檔案、快速找到它們，並透過刪除不必要的檔案來釋放你電腦上的儲存空間。
