什么是 JSON 以及如何使用方括弧和点符号访问值？

JSON 代表 JavaScript 对象表示法。它是一种轻量级、基于文本的数据格式，通常用于在服务器和网页应用之间交换数据。 

JSON 在网页开发中如此流行的原因之一是它既可被机器 `解析`，又可被人类 `读取`。   

由于 JSON 是语言无关的，你可以轻松地将 JSON 数据从 Java 应用发送到 Python 应用，或从 JavaScript 应用发送到 C# 应用。

JSON 支持多种数据类型，包括对象、数组、字串、布尔、null 和数字。 

这是一个 JSON 对象的示例：

```js
{
  "name": "Alice",
  "age": 30,
  "isStudent": false,
  "list of courses": ["Mathematics", "Physics", "Computer Science"]
}
```

如你所见，JSON 使用密钥-值点对来保存信息，每个点对之间用逗号分隔。每个密钥必须用双引号包裹，否则你将得到一个误差。

要从 JSON 对象中访问数据，你可以使用点符号或方括弧符号。在此示例中，我们使用点符号来访问 JSON 对象中的 `age`：

```js
import data from "./example.json" with { type: "json" };

console.log(data.age);
```

这个特定的示例使用了所谓的 `import` 语句，它将 JSON 对象导入到这个文件中，以便我们可以访问它。你将在未来的课程中学习更多关于 `import` 语句的内容。

你也可以使用方括弧表示法来访问 JSON 对象中的信息。以下是访问 `list of courses` 数组的示例：

```js
import data from "./example.json" with { type: "json" };

console.log(data["list of courses"]);
```

在这里使用方括弧表示法特别有用，因为密钥包含由空格分隔的多个单词。如果我们尝试使用点表示法，将导致误差。

总之，JSON 是一种多功能格式，可以保存多种数据类型，包括数组和嵌套对象。通过使用点表示法或方括弧表示法，你可以轻松访问保存在 JSON 对象中的值。
