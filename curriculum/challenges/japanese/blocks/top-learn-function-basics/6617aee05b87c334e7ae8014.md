# --description--

関数の中で宣言された変数は、その関数の中でのみ見えます。これを変数のスコープと呼び、この特定のスコープはローカルスコープと呼ばれます。関数の外で宣言された変数はグローバル変数と呼ばれ、プログラム全体で見えます。例えば、次のコードはエラーになります。

```js
function myFunction() {
  let x = 10;
}

console.log(x);
```

変数`x`は関数`myFunction`の中で宣言されているため、関数の外では見えません。このコードは`ReferenceError`が定義されていないため、`x`を発生させます。

関数の外で変数を宣言すると、それはグローバル変数と呼ばれ、プログラム全体で見えます。例えば、次のコードは動作します。

```js
let x = 10;

function myFunction() {
  console.log(x);
}

myFunction();
```

もし関数の中で同じ名前の変数が宣言されると、外側の変数を隠します。
