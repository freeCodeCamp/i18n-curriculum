# --description--

現在讓我們深入了解從 90 到 99 的數字。

在西班牙語中，數字 90 稱為 `noventa`。

這些數字是由使用十位數字的字 `noventa`（90），接著是連接詞 `y`（"and"），然後是個位數字（1-9）所組成。

| Number | Spanish |
| :--- | :--- |
| 90 | `Noventa` |
| 91 | `Noventa y uno` |
| 92 | `Noventa y dos` |
| 93 | `Noventa y tres` |
| 94 | `Noventa y cuatro` |
| 95 | `Noventa y cinco` |
| 96 | `Noventa y seis` |
| 97 | `Noventa y siete` |
| 98 | `Noventa y ocho` |
| 99 | `Noventa y nueve` |

數字 `noventa y uno` (91) 在這個範圍中是獨特的，因為它在陽性名詞前會變成 `noventa y un`。例如：

`Noventa y un años` - 九十一年。
