# --description--

Bob의 문장은 과거에 진행 중이던 동작을 설명하기 위해 과거 진행형 `I was checking`을 사용합니다.

과거 진행형 문장은 과거형 `was` 또는 `were` + `verb + ing`를 결합하여 만들 수 있습니다.

또한 그는 조치를 제안하기 위해 조동사 `should`를 사용합니다.
