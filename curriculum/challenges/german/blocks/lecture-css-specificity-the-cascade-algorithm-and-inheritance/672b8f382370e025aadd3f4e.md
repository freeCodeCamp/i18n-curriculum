# --interactive--

Vererbung ist ein zentrales Konzept in CSS, das bestimmt, wie Styles von Elternelementen an ihre Kindelemente weitergegeben werden.

Ähnlich wie in der realen Welt, wo Kinder oft Eigenschaften von ihren Eltern erben, können in CSS bestimmte Eigenschaften von Kindelementen von ihren Elternelementen übernommen werden.

Das ermöglicht eine effizientere Art, einheitliche Gestaltung über ein gesamtes Dokument hinweg anzuwenden.

In CSS werden nicht alle Eigenschaften standardmäßig vererbt. Zum Beispiel werden Eigenschaften wie `color`, `font-family` und `line-height` vererbt. Das bedeutet, wenn Sie die Textfarbe bei einem Elternelement festlegen, erben alle seine Kindelemente diese Farbe, sofern Sie sie nicht explizit überschreiben.

Betrachten Sie zum Beispiel folgendes Beispiel, bei dem ein Elternelement `div` seine Farbe mit einem inline Style setzt und das Kindelement `p` diese Farbe erbt:

:::interactive_editor

```html
<div style="color: blue;">
  This is the parent element.
  <p>This is the child element inheriting the color.</p>
</div>
```

:::

In diesem Fall zeigen sowohl das Elternelement `div` als auch das Kindelement `p` ihren Text in Blau an, weil die Farbe vererbt wird.

Andererseits werden Eigenschaften wie `margin`, `padding`, `border` und `background` standardmäßig nicht vererbt. Wenn Sie möchten, dass ein Kindelement diese Styles erbt, müssen Sie sie explizit setzen, entweder direkt am Kindelement oder mit dem Schlüsselwort `inherit`.

Das Schlüsselwort `inherit` kann verwendet werden, um die Vererbung einer Eigenschaft von einem Elternelement zu erzwingen, selbst wenn diese Eigenschaft normalerweise nicht vererbt wird.

Zum Beispiel, wenn ein bestimmtes Kindelement denselben `padding` wie sein Elternelement haben soll, können Sie `padding: inherit` am Kindelement setzen:

:::interactive_editor

```html
<div style="padding: 20px;">
  This is the parent element with padding.
  <p style="padding: inherit;">This is the child element inheriting the padding.</p>
</div>
```

:::

In diesem Fall erbt das Kindelement `p` den Innenabstand `20px` von seinem Elternelement `div`.

Vererbung ist besonders nützlich, um Konsistenz zu bewahren und Redundanz in Ihren Stylesheets zu reduzieren.

Anstatt dieselbe Stilregel für mehrere Elemente zu schreiben, können Sie sie einmal am Elternelement definieren, und die Kindelemente erben sie. Das macht Ihr CSS prägnanter und leichter zu verwalten.

Es ist jedoch wichtig zu beachten, dass Vererbung nur in eine Richtung funktioniert – vom Eltern- zum Kindelement. Wenn Sie einen Stil bei einem Kindelement überschreiben, wirkt sich das nicht auf das Elternelement aus.
