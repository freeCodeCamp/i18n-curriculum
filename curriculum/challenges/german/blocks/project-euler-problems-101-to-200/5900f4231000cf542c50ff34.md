# --description--

Wenn man drei schwarze Objekte $B$ und ein weißes Objekt $W$ hat, können diese auf 7 Arten so gruppiert werden:

$$(BBBW)\\;(B,BBW)\\;(B,B,BW)\\;(B,B,B,W)\\;(B,BB,W)\\;(BBB,W)\\;(BB,BW)$$

Auf wie viele Arten können sechzig schwarze Objekte $B$ und vierzig weiße Objekte $W$ auf diese Weise gruppiert werden?
