# --instructions--

使用 Python 創建一個埠掃描器。

在 `port_scanner.py` 檔案中，創建一個名為 `get_open_ports` 的函式（程式），該函式接受一個 `target` 引數和一個 `port_range` 引數。`target` 可以是 URL 或位址。`port_range` 是一個包含兩個數字的列表，表示要檢查的埠範圍的起始和結束數字。

以下是函式可能被呼叫的範例：

```py
get_open_ports("209.216.230.240", [440, 445])
get_open_ports("www.stackoverflow.com", [79, 82])
```

該函式（程式）應傳回指定範圍內的開啟埠列表。

`get_open_ports` 函式（程式）也應該接受第三個選用引數 `True`，以顯示「詳細」模式。如果這個引數設為 true，函式（程式）應該傳回描述性字串，而不是埠的列表。

以下是詳細模式中應該傳回的字串格式（`{}` 內的文字指出應該出現的資訊）：

```bash
Open ports for {URL} ({IP address})
PORT     SERVICE
{port}   {service name}
{port}   {service name}
```

你可以使用 `common_ports.py` 中的字典來取得每個埠的正確服務名稱。

例如，如果函式（程式）是這樣被呼叫的：

```py
port_scanner.get_open_ports("scanme.nmap.org", [20, 80], True)
```

它應該傳回以下內容：

```bash
Open ports for scanme.nmap.org (45.33.32.156)
PORT     SERVICE
22       ssh
80       http
```

請確保包含適當的空格和換行字元。

如果傳入 `get_open_ports` 函式的 URL 無效，該函式應該傳回字串："Error: Invalid hostname"。

如果傳入 `get_open_ports` 函式（程式）的 IP 位址無效，該函式（程式）應該傳回字串："Error:  Invalid IP address"。
