# --interactive--

In JavaScript haben alle arithmetischen Operatoren eine Form der zusammengesetzten Zuweisung. Zusammengesetzte Zuweisungsoperatoren bieten eine kurze Schreibweise für eine Operation an einer Variablen, gefolgt von der Speicherung des Ergebnisses in derselben Variablen. Sie kombinieren die Operation und die Zuweisung zu einer kürzeren Form wie `x += y`, was dem Schreiben von `x = x + y` entspricht, aber ohne den Variablennamen zu wiederholen. Anstatt also so etwas zu schreiben:

:::interactive_editor

```js
let num = 5;
num = num + 2;

console.log(num); // 7
```

:::

können Sie so etwas schreiben:

:::interactive_editor

```js
let num = 5;
num += 2;

console.log(num); // 7
```

:::

Beachten Sie, wie `num += 2` sowohl den Additions- als auch den Zuweisungsschritt in einem kombiniert. Das spart Zeit und reduziert Unordnung in Ihrem Code. Schauen wir uns die gebräuchlichsten zusammengesetzten Zuweisungsoperatoren in JavaScript genauer an.

Wie Sie bereits gesehen haben, erlaubt Ihnen der `+=`-Operator, einen Wert zu einer bestehenden Variablen hinzuzufügen. Er ist als Additionszuweisungsoperator bekannt. Der Additionszuweisungsoperator nimmt den aktuellen Wert der Variablen, addiert die angegebene Zahl dazu und weist das Ergebnis dann wieder der Variablen zu:

:::interactive_editor

```js
let total = 10;
total += 5;

console.log(total); // 15
```

:::

Wie Sie sich denken können, gibt es auch einen Subtraktionszuweisungsoperator, der durch `-=` dargestellt wird. Der Subtraktionszuweisungsoperator subtrahiert den angegebenen Wert vom aktuellen Wert der Variablen und weist den neuen Wert wieder der Variablen zu:

:::interactive_editor

```js
let score = 20;
score -= 7;

console.log(score); // 13
```

:::

Wenn Sie die Subtraktionszuweisung nicht verwendet hätten, hätten Sie so etwas gemacht:

:::interactive_editor

```js
let score = 20;
score = score - 7;

console.log(score); // 13
```

:::

Der Multiplikationszuweisungsoperator wird durch `*=` dargestellt. Er multipliziert den aktuellen Wert der Variablen mit der angegebenen Zahl und weist das Ergebnis wieder der Variablen zu:

:::interactive_editor

```js
let points = 5;
points *= 3;

console.log(points); // 15
```

:::

Schließlich gibt es noch den Divisionszuweisungsoperator, der durch `/=` dargestellt wird. Wie die anderen zuvor ermöglicht er es Ihnen, den aktuellen Wert einer Variablen durch eine von Ihnen angegebene Zahl zu teilen und das Ergebnis dann wieder der Variablen zuzuweisen:

:::interactive_editor

```js
let balance = 100;
balance /= 4;

console.log(balance); // 25
```

:::

Denken Sie daran, dass es für jeden Operator in JavaScript einen zusammengesetzten Zuweisungsoperator gibt. Neben den vier bereits genannten haben wir auch:

- Den Restzuweisungsoperator (`%=`), der eine Variable durch die angegebene Zahl teilt und den Rest der Variablen zuweist.

- Den Exponentzuweisungsoperator (`**=`), der eine Variable auf die Potenz der angegebenen Zahl hebt und das Ergebnis der Variablen neu zuweist.

- Den bitweisen UND-Zuweisungsoperator (`&=`), der eine bitweise UND-Operation mit der angegebenen Zahl durchführt und das Ergebnis der Variablen neu zuweist.

- Den bitweisen ODER-Zuweisungsoperator (`|=`), der eine bitweise ODER-Operation mit der angegebenen Zahl durchführt und das Ergebnis der Variablen neu zuweist.
