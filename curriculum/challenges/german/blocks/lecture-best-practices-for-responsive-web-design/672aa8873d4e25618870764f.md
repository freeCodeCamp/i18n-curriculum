# --interactive--

Responsives Webdesign ist ein Ansatz in der Webentwicklung, der darauf abzielt, Websites zu erstellen, die auf einer Vielzahl von Geräten – von Desktop-Computern bis hin zu Mobiltelefonen – ein optimales Anzeige- und Nutzungserlebnis bieten.

Das Kernprinzip des responsiven Designs ist Anpassungsfähigkeit – die Fähigkeit einer Website, ihr Layout und ihren Inhalt basierend auf der Bildschirmgröße und den Fähigkeiten des Geräts, auf dem sie betrachtet wird, anzupassen.

Responsives Design basiert typischerweise auf drei Hauptkomponenten – flüssigen Grids, flexiblen Bildern und Media Queries:

- Flüssige Grids verwenden relative Einheiten wie Prozentsätze anstelle von festen Einheiten wie Pixeln, sodass sich der Inhalt je nach Bildschirmgröße anpassen und umfließen kann.

- Flexible Bilder sind so eingestellt, dass sie sich innerhalb ihrer Container skalieren, wodurch sichergestellt wird, dass sie auf kleineren Bildschirmen nicht aus ihren Containern herausragen.

- Und Media Queries ermöglichen es Entwickler:innen, unterschiedliche Stile basierend auf den Eigenschaften des Geräts anzuwenden, hauptsächlich der Breite des Viewports. Mehr über Media Queries lernen Sie in zukünftigen Lektionen.

Die Beziehung zwischen responsivem Webdesign und Tools wie CSS Grid und Flexbox ist symbiotisch. Während responsives Design ein Konzept oder Ansatz ist, sind CSS Grid und Flexbox praktische Werkzeuge, die das Umsetzen responsiver Designs deutlich einfacher und effizienter machen.

In vorherigen Lektionen haben Sie gelernt, wie man mit Flexbox arbeitet, und in zukünftigen Lektionen werden Sie lernen, wie man mit CSS Grid arbeitet. Aber zunächst hier eine kurze Einführung in CSS Grid.

CSS Grid ist ein zweidimensionales Layoutsystem, das komplexere Anordnungen ermöglicht. Es eignet sich hervorragend für die Erstellung von Gesamtseitenlayouts sowie kleineren Komponentenlayouts.

Hier ist ein Beispiel, wie CSS Grid responsiv eingesetzt werden kann:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr;
  gap: 20px;
}

@media (min-width: 768px) {
  .grid-container {
    grid-template-columns: 1fr 1fr;
  }
}

@media (min-width: 1024px) {
  .grid-container {
    grid-template-columns: 1fr 1fr 1fr;
  }
}
```

:::

In diesem Beispiel beginnt das Grid auf kleinen Bildschirmen mit einer einzigen Spalte, wechselt auf mittelgroßen Bildschirmen zu zwei Spalten und auf größeren Bildschirmen zu drei Spalten. Das zeigt, wie Grid responsive Layouts erstellen kann, die sich an verschiedene Bildschirmgrößen anpassen.

Sowohl Flexbox als auch Grid bieten gegenüber älteren Layoutmethoden wie Floats oder tabellenbasierten Layouts erhebliche Vorteile. Sie bieten mehr Flexibilität und Kontrolle, benötigen weniger Code und sind mit Blick auf Responsivität entwickelt worden. Sie ermöglichen Entwickler:innen, komplexe, responsive Layouts mit relativ geringem Aufwand zu erstellen, was sie zu unverzichtbaren Werkzeugen bei der Umsetzung responsiven Webdesigns macht.

Es ist erwähnenswert, dass Flexbox und Grid oft zusammen in responsiven Designs verwendet werden. Flexbox wird typischerweise für Komponenten und eindimensionale Layouts eingesetzt, während Grid für die Gesamtseitenstruktur und zweidimensionale Layouts verwendet wird. Die Wahl zwischen beiden hängt oft von den spezifischen Layoutanforderungen des Designs ab.

Neben Flexbox und Grid spielen weitere CSS-Funktionen eine wichtige Rolle im responsiven Design.

Die `calc()`-Funktion ermöglicht beispielsweise das Mischen von Einheiten und das Durchführen von Berechnungen, was bei der Erstellung flexibler Layouts sehr nützlich sein kann.

Responsives Bildmaterial ist ein weiterer entscheidender Aspekt des responsiven Webdesigns. Das `srcset`-Attribut und das `picture`-Element in HTML5 erlauben es, unterschiedliche Bilddateien basierend auf den Fähigkeiten des Geräts auszuliefern, sodass Nutzer:innen auf Geräten mit kleineren Bildschirmen oder geringerer Auflösung nicht unnötig große Bilddateien herunterladen müssen.

Zusammenfassend ist responsives Webdesign ein Ansatz, der darauf abzielt, Websites zu erstellen, die auf jedem Gerät gut funktionieren.

Obwohl es sich eher um eine Designphilosophie als um eine spezifische Technologie handelt, beruht es stark auf CSS-Funktionen wie Media Queries und wird durch moderne Layout-Tools wie Flexbox und Grid erheblich erleichtert. Diese Werkzeuge bieten die Flexibilität und Kontrolle, die nötig sind, um wirklich responsive Designs zu erstellen, sodass Websites sich nahtlos an die ständig wachsende Vielfalt der Geräte anpassen, mit denen auf das Web zugegriffen wird.
