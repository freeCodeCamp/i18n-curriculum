# --explanation--

`Server` refere-se a um computador ou sistema poderoso que fornece serviços, dados ou recursos para outros dispositivos, frequentemente através de uma rede.Por exemplo:  

`The server hosts the company's website.` - Isso significa que o servidor é responsável por armazenar e entregar o conteúdo do site para os usuários que o acessam.

`Load` refere-se à quantidade de trabalho, demanda ou estresse colocado em um sistema ou recurso.Por exemplo:  

`The system crashed because it couldn't handle the high load during peak hours.` - Isso significa que o sistema ficou sobrecarregado devido à grande quantidade de demanda colocada sobre ele.
