# --description--

比較條件述語的另一種方法是三元運算子。三元運算子是撰寫 `if-else` 述語的簡寫方式。它用於評估條件並根據條件的結果傳回一個值。三元運算子的語法如下：

```javascript
let result = condition ? value1 : value2;
```

在上述範例中，如果條件為真，`result` 的值將會是 `value1`。否則，`result` 的值將會是 `value2`。

三元運算子經常用於根據條件指定變數的值。它們也用於根據條件傳回值。
