# --description--

给定一个格式错误的罗马数字字串，返回其标准罗马数字表示的数值。

输入只会使用加法形式，因此每个符号的值都会加到总数上。作为提醒，以下是符号和对应的数值：

| Symbol | Value |
| ------ | ----- |
| `"I"`  | 1     |
| `"V"`  | 5     |
| `"X"`  | 10    |
| `"L"`  | 50    |
| `"C"`  | 100   |
| `"D"`  | 500   |
| `"M"`  | 1000  |

重新编码时，每一步都使用尽可能大的符号，必要时使用减法点对（`"IV"`, `"IX"`, `"XL"`, `"XC"`, `"CD"`, `"CM"`）。

```py
def fix_numerals(s):
    values = {"I": 1, "V": 5, "X": 10, "L": 50, "C": 100, "D": 500, "M": 1000}
    encoding = [
        (1000, "M"), (900, "CM"), (500, "D"), (400, "CD"),
        (100, "C"), (90, "XC"), (50, "L"), (40, "XL"),
        (10, "X"), (9, "IX"), (5, "V"), (4, "IV"), (1, "I")
    ]

    total = sum(values[c] for c in s)

    result = ""
    for value, symbol in encoding:
        while total >= value:
            result += symbol
            total -= value
    return result
```
