# --description--

`div` 元素用作容器來群組其他 HTML 元素。當你想要群組將共用一組 CSS 樣式的 HTML 元素時，你主要會使用 `div` 元素。

在 `p` 元素下方，添加一個 `div` 元素。這個 `div` 將會是你的書籍卡片的容器。

# --hints--

你應該有一個 `div` 元素。

```js
assert.exists(document.querySelector('div'));
```

你的 `div` 元素應該位於你的 `p` 元素下方。

```js
assert.exists(document.querySelector('p + div'));
```
