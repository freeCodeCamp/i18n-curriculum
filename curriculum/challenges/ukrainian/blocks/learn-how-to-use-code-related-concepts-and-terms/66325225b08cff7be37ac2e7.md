# --description--

Відносні займенники, такі як `who`, `whom`, `which` і `that`, використовуються для зв’язку відносних умов з іменником або займенником.

`Who` і `whom` відносяться до людей. Різниця в тому, що `who` є підметом дієслова, тоді як `whom` використовується як додаток дієслова або прийменника.

`Which` використовується для конкретних речей або тварин, а `that` може використовуватися для людей, речей або тварин у визначальних умовах.

Приклади:

`The programmer who wrote this code is brilliant.` (`who` відноситься до програміста)

`The artist whom we met at the gallery is internationally recognized.` (`whom` відноситься до художника)

`The tool that you lent me was useful.` (`that` відноситься до інструмента)

`The book, which I bought yesterday, is already a bestseller.` (`which` відноситься до книги)

# --questions--

## --text--

Яке речення правильно використовує відносний займенник?

## --answers--

`The teacher which teaches us math is very kind.`

### --feedback--

`which` зазвичай використовується для речей, а не для людей. Тут правильним займенником має бути `who`.

---

`The car, that won the race, is very fast.`

### --feedback--

У цьому реченні використання `that` граматично правильне, але коми зайві. Має бути: `The car that won the race is very fast.`

---

`She is the one who made the cake.`

---

`The flowers whom are on the table smell lovely.`

### --feedback--

`Whom` використовується для додатків дієслів або прийменників, коли йдеться про людей, а не про речі, як-от квіти. Тут правильним займенником має бути `which` або `that`.
