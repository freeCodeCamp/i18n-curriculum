# --description--

到目前為止，你已經練習了如何用 `do` / `does` 提問以及用 `don't` / `doesn't` 造否定句。讓我們來複習一下。記住：

- 肯定句：`He does work on weekends.`（表示此人週末有工作）

- 否定：`He doesn't work on weekends.`（表示此人週末不工作）

- 疑問句：`Does he work on weekends?`（詢問此人是否在週末工作）

# --instructions--

閱讀對話窗摘錄並使用 `do` 或 `don't` 填空。
