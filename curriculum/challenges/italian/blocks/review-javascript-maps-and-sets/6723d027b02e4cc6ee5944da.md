## Set in JavaScript

- Un `Set` è un'opzione integrata per gestire raccolte di dati.
- I set garantiscono che ogni valore al loro interno appaia una sola volta, risultando utili per eliminare duplicati da un array o per gestire raccolte di valori distinti.
- Puoi creare un `Set` usando il costruttore `Set()`:

:::interactive_editor

```js
const set = new Set([1, 2, 3, 4, 5]);
console.log(set); // Set { 1, 2, 3, 4, 5 }
```

:::

- I set possono essere manipolati usando questi metodi:

  - `add()`: aggiunge un nuovo elemento al `Set`.
  - `delete()`: rimuove un elemento dal `Set`.
  - `has()`: verifica se un elemento esiste nel `Set`.
  - `clear()`: rimuove tutti gli elementi dal `Set`.
  - `keys()` e `values()`: entrambi restituiscono un `SetIterator` che contiene i valori del `Set`. Sono uguali perché `keys()` è un alias di `values()`.
  - `forEach()`: per iterare sui valori del `Set`.

## Weakset in JavaScript

- `WeakSet` è una raccolta di oggetti che permette di archiviare oggetti con riferimenti deboli.

## Set vs WeakSet

- A differenza dei set, un `WeakSet` non supporta tipi primitivi come numeri o stringhe.
- Un `WeakSet` archivia solo oggetti, e i riferimenti a quegli oggetti sono "deboli", cioè se l'oggetto non viene usato altrove nel codice, viene rimosso automaticamente per liberare memoria.

## Map in JavaScript

- Un `Map` è un oggetto integrato che contiene coppie chiave-valore proprio come un oggetto.
- Le map differiscono dagli oggetti JavaScript standard per la capacità di permettere chiavi di qualsiasi tipo, inclusi oggetti e funzioni.
- Un `Map` offre prestazioni migliori rispetto all'oggetto standard quando si tratta di aggiunte e rimozioni frequenti di coppie chiave-valore.
- Puoi creare un `Map` usando il costruttore `Map()`:

:::interactive_editor

```js
const map = new Map([
  ['flower', 'rose'],
  ['fruit', 'apple'],
  ['vegetable', 'carrot']
]);
console.log(map); // Map(3) { 'flower' => 'rose', 'fruit' => 'apple', 'vegetable' => 'carrot' }
```

:::

- Le map possono essere manipolate usando questi metodi:

  - `set()`: aggiunge una nuova coppia chiave-valore al `Map`.
  - `get()`: recupera il valore di una chiave dal `Map`.
  - `delete()`: rimuove una coppia chiave-valore dal `Map`.
  - `has()`: verifica se una chiave esiste nel `Map`.
  - `clear()`: rimuove tutte le coppie chiave-valore dal `Map`.

Nota che sia le map che i set hanno la proprietà `size` che restituisce il numero di elementi unici al loro interno.

## WeakMap in JavaScript

- Un `WeakMap` è una raccolta di coppie chiave-valore proprio come `Map`, ma con riferimenti deboli alle chiavi. Le chiavi devono essere oggetti e i valori possono essere qualsiasi cosa.

## Map vs WeakMap

- Le weakmap sono simili alle weakset in quanto archiviano solo oggetti e i riferimenti a quegli oggetti sono "deboli".
