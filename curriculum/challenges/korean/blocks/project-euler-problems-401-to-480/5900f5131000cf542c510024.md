# --description--

형태가 $n^{15} + 1$인 수는 모든 정수 $n > 1$에 대해 합성수입니다.

양의 정수 $n$과 $m$에 대해 $s(n, m)$을 $n^{15} + 1$의 서로 다른 소인수 중 $m$ 이하인 것들의 합으로 정의합니다.

예를 들어, $2^{15} + 1 = 3 × 3 × 11 × 331$입니다.

따라서 $s(2, 10) = 3$이고 $s(2, 1000) = 3 + 11 + 331 = 345$입니다.

또한 ${10}^{15} + 1 = 7 × 11 × 13 × 211 × 241 × 2161 × 9091$입니다.

그래서 $s(10, 100) = 31$이고 $s(10, 1000) = 483$입니다.

$1 ≤ n ≤ {10}^{11}$에 대해 $\sum s(n, {10}^8)$을 구하세요.
