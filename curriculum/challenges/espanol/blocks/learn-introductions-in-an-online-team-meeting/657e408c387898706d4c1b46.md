# --description--

En reuniones, frases como `We have a lot to talk about today` se usan con frecuencia. `We have` significa que algo pertenece o está relacionado con el grupo. `A lot` significa mucho o muchos, y `to talk about` significa que hay cosas para discutir. 

Comprender estas frases te ayuda a seguir lo que está planeado para una reunión o una discusión.
