# --interactive--

JavaScript 中的日期和时间一直以来都不容易处理。在专业的代码库中，你可能会使用一个已经为你解决了许多这些问题的库。

但有时你需要使用 JavaScript 的内置 `Date` 对象。

在本课中，我们将介绍 `Date` 对象的基础知识以及在 JavaScript 中处理日期和时间时你最常用的一些方法。

`Date` 对象是 JavaScript 中的一个内置对象，允许你处理日期和时间。以下是创建新建 `Date` 对象的基本语法：

```js
const now = new Date();
```

`new` 关键字用于创建 `Date` 对象的新实例，然后将该 `Date` 对象赋值给变量 `now`。如果你将 `now` 的值记录到控制台，你将看到基于运行代码的计算机系统时钟的当前日期和时间。

以下是如果你将 `now` 的值记录到控制台时会看到的示例响应：

:::interactive_editor

```js
const now = new Date();
console.log(now);
```

:::

时间使用的是 24 小时制格式，所以 `14:30` 是 `2:30 PM`。`GMT-0700` 是时区偏移量，`Pacific Daylight Time` 是时区名称。

你也可以通过提供年、月、日、小时、分钟、秒和毫秒值作为参数，将特定的日期和时间传递给 `Date` 对象。

下面是使用特定日期和时间创建新建 `Date` 对象的示例：

:::interactive_editor

```js
const specificDate = new Date("July 4, 1776 12:00:00");

console.log(specificDate);
```

:::

当你需要处理特定的日期和时间而不是当前的日期和时间时，这非常有用。输入始终需要是一个字串，且格式应能被 `Date` 对象识别。

要获取当前日期和时间，你可以使用 `Date.now()` 方法，它返回自 `January 1, 1970, 00:00:00 UTC` 以来的毫秒数。这被称为 Unix 纪元时间。

Unix 纪元时间是在计算机系统中表现日期和时间的常用方式，因为它是一个可以轻松保存和操作的整数。UTC 代表通用的协调时间，它是世界调节时钟和时间的主要时间标准。

如果你需要根据当前日期获取一个月中的某一天，可以使用 `getDate` 方法。以下是使用 `getDate` 方法的示例：

:::interactive_editor

```js
const now = new Date();
const date = now.getDate();
console.log(date); 
```

:::

在此示例中，我们使用 `Date` 对象创建一个新的日期实例并赋值给变量 `now`。然后，我们在 `now` 对象上调用 `getDate` 方法以获取月份中的日期，并赋值给变量 `date`。最后，我们将 `date` 的值记录到控制台，输出当前的月份日期。

`getDate` 将返回一个介于 `1` 和 `31` 之间的整数值，具体取决于月份中的日期。如果日期无效，它将返回 `NaN`（非数字）。

要获取月份，你可以像这样使用 `getMonth` 方法：

:::interactive_editor

```js
const now = new Date();
const month = now.getMonth();
console.log(month);
```

:::

月份是从零开始的，所以 `January` 是 `0`，`February` 是 `1`，依此类推。如果月份无效，它将返回 `NaN`。

如果你需要获取完整的年份，那么你可以像这样使用 `getFullYear` 方法：

:::interactive_editor

```js
const now = new Date();
const year = now.getFullYear();
console.log(year); 
```

:::

如果年份无效，它将返回 `NaN`。

`Date` 对象上还有许多其他方法，包括 `getHours`、`getMinutes`、`getSeconds` 等。我鼓励你通过 Mozilla 的文档或其他资源自行探索更多内容。
