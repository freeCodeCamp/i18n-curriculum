# --description--

使用者隱私就是你控制個人資料如何被收集、使用和在線上分享的權利。現代隱私框架依賴幾個核心區塊來保護你的數位足跡。

隱私保護的核心是**可識別個人身分的資訊（PII）**。PII 是任何能直接或間接辨識你身分的資料。

直接識別符包括你的全名、住家位址和國民身分證號碼，而間接識別符則包括你的 IP 位址、裝置序號或精確的定位歷史。

為了保護這些資料安全，公司必須維護**機密性**。機密性確保你的私人資料透過強加密和存取控制，免於未經授權的存取。

如果未經授權的第三方讀取你的記錄或洩漏你的 PII，你的機密性就被破壞了。

隱私框架透過**資料最小化**和**目的限制**來執行這項保護。資料最小化規定公司應只收集完成特定任務所需的最少 PII。

目的限制規定企業一旦為特定理由收集你的資料，就不能合法地用於其他用途，例如將你的聯絡資訊賣給廣告網路。

這些原則直接對抗侵入性的數位追蹤。**追蹤**是持續監控你在網際網路上的線上行為。

廣告網路使用腳本、裝置指紋和瀏覽器歷史記錄來追蹤你的行為，並在你不知情的情況下建立詳細的個人檔案。

為了防範這種追蹤，使用者同意是個人控制的基石。真正的隱私需要明確且知情的同意，也就是你必須主動選擇加入資料追蹤，而非被強迫接受。

最後，**隱私法保障被遺忘權**，也稱為資料刪除權。這個概念賦予你法律權利，要求組織在你關閉帳號後，永久刪除所有個人記錄和追蹤歷史。

你將在下一堂課中學習這些隱私法及其負責機構。
