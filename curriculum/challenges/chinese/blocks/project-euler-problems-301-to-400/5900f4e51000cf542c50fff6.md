# --description--

一个整数 $n$ 的整数分区是将 $n$ 写成正整数之和的一种方式。

仅在其加数顺序上不同的分区被视为相同。将 $n$ 分成不同部分的分区是指每个部分最多出现一次的 $n$ 的分区。

5 的不同部分的分区是：

5、4 + 1 和 3 + 2。

设 $f(n)$ 为将 $n$ 分成不同部分的任意此类分区的部分的最大乘积，设 $m(n)$ 为具有该乘积的任意此类 $n$ 分区的元素数量。

所以 $f(5) = 6$ 且 $m(5) = 2$。

对于 $n = 10$，乘积最大的分区是 $10 = 2 + 3 + 5$，其结果为 $f(10) = 30$ 和 $m(10) = 3$。它们的乘积为 $f(10) \times m(10) = 30 \times 3 = 90$。

可以验证，对于 $1 ≤ n ≤ 100$，$\sum f(n) \times m(n) = 1\\,683\\,550\\,844\\,462$。

查找 $\sum f(n) \times m(n)$，其中 $1 ≤ n ≤ {10}^{14}$。将你的答案对 $982\,451\,653$ 取模，该数是第 5\,000\,0000 个素数。
