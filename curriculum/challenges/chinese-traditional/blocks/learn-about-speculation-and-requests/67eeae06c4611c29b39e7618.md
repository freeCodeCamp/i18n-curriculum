# --explanation--

`Should have` + `Past Participle` 用來表達對過去的遺憾或建議。例如：

- `You should have run the program on a different machine to check for compatibility。`－在不同的機器上測試會很有幫助，但並未執行。

- `I should have run the tests before deploying the app。`－執行測試是正確的做法，但當時並未執行。
