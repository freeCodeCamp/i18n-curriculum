# --description--

Kama ulivyojifunza katika moduli za awali, mizunguko hutumika kurudia kifungu cha msimbo kwa idadi fulani ya mara. Katika funzo hili utajifunza jinsi ya kufanya kazi na aina tofauti za mizunguko katika Python.

Mzunguko wa kwanza tutakauelezea ni mzunguko wa `for`. Hapa kuna mfano wa kutumia mzunguko wa `for` kurudia kupitia orodha na kuchapisha kila kipengee kwenye konsoli:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)
```

Matokeo yatakuwa:

```md
Rust
Java
Python
C++
```

Angalia kwamba `print(language)` imepangwa ndani ya mzunguko. Bila upangaji huo, utapata `IndentationError`:

```python
"""
Traceback (most recent call last):
  File "<stdin>", line 4, in <module>
IndentationError: expected an indented block after 'for' statement on line 3
"""
```

Unaweza pia kutumia mzunguko wa `for` kurudia kupitia vitu vingine vinavyoweza kurudiwa kama mfuatano wa herufi. Hapa kuna mfano wa kutumia mzunguko wa `for` kurudia kupitia mfuatano wa herufi `code` na kuchapisha kila herufi:

```py
for char in 'code':
    print(char)
```

Matokeo yatakuwa:

```md
c
o
d
e
```

Kama ilivyo katika JavaScript, unaweza pia kupanga ndani mizunguko ya `for` katika Python. Hapa kuna mfano wa kutumia mzunguko wa `for` uliopangwa ndani:

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)
```

Mzunguko wa nje utarudia kupitia kila `category` katika orodha ya `categories`. Kwa kila `category`, mzunguko wa ndani utarudia kupitia kila `food` katika orodha ya `foods`. Hapa ni matokeo yatakayochapishwa kwenye konsoli:

```md
Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
```

Aina nyingine ya mzunguko unaoweza kutumia katika Python ni mzunguko wa `while`. Aina hii ya mzunguko itarudia kifungu cha msimbo hadi hali iwe `False`. Hapa kuna mfano wa kutumia mzunguko wa `while` kwa mchezo wa kubahatisha:

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')
```

Katika mfano huu tuna kigezo cha `secret_number` chenye thamani ya `3` na bahati ya awali ya `0`. Kisha tunatumia kitendakazi cha `input` kupata ingizo kutoka kwa mtumizi, kisha kubadilisha mfuatano wa herufi wa ingizo kuwa nambari kamili kwa kitendakazi cha `int()`, na kuweka thamani hiyo kwenye kigezo cha `guess`. Ikiwa mtumizi atabahatisha kwa usahihi kwa kuingiza `3`, mzunguko wa `while` unavunjwa na ujumbe wa `You got it!` unachapishwa kwenye konsoli. Vinginevyo, ujumbe wa `Wrong! Try again.` unachapishwa kwenye konsoli, na mzunguko unarudia, ukimwomba mtumizi abahatishie tena.

Hapa kuna mfano wa matokeo:

```md
Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
```

Kama ilivyo katika JavaScript, Python inaunga mkono kauli za `break` na `continue`.

Kauli ya `break` hutumika kusitisha utekelezaji wa mzunguko. Hapa kuna mfano wa kutumia kauli ya `break` kwa orodha ya `developer_names`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

Katika mfano huu, tunarudia kupitia orodha ya `developer_names` na kuchapisha kila jina kwenye konsoli. Ikiwa jina ni sawa na `Naomi`, basi tunavunja mzunguko. Hii husababisha jina la `Jess` tu kuchapishwa kwenye konsoli.

Kauli ya `continue` hutumika kuruka kurudia kwa sasa wa mzunguko na kuendelea na kurudia inayofuata. Hebu tubadilishe mfano wa awali kutumia kauli ya `continue` badala ya `break`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

Sasa matokeo kwenye konsoli yatakuwa tofauti. Majina ya `Jess` na `Tom` yamechapishwa kwa sababu kauli ya `continue` inaruka kurudia ya pili ya mzunguko wakati `developer` ni sawa na `Naomi`, na haichapishi jina hilo kwenye konsoli.

Mizunguko ya `for` na `while` inaweza kuunganishwa na kifungu cha `else`, ambacho hufanyika tu wakati mzunguko haujamalizika kwa kauli ya `break`. Hapa kuna mfano wa kutumia mizunguko mingi ya `for`:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

Katika mfano huu tuna orodha ya maneno ya bahati nasibu, na mzunguko wa `for` unatumika kurudia kupitia kila neno. Ndani ya mzunguko wa nje wa `for`, tuna mzunguko mwingine wa `for` kurudia kupitia kila herufi ya kila neno. Ikiwa toleo la herufi ndogo ni vokali, tunachapisha neno likifuatiwa na vokali zake, kisha tunavunja mzunguko wa ndani. Ikiwa neno halina vokali, basi tunachapisha ujumbe unaoonyesha hilo.

Hapa ni jinsi matokeo yanavyoonekana kwenye konsoli:

```md
'sky' has no vowels
'apple' contains the vowel 'a'
'rhythm' has no vowels
'fly' has no vowels
'orange' contains the vowel 'o'
```

Mizunguko ni ya kawaida sana katika Python, kwa hivyo ni muhimu kujizoeza nayo. Katika mafundisho yajayo, utajifunza jinsi ya kufanya kazi na kitendakazi cha `enumerate()` na `range()` katika mizunguko.
