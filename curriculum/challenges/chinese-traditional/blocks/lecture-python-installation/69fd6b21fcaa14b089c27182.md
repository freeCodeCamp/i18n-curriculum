# --description--

在上一課中，你學會了如何在你的區域裝置上安裝 Python。在本課中，你將學習如何在本地執行 Python 程式碼。

在你開始在本地建置 Python 腳本之前，你需要先安裝一個程式碼編輯器或整合開發環境（IDE）。IDE 是整合開發環境的縮寫，它包含了程式碼編輯器、測試工具、終端機等多種功能。

一個許多開發者常用的程式碼編輯器是 VS Code。前往 `https://code.visualstudio.com/download` 下載支援 Mac、Windows 和 Linux 環境的 VS Code。這就是本課將使用的編輯器。

其他常見的 Python 開發選擇包括：

- PyCharm
- Spyder

下載好你的編輯器後，打開它。打開後，你需要開啟一個資料夾。你可以點擊左上角的 `File` 選單選項，然後點擊 `Open Folder`。接著，你可以選擇或建立一個專案資料夾。為了課外練習，你可能想要一個名為 `python-projects` 的資料夾。

資料夾設定好後，你可以打開 VS Code 左側邊欄的檔案總管。點擊看起來像一張紙上有加號的 `New File` 圖示。然後建立一個名為 `main.py` 的檔案，按下 Enter。`.py` 是副檔名，表示這是一個 Python 檔案。

打開你的 `main.py` 檔案，輸入以下程式碼：

```py
print("Hello, world!")
```

要執行這段程式碼，你有幾種選擇。第一種是點擊位於右上角看起來像播放按鈕的執行按鈕。這會開啟終端機並執行你的 Python 腳本。你應該會看到文字 `"Hello, world!"`。

另一種選擇是打開終端機，手動使用以下指令執行程式：

```bash
python main.py
```

你必須在 `main.py` 所在的資料夾執行此指令。例如，如果 `main.py` 位於名為 `python-projects` 的資料夾中，先在終端機輸入：

```bash
cd python-projects
```

接著執行：

```bash
python main.py
```

在某些環境中，特別是在 macOS 和 Linux 系統上同時安裝了 Python 2 和 Python 3 時，你可能需要使用 `python3` 來取代 `python`。這種情況常見於 `python` 指令不存在或指向較舊版本的 Python。

```bash
python3 main.py
```
