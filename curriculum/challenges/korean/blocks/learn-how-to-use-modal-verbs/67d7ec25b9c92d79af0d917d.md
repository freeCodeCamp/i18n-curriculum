# --explanation--

`Must have happened`는 과거에 대한 논리적인 추측을 할 때 사용합니다.  

`Must`는 강한 확신을 나타냅니다. `Have` + `Past Participle` (예: `happened`)는 과거를 가리킵니다. 예를 들어:

- `He must have forgotten to update the system.` - 그가 업데이트하는 것을 기억하지 못했을 가능성이 큽니다.

- `They must have made some changes to the app.` - 앱이 업데이트되거나 개선되었을 가능성이 큽니다.

Jake는 증거(`unusual activity on the servers`)를 사용해 침해가 어젯밤에 `must have happened`라고 결론 내립니다. 이 구조는 논리적인 추정을 할 때 유용합니다.
