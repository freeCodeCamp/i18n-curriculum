# --description--

Si se llama a una función, pero no se proporciona un argumento, entonces el parámetro recibirá el valor de `undefined`. Esto puede llevar a resultados inesperados si la función no está diseñada para manejar este caso.

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal()); // Output: undefined is my favorite animal!
```

En el ejemplo anterior, la función `favoriteAnimal` se llama sin un argumento. El parámetro `animal` recibe el valor de `undefined`, que luego se concatena con la cadena `" is my favorite animal!"`.

Si no se proporciona un valor, puedes usar parámetros por defecto. Los parámetros por defecto te permiten asignar un valor predeterminado a un parámetro si no se proporciona ningún valor cuando se llama a la función.

```js
function favoriteAnimal(animal = 'Giant Panda') {
  return animal + " is my favorite animal!"
}
```

En el ejemplo anterior, el valor predeterminado del parámetro `animal` está establecido en `'Giant Panda'`. Si se llama a la función `favoriteAnimal` sin un argumento, al parámetro `animal` se le asignará el valor `'Giant Panda'`.
