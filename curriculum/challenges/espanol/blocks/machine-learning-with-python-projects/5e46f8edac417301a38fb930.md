# --instructions--

En este desafío, vas a predecir costos de atención médica usando un algoritmo de regresión.

Se te proporciona un conjunto de datos que contiene información sobre diferentes personas, incluidos sus costos de atención médica. Usa los datos para predecir los costos de atención médica basándote en datos nuevos.

Las dos primeras celdas de este cuaderno importan librerías y los datos.

Asegúrate de convertir los datos categóricos a números. Usa el 80% de los datos como el `train_dataset` y el 20% de los datos como el `test_dataset`.

Haz un `pop` de la columna "expenses" de estos conjuntos de datos para crear nuevos conjuntos llamados `train_labels` y `test_labels`. Usa estas etiquetas al entrenar tu modelo.

Crea un modelo y entrénalo con el `train_dataset`. Ejecuta la celda final de este cuaderno para verificar tu modelo. La celda final usará el `test_dataset` no visto para comprobar qué tan bien generaliza el modelo.

Para pasar el desafío, `model.evaluate` debe devolver un Error Absoluto Medio menor a 3500. Esto significa que predice correctamente los costos de atención médica dentro de $3500.

La celda final también predecirá gastos usando el `test_dataset` y graficará los resultados.
