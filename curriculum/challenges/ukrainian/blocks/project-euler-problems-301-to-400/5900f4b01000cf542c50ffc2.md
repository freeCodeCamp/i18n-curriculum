# --description--

Нехай $y_0, y_1, y_2, \ldots$ — послідовність випадкових беззнакових 32-бітних цілих чисел (тобто $0 ≤ y_i &lt; 2^{32}$, кожне значення однаково ймовірне).

Для послідовності $x_i$ задана така рекурсія:

- $x_0 = 0$ та
- $x_i = x_{i - 1} \mathbf{|} y_{i - 1}$, для $i > 0$. ($\mathbf{|}$ — оператор побітового OR)

Можна побачити, що зрештою існуватиме індекс $N$, такий що $x_i = 2^{32} - 1$ (шаблон бітів усіх одиниць) для всіх $i ≥ N$.

Знайдіть очікуване значення $N$. Наведіть відповідь, округлену до 10 знаків після десяткової крапки.
