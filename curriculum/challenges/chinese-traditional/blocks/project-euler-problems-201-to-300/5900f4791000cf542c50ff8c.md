# --description--

多項式 $P(x)$ 的根或零點是方程式 $P(x) = 0$ 的解。

將 $P_n$ 定義為係數為 $n$ 各位數字的多項式。

例如，$P_{5703}(x) = 5x^3 + 7x^2 + 3$。

我們可以看到：

- $P_n(0)$ 是 $n$ 的最後一位數字，
- $P_n(1)$ 是 $n$ 的各位數字之和，
- $Pn(10)$ 就是 $n$ 本身。

定義 $Z(k)$ 為不超過 $k$ 的正整數 $n$ 的數量，且對這些 $n$，多項式 $P_n$ 至少有一個整數根。

可以驗證 $Z(100\,000)$ 是 14696。

什麼是 $Z({10}^{16})$？
