# --description--

Express.js أو Express للاختصار، هو إطار عمل ويب بسيط ومرن مبني **على** Node.js. يُبسّط عملية بناء خوادم الويب وواجهات برمجة التطبيقات (APIs).

بدلاً من كتابة الكثير من شفرة القالب الجاهز باستخدام وحدة `http` الأصلية في Node، يتيح لك Express إنشاء تطبيقات ويب قوية ببضع أسطر فقط.

يمكنك اعتباره الطبقة السريعة وغير المتحيزة التي تجعل Node.js أسهل وأكثر قوة.

إليك ثلاثة أسباب رئيسية تجعل المطوّرين يحبون Express:

1. **بسيط وسريع** – يوفر Express الأساسيات فقط، مما يمنحك تحكمًا كاملاً مع السماح لك بإضافة الميزات حسب الحاجة عبر البرمجيات الوسيطة والإضافات.

2. **نظام بيئي ضخم** – يحتوي على مجموعة كبيرة من البرمجيات الوسيطة والإضافات للتسجيل، والتوثيق، والأمان، والجلسات، والمزيد.

3. **مناسب جدًا لواجهات برمجة التطبيقات** – نموذج التوجيه والتعامل مع الطلبات البسيط يجعله مناسبًا بشكل خاص لبناء واجهات برمجة التطبيقات RESTful.

مع Express، يمكنك إنشاء:

* تطبيقات ويب

* واجهات برمجة التطبيقات RESTful

* خلفيات تطبيقات الصفحة الواحدة (SPA)

* خدمات تعتمد بشكل كبير على البرمجيات الوسيطة

* أي شيء على الخادم باستخدام HTTP


## كيفية تثبيت Express

أولًا، تأكد من تثبيت Node.js و npm. إليك كيفية التحقق مما إذا كان node و npm مثبتين:

```bash
node -v
npm -v
```

إذا رأيت رقم إصدار، فهذا يعني أنهما مثبتان. إذا لم يكن كذلك، ستحتاج إلى تثبيتهما. يمكنك العثور على تعليمات التثبيت في درس سابق، أو على الموقع الرسمي لـ Node.js.

بعد ذلك، في الطرفية، اكتب الأوامر التالية:

```bash
mkdir my-app
cd my-app
npm init -y
npm install express
```

ستتعلم المزيد عن البرمجيات الوسيطة والإضافات في دروس لاحقة، لكن باختصار، البرمجيات الوسيطة هي شفرة تعمل بين استقبال الطلب وإرسال الاستجابة. تُستخدم عادةً لمعالجة الأخطاء، والتحقق من المُدخلات، وتعديل البيانات في الطلب أو الاستجابة، والمزيد. يمكنك ربط البرمجيات الوسيطة معًا لبناء منطق مرن وقابل لإعادة الاستخدام لمعالجة الطلبات.

الإضافات عادةً ما تكون وحدات Node.js خارجية من طرف ثالث توسع وظائف Express. يمكنك تثبيتها بشكل منفصل باستخدام npm، ومعظمها يُنفذ كبرمجيات وسيطة يمكن استخدامها في تطبيقك مع إعداد بسيط. بدلاً من كتابة الميزات الشائعة من الصفر، يمكنك استخدام إضافة للتعامل مع أشياء مثل التسجيل، والتوثيق، أو أفضل الممارسات الأمنية.

Express هو إطار عمل شائع لتطوير الخادم في Node.js. يمنحك اللبنات الأساسية لإنشاء خادم بسرعة، وإدارة المسارات، والتحكم في البرمجيات الوسيطة، وبناء واجهات برمجة التطبيقات — كل ذلك بشفرة نظيفة وسهلة القراءة.
