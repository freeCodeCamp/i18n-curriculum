# --interactive--

У CSS властивості `width` і `height` використовуються для керування розмірами елементів на вебсторінці.

Ці властивості можна задавати в різних одиницях, таких як пікселі (`px`), відсотки (`%`), одиниці вікна перегляду (`vw`, `vh`) та інші.

Властивість `width` визначає ширину елемента. Якщо ширина не вказана, то за замовчуванням вона встановлюється в `auto`. Це дозволяє браузеру визначати ширину елемента залежно від його вмісту, батьківського елемента та типу відображення. Для елемента `div` властивість `width: auto` змушує його розтягуватися на всю ширину батьківського контейнера.

Властивість `height` визначає висоту елемента. Аналогічно, висота за замовчуванням є `auto`, що означає, що вона підлаштовується під вміст всередині.

Ось приклад із використанням властивостей `width` і `height`:

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 100px;
      height: 100px;
      background-color: skyblue
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

У цьому прикладі ми маємо елемент `div` з класом `box`. Цей елемент займатиме `100px` по висоті і ширині, а колір фону буде `skyblue`.

Пікселі — це одиниця виміру фіксованого розміру в CSS, що дає точний контроль над розмірами.

Властивість `min-width` задає мінімальну ширину, яку може мати елемент. Навіть якщо вміст всередині менший, елемент не звузиться менше за це значення.

Властивість `min-height` задає мінімальну висоту елемента. Вона гарантує, що елемент не стане нижчим за встановлене значення.

Ось приклад:

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 50px;
      min-width: 100px;
      height: 50px;
      min-height: 100px;
      background-color: lightcoral;
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

Вищенаведений приклад демонструє, як працюють `min-width` і `min-height`.

Навіть якщо у блоку встановлені `width` і `height` по 50px, фактично він буде `100px` на `100px`. Це тому, що `min-width` і `min-height` встановлені в `100px`, які більші за задані `width` і `height`.

Пам’ятайте, якщо `min-width` або `min-height` більші за `width` або `height`, вони заміщають менші значення. Це гарантує, що елементи не стануть занадто маленькими, що важливо для підтримки послідовного та зручного дизайну.

Властивість `max-width` задає максимальну ширину, до якої елемент може розширюватися, навіть якщо для цього є достатньо місця.

Властивість `max-height` задає максимальну висоту, до якої елемент може розширюватися, незалежно від розміру вмісту.

Ось приклад:

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 200px;
      max-width: 150px;
      height: 200px;
      max-height: 150px;
      background-color: lightgreen;
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

Вищенаведений приклад демонструє, як `max-width` і `max-height` заміщають `width` і `height`. Навіть якщо у блоку встановлені `200px` на `200px`, фактично він буде `150px` на `150px`. Це тому, що `max-width` і `max-height` встановлені в `150px`, які менші.

Пам’ятайте, коли `max-width` або `max-height` менші за `width` або `height`, вони мають пріоритет. Це важливо для контролю максимальних розмірів елементів у ваших макетах.

CSS надає пріоритет `min-width` і `min-height` над `width` і `height`. `max-width` і `max-height` обмежують розміри, якщо значення перевищують їхні межі.

Наприклад, якщо ви встановите `width` у `600px` і `max-width` у `500px`, елемент буде обмежений шириною `500px`. Властивість `max-width` заміщує ширше значення, утримуючи елемент у межах заданого максимального розміру.

Це гарантує, що елементи залишаються в межах бажаних розмірів, незалежно від стандартних значень ширини та висоти.
