# --interactive--

輸入欄位上的標籤看似微不足道，但它們在讓表單更容易使用方面扮演著關鍵角色。

標籤有助於身心障礙者，使表單對所有人更友善，並最終提升表單轉換率。

讓我們深入探討為什麼為輸入元素添加標籤對於存取性和可用性很重要。

螢幕閱讀器經常依賴標籤來描述輸入欄位的用途。為了讓這個功能正常運作，標籤必須以程式方式與 `input` 相關聯。雖然有多種方法可以做到這一點，但最常見的是使用 HTML 的 `label` 元素。

以下是如何將 `label` 與 `input` 關聯的方法：

:::interactive_editor

```html
<form>
   <label for="name">Your Name</label>
   <input type="text" id="name" />
</form>
```

:::

在此範例中，`label` 元素的 `for` 屬性與 `input` 元素的 `id` 相關。此連線允許螢幕閱讀器在 `input` 處於聚焦狀態時宣告該 `label`，使螢幕閱讀器使用者能理解該 `input` 的用途。

將標籤與所有輸入欄位關聯也有助於 SEO。 有了適當的標籤，搜尋引擎將更好地理解頁面內容，進而對頁面搜尋等級產生正面影響。

但重要的是要了解你是在為人類製作內容，而不是為搜尋引擎。因此，你應該確保標籤盡可能清晰且具描述性。
