## Kufanya kazi na HTML, CSS, na JavaScript

Wakati HTML na CSS zinatoa muundo wa tovuti, JavaScript huleta mwingiliano kwenye tovuti kwa kuwezesha utendaji mgumu, kama kushughulikia ingizo la mtumizi, kuhuisha vipengele, na hata kujenga programu kamili za mtandao.

## Aina za data katika JavaScript

Aina za data husaidia programu kuelewa aina ya data inayofanyiwa kazi, iwe ni nambari, maandishi, au kitu kingine.

- **Nambari**: Nambari inawakilisha nambari kamili na nambari zenye sehemu za desimali. Mifano ya nambari kamili ni 7, 19, na 90. 
- **Nambari zenye sehemu za desimali**: Nambari yenye sehemu za desimali ni nambari yenye nukta ya desimali. Mifano ni 3.14, 0.5, na 0.0001.
- **Mfuatano wa herufi**: Mfuatano wa herufi ni mfululizo wa herufi, au maandishi, yaliyowekwa ndani ya nukuu. `"I like coding"` na `'JavaScript is fun'` ni mifano ya mfuatano wa herufi.
- **Thamani ya kweli au siyo kweli**: Thamani ya kweli au siyo kweli inawakilisha mojawapo ya thamani mbili zinazowezekana: `true` au `false`. Unaweza kutumia thamani ya kweli au siyo kweli kuwakilisha hali, kama `isLoggedIn = true`.
- **Undefined na Null**: Thamani ya `undefined` ni kigezo kilichotangazwa lakini hakijapewa thamani. Thamani ya `null` ni thamani tupu, au kigezo kilichopewa thamani ya `null` kwa makusudi.
- **Kitu**: Kitu ni mkusanyiko wa jozi za ufunguo-thamani. Ufunguo ni jina la sifa, na thamani ni thamani ya sifa.

Hapa, kitu cha `pet` kina sifa tatu au funguo: `name`, `age`, na `type`. Thamani ni `Fluffy`, `3`, na `dog`, kwa mtiririko huo.

```js
let pet = {
  name: "Fluffy",
  age: 3,
  type: "dog"
};
```

- **Alama (Symbol)**: Aina ya data ya Symbol ni thamani ya kipekee isiyobadilika ambayo inaweza kutumika kama kitambulisho cha sifa za kitu.

Katika mfano huu hapa chini, alama mbili zimetengenezwa zikiwa na maelezo sawa, lakini hazilingani.

:::interactive_editor

```js
const crypticKey1= Symbol("saltNpepper");
const crypticKey2= Symbol("saltNpepper");
console.log(crypticKey1 === crypticKey2); // false
```

:::

- **BigInt**: Wakati nambari ni kubwa mno kwa aina ya data ya `Number`, unaweza kutumia aina ya data ya BigInt kuwakilisha nambari kamili za urefu wowote.

Kwa kuongeza `n` mwishoni mwa nambari, unaweza kuunda BigInt.

```js
const veryBigNumber = 1234567890123456789012345678901234567890n;
```

## Vigezo katika JavaScript

- Vigezo vinaweza kutangazwa kwa kutumia neno muhimu la `let`. 

```js
let cityName;
```

- Ili kuweka thamani kwa kigezo, unaweza kutumia opereta ya mgawo `=`.

```js
cityName = "New York";
```

- Vigezo vilivyotangazwa kwa kutumia `let` vinaweza kupewa thamani mpya tena.

:::interactive_editor

```js
let cityName = "New York";
cityName = "Los Angeles";
console.log(cityName); // Los Angeles
```

:::

- Mbali na `let`, unaweza pia kutumia `const` kutangaza kigezo. Hata hivyo, kigezo cha `const` hakiwezi kupewa thamani mpya tena.

```js
const cityName = "New York";
cityName = "Los Angeles"; // TypeError: Assignment to constant variable.
```

- Vigezo vilivyotangazwa kwa kutumia `const` hutumika kutangaza constants, ambazo haziruhusiwi kubadilika katika msimbo mzima, kama `PI` au `MAX_SIZE`.

## Mikataba ya Kuitaja Vigezo

- Majina ya vigezo yanapaswa kuwa ya kueleweka na yenye maana.
- Majina ya vigezo yanapaswa kuwa camelCase kama `cityName`, `isLoggedIn`, na `veryBigNumber`.
- Majina ya vigezo hayapaswi kuanza na nambari. Yanapaswa kuanza na herufi, `_`, au `$`.
- Majina ya vigezo hayapaswi kuwa na nafasi au herufi maalum, isipokuwa `_` na `$`.
- Majina ya vigezo hayapaswi kuwa maneno muhimu yaliyohifadhiwa.
- Majina ya vigezo yanazingatia herufi kubwa na ndogo. `age` na `Age` ni vigezo tofauti.

## Mfuatano wa herufi na kutokubadilika kwa mfuatano wa herufi katika JavaScript

- Mfuatano wa herufi ni mfululizo wa herufi uliowekwa ndani ya nukuu. Unaweza kuunda mfuatano wa herufi kwa kutumia nukuu moja au alama za nukuu maradufu.

```js
let correctWay = 'This is a string';
let alsoCorrect = "This is also a string";
```

- Mfuatano wa herufi haukubadiliki katika JavaScript. Hii ina maana kwamba mara mfuatano wa herufi unapotengenezwa, huwezi kubadilisha herufi ndani ya mfuatano huo. Hata hivyo, bado unaweza kuweka mfuatano wa herufi mpya kwa thamani mpya.

```js
let firstName = "John";
firstName = "Jane"; // Reassigning the string to a new value
```

## Kuunganisha mfuatano wa herufi katika JavaScript

- Kuunganisha ni mchakato wa kuunganisha mfuatano wa herufi kadhaa au kuunganisha mfuatano wa herufi na vigezo vinavyoshikilia maandishi. Opereta ya `+` ni mojawapo ya njia rahisi na zinazotumika mara kwa mara kuunganisha mfuatano wa herufi. 

:::interactive_editor

```js
let studentName = "Asad";
let studentAge = 25;
let studentInfo = studentName + " is " + studentAge + " years old.";
console.log(studentInfo); // Asad is 25 years old.
```

:::

- Ikiwa unahitaji kuongeza au kuambatisha kwenye mfuatano wa herufi uliopo, basi unaweza kutumia opereta ya `+=`. Hii ni msaada wakati unataka kujenga juu ya mfuatano wa herufi kwa kuongeza maandishi zaidi kwa muda.

:::interactive_editor

```js
let message = "Welcome to programming, ";
message += "Asad!";
console.log(message); // Welcome to programming, Asad!
```

:::

- Njia nyingine ya kuunganisha mfuatano wa herufi ni kutumia njia ya `concat()`. Njia hii huunganisha mfuatano wa herufi mbili au zaidi pamoja.

:::interactive_editor

```js
let firstName = "John";
let lastName = "Doe";
let fullName = firstName.concat(" ", lastName);
console.log(fullName); // John Doe
```

:::

## Kurekodi ujumbe kwa `console.log()`

- Njia ya `console.log()` hutumika kurekodi ujumbe kwenye konsoli. Ni chombo muhimu kwa kurekebisha makosa na kujaribu msimbo wako.

:::interactive_editor

```js
console.log("Hello, World!");
// Output: Hello, World!
```

:::

## Alama za nukuu maradufu katika JavaScript

- Alama za nukuu maradufu hutumika hasa kuashiria mwisho wa usemi. Hii husaidia injini ya JavaScript kuelewa utofauti wa maagizo binafsi, jambo muhimu kwa utekelezaji sahihi.

```js
let message = "Hello, World!"; // first statement ends here
let number = 42; // second statement starts here
```

- Alama za nukuu maradufu husaidia kuzuia mkanganyiko katika utekelezaji wa msimbo na kuhakikisha maagizo yamekamilika kwa usahihi.

## Maoni katika JavaScript

- Kila mstari wa msimbo uliotangazwa kama maoni hauzingatiwi na injini ya JavaScript. Maoni hutumika kuelezea msimbo, kuandika kumbukumbu, au kuzima msimbo kwa muda.
- Maoni ya mstari mmoja hutengenezwa kwa kutumia `//`.

```js
// This is a single-line comment and will be ignored by the JavaScript engine
```

- Maoni ya mistari mingi hutengenezwa kwa kutumia `/*` kuanza maoni na `*/` kumaliza maoni.

```js
/*
This is a multi-line comment.
It can span multiple lines.
*/
```

## JavaScript kama lugha yenye aina za data zinazobadilika wakati wa utekelezaji

- JavaScript ni lugha yenye aina za data zinazobadilika wakati wa utekelezaji, ambayo ina maana kwamba huna haja ya kubainisha aina ya data ya kigezo unapotangaza. Injini ya JavaScript huamua aina ya data kiotomatiki kulingana na thamani iliyowekwa kwa kigezo.

```js
let error = 404; // JavaScript treats error as a number
error = "Not Found"; // JavaScript now treats error as a string
```

- Lugha nyingine, kama C#, ambazo si za aina zinazobadilika wakati wa utekelezaji, zitasababisha hitilafu:

```csharp
int error = 404; // value must always be an integer
error = "Not Found"; // This would cause an error in C#
```

## Kutumia opereta ya `typeof`

- Opereta ya `typeof` hutumika kuangalia aina ya data ya kigezo. Inarudisha mfuatano wa herufi unaoonyesha aina ya kigezo.

:::interactive_editor

```js
let age = 25;
console.log(typeof age); // "number"

let isLoggedIn = true;
console.log(typeof isLoggedIn); // "boolean"
```

:::

- Hata hivyo, kuna tabia maarufu katika JavaScript kuhusu `null`. Opereta ya `typeof` hurudisha `"object"` kwa thamani za `null`.

:::interactive_editor

```js
let user = null;
console.log(typeof user); // "object"
```
