# --description--

インターネットで情報を検索するのは大変な作業に感じることがあります。膨大な数のウェブサイトと大量のデータがあるため、欲しい結果を得るのは不可能に思えるかもしれません。

しかし、ブール値検索構文のようなものを使えば、必要な情報に絞り込むことができます。

ブール値検索構文とは、オンライン検索で正確に探しているものを見つけるために特別な単語を使う方法です。

ここに`freecodecamp curriculum`の基本的な検索クエリの例があります。

`freecodecamp curriculum`を検索すると、検索エンジンはこれらのキーワードの*いずれか*に一致する結果を返します。通常は関連度（クエリにどれだけ合致しているか、または関連しているか）で並べられます。

では、両方のキーワードに一致する結果だけが欲しい場合はどうでしょう？クエリを引用符で囲むことができると聞いたことがあるかもしれません。例えば`"freecodecamp curriculum"`のように。

しかし、このクエリは*フレーズ*`freecodecamp curriculum`に特に一致します。つまり、結果に表示されるのは`freecodecamp curriculum`が一緒に書かれているサイトに限定され、`freecodecamp`と`curriculum`が両方含まれているだけのサイトは含まれません。

両方の用語を含むサイトに一致させたいが、必ずしも隣接したフレーズでなくてもよい場合は、それぞれの用語の前にプラス記号を付けて次のようにします：`+freecodecamp +curriculum`

あるいは、カリキュラム以外のfreeCodeCampの取り組みに興味があるかもしれません。この場合、`curriculum`の前にマイナス記号を付けて次のようにします：`+freecodecamp -curriculum`。

これにより、`freecodecamp`を含む結果は返されますが、`curriculum`も含む結果は除外されます。

または、特定のサイト、例えば私たちのニュースプラットフォームを検索したい場合は、`site:`に続けて検索したいサイトのURLを使います：`site:freecodecamp.org/news curriculum`。

このクエリは`freecodecamp.org/news`から`curriculum`を含むすべての結果を返します。

これらのツールを組み合わせることで、必要な情報を正確に得るための非常に具体的なクエリを作成できます。

例えば、クエリ`site:freecodecamp.org/news +python -curriculum`は、カリキュラムに関する記事を除外しながら、私たちのニュースサイトでPythonの記事を検索します。

検索オペレーターは異なる場合があるため、使いたい検索エンジンのドキュメントを必ず確認してください。

しかし、これでウェブ検索から正確に欲しい結果を得る力がついたはずです。
