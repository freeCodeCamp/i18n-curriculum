# --description--

Nelle lezioni precedenti, hai imparato come aggiungere proprietà e valori `CSS` in questo modo:

```css
element {
 property: value;
}
```

Centra il contenuto dell'elemento `h1` impostando la sua proprietà `text-align` al valore `center`.

```js

assert.equal(h1TextAlign, 'center');
```
