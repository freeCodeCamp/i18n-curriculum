# --explanation--

Марія каже, `at the end of each day`, що чітко вказує на те, що запасні копії плануються щодня. Це означає, що завдання заплановане виконуватися один раз на день. Наприклад:

- `At the end of each day, the system runs a data backup.` — система зберігає всі дані щовечора, щоб уникнути їх втрати.

- `We ensure the servers are updated at the end of each day.` — оновлення застосовуються на серверах щонічно, щоб уникнути перебоїв у робочий час.
