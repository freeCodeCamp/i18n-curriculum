# --explanation--

Ein `participant` ist eine Person, die an einem Ereignis, einer Aktivität oder einem Treffen teilnimmt. Zum Beispiel:

`Each participant in the workshop received a certificate.` – Das bedeutet, dass jeder, der am Workshop teilgenommen hat, ein Zertifikat erhalten hat.

`Internal` bezieht sich auf Dinge innerhalb einer Firma oder Organisation. Es ist das Gegenteil von `external`. Zum Beispiel:

`This event is for internal employees only. It's not for external partners and clients.` – Das bedeutet, dass nur Personen aus der Firma an diesem Ereignis teilnehmen sollen. Personen von außerhalb der Firma sollen nicht teilnehmen.
