# --description--

分子小于其分母的正分数称为适当分数。

对于任意分母 $d$，将有 $d−1$ 个真分数；例如，当 $d = 12$ 时：

$$\frac{1}{12}, \frac{2}{12}, \frac{3}{12}, \frac{4}{12}, \frac{5}{12}, \frac{6}{12}, \frac{7}{12}, \frac{8}{12}, \frac{9}{12}, \frac{10}{12}, \frac{11}{12}$$

我们称不能约分的分数为不可约分数。

此外，我们将分母的弹性定义为其弹性真分数的比例；例如，$R(12) = \frac{4}{11}$。

事实上，$d = 12$ 是最小的分母，其弹性 $R(d) &lt; \frac{4}{10}$。

查找最小的分母 $d$，使其韧性 $R(d) &lt; \frac{15\\,499}{94\\,744}$。
