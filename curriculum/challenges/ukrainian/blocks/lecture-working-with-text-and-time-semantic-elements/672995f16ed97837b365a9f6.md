# --interactive--

Елемент `time` використовується для представлення конкретного моменту часу.

Ось приклад із використанням елемента `time` для представлення двадцяти сотень годин, або восьмої вечора.

:::interactive_editor

```html
<p>The reservations are for <time datetime="20:00">20:00 </time></p>
```

:::

Атрибут `datetime` використовується для перекладу дат і часу у формат, зрозумілий для машини.

Це важливо, бо допомагає з результатами пошукових систем і допомагає браузеру ефективніше обробляти інформацію про дату і час.

Значення атрибута `datetime` має бути або дійсним роком, дійсним місяцем, дійсним часом, локальною датою, глобальною датою або дійсним рядком тривалості.

Ось ще один приклад використання елемента time для представлення конкретної дати:

:::interactive_editor

```html
<p>
  The graduation will be on <time datetime="2024-06-15T15:00">June 15</time>
</p>
```

:::

Значення атрибута `datetime` записане у форматі ISO 8601. ISO 8601 — це міжнародний стандарт для представлення дат і часу.

Перша частина цього значення — це рік, місяць і день. Велика літера T у значенні — це роздільник між датою і часом.

П’ятнадцять сотень годин — це третя година дня.

Коли потрібно представити події, дати публікацій або призначені зустрічі, найкраще використовувати елемент `time`.
