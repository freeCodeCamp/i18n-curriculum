# --description--

`To roll back` significa tornare a uno stato o a una versione precedente di qualcosa. Per esempio:

`If the new app is not working, try rolling back to the old version.`

Un `previous version` è una forma precedente di qualcosa, come una versione più vecchia di un software. Usalo nella stessa frase di prima:

`If the new app is not working, try rolling back to the previous version.`
