# --explanation--

Wenn Sie jemanden in einem formellen Rahmen, wie einem Vorstellungsgespräch, höflich begrüßen möchten, können Sie `您好 (nín hǎo)` verwenden.

Um anzugeben, woher Sie kommen, können Sie die gebräuchliche Struktur `我是 (wǒ shì)` + Nationalität verwenden.

Wenn Ihr Beruf das Erstellen von Software und die Pflege von Anwendungen umfasst, können Sie Ihren Beruf als `开发者 (kāi fā zhě)` vorstellen.
