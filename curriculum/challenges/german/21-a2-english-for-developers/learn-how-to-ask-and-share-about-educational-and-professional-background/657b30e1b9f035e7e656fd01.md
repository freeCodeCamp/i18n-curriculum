# --description--

Ein `employee` ist eine Person, die für ein Unternehmen oder eine andere Person arbeitet. Über `employees' well-being` zu reden bedeutet, du diskutierst die Gesundheit und das Glück der Leute, die an einem Ort arbeiten. Das `'` nach `employees` zeigt, dass das Wohlergehen zu ihnen gehört. Zum Beispiel:

`The project's code` bedeutet, der Code gehört zu einem Projekt. Wenn du über mehrere Prjekte sprichst und sagst `the projects' code`, bedeutet es, der Code gehört zu mehr als einem Projekt.
