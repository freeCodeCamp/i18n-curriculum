# --description--

ABC를 모든 내각이 120도 미만인 삼각형이라고 하겠습니다. X를 삼각형 내부의 임의의 점이라고 하고 $XA = p$, $XC = q$, $XB = r$라고 하겠습니다.

페르마는 토리첼리에게 p + q + r이 최소가 되는 X의 위치를 찾으라고 도전했습니다.

토리첼리는 삼각형 ABC의 각 변 위에 정삼각형 AOB, BNC, AMC를 만들면 AOB, BNC, AMC의 외접원이 삼각형 내부의 한 점 T에서 만난다는 것을 증명했습니다. 더 나아가 T는 토리첼리/페르마 점이라고 불리며 $p + q + r$을 최소화한다는 것도 증명했습니다. 더욱 놀라운 점은 합이 최소가 될 때 $AN = BM = CO = p + q + r$이고 AN, BM, CO도 T에서 만난다는 것입니다.

합이 최소가 되고 a, b, c, p, q, r이 모두 양의 정수일 때 삼각형 ABC를 토리첼리 삼각형이라고 부르겠습니다. 예를 들어, $a = 399$, $b = 455$, $c = 511$은 토리첼리 삼각형의 예이며, 이때 $p + q + r = 784$입니다. 토리첼리 삼각형에 대해 $p + q + r ≤ 120000$인 모든 서로 다른 값의 합을 구하세요.
