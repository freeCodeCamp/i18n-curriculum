# --description--

Tuchukulie mifano michache ya haraka kuona jinsi uhalisia unavyofanya kazi. Angalia msimbo wa HTML na CSS ufuatao:

```html
<!-- index.html -->

<div class="main">
  <div class="list subsection"></div>
</div>
```

```css
/* rule 1 */
.subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

Katika mfano huu hapo juu, sheria zote mbili zinatumia vichaguzi vya darasa la vitu tu, lakini sheria ya 2 ni maalum zaidi kwa sababu inatumia vichaguzi vingi vya darasa la vitu.
