# --description--

動詞 `vivir` は「住む」という意味です。

他の人（「彼」や「彼女」）について話すときは、動詞は `vive` の形に活用されます。通常、その後に前置詞 `en` が続き、どこに住んでいるかを示します。構造は次の通りです：

**`Vive en` + 場所**

フリエタ は マテオ の質問に答えて、バレリア がどこに住んでいるかを伝えています。
