# --explanation--

`First off` は、最初に話すことや行うことを紹介するために使われます。これは、注目すべきことのリストや順序の始まりを示します。例えば：

- `First off, we need to address the project's scope.` – これは、プロジェクトのスコープに最初に取り組むことを意味します。

- `First off, let's review the current goals before we start planning.` – これは、計画に進む前に現在の目標をレビューすることが最初に行われることを意味します。
