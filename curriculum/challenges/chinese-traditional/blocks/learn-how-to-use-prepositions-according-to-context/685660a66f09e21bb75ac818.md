# --explanation--

`On vacation` 意思是暫時不工作，通常是為了休息或旅行。 例如：

`She's on vacation until next Monday.`－這表示她現在沒有工作。

`In July` 指的是發生在一年中第七個月的事情。介系詞 `in` 用於月份之前，以表示某事發生的時間。例如：

`We have a big release scheduled in July。`－這表示發佈會在該月份的某個時間進行。
