# --explanation--

`Integrate into`는 두 가지를 잘 작동하도록 결합하는 것을 의미합니다. 예를 들어:

`The new tool will integrate into our current workflow.` – 이 말은 도구가 팀이 이미 사용하는 방식의 일부가 된다는 뜻입니다.

`Existing`는 이미 존재하는 것을 가리킵니다. 예를 들어:

`We need to update the existing software.` – 이 말은 소프트웨어가 이미 있지만 개선이 필요하다는 뜻입니다.

`Infrastructure`는 비즈니스 운영에 필요한 기본 시스템이나 장비를 의미합니다. 예를 들어:

`Strong IT infrastructure helps companies stay secure.` – 이 말은 신뢰할 수 있는 기술 시스템이 회사 운영을 지원한다는 뜻입니다.
