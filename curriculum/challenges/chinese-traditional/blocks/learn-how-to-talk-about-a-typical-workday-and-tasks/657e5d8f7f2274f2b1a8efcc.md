# --description--

`Every` 和 `any` 常用於句子中，但意義不同。

`Every` 用來談論群組或類別中的所有項目。 例如，`Every student must complete the assignment` 意味著所有學生。

另一方面，`any` 用於疑問句或否定句中，表示一個、一些或全部。 例如，`Can any student answer the question?` 是在詢問是否至少有一個學生能回答這個問題。

# --questions--

## --text--

`every` 和 `any` 之間的差異是什麼？

## --answers--

`Every` 僅用於疑問句；`any` 用於所有項目。

### --feedback--

`Every` 是針對所有項目，而不僅限於問題中；`any` 是針對一個、一些或所有項目。
