# --interactive--

類別選擇器是 CSS 的關鍵零件，允許開發者標的多個具有相同類別屬性的元素並套用一致的樣式。

這使它們在整個網站套用樣式時非常多功能且高效。

類別選擇器是由一個句點（`.`）後接類別名稱所定義。它們可以套用到 HTML 文件中的任何元素。

以下是使用類別選擇器的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="highlight">Example paragraph</p>
```

```css
.highlight {
  color: green;
}
```

:::

在此範例中，任何具有類別 `highlight` 的元素，其文字顏色將設定為 `green`。

類別選擇器的特異性高於型別選擇器，但低於 ID 選擇器和行內樣式。

類別選擇器的特異性值為 `(0, 0, 1, 0)`。這表示類別選擇器可以改寫型別選擇器，但會被 ID 選擇器和行內樣式所改寫。

類別選擇器可以與其他選擇器結合，以創建更具特定性的規則。

以下是一個結合段落型別選擇器與類別選擇器的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="bold-text">Example paragraph</p>
<p class="bold-text">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p.bold-text {
  font-weight: bold;
}
```

:::

此規則僅適用於同時具有 `bold-text` 類別的 `p` 元素，使其文字加粗。
