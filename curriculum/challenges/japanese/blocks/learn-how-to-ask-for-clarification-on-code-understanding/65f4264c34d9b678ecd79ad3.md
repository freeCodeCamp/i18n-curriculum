# --description--

`Uncertain` は、決断に迷うような、何かについて確信が持てない状態を意味します。反対語は `certain` で、確信があることを意味します。例えば、`I am certain this is the right way` は方向性に確信があることを表します。

`Achieve` は、`achieve a good result in a test` のように努力の末に目標を達成することを意味します。コーディングでは、`achieve` はプログラマーがコードで達成したい目標を指すことが多いです。
