# --description--

このレッスンでリンクと画像を使う練習をするために、HTMLのプロジェクトが必要です。

- `odin-links-and-images` という名前の新しいディレクトリを作成してください。

- そのディレクトリ内に、`index.html` という名前の新しいファイルを作成してください。

- 通常のHTMLボイラープレートを記入してください。

- 最後に、`h1` を `body` に追加してください：`<h1>Homepage</h1>`

## アンカー要素
HTMLでリンクを作成するには、アンカー要素を使います。アンカー要素は、リンクにしたいテキストや他のHTML要素を `<a>` タグで囲むことで定義されます。

作成した `body` ページの `index.html` に次の内容を追加し、ブラウザで開いてください：

```html
<a>click me</a>
```

このリンクをクリックしても何も起こらないことに気づいたかもしれません。これは、アンカータグだけではリンク先がわからないためです。リンク先を指定する必要があります。これはHTMLの属性を使って行います。

HTML属性はHTML要素に追加情報を与え、常に要素の開始タグ内に書きます。属性は通常、名前と値の2つの部分で構成されますが、すべての属性が値を必要とするわけではありません。今回の場合、開くアンカータグに `href` （ハイパーリンク参照）属性を追加する必要があります。`href` 属性の値は、リンク先のURLです。

先ほど作成したアンカー要素に次の `href` 属性を追加し、もう一度クリックしてみてください。新しい変更を反映させるためにブラウザを更新するのを忘れないでください。

```html
<a href="https://www.theodinproject.com/about">click me</a>
```

デフォルトでは、`href` 属性がないアンカータグで囲まれたテキストは普通のテキストのように見えます。`href` 属性がある場合、ブラウザはそのテキストを青色にして下線を引き、リンクであることを示します。

アンカータグは他のHTMLドキュメントだけでなく、インターネット上のあらゆる種類のリソースにリンクするために使えることも覚えておいてください。動画、PDFファイル、画像などにリンクできますが、ほとんどの場合は他のHTMLドキュメントにリンクします。
