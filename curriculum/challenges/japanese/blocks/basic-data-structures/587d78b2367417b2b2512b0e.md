# --description--

配列の長さは、含めることができるデータ型と同様に固定されていません。配列は任意の要素数で定義でき、時間の経過とともに要素を追加したり削除したりできます。つまり、配列は<dfn>変更可能</dfn>です。このチャレンジでは、プログラムで配列を変更できる2つのメソッド、`Array.push()` と `Array.unshift()` を見ていきます。

どちらのメソッドも1つ以上の要素をパラメータとして受け取り、その要素をメソッドが呼び出された配列に追加します。`push()` メソッドは配列の末尾に要素を追加し、`unshift()` は配列の先頭に要素を追加します。次の例を考えてみましょう。

```js
let twentyThree = 'XXIII';
let romanNumerals = ['XXI', 'XXII'];

romanNumerals.unshift('XIX', 'XX');
```

`romanNumerals` は `['XIX', 'XX', 'XXI', 'XXII']` という値になります。

```js
romanNumerals.push(twentyThree);
```

`romanNumerals` は `['XIX', 'XX', 'XXI', 'XXII', 'XXIII']` という値になります。変数を渡すこともできるので、配列のデータを動的に変更する際にさらに柔軟に対応できます。

# --instructions--

`mixedNumbers` という関数が定義されており、配列を引数として渡しています。`push()` と `unshift()` を使って配列の先頭に `'I', 2, 'three'` を、末尾に `7, 'VIII', 9` を追加するように関数を修正してください。返される配列には1から9までの数字の表現が順番に含まれるようにします。
