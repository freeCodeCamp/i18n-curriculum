# --description--

El atributo `name` se usa para identificar los datos del formulario después de que se han enviado al servidor.

Aquí tienes un ejemplo de cómo usar el atributo `name`:

```html
<input type="email" name="email">
```

Agrega un atributo `name` al elemento `input` con el valor de `"name"`.

# --hints--

No debes eliminar el elemento `input` existente.

```js
assert.isNotNull(document.querySelector('fieldset input'));
```

No debes crear un nuevo elemento `input`.

```js
assert.lengthOf(document.querySelectorAll('fieldset input'), 1);
```

Tu elemento `input` debe tener un atributo `name` con el valor de `"name"`.

```js
const input = document.querySelector('fieldset input');
assert.strictEqual(input?.getAttribute('name'), 'name');
```
