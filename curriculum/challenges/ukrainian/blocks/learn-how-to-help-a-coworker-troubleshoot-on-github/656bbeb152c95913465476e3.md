# --description--

`Compile` у програмуванні означає перетворення коду, який пишуть люди, у форму, яку комп’ютери можуть зрозуміти та використовувати.

Це схоже на переклад рецепту, написаного однією мовою, на іншу мову, щоб хтось інший міг приготувати страву.

Наприклад, `When we compile Java code, it changes into a form that the computer can run.`
