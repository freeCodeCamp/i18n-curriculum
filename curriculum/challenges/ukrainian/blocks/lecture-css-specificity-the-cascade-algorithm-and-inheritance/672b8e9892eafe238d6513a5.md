# --interactive--

Селектори класу — це ключова частина CSS, що дозволяє розробникам націлюватися на кілька елементів з однаковим атрибутом класу та застосовувати послідовне стилювання.

Це робить їх дуже універсальними та ефективними для застосування стилів по всьому сайту.

Селектори класу визначаються крапкою (`.`) з наступним ім’ям класу. Їх можна застосувати до будь-якого елемента в HTML-документі.

Ось приклад із використанням селектора класу:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="highlight">Example paragraph</p>
```

```css
.highlight {
  color: green;
}
```

:::

У цьому прикладі будь-який елемент із класом `highlight` матиме колір тексту, встановлений у `green`.

Селектори класу мають вищу специфічність, ніж селектори типу, але нижчу за селектори ідентифікатора (ID) та рядкові стилі.

Значення специфічності для селектора класу — `(0, 0, 1, 0)`. Це означає, що селектори класу можуть заміщати селектори типу, але їх можуть заміщати селектори ID і рядкові стилі.

Селектори класу можна комбінувати з іншими селекторами, щоб створювати більш специфічні правила.

Ось приклад поєднання селектора типу абзацу з селектором класу:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="bold-text">Example paragraph</p>
<p class="bold-text">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p.bold-text {
  font-weight: bold;
}
```

:::

Це правило застосовується лише до елементів `p`, які також мають клас `bold-text`, роблячи їхній текст жирним.
