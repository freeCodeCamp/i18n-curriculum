# --interactive--

`grid-template-areas` 屬性讓你透過使用命名標籤來設計視覺網格佈局。

接著，你使用 `grid-area` 屬性將標籤指定到特定的網格項目。換句話說，這些命名的標籤也稱為「網格區域名稱」。

以下是 `grid-template-areas` 屬性的基本語法：

```css
grid-template-areas:
 'header header header'
 'left-sidebar main right-sidebar'
 'footer footer footer';
```

以下是基本語法中需要記住的要點：

- 像 `header` 和 `main` 這樣的值是網格區域的名稱。

- 字串中以空格分隔的每個值對應到一個行。

- 每個字串表述格線中的一列。

所以，在語法中，我們有一個 3×3 的網格容器。

在定義模板之後，你必須使用 `grid-area` 屬性和命名標籤（或區域）作為值，以告訴 CSS 特定元素屬於網格的該區域。

`grid-area` 屬性將網格項目連接到你在 `grid-template-areas` 中定義的命名區域。

展示 `grid-template-areas` 屬性功能的一種流行方式是建立經典的聖杯佈局。

如果你不知道它是什麼，Holy Grail 版面配置是一種包含標頭、頁腳、兩個側邊欄和主要內容區域的網頁設計樣式。它確保主要內容具有優先權，同時側邊欄和其他區塊會在版面配置中響應式調整。

有許多解決方案可以實作聖杯版面配置，但使用 `grid-template-areas` 和 `grid-area` 屬性是創建它最直接的方法。

以下是聖杯佈局的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="header">
    <h2>Header</h2>
  </div>
  <div class="sidebar-left">
    <h2>Left Sidebar</h2>
  </div>
  <div class="main"><h2>Main Content</h2></div>
  <div class="sidebar-right">
    <h2>Right Sidebar</h2>
  </div>
  <div class="footer">
    <h2>Footer</h2>
  </div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-areas:
    'header header header'
    'sidebar-left main sidebar-right'
    'footer footer footer';
  grid-gap: 10px;
  background-color: #2196F3;
  padding: 10px;
}

.header {
  grid-area: header;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.sidebar-left {
  grid-area: sidebar-left;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.main {
  grid-area: main;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.sidebar-right {
  grid-area: sidebar-right;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.footer {
  grid-area: footer;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}
```

:::

請注意，`grid-template-areas` 和 `grid-area` 屬性皆可獨立使用。

`grid-template-areas` 屬性專門用於透過在網格容器中映射命名的網格區域來定義視覺佈局。

另一方面，`grid-area` 屬性用於定位單一的網格項目，方法是指定它們的列和行位置，或取用使用 `grid-template-areas` 屬性定義的命名區域。
