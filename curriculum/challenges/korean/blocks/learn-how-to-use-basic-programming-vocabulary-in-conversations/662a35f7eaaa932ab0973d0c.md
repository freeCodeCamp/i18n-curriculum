# --description--

프로그래밍에서 `variables`, `data types` 그리고 `conditional statements`를 배우세요.

`variable`는 정보를 저장할 수 있는 상자와 같습니다. 이 정보는 숫자, 텍스트 또는 다른 유형의 데이터일 수 있습니다. 상자 안의 내용을 바꿀 수 있기 때문에 `variable`라고 부르며, 이는 변할 수 있다는 뜻입니다. 예를 들어 `x = 5`에서 `x`는 임시로 값 5를 저장합니다.

`data type`는 `variable`가 저장할 수 있는 데이터 유형을 지정하는 분류입니다. 일반적인 `data types`에는 정수(온전한 숫자), 부동소수점(소수점이 있는 숫자), 문자열(텍스트), 불리언(참 또는 거짓)이 포함됩니다. 예를 들어 `age = 21`라면 `data type`는 정수입니다.

`conditional statement`는 특정 조건이 참일 때만 컴퓨터에게 무언가를 하라고 지시하는 명령입니다. 실제 생활에서 특정 상황에 따라 결정을 내리는 것과 같습니다. 일반적인 `conditional statements`는 `if-else` 문장입니다. 예를 들어 `if (temperature > 20) {console.log("It's warm outside!");}`는 온도가 20도 이상일 때 `conditional statement`를 실행하는 `console.log("It's warm outside!")`입니다.
