# --explanation--

`Past Continuous` 시제는 과거 특정 시점에 진행 중이던 동작의 지속성이나 배경을 강조할 때 자주 사용됩니다. 과거 특정 시점에 진행 중이던 동작을 설명할 때 쓰입니다. 구조는 **`was`/`were` + `-ing`로 끝나는 동사**입니다.

Jake의 문장에서 `I was reviewing`는 어제 검토하는 동작이 진행 중이었음을 보여줍니다. 또 다른 예:

`She was testing the new feature when the system crashed.` - 여기서 시스템이 다운된 것은 그녀가 새로운 기능을 테스트하는 동작 중간에 일어난 일로, 테스트는 더 오래 진행된 동작이었습니다.
