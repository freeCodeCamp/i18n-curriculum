# --description--

Die RSA-Verschlüsselung basiert auf folgendem Verfahren:

Erzeugen Sie zwei verschiedene Primzahlen `p` und `q`.
Berechnen Sie `n=p*q` und `φ=(p-1)(q-1)`.
Finden Sie eine ganze Zahl `e`, `1 < e < φ`, so dass `gcd(e,φ) = 1`

Eine Nachricht in diesem System ist eine Zahl im Intervall `[0,n-1]`.
Ein zu verschlüsselnder Text wird dann auf irgendeine Weise in Nachrichten (Zahlen im Intervall `[0,n-1]`) umgewandelt.
Um den Text zu verschlüsseln, wird für jede Nachricht `m`, c=m<sup>e</sup> mod n berechnet.

Um den Text zu entschlüsseln, ist folgendes Verfahren nötig: Berechnen Sie `d`, so dass `ed=1 mod φ`, dann für jede verschlüsselte Nachricht `c`, berechnen Sie m=c<sup>d</sup> mod n.

Es existieren Werte von `e` und `m`, so dass m<sup>e</sup> mod n = m gilt.
Wir nennen Nachrichten `m`, für die m<sup>e</sup> mod n=m gilt, unverschleierte Nachrichten.

Ein Problem bei der Wahl von `e` ist, dass es nicht zu viele unverschleierte Nachrichten geben sollte.
Zum Beispiel sei `p=19` und `q=37`.
Dann `n=19*37=703` und `φ=18*36=648`.
Wenn wir `e=181` wählen, dann, obwohl `gcd(181,648)=1`, stellt sich heraus, dass alle möglichen Nachrichten
m `(0≤m≤n-1)` unverschleiert sind, wenn man m<sup>e</sup> mod n berechnet.
Für jede gültige Wahl von `e` existieren einige unverschleierte Nachrichten.
Es ist wichtig, dass die Anzahl der unverschleierten Nachrichten ein Minimum ist.

Für ein gegebenes `p` und `q` finden Sie die Summe aller Werte von `e`, `1 < e < φ(p,q)` und `gcd(e,φ)=1`, so dass die Anzahl der unverschleierten Nachrichten für diesen Wert von `e` ein Minimum ist.

# --solutions--

```js
function gcd(a, b) {
    if (b)
        return gcd(b, a % b);
    else
        return a;
}

function RSAEncryption(p, q) {
    let phi = (p - 1) * (q - 1);

    let best = Number.MAX_SAFE_INTEGER;
    let sum = 0;

    for (let e = 0; e < phi; ++e) {
        if (!(gcd(e, phi) == 1))
            continue;

        let msg = (gcd(p - 1, e - 1) + 1) * (gcd(q - 1, e - 1) + 1);

        if (best == msg) {
            sum += e;
        } else if (best > msg) {
            best = msg;
            sum = e;
        }
    }

    return sum;
}
```
