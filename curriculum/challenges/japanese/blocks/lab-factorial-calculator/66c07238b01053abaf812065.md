# --description--

階乗とは、ある整数とその整数より小さいすべての整数の積のことです。例えば、`5` の階乗は `5 * 4 * 3 * 2 * 1 = 120` です。このラボでは、ユーザーから数値を受け取り、その数値の階乗を計算する階乗計算機を作成します。

**目的:** 以下のユーザーストーリーを満たし、すべてのテストに合格してラボを完了してください。

**ユーザーストーリー:**

1. `num` という変数を宣言し、任意の数値を代入してください。代入する数値は `1` から `20` までの範囲内である必要があります。
1. `factorialCalculator` という名前の関数を作成し、数値を引数として受け取り、その数値の階乗を返してください。
1. 関数内で `result` という変数を宣言し、`1` の値を代入してください。ループを使って `1` から入力された数値（含む）までのすべての数値を繰り返し処理し、各数値に対して `result` 変数を現在の数値で乗算し、その結果を `result` 変数に代入してください。ここでは `for` ループ、`while` ループ、または `do...while` ループのいずれかを使うことができます。
1. `factorialCalculator` 関数を `num` を引数として呼び出し、その結果を `factorial` 変数に代入してください。
1. 最終的な出力を `Factorial of [num] is [factorial]` の形式で保存し、`resultMsg` 変数に代入してください。
1. `resultMsg` の値をコンソールに出力してください。

# --solutions--

```js
const num = 5;

function factorialCalculator(n) {
  let result = 1;

  for (let i = 2; i <= n; i++) {
    result *= i;
  }

  return result;
}

const factorial = factorialCalculator(num);
const resultMsg = `Factorial of ${num} is ${factorial}`;
console.log(resultMsg);
```
