# --description--

الدالة $lcm(a, b)$ تمثل أصغر مضاعف مشترك بين $a$ و $b$.

لتكن $A(n)$ المتوسط الحسابي لقيم $lcm(n, i)$ حيث $1 ≤ i ≤ n$.

مثلاً: $A(2) = \frac{2 + 2}{2} = 2$ و $A(10) = \frac{10 + 10 + 30 + 20 + 10 + 30 + 70 + 40 + 90 + 10}{10} = 32$.

لتكن $S(n) = \sum A(k)$ حيث $1 ≤ k ≤ n$.

$S(100) = 122\\,726$.

احسب $S(99\\,999\\,999\\,019)\bmod 999\\,999\\,017$.
