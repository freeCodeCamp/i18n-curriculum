# --description--

幅優先探索の方法では、キューを使って丸括弧の組み合わせを作る過程で異なる状態を追跡します。各状態は3つの要素を含む配列で表されます：

- 現在作成中の文字列
- これまでに使った開き丸括弧の数
- これまでに使った閉じ丸括弧の数

`queue` という名前の変数を作成し、空の文字列と丸括弧が0個使われている状態を表す配列 `['', 0, 0]` を含む配列で初期化してください。

# --hints--

`queue` 関数内で `genParentheses` という名前の変数を宣言してください。

```js
assert.match(genParentheses.toString(), /queue/);
```

`queue` を `['', 0, 0]` を含む配列で初期化してください。

```js

```
