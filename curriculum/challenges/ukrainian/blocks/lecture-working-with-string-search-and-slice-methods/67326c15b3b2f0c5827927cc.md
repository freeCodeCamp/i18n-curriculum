# --interactive--

Працюючи з рядками в JavaScript, часто потрібно витягти частину або підрядок із більшого рядка.

Наприклад, ви можете захотіти витягти частину слова, певну послідовність символів або просто фрагмент речення.

JavaScript надає кілька методів для цього завдання, одним із найпоширеніших є метод `slice()`.

Метод `slice()` дозволяє витягти частину рядка і повертає новий рядок, не змінюючи оригінальний рядок. Він приймає два параметри: початковий індекс і необов’язковий кінцевий індекс.

Ось базовий синтаксис:

```js
string.slice(startIndex, endIndex);
```

`startIndex` — це позиція, з якої починається витягування. `endIndex` — це позиція, де витягування закінчується. Якщо кінцевий індекс не вказано, `slice()` витягує до кінця рядка.

Розглянемо простий приклад витягування частини рядка:

:::interactive_editor

```js
let message = "Hello, world!";
let greeting = message.slice(0, 5);

console.log(greeting);  // Hello
```

:::

У цьому прикладі `slice(0, 5)` витягує символи, починаючи з індексу `0` і до, але не включно з індексом `5`. В результаті витягується слово `Hello`.

Якщо пропустити другий параметр, `slice()` витягне все від початкового індексу до кінця рядка:

:::interactive_editor

```js
let message = "Hello, world!";
let world = message.slice(7);

console.log(world);  // world!
```

:::

Тут `slice(7)` витягує рядок від індексу `7` до кінця рядка, отримуючи `world!`.

Ви також можете використовувати від’ємні числа як індекси. Коли ви використовуєте від’ємне число, воно рахується з кінця рядка назад:

:::interactive_editor

```js
let message = "JavaScript is fun!";
let lastWord = message.slice(-4);

console.log(lastWord);  // fun!
```

:::

У цьому випадку `slice(-4)` витягує останні чотири символи з рядка, отримуючи `fun!`.

Припустимо, ви хочете витягти розділ із середини рядка. Ви можете вказати і початковий, і кінцевий індекси, щоб точно контролювати, яку частину рядка хочете:

:::interactive_editor

```js
let message = "I love JavaScript!";
let language = message.slice(7, 17);

console.log(language);  // JavaScript
```

:::

Тут `slice(7, 17)` витягує підрядок, починаючи з індексу 7 і закінчуючи безпосередньо перед індексом `17`, що є словом `JavaScript`.

Метод `slice()` — це потужний інструмент для витягування частин рядка в JavaScript.

Ви вказуєте початковий і кінцевий індекси, і метод повертає новий рядок, що містить витягнуту частину.

З можливістю використовувати додатні, від’ємні та пропущені індекси, ви можете адаптувати його до різних ситуацій, не змінюючи оригінальний рядок.
