# --explanation--

`Get used to` は新しいものに適応することを意味します。通常、名詞か`-ing`形が続きます。例えば：

- `New remote employees need time to get used to asynchronous communication.` - リモートワーカーは最初は苦労するかもしれませんが、最終的にはこの働き方に適応します。

- `I can't get used to working without dual monitors.` - 話者は1つの画面だけを使うことに慣れるのが難しいと感じています。

アンナは、非同期のリモートワークは最初は慣れないかもしれませんが、時間が経つにつれて調整され、うまく機能するようになると言っています。
