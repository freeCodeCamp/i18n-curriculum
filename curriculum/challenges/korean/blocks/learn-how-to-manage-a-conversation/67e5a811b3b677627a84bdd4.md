# --explanation--

`Might`는 어떤 일이 가능하지만 확실하지 않음을 나타내는 데 사용됩니다. Jessica는 버그가 문제를 *일으킬 수도* 있다고 말하지만 아직 확정된 것은 아닙니다. `Simple Present`, `we've discovered a critical bug that affects the user interface`와 비교해 보세요. 이들은 버그가 확실히 문제를 일으킨다는 뜻입니다. 다른 예:

- `The delay might cause problems.` – 가능하지만 확실하지 않음.

- `The delay causes problems.` – 확실히 문제를 일으킴.
