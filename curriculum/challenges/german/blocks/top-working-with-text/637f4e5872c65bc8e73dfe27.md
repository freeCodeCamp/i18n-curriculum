# --description--

Was würden Sie erwarten, dass der folgende Text auf einer HTML-Seite ausgibt?

```html
<body>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor
  incididunt ut labore et dolore magna aliqua.

  Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris
  nisi ut aliquip ex ea commodo consequat.
</body>
```

Es sieht aus wie zwei Textabsätze, und daher könnten Sie erwarten, dass es auch so angezeigt wird. Das ist jedoch nicht der Fall, wie Sie in der Ausgabe unten sehen können:

Wenn der Browser in Ihrem HTML auf Zeilenumbrüche wie diese trifft, komprimiert er sie zu einem einzigen Leerzeichen. Das Ergebnis dieser Komprimierung ist, dass der gesamte Text zu einer langen Zeile zusammengefasst wird.

Wenn Sie Absätze in HTML erstellen möchten, müssen Sie das Absatz-Element verwenden, das nach jedem Absatz einen Zeilenumbruch hinzufügt. Ein Absatz-Element wird definiert, indem Sie den Textinhalt mit einem `<p>` Tag umschließen.
