# --description--

Прості числа 3, 7, 109 і 673 є досить особливими. Якщо взяти будь-які два прості числа і об’єднати їх у будь-якому порядку, результат завжди буде простим числом. Наприклад, взявши 7 і 109, обидва числа 7109 і 1097 є простими. Сума цих чотирьох простих чисел, 792, є найменшою сумою для множини з чотирьох простих чисел з такою властивістю.

Знайдіть найменшу суму для множини з п’яти простих чисел, для яких будь-які два прості числа при конкатенації утворюють інше просте число.
