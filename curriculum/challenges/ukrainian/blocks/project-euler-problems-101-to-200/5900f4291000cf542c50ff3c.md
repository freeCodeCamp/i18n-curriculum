# --description--

Розглянемо таку конфігурацію з 64 трикутників:

<img alt="64 трикутники, розташовані для створення більшого трикутника зі стороною довжиною 8 трикутників" src="https://cdn.freecodecamp.org/curriculum/project-euler/tri-colouring-a-triangular-grid-1.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Ми хочемо пофарбувати внутрішню частину кожного трикутника одним із трьох кольорів: червоним, зеленим або синім, так щоб жодні два сусідні трикутники не мали однаковий колір. Таке фарбування називатимемо дійсним. Тут два трикутники вважаються сусідніми, якщо вони мають спільну сторону. Примітка: якщо вони ділять лише вершину, то вони не є сусідами.

Наприклад, ось дійсне фарбування наведеної сітки:

<img alt="фарбована сітка з 64 трикутників" src="https://cdn.freecodecamp.org/curriculum/project-euler/tri-colouring-a-triangular-grid-2.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Фарбування C', яке отримане з фарбування C шляхом обертання або відображення, вважається відмінним від C, якщо вони не є ідентичними.

Скільки існує різних дійсних фарбувань для наведеної конфігурації?
