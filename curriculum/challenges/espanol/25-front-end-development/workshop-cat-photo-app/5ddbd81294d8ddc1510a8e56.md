# --description--

El texto de un enlace debe colocarse entre las etiquetas de apertura y cierre de un elemento de anclaje (`a`).

Aquí hay un ejemplo de un enlace con el texto `click here to go to freeCodeCamp.org`:

```html
<a href="https://www.freecodecamp.org">click here to go to freeCodeCamp.org</a>
```

Agrega el texto de anclaje `cat photos` al elemento de anclaje. Esto se convertirá en el texto del enlace.

# --hints--

Tu elemento de anclaje (`a`) debe tener una etiqueta de apertura. Las etiquetas de apertura tienen esta sintaxis: `<elementName>`.

```js
assert.exists(document.querySelector('a'));
```

Tu elemento de anclaje (`a`) debe tener una etiqueta de cierre. Las etiquetas de cierre tienen un `/` justo después del carácter `<`.

```js
assert.match(code, /<\/a\>/);
```

El texto de tu elemento de anclaje (`a`) debe ser `cat photos`. Asegúrate de poner el texto del enlace entre la etiqueta de apertura y la etiqueta de cierre del elemento de anclaje (`a`).

```js
assert.equal(
  document.querySelector('a')?.innerText?.trim().toLowerCase().replace(/\s+/g, ' '),
    'cat photos'
);
```
