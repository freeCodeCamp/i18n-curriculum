# --description--

JavaScript 中的 `throw` 语句用于抛出用户定义的异常。编程中的异常是指发生了意外事件，打断了程序的正常流程。

作为程序员，处理这些异常非常重要，这样当错误发生时你的程序不会意外崩溃。以下是 `throw` 语句的基本语法：

```js
throw expression;
```

这里的 `expression` 是表示你想要抛出的异常的对象或值。例子包括内置的异常类，比如 `Error`、`TypeError` 或 `RangeError` 类。下面是使用 `throw` 语句抛出 `TypeError` 的示例：

```js
function validateNumber(input) {
  if (typeof input !== "number") {
    throw new TypeError("Expected a number, but received " + typeof input);
  }
  return input * 2;
}
```

在这个例子中，我们正在检查 `input` 的类型是否不是 `number`。如果不是，则抛出带有自定义消息的 `TypeError`。否则，函数将返回 `input` 乘以 `2` 的结果。

如果你想抛出更通用的错误信息，那么你可以像这样引用 `Error` 构造函数：

```js
function divide(numerator, denominator) {
  if (denominator === 0) {
    throw new Error("Cannot divide by zero");
  }
  return numerator / denominator;
}
```

这是一个函数示例，用于查看分母是否为 `0`。如果是这种情况，它将抛出一个自定义错误信息，内容为 `Cannot divide by zero`。

在下一课中，我们将学习如何在 `try`/`catch` 块的上下文中抛出错误信息，这用于优雅地处理 JavaScript 中的异常。
