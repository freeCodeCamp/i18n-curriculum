# --description--

함수 안에서 선언된 변수는 함수 내부에서만 보입니다. 이것을 변수의 범위라고 하며, 이 특정 범위를 지역 범위라고 합니다. 함수 밖에서 선언된 변수는 전역 변수라고 하며 프로그램 전체에서 보입니다. 예를 들어, 다음 코드는 오류를 발생시킵니다:

```js
function myFunction() {
  let x = 10;
}

console.log(x);
```

변수 `x`는 함수 `myFunction` 안에서 선언되어 함수 밖에서는 보이지 않습니다. `ReferenceError`가 정의되지 않아 `x`가 발생합니다.

함수 밖에서 변수를 선언하면 전역 변수라고 하며 프로그램 전체에서 보입니다. 예를 들어, 다음 코드는 정상 작동합니다:

```js
let x = 10;

function myFunction() {
  console.log(x);
}

myFunction();
```

함수 안에서 같은 이름의 변수가 선언되면 외부 변수를 가립니다.
