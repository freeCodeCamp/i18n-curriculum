# --description--

Sei $x_1, x_2, \ldots, x_n$ eine Folge der Länge $n$, so dass:

- $x_1 = 2$
- für alle $1 &lt; i ≤ n : x_{i - 1} &lt; x_i$
- für alle $i$ und $j$ mit $1 ≤ i, j ≤ n : {(x_i)}^j &lt; {(x_j + 1)}^i$

Es gibt nur fünf solche Folgen der Länge 2, nämlich: {2,4}, {2,5}, {2,6}, {2,7} und {2,8}. Es gibt 293 solche Folgen der Länge 5; drei Beispiele sind unten angegeben: {2,5,11,25,55}, {2,6,14,36,88}, {2,8,22,64,181}.

Sei $t(n)$ die Anzahl solcher Folgen der Länge $n$. Es ist gegeben, dass $t(10) = 86195$ und $t(20) = 5227991891$.

Finden Sie $t({10}^{10})$ und geben Sie Ihre Antwort modulo $10^9$ an.
