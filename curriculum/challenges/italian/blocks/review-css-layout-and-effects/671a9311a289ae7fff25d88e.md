## Proprietà CSS overflow

- **Definizione**: Overflow indica il modo in cui gli elementi gestiscono il contenuto che supera, o "esce", dalle dimensioni dell'elemento contenitore. L'overflow è bidimensionale.
- **`overflow-x`**: L'asse x determina l'overflow orizzontale.
- **`overflow-y`**: l'asse y determina l'overflow verticale.
- **`overflow`**: Proprietà abbreviata per `overflow-x` e `overflow-y`. Se viene fornito un solo valore, entrambi gli overflow lo useranno. Se vengono forniti due valori, il `overflow-x` userà il primo e il `overflow-y` userà il secondo.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="overflow-box">
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
</div>
```

```css
.overflow-box {
  width: 140px;
  height: 70px;
  border: 2px solid black;
  overflow-x: auto;
  overflow-y: auto;
  white-space:nowrap;
}
```

## Proprietà CSS transform

- **Definizione**: Questa proprietà ti permette di applicare varie trasformazioni agli elementi, come ruotarli, scalarli, inclinarli o traslarli (spostarli) nello spazio 2D o 3D.
- **Funzione `translate()`**: Questa funzione serve a spostare un elemento dalla sua posizione attuale.
- **Funzione `scale()`**: Questa funzione ti permette di cambiare la dimensione di un elemento.
- **Funzione `rotate()`**: Questa funzione ti permette di ruotare un elemento.
- **Funzione `skew()`**: Questa funzione ti permette di inclinare un elemento.
- **Trasformazioni e accessibilità**: Se usi transform per nascondere o mostrare contenuti, assicurati che il contenuto sia ancora accessibile ai lettori di schermo e alla navigazione tramite tastiera. Il contenuto nascosto dovrebbe essere veramente nascosto, ad esempio usando `display: none` o `visibility: hidden`, invece di essere semplicemente spostato visivamente fuori schermo.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="transform-box translate">Translate</div>
  <div class="transform-box scale">Scale</div>
  <div class="transform-box rotate">Rotate</div>
  <div class="transform-box skew">Skew</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: column;
  align-items: center;
  gap: 16px;
  flex-wrap: wrap;
}

.transform-box {
  width: 100px;
  height: 100px;
  background: coral;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

/* translate() */
.translate {
  transform: translateX(20px);
}

/* scale() */
.scale {
  transform: scale(1.2);
}

/* rotate() */
.rotate {
  transform: rotate(15deg);
}

/* skew() */
.skew {
  transform: skew(10deg, 5deg);
}
```

## Il modello a riquadro (box model)

- **Definizione**: Nel modello a riquadro CSS, ogni elemento è circondato da un riquadro. Questo riquadro è composto da quattro componenti: l'area del contenuto, `padding`, `border`, `margin`.
- **Area del contenuto**: L'area del contenuto è la parte più interna del riquadro. È lo spazio che contiene il contenuto effettivo di un elemento, come testo o immagini.
- **`padding`**: La spaziatura interna è l'area immediatamente dopo l'area del contenuto. È lo spazio tra l'area del contenuto e il bordo di un elemento.
- **`border`**: Il bordo è il margine esterno o contorno di un elemento nel modello a riquadro CSS. È il confine visivo dell'elemento.
- **`margin`**: Il margine è lo spazio esterno al bordo di un elemento. Determina la distanza tra un elemento e gli altri elementi intorno a esso.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="box-model">Content</div>
```

```css
.box-model {
  width: 120px;
  background: lightblue;
  padding: 16px;
  border: 4px solid teal;
  margin: 20px;
  text-align: center;
}
```

:::

## Collasso del margine

- **Definizione**: Questo comportamento si verifica quando i margini verticali di elementi adiacenti si sovrappongono, risultando in un singolo margine uguale al più grande dei due. Questo comportamento si applica solo ai margini verticali (superiore e inferiore), non ai margini orizzontali (sinistro e destro).

## I valori delle proprietà `content-box` e `border-box`

- **Proprietà `box-sizing`**: Questa proprietà serve a determinare come vengono calcolati i `width` e `height` finali per un elemento HTML.
- **Valore `content-box`**: Nel modello `content-box`, i `width` e `height` che imposti per un elemento determinano le dimensioni dell'area del contenuto ma non includono il `padding`, `border` o `margin`.
- **Valore `border-box`**: Con `border-box`, i `width` e `height` di un elemento includono l'area del contenuto, il `padding` e il `border`, ma non includono il `margin`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="sizing-box content-box">Content-box</div>
<div class="sizing-box border-box">Border-box</div>
```

```css
.sizing-box {
  width: 150px;
  padding: 10px;
  border: 4px solid teal;
  margin-bottom: 10px;
  background: peachpuff;
  text-align: center;
}

.content-box {
  box-sizing: content-box;  
}

.border-box {
  box-sizing: border-box;
  background: lightgreen;
}
```

## Reset CSS

- **Definizione**: Un reset CSS è un foglio di stile che rimuove tutta o parte della formattazione predefinita che i browser web applicano agli elementi HTML. Opzioni di terze parti per i reset CSS includono `sanitize.css` e `normalize.css`.

## Proprietà CSS filter

- **Definizione**: Questa proprietà può essere usata per creare vari effetti come sfocatura, spostamento del colore e regolazione del contrasto.
- **Funzione `blur()`**: Questa funzione applica una sfocatura gaussiana all'elemento. La quantità è definita in pixel e rappresenta il raggio della sfocatura.
- **Funzione `brightness()`**: Questa funzione regola la luminosità dell'elemento. Un valore di 0% rende l'elemento completamente nero, mentre valori superiori al 100% aumentano la luminosità.
- **Funzione `contrast()`**: Questa funzione regola il contrasto dell'elemento. Un valore di 0% rende l'elemento completamente grigio, 100% lo mostra normalmente e valori superiori al 100% aumentano il contrasto.
- **Funzione `grayscale()`**: Questa funzione converte l'elemento in scala di grigi. La quantità è definita in percentuale, dove 100% è completamente in scala di grigi e 0% lascia l'immagine invariata.
- **Funzione `sepia()`**: Questa funzione applica una tonalità seppia all'elemento. Come la scala di grigi, usa un valore percentuale.
- **Funzione `hue-rotate()`**: Questa funzione applica una rotazione della tonalità all'elemento. Il valore è definito in gradi e rappresenta una rotazione intorno al cerchio dei colori.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <img class="filter blur" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Blur" />
  <img class="filter brightness" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Brightness" />
  <img class="filter contrast" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Contrast" />
  <img class="filter grayscale" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Grayscale" />
  <img class="filter sepia" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Sepia" />
  <img class="filter hue" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Hue Rotate" />
</div>
```

```css
.container {
  display: flex;
  gap: 12px;
  flex-wrap: wrap;
}

.filter {
  width: 120px;
}

/* blur() */
.blur {
  filter: blur(3px);
}

/* brightness() */
.brightness {
  filter: brightness(130%);
}

/* contrast() */
.contrast {
  filter: contrast(150%);
}

/* grayscale() */
.grayscale {
  filter: grayscale(100%);
}

/* sepia() */
.sepia {
  filter: sepia(100%);
}

/* hue-rotate() */
.hue {
  filter: hue-rotate(90deg);
}
```
