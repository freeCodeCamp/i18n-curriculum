# --description--

Dhana muhimu ya kwanza unayohitaji kuelewa ili kufanikiwa katika CSS ni mfano wa sanduku. Haijumuishi ugumu, lakini kuiruka sasa kutasababisha kukasirika sana baadaye.

Kila kitu kwenye ukurasa wa mtandao ni sanduku la mstatili. Masanduku haya yanaweza kuwa na masanduku mengine ndani yake na yanaweza kuwekwa pembeni kwa kila mmoja. Unaweza kupata wazo la jumla la jinsi hii inavyofanya kazi kwa kuweka mpaka kwenye kila kipengee kwenye ukurasa kama hii:

```css
* {
  border: 1px solid red;
}
```

# --assignment--

Ongeza mpaka kwenye kila kipengee kwenye ukurasa na uone jinsi masanduku yanavyopangwa.
