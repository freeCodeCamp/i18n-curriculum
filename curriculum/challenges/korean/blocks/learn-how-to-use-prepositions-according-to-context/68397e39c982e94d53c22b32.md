# --explanation--

전치사 `on`는 여러 의미가 있지만, 이 도전 과제에서는 논의하거나 작업하는 주제나 대상을 나타내는 데 사용해야 합니다.

`Working on the new software update`는 팀이 특정 프로젝트나 주제에 노력을 집중하고 있음을 의미합니다.

다른 문장들은 `on`를 시간(`on the weekend`, `on time`)이나 장소/장치(`on his computer`)를 나타내는 데 사용해 다른 의미를 가집니다.
