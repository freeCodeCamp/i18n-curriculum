# --interactive--

當你深入了解 JavaScript 時，你會很快發現執行數學操作是常見的任務。雖然基本的算術運算子可以處理簡單的計算，但 JavaScript 提供了內建的 `Math` 物件來應付更複雜的數學挑戰。

這個方便的工具提供多種方法，使進行進階計算和操作數字更加容易。讓我們來探索這些方法，看看它們如何簡化你的程式設計體驗。

`Math.random()` 方法會產生介於 `0`（包含）和 `1`（不包含）之間的隨機浮點數。這表示可能的輸出可以是 `0`，但實際上永遠不會達到 `1`。以下是使用 `Math.random()` 方法的範例：

:::interactive_editor

```js
const randomNum = Math.random();

console.log(randomNum);
// any number between 0 and 1 – 0 inclusive and 1 exclusive
```

:::

`Math.min()` 和 `Math.max()` 都會接受一組數字並分別傳回最小值和最大值。以下是同時使用這兩個方法的範例：

:::interactive_editor

```js
const smallest = Math.min(1, 5, 3, 9);
console.log(smallest); // 1

const largest = Math.max(1, 5, 3, 9);
console.log(largest); // 9
```

:::

第一個 `console.log()` 會記錄數字 `1`，因為 `1` 是該數字列表中最小的數字。第二個 `console.log()` 會記錄數字 `9`，因為 `9` 是該數字列表中最大的數字。

如果你想將數字向上或向下取整到最接近的整數，你可以使用 `Math.ceil()` 和 `Math.floor()` 方法。以下是使用 `Math.ceil()` 的範例：

:::interactive_editor

```js
console.log(Math.ceil(4.3)); // 5
```

:::

`Math.ceil()` 會將 `4.3` 無條件進位到最接近的整數，在此例中為 `5`。現在，讓我們來看看如何將數字無條件捨去：

:::interactive_editor

```js
console.log(Math.floor(4.7)); // 4
```

:::

`Math.floor()` 會將 `4.7` 向下取整到最接近的整數，在此例中為 `4`。`Math.round()` 是 `Math.ceil()` 和 `Math.floor()` 的混合體。它會根據小數點將數字四捨五入到最接近的整數：

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

所以，如果小數點小於 `5`，數字會向下取整。如果小數點是 `5` 或更大，數字會向上取整。`Math.floor()` 和 `Math.random()` 的一個實際應用是為兩個整數之間產生隨機數。以下是該語法：

:::interactive_editor

```js
const max = 10;
const min = 5;
const randomNum = Math.floor(Math.random() * (max - min + 1)) + min;
console.log(randomNum);
```

:::

在 `20` 和 `1` 之間產生隨機數看起來會是這樣：

:::interactive_editor

```js
const randomNumBtw1And20 = Math.floor(Math.random() * 20) + 1;
console.log(randomNumBtw1And20);
```

:::

另一個有用的 `Math` 方法是 `Math.trunc()` 方法。`Math.trunc()` 會移除數字的小數部分，只傳回整數部分，且不進行四捨五入：

:::interactive_editor

```js
console.log(Math.trunc(2.9)); // 2
console.log(Math.trunc(9.1)); // 9
```

:::

如果你需要取得一個數字的平方根或立方根，你可以分別使用 `Math.sqrt()` 和 `Math.cbrt()` 方法：

:::interactive_editor

```js
console.log(Math.sqrt(81)); // 9
console.log(Math.cbrt(27)); // 3
```

:::

第一個日誌述語會記錄 `9`，因為 `81` 的平方根是 `9`，而第二個日誌述語會記錄 `3`，因為 `27` 的立方根是 `3`。如果你需要取得數字的絕對值，你可以使用 `Math.abs()` 方法：

:::interactive_editor

```js
console.log(Math.abs(-5)); // 5
console.log(Math.abs(5)); // 5
```

:::

`Math.abs()` 傳回數字的絕對值，將負數轉為正數。我們將要看的最後一個方法是 `Math.pow()` 方法：

:::interactive_editor

```js
console.log(Math.pow(2, 3)); // 8
console.log(Math.pow(8, 2)); // 64
```

:::

`Math.pow()` 接受兩個數字並將第一個數字引發到第二個數字的次方。還有許多屬於 `Math` 物件的其他方法， 你可以自行探索。然而，這些只是 JavaScript 程式碼庫中較常用的幾個方法。
