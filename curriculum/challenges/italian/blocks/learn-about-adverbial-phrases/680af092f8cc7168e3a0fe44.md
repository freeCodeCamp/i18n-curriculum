# --explanation--

`To cost` significa richiedere un pagamento o avere un prezzo. Quando dici che qualcosa `costs more than expected`, significa che devi pagare più soldi di quanto avevi previsto. Per esempio:

`The event cost more than we budgeted for because of extra equipment.` – Questo significa che le spese finali sono state più alte di quanto pianificato inizialmente.
