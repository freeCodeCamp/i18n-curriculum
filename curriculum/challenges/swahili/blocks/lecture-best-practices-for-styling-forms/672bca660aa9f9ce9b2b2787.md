# --interactive--

Tujifunze kuhusu baadhi ya matatizo ya kawaida wakati wa kujaribu kupamba vipengele maalum vya ingizo kama vile ingizo la `datetime-local` na `color`.

Aina hizi maalum za ingizo hutegemea vipengele vya bandia vigumu kuunda vitu kama vile chaguo la tarehe na rangi. Hii inaleta changamoto kubwa katika kupamba ingizo hizi. Changamoto moja ni kwamba, kwa sababu mtindo wa msingi hutegemea kabisa kivinjari, CSS inayofanya chaguo liwezeonekana sawa katika kivinjari kimoja inaweza kutoa matokeo tofauti kabisa katika kivinjari kingine.

Hapa kuna mfano wa ingizo la rangi:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<form>
  <label for="favorite-color">Pick your favorite color:</label>
  <input type="color" id="favorite-color" name="favorite-color">
</form>
```

```css
input {
  padding: 8px 12px;
  margin: 8px 0;
  border-radius: 6px;
  border: 1px solid #ccc;
}

input[type="color"] {
  width: 60px;
  height: 40px;
  padding: 0;
  border: 2px solid #555;
  border-radius: 4px;
  cursor: pointer;
}

```

:::

Changamoto nyingine inaweza kuwa ugumu wa kipengele cha bandia. Fikiria kichaguzi cha tarehe; kuna sehemu nyingi zinazohamia hapa na muundo mgumu wa kipengele cha bandia unaweza kuleta changamoto kubwa katika kutumia mtindo kwa maeneo sahihi.

Hapa kuna mfano wa ingizo la tarehe:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<form>
  <label for="birthdate">Select your birthdate:</label>
  <input type="date" id="birthdate" name="birthdate">
</form>
```

```css
input {
  padding: 8px 12px;
  margin: 8px 0;
  border-radius: 6px;
  border: 1px solid #ccc;
}

input[type="date"] {
  padding: 6px 10px;
  border: 2px solid #555;
  border-radius: 4px;
  font-size: 14px;
  cursor: pointer;
}

input[type="date"]::-webkit-calendar-picker-indicator {
  background-color: #4CAF50;
  color: white;
  border-radius: 4px;
  cursor: pointer;
}

```

:::

Bila shaka, kwa vipengele hivi vigumu, pia unakumbwa na hatari ya kupoteza kitendakazi muhimu kwa bahati mbaya wakati unavipamba kwa mikono. Sio tu kwamba unaweza kupoteza viashiria muhimu kama hali ya makini au kipengee kilichochaguliwa bali pia unaweza kuvunja kichaguzi kabisa.

Kwa sababu hizi msanidi programu wengi hutegemea maktaba za JavaScript au sehemu maalum badala ya kutumia sehemu zilizojengwa ndani ya kivinjari.
