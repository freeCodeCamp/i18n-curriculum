# --explanation--

`Present Perfect Continuous`의 의문문 형태는 `Have/Has + subject + been + verb(-ing)`을 사용해 만듭니다. 이는 과거에 시작되어 아직 진행 중인 동작에 대해 묻는 데 사용됩니다. 예를 들어:

`Have they been testing the new software features?` - 누군가 테스트가 계속되고 있는지 확인합니다.
