# --interactive--

Die `prompt()`-Methode ist ein wichtiger Bestandteil der Interaktion von JavaScript mit dem Benutzer. Sie ist eine der einfachsten Möglichkeiten, um über ein kleines Pop-up-Dialogfeld eine Eingabe vom Benutzer zu erhalten.

Sie wird häufig verwendet, wenn die Webseite eine Information vom Benutzer benötigt, wie zum Beispiel einen Namen oder eine andere Form von Texteingabe.

Was genau macht also die `prompt()`-Methode? Sie öffnet ein Dialogfeld, das den Benutzer um eine Eingabe bittet, und gibt dann den vom Benutzer eingegebenen Text als String zurück.

Die `prompt()`-Methode nimmt zwei Argumente entgegen: Das erste ist die Nachricht, die im Dialogfeld angezeigt wird und typischerweise den Benutzer auffordert, eine Information einzugeben. Das zweite ist ein optionaler Standardwert, der das Eingabefeld initial ausfüllt.

```js
prompt(message, default);
```

Hier ist ein Beispiel, wie sie funktioniert.

**HINWEIS**: Dieses Beispiel enthält Code, den Sie noch nicht gelernt haben. Machen Sie sich keine Sorgen, alles im Code verstehen zu müssen. Es dient nur dazu, zu veranschaulichen, wie die `prompt()`-Methode funktioniert, und sicherzustellen, dass das Prompt nicht sofort beim Laden der Seite erscheint, was als störend empfunden werden kann. Wenn Sie den interaktiven Editor aktiviert haben, können Sie es selbst ausprobieren.

:::interactive_editor

```html
<button id="prompt-btn">Show Prompt</button>
<p id="output"></p>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("prompt-btn");
const output = document.getElementById("output");
btn.addEventListener("click", () => {
  const userName = prompt("What is your name?", "Guest");
  output.textContent = "Hello, " + userName + "!";
});
```

:::

In diesem Beispiel zeigt die `prompt()`-Methode beim Klicken auf den Button ein Dialogfeld mit der Nachricht `What is your name?` und einem Eingabefeld an, das zunächst den Wert `Guest` enthält.

Wenn der Benutzer seinen Namen eingibt und auf „OK“ drückt, speichert die Variable `userName` den eingegebenen Wert. Wenn der Benutzer „Abbrechen“ drückt, wird die Variable `userName` auf `null` gesetzt. `null` bedeutet, dass der Benutzer keine Eingabe gemacht hat. Der Ausgabesatz zeigt dann eine Begrüßungsnachricht mit dem angegebenen Namen oder `null`, falls der Benutzer abgebrochen hat. In zukünftigen Lektionen lernen Sie Techniken, um zu vermeiden, dass `null` angezeigt wird, wenn ein Benutzer das Prompt abbricht.

Beachten Sie, dass die `prompt()`-Methode die Ausführung des Skripts anhält, bis der Benutzer mit dem Dialogfeld interagiert.

Das bedeutet, dass der restliche JavaScript-Code erst ausgeführt wird, wenn der Benutzer entweder eine Eingabe macht und auf „OK“ klickt oder das Prompt abbricht.

Ein weiterer Punkt ist, dass `prompt()` zwar nützlich für schnelle Tests oder kleine Anwendungen ist, aber in modernen, komplexen Webanwendungen aufgrund seiner störenden Wirkung und inkonsistenten Verhaltens in verschiedenen Browsern meist vermieden wird.

Wenn Sie die `prompt()`-Methode verstehen, erhalten Sie eine einfache Möglichkeit, mit Benutzern zu interagieren und Informationen direkt über den Browser abzurufen, auch wenn sie in modernen Webanwendungen nicht weit verbreitet ist.
