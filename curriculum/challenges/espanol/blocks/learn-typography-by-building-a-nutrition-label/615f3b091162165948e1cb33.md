# --description--

Si inspeccionas tu elemento `.label` con las herramientas de desarrollador del navegador, puedes notar que en realidad mide 288 píxeles de ancho en lugar de 270. Esto se debe a que, por defecto, el navegador incluye el borde y el padding al determinar el tamaño de un elemento.

Para solucionar esto, restablece el modelo de caja creando un selector `*` y dándole una propiedad `box-sizing` de `border-box`.

# --hints--

Debes crear un selector `*`.

```js
assert(new __helpers.CSSHelp(document).getStyle('*'));
```

Tu selector `*` debe tener una propiedad `box-sizing` establecida en `border-box`.

```js
assert(new __helpers.CSSHelp(document).getStyle('*')?.boxSizing === 'border-box');
```
