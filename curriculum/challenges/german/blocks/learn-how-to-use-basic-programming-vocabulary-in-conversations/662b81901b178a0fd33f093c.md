## --text--

Wie deklariert man laut Sophie eine Variable in Python?

## --answers--

Sie deklarieren eine Variable, indem Sie sie nach einem Operator benennen.

### --feedback--

Diese Antwort ist falsch, denn die Variable wird nicht nach, sondern vor einem Operator benannt.

---

Variablen werden mit einem Befehl namens `declare` deklariert.

### --feedback--

Python benötigt keinen speziellen Befehl wie `declare`, um Variablen zu erstellen; es verwendet direkt die Zuweisung.

---

Sie verwenden den Variablennamen, gefolgt vom Zuweisungsoperator und einem Wert.

---

Variablen müssen deklariert werden, indem Sie `variable` vor den Namen tippen.

### --feedback--

Python verwendet keine Schlüsselwörter wie `variable`, um Variablen zu deklarieren. Es verwendet einfach den Variablennamen, den Zuweisungsoperator und den Wert.
