# --description--

さて、`shift()` と `pop()` を使って配列の先頭や末尾から要素を削除する方法は学びましたが、配列の途中から要素を削除したい場合や、一度に複数の要素を削除したい場合はどうすればいいでしょうか？そんな時に役立つのが `splice()` です。`splice()` を使うと、配列のどこからでも**連続した任意の数の要素を削除**できます。

`splice()` は最大3つのパラメータを取れますが、ここでは最初の2つに注目します。`splice()` の最初の2つのパラメータは整数で、`splice()` が呼び出されている配列の要素のインデックス（位置）を表します。配列は*ゼロインデックス*なので、配列の最初の要素を示すには `0` を使います。`splice()` の最初のパラメータは削除を開始する配列のインデックスを表し、2つ目のパラメータは削除する要素の数を示します。例えば：

```js
let array = ['today', 'was', 'not', 'so', 'great'];

array.splice(2, 2);
```

ここでは3番目の要素（インデックス2）から2つの要素を削除しています。`array` は `['today', 'was', 'great']` という値になります。

`splice()` は呼び出された配列を変更するだけでなく、削除された要素の値を含む新しい配列も返します：

```js
let array = ['I', 'am', 'feeling', 'really', 'happy'];

let newArray = array.splice(3, 2);
```

`newArray` は `['really', 'happy']` という値を持ちます。
