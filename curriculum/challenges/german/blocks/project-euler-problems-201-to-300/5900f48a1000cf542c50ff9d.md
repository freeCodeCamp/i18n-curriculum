# --description--

Barbara ist Mathematikerin und Basketballspielerin. Sie hat herausgefunden, dass die Wahrscheinlichkeit, einen Punkt zu erzielen, wenn sie aus der Entfernung $x$ wirft, genau ($1 - \frac{x}{q}$) beträgt, wobei $q$ eine reelle Konstante größer als 50 ist.

Während jeder Übung nimmt sie Würfe aus den Entfernungen $x = 1, x = 2, \ldots, x = 50$ und laut ihren Aufzeichnungen hat sie genau eine 2 % Chance, insgesamt genau 20 Punkte zu erzielen.

Finden Sie $q$ und geben Sie Ihre Antwort auf 10 Dezimalstellen gerundet an.
