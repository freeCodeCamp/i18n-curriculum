# --explanation--

`Encantada`は女性話者が「はじめまして」と言うときの別の一般的な表現です。

意味は「魅了された」または「喜んでいる」です。

| 形 | 性別 | 話者 | 意味 |
| :--- | :--- | :--- | :--- |
| `Encantado` | 男性形 (`-o`) | 男性話者 | はじめまして。 |
| `Encantada` | 女性形 (`-a`) | 女性話者 | はじめまして。 |
