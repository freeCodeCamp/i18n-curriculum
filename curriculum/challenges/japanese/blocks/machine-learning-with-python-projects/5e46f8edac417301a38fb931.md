# --instructions--

このチャレンジでは、SMSメッセージを「ham」か「spam」に分類する機械学習モデルを作成します。「ham」メッセージは友人から送られる普通のメッセージです。「spam」メッセージは広告や企業から送られるメッセージです。

メッセージの文字列を引数に取り、リストを返す`predict_message`という関数を作成してください。リストの最初の要素は、「ham」（0）か「spam」（1）の可能性を示す0から1の数値にしてください。リストの2番目の要素は、可能性が高い方の「ham」か「spam」の文字列にしてください。

このチャレンジでは、<a href="http://www.dt.fee.unicamp.br/~tiago/smsspamcollection/" target="_blank" rel="noopener noreferrer nofollow">SMS Spam Collection</a>データセットを使用します。データセットはすでに訓練データとテストデータに分けられています。

最初の2つのセルでライブラリとデータをインポートします。最後のセルでモデルと関数のテストを行います。コードはこれらのセルの間に追加してください。
