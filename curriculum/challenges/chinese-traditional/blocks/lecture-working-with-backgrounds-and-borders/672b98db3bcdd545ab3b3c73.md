# --interactive--

在 CSS 中，有多種方法可以為影像添加邊框，每種方法都提供不同的樣式選項和控制元件層次。

讓我們探索一些最常見且多功能的方法。

為你的影像添加邊框最直接的方法是使用 `border` 屬性。這個屬性是一個簡寫，允許你一次設定邊框的寬度、樣式和顏色。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

這個 CSS 規則會在所有 `img` 元素周圍添加一個寬度為 2 像素、實線的紅色邊框。你可以調整寬度、樣式（例如 `dashed`、`dotted` 或 `double`）和顏色以符合你的設計需求。

如果你需要對邊框的各個邊有更多控制元件，可以使用每個邊的特定邊框屬性：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border-top: 10px solid red;
  border-right: 10px dashed green;
  border-bottom: 10px dotted blue;
  border-left: 10px double purple;
}
```

:::

這允許你為影像的每一側創建獨特的邊框樣式。

另一種產生邊框效果的方法是使用 `outline` 屬性。雖然與邊框相似，但 outline 不會影響元素的尺寸或佈局：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  outline: 3px solid gold;
}
```

:::

這會在影像周圍產生金色輪廓，而不改變其大小或位置。

如果你想在你的邊框上創建圓角，你可以將 `border-radius` 屬性與邊框一起使用：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

請記住，這些技術可以結合並自訂，以創建獨特的邊框效果。方法的選擇取決於你特定的設計需求和你所需的複雜層次。
