# --description--

考虑以下形式的方程：$a^2 + b^2 = N$，$0 ≤ a ≤ b$，$a$、$b$ 和 $N$ 为整数。

对于 $N = 65$ 有两个方案：

$a = 1, b = 8$ 和 $a = 4, b = 7$。

我们将 $S(N)$ 称为所有满足 $a^2 + b^2 = N$，$0 ≤ a ≤ b$，且 $a$、$b$ 和 $N$ 为整数的方案中 $a$ 值的和。

因此 $S(65) = 1 + 4 = 5$。

查找所有仅由形式为 `4k + 1` 且 `4k + 1 < 150` 的素数整除的无平方因子 `N` 的 $\sum S(N)$。
