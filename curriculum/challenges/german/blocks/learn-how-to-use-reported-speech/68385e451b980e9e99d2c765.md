# --explanation--

`True` ist ein Adjektiv, das bedeutet, dass etwas korrekt, echt oder genau ist. Es wird verwendet, um einer Aussage oder Tatsache zuzustimmen. Es unterscheidet sich von `the truth`, das ein Substantiv ist und sich auf die tatsächlichen Fakten oder die Realität von etwas bezieht, oft in einem weiteren oder abstrakteren Sinn. Zum Beispiel:

- `It's true that the server crashed yesterday.` – Das bedeutet, die Aussage über den Absturz ist korrekt.

- `The truth is that we didn't have a backup.` – Das bedeutet, die tatsächliche Tatsache ist, dass keine Sicherungskopie existierte.
