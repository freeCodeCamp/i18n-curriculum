# --description--

$S_m = (x_1, x_2, \ldots, x_m)$를 $x_1 + x_2 + \cdots + x_m = m$을 만족하는 양의 실수 $m$-튜플이라고 할 때, $P_m = x_1 \times {x_2}^2 \times \cdots \times {x_m}^m$가 최대가 되는 $S_m$를 구하세요.

예를 들어, $[P_{10}] = 4112$임을 확인할 수 있습니다 ($[ ]$는 정수 부분 함수입니다).

$2 ≤ m ≤ 15$에 대해 $\sum {[P_m]}$를 구하세요.
