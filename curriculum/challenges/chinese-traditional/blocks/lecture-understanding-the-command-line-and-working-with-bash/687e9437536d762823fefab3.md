# --description--

讓我們來了解你可以使用的命令列捷徑，以加快生產力。

終端機和命令列捷徑可以幫助你簡化工作流程並最大化你的生產力。對於 Linux 和 macOS，兩者皆可追溯至 Unix，許多這些捷徑將會相同。但對於 Windows，則會有一些差異。

第一個快速鍵是向上箭頭鍵。按下這個鍵會讓你一次向後瀏覽你的指令歷史記錄，一次一個指令。接著，向下箭頭鍵可以用來向前瀏覽你的指令歷史記錄。這兩個鍵讓你能快速瀏覽你之前執行過的指令，以找到你需要再次執行的指令。

許多 shell 也會提供自動完成特性，根據你開始輸入的內容建議命令。可以使用 tab 鍵填入建議的其餘部分，快速將你的命令列填滿完整的語法。這些建議的產生方式會因 shell 而異。例如，在 zsh 中，這些建議是根據你最近的命令歷史紀錄。但在 PowerShell 中，則是根據可用的命令、變數和引數（較少考慮最近使用的頻率）。

有時候，一個指令可能會在你的終端機上產生大量輸出。這些額外的雜訊可能會妨礙或干擾你的工作流程。在 Linux 和 macOS 裝置上基於 *nix 的終端機中，你可以按 Control + L 來清除整個螢幕，並從一個全新的乾淨提示字元開始。在 PowerShell 中，你需要執行 `cls` 指令（你可以將它綁定到像 Control + L 這樣的按鍵組合）。

如果你需要中斷一個指令以停止其執行，你可以使用 Control + C 來終止該行程。這將結束指令並產生一個新的提示符，讓你可以繼續處理其他需要進行的工作。對於 PowerShell 使用者來說，Control + C 也用於複製文字－且只有在上下文不模糊時（例如沒有選取任何文字以供複製）才會用來終止行程。

接下來的幾個快速鍵僅在 *nix 基於終端機中可用，且沒有 PowerShell 替代方案。

有時你可能需要多工處理，允許一個處理程序或命令在背景中執行，同時你可以繼續處理另一項工作。按下 Control + Z 會將目前的處理程序放到背景工作，並傳回你到命令列，在那裡你可以繼續你的工作。當你需要將焦點轉回背景工作時，你可以使用 `fg` 將它還原。

最後，可能有些命令你需要連續執行兩次。你可以按上方向鍵向後循環到先前執行過的命令，但你也可以在命令列輸入兩個驚嘆號（`!!`）並按下 Enter 鍵－這將自動執行最後一次執行的命令。

許多終端機提供更多的捷徑，我鼓勵你閱讀你特定設定的文件，以找到最適合你的捷徑。但這些基本的通用捷徑應該能作為提升你生產力和成為終端機高手的絕佳起點。
