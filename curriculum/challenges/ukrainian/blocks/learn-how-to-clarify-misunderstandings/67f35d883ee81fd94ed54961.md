# --explanation--

`On schedule` означає, що справи йдуть згідно з запланованим графіком. Наприклад:

`The project is still on schedule for delivery next week.` — Це означає, що буде завершено вчасно.

`Miscommunication` стосується ситуації, коли інформація не була правильно передана або зрозуміла. Наприклад:

`There was a miscommunication about the meeting location.` — Люди не зрозуміли правильне місце.

`Right away` означає негайно або якнайшвидше. Наприклад:

`I'll send the update right away.` — Ви надішлете це без затримки.
