# --description--

У питанні `¿Quieres que te lo deletree?` кілька елементів працюють разом, щоб висловити пропозицію.

`Quieres que` використовується, щоб запитати, чи хтось хоче щось.

`Te` стосується особи, до якої звертаються. У цьому випадку — Каміли.

`Lo` стосується інформації, тут — е-пошти Матео.

`Deletree` походить від `deletrear`, що означає вимовляти слово по буквах.

Разом це питання використовується, щоб запропонувати Камілі продиктувати е-пошту по буквах.
