# --description--

Tofauti na kufanya kazi na miradi midogo ya vanilla HTML, CSS, na JavaScript, kuanzisha mradi mpya wa React kunajumuisha hatua zaidi ili kuhakikisha kila kitu kinafanya kazi vizuri. Badala ya kujaribu kusanidi mipangilio yote muhimu mwenyewe, kuna zana zitakazofanya hivyo kwa niaba yako.

Moja ya zana maarufu za kusanidi miradi ni Vite. Vite, ambayo inamaanisha "haraka" kwa Kifaransa, ni chombo cha kujenga kinacholenga kutoa uzoefu wa ukuzaji wa haraka kwa miradi ya kisasa ya mtandao. Vite inaweza kutumika na React, pamoja na maktaba na mifumo mingine kama Vue au Svelte. Unaweza hata kuitumia na miradi ya vanilla JavaScript.

Ili kuunda mradi mpya kwa kutumia Vite, utahitaji kutumia mstari wa amri. Ikiwa unatumia kompyuta ya Windows, basi unaweza kutumia Command Prompt au Windows PowerShell. Ikiwa unatumia Mac, basi unaweza kutumia programu ya Terminal.

Iwapo bado huna Node.js imewekwa, ambayo inahitajika ili npm ifanye kazi, hapa kuna kiungo cha tovuti rasmi ya Node.js: https://nodejs.org. Fuata maelekezo ya usakinishaji kisha rudi hapa kuendelea na usakinishaji wa React.

Mara mstari wa amri ukiwa wazi, unaweza kutumia amri ifuatayo:

```bash
npm create vite@latest my-react-app -- --template react
```

Amri hii inaunda mradi mpya wa React uitwao `my-react-app` kwa kutumia templeti ya React ya Vite.

Kisha unaweza kufungua mradi mpya na kuona msimbo wa msingi wa React ambao Vite imetoa kwako.

Kitu kizuri kuhusu Vite ni kwamba itatoa tu mafaili ambayo ni muhimu kabisa kuanza mradi wako wa React.

Ili kuendesha mradi kama ulivyo, utahitaji kusakinisha tegemezi kwa kutumia amri zifuatazo kwenye mstari wa amri:

```bash
cd my-react-app
npm install
```

Amri ya `cd my-react-app` inahakikisha unabadilisha hadi saraka sahihi ambapo mradi wako uko.

Amri ya `npm install` hutumika kusakinisha tegemezi zinazohitajika ili kuendesha mradi wako wa React ipasavyo. Tegemezi ni maktaba au vifurushi ambavyo mradi wako wa React unahitaji ili ufanye kazi vizuri, kama React yenyewe, ReactDOM, au vifurushi vingine vya watu wa tatu vinavyotoa utendaji zaidi.

Kuendesha `npm install` kutasoma faili ya `package.json` katika saraka ya mradi wako na kusakinisha tegemezi zote zilizoorodheshwa hapo, ikikuwezesha kujenga na kuendesha programu yako ya React bila kukosa sehemu muhimu yoyote.

Faili ya `package.json` ni faili muhimu ya usanidi katika miradi inayojumuisha metadata kuhusu mradi wako, ikiwa ni pamoja na jina lake, toleo, na tegemezi. Pia inaeleza skiripti, taarifa za leseni, na mipangilio mingine inayosaidia kusimamia mradi na tegemezi zake.

Mara tegemezi zikasakinishwa, utagundua saraka mpya katika mradi wako iitwayo `node_modules`.

Saraka ya `node_modules` ndiyo mahali ambapo vifurushi vyote na maktaba zinazohitajika na mradi wako huhifadhiwa. Saraka hii ina msimbo halisi wa tegemezi zilizoorodheshwa katika faili yako ya `package.json`, ikiwa ni pamoja na tegemezi za moja kwa moja za mradi wako na pia tegemezi za tegemezi hizo.

Ili kuendesha mradi wako, tumia amri ya `npm run dev` kisha fungua kichupo kipya cha kivinjari kwa `http://localhost:5173/`.

Utapata kuona templeti ya kuanzia ambayo Vite imetoa kwako.

Iwapo unahitaji kusitisha seva ya ndani, bonyeza `CTRL + C` kwenye mstari wa amri.

Ili kuona msimbo wa templeti hii ya kuanzia, unaweza kuingia ndani ya mradi wako katika saraka ya `src` na utapata faili ya `App.jsx`.

Kutoka hapo unaweza kuanza kubadilisha faili, kuhifadhi mabadiliko yako na kuona mabadiliko mapya yanaonyeshwa kwenye kivinjari.

Na kwa hayo, uko tayari kuanza kujenga programu yako ya React!
