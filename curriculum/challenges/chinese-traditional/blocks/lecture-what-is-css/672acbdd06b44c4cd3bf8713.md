# --interactive--

當你開始使用 HTML 和 CSS 時，你會注意到在你撰寫任何 CSS 之前，有些樣式已經套用到你的網頁上。這些樣式稱為「預設瀏覽器樣式」或「使用者代理樣式」。

這些預設瀏覽器樣式提供基本格式設定，以確保 HTML 元素在所有瀏覽器中都能以可讀的方式顯示。

然而，這些樣式在不同瀏覽器之間可能會略有差異，因此在設計網站的一致外觀時，了解它們非常重要。

讓我們來看看套用於各種 HTML 元素的一些常見預設瀏覽器樣式。

標題是最常使用的 HTML 元素之一，並且預設會以不同的大小和字重來呈現。

例如，`h1`（最高層級的標題）通常比較低層級的標題如 `h2`、`h3` 等字型大小更大且為粗體。

這些標題頁籤有助於定義你內容的階層體系和結構。

以下範例顯示從 `h1` 到 `h6` 的所有六個標題元素。

:::interactive_editor

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```

:::

另一個具有預設樣式的元素是 `hr` 元素，它會創建一條水平線，通常用於視覺上分隔內容區段。瀏覽器通常會對此元素套用簡單的線條樣式。

為了更好地理解它，讓我們來看這個程式碼範例：

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<hr>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

水平線條會以細線顯示，並在文字上下方留有間距，以區分內容區段。

接下來，我們將查看 `blockquote` 元素。

區塊引言用來顯示一段引用自其他來源的文字。瀏覽器通常會添加縮排，有時也會將文字斜體化。

縮排有助於將區塊引用與其餘文字區分開來，使內容清楚顯示是引用自其他來源。

讓我們來看看這個程式碼範例：

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<blockquote>I think, therefore I am. (Rene Descartes)</blockquote>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

在上述範例中，`blockquote` 元素會在文字上產生縮排，該縮排是由瀏覽器的預設樣式套用。

另一個具有預設樣式的元素是錨點元素（`<a>`），它以預設的藍色和底線樣式來使其可識別為聯結。

讓我們來看看以下的 HTML 範例：

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<a href="https://freecodecamp.org/">Visit the freeCodeCamp learn page</a>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

上述程式碼有四個段落元素，中間有一個錨點元素。錨點元素以藍色並帶有底線的樣式呈現，讓使用者知道可以前往 freeCodeCamp learn 頁面。

最後，我們將查看無序和有序列表元素。

瀏覽器會為列表添加基本格式，包括縮排和項目符號或數字，這取決於你使用的是無序列表（`ul`）還是有序列表（`ol`）。

讓我們來看一個範例：

:::interactive_editor

```html
<p>Sample Paragraph</p>
<ol>
  <li>item 1</li>
  <li>item 2</li>
  <li>item 3</li>
</ol>
<ul>
  <li>item</li>
  <li>another item</li>
  <li>yet another item</li>
</ul>
<p>Ending Paragraph</p>
```

:::

在上述 `code` 範例中，我們使用了無序列表和有序列表。這兩種列表預設都會稍微向右縮排。

在所有這些範例中，你看到瀏覽器如何套用預設間距、不同的字型大小和字重到這些 HTML 元素。

在後續課程中，你將學習如何使用 CSS reset 重設這些預設樣式中的部分。
