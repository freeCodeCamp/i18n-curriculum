# --explanation--

`data breach`は、許可されていない人が機密情報にアクセスしたときに起こります。例えば：

- `A data breach can happen if hackers get access to private information.` - サイバー犯罪者が情報を盗むと、それはデータ漏えいと呼ばれます。

- `Companies must act quickly if there is a data breach.` - データ漏えいが起きた場合、企業は迅速に対応して修正する必要があります。

ジェシカは、このリスクを防ぐためにデータを暗号化し、暗号化キーを安全に保管することが不可欠だと言っています。
