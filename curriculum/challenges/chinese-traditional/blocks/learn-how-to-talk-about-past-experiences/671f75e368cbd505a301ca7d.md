# --explanation--

`Me too` 用來表達同意或表示相同的述語也適用於說話者。這是一種簡潔且非正式的方式來回應共同的經驗或感受。 例如： 

Person 1: `I love Python programming.` 

第二個人：`Me too.` － 意思是他們也喜歡用 Python 寫程式。

在這個背景關係中，Maria 可以說 `Yeah, me too`，以快速讓 Brian 知道她也參加了 PyCon 上非常有趣的場次。
