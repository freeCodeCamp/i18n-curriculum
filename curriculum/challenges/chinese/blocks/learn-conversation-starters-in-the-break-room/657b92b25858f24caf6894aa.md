# --description--

表达式 `only if` 用于设置某事发生的条件。它就像是在说“如果那件事发生，这件事才会发生”。这是一种使一个动作依赖于另一个动作的约定方式。

例如，你的朋友可能会说，`I'll go out only if it doesn't rain`。这意味着他们会出去，但天气必须好——不能下雨。
