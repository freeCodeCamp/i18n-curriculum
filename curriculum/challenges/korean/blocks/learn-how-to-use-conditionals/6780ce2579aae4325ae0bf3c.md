# --explanation--

`Critical`은 매우 중요하거나 필수적인 것을 의미하며, `critical data`는 팀이 효과적으로 작업하기 위해 의존하는 중요한 정보를 가리킵니다. 이를 잃으면 지연이나 문제가 발생합니다. 예를 들어:

- `The backup includes all critical data in case of a failure.` - 저장된 복사본에는 문제를 복구하는 데 필수적인 정보가 포함되어 있습니다.

- `You must encrypt critical data to ensure it's secure.` - 중요한 정보를 암호화로 보호하는 것은 보안을 위해 필요합니다.
