# --explanation--

`that`の後に`I think`を使うのは、このような文では任意です。`I think that it's vital we make these updates.`と言うのも正しいです。ただし、日常会話では`that`を省略して、よりスムーズで素早い文にすることがよくあります。例えば：

`I think that we need more time to finish the project.` – この構造は正しいですが、ほとんどの人は`I think we need more time…`と言うでしょう。
