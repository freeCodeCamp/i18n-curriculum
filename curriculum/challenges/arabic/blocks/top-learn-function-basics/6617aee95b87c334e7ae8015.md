# --description--

إذا تم استدعاء دالة ولم تُقدَّم معلمة، فسيُعيَّن للمعلمة القيمة `undefined`. قد يؤدي هذا إلى نتائج غير متوقعة إذا لم تُصمم الدالة للتعامل مع هذه الحالة.

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal()); // Output: undefined is my favorite animal!
```

في المثال أعلاه، تُستدعى الدالة `favoriteAnimal` بدون معلمة. تُعيَّن المعلمة `animal` القيمة `undefined`، والتي تُدمج بعد ذلك مع السلسلة النصية `" is my favorite animal!"`.

إذا لم تُقدَّم قيمة، يمكنك استخدام المعلمات الافتراضية. تتيح المعلمات الافتراضية تعيين قيمة افتراضية للمعلمة إذا لم تُقدَّم قيمة عند استدعاء الدالة.

```js
function favoriteAnimal(animal = 'Giant Panda') {
  return animal + " is my favorite animal!"
}
```

في المثال أعلاه، تم تعيين القيمة الافتراضية للمعلمة `animal` إلى `'Giant Panda'`. إذا استُدعيت الدالة `favoriteAnimal` بدون معلمة، فستُعيَّن للمعلمة `animal` القيمة `'Giant Panda'`.

---

تضمن أن تُرجع الدالة `null` للدلالة على عدم تمرير معلمة.

---

تسمح للدالة بالتنفيذ بقيمة محددة مسبقًا، متجنبةً المشاكل المتعلقة بـ `undefined`.
