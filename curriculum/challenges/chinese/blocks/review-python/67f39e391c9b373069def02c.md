## 什么是 Python？

- **简介**：Python 是一种通用的编程语言，以其简洁和易用著称。Python 被广泛应用于数据科学和机器学习、网页开发、脚本编写与自动化、嵌入式系统和物联网等多个领域。
- **常见应用场景**：Python 用于数据科学、机器学习、网页开发、安全性、自动化以及像 Raspberry Pi 和兼容 MicroPython 的微型计算机等设备。

## 在你的局部环境中使用 Python

- **安装**：在 Windows、Mac 和 Linux 上安装 Python 的最佳方式是从官方 Python 网站下载安装程序（`https://www.python.org/`）。

## 变量

- **声明变量**：声明变量时，先写变量名，后跟赋值操作符（`=`），然后是值。这个值可以是数字、字串、布尔值等。以下是一些示例：

```py
name = 'John Doe'
age = 25
```

- **变量命名规范**：变量命名应遵循以下规范：

  - 变量名只能以字母或下划线（_）开头，不能以数字开头。
  - 变量名只能包含字母数字字符（a-z、A-Z、0-9）和下划线（_）。
  - 变量名区分大小写——`age`、`Age` 和 `AGE` 都被视为不同的变量。
  - 变量名不能是 Python 的保留关键字，如 `if`、`class` 或 `def`。
  - 多词变量名用下划线分隔。例如，`snake_case`。

## 评论

- **单行评论**：这种类型的评论适合用于在代码中留下简短的备注。

```py
# This is a single line comment
```

- **多行字串**：这种类型的字串可以用来留下更长的备注，或者注释掉代码的某些部分。

```py
"""
This is a multi-line string.
Here is some code commented out.

name = 'John Doe'
age = 25
"""
```

- **`print()` 函数**：要将数据打印到控制台，你可以像这样使用 `print()` 函数：

```py
print('Hello world!') # Hello world!
```

## Python 中的常见数据类型

- **简介**：Python 是一种动态类型语言，类似于 JavaScript，这意味着你不需要显式声明变量的类型。语言会根据你赋值给变量的内容自动识别变量的类型。
- **整数**：没有小数的整数：

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **浮点数**：带有小数的数字：

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **字串**：用引号括起来的一系列字符：

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **布尔**：表示 `True` 或 `False` 的值：

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **集合**：无序且唯一元素的集合：

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **字典**：由键值对组成的集合，用花括弧括起来：

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **元组**：不可变的有序集合，用括号括起来：

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **范围**：一系列数字，常用于循环：

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **列表**：支持不同数据类型的有序元素集合：

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**：表示没有值的特殊值：

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## 不可变类型和可变类型

- **不可变类型**：这些类型一旦声明就不能改变，尽管你可以将它们的变量指向新的内容，这称为重新赋值。它们包括整数、浮点数、复数、布尔、字串、元组、范围和 `None`。
- **可变类型**：这些类型一旦声明后可以改变。你可以添加、删除或更新它们的项。它们包括集合类型，如列表、集合和字典。
- **`type()` 函数**：要查看变量的类型，你可以像这样使用 `type()` 函数：

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **`isinstance()` 函数**：用于查看变量是否匹配特定的数据类型：

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## 处理字串

- **定义**：正如你从 JavaScript 中回忆的，字串是不可变的，这意味着创建后你无法更改它们。在 Python 中，你可以使用单引号或双引号。建议选择一种规则并坚持使用：

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **访问字串中的字符**：你可以使用方括弧表示法访问字串中的字符，方法如下：

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **转义字串**：如果你的字串包含引用符，可以使用反斜线（`\`）进行转义，示例如下：

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **字串串接**：要串接字串，你可以使用 `+` 操作符，示例如下：

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

另一种串接字串的方法是使用 `+=` 操作符。它用于在同一步执行串接和赋值，示例如下：

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**：这是格式化字面值字串的缩写。它允许你处理插值，同时用简洁且易读的语法进行串接：

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **字串切片**：这指的是你可以提取字串的部分内容。基本语法如下：

```py
str[start:stop:step]
```

起始位置表示提取开始的索引。结束位置是切片结束的索引，该位置不包含在内。步长位置表示切片时的递增间隔。以下是一些示例：

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **获取字串长度**：使用 `len()` 函数返回字串中的字符数：

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## 使用 `in` 操作符

- **`in` 操作符**：它返回一个布尔值，用于指定该字符或字符是否存在于字串中：

```py
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

## 常用字串方法

- **`upper()`**：此字串方法返回一个新字串，所有字符都转换为大写：

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`lower()`**：此字串方法返回一个新字串，所有字符都转换为小写：

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`strip()`**：此字串方法返回字串的拷贝，去除指定的前导和尾随字符（如果未传入参数，则去除前导和尾随的空白字符）。

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**：此方法返回一个新字串，将所有旧字串的出现替换为新字串。

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**：此方法用于使用指定的分隔符将字串拆分成列表。分隔符是指定拆分位置的字串。

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**：此方法用于将可迭代对象的元素用分隔符连接成一个字串。可迭代对象是可以像列表、字串或元组那样循环访问的集合。

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`startswith(prefix)`**：此字串方法返回布尔值，指示字串是否以指定的前缀开始：

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`endswith(suffix)`**：此字串方法返回布尔值，指示字串是否以指定的后缀结束：

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`find()`**：此字串方法返回子字串首次出现的索引。如果未找到，则返回 `-1`：

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`count(substring)`**：此字串方法统计子字串在字串中出现的次数：

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`capitalize()`**：此字串方法返回一个新字串，首字符大写，其余字符小写：

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`isupper()`**：此字串方法如果字串中所有字母均为大写，则返回 `True`，否则返回 `False`：

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`islower()`**：此字串方法如果字串中所有字母均为小写，则返回 `True`，否则返回 `False`：

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`title()`**：此字串方法返回一个新字串，每个单词的首字母均大写：

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`maketrans()`**：此字串方法用于创建一个 1 对 1 字符映射表，常与 `translate()` 方法配合使用，后者将该映射表应用于字串并返回翻译结果。

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## 整数和浮点数常用操作

- **基本数学操作**：在 Python 中，你可以对整数和浮点数进行基本的数学操作，包括加法、减法、乘法和除法：

```py
int_1 = 56
int_2 = 12
float_1 = 5.4
float_2 = 12.0

# Addition

print('Integer Addition:', int_1 + int_2) # Integer Addition: 68
print('Float Addition:', float_1 + float_2) # Float Addition: 17.4

# Subtraction

print('Int Subtraction:', int_1 - int_2) # Int Subtraction: 44
print('Float Subtraction:',  float_2 - float_1) # Float Subtraction: 6.6

# Multiplication

print('Int Multiplication:', int_1 * int_2) # Int Multiplication: 672
```

```py

print(int_1 + float_1) # 61.4
```

- **取模操作符（`%`）**：这个操作符返回一个数除以另一个数的余数：

- **向下取整除法（`//`）**：这个操作符用于除两个数，并将结果向下取整到最接近的整数：

- **幂运算操作符（`**`）**：这个操作符用于将一个数提升到另一个数的幂：

- **`float()` 函数**：你可以使用这个函数将整数转换为浮点数。

```py

print(float(num)) # 4.0
```

- **`int()` 函数**：你可以使用这个函数将浮点数转换为整数。

```py
num = 4.0

print(int(num)) # 4
```

- **`round()` 函数**：这个函数用于将数字四舍五入到最接近的整数：

```py
num_1 = 3.4
num_2 = 7.7

print(round(num_1)) # 3
print(round(num_2)) # 8
```

- **`abs()` 函数**：这个函数用于返回数字的绝对值：

```py
num = -13

print(abs(num)) # 13
```

- **`bin()` 函数**：这个函数用于将整数转换为其二进制表示的字串：

- **`oct()` 函数**：这个函数用于将整数转换为其八进制表示的字串：


- **`hex()` 函数**：这个函数用于将整数转换为其十六进制表示的字串：

```py
num = 56

print(hex(num))  # 0x38
```

- **`pow()` 函数**：这个函数用于将一个数提升到另一个数的幂：

```py
result = pow(2, 3)
```

## 使用函数

- **定义**：函数是可重用的代码片段，它们接受输入（参数）并返回输出。要调用函数，你需要引用函数名，后跟一对括号：

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call
print(result) # 7
```

如果函数没有显式返回值，那么默认返回值是 `None`：

```py
def greet():
    print('hello') 

result = greet() # hello
print(result) # None
```

你也可以像这样为参数提供默认值：

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

如果调用函数时参数数量不正确，你将得到一个 `TypeError`：

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

## Python 中的作用域

- **局部作用域**：指在函数或类内部声明的变量只能在该函数或类内部访问。


- **闭包作用域**：指嵌套在另一个函数内部的函数可以访问其外层函数的变量。

```py

    inner_func()

print(outer_func()) # Hello there!
```

- **全局作用域**：指在任何函数或类外部声明的变量，可以在程序的任何地方访问。

```py
tax = 0.70 

def get_total(subtotal):
    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **内置作用域**：Python 中为预定义的函数、模块、关键字和对象保留的名称。

```py
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```

## 使用 `if`、`elif` 和 `else` 语句

- **`if` 语句**：这些是用来判断某事是否为真的条件。如果条件计算结果为 `True`，那么该代码块将被运行。

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **`elif` 子句**：这些条件出现在 `if` 语句之后。如果 `elif` 条件计算结果为 `True`，那么该代码块将被运行。

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **`else` 子句**：如果没有其他条件计算结果为 `True`，则会运行此代码块。

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

你也可以像这样使用嵌套的 `if` 语句：

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## 真值和假值

- **定义**：在 Python 中，每个值都有一个固有的布尔值，或者说在逻辑上下文中它应该被视为 `True` 还是 `False` 的内置含义。许多值被认为是真值，也就是说，它们在逻辑上下文中计算结果为 `True`。其他值是假值，意味着它们计算结果为 `False`。以下是假值的一些示例：

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

其他值，比如非零数字和非空字串，是真值。

## 使用 `bool()` 函数

- **定义**：如果你想查看一个值是真值还是假值，可以使用内置的 `bool()` 函数。它会显式地将一个值转换为其布尔等价物，并为真值返回 `True`，为假值返回 `False`。以下是一些示例：

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## 布尔操作符和短路求值

- **定义**：这些是特殊的操作符，允许你组合多个表达式，从而在代码中创建更复杂的决策逻辑。Python 中有三个布尔操作符：`and`、`or` 和 `not`。
- **`and` 操作符**：该操作符接受两个操作对象，如果第一个操作对象为假值，则返回第一个操作对象，否则返回第二个操作对象。两个操作对象都必须为真值，表达式才会得到真值。

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

你也可以在条件式中这样使用 `and` 操作符：

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **`or` 操作符**：该操作符返回第一个操作对象（如果它为真值），否则返回第二个操作对象。如果至少有一个操作对象为真值，`or` 表达式就会得到真值。示例如下：

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

就像 `and` 操作符一样，你可以在条件式中这样使用 `or` 操作符：

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **短路求值**：`and` 和 `or` 操作符被称为短路操作符。短路求值意味着 Python 从左到右检查值，一旦确定最终结果就停止计算。
- **`not` 操作符**：该操作符接受一个操作对象，并取反其布尔值。它将真值转换为 `False`，将假值转换为 `True`。与之前介绍的操作符不同，`not` 总是返回 `True` 或 `False`。以下是一些示例：

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

下面是 `not` 操作符在条件式中的示例：

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```

## Python 列表

- **简介**：在 Python 中，列表数据类型是一个有序的元素序列，元素可以是字串、数字甚至其他列表。列表是可变的，并且索引从零开始。

```python
cities = ['Los Angeles', 'London', 'Tokyo']
```

- **访问列表中的元素**：要访问 `cities` 列表中的元素，可以通过引用序列中的索引号来实现：

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # Los Angeles
```

- **使用负索引访问元素**：要访问任何列表的最后一个元素，可以使用 `-1` 作为索引号：

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # Tokyo
```

- 负索引用于从列表末尾开始访问元素，而不是从索引 `0` 的开头开始。

- **使用 `list()` 构造函数创建列表**：列表也可以使用 `list()` 构造函数创建。`list()` 构造函数用于将可迭代对象转换为列表：

```python
developer = 'Jessica'

print(list(developer)) 
# Result: ['J', 'e', 's', 's', 'i', 'c', 'a']
```

- **查找列表长度**：你可以使用 `len()` 函数获取列表的长度：

```python
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

- **列表的可变性**：列表是可变的，这意味着只要传入有效的索引号，你可以更新列表中的任何元素。要更新特定索引处的列表元素，可以为该索引赋予新值：

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

- **索引超出范围误差**：如果传入的索引（无论是正数还是负数）超出列表的边界，则会收到 `IndexError`：

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
"""
```

- **从列表中移除元素**：可以使用 `del` 关键字从列表中移除元素：

```python
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

- **检查元素是否存在于列表中**：可以使用 `in` 关键字检查元素是否存在于列表中：

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **嵌套列表**：列表可以嵌套在其他列表中：

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

- 要访问嵌套列表，需要使用索引 `2`，因为列表的索引从零开始。

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

- 要进一步访问该嵌套列表中的第二种语言，需要使用索引 `1`：

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # Rust
```

- **从列表中解包值**：从列表中解包值是一种将列表中的值赋给新变量的技术。下面是一个示例，将 `developer` 列表解包到名为 `name`、`age` 和 `job` 的新变量中：

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer
```

- 如果赋值操作符左侧的变量数量与列表中的项总数不匹配，则会收到 `ValueError`。

- **收集列表中剩余的项**：要收集列表中剩余的元素，可以使用星号（`*`）操作符，如下所示：

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer
```

- **切片列表**：切片是使用切片操作符 `:` 访问列表部分内容的概念。要切片一个从索引 `1` 开始，到索引 `3` 之前结束的列表，可以使用以下语法：

```python
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie']
desserts[1:3] # ['Cookies', 'Ice Cream']
```

- **步长间隔**：还可以指定步长间隔，用于确定索引之间的增量。如果你想使用切片提取仅包含偶数的列表，下面是一个示例：

```python
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

## 列表方法

- **append()**：用于将一项添加到列表的末尾。下面是使用 `append()` 方法将数字 `6` 添加到这个 `numbers` 列表的示例：

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **追加列表**：`append()` 方法也可以用来将一个列表追加到另一个列表的末尾：

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

- **extend()**：用于将多个项添加到列表的末尾。下面是将数字 `6`、`8` 和 `10` 添加到 `numbers` 列表末尾的示例：

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

- **insert()**：用于在列表的特定索引处插入一项。下面是使用 `insert()` 方法的示例：

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

- **remove()**：用于从列表中移除一项。`remove()` 方法只会移除列表中第一次出现的该项：

```py
numbers = [1, 2, 3, 4, 5, 5, 5]
numbers.remove(5)

print(numbers) # [1, 2, 3, 4, 5, 5]

```

- **pop()**：用于移除列表中的特定项并返回它：

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

- 如果你没有为 `pop` 方法指定元素，那么默认移除最后一个元素。

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

- **clear()**：用于移除列表中的所有项：

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

- **sort()**：`sort()` 方法用于就地排序元素。下面是对一个随机 `numbers` 列表进行就地排序的示例：

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

- **sorted()**：一个内置函数，返回一个新的排序列表，而不是修改原列表：

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
print(numbers) # [19, 2, 35, 1, 67, 41]
```

- **reverse()**：用于反转列表中元素的顺序：

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **index()**：用于查找列表中元素第一次出现的位置索引：

```python
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

- 如果使用 `index()` 方法找不到该元素，结果将是 `ValueError`。

## Python 中的元组

- **定义**：元组是 Python 中用于创建有序值序列的数据类型。元组可以包含混合的数据类型：

```py
developer = ('Alice', 34, 'Rust Developer')
```

- 元组是不可变的，这意味着元组中的元素一旦创建就不能更改。如果你尝试更新元组中的某一项，你将得到一个 `TypeError`：

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **访问元组中的元素**：要访问元组中的元素，使用方括弧表示法和索引号：

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

- 可以使用负索引从元组末尾开始访问元素：

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- 如果你尝试传入一个超过或等于元组长度的索引号，你将收到一个 `IndexError`：

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- 元组也可以使用 `tuple()` 构造函数创建。在构造函数中，你可以传入不同的可迭代对象，比如字串、列表甚至其他元组。

```python
developer = 'Jessica'

print(tuple(developer)) 
# Result: ('J', 'e', 's', 's', 'i', 'c', 'a')
```

- **检查元组中的项**：要查看某项是否在元组中，可以使用 `in` 关键字，如下所示：

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **解包元组**：可以这样从元组中解包项：

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

- 如果你需要收集元组中剩余的元素，可以使用星号（`*`）操作符，如下所示：

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer
```

- **切片元组**：切片可以用来提取元组的一部分。例如，项 `pie` 和 `cookies` 可以被切片成一个单独的元组：

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

- **从元组中移除项**：从元组中移除项会引起 `TypeError`，因为元组是不可变的：

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

- **什么时候使用元组而不是列表？**：如果你需要一个动态的元素集合，可以添加、移除和更新元素，那么你应该使用列表。如果你知道你正在处理一个固定且不可变的数据集合，那么你应该使用元组。

## 常用元组方法

- **count()**：用于确定某项在元组中出现的次数。例如，你可以查看语言 `'Rust'` 在元组中出现了多少次：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

- 如果在 `count()` 方法中指定的项在元组中根本不存在，那么返回值将是 `0`：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

- 如果没有向 `count()` 方法传递任何参数，那么 Python 会返回一个 `TypeError`。

- **index()**：用于查找元组中某项所在的索引。下面是使用 `index()` 方法查找语言 `'Java'` 索引的示例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

- 如果找不到指定的项，Python 会返回一个 `ValueError`。

- 你可以向 `index()` 方法传递一个可选的起始索引，以指定从元组的哪个位置开始查找该项：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

- 你还可以向 `index()` 方法传递一个可选的结束索引，以指定在元组中查找该项时停止的位置：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

- **sorted()**：用于对任何可迭代对象中的元素进行排序，并返回一个新的排序列表。下面是使用 `sorted()` 函数创建一个新数字列表的示例：

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

- **修改排序行为**：你可以使用可选的 `reverse` 和 `key` 参数自定义可迭代对象的排序行为。下面是使用 `key` 参数按长度对元组中的项进行排序的示例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

- 你可以使用 `reverse` 参数创建一个按相反顺序排列的新值列表，如下所示：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

- **定义**：循环用于重复执行一段代码块指定的次数。

- **`for` 循环**：用于迭代一个序列（如列表、元组或字串），并对序列中的每一项执行一段代码。下面是使用 `for` 循环迭代列表并打印每种语言到控制台的示例：

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:

Rust
Java
Python
C++
```

```py
for char in 'code':

Result 

c
o
d
e
```

```py
categories = ['Fruit', 'Vegetable']
```

```py
secret_number = 3
guess = 0
```

- **`break` 和 `continue` 语句**：用于循环中修改循环的执行。

```py
developer_names = ['Jess', 'Naomi', 'Tom']
```

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']
```

## 范围及其在循环中的使用

- **`range()` 函数**：用于生成整数序列。

```py
range(start, stop, step)
```

- 必需的 `stop` 参数是一个整数（不包含该整数），表示生成的数字序列的终点。下面是使用 `range()` 函数的示例：

```py
for num in range(3):
    print(num)
```

- 如果未指定 `start` 参数，则默认值为 `0`。默认情况下，整数序列将以 `1` 递增。你可以使用可选的 `step` 参数来更改默认的递增值。下面是生成从 2 到但不包括 11（即包含 10）的偶数序列的示例：

```py
for num in range(2, 11, 2):
    print(num)
```

- 如果你不为 `range()` 函数提供任何参数，则会得到一个 `TypeError`。

- `range()` 函数只接受整数作为参数，不接受浮点数。使用浮点数也会导致 `TypeError`：

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- 你可以为 `step` 参数使用负整数，以生成递减顺序的整数序列：

```py
for num in range(40, 0, -10):
    print(num)
```

- `range()` 函数也可以与 `list` 构造函数一起使用来创建整数列表。`list` 构造函数用于将可迭代对象转换为列表。下面是生成包含 2 到 10 之间偶数的列表的示例：

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

## Python 中的 `enumerate()` 和 `zip()` 函数

- **`enumerate()`**：用于迭代一个序列，并跟踪该序列中每个项的索引。`enumerate()` 函数接受一个可迭代对象作为参数，并返回一个 `enumerate` 对象，该对象包含可迭代对象中每个项的索引和值。

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']
```

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']
```

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']

ID: 3
Name: Tom
ID: 4

```

## Python 中的列表推导式

- **定义**：列表推导式允许你通过在方括弧内直接结合循环和条件，在一行内创建一个新列表。这使代码更简洁，通常也更易读。

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

## 可迭代函数

- **`filter()`**：用于根据条件过滤可迭代对象中的元素。它返回一个只包含满足条件元素的迭代器。下面是一个创建仅包含长度超过四个字符单词的新列表的示例：

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**：用于对可迭代对象中的每一项应用一个函数，并返回一个包含结果的新可迭代对象。下面是使用 `map()` 函数将摄氏温度列表转换为华氏温度的示例：

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**：用于获取列表或元组等可迭代对象的总和。下面是使用 `sum()` 函数的示例：

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- 你还可以传入一个可选的 `start` 参数，用于设置求和的初始值。下面是使用 `start` 参数作为位置参数的更新示例：

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- 你也可以选择像下面这样将 `start` 参数作为关键字参数使用：

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## Lambda 函数

- lambda 函数通常作为另一个函数的参数使用。下面是一个 lambda 函数的示例：

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
```

## 字典

- **字典**：字典是内置的数据结构，用于保存密钥-值点对的集合。密钥必须是不可变的数据类型。以下是 Python 字典的一般语法：

```python
dictionary = {
    key1: value1,
    key2: value2
}
```

- **`dict()` 构造函数**：`dict()` 构造函数是构建字典的另一种方式。你将一个元组列表作为参数传递给 `dict()` 构造函数。这些元组的第一个元素是密钥，第二个元素是值。

```python
pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
```

- **方括弧表示法**：要访问密钥-值点对的值，可以使用称为方括弧表示法的语法。

```python
dictionary[key]
```

## 常用字典方法

- **`get()` 方法**：`get()` 方法检索与密钥关联的值。它类似于方括弧表示法，但允许你设置默认值，从而防止密钥不存在时引起误差。

```python
dictionary.get(key, default)
```

- **`keys()` 和 `values()` 方法**：`keys()` 和 `values()` 方法分别返回包含字典中所有密钥和值的视图对象。视图对象是一种查看字典内容的方式，而不需要创建数据的单独拷贝。

```python
pizza = {
    'name': 'Margherita Pizza',
    'price': 8.9,
    'calories_per_slice': 250
}

pizza.keys()
# dict_keys(['name', 'price', 'calories_per_slice'])

pizza.values()
# dict_values(['Margherita Pizza', 8.9, 250])
```

- **`items()` 方法**：`items()` 方法返回包含字典中所有点对（密钥和值）的视图对象，包括密钥和值。

```python
pizza.items()
# dict_items([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250)])
```

- **`clear()` 方法**：`clear()` 方法移除字典中的所有点对。

```python
pizza.clear()
```

- **`pop()` 方法**：`pop()` 方法移除第一个参数指定的密钥对应的点对，并返回其值。如果密钥不存在，则返回第二个参数指定的默认值。如果密钥不存在且未指定默认值，则引起 `KeyError`。

```python
pizza.pop('price', 10)
pizza.pop('total_price') # KeyError
```

- **`popitem()` 方法**：在 Python 3.7 及以上版本中，`popitem()` 方法移除最后插入的项。

```python
pizza.popitem()
```

- **`update()` 方法**：`update()` 方法用另一个字典的点对更新当前字典的点对。如果有相同的密钥，则其值会被覆盖。新的密钥将作为新的点对添加到字典中。

```python
pizza.update({ 'price': 15, 'total_time': 25 })
```

## 遍历字典

- **遍历值**：如果你需要遍历字典中的值，可以使用 `for` 循环配合 `values()` 来获取字典的所有值。

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,
    'Headphones': 70,
}

for price in products.values():
    print(price)
```

输出：

```md
990
600
250
70
```

- **遍历键**：如果你需要遍历上面 `products` 字典的键，可以直接写 `products.keys()` 或 `products`。

```python
for product in products.keys():
    print(product)
    
# Or

for product in products:
    print(product)
```

输出：

```md
Laptop
Smartphone
Tablet
Headphones
```

- **遍历键值对**：如果你需要同时遍历键及其对应的值，可以遍历 `products.items()`。你会得到包含键和值的单个元组。

```python
for product in products.items():

```

输出：

```md

('Tablet', 250)
```

要将键和值分别保存到不同的循环变量中，需要用逗号分隔它们。第一个变量保存键，第二个保存值。

```python
for product, price in products.items():
    print(product, price)
```

输出：

```md
Laptop 990
Smartphone 600
Tablet 250
Headphones 70
```

- **`enumerate()` 函数**：如果你需要在遍历字典时跟踪计数器，可以调用 `enumerate()` 函数。该函数返回一个 `enumerate` 对象，会为每一项分配一个整数，类似计数器。计数器可以从任意数字开始，但默认从 0 开始。

将索引和值分别赋值给不同的循环变量是使用 `enumerate()` 的常见方式。例如，使用 `products.items()`，你可以在索引之外获取整个键值对：

```python
for index, product in enumerate(products.items()):
    print(index, product)
```

输出：

```md

1 ('Smartphone', 600)
2 ('Tablet', 250)
3 ('Headphones', 70)
```

要自定义计数的初始值，可以向 `enumerate()` 传入第二个参数。例如，这里我们从 1 开始计数。

```python
for index, product in enumerate(products.items(), 1):
    print(index, product)
```

输出：

```md
1 ('Laptop', 990)
2 ('Smartphone', 600)
3 ('Tablet', 250)
4 ('Headphones', 70)
```

## 集合

- **集合**：集合是 Python 内置的数据结构，不允许有重复值。集合是可变且无序的，这意味着它们的元素没有存储在特定顺序中，因此你不能使用索引或密钥来访问它们。此外，集合只能包含不可变数据类型的值，比如数字、字串和元组。

- **定义集合**：要定义一个集合，需要将其元素写在花括弧内，并用逗号分隔。

```python
my_set = {1, 2, 3, 4, 5}
```

- **定义空集合**：如果你需要定义一个空集合，必须使用 `set()` 函数。仅写空花括弧会自动创建一个字典。

```python
set() # Set
{}    # Dictionary
```

## 常用集合方法

- **`add()` 方法**：你可以使用 `add()` 方法向集合添加元素，传入新元素作为参数。

```python
my_set.add(6)
```

- **`remove()` 和 `discard()` 方法**：要从集合中移除元素，你可以使用 `remove()` 方法或 `discard()` 方法，传入你想移除的元素作为参数。`remove()` 方法如果找不到元素会引起 `KeyError`，而 `discard()` 方法则不会。

```python
my_set.remove(4)
my_set.discard(4)
```

- **`clear()` 方法**：`clear()` 方法会移除集合中的所有元素。

```python
my_set.clear()
```

## 数学集合操作

- **`issubset()` 和 `issuperset()` 方法**：`issubset()` 和 `issuperset()` 方法分别用于查看一个集合是否是另一个集合的子集或超集。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 5}

print(your_set.issubset(my_set)) # True
print(my_set.issuperset(your_set)) # True
```

- **`isdisjoint()` 方法**：`isdisjoint()` 方法用于查看两个集合是否不相交，即它们没有共同的元素。

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

print(my_set.isdisjoint(your_set)) # True
```

- **并集操作符（`|`）**：并集操作符 `|` 返回一个包含两个集合所有元素的新集合。

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

my_set | your_set # {1, 2, 3, 4, 5, 6}
```

- **交集操作符（`&`）**：交集操作符 `&` 返回一个只包含两个集合共有元素的新集合。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set & your_set # {2, 3, 4}
```

- **差集操作符（`-`）**：差集操作符 `-` 返回一个包含第一个集合中不在另一个集合中的元素的新集合。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set - your_set # {1, 5}
```

- **对称差集操作符（`^`）**：对称差集操作符 `^` 返回一个包含只在第一个集合或第二个集合中出现但不同时出现的元素的新集合。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set ^ your_set # {1, 5, 6}
```

- **`in` 操作符**：你可以用 `in` 操作符查看一个元素是否在集合中。

```python
print(5 in my_set)
```

## Import 语句

- **Import 语句**：要访问内置模块中定义的元素，你需要使用 import 语句。import 语句通常写在文件的顶部。import 语句对函数、类、常量、变量以及模块中定义的任何其他元素的作用是相同的。

- **基本的 Import 语句**：你可以使用 `import` 关键字，后面跟上模块的名称：

```python
import module_name
```

然后，如果你需要调用该模块中的函数，可以使用点表示法，模块名称后跟函数名称。

```python
module_name.function_name()
```

例如，你可以在代码中写如下内容来导入 `math` 模块并计算 36 的平方根：

```python
import math

math.sqrt(36)
```

- **用不同名称导入模块**：如果你需要用不同的名称（也称为“别名”）导入模块，可以在 import 语句末尾使用 `as`，后面跟别名。这通常用于模块名称较长或避免命名冲突的情况。

```python
import module_name as module_alias
```

例如，要在代码中将 `math` 模块称为 `m`，可以这样赋予别名：

```python
import math as m
```

然后，你可以使用别名访问模块的元素：

```python
m.sqrt(36)
```

- **导入特定元素**：如果你不需要模块中的所有内容，可以使用 `from` 导入特定元素。在这种情况下，import 语句以 `from` 开头，后跟模块名称，再使用 `import` 关键字，最后列出你想导入的元素名称。

```python
from module_name import name1, name2
```

然后，你可以在 Python 脚本中直接使用这些名称，而无需模块前缀。例如：

```python
from math import radians, sin, cos

angle_degrees = 40
angle_radians = radians(angle_degrees)

sine_value = sin(angle_radians)
cos_value = cos(angle_radians)

print(sine_value) # 0.6427876096865393
print(cos_value)  # 0.766044443118978
```

这很有用，但如果你已有同名的函数或变量，可能会导致命名冲突。选择使用哪种 import 语句时要注意这一点。

如果你需要为这些名称赋予别名，也可以使用 `as` 关键字，后面跟别名。

```python
from module_name import name1 as alias1, name2 as alias2
```

- **带星号的 Import 语句（`*`）**：星号告诉 Python 你想导入该模块中的所有内容，但导入后不需要使用模块名称作为前缀。

```python
from module_name import *
```

例如，如果你用它导入 `math` 模块，就可以调用该模块中定义的任何函数，而无需指定模块名称作为前缀。

```python
from math import *
print(sqrt(36))  # 6.0
```

不过，这通常不推荐使用，因为它可能导致命名空间冲突，并且让人难以判断名称的来源。

## `if __name__ == '__main__'`

- **`__name__` 变量**：`__name__` 是 Python 中一个特殊的内置变量。当一个 Python 文件被直接执行时，Python 会将该变量的值设置为字串 `"__main__"`。但如果该 Python 文件作为模块被导入到另一个 Python 脚本中，`__name__` 变量的值则被设置为该模块的名称。

这就是为什么你经常会在 Python 脚本中看到这个条件式。它包含了你只想在 Python 脚本作为主程序运行时才执行的代码。

```python
if __name__ == '__main__': 
    # Code
```

## Python 中的常见错误

- **SyntaxError**：当你的代码不符合 Python 的语法规则时，Python 会引发此错误。例如，代码 `print("Hello there"` 会导致语法错误，错误消息为 `SyntaxError: '(' was never closed`，因为代码缺少一个右括号。
- **NameError**：当你尝试访问一个未定义的变量或函数时，Python 会引发 `NameError`。例如，如果你的代码中有一行 `print(username)`，但之前没有定义 `username` 变量，就会出现名称错误，错误消息为 `NameError: name 'username' is not defined`。
- **TypeError**：当你对两个或多个不兼容的数据类型执行操作时，Python 会抛出此错误。例如，如果你尝试将一个字串与一个数字相加，就会得到错误 `TypeError: can only concatenate str (not "int") to str`。
- **IndexError**：如果你访问列表或其他序列（如元组和字串）中不存在的索引，就会得到 `IndexError`。例如，在一个 `Hello world` 字串中，最后一个字符的索引是 `10`。如果你继续这样访问字符，`greet = "hello world"; print(greet[11])`，就会得到错误消息 `IndexError: string index out of range`。
- **AttributeError**：当你尝试使用某个类型的对象中不存在的方法或属性时，Python 会引发此错误。例如，在字串 `"hello".append("!")` 上调用 `.append()` 会导致错误，错误消息为 `AttributeError: 'str' object has no attribute 'append'`。

## 异常处理

- **`try...except`**：用于执行可能引起异常的代码块。`try` 块是你预期可能发生错误的地方，而 `except` 块则处理指定的异常，如果引发了该错误，就会执行该块。示例如下：

  ```py
  try:
    print(22 / 0)
  except ZeroDivisionError:
    print('You can\'t divide by zero!')
    # You can't divide by zero!
  ```

  你也可以链接多个 `except` 块，以便处理更多类型的异常：

  ```py
  try:
    number = int(input('Enter a number: '))
    print(22 / number)
  except ZeroDivisionError:
    print('You cannot divide by zero!')
    # You cannot divide by zero! prints when you enter 0
  except ValueError:
    print('Please enter a valid number!')
    # Please enter a valid number! prints when you don't enter an integer
  ```

- **`else` 和 `finally`**：这些块是对 `try...except` 的扩展。如果没有发生异常，`else` 块会执行。`finally` 块无论是否发生错误都会执行。

  ```py
  try:
    result = 100 / 4
  except ZeroDivisionError:
    print('You cannot divide by zero!') # This will not run
  else:
    print(f'Result is {result}') # Result is 25.0
  finally:
    print('Execution complete!') # Execution complete!
  ```

- **异常对象**：它让你可以访问异常本身，以便更好地调试和打印直接的错误信息。要访问异常对象，需要使用 `as` 关键字。示例如下：

  ```py
  try:
      value = int('This will raise an error')
  except ValueError as e:
      print(f'Caught an error: {e}')
      # Caught an error: invalid literal for int() with base 10: 'This will raise an error'
  ```

- **`raise` 语句**：允许你手动引发异常。当满足某个条件时，可以用它来抛出异常。示例如下：

  ```py
  def divide(a, b):
      if b == 0:
          raise ZeroDivisionError('You cannot divide by zero')
      return a / b
  ```

## 异常信号

`raise` 语句在你创建自定义异常时也非常有用，因为你可以用它抛出带有自定义消息的异常。下面是一个示例：

```py
class InvalidCredentialsError(Exception):
    def __init__(self, message="用户名或密码无效"):
        self.message = message
        super().__init__(self.message)

def login(username, password):
    stored_username = "admin"
    stored_password = "password123"
    
    if username != stored_username or password != stored_password:
        raise InvalidCredentialsError()
    
    return f"欢迎，{username}！"
```  

下面是如何将 `login` 函数与 `InvalidCredentialsError` 异常一起使用的示例：

```py
# 登录失败尝试
try:
    message = login("user", "wrongpassword")
except InvalidCredentialsError as e:
    print(f"登录失败：{e}")
else:
    print(message)

# 登录成功尝试
try:
    message = login("admin", "password123")
except InvalidCredentialsError as e:
    # 由于登录成功，此块不会执行
    print(f"登录失败：{e}")
else:
    # 如果 'try' 块没有异常，else 块会执行
    print(message)
```

`raise` 语句也可以与 `from` 关键字一起使用来链式异常，显示不同错误之间的关系：

```py
def parse_config(filename):
  try:
      with open(filename, 'r') as file:
          data = file.read()
          return int(data)
  except FileNotFoundError:
      raise ValueError('配置文件缺失') from None
  except ValueError as e:
      raise ValueError('配置格式无效') from e

config = parse_config('config.txt')
```

## Python 类和对象

- **类定义**：类是创建对象的蓝图。它通过属性和方法定义对象将具有的行为。以下是 Python 中类定义的一个基本示例：

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f'{self.name.upper()} says woof woof!')
```

- **创建对象**：对象是类的实例。通过调用类并传入必要的参数来创建对象。

```python
dog1 = Dog('Jack', 3)
dog2 = Dog('Thatcher', 5)

dog1.bark()  # JACK says woof woof!
dog2.bark()  # THATCHER says woof woof!
```

- **调用对象的方法**：你可以调用对象的方法来执行操作或获取信息。

```python
object_name1.method_name()
object_name2.method_name()
```

- **类和对象的区别**：类是可重用的模板，而对象是该类的具体实例，包含实际数据。

## 属性

- **实例属性**：在 `__init__()` 中使用 `self` 定义，对每个对象都是唯一的。
- **类属性**：直接在类内部定义，所有实例共享。 

```python
class Dog:
    species = 'French Bulldog'  # Class attribute

    def __init__(self, name):
        self.name = name  # Instance attribute

print(Dog.species) # French Bulldog

jack = Dog('Jack')
print(jack.name)     # Jack
print(jack.species)  # French Bulldog
```

## 方法

- **方法**：定义在类内部、用于操作对象属性的函数。

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
print(my_car_1.describe())  # This car is a red Tesla Model S
```

- **访问方法**：使用点符号调用对象的方法。下面是对两个不同汽车对象调用 `describe` 方法的示例：

```python
class Car:
    def __init__(self, color, model):
        self.color = color  
        self.model = model  

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
my_car_2 = Car('green', 'Lamborghini Revuelto')

print(my_car_1.describe()) # Calling method using the dot notation

print(my_car_2.describe()) # Calling method using the dot notation
```

## Dunder（魔法）方法

- **定义**：以双下划线开头和结尾的特殊方法（例如 `__init__`、`__len__`、`__str__`、`__eq__`）。Python 在内部用于内置操作时会使用它们。

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **间接调用 dunder 方法**：你不需要直接调用 dunder 方法。相反，当发生某些操作时，Python 会自动调用它们。这些操作包括：

  - **算术操作，如加法、减法、乘法、除法等**。加法时调用 `__add__()`，减法时调用 `__sub__()`，乘法时调用 `__mul__()`，除法时调用 `__truediv__()`。
        
  - **字串操作，如串接、重复、格式化和转换为文本**。串接时调用 `__add__()`，重复时调用 `__mul__()`，格式化时调用 `__format__()`，转换为文本时调用 `__str__()` 和 `__repr__()`，等等。
        
  - **比较操作，如相等、小于、大于等**。相等性检查时调用 `__eq__()`，小于时调用 `__lt__()`，大于时调用 `__gt__()`，等等。
        
  - **迭代操作，如使对象可迭代和遍历项**。调用 `__iter__()` 返回迭代器，调用 `__next__()` 获取下一项。

## 真实案例：购物车

- **带有双下划线方法的购物车类**：允许添加、移除、迭代和查看内容，具备内置行为。

```python
class Cart:
    def __init__(self):
```

## 什么是面向对象编程？

- **面向对象编程**：一种编程风格，开发者将代码中的所有内容都视为现实世界中的对象。它通常被称为 OOP。帮助你有效组织和管理代码的四个关键原则是**封装**、**继承**、**多态**和**抽象**。
- **类**：用于创建对象的蓝图。每个由类创建的对象都有定义数据的属性和决定对象行为的方法。

## 什么是封装？

- **封装**：将对象的属性和方法捆绑成一个单一单元。它让你可以通过一组简单的公共方法和属性来隐藏对象的内部状态，这些公共方法和属性就像门一样。在这些门的背后是私有的属性和方法，它们控制数据如何变化以及谁可以看到这些数据。
- **封装示例**：如果你想跟踪钱包余额，你会允许存款和取款，但你不希望任何人篡改钱包余额本身：

```py
class Wallet:
   def __init__(self, balance):
       self.__balance = balance # Private attribute

   def deposit(self, amount):
       if amount > 0:
           self.__balance += amount # Add to the balance safely

   def withdraw(self, amount):
       if 0 < amount <= self.__balance:
           self.__balance -= amount # Remove from the balance safely

account = Wallet(500)
print(account.__balance) # AttributeError: 'Wallet' object has no attribute '__balance'
```

- **单下划线和双下划线前缀属性的区别**：用单下划线作为属性和方法的前缀表示它们是供内部使用的。这是一种约定，并不会强制阻止外部访问这些属性。用双下划线作为前缀则有效地阻止了它们被类外部访问。

## 什么是 Getter 和 Setter？

- **Getter 和 Setter**：允许你控制类的属性如何被访问和修改的方法。你用 getter 来获取值，用 setter 来设置值。
- **属性（Properties）**：它们连接 getter 和 setter，允许访问数据。当你获取、设置或删除值时，属性会在后台运行额外的逻辑。
- **为什么用属性而不是方法**：使用属性代替方法可以提高代码的可读性和整洁度。它们让你像访问普通属性一样用点号语法访问值，而不需要括号。
- **创建 Getter**：要创建 getter，你使用 `@property` 装饰器。下面是一个获取圆半径的 getter：

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self): # A getter to get the radius
        return self._radius
  
    @property
    def area(self):  # A getter to calculate area
        return 3.14 * (self._radius ** 2)

my_circle = Circle(3)

print(my_circle.radius) # 3
print(my_circle.area) # 28.26
```

在 `__init__()` 中使用 self.radius 确保在对象创建时调用 setter，这样可以立即捕获无效的半径值。

- **创建 Setter**：要创建设置半径的 setter，你需要定义另一个同名方法，并在其上方使用 `@<property_name>.setter`：

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self):  # A getter to get the radius
        return self._radius

    @radius.setter
    def radius(self, value):  # A setter to set the radius
        if value <= 0:
            raise ValueError('Radius must be positive')
        self._radius = value

my_circle = Circle(3)
print('Initial radius:', my_circle.radius) # Initial radius: 3

my_circle.radius = 8
print('After modifying the radius:', my_circle.radius) # After modifying the radius: 8
```

- **Python 如何处理 Getter 和 Setter**：一旦你定义了 getter 和 setter，Python 会在你用普通属性语法这样访问时自动调用它们：

```py
my_circle.radius # This will call the getter
my_circle.radius = 4 # This will call the setter
```

在 setter 内部，不应该给属性名本身赋值，因为那会导致 `RecursionError`。

- **删除器（Deleter）**：在用 setter 和 getter 设置和获取值之后，你可以用 `deleter` 控制它如何被删除。删除器在你对属性使用 `del` 语句时运行自定义逻辑。要创建删除器，你使用 `@<property_name>.deleter` 装饰器。

```py
  # Deleter
    @radius.deleter
    def radius(self):
        print("Deleting radius...")
        del self._radius
```

## 什么是继承以及它如何促进代码复用？

- **继承**：子类使用父类的属性和方法的过程。继承促进代码复用，提供清晰的层级结构，并且可以在不重写所有内容的情况下自定义行为。要实现继承，子类需要接收父类的名称：

```py
class Parent:
    # Parent attributes and methods

class Child(Parent):
    # Child inherits, extends, and/or overrides where necessary
```

- **单继承和多继承**：当子类从单个父类继承属性和方法时，如上所示，这个过程称为**单继承**。当子类从多个父类继承属性和方法时，这就是**多继承**。语法如下：

```py
class Parent:
    # Attributes and methods for Parent

class Child:
    # Attributes and methods for Child

class GrandChild(Parent, Child):
    # GrandChild inherits from both Parent and Child
    # GrandChild can combine or override behavior from each
```

- **`super()` 函数**：一个函数，允许你在类对某个方法有不同实现或扩展该方法时，**调用**父类的方法，而无需重复代码。

## 什么是多态以及它如何促进代码复用？

- **多态**：面向对象编程的原则，允许不同的类使用相同的方法名，但每个类在调用时会有不同的实现。它的语法如下：

```py
class A:
   def action(self): ...

class B:
   def action(self): ...

class C:
   def action(self): ...

Class().method()  # Works for A, B, or C
```

- **基于继承的多态**：父类定义一个方法，每个子类根据自己的需要对其进行调整。

## 什么是名称重整及其工作原理？

- **名称重整**：Python 内部对以双下划线开头的属性进行重命名的过程，通过添加一个下划线和类名作为前缀，将 `__attribute` 变成 `_ClassName__attribute`。
- **名称重整的目的**：名称重整的主要目的是防止在使用继承时意外覆盖属性和方法。下面的代码可以帮助你更好地理解： 

```py
class Parent:
    def __init__(self):
        self.__data = 'Parent data'

class Child(Parent):
    def __init__(self):
        super().__init__()
        self.__data = 'Child data'

c = Child()
print(c.__dict__) # {'_Parent__data': 'Parent data', '_Child__data': 'Child data'}
```

## 什么是抽象以及它如何帮助保持复杂系统的有序？

- **抽象**：一种编程概念，通过隐藏对象或系统的复杂实现细节，仅展示必要的特色。在 Python 和其他编程语言中，抽象通过提高可重用性简化复杂系统。
- **抽象的例子**：生活中抽象的一个好例子是汽车，你只需使用方向盘、踏板和换挡杆，而不必了解发动机或刹车的工作原理。
- **Python 如何实现抽象**：Python 通过 `abc` 模块实现抽象。该模块提供了 `ABC` 类（抽象基类）和 `@abstractmethod` 装饰器。抽象基类（ABC）定义了子类必须实现的公共方法和属性，且不能被实例化。
- **抽象方法的定义方式**：抽象方法用 `@abstractmethod` 定义，必须在子类中重写，即使它有默认实现。抽象的基本语法如下：

```py
from abc import ABC, abstractmethod

# Define an abstract base class
class AbstractClass(ABC):
    @abstractmethod
    def abstract_method(self):
        pass

# Concrete subclass that implements the abstract method
class ConcreteClassOne(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassOne')

# Another concrete subclass
class ConcreteClassTwo(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassTwo')
```

## 算法与大 O 表示法

- **算法**：一组明确无歧义的指令，用于解决问题或执行任务。算法必须在有限步数内完成，每一步都必须精确且无歧义。

- **大 O 表示法**：描述算法在输入规模增加时的最坏情况性能或增长率。它关注资源使用随输入规模的增长情况，忽略常数因子和低阶项。

### 常见时间复杂度

- **O(1) - 常数时间**：算法所需时间与输入规模无关，始终相同。

```python
def check_even_or_odd(number):
    if number % 2 == 0:
        return 'Even'
    else:
        return 'Odd'
```

- **O(log n) - 对数时间**：随着输入增长，时间缓慢增加。常见于通过分数不断缩小问题规模的算法（如二分查找）。

- **O(n) - 线性时间**：运行时间与输入规模成正比增长。

```python
for grade in grades:
    print(grade)
```

- **O(n log n) - 线性对数时间**：高效排序算法（如归并排序和快速排序）的常见时间复杂度。

- **O(n²) - 平方时间**：运行时间按平方增长。通常出现在嵌套循环中。

```python
for i in range(n):
    for j in range(n):
        print("Hello, World!")
```

### 空间复杂度

- **O(1) - 常数空间**：算法使用的内存量与输入规模无关，始终相同。
- **O(n) - 线性空间**：内存使用量与输入规模成正比增长。
- **O(n²) - 平方空间**：内存使用量按平方增长。

## 解决问题的技巧

- **理解问题**：多次阅读问题描述。确定输入、预期输出，以及如何将输入转换为输出。

- **伪代码**：对算法逻辑的高级描述，与具体语言无关。使用通用书面语言混合编程结构，如 `IF`、`ELSE`、`FOR`、`WHILE`。

```md
GET original_string
SET reversed_string = ""
FOR EACH character IN original_string:
  ADD character TO THE BEGINNING OF reversed_string
DISPLAY reversed_string
```

- **边界情况**：算法应处理的边界上的特定有效输入。始终考虑并测试边界情况。

## 数组

- **静态数组**：在初始化时确定固定大小。元素存储在相邻的内存位置。程序执行期间大小不可更改。

- **动态数组**：可以在程序执行期间自动增长或缩小。通过在需要时自动拷贝到更大的数组来处理调整大小。

### Python 列表（动态数组）

```python
numbers = [3, 4, 5, 6]

# Access elements
numbers[0]  # 3

# Update elements
numbers[2] = 16

# Add elements
numbers.append(7)
numbers.insert(3, 15)  # Insert at specific index

# Remove elements
numbers.pop(2)  # Remove at specific index
numbers.pop()   # Remove last element
```

### 动态数组的时间复杂度

- **访问**：O(1)
- **末尾插入**：平均 O(1)，需要调整大小时为 O(n)
- **中间插入**：O(n)
- **删除**：中间为 O(n)，末尾为 O(1)

## 栈

- **栈**：后进先出（LIFO）数据结构。元素只能从顶部添加和移除。

- **入栈操作**：将元素添加到栈顶。时间复杂度：O(1)。

- **出栈操作**：从栈顶移除元素。时间复杂度：O(1)。

```python
# Using Python list as stack
stack = []

# Push operations
stack.append(1)
stack.append(2)
stack.append(3)

# Pop operations
top_element = stack.pop()  # Returns 3
```

## 队列

- **队列**：先进先出（FIFO）数据结构。元素从队尾添加，从队头移除。

- **入队操作**：将元素添加到队列的队尾。时间复杂度：O(1)。

- **出队操作**：从队列的队头移除元素。时间复杂度：O(1)。

```python
from collections import deque

# Using deque for efficient queue operations
queue = deque()

# Enqueue operations
queue.append(1)
queue.append(2)
queue.append(3)

# Dequeue operations
first_element = queue.popleft()  # Returns 1
```

## 链表

- **链表**：线性数据结构，每个节点包含数据和指向下一个节点的引用。节点像链条一样连接。

### 单向链表

- **结构**：每个节点有数据和一个指向下一个节点的引用。
- **遍历**：只能从头节点向尾节点单向移动。
- **头节点**：列表中的第一个节点，通常是唯一可以直接访问的节点。
- **尾节点**：列表中的最后一个节点，指向 `None`。

### 操作及时间复杂度

- **开头插入**：O(1)
- **末尾插入**：O(n) — 必须遍历到末尾
- **中间插入**：O(n) — 必须遍历到指定位置
- **开头删除**：O(1)
- **末尾删除**：O(n) — 必须遍历找到前一个节点
- **中间删除**：O(n) — 必须遍历找到目标节点

### 双向链表

- **结构**：每个节点有数据和两个引用：指向下一个节点和上一个节点。
- **遍历**：可以双向移动。
- **内存**：由于额外的引用，所需内存比单向链表多。

## 哈希映射和集合

### 映射和哈希映射

- **映射（抽象数据类型）**：管理键值对集合。每个键必须唯一，但值可以重复。

- **哈希映射**：映射抽象数据类型的具体实现，使用哈希技术。通过哈希函数为键生成哈希值，决定其在底层数组中的存储位置。

### Python 字典（哈希映射）

```python
# Creating dictionaries
my_dictionary = {
    "A": 1,
    "B": 2, 
    "C": 3
}

# Alternative creation
my_dictionary = dict(A=1, B=2, C=3)

# Access and modify
value = my_dictionary["A"]  # 1
my_dictionary["A"] = 4      # Update value
del my_dictionary["A"]      # Remove key-value pair

# Check membership
"C" in my_dictionary

# Get keys, values, items
my_dictionary.keys()
my_dictionary.values()
my_dictionary.items()
```

### 哈希映射的时间复杂度

- **平均情况**：插入、获取、删除均为 O(1)
- **最坏情况**：当发生大量哈希冲突时为 O(n)

### 集合

- **集合**：无序的唯一元素集合。不允许重复元素，也不维护特定顺序。

- **仅限不可变元素**：集合只能包含不可变数据类型（数字、字串、元组），因为哈希值必须保持不变。

```python
# Creating sets
numbers = {1, 2, 3, 4}
empty_set = set()  # Must use set(), not {}

# Add and remove elements
numbers.add(5)
numbers.remove(4)      # Raises KeyError if not found
numbers.discard(4)     # No error if not found

# Set operations
set_a = {1, 2, 3, 4}
set_b = {2, 3, 4, 5, 6}

# Union, intersection, difference, symmetric difference
set_a.union(set_b)                    # or set_a | set_b
set_a.intersection(set_b)             # or set_a & set_b
set_a.difference(set_b)               # or set_a - set_b
set_a.symmetric_difference(set_b)     # or set_a ^ set_b

# Subset and superset checks
set_a.issubset(set_b)
set_a.issuperset(set_b)
set_a.isdisjoint(set_b)

# Membership testing
5 in numbers
```

### 集合的时间复杂度

- **平均情况**：添加、删除、成员测试均为 O(1)
- **最坏情况**：由于哈希冲突为 O(n)

## 哈希冲突

- **哈希冲突**：当两个不同的密钥产生相同的哈希值时发生。

- **冲突解决策略**：
  - **链地址法**：每个数组索引指向一个链表，存储所有具有相同哈希值的元素
  - **开放地址法**：使用预定义的序列查找下一个可用索引

## 何时使用每种数据结构

- **列表**：当你需要有序、索引访问且事先不知道大小时使用
- **栈**：用于后进先出操作（撤销功能、表达式求值、回溯）
- **队列**：用于先进先出操作（任务调度、广度优先查找）
- **链表**：当频繁在开头插入/删除、大小未知且不需要随机访问时使用
- **哈希映射**：用于快速的键值查找、计数出现次数、缓存
- **集合**：用于唯一性检查、数学集合操作、去除副本

## 查找算法

查找算法让你在一定的项列表中查找目标。

在计算机科学中，有两种你应该了解的查找算法。它们是**线性查找**和**二分查找**算法。理解这两种算法的区别以及何时使用它们非常重要。

### 线性查找

- 线性查找通过迭代项列表，从头开始检查每一项，直到找到目标项。
- 如果找到目标项，则返回它在列表中的索引。
- 如果未找到目标，则返回 `-1`，这在大多数编程语言中表示**无效索引**。
- 由于线性查找会检查每一项直到找到目标，对于大型列表来说效率不高。
- 线性查找的时间复杂度是 `O(n)`，因为搜索列表所需的时间随着列表大小线性增长。
- 线性查找的空间复杂度是 `O(1)`，因为它不需要额外空间来搜索列表。

### 二分查找

- 二分查找通过将项列表分成两半，检查目标值是否在列表中间来工作。
- 二分查找的前提条件是列表中的项必须按升序排列。
- 二分查找是搜索大型列表更高效的算法，因为它将列表分成两半，忽略目标不在的那一半。
- 如果目标项在列表中间找到，则返回目标项的索引。
- 如果未找到该项，算法会检查目标项是在列表的左半部分还是右半部分。
- 它继续将剩余部分分成两半，直到找到目标项。
- 如果最终列表中仍未找到目标项，则返回 `-1`。
- 二分查找的时间复杂度是 `O(log n)`，因为搜索列表所需的时间随着列表大小对数增长。
- 二分查找的空间复杂度是 `O(1)`，因为它不需要额外空间来搜索列表。

### 线性查找与二分查找的区别

- 与线性查找相比，二分查找更适合大型项列表。
- 线性查找的时间复杂度是 `O(n)`，因为搜索列表所需的时间随着列表大小线性增长。
- 二分查找的时间复杂度是 `O(log n)`，因为搜索列表所需的时间随着列表大小对数增长。

## 排序算法与分治法

在计算机科学中，分治法是一种将问题拆分成更小子问题以便更容易解决的技术。递归通常用于分治法，而分治法是一种强大的策略，用于实现许多高效的排序算法，比如归并排序。

### 归并排序

- 归并排序是一种遵循分治法的排序算法。
- 它通过递归地将列表划分为更小的子列表，直到每个子列表只包含一个元素。
- 然后反复将子列表合并回去，合并时保持排序顺序。
- 归并排序的时间复杂度是 `O(n log n)`，因为列表不断被对半划分 `(log n)`，然后再合并 `(O(n))`。
- 归并排序的空间复杂度是 `O(n)`，因为它不是原地排序算法。

## 图概述

图是一组由边（连接）连接的节点（顶点）。每个节点可以连接到多个其他节点，形成一个网络。不同类型的图包括：

- 有向图：边有方向（从一个节点指向另一个节点），通常用直线和箭头表示。
- 无向图：边没有方向，用简单的线表示。
- 顶点图：每个节点关联一个标签或标识符。
- 有环图：包含周期（从同一节点出发并返回的路径）。
- 无环图（DAG）：不包含周期。
- 边带标签图：每条边都有一个标签，通常绘制在对应边旁边。
- 带权图：边关联权值（数值），可用于执行算术操作。
- 非连通图：包含两个或多个没有任何边连接的节点。

图被用于各种应用，如地图、网络、推荐系统、依赖关系解析。

## 图遍历

这涉及访问图中的所有节点。两个主要的算法是：

- **广度优先搜索（BFS）**
  - 使用队列。
  - 按层级进行探索。
  - 在无权图中查找最短路径。

- **深度优先搜索（DFS）**
  - 使用栈（或递归）。
  - 在回溯之前完全探索一个分支。
  - 对周期检测和路径查找很有用。

## 图的表现形式

图可以用两种主要方式表现：

- **邻接列表**
  - 每个节点都有一个邻居的列表。
  - 对于稀疏图来说空间高效。
  - 方便迭代邻居。

- **邻接矩阵**
  - 一个二维数组，行和列代表节点。
  - 对于大型图来说空间消耗大。
  - 快速查看两个节点之间是否存在边。

## 树

树是一种特殊类型的图，它是无周期且连通的。主要特性包括：

- 它们没有循环或周期（起点和终点节点相同的路径）。
- 它们必须是连通的（每个节点都可以从其他任何节点到达）。

### 常见的树类型

最常见的树类型有：

- 二叉树
  - 每个节点最多有两个子节点，分别是左子节点和右子节点。

- 二叉搜索树（BST）
  - 一种二叉树，其中每个左子节点都小于其父节点，每个右子节点都大于其父节点。

## 字典树

也称为前缀树，用于保存字串集合，其中每个节点表现一个字符。

共享的前缀只保存一次，使它们在自动补全和拼写检查等任务中非常高效。

查找和插入操作的时间复杂度为 O(L)，其中 L 是字串的长度。

## 优先级队列

优先级队列是一种抽象数据类型，其中每个元素都有一个优先级。

队列和栈只考虑插入的顺序，而优先级队列则考虑元素的优先级。

标准队列遵循 FIFO（先进先出），栈遵循 LIFO（后进先出）。然而，在优先级队列中，优先级较高的元素会先被处理，而不管它们的插入顺序。

## 堆

堆是一种特化的基于树的数据结构，具有一个非常具体的属性，称为堆属性。

堆属性决定了父节点和子节点之间的关系。堆有两种类型：

- 最大堆
  - 每个父节点的值大于或等于其子节点的值。
  - 最大的元素位于根节点。

- 最小堆
  - 每个父节点的值小于或等于其子节点的值。
  - 最小的元素位于根节点。

### Python `heapq` 模块示例

```py
import heapq

# Create empty heap
my_heap = []

# Insert elements
heapq.heappush(my_heap, 9)
heapq.heappush(my_heap, 3)
heapq.heappush(my_heap, 5)
print(my_heap) # [3, 9, 5]

# Remove smallest element
print(heapq.heappop(my_heap))  # 3
print(my_heap) # [5, 9]

# Push + Pop in one step
print(heapq.heappushpop(my_heap, 7)) # 5
print(my_heap) # [7, 9]

# Transform list into heap
nums = [5, 7, 3, 1]
heapq.heapify(nums)
```

### 使用优先级

```py
my_heap = []
heapq.heappush(my_heap, (3, "A"))
heapq.heappush(my_heap, (2, "B"))
heapq.heappush(my_heap, (1, "C"))

# Removes lowest number = highest priority
print(heapq.heappop(my_heap))  # (1, "C")
```

## 动态规划简介

- **定义**：动态规划是一种算法技术，通过将复杂问题分解为更简单的子问题并保存结果，避免重复计算，从而解决复杂问题。
- **重叠子问题**：在解决较大问题时，相同的较小问题会多次出现。我们通过保存这些子问题的解，避免反复计算。
- **最优子结构**：问题的最优解包含其子问题的最优解。这意味着我们可以通过组合较小部分的最优解来构建整体的最优解。

## 动态规划方案

- **记忆化（自顶向下方法）**：记忆化保存昂贵函数调用的结果，当相同输入再次出现时返回缓存的结果。

```py
def climb_stairs_memo(n, memo={}):
    """Dynamic programming with memoization"""
    # Check if we've already calculated this value
    if n in memo:
        return memo[n]  # Return cached result - O(1) lookup!
    
    # Base cases
    if n <= 2:
        return n
    
    # Calculate once and store in memo for future use
    memo[n] = climb_stairs_memo(n-1, memo) + climb_stairs_memo(n-2, memo)
    return memo[n]
```

- **表格法（自底向上方法）**：表格法从底层开始构建方案，使用表格填充子问题的解。

```py
def climb_stairs_tabulation(n):
    """Dynamic programming with tabulation"""
    if n <= 2:
        return n
    
    # Create array to store results for all steps from 0 to n
    dp = [0] * (n + 1)
    dp[1] = 1  # 1 way to reach step 1
    dp[2] = 2  # 2 ways to reach step 2
    
    # Build up the solution iteratively
    for i in range(3, n + 1):
        # Ways to reach step i = ways to reach (i-1) + ways to reach (i-2)
        dp[i] = dp[i-1] + dp[i-2]
    
    return dp[n]
```

## 使用动态规划的实际应用

- **路径优化**：GPS 系统使用动态规划算法来查找位置之间的最短路径。
- **文本处理**：拼写检查和自动完成功能通常依赖动态规划来计算单词之间的编辑距离。
- **金融建模**：投资策略和投资组合优化经常采用动态规划技术。
- **资源分配**：背包问题及其变体出现在调度、预算和资源管理中。

## 何时使用动态规划

你应该在以下情况下考虑使用动态规划：

- 问题可以拆分为重叠的子问题。
- 问题具有最优子结构。
- 朴素的递归解法会涉及重复计算。
- 你需要以牺牲空间复杂度为代价来优化时间复杂度。
