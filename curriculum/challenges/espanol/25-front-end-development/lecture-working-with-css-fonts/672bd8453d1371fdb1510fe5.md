# --description--

Una fuente externa es un archivo de fuente que no está incluido directamente dentro de los archivos de tu proyecto. Por lo regular están alojadas en un servidor separado. Un servidor es una computadora que proporciona datos o servicios a otras computadoras a través de una red. Aprenderás más sobre servidores en futuras lecciones. Las fuentes externas te dan más flexibilidad porque puedes usar fuentes personalizadas que pueden no estar instaladas en el dispositivo del usuario.

Google Fonts y Font Squirrel son recursos en línea populares para encontrar y usar fuentes gratuitas para propósitos de desarrollo web. Puedes incorporarlas a tus proyectos muy fácilmente.

Comencemos con Google Fonts. Este es un servicio de Google que ofrece una colección de fuentes, muchas de las cuales están diseñadas específicamente para el desarrollo web. En la interfaz de usuario de Google Fonts, puedes ver muchos elementos diferentes. Analicémoslos uno por uno. En la barra lateral izquierda, encontrarás:

- Una sección para ver, encontrar y filtrar fuentes.
- Una sección especial sobre Noto, una colección de fuentes de alta calidad con varios pesos, anchos y estilos que son perfectas para comunicarse en más de 1,000 idiomas y más de 150 sistemas de escritura.
- Una sección donde puedes encontrar y descargar íconos para tus proyectos web.
- Una sección donde puedes aprender más sobre las fuentes y sus mejores prácticas, seguida de preguntas frecuentes.

Para añadir una fuente de Google a tu sitio web, debes ir a la primera sección (Fuentes). Puedes personalizar el texto de vista previa en la barra lateral izquierda, donde dice "Vista previa". Solo escribe el texto que te gustaría ver y se actualizará automáticamente. Esto es útil para previsualizar la fuente con el texto que ya tienes en mente para tu sitio web. También puedes ajustar el tamaño de la fuente y filtrar las fuentes según sus características.

Para ver más información sobre una fuente, solo haz clic en ella y verás información importante sobre la fuente, como quién la diseñó, texto de vista previa y diferentes estilos, como claro, regular, medio, negrita y negro. Si haces clic en las opciones que están al principio, puedes ir al probador de tipos, ver glifos individuales y aprender más sobre la fuente y su licencia.

Puedes personalizar el tamaño de la fuente del texto de vista previa con el control deslizante azul ubicado junto a donde puedes ingresar el texto de vista previa. Una vez que estés listo para agregar la fuente a tu proyecto, haz clic en el botón azul "Obtener fuente" en la parte superior. Serás llevado a una página de resumen donde verás las fuentes que has seleccionado actualmente. Puedes tener múltiples fuentes seleccionadas simultáneamente.

A continuación, debes elegir si te gustaría descargar los archivos de fuente para agregarlos a tu proyecto como archivos locales, o si prefieres usarlas como fuentes externas y descargarlas de los servidores de Google cuando un usuario ingresa a tu sitio web. Haz clic en "Descargar todo" si deseas descargarlas, pero si prefieres utilizarlas como fuentes externas en los servidores de Google, haz clic en "Obtener código de inserción". Si haces clic en "Obtener código de inserción", verás las instrucciones que debes seguir para agregar estas fuentes externas a tu proyecto.

Para los proyectos de desarrollo web, tienes dos opciones. Puedes usar un elemento `link` o `@import`. Si eliges la opción del elemento `link`, puedes copiar y pegar el fragmento de HTML y las reglas CSS para agregarlas a tu proyecto. Debes integrar el código en el elemento `head` de tu archivo HTML y agregar las reglas CSS que se ajusten a tus necesidades. Por ejemplo, esto es lo que añadirías a tu archivo HTML para agregar todos los estilos de Roboto:

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap" rel="stylesheet">
```

Hay una regla CSS para cada estilo de fuente. Cada regla asigna las fuentes personalizadas con fuentes de respaldo en caso de que las fuentes personalizadas no se carguen correctamente.

Si eliges la opción `@import`, en su lugar, necesitarás agregar esa regla a tu archivo CSS. Aquí hay un ejemplo:

```css
@import url('https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap');
```

Las reglas CSS son exactamente las mismas.

Si solo deseas incluir estilos de fuentes específicos, también tienes la opción de eliminarlas de estos fragmentos. Los fragmentos de HTML y CSS se actualizarán automáticamente.

Y esto es Google Fonts. Es una de las opciones más populares para encontrar fuentes gratuitas y de código abierto para proyectos de desarrollo web.

Otra gran opción es Font Squirrel, donde puedes encontrar y descargar las fuentes personalizadas que visualizas para tu diseño. Si buscas una fuente y haces clic en un resultado, verás más detalles sobre la fuente, incluidos especímenes, prueba de manejo, glifos y kit de Webfont. También puedes ver muestras de los diferentes estilos y variaciones de la fuente, como delgado, claro, mediano, negrita y negro.

Una vez que hayas decidido sobre una fuente que quieres usar en tu proyecto, ve a la pestaña "Webfont Kit". Aquí, puedes verificar si la licencia de la fuente permite que la uses en `@font-face` embebidos CSS. También puedes elegir el subconjunto y formato.

Haciendo clic en el botón "Descargar Kit de @font-face" descargarás un archivo comprimido (zip). Si lo descomprimes, encontrarás una carpeta "web fonts", la licencia como un archivo de texto y un archivo HTML con instrucciones sobre cómo usar las fuentes web. La carpeta de fuentes web tendrá todos los archivos de fuente que necesitas para tu proyecto. Los encontrarás agrupados en carpetas específicas para los diferentes estilos de fuentes.

Si abres el archivo HTML en un navegador, tiene instrucciones sobre cómo agregar las fuentes a tu proyecto, cómo incluirlas en la hoja de estilo con la declaración `@font-face`, y cómo usarlas en las reglas CSS.

Esta es la manera de añadir fuentes personalizadas de Font Squirrel a tu proyecto. También puedes alojarlas en un servidor separado y usarlas como fuentes externas. Las fuentes externas son muy útiles para personalizar la apariencia y el diseño general de tu sitio web. Pueden hacer que tu sitio web sea verdaderamente único.

Sin embargo, también debes tener en cuenta que usar múltiples fuentes externas puede aumentar el tiempo de carga del sitio web, afectando su rendimiento general. Siempre debes esforzarte por encontrar un equilibrio entre el estilo y la eficiencia.
