# --description--

`Configuration` في البرمجيات والبرمجة تشير إلى إعداد أو ترتيب مكوِّنات النظام أو التطبيق. غالبًا ما تتضمن إعدادات تحدد كيفية عمل البرنامج.

على سبيل المثال، تغيير `configuration` لتطبيق يمكن أن يضبط كيفية أدائه أو تفاعله مع أنظمة أخرى.
