# --description--

你們中許多人已經使用電腦一段時間了。但你可能還沒想過登入你的電腦時最安全或最有保障的方法。

無論你使用 Mac 還是 PC，都有許多方法可以安全地登入你的電腦。

你首先應該確保你的電腦有設定密碼保護。

對於 Windows 使用者，請先開啟開始選單並選擇設定。接著，前往帳戶，然後選擇登入選項。你就可以為你的電腦設定密碼。

*注意：如果你遇到任何困難或使用不同版本的 Windows，請取用官方 Microsoft 文件或支援資源，以取得針對你特定版本的詳細說明。*

在建立你的密碼時，請確保它是一個長且複雜的密碼。你可以使用字母、數字和特殊字元的組合來提高安全性。

選擇像 `12345` 或 `password` 這樣簡單的密碼不是好主意。這些很容易被猜到，也很容易被駭客入侵。

此外，也不要使用基於個人資訊如你的名字、生日或位址的密碼。這些對駭客來說也很容易猜出。

當你創建你的密碼時，同時設定雙重驗證（2FA）也是一個好主意。2FA 作為額外的安全層，並且需要第二種驗證方式，以確保嘗試存取帳戶的人確實是授權的使用者。

如果你是 Mac 使用者，可以點擊 Apple 選單，然後前往系統設定。從那裡你可以進入 使用者與群組，並為你的電腦設定密碼。

部分 Mac 電腦配備了觸控 ID 特性，這通常被認為比一般密碼更安全。這個特性允許你使用指紋登入你的電腦。

對於 Windows 使用者，Windows Hello 特性提供更安全的登入方式。它使用臉部辨識或指紋等生物識別方法進行驗證，提供傳統密碼的替代方案。

完成本課程後，我強烈建議你研究這些額外的安全特性，並在你的電腦上設定它們，以便你能保持你的電腦安全無虞。
