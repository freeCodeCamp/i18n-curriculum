# --description--

Diese Aufgabe konzentriert sich auf das Relativpronomen `that`. Es wird verwendet, um eine Klausel einzuleiten, die mehr Details über ein Substantiv liefert.

Eine Klausel ist ein Teil eines Satzes, der mindestens ein Subjekt und ein Verb enthält. Sie kann eine vollständige Idee ausdrücken oder zusätzliche Informationen zum Hauptsatz hinzufügen.

Das Pronomen `that` hilft dabei, etwas genauer zu spezifizieren oder zu definieren, indem es mit einer Beschreibung oder einem notwendigen Detail verbunden wird.

Zum Beispiel `She read the book that won the Pulitzer Prize last year.`: In diesem Satz ist `that won the Pulitzer Prize last year` eine Relativklausel, die durch `that` eingeleitet wird. Die Klausel beschreibt oder spezifiziert das Buch und hilft dabei, es von anderen Büchern zu unterscheiden. Hier fungiert `that` als Relativpronomen, das die Klausel mit dem Substantiv `book` verbindet und beschreibt, welches bestimmte Buch sie gelesen hat.
