# --description--

Dieses Quiz überprüft Ihr Verständnis von Begrüßungen, Verabschiedungen, Nationalitäten, Berufen und Altersangaben auf Spanisch.

Um das Quiz zu bestehen, müssen Sie **mindestens 9 von 10** Fragen richtig beantworten.

Lesen Sie jede Frage und wählen Sie die richtige Antwort aus. Es gibt nur eine richtige Antwort pro Frage.
