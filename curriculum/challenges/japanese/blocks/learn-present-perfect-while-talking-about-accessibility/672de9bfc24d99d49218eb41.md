# --explanation--

`Specific` は、特定のものや明確なものを指します。例えば：

`She had specific questions about the project requirements.` - これは、彼女がプロジェクトの一般的な質問ではなく、特定の詳細について質問があったことを意味します。

`As expected` は、物事が人々の予想通りに進んだことを意味します。例えば：

`The software update improved performance, as expected, because the team fixed key bugs.` - これは、アップデートのバグ修正のおかげで、チームが予測した通りにパフォーマンスが向上したことを意味します。
