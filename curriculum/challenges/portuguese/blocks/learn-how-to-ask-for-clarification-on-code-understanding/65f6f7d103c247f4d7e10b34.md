# --description--

A palavra `clear` significa fácil de entender ou ver.Quando você quer comparar algo e mostrar que é mais `clear`, você adiciona `er` no final para tornar `clearer`.

Por exemplo:

- Esta explicação é `clear`.
- Esta explicação é `clearer` do que a outra.

Note como `clearer` é usado para comparar duas coisas, indicando que uma é mais fácil de entender do que a outra.
