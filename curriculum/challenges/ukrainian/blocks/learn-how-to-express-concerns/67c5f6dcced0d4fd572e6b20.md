# --explanation--

`Also` використовується як сполучник, щоб додати ще один запит, ідею або інформацію. Він пов’язує дві пов’язані думки, залишаючи їх окремими реченнями. Наприклад:

`We should review the budget. Also, let's check if we have enough resources.` — це означає, що перевірка ресурсів є додатковим кроком після перегляду бюджету.
