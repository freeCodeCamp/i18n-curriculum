# --explanation--

`Present Continuous` 時制は、話している時点での一時的な状態や問題を強調することが多いです。例えば：

- `The server isn't running diagnostics.` - 診断プロセスが今まさに行われていないことを示します。

- `Developers aren't debugging the application.` - 開発者がその作業を今していないことを示します。

`Properly` は、正しいまたは期待される方法で何かをしていることを意味します。

この文では、`aren't booting properly` がコンピューターが現在正しく起動していないことを指しています。
