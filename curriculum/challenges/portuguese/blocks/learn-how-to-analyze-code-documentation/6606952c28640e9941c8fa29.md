# --description--

Geralmente, `with` é usado para mostrar conexão ou uso.Por exemplo, `I write with a pen.` No entanto, em algumas frases, `with` vem depois da palavra com a qual é usada.Isso geralmente acontece quando você quer enfatizar o objeto ou o que alguém está usando ou lidando, especialmente em perguntas ou declarações sobre tecnologia ou trabalho.

Por exemplo, em tecnologia, você poderia dizer, `This program I work with is challenging.` Aqui, `with` é usado no final para enfatizar o programa com o qual é desafiador trabalhar.Outro exemplo poderia ser uma questão semelhante, `What else is he working with?` onde `with` no final pergunta sobre outras ferramentas ou tecnologias que estão sendo usadas.
