# --description--

`Step by step` は、一度に一段階ずつ何かを行うことを意味します。例えば、`debugging step by step` は問題を小さく管理しやすい部分に分けて解決することを意味します。

`Input` は、コーディングにおいてはプロジェクトに対して与えられる情報やアドバイスを意味することが多いです。

`Work together` は協力することを意味し、`pair up` は似ていて、二人が密接に一緒に働くことを意味します。

最後に、`to solve a problem` はそれに対する解決策を見つけようとすることを意味します。この場合、トムとサラはコードの可能性のあるバグの解決策を見つけようとしています。
