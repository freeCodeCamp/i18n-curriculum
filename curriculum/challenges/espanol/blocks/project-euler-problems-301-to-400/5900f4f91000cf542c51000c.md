# --description--

En la parábola $y = \frac{x^2}{k}$, se eligen tres puntos $A(a, \frac{a^2}{k})$, $B(b, \frac{b^2}{k})$ y $C(c, \frac{c^2}{k})$.

Sea $F(K, X)$ el número de cuádruples enteros $(k, a, b, c)$ tales que al menos un ángulo del triángulo $ABC$ es 45°, con $1 ≤ k ≤ K$ y $-X ≤ a &lt; b &lt; c ≤ X$.

Por ejemplo, $F(1, 10) = 41$ y $F(10, 100) = 12\\,492$.

Encuentra $F({10}^6, {10}^9)$.
