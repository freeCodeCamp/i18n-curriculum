# --explanation--

`Delays`는 어떤 일이 예상보다 오래 걸릴 때 발생합니다. 예를 들어:

`Poor time management can result in project delays.` - 팀이 작업을 잘 관리하지 못하면 프로젝트가 더 오래 걸릴 수 있습니다.

`Miscommunication`는 메시지가 명확하지 않아 사람들이 서로 오해할 때 발생합니다. 이는 실수와 혼란을 초래할 수 있습니다. 예를 들어:

`Miscommunication between departments delayed the product launch.` - 서로 다른 팀들이 조율되지 않아 차질이 생겼습니다.
