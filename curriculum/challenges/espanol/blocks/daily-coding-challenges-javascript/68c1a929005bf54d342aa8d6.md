# --description--

Para el quinto día de la Semana del Espacio, calcularás la "zona habitable" de una estrella: la región alrededor de una estrella donde las condiciones son "justas" para que exista agua líquida.

Dada la masa de una estrella, devuelve un arreglo con las distancias de inicio y fin de su Zona Ricitos de Oro en Unidades Astronómicas.

Para calcular la Zona de Ricitos de Oro:

1. Encuentra la luminosidad de la estrella elevando su masa a la potencia de 3.5.
2. El inicio de la zona es 0.95 veces la raíz cuadrada de su luminosidad.
3. El final de la zona es 1.37 veces la raíz cuadrada de su luminosidad.

- Devuelve las distancias redondeadas a dos decimales.

Por ejemplo, dado `1` como masa, devuelve `[0.95, 1.37]`.

# --solutions--

```js
function goldilocksZone(mass) {
  const luminosity = Math.pow(mass, 3.5);
  const start = 0.95 * Math.sqrt(luminosity);
  const end = 1.37 * Math.sqrt(luminosity);
  return [Number(start.toFixed(2)), Number(end.toFixed(2))];
}
```
