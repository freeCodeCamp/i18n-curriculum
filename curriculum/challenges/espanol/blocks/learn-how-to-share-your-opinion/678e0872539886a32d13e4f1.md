# --explanation--

`Lightweight` describe algo que no requiere muchos recursos o energía, lo que lo hace fácil de ejecutar, rápido y eficiente. Por ejemplo:

`The app is lightweight, so it runs smoothly even on older smartphones.` - Esto significa que la app usa muy poca memoria o potencia de procesamiento, lo que le permite funcionar bien en dispositivos con menos capacidad.

`Features` son funciones o herramientas específicas que un producto o software ofrece a los usuarios. Por ejemplo:

`The software has many useful features like file sharing and cloud storage.` - Esto significa que el software incluye varias funciones útiles que lo hacen más valioso y eficiente para los usuarios.
