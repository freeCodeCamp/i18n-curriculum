# --description--

Wichtige Aspekte der Variablennamen in Python sind:

- Einige Wörter sind reservierte Schlüsselwörter (z. B. `for`, `while`, `True`). Sie haben eine besondere Bedeutung in Python, daher können Sie sie nicht als Variablennamen verwenden.
- Variablennamen dürfen nicht mit einer Zahl beginnen und dürfen nur alphanumerische Zeichen oder Unterstriche enthalten.
- Variablennamen sind case-sensitiv, das heißt `my_var` ist anders als `my_Var` und `MY_VAR`.
- Schließlich ist es eine gängige Konvention, Variablennamen im `snake_case`-Stil zu schreiben, wobei jeder Leerraum durch einen Unterstrich ersetzt wird und die Wörter in Kleinbuchstaben geschrieben sind.

Entfernen Sie beide Aufrufe von `print()` und deklarieren Sie eine weitere Variable namens `alphabet`. Weisen Sie dieser Variable den String `'abcdefghijklmnopqrstuvwxyz'` zu.
