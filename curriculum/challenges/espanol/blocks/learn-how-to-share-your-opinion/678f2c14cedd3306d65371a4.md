# --explanation--

`Might` se usa para mostrar posibilidad o incertidumbre. A menudo se usa cuando algo no está garantizado pero podría ocurrir.

`Sure` se usa para mostrar certeza o confianza sobre una situación. Cuando alguien dice que no está seguro, significa que tiene dudas o no sabe con certeza.
