# --description--

整数座標を持ち、$|x| + |y| ≤ r$ を満たす点 ($x$,$y$) の集合 $S(r)$ を考えます。

点 $O$ を (0,0)、点 $C$ を ($\frac{r}{4}$,$\frac{r}{4}$) とします。

$S(r)$ に含まれる点 $B$ のうち、三角形 $OBC$ が鈍角を持つ、つまり最大の角度 $α$ が $90°<α<180°$ を満たす点の数を $N(r)$ とします。

例えば、$N(4)=24$、$N(8)=100$ です。

$N(1\,000\,000\,000)$ はいくつでしょうか？
