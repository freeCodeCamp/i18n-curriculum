# --interactive--

In JavaScript, una stringa è una sequenza di caratteri usata per rappresentare dati testuali. Le stringhe sono uno dei tipi di dato primitivi nel linguaggio, insieme a numeri, booleani, `null` e `undefined`.

Per creare una stringa in JavaScript, puoi usare virgolette singole (`'`) o virgolette doppie (`"`).

Ecco un esempio di creazione di due variabili che contengono valori stringa:

:::interactive_editor

```js
let singleQuotes = 'This is a string';
console.log(singleQuotes);
let doubleQuotes = "This is also a string";
console.log(doubleQuotes);
```

:::

Anche se puoi usare virgolette singole o doppie per creare stringhe, è importante essere coerenti. Se una stringa inizia con una virgoletta singola, deve anche finire con una virgoletta singola.

Lo stesso vale per le virgolette doppie. L’esempio seguente genera un errore perché inizia con una virgoletta doppia e finisce con una virgoletta singola:

```js
const improperStr = "Do not do this';
```

Un’altra cosa da considerare è che le stringhe sono immutabili. In programmazione, immutabilità significa che una volta creato qualcosa, non può essere modificato. Quindi, quando crei una stringa, non puoi cambiare direttamente i suoi caratteri. Invece, creeresti una nuova stringa se vuoi apportare modifiche.

Ecco un esempio di assegnazione di una nuova stringa a una variabile `developer`:

:::interactive_editor

```js
let developer = "Jessica";
console.log(developer);
developer = "Quincy";
console.log(developer);
```

:::

Poiché le stringhe sono immutabili, non possiamo aggiornare direttamente la prima stringa. Per questo stiamo assegnando una nuova stringa alla variabile `developer`.

Le stringhe sono una parte importante della programmazione e nelle lezioni future imparerai tecniche avanzate per manipolarle e sfruttarne appieno il potenziale per creare applicazioni dinamiche e interattive.
