# --description--

`Store information` significa conservare o salvare dati in una posizione specifica, di solito in una variabile. Le variabili sono come contenitori nel tuo codice che tengono informazioni, come numeri, testo o altri tipi di dati. Puoi pensarle come spazi di archiviazione nominati dove puoi consultare il `stored information` più avanti nel tuo codice.

Per esempio, `int age = 30;` Qui, la variabile `age` conserva il numero 30.

La parola `like` si usa come preposizione per fare esempi o per confrontare cose simili.

Per esempio, se dici `I love fruits like apples and oranges.` Stai usando `like` per fare esempi del tipo di frutta che ti piace, suggerendo che potrebbero esserci altri frutti in quella lista.
