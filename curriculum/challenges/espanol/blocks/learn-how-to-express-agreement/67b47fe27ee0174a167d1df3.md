# --explanation--

`Tend to` es una frase que se usa para describir algo que ocurre regularmente o que es probable que ocurra. Por ejemplo:

`She tends to be early for meetings.` - Esto significa que llegar temprano a las reuniones es un hábito o una ocurrencia regular para ella.

`Lag` se refiere a un retraso o tiempo de respuesta lento. Por ejemplo:

`The video lags when there is a poor internet connection.` - Esto significa que el video se retrasa o va lento debido a la conexión.

`Switch` significa cambiar de una cosa a otra. Por ejemplo:

`Users can switch between different tabs in the browser.` - Esto significa que los usuarios pueden cambiar de una pestaña a otra mientras navegan.
