# --description--

Para perguntar a alguém se quer algo, o espanhol usa o verbo `querer`.

A forma verbal `quieres` é usada com `tú` (você) para perguntar sobre os desejos ou preferências de outra pessoa. É comum na conversa cotidiana. Por exemplo:

- `¿Quieres ayuda?` - Você quer ajuda?

- `¿Quieres más tiempo?` - Você quer mais tempo?
