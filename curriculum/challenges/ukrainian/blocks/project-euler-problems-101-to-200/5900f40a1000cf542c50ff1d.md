# --description--

Використовуючи три різні літери з 26 літер алфавіту, можна утворити рядки довжиною три символи.

Прикладами є 'abc', 'hat' і 'zyx'.

Якщо розглянути ці три приклади, то для 'abc' два символи лексикографічно йдуть після свого сусіда зліва.

Для 'hat' є рівно один символ, який лексикографічно йде після сусіда зліва. Для 'zyx' таких символів немає.

Всього існує 10400 рядків довжиною 3, для яких рівно один символ лексикографічно йде після сусіда зліва.

Тепер розглянемо рядки довжиною $n ≤ 26$ з різних символів алфавіту.

Для кожного $n$, $p(n)$ — це число рядків довжиною $n$, для яких рівно один символ лексикографічно йде після сусіда зліва.

Яке максимальне значення $p(n)$?
