# --interactive--

Метод `toString()` — це базова функція в JavaScript, яка конвертує значення у його рядкове представлення. Це метод, який ви можете використовувати для чисел, булевих значень, масивів і об’єктів. Одне з найпоширеніших застосувань `toString()` — це конвертація числа у рядок. Ось приклад:

:::interactive_editor

```js
const num = 10;
console.log(num.toString()); // "10"
```

:::

Цей метод приймає необов’язковий аргумент radix, який є числом від `2` до `36`. Цей radix представляє основу системи числення, наприклад, основу 2 для двійкової або основу 8 для вісімкової. Якщо radix не вказано, за замовчуванням використовується основа 10, тобто десяткова. Ось приклад передачі `2` як аргументу методу `toString()`:

:::interactive_editor

```js
const num = 10;
console.log(num.toString(2)); // "1010"
```

:::

Результатом буде `1010`, що є двійковим представленням десяткового числа `10`.

Ви також можете використовувати метод `toString()` для конвертації масивів і об’єктів у рядки. Масиви мають кастомну реалізацію `toString()`, яка конвертує кожен елемент у рядок і об’єднує їх комами. Ось приклад:

:::interactive_editor

```js
const arr = [1, 2, 3];
console.log(arr.toString()); // "1,2,3"
```

:::

У цьому прикладі всі елементи масиву об’єднані разом, утворюючи рядок `1,2,3`.

Коли метод `toString()` використовується з об’єктами, результатом не буде рядкове представлення властивостей об’єкта.

:::interactive_editor

```js
const person = {
  name: "John",
  age: 30,
  isStudent: true
};

console.log(person.toString()); // "[object Object]"
```

:::

У цьому прикладі результатом буде стандартне рядкове представлення об’єкта, яке є `[object Object]`. Щоб отримати рядкове представлення властивостей об’єкта `person`, вам потрібно буде використати `JSON.stringify()`, про який ви дізнаєтеся більше в майбутніх уроках.

Підсумовуючи, метод `toString()` використовується для конвертації значень у рядки. Розуміння того, як він працює з різними типами даних і як кастомізувати його для власних об’єктів, значно покращить вашу здатність маніпулювати та відображати дані у ваших JavaScript-застосунках.
