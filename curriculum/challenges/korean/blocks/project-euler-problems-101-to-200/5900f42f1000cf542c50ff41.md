# --description--

양의 정수 $n$이 소수의 제곱으로 나누어지지 않으면 제곱 인수 없는 수라고 합니다. 따라서 1, 2, 3, 5, 6, 7, 10, 11은 제곱 인수 없는 수이고, 4, 8, 9, 12는 그렇지 않습니다.

$2^{50}$보다 작은 제곱 인수 없는 수는 몇 개일까요?
