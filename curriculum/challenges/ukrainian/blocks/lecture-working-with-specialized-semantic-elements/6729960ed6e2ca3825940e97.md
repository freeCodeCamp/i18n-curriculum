# --interactive--

Рядковий елемент коду використовується для представлення коротких фрагментів коду всередині тексту. Загальні випадки використання елемента коду — це технічні статті та сторінки документації.

Ось приклад використання елемента `code` для показу фрагмента CSS-коду:

:::interactive_editor

```html
<p>
  To set the text color to blue in CSS, use the following code:
  <code>color: blue;</code>
</p>
```

:::

У цьому прикладі властивість CSS `color` використовується для встановлення кольору тексту `blue`. Обгортаючи цей фрагмент коду в теги `<code>`, ми повідомляємо браузеру, що текст є рядковим кодом.

Браузер застосує стилі за замовчуванням для вмісту всередині елемента `code`. Стиль за замовчуванням — це моноширинний шрифт.

Елемент `code` призначений для представлення одного рядка коду. Якщо потрібно представити кілька рядків коду, слід розмістити елемент `code` всередині елемента з попередньо відформатованим текстом.

Елемент з попередньо відформатованим текстом використовується для представлення тексту з фіксованим форматуванням. Ось приклад використання цього елемента для показу оголошення CSS:

:::interactive_editor

```html
<pre>
  <code>
    body {
      color: red;
    }
  </code>
</pre>
```

:::

При використанні елемента `pre` потрібно звертати увагу на пробіли, оскільки він відображатиметься точно так, як записано в HTML-документі.

У браузері ви побачите, що код відступає на кілька пробілів вправо. Якщо змінити відступ у прикладі коду, це відобразиться на екрані.

Що стосується включення прикладів коду у ваш HTML-документ, слід використовувати елемент `code` для коротких рядкових прикладів.

Якщо потрібно показати довші фрагменти коду, слід використовувати елементи `pre` та `code`.
