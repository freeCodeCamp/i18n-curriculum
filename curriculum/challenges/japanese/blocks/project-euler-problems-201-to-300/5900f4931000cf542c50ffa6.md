# --description--

2つの円に囲まれた凸エリアを、次の条件を満たす場合にレンズ状の穴と呼びます。

- 両方の円の中心が格子点にあること。
- 2つの円が異なる2つの格子点で交差していること。
- 両方の円に囲まれた凸エリアの内部に格子点が存在しないこと。

次の円を考えます。

$$\begin{align}
  & C_0: x^2 + y^2 = 25 \\\\
  & C_1: {(x + 4)}^2 + {(y - 4)}^2 = 1 \\\\
  & C_2: {(x - 12)}^2 + {(y - 4)}^2 = 65
\end{align}$$

下の図には円 $C_0$、$C_1$、$C_2$ が描かれています。

<img alt="C_0, C_1 and C_2 circles" src="https://cdn.freecodecamp.org/curriculum/project-euler/lenticular-holes.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$C_0$ と $C_1$、および $C_0$ と $C_2$ はレンズ状の穴を形成しています。

正の実数の順序対 ($r_1$, $r_2$) をレンズ状ペアと呼びます。これは、半径がそれぞれ $r_1$ と $r_2$ の2つの円がレンズ状の穴を形成する場合に存在します。上の例では、($1$, $5$) と ($5$, $\sqrt{65}$) がレンズ状ペアであることが確認できます。

$L(N)$ を、$0 < r_1 ≤ r_2 ≤ N$ を満たす異なるレンズ状ペア ($r_1$, $r_2$) の数とします。$L(10) = 30$、$L(100) = 3442$ であることが確認できます。

$L(100\,000)$ を求めてください。
