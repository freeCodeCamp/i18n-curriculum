# --description--

JavaScript でよくある作業の一つに、配列の内容を繰り返し処理することがあります。その方法の一つが `for` ループです。このコードは配列 `arr` の各要素をコンソールに出力します。

```js
const arr = [10, 9, 8, 7, 6];

for (let i = 0; i < arr.length; i++) {
   console.log(arr[i]);
}
```

配列はゼロベースのインデックスを持つことを覚えておいてください。つまり配列の最後のインデックスは `length - 1` です。このループの条件は `i < arr.length` で、`i` が `length` と等しくなったときにループが終了します。この場合、最後の反復は `i === 4` で、つまり `i` が `arr.length - 1` と等しくなり `6` をコンソールに出力します。その後 `i` は `5` に増加し、`i < arr.length` が `false` になるためループが終了します。
