# --description--

순서가 중요하지 않은 항목 목록, 예를 들어 어떤 순서로든 구매할 수 있는 쇼핑 목록을 만들고 싶다면 순서 없는 목록을 사용할 수 있습니다.

순서 없는 목록은 `<ul>` 요소를 사용해 만들고, 목록 내 각 항목은 목록 항목 요소 `<li>`를 사용해 만듭니다.

순서 없는 목록의 각 항목은 글머리 기호로 시작합니다:

순서 있는 목록은 `<ol>` 요소를 사용해 만듭니다. 각 항목 역시 목록 항목 요소 `<li>`를 사용해 만듭니다. 하지만 순서 있는 목록의 각 항목은 숫자로 시작합니다:
