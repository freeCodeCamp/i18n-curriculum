# --description--

Eine weitere Möglichkeit, einen Graphen darzustellen, ist die Verwendung einer <dfn>Adjazenzmatrix</dfn>. Eine <dfn>Adjazenzmatrix</dfn> ist ein zweidimensionales (2D) Array, bei dem jedes verschachtelte Array genauso viele Elemente enthält wie das äußere Array. Anders gesagt, es handelt sich um eine Matrix oder ein Raster von Zahlen, wobei die Zahlen die Kanten darstellen.

**Hinweis**: Die Zahlen oben und links neben der Matrix sind nur Bezeichnungen für die Knoten. Innerhalb der Matrix bedeuten Einsen, dass eine Kante zwischen den Knoten der jeweiligen Zeile und Spalte existiert. Nullen bedeuten, dass keine Kante oder Beziehung besteht.

<pre>
    1 2 3
  \------
1 | 0 1 1
2 | 1 0 0
3 | 1 0 0
</pre>

Oben sehen Sie einen sehr einfachen, ungerichteten Graphen mit drei Knoten, wobei der erste Knoten mit dem zweiten und dritten Knoten verbunden ist. Unten folgt eine JavaScript-Implementierung desselben Graphen.

```js
var adjMat = [
  [0, 1, 1],
  [1, 0, 0],
  [1, 0, 0]
];
```

Im Gegensatz zu einer Adjazenzliste muss jede „Zeile“ der Matrix genauso viele Elemente enthalten wie Knoten im Graphen vorhanden sind. Hier haben wir eine 3x3-Matrix, was bedeutet, dass unser Graph drei Knoten hat. Ein gerichteter Graph sieht ähnlich aus. Unten sehen Sie einen Graphen, bei dem der erste Knoten eine Kante zum zweiten Knoten hat, und der zweite Knoten eine Kante zum dritten Knoten.

```js
var adjMatDirected = [
  [0, 1, 0],
  [0, 0, 1],
  [0, 0, 0]
];
```

Graphen können auch <dfn>Gewichte</dfn> auf ihren Kanten haben. Bisher hatten wir <dfn>gewichtlose</dfn> Kanten, bei denen nur das Vorhandensein oder Fehlen einer Kante binär ist (`0` oder `1`). Je nach Anwendung können unterschiedliche Gewichte verwendet werden.

# --instructions--

Erstellen Sie eine Adjazenzmatrix eines ungerichteten Graphen mit fünf Knoten. Diese Matrix soll in einem mehrdimensionalen Array dargestellt werden. Zwischen dem ersten und vierten Knoten, dem ersten und dritten Knoten, dem dritten und fünften Knoten sowie dem vierten und fünften Knoten bestehen Beziehungen. Alle Kantengewichte sind eins.
