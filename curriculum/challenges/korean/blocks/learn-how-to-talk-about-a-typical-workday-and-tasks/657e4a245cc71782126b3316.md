# --description--

To `go as expected`는 일이 계획이나 예상대로 진행된다는 뜻입니다. 이는 상황이 의도한 대로, 예상치 못한 문제가 없이 전개되기를 바라는 희망이나 요구를 표현할 때 자주 사용됩니다. 예를 들어:

`We hope the event goes as expected, without any disruptions.` - 그들은 이벤트가 계획대로 진행되고 아무 문제도 발생하지 않기를 바랍니다.
