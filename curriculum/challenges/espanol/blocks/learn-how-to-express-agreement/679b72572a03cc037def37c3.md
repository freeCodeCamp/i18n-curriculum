# --explanation--

`Doable` significa que algo es posible o alcanzable. A menudo se usa en conversaciones informales para expresar que una tarea o meta puede completarse. Por ejemplo:

- `Finishing this project by Friday is doable.` – Esto significa que completar el proyecto para el viernes es posible.

- `Is it doable within our budget?` – Esto pregunta si algo puede lograrse con el dinero disponible.
