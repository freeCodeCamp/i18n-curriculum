# --interactive--

Zuerst müssen Sie verstehen, wie Bilder funktionieren. Gängige Bildformate wie PNG und JPG werden als Rasterformate eingestuft. Das bedeutet im Wesentlichen, dass sie pixelbasiert sind und die Daten den Farbwert jedes Pixels speichern.

Ein großer Nachteil von rasterbasierten Bildern ist, dass sie nicht gut vergrößert werden können. Wenn Sie schon einmal versucht haben, ein PNG zu vergrößern, haben Sie vielleicht gesehen, dass es pixelig oder unscharf wird.

Ein SVG ist eine andere Art von Bild. SVG steht für skalierbare Vektorgrafik. Eine Vektorgrafik speichert Daten basierend auf Pfaden und Gleichungen, um Punkte, Linien und Kurven zu zeichnen. Das bedeutet, dass eine Vektorgrafik wie ein SVG auf jede Größe skaliert werden kann, ohne die Qualität zu beeinträchtigen.

SVGs haben speziell den zusätzlichen Vorteil, dass sie Daten in XML speichern. Das bedeutet, Sie können sie direkt in Ihrem Code als rohes HTML mit dem `svg`-Element verwenden. Es bedeutet auch, dass Sie die Farbe des Bildes programmatisch ändern können.

Um das Smiley-Gesicht rot zu färben, aktivieren Sie den interaktiven Editor und ändern Sie das `fill="yellow"` in `fill="red"`.

:::interactive_editor

```html
<svg width="100" height="100" viewBox="0 0 100 100" xmlns="http://www.w3.org/2000/svg">
  <circle cx="50" cy="50" r="45" stroke="black" stroke-width="4" fill="yellow" />
  <circle cx="35" cy="40" r="5" fill="black" />
  <circle cx="65" cy="40" r="5" fill="black" />
  <path d="M35 65 Q50 80 65 65" stroke="black" stroke-width="4" fill="transparent" />
</svg>
```

:::

Hier sind die grundlegenden Elemente für das Beispiel:

- Das `svg`-Element ist der Container für die gesamte Zeichnung. Es richtet den Bereich ein, in dem alle Formen erscheinen. Alles, was Sie mit SVG zeichnen möchten, wie Kreise, Linien oder Pfade, kommt in das `svg`-Element.
- Das `circle`-Element wird verwendet, um das Gesicht und die Augen zu erstellen. Ein großer Kreis bildet das gelbe Gesicht, und zwei kleinere Kreise bilden die Augen.
- Das `path`-Element wird verwendet, um das Lächeln zu zeichnen. Es erzeugt eine gebogene Linie für den Mund.
- Jedes SVG-Element hat Attribute, die sein Aussehen und seine Position im Zeichenbereich steuern.

Hier sind noch ein paar weitere Beispiele. Um die Farbe eines der Beispiele zu ändern, aktualisieren Sie den Wert des `fill`-Attributs auf eine beliebige benannte Farbe wie `red`, `green`, `blue`, `yellow` usw.

:::interactive_editor

Wann sollten Sie also ein SVG verwenden? Ein großartiger Anwendungsfall sind Icons. Wenn Sie benutzerdefinierte Aufzählungszeichen erstellen oder Icons zu Ihren Links hinzufügen möchten, um soziale Medienplattformen darzustellen, ist die Verwendung von SVGs der beste Weg. Eine der beliebtesten Icon-Bibliotheken, Font Awesome, verwendet SVG-Bilder für ihre Icons. SVGs sind auch ideal für Webseiten-Logos, weil sie perfekt skalieren. Sie ermöglichen es Ihnen, Ihr Layout an jedes responsive Design anzupassen, das Sie benötigen. Wenn Sie das nächste Mal ein SVG lokal haben, versuchen Sie, es mit einem Texteditor zu öffnen und mit dem Code zu experimentieren.
