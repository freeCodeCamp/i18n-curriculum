# --explanation--

`Across` wird verwendet, um etwas zu beschreiben, das über einen großen Bereich oder an vielen verschiedenen Orten passiert. Es kann eine Bewegung von einer Seite zur anderen oder etwas, das sich über eine Spanne erstreckt, anzeigen. Zum Beispiel:

- `The company has offices across the country.` – Die Büros sind an verschiedenen Standorten im Land verteilt.

- `She walked across the street.` – Sie ist von einer Straßenseite zur anderen gegangen.

`Across different screen sizes` bedeutet, dass das Problem bei vielen verschiedenen Bildschirmgrößen auftritt.
