# --description--

在互联网查找信息可能是一项艰巨的任务。网络上有如此多的网站，包含如此多的数据，似乎不可能获得你想要的结果。

但多亏了布尔查找语法，你可以将查找范围缩小到正是你需要的内容。

布尔搜索语法是一种使用特殊词汇来帮助你在在线查找时准确找到所需内容的方法。

这是一个针对 `freecodecamp curriculum` 的基本查找查询示例。

当你查找 `freecodecamp curriculum` 时，搜索引擎将返回匹配 *任意* 这些关键字的结果——通常按相关性排序（它们与更多查询内容匹配或相关的可能性）。

但是如果你只想要同时匹配两个关键字的结果呢？你可能听说过可以将查询用引号括起来，比如 `"freecodecamp curriculum"`。

但是，这个查询专门匹配 *短语* `freecodecamp curriculum`。这意味着你看到的结果将仅限于同时包含 `freecodecamp curriculum` 的网站，而不是同时包含 `freecodecamp` 和 `curriculum` 的网站。

如果你想匹配包含这两个术语但不一定作为短语相邻的站点，你可以像这样在每个术语前加上加号符号：`+freecodecamp +curriculum`

备选地，也许你对 freeCodeCamp 在课程之外的项目感兴趣。在这种情况下，你可以在术语 `curriculum` 前加上减号符号，如下所示：`+freecodecamp -curriculum`。

这将返回提到 `freecodecamp` 的结果，但排除同时提到 `curriculum` 的结果。

或者你有一个特定的网站想要查找，比如我们的新闻平台？你可以使用 `site:` 后跟你想查找的网站 URL ：`site:freecodecamp.org/news curriculum`。

此查询返回所有来自 `freecodecamp.org/news` 且包含单词 `curriculum` 的结果。

通过结合这些工具，你可以提出非常具体的查询，以获取你需要的准确信息。

例如，查询 `site:freecodecamp.org/news +python -curriculum` 会在我们的新闻网站上查找 Python 相关文章，但排除可能涉及我们课程的文章。

搜索操作符可能有所不同，因此始终重要的是查看你首选搜索引擎的文档。

但你现在应该能够从你的网页 `search` 中获得你想要的准确结果。
