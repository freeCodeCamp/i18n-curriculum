# --description--

У наведеному рівнянні $x$, $y$ та $n$ — натуральні числа.

$$\frac{1}{x} + \frac{1}{y} = \frac{1}{n}$$

Для межі $L$ визначимо $F(L)$ як кількість розв’язків, що задовольняють умову $x < y ≤ L$.

Ми можемо перевірити, що $F(15) = 4$ і $F(1000) = 1069$.

Знайдіть $F({10}^{12})$.
