# --description--

`AM` と `PM` は、日中の時間を表すために使われます。`AM` は真夜中から正午までの時間を意味します（例えば、`8:00 AM` は午前中です）。`PM` は正午から真夜中までの時間を意味します（例えば、`6:00 PM` は夕方です）。

英語では、`AM` と `PM` を使って一日を二つの期間に分けます。`AM` は `Ante Meridiem` を表し、これはラテン語で「正午前」を意味します。これは真夜中（`12:00 AM`）から正午（`12:00 PM`）までの時間を指します。ですから、`9:00 AM` と言うと午前9時を意味します。

`PM` は `Post Meridiem` を表し、これはラテン語で「正午後」を意味します。これは正午（`12:00 PM`）から真夜中直前（`11:59 PM`）までの時間を指します。ですから、`3:00 PM` と言うと午後3時を意味します。
