# --explanation--

`Scheduled` означає, що щось було заплановано або призначено відбутися у певний час. Наприклад:  

`The scheduled meeting will start at 2 PM.` - Це означає, що зустріч була запланована заздалегідь, щоб початися у вказаний час.

`Maintenance` означає дії, які виконують для підтримки чогось у робочому стані або для усунення проблем. Наприклад:  

`The team performed maintenance on the server to ensure it runs smoothly.` - Це означає, що команда працювала над підтримкою сервера в хорошому стані, виправляючи або оновлюючи його.
