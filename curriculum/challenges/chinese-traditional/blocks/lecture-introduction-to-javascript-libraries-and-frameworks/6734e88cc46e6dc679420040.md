# --description--

與處理較小的純 HTML、CSS 和 JavaScript 專案不同，啟動一個新的 React 專案包含更多步驟，以確保一切能正確執行。與其嘗試自己設定所有必要的組態，不如使用有工具可以為你完成這些工作。 

設定專案最受歡迎的工具之一是 Vite。Vite 在法文中意指「快速」，是一個建置工具，旨在為現代網頁專案提供更快速的開發體驗。Vite 可以與 React 一起使用，也可以與其他函式庫和框架如 Vue 或 Svelte 一起使用。你甚至可以將它用於純 JavaScript 專案。

要使用 Vite 創建一個新的專案，你需要使用命令列。如果你使用的是 Windows 電腦，則可以使用命令提示字元或 Windows PowerShell。如果你使用的是 Mac，則可以使用終端機應用程式。

一旦你打開命令列，你就可以使用以下指令：

```bash
npm create vite@latest my-react-app -- --template react
```

此指令使用 Vite 的 React 模板創建一個名為 `my-react-app` 的新 React 專案。

你接著可以打開新的專案，並查看 Vite 為你提供的 React 範本程式碼。

Vite 的優點在於它只會提供啟動你的 React 專案絕對必要的檔案。

要實際以原樣執行專案，你需要使用命令列中的以下指令安裝相依性：

```bash
cd my-react-app
npm install
```

`cd my-react-app` 指令確保你切換到專案所在的正確目錄。

`npm install` 指令用於安裝正確執行你的 React 專案所需的相依性。相依性是你的 React 專案為了正確運作所需的函式庫或套件，例如 React 本身、ReactDOM，或其他提供額外功能的第三方套件。

執行 `npm install` 將會讀取你專案目錄中的 `package.json` 檔案，並安裝那裡列出的所有相依性，讓你能夠在不遺漏任何必要組件的情況下建置並執行你的 React 應用程式。

`package.json` 檔案是在專案中包含關於你的專案的詮釋資料的關鍵設定檔案，包括其名稱、版本和相依性。它也定義了腳本、授權資訊以及其他有助於管理專案及其相依性的設定。

安裝完相依性後，你應該會在你的專案中看到一個名為 `node_modules` 的新資料夾。

`node_modules` 資料夾是儲存你的專案所需所有軟體包和函式庫的地方。此資料夾包含 `package.json` 檔案中列出的相依項目的實際程式碼，包括你的專案直接相依項目以及這些相依項目的任何相依項目。

要執行你的專案，執行 `npm run dev` 指令並在新的瀏覽器分頁開啟 `http://localhost:5173/`。

你應該會看到 Vite 為你提供的起始模板。

如果你需要停止區域的伺服器，請在命令列中按下 `CTRL + C`。

若要實際查看此啟動模板的程式碼，你可以進入你的專案中的 `src` 資料夾，並且你應該會看到 `App.jsx` 檔案。

從那裡你可以開始修改檔案、存檔你的變更，並在瀏覽器中看到新的變更顯示。

這樣，你就準備好開始建置你的 React 應用程式了！
