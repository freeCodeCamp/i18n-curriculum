# --explanation--

`There are` é uma frase comum usada para mostrar que algo existe ou está acontecendo.Por exemplo:  

`There are many files on the desktop.` – Isso significa que muitos arquivos existem lá.

`Issues` significa problemas ou preocupações que precisam ser corrigidos ou discutidos.Por exemplo:  

`The software has a few issues with login.` – Ele tem alguns problemas que afetam como ele funciona.
