# --description--

변의 길이가 $a=7$, $b=24$, $c=25$인 직각삼각형을 생각해보세요.

이 삼각형의 영역은 84이며, 완전수인 6과 28로 나누어집니다.

또한 $gcd(a,b) = 1$이고 $gcd(b,c) = 1$이므로 원시 직각삼각형입니다.

그리고 $c$는 완전 제곱수입니다.

다음 조건을 만족하는 직각삼각형을 완전한 직각삼각형이라고 부르겠습니다:

- 원시 직각삼각형이다
- 빗변이 완전 제곱수이다

다음 조건을 만족하는 직각삼각형을 초완전 직각삼각형이라고 부르겠습니다:

- 완전한 직각삼각형이다
- 영역이 완전수 6과 28의 배수이다

$c ≤ {10}^{16}$인 완전한 직각삼각형 중 초완전하지 않은 것은 몇 개나 존재할까요?
