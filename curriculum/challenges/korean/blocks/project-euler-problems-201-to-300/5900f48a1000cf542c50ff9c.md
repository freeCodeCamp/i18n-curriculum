# --description--

Albert가 양의 정수 $k$를 선택하면, 두 실수 $a$, $b$가 구간 [0,1]에서 균등 분포로 무작위로 선택됩니다.

그다음 ${(ka + 1)}^2 + {(kb + 1)}^2$의 제곱근을 계산하고 가장 가까운 정수로 반올림합니다. 결과가 $k$와 같으면 $k$ 점수를 얻고, 그렇지 않으면 점수를 얻지 못합니다.

예를 들어, $k = 6$, $a = 0.2$, $b = 0.85$일 때, ${(ka + 1)}^2 + {(kb + 1)}^2 = 42.05$입니다. 42.05의 제곱근은 6.484...이고 가장 가까운 정수로 반올림하면 6이 됩니다. 이것은 $k$와 같으므로 6점을 얻습니다.

$k = 1, k = 2, \ldots, k = 10$으로 10번 플레이하면, 총 점수의 기댓값은 소수점 다섯 자리까지 반올림하여 10.20914임을 보일 수 있습니다.

만약 $k = 1, k = 2, k = 3, \ldots, k = {10}^5$로 ${10}^5$번 플레이한다면, 총 점수의 기댓값을 소수점 다섯 자리까지 반올림한 값은 얼마일까요?
