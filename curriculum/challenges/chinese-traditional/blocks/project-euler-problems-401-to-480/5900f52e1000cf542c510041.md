# --description--

假循環線是由一個小圓在一個大圓內部滾動時，該小圓上一點所繪製的曲線。以原點為中心，並從最右側點開始的假循環線參數方程式為：

$$x(t) = (R - r) \cos(t) + r \cos(\frac{R - r}{r}t)$$

$$y(t) = (R - r) \sin(t) - r \sin(\frac{R - r}{r} t)$$

其中 $R$ 是大圓的半徑，$r$ 是小圓的半徑。

令 $C(R, r)$ 為半徑為 $R$ 和 $r$ 的內輪曲線上，具有整數座標且存在對應的 $t$ 值使得 $\sin(t)$ 和 $\cos(t)$ 為有理數的不同點集合。

令 $S(R, r) = \sum\_{(x,y) \in C(R, r)} |x| + |y|$ 為集合 $C(R, r)$ 中點的 $x$ 與 $y$ 座標絕對值之和。

令 $T(N) = \sum_{R = 3}^N \sum_{r=1}^{\left\lfloor \frac{R - 1}{2} \right\rfloor} S(R, r)$ 為 $S(R, r)$ 在 $R$ 和 $r$ 為正整數、且 $R \leq N$ 且 $2r < R$ 時的總和。

你被給予：

$$\begin{align}
  C(3, 1) = & \\{(3, 0), (-1, 2), (-1,0), (-1,-2)\\} \\\\
  C(2500, 1000) = & \\{(2500, 0), (772, 2376), (772, -2376), (516, 1792), (516, -1792), (500, 0), (68, 504), \\\\
                  &(68, -504),(-1356, 1088), (-1356, -1088), (-1500, 1000), (-1500, -1000)\\}
\end{align}$$

**注意：** (-625, 0) 不是 $C(2500, 1000)$ 的元素，因為對應的 $t$ 值中 $\sin(t)$ 不是有理數。

$S(3, 1) = (|3| + |0|) + (|-1| + |2|) + (|-1| + |0|) + (|-1| + |-2|) = 10$

$T(3) = 10$；$T(10) = 524$；$T(100) = 580\\,442$；$T({10}^3) = 583\\,108\\,600$。

尋找 $T({10}^6)$。
