# --interactive--

`switch` 述語和 `if/else if/else` 串鏈都是程式設計中用來根據特定條件執行不同程式碼區塊的控制元件結構。然而，它們有不同的特性和使用情境。

`switch` 述語會評估一個表達式，並將其值與一系列的 `case` 語句進行比對。當找到匹配時，會執行與該 `case` 相關的程式碼區塊。以下是 `switch` 述語的基本結構：

```js
switch (expression) {
  case value1:
    // code to be executed if expression === value1
    break;
  case value2:
    // code to be executed if expression === value2
    break;
  default:
    // code to be executed if expression doesn't match any case
}
```

每個 `case` 結尾的 `break` 述語非常重要。它告訴程式在執行完符合條件的 `case` 後離開 switch 區塊。若沒有它，程式會繼續執行後續的 case，這種行為稱為「fall-through」。

`switch` 述語通常用於當你將單一變數與多個可能的值進行比較時。當你需要針對單一變數檢查許多潛在條件時，它們特別有用。以下是一個使用 `switch` 述語來處理一週中各天的範例：

:::interactive_editor

```js
let dayOfWeek = 3; 

switch (dayOfWeek) {
    case 1:
        console.log("It's Monday! Time to start the week strong.");
        break;
    case 2:
        console.log("It's Tuesday! Keep the momentum going.");
        break;
    case 3:
        console.log("It's Wednesday! We're halfway there.");
        break;
    case 4:
        console.log("It's Thursday! Almost the weekend.");
        break;
    case 5:
        console.log("It's Friday! The weekend is near.");
        break;
    case 6:
        console.log("It's Saturday! Enjoy your weekend.");
        break;
    case 7:
        console.log("It's Sunday! Rest and recharge.");
        break;
    default:
        console.log("Invalid day! Please enter a number between 1 and 7.");
}
```

:::

當處理單一變數的多個可能值時，`switch` 述語可以更具可讀性且更簡潔。 

另一方面，`if/else if` 語句更具彈性。它們可以在每個語句中評估複雜條件和不同的變數。這使它們適用於更廣泛的情境。以下是你可能會選擇使用 `if/else` 語句而非 `switch` 語句的範例：

:::interactive_editor

```js
let creditScore = 720; 
let annualIncome = 60000; 
let loanAmount = 200000; 

let eligibilityStatus;

if (creditScore >= 750 && annualIncome >= 80000) {
    eligibilityStatus = "Eligible for premium loan rates.";
} else if (creditScore >= 700 && annualIncome >= 50000) {
    eligibilityStatus = "Eligible for standard loan rates.";
} else if (creditScore >= 650 && annualIncome >= 40000) {
    eligibilityStatus = "Eligible for subprime loan rates.";
} else if (creditScore < 650) {
    eligibilityStatus = "Not eligible due to low credit score.";
} else {
    eligibilityStatus = "Not eligible due to insufficient income.";
}

console.log(eligibilityStatus);
```

:::

在這個範例中，我們有一個人的年收入和信用分數，並檢查他們符合哪種類型的貸款資格。由於我們處理的是較複雜的邏輯評估和多個變數，因此這裡使用 `if/else` 述語比使用 `switch` 述語更好。

值得注意的是，JavaScript 中的 `switch` 述語使用嚴格比較（`===`），這表示它們不會執行型別強制轉換。這在可預測性和避免細微錯誤方面可能是一個優勢。

總結來說，雖然 `switch` 述語和 `if/else if` 串鏈都允許在你的程式碼中實現多分支邏輯，但它們各有不同的優勢。`switch` 述語擅長處理單一變數的多種可能值，而 `if/else if` 串鏈則為複雜條件提供更多彈性。兩者之間的選擇通常取決於你的程式碼的具體需求以及個人或團隊的程式碼風格偏好。
