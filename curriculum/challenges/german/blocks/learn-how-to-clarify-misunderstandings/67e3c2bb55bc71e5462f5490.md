# --explanation--

`Mentioned` bedeutet, kurz auf etwas zu verweisen oder darüber zu sprechen. Es wird oft verwendet, wenn jemand ein Thema im Gespräch anspricht. Zum Beispiel:

`She mentioned the meeting time in her email.` – Sie bezog sich auf die Besprechungszeit.

`Polymorphism` ist ein Programmierkonzept. Es bedeutet, dass eine einzelne Funktion, Methode oder ein Operator mit verschiedenen Datentypen oder Objekten verwendet werden kann. Zum Beispiel:

`In object-oriented programming, polymorphism allows a single method to behave differently based on the object that calls it.` – Hier bezieht sich `polymorphism` auf eine Methode, die je nach dem Objekt, das sie verwendet, unterschiedliche Aktionen ausführen kann. Zum Beispiel könnte dieselbe Methode `draw()` unterschiedlich reagieren, wenn sie von einem `Circle`-Objekt oder einem `Rectangle`-Objekt aufgerufen wird.
