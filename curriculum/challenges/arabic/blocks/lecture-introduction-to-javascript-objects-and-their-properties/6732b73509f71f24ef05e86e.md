# --interactive--

عند العمل مع JavaScript، ستواجه غالبًا هياكل بيانات معقدة تتضمن كائنات مسكنة ومصفوفات داخل الكائنات. تمثل هذه الهياكل بيانات غنية وهرمية، لكنها تتطلب فهمًا واضحًا لكيفية الوصول إلى البيانات والتعامل معها. دعنا نستكشف كيفية التنقل بفعالية في هذه الهياكل المسكنة.

الوصول إلى الخصائص من الكائنات المسكنة يتطلب استخدام ترميز النقطة أو ترميز الأقواس، تمامًا كما في الوصول إلى خصائص الكائنات البسيطة. لكنك ستحتاج إلى ربط هذه الطرق للوصول إلى العمق داخل الهيكل المسكن.

على سبيل المثال، لنأخذ كائنًا مسكنًا يمثل شخصًا مع معلومات الاتصال:

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};
```

للوصول إلى رقم هاتف العمل الخاص بـ `Alice`، ستربط طرق الوصول إلى الخصائص بهذا الشكل:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person.contact.phone.work); // "098-765-4321"
```

:::

يمكنك أيضًا استخدام ترميز الأقواس، وهو مفيد بشكل خاص عندما تحتوي أسماء الخصائص على مسافات أو أحرف خاصة، أو عندما تستخدم متغيرات للوصول إلى الخصائص:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person['contact']['phone']['work']); // "098-765-4321"
```

:::

الآن، دعنا نلقي نظرة على كيفية الوصول إلى بيانات حيث تكون إحدى خصائص الكائن قيمة مصفوفة. هنا كائن `person` معدل يتضمن مصفوفة من العناوين:

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};
```

إليك مثالًا على كيفية الوصول إلى مدينة عنوان العمل لـ `Alice`:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};

console.log(person.addresses[1].city); // "Workville"
```

:::

في هذا المثال، تشير `person.addresses` إلى مصفوفة العناوين. للوصول إلى العنوان الثاني في تلك المصفوفة، نستخدم ترميز الأقواس مع الفهرس `1`. ثم نستخدم ترميز النقطة للوصول إلى `city` من كائن العنوان ذلك.

فهم كيفية الوصول إلى الخصائص في الكائنات المسكنة والمصفوفات ضروري عند العمل مع هياكل بيانات معقدة. في ورش العمل والمختبرات القادمة، ستتاح لك الفرصة لممارسة التعامل مع هذه الأنواع من هياكل البيانات.
