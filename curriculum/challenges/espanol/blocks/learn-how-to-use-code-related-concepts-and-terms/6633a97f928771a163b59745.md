## --text--

¿Cuál oración describe con precisión la diferencia entre variables y constantes según la explicación de Sophie?

## --answers--

Las variables son unidades de almacenamiento flexibles, mientras que las constantes son unidades de almacenamiento seguras que no pueden alterarse.

### --feedback--

Esta opción, aunque algo precisa, no capta completamente la metáfora de llenar y vaciar versus cajas cerradas con llave que utilizó Sophie.

---

Las variables pueden almacenar múltiples tipos de datos a la vez, mientras que las constantes solo pueden almacenar un tipo.

### --feedback--

Esta sentencia es engañosa porque se enfoca en los tipos de datos en lugar de cómo se pueden alterar o permanecen fijos, que es el punto principal de la explicación de Sophie.

---

Las variables actúan como contenedores que se pueden llenar y vaciar, mientras que las constantes actúan como cajas cerradas con llave que mantienen inalterados los contenidos.

---

Las variables siempre están cambiando, y las constantes nunca se usan más de una vez.

### --feedback--

Constants are used repeatedly but their values do not change.
