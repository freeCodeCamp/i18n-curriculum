# --explanation--

`Automation`는 인간의 개입 없이 기술을 사용해 작업을 자동으로 수행하는 것입니다. 예를 들어:

`The company uses automation to schedule social media posts every day.` - 매번 수동 입력 없이 소프트웨어 도구를 사용해 게시물을 예약하는 것을 의미합니다.

`Data-driven`는 데이터 분석을 기반으로 결정을 내리는 것을 의미합니다. 예를 들어:

`The team made a data-driven decision to improve website performance based on user activity data.` - 가정이나 추측이 아닌 실제 사용자 행동 데이터에 근거해 결정했음을 나타냅니다.
