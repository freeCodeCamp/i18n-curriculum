# --description--

Nel coding, il termine `persistent` si riferisce spesso a qualcosa che continua a esistere o a verificarsi, specialmente un problema o un bug difficile da risolvere.

`Persistence` è la forma sostantivata, che indica lo stato di persistenza.

Per esempio, un `persistent error` nel codice è un errore che si presenta continuamente ed è difficile da correggere.
