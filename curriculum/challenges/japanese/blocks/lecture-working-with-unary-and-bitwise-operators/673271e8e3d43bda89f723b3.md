# --interactive--

単項演算子は、1つのオペランドに対して動作し、型変換、値の操作、特定の条件のチェックなどの操作を行います。よく使われる単項演算子とその動作を見てみましょう。

単項プラス演算子は、オペランドを数値に変換します。オペランドがすでに数値の場合は、そのまま変わりません。

:::interactive_editor

```js
const str = '42';
const strToNum = +str;

console.log(strToNum); // 42
console.log(typeof str); // string
console.log(typeof strToNum); // number
```

:::

単項プラスは、数値として扱いたいときに便利です。ご想像の通り、単項マイナス演算子もあります。これはオペランドの値を反転させます。単項プラスと似ていますが、符号を反転させる点が違います。

:::interactive_editor

```js
const str = '42';
const strToNegativeNum = -str;

console.log(strToNegativeNum); // -42
console.log(typeof str); // string
console.log(typeof strToNegativeNum); // number
```

:::

論理否定演算子は、感嘆符（`!`）で表される別の単項演算子です。オペランドのブール値を反転させます。つまり、オペランドが`true`なら`false`になり、`false`なら`true`になります。

:::interactive_editor

```js
let isOnline = true;
console.log(!isOnline); // false

let isOffline = false;
console.log(!isOffline); // true
```

:::

ビット単位の否定演算子は、あまり使われない単項演算子です。チルダの`~`で表され、数値の2進数表現を反転させます。コンピューターは数値を2進数（1と0）で保存しています。`~`演算子はすべてのビットを反転させ、1は0に、0は1に変えます。2進数とビットについては今後のレッスンで詳しく学びます。

:::interactive_editor

```js
const num = 5; // The binary for 5 is 00000101

console.log(~num); // -6
```

:::

この例では、`5`が`-6`になりました。`~`に`5`演算子を適用すると`- (5 + 1)`になり、2の補数表現により`-6`と等しくなるためです。2の補数は、コンピューターが負の数を2進数で表す方法です。ビット単位の否定は、ビットを直接操作するような低レベルのプログラミングでない限り、あまり使わないでしょう。

`void`キーワードは単項演算子で、式を評価して`undefined`を返します。

:::interactive_editor

```js
const result = void (2 + 2);

console.log(result); // undefined
```

:::

`void`は、リンクの遷移を防ぐためによく使われます。

```js
<a href="javascript:void(0);">Click Me</a>
```

最後に、前のレッスンで学んだ`typeof`演算子があります。これはオペランドの型を文字列として返します。

:::interactive_editor

```js
const value = 'Hello world';

console.log(typeof value); // string
```

:::
