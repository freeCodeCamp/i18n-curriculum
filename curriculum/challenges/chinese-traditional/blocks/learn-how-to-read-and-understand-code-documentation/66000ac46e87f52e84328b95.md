# --description--

`Interpret` 是指嘗試理解或找出某事物的意義，例如弄清楚一個困難的字、單字是什麼意思或某人想表達什麼。 例如，`He tried to interpret the instructions as best as he could.`

`Focus` 是當你特別注意某一個特定事物，而忽略其他事物。這就像透過相機觀看並調整，使只有一個物件清晰，而其他所有東西都模糊。例如，`She decided to focus on her math homework to finish it quickly.`
