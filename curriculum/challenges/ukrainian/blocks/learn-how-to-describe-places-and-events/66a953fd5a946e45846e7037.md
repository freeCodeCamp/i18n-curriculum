### --feedback--

Це складне слово, утворене поєднанням двох слів за допомогою `-`. Цей термін означає, що щось відбувається миттєво, без жодної затримки.

# --explanation--

`Real-time` означає, що щось відбувається миттєво, без жодної затримки. Наприклад:

- `We are having a real-time video chat.` — розмова відбувається миттєво, без затримки між відповідями.

`Collaboration` означає працювати разом з іншими для досягнення загальної мети. Наприклад:

- `The team collaborated to finish the project on time.` — команда працювала разом, щоб завершити проєкт.

`Real-time project collaboration` означає працювати разом над проєктом миттєво, навіть якщо учасники команди знаходяться в різних місцях. Наприклад:

- `Our team uses a shared platform for real-time project collaboration.` — команда працює разом над проєктом з негайними оновленнями та комунікацією.
