# --explanation--

`Present Perfect Continuous` 時制（`have/has been` + 動詞の`-ing`形）は、過去に始まり現在も続いているか、最近終わったばかりの動作を表すために使われます。これはしばしば活動や過程にフォーカスします。例えば：

`I have been studying the logs all morning.` – これは動作が以前に始まり、今も続いている可能性があることを意味します。
