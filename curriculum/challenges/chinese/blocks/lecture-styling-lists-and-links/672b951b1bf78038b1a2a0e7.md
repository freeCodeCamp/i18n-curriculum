# --interactive--

在 CSS 中，`list-style` 属性用于控制网页上列表的外观。

无论你是在处理有序列表（`ol`）还是无序列表（`ul`），`list-style` 属性允许你自定义列表项的显示方式。

`list-style` 属性实际上是另外三个属性的简写：

- `list-style-type`
- `list-style-position`
- `list-style-image`

每一个在定义列表外观时都扮演着不同的角色。

`list-style-type` 属性允许你定义列表中使用的项目符号或数字的类型。

对于无序列表，你可以选择多种项目符号样式，例如实心圆、空心圆或平方。

对于有序列表，你可以使用不同的编号系统，比如十进制、罗马数字，甚至字母字符。

这是使用 `list-style-type` 的示例：

:::interactive_editor

```html
<ul style="list-style-type: square;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

在此示例中，无序列表的项目符号更改为方块。

`list-style-type` 属性是这三者中最常用的，因为它直接影响你的列表中项目符号或编号样式的外观。

`list-style-position` 属性控制项目内容中项目符号或数字的位置。你可以使用两个值：`inside` 和 `outside`。

当你使用值 `outside` 时，项目符号或数字会出现在内容之外，这是默认行为。

当你使用值 `inside` 时，项目符号或数字会出现在内容内部，这可能导致文本换行并与项目符号或数字对齐。

这是使用 `list-style-position` 的示例：

:::interactive_editor

```html
<ul style="list-style-position: inside;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
<ul style="list-style-position: outside;">
  <li>Item 4</li>
  <li>Item 5</li>
  <li>Item 6</li>
</ul>
```

:::

在此示例中，为两个不同的无序列表标签提供了 `inside` 和 `outside` 值。

当你想控制列表内容的对齐方式时，`list-style-position` 属性非常有用，尤其是在单个列表项中有多行文本的情况下。

`list-style-image` 属性允许你将图像用作列表项的项目符号。这对于为你的列表添加独特的视觉风格非常有用。

这是使用 `list-style-image` 的示例：

:::interactive_editor

```html
<head>
  <style>
    ul {
      list-style-image: url('https://cdn.freecodecamp.org/platform/universal/freecodecamp-org-gravatar.jpeg');
      list-style-position: inside;
    }
  </style>
</head>
<body>
  <ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
  </ul>
</body>
```

:::

在此示例中，项目符号被替换为自定义的 freeCodeCamp 标志，为列表添加了个性化的风格。

使用 `list-style-image` 时，确保你选择的图像小且适合你网页的设计。如果图像过大或过于复杂，会使列表更难读取。

你可以将三个属性——`list-style-type`、`list-style-position` 和 `list-style-image`——合并为一个单一的 `list-style` 简写属性。

简写中值的顺序无关紧要，但三者都可以一起指定。

这是一个使用简写属性的示例：

:::interactive_editor

```html
<ul style="list-style: square inside url('https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg');">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

在此示例中，列表项使用方形项目符号，位置在内容内部，项目符号为自定义图像。

但是，如果图像不可用或未渲染，将使用平方项目符号作为备用。
