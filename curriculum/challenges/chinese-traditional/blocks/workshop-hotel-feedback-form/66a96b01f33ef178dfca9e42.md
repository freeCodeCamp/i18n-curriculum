# --description--

當你希望用戶從下拉菜單中進行選擇時，可以使用 `select` 和 `option` 元素。

下面是一個使用 `select` 和 `option` 元素創建不同城市下拉菜單的示例：

```html
<label for="city">Choose a City: </label>
<select id="city" name="city">
  <option value="new-york">New York</option>
  <option value="los-angeles">Los Angeles</option>
  <option value="chicago">Chicago</option>
  <option value="miami">Miami</option>
</select>
```

首先添加一個 `select` 元素，其 `name` 和 `id` 屬性設置爲 `"service"`。

# --hints--

你的 `fieldset` 應該有一個 `select` 元素。

```js
assert.exists(document.querySelector('fieldset:nth-of-type(4) select'));
```

你的 `select` 元素應將 `name` 屬性設置爲 `"service"`。

```js
assert.exists(document.querySelector('select[name="service"]'));
```

你的 `select` 元素應將 `id` 屬性設置爲 `"service"`。

```js
assert.exists(document.querySelector('select[id="service"]'));
```
