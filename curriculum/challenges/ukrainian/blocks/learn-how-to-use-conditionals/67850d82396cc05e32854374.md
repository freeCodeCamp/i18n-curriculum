# --explanation--

**Питання з хвостиком** — це коротке питання, додане в кінець речення, яке використовується для підтвердження інформації або запрошення до згоди. Воно утворюється так:

- Береться допоміжне дієслово (наприклад, `would`, `is`, `has`, `will`) з головного речення.

- Використовується протилежна форма: позитивна на від’ємну або від’ємна на позитивну.

- Додається відповідний займенник.

У реченні Лізи:

- Головна умова — `It would affect a lot of tech companies` (позитивна).

- Питання з хвостиком — `wouldn't it?` (від’ємна форма `would`).

Це підтверджує твердження і запрошує до згоди або зворотного зв’язку.
