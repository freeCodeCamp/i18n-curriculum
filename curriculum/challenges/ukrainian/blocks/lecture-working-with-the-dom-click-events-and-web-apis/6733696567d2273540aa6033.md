# --interactive--

API `Web Animations` (WAAPI) дозволяє створювати та керувати анімаціями безпосередньо у JavaScript. За допомогою WAAPI ви можете працювати з анімаціями більш динамічно, що полегшує їхнє маніпулювання.

Давайте детальніше розглянемо WAAPI, щоб ви могли почати працювати з анімаціями прямо у своєму JavaScript-коді. В основі WAAPI лежить конструктор `Animation`, який надає кілька методів і властивостей екземпляра, що дозволяють динамічно анімувати елементи.

Важливим методом конструктора `Animation` є `animate()`. Він дозволяє створити анімацію, задаючи ключові кадри та опції, такі як тривалість, напрямок, плавність і кількість ітерацій.

Ось базовий синтаксис методу `animate()`:

```js
element.animate(keyframes, options);
```

Розглянемо приклад.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="square" id="square"></div>
<script src="index.js"></script>
```

```css
body {
  background: #f1f1f1;
  display: flex;
  align-items: center;
  justify-content: center;
  height: 100vh;
}

.square {
  background: #1b1b32;
  width: 10rem;
  aspect-ratio: 1/1;
}
```

```js
const square = document.querySelector("#square");

const animation = square.animate(
  [{ transform: "translateX(0px)" }, { transform: "translateX(100px)" }],
  {
    duration: 2000, // makes animation lasts 2 seconds
    iterations: Infinity, // loops indefinitely
    direction: "alternate", // moves back and forth
    easing: "ease-in-out" // smooth easing
  }
);
```

:::

У браузері результатом буде синій квадрат, який рухається туди й назад по горизонталі.

Методи екземпляра конструктора `Animation` включають:

```js
play()
pause()
reverse()
finish()
cancel()
```

А властивості екземпляра включають:

```md
playbackRate
currentTime
startTime
effect
timeline
playState
finished
onfinish
oncancel
```

Давайте змінимо приклад, щоб використати методи `play()`, `pause()` і властивість `onfinish`. Ось нова версія:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="square" id="square"></div>

<button id="playBtn">Play</button>
<button id="pauseBtn">Pause</button>
<script src="index.js"></script>
```

```css
body {
  background: #f1f1f1;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  height: 100vh;
}
.square {
  background: #1b1b32;
  width: 10rem;
  aspect-ratio: 1/1;
  margin-bottom: 20px;
}
button {
  margin: 10px;
  padding: 10px 20px;
  font-size: 16px;
}
```

```js
const square = document.querySelector("#square");
const playBtn = document.querySelector("#playBtn");
const pauseBtn = document.querySelector("#pauseBtn");

const animation = square.animate(
  [{ transform: "translateX(0px)" }, { transform: "translateX(200px)" }],
  {
    duration: 5000, // Animation lasts 5 seconds
    // iterations: Infinity, // Loops indefinitely
    direction: "alternate", // Moves back and forth
    easing: "ease-in-out" // Smooth easing function
  }
);

// Set the onfinish property to log a message when the animation ends
animation.onfinish = () => {
  console.log("Animation finished!");
};

// Play the animation when the "Play" button is clicked
playBtn.addEventListener("click", () => {
  animation.play();
  console.log("You start the animation");
});

// Pause the animation when the "Pause" button is clicked
pauseBtn.addEventListener("click", () => {
  animation.pause();
  console.log("You pause the animation");
});
```

:::

У браузері буде показано синій блок, який рухається зліва направо при натисканні на кнопку `play`.

WAAPI розширює можливості анімацій CSS, надаючи більш динамічний контроль над анімаціями безпосередньо у JavaScript.

За допомогою анімацій CSS ви визначаєте анімації декларативно, використовуючи властивості, такі як `animation-name`, `animation-duration` і `animation-timing-function`. Ви також можете робити те саме, використовуючи метод `animate()` у WAAPI.

Різниця в тому, що ви можете керувати анімаціями, створеними методом `animate()`, більш безпосередньо і динамічно, тоді як у CSS-анімаціях потрібно значно більше робити, визначаючи кастомні стилі та активуючи їх у своєму JavaScript-файлі.

Анімації CSS ідеальні для простих і декларативних анімацій, які запускаються автоматично. Це включає ефекти наведення, переходи або анімації, які не потребують багато взаємодії після запуску. Якщо ваша анімація має реагувати на взаємодії користувача, такі як натискання, прокрутка, або ви хочете, щоб користувач міг ставити на паузу, змінювати напрямок чи швидкість динамічно, WAAPI — кращий вибір.

Ви можете поєднувати WAAPI і анімації CSS, коли хочете простоту CSS для базового налаштування, але потребуєте контролю анімацій під час виконання.
