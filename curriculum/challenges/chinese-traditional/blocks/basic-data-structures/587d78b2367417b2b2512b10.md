# --description--

好，那麼我們已經學會如何使用 `shift()` 和 `pop()` 從陣列的開頭和結尾移除元素，但如果我們想從中間某個位置移除元素呢？或者一次移除多個元素呢？這時候就用得上 `splice()`。`splice()` 允許我們做到這一點：**從陣列中的任何位置移除任意數量的連續元素**。

`splice()` 最多可以接受 3 個參數，但目前我們將專注於前 2 個。`splice()` 的前兩個參數是整數，表述 `splice()` 被呼叫的陣列中項目的索引或位置。請記住，陣列是*從零開始編號*的，所以要顯示陣列的第一個元素，我們會使用 `0`。`splice()` 的第一個參數表述從陣列中哪個索引開始移除元素，而第二個參數顯示要刪除的元素數量。例如：

```js
let array = ['today', 'was', 'not', 'so', 'great'];

array.splice(2, 2);
```

這裡我們從第三個元素（索引 2）開始移除 2 個元素。`array` 的值將會是 `['today', 'was', 'great']`。

`splice()` 不僅會修改它被呼叫的陣列，還會傳回一個包含被移除元素值的新陣列：

```js
let array = ['I', 'am', 'feeling', 'really', 'happy'];

let newArray = array.splice(3, 2);
```

`newArray` 的值是 `['really', 'happy']`。
