# --interactive--

تُعد طريقتا `parseFloat()` و `parseInt()` من الطرق الأساسية في JavaScript لتحويل السلاسل النصية إلى أرقام. هذه الطرق مفيدة جدًا عند التعامل مع مُدخلات المستخدم أو معالجة البيانات التي تأتي بصيغة سلسلة نصية ولكن يجب التعامل معها كقيم رقمية.

لنبدأ بـ `parseFloat()`. هذه الطريقة تُحلل معلمة سلسلة نصية وتُرجع عددًا عشريًا. صُممت لاستخراج رقم من بداية السلسلة النصية، حتى لو احتوت السلسلة على أحرف غير رقمية لاحقًا. تذكر أن الأعداد العشرية هي أرقام تحتوي على نقاط عشرية. إليك كيف تعمل `parseFloat()`:

:::interactive_editor

```js
console.log(parseFloat("3.14"));     // 3.14
console.log(parseFloat("3.14 abc")); // 3.14
console.log(parseFloat("3.14.5"));   // 3.14
console.log(parseFloat("abc 3.14")); // NaN
```

:::

كما ترى، تبدأ `parseFloat()` التحليل من بداية السلسلة النصية وتستمر حتى تواجه حرفًا لا يمكن أن يكون جزءًا من عدد عشري. إذا لم تجد رقمًا صالحًا في بداية السلسلة، تُرجع `NaN` (ليس رقمًا).

أما `parseInt()`، فهي تُحلل معلمة سلسلة نصية وتُرجع عددًا صحيحًا. مثل `parseFloat()`، تبدأ من بداية السلسلة النصية، لكنها تتوقف عند أول حرف غير رقمي. إليك كيف تعمل `parseInt()`:

:::interactive_editor

```js
console.log(parseInt("42"));       // 42
console.log(parseInt("42px"));     // 42
console.log(parseInt("3.14"));     // 3
console.log(parseInt("abc123"));   // NaN
```

:::

تتوقف `parseInt()` عن التحليل عند أول حرف غير رقمي تواجهه. بالنسبة للأعداد العشرية، تُرجع فقط الجزء الصحيح. إذا لم تجد عددًا صحيحًا صالحًا في بداية السلسلة، تُرجع `NaN`.

تتميز كلتا الطريقتين بسلوكيات جديرة بالملاحظة. تتجاهلان الفراغات البيضاء في البداية:

:::interactive_editor

```js
console.log(parseFloat("  3.14"));  // 3.14
console.log(parseInt("  42"));      // 42
```

:::

تتعاملان مع علامات الجمع والطرح في بداية السلسلة النصية:

:::interactive_editor

```js
console.log(parseFloat("+3.14"));  // 3.14
console.log(parseInt("-42"));      // -42
```

:::

يجدر بالذكر أنه رغم قوة هذه الطرق، إلا أن لها بعض القيود. على سبيل المثال، لا تتعامل مباشرة مع جميع صيغ الأرقام، مثل التدوين العلمي. للاحتياجات الأكثر تعقيدًا في التحليل، قد تحتاج إلى استخدام تقنيات أو مكتبات إضافية.

في الختام، تُعد `parseFloat()` و `parseInt()` أدوات قيمة لتحويل السلاسل النصية إلى أرقام في JavaScript. فهم كيفية عملهما وسلوكهما الخاص يسمح لك بالتعامل مع البيانات الرقمية بشكل أكثر فعالية في تطبيقاتك، خاصة عند التعامل مع مُدخلات المستخدم أو مصادر البيانات الخارجية.
