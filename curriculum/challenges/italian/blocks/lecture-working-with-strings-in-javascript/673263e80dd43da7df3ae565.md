# --interactive--

Quando lavori con le stringhe in JavaScript, può capitare di dover trovare la posizione di una specifica sottostringa all’interno di una stringa più grande.

Una sottostringa è una sequenza di caratteri che appare all’interno di una stringa più grande. Per esempio, nella stringa `hello world`, `hello` e `world` sono sottostringhe.

Per trovare la posizione di una sottostringa all’interno di una stringa, puoi usare il metodo `indexOf()`.
Il metodo `indexOf()` in JavaScript ti permette di cercare una sottostringa all’interno di una stringa.

Se la sottostringa viene trovata, `indexOf()` restituisce l’indice (o posizione) della prima occorrenza di quella sottostringa. Se la sottostringa non viene trovata, `indexOf()` restituisce `-1`, che indica che la ricerca non ha avuto successo.

Il metodo `indexOf()` prende due argomenti: il primo è la sottostringa che vuoi trovare all’interno della stringa più grande, e il secondo è una posizione di partenza opzionale per la ricerca. Se non fornisci una posizione di partenza, la ricerca inizierà dall’inizio della stringa.

In questo contesto, un **argomento** è un valore che dai a una funzione o a un metodo quando lo chiami, permettendo a quella funzione o metodo di svolgere il suo compito usando l’informazione specifica che fornisci. Imparerai di più sugli argomenti nelle lezioni future.

Ecco un esempio di utilizzo del metodo `indexOf()` per trovare la posizione della stringa `awesome`:

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("awesome!");
console.log(position); // 14
```

:::

In questo esempio, la parola `awesome` inizia all’indice `14` nella stringa `JavaScript is awesome!`, quindi il metodo `indexOf()` restituisce `14`.

Ora, vediamo cosa succede quando la sottostringa non viene trovata:

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("fantastic");
console.log(position); // -1
```

:::

Dato che la parola `fantastic` non appare nella stringa, il metodo restituisce `-1`.

Puoi anche specificare da dove iniziare la ricerca all’interno della stringa fornendo un secondo argomento a `indexOf()`. Ecco un esempio:

:::interactive_editor

```js
let sentence = "JavaScript is awesome, and JavaScript is powerful!";
let position = sentence.indexOf("JavaScript", 10);
console.log(position); // 27
```

:::

In questo caso, la ricerca di `JavaScript` inizia dopo il 10° carattere, quindi la seconda occorrenza di `JavaScript` viene trovata all’indice `27`.

È importante notare che il metodo `indexOf()` fa distinzione tra maiuscole e minuscole.

In questo esempio, il risultato sarebbe `-1` perché la lettera maiuscola `F` non viene trovata nella stringa `freeCodeCamp`.

:::interactive_editor

```js
console.log("freeCodeCamp".indexOf("F")) // -1
```

:::

Usare `indexOf()` può essere molto utile quando devi verificare se una sottostringa è presente in una stringa e determinare la sua posizione per operazioni successive.
