# --description--

通常，`with` 用來表示連接或使用。例如，`I write with a pen.` 然而，在某些句子中，`with` 會出現在它所搭配的字之後。當你想強調某人使用或處理的物件或事物時，尤其是在關於技術或工作的問題或述語中，這種情況通常會發生。

例如，在科技領域，你可能會說，`This program I work with is challenging.` 這裡，`with` 用在句尾是為了強調這個難以合作的程式。另一個例子可能是一個問題，像是，`What else is he working with?`，其中句尾的 `with` 是在詢問他正在使用的其他工具或技術。
