# --description--

フレーズ `give that a go` は、通常初めて何かを試してみて、それが好きかどうか、または成功できるかどうかを確かめることを意味します。

例えば、`I've never tried sushi before, but I'll give it a go.` 話者はこのフレーズを使って、寿司を体験してみることに対して前向きであることを示しています。
