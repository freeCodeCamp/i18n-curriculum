# --description--

Pythonの変数を扱う前に、データ型を理解することが重要です。データ型は変数が保持する値の種類を表します。例えば、数値、テキストの一部、またはアイテムのリストなどです。プログラミング言語はデータ型を使って、異なる種類の情報をどのように保存し操作するかを判断します。

PythonはJavaScriptのような動的型付け言語であり、変数の型を明示的に宣言する必要はありません。変数に代入された値に基づいて、言語がその変数のデータ型を判断します。

以下はその例です：

```python
name = 'John Doe' # Python knows this is a string
age = 25 # Python knows this is an integer
```

これは、C#、Java、C++のような静的型付け言語とは対照的です。これらの言語では、変数に型を宣言する必要があります。例えば：

```csharp
string name = 'John Doe'
int age = 25
```

Pythonの動的型付けの性質は、コーディングを非常に速く柔軟にしますが、型エラーはプログラムの実行時にしか検出されないため、予期しないバグが発生することがあります。

Pythonはプログラムの**実行中**にデータ型を判断するため、型に関する間違いはその時点で初めて発見されます。プログラムが実行されると、Pythonはコードを1行ずつ実行します。もし特定のオブジェクトが期待される動作をできない行に達すると、Pythonは停止してエラーを表示します。

一方で、いくつかの言語はプログラムを**コンパイル**してから実行します。コンパイルとは、コンピューターが事前にコードをチェックし、実行の準備をすることです。この段階で、これらの言語はプログラムが始まる前に型エラーを検出できます。

これらの言語をまだ学ぶ必要はありません。重要なのは以下の点です：

- **Pythonでは型エラーはプログラムの実行中に現れる**ため、コードが実際に動いているときに発覚します。

- **コンパイル言語はコンパイル段階で型エラーを検出する**ため、プログラムが実行される前に問題がわかります。

このため、Pythonではプログラムが特定のコード行に達するまで型の間違いに気づかない場合があります。

Pythonでよく使う一般的なデータ型は以下の通りです：

- 整数：小数点のない整数、例えば`10`や`-5`。

```python
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- 浮動小数点数：小数点を含む数値、例えば`4.41`や`-0.4`。

```python
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- 文字列：シングルクォーテーションまたはダブルクォーテーションで囲まれた文字のシーケンス、例えば`'Hello world!'`。

```python
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- ブール値：真または偽の型で、`True`や`False`と書きます。

```python
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- セット：順序のない一意の要素の集合、例えば`{0.5, 4, 'apple'}`。

```python
my_set_var = {7, 'hello', 8.5}
print('Set:', my_set_var) # Set: {7, 'hello', 8.5}
```

- 辞書：波括弧で囲まれたキーと値のペアの集合、例えば`{'name': 'John Doe', 'age': 28}`。

```python
my_dictionary_var = {'name': 'Alice', 'age': 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- タプル：丸括弧で囲まれた変更不可能な順序付きの集合、例えば`('apple', 4.5, 7)`。

```python
my_tuple_var = (7, 'hello', 8.5)
print('Tuple:', my_tuple_var) # Tuple: (7, 'hello', 8.5)
```

- レンジ：ループでよく使われる数値のシーケンス、例えば`range(5)`。

```python
my_range_var = range(5)
print('Range:', my_range_var) # Range: range(0, 5)
```

- リスト：異なるデータ型を含められる順序付きの要素の集合。

```python
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- None：値が存在しないことを表す特別な値。

```python
my_none_var = None
print('None:', my_none_var) # None: None
```

変数のデータ型を取得するには、`type()`関数を使います：

```python
my_var_1 = 'Hello world'
my_var_2 = 21

print(type(my_var_1)) # <class 'str'>
print(type (my_var_2)) # <class 'int'>
```

以下はこのレッスンで扱うすべてのデータ型と、ターミナルでの型の表示例です：

```python
my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 'hello', 8.5}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 'hello', 8.5)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>
```

組み込みの`isinstance()`関数を使うと、変数が特定のデータ型に一致するかどうかを確認できます。この関数はオブジェクトとチェックしたい型を受け取り、ブール値を返します。以下はその例です：

```python
isinstance('Hello world', str) # True
isinstance(True, bool) # True
isinstance(42, int) # True
isinstance('John Doe', int) # False
```
