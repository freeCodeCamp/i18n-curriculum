# --description--

プログラミングにおける `constant` は変数に似ていますが、一度値が代入されると、その値を変更できません。`Constants` はプログラム全体で変更する必要がないと分かっている値に使われます。変数と区別するために、大文字で書かれることが多いです。

例えば、`const PI = 3.14;` 多くのプログラミング言語では、`const` を使って定数を宣言します。ここで、`PI` は値 `constant` が代入された `3.14` です。
