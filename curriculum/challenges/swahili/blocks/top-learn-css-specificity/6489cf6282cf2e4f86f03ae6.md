# --description--

```css
/* rule 1 */
#subsection .list {
  background-color: yellow;
  color: blue;
}

/* rule 2 */
#subsection .main .list {
  color: red;
}
```

Katika mfano huu wa mwisho, sheria zote mbili zinatumia vichaguzi vya ID na darasa la vitu, kwa hivyo hakuna sheria inayotumia kichaguzi maalum zaidi kuliko kingine. Mfululizo basi unakagua idadi ya kila aina ya kichaguzi. Sheria zote mbili zina kichaguzi kimoja cha ID tu, lakini sheria ya 2 ina vichaguzi vingi vya darasa la vitu, hivyo sheria ya 2 ina umaalum zaidi!

Ingawa tamko la `color: red` lingechukua kipaumbele, tamko la `background-color: yellow` bado lingetumika kwa sababu hakuna tamko linalopingana nalo.
