# --description--

في الدرس السابق، أنشأت مجلد مشروع وثبت Express:

```bash
mkdir my-app
cd my-app
npm init -y
npm install express
```

كتذكير، هذه الأوامر تنشئ مجلد مشروع جديد، وتنتقل إلى مجلد المشروع باستخدام `cd`، وتُهيئ المشروع وتُنشئ ملف `package.json` أساسي، وتثبت Express.

بعد ذلك، افتح محرر شفرة، وأنشئ ملفًا باسم `index.js`، واكتب أول خادم Express لك:

```javascript
const express = require("express");
const app = express();
const port = 3000;

app.get("/", (req, res) => {
  res.send("Hello World!");
});

app.listen(port, () => {
  console.log(`Example app listening on port ${port}`);
});
```

يمكنك تشغيل الشفرة باستخدام الأمر التالي في الطرفية:

```bash
node index.js
```

افتح متصفحك على [http://localhost:3000](http://localhost:3000) وسترى الرسالة **"Hello World!"**

هكذا بسرعة تبدأ مع Express! الآن دعنا نستعرض الشفرة خطوة بخطوة.

في السطر الأول من الشفرة، تقوم بتحميل مكتبة Express وتعيينها إلى متغير يسمى `express`:

```javascript
const express = require("express");
```

المتغير `express` هو دالة مصنع. عند استدعائها، تنشئ وتُرجع نسخة جديدة من تطبيق Express:

```javascript
const app = express();
```

الآن لدى `app` وصول إلى جميع الطرق التي توفرها Express، مثل `.get()`، `.post()`، `.listen()`، وهكذا. هذا هو كائن الخادم الرئيسي الخاص بك.

السطر التالي يعرّف المتغير `port`. المنفذ هو نقطة نهاية شبكة يستمع إليها التطبيق، و`3000` هو منفذ شائع الاستخدام لتطوير محلي:

```javascript
const port = 3000;
```

الآن، لنلق نظرة على الجزء التالي هنا:

```javascript
app.get("/", (req, res) => {
  res.send("Hello World!");
});
```

هذا الجزء من الشفرة يعرّف مسارًا ويخبر الخادم كيف يستجيب للطلبات. `app.get()` يسجل معالِجًا لطلبات HTTP من نوع GET. طلب GET هو طريقة HTTP شائعة لاسترجاع البيانات.

المعلمة `"/"` داخل `app.get()` تمثل مسار المسار. في هذه الحالة، تشير إلى عنوان URL الجذر للتطبيق.

الجزء `(req, res) => { ... }` هو دالة استدعاء راجع تُنفذ عندما يتطابق طلب مع هذا المسار. كائن `req` يحتوي معلومات عن الطلب الوارد، وكائن `res` يُستخدم لإرسال استجابة إلى العميل.

أخيرًا، `res.send("Hello World!")` يرسل استجابة إلى العميل. في هذا المثال، الاستجابة هي النص العادي `"Hello World!"`.

الآن، لنلق نظرة على آخر جزء من الشفرة هنا:

```javascript
app.listen(port, () => {
  console.log(`Example app listening on port ${port}`);
});
```

طريقة `app.listen()` تبدأ تطبيق Express وتخبر Node بالاستماع لطلبات HTTP الواردة على المنفذ المحدد، في هذه الحالة، `3000`.

بمجرد بدء الخادم، تُنفذ دالة الاستدعاء الراجع وتُسجل الرسالة `Example app listening on port 3000` في وحدة التحكم، مؤكدة أن تطبيق Express بدأ بنجاح وجاهز لاستقبال الطلبات.
