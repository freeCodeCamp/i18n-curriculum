# --instructions--

У цьому завданні потрібно створити модель машинного навчання, яка класифікуватиме SMS-повідомлення як «ham» або «spam». Повідомлення «ham» — це звичайне повідомлення від друга. Повідомлення «spam» — це реклама або повідомлення від компанії.

Ви маєте створити функцію з назвою `predict_message`, яка прийматиме рядок повідомлення як аргумент і повертатиме список. Перший елемент списку має бути числом від нуля до одного, що вказує на ймовірність «ham» (0) або «spam» (1). Другий елемент списку має бути словом «ham» або «spam», залежно від того, що ймовірніше.

Для цього завдання ви використаєте набір даних <a href="http://www.dt.fee.unicamp.br/~tiago/smsspamcollection/" target="_blank" rel="noopener noreferrer nofollow">SMS Spam Collection</a>. Набір даних уже розділено на тренувальні та тестові дані.

Перші дві клітинки імпортують бібліотеки та дані. Остання клітинка тестує вашу модель і функцію. Додайте свій код між цими клітинками.
