# --explanation--

`Unexpected behavior` refere-se a resultados que diferem do que foi pretendido. Isso pode incluir erros, cálculos incorretos ou pular casos importantes. Alguns exemplos de comportamento inesperado em loops são:

- `Incorrect output` - O programa executa mas fornece resultados incorretos.

- `Infinite repetition` - O loop executa para sempre.

- `Unexpected termination` - O loop pode causar um erro que interrompe a execução.

- `System crash` - Raro, mas possível com problemas relacionados à memória.
