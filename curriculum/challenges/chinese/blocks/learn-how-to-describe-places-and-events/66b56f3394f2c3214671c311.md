`To roll out` 意味着推出或引入某个新事物，例如产品、服务或项目。它通常用于商业和技术上下文中。例如：
 
- 该公司计划在下个月推出一个新建 `software` 更新。

`By the end of` 用于设置截止日期或指定某事应完成的时间。例如：

- `I need to finish this report by the end of the day.` - 报告应在当天结束前完成。
