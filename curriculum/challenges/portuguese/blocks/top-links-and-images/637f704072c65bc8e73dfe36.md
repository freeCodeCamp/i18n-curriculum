## Elementos de âncora
Para criar um link em HTML, use o elemento de âncora. Um elemento de âncora é definido agregando o texto ou outro elemento do HTML que você que seja um link à tag `<a>`.

Adicione o seguinte ao `body` da página `index.html` que você criou e abra a página no navegador:

```html
<a>click me</a>
```

Você pode ter notado que clicar nesse link não faz nada. Isso acontece porque uma tag de âncora sozinha não sabe para onde você quer linkar. Você precisa informar um destino para onde ela deve ir. Você faz isso usando um atributo HTML.

Um atributo HTML fornece informações adicionais para um elemento HTML e sempre fica na tag de abertura do elemento. Um atributo geralmente é composto por duas partes: um nome e um valor; no entanto, nem todos os atributos exigem um valor. No seu caso, você precisa adicionar um atributo `href` (referência de hyperlink) à tag de abertura da âncora. O valor do atributo `href` é o destino para onde você quer que seu link vá.

Adicione o seguinte atributo `href` ao elemento âncora que você criou anteriormente e tente clicar nele novamente, não esqueça de atualizar o navegador para que as novas mudanças sejam aplicadas.

```html
<a href="https://www.theodinproject.com/about">click me</a>
```

Por padrão, qualquer texto encapsulado com uma tag de âncora sem um atributo `href` parecerá texto simples. Se o atributo `href` estiver presente, o navegador dará ao texto uma cor azul e vai sublinhá-lo para representar um link.

Vale lembrar que você pode usar tags de âncora para linkar para qualquer tipo de recurso na internet, não apenas outros documentos HTML. Você pode linkar para vídeos, arquivos pdf, imagens e assim por diante, mas na maioria das vezes, você estará linkando para outros documentos HTML.
