# --description--

Für ein $n$-Tuple ganzer Zahlen $t = (a_1, \ldots, a_n)$ seien $(x_1, \ldots, x_n)$ die Lösungen der Polynomgleichung $x^n + a_1x^{n - 1} + a_2x^{n - 2} + \ldots + a_{n - 1}x + a_n = 0$.

Betrachten Sie die folgenden zwei Bedingungen:

- $x_1, \ldots, x_n$ sind alle reell.
- Wenn $x_1, ..., x_n$ sortiert sind, gilt $⌊x_i⌋ = i$ für $1 ≤ i ≤ n$. ($⌊·⌋:$ Abrundungsfunktion.)

Im Fall von $n = 4$ gibt es 12 $n$-Tuple ganzer Zahlen, die beide Bedingungen erfüllen.

Wir definieren $S(t)$ als die Summe der Absolutwerte der ganzen Zahlen in $t$.

Für $n = 4$ können wir überprüfen, dass $\sum S(t) = 2087$ für alle $n$-Tuple $t$ gilt, die beide Bedingungen erfüllen.

Finden Sie $\sum S(t)$ für $n = 7$.
