# --interactive--

`toString()` メソッドは、JavaScriptの基本的な機能で、値をその文字列表現に変換します。これは数値、ブール値、配列、オブジェクトに使えるメソッドです。`toString()` の最も一般的な使い方の一つは、数値を文字列表現に変換することです。例を見てみましょう。

:::interactive_editor

```js
const num = 10;
console.log(num.toString()); // "10"
```

:::

このメソッドはオプションの基数を受け取ります。基数は`2`から`36`までの数字で、基数は2進数の基数2や8進数の基数8のように基数を表します。基数が指定されない場合はデフォルトで10進数の基数10になります。`2` を`toString()` メソッドの引数として渡す例を示します。

:::interactive_editor

```js
const num = 10;
console.log(num.toString(2)); // "1010"
```

:::

結果は`1010` となり、これは10進数の`10` の2進数表現です。

配列やオブジェクトを文字列に変換するために`toString()` メソッドも使えます。配列は`toString()` のカスタム実装を持っており、各要素を文字列に変換してカンマで結合します。例を見てみましょう。

:::interactive_editor

```js
const arr = [1, 2, 3];
console.log(arr.toString()); // "1,2,3"
```

:::

この例では、配列のすべての要素が結合されて文字列`1,2,3` になります。

オブジェクトに`toString()` メソッドを使うと、結果はオブジェクトのプロパティを文字列化したものにはなりません。

:::interactive_editor

```js
const person = {
  name: "John",
  age: 30,
  isStudent: true
};

console.log(person.toString()); // "[object Object]"
```

:::

この例では、結果はオブジェクトのデフォルトの文字列表現である`[object Object]` になります。`person` オブジェクトのプロパティを文字列化したい場合は、将来のレッスンで学ぶ`JSON.stringify()` を使う必要があります。

まとめると、`toString()` メソッドは値を文字列に変換するために使われます。異なるデータ型での動作や、自分のオブジェクト用にカスタマイズする方法を理解することで、JavaScriptアプリケーションでのデータ操作や表示能力が大きく向上します。
