# --explanation--

`Supposed to be` 意味着某事被期望或计划发生。例如：

`The new feature was supposed to be ready by last week.` — 这意味着它计划在上周准备好。

`Coming later` 意味着某事计划在未来发生。例如：

`The update is coming later this month。` — 这意味着更新将在未来某个时间发布，而不是现在。
