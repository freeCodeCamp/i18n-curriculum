# --interactive--

Специфічність CSS — це базове поняття, яке визначає, які стилі застосовуються до елемента, коли можуть застосовуватися кілька правил.

Розуміння специфічності допомагає розробникам вирішувати конфлікти між різними CSS-правилами і гарантує, що бажані стилі застосовуються послідовно.

Специфічність CSS обчислюється на основі типу використаних селекторів.

Найвища специфічність належить рядковим стилям, які застосовуються безпосередньо до елемента через атрибут `style`.

У цьому прикладі перший абзац буде червоним, тоді як інші елементи `p` будуть синіми. Це тому, що рядкові стилі мають вищу специфічність, ніж селектори типу, як-от селектор `p`, показаний у файлі `styles.css`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p style="color: red;">Red paragraph</p>
<p>Other paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p {
  color: blue;
}
```

:::

Далі селектори ідентифікаторів мають високий рівень специфічності.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Red paragraph</p>
<p>Other paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

p {
  color: blue;
}
```

:::

Потім вступають у гру селектори класів, селектори атрибутів і псевдокласи.

Прикладами є селектори класів, такі як `.container` і `.button`, селектори атрибутів, як `[type="text"]`, і псевдокласи, наприклад `:hover`. Вони мають помірний рівень специфічності.

**ПРИМІТКА**: Ви дізнаєтеся більше про псевдокласи у майбутніх уроках.

У цьому прикладі перший абзац буде червоним, бо селектор `id` має вищу специфічність, ніж класи та селектори типу. Елементи `.example-para` матимуть зелений колір, а решта абзаців — синій.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p class="example-para">Other paragraph</p>
<p class="example-para">Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

.example-para {
  color: green;
}

p {
  color: blue;
}
```

:::

Селектори типу, такі як `div` і `h1`, а також псевдоелементи, як `::before` і `::after`, мають нижчу специфічність порівняно з попередніми групами.

**ПРИМІТКА**: Ви дізнаєтеся більше про псевдоелементи у майбутніх уроках.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p>Other paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

p {
  color: blue;
}
```

:::

Нарешті, універсальний селектор, представлений зірочкою `*`, має найнижчу специфічність.

Ось приклад встановлення кольору для всіх елементів у червоний за допомогою селектора `*`. Проте ви не побачите червоних елементів, бо існують селектори ідентифікаторів і типу, які заміщають ці стилі, що підкреслює низьку специфічність селектора `*`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p>Other paragraph</p>
<p>Yet another paragraph</p>
```

```css
* {
  color: red;
}

#para {
  color: green; 
}

p {
  color: blue;
}
```

:::

Значення специфічності обчислюються як чотиричастинне значення `(a, b, c, d)`:

- `a`: Рядкові стилі (`1` або `0`).
- `b`: Кількість селекторів ідентифікаторів.
- `c`: Кількість селекторів класів, селекторів атрибутів і псевдокласів.
- `d`: Кількість селекторів типу, псевдоелементів і універсальних селекторів.

Кожна частина значення специфічності має різну вагу:

- **Рядкові стилі** (`a`) мають найвищу вагу, додаючи значення `1` до першої частини специфічності.
    
- **Селектори ідентифікаторів** (`b`) мають наступну за величиною вагу, кожен ідентифікатор додає `1` до другої частини специфічності.
    
- **Селектори класів, селектори атрибутів і псевдокласи** (`c`) мають помірну вагу, кожен додає `1` до третьої частини специфічності.
    
- **Селектори типу і псевдоелементи** (`d`) мають найнижчу вагу, кожен додає `1` до четвертої частини специфічності.
    
- **Універсальний селектор** (`*`): Універсальний селектор додає `0` до обчислення специфічності і не впливає на специфічність. Його включення в селектор не змінює значення специфічності.

Рядковий CSS має найвищу специфічність, бо застосовується безпосередньо до елемента. Він заміщує будь-який внутрішній або зовнішній CSS. Значення специфічності для рядкових стилів — `(1, 0, 0, 0)`.

Ось ще один приклад рядкового стилю:

:::interactive_editor

```html
<p style="color: red;">This text is red.</p>
```

:::

Внутрішній CSS визначається всередині елемента `style` у розділі `head` HTML-документа. Він має нижчу специфічність, ніж рядкові стилі. Внутрішній і зовнішній CSS мають однаковий рівень специфічності — специфічність визначається селекторами, а не тим, де CSS визначено. Якщо два правила мають однакову специфічність, перемагає те, що йде пізніше в документі. Тож якщо зовнішній файл стилів підключено після внутрішнього блоку `style`, перевага буде за зовнішніми правилами. Якщо раніше — виграють внутрішні.

Значення специфічності для внутрішніх стилів визначається селекторами. Наприклад, селектор ідентифікатора у внутрішньому CSS має значення специфічності `(0, 1, 0, 0)`.

:::interactive_editor

```html
<head>
  <style>
    #text {
      color: blue;
    }
  </style>
</head>
<body>
  <div id="text">This text is blue.</div>
</body>
```

:::

У цьому прикладі текст буде синім, якщо не застосовано рядковий стиль або більш специфічний селектор.

Зовнішній CSS підключається через елемент `link` у розділі `head` і записується у окремих файлах `.css`. Як і внутрішній CSS, специфічність зовнішніх стилів визначається селекторами. Зовнішній CSS забезпечує найкращу підтримуваність для більших проєктів.

Значення специфічності для зовнішніх стилів також визначається селекторами. Наприклад, селектор класу у зовнішньому CSS має значення специфічності `(0, 0, 1, 0)`.

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <div class="text">This text's color is defined in an external CSS file.</div>
</body>
```

```css
.text {
  color: purple;
}
```

:::

У цьому прикладі колір тексту визначено у файлі `styles.css` і він буде застосований, якщо не буде заміщений більш специфічним селектором або рядковим стилем.
