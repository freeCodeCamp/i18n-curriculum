# --description--

`Would you like...?` é frequentemente usado ao propor ou sugerir uma atividade. Por exemplo:

`Would you like to test the new app with me?` - É um convite educado para trabalharmos juntos no teste.

Quando Linda diz `Would you like to go cycling with me one of these days?`, é parecido com dizer "Ei, vamos andar de bicicleta juntos."
