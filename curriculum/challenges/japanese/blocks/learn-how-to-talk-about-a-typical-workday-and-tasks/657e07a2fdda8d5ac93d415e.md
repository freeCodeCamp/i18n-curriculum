# --description--

`then` のような言葉を使って、物事が起こる順序について話すことができます。

`Then` は、何かが終わった後の次のステップや行動を示すためによく使われます。例えば：

`First, I check my emails, then I start coding.` - これは、その人がコードを書き始める前にメールを確認することを意味します。

これにより、一連の行動やイベントが時間的にどのようにつながっているかを理解しやすくなります。
