# --description--

로봇은 1/5 원호(72°)씩 연속해서 움직이며, 각 단계마다 시계 방향 또는 반시계 방향 원호를 자유롭게 선택할 수 있지만 제자리에서 회전하지는 않습니다.

북쪽을 향해 시작하는 25개의 원호로 이루어진 70932가지 가능한 닫힌 경로 중 하나는 다음과 같습니다.
