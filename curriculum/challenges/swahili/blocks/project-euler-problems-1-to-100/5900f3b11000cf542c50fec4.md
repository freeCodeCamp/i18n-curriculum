# --description--

Kitendakazi cha Totient cha Euler, ${\phi}(n)$ (mara nyingine huitwa kitendakazi cha phi), hutumika kubaini idadi ya nambari zilizo chini ya `n` ambazo ni namba msingi na `n`. Kwa mfano, kwa kuwa 1, 2, 4, 5, 7, na 8, zote ni chini ya tisa na ni namba msingi na tisa, ${\phi}(9) = 6$.

<div style='margin-left: 4em;'>

| $n$ | $\text{Namba msingi}$ | $\displaystyle{\phi}(n)$ | $\displaystyle\frac{n}{{\phi}(n)}$ |
| --- | ------------------------- | ------------------------ | ---------------------------------- |
| 2   | 1                         | 1                        | 2                                  |
| 3   | 1,2                       | 2                        | 1.5                                |
| 4   | 1,3                       | 2                        | 2                                  |
| 5   | 1,2,3,4                   | 4                        | 1.25                               |
| 6   | 1,5                       | 2                        | 3                                  |
| 7   | 1,2,3,4,5,6               | 6                        | 1.1666...                          |
| 8   | 1,3,5,7                   | 4                        | 2                                  |
| 9   | 1,2,4,5,7,8               | 6                        | 1.5                                |
| 10  | 1,3,7,9                   | 4                        | 2.5                                |

</div>

Inaonekana kuwa `n` = 6 hutoa $\displaystyle\frac{n}{{\phi}(n)}$ kubwa zaidi kwa `n` ≤ 10.

Tafuta thamani ya `n` ≤ `limit` ambayo $\displaystyle\frac{n}{{\phi(n)}}$ ni kubwa zaidi.

```js
function totientMaximum(limit) {
  function getSievePrimes(max) {
    const primesMap = new Array(max).fill(true);
    primesMap[0] = false;
    primesMap[1] = false;
    const primes = [];
    for (let i = 2; i < max; i = i + 2) {
      if (primesMap[i]) {
        primes.push(i);
        for (let j = i * i; j < max; j = j + i) {
          primesMap[j] = false;
        }
      }
      if (i === 2) {
        i = 1;
      }
    }
    return primes;
  }

  const MAX_PRIME = 50;
  const primes = getSievePrimes(MAX_PRIME);
  let result = 1;

  for (let i = 0; result * primes[i] < limit; i++) {
    result *= primes[i];
  }
  return result;
}
```
