# --explanation--

**序数** 表示某物在序列中的位置，例如步骤、任务或阶段。示例包括：`first`、`second`、`third`、`fourth` 等。

要形成大多数序数词，你需要取 **基数词**（1、2、3、4……）并添加正确的结尾。大多数序数词遵循添加 `-th` 的模式，比如 4 变成 `fourth`，6 变成 `sixth`。但 1、2 和 3 是不规则的：1 变成 `first`，2 变成 `second`，3 变成 `third`。

在对话框中，`the first task` 意味着它是在初始阶段需要完成的第一个动作。
