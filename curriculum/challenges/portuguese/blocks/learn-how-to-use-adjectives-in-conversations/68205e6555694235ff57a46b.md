Esta frase de duas palavras refere-se à quantidade de tempo necessária para preparar ou instalar algo para que possa começar a funcionar.

# --explanation--

`Minimal` significa apenas uma quantidade muito pequena. Apenas o suficiente para atender a uma necessidade.Por exemplo:

`The task required minimal effort.` – Isso significa que foi muito fácil e não exigiu muita energia.

`Setup time` é o tempo que leva para preparar algo para uso.Por exemplo:

`The software has a short setup time.` – Isso significa que não leva muito tempo para deixá-lo pronto para uso.
