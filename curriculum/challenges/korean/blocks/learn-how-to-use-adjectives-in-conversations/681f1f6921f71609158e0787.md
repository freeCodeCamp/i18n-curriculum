# --explanation--

`Specify`는 무언가를 명확하게 말하거나 자세히 설명하는 것을 의미합니다. 예를 들어:

`Please specify the time and place of the meeting.` – 회의가 언제 어디서 열리는지 정확히 말하는 뜻입니다.

`What makes it`는 명사절입니다. 문장에서 명사처럼 작용하는 단어들의 집합으로, 어떤 것의 이유나 원인을 설명합니다. 예를 들어:

`She described what makes it so effective.` – 그녀가 어떤 것이 효과적인 이유를 설명했다는 뜻입니다. 여기서 절 `what makes it so effective`는 동사 `described`의 목적어 역할을 합니다.
