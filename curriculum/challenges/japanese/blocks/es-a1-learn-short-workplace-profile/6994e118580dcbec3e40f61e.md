# --description--

スペイン語では、他人のメールアドレスについて話すときに `su`（「彼の」または「彼女の」）を使います。

メールアドレスには、知っておくべきいくつかの重要な記号があります：

- `Correo` - メール。

- `@` - `Arroba` - アットマーク。

- `.` - `Punto` - ドット／ピリオド。

`Correo electrónico` はメールを正式かつ完全に指す言い方ですが、文脈によっては `correo` と言うだけで十分な場合が多いです。

ルナはカルロスのメールアドレスについて話しています。記号の発音練習にフォーカスしてください。
