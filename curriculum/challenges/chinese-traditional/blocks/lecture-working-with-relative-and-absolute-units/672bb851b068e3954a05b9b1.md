# --interactive--

使用 `calc()` 函式（程式），你可以直接在你的樣式表中執行計算，以動態決定屬性值。這表示你可以透過根據視窗大小或其他元素計算尺寸，來創建靈活且響應式的使用者介面。

`calc()` 是一個 CSS 函式（程式）。當你開始學習 JavaScript 時，你會學到更多關於函式（程式）的知識，但在本課程中，你將學習理解 `calc()` 運作所需的基礎知識。

函式（程式）是一個執行特定任務的程式碼區塊。有些函式（程式）已經在 CSS 中定義好，所以你可以直接使用它們，並傳遞任何必要的值來自訂它們執行任務的方式。

在程式設計的世界中，當我們執行由函式所執行的任務時，我們稱之為「呼叫」該函式。我們傳入函式的值稱為引數。 

如你在以下程式碼中所見，要呼叫一個函式，請寫出它的名稱，後面接著用小括弧包住以逗號分隔的引數。函式名稱與開小括弧之間不應有空格：

```css
function(argument1, argument2, argument3)
```

一個函式（程式）可能只需要一個值來知道該做什麼。在那種情況下，它只會接受一個引數。`calc()` 函式（程式）就是這樣。它接受一個引數，因為它需要知道要計算什麼。

為此，你傳遞一個稱為表達式的東西作為引數。表達式是由值和運算子組合而成，並產生結果。

這是你如何呼叫 `calc()` 函式（程式）。你寫下名稱 calc，接著是小括弧，在小括弧內，你寫下表達式。

```css
calc(expression)
```

該表達式會被計算以求得最終結果。「計算」只是表示值和運算子在幕後被轉換成單一值。結果會被指定到進行計算的 CSS 屬性。

你可以對表述長度、角度、時間、百分比、數字和顏色的值進行計算。你也可以將不同單位如像素、百分比和 em 結合起來。

對於數字，表達式中所有的值，也稱為運算元，必須有相應的單位，例如 `px`、`em` 和百分比（`%`）。根據運算子，不同的運算元可能會有不同的單位。

你可以在表達式中使用加法（`+`）、減法（`-`）、乘法（`*`）和除法（`/`）運算子。

如果有多個運算元和運算子，`calc()` 將遵循標準的運算子優先規則。你也可以為你的操作行為添加小括弧以建立操作順序（如果需要）。

在下面的範例中，你可以看到一個帶有文字 `Hello, World!` 的 `div`。

使用 CSS 型別選擇器選取 `div`，你可以將它設計為白色文字和深藍色背景： 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>Hello, World!</div>
```

```css
div {
  color: white;
  background-color: #1b1b32;
  width: calc(50% - 20px);
}
```

:::

這裡的新功能是寬度是動態計算的。注意我們如何呼叫 `calc()` 函式並將一個表達式作為引數傳遞。該表達式有兩個不同單位的運算元和一個運算子，減法運算子。

百分比是一種相對單位。該值（`50%`）將由父容器的寬度決定。然後，從該值中減去 `20px`。此表達式的結果將決定 `div` 的寬度。

`div` 的寬度約為其容器總寬度的一半，如果你調整父容器大小，寬度將會自動重新計算。

這就是使用 `calc()` 的主要優勢。你可以根據應用程式或視窗不同的面向動態決定 CSS 屬性的值。

如果你需要在計算中使用，該表達式也可以包含 CSS 函式（程式）和變數。你將在接下來的課程中學習更多關於 CSS 變數的內容。

很好。既然你已經了解了 `calc()` 函式（程式）的基礎，現在讓我們涵蓋一些最佳實踐。

首先，你必須在加法 (`+`) 和減法 (`-`) 運算子兩側加上空白。

例如，下面的表達式不是有效的表達式，因為減法運算子緊接在第二個運算元之前。

```css
calc(100% -30px)
```

減法 (`-`) 運算子必須被空白字元包圍，如此一來。添加空白字元將會創建有效的表達式。

```css
calc(100% - 30px)
```

這對於乘法和除法運算子不是必要的，但強烈建議這麼做。

如果你需要執行計算並將那些結果用於其他計算，你也可以巢狀使用 `calc()` 函式呼叫。

此外，如果你使用值零來表述你傳入 `calc()` 函式的表達式中的長度，必須包含單位。例如，這個表達式將無效：

```css
calc(100% - 0)
```

你需要為你的單位添加單位，例如 px。

```css
calc(100% - 0px)
```

你也應該知道，目前如果你使用乘法或除法運算子，其中一個運算元必須是無單位的。對於除法運算子，特別是右邊的運算元必須是無單位的。這將不是一個有效的表達式，因為兩個運算元都有單位（像素）。其中一個運算元，不論是 5 還是 50，必須是無單位的：

```css
calc(5px * 50px)
```

你需要在其中一個中省略單位。這兩種替代方案都是有效的：

```css
calc(5 * 50px)
calc(5px * 50)
```

這是一個使用除法運算子的範例。由於兩者都有單位，這將不是一個有效的表達式：

```css
calc(50% / 5%)
```

當你有除法 `operator` 時，應該從右側 `operand` 移除單位：

```css
calc(50% / 5)
```

作為網頁開發者，`calc()` 函式（程式）對你非常有幫助。使用這個函式（程式），你可以動態設定屬性值，以創建靈活且響應式的設計。
