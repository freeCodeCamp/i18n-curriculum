# --description--

以前に、`do` を使って質問をする方法を学びました。`do`（`I`、`you`、`we`、`they`の場合）や `does`（`he`、`she`、`it`の場合）と単語 `not` を組み合わせて、主動詞の直前に置くことで否定文を作ることもできます。

- `I do not work everyday.` - 構造は「主語 + `do` + `not` + 主動詞」です。

- `It does not function.` - 構造は「主語 + `does` + `not` + 主動詞」です。

実用的にするために、`do not` は `don't` に、`does not` は `doesn't` に省略されることがよくあります。

質問文と同様に、`don't` を使うと、その後に続く主動詞は誰について話していても常に原形になります。
