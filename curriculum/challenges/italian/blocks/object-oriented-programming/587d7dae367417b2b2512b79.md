# --description--

I costruttori `Bird` e `Dog` della sfida precedente hanno funzionato bene. Tuttavia, nota che tutti gli `Birds` creati con il costruttore `Bird` si chiamano automaticamente Albert, sono di colore blu e hanno due gambe. E se volessi uccelli con valori diversi per nome e colore? È possibile modificare manualmente le proprietà di ogni uccello, ma sarebbe un lavoro enorme:

```js
let swan = new Bird();
swan.name = "Carlos";
swan.color = "white";
```

Supponiamo che tu stia scrivendo un programma per tenere traccia di centinaia o addirittura migliaia di uccelli diversi in un aviario. Ci vorrebbe molto tempo per creare tutti gli uccelli e poi cambiare le proprietà con valori diversi per ciascuno. Per creare più facilmente oggetti `Bird` diversi, puoi progettare il costruttore Bird per accettare parametri:

```js
function Bird(name, color) {
  this.name = name;
  this.color = color;
  this.numLegs = 2;
}
```

Poi passa i valori come argomenti per definire ogni uccello unico nel costruttore `Bird`: `let cardinal = new Bird("Bruce", "red");` Questo crea una nuova istanza di `Bird` con le proprietà `name` e `color` impostate rispettivamente su `Bruce` e `red`. La proprietà `numLegs` è ancora impostata su 2. L'oggetto `cardinal` ha queste proprietà:

```js
cardinal.name
cardinal.color
cardinal.numLegs
```

Il costruttore è più flessibile. Ora è possibile definire le proprietà di ogni `Bird` al momento della sua creazione, che è uno dei modi in cui i costruttori JavaScript sono così utili. Raggruppano oggetti insieme in base a caratteristiche e comportamenti condivisi e definiscono un modello che automatizza la loro creazione.

# --instructions--

Crea un altro costruttore `Dog`. Questa volta, impostalo per ricevere i parametri `name` e `color`, e fai in modo che la proprietà `numLegs` sia fissa a 4. Poi crea un nuovo `Dog` salvato in una variabile `terrier`. Passa due stringhe come argomenti per le proprietà `name` e `color`.

# --hints--

`Dog` dovrebbe ricevere un argomento per `name`.

```js
assert(new Dog('Clifford').name === 'Clifford');
```

`Dog` dovrebbe ricevere un argomento per `color`.

```js
assert(new Dog('Clifford', 'yellow').color === 'yellow');
```

`Dog` dovrebbe avere la proprietà `numLegs` impostata a 4.

```js
assert(new Dog('Clifford').numLegs === 4);
```

`terrier` dovrebbe essere creato usando il costruttore `Dog`.

```js
assert(terrier instanceof Dog);
```
