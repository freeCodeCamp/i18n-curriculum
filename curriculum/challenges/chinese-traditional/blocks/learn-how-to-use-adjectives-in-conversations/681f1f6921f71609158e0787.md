# --explanation--

`Specify` 意味著清楚地陳述或詳細描述某事。 例如：

`Please specify the time and place of the meeting.`－這表示要準確說明會議的時間和地點。

`What makes it` 是一個名詞語句。一群在句子中充當名詞作用的字詞。它說明某事背後的原因或起因。例如：

`She described what makes it so effective.`－這表示她解釋了某事物有效的原因。在這裡，語句 `what makes it so effective` 作為動詞 `described` 的物件。
