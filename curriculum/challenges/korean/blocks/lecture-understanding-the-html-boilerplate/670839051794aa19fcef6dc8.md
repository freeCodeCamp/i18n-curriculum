# --description--

UTF-8, 또는 UCS 변환 형식 8은 웹에서 널리 사용되는 표준화된 문자 인코딩입니다. 문자 인코딩은 컴퓨터가 문자를 데이터로 저장하는 방법입니다. 기본적으로 웹 페이지의 모든 텍스트는 하나 이상의 바이트로 저장된 문자 수열입니다. 컴퓨팅에서 바이트는 8비트, 즉 이진 숫자로 구성된 데이터 단위입니다. UTF-8은 유니코드 문자 집합의 모든 문자를 지원하며, 여기에는 모든 문자 체계, 언어, 기술 기호의 문자와 기호가 포함됩니다. 다음은 `meta` 요소에 `charset` 속성을 사용해 문자 인코딩을 `UTF-8`로 설정하는 예입니다:

```html
<meta charset="UTF-8" />
```

문자 인코딩을 UTF-8로 설정하면, 강세가 있는 `"e"` 문자(`é`)가 페이지에 올바르게 표시됩니다. 다음은 UTF-8 문자 인코딩을 사용하는 확장된 코드 예시입니다:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Examples of the UTF-8 encoding</title>
  </head>
  <body>
    <p>Café</p>
  </body>
</html>
```

새로운 프로젝트를 만들 때마다 `meta` 요소에 `charset` 속성을 `UTF-8`로 설정해 포함해야 합니다.
