# --description--

En este laboratorio, necesitas crear un algoritmo para fusionar dos arreglos.

**Objetivo:** Cumplir con las historias de usuario a continuación y pasar todas las pruebas para completar el laboratorio.

**Historias de usuario:**

1. Crea una función `frankenSplice` que acepte dos arreglos y un índice.
1. Copia cada elemento del primer arreglo dentro del segundo arreglo, en orden, comenzando en el índice dado, y devuelve el arreglo resultante.
1. Los arreglos de entrada deben permanecer iguales luego de que se ejecute la función.

```js
function frankenSplice(arr1, arr2, n) {
  let result = arr2.slice();
  for (let i = 0; i < arr1.length; i++) {
    result.splice(n + i, 0, arr1[i]);
  }
  return result;
}
```
