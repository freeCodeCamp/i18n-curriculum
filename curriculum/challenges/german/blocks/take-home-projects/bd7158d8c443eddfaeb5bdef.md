# --description--

Erstellen Sie eine Full-Stack-JavaScript-App, die funktional ähnlich ist wie diese: <a href="https://voting-app.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://voting-app.freecodecamp.rocks/</a>. Verwenden Sie einen Website-Builder Ihrer Wahl, um das Projekt abzuschließen.

Hier sind die spezifischen User Stories, die Sie für dieses Projekt implementieren sollten:

**User Story:** Als authentifizierte Nutzerin oder authentifizierter Nutzer können Sie Ihre Umfragen speichern und später wieder darauf zugreifen.

**User Story:** Als authentifizierte Nutzerin oder authentifizierter Nutzer können Sie Ihre Umfragen mit Ihren Freundinnen und Freunden teilen.

**User Story:** Als authentifizierte Nutzerin oder authentifizierter Nutzer können Sie die aggregierten Ergebnisse Ihrer Umfragen sehen.

**User Story:** Als authentifizierte Nutzerin oder authentifizierter Nutzer können Sie Umfragen löschen, die Sie nicht mehr möchten.

**User Story:** Als authentifizierte Nutzerin oder authentifizierter Nutzer können Sie eine Umfrage mit beliebig vielen möglichen Optionen erstellen.

**User Story:** Als nicht authentifizierte oder authentifizierte Nutzerin oder Nutzer können Sie alle Umfragen sehen und abstimmen.

**User Story:** Als nicht authentifizierte oder authentifizierte Nutzerin oder Nutzer können Sie die Ergebnisse der Umfragen in Diagrammform sehen. (Dies könnte mit Chart.js oder Google Charts umgesetzt werden.)

**User Story:** Als authentifizierte Nutzerin oder authentifizierter Nutzer können Sie, wenn Ihnen die Optionen einer Umfrage nicht gefallen, eine neue Option erstellen.

Wenn Sie fertig sind, stellen Sie sicher, dass eine funktionierende Demo Ihres Projekts öffentlich gehostet wird. Reichen Sie dann die URL im Feld „Solution Link“ ein. Optional können Sie auch einen Link zum Quellcode Ihres Projekts im Feld „GitHub Link“ absenden.
