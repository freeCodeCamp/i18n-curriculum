# --description--

在程式設計中，`int` 代表「整數」。它是一種用來儲存整數的資料型別，整數是沒有小數部分或小數點的數字。整數可以是正數、負數或零。

當處理整數時，選擇 `int` 資料型別是必要的，例如計數、陣列中的索引，或任何不需要小數的情況。

範例：

計算項目：`int itemCount = 5;`

儲存年齡：`int age = 30;`
