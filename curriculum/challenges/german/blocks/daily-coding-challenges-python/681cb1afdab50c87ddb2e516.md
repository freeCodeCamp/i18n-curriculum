# --description--

Gegeben ist ein String. Geben Sie eine durcheinandergebrachte Version dieses Strings zurück, bei der jedes Wort nach folgenden Vorgaben umgewandelt wird:

- Der erste und der letzte Buchstabe der Wörter bleiben an ihrer Stelle
- Alle Buchstaben zwischen dem ersten und letzten Buchstaben werden alphabetisch sortiert
- Die Eingabe-Strings enthalten keine Satzzeichen und bestehen vollständig aus Kleinbuchstaben

```py
def jbelmu(text):
    words = text.split()
    jumbled = []

    for word in words:
        if len(word) <= 3:
            jumbled.append(word)
        else:
            first = word[0]
            last = word[-1]
            middle = ''.join(sorted(word[1:-1]))
            jumbled.append(first + middle + last)
    
    return ' '.join(jumbled)
```
