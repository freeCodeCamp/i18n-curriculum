# --description--

设 $n$ 为正整数。若整数三元组 ($a$, $b$, $c$) 满足以下条件，则称其为 $n$ 的因式分解三元组：

- $1 ≤ a ≤ b ≤ c$
- $a \times b \times c = n$。

定义 $f(n)$ 为因式分解三元组 ($a$, $b$, $c$) 的 $a + b + c$，该三元组使得 $\frac{c}{a}$ 最小。可以证明该三元组是唯一的。

例如，$f(165) = 19$，$f(100\\,100) = 142$ 和 $f(20!) = 4\\,034\\,872$。

查找 $f(43!)$。
