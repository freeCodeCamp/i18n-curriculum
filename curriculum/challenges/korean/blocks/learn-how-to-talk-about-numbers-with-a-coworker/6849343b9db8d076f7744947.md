# --explanation--

`Most`는 과반수, 즉 50% 이상을 의미합니다. 예를 들어:

`Most developers use version control tools.` — 대부분의 개발자가 버전 관리를 사용합니다.

`Around`는 약 또는 대략을 의미합니다. 숫자가 정확하지 않을 때 사용합니다. 예를 들어:

`Around 20 people joined the training session.` — 약 20명이 교육에 참석했습니다.
