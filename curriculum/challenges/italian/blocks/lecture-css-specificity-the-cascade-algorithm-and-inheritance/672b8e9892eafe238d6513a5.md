# --interactive--

I selettori di classe sono una parte fondamentale del CSS, permettendo agli sviluppatori di indirizzare più elementi con lo stesso attributo di classe e applicare uno stile coerente.

Questo li rende molto versatili ed efficienti per applicare stili su un intero sito web.

I selettori di classe si definiscono con un punto (`.`) seguito dal nome della classe. Possono essere applicati a qualsiasi elemento nel documento HTML.

Ecco un esempio che usa un selettore di classe:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="highlight">Example paragraph</p>
```

```css
.highlight {
  color: green;
}
```

:::

In questo esempio, ogni elemento con la classe `highlight` avrà il colore del testo impostato su `green`.

I selettori di classe hanno una specificità più alta rispetto ai selettori di tipo ma inferiore rispetto ai selettori ID e agli stili in linea.

Il valore di specificità per un selettore di classe è `(0, 0, 1, 0)`. Questo significa che i selettori di classe possono sovrascrivere i selettori di tipo, ma possono essere sovrascritti dai selettori ID e dagli stili in linea.

I selettori di classe possono essere combinati con altri selettori per creare regole più specifiche.

Ecco un esempio che combina un selettore di tipo paragrafo con un selettore di classe:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="bold-text">Example paragraph</p>
<p class="bold-text">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p.bold-text {
  font-weight: bold;
}
```

:::

Questa regola si applica solo agli elementi `p` che hanno anche la classe `bold-text`, rendendo il loro testo in grassetto.
