# --description--

Express.js oder kurz Express ist ein minimalistisches und flexibles Web-Framework, das **auf** Node.js aufbaut. Es vereinfacht den Prozess, Webserver und APIs zu erstellen.

Anstatt eine Menge Boilerplate-Code mit dem nativen `http`-Modul von Node zu schreiben, ermöglicht Express Ihnen, robuste Webanwendungen mit nur wenigen Zeilen zu erstellen.

Man kann es sich als die schnelle, unvoreingenommene Schicht vorstellen, die Node.js einfacher und leistungsfähiger macht.

Hier sind drei Hauptgründe, warum Entwickler Express lieben:

1. **Minimal und schnell** – Express bietet nur das Wesentliche, gibt Ihnen volle Kontrolle und lässt Sie Funktionen bei Bedarf über Middleware und Plugins hinzufügen.

2. **Riesiges Ökosystem** – Es gibt eine große Sammlung von Middleware und Plugins für Logging, Authentifizierung, Sicherheit, Sitzungen und mehr.

3. **Ideal für APIs** – Das einfache Routing- und Anfrageverarbeitungsmodell macht Express besonders gut geeignet für den Aufbau von RESTful APIs.

Mit Express können Sie erstellen:

* Webanwendungen

* RESTful APIs

* Backends für Single Page Applications (SPA)

* Middleware-lastige Dienste

* Alles Serverseitige mit HTTP


## So installieren Sie Express

Stellen Sie zuerst sicher, dass Node.js und npm installiert sind. So prüfen Sie, ob node und npm installiert sind:

```bash
node -v
npm -v
```

Wenn Sie eine Versionsnummer sehen, bedeutet das, dass sie installiert sind. Falls nicht, müssen Sie sie installieren. Installationsanleitungen finden Sie in einer früheren Lektion oder auf der offiziellen Node.js-Website.

Als Nächstes geben Sie im Terminal folgende Befehle ein:

```bash
mkdir my-app
cd my-app
npm init -y
npm install express
```

Sie werden in zukünftigen Lektionen mehr über Middleware und Plugins lernen, aber kurz gesagt: Middleware ist Code, der zwischen dem Empfang einer Anfrage und dem Senden einer Antwort ausgeführt wird. Sie wird häufig für Fehlerbehandlung, Validierung von Eingaben, Modifikation von Daten in der Anfrage oder Antwort und mehr verwendet. Sie können Middleware verketten, um flexible und wiederverwendbare Logik zur Verarbeitung von Anfragen zu erstellen.

Plugins sind in der Regel externe, von Dritten entwickelte Node.js-Module, die die Funktionalität von Express erweitern. Sie können diese separat mit npm installieren, und die meisten sind als Middleware implementiert, die Sie mit wenig Aufwand in Ihrer Anwendung verwenden können. Anstatt häufige Funktionen von Grund auf neu zu schreiben, können Sie ein Plugin nutzen, um Dinge wie Logging, Authentifizierung oder Sicherheits-Best-Practices zu übernehmen.

Express ist ein beliebtes Framework für die serverseitige Entwicklung mit Node.js. Es bietet Ihnen die Bausteine, um schnell einen Server aufzusetzen, Routen zu verwalten, Middleware zu steuern und APIs zu erstellen – alles mit sauberem und gut lesbarem Code.
