# --description--

Acha $a_n$ iwe mfululizo unaoelezwa kwa urudia: $a_1 = 1$, $\displaystyle a_n = \left(\sum_{k = 1}^{n - 1} k \times a_k\right)\bmod n$.

Hivyo vipengele vya kwanza 10 vya $a_n$ ni: 1, 1, 0, 3, 0, 3, 5, 4, 1, 9.

Acha $f(N, M)$ wakilishe idadi ya jozi $(p, q)$ zenye sifa:

$$ 1 \le p \le q \le N \\; \text{na} \\; \left(\sum_{i = p}^q a_i\right)\bmod M = 0$$

Inaonekana kuwa $f(10, 10) = 4$ kwa jozi (3,3), (5,5), (7,9) na (9,10).

Pia unapewa kuwa $f({10}^4, {10}^3) = 97\\,158$.

Tafuta $f({10}^{12}, {10}^6)$.
