# --description--

يوفر Express مجموعة من دوال البرمجيات الوسيطة المدمجة التي تساعد في أداء مهام شائعة، مثل تقديم الملفات الثابتة، وتحليل أجسام طلبات JSON، والمزيد. يمكنك أيضًا استخدام برمجيات وسيطة تابعة لأطراف ثالثة لإضافة وظائف إضافية.

**إليك بعض الأمثلة على البرمجيات الوسيطة المدمجة:**

1. `express.json()`: برمجية وسيطة مدمجة تقوم بتحليل الحمولة الواردة بصيغة JSON.

```js
app.use(express.json())
```

1. `express.static()`: برمجية وسيطة لتقديم الملفات الثابتة مثل الصور وملفات CSS وJavaScript.

```js
app.use(express.static('public'))
```

1. `express.urlencoded()`: برمجية وسيطة تحلل الطلبات الواردة التي تحتوي على حمولة مشفرة بصيغة URL، وهو التنسيق الذي ترسله عادة نماذج HTML. تجعل البيانات المحللة متاحة في `req.body`.

```js
app.use(express.urlencoded({ extended: true }))
```

**والآن، إليك بعض الأمثلة على البرمجيات الوسيطة التابعة لأطراف ثالثة:**

1. `cors`: برمجية وسيطة شهيرة تابعة لأطراف ثالثة لتمكين مشاركة الموارد عبر الأصول المختلفة.

```bash
npm install cors
```

```js
const cors = require('cors')
app.use(cors())
```

1. `morgan`: برمجية وسيطة لتسجيل طلبات HTTP.

```bash
npm install morgan
```

```js
const morgan = require('morgan')
app.use(morgan('tiny'))
```

الآن، لننظر إلى مثال على استخدام كل من البرمجيات الوسيطة المدمجة وتلك التابعة لأطراف ثالثة:

```js
const express = require('express')
const cors = require('cors')
const morgan = require('morgan')
const app = express()

app.use(express.json()) // Built-in middleware to parse JSON
app.use(cors())       // Third-party middleware for CORS
app.use(morgan('tiny')) // Third-party middleware for logging

app.get('/', (req, res) => {
  res.send('Hello, World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* البرمجية الوسيطة المدمجة `express.json()` تحلل الحمولة الواردة بصيغة JSON.
* البرمجيات الوسيطة التابعة لأطراف ثالثة `cors` تسمح بالطلبات عبر الأصول المختلفة، و`morgan` تسجل طلبات HTTP الواردة.

من المهم ملاحظة أن دوال البرمجيات الوسيطة تُضاف بالترتيب الذي تُستدعى به باستخدام `app.use()`.

يوفر Express برمجيات وسيطة مدمجة أساسية لمعالجة المهام الشائعة مثل تحليل JSON وتقديم الملفات الثابتة. يمكنك أيضًا توسيع الوظائف باستخدام برمجيات وسيطة تابعة لأطراف ثالثة لإضافة قدرات إضافية.
