# --explanation--

動詞 `to be` 的否定縮寫在英語中很常見。它們用來使句子更簡短且更自然。例如：

- `They're not` 是 `they are not` 的縮寫，用來表示某事不是真的或沒有發生。

- `Isn't` 是 `is not` 的縮寫，也用來表示某事不正確或未發生。
