# --questions--

## --text--

¿Qué oración usa correctamente `close to` para describir una ubicación física?

## --answers--

`He finished the project close to the deadline.`

### --feedback--

Este uso de `close to` se refiere al tiempo, no a la ubicación física.

---

`She's close to understanding the concept.`

### --feedback--

Este uso de `close to` se refiere a progreso o logro, no a espacio físico.

---

`They're close to signing the contract.`

### --feedback--

Esto trata sobre estar cerca de una decisión, no de un lugar.

---

`His desk is close to the window.`

4

# --explanation--

`Close to` puede describir la distancia física cuando una cosa está cerca de otra.

En la oración correcta, muestra que el escritorio está ubicado cerca de la ventana, lo cual es un ejemplo adecuado y claro de usar `close to` para describir la ubicación.

Las otras opciones usan `close to` de maneras no físicas, como tiempo, progreso o toma de decisiones.
