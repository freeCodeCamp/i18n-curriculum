# --interactive--

Nella lezione precedente, hai imparato a lavorare con gli eventi `"click"` e `"input"` usando il metodo `addEventListener()`. In questa lezione, imparerai come rimuovere i listener di eventi usando il metodo `removeEventListener()`.

Il metodo `removeEventListener()` serve a rimuovere un listener di eventi che era stato precedentemente aggiunto a un elemento usando il metodo `addEventListener()`. Questo è utile quando vuoi smettere di ascoltare un evento specifico su un elemento.

Ecco la sintassi base per il metodo `removeEventListener()`:

```js
element.removeEventListener("event", listener);
```

Proprio come il metodo `addEventListener()`, il metodo `removeEventListener()` prende due argomenti: l’evento che vuoi rimuovere e la funzione listener che era stata aggiunta in precedenza.

C’è anche un terzo argomento opzionale che può essere passato al metodo `removeEventListener()`. Questo argomento può essere o `options` o `useCapture`.

L’argomento `options` è un oggetto che specifica le opzioni per il listener di eventi, come se il listener debba essere passivo o eseguito una sola volta.

L’argomento `useCapture` è un valore booleano che specifica se l’evento deve essere catturato durante la fase di propagazione dell’evento.

La maggior parte delle volte, ti basterà passare solo gli argomenti evento e listener al metodo `removeEventListener()`.

Se vogliamo alternare il colore di sfondo tra grigio e blu, possiamo usare un listener di eventi per farlo.

Prima dobbiamo consultare l’elemento `button` insieme all’elemento `body`:

```js
const bodyEl = document.querySelector("body");
const btn = document.getElementById("btn");
```

Poi dobbiamo creare la funzione responsabile di alternare tra i colori grigio e blu:

```js
let isBgColorGrey = true;

function toggleBgColor() {
  bodyEl.style.backgroundColor = isBgColorGrey ? "blue" : "grey";
  isBgColorGrey = !isBgColorGrey;
}
```

Stiamo usando una variabile booleana `isBgColorGrey` per tenere traccia del colore di sfondo attuale. Se il colore di sfondo è grigio, lo cambiamo in blu, e viceversa.

Poi dobbiamo aggiungere un listener di eventi all’elemento `button` per chiamare la funzione `toggleBgColor` quando il pulsante viene cliccato:

```js
btn.addEventListener("click", toggleBgColor);
```

Possiamo aggiornare l’HTML per includere un elemento paragrafo che useremo per rimuovere il listener di eventi quando ci si passa sopra con il mouse:

```html
<link rel="stylesheet" href="styles.css" />
<h1>Event Listener examples</h1>
<p id="para">MouseOver this text to remove the event listener</p>
<button id="btn">Change background color</button>
<script src="index.js"></script>
```

Per rimuovere il listener di eventi quando si passa sopra il paragrafo, possiamo aggiungere un listener di eventi all’elemento paragrafo. Stiamo usando l’evento `mouseover` e passando una funzione che rimuove il listener di eventi dall’elemento `button`:

```js
const para = document.getElementById("para");
const btn = document.getElementById("btn");

para.addEventListener("mouseover", () => {
  btn.removeEventListener("click", toggleBgColor);
});
```

Quando si passa sopra il paragrafo, il listener di eventi per il clic sul pulsante viene rimosso e il colore di sfondo non cambierà più quando si clicca il pulsante.
