# --interactive--

У CSS існує кілька способів додати кордони навколо зображень, кожен із яких пропонує різні варіанти стилізації та рівні контролю.

Давайте розглянемо деякі з найпоширеніших і найуніверсальніших методів.

Найпростіший спосіб додати кордон до зображення — це використати властивість `border`. Ця властивість є скороченням, яке дозволяє одночасно встановити ширину, стиль і колір кордону.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

Це CSS-правило додає червоний суцільний кордон шириною 2 пікселі навколо всіх елементів `img`. Ви можете налаштувати ширину, стиль (наприклад, `dashed`, `dotted` або `double`) і колір відповідно до вашого дизайну.

Якщо вам потрібен більший контроль над окремими сторонами кордону, ви можете використати конкретні властивості кордону для кожної сторони:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border-top: 10px solid red;
  border-right: 10px dashed green;
  border-bottom: 10px dotted blue;
  border-left: 10px double purple;
}
```

:::

Це дозволяє створювати унікальні стилі кордону для кожної сторони зображення.

Інший спосіб створити ефект кордону — використати властивість `outline`. Хоча вона схожа на border, outline не впливає на розміри чи макет елемента:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  outline: 3px solid gold;
}
```

:::

Це створює золотисті обриси навколо зображення без зміни його розміру чи позиції.

Якщо ви хочете зробити заокруглені кути кордону, ви можете використати властивість `border-radius` разом із border:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

Пам’ятайте, ці техніки можна комбінувати і налаштовувати для створення унікальних ефектів кордону. Вибір методу залежить від ваших конкретних вимог до дизайну та рівня складності, який вам потрібен.
