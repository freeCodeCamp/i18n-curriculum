# --description--

计算机科学中的分治范式是一种递归地将问题分解为更小子问题的技术。该技术的关键方面之一是递归，即函数反复调用自身，直到达到基本情形。在本课中，我们将通过归并排序算法更好地理解分治技术的工作原理。

假设我们有这个数字列表：

```md
42 37 53 17
```

目标是使用归并排序算法将该列表从小到大排序。第一步是将该列表对半分割：

```md
42 37 | 53 17
```

然后我们需要查看列表的左侧：

```md
42 37
```

我们将该子列表再次对半分，直到每个子列表中只有一个项：

```md
42 | 37
```

只有一个项的列表默认已排序。接下来我们需要将每个只有一个元素的子列表合并成一个排序后的列表：

```md
37 42
```

然后我们对原始列表的右侧执行相同的进程：

```py
# right side of original list
53 17

# divide the list in half
53 | 17

# merge the lists in sorted order
17 53
```

现在原始列表的两个部分都已排序，我们将这两个部分合并并对元素进行排序：

```markdown
17 37 42 53
```

下面是算法在代码中的样子：

```py
def merge_sort(arr):
    if len(arr) <= 1:
        return arr

    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])

    sorted_list = []
    i = 0
    j = 0

    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            sorted_list.append(left[i])
            i += 1
        else:
            sorted_list.append(right[j])
            j += 1

    sorted_list.extend(left[i:])
    sorted_list.extend(right[j:])

    return sorted_list
```

归并排序的时间复杂度是 `O(n log n)`，因为列表不断被对半分割（`log n`），然后再合并（`O(n)`）。与冒泡排序等其他排序算法不同，归并排序不是原地排序，且空间复杂度为 `O(n)`。
