# --description--

三次元空間の2点 ($x_1$, $y_1$, $z_1$) と ($x_2$, $y_2$, $z_2$) が与えられたとき、これらの点間のマンハッタン距離は $|x_1 - x_2| + |y_1 - y_2| + |z_1 - z_2|$ と定義されます。

$C(r)$ を半径 $r$、中心が原点 $O(0, 0, 0)$ の球とします。

$I(r)$ を $C(r)$ の表面上にある整数座標のすべての点の集合とします。

$S(r)$ を $I(r)$ のすべての要素から原点 $O$ までのマンハッタン距離の合計とします。

例：$S(45)=34518$。

$S({10}^{10})$ を求めてください。
