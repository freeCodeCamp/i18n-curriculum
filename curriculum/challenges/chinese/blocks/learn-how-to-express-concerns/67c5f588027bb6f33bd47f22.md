当你说某事 `works smoothly`，意味着某事发生时没有困难或中止。`Smoothly` 来源于 `smooth`，后者描述某物没有粗糙或障碍，但它们用于不同的上下文。例如：

- `The table has a smooth surface.` - 这描述了该表格表面没有粗糙区域。

- `The software update helped the system run more smoothly。`——这意味着系统现在运行时出现的问题更少。
