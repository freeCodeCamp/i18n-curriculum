# --interactive--

## Nozioni di base sulle stringhe

- **Definizione**: Una stringa è una sequenza di caratteri racchiusa tra virgolette singole, virgolette doppie o segni di graduazione. Le stringhe sono tipi di dato primitivi e sono immutabili. L'immutabilità significa che una volta creata una stringa, non può essere modificata. 
- **Consultare i caratteri di una stringa**: Per consultare un carattere di una stringa puoi usare la notazione a parentesi e passare l'indice. Un indice è la posizione di un carattere all'interno di una stringa, ed è a base zero.

:::interactive_editor

```js
const developer = "Jessica";
console.log(developer[0]); // J
```

:::

- **`\n` (carattere di nuova riga)**: Puoi creare una nuova riga in una stringa usando il carattere di nuova riga `\n`.

:::interactive_editor

```js
const poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

- **Escape delle stringhe**: Puoi fare l'escape dei caratteri in una stringa mettendo delle backslash (`\`) davanti alle virgolette.

:::interactive_editor

```js
const statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

## Template literal (stringhe template) e interpolazione delle stringhe

- **Definizione**: I template literal sono definiti con i segni di graduazione (`). Permettono una manipolazione più semplice delle stringhe, incluso incorporare variabili direttamente all'interno di una stringa, una funzionalità nota come interpolazione delle stringhe.

:::interactive_editor

```js
const name = "Jessica";
const greeting = `Hello, ${name}!`; 
console.log(greeting); // "Hello, Jessica!"
```

:::

## ASCII, il metodo `charCodeAt()` e il metodo `fromCharCode()`

- **ASCII**: ASCII (American Standard Code for Information Interchange) è uno standard di codifica dei caratteri usato per rappresentare i caratteri base inglesi tramite valori numerici. Le lezioni precedenti introducono `charCodeAt()` e `fromCharCode()` usando esempi ASCII.
- **Unicode**: Le stringhe JavaScript usano internamente Unicode, specificamente la codifica UTF-16. Per i primi 128 caratteri (lettere latine base, cifre e simboli comuni), i valori Unicode corrispondono ai codici ASCII. Per questo motivo gli esempi basati su ASCII continuano a funzionare in JavaScript.
- **Il metodo `charCodeAt()`**: Questo metodo restituisce l'unità di codice UTF-16 del carattere a un indice specificato. Per i caratteri latini base, questo valore corrisponde al codice ASCII.

:::interactive_editor

```js
const letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

:::

- **Il metodo `fromCharCode()`**: Questo metodo converte un codice ASCII nel carattere corrispondente.

:::interactive_editor

```js
const char = String.fromCharCode(65);
console.log(char);  // A
```

:::

## Altri metodi comuni per le stringhe

- **Il metodo `indexOf`**: Questo metodo serve a cercare una sottostringa all'interno di una stringa. Se la sottostringa viene trovata, `indexOf` restituisce l'indice (o posizione) della prima occorrenza di quella sottostringa. Se la sottostringa non viene trovata, `indexOf` restituisce -1, che indica che la ricerca non ha avuto successo.

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.indexOf("fox")); // 16
console.log(text.indexOf("cat")); // -1
```
  
:::  

- **Il metodo `includes()`**: Questo metodo serve a verificare se una stringa contiene una sottostringa specifica. Se la sottostringa è presente nella stringa, il metodo restituisce true. Altrimenti, restituisce false.

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.includes("fox")); // true
console.log(text.includes("cat")); // false
```

:::

- **Il metodo `slice()`**: Questo metodo estrae una porzione di stringa e restituisce una nuova stringa, senza modificare quella originale. Prende due parametri: l'indice di partenza e l'indice di fine opzionale.

:::interactive_editor

```js
const text = "freeCodeCamp";
console.log(text.slice(0, 4));  // "free"
console.log(text.slice(4, 8));  // "Code"
console.log(text.slice(8, 12)); // "Camp"
```

:::

- **Il metodo `toUpperCase()`**: Questo metodo converte tutti i caratteri in lettere maiuscole e restituisce una nuova stringa con tutti i caratteri maiuscoli.

:::interactive_editor

```js
const text = "Hello, world!";
console.log(text.toUpperCase()); // "HELLO, WORLD!"
```

:::

- **Il metodo `toLowerCase()`**: Questo metodo converte tutti i caratteri di una stringa in minuscolo.

:::interactive_editor

```js
const text = "HELLO, WORLD!"
console.log(text.toLowerCase()); // "hello, world!"
```

:::

- **Il metodo `replace()`**: Questo metodo permette di trovare un valore specificato (come una parola o un carattere) in una stringa e sostituirlo con un altro valore. Il metodo restituisce una nuova stringa con la sostituzione e lascia invariata quella originale perché le stringhe JavaScript sono immutabili.

:::interactive_editor

```js
const text = "I like cats";
console.log(text.replace("cats", "dogs")); // "I like dogs"
```

:::

- **Il metodo `replaceAll()`**: Questo metodo permette di trovare tutte le occorrenze di un valore specificato (una parola, un carattere o un pattern) in una stringa e sostituirle con un altro valore. Funziona come `replace()`, ma invece di fermarsi alla prima corrispondenza aggiorna tutte le corrispondenze trovate nella stringa.

:::interactive_editor

```js
const text = "I love cats and cats are so much fun!";
console.log(text.replaceAll("cats", "dogs")); // "I love dogs and dogs are so much fun!"
```

:::

- **Il metodo `repeat()`**: Questo metodo serve a ripetere una stringa un numero specificato di volte.

:::interactive_editor

```js
const text = "Hello";
console.log(text.repeat(3)); // "HelloHelloHello"
```

:::

- **Il metodo `trim()`**: Questo metodo serve a rimuovere gli spazi bianchi sia dall'inizio sia dalla fine di una stringa.

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trim()); // "Hello, world!"
```

:::

- **Il metodo `trimStart()`**: Questo metodo rimuove gli spazi bianchi dall'inizio (o "start") della stringa.

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trimStart()); // "Hello, world!  "
```

:::

- **Il metodo `trimEnd()`**: Questo metodo rimuove gli spazi bianchi dalla fine della stringa.

:::interactive_editor

```js
const text = " Hello, world! ";
console.log(text.trimEnd()); // "  Hello, world!"
```

:::

- **Il metodo `prompt()`**: Questo metodo del `window` serve a ottenere informazioni da un utente tramite un modulo di dialogo a riquadro. Questo metodo prende due argomenti. Il primo argomento è il messaggio che apparirà all'interno del riquadro, tipicamente per invitare l'utente a inserire informazioni. Il secondo è un valore predefinito opzionale che riempirà inizialmente il campo di input.

```js
const answer = window.prompt("What's your favorite animal?"); // This will change depending on what the user answers
```
