## --text--

Welche der folgenden Aussagen beschreibt am besten eine integrierte Entwicklungsumgebung (IDE)?

## --answers--

Visual Studio Code ist eine integrierte Entwicklungsumgebung.

---

Eine integrierte Entwicklungsumgebung ist ein Werkzeug, das das Schreiben von Code erleichtert.

---

Eine IDE ist eine Sammlung von Werkzeugen, die den gesamten Softwareentwicklungszyklus unterstützt.
