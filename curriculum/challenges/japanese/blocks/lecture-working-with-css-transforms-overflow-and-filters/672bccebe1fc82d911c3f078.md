# --interactive--

CSSの`filter`プロパティは、ウェブページ上の要素にグラフィカルな効果を適用できる強力なツールです。元の素材を変更せずに、画像や背景、さらにはテキストの見た目を調整するのに特に役立ちます。

`filter`プロパティは、ぼかし、色のシフト、コントラストの調整など、さまざまな効果を作り出すために使えます。filterプロパティの仕組みを説明し、よく使われる例を見ていきましょう。`filter`プロパティの基本的な構文はシンプルです：

```css
selector {
  filter: function(amount);
}
```

ここで、`function`は適用したい特定のフィルター効果を表し、`amount`は通常、その効果の強さを決める値です。では、よく使われるフィルター関数とその用途を見ていきましょう。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: blur(2px);
}
```

:::

`blur`関数は要素にガウスぼかしを適用します。量はピクセル単位で指定し、ぼかしの半径を表します。このCSSルールはページ上のすべての画像に2ピクセルのぼかしを適用します。`blur`効果はデザインに奥行きを出したり、画像の一部をぼかしたりするのに便利です。

`brightness`関数は要素の明るさを調整します。値が`0%`だと要素は完全に黒くなり、`100%`を超える値は明るさを増加させます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: brightness(150%);
}
```

:::

このCSSルールは画像要素の明るさを`50%`だけ上げます。明るさの調整は画像を際立たせたり、色あせたような効果を作ったりするのに使えます。

`grayscale`関数は要素をグレースケールに変換します。量はパーセンテージで定義され、`100%`は完全なグレースケール、`0%`は画像を変化させません。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: grayscale(100%);
}
```

:::

このルールは画像要素を完全なグレースケールに変換します。`grayscale`はヴィンテージ風の見た目を作ったり、ページ上の特定の要素の強調を抑えたりするのに使えます。

`sepia`関数は要素にセピア調を適用します。グレースケールと同様にパーセンテージ値を使います：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: sepia(80%);
}
```

:::

このルールは画像要素に`80%`のセピア効果を適用します。セピア効果はヴィンテージや昔風の見た目を作るのに最適です。

`hue-rotate`関数は要素に色相回転を適用します。値は度数で定義され、色相環を回転させることを表します。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: hue-rotate(90deg);
}
```

:::

このルールは画像要素の色相を`90`度回転させます。色相回転はサイケデリックな効果を作ったり、画像の全体的な色調を調整したりするのに使えます。

`filter`プロパティの最も強力な点の一つは、複数の効果を組み合わせられることです。同じ要素に複数のフィルターをスペースで区切って適用できます：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {  
  filter: contrast(150%) brightness(110%) sepia(30%);  
}
```

:::

このルールは画像要素にコントラストの増加、わずかな明るさの増加、そして控えめなセピア効果を適用します。

フィルターを組み合わせることで、デザインのニーズに合わせた複雑でユニークな視覚効果を作り出せます。CSSのfilterプロパティはウェブ要素の見た目を創造的に操作できる多用途なツールです。

ここで紹介した一般的なフィルター関数以外にも、`contrast`、`invert`、`saturate`などがあります。強力な機能であるため、ユーザーを圧倒したりアクセシビリティを損なったりしないよう、使い方には注意が必要です。
