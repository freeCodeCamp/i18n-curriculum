# --description--

**目的:** 次のような機能を持つアプリを作成してください：<a href="https://choropleth-map.freecodecamp.rocks" target="_blank" rel="noopener noreferrer nofollow">https://choropleth-map.freecodecamp.rocks</a>。

以下のユーザーストーリーを満たし、すべてのテストに合格してください。必要なライブラリやAPIは自由に使って構いません。自分なりのスタイルを加えてください。

HTML、JavaScript、CSS、D3のSVGベースの可視化ライブラリを使えます。必要なDOM要素は各テストの時点でクエリされます。もしVueのようなフロントエンドフレームワークを使うと、動的コンテンツのテスト結果が正確でない場合があります。将来的には対応を検討していますが、現時点ではD3プロジェクトでこれらのフレームワークはサポートされていません。

**ユーザーストーリー #1:** コロプレスマップには対応する`id="title"`を持つタイトルがあること。

**ユーザーストーリー #2:** コロプレスマップには対応する`id="description"`を持つ説明要素があること。

**ユーザーストーリー #3:** コロプレスマップにはデータを表す対応する`class="county"`を持つ郡があること。

**ユーザーストーリー #4:** 郡には少なくとも4つの異なる塗りつぶし色が使われていること。

**ユーザーストーリー #5:** 各郡には対応する`data-fips`と`data-education`の値を含む`fips`と`education`のプロパティがあること。

**ユーザーストーリー #6:** コロプレスマップには提供された各データポイントに対応する郡があること。

**ユーザーストーリー #7:** 郡にはサンプルデータと一致する`data-fips`と`data-education`の値があること。

**ユーザーストーリー #8:** コロプレスマップには対応する`id="legend"`を持つ凡例があること。

**ユーザーストーリー #9:** 凡例には少なくとも4つの異なる塗りつぶし色が使われていること。

**ユーザーストーリー #10:** エリアにマウスオーバーすると、対応する`id="tooltip"`を持つツールチップが表示され、そのエリアの詳細情報が見られること。

**ユーザーストーリー #11:** ツールチップにはアクティブなエリアの`data-education`に対応する`data-education`プロパティがあること。

このプロジェクトを完成させるために必要なデータセットは以下の通りです：

- **米国教育データ:**`https://cdn.freecodecamp.org/testable-projects-fcc/data/choropleth_map/for_user_education.json`
- **米国郡データ:**`https://cdn.freecodecamp.org/testable-projects-fcc/data/choropleth_map/counties.json`

<a href='https://codepen.io/pen?template=MJjpwO' target="_blank" rel="noopener noreferrer nofollow">このCodePenテンプレート</a>を使い、`Save`をクリックして自分のペンを作成するか、任意の環境でテストを実行するためにこのCDNリンクを使ってください：`https://cdn.freecodecamp.org/testable-projects-fcc/v1/bundle.js`

完成したら、すべてのテストに合格した動作中のプロジェクトのURLを提出してください。
