# --description--

웹사이트가 텍스트만 표시할 수 있다면 꽤 지루할 것입니다. 다행히 HTML은 다양한 미디어를 표시할 수 있는 여러 요소를 제공합니다. 그중 가장 널리 사용되는 것이 이미지 요소입니다.

HTML에서 이미지를 표시하려면 `<img>` 요소를 사용하세요. 지금까지 만난 다른 요소들과 달리 `<img>`는 빈 요소입니다. 빈, void HTML 요소는 닫는 태그가 필요 없습니다.

열고 닫는 태그로 내용을 감싸는 대신, `src` 속성을 사용해 페이지에 이미지를 임베드합니다. 이 속성은 브라우저에 이미지 파일 위치를 알려줍니다. `src` 속성은 앵커 태그의 `href` 속성과 매우 비슷하게 작동합니다. 절대 경로와 상대 경로 모두를 사용해 이미지를 임베드할 수 있습니다.

예를 들어, 절대 경로를 사용해 The Odin Project 사이트에 있는 이미지를 표시할 수 있습니다:

자신의 웹사이트에 있는 이미지를 사용하려면 상대 경로를 사용할 수 있습니다.

- `images` 프로젝트 내에 `odin-links-and-images`라는 새 디렉터리를 만드세요.

- 다음으로 <a href="https://unsplash.com/photos/Mv9hjnEUHR4/download?force=true&w=640" target="_blank">이 이미지</a>를 다운로드해 방금 만든 images 디렉터리에 옮기세요.

- 이미지를 `dog.jpg`로 이름을 바꾸세요.

마지막으로 이미지를 `index.html` 파일에 추가하세요:

```html
<body>
  <h1>Homepage</h1>
	<a href="https://www.theodinproject.com/about">click me</a>

	<a href="pages/about.html">About</a>

	<img src="images/dog.jpg">
</body>
```

`index.html` 파일을 저장하고 브라우저에서 열어 찰스의 모습을 확인하세요.


## 상위 디렉터리

`about` 페이지에서 개 이미지를 사용하고 싶다면 어떻게 해야 할까요? 먼저 pages 디렉터리에서 한 단계 위로 올라가 상위 디렉터리로 이동한 다음 images 디렉터리에 접근해야 합니다.

상위 디렉터리로 가려면 상대 파일 경로에 점 두 개를 이렇게 사용하세요: `../.` 실제로 `body` 파일의 `about.html` 내에서, 앞서 추가한 제목 아래에 다음 이미지를 추가해 보겠습니다:

```html
<img src="../images/dog.jpg">
```

이 내용을 자세히 살펴보면:

- 먼저 pages 디렉터리의 상위 디렉터리인 `odin-links-and-images`로 이동합니다.

- 그다음 상위 디렉터리에서 `images` 디렉터리로 들어갑니다.

- 마지막으로 `dog.jpg` 파일에 접근합니다.

앞서 사용한 비유를 다시 들자면, 파일 경로에서 `../`를 사용하는 것은 현재 있는 방에서 나와 복도로 나가 다른 방으로 가는 것과 비슷합니다.

## Alt 속성

`src` 속성 외에도 모든 이미지 요소에는 `alt` (대체 텍스트) 속성이 있어야 합니다.

`alt` 속성은 이미지를 설명하는 데 사용됩니다. 이미지가 로드되지 않을 경우 대체로 표시되며, 시각 장애가 있는 사용자를 위해 화면 읽기 프로그램과 함께 이미지 내용을 설명하는 데도 사용됩니다.
