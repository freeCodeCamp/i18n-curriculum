# --explanation--

Camila diz `¿Cuál es tu correo electrónico?` A estrutura segue este padrão:

`Qual é o seu` + noun?

Esta estrutura é usada para solicitar informações pessoais específicas. Neste caso, `correo electrónico` refere-se a um endereço de e-mail usado para enviar e receber mensagens online.
