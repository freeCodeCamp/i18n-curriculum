# --description--

Dada una palabra o frase, devuelve una versión corregida donde cada palabra siga la regla "i antes de e excepto después de c".

- Si una palabra contiene `"ei"` que no está precedida por `"c"`, reemplázala por `"ie"`.
- Si una palabra contiene `"ie"` precedida por `"c"`, reemplázala por `"ei"`.
- Todas las demás palabras se dejan sin cambios.

# --solutions--

```py
import re

def i_before_e(sentence):
    words = []
    for word in sentence.split(" "):
        word = re.sub(r"([^c])ei", r"\1ie", word)
        word = re.sub(r"cie", "cei", word)
        words.append(word)
    return " ".join(words)
```
