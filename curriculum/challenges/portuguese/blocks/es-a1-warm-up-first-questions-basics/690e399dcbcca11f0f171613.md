# --explanation--

A palavra `Cómo` significa "How". 

É usado em perguntas como `¿Cómo estás?` ("How are you?").

Em espanhol, quando a palavra `cómo` é usada para fazer uma pergunta, ela carrega um acento na primeira letra `o`.
