# --description--

令 d(`n`) 定义为 `n` 的真因数之和（小于 `n` 且能整除 `n` 的数）。

如果 d(`a`) = `b` 且 d(`b`) = `a`，且 `a` ≠ `b`，那么 `a` 和 `b` 是一个友好点对，且 `a` 和 `b` 都被称为友好数。

例如，220 的真因数是 1、2、4、5、10、11、20、22、44、55 和 110；因此 d(220) = 284。284 的真因数是 1、2、4、71 和 142；所以 d(284) = 220。

计算所有小于 `n` 的亲和数之和。

```js
const sumAmicableNum = (n) => {
  const fsum = (n) => {
    let sum = 1;
    for (let i = 2; i <= Math.floor(Math.sqrt(n)); i++)
      if (Math.floor(n % i) === 0)
        sum += i + Math.floor(n / i);
    return sum;
  };
  let d = [];
  let amicableSum = 0;
  for (let i=2; i<n; i++) d[i] = fsum(i);
  for (let i=2; i<n; i++) {
    let dsum = d[i];
    if (d[dsum]===i && i!==dsum) amicableSum += i+dsum;
  }
  return amicableSum/2;
};
```
