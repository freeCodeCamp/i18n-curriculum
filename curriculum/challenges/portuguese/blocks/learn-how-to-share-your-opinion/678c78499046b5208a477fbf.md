`front-end`

### --feedback--

Refere-se à parte de um site ou aplicativo que os usuários veem e interagem diretamente. Esta é uma palavra composta. Lembre-se de usar `-`.

# --explanation--

`Go-to` refere-se a algo que é considerado a melhor ou mais confiável escolha em uma determinada situação. Por exemplo:

`Python is the go-to language for data analysis.` - Isso significa que Python é a linguagem mais usada ou confiável para análise de dados.

`Front-end development` refere-se à parte de um site ou aplicação que lida com o layout visual e a interface do usuário, com os quais os usuários interagem. Por exemplo:

`HTML and CSS are essential for front-end development.` - Isso significa que HTML e CSS são importantes para criar as partes visíveis de um site com as quais os usuários interagem.
