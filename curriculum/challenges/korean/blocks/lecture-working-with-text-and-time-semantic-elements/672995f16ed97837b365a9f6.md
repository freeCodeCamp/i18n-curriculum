# --interactive--

`time` 요소는 특정 순간을 나타내는 데 사용됩니다.

다음은 `time` 요소를 사용해 오후 8시, 즉 20:00을 나타낸 예시입니다.

:::interactive_editor

```html
<p>The reservations are for <time datetime="20:00">20:00 </time></p>
```

:::

`datetime` 속성은 날짜와 시간을 기계가 읽을 수 있는 형식으로 변환하는 데 사용됩니다.

이것은 검색 엔진 결과에 도움이 되고 브라우저가 날짜와 시간 정보를 더 효과적으로 처리하는 데 중요합니다.

`datetime` 속성의 값은 유효한 연도, 유효한 월, 유효한 시간, 지역 날짜, 전역 날짜 또는 유효한 기간 문자열이어야 합니다.

다음은 특정 날짜를 나타내기 위해 time 요소를 사용하는 또 다른 예시입니다:

:::interactive_editor

```html
<p>
  The graduation will be on <time datetime="2024-06-15T15:00">June 15</time>
</p>
```

:::

`datetime` 속성의 값은 ISO 8601 형식입니다. ISO 8601은 날짜와 시간을 나타내는 국제 표준입니다.

값의 첫 부분은 연도, 월, 일입니다. 값에 있는 대문자 T는 날짜와 시간 사이의 구분자입니다.

15:00은 오후 3시를 의미합니다.

이벤트, 출판 날짜 또는 약속을 나타내야 할 때는 `time` 요소를 사용하는 것이 가장 좋습니다.
