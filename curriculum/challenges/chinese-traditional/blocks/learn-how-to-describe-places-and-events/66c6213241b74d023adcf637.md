# --explanation--

當使用動詞 `to be` 形成疑問句時，主詞和動詞的通常語序會顛倒。這表示動詞會出現在主詞之前。例如：

- 你的專案已連接到 Git。

- 你的專案是否已連接到 Git？

在 Sarah 的句子中，她使用這種倒裝來詢問 Tom 他的專案是否連接到 Eclipse 中的 Git。
