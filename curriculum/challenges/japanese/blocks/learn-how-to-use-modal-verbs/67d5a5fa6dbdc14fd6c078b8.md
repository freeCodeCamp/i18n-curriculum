## --text--

どの文が文法的に正しいですか？

## --answers--

`What happen would if the system crashed?`

### --feedback--

正しい語順は `What would happen...` です。

---

`Would what happen if the system crashed?`

### --feedback--

疑問文では、`would` は `what` の後に来て、前には来ません。

---

`What would happens if the system crashed?`

### --feedback--

`Would` の後には動詞の原形が続くので、`happens` は `happen` であるべきです。

---

`What would happen if the system crashed?`

# --explanation--

もし `wh-` の疑問文（`when`、`who`、`why`、`how`）が助動詞を使う場合、助動詞は主語の前に来ます。主動詞は原形のままです。例えば：

- `What can you do to fix this?` - 可能性について尋ねる。

- `Why should we start early?` - 助言を求める。

- `How must we prepare?` - 要件について尋ねる。

- `When will you finish?` - 未来の行動について尋ねる。

- `Where could I find help?` - 選択肢について尋ねる。

このようにして、可能なこと、必要なこと、または期待されることについて尋ねています。
