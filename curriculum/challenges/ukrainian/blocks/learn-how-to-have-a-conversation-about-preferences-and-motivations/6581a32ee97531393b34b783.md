# --description--

`Encourage` означає надати комусь підтримку або впевненість, щоб щось зробити. Наприклад, коли ваш друг каже `You can solve this coding problem!`, він вас заохочує.

`Discourage` — це протилежне; означає змусити когось почуватися менш впевнено або менш схильним щось робити. Якщо той друг скаже `That programming language is too hard for beginners`, це може відбити у вас бажання вчитися.

`Discouraging` описує щось, що змушує вас почуватися пригніченим. Наприклад, `Finding a bug in your code after hours of work can be discouraging.`

`Encouraging` — це коли щось дає вам мужність або надію. Наприклад, `Getting your first program to run is very encouraging.`
