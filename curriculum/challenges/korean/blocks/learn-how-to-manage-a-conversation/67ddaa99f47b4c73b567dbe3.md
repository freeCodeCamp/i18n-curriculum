# --explanation--

`Sure thing`는 `Of course` 또는 `I'll do it`를 비공식적이고 친근하게 표현하는 방법입니다. 이는 화자가 주저 없이 요청에 동의한다는 뜻입니다. 예를 들어:

- **Ashley:** `Can you send me the file this afternoon?` - 이 사람은 파일을 요청하고 있습니다.

- **Sam:** `Sure thing.` - 이는 화자가 동의하며 요청한 대로 파일을 보내겠다는 의미입니다.
