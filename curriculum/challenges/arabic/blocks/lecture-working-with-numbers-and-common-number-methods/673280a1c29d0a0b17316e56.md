# --interactive--

طريقة `.toFixed()` هي دالة مدمجة في JavaScript تُنسق الرقم باستخدام التدوين الثابت. تفيد بشكل خاص عندما تحتاج إلى التحكم في عدد الأرقام العشرية في الرقم، خصوصًا عند عرض قيم العملات أو عند العمل مع قياسات دقيقة.

تُستدعى طريقة `.toFixed()` على رقم وتأخذ معلمة اختيارية واحدة، وهي عدد الأرقام التي تظهر بعد الفاصلة العشرية. تُرجع تمثيلًا نصيًا للرقم بعدد الأرقام العشرية المحددة. إليك مثالًا بسيطًا على كيفية عمل `.toFixed()`:

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed(2)); // "3.14"
```

:::

في هذه الحالة، نحدد عدد الأرقام العشرية إلى اثنين. إذًا، `3.14159` تصبح `3.14`. من المهم ملاحظة أن `.toFixed()` تُرجع سلسلة نصية، وليست رقمًا. ذلك لأن الطريقة مخصصة أساسًا لتنسيق الأرقام للعرض، وليس للحسابات الإضافية.

طريقة `.toFixed()` تُقرب الرقم إلى أقرب قيمة يمكن تمثيلها بعدد الأرقام العشرية المحددة. من المهم فهم هذا السلوك الخاص بالتقريب:

:::interactive_editor

```js
console.log((3.14159).toFixed(3));  // "3.142"
console.log((3.14449).toFixed(3));  // "3.144"
console.log((3.14550).toFixed(3));  // "3.146"
```

:::

كما ترى، `.toFixed()` تُقرب للأعلى عندما يكون الرقم التالي `5` أو أكبر، وتُقرب للأسفل خلاف ذلك. إذا استدعيت `.toFixed()` بدون معلمات، فإنها تفترض `0` أرقام عشرية:

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed()); // "3"
```

:::

طريقة `.toFixed()` مفيدة بشكل خاص عند العمل مع الحسابات المالية أو عرض الأسعار:

:::interactive_editor

```js
let price = 19.99;
let taxRate = 0.08;
let total = price + (price * taxRate);

console.log("Total: $" + total.toFixed(2)); // "Total: $21.59"
```

:::

في هذا المثال، تضمن `.toFixed(2)` أن الإجمالي يُعرض دائمًا برقمين عشريين، وهو المعيار في عرض العملات في العديد من البلدان.

في الختام، طريقة `.toFixed()` أداة قوية لتنسيق الأرقام في JavaScript، خصوصًا عندما تحتاج إلى التحكم في عرض الأرقام العشرية. رغم أنها تُستخدم أساسًا لتنسيق المخرجات، تذكر سلوكها جيدًا، خاصة عند الحاجة إلى حسابات دقيقة.
