# --description--

Kitendakazi cha mshale ni njia ya kisasa zaidi ya kuandika kitendakazi katika JavaScript. Hutoa sintaksia fupi zaidi ikilinganishwa na usemi wa kitendakazi wa jadi. Kitendakazi cha mshale huandikwa kwa kutumia sintaksia ya `=>`.

```js
const add = (a, b) => {
  return a + b;
}
```

Katika mfano hapo juu, kitendakazi cha mshale `add` kinapokea vigezo viwili `a` na `b` na hurudisha jumla yao. Neno kuu `return` hutumika kurudisha matokeo ya operesheni ya kuongeza.

Ikiwa kitendakazi cha mshale kina usemi wa kihesabu mmoja tu, mabano ya mviringo `{}` na neno kuu `return` vinaweza kuachwa. Usemi huo wa kihesabu utarudishwa kwa njia isiyo ya moja kwa moja.

```js
const add = (a, b) => a + b;
```

Katika mfano hapo juu, kitendakazi cha mshale `add` kinapokea vigezo viwili `a` na `b` na hurudisha jumla yao. Neno kuu `return` na mabano ya mviringo `{}` vimeachwa, na matokeo ya operesheni ya kuongeza hurudishwa kwa njia isiyo ya moja kwa moja.
