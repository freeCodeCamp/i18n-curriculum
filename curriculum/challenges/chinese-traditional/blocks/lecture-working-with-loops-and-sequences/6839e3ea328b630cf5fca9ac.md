# --description--

如你在先前的模組中所學，迴圈用於重複執行一個區塊的程式碼指定的次數。在本課程中，你將學習如何使用 Python 中不同型別的迴圈。

我們將先介紹的第一個迴圈是 `for` 迴圈。以下是一個使用 `for` 迴圈來迭代列表並將每個項目列印到主控台的範例：

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)
```

結果將會是：

```md
Rust
Java
Python
C++
```

請注意 `print(language)` 是縮排在迴圈內部。若沒有那個縮排，你會得到一個 `IndentationError`：

```python
"""
Traceback (most recent call last):
  File "<stdin>", line 4, in <module>
IndentationError: expected an indented block after 'for' statement on line 3
"""
```

你也可以使用 `for` 迴圈來迭代其他可迭代物件，例如字串。以下是使用 `for` 迴圈來迭代字串 `code` 並列印出每個字元的範例：

```py
for char in 'code':
    print(char)
```

結果將會是：

```md
c
o
d
e
```

就像在 JavaScript 中一樣，你也可以在 Python 中巢狀使用 `for` 迴圈。以下是使用巢狀 `for` 迴圈的範例：

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)
```

外層迴圈將會迭代 `categories` 列表中的每個 `category`。對於每個 `category`，內層迴圈將會迭代 `foods` 列表中的每個 `food`。以下是將會列印到主控台的結果：

```md
Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
```

你可以在 Python 中使用的另一種迴圈是 `while` 迴圈。這種類型的迴圈會重複執行一個區塊的程式碼，直到條件為 `False`。以下是一個使用 `while` 迴圈進行猜數字遊戲的範例：

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')
```

在這個範例中，我們有一個值為 `3` 的 `secret_number` 變數和一個初始猜測值為 `0`。然後我們使用 `input` 函式從使用者取得輸入，再用 `int()` 函式將輸入的字串轉換成整數，並指定給 `guess` 變數。如果使用者透過輸入 `3` 猜對了，則跳出 `while` 迴圈並在主控台列印訊息 `You got it!`。否則，主控台會列印訊息 `Wrong! Try again.`，迴圈會重複，提示使用者再次猜測。

以下是一個範例結果：

```md
Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
```

就像在 JavaScript 中一樣，Python 支援 `break` 和 `continue` 述語。

`break` 述語用於停止迴圈的執行。以下是使用 `break` 述語處理 `developer_names` 列表的範例：

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

在這個範例中，我們迭代 `developer_names` 的列表並將每個名稱列印到主控台。如果名稱與 `Naomi` 相等，則我們跳出迴圈。這導致只有名稱 `Jess` 被列印到主控台。

`continue` 述語用於跳過迴圈的當前迭代並進入下一次迭代。讓我們修改先前的範例，改用 `continue` 述語取代 `break`：

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

現在主控台中的結果會不同。因為當 `developer` 與 `Naomi` 相等時，`continue` 述語會跳過迴圈的第二次執行，並且不會將該名稱列印到主控台，所以會列印出 `Jess` 和 `Tom`。

`for` 和 `while` 迴圈都可以與 `else` 語句結合，該語句僅在迴圈未被 `break` 述語終止時執行。以下是使用多個 `for` 迴圈的範例：

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

在這個範例中，我們有一個隨機字、單字的列表，並使用 `for` 迴圈來遍歷每個字、單字。在外層的 `for` 迴圈內，我們有另一個 `for` 迴圈來遍歷每個字、單字的每個字母。如果該字母的小寫版本是母音，我們就列印該字、單字以及它包含的母音，然後跳出內層迴圈。如果該字、單字不包含母音，則列印一則表示此情況的訊息。

以下是在主控台中結果的顯示方式：

```md
'sky' has no vowels
'apple' contains the vowel 'a'
'rhythm' has no vowels
'fly' has no vowels
'orange' contains the vowel 'o'
```

迴圈在 Python 中非常常見，因此熟悉它們很重要。在接下來的幾堂課中，你將學習如何在迴圈中使用 `enumerate()` 和 `range()` 函式（程式）。
