# --description--

前のレッスンでは、複雑なデータをわかりやすく視覚的に表現するプロセスであるデータの可視化について紹介しました。このレッスンでは、D3ライブラリについて学びます。

D3ライブラリは、JavaScriptで魅力的で効果的なデータの可視化を作成するために使える無料のオープンソースライブラリです。

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-1.png" alt="A graph with four nodes labeled A, B, C, and D. Node A connects to B, C, and D. Node B connects to A and D. Node C connects to A. Node D connects to A and B. It's the same image as before.">


D3という名前はdata-driven documents（データ駆動型ドキュメント）の略で、データをDocument Object Model（DOM）の要素に動的に結びつけることを指しています。このライブラリは柔軟でウェブと自然に連携します。新しいグラフィカルな表現方法を一から学ぶ代わりに、`svg`や`canvas`要素を使ってデータの可視化を構築できます。

たとえば、D3を使って棒グラフを作成したい場合、1つのJSファイル内で次のことができます：

* チャートの寸法とマージンを表す変数を宣言する
    
* 水平方向の位置スケールを宣言する
    
* 垂直方向の位置スケールを宣言する
    
* 棒とラベルを含むSVGコンテナを作成する
    
<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-2.png" alt="A graph with four nodes labeled A, B, C, and D. Node A connects to B, C, and D. Node B connects to A and D. Node C connects to A. Node D connects to A and B. It's the same image as before.">

では、Chart.js、Plotly、Tableauなどの他の可視化ツールよりもD3を学ぶ理由は何でしょうか？

他のツールとは異なり、D3は多くの低レベルの制御を提供し、バニラのHTML、CSS、JavaScriptだけでカスタマイズ可能でインタラクティブな可視化を構築できます。さらに、ライブラリはJavaScriptで書かれているため、新しい言語を学ぶ必要がありません。

今後のワークショップやラボでは、選択、形状、スケールなどD3ライブラリの基本を学びます。散布図、棒グラフ、ダッシュボードアプリなどを作りながら、これらの重要な概念を習得します。
