# --description--

Sie können auf Anfragen mit einer Datei antworten, indem Sie die `res.sendFile(path)`-Methode verwenden. Sie können diese innerhalb des `app.get('/', ...)`-Route-Handlers platzieren. Im Hintergrund setzt diese Methode die passenden Kopfzeilen, um Ihrem Browser mitzuteilen, wie er die Datei, die Sie senden möchten, entsprechend ihrem Typ behandeln soll. Anschließend liest sie die Datei und sendet sie. Diese Methode benötigt einen absoluten Dateipfad. Wir empfehlen, die Node-globale Variable `__dirname` zu verwenden, um den Pfad so zu berechnen:

```js
absolutePath = __dirname + '/relativePath/file.ext'
```

# --instructions--

Senden Sie die `/views/index.html`-Datei als Antwort auf GET-Anfragen an den Pfad `/`. Wenn Sie Ihre Live-Anwendung ansehen, sollten Sie eine große HTML-Überschrift (und ein Formular, das wir später verwenden werden…) sehen, ohne angewendetes Styling.

**Hinweis:** Sie können die Lösung der vorherigen Aufgabe bearbeiten oder eine neue erstellen. Wenn Sie eine neue Lösung erstellen, bedenken Sie, dass Express die Routen von oben nach unten auswertet und den Handler für die erste Übereinstimmung ausführt. Sie müssen die vorherige Lösung auskommentieren, sonst antwortet der Server weiterhin mit einem String.

# --hints--
