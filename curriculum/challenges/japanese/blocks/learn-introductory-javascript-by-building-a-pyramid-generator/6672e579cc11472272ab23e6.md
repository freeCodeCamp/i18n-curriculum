# --description--


JavaScript は、ウェブを動かすプログラミング言語です。 ここまでに学んだ HTML や CSS とは異なり、JavaScript は、マークアップ (文書構造などの定義) ではなくロジック (論理) を記述するために使われることが一般的です。

In this project, you will learn the basics of JavaScript and apply those concepts to building a pyramid generator.

このピラミッド表示プログラムでは、文字の種類、ピラミッドの段数、およびピラミッドの方向を設定できるようにします。 その設定に基づいてピラミッドのような図形を生成します。

「コードをチェック」ボタンをクリックして次のステップに進み、プロジェクトを開始しましょう。
