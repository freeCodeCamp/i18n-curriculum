# --description--

I file sono classificati in base al loro contenuto e alla loro struttura. Il tipo di file determina come un file viene aperto e processato dai programmi per computer.

Identifichiamo i tipi di file in base alle loro estensioni. Un'estensione di file è un suffisso di tre o quattro lettere alla fine del nome del file. I tre tipi principali di file che troverai nelle applicazioni web sono HTML, CSS e JavaScript.

I file HTML contengono la struttura e il contenuto dell'applicazione web. Hanno un'estensione di file `.html`.

I file CSS definiscono gli stili. Hanno un'estensione di file `.css`.

E i file JavaScript aggiungono funzionalità più avanzate e interattività. Hanno un'estensione di file `.js`.

Durante lo sviluppo di applicazioni web, dovrai anche includere immagini. Questi sono alcuni dei tipi di file immagine più usati.

JPEG, che sta per "Joint Photographic Experts Group", è noto per la sua compressione efficiente. È ottimo per fotografie e immagini. Tuttavia, la compressione è lossy, il che significa che una parte della qualità dell'immagine viene persa per ridurre la dimensione del file.

PNG, che sta per "Portable Network Graphics", preserva la qualità dell'immagine anche dopo la compressione. Questo comporta file più grandi. PNG supporta la trasparenza ed è ottimo per immagini con bordi netti, come loghi e icone.

Il formato immagine GIF supporta sia l'animazione sia la trasparenza ma ha una palette di colori limitata. GIF sta per "Graphics Interchange Format".

Infine, abbiamo SVG, che sta per "Scalable Vector Graphics". È un formato di immagine vettoriale. Queste immagini possono essere scalate quanto serve senza perdere qualità.

Perfetto. Ora che conosci i formati immagine, vediamo alcuni formati video e audio.

MP3 è un formato audio noto per la sua compressione efficiente. È un formato lossy, quindi alcuni dati audio vengono persi durante la compressione per rendere i file più piccoli.

MP4 è uno dei formati video più comuni. Offre una buona compressione e supporta più codec audio e video, sottotitoli e metadati.

MOV, il formato multimediale QuickTime sviluppato da Apple, è principalmente associato a QuickTime Player.

Altri tipi di file audio e video popolari sono WAV, un formato audio lossless che mantiene la qualità originale dell'audio, e WebM, un formato video open source di alta qualità.

Se mai dovessi personalizzare i caratteri tipografici della tua applicazione web, lavorerai anche con i formati di carattere. Questi sono tre dei più usati.

TTF, che sta per "TrueType Font", è un formato di carattere compatibile con diversi sistemi operativi.

WOFF è un formato di carattere moderno, progettato specificamente per lo sviluppo web. Sta per "Web Open Font Format". Questi file sono più piccoli perché hanno una compressione migliore. Possono anche contenere metadati, inclusi dati sulla licenza.

E WOFF2 è il successore di WOFF con una compressione e prestazioni ancora più efficienti.

Puoi anche creare file di archivio se hai bisogno di raggruppare più cartelle e file. Il formato di archivio più usato è ZIP. ZIP offre una compressione lossless, quindi nessun dato viene perso durante il processo. È ampiamente supportato tra sistemi operativi e applicazioni software.

Infine, parliamo della documentazione. Nelle applicazioni web, di solito troverai un file chiamato `README` che contiene informazioni sull'applicazione, come come usarla, come installarla, la sua licenza e come contribuire.

Di solito sono scritti in un formato di file chiamato Markdown. Con Markdown puoi creare documenti strutturati con intestazioni, sottotitoli, collegamenti, immagini, elenchi e altro. I file Markdown hanno un'estensione `.md` o `.markdown`.

Qui puoi trovare il file `README` del repository GitHub di freeCodeCamp: https://github.com/freeCodeCamp/freeCodeCamp/blob/main/README.md

Puoi creare questa struttura dettagliata e questo formato usando Markdown.

Leggerai e scriverai sicuramente molti file `README` durante la tua carriera. Conoscere questi tipi di file comuni è essenziale per lo sviluppo web.

Dai blocchi fondamentali di HTML, CSS e JavaScript ai formati immagine, video e audio, ogni tipo di file ha un ruolo importante nella creazione di applicazioni web interattive.
