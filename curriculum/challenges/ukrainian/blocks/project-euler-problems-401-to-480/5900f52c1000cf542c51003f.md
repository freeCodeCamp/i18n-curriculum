# --description--

Функція $lcm(a, b)$ позначає найменший спільний кратник чисел $a$ та $b$.

Нехай $A(n)$ — це середнє значення $lcm(n, i)$ для $1 ≤ i ≤ n$.

Наприклад: $A(2) = \frac{2 + 2}{2} = 2$ та $A(10) = \frac{10 + 10 + 30 + 20 + 10 + 30 + 70 + 40 + 90 + 10}{10} = 32$.

Нехай $S(n) = \sum A(k)$ для $1 ≤ k ≤ n$.

$S(100) = 122\\,726$.

Знайдіть $S(99\\,999\\,999\\,019)\bmod 999\\,999\\,017$.
