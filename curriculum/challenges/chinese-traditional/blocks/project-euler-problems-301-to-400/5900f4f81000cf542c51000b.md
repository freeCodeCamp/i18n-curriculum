# --description--

對於任何正整數 $n$，第 $n$ 個弱 Goodstein 數列 `\{g1, g2, g3, \ldots\}` 定義為：

- $g_1 = n$
- 對於 $k > 1$，$g_k$ 是透過將 $g_{k - 1}$ 以 $k$ 進位表示，將其解讀為 $k + 1$ 進位數字，然後減去 1 來獲得的。

當 $g_k$ 變成 0 時，序列終止。

例如，第 6 個弱 Goodstein 數列是 $\\{6, 11, 17, 25, \ldots\\}$：

- $g_1 = 6$。
- $g_2 = 11$，因為 $6 = 110_2$，$110_3 = 12$，且 $12 - 1 = 11$。
- $g_3 = 17$，因為 $11 = 102_3$，$102_4 = 18$，且 $18 - 1 = 17$。
- $g_4 = 25$，因為 $17 = 101_4$，$101_5 = 26$，且 $26 - 1 = 25$。

等等。

可以證明每個弱 Goodstein 數列都會終止。

令 $G(n)$ 為第 $n$ 個弱 Goodstein 數列中非零元素的數量。

可以驗證 $G(2) = 3$、$G(4) = 21$ 和 $G(6) = 381$。

也可以驗證對於 $1 ≤ n &lt; 8$，$\sum G(n) = 2517$。

找出 $\sum G(n)$ 在 $1 ≤ n < 16$ 時的最後 9 位數字。
