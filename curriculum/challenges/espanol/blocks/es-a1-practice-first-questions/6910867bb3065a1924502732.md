# --explanation--

Este diálogo entre Mateo y Camila incluye estas frases comunes:

`¿Cómo te llamas?` significa "What is your name?" (informal).

`Mi nombre es...` significa "My name is".

`¿Y tú?` significa "And you?". Esto se usa para devolver una pregunta de manera informal.

`Me llamo...` es la forma más común de decir tu nombre.
