# --description--

A frase `Se llama...` significa "O nome dele/dela é...".

Isso é comumente usado para se referir ao nome de outra pessoa.

Luna está usando essa frase para responder à pergunta de Sebastián.

# --instructions--

Ouça o áudio e complete a frase abaixo.

## --sentence--

`BLANK BLANK Carlos Andrés Vélez.`

## --blanks--

`Se`

### --feedback--

Este é o pronome reflexivo para a terceira pessoa (ele/ela). Lembre-se de colocar a primeira letra em maiúscula.

---

`llama`

### --feedback--

Esta é a forma conjugada do verbo `llamarse` para "ele" ou "ela".

# --explanation--

Luna diz `Se llama Carlos Andrés Vélez.`, que significa "O nome dele é Carlos Andrés Vélez".

`Se` indica que Luna está falando sobre o nome de outra pessoa (terceira pessoa).

Note os acentos no nome do meio e no sobrenome desse personagem.
