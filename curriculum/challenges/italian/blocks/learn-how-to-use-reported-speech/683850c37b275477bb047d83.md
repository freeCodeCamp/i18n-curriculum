# --explanation--

Un `log` è un registro di eventi, azioni o messaggi creato da un sistema informatico o software. Per esempio:

`The system logs showed several errors before the crash.` – Significa che i record contenevano informazioni su problemi che si sono verificati.

Un `crash` è quando un sistema informatico o programma si blocca improvvisamente e non funziona più correttamente. Per esempio:

`The application crash caused all unsaved data to be lost.` – Significa che il programma è fallito inaspettatamente e i dati non sono stati salvati.
