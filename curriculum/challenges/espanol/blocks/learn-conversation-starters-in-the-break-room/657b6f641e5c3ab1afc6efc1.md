# --description--

Anteriormente, aprendiste a hacer preguntas usando `do`. También puedes usarlo para crear oraciones negativas combinando `do` (para `I`, `you`, `we`, `they`) o `does` (para `he`, `she`, `it`) con la palabra `not` y colocándola justo antes del verbo principal:

- `I do not work everyday.` - Sigue esta estructura: sujeto + `do` + `not` + verbo principal.

- `It does not function.` - Sigue esta estructura: sujeto + `does` + `not` + verbo principal.

La gente a menudo abrevia `do not` como `don't` y `does not` como `doesn't` para ser más práctico. 

Al igual que con las preguntas, cuando usas `don't`, el verbo principal que viene después siempre estará en su forma base, sin importar de quién estás hablando.
