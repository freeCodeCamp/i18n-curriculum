# --description--

표준 52장 카드 덱에서, 4장의 카드 집합이 페어가 없고 같은 무늬가 두 장 이상 없는 경우 바두기입니다.

$f(n)$을 4장 부분집합이 바두기인 $n$장 카드를 고르는 경우의 수라고 합시다. 예를 들어, 표준 52장 카드 덱에서 5장을 고르는 경우의 수는 $2\,598\,960$이며, 이 중 4장 부분집합이 바두기인 경우는 $514\,800$이므로 $f(5) = 514800$입니다.

$4 ≤ n ≤ 13$인 모든 $n$에 대해 $\sum f(n)$을 구하세요.
