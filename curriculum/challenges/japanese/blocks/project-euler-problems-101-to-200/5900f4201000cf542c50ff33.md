# --description--

任意の整数 $n$ に対して、次の3つの関数を考えます。

$$\begin{align}
  & f_{1,n}(x,y,z) = x^{n + 1} + y^{n + 1} − z^{n + 1}\\\\
  & f_{2,n}(x,y,z) = (xy + yz + zx) \times (x^{n - 1} + y^{n - 1} − z^{n - 1})\\\\
  & f_{3,n}(x,y,z) = xyz \times (x^{n - 2} + y^{n - 2} − z^{n - 2})
\end{align}$$

そして、それらの組み合わせとして

$$\begin{align}
  & f_n(x,y,z) = f_{1,n}(x,y,z) + f_{2,n}(x,y,z) − f_{3,n}(x,y,z)
\end{align}$$

を定義します。

$(x,y,z)$ を「次数 $k$ の黄金の三つ組」と呼ぶのは、$x$、$y$、$z$ がすべて $\frac{a}{b}$ の形の有理数であり、$0 < a < b \leq k$ を満たし、かつ $f_n(x,y,z) = 0$ となる整数 $n$ が（少なくとも）1つ存在するときです。

$s(x,y,z) = x + y + z$ とします。

$t = \frac{u}{v}$ は、次数35のすべての黄金の三つ組 $(x,y,z)$ に対する異なる $s(x,y,z)$ の和です。すべての $s(x,y,z)$ と $t$ は既約分数の形で表されます。

$u + v$ を求めてください。
