# --explanation--

La palabra `stressful` describe situaciones o eventos que causan estrés, tensión o ansiedad. A menudo se usa para hablar de experiencias que son desafiantes o exigentes. Por ejemplo:

- `The deadline was stressful because we had so much work to do.` - Significa que sintieron mucho estrés debido a la fecha límite.

- `Presenting to a large audience can be stressful.` - Significa que puedes sentirte realmente ansioso cuando presentas frente a una audiencia.

En esta pregunta, Lisa le está preguntando a James si arreglar el problema con la app y las credenciales fue una experiencia estresante para él.
