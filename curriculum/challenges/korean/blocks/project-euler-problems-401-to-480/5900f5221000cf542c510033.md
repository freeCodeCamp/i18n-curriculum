# --description--

줄리는 여동생 루이즈에게 다음과 같은 내기를 제안합니다.

누가 설거지를 할지 결정하기 위해 확률 게임을 하자고 제안합니다.

이 게임에서는 0과 1 사이에서 균등 분포하는 독립적인 난수 생성기를 사용합니다.

게임은 $S = 0$에서 시작합니다.

첫 번째 플레이어인 루이즈는 $S > 1$이 될 때까지 생성기에서 나온 서로 다른 난수를 $S$에 더하고, 마지막 난수 '$x$'를 기록합니다.

두 번째 플레이어인 줄리는 $S > 2$가 될 때까지 생성기에서 나온 서로 다른 난수를 $S$에 더하고, 마지막 난수 '$y$'를 기록합니다.

더 큰 수를 가진 플레이어가 이기고, 진 사람이 설거지를 합니다. 즉, $y > x$이면 두 번째 플레이어가 이깁니다.

예를 들어, 첫 번째 플레이어가 0.62와 0.44를 뽑으면, $0.62 + 0.44 > 1$이므로 첫 번째 플레이어의 차례가 끝나고 $x = 0.44$가 됩니다. 두 번째 플레이어가 0.1, 0.27, 0.91을 뽑으면, $0.62 + 0.44 + 0.1 + 0.27 + 0.91 > 2$가 되어 두 번째 플레이어의 차례가 끝나고 $y = 0.91$이 됩니다. $y > x$이므로 두 번째 플레이어가 이깁니다.

루이즈는 잠시 생각하다가 이렇게 말합니다: "그건 공평하지 않아".

두 번째 플레이어가 이길 확률은 얼마일까요? 소수점 아래 10자리까지 반올림하여 0.abcdefghij 형태로 답하세요.
