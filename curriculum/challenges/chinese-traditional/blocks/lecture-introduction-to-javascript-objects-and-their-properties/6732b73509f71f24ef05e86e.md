# --interactive--

在使用 JavaScript 時，你經常會遇到包含巢狀物件和物件內陣列的複雜資料結構。這些結構可以表述豐富的階層式資料，但也需要清楚了解如何存取和操作其中的資料。讓我們探討如何有效地導覽這些巢狀結構。

從巢狀物件存取屬性涉及使用點記法或括號記法，就像從簡單物件存取屬性一樣。然而，你需要串鏈這些存取器以深入巢狀結構。 

例如，讓我們考慮一個巢狀物件，表述一個具有聯絡資訊的人：

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};
```

要存取 `Alice` 的工作電話號碼，你可以這樣串鏈屬性存取器：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person.contact.phone.work); // "098-765-4321"
```

:::

你也可以使用括號表示法，當屬性名稱包含空格或特殊字元，或當你使用變數來存取屬性時，這種方法特別有用：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person['contact']['phone']['work']); // "098-765-4321"
```

:::

現在，讓我們來看看如何存取其中一個物件屬性值為陣列的資料。這裡是一個修改過的 `person` 物件，其中包含一個位址陣列：

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};
```

這裡是一個如何存取 `Alice` 的工作位址城市的範例：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};

console.log(person.addresses[1].city); // "Workville"
```

:::

在此範例中，`person.addresses` 取用的是位址陣列。要存取該陣列中的第二個位址，我們使用括號表示法和索引 `1`。接著，我們使用點表示法來存取該位址物件中的 `city`。

了解如何存取巢狀物件和陣列中的屬性，對於處理複雜資料結構至關重要。在未來的工作坊和實驗中，你將有機會練習操作這些類型的資料結構。
