# --description--

يمكنك التفكير في المتغيرات كـ"حاويات تخزين" للبيانات في شفرتك.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/variables-and-operators/top-learn-variables-and-operators.png" alt='the concept of variables in coding as boxes labeled "birthday" that store and provide access to data, specifically a date "16-09-2003"'>

المتغير هو "مخزن مسمّى" للبيانات. يمكنك استخدام المتغيرات لتخزين الأشياء الجيدة، الزوار، وبيانات أخرى.

لإنشاء متغير في JavaScript، استخدم الكلمة المفتاحية `let`.

العبارة أدناه تُنشئ (بمعنى آخر: تُعلن عن) متغيرًا باسم "message":

```js
let message;
```

الآن، يمكنك وضع بعض البيانات فيه باستخدام معامل التعيين `=`:

```js
let message;

// store the string 'Hello' in the variable named message
message = 'Hello'; 
```

لتكون مختصرًا، يمكنك دمج إعلان المتغير والتعيين في سطر واحد:

```js
let message = 'Hello';
```
