# --description--

Ángela hace una pregunta usando la estructura `¿Cuál es tu` + sustantivo.

Ella podría detenerse ahí. En cambio, añade otra pieza de información usando el conector `y` y un segundo sustantivo.

`y` es una conjunción usada para conectar dos sustantivos. Aquí está la estructura que ella usa:

`¿Cuál es tu` + noun + `y` + noun`?`

Esto permite que el hablante solicite más de un detalle en la misma pregunta.

# --explanation--

En la oración `¿Cuál es tu nombre y apellido?`, dos sustantivos están conectados con `y`:

- `Nombre`

- `Apellido`

La palabra `y` conecta dos piezas de información.

Ángela primero pide el `nombre` de la persona y luego añade el `apellido`. 

Esta estructura permite al hablante solicitar múltiples detalles en una sola pregunta:

`¿Cuál es tu correo y número?`

Aquí, la segunda pieza de información que Ángela pide es `apellido`.
