# --interactive--

Quando lavori con le immagini di sfondo in CSS, hai a disposizione diverse proprietà per controllare come queste immagini vengono visualizzate.

Le proprietà principali su cui ci concentreremo sono `background-size`, `background-repeat`, `background-position` e `background-attachment`.

Diamo prima un’occhiata alla proprietà `background-image`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

Il CSS sopra imposta un’immagine di sfondo di un gatto per l’elemento `body`.

Se vuoi impostare la dimensione dell’immagine di sfondo, puoi usare la proprietà `background-size`.

Puoi usare `contain` per scalare l’immagine il più grande possibile senza ritagliarla o distorcerla.

Ecco un esempio con `background-size: contain`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    min-height: 100px;
  }
</style>
```

:::

Stiamo impostando il `min-height` su `100px` così l’immagine di sfondo è visibile e il layout mantiene un’altezza di base, assicurando che anche con contenuti minimi il design appaia equilibrato e gradevole alla vista.

Se cambiamo la proprietà `background-size` usando il valore `cover`, allora l’immagine di sfondo si scalerà per coprire l’intero elemento `body` mantenendo il suo rapporto d’aspetto.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    min-height: 100px;
  }
</style>
```

:::

Negli esempi precedenti, probabilmente hai notato che l’immagine di sfondo si ripeteva continuamente.

Per impostazione predefinita, le immagini di sfondo si ripetono sia orizzontalmente sia verticalmente per riempire l’intero elemento. Tuttavia, puoi controllare questo comportamento.

Puoi usare la proprietà `background-repeat` con il valore impostato su `no-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

Con `background-size` impostato su `contain` e `background-repeat` impostato su `no-repeat`, l’immagine non si ripeterà più sullo schermo.

Se volessi ripetere l’immagine di sfondo orizzontalmente, puoi usare il valore `repeat-x` per la proprietà `background-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-x;
    min-height: 100px;
  }
</style>
```

:::

Se volessi ripetere l’immagine di sfondo verticalmente, puoi usare il valore `repeat-y` per la proprietà `background-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-y;
    min-height: 100px;
  }
</style>
```

:::

Per posizionare un’immagine di sfondo sullo schermo, puoi usare la proprietà `background-position`.

La proprietà `background-position` ti permette di impostare dove nell’elemento appare l’immagine di sfondo. Puoi usare parole chiave come `top`, `bottom`, `left`, `right` e `center`, oppure valori specifici in pixel o percentuali.

Ecco un esempio che usa `center` e `top` per `background-position`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

Questo codice CSS posiziona l’immagine di sfondo al centro dell’elemento orizzontalmente e in alto verticalmente.

Infine, `background-attachment` determina se l’immagine di sfondo scorre con il contenuto o rimane fissa quando la pagina viene scrollata.

I valori principali sono `scroll` (predefinito), dove l’immagine di sfondo scorre con il contenuto, e `fixed`, dove l’immagine di sfondo resta nella stessa posizione sullo schermo.

Ecco un esempio che usa il valore `fixed` per la proprietà `background-attachment`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-position: center top;
    background-attachment: fixed;
  }
</style>
```

:::

Questo codice CSS imposta l’immagine di sfondo in modo che rimanga fissa anche quando la pagina viene scrollata.

Se vuoi combinare alcune di queste proprietà in una sola riga, puoi farlo usando la proprietà abbreviata `background`.

Ecco un esempio:

:::interactive_editor

```html
<style>
  body {
    background: center top fixed
      url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

L’esempio sopra combina `background-image`, `background-position` e `background-attachment` in una sola riga.

Padroneggiando queste proprietà, avrai un ottimo controllo su come le immagini di sfondo vengono visualizzate nei tuoi design web, permettendo layout più belli e funzionali.
