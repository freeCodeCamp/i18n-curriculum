# --description--

字母 `q` 在西班牙語中稱為 `cu`。

它的發音類似於硬的 `k` 音，就像 `casa` 中的字母 `c`。

它總是出現在靜音的 `u` 之前，後面接著 `e` 或 `i`，例如在 `que` 或 `qui` 中。

一個範例是 `Queso`（「起司」）。
