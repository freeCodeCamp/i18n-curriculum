# --explanation--

`Navarro com dois erres.` segue a estrutura: `con` + número + substantivo plural

`Con` adiciona informações sobre ortografia. 

`Dos` fornece a quantidade.  

`Erres` refere-se a duas letras `r`.

Essa estrutura é comum ao esclarecer como uma palavra é escrita.
