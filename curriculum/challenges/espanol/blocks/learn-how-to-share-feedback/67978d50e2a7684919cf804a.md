# --explanation--

`To set goals` significa definir objetivos o metas específicas para alcanzar. Por ejemplo:

`The team set goals to improve customer satisfaction this year.` - Esto significa que el equipo estableció objetivos relacionados con la satisfacción del cliente.

`Quarter` se refiere a una cuarta parte de un año, o un período de tres meses. En el contexto empresarial, `the next quarter` significa el próximo período de tres meses en el año calendario o fiscal. Por ejemplo:

- `Our sales target for the next quarter is $1 million.` - Esto significa que el objetivo aplica para el próximo período de tres meses.

- `We review our performance at the end of each quarter.` - Esto se refiere a evaluar el progreso cada tres meses.
