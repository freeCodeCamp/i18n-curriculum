# --description--

En el formulario, los usuarios deben tener la capacidad de agregar varios tipos de comidas junto con sus conteos de calorías.

Cree un elemento `fieldset` con el `id` configurado como `breakfast`.

Dentro de ese elemento, cree un `legend` con el texto `Breakfast`, y un `div` vacío con la `class` configurada como `input-container`.
