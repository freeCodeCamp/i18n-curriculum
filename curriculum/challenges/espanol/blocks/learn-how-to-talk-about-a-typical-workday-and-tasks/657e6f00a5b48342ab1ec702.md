# --description--

Un `QA (Quality Assurance) tester` es alguien que verifica el software para asegurarse de que funcione bien y no tenga problemas.

`Quality` significa qué tan bueno o malo es algo, y `assurance` significa asegurarse de algo. Un `QA tester` tiene que probar el software para garantizar que su calidad sea buena y que funcione como debe.
