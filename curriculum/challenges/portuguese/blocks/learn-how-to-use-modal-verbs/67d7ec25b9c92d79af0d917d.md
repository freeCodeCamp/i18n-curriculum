# --explanation--

`Must have happened` é usado para fazer uma suposição lógica sobre algo no passado.  

`Must` expressa forte certeza. `Have` + `Past Participle` (como `happened`) refere-se a algo no passado.Por exemplo:

- `He must have forgotten to update the system.` - É provável que ele não tenha se lembrado de fazer a atualização.

- `They must have made some changes to the app.` - É provável que o app tenha sido atualizado ou melhorado.

Jake está usando evidências (`unusual activity on the servers`) para concluir que a violação `must have happened` na noite passada.Esta estrutura é útil para fazer suposições lógicas.
