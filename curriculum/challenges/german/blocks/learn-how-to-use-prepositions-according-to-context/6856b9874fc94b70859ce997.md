# --explanation--

`Update` bedeutet, den neuesten Stand oder die aktuellsten Informationen über etwas mitzuteilen. Zum Beispiel:

`She will update the team after each sprint.` – Das bedeutet, sie wird sie über den Fortschritt informieren.

`On` kann verwendet werden, um das Thema oder den Fokus einer Handlung oder Aussage zu zeigen. Es hilft dabei, das, worüber gesprochen wird, mit seinem Thema zu verbinden. Zum Beispiel:

`We had a meeting on software updates.` – Das bedeutet, das Treffen drehte sich um dieses Thema.
