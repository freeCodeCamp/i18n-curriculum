# --description--

`equals symbol` (`=`)는 할당에 자주 사용됩니다. 변수에 값을 할당하는 데 사용합니다. 변수 이름은 등호 기호 왼쪽에 쓰고, 변수에 저장하려는 값은 오른쪽에 씁니다.

예를 들어, `name = "Sophie";`는 문자열 `"Sophie"`를 변수 `name`에 할당합니다.

프로그래밍에서 `assign a value`는 변수에 특정 값을 주는 것을 의미합니다. `equals symbol` (`=`)는 이 목적에 자주 사용됩니다. 변수에 값을 할당하면 프로그램에 그 값을 나중에 사용할 수 있도록 기억하라고 지시하는 것입니다.

예를 들어, `age = 25`는 값 `25`를 변수 `age`에 할당합니다.
