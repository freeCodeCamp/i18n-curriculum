# --interactive--

L'elemento di annotazione non articolata, o elemento `u` in breve, viene usato per rappresentare testo in linea a cui è applicata un'annotazione non testuale.

Ecco un esempio di utilizzo dell'elemento `u` per evidenziare vari errori di ortografia:

:::interactive_editor

```html
<p>
  You can use the unarticulated annotation element to highlight
  <u>inccccort</u> <u>spling</u> <u>issses</u>.
</p>
```

:::

Nell'esempio, le parole `incorrect`, `spelling` e `issues` sono scritte in modo errato. Lo stile predefinito per l'elemento `u` è una sottolineatura nera sotto il testo.

In HTML4, l'elemento `u` veniva usato per scopi di definizione dello stile. Ma in HTML5, l'elemento `u` dovrebbe essere usato solo per indicare che il testo ha un'annotazione non testuale applicata.

Se vuoi definire lo stile di un testo con una sottolineatura, dovresti usare CSS invece di HTML.

L'elemento barrato, o elemento `s` in breve, dovrebbe essere usato per rappresentare quando un testo non è più accurato o rilevante. Ecco un esempio di utilizzo dell'elemento `s` per mostrare l'annullamento di un'attività:

:::interactive_editor

```html
<p><s>Tomorrow's hike will be meeting at noon.</s></p>

<p>Due to unforeseen weather conditions, the hike has been canceled.</p>
```

:::

In questo esempio, la prima frase è barrata perché l'escursione è stata annullata per motivi meteorologici.

L'elemento `s` non dovrebbe mai essere usato solo per mostrare modifiche a un documento. In quel caso, elementi più appropriati sarebbero l'elemento testo eliminato e l'elemento testo inserito.

L'elemento `ruby` rappresenta un testo piccolo mostrato sopra o sotto il testo principale. Viene tipicamente usato per mostrare la pronuncia dei caratteri dell'Asia orientale. Ecco l'esempio dell'elemento `ruby` dalla pagina MDN web docs:

:::interactive_editor

```html
<ruby> 明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp> </ruby>
```

:::

L'elemento `rp`, o elemento parentesi fallback ruby, viene usato come fallback per i browser che non supportano la visualizzazione delle annotazioni ruby.

L'elemento `rt`, o elemento testo ruby, viene usato per indicare il testo dell'annotazione ruby. Questo testo viene solitamente usato per la pronuncia o per dettagli di traduzione nella tipografia dell'Asia orientale.

Sebbene l'elemento `ruby` possa essere usato per altri tipi di annotazioni, il caso d'uso più comune è per la tipografia dell'Asia orientale.
