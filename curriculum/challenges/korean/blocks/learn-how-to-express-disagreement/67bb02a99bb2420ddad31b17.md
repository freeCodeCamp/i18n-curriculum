# --explanation--

`That's a good suggestion, but...`는 아이디어를 인정하면서 걱정을 제기하는 표현입니다. 이 구절은 존중하는 태도로 토론을 이어가면서도 걱정을 계속 논의할 수 있게 합니다. 예를 들어:

- **개발자:** `How about we use this new software to improve productivity?`

- **팀장:** `That's a good suggestion, but we should check if it's compatible with our current systems.` - 이 사람은 아이디어를 인정하지만 잠재적인 호환성 문제를 지적합니다.
