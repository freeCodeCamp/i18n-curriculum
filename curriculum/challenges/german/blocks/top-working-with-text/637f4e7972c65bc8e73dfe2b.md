# --description--

Sie haben vielleicht bemerkt, dass Sie in allen Beispielen dieser Lektion Elemente, die sich innerhalb anderer Elemente befinden, einrücken. Dies wird als Verschachteln von Elementen bezeichnet.

Wenn Sie Elemente innerhalb anderer Elemente verschachteln, entsteht eine Eltern-Kind-Beziehung zwischen ihnen. Die verschachtelten Elemente sind die Kinder, und das Element, in dem sie verschachtelt sind, ist der Elternteil.

Im folgenden Beispiel ist das `body`-Element der Elternteil und das `p` das Kind:

```html
<html>
  <head>
  </head>
  <body>
    <p>Lorem ipsum dolor sit amet.</p>
  </body>
 </html>
```

Wie in menschlichen Beziehungen können HTML-Elternelemente viele Kinder haben. Elemente auf derselben Verschachtelungsebene gelten als Geschwister.

Zum Beispiel sind die beiden `p`-Elemente im folgenden Code Geschwister, da sie beide Kinder des `body`-Tags sind und sich auf derselben Verschachtelungsebene befinden:

```html
<html>
  <head>
  </head>
  <body>
    <p>Lorem ipsum dolor sit amet.</p>
    <p>Ut enim ad minim veniam.</p>
  </body>
 </html>
```

Sie verwenden Einrückungen, um die Verschachtelungsebene für sich selbst und andere Entwickler, die in Zukunft mit Ihrem HTML arbeiten, klar und lesbar zu machen. Es wird empfohlen, alle Kind-Elemente um zwei Leerzeichen einzurücken.

Die Eltern-, Kinder- und Geschwisterbeziehungen zwischen Elementen werden später viel wichtiger, wenn Sie Ihr HTML mit CSS gestalten und mit JavaScript Verhalten hinzufügen. Für den Moment ist es jedoch nur wichtig, den Unterschied zu kennen, wie Elemente miteinander verbunden sind, und die Terminologie zu verstehen, die verwendet wird, um ihre Beziehungen zu beschreiben.
