# --description--

정수 $1 < a_1 < a_2 < \ldots < a_n$의 값이 주어졌을 때, 정수 값 $q_k ≥ 0$만 사용하여 선형 결합 $q_1a_1 + q_2a_2 + \ldots + q_na_n = b$를 고려하세요.

주어진 $a_k$ 집합에 대해 모든 $b$ 값이 가능한 것은 아닙니다. 예를 들어, $a_1 = 5$이고 $a_2 = 7$일 때, $b$가 1, 2, 3, 4, 6, 8, 9, 11, 13, 16, 18 또는 23이 되도록 하는 $q_1 ≥ 0$와 $q_2 ≥ 0$가 존재하지 않습니다.

사실, $a_1 = 5$와 $a_2 = 7$에 대해 $b$가 될 수 없는 가장 큰 값은 23입니다. 따라서 $f(5, 7) = 23$이라고 부릅니다. 마찬가지로, $f(6, 10, 15) = 29$이고 $f(14, 22, 77) = 195$임을 증명할 수 있습니다.

$p$, $q$, $r$가 소수이고 $p < q < r < 5000$일 때, $\sum f(pq, pr, qr)$를 구하세요.
