# --description--

`Frustration` は、何かを変えられなかったり達成できなかったりしてイライラしたり不満を感じる気持ちです。例えば、新しいスキルを習得しようとして難しいと感じるときに `frustrating` になることがあります。この感情は、物事が難しかったり計画通りに進まなかったりするときによく起こります。
