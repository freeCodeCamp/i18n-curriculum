# --interactive--

CSS 的 `transform` 屬性是一個強大的工具，允許你修改網頁上元素的視覺呈現，而不會影響其他元素的版面配置。它可讓你對元素套用各種轉換，例如旋轉、縮放、傾斜或在 2D 或 3D 空間中平移（移動）它們。

`transform` 屬性透過對元素的座標系統套用數學轉換來運作。這表示你可以操作元素的形狀和位置，同時保持它的原始位置和文件流程不變。

讓我們來探索一些常見的轉換函式。這裡有一個盒子元素的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
body {
  border: 2px solid black;
}

.box {
  width: 200px;
  height: 200px;
  background-color: red;
}
```

:::

我們已將 `body` 設定為實心黑色邊框，以便你可以看到巢狀在 `body` 元素內的 `.box` 元素。

`translate` 函式（程式）會將元素從其目前位置移動。以下是使用 `translate` 函式（程式）的更新範例：  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
body {
  border: 2px solid black;
}

.box {
  width: 200px;
  height: 200px;
  background-color: red;
  transform: translate(50px, 100px);
}
```

:::

這條 CSS 規則會將具有 `box` 類別的元素從其原始位置向右移動 50 像素，向下移動 100 像素。

`rotate` 函式（程式）會繞著固定點旋轉一個元素，這是先前對 `.box` 元素使用 `rotate` 函式（程式）的範例：  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: rotate(45deg);
}
```

:::

這將使元素順時針旋轉四十五度。

`scale` 函式允許你改變元素的大小。以下是一個範例：  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: scale(1.5, 2);
}
```

:::

這將使元素的寬度為原始大小的 1.5 倍，高度為 2 倍。

你可以在單一宣告中結合多個轉換：  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: translate(50px, 50px) rotate(45deg) scale(1.5);
}
```

:::

這將使元素向右和向下移動 50 像素，旋轉 45 度，並將其縮放為原始大小的 1.5 倍。

雖然 `transform` 屬性在創造視覺吸引人的設計方面非常強大，但在使用時考慮存取性非常重要。以下是一些需要注意的重要存取性問題。 

螢幕閱讀器可能無法準確傳達經過轉換的內容。例如，如果你使用 `transform` 重新排列元素的視覺順序，螢幕閱讀器仍會以原始 DOM 順序讀取內容。這可能會導致依賴螢幕閱讀器的使用者感到困惑。

使用 `scale` 調整文字大小時，請注意不要讓文字變得過小或過大。過小的文字可能難以閱讀，而過大的文字可能會超出其容器的上限溢位並變得無法閱讀。通常最好使用適當的字型樣式技術來調整文字大小。

如果你正在使用 `transform` 來製作動畫效果，請注意對動作敏感的使用者。過度或快速的動畫可能會造成不適，甚至觸發某些人的癲癇發作。請考慮提供讓使用者減少或關閉動畫的方式。當使用 3D 變形時，請記住並非所有使用者都以相同方式感知深度。請確保任何透過 3D 效果傳達的重要資訊，也能以 2D 格式或文字呈現。

如果你使用 `transform` 來隱藏或顯示內容，請確保內容仍可被螢幕閱讀器和鍵盤導覽存取。隱藏的內容應該是真正隱藏，例如使用 `display: none;` 或 `visibility: hidden;`，而不是僅僅視覺上移出螢幕。 

當將 `transform` 套用到像按鈕或聯結這類互動元素時，請確保可點擊區域保持直覺且容易成為標的。劇烈變形的按鈕在視覺上可能會令人困惑，或對於有運動障礙的使用者來說難以點擊。

總結來說，雖然 CSS 的 `transform` 屬性是用於創造視覺動態網頁設計的強大工具，但務必在考慮存取性的前提下負責任地使用它。請務必使用各種輔助技術測試你經過變形的元素，並考慮提供替代方式來存取可能受到變形影響的資訊或功能。
