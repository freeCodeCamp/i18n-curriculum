# --explanation--

`Public`는 모두가 이용할 수 있거나 커뮤니티가 공유하는 것을 의미합니다. 예를 들어:

`The repository is public so anyone can access it.` - 저장소가 비공개가 아니라 모두에게 열려 있음을 보여줍니다.

`Wi-Fi` (`Wireless Fidelity`의 줄임말)은 장치가 인터넷에 연결할 수 있게 하는 무선 인터넷 연결을 뜻합니다. 예를 들어:

`I connected my phone to the café's Wi-Fi to check my emails.` - 카페에서 케이블 없이 인터넷에 접근했음을 보여줍니다.
