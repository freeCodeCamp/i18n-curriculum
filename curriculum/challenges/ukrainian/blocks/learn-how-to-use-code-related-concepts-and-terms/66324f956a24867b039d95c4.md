# --description--

Слово `should` — це модальне дієслово, яке використовується для надання порад або пропозицій. За ним слідує базова форма дієслова. Наприклад, у реченні `You should try that restaurant` після `should` йде `try`, що є базовою формою дієслова.

Дієслово `pick` означає вибирати або обирати щось із кількох варіантів. У контексті програмування `picking a data type` означає вибір відповідного типу даних, наприклад цілого числа, рядка або булевого, залежно від того, що найкраще підходить для конкретного завдання.

`Match` означає переконатися, що дві речі підходять або добре поєднуються одна з одною. Наприклад, `This key matches the lock.` вказує, що ключ підходить до замка.
