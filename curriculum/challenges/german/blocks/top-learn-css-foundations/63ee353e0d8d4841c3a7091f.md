# --description--

Okay, Sie haben bisher schon einiges durchgenommen. Das Einzige, was jetzt noch fehlt, ist zu erklären, wie Sie all dieses CSS zu Ihrem HTML hinzufügen. Es gibt drei Methoden dafür.

Externes CSS ist die gebräuchlichste Methode, auf die Sie stoßen werden. Dabei wird eine separate Datei für das CSS erstellt und innerhalb der öffnenden und schließenden `<head>`-Tags eines HTML-Dokuments mit einem `<link>`-Element verlinkt (dies ist ein Void-Element und benötigt kein schließendes Tag):

Zuerst fügen Sie ein Void-Element `<link>`-Tag innerhalb der öffnenden und schließenden `<head>`-Tags der HTML-Datei hinzu. Das `href`-Attribut gibt den Speicherort der CSS-Datei an, entweder eine absolute URL oder, was Sie verwenden werden, eine URL relativ zum Speicherort der HTML-Datei. Im obigen Beispiel gehen Sie davon aus, dass sich beide Dateien im selben Verzeichnis befinden. Das `rel`-Attribut ist erforderlich und gibt die Beziehung zwischen der HTML-Datei und der verlinkten Datei an.

Dann haben Sie innerhalb der neu erstellten `styles.css`-Datei den Selektor (das `div` und `p`), gefolgt von einem Paar öffnender und schließender geschweifter Klammern, die einen „Deklarationsblock“ bilden. Schließlich platzieren Sie alle Deklarationen innerhalb des Deklarationsblocks. `color: white; ` ist eine Deklaration, wobei `color` die Eigenschaft und `white` der Wert ist, und `background-color: black;` ist eine weitere Deklaration.

Ein Hinweis zu Dateinamen: `styles.css` ist hier einfach der von Ihnen gewählte Dateiname. Sie können die Datei beliebig benennen, solange der Dateityp `.css` ist, wobei „style“ oder „styles“ am häufigsten verwendet wird.

Ein paar Vorteile dieser Methode sind:

1. Sie hält Ihr HTML und CSS getrennt, was dazu führt, dass die HTML-Datei kleiner ist und alles sauberer aussieht.
2. Sie müssen das CSS nur an einer Stelle bearbeiten, was besonders praktisch für Websites mit vielen Seiten ist, die alle ähnliche Stile verwenden.
