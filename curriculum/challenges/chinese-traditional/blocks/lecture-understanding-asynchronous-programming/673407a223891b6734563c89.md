# --description--

Fetch API 允許網頁應用程式進行網路請求，通常用於從伺服端擷取或傳送資料。這個 API 提供一個 `fetch()` 行為，你可以用來進行這些請求。讓我們來看一個如何使用 `fetch` 的基本範例：

```js
fetch('https://api.example.com/data')
```

在此範例中，我們正在對 `https://api.example.com/data` 發出 `GET` 請求。接著這會傳回我們需要轉換成 JSON 格式並可在任何地方使用的部分資料。 

預設情況下，Fetch API 使用 `GET` 方法來擷取資料。這將在下一課中涵蓋，以及其他常見的 HTTP 請求方法。

現在，讓我們討論一些常見的從網路擷取的資源型別。

在我們的網頁應用程式中，我們需要一些常見的資料，例如天氣資料、職業列表資料、國家名稱列表、國家代碼或國家旗標圖示／影像。使用這些資料，我們可以讓應用程式更具資訊性和互動性。多虧了 `Fetch API`，我們可以從網路取得這些資源。

影像是一些經常被擷取的資源。你可能會根據使用者操作，使用 fetch 靜態或動態地載入影像，並將它們顯示在你的網頁應用程式中。

程式本文檔是另一種經常被擷取的資源。這可能包括設定檔案、日誌檔案，甚至是你想在網頁上顯示的整份文件。

在某些情況下，你可能會擷取音訊或影片檔案。`Fetch API` 也能控制這些類型的資源，讓你能處理各種不同的資料型別。
