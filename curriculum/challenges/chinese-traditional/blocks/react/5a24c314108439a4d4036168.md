# --description--

既然你已經學會了 JSX 和 React 組件的基礎知識，現在是時候自己撰寫一個組件了。React 組件是 React 應用程式的核心建置區塊，因此熟悉撰寫它們非常重要。請記住，典型的 React 組件是一個繼承自 `React.Component` 的 ES6 `class`。它有一個 `render` 方法，該方法會傳回來自 JSX 的 HTML 或 `null`。這是 React 組件的基本形式。一旦你充分理解這點，就能準備開始建置更複雜的 React 專案。

# --instructions--

定義一個繼承自 `React.Component` 的類別 `MyComponent`。它的呈現方法應該傳回一個包含 `h1` 頁籤且內含文字：`My First React Component!` 的 `div`。請精確使用此文字，大小寫和標點符號皆須相符。也請確保為你的組件呼叫建構式。

使用 `ReactDOM.render()` 將此組件呈現到 DOM。你可以使用具有 `id='challenge-node'` 的 `div`。

# --solutions--

```jsx
// Change code below this line
class MyComponent extends React.Component {
  constructor(props) {
    super(props);
  }
  render() {
    return (
      <div>
        <h1>My First React Component!</h1>
      </div>
    );
  }
};

ReactDOM.render(<MyComponent />, document.getElementById('challenge-node'));
```
