Esta palabra se refiere a un calendario que muestra cuándo deben ocurrir las tareas o eventos.

# --explanation--

Un `project timeline` es un calendario que describe los hitos clave, fechas límite y tareas en un proyecto. Ayuda a los equipos a mantenerse organizados y completar el trabajo a tiempo. Por ejemplo:

`We need to adjust the project timeline to allow more time for testing.` - Esto significa que el cronograma necesita cambios para incluir más tiempo de prueba.
