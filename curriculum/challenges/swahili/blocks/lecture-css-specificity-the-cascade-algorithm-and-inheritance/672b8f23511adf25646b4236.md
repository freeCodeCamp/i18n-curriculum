# --interactive--

Algoriti ya Cascade ni mchakato unaotumika na kivinjari kuamua ni sheria gani za CSS zitumike wakati kuna mitindo mingi inayolenga kipengele kimoja. Inahakikisha kwamba mitindo inayofaa zaidi inatumiwa, kulingana na seti ya sheria zilizoelezwa vizuri.

Mchakato huanza na **uhusiano**. Kivinjari kwanza huchuja sheria zote za CSS ili kupata zile zinazotumika kwa kipengele kinachozungumziwa. Hii ni pamoja na kulinganisha vichaguzi na kuzingatia media queries zinazoweza kuwa zinatumika.

Media query ni mbinu ya CSS inayotumika kutekeleza mitindo kulingana na sifa za kifaa au eneo la kuona, kama upana wake, urefu wake, au mwelekeo wake.

Kisha, algoriti huzingatia **asili na umuhimu**. CSS inaweza kutoka vyanzo tofauti: mitindo ya msingi ya kivinjari (user-agent), mitindo iliyowekwa na mtumizi, na mitindo iliyotungwa na mwandishi (wewe).

Baada ya kuzingatia asili, algoriti hupima umuhimu wa kila sheria, ikitoa kipaumbele kwa sheria zilizo na `!important` ambazo hughairi sheria nyingine bila kujali chanzo chao.

Baada ya kuchuja kwa asili na umuhimu, algoriti huangalia **uwezo wa maalum**. Wakati sheria mbili kutoka asili na kiwango cha umuhimu sawa zinatumika, sheria yenye uwezo wa maalum zaidi ndiyo itakayotekelezwa.

Uwezo wa maalum ni kipimo cha jinsi kichaguzi kilivyo lengwa, ambapo vichaguzi maalum zaidi vina kipaumbele juu ya vichaguzi vya jumla.

Mwishowe, ikiwa kila kitu kingine ni sawa, **mpangilio wa kuonekana** huingia kazini. Wakati sheria mbili zina uwezo sawa wa maalum, ile inayojitokeza mwisho katika CSS ndiyo itatumika.

Hii ndiyo sababu mpangilio wa kuandika mitindo yako unaweza kuathiri matokeo.

Tuchukulie mfano.

Katika faili ya HTML, kuna kipengele kimoja cha aya. Kisha ndani ya CSS, tuna sheria mbili, kila moja ikilenga kipengele cha aya.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>example paragraph</p>
```

```css
p {
  color: blue;
}

p {
  color: green; 
}
```

:::

Sheria ya kwanza inaweka rangi ya maandishi ya aya zote kuwa buluu wakati sheria ya pili inaweka rangi ya maandishi ya aya zote kuwa kijani.

Basi ni rangi gani itatumika? Rangi ya kijani ndiyo itakayotekelezwa kwa vipengele vya aya.

Kwa kuzingatia uhusiano, asili na umuhimu, uwezo wa maalum, wigo, na mpangilio wa kuonekana, Algoriti ya Cascade inahakikisha kwamba CSS yako inafanya kazi kwa utabiri, ikikuruhusu kubuni kurasa za mtandao zenye muundo tata na wa kina zaidi.
