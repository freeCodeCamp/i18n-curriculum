# --description--

将数字增加或减少 1 是最常见的数值操作之一。

所以，为此有特殊的操作符：

- 递增 `++` 将变量增加 1：

```js
let counter = 2;

// works the same as counter = counter + 1, but is shorter
counter++;      

console.log(counter); // 3
```

- 递减 `--` 将变量减少 1：

```js
let counter = 2;

// works the same as counter = counter - 1, but is shorter
counter--;

console.log(counter); // 1
```

操作符 `++` 和 `--` 可以放在变量的前面或后面。

- 当操作符位于变量之后时，它是“后序形式”：`counter++`。

- “前缀形式”是操作符位于变量之前的情况：`++counter`。

这两个语句都执行相同的操作：将 `counter` 增加 `1`。

有区别吗？有，但你只能在使用 `++/--` 的返回值时看到。

让我们澄清一下。正如你所知，所有操作符都会返回一个值。递增/递减也不例外。前缀形式返回新值，而后序形式返回递增/递减之前的旧值。

要看到区别，以下是一个示例：

```js
let counter = 1;
let a = ++counter; // (*)

console.log(a); // 2
```

总结如下：

- 如果增量/减量的结果未被使用，使用哪种形式没有区别：

```js
let counter = 0;
counter++;
++counter;

console.log( counter ); // 2, the lines above did the same
```

- 如果你想增加一个值并立即使用该操作符的结果，你需要使用前缀形式：

```js
let counter = 0;
console.log( ++counter ); // 1
```

- 如果你想递增一个值但使用它之前的值，你需要使用后序形式：

```js 
let counter = 0;
console.log( counter++ ); // 0
```

# --questions--

## --text--

下面 JavaScript 代码中两个 `console.log` 语句的输出是什么？

```js
let counter = 1;
console.log(2 * ++counter); // Statement A
```

```js
let counter = 1;
console.log(2 * counter++); // Statement B
```

## --answers--

没有区别；两个 `console.log` 语句都会显示相同的结果。

---

语句 A 会记录 `4`，因为 `++counter` 在乘法之前递增值。语句 B 会记录 `2`，因为 `counter++` 在乘法之后递增值。

---

语句 A 会记录 `2`，因为 `++counter` 是语法错误。语句 B 会记录 `4`，因为 `counter++` 是递增计数器的正确方法。

---

两个语句都会记录 `3`，因为计数器在两个分支中都在 `console.log` 之前递增。
