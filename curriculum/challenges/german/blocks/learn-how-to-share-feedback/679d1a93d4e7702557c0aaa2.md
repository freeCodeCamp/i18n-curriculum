Dieses Wort beschreibt etwas, das mit Programmieren oder Softwareentwicklung zu tun hat.

# --explanation--

In `Past Simple`-Fragen bleibt das Hauptverb in seiner Grundform, weil das Hilfsverb `did` bereits die Vergangenheitsform trägt. Zum Beispiel:

`What time did you arrive?` (Nicht `did you arrived`)  

`Coding` als Adjektiv bezieht sich auf Dinge, die mit dem Schreiben von Software zu tun haben. Ein `coding issue` ist ein Problem im Code, das die Funktionalität eines Programms beeinträchtigt. Zum Beispiel:

`The development team is working on fixing a coding issue that caused system crashes.` – Das bedeutet, dass ein Problem im Code das System beeinträchtigte.
