- `A for loop is useful for going through a list.` - リストの各アイテムをチェックするのに使えます。

- `A while loop runs until a condition is false.` - ルールが真の間、ループが続きます。

- `Be careful, a while loop can run forever!` - 停止しなければ、ループは終わりません。
