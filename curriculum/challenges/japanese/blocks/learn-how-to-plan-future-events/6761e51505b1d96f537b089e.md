# --explanation--

`Going to` は未来の予定や意図を話すときに使います。例えば：

- `We are going to upgrade the system next month.` - これはアップグレードが予定されていて、まもなく行われることを意味します。
  
- `She is going to learn SQL next year.` - これは彼女が将来SQLを学ぶつもりであることを意味します。

この文では：

`We're` は `we are` の短縮形です。

`Going to` は行動が近い未来に予定されていることを示します。

`Need` は必要や義務を示す基本動詞です。
