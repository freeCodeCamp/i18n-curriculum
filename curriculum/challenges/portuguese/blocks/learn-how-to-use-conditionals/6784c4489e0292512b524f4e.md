# --explanation--

Lisa está usando o `Second Conditional` para discutir uma situação hipotética.

O `Second Conditional` é usado para falar sobre situações irreais ou improváveis no presente ou futuro.Segue a estrutura `if` + `Present Simple` ou `Past Continuous` (para a condição) e `might / would / could` + verbo base (para o resultado).Por exemplo:

- `If I had more time, I would learn a new programming language.` - Isso descreve uma situação hipotética onde ter mais tempo levaria ao aprendizado.

- `If the server were faster, it could handle more users.` - O servidor não é rápido agora, mas se fosse, poderia suportar mais usuários.
