# --description--

Нехай $C$ — коло з радіусом $r$, $x^2 + y^2 = r^2$. Ми вибираємо дві точки $P(a, b)$ та $Q(-a, c)$ так, щоб пряма, що проходить через $P$ і $Q$, була дотичною до $C$.

Наприклад, четвірка $(r, a, b, c) = (2, 6, 2, -7)$ задовольняє цю властивість.

Нехай $F(R, X)$ — кількість цілих четвірок $(r, a, b, c)$ з цією властивістю, де $0 < r ≤ R$ та $0 < a ≤ X$.

Ми можемо перевірити, що $F(1, 5) = 10$, $F(2, 10) = 52$ та $F(10, 100) = 3384$.

Знайдіть $F({10}^8, {10}^9) + F({10}^9, {10}^8)$.
