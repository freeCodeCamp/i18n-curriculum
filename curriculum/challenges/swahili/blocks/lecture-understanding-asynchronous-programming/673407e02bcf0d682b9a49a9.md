# --description--

Injini ya JavaScript ina uwezo wa kusoma, kuelewa, na kutekeleza msimbo wako. Inafanya kazi kama mabadilishaji unaochukua msimbo wako, kuubadilisha kuwa maagizo ambayo kompyuta inaweza kuelewa na kufanya kazi kwa mujibu wake.

Moja ya injini maarufu zaidi za JavaScript ni V8, iliyotengenezwa na Google, inayotumika katika Chrome na Node.js. Injini ya JavaScript hufanya kazi kwa hatua chache. Kwanza, inachanganua msimbo wako, ikiusoma mstari kwa mstari kuhakikisha hakuna kosa katika msimbo wa JavaScript. Kisha, inaubadilisha msimbo huu kuwa bytecode, ambayo ni toleo rahisi, la kati la msimbo wako ambalo ni rahisi kwa kompyuta kuelewa na kutekeleza. Mwishowe, inaendesha bytecode hii kutekeleza maagizo ya programu yako. Hapa kuna mfano wa msimbo wa JavaScript:

```js
const greeting = "Hello, World!";
console.log(greeting);
```

Unapotekeleza msimbo huu, injini ya JavaScript kwanza inauchanganua ili kuangalia kama kuna makosa ya sintaksia. Kuchanganua maana yake ni injini inasoma msimbo na kuuvunja kuwa muundo unaoweza kueleweka, ikikagua makosa njiani.

Kisha, inaandaa msimbo huo katika muundo wa kati (mara nyingi bytecode au msimbo wa mashine, kulingana na injini). Kuandaa ni mchakato wa kubadilisha msimbo unaoeleweka na binadamu kuwa muundo bora zaidi ambao kompyuta inaweza kuutekeleza kwa haraka zaidi.

Mwishowe, injini inatekeleza msimbo, ikichapisha `Hello, World!` kwenye konsoli.

Sasa, hebu tuzungumzie runtime ya JavaScript. Runtime ya JavaScript ni mazingira ambayo msimbo wako wa JavaScript unatekelezwa. Inajumuisha injini ya JavaScript (kama V8 katika Chrome au SpiderMonkey katika Firefox), ambayo inachakata na kutekeleza msimbo, pamoja na vipengele vya ziada vinavyotolewa na mazingira (kama kivinjari cha mtandao au Node.js, ambavyo utajifunza zaidi kuhusu katika funzo zijazo).

Wakati lugha kuu ya JavaScript inashughulikia mambo kama vile vigezo, mizunguko, na vitendakazi, runtime hutoa zana za ziada zinazomruhusu JavaScript kuingiliana na vitu nje ya lugha yenyewe, kama DOM (kwa ajili ya kurasa za mtandao) au Fetch API (kwa ajili ya kufanya maombi ya mtandao).

Kwa kifupi, runtime ndiyo inayomruhusu JavaScript kufanya zaidi ya kazi za msingi za programu – kama kuingiliana na kurasa za mtandao au kushughulikia matukio yanayotegemea muda – kwa kutoa vipengele hivi vya ziada zaidi ya lugha yenyewe.

Ingawa huna haja ya kujua kila undani wa injini na runtime ili kuandika JavaScript, kuwa na uelewa wa msingi kunaweza kusaidia kuandika msimbo wenye ufanisi zaidi na kutafuta hitilafu kwa ufanisi zaidi.
