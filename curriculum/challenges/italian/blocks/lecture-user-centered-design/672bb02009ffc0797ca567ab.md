# --interactive--

**NOTA**: Alcuni esempi interattivi potrebbero usare CSS e JavaScript che non hai ancora imparato. Non preoccuparti di capire tutto il codice. Lo scopo degli esempi è mostrarti anteprime di questi concetti di design per farti comprendere meglio come funzionano. Per vedere le anteprime, devi abilitare l'editor interattivo.

L'indicazione del progresso è un modo per mostrare agli utenti a che punto sono in un processo. Può essere usata in moduli, registrazioni e processi di configurazione. L'obiettivo è aiutare gli utenti a capire dove si trovano nel processo e quanto manca ancora da fare.

Per esempio, puoi usare una barra di indicazione del progresso per mostrare agli utenti cosa resta da fare quando compilano moduli. Non vuoi creare una situazione in cui l'utente deve compilare un modulo lungo e non sa quanti passaggi restano da completare. La trasparenza è fondamentale affinché l'utente sappia se ha abbastanza tempo per sedersi e completare il modulo o se deve tornare più tardi.

```js
const form = document.getElementById('multiStepForm');
const steps = form.querySelectorAll('.form-step');
const progressBar = form.querySelector('.progress-bar');
const progressText = form.querySelector('.progress-text');
const totalSteps = steps.length;

let currentStep = 0;

function updateProgress() {
  const percent = ((currentStep + 1) / totalSteps) * 100;
  progressBar.style.width = percent + '%';
  progressText.textContent = `Step ${currentStep + 1} of ${totalSteps}`;
}

function showStep(index) {
  steps.forEach((step, i) => {
    step.classList.toggle('active', i === index);
  });
  updateProgress();
}

form.querySelectorAll('.next-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    if (currentStep < totalSteps - 1) {
      currentStep++;
      showStep(currentStep);
    }
  });
});

form.querySelectorAll('.prev-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    if (currentStep > 0) {
      currentStep--;
      showStep(currentStep);
    }
  });
});

showStep(currentStep);

form.addEventListener('submit', e => {
  e.preventDefault();
  alert('Form submitted!');
});

```

Quando progetti una sezione di indicazione del progresso, ci sono alcune buone pratiche da tenere a mente. La prima considerazione è mantenerla semplice. Non vuoi sovraccaricare l'utente con troppe informazioni che lo frustrano e lo fanno abbandonare il sito.

La seconda considerazione è rendere possibile tornare ai passaggi precedenti. Questo è importante perché gli utenti potrebbero voler controllare le risposte precedenti o apportare modifiche.

Un'altra considerazione è rendere la sezione di indicazione del progresso facile da trovare. Se l'utente non la trova, non saprà a che punto è nel processo.

L'ultima considerazione è avere titoli di sezione chiari, percentuali o passaggi numerati. Se hai solo una barra di progresso senza contesto, l'utente non capirà cosa significa.

const form = document.getElementById('progressForm');
const steps = form.querySelectorAll('.form-step');
const progressBar = form.querySelector('.progress-bar');
const currentStepSpan = document.getElementById('currentStep');
const totalStepsSpan = document.getElementById('totalSteps');
const percentageSpan = document.getElementById('percentage');

const totalSteps = steps.length;
let currentStep = 0;

totalStepsSpan.textContent = totalSteps;

function updateProgress() {
  const percent = Math.round(((currentStep + 1) / totalSteps) * 100);
  progressBar.style.width = percent + '%';
  currentStepSpan.textContent = currentStep + 1;
  percentageSpan.textContent = percent + '%';

  // Update ARIA attribute
  form.querySelector('.progress-container').setAttribute('aria-valuenow', percent);
}

function showStep(index) {
  steps.forEach((step, i) => {
    step.classList.toggle('active', i === index);
  });
  updateProgress();
}

form.querySelectorAll('.next-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    if (currentStep < totalSteps - 1) {
      currentStep++;
      showStep(currentStep);
    }
  });
});

form.querySelectorAll('.prev-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    if (currentStep > 0) {
      currentStep--;
      showStep(currentStep);
    }
  });
});

form.addEventListener('submit', e => {
  e.preventDefault();
  alert('Form submitted! Thanks!');
});

showStep(currentStep);
