# --interactive--

Unapofanya kazi na picha za mandharinyuma katika CSS, una sifa kadhaa zinazopatikana ili kudhibiti jinsi picha hizi zinaonyeshwa.

Sifa kuu tutazozizingatia ni `background-size`, `background-repeat`, `background-position`, na `background-attachment`.

Kwanza, tuchukulie sifa ya `background-image`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

CSS hapo juu inaweka picha ya paka kama mandharinyuma kwa kipengele cha `body`.

Ikiwa unataka kuweka ukubwa wa picha ya mandharinyuma, unaweza kutumia sifa ya `background-size`.

Unaweza kutumia `contain` kupanua picha iwe kubwa iwezekanavyo bila kukata au kunyoosha.

Hapa kuna mfano ukiwa na `background-size: contain`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    min-height: 100px;
  }
</style>
```

:::

Tunaweka `min-height` kuwa `100px` ili picha ya mandharinyuma ionekane na mpangilio udumuishe urefu wa msingi, kuhakikisha kwamba hata kwa maudhui kidogo, muundo unaonekana sawa na kuvutia kwa macho.

Ikiwa tutabadilisha sifa ya `background-size` kutumia thamani ya `cover`, basi picha ya mandharinyuma itapanuka kufunika kipengele chote cha `body` huku ikidumisha uwiano wake wa sura.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    min-height: 100px;
  }
</style>
```

:::

Katika mifano iliyopita, huenda umeona kwamba picha ya mandharinyuma inarudia mara kwa mara.

Kwa chaguo-msingi, picha za mandharinyuma hurudia wima na usawa ili kujaza kipengele chote. Hata hivyo, unaweza kudhibiti tabia hii.

Unaweza kutumia sifa ya `background-repeat` na thamani yake kuwekwa kuwa `no-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

Kwa kuwa `background-size` imewekwa kuwa `contain` na `background-repeat` imewekwa kuwa `no-repeat`, picha haitarudi tena kwenye skrini.

Ikiwa ungependa kurudia picha ya mandharinyuma kwa usawa, unaweza kutumia thamani ya `repeat-x` kwa sifa ya `background-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-x;
    min-height: 100px;
  }
</style>
```

:::

Ikiwa ungependa kurudia picha ya mandharinyuma kwa wima, unaweza kutumia thamani ya `repeat-y` kwa sifa ya `background-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-y;
    min-height: 100px;
  }
</style>
```

:::

Ili kuweka picha ya mandharinyuma mahali kwenye skrini, unaweza kutumia sifa ya `background-position`.

Sifa ya `background-position` inakuwezesha kuweka mahali ndani ya kipengele ambapo picha ya mandharinyuma itaonekana. Unaweza kutumia maneno muhimu kama `top`, `bottom`, `left`, `right`, na `center`, au thamani maalum za pikseli au asilimia.

Hapa kuna mfano unaotumia `center` na `top` kwa `background-position`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

Msimbo huu wa CSS unaweka picha ya mandharinyuma katikati kwa usawa na juu kwa wima.

Mwishowe, `background-attachment` inaamua kama picha ya mandharinyuma itasogea pamoja na maudhui au itabaki imara wakati ukurasa unaposogezwa.

Thamani kuu ni `scroll` (kwa chaguo-msingi), ambapo picha ya mandharinyuma husogea na maudhui, na `fixed`, ambapo picha ya mandharinyuma inabaki mahali pale pale kwenye skrini.

Hapa kuna mfano unaotumia thamani ya `fixed` kwa sifa ya `background-attachment`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-position: center top;
    background-attachment: fixed;
  }
</style>
```

:::

Msimbo huu wa CSS unaweka picha ya mandharinyuma ibaki imara mahali hata ukurasa ukisogezwa.

Ikiwa ungependa kuunganisha sifa chache katika mstari mmoja, unaweza kufanya hivyo kwa kutumia sifa fupi ya `background`.

Hapa kuna mfano:

:::interactive_editor

```html
<style>
  body {
    background: center top fixed
      url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

Mfano hapo juu unaunganisha `background-image`, `background-position`, na `background-attachment` katika mstari mmoja.

Kwa kumudu sifa hizi, utakuwa na udhibiti mzuri juu ya jinsi picha za mandharinyuma zinaonyeshwa katika miundo yako ya mtandao, kuruhusu mipangilio yenye mvuto wa macho na yenye utendaji mzuri.
