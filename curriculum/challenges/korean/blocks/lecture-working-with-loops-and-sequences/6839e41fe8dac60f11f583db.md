# --description--

`range()` 함수는 정수 수열을 생성하는 데 사용됩니다. `range()` 함수의 기본 구문은 다음과 같습니다:

```py
range(start, stop, step)
```

필수 `stop` 인수는 생성되는 숫자 수열의 끝점을 나타내는 정수입니다. `range()` 함수를 사용하는 예는 다음과 같습니다:

```py
for num in range(3):
    print(num)
```

다음 코드는 `0`와 `2` 사이의 숫자 수열을 생성합니다. `3` 정수는 `stop` 인수가 비포함이기 때문에 포함되지 않습니다.

만약 `start` 인수가 지정되지 않으면 기본값은 `0`입니다. 그렇지 않으면 선택적 `start` 인수를 사용하여 정수 수열을 `0`이 아닌 다른 정수에서 시작할 수 있습니다. `1`와 `4` 사이의 정수 수열을 생성하는 예는 다음과 같습니다:

```py
for num in range(1, 5):
    print(num)
```

기본적으로 정수 수열은 `1`씩 증가합니다. 하지만 기본값을 변경하고 싶다면 선택적 `step` 인수를 사용할 수 있습니다. `2`와 `10` 사이의 짝수 정수 수열을 생성하는 예는 다음과 같습니다:

```py
for num in range(2, 11, 2):
    print(num)
```

앞서 언급했듯이 `range()` 함수에는 필수 인수가 하나뿐입니다. `range()`에 인수를 제공하지 않으면 `TypeError`이 생성됩니다:

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: range expected at least 1 argument, got 0
```

중요한 점은 `range()` 함수가 인수로 정수만 받으며, 부동소수점은 받지 않는다는 것입니다. 부동소수점은 `3.4`처럼 소수점이 있는 숫자임을 기억하세요. 만약 부동소수점을 인수로 전달하려 하면 `TypeError`이 발생합니다:

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

감소하는 순서로 정수 수열을 생성하려면 `step` 인수에 음수를 사용할 수 있습니다. 예시는 다음과 같습니다:

```py
for num in range(40, 0, -10):
    print(num)
```

다음 코드는 `40`, `30`, `20`, `10` 숫자를 그 순서대로 콘솔에 출력합니다.

`range()` 함수로 할 수 있는 또 다른 일은 `list` 생성자와 함께 사용하여 정수 리스트를 만드는 것입니다. `list` 생성자는 반복 가능한 객체를 리스트로 변환하는 데 사용됩니다. `2`와 `10` 사이의 짝수 정수 리스트를 생성하는 예는 다음과 같습니다:

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

`range()` 함수는 Python에서 정수 수열을 생성하는 매우 유용한 방법입니다. 익숙해지면 Python 프로그램에서 자주 사용하게 될 것입니다.
