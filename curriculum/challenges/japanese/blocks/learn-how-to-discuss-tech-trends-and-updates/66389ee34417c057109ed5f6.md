# --description--

`automate`とは、人の助けを必要とせずにプロセスを自動で動かすことを意味します。プログラミングでは、コードを書く、アプリケーションをテストする、エラーを見つけて修正するためにソフトウェアを使うことが含まれる場合があります。例えば、朝に自動で電源が入りコーヒーを淹れるコーヒーマシンは`automated`です。

`certain`という言葉は、すべてのものや人ではなく、特定のものや人について話すときに使われます。例えば、`certain books are on sale`と言うと、本の一部だけがセール中で、すべてではないことを意味します。
