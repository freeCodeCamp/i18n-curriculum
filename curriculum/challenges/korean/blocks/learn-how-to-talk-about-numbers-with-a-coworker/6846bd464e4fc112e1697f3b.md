# --explanation--

**서수**는 단계, 작업, 또는 단계와 같은 수열에서 어떤 것의 위치를 나타냅니다. 예를 들어 `first`, `second`, `third`, `fourth` 등이 있습니다.

대부분의 서수는 **기수**(1, 2, 3, 4...)에 올바른 어미를 붙여 만듭니다. 대부분의 서수는 `-th`를 붙이는 패턴을 따릅니다. 예를 들어 4는 `fourth`가 되고, 6은 `sixth`가 됩니다. 하지만 1, 2, 3은 불규칙합니다: 1은 `first`, 2는 `second`, 3은 `third`가 됩니다.

대화에서 `the first task`는 초기 단계에서 완료해야 할 첫 번째 작업임을 의미합니다.
