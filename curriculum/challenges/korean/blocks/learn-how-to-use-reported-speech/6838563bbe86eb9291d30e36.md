# --explanation--

`Past Perfect` 시제(`had` + `past participle` 형태의 동사)는 다른 과거 사건보다 먼저 일어난 행동을 말할 때 사용합니다. 어떤 행동이 먼저 일어났는지 보여줍니다. 예를 들어:

`The server had crashed before the backup started.` – 충돌이 먼저 일어났고, 그 다음에 백업이 시작되었습니다.

`Third Conditional` 구조(`if` + `past perfect` 형태의 동사: 가정 원인, 조동사 + `have` + `past participle` 형태의 동사: 가정 결과)는 현실과 다른 과거 조건과 그 가능한, 그러나 현실이 아닌 결과를 표현합니다. 다른 행동을 취했다면 상황이 어떻게 달라졌을지 말합니다. 예를 들어:

`If they had updated the software, they could have avoided the error.` – 이것은 업데이트가 일어나지 않았고, 오류가 피하지 못했다는 뜻입니다.
