# --explanation--

`To integrate`는 서로 다른 부분들을 하나의 시스템으로 작동하도록 결합하는 것을 의미합니다. 예를 들어:

`The new app integrates chat and calendar features in one place.` – 이는 새 앱이 채팅과 달력 기능을 함께 추가한다는 뜻입니다.

무언가가 `intuitive`라는 것은 설명 없이도 자연스럽게 사용하거나 이해하기 쉽다는 뜻입니다. 예를 들어:

`The new design is more intuitive, so users can navigate it easily.` – 이는 사용자가 단지 보고도 어떻게 사용하는지 이해할 수 있다는 뜻입니다.
