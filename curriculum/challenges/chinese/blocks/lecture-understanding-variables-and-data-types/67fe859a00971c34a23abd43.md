# --description--

在使用 Python 变量之前，理解数据类型非常重要。数据类型描述了变量所保存的值的种类。例如，一个数字、一段文本或一个列表项。编程语言使用数据类型，以便它们知道如何保存和处理不同种类的信息。

Python 是一种动态类型语言，类似于 JavaScript，这意味着你不需要为变量显式声明类型。语言会根据你赋值给变量的内容来判断变量的数据类型。

这是一些例子：

```python
name = 'John Doe' # Python knows this is a string
age = 25 # Python knows this is an integer
```

这与某些静态类型语言如 C#、Java 和 C++ 不同，在这些语言中，你必须像这样为变量声明类型：

```csharp
string name = 'John Doe'
int age = 25
```

Python 的动态类型指派特性使编码非常快速且更灵活，但由于类型错误仅在程序运行时检测，而非编译时检测，这可能导致意外的错误。

由于 Python 在你的程序**运行**时确定数据类型，类型相关的错误只有在那时才会被发现。当程序运行时，Python 会逐行执行你的代码。如果它遇到某一行时，期望某个对象以它无法实现的方式表现，Python 会停止并显示误差。

相比之下，某些语言会在你的程序 **编译** 之后再运行。编译意味着计算机会提前查看你的代码并准备运行。在此步骤中，这些语言可以在程序开始之前捕获类型误差。

你还不需要了解那些语言。重要的观点很简单：

- **在 Python 中，类型误差可能会在执行期间显现出来**，即程序实际运行并使用你的代码时。

- **编译型语言在编译步骤期间捕获类型误差**，在程序被允许运行之前。

因此，你可能要等到程序运行到那一行代码时，才会发现 Python 中的类型错误。

以下是你在 Python 中最常用的数据类型：

- 整数：没有小数的整数，例如 `10` 或 `-5`。

```python
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- 浮点数：带有小数点的数字，例如 `4.41` 或 `-0.4`。

```python
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- 字串：由单引号或双引号括起来的字符序列，如 `'Hello world!'`。

```python
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- 布尔：一种真或假的类型，写作 `True` 或 `False`。

```python
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- 集合：一个无序的唯一元素集合，如 `{4, 2, 0}`。
 
```python
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- 字典：一个由括弧状的花括弧包围的密钥-值点对集合，例如 `{'name': 'John Doe', 'age': 28}`。

```python
my_dictionary_var = {'name': 'Alice', 'age': 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- 元组：一个不可变的有序集合，用方括弧括起来，如 `(7, 8, 4)`。

```python
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- 范围：一组数字序列，通常用于循环，例如 `range(5)`。

```python
my_range_var = range(5)
print('Range:', my_range_var) # Range: range(0, 5)
```

- 列表：一个有序的集合，支持不同的数据类型的元素。
    
```python
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- None：一个表现值的缺失的特殊值。

```python
my_none_var = None
print('None:', my_none_var) # None: None
```

要获取变量的数据类型，你可以使用 `type()` 函数：

```python
my_var_1 = 'Hello world'
my_var_2 = 21

print(type(my_var_1)) # <class 'str'>
print(type (my_var_2)) # <class 'int'>
```

以下是本课涵盖的所有数据类型，以及它们在终端中的类型：

```python
my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 5, 8}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 5, 8)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>
```

内置的 `isinstance()` 函数让你查看一个变量是否匹配特定的数据类型。它接受一个对象和你想要检查的类型，然后返回一个布尔值。以下是一些示例：

```python
isinstance('Hello world', str) # True
isinstance(True, bool) # True
isinstance(42, int) # True
isinstance('John Doe', int) # False
```
