# --description--

Kigezo kilichotangazwa ndani ya kitendakazi kinaonekana tu ndani ya kitendakazi hicho. Hii huitwa wigo wa kigezo, na wigo huu maalum huitwa wigo wa ndani. Vigezo vilivyotangazwa nje ya kitendakazi huitwa vigezo vya kimataifa na vinaonekana katika programu nzima. Kwa mfano, msimbo ufuatao utatoa kosa:

```js
function myFunction() {
  let x = 10;
}

console.log(x);
```

Kigezo `x` kime tangazwa ndani ya kitendakazi `myFunction` na hakionekani nje ya kitendakazi hicho. Msimbo utatoa `ReferenceError` kwa sababu `x` haijatangazwa.

Ikiwa utatangaza kigezo nje ya kitendakazi, huitwa kigezo cha kimataifa na kinaonekana katika programu nzima. Kwa mfano, msimbo ufuatao utafanya kazi:

```js
let x = 10;

function myFunction() {
  console.log(x);
}

myFunction();
```

Ikiwa kigezo chenye jina sawa kitatangazwa ndani ya kitendakazi, basi kitaficha kigezo cha nje.
