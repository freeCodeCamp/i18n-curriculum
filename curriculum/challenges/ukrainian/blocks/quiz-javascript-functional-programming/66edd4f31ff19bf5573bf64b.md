### --question--

#### --text--

Що таке чиста функція?

#### --distractors--

Функція, яка змінює глобальні змінні і повертає або `undefined`, або `null`.

---

Функція, яка виводить результат у консоль.

---

Функція, яка змінює свою поведінку залежно від зовнішніх факторів і повертає `null`.

#### --answer--

Функція, яка завжди повертає однаковий результат для однакових вхідних даних і не має побічних ефектів.

### --question--

#### --text--

#### --distractors--

Несподіваний `TypeError` у вашому коді.

---

Функція, яка виконується надто довго і викликає збій програми.

---

Рекурсивний виклик функції, що породжує нескінченний цикл і викликає збій програми.

#### --answer--

Зміна стану програми, яка спостерігається поза межами функції.

#### --distractors--

---

Оптимізований спосіб написання рекурсивних функцій, щоб вони працювали вдвічі швидше за інші.

---

Процес створення, проєктування та тестування коду, щоб він відповідав стандартам ECMAScript.

#### --answer--

Процес перетворення функції з кількома аргументами у послідовність функцій, кожна з яких приймає один аргумент.

Який із наведених прикладів є прикладом каррінгу?

#### --distractors--

```js
const curriedAverage = (a, b, c) => a + b + c / 3
```

---

```js
const curriedAverage = (a, b, c) => a + b + c
```

---

```js
function curriedAverage(a) {
  return a 
}

function curried(c) {
  return c
}
```

#### --answer--

```js
function curriedAverage(a) {
  return function(b) {
    return function(c) {
      return (a + b + c) / 3;
    };
  };
}
```
