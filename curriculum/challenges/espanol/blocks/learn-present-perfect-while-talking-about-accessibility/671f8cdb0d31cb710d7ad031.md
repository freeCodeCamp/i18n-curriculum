# --explanation--

La forma negativa del tiempo `Present Perfect Continuous` (`hasn't/haven't + been + present participle`) se usa para describir una acción en curso que no ha estado ocurriendo como se esperaba a lo largo del tiempo. En este contexto, indica que el problema ha estado afectando solo a algunos videos, no a todos.
