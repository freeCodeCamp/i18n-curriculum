# --description--

Die `range()`-Funktion wird verwendet, um eine Folge von ganzen Zahlen zu erzeugen. Hier ist die grundlegende Syntax für die `range()`-Funktion:

```py
range(start, stop, step)
```

Das erforderliche `stop`-Argument ist eine ganze Zahl, die den Endpunkt für die erzeugte Zahlenfolge darstellt. Hier ist ein Beispiel für die Verwendung der `range()`-Funktion:

```py
for num in range(3):
    print(num)
```

Der folgende Code erzeugt eine Zahlenfolge zwischen `0` und `2`. Die ganze Zahl `3` ist nicht enthalten, da das `stop`-Argument nicht inklusive ist.

Wenn kein `start`-Argument angegeben wird, ist der Standardwert `0`. Andernfalls können Sie das optionale `start`-Argument verwenden, um die Folge ganzer Zahlen bei einer anderen ganzen Zahl als `0` zu starten. Hier ist ein Beispiel für die Erzeugung einer Folge ganzer Zahlen zwischen `1` und `4`:

```py
for num in range(1, 5):
    print(num)
```

Standardmäßig wird die Folge ganzer Zahlen um `1` erhöht. Wenn Sie diesen Standardwert ändern möchten, können Sie das optionale `step`-Argument verwenden. Hier ist ein Beispiel für die Erzeugung einer Folge gerader ganzer Zahlen zwischen `2` und `10`:

```py
for num in range(2, 11, 2):
    print(num)
```

Wie bereits erwähnt, gibt es nur ein erforderliches Argument für die `range()`-Funktion. Wenn Sie keine Argumente an `range()` übergeben, erhalten Sie eine `TypeError`:

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: range expected at least 1 argument, got 0
```

Es ist wichtig zu beachten, dass die `range()`-Funktion nur ganze Zahlen als Argumente akzeptiert, keine Fließkommazahlen. Denken Sie daran, dass Fließkommazahlen Zahlen mit Dezimalstellen sind, wie `3.4`. Wenn Sie versuchen, Fließkommazahlen als Argumente zu übergeben, erhalten Sie eine `TypeError`:

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

Wenn Sie eine Folge ganzer Zahlen in absteigender Reihenfolge erzeugen möchten, können Sie für das `step`-Argument eine negative ganze Zahl verwenden, wie hier:

```py
for num in range(40, 0, -10):
    print(num)
```

Der folgende Code gibt die Zahlen `40`, `30`, `20` und `10` in dieser Reihenfolge in der Konsole aus.

Eine weitere Möglichkeit, die Sie mit der `range()`-Funktion haben, ist die Erstellung einer Liste ganzer Zahlen, indem Sie sie mit dem `list`-Konstruktor verwenden. Der `list`-Konstruktor wird verwendet, um ein iterierbares Objekt in eine Liste umzuwandeln. Hier ist ein Beispiel für die Erzeugung einer Liste gerader ganzer Zahlen zwischen `2` und `10`:

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

Die `range()`-Funktion ist eine sehr praktische Möglichkeit, eine Folge ganzer Zahlen in Python zu erzeugen. Wenn Sie den Dreh raus haben, werden Sie sie wahrscheinlich oft in Ihren Python-Programmen verwenden.
