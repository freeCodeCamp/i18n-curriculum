# --description--

`That's strange` bedeutet, dass Sie etwas ungewöhnlich oder unerwartet finden. Zum Beispiel, wenn Sie sehen, dass es im Sommer schneit, könnten Sie sagen, `That's strange.` Es bedeutet, dass Sie nicht erwartet haben, dass das passiert.

`I'm sure` bedeutet, dass Sie sicher sind, dass etwas wahr ist. Zum Beispiel, wenn Sie wissen, dass der Geburtstag Ihres Freundes morgen ist, können Sie sagen, `I'm sure it's tomorrow.` Es bedeutet, dass Sie ohne Zweifel daran glauben.
