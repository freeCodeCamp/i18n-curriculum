# --description--

La frase `crazy part` si usa per esprimere che qualcosa è sorprendente o inaspettato. È come dire `the surprising thing is` o `the unexpected thing is`.

Per esempio, `The crazy part is that he finished all the work in one day.` Questo significa che è sorprendente che lui sia riuscito a farlo così in fretta.
