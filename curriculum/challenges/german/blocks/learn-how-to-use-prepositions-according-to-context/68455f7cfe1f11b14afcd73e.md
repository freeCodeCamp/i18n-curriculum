# --explanation--

`Layout` bezieht sich auf das Design oder die Struktur eines Raums. Zum Beispiel:  

`He shared the layout of the new office.` – Das bedeutet den Plan, wo Dinge platziert werden.

`Draft` ist eine frühe Version eines Dokuments. Zum Beispiel:  

`She wrote the first draft of the proposal.` – Es kann noch bearbeitet werden.

`Send it over` bedeutet, etwas zu übermitteln. Zum Beispiel:  

`I'll send it over once it's ready.` – Es bedeutet, eine Datei oder Nachricht zu teilen.

`To` zeigt, für wen die Handlung bestimmt ist. Zum Beispiel:  

`She gave the file to her manager.` – Es zeigt, wer es erhält.
