# --explanation--

`It was mentioned...` は**受動態**の構造でもあり、話者ではなく伝えられている内容にフォーカスしています。例えば：

`It was mentioned that the update improved security.` – これは話者を特定せずにアイデアを伝えています。

`it's` + 比較級の形容詞 + `to` + 動詞 の構造は、何かが他のものと比べてどのように行われるかを示すために使われます。例えば：

`It's faster to search with the new tool.` – これは新しいツールが古いツールと比べて検索時に時間を節約することを意味します。
