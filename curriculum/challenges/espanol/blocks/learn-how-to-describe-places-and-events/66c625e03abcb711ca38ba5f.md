# --explanation--

`This is` es una forma simple de comenzar una oración cuando describes algo.

`Frustrating` es un adjetivo que describe una sensación de estar molesto o irritado porque algo no funciona como se esperaba. Por ejemplo:

- `This is confusing.` - La situación es difícil de entender.

- `This is exciting.` - La situación te hace sentir emocionado.
