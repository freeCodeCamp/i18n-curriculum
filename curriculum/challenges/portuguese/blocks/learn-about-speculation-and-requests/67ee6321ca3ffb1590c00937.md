`We need to test the app with edge cases to make sure it works in all situations.` - Você precisa testar situações incomuns para garantir que o app está funcionando corretamente.

`The app showed unexpected behavior when I tried to open it.` - O aplicativo não funcionou como esperado quando você tentou abri-lo.
