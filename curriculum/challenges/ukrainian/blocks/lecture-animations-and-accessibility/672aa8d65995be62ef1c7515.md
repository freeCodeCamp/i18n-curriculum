# --interactive--

CSS анімації дозволяють створювати динамічні, візуально привабливі ефекти на вебсторінках без потреби в JavaScript або складному програмуванні. Вони забезпечують плавний перехід елементів між різними стилями протягом заданого часу.

У своїй основі CSS анімація складається з двох основних компонентів: правила `@keyframes` та властивості animation.

Правило `@keyframes` визначає етапи та стилі анімації. Воно вказує, які стилі повинен мати елемент у різні моменти анімації.

Ось приклад:

```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(0);
  }
}
```

Це правило `@keyframes`, з іменем `slide-in`, визначає анімацію, яка переміщує елемент зліва направо. Відсотки представляють прогрес анімації, де `0%` — початок, а `100%` — кінець.

Функція `translateX` у вашій анімації `@keyframes` контролює горизонтальне положення елемента під час анімації.

Щоб застосувати цю анімацію до елемента, використовуйте властивість `animation`. У цьому прикладі анімація повторюється безкінечно, щоб ви могли побачити її в дії (вам потрібно увімкнути інтерактивний редактор для перегляду):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="sliding-element">Hello, I slide in!</div>
```

```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(0);
  }
}

.sliding-element {
  animation: slide-in 2s ease-out infinite;
}
```

:::

Це застосовує анімацію `slide-in` до елемента з тривалістю 2 секунди та функцією часу `ease-out`.

Властивість `animation` фактично є скороченням для кількох окремих властивостей:

`animation-name`, яка вказує, яке правило `@keyframes` використовувати.

`animation-duration`, що задає, скільки часу має тривати анімація.

`animation-timing-function`, яка визначає, як анімація змінюється з часом — наприклад, `ease`, `linear`, `ease-in-out`.

`animation-delay`, що задає затримку перед початком анімації.

`animation-iteration-count`, яка встановлює, скільки разів анімація має повторюватися.

`animation-direction`, що визначає, чи анімація має відтворюватися вперед, назад або чергуватися.

`animation-fill-mode`, яка вказує, як елемент має бути стилізований до та після анімації.

`animation-play-state`, що дозволяє призупиняти та відновлювати анімацію.

Ви можете використовувати ці властивості окремо для більш точного контролю:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="complex-animation">Watch my colors change!</div>
```

```css
.complex-animation {
  animation-name: color-change;
  animation-duration: 3s;
  animation-timing-function: linear;
  animation-delay: 1s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}

@keyframes color-change {
  0% {
    background-color: red;
  }
  50% {
    background-color: blue;
  }
  100% {
    background-color: green;
  }
}
```

:::

Це створює анімацію, яка безперервно змінює колір фону елемента між червоним, синім і зеленим.

CSS анімації можуть запускатися різними подіями, наприклад, при наведенні курсору на елемент:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Hover over me!</button>
```

```css
.button {
  background-color: blue;
  transition: background-color 0.3s;
}

.button:hover {
  background-color: red;
}
```

:::

Хоча цей приклад використовує властивість `transition`, яка простіша для базових ефектів, він демонструє, як CSS може створювати інтерактивні анімовані елементи.

Важливо пам’ятати, що хоча CSS анімації потужні, їх слід використовувати помірковано. Надмірне використання анімацій може призвести до поганої продуктивності та відволікати або створювати проблеми для користувачів з певними потребами у доступності. Завжди враховуйте можливість надати опції для зменшення або вимкнення анімацій для користувачів, які віддають перевагу меншій кількості руху.

CSS анімації пропонують спосіб створення захопливих, інтерактивних вебдосвідів без залежності від JavaScript. Розуміючи принципи `@keyframes` та властивостей `animation`, ви зможете оживити свої вебдизайни ефективно та доступно.
