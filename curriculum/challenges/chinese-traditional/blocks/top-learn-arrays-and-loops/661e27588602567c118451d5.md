# --description--

既然你已經對陣列有基本的了解，讓我們來談談迴圈。迴圈用於多次執行一個區塊的程式碼。其中一種迴圈是 `while` 迴圈。只要條件為真，`while` 迴圈就會執行一個區塊的程式碼。`while` 迴圈的語法如下：

```javascript
while (condition) {
  // code block to be executed
}
```

例如，以下程式碼片段會列印從 1 到 5 的數字：

```javascript
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
```

上述程式碼片段將變數 `i` 初始化為值 `1`。只要 `i` 的值小於或相等於 `5`，`while` 迴圈就會執行該區塊的程式碼。在每次迭代中，`i` 的值會增加 `1`。
