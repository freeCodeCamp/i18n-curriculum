# --description--

У програмуванні `int` означає "ціле число". Це тип даних, який використовується для збереження цілих чисел, тобто чисел без дробової частини або десяткових знаків. Цілі числа можуть бути додатними, від’ємними або нулем.

Вибір типу даних `int` є важливим, коли працюють із цілими числами, наприклад, підрахунок елементів, індекси в масивах або будь-які випадки, коли дробові числа не потрібні.

Приклади:

Підрахунок елементів: `int itemCount = 5;`

Збереження віку: `int age = 30;`
