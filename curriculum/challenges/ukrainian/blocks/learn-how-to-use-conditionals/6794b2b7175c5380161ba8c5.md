# --explanation--

Давайте розглянемо, як аудіо дає підказки для відповіді на запитання:  

У `set up a reminder` слово `reminder` підказує, що завдання важливі. Якби це було не так, не було б потреби їх запам’ятовувати.  

Слово `daily` показує, що принаймні одне завдання, резервне копіювання роботи, потрібно виконувати регулярно.  

Ці слова допомагають зрозуміти, що завдання, про які говорить Марія, важливі і їх потрібно виконувати регулярно. Вони не є необов’язковими або вже виконаними. Це поточні завдання, які потребують уваги.
