# --interactive--

ID 選擇器是 CSS 中最強大的選擇器之一，允許開發者將樣式套用到具有唯一識別字的特定元素。

這使它們在標的需要獨特樣式的個別元素上非常有效。

ID 選取器由井字號符號（`#`）後接 ID 名稱所定義。它們在 HTML 文件中應該是唯一的，表示不應該有兩個元素共用相同的 ID。

這裡是一個使用 ID 選擇器的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="unique">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#unique {
  color: purple;
}
```

:::

在此範例中，ID 為 `unique` 的元素將其文字顏色設為 `purple`。

ID 選取器具有非常高的特異性，高於型別選取器和類別選取器，但低於行內樣式。ID 選取器的特異性值為 `(0, 1, 0, 0)`。

這表示 ID 選擇器可以改寫類別選擇器和型別選擇器，但可以被行內樣式改寫。
