# --explanation--

`Currently`は「現在の瞬間」を意味します。例えば：

`We currently have three open positions.` – これは「今、3つの求人がある」という意味です。

`Working on`は「特定のタスクやプロジェクトに忙しく取り組んでいる」という意味です。例えば：

`She is working on the design for the new homepage.` – これは「彼女が積極的にそれを設計している」という意味です。
