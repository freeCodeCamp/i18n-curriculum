# --interactive--

웹 폼을 만들 때 입력 요소의 모양과 동작이 중요합니다. 사용자 동작에 반응하는 입력이 있는 폼은 사용자 경험을 크게 향상시키며, 혼란이나 불편을 일으키지 않아야 합니다.

CSS는 폼을 더 직관적이고 사용자 친화적으로 만들 수 있는 여러 입력 가상 클래스를 제공합니다. 이 가상 클래스들을 자세히 살펴보겠습니다.

`:focus` 가상 클래스를 사용하면 사용자가 입력 요소를 선택하거나 클릭할 때 해당 입력 필드에 포커스를 맞출 수 있습니다. 이렇게 하면 사용자가 현재 어디에 입력하고 있는지 쉽게 알 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form>
  <input type="text" />
</form>
```

```css
input:focus {
  border: 2px solid crimson;
  outline: none;
}
```

:::

이름에서 알 수 있듯이 `:hover` 가상 클래스는 사용자가 요소 위에 커서를 올렸을 때 작동합니다. 사용자가 입력과 상호작용하기 전에 시각적 피드백을 제공하여 입력이 준비되었음을 효과적으로 알립니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form>
  <input type="text" />
</form>
```

```css
input:hover {
  background-color: orange;
}
```

:::

위 예제에서 사용자가 입력 요소 위에 마우스를 올리면 배경색이 주황색으로 바뀌어 해당 필드가 상호작용할 준비가 되었음을 알려줍니다.

`:checked` 가상 클래스는 사용자가 라디오 버튼이나 체크박스를 선택했을 때 다르게 스타일링할 수 있게 해줍니다. 이렇게 하면 사용자가 선택한 옵션을 쉽게 확인할 수 있습니다.

다음은 웹사이트 이용 약관에 동의하는 체크박스 예제입니다.

**참고**: 이 예제의 일부 CSS는 아직 배우지 않은 속성을 사용합니다. 이는 맞춤 체크박스를 만드는 방법을 이해하기 위한 예시일 뿐이며, 앞으로 수업과 워크숍에서 자세히 배우게 됩니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form>
  <label>
  Agree <input class="checkbox" type="checkbox" />
  </label>
</form>
```

```css
.checkbox {
  appearance: none;
  width: 18px;
  height: 18px;
  border: 2px solid #ccc;
  border-radius: 4px;
  display: inline-block;
  position: relative;
  cursor: pointer;
  transition: all 0.25s ease;
  vertical-align: middle; 
}

.checkbox:hover {
  border-color: #888;
}

.checkbox:checked {
  background-color: #4caf50;
  border-color: #4caf50;
}

.checkbox:checked::after {
  content: "";
  position: absolute;
  left: 4px;
  top: 0px;
  width: 5px;
  height: 10px;
  border: solid white;
  border-width: 0 2px 2px 0;
  transform: rotate(45deg);
}

.checkbox:focus {
  outline: 2px solid #90caf9;
  outline-offset: 2px;
}

```

:::

`:required`는 `required` 속성이 있는 입력 요소를 대상으로 합니다. 이는 사용자가 폼을 제출하려면 해당 필드를 반드시 작성해야 함을 알립니다.

다음은 필수 입력 필드가 포함된 폼 예제입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form action="#">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required />

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required />

  <label for="phone">Phone Number:</label>
  <input type="tel" id="phone" name="phone" />

  <button type="submit">Submit</button>
</form>
```

```css
input:required {
  border: 2px solid orange;
}
```

:::

폼 유효성 검사 시 `:valid` 가상 클래스를 사용해 유효성 검사 기준을 충족하는 입력 필드를 스타일링하고, `:invalid`를 사용해 기준을 충족하지 않는 입력 필드를 스타일링할 수 있습니다. 일반적으로 유효한 입력에는 녹색, 유효하지 않은 입력에는 빨간색을 사용합니다.

다음은 `:valid` 가상 클래스를 사용한 예제입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" />
</form> 
```

```css
input:valid {
  border-color: green;
}
```

:::

다음은 `:invalid` 가상 클래스를 사용한 예제입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form>  
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" />
</form>
```

```css
input:invalid {
  border-color: crimson;
}
```

:::

`:disabled` 가상 클래스는 비활성화된 입력 요소를 선택하고 스타일링할 수 있게 해줍니다. 이 요소들은 `disabled` 속성을 가지고 있어 사용자가 상호작용할 수 없습니다. 비활성화된 입력은 클릭, 포커스, 편집이 불가능합니다.

다음은 레이블과 비활성화된 입력 요소 예제입니다.

CSS에서는 비활성화된 입력을 대상으로 배경색을 `lightgray`로 지정하고, 커서 포인터를 `not-allowed`로 변경합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form>
  <label for="name">Name:</label>
  <input class="text-input" type="text" id="name" name="name" disabled />
</form>
```

```css
.text-input:disabled {
  background-color: lightgray;
  cursor: not-allowed;
}
```

:::

`cursor: not-allowed;` CSS 속성 값은 커서 모양을 변경하여 동작이 허용되지 않음을 나타냅니다.

이 속성이 적용된 요소는 일반적으로 원 안에 대각선이 그어진 원형 금지 기호가 있는 커서를 표시하여 사용자가 해당 요소가 상호작용 불가능하거나 클릭할 수 없음을 알 수 있게 합니다.

다음은 다른 입력 가상 클래스와 그 기능 예시입니다:

- `:autofill`: 브라우저가 자동으로 저장된 데이터로 입력 필드를 채울 때 스타일을 적용합니다.
- `:optional`: 필수가 아니며 비워둘 수 있는 입력 요소를 스타일링합니다.
- `:in-range`와 `:out-of-range`: 값이 지정된 범위 제약 조건 내에 있거나 벗어났는지에 따라 요소를 스타일링합니다.
