# --explanation--

`Evidence of` bedeutet Anzeichen oder Beweise dafür, dass etwas passiert ist. Zum Beispiel:

`The report showed evidence of user error.` – Der Bericht zeigte, dass eine Person einen Fehler gemacht hat.

Ein `brute force attack` ist, wenn jemand viele verschiedene Passwörter ausprobiert, bis eines funktioniert.

Ein `SQL injection` ist, wenn jemand versucht, in ein System einzubrechen, indem er schädlichen Code in ein Suchfeld oder Formular eingibt.

Jake spricht davon, nach Anzeichen für diese Arten von Angriffen zu suchen.
