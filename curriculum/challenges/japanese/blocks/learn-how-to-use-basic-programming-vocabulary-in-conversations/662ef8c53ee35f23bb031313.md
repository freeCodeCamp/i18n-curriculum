# --description--

`a lot of` の式は、可算名詞と不可算名詞の両方で使え、多量を示します。

可算名詞の場合、例えば `cars` には `a lot of cars` と言うことがあります。不可算名詞の場合、`water` のように `a lot of water` と言います。
