# --explanation--

質問 `¿Cuántos años tienes?` は誰かの年齢を尋ねています。

答えを見つけるには、フォームの `Edad` フィールドを見る必要があります。

フォームには `Edad`: 25 と表示されています。

したがって、正しい答えは **25** です。

この質問はスペイン語で一般的に次の構造で答えられます：

`Tengo` + 数字 + `años`

文字 **`ñ`** は単なる `n` に `virgulilla`（`˜`）が付いたものではなく、スペイン語のアルファベットで完全に別の文字であり、単語の発音と意味の両方を変えます。
