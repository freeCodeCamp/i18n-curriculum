# --description--

UTF-8, або UCS Transformation Format 8, — це стандартизоване кодування символів, яке широко використовується в інтернеті. Кодування символів — це метод, який комп’ютери застосовують для збереження символів у вигляді даних. По суті, увесь текст на вебсторінці — це послідовність символів, збережених у вигляді одного або кількох байтів. У комп’ютерній техніці байт — це одиниця даних, що складається з 8 бітів, або двійкових цифр. UTF-8 підтримує кожен символ із множини символів Unicode — це включає символи та знаки з усіх систем письма, мов і технічних символів. Ось приклад використання елемента `meta` з атрибутом `charset` для встановлення кодування символів у `UTF-8`:

```html
<meta charset="UTF-8" />
```

Встановивши кодування символів у UTF-8, ви забезпечите правильне відображення акцентованого символу `"e"` (`é`) на сторінці. Ось розширений приклад коду з використанням кодування символів UTF-8:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Examples of the UTF-8 encoding</title>
  </head>
  <body>
    <p>Café</p>
  </body>
</html>
```

Для кожного нового проєкту, який ви створюєте, слід додавати цей елемент `meta` з атрибутом `charset`, встановленим у `UTF-8`.
