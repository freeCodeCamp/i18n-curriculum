# --description--

從起始值 $a_1$ 以以下方式取得一個修改過的 Collatz 整數序列：

如果 `a_n` 可被 3 整除，則 $a_{n + 1} = \frac{a_n}{3}$。我們將此稱為一個大的向下步驟，"D"。

當 `a_n` 除以 3 餘數為 1 時，$a_{n + 1} = \frac{4a_n + 2}{3}$。我們將此稱為向上步驟，"U"。

當 `a_n` 除以 3 餘數為 2 時，$a_{n + 1} = \frac{2a_n - 1}{3}$。我們將此表示為一個向下的小步驟，「d」。

當某些 $a_n = 1$ 時，序列終止。

給定任何整數，我們可以列出步驟的序列。例如，如果 `a_1 = 231`，則序列 `\{a_n\} = \{231, 77, 51, 17, 11, 7, 10, 14, 9, 3, 1\}` 對應於步驟 `"DdDddUUdDD"`。

當然，也有其他序列是以相同的序列 `"DdDddUUdDD..."` 開頭。

例如，如果 $a_1 = 1004064$，則序列為 DdDddUUdDDDdUDUUUdDdUUDDDUdDD。

事實上，1004064 是最小可能的 $a_1 > {10}^6$，其開頭為序列 DdDddUUdDD。

大於 {10}^{15} 且以序列 "UDDDUdddDDUDDddDdDddDDUDDdUUDd" 開頭的最小 `a_1` 是多少？
