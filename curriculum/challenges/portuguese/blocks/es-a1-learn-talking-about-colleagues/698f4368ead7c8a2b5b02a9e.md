# --explanation--

Camila diz que Marisol `es ingeniera de control de calidad`, o que significa que ela é uma engenheira de controle de qualidade.

A palavra `ingeniera` significa "engenheira" na forma feminina. A forma masculina de "engenheira" é `ingeniero`.
