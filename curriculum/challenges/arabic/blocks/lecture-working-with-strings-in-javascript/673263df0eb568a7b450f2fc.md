# --interactive--

عند العمل مع السلاسل النصية في JavaScript، هناك أوقات تحتاج فيها إلى تضمين أحرف خاصة قد يسيء محرك JavaScript تفسيرها.

مهمتان شائعتان تتعلقان بإنشاء سطر جديد داخل سلسلة نصية وإضافة التهريب لبعض الأحرف (مثل علامات الاقتباس) لضمان ظهورها بشكل صحيح.

في العديد من لغات البرمجة، بما في ذلك JavaScript، يمكنك إنشاء سطر جديد في سلسلة نصية باستخدام حرف خاص يُسمى تخطى التسلسل. أكثر تخطى تسلسل شيوعًا للسطر الجديد هو `\n`.

على سبيل المثال، إذا أردت تقسيم سلسلة نصية إلى عدة أسطر، تستخدم `\n` حيث تريد أن يبدأ السطر الجديد:

:::interactive_editor

```js
let poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

يخبر تخطى التسلسل `\n` جافاسكريبت بإدخال فاصل سطر في تلك النقطة، مما يؤدي إلى عرض السلسلة النصية عبر عدة أسطر.

مفهوم مهم آخر عند العمل مع السلاسل النصية هو تهريب الأحرف. أحيانًا تحتاج إلى تضمين أحرف في السلسلة النصية يستخدمها JavaScript عادةً لشيء آخر، مثل علامات الاقتباس.

إذا استخدمت علامات الاقتباس داخل سلسلة نصية دون تهريبها، قد يسبب ذلك خطأ لأن JavaScript سيظن أنك تحاول إنهاء السلسلة.

على سبيل المثال، هذا سيسبب خطأ:

```js
let statement = "She said, "Hello!"";
```

يختلط الأمر على JavaScript لأنه يظن أن السلسلة تنتهي بعد كلمة `"said,"` بينما تريد أن تكون علامات الاقتباس حول `"Hello!"` جزءًا من السلسلة.

لإصلاح ذلك، يمكنك تهريب علامات الاقتباس الداخلية بوضع شرطة مائلة عكسية (`\`) قبلها:

:::interactive_editor

```js
let statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

تخبر الشرطة المائلة العكسية JavaScript بمعاملة علامات الاقتباس كأحرف حرفية، فتظهر بشكل صحيح في الناتج.

يمكنك أيضًا تهريب أحرف خاصة أخرى، مثل الشرطة المائلة العكسية نفسها (`\\`)، أو علامات الاقتباس المفردة داخل سلسلة محاطة بعلامات اقتباس مفردة (`\'`).

إليك مثالًا آخر باستخدام علامات الاقتباس المفردة:

:::interactive_editor

```js
let quote = 'It\'s a beautiful day!';
console.log(quote); // It's a beautiful day!
```

:::

بتهريب علامة الاقتباس المفردة بـ `\'`، يعرف JavaScript أن يضمّنها كجزء من السلسلة بدلًا من إنهائها مبكرًا.

التهريب وإنشاء الأسطر الجديدة ضروريان عند تنسيق المخرجات أو التعامل مع الأحرف الخاصة في السلاسل النصية. تساعدك هذه التقنيات على منع الأخطاء وضمان ظهور النص كما تريد بالضبط.
