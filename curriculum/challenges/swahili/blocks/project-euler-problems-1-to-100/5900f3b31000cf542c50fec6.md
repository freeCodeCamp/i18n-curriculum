# --description--

Chukulia sehemu, $\frac{n}{d}$, ambapo `n` na `d` ni nambari kamili chanya. Ikiwa `n` &lt; `d` na kigezo cha kawaida cha juu, ${{HCF}(n, d)} = 1$, inaitwa sehemu sahihi iliyorahisishwa.

Ikiwa tutaorodhesha seti ya sehemu sahihi iliyorahisishwa kwa `d` ≤ 8 kwa mpangilio wa ukubwa unaoongezeka, tunapata:

$$\frac{1}{8}, \frac{1}{7}, \frac{1}{6}, \frac{1}{5}, \frac{1}{4}, \frac{2}{7}, \frac{1}{3}, \frac{3}{8}, \frac{\textbf2}{\textbf5}, \frac{3}{7}, \frac{1}{2}, \frac{4}{7}, \frac{3}{5}, \frac{5}{8}, \frac{2}{3}, \frac{5}{7}, \frac{3}{4}, \frac{4}{5}, \frac{5}{6}, \frac{6}{7}, \frac{7}{8}$$

Inaonekana kwamba $\frac{2}{5}$ ni sehemu iliyoko mara moja kushoto ya $\frac{3}{7}$.

Kwa kuorodhesha seti ya sehemu sahihi iliyorahisishwa kwa `d` ≤ `limit` kwa mpangilio wa ukubwa unaoongezeka, pata nomino ya sehemu iliyoko mara moja kushoto ya $\frac{3}{7}$.

```js
function orderedFractions(limit) {
  const fractions = [];
  const fractionValues = {};
  const highBoundary = 3 / 7;
  let lowBoundary = 2 / 7;

  for (let denominator = limit; denominator > 2; denominator--) {
    let numerator = Math.floor((3 * denominator - 1) / 7);
    let value = numerator / denominator;
    if (value > highBoundary || value < lowBoundary) {
      continue;
    }
    fractionValues[value] = [numerator, denominator];
    fractions.push(value);
    lowBoundary = value;
  }

  fractions.sort();
  return fractionValues[fractions[fractions.length - 1]][0];
}
```
