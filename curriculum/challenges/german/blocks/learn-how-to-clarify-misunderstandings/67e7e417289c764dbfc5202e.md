# --explanation--

In der Phrase `limit what other parts can see` leitet `what` eine Nominalklausel ein, die als Objekt von `limit` dient.

Eine Nominalklausel ist eine Klausel, die innerhalb eines Satzes als Substantiv fungiert. Sie kann als Objekt oder Subjekt eines Satzes auftreten. Sie beginnt normalerweise mit Wörtern wie `that`, `what`, `whether`, `who`, `when`, `why`, `how` usw.

Nominalklauseln verwenden die normale Satzstellung (nicht die Fragewortstellung). Zum Beispiel:

- `The team is checking why the system crashed during the update.` – `Why the system crashed during the update` ist die Nominalklausel, die als Objekt von `checking` fungiert.

- `What the function returns depends on the input data.` – `What the function returns` ist eine Nominalklausel, die als Subjekt von `depends on` fungiert.
