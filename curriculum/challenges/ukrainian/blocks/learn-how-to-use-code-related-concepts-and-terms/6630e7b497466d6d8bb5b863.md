# --description--

У програмуванні `string` — це тип даних, який використовується для зберігання тексту. `string` може містити літери, числа та символи. Його часто представляють, укладаючи текст у лапки.

Наприклад, `"Hello, world!"` — це `string`.

# --questions--

## --text--

Що таке `string` у контексті програмування?

## --answers--

Фрагмент тексту

---

Тип числа

### --feedback--

Хоча `string` може містити числа, він головним чином використовується для тексту, а не лише числових значень.

---

Інструмент програмування

### --feedback--

`string` — це тип даних, який використовується в програмуванні, зокрема для тексту, а не інструмент.

---

Команда для запуску програми

### --feedback--

`String` — це тип даних для тексту, а не команда для виконання програм.
