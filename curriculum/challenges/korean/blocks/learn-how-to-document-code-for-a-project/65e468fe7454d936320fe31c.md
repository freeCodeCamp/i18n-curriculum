# --description--

동사 `to contain`는 무언가를 안에 담거나 가지는 것을 의미합니다. 문서나 README 파일의 맥락에서 `to contain`는 문서 안에 포함된 정보를 가리킵니다.

예를 들어, 병은 물을 `contain` 수 있으며, README 파일은 프로젝트에 관한 정보를 `contains` 수 있습니다.
