# --interactive--

在之前的课程中，我们一直在使用 `grid-template-columns` 属性并将值设置为几个分数单位。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

虽然下面的代码是完全有效的，但是有一种更简单的方法来重复部分或所有轨道列表。

`repeat()` 函数用于为列或行重复部分或所有轨道。 该函数接受重复计数和你希望重复的轨道。

下面是先前示例使用重复函数后的修改版本：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

浏览器中的样式不会改变，但这是为列编写重复值的一种更简洁的方式。

`repeat()` 函数将接受可用于行或列的任何有效模式。

这是一个使用 `repeat()` 函数将第 1 和第 3 列设置为 20 像素，第 2 和第 4 列设置为一个分数单位的示例。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(2, 20px 1fr);
  column-gap: 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

有时，你可能会选择写出每个单独的值，而不是使用 `repeat()` 函数。 但是有时这个函数会派上用场，特别是当你想要为轨道列表重复特定模式时。
