# --explanation--

`Installed`はソフトウェアがデバイスに設定されて適切に動作する状態を意味します。例えば：

`The installed program is working fine.` — 設定されたプログラムが正常に動作しています。

`By next day`、`by next week`、`by next month`は翌日／翌週／翌月の終わりまでにという意味です。例えば：

- `The update should be ready by next day.` — アップデートは翌日までに完了します。

- `We plan to finish testing by next month.` — テストは来月末までに終わらせるべきです。

トムはこれらのソフトウェアアップデートが緊急で、特定の期限内に完了しなければならないと言っています。
