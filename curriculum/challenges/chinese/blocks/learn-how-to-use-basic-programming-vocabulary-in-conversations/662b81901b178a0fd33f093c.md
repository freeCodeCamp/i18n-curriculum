## --text--

根据 Sophie，如何在 Python 中声明一个变量？

## --answers--

你通过以操作符命名来声明一个变量。

### --feedback--

这个选择是不正确的，因为变量不是以操作符命名，而是在操作符之前。

---

变量通过使用名为 `declare` 的命令来声明。

### --feedback--

Python 不需要像 `declare` 这样的特殊命令来创建变量；它直接使用赋值。

---

你使用变量名，后跟赋值操作符和一个值。

---

变量必须在名称前通过类型指派 `variable` 声明。

### --feedback--

Python 不使用任何像 `variable` 这样的关键字来声明变量。它只是使用变量名、赋值操作符和数值。
