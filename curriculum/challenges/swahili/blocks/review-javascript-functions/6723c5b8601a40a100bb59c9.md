## Kitendakazi za JavaScript

- Kitendakazi ni sehemu za msimbo zinazoweza kutumika tena ambazo hufanya kazi maalum.
- Kitendakazi kinaweza kutangazwa kwa kutumia neno kuu la `function` ikifuatiwa na jina, orodha ya vigezo, na sehemu ya msimbo inayofanya kazi hiyo.

:::interactive_editor

```js
function addNumbers(x, y, z) {
  return x + y + z;
}

console.log(addNumbers(5, 3, 8)); // Output: 16
```

:::

- Hoja ni thamani zinazopitishwa kwa kitendakazi wakati kinapoitwa.
- Wito wa kitendakazi ni mchakato wa kutekeleza kitendakazi katika programu kwa kubainisha jina la kitendakazi ikifuatiwa na mabano ya kawaida, kwa hiari ikiwa na hoja ndani ya mabano hayo.
- Kitendakazi kinapomaliza utekelezaji wake, daima hurejesha thamani.
- Kwa chaguo-msingi, thamani ya return ya kitendakazi ni `undefined`.
- Neno kuu la `return` hutumika kubainisha thamani itakayotolewa kutoka kwa kitendakazi na kumaliza utekelezaji wa kitendakazi hicho.
- Vigezo vya chaguo-msingi huruhusu kitendakazi kuwa na thamani zilizowekwa awali zitakazotumika ikiwa hoja haijatolewa wakati kitendakazi kinapoitwa. Hii hufanya kitendakazi kuwa na ufanisi zaidi na kuzuia makosa pale hoja fulani zinapoweza kutoletwa.

:::interactive_editor

```js
const calculateTotal = (amount, taxRate = 0.05) => {
  return amount + (amount * taxRate);
};

console.log(calculateTotal(100)); // Output: 105
```

:::

- Kitendakazi kisicho na jina ni kitendakazi ambacho hakina jina na kinaweza kupewa thamani katika vigezo. Kwa kuviweka katika vigezo, unaweza kuvitumia tena popote ambapo kigezo hicho kinapatikana.

:::interactive_editor

```js
const multiplyNumbers = function(firstNumber, secondNumber) {
  return firstNumber * secondNumber;
};

console.log(multiplyNumbers(4, 5)); // Output: 20
```

:::

## Kitendakazi cha mshale

- Kitendakazi cha mshale ni njia fupi zaidi ya kuandika kitendakazi katika JavaScript.

:::interactive_editor

```js
const calculateArea = (length, width) => {
  const area = length * width;
  return `The area of the rectangle is ${area} square units.`;
};

console.log(calculateArea(5, 10)); // Output: "The area of the rectangle is 50 square units."
```

:::

- Unapotangaza kitendakazi cha mshale, huna haja ya neno kuu la `function`.
- Ikiwa unatumia kigezo kimoja tu, unaweza kuachana na mabano ya kawaida yanayozunguka orodha ya vigezo.

:::interactive_editor

```js
const cube = x => {
  return x * x * x;
};

console.log(cube(3)); // Output: 27
```

:::

- Ikiwa mwili wa kitendakazi una usemi wa kihesabu mmoja tu, unaweza kuachana na mabano ya mviringo na neno kuu la `return`.

:::interactive_editor

```js
const square = number => number * number;

console.log(square(5)); // Output: 25
```

:::

## Wigo katika programu

- **Wigo wa kimataifa**: Huu ni wigo wa nje kabisa katika JavaScript. Vigezo vilivyotangazwa katika wigo wa kimataifa vinapatikana kutoka sehemu yoyote ya msimbo na huitwa vigezo vya kimataifa.
- **Wigo wa ndani**: Hii inahusu vigezo vilivyotangazwa ndani ya kitendakazi. Vigezo hivi vinapatikana tu ndani ya kitendakazi ambacho vimetangazwa na huitwa vigezo vya ndani.
- **Wigo wa block**: Block ni seti ya maagizo yaliyofungwa katika mabano ya mviringo `{}` kama vile katika maagizo ya `if`, au mizunguko.
- Uwezo wa kudhibiti upatikanaji wa vigezo kwa kutumia `let` na `const` hutoa udhibiti wa kina zaidi juu ya upatikanaji wa vigezo, kusaidia kuzuia makosa na kufanya msimbo wako uwe wa kutegemewa zaidi.
