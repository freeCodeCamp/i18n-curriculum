# --description--

このラボでは、文字列をタイトルケースに変換する関数を作成します。タイトルケースとは、各単語の最初の文字が大文字で、それ以外の文字が小文字になっている状態を指します。

`"Web Development Is Awesome"` はタイトルケースの文字列の例です。

**目的**：以下のユーザーストーリーを満たし、すべてのテストに合格してラボを完了してください。

**ユーザーストーリー：**

1. `titleCase` という名前の関数があり、文字列を引数として受け取る必要があります。
1. `titleCase` 関数は、各単語の最初の文字が大文字で、それ以外が小文字の文字列を返す必要があります。
1. `titleCase("I like to code")` は `"I Like To Code"` を返す必要があります。
1. `titleCase("javaScript is fun")` は `"Javascript Is Fun"` を返す必要があります。

# --solutions--

```js
function titleCase(str) {
  return str
    .split(' ')
    .map(word => word.charAt(0).toUpperCase() + word.substring(1).toLowerCase())
    .join(' ');
}
```
