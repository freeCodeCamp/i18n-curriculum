# --explanation--

`Adjusting to` 意味著變得適合或為新的情況或條件做出改變。 例如：

`The layout adjusts to different browsers.`－它會自動變更以符合瀏覽器設定。

`Screen sizes` 是指顯示器的實體尺寸，例如小型手機螢幕或大型桌上型顯示器。 例如：

`The app looks good on all screen sizes.`－它在任何型別的裝置上都能良好運作，無論顯示螢幕多大或多小。
