# --description--

Sehemu ndizo msingi wa React. Kila kitu katika React ni sehemu na hapa utajifunza jinsi ya kuunda moja.

Kuna njia mbili za kuunda sehemu ya React. Njia ya kwanza ni kutumia kitendakazi cha JavaScript. Kueleza sehemu kwa njia hii huunda *sehemu isiyo na hali ya kitendakazi*. Dhana ya hali katika programu itajadiliwa katika changamoto zijazo. Kwa sasa, fikiria sehemu isiyo na hali kama ile inayoweza kupokea data na kuonyesha, lakini haidhibiti wala kufuatilia mabadiliko ya data hiyo. (Tutajadili njia ya pili ya kuunda sehemu ya React katika changamoto inayofuata.)

Ili kuunda sehemu kwa kitendakazi, andika tu kitendakazi cha JavaScript kinachorejesha ama JSX au `null`. Jambo muhimu la kuzingatia ni kwamba React inahitaji jina la kitendakazi chako lianze na herufi kubwa. Hapa kuna mfano wa sehemu isiyo na hali ya kitendakazi inayoweka darasa la HTML katika JSX:

```jsx
const DemoComponent = function() {
  return (
    <div className='customClass' />
  );
};
```

Baada ya kubadilishwa, `<div>` itakuwa na darasa la CSS la `customClass`.

Kwa sababu sehemu ya JSX inawakilisha HTML, unaweza kuweka sehemu kadhaa pamoja kuunda ukurasa wa HTML mgumu zaidi. Hii ni moja ya faida kuu za usanifu wa sehemu unaotolewa na React. Inakuwezesha kuunda UI yako kutoka sehemu nyingi tofauti, zilizotengwa. Hii inafanya iwe rahisi kujenga na kudumisha interface za mtumiaji ngumu.

# --instructions--

Mhariri wa msimbo una kitendakazi kinachoitwa `MyComponent`. Kamilisha kitendakazi hiki ili kirejeshe kipengele kimoja cha `div` chenye mfuatano wa herufi fulani.

**Kumbuka:** Maandishi yanachukuliwa kuwa mtoto wa kipengele cha `div`, kwa hivyo hautaweza kutumia lebo inayojifunga yenyewe.

# --hints--

`MyComponent` inapaswa kurejesha JSX.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.length === 1;
  })()
);
```

`MyComponent` inapaswa kurejesha kipengele cha `div`.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.children().type() === 'div';
  })()
);
```

Kipengele cha `div` kinapaswa kuwa na mfuatano wa herufi.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.find('div').text() !== '';
  })()
);
```
