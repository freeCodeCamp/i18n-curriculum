# --explanation--

`To write something up` bedeutet, eine schriftliche Version von etwas zu erstellen, das gesprochen oder geplant wurde, normalerweise klar und organisiert. Zum Beispiel:

`After the interview, she wrote up her notes into a short article.` – Das bedeutet, dass sie ihre gesprochenen Notizen in ein schriftliches Format umgewandelt hat.
