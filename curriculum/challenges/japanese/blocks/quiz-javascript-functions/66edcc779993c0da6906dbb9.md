## --quiz--

### --question--

#### --text--

JavaScriptで関数を宣言する正しい方法はどれですか？

#### --distractors--

```js
func greet() {
  console.log("Hello there!");
}
```

---

```js
defineFunction greet() {
  console.log("Hello there!");
}
```

---

```js
def greet() {
  console.log("Hello there!");
}
```

#### --answer--

```js
function greet() {
  console.log("Hello there!");
}
```

### --question--

#### --text--

関数を呼び出す（または実行する）と何が起こりますか？

#### --distractors--

---

---

#### --answer--

関数内のコードが実行され、値が返されます。
