# --explanation--

Le risposte a domande con verbi ausiliari o modali non devono sempre usare lo stesso verbo. Per esempio:

- Domanda: `Should we leave early?`

- Risposta: `We could, but I'm not sure if it's necessary.` - La risposta usa `could` invece di ripetere `should`.
