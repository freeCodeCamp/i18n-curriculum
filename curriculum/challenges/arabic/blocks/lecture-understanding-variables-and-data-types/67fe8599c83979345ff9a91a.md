# --description--

لكل لغة برمجة طريقة لإخراج البيانات إلى الطرفية باستخدام طريقة، دالة، خاصية، أو كلمة مفتاحية مدمجة. في Python، يمكنك استخدام الدالة `print` لطباعة البيانات على الطرفية. دعنا نلقي نظرة أقرب على دالة `print` لتبدأ استخدامها بثقة.

أحد أول الأشياء التي تفعلها عند تعلم أي لغة برمجة هو كتابة برنامج بسيط `Hello world!`. يمكنك فعل ذلك بسهولة في Python باستخدام دالة `print` فقط.

للقيام بذلك، فقط ضع السلسلة النصية `Hello world!` بين الأقواس العادية المفتوحة والمغلقة التي تستخدمها لاستدعاء دالة `print`:

```python
print('Hello world!') # Hello world!
```

ستتعلم المزيد عن السلاسل النصية والدوال في Python في دروس لاحقة. الآن، اعتبر السلاسل النصية متتالية من الأحرف محاطة إما بعلامات اقتباس مفردة (`'`) أو مزدوجة (`"`).

في المثال `print('Hello world!')`، السلسلة النصية `'Hello world!'` هي **معلمة** تُمرر إلى دالة `print`. يمكنك أيضًا استخدام دالة `print` لعرض عدة قيم، أو معلمات، مرة واحدة بفصلها بفواصل. على سبيل المثال:

```python
print('My favorite colors are', 'blue', 'green', 'red')

# Output: My favorite colors are blue green red
```

تضيف Python تلقائيًا مسافة بين كل عنصر عندما تفصلهم بفواصل. هذا مفيد عندما تريد طباعة عدة معلومات معًا.
