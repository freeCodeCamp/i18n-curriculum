# --description--

양의 정수 ($a$,$b$,$c$) 세 개가 다음 조건을 만족하면 카르다노 삼중항이라고 합니다:

$$\sqrt[3]{a + b \sqrt{c}} + \sqrt[3]{a - b \sqrt{c}} = 1$$

예를 들어, (2,1,5)는 카르다노 삼중항입니다.

$a + b + c ≤ 1000$인 경우에 해당하는 카르다노 삼중항은 149개 존재합니다.

$a + b + c ≤ 110\\,000\\,000$인 경우에 몇 개의 카르다노 삼중항이 존재하는지 찾아보세요.
