# --description--

다항식 $n^4 + 4n^3 + 2n^2 + 5n$가 모든 정수 $n$에 대해 6의 배수임을 증명할 수 있습니다. 또한 6이 이 성질을 만족하는 가장 큰 정수임도 증명할 수 있습니다.

$M(a, b, c)$를 모든 정수 $n$에 대해 $n^4 + an^3 + bn^2 + cn$이 $m$의 배수가 되도록 하는 최대 $m$으로 정의합니다. 예를 들어, $M(4, 2, 5) = 6$입니다.

또한 $S(N)$을 $0 < a, b, c ≤ N$인 모든 경우에 대해 $M(a, b, c)$의 합으로 정의합니다.

$S(10) = 1\,972$이고 $S(10\,000) = 2\,024\,258\,331\,114$임을 확인할 수 있습니다.

피보나치 수열 $F_k$를 다음과 같이 정의합니다:

- $F_0 = 0$, $F_1 = 1$ 그리고
- $F_k = F_{k - 1} + F_{k - 2}$ ($k ≥ 2$).

$2 ≤ k ≤ 1\,234\,567\,890\,123$ 범위에서 $\sum S(F_k)$의 마지막 9자리를 구하세요.
