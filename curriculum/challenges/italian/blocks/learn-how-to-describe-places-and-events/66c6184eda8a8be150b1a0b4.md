# --explanation--

`Have to` si usa per esprimere che qualcosa è necessario o richiesto. Indica che non c'è altra scelta se non fare qualcosa. Per esempio:

- `You have to save your changes before closing the program.` - Questo significa che è necessario salvare le modifiche prima di chiudere il programma.

Quando Tom dice, `I'll have to check again`, sta affermando che per lui è necessario controllare le impostazioni un'altra volta, suggerendo che non ne è sicuro.
