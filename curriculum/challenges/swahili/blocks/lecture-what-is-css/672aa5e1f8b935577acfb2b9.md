# --interactive--

CSS, ambayo ni kifupi cha Cascading Style Sheets, ni sehemu muhimu sana ya ukuzaji wa mtandao wa kisasa. Ni lugha ya mitindo inayotumika kuweka mitindo kwa HTML. Kwa maneno rahisi, kama HTML ni muundo wa ukurasa wa mtandao, CSS ndiyo inayoufanya uonekane mzuri.

Hapa kuna mfano wa kupamba navbar.

**KUMBUKA**: Usijali kuelewa msimbo wa CSS kwa sasa. Utajifunza jinsi yote haya yanavyofanya kazi katika mafundisho na warsha zijazo. Hii ni kwa ajili ya kukupa wazo la kile unachoweza kufanya kwa kidogo cha CSS.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<nav class="navbar">
    <ul class="nav-links">
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Services</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>
```

```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: 'Segoe UI', sans-serif;
  background-color: #f4f4f4;
}

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  background-color: #333;
  padding: 1rem 2rem;
  color: white;
}

.navbar .logo {
  font-size: 1.5rem;
  font-weight: bold;
}

.nav-links {
  list-style: none;
  display: flex;
  gap: 1.5rem;
}

.nav-links a {
  color: white;
  text-decoration: none;
  font-weight: 500;
  transition: color 0.3s ease;
}

.nav-links a:hover {
  color: #ff9800;
}
```

:::

Jukumu kuu la CSS ni kutenganisha maudhui ya ukurasa wa mtandao na muonekano wake wa kuona. Utengano huu unaruhusu wasanidi programu wa mtandao kuunda tovuti zinazobadilika na rahisi kudumishwa.

Kwa CSS, unaweza kudhibiti mpangilio, rangi, fonti, na muonekano wa jumla wa kurasa za mtandao bila kubadilisha muundo wa HTML.

Tuchukulie mfano rahisi. Ukifikiria tovuti kama nyumba, HTML itakuwa msingi na mfumo wa kazi, wakati CSS itakuwa rangi, wallpaper, na mapambo yanayofanya nyumba ionekane nzuri na ya kipekee.

CSS hufanya kazi kwa kuchagua vipengele vya HTML na kuweka mitindo kwao.

Hapa kuna mfano unaoonyesha kipengele cha `button` kisichopambwa na kile kilichopambwa. Ili kuingiliana na mfano huu, utahitaji kuwezesha mhariri wa kuingiliana.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<button class="regular-btn">Unstyled Button</button>
<button class="round-btn">Round Button</button>
```

```css

.round-btn {
  font-size: 1rem;
  font-family: 'Segoe UI', sans-serif;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease, transform 0.2s ease;
  background-color: #2ecc71;
  color: white;
  border-radius: 50px;
  padding: 0.6rem 1.6rem;
}

.round-btn:hover {
  background-color: #27ae60;
  transform: translateY(-2px);
}
```

Mitindo hii inaweza kujumuisha vigezo kama rangi, ukubwa wa fonti, na vingine vingi. Kwa kubadilisha vigezo hivi, unaweza kubadilisha kwa kiasi kikubwa jinsi ukurasa wa mtandao unavyoonekana bila kubadilisha maudhui yake.

Moja ya vipengele vyenye nguvu vya CSS ni uwezo wake wa kuunda muundo unaobadilika.

Hii inamaanisha kuwa kwa CSS, unaweza kufanya tovuti yako ionekane nzuri kwenye kifaa chochote, iwe ni kompyuta ya mezani, kibao, au simu ya mkononi.

CSS inakuwezesha kurekebisha mipangilio, ukubwa wa fonti, na vipengele vingine vya kuona kulingana na ukubwa wa skrini ya kifaa kinachotazama tovuti.

Jaribu kuwezesha mhariri wa kuingiliana na rekebisha ukubwa wa dirisha la onyesho ili kuona jinsi mpangilio unavyobadilika kulingana na ukubwa wa skrini.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<header class="header">
    <h1>Welcome to My Responsive Site</h1>
    <p>This layout adapts to your screen size!</p>
</header>

<main class="content">
    <section class="box">Box 1</section>
    <section class="box">Box 2</section>
    <section class="box">Box 3</section>
</main>
```

```css
body {
  margin: 0;
  font-family: 'Segoe UI', sans-serif;
  background-color: #f9f9f9;
  color: #333;
}

.header {
  background-color: #4CAF50;
  color: white;
  padding: 2rem;
  text-align: center;
}

.content {
  display: flex;
  gap: 1rem;
  padding: 1rem;
  justify-content: center;
}

.box {
  flex: 1;
  min-width: 200px;
  background-color: #ddd;
  padding: 2rem;
  text-align: center;
  font-size: 1.2rem;
  border-radius: 8px;
  transition: background-color 0.3s;
}

.box:hover {
  background-color: #ccc;
}

/* Responsive: for tablets and below (≤ 768px) */
@media (max-width: 768px) {
  .content {
    flex-direction: column;
    align-items: center;
  }

  .box {
    width: 90%;
    font-size: 1.1rem;
  }
}

/* Responsive: for phones (≤ 480px) */
@media (max-width: 480px) {
  .header {
    padding: 1.5rem 1rem;
  }

  .header h1 {
    font-size: 1.5rem;
  }

  .header p {
    font-size: 1rem;
  }

  .box {
    font-size: 1rem;
    padding: 1.5rem;
  }
}
```

Kipengele kingine muhimu cha CSS ni asili yake ya kuendelea kuambatana, ambayo ndiyo chanzo cha neno "cascading" katika jina lake.

Hii inamaanisha kuwa mitindo inaweza kurithiwa na kubadilishwa, kuruhusu muundo wa mitindo wenye ngazi mbalimbali. Utajifunza zaidi jinsi hii inavyofanya kazi katika mafundisho yajayo.

CSS pia inaunga mkono matumizi ya faili za mitindo za nje.

Hii inamaanisha unaweza kuweka sheria zako zote za mitindo katika faili tofauti, ambayo inaweza kuunganishwa na kurasa nyingi za HTML.

Kipengele hiki kinaboresha sana urahisi wa kudumisha tovuti, hasa zile kubwa. Badala ya kubadilisha mitindo kwenye kila ukurasa mmoja mmoja, unaweza kufanya marekebisho katika faili moja la CSS ambayo yataathiri kurasa zote zilizounganishwa.

Katika ulimwengu wa ukuzaji wa mtandao, CSS ina jukumu muhimu katika kuunda tovuti zenye muonekano mzuri, zinazojibadilisha kulingana na kifaa, na rafiki kwa mtumizi.

Inawawezesha wasanidi programu kubadilisha hati rahisi za HTML kuwa uzoefu wa mtandao unaovutia ambao unavutia watumizi na kuboresha mwingiliano wao na maudhui ya mtandao.
