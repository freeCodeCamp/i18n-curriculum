# --interactive--

Працюючи з фоновими зображеннями в CSS, у вас є кілька властивостей, які дозволяють контролювати, як ці зображення відображаються.

Основні властивості, на яких ми зосередимося, це `background-size`, `background-repeat`, `background-position` і `background-attachment`.

Спершу подивімося на властивість `background-image`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

Вищенаведений CSS встановлює фонове зображення кота для елемента `body`.

Якщо ви хочете задати розмір фонового зображення, ви можете використати властивість `background-size`.

Ви можете використати `contain`, щоб масштабувати зображення максимально великим без обрізання або розтягування.

Ось приклад із `background-size: contain`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    min-height: 100px;
  }
</style>
```

:::

Ми встановлюємо `min-height` у `100px`, щоб фонове зображення було видимим, а макет зберігав базову висоту, гарантуючи, що навіть при мінімальному вмісті дизайн виглядає збалансованим і привабливим.

Якщо ми змінемо властивість `background-size` на значення `cover`, тоді фонове зображення масштабуватиметься, щоб покрити весь елемент `body`, зберігаючи пропорції.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    min-height: 100px;
  }
</style>
```

:::

У попередніх прикладах ви, мабуть, помітили, що фонове зображення безперервно повторюється.

За замовчуванням фонові зображення повторюються і по горизонталі, і по вертикалі, щоб заповнити весь елемент. Однак ви можете контролювати цю поведінку.

Ви можете використати властивість `background-repeat` зі значенням `no-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

Якщо `background-size` встановлено в `contain`, а `background-repeat` у `no-repeat`, зображення більше не повторюватиметься на екрані.

Якщо ви хочете, щоб фонове зображення повторювалося по горизонталі, використайте значення `repeat-x` для властивості `background-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-x;
    min-height: 100px;
  }
</style>
```

:::

Якщо ви хочете, щоб фонове зображення повторювалося по вертикалі, використайте значення `repeat-y` для властивості `background-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-y;
    min-height: 100px;
  }
</style>
```

:::

Щоб позиціонувати фонове зображення на екрані, ви можете використати властивість `background-position`.

Властивість `background-position` дозволяє задати, де саме в елементі з’явиться фонове зображення. Ви можете використати ключові слова, такі як `top`, `bottom`, `left`, `right` і `center`, або конкретні значення в пікселях чи відсотках.

Ось приклад із використанням `center` і `top` для `background-position`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

Цей CSS-код позиціонує фонове зображення по центру елемента по горизонталі і зверху по вертикалі.

Нарешті, `background-attachment` визначає, чи фонове зображення прокручується разом із вмістом, чи залишається зафіксованим при прокручуванні сторінки.

Основні значення — `scroll` (за замовчуванням), коли фонове зображення прокручується разом із вмістом, і `fixed`, коли фонове зображення залишається на одному місці на екрані.

Ось приклад із використанням значення `fixed` для властивості `background-attachment`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-position: center top;
    background-attachment: fixed;
  }
</style>
```

:::

Цей CSS-код встановлює фонове зображення так, щоб воно залишалося зафіксованим навіть при прокручуванні сторінки.

Якщо ви хочете об’єднати кілька властивостей в один рядок, ви можете зробити це за допомогою скороченої властивості `background`.

Ось приклад:

:::interactive_editor

```html
<style>
  body {
    background: center top fixed
      url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

Вищенаведений приклад об’єднує `background-image`, `background-position` і `background-attachment` в один рядок.

Опанувавши ці властивості, ви отримаєте чудовий контроль над тим, як фонові зображення відображаються у ваших вебдизайнах, що дозволить створювати більш привабливі та функціональні макети.
