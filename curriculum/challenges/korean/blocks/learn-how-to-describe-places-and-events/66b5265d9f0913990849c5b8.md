# --explanation--

`To touch on`는 주제를 간단히 언급하거나 다루는 것을 의미할 수 있습니다. 보통 자세히 설명하지 않고 주제를 다룰 때 사용합니다. 예를 들어:

- `The speaker touched on the new software update during the meeting.` - 화자는 소프트웨어 업데이트를 간단히 언급했지만 자세히 설명하지는 않았습니다.
