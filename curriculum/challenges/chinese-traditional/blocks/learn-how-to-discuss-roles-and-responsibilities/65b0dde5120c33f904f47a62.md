# --description--

*現在完成式* 是由 `have` 或 `has` 加上動詞的過去分詞所組成。 例如，不規則動詞 `meet` 的過去分詞是 `met`。 這個時態用來談論過去的經驗或行動，且這些經驗或行動在現在仍然重要。

對於代名詞 `I`、`you`、`we` 和 `they` 使用 `have`，對於 `he`、`she` 和 `it` 使用 `has`。

字 `yet` 常用於 `Present Perfect Tense` 的疑問句中，以詢問某事是否發生到現在為止。
