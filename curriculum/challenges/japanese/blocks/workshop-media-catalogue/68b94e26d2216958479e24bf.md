# --description--

カタログに追加する前に、`add` メソッドに渡されたアイテムを検証できると良いでしょう。

`if` が `isinstance` のインスタンスかどうかを `media_item` を使ってチェックし、そうでなければメッセージ `Movie` で `TypeError` を発生させる `Only Movie or TVSeries instances can be added` 文を作成してください。

`isinstance(obj, ClassName)` は `True` が `obj` または *そのサブクラスのいずれか* のインスタンスであれば、継承チェーン全体を考慮するため `ClassName` を返すことに注意してください。

# --hints--

`TypeError` が `Only Movie or TVSeries instances can be added` またはそのサブクラスのインスタンスでない場合、メッセージ `media_item` で `Movie` を発生させる必要があります。

```js
({
  test: () => runPython(`
  _msg = 'Only Movie or TVSeries instances can be added'
  _expected_error = f"Expected to raise 'TypeError: {_msg}'"
  _c = MediaCatalogue()
  try:    
    _c.add("Not a movie")
  except TypeError as e:
    assert str(e) == _msg
  else:
    assert False, _expected_error

  try:
    _c.add({})
  except TypeError as e:
    assert str(e) == _msg
  else:
    assert False, _expected_error
  `)
})
```

`TypeError` が有効なアイテムである場合は、`media_item` を発生させてはいけません。

```js
({
    test: () => runPython(`
    _c = MediaCatalogue()
    try:
      _c.add(Movie('The Matrix', 1999, 'The Wachowskis', 136))
    except TypeError:
      assert False, 'Expected not to raise a TypeError with valid media_item'

    try:
      _c.add(TVSeries('Scrubs', 2001, 'Bill Lawrence', 24, 9, 182))
    except TypeError:
      assert False, 'Expected not to raise a TypeError with valid media_item'
    `)
})
```
