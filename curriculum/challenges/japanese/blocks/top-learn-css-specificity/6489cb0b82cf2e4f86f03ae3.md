# --description--

特異性がどのように機能するかを視覚的に理解するために、いくつかの簡単な例を見てみましょう。以下のHTMLとCSSコードを考えてください。

```html
<!-- index.html -->

<div class="main">
  <div class="list subsection"></div>
</div>
```

```css
/* rule 1 */
.subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

上の例では、両方のルールがクラスセレクターのみを使っていますが、ルール2の方がより多くのクラスセレクターを使っているため、より特異性が高くなっています。
