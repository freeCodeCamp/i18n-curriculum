# --questions--

## --text--

どの文が `since` を `Present Perfect Continuous` 時制で正しく使っていますか？

## --answers--

`I have worked on the project since last week.`

### --feedback--

これは `Present Perfect Simple` 時制です。動詞の形は、進行中の動作を示すために `Present Perfect Continuous` 時制（`have been working`）である必要があります。

---

`I was testing the program since yesterday.`

### --feedback--

これは `Past Simple` 時制です。`Present Perfect Continuous` 時制では `have been` を使い、`was` は使いません。

---

`I have been testing them since the latest software update.`

---

`I will have been working on the project since next month.`

### --feedback--

`Future Perfect` 時制はここでは適切ではありません。`since` は `Present Perfect Continuous` 時制で使われます。

## --video-solution--

3

# --explanation--

`Since` は、まだ続いている動作の開始点を示すために `Present Perfect Continuous` 時制で使われます。
