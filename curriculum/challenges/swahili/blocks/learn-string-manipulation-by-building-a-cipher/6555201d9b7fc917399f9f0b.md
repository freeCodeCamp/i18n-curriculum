# --description--

Njia ya `.isalpha()` inarudisha `True` ikiwa herufi zote za mfuatano wa herufi ambayo inaitwa ni herufi. Kwa mfano, msimbo ulio hapa chini unarudisha `True`:

```py
'freeCodeCamp'.isalpha()
# True
```

Futa sharti la `char == ' '` na libadilishe kwa kuita njia ya `.isalpha()` kwenye `char`.
