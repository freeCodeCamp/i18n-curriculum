# --description--

`Strange` 意味着不寻常或意料之外。它就像查找与你通常看到的不同的东西。例如：`A strange sound in a car might mean a problem.`

`error` 是一种错误，尤其是会导致某些东西无法正常工作的错误。例如：`An error in a code means there is a mistake that needs fixing.`
