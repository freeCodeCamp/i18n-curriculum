# --description--

Обчисліть **n**<sup>тий</sup> член <em>послідовності</em>, тобто суму перших **n** членів відповідної <em>послідовності</em>. Неформально це значення, або його межа при прямує до нескінченності, також називається *сумою послідовності*, звідси й назва цього завдання. Для цього завдання використовуйте: $S_n = \displaystyle\sum_{k=1}^n \frac{1}{k^2}$.

# --instructions--

Напишіть функцію, яка приймає $a$ та $b$ як параметри і повертає суму членів послідовності від $a^{го}$ до $b^{го}$.

```js
function sum(a, b) {
  function fn(x) {
    return 1 / (x * x);
  }
  var s = 0;
  for (; a <= b; a++) s += fn(a);
  return s;
}
```
