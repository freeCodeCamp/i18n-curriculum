# --description--

HTML 註解對瀏覽器不可見；它們允許我們對你的程式碼進行註解，讓其他開發者或你未來的自己能閱讀並獲得關於程式碼中某些可能不清楚的部分的背景關係。

撰寫 HTML 註解很簡單：你只需將註解用 `<!--` 和 `-->` 頁籤包起來。例如：

```html
<h1> View the html to see the hidden comments </h1>

<!-- I am a html comment -->

<p>Some paragraph text</p>

<!-- I am another html comment -->
```

# --assignment--

為了練習在 HTML 中操作文字，請創建一個純粹的部落格文章頁面，該頁面使用不同的標題、使用段落，並且在段落中有些文字是加粗和斜體的。你可以使用 [Lorem Ipsum](https://loremipsum.io) 來產生假文字，取代真實文字，當你建置你的網站時。
