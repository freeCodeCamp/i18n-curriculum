# --interactive--

在之前的課程中，你學會了如何開始使用線性和放射狀漸層。在本課程中，我們將更深入探討如何使用 CSS 屬性來創建更具視覺吸引力和複雜的設計。

CSS 中的漸層允許你在兩個或多個指定顏色之間創建平滑的過渡。它們常用於為網頁設計添加深度、質感和視覺趣味，而無需使用影像檔案。 

CSS 支援兩種主要的漸層類型：線性漸層和放射狀漸層。

線性漸層會沿著直線在顏色之間產生漸進的混合。你可以控制這條線的方向和使用的顏色。以下是一個線性漸層的基本範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

這會創建一個漸層，從左邊的紅色過渡到右邊的藍色。`to right` 參數指定漸層的方向。你可以使用其他方向，例如 `to left`、`to top`、`to bottom`，甚至指定角度如 `45deg`。

你也可以在線性漸層中使用多個顏色停點：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="multi-color-gradient"></div>
```

```css
.multi-color-gradient {
  background: linear-gradient(to right, red, yellow, blue);
  height: 40vh;
}
```

:::

這會創建一個漸層，從紅色過渡到黃色再到藍色，均勻分佈在元素上。

另一方面，放射狀漸層會從中心點輻射出圓形或橢圓形的漸層。以下是一個範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::

這會創建一個從中心紅色開始並過渡到邊緣藍色的圓形漸層。你也可以指定放射狀漸層的形狀和大小：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="custom-radial-gradient"></div>
```

```css
.custom-radial-gradient {
  background: radial-gradient(ellipse at top left, red, blue);
  height: 40vh;
}
```

:::

這會從元素的左上角創建一個橢圓形漸層。線性和放射狀漸層都允許你在精確位置指定顏色停點。 例如：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="precise-gradient"></div>
```

```css
.precise-gradient {
  background: linear-gradient(to right, red 20%, yellow 40%, blue 80%);
  height: 40vh;
}
```

:::

這會創建一個漸層，其中紅色佔據前 `20%`，然後在 `40%` 過渡到黃色，最後在 `80%` 過渡到藍色。

漸層可以用在你在 CSS 中使用影像的任何地方。這使它們在建立背景、按鈕，甚至結合其他 CSS 屬性時用於文字效果時非常多功能。

請記住，雖然漸層可以提升你網站的視覺吸引力，但確保足夠的對比度以利存取性非常重要，尤其是在漸層置於文字背後時。

## --text--

CSS 中線性漸層和放射狀漸層的差異是什麼？

## --answers--

線性漸層只使用兩種顏色，而放射狀漸層使用多種顏色。

---
