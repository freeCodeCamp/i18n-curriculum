# --description--

Normalmente, o que diferencia um adjetivo de um advérbio é o sufixo `-ly` no final, adicionado a um adjetivo para criar um advérbio.Exemplos: `extreme` (adjetivo) x `extremely` (advérbio) / `quick` (adjetivo) x `quickly` (advérbio).

Além disso, saiba que é possível usar um advérbio como `extremely` para intensificar um adjetivo ou até outro advérbio.Se você diz que algo acontece `quickly`, é uma ação rápida.Se for `extremely quickly`, a ação é super rápida.
