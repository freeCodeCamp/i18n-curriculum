# --description--

Il verbo `vivir` significa "vivere".

Quando si parla di un'altra persona ("lui"/"lei"), il verbo si coniuga come `vive`. Di solito è seguito dalla preposizione `en` per indicare dove vive la persona, con questa struttura:

**`Vive en` + luogo**

Julieta sta rispondendo alla domanda di Mateo, dicendogli dove vive Valeria.
