# --interactive--

`button` 元素用於在它被作用時執行特定動作。以下是一個 `button` 元素的範例，其按鈕文字為 `Start Game`。

```html
<button>Start Game</button>
```

使用 `button` 元素的其他範例包括提交表單、顯示模態視窗，或切換側邊選單的開啟與關閉。`button` 元素有一個 `type` 屬性，用來控制按鈕被作用時的行為。`type` 屬性的第一個可能值是 `button` 型別。以下是使用 `button` 元素搭配 `button` 型別及文字為 `Show Alert` 的範例：

```html
<button type="button">Show Alert</button>
```

預設情況下，按鈕在作用時不會執行任何操作。然而，你可以為你的按鈕添加一些 JavaScript 程式碼，使按鈕具有互動性，例如在此情況下顯示警示。 

點擊 `Show Alert` 按鈕即可看到警示在螢幕上彈出。若要與範例互動，你需要啟用互動式編輯器。


**注意**：這個互動範例使用的是 JavaScript，但你不需要擔心理解 JavaScript 程式碼。你會在未來的模組中學習 JavaScript。 

:::interactive_editor

```html
<button type="button">Show Alert</button>
<script src="index.js"></script>
```

```js
const btn = document.querySelector("button");
btn.addEventListener("click", () => alert("You clicked on the alert button"));
```

:::

`type` 屬性的另一個可能值是 `submit` 值。以下是使用具有 `submit` 型別的 `button` 元素的範例。

:::interactive_editor

```html
<form action="">
  <label for="email">Email address:</label>
  <input type="email" id="email" name="email" />
  <button type="submit">Submit form</button>
</form>
```

:::

在這個 `form` 元素內，有一個用於使用者電子郵件位址的 `label` 和 `input` 元素。當使用者點擊送出按鈕時，他們的資料將會被傳送到伺服器並進行處理程序。`type` 屬性的第三個可能值是 `reset` 值。以下是一個包含重設和送出按鈕的 `form` 元素範例。

啟用互動式編輯器，並在預覽視窗中透過提供假電子郵件位址與電子郵件輸入欄互動。然後點擊重設按鈕，查看你的電子郵件從欄位中消失。 

:::interactive_editor

```html
<form action="">
  <label for="email">Email address:</label>
  <input type="email" id="email" name="email" />
  <button type="reset">Reset form</button>
  <button type="submit">Submit form</button>
</form>
```

:::

在這個修改過的範例中，使用了 `label` 和 `input` 元素來收集使用者的電子郵件位址。當使用者點擊重設按鈕時，將會清除他們所有的輸入資料。重要的是要注意，重設按鈕通常不是最佳選擇，因為它們可能導致使用者不小心重設資料。此外，表單中多個按鈕可能會使使用者介面變得雜亂。

在 HTML 中創建按鈕的另一種方式是使用 `input` 元素。`input` 元素也有一個 `type` 屬性，其可能的值包括 `submit`、`reset` 和 `button`。以下是使用 `input` 元素並將 `type` 設定為 `button` 的範例：

:::interactive_editor

```html
<input class="start-btn" type="button" value="Start Game" />
<script src="index.js"></script>
```

```js
document.addEventListener("DOMContentLoaded", () => {
  const btn = document.querySelector(".start-btn");
  btn.addEventListener("click", () => {
    const paraEl = document.createElement("p");
    const bodyEl = document.querySelector("body");

    bodyEl.appendChild(paraEl);
    paraEl.textContent = "The game has started!!!";
  });
});
```

:::

`value` 屬性用於顯示按鈕文字。那麼，使用 `input` 和 `button` 元素有什麼差別呢？`input` 元素是空元素，這表示它們不能有子節點，例如文字，且只能有起始標籤。另一方面，`button` 元素提供更多彈性，因為你可以在其中嵌套文字、影像和圖示。
