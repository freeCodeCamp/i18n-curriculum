# --interactive--

CSS，全名為 Cascading Style Sheets，是現代網頁開發中重要的組件。它是一種用於套用 HTML 樣式的標記語言。簡單來說，如果 HTML 是網頁的結構，CSS 就是讓它看起來漂亮的東西。

這裡是一個為導覽列設計樣式的範例。 

**注意**：暫時不用擔心嘗試理解 CSS 程式碼。你會在未來的課程和工作坊中學習這些運作方式。這只是讓你了解使用一點點 CSS 可以做什麼。 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<nav class="navbar">
    <ul class="nav-links">
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Services</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>
```

```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: 'Segoe UI', sans-serif;
  background-color: #f4f4f4;
}

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  background-color: #333;
  padding: 1rem 2rem;
  color: white;
}

.navbar .logo {
  font-size: 1.5rem;
  font-weight: bold;
}

.nav-links {
  list-style: none;
  display: flex;
  gap: 1.5rem;
}

.nav-links a {
  color: white;
  text-decoration: none;
  font-weight: 500;
  transition: color 0.3s ease;
}

.nav-links a:hover {
  color: #ff9800;
}
```

:::

CSS 的主要角色是將網頁內容與其視覺呈現分離。這種分離讓網頁開發者能夠創建更具彈性且易於維護的網站。

使用 CSS，你可以控制網頁的版面配置、顏色、字型和整體視覺外觀，而不改變 HTML 結構。

讓我們來考慮一個簡單的類比。如果你將網站視為一棟房子，`HTML` 就是地基和框架，而 `CSS` 則是讓房子在視覺上吸引人且獨特的油漆、桌布和裝飾。

CSS 是透過選取 HTML 元素並套用樣式來運作的。

這裡有一個範例，顯示未加樣式的 `button` 元素和已加樣式的元素。若要與此範例互動，你需要啟用互動式編輯器。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<button class="regular-btn">Unstyled Button</button>
<button class="round-btn">Round Button</button>
```

```css

.round-btn {
  font-size: 1rem;
  font-family: 'Segoe UI', sans-serif;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease, transform 0.2s ease;
  background-color: #2ecc71;
  color: white;
  border-radius: 50px;
  padding: 0.6rem 1.6rem;
}

.round-btn:hover {
  background-color: #27ae60;
  transform: translateY(-2px);
}
```

:::

這些樣式可以包含像是顏色、字型大小以及更多的屬性。透過更改這些屬性，你可以在不改變內容的情況下，大幅改變網頁的外觀。

CSS 最強大的特性之一是它能夠創建響應式設計。

這表示使用 CSS，你可以讓你的网站在任何裝置上看起來都很棒，無論是桌上型電腦、平板電腦還是智慧型手機。

CSS 允許你根據瀏覽網站的裝置螢幕大小調整版面配置、字型大小和其他視覺元素。

嘗試啟用互動式編輯器並調整預覽視窗的大小，以查看版面如何適應螢幕大小。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<header class="header">
    <h1>Welcome to My Responsive Site</h1>
    <p>This layout adapts to your screen size!</p>
</header>

<main class="content">
    <section class="box">Box 1</section>
    <section class="box">Box 2</section>
    <section class="box">Box 3</section>
</main>
```

```css
body {
  margin: 0;
  font-family: 'Segoe UI', sans-serif;
  background-color: #f9f9f9;
  color: #333;
}

.header {
  background-color: #4CAF50;
  color: white;
  padding: 2rem;
  text-align: center;
}

.content {
  display: flex;
  gap: 1rem;
  padding: 1rem;
  justify-content: center;
}

.box {
  flex: 1;
  min-width: 200px;
  background-color: #ddd;
  padding: 2rem;
  text-align: center;
  font-size: 1.2rem;
  border-radius: 8px;
  transition: background-color 0.3s;
}

.box:hover {
  background-color: #ccc;
}

/* Responsive: for tablets and below (≤ 768px) */
@media (max-width: 768px) {
  .content {
    flex-direction: column;
    align-items: center;
  }

  .box {
    width: 90%;
    font-size: 1.1rem;
  }
}

/* Responsive: for phones (≤ 480px) */
@media (max-width: 480px) {
  .header {
    padding: 1.5rem 1rem;
  }

  .header h1 {
    font-size: 1.5rem;
  }

  .header p {
    font-size: 1rem;
  }

  .box {
    font-size: 1rem;
    padding: 1.5rem;
  }
}
```

:::

CSS 的另一個重要特性是其層疊性，這也是其名稱中「層疊」一詞的由來。

這表示樣式可以被繼承和覆寫，允許樣式呈現階層式的結構。你將在未來的課程中學習更多這方面的運作方式。

CSS 也支援（某功能）使用外部樣式表。

這表示你可以將所有樣式規則保存在獨立的檔案中，然後將該檔案連結到多個 HTML 頁面。

這個特性大幅提升網站的維護性，尤其是較大型的網站。你不必在每個獨立頁面上更改樣式，而是可以在一個 `CSS` 檔案中進行更改，這些更改將影響所有連結的頁面。

在網頁開發的世界中，CSS 在建立視覺吸引人、響應式且使用者友善的網站中扮演著關鍵角色。

它允許開發者將簡單的 HTML `documents` 轉換成引人入勝的網頁體驗，吸引用戶的注意力並提升他們與網頁內容的互動。
