# --explanation--

`Could` na `might` ni vitenzi vya modal vinavyotumika kuelezea uwezekano.

`Could` inaonyesha kitu kinawezekana lakini si hakika. Kwa mfano:

`If we don't fix this bug, it could cause a system crash.` - Kuna uwezekano mfumo utaacha kufanya kazi.

`Might` inaonyesha kitu kinawezekana lakini si hakika kama `could`. Kwa mfano:

`We might need to rewrite some parts of the code.` - Inawezekana mabadiliko yatakuwa muhimu.
