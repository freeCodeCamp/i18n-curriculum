# --description--

Сполучник `so` використовується для введення результату або наслідку, пов’язаного з тим, що було сказано. Він часто показує причинно-наслідковий зв’язок між двома частинами речення.

Наприклад, `It was raining, so we canceled the picnic.` Це показує, що пікнік скасували через дощ.

# --questions--

## --text--

Яке речення правильно використовує `so`?

## --answers--

`I was hungry so I am eating.`

### --feedback--

Це речення майже правильно використовує `so`, але час у другій частині має відповідати першій. Правильно було б сказати: `I was hungry, so I ate.`

---

`She is late to work, so she misses the meeting.`

### --feedback--

Використання `so` натякає на наслідок, але часи не співпадають. Має бути: `She is late to work, so she will miss the meeting.`

---

`It started raining, so we took shelter.`

---

`He likes to read books, so he buys a lamp.`

### --feedback--

Це речення неправильно використовує `so`, бо купівля лампи не є прямим наслідком того, що подобається читати книги. Краще було б сказати: `He likes to read at night, so he bought a lamp.`
