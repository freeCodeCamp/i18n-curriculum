# --description--

`Robust` significa fuerte o poderoso. Por ejemplo:

`A robust system can handle many users.`

`Enough` significa tanto como sea necesario. Por ejemplo:

`Do we have enough chairs?`

Juntos, `robust enough` significa ser fuerte o efectivo al grado necesario. Por ejemplo, esta oración pregunta si la contraseña es suficientemente fuerte para detener a los hackers:

`Is the password robust enough to prevent hacking?`
