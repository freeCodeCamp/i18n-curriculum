# --explanation--

`Make it a habit` تعني القيام بشيء بانتظام حتى يصبح تلقائيًا. على سبيل المثال:

- `Try to make it a habit to write clean code.` - يجب أن تكتب شفرة سهلة القراءة بانتظام.

- `She made it a habit to learn one new programming tip every week.` - تتعلم بانتظام كجزء من روتينها.

يريد مارك اختبار شفرته في كل مرة حتى لا ينسى القيام بذلك في المستقبل.
