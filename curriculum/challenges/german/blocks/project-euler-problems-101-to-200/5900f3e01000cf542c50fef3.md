# --description--

Eine Reihe von fünf schwarzen quadratischen Fliesen soll durch eine Anzahl von farbigen rechteckigen Fliesen ersetzt werden, die aus Rot (Länge zwei), Grün (Länge drei) oder Blau (Länge vier) ausgewählt werden.

Wenn rote Fliesen gewählt werden, gibt es genau sieben Möglichkeiten, dies zu tun.

<img alt="Mögliche Arten, rote rechteckige Fliesen auf einer Reihe mit einer Länge von fünf Einheiten zu platzieren" src="https://cdn.freecodecamp.org/curriculum/project-euler/red-green-or-blue-tiles-1.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Wenn grüne Fliesen gewählt werden, gibt es drei Möglichkeiten.

<img alt="Mögliche Arten, grüne rechteckige Fliesen auf einer Reihe mit einer Länge von fünf Einheiten zu platzieren" src="https://cdn.freecodecamp.org/curriculum/project-euler/red-green-or-blue-tiles-2.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Und wenn blaue Fliesen gewählt werden, gibt es zwei Möglichkeiten.

<img alt="Mögliche Arten, blaue rechteckige Fliesen auf einer Reihe mit einer Länge von fünf Einheiten zu platzieren" src="https://cdn.freecodecamp.org/curriculum/project-euler/red-green-or-blue-tiles-3.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Unter der Annahme, dass die Farben nicht gemischt werden dürfen, gibt es 7 + 3 + 2 = 12 Möglichkeiten, die schwarzen Fliesen in einer Reihe mit einer Länge von fünf Einheiten zu ersetzen. Wie viele verschiedene Möglichkeiten gibt es, die schwarzen Fliesen in einer Reihe mit einer Länge von fünfzig Einheiten zu ersetzen, wenn die Farben nicht gemischt werden dürfen und mindestens eine farbige Fliese verwendet werden muss?

**Hinweis:** Dies steht im Zusammenhang mit Problem 117.
