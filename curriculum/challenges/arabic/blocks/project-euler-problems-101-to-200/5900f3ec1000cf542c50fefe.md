# --description--

الجذر التربيعي لـ $n$، $rad(n)$، هو حاصل ضرب العوامل الأولية المميزة لـ $n$. على سبيل المثال، $504 = 2^3 × 3^2 × 7$، إذًا $rad(504) = 2 × 3 × 7 = 42$.

نُعرّف الثلاثي من الأعداد الصحيحة الموجبة (a, b, c) بأنه abc-hit إذا:

1. $GCD(a, b) = GCD(a, c) = GCD(b, c) = 1$
2. $a &lt; b$
3. $a + b = c$
4. $rad(abc) &lt; c$

على سبيل المثال، (5, 27, 32) هو abc-hit، لأن:

1. $GCD(5, 27) = GCD(5, 32) = GCD(27, 32) = 1$
2. $5 &lt; 27$
3. $5 + 27 = 32$
4. $rad(4320) = 30 &lt; 32$

يتبين أن abc-hits نادرة جدًا ولا يوجد سوى واحد وثلاثون abc-hit لـ $c &lt; 1000$، مع $\sum{c} = 12523$.

احسب $\sum{c}$ لـ $c &lt; 120000$.
