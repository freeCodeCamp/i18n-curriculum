# --description--

关系数据库有广泛的应用，包括网页开发、库存管理系统、医疗系统、电子商务应用等。

一些最常见的关系数据库有 MySQL、PostgreSQL、SQLite 和 Microsoft SQL Server。让我们了解一下它们各自的基本情况。

MySQL 是一个开源的关系数据库管理系统。它因易于使用、可靠且高效地用于创建实际应用而在网页开发中非常受欢迎。它还有一个活跃的开发者社区，支持并维护它。

PostgreSQL 官方描述为“世界上最高级的开源关系数据库。”它是一个免费的、开源的对象关系数据库系统，以其可靠性、数据完整性和可扩展性著称。例如，你可以定义你自己的数据类型、自定义函数，并且可以用不同的编程语言编写代码，而无需重新编译你的数据库。

SQLite 是一个免费、开源且轻量级的基于文件的 SQL 数据库引擎。它是世界上使用最广泛的数据库引擎之一。SQLite 是自包含且无服务器的。它是一个零配置数据库，这意味着它不需要初始设置过程，非常适合快速入门。

这三个数据库是免费的开源数据库。还有其他流行的专有选项，包括 Microsoft SQL Server 和 Oracle database。

Microsoft SQL `server` 是微软开发并在各种应用中使用的流行关系数据库管理系统。

Oracle 数据库是一种可扩展的关系数据库管理系统，常用于需要高性能的企业应用。

太好了。既然你已经了解了可用的不同选项，现在让我们看看你如何安装 PostgreSQL。

首先，你需要访问官方网站：<a href="https://www.postgresql.org/" target="_blank">https://www.postgresql.org/</a>。

一旦你到达那里，点击 `"Download"` 按钮。

这将带你进入一个页面，在那里你可以选择你的操作系统。如果你点击其中一个，你将被带到该操作系统的说明页面。

我们将介绍 Windows 操作系统的安装进程，但该进程在各操作系统中类似。

在此页面，你可以通过点击 `"Download the installer"` 链接来下载安装程序。

这将带你进入一个页面，在那里你可以选择想要安装的 PostgreSQL 版本。如果你点击相应的图标，下载应该会开始。

然后，在你的 Downloads 文件夹或你默认保存下载内容的文件夹中，你应该能看到安装程序。

如果你双击它，你将启动安装进程。点击下一步。

选择 PostgreSQL 将被安装的目录。你会看到一个默认目录，但你可以自定义它。

然后，选择你想要安装的组件。有四个可能的组件：

- PostgreSQL 服务器，即数据库引擎本身。
- pgAdmin 4，是一个用户友好的数据库管理工具。
- Stack Builder，是一个用于下载和安装额外的 PostgreSQL 相关软件和扩展（程序、功能）的工具
- 以及命令行工具，用于直接从你的终端与 PostgreSQL 服务器交互。这包括 psql，一个用于执行 SQL 查询的交互式终端。

当你准备好时，点击 Next。

然后，选择一个目录来保存你的数据。你会看到一个默认路径，但你可以自定义它。准备好后，点击 Next。

现在你需要为数据库超级用户输入密码，该用户拥有对所有数据库和对象的无限制访问权限。输入你的密码，重新输入，然后点击下一步。

然后，选择服务器应监听的端口号。默认情况下，是 5432。然后，点击 下一步。

你需要为新的数据库集群选择一个区域设置。这对于数据库解释区域设置敏感的数据类型非常重要。然后，点击 下一步。

最后，你会看到一个预安装摘要。花点时间查看，然后点击 Next。

你需要确认是否要开始安装进程。如果你点击 Next，进程将开始。整个过程应该只需几分钟。

当进程完成后，根据你安装的组件，你可能需要选择是否要启动 Stack Builder 以下载和安装额外的工具、驱动程序和应用。

如果你决定启动 Stack Builder，你将看到一个可以开始安装的屏幕。备选地，你可以点击 Cancel 以后再进行此操作。

恭喜！现在你已经在你的计算机上安装了 PostgreSQL。对于其他操作系统，进程应该类似。之后，你可以开始使用数据库。

进入你的应用并查找 pgAdmin，这是一个开源的 PostgreSQL 管理和开发平台，如果你在组件中选择了它，应该已经安装。点击 pgAdmin 4 以打开它。

当应用启动时，你会看到一个初始装载屏幕。然后，你应该看到一个仪表盘，在那里你可以新建服务器、查看你的数据库、配置 pgAdmin 等。你还会找到指向文档的有用链接。

如果你点击左上角的 "Servers"，你需要输入安装过程中选择的密码。

输入密码后，你应该会看到一个屏幕，在那里你可以访问你的数据库，包括它们的表格。

你还会找到另一个非常有用的工具，psql，这是一个用于 SQL 查询的交互式终端。要打开它，请在你的系统中查找 psql。点击它。

它将打开 SQL Shell，在那里你可以开始运行命令。

这就是你如何在 Windows 上安装 PostgreSQL。该进程在 macOS 和 Linux 上非常相似。

备选地，如果你在 macOS 上安装它，可以使用 Homebrew，一个包管理器。安装好 Homebrew 后，运行此命令来安装 PostgreSQL：

```bash
brew install postgresql@<version>
```

你可以通过在 at (`@`) 符号后面写版本号来指定你想安装的版本。
