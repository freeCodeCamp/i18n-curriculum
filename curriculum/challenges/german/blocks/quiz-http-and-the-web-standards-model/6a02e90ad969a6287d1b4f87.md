### --question--

#### --text--

Was ist der Unterschied zwischen einer `401 Unauthorized`- und einer `403 Forbidden`-Antwort?

#### --distractors--

`401` bedeutet, der Server ist abgestürzt, während `403` bedeutet, die Ressource existiert nicht.

---

`401` ist ein Clientfehler, während `403` ein Serverfehler ist.

---

`401` bedeutet, die Ressource wurde dauerhaft verschoben, während `403` bedeutet, sie wurde vorübergehend verschoben.

#### --answer--

`401` bedeutet, dass Anmeldedaten erforderlich sind, während `403` bedeutet, dass der Client gültige Anmeldedaten hat, aber keine Berechtigung für die Ressource besitzt.
