## Nozioni di base su CSS

- **Cos’è CSS?**: Cascading Style Sheets (CSS) è un linguaggio di markup usato per applicare stili agli elementi HTML. CSS si usa per colori, immagini di sfondo, layout e altro.
- **Anatomia di base di una regola CSS**: Una regola CSS è composta da due parti principali: un selettore e un blocco di dichiarazioni. Un selettore è un pattern usato in CSS per identificare e mirare specifici elementi HTML da stilizzare. Un blocco di dichiarazioni applica un insieme di stili a un selettore o a più selettori.

Ecco la sintassi generale di una regola CSS:

```css
selector {
    property: value;
}
```

- **Elemento `meta name="viewport"`**: Questo elemento `meta` dà al browser istruzioni su come controllare le dimensioni e la scala della pagina su dispositivi diversi, in particolare su telefoni cellulari e tablet.
- **Stili predefiniti del browser**: Ogni elemento HTML avrà applicati gli stili predefiniti del browser. Questo di solito include elementi come margini e spaziature interne predefiniti.

## CSS in linea, interno ed esterno

- **CSS in linea**: Questi stili sono scritti direttamente all’interno di un elemento HTML usando l’attributo `style`. La maggior parte delle volte non userai CSS in linea per via della separazione delle responsabilità.

Ecco un esempio di CSS in linea:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p style="color: red;">This is a red paragraph.</p>
```

:::

- **CSS interno**: Questi stili sono scritti all’interno dei tag `<style>` nella sezione `head` di un documento HTML. Può essere utile per creare esempi di codice brevi, ma di solito non userai CSS interno.
- **CSS esterno**: Questi stili sono scritti in un file CSS separato e collegati al documento HTML usando l’elemento `link` nella sezione `head`. Per la maggior parte dei progetti, userai un file CSS esterno invece di CSS interno o in linea.

## Lavorare con le proprietà `width` e `height`

- **Proprietà `width`**: Questa proprietà specifica la larghezza di un elemento. Se non specifichi una larghezza, il valore predefinito è `auto`. Questo permette al browser di determinare la larghezza dell’elemento in base al suo contenuto, genitore e tipo di visualizzazione.
- **Proprietà `min-width`**: Questa proprietà specifica la larghezza minima per un elemento.
- **Proprietà `max-width`**: Questa proprietà specifica la larghezza massima per un elemento.
- **Proprietà `height`**: Questa proprietà specifica l’altezza di un elemento. Analogamente, l’altezza è `auto` per impostazione predefinita, il che significa che si adatta al contenuto interno.
- **Proprietà `min-height`**: Questa proprietà specifica l’altezza minima per un elemento.
- **Proprietà `max-height`**: Questa proprietà specifica l’altezza massima per un elemento.

## Tipi diversi di combinatori CSS

- **Combinatore discendente**: Questo combinatore si usa per mirare agli elementi che sono discendenti di un elemento genitore specificato. L’esempio seguente mira a tutti gli elementi `li` all’interno degli elementi `ul`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul>
    <li>Example item one</li>
    <li>Example item two</li>
    <li>Example item three</li>
</ul>
```

```css
ul li {
    background-color: yellow;
}
```

:::

- **Combinatore figlio (`>`)**: Questo combinatore si usa per selezionare gli elementi che sono figli diretti di un elemento genitore specificato. L’esempio seguente mira a tutti gli elementi `p` che sono figli diretti della classe `container`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <p>This will get styled.</p>

  <div>
    <p>This will not get styled.</p>
  </div>
</div>
```

```css
.container > p {
  background-color: black;
  color: white;
}
```

:::

- **Combinatore fratello immediato (`+`)**: Questo combinatore seleziona un elemento che segue immediatamente un elemento fratello specificato. L’esempio seguente selezionerà il paragrafo che segue immediatamente l’elemento `h2`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h2>I am a sub heading</h2>

<p>This paragraph element will get a red background.</p>
```

```css
h2 + p {
  background-color: red;
}
```

:::

- **Combinatore fratello successivo (`~`)**: Questo combinatore seleziona tutti i fratelli di un elemento specificato che vengono dopo di esso. L’esempio seguente stilerà solo il secondo paragrafo perché è l’unico che è fratello dell’elemento `ul` e condivide lo stesso genitore.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <p>This will not get styled.</p>
  <ul>
    <li>Example item one</li>
    <li>Example item two</li>
    <li>Example item three</li>
  </ul>
  <p>This will get styled.</p>
</div>
<p>This will not get styled.</p>
```

```css
ul ~ p {
  background-color: green;
}
```

:::

## Elementi in linea, a blocco e inline-block

- **Elementi in linea**: Gli elementi in linea occupano solo la larghezza necessaria e non iniziano su una nuova riga. Questi elementi scorrono all’interno del contenuto, permettendo al testo e ad altri elementi in linea di apparire accanto a loro. Elementi in linea comuni sono `span`, `anchor` e `img`.
- **Elementi a blocco**: Gli elementi a blocco iniziano su una nuova riga e occupano per impostazione predefinita tutta la larghezza disponibile, estendendosi per tutta la larghezza del loro contenitore. Alcuni elementi a blocco comuni sono `div`, `paragraph` e `section`.
- **Elementi inline-block**: Puoi impostare un elemento come `inline-block` usando la proprietà `display`. Questi elementi si comportano come elementi in linea ma possono avere un `width` e `height` impostati come gli elementi a blocco.

## Margine e spaziatura interna

- **Proprietà `margin`**: Questa proprietà si usa per applicare spazio esterno all’elemento, tra il bordo dell’elemento e gli elementi circostanti.
- **Proprietà `padding`**: Questa proprietà si usa per applicare spazio interno all’elemento, tra il contenuto e il suo bordo.
- **Shorthand `margin`**: Puoi specificare da 1 a 4 valori per impostare i lati del margine. Un valore si applica a tutti e quattro i lati; due valori impostano `top` e `bottom`, poi `right` e `left`; tre valori impostano `top`, orizzontale (`right` e `left`), poi `bottom`; quattro valori impostano `top`, `right`, `bottom`, `left`.
- **Shorthand `padding`**: Puoi specificare da 1 a 4 valori per impostare i lati della spaziatura interna. Un valore si applica a tutti e quattro i lati; due valori impostano `top` e `bottom`, poi `right` e `left`; tre valori impostano `top`, orizzontale (`right` e `left`), poi `bottom`; quattro valori impostano `top`, `right`, `bottom`, `left`.

## Specificità CSS

- **Specificità del CSS in linea**: Il CSS in linea ha la specificità più alta perché viene applicato direttamente all’elemento. Sovrascrive qualsiasi CSS interno o esterno. Il valore di specificità per gli stili in linea è (1, 0, 0, 0).
- **Specificità del CSS interno**: Il CSS interno è definito all’interno di un elemento `style` nella sezione `head` del documento HTML. Ha una specificità inferiore rispetto agli stili in linea. Sia il CSS interno che quello esterno condividono lo stesso livello di specificità, che è determinato dai loro selettori, non dalla loro posizione.
- **Specificità del CSS esterno**: Il CSS esterno è collegato tramite un elemento `link` nella sezione `head` ed è scritto in file `.css` separati. Come il CSS interno, la sua specificità è determinata dai selettori usati. Quando due regole hanno la stessa specificità, l’ordine di origine determina il vincitore — la regola che appare più tardi nel documento ha la precedenza. Il CSS esterno offre la migliore manutenibilità per progetti più grandi.
- **Selettore universale (`*`)**: un tipo speciale di selettore CSS che corrisponde a qualsiasi elemento nel documento. Viene spesso usato per applicare uno stile a tutti gli elementi della pagina, utile per resettare o normalizzare gli stili tra browser diversi. Il selettore universale ha il valore di specificità più basso di qualsiasi selettore. Contribuisce con 0 a tutte le parti del valore di specificità (0, 0, 0, 0).
- **Selettori di tipo**: Questi selettori mirano agli elementi in base al loro nome di tag. I selettori di tipo hanno una specificità relativamente bassa rispetto ad altri selettori. Il valore di specificità per un selettore di tipo è (0, 0, 0, 1).
- **Selettori di classe**: Questi selettori sono definiti da un punto (`.`) seguito dal nome della classe. Il valore di specificità per un selettore di classe è (0, 0, 1, 0). Questo significa che i selettori di classe possono sovrascrivere i selettori di tipo, ma possono essere sovrascritti dai selettori ID e dagli stili in linea.
- **Selettori ID**: I selettori ID sono definiti da un simbolo hash (`#`) seguito dal nome ID. I selettori ID hanno una specificità molto alta, superiore ai selettori di tipo e di classe, ma inferiore agli stili in linea. Il valore di specificità per un selettore ID è (0, 1, 0, 0).
- **Parola chiave `!important`**: usata per dare a una regola di stile la priorità più alta, permettendo di sovrascrivere qualsiasi altra dichiarazione per una proprietà. Quando usata, forza il browser ad applicare lo stile specificato, indipendentemente dalla specificità degli altri selettori. Devi usare `!important` con cautela perché può rendere il tuo CSS più difficile da mantenere e eseguire il debug.
- **Algoritmo della cascata**: Un algoritmo usato per decidere quali regole CSS applicare quando ci sono più stili che mirano allo stesso elemento. Garantisce che vengano usati gli stili più appropriati, basandosi su un insieme di regole ben definite.
- **Ereditarietà CSS**: Il processo con cui gli stili vengono trasmessi dagli elementi genitori ai loro figli. L’ereditarietà ti permette di definire stili a un livello più alto nell’albero del documento e farli applicare a più elementi senza specificarli esplicitamente per ciascuno.
