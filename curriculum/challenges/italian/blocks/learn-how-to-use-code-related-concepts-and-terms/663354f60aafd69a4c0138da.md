# --description--

Il `equals symbol` (`=`) è comunemente usato per l'assegnazione. Serve ad assegnare un valore a una variabile. Il nome della variabile si scrive a sinistra del simbolo uguale, e il valore che vuoi archiviare nella variabile si trova a destra.

Per esempio, `name = "Sophie";` assegna la stringa `"Sophie"` alla variabile `name`.

Assegnare in programmazione significa dare un valore specifico a una variabile. Il `equals symbol` (`=`) è comunemente usato per questo scopo. Quando assegni un valore a una variabile, stai dicendo al programma di ricordare quel valore per usarlo in seguito.

Per esempio, `age = 25` assegna il valore `25` alla variabile `age`.
