# --description--

In the previous lectures, you learned how to call (invoke) a function. 调用函数意味着执行函数内部的代码。

下面是提醒大家如何调用函数：

```js
function myFunction() {
  return "Hello World";
}

// function call
myFunction();
```

添加一个 `console.log` 语句，并在其中调用 `addTwoAndSeven` 函数。

现在你应该可以看到控制台记录了 `2` 和 `7` 的和。
