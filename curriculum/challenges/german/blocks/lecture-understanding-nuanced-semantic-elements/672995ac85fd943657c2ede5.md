# --interactive--

Das „bring attention to“-Element, `b`, wird häufig verwendet, um Schlüsselwörter in Zusammenfassungen oder Produktnamen in Bewertungen hervorzuheben. Üblicherweise zeigen Browser diesen Text fett an. Hier ist ein Beispiel, bei dem das `b`-Element verwendet wird, um Produktnamen in dieser Bewertung hervorzuheben:

:::interactive_editor

```html
<p>
  We tested several products, including the <b>SuperSound 3000</b> for audio
  quality, the <b>QuickCharge Pro</b> for fast charging, and the
  <b>EcoClean Vacuum</b> for cleaning. The first two performed well, but the
  <b>EcoClean Vacuum</b> did not meet expectations.
</p>
```

:::

Der Browser rendert diese Textteile als fetten Text. Diese visuelle Hervorhebung zieht die Aufmerksamkeit der Leser auf die Produktnamen.

Wenn Sie die Wichtigkeit des Textes betonen müssen, sollten Sie stattdessen das `strong`-Element verwenden, nicht das `b`-Element.

`strong` ist ein semantisches HTML-Element, das Text hervorhebt, der entscheidend oder dringend ist. Hier ist ein Beispiel, bei dem das `strong`-Element verwendet wird, um eine sehr wichtige Warnung über mögliche allergische Reaktionen, die Kunden auf ein Produkt haben könnten, zu kennzeichnen:

:::interactive_editor

```html
<p>
  <strong>Warning:</strong> This product may cause allergic reactions.
</p>
```

:::

Das `strong`-Element vermittelt dieses Gefühl von Dringlichkeit.

Optisch sind beide sehr ähnlich, da sie standardmäßig fett dargestellt werden. Ihre Bedeutungen sind jedoch ganz unterschiedlich. Während das „bring attention to“-Element nur die Aufmerksamkeit auf den Text lenkt, ohne eine höhere Wichtigkeit anzuzeigen, macht das `strong`-Element mehr als das. Es vermittelt ein Gefühl von Wichtigkeit oder Dringlichkeit. Das ist ihr Hauptunterschied.

Um zwischen ihnen zu wählen, überlegen Sie, welchen Zweck der Text hat und wie wichtig er im Kontext des umgebenden Inhalts ist.
