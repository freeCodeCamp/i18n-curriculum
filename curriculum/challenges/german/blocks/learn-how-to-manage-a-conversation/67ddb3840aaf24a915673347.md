# --explanation--

`That's it for now` bedeutet, dass alles, was erledigt oder gesagt werden musste, fürs Erste abgeschlossen ist. Es lässt offen, dass später noch mehr passieren kann. Zum Beispiel:

`That's it for now — let's pick this up tomorrow.` – Das bedeutet, der aktuelle Teil ist beendet, aber das Gespräch oder die Aufgabe kann später fortgesetzt werden.
