# --description--

Теорія кольору та її вплив на дизайн — це глибока тема, і в наступних завданнях розглядаються лише основи. На вебсайті колір може привертати увагу до вмісту, викликати емоції або створювати візуальну гармонію. Використання різних комбінацій кольорів може суттєво змінити вигляд сайту, і багато уваги приділяється вибору палітри кольорів, що пасує до вашого вмісту.

Колірне коло — корисний інструмент для візуалізації того, як кольори пов’язані між собою — це коло, де схожі відтінки розташовані поруч, а різні — далі один від одного. Коли два кольори розташовані навпроти один одного на колі, їх називають доповнюючими кольорами. Вони мають властивість, що при поєднанні вони «нейтралізують» один одного і створюють сірий колір. Однак, коли ці кольори розміщені поруч, вони виглядають яскравішими і створюють сильний візуальний контраст.

Деякі приклади доповнюючих кольорів із їхніми шістнадцятковими кодами:

<blockquote>red (#FF0000) and cyan (#00FFFF)<br>green (#00FF00) and magenta (#FF00FF)<br>blue (#0000FF) and yellow (#FFFF00)</blockquote>

Це відрізняється від застарілої моделі кольорів RYB, яку багатьом із нас викладали в школі, де використовуються інші основні та доповнюючі кольори. Сучасна теорія кольору використовує адитивну модель RGB (як на екрані комп’ютера) та субтрактивну модель CMY(K) (як у друку).

Існує багато онлайн-інструментів для вибору кольорів, які мають опцію знайти доповнення кольору.

**Примітка:** Використання кольору може бути потужним способом додати візуальний інтерес до сторінки. Однак колір не повинен бути єдиним способом передачі важливої інформації, оскільки користувачі з порушеннями зору можуть не зрозуміти цей вміст. Це питання буде розглянуто детальніше у завданнях із прикладної доступності.

# --instructions--

Змініть властивість `background-color` у класах `blue` і `yellow` на відповідні кольори. Зверніть увагу, як кольори виглядають по-різному поруч один з одним, ніж на білому фоні.

# --hints--

Елемент `div` з класом `blue` повинен мати властивість `background-color` зі значенням синій.

```js
const blueElement = document.querySelector('.blue');
const blueStyle = window.getComputedStyle(blueElement); 
assert.equal(blueStyle?.backgroundColor, 'rgb(0, 0, 255)');
```

Елемент `div` з класом `yellow` повинен мати властивість `background-color` зі значенням жовтий.

```js
const yellowElement = document.querySelector('.yellow');
const yellowStyle = window.getComputedStyle(yellowElement);
assert.equal(yellowStyle?.backgroundColor, 'rgb(255, 255, 0)');
```
