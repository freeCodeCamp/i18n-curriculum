# --description--

피타고라스 다각형을 다음 조건을 만족하는 볼록 다각형으로 정의합니다:

- 꼭짓점이 최소 세 개 이상이다,
- 세 꼭짓점이 일직선상에 있지 않다,
- 각 꼭짓점의 좌표가 정수이다,
- 각 변의 길이가 정수이다.

주어진 정수 $n$에 대해, 둘레가 $≤ n$인 서로 다른 피타고라스 다각형의 개수를 $P(n)$이라 정의합니다.

피타고라스 다각형은 서로 평행이동으로 겹치지 않는 한 서로 다른 것으로 간주합니다.

$P(4) = 1$, $P(30) = 3655$, $P(60) = 891045$임이 주어졌습니다.

$P(120)$을 구하세요.
