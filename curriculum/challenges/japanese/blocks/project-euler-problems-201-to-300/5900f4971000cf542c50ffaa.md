# --description--

整数座標を持つ4つの点が選ばれています：

$A(a, 0)$、$B(b, 0)$、$C(0, c)$、$D(0, d)$、ただし $0 < a < b$ および $0 < c < d$ です。

点 $P$ も整数座標を持ち、$AC$ 線上に選ばれます。このとき、三角形 $ABP$、$CDP$、$BDP$ の3つがすべて相似になります。

<img alt="points A, B, C, D and P creating three triangles: ABP, CDP, and BDP" src="https://cdn.freecodecamp.org/curriculum/project-euler/three-similar-triangles.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

3つの三角形が相似になるためには、$a = c$ であることが簡単に証明できます。

したがって、$a = c$ とすると、$AC$ 上に少なくとも1つの点 $P$（整数座標）が存在し、三角形 $ABP$、$CDP$、$BDP$ の3つがすべて相似になるような組み合わせ ($a$, $b$, $d$) を探します。

例えば、$(a, b, d) = (2, 3, 4)$ の場合、点 $P(1, 1)$ が上記の条件を満たすことが簡単に確認できます。なお、組み合わせ (2,3,4) と (2,4,3) は異なるものとみなされますが、点 $P(1, 1)$ は両方に共通しています。

もし $b + d < 100$ なら、点 $P$ が存在する異なる組み合わせ ($a$, $b$, $d$) は92個あります。

もし $b + d < 100\,000$ なら、点 $P$ が存在する異なる組み合わせ ($a$, $b$, $d$) は320471個あります。

もし $b + d < 100\,000\,000$ なら、点 $P$ が存在する異なる組み合わせ ($a$, $b$, $d$) はいくつありますか？
