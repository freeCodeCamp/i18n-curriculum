# --description--

到目前为止，你已经练习了如何用 `do` / `does` 提问以及用 `don't` / `doesn't` 构成否定句。让我们复习一下。记住：

- 肯定：`He does work on weekends.`（表示此人在周末工作）

- 否定：`He doesn't work on weekends.`（表示此人周末不工作）

- 疑问句：`Does he work on weekends?`（询问此人是否在周末工作）
