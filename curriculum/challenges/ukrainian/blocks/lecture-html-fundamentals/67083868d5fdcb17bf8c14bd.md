# --interactive--

HTML-сутність, або посилання на символ, — це множина символів, які використовують для представлення зарезервованого символу в HTML.

Припустімо, ви хочете відобразити текст `This is an <img/> element` на екрані. Якщо використати код, який зараз у редакторі, бажаного результату не буде. Навіть якщо додати атрибути `src` і `alt` до прикладу, у середині абзацу з’явиться зображення. Це не те, що потрібно. Щоб взаємодіяти з прикладом, потрібно увімкнути інтерактивний редактор.


:::interactive_editor

```html
<p>This is an <img /> element</p>
```

:::

Коли HTML-парсер бачить символ менше (`<`) перед назвою HTML-тега, він інтерпретує це як HTML-елемент. Саме тому ви не бачите на екрані бажаний результат `This is an <img/> element`.

Щоб виправити цю проблему, можна використати HTML-сутності. Ось оновлений приклад із правильними HTML-сутностями для символів менше (`<`) та більше (`>`). Тепер на екрані має з’явитися `This is an <img/> element`.

Увімкніть інтерактивний редактор і спробуйте додати `&lt;p&gt;learning is fun&lt;/p&gt;` під елементом абзацу. Ви маєте побачити `<p>learning is fun</p>` на екрані.

:::interactive_editor

```html
<p>This is an &lt;img /&gt; element</p>

```

:::

Такі посилання на символи називають іменованими посиланнями. Іменовані посилання починаються з амперсанда (`&`) і закінчуються крапкою з комою (`;`). Використовуючи іменоване посилання, HTML-парсер не сплутає його з реальним HTML-елементом.

Інший тип посилання на символ — десяткове числове посилання. Воно починається з амперсанда і символу решітки (`#`), за якими йде один або кілька десяткових знаків, і завершується крапкою з комою.

Ось приклад використання десяткового числового посилання для символу менше.

Увімкніть інтерактивний редактор і змініть код, щоб побачити різні символи. Ви можете використати `&#169;` для символу авторського права і `&#174;` для символу зареєстрованої торгової марки.

:::interactive_editor

```html
&#60;
```

:::

Останній тип посилання на символ — шістнадцяткове числове посилання. Воно починається з амперсанда, символу решітки і літери `x`. Потім ідуть один або кілька ASCII шістнадцяткових знаків і завершується крапкою з комою.

Ось приклад використання шістнадцяткового числового посилання для символу менше.

Увімкніть інтерактивний редактор і змініть код, щоб побачити різні символи. Ви можете використати `&#x20AC;` для символу євро і `&#x03A9;` для символу грецької великої літери Омега.

:::interactive_editor

```html
&#x3C;
```

:::
