# --explanation--

`Wouldn't` es una contracción de `would not`, usada para formar una pregunta, a menudo para expresar duda o indagar sobre algo. Por ejemplo:

`The system should be secure, but wouldn't it be safer with extra testing?` - En este caso, el hablante cuestiona si se necesita más testing para hacer el sistema más seguro.

`Vulnerabilities` se refieren a debilidades en un sistema, software o red que pueden ser explotadas por atacantes. En este contexto. Por ejemplo:

La app tenía varias vulnerabilidades que los hackers podrían explotar.
