# --description--

`div` 요소의 스타일이 점점 완성되고 있습니다. 하지만 텍스트 주변에 약간의 여백이 있으면 좋겠습니다.

Tailwind CSS를 사용해 요소에 `padding`을 적용하려면 다음 패턴을 사용하세요: `p-number`. 여기서 각 `number` 단위는 네 면 모두에 적용할 0.25rem(또는 4px)을 나타냅니다.

요소에 `padding`을 `32px` 적용하는 예시는 다음과 같습니다:

```html
<p class="p-8">This is padding.</p>
```

여는 `div` 태그 안에 `padding`을 `16px` 적용할 수 있는 올바른 유틸리티 클래스를 사용하세요.

# --hints--

열린 `div` 태그에는 `p-4` 유틸리티 클래스가 있어야 합니다.

```js
const divEl = document.querySelector("div");
assert.isTrue(divEl.classList.contains("p-4"));
```
