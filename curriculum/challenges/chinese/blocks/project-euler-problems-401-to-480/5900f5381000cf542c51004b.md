# --description--

在欧几里得平面上，一只蚂蚁从点 $A(0, 1)$ 走到点 $B(d, 1)$，其中 $d$ 是一个 `integer`。

在每一步中，位于点 ($x_0$, $y_0$) 的蚂蚁选择满足 $x_1 ≥ 0$ 且 $y_1 ≥ 1$ 的格点 ($x_1$, $y_1$) 中的一个，并以常量速度 $v$ 直接前往 ($x_1$, $y_1$)。$v$ 的值取决于 $y_0$ 和 $y_1$，具体如下：

- 如果 $y_0 = y_1$，则 $v$ 的值等于 $y_0$。
- 如果 $y_0 ≠ y_1$，则 $v$ 的值等于 $\frac{y_1 - y_0}{\ln y_1 - \ln y_0}$。

左侧图像是 $d = 4$ 的一种可能路径。首先蚂蚁以速度 $\frac{3 - 1}{\ln 3 - \ln 1} ≈ 1.8205$ 从 $A(0, 1)$ 移动到 $P_1(1, 3)$。然后所需时间为 $\frac{\sqrt{5}}{1.820} ≈ 1.2283$。

从 $P_1(1, 3)$ 到 $P_2(3, 3)$ 蚂蚁以速度 3 行进，因此所需时间是 $\frac{2}{3} ≈ 0.6667$。从 $P_2(3, 3)$ 到 $B(4, 1)$ 蚂蚁以速度 $\frac{1 - 3}{\ln 1 - \ln 3} ≈ 1.8205$ 行进，因此所需时间是 $\frac{\sqrt{5}}{1.8205} ≈ 1.2283$。

因此总共需要的时间是 $1.2283 + 0.6667 + 1.2283 = 3.1233$。

右侧图像是另一条路径。总所需时间计算为 $0.98026 + 1 + 0.98026 = 2.96052$。可以证明这是 $d = 4$ 时最快的路径。

<img alt="two possible paths for d = 4" src="https://cdn.freecodecamp.org/curriculum/project-euler/an-ant-on-the-move.jpg" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

设 $F(d)$ 为蚂蚁选择最快路径时所需的总时间。例如，$F(4) ≈ 2.960\\,516\\,287$。我们可以验证 $F(10) ≈ 4.668\\,187\\,834$ 和 $F(100) ≈ 9.217\\,221\\,972$。

查找 $F(10\,000)$。将你的答案四舍五入到九位小数。
