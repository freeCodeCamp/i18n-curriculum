# --interactive--

有几种方法可以从一个对象中删除属性，其中使用 `delete` 操作符是最直接且最常用的方法。

当你使用 `delete` 时，它会从对象中删除所选的属性。以下是如何使用 `delete` 操作符的示例：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  job: "Engineer"
};

delete person.job;

console.log(person.job); // undefined
```

:::

在此示例中，我们从一个具有三个属性的 `person` 对象开始：`name`、`age` 和 `job`。然后，我们使用 `delete` 操作符删除 `job` 属性。删除后，`person` 对象不再具有 `job` 属性。

另一种删除属性的方法是使用带有剩余参数的解构赋值。这种方法实际上并不会删除属性，而是创建一个不包含指定属性的新对象：

:::interactive_editor

```js
const person = {
  name: "Bob",
  age: 25,
  job: "Designer",
  city: "New York"
};

const { job, city, ...remainingProperties } = person;

// { name: "Bob", age: 25 }
console.log(remainingProperties);
```

:::

在此示例中，我们使用解构从 `person` 对象中提取 `job` 和 `city`，并将剩余属性收集到一个名为 `remainingProperties` 的新建对象中。这会创建一个不包含 `job` 和 `city` 属性的新对象。

理解如何从对象中移除属性是 JavaScript 编程中的一项重要技能。它允许你动态操作对象，并清理不必要的数据。
