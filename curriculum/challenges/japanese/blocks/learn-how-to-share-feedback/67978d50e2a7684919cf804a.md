# --explanation--

`To set goals` は、達成すべき具体的な目標やターゲットを定めることを意味します。例えば：

`The team set goals to improve customer satisfaction this year.` - これはチームが顧客満足に関する目標を設定したことを意味します。

`Quarter` は1年の4分の1、つまり3か月の期間を指します。ビジネスの文脈では、`the next quarter` はカレンダー年または会計年度の次の3か月間を意味します。例えば：

- `Our sales target for the next quarter is $1 million.` - これは目標が次の3か月間に適用されることを意味します。

- `We review our performance at the end of each quarter.` - これは3か月ごとに進捗を評価することを指します。
