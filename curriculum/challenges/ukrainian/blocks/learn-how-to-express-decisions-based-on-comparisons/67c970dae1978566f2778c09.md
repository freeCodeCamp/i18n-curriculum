# --explanation--

`Slow down` означає змусити щось працювати повільніше або менш ефективно. Наприклад:  

`Too many background apps can slow down your computer.` - Це означає, що комп’ютер стає менш адаптивним.  

`As much` використовується для порівняння двох речей, означаючи в такій же мірі або ступені. Наприклад:  

`This software doesn't use as much memory as the previous version.` - Це означає, що програмне забезпечення використовує менше пам’яті порівняно зі старим.
