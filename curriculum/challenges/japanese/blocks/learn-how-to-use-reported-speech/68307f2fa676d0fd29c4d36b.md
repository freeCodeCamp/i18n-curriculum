# --explanation--

`Was there` は`There was`の疑問形で、過去に何かが存在したか、起こったかを尋ねるために使われます。主語と動詞を入れ替えて疑問文を作ります。例えば：

- `There was a problem with the layout.`（肯定文 - 問題が起こったと言っている）  

- `Was there a problem with the layout?`（疑問文 - 問題が起こったかどうかを尋ねている）

これは過去の何かの存在や発生について尋ねるのに役立ちます。
