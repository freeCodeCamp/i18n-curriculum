# --description--

`equals symbol`（`=`）通常用于赋值。它用于将值赋值给变量。变量名写在等号左侧，想要保存到变量中的值写在右侧。

例如，`name = "Sophie";` 将字串 `"Sophie"` 赋值给变量 `name`。

在编程中，`assign a value` 意味着给变量赋予一个特定的值。通常使用 `equals symbol`（`=`）来实现这一目的。当你为变量赋值时，你是在告诉程序记住该值以便以后使用。

例如，`age = 25` 将值 `25` 赋值给变量 `age`。
