# --description--

文字列が有効なIPv4アドレスかどうかを判定してください。有効なIPv4アドレスは、ドット（`.`）で区切られた4つの整数で構成されています。各数字は以下の条件を満たす必要があります。

- 0から255までの範囲内であること（両端を含む）。
- 先頭にゼロがないこと（例：0は許可されますが、01は許可されません）。
- 数字のみが許可されます。

```py
def is_valid_ipv4(ipv4):
    parts = ipv4.split(".")    

    if len(parts) != 4:
        return False

    for part in parts:
        if not part.isdigit():
            return False

        num = int(part)
        if num < 0 or num > 255:
            return False

        if len(part) > 1 and part.startswith("0"):
            return False

    return True
```
