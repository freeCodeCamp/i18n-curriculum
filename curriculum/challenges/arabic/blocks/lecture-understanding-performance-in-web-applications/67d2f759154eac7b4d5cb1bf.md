# --description--

كما نعلم، الأداء المدرك ليس فقط عن مدى سرعة الموقع فعليًا — بل عن مدى السرعة التي يشعر بها المستخدمون. فمثلًا، هل سبق أن استخدمت موقعًا بدا بطيئًا رغم أنه لم يكن كذلك؟ هذا هو الأداء المدرك — مدى سرعة شعور المستخدم بالموقع.

تعديلات صغيرة يمكن أن تُحدث فرقًا كبيرًا في تجربة المستخدم. دعنا نستعرض بعض الطرق العملية لتحسين الأداء المدرك!

المؤشرات الدوارة قد تجعل المستخدمين يشعرون بأنهم ينتظرون لفترة أطول مما هي عليه فعليًا. بدلًا من ذلك، جرب شاشات الهيكل العظمي — عناصر نائب رمادية تحاكي المحتوى الحقيقي وتُحمّل تدريجيًا:

```html
<div class="skeleton"></div>
<script>
  setTimeout(() => {
    document.querySelector(".skeleton").innerHTML = "✅ Content Loaded!";
  }, 2000);
</script>
```

يشعر المستخدمون أن الصفحة تُحمّل بشكل تدريجي، بدلاً من الانتظار في الظلام.

بالإضافة إلى ذلك، لا يجب أن ينتظر المستخدمون تحميل كل شيء قبل أن يبدأوا بالتفاعل. حمّل المحتوى المهم أولًا، ثم حمّل الباقي في الخلفية. لقد ناقشنا هذا سابقًا مع التحميل الكسول:

```html
<img src="placeholder.jpg" data-src="real-image.jpg" class="lazy">
<script>
  document.addEventListener("DOMContentLoaded", function () {
    const lazyImages = document.querySelectorAll("img.lazy");
    lazyImages.forEach(img => {
      img.src = img.dataset.src;
    });
  });
</script>
```

هذا يجعل الصفحة تبدو متجاوبة، حتى لو لم يُحمّل كل شيء بعد.

والآن - ماذا لو كان موقعك يستطيع التنبؤ بما سينقر عليه المستخدم بعد ذلك؟ مع التحميل المسبق، يمكنك جلب الصفحة التالية قبل أن يطلبها المستخدم:

```html
<link rel="preload" href="next-page.html" as="document">
```

هذا يقلل من وقت التحميل المدرك إلى ما يقارب الصفر عند تنقل المستخدمين.

وأخيرًا، المستخدمون لا يحبون عدم اليقين. عندما ينقرون على زر، اعترف بفعلهم فورًا:

```html
<button onclick="this.innerText='⏳ Processing...'; setTimeout(() => this.innerText='✅ Done!', 2000);">
  Click Me
</button>
```

هذا يعني أنه حتى لو كان هناك تأخير، يشعر المستخدم بأن شيئًا ما يحدث.

لذا، في الختام، تحسين الأداء المدرك ليس فقط عن السرعة الخام، بل عن كيف يشعر المستخدم بالسرعة. استخدم شاشات الهيكل العظمي، وأعطِ أولوية للمحتوى المهم، وحمّل مسبقًا بذكاء، وقدم ردود فعل فورية لجعل موقعك يبدو أسرع.
