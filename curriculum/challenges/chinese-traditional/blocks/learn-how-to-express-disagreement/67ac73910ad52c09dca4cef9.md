# --explanation--

`Not necessarily` 是一種禮貌表達部分不同意見的方式。它不是直接說 `no`，而是暗示可能有其他因素需要考慮。在對話中，這個片語有助於保持討論的開放性和尊重。 例如：

- 使用者 1：`I think upgrading the computer's processor will fix the issue.`

- 第二個人：`Not necessarily. The problem could be with the software, not the hardware.`－這個人建議升級處理器可能無法解決問題，因為問題可能是由其他原因造成，例如軟體過時。
