# --description--

Neno `might` ni kitenzi cha modal kinachotumika kudhihirisha uwezekano au uwezo. Usemi `be able to` unaonyesha uwezo.

Pamoja, `might be able to` inaashiria kwamba kuna uwezekano wa kufanikisha jambo fulani, lakini si hakika.

Kwa mfano, `She might be able to finish the work by tomorrow.` Hii ina maana kwamba inawezekana atakamilisha kazi hiyo wakati huo, lakini si uhakika.
