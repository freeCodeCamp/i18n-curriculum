# --explanation--

Марк згадує `incompatible drivers` і `outdated software` як причини збою програмного забезпечення.

`Incompatible drivers` — це драйвери, які некоректно працюють із апаратним забезпеченням, а `outdated software` означає, що програмне забезпечення не оновлене до останньої версії. Обидва можуть спричинити збої. Наприклад:

- `The graphics card isn't working because of incompatible drivers.` — неправильний драйвер викликає несправність відеокарти.

- `Outdated software can lead to security vulnerabilities.` — старі програми можуть бути небезпечними для використання через відсутність оновлень.
