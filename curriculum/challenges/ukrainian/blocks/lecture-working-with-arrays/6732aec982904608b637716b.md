# --interactive--

Масиви в JavaScript є динамічними, що означає, що ви можете легко додавати або видаляти елементи з них. Існує чотири основні методи для додавання та видалення елементів з початку та кінця масиву: `push()`, `pop()`, `shift()` та `unshift()`. Давайте детально розглянемо кожен із цих методів.

Метод `push()` використовується для додавання одного або кількох елементів до кінця масиву. Повернене значення методу `push()` — це нова довжина масиву. Ось приклад додавання нового фрукта до існуючого масиву `fruits`:

:::interactive_editor

```js
const fruits = ["apple", "banana"];
const newLength = fruits.push("orange");
console.log(newLength); // 3
console.log(fruits); // ["apple", "banana", "orange"]
```

:::

У цьому прикладі ми починаємо з масиву під назвою `fruits`, який містить два елементи. Потім ми використовуємо метод `push()`, щоб додати рядок `orange` до кінця масиву.

Ви могли помітити, що ми використовуємо `const` при оголошенні масиву `fruits`. Але чому можливо додавати більше елементів до цього `fruits` масиву, якщо `fruits` є константою? Це можливо, тому що оголошення масиву з ключовим словом `const` створює референс на масив. Хоча сам масив є змінним і його можна модифікувати, ви не можете переназначити нове значення константі `fruits`, як ось так:

```js
const fruits = ["apple", "banana"];
fruits = ["This", "will", "not", "work"];
console.log(fruits); // Uncaught TypeError: Assignment to constant variable. 
```

Наступним методом, який ми розглянемо, є метод `pop()`. Метод `pop()` видаляє останній елемент з масиву і повертає цей елемент. Він також змінює оригінальний масив. Ось як це працює:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let lastFruit = fruits.pop();
console.log(fruits); // ["apple", "banana"]
console.log(lastFruit); // "orange"
```

:::

У цьому прикладі ми починаємо з масиву з трьох фруктів. Метод `pop()` видаляє останній елемент (`orange`) з масиву і повертає його. Оригінальний масив `fruits` змінено і тепер містить лише два елементи.

Метод `unshift()` додає один або кілька елементів на початок масиву і повертає його нову довжину. Він працює подібно до `push()`, але змінює початок масиву замість кінця. Ось приклад:

:::interactive_editor

```js
let numbers = [2, 3];
let newLength = numbers.unshift(1);
console.log(numbers); // [1, 2, 3]
console.log(newLength); // 3
```

:::

У цьому прикладі ми використовуємо `unshift()`, щоб додати число `1` на початок масиву `numbers`. Метод повертає нову довжину масиву, яка дорівнює `3`.

Нарешті, метод `shift()` видаляє перший елемент з масиву і повертає цей елемент. Він схожий на `pop()`, але працює на початку масиву замість кінця. Ось як це працює:

:::interactive_editor

```js
let colors = ["red", "green", "blue"];
let firstColor = colors.shift();
console.log(colors); // ["green", "blue"]
console.log(firstColor); // "red"
```

:::

У цьому прикладі ми починаємо з масиву з трьох кольорів. Метод `shift()` видаляє перший елемент (`red`) з масиву і повертає його. Оригінальний масив `colors` змінено і тепер містить лише два елементи.

Зверніть увагу, що хоча `push()` і `unshift()` можуть додавати кілька елементів одночасно, `pop()` і `shift()` видаляють лише по одному елементу за раз.
