# --explanation--

`Should`은 무언가를 제안하거나 권장할 때 사용되지만 `must`나 `have to`만큼 강하지 않습니다.

Jessica는 또한 `it's not mandatory`라고 말했는데, 이는 Mark가 정기적인 보안 감사를 반드시 해야 하는 엄격한 요구사항이 아니라는 뜻입니다. 단지 취약점을 조기에 발견하는 것이 좋기 때문에 권장하는 것일 뿐, 반드시 해야 하는 것은 아닙니다.
