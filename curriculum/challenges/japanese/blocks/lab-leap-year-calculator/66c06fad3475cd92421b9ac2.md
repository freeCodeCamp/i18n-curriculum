# --description--

うるう年とは`4`で割り切れる年のことですが、`100`で割り切れて`400`で割り切れない年は除きます。例えば、`2000`はうるう年ですが、`1900`はうるう年ではありません。また、うるう年は2月に29日という1日多い日があります。

**目的:** 以下のユーザーストーリーを満たし、すべてのテストに合格してラボを完了してください。

**ユーザーストーリー:**

1. `isLeapYear`という名前の関数を定義し、数値を引数として受け取ります。
2. 関数の外で、調べたい年の値を格納する`year`という変数を宣言してください。
3. 関数内で、`if/ else`文または三項演算子を使って、その年がうるう年かどうかを判定してください。
4. うるう年かどうかを判定するには、以下の条件を満たしてください：

   - 年が`4`で割り切れる場合はうるう年です。
   - ただし、年が`100`でも割り切れる場合はうるう年ではありません。
   - ただし、年が`400`でも割り切れる場合はうるう年です。

5. うるう年の場合は`[year] is a leap year.`を返し、そうでなければ`[year] is not a leap year.`を返してください。`[year]`は`isLeapYear`関数で定義したパラメータに置き換えます。
6. `isLeapYear`関数を`year`を引数にして呼び出し、その結果を`result`という変数に代入してください。
7. `result`変数を`console.log()`を使ってコンソールに出力してください。

function isLeapYear(year) {
  if ((year % 4 === 0 && year % 100 !== 0) || year % 400 === 0) {
    return(year + " is a leap year.");
  } else {
    return(year + " is not a leap year.");
  }
}
