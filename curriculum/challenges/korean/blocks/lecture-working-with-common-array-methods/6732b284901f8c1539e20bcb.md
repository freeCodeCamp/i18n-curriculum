# --interactive--

JavaScript의 `splice()` 메서드는 배열을 수정하는 강력한 방법입니다. 이 메서드는 배열의 중간을 포함해 어떤 위치에서든 요소를 추가하거나 제거할 수 있습니다. `splice()` 메서드의 반환값은 배열에서 제거된 항목들의 배열입니다. 만약 아무것도 제거되지 않았다면 빈 배열이 반환됩니다.

이 메서드는 원본 배열을 변경하여 새로운 배열을 생성하지 않고 제자리에서 수정한다는 점을 기억하세요. `splice()` 작업 시 이 점을 유념해야 합니다. 기본 구문은 다음과 같습니다:

```js
array.splice(startIndex, itemsToRemove, item1, item2)
```

`startIndex`는 배열 수정을 시작할 인덱스를 지정하고, `itemsToRemove`는 제거할 요소 수를 나타내는 선택적 매개변수입니다. `itemsToRemove`가 생략되면 `splice()`는 `startIndex`부터 배열 끝까지 모든 요소를 제거합니다. 그 다음 매개변수들(`item1`, `item2` 등)은 시작 인덱스부터 배열에 추가할 요소들입니다.

이제 배열 중간에서 요소를 제거하는 예제를 살펴보겠습니다:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango", "kiwi"];
let removed = fruits.splice(2, 2);

console.log(fruits);  // ["apple", "banana", "kiwi"]
console.log(removed); // ["orange", "mango"]
```

:::

이 예제에서 `splice(2, 2)`은 인덱스 `2`에서 시작해 `2`개의 요소를 제거합니다. 수정된 배열은 이제 `apple`, `banana`, `kiwi`만 포함합니다. 다음으로 배열 중간에 요소를 추가하는 방법을 보겠습니다:

:::interactive_editor

```js
let colors = ["red", "green", "blue"];
colors.splice(1, 0, "yellow", "purple");

console.log(colors); // ["red", "yellow", "purple", "green", "blue"]
```

:::

여기서 `splice(1, 0, "yellow", "purple")`은 인덱스 `1`에서 시작해 `0`개의 요소를 제거하고 `yellow`와 `purple`을 삽입합니다. 두 번째 매개변수(`0`)는 삽입 전에 제거할 요소가 없음을 의미합니다. `splice()`를 사용하면 요소를 동시에 제거하고 추가할 수도 있습니다:

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];
numbers.splice(1, 2, 6, 7, 8);

console.log(numbers); // [1, 6, 7, 8, 4, 5]
```

:::

이 경우 `splice(1, 2, 6, 7, 8)`은 인덱스 `1`에서 시작해 `2`개의 요소(`2`와 `3`)를 제거하고 `6`, `7`, `8`을 삽입합니다. 원본 배열을 변경하지 않고 유지하려면 `splice()`를 사용하기 전에 복사본을 만들어야 합니다:

:::interactive_editor

```js
let original = [1, 2, 3, 4, 5];
let copy = [...original];
copy.splice(2, 1, 6);

console.log(original); // [1, 2, 3, 4, 5]
console.log(copy);     // [1, 2, 6, 4, 5]
```

:::

이 예제에서는 `original` 배열을 수정하지 않고 복사본을 만들기 위해 전개 연산자(`...`)를 사용합니다. 전개 연산자는 `original` 배열의 요소들을 새 배열에 얕게 복사합니다. 이 내용은 이후 수업에서 더 자세히 배울 것입니다.

`copy.splice(2, 1, 6)`를 사용하면 복사한 배열에서 인덱스 `2`에 있는 요소(`3`)를 제거하고 그 위치에 새 요소 `6`를 삽입해 배열을 수정합니다.

`splice()`의 일반적인 용도 중 하나는 인덱스를 알고 있을 때 배열에서 단일 요소를 제거하는 것입니다:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango"];
let indexToRemove = fruits.indexOf("orange");
if (indexToRemove !== -1) {
    fruits.splice(indexToRemove, 1);
}

console.log(fruits); // ["apple", "banana", "mango"]
```

:::

이 예제에서는 먼저 `indexOf()` 메서드를 사용해 `orange` 배열에서 요소 `fruits`의 인덱스를 찾습니다. `indexOf()` 메서드는 주어진 요소가 배열에 처음 나타나는 인덱스를 반환하며, 요소가 없으면 `-1`를 반환합니다.

그다음 `indexToRemove`와 `-1`를 비교해 요소가 배열에 존재하는지 확인한 후 제거를 시도합니다. `indexToRemove`가 `-1`와 같지 않으면(즉, 요소가 발견되면) `splice()`를 사용해 `indexToRemove` 위치에서 한 개의 요소를 제거합니다.

`splice()`를 사용하면 배열의 모든 요소를 제거해 배열을 지울 수도 있습니다:

:::interactive_editor

```js
let array = [1, 2, 3, 4, 5];
array.splice(0);

console.log(array); // []
```

:::

`splice()`는 강력하지만, 매우 큰 배열에서 특히 배열 앞부분을 수정할 때는 다른 방법보다 효율이 떨어질 수 있습니다. 이는 `splice()`가 이후 모든 요소를 이동시켜야 할 수 있기 때문입니다. 이런 경우 배열 끝에서만 요소를 추가하거나 제거한다면 `push()`, `pop()`, `unshift()`, `shift()` 같은 메서드가 더 적합할 수 있습니다.

결론적으로 `splice()` 메서드는 JavaScript에서 배열을 수정하는 다재다능한 방법입니다. 배열의 어느 위치에서든 요소를 추가하거나 제거하는 정밀한 제어가 가능합니다. `splice()`를 효과적으로 사용하는 방법을 이해하면 JavaScript 코드에서 배열을 다루는 능력이 크게 향상됩니다.
