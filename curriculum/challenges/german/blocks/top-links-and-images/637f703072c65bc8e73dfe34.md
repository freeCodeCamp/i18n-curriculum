- Erstellen Sie ein neues Verzeichnis mit dem Namen `odin-links-and-images`.

- Erstellen Sie innerhalb dieses Verzeichnisses eine neue Datei mit dem Namen `index.html`.

- Füllen Sie die übliche Boilerplate-Code für HTML aus.

- Fügen Sie schließlich das folgende `h1` in das `body` ein: `<h1>Homepage</h1>`

## Ankerelemente
Um einen Link in HTML zu erstellen, verwenden Sie das Ankerelement. Ein Ankerelement wird definiert, indem Sie den Text oder ein anderes HTML-Element, das Sie als Link haben möchten, mit einem `<a>`-Tag umschließen.

Fügen Sie Folgendes in das `body` der `index.html`-Seite ein, die Sie erstellt haben, und öffnen Sie sie im Browser:

```html
<a>click me</a>
```

Vielleicht ist Ihnen aufgefallen, dass das Klicken auf diesen Link nichts bewirkt. Das liegt daran, dass ein Anker-Tag allein nicht weiß, wohin Sie verlinken möchten. Sie müssen ihm ein Ziel angeben. Dies tun Sie, indem Sie ein HTML-Attribut verwenden.

Ein HTML-Attribut gibt einem HTML-Element zusätzliche Informationen und steht immer im öffnenden Tag des Elements. Ein Attribut besteht normalerweise aus zwei Teilen: einem Namen und einem Wert; jedoch benötigen nicht alle Attribute einen Wert. In Ihrem Fall müssen Sie dem öffnenden Anker-Tag ein `href` (Hyperlink-Referenz) Attribut hinzufügen. Der Wert des `href`-Attributs ist das Ziel, zu dem Ihr Link führen soll.

Fügen Sie dem zuvor erstellten Ankerelement das folgende `href`-Attribut hinzu und versuchen Sie erneut, darauf zu klicken. Vergessen Sie nicht, den Browser zu aktualisieren, damit die neuen Änderungen übernommen werden.

```html
<a href="https://www.theodinproject.com/about">click me</a>
```

Standardmäßig sieht jeder Text, der mit einem Anker-Tag ohne `href`-Attribut umschlossen ist, wie normaler Text aus. Wenn das `href`-Attribut vorhanden ist, zeigt der Browser den Text in blauer Farbe und unterstrichen an, um zu signalisieren, dass es sich um einen Link handelt.

Es ist erwähnenswert, dass Sie Anker-Tags verwenden können, um auf jede Art von Ressource im Internet zu verlinken, nicht nur auf andere HTML-Dokumente. Sie können auf Videos, PDF-Dateien, Bilder und so weiter verlinken, aber meistens werden Sie auf andere HTML-Dokumente verlinken.
