# --description--

`Le` ist ein Pronomen, das „ihm“, „ihr“ oder „Ihnen“ darstellt.

`Le gusta` bedeutet „er mag“, wenn es sich auf Diego bezieht, der männlich ist. Das ist dasselbe, wenn man sich auf eine weibliche Person bezieht. Zum Beispiel:

- `Le gusta el fútbol.` – Er mag Fußball (bezieht sich auf einen Mann).

- `Le gusta el fútbol.` – Sie mag Fußball (bezieht sich auf eine Frau).
