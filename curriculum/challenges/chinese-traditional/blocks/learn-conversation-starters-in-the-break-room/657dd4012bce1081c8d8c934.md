# --description--

`Any` 經常用於疑問句中，詢問某物是否存在或可用，而不指定數量。當你不知道是否有某物或有多少，並且想要了解時，它非常有用。 例如：

`Are there any updates?`－這裡，`any` 用來詢問是否有更新而不指定數量。 

`Any` 常用於疑問句和否定句中，表示某些甚至一點點。以下是你可能在不同背景關係中使用它的方式：

- 在問題中：`Do you have any questions?`（不論有很多問題還是一個問題；你是在詢問是否有任何問題。）
- 在否定句中：`I don't have any money.`（你一毛錢都沒有。）
- 使用報價和請求時：`Could I have any of these cookies?`（哪一種餅乾都可以；任何餅乾都行。）
