# --interactive--

Einen String umzukehren ist eine häufige Programmieraufgabe, die in JavaScript mit einer Kombination aus String- und Array-Methoden erledigt werden kann. Der Prozess umfasst drei Hauptschritte:

- Den String in ein Array von Zeichen aufteilen.
- Das Array umkehren.
- Die Zeichen wieder zu einem String zusammenfügen.

Lassen Sie uns jeden dieser Schritte mit den `split()`, `reverse()` und `join()` Methoden genauer ansehen.

Der erste Schritt beim Umkehren eines Strings ist, ihn in ein Array einzelner Zeichen umzuwandeln. Das können wir mit der `split()` Methode machen. Die `split()` Methode teilt einen String in ein Array von Teilstrings auf und gibt an, wo die Aufteilung anhand eines angegebenen Trennzeichens erfolgen soll. Wenn kein Trennzeichen angegeben wird, gibt die Methode ein Array zurück, das den ursprünglichen String als einzelnes Element enthält. Beispiele für häufige Trennzeichen sind:

- Ein leerer String (`""`), der den String in einzelne Zeichen aufteilt.

- Ein einzelnes Leerzeichen (`" "`), das den String an jedem Leerzeichen aufteilt.

- Ein Bindestrich (`"-"`), der den String an jedem Bindestrich aufteilt.

Hier ist ein Beispiel, wie die `split()` Methode verwendet wird, um ein Array von Zeichen zu erstellen:

:::interactive_editor

```js
let str = "hello";
let charArray = str.split("");
console.log(charArray); // ["h", "e", "l", "l", "o"]
```

:::

In diesem Beispiel verwenden wir `split("")` (mit einem leeren String als Argument), um den String `hello` in ein Array seiner einzelnen Zeichen umzuwandeln. Sobald wir ein Array von Zeichen haben, können wir die `reverse()` Methode verwenden, um die Reihenfolge der Elemente im Array umzukehren.

Die `reverse()` Methode ist eine Array-Methode, die die Elemente eines Arrays an Ort und Stelle umkehrt. Das bedeutet, sie verändert das ursprüngliche Array, anstatt ein neues zu erstellen. So können wir sie verwenden:

:::interactive_editor

```js
let charArray = ["h", "e", "l", "l", "o"];
charArray.reverse();
console.log(charArray); // ["o", "l", "l", "e", "h"]
```

:::

In diesem Beispiel ändert `reverse()` die Reihenfolge der Elemente in `charArray` und kehrt sie von `["h", "e", "l", "l", "o"]` zu `["o", "l", "l", "e", "h"]` um.

Der letzte Schritt ist, das umgekehrte Array von Zeichen wieder in einen String umzuwandeln. Das erreichen wir mit der `join()` Methode. Die `join()` Methode erstellt und gibt einen neuen String zurück, indem alle Elemente eines Arrays zusammengefügt werden, getrennt durch einen angegebenen Trennstring. Wenn Sie die Zeichen ohne Trennzeichen zusammenfügen möchten, können Sie einen leeren String als Argument verwenden. Hier ist ein Beispiel:

:::interactive_editor

```js
let reversedArray = ["o", "l", "l", "e", "h"];
let reversedString = reversedArray.join("");
console.log(reversedString); // "olleh"
```

:::

In diesem Beispiel kombiniert `join("")` (mit einem leeren String als Argument) alle Zeichen im Array zu einem einzigen String, ohne Trennzeichen dazwischen.

Denken Sie daran, dass Strings in JavaScript unveränderlich sind, was bedeutet, dass Sie einen String nicht direkt durch Modifikation umkehren können. Deshalb müssen wir ihn in ein Array umwandeln, das Array umkehren und dann wieder in einen String umwandeln. Diese Kombination aus String- und Array-Methoden bietet eine leistungsstarke und flexible Möglichkeit, Strings in JavaScript zu bearbeiten.
