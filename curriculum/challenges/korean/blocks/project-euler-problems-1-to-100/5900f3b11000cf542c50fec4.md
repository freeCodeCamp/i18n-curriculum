# --description--

오일러의 오일러 피 함수, ${\phi}(n)$ (때때로 파이 함수라고도 함)는 `n`보다 작은 수 중 `n`와 서로소인 수의 개수를 구하는 데 사용됩니다. 예를 들어, 1, 2, 4, 5, 7, 8은 모두 9보다 작고 9와 서로소이므로 ${\phi}(9) = 6$입니다.

<div style='margin-left: 4em;'>

| $n$ | $\text{서로소}$ | $\displaystyle{\phi}(n)$ | $\displaystyle\frac{n}{{\phi}(n)}$ |
| --- | ------------------------- | ------------------------ | ---------------------------------- |
| 2   | 1                         | 1                        | 2                                  |
| 3   | 1,2                       | 2                        | 1.5                                |
| 4   | 1,3                       | 2                        | 2                                  |
| 5   | 1,2,3,4                   | 4                        | 1.25                               |
| 6   | 1,5                       | 2                        | 3                                  |
| 7   | 1,2,3,4,5,6               | 6                        | 1.1666...                          |
| 8   | 1,3,5,7                   | 4                        | 2                                  |
| 9   | 1,2,4,5,7,8               | 6                        | 1.5                                |
| 10  | 1,3,7,9                   | 4                        | 2.5                                |

</div>

`n` = 6이 `n` ≤ 10에 대해 최대 $\displaystyle\frac{n}{{\phi}(n)}$ 값을 만든다는 것을 알 수 있습니다.

`n` ≤ `limit`인 값 중에서 $\displaystyle\frac{n}{{\phi(n)}}$가 최대가 되는 값을 찾으세요.

function totientMaximum(limit) {
  function getSievePrimes(max) {
    const primesMap = new Array(max).fill(true);
    primesMap[0] = false;
    primesMap[1] = false;
    const primes = [];
    for (let i = 2; i < max; i = i + 2) {
      if (primesMap[i]) {
        primes.push(i);
        for (let j = i * i; j < max; j = j + i) {
          primesMap[j] = false;
        }
      }
      if (i === 2) {
        i = 1;
      }
    }
    return primes;
  }

  const MAX_PRIME = 50;
  const primes = getSievePrimes(MAX_PRIME);
  let result = 1;

  for (let i = 0; result * primes[i] < limit; i++) {
    result *= primes[i];
  }
  return result;
}
