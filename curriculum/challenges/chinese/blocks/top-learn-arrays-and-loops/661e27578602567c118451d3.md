# --description--

向数组添加新元素最常见的方法之一是使用 `push()` 方法。`push()` 方法将一个或多个元素添加到数组末尾，并返回数组的新长度。

例如，要为 `pet` 数组添加一个新建元素，你可以使用以下代码：

```javascript
const pet = ['cat', 'dog', 'bunny'];
pet.push('parrot');
console.log(pet); // Output: ['cat', 'dog', 'bunny', 'parrot']
```

要移除数组的最后一个元素，你可以使用 `pop()` 方法。`pop()` 方法从数组中移除最后一个元素并返回该元素。

例如，要从 `pet` 数组中移除最后一个元素，你可以使用以下代码：

```javascript
const pet = ['cat', 'dog', 'tiger'];
pet.pop();
console.log(pet); // Output: ['cat', 'dog']
```
