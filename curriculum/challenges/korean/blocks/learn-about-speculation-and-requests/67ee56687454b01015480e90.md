`for loop`은 루프가 몇 번 실행되어야 하는지 알 때 유용합니다. 예를 들어:

- `A for loop is useful for going through a list.` - 리스트의 각 항목을 확인할 때 사용할 수 있습니다.

- `The for loop stops when it reaches the last item.` - 더 이상 항목이 없으면 루프가 종료됩니다.

`while loop`은 반복 횟수가 조건에 따라 달라질 때 더 적합합니다. 예를 들어:

- `A while loop runs until a condition is false.` - 규칙이 참인 동안 계속 반복합니다.
