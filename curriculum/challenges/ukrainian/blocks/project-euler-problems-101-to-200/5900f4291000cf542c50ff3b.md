# --description--

Гіперекспонентація або тетрація числа $a$ за додатним цілим числом $b$, позначена як $a↑↑b$ або ${}^ba$, рекурсивно визначається так:

$a↑↑1 = a$,

$a↑↑(k+1) = a^{(a↑↑k)}$.

Отже, наприклад, $3↑↑2 = 3^3 = 27$, звідси $3↑↑3 = 3^{27} = 7625597484987$, а $3↑↑4$ приблизно дорівнює ${10}^{3.6383346400240996 \times {10}^{12}}$. Знайдіть останні 8 цифр числа $1777↑↑1855$.

# --hints--

```js
assert.strictEqual(hyperexponentation(), 95962097);
```
