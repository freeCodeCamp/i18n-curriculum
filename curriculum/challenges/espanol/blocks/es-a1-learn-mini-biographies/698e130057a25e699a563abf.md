# --description--

Para decir qué le gusta a alguien en español, puedes usar esta estructura:

**`Le gusta` ("A él/ella le gusta") + Lo que le gusta a la persona.**

El pronombre `le` se refiere a "él" o "ella". Deja claro que estás hablando de otra persona.

Para decir que a alguien **no** le gusta algo, debes añadir la palabra `no` antes de `le`.

**`No le gusta` ("A él/ella no le gusta") + Lo que no le gusta a la persona.**

Sebastián está hablando de lo que a Marisol le gusta hacer en su trabajo diario.
