# --explanation--

`Evidence of` significa señales o prueba de que algo ha ocurrido. Por ejemplo:

El informe mostró evidencia de error del usuario.

Un `brute force attack` es cuando alguien intenta muchas contraseñas diferentes hasta que una funciona.

Una `SQL injection` es cuando alguien intenta entrar en un sistema insertando código dañino en un cuadro de búsqueda o formulario.

Jake está hablando sobre cómo verificar signos de estos tipos de ataques.
