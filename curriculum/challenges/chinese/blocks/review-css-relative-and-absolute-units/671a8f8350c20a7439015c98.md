## 绝对单位

- **`px`（像素）**：这种绝对单位是 CSS 中的固定大小测量单位。它是最常见的绝对单位，能够精确控制维度。`1px` 始终等于 1/96 英寸。
- **`in`（英寸）**：该绝对单位等于 96px。
- **`cm`（厘米）**：该绝对单位等于 25.2/64 英寸。
- **`mm`（毫米）**：该绝对单位等于 1/10 厘米。
- **`q`（1/4 毫米）**：该绝对单位等于 1/40 厘米。
- **`pc`（派卡）**：该绝对单位等于 1/6 英寸。
- **`pt`（点）**：该绝对单位等于 1/72 英寸。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="units">
  <div class="unit px">px</div>
  <div class="unit inch">in</div>
  <div class="unit cm">cm</div>
  <div class="unit mm">mm</div>
  <div class="unit q">q</div>
  <div class="unit pc">pc</div>
  <div class="unit pt">pt</div>
</div>
```

```css
.units {
  display: flex;
  gap: 12px;
  flex-wrap: wrap;
  align-items: flex-end;
}

.unit {
  background: steelblue;
  color: white;
  text-align: center;
  padding: 4px;
}

.px {
  width: 40px;
  height: 40px;
}

.inch {
  width: 0.5in;
  height: 0.5in;
}

.cm {
  width: 1cm;
  height: 1cm;
}

.mm {
  width: 10mm;
  height: 10mm;
}

.q {
  width: 40q;
  height: 40q;
}

.pc {
  width: 6pc;
  height: 6pc;
}

.pt {
  width: 36pt;
  height: 36pt;
}
```

## 相对单位

- **百分比**：使用这些相对单位，可以将大小、尺寸和其他属性定义为其父元素的一定比例。 例如，如果在一个元素上设置 `width: 50%;` ，它将占据其父容器宽度的一半。
- **`em` 单位**：该单位是相对于元素的字体大小而言的。 如果你使用 `ems` 作为 `font-size` 属性，文本的大小将相对于父元素的字体大小。
- **`rem` 单位**：该单位是相对于根元素，即 `html` 元素的字体大小而言的。
- **`vh` 单位**：`vh` 代表 `"viewport height"`，`1vh` 等于视口高度的 1%。
- **`vw` 单位**：`vw` 代表 `"viewport width"`，`1vw` 等于视口宽度的 1%。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  <div class="box percent">50%</div>
  <div class="box em">2em</div>
  <div class="box rem">2rem</div>
  <div class="box vh">10vh</div>
  <div class="box vw">10vw</div>
</div>
```

```css
html {
  font-size: 16px;
}

.parent {
  width: 200px;
  font-size: 20px;
  border: 2px dashed #555;
  padding: 10px;
  display: flex;
  flex-direction: column;
  gap: 12px;
}

.box {
  background: seagreen;
  color: white;
  text-align: center;
  padding: 6px;
}

.percent {
  width: 50%;
}

.em {
  font-size: 2em;
}

.rem {
  font-size: 2rem;
}

.vh {
  height: 10vh;
}

.vw {
  width: 10vw;
}
```

## `calc` 函数

- **`calc()` 函数**：使用 `calc()` 函数，你可以直接在样式表中执行计算，动态确定属性值。 这意味着，你可以根据视口尺寸或其他元素计算尺寸，创建灵活的响应式用户界面。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="calc-box">calc()</div>
```

```css
.calc-box {
  width: calc(100% - 40px);
  padding: 10px;
  background: steelblue;
  color: white;
  text-align: center;
  border: 2px solid black;
}
```
