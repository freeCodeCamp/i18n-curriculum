# --description--

프로젝트를 진행하면서 여러 가지 해결책을 시도해 왔다고 상상해 보세요. 이런 지속적인 노력을 표현하려면 현재완료 시제를 사용할 수 있습니다. 이 시제는 과거의 행동을 현재와 연결합니다.

현재완료 시제는 조동사 `have` 또는 `has`와 주동사의 과거분사 형태를 사용해 만듭니다. 예를 들어 `I have been working on this for a long time` 또는 `he has worked on this for a long time`가 있습니다.

과거에 시작되어 지금도 관련 있는 행동에 대해 현재완료 시제를 사용해 연습해 보세요.

## --text--

### --feedback--
