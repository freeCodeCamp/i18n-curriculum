# --description--

Wie Sie in früheren Modulen gelernt haben, werden Schleifen verwendet, um einen Codeblock eine festgelegte Anzahl von Malen zu wiederholen. In dieser Lektion lernen Sie, wie Sie mit verschiedenen Arten von Schleifen in Python arbeiten.

Die erste Schleife, die wir besprechen, ist die `for`-Schleife. Hier ist ein Beispiel für die Verwendung einer `for`-Schleife, um eine Liste zu durchlaufen und jedes Element in der Konsole auszugeben:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)
```

Das Ergebnis wäre:

```md
Rust
Java
Python
C++
```

Beachten Sie, dass die `print(language)` innerhalb der Schleife eingerückt ist. Ohne diese Einrückung würden Sie einen `IndentationError` erhalten:

```python
"""
Traceback (most recent call last):
  File "<stdin>", line 4, in <module>
IndentationError: expected an indented block after 'for' statement on line 3
"""
```

Sie können auch eine `for`-Schleife verwenden, um andere iterierbare Objekte wie einen String zu durchlaufen. Hier ist ein Beispiel für die Verwendung einer `for`-Schleife, um den String `code` zu durchlaufen und jedes Zeichen auszugeben:

```py
for char in 'code':
    print(char)
```

Das Ergebnis wäre:

```md
c
o
d
e
```

Genau wie in JavaScript können Sie in Python auch `for`-Schleifen verschachteln. Hier ist ein Beispiel für die Verwendung einer verschachtelten `for`-Schleife:

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)
```

Die äußere Schleife durchläuft jedes `category` in der `categories`-Liste. Für jedes `category` durchläuft die innere Schleife jedes `food` in der `foods`-Liste. Hier ist das Ergebnis, das in der Konsole ausgegeben wird:

```md
Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
```

Eine weitere Art von Schleife, die Sie in Python verwenden können, ist die `while`-Schleife. Diese Art von Schleife wiederholt einen Codeblock, bis die Bedingung `False` ist. Hier ist ein Beispiel für die Verwendung einer `while`-Schleife für ein Ratespiel:

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')
```

In diesem Beispiel haben wir eine `secret_number`-Variable mit dem Wert `3` und eine Anfangsvermutung von `0`. Dann verwenden wir die `input`-Funktion, um eine Eingabe vom Benutzer zu erhalten, wandeln den Eingabe-String mit der `int()`-Funktion in eine Ganzzahl um und weisen ihn der `guess`-Variable zu. Wenn der Benutzer richtig rät, indem er `3` eingibt, wird die `while`-Schleife abgebrochen und die Nachricht `You got it!` in der Konsole ausgegeben. Andernfalls wird die Nachricht `Wrong! Try again.` in der Konsole ausgegeben, und die Schleife wiederholt sich, um den Benutzer erneut zum Raten aufzufordern.

Hier ist ein Beispielergebnis:

```md
Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
```

Genau wie in JavaScript unterstützt Python die `break`- und `continue`-Anweisungen.

Die `break`-Anweisung wird verwendet, um die Ausführung einer Schleife zu beenden. Hier ist ein Beispiel für die Verwendung der `break`-Anweisung für eine Liste von `developer_names`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

In diesem Beispiel durchlaufen wir eine Liste von `developer_names` und geben jeden Namen in der Konsole aus. Wenn der Name gleich `Naomi` ist, wird die Schleife abgebrochen. Dadurch wird nur der Name `Jess` in der Konsole ausgegeben.

Die `continue`-Anweisung wird verwendet, um die aktuelle Iteration einer Schleife zu überspringen und mit der nächsten Iteration fortzufahren. Lassen Sie uns das vorherige Beispiel so ändern, dass die `continue`-Anweisung anstelle von `break` verwendet wird:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

Jetzt ist das Ergebnis in der Konsole anders. Die Namen `Jess` und `Tom` werden ausgegeben, weil die `continue`-Anweisung die zweite Iteration der Schleife überspringt, wenn `developer` gleich `Naomi` ist, und diesen Namen nicht in der Konsole ausgibt.

Sowohl `for`- als auch `while`-Schleifen können mit einer `else`-Klausel kombiniert werden, die nur ausgeführt wird, wenn die Schleife nicht durch eine `break`-Anweisung beendet wurde. Hier ist ein Beispiel für die Verwendung mehrerer `for`-Schleifen:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

In diesem Beispiel haben wir eine Liste zufälliger Wörter, und eine `for`-Schleife wird verwendet, um jedes Wort zu durchlaufen. Innerhalb der äußeren `for`-Schleife haben wir eine weitere `for`-Schleife, um jeden Buchstaben jedes Wortes zu durchlaufen. Wenn die Kleinbuchstaben-Version des Buchstabens ein Vokal ist, geben wir das Wort gefolgt von den enthaltenen Vokalen aus und brechen die innere Schleife ab. Wenn das Wort keine Vokale enthält, geben wir eine Nachricht aus, die darauf hinweist.

So sieht das Ergebnis in der Konsole aus:

```md
'sky' has no vowels
'apple' contains the vowel 'a'
'rhythm' has no vowels
'fly' has no vowels
'orange' contains the vowel 'o'
```

Schleifen sind in Python sehr häufig, daher ist es wichtig, sich mit ihnen vertraut zu machen. In den nächsten Lektionen lernen Sie, wie Sie mit den `enumerate()`- und `range()`-Funktionen in Schleifen arbeiten.
