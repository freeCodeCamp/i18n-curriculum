`Code reuse` inahusu utaratibu wa kutumia tena msimbo uliopo katika sehemu mpya za programu. Kwa mfano:

`The "Login" class was used in multiple applications to handle user access, which is a good example of code reuse.` - Badala ya kuandika mantiki ya kuingia (login) kwa kila programu, msimbo uleule unatumika tena, hivyo kuboresha ufanisi.

`Extending` inamaanisha kuongeza vipengele vipya kwa kitu kilichopo tayari. Kwa mfano:

`The "AdminUser" class extends the basic "User" class by adding an "approveUser()" method.` - `AdminUser` huhifadhi kila kitu kutoka `User` na hujenga juu yake kwa uwezo mpya.
