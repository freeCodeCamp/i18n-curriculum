# --description--

숫자 512는 각 자리수를 어떤 거듭제곱으로 올린 값들의 합과 같아서 흥미롭습니다: $5 + 1 + 2 = 8$, 그리고 $8^3 = 512$입니다. 이 성질을 가진 또 다른 숫자는 $614656 = 28^4$입니다.

우리는 $a_n$을 이 수열의 $n$번째 항으로 정의하며, 숫자는 합을 가지려면 최소 두 자리 숫자여야 한다고 가정합니다.

$a_2 = 512$이고 $a_{10} = 614656$임이 주어졌습니다.

$a_{30}$을 찾으세요.
