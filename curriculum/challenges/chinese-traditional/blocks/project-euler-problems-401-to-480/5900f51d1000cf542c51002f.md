# --description--

令 $E(x_0, y_0)$ 為使用歐幾里得演算法判定 $x_0$ 和 $y_0$ 最大公因數所需的步驟數。更正式地說：

$$\begin{align}
  & x_1 = y_0, y_1 = x_0\bmod y_0 \\\\
  & x_n = y_{n - 1}, y_n = x_{n - 1}\bmod y_{n - 1}
\end{align}$$

$E(x_0, y_0)$ 是最小的 $n$，使得 $y_n = 0$。

我們有 $E(1, 1) = 1$、$E(10, 6) = 3$ 和 $E(6, 10) = 4$。

定義 $S(N)$ 為 $1 ≤ x$、$y ≤ N$ 時 $E(x, y)$ 的總和。

我們有 $S(1) = 1$、$S(10) = 221$ 和 $S(100) = 39\\,826$。

尋找 $S(5 \times {10}^6)$。
