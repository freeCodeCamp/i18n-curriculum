# --description--

Остання тема — це страшенний нескінченний цикл. Цикли — чудовий інструмент, коли потрібно, щоб ваша програма виконала блок коду певну кількість разів або доки не буде виконана умова, але їм потрібна умова завершення, яка припиняє цикл. Нескінченні цикли можуть зависати або викликати збій браузера, а також спричиняти хаос у виконанні програми, чого ніхто не хоче.

У вступі до цього розділу був приклад нескінченного циклу — у нього не було умови завершення, щоб вийти з циклу `while` всередині `loopy()`. НЕ викликайте цю функцію!

```js
function loopy() {
  while(true) {
    console.log("Hello, world!");
  }
}
```

Завдання програміста — переконатися, що умова завершення, яка повідомляє програмі, коли припинити виконання циклу, зрештою буде досягнута. Однією з помилок є збільшення або зменшення змінної-фішки у неправильному напрямку відносно умови завершення. Іншою помилкою є випадкове скидання змінної-фішки або індексу всередині коду циклу замість її збільшення або зменшення.

# --instructions--

Функція `myFunc()` містить нескінченний цикл, бо умова завершення `i != 4` ніколи не стане `false` (і не припинить цикл) — `i` збільшуватиметься на 2 щоразу, і пропускатиме 4, оскільки `i` спочатку непарне. Виправте оператор порівняння в умові завершення, щоб цикл виконувався лише для `i`, що менше або дорівнює 4.

# --hints--

Ваш код має змінити оператор порівняння в умові завершення (середня частина) циклу `for`.

```js
assert(__helpers.removeJSComments(code).match(/i\s*?<=\s*?4;/g).length == 1);
```

Ваш код має виправити оператор порівняння в умові завершення циклу.

```js
assert(!__helpers.removeJSComments(code).match(/i\s*?!=\s*?4;/g));
```
