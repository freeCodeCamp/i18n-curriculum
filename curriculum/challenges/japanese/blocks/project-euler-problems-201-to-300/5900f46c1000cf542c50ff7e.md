# --description--

畳は長方形のマットで、重ならないように部屋の床全体を覆うために使われます。

利用可能な畳の種類が1×2の寸法だけだとすると、畳で覆える部屋の形や大きさには明らかに制限があります。

この問題では、整数の寸法 $a$, $b$ を持ち、面積 $s = a \times b$ が偶数の長方形の部屋のみを考えます。ここで「面積」とは部屋の床の表面積を指し、一般性を失わずに $a ≤ b$ の条件を加えます。

畳を敷く際には1つのルールがあります：4枚の異なる畳の角が1点で交わる場所があってはいけません。例えば、4×4の部屋に対する以下の2つの敷き方を考えてみましょう。

<img alt="two arrangements of mats in 4x4 room" src="https://cdn.freecodecamp.org/curriculum/project-euler/tatami-free-rooms.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

左の敷き方は許容されますが、右の敷き方は許されません。中央の赤い「<strong><span style="color: red;">X</span></strong>」は4枚の畳が交わる点を示しています。

このルールのため、特定の偶数面積の部屋は畳で覆うことができません。これらを「畳なしの部屋」と呼びます。さらに、$T(s)$ を面積 $s$ の畳なしの部屋の数と定義します。

最小の畳なしの部屋は面積 $s = 70$、寸法は7×10です。面積 $s = 70$ の他の部屋はすべて畳で覆うことができ、それらは1×70、2×35、5×14です。したがって、$T(70) = 1$ となります。

同様に、$T(1320) = 5$ であることも確認できます。面積 $s = 1320$ の畳なしの部屋は正確に5つあり、それらは20×66、22×60、24×55、30×44、33×40です。実際、$s = 1320$ は $T(s) = 5$ となる最小の面積です。

$T(s) = 200$ となる最小の面積 $s$ を求めてください。
