# --explanation--

`Clarifying` 是以消除混淆的方式解釋某事。它幫助某人更好地理解一個概念或情況。 例如：

`Thanks for clarifying how the API works.` - 說話者感謝，因為說明讓 API 更容易理解。

`Abstraction` 是指透過只顯示必要的特性並隱藏複雜的內部運作來簡化某事物。例如：

`Using a login function is an example of abstraction - you don't need to know how the password is checked, just that it works.`－這顯示了使用者如何與簡單的介面互動，而技術邏輯則保持隱藏。
