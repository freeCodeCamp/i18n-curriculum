# --description--

Tatizo la Josephus ni fumbo la hisabati lenye maelezo mabaya: wafungwa $n$ wamesimama mduara, wakiwa wamepangwa kwa mfululizo kutoka $0$ hadi $n-1$.

Mtekelezaji hukimbia mduara huo, kuanzia kwa mfungwa nambari $0$, akiondoa kila mfungwa wa $k$ na kumuua.

Mchakato huu unaendelea, mduara unakuwa mdogo zaidi na zaidi, hadi mfungwa mmoja tu abaki, ambaye kisha hurejeshwa uhuru.

Kwa mfano, ikiwa kuna wafungwa $n=5$ na $k=2$, mfululizo wa wafungwa waliouawa (tuuitie "mfululizo wa kuua") utakuwa 1, 3, 0, na 4, na yule aliyeishi atakuwa #2.

Ikiwa $n, k > 0$ yoyote, gundua ni mfungwa gani atakayekuwa mwishowe mmojawapo.

Katika tukio moja, kulikuwa na wafungwa 41 na kila mfungwa wa tatu alikuwa akiuwawa ($k=3$).

Miongoni mwao alikuwa jamaa mwerevu aitwaye Josephus ambaye alitatua tatizo hilo, akasimama katika nafasi ya kuishi, na akaishi kuendelea kusimulia hadithi.

Nambari gani alikuwa yeye?

# --instructions--

Andika kitendakazi kinachochukua idadi ya awali ya wafungwa na `k` kama vigezo na kurudisha nambari ya mfungwa anayeelekea kuishi.

function josephus(init, kill) {
  const arr = Array.from(Array(init).keys());
  let curr = -1
  while (arr.length > 1) {
    curr = (curr + kill) % arr.length;
    arr.splice(curr, 1);
    curr--;
  }
  return arr[0];
}
