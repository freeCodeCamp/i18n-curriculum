# --interactive--

Wenn Sie mit Zahlen arbeiten und einen Wert um eins erhöhen oder verringern müssen, machen Ihnen die Inkrement- und Dekrementoperatoren die Arbeit leichter. Lassen Sie uns das einfach erklären.

Die Inkrement- und Dekrementoperatoren werden jeweils durch `++` und `--` dargestellt. Beide erlauben es Ihnen, den Wert einer Variablen um `1` anzupassen.

Anstatt etwas wie `x = x + 1` oder `x = x - 1` zu schreiben, können Sie einfach `x++` verwenden, um `1` hinzuzufügen, oder `x--`, um `1` zu subtrahieren. Das ist schneller, sauberer und leichter zu lesen.

Es gibt einen Unterschied darin, wie die Inkrement- und Dekrementoperatoren funktionieren: Sie kommen in zwei Formen vor, Präfix und Postfix, wobei der Unterschied darin besteht, wann der Wert aktualisiert wird. Beim Inkrementoperator ist Präfix `++x` und Postfix `x++`.

Präfix (`++x`) erhöht zuerst den Wert der Variablen und gibt dann den neuen Wert zurück. Postfix (`x++`) gibt zuerst den aktuellen Wert der Variablen zurück und erhöht ihn dann:

:::interactive_editor

```js
let x = 5;

console.log(++x); // 6
console.log(x); // 6
```

:::

Im obigen Code bedeutet `++x` „erhöhe `x` zuerst, dann verwende ihn“. Wenn Sie also `++x` ausgeben, erhalten Sie sofort den erhöhten Wert, der `6` ist.

Schauen wir uns nun ein Beispiel mit dem Postfix an:

:::interactive_editor

```js
let y = 5;

console.log(y++); // 5
console.log(y); // 6
```

:::

In diesem Beispiel bedeutet `y++` „verwende `y` zuerst, dann erhöhe ihn“. Wenn Sie `y++` ausgeben, erhalten Sie `5`, aber `y` wird nach dieser Codezeile zu `6`.

Der Dekrementoperator funktioniert genauso wie der Inkrementoperator, nur dass er den Wert um `1` verringert. Auch hier gibt es zwei Formen: Präfix (`--x`) verringert zuerst den Wert der Variablen und gibt dann den neuen Wert zurück. Und Postfix (`x--`) gibt zuerst den aktuellen Wert zurück und verringert ihn dann:

:::interactive_editor

```js
let x = 5;
console.log(--x); // 4
console.log(x); // 4

let y = 5;
console.log(y--); // 5
console.log(y); // 4
```

:::

Welche Form sollten Sie also verwenden: Präfix oder Postfix? In vielen Fällen ist es egal, welche Sie verwenden. Beide erledigen die Aufgabe. Wenn Sie den Wert jedoch sofort in einem Ausdruck verwenden, wird der Unterschied wichtig. Schauen wir uns dieses Beispiel an:

:::interactive_editor

```js
let a = 5;
let b = ++a;
console.log(b); // 6 (a was incremented before assignment)

let c = 5;
let d = c++;
console.log(d); // 5 (c was incremented after assignment)
```

:::

Wenn Sie also den aktualisierten Wert sofort benötigen, verwenden Sie Präfix. Wenn Sie zuerst den aktuellen Wert wollen und die Erhöhung erst später wichtig ist, wählen Sie Postfix.
