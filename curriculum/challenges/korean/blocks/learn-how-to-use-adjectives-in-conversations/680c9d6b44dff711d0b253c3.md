# --explanation--

`Actually`는 실제 진실을 보여주거나 오해를 바로잡을 때 사용합니다. 예를 들어:

`I actually finished the project early.` – 예상보다 일찍 끝냈다는 뜻입니다.

`Quite a few`는 단순히 몇 개보다 더 많은 수를 의미합니다. 예를 들어:

`There were quite a few people at the meeting.` – 많은 사람이 참석했다는 뜻입니다.

`Those`는 가까이 있지 않은 특정 항목을 가리키는 지시사입니다. 예를 들어:

`Those are my files over there.` – 더 멀리 있는 파일을 가리킵니다.
