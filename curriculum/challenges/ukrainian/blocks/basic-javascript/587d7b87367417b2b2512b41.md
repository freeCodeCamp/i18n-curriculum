# --description--

Ключове слово `let` — не єдиний новий спосіб оголошення змінних. У ES6 ви також можете оголошувати змінні за допомогою ключового слова `const`.

`const` має всі чудові функції, які має `let`, з додатковою перевагою, що змінні, оголошені за допомогою `const`, є лише для читання. Вони мають константне значення, що означає: після того, як змінній призначено значення за допомогою `const`, його не можна змінити:

```js
const FAV_PET = "Cats";
FAV_PET = "Dogs";
```

У консолі з’явиться помилка через повторне призначення значення `FAV_PET`.

Ви завжди повинні називати змінні, які не хочете змінювати, використовуючи ключове слово `const`. Це допомагає уникнути випадкового повторного призначення змінної, яка має залишатися константою.

**Примітка:** Розробники часто використовують ідентифікатори змінних у верхньому регістрі для незмінних значень, а у нижньому регістрі або верблюжому регістрі — для змінних значень (об’єктів і масивів). Ви дізнаєтеся більше про об’єкти, масиви, а також про незмінні та змінні значення у наступних завданнях. Також у наступних завданнях ви побачите приклади ідентифікаторів змінних у верхньому, нижньому або верблюжому регістрі.

# --instructions--

Змініть код так, щоб усі змінні були оголошені за допомогою `let` або `const`. Використовуйте `let`, коли змінна має змінюватися, і `const`, коли змінна має залишатися константою. Також перейменуйте змінні, оголошені за допомогою `const`, відповідно до загальних практик. Не змінюйте рядки, призначені змінним.

# --hints--

`var` не має існувати у вашому коді.

```js
assert.notMatch(code, /var/g);
```

Ви повинні змінити `fCC` на всі великі літери.

```js
assert.match(code, /(FCC)/);
assert.notMatch(code, /(fCC)/);
```

`FCC` має бути константною змінною, оголошеною за допомогою `const`.

```js

```

Рядок, призначений `FCC`, не слід змінювати.

```js
assert.equal(FCC, 'freeCodeCamp');
```

`fact` має бути оголошений за допомогою `let`.

```js
assert.match(code, /(let\s+fact)/g);
```

`console.log` слід змінити, щоб виводити змінні `FCC` і `fact`.

```js
assert.match(code, /console\.log\(\s*FCC\s*\,\s*fact\s*\)\s*;?/g);
```
