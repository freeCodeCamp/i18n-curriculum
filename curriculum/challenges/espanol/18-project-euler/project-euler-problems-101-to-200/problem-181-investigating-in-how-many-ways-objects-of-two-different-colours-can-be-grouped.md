# --description--

Teniendo tres objetos negros $B$ y un objeto blanco $W$ se pueden agrupar de 7 maneras así:

$$(BBBW)\\;(B,BBW)\\;(B,B,BW)\\;(B,B,B,W)\\;(B,BB,W)\\;(BBB,W)\\;(BB,BW)$$

¿De cuántas maneras se pueden agrupar sesenta objetos negros $B$ y cuarenta objetos blancos $W$ de esta forma?
