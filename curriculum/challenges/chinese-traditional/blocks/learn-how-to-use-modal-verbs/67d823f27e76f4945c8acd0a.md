# --explanation--

`Should` 是一個情態動詞，用來表達建議、推薦或最佳做法——但不是義務。

- `You should update your password regularly。`－為了安全起見，經常更改你的密碼是個好主意。

- `You should check for errors in the code before running it。`－在執行程式碼之前檢查錯誤是個好習慣。

既然 Jake 說 `we should focus on the critical systems first`，他是在給出強烈建議，但團隊並沒有嚴格義務去執行。
