# --interactive--

이전 수업에서 선형 그래디언트와 방사형 그래디언트를 사용하는 방법을 배웠습니다. 이번 수업에서는 CSS 속성을 활용해 더 시각적으로 매력적이고 복잡한 디자인을 만드는 방법을 자세히 살펴보겠습니다.

CSS의 그래디언트는 두 가지 이상의 지정된 색상 사이에 부드러운 전환을 만들 수 있게 해줍니다. 이미지 파일 없이도 웹 디자인에 깊이감, 질감, 시각적 흥미를 더할 때 자주 사용됩니다.

CSS는 두 가지 주요 그래디언트 유형을 지원합니다: 선형 그래디언트와 방사형 그래디언트입니다.

선형 그래디언트는 직선 방향을 따라 색상이 점진적으로 섞이도록 만듭니다. 이 선의 방향과 사용되는 색상을 제어할 수 있습니다. 다음은 선형 그래디언트의 기본 예시입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

이 코드는 왼쪽에서 빨간색, 오른쪽에서 파란색으로 전환되는 그래디언트를 만듭니다. `to right` 매개변수는 그래디언트의 방향을 지정합니다. `to left`, `to top`, `to bottom` 같은 다른 방향을 사용하거나 `45deg` 같은 각도를 지정할 수도 있습니다.

선형 그래디언트에서는 여러 색상 정지점도 사용할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="multi-color-gradient"></div>
```

```css
.multi-color-gradient {
  background: linear-gradient(to right, red, yellow, blue);
  height: 40vh;
}
```

:::

이 코드는 빨간색에서 노란색, 다시 파란색으로 고르게 분포된 그래디언트를 만듭니다.

반면 방사형 그래디언트는 중심점에서 원형 또는 타원형으로 퍼지는 그래디언트를 만듭니다. 예시는 다음과 같습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::

이 코드는 중앙에서 빨간색으로 시작해 가장자리에서 파란색으로 전환되는 원형 그래디언트를 만듭니다. 방사형 그래디언트의 모양과 크기도 지정할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="custom-radial-gradient"></div>
```

```css
.custom-radial-gradient {
  background: radial-gradient(ellipse at top left, red, blue);
  height: 40vh;
}
```

:::

이 코드는 요소의 왼쪽 위 모서리에서 시작하는 타원형 그래디언트를 만듭니다. 선형과 방사형 그래디언트 모두 정확한 위치에 색상 정지점을 지정할 수 있습니다. 예를 들어:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="precise-gradient"></div>
```

```css
.precise-gradient {
  background: linear-gradient(to right, red 20%, yellow 40%, blue 80%);
  height: 40vh;
}
```

:::

이 코드는 빨간색이 처음 `20%`를 차지하고, `40%`에서 노란색으로, 마지막으로 `80%`에서 파란색으로 전환되는 그래디언트를 만듭니다.

그래디언트는 CSS에서 이미지 대신 어디서든 사용할 수 있습니다. 이로 인해 배경, 버튼, 또는 다른 CSS 속성과 결합해 텍스트 효과를 만드는 데도 유용합니다.

그래디언트가 웹사이트의 시각적 매력을 높여주지만, 특히 텍스트 뒤에 그래디언트를 사용할 때는 접근성을 위해 충분한 대비를 유지하는 것이 중요합니다.
