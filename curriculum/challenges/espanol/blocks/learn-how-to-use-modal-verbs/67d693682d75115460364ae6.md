# --explanation--

Un `critical bug` es un error grave en el software que afecta la funcionalidad y debe ser corregido de inmediato. Por ejemplo:

- `We need to fix the critical bugs before launching the app.` - Los problemas más graves deben ser solucionados antes del lanzamiento.

- `A critical bug caused the website to crash.` - Un error grave hizo que el sitio web dejara de funcionar.
