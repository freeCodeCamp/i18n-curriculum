# --description--

從 26 個字母中取出三個不同的字母，可以形成長度為三的字串。

範例是 'abc'、'hat' 和 'zyx'。

當我們研究這三個範例時，我們看到對於 'abc'，有兩個字元在字典序上位於其左側鄰居之後。

對於「hat」，恰好有一個字元在字典序上位於其左側鄰居之後。對於「zyx」，沒有字元在字典序上位於其左側鄰居之後。

總共有 10400 個長度為 3 的字串，其中恰好有一個字元在字典序上位於其左側鄰居之後。

我們現在考慮由字母表中不超過 26 個不同字元組成的長度為 n 的字串。

對於每個 $n$，$p(n)$ 是長度為 $n$ 的字串數量，其中恰好有一個字元在字典序上位於其左側鄰居之後。

$p(n)$ 的最大值是多少？
