# --description--

Reactのコードに強力な機能を追加する他の複雑な概念もあります。しかし、Reactで作成したJSX要素にどうやってスタイルを適用するかという、よりシンプルな問題について疑問に思っているかもしれません。JSX要素にクラスを適用する方法が<a href="/learn/front-end-development-libraries/react/define-an-html-class-in-jsx" target="_blank" rel="noopener noreferrer nofollow">HTMLとは少し違うことはご存じでしょう</a>。

スタイルシートからスタイルをインポートする場合は、ほとんど違いはありません。JSX要素に`className`属性を使ってクラスを適用し、スタイルシートでそのクラスにスタイルを設定します。もう一つの方法は、ReactJS開発で非常に一般的なインラインスタイルを適用することです。

インラインスタイルはHTMLでのやり方に似ていますが、JSXにはいくつか違いがあります。HTMLでのインラインスタイルの例を示します。

```jsx
<div style="color: yellow; font-size: 16px">Mellow Yellow</div>
```

JSX要素では`style`属性を使いますが、JSXがトランスパイルされる仕組みのため、値を`string`に設定することはできません。代わりにJavaScriptの`object`に等しく設定します。例を示します。

```jsx
<div style={{color: "yellow", fontSize: 16}}>Mellow Yellow</div>
```

`fontSize`プロパティをキャメルケースにしているのに気づきましたか？これはReactがスタイルオブジェクト内でケバブケースのキーを受け付けないためです。ReactがHTMLで正しいプロパティ名を適用してくれます。

# --instructions--

コードエディターの`style`に`div`属性を追加して、テキストの色を赤、フォントサイズを`72px`にしてください。

フォントサイズは単位の`px`を省略して数値で設定するか、`"72px"`として書くこともできます。
# --hints--

コンポーネントは`div`要素をレンダリングする必要があります。

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(Colorful));
    return mockedComponent.children().type() === 'div';
  })()
);
```

`div`要素の色は`red`であるべきです。

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(Colorful));
    return mockedComponent.children().props().style.color === 'red';
  })()
);
```

`div`要素のフォントサイズは`72px`であるべきです。

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(Colorful));
    return (
      mockedComponent.children().props().style.fontSize === 72 ||
      mockedComponent.children().props().style.fontSize === '72' ||
      mockedComponent.children().props().style.fontSize === '72px'
    );
  })()
);
```
