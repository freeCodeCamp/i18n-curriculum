# --description--

每种编程语言都有某种方式通过内置的方法、函数、属性或关键字将数据输出到终端。在 Python 中，你可以使用 `print` 函数将数据打印到终端。让我们仔细看看 `print` 函数，这样你就可以自信地开始使用它。

学习任何编程语言时你做的第一件事之一就是编写一个简单的 `Hello world!` 程序。你可以用 Python 仅用 `print` 函数非常轻松地做到这一点。

为此，你只需将字串 `Hello world!` 放在你调用 `print` 函数时使用的开括号和闭括号之间：

```python
print('Hello world!') # Hello world!
```

你将在以后的课程中学习更多关于 Python 中的字串和函数。现在，只需将字串视为由单引号（`'`）或双引号（`"`）包围的一系列字符。

在 `print('Hello world!')` 示例中，字串 `'Hello world!'` 是传递给 `print` 函数的一个**参数**。你也可以使用 `print` 函数通过逗号分隔一次显示多值或多个参数。例如：

```python
print('My favorite colors are', 'blue', 'green', 'red')

# Output: My favorite colors are blue green red
```

当你用逗号分隔每个项时，Python 会自动在它们之间添加一个空格。这在你想要一起打印多条信息时非常有用。
