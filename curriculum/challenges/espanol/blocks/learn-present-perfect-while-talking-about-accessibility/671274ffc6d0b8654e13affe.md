# --explanation--

El tiempo `Present Perfect Continuous` se usa para describir acciones que comenzaron en el pasado y aún están ocurriendo. Se enfoca en la duración o continuación de la acción. La estructura de este tiempo es `have/has + been + verb(-ing)`. Por ejemplo:

- Hemos estado desarrollando una nueva app para accesibilidad. - Esto significa que el desarrollo comenzó en el pasado y aún continúa.

- `He has been fixing bugs in the software for several hours.` - Esto significa que comenzó a arreglar los errores antes y todavía está trabajando en ello.
