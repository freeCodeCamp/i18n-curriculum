# --description--

Nesta tarefa, você ouvirá a reação de Ángela ao descobrir o país de origem de Esteban. Preste atenção a essas expressões:

- `Ah` é um som natural de reação. Ele demonstra compreensão ou percepção.

- `de` é usado para falar sobre origem.

- `Muy bien` é uma reação positiva. Indica aprovação ou satisfação.

# --instructions--

Ouça o áudio e decida se a seguinte declaração é verdadeira ou falsa.
