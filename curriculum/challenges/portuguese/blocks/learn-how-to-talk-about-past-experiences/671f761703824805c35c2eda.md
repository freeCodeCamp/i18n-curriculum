# --explanation--

A frase criada usando `get` seguida de um adjetivo descreve uma mudança de estado ou o ato de se tornar de uma certa maneira.Por exemplo:

- `They always get excited when a new Python update is released.` - `Get excited` significa ficar entusiasmado ou empolgado. Isso significa que eles estão sempre cheios de entusiasmo quando uma nova atualização do Python é lançada.

- `He got tired after working on the project all night.` - Isso significa que ele não estava cansado antes, mas está cansado agora, depois de uma noite inteira trabalhando no projeto. 

Na frase de Maria, ela está perguntando se Brian se lembra do entusiasmo que eles costumavam sentir sobre novas tecnologias enquanto trabalhavam juntos.
