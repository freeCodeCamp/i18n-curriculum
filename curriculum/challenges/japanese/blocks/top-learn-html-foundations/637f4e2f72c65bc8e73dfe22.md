# --description--

`<head>` 要素は、ウェブページに関する重要なメタ情報や、ブラウザーで正しくウェブページをレンダリングするために必要なものを配置する場所です。`<head>` の中には、ウェブページ上に表示されるコンテンツを表示する要素を使ってはいけません。

## 文字セットのメタ要素
ウェブページの `meta` エンコーディングのために、`charset` 要素内に常に `head` タグを入れる必要があります：`<meta charset="utf-8">`。

エンコーディングを設定することは非常に重要です。なぜなら、これによってブラウザーで特殊な記号や異なる言語の文字が正しく表示されるようになるからです。

## タイトル要素
HTMLドキュメントのヘッダーに常に含めるべきもう一つの要素は `title` 要素です：

```html
<title>My First Webpage</title>
```

`title` 要素は、ウェブページに人間が読めるタイトルを付けるために使われ、ウェブページのブラウザータブに表示されます。

もし `title` 要素を含めなければ、ウェブページのタイトルはファイル名がデフォルトになります。あなたの場合は `index.html` となり、ユーザーにとってあまり意味のないものになります。多くのブラウザータブを開いている場合、ウェブページを見つけるのが非常に難しくなります。

HTMLドキュメントの `head` 内には他にも多くの要素を入れることができます。しかし、今のところここで扱った2つの要素を知っておくことが重要です。カリキュラムの残りの部分で `head` に入る他の要素を紹介していきます。

`index.html` ファイルに戻って、`head` 要素の中に `charset` `meta` 要素と `title` を追加しましょう。head 要素は HTML 要素の中に入り、開く `<html>` タグのすぐ下に常に最初の要素として置かれます：


```html
<!DOCTYPE html>

<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Webpage</title>
  </head>
</html>
```

`head` 要素の目的は何ですか？

## --answers--

`head` 要素は、ウェブページ上に表示されるすべての要素を表示するために使われます。

---

`head` 要素は、ウェブページに関する重要な情報を表示し、`meta` 要素でウェブページを正しくレンダリングするために使われます。

---

`head` 要素は、ウェブページの上部にヘッダーコンテンツを表示するために使われます。
