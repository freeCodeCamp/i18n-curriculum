# --description--

이전 수업에서는 복잡한 데이터를 명확하고 시각적으로 나타내는 과정인 데이터 시각화에 대해 배웠습니다. 이번 수업에서는 D3 라이브러리에 대해 알아보겠습니다.

D3 라이브러리는 JavaScript로 매력적이고 효과적인 데이터 시각화를 만들 때 사용할 수 있는 무료 오픈 소스 라이브러리입니다.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-1.png" alt="미국의 카운티별 데이터를 보여주는 녹색 색조 지도, 색상 범위는 연한 녹색(3%)에서 진한 녹색(60% 이상)까지">

D3라는 이름은 데이터 기반 문서(data-driven documents)의 약자로, 문서 객체 모델(DOM)의 요소에 데이터를 동적으로 바인딩하는 방식을 나타냅니다. 이 라이브러리는 유연하며 웹과 자연스럽게 작동합니다. 완전히 새로운 그래픽 표현 방식을 배울 필요 없이 `svg`와 `canvas` 요소로 데이터 시각화를 만들 수 있습니다.

예를 들어, D3를 사용해 막대 그래프를 만들고 싶다면 하나의 JS 파일에서 다음과 같이 할 수 있습니다:

* 차트 크기와 여백을 나타내는 변수를 선언하세요
    
* 가로 위치 스케일을 선언하세요
    
* 세로 위치 스케일을 선언하세요
    
* 막대와 레이블이 포함된 SVG 컨테이너를 만드세요
    
<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-2.png" alt="1947년부터 2015년까지 미국 국내총생산을 나타내는 막대 그래프, 값이 거의 0에서 약 18,000까지 상승">

그렇다면 Chart.js, Plotly, Tableau 같은 다른 시각화 도구보다 D3를 배워야 하는 이유는 무엇일까요?

다른 도구와 달리 D3는 많은 저수준 제어를 제공하여 순수 HTML, CSS, JavaScript만으로 맞춤형이고 인터랙티브한 시각화를 만들 수 있습니다. 게다가 이 라이브러리는 JavaScript로 되어 있어 완전히 새로운 언어를 배울 필요가 없습니다.

앞으로 진행될 워크숍과 실습에서 선택, 도형, 스케일 같은 D3 라이브러리의 기본 개념을 배우게 됩니다. 산점도, 막대 그래프, 대시보드 앱 등을 만들면서 이 핵심 개념을 익히게 됩니다.
