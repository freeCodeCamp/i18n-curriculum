# --description--

Betrachten Sie die folgende Anordnung von 64 Dreiecken:

<img alt="64 Dreiecke, angeordnet zu einem größeren Dreieck mit einer Seitenlänge von 8 Dreiecken" src="https://cdn.freecodecamp.org/curriculum/project-euler/tri-colouring-a-triangular-grid-1.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Wir möchten das Innere jedes Dreiecks mit einer von drei Farben einfärben: Rot, Grün oder Blau, so dass keine zwei benachbarten Dreiecke dieselbe Farbe haben. Eine solche Färbung wird als gültig bezeichnet. Zwei Dreiecke gelten als benachbart, wenn sie eine Kante teilen. Hinweis: Wenn sie nur einen Eckpunkt teilen, sind sie keine Nachbarn.

Zum Beispiel ist hier eine gültige Färbung des obigen Gitters dargestellt:

<img alt="gefärbtes Gitter aus 64 Dreiecken" src="https://cdn.freecodecamp.org/curriculum/project-euler/tri-colouring-a-triangular-grid-2.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Eine Färbung C', die durch Rotation oder Spiegelung aus einer Färbung C entsteht, gilt als verschieden von C, es sei denn, beide sind identisch.

Wie viele verschiedene gültige Färbungen gibt es für die obige Anordnung?
