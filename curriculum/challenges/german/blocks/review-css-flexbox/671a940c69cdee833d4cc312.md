## Einführung in CSS Flexbox und das Flex-Modell

- **Definition**: CSS Flexbox ist ein eindimensionales Layoutmodell, mit dem Sie Elemente innerhalb eines Containers in Reihen und Spalten anordnen können. 
- **Flex-Modell**: Dieses Modell definiert, wie Flex-Elemente innerhalb eines Flex-Containers angeordnet werden. Jeder Flex-Container hat zwei Achsen: die Hauptachse und die Querachse.

## Die `flex-direction`-Eigenschaft

- **Definition**: Diese Eigenschaft legt die Richtung der Hauptachse fest. Der Standardwert von `flex-direction` ist `row`, wodurch alle Flex-Elemente in derselben Reihe in der Richtung der Standardsprache Ihres Browsers (von links nach rechts oder von rechts nach links) angeordnet werden. 
- **`flex-direction: row-reverse;`**: Dies kehrt die Reihenfolge der Elemente in der Reihe um.
- **`flex-direction: column;`**: Dies richtet die Flex-Elemente vertikal statt horizontal aus.
- **`flex-direction: column-reverse;`**: Dies kehrt die Reihenfolge der Flex-Elemente vertikal um.

:::interactive_editor

```html
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: row;
  gap: 10px;
}

.box {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}
```

:::

## Die `flex-wrap`-Eigenschaft

- **Definition**: Diese Eigenschaft bestimmt, wie Flex-Elemente innerhalb eines Flex-Containers umgebrochen werden, um in den verfügbaren Platz zu passen. `flex-wrap` kann drei mögliche Werte annehmen: `nowrap`, `wrap` und `wrap-reverse`.
- **`flex-wrap: nowrap;`**: Dies ist der Standardwert. Flex-Elemente werden nicht in eine neue Zeile umgebrochen, selbst wenn ihre Breite die Breite des Containers überschreitet.
- **`flex-wrap: wrap;`**: Diese Eigenschaft bewirkt, dass die Elemente umgebrochen werden, wenn sie die Breite ihres Containers überschreiten.
- **`flex-wrap: wrap-reverse;`**: Diese Eigenschaft bricht Flex-Elemente in umgekehrter Reihenfolge um.
- **`flex-flow`-Eigenschaft**: Diese Eigenschaft ist eine Kurzschreibweise-Eigenschaft für `flex-direction` und `flex-wrap`.

:::interactive_editor

```html
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
  <div class="box">4</div>
  <div class="box">5</div>
</div>
```

```css
.container {
  display: flex;
  flex-wrap: wrap;
  width: 150px;
  background: #f0f0f0;
}

.box {
  width: 60px;
  padding: 10px;
  margin: 5px;
  background: skyblue;
  text-align: center;
}
```

:::

## Die `justify-content`-Eigenschaft

- **Definition**: Diese Eigenschaft richtet die Kindelemente entlang der Hauptachse des Flex-Containers aus.
- **`justify-content: flex-start;`**: In diesem Fall werden die Flex-Elemente am Anfang der Hauptachse ausgerichtet. Dies kann horizontal oder vertikal sein.
- **`justify-content: flex-end;`**: In diesem Fall sind die Flex-Elemente am Ende der Hauptachse ausgerichtet, horizontal oder vertikal.
- **`justify-content: center;`**: Dies zentriert die Flex-Elemente entlang der Hauptachse.
- **`justify-content: space-between;`**: Dies verteilt die Elemente gleichmäßig entlang der Hauptachse.
- **`justify-content: space-around;`**: Dies verteilt die Flex-Elemente gleichmäßig innerhalb der Hauptachse und fügt einen Abstand vor dem ersten und nach dem letzten Element hinzu.
- **`justify-content: space-evenly;`**: Dies verteilt die Elemente gleichmäßig entlang der Hauptachse.

:::interactive_editor

```html
<div class="container">
  <div class="box">A</div>
  <div class="box">B</div>
  <div class="box">C</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
  background: #eee;
}

.box {
  padding: 20px;
  background: salmon;
}
```

:::
  
## Die `align-items`-Eigenschaft

- **Definition**: Diese Eigenschaft wird verwendet, um Elemente entlang der Querachse zu verteilen. Denken Sie daran, dass die Querachse senkrecht zur Hauptachse steht.
- **`align-items: center;`**: Dies wird verwendet, um die Elemente entlang der Querachse zu zentrieren.
- **`align-items: flex-start;`**: Dies richtet die Elemente am Anfang der Querachse aus.
- **`align-items: stretch;`**: Dies wird verwendet, um die Flex-Elemente entlang der Querachse zu strecken.

:::interactive_editor

```html
<div class="container">
  <div class="box tall">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  align-items: center;
  height: 150px;
  background: #ddd;
}

.box {
  background: lightgreen;
  padding: 10px;
}

.tall {
  height: 100px;
}
```
