# --description--

要安裝 VS Code，你需要造訪他們的網站以取得適用於你的作業系統的安裝程式。請確定你造訪的是 VS Code 網站，而不是 Visual Studio 網站。

對於 Windows，你可以下載 EXE 安裝程式。下載完成後，執行該檔案，安裝精靈將引導你完成安裝程序。

對於 Mac，你可以下載一個包含可執行 `.app` 的 zip 壓縮檔。你也可以透過 homebrew 安裝 VS Code。

在你的 Mac 上開啟終端機應用程式並執行以下指令：

```sh
brew install --cask visual-studio-code
```

在 Linux 上，你可以直接下載 `.deb` 或 `.rpm` 檔案。安裝完成後，你可以使用適當的可執行檔或 CLI 指令（通常是 `code`）來啟動應用程式。現在你已經準備好撰寫程式碼。
