# --description--

设 $R(a, b, c)$ 为边长为 $a$、$b$ 和 $c$ 的三角形内三个不重叠圆所覆盖的最大面积。

设 $S(n)$ 为所有满足 $1 ≤ a ≤ b ≤ c &lt; a + b ≤ n$ 的整数三元组 $(a, b, c)$ 上 $R(a, b, c)$ 的平均值。

你得到 $S(2) = R(1, 1, 1) ≈ 0.31998$，$S(5) ≈ 1.25899$。

查找 $S(1803)$ 并将结果四舍五入到小数点后 5 位。
