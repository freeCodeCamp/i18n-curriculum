# --explanation--

`Haven't`は`have not`の短縮形で、`Present Perfect`時制で使われ、ある行動が起こっていないことを示します。

`Negative`は好ましくない、または批判的なものを指します。例えば：

`The user left a negative review about the app's performance.` - ここで`negative`はレビューの好ましくない性質を表しています。

`Comment`は誰かが提供した意見やフィードバックのことです。その複数形が`comments`です。例えば：

`I received a comment suggesting more features in the app.` - この文脈では、`comment`はアプリについてユーザーが提供したフィードバックを指します。
