# --description--

フレーズ `some of these` は、グループの特定の部分や、より大きな集合からの部分集合を指すときに使われます。個々の項目をすべて挙げることなく、グループの中のいくつかを言及したい場合によく使われます。

例えば、`Let's try some of these cookies.` 話者は大きなバッチからいくつかのクッキーを試すことを提案しています。
