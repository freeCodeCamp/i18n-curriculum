# --description--

To `use` означає використовувати щось для певної мети. Наприклад:

`She uses a notebook to take notes in class.` - Це показує, як вона застосовує зошит для письма.

`tool` — це те, що допомагає вам виконати роботу. У роботі інструменти часто є програмним забезпеченням або речами на комп’ютері. Наприклад:

`Zoom is a tool for online meetings.` - У цьому випадку інструмент — це програмне забезпечення, яке допомагає людям спілкуватися.
