# --explanation--

`Evidence of` означає ознаки або докази того, що щось сталося. Наприклад:

`The report showed evidence of user error.` — у звіті показано, що людина зробила помилку.

`brute force attack` — це коли хтось пробує багато різних паролів, поки один не спрацює.

`SQL injection` — це коли хтось намагається зламати систему, вводячи шкідливий код у поле пошуку або форму.

Джейк говорить про перевірку ознак таких типів атак.
