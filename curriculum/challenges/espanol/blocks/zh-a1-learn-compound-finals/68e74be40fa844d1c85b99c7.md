# --description--

Para pronunciar `iao`, comienzas con la lengua alta y cerca del paladar duro para `i`, luego haces una transición suave a la posición para `a`, y finalmente a la posición para `o`. El movimiento debe ser fluido, sin interrupción entre los tres sonidos.

Cuando `iao` se combina con las iniciales, obtienes sílabas como `biao`, `miao`, `diao`, `jiao`, `qiao`, `xiao`.

Practiquemos la pronunciación de estas sílabas, y luego usa `miao` en diferentes tonos como ejemplo.

PinyinCarácter
|--------|----------|
| `mīao`  | `喵`|
| `miáo`   | `苗`|
| `miǎo`  | `秒`|
| `miào`  | `妙`|
