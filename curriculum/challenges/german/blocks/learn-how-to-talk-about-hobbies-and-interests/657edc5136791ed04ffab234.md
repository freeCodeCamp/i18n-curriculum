# --description--

Eine `action figure` ist ein kleines Spielzeug, das wie eine Figur aus einem Film, Comic oder Videospiel aussieht. Menschen sammeln sie oft, weil sie die Figuren oder Geschichten mögen. Zum Beispiel:

`My brother got a new action figure for his birthday.` – Das bedeutet, er hat ein Spielzeug erhalten, das eine Figur darstellt, die ihm wahrscheinlich gefällt.
