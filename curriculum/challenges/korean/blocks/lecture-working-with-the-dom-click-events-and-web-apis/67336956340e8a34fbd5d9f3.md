# --interactive--

웹 페이지에서 부드러운 애니메이션을 만드는 것은 특히 타이밍을 제대로 다루는 방법을 모를 때 까다로울 수 있습니다. 좋은 소식은 `requestAnimationFrame()` API가 이를 더 쉽게 만들어준다는 점입니다. 

`requestAnimationFrame()`은 다음 화면 새로 고침 전에 애니메이션의 다음 단계를 예약할 수 있게 해주는 메서드로, 결과적으로 부드럽고 시각적으로 매력적인 경험을 제공합니다.

다음 화면 새로 고침은 브라우저가 웹 페이지의 시각적 표시를 새로 고치는 순간을 말합니다. 이는 대부분의 디스플레이에서 초당 약 60회(또는 60프레임) 여러 번 발생합니다.

`requestAnimationFrame()` 메서드를 사용하려면, 단순히 호출하고 콜백 함수를 전달하면 됩니다:

```js
requestAnimationFrame(callback);
```

`requestAnimationFrame()` 호출은 먼저 `animate()` 같은 애니메이션을 처리하는 함수 안에서, 그리고 전통적으로 `update()`라고 부르는 애니메이션을 업데이트하는 함수와 함께 이루어져야 합니다:

```js
function animate() {
 // Update the animation...
 // for example, move an element, change a style, and more.
 update();
 // Request the next frame
 requestAnimationFrame(animate);
}
```

`update()` 함수 안에서 마법이 일어납니다. 이 안에서 애니메이션할 원하는 모든 것을 변경할 수 있습니다. 예를 들어 스타일을 업데이트하거나 요소의 위치를 변경하는 것입니다:

```js
function update() {
 element.style.transform = `translateX(${position}px)`;
 position += 2;
}
```

애니메이션을 실제로 시작하는 것은 `requestAnimationFrame()`를 호출하고 이번에는 `animate` 함수를 `animate` 함수 밖에서 전달하는 것입니다:

```js
requestAnimationFrame(animate);
```

루프는 정지할 때까지 계속됩니다.

이제 다른 예제를 살펴보겠습니다. 이 예제의 HTML은 텍스트 `div`가 있는 `freeCodeCamp is Awesome` 요소입니다. CSS는 `div`를 직사각형으로 만들고 뷰포트의 왼쪽이나 오른쪽을 벗어나는 모든 것을 숨깁니다. JavaScript 코드는 `2px` 함수를 콜백으로 하여 `requestAnimationFrame(animate)`가 호출될 때마다 직사각형 `animate`를 오른쪽으로 이동시킵니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div id="rect" class="rect">freeCodeCamp is Awesome</div>
<script src="index.js"></script>
```

```css
body {
  overflow-x: hidden;
}

.rect {
  display: flex;
  align-items: center;
  justify-content: center;
  width: 400px;
  height: 250px;
  border-radius: 5px;
  background-color: #1b1b32;
  color: #f5f6f7;
  font-size: 2rem;
  position: absolute;
}
```

```js
const rect = document.getElementById("rect");

let position = 0;

function update() {
  // Move the rectangle 2px to the right
  rect.style.left = position + "px";
  position += 2;

  if (position > window.innerWidth) {
    // Move the rectangle just outside the left side of the screen
    position = -rect.offsetWidth;
  }
}

function animate() {
  update();

  //request the next frame
  requestAnimationFrame(animate);
}

// Start the animation
requestAnimationFrame(animate);
```
