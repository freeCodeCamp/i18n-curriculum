# --description--

Vamos a practicar cómo escribir algunos números entre 50 y 59. Para completar esta tarea, ten en cuenta que en chino, el número 50 se llama `五十 (wǔ shí)`.

Los números del 0 al 9 son:

| Número | Chino |
| :--- | :--- |
| 0 | `零 (líng)` |
| 1 | `一 (yī)` |
| 2 | `二 (èr)` |
| 3 | `三 (sān)` |
| 4 | `四 (sì)` |
| 5 | `五 (wǔ)` |
| 6 | `六 (liù)` |
| 7 | `七 (qī)` |
| 8 | `八 (bā)` |
| 9 | `九 (jiǔ)` |
