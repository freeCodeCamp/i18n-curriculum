# --description--

La frase `no worries` es una expresión informal utilizada para asegurarle a alguien que todo está bien y no debes preocuparse. Puede utilizarse para responder a un gracias, descartar una disculpa o asegurarle a alguien que un problema no es serio.

Por ejemplo, cuando tu amigo accidentalmente derrama una bebida y empieza a disculparse, puedes decir <code>no worries! Solo es agua, y se secará rápido</code> para hacer que el amigo se sienta mejor.
