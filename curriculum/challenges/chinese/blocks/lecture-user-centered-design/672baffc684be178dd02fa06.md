# --interactive--

**注意**：某些交互示例可能使用你尚未学习的 CSS。不要担心尝试理解所有的代码。示例的目的是向你展示这些设计概念的预览，以便你更好地理解其工作原理。要查看预览，你需要启用交互式编辑器。

卡片组件在电子商务、社交媒体和新闻网站中非常常见。它们用于帮助以结构化的方式显示信息。当你设计你的卡片时，了解最佳实践非常重要，这样你的用户才能轻松理解你试图传达的信息。

卡片设计的首要考虑应是简洁。你不希望你的卡片在视觉上杂乱或显示过多信息。例如，如果卡片设计在视觉上杂乱，用户将一次性需要处理过多信息。 

这是一个杂乱卡片设计的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2><strong><em>Product Name: Ultimate Edition</em></strong></h2>
  <p>
    This is a great product that you will love! It has many features and benefits that will make your life easier and more enjoyable. Whether you're at home or on the go, this product is perfect for all situations. Don’t miss out on this incredible offer, buy now and experience the difference!
  </p>
  <button>Buy Now</button>
  <button>Learn More</button>
  <button>Add to Wishlist</button>
  <p><strong>Only 3 left in stock!</strong></p>
  <p><em>Rated 4.8 stars by 2,391 customers</em></p>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}
.card p {
  font-size: 1em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

卡片上的信息较少且各项之间间距合理，使用户更容易处理信息，并允许页面上显示多个卡片。

这是一个简单卡片设计的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2>Product Name</h2>
  <button>Buy Now</button>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 200px;
  text-align: center;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

另一个需要考虑的点是用户可以点击卡片的哪个位置。有些卡片设计会有一个单独的按钮，使用户点击位置显而易见。其他卡片设计则会使整个卡片都可点击。当用户将鼠标悬停在卡片的任何部分时，卡片会改变颜色或有阴影效果，以表明卡片是可点击的。无论你选择哪种设计，都需要在你的网站中保持一致，并且让用户容易理解。 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <a href="#">
    <img src="https://placehold.co/150x150" alt="Example Product Image">
    <h2>Product Name</h2>
    <button>Buy Now</button>
  </a>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
  cursor: pointer;
}

.card:hover {
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
}

.card a {
  text-decoration: none;
  color: inherit;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

另一个需要考虑的是在你的卡片上使用媒体。选择高质量的媒体可以显著提升用户体验。如果你在产品卡片中使用图像或视频，质量越高，用户对该产品的兴趣就越大。但如果你使用低质量的媒体，用户可能不会信任你的产品和服务的质量。 

最后需要考虑的事项之一是颜色层次结构的使用。你要确保卡片上最重要的信息最为突出。你可以为重要元素（如调用按钮）使用鲜艳的颜色，为卡片上不太重要的项使用浅色。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2>Product Name</h2>
  <button>Buy Now</button>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
  text-align: center;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #28a745; 
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

当你继续开发网页应用时，请记住卡片设计的不同最佳实践。这将帮助你为你的用户创建更好的用户体验。
