# --description--

當你想詢問某個特定的事物時，你可以使用疑問字 `what`。`What` 通常用來談論事物。 

你通常會將 `what` 放在問題的開頭。然後接著使用動詞 `to be`（例如 `is` 或 `are`）或其他動詞，接著是主詞（例如 `you` 或 `the team`）。例如：

- `What is your favorite color?`－有人想知道你最喜歡的顏色。

- `What does this code do?`－有人想知道這段程式碼的工作內容或結果。
