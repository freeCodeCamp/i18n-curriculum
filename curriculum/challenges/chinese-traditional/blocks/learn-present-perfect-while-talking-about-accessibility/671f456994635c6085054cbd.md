# --explanation--

`Correctly` 是指以正確或預期的方式執行某事。 例如：

`Make sure the software updates correctly to avoid errors。` - 它強調更新應該在沒有問題或錯誤的情況下進行。

`User` 指的是與產品、服務或系統互動的人。例如：

`The app was designed with users in mind.`－這表示有人應該考慮使用該應用程式的人，凸顯標示對他們需求和體驗的關注。
