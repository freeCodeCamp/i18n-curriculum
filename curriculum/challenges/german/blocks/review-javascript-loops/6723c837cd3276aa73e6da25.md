# --interactive--

## Arbeiten mit Schleifen

- **`for` Schleife**: Diese Art von Schleife wird verwendet, um einen Codeblock eine bestimmte Anzahl von Malen zu wiederholen. Diese Schleife ist in drei Teile gegliedert: die Initialisierungsanweisung, die Bedingung und die Inkrement-/Dekrement-Anweisung. Die Initialisierungsanweisung wird vor Beginn der Schleife ausgeführt. Sie dient typischerweise dazu, eine Zählervariable zu initialisieren. Die Bedingung wird vor jeder Iteration der Schleife ausgewertet. Eine Iteration ist ein einzelner Durchlauf der Schleife. Wenn die Bedingung `true` ist, wird der Codeblock innerhalb der Schleife ausgeführt. Wenn die Bedingung `false` ist, endet die Schleife und Sie fahren mit dem nächsten Codeblock fort. Die Inkrement-/Dekrement-Anweisung wird nach jeder Iteration der Schleife ausgeführt. Sie dient typischerweise dazu, die Zählervariable zu erhöhen oder zu verringern.

:::interactive_editor

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

:::

- **`for...of` Schleife**: Diese Art von Schleife wird verwendet, wenn Sie Werte aus einem iterierbaren Objekt durchlaufen müssen. Beispiele für Iterierbare sind Arrays und Strings.

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

:::

- **`for...in` Schleife**: Diese Art von Schleife eignet sich am besten, wenn Sie die Eigenschaften eines Objekts durchlaufen müssen. Diese Schleife iteriert über alle aufzählbaren Eigenschaften eines Objekts, einschließlich geerbter Eigenschaften und nicht-numerischer Eigenschaften.

:::interactive_editor

```js
const fruit = {
  name: 'apple',
  color: 'red',
  price: 0.99
};

for (const prop in fruit) {
  console.log(fruit[prop]);
}
```

:::

- **`while` Schleife**: Diese Art von Schleife führt einen Codeblock so lange aus, wie die Bedingung `true` ist.

:::interactive_editor

```js
let i = 5;

while (i > 0) {
  console.log(i);
  i--;
}
```

:::

- **`do...while` Schleife**: Diese Art von Schleife führt den Codeblock mindestens einmal aus, bevor die Bedingung überprüft wird.

```js
let userInput;

do {
  userInput = prompt("Please enter a number between 1 and 10");
} while (Number(userInput) < 1 || Number(userInput) > 10);

alert("You entered a valid number!");
```

## `break` und `continue` Anweisungen

- **Definition**: Eine `break` Anweisung wird verwendet, um eine Schleife vorzeitig zu verlassen, während eine `continue` Anweisung verwendet wird, um die aktuelle Iteration einer Schleife zu überspringen und mit der nächsten fortzufahren.

:::interactive_editor

```js
// Example of break statement
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, and 4

// Example of continue statement 
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, 4, 6, 7, 8, and 9
```

:::
