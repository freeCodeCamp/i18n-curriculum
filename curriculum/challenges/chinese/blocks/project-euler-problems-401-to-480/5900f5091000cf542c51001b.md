# --description--

如果 $x$、$y$ 和 $x + y$ 都是正的完全平方数，则称格点 ($x$, $y$) 为不可接受的。

例如，（9,16）是不允许的，而（0,4），（3,1）和（9,4）则不允许。

考虑一条从点 ($x_1$, $y_1$) 到点 ($x_2$, $y_2$) 的路径，该路径仅使用单位步长向北或向东。 如果其所有中间点均为可接受点，则称该路径为可接受路径。

设 $P(n)$ 为从 (0, 0) 到 ($n$, $n$) 的可行 `路径` 数量。可以验证 $P(5) = 252$，$P(16) = 596\\,994\\,440$，且 $P(1\\,000)\bmod 1\\,000\\,000\\,007 = 341\\,920\\,854$。

查找 $P(10\,000\,000)\bmod 1\,000\,000\,007$。
