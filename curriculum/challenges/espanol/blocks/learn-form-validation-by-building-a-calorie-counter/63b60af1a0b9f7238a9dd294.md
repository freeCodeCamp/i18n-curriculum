# --description--

El menú desplegable necesita opciones para cada uno de los elementos `fieldset` de comida y ejercicio que creaste en los pasos anteriores. Usa el elemento `option` para crear una nueva opción para cada `fieldset`. El atributo `value` de cada opción debe ser el `id` del `fieldset`, y el texto de cada opción debe ser el texto del `legend`.

Establece la opción `Breakfast` como la opción `selected`.
