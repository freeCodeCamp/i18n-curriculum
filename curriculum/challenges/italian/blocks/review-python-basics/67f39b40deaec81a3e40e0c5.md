## Che cos’è Python?

- **Introduzione**: Python è un linguaggio di programmazione general-purpose noto per la sua semplicità e facilità d’uso. Python è usato in molti ambiti come data science e machine learning, sviluppo web, scripting e automazione, sistemi embedded e IoT, e molto altro.

## Python nel tuo ambiente locale

- **Installazione**: Il modo migliore per installare Python su Windows, Mac e Linux è scaricare l’installer dal sito ufficiale di Python (`https://www.python.org/`).

## Variabili

- **Dichiarare variabili**: Per dichiarare una variabile, inizi con il nome della variabile seguito dall’operatore di assegnazione (`=`) e poi il valore. Questo può essere un numero, stringa, booleano, ecc. Ecco alcuni esempi:

```py
name = 'John Doe'
age = 25
```

- **Convenzioni per i nomi delle variabili**: Ecco le convenzioni che dovresti usare per i nomi delle variabili:

  - I nomi delle variabili possono iniziare solo con una lettera o un underscore (_), non con un numero.
  - I nomi delle variabili possono contenere solo caratteri alfanumerici (a-z, A-Z, 0-9) e underscore (_).
  - I nomi delle variabili sono case-sensitive — `age`, `Age` e `AGE` sono considerati tutti unici.
  - I nomi delle variabili non possono essere una delle parole chiave riservate di Python come `if`, `class` o `def`.
  - I nomi delle variabili composti da più parole sono separati da underscore. Es. `snake_case`.

## Tipi di dati comuni in Python

- **Introduzione**: Python è un linguaggio a tipizzazione dinamica come JavaScript, il che significa che non devi dichiarare esplicitamente i tipi per le variabili. Il linguaggio sa qual è il tipo di una variabile in base a ciò che assegni alla variabile.
- **Intero**: Un numero intero senza decimali:

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **Float**: Un numero con decimali:

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **Stringa**: Una sequenza di caratteri racchiusa tra virgolette:

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **Booleano**: Un valore che rappresenta `True` o `False`:

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **Set**: Una collezione non ordinata di elementi unici:

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **Dizionario**: Una collezione di coppie chiave-valore, racchiusa tra parentesi graffe:

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **Tupla**: Una collezione ordinata e immutabile, racchiusa tra parentesi tonde:

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **Range**: Una sequenza di numeri, spesso usata nei loop:

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **Lista**: Una collezione ordinata di elementi che supporta tipi di dati diversi:

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**: Un valore speciale che rappresenta l’assenza di un valore:

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## Tipi immutabili e mutabili

- **Tipi immutabili**: Questi tipi non possono cambiare una volta dichiarati, anche se puoi puntare le loro variabili a qualcosa di nuovo, chiamato riassegnazione. Includono intero, float, complesso, booleano, stringa, tupla, range e `None`.
- **Tipi mutabili**: Questi tipi possono cambiare una volta dichiarati. Puoi aggiungere, rimuovere o aggiornare i loro elementi. Includono tipi di collezioni come lista, set e dizionario.
- **Funzione `type()`**: Per vedere il tipo di una variabile, puoi usare la funzione `type()` così:

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **Funzione `isinstance()`**: Questa serve per verificare se una variabile corrisponde a un tipo di dato specifico:

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## Lavorare con le stringhe

- **Definizione**: Come ricorderai da JavaScript, le stringhe sono immutabili, il che significa che non puoi cambiarle dopo che sono state create. In Python puoi usare sia virgolette singole che doppie. È consigliato scegliere una regola e mantenerla:

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **Consultare i caratteri delle stringhe**: Puoi consultare i caratteri delle stringhe usando la notazione a parentesi così:

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **Escape delle stringhe**: Puoi usare una barra rovesciata (`\`) se la tua stringa contiene virgolette così:

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **Concatenazione di stringhe**: Per concatenare stringhe puoi usare l’operatore `+` così:

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

Un altro modo per concatenare stringhe è usare l’operatore `+=`. Questo serve per fare concatenazione e assegnazione nello stesso passaggio così:

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**: È l’abbreviazione di formatted string literals. Ti permette di gestire l’interpolazione e anche di fare un po’ di concatenazione con una sintassi compatta e leggibile:

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **Slicing delle stringhe**: Questo è quando puoi estrarre porzioni di una stringa. Ecco la sintassi base:

```py
str[start:stop:step]
```

La posizione di inizio rappresenta l’indice da cui deve iniziare l’estrazione. La posizione di fine è dove la slice deve terminare. Questa posizione non è inclusiva. La posizione di step rappresenta l’intervallo con cui incrementare per lo slicing. Ecco alcuni esempi:

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **Ottenere la lunghezza di una stringa**: La funzione `len()` serve per restituire il numero di caratteri nella stringa:

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## Lavorare con l’operatore `in`

- **Operatore `in`**: Questo restituisce un booleano che specifica se il carattere o i caratteri esistono nella stringa o no:

```py
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False

```

## Metodi comuni delle stringhe

- **`str.upper()`**: Restituisce una nuova stringa con tutti i caratteri convertiti in maiuscolo:

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`str.lower()`**: Restituisce una nuova stringa con tutti i caratteri convertiti in minuscolo:

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`str.strip()`**: Restituisce una copia della stringa con i caratteri iniziali e finali specificati rimossi (se non viene passato alcun argomento al metodo, rimuove gli spazi bianchi iniziali e finali).

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**: Restituisce una nuova stringa con tutte le occorrenze della vecchia stringa sostituite da una nuova.

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**: Serve per dividere una stringa in una lista usando un separatore specificato. Un separatore è una stringa che indica dove deve avvenire la divisione.

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**: Serve per unire gli elementi di un iterabile in una stringa con un separatore. Un iterabile è una collezione di elementi su cui si può iterare come una lista, una stringa o una tupla.

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`str.startswith(prefix)`**: Restituisce un booleano che indica se una stringa inizia con il prefisso specificato:

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`str.endswith(suffix)`**: Restituisce un booleano che indica se una stringa termina con il suffisso specificato:

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`str.find()`**: Restituisce l’indice della prima occorrenza di una sottostringa. Se non viene trovata, restituisce `-1`:

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`str.count(substring)`**: Conta quante volte una sottostringa appare in una stringa:

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`str.capitalize()`**: Restituisce una nuova stringa con il primo carattere maiuscolo e gli altri caratteri minuscoli:

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`str.isupper()`**: Restituisce `True` se tutte le lettere nella stringa sono maiuscole e `False` altrimenti:

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`str.islower()`**: Restituisce `True` se tutte le lettere nella stringa sono minuscole e `False` altrimenti:

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`str.title()`**: Restituisce una nuova stringa con la prima lettera di ogni parola maiuscola:

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`str.maketrans()`**: Questo metodo serve per creare una tabella di mappature 1 a 1 per la traduzione dei caratteri. Viene spesso usato con il metodo `translate()` che applica quella tabella a una stringa e restituisce il risultato tradotto.

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## Operazioni comuni con interi e float

- **Operazioni matematiche di base**: In Python puoi fare operazioni matematiche di base con interi e float, inclusi addizione, sottrazione, moltiplicazione e divisione:

```py
float_1 = 5.4
float_2 = 12.0
```

```py
# Addition assignment 
my_var = 10

# Subtraction assignment
count = 14

print(count) # 11

# Multiplication assignment 
product = 65

price = 100

print(price) # 25.0

# Floor Division assignment 
total_pages = 23

bits %= 2

power = 2
power **= 3

print(power) # 8
```

## Lavorare con le funzioni

- **Definizione**: Le funzioni sono pezzi di codice riutilizzabili che prendono input (argomenti) e restituiscono un output. Per chiamare una funzione, devi fare riferimento al nome della funzione seguito da un insieme di parentesi tonde:

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call

```

Se una funzione non restituisce esplicitamente un valore, allora il valore di ritorno predefinito è `None`:

```py
def greet():

result = greet() # hello

```

Puoi anche fornire valori predefiniti ai parametri così:

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

Se chiami la funzione senza il numero corretto di argomenti, otterrai un `TypeError`:

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

## Ambito in Python

- **Ambito locale**: È quando una variabile dichiarata dentro una funzione o classe può essere consultata solo all’interno di quella funzione o classe.

```py
def my_func():
    num = 10
    print(num)
```

- **Ambito annidato**: È quando una funzione annidata dentro un’altra funzione può consultare le variabili della funzione in cui è annidata.

```py
def outer_func():
    msg = 'Hello there!'
    
    def inner_func():
        print(msg)
    inner_func()

print(outer_func()) # Hello there!
```

- **Ambito globale**: Si riferisce a variabili dichiarate fuori da qualsiasi funzione o classe che possono essere consultate da qualsiasi parte del programma.

```py
tax = 0.70 

def get_total(subtotal):
    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **Ambito integrato**: Nomi riservati in Python per funzioni, moduli, parole chiave e oggetti predefiniti.

```py
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```

## Lavorare con le istruzioni `if`, `elif` e `else`

- **Istruzioni `if`**: Sono condizioni usate per determinare se qualcosa è vero o no. Se la condizione valuta a `True`, allora quel blocco di codice viene eseguito.

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **Clausola `elif`**: Sono condizioni che seguono un’istruzione `if`. Un blocco `elif` viene eseguito solo se tutte le condizioni precedenti valutano a `False` e la sua condizione valuta a `True`.

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **Clausola `else`**: Viene eseguita se nessun’altra condizione valuta a `True`.

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

Puoi anche usare istruzioni `if` annidate così:

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## Valori truthy e falsy

- **Definizione**: In Python ogni valore ha un valore booleano intrinseco, o un senso integrato di se deve essere trattato come `True` o `False` in un contesto logico. Molti valori sono considerati truthy, cioè valutano a `True` in un contesto logico. Altri sono falsy, cioè valutano a `False`. Ecco alcuni esempi di valori falsy:

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

Altri valori come numeri diversi da zero e stringhe non vuote sono truthy.

## Lavorare con la funzione `bool()`

- **Definizione**: Se vuoi verificare se un valore è truthy o falsy, puoi usare la funzione integrata `bool()`. Converte esplicitamente un valore nel suo equivalente booleano e restituisce `True` per valori truthy e `False` per valori falsy. Ecco alcuni esempi:

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## Operatori booleani e short-circuiting

- **Definizione**: Sono operatori speciali che ti permettono di combinare più espressioni per creare logiche decisionali più complesse nel tuo codice. Ci sono tre operatori booleani in Python: `and`, `or` e `not`.
- **Operatore `and`**: Questo operatore prende due operandi e restituisce il primo operando se è falsy, altrimenti restituisce il secondo operando. Entrambi gli operandi devono essere truthy affinché un’espressione risulti truthy.

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

Puoi anche usare l’operatore `and` nelle condizioni così:

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **Operatore `or`**: Questo operatore restituisce il primo operando se è truthy, altrimenti restituisce il secondo operando. Un’espressione `or` risulta truthy se almeno un operando è truthy. Ecco un esempio:

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

Proprio come con l’operatore `and`, puoi usare l’operatore `or` nelle condizioni così:

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **Short-circuiting**: Gli operatori `and` e `or` sono noti come operatori short-circuit. Lo short-circuiting significa che Python controlla i valori da sinistra a destra e si ferma non appena determina il risultato finale.
- **Operatore `not`**: Questo operatore prende un solo operando e inverte il suo valore booleano. Converte valori truthy in `False` e valori falsy in `True`. A differenza degli operatori precedenti, `not` restituisce sempre `True` o `False`. Ecco alcuni esempi:

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

Ecco un esempio dell’operatore `not` in una condizione:

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
```
