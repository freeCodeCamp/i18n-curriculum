# --description--

Katika lugha yoyote ya programu, unahitaji kulinganisha thamani ili kufanya maamuzi. Kwa mfano, ikiwa hali ya hewa ni jua, utaenda nje, vinginevyo utabaki nyumbani. Mfano mwingine ni kuona kama mtumizi ana pointi za kutosha kupanda ngazi katika mchezo.

Taarifa ya `if-else` hutumika kufanya maamuzi katika JavaScript. Mara nyingi hutumika pamoja na vigezo vya kulinganisha ili kulinganisha thamani na kufanya maamuzi kulingana na matokeo.

Mfano wa taarifa ya if ni:

```javascript
let x = 5;

if (x > 3) {
  console.log("x is greater than 3");
} else {
  console.log("x is less than or equal to 3");
}
```

Katika mfano huu, tofauti ya `x` inalinganishwa na nambari `3` kwa kutumia kigezo cha `>`. Ikiwa `x` ni kubwa kuliko `3`, sehemu ya kwanza ya msimbo itatekelezwa. Vinginevyo, sehemu ya pili ya msimbo itatekelezwa.
