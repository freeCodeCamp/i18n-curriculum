# --interactive--

JavaScriptでは、関数とオブジェクトメソッドはどちらも再利用可能なコードをまとめる方法ですが、定義の仕方、使い方、動作するコンテキストにいくつか重要な違いがあります。これらの違いを理解することは、効果的で整理されたJavaScriptコードを書くために非常に重要です。

前のモジュールで学んだように、関数は特定のタスクを実行する再利用可能なコードのブロックです。

:::interactive_editor

```js
function greet(name) {
    return "Hello, " + name + "!";
}
console.log(greet("Alice")); // "Hello, Alice!"
```

:::

一方、オブジェクトメソッドはオブジェクトに関連付けられた関数です。オブジェクトのプロパティとして定義され、そのオブジェクトのデータにアクセスし操作できます。以下はメソッドを持つオブジェクトの例です。

:::interactive_editor

```js
const person = {
    name: "Bob",
    age: 30,
    sayHello: function() {
        return "Hello, my name is " + this.name;
    }
};

console.log(person.sayHello()); // "Hello, my name is Bob"
```

:::

この例では、`sayHello` は `person` オブジェクトのメソッドです。`this` キーワードによって、`sayHello` メソッドは `person` という名前のオブジェクトのプロパティにアクセスできます。`this` キーワードについては今後のレッスンで詳しく学びます。

関数とメソッドの違いの一つは呼び出し方です。関数は名前で呼び出しますが、メソッドは所属するオブジェクトのドット記法を使って呼び出します。例えば、`greet` 関数は `greet("Alice")` と呼び出しますが、`sayHello` メソッドは `person.sayHello()` と呼び出します。

もう一つの重要な違いは動作するコンテキストです。通常の関数は独自のスコープを持ちますが、特定のオブジェクトへの組み込みの参照はありません。一方、メソッドはそのオブジェクトに結びついており、`this` キーワードを使ってオブジェクトのプロパティや他のメソッドにアクセスできます。

重要なポイントは、メソッドはコードを論理的なオブジェクトに整理するのに役立ち、関数はより一般的で再利用可能なコードに使われることです。

関数とオブジェクトメソッドの違いを理解することは、JavaScriptの学習を進める上で重要です。最初は似ているように見えますが、それぞれを使うべきタイミングを見極めることで、より整理され効率的で良いコードを書けるようになります。
