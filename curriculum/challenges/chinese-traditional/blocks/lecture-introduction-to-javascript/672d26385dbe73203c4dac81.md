# --interactive--

JavaScript 是一種強大的程式設計語言，為網站帶來互動性和動態行為。

雖然 HTML 和 CSS 是用來結構內容和設定頁面元素樣式的標記語言，JavaScript 則超越這些，提供更複雜的功能，例如處理使用者輸入、動畫元素，甚至建置完整的網頁應用程式。

例如，當你點擊按鈕、提交表單或暫留在選單時，JavaScript 決定頁面的行為。

以下是這三者如何協同運作的範例：

:::interactive_editor

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: green;
    }
  </style>
</head>
<body>
  <h1>Hello, World!</h1>
  <button onclick="alert('Button clicked!')">Click me</button>
</body>
</html>
```

:::

在此範例中，使用 HTML 來定義內容：標題（一個 `h1` 元素）和按鈕（`button` 元素）。

CSS 用於套用標題的樣式，例如將文字設為綠色。JavaScript 用於在按鈕被點擊時顯示警示訊息。

總結來說，HTML 提供結構，CSS 添加樣式，而 JavaScript 啟用行為，創造完整且互動的網頁體驗。
