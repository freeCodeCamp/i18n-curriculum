# --interactive--

Nelle lezioni precedenti, hai imparato a lavorare con i loop `for`, i loop `for...in` e i loop `for...of`. In questa lezione, scoprirai il loop `while` e il loop `do...while`.

Un loop `while` eseguirà un blocco di codice finché la condizione è vera. Ecco la sintassi base per un loop `while`:

```js
while (condition) {
  // code block to be executed
}
```

La condizione viene verificata prima che il blocco di codice venga eseguito. Se la condizione è falsa, il blocco di codice non verrà eseguito.

I loop `while` sono utili quando non sai quante volte devi eseguire il blocco di codice. Ecco un esempio di utilizzo di un loop `while`:

:::interactive_editor

```js
let counter = 0;
while(counter < 5) {
  console.log(counter);
  counter++;
}
```

:::

In questo esempio, abbiamo una variabile chiamata `counter` inizializzata a `0`. Il loop `while` continuerà a eseguire finché il valore di `counter` è minore di `5`. All’interno del loop, stampiamo il valore di `counter` nella console e poi incrementiamo `counter` di `1`.

Un altro loop simile al loop `while` è il loop `do...while`. Ecco la sintassi base:

```js
do {
  // code block to be executed
} while (condition);
```

Una differenza chiave tra un loop `do...while` e un loop `while` è che il loop `do...while` eseguirà il blocco di codice almeno una volta prima di verificare la condizione.

Se la condizione è vera, il blocco di codice continuerà a essere eseguito. Se la condizione è falsa, il blocco di codice smetterà di essere eseguito.

Ecco un esempio di utilizzo di un loop `do...while`:

:::interactive_editor

```js
let counter = 0;
do {
  console.log(counter);
  counter++;
} while (counter < 5);  
```

:::

In questo esempio, abbiamo una variabile chiamata `counter` inizializzata a `0`. Il loop `do...while` stamperà il valore di `counter` nella console e poi incrementerà `counter` di `1`. Dopo aver eseguito il blocco di codice, verifica se il valore di `counter` è minore di `5`. Se lo è, il loop continuerà a eseguire. Altrimenti, il loop si fermerà.

Nella maggior parte dei casi, probabilmente userai più spesso il loop `while` rispetto al loop `do...while`. Tuttavia, è utile conoscere entrambi i tipi di loop e quando usarli.

Qual è la differenza principale tra un loop `while` e un loop `do...while`?

## --answers--
