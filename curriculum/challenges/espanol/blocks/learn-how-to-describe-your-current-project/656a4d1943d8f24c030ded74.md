# --description--

Cuando alguien usa `either` al final de una oración negativa, significa que la misma situación se aplica también a ellos. Por ejemplo: 

No me gustan las zanahorias. A mi hermana tampoco le gustan.

# --questions--

## --text--

¿Qué implica Bob con la frase `I'm not currently working on any major data analysis projects either`?

## --answers--

Él implica que está trabajando en diferentes proyectos.
