# --description--

La palabra `everyone` se usa para hablar sobre todos en el grupo. `Every-` se usa cuando se habla de todas las cosas o personas en un grupo, una por una.

Es importante observar que `everyone` suena como si estuviera hablando de muchas personas, pero lo usas con un verbo singular. Esto es similar a otras palabras que comienzan con `every-` como `everything` y `everybody`.

Por ejemplo, dices `Everyone goes to the party`, `Everything is right` o `Everybody works hard`.

`Make sure` significa verificar o asegurarse de que algo es correcto antes de continuar. Por ejemplo, `You make sure the code runs` significa que verificas el código para estar seguro de que no tiene errores y puede ejecutarse correctamente.

`Follow the rules` significa hacer lo que dicen las reglas. Si hay una regla que dice `use camelCase para variable names`, seguir las reglas significa que nombrar tus variables como `myVariable` en lugar de `myBLANKvariable`.
