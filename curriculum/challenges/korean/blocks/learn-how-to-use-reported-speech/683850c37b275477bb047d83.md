# --explanation--

`log`는 컴퓨터 시스템이나 소프트웨어가 생성하는 이벤트, 동작, 메시지의 기록입니다. 예를 들어:

`The system logs showed several errors before the crash.` – 이는 기록에 문제가 발생했다는 정보가 포함되어 있음을 의미합니다.

`crash`는 컴퓨터 시스템이나 프로그램이 갑자기 제대로 작동을 멈추는 경우입니다. 예를 들어:

`The application crash caused all unsaved data to be lost.` – 이는 프로그램이 예기치 않게 실패하여 데이터가 저장되지 않았음을 의미합니다.
