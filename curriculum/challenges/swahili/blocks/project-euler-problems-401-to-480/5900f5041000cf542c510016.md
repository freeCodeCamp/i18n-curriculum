# --description--

Ikiwa tukihesabu $a^2\bmod 6$ kwa $0 ≤ a ≤ 5$ tunapata: 0, 1, 4, 3, 4, 1.

Thamani kubwa zaidi ya a ambapo $a^2 ≡ a\bmod 6$ ni $4$.

Tuitaje $M(n)$ kuwa thamani kubwa zaidi ya $a < n$ ambapo $a^2 ≡ a (\text{mod } n)$. Hivyo $M(6) = 4$.

Tafuta $\sum M(n)$ kwa $1 ≤ n ≤ {10}^7$.
