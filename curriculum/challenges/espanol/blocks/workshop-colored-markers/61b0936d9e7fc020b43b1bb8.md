# --description--

Parece que la manga ha desaparecido, pero no te preocupes — aún está ahí. Lo que sucedió es que tu nuevo `div` de gorra está ocupando todo el ancho del marcador y está empujando la manga hacia abajo a la siguiente línea.

Esto es porque la propiedad `display` predeterminada para elementos `div` es `block`. Por lo tanto, cuando dos elementos `block` están uno al lado del otro, se apilan como bloques reales. Por ejemplo, tus elementos de marcador están todos apilados uno sobre otro.

Para posicionar dos elementos `div` en la misma línea, establece sus propiedades `display` a `inline-block`.

Crea una nueva regla para dirigirte a las clases `cap` y `sleeve`, y establece `display` en `inline-block`.

Tu regla CSS debe tener una propiedad `display` establecida en `inline-block`.

```js
assert.isTrue(new __helpers.CSSHelp(document).getStyle('.cap, .sleeve')?.display === 'inline-block' || new __helpers.CSSHelp(document).getStyle('.sleeve, .cap')?.display === 'inline-block');
```
