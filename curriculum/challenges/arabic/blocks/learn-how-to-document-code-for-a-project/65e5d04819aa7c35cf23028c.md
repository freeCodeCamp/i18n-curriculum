# --description--

كلمة `so` هي أداة ربط تُستخدم لإظهار السبب والنتيجة أو لشرح سبب شيء ما. في التكنولوجيا والبرمجة، غالبًا ما تربط `so` المشكلة أو الحالة بالحل أو النتيجة.

# --questions--

## --text--

اختر الجملة التي تُستخدم فيها `so` بشكل صحيح لربط السبب بالنتيجة في سياق تقني.

## --answers--

`The program crashed, so we must reboot the computer.`

---

`We wrote the code, so but it didn't compile.`

### --feedback--

تُستخدم كلمة `so` بشكل غير صحيح هنا. لا يجب استخدام `But` بعد `so`. يجب أن تكون الجملة إما `The code didn't compile, so we rewrote it`، أو `We wrote the code, but it didn't compile.`

---

`The server is online, so therefore we can't connect.`

### --feedback--

استخدام `so` و`therefore` معًا مكرر وغير ضروري. يجب أن تكون إما `The server is online, so we can connect` أو `The server is online; therefore, we can connect.`

---

`We need more data, so we can ignore the results.`

### --feedback--

هذه الجملة لا تربط السبب بالنتيجة بشكل منطقي. استخدام أفضل لـ `so` سيكون، `We need more data, so we will perform more tests.`
