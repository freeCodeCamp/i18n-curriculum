# --explanation--

`UI` 代表 `user interface`，指的是用户与产品交互的视觉元素。`设计师 (shè jì shī)` 意思是“designer”。

`UI 设计师 (shè jì shī)` 是指专注于设计产品外观和版型，使其易于使用且令人愉悦的职业。这是科技公司中非常常见的术语，人们通常直接使用 `UI`，而不将其翻译成中文。例如：

`我是 (wǒ shì) UI 设计师 (shè jì shī)。` – 我是 UI 设计师。
