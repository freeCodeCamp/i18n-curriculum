# --description--

情态动词 `can` 用于表示可能性或能力。例如，在句子 `You can see the stars at night` 中，`can` 表示在夜间可以看到星星。

`Change` 意味着使某物变得不同或变成不同的状态。在日常生活中，你可能会在下班后 `change` 衣服，表示你从工作服换成休闲服。

`Stay the same` 意味着保持不变或常量。例如，`The rules of the game stay the same every year.` 这意味着规则每年都不会改变。
