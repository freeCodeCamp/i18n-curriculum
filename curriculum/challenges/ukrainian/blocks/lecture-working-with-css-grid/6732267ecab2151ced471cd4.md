# --interactive--

У попередніх уроках ми трохи говорили про те, як створити простір між елементами сітки. Але в цьому уроці ми детальніше розглянемо, як використовувати властивості `row-gap`, `column-gap` і `gap` у макеті сітки.

Але спершу потрібно повторити, що таке трек у CSS сітці. Трек — це простір між двома сусідніми лініями сітки. Ці лінії автоматично створюються, коли ви використовуєте CSS Grid. У цьому контексті треки зазвичай означають рядки та колонки, які складають макет сітки.

Щоб створити розриви між колонками в CSS Grid, можна використати властивість `column-gap`. Допустимі значення для цієї властивості включають пікселі, одиницю `em`, відсотки або ключове слово `normal`.

Якщо ви використаєте значення `normal` для властивості `column-gap`, то результатом буде `0` для макетів сітки і `1em` для багатоколонкових макетів.

Ось приклад розмітки для макету сітки з чотирма колонками:

```html
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

Для CSS ми встановлюємо display у значення grid і властивість `column-gap` у 10 пікселів:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  height: 100px;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.grid-container div {
  background-color: darkblue;
}
```

:::

Якщо ми хочемо змінити приклад так, щоб було два рядки синіх блоків і створити більше простору між рядками, ми можемо використати властивість `row-gap`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  column-gap: 10px;
  row-gap: 30px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

У цьому зміненому прикладі ми встановлюємо властивість `row-gap` у 30 пікселів і змінюємо `grid-template-columns`, щоб використовувати лише дві одиниці `1fr` замість чотирьох, створюючи два рядки блоків.

Як і для властивості `column-gap`, допустимі значення для властивості `row-gap` можуть включати відсотки, `em` і пікселі.

Якщо ви хочете використати скорочення для створення розривів між рядками і колонками, можна застосувати властивість `gap`. Ось базовий синтаксис:

```css
gap: row-value optional-column-value;
```

Якщо ви вкажете одне значення для властивості `gap`, це значення буде застосоване і до рядків, і до колонок. Якщо вказати два значення, перше застосовується до рядка, а друге — до колонки:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 30px 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

Допустимі значення для скороченої властивості `gap` включають відсотки, пікселі, `em` і навіть значення `calc()`. Але тут не можна використовувати одиниці `fr`.

Властивості `row-gap`, `column-gap` і `gap` надають гнучкі способи контролювати простір між елементами в макеті CSS Grid. Використовуючи ці властивості, ви легко створите візуально привабливі сітки з послідовними і регульованими розривами між рядками і колонками.
