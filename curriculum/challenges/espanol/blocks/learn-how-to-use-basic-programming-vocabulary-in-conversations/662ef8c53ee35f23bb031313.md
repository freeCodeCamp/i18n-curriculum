# --description--

La expresión `a lot of` se puede usar tanto con sustantivos contables como incontables, para indicar un gran número o cantidad.

Para sustantivos contables, como `cars`, puedes decir `a lot of cars`. Para sustantivos incontables, como `water`, dirías `a lot of water`.
