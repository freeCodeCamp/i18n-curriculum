# --interactive--

在上一课中，你学习了如何通过使用 `addEventListener()` 方法处理 `"click"` 和 `"input"` 事件。在本课中，你将学习如何使用 `removeEventListener()` 方法移除事件监听器。

`removeEventListener()` 方法用于移除之前通过 `addEventListener()` 方法为元素添加的事件监听器。当你想停止监听元素上的特定事件时，这非常有用。

这是 `removeEventListener()` 方法的基本语法：

```js
element.removeEventListener("event", listener);
```

就像 `addEventListener()` 方法一样，`removeEventListener()` 方法接受两个参数：你想要移除的事件和之前添加的监听器函数。

`removeEventListener()` 方法还可以传入一个额外的可选第三个参数。该参数可以是 `options` 或 `useCapture`。

`options` 参数是一个对象，用于指定事件监听器的选项，例如事件监听器是否应该是被动的或只执行一次。 

`useCapture` 参数是一个布尔值，用于指定事件是否应在事件传播阶段被捕捉。

大多数情况下，你只需要将 `event` 和 `listener` 参数传递给 `removeEventListener()` 方法。

如果我们想在灰色和蓝色之间切换背景颜色，那么我们可以为此使用一个事件监听器。

我们首先需要访问 `button` 元素和 `body` 元素：

```js
const bodyEl = document.querySelector("body");
const btn = document.getElementById("btn");
```

然后我们需要创建负责在灰色和蓝色之间切换的函数：

```js
let isBgColorGrey = true;

function toggleBgColor() {
  bodyEl.style.backgroundColor = isBgColorGrey ? "blue" : "grey";
  isBgColorGrey = !isBgColorGrey;
}
```

我们使用布尔变量 `isBgColorGrey` 来跟踪当前的背景颜色。如果背景颜色是灰色，则将其更改为蓝色，反之亦然。

然后我们需要为 `button` 元素添加一个事件监听器，以便在点击按钮时调用 `toggleBgColor` 函数：

```js
btn.addEventListener("click", toggleBgColor);
```

我们可以更新超文本标记语言以包含一个段落元素，当鼠标悬停时将用于移除事件监听器：

```html
<link rel="stylesheet" href="styles.css" />
<h1>Event Listener examples</h1>
<p id="para">MouseOver this text to remove the event listener</p>
<button id="btn">Change background color</button>
<script src="index.js"></script>
```

要在鼠标悬停段落时移除事件监听器，我们可以为段落元素添加事件监听器。我们使用 `mouseover` 事件，并传入一个函数，该函数从 `button` 元素中移除事件监听器：

```js
const para = document.getElementById("para");
const btn = document.getElementById("btn");

para.addEventListener("mouseover", () => {
  btn.removeEventListener("click", toggleBgColor);
});
```

当鼠标悬停在段落上时，按钮点击事件的事件监听器将被移除，按钮被点击时背景颜色将不再改变。
