# --description--

型別選擇器（或元素選擇器）會選取所有指定元素型別的元素，而語法就是元素的名稱：

```html
<!-- index.html -->

<div>Hello, World!</div>
<div>Hello again!</div>
<p>Hi...</p>
<div>Okay, bye.</div>
```

```css
/* styles.css */

div {
  color: white;
}
```

在這裡，所有三個 `<div>` 元素都會被選取，而 `<p>` 元素不會。

# --questions--    

## --text--

以下哪一項最能描述上述的 CSS 程式碼？

## --answers--

程式碼將 `white` 顏色套用到 HTML 檔案中的所有元素。

---

程式碼將 `white` 顏色套用到 HTML 檔案中所有的 `div` 元素。

---

程式碼將 `white` 顏色套用到 HTML 檔案中所有的 `p` 元素。
