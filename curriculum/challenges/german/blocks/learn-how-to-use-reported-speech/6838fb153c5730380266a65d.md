`exploited`

### --feedback--

Das bedeutet, dass etwas ausgenutzt wurde, oft auf eine negative oder schädliche Weise. Es endet auf `-ed`.

# --explanation--

Ein `patch` ist ein kleines Update, das von Entwicklerinnen und Entwicklern veröffentlicht wird, um Probleme zu beheben, die Leistung zu verbessern oder Sicherheitslücken zu schließen. Zum Beispiel:

`The company released a patch to fix the login error.` – Das bedeutet, sie haben eine Lösung für das Problem veröffentlicht.

`To exploit` bedeutet, etwas auf eine Weise zu verwenden, die unfair ist oder Schaden verursacht, insbesondere indem eine Schwäche ausgenutzt wird. Zum Beispiel:

`Hackers exploited a flaw in the email server.` – Das bedeutet, sie haben die Schwäche genutzt, um Zugriff zu erhalten oder Schaden anzurichten.
