# --description--

`How much` 用於詢問某些無法以一般方式計數或沒有複數形式的事物的數量或程度。

在這個背景關係中，`code` 指的是某人正在處理的程式設計指令數量，其衡量方式不是透過計算個別條款，而是考量其體積或複雜度。
