# --description--

Tujadili vigezo na hoja katika muktadha wa kitendakazi cha mfano ufuatao:

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

Katika JavaScript, vigezo ni vitu vilivyomo kati ya mabano ya kawaida `()` katika tamko la kitendakazi. Hoja za kitendakazi ni thamani halisi unayoamua kuzipitisha kwa kitendakazi hicho.

Katika mfano hapo juu, ufafanuzi wa kitendakazi umeandikwa kwenye mstari wa kwanza: `function favoriteAnimal(animal)`. Kigezo, `animal`, kinapatikana ndani ya mabano ya kawaida. Unaweza pia kubadilisha `animal` na `pet`, `x`, au `blah`. Lakini katika kesi hii, kuitaja kigezo `animal` kunampa mtu anayesoma msimbo wako muktadha kidogo ili asihitaji kubahatisha kile `animal` kinaweza kuwa nacho baadaye.
