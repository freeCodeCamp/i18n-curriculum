# --explanation--

`Everything`는 여러 개를 가리키더라도 단수 명사로 취급됩니다. `How is everything?`이나 `Everything's fine` 같은 문장에서는 `everything`가 단수 주어로 간주되어 동사가 3인칭 단수형입니다. 예를 들어:

`Everything is ready for the presentation.` – `everything`가 여러 항목이나 작업을 가리키더라도 단수 동사 `is`를 사용합니다.
