# --explanation--

`Third Conditional`는 과거에 일어나지 않은 비현실적인 상황과 그 가능한 결과를 말할 때 사용합니다. 구조는 `If` + `past perfect`, 조동사(`could/might/should`) + `have` + `past participle`입니다. 이 조건문을 사용하면 `if` 절의 행동이 일어나지 않았고 결과도 발생하지 않았지만, 그것이 가능한 결과였다고 상상하는 것입니다. 예를 들어:

`If we had tested the system earlier, we might have prevented the failure.` – 이는 테스트가 이루어지지 않았고 실패가 발생했지만, 테스트가 이루어졌다면 실패를 피할 수 있었을지도 모른다는 뜻입니다.
