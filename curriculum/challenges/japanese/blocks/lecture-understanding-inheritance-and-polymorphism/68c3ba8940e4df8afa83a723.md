# --description--

継承はオブジェクト指向プログラミング（OOP）の次の重要な概念です。

この概念を詳しく見ていき、どのようにコードの再利用を可能にするかを理解しましょう。

継承を使うと、サブクラス（子クラス）がベースクラス（親クラス）の属性やメソッドを利用できます。これにより、コードを再利用し、明確なクラス階層を作成し、すべてを書き直すことなく動作をカスタマイズできます。既存のメソッドを拡張したり、子クラスでオーバーライドしたりしてカスタマイズできます。

継承の基本的な構文は次の通りです。

```py
class Parent:
    # Parent attributes and methods

class Child(Parent):
    # Child inherits, extends, and/or overrides where necessary
```

`Child` クラスが `Parent` クラスを継承するには、ParentをChildに渡す必要があります。

このスタイルは単一継承と呼ばれ、子クラスがちょうど1つの親クラスを継承します。

例を見てみましょう。

```py
class Animal:
    def __init__(self, name):
        self.name = name

    def sound(self):
        return f'{self.name} makes a sound'

class Dog(Animal):
    bark = 'woof! woof!! woof!!!'

jack = Dog('Jack')
print(jack.sound())  # Jack makes a sound
print(jack.bark)  # woof! woof!! woof!!!
```

親の `self.name` クラスから子の `sound()` クラスで `Animal` 属性と `Dog` メソッドを再利用できているのがわかります。

親の `sound()` クラスの `Animal` メソッドを子の `Dog` クラスでオーバーライドして、`sound()` が `bark` クラス変数を使えるようにしましょう。

```py
class Animal:
    def __init__(self, name):
        self.name = name

    def sound(self):
        return f'{self.name} makes a sound.'

class Dog(Animal):
    bark = 'woof! woof!! woof!!!'

    # Override sound() to use bark class variable
    def sound(self):
        return f'{self.name} barks {self.bark}'

jack = Dog('Jack')
print(jack.sound())  # Jack barks woof! woof!! woof!!!
```

`sound()` の戻り値を保持しつつ、後で bark クラス変数を追加したい場合は、`sound()` 関数を使って `super()` を拡張できます。

```py
class Animal:
    def __init__(self, name):
        self.name = name

    def sound(self):
        return f'{self.name} makes a sound'

class Dog(Animal):
    bark = 'woof! woof!! woof!!!'

    # Call Animal.sound(), then append bark
    def sound(self):
        base = super().sound()
        return f'{base}, then {self.name} barks {self.bark}'

jack = Dog('Jack')
print(jack.sound())  # Jack makes a sound, then Jack barks woof! woof!! woof!!!
```

この例では、`base` は `sound()` クラスの `Animal` メソッドを呼び出した結果で、その後 `Dog` クラス固有の音を追加しています。こうすることで、親の `Animal` クラスの元の動作を保ちながら機能を拡張できます。

また、複数の親クラスから継承できる多重継承もあります。

多重継承の基本的な構文は次の通りです。

```py
class Parent:
    # Attributes and methods for Parent

class Child:
    # Attributes and methods for Child

class GrandChild(Parent, Child):
    # GrandChild inherits from both Parent and Child
    # GrandChild can combine or override behavior from each
```

多重継承を簡単に示す例として、陸上を歩き水中を泳ぐことができるカエルがあります。

```py
class Walker:
    def walk(self):
        return 'I can walk on land'

class Swimmer:
    def swim(self):
        return 'I can swim in water'

# Amphibian inherits from both Walker and Swimmer
class Amphibian(Walker, Swimmer):
    def __init__(self, name):
        self.name = name

    def introduce(self):
        return f"I'm {self.name} the frog. {self.walk()} and {self.swim()}."

frog = Amphibian('Freddy')
print(frog.introduce())
# Output: I'm Freddy the frog. I can walk on land and I can swim in water.
```
