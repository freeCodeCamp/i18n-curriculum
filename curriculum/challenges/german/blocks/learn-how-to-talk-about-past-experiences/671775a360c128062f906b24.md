# --explanation--

`Which` ist ein Relativpronomen, das zusätzliche Informationen über etwas gibt, das zuvor im Satz erwähnt wurde. In diesem Fall bezieht sich `which` auf die gesamte Idee, die Updates einen Tag später zu überprüfen, und stellt klar, dass dies nicht ideal war. Zum Beispiel:

`She missed the deadline, which caused a delay in the project.` – Hier bezieht sich `which` auf `missing the deadline` und erklärt die daraus resultierende Verzögerung.

Ähnlich verwendet Jake `which`, um auszudrücken, dass das Überprüfen der Updates einen Tag später keine bewährte Vorgehensweise war.
