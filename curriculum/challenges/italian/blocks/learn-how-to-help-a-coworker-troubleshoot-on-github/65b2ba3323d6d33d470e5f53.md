# --description--

Tom usa `I'll` (abbreviazione di `I will`) per parlare di ciò che ha intenzione di fare più tardi.

`Will` si usa spesso quando qualcuno decide di fare qualcosa in futuro. Per esempio, `I'll call you later` significa che la persona ha in programma di chiamare in un momento successivo a ora.
