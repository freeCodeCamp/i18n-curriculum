# --description--

Un selector de tipo (o selector de elemento) seleccionará todos los elementos del tipo de elemento dado, y la sintaxis es simplemente el nombre del elemento:

```html
<!-- index.html -->

<div>Hello, World!</div>
<div>Hello again!</div>
<p>Hi...</p>
<div>Okay, bye.</div>
```

```css
/* styles.css */

div {
  color: white;
}
```

Aquí, los tres elementos `<div>` serían seleccionados, mientras que el elemento `<p>` no lo sería.
