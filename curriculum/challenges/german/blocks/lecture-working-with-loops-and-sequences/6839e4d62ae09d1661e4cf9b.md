# --description--

In den letzten Lektionen haben Sie sich mit Schleifen wie dieser vertraut gemacht:

```py
even_numbers = []

for num in range(21):
    if num % 2 == 0:
        even_numbers.append(num)

print(even_numbers)
```

Dieses Beispiel erstellt eine neue leere Liste namens `even_numbers` und durchläuft eine Folge von Zahlen zwischen `0` und `20`. Innerhalb der Schleife gibt es eine Bedingung, die überprüft, ob die aktuelle Zahl beim Teilen durch `0` einen Rest von `2` hat. Dies wird verwendet, um zu bestimmen, ob die Zahl gerade ist. Wenn die Bedingung `True` ist, wird die aktuelle `num` am Ende der `even_numbers` Liste angehängt. Schließlich geben wir die `even_numbers` Liste in der Konsole aus.

Obwohl dieser Code funktioniert, gibt es eine prägnantere Möglichkeit, dies zu schreiben, die List Comprehension verwendet. List Comprehension ermöglicht es Ihnen, eine neue Liste in einer einzigen Zeile zu erstellen, indem Sie eine Schleife und eine Bedingung direkt innerhalb eckiger Klammern kombinieren. Das macht den Code kürzer und oft leichter lesbar.

Hier ist das umgeschriebene Beispiel von zuvor mit eckigen Klammern:

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

In diesem umgeschriebenen Beispiel wird die `even_numbers` Liste mit nur einer Codezeile erstellt. Die List Comprehension durchläuft Zahlen von `0` bis `20` und schließt nur diejenigen ein, die durch `2` teilbar sind. Dieser Ansatz ist kompakter und eliminiert die Notwendigkeit für eine separate Schleife und einen bedingten Block.

Schauen wir uns ein weiteres Beispiel an, damit wir besser verstehen, wie List Comprehension funktioniert:

```py
numbers = [1, 2, 3, 4, 5]
result = [(num, 'Even') if num % 2 == 0 else (num, 'Odd') for num in numbers]
print(result)
```

In diesem Beispiel haben wir eine Liste von Zahlen und möchten eine neue Liste von Tuple erstellen, die anzeigt, welche Zahlen gerade oder ungerade sind. Im ersten Teil der List Comprehension verwenden wir eine `if` Anweisung, um zu überprüfen, ob die Zahl ohne Rest durch `2` teilbar ist. Wenn ja, ist das Ergebnis ein Tuple aus dieser Zahl gefolgt von dem Wort `Even`. Andernfalls ist das Ergebnis ein Tuple mit der Zahl gefolgt von dem Wort `Odd`.

So sieht das Ergebnis aus, wenn es in der Konsole ausgegeben wird:

```py
[(1, 'Odd'), (2, 'Even'), (3, 'Odd'), (4, 'Even'), (5, 'Odd')]
```

Eine weitere Möglichkeit, eine Liste aus einem bestehenden iterierbaren Objekt zu erstellen, ist die `filter()` Funktion. Hier ist ein Beispiel, wie man eine neue Liste nur mit Wörtern erstellt, die länger als vier Zeichen sind:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

Die `filter()` Funktion wird verwendet, um Elemente aus einem iterierbaren Objekt auszuwählen, die eine bestimmte Bedingung erfüllen. Die `filter()` Funktion akzeptiert eine Funktion und ein iterierbares Objekt als Argumente. In diesem Beispiel übergeben wir eine `is_long_word` Funktion an die `filter()` Funktion, um zu überprüfen, ob die aktuelle Wortlänge größer als `4` ist. Alle Wörter, die eine Zeichenanzahl größer als `4` haben, werden in eine neue Liste aufgenommen und der Variablen `long_words` zugewiesen.

Abgesehen von der `filter()` Funktion gibt es noch einige weitere Funktionen, die bei der Arbeit mit Listen hilfreich sind. Eine weitere Funktion, die Sie kennen sollten, ist die `map()` Funktion, die ein iterierbares Objekt nimmt und eine Funktion auf jedes seiner Elemente anwendet. Hier ist ein Beispiel für die Verwendung der `map()` Funktion, um eine Liste von Temperaturen von Celsius in Fahrenheit umzuwandeln:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

Genau wie die `filter()` Funktion akzeptiert `map()` eine Funktion und ein iterierbares Objekt als Argumente. Die `to_fahrenheit` Funktion nimmt eine Temperatur und wandelt sie von Celsius in Fahrenheit um.

Die letzte Funktion, die wir uns ansehen, ist die `sum()` Funktion. Diese Funktion wird verwendet, um die Summe aus einem iterierbaren Objekt wie einer Liste oder einem Tuple zu erhalten. Hier ist ein Beispiel für die Verwendung der `sum()` Funktion:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

Sie können auch ein optionales `start` Argument übergeben, das den Anfangswert für die Summierung festlegt. Hier ist ein aktualisiertes Beispiel, das das `start` Argument als Positionsargument verwendet:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

Sie können auch stattdessen das `start` Argument als Schlüsselwortargument verwenden:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

Beide Versionen liefern dasselbe Ergebnis, aber das Schlüsselwortargument ist etwas expliziter.

List Comprehension sowie andere Funktionen wie `map()`, `filter()` und `sum()` mögen anfangs etwas verwirrend erscheinen. Aber mit genügend Übung und Zeit werden Sie sich wohler fühlen, sie in Ihren Python-Programmen zu verwenden.
