# --description--

왼쪽에서 오른쪽으로 진행할 때, 어떤 자리 숫자도 그 왼쪽 숫자보다 작지 않으면 이를 증가하는 수라고 합니다. 예를 들어, 134468이 그렇습니다.

마찬가지로 어떤 자리 숫자도 그 오른쪽 숫자보다 작지 않으면 이를 감소하는 수라고 합니다. 예를 들어, 66420이 그렇습니다.

증가하는 수도 아니고 감소하는 수도 아닌 양의 정수를 "바운시" 수라고 부릅니다. 예를 들어, 155349가 그렇습니다.

n이 커질수록 n 이하의 바운시 수 비율이 증가하여, 백만 이하에는 비바운시 수가 12951개뿐이고, ${10}^{10}$ 이하에는 비바운시 수가 277032개뿐입니다.

구골(${10}^{100}$) 이하의 수 중 비바운시 수는 몇 개일까요?
