# --description--

In einer vorherigen Lektion haben Sie die binäre Suche kennengelernt und erfahren, wie sie sich von der linearen Suche unterscheidet.

In diesem Workshop setzen Sie das in die Praxis um, indem Sie einen Algorithmus für die binäre Suche erstellen.

Beginnen Sie damit, eine `binarySearch`-Funktion mit `searchList`- und `value`-Parametern zu erstellen.

`searchList` ist das Array von Elementen, in dem die Funktion suchen wird, und `value` ist das Ziel, das die Funktion im `searchList` finden soll.
