# --description--

在本实验中，你将构建一个函数，该函数返回一个数组，数组由每个提供的子数组中的最大数字组成。

请记住，你可以用简单的 `for` 循环迭代数组，并用数组语法 `arr[i]` 访问每个成员。

**目标：**实现以下用户需求并通过所有测试以完成实验。

**用户故事：**

1. 你应该创建一个函数 `largestOfAll`，它以一个数组的数组作为参数。
2. 该函数应返回一个包含每个子数组中最大数字的数组。

# --solutions--

```js
function largestOfAll(arr) {
  return arr.map(subArr => Math.max.apply(null, subArr));
}

largestOfAll([
  [4, 5, 1, 3],
  [13, 27, 18, 26],
  [32, 35, 37, 39],
  [1000, 1001, 857, 1]
]);
```
