# --explanation--

Katika **usemi ulioripotiwa**, usemi `said that` hutumika kuwasilisha kile mtu alichosema kwa njia tofauti, bila kutumia maneno yao halisi. Neno `that` ni hiari na mara nyingi halitumiki katika mazungumzo ya kila siku. Kwa mfano:

- Usemi wa moja kwa moja: `We need to update the software.`

- Usemi ulioripotiwa: `He said that we need to update the software.` au `He said we need to update the software.`

Toleo zote mbili ni sahihi na zina maana sawa.
