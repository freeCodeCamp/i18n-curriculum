# --interactive--

عند العمل مع الألوان في CSS، من الضروري فهم نموذج ألوان RGB. يشير RGB إلى الأحمر والأخضر والأزرق — الألوان الأساسية للضوء. تُدمج هذه الألوان الثلاثة بشدة مختلفة لإنشاء مجموعة واسعة من الألوان.

نموذج ألوان RGB هو نموذج ألوان إضافي، ما يعني أن الألوان تُنشأ بدمج الضوء بشدات متفاوتة.

يمكن أن تتراوح شدة كل لون من `0` (مما يعني عدم وجود ضوء) إلى `255` (مما يعني ضوء كامل). بمزج مستويات مختلفة من الأحمر والأخضر والأزرق، يمكنك إنتاج أي لون تراه على شاشتك. على سبيل المثال:

- يُمثّل اللون الأسود بـ `rgb(0, 0, 0)`، مما يعني عدم وجود ضوء لأي من الألوان الثلاثة.
- يُمثّل اللون الأبيض بـ `rgb(255, 255, 255)`، مما يعني شدة ضوء كاملة لكل الألوان الثلاثة.
- يُمثّل اللون الأحمر بـ `rgb(255, 0, 0)`، مع شدة كاملة للأحمر ولا شيء للأخضر أو الأزرق.

في CSS، تتيح لك الدالة `rgb()` تعريف الألوان باستخدام نموذج ألوان RGB. تأخذ الدالة ثلاث قيم، كل واحدة تمثل شدة الضوء الأحمر والأخضر والأزرق على التوالي. هذه هي الصيغة النحوية الأساسية:

```css
element {
  color: rgb(red, green, blue);
}
```

يمكن أن تتراوح قيم الأحمر والأخضر والأزرق من `0` إلى `255`، حيث يمثل `0` غياب ذلك اللون، ويمثل `255` التشبع الكامل. إليك مثالًا على كيفية استخدام دالة `rgb()` في CSS:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

سيغير هذا الشفرة لون نص الفقرة إلى الأحمر لأن قيمة الأحمر مضبوطة على `255`، بينما الأخضر والأزرق مضبوطان على `0`.

يوفر CSS أيضًا دالة `rgba()`، التي تضيف قيمة رابعة — ألفا — تتحكم في شفافية اللون. تتراوح قيمة الألفا من `0` (شفاف تمامًا) إلى `1` (غير شفاف تمامًا). إليك مثالًا على استخدام `rgba()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

سيطبق هذا الشفرة خلفية زرقاء نصف شفافة على عنصر `div` لأن قيم الأحمر والأخضر مضبوطة على `0`، وقيمة الأزرق مضبوطة على `255`، وقيمة الألفا مضبوطة على `0.5` (شفافية 50%).

نموذج ألوان RGB مفيد جدًا للوسائط الرقمية لأنه يتوافق مباشرة مع كيفية عرض الشاشات للألوان. تستخدم الشاشات وحدات بكسل صغيرة حمراء وخضراء وزرقاء لإنشاء الألوان التي تراها. بالتحكم في شدة هذه البكسلات عبر قيم RGB، يمكنك تحقيق مجموعة واسعة من الألوان لتصميم الويب الخاص بك.

بالإضافة إلى ذلك، يعد نموذج RGB مثاليًا عند العمل مع التصاميم الديناميكية. على سبيل المثال، إذا أردت إنشاء رسوم متحركة أو تأثيرات لونية تتطلب تغييرات في الوقت الحقيقي، فإن مرونة دالة RGB تتيح لك تعديل الألوان برمجيًا.

بفهم نموذج ألوان RGB وكيفية استخدام دالتي `rgb()` و `rgba()` في CSS، ستحصل على تحكم أكبر في كيفية عرض الألوان على صفحات الويب الخاصة بك. يمكنك إنشاء تصاميم نابضة بالحياة، وشفافية، وحتى تعديل الألوان برمجيًا لتأثيرات ديناميكية.
