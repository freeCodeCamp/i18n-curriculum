# --description--

En proyectos anteriores aprendiste sobre el método `setAttribute`. Otra forma de actualizar un atributo en JavaScript es utilizar la siguiente sintaxis:

```js
el.attribute = value;
```

Los nombres de propiedades para los valores de atributos HTML guionados, tales como `aria-label`, siga el caso de camel, convirtiéndose en `ariaLabel`.

```js
el.ariaLabel = "Aria Label Value";
```

Establece el atributo `aria-label` para la `input` al mismo valor que el atributo `id`.


# --hints--

Debes tener una `input.ariaLabel`.

```js
assert.match(code, /input\.ariaLabel\s*=/);
```


Debes dar `input.ariaLabel` el mismo valor que `input.id`.

```js
assert.match(code, /input\.ariaLabel\s*=\s*((letter\s*\+\s*number)|(input.id))/);
```
