# --description--

`for`ループは、コードのブロックを複数回実行するために使われる別のタイプのループです。`for`ループは、繰り返し回数が既知の場合に使われます。`for`ループの構文は次の通りです。

```javascript
for (initialization; condition; increment/decrement) {
  // code block to be executed
}
```

例えば、次のコードスニペットは1から5までの数字を出力します。

```javascript
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
```

`for`ループを使って配列を繰り返すには、配列のlengthプロパティを使うことができます。例えば、次のコードスニペットは配列の要素を出力します。

```javascript
const arr = [1, 2, 3, 4, 5];
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}
```
