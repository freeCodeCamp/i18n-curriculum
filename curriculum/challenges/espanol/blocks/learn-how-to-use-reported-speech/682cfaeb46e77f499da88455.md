# --explanation--

En el **discurso indirecto**, `to suggest` se usa para compartir consejos o ideas que alguien dio. Por ejemplo:  

- Discurso directo: `I think we should update our firewall.`

- Estilo indirecto: `Jake suggested updating the firewall.`

Esto muestra que alguien dio un consejo o una posible solución sin usar sus palabras exactas.
