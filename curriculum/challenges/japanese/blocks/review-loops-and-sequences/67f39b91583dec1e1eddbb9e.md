## Pythonのリスト

- **概要**: Pythonでは、リスト型は文字列、数値、または他のリストを含むことができる順序付きの要素のシーケンスです。リストは変更可能で、ゼロベースのインデックスが付けられています。

```python
cities = ['Los Angeles', 'London', 'Tokyo']
```

- **リストの要素へのアクセス**: `cities` リストの要素にアクセスするには、シーケンス内のインデックス番号を参照します。

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # Los Angeles
```

- **負のインデックスを使った要素へのアクセス**: 任意のリストの最後の要素にアクセスするには、インデックス番号として `-1` を使います。

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # Tokyo
```

- 負のインデックスは、リストの先頭のインデックス `0` ではなく、末尾から要素にアクセスするために使われます。

- **`list()` コンストラクタを使ったリストの作成**: リストは `list()` コンストラクタを使っても作成できます。`list()` コンストラクタはイテラブルをリストに変換するために使われます。

```python
developer = 'Jessica'

print(list(developer)) 
# Result: ['J', 'e', 's', 's', 'i', 'c', 'a']
```

- **リストの長さを調べる**: `len()` 関数を使ってリストの長さを取得できます。

```python
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

- **リストの変更可能性**: リストは変更可能で、有効なインデックス番号を指定すれば任意の要素を更新できます。特定のインデックスのリストを更新するには、そのインデックスに新しい値を代入します。

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

- **インデックス範囲外エラー**: リストの範囲外のインデックス（正または負）を指定すると、`IndexError` が発生します。

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
"""
```

- **リストから要素を削除する**: `del` キーワードを使ってリストから要素を削除できます。

```python
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

- **リストに要素が存在するか確認する**: `in` キーワードを使ってリストに要素が存在するか確認できます。

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **リストのネスト**: リストは他のリストの中にネストできます。

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

- ネストされたリストにアクセスするには、リストがゼロベースのインデックスなので `2` のインデックスを使います。

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

- さらにそのネストされたリストから2番目の言語にアクセスするには、インデックス `1` を使います。

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # Rust
```

- **リストからの値のアンパック**: リストから値をアンパックする手法は、リストの値を新しい変数に代入するために使われます。例えば、`developer` リストを `name`、`age`、`job` という新しい変数にアンパックする例は次の通りです。

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer
```

- 代入演算子の左側の変数の数がリスト内のアイテム数と一致しない場合、`ValueError` が発生します。

- **リストの残りの要素をまとめる**: リストの残りの要素をまとめるには、アスタリスク（`*`）演算子を使います。

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer
```

- **リストのスライス**: スライスは `:` スライス演算子を使ってリストの一部にアクセスする概念です。インデックス `1` から始まり `3` の直前までのリストをスライスするには、次の構文を使います。

```python
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie']
desserts[1:3] # ['Cookies', 'Ice Cream']
```

- **ステップ間隔**: インデックス間の増分を指定するステップ間隔も設定できます。例えば、スライスを使って偶数だけのリストを抽出する例は次の通りです。

```python
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

## リストのメソッド

- **append()**: リストの末尾にアイテムを追加するために使います。`append()` メソッドを使って `6` リストに数値 `numbers` を追加する例は次の通りです。

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **リストの追加**: `append()` メソッドは、あるリストの末尾に別のリストを追加するためにも使えます。

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

- **extend()**: 複数のアイテムをリストの末尾に追加するために使います。`6` リストの末尾に `8`、`10`、`numbers` の数値を追加する例は次の通りです。

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

- **insert()**: リストの特定のインデックスにアイテムを挿入するために使います。`insert()` メソッドを使った例は次の通りです。

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

- **remove()**: リストからアイテムを削除するために使います。`remove()` メソッドはリスト内の最初の出現だけを削除します。

```py
numbers = [1, 2, 3, 4, 5, 5, 5]
numbers.remove(5)

print(numbers) # [1, 2, 3, 4, 5, 5]

```

- **pop()**: リストから特定のアイテムを削除して返します。

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

- `pop` メソッドで要素を指定しない場合は、最後の要素が削除されます。

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

- **clear()**: リストのすべてのアイテムを削除します。

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

- **sort()**: `sort()` メソッドは要素をその場でソートします。ランダムな `numbers` のリストをその場でソートする例は次の通りです。

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

- **sorted()**: 元のリストを変更せずに新しいソート済みリストを返す組み込み関数です。

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
print(numbers) # [19, 2, 35, 1, 67, 41]
```

- **reverse()**: リストの要素の順序を逆にします。

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **index()**: リスト内で要素が最初に見つかるインデックスを調べるために使います。

```python
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

- `index()` メソッドで要素が見つからない場合、結果は `ValueError` になります。

## Pythonのタプル

- **定義**: タプルはPythonのデータ型で、順序付きの値のシーケンスを作成するために使います。タプルは異なるデータ型の混在も可能です。

```py
developer = ('Alice', 34, 'Rust Developer')
```

- タプルは変更不可能で、一度作成した要素は変更できません。要素を更新しようとすると `TypeError` が発生します。

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **タプルの要素へのアクセス**: タプルの要素にアクセスするにはブラケット記法とインデックス番号を使います。

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

- 負のインデックスを使ってタプルの末尾から要素にアクセスできます。

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- タプルの長さを超えるか等しいインデックス番号を指定すると `IndexError` が発生します。

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- タプルは `tuple()` コンストラクタを使っても作成できます。コンストラクタ内には文字列、リスト、他のタプルなどのイテラブルを渡せます。

```python
developer = 'Jessica'

print(tuple(developer)) 
# Result: ('J', 'e', 's', 's', 'i', 'c', 'a')
```

- **タプル内のアイテムの確認**: タプルにアイテムがあるか確認するには `in` キーワードを使います。

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **タプルのアンパック**: タプルからアイテムをアンパックする例は次の通りです。

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

- タプルの残りの要素をまとめるにはアスタリスク（`*`）演算子を使います。

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer
```

- **タプルのスライス**: スライスを使ってタプルの一部を抽出できます。例えば、`pie` と `cookies` のアイテムを別のタプルにスライスする例です。

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

- **タプルからのアイテム削除**: タプルは変更不可能なので、アイテムを削除しようとすると `TypeError` が発生します。

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

- **タプルとリストの使い分け**: 要素の追加、削除、更新が必要な動的なコレクションにはリストを使います。固定で変更不可のコレクションにはタプルを使います。

## タプルのよく使うメソッド

- **`count()`**: タプル内のアイテムが何回出現するか調べるために使います。例えば、言語 `'Rust'` がタプル内に何回あるか調べる例です。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

- `count()` メソッドで指定したアイテムがタプルに全く存在しない場合、戻り値は `0` になります。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

- `count()` メソッドに引数を渡さない場合、Pythonは `TypeError` を返します。

- **index()**: タプル内の特定のアイテムがあるインデックスを調べるために使います。言語 `index()` のインデックスを調べる `'Java'` メソッドの例は次の通りです。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

- 指定したアイテムが見つからない場合、Pythonは `ValueError` を返します。

- `index()` メソッドには検索開始位置を指定するオプションの開始インデックスを渡せます。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

- `index()` メソッドには検索終了位置を指定するオプションの終了インデックスも渡せます。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

- **`sorted()`**: 任意のイテラブルの要素をソートして新しいソート済みリストを返します。`sorted()` 関数を使って新しい数値リストを作成する例は次の通りです。

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

- **ソート動作の変更**: オプションの `reverse` と `key` 引数を使ってイテラブルのソート動作をカスタマイズできます。`key` 引数を使ってタプルのアイテムを長さでソートする例は次の通りです。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

- `reverse` 引数を使って逆順の新しいリストを作成できます。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

## Pythonのループ

- **定義**: ループはコードのブロックを決まった回数繰り返すために使います。

- **`for` ループ**: シーケンス（リスト、タプル、文字列など）を繰り返し処理し、そのシーケンスの各アイテムに対してコードブロックを実行します。`for` ループを使ってリストを繰り返し処理し、各言語をコンソールに出力する例は次の通りです。

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)

"""
Result 

Rust
Java
Python
C++
"""
```

- `for` ループを使って文字列 `code` を繰り返し処理し、各文字を出力する例は次の通りです。

```py
for char in 'code':
    print(char)

"""
Result 

c
o
d
e
"""
```

- `for` ループはネストできます。ネストされた `for` ループの例は次の通りです。

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)

"""
Result

Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
"""

```

- **`while` ループ**: 条件が `False` になるまでコードブロックを繰り返します。`while` ループを使った推測ゲームの例は次の通りです。

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')

"""
Result

Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
"""

```

- **`break` と `continue` 文**: ループの実行を変更するために使います。

- `break` 文は特定の条件が満たされたときにループを即座に終了します。`break` 文を使って `developer_names` のリストを処理する例は次の通りです。

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

- `continue` 文は現在の繰り返しをスキップして次の繰り返しに進みます。`continue` 文の代わりに `break` 文を使う例は次の通りです。

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

- `for` と `while` ループは `else` 節と組み合わせることができ、ループが `break` で終了しなかった場合にのみ実行されます。

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

## ループでのrangeの使い方

- **`range()` 関数**: 整数のシーケンスを生成するために使います。

```py
range(start, stop, step)
```

- 必須の `stop` 引数は整数（非包含）で、生成される数列の終点を表します。`range()` 関数を使った例は次の通りです。

```py
for num in range(3):
    print(num)
```

- `start` 引数が指定されない場合、デフォルトは `0` になります。デフォルトでは整数のシーケンスは `1` ずつ増加します。オプションの `step` 引数を使って増分値を変更できます。例えば、2から11未満（10を含む）までの偶数のシーケンスを生成する例は次の通りです。

```py
for num in range(2, 11, 2):
    print(num)
```

- `range()` 関数に引数を渡さないと `TypeError` になります。

- `range()` 関数は引数に整数のみを受け付け、浮動小数点数は受け付けません。浮動小数点数を使うと `TypeError` になります。

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- `step` 引数に負の整数を使うと、減少する整数のシーケンスを生成できます。

```py
for num in range(40, 0, -10):
    print(num)
```

- `range()` 関数は `list` コンストラクタと組み合わせて整数のリストを作成することもできます。`list` コンストラクタはイテラブルをリストに変換します。2から10までの偶数のリストを生成する例は次の通りです。

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

## Pythonの`enumerate()` と `zip()` 関数

- **`enumerate()`**: シーケンスを繰り返し処理し、そのシーケンスの各アイテムのインデックスを追跡します。`enumerate()` 関数はイテラブルを引数に取り、イテラブル内の各アイテムのインデックスと値からなる `enumerate` オブジェクトを返します。

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')

# Result
# Index 0 and language Spanish
# Index 1 and language English
# Index 2 and language Russian
# Index 3 and language Chinese

```

- `enumerate()` 関数は `for` ループの外でも使えます。

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

print(list(enumerate(languages)))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

- `enumerate()` 関数はオプションの `start` 引数を受け付け、カウントの開始値を指定します。この引数を省略するとカウントは `0` から始まります。

- **`zip()`**: 複数のイテラブルを並列に繰り返し処理します。`zip()` 関数を使って `developers` と `ids` を繰り返し処理する例は次の通りです。

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')


"""
Result

Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
"""

```

## Pythonのリスト内包表記

- **定義**: リスト内包表記は、ループと条件を大括弧内に直接組み合わせて1行で新しいリストを作成する方法です。コードが短くなり、読みやすくなることが多いです。

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

## イテラブルのメソッド

- **`filter()`**: 条件に基づいてイテラブルから要素をフィルタリングします。条件を満たす要素だけを含むイテレータを返します。4文字より長い単語だけの新しいリストを作成する例は次の通りです。

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**: イテラブルの各アイテムに関数を適用し、結果の新しいイテラブルを返します。`map()` 関数を使って摂氏温度のリストを華氏に変換する例は次の通りです。

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**: リストやタプルのようなイテラブルの合計を取得します。`sum()` 関数を使った例は次の通りです。

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- オプションの `start` 引数を渡して合計の初期値を設定できます。`start` 引数を位置引数として使った更新例は次の通りです。

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- `start` 引数をキーワード引数として使う例は次の通りです。

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## ラムダ関数

- **定義**: Pythonのラムダ関数は名前のない（無名）関数を簡潔に作成する方法です。  
- ラムダ関数は他の関数の引数としてよく使われます。ラムダ関数の例は次の通りです。

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

- ラムダ関数のベストプラクティスは、変数に代入しないこと、シンプルで読みやすく保つこと、短く一時的な関数に使うことです。
