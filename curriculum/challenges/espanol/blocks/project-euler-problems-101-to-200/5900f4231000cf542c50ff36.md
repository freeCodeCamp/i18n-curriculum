# --description--

Sea $N$ un entero positivo y sea $N$ dividido en $k$ partes iguales, $r = \frac{N}{k}$, de modo que $N = r + r + \cdots + r$.

Sea $P$ el producto de estas partes, $P = r × r × \cdots × r = r^k$.

Por ejemplo, si 11 se divide en cinco partes iguales, 11 = 2.2 + 2.2 + 2.2 + 2.2 + 2.2, entonces $P = {2.2}^5 = 51.53632$.

Sea $M(N) = P_{max}$ para un valor dado de $N$.

Resulta que el máximo para $N = 11$ se encuentra dividiendo once en cuatro partes iguales, lo que conduce a $P_{max} = {(\frac{11}{4})}^4$; es decir, $M(11) = \frac{14641}{256} = 57.19140625$, que es un decimal finito.

Sin embargo, para $N = 8$ el máximo se logra dividiéndolo en tres partes iguales, por lo que $M(8) = \frac{512}{27}$, que es un decimal no periódico.

Sea $D(N) = N$ si $M(N)$ es un decimal no periódico y $D(N) = -N$ si $M(N)$ es un decimal periódico.

Por ejemplo, $\sum D(N)$ para $5 ≤ N ≤ 100$ es 2438.

Encuentra $\sum D(N)$ para $5 ≤ N ≤ 10000$.
