# --interactive--

L'elemento `button` viene usato per eseguire un'azione particolare quando viene attivato. Ecco un esempio di un elemento `button` con il testo del pulsante `Start Game`.

```html
<button>Start Game</button>
```

Altri esempi di utilizzo dell'elemento `button` includono l'invio di un modulo, la visualizzazione di una finestra modale o l'apertura e chiusura di un menu laterale. L'elemento `button` ha un attributo `type` che controlla il comportamento del pulsante quando viene attivato. Il primo valore possibile per l'attributo `type` sarebbe il tipo `button`. Ecco un esempio di utilizzo dell'elemento `button` con il tipo `button` e un testo `Show Alert`:

```html
<button type="button">Show Alert</button>
```

Per impostazione predefinita, il pulsante non fa nulla quando viene attivato. Tuttavia, puoi aggiungere del codice JavaScript per rendere il pulsante interattivo, come mostrare un avviso in questo caso.

Clicca sul pulsante `Show Alert` per vedere comparire un avviso sullo schermo. Per interagire con l'esempio, devi abilitare l'editor interattivo.


**NOTA**: Questo esempio interattivo usa JavaScript ma non devi preoccuparti di capire il codice JavaScript. Imparerai JavaScript in moduli futuri.

:::interactive_editor

```html
<button type="button">Show Alert</button>
<script src="index.js"></script>
```

```js
const btn = document.querySelector("button");
btn.addEventListener("click", () => alert("You clicked on the alert button"));
```

:::

Un altro valore possibile per l'attributo `type` è il valore `submit`. Ecco un esempio di utilizzo di un elemento `button` con il tipo `submit`.

:::interactive_editor

```html
<form action="">
  <label for="email">Email address:</label>
  <input type="email" id="email" name="email" />
  <button type="submit">Submit form</button>
</form>
```

:::

All'interno di questo elemento `form` ci sono un elemento `label` e un elemento `input` per l'indirizzo email dell'utente. Quando l'utente clicca sul pulsante di invio, i suoi dati verranno inviati al server e processati. Il terzo valore possibile per l'attributo `type` è il valore `reset`. Ecco un esempio di un elemento `form` con pulsanti di reset e invio.

Abilita l'editor interattivo e interagisci con l'input email nella finestra di anteprima fornendo un indirizzo email falso. Poi clicca sul pulsante di reset per vedere la tua email sparire dal campo.

:::interactive_editor

```html
<form action="">
  <label for="email">Email address:</label>
  <input type="email" id="email" name="email" />
  <button type="reset">Reset form</button>
  <button type="submit">Submit form</button>
</form>
```

:::

In questo esempio modificato, un elemento `label` e un elemento `input` sono usati per raccogliere l'indirizzo email dell'utente. Quando l'utente clicca sul pulsante di reset, tutti i dati inseriti verranno cancellati. È importante notare che i pulsanti di reset di solito non sono la scelta migliore perché potrebbero portare gli utenti a resettare accidentalmente i loro dati. Inoltre, più pulsanti nel modulo potrebbero rendere l'interfaccia utente più confusa.

Un altro modo per creare pulsanti in HTML è usare l'elemento `input`. L'elemento `input` ha anche un attributo `type` con i valori possibili `submit`, `reset` e `button`. Ecco un esempio di utilizzo dell'elemento `input` con il `type` impostato su `button`:

:::interactive_editor

```html
<input class="start-btn" type="button" value="Start Game" />
<script src="index.js"></script>
```

```js
document.addEventListener("DOMContentLoaded", () => {
  const btn = document.querySelector(".start-btn");
  btn.addEventListener("click", () => {
    const paraEl = document.createElement("p");
    const bodyEl = document.querySelector("body");

    bodyEl.appendChild(paraEl);
    paraEl.textContent = "The game has started!!!";
  });
});
```

:::

L'attributo `value` viene usato per mostrare il testo del pulsante. Allora, qual è la differenza tra usare gli elementi `input` e `button`? Gli elementi `input` sono elementi void, il che significa che non possono avere nodi figli, come testo, e possono avere solo un tag di apertura. D'altra parte, l'elemento `button` offre più flessibilità perché puoi annidare testo, immagini e icone al suo interno.
