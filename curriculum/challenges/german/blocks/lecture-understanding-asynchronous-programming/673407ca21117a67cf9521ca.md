# --description--

Ein `Promise` ist ein Objekt, das die endgültige Fertigstellung oder das Scheitern einer asynchronen Operation darstellt. Es befindet sich zunächst in einem ausstehenden Zustand. Anschließend kann es entweder in einen erfüllten Zustand übergehen, wenn die Operation erfolgreich ist, oder in einen abgelehnten Zustand, wenn die Operation fehlschlägt. Hier ist ein Beispiel für das Erstellen eines `Promise`:

```js
const aPromise = new Promise((resolve, reject) => {
  setTimeout(() => {
    resolve("Operation successful!");
  }, 1000);
});
```

In diesem Beispiel erstellen wir ein Promise, das eine asynchrone Operation mit `setTimeout` simuliert. Nach einer Sekunde wird das Promise mit der Nachricht `Operation successful!` aufgelöst.

Eine weitere Möglichkeit, mit Promises zu arbeiten, ist die Verwendung der `.then`- und `.catch`-Methoden.

Die `.then()`-Methode wird in einem Promise verwendet, um anzugeben, was passieren soll, wenn das Promise erfüllt wird, während `.catch()` dazu dient, Fehler zu behandeln, die auftreten können. Sehen wir uns an, wie wir `.then` und `.catch` mit unserem Promise verwenden können:

```js
aPromise.then((result) => {
  console.log(result);  // Outputs: "Operation successful!"
}).catch((error) => {
  console.error(error);
});
```

In diesem Code oder den Anweisungen, was zu tun ist, wenn das Promise erfüllt wird, wird die an `.then()` übergebene Funktion mit dem aufgelösten Wert des Promise aufgerufen. Wenn ein Fehler auftritt, wird stattdessen die an `.catch()` übergebene Funktion aufgerufen.

Kommen wir nun zur Promise-Verkettung. Eine der mächtigen Funktionen von Promises ist, dass wir mehrere asynchrone Operationen hintereinander verketten können. Jeder `.then()` kann ein neues Promise zurückgeben, sodass Sie eine Folge von asynchronen Operationen nacheinander ausführen können. Hier ist ein Beispiel:

```js
fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => {
    console.log(data);
    return fetch('https://api.example.com/data2');
  })
  .then(response => response.json())
  .then(data2 => console.log(data2))
  .catch(error => console.error('Error:', error));
```

In diesem Beispiel führen wir zwei API-Aufrufe nacheinander aus. Der erste `.then()` parst die Antwort als JSON. Der zweite `.then()` gibt die Daten aus und macht einen weiteren API-Aufruf. Der dritte `.then()` parst die Antwort des zweiten API-Aufrufs, und der vierte `.then()` gibt diese Daten aus. Wenn an irgendeinem Punkt in dieser Kette ein Fehler auftritt, wird er vom `.catch()` am Ende abgefangen.

Es ist wichtig zu beachten, dass `.catch()` Fehler aus jedem der vorherigen Schritte in der Kette abfängt. Das bedeutet, dass Sie nicht für jeden einzelnen Schritt eine Fehlerbehandlung hinzufügen müssen, was Ihren Code erheblich vereinfachen kann.
