# --explanation--

Cuando alguien dice que está `struggling with` algo, significa que está teniendo dificultad o problemas con ello. Por ejemplo:

- El equipo está teniendo problemas para cumplir con la fecha límite.

`IDEs` significa `Integrated Development Environments`. Estas son herramientas de software que los desarrolladores usan para escribir, probar y depurar código. Ejemplos de `IDEs` populares incluyen Visual Studio Code, IntelliJ IDEA y Eclipse.

En el diálogo, Tom tiene problemas con sus IDEs y quiere hablar con Sarah al respecto.
