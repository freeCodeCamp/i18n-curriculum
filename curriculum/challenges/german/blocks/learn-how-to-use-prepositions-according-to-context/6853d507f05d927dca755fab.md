# --explanation--

`Currently` bedeutet „im gegenwärtigen Moment“. Zum Beispiel:

`We currently have three open positions.` – Das bedeutet, dass es gerade jetzt drei offene Stellen gibt.

`Working on` bedeutet, mit einer bestimmten Aufgabe oder einem Projekt beschäftigt zu sein. Zum Beispiel:

`She is working on the design for the new homepage.` – Das bedeutet, dass sie es aktiv entwirft.
