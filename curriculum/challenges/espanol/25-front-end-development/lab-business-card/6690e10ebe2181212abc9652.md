# --description--

Cumple las historias de usuario a continuación y haz que todas las pruebas se aprueben para completar el laboratorio.

**Instrucciones:**

1. Deberías enlazar la hoja de estilo `styles.css` usando la etiqueta `link` en la sección `head` del archivo HTML.
1. El fondo de tu página debería ser `rosybrown`. La fuente general debe ser `Arial`, con una alternativa de `sans-serif`.
1. Todos los elementos `p` deben tener un margen superior e inferior de `5px`.
1. Todos los enlaces en la página no deberían estar subrayados.
1. Deberías tener un `div` con un atributo `class` con un valor de `business-card` que contenga todos los otros elementos.
1. El selector `.business-card` debería tener propiedades para establecer el elemento con un ancho de `300px` y un color de fondo a tu elección. También deberías establecer un `padding` de `20px` alrededor y un margen superior de `100px`. El texto debería estar alineado al centro y el tamaño de la fuente debería ser `16px`. Los márgenes izquierdo y derecho deben establecerse en `auto`.
1. Dentro del elemento `.business-card`, debe haber un elemento `img` con un atributo `class` con un valor de `profile-image`. Puedes establecer la fuente de la imagen en `https://cdn.freecodecamp.org/curriculum/labs/flower.jpg` si lo deseas. Debe haber un `alt` con una descripción significativa.
1. El selector `.profile-image` debería tener una propiedad `max-width` con un valor de `100%`.
1. Dentro del elemento `.business-card`, después del elemento `img`, debería haber tres elementos `p` con un atributo `class` que tenga, respectivamente, un valor de `full-name`, `designation`, y `company`.
1. El primer elemento `p` debe contener tu nombre.
1. El segundo elemento `p` debe contener tu designación.
1. El tercer elemento `p` debe tener el nombre de tu empresa.
1. Debe haber un elemento `hr` debajo del tercer elemento `p`.
1. Después del elemento `hr`, debe haber dos elementos `p`. En el primer elemento `p`, deberías tener una dirección de correo electrónico como texto. En el segundo elemento `p`, el texto debe ser un número de teléfono. Después de los dos elementos `p`, debe haber un elemento `a` con el texto `Portfolio` que enlace a una URL válida. Agrega un elemento `hr` después del elemento `a` que contiene el enlace del portafolio.
1. Deberías tener otro elemento `div` con un atributo `class` con un valor de `social-media`. Dentro de este elemento, debe haber un elemento `h2` con el texto `Connect with me`.
1. Dentro del os elementos `.social-media`, debe haber tres elementos `a`. Los elementos `a` deben tener el texto `Twitter`, `LinkedIn` y `GitHub` respectivamente con enlaces a sitios web válidos.
