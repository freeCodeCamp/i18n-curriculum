# --explanation--

`That's true` 是用來同意某人的述語或承認他們所說的內容正確的片語。例如：

「That's true, we do need more time to finish the project。」－ 這表示同意需要更多時間來完成專案。  

`Cost-effective` 意味著某物在花費的金錢上提供良好的價值或有助於節省成本，同時仍能達到良好的結果。例如：

`Using cloud services can be more cost-effective than buying expensive servers。`－這表示雲端服務可以在仍然有用的同時幫助節省金錢。
