# --description--

Usualmente, `with` se utiliza para mostrar conexión o uso. Por ejemplo, `I write with a pen.` Sin embargo, en algunas oraciones, `with` aparece después de la palabra con la que se usa. Esto usualmente ocurre cuando se desea enfatizar el objeto o cosa que alguien está usando o manejando, especialmente en preguntas o declaraciones sobre tecnología o trabajo.

Por ejemplo, en tecnología, podrías decir, `This program I work with is challenging.` Aquí, `with` se utiliza al final para enfatizar el programa que es desafiante manejar. Otro ejemplo podría ser una pregunta como, `What else is he working with?` donde `with` al final pregunta sobre otras herramientas o tecnologías que se están utilizando.
