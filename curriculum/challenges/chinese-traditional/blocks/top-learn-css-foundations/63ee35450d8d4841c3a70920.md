# --description--

內部 CSS（或內嵌 CSS）是指將 CSS 直接加入 HTML 檔案中，而不是建立一個完全獨立的檔案。使用內部方法時，你會將所有規則放在一對開啟和關閉的 `<style>` 頁籤內，然後將這對 `<style>` 頁籤放置在 HTML 檔案的開啟和關閉 `<head>` 頁籤內。由於樣式是直接放在 `<head>` 頁籤內，因此不再需要外部方法所需的 `<link>` 元素。

除了這些差異之外，語法與外部方法（選取器、花括號、宣告）完全相同：

```html
<head>
  <style>
    div {
      color: white;
      background-color: black;
    }

    p {
      color: red;
    }
  </style>
</head>
<body>...</body>
```

這個方法可以用來為網站的單一頁面添加獨特的樣式，但它不像外部方法那樣保持分離，並且根據規則和宣告的數量，可能會導致 HTML 檔案變得相當大。

## --text--

以下哪一項是內部與外部 CSS 方法之間的差異？

## --answers--

外部方法將 CSS 規則放在獨立的檔案中，而內部方法則將 CSS 規則放在 HTML 檔案內。

---

內部方法將 CSS 與 HTML 分開，而外部方法則將 CSS 直接嵌入 HTML。

---

內部方法使用 `<link>` 元素將 CSS 聯結到 HTML，而外部方法則直接在 HTML 中嵌入 CSS。
