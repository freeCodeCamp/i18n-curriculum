# --description--

A frase `crazy part` é usada para expressar que algo é surpreendente ou inesperado.É como dizer `the surprising thing is` ou `the unexpected thing is`.

Por exemplo, `The crazy part is that he finished all the work in one day.` Isso significa que é surpreendente que ele tenha conseguido fazer isso tão rápido.
