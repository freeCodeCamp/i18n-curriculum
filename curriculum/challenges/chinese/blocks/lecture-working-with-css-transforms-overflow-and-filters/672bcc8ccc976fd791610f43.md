# --interactive--

CSS `transform` 属性是一个功能强大的工具，允许你修改网页上元素的视觉呈现，而不会影响其他元素的布局。 它可以对元素进行各种变换，如旋转、缩放、倾斜或在二维或三维空间中平移（移动）。

`transform` 属性的工作原理是对元素的坐标系应用数学变换。 这意味着你可以在保持元素原始位置和文档流不变的情况下，对其形状和位置进行操作。

让我们来探索一些常见的变换函数。 下面是一个盒子元素的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
body {
  border: 2px solid black;
}

.box {
  width: 200px;
  height: 200px;
  background-color: red;
}
```

:::

我们已将 `body` 设置为实线黑色边框，以便你能看到嵌套在 `body` 元素中的 `.box` 元素。

`translate` 函数将元素从其当前位置移动。 下面是一个使用 `translate` 函数的更新示例：  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
body {
  border: 2px solid black;
}

.box {
  width: 200px;
  height: 200px;
  background-color: red;
  transform: translate(50px, 100px);
}
```

:::

这条 CSS 规则将把类为 `box` 的元素从原来的位置向右移动 50 像素，向下移动 100 像素。

`rotate` 函数围绕一个固定点旋转元素，这是一个对 `.box` 元素使用 `rotate` 函数的示例：  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: rotate(45deg);
}
```

:::

这将使元素顺时针旋转 45 度。

`scale` 函数能让你改变元素的大小。 这是一个示例：  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: scale(1.5, 2);
}
```

:::

这将使元素的宽度和高度分别是原来的 1.5 倍和 2 倍。

你可以在一个声明中组合多个变换：  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: translate(50px, 50px) rotate(45deg) scale(1.5);
}
```

:::

这将使元素向右向下分别移动 50 像素，旋转 45 度，并缩放为原始大小的 1.5 倍。

虽然 `transform` 属性在创建具有视觉吸引力的设计方面功能强大，但在使用时必须考虑可访问性。 以下是一些需要牢记的重要的可访问性问题。 

屏幕阅读器可能无法准确传达变换后的内容。 例如，如果你使用 `transform` 重新排列元素的视觉顺序，屏幕阅读器仍将按照原始 DOM 顺序读取内容。 这可能会给依赖屏幕阅读器的用户带来困惑。

使用 `scale` 调整文字大小时，请注意不要使文字过小或过大。 过小的文本可能难以阅读，而过大的文本可能会溢出容器，变得无法阅读。 一般来说，最好使用适当的字体样式技术来调整文字大小。

如果你使用 `transform` 制作动画效果，请注意那些对动效敏感的用户。 对某些人来说，过多或过快的动画会引起不适，甚至引发癫痫发作。 考虑为用户提供减少或关闭动画的途径。 在使用 3D 变换时，请记住并非所有用户都能以相同的方式感知深度。 确保通过 3D 效果传达的任何关键信息也能以 2D 格式或文字形式提供。

如果你使用 `transform` 隐藏或显示内容，请确保屏幕阅读器和键盘导航仍可访问内容。 隐藏内容应真正隐藏，例如使用 `display: none;` 或 `visibility: hidden;`，而不仅仅是在视觉上移到屏幕外。 

在对按钮或链接等交互式元素应用 `transform` 时，请确保可点击区域保持直观且易于定位。 大幅变换的按钮可能会在视觉上造成混乱或难以点击，尤其是对有运动障碍的用户而言。

总之，虽然 CSS `transform` 属性是创建视觉动态网页设计的强大工具，但必须负责任地使用它，并考虑无障碍。始终使用各种辅助技术测试你的变换元素，并考虑为可能受变换影响的信息或功能性提供备选访问方式。
