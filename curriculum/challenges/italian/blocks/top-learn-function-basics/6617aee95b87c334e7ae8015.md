# --description--

Se una funzione viene chiamata, ma non viene fornito un argomento, allora al parametro viene assegnato il valore `undefined`. Questo può portare a risultati inattesi se la funzione non è progettata per gestire questo caso.

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal()); // Output: undefined is my favorite animal!
```

Nell'esempio sopra, la funzione `favoriteAnimal` viene chiamata senza un argomento. Al parametro `animal` viene assegnato il valore `undefined`, che viene poi concatenato con la stringa `" is my favorite animal!"`.

Se non viene fornito un valore, puoi usare i parametri predefiniti. I parametri predefiniti ti permettono di assegnare un valore predefinito a un parametro se non viene fornito alcun valore quando la funzione viene chiamata.

```js
function favoriteAnimal(animal = 'Giant Panda') {
  return animal + " is my favorite animal!"
}
```

Nell'esempio sopra, il valore predefinito del parametro `animal` è impostato su `'Giant Panda'`. Se la funzione `favoriteAnimal` viene chiamata senza un argomento, al parametro `animal` verrà assegnato il valore `'Giant Panda'`.

Impediscono che la funzione venga chiamata senza il numero richiesto di argomenti.

Garantiscono che la funzione restituisca `null` per indicare che non è stato passato alcun argomento.

Permettono alla funzione di eseguire con un valore predefinito, evitando problemi legati a `undefined`.
