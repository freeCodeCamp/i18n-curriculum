# --interactive--

CSS 重置是一種樣式表，用來移除瀏覽器對 HTML 元素套用的全部或部分預設格式。舉例來說，你可能注意到某些元素，如段落和標題，即使你沒有在訂製樣式表中明確定義，它們預設就已經有邊距。 

你也會在各種 HTML 元素的多種屬性中看到這種情況。不同瀏覽器也可能對 HTML 元素套用不同的預設樣式。Google Chrome 中的預設樣式可能與 Mozilla Firefox 或 Microsoft Edge 中的並不完全相同。這可能導致瀏覽器之間的樣式不一致，應盡可能避免。

為了控制代碼，CSS 重置會移除預設值的樣式。透過移除全部或部分預設值的樣式，你可以為你的設計取得一致的基準，並將跨瀏覽器和裝置的潛在不一致性降到最低。移除預設值的樣式也能讓樣式設定過程更簡單，因為你只會看到你在訂製樣式表中明確套用的樣式。CSS 重置有兩種主要方法：你可以定義訂製的 CSS 重置，或使用第三方的 CSS 重置。

訂製的 CSS 重置是你從頭創建的樣式表，以符合你的專案需求。這樣你可以控制將被重置的特定樣式，並擁有很大的彈性空間。然而，你也需要投入時間來開發和維護這些樣式表。以下是一個非常常見的 CSS 規則範例，用於重置所有 HTML 元素的 margin 和 padding：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Example Heading</h1>
<p>This is a paragraph.</p>
```

```css
* {
  margin: 0;
  padding: 0;
}
```

:::

它通常寫在 CSS 樣式表的頂部。星號選擇器是一種萬用字元選擇器，會匹配所有 HTML 元素，因此它們在四個邊的預設外距和預設內距都是零。這會給你一個起點，然後你可以在樣式表中更下方的更具體的 CSS 選擇器中進一步自訂它們。

你可以使用此方法來選取任何 HTML 元素並重設其預設樣式。只要創建一個訂製的樣式表，並使用適當的 CSS 選擇器來匹配元素並設定樣式。 

但這可能是一個耗時的處理程序。如果你想節省時間，你也可以使用第三方的 CSS 重置。這些樣式表已經預先建置好，所以你只需下載它們並將它們添加到你的專案中即可。第三方 CSS 重置的一個很好的範例是 `Normalize.css`。這個樣式表會為廣泛的 HTML 元素標準化樣式，同時仍保留一些有用的預設樣式，特別是那些對存取性很重要的樣式。它也修正了常見的錯誤和樣式不一致問題。

另一個選項是 `sanitize.css`。這是一個 CSS 程式庫，你可以用來確保預設樣式在所有主要現代瀏覽器中保持一致。這個程式庫與 `Normalize.css` 一起開發，因此它們會同步演進。它也有可供下載的個別樣式表，用於特定用途，例如標準化表單和排版。

有許多選項可用，但你應該選擇最符合你專案需求的那些。你也可以結合兩種方法，使用第三方 CSS 重置與訂製重置。

在使用 CSS 重置時，也必須考慮存取性。你的網頁應用程式應該對所有人都能存取。因此，你不應該重置可能對螢幕閱讀器或其他輔助技術有幫助的樣式。你還應該考慮這些額外樣式表對你的應用程式效率的影響，因為它們必須在訂製樣式套用之前被下載。 

透過移除預設樣式，CSS 重置為你提供一個空白的起點來實作你的設計。這會在不同瀏覽器和不同裝置之間帶來更一致且統一的使用者體驗。
