# --description--

SEO（検索エンジン最適化）とは、ウェブページを最適化して検索エンジンでより目立ち、上位に表示されるようにする手法です。サイトのSEOを改善する方法の一つに、`meta`要素を使ってウェブページの短い説明を提供することがあります。以下は、ガーデニングサイトのページ説明を設定するためにmeta要素を使った例です：

```html
<meta
  name="description"
  content="Discover expert tips and techniques for gardening in small spaces, choosing the right plants, and maintaining a thriving garden."
/>
```

`name`属性を`description`に設定すると、ブラウザーや検索エンジン、その他のウェブツールがこのメタデータを正しく解釈します。`content`属性には説明文を入れます。説明は短く簡潔にすることをお勧めします。これは、検索エンジンが結果ページのレイアウトに基づいて説明文を切り詰めることが多いためです。

他の種類の`meta`要素と同様に、`meta`の説明はウェブページ自体には表示されません。ページ説明が見られる場所の一つは、検索エンジンの結果ページのスニペットです。以下はfreeCodeCampのsubredditとGitHubリポジトリのページ結果スニペットの例です。

```md
r/freeCodeCamp: This is the official subreddit for the freeCodeCamp.org community. Learn to
code for free together with millions of other people...
```

```md
Our full-stack web development and machine learning curriculum is completely free and self-
paced. We have thousands of interactive coding challenges to help you...
```

例では、各ページの説明がサイトリンクのすぐ下に配置されています。ユーザーは数秒でページの概要を把握し、詳細を知るためにリンクをクリックするかどうかを判断できます。

`meta`の説明は検索エンジンのランキングに直接影響しませんが、魅力的な説明があることでウェブサイトへのアクセスが増える可能性があります。
