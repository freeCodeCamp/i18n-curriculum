# --description--

Для `depend on` використовується, щоб сказати, що щось може змінюватися залежно від різних факторів. Наприклад:

- `The project depends on good teamwork.` - Проєкт вдасться лише якщо команда добре працюватиме разом.

- `Developers depend on tests to find errors.` - Програмістам потрібні тести, щоб перевірити, чи є помилки в коді.

Зверніть увагу, що після `depend` завжди стоїть прийменник `on`.
