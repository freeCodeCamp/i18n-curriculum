# --explanation--

구조 `it's + adjective + to`은 행동의 성격을 설명할 때 사용됩니다. 예를 들어:

`It's necessary to prepare before an important presentation.` - 이것은 준비가 발표 전에 필수적인 행동임을 의미합니다.

이 문장에서 `it's healthy to have`은 의견 차이가 유익하다는 뜻입니다.
