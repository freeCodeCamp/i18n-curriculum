# --interactive--

在 JavaScript 中处理字符串时，有时可能需要定位特定子字符串在较大字符串中的位置。

子字符串是出现在较大字符串中的字符序列。 例如，在字符串 `hello world` 中，`hello` 和 `world` 是子字符串。

要确定子串在字符串中的位置，可以使用 `indexOf()` 方法。 JavaScript 中的 `indexOf()` 方法允许你搜索字符串中的子字符串。

如果找到子字符串，`indexOf()` 将返回该子字符串首次出现的索引（或位置）。 如果未找到子字符串，`indexOf()` 将返回 `-1`，表示搜索不成功。

`indexOf()` 方法需要两个参数：第一个参数是要在较大字符串中查找的子字符串，第二个参数是搜索的起始位置选项。 如果不提供起始位置，搜索将从字符串的起始位置开始。

在这个上下文中，**参数** 是你在调用函数或方法时提供的一个值，使该函数或方法能够使用你提供的特定信息来执行它的任务。你将在后续课程中学习更多关于参数的内容。

下面是使用 `indexOf()` 方法查找字符串 `awesome` 位置的示例：

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("awesome!");
console.log(position); // 14
```

:::

在此示例中，单词 `awesome` 从字符串 `JavaScript is awesome!` 中的索引 `14` 开始，因此 `indexOf()` 方法返回 `14`。

现在，让我们看看找不到子字符串时会发生什么：

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("fantastic");
console.log(position); // -1
```

:::

由于字符串中没有出现 `fantastic` 单词，因此方法返回 `-1`。

你还可以为 `indexOf()` 提供第二个参数，指定从字符串的哪里开始搜索。 这是一个示例：

:::interactive_editor

```js
let sentence = "JavaScript is awesome, and JavaScript is powerful!";
let position = sentence.indexOf("JavaScript", 10);
console.log(position); // 27
```

:::

在这种情况下，从第 10 个字符后开始搜索 `JavaScript`，因此 `JavaScript` 的第二次出现的位置是在索引 `27`。

需要注意的是，`indexOf()` 方法区分大小写。

在此示例中，以下内容将返回 `-1`，因为在字符串 `freeCodeCamp` 中找不到大写字母 `F`。

:::interactive_editor

```js
console.log("freeCodeCamp".indexOf("F")) // -1
```

:::

当你需要检查字符串中是否存在子字符串并确定其位置以便进行进一步操作时，使用 `indexOf()` 非常有用。
