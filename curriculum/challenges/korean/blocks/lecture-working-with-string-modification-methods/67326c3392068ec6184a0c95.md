# --interactive--

JavaScript를 사용할 때 문자열을 특정 횟수만큼 반복해야 하는 상황이 있을 수 있습니다.

반복되는 패턴을 생성하거나 단순히 텍스트를 복제할 때, `repeat()` 메서드는 이를 간단하고 효과적으로 처리할 수 있는 방법을 제공합니다.

`repeat()` 메서드는 JavaScript 내장 함수로, 문자열을 지정한 횟수만큼 반복할 수 있게 해줍니다. 기본 구문은 다음과 같습니다:

```js
string.repeat(count);
```

`string`는 반복하려는 문자열이고, `count`는 문자열을 반복할 횟수입니다. 예를 들어 보겠습니다:

:::interactive_editor

```js
let word = "Hello!";
let repeatedWord = word.repeat(3);
console.log(repeatedWord);  // "Hello!Hello!Hello!"
```

:::

이 경우 문자열 `Hello!`가 세 번 반복되어 `Hello!Hello!Hello!`가 됩니다.

`repeat()` 메서드는 유용하지만 몇 가지 예외와 제한 사항을 기억하세요.

`count` 매개변수는 0 이상의 숫자여야 합니다. 음수를 전달하면 JavaScript가 `RangeError`를 발생시킵니다.

```js
let word = "Test";
console.log(word.repeat(-1));  // Throws RangeError: Invalid count value
```

`count`는 유한한 숫자여야 합니다. 문자열을 무한히 반복하거나 `Infinity`를 횟수로 사용하면 `RangeError`가 발생합니다.

JavaScript에서 `Infinity`는 무한대를 나타내는 특별한 값입니다. 이는 어떤 유한한 숫자보다 큰 수를 나타낼 때 사용됩니다.

```js
let word = "Test";
console.log(word.repeat(Infinity));  // Throws RangeError: Invalid count value
```

만약 횟수가 정수가 아니라면(예: `2.5` 같은 소수), `repeat()` 메서드는 이를 가장 가까운 정수로 내림합니다.

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(2.5));  // "TestTest"
```

:::

횟수로 `0`를 전달하면 `repeat()` 메서드는 빈 문자열을 반환합니다.

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(0));  // ""
```

:::

`repeat()` 메서드는 문자열 복제를 포함하는 작업을 간소화하여 코드를 더 간결하고 읽기 쉽게 만듭니다.

반복되는 텍스트 패턴을 생성하거나 문자를 채워 넣을 때, `repeat()`는 루프나 더 복잡한 코드를 작성하지 않아도 되게 해줍니다.

`repeat()` 메서드에 숫자를 직접 전달하는 것에 제한되지 않습니다. 숫자 값을 저장하는 변수를 전달할 수도 있습니다.

:::interactive_editor

```js
let count = 4;
let word = "Test";
let repeatedWord = word.repeat(count);
console.log(repeatedWord); // TestTestTestTest
```

:::

이 예제에서 `count` 변수는 반복 횟수를 저장합니다. 이는 반복 횟수가 사용자 입력이나 프로그램 내 다른 동적 값에 따라 달라질 때 유용합니다.
