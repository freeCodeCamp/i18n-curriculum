# --interactive--

反轉字串是常見的程式設計任務，可以使用 JavaScript 結合字串和陣列的方法來完成。此處理程序包含三個主要步驟：

- 將字串拆分成字元陣列。
- 反轉陣列。
- 將字元重新組合成字串。

讓我們使用 `split()`、`reverse()` 和 `join()` 方法來探索這些步驟。

反轉字串的第一步是將它轉換成由單一字元組成的陣列。我們可以使用 `split()` 方法來做到這點。`split()` 方法會將字串分割成子字串陣列，並根據指定的分隔符決定每個分割點。如果沒有提供分隔符，該方法會傳回一個包含原始字串作為單一元素的陣列。常見分隔符的範例如下：

- 一個空字串（`""`），它將字串拆分成單一字元。

- 一個單一空格（`" "`），會在字串中每處空格出現的地方進行分割。

- 一個破折號（`"-"`），用來在每個破折號處拆分字串。

以下是使用 split 方法來創建字元陣列的範例：

:::interactive_editor

```js
let str = "hello";
let charArray = str.split("");
console.log(charArray); // ["h", "e", "l", "l", "o"]
```

:::

在這個範例中，我們使用 `split("")`（傳入空字串）將字串 `hello` 轉換成其各個字元的陣列。一旦我們有了字元陣列，就可以使用 `reverse()` 方法來反轉陣列中元素的順序。 

`reverse()` 方法是一個陣列方法，用來就地反轉陣列。這表示它會修改原始陣列，而不是建立新的陣列。以下是我們如何使用它：

:::interactive_editor

```js
let charArray = ["h", "e", "l", "l", "o"];
charArray.reverse();
console.log(charArray); // ["o", "l", "l", "e", "h"]
```

:::

在此範例中，`reverse()` 會改變 `charArray` 中元素的順序，將其從 `["h", "e", "l", "l", "o"]` 反轉為 `["o", "l", "l", "e", "h"]`。

最後一步是將反轉的字元陣列轉回字串。我們可以使用 `join()` 方法來完成這個動作。`join()` 方法會透過連接陣列中所有元素並以指定的分隔字串分隔，來創建並傳回一個新的字串。如果你想要不使用任何分隔符號來連接字元，可以使用空字串作為引數。以下是一個範例：

:::interactive_editor

```js
let reversedArray = ["o", "l", "l", "e", "h"];
let reversedString = reversedArray.join("");
console.log(reversedString); // "olleh"
```

:::

在這個範例中，`join("")`（傳入空字串作為引數）將陣列中的所有字元合併成一個沒有任何分隔符的字串。

請記住，JavaScript 中的字串是不可變的，這表示你無法直接透過修改字串來反轉它。這就是為什麼我們需要將字串轉換成陣列，反轉陣列，然後再將它轉換回字串。這種字串和陣列方法的組合，提供了一種強大且靈活的方式來操作 JavaScript 中的字串。
