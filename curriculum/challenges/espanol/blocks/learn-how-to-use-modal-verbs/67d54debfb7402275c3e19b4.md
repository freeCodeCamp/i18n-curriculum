# --explanation--

`Sensitive data` se refiere a información confidencial que debe ser protegida. Por ejemplo:

`Sensitive data, like passwords, should never be shared.` - La información privada, como las contraseñas, debe mantenerse en secreto.

Las `Encryption keys` se usan para bloquear y desbloquear datos cifrados. Por ejemplo:

`Encryption keys help keep online payments secure.` - Estos códigos especiales protegen las transacciones financieras.

Las directrices de Jessica aseguran que ambos se manejen de forma segura.
