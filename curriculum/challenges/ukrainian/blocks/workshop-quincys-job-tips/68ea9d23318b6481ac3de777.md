# --description--

Точно так само, як і рядковий елемент цитати, ви також можете додати атрибут `cite` до блочного елемента цитати.

Ось приклад блочного елемента цитати з атрибутом `cite`:

```html
<blockquote cite="https://www.freecodecamp.org/news/is-college-worth-it/">
  The first thing you should consider about education is this is an economic decision.
</blockquote>
```

Тепер додайте атрибут `cite` до блочного елемента цитати з URL `https://www.freecodecamp.org/news/learn-to-code-book/`.

# --hints--

Ваш елемент `blockquote` має мати атрибут `cite`.

```js
const firstBlockquoteEl = document.querySelector('section:first-of-type > h2 + blockquote');
assert.exists(firstBlockquoteEl?.getAttribute('cite'));
```

Атрибут `cite` повинен мати значення `https://www.freecodecamp.org/news/learn-to-code-book/`.

```js
const firstBlockquoteEl = document.querySelector('section:first-of-type > h2 + blockquote');
assert.equal(firstBlockquoteEl?.getAttribute('cite'), 'https://www.freecodecamp.org/news/learn-to-code-book/');
```
