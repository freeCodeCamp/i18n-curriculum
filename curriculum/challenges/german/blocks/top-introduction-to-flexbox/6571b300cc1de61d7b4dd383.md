# --description--

Die `flex`-Deklaration ist tatsächlich eine Kurzschreibweise für 3 Eigenschaften, die Sie an einem Flex-Element festlegen können. Diese Eigenschaften beeinflussen, wie sich Flex-Elemente innerhalb ihres Containers in der Größe anpassen. Sie haben bereits einige Kurzschreibweise-Eigenschaften gesehen, aber noch nicht offiziell definiert.

> Kurzschreibweise-Eigenschaften sind CSS-Eigenschaften, mit denen Sie die Werte mehrerer anderer CSS-Eigenschaften gleichzeitig festlegen können. Mit einer Kurzschreibweise-Eigenschaft können Sie prägnantere (und oft besser lesbare) Stylesheets schreiben, was Zeit und Aufwand spart.

In diesem Fall ist `flex` tatsächlich eine Kurzschreibweise für `flex-grow`, `flex-shrink` und `flex-basis`.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/flex-box/flexbox-04.png" alt="CSS code setting the flex property to 1 for a div element.">

Im obigen Screenshot entspricht `flex: 1`: `flex-grow: 1`, `flex-shrink: 1`, `flex-basis: 0`.

Sehr oft sehen Sie die Kurzschreibweise flex mit nur einem Wert definiert. In diesem Fall wird dieser Wert auf `flex-grow` angewendet. Wenn Sie also `flex: 1` auf unsere divs gesetzt haben, haben Sie tatsächlich eine Kurzschreibweise von `flex: 1 1 0` angegeben.
