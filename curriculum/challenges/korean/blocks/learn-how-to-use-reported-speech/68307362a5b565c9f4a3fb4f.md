# --explanation--

`It was mentioned...`는 **수동태** 구조이기도 하며, 누가 말했는지보다 공유된 아이디어에 포커스가 맞춰져 있습니다. 예를 들어:

`It was mentioned that the update improved security.` – 화자를 명시하지 않고 아이디어를 전달합니다.

`it's` + 비교 형용사 + `to` + 동사 구조는 어떤 것이 다른 것과 대비되어 어떻게 이루어지는지를 보여줄 때 사용합니다. 예를 들어:

`It's faster to search with the new tool.` – 이 문장은 새 도구가 이전 도구에 비해 검색할 때 시간을 절약한다는 뜻입니다.
