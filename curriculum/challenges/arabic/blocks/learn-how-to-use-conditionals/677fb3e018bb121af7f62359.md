# --explanation--

تُستخدم `First Conditional` لوصف حالات حقيقية وممكنة في المستقبل. تتكون من `if` + `Present Simple` (لشرط) و`will` + الفعل الأساسي (لنتيجة). على سبيل المثال:

`If you debug the code, the app will run smoothly.` - تصحيح الشفرة سيؤدي إلى عمل التطبيق بسلاسة.

في الجملة الصحيحة، الشرط هو `If we don't back up regularly`، والنتيجة هي `we could lose important information`. هذا يشير إلى احتمال حقيقي.
