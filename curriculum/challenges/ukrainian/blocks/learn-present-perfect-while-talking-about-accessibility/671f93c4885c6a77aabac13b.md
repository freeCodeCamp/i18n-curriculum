# --explanation--

`Concerning` використовується, щоб описати щось, що викликає занепокоєння або тривогу. Наприклад:

`The drop in performance is quite concerning.` — це означає, що падіння продуктивності є причиною для занепокоєння.

`Found out` — це минулий час від `find out`. Це означає виявити або дізнатися щось, що раніше було невідомо. Наприклад,

`She found out the reason for the delay.` — це стосується дізнання нової інформації про причину затримки.
