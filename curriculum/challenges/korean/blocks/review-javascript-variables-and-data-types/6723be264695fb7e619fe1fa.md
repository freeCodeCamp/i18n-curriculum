## HTML, CSS, 그리고 JavaScript 다루기

HTML과 CSS가 웹사이트 구조를 제공하는 반면, JavaScript는 사용자 입력 처리, 요소 애니메이션, 심지어 완전한 웹 애플리케이션 구축 같은 복잡한 기능을 가능하게 하여 웹사이트에 상호작용성을 부여합니다.

## JavaScript의 데이터 타입

데이터 타입은 프로그램이 숫자인지, 텍스트인지, 혹은 다른 종류의 데이터인지 이해하도록 돕습니다.

- **Number**: 숫자는 정수와 부동 소수점 값을 모두 나타냅니다. 정수 예시로는 7, 19, 90이 있습니다.
- **Floating point**: 부동 소수점 숫자는 소수점이 있는 숫자입니다. 예시로 3.14, 0.5, 0.0001이 있습니다.
- **String**: 문자열은 따옴표로 감싼 문자 또는 텍스트의 수열입니다. `"I like coding"`와 `'JavaScript is fun'`은 문자열 예시입니다.
- **Boolean**: 불리언은 두 가지 가능한 값 중 하나를 나타냅니다: `true` 또는 `false`. 불리언은 `isLoggedIn = true` 같은 조건을 나타내는 데 사용할 수 있습니다.
- **Undefined and Null**: `undefined` 값은 선언되었지만 값이 할당되지 않은 변수입니다. `null` 값은 빈 값이거나 의도적으로 `null` 값이 할당된 변수입니다.
- **Object**: 객체는 키-값 쌍의 모음입니다. 키는 속성 이름이고, 값은 속성 값입니다.

여기서 `pet` 객체는 `name`, `age`, `type` 세 개의 속성 또는 키를 가집니다. 값들은 각각 `Fluffy`, `3`, `dog`입니다.

```js
let pet = {
  name: "Fluffy",
  age: 3,
  type: "dog"
};
```

- **Symbol**: Symbol 데이터 타입은 객체 속성의 식별자로 사용할 수 있는 고유하고 변경 불가능한 값입니다.

아래 예시에서는 같은 설명을 가진 두 개의 심볼이 생성되었지만, 이들은 같지 않습니다.

:::interactive_editor

```js
const crypticKey1= Symbol("saltNpepper");
const crypticKey2= Symbol("saltNpepper");
console.log(crypticKey1 === crypticKey2); // false
```

:::

- **BigInt**: 숫자가 `Number` 데이터 타입에 너무 클 때, 임의 길이의 정수를 나타내기 위해 BigInt 데이터 타입을 사용할 수 있습니다.

숫자 끝에 `n`를 추가하면 BigInt를 만들 수 있습니다.

```js
const veryBigNumber = 1234567890123456789012345678901234567890n;
```

## JavaScript의 변수

- 변수는 `let` 키워드를 사용해 선언할 수 있습니다.

```js
let cityName;
```

- 변수에 값을 할당하려면 대입 연산자 `=`를 사용할 수 있습니다.

```js
cityName = "New York";
```

- `let`로 선언된 변수는 새 값을 다시 할당할 수 있습니다.

:::interactive_editor

```js
let cityName = "New York";
cityName = "Los Angeles";
console.log(cityName); // Los Angeles
```

:::

- `let` 외에도 `const`를 사용해 변수를 선언할 수 있습니다. 하지만 `const` 변수는 새 값을 다시 할당할 수 없습니다.

```js
const cityName = "New York";
cityName = "Los Angeles"; // TypeError: Assignment to constant variable.
```

- `const`로 선언된 변수는 코드 전체에서 변경이 허용되지 않는 상수를 선언할 때 사용됩니다. 예를 들어 `PI` 또는 `MAX_SIZE` 같은 경우입니다.

## 변수 명명 관례

- 변수 이름은 설명적이고 의미 있어야 합니다.
- 변수 이름은 `cityName`, `isLoggedIn`, `veryBigNumber`처럼 카멜 케이스여야 합니다.
- 변수 이름은 숫자로 시작하면 안 됩니다. 반드시 문자, `_`, 또는 `$`로 시작해야 합니다.
- 변수 이름에는 공백이나 특수 문자가 포함되면 안 되며, `_`와 `$`는 예외입니다.
- 변수 이름은 예약된 키워드를 사용하면 안 됩니다.
- 변수 이름은 대소문자를 구분합니다. `age`와 `Age`는 서로 다른 변수입니다.

## JavaScript의 문자열과 문자열 불변성

- 문자열은 따옴표로 감싼 문자 수열입니다. 작은따옴표와 큰따옴표를 사용해 만들 수 있습니다.

```js
let correctWay = 'This is a string';
let alsoCorrect = "This is also a string";
```

- JavaScript에서 문자열은 불변입니다. 즉, 문자열이 생성된 후에는 문자열 내 문자를 변경할 수 없습니다. 하지만 문자열에 새 값을 다시 할당할 수는 있습니다.

```js
let firstName = "John";
firstName = "Jane"; // Reassigning the string to a new value
```

## JavaScript에서 문자열 연결

- 연결은 여러 문자열을 합치거나 텍스트를 담은 변수와 문자열을 결합하는 과정입니다. `+` 연산자는 문자열을 연결하는 가장 간단하고 자주 쓰이는 방법 중 하나입니다.

:::interactive_editor

```js
let studentName = "Asad";
let studentAge = 25;
let studentInfo = studentName + " is " + studentAge + " years old.";
console.log(studentInfo); // Asad is 25 years old.
```

:::

- 기존 문자열에 추가하거나 덧붙여야 할 때는 `+=` 연산자를 사용할 수 있습니다. 시간이 지남에 따라 문자열에 더 많은 텍스트를 추가할 때 유용합니다.

:::interactive_editor

```js
let message = "Welcome to programming, ";
message += "Asad!";
console.log(message); // Welcome to programming, Asad!
```

:::

- 문자열을 연결하는 또 다른 방법은 `concat()` 메서드를 사용하는 것입니다. 이 메서드는 두 개 이상의 문자열을 함께 합칩니다.

:::interactive_editor

```js
let firstName = "John";
let lastName = "Doe";
let fullName = firstName.concat(" ", lastName);
console.log(fullName); // John Doe
```

:::

## `console.log()`로 메시지 출력하기

- `console.log()` 메서드는 콘솔에 메시지를 출력하는 데 사용됩니다. 디버깅과 코드 테스트에 유용한 도구입니다.

:::interactive_editor

```js
console.log("Hello, World!");
// Output: Hello, World!
```

:::

## JavaScript에서 세미콜론

- 세미콜론은 주로 문장의 끝을 표시하는 데 사용됩니다. 이는 JavaScript 엔진이 개별 명령어를 구분하는 데 도움을 주어 올바른 실행에 필수적입니다.

```js
let message = "Hello, World!"; // first statement ends here
let number = 42; // second statement starts here
```

- 세미콜론은 코드 실행의 모호성을 방지하고 문장이 올바르게 종료되도록 보장합니다.

## JavaScript의 주석

- 주석 처리된 코드는 JavaScript 엔진이 무시합니다. 주석은 코드를 설명하거나 메모를 남기거나 코드를 일시적으로 비활성화하는 데 사용됩니다.
- 한 줄 주석은 `//`를 사용해 만듭니다.

```js
// This is a single-line comment and will be ignored by the JavaScript engine
```

- 여러 줄 주석은 `/*`로 시작하고 `*/`로 끝냅니다.

```js
/*
This is a multi-line comment.
It can span multiple lines.
*/
```

## 동적 타입 언어로서의 JavaScript

- JavaScript는 동적 타입 언어로, 변수를 선언할 때 데이터 타입을 명시할 필요가 없습니다. JavaScript 엔진이 변수에 할당된 값에 따라 데이터 타입을 자동으로 결정합니다.

```js
let error = 404; // JavaScript treats error as a number
error = "Not Found"; // JavaScript now treats error as a string
```

- C# 같은 동적 타입이 아닌 다른 언어에서는 오류가 발생합니다:

```csharp
int error = 404; // value must always be an integer
error = "Not Found"; // This would cause an error in C#
```

## `typeof` 연산자 사용하기

- `typeof` 연산자는 변수의 데이터 타입을 확인하는 데 사용됩니다. 변수 타입을 나타내는 문자열을 반환합니다.

:::interactive_editor

```js
let age = 25;
console.log(typeof age); // "number"

let isLoggedIn = true;
console.log(typeof isLoggedIn); // "boolean"
```

:::

- 하지만 `null`에 관해 JavaScript에서 잘 알려진 특이점이 있습니다. `typeof` 연산자는 `"object"` 값에 대해 `null`를 반환합니다.

:::interactive_editor

```js
let user = null;
console.log(typeof user); // "object"
```

:::
