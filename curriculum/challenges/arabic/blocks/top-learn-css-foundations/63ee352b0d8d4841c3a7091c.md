# --description--

ماذا لو كان لديك مجموعتان من العناصر تشتركان في بعض إعلانات الأنماط الخاصة بهما؟

```css
.read {
  color: white;
  background-color: black;
  /* several unique declarations */
}

.unread {
  color: white;
  background-color: black;
  /* several unique declarations */
}
```

كلا محددي `.read` و `.unread` يشتركان في إعلانات `color: white;` و `background-color: black;`، لكن لكل منهما عدة إعلانات فريدة خاصة به. لتقليل التكرار، يمكنك جمع هذين المحددين معًا في قائمة مفصولة بفواصل:

```css
.read,
.unread {
  color: white;
  background-color: black;
}

.read {
  /* several unique declarations */
}

.unread {
  /* several unique declarations */
}
```

كلا المثالين أعلاه (مع التجميع وبدونه) سيعطيان نفس النتيجة، لكن المثال الثاني يقلل من تكرار الإعلانات ويسهل تعديل `color` أو `background-color` لكلا الصنفين مرة واحدة.
