# --description--

У попередньому уроці ви дізналися, як кожен символ у рядку можна ідентифікувати за його індексом (починаючи з нуля) і отримати доступ за допомогою дужкової нотації:

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
print(my_str[-1]) # d
```

**Зріз рядка** дозволяє витягти частину рядка або працювати лише з певною його частиною. Ось базовий синтаксис:

```python
string[start:stop]
```

Якщо ви хочете витягти символи з певного індексу до іншого, просто розділіть індекси `start` і `stop` двокрапкою:

```python
my_str = 'Hello world'
print(my_str[1:4]) # ell
```

Зверніть увагу, що індекс `stop` не включається, тому `[1:4]` витягнув символи від індексу `1` і до, але не включно, символу з індексом `4`.

Ви також можете опустити індекси `start` і `stop`, і Python за замовчуванням візьме `0` або кінець рядка відповідно. Наприклад, ось що станеться, якщо опустити індекс `start`:

```python
my_str = 'Hello world'
print(my_str[:7])  # Hello w
```

Це витягне все від індексу `0` до (але не включно) символу з індексом `7`. А ось що станеться, якщо опустити індекс `stop`:

```python
my_str = 'Hello world'
print(my_str[8:])  # rld
```

Це витягне все від символу з індексом `8` до кінця рядка.

Зверніть увагу, що зріз рядка не змінює оригінальний рядок:

```py
my_str = 'Hello world'
print(my_str[8:])  # rld
print(my_str)  # Hello world
```

Ви також можете опустити обидва індекси `start` і `stop`, що витягне весь рядок:

```python
my_str = 'Hello world'
print(my_str[:])  # Hello world
```

Окрім індексів `start` і `stop`, існує також необов’язковий параметр `step`, який використовується для вказівки кроку між кожним індексом у зрізі.

Ось синтаксис для цього:

```python
string[start:stop:step]
```

У прикладі нижче зріз починається з індексу `0`, зупиняється перед `11` і витягує кожен другий символ:

```python
my_str = 'Hello world'
print(my_str[0:11:2])  # Hlowrd
```

Корисний прийом із параметром `step` — це перевернути рядок, встановивши крок `-1` і залишивши `start` і `stop` порожніми:

```python
my_str = 'Hello world'
print(my_str[::-1]) # dlrow olleH
```
