# --description--

FRACTRAN ist eine Turing-vollständige esoterische Programmiersprache, die vom Mathematiker John Horton Conway erfunden wurde.

Ein FRACTRAN-Programm ist eine geordnete Liste positiver Brüche $P = (f_1, f_2, \\ldots, f_m)$ zusammen mit einer anfänglichen positiven ganzen Zahl $n$ als Eingabe.

Das Programm wird ausgeführt, indem die ganze Zahl $n$ wie folgt aktualisiert wird:

<ul>
  <li>für den ersten Bruch $f_i$ in der Liste, für den $nf_i$ eine ganze Zahl ist, ersetzen Sie $n$ durch $nf_i$;</li>
  <li>wiederholen Sie diese Regel, bis kein Bruch in der Liste mehr eine ganze Zahl ergibt, wenn er mit $n$ multipliziert wird, dann halten Sie an.</li>
</ul>

Conway gab ein Programm für Primzahlen in FRACTRAN an:

$\\dfrac{17}{91}$, $\\dfrac{78}{85}$, $\\dfrac{19}{51}$, $\\dfrac{23}{38}$, $\\dfrac{29}{33}$, $\\dfrac{77}{29}$, $\\dfrac{95}{23}$, $\\dfrac{77}{19}$, $\\dfrac{1}{17}$, $\\dfrac{11}{13}$, $\\dfrac{13}{11}$, $\\dfrac{15}{14}$, $\\dfrac{15}{2}$, $\\dfrac{55}{1}$

Ausgehend von $n=2$ wird dieses FRACTRAN-Programm $n$ zu $15=2\\times (\\frac{15}{2})$ ändern, dann zu $825=15\\times (\\frac{55}{1})$, und erzeugt die folgende Folge ganzer Zahlen:

$2$, $15$, $825$, $725$, $1925$, $2275$, $425$, $390$, $330$, $290$, $770$, $\\ldots$

Nach 2 enthält diese Folge die folgenden Potenzen von 2:

$2^2=4$, $2^3=8$, $2^5=32$, $2^7=128$, $2^{11}=2048$, $2^{13}=8192$, $2^{17}=131072$, $2^{19}=524288$, $\\ldots$

die die Primzahlpotenzen von 2 sind.

function fractran(progStr){
  var num = new Array();
  var den = new Array();
  var val ;
  var out="";
  function compile(prog){
    var regex = /\s*(\d*)\s*\/\s*(\d*)\s*(.*)/m;
    while(regex.test(prog)){
      num.push(regex.exec(prog)[1]);
      den.push(regex.exec(prog)[2]);
      prog = regex.exec(prog)[3];
    }
  }

  function step(val){
    var i=0;
    while(i<den.length && val%den[i] != 0) i++;
    return num[i]*val/den[i];
  }

  var seq=[]

  function exec(val){
    var i = 0;
    while(val && i<limit){
      seq.push(val)
      val = step(val);
      i ++;
    }
  }

  // Main
  compile(progStr);
  var limit = 10;
  exec(2);
  return seq;
}
