# --description--

Un `extension` è un componente software che aggiunge funzionalità specifiche a una più ampia applicazione software. `Extensions` può fornire nuove funzionalità o migliorare quelle esistenti.

Per esempio, un componente aggiuntivo per browser `extension` può aggiungere funzionalità speciali al tuo browser web, come bloccare le pubblicità o gestire le password.
