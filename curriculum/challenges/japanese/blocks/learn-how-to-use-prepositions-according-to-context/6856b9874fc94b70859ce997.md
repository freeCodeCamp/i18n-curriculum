# --explanation--

`Update` は、何かについて最新の状況や情報を共有することを意味します。例えば：

`She will update the team after each sprint.` – これは彼女が進捗状況を知らせるという意味です。

`On` は、行動や発言の主題やフォーカスを示すために使われます。話している内容とその話題をつなげる役割があります。例えば：

`We had a meeting on software updates.` – これは会議がその話題についてだったという意味です。
