# --explanation--

Eine `license` ist eine rechtliche Vereinbarung, die Ihnen die Nutzung von Software oder einem digitalen Produkt erlaubt. Unternehmen kaufen oft Lizenzen für Werkzeuge wie Microsoft Office, Adobe Creative Cloud oder Entwicklungsplattformen. Zum Beispiel:

`We have five licenses for our design software.` — Das bedeutet, dass fünf Personen diese Software legal nutzen dürfen.
