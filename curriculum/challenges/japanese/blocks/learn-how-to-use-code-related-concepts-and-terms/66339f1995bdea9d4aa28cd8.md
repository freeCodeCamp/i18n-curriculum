# --description--

`difference` という言葉は、2つ以上のものが同じでないことを指します。これは、項目や概念の間の異なる特徴や性質を強調します。

例えば、リンゴとオレンジを見ると、`difference` には色、味、食感が含まれます。

前置詞の `between` は、2つのものを比較したり、比較したときに存在する対比を話す際によく使われる `difference between` というフレーズで使われます。

例えば、`He explained the difference between Java and Python.`
