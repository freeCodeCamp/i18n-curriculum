# --description--

Das Wort `so` ist ein Bindewort, das Ursache und Wirkung verbindet oder den Grund für etwas erklärt. In der Technologie und Programmierung verbindet `so` oft das Problem oder die Situation mit der Lösung oder dem Ergebnis.

# --questions--

## --text--

Wählen Sie den Satz aus, in dem `so` korrekt verwendet wird, um Ursache und Wirkung in einem technischen Kontext zu verbinden.

## --answers--

`The program crashed, so we must reboot the computer.`

---

`We wrote the code, so but it didn't compile.`

### --feedback--

Das Wort `so` wird hier falsch verwendet. `But` sollte nicht nach `so` verwendet werden. Es sollte entweder `The code didn't compile, so we rewrote it` oder `We wrote the code, but it didn't compile.` sein.

---

`The server is online, so therefore we can't connect.`

### --feedback--

Die Verwendung von `so` und `therefore` zusammen ist überflüssig. Es sollte entweder `The server is online, so we can connect` oder `The server is online; therefore, we can connect.` sein.

---

`We need more data, so we can ignore the results.`

### --feedback--

Dieser Satz verbindet Ursache und Wirkung nicht logisch. Eine bessere Verwendung von `so` wäre `We need more data, so we will perform more tests.`.
