# --description--

`occasionally` は、何かが時々起こることを意味しますが、定期的または頻繁ではありません。`sometimes` より頻度は低く、`rarely` よりは高いです。

利便性のために、前回のチャレンジから副詞の頻度チャートを示します。

| 副詞         | 頻度      | 副詞       | 頻度      |
| ------------ | --------- | ---------- | --------- |
| Never        | 0%        | Often      | 70%       |
| Rarely       | 5%        | Generally  | 80%       |
| Seldom       | 15%       | Usually    | 90%       |
| Occasionally | 30%       | Always     | 100%      |
| Sometimes    | 50%       |            |           |
