# --interactive--

`innerHTML` と `createElement()` を使ってノードを作成する方法を見てみましょう。

まずは `innerHTML` から始めます。

`innerHTML` は `Element` オブジェクトのプロパティで、HTML マークアップを設定するために使えます。`innerHTML` を使うと、文字列で既存の要素の HTML 構造を設定し、必要なノードをすべて作成できます。

これは例です。空の `div` があり、その中に `innerHTML` プロパティを使っていくつかの要素を追加します：

```html
<div id="container">
  <!-- Add new elements here -->
</div>
```

まず、ID で要素を選択する必要があります：

```js
const container = document.getElementById("container");
```

次に、コンテナの `innerHTML` プロパティに文字列を設定します。この文字列には、作成したいノードと構造を表すために必要なすべてのマークアップが含まれていなければなりません。これは文字列内に HTML を書くようなものだと考えてください。

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
container.innerHTML = "<ul><li>Cheese</li><li>Tomato</li></ul>";
```

:::

この要素の `innerHTML` をピザの材料（チーズとトマト）の番号なしリストに設定します。

マークアップを理解しやすくするために、文字列を複数行に分けて書くこともできます。重要なのは、マークアップが作成したい HTML 構造を正確に表していることです。

このコードを実行すると、マークアップを検査したときに次のような HTML 構造が表示されます。

```html
<div id="container">
  <ul>
    <li>Cheese</li>
    <li>Tomato</li>
  </ul>
</div>
```

新しいノードは文字列が解析された後に動的に作成され、DOM に追加されました。

`innerHTML` は特定のシナリオで非常に役立ちます。ただし、いくつかのセキュリティリスクもあるため、よく理解しておく必要があります。文字列を完全にコントロールできない場合は使用しないでください。

たとえば、文字列がユーザーによって入力される場合、ユーザーが悪意のある内容をウェブサイトに挿入する可能性があるため、`innerHTML` を使うべきではありません。そのため、通常はプレーンテキストを挿入するために `textContent` を使うことが推奨されます。

新しいノードを作成するもう一つの方法は、`createElement()` メソッドを使うことです。この新しいメソッドでは、タグ名を指定して新しい要素を作成できます。

たとえば、画像要素を作成したい場合は、このメソッドを呼び出すときに `img` タグを文字列として渡します：

```js
document.createElement("img");
```

そして、この新しいオブジェクトを変数に代入できます：

```js
const img = document.createElement("img");
```

`createElement()` メソッドは、ドキュメントが `HTMLElement` の場合は新しい `HTMLDocument` オブジェクトを返します。そうでなければ、`Element` オブジェクトを返します。

この新しい要素が準備できたら、`appendChild()` メソッドを使って既存の別の要素の子として DOM に追加したり、他のメソッドを使って特定の場所に挿入したりできます。ニーズに最も合う方法を選んでください。

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
const img = document.createElement("img");
img.src = "https://cdn.freecodecamp.org/curriculum/cat-photo-app/lasagna.jpg";
img.alt = "A slice of lasagna on a plate.";
container.appendChild(img);
```

:::

`innerHTML` と `createElement()` を使ってノードを作成すると、ウェブサイトの構造や内容を動的に操作できます。これらの手法を組み合わせることで、インタラクティブなウェブアプリケーションを構築できます。
