# --interactive--

縮寫是字或片語的縮短形式。例如，「Dr」後面加上句點，是「doctor」這個字的縮寫。

縮寫有兩種常見形式：首字母縮略詞和首字母拼寫詞。

縮寫詞是由一個片語的首字母組成的字，每個字母表述該片語中一個字的第一個字母。

GUI 是一個縮寫的範例。它代表 Graphical User Interface。透過取每個字的第一個字母 G、U 和 I，你就得到縮寫 GUI。

縮寫詞是由片語的首字母組成，每個字母表述該片語中一個字的第一個字母。

例如，HTML 是一個首字母縮略詞；它代表 HyperText Markup Language，並且透過拼讀每個字母 H、T、M、L 來發音。

縮寫包括首字母縮略詞和首字母拼讀詞兩種型別。兩者的區別在於首字母縮略詞是以字的方式發音，而首字母拼讀詞是以單個字母分別發音。

它們對於撰寫更簡潔的文字非常有幫助，特別是在特定背景關係中它們是眾所周知且易於理解的情況下。 

如果你需要在 HTML 中顯示縮寫，例如首字母縮略詞或縮寫詞，縮寫元素正是你所需要的。當你第一次使用它們時，應該總是說明它們的完整含義。然後你可以使用縮寫元素來凸顯標示它們並提供更多細節。 

這裡有一個範例，你可以看到一段包含句子 `HTML is the foundation of the web` 的段落： 

:::interactive_editor

```html
<p><abbr>HTML</abbr> is the foundation of the web.</p>
```

:::

縮寫 HTML 位於縮寫元素中。在瀏覽器中，你會看到其實沒有任何改變。它仍然看起來像一般的文字。縮寫元素在背後提供有用的背景關係，但使用者仍會看到該縮寫像一般文字一樣。 

如果你想幫助使用者了解這個縮寫的意思，你可以使用 `title` 屬性顯示其完整表單。

`title` 屬性是選用的，但如果你決定包含它，必須是縮寫、首字母縮略詞或縮寫詞的人類可讀描述。

讓我們採用之前相同的範例，但為你的元素添加 `title` 屬性。它將是 `HyperText Markup Language`，即該縮寫的完整表單：

:::interactive_editor

```html
<p><abbr title="HyperText Markup Language">HTML</abbr> is the foundation of the web.</p>
```

:::

通常，當你為縮寫元素添加此屬性時，其樣式會改變。具體樣式取決於瀏覽器。有些瀏覽器可能會顯示虛線底線，而其他瀏覽器可能會將內容轉換為小型大寫字母。當使用者暫留在縮寫上時，完整的表單會以工具提示顯示。

雖然你不一定需要為網頁上的每個縮寫使用縮寫元素，但建議對那些可能不清楚以及可能需要額外背景關係的縮寫使用。

你應該運用最佳判斷，在資訊與呈現之間找到適當的平衡，以避免使文字雜亂，同時保持清晰且簡潔。
