# --interactive--

Aina ya `Number` ni mojawapo ya aina za data zinazotumika sana katika JavaScript na lugha nyingine za programu. Nambari zinaweza kuonekana rahisi, lakini kuna mengi ya kuchunguza linapokuja suala la nambari katika JavaScript. Hivyo, tuchunguze kwa undani zaidi. Katika JavaScript, aina ya data ya `Number` inawakilisha thamani ya nambari.

Tofauti na lugha nyingi za programu zinazotofautisha nambari kamili na nambari za desimali kuwa aina tofauti, JavaScript hutumia aina moja ya `Number` kuwakilisha nambari. Hii inamaanisha unaweza kufanya kazi na nambari kamili, desimali, na hata thamani maalum za nambari zote chini ya aina moja ya data ya `Number`.

Hapa kuna mfano wa msingi unaokuonyesha nambari kamili, nambari za desimali, na nambari hasi zote ni za aina number:

:::interactive_editor

```js
const wholeNumber = 50;
const decimalNumber = 4.5;
const negativeNumber = -7;

console.log(typeof wholeNumber); // number
console.log(typeof decimalNumber); // number
console.log(typeof negativeNumber); // number
```

:::

Aina ya `Number` ya JavaScript inajumuisha aina mbalimbali za thamani za nambari, kuanzia nambari kamili rahisi na nambari za desimali hadi kesi maalum kama `Infinity` na `NaN`, au "Not a Number". Hebu tuchambue aina kuu utakazokutana nazo. Nambari kamili ni nambari zisizo na sehemu za sehemu au desimali. Zinaweza kuwa chanya, hasi, au sifuri. Hapa kuna mifano:

:::interactive_editor

```js
const positiveInteger = 100;
const negativeInteger = -25;
const zero = 0;

console.log(typeof positiveInteger); // number
console.log(typeof negativeInteger); // number
console.log(typeof zero); // number
```

:::

Nambari za desimali ni nambari zilizo na pointi za desimali. Mara nyingi hujulikana kama "floats" na wasanidi programu wa JavaScript. Floats ni muhimu unapohitaji usahihi zaidi, kama unavyoshughulika na vipimo au sarafu. Hapa kuna mifano:

:::interactive_editor

```js
const floatingPointNumber = 4.5;
const anotherFloat = 89.56;
const oneMoreFloat = 16.462;

console.log(typeof floatingPointNumber); // number
console.log(typeof anotherFloat); // number
console.log(typeof oneMoreFloat); // number
```

:::

JavaScript inaweza wakilisha nambari ambazo ziko zaidi ya kikomo cha juu kwa kutumia `Infinity`. Utakutana na hili unapojaribu kugawanya nambari kwa sifuri au katika matukio machache, kuzidi kikomo cha juu cha aina ya `Number`. Hapa kuna mfano:

:::interactive_editor

```js
const infiniteNumber = 1 / 0;
console.log(infiniteNumber); // Infinity
console.log(typeof infiniteNumber); // number
```

:::

Wakati mwingine katika JavaScript, baadhi ya operesheni za kihesabu hazizalishi nambari halali. Kwa mfano, ukijaribu kufanya operesheni ya kihesabu kwenye kitu ambacho si nambari, utapata `NaN`, ambayo inasimama kwa "Not a Number":

:::interactive_editor

```js
const notANumber = 'hello world' / 2;
console.log(notANumber); // NaN
```

:::

Kushangaza, aina ya `NaN` pia ni `Number`:

:::interactive_editor

```js
const notANumber = 'hello world' / 2;
console.log(typeof notANumber); // number
```

:::

Mbali na mfumo wa kawaida wa desimali (base 10), JavaScript pia inaunga mkono nambari katika misingi tofauti kama binary, octal, na hexadecimal. Binary ni mfumo wa base-2 unaotumia tu tarakimu 1 na 0. Octal ni mfumo wa base-8 unaotumia tarakimu 0 hadi 7 tu. Hexadecimal ni mfumo wa base-16 unaotumia tarakimu 0 hadi 9 na herufi a hadi f, kama unavyoyaona katika rangi za hex za CSS.
