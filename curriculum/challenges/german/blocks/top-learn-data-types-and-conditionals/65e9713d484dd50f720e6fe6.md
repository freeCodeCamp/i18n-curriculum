# --description--

Je nachdem, welche Art von Arbeit Sie machen, arbeiten Sie möglicherweise mehr mit Textstücken als mit Zahlen. Ein **String** ist ein Textstück … und ein grundlegender Baustein der Sprache.

HTML sorgt für Struktur und Bedeutung des Texts, CSS ermöglicht es uns, ihn präzise zu gestalten, und JavaScript bietet viele Funktionen zur Manipulation von Strings. Dazu gehört das Erstellen von benutzerdefinierten Willkommensnachrichten und Eingabeaufforderungen, das Anzeigen der richtigen Textbezeichnungen bei Bedarf, das Sortieren von Begriffen in die gewünschte Reihenfolge und vieles mehr.

Strings sind ein grundlegender Datentyp in JavaScript. Sie werden verwendet, um Text darzustellen, und sind entweder in einfache, doppelte Anführungszeichen oder Backticks eingeschlossen:

```javascript
let greeting = "Hello World!";

let greeting2 = 'I am learning JavaScript!';

let greeting3 = `with The Odin Project!`;
```

Strings, die mit einfachen Anführungszeichen deklariert werden, und Strings, die mit doppelten Anführungszeichen deklariert werden, sind gleichwertig. Welche Sie verwenden, ist eine Frage der persönlichen Vorliebe — es ist jedoch gute Praxis, einen Stil zu wählen und ihn konsequent im Code zu verwenden.

Welche Aussage beschreibt die Vielseitigkeit und Verwendung von Strings in JavaScript in Verbindung mit HTML, CSS und JavaScript selbst korrekt?

## --answers--

Strings in JavaScript können nur mit CSS für das Design von Webseiten manipuliert werden.

---

JavaScript-Strings werden ausschließlich für mathematische Berechnungen verwendet und können nicht mit HTML oder CSS interagieren.

---

Strings ermöglichen die Textmanipulation in JavaScript, einschließlich, aber nicht beschränkt auf das Erstellen benutzerdefinierter Nachrichten, das Sortieren von Begriffen und das dynamische Aktualisieren von HTML-Inhalten mit CSS-Styling.
