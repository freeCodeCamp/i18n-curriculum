# --description--

Eine `assignment operator` in der Programmierung wird verwendet, um einer Variablen einen Wert zuzuweisen. In vielen Programmiersprachen, einschließlich Python, wird der `assignment operator` durch das Symbol `=` dargestellt. Es weist den Computer an, den Wert auf der rechten Seite des `=` in der Variablen zu speichern, die auf der linken Seite genannt ist. Zum Beispiel weist `number = 10` der Variablen `10` den Wert `number` zu.
