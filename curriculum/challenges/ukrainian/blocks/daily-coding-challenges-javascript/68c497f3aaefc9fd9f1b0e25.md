# --description--

Для фінального дня тижня космосу вам буде задано масу корисного навантаження в кілограмах (кг), яке ви хочете відправити на орбіту. Визначте кількість палива, необхідного для відправлення вашого корисного навантаження на орбіту, використовуючи такі правила:

- Ракетам потрібно 1 кг палива на кожні 5 кг маси, яку вони мають підняти.
- Саме паливо має масу. Тож коли ви додаєте паливо, маса для підйому зростає, що вимагає більше палива, що збільшує масу, і так далі.
- Щоб обчислити загальну кількість палива: почніть з маси корисного навантаження, обчисліть потрібне паливо для нього, додайте це паливо до загальної маси і обчисліть знову. Повторюйте цей процес, доки додаткова кількість палива не стане меншою за 1 кг, тоді зупиніться.
- Ігноруйте масу самої ракети. Обчислюйте лише паливо, потрібне для підйому корисного навантаження та його власного палива.

Наприклад, якщо маса корисного навантаження 50 кг, вам знадобиться 10 кг палива для його підйому (корисне навантаження / 5), що збільшує загальну масу до 60 кг, для якої потрібно 12 кг палива (ще 2 кг), що збільшує загальну масу до 62 кг, для якої потрібно 12,4 кг палива — 0,4 кг додатково — що менше за 1 кг, тому тут зупиняємося. Загальна маса для підйому становить 62,4 кг, з яких 50 кг — початкове корисне навантаження, а 12,4 кг — паливо.

- Поверніть кількість палива, округлену до одного десяткового знака.

function launchFuel(payload) {
  let totalMass = payload;
  let additionalFuel = totalMass / 5;
  let totalFuel = additionalFuel;

  while (additionalFuel >= 1) {
    totalMass += additionalFuel;
    additionalFuel = (totalMass / 5) - totalFuel;
    totalFuel += additionalFuel;
  }

  return Number(totalFuel.toFixed(1));
}
