# --interactive--

在之前的课程中，你首次接触了 `iframe` 元素。在本课中，你将学习更多关于如何使用 `iframe` 元素。该元素代表内联框架。它是一个内联元素，用于将其他超文本标记语言内容直接嵌入到超文本标记语言页面中。该超文本标记语言内容可以是视频、地图、另一个超文本标记语言元素，甚至是其他网页。

这是一个从 YouTube 嵌入流行 freeCodeCamp 课程的示例。要查看不同的视频，请启用交互式编辑器并将 `src` 值更改为你选择的视频。

:::interactive_editor

```html
<iframe
  width="400"
  height="400"
  src="https://www.youtube.com/embed/PkZNo7MFNFg?si=-UBVIUNM3csdeiWF"
  title="Learn JavaScript - Full Course for Beginners (YouTube video)"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
  referrerpolicy="strict-origin-when-cross-origin"
  allowfullscreen
></iframe>
```

:::

`src` 属性指定了要嵌入的页面的 URL。 `width` 属性指定 `iframe` 的宽度。 `height` 属性指定 `iframe` 的高度。 `allowfullscreen` 属性允许用户以全屏模式显示 `iframe` 。 为 `iframe` 指定 `title` 属性也是一种好的做法，因为它对可访问性非常重要。

另一方面，`allow` 属性让你定义 iframe 可以或不可以做什么。这称为允许列表。在上面的例子中，为它添加 `clipboard-write` 允许嵌入页面向你的剪贴板写入项。允许列表中的项可以用分号或空格分隔，也可以同时使用两者。

注意，视频可以来自任何地方。 它不一定要来自 YouTube 和 Vimeo 等视频服务。

别忘了你也可以在 `iframe` 元素中嵌入地图、另一个网页或直接的超文本标记语言。下面是一个嵌入地图的示例。

尝试通过放大和缩小与地图交互。

:::interactive_editor

```html
<h1>A Map from Openstreetmap.org Embedded with the iframe Element</h1>

<iframe
  width="425"
  height="350"
  src="https://www.openstreetmap.org/export/embed.html?bbox=3.006134033203125%2C6.150112578753815%2C3.6357879638671875%2C6.749850810550778&amp;layer=mapnik"
  title="Map of Lagos area, Nigeria"
  style="border: 1px solid black"
>
</iframe>
<br />
<small>
  <a href="https://www.openstreetmap.org/#map=11/6.4501/3.3210">
    View Larger Map
  </a>
</small>
```

:::

如果你想在 `iframe` 元素中直接嵌入 HTML，则必须使用 `srcdoc` 属性而不是 `src`。
