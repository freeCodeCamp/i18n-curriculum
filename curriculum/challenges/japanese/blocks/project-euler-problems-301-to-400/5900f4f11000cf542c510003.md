# --description--

ハーシャッド数（またはニーベン数）とは、その数字の桁の合計で割り切れる数のことです。

201 は、その桁の合計である 3 で割り切れるため、ハーシャッド数です。

201 の最後の桁を切り捨てると 20 になり、これもハーシャッド数です。

20 の最後の桁を切り捨てると 2 になり、これもハーシャッド数です。

最後の桁を再帰的に切り捨てても常にハーシャッド数になるものを、右切り捨て可能なハーシャッド数と呼びましょう。

また：

$\frac{201}{3} = 67$ は素数です。

桁の合計で割った結果が素数になるハーシャッド数を強いハーシャッド数と呼びましょう。

次に、2011 という素数を考えます。これの最後の桁を切り捨てると 201 になり、これは強いハーシャッド数であり、かつ右切り捨て可能です。このような素数を強い右切り捨て可能なハーシャッド素数と呼びましょう。

10000 未満の強い右切り捨て可能なハーシャッド素数の合計は 90619 であることが与えられています。

${10}^{14}$ 未満の強い右切り捨て可能なハーシャッド素数の合計を求めてください。
