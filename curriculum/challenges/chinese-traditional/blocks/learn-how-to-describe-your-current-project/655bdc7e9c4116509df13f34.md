# --description--

`use` 是指將某物用於特定目的。例如：

`She uses a notebook to take notes in class.`－這說明她如何使用筆記本來做筆記。

`tool` 是幫助你完成工作的東西。在工作中，工具通常是軟體或電腦上的東西。例如：

`Zoom is a tool for online meetings。` - 在這種情況下，該工具是幫助人們連線和交談的軟體。
