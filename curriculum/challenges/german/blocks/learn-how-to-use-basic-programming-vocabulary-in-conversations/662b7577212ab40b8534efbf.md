# --description--

`Pretty well` bedeutet ein hohes Maß oder ziemlich gut, aber nicht perfekt. Es ist eine Möglichkeit, jemandem zu sagen, dass Sie oder jemand anderes etwas gut genug kann, aber es noch Raum für Verbesserungen gibt.

Zum Beispiel `I did pretty well in the English test. I got most of the answers right.`
