# --description--

У наступних кількох уроках ми вивчимо списки, кортежі та діапазони — три базові типи послідовностей, які використовуються в Python.

Тип даних список — це впорядкована послідовність елементів, які можуть складатися з рядків, чисел або навіть інших списків. Списки змінні і використовують індексацію, починаючи з нуля, тобто перший елемент списку має індекс нуль.

Ось базовий синтаксис для списку:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
```

Щоб отримати доступ до елемента зі списку `cities`, ви можете звернутися до його номера індексу в послідовності. Ось приклад доступу до першого елемента списку `cities`:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # 'Los Angeles'
```

Від’ємна індексація використовується для доступу до елементів, починаючи з кінця списку, замість початку з індексу `0`. Щоб отримати доступ до останнього елемента будь-якого списку, можна використати `-1` ось так:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # 'Tokyo'
```

Ще один спосіб створити список — використати конструктор `list()`. Конструктор `list()` використовується для перетворення ітерованого об’єкта у список ось так:

```py
developer = 'Jessica'
list(developer) # ['J', 'e', 's', 's', 'i', 'c', 'a']
```

Ітерований об’єкт — це спеціальний тип об’єкта, по якому можна проходити в циклі по одному елементу за раз. Ви дізнаєтеся більше про цикли в Python пізніше.

Щоб отримати загальну кількість елементів у списку, можна використати функцію `len()` ось так:

```py
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

Якщо ви хочете оновити значення за певним індексом, можна зробити так:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

Оскільки списки змінні, ви можете оновити будь-який елемент у списку, якщо передасте дійсний номер індексу. Якщо ви передасте індекс (позитивний або від’ємний), який виходить за межі списку, ви отримаєте `IndexError`:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list assignment index out of range
"""
```

Якщо ви хочете видалити елемент зі списку, можна використати ключове слово `del` ось так:

```py
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

Іноді корисно перевірити, чи є елемент у списку. Для цього можна використати ключове слово `in` ось так:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

Іноді буває, що списки вкладені всередину інших списків ось так:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

У цьому прикладі ми маємо один вкладений список, що містить три популярні мови програмування. Щоб отримати доступ до вкладеного списку, потрібно звернутися до нього за індексом `2`, оскільки списки індексуються, починаючи з нуля:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

Потім, щоб отримати другу мову з цього вкладеного списку, потрібно звернутися до неї за індексом `1` ось так:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # 'Rust'
```

Ще одна поширена техніка, що використовується зі списками, — це розпакування значень.

Розпакування значень зі списку — це техніка, яка дозволяє призначити значення зі списку новим змінним. Ось приклад розпакування списку `developer` у нові змінні `name`, `age` і `job`.

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

Тут `name` має значення `'Alice'`, `age` має значення `34`, а `job` має значення `'Rust Developer'`.

Якщо потрібно зібрати всі залишкові елементи зі списку, можна використати оператор зірочки (`*`) ось так:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

У цьому прикладі `name` все ще матиме значення `'Alice'`, а `rest` — це список із двох елементів: числа `34` і рядка `'Rust Developer'`.

Якщо кількість змінних зліва від оператора присвоєння не співпадає із загальною кількістю елементів у списку, ви отримаєте `ValueError`:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job, city = developer

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: not enough values to unpack (expected 4, got 3)
"""
```

Остання концепція, яку ми розглянемо, — це оператор зрізу (`:`). Подібно до рядків, ви можете отримувати частини списку, використовуючи оператор зрізу ось так:

```py
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie', 'Brownies']
desserts[1:4] # ['Cookies', 'Ice Cream', 'Pie']
```

У цьому прикладі початковий індекс — `1`, оскільки він вказує на другий елемент списку. Потім ми використовуємо оператор зрізу з кінцевим індексом `4`, який включає все до (але не включно з) елементом за цим індексом.

Ще одна річ, яку можна зробити з оператором зрізу `:`, — це вказати крок, який визначає, на скільки збільшувати індекси. Припустимо, у вас є список чисел ось такого вигляду:

```py
numbers = [1, 2, 3, 4, 5, 6]
```

Якщо ви хочете отримати список лише парних чисел, можна використати оператор зрізу ось так:

```py
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

Перше парне число знаходиться за індексом `1`, тому це буде початковий індекс. Оскільки ми хочемо пройти до кінця списку, кінцевий індекс опускаємо. Нарешті, ми вказуємо `2` для необов’язкового кроку, щоб збільшення відбувалося на `2`, а не за замовчуванням `1`.

Списки — це корисна і гнучка структура даних, яку ви часто використовуватимете у своїх програмах на Python. У наступному уроці ви дізнаєтеся про загальні методи, які можна застосовувати до списків.
