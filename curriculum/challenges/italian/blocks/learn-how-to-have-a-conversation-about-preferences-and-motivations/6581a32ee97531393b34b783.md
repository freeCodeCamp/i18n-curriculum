# --description--

`Encourage` significa dare a qualcuno supporto o fiducia per fare qualcosa. Per esempio, quando il tuo amico dice `You can solve this coding problem!` ti sta incoraggiando.

`Discourage` è l'opposto; significa far sentire qualcuno meno sicuro o meno propenso a fare qualcosa. Se quell'amico dice `That programming language is too hard for beginners`, potrebbe scoraggiarti dal impararlo.

`Discouraging` descrive qualcosa che ti fa sentire scoraggiato. Come, `Finding a bug in your code after hours of work can be discouraging.`

`Encouraging` è quando qualcosa ti dà coraggio o speranza. Per esempio, `Getting your first program to run is very encouraging.`
