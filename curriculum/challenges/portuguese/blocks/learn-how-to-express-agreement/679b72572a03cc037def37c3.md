`Doable` significa que algo é possível ou realizável.É frequentemente usado em conversas informais para expressar que uma tarefa ou objetivo pode ser concluído.Por exemplo:

- `Finishing this project by Friday is doable.` – Isso significa que completar o projeto até sexta-feira é possível.
