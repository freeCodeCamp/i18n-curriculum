# --description--

誰かに何かを欲しいか尋ねるとき、スペイン語では動詞 `querer` を使います。

動詞の形 `quieres` は `tú`（あなた）と一緒に使われ、相手の望みや好みを尋ねるときに使います。日常会話でよく使われます。例えば：

- `¿Quieres ayuda?` - 助けが欲しいですか？

- `¿Quieres más tiempo?` - もっと時間が欲しいですか？
