# --description--

`Changes` en este contexto significa modificaciones o ediciones realizadas en el código.

`Descriptive` significa dar detalles sobre cómo luce, se siente, suena, huele o sabe algo. Es un adjetivo usado para proporcionar más información sobre un sustantivo, haciendo que la descripción sea más vívida y clara. Por ejemplo, en la oración `she wears a beautiful dress`, la palabra `beautiful` es `descriptive` porque les dice a los oyentes más sobre el vestido.

Un `message` es una pieza de información o comunicación que alguien envía a otra persona. Puede ser escrito, hablado, o incluso enviado por correo electrónico o texto.
