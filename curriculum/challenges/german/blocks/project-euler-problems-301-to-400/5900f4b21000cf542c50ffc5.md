# --description--

Sei $a_n$ eine Folge, die rekursiv definiert ist durch: $a_1 = 1$, $\displaystyle a_n = \left(\sum_{k = 1}^{n - 1} k \times a_k\right)\bmod n$.

Die ersten 10 Elemente von $a_n$ sind also: 1, 1, 0, 3, 0, 3, 5, 4, 1, 9.

Sei $f(N, M)$ die Anzahl der Paare $(p, q)$, so dass:

$$ 1 \le p \le q \le N \\; \text{und} \\; \left(\sum_{i = p}^q a_i\right)\bmod M = 0$$

Es ist ersichtlich, dass $f(10, 10) = 4$ mit den Paaren (3,3), (5,5), (7,9) und (9,10).

Es ist außerdem gegeben, dass $f({10}^4, {10}^3) = 97\\,158$.

Finden Sie $f({10}^{12}, {10}^6)$.
