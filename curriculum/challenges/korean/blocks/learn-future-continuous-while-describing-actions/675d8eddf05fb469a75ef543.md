# --explanation--

`Solution`는 문제를 해결하는 답변이나 방법을 의미합니다. 예를 들어:

`The engineers worked hard to find a solution to the network issue.` - 엔지니어들이 네트워크 문제를 해결하기 위해 열심히 일했다는 뜻입니다.

`Frustrated`는 무언가가 작동하지 않거나 지연될 때 느끼는 짜증이나 불만을 설명합니다. 예를 들어:

`The customers were frustrated by the long wait times.` - 지연 때문에 고객들이 느낀 짜증과 불만을 나타냅니다.

`Do`는 부정문 뒤에 오는 부가 의문문에서 동의를 확인하거나 점검할 때 사용됩니다. 예를 들어:

`We don't need to leave now, do we?` - 주절이 부정문이고, 부가 의문문(`do we?`)은 긍정문으로 대화 중 확인을 요청하는 형태입니다.
