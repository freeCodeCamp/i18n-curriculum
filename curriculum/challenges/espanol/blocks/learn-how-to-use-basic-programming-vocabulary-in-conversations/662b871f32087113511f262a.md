# --description--

El patrón `be` seguido de la forma `-ing` de un verbo se usa para hablar de acciones que están ocurriendo en el momento.

La frase `have trouble with` se utiliza para expresar dificultad en tratar con un problema específico o objeto. Por ejemplo, `She is having trouble with her computer` significa que ella está experimentando dificultades al usar su computadora.
