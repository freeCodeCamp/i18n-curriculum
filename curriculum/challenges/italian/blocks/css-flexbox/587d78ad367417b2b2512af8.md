# --description--

La proprietà `align-items` è simile a `justify-content`. Ricorda che la proprietà `justify-content` allineava gli elementi flex lungo l'asse principale. Per le righe, l'asse principale è una linea orizzontale e per le colonne è una linea verticale.

I contenitori flex hanno anche un **asse trasversale** che è opposto all'asse principale. Per le righe, l'asse trasversale è verticale e per le colonne è orizzontale.

CSS offre la proprietà `align-items` per allineare gli elementi flex lungo l'asse trasversale. Per una riga, indica a CSS come spingere gli elementi dell'intera riga verso l'alto o verso il basso all'interno del contenitore. E per una colonna, come spingere tutti gli elementi a sinistra o a destra all'interno del contenitore.

I diversi valori disponibili per `align-items` includono:

<ul><li><code>flex-start</code>: allinea gli elementi all'inizio del contenitore flex. Per le righe, questo allinea gli elementi in alto nel contenitore. Per le colonne, allinea gli elementi a sinistra del contenitore.</li><li><code>flex-end</code>: allinea gli elementi alla fine del contenitore flex. Per le righe, questo allinea gli elementi in basso nel contenitore. Per le colonne, allinea gli elementi a destra del contenitore.</li><li><code>center</code>: allinea gli elementi al centro. Per le righe, li allinea verticalmente (spazio uguale sopra e sotto gli elementi). Per le colonne, li allinea orizzontalmente (spazio uguale a sinistra e a destra degli elementi).</li><li><code>stretch</code>: estende gli elementi per riempire il contenitore flex. Per esempio, gli elementi delle righe sono estesi per riempire il contenitore flex dall'alto al basso. Questo è il valore predefinito se non viene specificato alcun valore per <code>align-items</code>.</li><li><code>baseline</code>: allinea gli elementi alle loro linee di base. La linea di base è un concetto tipografico, pensala come la linea su cui si appoggiano le lettere.</li></ul>

# --instructions--

Un esempio aiuta a mostrare questa proprietà in azione. Aggiungi la proprietà CSS `align-items` all'elemento `#box-container` e impostala con il valore `center`.

**Bonus**  
Prova le altre opzioni per la proprietà `align-items` nell'editor di codice per vedere le loro differenze. Ma tieni presente che solo il valore `center` supererà questa sfida.

# --hints--

L'elemento `#box-container` dovrebbe avere una proprietà `align-items` impostata con il valore `center`.

```js
const boxContainer = document.querySelector('#box-container');
const alignment = window.getComputedStyle(boxContainer)['align-items'];
assert.strictEqual(alignment, 'center');
```
