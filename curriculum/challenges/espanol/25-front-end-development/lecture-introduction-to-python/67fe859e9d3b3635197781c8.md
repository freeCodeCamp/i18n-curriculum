# --description--

Los enteros y flotantes son los principales tipos de datos numéricos en Python. Con ellos, puedes almacenar datos numéricos y realizar operaciones matemáticas.

Veamos qué son los enteros y flotantes, cómo realizar cálculos aritméticos con ellos y algunos métodos que Python ofrece para trabajar con ambos.

Los enteros son números enteros sin puntos decimales, ya sean positivos o negativos:

```python
my_int_1 = 56
my_int_2 = -4

print(type(my_int_1)) # <class 'int'>
print(type(my_int_2)) # <class 'int'>
```

Aquí está cómo realizar una operación de suma con enteros:

```python
my_int_1 = 56
my_int_2 = 12

sum_ints = my_int_1 + my_int_2
print('Integer Addition:', sum_ints) # Integer Addition: 68
```

Aquí está cómo realizar una resta con enteros:

```python
my_int_1 = 56
my_int_2 = 12

# Subtraction
diff_ints = my_int_1 - my_int_2
print('Integer Subtraction:', diff_ints) # Integer Subtraction: 44
```

Aquí está cómo realizar una operación de multiplicación con enteros:

```python
my_int_1 = 12
my_int_2 = 4

# Multiplication
product_ints = my_int_1 * my_int_2
print('Integer Multiplication:', product_ints) # Integer Multiplication: 48
```

Y aquí es cómo realizar una operación de división con enteros:

```python
my_int_1 = 56
my_int_2 = 12

# Division
div_ints = my_int_1 / my_int_2
print('Integer Division:', div_ints) # Integer Division: 4.666666666666667
```

Los flotantes son números positivos o negativos con puntos decimales, como `3.14`, `-0.5` o `0.0`.

```python
my_float_1 = -12.0
my_float_2 = 4.9

print(type(my_float_1)) # <class 'float'>
print(type(my_float_2)) # <class 'float'>
```

Aquí está una operación de suma con flotantes:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_addition = my_float_1 + my_float_2
print('Float Addition:', float_addition) # Float Addition: 17.4
```

Aquí está una operación de resta con flotantes:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_subtraction = my_float_2 - my_float_1
print('Float Subtraction:', float_subtraction) # Float Subtraction: 6.6
```

Aquí está una operación de multiplicación con flotantes:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_multiplication = my_float_2 * my_float_1
print('Float Multiplication:', float_multiplication) # Float Multiplication: 64.80000000000001
```

Y aquí es cómo realizar una operación de división con flotantes:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_division = my_float_2 / my_float_1
print('Float Division:', float_division) # Float Division: 2.222222222222222
```

Si sumas un entero y un flotante, el resultado se convierte automáticamente en un flotante:

```python
my_int = 56
my_float = 5.4

sum_int_and_float = my_int + my_float

print(sum_int_and_float) # 61.4
print(type(sum_int_and_float)) # <class 'float'>
```

Esto es cierto también para otras operaciones aritméticas básicas, como resta, multiplicación y división. Si mezclas enteros y flotantes, Python devolverá un flotante como resultado.

También puedes realizar cálculos aritméticos más complejos, como obtener el residuo de dos números con el operador de módulo, división con redondeo hacia abajo y exponentiación con tanto enteros como flotantes.

El operador de módulo (`%`) devuelve el residuo cuando el valor de la izquierda se divide entre el valor de la derecha:

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

mod_ints = my_int_1 % my_int_2
mod_floats = my_float_2 % my_float_1

print('Integer Modulus:', mod_ints) # Integer Modulus: 8
print('Float Modulus:', mod_floats) # Float Modulus: 1.1999999999999993
```

La división con redondeo divide dos números y redondea el resultado hacia abajo hasta el entero más cercano. Esto se hace con el operador de doble barra inclinada hacia adelante (`//`):

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

floor_div_ints = my_int_1 // my_int_2
floor_div_floats = my_float_2 // my_float_1

print('Integer Floor Division:', floor_div_ints) # Integer Floor Division: 4
print('Float Floor Division:', floor_div_floats) # Float Floor Division: 2.0
```

La exponentiación eleva un número a la potencia de otro, y se hace con el operador de doble asterisco (`**`):

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

exp_ints = my_int_1 ** my_int_2
exp_floats = my_float_1 ** my_float_2

print('Integer Exponentiation:', exp_ints) # Integer Exponentiation: 951166013805414055936
print('Float Exponentiation:',  exp_floats) # Float Exponentiation: 614787626.1765089
```

Python también ofrece funciones incorporadas para convertir datos numéricos o cadenas de texto en enteros o flotantes. Puedes usar la función `float()` para convertir un entero en un flotante agregando un punto decimal de `0` al entero:

```python
my_int_1 = 56
my_float_1 = float(my_int_1)

print(my_float_1)  # 56.0
print(type(my_float_1))  # <class 'float'>
```

Y puedes usar la función `int()` para convertir un flotante en un entero, lo que elimina el punto decimal y todo lo que le sigue en el flotante que le pasas:

```python
my_float = 12.92563
my_int = int(my_float)

print(my_int)  # 12
print(type(my_int))  # <class 'int'>
```

También puedes usar las mismas funciones incorporadas para convertir una cadena de texto en un flotante o entero:

```python
my_str_int = '45'
my_str_float = '7.8'

converted_int = int(my_str_int)
converted_float = float(my_str_float)

print(converted_int, type(converted_int))  # 45 <class 'int'>
print(converted_float, type(converted_float))  # 7.8 <class 'float'>
```

Aquí hay algunos otros métodos que Python ofrece para trabajar con enteros y flotantes.

- `round()`: Redondea un número a un número especificado de lugares decimales. Por defecto, esta función redondea al entero más cercano y devuelve un número entero sin lugares decimales:


```python
my_int_1 = 4.798
my_int_2 = 4.253

rounded_int_1 = round(my_int_1)
rounded_int_2 = round(my_int_2, 1)

print(rounded_int_1) # 5
print(rounded_int_2) # 4.3
```

- `abs()`: devuelve el valor absoluto de un número,


```python
num = -15

absolute_value = abs(num)
print(absolute_value) # 15
```

- `bin()`: convierte un entero en su representación binaria como cadena de texto.

- `oct()`: convierte un entero en su representación octal como cadena de texto.

- `hex()`: convierte un entero en su representación hexadecimal como cadena de texto.


```python
my_int = 56

binary_representation = bin(my_int)
print(binary_representation)  # 0b111000

octal_representation = oct(my_int)
print(octal_representation) # 0o70

hex_representation = hex(my_int)
print(hex_representation) # 0x38
```

- `pow()`: eleva un número a la potencia de otro o realiza exponentiación modular.


```python
result_1 = pow(2, 3)  # Equivalent to 2 ** 3
print(result_1)  # 8

result_2 = pow(2, 3, 5)  # (2 ** 3) % 5
print(result_2)  # 3
```
