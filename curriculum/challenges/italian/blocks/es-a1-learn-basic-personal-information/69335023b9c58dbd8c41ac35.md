# --explanation--

Ángela dice `¿Y tu profesión?`, una domanda di seguito breve. `Y` collega questa domanda all'argomento precedente.

Anche se non dice `¿Cuál es...?`, la domanda completa si capisce dal contesto:

`¿Cuál es tu profesión?`

Nelle conversazioni, lo spagnolo spesso omette parole ripetute quando il significato è chiaro.
