# --description--

Cuando quieres preguntar sobre algo específico, puedes usar la palabra interrogativa `what`. `What` se usa comúnmente para hablar sobre cosas. 

Usualmente pones `what` al comienzo de una pregunta. Después lo continuas con el verbo `to be` (cómo `is` o `are`) o otros verbos, y después el subjeto (cómo `you` o `the team`). Por ejemplo:

- `What is your favorite color?` – Alguien quiere saber el color que más te gusta.

- `What does this code do?` - Alguien quiere saber la función o el resultado del código.
