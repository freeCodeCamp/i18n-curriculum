# --description--

**Ziel:** Erstellen Sie eine Anwendung, die funktional ähnlich ist wie diese: <a href="https://p2p-video-chat-application.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://p2p-video-chat-application.freecodecamp.rocks/</a>.

Erfüllen Sie die untenstehenden User Stories und bringen Sie alle Tests zum Bestehen. Verwenden Sie beliebige Bibliotheken oder APIs, die Sie benötigen. Verleihen Sie der Anwendung Ihren eigenen Stil.

**User Story:** Beim Aufrufen fordert mich der Browser auf, auf meine Kamera und mein Mikrofon zuzugreifen.

**User Story:** Nachdem ich die Erlaubnis erteilt habe, werde ich aufgefordert, einen Raumnamen einzugeben.

**User Story:** Sobald ich den Raumnamen eingebe, wird ein Raum erstellt, falls es zuvor keinen Raum mit diesem Namen gab.

**User Story:** Ein Freund von mir kann anschließend dieselbe Website besuchen, denselben Raum eingeben und dem gleichen Raum beitreten, um dann mit mir per Video zu chatten.

**User Story:** Wenn ich einen Raumnamen eingebe und sich bereits zwei Personen in diesem Raum befinden, erhalte ich eine Benachrichtigung, dass der Raum voll ist.

**User Story:** Jeder kann jeden Raum erstellen oder betreten. Es kann beliebig viele Räume geben, aber alle müssen eindeutige Namen haben.

**User Story:** Ich kann wählen, der Website den Zugriff auf mein Mikrofon und meine Webcam zu verweigern. Wenn ich dies tue oder ein anderer Treiberfehler auftritt, sehe ich eine Fehlermeldung, die besagt, dass diese erforderlich sind.

**User Story:** Wenn ich den Schritt zur Eingabe des Raumnamens abbreche oder keinen Namen oder nur Leerzeichen eingebe, werde ich erneut aufgefordert, einen gültigen Raumnamen einzugeben.

**User Story:** Wenn eine der beiden Personen im Raum die Verbindung verliert, kann sie sich wieder mit demselben Raum verbinden und das Chatten fortsetzen.

Nachdem Sie diese User Stories implementiert haben, geben Sie die URL Ihrer Live-Anwendung und optional Ihr GitHub-Repository ein. Klicken Sie dann auf die Schaltfläche „Ich habe diese Aufgabe abgeschlossen“.
