# --description--

多边形是由直线段组成的扁平形状，所述直线段连接以形成闭合链或电路。 多边形由至少三个边组成，并且不自相交。

一组正数 $S$ 被称为生成多边形 $P$，如果：

- $P$ 的任意两边长度都不相同，
- $P$ 的每条边的长度都在 $S$ 中，且
- $S$ 不包含其他值。

例如：

集合 {3, 4, 5} 生成一个边长为 3、4 和 5 的多边形（三角形）。

集合{6,9,11,24}生成具有边6,9,11和24（四边形）的多边形。

集合{1,2,3}和{2,3,4,9}根本不生成任何多边形。

考虑序列 $s$，定义如下：

- $s_1 = 1$，$s_2 = 2$，$s_3 = 3$
- $s_n = s_{n - 1} + s_{n - 3}$，当 $n > 3$ 时。

设 $U_n$ 为集合 $\\{s_1, s_2, \ldots, s_n\\}$。例如，$U_{10} = \\{1, 2, 3, 4, 6, 9, 13, 19, 28, 41\\}$。

令 $f(n)$ 为生成至少一个多边形的 $U_n$ 的子集的数量。

例如，$f(5) = 7$，$f(10) = 501$ 和 $f(25) = 18\\,635\\,853$。

查找 $f({10}^{18})$ 的最后 9 位数字。
