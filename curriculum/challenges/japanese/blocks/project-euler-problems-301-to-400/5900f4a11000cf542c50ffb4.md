# --description--

古典的な「はしご交差」問題では、狭く平らな通りの反対側の壁に立てかけられた2本のはしごの長さ $x$ と $y$ が与えられます。また、2本のはしごが交差する通りの上の高さ $h$ も与えられ、通りの幅 ($w$) を求める問題です。

<img alt="はしご x と y が高さ h で交差し、幅 w の通りの反対側の壁に立てかけられている図" src="https://cdn.freecodecamp.org/curriculum/project-euler/integer-ladders.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

ここでは、4つの変数すべてが正の整数である場合のみを考えます。例えば、$x = 70$、$y = 119$、$h = 30$ のとき、$w = 56$ と計算できます。

実際、整数値 $x$、$y$、$h$ で $0 < x < y < 200$ の範囲では、$w$ が整数解となる三つ組 ($x$, $y$, $h$) は5つだけです：(70, 119, 30)、(74, 182, 21)、(87, 105, 35)、(100, 116, 35)、(119, 175, 40)。

整数値 $x$、$y$、$h$ で $0 < x < y < 1\,000\,000$ の範囲では、$w$ が整数解となる三つ組 ($x$, $y$, $h$) はいくつありますか？
