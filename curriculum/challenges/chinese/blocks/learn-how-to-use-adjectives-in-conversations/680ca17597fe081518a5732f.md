`Last time` 指的是最近的事件。例如：

`Last time we updated the server, we had some downtime.` – 这表示在上一次更新期间。

`Order` 意味着正式请求商品或设备。它的过去形式是 `ordered`。例如：

`The IT team ordered new laptops for the developers.` – 这意味着他们为笔记本电脑提交了一个请求。

`Double that` 意味着前一个数量的两倍。例如：

`Last time we ordered 10 routers; this time we need double that, so 20 routers.` – 这表明数量增加了两倍。
