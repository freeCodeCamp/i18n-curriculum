# --description--

Reactにはデフォルトプロップスを設定するオプションもあります。コンポーネント自体のプロパティとしてデフォルトプロップスを代入すると、必要に応じてReactがそのデフォルトプロップを割り当てます。これにより、明示的に値が指定されていない場合にプロップの値が何になるべきかを指定できます。たとえば、`MyComponent.defaultProps = { location: 'San Francisco' }`を宣言すると、locationプロップが文字列`San Francisco`に設定されます（別の値を指定しない限り）。Reactはプロップが未定義の場合にデフォルトプロップを割り当てますが、もしプロップの値として`null`を渡した場合は、そのまま`null`になります。

# --instructions--

コードエディターには`ShoppingCart`コンポーネントが表示されています。このコンポーネントにデフォルトプロップスを定義し、`items`というプロップに`0`という値を指定してください。

# --hints--

`ShoppingCart`コンポーネントがレンダリングされるはずです。

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(ShoppingCart));
    return mockedComponent.find('ShoppingCart').length === 1;
  })()
);
```

`ShoppingCart`コンポーネントは`{ items: 0 }`というデフォルトプロップを持つべきです。

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(ShoppingCart));
    mockedComponent.setProps({ items: undefined });
    return mockedComponent.find('ShoppingCart').props().items === 0;
  })()
);
```
