# --interactive--

Кастомні властивості CSS, також відомі як CSS-змінні, — це сутності, які визначають автори CSS і які містять конкретні значення для повторного використання по всьому документу. Це потужна функція, що дозволяє створювати більш ефективні, підтримувані та гнучкі таблиці стилів.

Синтаксис оголошення кастомної властивості простий. Він починається з двох дефісів (`--`), за якими йде ім’я властивості:

```css
:root {
  --main-color: #3498db;
}
```

У цьому прикладі ми оголошуємо кастомну властивість з ім’ям `--main-color` зі значенням `#3498db`. Псевдоклас `:root` часто використовується для оголошення глобальних кастомних властивостей, оскільки він представляє найвищий рівень батьківського елемента в DOM-дереві.

Щоб використати кастомну властивість, застосовуйте функцію `var()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
:root {
  --main-color: #3498db;
}

.button {
  background-color: var(--main-color);
}
```

:::

Це встановлює колір фону елементів із класом `button` у значення, що зберігається в `--main-color`.

Одна з ключових функцій кастомних властивостей полягає в тому, що вони підпорядковуються каскаду CSS. Це означає, що ви можете перевизначати їх для конкретних елементів або контекстів:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="alert">This is an alert message.</div>
```

```css
.alert {
  --main-color: #e74c3c;
  background-color: var(--main-color);
}
```

:::

У цьому випадку елементи з класом `alert` використовуватимуть інше значення `--main-color`, яке перекриває глобальне визначення.

Кастомні властивості також підтримують запасні варіанти. Якщо кастомна властивість не визначена або є недійсною, ви можете надати запасне значення:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="text">This is some text.</div>
```

```css
:root {
  --text-color: green;
}

.text {
  color: var(--text-color, green);
}
```

:::

Тут, якщо `--text-color` не визначена, властивість `color` за замовчуванням прийме значення `green`.

Кастомні властивості особливо корисні для створення темізованих дизайнів. Ви можете визначити множину властивостей для різних тем:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="light-theme">
  <p>This is some light themed text.</p>
</div>
```

```css
:root {
  --bg-color: black;
}

body {
  background-color: var(--bg-color);
}

.light-theme {
  --bg-color: white;
  --text-color: black;
  background-color: var(--bg-color);
  color: var(--text-color);
  padding: 10px;
}

```

:::

Перемикання тем стає таким простим, як додати або видалити клас у батьківському елементі `body`.

Кастомні властивості також можна використовувати з медіазапитами для створення адаптивних дизайнів:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <h2>Responsive Design</h2>
  <p>Resize the window to see the card adapt!</p>
</div>

```

```css
:root {
  --card-width: 90%;
  --card-bg: #f0f0f0;
  --card-padding: 1rem;
  --text-color: #333;
}

/* Tablet screens and up */
@media (min-width: 600px) {
  :root {
    --card-width: 70%;
    --card-bg: #e8f5e9;
    --card-padding: 1.5rem;
  }
}

/* Desktop screens and up */
@media (min-width: 1024px) {
  :root {
    --card-width: 50%;
    --card-bg: #d0f0ff;
    --card-padding: 2rem;
  }
}

body {
  font-family: system-ui, sans-serif;
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
  background-color: #fafafa;
}

.card {
  width: var(--card-width);
  background-color: var(--card-bg);
  padding: var(--card-padding);
  color: var(--text-color);
  border-radius: 8px;
  box-shadow: 0 2px 6px rgba(0,0,0,0.1);
  text-align: center;
  transition: all 0.3s ease;
}

```

:::

Такий підхід дозволяє створювати більш підтримувані адаптивні макети, оскільки ви можете змінювати значення в одному місці, а не по всій таблиці стилів. Це суттєво зменшує повторення у ваших стилях, особливо для часто використовуваних складних значень.

Кастомні властивості також можуть посилатися на інші кастомні властивості:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
:root {
  --primary-color: #3498db;
  --button-bg: var(--primary-color);
}

button {
  background-color: var(--button-bg);
}

```

:::

Це дозволяє створювати зв’язки між різними аспектами стилю, що полегшує підтримку узгоджених тем на вашому сайті.

Хоча кастомні властивості мають багато переваг, важливо пам’ятати про їх підтримку в браузерах. Вони добре підтримуються в сучасних браузерах, але старі браузери можуть їх не розпізнавати. Завжди майте запасний план при використанні передових функцій.

Підсумовуючи, кастомні властивості CSS надають потужний спосіб створювати більш динамічні, гнучкі та підтримувані таблиці стилів. Вони дозволяють створювати темізовані дизайни, спрощують адаптивні макети та дають змогу змінювати стилі під час виконання.

Оскільки веб-розробка продовжує розвиватися, кастомні властивості стають дедалі важливішим інструментом у наборі розробника, відкриваючи нові можливості для створення адаптивного та ефективного CSS.
