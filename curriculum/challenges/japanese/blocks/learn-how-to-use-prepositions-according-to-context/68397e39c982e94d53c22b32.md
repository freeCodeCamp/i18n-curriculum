# --explanation--

前置詞 `on` は多くの意味を持ちますが、このタスクでは話題や取り組んでいるテーマを示すために使う必要があります。

`Working on the new software update` はチームが特定のプロジェクトや話題に努力と注意を集中していることを意味します。

他の文は `on` を使って時間（`on the weekend`、`on time`）や場所・機器（`on his computer`）を示しており、異なる使い方です。
