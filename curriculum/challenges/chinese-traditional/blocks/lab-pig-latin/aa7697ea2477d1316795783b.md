# --description--

Pig Latin 是一種透過遵循特定轉換規則來改變英文單字的方法。

**目標：** 完成以下使用者故事並通過所有測試以完成實驗。

**使用者故事：**

1. 你應該創建一個 `translatePigLatin` 函式（程式），該函式接受一個字串作為引數。
1. 如果字串引數以子音開頭，你的函式應該取第一個子音或子音群，將它移到字的末尾，為它添加 `ay`，並傳回結果。
1. 如果字串引數以母音開頭，你的函式（程式）應該在結尾添加 `way` 並傳回結果。
1. 你的函式（程式）應該控制代碼處理字串引數，其中第一個母音出現在字的中間，並傳回適當轉換後的字串。
1. 如果字串引數沒有母音，你的函式應該在結尾添加 `ay` 並傳回結果。

注意：在此實驗的背景關係中，母音是 `a`、`e`、`i`、`o` 和 `u`。字母 `y` 不被視為母音。

```js
function translatePigLatin(str) {
  if (isVowel(str.charAt(0))) return str + "way";
  let front = [];
  str = str.split('');
  while (str.length && !isVowel(str[0])) {
    front.push(str.shift());
  }
  return [].concat(str, front).join('') + 'ay';
}

function isVowel(c) {
  return ['a', 'e', 'i', 'o', 'u'].indexOf(c.toLowerCase()) !== -1;
}
```
