# --description--

すべての桁が1で構成される数はレプユニットと呼ばれます。$R(k)$を長さ$k$のレプユニットと定義します。例えば、$R(6) = 111111$です。

$n$が正の整数で$GCD(n, 10) = 1$であるとき、$R(k)$が$n$で割り切れるような値$k$が必ず存在し、その中で最小の値を$A(n)$とします。例えば、$A(7) = 6$、$A(41) = 5$です。

$A(n)$が初めて10を超える最小の$n$は17です。

$A(n)$が初めて100万を超える最小の$n$を求めてください。

# --hints--

`repunitDivisibility()`は`1000023`を返すべきです。

```js
assert.strictEqual(repunitDivisibility(), 1000023);
```
