# --description--

La sentencia de Bob usa el tiempo pasado continuo `I was checking` para describir una acción que estaba en curso en el pasado.

Puedes crear sentencias en pasado continuo combinando el verbo to be en pasado `was` o `were` + `verb + ing`.

También usa el verbo modal `should` para sugerir un curso de acción.
