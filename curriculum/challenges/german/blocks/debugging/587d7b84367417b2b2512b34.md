# --description--

Sie können `typeof` verwenden, um die Datenstruktur oder den Typ einer Variable zu prüfen. Das ist beim Debuggen hilfreich, wenn Sie mit mehreren Datentypen arbeiten. Wenn Sie denken, dass Sie zwei Zahlen addieren, aber eine davon tatsächlich ein String ist, können die Ergebnisse unerwartet sein. Typfehler können sich in Berechnungen oder Funktionsaufrufen verstecken. Seien Sie besonders vorsichtig, wenn Sie auf externe Daten in Form eines JavaScript Object Notation (JSON)-Objekts zugreifen und damit arbeiten.

Hier sind einige Beispiele mit `typeof`:

```js
console.log(typeof "");
console.log(typeof 0);
console.log(typeof []);
console.log(typeof {});
```

Die Konsole zeigt nacheinander die Strings `string`, `number`, `object` und `object` an.

JavaScript erkennt sieben primitive (unveränderliche) Datentypen: `Boolean`, `Null`, `Undefined`, `Number`, `String`, `Symbol` (neu mit ES6) und `BigInt` (neu mit ES2020) sowie einen Typ für veränderliche Elemente: `Object`. Beachten Sie, dass Arrays in JavaScript technisch gesehen eine Art Objekt sind.
