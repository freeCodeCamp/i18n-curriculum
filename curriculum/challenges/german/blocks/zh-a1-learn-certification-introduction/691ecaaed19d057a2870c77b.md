# --description--

Mit den benutzerdefinierten Eingaben von freeCodeCamp können Sie Pinyin und chinesische Schriftzeichen direkt in den Aufgaben eingeben.

Wenn Sie eine chinesische Tastatur auf Ihrem Computer verwenden, können Sie Pinyin eingeben und die chinesischen Schriftzeichen aus den Vorschlägen auswählen. So richten Sie sie ein.

## Einrichtung unter Windows 11  

1. Öffnen Sie **Einstellungen** über das Startmenü.

2. Gehen Sie zu **Zeit und Sprache** und klicken Sie dann auf **Sprache und Region**.

3. Gehen Sie zu **Bevorzugte Sprachen** und wählen Sie **Sprache hinzufügen**.

4. Suchen Sie nach **Chinesisch** und wählen Sie **Chinesisch (vereinfacht)** aus.

5. Klicken Sie auf **Weiter** und folgen Sie den Schritten.

Sobald die Sprache installiert ist:  

- Finden Sie sie in Ihrer Liste der **Bevorzugten Sprachen**.

- Klicken Sie auf die Auslassungspunkte, um weitere Optionen anzuzeigen, und wählen Sie **Sprachoptionen**.

- Gehen Sie zu **Tastaturen**, klicken Sie auf **Tastatur hinzufügen** und wählen Sie das chinesische (vereinfacht) Layout aus.

Um zwischen Tastaturen zu wechseln:  

- Drücken Sie **Windows-Taste + Leertaste**.

- Oder klicken Sie auf das Sprachsymbol in der Taskleiste (oft mit **ENG** gekennzeichnet) und wählen Sie Chinesisch aus.

## Einrichtung auf dem Mac  

**HINWEIS:** Die Schritte können je nach Betriebssystem variieren.

1. Öffnen Sie das **Apple-Menü** und wählen Sie **Systemeinstellungen**.

2. Gehen Sie zu **Tastatur**.

3. Klicken Sie auf den Reiter **Eingabequellen**.

4. Wählen Sie die **Plus-Taste (+)**, klicken Sie auf **Chinesisch, vereinfacht** und wählen Sie aus der Liste **Pinyin – vereinfacht** aus.  

Um zwischen Tastaturen zu wechseln:

- Drücken Sie **Control + Leertaste**.

- Oder klicken Sie auf das Eingabesymbol in der Menüleiste und wählen Sie die Pinyin-Methode aus.

Jetzt sind Sie startklar.
