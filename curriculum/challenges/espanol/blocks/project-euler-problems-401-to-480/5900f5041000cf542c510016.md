# --description--

Si calculamos $a^2\bmod 6$ para $0 ≤ a ≤ 5$ obtenemos: 0, 1, 4, 3, 4, 1.

El valor más grande de a tal que $a^2 ≡ a\bmod 6$ es $4$.

Llamemos $M(n)$ al valor más grande de $a < n$ tal que $a^2 ≡ a (\text{mod } n)$. Entonces $M(6) = 4$.

Encuentra $\sum M(n)$ para $1 ≤ n ≤ {10}^7$.
