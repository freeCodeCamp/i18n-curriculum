# --interactive--

CSS Grid 是一個強大的版面配置系統，讓網頁開發者輕鬆創建複雜且響應式的網頁版面配置。

想像你正在佈置房間的家具——CSS Grid 就像是在地板上有一個隱形的格線，幫助你精確地將所有東西放在你想要的位置。

當我們建置網站時，經常需要在頁面上排列不同的元素。

在 CSS Grid 出現之前，這有時候會很棘手，尤其是對於複雜的版面配置。CSS Grid 透過將你的網頁分割成列和行，建立類似網格的結構，簡化了這個處理程序。

讓我們想像你正在使用一個包含多個巢狀項目的容器 `div`，如下所示：

:::interactive_editor

```html
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

:::

如果你想以網格格式設計那些元素，可以將 `display` 設為 `grid` 並像這樣套用行：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-gap: 20px;
}

.item {
  background-color: lightgray;
  padding: 20px;
  text-align: center;
  border: 1px solid #ccc;
}
```

:::

在這段程式碼中，我們告訴瀏覽器創建一個具有三個等寬行的網格，這就是 `1fr 1fr 1fr` 的意思，並且我們在每個網格項目之間添加了 20 像素的間隙。

現在，你可能會想：「那 Flexbox 呢？那不是也用於版面配置嗎？」

你說得對！Flexbox 是另一種 CSS 版面配置模型，而且也相當有用。但有一些關鍵差異。

Flexbox 是一維的，而 Grid 是二維的。這表示 Flexbox 非常適合在單一列或行中排列元素，而 Grid 則擅長建立同時包含列和行的版面配置。

Flexbox 是以內容為先，意即它會根據內容調整版面配置。Grid 則是以版面配置為先，允許你先創建版面配置，然後將項目放入其中。Grid 提供你更精確的放置控制。你可以告訴一個項目確切要佔據哪一列和哪一行。

這裡有一個 Flexbox 範例供比較：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
}
```

:::

這會創建一個彈性容器，其中的項目沿主軸均勻分布。

Grid 和 Flexbox 各有其優勢，且通常最佳的版面配置會結合兩者。你可能會使用 Grid 來做整體頁面佈局，然後使用 Flexbox 來對齊每個 grid 區域內的項目。

總結來說，CSS Grid 是一個強大的工具，允許精確的二維佈局。雖然一開始可能看起來複雜，但經過練習後，它會成為製作響應式且複雜網頁佈局的寶貴工具。
