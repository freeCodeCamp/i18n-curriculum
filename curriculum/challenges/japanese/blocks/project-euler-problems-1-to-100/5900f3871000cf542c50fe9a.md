# --description--

オイラーは驚くべき二次式を発見しました：

<div style='margin-left: 4em;'>$n^2 + n + 41$</div>

この式は、連続する整数値 $0 \le n \le 39$ に対して40個の素数を生成します。しかし、$n = 40$ のとき、$40^2 + 40 + 41 = 40(40 + 1) + 41$ は41で割り切れ、また $n = 41$ のとき、$41^2 + 41 + 41$ も明らかに41で割り切れます。

さらに、連続する値 $0 \le n \le 79$ に対して80個の素数を生成する驚異的な式 $n^2 - 79n + 1601$ が発見されました。係数 −79 と 1601 の積は −126479 です。

次の形の二次式を考えます：

<div style='margin-left: 4em;'>
  $n^2 + an + b$、ただし $|a| < range$ および $|b| \le range$<br>
  ここで $|n|$ は $n$ のモジュラス/絶対値です<br>
  例：$|11| = 11$ および $|-4| = 4$<br>
</div>

連続する $n$ の値（$n=0$ から始まる）に対して最大数の素数を生成する二次式の係数 $a$ と $b$ の積を求めてください。
