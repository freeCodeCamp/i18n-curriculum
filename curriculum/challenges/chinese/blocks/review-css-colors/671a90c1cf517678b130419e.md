## 色彩理论

- **色彩理论定义**：这是关于色彩之间如何相互作用以及它们如何影响我们的感知的研究。 内容包括色彩关系、色彩和谐以及色彩的心理影响。
- **原色**：这些颜色包括黄色、蓝色和红色，是所有其他颜色的基本色调。
- **间色**：这些颜色是由两种原色等量混合而成。 绿色、橙色和紫色都是间色。
- **复色**：这些颜色由一种原色和一种邻近的间色组合而成。 黄绿、蓝绿和蓝紫都属于复色。
- **暖色**：这些颜色包括红色、橙色和黄色，能唤起舒适、温暖和惬意的感觉。
- **冷色**：这些颜色包括蓝色、绿色和紫色，能唤起平静、安详和专业的感觉。
- **色轮**：色轮是一个圆形图，表示颜色之间的关系。 它是设计师的必备工具，因为它可以帮助他们选择色彩组合。
- **邻近色配色方案**：这些配色方案能营造出和谐舒缓的体验。 它们有类似的颜色，在色轮中彼此相邻。
- **互补色配色方案**：这些配色方案能产生强烈的对比和视觉冲击。 相对于彼此，它们的颜色位于色轮的两端。
- **三色配色方案**：该配色方案色彩鲜艳。 它们由彼此距离大致相等的颜色组成。 如果把它们相连，就会在色轮上形成一个等边三角形。
- **单色配色方案**： 在这种配色方案中，所有颜色都是通过调整明度、暗度和饱和度从相同的基色中衍生出来的。 这既能唤起统一和谐的感觉，又能形成对比。

## 在 CSS 中使用颜色的不同方法

- **命名颜色**：这些颜色是浏览器识别的预定义颜色名称。 包括 `blue`、`darkred`、`lightgreen`。
- **`rgb()` 函数**：RGB 代表红色、绿色和蓝色——光的三原色。这三种颜色以不同强度组合，创建出广泛的颜色范围。`rgb()` 函数允许你使用 RGB 颜色模型定义颜色。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>RGB Color</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

- **`rgba()` 函数**：该函数添加了第四个值 alpha，用于控制颜色的透明度。 如果未提供，alpha 值默认为`1`。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>RGBA Background</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

- **`hsl()` 函数**：HSL 代表色相、饱和度和亮度——是定义颜色的三个关键要素。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>HSL Color</p>
```

```css
p {
  color: hsl(120, 100%, 50%);
}
```

:::

- **`hsla()` 函数**：该函数添加了第四个值 alpha，用于控制颜色的不透明度。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>HSLA Background</div>
```


```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

- **十六进制**：hex 代码（十六进制代码的简称）是一个六个字符的字符串，用于表示 RGB 颜色模型中的颜色。 “hex”指的是十六进制计数系统，使用数字 0 至 9 和字母 A 至 F。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1 class="hex-text">Hex Text</h1>
<p class="hex-bg">Hex Background</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}

p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

## `box-shadow` 属性

- **定义**：`box-shadow` 属性在一个元素周围应用一个或多个阴影。
- **偏移值**：你必须指定水平（`offset-x`）和垂直（`offset-y`）值。正的 `offset-x` 会将阴影向右移动，负值则向左移动。正的 `offset-y` 会将阴影向下移动，负值则向上移动。如果值为 `0`，则无需包含单位。
- **模糊半径**：此可选的值控制阴影的模糊程度。如果未包含，则默认为 `0`，这会创建犀利的边缘。数值越大，阴影越柔和。
- **扩散半径**：此可选的值控制阴影的扩展或收缩程度。如果未包含，则默认为 `0`。
- **阴影颜色**：你可以使用命名颜色、十六进制值、`rgb()`、`rgba()`、`hsl()` 或 `hsla()` 函数来指定颜色。
- **`inset` 关键字**：添加 `inset` 关键字会将阴影放置在元素内部，而不是外部。
- **应用多个盒子阴影**：你可以通过逗号分隔来应用多个阴影。阴影从前到后分层。

### 语法

```css
box-shadow: offset-x offset-y blur-radius spread-radius color;
```

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="shadow-box">Shadow Color Example</div>
```

```css
.shadow-box {
  width: 200px;
  padding: 20px;
  background-color: lightblue;
  box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}
```

:::

## 线性和径向渐变

- **线性渐变**：这类渐变沿直线创建颜色之间的渐变混合。 你可以使用关键字来控制渐变的方向，如 `to top`、`to right`、`to bottom right`，或者使用角度，如 `45deg`。 你可以使用任何有效的 CSS 颜色和任意多的颜色停止点。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient">Linear Gradient</div>
```

```css
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

:::

- **径向渐变**：这些渐变可创建从中心点辐射开来的圆形或椭圆形渐变。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient">Radial Gradient</div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::
