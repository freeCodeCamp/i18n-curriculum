# --description--

集合 {1,2,..., $n$} が与えられたとき、$f(n, k)$ をその $k$ 要素部分集合のうち、要素の和が奇数になるものの数と定義します。例えば、$f(5,3) = 4$ です。なぜなら、集合 {1,2,3,4,5} には、要素の和が奇数になる3要素部分集合が4つあり、それは {1,2,4}、{1,3,5}、{2,3,4}、{2,4,5} だからです。

3つの値 $n$、$k$、$f(n, k)$ がすべて奇数のとき、それらを奇数トリプレット $[n, k, f(n, k)]$ と呼びます。

$n ≤ 10$ の範囲で奇数トリプレットはちょうど5つあり、それは $[1, 1, f(1, 1) = 1]$、$[5, 1, f(5, 1) = 3]$、$[5, 5, f(5, 5) = 1]$、$[9, 1, f(9, 1) = 5]$、$[9, 9, f(9, 9) = 1]$ です。

$n ≤ {10}^{12}$ の範囲で奇数トリプレットはいくつありますか？
