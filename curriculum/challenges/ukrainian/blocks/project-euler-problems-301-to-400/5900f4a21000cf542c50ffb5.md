# --description--

Еліс і Боб грають у гру Nim square.

Nim square — це звичайна гра Nim з трьома купами у нормальному режимі, але гравці можуть забирати лише квадратне число каменів із купи.

Кількість каменів у трьох купах представлена впорядкованою трійкою ($a$, $b$, $c$).

Якщо $0 ≤ a ≤ b ≤ c ≤ 29$, то кількість програшних позицій для наступного гравця дорівнює 1160.

Знайдіть кількість програшних позицій для наступного гравця, якщо $0 ≤ a ≤ b ≤ c ≤ 100\\,000$.
