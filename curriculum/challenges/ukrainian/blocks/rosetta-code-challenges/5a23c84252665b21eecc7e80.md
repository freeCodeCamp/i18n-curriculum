# --description--

Сірий код — це форма двійкового кодування, де переходи між послідовними числами відрізняються лише одним бітом.

Це корисне кодування для зменшення апаратних помилок при роботі зі значеннями, що швидко змінюються, і/або підключаються до повільнішого апаратного забезпечення як вхідні дані.

Воно також корисне для генерації вхідних даних для карт Карно у порядку зліва направо або зверху вниз.

# --instructions--

Створіть функцію для кодування числа у сірий код і декодування числа зі сірого коду. Функція повинна мати 2 параметри.

Перший параметр — булевий. Функція повинна кодувати, якщо він true, і декодувати, якщо false. Другий параметр — число, яке потрібно закодувати або декодувати.

Виведіть звичайні двійкові представлення, представлення у сірому коді та декодовані значення сірого коду для всіх 5-бітних двійкових чисел (від 0 до 31 включно, провідні нулі не обов’язкові).

Існує багато варіантів сірого коду. Нижче наведено кодування, яке називають «бінарним відображеним сірим кодом».

Кодування (старший біт — біт 0, b — двійкове число, g — сірий код):

<pre>if b[i-1] = 1
  g[i] = not b[i]
else
  g[i] = b[i]
</pre>

Або:

<pre>g = b xor (b logically right shifted 1 time)
</pre>

Декодування (старший біт — біт 0, b — двійкове число, g — сірий код):

<pre>b[0] = g[0]<br>
for other bits:
b[i] = g[i] xor b[i-1]
</pre>
