### --question--

#### --text--

Was ist eine relationale Datenbank?

#### --distractors--

Eine Datenbank, die so strukturiert ist, dass sie Nicht-Beziehungen zwischen gespeicherten Elementen erkennt.

---

Eine Datenbank, die unstrukturierte Daten speichert.

---

Ein flaches, dateibasiertes System.

#### --answer--

Eine Datenbank, die so strukturiert ist, dass sie Beziehungen zwischen Daten in Tabellen erkennt und verwaltet.

### --question--

#### --text--

---

Relationale Datenbanken können unstrukturierte Daten speichern.

---

Nicht-relationale Datenbanken sind langsamer als relationale Datenbanken.

#### --answer--

Nicht-relationale Datenbanken können unstrukturierte Daten speichern.
