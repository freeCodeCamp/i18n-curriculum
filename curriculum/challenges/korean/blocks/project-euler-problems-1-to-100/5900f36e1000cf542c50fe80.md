# --description--

10 미만의 자연수 중 3 또는 5의 배수를 모두 나열하면 3, 5, 6, 9가 됩니다. 이 배수들의 합은 23입니다.

제공된 매개변수 값 `number` 미만의 3 또는 5의 모든 배수의 합을 구하세요.

# --solutions--

```js
const multiplesOf3Or5 = (number) => {
  var total = 0;

  for(var i = 0; i < number; i++) {
    if(i % 3 == 0 || i % 5 == 0) {
      total += i;
    }
  }
  return total;
};
```
