# --description--

Il possessivo `'s` indica che qualcosa appartiene a qualcuno o a qualcosa, come in `Brian's book`, dove il libro appartiene a Brian.

`Dealing` significa gestire o occuparsi di qualcosa. Per esempio, `When dealing with homework, it's important to be focused.`

Un `function's signature` in programmazione è come una carta d'identità per una funzione. Ti dice il nome della funzione, cosa serve (parametri) e cosa restituisce (`return type`).

Il `return type` di una funzione è ciò che la funzione restituisce dopo essere stata eseguita, come una macchina distributrice che restituisce una bibita dopo aver inserito i soldi.
