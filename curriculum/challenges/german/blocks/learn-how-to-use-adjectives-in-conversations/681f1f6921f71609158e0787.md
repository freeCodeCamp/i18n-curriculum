# --explanation--

`Specify` bedeutet, etwas klar zu benennen oder ausführlich zu beschreiben. Zum Beispiel:

`Please specify the time and place of the meeting.` – Das bedeutet, genau zu sagen, wann und wo das Treffen stattfindet.

`What makes it` ist eine Nominalklausel. Eine Wortgruppe, die im Satz wie ein Substantiv funktioniert. Sie erklärt den Grund oder die Ursache für etwas. Zum Beispiel:

`She described what makes it so effective.` – Das bedeutet, sie erklärte den Grund, warum etwas wirksam ist. Hier fungiert die Klausel `what makes it so effective` als Objekt des Verbs `described`.
