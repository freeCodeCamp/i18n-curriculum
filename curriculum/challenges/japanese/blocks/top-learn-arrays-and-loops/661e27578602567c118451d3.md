# --description--

配列に新しい要素を追加する最も一般的な方法の一つは、`push()`メソッドを使うことです。`push()`メソッドは配列の末尾に1つ以上の要素を追加し、配列の新しい長さを返します。

例えば、`pet`配列に新しい要素を追加するには、次のコードを使えます。

```javascript
const pet = ['cat', 'dog', 'bunny'];
pet.push('parrot');
console.log(pet); // Output: ['cat', 'dog', 'bunny', 'parrot']
```

配列の最後の要素を削除するには、`pop()`メソッドを使えます。`pop()`メソッドは配列の最後の要素を削除し、その要素を返します。

例えば、`pet`配列から最後の要素を削除するには、次のコードを使えます。

```javascript
const pet = ['cat', 'dog', 'tiger'];
pet.pop();
console.log(pet); // Output: ['cat', 'dog']
```
