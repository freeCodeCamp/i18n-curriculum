# --interactive--

비교 연산자는 두 값을 비교하여 `true` 또는 `false` 결과를 반환할 수 있게 합니다. 그런 다음 그 결과를 사용해 결정을 내리거나 프로그램 흐름을 제어할 수 있습니다. 비교는 `if` 문, 반복문, 그리고 특정 조건에 따라 결정을 내려야 하는 다양한 상황에서 사용합니다. 가장 일반적인 비교 연산자를 살펴보고 어떻게 작동하는지 알아봅시다.

크다 연산자는 오른쪽 꺾쇠 괄호(`>`)로 나타내며, 왼쪽 값이 오른쪽 값보다 큰지 확인합니다:

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a > b); // false
console.log(b > a); // true
```

:::

크거나 같다 연산자는 오른쪽 꺾쇠 괄호와 등호(`>=`)로 나타내며, 왼쪽 값이 오른쪽 값보다 크거나 같은지 확인합니다:

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a >= b); // false
console.log(b >= a); // true
console.log(a >= c); // true
```

:::

작다 연산자는 왼쪽 꺾쇠 괄호(`<`)로 나타내며, `>`와 비슷하지만 반대입니다. 왼쪽 값이 오른쪽 값보다 작은지 확인합니다:

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a < b); // true
console.log(b < a); // false
```

:::

작거나 같다 연산자는 왼쪽 꺾쇠 괄호와 등호(`<=`)로 나타내며, 왼쪽 값이 오른쪽 값보다 작거나 같은지 확인합니다:

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a <= b); // true
console.log(b <= a); // false
console.log(a <= c); // true
```

:::
