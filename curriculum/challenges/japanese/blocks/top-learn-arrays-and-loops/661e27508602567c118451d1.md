# --description--

文字列や数字は基本的な要素ですが、スクリプトが複雑になるにつれて大量のデータを扱う方法が必要になります。幸い、JavaScriptにはそれに適したいくつかのデータ型があります。配列は、順序付けられたアイテムの集合（文字列、数字、またはその他のもの）です。

配列は複数の値を1つの変数に格納する方法です。これは特別な種類のオブジェクトで、lengthプロパティと番号付きのプロパティの連続を持っています。各番号付きプロパティは要素と呼ばれ、どんな型の値でも格納できます。

配列の例は以下の通りです：

```javascript
const fruits = ['apple', 'banana', 'orange'];
```
