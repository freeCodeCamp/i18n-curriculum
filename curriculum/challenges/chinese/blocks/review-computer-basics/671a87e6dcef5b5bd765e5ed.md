## 理解计算机、互联网和开发者工具基础知识

- **主板**：保存所有运行计算机所需的内存、连接器和硬盘。它作为计算机的主电路板。
- **中央处理单元(CPU)**：负责执行指令和执行计算的处理器。
- **随机存取存储器(RAM)**：计算机 CPU 的临时存储位置。
- **硬盘驱动器(HDD)**：一种永久的存储位置，即使计算机关闭也用于保存数据。
- **固态硬盘(SSD)**：非易变的闪存，可用来替代硬盘。
- **电源单元(PSU)**：负责将墙壁插座的电力转换成计算机可以使用的形式。
- **图形处理单元(GPU)**：负责在计算机屏幕上渲染视觉效果。
- **不同类型的互联网服务提供商**：互联网服务提供商（ISP）是一家提供互联网访问的公司。ISP 有不同的类型，包括拨号、DSL、有线、光纤和卫星。
- **安全登录你的计算机的方法**：安全登录你的计算机的方法示例包括使用强密码、启用双因素认证和使用密码管理器。
- **集成开发环境 (IDE)**：一种帮助开发者以高效方式编写、测试和调试代码的工具。
- **代码编辑器**：开发者用来编写和调试代码的工具。
- **Git**：一种流行的版本控制系统，允许开发者跟踪他们代码中的更改并与其他人协作。
- **基于云的仓库托管服务**：仓库是项目文件和版本历史的存储位置。流行的基于云的仓库托管服务包括 GitHub、GitLab 和 Bitbucket。
- **包管理器**：帮助开发者简化添加、更新和移除库及项目依赖的进程的工具。示例包括 npm、pip 和 Maven。
- **测试库和框架**：测试是在软件中进行的，以确保代码按预期工作。测试库和框架的示例包括 Jest、PHPUnit 和 JUnit。

## 处理文件、文件系统和媒体格式

- **命名文件的最佳实践**：你需要以易于理解和维护的方式为你的文件命名。例如，`about-us.html` 比 `page1.html` 更具描述性。
- **根目录**：文件系统中的顶层目录。目录是文件夹的另一种称呼。
- **Markdown**：一种常用于文档和 `README` 文件的标记语言。`README` 文件是包含有关项目的信息的文件，例如如何安装和使用它。
- **`index.html`**：表现用户访问网站时显示的默认页面。
- **使用 Explorer/Finder 创建、移动和删除文件和文件夹**：Explorer 是 Windows 中的文件管理器，Finder 是 macOS 中的文件管理器。你可以使用这些工具来创建、移动和删除文件和文件夹。
- **查找文件和文件夹**：你可以使用资源管理器或 Finder 中的查找功能性来查找计算机上的文件和文件夹。
- **超文本标记语言、CSS 和 JS 文件类型**：`.html` 扩展名用于超文本标记语言文件，`.css` 用于 CSS 文件，`.js` 用于 JavaScript 文件。
- **常见图像和图形格式**：`JPEG` 和 `PNG` 是常见的图像文件格式。`GIF` 是另一种支持动画的常见图像文件格式。`SVG` 是一种向量图形文件格式。
- **常见的音频和视频格式**：`MP3` 格式常用于音频文件。`MP4` 格式常用于视频文件。`MOV` 格式由 Apple 开发，常用于视频文件。
- **常见字体形式**：`TTF` 形式通常用于 TrueType 字体。`WOFF` 形式通常用于网页字体。`WOFF` 的继任者是 `WOFF2`，它提供了更好的压缩。
- **ZIP**：一种用于压缩文件和文件夹的文件格式。

## 有效浏览网页

- **什么是网页浏览器？**：一种允许用户访问和查看互联网网站的软件应用。
- **什么是搜索引擎**：一种允许用户在互联网中查找信息的工具。示例包括 Google、Bing 和 Yahoo。
- **常见的网页浏览器**：一些常见的网页浏览器示例包括 Google Chrome、Mozilla Firefox 和 Microsoft Edge。
- **常用查找策略**：你可以使用 `site:` 后跟网站的 URL 来查找该网站上的内容。你可以使用 `filetype:` 后跟扩展名来查找该类型的文件。你可以在查找词前加负号以排除包含该词的结果。你可以在查找词前加正号以包含包含该词的结果。
