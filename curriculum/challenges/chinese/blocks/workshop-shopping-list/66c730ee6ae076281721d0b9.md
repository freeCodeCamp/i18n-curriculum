# --description--

在前面的讲座中，你学习了如何使用 `pop` 方法从数组末尾移除项目。

下面是提醒你如何使用 `pop` 方法：

```js
let array = [1, 2, 3, 4, 5];
array.pop();

// [1, 2, 3, 4]
console.log(array); 
```

使用 `pop` 方法从 `shoppingList` 数组删除最后一个项目。
