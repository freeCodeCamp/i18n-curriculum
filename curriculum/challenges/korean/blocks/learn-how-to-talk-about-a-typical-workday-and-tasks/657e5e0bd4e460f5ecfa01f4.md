# --description--

대화에서, 특히 직장 내 규칙이나 절차에 대해 이야기할 때 `every`와 `any`는 서로 다른 상황을 설명하는 데 사용됩니다. `Every`는 그룹 내 모든 항목에 대해 말할 때 쓰이며, `any`는 질문이나 부정문에서 하나 이상의 항목을 가리킬 때 자주 사용됩니다.
