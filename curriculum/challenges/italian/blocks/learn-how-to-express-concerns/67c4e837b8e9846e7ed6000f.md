# --explanation--

`That's a good idea, but...` viene spesso usato per riconoscere positivamente un'idea introducendo però una preoccupazione o una limitazione. Per esempio:

`That's a good idea, but we don't have enough time to implement it.` - Questo significa che l'idea è buona, ma il tempo è un problema.
