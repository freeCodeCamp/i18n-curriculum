# --description--

もう一つできる面白いことは、ユーザーの現在位置にアクセスすることです。すべてのブラウザーには、この情報を提供できる組み込みのナビゲーターがあります。

ナビゲーターはユーザーの現在の経度と緯度を取得します。

このサイトが現在位置を知ることを許可するかブロックするかのプロンプトが表示されます。コードが正しければ、どちらを選んでもチャレンジは完了できます。

許可を選ぶと、出力画面のテキストが緯度と経度に変わるのが見えます。

これがそのコードです：

```js
if (navigator.geolocation) {
  navigator.geolocation.getCurrentPosition(function (position) {
    document.getElementById('data').innerHTML =
      'latitude: ' +
      position.coords.latitude +
      '<br>longitude: ' +
      position.coords.longitude;
  });
}
```

まず、`navigator.geolocation` オブジェクトが存在するか確認します。存在すれば、そのオブジェクトの `getCurrentPosition` メソッドが呼び出され、ユーザーの位置情報を非同期でリクエストします。リクエストが成功すると、そのメソッド内のコールバック関数が実行されます。この関数はドット記法を使って `position` オブジェクトの緯度と経度の値にアクセスし、HTMLを更新します。

# --instructions--

`script` タグ内に例のコードを追加して、ユーザーの現在位置を確認しHTMLに挿入してください。
