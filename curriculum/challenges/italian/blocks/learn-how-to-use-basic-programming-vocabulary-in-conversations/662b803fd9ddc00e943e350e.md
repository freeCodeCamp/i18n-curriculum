# --description--

Un `assignment operator` nella programmazione serve ad assegnare un valore a una variabile. In molti linguaggi di programmazione, incluso Python, il `assignment operator` è rappresentato dal simbolo `=`. Indica al computer di archiviare il valore a destra del `=` nella variabile indicata a sinistra. Per esempio, `number = 10` assegna il valore `10` alla variabile `number`.
