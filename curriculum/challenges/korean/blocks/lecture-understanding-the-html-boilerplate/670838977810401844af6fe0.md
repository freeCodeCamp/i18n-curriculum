# --description--

`link` 요소와 외부 스타일시트에 링크하는 방법을 배워보세요.

`link` 요소는 스타일시트나 사이트 아이콘 같은 외부 리소스에 링크할 때 사용합니다. 외부 CSS 파일에 `link` 요소를 사용하는 기본 구문은 다음과 같습니다:

```html
<link rel="stylesheet" href="./styles.css" />
```

`rel` 속성은 링크된 리소스와 HTML 문서 간의 관계를 지정하는 데 사용합니다. 이 경우, 이 링크된 리소스가 `stylesheet`임을 명시해야 합니다.

HTML과 CSS를 서로 다른 파일로 분리하는 것이 모범 사례로 여겨집니다. 개발자들은 HTML 문서에 모든 것을 작성하는 대신 외부 CSS 파일에 `link` 요소를 사용합니다.

`href` 속성은 외부 리소스의 URL 위치를 지정하는 데 사용합니다.

예제에서 슬래시가 뒤따르는 `dot`는 현재 폴더 또는 디렉터리에서 `styles.css` 파일을 찾으라는 의미입니다.

`link` 요소는 다음 예시처럼 `head` 요소 안에 배치해야 합니다:

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Examples of the link element</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

전문 코드베이스에서는 종종 여러 `link` 요소가 서로 다른 스타일시트, 폰트, 아이콘에 링크하는 것을 볼 수 있습니다. 다음은 *Playwright Cuba*라는 외부 Google 폰트에 `link` 요소를 사용해 링크하는 예시입니다:

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Playwrite+CU:wght@100..400&display=swap"
  rel="stylesheet"
/>
```

Google Fonts는 어떤 프로젝트에서든 사용할 수 있는 무료 오픈 소스 맞춤 폰트 세트입니다. 원하는 폰트를 선택하면 Google이 필요한 HTML과 CSS 코드를 제공합니다. 이 예시에서 `preconnect` 값은 `rel` 속성에 대해 브라우저가 `href` 속성에 지정된 값에 조기 연결을 생성하도록 지시합니다. 이는 이러한 외부 리소스의 로딩 시간을 단축하기 위해서입니다.

`link` 요소의 또 다른 일반적인 사용 사례는 아이콘에 링크하는 것입니다. 다음은 파비콘에 링크하는 예시입니다:

```html
<link rel="icon" href="favicon.ico" />
```

파비콘은 favorite icon의 줄임말로, 보통 사이트 제목 옆 브라우저 탭에 표시되는 작은 아이콘입니다. 많은 웹사이트가 브랜드 아이콘을 표시하기 위해 파비콘을 사용합니다.
