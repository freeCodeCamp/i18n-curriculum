# --description--

O método `.isalpha()` retorna `True` se todos os caracteres da string na qual ele é chamado forem letras.Por exemplo, o código abaixo retorna `True`:

```py
'freeCodeCamp'.isalpha()
# True
```

Exclua a condição `char == ' '` e substitua-a por uma chamada ao método `.isalpha()` em `char`.
