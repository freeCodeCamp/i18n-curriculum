# --interactive--

이전 수업에서는 웹 페이지에서 복잡하고 유동적인 레이아웃을 만들 때 사용할 수 있는 CSS 그리드를 소개했습니다. 이번 수업에서는 `fr` 단위를 사용해 유연한 그리드 레이아웃을 만드는 방법을 살펴보겠습니다.

그리드 컨테이너를 나타낼 이 HTML 마크업부터 시작해 보겠습니다:

```html
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

CSS 내부에서는 컨테이너에 대해 display 속성을 grid로 설정했습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 25% 25% 25% 25%;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

`grid-template-columns`은 각 열의 크기를 설정하는 데 사용됩니다. 이 경우 각 열 크기는 컨테이너의 `25%`가 됩니다. 그리고 gap 속성은 각 열 주위에 공간을 만듭니다.

지금까지 열 크기에 백분율을 사용했지만 `fr` 단위도 사용할 수 있습니다.

`fr` 단위는 그리드 컨테이너 공간의 분수를 나타내는 분수 단위입니다.

백분율 대신 `fr` 단위를 사용하도록 코드를 리팩터링하면 다음과 같습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

각 열은 사용 가능한 공간의 한 분수를 차지합니다. 네 개의 열이 있으므로 각 열은 그리드 컨테이너에서 사용 가능한 공간을 동일하게 나눠 갖습니다.

그리드 레이아웃을 만들기 시작하면 `fr` 단위를 더 자주 사용하게 될 것입니다. 이 단위는 공간을 유연하고 비례적으로 분배하는 방법을 제공해, 픽셀 값을 수동으로 조정하지 않고도 다양한 화면 크기에 맞게 반응형 레이아웃을 만들 수 있기 때문입니다.
