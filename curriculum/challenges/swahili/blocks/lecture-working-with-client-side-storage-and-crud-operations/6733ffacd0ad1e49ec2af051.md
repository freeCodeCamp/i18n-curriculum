# --description--

Kama ulivyojifunza katika mafundisho yaliyopita, vidakuzi ni data rahisi ambazo tovuti zinaweza kuhifadhi kwenye kifaa cha mtumizi.

Hata hivyo, unaweza kuhifadhi miundo ya data tata zaidi katika vidakuzi. Njia moja ya kawaida ni kutumia JSON kuhifadhi vitu au safu za data. Hapa kuna mfano:

```js
const userData = {
  name: "John Doe",
  age: 30,
  role: "admin"
};

document.cookie = "userInfo=" + JSON.stringify(userData) + "; path=/";
```

Katika mfano huu, tunaunda kitu chenye data za mtumizi, tukibadilisha kuwa mfuatano wa herufi wa JSON, kisha tukihifadhi katika kidakuzi. Tunapotaka kupata data hii, tunaweza kuchambua mfuatano wa herufi wa JSON kuwa kitu kwa kutumia `JSON.parse()`.

Sasa, unaweza kuwa unajiuliza kuhusu sehemu ya "vichwa vya HTTP" katika mada yetu. Kawaida, vidakuzi huwekwa na seva kwa kutumia vichwa vya HTTP.

Kwa mfano, seva inaweza kutuma kichwa kama hiki:

```http
Set-Cookie: username=John Doe; expires=Thu, 31 Dec 2024 6:00:00 IST; path=/
```

Kichwa hiki kinaambia kivinjari kuweka kidakuzi.

Pia tunaweza kuweka vidakuzi moja kwa moja katika kivinjari kwa kutumia JavaScript. Hii ni muhimu kuhifadhi data ambayo haihitaji kutumwa kwa seva mara moja.

Tafadhali kumbuka kuwa vidakuzi vina kikomo cha ukubwa cha takriban 4KB, na kuhifadhi data nyingi sana ndani yake kunaweza kupunguza kasi ya programu yako ya mtandao.

Kuhifadhi kiasi kikubwa cha data katika vidakuzi kunaweza kuongeza trafiki ya mtandao kwa sababu vidakuzi hutumwa na kila ombi la HTTP.
