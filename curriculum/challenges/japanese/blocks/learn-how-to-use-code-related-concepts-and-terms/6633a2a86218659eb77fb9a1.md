# --description--

助動詞 `can` は可能性や能力を示すために使われます。例えば、文 `You can see the stars at night` では、`can` が夜間に星を見ることが可能であることを示しています。

`Change` は「変える」または「変わる」という意味です。日常生活では、仕事の後に服を `change` することがあり、これは仕事着からカジュアルな服に着替えることを示します。

`Stay the same` は変わらず一定であることを意味します。例えば、`The rules of the game stay the same every year.` これはルールが年ごとに変わらないことを意味します。
