# --description--

在所有的工作坊和实验中，你都会使用 freeCodeCamp 的 Python 编辑器。但学习如何在本地计算机上设置 Python 也很重要。

在 Windows 和 Mac 上安装 Python 最简单的方法是从官方 Python 网站下载安装程序。本课后面我们还会讲如何在 Linux 上运行 Python。

转到 `https://www.python.org/` 并将鼠标悬停在“Downloads”上。会弹出一个模态窗口，显示适用于你操作系统（OS）的当前 Python 版本。

我们先讲如何在运行 macOS 的计算机上安装 Python：

- 点击显示当前 Python 版本的按钮（来自之前的模态窗口），你会自动开始下载一个 `.pkg` 安装文件。
- 下载完成后，打开 `.pkg` 安装程序，然后在弹出的窗口中点击“继续”。
- 持续点击“继续”按钮，直到看到“安装类型”部分。在那里，点击“安装”按钮。
- 如有必要，输入你的密码，然后开始安装。
- 之后，你应该会看到一条祝贺消息，说明 Python 已成功安装。
- 点击“关闭”按钮，安装完成！

你可以通过打开终端并运行 `python --version` 或 `python3 --version` 来验证安装。

你也可以通过在终端运行 `python` 或 `python3` 来打开 Python 解释器。Python 解释器是读取 Python 代码、将其翻译成计算机能理解的指令并执行这些指令的程序。

<dfn>终端</dfn>是一个基于文本的界面，允许你通过输入命令与计算机交互。每个操作系统都自带默认的终端应用程序。在 macOS 上，你可以使用 Terminal 应用。在 Windows 上，你可以使用命令提示符或 PowerShell。在 Linux 上，每个桌面环境都有自己的默认终端应用，比如 GNOME Terminal 或 Konsole。

请注意，在一些较旧的 macOS 和 Linux 系统上，`python` 可能保留给 Python 2，而 `python3` 专门用于 Python 3。如果你运行 `python --version` 并看到类似 `Python 2.7.18` 的 Python 2 版本，那么你的操作系统可能依赖于一些用旧版本 Python 编写的软件。如果是这样，你应该使用 `python3` 来运行你的 Python 代码。Python 2 已经停止维护，不应再用于任何新开发。

在 Windows 上安装 Python，请按照以下步骤操作：

- 转到 `https://www.python.org/`，将鼠标悬停在“Downloads”上。你应该会看到一个模态窗口，显示“Download for Windows”和带有当前 Python 版本的下载按钮。
- 点击版本号，你会自动开始下载一个 Windows 可执行文件（`.exe`）。
- 下载完成后，双击 Python 安装程序，按照提示操作。
- 当你看到 `Add python.exe to Path` 选项时，勾选该选项，然后点击 `Install Now`。这样以后操作会更方便。

你可以通过打开命令行外壳（如 PowerShell）并运行 `python --version` 来验证安装。你也可以通过运行 `python` 来打开 Python 解释器。

对于 Linux，大多数主流发行版如 Ubuntu、Debian 和 Fedora 都自带 Python。

只需打开终端并运行 `python --version` 或 `python3 --version`：

如果任一命令没有显示 Python 版本，你可以在 `https://www.python.org` 查找适合你 Linux 发行版的安装包，或者在线搜索推荐的安装方法。
