# --interactive--

リンクには `link`、`visited`、`hover`、`focus`、`active` といった異なる状態があります。これらの状態は、ユーザーがリンクを認識しやすくし、操作後に明確なフィードバックを提供するために重要であり、使いやすさとアクセシビリティの向上につながります。

これらの異なるリンク状態をスタイルすることは、リンクの現在の状態に関する視覚的な手がかりを提供するため、使いやすさとアクセシビリティの両方にとって非常に重要です。これにより、ユーザーはどのリンクを訪れたか、どのリンクとやり取りしているか、クリックすると何が起こるかを理解できます。

視覚障害や認知障害のあるユーザーにとって、これらの異なるスタイルはナビゲーションをより簡単で直感的にします。

さらに、明確なリンク状態はユーザーの操作に即時のフィードバックを提供し、混乱を減らしサイトのナビゲーション性を向上させることで、全体的なユーザー体験を高めます。

これらの状態は、CSSの `pseudo-classes` を使ってスタイルできます。

疑似クラスとは、セレクターに追加されるキーワードで、選択された要素の特別な状態を指定します。

例えば、`:hover` はユーザーのポインターがボタンにホバーしたときに色を変え、`:visited` はすでに訪れたリンクの色を変えます。

疑似クラスを使うと、HTMLに追加のマークアップを加えずに、要素の状態やユーザーの操作に基づいてスタイルを適用できます。

`pseudo-class` の構文は次のようになり、`A` がセレクター、`:B` が `pseudo-class` です：

```css
A:B {
  property: value;
}
```

異なるリンク状態のスタイル方法を理解するために、いくつかの例を見てみましょう。

`:link` 疑似クラスは未訪問のリンクをスタイルし、クリック可能であることを示します。

以下はアンカー要素をターゲットにして `:link` 疑似クラスを使う例です：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Normal state (unvisited link) */
a:link { 
  color: red;
}
```

:::

上記の例では、リンクのデフォルトの青色が未訪問時に `red` に変わります。

`:visited` はすでに訪問またはクリックされたリンクをスタイルし、ユーザーがどのリンクをクリックしたかを追跡しやすくします。以下は `:visited` 疑似クラスの使用例です：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="https://www.freecodecamp.org/learn/" target="_blank">freeCodeCamp</a>
```

```css
/* Visited link */
a:visited {
  color: green;
}
```

:::

このコードは、リンクがクリックされたときに `green` に色を変えます。

`:hover` はユーザーがリンクにホバーしたときにリンクのスタイルを変え、リンクがインタラクティブであることを視覚的に示します。以下は `:hover` 疑似クラスの使用例です：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Hover state */
a:hover {
  color: green;
}
```

:::

このコードは、マウスがホバーしたときにリンクの色を `green` に変えます。

`:focus` はリンクにフォーカスが当たったとき（キーボードでのナビゲーション時など）にリンクの周囲にスタイルを追加し、アクセシビリティを向上させます。

以下はリンクにフォーカスが当たったときに `outline` プロパティを使ってオレンジの実線アウトラインを適用する例です。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Focus state */
a:focus {
  outline: 2px solid orange;
}
```

:::

`:active` はリンクがクリックされている間にリンクのスタイルを変え、ユーザーの操作が認識されていることを即座にフィードバックします。以下は `:active` 疑似クラスの使用例です：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Active state */
a:active {
  color: pink;
}
```

:::

このコード例では、リンクがクリックされたときに即座に `pink` 色に変わります。
