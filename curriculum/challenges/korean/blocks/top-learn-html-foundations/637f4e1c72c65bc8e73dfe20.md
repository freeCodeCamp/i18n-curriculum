# --description--

HTML 보일러플레이트를 보여 주려면 먼저 작업할 HTML 파일이 필요합니다.

컴퓨터에 새 폴더를 만들고 이름을 `html-boilerplate`로 지정하세요. 그 폴더 안에 새 파일을 만들고 이름을 `index.html`로 지정하세요.

이미 doc, pdf, 이미지 파일 등 다양한 유형의 파일에 익숙할 것입니다.

컴퓨터에 HTML 파일을 만들고 싶다는 것을 알리려면, `.html` 파일을 만들 때처럼 파일 이름에 `index.html` 확장자를 붙여야 합니다.

HTML 파일 이름을 index로 지정한 점은 주목할 만합니다. 웹사이트의 홈페이지를 담을 HTML 파일은 항상 `index.html`로 이름을 지어야 합니다. 이는 웹 서버가 기본적으로 사용자가 웹사이트에 접속할 때 `index.html` 페이지를 찾기 때문이며, 없으면 큰 문제가 발생합니다.

## The DOCTYPE

모든 HTML 페이지는 doctype 선언으로 시작합니다. doctype의 목적은 브라우저에 문서를 렌더링할 때 어떤 HTML 버전을 사용할지 알려 주는 것입니다. 최신 HTML 버전은 HTML5이며, 그 버전의 doctype은 단순히 `<!DOCTYPE html>`입니다.

이전 HTML 버전의 doctype은 좀 더 복잡했습니다. 예를 들어, HTML4의 doctype 선언은 다음과 같습니다:

```html
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
```

하지만 아마도 이전 버전의 HTML을 사용할 일은 거의 없을 것이므로 항상 `<!DOCTYPE html>`을 사용할 것입니다.

앞서 만든 `index.html` 파일을 텍스트 편집기로 열고 첫 번째 줄에 `<!DOCTYPE html>`을 추가하세요.
