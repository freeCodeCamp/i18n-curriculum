# --description--

Радикал числа $n$, $rad(n)$, — це добуток різних простих множників числа $n$. Наприклад, $504 = 2^3 × 3^2 × 7$, отже $rad(504) = 2 × 3 × 7 = 42$.

Визначимо трійку натуральних чисел (a, b, c) як abc-хіт, якщо:

1. $GCD(a, b) = GCD(a, c) = GCD(b, c) = 1$
2. $a &lt; b$
3. $a + b = c$
4. $rad(abc) &lt; c$

Наприклад, (5, 27, 32) — це abc-хіт, тому що:

1. $GCD(5, 27) = GCD(5, 32) = GCD(27, 32) = 1$
2. $5 &lt; 27$
3. $5 + 27 = 32$
4. $rad(4320) = 30 &lt; 32$

Виявляється, що abc-хіти досить рідкісні, і існує лише тридцять один abc-хіт для $c &lt; 1000$, з $\sum{c} = 12523$.

Знайдіть $\sum{c}$ для $c &lt; 120000$.
