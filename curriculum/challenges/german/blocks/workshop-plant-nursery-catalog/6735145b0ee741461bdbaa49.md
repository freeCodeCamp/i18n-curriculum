# --description--

Alternativ können Sie Ihr Map-Objekt durchlaufen, indem Sie die `entries`-Methode verwenden, die einen Map-Iterator mit Schlüssel-Wert-Paaren zurückgibt.

Ändern Sie Ihre `displayCatalog`-Funktion so, dass sie die Einträge in Ihrer `catalog`-Map durchläuft und jeden Eintrag in der Konsole ausgibt.

# --hints--

Ihre `displayCatalog`-Funktion sollte die `entries`-Methode auf `catalog` aufrufen.

```js
const catalogCopy = new Map(catalog);
let flag = false;
const temp = catalog.entries;
try {
    catalog.entries = () => {
        flag = true;
        return catalogCopy.entries()
    };
    displayCatalog();    
    assert.isTrue(flag);
} finally {
    catalog.entries = temp;
}
```

Ihre `displayCatalog`-Funktion sollte jeden Wert, der in `catalog` enthalten ist, in der Konsole ausgeben.

```js
const testArr = [];
const temp = console.log;
try {
    console.log = obj => {testArr.push(obj)};
    displayCatalog();    
    assert.isNotEmpty(testArr);
    const ents = catalog.entries();
    for (let i = 0; i < catalog.size; i++) {
```
