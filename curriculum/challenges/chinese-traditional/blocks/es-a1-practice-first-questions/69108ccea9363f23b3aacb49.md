# --explanation--

這段 Mateo 和 Camila 之間的對話窗聚焦於職業。

`¿A qué te dedicas?` 意思是「你的職業是什麼？」或「你是做什麼工作的？」

**`Soy` + [profession]** 意思是「我是一位 [profession]」。

在西班牙語中，大多數職業會根據性別改變。然而，某些以 `-ista` 或 `-nte` 結尾的職業通常對兩種性別保持完全相同：

| Profession | Masculine | Feminine |
| :---: | :---: | :---: |
| Analyst | `analista` | `analista` |
| Journalist | `periodista` | `periodista` |
| Receptionist | `recepcionista` | `recepcionista` |
| Singer | `cantante` | `cantante` |
