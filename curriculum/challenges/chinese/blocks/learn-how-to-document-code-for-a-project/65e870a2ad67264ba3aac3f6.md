# --description--

在这个任务中，你将学习 `simple` 和 `simply` 之间的区别，并理解 `double` 和 `slashes` 的含义。

`Simple` 是一个形容词，意思是简单或不复杂，比如 `a simple task`。`Simply` 是一个副词，意思是以简单的方式，比如 `doing something simply`。

`Double` 意味着两个某物。`slashes` 是写作中使用的 `/` 符号。你为使 `slash` 变为复数形式而添加 `es`，因为以 `sh` 结尾的单词通过添加 `es` 来构成复数形式，就像 `dishes` 或 `brushes` 一样。
