# --interactive--

JavaScriptでは、テキストを扱うことがコーディングの重要な部分であり、しばしばテキストの断片を組み合わせたり結合したりする必要があります。この処理を文字列の連結と呼びます。

このレッスンでは、文字列の連結がどのように機能するかにフォーカスし、特に`+`演算子、`+=`演算子、そして`concat()`メソッドの使い方を学びます。

`+`演算子は、文字列を連結する最もシンプルで頻繁に使われる方法の一つです。複数の文字列を結合したり、テキストを保持する変数と文字列を組み合わせたりできます。

例を見てみましょう。

:::interactive_editor

```js
let firstName = "John";
let lastName = "Doe";

let fullName = firstName + " " + lastName; 
console.log(fullName); // John Doe
```

:::

この例では、`+`演算子を使って`firstName`と`lastName`の変数を空白（`" "`）と一緒に連結し、フルネームを作成しました。

`+`演算子を使って文字列を連結する際の欠点の一つは、連結する文字列間のスペースを注意深く管理しないと、スペースの問題が起こる可能性があることです。

スペースが抜けている例を見てみましょう。

:::interactive_editor

```js
let firstName = "John";
let lastName = "Doe";

let fullName = firstName + lastName; 
console.log(fullName); // "JohnDoe"
```

:::

文字列を連結するために`+`演算子を使う場合は、スペースの問題がないか必ず確認してください。

既存の文字列に追加や追記をしたい場合は、`+=`演算子を使うことができます。これは、時間をかけて文字列にさらにテキストを追加していきたいときに便利です。

`+=`演算子を使って一つの文字列に別の文字列を追記する例を見てみましょう。

:::interactive_editor

```js
let greeting = 'Hello';
greeting += ', John!';

console.log(greeting); // "Hello, John!"
```

:::

文字列は不変であることを覚えておくことが重要です。つまり、一度作成された文字列は変更できません。

この場合、元の`Hello`の文字列は変更されません。代わりに、greetingは新しい`Hello, John!`の文字列を参照しています。

文字列を連結するもう一つの方法は、`concat()`メソッドを使うことです。

`concat()`メソッドについて学ぶ前に、まずメソッドと関数がどのようなものかを大まかに理解することが重要です。

プログラミングでは、関数は特定のタスクを実行し、さまざまな入力で呼び出せる再利用可能なコードのブロックです。一方、メソッドはオブジェクトに関連付けられた関数の一種で、そのオブジェクト内のデータに対して操作を行います。

今後のレッスンで、JavaScriptの関数、オブジェクト、メソッドがどのように動作するかをさらに詳しく学びます。しかし今は、JavaScriptには`concat()`メソッドのように使えるメソッドが数多くあることを理解しておくことが大切です。

`concat()`メソッドを使って二つの文字列を結合する例を見てみましょう。

:::interactive_editor

```js
let str1 = 'Hello';
let str2 = 'World';

let result = str1.concat(' ', str2); 
console.log(result); // Hello World
```

:::

この例では、`concat()`メソッドを使って`str1`、空白（`' '`）、`str2`を一つの文字列に結合しています。

まとめると、`+`演算子は、特にいくつかの文字列や変数を組み合わせる単純な連結に最適です。

`+=`演算子は、文字列を段階的に構築したり、既存の文字列変数に新しい内容を追記したりするときに便利です。

最後に、`concat()`メソッドは複数の文字列をまとめて連結したい場合に役立ちます。
