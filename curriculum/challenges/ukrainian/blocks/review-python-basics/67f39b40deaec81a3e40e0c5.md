## Що таке Python?

- **Вступ**: Python — це мова програмування загального призначення, відома своєю простотою та легкістю у використанні. Python застосовують у багатьох сферах, таких як наука про дані та машинне навчання, веброзробка, скрипти та автоматизація, вбудовані системи та Інтернет речей, і багато іншого.
- **Загальні випадки використання**: Python використовується у науці про дані, машинному навчанні, веброзробці, кібербезпеці, автоматизації та на мікрокомп’ютерах, таких як Raspberry Pi та плати, сумісні з MicroPython.

## Python у вашому локальному середовищі

- **Встановлення**: Найкращий спосіб встановити Python на Windows, Mac і Linux — скачати інсталятор з офіційного сайту Python (`https://www.python.org/`).

## Змінні

- **Оголошення змінних**: Щоб оголосити змінну, почніть із імені змінної, потім використайте оператор присвоєння (`=`) і вкажіть значення. Це може бути число, рядок, булевий тип тощо. Ось кілька прикладів:

```py
name = 'John Doe'
age = 25
```

- **Конвенції іменування змінних**: Ось конвенції іменування, які слід використовувати для змінних:

  - Імена змінних можуть починатися лише з літери або підкреслення (_), а не з числа.
  - Імена змінних можуть містити лише алфавітно-цифрові символи (a-z, A-Z, 0-9) та підкреслення (_).
  - Імена змінних чутливі до регістру — `age`, `Age` і `AGE` вважаються різними.
  - Імена змінних не можуть бути зарезервованими ключовими словами Python, такими як `if`, `class` або `def`.
  - Імена змінних із кількома словами розділяються підкресленнями. Наприклад, `snake_case`.


## Коментарі

- **Однорядкові коментарі**: Ці коментарі використовують для коротких нотаток, які ви хочете залишити у своєму коді.

```py
# This is a single line comment
```

- **Багаторядкові рядки**: Ці рядки можна використовувати для більших нотаток або щоб перетворити частини коду на коментарі.

```py
"""
This is a multi-line string.
Here is some code commented out.

name = 'John Doe'
age = 25
"""
```

- **Функція `print()`**: Щоб вивести дані у консоль, можна використати функцію `print()` ось так:

```py
print('Hello world!') # Hello world!
```

## Загальні типи даних у Python

- **Вступ**: Python — це мова з динамічною типізацією, як і JavaScript, тобто вам не потрібно явно оголошувати типи змінних. Мова визначає тип змінної на основі того, що ви їй присвоюєте.
- **Ціле число**: Ціле число без десяткових знаків:

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **Число з плаваючою комою**: Число з десятковими знаками:

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **Рядок**: Послідовність символів, укладена в лапки:

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **Булевий тип**: Значення, що представляє або `True`, або `False`:

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **Множина**: Неупорядкована колекція унікальних елементів:

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **Словник**: Колекція пар ключ-значення, укладена у фігурні дужки:

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **Кортеж**: Незмінна впорядкована колекція, укладена в круглі дужки:

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **Діапазон**: Послідовність чисел, часто використовується у циклах:

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **Список**: Впорядкована колекція елементів, що підтримує різні типи даних:

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**: Спеціальне значення, що означає відсутність значення:

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## Незмінні та змінні типи

- **Незмінні типи**: Ці типи не можна змінити після оголошення, хоча ви можете призначити їхні змінні на щось нове, що називається переназначенням. До них належать ціле число, число з плаваючою комою, комплексне число, булевий, рядок, кортеж, діапазон і `None`.
- **Змінні типи**: Ці типи можна змінювати після оголошення. Ви можете додавати, видаляти або оновлювати їхні елементи. До них належать колекційні типи, такі як список, множина та словник.
- **Функція `type()`**: Щоб дізнатися тип змінної, можна використати функцію `type()` ось так:

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **Функція `isinstance()`**: Використовується для перевірки, чи відповідає змінна певному типу даних:

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## Робота з рядками

- **Визначення**: Як ви пам’ятаєте з JavaScript, рядки є незмінними, тобто їх не можна змінити після створення. У Python можна використовувати як одинарні, так і подвійні лапки. Рекомендується обрати один стиль і дотримуватися його:

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **Доступ до символів рядка**: Ви можете отримати доступ до символів рядка, використовуючи дужкову нотацію ось так:

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **Екранізація рядків**: Якщо рядок містить лапки, можна використати зворотну скісну риску (`\`) ось так:

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **Конкатенація рядків**: Щоб об’єднати рядки, можна використати оператор `+` ось так:

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

Інший спосіб об’єднати рядки — використати оператор `+=`. Він виконує конкатенацію та присвоєння в одному кроці ось так:

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**: Це скорочення для форматованих рядкових літералів. Вони дозволяють виконувати інтерполяцію та деяку конкатенацію з компактним і читабельним синтаксисом:

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **Вирізання рядків (slicing)**: Це коли ви можете витягти частини рядка. Ось базовий синтаксис:

```py
str[start:stop:step]
```

Початкова позиція — це індекс, з якого починається вирізання. Кінцева позиція — де вирізання закінчується, ця позиція не включається. Крок — це інтервал, з яким відбувається вирізання. Ось кілька прикладів:

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **Отримання довжини рядка**: Функція `len()` повертає кількість символів у рядку:

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## Робота з оператором `in`

- **Оператор `in`**: Повертає булеве значення, яке вказує, чи існує символ або символи у рядку:

```py
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

## Загальні методи рядків

- **`str.upper()`**: Повертає новий рядок, у якому всі символи перетворені на верхній регістр:

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`str.lower()`**: Повертає новий рядок, у якому всі символи перетворені на нижній регістр:

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`str.strip()`**: Повертає копію рядка з видаленими вказаними провідними та кінцевими символами (якщо аргумент не передано, видаляє пробіли на початку та в кінці).

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**: Повертає новий рядок, у якому всі входження старого рядка замінені на новий.

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**: Використовується для розбиття рядка на список за вказаним роздільником. Роздільник — це рядок, який вказує, де має відбутися розбиття.

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**: Використовується для об’єднання елементів ітерованого об’єкта в рядок із роздільником. Ітерований об’єкт — це колекція елементів, по яких можна пройтися циклом, наприклад список, рядок або кортеж.

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`str.startswith(prefix)`**: Повертає булеве значення, яке вказує, чи починається рядок із вказаного префікса:

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`str.endswith(suffix)`**: Повертає булеве значення, яке вказує, чи закінчується рядок вказаним суфіксом:

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`str.find()`**: Повертає індекс першого входження підрядка. Якщо підрядок не знайдено, повертає `-1`:

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`str.count(substring)`**: Рахує, скільки разів підрядок зустрічається у рядку:

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`str.capitalize()`**: Повертає новий рядок, у якому перший символ великий, а всі інші — у нижньому регістрі:

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`str.isupper()`**: Повертає `True`, якщо всі літери у рядку у верхньому регістрі, і `False`, якщо ні:

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`str.islower()`**: Повертає `True`, якщо всі літери у рядку у нижньому регістрі, і `False`, якщо ні:

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`str.title()`**: Повертає новий рядок, у якому перша літера кожного слова велика:

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`str.maketrans()`**: Цей метод створює таблицю відповідностей символів для трансляції. Часто використовується разом із методом `translate()`, який застосовує цю таблицю до рядка і повертає перекладений результат.

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## Загальні операції з цілими числами та числами з плаваючою комою

- **Основні математичні операції**: У Python можна виконувати базові математичні операції з цілими числами та числами з плаваючою комою, включно з додаванням, відніманням, множенням і діленням:

```py
int_1 = 56
int_2 = 12
float_1 = 5.4
float_2 = 12.0

# Addition

print('Integer Addition:', int_1 + int_2) # Integer Addition: 68
print('Float Addition:', float_1 + float_2) # Float Addition: 17.4

# Subtraction

print('Int Subtraction:', int_1 - int_2) # Int Subtraction: 44
print('Float Subtraction:',  float_2 - float_1) # Float Subtraction: 6.6

# Multiplication

print('Int Multiplication:', int_1 * int_2) # Int Multiplication: 672
print('Float Multiplication:', float_2 * float_1) # Float Multiplication: 64.80000000000001

# Division

print('Int Division:', int_1 / int_2) # Int Division: 4.666666666666667
print('Float Division:', float_2 / float_1) # Float Division: 2.222222222222222
```

Якщо додати число з плаваючою комою і ціле число, результат буде перетворений на число з плаваючою комою ось так:

```py
int_1 = 56
float_1 = 5.4

print(int_1 + float_1) # 61.4
```

- **Оператор за модулем (`%`)**: Повертає остачу від ділення одного числа на інше:

```py
int_1 = 56
int_2 = 12

print(int_1 % int_2) # 8
```

- **Цілочисельне ділення (`//`)**: Оператор, який ділить два числа і округлює результат вниз до найближчого цілого числа:

```py
int_1 = 56
int_2 = 12

print(int_1 // int_2) # 4
```

- **Оператор піднесення до степеня (`**`)**: Використовується для піднесення числа до степеня іншого числа:

```py
int_1 = 4
int_2 = 2

print(int_1 ** int_2) # 16
```

- **Функція `float()`**: Використовується для перетворення цілого числа на число з плаваючою комою.

```py
num = 4

print(float(num)) # 4.0
```

- **Функція `int()`**: Використовується для перетворення числа з плаваючою комою на ціле число.

```py
num = 4.0

print(int(num)) # 4
```

- **Функція `round()`**: Використовується для округлення числа до найближчого цілого числа:

```py
num_1 = 3.4
num_2 = 7.7

print(round(num_1)) # 3
print(round(num_2)) # 8
```

- **Функція `abs()`**: Використовується для отримання абсолютного значення числа:

```py
num = -13

print(abs(num)) # 13
```

- **Функція `bin()`**: Використовується для перетворення цілого числа у його двійкове представлення у вигляді рядка:

```py
num = 56

print(bin(num))  # 0b111000
```

- **Функція `oct()`**: Використовується для перетворення цілого числа у його вісімкове представлення у вигляді рядка:

```py
num = 56

print(oct(num))  # 0o70
```

- **Функція `hex()`**: Використовується для перетворення цілого числа у його шістнадцяткове представлення у вигляді рядка:

```py
num = 56

print(hex(num))  # 0x38
```

- **Функція `pow()`**: Використовується для піднесення числа до степеня іншого числа:

```py
result = pow(2, 3) 
print(result)  # 8
```

## Розширені присвоєння

- **Визначення**: Розширене присвоєння поєднує бінарну операцію з присвоєнням в одному кроці. Воно бере змінну, застосовує до неї операцію з іншим значенням і зберігає результат у ту ж змінну.

```py
# Addition assignment 
my_var = 10
my_var += 5

print(my_var) # 15

# Subtraction assignment
count = 14
count -= 3

print(count) # 11

# Multiplication assignment 
product = 65
product *= 7

print(product) # 455

# Division assignment 
price = 100
price /= 4

print(price) # 25.0

# Floor Division assignment 
total_pages = 23
total_pages //= 5

print(total_pages) # 4

# Modulo assignment 
bits = 35
bits %= 2

print(bits) # 1

# Exponentiation assignment 
power = 2
power **= 3

print(power) # 8
```

Існують й інші оператори розширеного присвоєння, наприклад для побітових операторів. До них належать `&=`, `^=`, `>>=` і `<<=`.

## Робота з функціями

- **Визначення**: Функції — це повторно використовувані частини коду, які приймають вхідні дані (аргументи) і повертають вихідні дані. Щоб викликати функцію, потрібно звернутися до її імені, після якого йдуть круглі дужки:

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call
print(result) # 7
```

Якщо функція явно не повертає значення, то за замовчуванням повертається `None`:

```py
def greet():
    print('hello') 

result = greet() # hello
print(result) # None
```

Ви також можете задавати значення за замовчуванням для параметрів ось так:

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

Якщо викликати функцію з неправильним числом аргументів, отримаєте `TypeError`:

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

## Загальні вбудовані функції

- **Функція `input()`**: Використовується для запиту вхідних даних від користувача:

```py
name = input('What is your name?') # User types 'Kolade' and presses Enter  
print('Hello', name) # Hello Kolade
```

- **Функція `int()`**: Використовується для перетворення числа, булевого значення або числового рядка на ціле число:

```py
print(int(3.14)) # 3
print(int('42')) # 42
print(int(True)) # 1
print(int(False)) # 0 
```

## Область видимості у Python

- **Локальна область видимості**: Змінна, оголошена всередині функції або класу, доступна лише в межах цієї функції або класу.

```py
def my_func():
    num = 10
    print(num)
```

- **Огортаюча область видимості**: Функція, вкладена в іншу функцію, може отримати доступ до змінних зовнішньої функції.

```py
def outer_func():
    msg = 'Hello there!'
    
    def inner_func():
        print(msg)
    inner_func()

print(outer_func()) # Hello there!
```

- **Глобальна область видимості**: Змінні, оголошені поза будь-якими функціями чи класами, доступні з будь-якої частини програми.

```py
tax = 0.70 

def get_total(subtotal):
    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **Вбудована область видимості**: Зарезервовані імена в Python для вбудованих функцій, модулів, ключових слів і об’єктів.

```py
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```

## Оператори порівняння

- **Рівність (`==`)**: Перевіряє, чи два значення рівні:

```py
print(3 == 4) # False
```

- **Нерівність (`!=`)**: Перевіряє, чи два значення не рівні:

```py
print(3 != 4) # True
```

- **Строго більше (`>`)**: Перевіряє, чи одне значення більше за інше:

```py
print(3 > 4) # False
```

- **Строго менше (`<`)**: Перевіряє, чи одне значення менше за інше:

```py
print(3 < 4) # True
```

- **Більше або дорівнює (`>=`)**: Перевіряє, чи одне значення більше або дорівнює іншому:

```py
print(3 >= 4) # False
```

- **Менше або дорівнює (`<=`)**: Перевіряє, чи одне значення менше або дорівнює іншому:

```py
print(3 <= 4) # True
```

## Робота з інструкціями `if`, `elif` та `else`

- **Інструкції `if`**: Це умови, які використовують, щоб визначити, чи щось є правдивим. Якщо умова оцінюється як `True`, виконується відповідний блок коду.

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **Інструкція `elif`**: Це умови, які йдуть після інструкції `if`. Блок `elif` виконується лише якщо всі попередні умови оцінюються як `False`, а власна умова — як `True`.

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **Умова `else`**: Виконується, якщо жодна інша умова не оцінюється як `True`.

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

Ви також можете використовувати вкладені інструкції `if` ось так:

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## Правдиві та хибні значення

- **Визначення**: У Python кожне значення має вбудоване булеве значення, тобто в логічному контексті його слід трактувати як `True` або `False`. Багато значень вважаються правдивими, тобто вони оцінюються як `True` у логічному контексті. Інші — хибними, тобто оцінюються як `False`. Ось кілька прикладів хибних значень:

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

Інші значення, як-от ненульові числа та непорожні рядки, є правдивими.

## Робота з функцією `bool()`

- **Визначення**: Якщо ви хочете перевірити, чи є значення правдивим або хибним, можна використати вбудовану функцію `bool()`. Вона явно перетворює значення на його булевий еквівалент і повертає `True` для правдивих значень і `False` для хибних. Ось кілька прикладів:

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## Булеві оператори та коротке замикання

- **Визначення**: Це спеціальні оператори, які дозволяють поєднувати кілька виразів для створення складнішої логіки прийняття рішень у коді. У Python є три булеві оператори: `and`, `or` і `not`.
- **Оператор `and`**: Цей оператор приймає два операнди і повертає перший операнд, якщо він хибний, інакше повертає другий. Для того, щоб вираз був правдивим, обидва операнди мають бути правдивими.

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

Ви також можете використовувати оператор `and` в умовах ось так:

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **Оператор `or`**: Цей оператор повертає перший операнд, якщо він правдивий, інакше повертає другий. Вираз `or` буде правдивим, якщо хоча б один операнд правдивий. Ось приклад:

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

Як і з оператором `and`, ви можете використовувати оператор `or` в умовах ось так:

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **Коротке замикання**: Оператори `and` і `or` називають операторами короткого замикання. Це означає, що Python перевіряє значення зліва направо і зупиняється, щойно визначає кінцевий результат.
- **Оператор `not`**: Цей оператор приймає один операнд і інвертує його булеве значення. Він перетворює правдиві значення на `False` і хибні на `True`. На відміну від попередніх операторів, `not` завжди повертає `True` або `False`. Ось кілька прикладів:

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

Ось приклад використання оператора `not` в умові:

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```
