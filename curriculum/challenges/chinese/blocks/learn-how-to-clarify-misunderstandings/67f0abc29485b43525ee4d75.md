# --explanation--

`Not that I know of` 用来表示，据你所知，某事不是真的或尚未发生。它通常用来礼貌地说 `no`，同时保留说话者可能错了的余地。例如：

- 人物 1：`Has the update been installed?`

- 人员 2：`Not that I know of.` - 这意味着你不相信更新已安装，但你并不完全确定。
