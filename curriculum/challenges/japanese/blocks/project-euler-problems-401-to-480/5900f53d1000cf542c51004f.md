# --description--

メビウス関数は $μ(n)$ と表され、次のように定義されます。

- $n$ が平方因子なしの場合、$μ(n) = (-1)^{ω(n)}$ （ここで $ω(n)$ は $n$ の異なる素因数の数です）
- $n$ が平方因子なしでない場合、$μ(n) = 0$ です。

区間 $[a, b]$ にある整数 $n$ のうち、$μ(n) = 1$ となるものの数を $P(a, b)$ とします。

区間 $[a, b]$ にある整数 $n$ のうち、$μ(n) = -1$ となるものの数を $N(a, b)$ とします。

例えば、$P(2, 10) = 2$ で $N(2, 10) = 4$ です。

次に、整数の組 $(a, b)$ の数 $C(n)$ を次の条件で定義します。

- $1 ≤ a ≤ b ≤ n$、
- $99 \times N(a, b) ≤ 100 \times P(a, b)$、かつ
- $99 \times P(a, b) ≤ 100 \times N(a, b)$。

例えば、$C(10) = 13$、$C(500) = 16\,676$、$C(10\,000) = 20\,155\,319$ です。

$C(20\,000\,000)$ を求めてください。
