# --description--

정수 변 길이를 가진 삼각형 $ABC$가 주어지고 $BC ≤ AC ≤ AB$입니다. $k$는 각 $ACB$의 각 이등분선입니다. $m$은 $ABC$의 외접원에서 점 $C$에서의 접선입니다. $n$은 $B$를 지나는 $m$과 평행한 직선입니다.

$n$과 $k$의 교점을 $E$라고 합니다.
