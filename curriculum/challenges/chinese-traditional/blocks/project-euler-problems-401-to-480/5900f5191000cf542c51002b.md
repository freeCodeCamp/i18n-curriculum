# --description--

設 $a$、$b$ 和 $c$ 為正數。

設 $W$、$X$、$Y$、$Z$ 為四個共線點，其中 $|WX| = a$、$|XY| = b$、$|YZ| = c$ 且 $|WZ| = a + b + c$。

令 $C_{\text{in}}$ 為以直徑 $XY$ 為圓徑的圓。

令 $C_{\text{out}}$ 為以直徑 $WZ$ 為圓徑的圓。

三元組 ($a$、$b$、$c$) 稱為*項鍊三元組*，如果你可以放置 $k ≥ 3$ 個不同的圓 $C_1$、$C_2$、\ldots、$C_k$，使得：

- $C_i$ 與任何 $C_j$（其中 $1 ≤ i$、$j ≤ k$ 且 $i ≠ j$）沒有共同的內部點，
- $C_i$ 與 $C_{\text{in}}$ 及 $C_{\text{out}}$ 皆相切，且 $1 ≤ i ≤ k$，
- $C_i$ 與 $C_{i + 1}$ 相切，對於 $1 ≤ i &lt; k$，且
- $C_k$ 與 $C_1$ 相切。

例如，(5, 5, 5) 和 (4, 3, 21) 是項鍊三元組，而可以證明 (2, 2, 5) 不是。

<img alt="a visual representation of a necklace triplet" src="https://cdn.freecodecamp.org/curriculum/project-euler/necklace-of-circles.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

設 $T(n)$ 為項鍊三元組 $(a, b, c)$ 的數量，其中 $a$、$b$ 和 $c$ 為正整數，且 $b ≤ n$。例如，$T(1) = 9$、$T(20) = 732$ 和 $T(3\,000) = 438\,106$。

尋找 $T(1\,000\,000\,000)$。
