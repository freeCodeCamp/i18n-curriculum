# --description--

Tom은 나중에 할 계획에 대해 이야기하기 위해 `I'll` (`I will`의 줄임말)을 사용합니다.

`Will`는 누군가가 미래에 무언가를 하기로 결정했을 때 자주 사용됩니다. 예를 들어, `I'll call you later`는 그 사람이 지금 이후에 전화를 할 계획임을 의미합니다.
