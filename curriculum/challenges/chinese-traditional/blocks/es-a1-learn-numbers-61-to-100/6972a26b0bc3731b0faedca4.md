# --description--

現在是時候學習 70 到 79 的數字了。

在西班牙語中，數字 70 稱為 `setenta`。

這些數字是由使用十位數字的字 `setenta`（70），接著是連接詞 `y`（"and"），然後是個位數字（1-9）所組成。

| Number | Spanish |
| :--- | :--- |
| 70 | `Setenta` |
| 71 | `Setenta y uno` |
| 72 | `Setenta y dos` |
| 73 | `Setenta y tres` |
| 74 | `Setenta y cuatro` |
| 75 | `Setenta y cinco` |
| 76 | `Setenta y seis` |
| 77 | `Setenta y siete` |
| 78 | `Setenta y ocho` |
| 79 | `Setenta y nueve` |

數字 `setenta y uno` (71) 在這個範圍中是獨特的，因為它在陽性名詞前會變成 `setenta y un`。例如：

`Setenta y un años` - 七十一年。
