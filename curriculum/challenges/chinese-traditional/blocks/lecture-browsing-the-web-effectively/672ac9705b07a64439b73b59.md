# --description--

在網際網路上搜尋資訊可能是一項艱鉅的任務。網路上有如此多的網站，擁有大量的資料，讓你似乎無法獲得想要的結果。

但多虧了像是布林值搜尋語法這類的東西，你可以將搜尋範圍縮小到正是你需要的內容。

布林值搜尋語法是一種使用特殊字來幫助你在線上搜尋時精確找到你想要的東西的方法。

這是一個針對 `freecodecamp curriculum` 的基本搜尋查詢範例。

當你搜尋 `freecodecamp curriculum` 時，搜尋引擎會傳回符合 *任一* 這些關鍵字的結果－通常依相關性排序（它們與你的查詢相符或相關的可能性有多高）。

但是如果你只想要同時符合兩個關鍵字的結果呢？你可能聽說過可以用引號將查詢包起來，例如 `"freecodecamp curriculum"`。

然而，這個查詢特別匹配 *phrase* `freecodecamp curriculum`。這表示你看到的結果將限於同時包含 `freecodecamp curriculum` 的網站，而非同時包含 `freecodecamp` 和 `curriculum` 的網站。

如果你想要符合同時包含這兩個詞彙的網站，但不一定要作為一個片語相鄰，你可以為每個詞彙加上加號符號前置式，如此：`+freecodecamp +curriculum`

或者，你可能對 freeCodeCamp 課程外的計畫感興趣。在這種情況下，你可以在詞彙 `curriculum` 前面加上減號符號，如此：`+freecodecamp -curriculum`。

這將傳回提及 `freecodecamp`，但排除同時提及 `curriculum` 的結果。

或者你可能有特定的網站想要搜尋，例如我們的新聞平台？你可以使用 `site:` 後接你想搜尋的網站 URL ：`site:freecodecamp.org/news curriculum`。

此查詢會傳回所有來自 `freecodecamp.org/news` 且包含字 `curriculum` 的結果。

透過結合這些工具，你可以提出非常具體的查詢，以取得你所需的精確資訊。

例如，查詢 `site:freecodecamp.org/news +python -curriculum` 將會在我們的新聞網站上搜尋 Python 文章，但排除可能談論我們課程的文章。

搜尋運算子可能會有所不同，因此務必檢查你偏好的搜尋引擎的文件。

但你現在應該能夠從你的網路搜尋中獲得你想要的精確結果。
