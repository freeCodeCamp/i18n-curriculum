# --description--

Na lição anterior, você aprendeu como instalar Python no seu dispositivo local. Nesta lição, você vai aprender como executar código Python localmente.

Antes de começar a criar seus scripts Python localmente, será necessário instalar um editor de código ou IDE. IDE significa ambiente de desenvolvimento integrado e possui recursos como editor de código, ferramentas de teste, terminal e mais.

Um editor de código popular que muitos desenvolvedores usam é o VS Code. Visite `https://code.visualstudio.com/download` para baixar o VS Code, que suporta ambientes Mac, Windows e Linux. Este é o editor que será usado nesta lição.

Outras opções comuns para desenvolvimento em Python incluem:

- PyCharm
- Spyder

Depois de baixar seu editor, você precisará abri-lo. Quando estiver aberto, será necessário abrir uma pasta. Você pode clicar na opção de menu `File` no canto superior esquerdo e clicar em `Open Folder`. A partir daí, você pode escolher ou criar uma pasta para seu projeto. Para prática extra fora da lição, talvez queira uma pasta chamada `python-projects`.

Depois de configurar sua pasta, você pode abrir o explorador, que é a barra lateral esquerda no VS Code. Clique no ícone `New File`, que parece uma folha de papel com um ícone de mais sobre ela. Então crie um arquivo chamado `main.py` e pressione enter. A extensão `.py` indica que este é um arquivo Python.

Abra seu arquivo `main.py` e digite o seguinte código:

```py
print("Hello, world!")
```

Para executar este código, você tem algumas opções. A primeira é clicar no botão de executar, que parece um botão de play localizado no canto superior direito. Isso abrirá um terminal e executará seu script Python lá. Você deve ver o texto `"Hello, world!"`.

Outra opção é abrir um terminal e executar o programa manualmente usando:

```bash
python main.py
```

Você deve executar este comando na pasta onde `main.py` está salvo. Por exemplo, se `main.py` estiver dentro de uma pasta chamada `python-projects`, primeiro use o seguinte no terminal:

```bash
cd python-projects
```

Depois execute:

```bash
python main.py
```

Em alguns ambientes, especialmente em sistemas macOS e Linux onde Python 2 e Python 3 estão instalados, pode ser necessário usar `python3` em vez de `python`. Isso é comum quando o comando `python` não existe ou aponta para uma versão mais antiga do Python.

```bash
python3 main.py
```

À medida que você avança no curso, deve tentar os exemplos das lições localmente e ver os resultados. Também deve tentar criar seus próprios exemplos para testar o que aprendeu nas lições.
