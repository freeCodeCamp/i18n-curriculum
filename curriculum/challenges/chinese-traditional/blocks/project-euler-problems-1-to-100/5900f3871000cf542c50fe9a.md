# --description--

歐拉發現了這個卓越的二次公式：

<div style='margin-left: 4em;'>$n^2 + n + 41$</div>

結果顯示該公式會為連續整數值 $0 \\le n \\le 39$ 產生 40 個質數。然而，當 $n = 40$ 時，$40^2 + 40 + 41 = 40(40 + 1) + 41$ 可被 41 整除，且當 $n = 41$ 時，$41^2 + 41 + 41$ 顯然也可被 41 整除。

令人難以置信的公式 $n^2 - 79n + 1601$ 被發現，該公式在連續的值 $0 \\le n \\le 79$ 產生 80 個質數。係數 −79 和 1601 的乘積是 −126479。

考慮以下形式的二次方程式：

<div style='margin-left: 4em;'>
  $n^2 + an + b$, where $|a| < range$ and $|b| \le range$<br>
  where $|n|$ is the modulus/absolute value of $n$<br>
  e.g. $|11| = 11$ and $|-4| = 4$<br>
</div>

找出二次表達式中係數 $a$ 和 $b$ 的乘積，此表達式能從 $n = 0$ 開始，產生最多連續質數。
