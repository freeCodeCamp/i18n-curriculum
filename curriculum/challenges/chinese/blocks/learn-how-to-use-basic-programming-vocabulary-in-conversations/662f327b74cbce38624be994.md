# --description--

短语 `no worries` 是一种随意的表达，用来向某人保证一切都好，他们不必担心。它可以用来回应感谢，驳回道歉，或向某人保证问题不严重。

例如，当你的朋友不小心洒了饮料并开始道歉时，你可以说 `no worries! It's just water, and it'll dry up quickly` 来让朋友感觉好些。
