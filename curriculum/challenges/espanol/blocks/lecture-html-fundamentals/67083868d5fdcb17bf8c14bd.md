# --interactive--

Una entidad HTML, o referencia de carácter, es un conjunto de caracteres utilizados para representar un carácter reservado en HTML. 

Supongamos que quieres mostrar el texto `This is an <img/> element` en la pantalla. Si usas el código que está actualmente en el editor, no mostrará el resultado deseado. Incluso si añadieras los atributos `src` y `alt` al ejemplo, se mostraría una imagen en medio del párrafo. No es el resultado deseado. Para interactuar con el ejemplo, necesitarás habilitar el editor interactivo.


:::interactive_editor

```html
<p>This is an <img /> element</p>
```

:::

Cuando el analizador de HTML ve el símbolo menor que (`<`) seguido de un nombre de etiqueta HTML, lo interpreta como un elemento HTML. Por eso no estás obteniendo el resultado deseado de `This is an <img/> element` en la pantalla.

Para solucionar este problema, puedes usar entidades HTML. Aquí tienes un ejemplo actualizado que usa las entidades HTML correctas para los símbolos menor que (`<`) y mayor que (`>`). Ahora deberías ver `This is an <img/> element` en la pantalla.

Activa el editor interactivo e intenta agregar un `&lt;p&gt;learning is fun&lt;/p&gt;` debajo del elemento párrafo. Deberías ver `<p>learning is fun</p>` en la pantalla.

:::interactive_editor

```html
<p>This is an &lt;img /&gt; element</p>

```

:::

Estos tipos de referencias de carácter se conocen como referencias de carácter nombradas.  Named references start with an ampersand sign (`&`) and end with a semicolon (`;`). Al usar una referencia de carácter nombrada, el analizador HTML no confundirá esto con un elemento HTML real. 

Otro tipo de referencia de carácter sería la referencia numérica decimal. Esta referencia de carácter comienza con un signo de ampersand y símbolo de almohadilla (`#`), seguido por uno o más dígitos decimales, terminando con un punto y coma.

Aquí tienes un ejemplo de cómo usar la referencia numérica decimal para el símbolo de menor que. 

Activa el editor interactivo y cambia el código para ver diferentes símbolos. Puedes usar `&#169;` para el símbolo de copyright y `&#174;` para el símbolo de marca registrada.

:::interactive_editor

```html
&#60;
```

:::

El último tipo de referencia de carácter sería la referencia numérica hexadecimal. Esta referencia de carácter comienza con un signo de ampersand, símbolo de almohadilla y la letra `x`. Luego es seguida por uno o más dígitos hexadecimales ASCII y termina con un punto y coma.

:::interactive_editor

```html
&#x3C;
```

:::
