## --text--

¿Cuál es una diferencia fundamental entre los algoritmos merge sort y quicksort?

## --answers--

Merge sort intercambia elementos adyacentes repetidamente hasta que están ordenados.

---

Quicksort dividirá la lista alrededor de un pivote y ordenará recursivamente las sublistas, mientras que merge sort divide la lista por la mitad y combina las mitades ordenadas.

---

Merge sort usa una tabla hash para contar elementos, mientras que quicksort usa una deque para reordenar elementos.

---

Quicksort siempre usa un ciclo while para ordenar, mientras que merge sort usa un ciclo for.
