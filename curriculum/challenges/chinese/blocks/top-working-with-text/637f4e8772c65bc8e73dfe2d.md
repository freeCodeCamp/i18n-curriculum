# --description--

超文本标记语言评论对浏览器不可见；它们允许我们对你的代码进行评论，以便其他开发者或你未来的自己可以阅读它们，并获得关于代码中某些可能不清楚内容的上下文。

编写超文本标记语言评论很简单：你只需用 `<!--` 和 `-->` 标签将评论括起来。例如：

```html
<h1> View the html to see the hidden comments </h1>

<!-- I am a html comment -->

<p>Some paragraph text</p>

<!-- I am another html comment -->
```

# --assignment--

为了练习在 HTML 中处理文本，创建一个使用不同标题、使用段落且段落中有部分文本加粗和斜体的简单博客文章页面。你可以使用 [Lorem Ipsum](https://loremipsum.io) 生成虚拟文本，代替真实文本来构建你的网站。
