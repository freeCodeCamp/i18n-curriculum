# --description--

To `depend on` é usado para dizer que algo pode mudar com base em coisas diferentes. Por exemplo:

- `The project depends on good teamwork.` - O projeto terá sucesso somente se a equipe trabalhar bem junta.

- `Developers depend on tests to find errors.` - Programadores precisam de testes para verificar se o código tem erros.

Note que depois de `depend` sempre há a preposição `on`.
