# --interactive--

Sifa za flex ni sifa unazoweza kutumia kwenye vyombo vya kubadilika ili kuamua mgawanyo wa vipengele vidogo. Tutazungumzia baadhi ya sifa zinazotumika sana: `flex-wrap`, `justify-content`, na `align-items`.

Tuanze na `flex-wrap`. Sifa hii inaamua jinsi vipande vya kubadilika vinavyopakiwa ndani ya chombo cha kubadilika ili kufaa nafasi iliyopo. `flex-wrap` inaweza kuchukua thamani tatu zinazowezekana: `nowrap`, `wrap`, na `wrap-reverse`. `nowrap` ni thamani ya chaguo-msingi—vipande vya kubadilika havitapakiwa kwenye mstari mpya, hata kama upana wao unazidi upana wa kontena.

Katika msimbo hapa chini, tuna vipande vitatu vya `div`. Tuwe makini na `width`. Kontena la `main` lililopakwa mpaka jekundu lina `width` ya `200px`, wakati vipande vitatu vya `div` vilivyo ndani yake vina `width` ya `240px` (`80px` kila kimoja):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::


Upana wa vipande vya `div` unazidi upana wa kontena lao, lakini kwa chaguo-msingi vitapunguzwa ili kufaa nafasi iliyopo. Ikiwa unataka vipande vipakwe mstari mpya wanapozidi upana wa kontena lao, unaweza kuweka `flex-wrap: wrap` kwenye chombo cha kubadilika:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  flex-wrap: wrap;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Vipande vya `div` vitapangwa upya kwa safu wanapozidi upana wa kontena lao. Unaweza kupakia vipande vya kubadilika kwa mpangilio wa nyuma kwa kutumia `flex-wrap: wrap-reverse`. Sifa ya `flex-flow` ni sifa ya ufupisho ya `flex-direction` na `flex-wrap`. Katika mfano huu, tumeweka `flex-direction` kuwa `column` na `flex-wrap` kuwa `wrap-reverse`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  flex-flow: column wrap-reverse;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Nzuri. Sasa tuzungumzie `justify-content`. `justify-content` huweka vipengele vidogo sambamba na mhimili mkuu wa chombo cha kubadilika. Ikiwa utaweka thamani `flex-start` kwa `justify-content`, vipande vya kubadilika vitawekwa mwanzoni mwa mhimili mkuu. Hii inaweza kuwa kwa usawa au wima:

Nzuri. Sasa unajua jinsi ya kugawanya vipande vya kubadilika kwenye mhimili mkuu. Lakini pia unaweza kutaka kuviweka kwenye mhimili wa msalaba. Kumbuka kwamba mhimili wa msalaba ni wima kwa mhimili mkuu. Unaweza kufanya hivi kwa kutumia sifa ya `align-items`. Ili kuweka vipande katikati ya mhimili wa msalaba, unahitaji tu kuongeza `align-items: center` kwenye chombo cha kubadilika:

:::interactive_editor

```html

<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css

  height: 300px;
  display: flex;
  align-items: center;
  border: 2px solid red;
}

  width: 80px;
  height: 50px;
}

#second-div {
  background-color: #5c4db2;

#third-div {
  background-color: #4da3b2;
```

Katika mfano huu, vipande vya kubadilika vimewekwa katikati ya mhimili wa msalaba, ambao kwa chaguo-msingi ni wima. Ikiwa mhimili wa msalaba ni usawa, vitakuwa katikati kwa usawa badala yake. Kinyume chake, `align-items: flex-start` huweka vipande mwanzoni mwa mhimili wa msalaba:

Ili kunyoosha vipande vya kubadilika kwenye mhimili wa msalaba, unaweza kutumia `align-items: stretch`. Hii pia hufanya kazi na vipengele vinavyopimwa kiotomatiki, kama vile visivyo na `width` au `height` vilivyowekwa, au vinavyopimwa kwa kiwango cha chini tu cha `width` au `height`. Vipande vya kubadilika vitanyoshwa kujaza kontena kwa mwelekeo wa mhimili wa msalaba.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box">Three</div>
</div>

```

```css
.container {
  display: flex;
  align-items: stretch; 
  border: 2px solid #444;
  height: 200px; 
}

.box {
  width: 100px;           
  border: 1px solid #888;
  background-color: lightblue;
}
```

:::

Na mwisho kabisa, unaweza kutumia sifa ya `align-self` kuweka ulinganifu tofauti kwenye mhimili wa msalaba kwa kipande cha kubadilika binafsi. Kwa mfano, unaweza kukinyoosha kwa `align-self: stretch`.
