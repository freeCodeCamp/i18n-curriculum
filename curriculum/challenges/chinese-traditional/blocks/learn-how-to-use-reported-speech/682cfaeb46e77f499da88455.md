# --explanation--

在**間接引語**中，`to suggest` 用於分享某人給出的建議或想法。例如：  

- 直接引語：`I think we should update our firewall.`

- 轉述：`Jake suggested updating the firewall.`

這表示有人給出了建議或可能的解法，但沒有使用他們的確切字詞。
