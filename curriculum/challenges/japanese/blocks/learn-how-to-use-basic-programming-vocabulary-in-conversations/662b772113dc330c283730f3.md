# --description--

プログラミングで `declare variables` とは、変数を作成し、その型と初期値を定義することを意味します。Pythonでは、変数宣言に明示的な型定義が不要なため、値を代入するだけで `declare a variable` できます。

プログラミングで `declare a variable` とは、変数を作成して名前を付けることを意味します。
