# --description--

조건문을 비교하는 또 다른 방법은 삼항 연산자입니다. 삼항 연산자는 `if-else` 문을 간단히 작성하는 방법입니다. 조건을 평가하고 그 결과에 따라 값을 반환하는 데 사용합니다. 삼항 연산자의 구문은 다음과 같습니다:

```javascript
let result = condition ? value1 : value2;
```

위 예제에서 조건이 참이면 `result`의 값은 `value1`가 됩니다. 그렇지 않으면 `result`의 값은 `value2`가 됩니다.

삼항 연산자는 조건에 따라 변수에 값을 할당할 때 자주 사용됩니다. 또한 조건에 따라 값을 반환할 때도 사용됩니다.
