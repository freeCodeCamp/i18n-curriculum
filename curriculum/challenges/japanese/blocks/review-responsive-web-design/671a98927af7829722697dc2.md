# --interactive--

## レスポンシブウェブデザイン

- **定義**: レスポンシブデザインの核心は適応性です。ウェブサイトが閲覧されているデバイスの画面サイズや機能に応じてレイアウトやコンテンツを調整できる能力を指します。
- **フルイドグリッド**: ピクセルのような固定単位の代わりにパーセンテージなどの相対単位を使い、画面サイズに応じてコンテンツがリサイズおよび再配置される仕組みです。
- **柔軟な画像**: コンテナ内でリサイズされるよう設定されており、小さい画面でもコンテナからはみ出さないようにします。

## メディアクエリ

- **定義**: 開発者がデバイスの特性、主にビューポートの幅に基づいて異なるスタイルを適用できるようにするものです。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="mq-example">The background of this paragraph will change when the viewport is at least 768px wide.</p>
```

```css
.mq-example {
  padding: 10px;
  border: 1px solid #ccc;
  background-color: lightyellow;
}

@media screen and (min-width: 768px) {
  /* Styles for screens at least 768px wide */
  .mq-example {
    background-color: lightblue;
  }
}
```

:::

- **`all` メディアタイプ**: すべてのデバイスに適しています。メディアタイプが指定されていない場合のデフォルトです。
- **`print` メディアタイプ**: ページ物や印刷プレビューモードで画面上に表示される文書向けです。
- **`screen` メディアタイプ**: 主に画面向けに設計されています。
- **`aspect-ratio`**: ビューポートの幅と高さの比率を表します。

```css
@media screen and (aspect-ratio: 16/9) {
  /* Styles for screens with a 16:9 aspect ratio */
}
```

- **`orientation`**: デバイスが横向き（ランドスケープ）か縦向き（ポートレート）かを示すために使われます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="orientation-example">This paragraph's background will change in landscape orientation.</p>
```

```css
.orientation-example {
  padding: 10px;
  border: 1px solid #ccc;
  background-color: lightyellow;
}

@media screen and (orientation: landscape) {
  /* Styles for landscape orientation */
  .orientation-example {
    background-color: lightgreen;
  }
}
```

:::

- **`resolution`**: 出力デバイスの解像度をdpi（ドット毎インチ）またはdpcm（ドット毎センチメートル）で表すために使われます。

```css
@media screen and (min-resolution: 300dpi) {
  /* Styles for high-resolution screens */
}
```

- **`hover`**: 主な入力機構が要素の上にホバーできるかどうかをテストするために使われます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button>Hover over me</button>
```

```css
button {
  padding: 10px 20px;
  font-size: 1rem;
  background-color: lightgray;
  border: 1px solid #999;
  cursor: pointer;
}

@media (hover: hover) {
  /* Styles for devices that support hover */
  button:hover {
    background-color: yellow;
  }
}
```

:::

- **`prefers-color-scheme`**: ユーザーがライトテーマかダークテーマかをリクエストしているかを検出するために使われます。
- **メディアクエリと論理演算子**: `and` 演算子は複数のメディア機能を組み合わせるために使われ、`not` と `only` はメディアクエリを否定または単独で使うために使われます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="logical-example">This paragraph's background will change when the screen is at least 768px wide AND in landscape orientation.</p>
```

```css
.logical-example {
  padding: 10px;
  border: 1px solid #ccc;
  background-color: lightyellow;
}

@media screen and (min-width: 768px) and (orientation: landscape) {
  /* Styles for landscape screens at least 768px wide */
  .logical-example {
    background-color: lightcoral;
  }
}
```

:::

## 一般的なメディアブレイクポイント

- **定義**: メディアブレイクポイントは、ウェブサイトのデザインにおいてレイアウトやコンテンツが異なる画面サイズに対応して調整される特定のポイントです。スマートフォン、タブレット、デスクトップ画面をターゲットにするための一般的なブレイクポイントがありますが、すべてのデバイスのあらゆる画面サイズを追いかけるのは賢明ではありません。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="breakpoint-example">This text will change size when the viewport width is at least 768px.</p>
```

```css
.breakpoint-example {
  font-size: 1rem;
  padding: 10px;
  border: 1px solid #ccc;
}

/* Styles for screens wider than 768px */
@media screen and (min-width: 768px) {
  .breakpoint-example {
    font-size: 1.125rem;
    background-color: lightblue;
  }
}
```

:::

- **小型デバイス（スマートフォン）**: 最大640px
- **中型デバイス（タブレット）**: 641pxから1024px
- **大型デバイス（デスクトップ）**: 1025px以上

## モバイルファーストアプローチ

- **定義**: モバイルファーストアプローチは、レスポンシブウェブデザインにおける設計哲学および開発戦略で、より大きな画面向けに設計する前にモバイルデバイス向けのウェブサイトを優先して作成します。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <p>The width of this container changes based on the viewport size.</p>
</div>
```

```css
/* Base styles for mobile */
.container {
  width: 100%;
  padding: 10px;
  background-color: lightblue;
}

/* Styles for larger screens */
@media screen and (min-width: 768px) {
  .container {
    width: 750px;
    margin: 0 auto;
    padding: 20px;
  }
}

@media screen and (min-width: 1024px) {
  .container {
    width: 960px;
  }
}
```

:::
