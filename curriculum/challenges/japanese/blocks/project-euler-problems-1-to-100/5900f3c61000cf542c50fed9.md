# --description--

立方体の6つの面にはそれぞれ異なる数字（0から9まで）が書かれており、2つ目の立方体にも同様に数字が書かれています。これら2つの立方体を並べて様々な2桁の数字を作ることができます。

例えば、平方数の64は次のように作れます：

<img alt="two cubes, one with the number 6 and the other with number 4" src="https://cdn-media-1.freecodecamp.org/project-euler/cube-digit-pairs.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

実際には、両方の立方体に数字を慎重に選ぶことで、100未満のすべての平方数：01、04、09、16、25、36、49、64、81 を表示することが可能です。

例えば、{0, 5, 6, 7, 8, 9} を一方の立方体に、{1, 2, 3, 4, 8, 9} をもう一方の立方体に置く方法があります。

ただし、この問題では6または9を上下逆さまにして使うことを許可します。つまり、{0, 5, 6, 7, 8, 9} と {1, 2, 3, 4, 6, 7} の組み合わせでも9つの平方数すべてを表示できます。そうでなければ09を作ることは不可能です。

異なる配置を判断する際には、各立方体の数字に注目し、順序は考慮しません。

<div style="margin-left: 4em;">
  {1, 2, 3, 4, 5, 6} は {3, 6, 4, 1, 2, 5} と同じです。<br>
  {1, 2, 3, 4, 5, 6} は {1, 2, 3, 4, 5, 9} とは異なります。
</div>

しかし、6と9を逆さまにできるため、最後の例の2つの異なるセットはどちらも2桁の数字を作る目的では拡張セット {1, 2, 3, 4, 5, 6, 9} を表しています。

2つの立方体の異なる配置のうち、すべての平方数を表示できるものは何通りありますか？
