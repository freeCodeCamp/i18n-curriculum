# --interactive--

JavaScript 中的陣列是有序的值群集，每個值由數字索引識別。JavaScript 陣列中的值可以是不同的資料型別，包括數字、字串、布林值、物件，甚至其他陣列。

要在 JavaScript 中創建陣列，你可以使用方括號 `[]`，並用逗號分隔值。以下是一個範例：

```js
let fruits = ["apple", "banana", "orange"];
```

在此範例中，我們宣告一個變數 `fruits` 並指定一個包含三個字串值的陣列：`apple`、`banana` 和 `orange`。

陣列的一個主要特性是它們是從零開始編號的，這表示陣列中的第一個元素的索引是 `0`，第二個元素的索引是 `1`，依此類推。你可以使用索引來存取陣列中的單一元素。例如：

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits[0]); // "apple"
console.log(fruits[2]); // "orange"
```

:::

在這個範例中，我們使用索引 `0` 來存取第一個元素（`apple`）和索引 `2` 來存取第三個元素（`orange`）。

JavaScript 中的陣列有一個特殊的 `length` 屬性，用來傳回陣列中元素的數量。你可以使用 `length` 關鍵字來存取這個屬性。例如：

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits.length); // 3
```

:::

JavaScript 中陣列的另一個關鍵特性是它們是動態的，這表示它們的大小可以在創建後改變。你可以使用各種陣列方法，像是 `push()`、`pop()`、`shift()`、`unshift()`、`splice()` 等，為陣列添加或移除元素。這些方法將在接下來的課程中教授。

JavaScript 陣列在你的程式中用於資料儲存時非常多功能且有用。在整個這個模組中，你將親自看到使用陣列如何幫助你高效管理和操作資料群集。
