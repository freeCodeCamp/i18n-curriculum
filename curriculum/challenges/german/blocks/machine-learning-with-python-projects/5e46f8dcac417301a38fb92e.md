Für diese Aufgabe vervollständigen Sie den Code, um Bilder von Hunden und Katzen zu klassifizieren. Ein Teil des Codes ist vorgegeben, aber einige Codeabschnitte müssen Sie ausfüllen, um diese Aufgabe abzuschließen. Lesen Sie die Anweisungen in jeder Textzelle, damit Sie wissen, was Sie in jeder Codezelle tun müssen.

Die erste Codezelle importiert die benötigten Bibliotheken. Die zweite Codezelle lädt die Daten herunter und legt wichtige Variablen fest. Die dritte Zelle ist der erste Ort, an dem Sie eigenen Code schreiben.

Die Struktur der heruntergeladenen Datensatzdateien sieht so aus (Sie werden feststellen, dass das Testverzeichnis keine Unterverzeichnisse hat und die Bilder nicht beschriftet sind):

```py
cats_and_dogs
|__ train:
    |______ cats: [cat.0.jpg, cat.1.jpg ...]
    |______ dogs: [dog.0.jpg, dog.1.jpg ...]
|__ validation:
    |______ cats: [cat.2000.jpg, cat.2001.jpg ...]
    |______ dogs: [dog.2000.jpg, dog.2001.jpg ...]
|__ test: [1.jpg, 2.jpg ...]
```

Sie können die Anzahl der Epochen und die Batch-Größe anpassen, wenn Sie möchten, aber das ist nicht erforderlich.

## Cell 3

Jetzt sind Sie dran! Setzen Sie jede der Variablen in dieser Zelle korrekt. (Sie sollten nicht mehr `None` entsprechen.)

Erstellen Sie Image-Generatoren für jeden der drei Bilddatensätze (Training, Validierung, Test). Verwenden Sie `ImageDataGenerator`, um die Bilder zu lesen / zu dekodieren und in Fließkomma-Tensoren umzuwandeln. Verwenden Sie das Argument `rescale` (und vorerst keine anderen Argumente), um die Tensoren von Werten zwischen 0 und 255 auf Werte zwischen 0 und 1 zu skalieren.

Für die `*_data_gen`-Variablen verwenden Sie die `flow_from_directory`-Methode. Übergeben Sie die Batch-Größe, das Verzeichnis, die Zielgröße (`(IMG_HEIGHT, IMG_WIDTH)`), den Klassenmodus und alles andere, was erforderlich ist. `test_data_gen` wird die kniffligste sein. Für `test_data_gen` stellen Sie sicher, dass Sie `shuffle=False` an die `flow_from_directory`-Methode übergeben. So bleibt die Reihenfolge der finalen Vorhersagen erhalten, die unser Test erwartet. Für `test_data_gen` ist es auch hilfreich, die Verzeichnisstruktur zu beachten.


Nachdem Sie den Code ausgeführt haben, sollte die Ausgabe so aussehen:

Kompilieren Sie das Modell, indem Sie Argumente übergeben, um den Optimierer und den Verlust festzulegen. Übergeben Sie auch `metrics=['accuracy']`, um die Trainings- und Validierungsgenauigkeit für jede Trainings-Epoche anzuzeigen.
