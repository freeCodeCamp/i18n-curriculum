# --description--

Il prossimo tipo di loop che imparerai si chiama loop `do...while`. Si chiama loop `do...while` perché eseguirà prima `do` una passata del codice all'interno del loop a prescindere da tutto, e poi continuerà a eseguire il loop `while` finché la condizione specificata risulta `true`.

```js
const ourArray = [];
let i = 0;

do {
  ourArray.push(i);
  i++;
} while (i < 5);
```

L'esempio sopra si comporta in modo simile ad altri tipi di loop, e l'array risultante sarà simile a `[0, 1, 2, 3, 4]`. Tuttavia, ciò che rende il loop `do...while` diverso dagli altri loop è come si comporta quando la condizione fallisce al primo controllo. Vediamo questo in azione. Ecco un normale loop `while` che eseguirà il codice nel loop finché `i < 5`:

```js
const ourArray = []; 
let i = 5;

while (i < 5) {
  ourArray.push(i);
  i++;
}
```

In questo esempio, inizializziamo il valore di `ourArray` a un array vuoto e il valore di `i` a 5. Quando eseguiamo il loop `while`, la condizione risulta `false` perché `i` non è minore di 5, quindi non eseguiamo il codice all'interno del loop. Il risultato è che `ourArray` finirà senza valori aggiunti, e sarà ancora simile a `[]` quando tutto il codice nell'esempio sopra avrà terminato l'esecuzione. Ora, dai un'occhiata a un loop `do...while`:

```js
const ourArray = []; 
let i = 5;

do {
  ourArray.push(i);
  i++;
} while (i < 5);
```

In questo caso, inizializziamo il valore di `i` a 5, proprio come abbiamo fatto con il loop `while`. Quando arriviamo alla riga successiva, non c'è una condizione da valutare, quindi entriamo nel codice dentro le parentesi graffe ed eseguiamo il codice. Aggiungeremo un singolo elemento all'array e poi incrementeremo `i` prima di arrivare al controllo della condizione. Quando infine valutiamo la condizione `i < 5` nell'ultima riga, vediamo che `i` è ora 6, il che fa fallire il controllo condizionale, quindi usciamo dal loop e abbiamo finito. Alla fine dell'esempio sopra, il valore di `ourArray` è `[5]`. In sostanza, un loop `do...while` garantisce che il codice all'interno del loop venga eseguito almeno una volta. Proviamo a far funzionare un loop `do...while` aggiungendo valori a un array.
