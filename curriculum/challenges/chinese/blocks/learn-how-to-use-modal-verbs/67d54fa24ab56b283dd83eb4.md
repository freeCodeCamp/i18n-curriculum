# --explanation--

`must` 和 `have to` 都表示义务或必要性，但它们的用法有细微差别。

`Must` 用于强制性的内部义务或官方规则。它通常听起来更正式或权威。例如：

`All sensitive data must be encrypted。`——一个严格的规则或要求。

`Have to` 用于外部义务，通常基于规则或情况，而非个人权威。例如：

你必须确保加密密钥被安全地保存。
