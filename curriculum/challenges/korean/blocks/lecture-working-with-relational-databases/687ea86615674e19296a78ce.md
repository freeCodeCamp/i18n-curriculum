# --description--

관계형 데이터베이스는 웹 개발, 재고 관리 시스템, 의료 시스템, 전자상거래 애플리케이션 등 다양한 분야에서 활용됩니다.

가장 흔한 관계형 데이터베이스로는 MySQL, PostgreSQL, SQLite, Microsoft SQL Server가 있습니다. 각 데이터베이스에 대해 간단히 알아봅시다.

MySQL은 오픈 소스 관계형 데이터베이스 관리 시스템입니다. 사용하기 쉽고 신뢰할 수 있으며 실제 애플리케이션 구축에 효율적이어서 웹 개발에 매우 인기가 많습니다. 또한 이를 지원하고 유지하는 활발한 개발자 커뮤니티가 있습니다.

PostgreSQL은 공식적으로 "세계에서 가장 진보된 오픈 소스 관계형 데이터베이스"로 설명됩니다. 신뢰성, 데이터 무결성, 확장성으로 잘 알려진 무료 오픈 소스 객체-관계형 데이터베이스 시스템입니다. 예를 들어, 사용자 정의 데이터 타입, 맞춤 함수 정의, 데이터베이스를 다시 컴파일하지 않고 다양한 프로그래밍 언어로 코드를 작성할 수 있습니다.

SQLite는 무료, 오픈 소스, 경량의 파일 기반 SQL 데이터베이스 엔진입니다. 세계에서 가장 널리 사용되는 데이터베이스 엔진 중 하나입니다. SQLite는 독립적이고 서버가 필요 없으며, 초기 설정 과정이 필요 없는 제로 구성 데이터베이스로 빠르게 시작하기에 좋습니다.

이 세 가지 데이터베이스는 모두 무료이자 오픈 소스입니다. Microsoft SQL Server와 Oracle 데이터베이스 같은 인기 있는 상용 옵션도 있습니다.

Microsoft SQL Server는 Microsoft에서 개발한 인기 있는 관계형 데이터베이스 관리 시스템으로 다양한 애플리케이션에서 사용됩니다.

Oracle 데이터베이스는 고성능이 필요한 엔터프라이즈 애플리케이션에 자주 사용되는 확장 가능한 관계형 데이터베이스 관리 시스템입니다.

좋습니다. 이제 다양한 옵션에 대해 알았으니 PostgreSQL을 설치하는 방법을 살펴봅시다.

먼저 공식 웹사이트 <a href="https://www.postgresql.org/" target="_blank">https://www.postgresql.org/</a>에 접속하세요.

사이트에 접속하면 "Download" 버튼을 클릭하세요.

그러면 운영체제를 선택할 수 있는 페이지로 이동합니다. 운영체제 중 하나를 클릭하면 해당 OS용 설치 지침이 나타납니다.

Windows 운영체제 설치 과정을 살펴보겠지만, 다른 운영체제도 비슷한 과정입니다.

이 페이지에서 "Download the installer" 링크를 클릭해 설치 프로그램을 다운로드하세요.

그러면 설치할 PostgreSQL 버전을 선택할 수 있는 페이지로 이동합니다. 해당 아이콘을 클릭하면 다운로드가 시작됩니다.

다운로드 폴더나 기본 다운로드 저장 위치에서 설치 프로그램을 찾을 수 있습니다.

설치 프로그램을 더블클릭하면 설치가 시작됩니다. Next를 클릭하세요.

PostgreSQL을 설치할 디렉터리를 선택하세요. 기본 경로가 표시되지만 원하는 대로 변경할 수 있습니다.

그다음 설치할 컴포넌트를 선택하세요. 네 가지 컴포넌트가 있습니다:

- PostgreSQL Server, 데이터베이스 엔진 자체
- pgAdmin 4, 데이터베이스 관리를 위한 사용자 친화적 도구
- Stack Builder, PostgreSQL 관련 추가 소프트웨어와 확장 기능을 다운로드 및 설치하는 도구
- Command Line Tools, 터미널에서 PostgreSQL 서버와 직접 상호작용할 수 있는 도구. 여기에는 SQL 쿼리를 실행하는 대화형 터미널인 psql이 포함됩니다.

준비가 되면 Next를 클릭하세요.

데이터를 저장할 디렉터리를 선택하세요. 기본 경로가 표시되지만 변경할 수 있습니다. 준비가 되면 Next를 클릭하세요.

이제 데이터베이스 슈퍼유저, 즉 모든 데이터베이스와 객체에 무제한 접근 권한이 있는 사용자의 비밀번호를 입력해야 합니다. 비밀번호를 입력하고 다시 입력한 후 Next를 클릭하세요.

그다음 서버가 수신할 포트 번호를 선택하세요. 기본값은 5432입니다. Next를 클릭하세요.

새 데이터베이스 클러스터에 사용할 로케일을 선택해야 합니다. 이는 로케일에 민감한 데이터 타입을 데이터베이스가 해석하는 데 중요합니다. Next를 클릭하세요.

마지막으로 설치 전 요약 화면이 나타납니다. 내용을 검토한 후 Next를 클릭하세요.

설치를 시작할지 확인해야 합니다. Next를 클릭하면 설치가 시작되며 몇 분 정도 걸립니다.

설치가 완료되면 설치한 컴포넌트에 따라 Stack Builder를 실행해 추가 도구, 드라이버, 애플리케이션을 다운로드 및 설치할지 선택할 수 있습니다.

Stack Builder를 실행하기로 결정하면 설치를 시작할 수 있는 화면이 나타납니다. 나중에 하려면 Cancel을 클릭하세요.

축하합니다! 이제 컴퓨터에 PostgreSQL이 설치되었습니다. 다른 운영체제도 비슷한 과정입니다. 이후 데이터베이스를 사용할 수 있습니다.

애플리케이션에서 pgAdmin을 검색하세요. pgAdmin은 PostgreSQL용 오픈 소스 관리 및 개발 플랫폼으로, 컴포넌트 선택 시 설치됩니다. pgAdmin 4를 클릭해 실행하세요.

애플리케이션이 시작되면 초기 로딩 화면이 나타납니다. 이후 새 서버 시작, 데이터베이스 확인, pgAdmin 설정 등 다양한 작업을 할 수 있는 대시보드가 표시됩니다. 문서 링크도 유용하게 제공됩니다.

왼쪽 상단의 "Servers"를 클릭하면 설치 시 설정한 비밀번호를 입력해야 합니다.

비밀번호를 입력하면 데이터베이스와 그 안의 표에 접근할 수 있는 화면이 나타납니다.

또한 SQL 쿼리를 위한 대화형 터미널인 psql도 매우 유용한 도구입니다. 시스템에서 psql을 검색해 실행하세요.

SQL Shell이 열리며 명령어 실행을 시작할 수 있습니다.

이것이 Windows에서 PostgreSQL을 설치하는 방법입니다. macOS와 Linux도 비슷한 과정입니다.

macOS에 설치할 경우 패키지 관리자 Homebrew를 사용할 수도 있습니다. Homebrew가 설치되어 있다면 다음 명령어로 PostgreSQL을 설치하세요:

```bash
brew install postgresql@<version>
```

설치할 버전을 지정하려면 at(`@`) 기호 뒤에 버전 번호를 적으세요.
