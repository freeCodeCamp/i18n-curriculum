# --explanation--

`To struggle` 意味着在某事上遇到困难。例如：

- `The program is struggling to load.`——这意味着程序当前在装载时遇到困难。

`Present Continuous` 时态用于描述**正在发生的**动作。

在上面的例子中，请注意，当用动词 `struggle` 构成 `Present Continuous` 时，最后的 `e` 被省略，变成了 `struggling`。这是以 `e` 结尾的动词的常见规则。

在 Tom 的句子中，他使用了 `Present Continuous` 时态来描述他在使用 IDE（集成开发环境）时遇到的持续困难。

你应该为 `struggle` 添加 `-ing`，因为动作正在发生。
