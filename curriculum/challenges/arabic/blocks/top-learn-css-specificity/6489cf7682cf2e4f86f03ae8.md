# --description--

تشير الوراثة إلى بعض خصائص CSS التي، عند تطبيقها على عنصر، يرثها العناصر التابعة لذلك العنصر حتى لو لم نكتب قاعدة صريحة لتلك العناصر التابعة. عادةً ما تُورث الخصائص المتعلقة بالطباعة مثل (`color`، `font-size`، `font-family`، وغيرها)، بينما معظم الخصائص الأخرى لا تُورث.

الاستثناء من ذلك هو عند استهداف عنصر مباشرةً، إذ يتفوق هذا دائمًا على الوراثة:

```html
<!-- index.html -->

<div id="parent">
  <div class="child"></div>
</div>
```

```css
/* styles.css */

#parent {
  color: red;
}

.child {
  color: blue;
}
```

على الرغم من أن عنصر `parent` له خصوصية أعلى باستخدام معرف ID، فإن عنصر `child` سيُطبق عليه نمط `color: blue` لأن هذا الإعلان يستهدفه مباشرة، بينما `color: red` من العنصر الأب يُورث فقط.
