# --description--

`typeof` を使うと、変数のデータ構造や型を確認できます。これは複数のデータ型を扱う際のデバッグに役立ちます。もし2つの数値を足しているつもりでも、実際には片方が文字列の場合、結果が予期しないものになることがあります。計算や関数の呼び出しで型のエラーが潜んでいることもあります。特にJavaScript Object Notation（JSON）オブジェクトの形で外部データにアクセスして操作するときは注意してください。

以下は `typeof` を使った例です：

```js
console.log(typeof "");
console.log(typeof 0);
console.log(typeof []);
console.log(typeof {});
```

順番に、コンソールには文字列 `string`、`number`、`object`、`object` が表示されます。

JavaScriptは7つのプリミティブ（不変）データ型を認識しています：`Boolean`、`Null`、`Undefined`、`Number`、`String`、`Symbol`（ES6で追加）、`BigInt`（ES2020で追加）、そして変更可能なアイテムの型として`Object`があります。JavaScriptでは配列は技術的にはオブジェクトの一種であることに注意してください。
