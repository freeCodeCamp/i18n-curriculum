# --interactive--

## JavaScript関数

- 関数は特定のタスクを実行する再利用可能なコードのブロックです。
- 関数は`function`キーワードの後に名前、パラメータのリスト、そしてタスクを実行するコードのブロックを続けて定義できます。

:::interactive_editor

```js
function addNumbers(x, y, z) {
  return x + y + z;
}

console.log(addNumbers(5, 3, 8)); // Output: 16
```

:::

- 引数は関数が呼び出されるときに渡される値です。
- 関数呼び出しとは、関数の名前の後に丸括弧を付けて、必要に応じて丸括弧内に引数を含めてプログラム内で関数を実行するプロセスです。
- 関数が実行を終えると、必ず戻り値を返します。
- デフォルトでは、関数の戻り値は`undefined`です。
- `return`キーワードは関数から返す値を指定し、関数の実行を終了させます。
- デフォルトパラメータは、関数が呼び出されたときに引数が渡されなかった場合に使われるあらかじめ定義された値を関数に持たせることができます。これにより関数がより柔軟になり、特定の引数が省略された場合のエラーを防げます。

:::interactive_editor

```js
const calculateTotal = (amount, taxRate = 0.05) => {
  return amount + (amount * taxRate);
};

console.log(calculateTotal(100)); // Output: 105
```

:::

- 無名関数は名前を持たない関数で、変数に代入できます。変数に代入することで、その変数がアクセス可能な場所ならどこでも再利用できます。

:::interactive_editor

```js
const multiplyNumbers = function(firstNumber, secondNumber) {
  return firstNumber * secondNumber;
};

console.log(multiplyNumbers(4, 5)); // Output: 20
```

:::

## アロー関数

- アロー関数はJavaScriptで関数を書くより簡潔な方法です。

:::interactive_editor

```js
const calculateArea = (length, width) => {
  const area = length * width;
  return `The area of the rectangle is ${area} square units.`;
};

console.log(calculateArea(5, 10)); // Output: "The area of the rectangle is 50 square units."
```

:::

- アロー関数を定義するときは`function`キーワードは不要です。
- パラメータが1つだけの場合、パラメータリストの丸括弧を省略できます。

:::interactive_editor

```js
const cube = x => {
  return x * x * x;
};

console.log(cube(3)); // Output: 27
```

:::

- 関数本体が単一の式だけの場合、波括弧と`return`キーワードを省略できます。

:::interactive_editor

```js
const square = number => number * number;

console.log(square(5)); // Output: 25
```

:::

## プログラミングにおけるスコープ

- **グローバルスコープ**：JavaScriptで最も外側のスコープです。グローバルスコープで宣言された変数はコードのどこからでもアクセスでき、グローバル変数と呼ばれます。
- **ローカルスコープ**：関数内で宣言された変数を指します。これらの変数は宣言された関数内でのみアクセス可能で、ローカル変数と呼ばれます。
- **ブロックスコープ**：ブロックとは、`{}`で囲まれた一連の文のことで、`if`文やループなどに使われます。
- `let`と`const`によるブロックスコープは、変数のアクセス制御をさらに細かくし、エラーを防ぎコードの予測可能性を高めます。
