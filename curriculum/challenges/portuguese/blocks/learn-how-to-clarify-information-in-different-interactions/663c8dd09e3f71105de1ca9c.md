# --description--

`Whoa` é uma palavra que você diz quando está surpreso ou impressionado com algo.Por exemplo, se você vir um arco-íris grande e bonito, você pode dizer, `Whoa!` para mostrar que está impressionado.Também pode ser usado para dizer a alguém para parar ou desacelerar.

Quando você diz que algo é `a mouthful`, você quer dizer que algo (como uma palavra ou frase) é longo ou difícil de dizer.Por exemplo, se uma palavra ou nome tem muitas letras e sons, você pode dizer `it's a mouthful`.Significa que é difícil dizer tudo de uma vez.
