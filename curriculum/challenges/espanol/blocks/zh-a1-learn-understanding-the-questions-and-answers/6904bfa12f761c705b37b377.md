# --description--

Wang Hua le hace una pregunta a alguien. Ella usa una palabra para referirse a la otra persona.

# --fillInTheBlank--

## --sentence--

`你好 (nǐ hǎo)，我是王华 (wǒ shì Wang Hua)，请问 (qǐng wèn) BLANK 叫什么名字 (jiào shén me míng zi)？`

## --blanks--

`你 (nǐ)`

### --feedback--

Esta palabra significa "tú" y se refiere a alguien con quien estás hablando.

# --explanation--

`你 (nǐ)` significa "tú". Se usa para hablar directamente con otra persona. Por ejemplo:

`你是刘明吗 (nǐ shì Liu Ming ma)？` – ¿Eres Liu Ming?

Has aprendido cómo usar `我 (wǒ)` para referirte a ti mismo. Tanto `我 (wǒ)` como `你 (nǐ)` son **pronombres personales**, lo que significa que se usan para referirse a personas.
