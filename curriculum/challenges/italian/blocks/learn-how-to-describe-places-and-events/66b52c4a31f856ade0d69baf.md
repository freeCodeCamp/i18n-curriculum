# --explanation--

`Currently` significa al momento presente o ora. Indica che qualcosa sta accadendo in questo momento. Per esempio:

- `She is currently debugging the software.` - Lei sta eseguendo il debug del software proprio ora.

- `We are currently testing the new app.` - Tu e il tuo team state testando la nuova app in questo momento.
