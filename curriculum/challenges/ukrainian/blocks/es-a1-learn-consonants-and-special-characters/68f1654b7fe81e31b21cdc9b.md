# --description--

Літера `r` у іспанській мові називається `erre`.

Цей приголосний має два різні вимови.

Одинарний `r` має м’який звук. Наприклад, `cero` ("zero").

Подвійний `r` (`rr`) або `r` на початку слова дає більш твердий звук. Наприклад, `perro` ("dog") або `ruido` ("noise").
