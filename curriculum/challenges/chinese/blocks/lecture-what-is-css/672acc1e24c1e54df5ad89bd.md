# --interactive--

在超文本标记语言 和 CSS 中，元素被分类为 **内联元素** 或 **块级元素**，这种分类决定了它们在文档版型中的行为。

理解此差异对于控制你的内容在网页上的显示方式至关重要。

块级元素是默认占据其容器可用全部宽度的元素，横跨其容器的宽度。

这些元素总是从新的一行开始，并将其他内容推到下一行，创建一个“块”内容。

块级元素默认应用 CSS 属性 `display: block;`。该属性确保元素拉伸以填充容器的宽度并出现在新行上。

一些常见的块级元素有 `div` 元素、段落、标题、有序列表、无序列表和区块元素。

这是一个块级元素的代码示例：

:::interactive_editor

```html
<p style="border: 2px solid red;">
  First paragraph
</p>
<p>Second paragraph</p>
```

:::

在此示例中，我们有两个段落 `Element`，其中第一个有一个红色的 `border`。

这两个段落元素不在同一行，因为它们默认是块级元素。

因此，两个段落元素都会占据其容器的全部宽度，在本例中该容器是 `body` 元素。

块级元素非常适合当你想让内容垂直堆叠时使用，例如段落、章节或较大的内容块。它们通常用于定义网页的版型和结构。

内联元素与块级元素不同，它们只占用所需的宽度且不会另起一行。这些元素在内容中流动，允许文本和其他内联元素与它们并排显示。

内联元素默认应用 CSS 属性 `display: inline;`。该属性确保元素保持在内容流内且不会换行。

常见的内联元素有 `span`、`anchor` 和 `img` 元素。

这是一个更好理解内联元素的示例：

:::interactive_editor

```html
<p>This is a
  <span style="color: red; border: 2px solid green;">red</span>
  word inside a paragraph.
</p>
```

:::

在此示例中，我们有一个嵌套在段落元素内的 `span` 元素。`span` 元素具有 `red` 文本颜色和一个 `green` 边框，以便你更好地看到高亮的单词。

如你所见，`span` 元素只占用单词 "red" 的空间，并且不会将后续内容推到新行。

内联元素最适合用于为一行内的较小部分文本或内容设置样式，例如强调一个词、创建超链接或为段落的部分应用特定样式。
