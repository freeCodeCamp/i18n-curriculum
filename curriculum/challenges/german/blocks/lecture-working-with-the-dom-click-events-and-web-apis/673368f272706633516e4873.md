# --interactive--

Sie kennen sich vielleicht schon mit dem Arbeiten mit Attributen in Ihren HTML- und CSS-Projekten aus. Aber wussten Sie, dass Sie Attribute in Ihrer JavaScript-Datei zu Ihren HTML-Elementen hinzufügen können?

In dieser Lektion lernen Sie, wie Sie mit der `setAttribute()`-Methode Attribute zu Ihren HTML-Elementen hinzufügen.

Hier ist die grundlegende Syntax:

```js
setAttribute(attribute, value);
```

Schauen wir uns ein paar Beispiele an, um besser zu verstehen, wie man die `setAttribute()`-Methode verwendet.

Im ersten Beispiel haben wir ein `p`-Element im HTML:

```html
<p id="para">I am a paragraph</p>
```

Um ein `class`-Attribut hinzuzufügen, müssen wir zuerst mit der `p`-Methode auf dieses `getElementById()`-Element zugreifen. Dann können wir die `setAttribute()`-Methode verwenden, um das `class`-Attribut hinzuzufügen und den Wert auf `my-class` zu setzen.

Wir verwenden `outerHTML`, um das gesamte HTML-Element einschließlich seiner Attribute zu protokollieren.

:::interactive_editor

```html
<p id="para">I am a paragraph</p>
<script src="index.js"></script>
```

```js
const para = document.getElementById("para");
para.setAttribute("class", "my-class");

console.log(`${para.outerHTML}`);
```

:::

Wenn ein Attribut bereits an einem HTML-Element vorhanden ist, können Sie seinen Wert mit der `setAttribute()`-Methode aktualisieren.

In diesem Beispiel haben wir ein `div`-Element mit einem `class`-Attribut, das auf `my-class` gesetzt ist:

```html
<div class="my-class"></div>
```

Um den Wert des `class`-Attributs auf example zu aktualisieren, können wir erneut die `setAttribute()`-Methode verwenden:

:::interactive_editor

```html
<div class="my-class"></div>
<script src="index.js"></script>
```

```js
const divEl = document.querySelector(".my-class");
divEl.setAttribute("class", "example");

console.log(`${divEl.outerHTML}`);
```

:::

Wann könnten Sie die `setAttribute()`-Methode in der Praxis verwenden?

Wenn Sie eine dynamische Bildergalerie erstellen, müssen Sie möglicherweise das `src`-Attribut eines Bild-Elements aktualisieren, wenn ein Benutzer auf ein Vorschaubild klickt.

Ein weiteres Beispiel ist, wenn Sie mit Formularvalidierung arbeiten und bestimmte Attribute wie `required` oder `minlength` zu einem `input`-Element hinzufügen müssen.
