# --description--

Дано рядок, що містить одне або кілька слів. Поверніть акронім цих слів, дотримуючись таких обмежень:

- Акронім повинен складатися з першої літери кожного слова, написаної з великої літери, якщо не вказано інше.
- Акронім має ігнорувати першу літеру таких слів, якщо вони не є першим словом у заданому рядку: `a`, `for`, `an`, `and`, `by` та `of`.
- Літери акроніма повинні повертатися в тому порядку, в якому вони йдуть.
- Акронім не повинен містити жодних пробілів.

```js
function buildAcronym(str) {
  const smallWords = ["a", "for", "an", "and", "by", "of"];
  const words = str.split(" ");
  let acronym = "";

  for (let i = 0; i < words.length; i++) {
    const word = words[i];
    const lowerWord = word.toLowerCase();

    if (i === 0 || !smallWords.includes(lowerWord)) {
      acronym += word[0].toUpperCase();
    }
  }

  return acronym;
}
```
