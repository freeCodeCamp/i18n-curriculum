# --description--

`Probably`는 어떤 일이 일어날 가능성이 있다고 생각하지만 100% 확신하지 못할 때 사용합니다.

예를 들어, `The code will probably run without errors after the update`라고 말할 수 있는데, 이는 그렇게 믿지만 여전히 작은 문제 가능성이 남아 있음을 의미합니다.
