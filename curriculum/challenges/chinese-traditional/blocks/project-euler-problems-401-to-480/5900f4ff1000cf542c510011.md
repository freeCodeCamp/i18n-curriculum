# --description--

可以證明多項式 $n^4 + 4n^3 + 2n^2 + 5n$ 對每個整數 $n$ 都是 6 的倍數。也可以證明 6 是滿足此性質的最大整數。

定義 $M(a, b, c)$ 為最大的 $m$，使得對所有整數 $n$，$n^4 + an^3 + bn^2 + cn$ 都是 $m$ 的倍數。 例如，$M(4, 2, 5) = 6$。

另外，定義 $S(N)$ 為所有 $0 < a, b, c ≤ N$ 的 $M(a, b, c)$ 之和。

我們可以驗證 $S(10) = 1\,972$ 和 $S(10\,000) = 2\,024\,258\,331\,114$。

令 $F_k$ 為費波那契數列：

- $F_0 = 0$、$F_1 = 1$ 和
- $F_k = F_{k - 1} + F_{k - 2}$，當 $k ≥ 2$。

找出 $\sum S(F_k)$ 中 $2 ≤ k ≤ 1\,234\,567\,890\,123$ 的最後 9 位數字。
