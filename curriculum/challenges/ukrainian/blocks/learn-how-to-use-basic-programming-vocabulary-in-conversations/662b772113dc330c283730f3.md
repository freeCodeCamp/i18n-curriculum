# --description--

У програмуванні `declare variables` означає створити їх і визначити їхній тип та ініціальне значення. У Python ви можете `declare a variable` просто призначивши їм значення, оскільки мова не вимагає явного визначення типу для оголошення змінної.

`declare a variable` у програмуванні означає створити змінну і дати їй ім’я.
