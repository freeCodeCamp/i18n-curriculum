# --explanation--

Un `data breach` ocurre cuando información sensible es accedida por personas no autorizadas. Por ejemplo:

- `A data breach can happen if hackers get access to private information.` - Si los ciberdelincuentes roban información, se llama una violación de datos.

- `Companies must act quickly if there is a data breach.` - Cuando ocurre una violación de datos, las empresas necesitan responder rápido para solucionarlo.

Jessica dice que encriptar datos y asegurar las claves de encriptación es esencial para prevenir este riesgo.
