# --description--

Модальні дієслова, як `might` і `can`, використовуються для вираження можливості та здатності.

`Might` вживається для чогось, що могло б статися або бути правдою, наприклад, `It might rain today`.

`Can` використовується, щоб говорити про чиїсь здібності щось робити, наприклад, `She can speak three languages`. Давайте потренуємося використовувати ці слова у реченнях.
