# --interactive--

Sifa ya `z-index` katika CSS hutumika kudhibiti mpangilio wa kupanga wima wa vipengele vyenye nafasi maalum vinavyogongana kwenye ukurasa. Wakati vipengele vingi vimepangiliwa juu ya kila kimoja, thamani ya `z-index` huamua ni kipengele gani kinaonekana juu zaidi. Kadri thamani ya `z-index` inavyoongezeka, ndivyo kipengele kinavyokaribia mtazamaji, wakati thamani za chini huweka kipengele nyuma zaidi katika muktadha wa kupanga.

Hata hivyo, `z-index` hufanya kazi tu kwa vipengele vyenye nafasi maalum, ambayo inamaanisha kipengele lazima kiwe na thamani ya nafasi tofauti na `static`, kama vile `relative`, `absolute`, au `fixed`. Thamani ya kawaida ya `z-index` ni `auto`, ambayo huweka kipengele katika mpangilio wa kawaida wa kupanga. Hebu tuchunguze mfano wenye masanduku matatu:

```html
<div class="container">
  <div class="box1">Box 1</div>
  <div class="box2">Box 2</div>
  <div class="box3">Box 3</div>
</div>
```

Sasa, tunaweza kutumia mitindo fulani kuweka masanduku hayo yakigongana kama ifuatavyo:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box1">Box 1</div>
  <div class="box2">Box 2</div>
  <div class="box3">Box 3</div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

.box1 {
  position: absolute;
  z-index: 1;
  background: lightcoral;
  top: 20px;
  left: 20px;
  width: 100px;
  height: 100px;
}

.box2 {
  position: absolute;
  z-index: 3;
  background: gold;
  top: 40px;
  left: 40px;
  width: 100px;
  height: 100px;
}

.box3 {
  position: absolute;
  z-index: 2;
  background: lightgreen;
  top: 60px;
  left: 60px;
  width: 100px;
  height: 100px;
}
```

:::

Kwa kontena, nafasi itakuwa `relative` na masanduku yote yaliyopangwa ndani yatawekwa na nafasi ya `absolute`. Kila sanduku lina thamani tofauti ya `z-index` ambayo husababisha masanduku kupangwa juu ya kila mmoja.

Unaweza kufikiria `z-index` kama njia ya kuunda tabaka kwenye ukurasa wa mtandao, na vipengele vyenye thamani za `z-index` kubwa zaidi vitawekwa juu ya vyenye thamani ndogo. Hii ni muhimu hasa kudhibiti jinsi vipengele vinavyogongana vinavyotenda katika mipangilio tata, kama vile modali, dirisha la pop-up, au kidokezo cha muda.
