# --explanation--

Un `log` es un registro de eventos, acciones o mensajes creados por un sistema informático o software. Por ejemplo:

`The system logs showed several errors before the crash.` – Esto significa que los registros tenían información sobre problemas que ocurrieron.

Un `crash` es cuando un sistema informático o programa deja de funcionar correctamente de repente. Por ejemplo:

`The application crash caused all unsaved data to be lost.` – Esto significa que el programa falló inesperadamente y los datos no se guardaron.
