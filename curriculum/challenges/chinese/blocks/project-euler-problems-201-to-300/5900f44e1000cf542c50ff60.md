# --description--

序列1,1,1,3,5,9,17,31,57,105,193,355,653,1201 ......由T1 = T2 = T3 = 1和Tn = Tn-1 + Tn定义-2 + Tn-3。

由 $T_1 = T_2 = T_3 = 1$ 和 $T_n = T_{n - 1} + T_{n - 2} + T_{n - 3}$ 定义。

可以证明 27 不整除该序列的任何项。事实上，27 是第一个具有此性质的奇数。

查找不整除上述序列任何项的第 ${124}^{\text{th}}$ 个奇数。
