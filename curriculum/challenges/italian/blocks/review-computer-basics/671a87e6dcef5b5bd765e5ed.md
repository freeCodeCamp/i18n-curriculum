## Comprendere le basi del computer, di internet e degli strumenti per sviluppatori

- **Scheda madre**: contiene tutta la memoria, i connettori e i dischi rigidi necessari per far funzionare il computer. Funziona come la scheda principale del circuito del computer.
- **Unità centrale di elaborazione (CPU)**: un processore responsabile dell'esecuzione delle istruzioni e dei calcoli.
- **Memoria ad accesso casuale (RAM)**: un'area di archiviazione temporanea per la CPU del computer.
- **Disco rigido (HDD)**: un'area di archiviazione permanente usata per conservare i dati anche quando il computer è spento.
- **Unità a stato solido (SSD)**: memoria flash non volatile che può sostituire un disco rigido.
- **Alimentatore (PSU)**: responsabile della conversione dell'elettricità dalla presa a muro in una forma utilizzabile dal computer.
- **Unità di elaborazione grafica (GPU)**: responsabile del rendering delle immagini sullo schermo del computer.
- **Diversi tipi di fornitori di servizi internet**: un fornitore di servizi internet (ISP) è un'azienda che fornisce accesso a internet. Esistono diversi tipi di ISP, tra cui dial-up, DSL, cavo, fibra ottica e satellite.
- **Modi sicuri per accedere al computer**: esempi di modi sicuri per accedere al computer includono l'uso di una password forte, l'attivazione dell'autenticazione a due fattori e l'uso di un gestore di password.
- **Ambiente di sviluppo integrato (IDE)**: uno strumento che aiuta gli sviluppatori a scrivere, testare e eseguire il debug del codice in modo efficiente.
- **Editor di codice**: uno strumento che gli sviluppatori usano per scrivere ed eseguire il debug del codice.
- **Git**: un sistema di controllo versione popolare che permette agli sviluppatori di tracciare le modifiche nel codice e collaborare con altri.
- **Servizi di hosting cloud per repository**: un repository è un luogo di archiviazione per i file di progetto e la cronologia delle versioni. I servizi di hosting cloud più popolari per repository includono GitHub, GitLab e Bitbucket.
- **Gestori di pacchetti**: strumenti che aiutano gli sviluppatori a semplificare il processo di aggiungere, aggiornare e rimuovere librerie e dipendenze di progetto. Esempi includono npm, pip e Maven.
- **Librerie e framework per i test**: i test vengono eseguiti nel software per assicurarsi che il codice funzioni come previsto. Esempi di librerie e framework per i test includono Jest, PHPUnit e JUnit.

## Lavorare con file, sistemi di file e formati multimediali

- **Buone pratiche per la denominazione dei file**: è importante dare ai file nomi facili da capire e mantenere. Per esempio, `about-us.html` è un nome più descrittivo di `page1.html`.
- **Directory radice**: la directory di livello superiore in un sistema di file. Directory è un altro nome per cartella.
- **Markdown**: un linguaggio di markup comunemente usato per la documentazione e i file `README`. Un file `README` è un file che contiene informazioni su un progetto, come come installarlo e usarlo.
- **`index.html`**: rappresenta la pagina predefinita che viene mostrata quando un utente visita un sito web.
- **Creare, spostare ed eliminare file e cartelle usando Explorer/Finder**: Explorer è il gestore di file in Windows, e Finder è il gestore di file in macOS. Puoi usare questi strumenti per creare, spostare ed eliminare file e cartelle.
- **Ricerca di file e cartelle**: puoi usare la funzione di ricerca in Explorer o Finder per trovare file e cartelle sul tuo computer.
- **Tipi di file HTML, CSS e JS**: l'estensione `.html` è usata per i file HTML, `.css` per i file CSS e `.js` per i file JavaScript.
- **Formati comuni di immagini e grafica**: `JPEG` e `PNG` sono formati comuni di file immagine. `GIF` è un altro formato comune di file immagine che supporta l'animazione. `SVG` è un formato di file per grafica vettoriale.
- **Formati comuni di audio e video**: il formato `MP3` è comunemente usato per file audio. Il formato `MP4` è comunemente usato per file video. Il formato `MOV` è stato sviluppato da Apple ed è comunemente usato per file video.
- **Formati comuni di caratteri tipografici**: il formato `TTF` è comunemente usato per i caratteri TrueType. Il formato `WOFF` è comunemente usato per i web fonts. Il successore di WOFF è `WOFF2`, che offre una compressione migliore.
- **ZIP**: un formato di file usato per comprimere file e cartelle.

## Navigare efficacemente sul web

- **Cos'è un browser web?**: un'applicazione software che permette agli utenti di consultare e visualizzare siti web su internet.
- **Cos'è un motore di ricerca**: uno strumento che permette agli utenti di cercare informazioni su internet. Esempi includono Google, Bing e Yahoo.
- **Browser web comuni**: alcuni esempi di browser web comuni includono Google Chrome, Mozilla Firefox e Microsoft Edge.
- **Strategie comuni di ricerca**: puoi usare `site:` seguito dall'URL di un sito web per cercare contenuti su quel sito. Puoi usare `filetype:` seguito da un'estensione di file per cercare file di quel tipo. Puoi anteporre un segno meno a un termine di ricerca per escludere risultati che contengono quel termine. Puoi anteporre un segno più a un termine di ricerca per includere risultati che contengono quel termine.
