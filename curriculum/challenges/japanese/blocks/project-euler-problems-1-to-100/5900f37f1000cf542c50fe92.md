# --description--

以下の情報が与えられていますが、ご自身で調べてみるのもよいでしょう。

<ul>
  <li>1900年1月1日は月曜日でした。</li>
  <li>9月は30日、<br>4月、6月、11月も30日です。<br>それ以外はすべて31日、<br>2月だけは例外で、<br>通常は28日、<br>うるう年は29日です。</li>
  <li>うるう年は4で割り切れる年ですが、100で割り切れる年は除き、ただし400で割り切れる年はうるう年です。</li>
</ul>

20世紀（1901年1月1日から2000年12月31日まで）に、月の初めが日曜日だったのは何回ありますか？

```js
function countingSundays(firstYear, lastYear) {
  let sundays = 0;

  for (let year = firstYear; year <= lastYear; year++) {
    for (let month = 0; month <= 11; month++) {
      const thisDate = new Date(year, month, 1);
      if (thisDate.getDay() === 0) {
        sundays++;
      }
    }
  }
  return sundays;
}
```
