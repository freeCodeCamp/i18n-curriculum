# --description--

Em programação, o termo `persistent` frequentemente se refere a algo que continua a existir ou ocorrer, especialmente um problema ou um bug que é difícil de resolver. 

`Persistence` é a forma substantiva, referindo-se ao estado de ser persistente.

Por exemplo, um `persistent error` no código é um erro que continua ocorrendo e é difícil de corrigir.
