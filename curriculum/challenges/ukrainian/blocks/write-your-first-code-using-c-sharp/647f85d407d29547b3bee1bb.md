# --description--

Тепер, коли ви завершили всі завдання «Напишіть свій перший код на C#», ви повинні отримати трофей за це на платформі Microsoft Learn. Якщо ви не бачите цей трофей на <a href="https://learn.microsoft.com/users/me/achievements#trophies-section" target="_blank" rel="noreferrer">сторінці своїх досягнень</a>, це означає, що ви не виконали всі завдання, і ваш трофей не може бути підтверджений.

Пов’яжіть свій обліковий запис Microsoft із обліковим записом freeCodeCamp і натисніть кнопку «Перевірити трофей» нижче, щоб завершити завдання. Цей трофей є **обов’язковим** для проходження сертифікаційного екзамену.
