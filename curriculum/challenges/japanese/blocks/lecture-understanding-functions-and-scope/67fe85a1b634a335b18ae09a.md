# --description--

Pythonでは、スコープは変数にアクセスできる範囲を決定します。これは変数の寿命を制御し、コードの異なる部分でどのように解決されるかを決めるものです。

スコープを正しく決定するために、Pythonは**LEGB**ルールに従います。これは以下の意味です：

- **ローカルスコープ (L)**：関数やクラス内で定義された変数。
    
- **エンクロージングスコープ (E)**：囲んでいる関数やネストされた関数内で定義された変数。
    
- **グローバルスコープ (G)**：モジュールやファイルのトップレベルで定義された変数。
    
- **組み込みスコープ (B)**：Pythonの組み込み関数、モジュール、キーワード、オブジェクトの予約名。

PythonはLEGBルールを使ってプログラム内の変数のスコープを解決します。これらのルールをそれぞれ詳しく見ていきましょう。

**ローカルスコープ**とは、関数やクラス内で宣言された変数は、その関数やクラスの中でのみアクセスできることを意味します。

例を示します：

```python
def my_func():
    my_var = 10
    print(my_var)
```

この場合、`my_func`関数は独自のスコープを持ち、関数の外からはアクセスできません。`my_func`を呼び出すと`10`が出力されますが、関数の外で`my_var`を出力しようとすると`NameError`になります：

```python
def my_func():
    my_var = 10 # Locally scoped to my_func
    print(my_var)

my_func() # 10

print(my_var) # NameError: name 'my_var' is not defined
```

**エンクロージングスコープ**とは、ある関数の中にネストされた関数が、その囲んでいる関数の変数にアクセスできることを意味します。

例えば：

```python
def outer_func():
    msg = 'Hello there!'

    def inner_func():
        print(msg)

    inner_func()

outer_func() # Hello there!
```

この例では、内側の関数`inner_func`は外側の関数`msg`で定義された`outer_func`変数に自由にアクセスできます。ただし、外側の関数はネストされた関数内で定義された変数にアクセスできないことに注意してください：

```python
def outer_func():
    msg = 'Hello there!'
    print(res)

    def inner_func():
        res = 'How are you?'
        print(msg)

    inner_func()

outer_func() # NameError: name 'res' is not defined
```

これは`res`が`inner_func`にローカルスコープであるためです。また、`outer_func`が`res`を呼び出す前に`inner_func`を出力しようとしていることにも注目してください。

一つの解決策は、`res`を囲んでいるスコープ（`outer_func`内）で空の文字列として初期化し、`inner_func`内で`res`を`nonlocal`キーワードで非ローカル変数にすることです：

```python
def outer_func():
    msg = 'Hello there!'
    res = ""  # Declare res in the enclosing scope

    def inner_func():
        nonlocal res  # Allow modification of an enclosing variable
        res = 'How are you?'
        print(msg)  # Accessing msg from outer_func()

    inner_func()
    print(res)  # Now res is accessible and modified

outer_func()

# Output:
# Hello there!
# How are you?
```

**グローバルスコープ**とは、関数やクラスの外で宣言された変数で、プログラムのどこからでもアクセスできるものを指します。ここでは、`my_var`は定義されていない関数内でもアクセス可能です：

```python
my_var = 100

def show_var():
    print(my_var)

show_var() # 100
print(my_var) # 100
```

また、関数内で定義されたローカルスコープの変数をグローバルにアクセス可能にしたい場合は、`global`キーワードを使えます：

```python
my_var_1 = 7

def show_vars():
    global my_var_2
    my_var_2 = 10
    print(my_var_1)
    print(my_var_2)

show_vars() # 7 10

# my_var_2 is now a global variable and can be accessed anywhere in the program
print(my_var_2) # 10
```

グローバル変数を変更したい場合は、`global`キーワードを使うこともできます：

```python
my_var = 10  # A global variable

def change_var():
    global my_var  # Allows modification of a global variable
    my_var = 20

change_var()

print(my_var)  # my_var is now modified globally to 20
```

最後に、**組み込みスコープ**とは、Pythonの組み込み関数、モジュール、キーワードのすべてを指し、プログラムのどこからでも利用可能です：

```python
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```
