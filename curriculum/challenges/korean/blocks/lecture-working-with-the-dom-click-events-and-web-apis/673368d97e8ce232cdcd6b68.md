# --interactive--

DOM에서 노드를 추가하거나 제거해야 할 때가 있습니다. 이를 위해 사용할 수 있는 몇 가지 웹 API가 있습니다.

이번 수업에서는 `appendChild()`와 `removeChild()` 메서드를 다룹니다.

`appendChild()` 메서드는 지정한 부모 노드의 자식 목록 끝에 노드를 추가하는 데 사용됩니다.

다음은 `appendChild()` 메서드의 기본 구문입니다:

```js
parentNode.appendChild(newNode);
```

`appendChild()` 메서드를 어떻게 사용하는지 더 잘 이해하기 위해 예제를 살펴보겠습니다.

이 예제에서는 HTML 안에 ID가 `desserts`인 순서 없는 목록 요소가 있습니다. 이 목록에는 `Cake`와 `Pie` 두 개의 목록 항목이 포함되어 있습니다:

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
```

JavaScript 파일 안에서 이 목록에 접근하려면 `getElementById()` 메서드를 사용할 수 있습니다:

```js
const dessertsList = document.getElementById("desserts");
```

`dessertsList`라는 변수가 `ul` 요소 중 ID가 `desserts`인 요소에 대한 참조를 저장합니다.

그다음 `createElement()` 메서드를 사용해 새 목록 항목 요소를 만들어야 합니다:

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");
```

다음 코드는 새 목록 항목 요소를 만들고 `listItem`라는 변수에 저장합니다.

`ul` 요소의 자식 목록 끝에 이 노드를 추가하려면 `appendChild()` 메서드를 사용할 수 있습니다:

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

dessertsList.appendChild(listItem);
```

이 코드를 실행하면 `ul` 요소의 자식 목록 끝에 새 목록 항목 요소가 추가된 것을 볼 수 있습니다.

:::interactive_editor

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
<script src="index.js"></script>
```

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

dessertsList.appendChild(listItem);
```

:::

문제는 새 목록 항목 요소가 비어 있다는 점입니다. 새 목록 항목 요소에 텍스트 내용을 추가하려면 `textContent` 속성을 사용할 수 있습니다:

:::interactive_editor

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
<script src="index.js"></script>
```

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

listItem.textContent = "Cookies";
dessertsList.appendChild(listItem);
```

:::

이제 목록에는 `Cake`, `Pie`, 그리고 `Cookies`가 표시됩니다.

DOM에서 노드를 제거하려면 `removeChild()` 메서드를 사용할 수 있습니다.

다음은 `removeChild()` 메서드의 기본 구문입니다:

```js
parentNode.removeChild(childNode);
```

이번에는 `section` 요소에서 마지막 단락 요소를 제거하는 예제를 살펴보겠습니다:

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
```

JavaScript 파일 안에서 `section` 요소에 접근하려면 `getElementById()` 메서드를 사용할 수 있습니다:

```js
const sectionEl = document.getElementById("example-section");
```

`section` 요소에 대한 참조를 얻으면 `querySelector()` 메서드를 사용해 마지막 단락에 접근할 수 있습니다:

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");
```

`:last-of-type` 가상 클래스를 사용해 `section` 요소 안의 마지막 단락 요소를 선택하고 있습니다.

이제 부모 노드와 자식 노드를 얻었으니 `section` 요소에서 마지막 단락 요소를 `removeChild()` 메서드로 제거할 수 있습니다:

:::interactive_editor

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
<script src="index.js"></script>
```

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");

sectionEl.removeChild(lastParagraph);
```

:::

그렇다면 언제 DOM에서 노드를 추가하거나 제거해야 할까요?

동적 콘텐츠를 다룰 때 DOM에서 노드를 추가하거나 제거해야 할 수 있습니다. 실제 사례로는 쇼핑 카트, 할 일 목록, 소셜 미디어 피드 등이 있습니다.
