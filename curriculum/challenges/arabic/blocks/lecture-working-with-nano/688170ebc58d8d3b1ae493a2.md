# --description--

عند العمل في الطرفية، لا تتوفر دائمًا محررات النصوص القياسية، لكن هناك برامج مصممة لتحرير الملفات مباشرة من داخل الطرفية.

تتضمن بيئات العمل في لينكس عادة محرر نصوص. Vi أو Vim هما الأكثر شيوعًا. قد تحتاج إلى تثبيت Emacs وNano يدويًا، لكنهما متوفران بسهولة في معظم التوزيعات.

التحدي مع Vim هو أن له منحنى تعلم حاد. إذا لم تكن معتادًا عليه، قد لا تعرف حتى كيف تغلق التطبيق! Emacs أسهل قليلاً في الاستخدام، لكنه يحتوي على مجموعة كبيرة من اختصارات لوحة المفاتيح.

Nano أبسط بكثير وأسهل في الاستخدام، خاصة للمبتدئين. يمكنك البدء بسرعة وتحرير ملف باستخدام Nano، ويقدم التطبيق مساعدة على الشاشة للاختصارات الأساسية.

المقابل لذلك هو نقص قابلية التوسعة. على عكس Vim أو Emacs، التي يمكن توسيعها بإضافات لتصبح بيئات تطوير كاملة، Nano بسيط جدًا.

لنلق نظرة على ما يمكن أن يفعله Nano. لفتح ملف، يمكنك إدخال `nano <filename>` في الطرفية.

بمجرد فتح الملف، سترى إصدار Nano الذي تستخدمه واسم الملف المفتوح في أعلى الشاشة.

أسفل ذلك محتوى الملف. يمكنك استخدام مفاتيح الأسهم لتحريك مؤشر الإدراج وإدخال أو حذف النص باستخدام لوحة المفاتيح. سيتحرك الملف تلقائيًا حسب الحاجة.

ثم، في الأسفل، سترى قائمة بالاختصارات المتاحة على لوحة المفاتيح، التي تبدو كالتالي:

```bash
^G Help   ^O Write Out    ^F Where Is   ^K Cut      ^T Execute  ^C Location     M-U Undo    M-A Set Mark    M-] To Bracket    M-B Previous
^X Exit   ^R Read File    ^\ Replace    ^U Paste    ^J Justify  ^/ Go To Line   M-E Redo    M-6 Copy        ^B Where Was      M-F Next
```

الاختصارات التي تبدأ برمز مؤشر الإدراج (`^`) تعني أنه يجب الضغط مع الاستمرار على مفتاح Ctrl ثم الضغط على الحرف المحدد. الاختصارات التي تبدأ بـ `M-` تعني أنه يجب الضغط مع الاستمرار على "مفتاح الميتا"، والذي في معظم الإعدادات يكون مفتاح Alt، ثم الضغط على الحرف.

لتحرير ملف في Nano، تحتاج فقط إلى معرفة بعض أوامر الاختصارات. بعد إجراء تغييرات على الملف، يمكنك الضغط على `Ctrl + O` لـ "الكتابة"، أي حفظ الملف. بعد إدخال الأمر، سترى القائمة في الأسفل تتغير إلى هذا الشكل:

```bash
File Name to write : <filename>
^G Get Help                 ^T  To Files
^C Cancel                   TAB Complete
```

اضغط Enter لحفظ الملف والعودة إلى التحرير، أو `Ctrl + C` للإلغاء والعودة إلى التحرير.

أثناء التحرير، يمكنك الضغط على `Ctrl + X` للخروج من Nano والعودة إلى الطرفية. إذا كان لديك تغييرات غير محفوظة عند محاولة الخروج، سترى هذا في القائمة السفلية:

```bash
Save modified buffer (ANSWERING "No" WILL DESTROY CHANGES) ?
                            Y Yes
^C Cancel                   N No
```

اضغط `Y` لحفظ التغييرات غير المحفوظة، أو `N` لتجاهلها.

بهذا المعرفة، ستكون مستعدًا بشكل أفضل في المرة القادمة التي تحتاج فيها لتحرير ملف بسرعة على خادم بعيد، حيث قد لا يتوفر لديك وصول إلى بيئة عمل رسومية كاملة.
