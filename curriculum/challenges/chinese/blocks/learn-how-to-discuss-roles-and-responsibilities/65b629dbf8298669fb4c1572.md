# --description--

单词 `everyone` 意味着群体中的每个人。`Every-` 用于逐一谈论群体中的所有事物或人。

重要的是要注意，`everyone` 听起来像是在谈论许多人，但你用它时要搭配单数动词。这类似于其他以 `every-` 开头的词，如 `everything` 和 `everybody`。

例如，你说 `Everyone goes to the party`、`Everything is right` 或 `Everybody works hard`。

`Make sure` 意味着在继续之前查看或确保某事是正确的。例如，`You make sure the code runs` 意味着你查看代码以确定它没有误差并且可以正确运行。

“遵守规则”意味着按照规则所说去做。如果有一条规则说 `use camelCase for variable names`，那么“遵守规则”意味着你给变量命名时使用 `myVariable` 而不是 `myBLANKvariable`。
