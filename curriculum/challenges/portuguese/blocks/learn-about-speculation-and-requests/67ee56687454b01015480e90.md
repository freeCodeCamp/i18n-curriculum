# --explanation--

Um `for loop` é útil quando você sabe quantas vezes o loop deve ser executado.Por exemplo:

- `A for loop is useful for going through a list.` - Você pode usá-lo para verificar cada item em uma lista.

- `The for loop stops when it reaches the last item.` - O loop termina quando não há mais itens.

Um `while loop` é melhor quando o número de repetições depende de uma condição.Por exemplo:

- `A while loop runs until a condition is false.` - O loop continua enquanto a regra for verdadeira.

- `Be careful, a while loop can run forever!` - Se você não pará-lo, o loop não vai terminar.
