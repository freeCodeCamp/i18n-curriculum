# --interactive--

JavaScriptで文字列を扱うとき、より大きな文字列の中にある特定の部分文字列の位置を見つける必要がある場合があります。

部分文字列とは、より大きな文字列の中に現れる文字のシーケンスのことです。例えば、文字列`hello world`の中で、`hello`や`world`は部分文字列です。

文字列の中の部分文字列の位置を見つけるには、`indexOf()`メソッドを使うことができます。
JavaScriptの`indexOf()`メソッドは、文字列の中で部分文字列を検索することができます。

部分文字列が見つかると、`indexOf()`はその部分文字列が最初に現れるインデックス（位置）を返します。部分文字列が見つからない場合は、`indexOf()`が`-1`を返し、検索が失敗したことを示します。

`indexOf()`メソッドは2つの引数を取ります。1つ目は大きな文字列の中で見つけたい部分文字列で、2つ目は検索の開始位置を指定するオプションです。開始位置を指定しなければ、検索は文字列の先頭から始まります。

ここで、**引数**とは、関数やメソッドを呼び出すときに渡す値のことで、その関数やメソッドが特定の情報を使って処理を行うために使われます。引数については今後のレッスンでさらに学びます。

以下は、`indexOf()`メソッドを使って文字列`awesome`の位置を見つける例です。

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("awesome!");
console.log(position); // 14
```

:::

この例では、文字列`awesome`の中で単語`14`はインデックス`JavaScript is awesome!`から始まるので、`indexOf()`メソッドは`14`を返します。

次に、部分文字列が見つからなかった場合を見てみましょう。

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("fantastic");
console.log(position); // -1
```

:::

文字列の中に単語`fantastic`が現れないため、このメソッドは`-1`を返します。

また、`indexOf()`に2つ目の引数を渡すことで、文字列の中のどこから検索を始めるかを指定することもできます。例を見てみましょう。

:::interactive_editor

```js
let sentence = "JavaScript is awesome, and JavaScript is powerful!";
let position = sentence.indexOf("JavaScript", 10);
console.log(position); // 27
```

:::

この場合、`JavaScript`の検索は10文字目以降から始まるため、`JavaScript`の2回目の出現がインデックス`27`で見つかります。

`indexOf()`メソッドは大文字・小文字を区別することに注意してください。

この例では、文字列`-1`の中に大文字の`F`が見つからないため、次のように`freeCodeCamp`が返されます。

:::interactive_editor

```js
console.log("freeCodeCamp".indexOf("F")) // -1
```

:::

`indexOf()`を使うと、文字列の中に部分文字列が存在するかどうかを確認し、その位置を調べてさらに操作を行うのにとても役立ちます。
