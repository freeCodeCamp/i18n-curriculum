# --description--

Después de `responsible for`, se usa el `-ing` formulario de verbos para describir estas tareas.

`Hiring` significa dar a alguien un trabajo. Un ejemplo sencillo podría ser, `The manager is hiring new workers.`

`Taking care of` significa cuidar de o asegurar el bienestar de alguien. Un ejemplo es, `She is taking care of her sick friend.`
