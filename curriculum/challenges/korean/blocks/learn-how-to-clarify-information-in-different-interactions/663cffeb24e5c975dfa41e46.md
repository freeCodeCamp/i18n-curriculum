# --description--

`Double check`는 무언가가 정확한지 다시 확인하는 것을 의미합니다. 예를 들어, `I wanted to be sure I locked the door, so I double checked it`라고 말할 수 있습니다. 이는 실수를 피하기 위해 조심스럽게 자신이 한 일을 다시 확인하고 있다는 뜻입니다.

`schedule`는 일이 언제 일어날지 보여주는 계획입니다. 예를 들어, `school schedule`는 수업이 언제 시작하고 끝나는지 보여줍니다. 이는 하루 중 다른 시간에 무엇을 해야 하는지 알 수 있게 도와줍니다. 프로젝트에서는 특정 부분이 시작하고 끝나는 날짜를 추정하는 데 사용할 수 있습니다.

`Accurate`는 무언가가 정확하고 실수가 없다는 뜻입니다. 예를 들어, 시계가 정확한 시간을 보여주면 `accurate`입니다. 이는 정보가 진실하고 정확하다는 의미입니다.
