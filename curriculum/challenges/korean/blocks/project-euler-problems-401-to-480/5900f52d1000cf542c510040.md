# --description--

제과사 필은 새로운 초콜릿으로 덮인 사탕을 만들고 있습니다. 각 사탕의 중심은 다음 방정식으로 정의되는 회전 타원체 모양입니다: $b^2x^2 + b^2y^2 + a^2z^2 = a^2b^2$.

필은 두께 1밀리미터인 균일한 초콜릿 코팅으로 사탕 중심 하나를 덮는 데 필요한 초콜릿 양을 알고 싶어 합니다.

만약 $a = 1$ mm이고 $b = 1$ mm라면, 필요한 초콜릿 양은 $\frac{28}{3} \pi$ mm<sup>3</sup>입니다.

만약 $a = 2$ mm이고 $b = 1$ mm라면, 필요한 초콜릿 양은 약 60.35475635 mm<sup>3</sup>입니다.

$a = 3$ mm이고 $b = 1$ mm일 때 필요한 초콜릿 양을 mm<sup>3</sup> 단위로 구하세요. 소수점 아래 8자리까지 반올림한 숫자로 답을 제출하세요.
