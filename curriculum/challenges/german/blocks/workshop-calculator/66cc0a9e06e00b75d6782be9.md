# --description--

In den vorherigen Lektionen haben Sie gelernt, wie man eine Funktion aufruft (invoked). Eine Funktion aufzurufen bedeutet, den Code innerhalb der Funktion auszuführen.

Hier eine Erinnerung daran, wie man eine Funktion aufruft:

```js
function myFunction() {
  return "Hello World";
}

// function call
myFunction();
```

Fügen Sie eine `console.log`-Anweisung hinzu, und rufen Sie darin die `addTwoAndSeven`-Funktion auf.

Sie sollten nun die Summe von `2` und `7` in der Konsole sehen.
