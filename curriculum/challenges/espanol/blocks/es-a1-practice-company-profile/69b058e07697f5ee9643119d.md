# --explanation--

Esta tarea refuerza las estructuras en tercera persona del singular usadas para describir una empresa.

`Se llama` se usa para dar nombres. Por ejemplo:

- `La empresa se llama Andes Commerce`. - La empresa se llama Andes Commerce.
- `El proyecto se llama Aurora`. - El proyecto se llama Aurora.

`Es` se usa para describir qué es algo. Por ejemplo:

- `Es una empresa internacional`. - Es una empresa internacional.
- `Es un curso avanzado`. - Es un curso avanzado.

`Está` se usa para indicar ubicación. Por ejemplo:

- `La oficina está en Lima`. - La oficina está en Lima.

`Tiene` expresa posesión o cantidad. Por ejemplo:

- `La empresa tiene veinte empleados`. - La empresa tiene veinte empleados.
- `El equipo tiene cinco miembros`. - El equipo tiene cinco miembros.

Estos términos también son importantes en este contexto:

- `Setenta y tres` - setenta y tres.
- `Sitio web` - sitio web.
