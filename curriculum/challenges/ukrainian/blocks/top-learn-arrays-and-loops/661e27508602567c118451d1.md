# --description--

Рядки та числа можуть бути нашими будівельними блоками, але коли ваші скрипти стають складнішими, вам знадобиться спосіб працювати з великою кількістю таких даних. На щастя, у JavaScript є кілька типів даних, які призначені саме для цього. Масив — це впорядкована колекція елементів (рядків, чисел або інших речей).

Масиви — це спосіб зберігати кілька значень у одній змінній. Вони є особливим типом об’єкта, який має властивість довжина та ряд пронумерованих властивостей. Кожна пронумерована властивість називається елементом, і кожен елемент може зберігати значення будь-якого типу.

Приклад масиву:

```javascript
const fruits = ['apple', 'banana', 'orange'];
```
