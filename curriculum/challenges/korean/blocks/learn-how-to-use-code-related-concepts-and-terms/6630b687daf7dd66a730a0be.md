# --description--

단어 `fundamental`는 집의 기초처럼 기본적이거나 필수적인 무언가를 의미합니다.

예를 들어, `Understanding grammar is fundamental to learning a new language.` 여기서 `fundamental`는 문법이 새로운 언어를 배우는 데 필수적이거나 기본적인 부분임을 강조하는 데 사용됩니다.

구문 `what kind of`는 무언가의 유형이나 성격을 묻는 데 사용됩니다. 이는 무언가를 더 명확하게 지정하거나 분류하는 일반적인 방법입니다.

예를 들어, 누군가 `what kind of music do you like?`라고 물으면 음악 장르에 대해 묻는 것입니다.
