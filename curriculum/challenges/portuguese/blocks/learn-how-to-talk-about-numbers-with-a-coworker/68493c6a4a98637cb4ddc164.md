# --explanation--

Tom diz `in the Backup Licenses section`, o que significa que há uma área ou pasta específica dentro do armazenamento em nuvem onde esses arquivos são salvos.

Aqui, `in` refere-se a estar dentro daquela seção. Outro exemplo:

`The answer is in the FAQ section.` — Você pode encontrar a resposta na parte de FAQ.
