# --description--

フレーズ `why don't` は、提案や申し出をする際によく使われます。これは主語（`we`、`you`、`they` など）と動詞の原形が続きます。構造は `why don't + subject + base verb` です。これは、行動を提案したり、誰かに何かをしてもらうよう丁寧に頼む表現です。

例：

`Why don't we go to the cinema tonight?` 話者が映画に行くことを提案しています。

`Why don't you try restarting the computer?` 話者が解決策としてコンピューターの再起動を提案しています。

フレーズ `check out` は、何かをより詳しく調べたり見ることを意味し、何かを探求したり学んだりすることを提案する際によく使われます。例えば、`Let's check out the new restaurant` はレストランを訪れて試してみることを示唆しています。
