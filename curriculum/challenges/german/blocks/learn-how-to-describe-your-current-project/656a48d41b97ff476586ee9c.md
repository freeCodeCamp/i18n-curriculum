# --description--

In diesem Zusammenhang bedeutet `out`, dass etwas verfügbar oder veröffentlicht ist. Zum Beispiel:

`The new book is out next week.` – Das bedeutet, dass das Buch dann verfügbar sein wird.

`Looking forward to` + ein Verb, das auf `-ing` endet, ist eine gängige Möglichkeit, Begeisterung für ein zukünftiges Ereignis auszudrücken. Zum Beispiel:

- `I'm looking forward to meeting you.` – Sie freuen sich darauf, jemanden zu treffen.

- `She's looking forward to traveling next month.` – Sie freut sich darauf, zu reisen.
