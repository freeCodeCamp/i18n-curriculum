# --description--

Moja ya matatizo makubwa ya kutangaza vigezo kwa kutumia neno muhimu la `var` ni kwamba unaweza kwa urahisi kuandika tena tamko la kigezo:

```js
var camper = "James";
var camper = "David";
console.log(camper);
```

Katika msimbo ulio juu, kigezo cha `camper` kilitangazwa awali kama `James`, kisha kimeandikwa tena kuwa `David`. Konsoli inaonyesha mfuatano wa herufi `David`.

Katika programu ndogo, huenda usakumbane na tatizo hili. Lakini kadri msimbo wako unavyokuwa mkubwa, unaweza kwa bahati mbaya kuandika tena kigezo ambacho hukusudia. Kwa sababu tabia hii haitupa hitilafu, utafuta na kurekebisha hitilafu huwa vigumu zaidi.

Neno muhimu liitwalo `let` lililetwa katika ES6, sasisho kubwa la JavaScript, kutatua tatizo hili linaloweza kutokea kwa neno muhimu la `var`. Utajifunza kuhusu vipengele vingine vya ES6 katika changamoto zijazo.

Ikiwa utabadilisha `var` na `let` katika msimbo ulio juu, itasababisha hitilafu:

```js
let camper = "James";
let camper = "David";
```

Hitilafu hiyo inaweza kuonekana katika konsoli ya kivinjari chako.

Kwa hivyo tofauti na `var`, unapotumia `let`, kigezo chenye jina moja kinaweza kutangazwa mara moja tu.
