# --description--

مصطلح `run code` يشير إلى تنفيذ أو تشغيل برنامج أو نص برمجي بواسطة الكمبيوتر. كلمة `run` هنا تعني بدء العملية ومتابعتها حتى ينفذ البرنامج ما صُمم من أجله، سواء كان ذلك حساب أرقام أو فرز بيانات أو أي مهمة أخرى.

عبارة `run code without managing servers` تعني تنفيذ برامج أو تطبيقات الكمبيوتر دون الحاجة للتعامل مع الخوادم الفعلية أو الافتراضية التي تعمل عليها الشفرة. هذا جانب أساسي في الحوسبة بدون خوادم، حيث يتولى مزود الخدمة السحابية إدارة الخوادم، مما يسمح للمطورين بالتركيز أكثر على كتابة الشفرة بدلاً من إدارة الخوادم.
