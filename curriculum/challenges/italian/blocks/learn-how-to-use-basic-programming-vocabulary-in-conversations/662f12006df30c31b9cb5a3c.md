# --description--

Un `concept` si riferisce generalmente a un'idea o principio usato per pianificare o definire azioni e operazioni. In vari ambiti, inclusa la programmazione, i `concepts` sono le idee o teorie fondamentali.

Per esempio, `Object-oriented programming is a concept in computer programming.`
