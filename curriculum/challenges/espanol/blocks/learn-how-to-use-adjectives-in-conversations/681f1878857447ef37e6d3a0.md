# --explanation--

`Not only ... but also ...` es una estructura común usada para enfatizar que dos afirmaciones relacionadas son verdaderas, siendo la segunda a menudo más sorprendente o importante. Por ejemplo:

`She is not only skilled but also very creative.` – Esto significa que tiene dos cualidades fuertes, y ambas son importantes.

`Adaptable` significa que algo puede cambiar o ajustarse fácilmente dependiendo de la situación. Por ejemplo:

`This program is adaptable to different types of teams.` – Esto significa que funciona bien en muchos tipos de entornos.
