# --description--

Тепер настав час вивчити числа від 70 до 79.

Іспанською числу 70 відповідає `setenta`.

Ці числа утворюються за допомогою слова для десятків `setenta` (70), потім сполучника `y` ("і"), а потім цифри одиниць (1-9).

| Число | Іспанською |
| :--- | :--- |
| 70 | `Setenta` |
| 71 | `Setenta y uno` |
| 72 | `Setenta y dos` |
| 73 | `Setenta y tres` |
| 74 | `Setenta y cuatro` |
| 75 | `Setenta y cinco` |
| 76 | `Setenta y seis` |
| 77 | `Setenta y siete` |
| 78 | `Setenta y ocho` |
| 79 | `Setenta y nueve` |

Число `setenta y uno` (71) у цьому діапазоні унікальне, бо перед іменником чоловічого роду воно стає `setenta y un`. Наприклад:

`Setenta y un años` - сімдесят один рік.
