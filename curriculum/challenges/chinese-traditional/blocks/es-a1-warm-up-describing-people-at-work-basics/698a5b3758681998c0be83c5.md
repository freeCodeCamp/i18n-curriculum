# --description--

有些符號已成為日常工作活動中不可或缺的元素。

以下是西班牙語中最常用的四個符號：

| 符號 | 西班牙語 | 意義 |
| :---: | :--- | :--- |
| `.` | `Punto` | 句點 / 點 |
| `@` | `Arroba` | 小老鼠符號（at 符號） |
| `-` | `Guión` | 連字號 / 破折號 |
| `_` | `Guión bajo` | 底線 |

請注意，`Guión` 這個字的 `o` 字母上有重音符號。

Camila 將協助你練習這些符號。反覆聆聽它們的發音，直到你感到更熟悉，然後將每個符號與其西班牙語名稱配對。
