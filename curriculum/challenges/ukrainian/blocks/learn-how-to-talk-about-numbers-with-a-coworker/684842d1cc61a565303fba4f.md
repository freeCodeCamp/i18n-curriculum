# --explanation--

`Make sure to` використовується, щоб висловити відповідальність або обіцянку, що мовник обов’язково щось зробить. Наприклад:

`Make sure to save your work before closing.` — Не забудь зберегти перед тим, як закрити програму.

`As needed` означає робити щось лише тоді, коли це стає необхідним. Наприклад:

`We can add more team members as needed.` — Вони додадуть людей лише якщо це стане необхідним.

У реченні Аліси вона каже, що регулярно інформуватиме Девіда і змінюватиме бюджетний план, якщо прогрес проєкту цього вимагатиме.
