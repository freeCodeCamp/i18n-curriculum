# --description--

`Store information` 是指將資料保存在特定位置，通常是在變數中。變數就像你程式碼中的容器，用來存放資訊，例如數字、文字或其他類型的資料。你可以將它們視為有名稱的儲存位置，之後可以在程式碼中存取 `stored information`。

例如，`int age = 30;` 在這裡，變數 `age` 儲存數字 30。

字 `like` 用作介系詞，用來舉例或比較相似的事物。

例如，如果你說，`I love fruits like apples and oranges.` 你是在使用 `like` 來舉例你喜歡的水果型別，暗示該列表中可能還有更多水果。
