# --description--

HTMLボイラープレートについて学びましょう。

HTMLボイラープレートとは何か気になりますか？それはウェブページのための既製のテンプレートのようなものです。家の基礎のように考えてください。ボイラープレートには、すべてのHTMLドキュメントに必要な基本構造と重要な要素が含まれています。時間を節約し、ページが正しく設定されていることを確実にします。例を示します：

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>freeCodeCamp</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
  </body>
</html>
```

このボイラープレートの重要な部分を分解してみましょう。まず、`DOCTYPE` 宣言があります：

```html
<!DOCTYPE html>
```

これはブラウザに使用しているHTMLのバージョンを伝えます。次に、`html` タグがあります：

```html
<!DOCTYPE html>
<html lang="en">
  <!--All other elements go inside here-->
</html>
```

これはすべてのコンテンツを包み、ページの言語を指定できます。`html` タグの中には、主に2つのセクションがあります — `head` と `body` です：

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <!--Important metadata goes here-->
  </head>
  <body>
    <!--Headings, paragraphs, images, etc. go inside here-->
  </body>
</html>
```

`head` セクションには重要な裏側の情報が含まれています：

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Document Title Goes Here</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

サイトのメタデータは`meta` 要素に含まれ、文字エンコーディングやTwitterのようなウェブサイトがページのリンクをどのようにプレビューするかなどの詳細があります。サイトのタイトルは`title` 要素にあり、ブラウザのタブやウィンドウに表示されるテキストを決定します。最後に、通常は`head` セクションで`link` 要素を使って外部スタイルシートをリンクします。

`body` セクションはすべてのコンテンツが入る場所です：

```html
<body>
  <h1>I am a main title</h1>
  <p>Example paragraph text</p>
</body>
```

では、なぜボイラープレートが重要なのでしょうか？それはページが正しく構造化され、異なるブラウザ間でうまく動作することを保証します。ボイラープレートを使うことで、よくあるミスを避け、ベストプラクティスに従うことができます。どんなウェブプロジェクトでも素晴らしい出発点です。自分のニーズに合わせてボイラープレートをカスタマイズできることを覚えておいてください。経験を積むにつれて、自分の好みの要素や`meta` タグを追加するかもしれません。個人用ボイラープレートを改善し続けることで、新しいプロジェクトを始める際に時間を節約できることがわかるでしょう。

次に新しいHTMLファイルを始めるときは、ボイラープレートの使用を検討してください。確実にしっかりした基盤を築くことができます。
