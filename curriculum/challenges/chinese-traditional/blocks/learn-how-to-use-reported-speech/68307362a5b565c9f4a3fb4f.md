# --explanation--

`It was mentioned...` 也是一個 **被動語態** 的結構，重點在於所分享的想法，而不是誰說的。 例如：

「提到更新提升了安全性。」－ 這告訴你這個想法，但沒有點出說話者。

結構 `it's` + 比較級形容詞 + `to` + 動詞 用來表示某事物與另一事物相比是如何進行的。例如：

`It's faster to search with the new tool。`－這表示與舊工具相比，使用新工具搜尋時可以節省時間。
