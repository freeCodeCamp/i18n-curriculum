# --description--

在本实验中，你将创建一个函数，该函数返回一个值在数组按升序排序后应插入的最低索引。

**目标**：完成以下用户故事并通过所有测试以完成实验。

**用户故事：**

1. 你应该有一个 `getIndexToIns` 函数，它接受两个参数：一个数组和一个数字。
2. 你应该使用 `sort` 方法将数组按升序排序。
3. 你的 `getIndexToIns` 函数应通过使用 `findIndex` 方法返回数字应插入的最低索引。
4. 你的 `getIndexToIns` 函数应始终返回一个数字。

**提示：**`findIndex` 方法是 JavaScript 中的内置数组方法。它接受一个回调函数并返回第一个满足条件的元素的索引。`findIndex` 和 `sort` 都是高阶函数。

**示例：**

- `getIndexToIns([1, 2, 3, 4], 1.5)` 应该返回 `1`，因为 `1.5` 大于 `1`（索引 `0`）且小于 `2`（索引 `1`）。
- `getIndexToIns([20, 3, 5], 19)` 应该返回 `2`，因为排序后为 `[3, 5, 20]`，`19` 小于 `20`（索引 `2`）且大于 `5`（索引 `1`）。

```js
function getIndexToIns(arr, num) {
  const sorted = [...arr].sort((a, b) => a - b);
  const index = sorted.findIndex(el => el >= num);

  return index === -1 ? sorted.length : index;
}
``` 
