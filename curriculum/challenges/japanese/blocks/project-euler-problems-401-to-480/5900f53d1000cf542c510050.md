# --description--

多角形のカーネルは、多角形の境界全体が見える点の集合として定義されます。極座標多角形とは、原点がそのカーネルの内部に厳密に含まれている多角形のことを指します。

この問題では、多角形は隣接する頂点が共線であってもかまいません。ただし、多角形は自己交差してはならず、面積がゼロであってはなりません。

例えば、以下のうち最初の多角形だけが極座標多角形です（2番目、3番目、4番目のカーネルは原点を厳密に含まず、5番目はカーネルを持ちません）：

<img alt="five example polygons" src="https://cdn.freecodecamp.org/curriculum/project-euler/polar-polygons.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

最初の多角形には3つの連続した共線頂点があることに注目してください。

$P(n)$ を、頂点 $(x, y)$ の座標が絶対値 $n$ 以下の整数である極座標多角形の数とします。

多角形は、囲む面積が同じでも辺の集合が異なれば別の多角形として数えます。例えば、頂点が [(0,0), (0,3), (1,1), (3,0)] の多角形は、頂点が [(0,0), (0,3), (1,1), (3,0), (1,0)] の多角形とは異なります。

例えば、$P(1) = 131$、$P(2) = 1\,648\,531$、$P(3) = 1\,099\,461\,296\,175$、および $P(343)\bmod 1\,000\,000\,007 = 937\,293\,740$ です。

$P(7^{13})\bmod 1\,000\,000\,007$ を求めてください。
