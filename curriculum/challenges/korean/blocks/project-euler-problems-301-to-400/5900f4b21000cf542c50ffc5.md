# --description--

$a_n$을 다음과 같이 재귀적으로 정의된 수열이라고 하겠습니다: $a_1 = 1$, $\displaystyle a_n = \left(\sum_{k = 1}^{n - 1} k \times a_k\right)\bmod n$.

따라서 $a_n$의 처음 10개 원소는 1, 1, 0, 3, 0, 3, 5, 4, 1, 9입니다.

$f(N, M)$이 다음 조건을 만족하는 쌍 $(p, q)$의 개수를 나타낸다고 하겠습니다:

$$ 1 \le p \le q \le N \\; \text{그리고} \\; \left(\sum_{i = p}^q a_i\right)\bmod M = 0$$

$f(10, 10) = 4$임을 알 수 있으며, 해당 쌍은 (3,3), (5,5), (7,9), (9,10)입니다.

또한 $f({10}^4, {10}^3) = 97\\,158$임이 주어졌습니다.

$f({10}^{12}, {10}^6)$를 구하세요.
