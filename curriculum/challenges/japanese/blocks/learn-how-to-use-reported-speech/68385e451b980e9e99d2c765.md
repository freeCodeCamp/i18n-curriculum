# --explanation--

`True` は、何かが正しい、本当である、または正確であることを意味する形容詞です。これは、発言や事実に同意するときに使われます。`the truth` とは異なり、`the truth` は名詞で、何かの実際の事実や現実を指し、より広範囲または抽象的な意味で使われます。例えば：

- `It's true that the server crashed yesterday.` – これは、事故に関する発言が正しいことを意味します。

- `The truth is that we didn't have a backup.` – これは、実際の事実としてバックアップが存在しなかったことを意味します。
