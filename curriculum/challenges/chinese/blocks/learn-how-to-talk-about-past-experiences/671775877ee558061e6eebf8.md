短语 `in real time` 指的是事件发生时立即发生的事情，没有延迟。它通常用于监控、通信或数据处理的上下文中。例如： 

`The system updates the dashboard in real time, so you can see changes instantly.` —— 意思是仪表盘在更新应用后会立即发生变化。
