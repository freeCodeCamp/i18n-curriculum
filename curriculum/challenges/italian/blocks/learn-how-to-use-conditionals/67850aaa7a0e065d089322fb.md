# --explanation--

La frase usa il `Second Conditional`, che è strutturato così:

`If` + `Past Simple` o `Past Continuous` tense, `might / would / could` + verbo base. In questo esempio:

`If they closed` è la condizione (immaginando una chiusura).

`It would affect` è il risultato (ciò che succederebbe di conseguenza).

Questo tipo di condizionale si usa per parlare di situazioni irreali o ipotetiche.
