# --description--

Як усім відомо, рівняння $x^2 = -1$ не має розв’язків для дійсних $x$.

Якщо ж ввести уявне число $i$, це рівняння має два розв’язки: $x = i$ та $x = -i$.

Якщо піти далі, рівняння ${(x - 3)}^2 = -4$ має два комплексні розв’язки: $x = 3 + 2i$ та $x = 3 - 2i$, які називаються комплексно спряженими одне одному.

Числа у формі $a + bi$ називаються комплексними числами.

Загалом $a + bi$ та $a − bi$ є комплексно спряженими одне одному. Гаусове ціле число — це комплексне число $a + bi$, у якому і $a$, і $b$ є цілими числами.

Звичайні цілі числа також є гаусовими цілими (з $b = 0$).

Щоб відрізнити їх від гаусових цілих з $b ≠ 0$, такі цілі числа називають «раціональними цілими».

Гаусове ціле число називається дільником раціонального цілого $n$, якщо результат ділення також є гаусовим цілим.

Наприклад, якщо поділити 5 на $1 + 2i$, можна спростити так:

Помножити чисельник і знаменник на комплексно спряжене до $1 + 2i$: $1 − 2i$.

Результат:

$$\frac{5}{1 + 2i} = \frac{5}{1 + 2i} \frac{1 - 2i}{1 - 2i} = \frac{5(1 - 2i)}{1 - {(2i)}^2} = \frac{5(1 - 2i)}{1 - (-4)} = \frac{5(1 - 2i)}{5} = 1 - 2i$$

Отже, $1 + 2i$ є дільником 5.

Зверніть увагу, що $1 + i$ не є дільником 5, бо:

$$\frac{5}{1 + i} = \frac{5}{2} - \frac{5}{2}i$$

Також зауважте, що якщо гаусове ціле ($a + bi$) є дільником раціонального цілого $n$, то його комплексно спряжене ($a − bi$) також є дільником $n$. Насправді, 5 має шість дільників з додатною дійсною частиною: {1, 1 + 2i, 1 − 2i, 2 + i, 2 − i, 5}.

Нижче наведена таблиця всіх дільників для перших п’яти додатних раціональних цілих:

| n | дільники гаусових цілих з додатною дійсною частиною | сума s(n) цих дільників |
|---|-----------------------------------------------------|-------------------------|
| 1 | 1                                                   | 1                       |
| 2 | 1, 1 + i, 1 - i, 2                                  | 5                       |
| 3 | 1, 3                                                | 4                       |
| 4 | 1, 1 + i, 1 - i, 2, 2 + 2i, 2 - 2i, 4              | 13                      |
| 5 | 1, 1 + 2i, 1 - 2i, 2 + i, 2 - i, 5                  | 12                      |

Для дільників з додатною дійсною частиною маємо: $\displaystyle\sum_{n=1}^5 s(n) = 35$.

Для $1 ≤ n ≤ {10}^5$, $\displaystyle\sum_{n = 1}^{{10}^5} s(n) = 17924657155$.

Яке значення має $\displaystyle\sum_{n=1}^{{10}^8} s(n)$?
