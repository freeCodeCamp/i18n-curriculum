# --description--

Щоб перевірити, чи існує властивість у заданому об’єкті, ви можете використати метод `.hasOwnProperty()`. `someObject.hasOwnProperty(someProperty)` повертає `true` або `false` залежно від того, чи знайдена властивість в об’єкті.

**Приклад**

```js
function checkForProperty(object, property) {
  return object.hasOwnProperty(property);
}

checkForProperty({ top: 'hat', bottom: 'pants' }, 'top'); // true
checkForProperty({ top: 'hat', bottom: 'pants' }, 'middle'); // false
```

Перший виклик функції `checkForProperty` повертає `true`, а другий — `false`.

# --instructions--

Змініть функцію `checkObj`, щоб перевірити, чи містить об’єкт, переданий у параметр функції `obj`, конкретну властивість, передану у параметр функції `checkProp`. Якщо властивість, передана у `checkProp`, знайдена в `obj`, поверніть значення цієї властивості. Якщо ні — поверніть `Not Found`.
