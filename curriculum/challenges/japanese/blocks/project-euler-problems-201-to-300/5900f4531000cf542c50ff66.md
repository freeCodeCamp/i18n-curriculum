# --description--

二項係数 $\displaystyle\binom{10}{3} = 120$ です。

$120 = 2^3 × 3 × 5 = 2 × 2 × 2 × 3 × 5$ で、$2 + 2 + 2 + 3 + 5 = 14$ となります。

したがって、$\displaystyle\binom{10}{3}$ の素因数分解の項の合計は $14$ です。

$\binom{20\\,000\\,000}{15\\,000\\,000}$ の素因数分解の項の合計を求めてください。
