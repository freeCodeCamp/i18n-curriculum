# --description--

Le stringhe e i numeri possono essere i nostri mattoni fondamentali, ma man mano che i tuoi script diventano più complessi, avrai bisogno di un modo per gestire grandi quantità di essi. Per fortuna, JavaScript ha un paio di tipi di dati pensati proprio per questo. Un array è una raccolta ordinata di elementi (stringhe, numeri o altre cose).

Gli array sono un modo per archiviare più valori in una singola variabile. Sono un tipo speciale di oggetto che ha una proprietà lunghezza e una serie di proprietà numerate. Ogni proprietà numerata si chiama elemento, e ogni elemento può contenere un valore di qualsiasi tipo.

Un esempio di array è:

```javascript
const fruits = ['apple', 'banana', 'orange'];
```
