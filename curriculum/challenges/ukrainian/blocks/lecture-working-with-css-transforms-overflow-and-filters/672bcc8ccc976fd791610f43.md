# --interactive--

Властивість CSS `transform` — це потужний інструмент, який дозволяє змінювати візуальне відображення елементів на вашій сторінці, не впливаючи на макет інших елементів. Вона дає змогу застосовувати різні трансформації до елементів, такі як обертання, масштабування, нахил або переміщення (трансляція) у 2D або 3D просторі.

Властивість `transform` працює шляхом застосування математичної трансформації до системи координат елемента. Це означає, що ви можете маніпулювати формою та позицією елемента, зберігаючи його початкове місце та потік документа незмінними.

Давайте розглянемо деякі загальні функції трансформації. Ось приклад елемента-блоку:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
body {
  border: 2px solid black;
}

.box {
  width: 200px;
  height: 200px;
  background-color: red;
}
```

:::

Ми встановили `body` з суцільним чорним кордоном, щоб ви могли бачити `.box` елемент, вкладений у `body` елемент.

Функція `translate` переміщує елемент з його поточного положення. Ось оновлений приклад із використанням функції `translate`:  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
body {
  border: 2px solid black;
}

.box {
  width: 200px;
  height: 200px;
  background-color: red;
  transform: translate(50px, 100px);
}
```

:::

Це CSS-правило перемістить елемент з класом `box` на 50 пікселів вправо і на 100 пікселів вниз від його початкової позиції.

Функція `rotate` обертає елемент навколо фіксованої точки, і це приклад використання функції `rotate` для `.box` елемента з попереднього прикладу:  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: rotate(45deg);
}
```

:::

Це оберне елемент на сорок п’ять градусів за годинниковою стрілкою.

Функція `scale` дозволяє змінювати розмір елемента. Ось приклад:  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: scale(1.5, 2);
}
```

:::

Це зробить елемент у півтора рази ширшим і вдвічі вищим за його початковий розмір.

Ви можете комбінувати кілька трансформацій в одному оголошенні:  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: translate(50px, 50px) rotate(45deg) scale(1.5);
}
```

:::

Це перемістить елемент на 50 пікселів вправо і вниз, оберне його на 45 градусів і масштабуватиме до півтора разів від початкового розміру.

Хоча властивість `transform` є потужним інструментом для створення візуально привабливих дизайнів, важливо враховувати доступність при її використанні. Ось кілька важливих питань доступності, які слід пам’ятати.

Читачі екрана можуть некоректно передавати трансформований вміст. Наприклад, якщо ви використовуєте `transform` для зміни візуального порядку елементів, читачі екрана все одно читатимуть вміст у початковому порядку DOM. Це може спричинити плутанину для користувачів, які покладаються на читачі екрана.

При використанні `scale` для зміни розміру тексту будьте обережні, щоб текст не став занадто маленьким або занадто великим. Надто дрібний текст може бути важким для читання, а надто великий текст може виходити за межі контейнера і ставати нечитаємим. Загалом краще використовувати правильні техніки стилізації шрифту для зміни розміру тексту.

Якщо ви використовуєте `transform` для анімаційних ефектів, враховуйте користувачів, чутливих до руху. Надмірні або швидкі анімації можуть викликати дискомфорт або навіть провокувати судоми у деяких людей. Розгляньте можливість надати користувачам спосіб зменшити або вимкнути анімації. При використанні 3D-трансформацій пам’ятайте, що не всі користувачі однаково сприймають глибину. Переконайтеся, що будь-яка критична інформація, передана через 3D-ефекти, також доступна у 2D-форматі або у вигляді тексту.

Якщо ви використовуєте `transform` для приховування або показу вмісту, переконайтеся, що вміст залишається доступним для читачів екрана та навігації за допомогою клавіатури. Прихований вміст має бути справді прихованим, наприклад, за допомогою `display: none;` або `visibility: hidden;`, а не просто візуально переміщеним за межі екрану.

При застосуванні `transform` до інтерактивних елементів, таких як кнопки або посилання, переконайтеся, що активна область залишається інтуїтивною та легкою для натискання. Сильно трансформована кнопка може бути візуально заплутаною або важкою для натискання, особливо для користувачів з порушеннями моторики.

На завершення, хоча властивість CSS `transform` є потужним інструментом для створення візуально динамічних вебдизайнів, важливо використовувати її відповідально, враховуючи доступність. Завжди тестуйте свої трансформовані елементи з різними допоміжними технологіями і розглядайте можливість надання альтернативних способів доступу до інформації або функціональності, які можуть бути вплинуті трансформаціями.
