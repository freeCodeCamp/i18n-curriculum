# --explanation--

Повне речення — `Gracias por venir`. Цей вираз має структуру:

`Gracias por` + інфінітив дієслова

Він використовується, щоб подякувати за якусь дію.

- `Por` вводить причину подяки.

- `Venir` описує дію прибуття в місце.

У цьому випадку Ángela дякує людині за те, що вона прийшла.
