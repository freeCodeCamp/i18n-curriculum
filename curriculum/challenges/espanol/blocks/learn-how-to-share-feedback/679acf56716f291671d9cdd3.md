# --explanation--

`Too much + noun` se usa para indicar una cantidad excesiva de algo, lo cual a menudo se considera negativo. Por ejemplo:

`Too much noise in the office makes it hard to concentrate.` - Esto significa que una cantidad excesiva de ruido afecta negativamente la concentración.
