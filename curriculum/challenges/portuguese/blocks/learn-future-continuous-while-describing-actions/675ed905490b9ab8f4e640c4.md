# --explanation--

`Public` significa algo que está disponível para todos ou compartilhado pela comunidade.Por exemplo: 

`The repository is public so anyone can access it.` - Isso mostra que o repositório está aberto para todos e não é privado.

`Wi-Fi` (abreviação de `Wireless Fidelity`) refere-se a uma conexão de internet sem fio que permite que dispositivos se conectem à internet.Por exemplo:

`I connected my phone to the café's Wi-Fi to check my emails.` - Isso mostra que você acessou a internet no café sem usar um cabo.
