# --description--

Manchmal müssen Sie ein Zeichen (oder eine Gruppe von Zeichen) finden, das ein- oder mehrmals hintereinander vorkommt. Das bedeutet, es erscheint mindestens einmal und kann wiederholt werden.

Sie können das `+` Zeichen verwenden, um zu prüfen, ob das der Fall ist. Denken Sie daran, dass das Zeichen oder Muster aufeinanderfolgend vorhanden sein muss. Das heißt, das Zeichen muss direkt hintereinander wiederholt werden.

Zum Beispiel würde `/a+/g` eine Übereinstimmung in `abc` finden und `["a"]` zurückgeben. Wegen des `+` würde es auch eine einzelne Übereinstimmung in `aabc` finden und `["aa"]` zurückgeben.

Wenn stattdessen der String `abab` geprüft würde, fände es zwei Übereinstimmungen und gäbe `["a", "a"]` zurück, weil die `a` Zeichen nicht hintereinander stehen – dazwischen befindet sich ein `b`. Schließlich, da im String `a` kein `bcd` vorkommt, würde keine Übereinstimmung gefunden.

# --instructions--

Sie möchten Übereinstimmungen finden, wenn der Buchstabe `s` ein- oder mehrmals in `Mississippi` vorkommt. Schreiben Sie einen Regex, der das `+` Zeichen verwendet.

# --hints--

Ihr Regex `myRegex` sollte das `+` Zeichen verwenden, um ein oder mehrere `s` Zeichen zu finden.

```js
assert(/\+/.test(myRegex.source));
```

Ihr Regex `myRegex` sollte 2 Treffer finden.

```js
assert(result.length == 2);
```

Die Variable `result` sollte ein Array mit zwei Übereinstimmungen von `ss` sein.

```js
assert(result[0] == 'ss' && result[1] == 'ss');
```
