# --explanation--

`Of course` طريقة مهذبة لإظهار الموافقة أو قول `yes` بدون تردد. يعني أن المتحدث سعيد أو مستعد لفعل شيء ما. على سبيل المثال:

`Of course, I'll help you with the report.` – هذا يعني أن المتحدث سعيد بالمساعدة.

`Will` يُستخدم غالبًا لتقديم وعود أو التعبير عن قرار في لحظة الكلام. على سبيل المثال:

`I'll email you the file this afternoon.` – هذا يُظهر وعدًا بفعل شيء قريبًا.
