# --description--

Sasa unapaswa kuona kadi moja tu ikionyeshwa kwenye ukurasa. Lakini kuna sehemu moja iliyokosekana ndani ya sehemu ya `Card`.

Kama unavyokumbuka kutoka kwa mafundisho ya awali, ni mbinu bora kujumuisha kigezo cha `key` wakati unaonyesha orodha ya sehemu kama hii:

```jsx
<div>
  {fruits.map(fruit => (
    <Card
      key={fruit.id}
      name={fruit.name}
      color={fruit.color}
    />
  ))}
</div>
```

Kigezo cha `key` husaidia React kutambua ni vitu gani vimebadilika, vimeongezwa, au vimeondolewa. Hii huongeza utendaji na kuruhusu masasisho yenye ufanisi zaidi.

Sahihisha sehemu ya `Card` ili ijumuishe kigezo cha `key` chenye thamani ya `profile.id`.

# --hints--

Unapaswa kuongeza kigezo cha `key` kwenye sehemu ya `Card`. Thamani inapaswa kuwa `profile.id`.

```js
assert.match(code, /<Card[^>]*key=\{\s*profile\.id\s*\}[^>]*\/?>/);
```
