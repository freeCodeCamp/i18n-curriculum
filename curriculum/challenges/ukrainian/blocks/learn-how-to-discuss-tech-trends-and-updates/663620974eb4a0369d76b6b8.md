# --description--

Фраза `some of these` використовується для позначення конкретної частини групи або підмножини елементів із більшої множини. Її часто застосовують, коли хочуть згадати кілька речей із групи, не перераховуючи кожну окремо.

Наприклад, `Let's try some of these cookies.` Доповідач пропонує спробувати кілька печив із більшої партії.
