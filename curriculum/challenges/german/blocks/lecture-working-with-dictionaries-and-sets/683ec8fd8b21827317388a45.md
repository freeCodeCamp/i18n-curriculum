# --description--

Sets sind eine der eingebauten Datenstrukturen von Python. Eine der Kernmerkmale von Sets ist, dass sie keine doppelten Werte speichern. Wenn Sie versuchen, einen doppelten Wert zu einem Set hinzuzufügen, wird nur einer davon gespeichert.

Sets sind veränderbar und ungeordnet, was bedeutet, dass ihre Elemente nicht in einer bestimmten Reihenfolge gespeichert werden, sodass Sie nicht mit Indizes oder Schlüsseln auf sie zugreifen können. Sie können nur Werte von unveränderlichen Datentypen wie Zahlen, Strings und Tuples enthalten. Außerdem unterstützen sie mathematische Mengenoperationen, einschließlich Vereinigung, Schnittmenge, Differenz und symmetrische Differenz.

Um ein Set zu definieren, müssen Sie nur seine Elemente in geschweifte Klammern schreiben und sie durch Kommas trennen. Dies ist ein Beispiel für ein Set von Zahlen:

```python
my_set = {1, 2, 3, 4, 5} 
```

Eine Besonderheit bei der Arbeit mit Sets ist, dass Sie, wenn Sie jemals ein leeres Set definieren müssen, die `set()`-Funktion verwenden müssen. Wenn Sie nur leere geschweifte Klammern schreiben, wie `{}`, erstellt Python automatisch ein Dictionary.

```python
set() # Set
{}    # Dictionary
```

Sie können ein Element mit der `.add()`-Methode zu einem Set hinzufügen und das neue Element als Argument übergeben:

```python
my_set.add(6)
```

In unserem Beispiel wäre das neue Set:

```python
{1, 2, 3, 4, 5, 6}
```

Wenn Sie versuchen, ein Element hinzuzufügen, das bereits im Set ist, wird nur eines behalten. In diesem Fall haben wir die Zahl 5 bereits im Set:

```python
my_set.add(5)
```

Also wird sich das Set nicht ändern:

```python
{1, 2, 3, 4, 5, 6}
```

Um ein Element aus dem Set zu entfernen, haben Sie zwei Möglichkeiten. Sie können entweder die `.remove()`-Methode oder die `.discard()`-Methode verwenden und das Element, das Sie entfernen möchten, als Argument übergeben.

Die `.remove()`-Methode löst eine `KeyError` aus, wenn das Element nicht gefunden wird, während die `.discard()`-Methode das nicht tut:

```python
my_set.remove(4)
my_set.discard(4)
```

Die `.clear()`-Methode entfernt alle Elemente aus dem Set:

```python
my_set.clear()
```

Python-Sets haben auch leistungsstarke Methoden, die gängige mathematische Mengenoperationen ausführen.

Die `.issubset()`- und die `.issuperset()`-Methoden prüfen, ob ein Set eine Teilmenge oder Obermenge eines anderen Sets ist.

Hier prüfen wir, ob `your_set` eine Teilmenge von `my_set` ist, was `False` ist, weil nicht alle Elemente von `your_set` in `my_set` sind.

Wir prüfen auch, ob `my_set` eine Obermenge von `your_set` ist. Das ist ebenfalls `False`, weil `my_set` nicht alle Elemente von `your_set` hat:

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

print(your_set.issubset(my_set)) # False
print(my_set.issuperset(your_set)) # False
```

Die `.isdisjoint()`-Methode prüft, ob zwei Sets disjunkt sind, das heißt, sie haben keine gemeinsamen Elemente. In diesem Fall ist das `False`, weil `my_set` und `your_set` gemeinsame Elemente haben – 2, 3 und 4:

```python
print(my_set.isdisjoint(your_set)) # False
```

Der Vereinigungsoperator `|` gibt ein neues Set mit allen Elementen aus beiden Sets zurück:

```python
my_set | your_set # {1, 2, 3, 4, 5, 6}
```

Der Schnittmengenoperator `&` gibt ein neues Set mit nur den Elementen zurück, die beide Sets gemeinsam haben:

```python
my_set & your_set # {2, 3, 4}
```

Der Differenzoperator `-` gibt ein neues Set mit den Elementen des ersten Sets zurück, die nicht in den anderen Sets sind. In diesem Beispiel sind die Zahlen 1 und 5 in `my_set`, aber NICHT in `your_set`:

```python
my_set - your_set # {1, 5}
```

Der symmetrische Differenzoperator `^` gibt ein neues Set mit den Elementen zurück, die entweder im ersten oder im zweiten Set sind, aber nicht in beiden. In diesem Fall sind 1 und 5 in `my_set`, aber nicht in `your_set`, also sind sie enthalten. Und die Zahl 6 ist in `your_set`, aber nicht in `my_set`, also ist sie ebenfalls enthalten:

```python
my_set ^ your_set # {1, 5, 6}
```

Jeder dieser Operatoren hat auch seinen entsprechenden zusammengesetzten Zuweisungsoperator, wenn Sie das Gleichheitszeichen daneben setzen. Diese Operatoren weisen das resultierende Set automatisch dem ersten Set im Ausdruck zu:

```python
|= &= -= ^=
```

Zum Beispiel findet der `-=`-Operator die Differenz zwischen den Sets und aktualisiert das erste Set mit diesem Ergebnis:

```python
my_set -= your_set
```

Danach wird `my_set` auf `{1, 5}` aktualisiert:

```python
print(my_set) # {1, 5}
```

Sie können mit dem `in`-Operator prüfen, ob ein Element in einem Set ist oder nicht. Hier prüfen wir, ob 5 in `my_set` ist. Das Ergebnis ist ein Boolean-Wert `True` oder `False`:

```python
print(5 in my_set)
```

Und das sind die Grundlagen von Sets. Sie sind sehr hilfreich, wenn Sie die Werte nicht in einer bestimmten Reihenfolge speichern müssen und wenn Sie nur eindeutige Werte speichern möchten.
