# --description--

Eine Nullstelle oder Wurzel eines Polynoms $P(x)$ ist eine Lösung der Gleichung $P(x) = 0$.

Definieren Sie $P_n$ als das Polynom, dessen Koeffizienten die Ziffern von $n$ sind.

Zum Beispiel gilt $P_{5703}(x) = 5x^3 + 7x^2 + 3$.

Wir sehen, dass:

- $P_n(0)$ die letzte Ziffer von $n$ ist,
- $P_n(1)$ die Summe der Ziffern von $n$ ist,
- $P_n(10)$ gleich $n$ selbst ist.

Definieren Sie $Z(k)$ als die Anzahl der positiven ganzen Zahlen $n$, die $k$ nicht überschreiten, für die das Polynom $P_n$ mindestens eine ganzzahlige Nullstelle hat.

Es kann überprüft werden, dass $Z(100\\,000)$ gleich 14696 ist.

Wie groß ist $Z({10}^{16})$?
