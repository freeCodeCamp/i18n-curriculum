# --description--

في الدروس القليلة الماضية، أصبحت معتادًا على العمل مع الحلقات مثل هذه:

```py
even_numbers = []

for num in range(21):
    if num % 2 == 0:
        even_numbers.append(num)

print(even_numbers)
```

ينشئ هذا المثال قائمة جديدة فارغة تُسمى `even_numbers` ويدور عبر متتالية أرقام بين `0` و `20`. داخل الحلقة، هناك شرط يتحقق مما إذا كان العدد الحالي له باقي `0` عند قسمته على `2`. يُستخدم هذا لتحديد ما إذا كان العدد زوجيًا. إذا كان الشرط `True`، تُضاف القيمة الحالية `num` إلى نهاية قائمة `even_numbers`. أخيرًا، نطبع قائمة `even_numbers` في وحدة التحكم.

بينما تعمل هذه الشفرة، هناك طريقة أكثر اختصارًا لكتابتها باستخدام تراكيب القوائم. تراكيب القوائم تتيح لك إنشاء قائمة جديدة في سطر واحد بدمج حلقة وشرط مباشرة داخل أقواس مربعة. هذا يجعل الشفرة أقصر وغالبًا أسهل في القراءة.

إليك المثال المعاد صياغته من قبل باستخدام الأقواس المربعة:

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

في هذا المثال المعاد صياغته، تُنشأ قائمة `even_numbers` باستخدام سطر واحد من الشفرة. تراكيب القوائم تدور عبر الأرقام من `0` إلى `20`، وتشمل فقط الأرقام القابلة للقسمة على `2`. هذا الأسلوب أكثر اختصارًا ويُلغي الحاجة إلى حلقة منفصلة وكتلة شرطية.

لنلقِ نظرة على مثال آخر لنفهم بشكل أفضل كيف تعمل تراكيب القوائم:

```py
numbers = [1, 2, 3, 4, 5]
result = [(num, 'Even') if num % 2 == 0 else (num, 'Odd') for num in numbers]
print(result)
```

في هذا المثال، لدينا قائمة أرقام ونريد إنشاء قائمة جديدة من صفوف مرتبة تشير إلى ما إذا كانت الأرقام زوجية أو فردية. في الجزء الأول من تركيب القائمة، نستخدم جملة `if` للتحقق مما إذا كان العدد يقبل القسمة على `2` بدون باقي. إذا كان الأمر كذلك، تكون النتيجة صف مرتب يحتوي على ذلك العدد متبوعًا بالكلمة `Even`. وإلا، تكون النتيجة صف مرتب يحتوي العدد متبوعًا بالكلمة `Odd`.

إليك كيف تبدو النتيجة عند طباعتها في وحدة التحكم:

```py
[(1, 'Odd'), (2, 'Even'), (3, 'Odd'), (4, 'Even'), (5, 'Odd')]
```

طريقة أخرى لإنشاء قائمة بدءًا من قابل للتكرار موجود هي الدالة `filter()`. إليك مثالًا لإنشاء قائمة جديدة تحتوي فقط على الكلمات التي يزيد طولها عن أربعة أحرف:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

تُستخدم الدالة `filter()` لاختيار عناصر من قابل للتكرار تحقق شرطًا معينًا. تقبل الدالة `filter()` دالة وقابل للتكرار كمعاملات. في هذا المثال، نمرر دالة `is_long_word` إلى `filter()` للتحقق مما إذا كان عدد أحرف الكلمة الحالية أكبر من `4`. تُضاف كل الكلمات التي يزيد عدد أحرفها عن `4` إلى قائمة جديدة وتُعيَّن إلى المتغير `long_words`.

بعيدًا عن الدالة `filter()`، هناك بعض الدوال الأخرى المفيدة عند العمل مع القوائم. دالة أخرى يجب معرفتها هي `map()`، التي تأخذ قابل للتكرار وتطبق دالة على كل عنصر منه. إليك مثالًا لاستخدام دالة `map()` لتحويل قائمة درجات الحرارة من سلسيوس إلى فهرنهايت:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

تمامًا مثل الدالة `filter()`، تقبل `map()` دالة وقابل للتكرار كمعاملات. تأخذ الدالة `to_fahrenheit` درجة حرارة وتحولها من سلسيوس إلى فهرنهايت.

الدالة الأخيرة التي سننظر إليها هي `sum()`. تُستخدم هذه الدالة للحصول على مجموع عناصر قابل للتكرار مثل قائمة أو صف مرتب. إليك مثالًا لاستخدام دالة `sum()`:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

يمكنك أيضًا تمرير معلمة اختيارية `start` التي تحدد القيمة الابتدائية للجمع. إليك مثالًا محدثًا يستخدم المعلمة `start` كمعلمة موضعية:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

يمكنك أيضًا اختيار استخدام المعلمة `start` كمعلمة مسماة هكذا بدلاً من ذلك:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

كلا النسختين ستنتجان نفس النتيجة، لكن المعلمة المسماة أكثر وضوحًا قليلاً.

قد تبدو تراكيب القوائم بالإضافة إلى دوال مثل `map()` و `filter()` و `sum()` مربكة قليلًا في البداية. لكن مع الممارسة والوقت الكافيين، ستشعر براحة أكبر عند استخدامها في برامج Python الخاصة بك.
