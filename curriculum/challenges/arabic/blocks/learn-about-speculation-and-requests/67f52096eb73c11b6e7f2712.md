# --explanation--

`Physical access` يعني الحصول على دخول إلى مكان، مثل مكتب أو غرفة خادم.

يحدث `physical access breach` عندما يدخل شخص ما موقعًا آمنًا بدون إذن. على سبيل المثال:

- `To avoid a physical access breach, we should require ID checks at the entrance.` - يمكن للشركة منع الدخول غير المصرح به بفحص الهويات عند الباب.

- `A physical access breach can happen if someone enters the server room without permission.` - يمكن أن تحدث مشكلة أمنية فعلية إذا دخل شخص غرفة الخادم بدون إذن.
