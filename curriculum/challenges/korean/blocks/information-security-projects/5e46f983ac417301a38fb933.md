# --instructions--

비밀번호는 절대 평문으로 저장해서는 안 됩니다. 비밀번호 목록이 발견될 경우를 대비해 해시로 저장해야 합니다. 하지만 모든 해시가 동일한 것은 아닙니다.

이번 프로젝트에서는 SHA-1으로 해시된 비밀번호를 알아내는 비밀번호 크래커를 만들어 좋은 보안의 중요성을 배우게 됩니다.

SHA-1 해시된 비밀번호를 입력받아, 상위 10,000개 비밀번호 중 하나라면 원래 비밀번호를 반환하는 함수를 만드세요. 만약 데이터베이스에 없는 비밀번호라면 "PASSWORD NOT IN DATABASE"를 반환하세요.

함수는 `top-10000-passwords.txt`에 있는 각 비밀번호를 해시한 뒤, 함수에 전달된 해시와 비교해야 합니다.

함수는 선택적 두 번째 인수 `use_salts`를 받을 수 있습니다. 이 값이 true로 설정되면, `known-salts.txt` 파일에 있는 각 솔트 문자열을 `top-10000-passwords.txt`의 각 비밀번호 앞뒤에 붙인 후 해시하고, 함수에 전달된 해시와 비교해야 합니다.

다음은 함수 테스트용 해시된 비밀번호입니다:

- `b305921a3723cd5d70a375cd21a61e60aabb84ec`는 "sammy123"을 반환해야 합니다
- `c7ab388a5ebefbf4d550652f1eb4d833e5316e3e`는 "abacab"을 반환해야 합니다
- `5baa61e4c9b93f3f0682250b6cf8331b7ee68fd8`는 "password"를 반환해야 합니다

`use_salts`가 `True`로 설정되었을 때 테스트할 해시된 비밀번호는 다음과 같습니다:

- `53d8b3dc9d39f0184144674e310185e41a87ffd5`는 "superman"을 반환해야 합니다
- `da5a4e8cf89539e66097acd2f8af128acae2f8ae`는 "q1w2e3r4t5"를 반환해야 합니다
- `ea3f62d498e3b98557f9f9cd0d905028b3b019e1`는 "bubbles1"을 반환해야 합니다

`hashlib` 라이브러리가 이미 가져와져 있습니다. 코드를 작성할 때 사용하는 것을 고려하세요. <a href="https://docs.python.org/3/library/hashlib.html" target="_blank" rel="noopener noreferrer nofollow">"hashlib"에 대해 더 알아보세요</a>.
