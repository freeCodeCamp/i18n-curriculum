# --description--

Considera grafos construidos con las unidades A: <img alt="graph unit A" src="https://cdn.freecodecamp.org/curriculum/project-euler/coloured-configurations-1.png" style="display: inline-block; background-color: white; padding: 10px;">
 y B: <img  alt="graph unit B" src="https://cdn.freecodecamp.org/curriculum/project-euler/coloured-configurations-2.png" style="display: inline-block; background-color: white; padding: 10px;">, donde las unidades están pegadas a lo largo de los bordes verticales como en el grafo <img alt="graph with four units glued along the vertical edges" src="https://cdn.freecodecamp.org/curriculum/project-euler/coloured-configurations-3.png" style="display: inline-block; background-color: white; padding: 10px;">.

Una configuración de tipo $(a,b,c)$ es un grafo construido así con $a$ unidades A y $b$ unidades B, donde los vértices del grafo están coloreados usando hasta $c$ colores, de modo que no haya dos vértices adyacentes con el mismo color. El grafo compuesto arriba es un ejemplo de una configuración de tipo $(2,2,6)$, de hecho de tipo $(2,2,c)$ para todo $c ≥ 4$

Sea $N(a,b,c)$ el número de configuraciones del tipo $(a,b,c)$. Por ejemplo, $N(1,0,3) = 24$, $N(0,2,4) = 92928$ y $N(2,2,3) = 20736$.

Encuentra los últimos 8 dígitos de $N(25,75,1984)$.
