## خاصية تجاوز السعة في CSS

- **التعريف**: يشير تجاوز السعة إلى الطريقة التي تتعامل بها العناصر مع المحتوى الذي يتجاوز، أو "يتجاوز السعة"، حجم العنصر الحاوي. تجاوز السعة ثنائي الأبعاد.
- **`overflow-x`**: المحور السيني يحدد تجاوز السعة الأفقي.
- **`overflow-y`**: المحور الصادي يحدد تجاوز السعة العمودي.
- **`overflow`**: خاصية مختصرة لـ `overflow-x` و `overflow-y`. إذا أعطيت قيمة واحدة، يستخدمها كلا التجاوزين. إذا أعطيت قيمتين، يستخدم `overflow-x` الأولى، و`overflow-y` الثانية.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="overflow-box">
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
</div>
```

```css
.overflow-box {
  width: 140px;
  height: 70px;
  border: 2px solid black;
  overflow-x: auto;
  overflow-y: auto;
  white-space:nowrap;
}
```

## خاصية التحويل في CSS

- **التعريف**: تتيح لك هذه الخاصية تطبيق تحولات مختلفة على العناصر، مثل التدوير، التكبير، الإمالة، أو الترجمة (التحريك) في فضاء ثنائي أو ثلاثي الأبعاد.
- **دالة `translate()`**: تُستخدم هذه الدالة لتحريك عنصر من موقعه الحالي.
- **دالة `scale()`**: تتيح لك هذه الدالة تغيير حجم العنصر.
- **دالة `rotate()`**: تتيح لك هذه الدالة تدوير العنصر.
- **دالة `skew()`**: تتيح لك هذه الدالة إمالة العنصر.
- **التحولات وإمكانية الوصول**: إذا كنت تستخدم التحويل لإخفاء أو إظهار المحتوى، تأكد من أن المحتوى لا يزال متاحًا لقارئ الشاشة والتنقل عبر لوحة المفاتيح. يجب أن يكون المحتوى المخفي مخفيًا فعليًا، مثل استخدام `display: none` أو `visibility: hidden`، وليس مجرد تحريكه بصريًا خارج الشاشة.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="transform-box translate">Translate</div>
  <div class="transform-box scale">Scale</div>
  <div class="transform-box rotate">Rotate</div>
  <div class="transform-box skew">Skew</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: column;
  align-items: center;
  gap: 16px;
  flex-wrap: wrap;
}

.transform-box {
  width: 100px;
  height: 100px;
  background: coral;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

/* translate() */
.translate {
  transform: translateX(20px);
}

/* scale() */
.scale {
  transform: scale(1.2);
}

/* rotate() */
.rotate {
  transform: rotate(15deg);
}

/* skew() */
.skew {
  transform: skew(10deg, 5deg);
}
```

## نموذج الصندوق

- **التعريف**: في نموذج الصندوق في CSS، كل عنصر محاط بصندوق. ويتكون هذا الصندوق من أربعة مكوّنات: منطقة المحتوى و`padding` و`border` و`margin`.
- **منطقة المحتوى**: منطقة المحتوى هي الجزء الأعمق من الصندوق. وهي المساحة التي تحتوي على المحتوى الفعلي للعنصر، مثل النص أو الصور.
- **`padding`**: الحشو هو المنطقة التي تلي منطقة المحتوى مباشرةً. وهو المساحة بين منطقة المحتوى وحدّ العنصر.
- **`border`**: الحدّ هو الحافة الخارجية أو الخط الخارجي للعنصر في نموذج الصندوق في CSS. وهو الحدّ المرئي للعنصر.
- **`margin`**: الهامش هو المساحة خارج حدّ العنصر. وهو يحدد المسافة بين عنصر والعناصر الأخرى المحيطة به.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="box-model">Content</div>
```

```css
.box-model {
  width: 120px;
  background: lightblue;
  padding: 16px;
  border: 4px solid teal;
  margin: 20px;
  text-align: center;
}
```

## تداخل الهوامش

- **التعريف**: يحدث هذا السلوك عندما تتداخل الهوامش الرأسية للعناصر المتجاورة، مما يؤدي إلى هامش واحد يساوي الأكبر بينهما. ينطبق هذا السلوك فقط على الهوامش الرأسية (الأعلى والأسفل)، وليس الهوامش الأفقية (اليسار واليمين).

## قيم خاصيتي `content-box` و `border-box`

- **خاصية `box-sizing`**: تُستخدم هذه الخاصية لتحديد كيفية حساب القيم النهائية لـ `width` و`height` لعنصر HTML.
- **قيمة `content-box`**: في نموذج `content-box`، تحدد قيمتا `width` و`height` التي تعيّنها لعنصر أبعاد مساحة المحتوى فقط، ولا تشمل `padding` أو `border` أو `margin`.
- **قيمة `border-box`**: مع `border-box`، تشمل قيمتا `width` و`height` للعنصر مساحة المحتوى و`padding` و`border`، لكنها لا تشمل `margin`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="sizing-box content-box">Content-box</div>
<div class="sizing-box border-box">Border-box</div>
```

```css
.sizing-box {
  width: 150px;
  padding: 10px;
  border: 4px solid teal;
  margin-bottom: 10px;
  background: peachpuff;
  text-align: center;
}

.content-box {
  box-sizing: content-box;  
}

.border-box {
  box-sizing: border-box;
  background: lightgreen;
}
```

## إعادة تعيين CSS

- **التعريف**: إعادة تعيين CSS هي ورقة أنماط تزيل كل أو بعض التنسيقات الافتراضية التي يطبقها متصفح الويب على عناصر HTML. تشمل الخيارات الخارجية لإعادة التعيين `sanitize.css` و `normalize.css`.

## خاصية فلتر CSS

- **التعريف**: يمكن استخدام هذه الخاصية لإنشاء تأثيرات مختلفة مثل التمويه، وتحويل الألوان، وتعديل التباين.
- **دالة `blur()`**: تطبق هذه الدالة تمويهًا غاوسيًا على العنصر. تُحدد الكمية بالبكسل وتمثل نصف قطر التمويه.
- **دالة `brightness()`**: تعدل هذه الدالة سطوع العنصر. القيمة 0% تجعل العنصر أسود تمامًا، والقيم فوق 100% تزيد السطوع.
- **دالة `contrast()`**: تعدل هذه الدالة تباين العنصر. القيمة 0% تجعل العنصر رماديًا تمامًا، 100% تظهر العنصر بشكل طبيعي، والقيم فوق 100% تزيد التباين.
- **دالة `grayscale()`**: تحول هذه الدالة العنصر إلى تدرج رمادي. تُحدد الكمية كنسبة مئوية، حيث 100% تعني تدرج رمادي كامل و0% تترك الصورة دون تغيير.
- **دالة `sepia()`**: تطبق هذه الدالة لون سيبيا على العنصر. تستخدم نسبة مئوية مثل التدرج الرمادي.
- **دالة `hue-rotate()`**: تطبق هذه الدالة تدويرًا للون على العنصر. تُحدد القيمة بالدرجات وتمثل تدويرًا حول دائرة الألوان.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <img class="filter blur" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Blur" />
  <img class="filter brightness" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Brightness" />
  <img class="filter contrast" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Contrast" />
  <img class="filter grayscale" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Grayscale" />
  <img class="filter sepia" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Sepia" />
  <img class="filter hue" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Hue Rotate" />
</div>
```

```css
.container {
  display: flex;
  gap: 12px;
  flex-wrap: wrap;
}

.filter {
  width: 120px;
}

/* blur() */
.blur {
  filter: blur(3px);
}

/* brightness() */
.brightness {
  filter: brightness(130%);
}

/* contrast() */
.contrast {
  filter: contrast(150%);
}

/* grayscale() */
.grayscale {
  filter: grayscale(100%);
}

/* sepia() */
.sepia {
  filter: sepia(100%);
}

/* hue-rotate() */
.hue {
  filter: hue-rotate(90deg);
}
```
