# --interactive--

CSS 리셋은 웹 브라우저가 HTML 요소에 기본적으로 적용하는 모든 또는 일부 기본 서식을 제거하는 스타일시트입니다. 예를 들어, 단락과 제목 같은 특정 요소들이 사용자 정의 스타일시트에서 명시적으로 정의하지 않아도 기본적으로 여백이 있다는 것을 눈치챘을 수 있습니다.

이것은 다양한 HTML 요소의 여러 속성에서도 볼 수 있습니다. 서로 다른 브라우저는 HTML 요소에 서로 다른 기본 스타일을 적용할 수도 있습니다. Google Chrome의 기본 스타일이 Mozilla Firefox나 Microsoft Edge와 정확히 같지 않을 수 있습니다. 이는 브라우저 간에 일관되지 않은 스타일을 초래할 수 있으며, 가능한 한 피해야 합니다.

이를 해결하기 위해 CSS 리셋은 기본 스타일을 제거합니다. 모든 또는 일부 기본 스타일을 제거함으로써 디자인의 일관된 기준선을 얻고 브라우저와 기기 간의 잠재적인 불일치를 최소화할 수 있습니다. 기본 스타일을 제거하면 사용자 정의 스타일시트에서 명시적으로 적용한 스타일만 보이기 때문에 스타일링 과정도 더 쉬워집니다. CSS 리셋에는 두 가지 주요 접근법이 있습니다: 직접 맞춤 CSS 리셋을 정의하거나 서드파티 CSS 리셋을 사용하는 방법입니다.

맞춤 CSS 리셋은 프로젝트의 필요에 맞게 처음부터 직접 만드는 스타일시트입니다. 이렇게 하면 리셋할 특정 스타일을 유연하게 많이 조절할 수 있습니다. 하지만 스타일시트를 개발하고 유지하는 데 시간 투자가 필요합니다. 모든 HTML 요소의 여백과 패딩을 리셋하는 매우 흔한 CSS 규칙 예시는 다음과 같습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Example Heading</h1>
<p>This is a paragraph.</p>
```

```css
* {
  margin: 0;
  padding: 0;
}
```

:::

보통 CSS 스타일시트 상단에 작성합니다. 별표 선택자는 모든 HTML 요소에 일치하는 와일드카드 선택자이므로, 모든 요소의 기본 여백과 네 면의 기본 패딩이 0이 됩니다. 이렇게 시작점을 잡고 스타일시트 아래쪽에서 더 구체적인 CSS 선택자를 사용해 원하는 대로 조정할 수 있습니다.

이 방법을 사용해 어떤 HTML 요소든 선택하여 기본 스타일을 리셋할 수 있습니다. 맞춤 스타일시트를 만들고 적절한 CSS 선택자를 사용해 요소를 선택하고 스타일을 설정하세요.

하지만 이 과정은 시간이 많이 걸릴 수 있습니다. 시간을 절약하고 싶다면 서드파티 CSS 리셋을 사용할 수도 있습니다. 이 스타일시트들은 이미 미리 만들어져 있어 다운로드해 프로젝트에 바로 추가할 수 있습니다. 서드파티 CSS 리셋의 훌륭한 예시는 `Normalize.css`입니다. 이 스타일시트는 광범위한 HTML 요소의 스타일을 표준화하면서도 특히 접근성에 중요한 유용한 기본 스타일을 유지합니다. 또한 흔한 버그와 스타일 불일치를 수정합니다.

또 다른 옵션은 `sanitize.css`입니다. 이 CSS 라이브러리는 모든 주요 최신 브라우저에서 기본 스타일이 일관되도록 보장하는 데 사용할 수 있습니다. 이 라이브러리는 `Normalize.css`와 함께 개발되어 함께 진화합니다. 또한 폼과 타이포그래피 표준화 같은 특정 목적을 위한 개별 스타일시트도 다운로드할 수 있습니다.

선택지는 많지만 프로젝트의 필요에 가장 잘 맞는 것을 선택하세요. 서드파티 CSS 리셋과 맞춤 리셋을 함께 사용하는 것도 가능합니다.

CSS 리셋을 사용할 때는 접근성도 고려하는 것이 중요합니다. 웹 애플리케이션은 모두가 접근할 수 있어야 합니다. 따라서 화면 읽기 프로그램이나 기타 보조 기술에 도움이 될 수 있는 스타일은 리셋하지 않아야 합니다. 또한 이러한 추가 스타일시트가 사용자 정의 스타일 적용 전에 다운로드되어야 하므로 애플리케이션 성능에 미칠 영향도 고려해야 합니다.

기본 스타일을 제거함으로써 CSS 리셋은 디자인을 구현할 수 있는 빈 시작점을 제공합니다. 이는 브라우저와 기기 전반에 걸쳐 보다 균일하고 일관된 사용자 경험을 만듭니다.
