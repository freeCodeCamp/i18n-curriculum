# --description--

`Gracefully` significa manejar problemas de manera suave, eficiente y no abrupta. Cuando un programa maneja errores `gracefully`, gestiona problemas sin fallar ni causar principales interrupciones. Por ejemplo, en lugar de detenerse repentinamente, un programa podría mostrar un mensaje de error útil.

Recuerda, `encounter` significa encontrarse con algo inesperadamente, y una `sentencia condicional` es un código que se ejecuta solo si una cierta condición es verdadera.
