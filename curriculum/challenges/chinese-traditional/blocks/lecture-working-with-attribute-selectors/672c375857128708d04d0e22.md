# --interactive--

在處理 HTML 中的有序列表時，`type` 屬性允許你指定使用的編號樣式，例如數字、字母或羅馬數字。 

可以在 `ol`（有序列表）元素上設定 `type` 屬性以控制編號格式：

- `1` 用於數字列表（例如 1、2、3、...），

- `A` 用於大寫字母列表（例如 A、B、C、...），

- `a` 用於小寫字母列表（例如 a、b、c、...），

- `I` 用於大寫羅馬數字（例如 I、II、III、...），

- `i` 代表小寫羅馬數字（例如 i、ii、iii、...）。

以下是如何使用屬性選擇器來標的具有特定 `type` 屬性的有序列表的方法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="A"] {
  color: purple;
  font-weight: bold;
}
```

:::

在此範例中，CSS 規則將套用紫色文字和粗體字型字重到任何使用大寫英文字母編號且由 `type="A"` 指定的有序列表。

同樣地，你可以透過選取帶有 `type="i"` 的項目，來標的使用小寫羅馬數字的有序列表：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="i">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="i"] {
  color: green;
}
```

:::

此規則將會將使用小寫羅馬數字的有序列表的文字顏色更改為綠色。

根據 `type` 屬性為有序列表設定樣式的能力，讓你對文件中列表的外觀擁有更大的控制權，並可根據內容的結構進行訂製。
