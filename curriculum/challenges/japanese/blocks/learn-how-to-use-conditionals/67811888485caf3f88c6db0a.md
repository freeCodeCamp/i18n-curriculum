# --explanation--

- `At the end of each day, the system runs a data backup.` - システムはデータ損失を防ぐために毎晩すべてのデータを保存します。

- `We ensure the servers are updated at the end of each day.` - サーバーには業務時間中の中断を避けるために毎晩アップデートが適用されます。
