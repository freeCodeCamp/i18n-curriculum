# --explanation--

`Specify` は、何かをはっきり述べたり詳しく説明したりすることを意味します。例えば：

`Please specify the time and place of the meeting.` – これは、会議がいつどこで行われるかを正確に言うことを意味します。

`What makes it` は名詞節です。文中で名詞のように働く語のかたまりで、何かの理由や原因を説明します。例えば：

`She described what makes it so effective.` – これは、彼女が何かが効果的である理由を説明したことを意味します。ここで節 `what makes it so effective` は動詞 `described` の目的語として機能しています。
