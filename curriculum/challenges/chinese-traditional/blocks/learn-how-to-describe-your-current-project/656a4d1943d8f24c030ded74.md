# --description--

當有人在否定句的結尾使用 `either`，表示相同的情況也適用於他們。 例如： 

我不喜歡胡蘿蔔。我妹妹也不喜歡它們。— 這裡，`either` 意味著說話者和他們的妹妹都不喜歡胡蘿蔔。
