# --interactive--

У попередніх модулях ви звикли працювати з літералами рядків ось так:

```js
const greeting = "Hello, World!";
```

Але JavaScript також має об’єкти рядків. І об’єкти рядків, і примітивні рядки використовуються для роботи з текстом, але вони працюють по-різному «під капотом». Об’єкт рядка створюється за допомогою функції конструктора рядка, яка обгортає примітивне значення в об’єкт. Ось як ви створюєте об’єкт рядка:

:::interactive_editor

```js
const greetingObject = new String("Hello, World!");

console.log(typeof greetingObject); // "object"
```

:::

Коли ми використовуємо оператор `typeof`, ми бачимо, що результат має тип object замість типу string. Одне з питань, яке могло вас цікавити, — як можна використовувати властивості, наприклад властивість `.length`, на примітивних рядках.

Коли ви використовуєте властивість `length` на примітивному рядку, JavaScript тимчасово обгортає примітивний рядок в об’єкт рядка, щоб виконати операцію. Саме тому ви можете використовувати властивість `length` і різні методи, такі як `repeat()`, `concat()` і `slice()`. Ці типи методів і властивостей називають методами екземпляра, властивостями екземпляра та статичними методами. Ви детальніше дізнаєтеся, як це працює, у майбутніх модулях.

Головна відмінність між об’єктом рядка і примітивним рядком полягає в тому, як вони пов’язані з пам’яттю та продуктивністю. Примітивні рядки зазвичай більш ефективні за пам’яттю і швидші порівняно з об’єктами рядків. Хоча ви переважно працюватимете з примітивними рядками у своєму коді, корисно розуміти, як працюють об’єкти рядків.
