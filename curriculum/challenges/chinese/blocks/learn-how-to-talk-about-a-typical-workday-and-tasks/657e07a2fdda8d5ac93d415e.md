# --description--

你可以使用像 `then` 这样的词来描述事情发生的顺序。 

`Then` 通常用于表示在完成某事之后的下一步或动作。例如：

`First, I check my emails, then I start coding.`——这意味着此人在开始编写代码之前会查看他们的电子邮件。

这有助于人们理解一系列动作或事件在时间上的连接方式。
