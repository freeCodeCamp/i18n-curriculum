# --description--

多角形とは、直線の線分がつながって閉じた鎖または回路を形成する平面図形です。多角形は少なくとも3辺を持ち、自己交差しません。

正の数の集合 $S$ が多角形 $P$ を生成すると言うのは、次の条件を満たす場合です：

- $P$ の辺の長さが同じものは2つありません、
- $P$ のすべての辺の長さは $S$ に含まれています、
- $S$ に他の値は含まれていません。

例えば：

集合 {3, 4, 5} は辺の長さが3、4、5の多角形（三角形）を生成します。

集合 {6, 9, 11, 24} は辺の長さが6、9、11、24の多角形（四角形）を生成します。

集合 {1, 2, 3} と {2, 3, 4, 9} はどちらも多角形を生成しません。

次のように定義されるシーケンス $s$ を考えます：

- $s_1 = 1$, $s_2 = 2$, $s_3 = 3$
- $n > 3$ のとき $s_n = s_{n - 1} + s_{n - 3}$

$U_n$ を集合 $\\{s_1, s_2, \ldots, s_n\\}$ とします。例えば、$U_{10} = \\{1, 2, 3, 4, 6, 9, 13, 19, 28, 41\\}$ です。

$f(n)$ を、$U_n$ の部分集合のうち少なくとも1つの多角形を生成するものの数とします。

例えば、$f(5) = 7$、$f(10) = 501$、$f(25) = 18\\,635\\,853$ です。

$f({10}^{18})$ の下9桁を求めてください。
