# --description--

**Vervollständigen Sie den folgenden Satz basierend auf dieser Geschichte:**

Julieta stellt sich am ersten Arbeitstag im Büro vor.

Sebastian fragt: „Wie alt sind Sie?“

Julieta antwortet, dass sie sechsunddreißig Jahre alt ist.

Wie kann Julieta auf diese Frage auf Spanisch antworten?

# --fillInTheBlank--

## --sentence--

`Tengo BLANK años.`

## --blanks--

`treinta y seis`

### --feedback--

Dies ist die Zahl 36 auf Spanisch.

# --explanation--

Basierend auf der Geschichte ist Julieta sechsunddreißig Jahre alt, was `treinta y seis años` (36) auf Spanisch entspricht.
