# --description--

In diesem Workshop überprüfen Sie die Arbeit mit den verschiedenen `Math` Objektmethoden, indem Sie einen Mathbot erstellen. Dieser Mathbot wird einige mathematische Operationen und Nachrichten in der Konsole protokollieren, um Ihnen das `Math` Objekt von JavaScript näherzubringen.

Beginnen Sie für diesen ersten Schritt damit, eine Variable namens `botName` zu erstellen und ihr den String-Wert `"MathBot"` zuzuweisen.

Erstellen Sie dann eine weitere Variable namens `greeting` und weisen Sie ihr den Satz `"Hi there! My name is [botName goes here] and I am here to teach you about the Math object!"` zu. Anstelle des `[botName goes here]` Platzhalters verwenden Sie die Variable `botName`.

Sie können Template-Literale oder String-Verkettung mit dem `+` Operator verwenden, um dies zu erreichen.

Zum Schluss protokollieren Sie die Variable `greeting` in der Konsole.

