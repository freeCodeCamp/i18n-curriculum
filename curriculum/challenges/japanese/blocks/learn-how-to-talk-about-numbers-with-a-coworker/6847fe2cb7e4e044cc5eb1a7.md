# --explanation--

⅗ を言葉で正しく書く方法は `three-fifths` です。次の形式を使います：

分子（基数）＋分母（分子が1より大きい場合は複数形）。例えば：

- ⅖ = `two-fifths`

- ⅘ = `four-fifths`

これは、データを説明したり、口頭で財務の詳細を共有したりするときに役立ちます。
