# --explanation--

`First Conditional` 用來談論未來真實或可能發生的情況及其可能的結果。它遵循的結構是條件使用 `if` + `Present Simple` 時態，結果使用 `will` 或 `can` + 動詞原形。 例如：

- `If you study hard, you will pass the exam.`－這描述了一個可能的情況及其結果。

- `If it rains tomorrow, we can cancel the picnic.`－這說明如果發生某個條件（下雨），可能會發生的情況。
