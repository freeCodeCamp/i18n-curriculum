# --interactive--

Масив у JavaScript — це впорядкована колекція значень, кожне з яких ідентифікується числовим індексом. Значення в масиві JavaScript можуть бути різних типів даних, включно з числами, рядками, булевими значеннями, об’єктами та навіть іншими масивами.

Щоб створити масив у JavaScript, ви можете використати квадратні дужки, `[]`, і розділити значення комами. Ось приклад:

```js
let fruits = ["apple", "banana", "orange"];
```

У цьому прикладі ми оголошуємо змінну `fruits` і призначаємо їй масив, що містить три рядкові значення: `apple`, `banana` та `orange`.

Однією з ключових характеристик масивів є те, що вони мають нульовий індекс, тобто перший елемент масиву має індекс `0`, другий елемент має індекс `1` і так далі. Ви можете отримати доступ до окремих елементів масиву, використовуючи їхній індекс. Наприклад:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits[0]); // "apple"
console.log(fruits[2]); // "orange"
```

:::

У цьому прикладі ми використовуємо індекс `0`, щоб отримати доступ до першого елемента (`apple`) і індекс `2`, щоб отримати доступ до третього елемента (`orange`).

Масиви в JavaScript мають спеціальну властивість `length`, яка повертає кількість елементів у масиві. Ви можете отримати доступ до цієї властивості, використовуючи ключове слово `length`. Наприклад:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits.length); // 3
```

:::

Ще одна ключова характеристика масивів у JavaScript — це їхня динамічність, тобто їхній розмір може змінюватися після створення. Ви можете додавати або видаляти елементи з масиву, використовуючи різні методи масивів, такі як `push()`, `pop()`, `shift()`, `unshift()`, `splice()` та інші. Ці методи будуть вивчені в наступних уроках.

Масиви JavaScript є універсальними та корисними для зберігання даних у ваших програмах. Протягом цього модуля ви на власному досвіді побачите, як робота з масивами допоможе вам ефективно керувати та маніпулювати колекціями даних.
