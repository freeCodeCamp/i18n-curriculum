# --explanation--

`Support` significa manejar o permitir que algo funcione correctamente. Por ejemplo:  

`The app can support 1,000 users at the same time.` – Puede manejar esa cantidad sin fallar.

`At once` significa hacer varias cosas al mismo tiempo. Por ejemplo:

`The software can open many files at once.` – Todos los archivos se abren juntos, no uno por uno.
