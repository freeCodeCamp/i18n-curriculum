# --explanation--

`Could have` + `Past Participle` si usa per parlare di qualcosa che era possibile ma non è avvenuto. Per esempio:

- `You could have added a comment to explain this function.` - Sarebbe stato utile se la persona avesse aggiunto un commento per spiegare cosa fa la funzione.

- `We could have added a search bar to make the website more user-friendly.` - Il team stava pensando di aggiungere una barra di ricerca, ma non era ancora stata fatta.
