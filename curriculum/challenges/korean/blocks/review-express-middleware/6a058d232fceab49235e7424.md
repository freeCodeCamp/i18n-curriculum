## 미들웨어란 무엇인가요?

- **미들웨어**는 서버에 대한 요청 수명 주기 동안 실행되는 함수입니다.
- 미들웨어는 **요청 객체**(`req`), **응답 객체**(`res`), 그리고 **`next` 함수**에 접근할 수 있습니다.
- 일반적인 용도: 요청 기록, 오류 처리, 들어오는 데이터 파싱, 정적 파일 제공 등이 있습니다.
- 미들웨어는 앱에 추가된 순서대로 실행됩니다.

### 미들웨어 작동 방식

1. 요청이 서버로 전송됩니다.
2. 미들웨어 함수들이 요청을 처리합니다.
3. 미들웨어가 `next()`를 호출하면 제어가 다음 미들웨어나 라우트 핸들러로 넘어갑니다.
4. 미들웨어가 `next()`를 호출하지 않으면 요청-응답 주기가 정지됩니다.

```javascript
app.use((req, res, next) => {
  console.log('Request URL:', req.url);
  console.log('Request Method:', req.method);
  next();
});
```

- `app.use()`는 애플리케이션에 미들웨어를 전역으로 추가합니다.

## 애플리케이션 레벨 미들웨어

- **애플리케이션 레벨 미들웨어**는 Express 앱 인스턴스에 바인딩됩니다.
- `app.use()` 또는 `app.get()`, `app.post()` 같은 라우트별 메서드를 사용해 등록합니다.
- 특정 라우트로 제한하지 않는 한 모든 들어오는 요청에 대해 실행됩니다.
- 사용 사례: 여러 라우트에 걸친 요청 기록, 인증, 오류 처리 등입니다.

```javascript
app.use((req, res, next) => {
  console.log('Request received at:', new Date());
  next();
});
```

## 라우터 레벨 미들웨어

- **라우터 레벨 미들웨어**는 메인 앱 대신 `express.Router()` 인스턴스에 바인딩됩니다.
- `router.use()`을 사용해 적용하며, 해당 라우터가 처리하는 라우트에만 영향을 줍니다.
- 앱을 독립적인 라우트 모듈로 모듈화하는 데 도움이 됩니다.
- `app.use('/path', router)`으로 라우터를 마운트해 미들웨어를 특정 라우트에 범위 지정하세요.

```javascript
const router = express.Router();

router.use((req, res, next) => {
  console.log('Request made to /menu route');
  next();
});

router.get('/drinks', (req, res) => {
  res.send('Welcome to the drinks menu!');
});

app.use('/menu', router);
```

## 오류 처리 미들웨어

- **오류 처리 미들웨어**는 요청-응답 주기 중 발생한 오류를 잡기 위해 사용하는 특별한 미들웨어입니다.
- 네 개의 매개변수 `err`, `req`, `res`, `next`를 가집니다. 이 네 매개변수 서명이 Express에 오류 핸들러임을 알립니다.
- 오류가 `next(err)`로 전달될 때 호출됩니다.
- 모든 다른 미들웨어와 라우트 뒤, 미들웨어 스택의 **마지막에** 배치해야 합니다.

```javascript
app.use((err, req, res, next) => {
  console.error(err.stack);
  res.status(500).send('Something went wrong!');
});
```

## 내장 및 서드파티 미들웨어

### 내장 미들웨어

- **`express.json()`**: JSON 페이로드가 포함된 들어오는 요청을 파싱합니다.

```javascript
app.use(express.json());
```

- **`express.static()`**: 이미지, CSS, JavaScript 같은 정적 파일을 제공합니다.

```javascript
app.use(express.static('public'));
```

- **`express.urlencoded()`**: URL 인코딩된 페이로드(HTML 폼에서 전송하는 형식)를 파싱하며, 파싱된 데이터를 `req.body`에서 사용할 수 있게 합니다.

```javascript
app.use(express.urlencoded({ extended: true }));
```

### 서드파티 미들웨어

- **`cors`**: 교차 출처 리소스 공유(CORS)를 활성화합니다.

```javascript
app.use(cors());
```

- **`morgan`**: HTTP 요청을 기록합니다.

```javascript
app.use(morgan('tiny'));
```

- 내장 미들웨어와 서드파티 미들웨어 함수 모두 `app.use()`로 호출된 순서대로 추가됩니다.
