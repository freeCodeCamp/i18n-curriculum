# --description--

`Computing` は、情報を管理し、問題を解決し、さまざまな作業を行うためにコンピューターとコンピューター技術を使用するプロセスや活動を指します。これは、単純な計算から複雑なプログラミングやデータ分析まで、幅広い活動を意味します。

現代の用法では、`computing` は `cloud computing`、`serverless computing` などのトピックも含み、技術の進化する性質を反映しています。
