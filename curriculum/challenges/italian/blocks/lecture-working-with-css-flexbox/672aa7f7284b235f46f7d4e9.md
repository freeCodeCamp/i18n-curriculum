# --interactive--

CSS flexbox è un modello di layout monodimensionale che ti permette di disporre gli elementi in righe e colonne all’interno di un contenitore. Puoi anche controllarne l’ordine e l’orientamento. Gli sviluppatori web lo usano per creare siti web e applicazioni responsivi che si adattano a diverse dimensioni e orientamenti dello schermo. Lo chiamiamo modello di layout monodimensionale perché si concentra sull’organizzazione degli elementi lungo un solo asse alla volta. L’asse può essere orizzontale o verticale.

Ci sono due concetti chiave che dovresti conoscere prima di iniziare a lavorare con flexbox: contenitore flex ed elemento flex.

Un contenitore flex è un elemento HTML con un layout flex. Puoi disporre e allineare gli elementi in vari modi all’interno di un contenitore flex. Per rendere un elemento HTML un contenitore flex, devi aggiungere `display: flex` ai suoi stili CSS.

Gli elementi flex sono i figli diretti di un contenitore flex. Questi elementi possono essere disposti e allineati all’interno di un contenitore flex in base alle sue proprietà. Possono anche restringersi o espandersi per adattarsi allo spazio disponibile.

Ecco un esempio con un contenitore `main` e tre elementi figli `div`:

```html
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

Se imposti solo `width`, `height` e `background-color` di questi elementi `div` nel foglio di stile CSS, ogni elemento figlio sarà posizionato su una riga propria perché il contenitore non è flex per impostazione predefinita.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Ma se aggiungi `display: flex` al contenitore `main`, gli elementi `div` verranno riorganizzati per stare sulla stessa riga e si restringeranno se necessario (per vedere le anteprime, abilita l’editor interattivo):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Per impostazione predefinita, un contenitore flex sarà un elemento di tipo blocco, quindi il contenitore stesso sarà su una riga propria rispetto ad altri elementi e contenitori.

Ora che conosci meglio contenitori flex ed elementi flex, dovresti anche conoscere le proprietà flex. Queste proprietà determinano come gli elementi flex saranno disposti, ridimensionati e distribuiti all’interno del contenitore flex. Alcune delle più comuni sono `flex-direction`, `justify-content`, `align-items` e `flex-wrap`.

Ottimo. Ora parliamo un po’ del modello flex. Questo modello definisce come gli elementi flex sono disposti all’interno di un contenitore flex. Ogni contenitore flex ha due assi:

- L’asse principale.
- L’asse trasversale.

L’orientamento di questi assi determina come le diverse proprietà influenzeranno il layout e la distribuzione degli elementi flex. Per impostazione predefinita, l’asse principale di un contenitore flex è orizzontale e l’asse trasversale è verticale. Gli elementi flex sono disposti nella direzione dell’asse principale. L’asse trasversale è perpendicolare all’asse principale.

La proprietà `flex-direction` imposta la direzione dell’asse principale. Il valore predefinito di `flex-direction` è `row`, che posiziona tutti gli elementi flex sulla stessa riga, nella direzione della lingua predefinita del tuo browser (da sinistra a destra o da destra a sinistra):

```css
flex-direction: row; /* Default */
```

Per invertire gli elementi nella riga, puoi usare `flex-direction: row-reverse`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: row-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Questo invertirà l’ordine degli elementi flex. Se vuoi invece allineare gli elementi flex verticalmente, devi solo impostare `flex-direction` su `column` nel contenitore flex. Questo cambierà la direzione dell’asse principale:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Ora gli elementi `div` saranno allineati verticalmente perché l’asse principale sarà verticale e l’asse trasversale orizzontale.

Puoi anche invertire l’ordine degli elementi flex verticalmente con `flex-direction: column-reverse`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

CSS flexbox è un modello di layout potente che offre un modo flessibile ed efficiente per disporre gli elementi all’interno di un contenitore. Capendo i concetti chiave di contenitori flex, elementi flex e le varie proprietà flex, puoi creare siti web dinamici e responsivi che si adattano a diverse dimensioni e orientamenti dello schermo.
