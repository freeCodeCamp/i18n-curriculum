# --explanation--

`To touch on` може означати коротко згадати або торкнутися теми. Це часто використовується, коли розглядають предмет без детального занурення. Наприклад:

- `The speaker touched on the new software update during the meeting.` - Доповідач коротко згадав про оновлення програмного забезпечення, але не заглиблювався в деталі.
