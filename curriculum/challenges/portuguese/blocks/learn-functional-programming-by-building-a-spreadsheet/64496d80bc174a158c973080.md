# --description--

Verifique se `length` é par usando sua função `isEven`.Se for o caso, retorne a média do número no índice `middle` e o número depois desse.Se for ímpar, retorne o número no índice `middle` – você precisará arredondar o valor de `middle` para cima.

# --hints--

Você deve retornar um valor da função `median`. 

```js
assert.exists(median([2,4,6,8]));
```

Se `length` for par, você deve retornar a média do número no índice `middle` e do número seguinte a ele. 

```js
assert.strictEqual(median([2,4,6,8]),5); 
assert.strictEqual(median([6,12,18,24]),15); 
```


Se `length` for ímpar, você deve retornar o valor no índice `middle`. 

```js
assert.strictEqual(median([2,4,6,8,10]),6); 
assert.strictEqual(median([3,6,9,12,15]),9); 
```
