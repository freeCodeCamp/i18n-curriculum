# --description--

列表推導式也接受 `if` 語句來從可迭代物件中篩選條款：

```py
nums = [1, 2, 3, 4, 5, 6]
even_nums = [num for num in nums if num % 2 == 0]
print(even_nums) # [2, 4, 6]
```

由於你想傳回只包含無效鍵的列表，請為你的推導式添加一個 `if` 語句，使得只有當 `value` 為假值時，才將每個 `key` 添加到列表中。
