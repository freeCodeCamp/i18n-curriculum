# --description--

定義 $f(n)$ 為數字 $n$ 各位數字階乘的總和。 例如，$f(342) = 3! + 4! + 2! = 32$。

定義 $sf(n)$ 為 $f(n)$ 的各位數字之和。所以 $sf(342) = 3 + 2 = 5$。

定義 $g(i)$ 為最小的正整數 $n$，使得 $sf(n) = i$。雖然 $sf(342)$ 是 5，$sf(25)$ 也是 5，且可以驗證 $g(5)$ 是 25。

定義 $sg(i)$ 為 $g(i)$ 各位數字的總和。所以 $sg(5) = 2 + 5 = 7$。

此外，可以驗證 $g(20)$ 是 267，且 $\sum sg(i)$ 對於 $1 ≤ i ≤ 20$ 是 156。

對於 $1 ≤ i ≤ 150$，$\sum sg(i)$ 是多少？
