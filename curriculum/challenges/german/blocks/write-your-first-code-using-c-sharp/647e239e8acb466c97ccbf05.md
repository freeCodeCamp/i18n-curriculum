## --text--

Was ist der Unterschied zwischen `Console.Write` und `Console.WriteLine`?

## --answers--

`Console.Write` gibt die Ausgabe in einer neuen Zeile aus.

---

`Console.WriteLine` gibt die Ausgabe in einer neuen Zeile aus.

---

`Console.WriteLine` fügt nach der Ausgabe eine neue Zeile an.
