# --explanation--

`Simple Past`では、質問は質問詞（例：`what`、`where`、`how`、`who`）で始まり、次の構造を使います：

**質問詞 + `did` + 主語 + 動詞の原形。**

リサの質問、`What did you find there?`は、ジェームズの行動の結果に関する具体的な情報を尋ねるために`what`を使っています。その他の例は以下の通りです：

- `Where did you go yesterday?` - 場所について尋ねる場合。

- `How did they solve the problem?` - 方法について尋ねる場合。

- `Who did you call last night?` - 人について尋ねる場合。

この対話では、リサはバグを検索中にジェームズが発見したことについて尋ねています。
