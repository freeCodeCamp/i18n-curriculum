# --explanation--

`Something` puede usarse en preguntas cuando el hablante asume que una cosa existe.

En `Is it something serious?`, James asume que hay un problema, porque María le dijo que el cliente se quejó de ello, y pregunta si es grave. En cambio, `anything` se usa en preguntas cuando el hablante no sabe si algo siquiera existe. Por ejemplo:

- `Is there something wrong with the software?` - Aquí, el hablante asume que hay un problema y pregunta para confirmar cuál es.

- `Is there anything wrong with the software?` - Aquí, el hablante solo tiene curiosidad por saber si hay algún problema. No está buscando una confirmación.
