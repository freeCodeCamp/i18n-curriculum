# --explanation--

- `Every company should have an incident response plan for cyberattacks.` - 企業はセキュリティ問題を処理する計画が必要です。

- `Without an incident response plan, it's hard to recover from a data breach.` - 計画がなければ、セキュリティ問題の修正は困難になる場合があります。
