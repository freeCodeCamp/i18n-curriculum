# --description--

`class` 是一个基本的概念，特别是在 `object-oriented programming` 中。它就像创建对象的蓝图。一个 `class` 定义了从该 `class` 创建的对象可以拥有的属性（attributes）和行为（`methods`）。

例如，`Person` 类可能有像 `name` 和 `age` 这样的属性，以及像 `talk()` 和 `walk()` 这样的行为。

`method` 是关联于对象的函数。`Methods` 定义了从 `class` 创建的对象可以执行的行为或动作。它们就像告诉对象该做什么的指令。

例如，在一个 `Calculator class` 中，`method` 可能是 `add(number1, number2)`，它将两个数字相加。
