# --description--

어떤 작업을 하느냐에 따라 숫자보다 텍스트 조각을 더 많이 다루게 될 수도 있습니다. **문자열**은 텍스트 조각이며… 언어의 기본 구성 요소입니다.

HTML은 텍스트에 구조와 의미를 제공하고, CSS는 텍스트를 정밀하게 스타일링할 수 있게 하며, JavaScript는 문자열을 조작하는 다양한 기능을 제공합니다. 여기에는 맞춤 환영 메시지와 프롬프트를 만들고, 필요할 때 올바른 텍스트 레이블을 표시하며, 용어를 원하는 순서로 정렬하는 등 여러 작업이 포함됩니다.

문자열은 JavaScript의 기본 데이터 유형입니다. 텍스트를 나타내는 데 사용하며, 작은따옴표, 큰따옴표 또는 백틱으로 감쌉니다:

```javascript
let greeting = "Hello World!";

let greeting2 = 'I am learning JavaScript!';

let greeting3 = `with The Odin Project!`;
```

작은따옴표로 선언한 문자열과 큰따옴표로 선언한 문자열은 동일하며, 어느 쪽을 사용할지는 개인 취향에 달려 있습니다 — 다만 한 가지 스타일을 선택해 코드에서 일관되게 사용하는 것이 좋은 습관입니다.
