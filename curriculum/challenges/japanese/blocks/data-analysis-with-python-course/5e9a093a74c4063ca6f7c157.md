## --text--

Python の標準ライブラリにおけるオブジェクト（リストやデータ型など）のメモリ上のサイズと、NumPy ライブラリのオブジェクトのサイズにはどのような関係がありますか？これを踏まえて、パフォーマンスにどのような影響がありますか？

## --answers--

標準の Python オブジェクトは NumPy オブジェクトよりもはるかに多くのメモリを使用しますが、標準の Python と NumPy の同等のオブジェクトに対する操作はほぼ同じ時間で完了します。
