# --interactive--

Елемент superscript використовується для відображення тексту у вигляді верхнього індексу. Верхній індекс — це символ або літера, надруковані вище за звичайну лінію тексту.

Ось приклад використання елемента superscript для ілюстрації степенів (щоб побачити попередній перегляд, увімкніть інтерактивний редактор):

:::interactive_editor

```html
<p>2<sup>2</sup> (2 squared) is 4.</p>
```

:::

У цьому прикладі число 2 обгорнуте в `sup` теги, щоб представити верхній індекс всередині абзацу. У вікні попереднього перегляду ви побачите, що друге число 2 менше і трохи вище за перше число 2.

Загальні випадки використання елемента superscript включають степені, верхні літери та порядкові числівники. Ось приклад використання елемента superscript для верхніх літер (щоб побачити попередній перегляд, увімкніть інтерактивний редактор):

:::interactive_editor

```html
<p>
  Monseigneur is often written as <strong>M<sup>gr</sup></strong>.
</p>
```

:::

Верхні літери — це літери, написані у вигляді верхнього індексу, зазвичай для позначення скорочень. Літери `g` і `r` обгорнуті в теги superscript, щоб проілюструвати скорочення в цьому прикладі.

Важливо зазначити, що елемент superscript слід використовувати лише з типографічною метою. Якщо ви хочете стилізувати текст із піднятою базовою лінією, слід використовувати CSS замість елемента superscript.

Щоб представити хімічні рівняння в HTML, використовуйте елемент subscript. Цей елемент створює нижній індекс, який відображає текст із зниженою базовою лінією та меншим розміром шрифту.

Ось приклад використання елемента subscript для показу хімічного представлення вуглекислого газу (щоб побачити попередній перегляд, увімкніть інтерактивний редактор):

:::interactive_editor

```html
<p>CO<sub>2</sub></p>
```

:::

Число два обгорнуте в `sub` теги, щоб показати, що цей символ має бути нижнім індексом.

Загальні випадки використання елемента subscript включають хімічні формули, сноски та нижні індекси змінних.
