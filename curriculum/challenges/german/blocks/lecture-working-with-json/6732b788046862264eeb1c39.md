# --description--

Was ist JSON und wie greifen Sie mit Klammer- und Punktnotation auf Werte zu?

JSON steht für JavaScript Object Notation. Es ist ein leichtgewichtiges, textbasiertes Datenformat, das häufig zum Austausch von Daten zwischen einem Server und einer Web-Anwendung verwendet wird.

Ein Grund, warum JSON in der Webentwicklung so beliebt ist, liegt darin, dass es sowohl maschinenlesbar als auch für Menschen gut lesbar ist.

Da JSON sprachunabhängig ist, können Sie JSON-Daten problemlos von einer Java-Anwendung an eine Python-Anwendung oder von einer JavaScript-Anwendung an eine C#-Anwendung senden.

JSON unterstützt viele Datentypen, darunter Objekte, Arrays, Strings, Booleans, null und Zahlen.

Hier ist ein Beispiel für ein JSON-Objekt:

```js
{
  "name": "Alice",
  "age": 30,
  "isStudent": false,
  "list of courses": ["Mathematics", "Physics", "Computer Science"]
}
```

Wie Sie sehen, verwendet JSON Schlüssel-Wert-Paare, um Informationen zu speichern, und jedes Paar ist durch ein Komma getrennt. Jeder Schlüssel muss in Anführungszeichen eingeschlossen sein, sonst erhalten Sie einen Fehler.

Um Daten aus einem JSON-Objekt zuzugreifen, können Sie entweder die Punktnotation oder die Klammernotation verwenden. In diesem Beispiel verwenden wir die Punktnotation, um auf das `age` aus dem JSON-Objekt zuzugreifen:

```js
import data from "./example.json" with { type: "json" };

console.log(data.age);
```

Dieses spezielle Beispiel verwendet eine sogenannte `import`-Anweisung, die das JSON-Objekt in diese Datei importiert, sodass wir darauf zugreifen können. Mehr über die `import`-Anweisung lernen Sie in einer zukünftigen Lektion.

Sie können auch die Klammernotation verwenden, um Informationen aus JSON-Objekten zuzugreifen. Hier ist ein Beispiel, wie Sie auf das `list of courses`-Array zugreifen:

```js
import data from "./example.json" with { type: "json" };

console.log(data["list of courses"]);
```

Die Verwendung der Klammernotation ist hier besonders nützlich, weil der Schlüssel aus mehreren Wörtern besteht, die durch Leerzeichen getrennt sind. Wenn wir versuchen würden, die Punktnotation zu verwenden, würde das zu einem Fehler führen.

Zusammenfassend ist JSON ein vielseitiges Format, das viele Datentypen speichern kann, einschließlich Arrays und verschachtelter Objekte. Mit der Punktnotation oder der Klammernotation können Sie einfach auf die in einem JSON-Objekt gespeicherten Werte zugreifen.
