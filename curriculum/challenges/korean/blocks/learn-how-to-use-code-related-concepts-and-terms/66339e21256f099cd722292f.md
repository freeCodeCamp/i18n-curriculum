# --description--

프로그래밍에서 `constant`는 변수와 비슷하지만, 한 번 값이 할당되면 그 값을 변경할 수 없습니다. `Constants`는 프로그램 전체에서 수정할 필요가 없을 것으로 예상되는 값을 다룰 때 사용합니다. 변수와 구분하기 위해 대문자로 작성하는 경우가 많습니다.

예를 들어, `const PI = 3.14;` 많은 프로그래밍 언어에서 `const`는 상수를 선언하는 데 사용됩니다. 여기서 `PI`는 값 `constant`가 할당된 `3.14`입니다.
