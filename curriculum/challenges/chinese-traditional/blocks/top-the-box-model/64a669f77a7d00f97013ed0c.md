# --description--

既然你已經了解了 HTML 和 CSS 的基本語法，我們將開始進入正題。你需要掌握的最重要的 CSS 技能是定位和版面配置。更改字型和顏色是關鍵技能，但能夠將元素精確地放置在網頁上的指定位置更為重要。畢竟，你能找到多少個網頁是所有元素完全堆疊在一起的呢？

一旦你了解了幾個關鍵的概念，學會在網頁上定位元素並不困難。不幸的是，許多學習者急於學習 HTML 和 CSS 以進入 JavaScript，結果錯過了這些基本的概念。這會導致挫折、痛苦（以及有趣的動圖），因為如果你無法將元素放置在頁面上你需要的位置，所有的 JavaScript 技能都毫無意義。所以，考慮到這一點，讓我們開始吧。
