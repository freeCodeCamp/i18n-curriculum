# --description--

螢幕放大鏡是幫助低視力及其他視覺障礙者更好地存取數位內容和網路的工具。

讓我們更深入探討這些工具是什麼以及它們在數位內容存取中扮演的角色。

螢幕放大鏡透過放大電腦或行動裝置螢幕上的文字、圖形及其他元素來運作。許多螢幕放大鏡允許使用者將顯示放大超過 200%。使用者接著可以使用他們的指標裝置或鍵盤來瀏覽頁面。此外，大多數放大鏡在其設定中提供訂製的縮放百分比及其他功能。

螢幕放大鏡主要幫助低視力的人閱讀文字，因為文件或應用程式中的小字型可能對他們來說很困難。透過放大文字，他們可以在不費力的情況下閱讀電子郵件、文章和其他內容。螢幕放大鏡也協助網頁瀏覽。它們幫助使用者找到並點擊可能難以看清的按鈕、聯結和其他互動元素。這種改善的可見度確保使用者能夠輕鬆瀏覽網站、填寫表單並參與線上活動。

因此，軟體開發人員需要讓他們的數位產品對低視力者可存取。部分考量包括：

- 使用可縮放字型，讓使用者可以調整頁面大小而不會破壞版面配置。
- 確保使用者介面透過響應式設計適應不同螢幕尺寸。
- 使用高對比度色彩結構綱目和可訂製的顏色。
- 實作一個非固定且極小的導覽列，讓使用放大鏡時用戶仍能看到內容。
- 使用一般的 HTML 文字而非文字的影像。
- 直接在觸發它的元素旁提供回授，還有更多。

所有主要的作業系統至少都有一個由其製造商內建的放大鏡：

- macOS 和 iOS 都有 Zoom。你可以在 macOS 上透過前往 設定，篩選「輔助使用」，然後點擊 `"Zoom"` 來開啟它。切換 `"Use keyboard shortcuts to zoom"` 選項以啟用它。
  - 你可以在 iPhone 上透過 *設定 > 輔助使用 > 縮放* 開啟它。
- Android 裝置具有放大功能。要開啟它，請前往 *Settings > Special Function > Accessibility > Magnification*。由於這可能因裝置而異，你可以在設定首頁搜尋「Magnification」以存取它。
- Windows 有放大鏡。你可以透過 *設定 > 存取輔助功能 > 放大鏡* 使用它。
- Linux 作業系統的放大鏡各有不同。它要麼是 Zoom，要麼是 Magnifier。

除了作業系統內建的之外，一些有用的第三方螢幕放大鏡有：

- ZoomText for Windows。
- ClaroView 適用於 macOS 和 Windows。
- iZoom for Windows。
- Zoomify－macOS 的螢幕放大鏡。
- LunarPluse for 視窗。
- macOS 的 Loupe。
