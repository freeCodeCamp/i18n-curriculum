# --description--

`equals symbol` (`=`) hutumika sana kwa mgawo wa thamani. Inatumika kuweka thamani katika kigezo. Jina la kigezo linaandikwa kushoto mwa alama ya usawa, na thamani unayotaka kuhifadhi katika kigezo iko kulia.

Kwa mfano, `name = "Sophie";` huweka mfuatano wa herufi `"Sophie"` katika kigezo `name`.

`assign a value` katika programu inamaanisha kutoa thamani maalum kwa kigezo. `equals symbol` (`=`) hutumika sana kwa madhumuni haya. Unapoweka thamani katika kigezo, unamwambia programu kukumbuka thamani hiyo kwa matumizi baadaye.

Kwa mfano, `age = 25` huweka thamani `25` katika kigezo `age`.
