# --description--

一個整數序列 $S = \\{s_i\\}$ 稱為一個 $n$-序列，如果它有 $n$ 個元素且每個元素 $s_i$ 滿足 $1 ≤ s_i ≤ n$。因此總共有 $n^n$ 個不同的 $n$-序列。

例如，序列 $S = \\{1, 5, 5, 10, 7, 7, 7, 2, 3, 7\\}$ 是一個 10 項序列。

對於任意序列 $S$，令 $L(S)$ 為 $S$ 中具有相同值的最長連續子序列的長度。 例如，對於上述給定的序列 $S$，$L(S) = 3$，因為有三個連續的 7。

令 $f(n) = \sum L(S)$，其中 $S$ 為所有長度為 n 的序列。

例如，$f(3) = 45$、$f(7) = 1\\,403\\,689$ 和 $f(11) = 481\\,496\\,895\\,121$。

求 $f(7\,500\,000)\bmod 1\,000\,000\,009$。
