# --explanation--

`Near` bedeutet räumliche Nähe. Zum Beispiel:

`The meeting room is near the kitchen.` – Das bedeutet, dass die beiden Orte nah beieinander liegen.

`Entrance` ist der Ort, an dem man ein Gebäude betritt. Zum Beispiel:

`Please use the main entrance to get inside.` – Das zeigt den Leuten, wo sie hineingehen sollen.
