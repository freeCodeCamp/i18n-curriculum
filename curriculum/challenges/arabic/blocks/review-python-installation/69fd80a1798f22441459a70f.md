## Python في بيئة العمل المحلية الخاصة بك

- **التثبيت**: أفضل طريقة لتثبيت Python على Windows وMac وLinux هي تنزيل المثبّت من الموقع الرسمي لـ Python (`https://www.python.org/`).
- **الطرفية**: واجهة نصية تتيح لك التفاعل مع الكمبيوتر عبر كتابة الأوامر. كل نظام تشغيل يأتي مع تطبيق طرفية افتراضي. على macOS، يمكنك استخدام تطبيق Terminal. على Windows، يمكنك استخدام Command Prompt أو PowerShell. على Linux، لكل بيئة سطح مكتب تطبيق طرفية افتراضي خاص بها، مثل GNOME Terminal أو Konsole.
- **IDE**: IDE تعني بيئة تطوير متكاملة وتحتوي على ميزات تشمل محرر شفرة، أدوات اختبار، طرفية والمزيد.
- **محررات الشفرة وبيئات التطوير المتكاملة الشهيرة لـ Python**: VS Code، PyCharm، وSpyder
- **تشغيل الشفرة محليًا**: لتشغيل شفرة Python، لديك عدة خيارات. الخيار الأول هو النقر على زر التشغيل في VS Code الذي يشبه زر التشغيل الموجود في الزاوية العلوية اليمنى. هذا سيفتح طرفية ويشغّل سكريبت Python هناك.

الخيار الآخر هو فتح طرفية وتشغيل البرنامج يدويًا. إليك مثال لتشغيل برنامج اسمه `main.py`:

```bash
python main.py
```

يجب تشغيل هذا الأمر من المجلد الذي يحوي `main.py`. على سبيل المثال، إذا كان `main.py` داخل مجلد اسمه `python-projects`، استخدم أولًا الأمر التالي في الطرفية:

```bash
cd python-projects
```

ثم شغّل:

```bash
python main.py
```

## استخدام القشرة التفاعلية لـ Python

- **التعريف**: القشرة التفاعلية برنامج يتيح لك كتابة الأوامر واحدة تلو الأخرى ورؤية النتائج. افتح تطبيق طرفية واكتب `python` ثم اضغط `Enter`. هذا سيبدأ قشرة Python التفاعلية. الرمز `>>>` يعني أن Python ينتظر منك كتابة أمر. جرّب طباعة `"Hello, world!"` في الطرفية:

```bash
print("Hello, world!")
```

يجب أن ترى النص يظهر هكذا:

```bash
>>> print("Hello, world!")
Hello, world!
```

بعد طباعة النص، يعود Python إلى الشكل التالي:

```bash
>>>
```

هذا يعني أنه يمكنك كتابة أمر آخر.

- **دورة القراءة، التقييم، الطباعة، التكرار (REPL)**: كلما كتبت أوامر، يقرأ المفسر المُدخَل، يقيمه، يطبع النتيجة، ثم يعود ليعرض `>>>` لتتمكن من كتابة المزيد من الأوامر.
