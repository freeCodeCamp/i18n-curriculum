# --explanation--

結構體 `what` 加上一個不定冠詞用來強調對某事的強烈感受或反應。它經常表達驚訝、讚嘆或情感。例如： 

`What a beautiful day it is!`－這裡，結構凸顯標示說話者對這一天的讚賞。 

同樣地，在 Maria 的句子中，`What a surprise` 強調她在 PyCon 見到 Brian 時的驚訝。  

`PyCon` 是一個知名的 Python 程式設計愛好者、開發者和專業人士的會議。它提供一個學習、網路交流和分享與 Python 相關知識的平台。 例如： 

`I attended PyCon last year and learned about the latest Python frameworks。`－這表示此人去年參加了 PyCon 並學到了很多。
