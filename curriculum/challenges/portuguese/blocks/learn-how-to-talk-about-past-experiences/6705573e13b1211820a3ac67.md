# --explanation--

A forma negativa do verbo `to be` no tempo `Simple Past` é `was not` ou `wasn't` para sujeitos singulares e `were not` ou `weren't` para sujeitos plurais.É usado para descrever algo que não aconteceu ou não era verdade.Por exemplo:  

- Singular: `The code wasn't running as expected.` - Um problema estava acontecendo com o código.

- Plural: `The changes weren't implemented correctly.` - Erros na implementação causaram os bugs.

Na frase de James, ele usa `wasn't` para explicar que o sistema estava falhando ao conectar-se ao banco de dados.Frases negativas com `wasn't` ou `weren't` indicam o que não ocorreu ou não funcionou como você poderia esperar.
