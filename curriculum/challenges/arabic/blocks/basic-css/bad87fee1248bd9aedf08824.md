# --description--

أحيانًا تريد تخصيص عنصر بحيث يكون له `margin` مختلف على كل جانب من جوانبه.

تسمح لك CSS بالتحكم في `margin` لكل من الجوانب الأربعة الفردية للعنصر باستخدام الخصائص `margin-top` و`margin-right` و`margin-bottom` و`margin-left`.

# --instructions--

أعطِ الصندوق الأزرق `margin` بقيمة `40px` في الأعلى وعلى الجانب الأيسر، و`20px` فقط في الأسفل وعلى الجانب الأيمن.

# --hints--

يجب أن تعطي صنف `blue-box` الجزء العلوي من العناصر `margin` بقيمة `40px`.

```js
const blueBox = document.querySelector('.blue-box');
const marginTop = window.getComputedStyle(blueBox)['margin-top'];
assert.strictEqual(marginTop, '40px');
```

يجب أن تعطي صنف `blue-box` الجانب الأيمن من العناصر `margin` بقيمة `20px`.

```js
const blueBox = document.querySelector('.blue-box');
const marginRight = window.getComputedStyle(blueBox)['margin-right'];
assert.strictEqual(marginRight, '20px');
```

يجب أن تعطي صنف `blue-box` الجزء السفلي من العناصر `margin` بقيمة `20px`.

```js
const blueBox = document.querySelector('.blue-box');
const marginBottom = window.getComputedStyle(blueBox)['margin-bottom'];
assert.strictEqual(marginBottom, '20px');
```

يجب أن تعطي صنف `blue-box` الجانب الأيسر من العناصر `margin` بقيمة `40px`.

```js
const blueBox = document.querySelector('.blue-box');
const marginLeft = window.getComputedStyle(blueBox)['margin-left'];
assert.strictEqual(marginLeft,'40px');
```
