# --explanation--

La frase `in real time` indica qualcosa che accade immediatamente mentre gli eventi si svolgono, senza ritardi. Viene spesso usata nel contesto di monitoraggio, comunicazione o elaborazione dati. Per esempio:

`The system updates the dashboard in real time, so you can see changes instantly.` – significa che la dashboard cambia immediatamente dopo che gli aggiornamenti sono stati applicati.

Nella frase di Jake, intende dire che, anche se controllava le vulnerabilità di sicurezza, non sempre lo faceva immediatamente mentre gli aggiornamenti avvenivano.
