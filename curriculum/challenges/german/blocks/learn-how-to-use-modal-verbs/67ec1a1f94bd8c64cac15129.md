# --explanation--

`Get to the bottom of` ist eine gebräuchliche Wendung, die bedeutet, die wahre Ursache eines Problems herauszufinden. Sie deutet darauf hin, aktiv zu werden, um etwas gründlich zu verstehen. Zum Beispiel:  

`We need to get to the bottom of these login errors before the next release.` – Das bedeutet, dass das Team herausfinden möchte, was wirklich die Fehler verursacht, damit sie diese beheben können.
