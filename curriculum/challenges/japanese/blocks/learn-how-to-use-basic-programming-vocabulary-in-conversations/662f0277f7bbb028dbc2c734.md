# --description--

`There are` は複数の物や人の存在や有無を示すために使われます。`Many` は数えられる名詞と一緒に使い、多くの数量を示します。

これらの言葉を組み合わせることで、多数の物が存在していることを表現できます。

例えば、`There are many options.` は多くの選択肢があることを意味します。
