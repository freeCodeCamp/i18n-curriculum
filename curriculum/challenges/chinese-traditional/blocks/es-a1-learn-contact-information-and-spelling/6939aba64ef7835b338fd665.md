# --description--

你還記得動詞會依據它們所取用的對象變化嗎？如果不記得，現在你記得了！

在幾個任務之前，你學會了如何使用動詞 `poder` 來詢問某人是否能為你做某事。

現在你將學習如何變化這個動詞來詢問 **你** 是否能做某事。你只需要使用以下結構：

`Puedo` + 不定詞動詞 + `...?`

讓我們來看一些範例：

- `¿Puedo usar la computadora?` - 我可以使用電腦嗎？

- `¿Puedo usar mi apodo?`－我可以使用我的暱稱嗎？

在這兩個範例中，你都是在請求許可。第一個是使用 `computer` 的許可，第二個是使用你的暱稱的許可。

# --explanation--

Basti 說 `¿Puedo usar mi apodo?`。

這裡，`puedo` 是 `poder` 的第一人稱單數形式，用於當某人為自己請求許可時。

結構體遵循以下樣式：

`¿Puedo` + 不定詞 + `...?`

在這種情況下：

- `Usar` 意思是使用

- `Mi apodo` 取用他的暱稱

所以 Basti 在問他是否被允許使用他的暱稱。
