# --description--

Como recordarás de las conferencias, en Python, declaras una variable escribiendo el nombre de la variable en el lado izquierdo del operador de asignación (`=`) y el valor a asignar en el lado derecho:

```py
variable_name = value
```

Crea una variable llamada `shift` y asigna el valor `5` a tu nueva variable.

# --hints--

Debes declarar una variable llamada `shift`. Presta atención para colocar el nombre de la variable al principio de la línea.

```js
({ test: () => assert(runPython(`_Node(_code).has_variable("shift")`)) })
```

Debes asignar el valor `5` a tu variable `shift`.

```js
({ test: () => assert(runPython(`_Node(_code).find_variable("shift").is_equivalent("shift = 5")`)) })
```
