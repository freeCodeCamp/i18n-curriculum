# --interactive--

프로그램을 만들 때, 일정 시간 후에 무언가가 실행되거나 정기적으로 반복되길 원할 때가 많습니다. 이럴 때 `setTimeout()`와 `setInterval()` 메서드가 유용합니다. 두 메서드 모두 함수와 지연 시간이라는 두 개의 매개변수를 받습니다.

각 메서드를 좀 더 자세히 살펴보고 웹 프로젝트에서 어떻게 사용하는지 알아보세요.

`setTimeout()`는 특정 시간 동안 동작을 지연시킬 수 있습니다. `setTimeout()` 메서드의 기본 구문은 다음과 같습니다:

```js
setTimeout(functionToRun, delay);
```

`functionToRun`는 지정한 지연 시간 후에 실행할 코드이고, `delay`는 함수를 실행하기 전에 기다릴 시간(밀리초 단위)입니다. 예시는 다음과 같습니다:

:::interactive_editor

```js
setTimeout(function () {
 console.log("This runs after 3 seconds");
}, 3000);
```

:::

콜백으로 화살표 함수를 사용할 수도 있습니다:

:::interactive_editor

```js
setTimeout(() => {
 console.log("This runs after 3 seconds"); 
}, 3000);
```

:::

`setInterval()`는 설정한 간격마다 함수를 반복 실행합니다. 몇 초마다 알람이 울리는 것과 같습니다. `setInterval()`는 데이터를 새로 고치거나 계속 움직이는 애니메이션을 만들 때처럼 무언가를 반복해서 실행하고 싶을 때 적합합니다.

`setTimeout()`와 마찬가지로 함수와 지연 시간을 매개변수로 받습니다:

```js
setInterval(functionToRun, delay);
```

`functionToRun`는 반복 실행할 코드이고, `delay`는 함수가 실행되는 간격(밀리초 단위)입니다. 예시는 다음과 같습니다:

:::interactive_editor

```js
setInterval(() => {
 console.log("This runs every 2 seconds");
}, 2000);

// Result:
// This runs every 2 seconds
// This runs every 2 seconds
// This runs every 2 seconds
// ...
```

:::

`setInterval()`는 지정한 간격마다 함수를 계속 실행하므로, 이를 멈추고 싶을 때가 있습니다. 이럴 때 `clearInterval()` 메서드를 사용하세요.

`clearInterval()`는 멈추고자 하는 `setInterval()`의 ID를 매개변수로 받습니다. 이 ID는 간격을 할당한 변수일 수 있습니다. 간격을 멈추는 한 가지 방법은 `setTimeout()` 안에서 하는 것으로, 일정 시간이 지나면 간격이 멈춥니다:

:::interactive_editor

```js
const intervalID = setInterval(() => {
 console.log("This will stop after 5 seconds");
}, 1000);

setTimeout(() => {
 clearInterval(intervalID);
}, 5000);
```

:::

`clearTimeout()` 메서드로 타임아웃도 멈출 수 있습니다:

:::interactive_editor

```js
let timeoutID = setTimeout(() => {
 console.log("This will not run");
}, 5000);

clearTimeout(timeoutID);
```

:::

이 코드를 사용하면 `setTimeout()`가 전혀 실행되지 않습니다. 더 우아한 방법은 DOM에서 버튼에 클릭 이벤트 리스너를 추가해 타임아웃을 멈추는 것입니다:

:::interactive_editor

```html
<h1>Cancel Timeout Example</h1>
<button id="cancelButton">Cancel Timeout</button>
<script src="index.js"></script>
```

```js
let timeoutID = setTimeout(() => {
  console.log("This will run if not canceled");
}, 5000);

document
.querySelector("#cancelButton")
.addEventListener("click", () => {
  clearTimeout(timeoutID);
  console.log("Timeout canceled!");
});
```

:::
