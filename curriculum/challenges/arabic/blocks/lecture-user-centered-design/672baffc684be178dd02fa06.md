# --interactive--

**ملاحظة**: قد تستخدم بعض الأمثلة التفاعلية CSS لم تتعلمه بعد. لا تقلق بشأن محاولة فهم كل الشفرة. الهدف من الأمثلة هو عرض معاينات لهذه المفاهيم التصميمية لتفهم كيف تعمل الأمور بشكل أفضل. لعرض المعاينات، تحتاج إلى تفعيل المحرر التفاعلي.

تُعد مكونات البطاقات شائعة جدًا في التجارة الإلكترونية ووسائل التواصل الاجتماعي ومواقع الأخبار. تُستخدم لمساعدة عرض المعلومات بطريقة منظمة. عند تصميم بطاقاتك، من المهم فهم أفضل الممارسات حتى يتمكن المستخدمون من فهم المعلومات التي تحاول إيصالها بسهولة.

أول اعتبار في تصميم البطاقة يجب أن يكون البساطة. لا تريد أن تكون بطاقاتك مزدحمة بصريًا أو تعرض معلومات كثيرة جدًا. على سبيل المثال، إذا كان تصميم البطاقة مزدحمًا بصريًا، فسيكون هناك معلومات كثيرة جدًا على المستخدم معالجتها دفعة واحدة.

إليك مثال على تصميم بطاقة مزدحمة:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2><strong><em>Product Name: Ultimate Edition</em></strong></h2>
  <p>
    This is a great product that you will love! It has many features and benefits that will make your life easier and more enjoyable. Whether you're at home or on the go, this product is perfect for all situations. Don’t miss out on this incredible offer, buy now and experience the difference!
  </p>
  <button>Buy Now</button>
  <button>Learn More</button>
  <button>Add to Wishlist</button>
  <p><strong>Only 3 left in stock!</strong></p>
  <p><em>Rated 4.8 stars by 2,391 customers</em></p>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}
.card p {
  font-size: 1em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

وجود معلومات أقل وتباعد جيد بين عناصر البطاقة يجعل من السهل على المستخدم معالجة المعلومات، ويسمح بعرض عدة بطاقات في الصفحة.

إليك مثال على تصميم بطاقة بسيطة:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2>Product Name</h2>
  <button>Buy Now</button>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 200px;
  text-align: center;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

شيء آخر يجب أخذه في الاعتبار هو مكان نقر المستخدم على البطاقة. بعض تصاميم البطاقات تحتوي على زر واحد فقط، مما يجعل مكان النقر واضحًا للمستخدم. تصاميم أخرى تجعل البطاقة بأكملها قابلة للنقر. عندما يحوم المستخدم فوق أي جزء من البطاقة، تتغير لون البطاقة أو يظهر تأثير ظل للدلالة على أن البطاقة قابلة للنقر. مهما كان التصميم الذي تختاره، يجب أن يكون متسقًا في موقعك وسهل الفهم للمستخدم.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <a href="#">
    <img src="https://placehold.co/150x150" alt="Example Product Image">
    <h2>Product Name</h2>
    <button>Buy Now</button>
  </a>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
  cursor: pointer;
}

.card:hover {
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
}

.card a {
  text-decoration: none;
  color: inherit;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

اعتبار آخر هو استخدام الوسائط في بطاقاتك. اختيار وسائط عالية الجودة يمكن أن يعزز تجربة المستخدم بشكل كبير. إذا كنت تستخدم صورًا أو فيديوهات لبطاقة منتج مثلاً، كلما كانت الجودة أعلى زاد اهتمام المستخدم بذلك المنتج. أما إذا استخدمت وسائط ذات جودة ضعيفة، فقد لا يثق المستخدم في جودة منتجاتك وخدماتك.

أحد الأمور الأخيرة التي يجب مراعاتها هو استخدام التسلسل اللوني. تريد التأكد من أن أهم المعلومات في البطاقة هي الأكثر بروزًا. يمكنك استخدام ألوان زاهية للعناصر المهمة مثل زر الدعوة إلى اتخاذ إجراء، وألوان فاتحة للعناصر الأقل أهمية في البطاقة.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2>Product Name</h2>
  <button>Buy Now</button>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
  text-align: center;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #28a745; 
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

أثناء استمرارك في العمل على تطبيقات الويب، تذكر أفضل الممارسات المختلفة لتصميم البطاقات. هذا سيساعدك على إنشاء تجارب مستخدم أفضل.
