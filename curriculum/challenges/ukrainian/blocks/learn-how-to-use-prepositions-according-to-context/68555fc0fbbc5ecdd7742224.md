# --explanation--

`On` використовується у фразі `planning on`, що означає мати намір або очікувати щось зробити. Наприклад:

`We're planning on launching next week.` – Це означає, що команда очікує або має намір це зробити.

`Bringing in` означає додавання людей до команди або групи. Наприклад:

`The manager is bringing in a new developer.` – Це означає, що когось додають.

`Additional` означає більше, ніж вже існує. Наприклад:

`We need additional help to finish the project.` – Це означає, що потрібна додаткова допомога.
