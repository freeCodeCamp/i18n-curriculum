# --description--

Em qualquer linguagem de programação, você precisa comparar valores para tomar decisões. Por exemplo, se o tempo está ensolarado, você sairá; caso contrário, ficará em casa. Outro exemplo seria verificar se o usuário tem pontos suficientes para subir de nível em um jogo.

A instrução condicional `if-else` é usada para tomar decisões em JavaScript. Ela é frequentemente usada com operadores de comparação para comparar valores e tomar decisões com base nos resultados.

Um exemplo de uma instrução if é:

```javascript
let x = 5;

if (x > 3) {
  console.log("x is greater than 3");
} else {
  console.log("x is less than or equal to 3");
}
```

Neste exemplo, a variável `x` é comparada ao número `3` usando o operador `>`. Se `x` for maior que `3`, o primeiro bloco de código será executado. Caso contrário, o segundo bloco de código será executado.
