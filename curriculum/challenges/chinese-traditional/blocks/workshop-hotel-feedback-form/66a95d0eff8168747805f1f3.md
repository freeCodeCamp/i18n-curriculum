# --description--

如果希望用戶從列表中選擇多個選項，可以使用複選框。

下面是一個如何使用複選框處理食物選項的示例：

```html
<fieldset>
  <legend>Food Options</legend>
  <input type="checkbox" id="pizza" name="food" value="pizza">
  <label for="pizza">Pizza</label>
  <input type="checkbox" id="burger" name="food" value="burger">
  <label for="burger">Burger</label>
</fieldset>
```

`value` 屬性用於指定提交表單時發送到服務器的值。

在你的 `legend` 元素下方，添加一個複選框 `input`，其 `id`、`name` 和 `value` 屬性都設置爲 `"ads"`。

在你的複選框 `input` 下方，添加一個 `label` 元素，其文本爲 `Social Media Ads`。 `for` 屬性應設置爲 `"ads"`。
