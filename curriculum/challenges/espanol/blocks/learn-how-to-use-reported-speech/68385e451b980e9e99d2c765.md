# --explanation--

`True` es un adjetivo que significa que algo es correcto, real o exacto. Se usa para estar de acuerdo con una afirmación o hecho. Es diferente de `the truth`, que es un sustantivo que se refiere a los hechos reales o la realidad sobre algo, y que a menudo se usa en un sentido más amplio o abstracto. Por ejemplo:

- `It's true that the server crashed yesterday.` – Esto significa que la afirmación sobre el fallo es correcta.

- `The truth is that we didn't have a backup.` – Esto significa que el hecho real es que no existía ninguna copia de seguridad.
