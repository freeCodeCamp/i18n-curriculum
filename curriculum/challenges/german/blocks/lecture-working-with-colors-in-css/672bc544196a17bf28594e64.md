# --interactive--

In einer vorherigen Lektion haben Sie gelernt, wie man mit linearen und radialen Verläufen arbeitet. In dieser Lektion tauchen wir tiefer ein, wie Sie mit den CSS-Eigenschaften arbeiten, um optisch ansprechendere und komplexere Designs zu erstellen.

Verläufe in CSS ermöglichen es Ihnen, sanfte Übergänge zwischen zwei oder mehr angegebenen Farben zu erzeugen. Sie werden oft verwendet, um Tiefe, Textur und visuelles Interesse in Webdesigns hinzuzufügen, ohne Bilddateien zu benötigen.

CSS unterstützt zwei Haupttypen von Verläufen: lineare Verläufe und radiale Verläufe.

Lineare Verläufe erzeugen eine allmähliche Mischung zwischen Farben entlang einer geraden Linie. Sie können die Richtung dieser Linie und die verwendeten Farben steuern. Hier ist ein einfaches Beispiel für einen linearen Verlauf:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

Dies erzeugt einen Verlauf, der von Rot links zu Blau rechts übergeht. Der `to right`-Parameter gibt die Richtung des Verlaufs an. Sie können auch andere Richtungen wie `to left`, `to top`, `to bottom` verwenden oder sogar Winkel wie `45deg` angeben.

Sie können auch mehrere Farb-Stopps in einem linearen Verlauf verwenden:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="multi-color-gradient"></div>
```

```css
.multi-color-gradient {
  background: linear-gradient(to right, red, yellow, blue);
  height: 40vh;
}
```

:::

Dies erzeugt einen Verlauf, der von Rot zu Gelb zu Blau übergeht, gleichmäßig über das Element verteilt.

Radiale Verläufe hingegen erzeugen kreisförmige oder elliptische Verläufe, die von einem zentralen Punkt ausgehen. Hier ist ein Beispiel:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::

Dies erzeugt einen kreisförmigen Verlauf, der in der Mitte mit Rot beginnt und zu Blau an den Rändern übergeht. Sie können auch die Form und Größe radialer Verläufe angeben:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="custom-radial-gradient"></div>
```

```css
.custom-radial-gradient {
  background: radial-gradient(ellipse at top left, red, blue);
  height: 40vh;
}
```

:::

Dies erzeugt einen elliptischen Verlauf, der von der oberen linken Ecke des Elements ausgeht. Sowohl lineare als auch radiale Verläufe erlauben es Ihnen, Farb-Stopps an genauen Positionen anzugeben. Zum Beispiel:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="precise-gradient"></div>
```

```css
.precise-gradient {
  background: linear-gradient(to right, red 20%, yellow 40%, blue 80%);
  height: 40vh;
}
```

:::

Dies erzeugt einen Verlauf, bei dem Rot die ersten `20%` einnimmt, dann bei `40%` zu Gelb übergeht und schließlich bei `80%` zu Blau.

Verläufe können überall dort verwendet werden, wo Sie in CSS ein Bild verwenden würden. Das macht sie vielseitig für Hintergründe, Schaltflächen oder sogar Texteffekte, wenn sie mit anderen CSS-Eigenschaften kombiniert werden.

Denken Sie daran, dass Verläufe zwar die optische Attraktivität Ihrer Website verbessern können, es aber wichtig ist, ausreichenden Kontrast für die Barrierefreiheit sicherzustellen, besonders wenn Verläufe hinter Text verwendet werden.

# --questions--

## --text--

Was ist der Unterschied zwischen linearen und radialen Verläufen in CSS?

## --answers--

Lineare Verläufe verwenden nur zwei Farben, während radiale Verläufe mehrere Farben verwenden.
