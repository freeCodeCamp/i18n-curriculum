# --description--

你是否曾点击网页却感觉没有任何反应？这种延迟会影响用户体验，并通过一个名为 Interaction to Next Paint，或 INP 的指标来衡量。

INP 通过测量用户交互（如点击或按键）到浏览器下一次更新显示之间的时间，评估页面的整体响应速度。较低的 INP 表示页面响应更快。

但是为什么 `INP` 很重要？较高的 `INP` 会让用户感觉页面无响应，导致挫败感。优化 `INP` 可确保用户及时收到反馈，提升他们的体验。

像 Chrome 的 DevTools 和网页性能 API 这样的工具可以通过跟踪交互延迟来帮助测量 INP。

为了提升 INP，最小化主线程工作，优化事件处理器，并确保用户交互触发及时的视觉更新。

通过理解和优化 INP，你可以创建感觉快速且响应灵敏的网页体验，保持你的用户满意。
