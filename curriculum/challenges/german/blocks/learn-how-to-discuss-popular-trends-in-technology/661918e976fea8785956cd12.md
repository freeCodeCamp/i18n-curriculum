# --description--

Wenn Sie `like` verwenden, um Dinge aufzulisten, bedeutet das, dass Sie konkrete Beispiele für etwas geben. Zum Beispiel, wenn Sie über Obst wie Äpfel, Bananen und Orangen sprechen, geben Sie Beispiele für verschiedene Obstsorten. `like` hilft Ihnen, zu zeigen, was Sie meinen, indem Sie spezifische Namen oder Dinge zum Thema nennen. Beispiel: `My son loves eating different kinds of fruit like apples, bananas, and oranges.`
