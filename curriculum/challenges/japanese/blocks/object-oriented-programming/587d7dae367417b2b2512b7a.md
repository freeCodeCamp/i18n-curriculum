# --description--

コンストラクタ関数が新しいオブジェクトを作成すると、そのオブジェクトはそのコンストラクタの<dfn>インスタンス</dfn>と呼ばれます。JavaScriptは`instanceof`演算子でこれを簡単に検証できます。`instanceof`はオブジェクトとコンストラクタを比較し、そのオブジェクトがそのコンストラクタで作成されたかどうかに応じて`true`または`false`を返します。例を示します。

```js
let Bird = function(name, color) {
  this.name = name;
  this.color = color;
  this.numLegs = 2;
}

let crow = new Bird("Alexis", "black");

crow instanceof Bird;
```

この`instanceof`メソッドは`true`を返します。

もしオブジェクトがコンストラクタを使わずに作成された場合、`instanceof`はそのオブジェクトがそのコンストラクタのインスタンスでないことを検証します。

```js
let canary = {
  name: "Mildred",
  color: "Yellow",
  numLegs: 2
};

canary instanceof Bird;
```

この`instanceof`メソッドは`false`を返します。

`House`コンストラクタの新しいインスタンスを作成し、`myHouse`と名付けて寝室の数を渡してください。それから、`instanceof`を使ってそれが`House`のインスタンスであることを検証してください。
