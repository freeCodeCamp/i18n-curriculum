# --description--

我们将 `pythagorean polygon` 定义为具有以下属性的凸多边形：

- 至少有三个顶点，
- 没有三个顶点对齐，
- 每个顶点都有整数坐标
- 每条边都有整数长度。

对于给定的整数 $n$，定义 $P(n)$ 为周长 $≤ n$ 的不同毕达哥拉斯多边形的数量。

毕达哥拉斯多边形应该被认为是不同的，只要它们都不是另一个的翻译即可。

已知 $P(4) = 1$，$P(30) = 3655$ 和 $P(60) = 891045$。

查找 $P(120)$。
