# --description--

在任何编程语言中，你都需要比较数值以做出决策。例如，如果天气晴朗，你会出去，否则你会待在家里。另一个例子是检查用户是否有足够的积分在游戏中升级。

`if-else` 条件式语句用于在 JavaScript 中做出决策。它通常与比较操作符一起使用，以比较值并根据结果做出决策。

if 语句的一个例子是：

```javascript
let x = 5;

if (x > 3) {
  console.log("x is greater than 3");
} else {
  console.log("x is less than or equal to 3");
}
```

在此示例中，使用 `>` 操作符将 `x` 变量与数字 `3` 进行比较。如果 `x` 大于 `3`，将执行第一块代码。否则，将执行第二块代码。
