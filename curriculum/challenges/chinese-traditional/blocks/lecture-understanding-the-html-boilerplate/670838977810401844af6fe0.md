# --description--

讓我們來了解 `link` 元素，以及如何使用它來聯結外部樣式表。

`link` 元素用於連結到外部資源，例如樣式表和網站圖示。以下是使用 `link` 元素連結外部 CSS 檔案的基本語法：

```html
<link rel="stylesheet" href="./styles.css" />
```

`rel` 屬性用來指定連結資源與 HTML 文件之間的關係。在這種情況下，我們需要指定該連結資源是 `stylesheet`。

將你的 HTML 和 CSS 分開放在不同的檔案中被視為最佳實務。開發者會使用 `link` 元素來引入外部 CSS 檔案，而不是將所有內容寫在 HTML 文件中。

`href` 屬性用於指定外部資源的 URL 位置。 

範例中 `dot` 後接斜線告訴電腦在目前的資料夾或目錄中尋找 `styles.css` 檔案。

`link` 元素應該放置在 `head` 元素內，如以下範例所示：

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Examples of the link element</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

你經常會在專業的程式碼庫中看到多個 `link` 元素，這些元素連結到不同的樣式表、字型和圖示。以下是一個使用 `link` 元素連結到名為 *Playwright Cuba* 的外部 Google 字型的範例：

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Playwrite+CU:wght@100..400&display=swap"
  rel="stylesheet"
/>
```

Google Fonts 是一組免費且開放原始碼的訂製字型，你可以在任何專案中使用。你可以選擇想要使用的字型，Google 將為你提供必要的 HTML 和 CSS 程式碼。在此範例中，`rel` 屬性的 `preconnect` 值告訴瀏覽器要對 `href` 屬性中指定的值建立早期連線。這樣做是為了加快這些外部資源的載入時間。

另一個 `link` 元素的常見使用案例是連結到圖示。以下是一個連結到 favicon 的範例：

```html
<link rel="icon" href="favicon.ico" />
```

favicon 是 favorite icon（最愛圖示）的縮寫，通常是在瀏覽器分頁標題旁邊顯示的小圖示。許多網站會使用 favicon 來顯示他們的品牌圖示。
