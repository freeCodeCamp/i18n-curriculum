# --description--

`Security sweeps` são inspeções ou buscas detalhadas realizadas em uma área ou edifício, geralmente para fins de segurança e proteção.Eles envolvem verificar quaisquer riscos potenciais, acesso não autorizado ou itens perigosos.Por exemplo:

`Security officers perform security sweeps to find any hidden dangers.` - Isso significa que os oficiais de segurança verificam cuidadosamente a área para ver se há algo perigoso que não seja fácil de ver.
