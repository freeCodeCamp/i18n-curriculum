# --explanation--

Wenn Sie gefragt werden, warum Sie Python als erste Sprache gewählt haben, ist es am besten, die spezifischen Vorteile von Python hervorzuheben.

Python ist bekannt dafür, anfängerfreundlich zu sein, mit einer einfachen Syntax, die leicht zu lesen und zu lernen ist. Außerdem verfügt Python über eine großartige Sammlung von Bibliotheken und Frameworks, was es zu einer praktischen Wahl für Anfänger macht.

Andererseits erklärt `I've always wanted to be a developer` nicht, warum Sie sich für Python entschieden haben. Es konzentriert sich auf den Wunsch, Entwickler zu werden, liefert aber keine Python-spezifische Begründung.
