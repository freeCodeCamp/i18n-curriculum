# --explanation--

Unapounda swali kwa kutumia vitenzi vya modal, muundo ni: kitenzi cha modal + kiwakilishi + kitenzi katika hali ya msingi + (kitu kinachotendeka)? Kwa mfano:

- `Can we complete the project by Friday?` - Kuuliza kama inawezekana kumaliza mradi ifikapo Ijumaa.

- `Should we update the software to the latest version?` - Kuomba ushauri au pendekezo kama kusahihisha programu.

Katika jibu sahihi, `can` imewekwa kabla ya kiwakilishi (`you`), na kitenzi (`explain`) kiko katika hali yake ya msingi.
