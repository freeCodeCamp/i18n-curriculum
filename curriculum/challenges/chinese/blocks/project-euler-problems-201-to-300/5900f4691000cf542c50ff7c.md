# --description--

一个小孩有一个由四十块拼图组成的“数字毛毛虫”，每块拼图上都有一个数字，拼接成一条线时，会依次显示数字 1 到 40。

每天晚上，孩子的父亲必须拿起散落在游戏室的毛毛虫的碎片。 他随机拿起碎片并按正确顺序放置。

当毛虫以这种方式建立时，它形成了逐渐融合在一起的不同部分。 段的数量从零开始（没有放置件），通常增加到大约十一或十二，然后在完成单个段（放置所有件）之前往往再次下降。

例如：

| Piece Placed | Segments So Far |
| -------------|-----------------|
|       12     |        1        |
|        4     |        2        |
|       29     |        3        |
|        6     |        4        |
|       34     |        5        |
|        5     |        4        |
|       35     |        4        |
|       …      |        …        |

设 $M$ 为毛毛虫随机整理过程中遇到的最大段数。对于一个由十个部分组成的毛毛虫，每个 $M$ 的可能性数量为

| M | Possibilities |
|---|---------------|
| 1 |      512      |
| 2 |     250912    |
| 3 |    1815264    |
| 4 |    1418112    |
| 5 |     144000    |

所以 $M$ 最可能的值是 3，平均值是 $\frac{385\\,643}{113\\,400} = 3.400732$，四舍五入到六位小数。

四十节毛毛虫的 $M$ 最可能的值是 11；但 $M$ 的平均值是多少？请将你的答案四舍五入到六位小数。
