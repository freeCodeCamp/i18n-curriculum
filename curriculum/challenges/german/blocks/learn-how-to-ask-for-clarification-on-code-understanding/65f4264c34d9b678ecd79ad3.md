# --description--

`Uncertain` bedeutet, sich bei etwas nicht sicher zu sein, wie ein unsicheres Gefühl bei einer Entscheidung. Das Gegenteil ist `certain`, was bedeutet, sicher zu sein. Zum Beispiel bedeutet `I am certain this is the right way`, sich über die Richtung sicher zu sein.

`Achieve` bedeutet, nach Anstrengung ein Ziel erfolgreich zu erreichen, wie zum Beispiel `achieve a good result in a test`. Im Programmieren bezieht sich `achieve` oft auf die Ziele, die ein Programmierer mit seinem Code erreichen möchte.
