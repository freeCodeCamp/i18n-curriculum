# --description--

このタスクでは、マテオが感謝された後に丁寧に返答するのを聞きます。彼のフレーズは、スペイン語で感謝の気持ちを認める標準的な言い方です。

「どういたしまして」を表す他の表現もあります。以下はその例です：

- `De nada` - どういたしまして。（フォーマルおよびインフォーマル）
- `No hay de qué` - 感謝されることは何もありません。（よりフォーマルまたは伝統的）
- `Con gusto` - 喜んで。（ラテンアメリカで一般的）
