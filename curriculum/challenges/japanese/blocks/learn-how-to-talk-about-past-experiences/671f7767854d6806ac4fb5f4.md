# --explanation--

`Miss` は何かに参加し損なうことを意味します。例えば：

`I missed the meeting because I was stuck in traffic.` - これは交通渋滞に長く巻き込まれてしまい、そのために仕事に遅れたことを意味します。

`design sprint` は、通常数日間の短期間でチームがデザインの問題を解決したりプロトタイプを開発したりする、時間制限のある協力的なプロセスです。例えば：

`The team completed a design sprint to test the new user interface concept.` これはチームがこの協力的なプロセスに参加してコンセプトをテストしたことを意味します。

この文脈では、マリアはデザインスプリントワークショップに積極的に参加していたため、サイバーセキュリティワークショップに参加できませんでした。
