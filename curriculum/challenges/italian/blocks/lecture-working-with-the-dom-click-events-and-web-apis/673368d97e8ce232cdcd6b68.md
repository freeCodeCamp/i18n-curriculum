# --interactive--

Ci saranno momenti in cui dovrai aggiungere o rimuovere nodi dal DOM e ci sono un paio di Web API che puoi usare.

In questa lezione, copriremo i metodi `appendChild()` e `removeChild()`.

Il metodo `appendChild()` serve per aggiungere un nodo alla fine della lista dei figli di un nodo genitore specificato.

Ecco la sintassi base per il metodo `appendChild()`:

```js
parentNode.appendChild(newNode);
```

Diamo un’occhiata a un esempio per capire meglio come usare il metodo `appendChild()`.

In questo esempio, abbiamo un elemento lista non ordinata con un ID `desserts` dentro l’HTML. Questa lista contiene due elementi lista `Cake` e `Pie`:

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
```

Per consultare quella lista dentro il file JavaScript, possiamo usare il metodo `getElementById()`:

```js
const dessertsList = document.getElementById("desserts");
```

Abbiamo una variabile chiamata `dessertsList` che archivia il riferimento all’elemento `ul` con l’ID `desserts`.

Poi dobbiamo creare un nuovo elemento lista usando il metodo `createElement()`:

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");
```

Il codice seguente creerà un nuovo elemento lista e lo archivierà in una variabile chiamata `listItem`.

Per aggiungere quel nodo alla fine della lista dei figli dell’elemento `ul`, possiamo usare il metodo `appendChild()`:

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

dessertsList.appendChild(listItem);
```

Se esegui questo codice, vedrai che un nuovo elemento lista viene aggiunto alla fine della lista dei figli dell’elemento `ul`.

:::interactive_editor

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
<script src="index.js"></script>
```

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

dessertsList.appendChild(listItem);
```

:::

Il problema è che il nuovo elemento lista è vuoto. Per aggiungere contenuto testuale al nuovo elemento lista, puoi usare la proprietà `textContent`:

:::interactive_editor

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
<script src="index.js"></script>
```

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

listItem.textContent = "Cookies";
dessertsList.appendChild(listItem);
```

:::

Ora la lista mostrerà `Cake`, `Pie` e `Cookies`.

Per rimuovere un nodo dal DOM, puoi usare il metodo `removeChild()`.

Ecco la sintassi base per il metodo `removeChild()`:

```js
parentNode.removeChild(childNode);
```

Diamo un’occhiata a un esempio in cui vogliamo rimuovere l’ultimo elemento paragrafo da questo elemento `section`:

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
```

Possiamo consultare l’elemento `section` dentro il file JavaScript usando il metodo `getElementById()`:

```js
const sectionEl = document.getElementById("example-section");
```

Una volta che abbiamo il riferimento all’elemento `section`, possiamo accedere all’ultimo paragrafo usando il metodo `querySelector()`:

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");
```

Stiamo usando la pseudo-classe `:last-of-type` per selezionare l’ultimo elemento paragrafo dentro l’elemento `section`.

Ora che abbiamo i nodi genitore e figlio, possiamo rimuovere l’ultimo elemento paragrafo dall’elemento `section` usando il metodo `removeChild()`:

:::interactive_editor

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
<script src="index.js"></script>
```

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");

sectionEl.removeChild(lastParagraph);
```

:::

Quindi, quando potresti voler aggiungere o rimuovere nodi dal DOM?

Se stai lavorando con contenuti dinamici, potresti dover aggiungere o rimuovere nodi dal DOM. Esempi reali includono carrelli della spesa, liste di cose da fare e feed dei social media.
