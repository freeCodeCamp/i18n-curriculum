# --description--

片語 `some of these` 用來指涉群組中的特定零件或較大集合中項目的子集合。當你想提及群組中的幾個項目而不逐一列出時，經常會使用此片語。

例如，`Let's try some of these cookies.` 講者建議嘗試從較大的批次中取幾塊餅乾。
