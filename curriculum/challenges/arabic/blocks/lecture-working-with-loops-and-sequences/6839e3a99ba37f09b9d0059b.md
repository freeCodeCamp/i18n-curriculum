# --description--

الصف المرتب هو نوع بيانات في Python يُستخدم لإنشاء متتالية مرتبة من القيم. يمكن أن تحتوي الصفوف المرتبة على مجموعة مختلطة من أنواع البيانات مثل هذا:

```py
developer = ('Alice', 34, 'Rust Developer')
```

الصفوف المرتبة تشبه القوائم، لكن بينما القوائم هي نوع بيانات قابل للتغيير، فإن الصفوف المرتبة غير قابلة للتغيير. هذا يعني أن العناصر في الصف المرتب لا يمكن تعديلها بعد إنشائه.

إذا حاولت تحديث أحد العناصر في الصف المرتب، ستحصل على `TypeError`:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: 'tuple' object does not support item assignment
"""
```

للوصول إلى عنصر من صف مرتب، يمكنك استخدام ترميز الأقواس ورقم الفهرس:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

إذا كنت بحاجة إلى الوصول إلى عناصر بدءًا من نهاية الصف المرتب، يمكنك استخدام الفهرسة السالبة. إليك مثالًا على استخدام فهرس سالب للوصول إلى العنصر قبل الأخير في الصف المرتب:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

إذا حاولت تمرير رقم فهرس يتجاوز أو يساوي طول الصف المرتب، فستتلقى `IndexError` مثل هذا:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list index out of range
"""
```

طريقة أخرى لإنشاء صف مرتب هي باستخدام دالة البناء `tuple()` مثل هذا:

```py
developer = 'Jessica'
tuple(developer) # ('J', 'e', 's', 's', 'i', 'c', 'a')
```

يمكنك تمرير متكررات مختلفة مثل السلاسل النصية والقوائم وحتى صفوف مرتبة أخرى إلى دالة البناء `tuple()`.

للتحقق مما إذا كان عنصر موجودًا في صف مرتب، يمكنك استخدام الكلمة المفتاحية `in` مثل هذا:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

يمكنك أيضًا فك تجميع العناصر من صف مرتب تمامًا كما فعلت مع القوائم:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

في هذا المثال، القيمة في `name` هي `'Alice'`، والقيمة في `age` هي `34`، والقيمة في `job` هي `'Rust Developer'`.

إذا كنت بحاجة إلى جمع أي عناصر متبقية من صف مرتب، يمكنك استخدام عامل النجمة (`*`) مثل هذا:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

هنا، القيمة في `name` هي `'Alice'`، و`rest` هي قائمة تتألف من الرقم `34` والسلسلة النصية `'Rust Developer'`.

تمامًا كما في القائمة، يمكنك استخدام عامل القطع على صف مرتب لاستخراج جزء منه. إليك مثالًا لاستخراج العناصر `'pie'` و`'cookies'` إلى صف مرتب منفصل:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

تذكر أن الرقم الأول يمثل فهرس البداية للاستخراج بينما الرقم الثاني يمثل فهرس النهاية. لكن لاحظ أن العنصر عند فهرس النهاية غير مشمول في الصف المرتب المستخرج.

إذا كنت بحاجة إلى حذف عنصر من صف مرتب، فهذا غير ممكن لأن الصفوف المرتبة غير قابلة للتغيير. لذا هذا المثال سينتج عنه خطأ:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

متى قد تستخدم صف مرتب بدلًا من قائمة؟

إذا كنت بحاجة إلى مجموعة ديناميكية من العناصر حيث يمكنك الإضافة والحذف والتحديث، فعليك استخدام قائمة. إذا كنت تعلم أنك تتعامل مع مجموعة ثابتة وغير قابلة للتغيير من البيانات، فعليك استخدام صف مرتب.

في الدرس التالي، سنلقي نظرة على بعض الطرق الشائعة التي ستستخدمها عند العمل مع الصفوف المرتبة.
