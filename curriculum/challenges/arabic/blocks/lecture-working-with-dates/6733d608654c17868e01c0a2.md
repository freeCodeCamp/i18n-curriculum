# --interactive--

في الدرس السابق، تعلمت كيفية العمل مع كائن `Date` في JavaScript. لكن هناك عدة طرق مختلفة لتنسيق التواريخ في JavaScript.

في هذا الدرس، سنلقي نظرة على كيفية العمل مع الطرق `toISOString()`، و`toString()`، و`toLocaleDateString()` لتنسيق التواريخ في JavaScript.

قبل أن نطلع على الطرق المختلفة، دعنا نراجع أولًا شكل كائن `Date`:

:::interactive_editor

```js
const date = new Date();
console.log(date);
```

:::

عند تسجيل كائن `date` في وحدة التحكم، سترى التاريخ والوقت الحاليين بناءً على إعدادات نظام المستخدم. إليك مثالًا على المخرجات التي قد تراها:

```js
Sun Sep 29 2024 19:45:37 GMT-0700 (Pacific Daylight Time)
```

إذا استخدمت الطريقة `toString()` على كائن `date`، سترى نفس المخرجات كما في الأعلى.

لتنسيق التاريخ بصيغة ISO الموسعة (ISO 8601)، يمكنك استخدام الطريقة `toISOString()` كما يلي:

:::interactive_editor

```js
const date = new Date();
console.log(date.toISOString());
```

:::

ISO 8601 هو معيار دولي لتمثيل التواريخ والأوقات. الصيغة هي `YYYY-MM-DDTHH:mm:ss.sssZ`.

إليك مثالًا على المخرجات التي قد تراها:

```js
2024-09-30T02:47:20.292Z
```

طريقة أخرى لتنسيق التاريخ هي استخدام الطريقة `toLocaleDateString()`. تتيح لك هذه الطريقة تنسيق التاريخ بناءً على لغة المستخدم. إليك الصيغة النحوية الأساسية لاستخدام الطريقة `toLocaleDateString()`:

:::interactive_editor

```js
const date = new Date();
console.log(date.toLocaleDateString());
```

:::

إليك مثالًا على المخرجات التي قد تراها:

```js
9/29/2024
```

تقبل الطريقة `toLocaleDateString()` معلمتين اختياريتين: `locales` و`options`.

المعلمة `locales` هي سلسلة نصية تمثل اللغة المراد استخدامها. على سبيل المثال، يمكنك تمرير `en-US` للإنجليزية (الولايات المتحدة) أو `fr-FR` للفرنسية (فرنسا). إذا لم تمرر معلمة `locales`، تُستخدم اللغة الافتراضية. إليك مثالًا على كيفية استخدام المعلمة `locales`:

:::interactive_editor

```js
const date = new Date();
console.log(date.toLocaleDateString("fr-FR"));
```

:::

المعلمة الاختيارية الثانية هي `options`. هذه المعلمة كائن يسمح لك بتحديد صيغة سلسلة التاريخ. إليك مثالًا على كيفية استخدام المعلمة `options`:

:::interactive_editor

```js
const date = new Date();
const options = {
  weekday: "long",
  year: "numeric",
  month: "long",
  day: "numeric",
};
console.log(date.toLocaleDateString("en-GB", options));
```

:::

في المثال أعلاه، حددنا الخيارات لتكون باللغة الإنجليزية (بريطانيا العظمى) ولتتضمن اسم اليوم الكامل، والسنة، والشهر، واليوم.

إليك مثالًا على المخرجات التي قد تراها:

```js
Sunday, September 29, 2024
```

في هذا الدرس، غطينا فقط بعض الطرق لتنسيق التواريخ في JavaScript. هناك العديد من الطرق والمكتبات الأخرى المتاحة لمساعدتك في تنسيق التواريخ في JavaScript. لكن الطريقتين `toISOString()` و`toLocaleDateString()` هما نقطة انطلاق جيدة لتنسيق التواريخ في JavaScript.
