# --description--

Ora passeremo a un'altra struttura dati ad albero, l'heap binario. Un heap binario è un albero binario parzialmente ordinato che soddisfa la proprietà di heap. La proprietà di heap specifica una relazione tra nodi genitori e figli. Puoi avere un heap massimo, in cui tutti i nodi genitori sono maggiori o uguali ai loro nodi figli, oppure un heap minimo, in cui vale il contrario. Gli heap binari sono anche alberi binari completi. Questo significa che tutti i livelli dell'albero sono completamente riempiti e se l'ultimo livello è parzialmente riempito, lo è da sinistra a destra.

Sebbene gli heap binari possano essere implementati come strutture ad albero con nodi che contengono riferimenti a sinistra e a destra, l'ordinamento parziale secondo la proprietà di heap ci permette di rappresentare l'heap con un array. La relazione genitore-figlio è ciò che ci interessa e con semplici calcoli aritmetici possiamo calcolare i figli di qualsiasi genitore e il genitore di qualsiasi nodo figlio.

Ad esempio, considera questa rappresentazione in array di un heap binario minimo:

```js
[ 6, 22, 30, 37, 63, 48, 42, 76 ]
```

Il nodo radice è il primo elemento, `6`. I suoi figli sono `22` e `30`. Se osserviamo la relazione tra gli indici dell'array di questi valori, per l'indice `i` i figli sono `2 * i + 1` e `2 * i + 2`. Allo stesso modo, l'elemento all'indice `0` è il genitore di questi due figli agli indici `1` e `2`. Più in generale, possiamo trovare il genitore di un nodo a qualsiasi indice con la seguente formula: `Math.floor((i - 1) / 2)`. Questi pattern rimangono validi man mano che l'albero binario cresce di dimensioni. Infine, possiamo fare una piccola regolazione per rendere questa aritmetica ancora più semplice saltando il primo elemento nell'array. Facendo questo si crea la seguente relazione per qualsiasi elemento a un dato indice `i`:

Esempio di rappresentazione in array:

```js
[ null, 6, 22, 30, 37, 63, 48, 42, 76 ]
```

Il figlio sinistro di un elemento: `i * 2`

Il figlio destro di un elemento: `i * 2 + 1`

Il genitore di un elemento: `Math.floor(i / 2)`

Una volta che hai capito la matematica, usare una rappresentazione in array è molto utile perché le posizioni dei nodi possono essere determinate rapidamente con questa aritmetica e l'uso della memoria si riduce perché non serve mantenere riferimenti ai nodi figli.

# --instructions--

Istruzioni: Qui creeremo un heap massimo. Inizia semplicemente creando un metodo `insert` che aggiunge elementi al nostro heap. Durante l'inserimento, è importante mantenere sempre la proprietà di heap. Per un heap massimo questo significa che l'elemento radice dovrebbe avere sempre il valore più grande nell'albero e tutti i nodi genitori dovrebbero essere maggiori dei loro figli. Per un'implementazione di heap con array, questo si realizza tipicamente in tre passaggi:

<ol>
  <li>Aggiungi il nuovo elemento alla fine dell'array.</li>
  <li>Se l'elemento è più grande del suo genitore, scambiali.</li>
  <li>Continua a scambiare finché il nuovo elemento è più piccolo del suo genitore o finché non raggiungi la radice dell'albero.</li>
</ol>

Infine, aggiungi un metodo `print` che restituisce un array di tutti gli elementi che sono stati aggiunti all'heap.

```js
var MaxHeap = function() {
    // Only change code below this line
    this.heap = [];
    this.parent = index => {
      return Math.floor((index - 1) / 2);
    }
    this.insert = element => {
      this.heap.push(element);
      this.heapifyUp(this.heap.length - 1);
    }
    this.heapifyUp = index => {
      let currentIndex = index,
      parentIndex = this.parent(currentIndex);
      while (currentIndex > 0 && this.heap[currentIndex] > this.heap[parentIndex]) {
        this.swap(currentIndex, parentIndex);
        currentIndex = parentIndex;
        parentIndex = this.parent(parentIndex);
      }
    }
    this.swap = (index1, index2) => {
      [this.heap[index1], this.heap[index2]] = [this.heap[index2], this.heap[index1]];
    }
    this.print = () => {
      return this.heap;
    }
    // Only change code above this line
};
```
