# --explanation--

`Weak passwords` sono password facili da indovinare o violare. Per esempio:

`123456 and qwerty are weak passwords.` - Una password forte dovrebbe essere lunga e contenere lettere, numeri e simboli.

`Shared credentials` sono dati di accesso (nome utente e password) usati da più persone. Per esempio:

`Using shared credentials increases security risks.` - Condividere le password può portare a consultazioni non autorizzate se finiscono nelle mani sbagliate.
