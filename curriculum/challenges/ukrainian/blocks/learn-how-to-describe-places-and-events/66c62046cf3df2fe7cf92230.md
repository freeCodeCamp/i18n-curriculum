# --explanation--

Щоб знайти відповідь, зверніть увагу на слово `but`, яке сигналізує про протиставлення або виняток.

Після `but` Сара згадує `they're not visible`, а потім умову `if the project isn't linked to Git.`.

Це говорить про те, що інструменти приховані, якщо проєкт не підключено до Git.

Фраза `if the project isn't` є ключовим маркером, який вводить умову, за якої інструменти Git не з’являться.

У реченні Сари вона використовує `they're not`, щоб пояснити, що інструменти Git не видимі, і `isn't`, щоб сказати, що проєкт не пов’язано з Git.
