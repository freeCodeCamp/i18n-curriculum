# --description--

Como todos sabemos, la ecuación $x^2 = -1$ no tiene soluciones para valores reales de $x$.

Sin embargo, si introducimos el número imaginario $i$, esta ecuación tiene dos soluciones: $x = i$ y $x = -i$.

Si vamos un paso más allá, la ecuación ${(x - 3)}^2 = -4$ tiene dos soluciones complejas: $x = 3 + 2i$ y $x = 3 - 2i$, que se llaman conjugados complejos el uno del otro.

Los números de la forma $a + bi$ se llaman números complejos.

En general, $a + bi$ y $a − bi$ son conjugados complejos el uno del otro. Un entero gaussiano es un número complejo $a + bi$ tal que tanto $a$ como $b$ son enteros.

Los enteros regulares también son enteros gaussianos (con $b = 0$).

Para distinguirlos de los enteros gaussianos con $b ≠ 0$ llamamos a esos enteros "enteros racionales."

Un entero gaussiano se llama divisor de un entero racional $n$ si el resultado también es un entero gaussiano.

Si por ejemplo dividimos 5 entre $1 + 2i$ podemos simplificar de la siguiente manera:

Multiplica el numerador y el denominador por el conjugado complejo de $1 + 2i$: $1 − 2i$.

El resultado es:

$$\frac{5}{1 + 2i} = \frac{5}{1 + 2i} \frac{1 - 2i}{1 - 2i} = \frac{5(1 - 2i)}{1 - {(2i)}^2} = \frac{5(1 - 2i)}{1 - (-4)} = \frac{5(1 - 2i)}{5} = 1 - 2i$$

Entonces $1 + 2i$ es un divisor de 5.

Ten en cuenta que $1 + i$ no es un divisor de 5 porque:

$$\frac{5}{1 + i} = \frac{5}{2} - \frac{5}{2}i$$

Ten en cuenta también que si el Entero Gaussiano ($a + bi$) es un divisor de un entero racional $n$, entonces su conjugado complejo ($a − bi$) también es un divisor de $n$. De hecho, 5 tiene seis divisores cuya parte real es positiva: {1, 1 + 2i, 1 − 2i, 2 + i, 2 − i, 5}.

La siguiente es una tabla de todos los divisores para los primeros cinco enteros racionales positivos:

Para divisores con partes reales positivas, entonces, tenemos: $\displaystyle\sum_{n=1}^5 s(n) = 35$.

Para $1 ≤ n ≤ {10}^5$, $\displaystyle\sum_{n = 1}^{{10}^5} s(n) = 17924657155$.

¿Qué es $\displaystyle\sum_{n=1}^{{10}^8} s(n)$?
