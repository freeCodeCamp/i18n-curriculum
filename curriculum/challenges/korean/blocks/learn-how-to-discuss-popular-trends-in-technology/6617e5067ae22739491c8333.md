# --description--

`chatbot`는 인간 사용자와의 대화를 시뮬레이션하도록 설계된 컴퓨터 프로그램입니다. 사람 대신 컴퓨터가 메시지에 답변합니다. 기업들은 웹사이트에서 질문에 답하거나 정보를 제공하기 위해 챗봇을 사용합니다. 그래서 온라인 쇼핑 중에 작은 채팅 창이 나타나면, 그것은 도움을 줄 준비가 된 챗봇입니다!
