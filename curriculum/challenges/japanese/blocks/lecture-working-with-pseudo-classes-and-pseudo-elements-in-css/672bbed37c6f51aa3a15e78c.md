# --interactive--

CSSで最も興味深い側面の一つが疑似要素の使い方です。この文脈での「疑似」とは「実際のものではない」という意味で、疑似要素は実際のHTML要素に直接対応しない仮想的または合成的な要素です。これにより、特定の要素の一部をスタイル設定したり、追加のHTMLを加えずにコンテンツを挿入したりできます。

疑似要素を適用するには、元の要素のセレクターに二重コロン（`::`）を付けます。セレクターはクラスやIDなど任意のタイプが使えます。疑似要素の基本的な構文は次のようになります。

```css
selector::pseudo-element {
  property: value;
}
```

この二重コロンが、単一コロンを使う疑似クラスと疑似要素を区別するポイントです。

疑似要素は要素のコンテンツの特定部分をスタイル設定したり、その前後にコンテンツを挿入したりできますが、単独で存在することはできません。疑似要素が付けられる要素は「起源要素」と呼ばれます。

まず、`::before`と`::after`疑似要素の例を見てみましょう。名前の通り、`::before`は要素のコンテンツの直前にコンテンツを挿入し、`::after`はその後に挿入します。

ここにボタン要素の例があります。CSSでは絶対位置指定と`::before`疑似要素を使って、ボタンの`Learn More`テキストの前に星を追加します。絶対位置指定については今後のレッスンで詳しく学びます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: lightseagreen;
  color: white;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
  position: relative;
}

.cta-button::before {
  content: "⭐";
  position: absolute;
  left: 3px;
  top: 8px;
  font-size: 0.75rem;
}
```

:::

`content`プロパティはボタンテキストの前に追加したいコンテンツを表します。この例では星を追加しています。コンテンツを挿入するだけでなく、スタイル設定もできることに気づくでしょう。次は同じボタンに`::after`疑似要素を使った例です。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: '➡️';
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}
```

:::

`::after`疑似要素では、`transition`プロパティを使って0.3秒かけてイージング効果で変化をアニメーション化し、突然ではなく滑らかで徐々に変化するようにしています。`transition`プロパティについては今後のレッスンで学びます。

また、`::before`と`::after`疑似要素で挿入したコンテンツに疑似クラスを付けることもできます。例えば、ホバー状態のコンテンツです。例を見てみましょう。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: "➡️";
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}

.cta-button:hover::after {
  transform: translateX(2px);
}
```

:::

ホバー状態の`transform: translateX(2px)`では、ユーザーがボタンにホバーするとコンテンツが`2px`だけ右に押されます。`::after`自体のtransitionプロパティがこの動作に`0.3s`かかることを保証しています。

`transform`プロパティは要素を回転、傾斜、拡大縮小、または特定方向に移動させるためのものです。これについても今後のレッスンで学びます。

次の例では`::first-letter`疑似要素を見ていきます。`::first-letter`疑似要素は要素のコンテンツの最初の文字を対象にしてスタイル設定できます。段落テキストの例です。最初の文字をスタイル設定したい場合、`::first-letter`疑似要素を次のように使います。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>freeCodeCamp lets you learn to code without having to pay.</p>
```

```css
p::first-letter {
  font-size: 4rem;
}
```

:::

最後の例では`::marker`疑似要素を見ます。これはリスト項目のマーカー、箇条書きや番号を選択してスタイル設定できます。`::marker`疑似要素はリストマーカーをカスタマイズしてウェブサイトのブランドアイデンティティを強化する方法を提供します。

番号なしリストと番号付きリストの例です。リスト項目のマーカーの色やサイズを変えたい場合、`::marker`疑似要素を次のように使えます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<ul>
  <li>Unordered list item 1</li>
  <li>Unordered list item 2</li>
  <li>Unordered list item 3</li>
  <li>Unordered list item 4</li>
</ul>

<ol>
  <li>Ordered list item 1</li>
  <li>Ordered list item 2</li>
  <li>Ordered list item 3</li>
  <li>Ordered list item 4</li>
</ol>
```

```css
li::marker {
  color: crimson;
  font-size: 1.5em;
  font-weight: bold;
}
```

:::

このレッスンではいくつかの疑似要素だけを扱いましたが、`::placeholder`、`::spelling-error`、`::selection`など他にもたくさんあります。ぜひ自分で探求してみてください。
