# --description--

如你從先前課程所回想，`not` 運算子是一元邏輯運算子，用來否定一個表達式：

```py
print(not True) # False
print(not False) # True
```

使用 `not` 運算子修正 `if` 述語的條件，使你的函式在 `shift` *不是* 整數時傳回 `Shift must be an integer value.`。
