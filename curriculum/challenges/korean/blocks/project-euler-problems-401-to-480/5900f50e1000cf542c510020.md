# --description--

$ n $개의 칸이 일렬로 있고, 가장 왼쪽 칸에 개구리가 있습니다. 개구리는 연속 점프로 가장 오른쪽 칸까지 갔다가 다시 가장 왼쪽 칸으로 돌아옵니다. 나가는 길에는 오른쪽으로 한 칸, 두 칸 또는 세 칸씩 점프하고, 돌아오는 길에는 비슷한 방식으로 왼쪽으로 점프합니다. 칸 밖으로 점프할 수 없습니다. 왕복 여행을 $ m $번 반복합니다.

$F(m, n)$을 개구리가 최대 한 칸만 방문하지 않고 여행할 수 있는 경우의 수라고 합시다.

예를 들어, $F(1, 3) = 4$, $F(1, 4) = 15$, $F(1, 5) = 46$, $F(2, 3) = 16$ 그리고 $F(2, 100)\bmod {10}^9 = 429\\,619\\,151$입니다.

$F(10, {10}^{12})$의 마지막 9자리를 구하세요.
