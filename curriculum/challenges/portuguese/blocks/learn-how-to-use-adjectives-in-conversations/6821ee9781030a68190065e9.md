### --feedback--

# --explanation--

`Need to` é usado quando algo é necessário.Por exemplo:  

`We need to finish the report by Friday.` – É importante completar o relatório antes de sexta-feira.

`Compare` significa examinar coisas para encontrar semelhanças ou diferenças.Por exemplo:  

`Let's compare both designs before we choose one.` – A equipe quer ver o que é semelhante ou diferente nos designs.

`Versions` referem-se a diferentes formas ou atualizações da mesma coisa.Por exemplo:  

`The latest version of the app has a new feature.` – Isso significa que uma versão mais recente do app inclui algo novo.
