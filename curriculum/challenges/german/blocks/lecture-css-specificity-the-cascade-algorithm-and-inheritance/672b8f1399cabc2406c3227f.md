# --interactive--

Das `!important`-Keyword in CSS wird verwendet, um einer Stilregel die höchste Priorität zu geben, sodass sie alle anderen Deklarationen für eine Eigenschaft überschreibt.

Wenn es verwendet wird, zwingt es den Browser, den angegebenen Stil anzuwenden, unabhängig von der Spezifität anderer Selektoren.

Angenommen, Sie haben ein HTML-Absatz-Element mit inline-Stilen wie diesem:

:::interactive_editor

```html
<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

:::

In diesem Beispiel hat das Absatz-Element eine Hintergrundfarbe, die auf `lightblue` gesetzt ist, und eine Textfarbe, die auf `black` gesetzt ist.

Die CSS-Datei wendet die folgenden Stile auf das Absatz-Element an:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black;
  color: white;
}
```

:::

Da inline-Stile eine höhere Priorität als Klassen-, ID- und Typselektoren haben, werden die schwarze Hintergrundfarbe und die weiße Textfarbe nicht auf dieses Absatz-Element angewendet.

Um diese inline-Stile zu überschreiben, können Sie das `!important`-Keyword so verwenden:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black !important;
  color: white !important;
}
```

:::

Das `!important`-Keyword wird nach dem CSS-Wert und vor dem Semikolon verwendet.

Jetzt werden dem Absatz-Element diese schwarzen und weißen Farben zugewiesen, weil die inline-Stile durch die Verwendung des `!important`-Keywords überschrieben werden.

Das `!important`-Keyword in CSS wird verwendet, um einer Stilregel die höchste Priorität zu geben und effektiv andere Deklarationen zu überschreiben, einschließlich solcher mit höherer Spezifität und inline-Stilen.

Das `!important`-Keyword ändert jedoch nicht die Spezifität des CSS-Selektors selbst. Es stellt einfach sicher, dass die Regel mit `!important` angewendet wird, selbst wenn es andere widersprüchliche Regeln mit höherer Spezifität gibt.

Ein weiterer geeigneter Anwendungsfall für das `!important`-Keyword ist das Überschreiben von Stilen aus Drittanbieter-Bibliotheken oder Frameworks, wenn Sie keine Kontrolle über das ursprüngliche CSS haben.

Das übermäßige Verwenden des `!important`-Keywords kann jedoch zu Schwierigkeiten bei der Wartung und beim Debugging Ihres CSS führen, da es die natürliche Kaskadierung der Stile unterbricht und zu unbeabsichtigten Folgen führen kann.

Daher ist es am besten, das `!important`-Keyword sparsam zu verwenden.
