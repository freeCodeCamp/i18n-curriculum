# --description--

Para hacer una pregunta usando `it is`, cambias el orden. En lugar de `it is`, se dice `is it`.  This is how you turn a statement into a question.

# --questions--

## --text--

¿Cuál de las siguientes opciones es un formulario correcto de una pregunta?

## --answers--

`It is lunchtime?`

### --feedback--

Esta sentence se puede entender, pero esta no es la manera común de form una pregunta.

---

`Lunchtime it is?`

### --feedback--

Este order no se usa comúnmente para preguntas.

---

`Is lunchtime it?`

### --feedback--

Este order es incorrecto para hacer preguntas.

---

`Is it lunchtime?`
