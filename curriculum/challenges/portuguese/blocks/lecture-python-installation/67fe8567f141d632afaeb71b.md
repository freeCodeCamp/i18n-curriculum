# --description--

Para todas as oficinas e laboratórios, você usará o editor de Python do freeCodeCamp. Mas é importante aprender a configurar o Python na sua máquina local.

A maneira mais fácil de instalar Python no Windows e Mac é baixar o instalador do site oficial do Python. Também vamos explicar como executar Python no Linux mais adiante nesta lição.

Vá para `https://www.python.org/` e passe o mouse sobre "Downloads". Um modal aparecerá mostrando a versão atual do Python para seu sistema operacional (SO).

Vamos explicar primeiro como instalar Python em um computador com macOS:

- Clique no botão que mostra a versão atual do Python (do modal anterior) e o arquivo de instalação `.pkg` começará a ser baixado automaticamente.
- Quando o instalador `.pkg` terminar de baixar, abra-o e clique em "Continuar" na janela que abrir.
- Continue clicando no botão "Continuar" até chegar na seção "Tipo de instalação". Lá, clique no botão "Instalar".
- Digite sua senha se necessário e inicie a instalação.
- Depois disso, você deve receber uma mensagem de parabéns dizendo que o Python foi instalado com sucesso.
- Clique no botão "Fechar" e pronto!

Você pode verificar a instalação abrindo o terminal e executando `python --version` ou `python3 --version`.

Também pode abrir o interpretador Python executando `python` ou `python3` no terminal. Um interpretador Python é o programa que lê o código Python, traduz para instruções que o computador entende e executa essas instruções.

Um <dfn>terminal</dfn> é uma interface baseada em texto que permite interagir com seu computador digitando comandos. Cada sistema operacional vem com um app de terminal padrão. No macOS, você pode usar o app Terminal. No Windows, pode usar o Prompt de Comando ou PowerShell. No Linux, cada ambiente de desktop tem seu próprio app de terminal padrão, como GNOME Terminal ou Konsole.

Note que, em alguns sistemas macOS e Linux mais antigos, `python` pode estar reservado para Python 2, enquanto `python3` é para Python 3 especificamente. Se você executar `python --version` e vir uma versão do Python 2 como `Python 2.7.18`, é possível que seu SO dependa de algum software escrito na versão antiga do Python. Nesse caso, você deve usar `python3` para executar seu código Python daqui para frente. Python 2 está em fim de vida e não deve ser usado para nenhum desenvolvimento novo.

Para instalar Python no Windows, siga estes passos:

- Vá para `https://www.python.org/` e passe o mouse sobre “Downloads“. Você verá um modal que diz "Download para Windows" e um botão para baixar a versão atual do Python.
- Clique no número da versão e o arquivo executável do Windows (`.exe`) começará a ser baixado automaticamente.
- Quando terminar de baixar o instalador do Python para Windows, dê um duplo clique nele e siga as instruções.
- Quando vir a opção `Add python.exe to Path`, marque essa opção e clique em `Install Now`. Isso facilitará as coisas para você depois.

Você pode verificar a instalação abrindo um shell de linha de comando como PowerShell e executando `python --version`. Também pode abrir o interpretador Python executando `python`.

Para Python no Linux, a maioria das distribuições principais como Ubuntu, Debian e Fedora já vem com Python.

Se algum desses comandos não mostrar uma versão do Python, você pode procurar um pacote de instalação para sua distribuição Linux em `https://www.python.org` ou pesquisar online a forma recomendada de instalar Python para sua distro.
