# --description--

격자점 집합 $S$가 타이타닉 집합이라고 불리려면, $S$에 있는 정확히 두 점을 지나는 직선이 존재해야 합니다.

타이타닉 집합의 예는 $S = \\{(0, 0), (0, 1), (0, 2), (1, 1), (2, 0), (1, 0)\\}$입니다. 여기서 (0, 1)과 (2, 0)을 지나는 직선은 $S$의 다른 어떤 점도 지나지 않습니다.

반면에, 집합 {(0, 0), (1, 1), (2, 2), (4, 4)}는 타이타닉 집합이 아닙니다. 이 집합의 어떤 두 점을 지나는 직선도 나머지 두 점을 모두 지나기 때문입니다.

양의 정수 $N$에 대해, 모든 점 ($x$, $y$)가 $0 ≤ x$, $y ≤ N$을 만족하는 타이타닉 집합 $S$의 개수를 $T(N)$이라 합시다. $T(1) = 11$, $T(2) = 494$, $T(4) = 33\\,554\\,178$, $T(111)\bmod {10}^8 = 13\\,500\\,401$, 그리고 $T({10}^5)\bmod {10}^8 = 63\\,259\\,062$임을 확인할 수 있습니다.

$T({10}^{11})\bmod {10}^8$를 구하세요.
