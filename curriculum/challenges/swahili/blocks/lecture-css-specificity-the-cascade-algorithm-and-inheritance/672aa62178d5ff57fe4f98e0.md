# --interactive--

Kipaumbele cha CSS ni dhana ya msingi inayobainisha ni mitindo gani itatumika kwa kipengele wakati sheria nyingi zinaweza kutumika.

Kuelewa kipaumbele husaidia wasanidi programu kutatua migongano kati ya sheria tofauti za CSS na kuhakikisha mitindo inayotakiwa inatumika kwa usahihi kila wakati.

Kipaumbele cha CSS huhesabiwa kulingana na aina ya vichaguzi vinavyotumika.

Kipaumbele cha juu zaidi kinatolewa kwa mitindo ndani ya mstari, ambayo hutumika moja kwa moja kwa kipengele kupitia sifa ya `style`.

Katika mfano huu, aya ya kwanza itakuwa nyekundu wakati vipengele vingine vya `p` vitakuwa buluu. Hii ni kwa sababu mitindo ndani ya mstari ina kipaumbele cha juu zaidi kuliko vichaguzi vya aina kama vile kichaguzi cha `p` kilichoonyeshwa katika faili ya `styles.css`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p style="color: red;">Red paragraph</p>
<p>Other paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p {
  color: blue;
}
```

:::

Baada ya hapo, vichaguzi vya ID vina kiwango cha juu cha kipaumbele.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Red paragraph</p>
<p>Other paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

p {
  color: blue;
}
```

:::

Kisha, vichaguzi vya darasa la vitu, vichaguzi vya sifa za kipengele, na darasa la bandia vinaingia kazini.

Mifano ni pamoja na vichaguzi vya darasa la vitu kama `.container` na `.button`, vichaguzi vya sifa kama `[type="text"]`, na darasa la bandia kama `:hover`. Hivi vina kiwango cha wastani cha kipaumbele.

**KUMBUKA**: Utajifunza zaidi kuhusu darasa la bandia katika mafundisho yajayo.

Katika mfano huu, aya ya kwanza itakuwa nyekundu kwa sababu kichaguzi cha `id` kina kipaumbele cha juu zaidi kuliko darasa la vitu na vichaguzi vya aina. Vipengele vya `.example-para` vitakuwa na rangi ya kijani wakati vipengele vingine vya aya vitakuwa na rangi ya buluu.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p class="example-para">Other paragraph</p>
<p class="example-para">Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

.example-para {
  color: green;
}

p {
  color: blue;
}
```

:::

Vichaguzi vya aina, kama `div` na `h1`, na vipengele vya bandia kama `::before` na `::after`, vina kipaumbele cha chini ikilinganishwa na makundi yaliyotangulia.

**KUMBUKA**: Utajifunza zaidi kuhusu vipengele vya bandia katika mafundisho yajayo.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p>Other paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

p {
  color: blue;
}
```

:::

Mwishowe, kichaguzi cha ulimwengu, kinachowakilishwa na nyota `*`, kina kipaumbele cha chini kabisa kati ya vyote.

Hapa kuna mfano wa kuweka rangi kwa vipengele vyote kuwa nyekundu kwa kutumia kichaguzi cha `*`. Hata hivyo, hutauoni vipengele vyovyote vyenye rangi nyekundu kwa sababu kuna vichaguzi vya ID na aina vinavyoghairi mitindo hiyo, jambo linaloonyesha kipaumbele kidogo cha kichaguzi cha `*`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p>Other paragraph</p>
<p>Yet another paragraph</p>
```

```css
* {
  color: red;
}

#para {
  color: green; 
}

p {
  color: blue;
}
```

:::

Thamani za kipaumbele huhesabiwa kama thamani ya sehemu nne `(a, b, c, d)`:

- `a`: Mitindo ndani ya mstari (`1` au `0`).
- `b`: Idadi ya vichaguzi vya ID.
- `c`: Idadi ya vichaguzi vya darasa la vitu, vichaguzi vya sifa za kipengele, na darasa la bandia.
- `d`: Idadi ya vichaguzi vya aina, vipengele vya bandia, na vichaguzi vya ulimwengu.

Kila sehemu ya thamani ya kipaumbele ina uzito tofauti:

- **Mitindo ndani ya mstari** (`a`) ina uzito wa juu zaidi, ikichangia thamani ya `1` kwa sehemu ya kwanza ya thamani ya kipaumbele.
    
- **Vichaguzi vya ID** (`b`) vina uzito unaofuata, ambapo kila ID huchangia `1` kwa sehemu ya pili ya thamani ya kipaumbele.
    
- **Vichaguzi vya darasa la vitu, vichaguzi vya sifa za kipengele, na darasa la bandia** (`c`) vina uzito wa wastani, ambapo kila moja huchangia `1` kwa sehemu ya tatu ya thamani ya kipaumbele.
    
- **Vichaguzi vya aina na vipengele vya bandia** (`d`) vina uzito wa chini zaidi, ambapo kila moja huchangia `1` kwa sehemu ya nne ya thamani ya kipaumbele.
    
- **Kichaguzi cha ulimwengu** (`*`): Kichaguzi cha ulimwengu huchangia `0` katika hesabu ya kipaumbele na hakibadilishi kipaumbele. Kuongeza kwake katika kichaguzi hakubadilishi thamani ya kipaumbele.

CSS ndani ya mstari ina kipaumbele cha juu zaidi kwa sababu hutumika moja kwa moja kwa kipengele. Inaghairi CSS ya ndani au ya nje. Thamani ya kipaumbele kwa mitindo ndani ya mstari ni `(1, 0, 0, 0)`.

Hapa kuna mfano mwingine wa mtindo ndani ya mstari:

:::interactive_editor

```html
<p style="color: red;">This text is red.</p>
```

:::

CSS ya ndani huandikwa ndani ya kipengele cha `style` katika sehemu ya `head` ya hati ya HTML. Ina kipaumbele kidogo kuliko mitindo ndani ya mstari. CSS ya ndani na ya nje zina kiwango sawa cha kipaumbele — kipaumbele kinaamuliwa na vichaguzi vinavyotumika, si mahali CSS imeandikwa. Wakati sheria mbili zina kipaumbele sawa, ile inayojitokeza baadaye katika hati ndiyo inashinda. Hivyo, kama faili la mtindo wa nje limeunganishwa baada ya kipengele cha `style` cha ndani, sheria za nje zinapata kipaumbele. Ikiwa limeunganishwa kabla, sheria za ndani ndizo zinashinda.

Thamani ya kipaumbele kwa mitindo ya ndani huamuliwa na vichaguzi vinavyotumika. Kwa mfano, kichaguzi cha ID ndani ya CSS ya ndani kina thamani ya kipaumbele ya `(0, 1, 0, 0)`.

:::interactive_editor

```html
<head>
  <style>
    #text {
      color: blue;
    }
  </style>
</head>
<body>
  <div id="text">This text is blue.</div>
</body>
```

:::

Katika mfano huu, maandishi yatakuwa ya buluu isipokuwa mtindo ndani ya mstari au kichaguzi chenye kipaumbele zaidi kitumike.

CSS ya nje inaunganishwa kupitia kipengele cha `link` katika sehemu ya `head` na huandikwa katika mafaili tofauti ya `.css`. Kama CSS ya ndani, kipaumbele cha mitindo ya nje kinaamuliwa na vichaguzi vinavyotumika. CSS ya nje hutoa utunzaji bora zaidi kwa miradi mikubwa.

Thamani ya kipaumbele kwa mitindo ya nje pia huamuliwa na vichaguzi vinavyotumika. Kwa mfano, kichaguzi cha darasa la vitu ndani ya CSS ya nje kina thamani ya kipaumbele ya `(0, 0, 1, 0)`.

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <div class="text">This text's color is defined in an external CSS file.</div>
</body>
```

```css
.text {
  color: purple;
}
```

:::

Katika mfano huu, rangi ya maandishi imefafanuliwa katika faili ya `styles.css` na itatumika isipokuwa ighairiwe na kichaguzi chenye kipaumbele zaidi au mtindo ndani ya mstari.
