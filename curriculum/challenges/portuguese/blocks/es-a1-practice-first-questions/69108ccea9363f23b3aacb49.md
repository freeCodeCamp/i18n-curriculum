# --explanation--

Este diálogo entre Mateo e Camila foca em profissões.

`¿A qué te dedicas?` significa "Qual é a sua profissão?" ou "O que você faz para viver?"

**`Soy` + [profession]** significa "Eu sou um/uma [profession]".

Em espanhol, a maioria das profissões muda com base no gênero. No entanto, algumas profissões que terminam em `-ista` ou `-nte` geralmente são mantidas exatamente iguais para ambos os gêneros:

| Profissão | Masculino | Feminino |
| :---: | :---: | :---: |
| Analista | `analista` | `analista` |
| Jornalista | `periodista` | `periodista` |
| Recepcionista | `recepcionista` | `recepcionista` |
| Cantor | `cantante` | `cantante` |
