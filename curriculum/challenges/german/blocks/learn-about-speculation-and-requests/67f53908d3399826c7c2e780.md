# --explanation--

`Want` drückt den Wunsch aus, etwas zu tun. In der Vergangenheit sagt man `wanted`, um zu zeigen, dass der Wunsch zu einem früheren Zeitpunkt bestand. Es folgt der Infinitiv eines Verbs (mit `to`). Zum Beispiel:

- `She wants to install the new software.` – Sie möchte die neue Software einrichten oder hinzufügen.

- `They wanted to upgrade their system last week.` – In der Vergangenheit planten sie ein Upgrade.
