# --description--

Erstellen Sie eine Full-Stack-JavaScript-App, die funktional ähnlich ist wie diese: <a href="https://chart-the-stock-market.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://chart-the-stock-market.freecodecamp.rocks/</a>. Verwenden Sie einen Website-Builder Ihrer Wahl, um das Projekt abzuschließen.

Hier sind die spezifischen User Stories, die Sie für dieses Projekt implementieren sollten:

**User Story:** Sie können ein Diagramm anzeigen, das die aktuellen Trendlinien für jede hinzugefügte Aktie zeigt.

**User Story:** Sie können neue Aktien anhand ihres Symbolnamens hinzufügen.

**User Story:** Sie können Aktien entfernen.

**User Story:** Sie können Änderungen in Echtzeit sehen, wenn ein anderer Benutzer eine Aktie hinzufügt oder entfernt. Dafür müssen Sie Web Sockets verwenden.

Wenn Sie fertig sind, stellen Sie sicher, dass eine funktionierende Demo Ihres Projekts öffentlich gehostet wird. Geben Sie dann die URL im Feld Lösung-Link ein. Optional können Sie auch einen Link zum Quellcode Ihres Projekts im GitHub-Link-Feld absenden.
