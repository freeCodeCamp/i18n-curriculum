# --description--

Dependiendo del tipo de trabajo que estés haciendo, podrías terminar trabajando más con fragmentos de texto que con números. Una **string** es un fragmento de texto… y es una estructura fundamental del lenguaje.

HTML proporciona estructura y significado al texto, CSS nos permite darle estilo con precisión, y JavaScript ofrece muchas funciones para manipular cadenas. Estas incluyen crear mensajes de bienvenida personalizados y avisos, mostrar las etiquetas de texto correctas cuando se necesitan, ordenar términos en el orden deseado, y mucho más.

Las cadenas son un tipo de dato fundamental en JavaScript. Se usan para representar texto y se encierran en comillas simples, dobles o backticks:

```javascript
let greeting = "Hello World!";

let greeting2 = 'I am learning JavaScript!';

let greeting3 = `with The Odin Project!`;
```

Las cadenas declaradas usando comillas simples y las cadenas declaradas usando comillas dobles son iguales, y cuál uses depende de tu preferencia personal — aunque es buena práctica elegir un estilo y usarlo consistentemente en tu código.
