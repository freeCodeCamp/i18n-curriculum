# --description--

設 $N$ 為正整數，且將 $N$ 分成 $k$ 個相等的零件，$r = \frac{N}{k}$，使得 $N = r + r + \cdots + r$。

設 $P$ 為這些零件的乘積，$P = r × r × \cdots × r = r^k$。

例如，如果 11 被分成五個相等的零件，11 = 2.2 + 2.2 + 2.2 + 2.2 + 2.2，則 $P = {2.2}^5 = 51.53632$。

令 $M(N) = P_{max}$，其中 $N$ 為給定值。

結果顯示，當 $N = 11$ 時的最大值是將十一分成四個相等的零件，這導致 $P_{max} = {(\frac{11}{4})}^4$；也就是說，$M(11) = \frac{14641}{256} = 57.19140625$，這是一個有限小數。

然而，對於 $N = 8$，最大值是透過將其分成三個相等的零件來達成，所以 $M(8) = \frac{512}{27}$，這是一個無限不循環小數。

令 $D(N) = N$，如果 $M(N)$ 是無限小數，且令 $D(N) = -N$，如果 $M(N)$ 是有限小數。

例如，$\sum D(N)$ 在 $5 ≤ N ≤ 100$ 時為 2438。

求 $\sum D(N)$，其中 $5 ≤ N ≤ 10000$。
