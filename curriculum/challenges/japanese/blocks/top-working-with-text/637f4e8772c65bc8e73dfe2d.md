# --description--

HTMLコメントはブラウザーには表示されません。コードにコメントを付けることで、他の開発者や将来の自分がコードの中でわかりにくい部分の背景を理解できるようにします。

HTMLコメントを書くのは簡単です。コメントを`<!--`と`-->`タグで囲むだけです。例えば：

```html
<h1> View the html to see the hidden comments </h1>

<!-- I am a html comment -->

<p>Some paragraph text</p>

<!-- I am another html comment -->
```

# --assignment--

HTMLでテキストを扱う練習として、さまざまな見出しを使い、段落を使い、段落内のテキストの一部を太字や斜体にしたシンプルなブログ記事ページを作成してください。サイトを作る際に実際のテキストの代わりに[Lorem Ipsum](https://loremipsum.io)を使ってダミーテキストを生成できます。
