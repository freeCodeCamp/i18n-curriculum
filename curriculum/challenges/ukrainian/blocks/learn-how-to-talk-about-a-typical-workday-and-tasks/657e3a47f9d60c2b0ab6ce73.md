# --description--

Для `make sure` використовується, щоб висловити впевненість або підтвердження чогось. Наприклад:

`I check the list twice to make sure I have everything.` — це означає, що ви дивитесь на список двічі, щоб переконатися, що нічого не пропущено.

To `work properly` означає працювати правильно або як очікується. Наприклад:

`It's important that all machines work properly for safety.` — це означає, що машини мають працювати належним чином, щоб люди були в безпеці.
