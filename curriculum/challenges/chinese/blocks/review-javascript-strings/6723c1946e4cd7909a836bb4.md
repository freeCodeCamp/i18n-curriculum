# --interactive--

## 字串基础

- **定义**：字串是用单引号、双引号或反引号包裹的字符序列。字串是原语数据类型且不可变。不可变性意味着一旦创建字串，就不能更改它。
- **访问字串中的字符**：要访问字串中的字符，可以使用方括弧表示法并传入索引号。索引是字符在字串中的位置，且从零开始。

:::interactive_editor

```js
const developer = "Jessica";
console.log(developer[0]); // J
```

:::

- **`\n`（换行字符）**：你可以使用 `\n` 换行字符在字串中创建换行。

:::interactive_editor

```js
const poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

- **转义字串**：你可以通过在引用前放置反斜线（`\`）来转义字串中的字符。

:::interactive_editor

```js
const statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

## 模板字面量（模板字串）和字串插值

- **定义**：模板字面量用反引号（`）定义。它们允许更方便地操作字串，包括直接在字串中嵌入变量，这称为字串插值。

:::interactive_editor

```js
const name = "Jessica";
const greeting = `Hello, ${name}!`; 
console.log(greeting); // "Hello, Jessica!"
```

:::

## ASCII、`charCodeAt()` 方法和 `fromCharCode()` 方法

- **ASCII**：ASCII（美国信息交换标准代码）是一种字符编码标准，用于使用数字值表现基本的英文字符。之前的课程通过 ASCII 示例介绍了 `charCodeAt()` 和 `fromCharCode()`。
- **Unicode**：JavaScript 字串在内部使用 Unicode，具体为 UTF-16 编码。对于前 128 个字符（基本拉丁字母、数字和常用符号），Unicode 值与 ASCII 代码相匹配。这就是基于 ASCII 的示例在 JavaScript 中仍然有效的原因。
- **`charCodeAt()` 方法**：此方法返回指定索引处字符的 UTF-16 代码单元。对于基本拉丁字符，该值与 ASCII 码匹配。

:::interactive_editor

```js
const letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

:::

- **`fromCharCode()` 方法**：此方法将 ASCII 码转换为对应的字符。

:::interactive_editor

```js
const char = String.fromCharCode(65);
console.log(char);  // A
```

:::

## 其他常用字串方法

- **`indexOf` 方法**：此方法用于在字串中查找子串。如果找到子串，`indexOf` 返回该子串首次出现的索引（位置）。如果未找到，`indexOf` 返回 -1，表示查找失败。

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.indexOf("fox")); // 16
console.log(text.indexOf("cat")); // -1
```
  
:::  

- **`includes()` 方法**：此方法用于检查字串是否包含特定子串。如果字串中包含该子串，方法返回 true，否则返回 false。

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.includes("fox")); // true
console.log(text.includes("cat")); // false
```

:::

- **`slice()` 方法**：该方法提取字符串的一部分，并返回一个新的字符串，而不修改原始字符串。 它需要两个参数：开始索引和选填的结束索引。

:::interactive_editor

```js
const text = "freeCodeCamp";
console.log(text.slice(0, 4));  // "free"
console.log(text.slice(4, 8));  // "Code"
console.log(text.slice(8, 12)); // "Camp"
```

:::

- **`toUpperCase()` 方法**：此方法将所有字符转换为大写字母，并返回一个包含所有大写字符的新字串。

:::interactive_editor

```js
const text = "Hello, world!";
console.log(text.toUpperCase()); // "HELLO, WORLD!"
```

:::

- **`toLowerCase()` 方法**：此方法将字串中的所有字符转换为小写。

:::interactive_editor

```js
const text = "HELLO, WORLD!"
console.log(text.toLowerCase()); // "hello, world!"
```

:::

- **`replace()` 方法**：此方法允许你在字串中查找指定值（如单词或字符）并替换为另一个值。该方法返回替换后的新字串，原字串保持不变，因为 JavaScript 字串是不可变的。

:::interactive_editor

```js
const text = "I like cats";
console.log(text.replace("cats", "dogs")); // "I like dogs"
```

:::

- **`replaceAll()` 方法**：此方法允许你查找字串中指定值（一个单词、字符或模式）的所有出现，并将它们替换为另一个值。它的工作方式类似于 `replace()`，但不是在找到第一个匹配后停止，而是更新字串中找到的每个匹配。

:::interactive_editor

```js
const text = "I love cats and cats are so much fun!";
console.log(text.replaceAll("cats", "dogs")); // "I love dogs and dogs are so much fun!"
```

:::

- **`repeat()` 方法**：此方法用于重复字串指定次数。

:::interactive_editor

```js
const text = "Hello";
console.log(text.repeat(3)); // "HelloHelloHello"
```

:::

- **`trim()` 方法**：此方法用于移除字串开头和结尾的空白字符。

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trim()); // "Hello, world!"
```

:::

- **`trimStart()` 方法**：此方法移除字串开头（或“开始”）的空白字符。

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trimStart()); // "Hello, world!  "
```

:::

- **`trimEnd()` 方法**：此方法移除字串结尾的空白字符。

:::interactive_editor

```js
const text = " Hello, world! ";
console.log(text.trimEnd()); // "  Hello, world!"
```

:::

- **`prompt()` 方法**：`window` 的此方法用于通过对话框形式从用户获取信息。此方法接受两个参数。第一个参数是显示在对话框中的消息，通常提示用户输入信息。第二个参数是可选的默认值，初始填充输入字段。

```js
const answer = window.prompt("What's your favorite animal?"); // This will change depending on what the user answers
```
