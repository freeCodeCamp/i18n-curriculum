# --explanation--

`Brute force attacks` 是指駭客嘗試多個密碼直到找到正確的密碼。 例如：

`The system detected a brute force attack on the login page.` - 攻擊嘗試了許多密碼猜測以入侵。

`SQL injection attempts` 發生在駭客試圖插入惡意 SQL 程式碼以存取或修改資料庫時。 例如：

`The firewall stopped two SQL injection attempts。` - 有人試圖使用有害程式碼入侵資料庫。
