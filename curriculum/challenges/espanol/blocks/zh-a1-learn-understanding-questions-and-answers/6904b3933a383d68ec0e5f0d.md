# --explanation--

`请问 (qǐng wèn)` es una frase educada que se usa para introducir una pregunta. Se traduce como "disculpa" o "¿puedo preguntar?", y muestra respeto cuando estás a punto de preguntar algo. A menudo se usa con extraños o en situaciones formales. Por ejemplo:

`请问你是刘明吗 (qǐng wèn nǐ shì liú míng ma)？` – Disculpa, ¿eres Liu Ming?
