# --interactive--

在使用 CSS 为网页设计样式时，最常用的定义颜色的方法之一就是使用十六进制颜色值。 十六进制颜色值或十六进制代码是 RGB 颜色模型中表示颜色的简明方法。  In this lesson, we'll explore what hex codes are, how they work in CSS, and when to use them.

hex 代码（十六进制代码的简称）是一个六个字符的字符串，用于表示 RGB 颜色模型中的颜色。 “hex”指的是十六进制计数系统，使用数字 `0` 至 `9` 和字母 `A` 至 `F`。 

在颜色方面，十六进制代码指定了组成特定颜色的红、绿、蓝（RGB）的量。 每个十六进制代码以一个哈希符号（`#`）开头，后面跟着六个字符，分别定义颜色中红、绿、蓝的强度。 

例如，黑色十六进制代码为 `#000000`，白色十六进制代码为 `#FFFFFF`。 前两个字符代表红色值，接着两个字符代表绿色值，最后两个字符代表蓝色值。

在 CSS 中，你可以使用十六进制代码为元素添加颜色。 这是基本语法：

```css
element {
  color: #RRGGBB;
}
```

这里，`#RRGGBB` 是实际十六进制代码的占位符。 `RR`、`GG` 和 `BB` 对分别代表红色、绿色和蓝色的强度。 

这些对的范围从 `00`（最低强度）到 `FF`（最高强度）。 数字越大，最终颜色中该颜色的含量就越多。 例如，让我们为一些元素应用十六进制颜色：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h1>This is a heading.</h1>
<p>This is a paragraph.</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}
p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

在第一个示例中，`#FF5733` 是橘红色。 具体情况如下：

- `FF` 是红色的最高值（完全饱和）。
- `57` 是适量的绿色。
- `33` 是少量的蓝色。

在第二个示例中，`#4CAF50` 是一种绿色：

- `4C` 是红色的低值。
- `AF` 是绿色的高值。
- `50` 是蓝色的适中值。

十六进制代码之所以流行，是因为它们精确且易于使用。 它们允许你对颜色进行微调，因此非常适合对一致性要求较高的设计。 

大多数设计软件（如 Adobe Photoshop 或 Figma）都会为你选择的颜色提供十六进制代码，这样就可以轻松地将这些值直接复制并粘贴到 CSS 文件中。

在网页开发中，十六进制代码通常用于定义文本颜色、背景、边框和其他视觉元素。 例如：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example</button>
```

```css
body {
  background-color: #F0F0F0; /* Light gray background */
}

button {
  background-color: #007BFF; /* A shade of blue */
  color: #FFFFFF; /* White text */
}
```

:::

这样可以确保整个网站或应用程序的颜色保持一致。

在某些情况下，你可能会注意到十六进制代码是以简写格式书写的，只使用了三个字符而不是六个字符。 当每对颜色中的两个字符都相同时，就可以这样做。 例如，`#FF5733` 不能被简写，但 `#FFFFFF`（白色）可以写成 `#FFF`。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h2>This is a heading.</h2>
```

```css
body {
  background-color: #000;
}

h2 {
  color: #FFF; /* White in shorthand */
}
```

:::

简写的 `#FFF` 等同于 `#FFFFFF`。 同样，`#000` 代表黑色，完整形式为 `#000000`。

十六进制代码因其精确性和兼容性而广受欢迎，可在所有浏览器中使用超过 1600 万种颜色。 当你需要特定的品牌颜色，想要控制命名颜色或 HSL 不容易定义的精确色调，或者从默认提供十六进制代码的设计软件中复制颜色时，它们都是理想之选。
