# --description--

Das `div`-Element sieht schon besser aus. Aber jetzt ist das dismiss `button` ein wenig zu nah am Absatz-Element.

Um Abstand zwischen den Flex-Elementen zu schaffen, können Sie das folgende Muster verwenden: `gap-number`, wobei jede `number`-Einheit 0,25rem (oder 4px) festen Abstand sowohl zwischen den Reihen als auch den Spalten in einem Flex-Layout darstellt.

Hier ist ein Beispiel, wie Sie einem Element ein `gap` zuweisen:

```html
<p class="flex gap-8">This is gap.</p>
```

Fügen Sie in Ihrem öffnenden `div`-Tag die richtige Utility-Klasse ein, um den `gap` auf `16px` einzustellen.

# --hints--

Ihr öffnender `div`-Tag sollte die Utility-Klasse `gap-4` enthalten.

```js
const divEl = document.querySelector("div");
assert.isTrue(divEl.classList.contains("gap-4"));
```
