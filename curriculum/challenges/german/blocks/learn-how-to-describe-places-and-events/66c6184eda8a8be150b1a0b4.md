# --explanation--

`Have to` wird verwendet, um auszudrücken, dass etwas notwendig oder erforderlich ist. Es zeigt an, dass es keine andere Wahl gibt, als etwas zu tun. Zum Beispiel:

- `You have to save your changes before closing the program.` – Das bedeutet, dass es notwendig ist, Änderungen zu speichern, bevor das Programm geschlossen wird.

Wenn Tom sagt, `I'll have to check again`, drückt er aus, dass es für ihn notwendig ist, die Einstellungen noch einmal zu überprüfen, was darauf hindeutet, dass er sich nicht sicher ist.
