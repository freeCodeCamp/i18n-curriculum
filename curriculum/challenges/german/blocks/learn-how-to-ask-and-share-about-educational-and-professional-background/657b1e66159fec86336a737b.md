# --description--

Typischerweise wird die `Past Simple`-Zeitform gebildet, indem man `-ed` zu regelmäßigen Verben hinzufügt. Zum Beispiel wird das Verb `play` zu `played`.

To `play around` bedeutet, etwas auf eine lockere oder experimentelle Weise auszuprobieren, oft einfach nur, um zu sehen, was passiert. Es wird normalerweise von `with` gefolgt. Zum Beispiel:

`He played around with the new software to see how it works.` – Das bedeutet, dass er die Software ausprobiert hat, ohne einen ernsthaften Plan, nur um sie zu erkunden.
