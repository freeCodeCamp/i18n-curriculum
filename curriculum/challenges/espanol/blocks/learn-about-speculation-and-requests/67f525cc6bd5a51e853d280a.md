# --explanation--

`Sounds good` es una forma casual de decir `I agree` o `That works for me`.

`Keep me posted` significa `Let me know if there are any updates`. Es una forma educada de pedir información continua sobre una situación. Por ejemplo:

- Si encuentras algún problema con el sistema, mantenme informado.

- `Keep me posted if you get any feedback from the client.` - Estás pidiendo a alguien que te informe si recibe alguna novedad del cliente.
