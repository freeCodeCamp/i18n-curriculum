# --description--

في الدرس السابق، تعلمت كيفية تثبيت Python على جهازك المحلي. في هذا الدرس، ستتعلم كيفية تشغيل شفرة Python محليًا.

قبل أن تبدأ في بناء سكريبتات Python محليًا، تحتاج إلى تثبيت محرر شفرة أو بيئة تطوير متكاملة (IDE). تعني IDE بيئة التطوير المتكاملة التي تحتوي على ميزات مثل محرر شفرة، أدوات اختبار، محطة طرفية والمزيد.

أحد محررات الشفرة الشهيرة التي يستخدمها العديد من المطوّرين هو VS Code. زر `https://code.visualstudio.com/download` لتحميل VS Code الذي يدعم بيئات العمل على Mac وWindows وLinux. هذا هو المحرر الذي ستستخدمه في هذا الدرس.

خيارات شائعة أخرى لتطوير Python تشمل ما يلي:

- PyCharm
- Spyder

بعد تحميل المحرر، افتحه. بعد الفتح، تحتاج إلى فتح مجلد. انقر على خيار قائمة `File` في الزاوية العلوية اليسرى ثم انقر على `Open Folder`. من هناك، يمكنك اختيار مجلد أو إنشاء مجلد لمشروعك. للممارسة الإضافية خارج الدرس، قد ترغب في مجلد باسم `python-projects`.

بعد إعداد المجلد، افتح المستكشف وهو الشريط الجانبي الأيسر في VS Code. انقر على أيقونة `New File` التي تشبه ورقة مع علامة زائد فوقها. ثم أنشئ ملفًا باسم `main.py` واضغط Enter. الامتداد `.py` يشير إلى أن هذا ملف Python.

افتح ملف `main.py` واكتب الشفرة التالية:

```py
print("Hello, world!")
```

لتشغيل هذه الشفرة، لديك عدة خيارات. الخيار الأول هو النقر على زر التشغيل الذي يشبه زر التشغيل الموجود في الزاوية العلوية اليمنى. هذا سيفتح محطة طرفية ويشغل سكريبت Python هناك. سترى النص `"Hello, world!"`.

خيار آخر هو فتح محطة طرفية وتشغيل البرنامج يدويًا باستخدام:

```bash
python main.py
```

يجب تشغيل هذا الأمر من المجلد الذي تم حفظ `main.py` فيه. على سبيل المثال، إذا كان `main.py` داخل مجلد يسمى `python-projects`، استخدم أولاً الأمر التالي في المحطة الطرفية:

```bash
cd python-projects
```

ثم شغّل:

```bash
python main.py
```

في بعض بيئات العمل، خاصة على أنظمة macOS وLinux حيث يتم تثبيت Python 2 وPython 3 معًا، قد تحتاج إلى استخدام `python3` بدلًا من `python`. هذا شائع عندما لا يوجد أمر `python` أو يشير إلى إصدار أقدم من Python.

```bash
python3 main.py
```

مع تقدمك في الدورة، جرب الأمثلة من الدروس محليًا وشاهد النتائج. حاول أيضًا ابتكار أمثلة خاصة بك لاختبار ما تعلمته من الدروس.
