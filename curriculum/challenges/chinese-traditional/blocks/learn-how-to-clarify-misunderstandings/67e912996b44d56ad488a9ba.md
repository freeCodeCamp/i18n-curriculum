# --explanation--

`Code reuse` 是指在程式的新零件中使用現有程式碼的做法。例如：

`The "Login" class was used in multiple applications to handle user access, which is a good example of code reuse。`－與其為每個應用程式撰寫登入邏輯，不如重複使用相同的程式碼，提升效率。

`Extending` 是指為已存在的東西添加新特性。 例如：

`"AdminUser"` 類別透過添加 `approveUser()` 方法來擴充基本的 `"User"` 類別。 - `AdminUser` 保留了 `User` 的所有內容，並在此基礎上建置了新的功能。
