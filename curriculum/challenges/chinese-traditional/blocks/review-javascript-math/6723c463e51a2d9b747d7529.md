## 操作 Number 資料型別

- **定義**：JavaScript 的 `Number` 型別包含整數、浮點數、`Infinity` 和 `NaN`。浮點數是帶有小數點的數字。正 `Infinity` 是大於任何其他數字的數值，而 `-Infinity` 則是小於任何其他數字的數值。`NaN`（`Not a Number`）代表無效的數值，例如字串 `"Jessica"`。

## 常見算術運算

- **加法運算子**：此運算子（`+`）用於計算兩個或多個數字的總和。
- **減法運算子**：此運算子（`-`）用於計算兩個數字的差。
- **乘法運算子**：此運算子（`*`）用於計算兩個或多個數字的乘積。
- **除法運算子**：此運算子（`/`）用於計算兩個數字的商。
- **除以零**：如果你嘗試除以零，JavaScript 會傳回 `Infinity`。
- **餘數運算子**：此運算子（`%`）傳回除法的餘數。
- **指數運算子**：此運算子（`**`）將一個數字提升到另一個數字的次方。

## 數字與字串的運算

- **說明**：當你使用 `+` 運算子搭配數字和字串時，JavaScript 會將數字強制轉換成字串並將兩個值串接起來。當你使用 `-`、`*` 或 `/` 運算子搭配字串和數字時，JavaScript 會將字串強制轉換成數字，結果會是數字。對於 `null` 和 `undefined`，JavaScript 在數學運算中將 `null` 視為 0，將 `undefined` 視為 `NaN`。

:::interactive_editor

```js
const result = 5 + '10';

console.log(result); // "510"
console.log(typeof result); // string

const subtractionResult = '10' - 5;
console.log(subtractionResult); // 5
console.log(typeof subtractionResult); // number

const multiplicationResult = '10' * 2;
console.log(multiplicationResult); // 20
console.log(typeof multiplicationResult); // number

const divisionResult = '20' / 2;
console.log(divisionResult); // 10
console.log(typeof divisionResult); // number

const result1 = null + 5;
console.log(result1); // 5
console.log(typeof result1); // number

const result2 = undefined + 5;
console.log(result2); // NaN
console.log(typeof result2); // number
```

:::

## 運算子優先順序

- **定義**：運算子優先決定運算式中操作的評估順序。優先較高的運算子會先於優先較低的運算子被評估。括號內的值會先被評估，且乘法／除法的優先會高於加法／減法。如果運算子具有相同的優先，則 JavaScript 會使用結合性。

:::interactive_editor

```js
const result = (2 + 3) * 4;

console.log(result); // 20

const result2 = 10 - 2 + 3;

console.log(result2); // 11

const result3 = 2 ** 3 ** 2;

console.log(result3); // 512
```

:::

- **定義**：結合性告訴我們當存在多個相同型別的運算子時，表達式的計算方向。它定義表達式應該從左到右 (`left-associative`) 還是從右到左 (`right-associative`) 進行計算。<dfn>例如</dfn>，指數運算子也是從右到左結合的：

:::interactive_editor

```js
const result4 = 5 ** 4 ** 1; 

console.log(result4); // 625
```

:::

## 累加與遞減運算子

- **累加運算子**：這個運算子用來將值加 1。前置式 `++num` 會先將變數的值加 1，再傳回新值。後置式 `num++` 則會先傳回目前的值，再將其加 1。

:::interactive_editor

```js
let x = 5;

console.log(++x); // 6
console.log(x); // 6


let y = 5;

console.log(y++); // 5
console.log(y); // 6
```

:::

- **遞減運算子**：這個運算子用來將值減 1。前置式和後置式的運作方式與累加運算子相同。

:::interactive_editor

```js
let num = 5;

console.log(--num); // 4
console.log(num--); // 4
console.log(num); // 3
```

:::

## 複合指定運算子

- **加法指定（`+=`）運算子**：這個運算子會將值相加，並將結果指定給變數。
- **減法指定（`-=`）運算子**：這個運算子會將值相減，並將結果指定給變數。
- **乘法指定（`*=`）運算子**：這個運算子會將值相乘，並將結果指定給變數。
- **除法指定（`/=`）運算子**：這個運算子會將值相除，並將結果指定給變數。
- **餘數指定（`%=`）運算子**：這個運算子會將變數除以指定數字，並將餘數指定給變數。
- **次方指定（`**=`）運算子**：這個運算子會將變數提升為指定數字的次方，並將結果重新指定給變數。

## 布林值與相等性

- **布林值定義**：布林值是一種只能有兩個值的資料型別：`true` 或 `false`。
- **相等（`==`）運算子**：這個運算子在檢查值是否相等前會進行型別轉換。

:::interactive_editor

```js
console.log(5 == '5'); // true
```

:::

- **嚴格相等（`===`）運算子**：這個運算子不會進行型別轉換，會檢查型別和值是否都相等。

:::interactive_editor

```js
console.log(5 === '5'); // false
```

:::

- **不相等 (`!=`) 運算子**：此運算子在檢查值是否不相等之前會使用型別強制轉換。
- **嚴格不相等 (`!==`) 運算子**：此運算子不執行型別強制轉換，並檢查型別和數值是否皆不相等。

## 比較運算子

- **大於（`>`）運算子**：這個運算子會檢查左邊的值是否大於右邊的值。
- **大於等於（`>=`）運算子**：這個運算子會檢查左邊的值是否大於或等於右邊的值。
- **小於（`<`）運算子**：這個運算子會檢查左邊的值是否小於右邊的值。
- **小於等於（`<=`）運算子**：這個運算子會檢查左邊的值是否小於或等於右邊的值。

## 一元運算子

- **一元加號運算子**：這個運算子會將其運算元轉換為數字。如果運算元本來就是數字，則不變。

:::interactive_editor

```js
const str = '42';
const num = +str;

console.log(num); // 42
console.log(typeof num); // number
```

:::

- **一元負號（`-`）運算子**：這個運算子會將運算元取負值。

:::interactive_editor

```js
const num = 4;
console.log(-num); // -4
```

:::

- **邏輯 NOT（`!`）運算子**：這個運算子會反轉運算元的布林值。如果運算元是 `true`，就會變成 `false`；如果是 `false`，就會變成 `true`。

## 位元運算子

- **位元 AND（`&`）運算子**：這個運算子會在兩個運算元的每個位元位置上，若兩者皆為 1，則傳回 1。
- **位元 AND 指定（`&=`）運算子**：這個運算子會對指定數字進行 `bitwise AND` 運算，並將結果重新指定給變數。
- **位元 OR（`|`）運算子**：這個運算子會在兩個運算元的每個位元位置上，只要其中之一為 1，就傳回 1。
- **位元 OR 指定（`|=`）運算子**：這個運算子會對指定數字進行 `bitwise OR` 運算，並將結果重新指定給變數。
- **位元 XOR（`^`）運算子**：這個運算子會在兩個運算元的每個位元位置上，僅當其中一個為 1（但不是兩者同時為 1）時傳回 1。
- **位元 NOT（`~`）運算子**：這個運算子會將數字的二進位表示反轉。
- **左移（`<<`）運算子**：這個運算子會將所有位元向左移動指定的位數。
- **右移（`>>`）運算子**：這個運算子會將所有位元向右移動。

## 條件判斷、Truthy 值、Falsy 值與三元運算子

- **`if/else if/else`**：`if` 判斷式會帶入一個條件，若該條件為 `truthy`，則執行一段程式碼區塊。如果條件為 `false`，則會進入 `else if` 區塊。如果這些條件都不是 `true`，則會執行 `else` 語句。`Truthy` 值是在布林情境（如 `true` 判斷式）下會被評估為 `if` 的任何值。`Falsy` 值則是在布林情境下會被評估為 `false` 的值。

:::interactive_editor

```js
const score = 87;

if (score >= 90) {
 console.log('You got an A'); 
} else if (score >= 80) {
 console.log('You got a B'); // You got a B
} else if (score >= 70) {
 console.log('You got a C');
} else {
 console.log('You failed! You need to study more!');
}
```

:::

- **三元運算子**：這個運算子常用來簡化 `if else` 判斷式的寫法。

:::interactive_editor

```js
const temperature = 30;
const weather = temperature > 25 ? 'sunny' : 'cool';

console.log(`It's a ${weather} day!`); // It's a sunny day!
```

:::

## 二元邏輯運算子

- **邏輯 AND (`&&`) 運算子**：此運算子會檢查兩個運算元是否皆為 `true`。如果兩者皆為 `true`，則會傳回第二個值。如果任一運算元為 `falsy`，則會傳回該 `falsy` 值。如果兩個運算元皆為 `falsy`，則會傳回第一個 `falsy` 值。

:::interactive_editor

```js
const result = true && 'hello';

console.log(result); // hello
```

:::

- **邏輯 OR (`||`) 運算子**：此運算子會檢查是否至少有一個運算元為真值。
- **Nullish Coalescing (`??`) 運算子**：只有當第一個值是 `null` 或 `undefined` 時，這個運算子才會傳回該值。

:::interactive_editor

```js
const userSettings = {
 theme: null,
 volume: 0,
 notifications: false,
};

let theme = userSettings.theme ?? 'light';
console.log(theme); // light
```

:::

## `Math` 物件

- **`Math.random()` 方法**：這個方法會產生一個介於 0（包含）和 1（不包含）之間的隨機浮點數。也就是說，可能的輸出可以是 0，但永遠不會到達 1。
- **`Math.max()` 方法**：這個方法會取一組數字並傳回最大值。
- **`Math.min()` 方法**：這個方法會取一組數字並傳回最小值。
- **`Math.ceil()` 方法**：這個方法會將數值無條件進位到最接近的整數。
- **`Math.floor()` 方法**：這個方法會將數值無條件捨去到最接近的整數。
- **`Math.round()` 方法**：這個方法會將數值四捨五入到最接近的整數。

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

- **`Math.trunc()` 方法**：這個方法會移除數字的小數部分，只傳回整數部分，不會進行四捨五入。
- **`Math.sqrt()` 方法**：這個方法會傳回數字的平方根。
- **`Math.cbrt()` 方法**：這個方法會傳回數字的立方根。
- **`Math.abs()` 方法**：這個方法會傳回數字的絕對值。
- **`Math.pow()` 方法**：這個方法會將第一個數字提升為第二個數字的次方。

## 常見的數字方法

- **`isNaN()`**：`NaN` 代表「不是數字」（Not-a-Number）。這是一個特殊值，用來表示無法表示或未定義的數值結果。`isNaN()` 函式屬性用來判斷一個值是否為 `NaN`。`Number.isNaN()` 提供了一種更可靠的方式來檢查 `NaN` 值，特別是在型別轉換可能導致全域 `isNaN()` 函式出現非預期結果的情況下。

:::interactive_editor

```js
console.log(isNaN(NaN));       // true
console.log(isNaN(undefined)); // true
console.log(isNaN({}));        // true

console.log(isNaN(true));      // false
console.log(isNaN(null));      // false
console.log(isNaN(37));        // false


console.log(Number.isNaN(NaN));        // true
console.log(Number.isNaN(Number.NaN)); // true
console.log(Number.isNaN(0 / 0));      // true

console.log(Number.isNaN("NaN"));      // false
console.log(Number.isNaN(undefined));  // false
```

:::

- **`parseFloat()` 方法**：這個方法會解析字串引數並傳回一個浮點數。它設計用來從字串開頭擷取數字，即使字串後面有非數字字元也沒關係。
- **`parseInt()` 方法**：這個方法會解析字串引數並傳回一個整數。`parseInt()` 在遇到第一個非數字時就會停止解析。對於浮點數，它只會傳回整數部分。如果在字串開頭找不到有效的整數，則會傳回 `NaN`。
- **`toFixed()` 方法**：這個方法是針對數字呼叫的，並接受一個選用的引數，指定小數點後要顯示幾位數。它會傳回指定小數位數的字串表示。
