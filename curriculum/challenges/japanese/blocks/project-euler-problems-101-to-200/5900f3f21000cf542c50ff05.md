# --description--

連続する素数 $p_1 = 19$ と $p_2 = 23$ を考えます。1219 は、末尾の数字が $p_1$ で構成され、かつ $p_2$ で割り切れる最小の数であることが確認できます。

実際、$p_1 = 3$ と $p_2 = 5$ を除いて、すべての連続する素数のペア $p_2 > p_1$ に対して、末尾の数字が $p_1$ で構成され、かつ $n$ が $p_2$ で割り切れるような値 $n$ が存在します。これらの $n$ の中で最小のものを $S$ とします。

$5 ≤ p_1 ≤ 1000000$ のすべての連続する素数ペアについて、$\sum{S}$ を求めてください。
