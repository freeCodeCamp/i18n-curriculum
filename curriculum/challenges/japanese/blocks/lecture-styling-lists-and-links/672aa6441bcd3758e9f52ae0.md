# --interactive--

マージンと line-height は、リスト項目の間隔を調整して読みやすさや見た目の良さを高めるために欠かせません。

まずは、マージンを使ってリスト項目の間隔を調整してみましょう！

マージンは `li` 要素にマージンのプロパティを適用することで、リスト項目同士の間にスペースを作ることができます。この方法では各リスト項目の外側の間隔をコントロールでき、項目間の隙間を増やしたり減らしたりできます。

3つのリスト項目を持つ番号なしリストの例を見てみましょう。

:::interactive_editor

```html
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

デフォルトでは、HTMLはリスト項目間にあまりスペースを適用しません。

各リスト項目の下にスペースを適用するには、次のように `margin-bottom` プロパティを使います。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  margin-bottom: 40px;
}
```

:::

この例では、番号なしリスト内の各リスト項目の下に `40px` のマージンが適用されます。

リスト項目の間隔を空ける別の方法として、`line-height` プロパティを使う方法があります。

`line-height` プロパティは、単一のリスト項目内のテキスト行間の垂直間隔を調整します。

主に各項目内のテキスト行間の間隔に影響しますが、項目が1行だけの場合はリスト項目間の全体的な間隔にも間接的に影響を与えます。

リスト項目が複数行のテキストを含む場合、`line-height` はその行間の間隔に影響しますが、別々のリスト項目間の間隔を直接調整するわけではありません。

個々のリスト項目間の間隔をコントロールするには、代わりに `margin` または `padding` プロパティを使います。

先ほどの番号なしリストを使い、リスト項目に `line-height` を適用した例がこちらです。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  line-height: 2; 
}
```

:::

この例では、`line-height: 2;` がフォントサイズの2倍の行の高さを設定し、各リスト項目内により多くの垂直スペースを作っています。

## --text--

各 `li` 要素の外側のスペースを調整してリスト項目間にスペースを作るには、どの CSS プロパティを使いますか？

## --answers--

`padding`

### --feedback--

このプロパティは要素の `border` 内のスペースに影響します。

---

`line-height`

### --feedback--

このプロパティはテキスト行間の間隔に影響し、要素の外側のスペースには影響しません。

---

`margin`

---

`border`

### --feedback--

このプロパティは要素の `border` に影響し、外側のスペースには影響しません。

## --text--

CSSルール `li { margin-bottom: 15px; }` はリスト項目にどんな効果を与えますか？

## --answers--

各リスト項目の内側に `15px` のスペースを追加します。

### --feedback--

このルールはリスト項目の外側のスペースに影響し、内側には影響しません。

---

各リスト項目のコンテンツと `15px` の間に `border` のスペースを追加します。

### --feedback--

このルールは各リスト項目の `border` の外側のスペースに影響します。

---

各リスト項目の下に `15px` のスペースを追加します。

---

各リスト項目の上に `15px` のスペースを追加します。

### --feedback--

このルールは各リスト項目の下にスペースを追加し、上には追加しません。
