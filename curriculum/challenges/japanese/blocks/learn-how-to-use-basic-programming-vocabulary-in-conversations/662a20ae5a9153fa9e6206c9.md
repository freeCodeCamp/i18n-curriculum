# --description--

`Many` と `much` は数量について話すときに使われますが、異なる種類の名詞と一緒に使われます。`Many` は数えられる名詞に使われます。数えられる名詞とは、`apples`、`cars`、`books` のように数えることができるものです。例えば、`There are many books on the shelf.` と言うことができます。

`Much` は数えられない名詞に使われます。数えられない名詞とは、`water`、`sand`、`information` のように個別に数えることができないものです。例えば、`There is not much water left in the bottle.` と言うことがあります。

### --feedback--

`Water` は数えられない名詞なので、数量を表すときは `much` を使い、`many` は使わないでください。

### --feedback--

`Advice` は数えられない名詞なので、`much advice` ではなく `many advice` と言うのが正しいです。

### --feedback--

`People` は個々のものを指すときは数えられる名詞なので、`many` を使い、`much` は使わないでください。
