# --description--

公平な4面サイコロを1回スローし、その値を$T$とします。

$T$個の公平な6面サイコロをスローし、それらの合計点を記録します。合計を$C$とします。

$C$個の公平な8面サイコロをスローし、それらの合計点を記録します。合計を$O$とします。

$O$個の公平な12面サイコロをスローし、それらの合計点を記録します。合計を$D$とします。

$D$個の公平な20面サイコロをスローし、それらの合計点を記録します。合計を$I$とします。

$I$の分散を求め、小数点以下4桁に丸めて答えてください。
