# --interactive--

Einer der interessantesten Aspekte von CSS ist die Verwendung von Pseudoelementen. In diesem Zusammenhang bedeutet „pseudo“ „nicht echt“, also sind Pseudoelemente virtuelle oder synthetische Elemente, die keinem tatsächlichen HTML-Element direkt entsprechen. Sie ermöglichen es Ihnen, bestimmte Teile eines Elements zu stylen oder Inhalte einzufügen, ohne zusätzliches HTML hinzuzufügen.

Um ein Pseudoelement anzuwenden, fügen Sie es an den Selektor des ursprünglichen Elements mit einem doppelten Doppelpunkt (`::`) an. Beachten Sie, dass der Selektor jeder Typ sein kann, wie zum Beispiel ein Klassen- oder ID-Selektor. So sieht die grundlegende Syntax von Pseudoelementen aus:

```css
selector::pseudo-element {
  property: value;
}
```

Dieser doppelte Doppelpunkt unterscheidet Pseudoelemente von Pseudoklassen, die einen einfachen Doppelpunkt verwenden.

Pseudoelemente erlauben es Ihnen, bestimmte Teile des Inhalts eines Elements zu stylen oder Inhalte davor oder danach einzufügen, aber sie können nicht eigenständig existieren. Das Element, an das ein Pseudoelement angehängt ist, nennt man sein Ursprungselement.

Beginnen wir mit Beispielen für die `::before` und `::after` Pseudoelemente. Wie ihre Namen vermuten lassen, ermöglicht `::before` das Einfügen von Inhalten direkt vor dem Inhalt des Elements, während `::after` das Einfügen von Inhalten danach erlaubt.

Hier ist ein Beispiel für ein Button-Element. Im CSS verwenden wir absolute Positionierung und das `::before` Pseudoelement, um vor dem `Learn More` Text des Buttons einen Stern hinzuzufügen. Mehr über absolute Positionierung lernen Sie in zukünftigen Lektionen.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: lightseagreen;
  color: white;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
  position: relative;
}

.cta-button::before {
  content: "⭐";
  position: absolute;
  left: 3px;
  top: 8px;
  font-size: 0.75rem;
}
```

:::

Die `content` Eigenschaft wird verwendet, um den Inhalt darzustellen, den Sie vor dem Button-Text hinzufügen möchten. In diesem Beispiel fügen wir einen Stern hinzu. Sie werden feststellen, dass Sie nicht nur Inhalte einfügen, sondern diese auch stylen können. Hier ist ein Beispiel für das `::after` Pseudoelement mit demselben Button:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: '➡️';
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}
```

:::

Im `::after` Pseudoelement wird die `transition` Eigenschaft verwendet, um Änderungen über 0,3 Sekunden mit einem Ease-Effekt zu animieren, wodurch eine sanfte und allmähliche Transformation entsteht statt einer plötzlichen. Mehr über die `transition` Eigenschaft lernen Sie in zukünftigen Lektionen.

Sie können auch eine Pseudoklasse an den Inhalt anhängen, den Sie mit den `::before` und `::after` Pseudoelementen einfügen. Zum Beispiel einen Hover-Zustand für den Inhalt. Hier ist ein Beispiel:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: "➡️";
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}

.cta-button:hover::after {
  transform: translateX(2px);
}
```

:::

Mit `transform: translateX(2px)` im Hover-Zustand wird der Inhalt jedes Mal um `2px` nach rechts verschoben, wenn der Benutzer über den Button hovert. Die transition-Eigenschaft im `::after` selbst sorgt dafür, dass der Vorgang `0.3s` dauert.

Genau das macht die `transform` Eigenschaft – sie ermöglicht es Ihnen, ein Element zu drehen, zu schrägen, zu skalieren oder in eine bestimmte Richtung zu verschieben. Mehr darüber lernen Sie in zukünftigen Lektionen.

Im nächsten Beispiel betrachten wir das `::first-letter` Pseudoelement. Das `::first-letter` Pseudoelement zielt auf den ersten Buchstaben des Inhalts eines Elements ab und erlaubt es Ihnen, diesen zu stylen. Hier ist ein Beispiel mit etwas Absatztext. Wenn wir den ersten Buchstaben stylen wollen, können wir das `::first-letter` Pseudoelement so verwenden:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>freeCodeCamp lets you learn to code without having to pay.</p>
```

```css
p::first-letter {
  font-size: 4rem;
}
```

:::

Im letzten Beispiel betrachten wir das `::marker` Pseudoelement, mit dem Sie den Marker, das Aufzählungszeichen oder die Nummerierung von Listenelementen zum Stylen auswählen können. Das `::marker` Pseudoelement bietet eine Möglichkeit, die Markenidentität Ihrer Website zu verbessern, indem Sie Listensymbole an Ihr Farbschema anpassen.

Hier ist ein Beispiel für eine ungeordnete Liste und eine geordnete Liste. Um die Farbe und Größe des Listensymbols zu ändern, können Sie das `::marker` Pseudoelement so verwenden:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<ul>
  <li>Unordered list item 1</li>
  <li>Unordered list item 2</li>
  <li>Unordered list item 3</li>
  <li>Unordered list item 4</li>
</ul>

<ol>
  <li>Ordered list item 1</li>
  <li>Ordered list item 2</li>
  <li>Ordered list item 3</li>
  <li>Ordered list item 4</li>
</ol>
```

```css
li::marker {
  color: crimson;
  font-size: 1.5em;
  font-weight: bold;
}
```

:::

In dieser Lektion haben wir nur einige wenige Pseudoelemente behandelt. Aber es gibt viele weitere wie `::placeholder`, `::spelling-error` und `::selection`, die ich Ihnen empfehle, selbst zu erkunden.
