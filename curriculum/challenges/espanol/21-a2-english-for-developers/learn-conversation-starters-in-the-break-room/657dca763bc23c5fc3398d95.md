# --description--

`Probably` se utiliza cuando piensas que algo es probable que suceda, pero no estás 100 % seguro.

Por ejemplo, podrías decir: `The code will probably run without errors after the update`, porque crees eso, pero todavía hay una pequeña posibilidad de un problema.
