# --description--

前のレッスンでは、ローカルデバイスにPythonをインストールする方法を学びました。このレッスンでは、Pythonコードをローカルで実行する方法を学びます。

Pythonスクリプトをローカルで作成し始める前に、コードエディターまたはIDEをインストールする必要があります。IDEとは統合開発環境のことで、コードエディター、テストツール、ターミナルなどの機能が含まれています。

多くの開発者が使う人気のコードエディターの一つがVS Codeです。`https://code.visualstudio.com/download` にアクセスして、Mac、Windows、Linux環境に対応したVS Codeをダウンロードしてください。このレッスンではこのエディターを使用します。

Python開発でよく使われる他の選択肢には以下があります：

- PyCharm
- Spyder

エディターをダウンロードしたら、それを開く必要があります。開いたら、フォルダーを開く必要があります。左上の`File`メニューオプションをクリックし、`Open Folder`をクリックしてください。そこから、プロジェクト用のフォルダーを選択または作成できます。レッスン外での練習用に、`python-projects`という名前のフォルダーを作るのも良いでしょう。

フォルダーの準備ができたら、VS Codeの左サイドバーにあるエクスプローラーを開けます。`New File`アイコン（紙にプラスのアイコンが重なったような形）をクリックしてください。次に、`main.py`というファイルを作成し、Enterを押します。`.py`はPythonファイルであることを示すファイル拡張子です。

`main.py`ファイルを開き、次のコードを入力してください：

```py
print("Hello, world!")
```

このコードを実行するにはいくつかの方法があります。最初の方法は、右上にある再生ボタンのような実行ボタンをクリックすることです。これでターミナルが開き、Pythonスクリプトが実行されます。`"Hello, world!"`というテキストが表示されるはずです。

もう一つの方法は、ターミナルを開いて手動で次のコマンドを使ってプログラムを実行することです：

```bash
python main.py
```

このコマンドは、`main.py`が保存されているフォルダーから実行する必要があります。例えば、`main.py`が`python-projects`というフォルダー内にある場合、まずターミナルで次のコマンドを使います：

```bash
cd python-projects
```

そして次に実行します：

```bash
python main.py
```

特にmacOSやLinuxの環境では、Python 2とPython 3の両方がインストールされている場合、`python3`の代わりに`python`を使う必要がある場合があります。これは`python`コマンドが存在しないか、古いバージョンのPythonを指している場合によくあります。

```bash
python3 main.py
```

コースを進める中で、レッスンの例をローカルで試して結果を確認してください。また、レッスンで学んだことを試すために自分で例を考えてみることもお勧めします。
