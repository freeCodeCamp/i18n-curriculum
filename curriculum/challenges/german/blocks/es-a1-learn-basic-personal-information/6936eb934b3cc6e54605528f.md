# --explanation--

Camila sagt `¿Cuál es tu correo electrónico?` Die Struktur folgt diesem Muster:

`¿Cuál es tu` + Substantiv?

Diese Struktur wird verwendet, um nach spezifischen persönlichen Informationen zu fragen. In diesem Fall bezieht sich `correo electrónico` auf eine E-Mail-Adresse, die zum Senden und Empfangen von Nachrichten online genutzt wird.
