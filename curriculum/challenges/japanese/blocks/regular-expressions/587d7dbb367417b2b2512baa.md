# --description--

複数回繰り返される単語にマッチさせたいとします。

```js
let repeatStr = "row row row your boat";
```

`/row row row/`を使うこともできますが、繰り返される特定の単語がわからない場合はどうでしょうか？<dfn>キャプチャグループ</dfn>を使うと繰り返される部分文字列を見つけられます。

キャプチャグループは、キャプチャしたい正規表現パターンを丸括弧で囲むことで作成します。この場合、英数字で構成される単語をキャプチャするのが目的なので、キャプチャグループは丸括弧で囲まれた`\w+`になります：`/(\w+)/`。

グループにマッチした部分文字列は一時的な「変数」として保存され、同じ正規表現内でバックスラッシュとキャプチャグループの番号（例：`\1`）を使ってアクセスできます。キャプチャグループは開き丸括弧の位置（左から右）によって自動的に番号付けされ、1から始まります。

以下の例は、スペースで区切られた3回繰り返される単語にマッチします：

```js
let repeatRegex = /(\w+) \1 \1/;
repeatRegex.test(repeatStr); // Returns true
repeatStr.match(repeatRegex); // Returns ["row row row", "row"]
```

文字列に対して`.match()`メソッドを使うと、マッチした部分文字列とキャプチャされたグループを含む配列が返されます。


# --instructions--

`reRegex`でキャプチャグループを使い、同じ数字が単一スペースで区切られてちょうど3回繰り返される文字列にマッチさせてください。

# --hints--

正規表現には数字のショートハンド文字クラスを使うべきです。

```js
assert(reRegex.source.match(/\\d/));
```

キャプチャグループを2回再利用する必要があります。

```js
assert(reRegex.source.match(/\\1|\\2/g).length >= 2);
```

正規表現は文字列`42 42 42`にマッチするべきです。

```js
reRegex.lastIndex = 0;
assert(reRegex.test('42 42 42'));
```

正規表現は文字列`100 100 100`にマッチするべきです。

```js
reRegex.lastIndex = 0;
assert(reRegex.test('100 100 100'));
```

正規表現は文字列`42 42 42 42`にマッチしてはいけません。

```js
assert.equal('42 42 42 42'.match(reRegex.source), null);
```

正規表現は文字列`42 42`にマッチしてはいけません。

```js
assert.equal('42 42'.match(reRegex.source), null);
```

正規表現は文字列`101 102 103`にマッチしてはいけません。

```js
reRegex.lastIndex = 0;
assert(!reRegex.test('101 102 103'));
```

正規表現は文字列`1 2 3`にマッチしてはいけません。

```js
reRegex.lastIndex = 0;
assert(!reRegex.test('1 2 3'));
```

正規表現は文字列`10 10 10`にマッチするべきです。

```js
reRegex.lastIndex = 0;
assert(reRegex.test('10 10 10'));
```

正規表現は文字列`42\t42\t42`にマッチしてはいけません。

```js
reRegex.lastIndex = 0;
assert(!reRegex.test('42\t42\t42'));
```

正規表現は文字列`42  42  42`にマッチしてはいけません。

```js
reRegex.lastIndex = 0;
assert(!reRegex.test('42  42  42'));
```
