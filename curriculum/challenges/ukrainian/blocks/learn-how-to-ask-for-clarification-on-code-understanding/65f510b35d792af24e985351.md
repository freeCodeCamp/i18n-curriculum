# --description--

У програмуванні термін `persistent` часто означає щось, що продовжує існувати або відбуватися, особливо проблему чи помилку, яку важко вирішити.

`Persistence` — це іменник, що позначає стан постійності.

Наприклад, `persistent error` у коді — це помилка, яка постійно виникає і її важко виправити.
