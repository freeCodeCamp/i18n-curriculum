# --interactive--

Il selettore di attributi in CSS è uno strumento potente che ti permette di selezionare elementi HTML in base ai loro attributi. Questo significa che puoi applicare stili agli elementi che hanno attributi specifici o anche determinati valori per quegli attributi.

È particolarmente utile quando vuoi definire lo stile degli elementi in modo dinamico o quando i nomi delle classi da soli non offrono abbastanza specificità.

Ad esempio, puoi usare il selettore di attributi per selezionare tutti i link che hanno un attributo `href`. Questo è particolarmente comodo quando vuoi applicare uno stile uniforme a tutti i link di una pagina.

Ecco come puoi usare il selettore di attributi per selezionare i link con l'attributo `href`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Example link with an href attribute</a>
<a>Example link without an href attribute</a>
```

```css
a {
  display: block;
}

a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

Questa regola CSS applicherà un colore blu e una sottolineatura a qualsiasi link che include un attributo `href`, assicurando che tutti i link cliccabili siano stilizzati in modo coerente.

Ma i selettori di attributi possono essere più specifici. Supponiamo che tu voglia selezionare solo i link che hanno un attributo `title`. L'attributo `title` spesso fornisce informazioni aggiuntive sul link, e potresti voler stilizzare questi link in modo diverso per indicare che hanno informazioni extra disponibili. Ecco come puoi farlo:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

Questa regola applica testo in grassetto e rimuove la sottolineatura da qualsiasi link con un attributo `title`. È un ottimo modo per differenziare visivamente questi link dagli altri nella pagina.

Puoi anche combinare i selettori di attributi per ottenere una precisione ancora maggiore. Per esempio, se vuoi stilizzare i link che hanno sia gli attributi `href` che `title`, puoi fare così:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[href][title] {
  display:block;
  color: green;
}
```

:::

In questo caso, solo i link che contengono entrambi gli attributi `href` e `title` saranno stilizzati con testo verde. Questo livello di controllo è ciò che rende i selettori di attributi così potenti in CSS.

Un altro esempio di selettore di attributi è quello che corrisponde a un singolo valore all’interno di una lista di valori separati da spazi in un attributo.

Ecco un esempio di un elemento àncora con più classi:

```html
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

Per selezionare questo specifico elemento àncora, puoi usare il seguente selettore:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

La sintassi `[attr~=value]` viene usata qui per selezionare tutti gli elementi àncora in cui l'attributo class contiene la parola `"primary"`.

Se hai bisogno di selezionare un elemento in cui il valore dell'attributo è preceduto da un valore specifico, puoi usare la sintassi `[attr^=value]`. Ecco un esempio:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

In questo esempio, il selettore `a[href^="https://"]` selezionerà tutti gli elementi àncora in cui il valore dell'attributo `href` inizia con `"https://"`.

Per selezionare gli elementi in cui il valore dell'attributo termina con un valore specifico, puoi usare la sintassi `[attr$=value]`. Ecco un esempio:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Visit Example Site</a>
```

```css
a[href$=".com"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

In questo esempio, il selettore `a[href$=".com"]` selezionerà tutti gli elementi àncora in cui il valore dell'attributo `href` termina con `.com`.

Usare i selettori di attributi non solo migliora la definizione dello stile della tua pagina web ma aumenta anche l’accessibilità rendendo gli elementi interattivi come i link più distinguibili in base ai loro attributi.
