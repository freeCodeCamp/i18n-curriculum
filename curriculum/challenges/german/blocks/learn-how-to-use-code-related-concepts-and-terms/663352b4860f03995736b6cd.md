# --description--

`Store information` bedeutet, Daten an einem bestimmten Ort zu behalten oder zu speichern, normalerweise in einer Variable. Variablen sind wie Container in Ihrem Code, die Informationen wie Zahlen, Text oder andere Datentypen enthalten. Sie können sich diese als benannte Speicherplätze vorstellen, auf die Sie später in Ihrem Code auf die `stored information` zugreifen können.

Zum Beispiel `int age = 30;` Hier speichert die Variable `age` die Zahl 30.

Das Wort `like` wird als Präposition verwendet, um Beispiele zu geben oder ähnliche Dinge zu vergleichen.

Zum Beispiel, wenn Sie sagen `I love fruits like apples and oranges.` verwenden Sie `like`, um Beispiele für die Art von Früchten zu geben, die Sie lieben, und deuten an, dass es möglicherweise noch mehr Früchte auf dieser Liste gibt.
