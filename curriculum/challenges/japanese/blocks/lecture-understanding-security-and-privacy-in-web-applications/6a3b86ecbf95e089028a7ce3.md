# --description--

ユーザープライバシーとは、個人データがどのように収集、使用、共有されるかをコントロールする権利のことです。現代のプライバシーフレームワークは、デジタル上の足跡を保護するためにいくつかの重要な要素に依存しています。

プライバシー保護の中心には**個人を特定できる情報（PII）**があります。PIIとは、あなたの身元を直接的または間接的に特定できるあらゆるデータのことです。

直接的な識別子には氏名、住所、国民識別番号が含まれ、間接的な識別子にはIPアドレス、デバイスのシリアル番号、正確な位置履歴などがあります。

このデータを安全に保つために、企業は**機密性**を維持する必要があります。機密性は、強力な暗号化とアクセス制御を使って、あなたのプライベートデータが不正アクセスから守られることを保証します。

もし不正な第三者があなたの記録を読み取ったりPIIを漏洩させたりした場合、機密性は破られたことになります。

プライバシーフレームワークはこれを**データ最小化**と**目的制限**によって実現しています。データ最小化は、企業が特定のタスクを完了するために必要な最小限のPIIだけを収集すべきだと定めています。

目的制限は、企業が特定の理由でデータを収集した後、そのデータを広告ネットワークへの連絡先情報の販売など、他の目的で使用することを法的に禁止しています。

これらの原則は侵入的なデジタルトラッキングに直接対抗します。**トラッキング**とは、インターネット上でのあなたのオンライン行動を継続的に監視することです。

広告ネットワークはスクリプト、デバイスフィンガープリンティング、ブラウザ履歴ログを使ってあなたの行動を追跡し、あなたの知らないうちに詳細なプロファイルを作成します。

このトラッキングから守るために、ユーザーの同意が個人のコントロールの基盤となります。本当のプライバシーは、情報に基づいた明確な同意を必要とし、強制されるのではなく自発的にデータトラッキングに参加することを意味します。

最後に、**プライバシー法は「忘れられる権利」**、別名データ消去を保証します。この概念は、アカウントを閉じた後に組織に対してあなたの個人記録やトラッキング履歴をデータベースから永久に削除するよう法的に要求する権利を与えます。

これらのプライバシー法とそれを担当する機関については次の講義で学びます。
