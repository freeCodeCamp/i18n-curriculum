# --description--

`UX` sta per user experience. Indica come una persona si sente quando usa un prodotto, un sistema o un servizio. Questo include siti web, app e dispositivi elettronici. Una `UX designer` rende le cose facili e piacevoli da usare al computer o al telefono.

Per esempio: `The UX of this website is excellent because it's easy to navigate and find the information you need.`
