## 使用浮點數

- **定義**：浮動用於將元素從頁面上的正常流程中移除，並將其定位在其容器的左側或右側。當發生這種情況時，文字會環繞該浮動內容。

```css
float: left;
float: right;
```

- **清除浮動**：`clear` 屬性用於判斷元素是否需要移動到浮動內容的下方。當你有多個浮動元素並排堆疊時，版面配置可能會出現重疊和塌陷的問題。因此，產生了 `clearfix` 技巧來解決這個問題。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="clearfix">
  <div class="box left">Left</div>
  <div class="box right">Right</div>
</div>
```

```css
.box {
  width: 100px;
  height: 60px;
  color: white;
  text-align: center;
  line-height: 60px;
}

.left {
  float: left;
  background: teal;
}

.right {
  float: right;
  background: purple;
}

.clearfix::after {
  content: "";
  display: block;
  clear: both;
}
```

:::

## 靜態、相對和絕對定位

- **靜態定位**：這是文件的正常流程。元素依序從上到下、從左到右定位。
- **相對定位**：這允許你使用 `top`、`left`、`right` 和 `bottom` 屬性在正常文件流程中定位元素。你也可以使用相對定位讓元素與頁面上的其他元素重疊。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="relative">Relative Element</div>
```

```css
.relative {
    position: relative;
    top: 30px;
    left: 30px;
}
```

:::

- **絕對定位**：這讓你可以將元素從正常的文件流程中取出，使其行為獨立於其他元素。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  Parent
  <div class="positioned">Absolute</div>
</div>
```

```css
.parent {
  position: relative;
  height: 120px;
  background: #eee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

## 固定與黏性定位

- **固定定位**：當元素使用 `position: fixed` 定位時，它會從正常的文件流程中移除，並相對於視窗定位，這表示即使使用者滾動，該元素仍會保持在相同位置。這通常用於像標頭或導覽列這類需要始終保持可見的元素。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="navbar">Fixed Navbar</div>
<div style="height: 600px;"></div>
```

```css
.navbar {
  position: fixed; 
  top: 0; 
  width: 100%;
  background: #0077ff;
  color: white;
  padding: 8px; 
}
```

:::

- **Sticky Positioning**：這種類型的定位會作為相對定位和固定定位的混合體。最初，元素的行為就像是相對定位，保持在文件的流程中。然而，一旦使用者將元素捲動過某個點，它就會「黏」在視窗（通常是頂部），並且行為就像是固定定位。這非常適合用來建立像是黏性導覽列這類元素，只有當使用者捲動到某個位置時才會變成固定。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Scroll down</p>
<div class="positioned">Sticky Element</div>
<div style="height: 500px;"></div>
```

```css
.positioned {
  position: sticky;
  top: 30px;
  left: 30px;
  background: #4caf50;
  color: white;
  padding: 6px;
}
```

:::

## 使用 `z-index` 屬性

- **定義**：CSS 中的 `z-index` 屬性用於控制頁面上重疊定位元素的垂直堆疊順序。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box box1">1</div>
  <div class="box box2">2</div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

.box {
  position: absolute;
  width: 100px;
  height: 100px;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

.box1 {
  background: lightcoral;
  top: 20px;
  left: 20px;
  z-index: 1;
}

.box2 {
  background: steelblue;
  top: 40px;
  left: 40px;
  z-index: 2;
}
```
