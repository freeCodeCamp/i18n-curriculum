# --questions--

## --text--

Welche der folgenden Möglichkeiten ist der richtige Weg, um jeden Buchstaben im String `"Giraffe Academy"` zu durchlaufen?

## --answers--

```python
for (letter) in "Giraffe Academy":
    print(letter)
```

---

```python
for key, letter in "Giraffe Academy":
    print(letter)
```

---

```python
for letter in "Giraffe Academy":
    print(letter)
```

---

```python
for letter, value in "Giraffe Academy":
    print(letter)
```
