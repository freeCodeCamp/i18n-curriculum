# --description--

D3 bietet mehrere Methoden, mit denen Sie Elemente in Ihrem Dokument hinzufügen und ändern können.

Die `select()`-Methode wählt ein Element aus dem Dokument aus. Sie nimmt ein Argument für den Namen des gewünschten Elements und gibt einen HTML-Knoten für das erste Element im Dokument zurück, das dem Namen entspricht. Hier ist ein Beispiel:

```js
const anchor = d3.select('a');
```

Das obige Beispiel findet das erste Anker-Tag auf der Seite und speichert einen HTML-Knoten dafür in der Variablen `anchor`. Sie können die Auswahl mit anderen Methoden verwenden. Der `d3`-Teil des Beispiels ist eine Referenz auf das D3-Objekt, über das Sie auf D3-Methoden zugreifen.

Zwei weitere nützliche Methoden sind `append()` und `text()`.

Die `append()`-Methode nimmt ein Argument für das Element, das Sie dem Dokument hinzufügen möchten. Sie fügt einem ausgewählten Element einen HTML-Knoten hinzu und gibt einen Verweis auf diesen Knoten zurück.

Die `text()`-Methode setzt entweder den Text des ausgewählten Knotens oder gibt den aktuellen Text zurück. Um den Wert zu setzen, übergeben Sie einen String als Argument innerhalb der runden Klammern der Methode.

Hier ist ein Beispiel, das eine ungeordnete Liste auswählt, ein Listenelement anhängt und Text hinzufügt:

```js
d3.select('ul')?.append('li').text('Very important item');
```

D3 erlaubt es Ihnen, mehrere Methoden mit Punkten zu verketten, um eine Reihe von Aktionen nacheinander auszuführen.
