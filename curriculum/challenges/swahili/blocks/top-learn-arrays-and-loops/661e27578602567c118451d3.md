# --description--

Moja ya njia zinazotumika sana kuongeza kipengele kipya katika safu ya data ni kwa kutumia njia ya `push()`. Njia ya `push()` huongeza kipengele kimoja au zaidi mwishoni mwa safu ya data na hurudisha urefu mpya wa safu hiyo.

Kwa mfano, kuongeza kipengele kipya katika safu ya data ya `pet`, unaweza kutumia msimbo ufuatao:

```javascript
const pet = ['cat', 'dog', 'bunny'];
pet.push('parrot');
console.log(pet); // Output: ['cat', 'dog', 'bunny', 'parrot']
```

Kuondoa kipengele cha mwisho katika safu ya data, unaweza kutumia njia ya `pop()`. Njia ya `pop()` huondoa kipengele cha mwisho kutoka katika safu ya data na hurudisha kipengele hicho.

Kwa mfano, kuondoa kipengele cha mwisho kutoka katika safu ya data ya `pet`, unaweza kutumia msimbo ufuatao:

```javascript
const pet = ['cat', 'dog', 'tiger'];
pet.pop();
console.log(pet); // Output: ['cat', 'dog']
```

## --text--

```javascript

animals.push('shark', 'bear');
const removed = animals.pop();

console.log(animals);
console.log(removed);
```
