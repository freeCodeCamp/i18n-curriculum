# --description--

In der folgenden Gleichung sind $x$, $y$ und $n$ positive ganze Zahlen.

$$\frac{1}{x} + \frac{1}{y} = \frac{1}{n}$$

Für eine Grenze $L$ definieren wir $F(L)$ als die Anzahl der Lösungen, die $x < y ≤ L$ erfüllen.

Wir können überprüfen, dass $F(15) = 4$ und $F(1000) = 1069$ gilt.

Finden Sie $F({10}^{12})$.
