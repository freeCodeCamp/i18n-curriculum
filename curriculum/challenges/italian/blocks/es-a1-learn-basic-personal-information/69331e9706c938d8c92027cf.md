# --explanation--

Ángela dice `Perfecto. Entonces, Esteban Navarro Huamán.`

`Perfecto` si usa per mostrare accordo o soddisfazione. Indica che l’informazione è corretta.

`Entonces` si usa dopo aver ascoltato un’informazione. Aiuta l’interlocutore a organizzare le idee e a ripetere chiaramente l’informazione.

Qui `entonces` introduce una conferma. Ángela ripete il nome completo per verificare che sia corretto.

Dice `Huamán`, non `Guzmán`. Quindi, l’istruzione è falsa.
