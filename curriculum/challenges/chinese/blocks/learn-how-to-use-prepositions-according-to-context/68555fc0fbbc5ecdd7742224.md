# --explanation--

`On` 用于短语 `planning on` 中，意思是打算或期望做某事。例如：

`We're planning on launching next week.` – 这意味着团队预计或打算这么做。

`Bringing in` 意味着将人员添加到团队或组中。例如：

`The manager is bringing in a new developer.` – 这意味着有人被添加了。

`Additional` 意味着比现有的更多。例如：

`We need additional help to finish the project.` – 这意味着需要额外的帮助。
