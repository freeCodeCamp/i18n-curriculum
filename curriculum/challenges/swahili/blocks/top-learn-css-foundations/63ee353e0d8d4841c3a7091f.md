# --description--

Sawa, umejifunza mengi hadi sasa. Kitu pekee kilichobaki kwa sasa ni kujifunza jinsi ya kuongeza CSS yote hii kwenye HTML yako. Kuna njia tatu za kufanya hivyo.

CSS ya nje ndiyo njia inayotumika zaidi utakayokutana nayo, na inahusisha kuunda faili tofauti kwa ajili ya CSS na kuiunganisha ndani ya lebo za kufungua na kufunga za `<head>` za HTML kwa kutumia kipengele cha `<link>` (ambacho ni kipengele kisicho na lebo ya kufunga):

Kwanza, unaongeza kipengele kisicho na lebo ya kufunga cha `<link>` ndani ya lebo za kufungua na kufunga za `<head>` za faili ya HTML. Sifa ya `href` ni mahali faili ya CSS iko, iwe ni URL kamili au, unayotumia hapa, URL inayohusiana na mahali faili ya HTML iko. Katika mfano hapo juu, unadhani mafaili yote mawili yako katika saraka moja. Sifa ya `rel` ni muhimu, na inaelezea uhusiano kati ya faili ya HTML na faili iliyounganishwa.

Kisha ndani ya faili mpya ya `styles.css`, una kichaguzi (`div` na `p`), kifuatiwa na mabano ya mviringo ya kufungua na kufunga, ambayo huunda "kizuizi cha tamko". Mwishowe, unaweka tamko lolote ndani ya kizuizi cha tamko. `color: white; ` ni tamko moja, ambapo `color` ni sifa na `white` ni thamani, na `background-color: black;` ni tamko lingine.

Kumbuka kuhusu majina ya faili: `styles.css` ni jina ulilochagua kwa faili hapa. Unaweza kuitaja faili kwa jina lolote unalotaka mradi tu aina ya faili iwe `.css`, ingawa "style" au "styles" ndizo zinazotumika zaidi.

Faida chache za njia hii ni:

1. Inatenga HTML na CSS yako, jambo ambalo hufanya faili ya HTML kuwa ndogo na kuonekana safi zaidi.
2. Unahitaji kuhariri CSS mahali pamoja tu, jambo ambalo ni rahisi hasa kwa tovuti zilizo na kurasa nyingi zinazoshiriki mitindo sawa.

## --answers--

Inaelezea mahali faili ya CSS iko ikilinganishwa na mahali faili ya HTML iko.

---

Inaelezea uhusiano kati ya faili ya HTML na faili iliyounganishwa.

Inaelezea aina ya faili inayounganishwa (mfano, "stylesheet").
