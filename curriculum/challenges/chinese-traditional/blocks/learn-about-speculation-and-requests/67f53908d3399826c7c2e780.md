# --explanation--

`Want` 表示想要做某事。在過去式中，你說 `wanted` 以表示該慾望存在於較早的時間。它後面接動詞不定詞（帶有 `to`）。例如：

- `She wants to install the new software.`－她希望將新的軟體安裝到系統中。

- `They wanted to upgrade their system last week。` - 過去，他們計劃升級系統。
