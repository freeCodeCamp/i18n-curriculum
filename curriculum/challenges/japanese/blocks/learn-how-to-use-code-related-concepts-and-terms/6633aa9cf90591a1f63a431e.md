# --description--

接続詞 `so` は、これまでに述べたことに関連する結果や効果を示すために使われます。文の2つの部分の間に因果関係を示すことが多いです。

例えば、`It was raining, so we canceled the picnic.` これは、雨のためにピクニックが中止されたことを示しています。

# --questions--

## --text--

どの文が `so` を正しく使っていますか？

## --answers--

`I was hungry so I am eating.`

### --feedback--

この文はほぼ `so` を正しく使っていますが、後半の時制は前半と一致させる必要があります。正しくは `I was hungry, so I ate.` と言うべきです。

---

`She is late to work, so she misses the meeting.`

### --feedback--

`so` の使い方は結果を示唆していますが、時制が一致していません。正しくは `She is late to work, so she will miss the meeting.` です。

---

`It started raining, so we took shelter.`

---

`He likes to read books, so he buys a lamp.`

### --feedback--

この文は `so` を正しく使っていません。なぜなら、ランプを買うことは本を読むのが好きなことの直接的な結果ではないからです。より良い使い方は `He likes to read at night, so he bought a lamp.` かもしれません。
