# --description--

如果呼叫 `function`，但沒有提供引數，則參數會被指定為 `undefined` 的值。如果函式（程式）沒有設計來控制代碼這種情況，可能會導致意外結果。

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal()); // Output: undefined is my favorite animal!
```

在上述範例中，函式（程式）`favoriteAnimal` 在沒有引數的情況下被呼叫。參數 `animal` 被指定為 `undefined`，然後與字串 `" is my favorite animal!"` 連接。

如果沒有提供值，你可以使用預設參數。預設參數允許你在呼叫函式時，如果沒有提供值，為參數指定預設值。

```js
function favoriteAnimal(animal = 'Giant Panda') {
  return animal + " is my favorite animal!"
}
```

在上述範例中，參數 `animal` 的預設值被設為 `'Giant Panda'`。如果函式（程式）`favoriteAnimal` 在沒有引數的情況下被呼叫，參數 `animal` 將會被指定為 `'Giant Panda'`。
