# --description--

前のステップで作成した食べ物と運動の `fieldset` 要素それぞれに対して、select メニューにオプションを追加する必要があります。`option` 要素を使って、各 `fieldset` に対する新しいオプションを作成してください。各オプションの `value` 属性は `id` の `fieldset` であり、オプションのテキストは `legend` のテキストにしてください。

`Breakfast` のオプションを `selected` のオプションとして設定してください。
