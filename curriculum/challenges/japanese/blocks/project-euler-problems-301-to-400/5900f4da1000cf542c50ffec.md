# --description--

二項係数 $\displaystyle\binom{{10}^{18}}{{10}^9}$ は90億（$9 × {10}^9$）桁を超える数字です。

$M(n, k, m)$ を二項係数 $\displaystyle\binom{n}{k}$ のモジュラス $m$ による剰余とします。

$1000 < p < q < r < 5000$ かつ $p$, $q$, $r$ が素数のとき、$\sum M({10}^{18}, {10}^9, p \times q \times r)$ を計算してください。
