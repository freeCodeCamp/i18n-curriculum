# --explanation--

Dopo che Lisa menziona il progetto, Bob fa una domanda per saperne di più sulle persone coinvolte. Bob usa la contrazione `Who's`, che sta per `Who is`, per chiedere informazioni sui membri del team.

Fai attenzione alla seconda parte della conversazione, dove Bob si concentra sul team, non sul progetto in sé.
