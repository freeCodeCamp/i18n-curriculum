# --description--

在这个任务中，你将学习短语 `to be willing`。

当有人说他们 `willing` 做某事时，意味着他们已经准备好并愿意去做，通常出于善意或合作。例如，`Brian is willing to help Sophie` 意味着 Brian 已经准备好并乐意帮助 Sophie。

这个短语常用于专业和个人上下文中，以表达对做某事或帮助某人的积极态度。
