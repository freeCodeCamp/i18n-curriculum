# --description--

$r ≤ n$인 60° 삼각형의 개수를 $T(n)$이라 할 때, $T(100) = 1234$, $T(1000) = 22767$, 그리고 $T(10000) = 359912$입니다.

$T(1053779)$를 구하세요.
