# --description--

`introductory` 部分は、文書の冒頭で主要なトピックを紹介します。例：`The introductory chapter of the book outlines the key concepts.`

`paragraph` は特定のテーマに関する文章のセクションです。例えば、`Each paragraph in the blog post focuses on a different technology trend.`

`To cover` は特定の内容を扱う、または含むことを意味します。例えば、技術ガイドでは`This section covers the steps to install software.`

### --feedback--

結論は文書の最後の部分で、述べたポイントをまとめるものです。`introductory paragraph` は主要なトピックを紹介します。

### --feedback--

情報源は本や記事など情報の出所です。重要ですが、`introductory paragraph` は文書の主要なトピックを紹介することが主な目的であり、単に情報源を一覧にすることではありません。

---

複雑なチャートやグラフを表示すること

### --feedback--

チャートやグラフはデータを視覚的に示すツールです。文書に登場する場合がありますが、`introductory paragraph` の目的はトピックを紹介することであり、データの視覚化ではありません。
