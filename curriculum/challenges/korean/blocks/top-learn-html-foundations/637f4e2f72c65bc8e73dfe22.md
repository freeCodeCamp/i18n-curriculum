# --description--

`<head>` 요소는 웹페이지에 중요한 메타 정보를 넣는 곳이며, 웹페이지가 브라우저에서 올바르게 렌더링되도록 필요한 내용을 포함합니다. `<head>` 내부에는 웹페이지에 내용을 표시하는 요소를 사용하면 안 됩니다.

## 문자 인코딩 메타 요소
웹페이지의 `meta` 인코딩을 위해 항상 `charset` 요소 안에 `head` 태그를 넣어야 합니다: `<meta charset="utf-8">`.

인코딩 설정은 매우 중요한데, 이것이 있어야 브라우저에서 특수 기호와 다양한 언어의 문자가 올바르게 표시됩니다.

## 제목 요소
HTML 문서의 헤드에 항상 포함해야 하는 또 다른 요소는 `title` 요소입니다:

```html
<title>My First Webpage</title>
```

`title` 요소는 웹페이지에 사람이 읽을 수 있는 제목을 부여하는 데 사용되며, 이 제목은 웹페이지의 브라우저 탭에 표시됩니다.

만약 `title` 요소를 포함하지 않으면, 웹페이지 제목은 기본적으로 파일 이름이 됩니다. 이 경우 `index.html`가 되는데, 이는 사용자에게 별로 의미가 없으며, 사용자가 여러 브라우저 탭을 열었을 때 웹페이지를 찾기 어렵게 만듭니다.

HTML 문서의 `head` 안에는 더 많은 요소가 들어갈 수 있습니다. 하지만 지금은 여기서 다룬 두 가지 요소만 아는 것이 중요합니다. 나머지 커리큘럼을 진행하면서 `head` 안에 들어가는 더 많은 요소를 소개할 것입니다.

`index.html` 파일로 돌아가서, `head` 요소 안에 `charset` `meta` 요소와 `title` 요소를 추가해 봅시다. head 요소는 HTML 요소 안에 들어가며, 항상 `<html>` 태그 바로 아래 첫 번째 요소여야 합니다:

```html
<!DOCTYPE html>

<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Webpage</title>
  </head>
</html>
```
