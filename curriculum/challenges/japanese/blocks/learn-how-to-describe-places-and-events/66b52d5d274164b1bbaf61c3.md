# --explanation--

`Present Continuous`時制は、話している瞬間に起こっている動作を表すために使われます。動詞`to be`（`am, is, are`）＋動詞の基本形＋`-ing`で構成されます。例えば：

- `She is coding a new feature right now.` - 彼女は今まさに新しい機能のコーディングをしています。

- `They are testing the software.` - 彼らは今まさにソフトウェアのテストをしています。
