# --explanation--

`Couple` bedeutet normalerweise zwei, kann aber manchmal auch eine kleine Anzahl oder `some` bedeuten. Zum Beispiel:

- `A couple of days` – Es kann zwei Tage oder einige Tage bedeuten.

- `A couple of friends` – Es kann zwei Freunde oder einige Freunde bedeuten.
