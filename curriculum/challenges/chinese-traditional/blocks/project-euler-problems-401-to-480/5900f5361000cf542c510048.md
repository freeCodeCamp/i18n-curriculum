# --description--

令 $f(n) = n^2 - 3n - 1$。

設 $p$ 為一個質數。

設 $R(p)$ 為最小的正整數 $n$，使得若存在此整數 $n$，則 $f(n)\bmod p^2 = 0$，否則 $R(p) = 0$。

令 $SR(L)$ 為所有不超過 $L$ 的質數的 $R(p)$ 之和。

尋找 $SR({10}^7)$。
