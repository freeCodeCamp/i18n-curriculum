# --description--

Betrachten Sie die unendliche Polynomreihe $A_G(x) = xG_1 + x^2G_2 + x^3G_3 + \cdots$, wobei $G_k$ das $k$-te Glied der Rekursionsfolge zweiter Ordnung $G_k = G_{k − 1} + G_{k − 2}, G_1 = 1$ und $G_2 = 4$ ist; das heißt, $1, 4, 5, 9, 14, 23, \ldots$.

Für diese Aufgabe interessieren wir uns für Werte von $x$, für die $A_G(x)$ eine positive ganze Zahl ist.

Die entsprechenden Werte von $x$ für die ersten fünf natürlichen Zahlen sind unten dargestellt.

| $x$                         | $A_G(x)$ |
|-----------------------------|----------|
| $\frac{\sqrt{5} − 1}{4}$    | $1$      |
| $\frac{2}{5}$               | $2$      |
| $\frac{\sqrt{22} − 2}{6}$   | $3$      |
| $\frac{\sqrt{137} − 5}{14}$ | $4$      |
| $\frac{1}{2}$               | $5$      |

Wir nennen $A_G(x)$ ein goldenes Nugget, wenn $x$ rational ist, da diese immer seltener werden; zum Beispiel ist das 20. goldene Nugget 211345365. Finden Sie die Summe der ersten dreißig goldenen Nuggets.
