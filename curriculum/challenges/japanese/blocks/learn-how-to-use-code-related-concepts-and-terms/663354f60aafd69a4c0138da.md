# --description--

`equals symbol`（`=`）は代入によく使われます。これは変数に値を代入するために使われます。変数名はイコール記号の左側に書き、変数にストアしたい値は右側に書きます。

例えば、`name = "Sophie";`は文字列`"Sophie"`を変数`name`に代入します。

プログラミングで`assign a value`とは、変数に特定の値を与えることを意味します。`equals symbol`（`=`）はこの目的でよく使われます。変数に値を代入すると、その値を後で使うためにプログラムに記憶させることになります。

例えば、`age = 25`は値`25`を変数`age`に代入します。
