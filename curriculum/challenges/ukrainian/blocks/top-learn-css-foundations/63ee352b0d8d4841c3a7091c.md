# --description--

Що робити, якщо у вас є дві групи елементів, які мають спільні деякі оголошення стилів?

```css
.read {
  color: white;
  background-color: black;
  /* several unique declarations */
}

.unread {
  color: white;
  background-color: black;
  /* several unique declarations */
}
```

Обидва наші селектори `.read` і `.unread` мають спільні оголошення `color: white;` і `background-color: black;`, але водночас мають кілька унікальних оголошень. Щоб зменшити повторення, ви можете об’єднати ці два селектори у список, розділений комами:

```css
.read,
.unread {
  color: white;
  background-color: black;
}

.read {
  /* several unique declarations */
}

.unread {
  /* several unique declarations */
}
```

Обидва наведені приклади (з групуванням і без нього) дадуть однаковий результат, але другий приклад зменшує повторення оголошень і полегшує редагування `color` або `background-color` для обох класів одночасно.
