## قوائم بايثون

- **مقدمة**: في بايثون، نوع البيانات القائمة هو متتالية مرتبة من العناصر التي يمكن أن تتكون من سلاسل نصية، أعداد أو حتى قوائم أخرى. القوائم قابلة للتغيير وفهارسها تبدأ من الصفر.

```python
cities = ['Los Angeles', 'London', 'Tokyo']
```

- **الوصول إلى عناصر في قائمة**: للوصول إلى عنصر من قائمة `cities`، يمكنك الإشارة إلى رقم الفهرس الخاص به في المتتالية:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # Los Angeles
```

- **الوصول إلى عناصر باستخدام الفهرسة السالبة**: للوصول إلى العنصر الأخير في أي قائمة، يمكنك استخدام `-1` كرقم الفهرس:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # Tokyo
```

- تُستخدم الفهرسة السالبة للوصول إلى العناصر بدءًا من نهاية القائمة بدلاً من البداية عند الفهرس `0`.

- **إنشاء قوائم باستخدام دالة البناء `list()`**: يمكن أيضًا إنشاء القوائم باستخدام دالة البناء `list()`. تُستخدم دالة البناء `list()` لتحويل كائن قابل للتكرار إلى قائمة:

```python
developer = 'Jessica'

print(list(developer)) 
# Result: ['J', 'e', 's', 's', 'i', 'c', 'a']
```

- **إيجاد طول القائمة**: يمكنك استخدام الدالة `len()` للحصول على طول القائمة:

```python
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

- **قابلية تغيير القوائم**: القوائم قابلة للتغيير، مما يعني أنه يمكنك تحديث أي عنصر في القائمة طالما أنك تمرر رقم فهرس صالح. لتحديث القوائم عند فهرس معين، يمكنك تعيين قيمة جديدة لذلك الفهرس:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

- **خطأ تجاوز الفهرس**: إذا مررت بفهرس (سواء كان موجبًا أو سالبًا) خارج حدود القائمة، فستتلقى خطأ `IndexError`:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
"""
```

- **إزالة عناصر من القائمة**: يمكن إزالة العناصر من القائمة باستخدام الكلمة المفتاحية `del`:

```python
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

- **التحقق من وجود عنصر في القائمة**: يمكن استخدام الكلمة المفتاحية `in` للتحقق مما إذا كان عنصر موجودًا في القائمة:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **تسكين القوائم**: يمكن تسكين القوائم داخل قوائم أخرى:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

- للوصول إلى القائمة المسكنة، ستحتاج إلى الوصول إليها باستخدام الفهرس `2` لأن الفهارس تبدأ من الصفر.

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

- للوصول إلى اللغة الثانية من تلك القائمة المسكنة، ستحتاج إلى الوصول إليها باستخدام الفهرس `1`:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # Rust
```

- **فك القيم من قائمة:** فك القيم من قائمة هو تقنية تُستخدم لتعيين القيم من القائمة إلى متغيرات جديدة. إليك مثال لفك قائمة `developer` إلى متغيرات جديدة تسمى `name` و `age` و `job` هكذا:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer
```

- إذا لم يتطابق عدد المتغيرات على الجانب الأيسر من معامل التعيين مع العدد الإجمالي للعناصر في القائمة، فستتلقى خطأ `ValueError`.

- **جمع العناصر المتبقية من القائمة**: لجمع أي عناصر متبقية من القائمة، يمكنك استخدام معامل النجمة (`*`) هكذا:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer
```

- **تقطيع القوائم**: التقطيع هو مفهوم الوصول إلى جزء من القائمة باستخدام معامل التقطيع `:`. لتقطيع قائمة تبدأ عند الفهرس `1` وتنتهي قبل الفهرس `3`، يمكنك استخدام الصيغة التالية:

```python
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie']
desserts[1:3] # ['Cookies', 'Ice Cream']
```

- **فواصل الخطوات**: من الممكن أيضًا تحديد فاصل خطوة يحدد مقدار الزيادة بين الفهارس. إليك مثالًا إذا أردت استخراج قائمة من الأعداد الزوجية فقط باستخدام التقطيع:

```python
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

## طرق القوائم

- **append()**: تُستخدم لإضافة عنصر إلى نهاية القائمة. إليك مثالًا لاستخدام طريقة `append()` لإضافة الرقم `6` إلى قائمة `numbers` هذه:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **إضافة قوائم**: يمكن أيضًا استخدام طريقة `append()` لإضافة قائمة واحدة إلى نهاية أخرى:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

- **extend()**: تُستخدم لإضافة عناصر متعددة إلى نهاية القائمة. إليك مثالًا لإضافة الأرقام `6` و `8` و `10` إلى نهاية قائمة `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

- **insert()**: تُستخدم لإدخال عنصر في فهرس محدد داخل القائمة. إليك مثالًا لاستخدام طريقة `insert()`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

- **remove():** تُستخدم لإزالة عنصر من القائمة. ستزيل طريقة `remove()` أول ظهور فقط للعنصر في القائمة:

```py
numbers = [1, 2, 3, 4, 5, 5, 5]
numbers.remove(5)

print(numbers) # [1, 2, 3, 4, 5, 5]

```

- **pop()**: تُستخدم لإزالة عنصر محدد من القائمة وإرجاعه:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

- إذا لم تحدد عنصرًا لطريقة `pop`، فسيتم إزالة العنصر الأخير.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

- **clear()**: تُستخدم لإزالة جميع العناصر من القائمة:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

- **sort()**: تُستخدم طريقة `sort()` لترتيب العناصر في مكانها. إليك مثالًا لترتيب قائمة عشوائية من `numbers` في مكانها:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

- **sorted()**: دالة مدمجة تُرجع قائمة مرتبة جديدة بدلاً من تعديل القائمة الأصلية:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
print(numbers) # [19, 2, 35, 1, 67, 41]
```

- **reverse()**: تُستخدم لعكس ترتيب العناصر في القائمة:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **index()**: تُستخدم لإيجاد أول فهرس يمكن العثور فيه على عنصر في القائمة:

```python
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

- إذا لم يُعثر على العنصر باستخدام طريقة `index()`، فستكون النتيجة `ValueError`.

## الصفوف المرتبة في بايثون

- **التعريف**: الصف المرتب هو نوع بيانات في بايثون يُستخدم لإنشاء متتالية مرتبة من القيم. يمكن أن تحتوي الصفوف المرتبة على مجموعة مختلطة من أنواع البيانات:

```py
developer = ('Alice', 34, 'Rust Developer')
```

- الصفوف المرتبة غير قابلة للتغيير، مما يعني أنه لا يمكن تعديل العناصر فيها بعد إنشائها. إذا حاولت تحديث أحد العناصر في الصف المرتب، ستحصل على خطأ `TypeError`:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **الوصول إلى عناصر من صف مرتب**: للوصول إلى عنصر من صف مرتب، استخدم ترميز الأقواس ورقم الفهرس:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

- يمكن استخدام الفهرسة السالبة للوصول إلى العناصر بدءًا من نهاية الصف المرتب:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- إذا حاولت تمرير رقم فهرس يتجاوز أو يساوي طول الصف المرتب، فستتلقى خطأ `IndexError`:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- يمكن أيضًا إنشاء صف مرتب باستخدام دالة البناء `tuple()`. داخل دالة البناء، يمكنك تمرير كائنات قابلة للتكرار مختلفة مثل السلاسل النصية والقوائم وحتى صفوف مرتبة أخرى.

```python
developer = 'Jessica'

print(tuple(developer)) 
# Result: ('J', 'e', 's', 's', 'i', 'c', 'a')
```

- **التحقق من وجود عناصر في صف مرتب**: للتحقق مما إذا كان عنصر موجودًا في صف مرتب، يمكنك استخدام الكلمة المفتاحية `in` هكذا:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **فك الصفوف المرتبة**: يمكن فك العناصر من صف مرتب هكذا:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

- إذا كنت بحاجة إلى جمع أي عناصر متبقية من صف مرتب، يمكنك استخدام معامل النجمة (`*`) هكذا:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer
```

- **تقطيع الصفوف المرتبة**: يمكن استخدام التقطيع لاستخراج جزء من صف مرتب. على سبيل المثال، يمكن تقطيع العناصر `pie` و `cookies` إلى صف مرتب منفصل:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

- **إزالة عناصر من الصفوف المرتبة**: إزالة عنصر من صف مرتب ستؤدي إلى رفع خطأ `TypeError` لأن الصفوف المرتبة غير قابلة للتغيير:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

- **متى تستخدم صف مرتب مقابل قائمة؟**: إذا كنت تحتاج إلى مجموعة ديناميكية من العناصر حيث يمكنك الإضافة والإزالة والتحديث، فعليك استخدام قائمة. إذا كنت تعرف أنك تعمل مع مجموعة ثابتة وغير قابلة للتغيير من البيانات، فعليك استخدام صف مرتب.

## طرق الصفوف المرتبة الشائعة

- **`count()`**: تُستخدم لتحديد عدد مرات ظهور عنصر في صف مرتب. على سبيل المثال، يمكنك التحقق من عدد مرات ظهور اللغة `'Rust'` في الصف المرتب:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

- إذا لم يكن العنصر المحدد في طريقة `count()` موجودًا على الإطلاق في الصف المرتب، فستكون قيمة الإرجاع `0`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

- إذا لم يتم تمرير أي معلمات إلى طريقة `count()`، فستُرجع بايثون خطأ `TypeError`.

- **index()**: تُستخدم لإيجاد الفهرس الذي يوجد فيه عنصر معين في الصف المرتب. إليك مثالًا لاستخدام طريقة `index()` لإيجاد فهرس اللغة `'Java'`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

- إذا لم يُعثر على العنصر المحدد، فستُرجع بايثون خطأ `ValueError`.

- يمكنك تمرير فهرس بداية اختياري إلى طريقة `index()` لتحديد مكان بدء البحث عن العنصر في الصف المرتب:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

- يمكنك أيضًا تمرير فهرس نهاية اختياري إلى طريقة `index()` لتحديد مكان إيقاف البحث عن العنصر في الصف المرتب:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

- **`sorted()`**: تُستخدم لترتيب العناصر في أي كائن قابل للتكرار وإرجاع قائمة مرتبة جديدة. إليك مثالًا لإنشاء قائمة جديدة من الأرقام باستخدام دالة `sorted()`:

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

- **تعديل سلوك الترتيب**: يمكنك تخصيص سلوك الترتيب لكائن قابل للتكرار باستخدام المعاملات الاختيارية `reverse` و `key`. إليك مثالًا لاستخدام معامل `key` لترتيب عناصر في صف مرتب حسب الطول:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

- يمكنك إنشاء قائمة جديدة من القيم بترتيب عكسي باستخدام معامل `reverse` هكذا:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

## الحلقات في بايثون

- **التعريف**: تُستخدم الحلقات لتكرار كتلة من الشفرة لعدد محدد من المرات.

- **حلقة `for`**: تُستخدم للتكرار على متتالية (مثل قائمة، صف مرتب أو سلسلة نصية) وتنفيذ كتلة من الشفرة لكل عنصر في تلك المتتالية. إليك مثالًا لاستخدام حلقة `for` للتكرار خلال قائمة وطباعة كل لغة إلى وحدة التحكم:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)

"""
Result 

Rust
Java
Python
C++
"""
```

- إليك مثالًا لاستخدام حلقة `for` للتكرار خلال السلسلة النصية `code` وطباعة كل حرف:

```py
for char in 'code':
    print(char)

"""
Result 

c
o
d
e
"""
```

- يمكن تداخل حلقات `for`. إليك مثالًا لاستخدام حلقة `for` مسكنة:

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)

"""
Result

Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
"""

```

- **حلقة `while`**: تكرر كتلة من الشفرة حتى يصبح الشرط `False`. إليك مثالًا لاستخدام حلقة `while` للعبة تخمين:

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')

"""
Result

Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
"""

```

- **عبارات `break` و `continue`**: تُستخدم في الحلقات لتعديل تنفيذ الحلقة.

- تُستخدم عبارة `break` للخروج من الحلقة فورًا عند تحقق شرط معين. إليك مثالًا لاستخدام عبارة `break` لقائمة `developer_names`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

- تُستخدم عبارة `continue` لتخطي التكرار الحالي والانتقال إلى التكرار التالي من الحلقة. إليك مثالًا لاستخدام عبارة `continue` بدلًا من `break`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

- يمكن دمج كل من حلقات `for` و `while` مع بند `else`، الذي يُنفذ فقط عندما لا تُنهى الحلقة بواسطة `break`:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

## الدوال النطاقية واستخدامها في الحلقات

- **دالة `range()`**: تُستخدم لتوليد متتالية من الأعداد الصحيحة.

```py
range(start, stop, step)
```

- المعلمة المطلوبة `stop` هي عدد صحيح (غير شامل) يمثل نقطة النهاية لمتتالية الأعداد التي يتم توليدها. إليك مثالًا لاستخدام دالة `range()`:

```py
for num in range(3):
    print(num)
```

- إذا لم يتم تحديد معلمة `start`، فسيكون الافتراضي `0`. بشكل افتراضي، ستزداد متتالية الأعداد الصحيحة بمقدار `1`. يمكنك استخدام المعلمة الاختيارية `step` لتغيير قيمة الزيادة الافتراضية. إليك مثالًا لتوليد متتالية من الأعداد الزوجية من 2 حتى قبل 11 (أي تشمل 10)

```py
for num in range(2, 11, 2):
    print(num)
```

- إذا لم تقدم أي معلمات إلى دالة `range()`، فستحصل على خطأ `TypeError`.

- تقبل دالة `range()` الأعداد الصحيحة فقط كمعلمات وليس الأعداد العشرية. استخدام الأعداد العشرية سيؤدي أيضًا إلى خطأ `TypeError`:

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- يمكنك استخدام عدد صحيح سالب للمعلمة `step` لتوليد متتالية من الأعداد الصحيحة بترتيب تنازلي:

```py
for num in range(40, 0, -10):
    print(num)
```

- يمكن أيضًا استخدام دالة `range()` لإنشاء قائمة من الأعداد الصحيحة باستخدامها مع دالة البناء `list`. تُستخدم دالة البناء `list` لتحويل كائن قابل للتكرار إلى قائمة. إليك مثالًا لتوليد قائمة من الأعداد الزوجية بين 2 و 10 شاملة:

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

## دالتي `enumerate()` و `zip()` في بايثون

- **`enumerate()`**: تُستخدم للتكرار على متتالية وتتبع الفهرس لكل عنصر في تلك المتتالية. تأخذ دالة `enumerate()` كائنًا قابلًا للتكرار كمعلمة وتُرجع كائن `enumerate` يتكون من الفهرس والقيمة لكل عنصر في الكائن القابل للتكرار.

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')

# Result
# Index 0 and language Spanish
# Index 1 and language English
# Index 2 and language Russian
# Index 3 and language Chinese

```

- يمكن أيضًا استخدام دالة `enumerate()` خارج حلقة `for`:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

print(list(enumerate(languages)))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

- تقبل دالة `enumerate()` أيضًا معلمة اختيارية `start` تحدد القيمة الابتدائية للعد. إذا تم حذف هذه المعلمة، يبدأ العد من `0`.

-  **`zip()`** : تُستخدم للتكرار على عدة كائنات قابلة للتكرار بالتوازي. إليك مثالًا لاستخدام دالة `zip()` للتكرار على `developers` و `ids`:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')


"""
Result

Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
"""

```

## تعبيرات القوائم في بايثون

- **التعريف**: تعبير القائمة يسمح لك بإنشاء قائمة جديدة في سطر واحد بدمج الحلقة والشرط مباشرة داخل أقواس مربعة. هذا يجعل الشفرة أقصر وغالبًا أسهل للقراءة.

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

## طرق الكائنات القابلة للتكرار

- **`filter()`**: تُستخدم لتصفية العناصر من كائن قابل للتكرار بناءً على شرط. تُرجع مُكررًا يحتوي فقط على العناصر التي تحقق الشرط. إليك مثالًا لإنشاء قائمة جديدة من الكلمات التي تزيد عن أربعة أحرف:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**: تُستخدم لتطبيق دالة على كل عنصر في كائن قابل للتكرار وإرجاع كائن قابل للتكرار جديد بالنتائج. إليك مثالًا لاستخدام دالة `map()` لتحويل قائمة درجات حرارة بالسلسيوس إلى فهرنهايت:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**: تُستخدم للحصول على مجموع من كائن قابل للتكرار مثل قائمة أو صف مرتب. إليك مثالًا لاستخدام دالة `sum()`:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- يمكنك أيضًا تمرير معلمة اختيارية `start` التي تحدد القيمة الابتدائية للجمع. إليك مثالًا محدثًا باستخدام معلمة `start` كمعلمة موضعية:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- يمكنك أيضًا اختيار استخدام معلمة `start` كمعلمة مسماة هكذا بدلاً من ذلك:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## دوال لامبدا

- **التعريف**: دالة لامبدا في بايثون هي طريقة مختصرة لإنشاء دالة بدون اسم (دالة مجهولة). 
- غالبًا ما تُستخدم دوال لامبدا كمعلمة لدالة أخرى. إليك مثالًا على دالة لامبدا:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

- أفضل الممارسات لاستخدام دوال لامبدا تشمل عدم تعيينها إلى متغير، والحفاظ عليها بسيطة وقابلة للقراءة، واستخدامها للدوال القصيرة والمؤقتة.
