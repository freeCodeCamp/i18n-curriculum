# --description--

设 $f(n)$ 为满足 $x$ 和 $y$ 为正整数，$x ≤ y$ 且 $x$ 和 $y$ 的最小公倍数等于 $n$ 的有序对 ($x$, $y$) 的数量。

设 $g$ 为 $f$ 的求和函数，即：$g(n) = \sum f(i)$，其中 $1 ≤ i ≤ n$。

已知 $g({10}^6) = 37\\,429\\,395$。

查找 $g({10}^{12})$。
