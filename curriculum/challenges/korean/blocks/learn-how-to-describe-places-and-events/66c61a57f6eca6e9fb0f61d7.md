무언가가 `built-in`라면 이미 존재하며 바로 사용할 수 있다는 뜻입니다. 예를 들어:

- `This laptop has a built-in camera.` - 카메라가 노트북에 포함되어 있습니다.

- `The software comes with built-in security features.` - 보안 기능이 소프트웨어에 이미 포함되어 있습니다.
