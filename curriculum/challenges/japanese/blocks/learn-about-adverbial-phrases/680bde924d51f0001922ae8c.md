# --explanation--

`To be based on something` は、あるアイデアや計画、決定がそれを基盤や出発点としていることを意味します。最終的な選択に影響を与えたり形作ったりするものを示します。例えば：

- `Their new pricing model is based on customer feedback.` – これは、顧客の意見や提案を使って価格を決めたことを意味します。

- `The movie is based on a true story.` – これは、実際の話が映画に描かれた出来事のインスピレーションになったことを意味します。
