# --description--

Erweiterte Zuweisung kombiniert eine binäre Operation mit einer Zuweisung in einem Schritt. Dabei wird einer Variablen eine Operation mit einem anderen Wert angewendet, und das Ergebnis wird wieder in dieselbe Variable gespeichert.

Wenn Sie mit einer Sprache wie JavaScript vertraut sind, haben Sie wahrscheinlich schon vom Additionszuweisungsoperator (`+=`) oder der Subtraktionszuweisung (`-=`) und anderen gehört. Diese gibt es auch in Python. Der einzige Unterschied ist, dass sie als **erweiterte Zuweisungen** bezeichnet werden.

Die grundlegende Syntax einer erweiterten Zuweisung sieht so aus:

```python
variable <operator>= value
```

Das ist eine effizientere Art, Folgendes zu tun:

```python
variable = variable <operator> value
```

Zum Beispiel hier ein Beispiel, wie man mit erweiterter Zuweisung `5` zu einer bestehenden Variable addiert:

```python
my_var = 10
my_var += 5

print(my_var) # 15
```

Und hier dasselbe ohne erweiterte Zuweisung:

```python
my_var = 10
my_var = my_var + 5

print(my_var) # 15
```

Der Vorteil der erweiterten Zuweisung ist, dass sie eine prägnante und gut lesbare Möglichkeit bietet, den Wert einer Variable zu aktualisieren, ohne den Variablennamen zu wiederholen. Dadurch wird Redundanz reduziert und potenzielle Fehler, die durch Tippfehler oder Ähnliches entstehen könnten, vermieden.

Jeder Operator kann mit einer erweiterten Zuweisung verwendet werden. Wir haben uns den Additionszuweisungsoperator (`+=`) angesehen, also schauen wir uns andere an.

- Der Subtraktionszuweisungsoperator (`-=`) subtrahiert den rechten Operanden von der linken Variable und speichert die Differenz in der linken Variable:

```python
count = 14
count -= 3

print(count) # 11
```

- Der Multiplikationszuweisungsoperator (`*=`) multipliziert die linke Variable mit dem rechten Operanden und speichert das Produkt wieder in der linken Variable:
    

```python
product = 65
product *= 7

print(product) # 455
```

- Der Divisionszuweisungsoperator (`/=`) dividiert die linke Variable durch den rechten und speichert das Ergebnis wieder in der linken Variable:
    

```python
price = 100
price /= 4

print(price) # 25.0
```

- Der Floor-Divisionsoperator (`//=`) führt eine ganzzahlige Division der linken Variable durch den rechten Operanden durch und speichert das Ergebnis wieder in der linken Variable:
    

```python
total_pages = 23
total_pages //= 5

print(total_pages) # 4
```

- Der Modulo-Zuweisungsoperator (`%=`) berechnet den Rest der Division der linken Variable durch den rechten und speichert ihn wieder in der linken Variable:
    

```python
bits = 35
bits %= 2

print(bits) # 1
```

- Der Exponentiationszuweisungsoperator (`**=`) hebt die linke Variable auf die Potenz des rechten Operanden und speichert das Ergebnis wieder in der linken Variable:
    

```python
power = 2
power **= 3

print(power) # 8
```

Einige erweiterte Zuweisungsoperatoren können Sie auch mit Strings verwenden. Zum Beispiel macht der Additionszuweisungsoperator das Verketten von Strings einfach:

```python
greet = 'Hello'
greet += ' World'

print(greet) # Hello World
```

Und der Multiplikationszuweisungsoperator kann verwendet werden, um einen String zu wiederholen:

```python
greet = 'Hello'
greet *= 3

print(greet) # HelloHelloHello
```

Andere erweiterte Zuweisungen werfen einen `TypeError`, wenn Sie sie mit Strings verwenden:

```python
greet = 'Hello'
greet -= ' World'

print(greet) # TypeError: unsupported operand type(s) for -=: 'str' and 'str'


greet = 'Hello'
greet /= 'World'

print(greet) # TypeError: unsupported operand type(s) for /=: 'str' and 'str' 
```

Falls Sie sich fragen, ob Inkrement- und Dekrementoperatoren (`++` und `--`) in Python funktionieren: Das tun sie nicht. Python vermeidet bewusst C-ähnliche Inkrement- und Dekrement-Abkürzungen, um die Sprache klar und explizit zu halten.

Anstelle von `x++` können Sie einfach `x += 1` schreiben, was deutlich macht, dass Sie den Wert von `x` um `1` erhöhen.

`++x` in Python zu schreiben, wendet einfach das unäre Plus zweimal an und erhöht nichts:

```python
my_var = 5

print(+my_var)   # 5
print(++my_var)  # 5
print(+++my_var) # 5

my_var += 1

print(my_var) # 6
```
