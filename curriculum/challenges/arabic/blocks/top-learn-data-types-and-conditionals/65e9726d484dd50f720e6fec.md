# --description--

تستطيع JavaScript أيضًا مقارنة الأنواع والقيم باستخدام معامل المساواة الصارمة `===`. هذا المعامل يتحقق مما إذا كانت القيمتان متساويتين ومن نفس النوع. على سبيل المثال:

```javascript
let x = 5;
let y = "5";

let result = x === y;

console.log(result); // false
```

في هذا المثال، ستحتوي المتغير `result` على القيمة `false` لأن `x` هو رقم و`y` هي سلسلة نصية. لكن إذا استخدمت المعامل `==`، ستكون النتيجة `true` لأن JavaScript ستحول السلسلة النصية إلى رقم وتقارن القيم.

يسمى هذا المعامل معامل المساواة الصارمة لأنه يتحقق من تساوي القيمة والنوع معًا. يُنصح غالبًا باستخدام المعامل `===` لتجنب النتائج غير المتوقعة عند مقارنة القيم.

يوجد أيضًا معامل عدم المساواة الصارم `!==` الذي يتحقق مما إذا كانت القيمتان غير متساويتين ومن نفس النوع.

هناك العديد من الطرق لاستخدام معامل المساواة الصارمة، ومن المهم فهم كيفية عمله.
