# --explanation--

**分数** 用于描述整体的部分。

`One-fourth`（¼）是四个相等部分中的一部分。其他例子：

- ½ = `one-half`

- ⅓ = `one-third`

- ¾ = `three-fourths`

在 Alice 的分支中，四分之一的开发预算用于后端开发，这显示了资金在一个类别内的分配情况。
