# --interactive--

JavaScript는 덧셈, 뺄셈, 곱셈, 나눗셈과 같은 기본 산술 연산을 수행할 수 있는 도구를 제공합니다. 또한 나머지와 거듭제곱 같은 더 복잡한 산술 연산을 위한 연산자도 포함되어 있습니다.

이 모든 도구를 산술 연산자라고 합니다. 이 연산자들을 자세히 살펴보고, 어떻게 사용하는지 그리고 어떻게 조합하는지 알아보겠습니다.

덧셈 연산자는 더하기 기호(`+`)입니다. 덧셈 연산자는 두 개 이상의 숫자의 합계를 구할 수 있게 해줍니다. 덧셈 연산에서는 숫자의 순서가 중요하지 않습니다:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 + num2;
const result2 = num2 + num1;
const result3 = num2 + num1 + num3;

console.log(result1); // 15
console.log(result2); // 15
console.log(result3); // 30
```

:::

뺄셈 연산자는 빼기 기호(`-`)입니다. 두 숫자 사이의 차이를 구할 수 있습니다. 보통 더 큰 수에서 더 작은 수를 빼고 싶을 때 빼기 기호를 사용하세요:

:::interactive_editor

```js
const difference = 10 - 5;
console.log(difference); // 5
```

:::

만약 더 작은 수가 먼저 오면, 결과는 음수가 됩니다:

:::interactive_editor

```js
const difference = 5 - 10;
console.log(difference); // -5
```

:::

숫자를 변수에 할당하고 변수 이름으로 뺄셈을 할 수도 있습니다:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const result = num1 - num2;

console.log(result); // 5
```

:::

JavaScript에서 곱셈 연산자는 별표(`*`)로 나타내며, 두 개 이상의 숫자의 곱을 구하는 데 사용합니다. 곱하는 숫자의 순서는 중요하지 않습니다:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 * num2;
const result2 = num2 * num1;
const result3 = num2 * num1 * num3;

console.log(result1); // 50
console.log(result2); // 50
console.log(result3); // 750
```

:::

JavaScript에서 나눗셈 연산자는 슬래시(`/`)이며, 전통적인 수학에서 사용하는 나눗셈 기호(`÷`)와 다릅니다. 나눗셈 연산을 수행할 때는 숫자의 순서가 중요합니다:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 / num2;
const result2 = num2 / num1;
const result3 = num2 / num1 / num3;

console.log(result1); // 2
console.log(result2); // 0.5
console.log(result3); // 0.03333333333333333
```

:::

0으로 나누기를 시도하면 JavaScript가 `Infinity`를 반환한다는 점을 기억하세요:

:::interactive_editor

```js
const result = 10 / 0; 

console.log(result); // Infinity
```

:::

예상치 못한 결과가 나오지 않도록 이런 계산은 피하세요.

나머지 연산자는 퍼센트 기호(`%`)로 나타내며, 나눗셈 후 남은 나머지를 반환합니다. 수학에서 나머지는 나눗셈을 수행한 후 남는 값입니다:

:::interactive_editor

```js
const num1 = 10;
const num2 = 3;
const remainder = num1 % num2;

console.log(remainder); // 1
```

:::

거듭제곱 연산자는 별표 두 개(`**`)로 나타내며, 한 숫자를 다른 숫자의 거듭제곱으로 올립니다:

:::interactive_editor

```js
const num1 = 2;
const num2 = 3;

const exponent = num1 ** num2;
console.log(exponent); // 8
```

:::

하나의 연산이나 표현식에서 여러 연산자를 섞어 사용할 수도 있습니다:

:::interactive_editor

```js
const result = 10 + 5 * 2 - 8 / 4;
console.log(result); // 18
```

:::

여러 연산자를 섞어 사용할 때 JavaScript 엔진은 연산자 우선순위라는 시스템을 따라 연산 순서를 결정합니다. 연산자 우선순위는 표현식에서 연산이 실행되는 순서를 결정합니다. 연산자 우선순위에 대해서는 앞으로 수업에서 더 자세히 배우게 됩니다.
