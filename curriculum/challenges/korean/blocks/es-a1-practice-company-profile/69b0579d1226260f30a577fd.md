# --explanation--

수량 질문에 답할 때는 다음과 같은 주요 숫자 패턴을 들으세요:

- 70대 숫자: `setenta y...`
- 30대 숫자: `treinta y...`
- 10대 숫자: `dieci...`

비슷하게 들리는 숫자를 구별할 수 있어야 이해가 잘 됩니다.

`¿Cuántos...?`는 **수량**에 대해 묻는 데 사용됩니다. 예를 들어:

`¿Cuántos empleados...?` - 직원이 몇 명인가요?

동사 `tiene`는 `tener`를 가리키므로 `la empresa` 동사의 3인칭 형태입니다. 예를 들어:

`La empresa tiene...` - 회사는 (3인칭 단수) 있습니다.
