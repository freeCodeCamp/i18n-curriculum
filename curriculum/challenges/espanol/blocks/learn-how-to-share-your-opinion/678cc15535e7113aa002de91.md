## --text--

¿Qué oración usa `I'd rather` correctamente?

## --answers--

`I'd rather use Python for data analysis than JavaScript.`

---

`I'd rather to use Python for data analysis than JavaScript.`

### --feedback--

`Rather` va seguido de la forma base del verbo, no de `to`.

---

`I'd rather using Python for data analysis than JavaScript.`

### --feedback--

`Rather` va seguido de la forma base del verbo, no de `using`.

---

`I'd rather use Python to data analysis than JavaScript.`

### --feedback--

La frase `to data analysis` debería ser `for data analysis`.

# --explanation--

`I'd rather` es una frase usada para expresar preferencia entre dos acciones u opciones. Va seguida de la forma base del verbo.
