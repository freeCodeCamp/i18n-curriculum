# --description--

`By the end of it` は、何かが終わろうとしているか、すでに終わっていることを意味します。

`A clearer picture` は比喩的に、より良いまたはより完全な理解を意味します。これは、メガネをきれいにして突然物事がはっきり見えるようになるのと似ています。

`a clearer picture` の例：`After studying, I have a clearer picture of the math problem.`
