# --description--

Kutolewa kwa mteremko wa mlima wa kuteleza, umbali wa usawa, na aina yake, amua kiwango cha ugumu wa mlima huo.

Ili kuamua kiwango:

- Hesabu unene wa mteremko kwa kugawanya mteremko na umbali.
- Kisha, hesabu unene uliorekebishwa kulingana na aina ya mlima:
  - `"Downhill"`: zidisha unene kwa 1.2
  - `"Slalom"`: zidisha unene kwa 0.9
  - `"Giant Slalom"`: zidisha unene kwa 1.0

Return:

- `"Green"` ikiwa unene uliorekebishwa ni mdogo au sawa na 0.1
- `"Blue"` ikiwa unene uliorekebishwa ni mkubwa kuliko 0.1 na mdogo au sawa na 0.25
- `"Black"` ikiwa unene uliorekebishwa ni mkubwa kuliko 0.25

```py
def get_hill_rating(drop, distance, hill_type):
    steepness = drop / distance

    if hill_type == "Downhill":
        steepness *= 1.2
    elif hill_type == "Slalom":
        steepness *= 0.9
    elif hill_type == "Giant Slalom":
        steepness *= 1.0

    if steepness <= 0.1:
        return "Green"
    elif steepness <= 0.25:
        return "Blue"
    else:
        return "Black"
```
