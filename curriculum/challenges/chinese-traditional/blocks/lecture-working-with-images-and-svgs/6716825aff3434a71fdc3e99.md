# --interactive--

首先，你需要了解影像如何運作。常見的影像格式如 PNG 和 JPG 被歸類為點陣格式。這基本上表示它們是基於像素的，資料會追蹤每個像素中的顏色值。

基於點陣的影像一個很大的缺點是它們無法良好放大。如果你曾經嘗試將 PNG 放大，你可能會看到它變得像素化或模糊。

SVG 是一種不同類型的影像。SVG 代表可縮放向量圖形。向量圖形根據路徑和方程式追蹤資料，以繪製點、線和曲線。這真正的意思是，像 SVG 這樣的向量圖形，可以縮放到任何大小而不影響品質。

SVG 特別具有將資料儲存在 XML 中的額外利益。這表示你可以直接在你的程式碼中以未經處理的 HTML 使用 `svg` 元素。這也表示你可以以程式方式變更影像的顏色。

若要將笑臉改為紅色，請啟用互動式編輯器，並將 `fill="yellow"` 改為 `fill="red"`。

:::interactive_editor

```html
<svg width="100" height="100" viewBox="0 0 100 100" xmlns="http://www.w3.org/2000/svg">
  <circle cx="50" cy="50" r="45" stroke="black" stroke-width="4" fill="yellow" />
  <circle cx="35" cy="40" r="5" fill="black" />
  <circle cx="65" cy="40" r="5" fill="black" />
  <path d="M35 65 Q50 80 65 65" stroke="black" stroke-width="4" fill="transparent" />
</svg>
```

:::

以下是範例的基本元素：

- `svg` 元素是整個繪圖的容器。它設定所有形狀出現的空間。你想用 SVG 繪製的所有東西，例如圓形、線條或路徑，都放在 `svg` 元素內。
- `circle` 元素用於製作臉部和眼睛。一個大圓形構成黃色臉部，兩個較小的圓形構成眼睛。
- `path` 元素用於繪製笑臉。它創建一條嘴巴的曲線。
- 每個 SVG 元素都有控制其外觀和在繪圖區域中位置的屬性。

這裡有更多範例。若要更改任何範例的顏色，請將 `fill` 屬性的值更新為任何命名顏色，例如 `red`、`green`、`blue`、`yellow` 等。 

:::interactive_editor

那麼你什麼時候會想使用 SVG 呢？一個很好的使用案例是圖示。如果你想創建訂製的項目符號，或為你的聯結添加圖示以表述社群媒體平台，使用 SVG 是最佳方法。最受歡迎的圖示庫之一 Font Awesome 就是使用 SVG 影像作為他們的圖示。SVG 也非常適合用於網頁標誌，因為它們可以完美縮放。它們讓你能夠將版面配置調整為任何你需要的響應式設計。下次你有區域的 SVG 時，試著用文字編輯器打開並玩弄裡面的程式碼。
