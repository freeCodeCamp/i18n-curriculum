## ミドルウェアとは何ですか？

- **ミドルウェア** は、サーバーへのリクエストのライフサイクル中に実行される関数です。
- ミドルウェアは **リクエストオブジェクト** (`req`)、**レスポンスオブジェクト** (`res`)、および **`next` 関数** にアクセスできます。
- 一般的な用途：リクエストのログ記録、エラー処理、受信データのパース、静的ファイルの提供などです。
- ミドルウェアはアプリに追加された順に実行されます。

### ミドルウェアの動作

1. サーバーにリクエストが送信されます。
2. ミドルウェア関数がリクエストを処理します。
3. ミドルウェアが `next()` を呼び出すと、制御は次のミドルウェアまたはルートハンドラに渡されます。
4. ミドルウェアが `next()` を呼び出さない場合、リクエスト-レスポンスのサイクルはストップします。

```javascript
app.use((req, res, next) => {
  console.log('Request URL:', req.url);
  console.log('Request Method:', req.method);
  next();
});
```

- `app.use()` はミドルウェアをアプリケーション全体にグローバルに追加します。

## アプリケーションレベルのミドルウェア

- **アプリケーションレベルのミドルウェア** は Express アプリのインスタンスにバインドされます。
- `app.use()` や `app.get()`、`app.post()` などのルート固有のメソッドを使って登録します。
- 特定のルートに制限しない限り、すべての受信リクエストに対して実行されます。
- 使用例：ログ記録、認証、複数ルートにまたがるエラー処理など。

```javascript
app.use((req, res, next) => {
  console.log('Request received at:', new Date());
  next();
});
```

## ルーター レベルのミドルウェア

- **ルーター レベルのミドルウェア** はメインアプリではなく `express.Router()` のインスタンスにバインドされます。
- `router.use()` を使って適用し、そのルーターが処理するルートにのみ影響します。
- アプリを分割して独立したルートモジュールにするのに役立ちます。
- `app.use('/path', router)` でルーターをマウントして、ミドルウェアを特定のルートにスコープできます。

```javascript
const router = express.Router();

router.use((req, res, next) => {
  console.log('Request made to /menu route');
  next();
});

router.get('/drinks', (req, res) => {
  res.send('Welcome to the drinks menu!');
});

app.use('/menu', router);
```

## エラー処理ミドルウェア

- **エラー処理ミドルウェア** はリクエスト-レスポンスサイクル中のエラーをキャッチするための特別なミドルウェアです。
- **4つのパラメータ** を持ちます：`err`、`req`、`res`、`next`。この4パラメータのシグネチャが Express にエラーハンドラであることを示します。
- エラーが `next(err)` に渡されたときに呼び出されます。
- 他のすべてのミドルウェアやルートの後、**ミドルウェアスタックの最後に配置する必要があります**。

```javascript
app.use((err, req, res, next) => {
  console.error(err.stack);
  res.status(500).send('Something went wrong!');
});
```

## 組み込みミドルウェアとサードパーティミドルウェア

### 組み込みミドルウェア

- **`express.json()`**：JSON ペイロードを含む受信リクエストをパースします。

```javascript
app.use(express.json());
```

- **`express.static()`**：画像、CSS、JavaScript などの静的ファイルを提供します。

```javascript
app.use(express.static('public'));
```

- **`express.urlencoded()`**：URLエンコードされたペイロード（HTMLフォームから送信される形式）を含む受信リクエストをパースし、`req.body` でパース済みデータを利用可能にします。

```javascript
app.use(express.urlencoded({ extended: true }));
```

### サードパーティミドルウェア

- **`cors`**：クロスオリジンリソースシェアリングを有効にします。

```javascript
app.use(cors());
```

- **`morgan`**：HTTPリクエストをログに記録します。

```javascript
app.use(morgan('tiny'));
```

- 組み込みミドルウェアとサードパーティミドルウェアの両方は、`app.use()` で呼び出された順に追加されます。
