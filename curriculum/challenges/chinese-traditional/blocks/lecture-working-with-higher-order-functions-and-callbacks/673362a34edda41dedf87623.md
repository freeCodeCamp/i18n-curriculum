# --interactive--

`map` 方法是 JavaScript 中一個強大且廣泛使用的函式（程式），用於操作陣列。它的設計目的是透過將給定的函式應用到原始陣列的每個元素，來創建一個新的陣列。

此方法不會修改原始陣列，而是傳回一個新陣列，該陣列包含對每個元素套用函式後的結果。

以下是對數字陣列使用 `map` 方法的範例：

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map((num) => num * 2);
console.log(numbers); // [1, 2, 3, 4, 5]
console.log(doubled); // [2, 4, 6, 8, 10]
```

:::

為了創建一個新的陣列，其中每個數字都被加倍，我們正在使用 `map` 方法。`map` 方法接受一個回呼函式，該函式會對陣列中的每個元素進行呼叫。

在這種情況下，陣列中的每個數字都會乘以 `2`。結果將會是一個新的陣列，包含數字 `2,4,6,8,10`。

回呼函式（程式）最多可以接受三個引數。

第一個引數是正在處理的當前元素。

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element) => {
  console.log("Element:", element);
  return element * 2;
});
```

:::

第二個引數是正在處理的當前元素的索引。

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index) => {
  console.log("Element:", element);
  console.log("Index:", index);
  return element * 2;
});
```

:::

第三個引數是 `map` 被呼叫的陣列。

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index, array) => {
  console.log("Element:", element);
  console.log("Index:", index);
  console.log("Array:", array);
  return element * 2;
});
```

:::

理解並有效使用 `map` 方法可以大幅提升你操作 JavaScript 陣列的能力。在未來的課程中，我們將深入探討 `map` 的更多進階用法，並探索它如何成為建造動態且高效程式的強大工具。
