# --instructions--

Ви проаналізуєте набір даних про глобальну зміну середнього рівня моря з 1880 року. Ви використаєте ці дані, щоб передбачити зміну рівня моря до 2050 року.

Використайте дані для виконання таких завдань:

- Використайте Pandas, щоб імпортувати дані з `epa-sea-level.csv`.
- Використайте matplotlib, щоб створити точковий графік, використовуючи стовпець `Year` як вісь x і стовпець `CSIRO Adjusted Sea Level` як вісь y.
- Використайте функцію `linregress` з `scipy.stats`, щоб отримати нахил і перетин з віссю y для лінії найкращого наближення. Накладіть цю лінію поверх точкового графіка. Зробіть так, щоб лінія проходила через 2050 рік, щоб передбачити підвищення рівня моря у 2050 році.
- Побудуйте нову лінію найкращого наближення, використовуючи лише дані з 2000 року до найновішого року в наборі даних. Також зробіть так, щоб ця лінія проходила через 2050 рік, щоб передбачити підвищення рівня моря у 2050 році, якщо темп підвищення збережеться з 2000 року.
- Підпис осі x повинен бути `Year`, підпис осі y — `Sea Level (inches)`, а заголовок — `Rise in Sea Level`.

Шаблонний код також містить команди для збереження та повернення зображення.
