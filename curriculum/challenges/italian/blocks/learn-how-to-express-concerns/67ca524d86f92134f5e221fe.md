# --explanation--

`How` si usa per chiedere il modo o il metodo per fare qualcosa. Quando `how` segue un'altra frase interrogativa (come, in questo caso, `Did they say`), il resto della frase segue una struttura affermativa (`we could fix it`) invece dell'ordine interrogativo. Per esempio:

- `Did she explain how we can improve communication?` - Questo è corretto; ciò che segue `how` segue una struttura affermativa. Sarebbe scorretto usare `Did she explain how can we improve communication?` qui.

- `Can you show me how I should submit the report?` - Ciò che segue `how` (`I should submit the report`) segue una struttura affermativa (`I should` invece di `should I`).
