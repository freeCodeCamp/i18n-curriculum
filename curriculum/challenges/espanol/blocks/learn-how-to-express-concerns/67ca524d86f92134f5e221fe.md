# --explanation--

`How` se usa para preguntar sobre la manera o método de hacer algo. Cuando `how` sigue a otra frase interrogativa (como, en este caso, `Did they say`), el resto de la oración sigue una estructura afirmativa (`we could fix it`) en lugar del orden interrogativo. Por ejemplo:

- `Did she explain how we can improve communication?` - Esto es correcto; lo que viene después de `how` sigue una estructura afirmativa. Sería incorrecto usar `Did she explain how can we improve communication?` aquí.

- ¿Puedes mostrarme cómo debo enviar el informe? - Lo que viene después de `how` (`I should submit the report`) sigue una estructura afirmativa (`I should` en lugar de `should I`).
