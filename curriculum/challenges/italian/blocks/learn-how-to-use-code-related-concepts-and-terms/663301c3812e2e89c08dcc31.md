# --description--

L’espressione `thanks for` si usa per mostrare gratitudine per azioni specifiche o aiuti ricevuti. La struttura è comunemente seguita da una forma `-ing` di un verbo (un gerundio) o da un sostantivo per specificare il motivo del ringraziamento.

Esempi:

`Thanks for helping me with my homework.`

`Thanks for the advice.`
