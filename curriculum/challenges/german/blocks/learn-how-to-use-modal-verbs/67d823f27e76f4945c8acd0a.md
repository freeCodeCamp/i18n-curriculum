# --explanation--

`Should` ist ein Modalverb, das Rat, Empfehlung oder den besten Handlungsweg ausdrückt – aber keine Verpflichtung.

- `You should update your password regularly.` – Es ist eine gute Idee, Ihr Passwort aus Sicherheitsgründen häufig zu ändern.

- `You should check for errors in the code before running it.` – Es ist eine gute Praxis, Fehler zu finden, bevor Sie den Code ausführen.

Da Jake `we should focus on the critical systems first` sagt, gibt er eine starke Empfehlung, aber das Team ist nicht strikt verpflichtet, dies zu tun.
