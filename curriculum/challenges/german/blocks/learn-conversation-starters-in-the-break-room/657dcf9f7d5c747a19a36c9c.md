# --description--

Wenn Sie nach der Existenz bestimmter Orte oder Dinge an Ihrem aktuellen Standort fragen möchten, können Sie die Frageform `Are there any ... around here?` verwenden. Zum Beispiel:

`Are there any Wi-Fi spots around here?` – Sie fragen, ob Sie in der Nähe dieses Ortes WLAN finden können.

Der Ausdruck `around here` bezieht sich auf die umliegende Gegend.
