# --description--

让我们来了解超出学校算术范围的 JavaScript 操作符特色。

通常，算术操作符 `+` 用于求和数字。

但是，如果二元 `+` 应用于字串，它会将它们合并（连接）：

```js
let s = "my" + "string";
console.log(s); // mystring
```

请注意，如果任何一个操作对象是字串，则另一个操作对象也会被转换为字串。

例如：

```js
console.log(1 + '2'); // '12'
console.log('1' + 2); // '12'
```

看，第一操作对象是字串还是第二个操作对象都无所谓。

这是一个更复杂的示例：

```js
console.log(2 + 2 + '1') // "41" and not "221"
```

这里，操作符一个接一个地工作。第一个 `+` 将两个数字相加，所以它返回 `4`，然后下一个 `+` 将字串 `1` 添加到它上面，所以就像 `4 + '1' = '41'`。

```js
console.log('1' + 2 + 2) // "122" and not "14"
```

这里，第一个 `+` 将字串和数字连接起来，所以它返回 `'12'`，然后下一个 `+` 将 `2` 添加到它上面，所以就像 `'12' + 2 = '122'`。

二元 `+` 是唯一以这种方式支持字串的操作符。其他算术操作符仅对数字有效，并且总是将它们的操作对象转换为数字。

这是减法和除法的演示：

```js
console.log( 6 - '2' ); // 4, converts '2' to a number
console.log( '6' / '2' ); // 3, converts both operands to numbers
```

console.log('3' + 2 - 1);
