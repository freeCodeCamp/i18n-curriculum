# --explanation--

`Unexpected behavior` se refiere a resultados que difieren de lo que se pretendía. Esto puede incluir errores, cálculos incorrectos o saltarse casos importantes. Algunos ejemplos de comportamiento inesperado en bucles son:

- `Incorrect output` - El programa se ejecuta pero da resultados incorrectos.

- `Infinite repetition` - El ciclo se ejecuta para siempre.

- `Unexpected termination` - El ciclo podría causar un error que detenga la ejecución.

- `System crash` - Raro, pero posible con problemas relacionados con la memoria.
