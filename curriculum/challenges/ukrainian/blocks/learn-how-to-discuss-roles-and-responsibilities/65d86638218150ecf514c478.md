# --description--

Родовий відмінок (`'s`) використовується, щоб показати власність або зв’язок між двома речами.

Для іменників в однині додають `'s` в кінець слова, щоб показати приналежність. Наприклад, `the cat's toy` означає іграшку, що належить коту.

Для іменників у множині, які вже закінчуються на `-s`, просто додають `'` в кінці. Наприклад, `the teachers' lounge` означає зал для вчителів.

`facility` — це місце, де відбувається певна діяльність, наприклад, фабрика, офіс або школа.
