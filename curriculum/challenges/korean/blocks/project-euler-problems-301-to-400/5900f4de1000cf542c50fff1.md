# --description--

기하학적 삼각형을 변의 길이가 정수이고 $a ≤ b ≤ c$를 만족하며 변들이 등비수열을 이루는 삼각형, 즉 $b^2 = a \times c$인 삼각형으로 정의합니다.

이런 기하학적 삼각형의 예로 변의 길이가 $a = 144$, $b = 156$, $c = 169$인 삼각형이 있습니다.

둘레가 $\text{perimeter} ≤ {10}^6$인 기하학적 삼각형은 $861\\,805$개입니다.

둘레가 $\text{perimeter} ≤ 2.5 \times {10}^{13}$인 기하학적 삼각형은 몇 개일까요?
