# --interactive--

У JavaScript рядки розглядаються як послідовності символів, і до кожного символу рядка можна отримати доступ за допомогою дужкової нотації. Це дозволяє отримати конкретний символ із рядка на основі його позиції, яку називають індексом.

Індекс — це позиція символу в рядку, і він починається з нуля. Це означає, що перший символ рядка має індекс `0`, другий символ — індекс `1`, і так далі.

Наприклад, у рядку `hello` символ `h` знаходиться на індексі `0`, `e` — на індексі `1`, `l` — на індексі `2`, і так далі.

Дужкова нотація використовує квадратні дужки (`[]`) і індекс символу, до якого ви хочете отримати доступ. Розглянемо приклад:

:::interactive_editor

```js
let greeting = "hello";
console.log(greeting[1]); // "e"
```

:::

У цьому прикладі ми можемо отримати символ на індексі `1`, яким є `e`.

Щоб отримати останній символ рядка, можна використати довжину рядка мінус один.
Властивість `length` рядка показує, скільки символів він містить, тому щоб отримати останній символ, потрібно від довжини відняти один:

:::interactive_editor

```js
let greeting = "hello";
console.log(greeting[greeting.length - 1]); // "o"
```

:::

У цьому випадку `length` рядка `hello` дорівнює `5`, а останній символ (`o`) знаходиться на індексі `4`, який дорівнює `5 - 1`.

Якщо потрібно отримати кілька символів, можна використати дужкову нотацію так:

:::interactive_editor

```js
let greeting = "hello";
let firstTwo = greeting[0] + greeting[1]; // "he"
console.log(firstTwo);
```

:::

У цьому прикладі ми об’єднуємо перший і другий символи за допомогою дужкової нотації, щоб утворити рядок `he`.

Дужкова нотація корисна, коли потрібно отримати доступ до конкретних символів рядка, наприклад, щоб витягти ініціали з імені або перевірити певну літеру для валідації.
