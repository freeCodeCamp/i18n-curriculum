# --interactive--

Tujifunze kuhusu DOM na kwa nini ni muhimu sana kwa ukuzaji wa mtandao. DOM inasimama kwa Document Object Model. Ni kiolesura cha programu kinachoturuhusu kuingiliana na hati za HTML.

Kwa kutumia DOM, unaweza kuongeza, kubadilisha, au kufuta vitu kwenye ukurasa wa mtandao. Hata unaweza kufanya tovuti yako iwe ya kuingiliana kwa kufanya vitu visikilize na kujibu matukio.

Katika DOM, hati ya HTML inawakilishwa kama mti wa nodi. Kila nodi ya kipengele inawakilisha kipengele cha HTML kutoka kwa hati ya HTML:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>DOM Example</title>
  </head>
  <body>
    <h1>What is the DOM?</h1>
    <h2>Let's learn about the DOM</h2>
  </body>
</html>
```

Hii ni mchoro unaowakilisha muundo wa msingi wa DOM wa mfano wetu:

```md
Document
========

HTML
-----

- Head
  - Title
- Body
  - H1
  - H2   
```

Bila shaka, hii inaweza kuwa na maelezo zaidi na kuwa tata zaidi kulingana na muundo wa alama za uandishi za HTML za ukurasa wa mtandao.

Hati ya HTML ni nodi mzizi katika mfuatano wa DOM. Ina nodi moja ya mtoto, kipengele cha `html`. Hiki ndicho kipengele mzizi cha hati ya HTML, kwani nodi zote nyingine zinatoka kwake.

Kipengele cha `html` kina watoto wawili: `head` na `body`.

Kipengele cha `head` kina metadata kuhusu hati. Kinatoa taarifa muhimu kuhusu ukurasa wa mtandao.

Kipengele cha `body` kina maudhui yanayoonekana ya ukurasa wa mtandao.

Unaweza kupata upatikanaji wa vitu hivi kwa kutumia JavaScript, kufanya kazi navyo katika msimbo wako, na hata kubadilisha kwa nguvu. Hiyo ndiyo nguvu ya DOM na API za mtandao.

Ili kupata vitu hivi katika JavaScript, unaweza kutumia njia za `getElementById()` na `querySelector()`. Njia hizi ni API za mtandao kwa sababu zinatoa njia za kawaida za kuingiliana na DOM kwa kutumia JavaScript.

Kwa `getElementById()`, unaweza kupata kitu kinachowakilisha kipengele cha HTML chenye `id` iliyotajwa. Kumbuka kuwa `id`s lazima ziwe za kipekee katika kila hati ya HTML, hivyo njia hii itarudisha kitu kimoja tu cha `Element`. Hapa unaweza kuona mfano:

```js
const container = document.getElementById("container");
```

Mstari huu wa msimbo wa JavaScript unapata kipengele chenye thamani ya `id` ya `container` na kuweka kitu hicho katika thabiti ya JavaScript. Lazima upitishe `id` ndani ya alama za nukuu kama hoja. Ukichapisha kitu hiki kwenye konsoli, utaona matokeo:

:::interactive_editor

```html
<div id="container">
  <h1>Hello, World!</h1>
  <p>Welcome to learning about the DOM.</p>
</div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
console.log(container); 
```

:::

`querySelector()` ni pana zaidi kuliko `getElementById()`. Kwa `querySelector()`, unaweza kupata kipengele cha kwanza katika hati ya HTML kinacholingana na kichaguzi cha CSS kilichopitishwa kama hoja.

Katika mfano huu, utapata kipengele kinacholingana na kichaguzi cha `section`:

:::interactive_editor

```html
<section>
  <h2>Section Title</h2>
  <p>This is a section of the webpage.</p>
</section>
<script src="index.js"></script>  
```

```js
const sectionEl = document.querySelector("section");
console.log(sectionEl);
```

:::

Ikiwa unataka kuchagua kipengele kwa jina la darasa la vitu kwa kutumia `querySelector()`, unahitaji kuweka kiambishi awali cha nukta (`.`) kabla ya jina la darasa. Hapa kuna mfano:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="highlight">
  <p>This content is highlighted.</p>
</div>
<script src="index.js"></script>
```

```css
body {
  background-color: lightgray;
}

.highlight {
  background-color: yellow;
}
```

```js
const highlightEl = document.querySelector(".highlight");
console.log(highlightEl);
```

:::

Pia una njia nyingine za kulinganisha vitu vingi, kama `getElementsByClassName()` na `querySelectorAll()`. Utajifunza zaidi kuhusu hizi katika mafundisho yajayo.

Kwa kuelewa jinsi ya kutumia API za mtandao kwa ufanisi kudhibiti DOM, unaweza kuunda programu za mtandao zenye nguvu na zinazoweza kuingiliana.
