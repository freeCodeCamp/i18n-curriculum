# --explanation--

`Server`는 네트워크를 통해 다른 장치에 서비스, 데이터 또는 자원을 제공하는 강력한 컴퓨터나 시스템을 의미합니다. 예를 들어:  

`The server hosts the company's website.` - 이는 서버가 웹사이트의 콘텐츠를 저장하고 접근하는 사용자에게 전달하는 역할을 한다는 뜻입니다.

`Load`는 시스템이나 자원에 가해지는 작업량, 수요 또는 부담을 의미합니다. 예를 들어:  

`The system crashed because it couldn't handle the high load during peak hours.` - 이는 시스템이 많은 수요로 인해 과부하가 걸렸다는 뜻입니다.
