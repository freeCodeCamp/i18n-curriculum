# --description--

Sei $s_k$ die Anzahl der Einsen, wenn man die Zahlen von 0 bis $k$ in Binärschreibweise darstellt.

Zum Beispiel: Schreibt man die Zahlen von 0 bis 5 in Binärform, erhält man 0, 1, 10, 11, 100, 101. Es gibt sieben Einsen, also ist $s_5 = 7$.

Die Folge $S = \\{s_k : k ≥ 0\\}$ beginnt mit $\\{0, 1, 2, 4, 5, 7, 9, 12, \ldots\\}$.

Ein Spiel wird von zwei Spielern gespielt. Vor Spielbeginn wird eine Zahl $n$ gewählt. Ein Zähler $c$ startet bei 0. In jedem Zug wählt der Spieler eine Zahl von 1 bis $n$ (einschließlich) und erhöht $c$ um diese Zahl. Der neue Wert von $c$ muss ein Element von $S$ sein. Gibt es keinen gültigen Zug mehr, verliert der Spieler.

Zum Beispiel, mit $n = 5$ und Start bei $c = 0$:

- Spieler 1 wählt 4, also wird $c$ zu $0 + 4 = 4$.
- Spieler 2 wählt 5, also wird $c$ zu $4 + 5 = 9$.
- Spieler 1 wählt 3, also wird $c$ zu $9 + 3 = 12$.
- usw.

Beachten Sie, dass $c$ immer in $S$ liegen muss und jeder Spieler $c$ höchstens um $n$ erhöhen kann.

Sei $M(n)$ die höchste Zahl, die der erste Spieler in seinem ersten Zug wählen kann, um einen Sieg zu erzwingen, und $M(n) = 0$, wenn es keinen solchen Zug gibt. Zum Beispiel ist $M(2) = 2$, $M(7) = 1$ und $M(20) = 4$.

Es kann überprüft werden, dass $\sum M{(n)}^3 = 8150$ für $1 ≤ n ≤ 20$ gilt.

Finden Sie $\sum M{(n)}^3$ für $1 ≤ n ≤ 1000$.
