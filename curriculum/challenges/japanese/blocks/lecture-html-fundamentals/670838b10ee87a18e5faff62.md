# --interactive--

`script` 要素は実行可能なコードを埋め込むために使われます。ほとんどの開発者はこれを使ってJavaScriptコードを実行します。JavaScriptはウェブページにインタラクティブな機能を追加するために使われます。JavaScriptの一般的な使用例には、インタラクティブなゲーム、画像スライダー、ユーザー入力をリアルタイムで検証する動的なフォームなどがあります。

以下はHTMLドキュメントで`script` 要素を使った例です。`//` の前にある`alert("Welcome to freeCodeCamp");` を削除すると、アラートが表示されるはずです。プレビューを見るにはインタラクティブエディターを有効にしてください。

:::interactive_editor

```html
<body>
  <script>
    // alert("Welcome to freeCodeCamp");
  </script>
</body>
```

:::

技術的にはすべてのJavaScriptコードを`script` タグ内に書くこともできますが、外部JavaScriptファイルにリンクするほうがベストプラクティスとされています。以下は`script` 要素を使って外部JavaScriptファイルにリンクする例です。

```html
<script src="path-to-javascript-file.js"></script>
```

ここで`src` 属性は外部JavaScriptファイルの場所を指定するために使われています。`src` は「source」の略です。すべてのJavaScriptをHTMLドキュメント内に置くことが推奨されない理由は、関心の分離という設計原則にあります。関心の分離とは、プログラムを異なるセクションに分け、それぞれのセクションが異なる関心事を扱うようにすることです。この場合、JavaScriptコードをHTMLコードから分離したいのです。
