# --interactive--

웹사이트 스타일링에서 색상을 다루는 가장 다재다능하고 직관적인 방법 중 하나는 HSL 색상 모델을 사용하는 것입니다. HSL은 Hue, Saturation, Lightness의 약자로, 색상을 정의하는 세 가지 주요 요소입니다.

HSL 색상 모델은 인간이 색상을 인지하는 방식과 더 일치하는 방식으로 색상을 나타냅니다. 이 수업에서는 HSL 색상 모델이 무엇인지, 그리고 CSS에서 `hsl()` 함수가 어떻게 사용되는지 살펴보겠습니다.

Hue는 색상 유형으로, 색상환에서 각도로 나타내며 `0`도에서 `360`도까지 범위가 있습니다. 예를 들어, `0`도는 빨간색, `120`도는 초록색, `240`도는 파란색을 나타냅니다. Hue를 조정하면 스펙트럼상의 다양한 색상으로 이동합니다.

Saturation은 색상의 강도 또는 순도를 나타냅니다. `0%`% (완전히 탈채도된 회색조 색상)에서 `100%`%(가장 생생한 색상)까지 백분율로 측정합니다. `100%`%의 채도를 가진 색상은 완전히 선명하며, `0%`%의 채도를 가진 색상은 회색조로 보입니다.

Lightness는 색상의 밝거나 어두운 정도를 나타내며, 역시 백분율로 측정합니다. 밝기 값이 `0%`%이면 검은색이 되고, `50%`%이면 Hue의 정상 톤이 되며, `100%`%이면 흰색이 됩니다.

CSS에서는 `hsl()` 함수를 사용해 HSL 색상 모델로 색상을 정의합니다. 기본 구문은 다음과 같습니다:

```css
element {
  color: hsl(hue, saturation, lightness);
}
```

예제를 통해 자세히 살펴보겠습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
body {
  background-color: hsla(0, 0%, 1%, 1.00);
}

p {
  color: hsl(120, 100%, 50%);
}
```

:::

이 경우, Hue는 `120`도이며, 이는 초록색에 해당합니다. 채도는 `100%`%로 초록색이 완전히 선명합니다. 밝기는 `50%`%로 정상 톤이며, 너무 어둡거나 밝지 않습니다. 결과적으로 단락의 텍스트 색상은 밝은 초록색이 됩니다.

HSL 색상 모델의 주요 장점 중 하나는 직관적이라는 점입니다. Hue(핵심 색상)를 변경하지 않고도 채도와 밝기 값을 조절해 색상의 생동감이나 밝기를 쉽게 조절할 수 있습니다.

예를 들어, 같은 색상의 다양한 음영이나 색조를 만들고 싶다면 밝기 값을 조절하면 됩니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="light">This is a light blue div.</div>
<div class="dark">This is a dark blue div.</div>
```

```css
div.light {
  background-color: hsl(240, 100%, 80%);
}

div.dark {
  background-color: hsl(240, 100%, 20%);
  color: hsl(0, 0%, 100%);
}
```

:::

여기서 두 `div` 요소는 같은 Hue(파란색인 `240`도)를 사용하지만, 하나는 밝기 값이 `80%`%로 더 밝은 파란색이고, 다른 하나는 `20%`%로 더 어두운 파란색입니다.

RGB 모델에 투명도를 포함하는 `rgba()` 함수가 있듯이, HSL 모델에도 `hsla()` 함수가 있습니다. 이 함수의 네 번째 매개변수는 알파 값으로, 색상의 불투명도를 제어합니다. 기본 구문은 다음과 같습니다:

```css
element {
  background-color: hsla(hue, saturation, lightness, alpha);
}
```

예제를 살펴보겠습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

이 코드는 Hue가 `div`도(빨간색), 채도가 `0`%, 밝기가 `100%`%, 알파가 `50%`% (50% 불투명도)인 반투명 빨간색 배경을 `0.5`에 적용합니다. 알파 값을 백분율로 지정할 수도 있는데, 예를 들어 `hsla(0, 100%, 50%, 50%)`%는 `hsla(0, 100%, 50%, 0.5)`%와 같습니다.

HSL 색상 모델은 색상 조합을 만들고 음영이나 색조를 쉽게 조절해야 할 때 특히 유용합니다.

Hue를 채도와 밝기에서 분리했기 때문에, 그림자나 하이라이트를 위해 밝기를 조절하면서도 일관된 색상 테마를 유지할 수 있습니다. 이러한 유연성 덕분에 HSL은 디자이너와 개발자 모두에게 인기 있는 모델입니다.

또한 HSL은 16진수 코드나 RGB 같은 다른 방법에 비해 색상 값을 더 읽기 쉽고 이해하기 쉽게 만듭니다. 색상의 의도를 더 명확히 파악하고 다양한 디자인을 실험할 때 적절히 조절할 수 있습니다.

HSL 색상 모델을 사용하면 색조, 채도, 밝기를 더 유연하게 제어할 수 있어 CSS에서 풍부하고 역동적인 디자인을 만드는 데 강력한 도구가 됩니다.
