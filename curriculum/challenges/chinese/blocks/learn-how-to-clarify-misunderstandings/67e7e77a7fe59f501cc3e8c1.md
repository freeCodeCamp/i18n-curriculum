# --questions--

## --text--

哪个句子正确使用了名词性从句？

## --answers--

`I don't understand what does this code do.`

### --feedback--

名词性从句不使用疑问句语序。

---

`She asked me whether can AI replace programmers.`

### --feedback--

名词从句使用语句词序（`AI can`）而不是疑问词序（`can AI`）。

---

`The developer explained how the algorithm works.`

---

`We need to know when the update release.`

### --feedback--

名词从句中的动词结构不完整。

## --video-solution--

3

# --explanation--

在句子 `the developer explained how the algorithm works` 中，名词性从句被正确用作动词 `explained` 的宾语。该名词性从句提供了关于开发者解释内容的详细信息。

其他句子要么在名词从句中错误使用词序，要么动词结构不完整，导致语法错误。
