# --description--

145 是一个有趣的数字，因为 1! + 4! + 5! = 1 + 24 + 120 = 145。

查找等于其各位数字阶乘之和的数字及这些数字的和。

**注意：** 由于 1! = 1 和 2! = 2 不是求和，因此不包括在内。

```js
assert.deepEqual(digitFactorial(), { sum: 40730, numbers: [145, 40585] });
```
