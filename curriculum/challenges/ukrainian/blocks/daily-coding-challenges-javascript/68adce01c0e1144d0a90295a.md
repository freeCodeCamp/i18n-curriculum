# --description--

Дано матрицю (масив масивів), поверніть матрицю на 90 градусів за годинниковою стрілкою та поверніть її. Наприклад, для `[[1, 2], [3, 4]]`, яка виглядає так:

| 1 | 2 |
|---|---|
| 3 | 4 |

Ви маєте повернути `[[3, 1], [4, 2]]`, яка виглядає так:

| 3 | 1 |
|---|---|
| 4 | 2 |

```js
function rotate(matrix) {
  const n = matrix.length;
  const result = Array.from({ length: n }, () => Array(n).fill(0));

  for (let i = 0; i < n; i++) {
    for (let j = 0; j < n; j++) {
      result[j][n - 1 - i] = matrix[i][j];
    }
  }

  return result;
}
```
