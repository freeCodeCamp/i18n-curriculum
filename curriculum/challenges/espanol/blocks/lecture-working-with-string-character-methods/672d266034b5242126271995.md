# --interactive--

En programación, comprender cómo se representan los caracteres como números es fundamental. Aquí es donde entra en juego ASCII. ASCII, abreviatura de American Standard Code for Information Interchange, es un estándar de codificación de caracteres utilizado en computadoras para representar texto. Asigna un valor numérico a cada carácter, lo cual es reconocido universalmente por las máquinas.

En esta lección, exploraremos qué es ASCII, cómo funciona y cómo los métodos de JavaScript como `charCodeAt()` y `fromCharCode()` se relacionan con la codificación de caracteres. Aunque las cadenas de JavaScript usan Unicode (UTF-16) internamente, los valores ASCII coinciden con los primeros 128 caracteres Unicode, por eso los ejemplos basados en ASCII funcionan en JavaScript.

ASCII es un sistema para codificar caracteres como letras, dígitos y símbolos en valores numéricos. Cada carácter se asigna a un número específico.

Por ejemplo, la letra mayúscula `A` está representada por el número `65` en ASCII, mientras que la minúscula `a` es representada por `97`. Esta codificación permite a las computadoras almacenar y manipular texto.

El estándar ASCII cubre 128 caracteres, incluyendo:

- Letras inglesas mayúsculas y minúsculas (A-Z, a-z).
- Números (0-9).
- Signos de puntuación y símbolos comunes (!, @, #, y así sucesivamente).
- Caracteres de control (como nueva línea y tabulación).

En JavaScript, puedes acceder al código numérico de un carácter usando el método `charCodeAt()`. Este método devuelve la unidad de código UTF-16 del carácter en un índice especificado. Para los primeros 128 caracteres, este valor coincide con el código ASCII.

Veamos un ejemplo:

:::interactive_editor

```js
let letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

:::

En este ejemplo, `A` es el primer carácter de la cadena, y llamar a `charCodeAt(0)` devuelve su código numérico (que coincide con su valor ASCII para caracteres latinos básicos), `65`.

También puedes usar este método con otros caracteres para encontrar sus valores numéricos de código:

:::interactive_editor

```js
let symbol = "!";
console.log(symbol.charCodeAt(0));  // 33
```

:::

Aquí, el código numérico para el signo de exclamación `!` se devuelve como `33` (que coincide con su valor ASCII).

Mientras que `charCodeAt()` te ayuda a obtener el código numérico de un carácter, el método `fromCharCode()` te permite hacer lo contrario: convertir una unidad de código UTF-16 (que coincide con ASCII para caracteres básicos) en su carácter correspondiente.

Veamos esto en acción:

:::interactive_editor

```js
let char = String.fromCharCode(65);
console.log(char);  //  A
```

:::

En este ejemplo, `fromCharCode(65)` convierte el código numérico `65` (que coincide con el valor ASCII de `A`) de nuevo al carácter `A`.

Otro ejemplo sería convertir el número `97` en su letra minúscula correspondiente:

:::interactive_editor

```js
let char = String.fromCharCode(97);
console.log(char);  // a
```

:::

Estos métodos son particularmente útiles cuando necesitas manipular o comparar caracteres basándote en sus valores numéricos de código.

Por ejemplo, podrías usar `charCodeAt()` para verificar si un carácter es mayúscula, minúscula o un dígito comparando su valor ASCII.

Por otro lado, `fromCharCode()` puede ser utilizado para generar dinámicamente caracteres a partir de sus códigos ASCII.
