# --description--

Рядок довжиною `n` одиниць має розміщені на ньому червоні блоки мінімальної довжини `m` одиниць, причому будь-які два червоні блоки (які можуть мати різну довжину) розділені принаймні одним чорним квадратом.

Нехай функція підрахунку заповнення, $F(m, n)$, представляє кількість способів, якими можна заповнити рядок.

Наприклад, $F(3, 29) = 673135$ і $F(3, 30) = 1089155$.

Тобто для m = 3 видно, що n = 30 — це найменше значення, при якому функція підрахунку заповнення вперше перевищує один мільйон.

Аналогічно, для m = 10 можна перевірити, що $F(10, 56) = 880711$ і $F(10, 57) = 1148904$, тому n = 57 — це найменше значення, при якому функція підрахунку заповнення вперше перевищує один мільйон.

Для m = 50 знайдіть найменше значення `n`, при якому функція підрахунку заповнення вперше перевищує один мільйон.

**Примітка:** Це складніша версія завдання 114.
