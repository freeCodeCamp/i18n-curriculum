# --interactive--

JavaScript에서 문자열을 다룰 때, 문자열에 특정 부분 문자열이 포함되어 있는지 확인해야 하는 경우가 많습니다. 부분 문자열은 그 문자열의 더 작은 일부입니다.

예를 들어, 사용자의 입력에 특정 단어나 문자가 포함되어 있는지 확인한 후 어떤 작업을 수행하고 싶을 수 있습니다. 이를 달성하는 한 가지 방법은 `includes()` 메서드를 사용하는 것입니다.

`includes()` 메서드는 문자열에 특정 부분 문자열이 포함되어 있는지 확인하는 데 사용됩니다. 부분 문자열이 문자열 내에 있으면 이 메서드는 `true`를 반환하고, 그렇지 않으면 `false`를 반환합니다.

기본 구문은 다음과 같습니다:

```js
string.includes(searchValue);
```

구문에서 `searchValue`는 문자열 내에서 찾고자 하는 부분 문자열입니다. 예를 들어:

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("awesome");

console.log(result);  // true
```

:::

이 예제에서 `awesome`라는 단어가 `JavaScript is awesome!` 문자열 내에 있으므로 `includes()` 메서드는 `true`를 반환합니다.

중요한 점은 `includes()` 메서드는 대소문자를 구분한다는 것입니다. 즉, 문자들의 대소문자까지 정확히 일치해야 합니다.

예를 들어:

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("Awesome");

console.log(result);  // false
```

:::

`Awesome` (대문자 `A`)는 `awesome` (소문자 `a`)와 일치하지 않으므로 결과는 `false`입니다.

또한 `includes()` 메서드를 사용해 문자열 내 특정 인덱스부터 부분 문자열을 찾도록 두 번째 매개변수를 제공할 수 있습니다:

:::interactive_editor

```js
let text = "Hello, JavaScript world!";
let result = text.includes("JavaScript", 7);

console.log(result);  // true
```

:::

여기서는 부분 문자열 `JavaScript`를 문자열의 7번째 위치부터 검색하여 그 이전 문자는 건너뜁니다.

`includes()` 메서드는 `true` 또는 `false` 결과만 반환합니다. 부분 문자열이 문자열 내 어디에 위치하는지, 몇 번 나타나는지에 대한 정보는 제공하지 않습니다. 그런 세부 정보가 필요하다면 `indexOf()` 메서드 같은 다른 메서드가 더 적합할 수 있습니다.
