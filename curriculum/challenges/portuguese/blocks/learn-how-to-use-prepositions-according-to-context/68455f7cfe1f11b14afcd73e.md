`He shared the layout of the new office.` – Isso significa o plano de onde as coisas ficam.

`Draft` é uma versão inicial de um documento.Por exemplo:  

`She wrote the first draft of the proposal.` – Ainda pode ser editado.

`Send it over` significa entregar algo.Por exemplo:  

`I'll send it over once it's ready.` – Significa compartilhar um arquivo ou mensagem.

`To` mostra para quem a ação é destinada.Por exemplo:  

`She gave the file to her manager.` – Indica quem o recebe.
