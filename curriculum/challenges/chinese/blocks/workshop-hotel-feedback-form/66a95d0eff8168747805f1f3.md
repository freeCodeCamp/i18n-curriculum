# --description--

如果希望用户从列表中选择多个选项，可以使用复选框。

下面是一个如何使用复选框处理食物选项的示例：

```html
<fieldset>
  <legend>Food Options</legend>
  <input type="checkbox" id="pizza" name="food" value="pizza">
  <label for="pizza">Pizza</label>
  <input type="checkbox" id="burger" name="food" value="burger">
  <label for="burger">Burger</label>
</fieldset>
```

`value` 属性用于指定提交表单时发送到服务器的值。

在你的 `legend` 元素下方，添加一个复选框 `input`，其 `id`、`name` 和 `value` 属性都设置为 `"ads"`。

在你的复选框 `input` 下方，添加一个 `label` 元素，其文本为 `Social Media Ads`。 `for` 属性应设置为 `"ads"`。
