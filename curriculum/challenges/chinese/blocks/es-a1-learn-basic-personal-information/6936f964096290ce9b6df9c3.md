# --description--

要问某人是否想要某物，西班牙语使用动词 `querer`。

形式 `quieres` 与 `tú`（你）一起使用，用于询问另一个人的愿望或偏好。这在日常对话中很常见。例如：

- `¿Quieres ayuda?` - 你想要帮助吗？

- `¿Quieres más tiempo？` - 你需要更多时间吗？
