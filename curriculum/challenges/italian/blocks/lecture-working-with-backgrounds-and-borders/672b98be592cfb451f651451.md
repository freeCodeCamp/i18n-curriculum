# --interactive--

Un gradiente di sfondo in CSS è una transizione graduale tra due o più colori che può essere applicata allo sfondo di un elemento. I gradienti ti permettono di creare sfondi visivamente accattivanti senza dover usare immagini.

Ci sono due tipi principali di gradienti in CSS: gradienti lineari e gradienti radiali.

Un gradiente lineare sfuma i colori lungo una linea retta. Puoi definire la direzione e i colori coinvolti.

Ecco la sintassi di base:

```css
background: linear-gradient(direction, color-stop1, color-stop2, ...);
```

In questo esempio, stiamo usando la proprietà CSS `background` con un valore di `linear gradient`.

La direzione specifica la direzione del gradiente. Può essere un angolo (come `45deg`), una parola chiave (come `to right`, `to bottom`) o un lato/angolo.

`color-stop` specifica i colori e le posizioni in cui avvengono le transizioni del gradiente.

Per capire meglio come funzionano i gradienti lineari, diamo un’occhiata al seguente esempio:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient{
  background: linear-gradient(to right, red, yellow);
  height: 40vh;
}
```

:::

Questo CSS crea un gradiente lineare che sfuma da `red` sul `left` a `yellow` sul `right`. Il gradiente è applicato a un elemento con un’altezza pari a `40%` dell’altezza del viewport. Imparerai di più sulle unità `vh` in una lezione futura.

La direzione `to right` significa che il gradiente scorre orizzontalmente da sinistra a destra.

Un altro tipo di gradiente è il gradiente `radial`.

Un gradiente radiale sfuma i colori irradiando da un’origine (di solito il centro) verso l’esterno in una forma circolare o ellittica.

Ecco la sintassi di base:

```css
background: radial-gradient(shape size at position, color-stop1, color-stop2, ...)
```

Nella sintassi, `shape` specifica la forma del gradiente che può essere `circle` o `ellipse`.

`size` determina la dimensione della forma finale del gradiente che può essere `closest-side`, `closest-corner`, `farthest-side` o `farthest-corner`.

`position` determina la posizione del centro del gradiente che può essere specificata usando parole chiave (come `center`, `top left`, `bottom right`) o valori precisi (come `50% 50%`, `10px 20px`).

Infine, le fermate di colore sono una lista di colori attraverso cui il gradiente sfuma. Ogni fermata di colore può opzionalmente includere un valore di posizione (percentuale o lunghezza) che indica dove il colore deve essere posizionato.

Un esempio sarebbe:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient{
  background: radial-gradient(circle closest-side at center, red, yellow 50%, green);
  height: 60vh;
}
```

:::

Questo CSS crea un gradiente radiale circolare centrato nell’elemento. Inizia con `red` al centro, sfuma verso `yellow` a `50%` del raggio, e termina con `green`.

La parola chiave `closest-side` fa sì che la forma finale del gradiente si adatti al lato più vicino dell’elemento. Il gradiente è applicato a un elemento con un’altezza pari a `60%` dell’altezza del viewport.

Capire come lavorare con i gradienti CSS può migliorare significativamente i tuoi design fornendo sfondi visivamente accattivanti senza bisogno di immagini.

Con opzioni come i gradienti lineari per transizioni morbide e i gradienti radiali per effetti circolari, offrono sia flessibilità sia creatività nel design web.
