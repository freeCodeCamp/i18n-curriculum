# --explanation--

Com a entrada Pinyin-Hanzi, você pode digitar as letras mais um número de tom para convertê-las automaticamente em caracteres. Por exemplo, aqui, `ni3 hao3` se torna `你好` e `shi4 jie4` se torna `世界`.

*Nota: Tanto o Pinyin-Tone Input quanto o Pinyin-Hanzi Input convertem sua entrada de letras e números em Pinyin ou caracteres chineses. O sistema decide automaticamente qual conversão aplicar, então você não precisa se preocupar com o processo.*
