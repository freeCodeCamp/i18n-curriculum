## --quiz--

### --question--

#### --text--

次のうち、クラスを定義する正しい方法はどれですか？

#### --distractors--

```py
def className(class):
    pass
```

---

```py
self className:
    pass
```

---

```py
class className
    pass
```

#### --answer--

```py
class className:
    pass
```

### --question--

#### --text--

特別なメソッド `__init__` の目的は何ですか？

#### --distractors--

クラスのインスタンスを作成するためのショートカットを設定します。

---

オブジェクトが重複した属性を持つのを防ぎます。

---

`self` の値を現在のオブジェクトに設定します。

#### --answer--

インスタンス化時にオブジェクトの属性を初期化します。

### --question--

#### --text--

インスタンス属性とクラス属性の違いは何ですか？

#### --distractors--

インスタンス属性はオブジェクトからアクセスできるメソッドを定義し、クラス属性はクラスからアクセスできるメソッドを定義します。

---

インスタンス属性は `__init__` メソッド内で設定され、クラス属性は `__class__` メソッド内で設定されます。

---

インスタンス属性はクラスから直接アクセスでき、クラス属性は実際のインスタンスからのみアクセスできます。

#### --answer--

インスタンス属性は特定のオブジェクトに属し、クラス属性はクラスに属します。
