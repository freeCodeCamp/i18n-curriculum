# --description--

`Work overtime` 是指工作時間超過平常的工作排程。 例如，`I usually work overtime during project deadlines` 意味著為了趕上專案截止日期而加班。

`Unless` 用來說明某事的例外或介紹使前述述語不成立的條件。它的意思是 `if it's not the case`。例如，`I walk to work unless it rains` 意味著如果沒有下雨，我就走路去上班。

`Urgent` 意味著某事非常重要且需要立即關注。 例如，`The project is urgent` 意味著該專案需要立即完成。

在否定句中，`don't/doesn't have to` 表示某事不是必須的。 例如，`I don't have to work late` 意味著加班不是必須的。
