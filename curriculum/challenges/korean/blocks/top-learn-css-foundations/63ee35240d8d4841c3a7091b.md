# --description--

클래스 선택자는 주어진 `class`를 가진 모든 요소를 선택합니다. `class`는 HTML 요소에 붙이는 속성입니다. HTML 태그에 클래스를 추가하고 CSS에서 선택하는 방법은 다음과 같습니다:

```html
<!-- index.html -->

<div class="alert-text">
  Please agree to our terms of service.
</div>
```

```css
/* styles.css */

.alert-text {
  color: red;
}
```

`class` 선택자의 구문에 주목하세요: 점(.) 다음에 클래스 속성의 대소문자를 구분하는 값이 바로 옵니다. 클래스는 고유할 필요가 없으므로 원하는 만큼 많은 요소에 같은 `class`를 사용할 수 있습니다.

`class` 속성으로 할 수 있는 또 다른 방법은 단일 요소에 여러 클래스를 공백으로 구분하여 추가하는 것입니다. 예를 들어 `class="alert-text severe-alert"`처럼요. 공백이 `class` 이름을 구분하는 데 사용되므로, 여러 단어로 된 이름에는 공백 대신 하이픈을 사용해야 합니다.

## ID 선택자
ID 선택자는 `class` 선택자와 비슷합니다. HTML 요소에 붙이는 또 다른 속성인 `id`를 가진 요소를 선택합니다:

```html
<!-- index.html -->

<div id="title">My Awesome 90's Page</div>
```

```css
/* styles.css */

#title {
  background-color: red;
}
```

점(.) 대신 해시 기호(#)를 사용하며, 그 뒤에 `id` 속성의 대소문자를 구분하는 값이 바로 옵니다. 흔한 실수는 클래스만으로 충분할 때 `id` 속성을 과도하게 사용하는 것입니다. 특정성 활용이나 현재 페이지의 섹션으로 링크를 리다이렉트하는 등 `id`를 사용하는 것이 합리적이거나 필요한 경우가 있지만, `id`는 가능한 한 적게(또는 아예) 사용해야 합니다.

클래스와 ID의 가장 큰 차이는 요소가 가질 수 있는 `id`가 하나뿐이라는 점입니다. `id`는 한 페이지에서 중복될 수 없으며, `id` 속성에는 공백이 전혀 포함되어서는 안 됩니다.
