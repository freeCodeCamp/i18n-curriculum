# --explanation--

マテオは`¿Dónde vive Valeria?`と尋ねています。これは「バレリアはどこに住んでいますか？」という意味です。

`¿Dónde...?`は「どこ…？」という意味です。

質問文の最初に開きの疑問符`¿`を入れ、`ó`の文字にアクセント記号を付けることを忘れないでください。

`vive`は「住む」という意味で、動詞`vivir`の三人称単数形です。
