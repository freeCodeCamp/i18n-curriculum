# --description--

ある正の整数 $n$ は、$n + reverse(n)$ の和がすべて奇数の（10進）数字で構成される性質を持つ場合があります。例えば、$36 + 63 = 99$ や $409 + 904 = 1313$ です。このような数を可逆数と呼びます。したがって、36、63、409、904 は可逆数です。$n$ も $reverse(n)$ も先頭にゼロを含んではいけません。

1000未満には120個の可逆数があります。

10億未満（${10}^9$）にはいくつの可逆数があるでしょうか？
