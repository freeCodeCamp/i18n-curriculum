# --description--

`To be all set` bedeutet, vollständig vorbereitet oder startklar zu sein. Zum Beispiel zeigt `Once you finish your training, you'll be all set to start working`, dass Sie nach dem Training bereit sind, mit der Arbeit zu beginnen.

`Things` in `first things first` betont die Wichtigkeit, die wichtigsten Aufgaben vor allen anderen zu erledigen.

`To install` bedeutet, Software oder Geräte so einzurichten, dass sie einsatzbereit sind. Zum Beispiel `You need to install the software on your computer.`

`Correctly` bedeutet auf die richtige Weise oder fehlerfrei. Zum Beispiel impliziert `Make sure you follow the instructions correctly`, dass Sie Anweisungen genau befolgen sollten.
