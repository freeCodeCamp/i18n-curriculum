# --description--

Se colocan discos N×N en un tablero de juego cuadrado. Cada disco tiene un lado negro y un lado blanco.

En cada turno, puedes elegir un disco y voltear todos los discos en la misma fila y la misma columna que este disco: así se voltean $2 × N - 1$ discos. El juego termina cuando todos los discos muestran su lado blanco. El siguiente ejemplo muestra un juego en un tablero de 5×5.

<img alt="animation showing game on 5x5 board" src="https://cdn.freecodecamp.org/curriculum/project-euler/cross-flips.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Se puede demostrar que 3 es el número mínimo de turnos para terminar este juego.

El disco inferior izquierdo en el tablero $N×N$ tiene las coordenadas (0, 0); el disco inferior derecho tiene las coordenadas ($N - 1$, $0$) y el disco superior izquierdo tiene las coordenadas ($0$, $N - 1$).

Sea $C_N$ la siguiente configuración de un tablero con discos de $N × N$: Un disco en ($x$, $y$) que cumple $N - 1 \le \sqrt{x^2 + y^2} \lt N$, muestra su lado negro; de lo contrario, muestra su lado blanco. $C_5$ se muestra arriba.

Sea $T(N)$ el número mínimo de turnos para terminar un juego comenzando desde la configuración $C_N$ o 0 si la configuración $C_N$ no tiene solución. Hemos demostrado que $T(5) = 3$. También se te da que $T(10) = 29$ y $T(1\\,000) = 395\\,253$.

Encuentra $\displaystyle \sum_{i = 3}^{31} T(2^i - i)$.
