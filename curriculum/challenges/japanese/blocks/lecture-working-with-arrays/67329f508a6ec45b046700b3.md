# --interactive--

JavaScriptの配列は、数値のインデックスで識別される値の順序付きコレクションです。JavaScriptの配列内の値は、数値、文字列、ブール値、オブジェクト、さらには他の配列など、さまざまなデータ型を含むことができます。

JavaScriptで配列を作成するには、大括弧を使い、`[]`、値をカンマで区切ってください。例を示します。

```js
let fruits = ["apple", "banana", "orange"];
```

この例では、変数`fruits`を宣言し、3つの文字列値`apple`、`banana`、`orange`を含む配列を代入しています。

配列の主な特徴の1つは、ゼロから始まるインデックスを持つことです。つまり、配列の最初の要素はインデックス`0`、2番目の要素はインデックス`1`を持ちます。このインデックスを使って配列の個々の要素にアクセスできます。例えば：

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits[0]); // "apple"
console.log(fruits[2]); // "orange"
```

:::

この例では、インデックス`0`を使って最初の要素（`apple`）にアクセスし、インデックス`2`を使って3番目の要素（`orange`）にアクセスしています。

JavaScriptの配列には特別な`length`プロパティがあり、配列内の要素数を返します。このプロパティには`length`キーワードを使ってアクセスできます。例えば：

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits.length); // 3
```

:::

JavaScriptの配列のもう1つの重要な特徴は動的であることです。つまり、作成後にサイズを変更できます。`push()`、`pop()`、`shift()`、`unshift()`、`splice()`などのさまざまな配列メソッドを使って要素を追加したり削除したりできます。これらのメソッドは今後のレッスンで学びます。

JavaScriptの配列は多用途で、プログラム内のデータ保存に役立ちます。このモジュールを通じて、配列を使うことでデータのコレクションを効率的に管理・操作できることを実感できるでしょう。
