# --description--

A `assign tasks` significa dar trabalho ou deveres específicos a alguém. Por exemplo:

- `The manager assigns tasks to the team every morning.` - O gerente distribui o trabalho para cada membro da equipe.

- `They assign tasks based on skills.` - Eles distribuem o trabalho dependendo do que cada pessoa é boa.
