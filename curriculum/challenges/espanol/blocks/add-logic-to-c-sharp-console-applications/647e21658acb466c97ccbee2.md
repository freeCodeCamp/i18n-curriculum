## --text--

¿Cuál de las siguientes afirmaciones es verdadera sobre mostrar/eliminar las llaves para los bloques de código asociados con una sentencia `if`? 

## --answers--

Las llaves no se pueden eliminar del bloque de código para las sentencias `else if` y `else`.

---

Si se eliminan las llaves de los bloques de código de un `if-elseif-else`, también se debe eliminar el espacio en blanco.

---

Elige siempre un estilo que mejore la legibilidad.
