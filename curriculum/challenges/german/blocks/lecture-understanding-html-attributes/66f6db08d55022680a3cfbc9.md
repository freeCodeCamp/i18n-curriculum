# --interactive--

HTML, das für HyperText Markup Language steht, ist eine Auszeichnungssprache zur Erstellung von Webseiten. Wenn Sie eine Webseite besuchen und Inhalte wie Absätze, Überschriften, Links, Bilder und Videos sehen, ist das HTML.

Hier ist ein kleines Beispiel mit HTML-Elementen. Versuchen Sie, einige Texte im Editor zu bearbeiten, und sehen Sie, wie sich die Änderungen im Vorschaufenster aktualisieren.

:::interactive_editor

```html
<h1>Main heading element</h1>

<p>I am a paragraph element.</p>
```

:::

HTML stellt den Inhalt und die Struktur einer Webseite durch die Verwendung von Elementen dar. Die meisten Elemente haben ein öffnendes Tag und ein schließendes Tag. Manchmal werden diese Tags auch als Start- und Endtags bezeichnet. Zwischen diesen beiden Tags befindet sich der Inhalt. Dieser Inhalt kann Text oder andere HTML-Elemente sein.

Hier ist ein weiteres Beispiel für ein Absatz-Element. Ändern Sie den Text im Editor zu `I love coding!` und sehen Sie die Ergebnisse im Vorschaufenster.

:::interactive_editor

```html
<p>I am a paragraph element.</p>
```

:::

Sowohl öffnende als auch schließende Tags beginnen mit einer linken spitzen Klammer (`<`) und enden mit einer rechten spitzen Klammer (`>`), wobei der Tag-Name zwischen diesen spitzen Klammern steht. Obwohl HTML-Tag-Namen nicht zwischen Groß- und Kleinbuchstaben unterscheiden, ist es eine weit verbreitete Konvention und bewährte Vorgehensweise, sie in Kleinbuchstaben zu schreiben.

Hier ist eine genauere Ansicht nur der öffnenden und schließenden Absatz-Tags:

```html
<p>
```

```html
</p>
```

Was ein öffnendes Tag von einem schließenden Tag unterscheidet, ist der Schrägstrich (`/`), der unmittelbar nach der linken spitzen Klammer im schließenden Tag steht. Einige HTML-Elemente haben kein schließendes Tag. Diese werden als Void-Elemente bezeichnet.

Hier ist ein Beispiel für ein Bild-Element, das ein Void-Element ist:

```html
<img>
```

Beachten Sie, dass dieses Bild-Element kein schließendes Tag hat und keinen Inhalt besitzt. Void-Elemente können keinen Inhalt haben und besitzen nur ein öffnendes Tag.

Manchmal sehen Sie Void-Elemente, die ein `/` vor dem `>` verwenden, so wie hier:

```html
<img />
```

Während viele Code-Formatter wie Prettier sich dafür entscheiden, das `/` bei Void-Elementen einzufügen, besagt die HTML-Spezifikation, dass das Vorhandensein des `/` „den Start-Tag nicht als selbstschließend kennzeichnet, sondern unnötig ist und keinerlei Wirkung hat“.

In der realen Entwicklung werden Sie beide Formen sehen, daher ist es wichtig, mit beiden vertraut zu sein.

Wenn Sie ein Bild anzeigen möchten, müssen Sie ein paar Attribute in Ihrem Bildelement einfügen. Ein Attribut ist ein spezieller Wert, der verwendet wird, um das Verhalten eines HTML-Elements anzupassen.

Hier ist ein Beispiel für ein Bild-Element mit einem `src`-Attribut. Ändern Sie den Wert des `src`-Attributs zu `"https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"` und Sie werden sehen, wie sich das Bild zu zwei friedlich schlafenden Katzen ändert.

:::interactive_editor

```html
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/running-cats.jpg" />
```

:::

Das `src`-Attribut wird verwendet, um den Speicherort für dieses Bild anzugeben. Für Bild-Elemente ist es eine bewährte Vorgehensweise, ein weiteres Attribut namens `alt`-Attribut hinzuzufügen. Das `alt`-Attribut wird verwendet, um kurzen, beschreibenden Text für die Bilder bereitzustellen.

Hier ist ein Beispiel für ein Bild-Element mit den `src`- und `alt`-Attributen. Versuchen Sie, das Bild zu zerstören, indem Sie den Wert von `src` auf `"https://.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"` ändern. Sie werden sehen, dass das Bild verschwindet und nur der `alt`-Text angezeigt wird.

:::interactive_editor

```html
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" alt="Two tabby kittens sleeping together on a couch." />
```

:::

Vielleicht fragen Sie sich, ob HTML allein ausreicht, um eine Webseite zu erstellen. Die Antwort lautet: Es kommt darauf an. Wenn Sie ein kleines Übungsprojekt erstellen, das nur Text und Bilder anzeigt, könnte HTML allein ausreichen. Wenn Sie jedoch eine moderne professionelle Webseite erstellen, benötigen Sie HTML, CSS und JavaScript.

HTML ist für den Inhalt und die Struktur zuständig. CSS ist für das Styling zuständig. JavaScript sorgt für Interaktivität auf Ihren Webseiten. Eine gute Analogie ist, HTML, CSS und JavaScript mit einem kompletten Gebäude zu vergleichen.

HTML stellt die Bausteine, den Beton und das Eisen dar, aus denen die Wände bestehen. Es ist das Fundament, das das Gebäude stark macht. CSS steht für das Innen- und Außendesign, das das Gebäude schön aussehen lässt. JavaScript steht für das elektrische und Wassersystem, das den ununterbrochenen Zugang zu Wasser und Strom sicherstellt.
