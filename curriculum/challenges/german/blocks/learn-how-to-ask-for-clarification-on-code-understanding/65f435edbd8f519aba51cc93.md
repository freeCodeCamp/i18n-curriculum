# --description--

Ein `function` im Programmieren ist eine Reihe von Anweisungen innerhalb eines Programms. Sie ist dafür gedacht, eine bestimmte Aufgabe auszuführen. Sie definieren eine `function` einmal, indem Sie festlegen, was sie tun soll, und können sie dann so oft verwenden, wie Sie sie brauchen, indem Sie ihren Namen aufrufen.

Wenn etwas im Code `missing` ist, bedeutet das, dass ein notwendiger Teil fehlt oder übersehen wurde.

Wenn Code `working as expected` ist, funktioniert er genau so, wie er soll, ohne Fehler oder Probleme.
