# --description--

In dieser Aufgabe hören Sie, wie Basti seine vollständige E-Mail-Adresse nennt.

Einige nützliche Wörter erscheinen in diesem Satz:

`Correo` (Kurzform von `correo electrónico`) bedeutet E-Mail.

`Completo` bedeutet „vollständig“ oder „komplett“.

`Arroba` ist das spanische Wort für das `@`-Symbol, das in E-Mail-Adressen verwendet wird.
