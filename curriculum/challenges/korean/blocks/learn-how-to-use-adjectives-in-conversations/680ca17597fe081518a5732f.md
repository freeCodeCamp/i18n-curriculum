# --explanation--

`Last time`는 가장 최근의 사건을 가리킵니다. 예를 들어:

`Last time we updated the server, we had some downtime.` – 이는 이전 업데이트 동안을 나타냅니다.

`Order`는 공식적으로 상품이나 장비를 요청하는 것을 의미합니다. 과거형은 `ordered`입니다. 예를 들어:

`The IT team ordered new laptops for the developers.` – 이는 노트북을 요청했다는 뜻입니다.

`Double that`는 이전 수량의 두 배를 의미합니다. 예를 들어:

`Last time we ordered 10 routers; this time we need double that, so 20 routers.` – 이는 수량이 두 배로 증가했음을 보여줍니다.
