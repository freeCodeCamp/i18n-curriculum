# --description--

要访问数组的元素，你可以使用索引号。索引号从 0 开始，所以数组的第一个元素位于索引 0，第二个元素位于索引 1，依此类推。

例如，要访问 `fruits` 数组的第一个元素，你可以使用以下代码：

```javascript
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits[0]); // Output: apple
console.log(fruits[2]); // Output: orange
```

如果索引超出范围，JavaScript 会返回 `undefined`。这意味着如果你尝试访问上述示例中的 `fruits[3]`，它将返回 `undefined`。
