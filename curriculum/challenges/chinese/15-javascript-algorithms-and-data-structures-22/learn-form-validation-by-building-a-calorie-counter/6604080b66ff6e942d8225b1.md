# --description--

当你打开控制台时，你应该看到以下结果：

```js
[ '1e3', index: 0, input: '1e3', groups: undefined ]
```

`match` 方法返回一个包含字符串中找到的所有匹配项的数组。

以下是该信息的完整分类：

- `"1e3"` 是与 `/\d+e\d+/i` 正则表达式匹配的值。
- `index: 0` 是字符串中匹配值的索引。
- `input: '1e3'` 是匹配的原始字符串。
- `groups: undefined` 是匹配的组，在本例中未使用。 你将在以后的项目中了解有关群组的更多信息。

现在是时候测试有效输入了。 将控制台语句更新为以下内容：`console.log(isInvalidInput("10"))`。

打开控制台查看结果。 你将在下一步中详细了解此结果的含义。
