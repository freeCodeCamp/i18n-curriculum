# --instructions--

Für diese Aufgabe müssen Sie drei Mathe-Spiele mit Python erstellen, die Folgendes tun:

- Scatter-Plot-Spiel
  - Zufällig Punkte auf einem Diagramm generieren, und der Spieler muss die (x,y)-Koordinaten eingeben
  - Für mehr Schwierigkeit das Diagramm größer machen
- Algebra-Übungsspiel
  - Ein- und zweistufige Aufgaben mit zufälligen Ganzzahlen generieren, und der Spieler muss die Antwort eingeben
  - Verwenden Sie positive und negative Werte. Für mehr Schwierigkeit die Zahlen größer machen
- Projektil-Spiel
  - Eine „Wand“ mit zufälliger Höhe und Position anzeigen. Der Spieler muss Schieberegler bewegen, um eine parabolische Bahn einzustellen, die die Wand überquert
  - Für mehr Schwierigkeit eine zweite Stufe erstellen, in der Spieler a, b und c ohne Schieberegler eingeben
