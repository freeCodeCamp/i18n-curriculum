# --description--

在专业上下文中，动词 `to document` 意味着记录信息以供将来使用。它涉及编写细节、步骤或重要点。 

`Future reference` 指的是在以后使用这些已记录的信息，通常用于解决问题或查全过程。例如：

`They are documenting their studies for future reference.`
