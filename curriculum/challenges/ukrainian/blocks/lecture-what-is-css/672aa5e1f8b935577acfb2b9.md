# --interactive--

CSS, що означає каскадні таблиці стилів, є ключовим компонентом сучасної веброзробки. Це мова розмітки, яка використовується для застосування стилів до HTML. Простими словами, якщо HTML — це структура вебсторінки, то CSS — це те, що робить її привабливою.

Ось приклад стилізації панелі навігації.

**ПРИМІТКА**: Не хвилюйтеся, якщо поки що не розумієте CSS-код. Ви навчитеся, як це все працює, у майбутніх уроках і практичних заняттях. Це лише для того, щоб дати уявлення про те, що можна зробити з невеликою кількістю CSS.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<nav class="navbar">
    <ul class="nav-links">
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Services</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>
```

```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: 'Segoe UI', sans-serif;
  background-color: #f4f4f4;
}

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  background-color: #333;
  padding: 1rem 2rem;
  color: white;
}

.navbar .logo {
  font-size: 1.5rem;
  font-weight: bold;
}

.nav-links {
  list-style: none;
  display: flex;
  gap: 1.5rem;
}

.nav-links a {
  color: white;
  text-decoration: none;
  font-weight: 500;
  transition: color 0.3s ease;
}

.nav-links a:hover {
  color: #ff9800;
}
```

:::

Основна роль CSS — відокремити вміст вебсторінки від її візуального оформлення. Це розділення дозволяє розробникам створювати більш гнучкі та зручні для підтримки сайти.

За допомогою CSS ви можете контролювати макет, кольори, шрифти та загальний візуальний вигляд вебсторінок без зміни структури HTML.

Розглянемо просту аналогію. Якщо уявити вебсайт як будинок, то HTML — це фундамент і каркас, а CSS — це фарба, шпалери та декорації, які роблять будинок привабливим і унікальним.

CSS працює, вибираючи HTML-елементи та застосовуючи до них стилі.

Ось приклад, який показує нестилізований `button` елемент і стилізований. Щоб взаємодіяти з цим прикладом, потрібно увімкнути інтерактивний редактор.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<button class="regular-btn">Unstyled Button</button>
<button class="round-btn">Round Button</button>
```

```css

.round-btn {
  font-size: 1rem;
  font-family: 'Segoe UI', sans-serif;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease, transform 0.2s ease;
  background-color: #2ecc71;
  color: white;
  border-radius: 50px;
  padding: 0.6rem 1.6rem;
}

.round-btn:hover {
  background-color: #27ae60;
  transform: translateY(-2px);
}
```

:::

Ці стилі можуть включати властивості, такі як колір, розмір шрифту та багато інших. Змінюючи ці властивості, ви можете кардинально змінити вигляд вебсторінки без зміни її вмісту.

Одна з найпотужніших функцій CSS — це можливість створювати адаптивний дизайн.

Це означає, що за допомогою CSS ви можете зробити свій вебсайт привабливим на будь-якому пристрої — чи то настільний комп’ютер, планшет або смартфон.

CSS дозволяє налаштовувати макети, розміри шрифтів та інші візуальні елементи залежно від розміру екрану пристрою, який переглядає сайт.

Спробуйте увімкнути інтерактивний редактор і змінити розмір вікна попереднього перегляду, щоб побачити, як макет адаптується до розміру екрану.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<header class="header">
    <h1>Welcome to My Responsive Site</h1>
    <p>This layout adapts to your screen size!</p>
</header>

<main class="content">
    <section class="box">Box 1</section>
    <section class="box">Box 2</section>
    <section class="box">Box 3</section>
</main>
```

```css
body {
  margin: 0;
  font-family: 'Segoe UI', sans-serif;
  background-color: #f9f9f9;
  color: #333;
}

.header {
  background-color: #4CAF50;
  color: white;
  padding: 2rem;
  text-align: center;
}

.content {
  display: flex;
  gap: 1rem;
  padding: 1rem;
  justify-content: center;
}

.box {
  flex: 1;
  min-width: 200px;
  background-color: #ddd;
  padding: 2rem;
  text-align: center;
  font-size: 1.2rem;
  border-radius: 8px;
  transition: background-color 0.3s;
}

.box:hover {
  background-color: #ccc;
}

/* Responsive: for tablets and below (≤ 768px) */
@media (max-width: 768px) {
  .content {
    flex-direction: column;
    align-items: center;
  }

  .box {
    width: 90%;
    font-size: 1.1rem;
  }
}

/* Responsive: for phones (≤ 480px) */
@media (max-width: 480px) {
  .header {
    padding: 1.5rem 1rem;
  }

  .header h1 {
    font-size: 1.5rem;
  }

  .header p {
    font-size: 1rem;
  }

  .box {
    font-size: 1rem;
    padding: 1.5rem;
  }
}
```

:::

Ще одна важлива функція CSS — це його каскадна природа, звідки й походить слово «каскадні» у назві.

Це означає, що стилі можуть успадковуватися та перевизначатися, що дозволяє створювати ієрархічну структуру стилізації. Ви дізнаєтеся більше про це у майбутніх уроках.

CSS також підтримує використання зовнішніх таблиць стилів.

Це означає, що всі правила стилів можна зберігати в окремому файлі, який потім можна підключати до кількох HTML-сторінок.

Ця функція значно покращує підтримку сайтів, особливо великих. Замість того, щоб змінювати стилі на кожній окремій сторінці, ви можете внести зміни в один CSS-файл, які вплинуть на всі підключені сторінки.

У світі веброзробки CSS відіграє життєво важливу роль у створенні візуально привабливих, адаптивних і зручних для користувачів сайтів.

Він дозволяє розробникам перетворювати прості HTML-документи на захопливі вебдосвіди, які привертають увагу користувачів і покращують їхню взаємодію з вебвмістом.
