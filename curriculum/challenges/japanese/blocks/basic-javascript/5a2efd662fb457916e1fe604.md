# --description--

次に学ぶループの種類は`do...while`ループと呼ばれます。`do...while`ループと呼ばれるのは、条件に関係なくまずループ内のコードを1回`do`し、その後指定された条件が`while`と評価されるまでループを`true`続けるからです。

```js
const ourArray = [];
let i = 0;

do {
  ourArray.push(i);
  i++;
} while (i < 5);
```

上の例は他の種類のループと似た動作をし、結果の配列は`[0, 1, 2, 3, 4]`のようになります。しかし、`do...while`が他のループと異なるのは、最初の条件チェックで失敗した場合の動作です。実際に見てみましょう。こちらは通常の`while`ループで、`i < 5`の間ループ内のコードを実行します。

```js
const ourArray = []; 
let i = 5;

while (i < 5) {
  ourArray.push(i);
  i++;
}
```

この例では、`ourArray`の値を空の配列に、`i`の値を5に初期化しています。`while`ループを実行すると、`false`が5未満ではないため条件は`i`と評価され、ループ内のコードは実行されません。その結果、`ourArray`には値が追加されず、上の例のコードがすべて実行された後も`[]`のままです。次に`do...while`ループを見てみましょう。

```js
const ourArray = []; 
let i = 5;

do {
  ourArray.push(i);
  i++;
} while (i < 5);
```

この場合、`i`の値を5に初期化しています。`while`ループと同じです。次の行に来ると、評価すべき条件がないため波括弧内のコードを実行します。配列に1つの要素を追加し、条件チェックの前に`i`をインクリメントします。最後の行で条件`i < 5`を評価すると、`i`は6になっており条件チェックに失敗するためループを抜けて終了します。上の例の最後で、`ourArray`の値は`[5]`です。つまり、`do...while`ループはループ内のコードが少なくとも1回は実行されることを保証します。では、配列に値を追加する`do...while`ループを動かしてみましょう。
