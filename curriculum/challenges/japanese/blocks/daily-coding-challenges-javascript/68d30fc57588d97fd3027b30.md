# --description--

1994年10月28日、Netscape Navigatorがリリースされ、何百万人もの人々が初期のウェブを探索するのを助けました。

Netscape Navigatorで実行したブラウザコマンドの配列が与えられたとき、以下のルールに従ってすべてのコマンドを実行した後の現在のページを返してください。

- 常に`"Home"`ページから開始し、このページはコマンド配列には含まれません。
- 有効なコマンドは次の通りです：
  - `"Visit Page"`: `"Page"`は訪問するページの名前です。例えば、`"Visit About"`は`"About"`ページに移動します。新しいページを訪問するときは、必ず前方の履歴を破棄してください。
  - `"Back"`: 履歴の前のページに戻るか、前のページがなければ現在のページにとどまります。
  - `"Forward"`: 履歴の先に進むか、先のページがなければ現在のページにとどまります。

例えば、`["Visit About Us", "Back", "Forward"]`が与えられたら、`"About Us"`を返してください。

```js
function navigate(commands) {
  const history = ["Home"];
  let currentPageIndex = 0;

  for (const command of commands) {
    if (command.startsWith("Visit")) {
      history.splice(currentPageIndex + 1);
      history.push(command.slice(6));
      currentPageIndex++;
    } else if(command === "Back" && currentPageIndex > 0) {
      currentPageIndex--;
    } else if (command === "Forward" && currentPageIndex < history.length - 1) {
      currentPageIndex++;
    }
  }

  return history[currentPageIndex];
}
```
