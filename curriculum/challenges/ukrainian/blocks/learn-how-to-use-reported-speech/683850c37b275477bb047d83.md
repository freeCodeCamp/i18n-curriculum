# --explanation--

`log` — це запис подій, дій або повідомлень, створений комп’ютерною системою або програмним забезпеченням. Наприклад:

`The system logs showed several errors before the crash.` — це означає, що записи містили інформацію про проблеми, які виникли.

`crash` — це коли комп’ютерна система або програма раптово перестає працювати правильно. Наприклад:

`The application crash caused all unsaved data to be lost.` — це означає, що програма несподівано зламалась, і дані не були збережені.
