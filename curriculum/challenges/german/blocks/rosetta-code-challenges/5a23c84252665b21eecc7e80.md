# --description--

Gray-Code ist eine Form der binären Codierung, bei der sich Übergänge zwischen aufeinanderfolgenden Zahlen nur in einem Bit unterscheiden.

Diese Codierung ist nützlich, um Hardware-Datenkonflikte bei Werten zu reduzieren, die sich schnell ändern und/oder als Eingaben an langsamere Hardware angeschlossen sind.

Sie ist auch hilfreich, um Eingaben für Karnaugh-Karten von links nach rechts oder von oben nach unten zu erzeugen.

# --instructions--

Erstellen Sie eine Funktion, um eine Zahl in Gray-Code zu codieren und eine Zahl aus Gray-Code zu decodieren. Die Funktion soll zwei Parameter haben.

Der erste Parameter ist ein Boolean. Die Funktion soll für true codieren und für false decodieren. Der zweite Parameter ist die Zahl, die codiert oder decodiert werden soll.

Geben Sie die normalen binären Darstellungen, die Gray-Code-Darstellungen und die decodierten Gray-Code-Werte für alle 5-Bit-binären Zahlen (0–31 inklusive, führende Nullen sind nicht notwendig) aus.

Es gibt viele mögliche Gray-Codes. Der folgende codiert den sogenannten „binary reflected Gray code“.

Codierung (MSB ist Bit 0, b ist binär, g ist Gray-Code):

<pre>if b[i-1] = 1
  g[i] = not b[i]
else
  g[i] = b[i]
</pre>

Oder:

<pre>g = b xor (b logisch um 1 nach rechts verschoben)
</pre>

Decodierung (MSB ist Bit 0, b ist binär, g ist Gray-Code):

<pre>b[0] = g[0]<br>
für andere Bits:
b[i] = g[i] xor b[i-1]
</pre>

function gray(enc, number){
  if(enc){
      return number ^ (number >> 1);
  }else{
      let n = number;

      while (number >>= 1) {
          n ^= number;
      }
      return n;
  }
}
