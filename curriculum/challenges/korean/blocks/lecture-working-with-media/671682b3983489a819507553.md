# --interactive--

이전 수업에서 `iframe` 요소를 처음 접했습니다. 이번 수업에서는 `iframe` 요소를 다루는 방법을 더 배웁니다. 이 요소는 인라인 프레임을 의미합니다. HTML 페이지 내에 다른 HTML 콘텐츠를 직접 임베드하는 데 사용하는 인라인 요소입니다. 그 HTML 콘텐츠는 동영상, 지도, 다른 HTML 요소 또는 다른 웹 페이지일 수 있습니다.

다음은 YouTube에서 인기 있는 freeCodeCamp 강의를 임베드한 예입니다. 다른 동영상을 보려면 인터랙티브 편집기를 활성화하고 `src` 값을 원하는 동영상으로 바꾸세요.

:::interactive_editor

```html
<iframe
  width="400"
  height="400"
  src="https://www.youtube.com/embed/PkZNo7MFNFg?si=-UBVIUNM3csdeiWF"
  title="Learn JavaScript - Full Course for Beginners (YouTube video)"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
  referrerpolicy="strict-origin-when-cross-origin"
  allowfullscreen
></iframe>
```

:::

`src` 속성은 임베드하려는 페이지의 URL을 지정합니다. `width` 속성은 `iframe`의 너비를 지정합니다. `height` 속성은 `iframe`의 높이를 지정합니다. `allowfullscreen` 속성은 사용자가 `iframe`를 전체 화면 모드로 표시할 수 있게 합니다. 접근성 측면에서 `title` 속성을 `iframe`에 지정하는 것도 좋은 습관입니다.

반면 `allow` 속성은 iframe이 할 수 있는 것과 할 수 없는 것을 정의할 수 있게 합니다. 이를 허용 목록이라고 합니다. 위 예제에서 `clipboard-write`을 추가하면 임베드된 페이지가 클립보드에 항목을 쓸 수 있게 허용합니다. 허용 목록의 항목은 세미콜론이나 공백으로 구분할 수 있으며, 두 가지를 함께 사용할 수도 있습니다.

동영상은 어디에서든 올 수 있다는 점을 기억하세요. YouTube나 Vimeo 같은 동영상 서비스에서만 올 필요는 없습니다.

지도, 다른 웹 페이지, 또는 직접 작성한 HTML도 `iframe` 요소 내에 임베드할 수 있다는 점도 잊지 마세요. 다음은 임베드된 지도 예시입니다.

지도를 확대하거나 축소하며 상호작용해 보세요.

:::interactive_editor

```html
<h1>A Map from Openstreetmap.org Embedded with the iframe Element</h1>

<iframe
  width="425"
  height="350"
  src="https://www.openstreetmap.org/export/embed.html?bbox=3.006134033203125%2C6.150112578753815%2C3.6357879638671875%2C6.749850810550778&amp;layer=mapnik"
  title="Map of Lagos area, Nigeria"
  style="border: 1px solid black"
>
</iframe>
<br />
<small>
  <a href="https://www.openstreetmap.org/#map=11/6.4501/3.3210">
    View Larger Map
  </a>
</small>
```

:::

`iframe` 요소 내에 직접 HTML을 임베드하려면 `srcdoc` 대신 `src` 속성을 사용해야 합니다.
