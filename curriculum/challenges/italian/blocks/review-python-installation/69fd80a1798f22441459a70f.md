## Python nel tuo ambiente locale

- **Installazione**: Il modo migliore per installare Python su Windows, Mac e Linux è scaricare l'installer dal sito ufficiale di Python (`https://www.python.org/`).
- **Terminale**: Un'interfaccia testuale che ti permette di interagire con il computer digitando comandi. Ogni sistema operativo ha un'app terminale predefinita. Su macOS puoi usare l'app Terminale. Su Windows puoi usare Prompt dei comandi o PowerShell. Su Linux ogni ambiente desktop ha la sua app terminale predefinita, come GNOME Terminal o Konsole.
- **IDE**: IDE sta per ambiente di sviluppo integrato e include funzionalità come un editor di codice, strumenti di test, un terminale e altro.
- **Editor di codice e IDE popolari per Python**: VS Code, PyCharm e Spyder
- **Eseguire codice localmente**: Per eseguire codice Python hai diverse opzioni. La prima è cliccare sul pulsante di esecuzione in VS Code, che assomiglia a un pulsante play e si trova nell'angolo in alto a destra. Questo aprirà un terminale ed eseguirà il tuo script Python lì.

Un'altra opzione è aprire un terminale ed eseguire manualmente il programma. Ecco un esempio per eseguire un programma chiamato `main.py`:

```bash
python main.py
```

Devi eseguire questo comando dalla cartella in cui è salvato `main.py`. Per esempio, se `main.py` si trova dentro una cartella chiamata `python-projects`, usa prima questo comando nel terminale:

```bash
cd python-projects
```

Poi esegui:

```bash
python main.py
```

## Usare la shell interattiva di Python

- **Definizione**: Una shell interattiva è un programma che ti permette di digitare comandi uno alla volta e vedere i risultati. Apri un'app terminale e digita `python` e premi `Enter`. Questo avvierà una shell interattiva di Python. Il simbolo `>>>` indica che Python aspetta che tu digiti un comando. Prova a stampare `"Hello, world!"` nel terminale:

```bash
print("Hello, world!")
```

Dovresti vedere il testo apparire così:

```bash
>>> print("Hello, world!")
Hello, world!
```

Dopo che il testo è stato stampato, Python torna a questo:

```bash
>>>
```

Questo significa che puoi digitare un altro comando.

- **Ciclo Read, Evaluate, Print, Loop (REPL)**: Ogni volta che digiti comandi, l'interprete leggerà l'input, lo valuterà, stamperà il risultato e tornerà a mostrare il `>>>` così puoi digitare altri comandi.
