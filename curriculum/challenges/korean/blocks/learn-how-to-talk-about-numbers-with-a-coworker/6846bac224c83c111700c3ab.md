# --explanation--

**기본 수(cardinal numbers)**는 "몇 개"인지를 알려줍니다 — 예를 들어 `one`, `two`, `three`처럼요.

문장 `We need three new developers for the next sprint.`에서 `three`는 **기본 수**입니다.

다른 모든 문장은 순서나 위치를 나타내는 잘못된 형태를 사용했습니다.
