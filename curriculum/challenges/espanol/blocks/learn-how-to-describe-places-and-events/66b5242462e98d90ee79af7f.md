# --explanation--

`Might` se usa para expresar posibilidad o incertidumbre. Sugiere que algo es posible, pero no seguro. Por ejemplo:

- `The system might crash if too many users log in at the same time.` - Existe la posibilidad de un fallo del sistema, pero no es seguro.

- `He might finish the coding task by the end of the day.` - Él podría terminar la tarea, pero no es definitivo.
