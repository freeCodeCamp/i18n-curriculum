# --explanation--

`Fail`는 무언가가 제대로 작동하지 않는다는 뜻입니다. 정상적인 작동에 영향을 주는 문제를 가리킵니다. 예를 들어:

- `The server is failing because of high traffic.` - 너무 많은 사용자가 동시에 접속해서 서버가 다운되었습니다.

- `The app failed during the beta testing phase.` - 애플리케이션이 사용자 테스트에서 제대로 작동하지 않았습니다.
