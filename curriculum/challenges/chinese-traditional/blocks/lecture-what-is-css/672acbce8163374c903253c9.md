# --description--

<meta viewport> 元素是在響應式網頁設計中一個關鍵的組件。

這是一個特殊的 HTML meta 元素，提供瀏覽器如何控制頁面尺寸和縮放的指示，特別是在行動電話和平板裝置上。

讓我們來看看 meta viewport 元素的基本語法：

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

此元素通常放置在你的 HTML `head` 區段中。但該元素的每個零件代表什麼意思？

`width=device-width` 這部分告訴瀏覽器將頁面的寬度設定為與裝置的螢幕寬度相符。這對於建立能適應不同螢幕尺寸的響應式版面配置至關重要。

`initial-scale=1.0` 設定頁面首次載入時的初始縮放層次。值為 1.0 表示頁面以 `100%` 縮放顯示，沒有任何縮放。

透過使用 meta viewport 元素，你確保你的網頁能在行動裝置上正確顯示。

如果沒有它，行動瀏覽器通常會以桌面螢幕寬度來呈現頁面，然後再縮放，這可能導致文字過小且難以閱讀，造成不佳的使用者體驗。

<meta viewport> 元素也允許你控制使用者是否可以放大或縮小你的網頁。

雖然可以使用 `user-scalable=no` 屬性來停用縮放，但通常建議為了無障礙性而避免這麼做。

許多使用者依賴縮放功能以提升可讀性，特別是視覺障礙者。

以下是你不該做的範例：

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0, user-scalable=no">
```

相反地，設計你的网站以在不同縮放層級下具有響應性和可讀性會更好，確保所有使用者都能舒適地存取你的內容。

<meta viewport> 元素在建立行動友善網站時扮演關鍵角色。

它確保你精心設計的響應式設計能如預期般在各種裝置上顯示，為所有訪問你網站的使用者提供更佳的使用體驗。
