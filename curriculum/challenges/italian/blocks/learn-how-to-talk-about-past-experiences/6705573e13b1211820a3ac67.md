# --explanation--

La forma negativa del verbo `to be` nel tempo `Simple Past` è `was not` o `wasn't` per soggetti singolari, e `were not` o `weren't` per soggetti plurali. Si usa per descrivere qualcosa che non è successo o non era vero. Per esempio:  

- Singolare: `The code wasn't running as expected.` - Si stava verificando un problema con il codice.

- Plurale: `The changes weren't implemented correctly.` - Gli errori nell'implementazione hanno causato i bug.

Nella frase di James, usa `wasn't` per spiegare che il sistema non riusciva a collegarsi al database. Le frasi negative con `wasn't` o `weren't` indicano ciò che non è accaduto o non ha funzionato come ci si aspettava.
