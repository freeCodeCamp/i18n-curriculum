# --description--

`Encourage` inamaanisha kumpa mtu msaada au kujiamini kufanya jambo fulani. Kwa mfano, wakati rafiki yako anasema, `You can solve this coding problem!` wanakutia moyo.

`Discourage` ni kinyume chake; inamaanisha kumfanya mtu ajisikie hana kujiamini au asipende kufanya jambo fulani. Ikiwa rafiki huyo atasema, `That programming language is too hard for beginners`, inaweza kukufanya usihisi hamu ya kujifunza hilo.

`Discouraging` inaelezea kitu kinachokufanya ujisikie kukata tamaa. Kama vile, `Finding a bug in your code after hours of work can be discouraging.`

`Encouraging` ni pale kitu kinapokupa ujasiri au matumaini. Kwa mfano, `Getting your first program to run is very encouraging.`
