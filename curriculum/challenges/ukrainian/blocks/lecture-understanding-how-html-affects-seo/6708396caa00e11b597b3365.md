# --description--

Протокол open graph дає змогу контролювати, як вміст вашого вебсайту відображається на різних соціальних мережах, таких як Facebook, LinkedIn та багатьох інших. Встановлюючи ці властивості open graph, ви можете заохотити користувачів натиснути на ваш вміст і взаємодіяти з ним. Ви можете встановити ці властивості через набір `meta` елементів у вашому HTML `head` розділі.

Першою важливою OG властивістю, яку слід включити, буде `title`. Ось приклад встановлення OG `title` для головної сторінки freeCodeCamp:

```html
<meta content="freeCodeCamp.org" property="og:title" />
```

Для атрибута `property` потрібно вказати, що він є `og:title`. Атрибут `content` — це місце, де ви напишете заголовок, який хочете відобразити для соціальних мереж.

Наступною важливою OG властивістю буде `type`. Ось приклад використання OG `type` для головної сторінки freeCodeCamp:

```html
<meta property="og:type" content="website" />
```

Властивість `type` використовується для представлення типу вмісту, який поширюється у соціальних мережах. Прикладами такого вмісту є статті, вебсайти, відео або музика.

Третьою важливою OG властивістю буде `image`. Ось приклад встановлення OG `image` для головної сторінки freeCodeCamp:

```html
<meta
  content="https://cdn.freecodecamp.org/platform/universal/fcc_meta_1920X1080-indigo.png"
  property="og:image"
/>
```

У цьому прикладі зображення open graph вказує на логотип freeCodeCamp. Всі ці зображення мають бути високої якості з хорошими розмірами та співвідношеннями. Більшість соціальних мереж мають вимоги до зображень, які допомагають переконатися, що ваш вміст добре відображається на їхніх сайтах. Наприклад, на сторінці документації developers.facebook.com зазначено:

"використовуйте зображення розміром щонайменше 1200 на 630 пікселів для найкращого відображення на пристроях з високою роздільністю. Мінімум — це зображення розміром 600 на 315 пікселів для відображення постів із посиланнями з більшими зображеннями."

Четвертою важливою OG властивістю буде `url`. Ось приклад встановлення OG `url` для головної сторінки freeCodeCamp:

```html
<meta property="og:url" content="https://www.freecodecamp.org" />
```

Існує багато інших OG властивостей, які ви можете встановити, як-от `description`, `audio`, `video` і `locale`. Однак open graph `url`, `image`, `type` і `title` — це найважливіші, які слід включити.

Отже, як ці властивості open graph впливають на оптимізацію для пошукових систем? Коли ваш вміст поширюється у соціальних мережах, добре продумані OG властивості можуть покращити вигляд вашого вмісту у стрічках користувачів. Це може призвести до вищих показників кліків, що може сигналізувати пошуковим системам про релевантність і привабливість вашого вмісту.
