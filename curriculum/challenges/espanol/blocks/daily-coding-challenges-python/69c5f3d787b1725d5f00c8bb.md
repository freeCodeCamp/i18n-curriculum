# --description--

Dada una cadena, devuelve una versión comprimida de la cadena usando las siguientes reglas:

- La primera aparición de una palabra permanece sin cambios.
- Las apariciones posteriores se reemplazan con la posición de la primera aparición, donde la primera palabra está en la posición 1.
- Las palabras están separadas por un solo espacio.

Por ejemplo, dado `"practice makes perfect and perfect practice makes perfect"`, devuelve `"practice makes perfect and 3 1 2 3"`.

```py
def compress(s):
    words = s.split(' ')
    seen = {}
    result = []
    for i, word in enumerate(words):
        if word in seen:
            result.append(str(seen[word]))
        else:
            seen[word] = i + 1
            result.append(word)
    return ' '.join(result)
```
