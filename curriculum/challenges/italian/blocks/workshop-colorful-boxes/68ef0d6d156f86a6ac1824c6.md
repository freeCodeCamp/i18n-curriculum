# --description--

Ora userai la proprietà `align-content`. Questa proprietà controlla l’allineamento degli elementi lungo l’asse trasversale. Mentre il valore predefinito di `align-content` è `stretch`, le opzioni più rilevanti che puoi usare per `align-content` sono: `space-between`, `start`, `center`, `end`, `space-around` e `space-evenly`.

Aggiungi la proprietà `align-content` con il valore `space-between` per vedere come si allineeranno i riquadri nell’elemento `.flex-container`.

# --hints--

Il tuo selettore `.flex-container` dovrebbe avere la proprietà `align-content` con il valore `space-between`.

```js
assert.equal(new __helpers.CSSHelp(document).getStyle('.flex-container')?.getPropVal('align-content'), 'space-between');
```
