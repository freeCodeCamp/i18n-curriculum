# --description--

Un `plugin` es un componente que añade una función específica a un programa de computadora existente. Cuando un programa soporta `plugins`, permite la personalización.  
En el contexto de un IDE, los `plugins` pueden proporcionar funcionalidad adicional, como nuevas herramientas para codificar, depurar o mejorar la interfaz de usuario.

Por ejemplo, un formateador de código en un IDE a menudo es un `plugin`.

## --answers--
