# --interactive--

Al diseñar sitios web, una de las formas más versátiles e intuitivas de trabajar con color es usando el modelo de color HSL. HSL significa matiz, saturación y luminosidad — tres componentes clave que definen un color.

El modelo de color HSL representa los colores de una manera más alineada con cómo los humanos perciben el color. En esta lección, exploraremos qué es el modelo de color HSL y cómo se usa la función `hsl()` en CSS.

El matiz es el tipo de color, representado como un ángulo en la rueda de colores, que va desde `0` hasta `360` grados. Por ejemplo, `0` grados representa el rojo, `120` grados es verde y `240` grados es azul. Al ajustar el matiz, se desplaza a través de diferentes colores en el espectro.

La saturación se refiere a la intensidad o pureza del color. Se mide como un porcentaje desde `0%` (un color completamente desaturado, en escala de grises) hasta `100%` (la forma más vibrante de ese color). Un color con saturación `100%` es totalmente vívido, mientras que un color con saturación `0%` aparece como un tono de gris.

La luminosidad determina qué tan claro u oscuro es el color, también medido como un porcentaje. Un valor de luminosidad `0%` producirá negro, `50%` dará el tono normal del matiz y `100%` resultará en blanco.

En CSS, la función `hsl()` se usa para definir colores usando el modelo de color HSL. La sintaxis moderna usa valores separados por espacios:

```css
element {
  color: hsl(hue saturation lightness);
}
```

También puedes encontrar la sintaxis antigua con valores separados por comas en código existente, que aún es compatible con los navegadores:

```css
element {
  color: hsl(hue, saturation, lightness);
}
```

Vamos a desglosar esto con un ejemplo:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
body {
  background-color: hsl(0 0% 1% / 1);
}

p {
  color: hsl(120 100% 50%);
}
```

:::

En este caso, el matiz es `120` grados, que corresponde al verde. La saturación es `100%`, por lo que el verde será totalmente vívido. La luminosidad es `50%`, así que está en su tono normal — ni muy oscuro ni muy claro. Como resultado, el color del texto del párrafo será un verde brillante.

Una de las principales ventajas del modelo de color HSL es su naturaleza intuitiva. Facilita ajustar la viveza o luminosidad de un color modificando los valores de saturación y luminosidad sin tener que alterar el color base (matiz).

Por ejemplo, si quieres crear diferentes tonos o tintes del mismo color, simplemente puedes ajustar el valor de luminosidad.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="light">This is a light blue div.</div>
<div class="dark">This is a dark blue div.</div>
```

```css
div.light {
  background-color: hsl(240 100% 80%);
}

div.dark {
  background-color: hsl(240 100% 20%);
  color: hsl(0 0% 100%);
}
```

:::

Aquí, ambos elementos `div` usan el mismo matiz (`240` grados, que es azul), pero uno tiene una luminosidad de `80%` (un tono más claro de azul) y el otro una luminosidad de `20%` (un tono más oscuro de azul).

Al igual que el modelo RGB, la función `hsl()` también admite un valor alfa opcional para controlar la transparencia. Se añade después de un separador `/` como cuarto parámetro. Aquí está la sintaxis básica:

```css
element {
  background-color: hsl(hue saturation lightness / alpha);
}
```

Veamos un ejemplo:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: hsl(0 100% 50% / 0.5);
}
```

:::

Este código daría al `div` un fondo rojo semitransparente, donde el matiz está en `0` grados (rojo), la saturación es `100%`, la luminosidad es `50%` y el alfa es `0.5` (50 % de opacidad). También puedes especificar el alfa como porcentaje — por ejemplo, `hsl(0 100% 50% / 50%)` equivale a `hsl(0 100% 50% / 0.5)`.

También verás comúnmente la función `hsla()` en CSS existente. Esta fue la forma original de incluir transparencia en colores HSL, usando valores separados por comas:

```css
element {
  background-color: hsla(hue, saturation, lightness, alpha);
}
```

Por ejemplo, `hsla(0, 100%, 50%, 0.5)` es el equivalente antiguo de `hsl(0 100% 50% / 0.5)`. Ambos son compatibles con los navegadores, pero ahora se prefiere la sintaxis moderna `hsl()` con el separador `/`.

El modelo de color HSL es especialmente útil cuando necesitas crear esquemas de color y ajustar tonos o tintes fácilmente.

Porque HSL separa el matiz de la saturación y la luminosidad, puedes mantener un tema de color consistente mientras ajustas la luminosidad para sombras o luces. Este nivel de flexibilidad hace que HSL sea favorito tanto para diseñadores como para desarrolladores.

Además, HSL hace que los valores de color sean más legibles y comprensibles en comparación con otros métodos como los códigos hexadecimales o RGB. Puedes ver más claramente la intención detrás del color y ajustar en consecuencia, especialmente al experimentar con diferentes diseños.

Con el modelo de color HSL, obtienes más flexibilidad para controlar tonos, saturación y luminosidad, convirtiéndolo en una herramienta poderosa para crear diseños ricos y dinámicos en CSS.
