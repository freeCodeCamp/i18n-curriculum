# --description--

Para `monitor` significa ver y comprobar algo durante un período de tiempo para asegurarse de que está funcionando correctamente o de que está seguro. Por ejemplo, `The security team monitors the cameras to keep the building safe.`

`Access control` se refiere a la forma en que la entrada a un lugar o el uso de un recurso está restringido y regulado, para asegurarse de que sólo las personas autorizadas pueden entrar en un lugar o usar algo. Por ejemplo, `The company uses access controls to restrict entry to the server room`
