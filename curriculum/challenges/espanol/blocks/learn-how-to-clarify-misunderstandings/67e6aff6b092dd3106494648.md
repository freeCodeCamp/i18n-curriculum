# --explanation--

`Encapsulation` es un principio clave en la programación orientada a objetos. Significa agrupar datos (variables) y métodos (funciones) que operan sobre los datos dentro de una sola unidad, a menudo una clase.

Este concepto ayuda a ocultar los detalles internos de cómo se gestionan los datos, exponiendo solo lo necesario al mundo exterior. Esto mejora la seguridad y hace que el código sea más organizado y fácil de mantener.
