# --interactive--

Absolute Positionierung ermöglicht es Ihnen, ein Element aus dem normalen Dokumentenfluss zu nehmen, sodass es unabhängig von anderen Elementen agiert. Wenn ein Element absolut positioniert ist, wird es in einer eigenen Ebene platziert, völlig getrennt von allem anderen im Layout. Das ist nützlich, um schwebende UI-Funktionen wie Modale, Tooltips oder Dropdown-Menüs zu erstellen, die andere Elemente auf der Seite überlappen können.

Standardmäßig werden absolut positionierte Elemente relativ zum nächsten positionierten Vorfahren platziert. Wenn kein positionierter Vorfahre gefunden wird, wird das Element relativ zum initialen Containing Block positioniert, der normalerweise der Viewport des Browsers ist. Sie können das absolut positionierte Element mit den `top`, `bottom`, `left` und `right`-Eigenschaften verschieben, um anzugeben, wie weit es von den Rändern seines Containing Elements entfernt sein soll.

Zum Beispiel verschieben `top: 30px` und `left: 30px` das Element `30px` vom oberen und linken Rand des Containing Blocks weg. Hier ist ein Beispiel, wie Sie absolute Positionierung in CSS auf ein Element anwenden:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="positioned">Absolutely Positioned</div>
```

```css
body {
  background-color: #eeeeee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

Wenn dieser Code angewendet wird, wird das Element aus dem normalen Dokumentenfluss entfernt und `30px` vom oberen und linken Rand seines Containing Blocks platziert.

Absolute Positionierung ist besonders hilfreich, um Elemente zu erstellen, die über dem restlichen Inhalt schweben sollen, wie Overlays oder Modalfenster. Da das Element jedoch aus dem Dokumentenfluss entfernt wird, kann es auch Lücken erzeugen oder dazu führen, dass andere Elemente zusammenrücken, wenn es nicht richtig gehandhabt wird.
