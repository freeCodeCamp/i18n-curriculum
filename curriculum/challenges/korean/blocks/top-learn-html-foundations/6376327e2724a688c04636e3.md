# --description--

HTML과 CSS는 인터넷에서 무언가를 볼 때 보이는 모든 것을 만드는 데 함께 작동하는 두 가지 언어입니다. HTML은 웹페이지를 구성하는 원시 데이터입니다. 모든 텍스트, 링크, 카드, 목록, 버튼은 HTML로 만들어집니다. CSS는 그런 단순한 요소에 스타일을 더합니다. HTML은 웹페이지에 정보를 넣고, CSS는 그 정보를 배치하고, 색을 입히고, 폰트를 바꾸며 멋지게 보이도록 만듭니다!
