# --description--

O tempo `present simple` é usado para falar sobre ações regulares ou rotinas.

Lembre-se, para formar o `present simple`, você usa o verbo base para `I`, `you`, `we`, `they` (`I code`, `they test`) e adiciona `-s` ou `-es` para `he`, `she`, `it` (`he codes`, `she tests`).

Em um contexto de trabalho, descrever tarefas diárias é tipicamente feito usando o tempo `present simple`, pois são ações habituais.
