# --description--

In qualsiasi linguaggio di programmazione, devi confrontare i valori per prendere decisioni. Per esempio, se il tempo è soleggiato, uscirai, altrimenti resterai a casa. Un altro esempio è verificare se un utente ha abbastanza punti per salire di livello in un gioco.

L'istruzione condizionale `if-else` viene usata per prendere decisioni in JavaScript. Spesso si usa con operatori di confronto per confrontare valori e decidere in base ai risultati.

Un esempio di istruzione if è:

```javascript
let x = 5;

if (x > 3) {
  console.log("x is greater than 3");
} else {
  console.log("x is less than or equal to 3");
}
```

In questo esempio, la variabile `x` viene confrontata con il numero `3` usando l'operatore `>`. Se `x` è maggiore di `3`, verrà eseguito il primo blocco di codice. Altrimenti, verrà eseguito il secondo blocco di codice.
