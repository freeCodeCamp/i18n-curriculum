# --explanation--

`Phishing`는 사기꾼이 비밀번호나 신용카드 정보 같은 개인 정보를 공유하도록 속이기 위해 주로 가짜 이메일이나 웹사이트를 통해 사용하는 방법입니다. 예를 들어:

`The phishing email looked like it came from our bank.` - 이 말은 이메일이 가짜였지만 개인 정보를 훔치기 위해 만들어졌다는 뜻입니다.

`Attack`는 무언가를 해치거나 손상시키려는 행위를 의미합니다. 신체적, 언어적, 또는 사이버 보안 분야에서 모두 해당됩니다. 예를 들어:

- `The attack on the server caused it to shut down.` - 이 말은 의도적인 행위가 서버를 손상시켰다는 뜻입니다.
  
- `She defended herself during the verbal attack.` - 이 말은 누군가가 그녀에게 해로운 말을 했다는 뜻입니다.
