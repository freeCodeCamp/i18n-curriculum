# --description--


La propiedad `flex-wrap` determina cómo se comportan tus elementos flexibles cuando el contenedor es demasiado pequeño. Establecerlo en `wrap` permitirá a los elementos pasar a la siguiente fila o columna. `nowrap` (por defecto) evitará que los elementos se envuelvan y los disminuirá si es necesario.


Haz que tus elementos flexibles se envuelvan a la siguiente fila cuando se queden sin espacio.

# --hints--

Tu selector `.gallery` debe tener la propiedad `flex-wrap` configurada como `wrap` en el valor.

```js
assert.equal(new __helpers.CSSHelp(document).getStyle('.gallery')?.flexWrap, 'wrap');
```
