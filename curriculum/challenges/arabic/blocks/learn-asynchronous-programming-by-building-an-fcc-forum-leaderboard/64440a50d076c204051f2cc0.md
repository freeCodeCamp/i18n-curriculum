# --description--

تحتاج إلى دالة لتحويل عدد المشاهدات إلى صيغة أكثر قابلية للقراءة. على سبيل المثال، إذا كان عدد المشاهدات `1000`، يجب أن يعرض كـ `1k` وإذا كان عدد المشاهدات `100,000` يجب أن يعرض كـ `100k`.

أنشئ دالة `viewCount` بمعلمة `views`. إذا كانت `views` أكبر من أو تساوي `1000`، أرجع سلسلة نصية بقيمة `views` مقسومة على `1000` مع إضافة الحرف `k` في نهايتها. تأكد من تقريب `views / 1000` إلى أقرب عدد صحيح أقل.

وإلا، أرجع قيمة `views`.

على سبيل المثال، إذا كانت `views` تساوي `1000` يجب أن تكون قيمة الإرجاع هي السلسلة النصية `"1k"`.

  const currentTime = new Date();
  const lastPost = new Date(time);

  const timeDifference = currentTime - lastPost;

  const minutesAgo = Math.floor(timeDifference / msPerMinute);

  const daysAgo = Math.floor(hoursAgo / 24);

    return `${minutesAgo}m ago`;
  }

  if (hoursAgo < 24) {
    return `${hoursAgo}h ago`;
  }

  return `${daysAgo}d ago`;
};
