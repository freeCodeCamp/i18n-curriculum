# --description--

文字列を扱うとき、異なるテキストの断片を組み合わせる操作はよく行う一般的な作業です。

Pythonでは、プラス (`+`) 演算子を使って複数の文字列を結合できます。この処理は**文字列の連結**と呼ばれます。プラス演算子で2つの文字列を連結する方法は次の通りです。

```python
my_str_1 = 'Hello'
my_str_2 = "World"

str_plus_str = my_str_1 + ' ' + my_str_2
print(str_plus_str) # Hello World
```

ただし、これは文字列同士でのみ機能します。文字列と数値を連結しようとすると、`TypeError`が発生します。

```python
name = 'John Doe'
age = 26

name_and_age = name + age
print(name_and_age) # TypeError: can only concatenate str (not "int") to str
```

これは、Pythonが連結時に整数などの他のデータ型を自動的に文字列に変換しないためです。連結する前にすべての要素が文字列である必要があります。これを解決するには、組み込みの`str()`関数を使って数値を文字列に変換できます。この関数は元のオブジェクトを変更せずに、その文字列表現を返します。

```python
name = 'John Doe'
age = 26

name_and_age = name + str(age)
print(name_and_age) # John Doe26
```

連結には複合代入演算子も使えます。これはプラスとイコールの記号 (`+=`) で表され、連結と代入を一度に行います。使い方は次の通りです。

```py
name = 'John Doe'
age = 26

name_and_age = name  # Start with the name
name_and_age += str(age)  # Append the age as string

print(name_and_age)  # John Doe26
```

変数や式を文字列に挿入する処理は**文字列の補間**と呼ばれます。Pythonには**f-strings**（フォーマット済み文字列リテラルの略）という文字列の種類があり、簡潔で読みやすい構文で補間を扱えます。

f-stringsは引用符の前に`f`（小文字または大文字）を付け、波括弧（`{}`）で囲まれた置換フィールド内に変数や式を埋め込めます。例は次の通りです。

```python
name = 'John Doe'
age = 26
name_and_age = f'My name is {name} and I am {age} years old'
print(name_and_age) # My name is John Doe and I am 26 years old

num1 = 5
num2 = 10
print(f'The sum of {num1} and {num2} is {num1 + num2}') # The sum of 5 and 10 is 15
```

上の例では、`str()`関数で非文字列型を変換する必要がないことに注目してください。`age`、`num1`、`num2`変数の値は補間の過程で内部的に文字列に変換されます。
