# --description--

React 中的條件式算繪允許你創建動態使用者介面。它用於根據應用程式中的特定條件或狀態顯示不同的內容。

使用條件呈現的最常見方法包括使用 `if` 述語、三元 (`?:`) 運算子和邏輯 AND (`&&`) 運算子。

最簡單的條件呈現形式使用 `if` 述語。以下是一個範例：

```jsx
function Greeting({ isLoggedIn }) {
  if (isLoggedIn) {
    return <h1>Welcome back!</h1>;
  }
  return <h1>Please sign in</h1>;
}
```

在此範例中，`Greeting` 組件會檢查 `isLoggedIn` prop。如果它是 `true`，則傳回歡迎訊息，否則會提示使用者登入。

以下是在父層 `App` 組件內使用 `Greeting` 組件的範例：

```jsx
function App() {
  return (
    <div className="App">
      <Greeting isLoggedIn={false} />
    </div>
  );
}
```

對於較簡單的條件，三元運算子（`?:`）經常直接用在 JSX 中。它允許行內條件呈現，能讓你的程式碼更簡潔：

```jsx
function Greeting({ isLoggedIn }) {
  return <h1>{isLoggedIn ? "Welcome back!" : "Please sign in."}</h1>;
}
```

這段程式碼達成與前一個範例相同的結果，但以更精簡的形式呈現。三元運算子檢查 `isLoggedIn` 並呈現適當的訊息。

另一個常見的條件呈現樣式是使用邏輯 AND (`&&`) 運算子。當你想根據條件呈現某些東西或不呈現任何東西時，這特別有用：

```jsx
function Notification({ message }) {
  return (
    <div>
      {message && <p>{message}</p>}
    </div>
  );
}
```

在此範例中，帶有訊息的段落元素只有在 `message` prop 為真值時才會被呈現。如果 `message` 為假值－意指它是空字串、`null` 或 `undefined`，則不會有任何內容呈現在螢幕上。

透過精通這些條件式呈現的技巧，你可以建置更具互動性且使用者友善的應用程式，能夠適應變化的資料和使用者互動。
