# --explanation--

Nelle `Simple Past`, le domande possono iniziare con una parola interrogativa (ad esempio, `what`, `where`, `how`, `who`) usando la seguente struttura:

**Parola interrogativa + `did` + soggetto + verbo base.**

La domanda di Lisa, `What did you find there?`, usa `what` per chiedere informazioni specifiche sul risultato delle azioni di James. Altri esempi includono:

- `Where did you go yesterday?` - Stai chiedendo di un luogo.

- `How did they solve the problem?` - Stai chiedendo di un metodo.

- `Who did you call last night?` - Stai chiedendo di una persona.

Nel dialogo, Lisa chiede della scoperta di James durante la ricerca del bug.
