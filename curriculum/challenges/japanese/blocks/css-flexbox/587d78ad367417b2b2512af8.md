# --description--

`align-items` プロパティは `justify-content` に似ています。`justify-content` プロパティがフレックスアイテムを主軸に沿って揃えていたことを思い出してください。行の場合、主軸は水平線で、列の場合は垂直線です。

フレックスコンテナには主軸の反対方向にある **交差軸** もあります。行の場合、交差軸は垂直で、列の場合は交差軸は水平です。

CSS は交差軸に沿ってフレックスアイテムを揃えるために `align-items` プロパティを提供しています。行の場合、これはコンテナ内でアイテム全体を上下に押し出す方法をCSSに指示します。列の場合は、コンテナ内でアイテム全体を左右に押し出す方法を指示します。

`align-items` に使えるさまざまな値は次の通りです：

<ul><li><code>flex-start</code>: アイテムをフレックスコンテナの開始位置に揃えます。行の場合はコンテナの上端に揃え、列の場合はコンテナの左端に揃えます。</li><li><code>flex-end</code>: アイテムをフレックスコンテナの終了位置に揃えます。行の場合はコンテナの下端に揃え、列の場合はコンテナの右端に揃えます。</li><li><code>center</code>: アイテムを中央に揃えます。行の場合は垂直方向に中央揃え（アイテムの上下に均等なスペース）、列の場合は水平方向に中央揃え（アイテムの左右に均等なスペース）になります。</li><li><code>stretch</code>: アイテムを伸ばしてフレックスコンテナを埋めます。例えば、行のアイテムはコンテナの上下いっぱいに伸ばされます。<code>align-items</code> の値が指定されていない場合のデフォルト値です。</li><li><code>baseline</code>: アイテムをベースラインに揃えます。ベースラインはテキストの概念で、文字が乗る線のことです。</li></ul>

# --instructions--

このプロパティの動作を示す例です。`align-items` 要素に CSS プロパティ `#box-container` を追加し、値を `center` にしてください。

**ボーナス**  
コードエディターで `align-items` プロパティの他のオプションを試して違いを確認してください。ただし、このチャレンジをクリアできるのは `center` の値だけです。

# --hints--

`#box-container` 要素には `align-items` プロパティを `center` の値で設定してください。

```js
const boxContainer = document.querySelector('#box-container');
const alignment = window.getComputedStyle(boxContainer)['align-items'];
assert.strictEqual(alignment, 'center');
```
