# --description--

React 是用于创建用户界面的最流行的 JavaScript 库之一，也是用于创建网页应用的库之一。React 最初由 Facebook 开发和维护，由于其效率、灵活性和特色，在网页开发中获得了极大的人气。

React 中的一个基本概念是创建可重用的 UI 组件。这些组件，例如按钮、卡片和头像组件，可以在你的应用中轻松重用。你可以将这些组件嵌套在彼此内部，以构建更复杂、动态的接口。

React 的一个关键优势是这些自定义组件可以在数据变化时独立更新和渲染。

与传统 JavaScript 需要直接操作 DOM（文档对象模型）不同，React 使用虚拟 DOM，这提高了性能和效率。你将在后续课程中了解更多关于虚拟 DOM 及其工作原理。

除了可重用组件，React 还提供了一种强大的方式来管理你的应用状态。状态指的是决定组件如何渲染和行为的数据。

使用 React，你可以轻松跟踪和更新组件的状态，确保 UI 反映最新的数据。你将在后续课程中学习更多关于状态操作的内容。

虽然 JavaScript 生态系统中有许多库，但 freeCodeCamp 主要专注于教授 React，因为它在行业中的广泛使用和需求。

接下来的几节课将深入讲解创建自定义组件、管理状态等内容。
