# --description--

HTML 和 CSS 主要是用來呈現資訊。它們不會用來撰寫邏輯。你將在下一章節學習的 JavaScript，則是用來讓網頁執行各種功能。僅使用 HTML 和 CSS 就能完成相當多的事情，而你絕對需要兩者。在我們的課程中，以下課程將專注於提供你在進入 JavaScript 內容後成功所需的工具。
