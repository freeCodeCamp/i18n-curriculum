# --description--

jortSortは、ユーザーが作業を行い、二度とソートする必要がないため効率が保証されるソートツールセットです。これは元々、Jenn "Moneydollars" Schifferが権威あるJSConf2014で発表しました。

jortSortは、比較可能なオブジェクトの単一の配列を引数に取る関数であるべきです。次に、その配列を昇順にソートし、ソートされた配列と元の配列を比較します。配列が一致する場合（つまり元の配列がすでにソートされていた場合）、関数はtrueを返します。配列が一致しない場合（つまり元の配列がソートされていなかった場合）、関数はfalseを返します。

```js
function jortsort(array) {
  // sort the array
  var originalArray = array.slice(0);
  array.sort( function(a,b){return a - b} );

  // compare to see if it was originally sorted
  for (var i = 0; i < originalArray.length; ++i) {
    if (originalArray[i] !== array[i]) return false;
  }

  return true;
};
```
