# --interactive--

CSS 可以通过三种主要方式应用到网页中：内联、内部或外部。

每个方法都有其自身的使用分支、优点和局限性，知道何时使用每个方法对于编写简洁、高效且易于维护的代码至关重要。

让我们分解三种 CSS 类型以及你应该何时使用它们。

内联 CSS 是直接在超文本标记语言元素中使用 `style` 属性编写的。它将样式应用于特定元素。

这是一个使用内联 CSS 的示例：

:::interactive_editor

```html
<p style="color: green;">This is an inline-styled paragraph.</p>
```

:::

在此示例中，我们使用 `style` 属性将段落文本设置为 `green`。

内联 CSS 通常用于快速的一次性样式或为特定元素覆写其他样式。

但是，在大多数分支中应避免这样做，因为它会使超文本标记语言杂乱，并且使代码更难维护。

大多数情况下，使用内部或外部 CSS 更有利于保持你的样式有序且易于维护。

内部 CSS 写在 HTML 文档的 `head` 部分内的 `style` 标签中。它将样式应用于整个页面，当你需要为单个文档设置样式时非常有用。

这是一个内部 CSS 的示例：

:::interactive_editor

```html
<head>
  <style>
    p {
      color: blue;
    }
  </style>
</head>
<body>
  <p>This paragraph is styled using internal CSS.</p>
</body>
```

:::

在上述示例中，内部 CSS 将蓝色文本应用于页面上的所有 `p` 元素。

内部 CSS 最适合在你需要为特定页面应用样式而不是跨多个页面时使用。它适用于单页网站或样式不需要在其他地方重用的情况。

但是，也存在一些缺点，比如不能促进跨多个页面的可重用性。此外，像内联 CSS 一样，它将超文本标记语言和 CSS 混合在一起，使得代码在较大的项目中更难维护。

外部 CSS 写在单独的 `.css` 文件中，并使用 `head` 部分的 `link` 元素链接到超文本标记语言文档。

它允许你为多个页面设置一致的样式，是专业网页开发中首选的方法。

下面是为所有段落元素设置样式的示例：

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <p>This paragraph is styled using external CSS.</p>
</body>
```

```css
p {
  color: red;
}
```

:::

在之前的课程中，你已经了解到 `link` 元素具有一个 `rel` 属性，该属性指定当前文档与链接资源之间的关系，例如链接到样式表或外部资源。

另一方面，`href` 属性指定了链接资源的 URL，指明了资源应从何处获取。

外部 CSS 非常适合大型项目，在多个页面之间维护一致的样式。

它通过让 HTML 处理结构，CSS 处理样式，促进了关注点分离，使代码更易维护和扩展。

理解何时使用每种类型的 CSS 对于高效且有效的网页开发至关重要。

在大多数分支中，外部 CSS 应该是你的首选方法，尤其是对于更大且更复杂的项目。
