# --explanation--

`While` puede usarse para contrastar dos ideas. Funciona como `even though` o `although` para mostrar que ambas cosas ocurrieron al mismo tiempo. Por ejemplo:

`While the design looked good, the app didn't work properly.` – Esto significa que el diseño era bueno, pero había un problema con cómo funcionaba la app.

En este caso, hubo aspectos positivos (la rápida respuesta del equipo) y negativos (confusión sobre responsabilidades) acerca del incidente. `While` se usa para contrastar ambos.
