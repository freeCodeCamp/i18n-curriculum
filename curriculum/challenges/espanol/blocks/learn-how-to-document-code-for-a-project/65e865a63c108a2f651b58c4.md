# --description--

En esta tarea, aprenderás la diferencia entre `specific` y `specify`. `Specific` es un adjetivo que describe algo claramente definido o detallado, mientras que `specify` es un verbo que significa declarar algo en detalle.

Por ejemplo, una instrucción `specific` podría ser `use port 8080`, mientras que `specify` puede significar indicar ese detalle: `specify the port number.`

`Setup` se refiere a la disposición o preparación de algo, a menudo utilizado en el contexto de preparar software o equipos, como `setup instructions for the software.`
