# --explanation--

Цей тип питання використовує `going to` для запитання про чиїсь **майбутні дії або плани**. Ви застосовуєте його, коли хочете дізнатися, що хтось `intends to do` або вже запланував. Наприклад:  

- `What are you going to do about the missing files?` — це питання про майбутню дію людини для розв’язання проблеми.  

- `When are you going to update the system?` — це питання про конкретний план на майбутнє.
