# --explanation--

`Have to` використовується, щоб висловити, що щось є необхідним або обов’язковим. Воно вказує, що немає іншого вибору, окрім як зробити щось. Наприклад:

- `You have to save your changes before closing the program.` - Це означає, що потрібно зберегти зміни перед закриттям програми.

Коли Том каже `I'll have to check again`, він стверджує, що йому потрібно перевірити налаштування ще раз, натякаючи, що він не впевнений у цьому.
