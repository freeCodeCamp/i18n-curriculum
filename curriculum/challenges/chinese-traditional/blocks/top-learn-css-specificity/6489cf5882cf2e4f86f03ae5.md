# --description--

現在，讓我們稍微改變一下：

```html
<!-- index.html -->

<div class="main">
  <div class="list" id="subsection"></div>
</div>
```

```css
/* rule 1 */
#subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

在上述範例中，儘管規則 2 有比 ID 選擇器更多的類別選擇器，但規則 1 更具特異性，因為 ID 優於類別。
