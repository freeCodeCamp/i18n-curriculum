# --description--

`Anyone who` використовується для позначення будь-якої людини, яка відповідає певній умові або вимозі.

У технологіях його часто використовують для опису людей із певними навичками або знаннями. Наприклад, `Anyone who understands JavaScript can modify this script.`

# --questions--

## --text--

Яке речення правильно використовує `Anyone who` у технологічному контексті?

Це речення побудоване неправильно. Правильна форма була б, `It is not clear who can join the meeting.`
