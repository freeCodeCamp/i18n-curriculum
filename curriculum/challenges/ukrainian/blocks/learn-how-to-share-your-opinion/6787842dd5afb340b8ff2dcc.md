# --explanation--

Коли вас запитують, чому ви обрали Python як першу мову, найкраще підкреслити конкретні переваги Python.

Python відомий своєю дружністю до початківців, з простим синтаксисом, який легко читати і вивчати. Крім того, Python має чудову колекцію бібліотек і фреймворків, що робить його практичним вибором для початківців.

З іншого боку, сказати `I've always wanted to be a developer` не пояснює, чому ви обрали Python. Це зосереджується на бажанні стати розробником, але не містить жодних причин, пов’язаних саме з Python.
