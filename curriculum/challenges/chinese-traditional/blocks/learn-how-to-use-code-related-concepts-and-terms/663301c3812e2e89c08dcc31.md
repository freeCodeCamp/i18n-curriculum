# --description--

表達式 `thanks for` 用來表示對特定行動或所獲得幫助的感謝。該結構通常後接動詞的 `-ing` 形式（動名詞）或名詞，以說明感謝的內容。

範例：

`Thanks for helping me with my homework.`

`Thanks for the advice.`
