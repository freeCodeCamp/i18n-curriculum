# --interactive--

通用选择器（`*`）是一种特殊类型的 CSS 选择器，用于匹配文档中的任何元素。

它通常用于将一种样式应用于页面上的所有元素，这对于在不同浏览器中重置或规范样式非常有用。

通用选择器可用于选择特定上下文中的所有元素，或在整个文档中全局选择所有元素。

以下是使用通用选择器为整个超文本标记语言文档设置 `margin` 和 `padding` 的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1>Heading element</h1>
<p>example paragraph element</p>
```

```css
* {
  margin: 0;
  padding: 0;
}
```

:::

在此代码示例中，`*` 选择器将所有元素的 margin 和 padding 重置为零，这是 CSS 重置中常用的技术。

通用选择器具有所有选择器中最低的特异性值。它对特异性值的所有部分贡献为 0，即 `(0, 0, 0, 0)`。

这意味着任何其他选择器，包括类型选择器、类选择器、ID 选择器和内联样式，都会覆写由通用的选择器设置的样式。

让我们来看以下 `HTML` 和 `CSS` 示例：

:::interactive_editor

```html
<head>
  <style>
    * {
      color: blue;
    }
    p {
      color: red;
    }
    .highlight {
      color: green;
    }
    #unique {
      color: purple;
    }
  </style>
</head>
<body>
  <p id="unique" class="highlight">This text has multiple styles applied.</p>
</body>
```

:::

通用的 `*` 选择器将所有元素的文本颜色设置为 `blue`。

但是，`p` 类型选择器通过专门为 `p` 元素设置文本颜色为 `red` 来覆写此设置。

如果一个元素具有 `highlight` 类，类选择器具有优先级，将文本颜色更改为 `green`。

最后，具有最高特异性的 `#unique` ID 选择器将覆写所有其他选择器，将文本颜色设置为 `purple`。
