# --description--

以下遊戲是組合遊戲理論的經典範例：

兩位玩家從一條長度為 $n$ 的白色方格條開始，並輪流進行。每回合，玩家選擇兩個相鄰的白色方格並將它們塗成黑色。第一個無法進行移動的玩家輸。

- $n = 1$：沒有有效的移動，因此第一個玩家自動輸掉。
- $n = 2$：只有一個有效移動，之後第二個玩家輸掉。
- $n = 3$：兩個有效移動，但兩者都會留下第二個玩家輸掉的局面。
- $n = 4$：第一位玩家有三個有效的移動；他能透過塗上中間的兩個方格來贏得遊戲。
- $n = 5$：第一位玩家有四個有效移動（如下方以紅色顯示）；但無論玩家做什麼，第二位玩家（藍色）都會獲勝。

<img alt="valid starting moves for strip with 5 squares" src="https://cdn.freecodecamp.org/curriculum/project-euler/paper-strip-game.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

所以，對於 $1 ≤ n ≤ 5$，有 3 個 $n$ 的值使得第一個玩家可以強制獲勝。

同樣地，對於 $1 ≤ n ≤ 50$，有 40 個 $n$ 的值使得第一位玩家可以強制獲勝。

對於 $1 ≤ n ≤ 1\,000\,000$，有多少個 $n$ 的值使得第一位玩家可以強制獲勝？
