# --explanation--

`final report`は、プロジェクト、イベント、または活動の結果、進捗、または調査結果をまとめた完全で完成された文書です。通常、タスクの最後に作成され、何が行われ、何が学ばれたかを振り返るために使われます。例えば：

`The final report showed that the training program improved employee performance.` – これは、トレーニングプログラム終了後の結果の要約を示しています。
