# --description--

ميزة الدوال هي أنه يمكن استدعاؤها بمعلمات مختلفة، مما يسمح لك بإعادة استخدام نفس الشفرة مع قيم متنوعة.

إليك بعض أمثلة استدعاءات الدوال مع معلمات سلسلة نصية مختلفة:

```js
// function definition
function greetUser(name) {
  console.log(`Hello ${name}!`);
}

// function calls
greetUser("John"); // "Hello John!"
greetUser("Jane"); // "Hello Jane!"
```

أضف `console.log` يستدعي الدالة `calculateSum` بالمعلمات `2` و `5`.
