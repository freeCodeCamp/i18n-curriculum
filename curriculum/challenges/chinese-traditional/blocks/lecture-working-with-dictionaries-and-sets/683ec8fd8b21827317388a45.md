# --description--

集合是 Python 內建的資料結構之一。集合的核心特性之一是它們不會儲存重複的值。如果你嘗試將重複的值添加到集合中，只有其中一個會被儲存。

集合是可變且無序的，這表示它們的元素不會以任何特定順序儲存，因此你無法使用索引或鍵來存取它們。它們只能包含不可變的資料型別值，例如數字、字串和元組。並且它們支援數學集合操作，包括聯集、交集、差集和對稱差集。

要定義一個集合，你只需要將其元素寫在大括號內並用逗號分隔。這是一個數字集合的範例：

```python
my_set = {1, 2, 3, 4, 5} 
```

使用集合時的一個特點是，如果你需要定義一個空集合，必須使用 `set()` 函式（程式）。如果你只是寫空的大括號，例如 `{}`，Python 會自動創建一個字典。

```python
set() # Set
{}    # Dictionary
```

你可以使用 `.add()` 方法，並將新元素作為引數傳入，為集合添加元素：

```python
my_set.add(6)
```

在我們的範例中，新的集合將會是：

```python
{1, 2, 3, 4, 5, 6}
```

如果你嘗試添加已經在集合中的元素，則只會保留一個。在這種情況下，我們的集合中已經有數字 5：

```python
my_set.add(5)
```

所以集合將不會改變：

```python
{1, 2, 3, 4, 5, 6}
```

要從集合中移除元素，你有兩個選項。你可以使用 `.remove()` 方法或 `.discard()` 方法，並將你想要移除的元素作為引數傳入。

如果找不到元素，`.remove()` 方法會引發 `KeyError`，而 `.discard()` 方法則不會：

```python
my_set.remove(4)
my_set.discard(4)
```

`.clear()` 方法會移除集合中的所有元素：

```python
my_set.clear()
```

Python 集合也有強大的方法，可以執行常見的數學集合操作。

`.issubset()` 和 `.issuperset()` 方法分別檢查一個集合是否為另一個集合的子集合或超集合。

這裡，我們正在檢查 `your_set` 是否為 `my_set` 的子集合，結果是 `False`，因為並非所有 `your_set` 的元素都在 `my_set` 中。

我們也在檢查 `my_set` 是否為 `your_set` 的超集合。這也是 `False`，因為 `my_set` 並不包含 `your_set` 的所有元素：

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

print(your_set.issubset(my_set)) # False
print(my_set.issuperset(your_set)) # False
```

`.isdisjoint()` 方法會檢查兩個集合是否不相交，也就是它們沒有任何共同的元素。在這個例子中，結果是 `False`，因為 `my_set` 和 `your_set` 確實有共同的元素－2、3 和 4：

```python
print(my_set.isdisjoint(your_set)) # False
```

聯集（數學）運算子 `|` 傳回一個包含兩個集合所有元素的新集合：

```python
my_set | your_set # {1, 2, 3, 4, 5, 6}
```

交集運算子 `&` 傳回一個只包含集合共同元素的新集合：

```python
my_set & your_set # {2, 3, 4}
```

差集運算子 `-` 傳回一個新集合，該集合包含第一個集合中不在其他集合內的元素。在此範例中，數字 1 和 5 在 `my_set` 中，但不在 `your_set` 中：

```python
my_set - your_set # {1, 5}
```

對稱差運算子 `^` 傳回一個新集合，該集合包含只存在於第一個集合或第二個集合中，但不在兩者皆有的元素。在此情況下，1 和 5 在 `my_set` 中但不在 `your_set` 中，因此被包含。而數字 6 在 `your_set` 中但不在 `my_set` 中，因此也被包含：

```python
my_set ^ your_set # {1, 5, 6}
```

這些運算子中的每一個，如果你在旁邊加上相等號，也都有對應的複合指定運算子。這些運算子會自動將結果集合指定給運算式中的第一個集合：

```python
|= &= -= ^=
```

例如，`-=` 運算子會找出集合之間的差異，並用該結果更新第一個集合：

```python
my_set -= your_set
```

之後，`my_set` 將會更新為 `{1, 5}`：

```python
print(my_set) # {1, 5}
```

你可以使用 `in` 運算子來檢查元素是否在集合中。這裡，我們正在檢查 5 是否在 `my_set` 中。結果將會是布林值 `True` 或 `False`：

```python
print(5 in my_set)
```

這些就是集合的基本原理。當你不需要以特定順序儲存值，且只需要儲存唯一值時，它們非常有用。
