# --explanation--

`Exactly`는 상대방과 완전히 동의한다는 점에서 `absolutely`와 거의 같은 방식으로 작동합니다. 예를 들어:

- **Vicky:** `Focusing on understanding the problem first will prevent frustration later.`

- **Jack:** `Exactly.` - 이는 그가 상대방의 말에 동의한다는 뜻입니다.
