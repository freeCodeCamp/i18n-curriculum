# --description--

正の整数 $A$ が「アレクサンドリア整数」と呼ばれるのは、整数 $p$、$q$、$r$ が存在して次の条件を満たす場合です：

$$A = p \times q \times r$$

かつ

$$\frac{1}{A} = \frac{1}{p} + \frac{1}{q} + \frac{1}{r}$$


例えば、630 はアレクサンドリア整数です（$p = 5$、$q = −7$、$r = −18$）。実際、630 は6番目のアレクサンドリア整数で、最初の6つのアレクサンドリア整数は 6、42、120、156、420、630 です。

150000番目のアレクサンドリア整数を見つけてください。
