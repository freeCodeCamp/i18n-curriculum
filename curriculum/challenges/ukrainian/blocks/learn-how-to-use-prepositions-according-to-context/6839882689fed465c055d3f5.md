# --explanation--

`Server room` означає спеціальну кімнату, де зберігаються сервери та важливе обладнання. Наприклад:

`The server room is located on the first floor.` – це область, де зберігаються сервери.

`Easy access` означає, що щось легко досягти або використати. Наприклад:

`The office has easy access to public transport.` – це означає, що до офісу легко дістатися.
