# --explanation--

Maria는 `at the end of each day`라고 말했으며, 이는 백업이 매일 예약된다는 것을 분명히 나타냅니다. 즉, 작업이 하루에 한 번 계획되어 있다는 뜻입니다. 예를 들어:

- `At the end of each day, the system runs a data backup.` - 시스템이 매일 저녁 모든 데이터를 저장하여 손실을 방지합니다.

- `We ensure the servers are updated at the end of each day.` - 근무 시간 중 중단을 피하기 위해 서버에 매일 밤 업데이트가 적용됩니다.
