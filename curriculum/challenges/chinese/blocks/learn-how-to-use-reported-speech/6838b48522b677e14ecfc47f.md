# --explanation--

`Past Continuous` 时态（`was/were` + 动词的 `-ing` 形式）描述过去一段时间内正在发生的动作。它通常与 `Simple Past` 事件一起使用，以显示持续动作被中止或发生在另一个事件之前。例如：

`She was working on the report when the computer crashed.` – 工作正在进行时，计算机发生了崩溃（`Simple Past`）。
