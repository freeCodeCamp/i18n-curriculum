# --description--

في React، تحديث المصفوفات في الحالة أمر بسيط إلى حد كبير، لكن قد يكون من السهل الوقوع في خطأ، خاصة إذا كنت قادمًا من JavaScript العادية حيث يمكنك تعديل المصفوفات مباشرة.

في React، تُعامل الحالة على أنها غير قابلة للتغيير حتى تتمكن من التعرف على التغييرات وإجراء التحديثات المناسبة لواجهة المستخدم.

لننظر كيف يمكنك تحديث المصفوفات المحتفظ بها في الحالة في React.

أحد أكثر الأخطاء شيوعًا عند تحديث المصفوفات في حالة React هو تعديل المصفوفة مباشرة باستخدام طرق مثل `push()`, `pop()`, أو `splice()`. هذه الطرق تُغير المصفوفة الأصلية، وReact لا يسمح بذلك.

تعتمد React على مرجع مصفوفة جديد لاكتشاف التغييرات، لذا فإن تعديل المصفوفة مباشرة قد يمنع المكوِّن من إعادة العرض كما هو متوقع.

إليك مثالًا على استخدام طريقة `push()` لإضافة عنصر إلى مصفوفة في الحالة، وهذا لن يعمل:

```jsx
import { useState } from "react";

function ItemsList() {
  const [items, setItems] = useState([
    { id: 0, name: "Item 1" },
    { id: 1, name: "Item 2" },
    { id: 2, name: "Item 3" },
  ]);

  const addItem = () => {
    const newItem = { id: items.length + 1, name: `Item ${items.length + 1}` };
    items.push(newItem); // This modifies the original array
    setItems(items); // React will not detect this change
  };

  return (
    <div>
      <button onClick={addItem}>Add Item</button>
      <ul>
        {items.map((item) => (
          <li key={item.id}>{item.name}</li>
        ))}
      </ul>
    </div>
  );
}

export default ItemsList;
```

إذا نقرت على زر `Add Item`، فلن يحدث شيء في واجهة المستخدم.

قد تميل أيضًا إلى إزالة عناصر من المصفوفة باستخدام طريقة `pop()`:

```js
const removeItem = () => {
  items.pop(); // Modifies the original array
  setItems(items); // React will not detect this change, either
};
```

لتحديث مصفوفة في الحالة، المفتاح هو إنشاء مصفوفة جديدة، إجراء العمليات عليها، ثم تمريرها إلى React بدلاً من تعديل المصفوفة الموجودة.

لأنها مصفوفة جديدة، ستعرف React أن الحالة قد تغيرت، وستُشغّل إعادة العرض.

إليك كيف يمكنك إضافة عنصر إلى مصفوفة `items` باستخدام معامل النشر:

```js
const addItem = () => {
  const newItem = {
    id: items.length + 1,
    name: `Item ${items.length + 1}`,
  };

  // Creates a new array
  setItems((prevItems) => [...prevItems, newItem]);
};
```

`[...prevItems, newItem]` ينشئ مصفوفة جديدة بنسخ كل العناصر في مصفوفة `items` الموجودة في الحالة، ثم يضيف `newItem` في النهاية، مما يزيد من `id` ورقم العنصر.

إذا أردت إزالة شيء من المصفوفة، يمكنك استخدام طريقة `filter()`، التي تُرجع مصفوفة جديدة بعد تصفية ما تريد إزالته:

```js
const removeItem = (id) => {
  setItems((prevItems) => prevItems.filter((item) => item.id !== id));
};
```

إليك الشفرة الكاملة:

```jsx
import { useState } from "react";

function ItemsList() {
  const [items, setItems] = useState([
    { id: 0, name: "Item 1" },
    { id: 1, name: "Item 2" },
    { id: 2, name: "Item 3" },
  ]);

  const addItem = () => {
    const newItem = { id: items.length + 1, name: `Item ${items.length + 1}` };
    setItems((prevItems) => [...prevItems, newItem]); // Creates a new array
  };

  const removeItem = (id) => {
    setItems((prevItems) => prevItems.filter((item) => item.id !== id)); // Creates a new array
  };

  return (
    <div>
      <button onClick={addItem}>Add Item</button>
      <ul>
        {items.map((item) => (
          <li key={item.id}>
            {item.name}{" "}
            <button onClick={() => removeItem(item.id)}>Remove</button>
          </li>
        ))}
      </ul>
    </div>
  );
}

export default ItemsList;
```

هذه بعض الطرق الشائعة التي يمكنك بها تحديث مصفوفة في الحالة.
