# --description--

У файлі `index.pug` є форма входу. Вона прихована через рядковий JavaScript `if showLogin`, після якого відступом йде форма.

У `res.render` для цієї сторінки додайте нову змінну до об’єкта `showLogin: true`. Коли ви оновите сторінку, форма має з’явитися! Ця форма налаштована на **POST** за адресою `/login`. Отже, саме тут потрібно налаштувати прийом POST-запиту та автентифікацію користувача.

Для цього завдання додайте маршрут `/login`, який прийматиме POST-запити. Щоб автентифікувати на цьому маршруті, потрібно додати підпрограмне забезпечення перед тим, як відправити відповідь. Це робиться, просто передавши ще один аргумент із підпрограмним забезпеченням перед відповіддю. Підпрограмне забезпечення, яке потрібно використати, — `passport.authenticate('local')`.

`passport.authenticate` також може приймати деякі опції як аргумент, наприклад `{ failureRedirect: '/' }`, що дуже корисно, тому обов’язково додайте це також. Після використання підпрограмного забезпечення (яке викликається лише якщо автентифікація пройшла успішно) додайте відповідь, яка перенаправить користувача на `/profile`. Додайте цей маршрут і зробіть так, щоб він відтворював вигляд `profile.pug`.

Якщо автентифікація була успішною, об’єкт користувача буде збережений у `req.user`.

На цьому етапі, якщо ви введете ім’я користувача та пароль у форму, має відбутися перенаправлення на домашню сторінку `/`, а консоль вашого сервера має показати `'User {USERNAME} attempted to log in.'`, оскільки наразі ми не можемо увійти користувача, який не зареєстрований.

Відправте свою сторінку, коли вважатимете, що все зроблено правильно. Якщо виникають помилки, ви можете <a href="https://forum.freecodecamp.org/t/advanced-node-and-express/567135#how-to-use-passport-strategies-7" target="_blank" rel="noopener noreferrer nofollow">переглянути проєкт, виконаний до цього моменту</a>.

POST-запит до `/login` має коректно перенаправляти на `/`.

```js
  const url = new URL("/login", code);
  const res = await fetch(url, { method: 'POST' });
  const data = await res.text();
  assert.match(
    data,
    /Looks like this page is being rendered from Pug into HTML!/,
    'A login attempt at this point should redirect to the homepage since we do not have any registered users'
  );
```
