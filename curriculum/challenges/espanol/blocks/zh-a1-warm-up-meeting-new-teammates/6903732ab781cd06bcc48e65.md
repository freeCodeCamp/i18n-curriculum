# --explanation--

`UI` significa "interfaz de usuario", que se refiere a los elementos visuales de un producto con los que los usuarios interactúan. `设计师 (shè jì shī)` significa "diseñador". Juntos, `UI 设计师 (UI shè jì shī)` se refiere a una profesión que se enfoca en diseñar interfaces de usuario. Por ejemplo:

`我是 UI 设计师。(wǒ shì UI shè jì shī.)` – Soy diseñador de UI.
