# --instructions--

Python을 사용해 포트 스캐너를 만드세요.

`port_scanner.py` 파일에서 `get_open_ports` 인수와 `target` 인수를 받는 `port_range` 함수를 만드세요. `target`는 URL이나 IP 주소일 수 있습니다. `port_range`는 검사할 포트 범위의 시작과 끝 숫자를 나타내는 두 숫자의 리스트입니다.

함수가 호출되는 예시는 다음과 같습니다:

```py
get_open_ports("209.216.230.240", [440, 445])
get_open_ports("www.stackoverflow.com", [79, 82])
```

함수는 주어진 범위 내에서 열린 포트들의 리스트를 반환해야 합니다.

`get_open_ports` 함수는 선택적으로 세 번째 인수로 `True`를 받아 "상세 모드"를 나타낼 수 있습니다. 이 값이 true로 설정되면 함수는 포트 리스트 대신 설명 문자열을 반환해야 합니다.

상세 모드에서 반환할 문자열 형식은 다음과 같습니다 (`{}` 안의 텍스트는 표시할 정보입니다):

```bash
Open ports for {URL} ({IP address})
PORT     SERVICE
{port}   {service name}
{port}   {service name}
```

각 포트에 맞는 서비스 이름을 얻으려면 `common_ports.py` 딕셔너리를 사용할 수 있습니다.

예를 들어, 함수가 다음과 같이 호출되면:

```py
port_scanner.get_open_ports("scanme.nmap.org", [20, 80], True)
```

다음과 같이 반환해야 합니다:

```bash
Open ports for scanme.nmap.org (45.33.32.156)
PORT     SERVICE
22       ssh
80       http
```

적절한 공백과 줄 바꿈 문자를 포함하세요.

`get_open_ports` 함수에 전달된 URL이 유효하지 않으면 함수는 "Error: Invalid hostname" 문자열을 반환해야 합니다.

`get_open_ports` 함수에 전달된 IP 주소가 유효하지 않으면 함수는 "Error:  Invalid IP address" 문자열을 반환해야 합니다.
