# --description--

整数の辺の長さ $N×N$ の正方形の紙が、原点に角を合わせて、2辺が $x$ 軸と $y$ 軸に沿うように置かれています。次に、以下のルールに従って切り分けます。

- 切断は、正方形の異なる辺上にあり、整数座標を持つ2点間の直線のみで行います。
- 2つの切断線は交差してはいけませんが、複数の切断線が同じ辺の点で交わることは可能です。
- これ以上合法的な切断ができなくなるまで続けます。

反射や回転を区別して数えたとき、$C(N)$ を $N×N$ の正方形の切断方法の数とします。例えば、$C(1) = 2$、$C(2) = 30$（下図参照）です。

$C(30)\bmod {10}^8$ の値はいくつでしょうか？
