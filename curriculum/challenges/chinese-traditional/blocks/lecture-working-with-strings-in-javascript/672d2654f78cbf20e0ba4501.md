# --interactive--

在 JavaScript 中，字串被視為字元序列，且可以使用括號表示法存取字串中的每個字元。這讓你能根據字元的位置（稱為其索引）來取得字串中的特定字元。

索引是字串中一個字元的位置，且是從零開始計數。這表示字串的第一個字元索引為 `0`，第二個字元索引為 `1`，依此類推。

例如，在字串 `hello` 中，字元 `h` 位於索引 `0`，`e` 位於索引 `1`，`l` 位於索引 `2`，依此類推。

括號表示法使用方括號（`[]`）和你想要存取的字元索引。讓我們來看一個範例：

:::interactive_editor

```js
let greeting = "hello";
console.log(greeting[1]); // "e"
```

:::

在這個範例中，我們可以存取索引 `1` 的字元，該字元是 `e`。

要取得字串的最後一個字元，你可以使用字串的長度減一。  
字串的 `length` 屬性告訴你它包含多少個字元，所以要存取最後一個字元，你需要從長度中減去一：

:::interactive_editor

```js
let greeting = "hello";
console.log(greeting[greeting.length - 1]); // "o"
```

:::

在這種情況下，`hello` 的 `length` 是 `5`，最後一個字元（`o`）位於索引 `4`，也就是 `5 - 1`。

如果你想取得多個字元，你可以使用括號表示法，如下所示：

:::interactive_editor

```js
let greeting = "hello";
let firstTwo = greeting[0] + greeting[1]; // "he"
console.log(firstTwo);
```

:::

在此範例中，我們使用括號表示法將第一個和第二個字元串接起來，形成字串 `he`。

當你需要存取字串中特定字元時，括號表示法非常有用，例如從名字中擷取首字母或檢查特定字元以進行驗證。
