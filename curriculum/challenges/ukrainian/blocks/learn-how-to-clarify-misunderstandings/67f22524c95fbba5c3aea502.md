# --explanation--

`On track` означає, що проєкт або завдання просувається за планом. Наприклад:

`We're on track to finish the website next month.` — Це означає, що команда, ймовірно, досягне мети.

`Release` означає запустити або зробити щось доступним. Наприклад:

`They plan to release the new version tomorrow.` — Це означає, що це буде оприлюднено.

`By the end of` встановлює часові рамки, тобто до певного моменту часу. Наприклад:

`The report is due by the end of the day.` — Це означає, що це має бути завершено до кінця дня.
