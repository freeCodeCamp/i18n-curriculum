# --description--

다음 규칙을 따르는 산술 표현식에서 나올 수 있는 양의 정수를 도달 가능한 숫자라고 합니다:

- 1부터 9까지의 숫자를 순서대로 정확히 한 번씩 사용합니다.
- 연속된 숫자들은 연결하여 사용할 수 있습니다(예: 숫자 2, 3, 4를 사용해 234를 만들 수 있습니다).
- 덧셈, 뺄셈, 곱셈, 나눗셈의 네 가지 이항 산술 연산만 허용됩니다.
- 각 연산은 여러 번 사용하거나 전혀 사용하지 않아도 됩니다.
- 단항 마이너스는 허용되지 않습니다.
- 연산 순서를 정의하기 위해 (중첩된) 소괄호를 원하는 만큼 사용할 수 있습니다.

예를 들어, 42는 도달 가능한 숫자입니다. 왜냐하면 $\frac{1}{23} \times ((4 \times 5) - 6) \times (78 - 9) = 42$이기 때문입니다.

모든 양의 도달 가능한 정수의 합은 얼마인가요?
