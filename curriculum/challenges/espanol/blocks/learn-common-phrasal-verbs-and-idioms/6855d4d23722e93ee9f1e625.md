# --explanation--

Las `Security questions` son preguntas usadas para verificar la identidad de un usuario, usualmente al iniciar sesión o recuperar una cuenta. Por ejemplo:

`One of the security questions asked for my mother's maiden name.` – Esto significa que se le pregunta al usuario el nombre de soltera de su madre como una forma de verificar la identidad del usuario.

`Password recovery` es el proceso de recuperar el acceso a una cuenta cuando la contraseña se olvida o se pierde. Por ejemplo:

`The app sent me a reset link for password recovery.` – Esto significa que el usuario recibió ayuda para restablecer su contraseña olvidada.
