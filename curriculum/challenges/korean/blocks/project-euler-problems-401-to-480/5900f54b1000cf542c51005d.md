# --description--

$a_k$, $b_k$, $c_k$가 표현식 $\frac{1}{x} = {\left(\frac{k}{x} \right)}^2 (k + x^2) - kx$의 세 해(실수 또는 복소수)를 나타낸다고 하겠습니다.

예를 들어, $k = 5$일 때, $\\{a_5, b_5, c_5\\}$는 대략 $\\{5.727244, -0.363622 + 2.057397i, -0.363622 - 2.057397i\\}$입니다.

모든 정수 $p$, $k$에 대해 $1 ≤ p, k ≤ n$일 때, $S(n) = \displaystyle\sum_{p = 1}^n \sum_{k = 1}^n {(a_k + b_k)}^p {(b_k + c_k)}^p {(c_k + a_k)}^p$라고 정의합니다.

흥미롭게도, $S(n)$은 항상 정수입니다. 예를 들어, $S(4) = 51\\,160$입니다.

$S({10}^6) \text{ modulo } 1\\,000\\,000\\,007$ 값을 구하세요.
