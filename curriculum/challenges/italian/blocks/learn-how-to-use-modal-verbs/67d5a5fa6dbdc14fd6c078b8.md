## --text--

Quale frase è grammaticalmente corretta?

## --answers--

`What happen would if the system crashed?`

### --feedback--

L'ordine corretto delle parole è `What would happen...`.

---

`Would what happen if the system crashed?`

### --feedback--

In una domanda, `would` viene dopo `what`, non prima.

---

`What would happens if the system crashed?`

### --feedback--

`Would` è seguito dalla forma base del verbo, quindi `happens` dovrebbe essere `happen`.

---

`What would happen if the system crashed?`

## --video-solution--

4

# --explanation--

Se una domanda `wh-` (`when`, `who`, `why`, `how`) usa un verbo modale, il verbo modale viene prima del soggetto. Il verbo principale resta nella sua forma base. Per esempio:

- `What can you do to fix this?` - Chiedere delle possibilità.

- `Why should we start early?` - Chiedere un consiglio.

- `How must we prepare?` - Chiedere dei requisiti.

- `When will you finish?` - Chiedere delle azioni future.

- `Where could I find help?` - Chiedere delle opzioni.

In questo modo, stai chiedendo cosa è possibile, necessario o previsto.
