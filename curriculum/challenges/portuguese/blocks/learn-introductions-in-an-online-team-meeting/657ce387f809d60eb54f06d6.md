# --description--

`Go-to person` é usado para descrever uma pessoa que é especialista ou a melhor pessoa para procurar em um campo específico ou para um problema particular. 

Por exemplo, `He's the go-to guy for computer problems` significa que ele é a pessoa mais indicada para ajudar com problemas de computador. 

# --fillInTheBlank--

## --sentence--

## --blanks--

`specific`

### --feedback--

Esta palavra é usada para indicar uma área particular ou claramente definida.

---

`go-to`

### --feedback--

Isso se refere a uma pessoa em quem todos confiam porque ela é muito prestativa ou habilidosa em algo. Esta é uma palavra composta, lembre-se de usar `-`.
