# --explanation--

`We'd prefer` ist eine höfliche Art auszudrücken, was Sie mehr möchten. Zum Beispiel:

`We'd prefer faster processors for the new computers.` – Dies zeigt ein stärkeres Interesse an besseren Optionen.

`Since` wird verwendet, um den Grund hinter einer Entscheidung oder Situation zu erklären. Zum Beispiel:

`We upgraded the servers since the old ones were too slow.` – Dies zeigt die Ursache für das Upgrade.
