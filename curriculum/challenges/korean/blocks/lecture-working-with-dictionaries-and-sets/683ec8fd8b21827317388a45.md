# --description--

집합은 Python의 내장 자료 구조 중 하나입니다. 집합의 핵심 특징 중 하나는 중복된 값을 저장하지 않는다는 점입니다. 중복된 값을 집합에 추가하려고 하면, 하나만 저장됩니다.

집합은 변경 가능하고 순서가 없기 때문에, 요소들이 특정한 순서로 저장되지 않아서 인덱스나 키로 접근할 수 없습니다. 숫자, 문자열, 튜플 같은 변경 불가능한 자료형의 값만 포함할 수 있습니다. 그리고 합집합, 교집합, 차집합, 대칭 차집합 같은 수학적 집합 연산을 지원합니다.

집합을 정의하려면 중괄호 안에 요소들을 작성하고 쉼표로 구분하면 됩니다. 다음은 숫자 집합의 예입니다:

```python
my_set = {1, 2, 3, 4, 5} 
```

집합을 다룰 때 한 가지 특이한 점은 빈 집합을 정의하려면 `set()` 함수를 사용해야 한다는 것입니다. 빈 중괄호 `{}`만 쓰면 Python이 자동으로 딕셔너리를 만듭니다.

```python
set() # Set
{}    # Dictionary
```

`.add()` 메서드를 사용해 집합에 요소를 추가할 수 있으며, 새 요소를 인수로 전달하세요:

```python
my_set.add(6)
```

예제에서 새 집합은 다음과 같습니다:

```python
{1, 2, 3, 4, 5, 6}
```

이미 집합에 있는 요소를 추가하려고 하면 하나만 유지됩니다. 이 경우, 숫자 5가 이미 집합에 있습니다:

```python
my_set.add(5)
```

그래서 집합은 변하지 않습니다:

```python
{1, 2, 3, 4, 5, 6}
```

집합에서 요소를 제거하려면 두 가지 방법이 있습니다. `.remove()` 메서드나 `.discard()` 메서드를 사용하고, 제거할 요소를 인수로 전달하세요.

`.remove()` 메서드는 요소가 없으면 `KeyError` 예외를 발생시키지만, `.discard()` 메서드는 그렇지 않습니다:

```python
my_set.remove(4)
my_set.discard(4)
```

`.clear()` 메서드는 집합의 모든 요소를 제거합니다:

```python
my_set.clear()
```

Python 집합에는 일반적인 수학적 집합 연산을 수행하는 강력한 메서드도 있습니다.

`.issubset()` 메서드와 `.issuperset()` 메서드는 각각 한 집합이 다른 집합의 부분집합인지 또는 상위집합인지 확인합니다.

여기서는 `your_set`이 `my_set`의 부분집합인지 확인하는데, `False`입니다. 왜냐하면 `your_set`의 모든 요소가 `my_set`에 있지 않기 때문입니다.

또한 `my_set`이 `your_set`의 상위집합인지 확인하는데, 이것도 `False`입니다. `my_set`가 `your_set`의 모든 요소를 가지고 있지 않기 때문입니다:

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

print(your_set.issubset(my_set)) # False
print(my_set.issuperset(your_set)) # False
```

`.isdisjoint()` 메서드는 두 집합이 서로소인지 확인합니다. 서로소란 공통 요소가 없다는 뜻입니다. 이 경우는 `False`가 아닙니다. `my_set`과 `your_set`는 2, 3, 4라는 공통 요소가 있기 때문입니다:

```python
print(my_set.isdisjoint(your_set)) # False
```

합집합 연산자 `|`는 두 집합의 모든 요소를 포함하는 새 집합을 반환합니다:

```python
my_set | your_set # {1, 2, 3, 4, 5, 6}
```

교집합 연산자 `&`는 두 집합에 공통으로 있는 요소만 포함하는 새 집합을 반환합니다:

```python
my_set & your_set # {2, 3, 4}
```

차집합 연산자 `-`는 첫 번째 집합에는 있지만 다른 집합에는 없는 요소로 새 집합을 만듭니다. 이 예에서는 숫자 1과 5가 `my_set`에는 있지만 `your_set`에는 없습니다:

```python
my_set - your_set # {1, 5}
```

대칭 차집합 연산자 `^`는 첫 번째 집합 또는 두 번째 집합 중 하나에만 있는 요소로 새 집합을 만듭니다. 이 경우 1과 5는 `my_set`에 있지만 `your_set`에는 없으므로 포함됩니다. 그리고 숫자 6은 `your_set`에 있지만 `my_set`에는 없으므로 포함됩니다:

```python
my_set ^ your_set # {1, 5, 6}
```

이 연산자들 각각은 등호를 붙인 복합 대입 연산자도 있습니다. 이 연산자들은 결과 집합을 표현식의 첫 번째 집합에 자동으로 할당합니다:

```python
|= &= -= ^=
```

예를 들어 `-=` 연산자는 집합 간 차이를 찾아 첫 번째 집합을 그 결과로 업데이트합니다:

```python
my_set -= your_set
```

이후 `my_set`은 `{1, 5}`으로 업데이트됩니다:

```python
print(my_set) # {1, 5}
```

`in` 연산자를 사용해 요소가 집합에 있는지 확인할 수 있습니다. 여기서는 5가 `my_set`에 있는지 확인합니다. 결과는 불리언 값 `True` 또는 `False`입니다:

```python
print(5 in my_set)
```

이것이 집합의 기본입니다. 특정 순서로 값을 저장할 필요가 없고, 고유한 값만 저장해야 할 때 매우 유용합니다.
