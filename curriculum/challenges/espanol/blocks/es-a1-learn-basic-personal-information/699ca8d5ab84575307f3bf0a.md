# --description--

Las personas se comunican de diferentes maneras, dependiendo de la situación.

Puedes usar el `teléfono` para hablar directamente con alguien, o un `mensaje de texto`, que es un mensaje escrito corto enviado por teléfono.

También puedes usar el `correo`, que se refiere al correo tradicional, como cartas enviadas a una dirección domiciliaria. Hoy en día, muchos mensajes se envían en línea. En estos casos, se llama `correo electrónico`.

`Electrónico` describe algo que usa tecnología o dispositivos electrónicos.
