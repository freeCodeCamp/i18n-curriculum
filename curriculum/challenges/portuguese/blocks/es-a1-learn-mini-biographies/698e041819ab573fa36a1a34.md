# --description--

O papel de um engenheiro é muito comum em contextos profissionais. Em espanhol, a palavra "ingeniero" muda conforme o gênero. `Ingeniero` é a forma masculina, enquanto `ingeniera` é a forma feminina.

Para especificar a área, você pode usar esta estrutura:

**`Ingeniero(a)` + `de` + campo**.

Algumas áreas da engenharia omitem a palavra `de`.

Por exemplo:

- `Ingeniero de software` - engenheiro de software.

- `Ingeniero civil` - engenheiro civil.

Sebastián está falando sobre o papel de Marisol.
