# --description--

Partendo dall'angolo in alto a sinistra di una griglia 2×2, e potendo muoversi solo verso destra e verso il basso, ci sono esattamente 6 percorsi fino all'angolo in basso a destra.

Quanti percorsi di questo tipo ci sono attraverso un dato `gridSize`?

# --solutions--

```js
function latticePaths(gridSize) {
  let paths = 1;

  for (let i = 0; i < gridSize; i++) {
    paths *= (2 * gridSize) - i;
    paths /= i + 1;
  }
  return paths;
}
```
