# --description--

Sei $S(A)$ die Summe der Elemente in der Menge A mit der Größe n. Wir nennen sie eine spezielle Summe-Menge, wenn für zwei nichtleere disjunkte Teilmengen B und C die folgenden Eigenschaften gelten:

1. $S(B) ≠ S(C)$; das heißt, die Summen der Teilmengen dürfen nicht gleich sein.
2. Wenn B mehr Elemente als C enthält, dann gilt $S(B) > S(C)$.

Für dieses Problem nehmen wir an, dass eine gegebene Menge n streng ansteigende Elemente enthält und bereits die zweite Regel erfüllt.

Überraschenderweise muss bei n = 4 von den 25 möglichen Teilmengenpaaren, die aus einer Menge gebildet werden können, nur 1 Paar auf Gleichheit (erste Regel) getestet werden. Ebenso müssen bei n = 7 nur 70 von den 966 Teilmengenpaaren getestet werden.

Für n = 12, wie viele der 261625 möglichen Teilmengenpaare müssen auf Gleichheit getestet werden?

**Hinweis:** Dieses Problem steht in Zusammenhang mit Problem 103 und Problem 105.
