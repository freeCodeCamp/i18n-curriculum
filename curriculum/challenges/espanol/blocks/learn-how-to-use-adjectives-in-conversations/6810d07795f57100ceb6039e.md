# --explanation--

`Alright` es una forma informal de empezar a hablar y llamar la atención de alguien. Por ejemplo:

`Alright, let's get started.` - Muestra que estás listo para comenzar.

`Here's` es la contracción de `here is`. Puedes usarlo para introducir o presentar algo. Por ejemplo:

`Here's the report you asked for.` - Estás mostrando el informe a alguien.

`Pitch` a menudo significa una presentación breve para promover o explicar algo. Por ejemplo:

`She gave a pitch to investors.` - Esto significa que hizo una presentación de negocios.
