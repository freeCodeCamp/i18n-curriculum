# --explanation--

`It's like` se usa para introducir una comparación o describir una situación de una manera que facilita su comprensión. Sugiere que algo **se siente similar a** otra situación. Por ejemplo:

`It's like he's angry at me.` - Esto significa que la situación **se siente como si** otra persona estuviera enojada contigo.
