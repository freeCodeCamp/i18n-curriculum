# --explanation--

`The developers released a fix for the login issue.` - Isso significa que eles forneceram uma solução para resolver o problema de login.

`Right away` significa imediatamente ou o mais rápido possível.Por exemplo:

`I'll send you the updated file right away.` - Isso significa que o arquivo será enviado sem demora.
