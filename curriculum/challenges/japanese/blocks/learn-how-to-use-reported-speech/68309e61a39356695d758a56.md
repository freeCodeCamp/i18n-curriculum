# --explanation--

`To notice` は、何かが変わったり目立ったりしたときに、見る、聞く、感じることで気づくことを意味します。例えば：

`Did you notice the new button on the homepage?` – これは誰かが変化を見たかどうかを尋ねるのと同じ意味です。

トムの質問、`Were the changes noticed?` は**受動態**です。気づいた人（主語）ではなく、変化（目的語）に焦点を当てています。質問における**受動態**の別の例：

`Was the mistake corrected before the launch?` – これは誰が修正したかではなく、修正に焦点を当てています。
