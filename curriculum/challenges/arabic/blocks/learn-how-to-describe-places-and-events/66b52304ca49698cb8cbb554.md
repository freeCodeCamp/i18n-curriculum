# --explanation--

في زمن `Present Simple`، عادةً ما تنتهي صيغة المفرد الغائب للأفعال بـ `-s` أو `-es`. على سبيل المثال:

- `He codes every day.` - ينتهي الفعل `codes` بـ `-s` لأنه يشير إلى `he`.

- `She fixes bugs in the system.` - ينتهي الفعل `fixes` بـ `-es` لأنه يشير إلى `she`.

يعني `Concern` القلق أو جعل شخص يشعر بالتوتر بشأن شيء ما. على سبيل المثال:

- `The security of the network concerns the IT team.` - فريق تكنولوجيا المعلومات يشعر بالقلق بشأن أمان الشبكة.

- `It concerns me that the software isn't updating correctly.` - أنت قلق من أن البرنامج لا يتم تحديثه بشكل صحيح.
