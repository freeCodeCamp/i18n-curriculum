# --explanation--

Stellen Sie sich vor, Sie schreiben Code und möchten schnell die richtige Funktion oder Variable finden, ohne alles vollständig eintippen zu müssen.

`Auto-completion` hilft, indem es vorschlägt, was Sie wahrscheinlich schreiben möchten, basierend auf dem, was Sie bereits getippt haben. Das macht das Programmieren schneller und reduziert Fehler.
