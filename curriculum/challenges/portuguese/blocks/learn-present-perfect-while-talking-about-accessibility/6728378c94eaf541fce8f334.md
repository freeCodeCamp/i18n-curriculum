# --explanation--

`Feedback` refere-se a respostas ou avaliações dos usuários sobre o desempenho de um produto.Por exemplo:

`The feedback from users helped improve the app.` - Isso significa que as opiniões dadas pelos usuários fornecem uma visão sobre o que funciona bem e o que precisa ser melhorado.

`Bugs` referem-se a problemas ou erros em software que fazem com que ele funcione incorretamente.Por exemplo:

`The team is working hard to fix the bugs before the release.` - Isso indica que problemas específicos no software precisam ser corrigidos para que ele funcione corretamente.
