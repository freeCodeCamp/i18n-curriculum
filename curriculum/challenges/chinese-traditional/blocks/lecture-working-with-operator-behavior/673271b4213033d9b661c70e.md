# --interactive--

在 JavaScript 中，所有算術運算子都有複合指定表單。複合指定運算子提供一種簡潔的縮寫，用於對變數進行操作，然後將結果存回同一個變數。它們將操作和指定結合成較短的形式，例如 `x += y`，這等同於寫成 `x = x + y`，但不需要重複變數名稱。取代寫成這樣：

:::interactive_editor

```js
let num = 5;
num = num + 2;

console.log(num); // 7
```

:::

你可以這樣寫：

:::interactive_editor

```js
let num = 5;
num += 2;

console.log(num); // 7
```

:::

注意 `num += 2` 如何將加法和指定兩個步驟合併成一步。這節省了時間並減少了你程式碼中的雜亂。讓我們深入探討 JavaScript 中最常見的複合指定運算子。

如你已經看到的，`+=` 運算子讓你將一個值加到現有的變數。它被稱為加法指派運算子。加法指派運算子會取變數的當前值，將指定的數字加上，然後將結果指派回該變數：

:::interactive_editor

```js
let total = 10;
total += 5;

console.log(total); // 15
```

:::

如你所料，有一個以 `-=` 表示的減法指定運算子。減法指定運算子會從變數的當前值中減去指定的值，並將新值指定回變數：

:::interactive_editor

```js
let score = 20;
score -= 7;

console.log(score); // 13
```

:::

如果你沒有使用減法指派，你會做類似這樣的事情：

:::interactive_editor

```js
let score = 20;
score = score - 7;

console.log(score); // 13
```

:::

乘法指定運算子由 `*=` 表述。它將變數的當前值乘以指定的數字，然後重新指定回該變數：

:::interactive_editor

```js
let points = 5;
points *= 3;

console.log(points); // 15
```

:::

最後，有一個除法指定運算子，表示為 `/=`。就像之前的其他運算子一樣，它讓你將變數的當前值除以你指定的數字，然後將結果指定回該變數：

:::interactive_editor

```js
let balance = 100;
balance /= 4;

console.log(balance); // 25
```

:::

請記住，JavaScript 中每個運算子都有一個複合指定運算子。因此，除了已經提到的四個之外，我們還有：

- 餘數指定運算子（`%=`），會將變數除以指定的數字，並將餘數指定給該變數。

- 指數指定運算子（`**=`），將變數提升到指定數字的次方，並將結果重新指定給該變數。

- 以 bit 為單元逐一 AND 指定運算子（`&=`），會對指定的數字執行以 bit 為單元逐一 AND 操作，並將結果重新指定給變數。

- 以 bit 為單元逐一 OR 指派運算子（`|=`），會對指定的數字執行以 bit 為單元逐一 OR 操作，並將結果重新指定給變數。
