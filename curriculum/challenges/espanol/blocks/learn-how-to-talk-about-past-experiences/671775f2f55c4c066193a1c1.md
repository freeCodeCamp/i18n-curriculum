# --explanation--

`Exactly` se usa para confirmar o estar completamente de acuerdo con algo que alguien más ha dicho. Indica que la afirmación es totalmente correcta. Por ejemplo: 

Persona 1: `Did you mean we should start testing today?`

Persona 2: `Exactly.` - significa que estás seguro de que las pruebas deben comenzar hoy.

En este contexto, Jake podría decir `Exactly` para confirmar que efectivamente estaba solucionando un problema de seguridad en ese entonces, como mencionó Linda.
