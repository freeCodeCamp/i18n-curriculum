# --description--

Нехай $S(A)$ представляє суму елементів множини A розміру n. Ми назвемо її особливою сумою підмножини, якщо для будь-яких двох непорожніх неперетинних підмножин B і C виконуються такі властивості:

1. $S(B) ≠ S(C)$; тобто суми підмножин не можуть бути рівними.
2. Якщо B містить більше елементів, ніж C, тоді $S(B) > S(C)$.

Якщо $S(A)$ мінімізована для заданого n, ми назвемо її оптимальною особливою сумою підмножини. Перші п’ять оптимальних особливих сум підмножин наведені нижче.

$$\begin{align}
  & n = 1: \\{1\\} \\\\
  & n = 2: \\{1, 2\\} \\\\
  & n = 3: \\{2, 3, 4\\} \\\\
  & n = 4: \\{3, 5, 6, 7\\} \\\\
  & n = 5: \\{6, 9, 11, 12, 13\\} \\\\
\end{align}$$

Здається, що для заданої оптимальної множини $A = \\{a_1, a_2, \ldots, a_n\\}$ наступна оптимальна множина має вигляд $B = \\{b, a_1 + b, a_2 + b, \ldots, a_n + b\\}$, де b — це «середній» елемент попереднього рядка.

Застосовуючи це «правило», ми очікували б, що оптимальна множина для $n = 6$ буде $A = \\{11, 17, 20, 22, 23, 24\\}$, з $S(A) = 117$. Однак це не оптимальна множина, оскільки ми просто застосували алгоритм для отримання близької до оптимальної множини. Оптимальна множина для $n = 6$ — це $A = \\{11, 18, 19, 20, 22, 25\\}$, з $S(A) = 115$ та відповідним рядком множини: `111819202225`.

Враховуючи, що A — оптимальна особлива сума підмножини для $n = 7$, знайдіть її рядок множини.

**Примітка:** Це завдання пов’язане із завданнями 105 та 106.
