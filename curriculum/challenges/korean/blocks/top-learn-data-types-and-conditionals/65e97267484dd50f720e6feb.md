# --description--

JavaScript에는 문자열, 숫자, 불리언 등 다양한 데이터 유형이 있다는 것을 알고 있습니다. 비교 연산자를 사용해 이러한 데이터 유형을 비교하고 결과에 따라 결정을 내릴 수 있습니다.

JavaScript에서 사용할 수 있는 비교 연산자는 다음과 같습니다:

| 연산자 | 설명                              | 예시         |
|--------|----------------------------------|--------------|
| ==     | 같음 (값 비교)                   | x == y       |
| !=     | 같지 않음                       | x != y       |
| >      | 큼                              | x > y        |
| <      | 작음                            | x < y        |
| >=     | 크거나 같음                     | x >= y       |
| <=     | 작거나 같음                     | x <= y       |

변수 할당에서 비교를 사용하려면 `==` 연산자를 사용할 수 있습니다. 예를 들어:

```javascript
let x = 5;
let y = 5;

let result = x == y;

console.log(result); // true
```
