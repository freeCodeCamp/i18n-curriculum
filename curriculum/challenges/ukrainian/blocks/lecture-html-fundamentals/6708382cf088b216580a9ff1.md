# --interactive--

Атрибут `id` додає унікальний ідентифікатор до HTML-елемента.

Ось приклад елемента `h1` з `id` `title`.

Під елементом `h1` додайте елемент `h2` з атрибутом `id`, встановленим у `"subtitle"`. Ви можете написати будь-який текст для `h2`, і побачите зміни у вікні попереднього перегляду. Щоб взаємодіяти з прикладом, потрібно увімкнути інтерактивний редактор.

:::interactive_editor

```html
<h1 id="title">Movie Review Page</h1>

```

:::

Ви можете посилатися на ім’я `id` елемента `title` у вашому JavaScript або CSS. Ось приклад CSS, який посилається на ідентифікатор `id` елемента `title`, щоб змінити колір тексту `color` на `red`.

**ПРИМІТКА**: У цьому інтерактивному прикладі для вас вже підготовлено частину CSS. Не хвилюйтеся, якщо ви не розумієте CSS-код — ви дізнаєтеся про нього більше в майбутніх уроках. Але якщо хочете побачити, як колір тексту зміниться на синій, увімкніть інтерактивний редактор, натисніть на вкладку `styles.css` і змініть `color: red;` на `color: blue;`.

:::interactive_editor

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>Review page Example</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
    <h1 id="title">Movie Review Page</h1>
  </body>
</html>
```

```css
#title {
  color: red;
}
```

:::

Символ решітки (`#`) перед `title` повідомляє комп’ютеру, що ви хочете націлитися на елемент з таким `id`. Імена `id` не повинні використовуватися більше одного разу і завжди мають бути унікальними. Ще одна важлива річ про значення `id` — вони не можуть містити пробілів. Ось приклад застосування слів `main` і `heading` до значення атрибута `id`:

```html
<h1 id="main heading">Main heading</h1>
```

Браузери сприйматимуть цей пробіл як частину `id`, що призведе до небажаних проблем зі стилізацією та скриптами. Значення атрибутів `id` повинні містити лише літери, цифри, підкреслення та дефіси.

На відміну від атрибута `id`, значення атрибута `class` не має бути унікальним і може містити пробіли.

Ось приклад застосування класу з назвою `box` до елемента `div`.

```html
<div class="box"></div>
```

Якщо ви хочете додати кілька імен класів до елемента, це можна зробити, розділяючи імена пробілом. Ось оновлений приклад застосування кількох класів до елемента `div`.

```html
<div class="box red-box"></div>
```

Ось ще один приклад застосування кількох класів до кількох елементів `div`.

**ПРИМІТКА**: У цьому інтерактивному прикладі для вас вже підготовлено частину CSS. Не хвилюйтеся, якщо ви не розумієте CSS-код — ви дізнаєтеся про нього більше в майбутніх уроках. Але якщо хочете змінити колір першого та третього блоків, увімкніть інтерактивний редактор, натисніть на вкладку `styles.css` і змініть `background-color: red;` на `background-color: black;`.

:::interactive_editor

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>Colored boxes example</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
    <div class="box red-box"></div>
    <div class="box blue-box"></div>
    <div class="box red-box"></div>
    <div class="box blue-box"></div>
  </body>
</html>
```

```css
.box {
  width: 100px;
  height: 100px;
}

.red-box {
  background-color: red;
}

.blue-box {
  background-color: blue;
}
```

:::

Отже, підсумовуючи, коли слід використовувати `id`, а коли `class`? Класи найкраще використовувати, коли потрібно застосувати множину стилів до багатьох елементів. Якщо хочете націлитися на конкретний елемент, краще використовувати `id`, бо ці значення мають бути унікальними.

# --questions--

## --text--

Коли слід використовувати `id`, а коли `class`?

## --answers--

Використовуйте `class`, коли потрібен унікальний ідентифікатор, який має застосовуватися лише до одного елемента.

### --feedback--

Пам’ятайте, що ім’я `id` унікальне і має використовуватися лише один раз у HTML-документі.

---

Використовуйте `id`, коли потрібен унікальний ідентифікатор для конкретного елемента, а `class` — коли хочете застосувати стилі або поведінку до кількох елементів.

---

Використовуйте `id`, коли хочете послідовно стилізувати кілька елементів у різних частинах вашої вебсторінки.

### --feedback--

Пам’ятайте, що ім’я `id` унікальне і має використовуватися лише один раз у HTML-документі.

---
