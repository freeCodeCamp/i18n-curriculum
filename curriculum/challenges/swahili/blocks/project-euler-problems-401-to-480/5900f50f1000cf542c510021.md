# --description--

Acha $n$ iwe nambari kamili chanya. Matatu ya nambari kamili ($a$, $b$, $c$) huitwa matatu ya ugawaji ya $n$ ikiwa:

- $1 ≤ a ≤ b ≤ c$
- $a \times b \times c = n$.

Tambua $f(n)$ kuwa $a + b + c$ kwa matatu ya ugawaji ($a$, $b$, $c$) ya $n$ ambayo hupunguza $\frac{c}{a}$. Inaweza kuonyeshwa kuwa matatu haya ni ya kipekee.

Kwa mfano, $f(165) = 19$, $f(100\\,100) = 142$ na $f(20!) = 4\\,034\\,872$.

Tafuta $f(43!)$.
