# --explanation--

`Around` bedeutet ungefähr oder nahe an Zeit oder Ort. Zum Beispiel:

`We plan to deploy the update around Friday.` – Das bedeutet in der Nähe dieses Tages, nicht genau.

`Mid-quarter` bezieht sich auf den Mittelpunkt eines dreimonatigen Geschäftszeitraums. Zum Beispiel:

`The review will happen in mid-quarter.` – Das bedeutet, dass es zur Mitte des aktuellen Quartals stattfinden wird.
