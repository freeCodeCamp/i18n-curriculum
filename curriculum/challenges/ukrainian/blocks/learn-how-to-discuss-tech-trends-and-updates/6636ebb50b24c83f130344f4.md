# --description--

Слово `without`, за яким слідує дієслово у формі `-ing`, використовується для позначення відсутності дії або стану. Це загальна конструкція, яка описує виконання однієї дії, одночасно не виконуючи іншу.

Наприклад, `She left without saying goodbye.` Це означає, що вона пішла і не попрощалася.
