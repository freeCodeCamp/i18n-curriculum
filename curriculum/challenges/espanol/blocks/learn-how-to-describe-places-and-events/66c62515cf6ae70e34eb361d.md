# --explanation--

A veces, cuando no estás completamente seguro de algo, pero crees que debería ser cierto, puedes usar la frase `It's supposed to` + verbo. 

Esto expresará que se espera que algo sea de cierta manera, incluso si aún no lo has confirmado. Por ejemplo:

- `The update is supposed to fix the bug.` - Esto significa que esperas que la actualización solucione el error, pero aún no lo has confirmado.
