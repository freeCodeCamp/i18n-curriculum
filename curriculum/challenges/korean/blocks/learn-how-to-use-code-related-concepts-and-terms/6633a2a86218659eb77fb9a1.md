# --description--

조동사 `can`는 가능성이나 능력을 나타낼 때 사용합니다. 예를 들어, 문장 `You can see the stars at night`에서 `can`는 밤에 별을 볼 수 있음을 의미합니다.

`Change`는 다르게 만들거나 변하게 하다는 뜻입니다. 일상생활에서는 퇴근 후에 옷을 `change`할 수 있는데, 이는 작업복에서 평상복으로 갈아입는 것을 나타냅니다.

`Stay the same`는 변하지 않고 일정하게 유지된다는 뜻입니다. 예를 들어, `The rules of the game stay the same every year.`는 규칙이 해마다 변하지 않는다는 의미입니다.
