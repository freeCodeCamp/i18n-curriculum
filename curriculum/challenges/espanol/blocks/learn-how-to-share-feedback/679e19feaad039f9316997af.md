# --explanation--

La frase `Thanks for` + `-ing` se usa para expresar gratitud por una acción que ha ocurrido o está ocurriendo. El verbo que sigue a `for` toma la forma `-ing` porque actúa como un sustantivo (gerundio). Por ejemplo:

`Thanks for helping me with the report.` - Esto significa que el hablante aprecia la ayuda dada para el informe.
