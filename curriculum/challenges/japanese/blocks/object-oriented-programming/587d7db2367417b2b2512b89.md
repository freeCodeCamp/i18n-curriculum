# --description--

これまでに見てきたように、振る舞いは継承を通じて共有されます。しかし、継承が最適な解決策でない場合もあります。継承は`Bird`や`Airplane`のような関係のないオブジェクトにはうまく機能しません。両方とも飛ぶことができますが、`Bird`は`Airplane`の一種ではなく、その逆も同様です。

関係のないオブジェクトには、<dfn>ミックスイン</dfn>を使う方が良いです。ミックスインは他のオブジェクトが関数の集合を使えるようにします。

```js
let flyMixin = function(obj) {
  obj.fly = function() {
    console.log("Flying, wooosh!");
  }
};
```

`flyMixin`は任意のオブジェクトを受け取り、そのオブジェクトに`fly`メソッドを与えます。

```js
let bird = {
  name: "Donald",
  numLegs: 2
};

let plane = {
  model: "777",
  numPassengers: 524
};

flyMixin(bird);
flyMixin(plane);
```

ここでは`bird`と`plane`が`flyMixin`に渡され、`fly`関数がそれぞれのオブジェクトに代入されます。これで`bird`と`plane`は両方とも飛べるようになります：

```js
bird.fly();
plane.fly();
```

コンソールには文字列`Flying, wooosh!`が2回表示されます。`.fly()`の呼び出しごとに1回ずつです。

ミックスインが関係のないオブジェクト`fly`と`bird`で同じ`plane`メソッドを再利用できることに注目してください。
