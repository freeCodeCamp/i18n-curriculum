# --description--

L'obiettivo ora è fare in modo che il `div` non occupi tutta la larghezza della pagina. La proprietà CSS `width` è perfetta per questo.

Puoi usare il selettore `id` per mirare a un elemento specifico con un attributo `id`. Un selettore `id` si definisce posizionando il simbolo cancelletto `#` direttamente davanti al valore `id` dell'elemento. Per esempio, se un elemento ha l'id `id` di `cat`, allora lo mireresti così:

```css 
#cat {
  width: 250px;
}
```

Usa il selettore `#menu` per dare al tuo elemento una larghezza di `300px`.

# --hints--

Dovresti avere un selettore `#menu`.

```js
const hasDiv = new __helpers.CSSHelp(document).getStyle("#menu");
assert(hasDiv);
```

Dovresti impostare la proprietà `width` su `300px`.

```js
const hasWidth = new __helpers.CSSHelp(document).getCSSRules().some(x => x.style.width === '300px');
assert(hasWidth);
```

Il tuo `div` dovrebbe avere una larghezza di 300px.

```js
const divWidth = new __helpers.CSSHelp(document).getStyle("#menu")?.getPropertyValue('width');
assert(divWidth === '300px');
```
