# --description--

動詞 `to contain` 意指在內含或擁有某物。在文件或 README 檔案的背景關係中，`to contain` 指的是包含在文件內的資訊。

例如，瓶子可以 `contain` 水，就像 README 檔案 `contains` 關於專案的資訊。

# --questions--

## --text--

在文件的背景關係中，`to contain` 是什麼意思？

## --answers--

排除

### --feedback--

這表示要將某物排除在外，這與包含相反。`To contain` 意味著包含或擁有某物。

### --feedback--

這表示移除或抹去某物，這與 `containing` 不同。`Containing` 意味著持有或包含，而非移除。

---

### --feedback--

這個表達式表示要複製某物。然而，`Containing` 是指包含某物，而不是複製它。
