# --explanation--

Mark diz, `If the application has memory leaks, it can lead to crashes`.

O termo chave aqui é `memory leaks`.Eles ocorrem quando um programa aloca memória para tarefas temporárias mas não a libera quando as tarefas são concluídas.Por exemplo:

- `If the application uses too much memory, it could crash.` - Isso sugere que o uso de memória é o problema que leva a falhas.

- `Memory leaks can cause performance issues and crashes.` - Isso mostra como vazamentos de memória são um fator que causa falhas.
