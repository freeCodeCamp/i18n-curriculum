# --explanation--

`fraction of` は、全体の一部や部分を意味します。例えば：

`We used a fraction of the budget for testing.` — 彼らはテストに使うお金のごく一部しか使いませんでした。

`Resources` は、予算、道具、人、時間など、プロジェクトを完了するために使うものです。例えば：

`The manager allocated more resources to speed up development.` — マネージャーは作業を早くするために追加の支援を提供しました。

デイビッドの文では、プロジェクトの総予算や支援のうち、予期しない費用のためにまだどれだけ残っているかを尋ねています。
