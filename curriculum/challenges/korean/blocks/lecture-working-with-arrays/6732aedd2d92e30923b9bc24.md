# --interactive--

배열 구조 분해는 JavaScript에서 배열의 값을 추출하여 더 간결하고 읽기 쉬운 방식으로 변수에 할당할 수 있게 해주는 기능입니다. 배열 요소를 개별 변수로 풀어내는 편리한 구문을 제공합니다.

이 기법은 특히 배열과 여러 값을 반환하는 함수 작업 시 유용합니다. 배열 구조 분해를 사용하는 예시는 다음과 같습니다:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];

let [first, second, third] = fruits;

console.log(first);  // "apple"
console.log(second); // "banana"
console.log(third);  // "orange"
```

:::

이 예시에서는 세 개의 요소를 가진 `fruits` 배열이 있습니다. 배열 구조 분해를 사용하여 첫 번째 요소를 `first` 변수에, 두 번째 요소를 `second` 변수에, 세 번째 요소를 `third` 변수에 할당합니다. 이렇게 하면 인덱스 표기법을 사용하지 않고도 배열의 개별 요소에 쉽게 접근할 수 있습니다.

배열 구조 분해 대신 각 요소에 인덱스로 접근하면 다음과 같습니다:

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange"];

const first = fruits[0];
const second = fruits[1];
const third = fruits[2];

console.log(first); // "apple"
console.log(second); // "banana"
console.log(third); // "orange"
```

:::

배열 구조 분해는 관심 없는 요소를 쉼표로 건너뛸 수도 있습니다. 예를 들어:

:::interactive_editor

```js
let colors = ["red", "green", "blue", "yellow"];
let [firstColor, , thirdColor] = colors;

console.log(firstColor); // "red"
console.log(thirdColor); // "blue"
```

:::

이 예시에서는 `colors` 배열의 두 번째 요소를 추가 쉼표로 건너뜁니다. 이로써 `red`가 `firstColor`에, `blue`이 `thirdColor`에 할당되어 `green`는 무시됩니다.

배열 구조 분해의 또 다른 강력한 기능은 기본값을 사용할 수 있다는 점입니다. 할당하려는 변수보다 배열 요소가 적을 경우 기본값을 제공할 수 있습니다:

:::interactive_editor

```js
let numbers = [1, 2];
let [a, b, c = 3] = numbers;

console.log(a); // 1
console.log(b); // 2
console.log(c); // 3
```

:::

여기서는 `3` 배열에 세 번째 요소가 없기 때문에 `c`에 기본값 `numbers`을 할당합니다.

이제 세 개의 점(`...`)으로 표시되는 나머지 구문에 대해 설명하겠습니다. 나머지 구문은 구조 분해되지 않은 배열의 나머지 요소를 새 배열로 캡처할 수 있게 해줍니다. 작동 방식은 다음과 같습니다:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango", "kiwi"];
let [first, second, ...rest] = fruits;

console.log(first);  // "apple"
console.log(second); // "banana"
console.log(rest);   // ["orange", "mango", "kiwi"]
```

:::

이 예시에서 `first`와 `second`는 `fruits` 배열의 처음 두 요소를 캡처하고, `rest`은 나머지 모든 요소를 새 배열로 캡처합니다. 나머지 구문은 구조 분해 패턴에서 마지막 요소여야 합니다.

배열 구조 분해는 코드를 더 간결하고 읽기 쉽게 만드는 강력한 기능입니다. 배열 작업 시 특히 유용하며, 배열에서 특정 요소를 추출해야 할 때 효과적입니다.
