# --description--

Let $y_0, y_1, y_2, \ldots$ be a sequence of random unsigned 32 bit integers (i.e. $0 ≤ y_i &lt; 2^{32}$, every value equally likely).

Para la secuencia $x_i$ se da la siguiente recursión:

- $x_0 = 0$ y
- $x_i = x_{i - 1} \mathbf{|} y_{i - 1}$, para $i > 0$. ($\mathbf{|}$ es el operador OR a nivel de bits)

Se puede ver que eventualmente habrá un índice $N$ tal que $x_i = 2^{32} - 1$ (un patrón de bits de todos unos) para todo $i ≥ N$.

Encuentra el valor esperado de $N$. Da tu respuesta redondeada a 10 dígitos después del punto decimal.
