# --explanation--

Unapoulizwa kwa nini ulichagua Python kama lugha yako ya kwanza, ni bora kuonyesha faida maalum za Python.

Python inajulikana kwa urahisi wake kwa wanaoanza, ikiwa na sintaksia rahisi ambayo ni rahisi kusoma na kujifunza. Zaidi ya hayo, Python ina mkusanyiko mzuri wa maktaba na mifumo ya kazi, ambayo hufanya iwe chaguo la vitendo kwa wanaoanza.

Kwa upande mwingine, kusema, `I've always wanted to be a developer` haelezi kwa nini ulichagua Python. Inazingatia hamu ya kuwa msanidi programu lakini haijumuishi sababu yoyote maalum ya Python.
