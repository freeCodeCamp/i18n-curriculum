# --explanation--

`Recommend` означає запропонувати або порадити щось як хорошу ідею чи рішення. Марія використовує `going to` у своєму запитанні, щоб дізнатися про **майбутні дії** або плани. Наприклад:  

`What are you going to recommend for the project?` — це запитання про те, що хтось запропонує як рішення або покращення.  

У цьому реченні Марія хоче знати, що Джейк запропонує для покращення безпеки в майбутньому. Вона не питає про минулі дії чи проблеми, а про ідеї для подальших кроків.
