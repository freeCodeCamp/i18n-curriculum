# --description--

Triplet ya nambari kamili chanya ($a$,$b$,$c$) huitwa Triplet ya Cardano ikiwa inakidhi sharti:

$$\sqrt[3]{a + b \sqrt{c}} + \sqrt[3]{a - b \sqrt{c}} = 1$$

Kwa mfano, (2,1,5) ni Triplet ya Cardano.

Kuna Triplets 149 za Cardano ambazo $a + b + c ≤ 1000$.

Tafuta ni Triplets ngapi za Cardano zipo kwa vile $a + b + c ≤ 110\\,000\\,000$.
