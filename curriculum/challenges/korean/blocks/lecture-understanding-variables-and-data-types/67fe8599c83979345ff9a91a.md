# --description--

모든 프로그래밍 언어에는 내장 메서드, 함수, 속성 또는 키워드를 사용해 터미널에 데이터를 출력하는 방법이 있습니다. Python에서는 `print` 함수를 사용해 터미널에 데이터를 출력할 수 있습니다. `print` 함수를 자세히 살펴보면서 자신 있게 사용할 수 있도록 하겠습니다.

어떤 프로그래밍 언어를 배우든 가장 먼저 하는 일 중 하나는 간단한 `Hello world!` 프로그램을 작성하는 것입니다. Python에서는 `print` 함수만으로 아주 쉽게 할 수 있습니다.

그렇게 하려면 `Hello world!` 함수를 호출할 때 사용하는 여는 소괄호와 닫는 소괄호 사이에 문자열 `print`를 넣으면 됩니다.

```python
print('Hello world!') # Hello world!
```

앞으로 수업에서 Python의 문자열과 함수에 대해 더 배우게 됩니다. 지금은 문자열을 작은따옴표(`'`) 또는 큰따옴표(`"`)로 둘러싸인 문자 수열로 생각하세요.

`print('Hello world!')` 예제에서 문자열 `'Hello world!'`는 `print` 함수에 전달된 **인수**입니다. 또한 `print` 함수를 사용해 여러 값, 즉 인수를 한 번에 쉼표로 구분해 출력할 수 있습니다. 예를 들어:

```python
print('My favorite colors are', 'blue', 'green', 'red')

# Output: My favorite colors are blue green red
```

Python은 쉼표로 구분된 각 항목 사이에 자동으로 공백을 추가합니다. 여러 정보를 함께 출력할 때 유용합니다.
