# --description--

`input`-Elemente können ein `size`-Attribut haben. Dieses Attribut definiert die Anzahl der Zeichen, die sichtbar sein sollen, während der Benutzer in die Eingabe tippt. Der Wert von `size` sollte eine nicht negative ganze Zahl größer als null sein. Wenn `size` nicht angegeben ist oder mit einem ungültigen Wert angegeben wird, hat die Eingabe die vom Browser voreingestellte Standardbreite.

```html
<label for="lastName">Last Name:</label>
<input id="lastName" name="lastName" type="text" size="10" />
```

Geben Sie den Eingaben für Name und E-Mail ein `size`-Attribut mit dem Wert `"20"`.

# --hints--

```js
const label2 = document.querySelector('label:nth-of-type(2) + input');
assert.strictEqual(label2?.getAttribute('size'), '20');
```
