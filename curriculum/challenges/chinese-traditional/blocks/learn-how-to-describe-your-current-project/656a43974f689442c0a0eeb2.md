# --description--

`Currently` 來自 `current`，意指正在發生。加上 `-ly` 後會將形容詞變成副詞。例如：

`She is currently reading a book.`－她現在正在讀一本書。

`Trends` 是許多人正在討論或實踐的流行風格或想法。單數的字 `trend` 變成複數的 `trends`。例如：

- `One current trend is using eco-friendly products.` - 這裡，`trend`（單數）指的是一個流行的想法。

- `She writes about fashion trends.` - 這裡，`trends`（複數）是指許多不同的流行風格。

`Tech trends` 指向技術領域中熱門的發展。
