# --description--

$N(i)$ を、$n!$ が $(i!)^{1234567890}$ で割り切れる最小の整数 $n$ とします。

$S(u) = \sum N(i)$ （$10 ≤ i ≤ u$ の範囲での和）と定義します。

$S(1000)=614\\,538\\,266\\,565\\,663$ です。

$S(1\\,000\\,000)\bmod {10}^{18}$ を求めてください。
