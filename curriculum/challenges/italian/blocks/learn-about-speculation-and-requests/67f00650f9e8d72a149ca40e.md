# --explanation--

Un `suspicious email` è un'email che sembra poco affidabile o pericolosa. Potrebbe far parte di un `phishing attack`, in cui qualcuno cerca di rubare informazioni. Per esempio:

`This looks like a suspicious email. Can you check it?` - Pensi che l'email possa essere pericolosa e chiedi a qualcuno di darle un'occhiata.
