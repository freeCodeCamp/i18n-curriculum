# --explanation--

`Perhaps` 意思是「也許」或「可能」。當某人不完全確定但想要提出建議時會使用。 例如：

`Perhaps we can try a new tool for project tracking。`－這表示說話者正在建議這個想法作為一種可能性。

前置式 `un-` 會加在形容詞前面以創建相反的意思。在 `unnecessary` 中，基底字是 `necessary`（意指需要的），而 `un-` 使其變成「不需要的」。另一個例子：

`It was an unfair decision.` – 「不公平」是「公平」的反義詞，意指不公正或不相等。
