# --description--

```css
/* rule 1 */
#subsection .list {
  background-color: yellow;
  color: blue;
}

/* rule 2 */
#subsection .main .list {
  color: red;
}
```

في هذا المثال النهائي، كلا القاعدتين تستخدمان محددات ID وclass، لذا لا تستخدم أي قاعدة محددًا أكثر تحديدًا من الأخرى. بعد ذلك، يتحقق التسلسل من كميات كل نوع من المحددات. كلا القاعدتين تحتويان على محدد ID واحد فقط، لكن القاعدة الثانية تحتوي على محددات class أكثر، لذلك القاعدة الثانية لها تحديدية أعلى!

بينما يأخذ الإعلان `color: red` الأسبقية، سيظل الإعلان `background-color: yellow` مطبقًا لأنه لا يوجد إعلان متضارب معه.
