# --interactive--

Gli operatori unari agiscono su un singolo operando per eseguire operazioni come la conversione di tipo, la manipolazione del valore o il controllo di certe condizioni. Vediamo alcuni operatori unari comuni e come funzionano.

L'operatore unario più converte il suo operando in un numero. Se l'operando è già un numero, rimane invariato.

:::interactive_editor

```js
const str = '42';
const strToNum = +str;

console.log(strToNum); // 42
console.log(typeof str); // string
console.log(typeof strToNum); // number
```

:::

L'operatore unario più è utile quando vuoi assicurarti di lavorare con un valore numerico. Come puoi immaginare, esiste un operatore unario di negazione. Esso nega il valore dell'operando. Funziona in modo simile all'operatore unario più, ma inverte il segno.

:::interactive_editor

```js
const str = '42';
const strToNegativeNum = -str;

console.log(strToNegativeNum); // -42
console.log(typeof str); // string
console.log(typeof strToNegativeNum); // number
```

:::

L'operatore logico NOT, rappresentato da un punto esclamativo (`!`), è un altro operatore unario. Inverte il valore booleano del suo operando. Quindi, se l'operando è `true`, diventa `false`, e se è `false`, diventa `true`. 

:::interactive_editor

```js
let isOnline = true;
console.log(!isOnline); // false

let isOffline = false;
console.log(!isOffline); // true
```

:::

L'operatore bitwise NOT è un operatore unario meno comune. Rappresentato da una tilde, `~`, inverte la rappresentazione binaria di un numero. I computer archiviano i numeri in formato binario (1 e 0). L'operatore `~` inverte ogni bit, cioè trasforma tutti gli 1 in 0 e tutti gli 0 in 1. Imparerai di più su binario e bit in una lezione futura.

:::interactive_editor

```js
const num = 5; // The binary for 5 is 00000101

console.log(~num); // -6
```

:::

In questo esempio, `5` è diventato `-6` perché applicando l'operatore `~` a `5` ottieni `- (5 + 1)`, che equivale a `-6` a causa della rappresentazione in complemento a due. Il complemento a due è un modo in cui i computer rappresentano i numeri negativi in binario. Probabilmente non userai spesso l'operatore bitwise NOT a meno che tu non stia lavorando con attività di programmazione a basso livello come la manipolazione diretta dei bit.

La parola chiave `void` è un operatore unario che valuta un'espressione e restituisce `undefined`.

:::interactive_editor

```js
const result = void (2 + 2);

console.log(result); // undefined
```

:::

`void` è anche comunemente usato nei collegamenti ipertestuali per impedire la navigazione:

```js
<a href="javascript:void(0);">Click Me</a>
```

Infine, c’è l’operatore `typeof` che hai già imparato nelle lezioni precedenti. Questo restituisce il tipo del suo operando come stringa.

:::interactive_editor

```js
const value = 'Hello world';

console.log(typeof value); // string
```

:::
