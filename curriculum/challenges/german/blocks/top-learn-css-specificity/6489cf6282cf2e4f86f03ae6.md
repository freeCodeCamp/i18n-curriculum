# --description--

```css
/* rule 1 */
#subsection .list {
  background-color: yellow;
  color: blue;
}

/* rule 2 */
#subsection .main .list {
  color: red;
}
```

In diesem letzten Beispiel verwenden beide Regeln ID- und Klassen-Selektoren, sodass keine der Regeln einen spezifischeren Selektor als die andere verwendet. Die Kaskade prüft dann die Anzahl der jeweiligen Selektortypen. Beide Regeln haben nur einen ID-Selektor, aber Regel 2 hat mehr Klassen-Selektoren, daher hat Regel 2 eine höhere Spezifität!

Während die `color: red`-Deklaration Vorrang hätte, würde die `background-color: yellow`-Deklaration dennoch angewendet, da es keine widersprüchliche Deklaration dafür gibt.
