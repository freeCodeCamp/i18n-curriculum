# --description--

음악부터 의료 분야까지 모든 산업은 크고 복잡한 데이터셋을 다룹니다. 여기에는 환자의 의료 기록, 시간에 따른 재생 목록 트렌드, 소셜 미디어 참여 지표 등이 포함될 수 있습니다.

이 정보를 이해하기 위해 기업들은 다양한 도구를 사용해 팀이 데이터를 분석하고 제품과 서비스에 대해 정보에 기반한 결정을 내리도록 돕습니다.

이 데이터를 이해하고 전달하는 데 가장 강력한 도구 중 하나가 **데이터 시각화**입니다.

데이터 시각화는 복잡한 데이터를 명확하고 시각적인 형식으로 만듭니다. 이러한 시각 자료의 예로는 원형 차트, 산점도, 막대 그래프 등이 있습니다. 좋은 시각화는 기업이 데이터에서 핵심 정보, 트렌드, 관계를 빠르게 파악하는 데 도움을 줍니다.

비즈니스에서 널리 사용되는 데이터 시각화 도구로는 Microsoft Excel, Power BI, Google Charts, Tableau 등이 있습니다.

앞으로 진행될 수업과 워크숍에서는 D3.js JavaScript 라이브러리를 사용해 인터랙티브한 시각화를 만드는 방법을 배우게 됩니다.

이 모듈의 목표는 데이터 시각화의 기본을 소개하고 JavaScript로 직접 시각화를 만드는 과정을 안내하는 것입니다.
