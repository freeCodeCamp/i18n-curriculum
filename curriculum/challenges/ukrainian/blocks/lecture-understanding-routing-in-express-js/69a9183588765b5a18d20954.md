# --description--

В Express об’єкт `res` (response) використовується для надсилання відповідей клієнту після обробки вхідних запитів. Express надає кілька вбудованих методів для надсилання різних типів відповідей. Наприклад, ви можете надіслати простий текст, JSON-дані, перенаправити користувача на іншу URL-адресу або відтворити шаблон.

Одним із найпоширеніших методів відповіді є `res.send()`:

```javascript
app.get("/text", (req, res) => {
  res.send("Hello, World!");
});
```

Метод `res.send()` надсилає відповідь клієнту. Він може надсилати простий текст, HTML, JSON або навіть бінарні дані. У цьому прикладі, коли клієнт відвідує `/text`, сервер відповідає текстом `Hello, World!`.

Ще одним часто використовуваним методом відповіді є `res.json()`.

Цей метод надсилає JSON-відповідь клієнту. Він особливо корисний при створенні API, де часто потрібно повертати структуровані дані, такі як об’єкти або масиви. Express автоматично конвертує JavaScript-об’єкт у рядок у форматі JSON:

```javascript
app.get("/user", (req, res) => {
  res.json({ name: "John Doe", age: 30 });
});
```

Якщо клієнт надсилає запит на `/user`, сервер відповідає таким чином:

```json
{
  "name": "John Doe",
  "age": 30
}
```

Коли потрібно контролювати HTTP-статус відповіді, можна використовувати `res.status()`. Цей метод встановлює код статусу і зазвичай ланцюжком викликається разом із іншим методом відповіді, наприклад `res.send()` або `res.json()`:

```javascript
app.get("/notfound", (req, res) => {
  res.status(404).send("Not Found");
});
```

Тут сервер встановлює код статусу `404` і надсилає повідомлення `Not Found` клієнту.

Якщо ви хочете перенаправити користувача на іншу URL-адресу, можна використати `res.redirect()`:

```javascript
app.get("/oldpage", (req, res) => {
  res.redirect("/newpage");
});
```

Коли клієнт відвідує `/oldpage`, його автоматично перенаправляють на `/newpage`.

Нарешті, `res.render()` використовується при роботі з шаблонними рушіями, такими як Pug, EJS або Handlebars. Він відтворює шаблон подання і надсилає отриманий HTML клієнту:

```javascript
app.get("/profile", (req, res) => {
  res.render("profile", { username: "john_doe" });
});
```

У цьому прикладі сервер відтворює подання `profile` і передає йому змінну `username`, яку потім можна відобразити в HTML.

Ці методи відповіді дозволяють повертати різні типи даних залежно від характеру запиту. Чи то надсилання тексту, JSON-даних, чи перенаправлення клієнта — Express має гнучкий набір інструментів для цього.
