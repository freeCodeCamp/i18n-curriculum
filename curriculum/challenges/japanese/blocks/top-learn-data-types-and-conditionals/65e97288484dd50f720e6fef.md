# --description--

より複雑な条件分岐は複数の条件を含む場合があります。だからこそ、複数の条件を組み合わせるために論理演算子が使われます。論理演算子には `&&`、`||`、`!` があり、それぞれ `and`、`or`、`not` を表すために使われます。

論理演算子 `||` は2つのブール値の条件を組み合わせるために使われます。条件のうち少なくとも1つが `true` の場合は `true` を返します。そうでなければ `false` を返します：

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a > b || a > c) {
  console.log("At least one of the conditions is true");
} else {
  console.log("Both of the conditions are false");
}
```

論理演算子 `&&` は2つのブール値の条件を組み合わせるために使われます。両方の条件が `true` の場合にのみ `true` を返します。そうでなければ `false` を返します：

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a < b && a < c) {
  console.log("Both of the conditions are true");
} else {
  console.log("At least one of the conditions is false");
}
```

論理演算子 `!` はブール値の条件を否定するために使われます。条件が `true` の場合は `false` を返します。そうでなければ `false` を返します：

```javascript
let a = 5;
let b = 10;

if (!(a > b)) {
  console.log("The condition is false");
} else {
  console.log("The condition is true");
}
```

上記の例では、コードを読みやすくするために余分な丸括弧が使われています。必ずしも使う必要はありません。

# --questions--

## --text--

ウェブアプリでユーザーに `if` を表示するために、次の条件をチェックする `Welcome!` 文を書く任務があります：

1. ユーザーはプレミアムアカウント（`isPremium`）を持っているか、または1年以上のメンバーである（`membershipDuration` > 12ヶ月）。

1. ユーザーは現在サービスからブロックされていない（`!isBlocked`）。

どの if文 がこれらの条件を正しくチェックしていますか？

## --answers--

```javascript
if (isPremium && membershipDuration > 12 && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (isPremium || (membershipDuration > 12 && !isBlocked)) {
  console.log("Welcome!");
}
```

---

```javascript
if ((isPremium || membershipDuration > 12) && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (!isPremium || membershipDuration <= 12 || isBlocked) {
  console.log("Welcome!");
}
```
