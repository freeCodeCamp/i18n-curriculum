## 色彩理論

- **色彩理論定義**：這是研究顏色如何相互作用以及它們如何影響我們的感知。它涵蓋色彩關係、色彩和諧以及色彩的心理影響。
- **主要顏色**：這些顏色為黃色、藍色和紅色，是所有其他顏色的基本色調。
- **次要色彩**：這些色彩是由混合相等量的兩種主要色彩所產生。綠色、橘色和紫色是次要色彩的範例。
- **第三色**：這些顏色是由原色與相鄰的次色混合而成。黃綠、藍綠和藍紫是第三色的例子。
- **暖色系**：這些顏色包括紅色、橘色和黃色，喚起舒適、溫暖和溫馨的感覺。
- **冷色調**：這些顏色包括藍色、綠色和紫色，喚起平靜、安詳和專業的感覺。
- **色彩輪**：色彩輪是一個圓形圖表，顯示顏色之間的關係。它是設計師的重要工具，因為它幫助他們選擇顏色組合。
- **類似色結構綱目**：這些色彩結構綱目創建連貫且舒緩的體驗。它們具有類似色，這些色彩在色輪上彼此相鄰。
- **互補色結構綱目**：這些色彩結構綱目產生高度對比和視覺衝擊。它們的顏色位於色輪的相對兩端。
- **三元色結構綱目**：這種結構綱目具有鮮豔的顏色。它們由彼此大約等距的顏色組成。如果將它們連接起來，會在色輪上形成一個等邊三角形。
- **單色結構綱目**：對於此結構綱目，所有顏色皆由相同的基色透過調整其明度、暗度和飽和度而產生。這喚起統一與和諧的感覺，同時仍創造對比。

## 在 CSS 中使用顏色的不同方式

- **命名顏色**：這些顏色是瀏覽器識別的預定義顏色名稱。範例包括 `blue`、`darkred`、`lightgreen`。
- **`rgb()` 函式（程式）**：RGB 代表紅色、綠色和藍色——光的三原色。這三種顏色以不同強度組合，產生廣泛的顏色範圍。`rgb()` 函式（程式）允許你使用 RGB 色彩模型定義顏色。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>RGB Color</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

- **`rgba()` 函式（程式）**：此函式（程式）會添加第四個值 alpha，用來控制顏色的透明度。如果未提供，alpha 值預設為 `1`。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>RGBA Background</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

- **`hsl()` 函式（程式）**：HSL 代表色相、飽和度和亮度——定義顏色的三個關鍵組件。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>HSL Color</p>
```

```css
p {
  color: hsl(120, 100%, 50%);
}
```

:::

- **`hsla()` 函式（程式）**：此函式（程式）新增第四個數值 alpha，用來控制顏色的不透明度。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>HSLA Background</div>
```


```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

- **十六進位**：十六進位碼（hex code 的縮寫）是一個六字元字串，用來表述 RGB 色彩模型中的顏色。「hex」指的是使用數字 0 到 9 及字母 A 到 F 的 16 進位制系統。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1 class="hex-text">Hex Text</h1>
<p class="hex-bg">Hex Background</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}

p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

## `box-shadow` 屬性

- **定義**：`box-shadow` 屬性會在元素周圍套用一個或多個陰影。
- **位移值**：你必須指定水平 (`offset-x`) 和垂直 (`offset-y`) 值。正的 `offset-x` 會將陰影向右移動，負值則向左移動。正的 `offset-y` 會將陰影向下移動，負值則向上移動。如果值為 `0`，則不需要包含單位。
- **模糊半徑**：此選用值控制陰影的模糊程度。如果未包含，預設為 `0`，會產生銳利的邊緣。數值越高，陰影越柔和。
- **Spread Radius**：這個選用值控制陰影擴展或縮小的程度。如果未包含，則預設為 `0`。
- **陰影顏色**：你可以使用命名顏色、十六進位值、`rgb()`、`rgba()`、`hsl()` 或 `hsla()` 函式指定顏色。
- **`inset` 關鍵字**：添加 `inset` 關鍵字會將陰影放置在元素內部，而非外部。
- **套用多重盒子陰影**：你可以用逗號分隔來套用多個陰影。陰影會從前到後分層。

### 語法

```css
box-shadow: offset-x offset-y blur-radius spread-radius color;
```

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="shadow-box">Shadow Color Example</div>
```

```css
.shadow-box {
  width: 200px;
  padding: 20px;
  background-color: lightblue;
  box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}
```

:::

## 線性與放射漸層

- **線性漸層**：這些漸層會沿著直線在顏色之間產生漸進混合。你可以使用像 `to top`、`to right`、`to bottom right` 這類關鍵字或 `45deg` 這類角度來控制這條線的方向。你可以使用任何有效的 CSS 顏色以及任意數量的顏色停點。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient">Linear Gradient</div>
```

```css
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

:::

- **放射狀漸層**：這些漸層會從中心點輻射出圓形或橢圓形的漸層。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient">Radial Gradient</div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::
