# --description--

Kwa namba msingi $p$ acheni $S(p) = (\sum (p - k)!)\bmod (p)$ kwa $1 ≤ k ≤ 5$.

Kwa mfano, ikiwa $p = 7$,

$$(7 - 1)! + (7 - 2)! + (7 - 3)! + (7 - 4)! + (7 - 5)! = 6! + 5! + 4! + 3! + 2! = 720 + 120 + 24 + 6 + 2 = 872$$

Kwa kuwa $872\bmod (7) = 4$, basi $S(7) = 4$.

Inaweza kuthibitishwa kuwa $\sum S(p) = 480$ kwa $5 ≤ p &lt; 100$.

Tafuta $\sum S(p)$ kwa $5 ≤ p &lt; {10}^8$.
