# --explanation--

**序数**は、ステップやタスク、段階などのシーケンスの中での位置を示します。例としては、`first`、`second`、`third`、`fourth`などがあります。

この会話では、`the first task`が初期段階で完了すべき最初のアクションであることを意味しています。
