# --description--

將新元素添加到陣列中最常見的方法之一是使用 `push()` 方法。`push()` 方法會將一個或多個元素添加到陣列的末端，並傳回陣列的新長度。

例如，要為你的 `pet` 陣列添加一個新元素，你可以使用以下程式碼：

```javascript
const pet = ['cat', 'dog', 'bunny'];
pet.push('parrot');
console.log(pet); // Output: ['cat', 'dog', 'bunny', 'parrot']
```

要移除陣列的最後一個元素，你可以使用 `pop()` 方法。`pop()` 方法會從陣列中移除最後一個元素並傳回該元素。

例如，要從 `pet` 陣列中移除最後一個元素，你可以使用以下程式碼：

```javascript
const pet = ['cat', 'dog', 'tiger'];
pet.pop();
console.log(pet); // Output: ['cat', 'dog']
```
