# --interactive--

Datum und Uhrzeit in JavaScript waren nicht immer einfach zu handhaben. In einem professionellen Codebase werden Sie wahrscheinlich eine Bibliothek verwenden, die viele dieser Probleme für Sie gelöst hat.

Aber es wird Zeiten geben, in denen Sie mit dem eingebauten `Date`-Objekt von JavaScript arbeiten müssen.

In dieser Lektion gehen wir die Grundlagen des `Date`-Objekts durch und einige der gebräuchlichsten Methoden, die Sie beim Arbeiten mit Datum und Uhrzeit in JavaScript verwenden werden.

Das `Date`-Objekt ist ein eingebautes Objekt in JavaScript, das Ihnen erlaubt, mit Datum und Uhrzeit zu arbeiten. Hier ist die grundlegende Syntax, um ein neues `Date`-Objekt zu erstellen:

```js
const now = new Date();
```

Das Schlüsselwort `new` wird verwendet, um eine neue Instanz des `Date`-Objekts zu erstellen, und das `Date`-Objekt wird dann der Variablen `now` zugewiesen. Wenn Sie den Wert von `now` in der Konsole ausgeben würden, sehen Sie das aktuelle Datum und die aktuelle Uhrzeit basierend auf der Systemuhr des Computers, der den Code ausführt.

Hier ist ein Beispiel für eine Antwort, die Sie sehen würden, wenn Sie den Wert von `now` in der Konsole ausgeben:

:::interactive_editor

```js
const now = new Date();
console.log(now);
```

:::

Für die Uhrzeit wird das 24-Stunden-Format verwendet, also ist `14:30` `2:30 PM`. `GMT-0700` ist der Zeitzonenversatz, und `Pacific Daylight Time` ist der Zeitzonenname.

Sie können auch ein bestimmtes Datum und eine bestimmte Uhrzeit an das `Date`-Objekt übergeben, indem Sie die Werte für Jahr, Monat, Tag, Stunde, Minute, Sekunde und Millisekunde als Argumente angeben.

Hier ist ein Beispiel, wie man ein neues `Date`-Objekt mit einem bestimmten Datum und einer bestimmten Uhrzeit erstellt:

:::interactive_editor

```js
const specificDate = new Date("July 4, 1776 12:00:00");

console.log(specificDate);
```

:::

Das ist nützlich, wenn Sie mit einem bestimmten Datum und einer bestimmten Uhrzeit arbeiten müssen, anstatt mit dem aktuellen Datum und der aktuellen Uhrzeit. Die Eingabe muss immer ein String sein, und das Format sollte vom `Date`-Objekt erkannt werden.

Um das aktuelle Datum und die aktuelle Uhrzeit zu erhalten, können Sie die `Date.now()`-Methode verwenden, die die Anzahl der Millisekunden seit `January 1, 1970, 00:00:00 UTC` zurückgibt. Dies ist als Unix-Epoch-Zeit bekannt.

Die Unix-Epoch-Zeit ist eine gängige Methode, um Datum und Uhrzeit in Computersystemen darzustellen, weil es sich um eine Ganzzahl handelt, die leicht gespeichert und verarbeitet werden kann. UTC steht für Coordinated Universal Time, den primären Zeitstandard, nach dem die Welt Uhren und Zeit reguliert.

Wenn Sie den Tag des Monats basierend auf dem aktuellen Datum erhalten möchten, können Sie die `getDate`-Methode verwenden. Hier ist ein Beispiel für die Verwendung der `getDate`-Methode:

:::interactive_editor

```js
const now = new Date();
const date = now.getDate();
console.log(date); 
```

:::

In diesem Beispiel erstellen wir eine neue Datum-Instanz mit dem `Date`-Objekt und weisen sie der Variablen `now` zu. Dann rufen wir die `getDate`-Methode auf dem `now`-Objekt auf, um den Tag des Monats zu erhalten, und weisen ihn der Variablen `date` zu. Schließlich geben wir den Wert von `date` in der Konsole aus, was den aktuellen Tag des Monats ausgibt.

`getDate` gibt einen ganzzahligen Wert zwischen `1` und `31` zurück, abhängig vom Tag des Monats. Wenn das Datum ungültig ist, gibt es `NaN` (Not a Number) zurück.

Um den Monat zu erhalten, können Sie die `getMonth`-Methode so verwenden:

:::interactive_editor

```js
const now = new Date();
const month = now.getMonth();
console.log(month);
```

:::

Der Monat ist nullbasiert, also ist `January` `0`, `February` ist `1` und so weiter. Wenn der Monat ungültig ist, gibt die Methode `NaN` zurück.

Wenn Sie das volle Jahr erhalten möchten, können Sie die `getFullYear`-Methode so verwenden:

:::interactive_editor

```js
const now = new Date();
const year = now.getFullYear();
console.log(year); 
```

:::

Wenn das Jahr ungültig ist, gibt es `NaN` zurück.

Es gibt viele weitere Methoden, die auf dem `Date`-Objekt verfügbar sind, darunter `getHours`, `getMinutes`, `getSeconds` und so weiter. Ich empfehle Ihnen, weitere auf eigene Faust durch die Dokumentation von Mozilla oder andere Ressourcen zu erkunden.

Wie würden Sie ein `Date`-Objekt für den 4. Juli 1776 um 12:00:00 Uhr erstellen?

## --answers--

`const specificDate = new Date(1776, 6, 4, 12, 0, 0);`

### --feedback--

Sie können das Datum und die Uhrzeit als String-Format oder als separate numerische Werte angeben.

---

`const specificDate = new Date("July 4, 1776 12:00:00")`

---

`const specificDate = new Date(1776, 7, 4, 12, 0, 0);`

### --feedback--

Sie können das Datum und die Uhrzeit als String-Format oder als separate numerische Werte angeben.

---

`const specificDate = new Date(1776, 6, 4);`
