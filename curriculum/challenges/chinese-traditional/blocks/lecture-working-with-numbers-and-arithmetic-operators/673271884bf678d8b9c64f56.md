# --interactive--

JavaScript 提供工具來對數字執行基本的算術操作，例如加法、減法、乘法和除法。JavaScript 也包含用於更複雜算術操作的運算子，例如餘數和冪次方。

所有這些工具都稱為算術運算子。讓我們詳細看看這些運算子、如何使用它們以及如何將它們組合起來。 

加法運算子是加號（`+`）。加法運算子允許你找出兩個或多個數字的總和。在加法操作中，數字的順序不重要：

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 + num2;
const result2 = num2 + num1;
const result3 = num2 + num1 + num3;

console.log(result1); // 15
console.log(result2); // 15
console.log(result3); // 30
```

:::

減法運算子是減號（`-`）。它允許你找出兩個數字之間的差異。當你想從一個數字中減去另一個數字時，通常是從較大的數字中減去較小的數字，請使用減號：

:::interactive_editor

```js
const difference = 10 - 5;
console.log(difference); // 5
```

:::

如果較小的數字先出現，你會得到負數結果：

:::interactive_editor

```js
const difference = 5 - 10;
console.log(difference); // -5
```

:::

你也可以將數字指定給變數，並使用變數名稱進行減法：

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const result = num1 - num2;

console.log(result); // 5
```

:::

在 JavaScript 中，乘法運算子由星號（`*`）表述，用於計算兩個或多個數字的乘積。你相乘的數字順序不影響結果：

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 * num2;
const result2 = num2 * num1;
const result3 = num2 * num1 * num3;

console.log(result1); // 50
console.log(result2); // 50
console.log(result3); // 750
```

:::

在 JavaScript 中，除法運算子是斜線（`/`），這與傳統數學中使用的除法符號（`÷`）不同。你使用除法運算子來執行除法操作。在這種情況下，被除數的順序很重要：

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 / num2;
const result2 = num2 / num1;
const result3 = num2 / num1 / num3;

console.log(result1); // 2
console.log(result2); // 0.5
console.log(result3); // 0.03333333333333333
```

:::

重要的是要注意，如果你嘗試除以零，JavaScript 將傳回 `Infinity`：

:::interactive_editor

```js
const result = 10 / 0; 

console.log(result); // Infinity
```

:::

請確保避免那些型別的計算，以免你的程式碼出現意外結果。 

餘數運算子，以百分比符號（`%`）表述，傳回除法的餘數。數學中的餘數是執行除法後剩餘的值：

:::interactive_editor

```js
const num1 = 10;
const num2 = 3;
const remainder = num1 % num2;

console.log(remainder); // 1
```

:::

指數運算子，由雙星號（`**`）表述，引發一個數字的次方於另一個數字：

:::interactive_editor

```js
const num1 = 2;
const num2 = 3;

const exponent = num1 ** num2;
console.log(exponent); // 8
```

:::

在單一操作或表達式中混合運算子是可能的：

:::interactive_editor

```js
const result = 10 + 5 * 2 - 8 / 4;
console.log(result); // 18
```

:::

當你在單一表達式中混合不同的運算子時，JavaScript 引擎會遵循一個稱為運算子優先的系統來決定操作的執行順序。運算子優先決定表達式中操作執行的順序。你將在未來的課程中學習更多關於運算子優先的內容。
