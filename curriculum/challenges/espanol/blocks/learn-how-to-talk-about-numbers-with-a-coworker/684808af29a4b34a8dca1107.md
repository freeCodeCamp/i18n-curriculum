# --explanation--

La `user interface` (a menudo abreviada como `UI`) es la parte visual e interactiva de un producto digital — lo que los usuarios hacen clic, ven o tocan. Por ejemplo:

`The user interface includes the navigation bar and search field.` — La parte de la app que los usuarios ven tiene elementos como el menú y el cuadro de búsqueda.

Consumir (`consume`) significa usar algo por completo, como tiempo, energía o presupuesto. Por ejemplo:

`Testing often consumes a large part of the schedule.` — Las pruebas usualmente toman mucho tiempo en el plan del proyecto.
