# --description--

オープングラフプロトコルは、FacebookやLinkedInなどのさまざまなソーシャルメディアプラットフォームでウェブサイトのコンテンツがどのように表示されるかを制御できるようにします。これらのオープングラフのプロパティを設定することで、ユーザーがクリックしてコンテンツに関わりたくなるように誘導できます。これらのプロパティは、HTMLの`meta`セクション内にある`head`要素の集合を通じて設定できます。

最初に含めるべき重要なOGプロパティは`title`です。以下はfreeCodeCampのホームページでOG `title`を設定する例です。

```html
<meta content="freeCodeCamp.org" property="og:title" />
```

`property`属性には、`og:title`であることを指定する必要があります。`content`属性には、ソーシャルメディアサイトに表示したいタイトルを書きます。

次に重要なOGプロパティは`type`です。以下はfreeCodeCampのホームページでOG `type`を使う例です。

```html
<meta property="og:type" content="website" />
```

`type`プロパティは、ソーシャルメディアで共有されるコンテンツの種類を表すために使われます。この記事、ウェブサイト、動画、音楽などが例として挙げられます。

3番目に重要なOGプロパティは`image`です。以下はfreeCodeCampのホームページでOG `image`を設定する例です。

```html
<meta
  content="https://cdn.freecodecamp.org/platform/universal/fcc_meta_1920X1080-indigo.png"
  property="og:image"
/>
```

この例では、オープングラフ画像がfreeCodeCampのロゴを指しています。これらの画像はすべて、高品質で適切な寸法と比率であるべきです。ほとんどのソーシャルメディアプラットフォームは、画像要件の基準を設けており、コンテンツがサイト上でうまく表示されるようにサポートしています。例えば、developers.facebook.comのドキュメントページには次のように記載されています。

「高解像度デバイスで最適に表示するには、少なくとも1200×630ピクセルの画像を使用してください。最低でも600×315ピクセルの画像を使用して、リンクページ投稿で大きな画像を表示することを推奨します。」

4番目に重要なOGプロパティは`url`です。以下はfreeCodeCampのホームページでOG `url`を設定する例です。

```html
<meta property="og:url" content="https://www.freecodecamp.org" />
```

`description`、`audio`、`video`、`locale`など、設定できるOGプロパティは他にもたくさんあります。しかし、オープングラフの`url`、`image`、`type`、`title`が最も重要なものです。

では、これらのオープングラフプロパティは検索エンジン最適化（SEO）にどのように影響するのでしょうか？コンテンツがソーシャルメディアで共有されるとき、よく作り込まれたOGプロパティはユーザーのフィードでのコンテンツの見た目を向上させます。これによりクリック率が高まり、検索エンジンに対してコンテンツが関連性があり魅力的であるというシグナルを送ることができます。
