# --description--

正の整数 $n$ を考えます。次の条件を満たす nim の局面を考えます：

- $n$ 個の空でない山がある。
- 各山のサイズは $2^n$ 未満である。
- どの2つの山も同じサイズではない。

$W(n)$ を、上記の条件を満たす勝ち局面の数とします（勝ち局面とは、先手が必勝戦略を持つ局面です）。

例えば、$W(1) = 1$、$W(2) = 6$、$W(3) = 168$、$W(5) = 19\,764\,360$、そして $W(100)\bmod 1\,000\,000\,007 = 384\,777\,056$ です。

$W(10\,000\,000)\bmod 1\,000\,000\,007$ を求めてください。
