# --interactive--

خوارزمية التتابع هي العملية التي يستخدمها المتصفح لتحديد أي قواعد CSS يطبق عندما تستهدف أنماط متعددة نفس العنصر. تضمن استخدام الأنماط الأنسب بناءً على مجموعة من القواعد المحددة جيدًا.

تبدأ العملية بـ **الأهمية**. يقوم المتصفح أولًا بتصفية جميع قواعد CSS ليجد تلك التي تنطبق فعليًا على العنصر المعني. يشمل ذلك مطابقة المحددات والنظر في استعلامات الوسائط التي قد تكون سارية.

استعلام الوسائط هو تقنية CSS تُستخدم لتطبيق الأنماط بناءً على خصائص الجهاز أو إطار العرض، مثل العرض أو الارتفاع أو الاتجاه.

بعد ذلك، تنظر الخوارزمية في **الأصل والأهمية**. يمكن أن تأتي أنماط CSS من مصادر مختلفة: أنماط المتصفح الافتراضية (وكيل المستخدم)، وأنماط يحددها المستخدم، وأنماط يكتبها المؤلف (أنت).

بعد النظر في الأصل، تقيم الخوارزمية أهمية كل قاعدة، مع إعطاء أولوية للقواعد المعلمة بـ `!important`، التي تتجاوز القواعد الأخرى بغض النظر عن مصدرها.

بعد التصفية حسب الأصل والأهمية، تنظر الخوارزمية في **التحديد**. عندما تنطبق قاعدتان من نفس الأصل ومستوى الأهمية، تُطبق القاعدة ذات التحديد الأعلى.

التحديد هو مقياس لمدى استهداف المحدد، حيث تأخذ المحددات الأكثر تحديدًا الأولوية على المحددات العامة.

أخيرًا، إذا كانت كل العوامل الأخرى متساوية، يأتي دور **ترتيب الظهور**. عندما يكون لقاعدتين نفس التحديد، تُطبق القاعدة التي تظهر أخيرًا في CSS.

لهذا السبب، يمكن لترتيب كتابة أنماطك أن يؤثر أحيانًا على النتيجة.

لنلقِ نظرة على مثال.

في ملف HTML، يوجد عنصر فقرة واحد. ثم داخل CSS، لدينا قاعدتان، كل واحدة تستهدف عنصر الفقرة.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>example paragraph</p>
```

```css
p {
  color: blue;
}

p {
  color: green; 
}
```

:::

القاعدة الأولى تحدد لون نص جميع عناصر الفقرة باللون الأزرق، بينما القاعدة الثانية تحدد لون نص جميع عناصر الفقرة باللون الأخضر.

فأي لون سيتم تطبيقه؟ سيتم تطبيق اللون الأخضر على عناصر الفقرة.

من خلال النظر في الأهمية، الأصل والأهمية، التحديد، وترتيب الظهور، تضمن خوارزمية التتابع أن يتصرف CSS الخاص بك بشكل متوقع، مما يسمح لك بتصميم صفحات ويب أكثر تعقيدًا وتفصيلًا.
