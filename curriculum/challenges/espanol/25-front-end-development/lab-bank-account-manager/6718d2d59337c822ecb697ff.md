# --description--

**Objetivo:** Completa las instrucciones que se indican a continuación y termina todas las pruebas para aprobar el laboratorio.

**Instrucciones:**

1. Debes definir una clase llamada `BankAccount` con un constructor. El constructor debe establecer el valor predeterminado del `balance` a `0` e inicializar un array vacío llamado `transactions` para almacenar registros de transacciones como objetos.

2. Cada transacción almacenada en el array `transactions` debe ser un objeto con dos propiedades: `type` y `amount`. La propiedad `type` debería ser `deposit` o `withdraw`, y la propiedad `amount` debería ser la cantidad depositada o retirada.

3. Debes definir un método llamado `deposit` que toma la cantidad depositada como un parámetro. Cuanto la cantidad del depósito es mayo a `0` debería:

   - Empujar un nuevo objeto al array transacciones con el tipo `deposit` y la cantidad depositada.
   - Actualiza el balance.
   - Devuelve `"Se han depositado exitosamente $[amount]. Nuevo balance: $[balance]"`.

4. Si la cantidad es menos o igual a `0`, el método `deposit` debería devolver `"Deposit amount must be greater than zero."`.

5. Debes definir un método llamado `withdraw` que tome una cantidad como parámetro. Este método debe actualizar el balance en función a los retiros. Cuando la cantidad retirada es mayor a `0` y menos o igual al balance actual, debería:

   - Empujar un nuevo objeto al array de transacciones con el tipo `withdraw` y la cantidad retirada.
   - Actualiza el balance.
   - Devuelve `"Se han retirado exitosamente $[amount]. Nuevo balance: $[balance]"`.

6. Si la cantidad a retirar es menos o igual a `0` o mayor al balance actual, el método `withdraw` debe devolver `"Insufficient balance or invalid amount."`.


7. Debes definir un método llamado `checkBalance` que devuelve el balance actual en el formato `"Current balance: $[balance]"`.

8. Debes definir un método llamado `listAllDeposits` que iterará a través del array de transacciones y devuelve todos los depósitos en el formato `"Deposits: amount,amount,..."`.

9. Debes definir un método llamado `listAllWithdrawals` que iterará a través del array de transacciones y deuelve todos los retiros en el formato `"Withdrawals: amount,amount,..."`.

10. Debes crear una nueva instancia de `BankAccount` llamada `myAccount`.

11. Tu `myAccount` cuenta bancaria deberia tener al menos cinco transacciones.

12. Tu `myAccount` cuenta bancaria debe tener al menos dos depositos.

13. Tu `myAccount` cuenta bancaria debe tener al menos dos retiros.

14. Tu `myAccount` cuenta bancaria debe tener un balance mayor a `$100`.

class BankAccount {
    constructor() {
        this.balance = 0;
        this.transactions = [];
    }

    deposit(amount) {
        if (amount > 0) {
            this.transactions.push({ type: 'deposit', amount });
            this.balance += amount;
            return `Successfully deposited $${amount}. New balance: $${this.balance}`;
        } else {
            return 'Deposit amount must be greater than zero.';
        }
    }

    withdraw(amount) {
        if (amount > 0 && amount <= this.balance) {
            this.transactions.push({ type: 'withdraw', amount });
            this.balance -= amount;
            return `Successfully withdrew $${amount}. New balance: $${this.balance}`;
        } else {
            return 'Insufficient balance or invalid amount.';
        }
    }

    checkBalance() {
        return `Current balance: $${this.balance}`;
    }

    listAllDeposits() {
        const deposits = this.transactions
            .filter(transaction => transaction.type === 'deposit')
            .map(transaction => transaction.amount);
        return `Deposits: ${deposits}`;
    }

    listAllWithdrawals() {
        const withdrawals = this.transactions
            .filter(transaction => transaction.type === 'withdraw')
            .map(transaction => transaction.amount);
        return `Withdrawals: ${withdrawals}`;
    }
}

const myAccount = new BankAccount();

console.log(myAccount.deposit(0)); 
console.log(myAccount.deposit(20)); 
console.log(myAccount.deposit(500));
console.log(myAccount.withdraw(30));
console.log(myAccount.withdraw(55));
console.log(myAccount.withdraw(100));
console.log(myAccount.checkBalance());
console.log(myAccount.listAllDeposits());
console.log(myAccount.listAllWithdrawals());
