# --explanation--

Ángela diz `Perfecto. Entonces, Esteban Navarro Huamán.`

`Perfecto` é usado para mostrar concordância ou satisfação. Indica que a informação está correta.

`Entonces` é usado após ouvir informações. Ele ajuda o falante a organizar ideias e repetir a informação claramente.

Aqui, `entonces` introduz uma confirmação. Ángela repete o nome completo para verificar se está correto.

Ela diz `Huamán`, não `Guzmán`. Portanto, a declaração é falso.
