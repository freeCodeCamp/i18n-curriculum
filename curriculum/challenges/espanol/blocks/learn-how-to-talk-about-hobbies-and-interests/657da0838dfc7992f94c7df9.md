# --description--

Recomendar significa decir que algo es una buena opción para alguien. Es cuando das un consejo sobre lo que otra persona debería hacer, usar o probar. Por ejemplo:

`They recommended testing the app on different devices.` - Esto sugiere la mejor acción para la calidad.

Cuando tengas acciones que sigan al verbo `recommend`, úsalas en la forma `-ing`. Por ejemplo, `I recommend reading this book if you want to understand this topic better`.
