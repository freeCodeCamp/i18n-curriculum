# --description--

在以下方程中，`x`、`y` 和 `n` 是正整数。

$$\frac{1}{x} + \frac{1}{y} = \frac{1}{n}$$

对于一个极限 $L$，我们定义 $F(L)$ 为满足 $x < y ≤ L$ 的解的数量。

我们可以验证 $F(15) = 4$ 和 $F(1000) = 1069$。

查找 $F({10}^{12})$。
