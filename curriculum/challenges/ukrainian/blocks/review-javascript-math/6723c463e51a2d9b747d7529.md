# --interactive--

## Робота з типом даних Number

- **Визначення**: `Number` у JavaScript включає цілі числа, числа з плаваючою комою, `Infinity` та `NaN`. Числа з плаваючою комою — це числа з десятковою крапкою. Додатне `Infinity` — це число, більше за будь-яке інше число, а `-Infinity` — число, менше за будь-яке інше число. `NaN` (`Not a Number`) представляє недійсне числове значення, (наприклад, рядок `"Jessica"`).

## Поширені арифметичні дії

- **Оператор додавання**: цей оператор (`+`) використовується для обчислення суми двох або більше чисел.
- **Оператор віднімання**: цей оператор (`-`) використовується для обчислення різниці між двома числами.
- **Оператор множення**: цей оператор (`*`) використовується для обчислення добутку двох або більше чисел.
- **Оператор ділення**: цей оператор (`/`) використовується для обчислення частки між двома числами.
- **Ділення на нуль**: якщо ви спробуєте поділити на нуль, JavaScript поверне `Infinity`.
- **Оператор остачі**: цей оператор (`%`) повертає остачу від ділення.
- **Оператор піднесення до степеня**: цей оператор (`**`) підносить одне число до степеня іншого.

## Обчислення з числами та рядками

- **Пояснення**: коли ви використовуєте оператор `+` з числом і рядком, JavaScript перетворює число на рядок і конкатенує обидва значення. Коли ви використовуєте оператори `-`, `*` або `/` з рядком і числом, JavaScript перетворює рядок на число, і результатом буде число. Для `null` і `undefined` JavaScript трактує `null` як 0, а `undefined` як `NaN` у математичних операціях.

:::interactive_editor

```js
const result = 5 + '10';

console.log(result); // "510"
console.log(typeof result); // string

const subtractionResult = '10' - 5;
console.log(subtractionResult); // 5
console.log(typeof subtractionResult); // number

const multiplicationResult = '10' * 2;
console.log(multiplicationResult); // 20
console.log(typeof multiplicationResult); // number

const divisionResult = '20' / 2;
console.log(divisionResult); // 10
console.log(typeof divisionResult); // number

const result1 = null + 5;
console.log(result1); // 5
console.log(typeof result1); // number

const result2 = undefined + 5;
console.log(result2); // NaN
console.log(typeof result2); // number
```

:::

## Пріоритет операторів

- **Визначення**: Пріоритет операторів визначає порядок, у якому операції оцінюються у виразі. Оператори з вищим пріоритетом оцінюються раніше за оператори з нижчим пріоритетом. Значення в дужках оцінюються першими, а множення/ділення мають вищий пріоритет, ніж додавання/віднімання. Якщо оператори мають однаковий пріоритет, тоді JavaScript використовує асоціативність.

:::interactive_editor

```js
const result = (2 + 3) * 4;

console.log(result); // 20

const result2 = 10 - 2 + 3;

console.log(result2); // 11

const result3 = 2 ** 3 ** 2;

console.log(result3); // 512
```

:::

- **Визначення**: Асоціативність вказує напрямок, у якому оцінюється вираз, коли існує кілька операторів одного типу. Вона визначає, чи слід оцінювати вираз зліва направо (`left-associative`) або справа наліво (`right-associative`). Наприклад, оператор піднесення до степеня також асоціативний справа наліво:

:::interactive_editor

```js
const result4 = 5 ** 4 ** 1; 

console.log(result4); // 625
```

:::

## Оператори інкременту та декременту

- **Оператор інкременту**: цей оператор використовується для збільшення значення на одиницю. Префіксний запис `++num` спочатку збільшує значення змінної, а потім повертає нове значення. Постфіксний запис `num++` спочатку повертає поточне значення змінної, а потім збільшує його.

:::interactive_editor

```js
let x = 5;

console.log(++x); // 6
console.log(x); // 6


let y = 5;

console.log(y++); // 5
console.log(y); // 6
```

:::

- **Оператор декременту**: цей оператор використовується для зменшення значення на одиницю. Префіксний та постфіксний записи працюють так само, як і раніше з оператором інкременту.

:::interactive_editor

```js
let num = 5;

console.log(--num); // 4
console.log(num--); // 4
console.log(num); // 3
```

:::

## Оператори складеного присвоєння

- **Оператор додавання з присвоєнням (`+=`)**: цей оператор виконує додавання значень і призначає результат змінній.
- **Оператор віднімання з присвоєнням (`-=`)**: цей оператор виконує віднімання значень і призначає результат змінній.
- **Оператор множення з присвоєнням (`*=`)**: цей оператор виконує множення значень і призначає результат змінній.
- **Оператор ділення з присвоєнням (`/=`)**: цей оператор виконує ділення значень і призначає результат змінній.
- **Оператор присвоєння остачі (`%=`)**: цей оператор ділить змінну на вказане число і призначає остачу змінній.
- **Оператор піднесення до степеня з присвоєнням (`**=`)**: цей оператор підносить змінну до степеня вказаного числа і повторно призначає результат змінній.

## Булеві значення та рівність

- **Визначення булевого типу**: булевим називають тип даних, який може мати лише два значення: `true` або `false`.
- **Оператор рівності (`==`)**: цей оператор виконує перетворення типів перш ніж перевірити, чи значення рівні.

:::interactive_editor

```js
console.log(5 == '5'); // true
```

:::

- **Оператор суворої рівності (`===`)**: цей оператор не виконує перетворення типів і перевіряє рівність як типів, так і значень.

:::interactive_editor

```js
console.log(5 === '5'); // false
```

:::

- **Оператор нерівності (`!=`)**: Цей оператор використовує перетворення типів перед перевіркою, чи не рівні значення.
- **Оператор суворої нерівності (`!==`)**: Цей оператор не виконує перетворення типів і перевіряє, чи не рівні як типи, так і значення.

## Оператори порівняння

- **Оператор «більше ніж» (`>`)**: цей оператор перевіряє, чи значення зліва більше за значення справа.
- **Оператор «більше або дорівнює» (`>=`)**: цей оператор перевіряє, чи значення зліва більше або рівне значенню справа.
- **Оператор «менше ніж» (`<`)**: цей оператор перевіряє, чи значення зліва менше за значення справа.
- **Оператор «менше або дорівнює» (`<=`)**: цей оператор перевіряє, чи значення зліва менше або рівне значенню справа.

## Унарні оператори

- **Унарний оператор плюс**: цей оператор перетворює свій операнд у число. Якщо операнд уже є числом, то він залишається без змін.

:::interactive_editor

```js
const str = '42';
const num = +str;

console.log(num); // 42
console.log(typeof num); // number
```

:::

- **Унарний оператор заперечення (`-`)**: цей оператор змінює знак операнда на протилежний.

:::interactive_editor

```js
const num = 4;
console.log(-num); // -4
```

:::

- **Логічний оператор NOT (`!`)**: цей оператор інвертує булеве значення свого операнда. Тобто, якщо операнд дорівнює `true`, він стає `false`, а якщо `false`, то стає `true`.

## Побітові оператори

- **Побітовий оператор AND (`&`)**: цей оператор повертає 1 у кожній позиції біта, для якої відповідні біти обох операндів дорівнюють 1.  
- **Побітовий оператор присвоєння AND (`&=`)**: цей оператор виконує операцію `bitwise AND` з вказаним числом і повторно призначає результат змінній.  
- **Побітовий оператор OR (`|`)**: цей оператор повертає 1 у кожній позиції біта, для якої відповідні біти одного або обох операндів дорівнюють 1.  
- **Побітовий оператор присвоєння OR (`|=`)**: цей оператор виконує операцію `bitwise OR` з вказаним числом і повторно призначає результат змінній.  
- **Побітовий оператор XOR (`^`)**: цей оператор повертає 1 у кожній позиції біта, для якої відповідні біти одного, але не обох, операндів дорівнюють 1.  
- **Побітовий оператор NOT (`~`)**: цей оператор інвертує бінарне представлення числа.  
- **Побітовий оператор зсуву вліво (`<<`)**: цей оператор зсуває всі біти вліво на вказану кількість позицій.  
- **Побітовий оператор зсуву вправо (`>>`)**: цей оператор зсуває всі біти вправо.

## Умовні інструкції, правдиві значення, хибні значення та тернарний оператор

- **`if/else if/else`**: інструкція `if` приймає умову і запускає блок коду, якщо ця умова `truthy`. Якщо умова `false`, тоді виконується блок `else if`. Якщо жодна з цих умов не `true`, тоді виконується умова `else`. `Truthy` значення — це будь-які значення, які при оцінці в булевому контексті, наприклад в інструкції `if`, дають результат `true`. `Falsy` значення — це значення, які при оцінці в булевому контексті дають результат `false`.

:::interactive_editor

```js
const score = 87;

if (score >= 90) {
 console.log('You got an A'); 
} else if (score >= 80) {
 console.log('You got a B'); // You got a B
} else if (score >= 70) {
 console.log('You got a C');
} else {
 console.log('You failed! You need to study more!');
}
```

:::

- **Тернарний оператор**: цей оператор часто використовується як коротший спосіб запису інструкцій `if else`.

:::interactive_editor

```js
const temperature = 30;
const weather = temperature > 25 ? 'sunny' : 'cool';

console.log(`It's a ${weather} day!`); // It's a sunny day!
```

:::

## Бінарні логічні оператори

- **Логічний AND (`&&`)**: Цей оператор перевіряє, чи обидва операнди є `true`. Якщо обидва `true`, він повертає друге значення. Якщо будь-який операнд `falsy`, він повертає `falsy`. Якщо обидва операнди `falsy`, він повертає перше `falsy` значення.

:::interactive_editor

```js
const result = true && 'hello';

console.log(result); // hello
```

:::

- **Логічний OR (`||`)**: Цей оператор перевіряє, чи принаймні один з операндів є правдивим.
- **Оператор об’єднання з null (`??`)**: Цей оператор повертає значення, лише якщо перше є `null` або `undefined`.

:::interactive_editor

```js
const userSettings = {
 theme: null,
 volume: 0,
 notifications: false,
};

let theme = userSettings.theme ?? 'light';
console.log(theme); // light
```

:::

## Об’єкт `Math`

- **Метод `Math.random()`**: цей метод генерує випадкове число з плаваючою комою між 0 (включно) та 1 (виключно). Це означає, що можливе вихідне значення може бути 0, але ніколи не досягне 1.
- **Метод `Math.max()`**: цей метод приймає множину чисел і повертає максимальне значення.
- **Метод `Math.min()`**: цей метод приймає множину чисел і повертає мінімальне значення.
- **Метод `Math.ceil()`**: цей метод округлює значення вгору до найближчого цілого числа.
- **Метод `Math.floor()`**: цей метод округлює значення вниз до найближчого цілого числа.
- **Метод `Math.round()`**: цей метод округлює значення до найближчого цілого числа.

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

- **Метод `Math.trunc()`**: цей метод видаляє десяткову частину числа, повертаючи лише цілу частину без округлення.
- **Метод `Math.sqrt()`**: цей метод повертає квадратний корінь числа.
- **Метод `Math.cbrt()`**: цей метод повертає кубічний корінь числа.
- **Метод `Math.abs()`**: цей метод повертає абсолютне значення числа.
- **Метод `Math.pow()`**: цей метод приймає два числа і підносить перше до степеня другого.

## Поширені методи для чисел

- **`isNaN()`**: `NaN` означає Not-a-Number (укр. «не число»). Це спеціальне значення, яке представляє неможливий або невизначений числовий результат. Властивість функції `isNaN()` використовується, щоб визначити, чи є значення `NaN` чи ні. `Number.isNaN()` надає більш надійний спосіб перевірки на `NaN`, особливо у випадках, коли перетворення типів може призвести до несподіваних результатів із глобальною функцією `isNaN()`.

:::interactive_editor

```js
console.log(isNaN(NaN));       // true
console.log(isNaN(undefined)); // true
console.log(isNaN({}));        // true

console.log(isNaN(true));      // false
console.log(isNaN(null));      // false
console.log(isNaN(37));        // false


console.log(Number.isNaN(NaN));        // true
console.log(Number.isNaN(Number.NaN)); // true
console.log(Number.isNaN(0 / 0));      // true

console.log(Number.isNaN("NaN"));      // false
console.log(Number.isNaN(undefined));  // false
```

:::

- **Метод `parseFloat()`**: цей метод аналізує рядковий аргумент і повертає число з плаваючою комою. Він призначений для вилучення числа з початку рядка, навіть якщо пізніше в рядку є нечислові символи.
- **Метод `parseInt()`**: цей метод аналізує рядковий аргумент і повертає ціле число. `parseInt()` припиняє аналіз при першому нецифровому символі. Для чисел з плаваючою комою він повертає лише цілу частину. Якщо на початку рядка не вдається знайти дійсне ціле число, він повертає `NaN`.
- **Метод `toFixed()`**: цей метод викликається для числа і приймає один необов’язковий аргумент — кількість цифр після десяткової крапки. Він повертає рядкове представлення числа з вказаною кількістю десяткових знаків.
