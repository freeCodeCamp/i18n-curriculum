# --explanation--

Die Verwendung von `that` nach `I think` ist in solchen Sätzen optional. Auch die Formulierung `I think that it's vital we make these updates.` ist korrekt. Im alltäglichen Gespräch ist es jedoch üblich, `that` wegzulassen, um den Satz flüssiger und schneller zu machen. Zum Beispiel:

`I think that we need more time to finish the project.` – Diese Struktur ist korrekt. Die meisten würden jedoch sagen `I think we need more time…`.
