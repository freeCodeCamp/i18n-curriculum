# --explanation--

`More user-friendly` significa que algo es más fácil de usar en comparación con otra opción. `More` se usa en esta forma comparativa porque `user-friendly` es un adjetivo más largo. Por ejemplo:

`This new app is more user-friendly than the old one because it has a simpler design.` - Esto significa que la nueva app es más fácil de usar que la anterior.
