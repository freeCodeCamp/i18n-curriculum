# --description--

Quando você deseja que os usuários façam seleções em um menu suspenso, você pode usar os elementos <dfn>select</dfn> e <dfn>option</dfn>.

Aqui está um exemplo de uso dos elementos `select` e `option` para criar uma lista suspensa para diferentes cidades:

```html
<label for="city">Choose a City: </label>
<select id="city" name="city">
  <option value="new-york">New York</option>
  <option value="los-angeles">Los Angeles</option>
  <option value="chicago">Chicago</option>
  <option value="miami">Miami</option>
</select>
```

Comece adicionando um elemento `select` com os atributos `name` e `id` definidos como `"service"`.
