# --interactive--

`grid-column`와 `grid-row` 속성은 그리드 레이아웃 내에서 그리드 항목의 가로 및 세로 위치를 지정할 수 있게 해줍니다.

즉, 두 속성 모두 그리드 항목이 시작하고 끝나는 위치를 그리드 선을 참조하여 제어할 수 있습니다. 이 그리드 선은 이미 `grid-template-rows`와 `grid-template-columns` 속성을 사용해 정의한 행과 열을 구분하는 경계입니다.

이제 `grid-row`와 `grid-column`의 구문과 실용적인 예제를 살펴보겠습니다.

`grid-row` 속성의 구문은 다음과 같습니다:

```css
grid-row: <start-line> / <end-line>;
```

그리고 `grid-column`의 구문은 다음과 같습니다:

```css
grid-column: <start-line> / <end-line>;
```

`<start-line>`은 항목이 시작하는 그리드 선이고 `<end-line>`은 항목이 끝나는 그리드 선입니다. 두 값 모두 1부터 시작하는 1-인덱스입니다.

행에 대한 그리드 선은 `grid-template-rows` 속성에 지정된 행 수를 기준으로 생성된다는 점을 기억하세요. 열도 `grid-template-columns` 속성에 따라 동일하게 적용됩니다.

실용적인 예제로, `grid-template-rows`와 `grid-template-columns` 속성을 모두 사용할 때 브라우저가 생성하는 선부터 시작해 보겠습니다.

다음은 4개의 열과 3개의 행이 있는 그리드 예제입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}
```

:::

그리드 컨테이너(`grid` 클래스)를 살펴보면 각 행과 열은 두 개의 선으로 경계가 나뉘어 있습니다 – 행이나 열의 시작 부분에 시작 선, 끝 부분에 끝 선이 있습니다.

이 선들을 `grid-row`와 `grid-column` 속성을 사용해 타겟팅하여 항목을 어디에 배치할지 결정할 수 있습니다.

첫 번째 그리드 항목이 처음 두 열을 차지하도록 만드는 방법은 다음과 같습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
}
```

:::

이렇게 하면 첫 번째 그리드 항목이 열 1에서 시작해 열 3 바로 전까지 끝나도록 지정하는 것입니다.

첫 번째 항목은 이제 두 열을 차지하고, 네 번째 항목은 두 번째 행으로 밀려납니다. 멋지죠!

첫 번째 항목이 두 행을 차지하게 하려면 `grid-row`를 `1 / 3`로 지정할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
  grid-row: 1 / 3;
}
```

:::

또한 `span` 키워드를 사용해 그리드 항목이 어느 행과 열을 차지할지 지정할 수 있습니다. 예를 들어, `1 / 3`는 `1 / span 2`와 같습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / span 2;
  grid-row: 1 / span 2;
}
```

:::

이 기법을 그리드의 어떤 항목에도 계속 적용해 원하는 위치에 배치할 수 있습니다.

계속해서, 항목들로 마손리 레이아웃을 만든 방법은 다음과 같습니다.

새로운 예제입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(4, 100px); /* 4 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / span 2;
}

.item4 {
  grid-column: 1 / span 3;
}

.item6 {
  grid-column: 1 / span 2;
}

.item7 {
  grid-column: 3 / span 2;
}
```
