# --description--

Für feste ganze Zahlen $a$, $b$, $c$ definieren Sie die verrückte Funktion $F(n)$ wie folgt:

$$\begin{align}
  & F(n) = n - c \\;\text{ für alle } n > b \\\\
  & F(n) = F(a + F(a + F(a + F(a + n)))) \\;\text{ für alle } n ≤ b.
\end{align}$$

Definieren Sie außerdem $S(a, b, c) = \displaystyle\sum_{n = 0}^b F(n)$.

Zum Beispiel, wenn $a = 50$, $b = 2000$ und $c = 40$, dann ist $F(0) = 3240$ und $F(2000) = 2040$. Außerdem gilt $S(50, 2000, 40) = 5\\,204\\,240$.

Finden Sie die letzten 9 Ziffern von $S({21}^7, 7^{21}, {12}^7)$.
