# --description--

Impara `variables`, `data types` e `conditional statements` nella programmazione.

Un `variable` è come un riquadro dove puoi archiviare informazioni. Queste informazioni possono essere numeri, testo o altri tipi di dati. Puoi cambiare ciò che c’è nel riquadro, per questo si chiama `variable` – perché può variare. Per esempio, in `x = 5`, `x` contiene temporaneamente il valore 5.

Un `data type` è una classificazione che specifica il tipo di dati che un `variable` può contenere. I `data types` comuni includono numeri interi (numeri interi), float (numeri con una parte decimale), stringhe (testo) e booleani (vero o falso). Per esempio, se `age = 21`, il `data type` è un numero intero.

Un `conditional statement` è un’istruzione che dice al computer di fare qualcosa solo se una condizione specifica è vera. È come prendere una decisione nella vita reale basandosi su una certa situazione. I `conditional statements` comuni sono le istruzioni `if-else`. Per esempio, `if (temperature > 20) {console.log("It's warm outside!");}` è un `conditional statement` che esegue `console.log("It's warm outside!")` se la temperatura è sopra i 20 gradi.
