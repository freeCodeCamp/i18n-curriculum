# --explanation--

`To set goals` 意味著定義要達成的具體目標或標的。例如：

`The team set goals to improve customer satisfaction this year.`－這表示團隊訂立了與客戶滿意度相關的目標。

`Quarter` 指一年中的四分之一，或三個月的期間。在商務背景關係中，`the next quarter` 意指日曆年或財政年度中即將到來的三個月期間。舉例來說：

- `Our sales target for the next quarter is $1 million。`－這表示標的適用於接下來的三個月期間。

- `We review our performance at the end of each quarter。`－這是指每三個月評估進度。
