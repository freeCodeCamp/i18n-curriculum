## リストのスタイリング

- **`line-height` プロパティ**: このプロパティはテキストの行間にスペースを作るために使います。許容される`line-height`の値にはキーワード`normal`、数字、パーセンテージ、`em`単位のような長さの単位が含まれます。
- **`list-style-type` プロパティ**: このプロパティはリスト項目のマーカーを指定するために使います。許容される値にはサークル、ディスク、または10進数などがあります。

`list-style-type`を使って箇条書きのスタイルを変える例を示します：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
ul {
  list-style-type: square;
}
```

:::

- **`list-style-position` プロパティ**: このプロパティはリストマーカーの位置を設定するために使います。許容される値はinsideとoutsideの2つだけです。

`inside`と`outside`の違いを示す例はこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="inside-list">
  <li>Item with inside position</li>
  <li>Item with inside position</li>
</ul>
<ul class="outside-list">
  <li>Item with outside position</li>
  <li>Item with outside position</li>
</ul>
```

```css
.inside-list {
  list-style-position: inside;
}

.outside-list {
  list-style-position: outside;
}
```

:::

- **`list-style-image` プロパティ**: このプロパティはリスト項目のマーカーに画像を使うために使います。一般的な使い方は、`url`関数を使い、有効な画像の場所を値として設定することです。

## `margin`を使ったリスト項目の間隔調整

- `line-height`以外にも、CSSではマージンを使ってリスト項目の間隔や読みやすさを向上させることができます。
- マージンは各`li`要素の外側にスペースを作り、リスト項目間の隙間を調整できます。
- `margin-bottom`は各リスト項目の下にスペースを作るために使います。例えば、`margin-bottom: 10px;`は各リスト項目の下に10ピクセルの隙間を作ります。

`margin-bottom`を使ってリスト項目の間隔を調整する例を示します：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="list">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
.list li {
  margin-bottom: 20px;
}
```

:::

## リンクのスタイリング

- **`pseudo-class`**: これはセレクターに追加するキーワードで、特定の状態に基づいて要素を選択できます。一般的な状態には`:hover`、`:visited`、`:focus`があります。
- **`:link pseudo-class`**: この疑似クラスはユーザーがまだ訪れていないリンクをスタイリングするために使います。

`:link`疑似クラスを使った例はこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:link {
  color: red;
}
```

:::

- **`:visited pseudo-class`**: この疑似クラスはユーザーがすでに訪れたリンクをスタイリングするために使います。
- **`:hover pseudo-class`**: この疑似クラスはユーザーが要素にマウスオーバーしている状態をスタイリングするために使います。

`:hover`疑似クラスを使った例はこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Hover over me!</a>
```

```css
a:hover {
  color: green;
  text-decoration: underline;
}
```

:::

- **`:focus pseudo-class`**: この疑似クラスは要素がフォーカスを受けたときにスタイリングするために使います。例としては、クリックやタブでフォーカスを受けた`input`や`select`要素があります。

`:focus`疑似クラスを使った例はこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:focus {
  outline: 2px solid orange;
}
```

:::

- **`:active pseudo-class`**: この疑似クラスはユーザーが要素をアクティブにしたときにスタイリングするために使います。一般的な例はユーザーがボタンをクリックしたときです。

`:active`疑似クラスを使った例はこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Click me!</a>
```

```css
a:active {
  color: pink;
}
```

:::

## 背景とボーダーの操作

- **`background-size` プロパティ**: このプロパティは要素の背景サイズを設定するために使います。よく使われる値には、背景画像が要素全体を覆う`cover`や、背景画像が要素内に収まる`contain`があります。

`background-size: cover`を使った例はこちらです：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

`background-size: contain`を使った例はこちらです：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **`background-repeat` プロパティ**: このプロパティは背景画像が水平方向と垂直方向にどのように繰り返されるかを決定します。`background-repeat`のデフォルト値は`repeat`で、画像が水平方向と垂直方向の両方で繰り返されます。繰り返しなしを指定する場合は`no-repeat`プロパティを使います。

`background-repeat: no-repeat`を使った例はこちらです：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **`background-position` プロパティ**: このプロパティは背景画像の位置を指定します。特定の長さ、パーセンテージ、または`top`、`bottom`、`left`、`right`、`center`のようなキーワード値を設定できます。

`background-position`を使った例はこちらです：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

- **`background-attachment` プロパティ**: このプロパティは背景画像がコンテンツと一緒にスクロールするか、画面上で固定されるかを指定します。主な値は`scroll`（デフォルト）で、背景画像がコンテンツと一緒にスクロールし、`fixed`では背景画像が画面上の同じ位置に固定されます。
- **`background-image` プロパティ**: このプロパティは要素の背景画像を設定します。複数の背景画像を同時に設定でき、値として`url`、`radial-gradient`、`linear-gradient`関数のいずれかを使えます。

`background-image`を使った例はこちらです：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **`background` プロパティ**: これはすべての背景プロパティを1つの宣言で設定する省略形プロパティです。背景画像を設定し、繰り返しなしにする例は`background: no-repeat url("example-url-goes-here");`です。

`background`省略形プロパティを使った例はこちらです：

:::interactive_editor

```html
<style>
  body {
    background: center top no-repeat url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **背景と前景色の良好なコントラスト**: テキストを読みやすくするために、背景色と前景色のコントラストが良好であることが重要です。ウェブコンテンツアクセシビリティガイドライン（WCAG）は、通常のテキストに対して4.5:1、見出しなどの大きなテキストに対して3:1の最小コントラスト比を推奨しています。

## ボーダー

- **`border-top` プロパティ**: このプロパティは要素の上側ボーダーのスタイルを設定します。`border-top: 3px solid blue;`は要素の上側に3ピクセル幅の青色の実線ボーダーを設定します。
- **`border-right` プロパティ**: このプロパティは要素の右側ボーダーのスタイルを設定します。`border-right: 2px solid red;`は要素の右側に2ピクセル幅の赤色の実線ボーダーを設定します。
- **`border-bottom` プロパティ**: このプロパティは要素の下側ボーダーのスタイルを設定します。`border-bottom: 1px dashed green;`は要素の下側に1ピクセル幅の緑色の破線ボーダーを設定します。
- **`border-left` プロパティ**: このプロパティは要素の左側ボーダーのスタイルを設定します。`border-left: 4px dotted orange;`は要素の左側に4ピクセル幅のオレンジ色の点線ボーダーを設定します。

個別のボーダープロパティを使った例はこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="bordered-box">Box with different borders</div>
```

```css
.bordered-box {
  border-top: 3px solid blue;
  border-right: 2px solid red;
  border-bottom: 1px dashed green;
  border-left: 4px dotted orange;
  padding: 20px;
}
```

:::

- **`border` プロパティ**: これは要素のボーダーの幅、スタイル、色を一括で設定する省略形プロパティです。`border: 1px solid black;`は1ピクセル幅の黒色の実線ボーダーを設定します。

`border`省略形プロパティを使った例はこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

- **`border-radius` プロパティ**: このプロパティは要素のボーダーに丸みをつけるために使います。

`border-radius`を使った例はこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

- **`border-style` プロパティ**: このプロパティは要素のボーダースタイルを設定します。許容される値には`solid`、`dashed`、`dotted`、`double`などがあります。

異なる`border-style`の値を使った例はこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="solid-border">Solid border</div>
<div class="dashed-border">Dashed border</div>
<div class="dotted-border">Dotted border</div>
<div class="double-border">Double border</div>
```

```css
.solid-border {
  border: 3px solid blue;
  margin-bottom: 10px;
  padding: 10px;
}

.dashed-border {
  border: 3px dashed red;
  margin-bottom: 10px;
  padding: 10px;
}

.dotted-border {
  border: 3px dotted green;
  padding: 10px;
}

.double-border {
  border: 3px double blueviolet;
  padding: 10px;
}
```

:::

## グラデーション

- **`linear-gradient()` 関数**: このCSS関数は複数の色の間で直線的な移行を作成するために使います。

`linear-gradient()`を使った例はこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

- **`radial-gradient()` 関数**: このCSS関数は特定の点から放射状に広がる画像を作成し、複数の色の間で徐々に移行します。円形や楕円形のような形状です。

`radial-gradient()`を使った例はこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```
