# --explanation--

`user interface`（しばしば`UI`と略される）は、デジタル製品の視覚的かつインタラクティブな部分で、ユーザーがクリックしたり見たり触れたりするものです。例えば：

`The user interface includes the navigation bar and search field.` — ユーザーが見るアプリの部分には、メニューや検索ボックスなどがあります。

`consume`は、時間やエネルギー、予算などを使い果たすことを意味します。例えば：

`Testing often consumes a large part of the schedule.` — テストは通常、プロジェクト計画の中で多くの時間を要します。

アリスは、UIデザインとフロントエンドのコーディングが予算の27％を使うと説明しています。これはかなりの割合です。
