# --interactive--

Beim Arbeiten mit Strings in JavaScript gibt es viele Situationen, in denen Sie die Groß- und Kleinschreibung des Texts anpassen müssen, zum Beispiel um alle Buchstaben für eine Überschrift in Großbuchstaben umzuwandeln oder Text für Einheitlichkeit in Kleinbuchstaben zu konvertieren.

Glücklicherweise macht JavaScript dies mit zwei eingebauten Methoden einfach: `toUpperCase()` und `toLowerCase()`.

Die `toUpperCase()`-Methode wandelt alle Zeichen in Großbuchstaben um und gibt einen neuen String mit allen Großbuchstaben zurück. Das ist nützlich, wenn Sie Text hervorheben oder Konsistenz im Format von Strings schaffen möchten.

Sehen wir uns ein Beispiel an:

:::interactive_editor

```js
let greeting = "Hello, World!";
let uppercaseGreeting = greeting.toUpperCase();
console.log(uppercaseGreeting);  // "HELLO, WORLD!"
```

:::

In diesem Code wandelt die `toUpperCase()`-Methode den gesamten String in Großbuchstaben um.

Der ursprüngliche String bleibt unverändert, weil `toUpperCase()` einen neuen String zurückgibt, anstatt den Originalstring zu verändern.

Andererseits wandelt die `toLowerCase()`-Methode alle Zeichen in einem String in Kleinbuchstaben um. Das ist hilfreich, wenn Sie Eingaben standardisieren müssen, zum Beispiel beim Vergleichen von Benutzereingaben oder bei Groß-/Kleinschreibungs-unabhängigen Prüfungen.

Sehen wir uns ein Beispiel an:

:::interactive_editor

```js
let shout = "I AM LEARNING JAVASCRIPT!";
let lowercaseShout = shout.toLowerCase();
console.log(lowercaseShout);  // "i am learning javascript!"
```

:::

Die `toLowerCase()`-Methode wandelt alle Zeichen in Kleinbuchstaben um, wodurch der String weniger „aggressiv“ wirkt, während der ursprüngliche String unverändert bleibt.

Zusammenfassend sind die `toUpperCase()`- und `toLowerCase()`-Methoden in JavaScript leistungsstarke Werkzeuge, um Strings komplett in Groß- oder Kleinbuchstaben umzuwandeln.

Diese Methoden sind besonders nützlich, um Texteingaben zu standardisieren, Groß-/Kleinschreibungs-unabhängige Vergleiche durchzuführen und Designkonsistenz sicherzustellen.

Mit diesen einfachen, aber effektiven Methoden können Sie die Textmanipulation kontrollierter und vorhersehbarer handhaben.
