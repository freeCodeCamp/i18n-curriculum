# --description--

Acha $R(a, b, c)$ iwe eneo kubwa zaidi lililofunikwa na mizunguko mitatu isiyovurugika ndani ya pembetatu yenye urefu wa pande $a$, $b$ na $c$.

Acha $S(n)$ iwe thamani ya wastani ya $R(a, b, c)$ kwa makundi yote ya nambari kamili tatu $(a, b, c)$ zenye sifa $1 ≤ a ≤ b ≤ c &lt; a + b ≤ n$.

Umepokea $S(2) = R(1, 1, 1) ≈ 0.31998$, $S(5) ≈ 1.25899$.

Tafuta $S(1803)$ ukizungusha hadi sehemu 5 nyuma ya nukta ya desimali.
