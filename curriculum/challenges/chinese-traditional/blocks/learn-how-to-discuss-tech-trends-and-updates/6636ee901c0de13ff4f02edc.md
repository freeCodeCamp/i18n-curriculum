# --description--

術語 `run code` 指的是由電腦執行或操作程式或腳本的動作。在此背景關係中，`run` 意味著啟動行程並持續執行直到程式完成其設計的功能，無論是計算數字、排序資料或其他任何操作。

「`run code without managing servers`」這個詞語是指在不需要控制代碼執行所在的實體或虛擬伺服器的情況下執行電腦程式或應用程式。這是無伺服器運算的一個關鍵面向，在這種模式中，雲端提供者管理伺服器，使開發者能夠更專注於撰寫程式碼，而非伺服器管理。
