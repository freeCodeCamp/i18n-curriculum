# --description--

`plugin`는 기존 컴퓨터 프로그램에 특정 기능을 추가하는 컴포넌트입니다. 프로그램이 `plugins`를 지원하면 맞춤 설정이 가능해집니다. 
IDE 환경에서는 `plugins`가 코딩, 디버깅, 사용자 인터페이스 향상과 같은 추가 기능을 제공할 수 있습니다.

예를 들어, IDE에서 코드 포매터는 종종 `plugin`입니다.

# --questions--

## --text--

프로그래밍 맥락에서 `plugin`는 무엇을 의미하나요?

## --answers--

### --feedback--

`plugin`는 주요 소프트웨어가 아니라, 주요 소프트웨어의 기능을 확장하거나 향상하는 애드온입니다.

---

컴퓨터 바이러스의 한 종류입니다.

### --feedback--

`plugin`는 합법적인 소프트웨어 컴포넌트이며 바이러스가 아닙니다. 소프트웨어에 기능을 추가하기 위한 것입니다.

---

기존 프로그램에 기능을 추가하는 소프트웨어 컴포넌트입니다.

---

다른 소프트웨어 없이 독립적으로 작동하는 애플리케이션입니다.

### --feedback--

`plugin`는 독립형이 아니며, 다른 프로그램이 있어야 작동하고 그 프로그램에 기능을 추가합니다.
