# --explanation--

`It's`는 이 문맥에서 `it has`의 축약형으로, 어떤 일이 발생했거나 경험되었음을 나타낼 때 사용합니다.

`Issues`는 문제나 도전 과제를 가리킵니다. 문장에서는 시스템이 겪고 있는 어려움을 강조합니다.

단어 `connecting`는 동사 `-ing`의 `connect` 형태입니다. 무언가를 연결하거나 잇는 동작을 설명할 때 사용합니다. 예를 들어:

`The engineer is connecting the wires to complete the circuit.` - 전선을 연결하는 진행 중인 동작을 설명합니다.
