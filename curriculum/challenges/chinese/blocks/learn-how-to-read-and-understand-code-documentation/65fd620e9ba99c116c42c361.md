# --description--

开发者经常使用 `once` 来表示工作流中任务的时机。它的意思是 `as soon as` 或 `when`。

例如，`Once you receive the client's feedback, update the project documentation.` 这意味着在收到反馈后应立即开始更新文档，显示项目生命周期中操作的序列。
