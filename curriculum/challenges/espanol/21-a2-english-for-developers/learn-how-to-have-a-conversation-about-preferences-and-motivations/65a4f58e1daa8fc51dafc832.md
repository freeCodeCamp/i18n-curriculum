# --description--

`Which` se utiliza al preguntar sobre uno o más elementos de un conjunto establecido de opciones. Por ejemplo, `Which computer do you prefer?` pregunta por una elección de computadoras conocidas.

`What` es más abierto, se utiliza cuando cualquier respuesta es posible, como `What do you like to do?`

`Absolute` significa completo o total. Se usa para enfatizar que algo es tanto como puede ser. Por ejemplo, `My absolute favorite game is chess,` significa que el ajedrez es el favorito superior, sin ninguna duda.
