# --explanation--

ジェームズは`Present Perfect`時制（`has been`）を使っています。これは、過去に始まって現在も続いている問題を説明しているためです。もし`Communication was a challenge`を使っていたら、その問題は完全に過去のものだと示唆します。次の文を比べてみてください：

- `The new software has been difficult to use.` - 過去に難しくなり、今もそうである。

- `The new software was difficult to use.` - 以前は難しかったが、今はそうでないかもしれない。
