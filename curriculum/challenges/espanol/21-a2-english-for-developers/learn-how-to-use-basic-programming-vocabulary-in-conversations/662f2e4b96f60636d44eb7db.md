# --description--

A `class` is a fundamental concept, especially in `object-oriented programming`. Actúa como un plano para crear objetos. Un `class` define las propiedades (atributos) y comportamientos (`methods`) que los objetos creados a partir del `class` pueden tener.

Por ejemplo, una `class` `Person` podría tener propiedades como `name` y `age`, y comportamientos como `talk()` y `walk()`.

Un `method` es una función que está asociada con un objeto. Los `methods` definen los comportamientos o acciones que un objeto creado a partir de un `class` puede realizar. Son como instrucciones que le dicen al objeto qué hacer.

Por ejemplo, en una `Calculator class`, un `method` podría ser `add(number1, number2)`, que suma dos números.
