`Graphics driver`는 컴퓨터가 이미지와 비디오를 어떻게 표시하는지 제어하는 소프트웨어입니다. 예를 들어:

`The graphics driver manages how images appear on your screen.` – 이 말은 드라이버가 시각적 콘텐츠 표시를 제어하는 데 도움을 준다는 뜻입니다.

`UI framework`는 애플리케이션의 시각적 요소를 만드는 데 사용되는 구조나 도구 모음을 뜻합니다. 예를 들어:

`The UI framework helps developers create user-friendly interfaces.` – 이 말은 메뉴, 버튼, 그리고 사용자가 상호작용하는 프로그램의 다른 부분을 만드는 구성 요소를 제공한다는 뜻입니다.
