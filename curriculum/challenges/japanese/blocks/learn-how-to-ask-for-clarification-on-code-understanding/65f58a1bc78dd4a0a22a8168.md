# --description--

`Encounter` は、特に過去形の `encountered` で、通常は予期せず何かに出くわすことを意味します。例えば `I encountered an error while testing the software.` のように、探していないものを見つけることを示します。

`Pinpoint` は、何かを非常に正確に特定することを意味します。例えば `After reviewing the code, I was able to pinpoint the error.` のように、問題がどこにあり何であるかを正確に見つけることです。

コーディングでは、問題を見つけることはしばしば `encountering` 予期しない問題に直面し、その原因を `pinpoint` ことに関わります。
