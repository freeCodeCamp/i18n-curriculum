# --description--

マークダウンでイタリック体のテキストが含まれている可能性がある文字列が与えられたとき、対応するHTML文字列を返してください。

- マークダウンのイタリック体のテキストは、単一のアスタリスク（`*`）または単一のアンダースコア（`_`）で始まり終わる任意のテキストです。
- テキストとアスタリスクまたはアンダースコアの間にスペースは入れられませんが、テキスト内にはスペースがあってもかまいません。
- すべてのイタリック体の部分をHTMLの`i`タグに変換して文字列を返してください。

たとえば、`"*This is italic*"`が与えられたら、`"<i>This is italic</i>"`を返します。

注意：コンソールは文字列内のHTMLタグをログに表示しない場合があります。タグを含むログを確認するにはブラウザのコンソールをチェックしてください。

# --hints--

`parseItalics("*This is italic*")`は`"<i>This is italic</i>"`を返すべきです。

```js
assert.equal(parseItalics("*This is italic*"), "<i>This is italic</i>");
```

`parseItalics("_This is also italic_")`は`"<i>This is also italic</i>"`を返すべきです。

```js
assert.equal(parseItalics("_This is also italic_"), "<i>This is also italic</i>");
```

`parseItalics("*This is not italic *")`は`"*This is not italic *"`を返すべきです。

```js
assert.equal(parseItalics("*This is not italic *"), "*This is not italic *");
```

`parseItalics("_ This is also not italic_")`は`"_ This is also not italic_"`を返すべきです。

```js
assert.equal(parseItalics("_ This is also not italic_"), "_ This is also not italic_");
```

`parseItalics("The *quick* brown fox _jumps_ over the *lazy* dog.")`は`"The <i>quick</i> brown fox <i>jumps</i> over the <i>lazy</i> dog."`を返すべきです。

```js
assert.equal(parseItalics("The *quick* brown fox _jumps_ over the *lazy* dog."), "The <i>quick</i> brown fox <i>jumps</i> over the <i>lazy</i> dog.");
```
