# --interactive--

JavaScript에서 배열은 숫자 인덱스로 식별되는 값들의 순서 있는 모음입니다. JavaScript 배열의 값들은 숫자, 문자열, 불리언, 객체, 심지어 다른 배열 등 다양한 데이터 타입일 수 있습니다.

JavaScript에서 배열을 만들려면 대괄호를 사용하고 `[]` 안에 값을 쉼표로 구분하여 넣으세요. 예를 들어 다음과 같습니다:

```js
let fruits = ["apple", "banana", "orange"];
```

이 예제에서는 변수 `fruits` 을 선언하고 세 개의 문자열 값 `apple`, `banana`, `orange` 를 포함하는 배열을 할당했습니다.

배열의 주요 특징 중 하나는 0부터 시작하는 인덱스를 가진다는 점입니다. 즉, 배열의 첫 번째 요소는 인덱스 `0` 를 가지며, 두 번째 요소는 인덱스 `1` 를 갖습니다. 배열의 개별 요소는 인덱스를 사용해 접근할 수 있습니다. 예를 들어:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits[0]); // "apple"
console.log(fruits[2]); // "orange"
```

:::

이 예제에서는 인덱스 `0` 를 사용해 첫 번째 요소(`apple`)에 접근하고, 인덱스 `2` 를 사용해 세 번째 요소(`orange`)에 접근했습니다.

JavaScript 배열에는 배열에 포함된 요소 수를 반환하는 특별한 `length` 속성이 있습니다. 이 속성은 `length` 키워드를 사용해 접근할 수 있습니다. 예를 들어:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits.length); // 3
```

:::

JavaScript 배열의 또 다른 중요한 특징은 동적이라는 점입니다. 즉, 배열이 생성된 후에도 크기가 변경될 수 있습니다. `push()`, `pop()`, `shift()`, `unshift()`, `splice()` 등 다양한 배열 메서드를 사용해 배열에 요소를 추가하거나 제거할 수 있습니다. 이 메서드들은 앞으로 수업에서 배울 예정입니다.

JavaScript 배열은 프로그램 내 데이터 저장에 매우 유용하고 다재다능합니다. 이 모듈을 통해 배열을 다루는 방법을 직접 경험하며 데이터 모음을 효율적으로 관리하고 조작하는 방법을 배우게 될 것입니다.
