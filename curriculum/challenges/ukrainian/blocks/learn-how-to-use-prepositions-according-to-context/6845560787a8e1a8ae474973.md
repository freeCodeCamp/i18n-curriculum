# --explanation--

`Between` означає у просторі, що розділяє дві речі. `Between ... and ...` використовується, щоб показати, що щось знаходиться посередині між двома конкретними речами. Наприклад:

`The break room is between the IT department and HR.` – Це означає, що кімната для відпочинку знаходиться у просторі, який розділяє ці дві команди.

`Short walk` означає коротку або легку відстань для ходьби. Наприклад:

`The café is just a short walk from here.` – Це означає, що туди не потрібно довго йти.
