# --explanation--

Ein `sketch` ist eine grobe Zeichnung, die verwendet wird, um eine Idee zu planen oder zu visualisieren. Zum Beispiel:

`She made a quick sketch of the website layout before designing it digitally.` – Das bedeutet, sie hat eine einfache Zeichnung erstellt, bevor sie mit dem eigentlichen Design begonnen hat.

Ein `wireframe` ist eine einfache Skizze oder ein Entwurf eines digitalen Designs, der die Struktur ohne Details zeigt. Zum Beispiel:

`The team created wireframes to plan the website's navigation before adding colors and images.` – Das bedeutet, sie haben zuerst eine grundlegende Struktur entworfen, bevor sie an den visuellen Details gearbeitet haben.
