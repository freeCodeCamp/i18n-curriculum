# --description--

Слово `super` часто неформально використовується перед прикметником, щоб підсилити його значення, подібно до `very` або `extremely`, щоб посилити значення прикметника, що йде після.

Наприклад, сказати `this book is super interesting` означає, що книга надзвичайно цікава, більше ніж просто цікава.

`Helpful` описує когось або щось, що надає цінну допомогу або підтримку. Разом `super helpful` означає надання виняткової допомоги або бути надзвичайно корисним.
