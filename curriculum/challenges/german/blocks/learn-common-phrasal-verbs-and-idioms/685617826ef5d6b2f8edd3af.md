# --explanation--

Eine `warning` ist eine Nachricht oder ein Zeichen, das Ihnen mitteilt, dass Gefahr oder etwas nicht in Ordnung sein könnte. Zum Beispiel:

`The system shows a warning when your password is too weak.` – Das bedeutet, dass eine Nachricht angezeigt wird, die besagt, dass Ihr schwaches Passwort unsicher ist.

Etwas `suspicious` erscheint unehrlich, gefährlich oder ungewöhnlich auf eine Weise, die Sie unsicher oder besorgt macht. Zum Beispiel:

`There was suspicious activity in the login history.` – Das bedeutet, dass etwas Seltsames oder möglicherweise Schädliches bemerkt wurde.
