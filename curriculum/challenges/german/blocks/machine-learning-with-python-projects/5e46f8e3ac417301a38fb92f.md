# --instructions--

In dieser Aufgabe erstellen Sie einen Buch-Empfehlungsalgorithmus mit **K-Nearest Neighbors**.

In diesem Projekt verwenden Sie den Book-Crossings-Datensatz, der 1,1 Millionen Bewertungen (Skala von 1 bis 10) von 270.000 Büchern durch 90.000 Nutzer enthält. Der Datensatz ist bereits im Notebook importiert, daher ist kein zusätzlicher Download erforderlich.

Verwenden Sie `NearestNeighbors` aus `sklearn.neighbors`, um ein Modell zu entwickeln, das Bücher zeigt, die einem gegebenen Buch ähnlich sind. Der Nearest-Neighbors-Algorithmus misst die Distanz, um die „Nähe“ von Instanzen zu bestimmen.

Erstellen Sie eine Funktion namens `get_recommends`, die einen Buchtitel (aus dem Datensatz) als Argument nimmt und eine Liste von 5 ähnlichen Büchern mit deren Distanzen zum Buch-Argument zurückgibt.

Dieser Code:

```py
get_recommends("The Queen of the Damned (Vampire Chronicles (Paperback))")
```

sollte Folgendes zurückgeben:

```py
[
  'The Queen of the Damned (Vampire Chronicles (Paperback))',
  [
    ['Catch 22', 0.793983519077301], 
    ['The Witching Hour (Lives of the Mayfair Witches)', 0.7448656558990479], 
    ['Interview with the Vampire', 0.7345068454742432],
    ['The Tale of the Body Thief (Vampire Chronicles (Paperback))', 0.5376338362693787],
    ['The Vampire Lestat (Vampire Chronicles, Book II)', 0.5178412199020386]
  ]
]
```

Beachten Sie, dass die von `get_recommends()` zurückgegebenen Daten eine Liste sind. Das erste Element in der Liste ist der an die Funktion übergebene Buchtitel. Das zweite Element in der Liste ist eine Liste von fünf weiteren Listen. Jede dieser fünf Listen enthält ein empfohlenes Buch und die Distanz vom empfohlenen Buch zum übergebenen Buch.

Wenn Sie den Datensatz grafisch darstellen (optional), werden Sie feststellen, dass die meisten Bücher nicht häufig bewertet werden. Um statistische Signifikanz sicherzustellen, entfernen Sie aus dem Datensatz Nutzer mit weniger als 200 Bewertungen und Bücher mit weniger als 100 Bewertungen.

Die ersten drei Zellen importieren Bibliotheken, die Sie benötigen könnten, und die zu verwendenden Daten. Die letzte Zelle dient zum Testen. Schreiben Sie Ihren gesamten Code zwischen diesen Zellen.
