# --description--

现在是学习 70 到 79 的数字的时候了。

在西班牙语中，数字 70 被称为 `setenta`。

这些数字是通过使用表示十位数的单词 `setenta`（70），后跟连词 `y`（“和”），然后是个位数字（1-9）组成的。

| Number | Spanish |
| :--- | :--- |
| 70 | `Setenta` |
| 71 | `Setenta y uno` |
| 72 | `Setenta y dos` |
| 73 | `Setenta y tres` |
| 74 | `Setenta y cuatro` |
| 75 | `Setenta y cinco` |
| 76 | `Setenta y seis` |
| 77 | `Setenta y siete` |
| 78 | `Setenta y ocho` |
| 79 | `Setenta y nueve` |

数字 `setenta y uno` (71) 在这个范围内是独特的，因为它在阳性名词前变成 `setenta y un`。例如：

`Setenta y un años` - 七十一年。
