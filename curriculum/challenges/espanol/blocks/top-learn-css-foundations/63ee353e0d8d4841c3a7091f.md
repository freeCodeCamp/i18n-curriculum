# --description--

Bien, has avanzado bastante hasta ahora. Lo único que queda por hacer es explicar cómo agregar todo este CSS a tu HTML. Hay tres métodos para hacerlo.

El CSS externo es el método más común que encontrarás, y consiste en crear un archivo separado para el CSS y enlazarlo dentro de las etiquetas `<head>` de apertura y cierre de un HTML con un elemento `<link>` (que es un elemento vacío y no requiere una etiqueta de cierre):

Primero, agregas una etiqueta de elemento vacío `<link>` dentro de las etiquetas de apertura y cierre `<head>` del archivo HTML. El atributo `href` es la ubicación del archivo CSS, ya sea una URL absoluta o, lo que utilizarás, una URL relativa a la ubicación del archivo HTML. En el ejemplo anterior, asumes que ambos archivos están ubicados en el mismo directorio. El atributo `rel` es obligatorio y especifica la relación entre el archivo HTML y el archivo vinculado.

Luego, dentro del archivo recién creado `styles.css`, tienes el selector (el `div` y `p`), seguido de un par de llaves de apertura y cierre, que crean un “bloque de declaraciones”. Finalmente, colocas cualquier declaración dentro del bloque de declaraciones. `color: white;` es una declaración, donde `color` es la propiedad y `white` es el valor, y `background-color: black;` es otra declaración.

Una nota sobre los nombres de archivo: `styles.css` es simplemente el nombre que elegiste para el archivo aquí. Puedes nombrar el archivo como quieras siempre que el tipo de archivo sea `.css`, aunque “style” o “styles” es lo más común.

Un par de las ventajas de este método son:

1. Mantiene tu HTML y CSS separados, lo que resulta en que el archivo HTML sea más pequeño y que todo se vea más limpio.
2. Solo necesitas editar el CSS en un solo lugar, lo cual es especialmente útil para sitios web con muchas páginas que comparten estilos similares.
