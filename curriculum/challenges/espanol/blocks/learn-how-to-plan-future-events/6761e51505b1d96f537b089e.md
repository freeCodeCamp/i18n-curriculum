# --explanation--

`Going to` se usa para hablar de acciones planificadas o intenciones para el futuro. Por ejemplo:

- `We are going to upgrade the system next month.` - Esto significa que la actualización está planeada y ocurrirá pronto.
  
- `She is going to learn SQL next year.` - Esto significa que ella tiene la intención de aprender SQL en el futuro.

En esta oración:

`We're` es la contracción de `we are`.

`Going to` indica que la acción está planeada para un futuro cercano.

`Need` es el verbo base que indica algo necesario o requerido.
