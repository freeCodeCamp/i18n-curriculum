# --description--

Eine der häufigsten Methoden, um ein neues Element zu einem Array hinzuzufügen, ist die Verwendung der `push()`-Methode. Die `push()`-Methode fügt ein oder mehrere Elemente am Ende eines Arrays hinzu und gibt die neue Länge des Arrays zurück.

Zum Beispiel können Sie, um ein neues Element zum `pet`-Array hinzuzufügen, den folgenden Code verwenden:

```javascript
const pet = ['cat', 'dog', 'bunny'];
pet.push('parrot');
console.log(pet); // Output: ['cat', 'dog', 'bunny', 'parrot']
```

Um das letzte Element eines Arrays zu entfernen, können Sie die `pop()`-Methode verwenden. Die `pop()`-Methode entfernt das letzte Element aus einem Array und gibt dieses Element zurück.

Zum Beispiel können Sie, um das letzte Element aus dem `pet`-Array zu entfernen, den folgenden Code verwenden:

```javascript
const pet = ['cat', 'dog', 'tiger'];
pet.pop();
console.log(pet); // Output: ['cat', 'dog']
```
