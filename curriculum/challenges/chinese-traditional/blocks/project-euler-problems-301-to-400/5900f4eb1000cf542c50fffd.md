# --description--

多邊形是一種由直線線段組成的平面形狀，這些線段連接起來形成封閉的串鏈或迴路。多邊形至少有三條邊，且不會自我相交。

一組正數 $S$ 被稱為產生多邊形 $P$，如果：

- $P$ 的任兩邊長度皆不相同，
- $P$ 的每一邊長度都在 $S$ 中，且
- $S$ 不包含其他值。

例如：

集合 {3、4、5} 會產生一個邊長為 3、4 和 5 的多邊形（三角形）。

集合 {6、9、11、24} 會產生一個邊長為 6、9、11 和 24（四邊形）的多邊形。

集合 {1, 2, 3} 和 {2, 3, 4, 9} 完全不會產生任何多邊形。

考慮序列 $s$，定義如下：

- $s_1 = 1$、$s_2 = 2$、$s_3 = 3$
- $s_n = s_{n - 1} + s_{n - 3}$，當 $n > 3$。

設 $U_n$ 為集合 $\\{s_1, s_2, \ldots, s_n\\}$。例如，$U_{10} = \\{1, 2, 3, 4, 6, 9, 13, 19, 28, 41\\}$。

令 $f(n)$ 為產生至少一個多邊形的 $U_n$ 子集合數量。

例如，$f(5) = 7$、$f(10) = 501$ 和 $f(25) = 18\\,635\\,853$。

找出 $f({10}^{18})$ 的最後 9 位數。
