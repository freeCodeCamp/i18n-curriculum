# --description--

양의 정수 $A$가 다음과 같은 정수 $p$, $q$, $r$가 존재할 때 "알렉산드리아 정수"라고 부릅니다:

$$A = p \times q \times r$$

그리고

$$\frac{1}{A} = \frac{1}{p} + \frac{1}{q} + \frac{1}{r}$$


예를 들어, 630은 알렉산드리아 정수입니다 ($p = 5$, $q = −7$, $r = −18$). 실제로 630은 6번째 알렉산드리아 정수이며, 처음 6개의 알렉산드리아 정수는 6, 42, 120, 156, 420, 630입니다.

150000번째 알렉산드리아 정수를 찾으세요.
