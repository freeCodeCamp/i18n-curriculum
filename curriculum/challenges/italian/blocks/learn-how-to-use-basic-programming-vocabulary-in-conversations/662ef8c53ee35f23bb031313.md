# --description--

L'espressione `a lot of` può essere usata sia con nomi numerabili sia con nomi non numerabili, per indicare un numero o una quantità grande.

Per i nomi numerabili, come `cars`, potresti dire `a lot of cars`. Per i nomi non numerabili, come `water`, diresti `a lot of water`.
