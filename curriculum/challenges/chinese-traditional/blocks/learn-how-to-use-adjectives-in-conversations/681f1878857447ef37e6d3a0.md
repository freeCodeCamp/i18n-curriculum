# --explanation--

`Not only ... but also ...` 是一個常用的結構，用來強調兩個相關的述語都為真，且第二個述語通常更令人驚訝或更重要。 例如：

`She is not only skilled but also very creative.`－這表示她有兩個強項，且兩者都很重要。

`Adaptable` 意味著某物可以根據情況輕鬆改變或調整。 例如：

`This program is adaptable to different types of teams。`－這表示它在多種環境中都能良好運作。
