# --description--

Porque você está apenas aumentando `i` em `1`, você pode usar o <dfn>operador de incremento</dfn> `++`.Este operador aumenta o valor de uma variável em 1, atualizando a atribuição para essa variável.Por exemplo, `test` se tornaria `8` aqui:

```js
let test = 7;
test++;
```

Substitua sua atribuição de adição pelo operador de incremento para a iteração do seu loop.

Seu loop `for` deve usar o operador de incremento em `i`.
