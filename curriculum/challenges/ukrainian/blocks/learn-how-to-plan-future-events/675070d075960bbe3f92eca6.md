# --explanation--

Скорочення `We'll` означає `We will`, що вказує на майбутню дію або план. Наприклад:

- `I'll call you tomorrow.` — це означає, що дія виклику відбудеться у майбутньому.

- `We'll review the report later.` — це вказує, що повторення заплановане, але ще не відбулося.
