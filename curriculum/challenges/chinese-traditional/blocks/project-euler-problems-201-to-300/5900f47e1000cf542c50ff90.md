# --description--

考慮以下形式的方程式：$a^2 + b^2 = N$，$0 ≤ a ≤ b$，其中 $a$、$b$ 和 $N$ 為整數。

對於 $N = 65$ 有兩個解法：

$a = 1, b = 8$ 和 $a = 4, b = 7$。

我們將所有滿足 $a^2 + b^2 = N$、$0 ≤ a ≤ b$、且 $a$、$b$ 和 $N$ 為整數的解法中 $a$ 的值加總稱為 $S(N)$。

因此 $S(65) = 1 + 4 = 5$。

求所有只被形如 $4k + 1$ 且 $4k + 1 < 150$ 的質數整除的無平方因子 $N$ 的 $\sum S(N)$。
