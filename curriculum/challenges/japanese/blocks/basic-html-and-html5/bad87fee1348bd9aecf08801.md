# --description--

HTML5はより説明的なHTMLタグを導入します。これには`main`、`header`、`footer`、`nav`、`video`、`article`、`section`などが含まれます。

これらのタグはHTMLに説明的な構造を与え、HTMLを読みやすくし、検索エンジン最適化（SEO）やアクセシビリティに役立ちます。`main`のHTML5タグは、検索エンジンや他の開発者がページの主要コンテンツを見つけやすくします。

使用例として、2つの子要素がネストされた`main`要素があります。

```html
<main> 
  <h1>Hello World</h1>
  <p>Hello Paragraph</p>
</main>
```

**注意：** 多くの新しいHTML5タグとその利点は、Applied Accessibilityセクションで扱われています。

# --instructions--

2つ目の`p`要素を作成し、以下のkitty ipsumテキストを入れてください：`Purr jump eat the grass rip the couch scratched sunbathe, shed everywhere rip the couch sleep in the sink fluffy fur catnip scratched.`

次に、`main`要素を作成し、`p`要素の中に2つの`main`要素だけをネストしてください。

# --hints--

Kitty Ipsumテキストを含む`p`要素が2つあるはずです。

```js
assert.lengthOf(document.querySelectorAll('p'),2);
```

それぞれの`p`要素には終了タグが必要です。

```js
assert.match(code,/<\/p>/g);
assert.strictEqual(code.match(/<\/p>/g).length,code.match(/<p/g).length);
```

`p`要素には、提供された追加の`kitty ipsum`テキストの最初の数語を含めてください。

```js
assert.match(document.querySelectorAll('p')[1].textContent,/Purr\s+jump\s+eat/gi);
```

コードには`main`要素が1つあるべきです。

```js
assert.lengthOf(document.querySelectorAll('main'),1);
```

`main`要素には2つの段落要素が子要素として含まれているべきです。

```js
const main = document.querySelector('main');
const children = main.querySelectorAll("p"); 
assert.lengthOf(children,2);
```

最初の段落タグの前に`main`の開始タグが来るべきです。

```js
assert.match(code,(/<main>\s*?<p>/g));
```

2つ目の段落の終了タグの後に`main`の終了タグが来るべきです。

```js
assert.match(code,(/<\/p>\s*?<\/main>/g));
```
