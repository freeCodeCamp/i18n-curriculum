# --description--

現在のパターンはまだ機能しません。 `/+-\s/` looks for `+`, `-`, and a space *in order*. つまり `+- hello` にはマッチしますが、`+hello` にはマッチしません。

これらの文字それぞれに個別に一致させるためには、文字クラスを使う必要があります。 文字クラスにするには、マッチさせたい文字のセットを角括弧で囲みます。 For example, this pattern will match the characters `h`, `e`, `l`, or `o`:

```js
const regex = /[helo]/;
```

では、`+-\s` というパターンを文字クラスに変更しましょう。 Note that you no longer need to escape the `+` character, because you are using a character class.
