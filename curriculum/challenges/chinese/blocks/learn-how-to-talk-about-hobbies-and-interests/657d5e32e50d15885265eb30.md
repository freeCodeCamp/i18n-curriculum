# --description--

在 `Present Perfect` 时态中，`have` 可以在 `I`、`you`、`we` 或 `they` 之后缩写为 `'ve`，而 `has` 可以在 `he`、`she` 或 `it` 之后缩写为 `'s`。例如：

- `I've walked to the park.` - `I've` 是 `I have` 的缩写。

- `She's seen the movie.` - `She's` 是 `she has` 的缩写。
