# --description--

`Computing`는 정보를 관리하고 문제를 해결하며 다양한 작업을 수행하기 위해 컴퓨터와 컴퓨터 기술을 사용하는 과정이나 활동을 의미합니다. 이는 간단한 계산부터 복잡한 프로그래밍과 데이터 분석에 이르기까지 광범위한 활동을 포함합니다.

현대적 의미에서 `computing`는 `cloud computing`, `serverless computing` 등과 같은 주제도 포함하여 기술의 진화하는 특성을 반영합니다.
