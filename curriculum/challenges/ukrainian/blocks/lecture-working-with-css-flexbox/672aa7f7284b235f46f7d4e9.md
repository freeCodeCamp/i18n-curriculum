# --interactive--

CSS flexbox — це одномірна модель макета, яка дозволяє розташовувати елементи в рядках і стовпцях усередині контейнера. Ви також можете керувати їхнім порядком і орієнтацією. Веброзробники використовують її для створення адаптивних вебсайтів і вебзастосунків, які підлаштовуються під різні розміри екранів і орієнтації. Ми називаємо flexbox одномірною моделлю макета, бо вона зосереджена на розташуванні елементів уздовж однієї осі за раз. Ось ця вісь може бути горизонтальною або вертикальною.

Існують два ключові поняття, які варто знати перед початком роботи з flexbox: гнучкий контейнер і гнучкий елемент.

Гнучкий контейнер — це HTML-елемент із гнучким макетом. Ви можете розташовувати й вирівнювати елементи різними способами всередині гнучкого контейнера. Щоб зробити HTML-елемент гнучким контейнером, потрібно додати `display: flex` до його CSS-стилів.

Гнучкі елементи — це прямі дочірні елементи гнучкого контейнера. Ці елементи можна розташовувати й вирівнювати всередині гнучкого контейнера залежно від його властивостей. Вони також можуть стискатися або розширюватися, щоб підлаштуватися під доступний простір.

Ось приклад із `main` контейнером і трьома дочірніми `div` елементами:

```html
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

Якщо ви встановите лише `width`, `height` і `background-color` цих `div` елементів у CSS-стилях, кожен дочірній елемент розміститься на власному рядку, бо контейнер за замовчуванням не є гнучким.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Але якщо додати `display: flex` до `main` контейнера, `div` елементи будуть переставлені так, щоб поміститися в одному рядку, і вони за потреби стиснуться (щоб побачити попередній перегляд, увімкніть інтерактивний редактор):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

За замовчуванням гнучкий контейнер буде елементом блочного рівня, тому сам контейнер займатиме власний рядок відносно інших елементів і контейнерів.

Тепер, коли ви знаєте більше про гнучкі контейнери і гнучкі елементи, варто також знати про гнучкі властивості. Ці властивості визначають, як гнучкі елементи будуть розташовані, змінювати розмір і розподілятися всередині гнучкого контейнера. Найпоширеніші з них — `flex-direction`, `justify-content`, `align-items` і `flex-wrap`.

Чудово. Тепер поговоримо трохи про модель flex. Ця модель визначає, як гнучкі елементи розташовуються всередині гнучкого контейнера. Кожен гнучкий контейнер має дві осі:

- Головна вісь.
- Поперечна вісь.

Орієнтація цих осей визначає, як різні властивості впливатимуть на макет і розподіл гнучких елементів. За замовчуванням головна вісь гнучкого контейнера горизонтальна, а поперечна — вертикальна. Гнучкі елементи розташовуються в напрямку головної осі. Поперечна вісь перпендикулярна до головної.

Властивість `flex-direction` задає напрямок головної осі. За замовчуванням значення `flex-direction` — `row`, що розміщує всі гнучкі елементи в одному рядку в напрямку мови вашого браузера за замовчуванням (зліва направо або справа наліво):

```css
flex-direction: row; /* Default */
```

Щоб змінити порядок елементів у рядку на зворотний, можна використати `flex-direction: row-reverse`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: row-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Це змінить порядок гнучких елементів на зворотний. Якщо ви хочете вирівняти гнучкі елементи вертикально, потрібно лише встановити `flex-direction` у `column` у гнучкому контейнері. Це змінить напрямок головної осі:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Тепер `div` елементи будуть вирівняні вертикально, бо головна вісь стане вертикальною, а поперечна — горизонтальною.

Ви також можете змінити порядок гнучких елементів вертикально за допомогою `flex-direction: column-reverse`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

CSS flexbox — це потужна модель макета, яка забезпечує гнучкий і ефективний спосіб розташування елементів усередині контейнера. Розуміючи ключові поняття гнучких контейнерів, гнучких елементів і різних гнучких властивостей, ви зможете створювати динамічні та адаптивні вебсайти, які підлаштовуються під різні розміри екранів і орієнтації.
