# --description--

你將需要對用戶添加的條目進行編號。 要獲取所有數字輸入，你可以使用 <dfn>querySelectorAll()</dfn> 方法。

`querySelectorAll()` 方法返回與選擇器匹配的所有元素的 `NodeList`。 `NodeList` 是一個類似數組的對象，因此你可以使用括號表示法訪問其中的元素。

聲明一個 `entryNumber` 變量並賦予其 `targetInputContainer.querySelectorAll()` 的值。 你還不需要向查詢選擇器傳遞參數。
