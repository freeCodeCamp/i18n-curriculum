# --description--

doctypeを宣言した後、`<html>` 要素を指定する必要があります。これはドキュメントのルート要素と呼ばれ、ドキュメント内の他のすべての要素はこの要素の子孫要素になります。

これは後でJavaScriptでHTMLを操作する際により重要になります。今は、`html` 要素はすべてのHTMLドキュメントに含めるべきだと覚えておいてください。

`index.html` ファイルに戻り、`<html>` 要素を開くタグと終了タグを入力して追加しましょう。

```html
<!DOCTYPE html>
<html lang="en">
</html>
```

## lang属性とは何ですか？
`lang` はその要素内のテキストコンテンツの言語を指定します。この属性は主にウェブページのアクセシビリティを向上させるために使われます。例えばスクリーンリーダーなどの支援技術が言語に応じて適切な発音を行うために役立ちます。
