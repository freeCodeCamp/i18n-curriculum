# --description--

El español a menudo usa las palabras `cuál` y `cómo` para formar preguntas comunes. 

`¿Cuál ... ?` se usa para pedir **información específica**, especialmente antes del verbo `ser`. Por ejemplo:

- `¿Cuál es tu nombre?` – ¿Cuál es tu nombre?

- `¿Cuál es tu número de teléfono?` – ¿Cuál es tu número de teléfono?

`¿Cómo ... ?` se usa para preguntar **cómo** se hace, se escribe o se pronuncia algo. Por ejemplo:

- `¿Cómo te llamas?` – ¿Cómo te llamas?

- `¿Cómo se escribe?` – ¿Cómo se escribe?
