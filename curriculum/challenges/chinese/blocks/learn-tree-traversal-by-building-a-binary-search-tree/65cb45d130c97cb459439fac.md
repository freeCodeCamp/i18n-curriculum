# --description--

为了找到右子树中的最小值，你需要遍历给定节点的左子节点，直到到达子树中最左边（最小）的节点。

为此，编写一个 `while` 循环，该循环在 `node.left is not None` 时运行，并将 `pass` 移动到 `while` 块内。 此条件检查是否存在左孩子。 只要有左孩子，循环就继续，并会找到一个较小的值。
