# --description--

레이저 물리학에서 "흰색 셀"은 레이저 빔의 지연선 역할을 하는 거울 시스템입니다. 빔은 셀에 들어가 거울에 부딪히며 여러 번 반사되다가 결국 다시 밖으로 나옵니다.

이번에 다룰 특정 흰색 셀은 방정식 $4{x}^2 + y^2 = 100$인 타원입니다.

위쪽에서 $−0.01 ≤ x ≤ +0.01$ 구간에 해당하는 부분이 비어 있어 빛이 구멍을 통해 들어오고 나갈 수 있습니다.

이 문제에서 빛의 빔은 흰색 셀 바로 밖의 점 (0.0, 10.1)에서 시작하며, 빔은 처음에 (1.4, -9.6)에서 거울에 부딪힙니다.

레이저 빔이 타원의 표면에 닿을 때마다, 일반적인 반사 법칙인 "입사각과 반사각이 같다"를 따릅니다. 즉, 입사 빔과 반사 빔이 입사점에서의 법선과 같은 각도를 이룹니다.

왼쪽 그림에서 빨간 선은 레이저 빔과 흰색 셀 벽이 처음 두 번 접촉하는 점을 나타내고, 파란 선은 첫 번째 반사점에서 타원에 접하는 접선입니다.

주어진 타원의 임의 점 (x, y)에서 접선의 기울기 m은 다음과 같습니다: $m = −4 × \frac{x}{y}$

법선은 입사점에서 이 접선에 수직입니다.

오른쪽 애니메이션은 빔의 처음 10번 반사를 보여줍니다.

빔이 나가기 전에 흰색 셀 내부 표면에 몇 번 부딪히나요?
