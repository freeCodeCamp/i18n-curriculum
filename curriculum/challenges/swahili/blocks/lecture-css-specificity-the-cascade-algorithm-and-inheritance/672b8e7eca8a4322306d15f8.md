# --interactive--

Kichaguzi cha universal (`*`) ni aina maalum ya kichaguzi cha CSS kinacholingana na kipengele chochote katika hati.

Mara nyingi hutumika kuweka mtindo kwa vipengele vyote kwenye ukurasa, jambo ambalo linaweza kusaidia kwa kusahihisha au kawaisha mitindo katika vivinjari tofauti.

Kichaguzi cha universal kinaweza kutumika kuchagua vipengele vyote ndani ya muktadha maalum au kwa ujumla katika hati nzima.

Hapa kuna mfano wa kutumia kichaguzi cha universal kuweka `margin` na `padding` kwa hati nzima ya HTML:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1>Heading element</h1>
<p>example paragraph element</p>
```

```css
* {
  margin: 0;
  padding: 0;
}
```

:::

Katika mfano huu wa msimbo, kichaguzi cha `*` kinapunguza ukingo na nafasi ya ndani ya vipengele vyote kuwa sifuri, ambayo ni mbinu ya kawaida inayotumika katika kusahihisha mitindo ya CSS.

Kichaguzi cha universal kina thamani ya chini kabisa ya ufafanuzi kati ya vichaguzi vyote. Kinachangia 0 kwa sehemu zote za thamani ya ufafanuzi `(0, 0, 0, 0)`.

Hii ina maana kwamba kichaguzi kingine chochote, ikiwa ni pamoja na vichaguzi vya aina, darasa la vitu, kitambulisho, na mitindo ndani ya mstari, kitaghairi mitindo iliyowekwa na kichaguzi cha universal.

Tuchunguze mfano ufuatao wa HTML na CSS:

:::interactive_editor

```html
<head>
  <style>
    * {
      color: blue;
    }
    p {
      color: red;
    }
    .highlight {
      color: green;
    }
    #unique {
      color: purple;
    }
  </style>
</head>
<body>
  <p id="unique" class="highlight">This text has multiple styles applied.</p>
</body>
```

:::

Kichaguzi cha universal cha `*` kinaweka rangi ya maandishi kuwa `blue` kwa vipengele vyote.

Hata hivyo, kichaguzi cha aina cha `p` kinaghairi hili kwa kuweka rangi ya maandishi kuwa `red` hasa kwa vipengele vya `p`.

Ikiwa kipengele kina darasa la `highlight`, kichaguzi cha darasa la vitu kinapata kipaumbele, na kubadilisha rangi ya maandishi kuwa `green`.

Mwishowe, kichaguzi cha kitambulisho cha `#unique`, ambacho kina ufafanuzi wa juu zaidi, kitaghairi vingine vyote, na kuweka rangi ya maandishi kuwa `purple`.
