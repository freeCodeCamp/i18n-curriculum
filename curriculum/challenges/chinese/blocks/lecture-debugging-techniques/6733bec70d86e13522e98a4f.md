# --description--

JavaScript 中的 `throw` 语句用于抛出用户定义的异常。编程中的异常是指当发生意外事件并中断程序的正常流程时。

作为程序员，处理这些异常非常重要，这样当发生误差时，你的程序才不会意外崩溃。以下是 `throw` 语句的基本语法：

```js
throw expression;
```

此分支中的 `expression` 将是表现你想要抛出的异常的对象或值。示例包括内置的异常类，如 `Error`、`TypeError` 或 `RangeError` 类。以下是使用 `throw` 语句抛出 `TypeError` 的示例：

```js
function validateNumber(input) {
  if (typeof input !== "number") {
    throw new TypeError("Expected a number, but received " + typeof input);
  }
  return input * 2;
}
```

在此示例中，我们正在查看 `input` 的类型是否不是 `number` 类型。如果不是，则抛出带有自定义消息的 `TypeError`。否则，函数将返回将 `input` 乘以 2 的结果。 

如果你想抛出更通用的错误信息，那么你可以像这样引用 `Error` 构造函数：

```js
function divide(numerator, denominator) {
  if (denominator === 0) {
    throw new Error("Cannot divide by zero");
  }
  return numerator / denominator;
}
```

这是一个函数示例，用于查看分母是否为 `0`。如果是这种情况，它将抛出一个自定义错误信息，内容为 `Cannot divide by zero`。

在下一课中，我们将学习如何在用于优雅地处理 JavaScript 异常的 `try`/`catch` 块的上下文中抛出误差消息。
