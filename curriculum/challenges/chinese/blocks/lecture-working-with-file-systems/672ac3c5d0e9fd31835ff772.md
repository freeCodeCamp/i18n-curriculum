# --description--

要在 Windows 上使用资源管理器创建 `file`，你可以点击左上角的 `"New"` 选项。这样会显示你可以创建的不同类型的 `file` 列表。 

你也可以在相应的应用中创建文件。让我们创建一个文本文件来向你展示这个过程。

当你点击 "New" 并选择文件类型时，你将在当前文件夹中看到一个默认名称的新文本文件。你可以根据需要重命名它。

创建文件夹的过程完全相同。点击左上角的 `"New"` 按钮，然后选择 `"Folder"` 选项，或者右键点击空白处，进入 `"New"`，然后选择 `"Folder"` 选项。你将在当前位置看到一个新的空文件夹。

你可以通过选择文件和文件夹，然后点击顶部的 `"Rename"` 选项来重命名它们。当文件的当前名称被高亮时，你可以开始输入文件的新名称。按 Enter 键确认更改。

备选地，你可以右键点击该 `file` 或 `Folder` 并选择 `"Rename"` 选项，或者使用 F2 键盘快捷键。输入新名称并按 Enter 键确认更改。 

如果你已将目标文件夹作为标签（页）打开，可以通过拖放将文件或文件夹移动到另一个文件夹中。 

你也可以选择 `file` 或 `folder`，然后选择顶部的 `"Cut"` 选项。

然后，转到目标文件夹，点击顶部的 `"Paste"` 按钮进行粘贴。如果你之前复制或剪切了该文件，该按钮将被启用。

你也可以通过右键点击空白处，然后点击粘贴图标来完成此操作。这将从其原始位置移除该文件或文件夹。

要删除一个文件或文件夹，点击它然后点击顶部的垃圾桶图标。如果你右键点击它，也会找到一个“Delete”选项。

太好了。现在让我们看看你如何在 macOS 上使用 `Finder` 来做到这一点。 

Finder 是 macOS 上的默认文件管理器。

要在 macOS 上创建文件，你需要打开一个允许你创建所需类型文档的应用。目前，Finder 中没有内置方法可以直接创建文件。

例如，要创建一个简单的纯文本文件，你可以打开 TextEdit 并储存你的文件。该进程对任何类型的文件都完全相同。

打开 `TextEdit` 有很多方法。其中一种是进入 Launchpad。然后，查找一个可以创建你感兴趣的文件类型的应用程序。在这种情况下，就是 `TextEdit`。点击图标以打开该应用程序。

进入应用后，你应该能在 `File` 菜单中看到储存你的文件的选项。该菜单位于左上角，紧挨着 Apple 菜单图标。

在某个 App 菜单选项中，你应该看到一个用于储存文件的命令。这通常位于 `File` 菜单下。

在它旁边，你会看到保存你的文件的键盘快捷键。这通常是在 macOS 上的 `Command + S`。

当你第一次保存 `file` 时，你应该会看到一个对话框窗口，在那里你可以输入 `file` 的名称并选择位置。

通常，文件扩展名由应用自动添加，但你可以在保存文件之前更改它。

储存文件后，你可以通过点击 Dock 中的 Finder 图标打开 Finder。你应该能在你储存文件的文件夹中看到你的新文件。

要创建一个新文件夹，你应该前往想要创建该新文件夹的位置，然后右键点击空白处。你会看到一个选项列表。第一个选项是 `"New Folder"`。

如果你点击它，你可以为你的新建文件夹赋值一个名称。输入名称并按 Enter 确认。

在 Finder 中有多种方法可以移动文件和文件夹。如果你需要将它们移动到 Finder “Favorites” 部分列出的文件夹，只需拖放即可。

如果你需要将它移动到不同的文件夹，你有两个选项。 

你可以将目标文件夹作为标签（页）打开，然后将文件或文件夹拖放到该标签（页）中。

或者你可以使用键盘快捷键。首先，使用 `Command + C` 拷贝文件，进入新建文件夹，然后使用 `Command + Option + V` 将文件或文件夹粘贴到该位置。这也会将其从原始文件夹中移除。

要删除一个文件或文件夹，你可以右键点击它并选择 "Move to trash"。

你也可以将它们拖放到 Dock 上的垃圾图标中。这是等效的。

当 `file` 或 `folder` 在回收站中时，你可以右键点击它立即删除，或者你可以清空回收站以永久删除所有已删除的文件和文件夹。

了解如何在 Windows 和 macOS 上创建、移动和删除文件及文件夹非常重要。通过掌握这些技能，你可以整理你的文件，快速定位它们，并通过删除不必要的文件释放计算机上的存储空间。
