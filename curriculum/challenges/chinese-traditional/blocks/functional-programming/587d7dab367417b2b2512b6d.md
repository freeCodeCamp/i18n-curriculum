# --description--

過去幾個挑戰涵蓋了許多遵循函式程式設計原則的有用陣列和字串方法。我們也學習了 `reduce`，這是一個強大的方法，用來將問題簡化成更簡單的形式。從計算平均值到排序，任何陣列操作都可以透過應用它來完成。請記得 `map` 和 `filter` 是 `reduce` 的特殊案例。

讓我們結合所學來解決一個實際問題。

許多內容管理網站（CMS）會將文章標題添加到 URL 的部分，以便簡單地建立書籤。 例如，如果你撰寫一篇標題為 `Stop Using Reduce` 的 Medium 文章，該 URL 很可能會包含標題字串的某種形式（`.../stop-using-reduce`）。你可能已經在 freeCodeCamp 網站上注意到這一點。

# --instructions--

填寫 `urlSlug` 函式，使其將字串 `title` 轉換並傳回用於 URL 的連字號版本。你可以使用本節涵蓋的任何方法，但不要使用 `replace`。以下是需求：

輸入是一個包含空格和標題大小寫字的字串

輸出是一個字串，字與字之間的空格被 `-` 取代。

輸出應該全部為小寫字母

輸出不應該有任何空格

# --hints--
