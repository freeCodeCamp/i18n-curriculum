# --explanation--

`Might`는 가능성에 대해 이야기할 때 사용합니다 — 일어날 수도 있지만 확실하지 않은 상황입니다. 예를 들어:

- `It might rain tomorrow.` - 가능성은 있지만 보장되지는 않습니다.

- `We might need more time to finish the project.` - 가능성은 있지만 확인되지는 않았습니다.
