# --explanation--

`user interface`（UI）是指系統或軟體中使用者直接互動的零件，例如按鈕、選單和螢幕。例如： 

`The user interface was redesigned to be more intuitive。`－意指使用者交談、互動的軟體零件被改進了。

`User feedback` 是指使用者對產品或服務體驗所回報的意見、建議或問題。例如： 

`The company improved the app based on user feedback。`－意指來自使用者的評論和建議已被閱讀並用來改進該應用程式。

在這個背景關係中，Linda 解釋團隊依賴使用者回授來持續改進使用者介面，確保它符合使用者的需求。
