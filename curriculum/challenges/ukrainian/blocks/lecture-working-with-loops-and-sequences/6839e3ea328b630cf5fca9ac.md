# --description--

Як ви дізналися в попередніх модулях, цикли використовуються для повторення блоку коду певну кількість разів. У цьому уроці ви навчитеся працювати з різними типами циклів у Python.

Перший цикл, який ми розглянемо, — це цикл `for`. Ось приклад використання циклу `for` для ітерування по списку та виведення кожного елемента в консоль:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)
```

Результат буде таким:

```md
Rust
Java
Python
C++
```

Зверніть увагу, що `print(language)` відступлений всередині циклу. Без цього відступу ви отримаєте `IndentationError`:

```python
"""
Traceback (most recent call last):
  File "<stdin>", line 4, in <module>
IndentationError: expected an indented block after 'for' statement on line 3
"""
```

Ви також можете використовувати цикл `for` для ітерування по інших ітерованих об’єктах, наприклад рядку. Ось приклад використання циклу `for` для проходження по рядку `code` та виведення кожного символу:

```py
for char in 'code':
    print(char)
```

Результат буде таким:

```md
c
o
d
e
```

Як і в JavaScript, у Python можна вкладати цикли `for`. Ось приклад використання вкладеного циклу `for`:

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)
```

Зовнішній цикл ітеруватиме по кожному `category` у списку `categories`. Для кожного `category` внутрішній цикл ітеруватиме по кожному `food` у списку `foods`. Ось результат, який буде виведено в консоль:

```md
Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
```

Ще один тип циклу, який можна використовувати в Python, — це цикл `while`. Цей тип циклу повторює блок коду, доки умова не стане `False`. Ось приклад використання циклу `while` для гри в угадайку:

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')
```

У цьому прикладі у нас є змінна `secret_number` зі значенням `3` та початкова спроба `0`. Потім ми використовуємо функцію `input` для отримання вхідних даних від користувача, потім конвертуємо рядок у ціле число за допомогою функції `int()` і призначаємо це значення змінній `guess`. Якщо користувач вгадає правильно, ввівши `3`, цикл `while` переривається, і в консоль виводиться повідомлення `You got it!`. Інакше в консоль виводиться повідомлення `Wrong! Try again.`, і цикл повторюється, пропонуючи користувачу вгадати знову.

Ось приклад результату:

```md
Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
```

Як і в JavaScript, Python підтримує інструкції `break` та `continue`.

Інструкція `break` використовується для припинення виконання циклу. Ось приклад використання інструкції `break` для списку `developer_names`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

У цьому прикладі ми ітеруємо по списку `developer_names` і виводимо кожне ім’я в консоль. Якщо ім’я дорівнює `Naomi`, то ми виходимо з циклу. Це призводить до того, що в консоль буде виведено лише ім’я `Jess`.

Інструкція `continue` використовується, щоб пропустити поточну ітерацію циклу і перейти до наступної. Змінімо попередній приклад, використовуючи інструкцію `continue` замість `break`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

Тепер результат у консолі буде іншим. Імена `Jess` і `Tom` виводяться, тому що інструкція `continue` пропускає другу ітерацію циклу, коли `developer` дорівнює `Naomi`, і це ім’я не виводиться в консоль.

Цикли `for` і `while` можна поєднувати з `else` умовою, яка виконується лише тоді, коли цикл не був завершений інструкцією `break`. Ось приклад використання кількох циклів `for`:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

У цьому прикладі у нас є список випадкових слів, і використовується цикл `for` для проходження по кожному слову. Всередині зовнішнього циклу `for` є ще один цикл `for` для проходження по кожній літері кожного слова. Якщо версія літери в нижньому регістрі є голосною, ми виводимо слово разом із голосними, які воно містить, а потім виходимо з внутрішнього циклу. Якщо слово не містить голосних, ми виводимо відповідне повідомлення.

Ось як виглядає результат у консолі:

```md
'sky' has no vowels
'apple' contains the vowel 'a'
'rhythm' has no vowels
'fly' has no vowels
'orange' contains the vowel 'o'
```

Цикли дуже поширені в Python, тому важливо добре з ними ознайомитися. У наступних кількох уроках ви навчитеся працювати з функціями `enumerate()` і `range()` у циклах.
