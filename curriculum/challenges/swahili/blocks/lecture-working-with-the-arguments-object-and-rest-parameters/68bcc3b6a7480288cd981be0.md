# --interactive--

Kama unavyokumbuka kutoka mafundisho na warsha zilizopita, unaweza kuunda kitendakazi chenye vigezo kadhaa na kuitisha kitendakazi hicho kwa hoja.

Hapa kuna ukumbusho:

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call
getSum(3, 4); // 7
```

Lakini vipi ikiwa una kitendakazi kinaitwa na hoja zaidi kuliko ilivyotangazwa kukubali?

:::interactive_editor

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call with extra argument
console.log(getSum(3, 4, 5)); // 7
```

:::

JavaScript haitupa kosa katika kesi hii. Badala yake itapuuzia hoja ya ziada na kuongeza tu nambari `3` na `4` pamoja. Vitendakazi vinavyokubali idadi inayobadilika ya hoja vinajulikana kama <dfn>vitendakazi vya variadic</dfn>.

Ikiwa unafanya kazi na vitendakazi vya variadic, basi unaweza kutumia kitu cha `arguments`. Kitu hiki kinachofanana na safu ya data kina thamani za hoja zilizopitishwa katika kitendakazi.

Hapa kuna mfano:

:::interactive_editor

```js
function logArgs() {
  for (const arg of arguments) {
    console.log(arg);
  }
}

logArgs(1, 2, 3);
// result:
// 1
// 2
// 3

logArgs("example"); // "example"
```

:::

Kwa kuwa kitu cha `arguments` kinafanana na safu ya data, unaweza kupata hoja kwa kielezo maalum hivi:

:::interactive_editor

```js
function getArg() {
  return arguments[1];
}

console.log(getArg(2, 4, 6)); // 4
```

:::

Unaweza pia kutumia sifa ya `length` hivi kupata idadi ya hoja ambazo kitendakazi kilitishiwa nazo:

:::interactive_editor

```js
function getArgs() {
  return arguments.length;
}

console.log(getArgs("Example")); // 1
console.log(getArgs("Another", "Example")); // 2
```

:::

Ingawa kitu cha `arguments` kinaonekana kama safu halisi, hakina njia za `Array` zilizojengwa ndani kama `includes` au `push`. Ili kupata upatikanaji wa njia hizo, unahitaji kwanza kubadilisha kitu cha `arguments` kuwa safu halisi kwa kutumia kitu kama `slice`, `Array.from()` au opereta ya spread:

:::interactive_editor

```js
function hasCat() {
  return [...arguments].includes("cat");
}

console.log(hasCat("dog", "chicken", "cat")); // true
console.log(hasCat("dog", "chicken", "horse")); // false
```

:::

Ingawa inawezekana kufanya kazi na kitu cha `arguments` kwa vitendakazi vya variadic, programu za kisasa za JavaScript kawaida zitumia sintaksia ya kigezo cha rest. Utajifunza zaidi kuhusu hilo katika funzo lijalo.
