# --description--

للوصول إلى عناصر المصفوفة، يمكنك استخدام رقم الفهرس. يبدأ رقم الفهرس من 0، لذا يكون العنصر الأول في المصفوفة عند الفهرس 0، والعنصر الثاني عند الفهرس 1، وهكذا.

على سبيل المثال، للوصول إلى العنصر الأول في مصفوفة `fruits`، يمكنك استخدام الشفرة التالية:

```javascript
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits[0]); // Output: apple
console.log(fruits[2]); // Output: orange
```

إذا كان الفهرس خارج النطاق، ستُرجع JavaScript القيمة `undefined`. هذا يعني أنه إذا حاولت الوصول إلى `fruits[3]` في المثال أعلاه، فستُرجع `undefined`.
