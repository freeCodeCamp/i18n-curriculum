# --explanation--

最終システムチェックが完了しなかった場合に何が起こるかを理解するには、アップデートの準備におけるそれらの役割を考えてみてください。

最終チェックはシステムアップデートの準備状況を確認するためのものであり、遅れるとアップデートの延期につながる可能性があります。

この関連を確認するには、`The final system checks are set`というフレーズを探してください。
