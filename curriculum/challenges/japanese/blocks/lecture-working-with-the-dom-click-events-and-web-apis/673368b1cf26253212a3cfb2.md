# --description--

`querySelectorAll()` メソッドについて学びましょう。このメソッドを使うと、特定の CSS セレクターに一致するすべての DOM 要素のリストを取得できます。

`querySelectorAll()` メソッドは次のように呼び出せます。

```js
document.querySelectorAll(selectors);
```

`document` オブジェクトで呼び出し、CSS セレクターを含む文字列を引数として渡します。引数は有効な CSS セレクター文字列である必要があります。そうでない場合は `SyntaxError` 例外がスローされます。

`querySelectorAll()` は `NodeList` オブジェクトを返します。これは指定した CSS セレクターに一致するノードのコレクションです。

リストには、セレクターに一致する各要素ごとに 1 つの `Element` オブジェクトが含まれます。もし一致する要素がなければ、リストは空になります。

要素は HTML ドキュメントに現れる順番で並びます。例えば、特定のタイプのすべての要素をマッチさせることができます。

```js
document.querySelectorAll("div");
```

特定のクラスを持つすべての要素：

```js
document.querySelectorAll(".rounded");
```

特定の ID を持つすべての要素：

```js
document.querySelectorAll("#logo");
```

または特定の属性を持つすべての要素、例えば特定の URL にユーザーを誘導するすべてのリンク：

```js
document.querySelectorAll("a[href='https://www.freecodecamp.org/']");
```

さらに、このような複雑な CSS セレクターも使えます。これはクラス `ingredients` を持つ番号なしリスト要素内のすべてのリストアイテムにマッチします：

```js
document.querySelectorAll("ul.ingredients li");
```

このメソッドはノードのコレクションを返すので、このコレクションを変数に代入して、個々の要素を操作するプログラムで使えます。一般的な構文の例を見てみましょう：

```js
const matches = document.querySelectorAll(selectors); 
```

例えば、3 つのリストアイテムを持つこの番号なしリストがあるとします：

```html
<ul class="ingredients">
  <li>Flour</li>
  <li>Cheese</li>
  <li>Water</li>
</ul>
```

`querySelectorAll()` メソッドを呼び出して、クラス `ingredients` を持つ番号なしリスト内のすべてのリストアイテムにマッチさせ、その戻り値を `matches` 定数に代入できます：

```js
const matches = document.querySelectorAll("ul.ingredients li");
```

これにより 3 つのノードのコレクションが返されます。各ノードはリストアイテム要素を表します：

```js
// NodeList(3)
{
  0: `<li>Flour</li>`,
  1: `<li>Cheese</li>`,
  2: `<li>Water</li>`,
  length: 3,
}
```

このコレクションは他の JavaScript の配列と同じように扱えます。

例えば、`console.log()` でコンソールに出力できます：

```js
console.log(matches);
```

`length` プロパティでコレクションの長さを確認することもできます：

```js
console.log(matches.length);
```

出力はコレクション内のノード数を表す整数になります：

```md
3
```

また、対応するインデックスを使ってコレクションの個々の要素にアクセスできます。最初の要素はインデックス `0` にあります：

```js
console.log(matches[0]);
console.log(matches[1]);
console.log(matches[2]);
```

これが出力です：

```html
<li>Flour</li>
<li>Cheese</li>
<li>Water</li>
```

要素はコンソールに一つずつ出力されます。

`for` ループで要素を繰り返すこともできます：

```js
for (let i = 0; i < matches.length; i++) {
  console.log(matches[i]);    
}
```

この場合、出力はまったく同じになります。しかし、コレクション内のすべての要素を操作するには非常に強力です。

JavaScript の `querySelectorAll()` メソッドは、CSS セレクターに基づいて複数の要素を選択する強力なツールです。このメソッドを使えば、ウェブページ内の特定の要素を選択して必要に応じて操作できます。
