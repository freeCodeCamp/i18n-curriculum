# --description--

Коли ви хочете запитати про наявність конкретних місць або речей у вашому поточному місці, ви можете використати структуру питання `Are there any ... around here?`. Наприклад:

`Are there any Wi-Fi spots around here?` — ви питаєте, чи можна знайти Wi-Fi поблизу цього місця.

Фраза `around here` означає найближчу область.
