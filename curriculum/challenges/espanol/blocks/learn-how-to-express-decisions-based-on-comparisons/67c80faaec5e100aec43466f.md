# --explanation--

`More robust` significa más fuerte o más confiable en comparación con otra cosa. Por ejemplo:

`This software is more robust than the previous version because it can handle larger data sets.` - Aquí, `more robust` significa que el software es más fuerte y capaz.

`When it comes to` introduce un tema específico, que significa "respecto a" o "acerca de". Por ejemplo:

`When it comes to customer support, this company is very responsive.` - Aquí, `when it comes to` introduce el tema del soporte al cliente.
