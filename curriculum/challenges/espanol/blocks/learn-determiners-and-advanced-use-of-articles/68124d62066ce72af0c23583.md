## --blanks--

`every`

### --feedback--

Este determinante significa que ocurre regularmente en cada periodo (aquí, cada dos semanas).

---

`any`

### --feedback--

Este determinante se usa para problemas no específicos - aún no conocidos.

# --explanation--

`Every two weeks` significa que algo ocurre una vez en cada período de dos semanas; esto a menudo se llama `bi-weekly`.

`Any problems` usa `any` para referirse a cualquier problema que pueda ocurrir, incluso si aún no sabes cuáles son.

**Nota:** `Bi-weekly` a veces puede significar "dos veces por semana" o "cada dos semanas", pero aquí significa "cada dos semanas" según el contexto.
