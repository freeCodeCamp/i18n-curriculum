# --description--

약분할 수 없는 분수를 코어실리언트 분수라고 부르겠습니다.

또한 분모의 코어실리언스 $R(d)$를 코어실리언트인 진분수의 비율로 정의하겠습니다. 예를 들어, $R(12) = \frac{4}{11}$입니다.

수 $d > 1$의 코어실리언스는 $φ(d)$가 오일러 피 함수일 때 $\frac{φ(d)}{d − 1}$입니다.

또한 수 $n > 1$의 코어실리언스를 $C(n) = \frac{n − φ(n)}{n − 1}$로 정의합니다.

소수 $p$의 코어실리언스는 $C(p) = \frac{1}{p − 1}$입니다.

$C(n)$이 단위 분수인 합성 정수 $1 < n ≤ 2 × {10}^{11}$ 모두의 합을 구하세요.
