# --instructions--

Для цього проєкту ви візуалізуєте часові ряди даних за допомогою лінійної діаграми, гістограми та блокових діаграм. Ви використаєте Pandas, Matplotlib і Seaborn для візуалізації набору даних, що містить кількість переглядів сторінок щодня на форумі freeCodeCamp.org з 2016-05-09 по 2019-12-03. Візуалізації допоможуть вам зрозуміти шаблони відвідувань і визначити річний та місячний зріст.

Використайте дані, щоб виконати такі завдання:

- Використайте Pandas, щоб імпортувати дані з файлу "fcc-forum-pageviews.csv". Встановіть індекс у стовпець `date`.
- Очистіть дані, відфільтрувавши дні, коли перегляди сторінок були у верхніх 2,5% або нижніх 2,5% набору даних.
- Створіть функцію `draw_line_plot`, яка використовує Matplotlib для побудови лінійної діаграми, схожої на "examples/Figure_1.png". Заголовок має бути `Daily freeCodeCamp Forum Page Views 5/2016-12/2019`. Мітка по осі x — `Date`, а по осі y — `Page Views`.
- Створіть функцію `draw_bar_plot`, яка малює гістограму, схожу на "examples/Figure_2.png". Вона має показувати середню кількість переглядів сторінок за день для кожного місяця, згруповану за роками. Легенда має показувати мітки місяців і мати заголовок `Months`. На діаграмі мітка по осі x — `Years`, а по осі y — `Average Page Views`.
- Створіть функцію `draw_box_plot`, яка використовує Seaborn для побудови двох суміжних блокових діаграм, схожих на "examples/Figure_3.png". Ці блокові діаграми мають показувати, як розподілені значення в межах певного року або місяця, і як вони змінюються з часом. Заголовок першої діаграми — `Year-wise Box Plot (Trend)`, а другої — `Month-wise Box Plot (Seasonality)`. Переконайтеся, що мітки місяців знизу починаються з `Jan`, а осі x і y мають правильні підписи. Шаблонний код містить команди для підготовки даних.

Для кожної діаграми використовуйте копію датафрейму.

Шаблонний код також містить команди для збереження та повернення зображення.
