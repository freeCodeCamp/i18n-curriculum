# --explanation--

`It can't be` wird verwendet, wenn jemand sicher ist, dass etwas nicht wahr oder nicht möglich ist. Zum Beispiel:

- `It can't be true!` – Sie sind sicher, dass es falsch ist.

- `That error can't be from my code.` – Sie sind überzeugt, dass Ihr Code korrekt ist.

Jake ist sicher, dass die Sicherheitslücke nicht zufällig war – er glaubt, dass sie aus einem bestimmten Grund gezielt verursacht wurde.
