# --explanation--

`To pick up` bedeutet in diesem Zusammenhang, etwas fortzusetzen, das zuvor pausiert wurde. Zum Beispiel:

`Let's pick up this discussion after lunch.` – Das bedeutet, die Diskussion später fortzusetzen.

`To leave off` bezieht sich auf den Punkt, an dem etwas gestoppt wurde. Zum Beispiel:

`Where did we leave off in the book?` – Das fragt nach dem letzten Teil, der vor dem Anhalten gelesen wurde.
