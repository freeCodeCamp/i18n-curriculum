# --explanation--

`To be better` 表示某些不同的东西可能是改进。例如：

`Wouldn't it be better to start earlier and avoid last-minute stress?` - 这意味着提前开始可能会有所改善。

`Differently` 意味着以不同的方式。它描述的是 **如何** 做某事。这与 `different` 形成对比，后者用于描述事物、人或情况的不同。其他示例：

- `We need a different plan.` - 这里，`different` 描述了 plan。

- `We need to plan differently.`——这里的描述是关于如何以另一种方式进行规划。
