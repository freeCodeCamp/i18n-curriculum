# --interactive--

이전에 적절한 제목 레벨 구조에 대해 배웠습니다. 이제 좋은 제목 구조가 접근성에 어떻게 영향을 미치는지 배우겠습니다.

제목을 올바르게 사용하면 사용자가 웹 페이지를 탐색하고 이해할 수 있도록 시각적 계층 구조를 만듭니다. 논리적인 제목 계층 구조를 사용하면 화면 읽기 프로그램 사용자가 콘텐츠 구조를 이해하고 빠르게 탐색할 수 있습니다. 뒤따르는 내용을 정확히 설명하는 적절한 제목 텍스트를 만들면 모든 사용자가 사이트에서 필요한 정보를 쉽게 찾을 수 있습니다. 추가로, 잘 구성된 제목은 사이트의 SEO 향상에도 도움이 될 수 있습니다.

기본 예시는 다음과 같습니다:

:::interactive_editor

```html
<h1> Music Store Page</h1>

<h2>Featured Albums</h2>
<p>Check out our featured albums below:</p>
```

:::

제목을 사이트의 기초라고 생각하세요. 좋은 기초가 없으면 콘텐츠의 접근성이 떨어집니다.

제목을 올바르게 사용하여 보조 기술을 사용하는 사람들에게 친화적인 웹 프로젝트를 만드는 방법을 살펴보겠습니다.

`h1`부터 `h6`까지의 제목은 화면 읽기 프로그램 사용자를 위한 탐색 구조를 만듭니다. 화면 읽기 프로그램은 페이지의 모든 제목을 나열할 수 있어 사용자가 필요한 섹션으로 바로 이동할 수 있습니다. 따라서 제목을 올바르게 배열하는 것은 이 사용자들이 불필요한 콘텐츠를 피하고 원하는 정보를 빠르게 찾도록 돕는 데 중요합니다.

:::interactive_editor

```html
<h1>I am an h1 heading</h1>
<h2>I am an h2 heading</h2>
<h3>I am an h3 heading</h3>
<h4>I am an h4 heading</h4>
<h5>I am an h5 heading</h5>
<h6>I am an h6 heading</h6>
```

:::

부분 시각 또는 인지 장애가 있는 사람들은 인지 부담을 줄이기 위해 정보를 빠르고 쉽게 처리해야 합니다.

제목을 올바르게 배열하는 것도 중요하지만, 제목 텍스트를 명확하고 설명적으로 만드는 것도 필요합니다.

제목을 올바르게 사용하기 위해 따라야 할 주요 모범 사례는 다음과 같습니다:

- 명확한 조직을 반영하는 계층 구조로 제목을 사용하세요. 예를 들어, 페이지 제목은 `h1`이어야 하며, 주요 섹션은 `h2` 제목으로, 하위 섹션은 `h3` 제목으로, 그리고 `h6`까지 이어집니다.
- `h1`에서 `h3`로, 또는 `h2`에서 `h4`로 건너뛰지 마세요.
- 각 제목 뒤에 오는 내용을 요약하는 명확하고 설명적인 텍스트를 사용하세요.
- 제목을 단독으로 사용하지 마세요 — 제목 뒤에는 반드시 내용이 따라야 합니다.
- 텍스트를 제목처럼 보이게 꾸미지 말고, 필요한 곳에 적절한 제목을 사용하세요.
- 각 페이지에는 주제나 제목을 나타내는 단일 `h1` 요소가 있어야 합니다.

다음은 페이지에서 제목을 어떻게 사용해야 하는지 나타내는 기본 마크업입니다:

:::interactive_editor

```html
<!-- Page title -->
<h1>What is HTML</h1>

<!-- First section -->
<section>
  <h2>Introduction to HTML</h2>
  <p>
    HTML stands for HyperText Markup Language. It is the standard language for
    creating web pages.
  </p>
</section>
<!-- Second section -->
<section>
  <h2>History of HTML</h2>
  <p>HTML began to take shape in the early 90s</p>
  <h3>Origins</h3>
  <p>
    HTML was created by Tim Berners-Lee in 1991. It has evolved significantly
    over the years.
  </p>
</section>
```

:::
