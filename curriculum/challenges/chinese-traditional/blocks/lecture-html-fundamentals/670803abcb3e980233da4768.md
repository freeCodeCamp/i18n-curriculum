# --interactive--

`div` 元素用作容器以群組其他元素。

這是一個 `div` 元素的範例。為你的 `div` 元素添加另一個段落元素，並在預覽視窗中查看變更。若要查看預覽，你需要啟用互動式編輯器。

:::interactive_editor

```html
<div>
  <p>Example paragraph element.</p>
</div>
```

:::

當你想要將會共用一組 CSS 樣式的 HTML 元素群組時，你主要會使用 `div` 元素。你將在未來的課程和工作坊中學習更多關於 CSS 的內容。

即使 `div` 元素在實際的程式碼庫中很常見，你也應該小心不要過度使用它。有時候使用其他元素會更合適。

例如，如果你想將你的內容劃分為多個區段，那麼使用 `section` 元素會比使用 `div` 元素更合適。

在第一個 `section` 元素下方添加另一個 `section` 元素。然後在 `section` 元素內添加 `h2` 和 `p` 元素。你可以使用任何你喜歡的文字，並且你會在預覽視窗中看到變更。若要與範例互動，你需要啟用互動式編輯器。

:::interactive_editor

```html
<section>
  <h2>Mammals</h2>
  <p>
    Mammals are warm-blooded animals with fur or hair. Most give birth to live
    young.
  </p>
  <ul>
    <li>Lion</li>
    <li>Elephant</li>
    <li>Dolphin</li>
  </ul>
</section>
```

:::

`section` 元素具有語意意義，而 `div` 元素則沒有語意。語意是語言中字或片語的意義。在 HTML 這種語言中，元素本身也有其語意意義。因此，若你使用 `section` 元素，瀏覽器會擷取其語意並理解要將其視為一個區段－無論是在桌面、行動裝置，還是其他裝置。
