# --explanation--

`We've` 是 `we have` 的縮寫。

`Made` 是不規則動詞 `make` 的過去分詞。`Have made` 描述過去創建或完成且對現在有影響的事物。

在這句話中，`have had` 是現在完成式。這個結構 `have` + `had` 表示某事從過去開始並且仍然影響現在。 

- `Have` 是用於現在完成式的助動詞。

- `Had` 是 `have` 的過去分詞，意思是經歷或擁有。

合起來，`have had` 表示變更較早開始，並持續產生影響。

總結來說，這些不規則動詞（`make` 的過去式為 `made`，`have` 的過去式為 `had`）構成現在完成式，表示從過去開始且至今仍相關的動作。
