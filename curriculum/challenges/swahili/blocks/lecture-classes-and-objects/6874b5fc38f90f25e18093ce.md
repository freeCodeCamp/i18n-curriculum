# --description--

Katika funzo la mwisho, ulijifunza kuhusu madarasa ya vitu na jinsi yanavyotumika kama ramani za kuunda vitu.

Hapa, tutaangazia kwa undani zaidi sifa za kipengele na njia.

Tuchunguze kwanza sifa za kipengele, kisha njia.

Sifa za kipengele ni vigezo vinavyomilikiwa na kitu, hivyo vinahifadhi data. Kuna aina mbili za sifa za kipengele: sifa za mfano na sifa za darasa la vitu.

Sifa za mfano ni za kipekee kwa kila kitu kinachoundwa kutoka darasa la vitu, na kawaida huziweka kwa njia ya `__init__`. Sifa za darasa la vitu, kwa upande mwingine, zinamilikiwa na darasa la vitu lenyewe na zinashirikiwa na mifano yote ya darasa hilo.

Ili kupata sifa ya kipengele, unatumia uandishi wa nukta.

Hapa kuna mifano ya sifa za mfano na sifa za darasa la vitu, na jinsi ya kuzifikilia kutoka kwa vitu:

```python
class Dog:
    species = "French Bulldog" # Class attribute

    def __init__(self, name):
        self.name = name # Instance attribute

print(Dog.species) # French Bulldog

dog1 = Dog("Jack")
print(dog1.name)    # Jack
print(dog1.species) # French Bulldog

dog2 = Dog("Tom")
print(dog2.name)    # Tom
print(dog2.species) # French Bulldog
```

Kumbuka kuwa unaweza kufikia sifa za darasa la vitu moja kwa moja kutoka kwa darasa la vitu lenyewe, lakini unahitaji kuunda kitu na kukipatia data kwanza kabla hujafikia sifa za mfano.

Magari ni mfano mwingine mzuri, kwa kuwa magari yote yana mfano na rangi:

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

car_1 = Car("red", "Toyota Corolla")
car_2 = Car("green", "Lamborghini Revuelto")

print(car_1.model) # Toyota Corolla
print(car_2.model) # Lamborghini Revuelto

print(car_1.color) # red
print(car_2.color) # green
```

Njia ni vitendakazi vilivyofafanuliwa ndani ya darasa la vitu. Kwa njia hizo, kitu chochote kilichofafanuliwa kutoka darasa la vitu kinaweza kufanya vitendo vinavyofanya kazi au kubadilisha data yake mwenyewe. Pia unapata njia kwa kutumia uandishi wa nukta.

Kwa mfano, mbwa wanaweza kulia. Hivyo tunaweza kuwa na njia ya `bark` katika darasa la `Dog` kama ulivyoona katika funzo la awali:

```python
class Dog:
   species = "French Bulldog"

   def __init__(self, name):
     self.name = name

   def bark(self):
       return f"{self.name} says woof woof!"

jack = Dog("Jack")
jill = Dog("Jill")

print(jack.bark()) # Jack says woof woof!
print(jill.bark()) # Jill says woof woof!
```

Darasa la `Car` pia linaweza kuwa na njia ya `describe`:

```python
class Car:
    def __init__(self, color, model):
        self.color = color  # Instance attribute
        self.model = model  # Instance attribute

    def describe(self):
        return f"This car is a {self.color} {self.model}"

car_1 = Car("red", "Toyota Corolla")
car_2 = Car("green", "Lamborghini Revuelto")

print(car_1.describe()) # This car is a red Toyota Corolla
print(car_2.describe()) # This car is a green Lamborghini Revuelto
```
