# --explanation--

Lisa anatumia `Second Conditional` kujadili hali ya kufikirika.

`Second Conditional` hutumika kuzungumzia hali zisizo halisi au zisizowezekana sasa au baadaye. Inafuata muundo `if` + `Present Simple` au `Past Continuous` (kwa sharti) na `might / would / could` + kitenzi cha msingi (kwa matokeo). Kwa mfano:

- `If I had more time, I would learn a new programming language.` - Hii inaelezea hali ya kufikirika ambapo kuwa na muda zaidi kungepelekea kujifunza.

- `If the server were faster, it could handle more users.` - Seva si haraka sasa, lakini kama ingekuwa, ingeweza kuunga mkono watumiaji wengi zaidi.
