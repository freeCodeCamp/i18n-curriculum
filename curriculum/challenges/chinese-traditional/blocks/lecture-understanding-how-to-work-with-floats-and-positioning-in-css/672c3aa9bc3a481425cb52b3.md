# --interactive--

CSS 中的 `z-index` 屬性用於控制頁面上重疊定位元素的垂直堆疊順序。當多個元素堆疊在彼此之上時，`z-index` 值決定哪個元素顯示在最上層。`z-index` 值越高，元素越接近觀者，而較低的值則將元素置於堆疊的較後方。

然而，`z-index` 只對已定位的元素有效，這表示該元素必須有非 `static` 的定位值，例如 `relative`、`absolute` 或 `fixed`。預設的 `z-index` 值是 `auto`，會將元素置於預設的堆疊順序中。讓我們來看一個有三個方塊的範例：

```html
<div class="container">
  <div class="box1">Box 1</div>
  <div class="box2">Box 2</div>
  <div class="box3">Box 3</div>
</div>
```

現在，我們可以套用一些樣式，使方塊彼此重疊，如下所示：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box1">Box 1</div>
  <div class="box2">Box 2</div>
  <div class="box3">Box 3</div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

.box1 {
  position: absolute;
  z-index: 1;
  background: lightcoral;
  top: 20px;
  left: 20px;
  width: 100px;
  height: 100px;
}

.box2 {
  position: absolute;
  z-index: 3;
  background: gold;
  top: 40px;
  left: 40px;
  width: 100px;
  height: 100px;
}

.box3 {
  position: absolute;
  z-index: 2;
  background: lightgreen;
  top: 60px;
  left: 60px;
  width: 100px;
  height: 100px;
}
```

:::

對於容器，定位將設定為 `relative`，而所有巢狀在其中的方塊將設定為 `absolute` 定位。每個方塊的 `z-index` 值不同，導致方塊彼此層疊。

你可以將 `z-index` 視為在網頁上創建圖層的一種方式，具有較高 `z-index` 值的元素會被放置在較低值元素的上方。這對於控制重疊元素在複雜版面配置中的行為特別有用，例如模態視窗、彈出視窗或工具提示。
