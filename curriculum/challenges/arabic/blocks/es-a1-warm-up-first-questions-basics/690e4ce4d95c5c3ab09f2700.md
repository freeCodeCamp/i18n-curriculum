# --explanation--

الكلمة `Encantada` هي طريقة بديلة وشائعة للمتحدثة الأنثى لتقول "سعيد بلقائك".

تعني "مسحورة" أو "مبتهجة".

| الصيغة | الجنس | المتحدث | المعنى |
| :--- | :--- | :--- | :--- |
| `Encantado` | مذكر (`-o`) | متحدث ذكر | سعيد بلقائك. |
| `Encantada` | مؤنث (`-a`) | متحدثة أنثى | سعيد بلقائك. |
