# --explanation--

كلمات مثل `everything`، `something`، `anything`، و`nothing` هي ضمائر غير محددة تأخذ أفعالًا بصيغة المفرد، رغم أنها قد تشير إلى عدة عناصر أو مفاهيم. على سبيل المثال:

- `Everything was ready for the presentation.` - تشير `Everything` إلى كل الأشياء وتتطلب فعلًا بصيغة المفرد.

- `Something is wrong with the code.` - تشير `Something` إلى شيء غير محدد وتأخذ فعلًا بصيغة المفرد.

- `Is anything missing from the report?` - تشير `Anything` إلى أي شيء بشكل عام وتستخدم فعلًا بصيغة المفرد.

- `Nothing was found in the logs.`  - تشير `Nothing` إلى غياب الأشياء وتتطلب فعلًا بصيغة المفرد.

في الجملة `Everything else was running smoothly` تستخدم `was` لأن `everything` تُعامل كمفرد، رغم أنها تشير إلى عدة عناصر.
