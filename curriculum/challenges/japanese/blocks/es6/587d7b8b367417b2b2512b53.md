# --description--

ES6は、<dfn>クラス</dfn>キーワードを使ってオブジェクトを作成する新しい構文を提供します。

ES5では、`constructor`関数を定義し、`new`キーワードを使ってオブジェクトをインスタンス化することでオブジェクトを作成できます。

ES6では、`class`宣言に`constructor`メソッドがあり、`new`キーワードで呼び出されます。`constructor`メソッドが明示的に定義されていない場合は、引数なしで暗黙的に定義されます。

```js
// Explicit constructor
class SpaceShuttle {
  constructor(targetPlanet) {
    this.targetPlanet = targetPlanet;
  }
  takeOff() {
    console.log("To " + this.targetPlanet + "!");
  }
}

// Implicit constructor 
class Rocket {
  launch() {
    console.log("To the moon!");
  }
}

const zeus = new SpaceShuttle('Jupiter');
// prints To Jupiter! in console
zeus.takeOff();

const atlas = new Rocket();
// prints To the moon! in console
atlas.launch();
```

`class`キーワードは新しい関数を宣言し、その関数にコンストラクタが追加されることに注意してください。このコンストラクタは、`new`が呼び出されて新しいオブジェクトが作成されるときに実行されます。

**注意:** ES6のクラス名には慣例としてUpperCamelCaseを使うべきであり、上記の`SpaceShuttle`がその例です。

`constructor`メソッドは、クラスで作成されたオブジェクトを生成および初期化するための特別なメソッドです。これはJavaScriptのアルゴリズムとデータ構造認定のオブジェクト指向プログラミングセクションでさらに学びます。

# --instructions--

`class`キーワードを使い、`constructor`を書いて`Vegetable`クラスを作成してください。

`Vegetable`クラスは、`name`に渡されるプロパティ`constructor`を持つvegetableオブジェクトを作成できます。
