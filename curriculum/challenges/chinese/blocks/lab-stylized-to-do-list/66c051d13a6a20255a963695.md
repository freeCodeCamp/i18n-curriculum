# --description--

在本实验中，你将练习当 `links` 被悬停、聚焦、点击和访问时可以应用的不同样式。

**目标：**实现以下用户需求并通过所有测试以完成实验。

**用户故事：**

1. 你应该有一个带有类 `todo-list` 的无序列表。
2. 在无序列表中，你应该有四个列表项。
3. 在每个列表项内，应包含：

   - 一个 `input` 元素，其类型为 `checkbox`，且 `id` 设置为唯一值。
   - 一个 `label` 元素，其 `for` 属性设置为对应 `input` 元素的 `id`。
   - 带有 `sub-item` 类的无序列表。
   - 一个包含锚点元素的列表项。锚点应具有类 `sub-item-link`、有效的 `href` 值和使链接在新建标签（页）中打开的 `target` 值。

4. 你的 `a` 元素不应有任何文本装饰。
5. 你应该将未访问 `链接` 的文本颜色设置为你选择的颜色。
6. 当你的 `links` 被访问时，颜色应更改为你选择的另一种颜色。
7. 当你的 `link` 被悬停时，颜色应更改为你选择的另一种颜色。
8. 当你的 `links` 获得焦点时，链接周围应有彩色轮廓。
9. 当你的 `links` 被点击时，颜色应更改为你选择的另一种颜色。

**注意：** 确保在 HTML 中链接样式表以应用 CSS。
