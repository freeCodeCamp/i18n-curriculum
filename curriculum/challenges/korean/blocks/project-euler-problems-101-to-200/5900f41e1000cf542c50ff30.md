# --description--

ABCD를 대각선 AC와 BD를 가진 볼록 사각형이라고 합시다. 각 꼭짓점에서 대각선은 두 변과 각각 각을 이루어 총 여덟 개의 모서리 각을 만듭니다.

<img alt="대각선 AC와 BD를 가진 볼록 사각형 ABCD" src="https://cdn.freecodecamp.org/curriculum/project-euler/integer-angled-quadrilaterals.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

예를 들어, 꼭짓점 A에서 두 각은 CAD, CAB입니다.

이렇게 여덟 개의 모서리 각이 모두 도 단위로 정수 값을 가지는 사각형을 "정수 각 사각형"이라고 부릅니다. 정수 각 사각형의 예로는 모든 여덟 각이 45°인 정사각형이 있습니다. 또 다른 예는 DAC = 20°, BAC = 60°, ABD = 50°, CBD = 30°, BCA = 40°, DCA = 30°, CDB = 80°, ADB = 50°인 경우입니다.

서로 닮지 않은 정수 각 사각형의 총 개수는 얼마일까요?

**참고:** 계산 중 각도가 정수에 ${10}^{-9}$ 이내로 근접하면 정수로 간주해도 됩니다.
