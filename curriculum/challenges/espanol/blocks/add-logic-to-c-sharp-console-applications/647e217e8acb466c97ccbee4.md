## --text--

¿Bajo qué condición es mejor usar una sentencia `switch-case` que una sentencia `if-elseif-else`?

## --answers--

Una sentencia `switch-case` es apropiada cuando se requieren más de 2 o 3 bloques de código `else if`.

---

Una sentencia `switch-case` es apropiada cuando la cantidad de patrones `case` es pequeña.

---

Una sentencia `switch-case` es apropiada cuando la sentencia de selección se encuentra dentro de un bucle.
