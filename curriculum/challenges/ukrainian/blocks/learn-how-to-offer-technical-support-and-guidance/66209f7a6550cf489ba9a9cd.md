# --description--

`To handle` щось означає мати справу з цим або керувати цим. У програмуванні `handling` передбачає керування даними або операціями. Наприклад, `to handle user input means to manage what happens when a user enters data`.

`Error handling` — це робота з помилками, які виникають під час запуску програми. Це пов’язано з написанням коду, який може керувати цими помилками, не даючи програмі аварійно завершитись. Наприклад, `using try-except blocks in Python to handle errors that might occur when a program runs`.
