# --description--

In programmazione, una `string` è un tipo di dato usato per memorizzare testo. Una `string` può contenere lettere, numeri e simboli. Spesso viene rappresentata racchiudendo il testo tra virgolette.

Per esempio, `"Hello, world!"` è una `string`.
