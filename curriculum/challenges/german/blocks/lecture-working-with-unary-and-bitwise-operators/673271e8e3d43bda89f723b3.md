# --interactive--

Unäre Operatoren wirken auf einen einzelnen Operanden, um Operationen wie Typumwandlung, Wertmanipulation oder das Überprüfen bestimmter Bedingungen durchzuführen. Schauen wir uns einige gängige unäre Operatoren und ihre Funktionsweise an.

Der unäre Plus-Operator wandelt seinen Operanden in eine Zahl um. Wenn der Operand bereits eine Zahl ist, bleibt er unverändert.

:::interactive_editor

```js
const str = '42';
const strToNum = +str;

console.log(strToNum); // 42
console.log(typeof str); // string
console.log(typeof strToNum); // number
```

:::

Der unäre Plus-Operator ist praktisch, wenn Sie sicherstellen möchten, dass Sie mit einem numerischen Wert arbeiten. Wie Sie sich denken können, gibt es auch einen unären Negationsoperator. Er negiert den Wert des Operanden. Er funktioniert ähnlich wie der unäre Plus-Operator, nur dass er das Vorzeichen umkehrt.

:::interactive_editor

```js
const str = '42';
const strToNegativeNum = -str;

console.log(strToNegativeNum); // -42
console.log(typeof str); // string
console.log(typeof strToNegativeNum); // number
```

:::

Der logische NOT-Operator, dargestellt durch ein Ausrufezeichen (`!`), ist ein weiterer unärer Operator. Er kehrt den Boolean-Wert seines Operanden um. Wenn der Operand also `true` ist, wird er zu `false`, und wenn er `false` ist, wird er zu `true`.

:::interactive_editor

```js
let isOnline = true;
console.log(!isOnline); // false

let isOffline = false;
console.log(!isOffline); // true
```

:::

Der bitweise NOT-Operator ist ein weniger häufig verwendeter unärer Operator. Dargestellt durch eine Tilde, `~`, invertiert er die binäre Darstellung einer Zahl. Computer speichern Zahlen im Binärformat (1en und 0en). Der `~`-Operator kehrt jedes Bit um, das heißt, er ändert alle 1en in 0en und alle 0en in 1en. Mehr über Binärzahlen und Bits lernen Sie in einer zukünftigen Lektion.

:::interactive_editor

```js
const num = 5; // The binary for 5 is 00000101

console.log(~num); // -6
```

:::

In diesem Beispiel wurde `5` zu `-6`, weil Sie durch Anwenden des `~`-Operators auf `5` `- (5 + 1)` erhalten, was aufgrund der Zweierkomplement-Darstellung `-6` entspricht. Das Zweierkomplement ist eine Methode, mit der Computer negative Zahlen im Binärformat darstellen. Den bitweisen NOT-Operator werden Sie wahrscheinlich nicht oft verwenden, außer Sie arbeiten an niedrigstufigen Programmieraufgaben wie der direkten Manipulation von Bits.

Das Schlüsselwort `void` ist ein unärer Operator, der einen Ausdruck auswertet und `undefined` zurückgibt.

:::interactive_editor

```js
const result = void (2 + 2);

console.log(result); // undefined
```

:::

`void` wird auch häufig in Hyperlinks verwendet, um die Navigation zu verhindern:

```js
<a href="javascript:void(0);">Click Me</a>
```

Schließlich gibt es den `typeof`-Operator, den Sie in vorherigen Lektionen kennengelernt haben. Er gibt den Typ seines Operanden als String zurück.

:::interactive_editor

```js
const value = 'Hello world';

console.log(typeof value); // string
```

:::
