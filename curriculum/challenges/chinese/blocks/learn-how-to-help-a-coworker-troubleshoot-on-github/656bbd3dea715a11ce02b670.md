# --description--

在编码中，尤其是在使用像 Git 这样的版本控件系统时，`push` 意味着将更改从局部的仓库上传到远程仓库。 

过去式 `pushed` 表示此操作已完成。 

例如，`He pushed his code to the GitHub repository` 意味着他将最近的代码更改上传到了 GitHub。
