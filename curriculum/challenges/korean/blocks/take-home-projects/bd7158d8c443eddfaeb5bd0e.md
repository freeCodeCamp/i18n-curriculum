다음과 기능적으로 유사한 풀스택 JavaScript 앱을 만드세요: <a href="https://chart-the-stock-market.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://chart-the-stock-market.freecodecamp.rocks/</a>. 원하는 사이트 빌더를 사용해 프로젝트를 완성하세요.

이 프로젝트에서 구현해야 할 구체적인 사용자 스토리는 다음과 같습니다:

**사용자 스토리:** 추가한 각 주식의 최근 추세선을 보여주는 그래프를 볼 수 있습니다.

**사용자 스토리:** 주식 심볼 이름으로 새 주식을 추가할 수 있습니다.

**사용자 스토리:** 주식을 제거할 수 있습니다.

**사용자 스토리:** 다른 사용자가 주식을 추가하거나 제거할 때 실시간으로 변경 사항을 볼 수 있습니다. 이를 위해 Web Sockets를 사용해야 합니다.

완료하면 프로젝트의 작동하는 데모를 공개된 곳에 호스트했는지 확인하세요. 그런 다음 솔루션 링크 필드에 URL을 제출하세요. 선택 사항으로 GitHub 링크 필드에 프로젝트 소스 코드 링크도 제출할 수 있습니다.
