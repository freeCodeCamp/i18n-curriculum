`data-driven`

Refere-se a tomar decisões com base em dados em vez de intuição. Esta é uma palavra composta. Lembre-se de usar `-`.

# --explanation--

`Automation` é o uso da tecnologia para executar tarefas automaticamente sem intervenção humana. Por exemplo:

`The company uses automation to schedule social media posts every day.` - Isso significa usar ferramentas de software para agendar publicações sem entrada manual a cada vez.

`Data-driven` significa tomar decisões com base na análise de dados. Por exemplo:

`The team made a data-driven decision to improve website performance based on user activity data.` - Isso indica que a decisão foi baseada em dados reais sobre o comportamento do usuário, e não em suposições ou palpites.
