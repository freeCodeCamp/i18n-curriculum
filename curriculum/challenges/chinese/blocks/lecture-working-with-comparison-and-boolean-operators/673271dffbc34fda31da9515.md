# --interactive--

比较运算符允许你比较两个值，并返回 `true` 或 `false` 结果。 然后，你就可以利用结果做出决策或控制程序的流程。  You use comparisons in `if` statements, loops, and many other situations where you need to make decisions based on certain conditions. Let's dive into the most common comparison operators and see how they work.

大于操作符，用右角方括弧（`>`）表现，查看左边的值是否大于右边的值：

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a > b); // false
console.log(b > a); // true
```

:::

大于或相等操作符，由一个直角方括弧和等号（`>=`）表现，用于查看左侧的值是否大于或相等于右侧的值：

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a >= b); // false
console.log(b >= a); // true
console.log(a >= c); // true
```

:::

小于操作符，由左尖括弧（`<`）表现，工作方式类似于 `>`，但方向相反。它查看左边的值是否小于右边的值：

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a < b); // true
console.log(b < a); // false
```

:::

小于或相等操作符，由左尖括弧和等号（`<=`）表现，用于查看左侧的值是否小于或相等于右侧的值：

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a <= b); // true
console.log(b <= a); // false
console.log(a <= c); // true
```

:::
