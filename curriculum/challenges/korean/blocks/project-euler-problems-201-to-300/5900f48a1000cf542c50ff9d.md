# --description--

Barbara는 수학자이자 농구 선수입니다. 그녀는 거리 $x$에서 슛을 할 때 점수를 얻을 확률이 정확히 ($1 - \frac{x}{q}$)임을 발견했으며, 여기서 $q$는 50보다 큰 실수 상수입니다.

각 연습에서 그녀는 거리 $x = 1, x = 2, \ldots, x = 50$에서 슛을 시도하며, 기록에 따르면 총 20점을 정확히 얻을 확률이 2%입니다.

$q$를 찾아서 소수점 10자리까지 반올림하여 답을 제출하세요.
