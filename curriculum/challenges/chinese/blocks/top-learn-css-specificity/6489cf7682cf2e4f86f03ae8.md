# --description--

继承是指某些 CSS 属性，当应用于一个元素时，即使我们没有为该元素的后代显式编写规则，这些属性也会被该元素的后代继承。基于排版的属性（如 `color`、`font-size`、`font-family` 等）通常会被继承，而大多数其他属性则不会。

例外情况是直接定位到一个元素，因为这总是优先于继承：

```html
<!-- index.html -->

<div id="parent">
  <div class="child"></div>
</div>
```

```css
/* styles.css */

#parent {
  color: red;
}

.child {
  color: blue;
}
```

尽管 `parent` 元素具有带 ID 的更高特异性，但 `child` 元素会应用 `color: blue` 样式，因为该声明直接针对它，而来自父元素的 `color: red` 只是继承而来。
