# --explanation--

`On track` は、プロジェクトやタスクが計画通りに進んでいることを意味します。例えば：

`We're on track to finish the website next month.` — これはチームが目標を達成しそうだという意味です。

`Release` は、何かを開始したり公開したりすることを意味します。例えば：

`They plan to release the new version tomorrow.` — これはそれが一般に公開されることを意味します。

`By the end of` は、特定の時点までの期間を設定します。例えば：

`The report is due by the end of the day.` — これはその日の終わりまでに完了しなければならないことを意味します。
