# --description--

Un triplete de enteros positivos ($a$,$b$,$c$) se llama Triplete de Cardano si cumple la condición:

$$\sqrt[3]{a + b \sqrt{c}} + \sqrt[3]{a - b \sqrt{c}} = 1$$

Por ejemplo, (2,1,5) es un Triplete de Cardano.

Existen 149 Tripletes de Cardano para los cuales $a + b + c ≤ 1000$.

Encuentra cuántos Tripletes de Cardano existen tal que $a + b + c ≤ 110\\,000\\,000$.
