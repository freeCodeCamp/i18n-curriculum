# --explanation--

`However` — це слово-перехід, яке використовується на початку речення для введення протилежної ідеї. Наприклад:

`The task seemed easy. However, it turned out to be quite challenging.` — тут `however` протиставляє початкове очікування реальності.

`Documentation` означає письмові або цифрові записи, які пояснюють або деталізують конкретний процес, продукт чи систему. Наприклад:

`The team updated the project documentation to include the new features.` — у цьому випадку `documentation` стосується записів про проєкт.
