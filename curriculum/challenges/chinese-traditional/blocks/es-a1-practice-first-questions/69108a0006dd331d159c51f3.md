# --explanation--

這段 Mateo 和 Camila 之間的對話窗包含關於出身和國籍的常用片語和問題。 

短語 `Mucho gusto`（「很高興認識你」）用於禮貌地開始對話。

`¿De dónde eres?` 意思是「你來自哪裡？」（非正式）。

**`Soy de` + [country]** 意思是「我來自 + [country]」。

`Colombiana` 是指來自哥倫比亞的女性國籍形式。男性形式是 `colombiano`。

`Puertorriqueño` 是來自波多黎各的男性國籍形式。女性形式是 `puertorriqueña`。
