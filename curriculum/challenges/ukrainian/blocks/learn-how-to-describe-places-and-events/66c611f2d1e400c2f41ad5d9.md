# --explanation--

У `Present Simple` часі заперечна форма створюється за допомогою `doesn't` або `isn't` для третіх осіб однини (`he, she, it`).

- Використовуйте `doesn't` з дієсловами дії, щоб показати, що щось не відбувається, наприклад `It doesn't work.`

- Використовуйте `isn't` з формами `to be`, щоб показати, що щось не є правдою, наприклад `isn't set up.`

У цьому реченні вам знадобляться і `doesn't`, і `isn't`, щоб правильно описати, чому функція не працює, як очікувалося.
