# --explanation--

`Code reuse` 指的是在程序的新部分中使用现有代码的做法。例如：

`The "Login" class was used in multiple applications to handle user access, which is a good example of code reuse。` - 不用为每个应用编写登录逻辑，复用相同的代码，提高了效率。

`Extending` 意味着为已经存在的东西添加新建特色。例如：

`"AdminUser"` 类通过添加一个 `"approveUser()"` 方法扩展了基本的 `"User"` 类。- `AdminUser` 保留了 `User` 的所有内容，并在此基础上添加了新功能。
