# --explanation--

`Simplifying complex systems` significa hacer algo grande o difícil más fácil de usar o de entender.

`Hiding unnecessary details` significa mantener las partes internas (como cómo funciona algo) ocultas para el usuario, de modo que solo vea lo que necesita para interactuar.

La pregunta de Mark muestra que él entiende la abstracción como una forma de manejar la complejidad — no eliminando partes, sino organizando sistemas para que los usuarios solo traten con lo que es relevante.
