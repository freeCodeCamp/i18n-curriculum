# --description--

固定された整数 $a$、$b$、$c$ に対して、クレイジー関数 $F(n)$ を次のように定義します：

$$\begin{align}
  & F(n) = n - c \\;\text{ すべての } n > b \text{ に対して} \\\\
  & F(n) = F(a + F(a + F(a + F(a + n)))) \\;\text{ すべての } n ≤ b \text{ に対して}.
\end{align}$$

また、$S(a, b, c) = \displaystyle\sum_{n = 0}^b F(n)$ と定義します。

例えば、$a = 50$、$b = 2000$、$c = 40$ のとき、$F(0) = 3240$、$F(2000) = 2040$ となります。また、$S(50, 2000, 40) = 5\,204\,240$ です。

$S({21}^7, 7^{21}, {12}^7)$ の下9桁を求めてください。
