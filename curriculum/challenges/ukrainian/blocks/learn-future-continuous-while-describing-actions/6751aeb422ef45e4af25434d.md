# --explanation--

`Stress tests` — це оцінювання, створені для перевірки роботи системи під високим навантаженням або в екстремальних умовах. Наприклад:  

`The developers ran stress tests on the website to check its performance during peak traffic.` — це означає, що розробники тестували сайт, щоб переконатися, що він може одночасно обробляти велику кількість відвідувачів.

`Unexpected` означає щось неплановане або несподіване. Наприклад:  

`The system crashed due to unexpected behavior during the update.` — це означає, що система зламалася через проблеми, які не були передбачені під час оновлення.
