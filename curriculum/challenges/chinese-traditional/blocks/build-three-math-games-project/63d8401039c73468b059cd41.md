# --instructions--

在這個挑戰中，你需要使用 Python 創建三個數學遊戲，並完成以下功能：

- 散佈圖遊戲
  - 隨機在圖表上產生點，玩家必須輸入 (x,y) 座標。
  - 為了增加難度，將圖表做得更大
- 代數練習遊戲
  - 產生帶有隨機整數值的一步和兩步問題，玩家必須輸入答案
  - 使用正值和負值。若要增加難度，請使數字變得更大。
- 射擊遊戲
  - 顯示一個具有隨機高度和位置的「牆壁」。玩家必須移動滾軸來調整拋物線路徑以越過牆壁。
  - 為了增加難度，製作第二個層次，讓玩家在沒有滾軸的情況下輸入 a、b 和 c。
