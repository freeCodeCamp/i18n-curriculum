# --description--

ウェブブラウザはインターネット上のページを閲覧するために使うアプリケーションです。freeCodeCampのカリキュラムを進めているなら、今まさにウェブブラウザを使っている可能性が非常に高いです！

2024年現在、最も人気のあるブラウザはMicrosoft Edge、Firefox、Google Chrome、Safariです。他にも多くの選択肢があり、市場シェアは常に変動していますが、この4つが現時点で最も一般的です。

お使いのOSにはデフォルトでブラウザがインストールされていることが多いです。WindowsならMicrosoft Edge、macOSならSafari、LinuxならおそらくFirefoxが入っています。

では、別のブラウザを使いたい場合はどうすればいいでしょうか？

良いニュースは、すでにデフォルトのブラウザがあるということです。使いたいブラウザのウェブページにアクセスし、お使いのOSに合ったインストーラーをダウンロードすればいいのです。

ブラウザを選ぶ際は、速度やセキュリティ機能、よく訪れるウェブサイトとの互換性などを考慮してください。

ブラウザによっては独自の機能もあります。例えばOperaは組み込みのVPNを持ち、Braveはプライバシー保護と広告ブロックに重点を置いています。

インストールは通常簡単です。インストーラーをダウンロードしたら実行し、画面の指示に従ってください。

ほとんどのブラウザは、現在のデフォルトブラウザからブックマークや設定をインポートするオプションを提供しており、移行が楽になります。

または、macOSのHomebrewで`brew install --cask google-chrome` のようなOSのパッケージマネージャーを使ってブラウザをインストールすることもできます。Arch Linuxなら`yay -S brave-bin` などです。

利用できるパッケージマネージャーやパッケージ名はOSによって異なります。

覚えておいてほしいのは、ブラウザは1つに限定されないということです。多くの人は用途に応じて複数のブラウザを使い分けています。例えば仕事用とプライベート用で別々に使うこともあります。

いろいろなブラウザを試して、自分に合ったものを見つけてください。
