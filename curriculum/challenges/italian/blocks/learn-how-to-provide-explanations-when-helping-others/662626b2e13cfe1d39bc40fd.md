# --description--

`To act up` significa comportarsi male o non funzionare correttamente. Per esempio, se un telefono continua a bloccarsi, puoi dire `it's acting up`. È come qualcosa che non funziona come dovrebbe.

`To open` in questo contesto significa consultare o avviare qualcosa, come un file o un programma. Per esempio, `opening a document` significa consultarlo per visualizzarlo o modificarlo.
