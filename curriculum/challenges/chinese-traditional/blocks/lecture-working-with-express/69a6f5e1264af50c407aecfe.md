# --description--

在上一課中，你創建了一個專案資料夾並安裝了 Express：

```bash
mkdir my-app
cd my-app
npm init -y
npm install express
```

提醒你，這些指令會創建一個新的專案資料夾，`cd` 進入該專案資料夾，初始化專案並創建一個基本的 `package.json` 檔案，然後安裝 Express。

接著，打開程式碼編輯器，創建一個名為 `index.js` 的檔案，並撰寫你的第一個 Express 伺服器：

```javascript
const express = require("express");
const app = express();
const port = 3000;

app.get("/", (req, res) => {
  res.send("Hello World!");
});

app.listen(port, () => {
  console.log(`Example app listening on port ${port}`);
});
```

你可以在終端機中使用以下指令來執行你的程式碼：

```bash
node index.js
```

打開瀏覽器並前往 [http://localhost:3000](http://localhost:3000)，你會看到訊息 **"Hello World!"**

這就是使用 Express 快速開始的方式！現在讓我們逐行解說程式碼。

在第一行程式碼中，你正在載入 Express 函式庫並將它指定給一個名為 `express` 的變數：

```javascript
const express = require("express");
```

`express` 變數是一個工廠函式。當你呼叫它時，它會創建並傳回一個新的 Express 應用程式實體：

```javascript
const app = express();
```

`app` 現在可以存取 Express 提供的所有方法，例如 `.get()`、`.post()`、`.listen()` 等等。這是你的主要伺服器物件。

下一行定義了 `port` 變數。埠是你的應用程式監聽的網路端點，而 `3000` 是區域開發常用的埠號：

```javascript
const port = 3000;
```

現在，讓我們來看看下一段程式碼：

```javascript
app.get("/", (req, res) => {
  res.send("Hello World!");
});
```

這段程式碼定義了一個路由，並告訴伺服器如何回應請求。`app.get()` 註冊了一個處理常式來處理 HTTP GET 請求。GET 請求是用來取得資料的常用 HTTP 方法。

`"/"` 引數在 `app.get()` 中代表路由路徑。在這個例子中，它指向應用程式的根 URL。

`(req, res) => { ... }` 部分是一個回呼函式，當請求符合此路由時會執行。`req` 物件包含有關傳入請求的資訊，而 `res` 物件用來傳送回應給客戶端。

最後，`res.send("Hello World!")` 傳送回應給客戶端。在這個範例中，回應是純文字 `"Hello World!"`。

現在，讓我們來看看最後一段程式碼：

```javascript
app.listen(port, () => {
  console.log(`Example app listening on port ${port}`);
});
```

`app.listen()` 方法啟動 Express 應用程式，並告訴 Node 在指定的埠（此例為 `3000`）監聽傳入的 HTTP 請求。

伺服器啟動後，回呼函式會執行，並在主控台記錄訊息 `Example app listening on port 3000`，確認 Express 應用程式已成功啟動並準備接收請求。
