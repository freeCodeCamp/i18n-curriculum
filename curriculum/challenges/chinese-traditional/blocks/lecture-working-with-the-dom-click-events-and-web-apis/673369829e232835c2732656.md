# --interactive--

對話盒讓你向使用者顯示重要的資訊或操作。利用 HTML 內建的 `dialog` 元素，你可以輕鬆在你的網頁應用程式中創建這些對話盒（包含模態和非模態對話盒）。

模態對話窗是一種對話窗型別，強制使用者在能夠存取應用程式或網頁的其他部分之前與其互動。它有效地阻擋與其他內容的互動，直到使用者完成某個動作，例如關閉對話窗或提交表單。

相反地，非模態對話盒允許使用者在對話盒開啟時繼續與頁面或應用程式的其他零件互動。它不會阻止存取其餘內容。

在本課程中，你將學習如何使用 `showModal()` 和 `close()` 方法開啟及關閉模態視窗。

當你想確保使用者專注於模態視窗的特定動作或訊息時，你可以使用 `showModal()` 方法來開啟模態對話窗。這將為頁面上的其他條款添加背景遮罩並將其停用。這非常適合用於顯示表單、確認和需要使用者操作的重要資訊的模態視窗。 

以下是 modal 對話窗的 HTML：

:::interactive_editor

```html
<dialog id="my-modal">
  <p>This is a modal dialog.</p>
</dialog>
```

:::

目前，你無法在頁面上看到任何內容，因為 modal 在初始呈現時是關閉的。你可以使用 `showModal()` 方法自動開啟 modal：

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
dialog.showModal();
```

:::

瀏覽器中的結果將顯示一個帶有文字 `This is a modal dialog.` 的模態視窗。

最好將控制元件交給使用者。為了達成這一點，你可以為按鈕添加點擊事件監聽器，並使用 `showModal()` 方法：

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.showModal();
});
```

:::

如果你需要顯示對話窗，同時仍允許與對話窗外的內容互動，則你可以使用 `show()` 方法：

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});
```

:::

要關閉模式視窗，你可以在 `dialog` 元素內為模式視窗添加一個按鈕，並使用 `close()` 方法：

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
  <button id="close-modal-btn">Close Modal</button>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");
const closeButton = document.getElementById("close-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});

closeButton.addEventListener("click", () => {
  dialog.close();
});
```

:::

## --text--

模態對話窗與非模態對話窗有何不同？

## --answers--

模態對話盒允許與頁面其他部分互動，而非模態對話盒則不允許。
