# --description--

乗算と除算の演算子を解析して評価したので、加算と減算の演算子についても同様に処理する必要があります。

`infix` 変数を宣言し、数値（小数を含む）に続き `+` または `-` の演算子、その後に別の数値が続く正規表現を代入してください。
