# --interactive--

在之前的课程中，你学习了如何使用 `addEventListener()` 方法处理事件。但在 JavaScript 中还有另一种不推荐的处理事件的方式。

内联事件处理器是超文本标记语言元素上的特殊属性，用于在事件发生时执行 JavaScript 代码。

这是一个带有内联点击事件处理器的 `button` 元素示例：

:::interactive_editor

```html
<button onclick="alert('Hello World!')">Show alert</button>
```

:::

当用户点击按钮时，调用 `alert` 函数并显示带有消息 `Hello World!` 的警告对话框。

使用内联事件处理器的另一种方法是调用在 HTML 文档中的 `script` 标签中定义的函数。 

下面是定义一个名为 `changeBgColor` 的函数并从内联点击事件处理器调用它的示例：

:::interactive_editor

```html
<script>
  function changeBgColor() {
    document.body.style.backgroundColor = "lightblue";
  }
</script>

<button onclick="changeBgColor()">Change background color</button>
```

:::

当用户点击按钮时，会调用 `changeBgColor` 函数，页面的背景颜色会变为浅蓝色。虽然可以这样使用内联事件处理器，但这并不是最佳实践。 

有一个原因是，内联事件处理器只能用于为一个元素添加一个事件监听器。如果你想为同一个元素添加多个事件监听器，你需要使用 `addEventListener()`。另一个原因是内联事件处理器将超文本标记语言和 JavaScript 代码混合在一起，这会使你的代码更难读取和维护。最好通过使用 `addEventListener()` 将事件监听器添加到元素，从而将超文本标记语言代码和 JavaScript 代码分开。

内联事件处理器不推荐在现代 JavaScript 中使用。因此，在处理 JavaScript 中的事件时，最好坚持使用 `addEventListener()` 方法。
