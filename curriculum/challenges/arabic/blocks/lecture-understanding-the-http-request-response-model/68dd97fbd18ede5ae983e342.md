# --description--

في الدرس السابق، تعلمت أن الخادم يُحلل طلب HTTP ويرسل استجابة. تتضمن تلك الاستجابة أصولًا مختلفة تراها على الويب.

تشمل الأصول التي تُعاد في استجابة HTTP أصولًا نصية، وأصولًا وسائطية، وخطوطًا، وهكذا. كل أصل من هذه الأصول يأتي مع رأس **Content-Type**، الذي يخبر المتصفح بكيفية معالجة الأصل وعرضه بشكل صحيح.

كانت الأصول النصية دائمًا جزءًا أساسيًا من الويب. في الواقع، بدأ الويب كنص نقي. في الوقت الحالي، تشمل الأصول النصية ما يلي:

* **HTML (لغة الترميز النصي الفائق)**: الهيكل الأساسي لأي صفحة ويب. بدون HTML، لا يمكن لصفحة الويب أن توجد.
    
* **CSS (أوراق الأنماط المتتالية)**: الأصل الذي يتحكم في مظهر صفحة الويب. باستخدام CSS، تطبق أنماطًا مثل الألوان، والخطوط، والتخطيط على HTML.
    
* **JavaScript**: تضيف JavaScript التفاعلية والمحتوى الديناميكي إلى HTML وCSS.
    
* **JSON و XML**: هذان التنسيقان يسمحان لك بنقل البيانات المهيكلة بين الخوادم والمتصفحات.
    
تشمل الأصول الوسائطية الصور، والصوتيات، والفيديوهات. تجلب هذه الأصول الويب إلى الحياة من خلال الصور والصوت.

تعتمد العديد من المواقع على الصور لنقل المعلومات البصرية. تأتي الصور بصيغ مختلفة مثل `JPG`، و`JPEG`، و`PNG`، و`WEBP`، و`GIF`.

تقدم مواقع مثل YouTube وVimeo محتوى الفيديو بصيغ مثل `MP4`، و`MOV`، و`AVI`، و`WebM`، والمزيد.

الصوت جزء أساسي من العديد من المنصات إذ يسمح للمستخدمين ببث الموسيقى، والبودكاست، ومحتوى الصوت الآخر. تشمل صيغ الصوت الشائعة `MP3`، و`AAC`، وOgg، و`WAV`. تعتمد منصات مثل Spotify وYouTube Music على هذه الصيغ لتقديم تجارب صوتية عالية الجودة.

تشمل الأصول الأخرى التي تُعاد في استجابة HTTP مستندات مثل `PDF`، و`DOCX`، و`PAGES`، وأرشيفات مضغوطة مثل `ZIP` و`RAR`، وملفات خطوط مثل `WOFF`، و`WOFF2`، و`TTF`، وبرامج تنفيذية مثل `EXE`.

إليك جدولًا بهذه الأصول الشائعة وأنواع المحتوى الخاصة بها:

| نوع الأصل | رأس Content-Type |
| --- | --- |
| HTML | text/html |
| CSS | text/css |
| JavaScript | application/javascript |
| JPEG | image/jpeg |
| PNG | image/png |
| GIF | image/gif |
| SVG | image/svg+xml |
| WebP | image/webp |
| WOFF | font/woff |
| WOFF2 | font/woff2 |
| TTF | font/ttf |
| OTF | font/otf |
| MP4 | video/mp4 |
| WebM | video/webm |
| Ogg (فيديو) | video/ogg |
| MP3 | audio/mpeg |
| AAC | audio/aac |
| WAV | audio/wav |
| Ogg (صوت) | audio/ogg |
| JSON | application/json |
| XML | application/xml |
| PDF | application/pdf |
| ZIP | application/zip |
