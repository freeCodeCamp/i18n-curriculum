# --description--

دعنا نتعلم عن قالب HTML الجاهز.

ما هو قالب HTML الجاهز، تسأل؟ إنه مثل قالب جاهز لصفحات الويب الخاصة بك. فكر فيه كأساس للمنزل. القالب الجاهز يشمل الهيكل الأساسي والعناصر الضرورية التي يحتاجها كل مستند HTML. يوفر عليك الوقت ويساعد في ضمان إعداد صفحاتك بشكل صحيح. إليك مثالًا:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>freeCodeCamp</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
  </body>
</html>
```

دعنا نحلل الأجزاء الرئيسية لهذا القالب الجاهز. أولًا، هناك إعلان `DOCTYPE`:

```html
<!DOCTYPE html>
```

يخبر المتصفحات بإصدار HTML الذي تستخدمه. بعد ذلك، يأتي وسم `html`:

```html
<!DOCTYPE html>
<html lang="en">
  <!--All other elements go inside here-->
</html>
```

هذا يحيط بكل المحتوى الخاص بك، ويمكن أن يحدد لغة صفحتك. داخل وسم `html`، ستجد قسمين رئيسيين - `head` و `body`:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <!--Important metadata goes here-->
  </head>
  <body>
    <!--Headings, paragraphs, images, etc. go inside here-->
  </body>
</html>
```

قسم `head` يحتوي على معلومات مهمة خلف الكواليس:

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Document Title Goes Here</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

البيانات الوصفية لموقعك، الموجودة في عناصر `meta`، تحتوي على تفاصيل مثل ترميز الأحرف، وكيف يجب أن تعرض مواقع مثل Twitter رابط صفحتك. عنوان موقعك، الموجود في عنصر `title`، يحدد النص الذي يظهر في تبويب أو نافذة المتصفح. وأخيرًا، عادةً ما تربط أوراق الأنماط الخارجية لصفحتك في قسم `head` باستخدام عناصر `link`.

قسم `body` هو المكان الذي تضع فيه كل محتواك:

```html
<body>
  <h1>I am a main title</h1>
  <p>Example paragraph text</p>
</body>
```

لماذا القالب الجاهز مهم؟ لأنه يضمن أن صفحاتك منظمة بشكل صحيح وتعمل جيدًا عبر متصفحات مختلفة. استخدام قالب جاهز يساعدك على تجنب الأخطاء الشائعة واتباع أفضل الممارسات. إنه نقطة انطلاق رائعة لأي مشروع ويب. تذكر، يمكنك تخصيص قالبك الخاص ليناسب احتياجاتك. مع اكتساب الخبرة، قد تضيف عناصرك المفضلة أو وسوم `meta` الخاصة بك. مع استمرار تحسين قالبك الشخصي، ستجد أنه يوفر عليك الوقت عند بدء مشاريع جديدة.

في المرة القادمة التي تبدأ فيها ملف HTML جديد، فكر في استخدام قالب جاهز. سيمنحك بالتأكيد أساسًا قويًا للبناء عليه.
