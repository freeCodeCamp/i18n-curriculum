# --explanation--

Im **indirekten Rede** wird der Ausdruck `said that` verwendet, um wiederzugeben, was jemand gesagt hat, aber nicht mit den genauen Worten. Das Wort `that` ist optional und wird im Alltag oft weggelassen. Zum Beispiel:

- Direkte Rede: `We need to update the software.`

- Indirekte Rede: `He said that we need to update the software.` oder `He said we need to update the software.`

Beide Versionen sind korrekt und bedeuten dasselbe.
