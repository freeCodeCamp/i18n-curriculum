# --description--

Es hora de agregar algo de color al marcador. Recuerda que una forma de añadir color a un elemento es utilizando el nombre del <dfn>color</dfn> en inglés, como `black` (negro), `cyan` (cian) o `yellow` (amarillo).

Como recordatorio, así es como puedes dirigir a la clase `freecodecamp`:

```css
.freecodecamp {
  
}
```

Crea una nueva regla CSS que dirija a la clase `marker` y establece su propiedad `background-color` a `red`.

**Nota:** No verás cambios después de añadir el CSS.

# --hints--

Debes crear un selector de clases para apuntar a la clase `marker`.

```js
assert(new __helpers.CSSHelp(document).getStyle('.marker'));
```

La regla CSS de tu selector `.marker` debe tener una propiedad `background-color` con el valor `red` (rojo).

```js
assert(new __helpers.CSSHelp(document).getStyle('.marker')?.backgroundColor === 'red');
```
