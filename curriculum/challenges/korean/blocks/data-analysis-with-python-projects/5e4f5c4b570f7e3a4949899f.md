# --instructions--

1880년 이후 전 세계 평균 해수면 변화를 담은 데이터셋을 분석할 것입니다. 이 데이터를 사용해 2050년까지의 해수면 변화를 예측하세요.

다음 작업을 완료하기 위해 데이터를 사용하세요:

- Pandas를 사용해 `epa-sea-level.csv`에서 데이터를 가져오세요.
- matplotlib를 사용해 `Year` 열을 x축으로, `CSIRO Adjusted Sea Level` 열을 y축으로 하는 산점도를 만드세요.
- `linregress`에서 `scipy.stats` 함수를 사용해 최적 적합선의 기울기와 y절편을 구하세요. 산점도 위에 최적 적합선을 그리세요. 이 선이 2050년을 지나도록 만들어 2050년 해수면 상승을 예측하세요.
- 2000년부터 데이터셋의 최신 연도까지의 데이터만 사용해 새로운 최적 적합선을 그리세요. 이 선도 2050년을 지나도록 만들어 2000년 이후 상승률이 계속된다면 2050년 해수면 상승을 예측하세요.
- x축 레이블은 `Year`, y축 레이블은 `Sea Level (inches)`, 제목은 `Rise in Sea Level`이어야 합니다.

보일러플레이트에는 이미지 저장 및 반환 명령도 포함되어 있습니다.
