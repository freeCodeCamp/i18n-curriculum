# --interactive--

يُستخدم عنصر `button` لأداء إجراء معين عند تفعيله. إليك مثالًا على عنصر `button` مع نص الزر `Start Game`.

```html
<button>Start Game</button>
```

تشمل أمثلة أخرى لاستخدام عنصر `button` إرسال نموذج، عرض نافذة منبثقة، أو تبديل قائمة جانبية مفتوحة ومغلقة. يحتوي عنصر `button` على خاصية `type` التي تتحكم في سلوك الزر عند تفعيله. القيمة الأولى الممكنة لخاصية `type` هي النوع `button`. إليك مثالًا على استخدام عنصر `button` مع النوع `button` ونص `Show Alert`:

```html
<button type="button">Show Alert</button>
```

افتراضيًا، لن يفعل الزر أي شيء عند تفعيله. مع ذلك، يمكنك إضافة شفرة JavaScript لجعل الزر تفاعليًا، مثل عرض تنبيه في هذه الحالة.

انقر على زر `Show Alert` لترى تنبيهًا يظهر على الشاشة. للتفاعل مع المثال، تحتاج إلى تفعيل المحرر التفاعلي.


**ملاحظة**: يستخدم هذا المثال التفاعلي JavaScript لكن لا تحتاج للقلق بشأن فهم شفرة JavaScript. ستتعلم عنها في الوحدات القادمة.

:::interactive_editor

```html
<button type="button">Show Alert</button>
<script src="index.js"></script>
```

```js
const btn = document.querySelector("button");
btn.addEventListener("click", () => alert("You clicked on the alert button"));
```

:::

قيمة أخرى ممكنة لخاصية `type` هي القيمة `submit`. إليك مثالًا على استخدام عنصر `button` مع النوع `submit`.

:::interactive_editor

```html
<form action="">
  <label for="email">Email address:</label>
  <input type="email" id="email" name="email" />
  <button type="submit">Submit form</button>
</form>
```

:::

داخل عنصر `form` هذا، يوجد عنصر `label` وعنصر `input` لعنوان البريد الإلكتروني للمستخدم. عند نقر المستخدم على زر الإرسال، تُرسل بياناته إلى الخادم ليتم معالجتها. القيمة الثالثة الممكنة لخاصية `type` هي القيمة `reset`. إليك مثالًا على عنصر `form` يحتوي على أزرار إعادة تعيين وإرسال.

فعّل المحرر التفاعلي وتفاعل مع مُدخَل البريد الإلكتروني في نافذة المعاينة بإدخال بريد إلكتروني وهمي. ثم انقر على زر إعادة التعيين لترى بريدك الإلكتروني يختفي من الحقل.

:::interactive_editor

```html
<form action="">
  <label for="email">Email address:</label>
  <input type="email" id="email" name="email" />
  <button type="reset">Reset form</button>
  <button type="submit">Submit form</button>
</form>
```

:::

في هذا المثال المعدل، يُستخدم عنصر `label` وعنصر `input` لجمع عنوان البريد الإلكتروني للمستخدم. عند نقر المستخدم على زر إعادة التعيين، يمسح كل بيانات المُدخَل الخاصة به. من المهم ملاحظة أن أزرار إعادة التعيين عادةً ليست فكرة جيدة لأنها قد تؤدي إلى إعادة تعيين البيانات عن طريق الخطأ. أيضًا، وجود عدة أزرار في النموذج قد يزدحم واجهة المستخدم.

طريقة أخرى لإنشاء أزرار في HTML هي استخدام عنصر `input`. يحتوي عنصر `input` أيضًا على خاصية `type` بالقيم الممكنة `submit` و`reset` و`button`. إليك مثالًا على استخدام عنصر `input` مع تعيين `type` إلى `button`:

:::interactive_editor

```html
<input class="start-btn" type="button" value="Start Game" />
<script src="index.js"></script>
```

```js
document.addEventListener("DOMContentLoaded", () => {
  const btn = document.querySelector(".start-btn");
  btn.addEventListener("click", () => {
    const paraEl = document.createElement("p");
    const bodyEl = document.querySelector("body");

    bodyEl.appendChild(paraEl);
    paraEl.textContent = "The game has started!!!";
  });
});
```

:::

تُستخدم خاصية `value` لعرض نص الزر. فما الفرق بين استخدام عنصري `input` و`button`؟ عناصر `input` هي عناصر فارغة، مما يعني أنها لا يمكن أن تحتوي على عقد فرعية مثل النص، ويمكن أن يكون لها وسم بداية فقط. من ناحية أخرى، يوفر عنصر `button` مرونة أكبر لأنه يمكنك تسكين نصوص وصور وأيقونات بداخله.
