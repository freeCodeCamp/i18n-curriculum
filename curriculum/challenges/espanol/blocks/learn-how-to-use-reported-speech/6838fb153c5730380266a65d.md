# --explanation--

Un `patch` es una pequeña actualización lanzada por desarrolladores de software para corregir problemas, mejorar el rendimiento o solucionar problemas de seguridad. Por ejemplo:

`The company released a patch to fix the login error.` – Esto significa que publicaron una solución para el problema.

`To exploit` significa usar algo de una manera que no es justa o que causa daño, especialmente aprovechándose de una debilidad. Por ejemplo:

`Hackers exploited a flaw in the email server.` – Esto significa que usaron la debilidad para obtener acceso o causar daño.
