# --description--

在之前的課程中，你學到了什麼是相依性以及如何為你的專案選擇正確的相依性。

在本課程中，你將學習如何使用 npm 安裝和移除依賴項。

要安裝相依性，請使用 `npm install` 指令，如下所示：

```bash
npm install dependencyName
```

你也可以將 `install` 縮寫為 `i`，如下所示：

```bash
npm i dependencyName
```

安裝軟體包後，npm 會在你的 `package.json` 檔案中新增一個 `dependencies` 區段（如果尚未存在），並列出你安裝的軟體包。舉例來說，當你安裝 `chalk` 軟體包時，畫面會是這樣：

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "chalk": "^5.6.2"
  }
}
```

當你安裝相依性時，兩個新項目會出現在你的專案根目錄中：

* `node_modules/`：一個資料夾，包含你安裝的軟體包的實際程式碼。
    
* `package-lock.json`：一個儲存那些軟體包精確版本詳細資訊的檔案，包括它們自己的相依性。（你會在後面的課程中學到更多關於這個檔案的內容。）
    

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/how-can-you-install-and-remove-dependencies-1.png" alt="">

大多數專案需要多於一個相依性。npm 讓你可以一次安裝多個套件，而不是一個一個安裝，只要輸入以空格分隔的套件名稱列表即可：

```bash
npm install express lodash bcrypt
```

就像之前一樣，當你一次安裝多個相依性時，npm 會更新你的 `package.json` 和 `package-lock.json` 檔案以及你的 `node_modules`，以包含它們。以下是 `package.json` 的範例：

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "bcrypt": "^6.0.0",
    "chalk": "^5.6.2",
    "express": "^5.1.0",
    "lodash": "^4.17.21"
  }
}
```

要安裝 `devDependency`，請為 `install` 指令添加 `--save-dev` 旗標。你也可以使用較短的 `-D` 版本：

```bash
npm i nodemon -D
```

在執行此指令後，npm 會更新你的 `package.json` 檔案，並將該軟體包添加到 `devDependencies` 區段。以下是範例：

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "bcrypt": "^6.0.0",
    "chalk": "^5.6.2",
    "express": "^5.1.0",
    "lodash": "^4.17.21"
  },
  "devDependencies": {
    "nodemon": "^3.1.10"
  }
}
```

每當你執行 `npm install` 指令時，npm 會顯示一則訊息告訴你安裝是否成功。成功的訊息通常看起來像這樣：

```bash
user@Kolade ~/Desktop/fCC/script-code/node/npm/install-uninstall-deps  % npm install express lodash bcrypt

added 72 packages, and audited 74 in 8s

17 packages are looking for funding
  run `npm fund` for details

found 0 vulnerabilities

user@Kolade ~/Desktop/fCC/script-code/node/npm/install-uninstall-deps  % npm install nodemon -D

added 27 packages, and audited 101 in 16s

21 packages are looking for funding
  run `npm fund` for details

found 0 vulnerabilities
```

要解除安裝或移除軟體包，請使用 `npm uninstall` 指令：

```bash
npm uninstall dependencyName
```

例如，要從你的專案中移除 `chalk`，你可以執行這個指令：

```bash
npm uninstall chalk
```

你也可以透過以空格分隔每個軟體包名稱來一次卸載多個軟體包：

```bash
npm uninstall express lodash bcrypt
```

你也會收到一則訊息，顯示解除安裝指令是成功還是失敗：

```bash
user@Kolade ~/Desktop/fCC/script-code/node/npm/install-uninstall-deps  % npm uninstall chalk

added 1 package, removed 1 package, and audited 101 packages in 1s

20 packages are looking for funding
  run `npm fund` for details

found 0 vulnerabilities

user@Kolade ~/Desktop/fCC/script-code/node/npm/install-uninstall-deps  % npm uninstall express lodash bcrypt

removed 70 packages, and audited 31 in 1s

4 packages are looking for funding
  run `npm fund` for details

found 0 vulnerabilities
```

你的 `package.json` 檔案將會自動更新：

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "devDependencies": {
    "nodemon": "^3.1.10"
  }
}
```

要移除 `devDependency`，你只需執行：

```bash
npm uninstall nodemon
```

請注意，解除安裝時你不需要 `-D` 旗標。npm 將自動從正確的區段中移除軟體包，在此情況下為 `devDependencies`。

在解除安裝 `devDependency` 後，你的 `package.json` 將會再次更新：

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC"
}
```
