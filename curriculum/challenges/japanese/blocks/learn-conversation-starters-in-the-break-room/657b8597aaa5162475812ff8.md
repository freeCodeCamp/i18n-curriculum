# --description--

誰かが将来したいことについて話すとき、`want to`を使って願望や計画を表せます。例えば：

- `I want to visit Paris next summer.` - これは将来そこに行きたいという意味です。

- `He wants to learn guitar.` - これは彼がいつかギターを弾きたい、または計画しているという意味です。
