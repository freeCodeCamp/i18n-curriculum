## --text--

Quando è appropriato usare una struttura `switch-case` piuttosto che una struttura `if-elseif-else`?

## --answers--

Una struttura `switch-case` è appropriata quando sono necessari più di 2-3 blocchi di codice `else if`.

---

Una struttura `switch-case` è appropriata quando il numero di pattern `case` è piccolo.

---

Una struttura `switch-case` è appropriata quando l'istruzione di selezione si trova all'interno di un loop.
