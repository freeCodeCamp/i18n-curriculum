# --explanation--

`Real-world` significa algo que acontece na vida real, não em um teste ou em uma situação inventada.Por exemplo:  

`We tested the new app in real-world conditions to see how it works.` - Isso significa que o app foi testado em situações normais do dia a dia para garantir que funcione bem.

`Feedback` significa conselhos ou comentários para ajudar alguém a melhorar.Por exemplo:  

`The project manager gave feedback on my work so I can do better next time.` - Isso significa que o gerente de projeto oferece conselhos para ajudar você a melhorar seu trabalho.

`Roll it out` significa lançar ou liberar um produto ou serviço para um público mais amplo.Por exemplo:

`We will roll out the new feature to all users next month.` - Isso indica que a nova funcionalidade será lançada para uso público, passando de teste para estar disponível para todos.
