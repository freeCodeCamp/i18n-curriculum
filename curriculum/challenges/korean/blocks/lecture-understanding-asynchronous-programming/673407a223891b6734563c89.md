# --description--

Fetch API는 웹 애플리케이션이 네트워크 요청을 할 수 있게 해 주며, 보통 서버로부터 데이터를 가져오거나 보내는 데 사용됩니다. 이 API는 이러한 요청을 할 때 사용할 수 있는 `fetch()` 메서드를 제공합니다. `fetch`를 사용하는 기본 예제를 살펴보겠습니다:

```js
fetch('https://api.example.com/data')
```

이 예제에서는 `GET`에 `https://api.example.com/data` 요청을 하고 있습니다. 그러면 JSON 형식으로 변환해야 하는 데이터를 반환하며, 이 데이터를 원하는 곳 어디서든 사용할 수 있습니다.

Fetch API는 기본적으로 데이터를 가져올 때 `GET` 메서드를 사용합니다. 이 메서드와 다른 일반적인 HTTP 요청 메서드는 다음 수업에서 다룰 예정입니다.

이제 네트워크에서 가져오는 일반적인 리소스 유형에 대해 이야기해 보겠습니다.

웹 애플리케이션에서는 날씨 데이터, 직업 목록 데이터, 국가 이름 목록, 국가 코드 또는 국가 깃발 아이콘/이미지 같은 일반적인 데이터가 필요합니다. 이러한 데이터를 사용하면 애플리케이션을 더 정보성 있고 상호작용적으로 만들 수 있습니다. Fetch API 덕분에 네트워크에서 이러한 리소스를 가져올 수 있습니다.

이미지는 자주 가져오는 리소스 중 하나입니다. 사용자의 동작에 따라 이미지를 정적으로 또는 동적으로 불러와 웹 애플리케이션에 표시할 때 fetch를 사용할 수 있습니다.

텍스트 파일도 자주 가져오는 리소스 유형입니다. 여기에는 구성 파일, 로그 파일 또는 웹페이지에 표시하려는 전체 문서가 포함될 수 있습니다.

경우에 따라 오디오나 비디오 파일을 가져올 수도 있습니다. Fetch API는 이러한 유형의 리소스도 처리할 수 있어 다양한 데이터 유형을 다룰 수 있습니다.
