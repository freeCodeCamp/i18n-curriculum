# --description--

6174 — це особливе число; якщо відсортувати його цифри за зростанням і відняти це число від числа, отриманого сортуванням цифр за спаданням, отримаємо $7641 - 1467 = 6174$.

Ще більш дивовижно, що якщо почати з будь-якого 4-значного числа і повторювати цей процес сортування та віднімання, ми врешті-решт отримаємо 6174 або одразу 0, якщо всі цифри однакові.

Це також працює з числами, у яких менше ніж 4 цифри, якщо доповнити число провідними нулями до 4 цифр.

Наприклад, почнемо з числа 0837:

$$\begin{align}
  & 8730 - 0378 = 8352 \\\\
  & 8532 - 2358 = 6174
\end{align}$$

6174 називають константою Капрекара. Процес сортування, віднімання і повторення цього, доки не буде досягнуто 0 або константи Капрекара, називають рутинною Капрекара.

Можна розглянути рутинну Капрекара для інших основ числення та кількості цифр. На жаль, не гарантовано, що константа Капрекара існує у всіх випадках; рутина може потрапити у цикл для деяких вхідних чисел або константа, до якої рутина приходить, може бути різною для різних вхідних чисел. Проте можна довести, що для 5 цифр і основи $b = 6t + 3 ≠ 9$ існує константа Капрекара.

Наприклад:
основа 15: ${(10, 4, 14, 9, 5)}\_{15}$
основа 21: $(14, 6, 20, 13, 7)\_{21}$

Визначимо $C_b$ як константу Капрекара в основі $b$ для 5 цифр. Визначимо функцію $sb(i)$ так:

- 0, якщо $i = C_b$ або якщо $i$, записане в основі $b$, складається з 5 ідентичних цифр
- кількість ітерацій, які потрібні рутині Капрекара в основі $b$, щоб дійти до $C_b$, інакше

Зверніть увагу, що $sb(i)$ можна визначити для всіх цілих чисел $i < b^5$. Якщо $i$, записане в основі $b$, має менше ніж 5 цифр, число доповнюють провідними нулями до 5 цифр перед застосуванням рутини Капрекара.

Визначимо $S(b)$ як суму $sb(i)$ для $0 < i < b^5$. Наприклад, $S(15) = 5\,274\,369$, $S(111) = 400\,668\,930\,299$.

Знайдіть суму $S(6k + 3)$ для $2 ≤ k ≤ 300$. Наведіть останні 18 цифр як відповідь.
