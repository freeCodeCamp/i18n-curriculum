# --interactive--

이전 수업에서 JavaScript에서 `Date` 객체를 다루는 방법을 배웠습니다. 하지만 JavaScript에서 날짜를 포맷하는 몇 가지 다른 방법이 있습니다.

이번 수업에서는 JavaScript에서 날짜를 포맷하기 위해 `toISOString()`, `toString()`, 그리고 `toLocaleDateString()` 메서드를 사용하는 방법을 살펴보겠습니다.

다양한 메서드를 보기 전에, 먼저 `Date` 객체가 어떻게 생겼는지 검토해 보겠습니다:

:::interactive_editor

```js
const date = new Date();
console.log(date);
```

:::

`date` 객체를 콘솔에 출력하면 사용자의 시스템 설정을 기준으로 현재 날짜와 시간이 표시됩니다. 다음은 출력 예시입니다:

```js
Sun Sep 29 2024 19:45:37 GMT-0700 (Pacific Daylight Time)
```

만약 `toString()` 객체에 `date` 메서드를 사용한다면 위와 같은 출력 결과를 보게 됩니다.

날짜를 확장된 ISO 형식(ISO 8601)으로 포맷하려면 다음과 같이 `toISOString()` 메서드를 사용할 수 있습니다:

:::interactive_editor

```js
const date = new Date();
console.log(date.toISOString());
```

:::

ISO 8601은 날짜와 시간을 나타내는 국제 표준입니다. 형식은 `YYYY-MM-DDTHH:mm:ss.sssZ` 입니다.

다음은 출력 예시입니다:

```js
2024-09-30T02:47:20.292Z
```

날짜를 포맷하는 또 다른 방법은 `toLocaleDateString()` 메서드를 사용하는 것입니다. 이 메서드는 사용자의 로케일을 기준으로 날짜를 포맷할 수 있게 해줍니다. `toLocaleDateString()` 메서드를 사용하는 기본 구문은 다음과 같습니다:

:::interactive_editor

```js
const date = new Date();
console.log(date.toLocaleDateString());
```

:::

다음은 출력 예시입니다:

```js
9/29/2024
```

`toLocaleDateString()` 메서드는 두 개의 선택적 매개변수인 `locales` 과 `options` 를 받습니다.

`locales` 매개변수는 사용할 로케일을 나타내는 문자열입니다. 예를 들어, 영어(미국)를 위해 `en-US` 를, 프랑스어(프랑스)를 위해 `fr-FR` 를 전달할 수 있습니다. `locales` 매개변수를 전달하지 않으면 기본 로케일이 사용됩니다. `locales` 매개변수를 사용하는 예시는 다음과 같습니다:

:::interactive_editor

```js
const date = new Date();
console.log(date.toLocaleDateString("fr-FR"));
```

:::

두 번째 선택적 매개변수는 `options` 매개변수입니다. 이 매개변수는 날짜 문자열의 형식을 지정할 수 있는 객체입니다. `options` 매개변수를 사용하는 예시는 다음과 같습니다:

:::interactive_editor

```js
const date = new Date();
const options = {
  weekday: "long",
  year: "numeric",
  month: "long",
  day: "numeric",
};
console.log(date.toLocaleDateString("en-GB", options));
```

:::

위 예시에서는 옵션을 영어(영국)로 지정하고, 요일 전체 이름, 연도, 월, 일을 포함하도록 설정했습니다.

다음은 출력 예시입니다:

```js
Sunday, September 29, 2024
```

이번 수업에서는 JavaScript에서 날짜를 포맷하는 몇 가지 방법만 다뤘습니다. 날짜를 포맷하는 데 도움이 되는 다른 많은 메서드와 라이브러리가 있습니다. 하지만 `toISOString()` 와 `toLocaleDateString()` 메서드는 JavaScript에서 날짜를 포맷하는 좋은 출발점입니다.
