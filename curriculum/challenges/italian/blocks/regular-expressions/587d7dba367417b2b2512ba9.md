# --description--

<dfn>Lookahead</dfn> sono pattern che dicono a JavaScript di guardare avanti nella tua stringa per verificare pattern più avanti. Questo può essere utile quando vuoi cercare più pattern nella stessa stringa.

Esistono due tipi di lookahead: <dfn>lookahead positivo</dfn> e <dfn>lookahead negativo</dfn>.

Un lookahead positivo verifica che l'elemento nel pattern di ricerca sia presente, ma non lo corrisponde effettivamente. Un lookahead positivo si usa come `(?=...)` dove `...` è la parte richiesta che non viene corrisposta.

Al contrario, un lookahead negativo verifica che l'elemento nel pattern di ricerca non sia presente. Un lookahead negativo si usa come `(?!...)` dove `...` è il pattern che non vuoi che sia presente. Il resto del pattern viene restituito se la parte di lookahead negativo non è presente.

I lookahead possono essere un po' complicati ma alcuni esempi aiutano.

```js
let quit = "qu";
let noquit = "qt";
let quRegex= /q(?=u)/;
let qRegex = /q(?!u)/;
quit.match(quRegex);
noquit.match(qRegex);
```

Entrambe queste chiamate `match` restituirebbero `["q"]`.

Un uso più pratico dei lookahead è controllare due o più pattern in una stringa. Ecco un controllo password (naïf) semplice che cerca tra 3 e 6 caratteri e almeno un numero:

```js
let password = "abc123";
let checkPass = /(?=\w{3,6})(?=\D*\d)/;
checkPass.test(password);
```
