# --explanation--

`Need` wird immer gefolgt von `to` + Infinitiv verwendet, wenn es mit einer anderen Handlung kombiniert wird. Diese Struktur drückt Notwendigkeit oder Erfordernis aus.

Jake sagt, `We need to find out how this happened`. In diesem Satz:

- `Need` drückt aus, dass etwas erforderlich ist.

- `To find out` folgt dem korrekten Muster (`need to` + Infinitiv) und bedeutet, etwas herauszufinden oder zu lernen.
