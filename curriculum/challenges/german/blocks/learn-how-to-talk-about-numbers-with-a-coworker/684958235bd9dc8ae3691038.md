`by next week`

### --feedback--

Diese drei Wörter zeigen eine Frist an.

# --explanation--

`Installed` bedeutet, dass Software auf einem Gerät eingerichtet und angewendet wird, sodass sie richtig funktioniert. Zum Beispiel:

`The installed program is working fine.` — Das eingerichtete Programm funktioniert gut.

`By next day`, `by next week`, `by next month` bedeuten spätestens bis zum Ende des nächsten Tages/Woche/Monats. Zum Beispiel:

- `The update should be ready by next day.` — Das Update wird am folgenden Tag abgeschlossen sein.

- `We plan to finish testing by next month.` — Das Testen sollte vor Ende des nächsten Monats erledigt sein.

Tom sagt, dass diese Software-Updates dringend sind und innerhalb eines bestimmten Zeitrahmens abgeschlossen werden müssen.
