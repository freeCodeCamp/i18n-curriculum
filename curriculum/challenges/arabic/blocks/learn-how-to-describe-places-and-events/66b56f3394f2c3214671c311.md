# --explanation--

`To roll out` تعني إطلاق أو تقديم شيء جديد، مثل منتج أو خدمة أو مشروع. تُستخدم غالبًا في سياقات الأعمال والتكنولوجيا. على سبيل المثال:
 
- `The company plans to roll out a new software update next month.` - تخطط الشركة لإطلاق التحديث الجديد للشركة الشهر المقبل.

`By the end of` تُستخدم لتحديد موعد نهائي أو وقت يجب إكمال شيء ما قبله. على سبيل المثال:

- `I need to finish this report by the end of the day.` - يجب إكمال التقرير قبل نهاية اليوم.
