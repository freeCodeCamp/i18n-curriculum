# --description--

抽象メソッドとして認識されるには、メソッドは`@abstractmethod`デコレーターで装飾されている必要があります。

<dfn>デコレーター</dfn>はPythonで関数の動作を変更するために使われます。`spam`という名前のデコレーターの使い方の例を示します。

```py
@spam
def foo():
    pass
```

インポート文を修正して`abstractmethod`デコレーターをインポートし、`solve`クラスの`analyze`メソッドと`Equation`メソッドの両方を装飾してください。これにより2つの例外が発生します。

`ABC`を継承したクラスに抽象メソッドがある場合、そのクラスはもうインスタンス化できません。したがって、エラーを解消するために`Equation`インスタンスを削除してください。

もう一つのエラーは、`LinearEquation`クラスがインターフェースで定義されたすべての抽象メソッドを実装しなければならないために発生します。`LinearEquation`クラス内にもそれらを定義してください。具象クラスで`abstractmethod`デコレーターを使ってはいけません。
