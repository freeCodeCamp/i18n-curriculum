# --description--

تعتمد تطبيقات React غالبًا على واجهات برمجة التطبيقات الخارجية وقواعد البيانات للمحتوى الديناميكي. للوصول إلى البيانات من تلك الواجهات وقواعد البيانات، تحتاج إلى استخدام بعض تقنيات جلب البيانات.

لنلقِ نظرة على كيفية عمل جلب البيانات في React والخيارات المختلفة المتاحة لك لجلب البيانات.

React لا تفرض طريقة معينة لجلب بياناتك، وهذا يعني على المستوى الأساسي، يمكنك استخدام واجهة Fetch API المدمجة، التي تدعمها جميع المتصفحات الحديثة.

يمكنك أيضًا استخدام Axios وSWR. Axios هي مكتبة طلبات HTTP قائمة على الوعود مبنية فوق كائن XMLHttpRequest، وSWR هو hook في React لجلب البيانات أنشأه فريق Vercel.

لنبدأ بمثال. تحتاج أولًا إلى استيراد الـ `useState` و `useEffect` hooks:

```js
import { useState, useEffect } from "react";
```

بعدها ستحتاج إلى إنشاء ثلاثة متغيرات حالة تسمى `loading` و `data` و `error`:

```js
const [data, setData] = useState(null);
const [loading, setLoading] = useState(true);
const [error, setError] = useState(null);
```

المتغير `loading` يتتبع ما إذا كانت البيانات لا تزال تُجلب. المتغير `data` يمثل البيانات نفسها، والمتغير `error` يلتقط أي أخطاء قد تحدث أثناء عملية جلب البيانات.

بما أن جلب البيانات هو تأثير جانبي، فمن الأفضل استخدام Fetch API داخل hook `useEffect`.

إليك مثال على ذلك:

```js
useEffect(() => {
  fetch("https://jsonplaceholder.typicode.com/posts")
    .then((res) => res.json())
    .then((data) => {
      setData(data);
      setLoading(false);
    })
    .catch((err) => {
      setError(err);
      setLoading(false);
    });
}, []);
```

هذا الـ `useEffect` يجلب البيانات باستخدام Fetch API ويضبط جميع الحالات.

يمكنك تحسين الأمور باستخدام `async`/`await` بدلاً من صيغة `.then()`. هذا يعني أنه يجب أن يكون لديك دالة منفصلة داخل `useEffect` لأنك لا تستطيع إضافة بادئة `async` على `useEffect` مباشرة:

```js
useEffect(() => {
  const fetchData = async () => {
    try {
      const res = await fetch("https://jsonplaceholder.typicode.com/posts");
       
      if (!res.ok) {
        throw new Error("Network response was not ok");
      }

      const data = await res.json();
      setData(data);
    } catch (err) {
      setError(err);
    } finally {
      setLoading(false);
    }
  };

  fetchData();
}, []);
```

بعدها يمكنك استخدام كل تلك الحالات لعرض البيانات من API.

إليك الشفرة الكاملة:

```jsx
import { useState, useEffect } from "react";

const FetchPosts = () => {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    const fetchData = async () => {
      try {
        const res = await fetch("https://jsonplaceholder.typicode.com/posts");
       
        if (!res.ok) {
          throw new Error("Network response was not ok");
        }

        const data = await res.json();
        setData(data);
      } catch (err) {
        setError(err);
      } finally {
        setLoading(false);
      }
    };

    fetchData();
  }, []);

  if (loading) {
    return <p>Loading...</p>;
  }

  if (error) {
    return <p>{error.message}</p>;
  }

  return (
    <ul>
      {data.map((post) => (
        <li key={post.id}>{post.title}</li>
      ))}
    </ul>
  );
};

export default FetchPosts;
```

في واجهة المستخدم، سترى `Loading...` على الشاشة أثناء جلب البيانات، ثم تظهر البيانات أو الخطأ حسب نجاح جلب البيانات.

تذكر أننا تحدثنا أيضًا عن جلب البيانات باستخدام Axios وSWR. لنلقِ نظرة على مثال باستخدام Axios.

أولًا ستحتاج إلى تثبيت Axios من سطر الأوامر هكذا:

```sh
npm i axios
```

بعدها ستحتاج إلى استيراد Axios هكذا:

```js
import axios from "axios";
```

يمكنك بعدها استخدام نفس متغيرات الحالة السابقة وجلب البيانات من API باستخدام `axios.get`:

```js
const [data, setData] = useState(null);
const [loading, setLoading] = useState(true);
const [error, setError] = useState(null);

useEffect(() => {
  const fetchData = async () => {
    try {
      const res = await axios.get(
        "https://jsonplaceholder.typicode.com/users"
      );
      setData(res.data);
    } catch (err) {
      setError(err);
    } finally {
      setLoading(false);
    }
  };

  fetchData();
}, []);
```

ربما لاحظت أنه لا يوجد سطر `await res.json()` في هذا المثال. ذلك لأن Axios يقوم بتحليل JSON تلقائيًا، فلا حاجة لذلك.

المثال الأخير الذي سننظر إليه هو استخدام hook `useSWR` لجلب البيانات.

تمامًا كما هو الحال مع Axios، ستحتاج إلى تثبيت SWR هكذا:

```sh
npm install swr
```

بعدها ستحتاج إلى استيراد hook `useSWR` إلى الملف هكذا:

```js
import useSWR from "swr";
```

مقارنة بالأمثلة السابقة، صيغة SWR أقصر بكثير. ما عليك فعله هو إنشاء دالة fetcher وتمريرها إلى hook `useSWR` كمعامل ثاني (نقطة النهاية هي المعامل الأول).

كما يمكنك تفكيك كل من حالات البيانات والخطأ من hook `useSWR`، لذلك لا تحتاج إلى hook `useState`.

إليك الصيغة:

```js
const fetcher = (url) => fetch(url).then((res) => res.json());
const { data, error } = useSWR(endpoint, fetcher);
```

لاحظ أن اسم "fetcher" هنا هو مجرد اصطلاح، لذا يمكنك تسمية المتغير كما تريد.

إليك مكوِّن يجلب مهام من JSON Placeholder API:

```jsx
import useSWR from "swr";

const fetcher = (url) => fetch(url).then((res) => res.json());

const FetchTodos = () => {
  const { data, error } = useSWR(
    "https://jsonplaceholder.typicode.com/todos",
    fetcher
  );

  if (!data) {
    return <h2>Loading...</h2>;
  }
  if (error) {
    return <h2>Error: {error.message}</h2>;
  }

  return (
    <>
      <h2>Todos</h2>
      <div>
        {data.map((todo) => (
          <h3 key={todo.id}>{todo.title}</h3>
        ))}
      </div>
    </>
  );
};

export default FetchTodos;
```

كما تعلمت في درس سابق عن الـ custom hooks، جلب البيانات هو منطق يمكنك استخراجه إلى hook مخصص. لذلك، إذا كنت تجلب البيانات في مكونات وصفحات متعددة، فمن الأفضل إنشاء hook `useFetch`.

إليك hook `useFetch` الذي يستخدم SWR لجلب البيانات:

```jsx
import useSWR from "swr";

const fetcher = (url) => fetch(url).then((res) => res.json());

const useFetch = (url) => {
  const { data, error } = useSWR(url, fetcher);

  return {
    data,
    loading: !data && !error,
    error,
  };
};

export default useFetch;
```

وهنا كيفية استخدام hook `useFetch` لإعادة كتابة المثال الأول الذي يجلب المنشورات من JSON Placeholder API:

```jsx
import useFetch from "./useFetch";

const FetchPosts = () => {
  const { data, loading, error } = useFetch(
    "https://jsonplaceholder.typicode.com/posts"
  );

  if (loading) {
    return <h2>Loading...</h2>;
  }

  if (error) {
    return <h2>{error.message}</h2>;
  }

  return (
    <>
      <h2>Posts</h2>
      <ul>
        {data.map((post) => (
          <li key={post.id}>{post.title}</li>
        ))}
      </ul>
    </>
  );
};

export default FetchPosts;
```
