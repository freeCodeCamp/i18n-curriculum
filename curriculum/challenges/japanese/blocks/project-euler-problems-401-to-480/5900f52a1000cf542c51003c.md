# --description--

すべての整数 $n > 1$ に対して、関数族 $f_{n, a, b}$ は次のように定義されます：

$f_{n, a, b}(x) ≡ ax + b\bmod n$ ただし $a, b, x$ は整数で、$0 \lt a \lt n$、$0 \le b \lt n$、$0 \le x \lt n$。

すべての $0 \le x \lt n$ に対して $f_{n, a, b}(f_{n, a, b}(x)) \equiv f_{n, a, b}(x)\bmod n$ が成り立つ場合、$f_{n, a, b}$ を退縮と呼びます。

$R(n)$ を $n$ に対する退縮の数とします。

次のことが与えられています：

$$\sum_{k = 1}^{99\\,999} R(\displaystyle\binom{100\\,000}{k}) \equiv 628\\,701\\,600\bmod 1\\,000\\,000\\,007$$

$$\sum_{k = 1}^{9\\,999\\,999} R(\displaystyle\binom{10\\,000\\,000}{k})$$ を求めてください。答えは $1\\,000\\,000\\,007$ で割った余りで答えてください。
