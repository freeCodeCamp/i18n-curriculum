# --interactive--

Der `typeof`-Operator in JavaScript ist ein einfaches, aber mächtiges Werkzeug, mit dem Sie den Datentyp einer Variablen oder eines Werts sehen können. Er gibt immer einen String zurück, der den Typ angibt.

Schauen wir uns ein paar Beispiele an:

:::interactive_editor

```js
let num = 42;
console.log(typeof num); // "number"
```

:::

In diesem ersten Beispiel haben wir eine Variable namens `num` erstellt und ihr die Zahl `42` zugewiesen. Wenn Sie den `typeof`-Operator auf die Variable mit dem Namen `num` anwenden, gibt er den String `number` zurück.

Hier ist ein weiteres Beispiel für die Verwendung des `typeof`-Operators auf einer Variablen namens `isUserLoggedIn`:

:::interactive_editor

```js
let isUserLoggedIn = true;
console.log(typeof isUserLoggedIn); // "boolean"
```

:::

Wenn Sie den `typeof`-Operator auf die `isUserLoggedIn`-Variable anwenden, gibt er einen String `boolean` zurück, weil der Boolean `true` der Variablen zugewiesen wurde.

Die Verwendung des `typeof`-Operators kann besonders nützlich sein, wenn Sie debuggen oder versuchen zu verstehen, mit welcher Art von Daten Sie in Ihrem Code arbeiten.

Es gibt jedoch eine bekannte Eigenheit in JavaScript, wenn es um `null` geht.

Schauen wir uns ein Beispiel an:

:::interactive_editor

```js
let exampleVariable = null;
console.log(typeof exampleVariable); // "object"
```

:::

In diesem Beispiel haben wir eine Variable namens `exampleVariable` und haben ihr den Wert `null` zugewiesen. Aber wenn wir den `typeof`-Operator verwenden, gibt er den String `object` zurück.

Dies wird allgemein als Bug in JavaScript angesehen, der bis in die frühen Tage der Sprache zurückreicht. Der Grund für dieses Verhalten liegt in der Art und Weise, wie JavaScript ursprünglich entworfen wurde.

Als die Sprache erstmals implementiert wurde, wurden Werte wie `null` als eine spezielle Art von Objekt dargestellt, was zu diesem unerwarteten Ergebnis führte.

Leider ist dies ein Teil der Sprache geworden, und obwohl es verwirrend ist, müssen Sie sich dessen bewusst sein.
