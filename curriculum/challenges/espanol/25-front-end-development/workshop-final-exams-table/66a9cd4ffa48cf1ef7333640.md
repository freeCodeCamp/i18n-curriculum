# --description--

Su tabla está casi completa, pero queda algo más por añadir.

Sería bueno si el elemento `td` usado para el `Promedio de Calificaciones` abarcará dos columnas en lugar de solo una.

Para hacer esto, puede usar el atributo <dfn>colspan</dfn> en el elemento `td`.

El atributo `colspan` especifica el número de columnas que debe abarcar una celda.

Aquí hay un ejemplo del atributo `colspan` usado para una tabla de deportes:

```html
<tr>
  <td colspan="3">Total Points</td>
</tr>
```

Dentro de la etiqueta de apertura `td`, agregue el atributo `colspan` y asígnelo a `"2"`.

¡Y con ese cambio, su tabla está completa!
