# --description--

Застосунки React часто покладаються на зовнішні API та бази даних для динамічного вмісту. Щоб отримати дані з цих API та баз даних, потрібно використовувати певні техніки отримання даних.

Давайте розглянемо, як працює отримання даних у React і які варіанти доступні для отримання даних.

React не нав’язує, як саме ви маєте отримувати свої дані, це означає, що на базовому рівні ви можете використовувати вбудований Fetch API, який підтримують усі сучасні браузери.

Ви також можете використовувати Axios і SWR. Axios — це бібліотека HTTP-запитів на основі промісів, побудована поверх об’єкта XMLHttpRequest, а SWR — це React-хук для отримання даних, створений командою Vercel.

Почнемо з прикладу. Спочатку потрібно імпортувати хуки `useState` та `useEffect`:

```js
import { useState, useEffect } from "react";
```

Потім потрібно створити три змінні стану з іменами `loading`, `data` та `error`:

```js
const [data, setData] = useState(null);
const [loading, setLoading] = useState(true);
const [error, setError] = useState(null);
```

Змінна `loading` відстежуватиме, чи ще триває отримання даних. Змінна `data` представляє самі дані, а змінна `error` зафіксує будь-які помилки, які можуть виникнути під час процесу отримання даних.

Оскільки отримання даних — це побічний ефект, найкраще використовувати Fetch API всередині хука `useEffect`.

Ось приклад цього:

```js
useEffect(() => {
  fetch("https://jsonplaceholder.typicode.com/posts")
    .then((res) => res.json())
    .then((data) => {
      setData(data);
      setLoading(false);
    })
    .catch((err) => {
      setError(err);
      setLoading(false);
    });
}, []);
```

Цей `useEffect` отримує дані за допомогою Fetch API і встановлює всі стани.

Ви можете покращити код, використовуючи `async`/`await` замість синтаксису `.then()`. Це означає, що потрібно мати окрему функцію всередині `useEffect`, бо не можна префіксувати `useEffect` ключовим словом `async`:

```js
useEffect(() => {
  const fetchData = async () => {
    try {
      const res = await fetch("https://jsonplaceholder.typicode.com/posts");
       
      if (!res.ok) {
        throw new Error("Network response was not ok");
      }

      const data = await res.json();
      setData(data);
    } catch (err) {
      setError(err);
    } finally {
      setLoading(false);
    }
  };

  fetchData();
}, []);
```

Потім ви можете використовувати всі ці стани, щоб відтворити дані з API.

Ось повний код:

```jsx
import { useState, useEffect } from "react";

const FetchPosts = () => {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    const fetchData = async () => {
      try {
        const res = await fetch("https://jsonplaceholder.typicode.com/posts");
       
        if (!res.ok) {
          throw new Error("Network response was not ok");
        }

        const data = await res.json();
        setData(data);
      } catch (err) {
        setError(err);
      } finally {
        setLoading(false);
      }
    };

    fetchData();
  }, []);

  if (loading) {
    return <p>Loading...</p>;
  }

  if (error) {
    return <p>{error.message}</p>;
  }

  return (
    <ul>
      {data.map((post) => (
        <li key={post.id}>{post.title}</li>
      ))}
    </ul>
  );
};

export default FetchPosts;
```

У інтерфейсі користувача ви побачите `Loading...` на екрані, коли дані отримуються, а потім відобразяться дані або помилка залежно від того, чи було успішно отримано дані.

Пам’ятаєте, ми також говорили про отримання даних за допомогою Axios і SWR. Давайте розглянемо приклад із Axios.

Спочатку потрібно встановити Axios через командний рядок так:

```sh
npm i axios
```

Потім потрібно імпортувати Axios так:

```js
import axios from "axios";
```

Потім можна використовувати ті самі змінні стану, що й раніше, і отримувати дані з API за допомогою `axios.get`:

```js
const [data, setData] = useState(null);
const [loading, setLoading] = useState(true);
const [error, setError] = useState(null);

useEffect(() => {
  const fetchData = async () => {
    try {
      const res = await axios.get(
        "https://jsonplaceholder.typicode.com/users"
      );
      setData(res.data);
    } catch (err) {
      setError(err);
    } finally {
      setLoading(false);
    }
  };

  fetchData();
}, []);
```

Ви, мабуть, помітили, що в цьому прикладі немає рядка `await res.json()`. Це тому, що Axios автоматично аналізує JSON, тож у цьому немає потреби.

Останній приклад — використання хука `useSWR` для отримання даних.

Як і з Axios, потрібно встановити SWR так:

```sh
npm install swr
```

Потім потрібно імпортувати хук `useSWR` у файл так:

```js
import useSWR from "swr";
```

Порівняно з попередніми прикладами, синтаксис SWR значно коротший. Вам потрібно створити функцію fetcher і передати її у хук `useSWR` як другий параметр (кінцева точка — це перший параметр).

Також ви можете деструктурувати і стан даних, і стан помилки з хука `useSWR`, тому хук `useState` не потрібен.

Ось синтаксис:

```js
const fetcher = (url) => fetch(url).then((res) => res.json());
const { data, error } = useSWR(endpoint, fetcher);
```

Зверніть увагу, що ім’я "fetcher" тут — це лише конвенція, тож ви можете назвати змінну як завгодно.

Ось компонент, який отримує завдання з JSON Placeholder API:

```jsx
import useSWR from "swr";

const fetcher = (url) => fetch(url).then((res) => res.json());

const FetchTodos = () => {
  const { data, error } = useSWR(
    "https://jsonplaceholder.typicode.com/todos",
    fetcher
  );

  if (!data) {
    return <h2>Loading...</h2>;
  }
  if (error) {
    return <h2>Error: {error.message}</h2>;
  }

  return (
    <>
      <h2>Todos</h2>
      <div>
        {data.map((todo) => (
          <h3 key={todo.id}>{todo.title}</h3>
        ))}
      </div>
    </>
  );
};

export default FetchTodos;
```

Як ви дізналися в попередньому уроці про кастомні хуки, логіку отримання даних можна винести у кастомний хук. Тож якщо ви отримуєте дані в кількох компонентах і сторінках, найкраще створити `useFetch` хук.

Ось `useFetch` хук, який використовує SWR для отримання даних:

```jsx
import useSWR from "swr";

const fetcher = (url) => fetch(url).then((res) => res.json());

const useFetch = (url) => {
  const { data, error } = useSWR(url, fetcher);

  return {
    data,
    loading: !data && !error,
    error,
  };
};

export default useFetch;
```

А ось як використати хук `useFetch`, щоб переписати перший приклад, який отримує пости з JSON Placeholder API:

```jsx
import useFetch from "./useFetch";

const FetchPosts = () => {
  const { data, loading, error } = useFetch(
    "https://jsonplaceholder.typicode.com/posts"
  );

  if (loading) {
    return <h2>Loading...</h2>;
  }

  if (error) {
    return <h2>{error.message}</h2>;
  }

  return (
    <>
      <h2>Posts</h2>
      <ul>
        {data.map((post) => (
          <li key={post.id}>{post.title}</li>
        ))}
      </ul>
    </>
  );
};

export default FetchPosts;
```
