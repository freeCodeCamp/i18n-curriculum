# --interactive--

Wenn Sie mit CSS eine Webseite gestalten, ist eine der gebräuchlichsten Methoden zur Definition von Farben die Verwendung von Hex-Farbwerten. Hexadezimale Farbwerte, oder Hex-Codes, sind eine kompakte Möglichkeit, Farben im RGB-Farbmodell darzustellen. In dieser Lektion werden wir untersuchen, was Hex-Codes sind, wie sie in CSS funktionieren und wann Sie sie verwenden sollten.

Ein Hex-Code (Kurzform für hexadezimaler Code) ist ein sechsstelliger String, der verwendet wird, um Farben im RGB-Farbmodell darzustellen. Das „Hex“ bezieht sich auf das Zahlensystem zur Basis 16, das die Ziffern `0` bis `9` und die Buchstaben `A` bis `F` verwendet.

Im Kontext von Farben geben Hex-Codes die Mengen an Rot, Grün und Blau (RGB) an, aus denen eine bestimmte Farbe besteht. Jeder Hex-Code beginnt mit einem Rautezeichen (`#`), gefolgt von sechs Zeichen, die die Intensität von Rot, Grün und Blau in der Farbe definieren.

Zum Beispiel ist der Hex-Code für Schwarz `#000000`, und der Hex-Code für Weiß ist `#FFFFFF`. Die ersten zwei Zeichen stellen den Rotwert dar, die nächsten zwei den Grünwert und die letzten zwei den Blauwert.

In CSS können Sie Farben auf Elemente anwenden, indem Sie Hex-Codes verwenden. Hier ist die grundlegende Syntax:

```css
element {
  color: #RRGGBB;
}
```

Hier ist `#RRGGBB` ein Platzhalter für den tatsächlichen Hex-Code. Die Paare `RR`, `GG` und `BB` repräsentieren jeweils die Intensität von Rot, Grün und Blau.

Diese Paare können von `00` (niedrigste Intensität) bis `FF` (höchste Intensität) reichen. Je höher die Zahl, desto mehr von dieser Farbe ist im Endergebnis enthalten. Zum Beispiel wenden wir einige Hex-Farben auf ein paar Elemente an:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h1>This is a heading.</h1>
<p>This is a paragraph.</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}
p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

Im ersten Fall ist `#FF5733` eine rötlich-orange Farbe. Hier die Aufschlüsselung:

- `FF` ist der höchste Wert für Rot (voll gesättigt).
- `57` ist eine moderate Menge Grün.
- `33` ist eine kleine Menge Blau.

Im zweiten Fall ist `#4CAF50` ein Grünton:

- `4C` ist ein niedriger Wert für Rot.
- `AF` ist ein starker Wert für Grün.
- `50` ist ein moderater Wert für Blau.

Hex-Codes sind beliebt, weil sie präzise und einfach zu verwenden sind. Sie ermöglichen es Ihnen, Ihre Farben fein abzustimmen, was sie ideal für Designs macht, bei denen Konsistenz wichtig ist.

Die meisten Designprogramme, wie Adobe Photoshop oder Figma, liefern Hex-Codes für die von Ihnen ausgewählten Farben, was es einfach macht, diese Werte direkt in Ihre CSS-Dateien zu kopieren und einzufügen.

In der Webentwicklung werden Hex-Codes häufig verwendet, um Textfarben, Hintergründe, Rahmen und andere visuelle Elemente zu definieren. Zum Beispiel:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example</button>
```

```css
body {
  background-color: #F0F0F0; /* Light gray background */
}

button {
  background-color: #007BFF; /* A shade of blue */
  color: #FFFFFF; /* White text */
}
```

:::

So stellen Sie sicher, dass Ihre Farben auf der gesamten Website oder Anwendung konsistent bleiben.

In manchen Fällen sehen Sie Hex-Codes in Kurzform, die nur drei Zeichen anstelle von sechs verwenden. Das ist möglich, wenn beide Zeichen in jedem Farbpaare gleich sind. Zum Beispiel kann `#FF5733` nicht verkürzt werden, aber `#FFFFFF` (Weiß) kann als `#FFF` geschrieben werden.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h2>This is a heading.</h2>
```

```css
body {
  background-color: #000;
}

h2 {
  color: #FFF; /* White in shorthand */
}
```

:::

Die Kurzform `#FFF` entspricht `#FFFFFF`. Ebenso würde `#000` Schwarz darstellen, was in der Langform `#000000` ist.

Hex-Codes sind beliebt wegen ihrer Genauigkeit und Kompatibilität und ermöglichen über 16 Millionen Farben in allen Browsern. Sie sind ideal, wenn Sie bestimmte Markenfarben benötigen, genaue Farbtöne steuern wollen, die nicht leicht durch benannte Farben oder HSL definiert werden können, oder wenn Sie Farben aus Designprogrammen übernehmen, die standardmäßig Hex-Codes liefern.
