# --description--

Sie können die Variable `__name__` verwenden, um festzustellen, ob ein Python-Skript als Hauptprogramm ausgeführt wird oder ob es als Modul importiert wird (Code, der in einer anderen Python-Datei geschrieben ist).

Wenn der Wert von `__name__` auf `'__main__'` gesetzt ist, bedeutet das, dass das aktuelle Skript das Hauptprogramm ist und kein Modul.

In diesem Projekt verwenden Sie das aktuelle Skript als Hauptprogramm.

Erstellen Sie eine `if`-Anweisung, die überprüft, ob der Wert von `__name__` `'__main__'` ist.

Verwenden Sie das Schlüsselwort `pass` im Körper der `if`-Anweisung.

    if len(array) <= 1:
        return
   
    middle_point = len(array) // 2
    left_part = array[:middle_point]
    right_part = array[middle_point:]

    merge_sort(left_part)
    merge_sort(right_part)

    left_array_index = 0
    right_array_index = 0
    sorted_index = 0

    while left_array_index < len(left_part) and right_array_index < len(right_part):

            left_array_index += 1
