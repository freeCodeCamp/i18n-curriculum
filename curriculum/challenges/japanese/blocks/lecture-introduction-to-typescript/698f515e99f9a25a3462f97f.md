# --description--

このレッスンでは、関数型の扱い方を学びます。まずはパラメータの型注釈から始めましょう。

基本的な関数の例を示します。

```ts
function circleArea(radius) {
  return Math.PI * radius * radius;
}
```

現在、TypeScriptは`Parameter 'radius' implicitly has an 'any' type`というメッセージを表示します。`any`型は値が任意の型を持つ可能性があることを示します。この型については今後のレッスンで詳しく学びます。

エラーを解消し、パラメータに明示的に型を付けるには、次のようにします。

```ts
function circleArea(radius: number) {
  return Math.PI * radius * radius;
}
```

`circleArea`関数を数値以外の型で呼び出そうとすると、TypeScriptは次のようなエラーを表示します。

```ts
function circleArea(radius: number) {
  return Math.PI * radius * radius;
}

// Argument of type 'string' is not assignable to parameter of type 'number'
circleArea("3");
```

関数が期待するより多くの引数を渡そうとすると、TypeScriptはエラーを表示します。

```ts
function circleArea(radius: number) {
  return Math.PI * radius * radius;
}

// Expected 1 arguments, but got 2.
circleArea(3, 4);
```

引数が不足している場合も、TypeScriptはエラーを表示します。

```ts
function circleArea(radius: number) {
  return Math.PI * radius * radius;
}

// Expected 1 arguments, but got 0.
// An argument for 'radius' was not provided.
circleArea();
```

パラメータをオプションにするには、パラメータの近くに`?`記号を使います。

```ts
function area(radius: number, height?: number): number {
  const baseArea = Math.PI * radius * radius;

  if (height !== undefined) {
    const sideArea = 2 * Math.PI * radius * height;
    return baseArea + sideArea;
  }

  return baseArea;
}
```

関数内の条件チェックは、`height`引数が渡されなかった場合に備えています。

戻り値に型を付けたい場合は、次のように戻り値の型注釈を追加できます。

```ts
function circleArea(radius: number): number {
  return Math.PI * radius * radius;
}
```

戻り値の型注釈はパラメータリストの後に書きます。変数の型注釈と同様に、多くの場合TypeScriptは戻り値の型を賢く推論します。関数の意図を明確にしたり、戻り値の意図しない変更を防ぐために、明示的な戻り値の型注釈を使うことをお勧めします。

次の例は非同期関数とPromiseを扱います。関数がPromiseを返す場合は、次のように`Promise`型を使えます。

```ts
async function getCityTemperature(city: string): Promise<number> {
  const response = await fetch(`https://api.example.com/weather?city=${city}`);
  const data = await response.json(); 
  return data.temperature;
}
```

この場合、戻り値の型を省略するとTypeScriptは`Promise<any>`として推論することがあります。したがって、戻り値の型を明示的に指定するのが良い状況です。

ここまで名前付き関数の例だけを見てきましたが、無名関数では型はどのように機能するのでしょうか？

次は`forEach`に渡されるコールバックで無名関数を使った例です。

```ts
const languages = ["JavaScript", "Python", "TypeScript"];

languages.forEach((lang) => {
  console.log(`${lang.toUpperCase()} has ${lang.length} characters`);
});
```

`lang`パラメータには型注釈がありません。代わりにTypeScriptはコンテキスト型付けを使い、現れる文脈から型を推論します。
