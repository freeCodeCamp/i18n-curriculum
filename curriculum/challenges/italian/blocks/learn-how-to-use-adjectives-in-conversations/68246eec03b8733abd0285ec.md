# --explanation--

`Support` significa gestire o permettere a qualcosa di funzionare correttamente. Per esempio:  

`The app can support 1,000 users at the same time.` – Può gestire quel numero senza bloccarsi.

`At once` significa fare più cose contemporaneamente. Per esempio:

`The software can open many files at once.` – Tutti i file si aprono insieme, non uno per uno.
