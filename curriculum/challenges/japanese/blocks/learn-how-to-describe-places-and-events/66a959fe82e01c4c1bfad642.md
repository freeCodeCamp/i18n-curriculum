# --explanation--

`Might` は可能性を表すために使われます。何かが可能であるけれども確実ではないことを意味します。例えば：

- `I might go to the party if I finish my work.` - 行くかもしれないけど、確かではない。

- `She might call you later.` - 彼女が電話をかけるかもしれないけど、確実ではない。
