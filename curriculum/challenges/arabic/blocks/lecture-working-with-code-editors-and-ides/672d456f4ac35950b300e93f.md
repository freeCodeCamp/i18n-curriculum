# --description--

من المحتمل أنك تعرف بالفعل بعض الاختصارات الأساسية، التي ورثتها من نظام التشغيل الخاص بك. اختصارات مثل `Ctrl + S` للحفظ، و`Ctrl + C` للنسخ، و`Ctrl + V` للصق تعمل كلها في VS Code.

لكن هناك بعض الاختصارات الخاصة بالتطبيق والتي يمكنها رفع إنتاجيتك. يجدر بالذكر أن بعض هذه الاختصارات قد تختلف حسب نظام التشغيل.

على سبيل المثال، `Shift + Alt + F` يشغل أداة التنسيق المهيأة لديك (مثل prettier، لمشروع JavaScript) على الملف المفتوح حاليًا.

أو `Ctrl + Shift + F` (ويندوز)، أو `Cmd + Shift + F` (ماك)، للبحث في محتوى النص لجميع الملفات في مساحة العمل. ثم `Ctrl + Shift + H` (ويندوز)، أو `Cmd + Shift + H` (ماك)، إذا أردت تشغيل البحث والاستبدال.

إذا احتجت إلى حذف سطر، `Ctrl + Shift + K` (ويندوز)، أو `Cmd + Shift + K` (ماك)، سيحذفه.

هل تحتاج إلى مساحة إضافية لكل شفرتك؟ `Ctrl + B` (ويندوز)، أو `Cmd + B` (ماك)، يخفي الشريط الجانبي - الذي يحتوي على قائمة الملفات وقائمة الإضافات.

أو ربما لا تستطيع رؤية شفرتك؟ `Ctrl + plus` (ويندوز)، أو `Cmd + plus` (ماك)، يزيد من تكبير المحرر، و`Ctrl + minus` (ويندوز)، أو `Cmd + minus` (ماك)، يقلله.

ميزة أخرى مفيدة في VS Code هي التحرير متعدد الأسطر، الذي يسمح لك بإجراء تغييرات عبر عدة أسطر دفعة واحدة. على سبيل المثال، يمكنك وضع عدة مؤشرات باستخدام `Ctrl + Alt + down` و`Ctrl + Alt + up` (ويندوز/لينكس)، أو `Option + Command + down` و`Option + Command + up` (ماك)، لتمديد المؤشر إلى أسطر متتالية. يمكنك أيضًا الضغط مع الاستمرار على `Alt` (ويندوز/لينكس) أو `Option` (ماك) والنقر لإضافة مؤشرات على أسطر منفصلة حيثما تحتاجها. بمجرد وضع المؤشرات، أي شيء تكتبه أو تحذفه سيطبق على كل تلك المواقع في نفس الوقت — مما يوفر الكثير من الوقت عند العمل على شفرة متكررة.

أخيرًا، إذا نسيت أيًا من هذه الاختصارات، لديك دائمًا `Ctrl + Shift + P` (ويندوز)، أو `Cmd + Shift + P` (ماك)، التي تفتح لوحة الأوامر لتختار ما تحتاجه.

مع هذه المعرفة، وربما بعض الممارسة، أنت في طريقك لتصبح مستخدمًا محترفًا لـ VS Code.
