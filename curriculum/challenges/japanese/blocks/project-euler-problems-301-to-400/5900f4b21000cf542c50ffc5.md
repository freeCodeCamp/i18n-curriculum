# --description--

数列 $a_n$ を次のように再帰的に定義します：$a_1 = 1$、$\displaystyle a_n = \left(\sum_{k = 1}^{n - 1} k \times a_k\right)\bmod n$。

したがって、$a_n$ の最初の10項は次の通りです：1、1、0、3、0、3、5、4、1、9。

関数 $f(N, M)$ を、次の条件を満たす組 $(p, q)$ の数とします：

$$ 1 \le p \le q \le N \\; \text{かつ} \\; \left(\sum_{i = p}^q a_i\right)\bmod M = 0$$

$f(10, 10) = 4$ であり、その組は (3,3)、(5,5)、(7,9)、(9,10) であることがわかります。

また、$f({10}^4, {10}^3) = 97\\,158$ であることも与えられています。

$f({10}^{12}, {10}^6)$ を求めてください。
