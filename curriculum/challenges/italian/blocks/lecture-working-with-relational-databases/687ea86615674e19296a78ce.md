# --description--

I database relazionali hanno un ampio campo di applicazioni, tra cui sviluppo web, sistemi di gestione dell'inventario, sistemi sanitari, applicazioni di e-commerce e altro ancora.

Alcuni dei database relazionali più comuni sono MySQL, PostgreSQL, SQLite e Microsoft SQL Server. Scopriamo qualcosa su ciascuno di loro.

MySQL è un sistema di gestione di database relazionale open source. È molto popolare per lo sviluppo web perché è facile da usare, affidabile ed efficiente per costruire applicazioni reali. Ha anche una comunità attiva di sviluppatori che lo supportano e lo mantengono.

PostgreSQL è ufficialmente descritto come "Il database relazionale open source più avanzato al mondo". È un sistema di database oggetto-relazionale gratuito e open source noto per la sua affidabilità, integrità dei dati ed estensibilità. Per esempio, puoi definire i tuoi tipi di dati, funzioni personalizzate e scrivere codice in diversi linguaggi di programmazione senza dover ricompilare il database.

SQLite è un motore di database SQL gratuito, open source e leggero basato su file. È uno dei motori di database più usati al mondo. SQLite è autonomo e senza server. È un database a zero configurazione, il che significa che non richiede un processo di configurazione iniziale, ottimo per iniziare rapidamente.

Questi tre database sono gratuiti e open source. Ci sono anche altre opzioni proprietarie popolari, tra cui Microsoft SQL Server e Oracle database.

Microsoft SQL Server è un sistema di gestione di database relazionale popolare sviluppato da Microsoft e usato in varie applicazioni.

Oracle database è un sistema di gestione di database relazionale scalabile spesso usato per applicazioni aziendali che richiedono alte prestazioni.

Ottimo. Ora che conosci meglio le diverse opzioni disponibili, vediamo come puoi installare PostgreSQL.

Per prima cosa, devi andare sul sito ufficiale: <a href="https://www.postgresql.org/" target="_blank">https://www.postgresql.org/</a>.

Una volta lì, clicca sul pulsante "Download".

Questo ti porterà a una pagina dove puoi selezionare il tuo sistema operativo. Se clicchi su uno, verrai indirizzato alle istruzioni per quel sistema operativo.

Passeremo attraverso il processo di installazione per il sistema operativo Windows, ma il processo è simile per tutti i sistemi operativi.

Da questa pagina, puoi scaricare l'installer cliccando sul collegamento "Download the installer".

Questo ti porterà a una pagina dove puoi scegliere la versione di PostgreSQL che vuoi installare. Se clicchi sull'icona corrispondente, il download dovrebbe partire.

Poi, nella cartella Download o nella cartella dove salvi i download per impostazione predefinita, dovresti vedere l'installer.

Se fai doppio clic su di esso, inizierai il processo di configurazione. Clicca su Avanti.

Scegli la directory dove PostgreSQL sarà installato. Vedrai una di default ma puoi personalizzarla.

Poi, seleziona il componente che vuoi installare. Ci sono quattro componenti possibili:

- PostgreSQL Server, il motore del database stesso.
- pgAdmin 4, uno strumento user-friendly per la gestione del database.
- Stack Builder, uno strumento per scaricare e installare software e estensioni aggiuntive relative a PostgreSQL
- E Command Line Tools, per interagire direttamente con il server PostgreSQL dal terminale. Include psql, un terminale interattivo per eseguire query SQL.

Quando sei pronto, clicca su Avanti.

Poi, seleziona una directory per archiviare i tuoi dati. Vedrai un percorso predefinito ma puoi personalizzarlo. Quando sei pronto, clicca su Avanti.

Ora dovrai inserire una password per il superutente del database, l'utente con accesso illimitato a tutti i database e oggetti. Digita la tua password, riscrivila e poi clicca su Avanti.

Poi, seleziona il numero di porta su cui il server deve ascoltare. Per impostazione predefinita è 5432. Poi clicca su Avanti.

Dovrai scegliere una localizzazione per il nuovo cluster di database. Questo è importante affinché il database interpreti correttamente i tipi di dati sensibili alla localizzazione. Poi clicca su Avanti.

Infine, vedrai un riepilogo pre-installazione. Prenditi un momento per rivederlo e clicca su Avanti.

Dovrai confermare di voler iniziare il processo di installazione. Se clicchi su Avanti, il processo partirà. Dovrebbe durare solo pochi minuti.

Quando il processo è completato, in base ai componenti che hai installato, potresti dover scegliere se vuoi avviare Stack Builder per scaricare e installare strumenti, driver e applicazioni aggiuntive.

Se decidi di avviare Stack Builder, vedrai una schermata dove puoi iniziare l'installazione. In alternativa, puoi cliccare su Annulla per farlo in un secondo momento.

Congratulazioni! Ora hai PostgreSQL installato sul tuo computer. Il processo dovrebbe essere simile per altri sistemi operativi. Dopo questo, puoi iniziare a usare il database.

Vai alle tue app e cerca pgAdmin, una piattaforma open source di amministrazione e sviluppo per PostgreSQL che dovrebbe essere stata installata se l'hai selezionata tra i componenti. Clicca su pgAdmin 4 per aprirla.

Vedrai una schermata di caricamento iniziale all'avvio dell'applicazione. Poi dovresti vedere una dashboard dove puoi avviare un nuovo server, controllare i tuoi database, configurare pgAdmin e altro. Troverai anche collegamenti utili alla documentazione.

Se clicchi su "Servers" in alto a sinistra, dovrai inserire la password che hai scelto durante l'installazione.

Dopo aver inserito la password, dovresti vedere una schermata dove avrai accesso ai tuoi database, incluse le loro tabelle.

Troverai anche un altro strumento molto utile, psql, un terminale interattivo per query SQL. Per aprirlo, cerca psql nel tuo sistema. Cliccaci sopra.

Si aprirà la SQL Shell, dove puoi iniziare a eseguire comandi.

Ecco come puoi installare PostgreSQL su Windows. Il processo è molto simile per macOS e Linux.

In alternativa, se lo stai installando su macOS, puoi usare Homebrew, un gestore di pacchetti. Una volta installato Homebrew, esegui questo comando per installare PostgreSQL:

```bash
brew install postgresql@<version>
```

Puoi specificare la versione che vuoi installare scrivendo il numero dopo il simbolo at (`@`).
