# --interactive--

У CSS властивість `list-style` використовується для керування зовнішнім виглядом списків на сторінці.

Чи працюєте ви з впорядкованими списками (`ol`) чи з невпорядкованими списками (`ul`), властивість `list-style` дозволяє налаштувати, як відображаються елементи списку.

Властивість `list-style` насправді є скороченою властивістю для трьох інших властивостей:

- `list-style-type`
- `list-style-position`
- `list-style-image`

Кожна з них виконує різну роль у визначенні зовнішнього вигляду списків.

Властивість `list-style-type` дозволяє визначити тип маркера або числа, що використовується у списку.

Для невпорядкованих списків ви можете вибрати серед кількох стилів маркерів, таких як диски, кола або квадрати.

Для впорядкованих списків можна використовувати різні системи нумерації, наприклад десяткову, римські цифри або навіть алфавітні символи.

Ось приклад використання `list-style-type`:

:::interactive_editor

```html
<ul style="list-style-type: square;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

У цьому прикладі маркери невпорядкованого списку змінені на квадрати.

Властивість `list-style-type` є найпоширенішою з трьох, оскільки вона безпосередньо впливає на зовнішній вигляд стилю маркера або нумерації у ваших списках.

Властивість `list-style-position` керує позицією маркера або числа відносно вмісту елемента списку. Існує два значення, які ви можете використовувати: `inside` та `outside`.

Коли ви використовуєте значення `outside`, маркер або число з’являється поза межами вмісту, що є поведінкою за замовчуванням.

А коли ви використовуєте значення `inside`, маркер або число з’являється всередині вмісту, що може спричинити перенесення тексту та вирівнювання з маркером або числом.

Ось приклад використання `list-style-position`:

:::interactive_editor

```html
<ul style="list-style-position: inside;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
<ul style="list-style-position: outside;">
  <li>Item 4</li>
  <li>Item 5</li>
  <li>Item 6</li>
</ul>
```

:::

У цьому прикладі наведені значення `inside` та `outside` для двох різних тегів невпорядкованого списку.

Властивість `list-style-position` може бути корисною, коли ви хочете контролювати вирівнювання вмісту списку, особливо у випадках, коли в одному елементі списку є кілька рядків тексту.

Властивість `list-style-image` дозволяє використовувати зображення як маркер для елементів списку. Це може бути корисно для додавання унікального візуального стилю вашим спискам.

Ось приклад використання `list-style-image`:

:::interactive_editor

```html
<head>
  <style>
    ul {
      list-style-image: url('https://cdn.freecodecamp.org/platform/universal/freecodecamp-org-gravatar.jpeg');
      list-style-position: inside;
    }
  </style>
</head>
<body>
  <ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
  </ul>
</body>
```

:::

У цьому прикладі маркери замінені на кастомний логотип freeCodeCamp, що додає персоналізований вигляд списку.

При використанні `list-style-image` переконайтеся, що вибране зображення є маленьким і підходить для дизайну вашої сторінки. Якщо зображення занадто велике або складне, це може ускладнити читання списку.

Ви можете поєднати три властивості – `list-style-type`, `list-style-position` та `list-style-image` – в одну скорочену властивість `list-style`.

Порядок значень у скороченій властивості не має значення, але всі три можна вказати разом.

Ось приклад використання скороченої властивості:

:::interactive_editor

```html
<ul style="list-style: square inside url('https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg');">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

У цьому прикладі елементи списку використовують квадратні маркери, розташовані всередині вмісту, з кастомним зображенням як маркером.

Однак, якщо зображення недоступне або не відтворюється, квадратні маркери будуть використані як запасний варіант.
