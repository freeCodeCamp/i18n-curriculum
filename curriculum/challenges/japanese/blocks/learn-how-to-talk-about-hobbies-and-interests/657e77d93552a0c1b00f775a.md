# --description--

`Would you like...?` は、活動を提案したり勧めたりするときによく使われます。例えば：

`Would you like to test the new app with me?` - これは一緒にテスト作業をする丁寧な招待です。

リンダが `Would you like to go cycling with me one of these days?` と言うときは、「ねえ、一緒にサイクリングに行こうよ」と言っているのと似ています。
