# --description--

Express.js，簡稱 Express，是一個建立在 Node.js **之上** 的極簡且彈性的網頁框架。它簡化了建置網頁伺服器和 API 的流程。

與其使用 Node 原生的 `http` 模組寫大量樣板程式碼，Express 讓你只需幾行程式碼就能創建強固的網頁應用程式。

你可以把它想成是讓 Node.js 更簡單且更強大的快速、無意見的層。

以下是開發者喜愛 Express 的三個主要原因：

1. **極簡且快速** — Express 只提供必要的功能，讓你完全掌控，同時可透過中介層和外掛程式按需添加功能。

2. **龐大生態系** — 它擁有大量用於記錄、驗證、安全、會話等的中介層和外掛程式。

3. **非常適合 API** — 它簡單的路由和請求處理模型，使 Express 特別適合建置 RESTful API。

使用 Express，你可以創建：

* 網頁應用程式

* RESTful API

* 單頁應用程式（SPA）後端

* 以中介層為主的服務

* 任何基於 HTTP 的伺服端應用


## 如何安裝 Express

首先，確定你已安裝 Node.js 和 npm。以下是檢查 node 和 npm 是否安裝的方法：

```bash
node -v
npm -v
```

如果你看到版本號，代表已安裝。若沒有，你需要先安裝。安裝說明可參考之前的課程或官方 Node.js 網站。

接著，在你的終端機輸入以下指令：

```bash
mkdir my-app
cd my-app
npm init -y
npm install express
```

你將在未來課程中學到更多關於中介層和外掛程式的內容，簡單來說，中介層是介於接收請求和送出回應之間執行的程式碼。它常用於錯誤處理、輸入驗證、修改請求或回應中的資料等。你可以串接多個中介層，建立靈活且可重複使用的邏輯來處理請求。

外掛程式通常是外部第三方的 Node.js 模組，用來擴充 Express 的功能。你可以用 npm 單獨安裝它們，大多數外掛程式是以中介層形式實作，只需簡單設定即可在你的應用中使用。你不必從頭寫常見功能，可以用外掛程式來處理記錄、驗證或安全最佳實踐等。

Express 是 Node.js 伺服端開發中非常受歡迎的框架。它提供建置伺服器、處理路由、管理中介層和建構 API 的基礎元件，且程式碼乾淨易讀。
