# --description--

開發者經常使用 `once` 來顯示工作流程中任務的時機。它的意思是 `as soon as` 或 `when`。

例如，`Once you receive the client's feedback, update the project documentation.` 這表示在收到回授後應立即開始更新文件，顯示專案生命週期中動作的順序。
