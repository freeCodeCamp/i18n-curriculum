# --interactive--

JavaScript — це потужна мова програмування, яка додає інтерактивність і динамічну поведінку вебсторінкам.

У той час як HTML і CSS — це мови розмітки, які використовують для структурування вмісту та стилізації елементів на сторінці, JavaScript виходить за межі цього, дозволяючи реалізовувати складніші функції, такі як обробка вхідних даних користувача, анімація елементів і навіть створення повноцінних вебзастосунків.

Наприклад, коли ви натискаєте на кнопку, відправляєте форму або наводите курсор на меню, JavaScript визначає, як поводитиметься сторінка.

Ось приклад того, як ці три працюють разом:

:::interactive_editor

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: green;
    }
  </style>
</head>
<body>
  <h1>Hello, World!</h1>
  <button onclick="alert('Button clicked!')">Click me</button>
</body>
</html>
```

:::

У цьому прикладі HTML використовується для визначення вмісту: заголовка (елемент `h1`) і кнопки (елемент `button`).

CSS застосовується для стилізації заголовка, наприклад, щоб зробити текст зеленим. JavaScript використовується для відображення повідомлення alert, коли натискають на кнопку.

Підсумовуючи, HTML забезпечує структуру, CSS додає стилі, а JavaScript відповідає за поведінку, створюючи повний і інтерактивний вебдосвід.
