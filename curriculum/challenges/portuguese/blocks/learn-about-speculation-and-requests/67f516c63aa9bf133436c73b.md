# --explanation--

`Might have to` significa que há uma possibilidade de que uma ação seja necessária.Por exemplo:

`We might have to update the software to fix the issue.` - A equipe pode precisar atualizar o software para resolver o problema.

`Escalate` significa reportar ou elevar um problema para um nível superior de responsabilidade.Por exemplo:

`I had to escalate the problem because it wasn't resolved at the lower level.` - Você precisou encaminhar o problema para alguém de nível superior porque não foi resolvido pela equipe.
