# --description--

Der Ausdruck `some of these` wird verwendet, um sich auf einen bestimmten Teil einer Gruppe oder eine Teilmenge von Elementen aus einer größeren Menge zu beziehen. Er wird oft benutzt, wenn Sie einige Dinge aus einer Gruppe erwähnen möchten, ohne jedes einzelne aufzulisten.

Zum Beispiel `Let's try some of these cookies.` Der Sprecher schlägt vor, ein paar Kekse aus einer größeren Charge zu probieren.

### --feedback--

Diese Präposition wird verwendet, um die allgemeine Menge mit den spezifischen besprochenen Elementen zu verbinden und so einen partitivischen Ausdruck zu bilden.

---

### --feedback--

Dieses Wort gibt an, welche Gruppe von Elementen gemeint ist, und verweist auf eine bestimmte, vorhandene Gruppe.
