### --feedback--

Esta palabra significa tener dificultad con algo. Recuerda eliminar la `e` y agregar `-ing` al final de la palabra.

# --explanation--

`To struggle` significa tener dificultad con algo. Por ejemplo:

- `The program is struggling to load.` - Esto significa que el programa actualmente tiene dificultades para cargar.

El tiempo `Present Continuous` se usa para describir acciones que **están ocurriendo ahora mismo**.

En el ejemplo anterior, nota que al formar el tiempo `Present Continuous` con el verbo `struggle`, se elimina la `e` final, resultando en `struggling`. Esta es una regla común para los verbos que terminan en `e`.

En la oración de Tom, él usa el tiempo `Present Continuous` para describir su dificultad continua con los IDEs (Entornos de Desarrollo Integrados).

Debes agregar `-ing` a `struggle` porque la acción está ocurriendo ahora mismo.
