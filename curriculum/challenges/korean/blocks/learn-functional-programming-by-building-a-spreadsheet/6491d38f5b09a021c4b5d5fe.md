# --description--

배열에는 `.every()` 메서드가 있습니다. `.some()` 메서드처럼 `.every()`는 배열의 요소를 인수로 받는 콜백 함수를 인수로 받습니다. `.every()` 메서드는 콜백 함수가 배열의 모든 요소에 대해 `true`를 반환하면 `true`를 반환합니다.

다음은 배열의 모든 요소가 대문자인지 확인하는 `.every()` 메서드 호출 예시입니다.

```js
const arr = ["A", "b", "C"];
arr.every(letter => letter === letter.toUpperCase());
```

`everyeven` 속성을 `spreadsheetFunctions`에 추가하세요 - `.every()` 메서드를 사용해 배열의 모든 요소가 짝수인지 확인하세요.

