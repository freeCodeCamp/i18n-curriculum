# --explanation--

Usas la estructura `Second Conditional` (en el discurso de Maria, `It'd be helpful if we had...`) para describir una situación hipotética. La estructura del segundo condicional es `It would` + verbo base ... `if` + sujeto + verbo en `Past Simple`. Se usa para describir situaciones irreales o hipotéticas en el presente o futuro. Por ejemplo:

`We would finish earlier if we had more team members.` - Esto significa que no tienen más miembros en el equipo, pero si los tuvieran, podrían terminar antes. Estamos hablando de una hipótesis.
