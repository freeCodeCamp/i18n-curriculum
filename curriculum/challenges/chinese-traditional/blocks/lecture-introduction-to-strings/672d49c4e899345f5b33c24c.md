# --interactive--

先前的課程已經為你介紹過 `console.log()`，但本課程將更深入探討它的用途和使用方式。

在 JavaScript 中，`console.log()` 是一個簡單但強大的工具，用於在瀏覽器的主控台顯示訊息或輸出資訊。它主要被開發者用來在撰寫程式時偵錯和檢查程式碼。

你可以使用 `console.log()` 將文字或變數記錄到主控台，並確保你的程式碼正在正確執行。

要使用 `console.log()`，你在小括弧內呼叫該方法並放入你想輸出的值或訊息。以下是一些範例：

:::interactive_editor

```js
console.log("Hello, world!");

let num = 5;
console.log(num); // 5
```

:::

第一個範例會在瀏覽器的主控台中列印 `Hello, world!`，而第二個範例會列印數值 `5`。

這是另一個使用 `console.log()` 的範例：

:::interactive_editor

```js
let name = "Alice";
console.log("Hello, " + name + "!"); // Hello, Alice!
```

:::

你也可以傳入多個以逗號分隔的值到 `console.log()`。例如：

:::interactive_editor

```js
let name = "Alice";
let age = 25;
console.log("Name:", name, "Age:", age); // Name: Alice Age: 25
```

:::

這有助於一次記錄多筆資訊。

`console.log()` 方法幫助你在程式執行時監控你的程式碼，使你更容易發現錯誤並了解你的程式行為。
