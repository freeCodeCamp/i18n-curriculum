# --description--

`Got it!` bedeutet, dass Sie etwas verstehen. Zum Beispiel, wenn jemand erklärt, wie man eine Aufgabe erledigt, und Sie es verstehen, können Sie sagen, `Got it!`. Das bedeutet, Sie wissen jetzt, was zu tun ist.

`To clear up` bedeutet, etwas leicht verständlich zu machen oder ein Missverständnis zu klären. Zum Beispiel, wenn es Verwirrung über die Uhrzeit eines Treffens gibt und Sie es klar erklären, können Sie fragen, ob Sie `cleared up` die Verwirrung. Das bedeutet, Sie machen alles für alle Beteiligten klar.
