# --description--

學習程式設計中的 `variables`、`data types` 和 `conditional statements`。

`variable` 就像一個可以儲存資訊的盒子。這些資訊可以是數字、文字，或其他型別的資料。你可以改變盒子裡的內容，這就是為什麼它被稱為 `variable`－因為它可以變化。例如，在 `x = 5` 中，`x` 臨時儲存數值 5。

`data type` 是一種分類，用來指定 `variable` 可以儲存的資料型別。常見的 `data types` 包括整數（整數）、浮點數（帶小數的數字）、字串（文字）和布林值（真或假）。例如，如果 `age = 21`，則 `data type` 是整數。

`conditional statement` 是一個指令，告訴電腦只有在特定條件為真時才執行某些操作。這就像在現實生活中根據某種情況做出決定。常見的 `conditional statements` 是 `if-else` 述語。<dfn>例如</dfn>，`if (temperature > 20) {console.log("It's warm outside!");}` 是一個 `conditional statement`，當溫度高於 20 度時會執行 `console.log("It's warm outside!")`。
