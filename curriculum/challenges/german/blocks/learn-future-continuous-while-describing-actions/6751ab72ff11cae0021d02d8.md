# --explanation--

`Server` bezeichnet einen leistungsstarken Computer oder ein System, das anderen Geräten oft über ein Netzwerk Dienste, Daten oder Ressourcen bereitstellt. Zum Beispiel:  

`The server hosts the company's website.` – Das bedeutet, der Server ist dafür verantwortlich, die Inhalte der Website zu speichern und an Benutzer zu liefern, die darauf zugreifen.

`Load` bezeichnet die Menge an Arbeit, Nachfrage oder Belastung, die an ein System oder eine Ressource gestellt wird. Zum Beispiel:  

`The system crashed because it couldn't handle the high load during peak hours.` – Das bedeutet, das System wurde aufgrund der großen Menge an Nachfrage überlastet.
