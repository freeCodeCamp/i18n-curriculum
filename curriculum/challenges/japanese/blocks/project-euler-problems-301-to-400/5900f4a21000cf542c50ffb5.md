# --description--

アリスとボブは Nim Square というゲームをします。

Nim Square は、普通の3つのヒープを使った通常の Nim と同じですが、プレイヤーはヒープから取り除ける石の数が平方数に限られています。

3つのヒープの石の数は順序付けられた三つ組 ($a$, $b$, $c$) で表されます。

もし $0 ≤ a ≤ b ≤ c ≤ 29$ なら、次のプレイヤーが負ける位置の数は1160です。

$0 ≤ a ≤ b ≤ c ≤ 100\,000$ の場合の、次のプレイヤーが負ける位置の数を求めてください。
