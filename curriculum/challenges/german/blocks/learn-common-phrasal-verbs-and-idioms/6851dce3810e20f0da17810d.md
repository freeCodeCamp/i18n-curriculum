# --explanation--

`On another note` wird verwendet, um in einem Gespräch ein neues Thema einzuführen. Zum Beispiel:

`On another note, have you seen the new project updates?` – Das bedeutet, dass der Sprecher das Thema wechselt.

`To touch base on something` bedeutet, Kontakt aufzunehmen oder eine kurze Diskussion über ein Thema zu führen. Zum Beispiel:

`We should touch base on the client's feedback before the meeting.` – Das bedeutet, ein kurzes Gespräch über das, was der Client gesagt hat, zu führen.
