# --explanation--

`Real-time` يعني حدوث الشيء فورًا، بدون أي تأخير. على سبيل المثال:

- `We are having a real-time video chat.` - المحادثة تحدث فورًا، بدون تأخير بين الردود.

`Collaboration` يعني العمل مع الآخرين لتحقيق هدف مشترك. على سبيل المثال:

- `The team collaborated to finish the project on time.` - الفريق عمل معًا لإكمال المشروع في الوقت المحدد.

`Real-time project collaboration` يعني العمل معًا على مشروع فورًا، حتى لو كان أعضاء الفريق في مواقع مختلفة. على سبيل المثال:

- `Our team uses a shared platform for real-time project collaboration.` - الفريق يعمل معًا على المشروع مع تحديثات واتصالات فورية.
