## --text--

如果用户提供字串 `"random"`，以下代码的结果将是什么？

```python
try:
    number = int(input("Enter a number: "))
    print(number)
except:
    print("Invalid Input")
```

## --answers--

不会有任何输出到控制台。

---

字串 `"random"` 将输出到控制台。

---

字串 `"Invalid Input"` 将输出到控制台。

---

程序会崩溃。
