# --description--

Wenn eine Schachtel einundzwanzig farbige Scheiben enthält, bestehend aus fünfzehn blauen Scheiben und sechs roten Scheiben, und zwei Scheiben zufällig entnommen werden, lässt sich die Wahrscheinlichkeit berechnen, zwei blaue Scheiben zu ziehen.

$${P(BB)} = \frac{15}{21}×\frac{14}{20} = \frac{1}{2}$$

Die nächste solche Anordnung, bei der die Wahrscheinlichkeit genau 50 % beträgt, zwei blaue Scheiben zufällig zu ziehen, ist eine Schachtel mit fünfundachtzig blauen Scheiben und fünfunddreißig roten Scheiben.

Indem Sie die erste Anordnung finden, die insgesamt mehr als `limit` Scheiben enthält, bestimmen Sie die Anzahl der blauen Scheiben, die die Schachtel enthalten würde.

```js
function arrangedProbability(limit) {
  // Based on https://www.mathblog.dk/project-euler-100-blue-discs-two-blue/
  let blue = 15;
  let discs = 21;

  while (discs < limit) {
    const nextBlue = 3 * blue + 2 * discs - 2;
    const nextDiscs = 4 * blue + 3 * discs - 3;

    blue = nextBlue;
    discs = nextDiscs;
  }
  return blue;
}
```
