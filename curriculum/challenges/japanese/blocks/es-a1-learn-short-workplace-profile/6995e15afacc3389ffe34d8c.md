# --explanation--

`¿Cuál es...?`は「～は何ですか？」という意味です。`a`の文字にアクセント記号を付けることを忘れないでください。

この文脈で`su`は「彼の」または「彼女の」という意味です。

`Correo electrónico`は「メール」を意味します。日常会話では`correo`と略すことが非常に一般的です。正式な場面では完全な形がよく使われます。

`correo`は「郵便」を意味し、`electrónico`は「電子的な」という意味です。`o`の最初の文字`electrónico`にアクセント記号を付けることを忘れないでください。
