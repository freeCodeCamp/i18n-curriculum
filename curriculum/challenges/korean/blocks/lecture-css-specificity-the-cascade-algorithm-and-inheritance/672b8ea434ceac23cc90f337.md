# --interactive--

ID 선택자는 CSS에서 가장 강력한 선택자 중 하나로, 개발자가 고유 식별자를 가진 특정 요소에 스타일을 적용할 수 있게 합니다.

이로 인해 고유한 스타일이 필요한 개별 요소를 타깃팅하는 데 매우 효과적입니다.

ID 선택자는 해시 기호(`#`) 다음에 ID 이름이 옵니다. HTML 문서 내에서 고유해야 하므로 두 요소가 같은 ID를 공유해서는 안 됩니다.

다음은 ID 선택자를 사용한 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="unique">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#unique {
  color: purple;
}
```

:::

이 예에서 ID가 `unique`인 요소는 텍스트 색상이 `purple`로 설정됩니다.

ID 선택자는 타입 선택자와 클래스 선택자보다 특이성이 매우 높지만 인라인 스타일보다는 낮습니다. ID 선택자의 특이성 값은 `(0, 1, 0, 0)`입니다.

즉, ID 선택자는 클래스 선택자와 타입 선택자를 재정의할 수 있지만 인라인 스타일에는 재정의됩니다.
