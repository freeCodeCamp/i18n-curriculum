# --explanation--

`Sure thing` — це неформальний і дружній спосіб сказати `Of course` або `I'll do it`. Він показує, що мовник погоджується з проханням без вагань. Наприклад:

- **Ashley:** `Can you send me the file this afternoon?` — ця людина просить файл.

- **Sam:** `Sure thing.` — це означає, що мовник погоджується і надішле файл, як було запрошено.
