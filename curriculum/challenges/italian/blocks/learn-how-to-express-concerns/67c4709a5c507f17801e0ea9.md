# --explanation--

Un `project timeline` è un programma che delinea le tappe fondamentali, le scadenze e le attività di un progetto. Aiuta i team a rimanere organizzati e a completare il lavoro in tempo. Per esempio:

`We need to adjust the project timeline to allow more time for testing.` - Questo significa che il programma necessita di modifiche per includere più tempo per i test.
