# --description--

このワークショップでは、10進数と2進数のコンバーターを作成し、両方の数値システムを練習します。変換には再帰も使います。

このプロジェクトのHTMLとCSSはすべて用意されています。

準備ができたら、`.getElementById()` メソッドを使って、idが `input` の `"number-input"` 要素を取得し、`numberInput` という変数にストアしてください。同じメソッドを使って、idが `button` の `"convert-btn"` 要素を取得し、`convertBtn` という変数にストアし、idが `output` の `"result"` 要素を取得して、`result` という変数にストアしてください。

**注意**：このプロジェクトは正の数のみを2進数に変換します。
