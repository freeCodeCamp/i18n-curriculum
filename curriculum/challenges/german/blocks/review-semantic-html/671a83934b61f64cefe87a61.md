# --interactive--

## Bedeutung von semantischem HTML

- **Strukturelle Hierarchie für Überschriftselemente**: Es ist wichtig, das korrekte Überschriftselement zu verwenden, um die strukturelle Hierarchie des Inhalts beizubehalten. Das `h1`-Element ist die höchste Ebene der Überschrift und das `h6`-Element ist die niedrigste Ebene der Überschrift.
- **Präsentative HTML-Elemente**: Elemente, die das Aussehen von Inhalten definieren. Zum Beispiel die veralteten `center`-, `big`- und `font`-Elemente.
- **Semantische HTML-Elemente**: Elemente, die Bedeutung und Struktur enthalten. Zum Beispiel `header`, `nav`, `figure`.

## Semantische HTML-Elemente

- **Header-Element**: Wird verwendet, um die Kopfzeile eines Dokuments oder Abschnitts zu definieren.

:::interactive_editor

  ```html
  <header>
    <h1>CatPhotoApp</h1>
    <p>Welcome to our cat gallery.</p>
  </header>
  ```

:::

- **Main-Element**: Wird verwendet, um den Hauptinhalt der Webseite zu enthalten.

:::interactive_editor

  ```html
  <main>
    <section>
      <h2>Cat Photos</h2>
      <p>Browse adorable cat pictures.</p>
    </section>
  </main>
  ```

:::

- **Section-Element**: Wird verwendet, um Inhalte in kleinere Abschnitte zu unterteilen.

:::interactive_editor

  ```html
  <section>
    <h2>About Me</h2>
    <p>Hi, I am Jane Doe and I am a web developer.</p>
  </section>
  ```

:::

- **Navigationsabschnitt (`nav`)-Element**: Stellt einen Abschnitt mit Navigationslinks dar.

:::interactive_editor

  ```html
  <nav>
    <ul>
      <li><a href="#photos">Photos</a></li>
      <li><a href="#videos">Videos</a></li>
    </ul>
  </nav>
  ```

:::

- **Figure-Element**: Wird verwendet, um Illustrationen und Diagramme zu enthalten.

:::interactive_editor

  ```html
  <figure>
    <img
      src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"
      alt="Two tabby kittens sleeping together on a couch."
    />
    <figcaption>Cats <strong>hate</strong> other cats.</figcaption>
  </figure>
  ```

:::

- **Emphasis (`em`)-Element**: Markiert Text, der betont wird.

:::interactive_editor

  ```html
  <p>
    Never give up on <em>your</em> dreams.
  </p>
  ```

:::

- **Idiomatic Text (`i`)-Element**: Wird verwendet, um alternative Stimme oder Stimmung, idiomatische Begriffe aus einer anderen Sprache, technische Begriffe und Gedanken hervorzuheben.

:::interactive_editor

  ```html
  <p>
    There is a certain <i lang="fr">je ne sais quoi</i> in the air.
  </p>
  ```

:::

Das `lang`-Attribut innerhalb des offenen `i`-Tags wird verwendet, um die Sprache des Inhalts anzugeben. In diesem Fall wäre die Sprache Französisch. Das `i`-Element zeigt nicht an, ob der Text wichtig ist oder nicht, sondern nur, dass er sich irgendwie vom umgebenden Text unterscheidet.

- **Strong Importance (`strong`)-Element**: Markiert Text mit starker Bedeutung.

:::interactive_editor

  ```html
  <p>
    <strong>Warning:</strong> This product may cause allergic reactions.
  </p>
  ```

:::

- **Bring Attention To (`b`)-Element**: Wird verwendet, um Aufmerksamkeit auf Text zu lenken, der für die Bedeutung des Inhalts nicht wichtig ist. Es wird häufig verwendet, um Schlüsselwörter in Zusammenfassungen oder Produktnamen in Bewertungen hervorzuheben.

:::interactive_editor

  ```html
  <p>
    We tested several products, including the <b>SuperSound 3000</b> for audio quality, the <b>QuickCharge Pro</b> for fast charging, and the <b>Ecoclean Vacuum</b> for cleaning. The first two performed well, but the <b>Ecoclean Vacuum</b> did not meet expectations.
  </p>
  ```

:::

- **Description List (`dl`)-Element**: Wird verwendet, um eine Liste von Begriff-Beschreibungs-Gruppierungen darzustellen.
- **Description Term (`dt`)-Element**: Wird verwendet, um den definierten Begriff darzustellen.
- **Description Details (`dd`)-Element**: Wird verwendet, um die Beschreibung des Begriffs darzustellen.

:::interactive_editor

  ```html
  <dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
  </dl>
  ```

:::

- **Block Quotation (`blockquote`)-Element**: Wird verwendet, um einen Abschnitt darzustellen, der aus einer anderen Quelle zitiert wird. Dieses Element hat ein `cite`-Attribut. Der Wert des `cite`-Attributs ist die URL der Quelle.

:::interactive_editor

  ```html
  <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
    "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
  </blockquote>
  ```

:::

- **Citation (`cite`)-Element**: Wird verwendet, um die Quelle des referenzierten Werks visuell zuzuordnen. Markiert den Titel der Referenz.

:::interactive_editor

  ```html
  <div>
    <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
      "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
    </blockquote>
    <p>
      -Quincy Larson, <cite>How to learn to Code and Get a Developer Job [Full Book].</cite>
    </p>
  </div>
  ```

:::

- **Inline Quotation (`q`)-Element**: Wird verwendet, um ein kurzes inline-Zitat darzustellen.

:::interactive_editor

  ```html
  <p>
    As Quincy Larson said,
    <q cite="https://www.freecodecamp.org/news/learn-to-code-book/">
      Momentum is everything.
    </q>
  </p>
  ```

:::

- **Abbreviation (`abbr`)-Element**: Wird verwendet, um eine Abkürzung oder ein Akronym darzustellen. Um Benutzern zu helfen, die Abkürzung oder das Akronym zu verstehen, können Sie mit dem `title`-Attribut die vollständige Form, eine menschenlesbare Beschreibung, anzeigen.

:::interactive_editor

  ```html
  <p>
    <abbr title="HyperText Markup Language">HTML</abbr> is the foundation of the web.
  </p>
  ```

:::

- **Contact Address (`address`)-Element**: Wird verwendet, um die Kontaktinformationen darzustellen.
- **(Date) Time (`time`)-Element**: Wird verwendet, um ein Datum und/oder eine Uhrzeit darzustellen. Das `datetime`-Attribut wird verwendet, um Daten und Zeiten in ein maschinenlesbares Format zu übersetzen.

:::interactive_editor

  ```html
  <p>
    The reservations are for the <time datetime="20:00">20:00 </time>
  </p>
  ```

:::

- **ISO 8601 Date (`datetime`)-Attribut**: Wird verwendet, um Daten und Zeiten in einem maschinenlesbaren Format darzustellen. Das Standardformat ist `YYYY-MM-DDThh:mm:ss`, wobei das große `T` als Trennzeichen zwischen Datum und Uhrzeit dient.
- **Superscript (`sup`)-Element**: Wird verwendet, um hochgestellten Text darzustellen. Häufige Anwendungsfälle für das `sup`-Element sind Exponenten, hochgestellte Buchstaben und Ordnungszahlen.

:::interactive_editor

  ```html
  <p>
    2<sup>2</sup> (2 squared) is 4.
  </p>
  ```

:::

- **Subscript (`sub`)-Element**: Wird verwendet, um tiefgestellten Text darzustellen. Häufige Anwendungsfälle für das Subscript-Element sind chemische Formeln, Fußnoten und Variablen-Indizes.

:::interactive_editor

  ```html
  <p>
    CO<sub>2</sub>
  </p>
  ```

:::

- **Inline Code (`code`)-Element**: Wird verwendet, um einen Ausschnitt von Computercode darzustellen.
- **Preformatted Text (`pre`)-Element**: Stellt vorformatierten Text dar.

:::interactive_editor

  ```html
  <pre>
    <code>
      body {
        color: red;
      }
    </code>
  </pre>
  ```

:::

- **Unarticulated Annotation (`u`)-Element**: Wird verwendet, um einen Bereich von inline-Text darzustellen, der so gerendert werden soll, dass er eine nicht-textuelle Anmerkung anzeigt.

:::interactive_editor

  ```html
  <p>
    You can use the unarticulated annotation element to highlight
    <u>inccccort</u> <u>spling</u> <u>issses</u>.
  </p>
  ```

:::

- **Ruby Annotation (`ruby`)-Element**: Wird verwendet, um Text mit Aussprache- oder Bedeutungs-Erklärungen zu annotieren. Ein Beispiel ist die ostasiatische Typografie.
- **Ruby fallback parenthesis (`rp`)-Element**: Wird als Fallback für Browser verwendet, die keine Unterstützung für die Anzeige von Ruby-Anmerkungen bieten.
- **Ruby text (`rt`)-Element**: Wird verwendet, um Text für die Ruby-Anmerkung anzugeben. Üblicherweise für Aussprache oder Übersetzungsdetails in der ostasiatischen Typografie.

:::interactive_editor

  ```html
  <ruby>
    明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp>
  </ruby>
  ```

:::

- **Strikethrough (`s`)-Element**: Wird verwendet, um Inhalte darzustellen, die nicht mehr korrekt oder relevant sind.

:::interactive_editor

  ```html
  <p>
    <s>Tomorrow's hike will be meeting at noon.</s>
  </p>
  <p>
    Due to unforeseen weather conditions, the hike has been canceled.
  </p>
  ```

:::

- **Interne Links**: Werden verwendet, um auf einen anderen Abschnitt der Seite zu verlinken, indem `href="#id"` auf einem `a`-Element verwendet wird und dem Ziel-Element dieselbe `id` zugewiesen wird. Dies wird häufig für Skip-Links, Inhaltsverzeichnisse oder lange Seiten mit mehreren Abschnitten verwendet.

:::interactive_editor

```html
<a href="#about-section">Go to About Section</a>

<section id="about-section">
  <h2>About</h2>
  <p>This is the about section of the page.</p>
</section>
```

:::
