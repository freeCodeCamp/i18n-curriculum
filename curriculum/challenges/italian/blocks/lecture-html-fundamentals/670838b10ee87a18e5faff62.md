# --interactive--

L'elemento `script` viene usato per incorporare codice eseguibile. La maggior parte degli sviluppatori lo usa per eseguire codice JavaScript. JavaScript serve per aggiungere interattività alle tue pagine web. Esempi comuni di utilizzo di JavaScript includono giochi interattivi, slider di immagini e moduli dinamici che convalidano l'input dell'utente in tempo reale.

Ecco un esempio di utilizzo dell'elemento `script` in un documento HTML. Rimuovi `//` davanti a `alert("Welcome to freeCodeCamp");` e vedrai apparire un alert. Per vedere le anteprime, devi abilitare l'editor interattivo.

:::interactive_editor

```html
<body>
  <script>
    // alert("Welcome to freeCodeCamp");
  </script>
</body>
```

:::

Anche se tecnicamente puoi scrivere tutto il tuo codice JavaScript all'interno dei tag `script`, è considerata una buona pratica collegare invece un file JavaScript esterno. Ecco un esempio di utilizzo dell'elemento `script` per collegare un file JavaScript esterno:

```html
<script src="path-to-javascript-file.js"></script>
```

L'attributo `src` viene usato qui per specificare la posizione di quel file JavaScript esterno. `src` sta per "source". Il motivo per cui non è consigliato inserire tutto il JavaScript dentro il documento HTML è la separazione delle responsabilità. La separazione delle responsabilità è un principio di progettazione in cui si dividono i programmi in sezioni distinte e ogni sezione si occupa di una responsabilità separata. In questo caso, vogliamo separare il codice JavaScript dal codice HTML.
