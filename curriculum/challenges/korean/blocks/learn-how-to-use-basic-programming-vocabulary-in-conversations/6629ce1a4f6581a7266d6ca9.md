# --description--

**현재완료진행형 시제**는 과거에 시작되어 지금도 계속되고 있는 행동이나 상황을 말할 때 사용합니다. 활동의 지속 시간이나 진행 중인 과정을 강조할 때 매우 유용합니다.

**현재완료진행형**을 만드는 방법은 다음과 같습니다: 주어 + `has/have been` + `-ing` 형태의 동사. 이 시제는 `for`와 `since` 같은 시간 표현과 자주 함께 쓰입니다. `For`는 2년, 6개월 같은 기간과 함께 쓰이고, `since`는 2018년, 오늘 아침 같은 특정 시점과 함께 쓰입니다.

예시:

`I have been studying English for two years.` 이것은 내가 2년 전에 영어 공부를 시작했고 지금도 계속 공부하고 있다는 뜻입니다.

`She has been working here since 2018.` 이것은 그녀가 2018년에 여기서 일하기 시작했고 지금도 계속 일하고 있다는 뜻입니다.

Brian의 말을 듣고 빈칸을 채우세요.
