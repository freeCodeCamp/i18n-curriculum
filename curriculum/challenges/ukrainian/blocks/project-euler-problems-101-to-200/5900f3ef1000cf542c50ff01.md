# --description--

Число, що складається повністю з одиниць, називається репьюнітом. Позначимо $R(k)$ як репьюніт довжини $k$; наприклад, $R(6) = 111111$.

Якщо $n$ — додатне ціле число і $GCD(n, 10) = 1$, можна довести, що завжди існує таке значення $k$, для якого $R(k)$ ділиться на $n$, і позначимо $A(n)$ найменше таке значення $k$; наприклад, $A(7) = 6$ і $A(41) = 5$.

Найменше значення $n$, для якого $A(n)$ вперше перевищує десять, дорівнює 17.

Знайдіть найменше значення $n$, для якого $A(n)$ вперше перевищує мільйон.
