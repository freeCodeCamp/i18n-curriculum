# --description--

為了在本課程中練習使用聯結和影像，你需要一個 HTML 專案來操作。

- 創建一個名為 `odin-links-and-images` 的新目錄。

- 在該目錄中，創建一個名為 `index.html` 的新檔案。

- 填寫常見的 HTML 標準範本。

- 最後，將以下 `h1` 添加到 `body`：`<h1>Homepage</h1>`

## 錨點元素
要在 HTML 中創建聯結，你使用錨點元素。錨點元素是透過用 `<a>` 標籤包裹你想要成為聯結的文字或其他 HTML 元素來定義的。

將以下內容添加到你創建的 `index.html` 頁面的 `body` 中，並在瀏覽器中打開：

```html
<a>click me</a>
```

你可能已經注意到點擊這個連結沒有任何反應。這是因為單獨的 anchor 標記不會知道你想要連結到哪裡。你必須告訴它一個目的地。你可以透過使用 HTML 屬性來做到這點。

HTML 屬性為 HTML 元素提供額外的資訊，且總是放在元素的起始標記內。屬性通常由兩部分組成：名稱和數值；不過並非所有屬性都需要數值。就你的情況而言，你需要在起始的 anchor 標記中添加一個 `href`（超連結參考）屬性。`href` 屬性的數值就是你希望連結導向的目的地。

將以下的 `href` 屬性添加到你之前建立的 anchor 元素，然後再試著點擊它，別忘了重新整理瀏覽器以套用新的變更。

```html
<a href="https://www.theodinproject.com/about">click me</a>
```

預設情況下，任何用沒有 `href` 屬性的錨點標記包裹的文字看起來都像純文字。如果存在 `href` 屬性，瀏覽器會將文字顯示為藍色並加底線，以表示它是聯結。

值得注意的是，你可以使用 anchor 標記連結到網際網路上的任何資源，而不只是其他 HTML 文件。你可以連結到影片、pdf 檔案、圖片等等，但大多數情況下，你會連結到其他 HTML 文件。
