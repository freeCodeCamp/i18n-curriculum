# --description--

`Compile` はプログラミングにおいて、人が書いたコードをコンピューターが理解し使える形に変換することを意味します。

これは、ある言語で書かれたレシピを別の言語に翻訳して、別の人が料理できるようにするのと似ています。

例えば、`When we compile Java code, it changes into a form that the computer can run.`
