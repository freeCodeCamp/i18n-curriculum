# --explanation--

`Server` означає потужний комп’ютер або систему, яка надає послуги, дані або ресурси іншим пристроям, часто через мережу. Наприклад:  

`The server hosts the company's website.` — це означає, що сервер відповідає за зберігання та доставку вмісту вебсайту користувачам, які отримують до нього доступ.

`Load` означає кількість роботи, навантаження або стресу, що покладається на систему чи ресурс. Наприклад:  

`The system crashed because it couldn't handle the high load during peak hours.` — це означає, що система була перевантажена через велику кількість навантаження, яке на неї поклали.
