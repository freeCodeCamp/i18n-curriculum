# --interactive--

CSS 的 `filter` 屬性是一個強大的工具，允許你對網頁上的元素套用圖形效果。它特別適用於調整影像、背景，甚至文字的視覺呈現，而不會修改原始資產。

`filter` 屬性可用於創建各種效果，例如模糊、色彩偏移和對比度調整。讓我們討論 `filter` 屬性如何運作並探索一些常見範例。`filter` 屬性的基本語法很簡單：

```css
selector {
  filter: function(amount);
}
```

在這裡，`function` 表述你想要套用的特定濾鏡效果，而 `amount` 通常是決定效果強度的數值。現在讓我們來看看一些常見的濾鏡函式（程式）及其用途。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: blur(2px);
}
```

:::

`blur` 函式會對元素套用高斯模糊，模糊的程度以像素為單位，並表述模糊的半徑。這條 CSS 規則會對頁面上的所有影像套用 2 像素的模糊。`blur` 效果可用於為你的設計創造深度或遮蔽影像的零件。 

`brightness` 函式（程式）調整元素的亮度。值為 `0%` 時，元素將完全變黑，而超過 `100%` 的值會增加亮度。 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: brightness(150%);
}
```

:::

這條 CSS 規則將影像元素的亮度提高 `50%`。亮度調整可用於讓影像更突出或產生洗淡效果。

`grayscale` 函式將元素轉換為灰階。數值以百分比定義，其中 `100%` 表示完全灰階，`0%` 則保持影像不變。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: grayscale(100%);
}
```

:::

此規則將把影像元素轉換為全灰階。`grayscale` 可用於創建復古風格或弱化頁面上的某些元素。

`sepia` 函式（程式）會將元素套用棕褐色調。像灰階一樣，它使用百分比數值： 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: sepia(80%);
}
```

:::

此規則對影像元素套用 `80%` 的褐色調效果。褐色調效果非常適合營造復古或懷舊風格。

`hue-rotate` 函式（程式）對元素套用色相旋轉。該數值以度數定義，並表述繞色彩圓的旋轉。 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: hue-rotate(90deg);
}
```

:::

此規則將影像元素的色相旋轉 `90` 度。色相旋轉可用於創建迷幻效果或調整影像的整體色彩結構綱目。

`filter` 屬性最強大的功能之一是能夠結合多個效果。你可以透過空格分隔，將多個濾鏡套用到同一個元素： 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {  
  filter: contrast(150%) brightness(110%) sepia(30%);  
}
```

:::

此規則對 `image` 元素套用增加的對比度、略微增加的亮度以及細微的棕褐色效果。 

透過組合濾鏡，你可以創建符合你設計需求的複雜且獨特的視覺效果。CSS filter 屬性是一個多功能工具，允許對網頁元素進行創意的視覺操作。 

雖然我們已涵蓋一些最常見的濾鏡函式，但還有其他可用的，例如 `contrast`、`invert` 和 `saturate`。如同任何強大的功能一樣，謹慎使用濾鏡以增強你的設計，而不會讓你的使用者感到負擔或影響存取性，是非常重要的。
