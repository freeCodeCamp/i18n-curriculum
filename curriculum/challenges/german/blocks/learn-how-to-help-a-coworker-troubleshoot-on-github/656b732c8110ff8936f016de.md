# --description--

In der Softwareentwicklung ist ein `branch` wie eine separate Entwicklungslinie. Man kann es sich als eine Möglichkeit vorstellen, gleichzeitig an verschiedenen Versionen eines Projekts zu arbeiten.

Zum Beispiel kann eine `branch` für neue Funktionen sein, während eine andere für das Beheben von Bugs zuständig ist.
