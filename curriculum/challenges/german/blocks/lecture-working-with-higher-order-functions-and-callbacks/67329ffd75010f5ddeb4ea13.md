# --interactive--

In JavaScript ist eine Callback-Funktion eine Funktion, die als Argument an eine andere Funktion übergeben wird, damit die äußere Funktion sie an einem bestimmten Punkt aufrufen kann.

Dieses Konzept ist grundlegend, um viele Aspekte von JavaScript zu verstehen, einschließlich wie die `forEach`-Methode funktioniert.

Beginnen wir damit, zu verstehen, was eine Callback-Funktion in einem einfachen Kontext ist.

Stellen Sie sich vor, Sie haben eine Funktion, und innerhalb dieser Funktion möchten Sie etwas mehrfach ausführen. Anstatt den gesamten Code in einer großen Funktion zu schreiben, können Sie eine zweite Funktion (die Callback-Funktion) übergeben, die immer dann ausgeführt wird, wenn Sie diese bestimmte Aufgabe erledigen möchten. Das ermöglicht flexibleren und modulareren Code.

Nun führen wir die `forEach`-Methode ein. `forEach` ist eine eingebaute Methode für Arrays in JavaScript. Sie erlaubt es Ihnen, jedes Element in einem Array zu durchlaufen und eine Operation auf jedem Element auszuführen. Die Operation, die Sie ausführen möchten, wird in einer Callback-Funktion definiert, die Sie an `forEach` übergeben.

Hier ist ein Beispiel, wie `forEach` mit einer Callback-Funktion funktioniert:

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];

numbers.forEach(function(number) {
  console.log(number * 2);
});
```

:::

In diesem Beispiel haben wir ein Array von Zahlen. Wir verwenden die `forEach`-Methode auf diesem Array und übergeben eine Callback-Funktion als Argument an `forEach`.

Diese Callback-Funktion nimmt einen Parameter, der jedes Element im Array darstellt. Die `forEach`-Methode ruft diese Callback-Funktion einmal für jedes Element im Array auf.

Die Callback-Funktion multipliziert in diesem Fall jede Zahl mit `2` und gibt das Ergebnis aus. Wenn dieser Code ausgeführt wird, wird Folgendes ausgegeben:

```md
2
4
6
8
10
```

Es ist wichtig zu verstehen, dass die Callback-Funktion für jedes Element im Array in der Reihenfolge aufgerufen wird. `forEach` übernimmt die Schleife für Sie, sodass Sie keine `for`-Schleife selbst schreiben müssen.

Sie können auch eine Pfeilfunktion als Callback verwenden, was Ihren Code noch kompakter macht:

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];
numbers.forEach(number => console.log(number * 2));
```

:::

Dies macht genau dasselbe wie das vorherige Beispiel, aber mit weniger Code.

Die Callback-Funktion in `forEach` kann tatsächlich bis zu drei Argumente annehmen: das aktuelle Element, den Index des aktuellen Elements und das Array, auf dem `forEach` aufgerufen wurde.

Hier ist ein Beispiel, das alle drei verwendet:

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];
numbers.forEach((number, index, array) => {
  console.log(`Element ${number} is at index ${index} in array ${array}`);
});
```

:::

Dies würde Informationen über jedes Element, seinen Index und das ursprüngliche Array ausgeben.

Das Verständnis von Callback-Funktionen und Methoden wie `forEach` ist wichtig, wenn Sie in JavaScript weiterkommen. Sie bilden die Grundlage für viele fortgeschrittene Konzepte in der Sprache, insbesondere im asynchronen Programmieren, das Sie in zukünftigen Lektionen lernen werden.
