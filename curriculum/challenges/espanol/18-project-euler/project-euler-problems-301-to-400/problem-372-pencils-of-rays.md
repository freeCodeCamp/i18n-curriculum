# --description--

Sea $R(M, N)$ el número de puntos de red ($x$, $y$) que satisfacen $M \lt x \le N$, $M \lt y \le N$ y $\left\lfloor\frac{y^2}{x^2}\right\rfloor$ es impar.

Podemos verificar que $R(0, 100) = 3\\,019$ y $R(100, 10\\,000) = 29\\,750\\,422$.

Calcula $R(2 \times {10}^6, {10}^9)$.

**Note:** $\lfloor x\rfloor$ representa la función de piso.
