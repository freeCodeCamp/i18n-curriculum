# --description--

Uma expressão regular, ou regex, é um padrão usado para corresponder a uma combinação específica de caracteres dentro de uma string.É uma alternativa válida às declarações condicionais `if`/`else` quando você precisa corresponder ou encontrar padrões dentro de uma string para fins de validação, substituição de caracteres e outros.

A função `compile()` do módulo `re` compila a string passada como argumento em um objeto de expressão regular que pode ser usado por outros métodos `re`.

Declare uma nova variável `pattern` e atribua o valor de `re.compile('i')` a essa variável.
