# --description--

У розмові, коли ви погоджуєтеся з кимось, але хочете додати більше інформації, ви можете використати `Yes, but...` для продовження своєї думки. Наприклад:

- **Керівник команди:** `Will you join the team meeting?`

- **Стажер:** `Yes, but only for the first 30 minutes.` — ця людина погоджується, але не може залишатися весь час.
