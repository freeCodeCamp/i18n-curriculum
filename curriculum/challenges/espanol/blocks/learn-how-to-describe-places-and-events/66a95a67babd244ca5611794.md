# --explanation--

`Does` se usa para formar una pregunta en el tiempo `Present Simple` para sujetos en tercera persona del singular.

`Yet` se usa a menudo en preguntas para preguntar si algo que se espera ha ocurrido. Implica que la acción o el evento se anticipa pero no ha sucedido hasta este momento. Por ejemplo:

- ¿Ya terminaste tu tarea? (Esperas que la tarea esté terminada pronto, si no es que ya lo está).
- `Has the movie started yet?` (Esperas que la película comience pronto, si es que no ha empezado ya).
