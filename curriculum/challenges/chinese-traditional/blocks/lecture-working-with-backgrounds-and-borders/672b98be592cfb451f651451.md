# --interactive--

CSS 中的背景漸層是在元素背景上應用的兩種或多種顏色之間的平滑過渡。漸層讓你能夠創建視覺上吸引人的背景，而不需要使用影像。

CSS 中有兩種主要的漸層型別：線性漸層和放射狀漸層。

線性漸層沿著一條直線過渡顏色。你可以定義方向和涉及的顏色。

這是基本語法：

```css
background: linear-gradient(direction, color-stop1, color-stop2, ...);
```

在此範例中，我們使用 `background` CSS 屬性，其值為 `linear gradient`。

方向指定漸層的方向。它可以是角度（例如 `45deg`）、關鍵字（例如 `to right`、`to bottom`）或邊/角落。

`color-stop` 指定漸層過渡發生的顏色和位置。

為了更好地理解線性漸層的運作方式，讓我們來看看以下範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient{
  background: linear-gradient(to right, red, yellow);
  height: 40vh;
}
```

:::

這個 CSS 產生一個線性漸層，從 `left` 的 `red` 過渡到 `right` 的 `yellow`。漸層套用於高度為視窗高度 `40%` 的元素。你將在未來的課程中學習更多關於 `vh` 單位的內容。

`to right` 方向表示漸層從左到右水平執行。

另一種漸層型別是 `radial` 漸層。

放射狀漸層會從一個起點（通常是中心）向外以圓形或橢圓形的形狀輻射顏色過渡。

這是基本語法：

```css
background: radial-gradient(shape size at position, color-stop1, color-stop2, ...)
```

在語法中，`shape` 指定漸層的形狀，可以是 `circle` 或 `ellipse`。

`size` 決定漸層結束形狀的大小，該形狀可以是 `closest-side`、`closest-corner`、`farthest-side` 或 `farthest-corner`。

`position` 決定漸層中心的位置，可以使用關鍵字（例如 `center`、`top left`、`bottom right`）或精確數值（例如 `50% 50%`、`10px 20px`）來指定。

最後，顏色停止點是一個顏色的列表，漸層會通過這些顏色。每個顏色停止點可以選填一個位置值（百分比或長度），用來指示顏色應該放置的位置。

一個範例會是：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient{
  background: radial-gradient(circle closest-side at center, red, yellow 50%, green);
  height: 60vh;
}
```

:::

這個 CSS 在元素中間創建一個圓形放射狀漸層。它從中心的 `red` 開始，過渡到半徑的 `50%` 處的 `yellow`，最後以 `green` 結束。

`closest-side` 關鍵字使漸層的結束形狀符合元素的最近邊。漸層套用於高度為視窗高度 `60%` 的元素。

了解如何使用 CSS 漸層可以大幅提升你的設計，因為它能提供視覺上吸引人的背景，而無需使用影像。

有像線性漸層用於平滑過渡和放射狀漸層用於圓形效果這樣的選項，可為網頁設計提供彈性和創造力。
