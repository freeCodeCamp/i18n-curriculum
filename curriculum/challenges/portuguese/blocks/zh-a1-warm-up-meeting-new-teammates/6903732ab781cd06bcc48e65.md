# --explanation--

`UI` significa "user interface", que se refere aos elementos visuais de um produto com os quais os usuários interagem. `设计师 (shè jì shī)` significa "designer". Juntos, `UI 设计师 (UI shè jì shī)` significam uma profissão que foca no design de interfaces de usuário. Por exemplo:

`我是 UI 设计师。(wǒ shì UI shè jì shī.)` – Eu sou um designer de UI.
