# --description--

Ahora que ha añadido la sección de "about", pruebe haciendo clic en el enlace `About` para ver cómo la página salta hacia esa sección.

La siguiente sección en la página del blog será una lista de publicaciones que hablan sobre el Sr. Whiskers.

Agregue otro elemento `section` con un `id` configurado como `"posts"`.

Dentro del elemento `section`, añada un elemento `h2` con el texto de `Posts`.
