# --interactive--

CSS Flexbox ist ein eindimensionales Layoutmodell, mit dem Sie Elemente innerhalb eines Containers in Reihen und Spalten anordnen können. Sie können auch deren Reihenfolge und Ausrichtung steuern. Webentwickler verwenden es, um responsive Websites und Webanwendungen zu erstellen, die sich an verschiedene Bildschirmgrößen und Ausrichtungen anpassen. Wir sprechen von einem eindimensionalen Layoutmodell, weil es sich darauf konzentriert, Elemente jeweils entlang einer einzigen Achse anzuordnen. Die Achse kann entweder horizontal oder vertikal sein.

Es gibt zwei wichtige Konzepte, die Sie kennen sollten, bevor Sie mit Flexbox arbeiten: Flex-Container und Flex-Element.

Ein Flex-Container ist ein HTML-Element mit einem Flex-Layout. Sie können Elemente innerhalb eines Flex-Containers auf verschiedene Arten anordnen und ausrichten. Um ein HTML-Element zu einem Flex-Container zu machen, müssen Sie `display: flex` zu seinen CSS-Stilen hinzufügen.

Flex-Elemente sind die direkten Kinder eines Flex-Containers. Diese Elemente können basierend auf den Eigenschaften des Containers innerhalb des Flex-Containers angeordnet und ausgerichtet werden. Sie können sich auch verkleinern oder vergrößern, um den verfügbaren Platz auszufüllen.

Dies ist ein Beispiel mit einem `main`-Container und drei untergeordneten `div`-Elementen:

```html
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

Wenn Sie nur die `width`, `height` und `background-color` dieser `div`-Elemente im CSS-Stylesheet festlegen, wird jedes Kindelement in einer eigenen Zeile platziert, da der Container standardmäßig kein Flex-Container ist.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Wenn Sie jedoch `display: flex` zum `main`-Container hinzufügen, werden die `div`-Elemente so umgeordnet, dass sie in dieselbe Zeile passen, und sie verkleinern sich bei Bedarf (um die Vorschauen zu sehen, aktivieren Sie den interaktiven Editor):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Standardmäßig ist ein Flex-Container ein Block-Element, sodass der Container selbst relativ zu anderen Elementen und Containern in einer eigenen Zeile steht.

Jetzt, wo Sie mehr über Flex-Container und Flex-Elemente wissen, sollten Sie auch die Flex-Eigenschaften kennen. Diese Eigenschaften bestimmen, wie Flex-Elemente innerhalb des Flex-Containers angeordnet, skaliert und verteilt werden. Einige der am häufigsten verwendeten sind `flex-direction`, `justify-content`, `align-items` und `flex-wrap`.

Prima. Sprechen wir nun ein wenig über das Flex-Modell. Dieses Modell definiert, wie Flex-Elemente innerhalb eines Flex-Containers angeordnet werden. Jeder Flex-Container hat zwei Achsen:

- Die Hauptachse.
- Die Querachse.

Die Ausrichtung dieser Achsen bestimmt, wie verschiedene Eigenschaften das Layout und die Verteilung der Flex-Elemente beeinflussen. Standardmäßig ist die Hauptachse eines Flex-Containers horizontal und die Querachse vertikal. Flex-Elemente werden in Richtung der Hauptachse angeordnet. Die Querachse steht senkrecht zur Hauptachse.

Die `flex-direction`-Eigenschaft legt die Richtung der Hauptachse fest. Der Standardwert von `flex-direction` ist `row`, wodurch alle Flex-Elemente in derselben Zeile in der Richtung der Standardsprache Ihres Browsers (von links nach rechts oder von rechts nach links) angeordnet werden:

```css
flex-direction: row; /* Default */
```

Um die Reihenfolge der Elemente in der Zeile umzukehren, können Sie `flex-direction: row-reverse` verwenden:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: row-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Dadurch wird die Reihenfolge der Flex-Elemente umgekehrt. Wenn Sie die Flex-Elemente stattdessen vertikal ausrichten möchten, müssen Sie nur `flex-direction` auf `column` im Flex-Container setzen. Dadurch ändert sich die Richtung der Hauptachse:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Jetzt werden die `div`-Elemente vertikal ausgerichtet, weil die Hauptachse vertikal und die Querachse horizontal ist.

Sie können die Reihenfolge der Flex-Elemente auch vertikal mit `flex-direction: column-reverse` umkehren:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

CSS Flexbox ist ein leistungsstarkes Layoutmodell, das eine flexible und effiziente Möglichkeit bietet, Elemente innerhalb eines Containers anzuordnen. Wenn Sie die Schlüsselkonzepte von Flex-Containern, Flex-Elementen und den verschiedenen Flex-Eigenschaften verstehen, können Sie dynamische und responsive Websites erstellen, die sich an unterschiedliche Bildschirmgrößen und Ausrichtungen anpassen.
