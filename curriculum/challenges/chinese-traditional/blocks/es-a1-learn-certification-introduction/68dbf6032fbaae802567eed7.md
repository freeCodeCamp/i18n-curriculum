# --description--

如果你要使用電腦來進行西班牙語課程，你需要確保它已準備好輸入重音符號和特殊字元，例如 `ñ`。 

設定這個很快，無論你使用 Windows 還是 Mac。只需幾分鐘。

## 在 Windows 11 上設定

1. 從開始選單開啟**設定**。

2. 前往 **時間與語言**，然後點擊 **語言與地區**。

3. 前往 **Preferred languages** 並選擇 **Add a language**。

4. 搜尋 **Spanish** 並選擇像是 `Español (Latin America)` 的變體。

5. 點擊 **Next**，然後依照步驟操作。

語言安裝完成後：  

- 在你的**偏好語言**列表中找到它。

- 點擊省略符號以顯示其他選項，可選方案，並選擇 **Language options**。

- 前往 **Keyboards**，點擊 **Add a keyboard**，然後選擇你偏好的西班牙語配置。

要在鍵盤之間切換：  

- 按下 **Windows key + Spacebar**。

- 或者點擊工作列上的語言圖示（通常標示為 **ENG**），然後選擇西班牙語。

## 在 Mac 上設定

1. 打開 **Apple 選單** 並選擇 **系統設定**。

2. 前往 **Keyboard**。

3. 前往 **文字輸入** > **輸入來源**。

4. 點擊 **Edit...**

5. 點擊底部的 **plus (+)** 按鈕，並從列表中選擇 **Spanish**。

6. 點擊 **Add**。

現在你已經準備好開始學習西班牙語課程並完成任務，包括那些需要重音符號和特殊字元的任務。
