# --explanation--

`Really` 和 `too` 都是 **強調詞**，但它們有不同的意思。`Really` 強化它所修飾的形容詞而不顯示過度。另一方面，`too` 顯示過度，意指超過可接受或可處理的範圍。 例如：

- `The project is really important.`－這表示該專案非常重要，但不一定過於重要。
 
- `This task is too complicated.`－這表示它比應該的更複雜。
