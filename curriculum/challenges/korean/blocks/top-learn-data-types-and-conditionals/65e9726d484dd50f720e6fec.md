# --description--

JavaScript는 엄격한 동등 연산자 `===`를 사용해 타입과 값을 비교할 수도 있습니다. 이 연산자는 두 값이 같고 같은 타입인지 확인합니다. 예를 들어:

```javascript
let x = 5;
let y = "5";

let result = x === y;

console.log(result); // false
```

이 예제에서 `result` 변수는 `false`를 담게 되는데, `x`은 숫자이고 `y`는 문자열이기 때문입니다. 하지만 `==` 연산자를 사용하면 JavaScript가 문자열을 숫자로 변환해 값을 비교하므로 결과는 `true`가 됩니다.

이 연산자를 엄격한 동등 연산자라고 하는 이유는 값과 타입 모두 같음을 검사하기 때문입니다. 값 비교 시 예상치 못한 결과를 피하려면 `===` 연산자를 사용하는 것이 권장됩니다.

또한 두 값이 같지 않고 같은 타입인지 확인하는 엄격한 부등 연산자 `!==`도 있습니다.

엄격한 동등 연산자를 사용하는 방법은 다양하며, 어떻게 작동하는지 이해하는 것이 중요합니다.
