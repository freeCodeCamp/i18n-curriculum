# --explanation--

`Forward`는 앞으로의 행동이나 지금부터의 시간을 말할 때 사용합니다. 예를 들어:

`Going forward, we will hold weekly meetings.` – 이는 지금부터 주간 회의를 진행한다는 뜻입니다.

`To happen`는 이벤트나 상황이 발생하거나 일어나는 것을 의미합니다. 예를 들어:

`If problems happen, please report them immediately.` – 이는 문제가 발생하면 누군가에게 알려야 한다는 뜻입니다.
