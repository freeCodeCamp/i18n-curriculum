# --interactive--

前のレッスンでは、`iframe` 要素を初めて紹介しました。このレッスンでは、`iframe` 要素の使い方についてさらに学びます。この要素はインラインフレームを表します。HTMLページ内に他のHTMLコンテンツを直接埋め込むためのインライン要素です。そのHTMLコンテンツは動画、地図、別のHTML要素、あるいは他のウェブページである場合があります。

ここでは、YouTubeから人気のfreeCodeCampコースを埋め込む例を示します。別の動画を表示したい場合は、インタラクティブエディターを有効にして、`src` の値をお好きな動画に変更してください。

:::interactive_editor

```html
<iframe
  width="400"
  height="400"
  src="https://www.youtube.com/embed/PkZNo7MFNFg?si=-UBVIUNM3csdeiWF"
  title="Learn JavaScript - Full Course for Beginners (YouTube video)"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
  referrerpolicy="strict-origin-when-cross-origin"
  allowfullscreen
></iframe>
```

:::

`src` 属性は埋め込みたいページのURLを指定します。`width` 属性は`iframe` の幅を指定します。`height` 属性は`iframe` の高さを指定します。`allowfullscreen` 属性はユーザーが`iframe` を全画面モードで表示できるようにします。また、アクセシビリティのために`title` 属性を`iframe` に指定することも良い習慣です。

一方、`allow` 属性はiframeができることやできないことを定義できます。これは許可リストと呼ばれます。上の例では、`clipboard-write` を追加することで埋め込まれたページがクリップボードに項目を書き込めるようになります。許可リスト内の項目はセミコロンやスペースで区切ることができ、両方を一緒に使うことも可能です。

動画はどこからでも来ることができます。YouTubeやVimeoのような動画サービスからである必要はありません。

また、地図や別のウェブページ、直接HTMLを`iframe` 要素内に埋め込むことも忘れないでください。以下は埋め込み地図の例です。

地図をズームイン・ズームアウトして操作してみてください。

:::interactive_editor

```html
<h1>A Map from Openstreetmap.org Embedded with the iframe Element</h1>

<iframe
  width="425"
  height="350"
  src="https://www.openstreetmap.org/export/embed.html?bbox=3.006134033203125%2C6.150112578753815%2C3.6357879638671875%2C6.749850810550778&amp;layer=mapnik"
  title="Map of Lagos area, Nigeria"
  style="border: 1px solid black"
>
</iframe>
<br />
<small>
  <a href="https://www.openstreetmap.org/#map=11/6.4501/3.3210">
    View Larger Map
  </a>
</small>
```

:::

`iframe` 要素内に直接HTMLを埋め込みたい場合は、`srcdoc` 属性を`src` の代わりに使う必要があります。
