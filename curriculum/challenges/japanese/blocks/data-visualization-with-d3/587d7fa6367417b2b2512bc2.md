# --description--

D3には、ドキュメント内の要素を追加・変更できるいくつかのメソッドがあります。

`select()` メソッドは、ドキュメントから1つの要素を選択します。引数に欲しい要素の名前を指定し、その名前に一致するドキュメント内の最初の要素のHTMLノードを返します。例を示します。

```js
const anchor = d3.select('a');
```

上の例では、ページ内の最初のアンカータグを見つけ、そのHTMLノードを変数`anchor`に保存しています。この選択は他のメソッドと組み合わせて使えます。例の`d3`はD3オブジェクトへの参照で、D3メソッドにアクセスする方法です。

他に便利なメソッドとして`append()`と`text()`があります。

`append()` メソッドは、ドキュメントに追加したい要素を引数に取ります。選択した要素にHTMLノードを追加し、そのノードへのハンドルを返します。

`text()` メソッドは、選択したノードのテキストを設定するか、現在のテキストを取得します。値を設定するには、メソッドの丸括弧内に文字列を引数として渡します。

以下は、番号なしリストを選択し、リスト項目を追加してテキストを入れる例です。

```js
d3.select('ul')?.append('li').text('Very important item');
```

D3では、複数のメソッドをピリオドでつなげて連続して操作を行うことができます。

# --instructions--

`select` メソッドを使ってドキュメント内の`body`タグを選択してください。次に`append`で`h1`タグを追加し、`Learning D3`要素にテキスト`h1`を入れてください。
