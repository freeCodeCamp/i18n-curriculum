# --explanation--

Джейк підкреслює терміновість розслідування порушення безпеки.

`Need to` + дієслово — це загальна конструкція, яка використовується для вираження чогось важливого або необхідного. Наприклад:

`We need to improve security.` — це означає, що покращення безпеки є необхідним.

`Find out` означає дізнатися або отримати інформацію. Наприклад:

`I need to find out who accessed the system.` — це означає, що мовник повинен розслідувати і отримати інформацію.
