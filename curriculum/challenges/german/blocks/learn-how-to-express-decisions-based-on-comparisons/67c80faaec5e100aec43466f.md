# --explanation--

`More robust` bedeutet stärker oder zuverlässiger im Vergleich zu etwas anderem. Zum Beispiel:

`This software is more robust than the previous version because it can handle larger data sets.` – Hier bedeutet `more robust`, dass die Software stärker und leistungsfähiger ist.

`When it comes to` leitet ein bestimmtes Thema ein und bedeutet „bezüglich“ oder „über“. Zum Beispiel:

`When it comes to customer support, this company is very responsive.` – Hier führt `when it comes to` das Thema Kundensupport ein.
