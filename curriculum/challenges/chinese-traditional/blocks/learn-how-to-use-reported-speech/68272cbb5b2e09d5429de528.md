# --explanation--

`Several` 意味著多於幾個，但不是全部。 例如：

`Several users reported issues with the new software.`－這表示超過兩個使用者遇到問題，但不是所有人。

`To target` 意思是瞄準或將某物指向某人或某物。例如：

`The hackers targeted customers by sending fake emails.`－這表示他們試圖專門欺騙這些客戶。
