# --explanation--

`Buenas noches` — це привітання, яке використовують вночі. Воно перекладається як "на добраніч".

Ви можете використовувати `Buenas noches` як для привітання ввечері, **так і** для привітання або прощання з кимось вночі. Наприклад, речення нижче підходить і для привітання, і для прощання вночі:

`Buenas noches, señora López.` – На добраніч, пані Лопес.
