이 도전 과제에서는 Python을 사용해 데이터를 입력받아 다음 작업을 수행하는 데이터 그래프 탐색기를 만들어야 합니다:

- .csv 파일을 세 가지 방법으로 가져오기

  - 코드에 URL 직접 넣기

- 열 이름을 리스트로 저장하기
- 한두 개 열을 선택해 데이터를 Numpy 배열로 변환하기
- 데이터를 산점도나 선 그래프로 표시하기
