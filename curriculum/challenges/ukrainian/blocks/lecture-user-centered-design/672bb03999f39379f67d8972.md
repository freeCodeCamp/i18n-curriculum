# --interactive--

**ПРИМІТКА**: Деякі інтерактивні приклади можуть використовувати CSS, який ви ще не вивчали. Не хвилюйтеся, якщо не розумієте весь код. Мета прикладів — показати вам попередній перегляд цих концепцій дизайну, щоб ви краще зрозуміли, як все працює. Щоб побачити попередній перегляд, потрібно увімкнути інтерактивний редактор.

Поступове розкриття — це шаблон дизайну, який використовується для показу користувачам лише релевантного вмісту залежно від їхньої поточної дії, приховуючи решту. Це робиться для зменшення когнітивного навантаження та зроблення користувацького досвіду більш інтуїтивним.

```js
const btn = document.getElementById('show-details-btn');
const content = document.getElementById('extra-content');

btn.addEventListener('click', () => {
  const isHidden = content.classList.toggle('hidden');
  btn.setAttribute('aria-expanded', !isHidden);
  if (!isHidden) {
    content.focus();
  }
});
```

Наприклад, коли ви користуєтеся пошуковою сторінкою Google, більшість ваших пошуків будуть простими. Тому простий інтерфейс із дуже малою кількістю опцій має сенс. Але для тих небагатьох випадків, коли потрібен більш просунутий пошук, Google надає опцію розширеного пошуку, яка відкриває більше параметрів.

Ви можете знайти опцію розширеного пошуку Google у меню налаштувань на головній сторінці Google. Або ви можете перейти до `google.com/advanced_search`.

Ще одним прикладом поступового розкриття є кнопка «Більше деталей» на продуктах на сайті електронної комерції. Коли ви заходите на сайт, як-от Amazon, ви бачите список зображень продуктів у різних категоріях. Це дозволяє користувачам зручно робити покупки і не перевантажуватися надлишком інформації.

Якщо користувач зацікавлений у певному продукті, він може натиснути на зображення продукту, щоб побачити більше деталей про нього. Відразу показувати всю інформацію може бути надто складно для користувача. Тож, використовуючи поступове розкриття, користувач може вибрати, коли побачити додаткову інформацію.

```js
document.querySelectorAll('.details-btn').forEach(button => {
  button.addEventListener('click', () => {
    const product = button.closest('.product');
    const details = product.querySelector('.product-details');
    const isHidden = details.classList.toggle('hidden');
    product.setAttribute('aria-expanded', !isHidden);
    if (!isHidden) {
      details.focus();
      button.textContent = 'Hide details';
    } else {
      button.textContent = 'More details';
    }
  });
});
```
