# --explanation--

`To impact` bedeutet, eine Auswirkung oder einen Einfluss auf etwas zu haben, oft in bedeutendem Maße. Zum Beispiel:

`Delays in shipping can impact customer experience.` – Das bedeutet, dass Lieferverzögerungen einen negativen Einfluss darauf haben können, wie Kunden den Service wahrnehmen.

In diesem Dialog sagt Maria, dass ungelöste Probleme die Kundenzufriedenheit negativ beeinflussen könnten.
