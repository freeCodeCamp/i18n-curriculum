# --description--

스페인어에서 다른 사람의 이메일 주소를 말할 때 `su` ("그의"/"그녀의")를 사용하세요.

이메일 주소에는 익숙해져야 할 몇 가지 주요 기호도 있습니다:

- `Correo` - 이메일.

- `@` - `Arroba` - 골뱅이(@).

- `.` - `Punto` - 점/마침표.

`Correo electrónico`는 이메일을 공식적이고 완전하게 지칭하는 방법이지만, 상황에 따라 보통 `correo`만 말해도 괜찮습니다.

Luna가 Carlos의 이메일 주소에 대해 말하고 있습니다. 기호를 발음하는 연습에 포커스를 맞추세요.
