# --interactive--

## Kufanya kazi na floats

- **Ufafanuzi**: Floats hutumika kuondoa kipengele kutoka katika mtiririko wake wa kawaida kwenye ukurasa na kuuweka upande wa kushoto au kulia wa kontena lake. Hii husababisha maandishi kuzunguka kipengele kilichofloata.

```css
float: left;
float: right;
```

- **Kufuta floats**: Sifa ya `clear` hutumika kuamua kama kipengele kinahitaji kuhamishwa chini ya kipengele kilichofloata. Unapokuwa na vipengele vingi vilivyojaa kando kando, kunaweza kuwa na matatizo ya kuungana au kushikana katika muundo. Hivyo, mbinu ya `clearfix` ilitengenezwa kutatua tatizo hili.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="clearfix">
  <div class="box left">Left</div>
  <div class="box right">Right</div>
</div>
```

```css
.box {
  width: 100px;
  height: 60px;
  color: white;
  text-align: center;
  line-height: 60px;
}

.left {
  float: left;
  background: teal;
}

.right {
  float: right;
  background: purple;
}

.clearfix::after {
  content: "";
  display: block;
  clear: both;
}
```

:::

## Upangaji wa static, relative na absolute

- **Upangaji wa static**: Huu ni mtiririko wa kawaida wa hati. Vipengele huwekwa kutoka juu kwenda chini na kushoto kwenda kulia mfululizo.
- **Upangaji wa relative**: Hii inakuwezesha kutumia sifa za `top`, `left`, `right` na `bottom` kuweka kipengele ndani ya mtiririko wa kawaida wa hati. Pia unaweza kutumia upangaji wa relative kufanya vipengele vishikane na vipengele vingine kwenye ukurasa.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="relative">Relative Element</div>
```

```css
.relative {
    position: relative;
    top: 30px;
    left: 30px;
}
```

:::

- **Upangaji wa absolute**: Hii inakuwezesha kuondoa kipengele kutoka katika mtiririko wa kawaida wa hati, na kufanya kitendeke huru bila kuathiriwa na vipengele vingine.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  Parent
  <div class="positioned">Absolute</div>
</div>
```

```css
.parent {
  position: relative;
  height: 120px;
  background: #eee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

## Upangaji wa fixed na sticky

- **Upangaji wa fixed**: Kipengele kinapowekwa kwa `position: fixed`, kinaondolewa kutoka katika mtiririko wa kawaida wa hati na kuwekwa kulingana na eneo la kuona (viewport), maana yake kinabaki mahali pale hata mtumizi anapopiga skrini chini au juu. Hii hutumika sana kwa vipengele kama vichwa vya ukurasa au barua za urambazaji ambazo zinahitaji kuonekana kila wakati.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="navbar">Fixed Navbar</div>
<div style="height: 600px;"></div>
```

```css
.navbar {
  position: fixed; 
  top: 0; 
  width: 100%;
  background: #0077ff;
  color: white;
  padding: 8px; 
}
```

:::

- **Upangaji wa sticky**: Aina hii ya upangaji ni mchanganyiko kati ya relative na fixed. Awali, kipengele hufanya kazi kama kinavyopangwa kwa relative, kikibaki ndani ya mtiririko wa hati. Lakini mara mtumizi anapopiga skrini na kipengele kinapopita sehemu fulani, kinagandamana na eneo la kuona (kawaida juu) na hufanya kazi kama fixed. Hii ni nzuri kwa kuunda vipengele kama barua za urambazaji zinazogandamana, ambazo zinakuwa fixed tu mtumizi anapofika sehemu fulani ya ukurasa.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Scroll down</p>
<div class="positioned">Sticky Element</div>
<div style="height: 500px;"></div>
```

```css
.positioned {
  position: sticky;
  top: 30px;
  left: 30px;
  background: #4caf50;
  color: white;
  padding: 6px;
}
```

:::

## Kufanya kazi na sifa ya `z-index`

- **Ufafanuzi**: Sifa ya `z-index` katika CSS hutumika kudhibiti mpangilio wa wima wa vipengele vyenye nafasi maalum vinavyoshikamana kwenye ukurasa.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box box1">1</div>
  <div class="box box2">2</div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

.box {
  position: absolute;
  width: 100px;
  height: 100px;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

.box1 {
  background: lightcoral;
  top: 20px;
  left: 20px;
  z-index: 1;
}

.box2 {
  background: steelblue;
  top: 40px;
  left: 40px;
  z-index: 2;
}
```
