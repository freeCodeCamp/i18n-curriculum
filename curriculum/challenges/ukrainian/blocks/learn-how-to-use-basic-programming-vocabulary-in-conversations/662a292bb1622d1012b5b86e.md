# --description--

Навчіться використовувати `how much` і `how many` для запитань про кількість. `How much` використовується з незлічуваними іменниками, щоб дізнатися про обсяг або ступінь чогось, наприклад `time`, `water` або `money`. Наприклад, `How much money do you have?` запитує про кількість грошей, які не можна порахувати поодинці.

`How many` використовується з злічуваними іменниками, коли можна порахувати елементи, наприклад `books`, `apples` або `people`. Наприклад, `How many books do you own?` запитує про злічувану кількість книг.
