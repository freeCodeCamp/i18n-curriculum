# --description--

Ganze Zahlen und Gleitkommazahlen sind die wichtigsten numerischen Datentypen in Python. Mit ihnen können Sie numerische Daten speichern und mathematische Operationen durchführen.

Schauen wir uns an, was ganze Zahlen und Gleitkommazahlen sind, wie man arithmetische Berechnungen mit ihnen durchführt und welche Methoden Python für die Arbeit mit beiden bereitstellt.

Ganze Zahlen sind Zahlen ohne Dezimalstellen, entweder positiv oder negativ:

```python
my_int_1 = 56
my_int_2 = -4

print(type(my_int_1)) # <class 'int'>
print(type(my_int_2)) # <class 'int'>
```

So führen Sie eine Addition mit ganzen Zahlen durch:

```python
my_int_1 = 56
my_int_2 = 12

sum_ints = my_int_1 + my_int_2
print('Integer Addition:', sum_ints) # Integer Addition: 68
```

So führen Sie eine Subtraktion mit ganzen Zahlen durch:

```python
my_int_1 = 56
my_int_2 = 12

# Subtraction
diff_ints = my_int_1 - my_int_2
print('Integer Subtraction:', diff_ints) # Integer Subtraction: 44
```

So führen Sie eine Multiplikation mit ganzen Zahlen durch:

```python
my_int_1 = 12
my_int_2 = 4

# Multiplication
product_ints = my_int_1 * my_int_2
print('Integer Multiplication:', product_ints) # Integer Multiplication: 48
```

Und so führen Sie eine Division mit ganzen Zahlen durch:

```python
my_int_1 = 56
my_int_2 = 12

# Division
div_ints = my_int_1 / my_int_2
print('Division:', div_ints) # Division: 4.666666666666667
```

Gleitkommazahlen sind positive oder negative Zahlen mit Dezimalstellen, wie `3.14`, `-0.5` oder `0.0`.

```python
my_float_1 = -12.0
my_float_2 = 4.9

print(type(my_float_1)) # <class 'float'>
print(type(my_float_2)) # <class 'float'>
```

Hier ist eine Addition mit Gleitkommazahlen:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_addition = my_float_1 + my_float_2
print('Float Addition:', float_addition) # Float Addition: 17.4
```

Hier ist eine Subtraktion mit Gleitkommazahlen:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_subtraction = my_float_2 - my_float_1
print('Float Subtraction:', float_subtraction) # Float Subtraction: 6.6
```

Hier ist eine Multiplikation mit Gleitkommazahlen:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_multiplication = my_float_2 * my_float_1
print('Float Multiplication:', float_multiplication) # Float Multiplication: 64.80000000000001
```

Und hier ist eine Division mit Gleitkommazahlen:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_division = my_float_2 / my_float_1
print('Float Division:', float_division) # Float Division: 2.222222222222222
```

Wenn Sie eine ganze Zahl und eine Gleitkommazahl addieren, wird das Ergebnis automatisch in eine Gleitkommazahl umgewandelt:

```python
my_int = 56
my_float = 5.4

sum_int_and_float = my_int + my_float

print(sum_int_and_float) # 61.4
print(type(sum_int_and_float)) # <class 'float'>
```

Das gilt auch für andere grundlegende arithmetische Operationen wie Subtraktion, Multiplikation und Division. Wenn Sie ganze Zahlen und Gleitkommazahlen mischen, gibt Python als Ergebnis eine Gleitkommazahl zurück.

Sie können auch komplexere arithmetische Berechnungen durchführen, wie den Rest zweier Zahlen mit dem Modulo-Operator, die Ganzzahldivision und die Potenzierung mit ganzen Zahlen und Gleitkommazahlen.

Der Modulo-Operator (`%`) gibt den Rest zurück, wenn der Wert links durch den Wert rechts geteilt wird:

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

mod_ints = my_int_1 % my_int_2
mod_floats = my_float_2 % my_float_1

print('Integer Modulo:', mod_ints) # Integer Modulo: 8
print('Float Modulo:', mod_floats) # Float Modulo: 1.1999999999999993
```

Die Ganzzahldivision teilt zwei Zahlen und gibt die größte ganze Zahl zurück, die kleiner oder gleich dem Ergebnis ist. Dies geschieht mit dem doppelten Schrägstrich-Operator (`//`):

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

floor_div_ints = my_int_1 // my_int_2
floor_div_floats = my_float_2 // my_float_1

print('Integer Floor Division:', floor_div_ints) # Integer Floor Division: 4
print('Float Floor Division:', floor_div_floats) # Float Floor Division: 2.0
```

Die Potenzierung hebt eine Zahl auf die Potenz einer anderen und wird mit dem doppelten Sternchen-Operator (`**`) durchgeführt:

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

exp_ints = my_int_1 ** my_int_2
exp_floats = my_float_1 ** my_float_2

print('Integer Exponentiation:', exp_ints) # Integer Exponentiation: 951166013805414055936
print('Float Exponentiation:',  exp_floats) # Float Exponentiation: 614787626.1765089
```

Manchmal fällt Ihnen vielleicht auf, dass das Ergebnis einer Operation mit Gleitkommazahlen mehr Dezimalstellen hat als erwartet. Zum Beispiel ist die Summe `0.1 + 0.2` gleich `0.30000000000000004` statt `0.3`.

Das passiert, weil Zahlen im Binärformat gespeichert werden und einige Brüche nicht exakt binär dargestellt werden können. Deshalb werden sie als endliche Annäherungen gespeichert, ähnlich wie der Bruch `1/3` nicht mit einer endlichen Anzahl von Dezimalziffern dargestellt werden kann und nach einer bestimmten Anzahl seiner unendlichen Ziffern (`0.33333...`) abgeschnitten wird.

Das führt zu kleinen Rundungsfehlern.

Python stellt auch eingebaute Funktionen zur Verfügung, um numerische Daten oder Strings in ganze Zahlen oder Gleitkommazahlen umzuwandeln.

Die `float()`-Funktion gibt eine Gleitkommazahl zurück, die aus der angegebenen Zahl konstruiert wird:

```python
my_int_1 = 56
my_float_1 = float(my_int_1)

print(my_float_1)  # 56.0
print(type(my_float_1))  # <class 'float'>
```

Die `int()`-Funktion gibt eine ganze Zahl zurück, die aus der angegebenen Zahl konstruiert wird:

```python
my_float = 12.92563
my_int = int(my_float)

print(my_int)  # 12
print(type(my_int))  # <class 'int'>
```

Außerdem können Sie dieselben eingebauten Funktionen verwenden, um einen String entweder in eine Gleitkommazahl oder eine ganze Zahl umzuwandeln:

```python
my_str_int = '45'
my_str_float = '7.8'

converted_int = int(my_str_int)
converted_float = float(my_str_float)

print(converted_int, type(converted_int))  # 45 <class 'int'>
print(converted_float, type(converted_float))  # 7.8 <class 'float'>
```

Hier sind einige weitere Methoden, die Python für die Arbeit mit ganzen Zahlen und Gleitkommazahlen bereitstellt.

- `round()`: Rundet eine Zahl auf die angegebene Anzahl von Dezimalstellen. Standardmäßig rundet diese Funktion auf die nächste ganze Zahl und gibt eine ganze Zahl ohne Dezimalstellen zurück:
    

```python
my_int_1 = 4.798
my_int_2 = 4.253

rounded_int_1 = round(my_int_1)
rounded_int_2 = round(my_int_2, 1)

print(rounded_int_1) # 5
print(rounded_int_2) # 4.3
```

- `abs()`: gibt den absoluten Wert einer Zahl zurück,
    

```python
num = -15

absolute_value = abs(num)
print(absolute_value) # 15
```

- `pow()`: hebt eine Zahl auf die Potenz einer anderen oder führt modulare Potenzierung durch.
    

```python
result_1 = pow(2, 3)  # Equivalent to 2 ** 3
print(result_1)  # 8

result_2 = pow(2, 3, 5)  # (2 ** 3) % 5
print(result_2)  # 3
```
