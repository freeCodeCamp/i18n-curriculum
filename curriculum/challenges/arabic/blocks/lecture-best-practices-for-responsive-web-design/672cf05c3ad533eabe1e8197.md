# --interactive--

استعلامات الوسائط مكوِّن أساسي في تصميم الويب المتجاوب، تتيح للمطوِّرين تطبيق أنماط مختلفة بناءً على خصائص جهاز المستخدم أو المتصفح. توفر طريقة لتخصيص عرض المحتوى لمجموعة متنوعة من الأجهزة دون تغيير المحتوى نفسه.

في جوهرها، تتكون استعلامات الوسائط من نوع وسائط وواحد أو أكثر من التعبيرات التي تتحقق من شروط محددة. إذا تحققت هذه الشروط، تُطبَّق الأنماط المقابلة. تتيح هذه الآلية إنشاء تصاميم متجاوبة تتكيف مع أحجام الشاشات المختلفة، ودقة الوضوح، وقدرات الأجهزة.

الصيغة الأساسية لاستعلام الوسائط في CSS تبدو هكذا:

```css
@media mediatype and (feature: value) {
  /* CSS rules go here */
}
```

في هذا الهيكل، يحدد mediatype نوع الوسائط التي ينطبق عليها الاستعلام، وزوج feature: value يحدد الشرط الذي يجب تحقيقه لتطبيق الأنماط.

تصف أنواع الوسائط الفئة العامة للجهاز. دعنا نتحدث عن أكثر أنواع الوسائط استخدامًا:

- `all` مناسب لجميع الأجهزة. هذا هو الافتراضي إذا لم يُحدد نوع وسائط.

- `print` مخصص للمواد المطبوعة والوثائق التي تُعرض على الشاشة في وضع معاينة الطباعة.

- `screen` مخصص أساسًا للشاشات.

في الماضي، كانت هناك أنواع وسائط أكثر، مثل `handheld` و `tv`، لكن معظمها أصبح مهجورًا لصالح استخدام الميزات لاستهداف الأجهزة بدقة أكبر.

تصف ميزات الوسائط خصائص محددة لوكيل المستخدم، أو جهاز الإخراج، أو بيئة العمل. دعنا نتحدث عن بعض ميزات الوسائط الأكثر استخدامًا. تشير `width` و `height` إلى عرض وارتفاع إطار العرض، وغالبًا ما تُستخدم مع بادئات `min-` أو `max-` لاستعلامات النطاق. إليك مثالًا:

```css
@media screen and (min-width: 768px) {
  /* Styles for screens at least 768px wide */
}
```

تصف `aspect-ratio` النسبة بين عرض وارتفاع إطار العرض. إليك مثالًا:

```css
@media screen and (aspect-ratio: 16/9) {
  /* Styles for screens with a 16:9 aspect ratio */
}
```

تشير ميزة `orientation` إلى ما إذا كان الجهاز في وضع أفقي (landscape) أو عمودي (portrait). إليك مثالًا:

```css
@media screen and (orientation: landscape) {
  /* Styles for landscape orientation */
}
```

تصف ميزة `resolution` دقة جهاز الإخراج بوحدات النقاط لكل إنش (dpi) أو النقاط لكل سنتيمتر (dpcm). إليك مثالًا:

```css
@media screen and (min-resolution: 300dpi) {
  /* Styles for high-resolution screens */
}
```

تختبر ميزة `hover` ما إذا كان آلية الإدخال الأساسية يمكنها التحويم فوق العناصر. إليك مثالًا:

```css
@media (hover: hover) {
  /* Styles for devices that support hover */
}
```

تكتشف ميزة `prefers-color-scheme` ما إذا كان المستخدم قد طلب سمة ألوان فاتحة أو داكنة. إليك مثالًا:

```css
@media (prefers-color-scheme: dark) {
  /* Styles for dark mode */
}
```

يمكن لاستعلامات الوسائط أيضًا دمج شروط متعددة باستخدام المعاملات المنطقية. يُستخدم المعامل `and` لدمج ميزات وسائط متعددة، بينما يمكن استخدام `not` و `only` لنفي أو عزل استعلامات الوسائط. إليك مثالًا يجمع ميزات متعددة:

```css
@media screen and (min-width: 768px) and (orientation: landscape) {
  /* Styles for landscape screens at least 768px wide */
}
```

من الممكن أيضًا استهداف استعلامات متعددة في قائمة مفصولة بفواصل، والتي تعمل كمعامل "أو":

```css
@media screen and (min-width: 768px), print {
  /* Styles for screens at least 768px wide OR for print */
}
```

عند العمل مع استعلامات الوسائط، من المهم مراعاة التراكم (cascade). استعلامات الوسائط لا تزيد من التحديدية – بل تجمع القواعد الشرطية فقط. قواعد التراكم العادية في CSS لا تزال تنطبق داخل كل استعلام وسائط.

في الممارسة العملية، تُستخدم استعلامات الوسائط غالبًا لإنشاء تخطيطات متجاوبة. نمط شائع هو تعريف نمط أساسي للأجهزة المحمولة ثم استخدام استعلامات الوسائط لتحسين التخطيط للشاشات الأكبر:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer nec odio. Praesent libero. Sed cursus ante dapibus diam.</p>
  <p>Sed nisi. Nulla quis sem at nibh elementum imperdiet. Duis sagittis ipsum. Praesent mauris. Fusce nec tellus sed augue semper porta.</p>
  <p>Mauris massa. Vestibulum lacinia arcu eget nulla. Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos.</p>
</div>
```

```css
/* Base styles for mobile */
.container {
  width: 100%;
  padding: 15px;
}

/* Styles for tablets */
@media screen and (min-width: 768px) {
  .container {
    width: 750px;
    margin: 0 auto;
  }
}

/* Styles for desktops */
@media screen and (min-width: 1024px) {
  .container {
    width: 960px;
  }
}
```

:::

تضمن هذه الطريقة، المعروفة باسم "تصميم متجاوب يبدأ بالمحمول"، أن الأنماط الأساسية مناسبة للأجهزة المحمولة، مع إضافة تحسينات للشاشات الأكبر.

في الختام، استعلامات الوسائط هي أداة قوية في CSS تتيح إنشاء تصاميم ويب متجاوبة وقابلة للتكيف. بفهم كيفية استخدام أنواع وميزات الوسائط المختلفة، يمكن للمطوِّرين إنشاء مواقع توفر تجارب مستخدم مثالية عبر مجموعة واسعة من الأجهزة والتفضيلات. مع استمرار تطور تقنيات الويب، يساعد البقاء على اطلاع بالميزات الجديدة في إنشاء تصاميم متجاوبة أكثر دقة وسهولة في الاستخدام.
