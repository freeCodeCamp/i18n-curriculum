# --description--

Uma `tray` é um recipiente plano e raso usado para carregar ou segurar coisas.Por exemplo, você usa uma `tray` para levar comida ou bebida da cozinha para a mesa.

`Enough` significa ter o quanto você precisa.Por exemplo, se você tem `enough water`, isso significa que você tem toda a água que precisa para beber e não precisa de mais.Pode ser usado para falar sobre quantidade, como ter `enough food`, ou para falar sobre qualidade, como quando algo é `good enough`.
