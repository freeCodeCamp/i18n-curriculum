# --description--

年齢について話すときは、数字に `años` を付けます。例えば：

`Veinte años` - 20歳。

20代の数字は `veinti-` で始まります：

- `Veintiuno años` - 21歳。

- `Veintidós años` - 22歳。

マテオ は年齢を言っています。

# --explanation--

`Veintiocho años` は「28歳」を意味します。

`Años` は「歳」を意味します。例えば：

`Tengo veintiocho años.` - 私は28歳です。

`ñ` の中の特別な文字 `años` に注目してください。
