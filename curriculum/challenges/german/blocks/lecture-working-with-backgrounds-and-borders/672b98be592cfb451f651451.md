# --interactive--

Ein Hintergrundverlauf in CSS ist ein sanfter Übergang zwischen zwei oder mehr Farben, der auf den Hintergrund eines Elements angewendet werden kann. Verläufe ermöglichen es Ihnen, optisch ansprechende Hintergründe zu erstellen, ohne Bilder zu benötigen.

Es gibt zwei Haupttypen von Verläufen in CSS: lineare Verläufe und radiale Verläufe.

Ein linearer Verlauf wechselt die Farben entlang einer geraden Linie. Sie können die Richtung und die beteiligten Farben definieren.

Hier ist die grundlegende Syntax:

```css
background: linear-gradient(direction, color-stop1, color-stop2, ...);
```

In diesem Beispiel verwenden wir die `background`-CSS-Eigenschaft mit dem Wert `linear gradient`.

Die Richtung gibt die Richtung des Verlaufs an. Sie kann ein Winkel sein (wie `45deg`), ein Schlüsselwort (wie `to right`, `to bottom`) oder eine Seite/eine Ecke.

`color-stop` gibt die Farben und Positionen an, an denen die Übergänge des Verlaufs stattfinden.

Um besser zu verstehen, wie lineare Verläufe funktionieren, schauen wir uns das folgende Beispiel an:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient{
  background: linear-gradient(to right, red, yellow);
  height: 40vh;
}
```

:::

Dieses CSS erzeugt einen linearen Verlauf, der von `red` am `left` zu `yellow` am `right` übergeht. Der Verlauf wird auf ein Element mit einer Höhe von `40%` der Viewport-Höhe angewendet. Mehr über `vh`-Einheiten lernen Sie in einer zukünftigen Lektion.

Die `to right`-Richtung bedeutet, dass der Verlauf horizontal von links nach rechts verläuft.

Ein weiterer Verlaufstyp ist der `radial`-Verlauf.

Ein radialer Verlauf wechselt die Farben, die von einem Ursprung (meistens der Mittelpunkt) aus strahlenförmig nach außen in einer kreisförmigen oder elliptischen Form verlaufen.

Hier ist die grundlegende Syntax:

```css
background: radial-gradient(shape size at position, color-stop1, color-stop2, ...)
```

In der Syntax gibt `shape` die Form des Verlaufs an, die `circle` oder `ellipse` sein kann.

`size` bestimmt die Größe der Endform des Verlaufs, die `closest-side`, `closest-corner`, `farthest-side` oder `farthest-corner` sein kann.

`position` bestimmt die Position des Mittelpunkts des Verlaufs, die mit Schlüsselwörtern (wie `center`, `top left`, `bottom right`) oder genauen Werten (wie `50% 50%`, `10px 20px`) angegeben werden kann.

Zuletzt sind Farbstopps eine Liste von Farben, durch die der Verlauf übergeht. Jeder Farbstopp kann optional einen Positionswert (Prozent oder Länge) enthalten, der angibt, wo die Farbe platziert werden soll.

Ein Beispiel wäre:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient{
  background: radial-gradient(circle closest-side at center, red, yellow 50%, green);
  height: 60vh;
}
```

:::

Dieses CSS erzeugt einen kreisförmigen radialen Verlauf, der im Element zentriert ist. Er beginnt mit `red` im Zentrum, wechselt zu `yellow` bei `50%` des Radius und endet mit `green`.

Das Schlüsselwort `closest-side` sorgt dafür, dass die Endform des Verlaufs an die nächste Seite des Elements angepasst wird. Der Verlauf wird auf ein Element mit einer Höhe von `60%` der Viewport-Höhe angewendet.

Das Verständnis, wie man mit CSS-Verläufen arbeitet, kann Ihre Designs erheblich verbessern, indem es optisch ansprechende Hintergründe ohne die Notwendigkeit von Bildern bietet.

Mit Optionen wie linearen Verläufen für sanfte Übergänge und radialen Verläufen für kreisförmige Effekte bieten sie sowohl Flexibilität als auch Kreativität im Webdesign.
