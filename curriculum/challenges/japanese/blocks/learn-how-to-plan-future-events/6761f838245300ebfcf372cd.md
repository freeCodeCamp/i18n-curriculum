# --explanation--

`Unauthorized`は「許可されていない」または「承認されていない」という意味です。これは接頭辞`un-`を、許可されたまたは承認されたことを意味する語幹`authorized`に付けて作られます。例えば：

`Unauthorized users tried to access the system.` - これは許可を持たないユーザーがアクセスしようとしたことを意味します。
