# --description--

Sea `r` el residuo cuando ${(a − 1)}^n + {(a + 1)}^n$ se divide por $a^2$.

Por ejemplo, si $a = 7$ y $n = 3$, entonces $r = 42: 6^3 + 8^3 = 728 ≡ 42 \\ \text{mod}\\ 49$. Y a medida que `n` varía, también lo hará `r`, pero para $a = 7$ resulta que $r_{max} = 42$.

Para $3 ≤ a ≤ 1000$, encuentra $\sum{r}_{max}$.
