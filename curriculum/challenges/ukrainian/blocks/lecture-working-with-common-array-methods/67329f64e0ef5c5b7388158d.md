# --interactive--

У JavaScript метод `indexOf()` корисний для пошуку першого індексу конкретного елемента в масиві. Якщо елемент не знайдено, він поверне `-1`. Ось базовий синтаксис:

```js
array.indexOf(element, fromIndex)
```

`element` представляє значення, яке ви хочете знайти в масиві, а параметр `fromIndex` — позицію, з якої має починатися пошук. Параметр `fromIndex` є необов’язковим. Якщо `fromIndex` не вказано, пошук починається з початку масиву. Розглянемо приклад:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "banana"];
let index = fruits.indexOf("banana");
console.log(index); // 1
```

:::

У цьому прикладі у нас є масив `fruits`, що містить різні назви фруктів. Ми використовуємо метод `indexOf()`, щоб знайти індекс рядка `banana` у масиві `fruits`. Оскільки `banana` присутній на індексі `1`, метод повертає `1`, яке зберігається у змінній `index` і виводиться в консоль.

Якщо елемент, який ви шукаєте, не знайдено в масиві, `indexOf()` повертає `-1`. Наприклад:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let index = fruits.indexOf("grape");
console.log(index); // -1
```

:::

Тут ми шукаємо рядок `grape` у масиві fruits за допомогою `indexOf()`. Оскільки `grape` відсутній у масиві, метод повертає `-1`, яке зберігається у змінній `index` і виводиться в консоль.

Якщо ви хочете почати пошук елемента після певного індексу, ви можете передати другий аргумент, як у цьому прикладі:

:::interactive_editor

```js
let colors = ["red", "green", "blue", "yellow", "green"];
let index = colors.indexOf("green", 3);
console.log(index); // 4
```

:::

У цьому прикладі пошук не починається з початку масиву, а починається з індексу `3`, який дорівнює `yellow`, і отримує вихідні дані `4`.
