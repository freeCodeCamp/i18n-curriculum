# --description--

Нехай $H$ — гіпербола, визначена рівнянням $12x^2 + 7xy - 12y^2 = 625$.

Далі визначимо $X$ як точку (7, 1). Можна побачити, що $X$ належить $H$.

Тепер визначимо послідовність точок у $H, \\{P_i : i ≥ 1\\}$, таку що:

- $P_1 = (13, \frac{61}{4})$.
- $P_2 = (\frac{-43}{6}, -4)$.
- Для $i > 2$, $P_i$ — це єдина точка в $H$, відмінна від $P_{i - 1}$, така що пряма $P_iP_{i - 1}$ паралельна прямій $P_{i - 2}X$. Можна показати, що $P_i$ визначена однозначно, а її координати завжди раціональні.

<img alt="animation showing defining points P_1 to P_6" src="https://cdn.freecodecamp.org/curriculum/project-euler/sequence-of-points-on-a-hyperbola.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Відомо, що $P_3 = (\frac{-19}{2}, \frac{-229}{24})$, $P_4 = (\frac{1267}{144}, \frac{-37}{12})$ та $P_7 = (\frac{17\\,194\\,218\\,091}{143\\,327\\,232}, \frac{274\\,748\\,766\\,781}{1\\,719\\,926\\,784})$.

Знайдіть $P_n$ для $n = {11}^{14}$ у такому форматі: якщо $P_n = (\frac{a}{b}, \frac{c}{d})$, де дроби є в найпростіших формах, а знаменники додатні, тоді відповідь дорівнює $(a + b + c + d)\bmod 1\\,000\\,000\\,007$.

Для $n = 7$ відповідь була б: $806\\,236\\,837$.
