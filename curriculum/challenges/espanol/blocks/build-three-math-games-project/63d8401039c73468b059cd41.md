# --instructions--

Para este desafío, necesitas crear tres juegos matemáticos usando Python que hagan lo siguiente:

- Juego de diagrama de dispersión
  - Genera puntos aleatorios en un gráfico y el jugador tiene que ingresar las coordenadas (x,y)
  - Para mayor dificultad, haz el gráfico más grande
- Juego de práctica de álgebra
  - Genera problemas de un paso y dos pasos con valores enteros aleatorios y el jugador tiene que ingresar la respuesta
  - Usa valores positivos y negativos. Para mayor dificultad, haz que los números sean más grandes
- Juego de proyectiles
  - Muestra un "muro" con altura y ubicación aleatorias. El jugador tiene que mover los deslizadores para ajustar una trayectoria parabólica y superar el muro.
  - Para mayor dificultad, crea un segundo nivel donde los jugadores ingresen a, b y c sin deslizadores
