# --interactive--

Поле і відступ — це важливі властивості в CSS для створення добре структурованих, читабельних і візуально привабливих вебсторінок.

Поля контролюють простір поза межами елемента, допомагаючи відокремити його від інших елементів і визначити структуру макета, тоді як відступи контролюють простір всередині елемента, покращуючи читабельність вмісту та естетичний вигляд.

Щоб краще зрозуміти відмінності між `margin` і `padding`, давайте розглянемо кілька прикладів.

Ось HTML-приклад трьох абзаців на сторінці:

:::interactive_editor

```html
<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

:::

Щоб додати пробіл зверху кожного абзацу, ви можете використати властивість `margin-top` так:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  margin-top: 20px;
  border: 2px solid black;
}
```

:::

У цьому прикладі ми застосовуємо `20px` поле лише зверху кожного абзацу.

Також ми додали суцільний чорний кордон з усіх чотирьох сторін, щоб краще бачити поле.

Чотири різні властивості `margin` — це `margin-top`, `margin-right`, `margin-bottom` і `margin-left`.

Ось приклад використання всіх чотирьох властивостей:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 30px;
  margin-left: 40px;
  border: 2px solid black;
}
```

:::

У цьому прикладі ми призначаємо значення пробілу з усіх чотирьох сторін абзацу. Також додано суцільний чорний кордон, щоб краще бачити поля.

Інший спосіб використання властивості `margin` — це скорочене записування. Ви можете вказати одне, два, три або чотири значення одночасно. Давайте розглянемо ці варіанти разом.

Якщо використати одне значення у скороченій властивості `margin`, це саме значення буде застосоване до всіх чотирьох сторін цільового елемента.

Ось приклад використання одного значення у скороченій властивості `margin`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px;
}
```

:::

Цей приклад коду застосує `10px` `margin` рівномірно з усіх чотирьох сторін абзацу.

Якщо вказати два значення, перше застосовується до `top` і `bottom`, а друге — до `left` і `right` сторін елемента.

Ось приклад використання двох значень для скороченої властивості `margin`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px;
}
```

:::

Це встановлює поля `top` і `bottom` рівними `10px`, а `20px` — для `left` і `right` абзацу.

Якщо вказати три значення, перше застосовується до поля `top`, друге — до полів `left` і `right`, а третє — до поля `bottom`.

Ось приклад для кращого розуміння:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px;
}
```

:::

Це встановлює поле `10px` для `top`, `20px` для `left` і `right`, а `30px` для `bottom`.

При використанні чотирьох значень ви отримуєте більше контролю, оскільки можете незалежно вказати кожне значення поля для кожної сторони цільового елемента.

Перше значення відповідає `top`, друге — `right`, третє — `bottom`, а четверте — `left`.

Ось приклад використання скороченої властивості поля з чотирма значеннями:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px 40px;
}
```

:::

Це встановлює поле `10px` для `top`, `20px` для `right`, `30px` для `bottom` і `40px` для `left`.

Властивість `padding` використовується для додавання простору всередині елемента, між вмістом і його кордоном.

Як і властивість `margin`, чотири властивості `padding` — це `padding-top`, `padding-right`, `padding-bottom` і `padding-left`.

Ось приклад, як встановити `padding` для абзацу:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 30px;
  padding-left: 40px;
  border: 2px solid black;
}
```

:::

Це встановлює відступ `10px` для `top`, `20px` для `right`, `30px` для `bottom` і `40px` для `left`.

Як бачите, `padding` застосовується до вмісту, який знаходиться всередині кордону, на відміну від `margin`, що застосовується поза межами кордону.

Так само, як і для властивості `margin`, ви можете обрати скорочений запис для властивості `padding`.

Ви можете вказати одне, два, три або чотири значення і для скороченої властивості `padding`.

Ось приклад використання скороченої властивості `padding` для абзацу з попереднього прикладу:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding: 10px 20px 30px 40px;
  border: 2px solid black;
}
```

:::

У цьому прикладі, використовуючи скорочений запис, код встановить відступ `10px` для `top`, `20px` для `right`, `30px` для `bottom` і `40px` для `left` абзацу.
