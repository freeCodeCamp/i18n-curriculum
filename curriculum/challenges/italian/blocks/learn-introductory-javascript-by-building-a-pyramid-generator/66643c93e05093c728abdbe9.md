# --description--

Prima di andare avanti, prenditi un momento per rivedere come funzionano le funzioni.

Dichiara una funzione chiamata `addTwoNumbers`. Questa funzione dovrebbe prendere due argomenti e restituire la somma di questi due argomenti.

La tua funzione non dovrebbe usare valori codificati direttamente. Un esempio di funzione con valori codificati potrebbe essere:

```js
function sayName(firstName, lastName) {
  return "John Doe";
}

sayName("Camper", "Cat");
```

Questa funzione restituirebbe `"John Doe"` indipendentemente dagli argomenti passati ai parametri `firstName` e `lastName`, quindi `"John Doe"` è considerato un valore codificato direttamente.

Dichiara una variabile `sum` e assegnale il valore della chiamata alla tua funzione `addTwoNumbers` con `5` e `10` come argomenti. Stampa la variabile `sum` nella console.
