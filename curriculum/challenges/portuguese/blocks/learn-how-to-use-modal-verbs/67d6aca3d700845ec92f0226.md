`Could` é usado para descrever algo que é possível mas não garantido. Por exemplo:

- `This bug could cause the app to crash.` - Há uma possibilidade de que o app pare de funcionar por causa deste bug.

- `We could add a new feature to improve user experience.` - Adicionar uma funcionalidade é uma forma possível de melhorar o aplicativo.
