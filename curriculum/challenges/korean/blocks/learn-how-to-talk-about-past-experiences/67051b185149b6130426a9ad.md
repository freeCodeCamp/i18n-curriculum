# --explanation--

`Simple Past`에서 질문은 보통 질문 단어(e.g., `what`, `where`, `how`, `who`)로 시작하며 다음 구조를 사용합니다:

**질문 단어 + `did` + 주어 + 기본 동사.**

Lisa의 질문인 `What did you find there?`는 James의 행동 결과에 관한 구체적인 정보를 묻기 위해 `what`를 사용합니다. 다른 예는 다음과 같습니다:

- `Where did you go yesterday?` - 위치에 대해 묻는 경우.

- `How did they solve the problem?` - 방법에 대해 묻는 경우.

- `Who did you call last night?` - 사람에 대해 묻는 경우.

대화에서 Lisa는 버그를 찾는 과정에서 James가 발견한 것에 대해 묻고 있습니다.
