## Абсолютні одиниці

- **`px` (пікселі)**: Ця абсолютна одиниця є фіксованою одиницею вимірювання в CSS. Вона є найпоширенішою абсолютною одиницею і забезпечує точний контроль над розмірами. `1px` завжди дорівнює 1/96 дюйма.
- **`in` (дюйм)**: Ця абсолютна одиниця дорівнює 96px.
- **`cm` (сантиметри)**: Ця абсолютна одиниця дорівнює 25.2/64 дюйма.
- **`mm` (міліметри)**: Ця абсолютна одиниця дорівнює 1/10 сантиметра.
- **`q` (четверті міліметра)**: Ця абсолютна одиниця дорівнює 1/40 сантиметра.
- **`pc` (піки)**: Ця абсолютна одиниця дорівнює 1/6 дюйма.
- **`pt` (пункти)**: Ця абсолютна одиниця дорівнює 1/72 дюйма.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="units">
  <div class="unit px">px</div>
  <div class="unit inch">in</div>
  <div class="unit cm">cm</div>
  <div class="unit mm">mm</div>
  <div class="unit q">q</div>
  <div class="unit pc">pc</div>
  <div class="unit pt">pt</div>
</div>
```

```css
.units {
  display: flex;
  gap: 12px;
  flex-wrap: wrap;
  align-items: flex-end;
}

.unit {
  background: steelblue;
  color: white;
  text-align: center;
  padding: 4px;
}

.px {
  width: 40px;
  height: 40px;
}

.inch {
  width: 0.5in;
  height: 0.5in;
}

.cm {
  width: 1cm;
  height: 1cm;
}

.mm {
  width: 10mm;
  height: 10mm;
}

.q {
  width: 40q;
  height: 40q;
}

.pc {
  width: 6pc;
  height: 6pc;
}

.pt {
  width: 36pt;
  height: 36pt;
}
```

## Відносні одиниці

- **Відсотки**: Ці відносні одиниці дозволяють визначати розміри, габарити та інші властивості як пропорцію від батьківського елемента. Наприклад, якщо ви встановите `width: 50%;` для елемента, він займатиме половину ширини свого батьківського контейнера.
- **`em` одиниця**: Ці одиниці відносні до розміру шрифту елемента. Якщо ви використовуєте `ems` для властивості `font-size`, розмір тексту буде відносним до розміру шрифту батьківського елемента.
- **`rem` одиниця**: Ці одиниці відносні до розміру шрифту кореневого елемента, яким є `html` елемент.
- **`vh` одиниця**: `vh` означає `"viewport height"`, а `1vh` дорівнює 1% висоти вікна перегляду.
- **`vw` одиниця**: `vw` означає `"viewport width"`, а `1vw` дорівнює 1% ширини вікна перегляду.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  <div class="box percent">50%</div>
  <div class="box em">2em</div>
  <div class="box rem">2rem</div>
  <div class="box vh">10vh</div>
  <div class="box vw">10vw</div>
</div>
```

```css
html {
  font-size: 16px;
}

.parent {
  width: 200px;
  font-size: 20px;
  border: 2px dashed #555;
  padding: 10px;
  display: flex;
  flex-direction: column;
  gap: 12px;
}

.box {
  background: seagreen;
  color: white;
  text-align: center;
  padding: 6px;
}

.percent {
  width: 50%;
}

.em {
  font-size: 2em;
}

.rem {
  font-size: 2rem;
}

.vh {
  height: 10vh;
}

.vw {
  width: 10vw;
}
```

## `calc` функція

- **`calc()` функція**: За допомогою функції `calc()` ви можете виконувати обчислення безпосередньо у своїх таблицях стилів, щоб динамічно визначати значення властивостей. Це означає, що ви можете створювати гнучкі та адаптивні інтерфейси користувача, обчислюючи розміри на основі розміру вікна перегляду або інших елементів.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="calc-box">calc()</div>
```

```css
.calc-box {
  width: calc(100% - 40px);
  padding: 10px;
  background: steelblue;
  color: white;
  text-align: center;
  border: 2px solid black;
}
```
