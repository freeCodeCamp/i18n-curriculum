# --description--

フレーズ **`Soy` + [国籍]** は、スペイン語で自分の国籍を言う最も一般的な方法です。英語の "I am + [nationality]" に相当します。

このフレーズは、`¿Cuál es tu nacionalidad?` という質問に対する直接的な答えとして自己紹介で使われます。ここで `nacionalidad` は「国籍」を意味します。

スペイン語では、国籍は文の先頭で使われる場合を除き大文字にしません。また、話者の性別に合わせて性別一致させる必要があります。例えば：

| 国名 | 国籍（女性形） | 国籍（男性形） |
| :---: | :---: | :---: |
| **`México`** | `mexicana` | `mexicano` |
| **`Argentina`** | `argentina` | `argentino` |
| **`Perú`** | `peruana` | `peruano` |
| **`Uruguay`** | `uruguaya` | `uruguayo` |
| **`El Salvador`** | `salvadoreña` | `salvadoreño` |
