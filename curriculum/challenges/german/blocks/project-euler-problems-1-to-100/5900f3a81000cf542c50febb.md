# --description--

Die Primzahlen 3, 7, 109 und 673 sind ziemlich bemerkenswert. Wenn man zwei beliebige Primzahlen nimmt und sie in beliebiger Reihenfolge aneinanderhängt, ist das Ergebnis immer eine Primzahl. Zum Beispiel sind sowohl 7109 als auch 1097 Primzahlen, wenn man 7 und 109 nimmt. Die Summe dieser vier Primzahlen, 792, stellt die kleinste Summe für eine Menge von vier Primzahlen mit dieser Eigenschaft dar.

Finden Sie die kleinste Summe für eine Menge von fünf Primzahlen, bei der jede Kombination aus zwei Primzahlen aneinandergereiht eine weitere Primzahl ergibt.
