# --description--

Daniel é do Canadá e vai fazer uma entrevista em uma empresa chinesa. A empresa exige habilidades básicas de comunicação em chinês. O entrevistador é o gerente de P&D. Daniel precisa preparar uma breve autoapresentação em chinês. Ajude-o a completar suas frases para que ele possa causar uma boa impressão.

# --instructions--

Encontre as palavras corretas da lista abaixo para ajudar Daniel.  

**IMPORTANTE**: Você pode usar cada palavra **apenas uma vez**.

`开发者 (kāi fā zhě)`, `您好 (nín hǎo)`, `我是 (wǒ shì)`.

`我是 (wǒ shì)`

### --feedback--

Esta frase significa "I am", que Daniel usa para apresentar sua nacionalidade.

`开发者 (kāi fā zhě)`

### --feedback--

Esta palavra indica uma profissão envolvendo desenvolvimento de software.

# --explanation--

Quando você quer cumprimentar alguém educadamente em um ambiente formal, como uma entrevista, você pode usar `您好 (nín hǎo)`.

Para apresentar de onde você é, você pode usar a estrutura comum `我是 (wǒ shì)` + nacionalidade.

Se o seu trabalho envolve criar software e manter aplicações, você pode apresentar sua profissão como `开发者 (kāi fā zhě)`.
