# --description--

모든 사람이 알다시피 방정식 $x^2 = -1$은 실수 $x$에 대해 해가 없습니다.

하지만 허수 $i$를 도입하면 이 방정식은 두 해를 갖습니다: $x = i$ 와 $x = -i$.

한 단계 더 나아가면 방정식 ${(x - 3)}^2 = -4$는 두 개의 복소수 해를 갖는데, $x = 3 + 2i$ 와 $x = 3 - 2i$이며, 이들은 서로의 복소수 켤레라고 부릅니다.

형태가 $a + bi$인 수를 복소수라고 합니다.

일반적으로 $a + bi$ 와 $a − bi$는 서로의 복소수 켤레입니다. 가우스 정수는 $a$와 $b$가 모두 정수인 복소수 $a + bi$입니다.

일반 정수도 가우스 정수에 포함되며 ($b = 0$인 경우) 이를 가우스 정수와 구분하기 위해 $b ≠ 0$인 가우스 정수와 달리 "유리 정수"라고 부릅니다.

가우스 정수는 유리 정수 $n$의 약수라고 할 때, 그 결과도 가우스 정수여야 합니다.

예를 들어 5를 $1 + 2i$로 나누면 다음과 같이 간단히 할 수 있습니다:

분자와 분모에 $1 + 2i$의 복소수 켤레인 $1 − 2i$를 곱합니다.

결과는 다음과 같습니다:

$$\frac{5}{1 + 2i} = \frac{5}{1 + 2i} \frac{1 - 2i}{1 - 2i} = \frac{5(1 - 2i)}{1 - {(2i)}^2} = \frac{5(1 - 2i)}{1 - (-4)} = \frac{5(1 - 2i)}{5} = 1 - 2i$$

따라서 $1 + 2i$는 5의 약수입니다.

반면 $1 + i$는 5의 약수가 아닙니다. 왜냐하면:

$$\frac{5}{1 + i} = \frac{5}{2} - \frac{5}{2}i$$

또한 가우스 정수 ($a + bi$)가 유리 정수 $n$의 약수라면, 그 복소수 켤레 ($a − bi$)도 $n$의 약수임을 주목하세요. 실제로 5는 실수 부분이 양수인 여섯 개의 약수를 갖습니다: {1, 1 + 2i, 1 − 2i, 2 + i, 2 − i, 5}.

다음은 처음 다섯 개의 양의 유리 정수에 대한 모든 약수의 표입니다:

| n | 실수 부분이 양수인 가우스 정수 약수들 | 이 약수들의 합 s(n) |
|---|---------------------------------------|--------------------|
| 1 | 1                                     | 1                  |
| 2 | 1, 1 + i, 1 - i, 2                    | 5                  |
| 3 | 1, 3                                  | 4                  |
| 4 | 1, 1 + i, 1 - i, 2, 2 + 2i, 2 - 2i, 4 | 13                 |
| 5 | 1, 1 + 2i, 1 - 2i, 2 + i, 2 - i, 5    | 12                 |

따라서 실수 부분이 양수인 약수들에 대해 $\displaystyle\sum_{n=1}^5 s(n) = 35$입니다.

$1 ≤ n ≤ {10}^5$일 때, $\displaystyle\sum_{n = 1}^{{10}^5} s(n) = 17924657155$입니다.

그렇다면 $\displaystyle\sum_{n=1}^{{10}^8} s(n)$의 값은 얼마일까요?
