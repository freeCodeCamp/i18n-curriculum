# --description--

더 복잡한 조건문은 여러 조건을 포함할 수 있습니다. 그래서 논리 연산자를 사용해 여러 조건을 결합합니다. 논리 연산자는 `&&`, `||` 그리고 `!`이며 각각 `and`, `or` 그리고 `not`를 나타내는 데 사용됩니다.

논리 연산자 `||`는 두 불리언 조건을 결합하는 데 사용합니다. 조건 중 적어도 하나가 `true`일 때 `true`을 반환합니다. 그렇지 않으면 `false`를 반환합니다:

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a > b || a > c) {
  console.log("At least one of the conditions is true");
} else {
  console.log("Both of the conditions are false");
}
```

논리 연산자 `&&`는 두 불리언 조건을 결합하는 데 사용합니다. 두 조건 모두 `true`일 때만 `true`을 반환합니다. 그렇지 않으면 `false`을 반환합니다:

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a < b && a < c) {
  console.log("Both of the conditions are true");
} else {
  console.log("At least one of the conditions is false");
}
```

논리 연산자 `!`는 불리언 조건을 부정하는 데 사용합니다. 조건이 `true`일 때 `false`를 반환합니다. 그렇지 않으면 `false`을 반환합니다:

```javascript
let a = 5;
let b = 10;

if (!(a > b)) {
  console.log("The condition is false");
} else {
  console.log("The condition is true");
}
```

위 예제에서 추가된 소괄호 쌍은 코드를 더 읽기 쉽게 만들기 위해 사용했습니다. 반드시 사용해야 하는 것은 아닙니다.

# --questions--

## --text--

웹 앱에서 사용자에게 `if`를 표시하기 위해 다음 조건을 확인하는 `Welcome!` 문을 작성하는 임무가 있습니다:

1. 사용자는 프리미엄 계정(`isPremium`)을 가지고 있거나 1년 이상 회원(`membershipDuration` > 12개월)이어야 합니다.

1. 사용자는 현재 서비스 이용이 차단되어 있지 않아야 합니다(`!isBlocked`).

어떤 if 문이 이 조건들을 올바르게 확인합니까?

## --answers--

```javascript
if (isPremium && membershipDuration > 12 && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (isPremium || (membershipDuration > 12 && !isBlocked)) {
  console.log("Welcome!");
}
```

---

```javascript
if ((isPremium || membershipDuration > 12) && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (!isPremium || membershipDuration <= 12 || isBlocked) {
  console.log("Welcome!");
}
```
