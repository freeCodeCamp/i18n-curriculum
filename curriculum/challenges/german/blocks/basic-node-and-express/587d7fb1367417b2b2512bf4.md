# --description--

Middleware kann an einer bestimmten Route mit `app.METHOD(path, middlewareFunction)` eingebunden werden. Middleware kann auch innerhalb einer Routendefinition verkettet werden.

Sehen Sie sich folgendes Beispiel an:

```js
app.get('/user', function(req, res, next) {
  req.user = getTheUserSync();  // Hypothetical synchronous operation
  next();
}, function(req, res) {
  res.send(req.user);
});
```

Dieser Ansatz ist nützlich, um die Serveroperationen in kleinere Einheiten aufzuteilen. Das führt zu einer besseren App-Struktur und der Möglichkeit, Code an verschiedenen Stellen wiederzuverwenden. Dieser Ansatz kann auch verwendet werden, um eine Validierung der Daten durchzuführen. An jedem Punkt des Middleware-Stacks können Sie die Ausführung der aktuellen Kette blockieren und die Kontrolle an speziell dafür vorgesehene Funktionen zur Fehlerbehandlung übergeben. Oder Sie können die Kontrolle an die nächste passende Route weitergeben, um Sonderfälle zu behandeln. Wie das funktioniert, sehen wir im Abschnitt zu fortgeschrittenem Express.

# --instructions--

Verketten Sie in der Route `app.get('/now', ...)` eine Middleware-Funktion und den finalen Handler. In der Middleware-Funktion sollen Sie die aktuelle Zeit im `req.time`-Schlüssel des Anfrageobjekts hinzufügen. Sie können `new Date().toString()` verwenden. Im Handler antworten Sie mit einem JSON-Objekt, das die Struktur `{time: req.time}` hat.

**Hinweis:** Der Test wird nicht bestehen, wenn Sie die Middleware nicht verketten. Wenn Sie die Funktion an einer anderen Stelle einbinden, schlägt der Test fehl, auch wenn das Ausgabeergebnis korrekt ist.

# --hints--

Der Endpunkt /now sollte Middleware eingebunden haben

```js
  const response = await fetch(code + '/_api/chain-middleware-time');
  if (!response.ok) {
    throw new Error(await response.text());
  }
  const data = await response.json();
  assert.equal(
    data.stackLength,
    2,
    '"/now" route has no mounted middleware'
  );
```

Der `/now`-Endpunkt sollte die aktuelle Zeit zurückgeben.

```js
  const response = await fetch(code + '/_api/chain-middleware-time');
  if (!response.ok) {
    throw new Error(await response.text());
  }
  const data = await response.json();
  var now = new Date();
  assert.isAtMost(
    Math.abs(new Date(data.time) - now),
    20000,
    'the returned time is not between +- 20 secs from now'
  );
```
