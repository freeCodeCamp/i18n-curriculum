# --explanation--

`Must have happened` si usa per fare un’ipotesi logica su qualcosa accaduto nel passato.

`Must` esprime una forte certezza. `Have` + `Past Participle` (come `happened`) si riferisce a qualcosa nel passato. Per esempio:

- `He must have forgotten to update the system.` - Probabilmente non si è ricordato di fare l’aggiornamento.

- `They must have made some changes to the app.` - Probabilmente l’app è stata aggiornata o migliorata.

Jake usa delle prove (`unusual activity on the servers`) per concludere che la violazione `must have happened` la scorsa notte. Questa struttura è utile per fare assunzioni logiche.
