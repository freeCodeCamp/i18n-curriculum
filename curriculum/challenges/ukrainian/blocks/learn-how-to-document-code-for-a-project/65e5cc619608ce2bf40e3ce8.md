# --description--

У цьому завданні ви дізнаєтеся про фрази на кшталт `Did you get it?`, `Got it` і `What else?`. Ці фрази є загальними в англійських розмовах. `Did you get it?` — це питання, яке запитує, чи хтось щось зрозумів. `Got it` — це короткий спосіб сказати `I understood.`. `What else?` використовується, щоб попросити додаткову інформацію або елементи.

Наприклад, на зустрічі хтось може сказати `Did you get it?`, а ви можете відповісти `Got it, what else do we need to discuss?`.
