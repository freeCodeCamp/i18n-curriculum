# --description--

`plugin` 是一個為現有電腦程式添加特定功能的組件。當程式支援 `plugins` 時，它啟用訂製功能。  
在整合開發環境的背景關係中，`plugins` 可以提供額外的功能，例如新的程式編寫、偵錯或增強使用者介面的工具。

例如，整合開發環境中的程式碼格式化工具通常是 `plugin`。
