# --interactive--

CSS에서 `z-index` 속성은 페이지에서 겹치는 위치 지정 요소들의 수직 쌓임 순서를 제어하는 데 사용됩니다. 여러 요소가 겹쳐 쌓일 때, `z-index` 값이 어떤 요소가 위에 나타날지 결정합니다. `z-index` 값이 높을수록 요소가 사용자에게 더 가까이 위치하며, 낮은 값은 요소를 쌓임에서 더 뒤쪽에 배치합니다.

하지만 `z-index`는 위치 지정된 요소에만 작동하는데, 이는 요소가 `static`가 아닌 위치 값, 예를 들어 `relative`, `absolute`, 또는 `fixed` 중 하나를 가져야 한다는 뜻입니다. 기본 `z-index` 값은 `auto`로, 이는 요소를 기본 쌓임 순서에 배치합니다. 세 개의 박스 예제를 살펴보겠습니다:

```html
<div class="container">
  <div class="box1">Box 1</div>
  <div class="box2">Box 2</div>
  <div class="box3">Box 3</div>
</div>
```

이제 박스들이 서로 겹치도록 위치를 지정하는 스타일을 적용할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box1">Box 1</div>
  <div class="box2">Box 2</div>
  <div class="box3">Box 3</div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

.box1 {
  position: absolute;
  z-index: 1;
  background: lightcoral;
  top: 20px;
  left: 20px;
  width: 100px;
  height: 100px;
}

.box2 {
  position: absolute;
  z-index: 3;
  background: gold;
  top: 40px;
  left: 40px;
  width: 100px;
  height: 100px;
}

.box3 {
  position: absolute;
  z-index: 2;
  background: lightgreen;
  top: 60px;
  left: 60px;
  width: 100px;
  height: 100px;
}
```

:::

컨테이너는 `relative` 위치 지정이 설정되고, 내부에 중첩된 모든 박스는 `absolute` 위치 지정이 설정됩니다. 각 박스는 `z-index`에 대해 서로 다른 값을 가지며, 이로 인해 박스들이 서로 위에 겹쳐집니다.

`z-index`를 웹페이지에서 레이어를 만드는 방법으로 생각할 수 있으며, `z-index` 값이 높은 요소가 낮은 값의 요소 위에 배치됩니다. 이는 모달, 팝업, 툴팁 같은 복잡한 레이아웃에서 겹치는 요소의 동작을 제어하는 데 특히 유용합니다.
