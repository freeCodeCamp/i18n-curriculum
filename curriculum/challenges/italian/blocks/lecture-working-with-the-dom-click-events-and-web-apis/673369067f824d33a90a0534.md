# --interactive--

Quando l’utente clicca su un pulsante o c’è una modifica in un modulo, questo è noto come evento. Nei tuoi programmi, avrai bisogno di un modo per ascoltare questi eventi e rispondere a essi.

Il metodo `addEventListener()` viene usato per ascoltare gli eventi. Prende due argomenti: l’evento che vuoi ascoltare e una funzione che verrà chiamata quando l’evento si verifica.

Ecco la sintassi di base:

```js
element.addEventListener("event", listener);
```

L’elemento è l’elemento HTML da monitorare per gli eventi, e event specifica il tipo di evento da ascoltare, come `"click"`.

Il listener è un oggetto che riceverà la notifica quando l’evento si verifica. La maggior parte delle volte, sarà una funzione che definisci per gestire l’evento.

Ecco un esempio:

```js
element.addEventListener("click", () => {
  // code to run when the click event occurs
});
```

Puoi anche scegliere di passare un riferimento a una funzione in questo modo:

```js
function handleClick() {
  // code to run when the click event occurs
}

element.addEventListener("click", handleClick);
```

A volte potresti voler creare una funzione separata per gestire l’evento. Questo può rendere il tuo codice più facile da leggere e mantenere.

L’argomento listener può anche essere `null`, oppure può essere un oggetto che usa l’interfaccia `EventListener`. L’interfaccia `EventListener` definisce un singolo metodo chiamato `handleEvent()`. Questo metodo viene chiamato automaticamente ogni volta che si verifica l’evento che stai ascoltando. Usare questa interfaccia permette allo stesso oggetto di gestire più eventi se necessario.

La maggior parte delle volte, non avrai bisogno di usare questa interfaccia, e potrai invece passare una funzione che verrà chiamata quando l’evento si verifica.

Ora, diamo un’occhiata a un esempio per capire meglio come funziona:

In questo esempio, abbiamo un elemento `button` con l’id `btn`:

```html
<button id="btn">Show alert</button>
```

Vogliamo ascoltare l’evento `"click"` su questo pulsante e mostrare un alert quando il pulsante viene cliccato.

Prima dobbiamo consultare quell’elemento `button` nel nostro codice JavaScript. Poi dobbiamo aggiungere un event listener per ascoltare l’evento click e mostrare un alert quando il pulsante viene cliccato:

:::interactive_editor

```html
<button id="btn">Show alert</button>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("btn");

btn.addEventListener("click", () => alert("You clicked the button"));
```

:::

Ora ogni volta che il pulsante viene cliccato, l’utente vedrà un messaggio di alert mostrato sullo schermo così:

```md
You clicked the button
```

Un altro tipo di evento che puoi ascoltare è l’evento `"input"`. Questo evento viene attivato quando il valore di un elemento input cambia.

Ecco un esempio:

```html
<input type="text" id="input" placeholder="Type something" />
```

Vogliamo ascoltare l’evento input su questo elemento input e registrare il valore dell’input nella console ogni volta che l’utente digita qualcosa.

Prima dobbiamo consultare quell’elemento `input` nel nostro codice JavaScript. Poi dobbiamo aggiungere un event listener per ascoltare l’evento input e registrare il valore dell’input nella console ogni volta che l’utente digita qualcosa:

:::interactive_editor

```html
<input type="text" id="input" placeholder="Type something" />
<script src="index.js"></script>
```

```js
const input = document.getElementById("input");

input.addEventListener("input", () => {
  console.log(input.value);
});
```

:::

Ci sono molti altri eventi che puoi ascoltare usando il metodo `addEventListener()`. Alcuni eventi comuni includono `mouseover`, `mouseout`, `keydown`, `keyup` e `submit`.

Nelle lezioni future, tratteremo più eventi e come usare il metodo `addEventListener()` per ascoltarli.
