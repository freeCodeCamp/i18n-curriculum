# --interactive--

Один із найцікавіших аспектів CSS — це використання псевдоелементів. У цьому контексті «псевдо» означає «не справжній», тому псевдоелементи — це віртуальні або синтетичні елементи, які безпосередньо не відповідають жодним реальним HTML-елементам. Вони дозволяють стилізувати конкретні частини елемента або вставляти вміст без додавання додаткового HTML.

Щоб застосувати псевдоелемент, приєднайте його до селектора оригінального елемента, використовуючи подвійний двокрапку (`::`). Зверніть увагу, що селектор може бути будь-якого типу, наприклад, селектор класу або ідентифікатора. Ось як виглядає базовий синтаксис псевдоелементів:

```css
selector::pseudo-element {
  property: value;
}
```

Ця подвійна двокрапка відрізняє псевдоелементи від псевдокласів, які використовують одинарну двокрапку.

Псевдоелементи дозволяють стилізувати конкретні частини вмісту елемента або вставляти вміст перед ним чи після нього, але вони не можуть існувати самостійно. Елемент, до якого приєднаний псевдоелемент, називається його елементом-джерелом.

Почнемо з прикладів для псевдоелементів `::before` та `::after`. Як випливає з їхніх назв, `::before` дозволяє вставити вміст безпосередньо перед вмістом елемента, а `::after` — після нього.

Ось приклад елемента кнопки. У CSS ми використаємо абсолютне позиціонування та псевдоелемент `::before`, щоб додати зірочку перед текстом `Learn More` кнопки. Ви дізнаєтеся більше про абсолютне позиціонування в майбутніх уроках.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: lightseagreen;
  color: white;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
  position: relative;
}

.cta-button::before {
  content: "⭐";
  position: absolute;
  left: 3px;
  top: 8px;
  font-size: 0.75rem;
}
```

:::

Властивість `content` використовується для представлення вмісту, який ви хочете додати перед текстом кнопки. У цьому прикладі ми додаємо зірочку. Ви помітите, що можна не лише вставляти вміст, а й стилізувати його. Ось приклад псевдоелемента `::after` з тією ж кнопкою:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: '➡️';
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}
```

:::

У псевдоелементі `::after` властивість `transition` використовується для анімації змін протягом 0,3 секунди з ефектом згладжування, створюючи плавну та поступову трансформацію замість раптової. Ви дізнаєтеся більше про властивість `transition` у майбутніх уроках.

Ви також можете приєднати псевдоклас до вмісту, який вставляєте за допомогою псевдоелементів `::before` та `::after`. Наприклад, стан наведення для вмісту. Ось приклад:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: "➡️";
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}

.cta-button:hover::after {
  transform: translateX(2px);
}
```

:::

З `transform: translateX(2px)` у стані наведення вміст зсувається вправо на `2px` щоразу, коли користувач наводить курсор на кнопку. Властивість transition у самому `::after` забезпечує, що цей процес триває `0.3s`.

Саме це робить властивість `transform` — вона дозволяє обертати, нахиляти, масштабувати або переміщувати елемент у певному напрямку. Ви дізнаєтеся більше про це в майбутніх уроках.

У наступному прикладі ми розглянемо псевдоелемент `::first-letter`. Псевдоелемент `::first-letter` націлюється на першу літеру вмісту елемента, дозволяючи її стилізувати. Ось приклад тексту абзацу. Якщо ми хочемо стилізувати першу літеру, можемо використати псевдоелемент `::first-letter` так:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>freeCodeCamp lets you learn to code without having to pay.</p>
```

```css
p::first-letter {
  font-size: 4rem;
}
```

:::

У останньому прикладі ми розглянемо псевдоелемент `::marker`, який дозволяє вибрати маркер, булет або нумерацію елементів списку для стилізації. Псевдоелемент `::marker` дає змогу покращити бренд вашого сайту, кастомізуючи маркери списку відповідно до вашої колірної схеми.

Ось приклад невпорядкованого списку та впорядкованого списку. Щоб змінити колір і розмір маркера елемента списку, можна використати псевдоелемент `::marker` так:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<ul>
  <li>Unordered list item 1</li>
  <li>Unordered list item 2</li>
  <li>Unordered list item 3</li>
  <li>Unordered list item 4</li>
</ul>

<ol>
  <li>Ordered list item 1</li>
  <li>Ordered list item 2</li>
  <li>Ordered list item 3</li>
  <li>Ordered list item 4</li>
</ol>
```

```css
li::marker {
  color: crimson;
  font-size: 1.5em;
  font-weight: bold;
}
```

:::

У цьому уроці ми розглянули лише кілька псевдоелементів. Але існує багато інших, як `::placeholder`, `::spelling-error` та `::selection`, які я рекомендую вам дослідити самостійно.
