# --interactive--

عند العمل مع السلاسل النصية في JavaScript، تحتاج غالبًا إلى استخراج جزء أو سلسلة فرعية من سلسلة أكبر.

على سبيل المثال، قد ترغب في استخراج جزء من كلمة، أو متتالية أحرف محددة، أو مجرد مقطع من جملة.

توفر JavaScript عدة طرق لهذه المهمة، وأشهرها طريقة `slice()`.

تسمح لك طريقة `slice()` باستخراج جزء من سلسلة نصية وتُرجع سلسلة جديدة، دون تعديل السلسلة الأصلية. تأخذ معلمتين: فهرس البداية والفهرس النهائي الاختياري.

إليك الصيغة النحوية الأساسية:

```js
string.slice(startIndex, endIndex);
```

`startIndex` هو الموضع الذي يبدأ منه الاستخراج. `endIndex` هو الموضع الذي ينتهي عنده الاستخراج. إذا لم يُحدد، تستخرج `slice()` حتى نهاية السلسلة.

لننظر إلى مثال بسيط لاستخراج جزء من سلسلة نصية:

:::interactive_editor

```js
let message = "Hello, world!";
let greeting = message.slice(0, 5);

console.log(greeting);  // Hello
```

:::

في هذا المثال، `slice(0, 5)` يستخرج الأحرف بدءًا من الفهرس `0` وحتى قبل الفهرس `5`. والنتيجة هي استخراج الكلمة `Hello`.

إذا حذفت المعلمة الثانية، ستستخرج `slice()` كل شيء من فهرس البداية حتى نهاية السلسلة:

:::interactive_editor

```js
let message = "Hello, world!";
let world = message.slice(7);

console.log(world);  // world!
```

:::

هنا، `slice(7)` يستخرج السلسلة من الفهرس `7` حتى نهاية السلسلة، والنتيجة هي `world!`.

يمكنك أيضًا استخدام أرقام سالبة كفهرس. عند استخدام رقم سالب، يُحسب من نهاية السلسلة إلى الوراء:

:::interactive_editor

```js
let message = "JavaScript is fun!";
let lastWord = message.slice(-4);

console.log(lastWord);  // fun!
```

:::

في هذه الحالة، `slice(-4)` يستخرج آخر أربعة أحرف من السلسلة، مما يعطينا `fun!`.

لنفترض أنك تريد استخراج قسم من منتصف السلسلة. يمكنك تحديد فهارس البداية والنهاية للتحكم بدقة في الجزء الذي تريد استخراجه:

:::interactive_editor

```js
let message = "I love JavaScript!";
let language = message.slice(7, 17);

console.log(language);  // JavaScript
```

:::

هنا، `slice(7, 17)` يستخرج السلسلة الفرعية بدءًا من الفهرس 7 وحتى قبل الفهرس `17`، وهي الكلمة `JavaScript`.

طريقة `slice()` أداة قوية لاستخراج أجزاء من سلسلة نصية في JavaScript.

تحدد فهارس البداية والنهاية، وتُرجع الطريقة سلسلة جديدة تحتوي الجزء المستخرج.

مع خيارات الفهارس الموجبة والسالبة والمحفوفة، يمكنك تكييفها مع مواقف مختلفة دون تغيير السلسلة الأصلية.
