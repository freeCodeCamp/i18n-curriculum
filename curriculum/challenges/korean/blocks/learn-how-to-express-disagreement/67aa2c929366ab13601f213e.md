# --explanation--

`Productivity`는 일정 시간 내에 사람이 완료할 수 있는 작업량을 의미합니다. 더 나은 도구(예: 업그레이드된 컴퓨터)는 직원들이 더 효율적으로 일하도록 도울 수 있습니다. 예를 들어:

- `Using automation tools can boost productivity.` - 작업 자동화는 사람들이 더 효율적으로 일하고 더 많은 일을 완료하도록 돕습니다.

- `Distractions reduce productivity when working remotely.` - 방해 요소는 집중하기 어렵게 만들고 작업을 효과적으로 완료하는 데 방해가 됩니다.
