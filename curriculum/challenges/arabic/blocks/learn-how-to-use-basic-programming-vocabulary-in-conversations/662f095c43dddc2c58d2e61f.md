# --description--

تُستخدم كلمة `super` غالبًا بشكل غير رسمي قبل الصفة لتقوية معناها، مثل `very` أو `extremely`، لتعزيز معنى الصفة التي تليها.

على سبيل المثال، قولك `this book is super interesting` يعني أن الكتاب مثير للاهتمام للغاية، أكثر من مجرد كونه مثيرًا للاهتمام.

`Helpful` تصف شخصًا أو شيئًا يقدم مساعدة أو دعمًا ذا قيمة. معًا، تعني `super helpful` تقديم مساعدة استثنائية أو أن تكون مفيدًا للغاية.
