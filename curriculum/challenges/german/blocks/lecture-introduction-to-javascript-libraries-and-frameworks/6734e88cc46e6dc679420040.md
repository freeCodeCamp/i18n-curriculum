# --description--

Im Gegensatz zur Arbeit mit kleineren Vanilla-HTML-, CSS- und JavaScript-Projekten umfasst das Starten eines neuen React-Projekts einige zusätzliche Schritte, um sicherzustellen, dass alles korrekt läuft. Anstatt zu versuchen, alle notwendigen Konfigurationen selbst einzurichten, gibt es Tools, die das für Sie übernehmen.

Eines der beliebtesten Tools zur Einrichtung von Projekten ist Vite. Vite, was auf Französisch „schnell“ bedeutet, ist ein Build-Tool, das darauf abzielt, eine schnellere Entwicklungsumgebung für moderne Webprojekte bereitzustellen. Vite kann mit React sowie mit anderen Bibliotheken und Frameworks wie Vue oder Svelte verwendet werden. Sie können es sogar mit Vanilla-JavaScript-Projekten nutzen.

Um ein neues Projekt mit Vite zu erstellen, müssen Sie die Kommandozeile verwenden. Wenn Sie einen Windows-Rechner nutzen, können Sie die Eingabeaufforderung oder Windows PowerShell verwenden. Wenn Sie einen Mac verwenden, können Sie die Terminal-App nutzen.

Falls Sie Node.js nicht installiert haben, was für npm erforderlich ist, finden Sie hier einen Link zur offiziellen Node.js-Webseite: https://nodejs.org. Folgen Sie den Installationsanweisungen und kehren Sie dann hierher zurück, um mit der React-Installation fortzufahren.

Sobald Sie die Kommandozeile geöffnet haben, können Sie den folgenden Befehl verwenden:

```bash
npm create vite@latest my-react-app -- --template react
```

Dieser Befehl erstellt ein neues React-Projekt namens `my-react-app` mit dem React-Template von Vite.

Sie können dann das neue Projekt öffnen und den React-Boilerplate-Code sehen, den Vite für Sie bereitgestellt hat.

Das Tolle an Vite ist, dass es nur die Dateien bereitstellt, die absolut notwendig sind, um mit Ihrem React-Projekt zu starten.

Um das Projekt tatsächlich so auszuführen, wie es ist, müssen Sie die Abhängigkeiten mit den folgenden Befehlen in der Kommandozeile installieren:

```bash
cd my-react-app
npm install
```

Der `cd my-react-app`-Befehl stellt sicher, dass Sie in das richtige Verzeichnis wechseln, in dem sich Ihr Projekt befindet.

Der `npm install`-Befehl wird verwendet, um die Abhängigkeiten zu installieren, die benötigt werden, damit Ihr React-Projekt ordnungsgemäß läuft. Abhängigkeiten sind Bibliotheken oder Pakete, die Ihr React-Projekt benötigt, um korrekt zu funktionieren, wie React selbst, ReactDOM oder andere Drittanbieterpakete, die zusätzliche Funktionalität bereitstellen.

Das Ausführen von `npm install` liest die `package.json`-Datei in Ihrem Projektverzeichnis und installiert alle dort aufgeführten Abhängigkeiten, sodass Sie Ihre React-Anwendung ohne fehlende Komponenten bauen und ausführen können.

Die `package.json`-Datei ist eine wichtige Konfigurationsdatei in Projekten, die Metadaten über Ihr Projekt enthält, einschließlich Name, Version und Abhängigkeiten. Sie definiert auch Skripte, Lizenzinformationen und andere Einstellungen, die bei der Verwaltung des Projekts und seiner Abhängigkeiten helfen.

Sobald die Abhängigkeiten installiert sind, sollten Sie einen neuen Ordner in Ihrem Projekt namens `node_modules` bemerken.

Der `node_modules`-Ordner ist der Speicherort aller Pakete und Bibliotheken, die Ihr Projekt benötigt. Dieser Ordner enthält den eigentlichen Code für die in Ihrer `package.json`-Datei aufgeführten Abhängigkeiten, einschließlich der direkten Abhängigkeiten Ihres Projekts und aller Abhängigkeiten dieser Abhängigkeiten.

Um Ihr Projekt auszuführen, führen Sie den `npm run dev`-Befehl aus und öffnen Sie einen neuen Browser-Tab unter `http://localhost:5173/`.

Sie sollten die Starter-Vorlage sehen, die Vite für Sie bereitgestellt hat.

Wenn Sie den lokalen Server stoppen müssen, drücken Sie `CTRL + C` in der Kommandozeile.

Um den Code für diese Starter-Vorlage tatsächlich zu sehen, können Sie in Ihrem Projekt in den `src`-Ordner gehen, und Sie sollten die `App.jsx`-Datei sehen.

Von dort aus können Sie die Datei bearbeiten, Ihre Änderungen speichern und die neuen Änderungen im Browser sehen.

Damit sind Sie bereit, Ihre React-Anwendung zu entwickeln!
