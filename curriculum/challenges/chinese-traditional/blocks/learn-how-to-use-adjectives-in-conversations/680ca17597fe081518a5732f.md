# --explanation--

`Last time` 指的是最近的事件。 例如：

`Last time we updated the server, we had some downtime.` – 這顯示在先前的更新期間。

`Order` 是正式請求貨物或設備的意思。它的過去式是 `ordered`。例如：

`The IT team ordered new laptops for the developers.` – 這表示他們已經為筆記型電腦提出了請求。

`Double that` 意味著前一數量的兩倍。 例如：

`Last time we ordered 10 routers; this time we need double that, so 20 routers.`－這表示數量增加了兩倍。
