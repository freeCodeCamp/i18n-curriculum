# --description--

在上一节课中，你创建了一个项目文件夹并安装了 Express：

```bash
mkdir my-app
cd my-app
npm init -y
npm install express
```

作为提醒，这些命令创建了一个新的项目文件夹，`cd` 进入项目文件夹，初始化项目并创建了一个基本的 `package.json` 文件，同时安装了 Express。

接下来，打开一个代码编辑器，创建一个名为 `index.js` 的文件，并编写你的第一个 Express 服务器：

```javascript
const express = require("express");
const app = express();
const port = 3000;

app.get("/", (req, res) => {
  res.send("Hello World!");
});

app.listen(port, () => {
  console.log(`Example app listening on port ${port}`);
});
```

你可以在终端中使用以下命令运行你的代码：

```bash
node index.js
```

打开浏览器访问 [http://localhost:3000](http://localhost:3000)，你将看到消息 **"Hello World!"**

这就是使用 Express 快速入门的方式！现在让我们逐步讲解这段代码。

在第一行代码中，你正在装载 Express 库并赋值给一个名为 `express` 的变量：

```javascript
const express = require("express");
```

`express` 变量是一个工厂函数。当你调用它时，它会创建并返回一个新的 Express 应用实例：

```javascript
const app = express();
```

`app` 现在可以访问 Express 提供的所有方法，比如 `.get()`、`.post()`、`.listen()` 等等。这是你的主要服务器对象。

下一行定义了 `port` 变量。端口是你的应用监听的网络端点，`3000` 是本地开发中常用的端口：

```javascript
const port = 3000;
```

现在，让我们看看接下来的部分：

```javascript
app.get("/", (req, res) => {
  res.send("Hello World!");
});
```

这段代码定义了一个路由，并告诉服务器如何响应请求。`app.get()` 为 HTTP GET 请求注册了一个处理程序。GET 请求是常用的 HTTP 方法，用于获取数据。

`"/"` 参数在 `app.get()` 中表示路由路径。在这个例子中，它指向应用的根 URL。

`(req, res) => { ... }` 部分是一个回调函数，当请求匹配此路由时运行。`req` 对象包含关于传入请求的信息，`res` 对象用于向客户端发送响应。

最后，`res.send("Hello World!")` 向客户端发送响应。在这个例子中，响应是纯文本 `"Hello World!"`。

现在，让我们看看最后一段代码：

```javascript
app.listen(port, () => {
  console.log(`Example app listening on port ${port}`);
});
```

`app.listen()` 方法启动 Express 应用，并告诉 Node 在指定端口（这里是 `3000`）监听传入的 HTTP 请求。

服务器启动后，回调函数执行，消息 `Example app listening on port 3000` 会被记录到控制台，确认 Express 应用已成功启动并准备好接收请求。
