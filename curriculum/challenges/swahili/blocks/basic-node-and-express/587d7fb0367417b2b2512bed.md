# --description--

Kufanya kazi kwenye changamoto hizi kutahusisha kuandika msimbo wako kwa kutumia mojawapo ya njia zifuatazo:

- Nakili <a href="https://github.com/freeCodeCamp/boilerplate-express/" target="_blank" rel="noopener noreferrer nofollow">hii repo ya GitHub</a> na ukamilishe changamoto hizi kwa ndani ya kompyuta yako.
- Tumia mtengenezaji wa tovuti unayempenda kukamilisha mradi. Hakikisha unajumuisha mafaili yote kutoka kwenye repo yetu ya GitHub.

Wakati wa mchakato wa ukuzaji, ni muhimu kuwa na uwezo wa kuangalia kinachoendelea katika msimbo wako.

Node ni mazingira ya JavaScript tu. Kama JavaScript upande wa mteja, unaweza kutumia konsoli kuonyesha taarifa muhimu za kutafuta makosa. Kwenye kompyuta yako ya ndani, utaona matokeo ya konsoli kwenye terminali.

Tunapendekeza uweke terminali wazi wakati unafanya kazi kwenye changamoto hizi. Kwa kusoma matokeo kwenye terminali, utaona hitilafu zozote zinazoweza kutokea.

Seva lazima ianzishwe upya baada ya kufanya mabadiliko kwenye mafaili yake.

Unaweza kuacha seva kutoka kwenye terminali kwa kutumia `Ctrl + C` na kuianzisha moja kwa moja kwa kutumia Node (`node mainEntryFile.js`) au kwa kutumia script ya kuendesha kwenye faili la `package.json` kwa `npm run`.

Kwa mfano, script ya `"start": "node server.js"` itaendeshwa kutoka kwenye terminali kwa kutumia `npm run start`.

Ili kutekeleza kuanzisha upya seva moja kwa moja wakati faili inahifadhiwa, Node hutoa bendera ya `--watch` ambayo unaweza kuongeza kwenye script yako ya kuanzisha `"start": "node --watch server.js"` au unaweza kusakinisha kifurushi cha npm kama `nodemon`. Tutakuachia hii kama zoezi kwako.

# --instructions--

Badilisha faili la `myApp.js` ili kuandika "Hello World" kwenye konsoli.
