# --description--

ここ数レッスンで、次のようなループの使い方に慣れてきました。

```py
even_numbers = []

for num in range(21):
    if num % 2 == 0:
        even_numbers.append(num)

print(even_numbers)
```

この例では、`even_numbers` という新しい空のリストを作成し、`0` から `20` までの数値のシーケンスをループしています。ループ内では、現在の数値を `0` で割ったときの余りが `2` かどうかを条件でチェックしています。これは数値が偶数かどうかを判定するためです。条件が `True` の場合、現在の `num` が `even_numbers` リストの末尾に追加されます。最後に、`even_numbers` リストをコンソールに出力しています。

このコードは動作しますが、リスト内包表記を使うともっと簡潔に書けます。リスト内包表記は、ループと条件を大括弧の中に直接組み合わせて1行で新しいリストを作成できる方法です。これによりコードが短くなり、読みやすくなることが多いです。

先ほどの例を大括弧を使って書き換えたものがこちらです。

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

この書き換え例では、`even_numbers` リストが1行のコードで作成されています。リスト内包表記は `0` から `20` までの数値をループし、`2` で割り切れる数だけを含めています。この方法はよりコンパクトで、別のループや条件ブロックが不要になります。

リスト内包表記の仕組みをよりよく理解するために、別の例を見てみましょう。

```py
numbers = [1, 2, 3, 4, 5]
result = [(num, 'Even') if num % 2 == 0 else (num, 'Odd') for num in numbers]
print(result)
```

この例では、数値のリストがあり、偶数か奇数かを示すタプルの新しいリストを作成したいです。リスト内包表記の最初の部分で、`if` 文を使って数値が `2` で割り切れるかをチェックしています。割り切れる場合は、その数値と `Even` という単語を含むタプルが結果になります。そうでなければ、その数値と `Odd` という単語を含むタプルになります。

コンソールに出力した結果は次のようになります。

```py
[(1, 'Odd'), (2, 'Even'), (3, 'Odd'), (4, 'Even'), (5, 'Odd')]
```

既存のイテラブルからリストを作成する別の方法は `filter()` 関数です。こちらは4文字より長い単語だけを集めた新しいリストを作る例です。

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

`filter()` 関数は、特定の条件を満たすイテラブルの要素を選択するために使います。`filter()` 関数は関数とイテラブルを引数に取ります。この例では、`is_long_word` 関数を `filter()` 関数に渡して、現在の単語の文字数が `4` より大きいかをチェックしています。文字数が `4` より大きい単語だけが新しいリストに追加され、`long_words` 変数に代入されます。

`filter()` 関数以外にも、リスト操作に便利な関数がいくつかあります。もう一つ知っておきたいのは `map()` 関数で、これはイテラブルの各要素に関数を適用します。こちらは `map()` 関数を使って摂氏の温度リストを華氏に変換する例です。

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

`filter()` 関数と同様に、`map()` 関数は関数とイテラブルを引数に取ります。`to_fahrenheit` 関数は温度を摂氏から華氏に変換します。

最後に紹介するのは `sum()` 関数です。この関数はリストやタプルのようなイテラブルの合計を取得するために使います。こちらは `sum()` 関数の使用例です。

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

オプションの `start` 引数を渡すこともでき、これは合計の初期値を設定します。こちらは `start` 引数を位置引数として使った更新例です。

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

また、`start` 引数をキーワード引数として使うこともできます。

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

どちらの方法でも同じ結果になりますが、キーワード引数のほうが少し明示的です。

リスト内包表記や `map()`、`filter()`、`sum()` のような関数は最初は少し混乱するかもしれません。でも練習と時間をかければ、Pythonプログラムで使いこなせるようになります。
