# --description--

檔案管理應用程式是一種能讓你輕鬆儲存、組織及編輯你在電腦上的檔案的方式。無論你使用 Mac 或 Windows 電腦，都有內建的應用程式可以用來組織你的檔案。

如果你有 Windows 電腦，預設的檔案管理員是 File Explorer。它用於瀏覽、搜尋和管理檔案及資料夾，以及執行複製、移動和刪除等檔案操作。

要找到檔案總管，你可以前往開始選單或按下鍵盤上的 Windows 標誌鍵。

要固定資料夾，你可以用滑鼠右鍵點擊並選擇「Pin」。要移動檔案或資料夾，你首先需要選取它，然後選擇「Cut」，接著導航到新位置並貼上。

如果你有 Mac 電腦，預設的檔案管理員是 Finder。它提供存取區域的和遠端檔案、支援檔案預覽，並提供多種組織特性，例如頁籤和智慧型資料夾。

若要存取 Mac 上的 Finder，請前往底部的 Dock 並點擊 Finder 圖示。

你會注意到左側有一個側邊欄，裡面放著你最喜愛的資料夾。

組織你的檔案的一種方法是使用標記。要為檔案或資料夾加上標記，你可以在 Finder 中選取該項目，然後用滑鼠右鍵點擊。你會在下拉選單中看到「Tags...」選項，並且可以套用訂製的彩色標記。

要搜尋標記的項目，你可以捲動到側邊欄底部並點擊對應顏色的標記，或者你可以在搜尋欄位中依顏色名稱搜尋該標記。

另一種組織你的檔案的方法是使用智慧資料夾。智慧資料夾是根據你設定的條件（例如檔案型別、日期或關鍵字）自動組織的檔案群集。

無論你是在 Windows 上使用 File Explorer 還是在 Mac 上使用 Finder，這些應用程式都提供彈性，讓你能夠透過像是標記、搜尋和智慧資料夾等特性來管理你的檔案。
