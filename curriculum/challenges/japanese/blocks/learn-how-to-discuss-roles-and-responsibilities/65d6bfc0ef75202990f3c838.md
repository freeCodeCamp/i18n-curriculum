# --description--

`Smoothly` は、何かが問題やストップなしに行われる様子を表す副詞です。これは、平らで均一な表面を持つ、または突然の変化や困難がないことを意味する形容詞 `smooth` に由来します。

例えば、`The table is smooth` と言うと、表面が平らで均一であることを指します。`The meeting ran smoothly` と言うと、会議が問題やストップなしに行われたことを意味します。
