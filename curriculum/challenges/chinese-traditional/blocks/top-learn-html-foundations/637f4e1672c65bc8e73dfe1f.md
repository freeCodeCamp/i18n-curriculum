# --description--

你可以將元素視為內容的容器。開啟和關閉的標籤告訴瀏覽器該元素包含什麼內容。瀏覽器接著可以使用該資訊來判斷應該如何解讀和格式化內容。

有些 HTML 元素沒有結束標籤。這些元素通常看起來像 `<br />` 或 `<img/>`，但有些也可以不用結束斜線，例如 `<br>` 或 `<img>`。這些被稱為空元素或空零件，因為它們不包裹任何內容。你會在後續課程中遇到一些這樣的元素，但大多數零件都會有開啟和結束標籤。

HTML 有大量預先定義的標記，你可以用來創建各種不同的元素。使用正確的標記對內容非常重要。使用正確的標記會對你網站的兩個方面產生重大影響：它們在搜尋引擎中的等級；以及對依賴輔助技術（例如螢幕閱讀器）來存取網際網路的使用者的可存取性。

使用正確的元素來呈現內容稱為語意化 HTML。你將在課程的後續部分更深入地探討這個主題。
