# --description--

如果我們列出所有小於 10 且是 3 或 5 的倍數的自然數，我們會得到 3、5、6 和 9。這些倍數的總和是 23。

找出所有小於提供的參數值 `number` 的 3 或 5 的倍數之和。

# --solutions--

```js
const multiplesOf3Or5 = (number) => {
  var total = 0;

  for(var i = 0; i < number; i++) {
    if(i % 3 == 0 || i % 5 == 0) {
      total += i;
    }
  }
  return total;
};
```
