# --description--

Hay varias formas de escribir el número $\frac{1}{2}$ como una suma de cuadrados inversos usando enteros distintos.

Por ejemplo, los números {2,3,4,5,7,12,15,20,28,35} pueden ser usados:

$$\frac{1}{2} = \frac{1}{2^2} + \frac{1}{3^2} + \frac{1}{4^2} + \frac{1}{5^2} + \frac{1}{7^2} + \frac{1}{{12}^2} + \frac{1}{{15}^2} + \frac{1}{{20}^2} + \frac{1}{{28}^2} + \frac{1}{{35}^2}$$

De hecho, usando solo enteros entre 2 y 45 inclusive, hay exactamente tres formas de hacerlo, las otras dos siendo: {2,3,4,6,7,9,10,20,28,35,36,45} y {2,3,4,6,7,9,12,15,28,30,35,36,45}.

¿Cuántas formas hay de escribir el número $\frac{1}{2}$ como una suma de cuadrados inversos usando enteros distintos entre 2 y 80 inclusive?
