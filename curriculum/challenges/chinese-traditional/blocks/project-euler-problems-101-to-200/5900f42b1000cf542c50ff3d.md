# --description--

設 $S_m = (x_1, x_2, \ldots, x_m)$ 為一組 $m$ 個正實數的 $m$ 元組，且滿足 $x_1 + x_2 + \cdots + x_m = m$，使得 $P_m = x_1 \times {x_2}^2 \times \cdots \times {x_m}^m$ 達到最大值。

例如，可以驗證 $[P_{10}] = 4112$（[ ] 是整數部分函式）。

求 $\sum {[P_m]}$，其中 $2 ≤ m ≤ 15$。
