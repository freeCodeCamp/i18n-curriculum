# --description--

モバイル端末で中国語カリキュラムを進める場合、freeCodeCampのカスタム入力を使ってピンインや中国語の文字を直接入力できます。しかし、端末の中国語キーボードを使いたい場合は、以下の手順で設定してください。

## 言語キーボードの追加  

### Androidの場合

**注意:** Androidの設定手順は端末によって異なる場合があります。

1. 設定を開きます。

2. システム（または端末によっては一般管理）に進みます。

3. キーボード設定（または言語と入力）をタップします。  

4. 言語とタイプ（またはキーボードを追加）をタップします。 
 
5. 中国語（簡体字またはCN）をオンにするか、必要に応じてダウンロードします。  

6. 保存するか、表示されたら完了をタップします。

### iPhoneの場合

1. 設定を開きます。

2. 一般 > キーボードに進みます。

3. キーボードをタップします。

4. 新しいキーボードを追加をタップします。  

5. 中国語（簡体字）を選択します。  

6. キーボードの地球儀アイコンで言語を切り替えます。

## サードパーティ製キーボードの利用  

スマホやタブレットにサードパーティ製のキーボードアプリをインストールして、中国語入力をより簡単にすることもできます。これらのキーボードは、スマートな文字予測やピンインや手書きなど複数の入力方式のサポートなど便利な機能が付いていることが多いです。
