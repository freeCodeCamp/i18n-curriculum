# --instructions--

Erstellen Sie ein sicheres Multiplayer-Spiel, in dem jeder Spieler seinen Avatar bewegen kann, mindestens ein sammelbares Objekt vorhanden ist und der Rang der Spieler basierend auf ihrer Punktzahl berechnet wird.

Für Details konsultieren Sie die untenstehenden Tests.

Stellen Sie sicher, dass Ihr Spiel sicher ist! Fügen Sie diese Sicherheitsmaßnahmen ein:

- Der Client darf den MIME-Typ nicht erraten oder ausspähen können
- Verhindern Sie XSS-Angriffe
- Es darf nichts von der Website im Client zwischengespeichert werden
- Die Kopfzeilen geben an, dass die Seite von `PHP 7.4.3` betrieben wird

**Hinweis**: `helmet@^3.21.3` wird für die User Stories benötigt. Das bedeutet, Sie müssen die vorherige Version der Helmet-Dokumentation verwenden, um Informationen zu erhalten, wie Sie die User Stories umsetzen.
