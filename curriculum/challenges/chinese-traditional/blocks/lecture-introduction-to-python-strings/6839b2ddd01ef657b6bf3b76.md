# --description--

Python 提供了許多內建的方法，使操作字串變得輕而易舉。它們包括但不限於以下內容：

- `upper()`: 傳回一個所有字元皆轉換為大寫的新字串。
    

```python
my_str = 'hello world'

uppercase_my_str = my_str.upper()
print(uppercase_my_str)  # HELLO WORLD
```

- `lower()`：傳回一個所有字元皆轉換為小寫的新字串。
    

```python
my_str = 'Hello World'

lowercase_my_str = my_str.lower()
print(lowercase_my_str)  # hello world
```

- `strip()`: 傳回一個新的字串，移除指定的前置和後置字元。如果沒有傳入引數，則移除前置和後置的空白字元。
    

```python
my_str = '  hello world  '

trimmed_my_str = my_str.strip()
print(trimmed_my_str)  # "hello world"
```

- `replace(old, new)`：傳回一個新的字串，所有出現的 `old` 都被 `new` 取代。
    

```python
my_str = 'hello world'

replaced_my_str = my_str.replace('hello', 'hi')
print(replaced_my_str)  # hi world
```

- `split(separator)`：將字串依指定的分隔符拆分成字串列表。如果未指定分隔符，則依空白字元拆分。
    

```python
my_str = 'hello world'

split_words = my_str.split()
print(split_words)  # ['hello', 'world']
```

- `join(iterable)`：將可迭代物件的元素以分隔符串接成字串。
    

```python
my_list = ['hello', 'world']

joined_my_str = ' '.join(my_list)
print(joined_my_str)  # hello world
```

- `startswith(prefix)`：傳回一個布林值，表示字串是否以指定的前置式開始。
    

```python
my_str = 'hello world'

starts_with_hello = my_str.startswith('hello')
print(starts_with_hello)  # True
```

- `endswith(suffix)`：傳回一個布林值，表示字串是否以指定的後綴結尾。
    

```python
my_str = 'hello world'

ends_with_world = my_str.endswith('world')
print(ends_with_world)  # True
```

- `find(substring)`：傳回 `substring` 第一次出現的位置索引，若找不到則傳回 `-1`。
    

```python
my_str = 'hello world'

world_index = my_str.find('world')
print(world_index)  # 6
```

- `count(substring)`：傳回子字串在字串中出現的次數。
    

```python
my_str = 'hello world'

o_count = my_str.count('o')
print(o_count)  # 2
```

- `capitalize()`: 傳回一個新的字串，首個字元為大寫，其餘字元為小寫。
    

```python
my_str = 'hello world'

capitalized_my_str = my_str.capitalize()
print(capitalized_my_str)  # Hello world
```

- `isupper()`：如果字串中所有字母都是大寫，則傳回 `True`，否則傳回 `False`。
    

```python
my_str = 'hello world'

is_all_upper = my_str.isupper()
print(is_all_upper)  # False
```

- `islower()`：如果字串中所有字母都是小寫，則傳回 `True`，否則傳回 `False`。
    

```python
my_str = 'hello world'

is_all_lower = my_str.islower()
print(is_all_lower)  # True
```

- `title()`：傳回一個新字串，其中每個字的第一個字母皆為大寫。
    

```python
my_str = 'hello world'

title_case_my_str = my_str.title()
print(title_case_my_str)  # Hello World
```
