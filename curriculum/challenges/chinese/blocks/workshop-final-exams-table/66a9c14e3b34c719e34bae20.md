# --description--

要在表格中添加学生数据，需要使用表格行和表格数据元素。

<dfn>表格数据</dfn>元素 `td`，用于创建表格中的单元格。

下面是一个将 `td` 元素用于体育运动员表格的示例：

```html
<tr>
  <td>1</td>
  <td>John Doe</td>
  <td>USA</td>
</tr>
```

在表体元素中，添加一个表格行元素 `tr` 和三个表格数据元素 `td`。

第一个表格数据元素应包含姓氏 `Davis`。

第二个表格数据元素应包含名字 `Alex`。

第三个表格数据元素应包含分数 `54`。



你应该在 `tr` 元素中包含一个文本为 `54` 的 `td` 元素。

```js
assert.strictEqual(document.querySelectorAll('tbody tr td')[2]?.textContent, '54');
```
