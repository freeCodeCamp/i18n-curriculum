# --description--

`equals symbol`（`=`）通常用於指派。它用來將值指定給變數。變數名稱寫在 `equals symbol` 的左邊，而你想要儲存在變數中的值則寫在右邊。

例如，`name = "Sophie";` 將字串 `"Sophie"` 指定給變數 `name`。

在程式設計中，`assign a value` 意味著將特定的值指定給一個變數。`equals symbol`（`=`）通常用於此目的。當你為變數指定值時，你是在告訴程式記住該值以便稍後使用。

例如，`age = 25` 將值 `25` 指定給變數 `age`。
