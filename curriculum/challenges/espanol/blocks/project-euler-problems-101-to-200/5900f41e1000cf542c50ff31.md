# --description--

Considera el número 45656.

Se puede ver que cada par de dígitos consecutivos de 45656 tiene una diferencia de uno.

Un número para el cual cada par de dígitos consecutivos tiene una diferencia de uno se llama número escalonado.

Un número pandigital contiene cada dígito decimal del 0 al 9 al menos una vez.

¿Cuántos números pandigitales escalonados menores que ${10}^{40}$ hay?
