# --description--

網頁瀏覽器是你用來瀏覽網際網路頁面的應用程式。如果你正在學習 freeCodeCamp 課程，很有可能你現在正在使用網頁瀏覽器！

截至 2024 年，最受歡迎的瀏覽器是 Microsoft Edge、Firefox、Google Chrome 和 Safari。還有相當多其他選項，且市場佔有率持續變動，但這四個目前是最常見的。

你的作業系統很可能預設安裝了瀏覽器。對於 Windows，你會有 Microsoft Edge。對於 macOS，你會有 Safari。對於 Linux，你可能會有 Firefox。

但是如果你想使用不同的瀏覽器呢？

好消息是你已經有一個預設瀏覽器。所以你可以瀏覽到你想要的瀏覽器的網頁，並下載適用於你的作業系統的安裝程式。

在選擇瀏覽器時，請考慮速度、安全特性以及與你經常造訪的網站的相容性等因素。

某些瀏覽器提供獨特的特性。例如，Opera 內建 VPN，而 Brave 則專注於隱私和廣告阻擋。

安裝通常很簡單。在下載安裝程式後，只要執行它並按照螢幕上的指示操作即可。

大多數瀏覽器會提供從你目前的預設瀏覽器匯入書籤和設定，讓過渡更輕鬆。

或者，你可以透過你的作業系統的軟體包管理員安裝瀏覽器，例如在 macOS 使用 Homebrew 時執行 `brew install --cask google-chrome`，或在 Arch Linux 執行 `yay -S brave-bin`。

可用於你的軟體包管理員和軟體包名稱會依據你的作業系統而有所不同。

請記住，你不只限於使用一個瀏覽器。許多人會根據不同用途使用不同的瀏覽器－或許一個用於工作，另一個用於個人瀏覽。

歡迎嘗試不同的瀏覽器，以找出最適合你的。
