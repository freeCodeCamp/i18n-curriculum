# --description--

Python의 특수 메서드, 일명 "매직 메서드" 또는 "더블 언더 메서드"는 양쪽에 이중 밑줄(`__`)이 붙은 특별한 Python 메서드입니다. "dunder"라는 단어 자체는 double(이중)의 d와 underscore(밑줄)의 under에서 왔습니다.

아마도 이미 특수 메서드를 사용했지만 몰랐을 수도 있습니다. `3 + 4` 같은 코드를 작성할 때마다 Python은 내부적으로 조용히 `3.__add__(4)`를 실행합니다. 이것이 특수 메서드가 작동하는 방식입니다. 그래서 특수 메서드를 직접 호출할 수는 있지만, 거의 그렇게 하지 않습니다. `3 + 4` 같은 표현이 `3.__add__(4)`를 직접 호출하는 것보다 훨씬 명확하고 읽기 쉽습니다.

`__add__` 외에도 `__init__()`는 클래스 초기화 메서드로서 가장 자주 보고 사용하는 특수 메서드 중 하나입니다. `__len__()`와 `__str__()` 같은 다른 메서드도 있습니다.

특수 메서드는 프로그래밍하는 사람과 Python 인터프리터 사이의 활동을 조율하는 감독자라고 생각하세요.

특수 메서드를 직접 호출할 필요는 없습니다. 특정 동작이 발생할 때 Python이 자동으로 호출합니다. 이런 동작에는 다음이 포함됩니다:

- **덧셈, 뺄셈, 곱셈, 나눗셈 등 산술 연산**. 예를 들어 덧셈에는 `__add__()`, 뺄셈에는 `__sub__()`, 곱셈에는 `__mul__()`, 나눗셈에는 `__truediv__()`가 호출됩니다.
    
- **문자열 연결, 반복, 포맷팅, 텍스트 변환 같은 문자열 연산**. 연결에는 `__add__()`, 반복에는 `__mul__()`, 포맷팅에는 `__format__()`, 텍스트 변환에는 `__str__()`와 `__repr__()` 등이 호출됩니다.
    
- **동등성, 작음, 큼 등 비교 연산**. 동등성 검사에는 `__eq__()`, 작음에는 `__lt__()`, 큼에는 `__gt__()` 등이 호출됩니다.
    
- **객체를 반복 가능한 상태로 만들고 항목을 순회하는 반복 연산**. 반복자 반환에는 `__iter__()`, 다음 항목 가져오기는 `__next__()`가 호출됩니다.
    

일반적으로 문자열이나 숫자 같은 Python 내장 데이터 타입은 덧셈, 연결, 동등성 비교, 반복 등 기본 동작을 이미 알고 있습니다.

하지만 직접 클래스를 만들면 Python은 자동으로 처리하는 방법을 모릅니다.

이때 특수 메서드가 등장합니다 — Python 내장 동작을 사용자 정의할 수 있게 해줍니다.

아래 클래스에서 만든 책 객체의 페이지 수를 얻거나, 비교하거나, 객체를 읽기 쉬운 문자열로 나타내고 싶다고 가정해 보세요. 특수 메서드가 없으면 다음과 같은 일이 발생합니다:

```python
class Book:
   def __init__(self, title, pages):
       self.title = title
       self.pages = pages

book1 = Book("Built Wealth Like a Boss", 420)
book2 = Book("Be Your Own Start", 420)

print(len(book1)) # TypeError: object of type 'Book' has no len()
print(str(book1)) # <__main__.Book object at 0x102ed2900>
print(book1 == book2) # False even though they have the same number of pages
```

예제에서:

- `len(book1)`는 실패했는데, Python이 `__len__()` 없이는 책 객체의 길이를 알 수 없기 때문입니다.
    
- `str(book1)`는 `<__main__.Book object at 0x102ed2900>` 같은 기본 표현을 출력했는데, `__str__()`을 사용하지 않았기 때문입니다.
    
- `book1 == book2`는 `False` 결과를 냈는데, Python이 내용이 아니라 메모리 상에서 두 객체가 같은지 확인했기 때문입니다.
    

아래는 `__len__()` 클래스에서 만든 객체를 더 쉽게 다루기 위해 `__str__()`, `__eq__()`, `Book` 특수 메서드를 직접 정의하는 방법입니다:

```python
class Book:
   def __init__(self, title, pages):
       self.title = title
       self.pages = pages

   def __len__(self):
       return self.pages

   def __str__(self):
       return f"'{self.title}' has {self.pages} pages"

   def __eq__(self, other):
       return self.pages == other.pages
  
book1 = Book("Built Wealth Like a Boss", 420)
book2 = Book("Be Your Own Start", 420)

print(len(book1)) # 420
print(len(book2)) # 420
print(str(book1)) # 'Built Wealth Like a Boss' has 420 pages
print(str(book2)) # 'Be Your Own Start' has 420 pages
print(book1 == book2) # True
```

또 다른 예는 쇼핑 카트로, 다음과 같은 작업을 합니다:

- 카트에 아이템 추가
    
- 카트에서 아이템 제거
    
- 카트에 있는 아이템 수 확인
    
- 카트에 어떤 아이템이 있는지 확인
    
- 특정 아이템이 카트에 있는지 확인
    
- 카트에서 특정 인덱스의 아이템 반환 또는 표시
    

아이템을 추가하거나 제거하는 메서드가 있을 수 있지만, 나머지 기능을 위해 다음과 같은 특수 메서드를 만들 수 있습니다:

- 카트 아이템 수를 얻는 `__len__()`
    
- 카트 아이템을 순회하는 `__iter__()` (아이템을 볼 수 있게)
    
- 특정 아이템이 카트에 있는지 확인하는 `__contains__()`
    
- 특정 인덱스의 아이템을 반환하거나 표시하는 `__getitem__()`
    

아래는 이런 사용자 정의 메서드와 특수 메서드를 가진 `Cart` 클래스 예제입니다:

```python
class Cart:
   def __init__(self):
       self.items = []

   def add(self, item):
       self.items.append(item)

   def remove(self, item):
       if item in self.items:
           self.items.remove(item)
       else:
           print(f'{item} is not in cart')

   def list_items(self):
       return self.items

   def __len__(self):
       return len(self.items)

   def __getitem__(self, index):
       return self.items[index]

   def __contains__(self, item):
       return item in self.items

   def __iter__(self):
       return iter(self.items)
```

그리고 이렇게 사용할 수 있습니다:

```python
cart = Cart()
cart.add('Laptop')
cart.add('Wireless mouse')
cart.add('Ergo keyboard')
cart.add('Monitor')

for item in cart:
   print(item, end=' ') # Laptop Wireless mouse Ergo keyboard Monitor

print(len(cart)) # 4
print(cart[3]) # Monitor

print('Monitor' in cart) # True
print('banana' in cart) # False

cart.remove('Ergo keyboard')

print(cart.list_items()) # ['Laptop', 'Wireless mouse', 'Monitor']

cart.remove('banana') # banana is not in cart
```

이것이 Python에서 특수 메서드를 실제로 사용하는 몇 가지 방법입니다.
