# --explanation--

短语动词是与介词或副词结合形成新含义的动词。在本部分对话中，你会看到 `to send something` 和 `to send something over` 之间的区别。

`To send something` 意味着使某物从一个地方或人传到另一个地方或人，而 `to send something over` 则增加了将其交付给某人或某地的含义，通常是非正式或数字的。来看每种情况的一个例子：

- `I'll send the letter tomorrow.`——一个基本的传递语句。

- `Can you send over the file by email?`——这强调将某物*发送给某人*，通常是短距离或数字方式。添加 `over` 使短语在日常对话中听起来更自然。
