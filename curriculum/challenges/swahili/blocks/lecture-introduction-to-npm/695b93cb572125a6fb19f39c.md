# --description--

Katika funzo la awali, ulijifunza ni utegemezi gani na jinsi ya kuchagua utegemezi sahihi kwa mradi wako.

Katika funzo hili, utajifunza jinsi ya kusakinisha na kufuta utegemezi kwa kutumia npm.

Ili kusakinisha utegemezi, tumia amri ya `npm install` kama ifuatavyo:

```bash
npm install dependencyName
```

Unaweza pia kufupisha `install` kuwa `i` kama ifuatavyo:

```bash
npm i dependencyName
```

Baada ya kusakinisha kifurushi, npm huongeza sehemu mpya ya `dependencies` kwenye faili yako ya `package.json` (ikiwa haipo tayari) na kuorodhesha kifurushi ulichosakinisha. Kwa mfano, hivi ndivyo inavyoonekana unapoweka kifurushi cha `chalk`:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "chalk": "^5.6.2"
  }
}
```

Unapoweka utegemezi, vitu viwili vipya vinaonekana kwenye folda kuu ya mradi wako:

* `node_modules/`: folda inayoshikilia msimbo halisi wa vifurushi ulivyoweka.
    
* `package-lock.json`: faili inayohifadhi taarifa za kina kuhusu matoleo halisi ya vifurushi hivyo, pamoja na utegemezi wao wenyewe. (Utajifunza zaidi kuhusu faili hii katika funzo lijalo.)
    

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/how-can-you-install-and-remove-dependencies-1.png" alt="">

Miradi mingi inahitaji zaidi ya utegemezi mmoja. Badala ya kuvisakinisha moja moja, npm inakuwezesha kusakinisha vifurushi vingi kwa wakati mmoja kwa kuingiza orodha ya majina ya vifurushi iliyotenganishwa kwa nafasi:

```bash
npm install express lodash bcrypt
```

Kama ilivyokuwa awali, unapoweka utegemezi mwingi kwa wakati mmoja, npm husasisha faili zako za `package.json` na `package-lock.json` pamoja na `node_modules` ili kuviingiza. Hivi ndivyo `package.json` itakavyoonekana:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "bcrypt": "^6.0.0",
    "chalk": "^5.6.2",
    "express": "^5.1.0",
    "lodash": "^4.17.21"
  }
}
```

Ili kusakinisha `devDependency`, ongeza bendera ya `--save-dev` kwenye amri ya `install`. Unaweza pia kutumia toleo fupi la `-D`:

```bash
npm i nodemon -D
```

Baada ya kuendesha amri hii, npm husasisha faili yako ya `package.json` na kuongeza kifurushi chini ya sehemu ya `devDependencies`. Hapa kuna mfano wa jinsi inavyoonekana:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "bcrypt": "^6.0.0",
    "chalk": "^5.6.2",
    "express": "^5.1.0",
    "lodash": "^4.17.21"
  },
  "devDependencies": {
    "nodemon": "^3.1.10"
  }
}
```

Ili kufuta au kuondoa kifurushi, tumia amri ya `npm uninstall`:

```bash
npm uninstall dependencyName
```

Kwa mfano, ili kufuta `chalk` kutoka kwenye mradi wako, utatekeleza amri hii:

```bash
npm uninstall chalk
```

Unaweza pia kufuta vifurushi vingi kwa wakati mmoja kwa kuorodhesha majina yao na nafasi kati ya kila jina la kifurushi:

```bash
npm uninstall express lodash bcrypt
```

Na faili yako ya `package.json` itasasishwa moja kwa moja:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "devDependencies": {
    "nodemon": "^3.1.10"
  }
}
```

Ili kufuta `devDependency`, unaweza tu kuendesha:

```bash
npm uninstall nodemon
```

Kumbuka huna haja ya bendera ya `-D` wakati wa kufuta. npm itafuta kifurushi moja kwa moja kutoka sehemu sahihi, yaani `devDependencies` katika kesi hii.

Baada ya kufuta `devDependency`, `package.json` yako itasasishwa tena:

