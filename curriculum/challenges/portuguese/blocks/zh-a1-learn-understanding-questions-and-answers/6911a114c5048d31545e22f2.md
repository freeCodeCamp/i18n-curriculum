# --description--

Você aprendeu a se apresentar dizendo de onde você é usando `我是 (wǒ shì)` + nacionalidade. Você também aprendeu a usar `你 (nǐ)` para se referir à pessoa com quem está falando.

Agora, se você quiser falar sobre a nacionalidade da pessoa com quem está falando, qual estrutura deve usar?

# --instructions--

`BLANK 中国人吗 (zhōng guó rén ma)？`

## --blanks--

`你是 (nǐ shì)`

### --feedback--

Isso significa "você é".

# --explanation--

`你是 (nǐ shì)` significa "você é". Você pode usar a estrutura `你是 (nǐ shì)` + nacionalidade para falar sobre a nacionalidade da pessoa com quem está falando. Por exemplo:

`你是美国人吗 (nǐ shì měi guó rén ma)？` – Você é americano?

Da mesma forma, você pode usar `你是 (nǐ shì)` + nome ou profissão para falar sobre o nome ou profissão da outra pessoa. Exemplos:

`你是刘明吗 (nǐ shì liú míng ma)？` – Você é Liu Ming?

`你是开发者吗 (nǐ shì kāi fā zhě ma)？` – Você é um desenvolvedor?
