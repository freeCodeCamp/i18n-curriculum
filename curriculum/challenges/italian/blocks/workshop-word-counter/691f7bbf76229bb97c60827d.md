# --description--

In questo workshop, eserciterai l'uso dei loop `for...of` costruendo una funzione che conta quante volte una stringa appare in un array di stringhe.

Prima di scrivere quella funzione, creerai una più semplice che stampa ogni carattere di una stringa nella console.

Inizia definendo una funzione vuota chiamata `printCharacters` con il parametro `str`.
