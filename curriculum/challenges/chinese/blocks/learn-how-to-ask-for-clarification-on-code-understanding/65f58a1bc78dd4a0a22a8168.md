# --description--

`Encounter`，特别是它的过去式 `encountered`，意味着遇到某事，通常是意外的。例如，`I encountered an error while testing the software.` 它暗示在没有寻找的情况下查找到某物。

`Pinpoint` 意味着非常精确地识别某物。例如，`After reviewing the code, I was able to pinpoint the error.` 这意味着准确找到问题的所在和内容。

在编码中，查找问题通常是关于 `encountering` 意外问题并尝试 `pinpoint` 它们的原因。
