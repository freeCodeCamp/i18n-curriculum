# --description--

في أي لغة برمجة، تحتاج إلى مقارنة القيم لاتخاذ قرارات. على سبيل المثال، إذا كان الطقس مشمسًا، ستخرج، وإلا ستبقى في المنزل. مثال آخر هو التحقق مما إذا كان لدى المستخدم نقاط كافية للترقية في لعبة.

تُستخدم جملة `if-else` الشرطية لاتخاذ القرارات في JavaScript. غالبًا ما تُستخدم مع عوامل المقارنة لمقارنة القيم واتخاذ القرارات بناءً على النتائج.

مثال على جملة if الشرطية هو:

```javascript
let x = 5;

if (x > 3) {
  console.log("x is greater than 3");
} else {
  console.log("x is less than or equal to 3");
}
```

في هذا المثال، تُقارن المتغير `x` بالعدد `3` باستخدام عامل `>`. إذا كان `x` أكبر من `3`، سينفذ أول كتلة شفرة. وإلا، سينفذ كتلة الشفرة الثانية.
