# --description--

正の整数 $n$ に対して、$σ_2(n)$ をその約数の平方の和とします。例えば、

$$σ_2(10) = 1 + 4 + 25 + 100 = 130$$

$σ_2(n)$ が完全な平方数となるような $0 < n < 64\,000\,000$ のすべての $n$ の和を求めてください。
