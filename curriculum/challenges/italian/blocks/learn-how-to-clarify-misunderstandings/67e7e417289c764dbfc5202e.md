# --explanation--

Nella frase `limit what other parts can see`, `what` introduce una clausola nominale, che funge da oggetto di `limit`.

Una clausola nominale è una clausola che svolge la funzione di nome all'interno di una frase. Può funzionare come oggetto o soggetto della frase. Di solito inizia con parole come `that`, `what`, `whether`, `who`, `when`, `why`, `how`, ecc.

Le clausole nominali usano l'ordine normale delle parole di una frase affermativa (non l'ordine interrogativo). Per esempio:

- `The team is checking why the system crashed during the update.` - `Why the system crashed during the update` è la clausola nominale che funge da oggetto di `checking`.
   
- `What the function returns depends on the input data.` - `What the function returns` è una clausola nominale che funge da soggetto di `depends on`.
