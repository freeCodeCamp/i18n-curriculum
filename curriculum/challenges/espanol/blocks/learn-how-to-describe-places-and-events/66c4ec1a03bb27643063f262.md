# --explanation--

El tiempo `Present Continuous` se usa para describir acciones que están ocurriendo en este momento. Cuando quieres decir que algo **no** está ocurriendo, usas la forma negativa. Por ejemplo:

- `He is not working today.` - Esto significa que él actualmente no está disponible para trabajar.
