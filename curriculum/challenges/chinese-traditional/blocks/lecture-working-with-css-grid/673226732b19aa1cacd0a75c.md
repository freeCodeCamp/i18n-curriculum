# --interactive--

在上一課中，你已經介紹了 CSS grid，它可以用來在你的網頁中創建複雜且流動的版面配置。在本課中，我們將探討如何使用 `fr` 單位來創建彈性網格版面配置。

讓我們從這個將表述我們的網格容器的 HTML 標記開始：

```html
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

在 CSS 中，我們將容器的 display 屬性設為 grid。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 25% 25% 25% 25%;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

`grid-template-columns` 用於設定每個行的大小。在這種情況下，每個行的大小將是容器的 `25%`。然後使用 gap 屬性來為每個行周圍創建間距。

到目前為止，我們一直使用百分比來設定行大小，但我們也可以使用 `fr` 單位。

`fr` 單位是一種分數單位，用來表述網格容器空間的一部分。

以下是將程式碼重構為使用 `fr` 單位取代百分比後的樣子。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

每個行將佔用可用空間的一個分數。由於有四個行，每個行將在網格容器中擁有相等的可用空間份額。

當你開始建置你的網格佈局時，你會發現自己越來越想使用 `fr` 單位，因為它們提供了一種靈活且成比例的空間分配方式，讓你能夠創建響應式佈局，適應不同的螢幕尺寸，而無需手動調整像素值。
