# --description--

Ili kuhuisha kipengele, unahitaji kujua kuhusu sifa za animation na kanuni ya `@keyframes`. Sifa za animation zinaongoza jinsi animation inavyopaswa kufanya kazi na kanuni ya `@keyframes` inaongoza kinachotokea wakati wa animation hiyo. Kuna sifa nane za animation kwa jumla. Changamoto hii itakuwa rahisi na itashughulikia mbili muhimu kwanza:

`animation-name` huweka jina la animation, ambalo hutumika baadaye na `@keyframes` kuambia CSS ni kanuni gani zinaendana na animations gani.

`animation-duration` huweka muda wa urefu wa animation.

`@keyframes` ni jinsi ya kubainisha hasa kinachotokea ndani ya animation kwa muda wote. Hii hufanywa kwa kutoa sifa za CSS kwa "fremu" maalum wakati wa animation, kwa asilimia kuanzia 0% hadi 100%. Ukilinganisha na filamu, sifa za CSS kwa 0% ni jinsi kipengele kinavyoonekana katika sehemu ya mwanzo ya filamu. Sifa za CSS kwa 100% ni jinsi kipengele kinavyoonekana mwishoni, kabla ya kuanza kuonyesha majina ya wahusika. Kisha CSS hutumia uchawi kuhamisha kipengele kwa muda uliotolewa ili kuigiza sehemu hiyo. Hapa kuna mfano wa kuonyesha matumizi ya `@keyframes` na sifa za animation:

```css
#anim {
  animation-name: colorful;
  animation-duration: 3s;
}

@keyframes colorful {
  0% {
    background-color: blue;
  }
  100% {
    background-color: yellow;
  }
}
```

Kwa kipengele chenye kitambulisho cha `anim`, kipande cha msimbo kilicho juu kinaweka `animation-name` kuwa `colorful` na kuweka `animation-duration` kwa sekunde 3. Kisha kanuni ya `@keyframes` inaunganisha na sifa za animation zenye jina `colorful`. Huweka rangi kuwa buluu mwanzoni mwa animation (0%) ambayo itabadilika kuwa njano mwishoni mwa animation (100%). Huna kikomo cha mabadiliko ya mwanzo-mwisho tu, unaweza kuweka sifa kwa kipengele kwa asilimia yoyote kati ya 0% na 100%.

# --instructions--

Tengeneza animation kwa kipengele chenye kitambulisho cha `rect`, kwa kuweka `animation-name` kuwa `rainbow` na `animation-duration` kuwa sekunde 4.
