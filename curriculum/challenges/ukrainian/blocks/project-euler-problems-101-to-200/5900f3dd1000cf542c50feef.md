# --description--

Рухаючись зліва направо, якщо жодна цифра не менша за цифру ліворуч від неї, таке число називають зростаючим; наприклад, 134468.

Аналогічно, якщо жодна цифра не менша за цифру праворуч від неї, таке число називають спадним; наприклад, 66420.

Ми назвемо додатне ціле число, яке не є ні зростаючим, ні спадним, «числом з коливаннями»; наприклад, 155349.

Очевидно, що чисел з коливаннями не може бути серед чисел менших за сто, але трохи більше половини чисел менших за тисячу (525) є числами з коливаннями. Насправді, найменше число, для якого частка чисел з коливаннями вперше досягає 50%, — це 538.

Дивовижно, але числа з коливаннями стають дедалі поширенішими, і до того часу, як ми досягаємо 21780, частка чисел з коливаннями дорівнює 90%.

Знайдіть найменше число, для якого частка чисел з коливаннями дорівнює рівно 99%.
