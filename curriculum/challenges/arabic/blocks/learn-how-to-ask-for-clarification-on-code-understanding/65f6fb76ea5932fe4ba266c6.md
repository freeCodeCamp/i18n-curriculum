# --description--

`By the end of it` تعني عندما ينتهي شيء ما أو يكون على وشك الانتهاء.

`A clearer picture` تعني مجازًا فهمًا أفضل أو أكثر اكتمالًا. يشبه الأمر عندما تنظف نظارتك وترى الأشياء بوضوح أكبر فجأة.

مثال على `a clearer picture`: `After studying, I have a clearer picture of the math problem.`
