# --description--

freeCodeCampのカスタム入力を使えば、タスク内で直接ピンインや中国語の文字を入力できます。

コンピューターで中国語キーボードを使う場合、ピンインを入力して候補から中国語の文字を選べます。設定方法は以下の通りです。

## Windows 11での設定  

1. スタートメニューから**設定**を開いてください。

2. **時刻と言語**に進み、次に**言語と地域**をクリックします。

3. **優先する言語**に行き、**言語を追加する**を選んでください。

4. **Chinese**を検索し、**Chinese (Simplified)**を選びます。

5. **次へ**をクリックし、指示に従ってください。

言語がインストールされたら：  

- **優先する言語**リストで見つけてください。

- 省略記号をクリックして追加オプションを表示し、**言語オプション**を選びます。

- **キーボード**に進み、**キーボードを追加**をクリックして中国語（簡体字）レイアウトを選択してください。

キーボードを切り替えるには：  

- **Windowsキー + スペースバー**を押します。

- またはタスクバーの言語アイコン（通常は**ENG**と表示）をクリックして中国語を選んでください。

## Macでの設定  

**注意：** 使用しているOSによって手順が異なる場合があります。

1. **Appleメニュー**を開き、**システム環境設定**を選択します。

2. **キーボード**に進みます。

3. **入力ソース**タブをクリックします。

4. **プラス（+）**ボタンを選び、**Chinese, Simplified**をクリックしてリストから**Pinyin - Simplified**を選択してください。  

キーボードを切り替えるには：

- **Control + スペースバー**を押します。

- またはメニューバーの入力アイコンをクリックしてピンイン方式を選んでください。

これで設定は完了です。
