# --explanation--

這句話使用 `will` 來顯示 `Future Simple` 時態，指出該動作預期會發生。 

結構 `will + verb` 通常用來談論未來的事件、計畫或預測。例如：

- `They will be at the meeting tomorrow.`－這表示他們計劃在未來出席會議。

- `We will start the project next week。`－這表示該專案預定在未來開始。

使用 `will` 是表達未來意圖、承諾或排程的簡單方法。
