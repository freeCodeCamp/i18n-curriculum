# --explanation--

Uma `overall reaction` significa a forma geral como um grupo de pessoas respondeu a algo, olhando para o quadro completo, não apenas uma pessoa ou um momento. Por exemplo:

`The overall reaction to the new security policy was positive.` – Isso significa que a maioria das pessoas teve uma boa resposta a ela.
