# --explanation--

Una `security breach` ocurre cuando alguien obtiene acceso no autorizado a información, sistemas o redes. Es un término importante en ciberseguridad. Por ejemplo:

`A weak password caused the security breach.` - El sistema no estaba seguro debido a una contraseña fácil, y alguien entró sin permiso.
