# --description--

對於正整數 $n$，定義 $f(n)$ 為最小的正整數倍數 $n$，其以十進位表示時僅使用數字 $≤ 2$。

因此 $f(2) = 2$、$f(3) = 12$、$f(7) = 21$、$f(42) = 210$、$f(89) = 1\\,121\\,222$。

此外，$\displaystyle\sum_{n = 1}^{100} \frac{f(n)}{n} = 11\\,363\\,107$。

求 $\displaystyle\sum_{n = 1}^{10\\,000} \frac{f(n)}{n}$。
