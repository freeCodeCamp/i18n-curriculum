# --explanation--

`Which` inaweza kutumika kuanzisha taarifa za ziada kuhusu kifungu kilichotangulia. Hii huitwa **kifungu kisichoainisha** — kinaongeza maelezo lakini si muhimu kwa maana kuu ya sentensi. Kwa mfano:

`The system crashed during the update, which caused a delay.` – Sehemu baada ya `which` hutoa taarifa zaidi kuhusu matokeo ya ajali, lakini inaweza kuachwa bila kubadilisha maana kuu.
