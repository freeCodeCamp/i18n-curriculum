# --description--

`security incident`는 회사나 조직의 안전이나 보안에 해로울 수 있는 이벤트입니다. 예를 들어, `A security incident like a stolen password needs to be reported immediately`가 있습니다.

`harm`와 `harmful`에 대한 설명:

`Harm`는 누군가나 무언가에 대한 손상이나 부상을 의미합니다. 부정적인 영향을 주거나 누군가 또는 무언가를 위험에 빠뜨리는 것을 말합니다. 예를 들어:

`Touching hot surfaces can cause harm.` - 뜨거운 표면에 닿으면 부상이나 손상이 발생할 수 있다는 뜻입니다.

`Harmful`는 해를 끼칠 수 있는 것을 설명할 때 쓰이는 형용사입니다. 손상이나 부상으로 이어질 수 있는 위험하거나 위험한 상태를 의미합니다. 예를 들어:

`Smoking is harmful to health.` - 흡연이 건강 문제나 손상을 일으킬 수 있음을 나타냅니다.

이 용어들은 특히 건강, 보안, 환경과 같은 맥락에서 안전과 위험을 논할 때 자주 사용됩니다.
