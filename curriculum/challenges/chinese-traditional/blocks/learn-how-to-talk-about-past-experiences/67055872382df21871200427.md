# --explanation--

`Properly` 是形容詞 `proper` 的副詞形式。這兩個字都可以表示以正確或適當的方式做某事，但它們的用法會依據句子結構而有所不同。例如：  

- `The application didn't function properly after the update.`－這表示更新為程式碼帶來了錯誤。副詞 `properly` 修飾動詞 `function`。

- `This wasn't a proper configuration of the database。` - 形容詞 `proper` 描述名詞 `configuration`。

在這裡，James 使用 `properly` 來描述系統沒有以正確的方式連接到資料庫。
