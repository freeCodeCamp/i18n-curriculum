# --explanation--

`Came in` は過去形で、何かが「到着した」または「受け取られた」という意味です。メッセージや結果、フィードバックが届いたり利用可能になったりしたことを話すときによく使われます。例えば：

`The test results came in yesterday.` - これは結果が昨日届いたことを意味します。

この文では、`since that first feedback came in` が最初のフィードバックが届いたことを意味し、彼らの進歩の始まりを示しています。
