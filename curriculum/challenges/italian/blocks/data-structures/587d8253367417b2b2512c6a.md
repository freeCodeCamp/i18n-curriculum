# --description--

Gli array sono oggetti JavaScript che possono contenere molti elementi diversi.

```js
var complexArr = [1, 5, "2", "Word", {"name": "James"}];
```

Fondamentalmente, quello che succede dietro le quinte è che il tuo browser assegna automaticamente la quantità giusta di memoria per quell’array. Cambierà anche se aggiungi o rimuovi dati.

Tuttavia, nel mondo delle alte prestazioni e dei diversi tipi di elementi, a volte serve essere più precisi su quanta memoria viene assegnata a un array.

<dfn>Gli array tipizzati</dfn> sono la soluzione a questo problema. Ora puoi specificare quanta memoria vuoi assegnare a un array. Qui sotto trovi una panoramica base dei diversi tipi di array disponibili e la dimensione in byte di ogni elemento in quell’array.

<table><tbody><tr><th>Tipo</th><th>Dimensione di ogni elemento in byte</th></tr><tr><td><code>Int8Array</code></td><td>1</td></tr><tr><td><code>Uint8Array</code></td><td>1</td></tr><tr><td><code>Uint8ClampedArray</code></td><td>1</td></tr><tr><td><code>Int16Array</code></td><td>2</td></tr><tr><td><code>Uint16Array</code></td><td>2</td></tr><tr><td><code>Int32Array</code></td><td>4</td></tr><tr><td><code>Uint32Array</code></td><td>4</td></tr><tr><td><code>Float32Array</code></td><td>4</td></tr><tr><td><code>Float64Array</code></td><td>8</td></tr></tbody></table>

Ci sono due modi per creare questo tipo di array. Uno è crearli direttamente. Qui sotto c’è come creare un `Int16Array` di lunghezza 3.

```js
var i8 = new Int16Array(3);
console.log(i8);
// Returns [0, 0, 0]
```

Puoi anche creare un <dfn>buffer</dfn> per assegnare quanta memoria (in byte) vuoi che l’array occupi. **Nota**  
Per creare array tipizzati usando i buffer, devi assegnare un numero di byte che sia un multiplo dei byte elencati sopra.

```js
// Create same Int16Array array differently
var byteSize = 6; // Needs to be multiple of 2
var buffer = new ArrayBuffer(byteSize);
var i8View = new Int16Array(buffer);
buffer.byteLength; // Returns 6
i8View.byteLength; // Returns 6
console.log(i8View); // Returns [0, 0, 0]
```

I <dfn>buffer</dfn> sono oggetti a uso generale che contengono dati. Non puoi consultarli normalmente. Per consultarli devi prima creare una <dfn>view</dfn>.

```js
i8View[0] = 42;
console.log(i8View); // Returns [42, 0, 0]
```

**Nota**  
Gli array tipizzati non hanno alcuni dei metodi che hanno gli array tradizionali come `.pop()` o `.push()`. Gli array tipizzati falliscono anche `Array.isArray()` che verifica se qualcosa è un array. Anche se più semplici, questo può essere un vantaggio per motori JavaScript meno sofisticati che li implementano.

# --instructions--

Prima crea un `buffer` di 64 byte. Poi crea un array tipizzato `Int32Array` con una view chiamata `i32View`.

# --hints--

Il tuo `buffer` dovrebbe essere grande 64 byte.

```js
assert(buffer.byteLength === 64);
```

La tua view `i32View` del buffer dovrebbe essere grande 64 byte.

```js
assert(i32View.byteLength === 64);
```

La tua view `i32View` del buffer dovrebbe avere 16 elementi.

```js
assert(i32View.length === 16);
```
