## --blanks--

`support`

### --feedback--

A palavra descreve a capacidade do programa de lidar com diferentes tarefas ou funcionalidades.

---

`feature`

### --feedback--

Esta palavra descreve uma função ou capacidade específica que um programa de software oferece aos seus usuários.

# --explanation--

No desenvolvimento de software, `support` significa que um programa ou ferramenta é capaz de funcionar com um determinado recurso ou funcionalidade.Uma `feature` refere-se a uma função ou capacidade específica que um programa de software oferece aos seus usuários.Por exemplo:

- `VS Code supports multiple programming languages.` - O VS Code pode trabalhar com diferentes linguagens.

- `One feature of VS Code is the ability to integrate with Git.` - Isso se refere a uma capacidade específica que o VS Code oferece aos seus usuários.
