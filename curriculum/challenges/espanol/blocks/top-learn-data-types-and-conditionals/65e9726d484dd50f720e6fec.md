# --description--

JavaScript también tiene la capacidad de comparar tipos y valores usando el operador de igualdad estricta `===`. Este operador verifica si los dos valores son iguales y del mismo tipo. Por ejemplo:

```javascript
let x = 5;
let y = "5";

let result = x === y;

console.log(result); // false
```

En este ejemplo, la variable `result` contendrá `false` porque `x` es un número y `y` es una cadena. Pero si usas el operador `==`, el resultado será `true` porque JavaScript convertirá la cadena a un número y comparará los valores.

Este operador se llama operador de igualdad estricta porque verifica tanto la igualdad de valor como de tipo. A menudo se recomienda usar el operador `===` para evitar resultados inesperados al comparar valores.

También existe un operador de desigualdad estricta `!==` que verifica si los dos valores no son iguales y del mismo tipo.

Hay muchas formas de usar el operador de igualdad estricta, y es importante entender cómo funciona.
