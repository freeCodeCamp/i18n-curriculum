# --interactive--

تعمل العوامل الأحادية على معامل واحد لأداء عمليات مثل تحويل النوع، تعديل القيمة، أو التحقق من شروط معينة. لنلقِ نظرة على بعض العوامل الأحادية الشائعة وكيف تعمل.

عامل الجمع الأحادي يحول المعامل إلى رقم. إذا كان المعامل رقمًا بالفعل، يبقى دون تغيير.

:::interactive_editor

```js
const str = '42';
const strToNum = +str;

console.log(strToNum); // 42
console.log(typeof str); // string
console.log(typeof strToNum); // number
```

:::

عامل الجمع الأحادي مفيد عندما تريد التأكد من أنك تتعامل مع قيمة رقمية. كما قد تتوقع، هناك عامل نفي أحادي. ينفي قيمة المعامل. يعمل بطريقة مشابهة لعامل الجمع الأحادي، لكنه يعكس الإشارة.

:::interactive_editor

```js
const str = '42';
const strToNegativeNum = -str;

console.log(strToNegativeNum); // -42
console.log(typeof str); // string
console.log(typeof strToNegativeNum); // number
```

:::

عامل النفي المنطقي، الممثل بعلامة التعجب (`!`)، هو عامل أحادي آخر. يعكس القيمة المنطقية للمعامل. فإذا كان المعامل `true`، يصبح `false`، وإذا كان `false`، يصبح `true`.

:::interactive_editor

```js
let isOnline = true;
console.log(!isOnline); // false

let isOffline = false;
console.log(!isOffline); // true
```

:::

عامل النفي على مستوى البتات هو عامل أحادي أقل استخدامًا. ممثلًا بالمد `~`، يعكس التمثيل الثنائي للرقم. تخزن الحواسيب الأرقام بصيغة ثنائية (1 و0). عامل `~` يعكس كل بت، أي يغير كل 1 إلى 0 وكل 0 إلى 1. ستتعلم المزيد عن الثنائي والبتات في درس قادم.

:::interactive_editor

```js
const num = 5; // The binary for 5 is 00000101

console.log(~num); // -6
```

:::

في هذا المثال، أصبح `5` هو `-6` لأن تطبيق عامل `~` على `5` يعطي `- (5 + 1)`، وهو ما يساوي `-6` بسبب تمثيل المكمل الثنائي. المكمل الثنائي هو طريقة تمثيل الأعداد السالبة في الحواسيب بصيغة ثنائية. على الأرجح لن تستخدم عامل النفي على مستوى البتات كثيرًا إلا إذا كنت تعمل على مهام برمجية منخفضة المستوى مثل تعديل البتات مباشرة.

الكلمة المفتاحية `void` هي عامل أحادي يقيم تعبيرًا ويُرجع `undefined`.

:::interactive_editor

```js
const result = void (2 + 2);

console.log(result); // undefined
```

:::

يُستخدم `void` أيضًا في الروابط لمنع التنقل:

```js
<a href="javascript:void(0);">Click Me</a>
```

أخيرًا، هناك عامل `typeof` الذي تعلمته في دروس سابقة. هذا العامل يُرجع نوع المعامل كسلسلة نصية.

:::interactive_editor

```js
const value = 'Hello world';

console.log(typeof value); // string
```

:::
