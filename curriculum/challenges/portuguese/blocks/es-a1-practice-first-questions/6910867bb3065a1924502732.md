`te`

# --explanation--

Este diálogo entre Mateo e Camila inclui essas frases comuns:

`¿Cómo te llamas?` significa "Qual é o seu nome?" (informal).

`Mi nombre es...` significa "Meu nome é".

`¿Y tú?` significa "E você?". Isso é usado para retornar uma pergunta de forma informal.

`Me llamo...` é a forma mais comum de dizer seu nome.
