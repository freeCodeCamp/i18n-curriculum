# --description--

Considera el triángulo con lados $\sqrt{5}$, $\sqrt{65}$ y $\sqrt{68}$. Se puede demostrar que este triángulo tiene un área de 9.

$S(n)$ es la suma de las áreas de todos los triángulos con lados $\sqrt{1 + b^2}$, $\sqrt{1 + c^2}$ y $\sqrt{b^2 + c^2}$ (para enteros positivos $b$ y $c$) que tienen un área entera que no excede $n$.

El triángulo de ejemplo tiene $b = 2$ y $c = 8$.

$S({10}^6) = 18\\,018\\,206$.

Encuentra $S({10}^{10})$.
