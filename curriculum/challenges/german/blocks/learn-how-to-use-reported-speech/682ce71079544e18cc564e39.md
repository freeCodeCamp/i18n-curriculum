# --explanation--

`To prevent` bedeutet, Maßnahmen zu ergreifen, damit etwas nicht passiert. Zum Beispiel:

`They installed new software to prevent viruses.` – Das bedeutet, die Software hilft, Viren zu stoppen, bevor sie Probleme verursachen.

Ein `incident` ist ein unerwartetes Ereignis, oft negativ oder problematisch. Zum Beispiel:

`There was a security incident last night that shut down the server.` – Das bedeutet, es ist etwas passiert, das Probleme mit dem System verursacht hat.
