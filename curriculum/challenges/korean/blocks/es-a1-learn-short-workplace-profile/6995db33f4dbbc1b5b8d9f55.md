# --description--

질문 단어 `¿Dónde...`는 "어디...?"라는 뜻입니다.

만약 단어 `Dónde`가 질문을 시작한다면, `o` 글자에 강세 부호가 있어야 합니다.

Sebastián은 Carlos에 대해 무언가를 묻기 위해 이 단어를 사용하고 있습니다.

# --explanation-- 

동사 `trabajar`는 "일하다"라는 뜻입니다.

`Trabaja`는 이 동사의 3인칭 단수형입니다.

Sebastián은 `¿Dónde trabaja?`라고 묻는데, 이는 "그는 어디에서 일하나요?"라는 뜻입니다. 이것이 스페인어에서 누군가가 현재 어디에서 일하는지 묻는 표준적인 방법입니다.
