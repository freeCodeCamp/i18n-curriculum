# --interactive--

Njia ya `toString()` ni kipengele cha msingi katika JavaScript kinachobadilisha thamani kuwa mfuatano wa herufi unaowakilisha thamani hiyo. Ni njia unayoweza kutumia kwa nambari, thamani za kweli au siyo kweli, safu za data, na vitu. Mojawapo ya matumizi ya kawaida ya `toString()` ni kubadilisha nambari kuwa mfuatano wa herufi unaowakilisha nambari hiyo. Hapa kuna mfano:

:::interactive_editor

```js
const num = 10;
console.log(num.toString()); // "10"
```

:::

Njia hii inakubali msingi wa nambari wa hiari ambao ni nambari kutoka `2` hadi `36`. Msingi huu wa nambari unawakilisha msingi, kama msingi wa 2 kwa binary au msingi wa 8 kwa octal. Ikiwa msingi haujatolewa, kwa chaguo-msingi ni msingi wa 10, yaani desimali. Hapa kuna mfano wa kupitisha `2` kama hoja kwa njia ya `toString()`:

:::interactive_editor

```js
const num = 10;
console.log(num.toString(2)); // "1010"
```

:::

Matokeo yatakuwa `1010` ambayo ni uwakilishi wa binary kwa nambari ya desimali `10`.

Pia unaweza kutumia njia ya `toString()` kubadilisha safu za data na vitu kuwa mfuatano wa herufi. Safu za data zina utekelezaji maalum wa `toString()` unaobadilisha kila kipengele kuwa mfuatano wa herufi na kuviunganisha kwa koma. Hapa kuna mfano:

:::interactive_editor

```js
const arr = [1, 2, 3];
console.log(arr.toString()); // "1,2,3"
```

:::

Katika mfano huu vipengele vyote vya safu ya data vimeunganishwa pamoja kuunda mfuatano wa herufi `1,2,3`.

Wakati njia ya `toString()` inapotumika kwa vitu, matokeo hayatakuwa toleo la mfuatano wa herufi la mali za kitu hicho.

:::interactive_editor

```js
const person = {
  name: "John",
  age: 30,
  isStudent: true
};

console.log(person.toString()); // "[object Object]"
```

:::

Katika mfano huu, matokeo yatakuwa uwakilishi wa mfuatano wa herufi wa chaguo-msingi kwa kitu hicho ambacho ni `[object Object]`. Ili kupata toleo la mfuatano wa herufi la mali za kitu cha `person` utahitaji kutumia `JSON.stringify()` ambayo utajifunza zaidi katika mafundisho yajayo.

Kwa kumalizia, njia ya `toString()` hutumika kubadilisha thamani kuwa mfuatano wa herufi. Kuelewa jinsi inavyofanya kazi na aina tofauti za data na jinsi ya kuibadilisha kwa vitu vyako binafsi kunaweza kukuza sana uwezo wako wa kushughulikia na kuonyesha data katika programu zako za JavaScript.
