# --explanation--

`Made sure`は、何かが真実であるか、正しく機能しているかを確認するために行動を起こすことを意味します。例えば：

`They made sure the software was ready before launch.` – 彼らはすべてが正しく動作するか確認しました。

`As`は、別の動作と同時に何かが起こることを表すために使われます。例えば：

`The system adds more resources as demand increases.` – より多くの人が使い始めると同時に調整されます。
