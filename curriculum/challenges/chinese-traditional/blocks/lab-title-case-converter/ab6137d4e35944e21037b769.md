# --description--

在本實驗中，你將創建一個函式（程式），將字串轉換為標題大小寫。標題大小寫意指每個字的第一個字母大寫，其餘字母則為小寫。

「Web Development Is Awesome」是一個標題式大小寫的字串範例。

**目標**：完成以下使用者故事並通過所有測試以完成實驗。

**使用者故事：**

1. 你應該有一個 `titleCase` 函式（程式），它接受一個字串作為引數。
1. `titleCase` 函式應該傳回一個字串，其中每個字的第一個字母大寫，其餘字母小寫。
1. `titleCase("I like to code")` 應該傳回 `"I Like To Code"`。
1. `titleCase("javaScript is fun")` 應該傳回 `"Javascript Is Fun"`。

# --solutions--

```js
function titleCase(str) {
  return str
    .split(' ')
    .map(word => word.charAt(0).toUpperCase() + word.substring(1).toLowerCase())
    .join(' ');
}
```
