# --description--

このタスクでは、`simple` と `simply` の違いを学び、`double` と `slashes` が何を意味するか理解します。

`Simple` は「簡単な」や「複雑でない」という意味の形容詞で、`a simple task` のような言葉です。`Simply` は副詞で、「簡単に」や「単純に」という意味があり、`doing something simply` のような言葉です。

`Double` は「2つの何か」を意味します。そして `slashes` は文章で使われる `/` 記号のことです。`es` を加えて `slash` を複数形にします。なぜなら `sh` で終わる単語は `es` を加えて複数形を作るからで、`dishes` や `brushes` のような例があります。
