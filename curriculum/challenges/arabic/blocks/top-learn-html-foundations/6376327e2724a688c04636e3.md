# --description--

HTML و CSS لغتان تعملان معًا لإنشاء كل ما تراه عند تصفح شيء على الإنترنت. HTML هي البيانات الخام التي يُبنى منها صفحة الويب. كل النصوص والروابط والبطاقات والقوائم والأزرار تُنشأ باستخدام HTML. أما CSS فهي تضيف الأسلوب لتلك العناصر البسيطة. HTML تضع المعلومات على صفحة الويب، وCSS تحدد موضع تلك المعلومات، وتعطيها اللون، وتغير الخط، وتجعلها تبدو رائعة!
