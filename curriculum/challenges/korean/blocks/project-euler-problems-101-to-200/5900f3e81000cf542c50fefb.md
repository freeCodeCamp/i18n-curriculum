# --description--

$n$의 루트값 $rad(n)$은 $n$의 서로 다른 소인수들의 곱입니다. 예를 들어, $504 = 2^3 × 3^2 × 7$이므로 $rad(504) = 2 × 3 × 7 = 42$입니다.

$1 ≤ n ≤ 10$에 대해 $rad(n)$을 계산하고, $rad(n)$을 기준으로 정렬하며 루트값이 같으면 $n$을 기준으로 정렬하면 다음과 같습니다:
