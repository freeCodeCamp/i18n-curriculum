# --description--

직선 격자는 격자선 사이 간격이 반드시 등간격일 필요가 없는 직교 격자입니다.

이러한 격자의 예로는 로그 그래프 용지가 있습니다.

다음 성질을 가진 데카르트 좌표계의 직선 격자를 고려하세요:

- 격자선은 데카르트 좌표계의 축에 평행합니다.
- 수직 격자선이 $N + 2$개, 수평 격자선이 $N + 2$개 있습니다. 따라서 $(N + 1) \times (N + 1)$개의 직사각형 셀이 있습니다.
- 두 바깥쪽 수직 격자선의 방정식은 $x = -1$과 $x = 1$입니다.
- 두 바깥쪽 수평 격자선의 방정식은 $y = -1$과 $y = 1$입니다.
- 격자 셀은 단위원과 겹치는 경우 빨간색으로 칠하고, 그렇지 않으면 검은색입니다.

이 문제에서는 빨간색 셀이 차지하는 영역을 최소화하도록 나머지 $N$개의 내부 수평 격자선과 $N$개의 내부 수직 격자선의 위치를 찾으세요.

예를 들어, $N = 10$일 때 해답 그림은 다음과 같습니다:

<img alt="solution for N = 10" src="https://cdn.freecodecamp.org/curriculum/project-euler/enmeshed-unit-circle.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$N = 10$일 때 빨간색 셀이 차지하는 영역은 소수점 아래 10자리까지 반올림하면 3.3469640797입니다.

$N = 400$에 대해 빨간색 셀이 차지하는 영역을 소수점 아래 10자리까지 반올림한 값을 답으로 제출하세요.
