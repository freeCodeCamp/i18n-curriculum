# --explanation--

Vamos revisar o vocabulário e a gramática principais nesta frase:  

`Hard drive` é a parte de um computador onde todos os arquivos e programas são armazenados.Por exemplo:  

`The hard drive is almost full, so I need to delete some old files.` – Isso significa que o disco rígido não tem muito espaço sobrando.  

`Failing` significa que algo está começando a parar de funcionar.Por exemplo: 

`The hard drive is failing, so we need to back up the files.` – Isso significa que o disco rígido está falhando e pode parar de funcionar em breve.  

`Is failing` é o tempo `Present Continuous`, usado para descrever algo que está acontecendo agora ou temporariamente.Por exemplo:  

`The hard drive is failing` – Isso significa que o disco rígido atualmente tem problemas e é uma questão contínua.

Nesta frase, `is failing` descreve a condição do disco rígido.Mostra que o problema está acontecendo agora e pode fazer com que o computador pare de funcionar em breve.
