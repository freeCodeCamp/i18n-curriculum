# --explanation--

Jessica sagt Mark, er soll `regular security audits` durchführen, um die Sicherheit zu gewährleisten.

Eine `security audit` ist ein Prozess, bei dem ein System auf Schwachstellen überprüft wird. Zum Beispiel:

`After the security audit, the team made improvements to the system.` – Nach der Überprüfung wurden Änderungen vorgenommen, um die Sicherheit zu verbessern.

`Regular` bedeutet, dass etwas beständig über die Zeit geschieht. Zum Beispiel:

`Regular testing helps find problems before they get worse.` – Häufiges Testen hilft, Probleme frühzeitig zu erkennen.
