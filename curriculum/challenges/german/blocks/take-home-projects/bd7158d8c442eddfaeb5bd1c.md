**Ziel:** Bauen Sie eine App, die funktional ähnlich ist wie diese: <a href="https://codepen.io/freeCodeCamp/full/obYBjE" target="_blank" rel="noopener noreferrer nofollow">https://codepen.io/freeCodeCamp/full/obYBjE</a>.

Erfüllen Sie die untenstehenden User Stories und bringen Sie alle Tests zum Bestehen. Verwenden Sie beliebige Bibliotheken oder APIs, die Sie benötigen. Verleihen Sie ihr Ihren eigenen Stil.

**User Story:** Mir wird eine zufällige Reihe von Tastendrücken angezeigt.

**User Story:** Jedes Mal, wenn ich eine Reihe von Tastendrücken korrekt eingebe, sehe ich dieselbe Reihe von Tastendrücken, aber mit einem zusätzlichen Schritt.

**User Story:** Ich höre einen Ton, der zu jeder Taste gehört, sowohl wenn die Reihe von Tastendrücken abgespielt wird, als auch wenn ich persönlich eine Taste drücke.

**User Story:** Wenn ich die falsche Taste drücke, werde ich darüber informiert, und die Reihe von Tastendrücken beginnt erneut, damit ich das Muster wiederholen und es erneut versuchen kann.

**User Story:** Ich kann sehen, wie viele Schritte die aktuelle Reihe von Tastendrücken enthält.

**User Story:** Wenn ich neu starten möchte, kann ich eine Taste drücken, und das Spiel beginnt wieder mit einem einzelnen Schritt.

**User Story:** Ich kann im strengen Modus spielen, bei dem ich, wenn ich eine Taste falsch drücke, darüber informiert werde und das Spiel mit einer neuen zufälligen Reihe von Tastendrücken neu startet.

**User Story:** Ich kann das Spiel gewinnen, indem ich eine Reihe von 20 Schritten korrekt eingebe. Ich werde über meinen Sieg informiert, und das Spiel beginnt von vorne.
