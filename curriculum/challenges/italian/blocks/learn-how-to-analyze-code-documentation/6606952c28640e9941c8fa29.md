# --description--

Di solito, `with` si usa per indicare connessione o uso. Per esempio, `I write with a pen.` Tuttavia, in alcune frasi, `with` segue la parola con cui viene usato. Questo accade soprattutto quando vuoi enfatizzare l’oggetto o la cosa con cui qualcuno sta lavorando o che sta usando, specialmente in domande o affermazioni riguardanti tecnologia o lavoro.

Per esempio, in ambito tecnologico, potresti dire `This program I work with is challenging.` Qui `with` viene usato alla fine per sottolineare il programma con cui è difficile lavorare. Un altro esempio potrebbe essere una domanda come `What else is he working with?` dove `with` alla fine chiede di altri strumenti o tecnologie usati.
