# --description--

HTML 和 CSS 是两种协同工作的语言，用于创建你在互联网中看到的所有内容。HTML 是构建网页的未经处理的数据。所有的文本、链接、卡片、列表和按钮都是用 HTML 创建的。CSS 则为这些普通元素添加样式。HTML 将信息放置在网页上，CSS 则定位这些信息，赋予颜色，更改字体，并使其看起来很棒！

`HTML` 和 `CSS` 在创建网页中的主要职责是什么？

## --answers--

超文本标记语言 结构化内容，而 CSS 样式化元素。
