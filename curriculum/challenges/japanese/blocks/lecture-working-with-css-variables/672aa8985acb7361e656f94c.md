# --interactive--

CSSカスタムプロパティは、CSS変数とも呼ばれ、CSSの作成者が定義し、ドキュメント全体で再利用できる特定の値を含むエンティティです。これは、より効率的で保守しやすく柔軟なスタイルシートを実現する強力な機能です。

カスタムプロパティを宣言する構文はシンプルです。二つのダッシュ（`--`）に続けてプロパティ名を書きます。

```css
:root {
  --main-color: #3498db;
}
```

この例では、`--main-color`という名前のカスタムプロパティを`#3498db`という値で宣言しています。`:root`疑似クラスは、DOMツリーの最上位の親を表すため、グローバルなカスタムプロパティを宣言する際によく使われます。

カスタムプロパティを使うには、`var()`関数を使います。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
:root {
  --main-color: #3498db;
}

.button {
  background-color: var(--main-color);
}
```

:::

これにより、`button`クラスを持つ要素の背景色が`--main-color`に格納された値に設定されます。

カスタムプロパティの重要な特徴の一つは、CSSのカスケードに従うことです。つまり、特定の要素やコンテキストに対して再定義できます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="alert">This is an alert message.</div>
```

```css
.alert {
  --main-color: #e74c3c;
  background-color: var(--main-color);
}
```

:::

この場合、`alert`クラスを持つ要素は、グローバル定義を上書きして異なる`--main-color`の値を使います。

カスタムプロパティはフォールバック値もサポートしています。カスタムプロパティが定義されていないか無効な場合、フォールバック値を指定できます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="text">This is some text.</div>
```

```css
:root {
  --text-color: green;
}

.text {
  color: var(--text-color, green);
}
```

:::

ここでは、`--text-color`が定義されていなければ、`color`は`green`にデフォルト設定されます。

カスタムプロパティはテーマ対応デザインを作るのに特に便利です。異なるテーマ用に一連のプロパティを定義できます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="light-theme">
  <p>This is some light themed text.</p>
</div>
```

```css
:root {
  --bg-color: black;
}

body {
  background-color: var(--bg-color);
}

.light-theme {
  --bg-color: white;
  --text-color: black;
  background-color: var(--bg-color);
  color: var(--text-color);
  padding: 10px;
}

```

:::

テーマの切り替えは、`body`要素にクラスを追加または削除するだけで簡単にできます。

カスタムプロパティはメディアクエリと組み合わせてレスポンシブデザインを作ることもできます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <h2>Responsive Design</h2>
  <p>Resize the window to see the card adapt!</p>
</div>

```

```css
:root {
  --card-width: 90%;
  --card-bg: #f0f0f0;
  --card-padding: 1rem;
  --text-color: #333;
}

/* Tablet screens and up */
@media (min-width: 600px) {
  :root {
    --card-width: 70%;
    --card-bg: #e8f5e9;
    --card-padding: 1.5rem;
  }
}

/* Desktop screens and up */
@media (min-width: 1024px) {
  :root {
    --card-width: 50%;
    --card-bg: #d0f0ff;
    --card-padding: 2rem;
  }
}

body {
  font-family: system-ui, sans-serif;
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
  background-color: #fafafa;
}

.card {
  width: var(--card-width);
  background-color: var(--card-bg);
  padding: var(--card-padding);
  color: var(--text-color);
  border-radius: 8px;
  box-shadow: 0 2px 6px rgba(0,0,0,0.1);
  text-align: center;
  transition: all 0.3s ease;
}

```

:::

この方法は、スタイルシート全体で値を変更するのではなく、一箇所で値を変えられるため、レスポンシブレイアウトの保守性を高めます。特に頻繁に使われる複雑な値の繰り返しを大幅に減らせます。

カスタムプロパティは他のカスタムプロパティを参照することもできます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
:root {
  --primary-color: #3498db;
  --button-bg: var(--primary-color);
}

button {
  background-color: var(--button-bg);
}

```

:::

これにより、異なるスタイルの側面間で関係性を作り、サイト全体で一貫したテーマを保ちやすくなります。

カスタムプロパティには多くの利点がありますが、ブラウザの対応状況に注意が必要です。最新のブラウザではよくサポートされていますが、古いブラウザでは認識されない場合があります。最新機能を使う場合は必ずフォールバックの準備をしてください。

まとめると、CSSカスタムプロパティは、より動的で柔軟かつ保守しやすいスタイルシートを作る強力な方法を提供します。テーマ対応デザインの作成、レスポンシブレイアウトの簡素化、実行時のスタイル操作を可能にします。

ウェブ開発が進化し続ける中で、カスタムプロパティは開発者のツールキットにおいてますます重要な役割を果たし、適応性と効率性の高いCSSを作る新たな可能性を提供しています。
