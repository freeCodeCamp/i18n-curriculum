# --description--

Sarah sagt, dass ihre Arbeit als QA-Testerin nicht immer gleich ist. An manchen Tagen testet sie einen Teil der Software, an anderen Tagen etwas anderes.

Dieses Verständnis der Vielfalt hilft Ihnen zu erkennen, dass Berufe wie das QA-Testen von Tag zu Tag wechseln können und nicht auf eine einzige Routine beschränkt sind.
