# --description--

이 도전 과제는 발표 기법과 관련된 용어를 이해하고 사용하는 데 도움이 됩니다. 발표에서 `interactive` 요소는 청중의 참여를 유도하거나 발표 진행에 영향을 미치도록 합니다. 예를 들어, `The interactive quiz in the presentation kept everyone engaged.`

`instead of` 표현식은 한 가지를 선택하고 다른 것을 선택하지 않는 상황을 말할 때 사용합니다. 선택하지 않은 것을 나타냅니다. 예를 들어, `I had tea instead of coffee`라고 말하면 차를 마시고 커피는 마시지 않겠다는 뜻입니다.

`demo`는 demonstration의 줄임말로, 어떤 것이 어떻게 작동하는지 실질적으로 보여주거나 설명하는 것입니다. 발표 맥락에서는 `demo`가 제품이나 소프트웨어 기능을 실제로 보여주는 것을 포함할 수 있습니다. 예를 들어, `The software demo helped the clients understand its features.`라고 말하는 것과 같습니다.
