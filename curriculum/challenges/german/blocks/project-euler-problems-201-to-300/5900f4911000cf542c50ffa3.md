# --description--

Wir definieren ein pythagoreisches Polygon als ein konvexes Polygon mit den folgenden Eigenschaften:

- Es gibt mindestens drei Eckpunkte,
- keine drei Eckpunkte liegen auf einer Linie,
- jeder Eckpunkt hat ganzzahlige Koordinaten,
- jede Kante hat eine ganzzahlige Länge.

Für eine gegebene ganze Zahl $n$ definieren Sie $P(n)$ als die Anzahl der verschiedenen pythagoreischen Polygone, deren Umfang $≤ n$ ist.

Pythagoreische Polygone gelten als verschieden, solange keines eine Verschiebung eines anderen ist.

Es ist gegeben, dass $P(4) = 1$, $P(30) = 3655$ und $P(60) = 891045$.

Finden Sie $P(120)$.
