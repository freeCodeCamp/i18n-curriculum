# --description--

In previous lessons, you learned how to get the maximum and minimum values from a range of numbers like this:

```js
Math.max(1, 2, 3, 4, 5); // 5
Math.min(1, 2, 3, 4, 5); // 1
```

創建一個名爲 `maxNum` 的變量，將 `3, 125, 55, 24` 中最大數字賦值給它。 然後，將 `maxNum` 的值記錄到控制檯。

在下面創建一個名爲 `minNum` 的變量，將 `6, 90, 14, 90, 2` 中的最小數賦給它。 然後，將 `minNum` 的值記錄到控制檯。
