# --explanation--

`How they look` es una cláusula que comienza con `how`, la cual introduce un tema o sujeto. Por ejemplo:

`He didn't like how the icons looked on the screen.` – Esto significa que no le gustó la apariencia de los íconos.

Cláusulas como `how they look` se llaman **noun clauses**, y pueden actuar como el objeto de una oración.

En este contexto, `Let's start with how they look` significa que lo primero que hay que examinar es la forma en que las dos versiones se ven visualmente.
