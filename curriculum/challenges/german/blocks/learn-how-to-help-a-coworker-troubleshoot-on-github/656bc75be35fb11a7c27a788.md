# --description--

Der Ausdruck `Got it` wird häufig im Gespräch verwendet, um anzuzeigen, dass jemand etwas verstanden hat. Es ist eine lockere und schnelle Art zu bestätigen, dass Sie die Information erhalten und verstanden haben. Zum Beispiel:

Lucy: `You need to finish the report by tomorrow.`
Bob: `Got it!` 

`Got it` bedeutet, dass Bob die Anweisung versteht.
