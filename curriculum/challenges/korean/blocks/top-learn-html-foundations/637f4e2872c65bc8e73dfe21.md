# --description--

doctype를 선언한 후에는 `<html>` 요소를 제공해야 합니다. 이것은 문서의 루트 요소라고 하며, 문서 내의 모든 다른 요소가 이 요소의 자손 요소가 된다는 뜻입니다.

이것은 나중에 JavaScript로 HTML을 조작하는 방법을 배울 때 더 중요해집니다. 지금은 `html` 요소가 모든 HTML 문서에 포함되어야 한다는 것만 알아두세요.

`index.html` 파일로 돌아가서 `<html>` 요소를 여는 태그와 닫는 태그를 입력하여 추가해 봅시다.

```html
<!DOCTYPE html>
<html lang="en">
</html>
```

## lang 속성이란 무엇인가요?
`lang`는 해당 요소 내 텍스트 내용의 언어를 지정합니다. 이 속성은 주로 웹페이지의 접근성을 향상시키기 위해 사용됩니다. 예를 들어 화면 읽기 프로그램 같은 보조 기술이 언어에 맞게 조정되어 올바른 발음을 할 수 있도록 도와줍니다.
