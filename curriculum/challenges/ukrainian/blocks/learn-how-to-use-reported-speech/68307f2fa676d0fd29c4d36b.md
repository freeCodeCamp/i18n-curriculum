# --explanation--

`Was there` — це питальна форма (інтерогатив) `There was`, яка використовується, щоб дізнатися, чи щось існувало або відбулося в минулому. Вона змінює порядок підмета і дієслова, щоб утворити питання. Наприклад:

- `There was a problem with the layout.` (інструкція — ви кажете, що проблема сталася)  

- `Was there a problem with the layout?` (питання — ви питаєте, чи сталася проблема)

Це допомагає дізнатися про наявність або факт події в минулому.
