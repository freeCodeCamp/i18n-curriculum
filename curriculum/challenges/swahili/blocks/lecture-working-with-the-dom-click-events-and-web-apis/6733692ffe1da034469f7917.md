# --interactive--

Kutakuwa na nyakati ambapo utahitaji kubadilisha mitindo ya kipengele cha HTML ndani ya faili yako ya JavaScript. Mfano wa hili ni pale unapotakiwa kuficha au kuonyesha menyu wakati mtumizi anapobofya kitufe.

Katika funzo hili, utajifunza jinsi ya kubadilisha mitindo ukitumia sifa za `Element.style` na `Element.classList`.

Sifa ya `Element.style` ni sifa ya kusoma tu inayowakilisha mtindo ndani ya mstari wa kipengele. Unaweza kutumia sifa hii kupata au kuweka mtindo wa kipengele.

Hapa kuna mfano wa kipengele cha aya chenye ID ya `para`:

```html
<p id="para">This is a paragraph</p>
```

Unaweza kutumia sifa ya `style` kubadilisha rangi ya kipengele cha aya kuwa nyekundu:

:::interactive_editor

```html
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```js
const paraEl = document.getElementById("para");
paraEl.style.color = "red";
```

:::

Sifa ya `style` inaweza kutumika kuweka sifa nyingi za CSS, kama vile `color`, `background-color`, `font-size`, `font-weight`, na kadhalika.

Njia nyingine ya kubadilisha mitindo ni kwa kutumia sifa ya `Element.classList`. Sifa ya `classList` ni sifa ya kusoma tu inayoweza kutumika kuongeza, kuondoa, au kubadilisha madarasa kwenye kipengele. Tuchunguze mifano michache.

Katika mfano huu, utaongeza darasa linaloitwa `highlight` kwenye kipengele cha aya chenye ID ya `para`. Katika CSS, tutaweka darasa la `highlight` kubadilisha rangi ya nyuma ya kipengele cha aya kuwa njano. Katika JavaScript, tutaongeza darasa la `highlight` kwenye kipengele cha aya kwa kutumia njia ya `classList.add()`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```css
.highlight {
  background-color: yellow;
}
```

```js
const paraEl = document.getElementById("para");
paraEl.classList.add("highlight");
```

:::

Unaweza pia kuongeza madarasa mengi kwenye kipengele kwa kuyapitia kama hoja kwa njia ya `classList.add()`:

```js
classList.add("class1", "class2", "class3");
```

Ikiwa unahitaji kuondoa darasa kutoka kwenye kipengele, unaweza kutumia njia ya `classList.remove()` kama ifuatavyo:

```js
classList.remove("highlight");
```

Ili kubadilisha darasa kwenye kipengele, unaweza kutumia njia ya `classList.toggle()`. Katika mfano huu, tuna kipengele cha `button` chenye ID ya `toggle-btn` na kipengele cha `nav` chenye ID ya `menu`. Ndani ya CSS, tumeongeza mitindo kwa menyu na darasa linaloitwa `show` litakalobadilisha sifa ya `display` ya `.menu` kuwa `block`. Ndani ya JavaScript, tunapata kipengele cha menyu na kipengele cha `button` kwa kutumia njia ya `getElementById()`. Kisha, tunaongeza msikilizaji wa tukio kwenye kipengele cha `button` utakao badilisha darasa la `show` kwenye kipengele cha `.menu` wakati kitufe kinapobofya:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button id="toggle-btn">Toggle Menu</button>

<nav id="menu" class="menu">
  <ul>
    <li>Home</li>
    <li>About</li>
    <li>Products</li>
  </ul>
</nav>
<script src="index.js"></script>
```

```css
.menu {
  display: none;
  background-color: lightgray;
  width: 50%;
  padding: 10px;
}

.menu.show {
  display: block;
}
```

```js
const menu = document.getElementById("menu");
const toggleBtn = document.getElementById("toggle-btn");

toggleBtn.addEventListener("click", () => menu.classList.toggle("show"));
```

:::

Mtumizi anapobofya kitufe, darasa la `show` litaongezwa kwenye kipengele cha `.menu`, na menyu itaonyeshwa. Ikiwa mtumizi atabofya kitufe tena, darasa la `show` litaondolewa kwenye kipengele cha `.menu`, na menyu itafichwa.

Kwa kutumia njia na sifa kama `style`, `add()`, `remove()`, na `toggle()`, unaweza kwa urahisi kubadilisha mitindo ya kipengele kwa JavaScript, ukitengeneza kurasa za mtandao zenye nguvu na zinazoshirikisha.
