# --interactive--

Un loop `for...of` si usa quando devi iterare sui valori di un iterabile. Esempi di iterabili sono gli array e le stringhe.

Ecco la sintassi base per un loop `for...of`:

```js
for (variable of iterable) {
  // code block to be executed
}
```

La variabile nell'esempio rappresenta il valore corrente dell'iterabile su cui si sta iterando.

Se hai un array di numeri, la variabile sarà il numero corrente nell'array. Se hai una stringa, la variabile sarà il carattere corrente nella stringa.

Diamo un'occhiata a qualche esempio per capire meglio come funziona il loop `for...of`.

Nel primo esempio abbiamo un array di numeri e vogliamo iterare su ogni numero e stamparlo nella console.

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

:::

Abbiamo creato una variabile chiamata `num` che rappresenterà il numero corrente nell'array. Alla prima iterazione, `num` sarà `1`, alla seconda iterazione `num` sarà `2`, e così via.

Dentro il loop, stiamo stampando il numero corrente nella console.

Ecco un altro esempio in cui abbiamo una stringa e vogliamo iterare su ogni carattere e stamparlo nella console.

:::interactive_editor

```js
const str = 'freeCodeCamp';

for (let char of str) {
  console.log(char);
}
```

:::

In questo esempio, abbiamo creato una variabile chiamata `char` che rappresenterà il carattere corrente nella stringa.

Ad ogni iterazione, il loop stamperà il carattere corrente nella console.

È importante notare che puoi usare `let` o `const` quando dichiari la variabile in un loop `for...of`.

Se però usi `const`, assicurati che il valore della variabile non cambi dentro il loop. Se cambia, otterrai un errore.

Ecco un esempio di uso di `const` che genera un errore:

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
  num = num + 1; // This will cause an error
}
```

In questo esempio, stiamo cercando di cambiare il valore di `num` dentro il loop. Poiché abbiamo dichiarato `num` con `const`, otterremo un errore. Quindi, se devi cambiare il valore della variabile dentro il loop, usa `let` invece.

Diamo un’ultima occhiata a un esempio con un array di oggetti.

:::interactive_editor

```js
const people = [
  { name: 'John', age: 30 },
  { name: 'Jane', age: 25 },
  { name: 'Jim', age: 40 }
];

for (const person of people) {
  console.log(`${person.name} is ${person.age} years old`);
}
```

:::

In questo esempio, abbiamo un array di oggetti chiamato `people`. Ogni oggetto ha una proprietà `name` e una proprietà `age`.

Quando iteriamo sull’array, creiamo una variabile chiamata `person` che rappresenterà l’oggetto corrente nell’array.

Dentro il loop, stiamo stampando un messaggio nella console.

Il primo messaggio sarà `John is 30 years old`, il secondo `Jane is 25 years old`, e il terzo `Jim is 40 years old`.

I loop `for...of` sono molto utili quando devi iterare sui valori di un iterabile come un array o una stringa. Sono anche facili da leggere e possono rendere il tuo codice più conciso.
