# --interactive--

Інструкція `break` використовується, щоб достроково вийти з циклу, тоді як інструкція `continue` використовується, щоб пропустити поточну ітерацію циклу і перейти до наступної.

Ось приклад використання інструкції `break` у циклі `for`:

:::interactive_editor

```js
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}
```

:::

У наведеному вище прикладі цикл починає рахунок з `0` і поки `i` менше за `10`, цикл продовжуватиме виконуватися.

Всередині циклу ми перевіряємо, чи дорівнює `i` значенню `5`. Якщо так, ми використовуємо інструкцію `break`, щоб достроково вийти з циклу. Якщо ні, ми виводимо значення `i` у консоль. Отже, вихідні дані коду виведуть числа `0`, `1`, `2`, `3` і `4`.

Інструкція `break` корисна, коли ви хочете достроково вийти з циклу на основі певної умови. Наприклад, якщо ви шукаєте конкретне значення в масиві, ви можете використати інструкцію `break`, щоб вийти з циклу, щойно знайдете це значення.

Іноді ви можете захотіти пропустити певну ітерацію циклу, не виходячи з нього повністю. Саме тут на допомогу приходить інструкція `continue`. Ось приклад використання інструкції `continue` у циклі `for`:

:::interactive_editor

```js
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}
```

:::

Як і раніше, ми ініціалізували `i` значенням `0` і маємо умову, що цикл виконуватиметься, поки `i` менше за `10`.

Всередині циклу, коли `i` дорівнює `5`, ми використовуємо інструкцію `continue`, щоб пропустити поточну ітерацію і перейти до наступної.

Вихідні дані цього коду виведуть числа `0`, `1`, `2`, `3`, `4`, `6`, `7`, `8` і `9`. Число `5` пропускається через інструкцію `continue`.

Ще одна річ, яку можна зробити з інструкціями `break` і `continue`, — це використовувати мітки, щоб вказати, який саме цикл ви хочете перервати або продовжити.

Це корисно, коли у вас вкладені цикли, і ви хочете контролювати потік зовнішнього циклу зсередини внутрішнього.

Ось приклад використання міток з інструкцією `break`:

:::interactive_editor

```js
outerLoop: for (let i = 0; i < 3; i++) {
  innerLoop: for (let j = 0; j < 3; j++) {
    if (i === 1 && j === 1) {
      break outerLoop;
    }
    console.log(`i: ${i}, j: ${j}`);
  }
}
```

:::

У цьому прикладі ми маємо зовнішній цикл `for` з міткою `outerLoop` і внутрішній цикл `for` з міткою `innerLoop`.

Коли `i` дорівнює `1` і `j` дорівнює `1`, ми використовуємо інструкцію `break` з міткою `outerLoop`, щоб достроково вийти з зовнішнього циклу. Це припинить виконання як внутрішнього, так і зовнішнього циклів.

Вихідні дані цього коду виведуть у консоль наступне:

```md
"i: 0, j: 0"
"i: 0, j: 1"
"i: 0, j: 2"
"i: 1, j: 0"
```

Зазвичай вам не доведеться використовувати мітки з інструкціями `break` і `continue`, але добре знати, що така можливість існує, якщо вона вам знадобиться.
