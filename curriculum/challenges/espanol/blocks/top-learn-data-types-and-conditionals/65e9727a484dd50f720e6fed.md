# --description--

En cualquier lenguaje de programación, necesitas comparar valores para tomar decisiones. Por ejemplo, si el clima está soleado, saldrás, de lo contrario te quedarás en casa. Otro ejemplo sería ver si un usuario tiene suficientes puntos para subir de nivel en un juego.

La estructura condicional `if-else` se usa para tomar decisiones en JavaScript. A menudo se utiliza con operadores de comparación para comparar valores y tomar decisiones basadas en los resultados.

Un ejemplo de una estructura if es:

```javascript
let x = 5;

if (x > 3) {
  console.log("x is greater than 3");
} else {
  console.log("x is less than or equal to 3");
}
```

En este ejemplo, la variable `x` se compara con el número `3` usando el operador `>`. Si `x` es mayor que `3`, se ejecutará el primer bloque de código. De lo contrario, se ejecutará el segundo bloque de código.
