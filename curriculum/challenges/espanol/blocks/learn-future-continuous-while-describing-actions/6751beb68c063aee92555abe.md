# --explanation--

`Notification` se refiere a un mensaje o alerta enviado para informar a las personas sobre algo importante. Por ejemplo:  

Recibimos una notificación sobre la próxima actualización de software. - Esto significa que se envió un mensaje para alertar a los usuarios sobre la actualización de software.

La `Maintenance window` es el tiempo programado reservado para actualizaciones del sistema, reparaciones o pruebas. Por ejemplo:  

`The website will be down during the maintenance window from 2 AM to 4 AM.` - Esto significa que el sitio web no estará disponible mientras se realiza el trabajo de mantenimiento durante el período especificado.

`Inform` significa compartir información o hacer que alguien sepa sobre algo. Por ejemplo:  

El equipo informará a los usuarios sobre el lanzamiento de la nueva función.
