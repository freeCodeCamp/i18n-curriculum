HTML na CSS zinahusiana zaidi na kuwasilisha taarifa. Hazitumiki kuandika mantiki ya programu. JavaScript, ambayo utajifunza katika sehemu inayofuata, hutumiwa kufanya kurasa za mtandao zifanye mambo. Kuna mengi unayoweza kufanya kwa kutumia HTML na CSS tu, na hakika utahitaji zote mbili. Katika mtaala wetu mzima, mafundisho yafuatayo yanazingatia kukupa zana unazohitaji kufanikiwa unapoingia kwenye maudhui ya JavaScript.
