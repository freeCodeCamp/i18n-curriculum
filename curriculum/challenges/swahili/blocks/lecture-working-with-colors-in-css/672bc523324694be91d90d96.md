# --interactive--

Unapopanga mtindo wa tovuti, mojawapo ya njia zinazobadilika na kueleweka kwa urahisi za kutumia rangi ni kwa kutumia mfano wa rangi wa HSL. HSL inasimama kwa Hue, Saturation, na Lightness — sehemu kuu tatu zinazobainisha rangi.

Mfano wa rangi wa HSL unawakilisha rangi kwa njia inayolingana zaidi na jinsi wanadamu wanavyotambua rangi. Katika funzo hili, tutaangalia ni nini mfano wa rangi wa HSL na jinsi kitendakazi cha `hsl()` kinavyotumika katika CSS.

Hue ni aina ya rangi, inayo wakilishwa kama pembe kwenye gurudumu la rangi, ambalo linaanzia `0` hadi `360` digrii. Kwa mfano, `0` digrii inawakilisha nyekundu, `120` digrii ni kijani, na `240` digrii ni buluu. Kwa kurekebisha hue, unabadilisha rangi tofauti kwenye mzunguko wa rangi.

Saturation inahusu ukali au usafi wa rangi. Inapimwa kwa asilimia kutoka `0%` (rangi isiyo na rangi kabisa, ya kijivu) hadi `100%` (nafasi yenye nguvu zaidi ya rangi hiyo). Rangi yenye saturation ya `100%` ni angavu kabisa, wakati rangi yenye saturation ya `0%` inaonekana kama kivuli cha kijivu.

Lightness inaamua jinsi rangi inavyokuwa nyepesi au giza, pia inapimwa kwa asilimia. Thamani ya lightness ya `0%` itatoa nyeusi, `50%` itakupa tone ya kawaida ya hue, na `100%` itasababisha nyeupe.

Katika CSS, kitendakazi cha `hsl()` kinatumika kufafanua rangi kwa kutumia mfano wa rangi wa HSL. Sintaksia ya kisasa hutumia thamani zilizoachwa nafasi:

```css
element {
  color: hsl(hue saturation lightness);
}
```

Pia unaweza kukutana na sintaksia ya zamani inayotumia koma kama mgawanyiko katika msimbo uliopo, ambayo bado inasaidiwa na vivinjari:

```css
element {
  color: hsl(hue, saturation, lightness);
}
```

Tuchambue kwa mfano:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
body {
  background-color: hsl(0 0% 1% / 1);
}

p {
  color: hsl(120 100% 50%);
}
```

:::

Katika kesi hii, hue ni `120` digrii, ambayo inalingana na kijani. Saturation ni `100%`, hivyo kijani kitakuwa angavu kabisa. Lightness ni `50%`, kwa hiyo iko kwenye tone yake ya kawaida — si giza sana wala nyepesi sana. Matokeo yake, rangi ya maandishi ya aya itakuwa kijani angavu.

Moja ya faida kuu za mfano wa rangi wa HSL ni asili yake inayoweza kueleweka kwa urahisi. Inafanya iwe rahisi kurekebisha ukali wa rangi au uangavu kwa kubadilisha thamani za saturation na lightness bila kubadilisha rangi kuu (hue).

Kwa mfano, ikiwa unataka kuunda vivuli tofauti au rangi nyepesi za rangi ile ile, unaweza tu kurekebisha thamani ya lightness.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="light">This is a light blue div.</div>
<div class="dark">This is a dark blue div.</div>
```

```css
div.light {
  background-color: hsl(240 100% 80%);
}

div.dark {
  background-color: hsl(240 100% 20%);
  color: hsl(0 0% 100%);
}
```

:::

Hapa, sehemu zote mbili za `div` zinatumia hue ile ile (`240` digrii, ambayo ni buluu), lakini moja ina lightness ya `80%` (kivuli nyepesi cha buluu), na nyingine ina lightness ya `20%` (kivuli giza cha buluu).

Kama mfano wa RGB, kitendakazi cha `hsl()` pia kinaunga mkono thamani ya alpha ya hiari kudhibiti uwazi. Unaongeza baada ya mgawanyiko wa `/` kama kigezo cha nne. Hii ndiyo sintaksia ya msingi:

```css
element {
  background-color: hsl(hue saturation lightness / alpha);
}
```

Tuchukulie mfano:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: hsl(0 100% 50% / 0.5);
}
```

:::

Msimbo huu utatoa `div` nyuma ya rangi nyekundu yenye uwazi wa nusu, ambapo hue imewekwa kuwa `0` digrii (nyekundu), saturation ni `100%`, lightness ni `50%`, na alpha ni `0.5` (uwazi wa 50%). Pia unaweza kubainisha alpha kama asilimia — kwa mfano, `hsl(0 100% 50% / 50%)` ni sawa na `hsl(0 100% 50% / 0.5)`.

Pia utaona mara nyingi kitendakazi cha `hsla()` katika CSS zilizopo. Hii ilikuwa njia ya awali ya kujumuisha uwazi katika rangi za HSL, kwa kutumia thamani zilizoachwa koma:

```css
element {
  background-color: hsla(hue, saturation, lightness, alpha);
}
```

Kwa mfano, `hsla(0, 100%, 50%, 0.5)` ni sawa ya zamani ya `hsl(0 100% 50% / 0.5)`. Zote zinasaidiwa na vivinjari, lakini sintaksia ya kisasa ya `hsl()` yenye mgawanyiko wa `/` sasa inapendekezwa zaidi.

Mfano wa rangi wa HSL ni muhimu hasa unapohitaji kuunda mipangilio ya rangi na kurekebisha vivuli au rangi nyepesi kwa urahisi.

Kwa sababu HSL inatenganisha hue kutoka saturation na lightness, unaweza kuweka mandhari ya rangi thabiti huku ukibadilisha lightness kwa ajili ya vivuli au mwangaza. Ngazi hii ya kubadilika inafanya HSL kuwa kipenzi kwa wabunifu na wasanidi programu.

Zaidi ya hayo, HSL hufanya thamani za rangi kuwa rahisi kusomeka na kueleweka ikilinganishwa na njia nyingine kama nambari za hex au RGB. Unaweza kuona kwa uwazi nia nyuma ya rangi na kurekebisha ipasavyo, hasa unapojaribu miundo tofauti.

Kwa mfano wa rangi wa HSL, unapata kubadilika zaidi katika kudhibiti tone za rangi, saturation, na lightness, na kufanya kuwa chombo chenye nguvu kwa kuunda miundo tajiri na yenye nguvu katika CSS.
