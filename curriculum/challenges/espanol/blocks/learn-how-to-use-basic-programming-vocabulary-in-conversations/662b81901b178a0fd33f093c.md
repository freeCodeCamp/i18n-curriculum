## --text--

¿Cómo declaras una variable en Python según Sophie?

## --answers--

Declara una variable nombrándola después de un operador.

### --feedback--

Esta elección es incorrecta, porque la variable no se nombra después, sino antes, de un operador.

---

Las variables se declaran usando un comando llamado `declare`.

### --feedback--

Python no requiere un comando especial como `declare` para crear variables; utiliza la asignación directamente.

---

Usas el nombre de la variable seguido del operador de asignación y un valor.

---

Las variables deben declararse escribiendo `variable` antes del nombre.

### --feedback--

Python no utiliza ninguna palabra clave como `variable` para declarar variables. Simplemente utiliza el nombre de la variable, el operador de asignación y el valor.
