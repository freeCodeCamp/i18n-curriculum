# --description--

数字の6を考えます。6の約数は1、2、3、6です。

1から6までのすべての数字は、6の異なる約数の和として表せます：

$1 = 1$、$2 = 2$、$3 = 1 + 2$、$4 = 1 + 3$、$5 = 2 + 3$、$6 = 6$。

ある数$n$が実用数（practical number）と呼ばれるのは、1から$n$までのすべての数字が$n$の異なる約数の和として表せる場合です。

差が6の連続する素数のペアはセクシーペア（"sex"はラテン語で「6」）と呼ばれます。最初のセクシーペアは(23, 29)です。

時々、トリプルペアと呼ばれるものが見つかります。これは、3つの連続したセクシー素数ペアで、それぞれのペアの2番目の数が次のペアの1番目の数になっているものです。

次の条件を満たす数$n$をエンジニアの楽園と呼びます：

- ($n - 9$, $n - 3$)、($n - 3$, $n + 3$)、($n + 3$, $n + 9$)がトリプルペアを形成し、
- $n - 8$、$n - 4$、$n$、$n + 4$、$n + 8$のすべてが実用数である

最初の4つのエンジニアの楽園の和を求めてください。
