# --description--

Sie haben vielleicht bemerkt, dass die Großbuchstaben trotz der Verschlüsselung unverändert geblieben sind. Das liegt daran, dass die Übersetzungstabelle keine Großbuchstaben enthält.

Um das zu beheben, müssen Sie die Strings, die an Ihren `str.maketrans()`-Aufruf übergeben werden, mit der `upper()`-Methode ändern:

```py
greet = "Hello"
print(greet.upper()) # HELLO
```

Aktualisieren Sie Ihren `str.maketrans()`-Aufruf, indem Sie an jedes Argument die Großbuchstabenversion des Arguments anhängen.
