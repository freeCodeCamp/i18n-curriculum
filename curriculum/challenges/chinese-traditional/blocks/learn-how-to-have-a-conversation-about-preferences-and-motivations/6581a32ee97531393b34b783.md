# --description--

`Encourage` 是指給予某人支援或信心去做某事。 例如，當你的朋友說，`You can solve this coding problem!` 他們是在鼓勵你。 

`Discourage` 是相反的意思；它表示讓某人感到較不自信或較不可能去做某事。如果那位朋友說，`That programming language is too hard for beginners`，那可能會讓你對學習它感到氣餒。

`Discouraging` 描述使你感到氣餒的事物。例如，`Finding a bug in your code after hours of work can be discouraging.`

`Encouraging` 是當某事物給你勇氣或希望。 例如，`Getting your first program to run is very encouraging.`
