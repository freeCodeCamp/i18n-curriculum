# --description--

Im Elements-Bereich können Sie die gesamte HTML-Struktur Ihrer Seite sehen. Sie können auf eines der Elemente in diesem Bereich klicken, um genau dieses Element auszuwählen. Alternativ können Sie auf das unten links blau hervorgehobene Symbol klicken und mit der Maus über ein beliebiges Element auf der Seite fahren.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/the-box-model/inspector-icon.png" alt="Eine Web-Entwicklungsumgebung mit geöffneten Entwickler-Tools, die ein blaues Inspektor-Symbol oben links hervorhebt, um Webseiten-Elemente auszuwählen und die HTML-Struktur sowie Stile anzuzeigen.">

Wenn ein Element ausgewählt ist, zeigt der Styles-Tab alle aktuell angewendeten Stile sowie alle Stile, die überschrieben werden (gekennzeichnet durch einen durchgestrichenen Text). Wenn Sie zum Beispiel mit dem Inspektor auf die Überschrift „Your Career in Web Development Starts Here“ auf der <a href="https://www.theodinproject.com/" target="_blank">Startseite von The Odin Project</a> klicken, sehen Sie auf der rechten Seite alle Stile, die das Element gerade beeinflussen, wie unten dargestellt:
