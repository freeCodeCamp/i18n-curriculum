Qui sotto c’è un codice per trovare il valore più piccolo da una lista di valori. Una riga contiene un errore che farà sì che il codice non funzioni come previsto. Qual è la riga?:

```python
smallest = None
print("Before:", smallest)
for itervar in [3, 41, 12, 9, 74, 15]:
    if smallest is None or itervar < smallest:
        smallest = itervar
        break
    print("Loop:", itervar, smallest)
print("Smallest:", smallest)
```
