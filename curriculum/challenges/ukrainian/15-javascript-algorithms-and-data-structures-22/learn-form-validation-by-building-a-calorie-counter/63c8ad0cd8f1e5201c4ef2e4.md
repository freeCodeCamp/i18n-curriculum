# --description--

Параметр `list` буде результатом селектора запиту, який повертатиме `NodeList`. `NodeList` — це список елементів, як масив. Він містить елементи, які відповідають селектору запиту. Вам потрібно буде ітерувати над цими елементами у списку.

У попередніх кроках ви навчилися як проходити по масиву за допомогою циклу `for`. Ви також можете використати цикл <dfn>for...of</dfn> для проходу по масиву та `NodeList`.

Цикл `for...of` використовується для ітерації над елементами в ітерованому об’єкті як масив. Змінна, оголошена в циклі, представляє поточний елемент, над яким ітерують.

```js
for (const element of elementArray) {
  console.log(element);
}
```

Create a `for...of` loop that loops through the `list`. Для назви змінної циклу використайте `const`, щоб оголосити змінну під назвою `item`.
