# --explanation--

Під час прослуховування частини діалогу Сари зверніть увагу на слово `if`. Це слово часто вводить причину проблеми, пояснюючи, що має статися, щоб щось інше відбулося.

Після знаходження `if` визначте частину речення, яка описує, що відбувається, коли ця умова не виконується (наслідок). У реченні Сари:

- **Наслідок:** `The auto-completion doesn't work`.

- **Причина**, введена `if`: `If the project isn't set up correctly`.

Розуміння цього зв’язку допомагає чітко пояснювати проблеми та їхні рішення.
