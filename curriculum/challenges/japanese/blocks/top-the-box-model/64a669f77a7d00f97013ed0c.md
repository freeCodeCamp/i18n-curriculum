# --description--

HTML と CSS の基本的な構文が理解できたので、ここから本格的に進めていきます。CSS で習得すべき最も重要なスキルは、ポジショニングとレイアウトです。フォントや色を変えることも重要なスキルですが、ウェブページ上で要素を正確に配置できることのほうがさらに重要です。実際、すべての要素がただ単に積み重なっているだけのウェブページがどれだけあるでしょうか？

ウェブページ上の要素を配置することは、いくつかの重要な概念を理解すればそれほど難しくありません。残念ながら、多くの学習者は JavaScript に進むために HTML と CSS の学習を急ぎ、これらの基本的な概念を見逃してしまいます。これがフラストレーションや苦痛、（そして面白い GIF）につながります。なぜなら、どんなに JavaScript のスキルがあっても、必要な場所に要素を配置できなければ意味がないからです。そういうわけで、さっそく始めましょう。
