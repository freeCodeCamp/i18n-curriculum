# --description--

`Make sure` означає перевірити або підтвердити, що щось зроблено правильно або що умова виконана. Це включає виконання необхідних кроків для забезпечення точності або завершення. Наприклад, `Please make sure you turn off the lights before you leave.`

`Follow instructions` означає дотримуватися вказівок або правил, наданих для виконання завдання. Це включає виконання кожного кроку у визначеному порядку та способом. Прикладом є `follow the instructions on the box to assemble the table`.
