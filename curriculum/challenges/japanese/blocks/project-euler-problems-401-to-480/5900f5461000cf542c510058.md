# --description--

$\varphi$ を黄金比とします：$\varphi = \frac{1+\sqrt{5}}{2}$。

驚くべきことに、すべての正の整数は、$\varphi$ のべきの和として表すことができ、この和の中で各べきは高々一度だけ使われるという条件を課しても可能です。

それでも、この表現は一意ではありません。

隣接する指数のべきが使われず、かつ表現が有限であることを要求することで、一意にすることができます。

例えば：

$2 = \varphi + \varphi^{-2}$ および $3 = \varphi^{2} + \varphi^{-2}$

この $\varphi$ のべきの和を表すために、負の指数が始まる位置を示す小数点を含む0と1の文字列を使います。これをフィジタル数の基数での表現と呼びます。

したがって、$1 = 1_{\varphi}$、$2 = 10.01_{\varphi}$、$3 = 100.01_{\varphi}$、$14 = 100100.001001_{\varphi}$ となります。フィジタル数の基数で表された1、2、14の文字列は回文の一方、3の文字列は回文ではありません（フィジタル小数点が中央の文字ではないためです）。

フィジタル表現が回文となる1000以下の正の整数の和は4345です。

フィジタル表現が回文となる $10^{10}$ 以下の正の整数の和を求めてください。
