# --description--

Ahora añadirás algunos elementos que eventualmente diseñarás para convertirlos en marcadores de color.

Primero, dentro del elemento `body`, añade un elemento `div` ya su atributo `class` dale el valor `container`. Asegúrate que el elemento `div` esté debajo del elemento `h1`.

# --hints--

Debe haber una etiqueta de apertura `div`.

```js
assert(code.match(/<div.*>/i));
```

Debe haber una etiqueta de cierre `div`.

```js
assert(code.match(/<\/div\s*>/i));
```

El elemento `div` debe estar dentro del elemento `body`.

```js
assert(document.querySelector('div')?.parentElement?.localName === 'body');
```

El elemento `div` debe estar después del elemento `h1`.

```js
assert.exists(document.querySelector('h1 + div'));
```

El elemento `div` debe tener un atributo `class` establecido en `container`.

```js
assert(document.querySelector('div')?.classList?.contains('container'));
```
