# --description--

`Object-oriented programming`（OOP）是一種基於 `objects` 概念的程式設計範式，`objects` 可以包含資料和程式碼：資料以欄（通常稱為屬性或屬性）形式存在，程式碼則以程序（通常稱為方法）形式存在。

`OOP` 的一個常見範例是在像 Java 或 Python 這樣的程式語言中建立類別。這個類別定義物件的結構，包括它們的屬性和方法。例如，`Car` 類別可能包含像 `color` 和 `model` 這樣的屬性，以及像 `drive()` 和 `brake()` 這樣的方法。

在此背景關係中，術語 `oriented` 意指對齊或指向特定目的或原則。

例如，freeCodeCamp 的 Responsive Web Design 課程被描述為 `project-oriented`，因為它的結構是圍繞完成專案，這表示教學方法專注於實務應用，而不僅僅是理論知識。
