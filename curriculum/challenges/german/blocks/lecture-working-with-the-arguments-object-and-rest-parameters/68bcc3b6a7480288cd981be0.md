# --interactive--

Wie Sie sich aus früheren Lektionen und Workshops erinnern, können Sie Funktionen mit einer bestimmten Anzahl von Parametern erstellen und diese Funktion mit Argumenten aufrufen.

Hier eine Erinnerung:

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call
getSum(3, 4); // 7
```

Aber was passiert, wenn eine Funktion mit mehr Argumenten aufgerufen wird, als sie definiert wurde zu akzeptieren?

:::interactive_editor

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call with extra argument
console.log(getSum(3, 4, 5)); // 7
```

:::

JavaScript wirft in diesem Fall keinen Fehler. Stattdessen ignoriert es das zusätzliche Argument und addiert einfach die Zahlen `3` und `4` zusammen. Funktionen, die eine variable Anzahl von Argumenten akzeptieren, werden als <dfn>variadische Funktionen</dfn> bezeichnet.

Wenn Sie mit variadischen Funktionen arbeiten, können Sie das `arguments`-Objekt nutzen. Dieses array-ähnliche Objekt enthält die Werte der an die Funktion übergebenen Argumente.

Hier ein Beispiel:

:::interactive_editor

```js
function logArgs() {
  for (const arg of arguments) {
    console.log(arg);
  }
}

logArgs(1, 2, 3);
// result:
// 1
// 2
// 3

logArgs("example"); // "example"
```

:::

Da das `arguments`-Objekt array-ähnlich ist, können Sie auf ein Argument an einem bestimmten Index so zugreifen:

:::interactive_editor

```js
function getArg() {
  return arguments[1];
}

console.log(getArg(2, 4, 6)); // 4
```

:::

Sie können auch die `length`-Eigenschaft so verwenden, um die Anzahl der Argumente zu erhalten, mit denen die Funktion aufgerufen wurde:

:::interactive_editor

```js
function getArgs() {
  return arguments.length;
}

console.log(getArgs("Example")); // 1
console.log(getArgs("Another", "Example")); // 2
```

:::

Obwohl das `arguments`-Objekt wie ein echtes Array wirkt, besitzt es keine eingebauten `Array`-Methoden wie `includes` oder `push`. Um Zugriff auf diese Methoden zu erhalten, müssten Sie das `arguments`-Objekt zuerst in ein echtes Array umwandeln, zum Beispiel mit `slice`, `Array.from()` oder dem Spread-Operator:

:::interactive_editor

```js
function hasCat() {
  return [...arguments].includes("cat");
}

console.log(hasCat("dog", "chicken", "cat")); // true
console.log(hasCat("dog", "chicken", "horse")); // false
```

:::

Obwohl es möglich ist, mit dem `arguments`-Objekt für variadische Funktionen zu arbeiten, verwenden moderne JavaScript-Anwendungen normalerweise die Rest-Parameter-Syntax. Mehr dazu lernen Sie in einer zukünftigen Lektion.

## --answers--

Das `arguments`-Objekt ist immer leer, es sei denn, es wird explizit initialisiert.
