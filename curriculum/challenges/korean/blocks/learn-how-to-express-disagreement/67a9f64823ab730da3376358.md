# --explanation--

누군가 의견을 물었을 때 동의하지 않으면 정중하고 분명하게 답해야 합니다. 예를 들어:

- `I don't think so.` - 간단하고 직접적으로 반대하는 방법입니다.

- `I'm not sure about that.` - 의심을 부드럽게 표현하는 방법입니다.

- `I don't really agree.` - 너무 강하지 않게 정중하게 반대하는 방법입니다.
