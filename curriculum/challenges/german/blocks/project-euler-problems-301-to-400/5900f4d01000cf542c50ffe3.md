# --description--

Sei $a_n$ die größte reelle Wurzel eines Polynoms $g(x) = x^3 - 2^n \times x^2 + n$.

Zum Beispiel ist $a_2 = 3.86619826\ldots$

Finden Sie die letzten acht Ziffern von $\displaystyle\sum_{i = 1}^{30} \lfloor {a_i}^{987654321}\rfloor$.

**Hinweis:** $\lfloor a\rfloor$ steht für die Abrundungsfunktion (floor function).
