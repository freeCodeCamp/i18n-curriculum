# --description--

Un `plugin` è un componente che aggiunge una funzionalità specifica a un programma per computer esistente. Quando un programma supporta `plugins`, permette la personalizzazione.  
Nel contesto di un IDE, `plugins` può fornire funzionalità aggiuntive, come nuovi strumenti per scrivere codice, eseguire il debug o migliorare l'interfaccia utente.

Per esempio, un formatter di codice in un IDE è spesso un `plugin`.

# --questions--

## --text--

Cosa significa `plugin` in un contesto di programmazione?

## --answers--

È il software principale che gira su un computer.

### --feedback--

Un `plugin` non è il software principale; è un componente aggiuntivo che migliora o estende le capacità del software principale.

---

È un tipo di virus informatico.

### --feedback--

Un `plugin` è un componente software legittimo, non un virus. Serve ad aggiungere funzionalità al software.

---

È un componente software che aggiunge funzionalità a un programma esistente.

---

È un'applicazione autonoma che non richiede altro software per funzionare.

### --feedback--

Un `plugin` non è autonomo; richiede un altro programma per funzionare e gli aggiunge funzionalità.
