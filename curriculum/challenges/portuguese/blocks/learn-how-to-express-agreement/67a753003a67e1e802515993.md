# --explanation--

`Collaborating` significa trabalhar com outras pessoas para alcançar um objetivo comum.É frequentemente usado em trabalho em equipe ou baseado em projetos.Por exemplo:

`The designers and developers are collaborating to improve the user experience.` – Isso significa que eles estão trabalhando juntos para tornar a interface mais amigável ao usuário. 

`Implementar` significa colocar um plano ou decisão em ação.Por exemplo:

`The team will implement a new authentication system next month.` – Isso significa que eles irão configurar e começar a usar uma nova forma para os usuários fazerem login.

`Front-end design` refere-se à parte de um site ou aplicativo com a qual os usuários interagem diretamente.Por exemplo:
  
O design front-end do app foi atualizado para torná-lo mais moderno e intuitivo. – Isso significa que os elementos visuais e interativos foram alterados para melhorar a experiência do usuário.
