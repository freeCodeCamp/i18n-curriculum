# --description--

如果你有兩個群組的元素共用部分樣式宣告怎麼辦？

```css
.read {
  color: white;
  background-color: black;
  /* several unique declarations */
}

.unread {
  color: white;
  background-color: black;
  /* several unique declarations */
}
```

我們的 `.read` 和 `.unread` 選擇器都共用 `color: white;` 和 `background-color: black;` 的宣告，但其他部分各自有幾個獨特的宣告。為了減少重複，你可以將這兩個選擇器以逗號分隔的列表方式群組在一起：

```css
.read,
.unread {
  color: white;
  background-color: black;
}

.read {
  /* several unique declarations */
}

.unread {
  /* several unique declarations */
}
```

上述兩個範例（有群組與無群組）將會有相同的結果，但第二個範例減少了宣告的重複，並且讓你更容易同時編輯兩個類別的 `color` 或 `background-color`。

你如何將單一規則套用到兩個不同的選擇器 `.red-box` 和 `.yellow-box`？

## --answers--

.red-box,
.yellow-box {
  width: 25px;
  height: 25px;
}
