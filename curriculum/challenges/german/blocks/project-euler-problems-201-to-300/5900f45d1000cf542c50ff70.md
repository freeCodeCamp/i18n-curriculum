# --description--

Für eine positive ganze Zahl $n$ sei $σ(n)$ die Summe aller Teiler von $n$, also z. B. $σ(6) = 1 + 2 + 3 + 6 = 12$.

Eine vollkommene Zahl ist, wie Sie wahrscheinlich wissen, eine Zahl mit $σ(n) = 2n$.

Definieren wir den Quotienten der Vollkommenheit einer positiven ganzen Zahl als $p(n) = \frac{σ(n)}{n}$.

Finden Sie die Summe aller positiven ganzen Zahlen $n ≤ {10}^{18}$, für die $p(n)$ die Form $k + \frac{1}{2}$ hat, wobei $k$ eine ganze Zahl ist.
