# --description--

アロー関数はJavaScriptで関数を書くよりモダンな方法です。従来の関数式と比べて、よりコンパクトな構文を提供します。アロー関数は`=>`構文を使って定義されます。

```js
const add = (a, b) => {
  return a + b;
}
```

上の例では、アロー関数`add`が2つのパラメータ`a`と`b`を受け取り、それらの合計を戻り値として返します。加算操作の結果を返すために`return`キーワードが使われています。

アロー関数が1つの式だけを持つ場合、波括弧`{}`と`return`キーワードは省略できます。その式は暗黙的に戻り値として返されます。

```js
const add = (a, b) => a + b;
```

上の例では、アロー関数`add`が2つのパラメータ`a`と`b`を受け取り、それらの合計を戻り値として返します。`return`キーワードと波括弧`{}`は省略されており、加算操作の結果が暗黙的に戻り値として返されています。
