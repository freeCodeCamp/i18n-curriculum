# --description--
開始的最簡單方法是創建一個包含 JavaScript 程式碼的 HTML 檔案。在你的電腦某處的檔案中輸入基本的 HTML 骨架：

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>
  <script>
    // Your JavaScript goes here!
    console.log("Hello, World!")
  </script>
</body>
</html>
```

將此檔案存檔並在網頁瀏覽器中開啟（你可以使用 <a href="https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer" target="_blank" >Visual Studio Code 上的 "Live Server"</a> 來做到這點！），然後透過在空白網頁上按右鍵並選擇「檢查」或「檢查元素」來開啟瀏覽器的主控台。在「開發人員工具」面板中找到並選取「主控台」標籤，你應該會看到我們的 `console.log` 述語的輸出。

> `console.log()` 是在瀏覽器的開發者主控台中列印內容的指令。你可以使用它將以下任何文章和練習的結果列印到主控台。我們鼓勵你跟著本課程及未來課程中的所有範例一起撰寫程式碼。

另一種在網頁中包含 JavaScript 的方式是透過外部腳本。這與將外部 CSS 文件聯結到你的網站非常相似。

```html
  <script src="javascript.js"></script>
```

JavaScript 檔案的擴充套件是 `.js`，類似樣式表的 `.css`。外部 JavaScript 檔案用於較複雜的腳本。

# --questions--

## --text--

哪個述語正確描述如何在 HTML 文件中包含 JavaScript？

## --answers--

JavaScript 只能內部包含在 HTML 主體的 `<script>` 頁籤中。

---

JavaScript 程式碼必須寫在 HTML 文件的 `<head>` 區段中，才能正確執行。

---

JavaScript 可以直接使用 `<script>` 頁籤包含在 HTML 檔案中，或透過 `<script src="javascript.js">` 頁籤外部連結。 

---

外部 JavaScript 檔案需要在 `<script>` 頁籤中使用特殊屬性，才能被網頁瀏覽器識別。
