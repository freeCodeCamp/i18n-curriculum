# --description--

`extension`는 더 큰 소프트웨어 애플리케이션에 특정 기능을 추가하는 소프트웨어 컴포넌트입니다. `Extensions`는 새로운 기능을 제공하거나 기존 기능을 향상할 수 있습니다.

예를 들어, 브라우저 `extension`는 광고 차단이나 비밀번호 관리 같은 특별한 기능을 웹 브라우저에 추가할 수 있습니다.

## --text--

프로그래밍 맥락에서 `extension`는 무엇을 의미하나요?

## --answers--

프로그램을 종료하는 명령어입니다.

### --feedback--

`extension`는 명령어가 아니라 기능을 향상하는 소프트웨어 애드온입니다.

---

기존 프로그램의 기능을 향상하는 소프트웨어 애드온입니다.

---

프로젝트를 완료하기 위해 주어진 추가 시간을 의미합니다.

### --feedback--

`extension`가 다른 맥락에서는 추가 시간을 의미할 수 있지만, 프로그래밍에서는 소프트웨어 컴포넌트를 가리킵니다.

---

저장을 위해 사용되는 데이터 형식의 한 종류입니다.

### --feedback--

`extension`는 소프트웨어 기능을 향상하는 것이며, 데이터 저장 형식을 의미하지 않습니다.
