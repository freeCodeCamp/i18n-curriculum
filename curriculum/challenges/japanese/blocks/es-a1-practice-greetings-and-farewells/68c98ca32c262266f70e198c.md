# --explanation--

それぞれの別れの挨拶は、相手にいつまた会うかによって少しずつ意味が違います。

| Expresión          | Meaning         | When to use                                                   |
|--------------------|-----------------|----------------------------------------------------------------|
| `Adiós`            | Goodbye         | 一般的またはややフォーマルな別れの挨拶です。 |
| `Hasta luego`      | See you later   | また会う予定だけどすぐではない場合に使います。    |
| `Hasta mañana`     | See you tomorrow| 次の日に会う予定がある場合に使います。                |
| `Hasta pronto`     | See you soon    | すぐに会う予定がある場合に使います。                   |
| `Chao`             | Bye             | くだけた親しい間柄で使います。 |
