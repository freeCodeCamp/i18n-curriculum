# --interactive--

## أهمية HTML الدلالي

- **التدرج الهيكلي لعناصر العناوين**: من المهم استخدام عنصر العنوان الصحيح للحفاظ على التدرج الهيكلي للمحتوى. عنصر `h1` هو أعلى مستوى للعناوين وعنصر `h6` هو أدنى مستوى للعناوين.
- **عناصر HTML العرضية**: عناصر تحدد مظهر المحتوى. مثل العناصر المهجورة `center`، `big` و`font`.
- **عناصر HTML الدلالية**: عناصر تحمل معنى وهيكلًا. مثل `header`، `nav`، `figure`.

## عناصر HTML الدلالية

- **عنصر Header**: يُستخدم لتعريف القسم العلوي لوثيقة أو قسم.

:::interactive_editor

  ```html
  <header>
    <h1>CatPhotoApp</h1>
    <p>Welcome to our cat gallery.</p>
  </header>
  ```

:::

- **عنصر Main**: يُستخدم لاحتواء المحتوى الرئيسي لصفحة الويب.

:::interactive_editor

  ```html
  <main>
    <section>
      <h2>Cat Photos</h2>
      <p>Browse adorable cat pictures.</p>
    </section>
  </main>
  ```

:::

- **عنصر Section**: يُستخدم لتقسيم المحتوى إلى أقسام أصغر.

:::interactive_editor

  ```html
  <section>
    <h2>About Me</h2>
    <p>Hi, I am Jane Doe and I am a web developer.</p>
  </section>
  ```

:::

- **عنصر قسم التنقل (`nav`)**: يمثل قسمًا يحتوي على روابط تنقل.

:::interactive_editor

  ```html
  <nav>
    <ul>
      <li><a href="#photos">Photos</a></li>
      <li><a href="#videos">Videos</a></li>
    </ul>
  </nav>
  ```

:::

- **عنصر Figure**: يُستخدم لاحتواء الرسوم التوضيحية والمخططات.

:::interactive_editor

  ```html
  <figure>
    <img
      src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"
      alt="Two tabby kittens sleeping together on a couch."
    />
    <figcaption>Cats <strong>hate</strong> other cats.</figcaption>
  </figure>
  ```

:::

- **عنصر التأكيد (`em`)**: يميّز النص الذي يحمل تأكيدًا أو توكيدًا.

:::interactive_editor

  ```html
  <p>
    Never give up on <em>your</em> dreams.
  </p>
  ```

:::

- **عنصر النص الاصطلاحي (`i`)**: يُستخدم لتسليط الضوء على صوت أو مزاج بديل، أو مصطلحات اصطلاحية من لغة أخرى، أو مصطلحات تقنية، أو أفكار.

:::interactive_editor

  ```html
  <p>
    There is a certain <i lang="fr">je ne sais quoi</i> in the air.
  </p>
  ```

:::

تُستخدم الخاصية `lang` داخل وسم `i` المفتوح لتحديد لغة المحتوى. في هذه الحالة، اللغة ستكون الفرنسية. لا يشير عنصر `i` إلى أهمية النص، بل يظهر فقط أنه مختلف بطريقة ما عن النص المحيط.

- **عنصر الأهمية القوية (`strong`)**: يميّز النص الذي يحمل أهمية قوية.

:::interactive_editor

  ```html
  <p>
    <strong>Warning:</strong> This product may cause allergic reactions.
  </p>
  ```

:::

- **عنصر لفت الانتباه (`b`)**: يُستخدم لجذب الانتباه إلى نص ليس مهمًا لمعنى المحتوى. يُستخدم عادةً لتسليط الضوء على الكلمات المفتاحية في الملخصات أو أسماء المنتجات في المراجعات.

:::interactive_editor

  ```html
  <p>
    We tested several products, including the <b>SuperSound 3000</b> for audio quality, the <b>QuickCharge Pro</b> for fast charging, and the <b>Ecoclean Vacuum</b> for cleaning. The first two performed well, but the <b>Ecoclean Vacuum</b> did not meet expectations.
  </p>
  ```

:::

- **عنصر قائمة الوصف (`dl`)**: يُستخدم لتمثيل قائمة من مجموعات المصطلح والوصف.
- **عنصر مصطلح الوصف (`dt`)**: يُستخدم لتمثيل المصطلح الذي يتم تعريفه.
- **عنصر تفاصيل الوصف (`dd`)**: يُستخدم لتمثيل وصف المصطلح.

:::interactive_editor

  ```html
  <dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
  </dl>
  ```

:::

- **عنصر الاقتباس المقتبس (`blockquote`)**: يُستخدم لتمثيل قسم مقتبس من مصدر آخر. يحتوي هذا العنصر على خاصية `cite`. قيمة الخاصية `cite` هي عنوان URL للمصدر.

:::interactive_editor

  ```html
  <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
    "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
  </blockquote>
  ```

:::

- **عنصر الاقتباس (`cite`)**: يُستخدم لنسبة مصدر العمل المشار إليه بصريًا. يميّز عنوان المرجع.

:::interactive_editor

  ```html
  <div>
    <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
      "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
    </blockquote>
    <p>
      -Quincy Larson, <cite>How to learn to Code and Get a Developer Job [Full Book].</cite>
    </p>
  </div>
  ```

:::

- **عنصر الاقتباس المضمّن (`q`)**: يُستخدم لتمثيل اقتباس قصير مضمّن.

:::interactive_editor

  ```html
  <p>
    As Quincy Larson said,
    <q cite="https://www.freecodecamp.org/news/learn-to-code-book/">
      Momentum is everything.
    </q>
  </p>
  ```

:::

- **عنصر الاختصار (`abbr`)**: يُستخدم لتمثيل اختصار أو رمز مختصر. لمساعدة المستخدمين على فهم الاختصار أو الرمز، يمكنك عرض شكله الكامل، وهو وصف قابل للقراءة البشرية، باستخدام الخاصية `title`.

:::interactive_editor

  ```html
  <p>
    <abbr title="HyperText Markup Language">HTML</abbr> is the foundation of the web.
  </p>
  ```

:::

- **عنصر عنوان الاتصال (`address`)**: يُستخدم لتمثيل معلومات الاتصال.
- **عنصر الوقت (`time`)**: يُستخدم لتمثيل تاريخ و/أو وقت. تُستخدم الخاصية `datetime` لترجمة التواريخ والأوقات إلى صيغة قابلة للقراءة آليًا.

:::interactive_editor

  ```html
  <p>
    The reservations are for the <time datetime="20:00">20:00 </time>
  </p>
  ```

:::

- **خاصية تاريخ ISO 8601 (`datetime`)**: تُستخدم لتمثيل التواريخ والأوقات بصيغة قابلة للقراءة آليًا. الصيغة القياسية هي `YYYY-MM-DDThh:mm:ss`، حيث الحرف الكبير `T` هو فاصل بين التاريخ والوقت.
- **عنصر مرتفع (`sup`)**: يُستخدم لتمثيل نص مرتفع. من الاستخدامات الشائعة لعنصر `sup` الأسس، الحروف العليا، والأعداد الترتيبية.

:::interactive_editor

  ```html
  <p>
    2<sup>2</sup> (2 squared) is 4.
  </p>
  ```

:::

- **عنصر النص المنخفض (`sub`)**: يُستخدم لتمثيل النص المنخفض. ومن حالات الاستخدام الشائعة لعنصر النص المنخفض الصيغ الكيميائية والحواشي ومنخفضات المتغيرات.

:::interactive_editor

  ```html
  <p>
    CO<sub>2</sub>
  </p>
  ```

:::

- **عنصر شفرة مضمّنة (`code`)**: يُستخدم لتمثيل جزء من شفرة الكمبيوتر.
- **عنصر نص منسق مسبقًا (`pre`)**: يمثل نصًا منسقًا مسبقًا.

:::interactive_editor

  ```html
  <pre>
    <code>
      body {
        color: red;
      }
    </code>
  </pre>
  ```

:::

- **عنصر التعليق غير المعلن (`u`)**: يُستخدم لتمثيل جزء من النص المضمّن يجب عرضه بطريقة تشير إلى وجود تعليق غير نصي.

:::interactive_editor

  ```html
  <p>
    You can use the unarticulated annotation element to highlight
    <u>inccccort</u> <u>spling</u> <u>issses</u>.
  </p>
  ```

:::

- **عنصر توضيح روبي (`ruby`)**: يُستخدم لتوضيح النص بالنطق أو شرح المعنى. مثال على الاستخدام هو الطباعة في شرق آسيا.
- **عنصر قوس النظام البديل لروبي (`rp`)**: يُستخدم كنظام بديل للمتصفحات التي لا تدعم عرض توضيحات روبي.
- **عنصر نص روبي (`rt`)**: يُستخدم لتحديد النص الخاص بتوضيح روبي. يُستخدم عادةً للنطق أو تفاصيل الترجمة في الطباعة شرق آسيوية.

:::interactive_editor

  ```html
  <ruby>
    明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp>
  </ruby>
  ```

:::

- **عنصر شطب (`s`)**: يُستخدم لتمثيل محتوى لم يعد دقيقًا أو ذا صلة.

:::interactive_editor

  ```html
  <p>
    <s>Tomorrow's hike will be meeting at noon.</s>
  </p>
  <p>
    Due to unforeseen weather conditions, the hike has been canceled.
  </p>
  ```

:::

- **الروابط الداخلية**: تُستخدم للربط بقسم آخر من الصفحة باستخدام `href="#id"` على عنصر `a` ومنح العنصر الوجهة نفس `id`. يُستخدم هذا عادةً لروابط التخطي، جدول المحتويات، أو الصفحات الطويلة التي تحتوي على عدة أقسام.

:::interactive_editor

```html
<a href="#about-section">Go to About Section</a>

<section id="about-section">
  <h2>About</h2>
  <p>This is the about section of the page.</p>
</section>
```

:::
