# --interactive--

Подія `DOMContentLoaded` спрацьовує, коли весь HTML-документ завантажено та проаналізовано. Якщо у вас є зовнішні таблиці стилів або зображення, подія `DOMContentLoaded` не чекатиме їхнього завантаження. Вона чекатиме лише на завантаження HTML.

Це відрізняється від події `load`, яка чекає на завантаження всього, включно із зовнішніми таблицями стилів, зображеннями тощо.

Ось приклад синтаксису для використання події `DOMContentLoaded`:

```js
document.addEventListener("DOMContentLoaded", function () {
  console.log("DOM is loaded.");
});
```

Після завантаження DOM функція буде виконана, і повідомлення `DOM is loaded.` буде виведено в консоль.

Тепер розглянемо інший приклад із використанням події `DOMContentLoaded`. У цьому прикладі в HTML є зображення. Щоб оновити зображення, ми можемо створити функцію, яка змінює атрибути `src` і `alt` для зображення:

:::interactive_editor

```html
<h1>Image Change on DOM Loaded</h1>
<img
  id="example-img"
  src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg"
  alt="Cat relaxing"
/>

<script src="index.js"></script>
```

```js
function changeImg() {
  const img = document.getElementById("example-img");
  img.src =
    "https://cdn.freecodecamp.org/curriculum/responsive-web-design-principles/FCCStickers-CamperBot200x200.jpg";
  img.alt = "CamperBot sticker";
  console.log("image was just changed");
}

changeImg();
```

:::

Потім ми можемо перевірити, чи DOM ще завантажується, і додати слухача події для `DOMContentLoaded`. Якщо подія `DOMContentLoaded` вже спрацювала, ми можемо безпосередньо викликати функцію `changeImg`:

```js
if (document.readyState === "loading") {
  document.addEventListener("DOMContentLoaded", changeImg);
} else {
  console.log("DOMContentLoaded has already fired");
  changeImg();
}
```

Якщо ви запустите цей код і оновите сторінку, ви побачите, що зображення зміниться на нове після завантаження DOM. Це відбувається досить швидко, але ви помітите невелике мерехтіння під час зміни зображення.

Подія `DOMContentLoaded` корисна, коли ви хочете запустити JavaScript-код одразу після завантаження DOM, але до того, як будуть завантажені всі зовнішні ресурси, такі як зображення та таблиці стилів.
