# --description--

Herzlichen Glückwunsch! Sie haben die Lektionen zu React und Redux abgeschlossen. Es gibt noch einen letzten Punkt, den Sie beachten sollten, bevor Sie weitermachen. Normalerweise schreiben Sie React-Apps nicht in einem Code-Editor wie diesem. Diese Aufgabe gibt Ihnen einen Einblick, wie die Syntax aussieht, wenn Sie mit einem Dateisystem auf Ihrem eigenen Rechner arbeiten. Der Code sollte ähnlich aussehen, abgesehen von der Verwendung der `import`-Anweisungen (diese binden alle Abhängigkeiten ein, die Ihnen in den Aufgaben bereitgestellt wurden).

Schließlich erfordert das Schreiben von React- und Redux-Code in der Regel eine gewisse Konfiguration. Das kann schnell kompliziert werden. Wenn Sie daran interessiert sind, auf Ihrem eigenen Rechner zu experimentieren, ist die <a href="https://www.freecodecamp.org/news/install-react-with-create-react-app/" target="_blank" rel="noopener noreferrer nofollow">Create React App</a> vorkonfiguriert und sofort einsatzbereit.

Alternativ können Sie Babel als JavaScript-Preprozessor in CodePen aktivieren, React und ReactDOM als externe JavaScript-Ressourcen hinzufügen und dort ebenfalls arbeiten.
