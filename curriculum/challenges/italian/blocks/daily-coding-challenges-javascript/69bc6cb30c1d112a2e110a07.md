# --description--

Benvenuto alla 250ª sfida quotidiana!

Data una stringa codificata, decodificala usando una chiave di cifratura e restituiscila.

Per trovare la chiave:

- Guarda tutte le sfide quotidiane fino a oggi il cui numero di sfida è un multiplo di 25 (inclusa questa).
- Prendi la prima lettera di ciascuno di quei titoli di sfida e combinane una stringa. Se il titolo inizia con un carattere non alfabetico, trova la sua prima lettera.

Per decodificare il messaggio, esamina ogni lettera del messaggio codificato e:

  1. Guarda la lettera corrispondente nella chiave (ripeti la chiave se il messaggio è più lungo della chiave).
  2. Converti la lettera della chiave nel suo numero corrispondente: `"A"` = 1, `"B"` = 2, ..., `"Z"` = 26.
  3. Sposta la lettera codificata indietro nell'alfabeto di quel numero.
  4. Se lo spostamento va oltre `"A"`, torna indietro fino a `"Z"`.

Per esempio, se il messaggio codificato inizia con `"Y"` e la prima lettera della chiave è `"V"` (22), sposta `"Y"` indietro di 22 posizioni per ottenere `"C"`. Ripeti questo processo per ogni lettera per decodificare il messaggio completo.

- Solo le lettere vengono spostate, gli spazi vengono restituiti così come sono.
- Tutte le lettere date e restituite sono maiuscole.

```js
function decode(message) {
  const key = "VLHCGMDLNH";
  let keyIndex = 0;
  return message.split('').map(char => {
    if (char === ' ') return char;
    const shift = key[keyIndex % key.length].charCodeAt(0) - 64;
    keyIndex++;
    return String.fromCharCode(((char.charCodeAt(0) - 65 - shift + 26) % 26) + 65);
  }).join('');
}
```
