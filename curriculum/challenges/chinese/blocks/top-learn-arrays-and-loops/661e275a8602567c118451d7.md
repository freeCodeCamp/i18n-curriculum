# --description--

现在你已经了解了迭代值和数组的最常见方法，还有一些迭代数组的其他方法。数组有一个内置方法叫做 `map()`，用于通过将函数应用到原数组的每个元素来创建一个新数组。`map()` 方法不会改变原数组。`map()` 方法的语法如下：

```javascript
const array = [1, 2, 3, 4, 5];

const newArray = array.map((arrayValue) => {
  return arrayValue * 2;
});

console.log(newArray); // Output: [2, 4, 6, 8, 10]
```

`map()` 方法通过将函数 `(arrayValue) => { return arrayValue * 2; }` 应用于原始数组的每个元素来创建一个新数组。当你想转换数组的元素而不改变原始数组时，这尤其有用。
