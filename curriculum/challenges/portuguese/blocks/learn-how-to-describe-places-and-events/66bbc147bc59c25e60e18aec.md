# --explanation--

O tempo `Present Continuous` é usado para descrever ações acontecendo agora mesmo.Para expressar que algo **não está** acontecendo, a forma negativa é usada. 

Na frase `The live preview isn't showing`, `isn't` é uma contração de `is not`, tornando a frase negativa. 

Isso indica que a visualização ao vivo não está funcionando como esperado.Então, Tom está relatando que a visualização ao vivo atualmente não está visível no Visual Studio Code.
