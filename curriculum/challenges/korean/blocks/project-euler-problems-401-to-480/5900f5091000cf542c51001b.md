# --description--

격자점 ($x$, $y$)가 $x$, $y$ 그리고 $x + y$가 모두 양의 완전제곱수일 때, 이를 허용되지 않는 점이라고 부르겠습니다.

예를 들어, (9, 16)은 허용되지 않지만, (0, 4), (3, 1), (9, 4)는 그렇지 않습니다.

단위 북쪽 또는 동쪽 방향의 이동만을 사용하여 점 ($x_1$, $y_1$)에서 점 ($x_2$, $y_2$)까지 가는 경로를 생각해봅시다. 중간에 허용되지 않는 점이 하나도 없는 경로를 허용 경로라고 부르겠습니다.

$P(n)$을 (0, 0)에서 ($n$, $n$)까지의 허용 경로 수라고 할 때, $P(5) = 252$, $P(16) = 596\\,994\\,440$, 그리고 $P(1\\,000)\bmod 1\\,000\\,000\\,007 = 341\\,920\\,854$임을 확인할 수 있습니다.

$P(10\\,000\\,000)\bmod 1\\,000\\,000\\,007$을 구하세요.
