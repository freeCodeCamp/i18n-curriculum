# --description--

La propiedad `transform-origin` se utiliza para establecer el punto alrededor del cual se aplica una transformación CSS. Por ejemplo, cuando aplicas una transformación `rotate` (como harás más adelante en este proyecto), el `transform-origin` determinará alrededor de qué punto se rota el elemento.

Asigna al selector `.line` una propiedad `transform-origin` de `0% 0%`. Esto desplazará el punto de origen al `0%` desde la izquierda y `0%` desde arriba, estableciéndolo en la esquina superior izquierda del elemento.

# --hints--

Tu selector `.line` debe tener una propiedad `transform-origin` establecida en `0% 0%`.

```js
const transformOrigin = new __helpers.CSSHelp(document).getStyle('.line')?.transformOrigin;
assert.oneOf(transformOrigin, ['0% 0%', '0% 0% 0px']);
```
