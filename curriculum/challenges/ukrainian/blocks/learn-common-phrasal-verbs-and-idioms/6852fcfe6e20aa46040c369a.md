# --explanation--

`Perhaps` означає «можливо» або «ймовірно». Його використовують, коли хтось не зовсім впевнений, але хоче запропонувати щось. Наприклад:

`Perhaps we can try a new tool for project tracking.` – Це означає, що мовник пропонує ідею як можливість.

Префікс `un-` додається до прикметників, щоб утворити протилежне значення. У `unnecessary` основне слово — `necessary` (означає потрібний), а `un-` робить його значення «непотрібний». Інший приклад:

`It was an unfair decision.` – `Unfair` є протилежним до `fair`, означаючи несправедливий або нерівний.
