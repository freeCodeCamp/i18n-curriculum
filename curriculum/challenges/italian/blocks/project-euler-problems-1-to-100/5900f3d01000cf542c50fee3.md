# --description--

Se un riquadro contiene ventuno dischi colorati, composto da quindici dischi blu e sei dischi rossi, e si estraggono due dischi a caso, si può vedere che la probabilità di estrarre due dischi blu è:

$${P(BB)} = \frac{15}{21}×\frac{14}{20} = \frac{1}{2}$$

La successiva disposizione di questo tipo, per cui c'è esattamente il 50% di probabilità di estrarre due dischi blu a caso, è un riquadro contenente ottantacinque dischi blu e trentacinque dischi rossi.

Trovando la prima disposizione che contiene più di `limit` dischi in totale, determina il numero di dischi blu che il riquadro conterrebbe.

```js
function arrangedProbability(limit) {
  // Based on https://www.mathblog.dk/project-euler-100-blue-discs-two-blue/
  let blue = 15;
  let discs = 21;

  while (discs < limit) {
    const nextBlue = 3 * blue + 2 * discs - 2;
    const nextDiscs = 4 * blue + 3 * discs - 3;

    blue = nextBlue;
    discs = nextDiscs;
  }
  return blue;
}
```
