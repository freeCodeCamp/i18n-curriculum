# --description--

Un `hard drive` è un componente nei computer usato per archiviare dati in modo permanente. È come un grande archivio dove vengono conservati tutti i file digitali. Per esempio, quando salvi documenti, foto o musica sul tuo computer, questi vengono archiviati nel `hard drive`.

È diverso dall'archiviazione temporanea, che è come una scrivania dove le cose vengono tenute solo per un breve periodo mentre sono in uso.
