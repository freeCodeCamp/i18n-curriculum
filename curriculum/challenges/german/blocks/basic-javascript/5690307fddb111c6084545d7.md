# --description--

Die Reihenfolge ist wichtig in `if`, `else if`-Anweisungen.

Die Funktion wird von oben nach unten ausgeführt, daher sollten Sie darauf achten, welche Anweisung zuerst kommt.

Nehmen Sie diese beiden Funktionen als Beispiel.

Hier ist die erste:

```js
function foo(x) {
  if (x < 1) {
    return "Less than one";
  } else if (x < 2) {
    return "Less than two";
  } else {
    return "Greater than or equal to two";
  }
}
```

Und die zweite vertauscht nur die Reihenfolge der Anweisungen:

```js
function bar(x) {
  if (x < 2) {
    return "Less than two";
  } else if (x < 1) {
    return "Less than one";
  } else {
    return "Greater than or equal to two";
  }
}
```

Obwohl diese beiden Funktionen fast identisch aussehen, erhalten wir unterschiedliche Ausgaben, wenn wir beiden eine Zahl übergeben.

```js
foo(0)
bar(0)
```

`foo(0)` gibt den String `Less than one` zurück, und `bar(0)` gibt den String `Less than two` zurück.

# --instructions--

Ändern Sie die logische Reihenfolge in der Funktion so, dass sie in allen Fällen die korrekten Anweisungen zurückgibt.
