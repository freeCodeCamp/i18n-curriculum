# --description--

`Any` 通常用于疑问句中，询问某物是否存在或可用，而不指定数量。当你不知道是否有某物或有多少时，并且想要查找时，它非常有用。例如：

`Are there any updates?` - 这里，`any` 用于询问是否有更新，而不指定数量。 

`Any` 通常用于疑问句和否定语句中，表示一些甚至一点点。以下是你可能在不同上下文中使用它的方式：

- 在问题中：`Do you have any questions?`（无论有多个问题还是只有一个问题；你是在询问是否存在任何问题。）
- 在否定句中：`I don't have any money.`（你没有任何钱。）
- 使用 offers 和 请求：`Could I have any of these cookies?`（哪种饼干无所谓；任何饼干都可以。）
