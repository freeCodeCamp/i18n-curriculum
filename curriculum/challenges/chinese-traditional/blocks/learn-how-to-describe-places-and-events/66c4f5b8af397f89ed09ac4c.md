# --explanation--

想像你正在撰寫程式碼，並且想要快速找到正確的 `function` 或 `variable`，而不需要全部輸入。 

`Auto-completion` 透過根據你已輸入的內容建議你可能想要撰寫的內容來提供幫助。這使得編碼更快速並減少錯誤。
