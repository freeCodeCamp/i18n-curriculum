# --description--

`Open-source` 프로젝트는 소스 코드가 자유롭게 공개되어 수정하거나 공유할 수 있는 프로젝트입니다. 예를 들어, Firefox 웹 브라우저는 오픈 소스 프로젝트로, 누구나 어떻게 만들어졌는지 볼 수 있고 개선에 참여할 수 있습니다.

`To work on` 무언가에, 특히 전치사 `on` 와 함께 쓰일 때, 시간에 걸쳐 작업이나 프로젝트에 노력을 기울인다는 뜻입니다. 예를 들어, `I'm working on a new app` 라고 말하면 개발 중임을 의미합니다.
