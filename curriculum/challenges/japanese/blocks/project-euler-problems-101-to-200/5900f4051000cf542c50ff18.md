# --description--

ご存じの通り、方程式 $x^2 = -1$ は実数 $x$ に対して解を持ちません。

しかし、虚数 $i$ を導入すると、この方程式は2つの解を持ちます：$x = i$ と $x = -i$ です。

さらに進むと、方程式 ${(x - 3)}^2 = -4$ は2つの複素解を持ちます：$x = 3 + 2i$ と $x = 3 - 2i$ で、これらは互いに複素共役と呼ばれます。

形が $a + bi$ の数は複素数と呼ばれます。

一般に $a + bi$ と $a − bi$ は互いに複素共役です。ガウス整数とは、$a$ と $b$ の両方が整数である複素数 $a + bi$ のことです。

通常の整数もガウス整数に含まれます（$b = 0$ の場合）。

$b ≠ 0$ のガウス整数と区別するために、こうした整数を「有理整数」と呼びます。

ガウス整数が有理整数 $n$ の約数であるとは、その結果もガウス整数になる場合を指します。

例えば、5 を $1 + 2i$ で割ると、次のように簡単化できます。

分子と分母に $1 + 2i$ の複素共役である $1 − 2i$ を掛けます。

結果は次の通りです：

$$\frac{5}{1 + 2i} = \frac{5}{1 + 2i} \frac{1 - 2i}{1 - 2i} = \frac{5(1 - 2i)}{1 - {(2i)}^2} = \frac{5(1 - 2i)}{1 - (-4)} = \frac{5(1 - 2i)}{5} = 1 - 2i$$

したがって、$1 + 2i$ は 5 の約数です。

ただし、$1 + i$ は 5 の約数ではありません。なぜなら：

$$\frac{5}{1 + i} = \frac{5}{2} - \frac{5}{2}i$$

また、ガウス整数 ($a + bi$) が有理整数 $n$ の約数であれば、その複素共役 ($a − bi$) も $n$ の約数です。実際、5 は実部が正の約数を6つ持ちます：{1, 1 + 2i, 1 − 2i, 2 + i, 2 − i, 5}。

以下は最初の5つの正の有理整数に対するすべての約数のテーブルです：

| n | 実部が正のガウス整数の約数 | これらの約数の和 s(n) |
|---|-----------------------------|-----------------------|
| 1 | 1                           | 1                     |
| 2 | 1, 1 + i, 1 - i, 2          | 5                     |
| 3 | 1, 3                        | 4                     |
| 4 | 1, 1 + i, 1 - i, 2, 2 + 2i, 2 - 2i, 4 | 13            |
| 5 | 1, 1 + 2i, 1 - 2i, 2 + i, 2 - i, 5 | 12                |

実部が正の約数に関して、$\displaystyle\sum_{n=1}^5 s(n) = 35$ となります。

$1 ≤ n ≤ {10}^5$ の範囲で、$\displaystyle\sum_{n = 1}^{{10}^5} s(n) = 17924657155$ です。

では、$\displaystyle\sum_{n=1}^{{10}^8} s(n)$ はいくつでしょうか？
