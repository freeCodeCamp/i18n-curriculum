# --description--

`One of these days` は、近いうちに何かをする予定だけど、正確な日時はまだ決まっていないときに使うフレーズです。「近いうちに」や「近い将来に」と言う意味合いです。例えば：

`One of these days, I'll visit Paris.` - これはパリを訪れたいと思っていて、近いうちに行きたいけれど、まだ具体的な日付はわからないという意味です。
