# --explanation--

구동사는 전치사나 부사와 결합해 새로운 의미를 만드는 동사입니다. 대화의 이 부분에서는 `to send something`와 `to send something over`의 차이를 볼 수 있습니다.

`To send something`는 무언가를 한 장소나 사람에게 옮기는 것을 의미하는 반면, `to send something over`는 사람이나 장소에 전달하는 개념을 더하며, 보통 비공식적이거나 디지털 방식입니다. 각각의 예를 살펴보세요:

- `I'll send the letter tomorrow.` - 무언가를 옮긴다는 기본적인 표현입니다.

- `Can you send over the file by email?` - 보통 짧은 거리나 디지털로 *누군가에게* 무언가를 보내는 것을 강조합니다. `over`가 추가되어 일상 대화에서 더 자연스럽게 들립니다.
