## --text--

Qual frase usa corretamente `across` para descrever uma localização?

## --answers--

`He walked across the code to find the error.`

### --feedback--

`Across` é usado para espaço físico, não para coisas abstratas como código.

---

`She sent the report across her thoughts.`

### --feedback--

Esta não é uma forma natural ou correta de usar `across` para descrever uma localização.

---

`The conference room is across the hall from the kitchen.`

---

`They discussed the topic across the lunch break.`

### --feedback--

Este uso de `across` está relacionado ao tempo, não à localização física.

## --video-solution--

3

# --explanation--

`Across` é usado para descrever algo do outro lado de um espaço ou área.

Na frase correta, mostra que a sala de conferências está localizada do lado oposto do corredor em relação à cozinha.
