# --interactive--

Створювати плавні анімації на вебсторінці може бути складно, особливо якщо ви не впевнені, як правильно працювати з таймінгом. Гарна новина в тому, що API `requestAnimationFrame()` робить це простіше.

`requestAnimationFrame()` — це метод, який дозволяє запланувати наступний крок вашої анімації перед наступним оновленням екрану, що забезпечує плавний і приємний візуальний ефект.

Наступне оновлення екрану — це момент, коли браузер оновлює візуальне відображення вебсторінки. Це відбувається кілька разів на секунду, зазвичай близько 60 разів (або 60 кадрів на секунду) на більшості дисплеїв.

Щоб використати метод `requestAnimationFrame()`, потрібно просто викликати його і передати функцію зворотного виклику:

```js
requestAnimationFrame(callback);
```

Виклик `requestAnimationFrame()` має відбуватися всередині функції, яка керує анімацією, наприклад `animate()`, разом із функцією для оновлення анімації, традиційно званою `update()`:

```js
function animate() {
 // Update the animation...
 // for example, move an element, change a style, and more.
 update();
 // Request the next frame
 requestAnimationFrame(animate);
}
```

Функція `update()` — це те місце, де відбувається магія. Усередині неї ви можете змінювати все, що хочете анімувати. Наприклад, оновлювати стиль або змінювати позицію елемента:

```js
function update() {
 element.style.transform = `translateX(${position}px)`;
 position += 2;
}
```

Що врешті запускає анімацію — це виклик `requestAnimationFrame()` із передачею функції `animate`, цього разу поза функцією `animate`:

```js
requestAnimationFrame(animate);
```

Цикл триватиме, доки ви його не зупините.

Тепер подивімося на інший приклад. HTML для цього прикладу — це елемент `div` з текстом `freeCodeCamp is Awesome`. CSS робить `div` прямокутником і приховує все, що виходить за межі вікна перегляду зліва або справа. JavaScript-код рухає прямокутник `2px` вправо при кожному виклику `requestAnimationFrame(animate)` з функцією `animate` як функцією зворотного виклику:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div id="rect" class="rect">freeCodeCamp is Awesome</div>
<script src="index.js"></script>
```

```css
body {
  overflow-x: hidden;
}

.rect {
  display: flex;
  align-items: center;
  justify-content: center;
  width: 400px;
  height: 250px;
  border-radius: 5px;
  background-color: #1b1b32;
  color: #f5f6f7;
  font-size: 2rem;
  position: absolute;
}
```

```js
const rect = document.getElementById("rect");

let position = 0;

function update() {
  // Move the rectangle 2px to the right
  rect.style.left = position + "px";
  position += 2;

  if (position > window.innerWidth) {
    // Move the rectangle just outside the left side of the screen
    position = -rect.offsetWidth;
  }
}

function animate() {
  update();

  //request the next frame
  requestAnimationFrame(animate);
}

// Start the animation
requestAnimationFrame(animate);
```

:::

Результатом у браузері буде анімована титульна картка, що пливе по екрану.
