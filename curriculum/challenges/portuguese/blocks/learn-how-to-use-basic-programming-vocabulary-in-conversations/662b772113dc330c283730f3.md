# --description--

Em programação, `declare variables` significa criá-las e definir seu tipo e valor inicial. Em Python, você pode `declare a variable` simplesmente atribuindo um valor a ela, pois a linguagem não exige definição explícita de tipo para declaração de variável.

Declarar uma variável em programação significa criá-la e dar um nome a ela.
