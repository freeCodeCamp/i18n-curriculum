# --description--

JavaScriptには文字列、数値、ブール値などさまざまなデータ型があることはご存知ですね。比較演算子を使ってこれらのデータ型を比較し、その結果に基づいて処理を分岐させることができます。

JavaScriptで使える比較演算子は次の通りです：

| 演算子 | 説明                                   | 例           |
|--------|----------------------------------------|--------------|
| ==     | 等しい（値の比較）                     | x == y       |
| !=     | 等しくない                             | x != y       |
| >      | より大きい                            | x > y        |
| <      | より小さい                            | x < y        |
| >=     | 以上                                 | x >= y       |
| <=     | 以下                                 | x <= y       |

変数への代入で比較を使うには、`==` 演算子を使うことができます。例えば：

```javascript
let x = 5;
let y = 5;

let result = x == y;

console.log(result); // true
```
