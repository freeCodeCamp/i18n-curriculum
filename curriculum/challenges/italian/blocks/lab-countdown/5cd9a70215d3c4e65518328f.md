# --description--

In questo laboratorio, costruirai una `countdown` funzione che prende un numero di partenza e genera una lista per il conto alla rovescia.

Quando chiami `countdown(n)`, dovrebbe restituire un array che inizia da `n` e conta alla rovescia di `1` fino a raggiungere `1`.

Per esempio:

- `countdown(5)` dovrebbe restituire `[5, 4, 3, 2, 1]`
- `countdown(1)` dovrebbe restituire `[1]`

Se il numero di partenza è inferiore a `1`, non c'è nulla da contare alla rovescia, quindi restituisci un array vuoto:

- `countdown(0)` dovrebbe restituire `[]`
- `countdown(-3)` dovrebbe restituire `[]`

Per completare il laboratorio, devi costruire il risultato usando la ricorsione e non devi usare loop (`for`, `while`), `Array.from()` o metodi di iterazione degli array (`forEach`, `map`, `filter`, `reduce`). Ogni chiamata dovrebbe produrre il proprio array risultato (non usare variabili globali per archiviare il conto alla rovescia).

**Obiettivo:** Soddisfa le storie utente qui sotto e fai passare tutti i test per completare il laboratorio.

**Storie utente:**

1. Devi creare una funzione chiamata `countdown`.
2. La funzione `countdown` deve prendere un singolo argomento `n` (un numero).
3. Se `n` è inferiore a `1`, `countdown` deve restituire un array vuoto.
4. Altrimenti, `countdown` deve restituire un array contenente i numeri interi da `n` fino a `1` in ordine decrescente.
5. Devi usare la ricorsione (la funzione deve chiamare se stessa) e non devi usare loop di alcun tipo (`for`, `while`, `Array.from()` o metodi di ordine superiore come `forEach`, `map`, `filter`, `reduce`).
6. Ogni chiamata ricorsiva deve usare un valore più piccolo della chiamata precedente (per esempio, `n - 1`) così la funzione raggiunge il suo caso base.
7. La soluzione non deve usare variabili globali per archiviare o memorizzare nella cache l'array risultato.
8. Chiamare `countdown` più volte con input diversi deve sempre restituire il risultato corretto e indipendente.

function countdown(n) {

  }

  return [n, ...countdown(n - 1)];
}
