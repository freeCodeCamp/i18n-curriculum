# --explanation--

`Not very` + adjetivo é frequentemente usado para suavizar uma descrição negativa. Ao invés de dizer diretamente que algo foi `bad` ou `ineffective`, os falantes podem dizer `not very effective` para soar mais educados ou menos severos. Por exemplo:

- `The meeting was not very helpful.` – Esta é uma forma mais suave de dizer que a reunião não ajudou a equipe de forma alguma.

- `His presentation wasn't very clear.` – Isso é mais suave do que dizer que foi confuso ou ruim.
