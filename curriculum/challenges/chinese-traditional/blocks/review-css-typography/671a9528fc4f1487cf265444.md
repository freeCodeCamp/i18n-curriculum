## 排版介紹

- **定義**：排版是選擇合適字型和格式，使文字在視覺上吸引人且易於閱讀的藝術。 「字型」指的是個別字元的設計和排列方式。
- **字型定義**：字型是一組字元、數字和符號的整體設計與風格。它就像是一個字體家族的藍圖。
- **字型定義**：字型是字體的特定變體，具有特定的特徵，例如大小、粗細、樣式和寬度。
- **襯線字型**：這種字型具有經典風格，字元末端帶有小線條。襯線字型通常用於列印材料，如書籍。
- **無襯線字型**：這種字型看起來較為現代，字元末端沒有小線條。無襯線字型常用於數位設計，因為它們在螢幕上易於閱讀。一些範例包括 Helvetica、Arial 和 Roboto。
- **字型粗細**：這是字元的粗細度，包括細體、常規、粗體和黑體。
- **字型樣式**：這是字元的傾斜和方向，例如斜體和傾斜體。
- **字型寬度**：這是字元所佔用的水平空間，例如壓縮和擴展。
- **Baseline**：這是大多數字元所依附的虛擬線。
- **大寫字母高度**：這是大寫字母的高度，從基線量測到頂部。
- **X-height**：這是小寫字母的平均高度，不包括升部和降部。
- **Ascenders**：這些是小寫字母中超出 x-height 的零件，例如字母 `h`、`b`、`d` 和 `f` 的頂部。
- **Descenders**：這些是小寫字母中延伸到基線以下的零件，例如 'y'、'g'、'p' 和 'q' 的尾巴。
- **字距調整**：這是如何調整特定字元對組之間的間距，以提升它們的可讀性和美觀度。例如，減少字母 A 和 V 之間的間距。
- **Tracking**：這是調整一個區塊中所有字元之間間距的方式。它本質上是字元之間的距離。它會影響文字的密集度與開闊度。
- **行距**：這是文字行之間的垂直間距。它是從一行的基線測量到下一行的基線。
- **字型的最佳實務**：你應該選擇清晰且簡單的字型，使你的設計易於理解。這對於你網站的主要文字尤其重要。如果字型易於閱讀，使用者更有可能與你的內容互動。你應該最多使用兩到三種字型，以建立視覺一致性。使用過多字型會使文字更難閱讀，並削弱你的品牌識別度。這也可能影響使用者體驗，因為會增加網站的載入時間。你可以使用字型大小來為標題、副標題、段落及其他元素創建視覺階層體系。例如，網頁上的主標題應該使用較大的字型，接著是字型大小較小的副標題。這將為階層體系中的每個元素賦予特定的字型大小，幫助使用者透過視覺導覽結構。

  
## CSS `font-family` 屬性

- **定義**：字型家族是一組具有共同設計的字型。所有屬於同一家族的字型都基於相同的核心字體，但它們在樣式、粗細和寬度上也有變化。你可以以優先權從高到低的順序，用逗號分隔來指定多個字型家族。這些替代字型將作為備用選項。你應該始終在 `font-family` 列表的最後包含一個泛型字型家族。

```css
font-family: Arial, Lato;
```

- **常見字型家族**：以下是 CSS 中常用的一些字型家族：serif、sans-serif、monospace、cursive、fantasy

## 網頁安全字型

- **定義**：這些字型是非常可能安裝在電腦或裝置上的字型子集。當瀏覽器必須算繪字型時，它會嘗試在使用者系統中尋找字型檔案。但如果找不到該字型，通常會回退到預設系統字型。

- **網頁安全字型**：
  - 無襯線字型常用於網頁開發，因為它們的字元末端沒有小「腳」或線條，因此在螢幕上易於閱讀。一些網頁安全的無襯線字型範例有：Arial、Verdana 和 Trebuchet MS。
  - 相較之下，襯線字型的字元末端確實有小「腳」，因此它們通常用於傳統列印。網頁安全的襯線字型包括：Times New Roman 和 Georgia。

## At-rules 與 `@font-face` At-rule

- **定義**：At-rules 是提供瀏覽器指令的述語。你可以使用它們來定義樣式表的各種面向，例如媒體查詢、keyframes、字型臉（font faces）等。
- **`@font-face`**：這允許你透過指定字型檔案、格式及其他重要屬性，如粗細和樣式，來定義訂製字型。為了使 `@font-face` at-rule 有效，你還需要指定 `src` 屬性。此屬性包含對字型資源的引用。

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2"),
       url("path/to/font.woff"),
       url("path/to/font.otf");
}
```

- **字型格式**：對於每個字型資源，你也可以指定格式。這是選用的。它是瀏覽器關於字型格式的提示。如果省略格式，資源將會被下載，並在下載後偵測格式。如果格式無效，資源將不會被下載。可能的字型格式包括 `collection`、`embedded-opentype`、`opentype`、`svg`、`truetype`、`woff` 和 `woff2`。

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
       url("path/to/font.otf") format("opentype"),
       url("path/to/font.woff") format("woff");
}
```

- **`woff` 和 `woff2`**：`woff` 代表「Web Open Font Format」。它是一種由 Mozilla 與 Type Supply、LettError 及其他組織合作開發的廣泛使用字型格式。`woff` 和 `woff2` 之間的差異在於用來壓縮資料的演算法。
- **OpenType**：這是一種由 Microsoft 和 Adobe 開發的可縮放電腦字型格式，允許使用者存取字型中的額外功能。它廣泛應用於主要作業系統。
- **`tech()`**：這用於指定字型資源的技術。這是選用的。

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
       url("path/to/font.otf") format("opentype") tech(color-COLRv1),
       url("path/to/font.woff") format("woff");
}
```

## 使用外部字型

- **定義**：外部字型是未直接包含在你的專案檔案中的字型檔案。它們通常託管在獨立的伺服器上。若要在你的專案中包含這些外部字型，你可以在 CSS 中使用 `link` 元素或 `@import` 述語。
- **Google Fonts**：這是 Google 的一項服務，提供一個字型群集，其中許多字型是專為網頁開發設計的。

以下是使用 `link` 元素的範例：

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap" rel="stylesheet">
```

```css
.roboto-thin {
  font-family: "Roboto", sans-serif;
  font-weight: 100;
  font-style: normal;
}
```

以下是使用 `@import` 述語的範例：

```css
@import url('https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap');
```

- **Font Squirrel**：這是另一個你可以用來為你的專案添加訂製外部字型的熱門資源。

## `text-shadow` 屬性

- **定義**：此屬性用於為文字添加陰影。你需要指定 X 和 Y 偏移量，分別表述陰影相對於文字的水平和垂直距離。這些數值為必填。X 偏移量和 Y 偏移量的正值會分別將陰影向右和向下移動，而負值則會將陰影向左和向上移動。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Typography Shadow</p>
```

```css
  p {
    text-shadow: 3px 2px;
  }
```

:::

- **陰影顏色**：你也可以透過在偏移量之前或之後指定此值來自訂陰影的顏色。如果未指定顏色，瀏覽器會自動決定顏色，因此通常最好明確設定。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Colored Shadow</p>
```

```css
p {
  text-shadow: 3px 2px #00ffc3;
}
```

:::

- **模糊半徑**：模糊半徑為選用，但會讓陰影看起來更平滑且更細緻。模糊半徑的預設值為零。數值越高，模糊越大，代表陰影變得越淡。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Blurred Shadow</p>
```

```css
p {
  text-shadow: 3px 2px 3px #00ffc3;
}
```

:::

- **套用多重文字陰影**：文字可以有多個陰影。陰影會以層疊方式套用，從前到後，第一個陰影位於最上層。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Multiple Shadows</p>
```

```css
p {
  text-shadow:
    3px 2px 3px #00ffc3,
    -3px -2px 3px #0077ff,
    5px 4px 3px #dee7e5;
}
```

:::
