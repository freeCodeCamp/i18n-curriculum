# --description--

如你所見，flexbox 不僅僅是一個單一的 CSS 屬性，而是一整套屬性工具箱，你可以用來將物件放置在你需要的位置。其中一些屬性屬於 flex 容器，而有些則屬於 flex 項目。這是一個簡單但重要的概念。

flex 容器是任何具有 `display: flex` 的元素。flex 項目是直接位於 flex 容器內的任何元素。

## --text--

flex 容器和 flex 項目有什麼差別？

## --answers--

Flex 容器是具有 `display: flex` 的元素，而 flex 項目是位於 Flex 容器內的元素。

---

Flex 容器是 flex 項目中的元素，而 flex 項目定義整體佈局。

---

Flex 容器是使用 flexbox 的元素，而 flex 項目是行內元素。

---

Flex 容器是定義版面結構的元素，而 flex 項目則控制整體頁面設計。
