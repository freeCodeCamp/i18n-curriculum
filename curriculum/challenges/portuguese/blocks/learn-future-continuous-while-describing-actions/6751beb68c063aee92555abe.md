# --explanation--

`Notification` refere-se a uma mensagem ou alerta enviado para informar as pessoas sobre algo importante. Por exemplo:  

`We received a notification about the upcoming software update.` - Isso significa que uma mensagem foi enviada para alertar os usuários sobre a atualização do software.

`Maintenance window` é o tempo agendado reservado para atualizações, reparos ou testes do sistema. Por exemplo:  

`The website will be down during the maintenance window from 2 AM to 4 AM.` - Isso significa que o site não estará disponível enquanto o trabalho de manutenção estiver sendo realizado durante o período especificado.

`Inform` significa compartilhar informações ou avisar alguém sobre algo. Por exemplo:  

`The team will inform the users about the new feature rollout.` - Isso significa que a equipe informará os usuários sobre a introdução da nova funcionalidade.
