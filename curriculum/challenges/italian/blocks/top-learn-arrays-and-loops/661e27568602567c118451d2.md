# --description--

Per consultare gli elementi di un array, puoi usare il numero dell'indice. L'indice parte da 0, quindi il primo elemento di un array si trova all'indice 0, il secondo elemento all'indice 1 e così via.

Per esempio, per consultare il primo elemento dell'array `fruits`, puoi usare il seguente codice:

```javascript
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits[0]); // Output: apple
console.log(fruits[2]); // Output: orange
```

Se l'indice è fuori intervallo, JavaScript restituirà `undefined`. Questo significa che se provi a consultare `fruits[3]` nell'esempio sopra, restituirà `undefined`.
