# --explanation--

Die `Past Continuous`-Zeitform (`was/were` + Verb in der `-ing`-Form) beschreibt eine Handlung, die über einen Zeitraum in der Vergangenheit andauerte. Sie wird oft zusammen mit einem `Simple Past`-Ereignis verwendet, um zu zeigen, dass die andauernde Handlung unterbrochen wurde oder vor einem anderen Ereignis stattfand. Zum Beispiel:

`She was working on the report when the computer crashed.` – Die Handlung des Arbeitens war im Gange, als der Absturz (`Simple Past`) geschah.
