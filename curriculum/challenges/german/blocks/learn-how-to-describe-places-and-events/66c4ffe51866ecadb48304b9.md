# --explanation--

Eine `tag question` ist eine kurze Frage, die am Ende eines Satzes hinzugefügt wird, um etwas zu bestätigen oder zu überprüfen. Sie wird oft verwendet, wenn der Sprecher erwartet, dass der Zuhörer zustimmt. Zum Beispiel:

- `It's a nice day, isn't it?` – Der Sprecher erwartet, dass der Zuhörer zustimmt, dass es ein schöner Tag ist.

- `They are coming to the meeting, aren't they?` – Der Sprecher erwartet, dass der Zuhörer zustimmt, dass sie zum Treffen kommen.

Wenn Tom sagt, `Isn't it automatic?`, bittet er um Bestätigung, dass die automatische Vervollständigung automatisch erfolgen sollte.
