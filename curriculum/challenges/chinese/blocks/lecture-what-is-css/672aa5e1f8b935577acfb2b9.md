# --interactive--

CSS，全称为层叠样式表，是现代网页开发中一个关键的组件。它是一种用于为 `HTML` 应用样式的标记语言。简单来说，如果 `HTML` 是网页的结构，`CSS` 就是让它看起来美观的东西。

这是一个为导航栏设置样式的示例。 

**注意**：暂时不用担心理解 CSS 代码。你将在后续课程和研讨会中学习所有这些内容的工作原理。这只是为了让你了解使用一点点 CSS 可以做什么。 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<nav class="navbar">
    <ul class="nav-links">
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Services</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>
```

```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: 'Segoe UI', sans-serif;
  background-color: #f4f4f4;
}

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  background-color: #333;
  padding: 1rem 2rem;
  color: white;
}

.navbar .logo {
  font-size: 1.5rem;
  font-weight: bold;
}

.nav-links {
  list-style: none;
  display: flex;
  gap: 1.5rem;
}

.nav-links a {
  color: white;
  text-decoration: none;
  font-weight: 500;
  transition: color 0.3s ease;
}

.nav-links a:hover {
  color: #ff9800;
}
```

:::

CSS 的主要作用是将网页内容与其视觉呈现分离。 这种分离使网页开发者能够创建更灵活且易于维护的网站。

使用 CSS，你可以控制网页的版型、颜色、字体和整体视觉外观，而无需更改 HTML 结构。

让我们考虑一个简单的类比。如果你把网站看作一座房子，`HTML` 就是地基和框架，而 `CSS` 则是使房子在视觉上吸引人且独特的油漆、墙纸和装饰。

CSS 通过选择超文本标记语言元素并应用样式来工作。

这是一个示例，展示了未样式化的 `button` 元素和已样式化的 `button` 元素。要与此示例交互，你需要启用交互式编辑器。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<button class="regular-btn">Unstyled Button</button>
<button class="round-btn">Round Button</button>
```

```css

.round-btn {
  font-size: 1rem;
  font-family: 'Segoe UI', sans-serif;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease, transform 0.2s ease;
  background-color: #2ecc71;
  color: white;
  border-radius: 50px;
  padding: 0.6rem 1.6rem;
}

.round-btn:hover {
  background-color: #27ae60;
  transform: translateY(-2px);
}
```

:::

这些样式可以包括颜色、字体大小等属性。通过更改这些属性，你可以在不更改内容的情况下显著改变网页的外观。

CSS 最强大的方面之一是它能够创建响应式设计。

这意味着使用 CSS，你可以让你的网站在任何设备上看起来都很棒，无论是台式计算机、平板电脑还是智能手机。

CSS 允许你根据查看网站的设备的屏幕大小调整版型、字体大小和其他视觉元素。

尝试启用交互式编辑器并调整预览窗口的大小，以查看版型如何适应屏幕尺寸。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<header class="header">
    <h1>Welcome to My Responsive Site</h1>
    <p>This layout adapts to your screen size!</p>
</header>

<main class="content">
    <section class="box">Box 1</section>
    <section class="box">Box 2</section>
    <section class="box">Box 3</section>
</main>
```

```css
body {
  margin: 0;
  font-family: 'Segoe UI', sans-serif;
  background-color: #f9f9f9;
  color: #333;
}

.header {
  background-color: #4CAF50;
  color: white;
  padding: 2rem;
  text-align: center;
}

.content {
  display: flex;
  gap: 1rem;
  padding: 1rem;
  justify-content: center;
}

.box {
  flex: 1;
  min-width: 200px;
  background-color: #ddd;
  padding: 2rem;
  text-align: center;
  font-size: 1.2rem;
  border-radius: 8px;
  transition: background-color 0.3s;
}

.box:hover {
  background-color: #ccc;
}

/* Responsive: for tablets and below (≤ 768px) */
@media (max-width: 768px) {
  .content {
    flex-direction: column;
    align-items: center;
  }

  .box {
    width: 90%;
    font-size: 1.1rem;
  }
}

/* Responsive: for phones (≤ 480px) */
@media (max-width: 480px) {
  .header {
    padding: 1.5rem 1rem;
  }

  .header h1 {
    font-size: 1.5rem;
  }

  .header p {
    font-size: 1rem;
  }

  .box {
    font-size: 1rem;
    padding: 1.5rem;
  }
}
```

:::

CSS 的另一个重要特色是其层叠特性，这也是其名称中“层叠”一词的由来。

这意味着样式可以被继承和覆盖，从而允许样式具有层级结构。你将在后续课程中学习更多关于这如何工作的内容。

CSS 也支持使用外部样式表。

这意味着你可以将所有样式规则保存在单独的文件中，然后将其链接到多个 HTML 页面。

此特色极大地提升了网站的维护性，尤其是大型网站。你无需在每个单独页面上更改样式，只需在一个 `CSS` 文件中进行更改，即可影响所有链接的页面。

在网页开发领域，CSS 在创建视觉吸引力强、响应迅速且用户友好的网站中起着至关重要的作用。

它允许开发者将简单的 `HTML` 文档转换为引人入胜的网页体验，从而捕捉用户的注意力并增强他们与网页内容的互动。
