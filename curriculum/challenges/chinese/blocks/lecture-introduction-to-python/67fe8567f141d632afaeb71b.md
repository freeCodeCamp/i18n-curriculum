# --description--

在上一课中，你了解了什么是 Python 以及你可以用它做什么。现在，让我们看看如何在你的局部机器上设置 Python。

在 Windows 和 Mac 上安装 Python 最简单的方法是从官方 Python 网站下载安装程序。我们稍后还会讲解在 Linux 上运行 Python。

访问 `https://www.python.org/` 并将鼠标悬停在 "Downloads" 上。将出现一个模态窗口，显示适用于你的操作系统（OS）的当前 Python 版本。

我们将首先介绍如何在运行 macOS 的计算机上安装 Python：

- 点击显示当前 Python 版本（来自上一个模态框）的按钮，你将自动开始下载 `.pkg` 安装文件。
- `.pkg` 安装程序下载完成后，打开它，然后在打开的窗口中点击“Continue”。
- 继续点击 "Continue" 按钮，直到到达 "Installation Type" 部分。在那里，点击 "Install" 按钮。
- 如有必要，输入你的密码，然后开始安装。
- 之后，你应该收到一条祝贺消息，说明 Python 已成功安装。
- 点击 `Close` 按钮，即可完成！

你可以通过打开终端并运行 `python --version` 或 `python3 --version` 来验证安装。

你也可以通过在终端运行 `python` 或 `python3` 来打开 Python 解释器。

<dfn>终端</dfn>是一个基于文本的接口，允许你通过输入命令与计算机交互。每个操作系统都自带一个默认的终端应用程序。在 macOS 上，你可以使用 Terminal 应用程序。在 Windows 上，你可以使用 Command Prompt 或 PowerShell。在 Linux 上，每个桌面环境都有自己的默认终端应用程序，比如 GNOME Terminal 或 Konsole。

请注意，在某些较旧的 macOS 和 Linux 系统上，`python` 可能被保留用于 Python 2，而 `python3` 专门用于 Python 3。如果你运行 `python --version` 并看到类似 `Python 2.7.18` 的 Python 2 版本，那么你的操作系统可能依赖于某些用旧版本 Python 编写的软件。如果是这种情况，你应该使用 `python3` 来转发你的 Python 代码。Python 2 已经终止支持，不应再用于任何新开发。

要在 `Windows` 上安装 Python，请按照以下步骤操作：

- 访问 `https://www.python.org/`，并将鼠标悬停在“Downloads”上。你应该会看到一个模态窗口，显示“Download for Windows”以及带有当前 Python 版本的下载按钮。
- 点击版本号，你将自动开始下载 Windows 可执行文件（`.exe`）。
- 当你完成下载适用于 Windows 的 Python 安装程序后，双击它并按照说明操作。
- 当你看到选项 `Add python.exe to Path` 时，查看该选项，然后点击 `Install Now`。这样做以后会让你更轻松。


你可以通过打开像 PowerShell 这样的命令行终端并运行 `python --version` 来验证安装。你也可以通过运行 `python` 来打开 Python 解释器。

对于 Linux 上的 Python，大多数主要发行版如 Ubuntu、Debian 和 Fedora 都随附 Python。

只需打开终端并运行 `python --version`，或 `python3 --version`：

如果任一命令未显示 Python 版本，你可以在 `https://www.python.org` 查找适合你 Linux 版本的安装包，或在线查找适合你发行版的推荐安装方法。
