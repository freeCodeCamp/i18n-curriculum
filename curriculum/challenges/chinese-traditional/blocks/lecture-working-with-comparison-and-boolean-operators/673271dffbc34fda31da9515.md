# --interactive--

比較運算子允許你比較兩個值並傳回 `true` 或 `false` 的結果。然後你可以使用該結果來做出決策或控制你的程式流程。你會在 `if` 述語、迴圈以及許多其他需要根據特定條件做出決策的情況中使用比較。讓我們深入了解最常見的比較運算子並看看它們如何運作。

大於運算子，由一個直角括號（`>`）表述，檢查左邊的值是否大於右邊的值：

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a > b); // false
console.log(b > a); // true
```

:::

大於或相等運算子，由直角括號和等號（`>=`）表述，檢查左邊的值是否大於或相等於右邊的值：

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a >= b); // false
console.log(b >= a); // true
console.log(a >= c); // true
```

:::

小於運算子，由左尖括號（`<`）表述，運作方式類似於 `>`，但方向相反。它檢查左邊的值是否小於右邊的值：

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a < b); // true
console.log(b < a); // false
```

:::

小於或相等運算子，由左尖括號和等號（`<=`）表述，檢查左邊的值是否小於或相等於右邊的值：

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a <= b); // true
console.log(b <= a); // false
console.log(a <= c); // true
```

:::
