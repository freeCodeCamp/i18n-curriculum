# --description--

연결어 `so`는 앞서 말한 내용과 관련된 결과나 효과를 소개할 때 사용합니다. 보통 문장의 두 부분 사이에 원인과 결과 관계를 나타냅니다.

예를 들어, `It was raining, so we canceled the picnic.` 이 문장은 비 때문에 소풍이 취소되었음을 보여줍니다.

# --questions--

## --text--

어떤 문장이 `so`를 올바르게 사용하고 있나요?

## --answers--

`I was hungry so I am eating.`

### --feedback--

이 문장은 거의 `so`를 올바르게 사용했지만, 두 번째 부분의 시제가 첫 번째 부분과 일치해야 합니다. 올바른 문장은 `I was hungry, so I ate.`입니다.

---

`She is late to work, so she misses the meeting.`

### --feedback--

`so`의 사용은 결과를 암시하지만, 시제가 맞지 않습니다. 다음과 같이 해야 합니다: `She is late to work, so she will miss the meeting.`

---

`It started raining, so we took shelter.`

---

`He likes to read books, so he buys a lamp.`

### --feedback--

이 문장은 `so`를 올바르게 사용하지 않았습니다. 책 읽기를 좋아하는 것이 직접적으로 램프를 사는 결과가 아니기 때문입니다. 더 나은 사용 예는 `He likes to read at night, so he bought a lamp.`입니다.
