# --explanation--

El `First Conditional` se usa para describir situaciones reales y posibles en el futuro. Se forma con `if` + `Present Simple` (para la condición) y `will` + verbo base (para el resultado). Por ejemplo:

`If you debug the code, the app will run smoothly.` - Arreglar el código hará que la app funcione correctamente.

En la oración correcta, la condición es `If we don't back up regularly`, y el resultado es `we could lose important information`. Esto sugiere una posibilidad real.
