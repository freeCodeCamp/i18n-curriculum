# --description--

`automate` bedeutet, einen Prozess so zu gestalten, dass er von selbst abläuft, ohne dass eine Person eingreifen muss. In der Programmierung kann das bedeuten, Software zu verwenden, um Code zu schreiben, Anwendungen zu testen oder Fehler zu finden und zu beheben. Zum Beispiel ist eine Kaffeemaschine, die morgens von selbst angeht, um Kaffee zu machen, `automated`.

Das Wort `certain` wird verwendet, um über bestimmte Dinge oder Personen zu sprechen, aber nicht über alle. Wenn Sie zum Beispiel sagen `certain books are on sale`, bedeutet das, dass nur einige Bücher im Angebot sind, nicht alle.
