# --explanation--

`Might` é usado para expressar possibilidade ou incerteza.Sugere que algo é possível, mas não certo.Por exemplo:

- `The system might crash if too many users log in at the same time.` - Há uma possibilidade de falha no sistema, mas não é certa.

- `He might finish the coding task by the end of the day.` - Ele poderia terminar a tarefa, mas não é definitivo.
