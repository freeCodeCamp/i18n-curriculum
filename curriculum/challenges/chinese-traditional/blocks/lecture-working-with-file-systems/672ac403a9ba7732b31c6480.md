# --description--

檔案根據其內容和結構進行分類。檔案型別決定了檔案如何被程式開啟和處理。

我們根據副檔名來識別檔案型別。副檔名是檔名結尾的三或四個字母後綴。你會在網頁應用程式中找到的三種主要檔案型別是 HTML、CSS 和 JavaScript。

HTML 檔案包含網頁應用程式的結構和內容。它們具有 `.html` 副檔名。

CSS 檔案定義樣式。它們具有 `.css` 副檔名。

而 JavaScript 檔案則添加更多進階功能和互動性。它們的副檔名為 `.js`。

當你開發網頁應用程式時，你也需要包含影像。這些是最廣泛使用的影像檔案型別。

JPEG，全名為「Joint Photographic Experts Group」，以其高效壓縮而聞名。它非常適合用於照片和影像。然而，壓縮是有損的，這表示為了減少檔案大小，部分影像品質會流失。

PNG，代表「可攜式網路圖形」，即使經過壓縮仍能保留影像品質。這會導致較大的檔案。PNG 支援透明度，非常適合具有銳利邊緣的影像，例如標誌和圖示。

GIF 影像格式支援動畫和透明度，但調色盤有限。GIF 代表「Graphics Interchange Format」。

最後，我們有 SVG，代表「可縮放向量圖形」。它是一種向量影像格式。這些影像可以依需要無損品質地縮放。

太好了。既然你已經了解了影像格式，現在讓我們來看看一些影片和音訊格式。

MP3 是一種以高效壓縮著稱的音訊格式。它是一種有損格式，因此在壓縮過程中會遺失部分音訊資料，以使這些檔案更小。

MP4 是最常見的影片格式之一。它提供良好的壓縮效果，並支援多種音訊和影片編碼、字幕以及詮釋資料。

MOV 是 Apple 開發的 QuickTime 多媒體檔案格式，主要與 QuickTime Player 相關。

其他常見的音訊和影片檔案類型有 WAV，一種保持音訊原始品質的無損音訊格式，以及 WebM，一種高品質的開放原始碼影片格式。

如果你需要自訂你的網頁應用程式的字型，你也會使用字型格式。這三種是最廣泛使用的。

TTF，全名為「TrueType Font」，是一種與不同作業系統相容的字型格式。

WOFF 是一種現代字型格式，專為網頁開發用途設計。它代表「Web Open Font Format」。這些檔案較小，因為它們具有更好的壓縮效果。它們也可以儲存詮釋資料，包括授權資訊。

而 WOFF2 是 WOFF 的後繼者，具有更高效的壓縮和效率。

如果你需要將多個資料夾和檔案群組，你也可以創建壓縮檔案。最廣泛使用的壓縮格式是 ZIP。ZIP 提供無損壓縮，因此在過程中不會遺失任何資料。它在各種作業系統和軟體應用中廣泛支援。

最後，讓我們談談文件。在網頁應用程式中，你通常會找到一個名為 `README` 的檔案，裡面包含關於應用程式的資訊，例如如何使用它、如何安裝它、其授權條款，以及如何貢獻。

它們通常以一種稱為 Markdown 的檔案格式撰寫。使用 Markdown，你可以創建具有標題、副標題、連結、圖片、列表等結構的文件。Markdown 檔案具有 `.md` 或 `.markdown` 擴充套件。

你可以在這裡找到 freeCodeCamp GitHub 倉庫的 `README` 檔案：https://github.com/freeCodeCamp/freeCodeCamp/blob/main/README.md

你可以使用 Markdown 創建這個詳細的結構和格式。

你在職業生涯中肯定會閱讀和撰寫許多 `README` 檔案。了解這些常見的檔案型別對於網頁開發至關重要。 

從 HTML、CSS 和 JavaScript 的核心區塊，到影像、影片 和 音訊 格式，每種檔案型別在建立互動式網頁應用程式中都有重要的角色。
