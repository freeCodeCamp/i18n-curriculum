# --explanation--

`Make sure` significa tomar uma ação para confirmar que algo está correto ou concluído.Por exemplo:  

`Please make sure the report is submitted before the deadline.` – Isso significa verificar e garantir que o relatório seja entregue no prazo.  

`Reliable` significa que algo ou alguém pode ser confiável para desempenhar bem ou para ser consistente ao longo do tempo.Por exemplo:

`The server is reliable; it has been running without any issues for months.` - Isso significa que o servidor funciona de forma consistente e não causa problemas.

`More reliable` significa que algo se tornou ainda mais confiável ou seguro.Por exemplo:

`The new software update makes the app more reliable.` - Isso significa que o aplicativo se tornou mais confiável ou estável após a atualização.
