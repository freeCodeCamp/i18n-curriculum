# --interactive--

في درس سابق، تعلمت كيف تبدأ العمل مع التدرجات الخطية والدائرية. في هذا الدرس، سنغوص أعمق في كيفية استخدام خصائص CSS لإنشاء تصاميم أكثر جاذبية وتعقيدًا بصريًا.

تسمح التدرجات في CSS بإنشاء انتقالات سلسة بين لونين أو أكثر محددين. غالبًا ما تُستخدم لإضافة عمق وملمس وجاذبية بصرية لتصاميم الويب دون الحاجة إلى ملفات صور.

يدعم CSS نوعين رئيسيين من التدرجات: التدرجات الخطية والتدرجات الدائرية.

تُنشئ التدرجات الخطية مزيجًا تدريجيًا بين الألوان على طول خط مستقيم. يمكنك التحكم في اتجاه هذا الخط والألوان المستخدمة. إليك مثالًا أساسيًا على تدرج خطي:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

هذا يُنشئ تدرجًا ينتقل من الأحمر على اليسار إلى الأزرق على اليمين. تحدد المعلمة `to right` اتجاه التدرج. يمكنك استخدام اتجاهات أخرى مثل `to left`، `to top`، `to bottom`، أو حتى تحديد زوايا مثل `45deg`.

يمكنك أيضًا استخدام عدة توقفات ألوان في التدرج الخطي:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="multi-color-gradient"></div>
```

```css
.multi-color-gradient {
  background: linear-gradient(to right, red, yellow, blue);
  height: 40vh;
}
```

:::

هذا يُنشئ تدرجًا ينتقل من الأحمر إلى الأصفر ثم إلى الأزرق، موزعة بالتساوي عبر العنصر.

أما التدرجات الدائرية، فتُنشئ تدرجات دائرية أو بيضاوية تشع من نقطة مركزية. إليك مثالًا:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::

هذا يُنشئ تدرجًا دائريًا يبدأ بالأحمر في المركز وينتقل إلى الأزرق عند الحواف. يمكنك أيضًا تحديد شكل وحجم التدرجات الدائرية:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="custom-radial-gradient"></div>
```

```css
.custom-radial-gradient {
  background: radial-gradient(ellipse at top left, red, blue);
  height: 40vh;
}
```

:::

هذا يُنشئ تدرجًا بيضاويًا يبدأ من الزاوية العلوية اليسرى للعنصر. تسمح كل من التدرجات الخطية والدائرية بتحديد توقفات الألوان في مواقع دقيقة. على سبيل المثال:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="precise-gradient"></div>
```

```css
.precise-gradient {
  background: linear-gradient(to right, red 20%, yellow 40%, blue 80%);
  height: 40vh;
}
```

:::

هذا يُنشئ تدرجًا حيث يشغل الأحمر أول `20%`، ثم ينتقل إلى الأصفر عند `40%`، وأخيرًا إلى الأزرق عند `80%`.

يمكن استخدام التدرجات في أي مكان تستخدم فيه صورة في CSS. هذا يجعلها متعددة الاستخدامات لإنشاء خلفيات، أزرار، أو حتى تأثيرات نصية عند دمجها مع خصائص CSS أخرى.

تذكر، بينما تعزز التدرجات الجاذبية البصرية لموقعك، من المهم ضمان تباين كافٍ لإمكانية الوصول، خاصة عند استخدام التدرجات خلف النص.
