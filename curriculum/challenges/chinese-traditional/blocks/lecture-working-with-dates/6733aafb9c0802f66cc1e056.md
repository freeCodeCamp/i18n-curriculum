# --interactive--

JavaScript 中的日期和時間並不總是容易處理。在專業的程式碼庫中，你可能會使用已經為你解決許多這些問題的函式庫。

但有時你需要使用 JavaScript 內建的 `Date` 物件。

在本課程中，我們將介紹 `Date` 物件的基礎知識，以及你在使用 JavaScript 處理日期和時間時會用到的一些最常見的方法。

`Date` 物件是 JavaScript 中的內建物件，允許你操作日期和時間。以下是建立新的 `Date` 物件的基本語法：

```js
const now = new Date();
```

`new` 關鍵字用於創建 `Date` 物件的新實體，然後將該 `Date` 物件指定給變數 `now`。如果你將 `now` 的值記錄到主控台，你會看到根據執行程式碼的電腦系統時鐘的當前日期和時間。

以下是你將 `now` 的值記錄到主控台時會看到的範例回應：

:::interactive_editor

```js
const now = new Date();
console.log(now);
```

:::

時間使用的是 24 小時制格式，所以 `14:30` 是 `2:30 PM`。`GMT-0700` 是時區偏移量，而 `Pacific Daylight Time` 是時區名稱。

你也可以透過提供年、月、日、時、分、秒和毫秒的引數，將特定的日期和時間傳入 `Date` 物件。

這裡是一個使用特定日期和時間建立新的 `Date` 物件的範例：

:::interactive_editor

```js
const specificDate = new Date("July 4, 1776 12:00:00");

console.log(specificDate);
```

:::

當你需要處理特定的日期和時間，而非目前的日期和時間時，這非常有用。輸入必須始終為字串，且格式應該能被 `Date` 物件識別。

要取得目前的日期和時間，你可以使用 `Date.now()` 方法，該方法會傳回自 `January 1, 1970, 00:00:00 UTC` 起的毫秒數。這稱為 Unix 紀元時間。

Unix 紀元時間是在電腦系統中表述日期和時間的常見方式，因為它是一個整數，可以輕鬆儲存和操作。UTC 代表協調世界時，是全球調整時鐘和時間的主要時間標準。

如果你需要根據當前日期取得一個月份中的日期，你可以使用 `getDate` 方法。以下是使用 `getDate` 方法的範例：

:::interactive_editor

```js
const now = new Date();
const date = now.getDate();
console.log(date); 
```

:::

在此範例中，我們使用 `Date` 物件創建一個新的日期實體並將其指定給變數 `now`。接著，我們在 `now` 物件上呼叫 `getDate` 方法以取得當月的日期，並將其指定給變數 `date`。最後，我們將 `date` 的值記錄到主控台，這將輸出當前的日期。

`getDate` 將傳回介於 `1` 和 `31` 之間的整數值，取決於當月的日期。如果日期無效，將傳回 `NaN`（非數字）。

要取得月份，你可以像這樣使用 `getMonth` 方法：

:::interactive_editor

```js
const now = new Date();
const month = now.getMonth();
console.log(month);
```

:::

月份是從零開始計算，所以 `January` 是 `0`，`February` 是 `1`，依此類推。如果月份無效，將會傳回 `NaN`。

如果你需要取得完整年份，則可以像這樣使用 `getFullYear` 方法：

:::interactive_editor

```js
const now = new Date();
const year = now.getFullYear();
console.log(year); 
```

:::

如果年份無效，將傳回 `NaN`。

`Date` 物件上還有許多其他方法，包括 `getHours`、`getMinutes`、`getSeconds` 等等。我鼓勵你透過 Mozilla 的文件或其他資源自行探索更多。
