# --explanation--

`Physical access`는 사무실이나 서버실 같은 장소에 들어가는 것을 뜻합니다.

`physical access breach`는 누군가 허가 없이 보안 구역에 들어갈 때 발생합니다. 예를 들어:

- `To avoid a physical access breach, we should require ID checks at the entrance.` - 회사는 출입문에서 신분증을 확인하여 무단 출입을 막을 수 있습니다.

- `A physical access breach can happen if someone enters the server room without permission.` - 누군가 허가 없이 서버실에 들어가면 물리적 보안 문제가 발생할 수 있습니다.
