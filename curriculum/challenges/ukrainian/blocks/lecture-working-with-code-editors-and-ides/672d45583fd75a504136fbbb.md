# --description--

Щоб встановити VS Code, вам потрібно відвідати їхній вебсайт, щоб отримати відповідний інсталятор для вашої операційної системи. Переконайтеся, що ви заходите на сайт VS Code, а не на сайт Visual Studio.

Для Windows можна скачати інсталятор у форматі EXE. Після завантаження запустіть файл, і майстер налаштування проведе вас через процес встановлення.

Для Mac можна скачати zip-архів, який містить `.app`, який ви можете запустити. Також можна встановити VS Code через homebrew.

Відкрийте застосунок Terminal на вашому Mac і виконайте таку команду:

```sh
brew install --cask visual-studio-code
```

На Linux можна безпосередньо скачати `.deb` або `.rpm` файл. Після встановлення ви можете використовувати відповідний виконуваний файл або команду інтерфейсу командного рядка (зазвичай `code`), щоб запустити застосунок. І тепер ви готові писати код.
