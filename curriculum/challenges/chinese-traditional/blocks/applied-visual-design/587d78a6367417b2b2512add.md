# --description--

透過操作不同的選擇器和屬性，你可以製作有趣的形狀。其中一個較容易嘗試的是新月形狀。這個挑戰需要你使用設定元素陰影的 `box-shadow` 屬性，以及控制元素角落圓潤度的 `border-radius` 屬性。

你將創建一個圓形、透通的物件，並帶有一個稍微偏移到一側的清晰陰影——陰影實際上就是你看到的月亮形狀。

為了創建一個圓形物件，`border-radius` 屬性應該設定為 50％ 的值。

你可能會回想起先前的挑戰中，`box-shadow` 屬性依序接受 `offset-x`、`offset-y`、`blur-radius`、`spread-radius` 和 `color` 值。`blur-radius` 和 `spread-radius` 值是選用的。

# --instructions--

在編輯器中操作方形元素以生成月亮形狀。首先，將 `background-color` 更改為 `transparent`，然後將 `border-radius` 屬性設為 50% 以形成圓形。最後，更改 `box-shadow` 屬性，將 `offset-x` 設為 25px，`offset-y` 設為 10px，`blur-radius` 設為 0，`spread-radius` 設為 0，並將 `color` 設為 `blue`。

# --hints--


`box-shadow` 屬性的值應該設定為 `offset-x` 為 25px、`offset-y` 為 10px、`blur-radius` 為 0、`spread-radius` 為 0，最後 `color` 為 `blue`。
