# --description--

`To be into...`는 어떤 것에 관심이 있거나 즐긴다는 뜻입니다.

`These`는 대화에서 이전에 언급된 특정 항목이나 아이디어를 가리키는 지시 대명사입니다.

Sarah가 `Are you into these activities?`라고 물을 때, 그녀는 Tom이 방금 언급한 것을 가리키기 위해 `these`를 사용합니다. 이는 전체 문구를 반복하지 않고도 그녀가 말하는 활동을 정확히 지정하는 방법입니다.
