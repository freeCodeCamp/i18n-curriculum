# --description--

在之前的课程中，你学习了如何通过索引（从零开始）识别字串中的每个字符，并使用方括弧表示法进行访问：

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
print(my_str[-1]) # d
```

**字串切片** 让你提取字串的一部分或仅处理其中的特定部分。基本语法如下：

```python
string[start:stop]
```

如果你想从某个索引提取字符到另一个索引，只需用冒号分隔 `start` 和 `stop` 索引：

```python
my_str = 'Hello world'
print(my_str[1:4]) # ell
```

请注意，`stop` 索引是不包含的，所以 `[1:4]` 只是提取了从索引 `1` 开始，到但不包括索引 `4` 处的字符。

你也可以省略 `start` 和 `stop` 索引，Python 会分别默认使用 `0` 或字串的末尾。例如，如果你省略 `start` 索引，会发生以下情况：

```python
my_str = 'Hello world'
print(my_str[:7])  # Hello w
```

这会提取从索引 `0` 到（但不包括）索引 `7` 处的字符的所有内容。如果你省略了 `stop` 索引，情况如下：

```python
my_str = 'Hello world'
print(my_str[8:])  # rld
```

这会提取从索引为 `8` 的字符开始直到字串末尾的所有内容。

请注意，切片一个字串不会修改原始字串：

```py
my_str = 'Hello world'
print(my_str[8:])  # rld
print(my_str)  # Hello world
```

你也可以省略 `start` 和 `stop` 索引，这将提取整个字串：

```python
my_str = 'Hello world'
print(my_str[:])  # Hello world
```

除了 `start` 和 `stop` 索引外，还有一个可选的 `step` 参数，用于指定切片中每个索引之间的增量。

这是其语法：

```python
string[start:stop:step]
```

在下面的示例中，切片从索引 `0` 开始，在 `11` 之前停止，并提取每隔一个的字符：

```python
my_str = 'Hello world'
print(my_str[0:11:2])  # Hlowrd
```

一个你可以对 `step` 参数做的有用技巧是通过将 step 设置为 `-1`，并将 `start` 和 `stop` 留空来反转一个字串：

```python
my_str = 'Hello world'
print(my_str[::-1]) # dlrow olleH
```
