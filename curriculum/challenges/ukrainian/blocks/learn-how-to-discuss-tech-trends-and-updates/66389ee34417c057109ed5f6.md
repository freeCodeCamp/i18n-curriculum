# --description--

`To automate` означає змусити процес працювати самостійно без потреби в допомозі людини. У програмуванні це може означати використання програмного забезпечення для написання коду, тестування застосунків або пошуку й виправлення помилок. Наприклад, кавоварка, яка вмикається сама вранці, щоб приготувати каву, є `automated`.

Слово `certain` використовується, щоб говорити про певні речі або людей, але не про всі. Наприклад, якщо ви скажете, що лише `certain books are on sale`, це означає, що не всі книги продаються зі знижкою.
