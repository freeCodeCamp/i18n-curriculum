# --description--

一列長度為 `n` 單位的列上放置了紅色區塊，紅色區塊的最小長度為 `m` 單位，且任意兩個紅色區塊（允許長度不同）之間至少以一個黑色方格分隔。

讓填充計數函式（`fill-count function`）`$F(m, n)$` 表述一列可以被填充的方式數量。

例如，$F(3, 29) = 673135$ 和 $F(3, 30) = 1089155$。

也就是說，當 m = 3 時，可以看到 n = 30 是填充計數函式首次超過一百萬的最小值。

以相同方式，當 m = 10 時，可以驗證 $F(10, 56) = 880711$ 和 $F(10, 57) = 1148904$，因此 n = 57 是填充計數函式首次超過一百萬的最小值。

當 m = 50 時，找出使填充計數函式首次超過一百萬的最小 `n` 值。

**注意：** 這是問題 114 的較困難版本。
