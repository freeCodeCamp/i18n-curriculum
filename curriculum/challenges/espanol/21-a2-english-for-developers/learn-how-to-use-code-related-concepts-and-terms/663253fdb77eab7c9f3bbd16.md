# --description--

Esta tarea se enfoca en el pronombre relativo `that`. Se utiliza para introducir una cláusula que proporciona más detalles sobre un sustantivo.

Una cláusula es una parte de una oración que contiene al menos un sujeto y un verbo. Puede expresar una idea completa o agregar información adicional a la oración principal.

El pronombre `that` ayuda a especificar o definir algo más claramente, conectándolo a una descripción o un detalle necesario.

Por ejemplo, `She read the book that won the Pulitzer Prize last year.` En esta oración, `that won the Pulitzer Prize last year` es una cláusula relativa introducida por `that`. La cláusula describe o especifica el libro, ayudando a distinguirlo de otros libros. Aquí, `that` actúa como un pronombre relativo que conecta la cláusula con el sustantivo `book`, y describe qué libro en particular ella leyó.
