## --text--

根據 Sophie，如何在 Python 中宣告一個變數？

## --answers--

你透過以運算子命名來宣告一個變數。

### --feedback--

這個選擇是不正確的，因為變數不是在運算子之後命名，而是在運算子之前命名。

---

變數是透過使用名為 `declare` 的指令來宣告的。

### --feedback--

Python 不需要像 `declare` 這樣的特殊指令來創建變數；它直接使用指派。

---

你使用變數名稱，後面接著指定運算子和一個值。

---

變數必須在名稱前輸入 `variable` 來宣告。

### --feedback--

Python 不使用任何像 `variable` 這樣的關鍵字來宣告變數。它僅使用變數名稱、指定運算子和數值。
