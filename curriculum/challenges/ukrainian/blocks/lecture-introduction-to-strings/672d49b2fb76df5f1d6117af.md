# --interactive--

У JavaScript робота з текстом є важливою частиною кодування, і часто вам потрібно об’єднати або з’єднати частини тексту разом. Цей процес називається конкатенацією рядків.

У цьому уроці ми зосередимося на тому, як працює конкатенація рядків, зокрема з використанням оператора `+`, оператора `+=` та методу `concat()`.

Оператор `+` — один із найпростіших і найчастіше використовуваних способів конкатенації рядків. Він дозволяє з’єднувати кілька рядків або комбінувати рядки зі змінними, які містять текст.

Ось приклад:

:::interactive_editor

```js
let firstName = "John";
let lastName = "Doe";

let fullName = firstName + " " + lastName; 
console.log(fullName); // John Doe
```

:::

У цьому прикладі ми використали оператор `+` для конкатенації змінних `firstName` і `lastName` разом із пробілом (`" "`) для створення повного імені.

Одним із недоліків використання оператора `+` для конкатенації рядків є те, що це може призвести до проблем із пробілами, якщо не контролювати пробіли між конкатенованими рядками.

Ось приклад, де бракує пробілу:

:::interactive_editor

```js
let firstName = "John";
let lastName = "Doe";

let fullName = firstName + lastName; 
console.log(fullName); // "JohnDoe"
```

:::

Коли ви використовуєте оператор `+` для конкатенації рядків, важливо ретельно перевіряти можливі проблеми з пробілами.

Якщо потрібно додати або приєднати до існуючого рядка, тоді можна використати оператор `+=`. Це корисно, коли ви хочете поступово будувати рядок, додаючи до нього більше тексту.

Ось приклад додавання одного рядка до іншого за допомогою оператора `+=`:

:::interactive_editor

```js
let greeting = 'Hello';
greeting += ', John!';

console.log(greeting); // "Hello, John!"
```

:::

Важливо пам’ятати, що рядки є незмінними, тобто після створення рядка його не можна змінити.

У цьому випадку оригінальний рядок `Hello` не змінюється. Натомість змінна greeting тепер посилається на новий рядок `Hello, John!`.

Ще один спосіб конкатенувати рядки — це використання методу `concat()`.

Перш ніж почати вивчати метод `concat()`, важливо спочатку зрозуміти, що таке метод і функція на більш загальному рівні.

У програмуванні функція — це багаторазовий блок коду, який виконує певне завдання і може викликатися з різними вхідними даними. Метод, натомість, — це тип функції, пов’язаний з об’єктом, тобто він працює з даними, що містяться в цьому об’єкті.

У майбутніх уроках ми глибше розглянемо, як працюють функції, об’єкти та методи у JavaScript. Але наразі важливо розуміти, що у JavaScript є десятки методів, які ви можете використовувати, наприклад метод `concat()`.

Ось приклад використання методу `concat()` для об’єднання двох рядків:

:::interactive_editor

```js
let str1 = 'Hello';
let str2 = 'World';

let result = str1.concat(' ', str2); 
console.log(result); // Hello World
```

:::

У цьому прикладі ми використовуємо метод `concat()` для об’єднання `str1`, пробілу (`' '`) і `str2` в один рядок.

Підсумовуючи, оператор `+` найкраще підходить для простої конкатенації, особливо коли потрібно об’єднати кілька рядків або змінних.

Оператор `+=` корисний, коли ви поступово будуєте рядок або додаєте новий вміст до існуючої змінної рядка.

Нарешті, метод `concat()` вигідно використовувати, коли потрібно конкатенувати кілька рядків разом.
