# --interactive--

你可能已经看到过网页上的链接在被点击后变为紫色。 这是因为链接的状态发生了变化，所以会应用不同的样式。 链接可以处于五种不同的状态。

第一种是默认状态，即 `:link`。 此状态表示用户尚未访问、单击或交互的链接。 你可以将此状态视为对页面上的所有链接提供的基本样式。 其他状态则在此基础上建立。

启用交互式编辑器并查看预览窗口中的链接。它应为表现默认状态的默认蓝色。如果你点击它，它将变成紫色。 

:::interactive_editor

```html
<a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
```

:::

第二种状态是 `:visited`，当用户已经访问过被链接到的页面时适用。 默认情况下，这将把链接变成紫色——但是你可以利用 CSS 为用户提供不同的视觉指示。 这有助于让人知道他们已经阅读了你的部分文档。 或者，这个网站是他们可以信任的，因为他们以前使用过它。

点击预览窗口中的链接，你应该会看到已访问的链接变为棕色。要查看预览，你需要启用交互式编辑器。

**注意**：为了让你看到链接样式的变化，已经为这个交互示例提供了一些 CSS。不要担心尝试理解这些 CSS 代码，因为你将在未来的模块中学习它们的全部含义。 

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:visited {
  color: brown;
}
```

:::

第三种状态是 `:hover`。 当用户将光标悬停在链接上时，适用此状态。 这种状态有助于为链接提供额外的关注，以确保用户确实打算点击它。

启用交互式编辑器并尝试将你的鼠标悬停在链接上。你将看到颜色变为红色。 

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:hover {
  color: red;
}
```

:::

然后是 `:focus`。 当我们聚焦一个链接时，适用此状态。

点击预览窗口中空白区域的任意部分，然后按键盘上的 `tab`。你应该会看到链接变为绿色。要查看预览，你需要启用交互式编辑器。

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:focus {
  color: green;
}
```

:::

最后是 `:active`。 此状态适用于由用户激活的链接。 在大多数情况下，这通常意味着通过左击鼠标的主按键来单击链接。 这种状态有助于向用户显示他们点击的元素是交互式的。

启用交互式编辑器并点击该 `link`。你应该会看到该 `link` 变成黑色。这个过程非常快，所以你可能需要点击几次才能看到颜色变化。 

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:active {
  color: black;
}
```

:::

当你使用这些状态来样式化你的链接时，你需要按照一个特定的顺序来写你的 CSS：`link`、`visited`、`hover`、`focus`，然后是 `active`。

现在，你应该知道如何在页面中赋予链接你自己的个人风格，同时仍然提供用户需要的关于每个链接状态的重要信息。
