# --description--

Jetzt, da Sie ein grundlegendes Verständnis von Arrays haben, sprechen wir über Schleifen. Schleifen werden verwendet, um einen Codeblock mehrfach auszuführen. Eine dieser Schleifen ist die `while`-Schleife. Die `while`-Schleife führt einen Codeblock so lange aus, wie die Bedingung wahr ist. Die Syntax der `while`-Schleife sieht wie folgt aus:

```javascript
while (condition) {
  // code block to be executed
}
```

Zum Beispiel gibt der folgende Code-Ausschnitt die Zahlen von 1 bis 5 aus:

```javascript
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
```

Der obige Code-Ausschnitt initialisiert eine Variable `i` mit dem Wert `1`. Die `while`-Schleife führt den Codeblock so lange aus, wie der Wert von `i` kleiner oder gleich `5` ist. Der Wert von `i` wird in jeder Iteration um `1` erhöht.
