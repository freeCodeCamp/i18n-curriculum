# --description--

Umepatiwa taarifa zifuatazo, lakini unaweza pia kupendelea kufanya utafiti kwa ajili yako mwenyewe.

<ul>
  <li>1 Jan 1900 ilikuwa Jumatatu.</li>
  <li>Siku thelathini zina Septemba,<br>Aprili, Juni na Novemba.<br>Zote zingine zina thelathini na moja,<br>Isipojumuisha Februari pekee,<br>Ambayo ina ishirini na nane, mvua au jua.<br>Na katika miaka ya kuruka, ishirini na tisa.</li>
  <li>Mwaka wa kuruka hutokea katika mwaka wowote unaogawanyika kwa usawa na 4, lakini si katika karne isipokuwa ukigawanyika kwa 400.</li>
</ul>

Ni Jumapili ngapi zilikuwa tarehe moja ya mwezi wakati wa karne ya ishirini (1 Jan 1901 hadi 31 Des 2000)?

```js
function countingSundays(firstYear, lastYear) {
  let sundays = 0;

  for (let year = firstYear; year <= lastYear; year++) {
    for (let month = 0; month <= 11; month++) {
      const thisDate = new Date(year, month, 1);
      if (thisDate.getDay() === 0) {
        sundays++;
      }
    }
  }
  return sundays;
}
```
