# --description--

Denken Sie daran, `their` ist ein Possessivpronomen. Es zeigt an, dass etwas einer Gruppe von Personen gehört. Zum Beispiel:

`Their opinions are important.` – Das bedeutet, die Meinungen einer Gruppe von Personen sind wichtig.

`Thoughts` bedeutet, was jemand denkt oder seine Ideen. Wenn Sie nach der `thoughts` von jemandem fragen, möchten Sie wissen, was er über etwas denkt oder fühlt. Zum Beispiel:

`What are their thoughts on this book?` – Das fragt nach der Meinung der Gruppe über das Buch.
