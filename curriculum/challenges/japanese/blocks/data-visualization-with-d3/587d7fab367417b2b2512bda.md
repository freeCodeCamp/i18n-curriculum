# --description--

棒グラフと散布図のチャートはどちらもデータを直接SVGにプロットしました。しかし、棒の高さやデータポイントのいずれかがSVGの高さや幅の値より大きい場合、それはSVGエリアの外に出てしまいます。

D3にはデータをプロットするのに役立つスケールがあります。`scales`は、生のデータポイントのセットをSVGのピクセルにどのようにマッピングするかをプログラムに伝える関数です。

例えば、100x500サイズのSVGがあり、複数の国の国内総生産（GDP）をプロットしたいとします。数値のセットは数十億ドルや数兆ドルの範囲になります。D3にスケールの種類を指定して、大きなGDPの値をその100x500サイズのエリアに配置する方法を教えます。

生のデータをそのままプロットすることはほとんどありません。プロットする前に、データセット全体のスケールを設定して、`x`と`y`の値がSVGの幅と高さに収まるようにします。

D3にはいくつかのスケールの種類があります。線形スケール（通常は定量データに使われます）には、D3のメソッド`scaleLinear()`があります。

```js
const scale = d3.scaleLinear();
```

デフォルトでは、スケールは恒等関係を使います。入力の値と出力の値は同じです。これを変更する方法は別のチャレンジで扱います。
