# --description--

En Express, el objeto `res` (response) se usa para enviar respuestas al cliente después de procesar las solicitudes entrantes. Express ofrece varios métodos integrados para enviar diferentes tipos de respuestas. Por ejemplo, puedes enviar texto plano, datos JSON, redirigir a un usuario a otra URL o renderizar una plantilla.

Uno de los métodos de respuesta más usados es `res.send()`:

```javascript
app.get("/text", (req, res) => {
  res.send("Hello, World!");
});
```

El método `res.send()` envía una respuesta al cliente. Puede enviar texto plano, HTML, JSON o incluso datos binarios. En este ejemplo, cuando un cliente visita `/text`, el servidor responde con el texto `Hello, World!`.

Otro método de respuesta común es `res.json()`.

Este método envía una respuesta JSON al cliente. Es especialmente útil al construir APIs, donde a menudo necesitas devolver datos estructurados como objetos o arreglos. Express convierte automáticamente un objeto JavaScript en una cadena con formato JSON:

```javascript
app.get("/user", (req, res) => {
  res.json({ name: "John Doe", age: 30 });
});
```

Si un cliente envía una solicitud a `/user`, el servidor responde con lo siguiente:

```json
{
  "name": "John Doe",
  "age": 30
}
```

Cuando necesitas controlar el código de estado HTTP de una respuesta, puedes usar `res.status()`. Este método establece el código de estado y normalmente se encadena con otro método de respuesta como `res.send()` o `res.json()`:

```javascript
app.get("/notfound", (req, res) => {
  res.status(404).send("Not Found");
});
```

Aquí, el servidor establece el código de estado en `404` y envía el mensaje `Not Found` de vuelta al cliente.

Si quieres enviar al usuario a una URL diferente, puedes usar `res.redirect()`:

```javascript
app.get("/oldpage", (req, res) => {
  res.redirect("/newpage");
});
```

Cuando un cliente visita `/oldpage`, es redirigido automáticamente a `/newpage`.

Finalmente, `res.render()` se usa cuando trabajas con motores de plantillas como Pug, EJS o Handlebars. Renderiza una plantilla de vista y envía el HTML resultante al cliente:

```javascript
app.get("/profile", (req, res) => {
  res.render("profile", { username: "john_doe" });
});
```

En este ejemplo, el servidor renderiza una vista `profile` y le pasa una variable `username`, que luego puede mostrarse en el HTML.

Estos métodos de respuesta te permiten devolver diferentes tipos de datos según la naturaleza de la solicitud. Ya sea que envíes texto, datos JSON o redirijas al cliente, Express tiene un conjunto flexible de herramientas para manejarlo.
