# --description--

Una `version` è uno dei campi obbligatori del tuo file package.json. Questo campo descrive la versione attuale del tuo progetto. Ecco un esempio:

```json
"version": "1.2.0",
```

# --instructions--

Aggiungi una `version` al file package.json del tuo progetto.

# --hints--

package.json dovrebbe avere una chiave "version" valida

```js
fetch(code + '/_api/package.json')
  .then(response => response.json())
  .then(
    data => {
      assert.exists(data.version, '"version" is missing');
    },
    error => {
      throw new Error(error.message || error.responseText);
    }
  );
```
