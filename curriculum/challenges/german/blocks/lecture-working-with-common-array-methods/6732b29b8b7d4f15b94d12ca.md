# --interactive--

Eine flache Kopie eines Arrays ist ein neues Array, das dieselben Elemente wie das Original enthält. Wenn das Array nur primitive Werte wie Zahlen oder Strings enthält, ist das neue Array völlig unabhängig. Enthält das Array jedoch andere Arrays, haben sowohl das Original als auch die Kopie Referenzen auf dieselben inneren Arrays. Das bedeutet, wenn Sie etwas in einem gemeinsamen inneren Array ändern, sehen Sie diese Änderung in beiden Arrays.

Flache Kopien sind hilfreich, wenn Sie die oberste Struktur ändern möchten, zum Beispiel Elemente hinzufügen, entfernen oder neu anordnen, ohne das Originalarray oder die inneren Arrays zu verändern.

Es gibt mehrere Methoden, um flache Kopien von Arrays zu erstellen, und wir werden einige der gebräuchlichsten betrachten: `concat()`, `slice()` und den Spread-Operator.

Beginnen wir mit der `concat()`-Methode. Diese Methode erstellt ein neues Array, indem zwei oder mehr Arrays zusammengeführt werden. Wenn sie mit nur einem Array verwendet wird, erzeugt sie effektiv eine flache Kopie. Hier ein Beispiel:

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const copyArray = [].concat(originalArray);

console.log(copyArray); // [1, 2, 3]
console.log(copyArray === originalArray); // false
```

:::

In diesem Beispiel verwenden wir die `concat()`-Methode, um ein leeres Array an `originalArray` anzuhängen. Dadurch entsteht ein neues Array, das eine flache Kopie von `originalArray` ist.

Das `copyArray` enthält dieselben Elemente wie `originalArray`, ist aber ein anderes Array-Objekt, weshalb der strikte Gleichheitsvergleich (`===`) `false` zurückgibt.

Eine weitere Methode, um eine flache Kopie zu erstellen, ist die `slice()`-Methode. Wenn sie ohne Argumente aufgerufen wird, gibt `slice()` eine flache Kopie des gesamten Arrays zurück. So funktioniert das:

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const copyArray = originalArray.slice();

console.log(copyArray); // [1, 2, 3]
console.log(copyArray === originalArray); // false
```

:::

In diesem Fall erstellt `originalArray.slice()` ein neues Array, das eine flache Kopie von `originalArray` ist. Auch hier enthält das `copyArray` dieselben Elemente, ist aber ein anderes Array-Objekt.

Der Spread-Operator (`...`), eingeführt in ES6, bietet eine weitere kompakte Möglichkeit, flache Kopien von Arrays zu erstellen. Hier ein Beispiel:

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const copyArray = [...originalArray];

console.log(copyArray); // [1, 2, 3]
console.log(copyArray === originalArray); // false
```

:::

Der Spread-Operator (`...`) verteilt die Elemente von `originalArray` in ein neues Array und erstellt so effektiv eine flache Kopie. Wichtig ist, dass alle diese Methoden neue Array-Objekte erzeugen, sodass Sie die Kopie ändern können, ohne das Originalarray zu beeinflussen. Zum Beispiel:

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const copyArray = [...originalArray];

copyArray.push(4);
console.log(originalArray); // [1, 2, 3]
console.log(copyArray);     // [1, 2, 3, 4]
```

:::

In diesem Beispiel wirkt sich das Hinzufügen eines Elements zu `copyArray` nicht auf `originalArray` aus.

Zusammenfassend lassen sich flache Kopien von Arrays leicht mit Methoden wie `concat()`, `slice()` oder dem Spread-Operator erstellen. Diese Methoden sind nützlich, um neue Arrays zu erzeugen, die unabhängig vom Originalarray bearbeitet werden können.
