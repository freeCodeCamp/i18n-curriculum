# --description--

**Objetivo:** Cumplir con las historias de usuario a continuación y pasar todas las pruebas para completar el laboratorio.

**Historia de usuario:**

1. Deberías exportar un componente `Footer`.
2. Tu componente `Footer` debería devolver un elemento `footer` que contiene todos los demás elementos.
3. Tu elemento `footer` debería tener:

- Al menos tres listas desordenadas, cada una con al menos dos elementos de lista.
- Al menos un elemento de párrafo con un símbolo de copyright `©`.
- Al menos tres enlaces con el valor `href` configurado en `#` y el contenido del enlace establecido como un icono o texto de tu elección.

# --hints--

Deberías exportar un componente `Footer`.

```js
assert.isFunction(window.index.Footer);
```

Tu componente `Footer` debería devolver un elemento `footer`.

```js
const footer = document.getElementById('root').firstElementChild;
assert.equal(footer.tagName, 'FOOTER');
```

Tu componente `Footer` debería contener solo un único elemento `footer`.

```js
assert.lengthOf(document.querySelectorAll('footer'), 1);
```

Tu elemento `footer` no debería tener hermanos.

```js
assert.lengthOf(document.getElementById('root').children, 1);
```

Tu elemento `footer` debería contener al menos tres listas desordenadas.

```js
assert.isAtLeast(document.querySelectorAll('ul').length, 3);
```

Cada una de tus listas desordenadas debería tener al menos dos elementos de lista.

```js
const uls = [...document.querySelectorAll('ul')];
uls.forEach(ul => {
    assert.isAtLeast(ul.querySelectorAll('li').length, 2);
  }
);
```

Tu `footer` debería tener al menos un elemento de párrafo.

```js
assert.isAtLeast(document.querySelectorAll('p').length, 1);
```

Deberías tener un símbolo de copyright (`©`) dentro de uno de tus párrafos.

```js
const texts = [...document.querySelectorAll('p')].map(p => p.textContent);
const hasCopyright = texts.some(text => text.includes('©'));
assert.isTrue(hasCopyright);
```

Tu `footer` debería tener al menos tres enlaces con el valor `href` configurado en `#`.

```js
const links = document.querySelectorAll('a[href="#"]');
assert.isAtLeast(links.length, 3);
```

Ninguno de tus enlaces debería estar vacío.

```js
const links = [...document.querySelectorAll('a[href="#"]')];
links.forEach(link => {
    assert.isNotEmpty(link.textContent);
  }
);
```
