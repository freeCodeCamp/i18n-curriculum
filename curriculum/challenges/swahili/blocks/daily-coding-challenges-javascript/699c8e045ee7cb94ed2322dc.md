# --description--

Ikiwa umepewa matriksi (safu ya safu za data) iliyojaa thamani mbili tofauti, rudisha matriksi mpya ambapo matukio yote ya thamani moja yamebadilishwa na nyingine.

Kwa mfano, ukipata:

```js
[
  ["a", "b"],
  ["a", "a"]
]
```

Rudisha:

```js
[
  ["b", "a"],
  ["b", "b"]
]
```

```js
function invertMatrix(matrix) {
  if (matrix.length === 0) return [];

  const values = [...new Set(matrix.flat())];
  const [val1, val2] = values;

  return matrix.map(row =>
    row.map(cell => (cell === val1 ? val2 : val1))
  );
}
```
