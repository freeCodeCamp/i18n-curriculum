# --description--

ピッグ・ラテンは英単語を変形する方法です。ルールは次の通りです：

\- 単語が子音で始まる場合、最初の子音または子音の連続を取り出し、単語の末尾に移動してから`ay`を追加します。

\- 単語が母音で始まる場合は、単に末尾に`way`を追加します。

# --instructions--

与えられた文字列をピッグ・ラテンに翻訳してください。入力される文字列はすべて小文字の英単語であることが保証されています。

```js
function translatePigLatin(str) {
  if (isVowel(str.charAt(0))) return str + "way";
  var front = [];
  str = str.split('');
  while (str.length && !isVowel(str[0])) {
    front.push(str.shift());
  }
  return [].concat(str, front).join('') + 'ay';
}

function isVowel(c) {
  return ['a', 'e', 'i', 'o', 'u'].indexOf(c.toLowerCase()) !== -1;
}
```
