# --explanation--

Você usa a estrutura `Second Conditional` (no discurso da Maria, `It'd be helpful if we had...`) para descrever uma situação hipotética. A estrutura do segundo condicional é `It would` + verbo base ... `if` + sujeito + verbo no `Past Simple`.  It is used to describe unreal or hypothetical situations in the present or future. Por exemplo:

`We would finish earlier if we had more team members.` - Isso significa que eles não têm mais membros na equipe, mas se tivessem, poderiam terminar mais cedo. Estamos falando de uma hipótese.
