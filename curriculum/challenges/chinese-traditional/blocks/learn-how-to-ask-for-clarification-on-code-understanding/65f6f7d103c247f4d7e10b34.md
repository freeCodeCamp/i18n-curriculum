# --description--

字、單字 `clear` 意思是容易理解或看見。當你想比較某事物並顯示它更 `clear` 時，你會在結尾加上 `er` 使其變成 `clearer`。

例如：

- 這個說明是 `clear`。
- 這個說明比另一個 `clearer`。

注意 `clearer` 是用來比較兩個事物，表示其中一個比另一個更容易理解。
