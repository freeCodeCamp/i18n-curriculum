# --description--

Цей тест перевіряє ваше розуміння базових концепцій програмування.

Щоб пройти тест, потрібно правильно відповісти щонайменше на 18 із 20 питань нижче.

Читайте кожне питання і вибирайте правильну відповідь. Для кожного питання є лише одна правильна відповідь.
