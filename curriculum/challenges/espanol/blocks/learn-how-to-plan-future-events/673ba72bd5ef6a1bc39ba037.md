# --explanation--

`Downtime` es un período cuando un sistema o servicio no está disponible o no está operativo. A menudo se programa para realizar mantenimiento, actualizaciones o reparaciones. 

`Server downtime` se refiere específicamente al tiempo durante el cual un servidor — un sistema informático que proporciona datos, servicios o recursos a otros dispositivos — está temporalmente no disponible. Por ejemplo:

- `The server downtime was scheduled at midnight to minimize the impact on users.` - Esto significa que el servidor estuvo fuera de servicio a medianoche para reducir las molestias a los usuarios.

- `Unexpected downtime occurred due to a power failure.` - Esto significa que el sistema se volvió inaccesible sin planificación previa debido a un corte de energía.
