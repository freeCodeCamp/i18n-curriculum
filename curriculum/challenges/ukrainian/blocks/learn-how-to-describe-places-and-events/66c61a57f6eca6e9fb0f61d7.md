# --explanation--

`Built-in` означає функції або інструменти, які включені як частина програмного забезпечення або пристрою з самого початку, без необхідності додавати чи встановлювати їх окремо.

Коли щось є `built-in`, воно вже є і готове до використання. Наприклад:

- `This laptop has a built-in camera.` — камера включена як частина ноутбука.

- `The software comes with built-in security features.` — функції безпеки вже включені у програмне забезпечення.
