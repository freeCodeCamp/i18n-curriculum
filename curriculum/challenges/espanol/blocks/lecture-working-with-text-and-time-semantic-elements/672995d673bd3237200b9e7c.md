# --interactive--

Una abreviatura es una forma abreviada de una palabra o frase. Por ejemplo, "Dr" seguido de un punto, es una abreviatura de la palabra "doctor".

Hay dos formas comunes de abreviaturas: acrónimos e inicialismos.

Un acrónimo es una palabra formada a partir de las iniciales de una frase, donde cada letra representa la primera letra de una palabra en esa frase.

GUI es un ejemplo de un acrónimo. Significa Graphical User Interface. Al tomar las primeras letras de cada palabra G, U e I, obtienes el acrónimo GUI.

Un acrónimo se forma a partir de las letras iniciales de una frase, donde cada letra representa la primera letra de una palabra en esa frase.

Por ejemplo, HTML es un acrónimo; significa HyperText Markup Language y se pronuncia deletreando cada letra H, T, M, L.

Tanto los acrónimos como las siglas son tipos de abreviaturas. La diferencia es que los acrónimos se pronuncian como palabras y las siglas se pronuncian como letras individuales.

Son muy útiles para escribir texto más conciso, especialmente cuando son bien conocidos y fáciles de entender en un contexto determinado. 

If you need to display abbreviations such as acronyms or initialisms in HTML, the abbreviation element is exactly what you're looking for. Siempre debes explicar su significado completo la primera vez que los usas. Luego puedes usar el elemento de abreviación para resaltarlos y proporcionar más detalles. 

Aquí hay un ejemplo donde puedes ver un párrafo con la frase `HTML es la base de la web`: 

:::interactive_editor

```html
<p><abbr>HTML</abbr> is the foundation of the web.</p>
```

:::

The initialism HTML is within an abbreviation element. En el navegador, verás que realmente nada ha cambiado. Todavía se ve como texto normal.  The abbreviation element is providing helpful context behind the scenes, but users will still see the initialism as normal text. 

Si quieres ayudar a los usuarios a entender qué significa esta sigla, puedes mostrar su forma completa con el atributo `title`.

El atributo `title` es opcional, pero si decides incluirlo, debe ser una descripción legible para humanos de la abreviatura, acrónimo o sigla.

Tomemos el mismo ejemplo de antes, pero añade el atributo `title`. Será `HyperText Markup Language`, la forma extendida de la sigla:

:::interactive_editor

```html
<p><abbr title="HyperText Markup Language">HTML</abbr> is the foundation of the web.</p>
```

:::

Por lo general, el estilo del elemento de abreviación cambiará cuando agregas este atributo. El estilo específico dependerá del navegador.  Some browsers may display a dotted underline, while others may convert the contents to small caps.  When the user hovers over the abbreviation, the full form is displayed as a tooltip.

Aunque no es necesario usar el elemento de abreviatura para cada abreviatura en tu página web, se recomienda para aquellas que podrían ser confusas y para las que podrían necesitar contexto adicional.

Debes usar tu mejor criterio para encontrar el equilibrio adecuado entre información y presentación, evitando saturar el texto y siendo claro y conciso.
