# --description--

在 Python 中，類別和物件密切合作以組織和管理資料。你建置一個類別來定義共用行為，然後創建使用這些行為的物件。

換句話說，類別就像你用來創建物件的藍圖或模板。

讓我們來看看什麼是類別，以及如何使用它們來創建物件。

要創建一個類別，你使用 `class` 關鍵字，後面接類別名稱和冒號。然後在類別內，你可以添加初始化函式，以及任何屬性和方法。

屬性就像類別中的變數，用來儲存資料。方法是在類別中定義的函式（程式），是使用類別創建的物件可以執行的行為。

以下是類別的基本語法：

```python
class ClassName:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def sample_method(self):               
        print(self.name.upper())
```

* `class ClassName` 是由 `class` 關鍵字用來創建類別，接著是類別的名稱，這裡稱為 `ClassName`。在 Python 中，命名類別時通常使用 **PascalCase** 慣例。
    
* `def __init__(self, name, age)` 是在創建新物件時自動呼叫的特殊方法。它初始化將由類別創建的物件的屬性。除此之外，`__init__` 的第一個參數總是對正在創建或使用的特定物件的引用。依慣例，這個參數命名為 `self`，但技術上你可以使用任何名稱。`self` 讓你存取物件自身的屬性和方法。
    
* `self.name = name` 和 `self.age = age` 是物件將擁有的屬性。
    
* `def sample_method(self):` 是每個創建的物件都可以呼叫的行為。
    
* `print(self.name.upper())` 是 `sample_method` 方法會執行的動作，在此情況下，它會列印大寫的名稱。
    

如果這聽起來很多，不用擔心。讓我們來看看一個類似的 `Dog` 類別範例，以及你如何從中創建物件：

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name.upper()} says woof woof!")
```

使用這個 `Dog` 類別，你可以創建一個物件。以下是從類別創建物件的基本語法：

```python
object_1 = ClassName(attribute_1, attribute_2)
object_2 = ClassName(attribute_1, attribute_2)
```

你也可以從每個物件呼叫類別中定義的任何方法：

```python
object_1.method_name()
object_2.method_name()
```

現在讓我們使用 `Dog` 類別作為藍圖來創建兩隻狗：

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name.upper()} says woof woof! I'm {self.age} years old!")

dog_1 = Dog("Jack", 3)
dog_2 = Dog("Thatcher", 5)

# Call the bark method
dog_1.bark()  # JACK says woof woof! I'm 3 years old!
dog_2.bark()  # THATCHER says woof woof! I'm 5 years old!
```

如你所見，我們使用 `Dog` 類別創建了兩個狗物件。初始化 `dog_1` 時，傳入字串 `Jack` 和數字 `3`，這會設定該實體的 `name` 和 `age` 屬性。而 `dog_2` 則分別以字串 `Thatcher` 和數字 `5` 作為其 `name` 和 `age` 進行初始化。

然後當你對 `dog_1` 和 `dog_2` 呼叫 `.bark()` 方法時，你可以看到兩者的輸出如何不同，並使用你在建立每個物件時傳入的獨特 `name` 和 `age` 屬性。

總結來說，類別和物件的差異在於類別是模板或藍圖，而物件是使用該模板所創建的。

此外，類別定義物件應該擁有的資料和行為，而物件則持有實際的資料並使用該行為。你撰寫類別一次，就可以從中建立許多物件，每個物件擁有不同的資料。
