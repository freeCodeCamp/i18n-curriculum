# --description--

Das Wort `what` kann verwendet werden, um eine Klausel einzuleiten. Es führt typischerweise Nominalklauseln ein, die mehr Informationen über ein Substantiv liefern oder als Teil einer Aussage fungieren.

Eine Nominalklausel ist eine Klausel, die innerhalb eines größeren Satzes als Substantiv fungiert. Sie enthält typischerweise ein Subjekt und ein Verb und kann als Subjekt, Objekt oder Ergänzung im Satz auftreten.

Zum Beispiel ist in dem Satz `She doesn't know what to buy` `what to buy` eine Klausel, wobei `what` für das steht, worüber sie entscheiden muss, es zu kaufen.

Ein weiteres Beispiel ist `Tell me what you heard.` `What you heard` ist die Klausel, wobei `what` die spezifischen Dinge erklärt, die gehört wurden, und als Objekt des Verbs `heard` fungiert.

`These` ist ein pluraler Demonstrativpronomen, das sich auf bestimmte zuvor erwähnte oder aus dem Kontext verstandene Gegenstände bezieht.
