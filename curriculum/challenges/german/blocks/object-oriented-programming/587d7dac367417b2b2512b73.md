# --description--

Denken Sie an Dinge, die Menschen jeden Tag sehen, wie Autos, Geschäfte und Vögel. Das sind alles <dfn>Objekte</dfn>: greifbare Dinge, die Menschen beobachten und mit denen sie interagieren können.

Welche Eigenschaften haben diese Objekte? Ein Auto hat Räder. Geschäfte verkaufen Waren. Vögel haben Flügel.

Diese Eigenschaften, oder <dfn>Eigenschaften</dfn>, definieren, woraus ein Objekt besteht. Beachten Sie, dass ähnliche Objekte dieselben Eigenschaften teilen, aber unterschiedliche Werte für diese Eigenschaften haben können. Zum Beispiel haben alle Autos Räder, aber nicht alle Autos haben dieselbe Anzahl an Rädern.

Objekte in JavaScript werden verwendet, um reale Objekte zu modellieren, indem sie Eigenschaften und Verhalten erhalten, genau wie ihre realen Gegenstücke. Hier ist ein Beispiel, das diese Konzepte verwendet, um ein `duck` Objekt zu erstellen:

```js
let duck = {
  name: "Aflac",
  numLegs: 2
};
```

Dieses `duck` Objekt hat zwei Eigenschaft-/Wert-Paare: eine `name` von `Aflac` und eine `numLegs` von 2.

# --instructions--

Erstellen Sie ein `dog` Objekt mit `name` und `numLegs` Eigenschaften und setzen Sie diese jeweils auf einen String und eine Zahl.

# --hints--

`dog` sollte ein Objekt sein.

```js
assert(typeof dog === 'object');
```

`dog` sollte eine `name` Eigenschaft haben, die auf einen String gesetzt ist.

```js
assert(typeof dog.name === 'string');
```

`dog` sollte eine `numLegs` Eigenschaft haben, die auf eine Zahl gesetzt ist.

```js
assert(typeof dog.numLegs === 'number');
```
