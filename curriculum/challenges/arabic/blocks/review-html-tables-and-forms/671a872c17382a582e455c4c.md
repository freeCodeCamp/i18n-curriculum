# --interactive--

## عناصر وخصائص نموذج HTML

- **عنصر `form`**: يُستخدم لإنشاء نموذج HTML لإدخال المستخدم.
- **خاصية `action`**: تُستخدم لتحديد عنوان URL الذي تُرسل إليه بيانات النموذج.
- **خاصية `method`**: تُستخدم لتحديد طريقة HTTP التي تُستخدم عند إرسال بيانات النموذج. أكثر الطرق شيوعًا هي `GET` و `POST`.

```html
<form method="value-goes-here" action="url-goes-here">
  <!-- inputs go inside here -->
</form>
```

- **عنصر `input`**: يُستخدم لإنشاء حقل إدخال لإدخال المستخدم.
- **خاصية `type`**: تُستخدم لتحديد نوع حقل الإدخال. مثلًا: `text`، `email`، `number`، `radio`، `checkbox`، إلخ.
- **خاصية `placeholder`**: تُستخدم لإظهار تلميح للمستخدم يوضح ما يجب إدخاله في حقل الإدخال.
- **خاصية `value`**: تُستخدم لتحديد قيمة الإدخال. إذا كان الإدخال من نوع `button`، يمكن استخدام خاصية `value` لتعيين نص الزر.
- **خاصية `name`**: تُستخدم لتعيين اسم لحقل الإدخال، ويعمل كالمفتاح عند إرسال بيانات النموذج. بالنسبة لأزرار الاختيار، إعطاؤها نفس `name` يجمعها في مجموعة واحدة، بحيث يمكن اختيار خيار واحد فقط في المجموعة في كل مرة.
- **خاصية `size`**: تُستخدم لتحديد عدد الأحرف التي يجب أن تظهر أثناء كتابة المستخدم في الإدخال.
- **خاصية `min`**: يمكن استخدامها مع أنواع الإدخال مثل `number` لتحديد الحد الأدنى للقيمة المسموح بها في حقل الإدخال.
- **خاصية `max`**: يمكن استخدامها مع أنواع الإدخال مثل `number` لتحديد الحد الأقصى للقيمة المسموح بها في حقل الإدخال.
- **خاصية `minlength`**: تُستخدم لتحديد الحد الأدنى لعدد الأحرف المطلوبة في حقل الإدخال.
- **خاصية `maxlength`**: تُستخدم لتحديد الحد الأقصى لعدد الأحرف المسموح بها في حقل الإدخال.
- **خاصية `required`**: تُستخدم لتحديد أن حقل الإدخال يجب ملؤه قبل إرسال النموذج.
- **خاصية `disabled`**: تُستخدم لتحديد أن حقل الإدخال يجب أن يكون معطلاً.
- **خاصية `readonly`**: تُستخدم لتحديد أن حقل الإدخال للقراءة فقط.

:::interactive_editor

```html
<!-- Text input -->
<input 
  type="text"
  id="name"
  name="name"
  placeholder="e.g. Quincy Larson" 
  size="20"
  minlength="5"
  maxlength="30"
  required
/>

<!-- Number input -->
<input 
  type="number"
  id="quantity"
  name="quantity"
  min="2"
  max="10"
  disabled
/>

<!-- Button -->
<input type="button" value="Show Alert" />
```

:::

- **عنصر `label`**: يُستخدم لإنشاء مسمى لحقل الإدخال.
- **خاصية `for`**: تُستخدم لتحديد أي حقل إدخال يرتبط به المسمى.
- **الربط الضمني للنموذج**: يمكن ربط المدخلات بالمسمى عن طريق تغليف حقل الإدخال داخل عنصر `label`.

:::interactive_editor

```html
<form action="">
  <label>
    Full Name:
    <input type="text" />
  </label>
</form>
```

:::

- **الربط الصريح للنموذج**: يمكن ربط المدخلات بالمسمى باستخدام خاصية `for` على عنصر `label`.

:::interactive_editor

```html
<form action="">
  <label for="email">Email Address: </label>
  <input type="email" id="email" />
</form>
```

:::

- **عنصر `button`**: يُستخدم لإنشاء زر قابل للنقر. يمكن أن يحتوي الزر أيضًا على خاصية `type`، تُستخدم للتحكم في سلوك الزر عند تفعيله. مثلًا: `submit`، `reset`، `button`.

:::interactive_editor

```html
<button type="button">Show Form</button>
<button type="submit">Submit Form</button>
<button type="reset">Reset Form</button>
```

:::

- **عنصر `fieldset`**: يُستخدم لتجميع المدخلات ذات الصلة معًا.
- **عنصر `legend`**: يُستخدم لإضافة تسمية توضيحية تصف مجموعة المدخلات.

:::interactive_editor

```html
<!-- Radio group -->
<fieldset>
  <legend>Was this your first time at our hotel?</legend>

  <label for="yes-option">Yes</label>
  <input id="yes-option" type="radio" name="hotel-stay" value="yes" />

  <label for="no-option">No</label>
  <input id="no-option" type="radio" name="hotel-stay" value="no" />
</fieldset>

<!-- Checkbox group -->
<fieldset>
  <legend>
    Why did you choose to stay at our hotel? (Check all that apply)
  </legend>

  <label for="location">Location</label>
  <input type="checkbox" id="location" name="location" value="location" />

  <label for="price">Price</label>
  <input type="checkbox" id="price" name="price" value="price" />
</fieldset>
```

:::

- **حالة التركيز**: هي حالة حقل الإدخال عندما يختاره المستخدم.

## العمل مع عناصر وخصائص جدول HTML

- **عنصر الجدول**: يُستخدم لإنشاء جدول HTML.
- **عنصر رأس الجدول (`thead`)**: يُستخدم لتجميع محتوى الرأس في جدول HTML.
- **عنصر صف الجدول (`tr`)**: يُستخدم لإنشاء صف في جدول HTML.
- **عنصر رأس الجدول (`th`)**: يُستخدم لإنشاء خلية رأس في جدول HTML.
- **عنصر جسم الجدول (`tbody`)**: يُستخدم لتجميع محتوى الجسم في جدول HTML.
- **عنصر خلية بيانات الجدول (`td`)**: يُستخدم لإنشاء خلية بيانات في جدول HTML.
- **عنصر تذييل الجدول (`tfoot`)**: يُستخدم لتجميع محتوى التذييل في جدول HTML.
- **عنصر `caption`**: يُستخدم لإضافة عنوان لجدول HTML.
- **خاصية `colspan`**: تُستخدم لتحديد عدد الأعمدة التي يجب أن تمتد عليها خلية الجدول.

:::interactive_editor

```html
<table>
  <caption>Exam Grades</caption>

  <thead>
    <tr>
      <th>Last Name</th>
      <th>First Name</th>
      <th>Grade</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>Davis</td>
      <td>Alex</td>
      <td>54</td>
    </tr>

    <tr>
      <td>Doe</td>
      <td>Samantha</td>
      <td>92</td>
    </tr>

    <tr>
      <td>Rodriguez</td>
      <td>Marcus</td>
      <td>88</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <td colspan="2">Average Grade</td>
      <td>78</td>
    </tr>
  </tfoot>
</table>
```

:::

## العمل مع أدوات HTML

- **مدقق HTML**: أداة تتحقق من صيغة شفرة HTML لضمان صحتها.
- **مفتش DOM**: أداة تسمح لك بفحص وتعديل هيكل HTML لصفحة ويب.
- **أدوات المطوِّر**: مجموعة أدوات مطوِّر الويب مدمجة مباشرة في المتصفح تساعدك على معالجة الأخطاء / تصحيح الأخطاء، وتحليل، وفحص صفحات الويب.
