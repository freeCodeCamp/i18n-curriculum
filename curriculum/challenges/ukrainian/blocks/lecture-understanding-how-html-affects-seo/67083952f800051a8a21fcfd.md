# --description--

SEO, або оптимізація для пошукових систем, — це практика, яка оптимізує вебсторінки, щоб вони ставали більш помітними та займали вищі позиції у пошукових системах. Один зі способів покращити SEO вашого сайту — надати короткий опис для вебсторінки за допомогою елемента `meta`. Ось приклад використання метаелемента для встановлення опису сторінки для сайту про садівництво:

```html
<meta
  name="description"
  content="Discover expert tips and techniques for gardening in small spaces, choosing the right plants, and maintaining a thriving garden."
/>
```

Встановивши атрибут `name` у значення `description`, ви забезпечуєте правильне тлумачення цих метаданих браузерами, пошуковими системами та іншими вебінструментами. Атрибут `content` — це місце, де ви розміщуєте свій опис. Рекомендується тримати описи короткими й лаконічними, оскільки пошукові системи часто обрізають опис залежно від макету сторінки результатів.

Подібно до інших типів елементів `meta`, опис `meta` не буде видимим безпосередньо на вебсторінці. Одне з місць, де можна побачити опис сторінки — це фрагмент на сторінці результатів пошукової системи. Ось кілька прикладів фрагментів результатів для сабреддіту freeCodeCamp і репозиторіїв GitHub:

```sh
r\FreeCodeCamp: This is the official subreddit for the freeCodeCamp.org community. Learn to
code for free together with millions of other people...
```

```sh
Our full-stack web development and machine learning curriculum is completely free and self-
paced. We have thousands of interactive coding challenges to help you...
```

У прикладах описи сторінок розташовані безпосередньо під посиланнями на сайти. За кілька секунд користувачі можуть отримати загальне уявлення про те, про що сторінка, і вирішити натиснути на посилання для отримання додаткової інформації.

Хоча описи `meta` безпосередньо не впливають на рейтинг сайту в пошукових системах, наявність сильного опису може призвести до збільшення трафіку на ваш вебсайт.
