# --description--

Im Spanischen verwendet man zur Bestätigung von Informationen das Wort `sí`.

Es ist sehr wichtig, das Akzentzeichen über dem Buchstaben `i` zu setzen. Ohne das Akzentzeichen bedeutet `si` „wenn“, also eine Bedingung, was die Bedeutung Ihres Satzes verändern kann.

Julieta bestätigt, dass Valeria Product Owner ist.
