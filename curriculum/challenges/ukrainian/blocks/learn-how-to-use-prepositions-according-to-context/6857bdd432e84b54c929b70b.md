# --explanation--

`Storage room` — це приміщення, яке використовується для зберігання речей, які не використовуються негайно. Наприклад:

`The extra cables are in the storage room.` — це означає, що кабелі зберігаються там.

`Go back` означає повернутися або рухатися у протилежному напрямку. Наприклад:

`Go back to the entrance if you're lost.` — це означає повернутися до початкової точки.

`Towards` означає у напрямку чогось. Наприклад:

`Walk towards the lobby.` — це вказує комусь рухатися в тому напрямку.
