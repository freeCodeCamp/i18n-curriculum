# --interactive--

이전 수업에서는 `console.log()`를 소개했지만 이번 수업에서는 그 목적과 사용법을 더 깊이 다룹니다.

JavaScript에서 `console.log()`는 브라우저 콘솔에 메시지나 정보를 출력하는 간단하면서도 강력한 도구입니다. 주로 개발자가 프로그램 작업 중에 코드를 디버깅하고 검사할 때 사용합니다.

`console.log()`를 사용해 텍스트나 변수를 콘솔에 기록하여 코드가 제대로 실행되는지 확인할 수 있습니다.

`console.log()`를 사용하려면 출력하려는 값이나 메시지를 소괄호 안에 넣어 메서드를 호출하세요. 다음은 몇 가지 예입니다:

:::interactive_editor

```js
console.log("Hello, world!");

let num = 5;
console.log(num); // 5
```

:::

첫 번째 예는 브라우저 콘솔에 `Hello, world!`를 출력하고, 두 번째 예는 값 `5`를 출력합니다.

다음은 `console.log()`를 사용하는 또 다른 예입니다:

:::interactive_editor

```js
let name = "Alice";
console.log("Hello, " + name + "!"); // Hello, Alice!
```

:::

또한 여러 값을 쉼표로 구분하여 `console.log()`에 전달할 수 있습니다. 예를 들어:

:::interactive_editor

```js
let name = "Alice";
let age = 25;
console.log("Name:", name, "Age:", age); // Name: Alice Age: 25
```

:::

이 방법은 여러 정보를 한 번에 기록할 때 유용합니다.

`console.log()` 메서드는 코드 실행을 모니터링하는 데 도움을 주어 실수를 발견하고 프로그램 동작을 이해하기 쉽게 만듭니다.
