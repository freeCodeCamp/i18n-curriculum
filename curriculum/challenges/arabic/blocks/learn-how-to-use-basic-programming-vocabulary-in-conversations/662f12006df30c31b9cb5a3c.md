# --description--

يشير `concept` عمومًا إلى فكرة أو مبدأ يُستخدم لتخطيط أو تحديد الإجراءات والعمليات. في مواضيع مختلفة، بما في ذلك البرمجة، تُعد `concepts` الأفكار أو النظريات الأساسية.

على سبيل المثال، `Object-oriented programming is a concept in computer programming.`
