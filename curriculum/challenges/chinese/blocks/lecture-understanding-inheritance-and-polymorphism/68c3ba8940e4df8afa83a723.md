# --description--

继承是我们将要介绍的面向对象编程（OOP）的下一个关键概念。

让我们更深入地了解这个概念以及它如何让你编写可重用的代码。

通过继承，子类（或子类）可以使用基类（或父类）的属性和方法。这允许你重用代码，创建清晰的类层次结构，并在不重写所有内容的情况下自定义行为。你可以通过扩展现有方法或在子类中重写它们来进行自定义。

以下是继承的基本语法：

```py
class Parent:
    # Parent attributes and methods

class Child(Parent):
    # Child inherits, extends, and/or overrides where necessary
```

为了让 `Child` 类继承自 `Parent` 类，你必须将 `Parent` 传递给 `Child`。

这种风格称为单继承，因为子类仅从一个父类继承。

这是一个示例：

```py
class Animal:
    def __init__(self, name):
        self.name = name

    def sound(self):
        return f'{self.name} makes a sound'

class Dog(Animal):
    bark = 'woof! woof!! woof!!!'

jack = Dog('Jack')
print(jack.sound())  # Jack makes a sound
print(jack.bark)  # woof! woof!! woof!!!
```

你可以看到我们能够在子类 `Dog` 中重用父类 `Animal` 的 `self.name` 属性和 `sound()` 方法。

让我们在子类 `Dog` 中覆写父类 `Animal` 的 `sound()` 方法，这样我们就可以让 `sound()` 使用 `bark` 类变量：

```py
class Animal:
    def __init__(self, name):
        self.name = name

    def sound(self):
        return f'{self.name} makes a sound.'

class Dog(Animal):
    bark = 'woof! woof!! woof!!!'

    # Override sound() to use bark class variable
    def sound(self):
        return f'{self.name} barks {self.bark}'

jack = Dog('Jack')
print(jack.sound())  # Jack barks woof! woof!! woof!!!
```

如果你想保留 `sound()` 的返回值并稍后添加 `bark` 类变量，你可以通过使用 `super()` 函数来扩展 `sound()`：

```py
class Animal:
    def __init__(self, name):
        self.name = name

    def sound(self):
        return f'{self.name} makes a sound'

class Dog(Animal):
    bark = 'woof! woof!! woof!!!'

    # Call Animal.sound(), then append bark
    def sound(self):
        base = super().sound()
        return f'{base}, then {self.name} barks {self.bark}'

jack = Dog('Jack')
print(jack.sound())  # Jack makes a sound, then Jack barks woof! woof!! woof!!!
```

在此示例中，`base` 是调用 `Animal` 类的 `sound()` 方法的结果，然后我们将 `Dog` 类的特定声音追加到它上面。这样，你可以扩展父类 `Animal` 的功能性，同时仍保持其原有行为。

还有多重继承，其中子类可以继承自多个父类。

以下是多重继承的基本语法：

```py
class Parent:
    # Attributes and methods for Parent

class Child:
    # Attributes and methods for Child

class GrandChild(Parent, Child):
    # GrandChild inherits from both Parent and Child
    # GrandChild can combine or override behavior from each
```

演示多重继承的一个简单方法是用青蛙，它既能在陆地上行走，也能在水中游泳：

```py
class Walker:
    def walk(self):
        return 'I can walk on land'

class Swimmer:
    def swim(self):
        return 'I can swim in water'

# Amphibian inherits from both Walker and Swimmer
class Amphibian(Walker, Swimmer):
    def __init__(self, name):
        self.name = name

    def introduce(self):
        return f"I'm {self.name} the frog. {self.walk()} and {self.swim()}."

frog = Amphibian('Freddy')
print(frog.introduce())
# Output: I'm Freddy the frog. I can walk on land and I can swim in water.
```
