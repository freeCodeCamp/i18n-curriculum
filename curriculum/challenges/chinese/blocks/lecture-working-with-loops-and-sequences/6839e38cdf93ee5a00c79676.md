# --description--

在上一课中，你已经了解了列表数据类型，并学习了如何访问列表中的元素以及列表切片。在本课中，你将继续学习列表以及一些关联的常用方法，如 `append()`、`pop()` 和 `sort()`。

我们将首先查看的方法是 `append()` 方法。它用于将一项添加到列表的末尾。下面是使用 `append()` 方法将数字 `6` 添加到数字列表中的示例：

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

如果你想将一个列表添加到另一个列表的末尾，你也可以像这样使用 `append()` 方法：

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

注意整个 `even_numbers` 列表是嵌套在 `numbers` 列表中的。

但是如果你想将 `even_numbers` 列表中的所有单个数字添加到 `numbers` 列表的末尾，那么你可以使用 `extend()` 方法。

`extend()` 方法类似于 `append()` 方法，但使用 `extend()` 你可以将一个列表中的多个元素添加到另一个列表中。下面是一个示例，演示如何将数字 `6`、`8` 和 `10` 从一个列表添加到 `numbers` 列表的末尾：

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

正如你所见，嵌套列表消失了，现在它只是一个数字列表。

要在列表的特定索引处插入元素，你可以使用 `insert()` 方法。该方法接受两个参数：你希望插入新项的索引和你想插入的项。

下面是使用 `insert()` 方法的示例：

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

以下代码将在 `numbers` 列表的索引 `2` 处插入数字 `2.5`。

如果你想从列表中移除一个元素，可以使用 `remove()` 方法。`remove()` 方法将要移除的元素值作为参数：

```py
numbers = [10, 20, 30, 40, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50]
```

重要的是要注意，此方法只会移除第一处出现的项，而不是全部。

```py
numbers = [10, 20, 30, 40, 50, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50, 50]
```

要删除列表中指定索引处的元素，你可以像这样使用 `pop()` 方法：

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

如果你没有为 `pop` 方法指定元素，则会移除最后一个元素。

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

如果你需要清空列表，那么你可以像这样使用 `clear()` 方法：

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

接下来我们将查看的 `sort()` 方法。此方法用于就地排序元素。以下是就地排序一个随机数字列表的示例：

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

与 `sort()` 方法相反，`sorted()` 函数适用于任何可迭代对象，并返回一个新的排序列表，而不是修改原始列表。例如：

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(numbers) # [19, 2, 35, 1, 67, 41]
print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
```

作为提醒，迭代器是一种特殊类型的对象，你可以对其进行循环，从而一次访问每一项。你将在后续课程中学习更多关于 Python 中循环的工作原理。

`sort()` 方法和 `sorted()` 函数都接受可选的 `key` 和 `reverse` 参数。你将在未来学习元组的课程中了解这些可选参数的更多内容。你还将在未来的课程中了解更多像 `sorted()` 这样的内置函数。

接下来我们将查看的 `reverse()` 方法。此方法将原地反转元素列表，如下所示：

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

我们将要看的最后一个方法是 `index` 方法。它用于查找元素在列表中首次出现的索引。下面是使用 `index` 方法在 `programming_languages` 列表中查找语言 `'Java'` 的示例：

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

如果找不到该元素，则 Python 会抛出 `ValueError`：

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: 'JavaScript' is not in list
"""
```

Python 列表还有一些其他的方法，但这组初始的方法列表是一个很好的起点。
