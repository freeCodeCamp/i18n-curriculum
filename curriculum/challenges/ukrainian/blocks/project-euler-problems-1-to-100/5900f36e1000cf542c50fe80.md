# --description--

Якщо перелічити всі натуральні числа менші за 10, які є кратними 3 або 5, отримаємо 3, 5, 6 і 9. Сума цих кратних дорівнює 23.

Знайдіть суму всіх кратних 3 або 5, менших за задане значення параметра `number`.

# --solutions--

```js
const multiplesOf3Or5 = (number) => {
  var total = 0;

  for(var i = 0; i < number; i++) {
    if(i % 3 == 0 || i % 5 == 0) {
      total += i;
    }
  }
  return total;
};
```
