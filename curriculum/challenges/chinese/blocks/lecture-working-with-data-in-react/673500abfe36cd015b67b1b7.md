# --description--

React 中的条件式渲染允许你创建动态用户界面。它用于根据你的应用中的某些条件或状态显示不同的内容。

使用条件式渲染的最常见方法包括使用 `if` 语句、三元 (`?:`) 操作符和逻辑与 (`&&`) 操作符。

条件式渲染最简单的形式使用 `if` 语句。下面是一个示例：

```jsx
function Greeting({ isLoggedIn }) {
  if (isLoggedIn) {
    return <h1>Welcome back!</h1>;
  }
  return <h1>Please sign in</h1>;
}
```

在此示例中，`Greeting` 组件查看 `isLoggedIn` 属性。如果它是 `true`，则返回欢迎消息，否则提示用户登录。

下面是一个在父级 `App` 组件内使用 `Greeting` 组件的示例：

```jsx
function App() {
  return (
    <div className="App">
      <Greeting isLoggedIn={false} />
    </div>
  );
}
```

对于更简单的条件式，三元操作符（`?:`）通常直接在 JSX 中使用。它允许内联条件式渲染，这可以使你的代码更简洁：

```jsx
function Greeting({ isLoggedIn }) {
  return <h1>{isLoggedIn ? "Welcome back!" : "Please sign in."}</h1>;
}
```

这段代码实现了与前一个示例相同的结果，但形式更紧凑。三元操作符检查 `isLoggedIn` 并渲染相应的消息。

另一个常见的条件式渲染模式是使用逻辑 AND (`&&`) 操作符。当你想根据条件渲染某些内容或不渲染任何内容时，这尤其有用：

```jsx
function Notification({ message }) {
  return (
    <div>
      {message && <p>{message}</p>}
    </div>
  );
}
```

在此示例中，带有消息的段落元素仅在 `message` 属性为真值时才被渲染。如果 `message` 为假值——意味着它是空字串、`null` 或 `undefined`，则不会渲染任何内容到屏幕。

通过掌握这些条件式渲染的技术，你可以构建更具交互性和用户友好的应用，能够适应不断变化的数据和用户交互。
