# --description--

Per ottenere una lista di riassunti di libri, puoi usare la proprietà `about` di ogni libro.

Crea una funzione `getBookSummaries` con un singolo parametro, che accetta un array di oggetti libro.

La funzione `getBookSummaries` dovrebbe restituire un array di stringhe che rappresentano il riassunto di ogni libro.
