# --interactive--

Die `map`-Methode ist eine leistungsstarke und weit verbreitete Funktion in JavaScript, die auf Arrays angewendet wird. Sie dient dazu, ein neues Array zu erstellen, indem eine gegebene Funktion auf jedes Element des ursprünglichen Arrays angewendet wird.

Diese Methode verändert das ursprüngliche Array nicht, sondern gibt stattdessen ein neues Array zurück, das die Ergebnisse der auf jedes Element angewendeten Funktion enthält.

Hier ist ein Beispiel für die Verwendung der `map`-Methode auf einem Array von Zahlen:

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map((num) => num * 2);
console.log(numbers); // [1, 2, 3, 4, 5]
console.log(doubled); // [2, 4, 6, 8, 10]
```

:::

Um ein neues Array zu erstellen, in dem jede Zahl verdoppelt wird, verwenden wir die `map`-Methode. Die `map`-Methode akzeptiert eine Callback-Funktion, bei der die Funktion auf jedes einzelne Element im Array aufgerufen wird.

In diesem Fall wird jede Zahl im Array mit `2` multipliziert. Das Ergebnis ist ein neues Array mit den Zahlen `2,4,6,8,10`.

Die Callback-Funktion kann bis zu drei Argumente annehmen.

Das erste Argument ist das aktuell verarbeitete Element.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element) => {
  console.log("Element:", element);
  return element * 2;
});
```

:::

Das zweite Argument ist der Index des aktuell verarbeiteten Elements.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index) => {
  console.log("Element:", element);
  console.log("Index:", index);
  return element * 2;
});
```

:::

Das dritte Argument ist das Array, auf dem `map` aufgerufen wird.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index, array) => {
  console.log("Element:", element);
  console.log("Index:", index);
  console.log("Array:", array);
  return element * 2;
});
```

:::

Das Verständnis und die effektive Nutzung der `map`-Methode können Ihre Fähigkeit, mit Arrays in JavaScript zu arbeiten, erheblich verbessern. In zukünftigen Lektionen werden wir tiefer in fortgeschrittene Anwendungen von `map` eintauchen und erkunden, wie sie ein mächtiges Werkzeug zum Erstellen dynamischer und effizienter Programme sein kann.
