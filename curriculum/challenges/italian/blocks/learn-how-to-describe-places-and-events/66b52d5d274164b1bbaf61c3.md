# --explanation--

Il tempo `Present Continuous` si usa per descrivere azioni che stanno avvenendo nel momento in cui si parla. Si forma usando il verbo `to be` (`am, is, are`) + la forma base del verbo + `-ing`. Per esempio:

- `She is coding a new feature right now.` - Lei sta programmando una nuova funzionalità in questo momento.

- `They are testing the software.` - Loro stanno testando il software proprio adesso.
