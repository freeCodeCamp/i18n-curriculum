# --description--

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/the-box-model/box-model-lines.png" alt="an image showing the odin project website surrounded by red borders indicating that the site is build out of multiple components" style="width: 95%; height: 95%;">

さて、上の画像にはいくつかの円があるかもしれませんが、レイアウトに関しては、それらは円ではなく長方形のボックスのように組み合わさっています。最終的にウェブページのレイアウトを決めてすべての要素を配置するというのは、これらのボックスをどのようにネストして積み重ねるかを決めることです。

唯一の本当の複雑さは、パディング、マージン、ボーダーを使ってこれらのボックスのサイズやボックス間のスペースを操作する方法がたくさんあることです。しかし簡単にまとめると：

- `padding` はボックスのボーダーとコンテンツの間のスペースを増やします。
- `margin` はボックスのボーダーと隣接するボックスのボーダーの間のスペースを増やします。
- `border` はマージンとパディングの間にスペース（たとえ1ピクセルか2ピクセルでも）を追加します。

図をよく見て理解してください。
