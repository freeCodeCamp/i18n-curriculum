# --description--

`Maybe next time` se usa para sugerir cortésmente la posibilidad de que alguien participe o se una a una actividad o evento en el futuro. Puede usarse de forma independiente o como parte de una oración.

## --blanks--

`Maybe`

### --feedback--

Esta palabra se usa para mostrar que algo es posible pero no seguro. La primera letra está en mayúscula.

---

`next`

### --feedback--

Esta palabra significa la cosa que viene inmediatamente después.

---

`time`

### --feedback--

Esta palabra se refiere a minutos, horas, días o un momento cuando algo sucede.
