# --explanation--

一個範例是 `the "Student" class inherits properties and methods from the "Person" class`。

- `Student` 和 `Person` 都是 `classes`。`class` 定義了用來建立具有特定資料和行為的物件的藍圖。

- `Student` 繼承自 `Person` 的特性，表示它不需要重新定義共用的特質 — 它會重複使用它們。

- `Properties` 是類別內的變數或資料，例如 `name` 或 `age`，用來描述物件。

- `Methods` 是在類別中定義的函式（程式），例如 `enroll()` 或 `introduce()`，用來描述物件可以執行的動作。
