# --description--

ジェームズは、話している瞬間やその周辺で起こっている動作について話すために`Present Continuous`時制を使います。

`Present Continuous`は、動詞`to be`（`am/is/are`）の現在形に続けて、主動詞の`-ing`形を付けて作られます。例えば、`I am using`は`I am using special tools`の中で、ジェームズが現在取り組んでいる動作を示しています。

この時制は、一時的または進行中の動作に使われることが多いです。ほかの例としては：

- `She is studying for the interview.` - 彼女は現在勉強に集中しています。

- `They are working on a new project.` - 彼らは現在プロジェクトに関わっています。

- `He is walking to the station.` - 彼は駅へ歩いている途中です。
