# --explanation--

`Present Simple` 시제에서는 부정형을 만들 때 `doesn't` 또는 3인칭 단수 주어(`isn't`)에 대해 `he, she, it`을 사용합니다.

- 동작 동사에는 `doesn't`을 사용하여 무언가가 일어나지 않음을 나타냅니다. 예: `It doesn't work.`

- `isn't`의 형태에는 `to be`를 사용하여 무언가가 사실이 아님을 나타냅니다. 예: `isn't set up.`

이 문장에서는 기능이 예상대로 작동하지 않는 이유를 올바르게 설명하려면 `doesn't`와 `isn't`가 모두 필요합니다.
