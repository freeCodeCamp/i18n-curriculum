# --explanation--

`Present Simple`時制は、一時的な出来事ではなく、常に真である行動や状況を指すことが多いです。例えば：

- `The system updates automatically every night.` - 定期的に行われる予定されたルーティンを指します。

- `The app saves data to the cloud.` - アプリの機能に関する一般的な事実を述べています。

`Securely`は、安全や適切な動作を確保するために、しっかりと固定されたり、位置づけられたり、行われたりしていることを意味します。

この文では、`isn't connected securely`が電源ケーブルが正しく接続されていない状態を指し、そのためコンピューターが起動しないことを表しています。
