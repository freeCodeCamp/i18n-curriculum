# --interactive--

CSS 默认情况下不会对文字应用任何阴影。 这是一个没有任何阴影的段落示例。 但是，如果你确实需要添加阴影，可以使用 `text-shadow` 属性。

```css
text-shadow: /* Values */
```

在 CSS 中，你可以通过 `X` 偏移、`Y` 偏移、模糊半径和颜色的组合来描述阴影。 首先，你需要指定 `X` 和 `Y` 偏移，它们分别代表阴影与文本的水平和垂直距离。 这些值是必需的。

下面是一个如何在阴影上应用 `X` 和 `Y` 偏移的示例。 我们应用 `text-shadow` 属性，其 x 偏移为 `3px`，y 偏移为 `2px`：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 3px 2px;
}
```

:::

在浏览器中，文本和阴影的外观相似，但我们也可以通过指定偏移之前或之后的值来定制阴影的颜色。 我们来设置阴影颜色。 我们将在这里使用十六进制颜色，但你也可以使用任何有效的颜色格式。 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 3px 2px #00ffc3;
}
```

:::

我们将颜色写在偏移之后，但如果你愿意，也可以将颜色写在偏移之前。 这是等价的：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: #00ffc3 3px 2px;
}
```

:::

在浏览器中，阴影会有特定的颜色，因此我们可以很容易地将其与文本区分开来。 我们区分了阴影和文字，了解正负值对阴影偏移的影响也很重要。 `X` 偏移和 `Y` 偏移的正值将分别使阴影向右和向下移动，而负值将使阴影向左和向上移动。 下面是一个使用负值的例子：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: -3px -2px #00ffc3;
}
```

:::

现在，阴影将相对文字向左和向上移动。 很好。 但阴影看起来并不美观，因为它看起来和原来的文字一模一样，只是放在了文字的后面。 为了让它看起来更好看，我们需要添加第三个值，即模糊半径。 这个值是可选的，但能让阴影看起来更平滑、更细腻。 模糊半径的默认值为零。 数值越大，模糊程度越高，这意味着阴影会变浅。

在这里，我们将模糊半径设置为 `3px`，偏移也恢复为正值：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 3px 2px 3px #00ffc3;
}
```

:::

现在，它开始像一个阴影了。 它更模糊、更细腻，因此我们可以把注意力集中在主要文字上，而阴影则为背景增添了一些风格。 在此基础上，我们可以对数值稍作调整，直到找到适合我们需要的偏移、颜色和模糊半径组合。

知道文字可以有多个阴影也很有帮助。 你只需将它们写入同一个 `text-shadow` 属性中，并用逗号分隔。 他们将从前到后分层应用，第一个阴影位于顶层。

下面是一个分层应用了三种阴影的段落示例。 请注意，所有这些阴影都定义在同一个 `text-shadow` 属性中，并用逗号隔开：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 
    3px 2px 3px #00ffc3, 
    -3px -2px 3px #0077ff, 
    5px 4px 3px #dee7e5;
}
```

:::

`text-shadow` 是一个功能强大的 CSS 属性，可让你的文本脱颖而出。 它还能增强网页应用的整体设计。 通过选择偏移、模糊和颜色，你可以创建多种阴影效果。
