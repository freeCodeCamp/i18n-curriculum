# --description--

In React, aggiornare gli array nello stato è abbastanza semplice, ma è facile commettere errori, soprattutto se vieni da JavaScript puro dove puoi modificare gli array direttamente.

In React, lo stato è trattato come immutabile così da poter riconoscere le modifiche e aggiornare correttamente l'interfaccia utente.

Vediamo come puoi aggiornare gli array contenuti nello stato in React.

Uno degli errori più comuni quando si aggiornano gli array nello stato di React è modificare direttamente l'array usando metodi come `push()`, `pop()` o `splice()`. Questi metodi mutano l'array originale, cosa che React non permette.

React si basa su un nuovo riferimento all'array per rilevare le modifiche, quindi modificare direttamente l'array può impedire al componente di renderizzarsi di nuovo come previsto.

Ecco un esempio di utilizzo del metodo `push()` per aggiungere a un array nello stato, che non funzionerà:

```jsx
import { useState } from "react";

function ItemsList() {
  const [items, setItems] = useState([
    { id: 0, name: "Item 1" },
    { id: 1, name: "Item 2" },
    { id: 2, name: "Item 3" },
  ]);

  const addItem = () => {
    const newItem = { id: items.length + 1, name: `Item ${items.length + 1}` };
    items.push(newItem); // This modifies the original array
    setItems(items); // React will not detect this change
  };

  return (
    <div>
      <button onClick={addItem}>Add Item</button>
      <ul>
        {items.map((item) => (
          <li key={item.id}>{item.name}</li>
        ))}
      </ul>
    </div>
  );
}

export default ItemsList;
```

Se clicchi il pulsante `Add Item`, non succede nulla nell'interfaccia utente.

Potrebbe anche venirti voglia di rimuovere elementi dall'array con il metodo `pop()`:

```js
const removeItem = () => {
  items.pop(); // Modifies the original array
  setItems(items); // React will not detect this change, either
};
```

Per aggiornare un array nello stato, la chiave è creare un nuovo array, fare le tue operazioni e passarlo a React, invece di mutare l'array esistente.

Poiché è un array nuovo, React saprà che lo stato è cambiato e attiverà una nuova renderizzazione.

Ecco come puoi aggiungere all'array `items` usando l'operatore di espansione:

```js
const addItem = () => {
  const newItem = {
    id: items.length + 1,
    name: `Item ${items.length + 1}`,
  };

  // Creates a new array
  setItems((prevItems) => [...prevItems, newItem]);
};
```

`[...prevItems, newItem]` crea un nuovo array copiando tutti gli elementi dell'array `items` esistente nello stato, poi aggiunge `newItem` alla fine, che incrementa `id` e il numero dell'elemento.

Se vuoi rimuovere qualcosa dall'array, puoi usare il metodo `filter()`, che restituisce un nuovo array dopo aver filtrato ciò che vuoi rimuovere:

```js
const removeItem = (id) => {
  setItems((prevItems) => prevItems.filter((item) => item.id !== id));
};
```

Ecco il codice completo:

```jsx
import { useState } from "react";

function ItemsList() {
  const [items, setItems] = useState([
    { id: 0, name: "Item 1" },
    { id: 1, name: "Item 2" },
    { id: 2, name: "Item 3" },
  ]);

  const addItem = () => {
    const newItem = { id: items.length + 1, name: `Item ${items.length + 1}` };
    setItems((prevItems) => [...prevItems, newItem]); // Creates a new array
  };

  const removeItem = (id) => {
    setItems((prevItems) => prevItems.filter((item) => item.id !== id)); // Creates a new array
  };

  return (
    <div>
      <button onClick={addItem}>Add Item</button>
      <ul>
        {items.map((item) => (
          <li key={item.id}>
            {item.name}{" "}
            <button onClick={() => removeItem(item.id)}>Remove</button>
          </li>
        ))}
      </ul>
    </div>
  );
}

export default ItemsList;
```

Questi sono alcuni modi comuni per aggiornare un array nello stato.
