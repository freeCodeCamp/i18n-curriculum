# --description--

令 $H$ 为由方程 $12x^2 + 7xy - 12y^2 = 625$ 定义的双曲线。

接下来，定义 $X$ 为点 (7, 1)。可以看出 $X$ 在 $H$ 中。

现在我们定义 $H$ 中的点序列 $\{P_i : i ≥ 1\}$，为：

- $P_1 = (13, \frac{61}{4})$。
- $P_2 = (\frac{-43}{6}, -4)$。
- 对于 $i > 2$，$P_i$ 是 $H$ 中唯一不同于 $P_{i - 1}$ 且使得直线 $P_iP_{i - 1}$ 与直线 $P_{i - 2}X$ 平行的点。可以证明 $P_i$ 是良好定义的，并且其坐标始终是有理数。

<img alt="animation showing defining points P_1 to P_6" src="https://cdn.freecodecamp.org/curriculum/project-euler/sequence-of-points-on-a-hyperbola.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

已知 $P_3 = (\frac{-19}{2}, \frac{-229}{24})$，$P_4 = (\frac{1267}{144}, \frac{-37}{12})$ 和 $P_7 = (\frac{17\\,194\\,218\\,091}{143\\,327\\,232}, \frac{274\\,748\\,766\\,781}{1\\,719\\,926\\,784})$。

查找 $n = {11}^{14}$ 时的 $P_n$，格式如下：如果 $P_n = (\frac{a}{b}, \frac{c}{d})$，其中分数为最简形式且分母为正，则答案为 $(a + b + c + d)\bmod 1\\,000\\,000\\,007$。

对于 $n = 7$，答案将是：$806\\,236\\,837$。
