# --description--

在這個專案中，你將建置一個應用程式，透過名稱或 ID 搜尋 RPG 遊戲中的生物，並將結果顯示給使用者。為了取得生物的資料和影像，你將使用 freeCodeCamp 的 <a href="https://rpg-creature-api.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">RPG Creature API</a>。

 **注意：** 前 13 個步驟必須在 `index.html` 檔案中完成。 

**目標：** 建置一個功能上類似於 <a href="https://rpg-creature-search-app.freecodecamp.rocks" target="_blank" rel="noopener noreferrer nofollow">https://rpg-creature-search-app.freecodecamp.rocks</a> 的應用程式。

**使用者故事：**

1. 你應該有一個 `input` 元素，其 `id` 為 `"search-input"`，且為必填。
1. 你應該有一個 `button` 元素，其 `id` 為 `"search-button"`。
1. 你應該有一個 `id` 為 `"creature-name"` 的元素。
1. 你應該有一個 `id` 為 `"creature-id"` 的元素。
1. 你應該有一個 `id` 為 `"weight"` 的元素。
1. 你應該有一個 `id` 為 `"height"` 的元素。
1. 你應該有一個 `id` 為 `"types"` 的元素。
1. 你應該有一個 `id` 為 `"hp"` 的元素。
1. 你應該有一個 `id` 為 `"attack"` 的元素。
1. 你應該有一個 `id` 為 `"defense"` 的元素。
1. 你應該有一個 `id` 為 `"special-attack"` 的元素。
1. 你應該有一個 `id` 為 `"special-defense"` 的元素。
1. 你應該有一個 `id` 為 `"speed"` 的元素。
1. 當 `#search-input` 元素包含值 `Red` 且點擊 `#search-button` 元素時，應該出現一個顯示文字 `"Creature not found"` 的警示。
1. 當 `#search-input` 元素包含值 `Pyrolynx` 且點擊 `#search-button` 元素時，`#creature-name`、`#creature-id`、`#weight`、`#height`、`#hp`、`#attack`、`#defense`、`#special-attack`、`#special-defense` 和 `#speed` 元素中的值應分別為 `PYROLYNX`、`#1` 或 `1`、`Weight: 42` 或 `42`、`Height: 32` 或 `32`、`65`、`80`、`50`、`90`、`55` 和 `100`。
1. 當 `#search-input` 元素包含值 `Pyrolynx` 且點擊 `#search-button` 元素時，應該在 `#types` 元素內新增一個包含文字 `FIRE` 的單一元素。每次搜尋之間應清除 `#types` 元素內容。
1. 當 `#search-input` 元素包含值 `2` 且點擊 `#search-button` 元素時，`#creature-name`、`#creature-id`、`#weight`、`#height`、`#hp`、`#attack`、`#defense`、`#special-attack`、`#special-defense` 和 `#speed` 元素中的值應分別為 `AQUOROC`、`#2` 或 `2`、`Weight: 220` 或 `220`、`Height: 53` 或 `53`、`85`、`90`、`120`、`60`、`70` 和 `40`。
1. 當 `#search-input` 元素包含值 `2` 且點擊 `#search-button` 元素時，應該在 `#types` 元素內分別添加包含文字值 `WATER` 和 `ROCK` 的兩個元素。`#types` 元素內容應在每次搜尋之間清除。

const creatureID = document.getElementById('creature-id');
const creatureName = document.getElementById('creature-name');
const specialName = document.getElementById('special-name');
const specialDescription = document.getElementById('special-description');
const types = document.getElementById('types');
const height = document.getElementById('height');
const weight = document.getElementById('weight');
const hp = document.getElementById('hp');
const attack = document.getElementById('attack');
const defense = document.getElementById('defense');
const specialAttack = document.getElementById('special-attack');
const specialDefense = document.getElementById('special-defense');
const speed = document.getElementById('speed');
const searchForm = document.getElementById('search-form');
const searchInput = document.getElementById('search-input');

const getCreature = async () => {
  try {
    const creatureNameOrId = searchInput.value.toLowerCase();
    const response = await fetch(
      `https://rpg-creature-api.freecodecamp.rocks/api/creature/${creatureNameOrId}`
    );
    const data = await response.json();

    // Set Creature info
    creatureName.textContent = `${data.name.toUpperCase()}`;
    creatureID.textContent = `#${data.id}`;
    weight.textContent = `Weight: ${data.weight}`;
    height.textContent = `Height: ${data.height}`;
    specialName.textContent = data.special.name;
    specialDescription.textContent = data.special.description;

    // Set stats
    hp.textContent = data.stats[0].base_stat;
    attack.textContent = data.stats[1].base_stat;
    defense.textContent = data.stats[2].base_stat;
    specialAttack.textContent = data.stats[3].base_stat;
    specialDefense.textContent = data.stats[4].base_stat;
    speed.textContent = data.stats[5].base_stat;

    // Set types
    types.innerHTML = data.types
      .map(obj => `<span class="type ${obj.name}">${obj.name}</span>`)
      .join('');
  } catch (err) {
    resetDisplay();
    alert('Creature not found');
    console.log(`Creature not found: ${err}`);
  }
};

const resetDisplay = () => {
  // reset stats
  creatureName.textContent = '';
  creatureID.textContent = '';
  height.textContent = '';
  weight.textContent = '';
  types.innerHTML = '';
  specialName.innerHTML = '';
  specialDescription.innerHTML = '';
  hp.textContent = '';
  attack.textContent = '';
  defense.textContent = '';
  specialAttack.textContent = '';
  specialDefense.textContent = '';
  speed.textContent = '';
};

searchForm.addEventListener('submit', e => {
  e.preventDefault();
  getCreature();
});
