# --description--

`Different from` é uma frase que você usa para mostrar que duas coisas não são iguais. Por exemplo:

- `My phone is different from yours.` - Isso significa que os dois telefones não são iguais.

- `Her job is different from mine.` - Isso significa que o trabalho dela e o seu trabalho não são os mesmos.

Note que `different` é o adjetivo e geralmente é seguido por `from` nessa estrutura.
