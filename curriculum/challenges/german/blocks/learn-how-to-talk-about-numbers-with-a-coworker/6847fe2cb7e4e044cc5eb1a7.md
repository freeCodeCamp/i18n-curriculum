# --explanation--

Die korrekte Schreibweise von ⅗ in Worten ist `three-fifths`. Verwenden Sie dieses Format:

Zähler (in Kardinalform) + Nenner (im Plural, wenn der Zähler > 1 ist). Zum Beispiel:

- ⅖ = `two-fifths`

- ⅘ = `four-fifths`

Dies ist hilfreich, wenn Sie Daten erklären oder finanzielle Details in gesprochenen Präsentationen teilen.
