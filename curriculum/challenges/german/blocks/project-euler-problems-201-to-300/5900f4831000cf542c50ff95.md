# --description--

Gegeben sind die Werte ganzer Zahlen $1 < a_1 < a_2 < \ldots < a_n$. Betrachten Sie die Linearkombination $q_1a_1 + q_2a_2 + \ldots + q_na_n = b$, wobei nur ganzzahlige Werte $q_k ≥ 0$ verwendet werden.

Beachten Sie, dass für eine gegebene Menge von $a_k$ nicht alle Werte von $b$ möglich sind. Zum Beispiel gibt es für $a_1 = 5$ und $a_2 = 7$ keine $q_1 ≥ 0$ und $q_2 ≥ 0$, so dass $b$ gleich 1, 2, 3, 4, 6, 8, 9, 11, 13, 16, 18 oder 23 sein könnte.

Tatsächlich ist 23 der größte unmögliche Wert von $b$ für $a_1 = 5$ und $a_2 = 7$. Wir nennen daher $f(5, 7) = 23$. Ebenso lässt sich zeigen, dass $f(6, 10, 15)=29$ und $f(14, 22, 77) = 195$ gilt.

Finden Sie $\sum f(pq,pr,qr)$, wobei $p$, $q$ und $r$ Primzahlen sind und $p < q < r < 5000$.
