# --description--

`log in` とは、コンピューターやウェブサイトにアクセスするためにユーザー名とパスワードを入力することを意味します。例えば：

`Every morning, I log in to my computer to start work.` - これは毎朝、コンピューターを開いて仕事を始めるためにユーザー名とパスワードを入力することを意味します。

`check emails` とは、メールアカウントの新しいメッセージを見て読むことを意味します。例えば：

`After logging in, I check my emails to see if I have any new messages.` - これはコンピューターやメールアカウントを開いた後、誰かから新しいメッセージが届いているか確認することを意味します。
