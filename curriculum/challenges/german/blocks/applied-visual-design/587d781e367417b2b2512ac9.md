# --description--

CSS behandelt jedes HTML-Element als seine eigene Box, die üblicherweise als <dfn>CSS-Boxmodell</dfn> bezeichnet wird. Block-Elemente beginnen automatisch in einer neuen Zeile (denken Sie an Überschriften, Absätze und divs), während inline-Elemente innerhalb des umgebenden Inhalts sitzen (wie Bilder oder spans). Das Standardlayout der Elemente auf diese Weise wird als <dfn>normaler Fluss</dfn> eines Dokuments bezeichnet, aber CSS bietet die Eigenschaft position, um diesen zu überschreiben.

Wenn die Position eines Elements auf `relative` gesetzt wird, können Sie angeben, wie CSS es *relativ* zu seiner aktuellen Position im normalen Fluss der Seite verschieben soll. Es wird mit den CSS-Versatz-Eigenschaften `left` oder `right` sowie `top` oder `bottom` kombiniert. Diese geben an, um wie viele Pixel, Prozente oder ems das Element *weg* von seiner normalen Position verschoben wird. Das folgende Beispiel verschiebt den Absatz 10 Pixel vom unteren Rand weg:

```css
p {
  position: relative;
  bottom: 10px;
}
```

Die Änderung der Position eines Elements auf relativ entfernt es nicht aus dem normalen Fluss – andere Elemente um es herum verhalten sich weiterhin so, als wäre dieses Element an seiner Standardposition.

**Hinweis:** Die Positionierung gibt Ihnen viel Flexibilität und Kontrolle über das visuelle Layout einer Seite. Es ist wichtig zu bedenken, dass unabhängig von der Position der Elemente die zugrunde liegende HTML-Auszeichnung organisiert sein und beim Lesen von oben nach unten Sinn ergeben sollte. So greifen Nutzer mit Sehbeeinträchtigung (die auf Hilfsmittel wie Screenreader angewiesen sind) auf Ihre Inhalte zu.

Ändern Sie die `position` des `h2` auf `relative` und verwenden Sie einen CSS-Versatz, um es 15 Pixel vom `top` weg zu verschieben, wo es im normalen Fluss sitzt. Beachten Sie, dass dies keine Auswirkungen auf die Positionen der umgebenden h1- und p-Elemente hat.

# --hints--

Das `h2`-Element sollte eine `position`-Eigenschaft mit dem Wert `relative` haben.

```js
const h2Element =document.querySelector('h2')
const h2Style = window.getComputedStyle(h2Element);
assert.equal(h2Style?.position, 'relative');
```

Ihr Code sollte einen CSS-Versatz verwenden, um das `h2` relativ 15px vom `top` weg zu positionieren, wo es normalerweise sitzt.

```js
const h2Element =document.querySelector('h2')
const h2Style = window.getComputedStyle(h2Element);
assert.equal(h2Style?.top, '15px');
```
