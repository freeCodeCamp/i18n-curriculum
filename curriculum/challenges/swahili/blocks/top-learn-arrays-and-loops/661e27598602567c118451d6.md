# --description--

Mzunguko wa `for` ni aina nyingine ya mzunguko unaotumika kutekeleza kipande cha msimbo mara nyingi. Mzunguko wa `for` hutumika wakati idadi ya marudio inajulikana. Sintaksia ya mzunguko wa `for` ni kama ifuatavyo:

```javascript
for (initialization; condition; increment/decrement) {
  // code block to be executed
}
```

Kwa mfano, kipande cha msimbo kilicho hapa chini kinachapisha nambari kutoka 1 hadi 5:

```javascript
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
```

Ili rudia juu ya safu ya data kwa kutumia mzunguko wa `for`, unaweza kutumia sifa ya urefu wa safu hiyo. Kwa mfano, kipande cha msimbo kilicho hapa chini kinachapisha vipengele vya safu ya data:

```javascript
const arr = [1, 2, 3, 4, 5];
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}
```
