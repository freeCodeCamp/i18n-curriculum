# --interactive--

Schleifen in der Programmierung werden verwendet, um einen Codeblock mehrfach zu wiederholen.

Ein Beispiel für eine Schleife wäre, wenn Sie ein Programm entwerfen, das eine Liste von Elementen ausgeben muss. Sie könnten eine Schleife verwenden, um jedes einzelne Element der Liste auszugeben.

Ein weiteres Beispiel ist, wenn Sie ein Spiel entwerfen und eine Spielfigur über den Bildschirm bewegen möchten. Sie könnten eine Schleife verwenden, um die Figur bei jedem Durchlauf der Schleife um eine bestimmte Anzahl von Pixeln zu verschieben.

In JavaScript gibt es verschiedene Arten von Schleifen, die Sie verwenden können. In dieser Lektion behandeln wir die `for`-Schleife. Hier ist die grundlegende Syntax für eine `for`-Schleife:

```js
for (initialization; condition; increment or decrement) {
  // code block to be executed
}
```

Die Initialisierung wird vor Beginn der Schleife ausgeführt. Sie wird typischerweise verwendet, um eine Zählvariable zu initialisieren. Eine Zählvariable ist eine Variable, die verfolgt, wie oft die Schleife bereits durchlaufen wurde.

Die Bedingung wird vor jeder Iteration der Schleife ausgewertet. Eine Iteration ist ein einzelner Durchlauf der Schleife.

Wenn die Bedingung wahr ist, wird der Codeblock innerhalb der Schleife ausgeführt. Wenn die Bedingung falsch ist, stoppt die Schleife und Sie fahren mit dem nächsten Codeblock fort.

Der letzte Teil der Schleife ist die Inkrement-/Dekrement-Anweisung. Diese Anweisung wird nach jeder Iteration der Schleife ausgeführt. Sie wird typischerweise verwendet, um die Zählvariable zu erhöhen oder zu verringern.

:::interactive_editor

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

:::

Im ersten Teil des obigen Beispiels initialisieren wir eine Zählvariable `i` mit `0`. Es ist eine gängige Konvention, `i` als Zählvariable in einer `for`-Schleife zu verwenden.

Der nächste Schritt ist die Überprüfung der Bedingung. In diesem Fall wird geprüft, ob `i` kleiner als `5` ist. Da `i` `0` ist, ist die Bedingung wahr, und der Codeblock innerhalb der Schleife wird ausgeführt.

Der Codeblock innerhalb der Schleife gibt den Wert von `i` in der Konsole aus. Der Wert von `i` ist `0`, also zeigt die Konsole den Wert von `0` an.

Dann wird die Inkrement-Anweisung ausgeführt. In diesem Fall erhöhen wir `i` um `1`. Somit ist `i` jetzt `1`.

Anschließend prüfen wir die Bedingung erneut, nämlich ob `i` kleiner als `5` ist. Da `i` jetzt `1` ist, ist die Bedingung weiterhin wahr, und der Codeblock innerhalb der Schleife wird erneut ausgeführt.

Wir wiederholen diesen Vorgang so lange, bis die Bedingung falsch ist. In diesem Fall ist die Bedingung falsch, wenn `i` `5` ist, und die Schleife stoppt.

Wenn Sie mit Schleifen arbeiten, sollten Sie darauf achten, keine Bedingung zu erstellen, die immer wahr ist. Wenn das passiert, läuft die Schleife unendlich und Ihr Programm stürzt ab. Das nennt man eine Endlosschleife.

Es ist möglich, verschachtelte `for`-Schleifen zu erstellen. Eine verschachtelte Schleife ist eine Schleife, die innerhalb einer anderen Schleife liegt. Wir werden später Beispiele sehen, wann Sie das verwenden könnten.

Schleifen können in der Programmierung sehr nützlich sein, wenn Sie einen Codeblock eine bestimmte Anzahl von Malen wiederholen müssen. Auch wenn das Arbeiten mit `for`-Schleifen anfangs knifflig sein kann, werden Sie mit Übung schnell den Dreh rausbekommen.
