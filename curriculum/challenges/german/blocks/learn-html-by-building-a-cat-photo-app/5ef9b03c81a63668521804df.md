# --description--

Das `id`-Attribut wird verwendet, um bestimmte HTML-Elemente zu identifizieren. Der Wert jedes `id`-Attributs muss sich von allen anderen `id`-Werten auf der gesamten Seite unterscheiden.

Hier ist ein Beispiel für ein `input`-Element mit einem `id`-Attribut:

```html
<input id="email">
```

Fügen Sie dem Optionsschaltflächen-Element ein `id`-Attribut mit dem Wert `indoor` hinzu. Wenn Elemente mehrere Attribute haben, spielt die Reihenfolge der Attribute keine Rolle.
