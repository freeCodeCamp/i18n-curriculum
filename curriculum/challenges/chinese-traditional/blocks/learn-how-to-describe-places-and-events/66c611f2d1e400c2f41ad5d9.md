# --explanation--

在 `Present Simple` 時態中，否定形式是使用 `doesn't` 或 `isn't` 為第三人稱單數主詞（`he, she, it`）創建的。 

- 使用 `doesn't` 搭配動作動詞來表示某事沒有發生，例如 `It doesn't work.`

- 使用 `isn't` 搭配 `to be` 的表單形式來表示某事不成立，例如 `isn't set up.`

在這句話中，你需要同時使用 `doesn't` 和 `isn't` 來正確描述為什麼該特性沒有如預期運作。
