# --interactive--

Діалоги дозволяють відображати важливу інформацію або дії для користувачів. За допомогою вбудованого в HTML елемента `dialog` ви можете легко створювати ці діалоги (як модальні, так і немодальні) у своїх веб-застосунках.

Модальний діалог — це тип діалогу, який змушує користувача взаємодіяти з ним, перш ніж він зможе отримати доступ до решти застосунку або сторінки. Він ефективно блокує взаємодію з іншим вмістом, доки користувач не виконає дію, наприклад, не закриє діалог або не відправить форму.

Натомість немодальний діалог дозволяє користувачу продовжувати взаємодіяти з іншими частинами сторінки або застосунку навіть коли діалог відкритий. Він не заважає доступу до решти вмісту.

У цьому уроці ви навчитеся відкривати та закривати модальні вікна за допомогою методів `showModal()` та `close()`.

Коли потрібно переконатися, що користувач зосереджений на певній дії або повідомленні модального вікна, ви можете відкрити модальний діалог за допомогою методу `showModal()`. Це додасть затемнення до інших елементів на сторінці та вимкне їх. Це ідеально підходить для модальних вікон, які відображають форми, підтвердження та критичну інформацію, що вимагає дії користувача.

Ось HTML для модального діалогу:

:::interactive_editor

```html
<dialog id="my-modal">
  <p>This is a modal dialog.</p>
</dialog>
```

:::

Поки що ви нічого не бачите на сторінці, бо модальне вікно закрите при ініціальному відтворенні. Ви можете автоматично відкрити модальне вікно, використовуючи метод `showModal()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
dialog.showModal();
```

:::

Результат у браузері покаже модальне вікно з текстом `This is a modal dialog.`

Найкраще надати контроль користувачу. Для цього можна додати слухача події click на кнопку і використати метод `showModal()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.showModal();
});
```

:::

Якщо потрібно показати діалог, дозволяючи при цьому взаємодію з вмістом поза межами діалогу, тоді можна використати метод `show()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});
```

:::

Щоб закрити модальне вікно, можна додати кнопку до модального вікна всередині елемента `dialog` і використати метод `close()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
  <button id="close-modal-btn">Close Modal</button>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");
const closeButton = document.getElementById("close-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});

closeButton.addEventListener("click", () => {
  dialog.close();
});
```

:::
