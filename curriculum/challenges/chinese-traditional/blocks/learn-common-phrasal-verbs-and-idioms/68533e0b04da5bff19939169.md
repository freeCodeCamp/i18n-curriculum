# --explanation--

結構體 `help` + `to` + 動詞 用來表示某人支援（某功能）或協助做某事。說 `help` + 動詞 而不加 `to`（例如，`help make`）也是正確的。另一個例子： 

`He helped to fix the code.` 或 `He helped fix the code.` — 兩者皆表示他在解決問題時提供了支援。
