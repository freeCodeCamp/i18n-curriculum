# --explanation--

フレーズ`limit what other parts can see`の中で、`what`は名詞節を導入しており、それが`limit`の目的語として機能しています。

名詞節とは、文の中で名詞の役割を果たす節のことです。文の目的語や主語として使われます。通常、`that`、`what`、`whether`、`who`、`when`、`why`、`how`などの語で始まります。

名詞節は通常の文の語順（疑問文の語順ではない）を使います。例えば：

- `The team is checking why the system crashed during the update.` - `Why the system crashed during the update`は`checking`の目的語として機能する名詞節です。
   
- `What the function returns depends on the input data.` - `What the function returns`は`depends on`の主語として機能する名詞節です。
