### --question--

#### --text--

Cos'è un dizionario?

#### --distractors--

Una struttura dati che contiene solo stringhe e liste.

---

Una struttura dati che è una raccolta di coppie chiave-valore che accetta solo numeri grandi.

---

Una struttura dati che contiene solo una raccolta di tuple.

#### --answer--

Una struttura dati che è una raccolta di coppie chiave-valore.

### --question--

#### --text--

Quale delle seguenti è la sintassi corretta per un dizionario?

#### --distractors--

```py
dictionary = {
    <key1: value1>,
    <key2: value2>
}
```

---

```py
dictionary = {
    (key1: value1),
    (key2: value2)
}
```

---

```py
dictionary = {
    [key1: value1],
    [key2: value2]
}
```

#### --answer--

```py
dictionary = {
    key1: value1,
    key2: value2
}
```
