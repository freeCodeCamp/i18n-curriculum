## Властивість CSS overflow

- **Визначення**: Overflow означає, як елементи обробляють вміст, який перевищує або "виходить за межі" розміру контейнера. Overflow є двовимірним.
- **`overflow-x`**: Вісь x визначає горизонтальне переповнення.
- **`overflow-y`**: вісь y визначає вертикальне переповнення.
- **`overflow`**: Скорочена властивість для `overflow-x` і `overflow-y`. Якщо задано одне значення, обидва переповнення використовуватимуть його. Якщо задано два значення, `overflow-x` використовуватиме перше, а `overflow-y` — друге.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="overflow-box">
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
</div>
```

```css
.overflow-box {
  width: 140px;
  height: 70px;
  border: 2px solid black;
  overflow-x: auto;
  overflow-y: auto;
  white-space:nowrap;
}
```

## Властивість CSS transform

- **Визначення**: Ця властивість дозволяє застосовувати різні трансформації до елементів, такі як обертання, масштабування, нахил або переміщення (трансляція) у 2D або 3D просторі.
- **`translate()` функція**: Ця функція використовується для переміщення елемента з його поточного положення.
- **`scale()` функція**: Ця функція дозволяє змінити розмір елемента.
- **`rotate()` функція**: Ця функція дозволяє обертати елемент.
- **`skew()` функція**: Ця функція дозволяє нахиляти елемент.
- **Трансформації та доступність**: Якщо ви використовуєте transform для приховування або показу вмісту, переконайтеся, що вміст залишається доступним для читачів екрана та навігації за допомогою клавіатури. Прихований вміст має бути справді прихованим, наприклад, за допомогою `display: none` або `visibility: hidden`, а не просто візуально переміщеним поза межі екрану.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="transform-box translate">Translate</div>
  <div class="transform-box scale">Scale</div>
  <div class="transform-box rotate">Rotate</div>
  <div class="transform-box skew">Skew</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: column;
  align-items: center;
  gap: 16px;
  flex-wrap: wrap;
}

.transform-box {
  width: 100px;
  height: 100px;
  background: coral;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

/* translate() */
.translate {
  transform: translateX(20px);
}

/* scale() */
.scale {
  transform: scale(1.2);
}

/* rotate() */
.rotate {
  transform: rotate(15deg);
}

/* skew() */
.skew {
  transform: skew(10deg, 5deg);
}
```

## Модель блоку (box model)

- **Визначення**: У CSS-моделі блоку кожен елемент оточений блоком. Цей блок складається з чотирьох компонентів: області вмісту, `padding`, `border`, `margin`.
- **Область вмісту**: Область вмісту — це найвнутрішніша частина блоку. Це простір, який містить фактичний вміст елемента, наприклад текст або зображення.
- **`padding`**: Відступ — це область безпосередньо після області вмісту. Це простір між областю вмісту та кордоном елемента.
- **`border`**: Кордон — це зовнішній край або обриси елемента в CSS-моделі блоку. Це візуальна межа елемента.
- **`margin`**: Поле — це простір поза кордоном елемента. Воно визначає відстань між елементом і іншими елементами навколо нього.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="box-model">Content</div>
```

```css
.box-model {
  width: 120px;
  background: lightblue;
  padding: 16px;
  border: 4px solid teal;
  margin: 20px;
  text-align: center;
}
```

:::

## Злиття полів (margin collapse)

- **Визначення**: Ця поведінка виникає, коли вертикальні поля суміжних елементів перекриваються, утворюючи одне поле, рівне більшому з двох. Ця поведінка застосовується лише до вертикальних полів (верхнього та нижнього), а не до горизонтальних (лівого та правого).

## Значення властивостей `content-box` і `border-box`

- **Властивість `box-sizing`**: Ця властивість використовується для визначення, як обчислюються фінальні `width` і `height` для HTML-елемента.
- **Значення `content-box`**: У моделі `content-box` `width` і `height`, які ви задаєте для елемента, визначають розміри області вмісту, але не включають `padding`, `border` або `margin`.
- **Значення `border-box`**: У `border-box` `width` і `height` елемента включають область вмісту, `padding` і `border`, але не включають `margin`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="sizing-box content-box">Content-box</div>
<div class="sizing-box border-box">Border-box</div>
```

```css
.sizing-box {
  width: 150px;
  padding: 10px;
  border: 4px solid teal;
  margin-bottom: 10px;
  background: peachpuff;
  text-align: center;
}

.content-box {
  box-sizing: content-box;  
}

.border-box {
  box-sizing: border-box;
  background: lightgreen;
}
```

## CSS Reset

- **Визначення**: CSS reset — це таблиця стилів, яка видаляє всі або частину стандартного форматування, яке веббраузери застосовують до HTML-елементів. Сторонні варіанти CSS reset включають `sanitize.css` і `normalize.css`.

## Властивість CSS filter

- **Визначення**: Ця властивість може використовуватися для створення різних ефектів, таких як розмиття, зміна кольору та регулювання контрасту.
- **`blur()` функція**: Ця функція застосовує гаусове розмиття до елемента. Кількість визначається в пікселях і представляє радіус розмиття.
- **`brightness()` функція**: Ця функція регулює яскравість елемента. Значення 0% зробить елемент повністю чорним, а значення понад 100% збільшать яскравість.
- **`contrast()` функція**: Ця функція регулює контрастність елемента. Значення 0% зробить елемент повністю сірим, 100% — нормальним, а значення понад 100% збільшать контраст.
- **`grayscale()` функція**: Ця функція перетворює елемент у відтінки сірого. Кількість визначається у відсотках, де 100% — повністю сірий, а 0% залишає зображення без змін.
- **`sepia()` функція**: Ця функція застосовує сепію до елемента. Як і для відтінків сірого, використовується відсоткове значення.
- **`hue-rotate()` функція**: Ця функція застосовує обертання відтінку до елемента. Значення визначається в градусах і представляє обертання навколо кола кольорів.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <img class="filter blur" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Blur" />
  <img class="filter brightness" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Brightness" />
  <img class="filter contrast" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Contrast" />
  <img class="filter grayscale" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Grayscale" />
  <img class="filter sepia" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Sepia" />
  <img class="filter hue" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Hue Rotate" />
</div>
```

```css
.container {
  display: flex;
  gap: 12px;
  flex-wrap: wrap;
}

.filter {
  width: 120px;
}

/* blur() */
.blur {
  filter: blur(3px);
}

/* brightness() */
.brightness {
  filter: brightness(130%);
}

/* contrast() */
.contrast {
  filter: contrast(150%);
}

/* grayscale() */
.grayscale {
  filter: grayscale(100%);
}

/* sepia() */
.sepia {
  filter: sepia(100%);
}

/* hue-rotate() */
.hue {
  filter: hue-rotate(90deg);
}
```
