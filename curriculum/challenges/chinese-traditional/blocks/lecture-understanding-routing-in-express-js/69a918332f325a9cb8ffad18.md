# --description--

在 Express 中，**路由處理常式**（也稱為路由回呼函式或控制器函式）是定義當客戶端對特定路由發送請求時會發生什麼事情的函式。

路由處理常式會處理進來的請求，執行任何必要的邏輯，例如查詢資料庫或進行計算，然後將回應送回給客戶端。

讓我們用一個簡單的範例來說明（其實你一直都在使用這個範例）：

```javascript
app.get("/menu", (req, res) => {
  res.send("Welcome to the menu!");
});
```

`.get()` 方法是 Express 的路由方法之一。它監聽指定路由上的 **GET** 請求。在這個範例中，它監聽對 `/menu` 的請求。

函式 `(req, res) => { ... }` 是 **路由處理常式**。每當客戶端對 `/menu` 發送 GET 請求時，它就會執行。

在處理常式內：

* `req`（request）是一個物件，包含有關進來請求的資訊，例如路由參數、查詢字串、標頭和主體資料。
    
* `res`（response）是一個用來將回應送回給客戶端的物件。
    

在這個例子中，`res.send("Welcome to the menu!")` 將純文字回應送給客戶端。

當使用者在瀏覽器中造訪 `/menu`，或對該路由發送 GET 請求時，伺服器會執行這個處理常式並回應訊息 `Welcome to the menu!`。

路由處理常式是 Express 應用程式的核心。它們讓你定義網頁應用程式中不同路由如何回應請求。你也可以在這些處理常式中加入邏輯，例如處理表單資料、與資料庫互動或執行驗證。
