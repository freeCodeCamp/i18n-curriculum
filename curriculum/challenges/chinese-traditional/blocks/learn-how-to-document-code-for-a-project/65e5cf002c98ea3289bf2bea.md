# --description--

當以 `y` 結尾的名詞變成複數時，`y` 通常會變成 `ies`。這條規則在技術領域中特別有用，因為某些術語經常以 `y` 結尾。

# --questions--

## --text--

以 `y` 結尾的名詞如何變成複數？

## --answers--

將 `y` 改為 `ys`

### --feedback--

這不是一般的規則。通常，`y` 變成複數時會改成 `ies`，例如 `query` 變成 `queries`。

---

透過添加 `s`

### --feedback--

對大多數以 `y` 結尾的名詞來說，單純加 `s` 是不正確的。舉例來說，`directory` 變成 `directories`，而不是 `directorys`。

---

將 `y` 改為 `ies`

---

透過移除 `y`

### --feedback--

移除 `y` 並不是形成複數的正確方法。`y` 通常會變成 `ies`，像是 `binary` 變成 `binaries`。
