# --description--

你可以看到什么也没有发生。 尽管将 `top` 和 `left` 都设置为 `100px`，但 `.cat-head` 元素并未移动。 但 `relative` 定位的情况并非如此。

当你使用 `relative` 值时，元素仍会根据文档的正常流程进行定位，但 `top`、`left`、`bottom` 和 `right` 值会被激活。

将你的 `.cat-head` 的位置设置为 `relative` 而非 `static`，保持 `top` 和 `left` 属性不变。
