# --explanation--

`The additional contact options were highlighted...` は**受動態**の構造です。これは誰かが連絡オプションを素晴らしい改善点として指摘したことを意味しますが、誰が言ったかは明示されていません（ユーザーとそのフィードバックについて話していることはすでに明らかだからです）。焦点は行動と機能自体にあります。別の例：

`The security updates were highlighted in the weekly report.` – これは報告書がそれらを指摘したことを意味しますが、誰が書いたかには焦点を当てていません。
