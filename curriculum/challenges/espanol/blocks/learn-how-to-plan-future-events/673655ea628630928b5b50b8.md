# --explanation--

Un ataque de `Denial of Service (DoS)` es un ciberataque destinado a hacer que una red o servicio no esté disponible para los usuarios previstos, generalmente saturándolo con tráfico. Este tipo de ataque interrumpe el servicio normal, causando lentitud o fallos completos. Por ejemplo:

- `The system went down due to a denial of service attack.` - Esto significa que el sistema se volvió inaccesible porque fue saturado por tráfico malicioso.
  
- `Enhanced firewall settings can help prevent DoS attacks.` - Esto significa que configuraciones mejoradas del firewall pueden reducir el riesgo de tales ataques bloqueando el tráfico no deseado.
