# --description--

JavaScriptは、インタラクティブなウェブサイトを作成できる強力な言語です。

*注意*: このカリキュラムの残りのすべてのプロジェクトでは、HTMLとCSSの基本的な知識が必要です。HTMLとCSSが初めての場合は、[Responsive Web Design Certification](https://www.freecodecamp.org/learn/2022/responsive-web-design/)を確認してください。

まず、`DOCTYPE`、`html`、`head`、`body`を含む標準的なHTMLボイラープレートを作成し、次に`meta`用の`charset`タグを追加してください。`title`要素を追加し、テキストに`RPG - Dragon Repeller`を使用してください。スタイルシート用に`link`タグを含めて、`styles.css`ファイルをリンクしてください。

最後に、`div`内に`id`が`game`に設定された`body`要素を作成してください。

# --hints--

コードには`DOCTYPE`参照が含まれている必要があります。

```js
assert(code.match(/<!DOCTYPE/gi));
```

`DOCTYPE`参照の後にスペースを入れてください。

```js
assert(code.match(/<!DOCTYPE\s+/gi));
```

ドキュメントタイプは`html`に設定してください。

```js
assert(code.match(/<!DOCTYPE\s+html/gi));
```

`DOCTYPE`宣言はタイプの後に`>`で閉じてください。

```js
assert(code.match(/<!DOCTYPE\s+html\s*>/gi));
```

`html`要素には開始タグが必要です。`lang`属性を忘れないでください。

```js
assert(code.match(/<html\s+lang\s*=\s*('|")en\1\s*>/gi));
```

`html`要素には終了タグが必要です。

```js
assert(code.match(/<\/html\s*>/));
```

`DOCTYPE`宣言はHTMLの先頭に置いてください。

```js
assert(__helpers.removeHtmlComments(code).match(/^\s*<!DOCTYPE\s+html\s*>/i));
```

開始`head`タグを含めてください。

```js
assert(code.match(/<head\s*>/i));
```

終了`head`タグを含めてください。

```js
assert(code.match(/<\/head\s*>/i));
```

開始`body`タグを含めてください。

```js
assert(code.match(/<body\s*>/i));
```

終了`body`タグを含めてください。

```js
assert(code.match(/<\/body\s*>/i));
```

`head`と`body`要素は兄弟要素である必要があります。

```js
assert.match(code, /<head\s*>.*<\/head\s*>.*<body\s*>.*<\/body\s*>/s)
```

`head`要素は`html`要素の中にある必要があります。

```js
assert.match(code, /<html[^>]*>.*<head\s*>.*<\/head\s*>.*<\/html\s*>/s);
```

`body`要素は`html`要素の中にある必要があります。

```js
assert.match(code, /<html[^>]*>.*<body\s*>.*<\/body\s*>.*<\/html\s*>/s);
```

コードには`meta`要素が必要です。

```js
const meta = document.querySelector('meta');
assert.exists(meta);
```

`meta`要素には値が`charset`の`UTF-8`属性が必要です。

```js
assert.match(code, /<meta[\s\S]+?charset=('|"|`)UTF-8\1/i)
```

コードには`title`要素が必要です。

```js
const title = document.querySelector('title');
assert.exists(title);
```

終了`title`タグを含めてください。

```js
assert(code.match(/<\/title\s*>/i));
```

コードには`link`要素が必要です。

```js
const link = document.querySelector('head > link');
assert.exists(link);
```

`div`要素を含めてください。

```js
const div = document.querySelector('div');
assert.exists(div);
```

`div`要素には値が`id`の`game`属性が必要です。

```js
const div = document.querySelector('div');
assert.equal(div?.id, 'game');
```

`div`要素は`body`要素の中にある必要があります。

```js
const div = document.querySelector('div');
assert.equal(div?.parentElement?.localName, 'body');
```
