# --description--
最简单的入门方法是在一个 HTML 文件中创建包含 JavaScript 代码的内容。在你的计算机上的某个位置，将基本的 HTML 骨架输入到一个文件中：

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>
  <script>
    // Your JavaScript goes here!
    console.log("Hello, World!")
  </script>
</body>
</html>
```

将此文件储存并在网页浏览器中打开（你可以使用 <a href="https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer" target="_blank" >Visual Studio Code 上的 "Live Server"</a> 来完成此操作！），然后通过右键点击空白网页并选择 "Inspect" 或 "Inspect Element" 打开浏览器的控制台。在“开发者工具”窗格中查找并选择“Console”标签（页），你应该能看到我们 `console.log` 语句的输出。

> `console.log()` 是在浏览器的开发者控制台打印内容的命令。你可以使用它将以下任何文章和练习中的结果打印到控制台。我们鼓励你跟随本课及未来课程中的所有示例进行编码。

另一种在网页中包含 JavaScript 的方式是通过外部脚本。这与将外部 CSS 文档链接到你的网站非常相似。

```html
  <script src="javascript.js"></script>
```

JavaScript 文件的扩展名是 `.js`，类似于样式表的 `.css`。外部 JavaScript 文件用于更复杂的脚本。

# --questions--

## --text--

哪个语句准确描述了如何在超文本标记语言文档中包含 JavaScript？

## --answers--

JavaScript 只能在超文本标记语言体内的 `<script>` 标签内部包含。

---

JavaScript 代码必须写在 HTML 文档的 `<head>` 部分，才能正确执行。

---

JavaScript 可以通过 `<script>` 标签直接包含在超文本标记语言文件中，或者通过 `<script src="javascript.js">` 标签外部链接。 

---

外部 JavaScript 文件需要在 `<script>` 标签中添加特殊属性，才能被网页浏览器识别。
