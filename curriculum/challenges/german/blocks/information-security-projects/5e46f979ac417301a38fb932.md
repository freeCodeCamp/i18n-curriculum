# --instructions--

Erstellen Sie einen Portscanner mit Python.

Erstellen Sie in der `port_scanner.py` eine Funktion namens `get_open_ports`, die ein `target`-Argument und ein `port_range`-Argument entgegennimmt. `target` kann eine URL oder IP-Adresse sein. `port_range` ist eine Liste von zwei Zahlen, die die erste und letzte Zahl des zu überprüfenden Portbereichs angeben.

Hier sind Beispiele, wie die Funktion aufgerufen werden kann:

```py
get_open_ports("209.216.230.240", [440, 445])
get_open_ports("www.stackoverflow.com", [79, 82])
```

Die Funktion soll eine Liste der offenen Ports im angegebenen Bereich zurückgeben.

Die `get_open_ports`-Funktion soll außerdem ein optionales drittes Argument vom Typ `True` annehmen, um den „Verbose“-Modus anzugeben. Wenn dieser auf true gesetzt ist, soll die Funktion einen beschreibenden String anstelle einer Liste von Ports zurückgeben.

Hier ist das Format des Strings, der im Verbose-Modus zurückgegeben werden soll (Text innerhalb von `{}` zeigt die anzuzeigenden Informationen):

```bash
Open ports for {URL} ({IP address})
PORT     SERVICE
{port}   {service name}
{port}   {service name}
```

Sie können das Wörterbuch in `common_ports.py` verwenden, um den korrekten Dienstnamen für jeden Port zu erhalten.

Wenn die Funktion zum Beispiel so aufgerufen wird:

```py
port_scanner.get_open_ports("scanme.nmap.org", [20, 80], True)
```

soll sie Folgendes zurückgeben:

```bash
Open ports for scanme.nmap.org (45.33.32.156)
PORT     SERVICE
22       ssh
80       http
```

Achten Sie darauf, die richtigen Abstände und Zeilenumbrüche einzuhalten.

Wenn die an die `get_open_ports`-Funktion übergebene URL ungültig ist, soll die Funktion den String „Error: Invalid hostname“ zurückgeben.

Wenn die an die `get_open_ports`-Funktion übergebene IP-Adresse ungültig ist, soll die Funktion den String „Error:  Invalid IP address“ zurückgeben.
