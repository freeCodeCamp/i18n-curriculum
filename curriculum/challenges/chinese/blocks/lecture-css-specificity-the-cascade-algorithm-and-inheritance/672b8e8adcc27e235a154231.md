# --interactive--

类型选择器，也称为元素选择器，根据它们的标签名称定位元素。

这些选择器是 CSS 中的基础，允许你为特定的超文本标记语言元素的所有实例应用样式。

类型选择器使用简单，直接写成你想要设置样式的元素的标签名称。

下面是一个类型选择器示例，针对页面上的所有段落元素：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  color: blue;
}
```

:::

在此示例中，所有 `p` 元素的文本颜色将设置为 `blue`。

类型选择器的特异性相对于其他选择器较低。类型选择器的特异性值为 `(0, 0, 0, 1)`。

这意味着类型选择器会被类选择器、ID 选择器和内联样式覆盖，但除非存在那些更高特异性的规则，否则仍然可以应用样式。

让我们来看一个示例，其中类选择器将覆写类型选择器的样式。

这是一个包含两个段落元素的示例：

:::interactive_editor

```html
<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

:::

两个段落元素都有一个名为 `para` 的类。

在 CSS 文件中，类型选择器定位段落，类选择器定位具有 `para` 类的元素。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

```css
p {
  color: blue;
}

.para {
  color: red;
}
```

:::

页面上所有带有 `para` 类的段落的文本颜色将被设置为 `red`，而不是 `blue`，因为类选择器的特异性高于类型选择器。
