# --description--

Una `convention` è come una regola o un modo in cui la maggior parte delle persone concorda di fare le cose, specialmente nel lavoro. Per esempio, nel coding, una regola comune è scrivere codice in un certo stile.

`To follow` è come attenersi o fare ciò che queste regole dicono. Quindi, seguire le regole di coding significa scrivere codice nel modo che la maggior parte delle persone concorda sia il migliore.
