# --explanation--

`To integrate` bedeutet, verschiedene Teile so zusammenzuführen, dass sie als ein System funktionieren. Zum Beispiel:

`The new app integrates chat and calendar features in one place.` – Das bedeutet, die neue App fügt Chat- und Kalenderfunktionen zusammen.

Etwas `intuitive` ist leicht zu benutzen oder zu verstehen, ohne Erklärung. Zum Beispiel:

`The new design is more intuitive, so users can navigate it easily.` – Das bedeutet, Nutzer können verstehen, wie man es benutzt, nur durch Anschauen.
