# --description--

Il metodo `fetch()` restituisce un `Promise`, che è un oggetto segnaposto che verrà risolto se la tua richiesta ha successo, oppure rifiutato se la tua richiesta fallisce.

Se il `Promise` viene risolto, restituisce un oggetto `Response`, e puoi usare il metodo `.then()` per consultare il `Response`.

Ecco come puoi concatenare `.then()` al metodo `fetch()`:

```js
fetch("sample-url-goes-here")
  .then((res) => res)
```

Concatena il metodo `.then()` alla tua chiamata `fetch`. All’interno del metodo `.then()`, aggiungi una funzione di callback con `res` come parametro, quindi registra il `res` nella console così puoi vedere l’oggetto `Response`. Apri la console del browser ed espandi l’oggetto `Response` per vedere cosa contiene.

Ancora, non terminare il codice con un punto e virgola.
