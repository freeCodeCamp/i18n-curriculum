# --description--

`Store information` significa mantener o guardar datos en una ubicación específica, generalmente en una variable. Las variables son como contenedores en el código que contienen información, como números, texto u otros tipos de datos. Puedes pensar en ellas como espacios de almacenamiento nombrados donde puedes acceder a la `información almacenada` más tarde en el código.

Por ejemplo, `int age = 30;` Aquí, la variable `age` almacena el número 30.

La palabra `like` se usa como una preposición para dar ejemplos o para comparar cosas similares.

Por ejemplo, si dices, `I love fruits like apples and oranges.` Estás usando `like` para dar ejemplos del tipo de frutas que amas, sugiriendo que podría haber más frutas en esa lista.
