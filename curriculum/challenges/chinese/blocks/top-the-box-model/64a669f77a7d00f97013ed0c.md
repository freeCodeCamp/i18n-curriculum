# --description--

既然你已经了解了 HTML 和 CSS 的基本语法，我们将进入更深入的内容。你需要掌握的最重要的 CSS 技能是定位和版型。更改字体和颜色是关键技能，但能够将内容准确地放置在网页上的指定位置则更为关键。毕竟，你能找到多少网页是所有元素完全堆叠在一起的呢？

一旦你理解了几个关键的概念，学习如何在网页上定位元素并不难。不幸的是，许多学习者急于学习超文本标记语言和 CSS，以便尽快进入 JavaScript，结果错过了这些基础概念。这会导致挫败感、痛苦（以及有趣的动图），因为如果你不能将元素放置在页面上所需的位置，所有的 JavaScript 技能都是毫无意义的。考虑到这一点，我们开始吧。
