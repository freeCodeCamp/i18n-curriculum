# --description--

動詞の`be`形に続く`-ing`形は、今まさに起こっている動作について話すときに使います。

`have trouble with`というフレーズは、特定の問題や対象に対して困難を感じていることを表現するときに使います。例えば、`She is having trouble with her computer`は彼女がコンピューターの使用に困っていることを意味します。
