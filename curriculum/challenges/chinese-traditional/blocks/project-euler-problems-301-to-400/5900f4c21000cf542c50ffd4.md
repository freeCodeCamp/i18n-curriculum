# --description--

對於固定整數 $a$、$b$、$c$，定義瘋狂函式（函式、函數）`F(n)` 如下：

$$\begin{align}
  & F(n) = n - c \\;\text{ 對所有 } n > b \\\\
  & F(n) = F(a + F(a + F(a + F(a + n)))) \\;\text{ 對所有 } n ≤ b.
\end{align}$$

此外，定義 $S(a, b, c) = \displaystyle\sum_{n = 0}^b F(n)$。

例如，如果 $a = 50$、$b = 2000$ 和 $c = 40$，那麼 $F(0) = 3240$ 和 $F(2000) = 2040$。此外，$S(50, 2000, 40) = 5\\,204\\,240$。

找出 $S({21}^7, 7^{21}, {12}^7)$ 的最後 9 位數字。
