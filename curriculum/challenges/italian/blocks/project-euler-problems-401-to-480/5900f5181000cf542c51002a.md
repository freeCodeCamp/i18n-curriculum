# --description--

Una sequenza di numeri interi $S = \\{s_i\\}$ si chiama sequenza n se ha $n$ elementi e ogni elemento $s_i$ soddisfa $1 ≤ s_i ≤ n$. Quindi ci sono in totale $n^n$ sequenze n distinte.

Per esempio, la sequenza $S = \\{1, 5, 5, 10, 7, 7, 7, 2, 3, 7\\}$ è una sequenza 10.

Per ogni sequenza $S$, sia $L(S)$ la lunghezza della più lunga sottosequenza contigua di $S$ con lo stesso valore. Per esempio, per la sequenza $S$ data sopra, $L(S) = 3$, a causa dei tre 7 consecutivi.

Sia $f(n) = \sum L(S)$ per tutte le sequenze n $S$.

Per esempio, $f(3) = 45$, $f(7) = 1\\,403\\,689$ e $f(11) = 481\\,496\\,895\\,121$.

Trova $f(7\\,500\\,000)\bmod 1\\,000\\,000\\,009$.
