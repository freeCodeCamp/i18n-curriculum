# --explanation--

Kifupisho `We'll` kinasimamia `We will`, kinachoashiria hatua ya baadaye au mpango. Kwa mfano:

- `I'll call you tomorrow.` - Hii ina maana hatua ya kuita itatokea baadaye.

- `We'll review the report later.` - Hii inaonyesha kuwa ukaguzi umepangwa lakini bado haujafanyika.
