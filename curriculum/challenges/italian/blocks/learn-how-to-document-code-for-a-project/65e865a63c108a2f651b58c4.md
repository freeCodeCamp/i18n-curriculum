# --description--

In questa attività imparerai la differenza tra `specific` e `specify`. `Specific` è un aggettivo che descrive qualcosa di chiaramente definito o dettagliato, mentre `specify` è un verbo che significa esporre qualcosa in dettaglio.

Per esempio, un'istruzione `specific` potrebbe essere `use port 8080`, mentre `specify` può significare dichiarare quel dettaglio: `specify the port number.`

`Setup` si riferisce all'organizzazione o alla preparazione di qualcosa, spesso usato nel contesto della preparazione di software o attrezzature, come `setup instructions for the software.`
