# --description--

Kabla ya kuendelea, chukua muda kidogo ukague jinsi vitendakazi vinavyofanya kazi.

Tangaza kitendakazi kinachoitwa `addTwoNumbers`. Kitendakazi hiki kinapaswa kuchukua hoja mbili na kurudisha jumla ya hoja hizo mbili.

Kitendakazi chako hakipaswi kutumia thamani zilizowekwa moja kwa moja. Mfano wa kitendakazi kilicho na thamani iliyowekwa moja kwa moja kinaweza kuwa:

```js
function sayName(firstName, lastName) {
  return "John Doe";
}

sayName("Camper", "Cat");
```

Kitendakazi hiki kitarudisha `"John Doe"` bila kujali hoja zitakazotolewa kwa vigezo `firstName` na `lastName`, kwa hivyo `"John Doe"` inachukuliwa kuwa thamani iliyowekwa moja kwa moja.

Tangaza kigezo cha `sum` na weka thamani yake kwa wito wa kitendakazi chako cha `addTwoNumbers` ukiwa na `5` na `10` kama hoja. Andika kigezo cha `sum` kwenye konsoli.
