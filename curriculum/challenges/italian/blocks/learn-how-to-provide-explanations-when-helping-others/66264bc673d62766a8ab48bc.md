# --description--

La parola `which` si usa per chiedere informazioni specifiche all’interno di un insieme di opzioni conosciute. Per esempio, quando qualcuno chiede `Which book did you read?` vuole sapere quale libro specifico tra diversi possibili.

In un contesto tecnico, chiedere `Which software are you using?` significa chiedere il nome specifico di un software tra opzioni conosciute.
