# --description--

型セレクター（または要素セレクター）は、指定された要素タイプのすべての要素を選択し、構文は単に要素の名前です：

```html
<!-- index.html -->

<div>Hello, World!</div>
<div>Hello again!</div>
<p>Hi...</p>
<div>Okay, bye.</div>
```

```css
/* styles.css */

div {
  color: white;
}
```

ここでは、3つの`<div>`要素すべてが選択され、`<p>`要素は選択されません。

# --questions--    

## --text--

上記のCSSコードを最もよく説明しているのはどれですか？

## --answers--

このコードは、HTMLファイル内のすべての要素に`white`色を適用します。

---

このコードは、HTMLファイル内のすべての`white`要素に`div`色を適用します。

---

このコードは、HTMLファイル内のすべての`white`要素に`p`色を適用します。
