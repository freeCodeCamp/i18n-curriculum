# --interactive--

## دوال JavaScript

- الدوال هي كتل قابلة لإعادة الاستخدام من الشفرة تؤدي مهمة محددة.
- يمكن تعريف الدوال باستخدام الكلمة المفتاحية `function` متبوعة باسم، وقائمة معلمات، وكتلة شفرة تؤدي المهمة.

:::interactive_editor

```js
function addNumbers(x, y, z) {
  return x + y + z;
}

console.log(addNumbers(5, 3, 8)); // Output: 16
```

:::

- المعلمات هي القيم التي تُمرر إلى الدالة عند استدعائها.
- استدعاء الدالة هو عملية تنفيذ الدالة في البرنامج بتحديد اسم الدالة متبوعًا بأقواس عادية، مع إمكانية تضمين المعلمات داخل الأقواس.
- عند انتهاء تنفيذ الدالة، سترجع دائمًا قيمة.
- افتراضيًا، تكون قيمة الإرجاع للدالة هي `undefined`.
- تُستخدم الكلمة المفتاحية `return` لتحديد القيمة التي سترجع من الدالة وتنهي تنفيذها.
- تسمح المعلمات الافتراضية للدوال بأن يكون لها قيم محددة مسبقًا تُستخدم إذا لم تُمرر معلمة عند استدعاء الدالة. هذا يجعل الدوال أكثر مرونة ويمنع الأخطاء في الحالات التي قد تُحذف فيها بعض المعلمات.

:::interactive_editor

```js
const calculateTotal = (amount, taxRate = 0.05) => {
  return amount + (amount * taxRate);
};

console.log(calculateTotal(100)); // Output: 105
```

:::

- الدوال المجهولة هي دوال بدون اسم يمكن تعيينها إلى متغيرات. بتعيينها إلى متغيرات، يمكنك إعادة استخدامها في أي مكان يمكن الوصول فيه إلى المتغير.

:::interactive_editor

```js
const multiplyNumbers = function(firstNumber, secondNumber) {
  return firstNumber * secondNumber;
};

console.log(multiplyNumbers(4, 5)); // Output: 20
```

:::

## الدوال السهمية

- الدوال السهمية هي طريقة أكثر اختصارًا لكتابة الدوال في JavaScript.

:::interactive_editor

```js
const calculateArea = (length, width) => {
  const area = length * width;
  return `The area of the rectangle is ${area} square units.`;
};

console.log(calculateArea(5, 10)); // Output: "The area of the rectangle is 50 square units."
```

:::

- عند تعريف دالة سهمية، لا تحتاج إلى الكلمة المفتاحية `function`.
- إذا كنت تستخدم معلمة واحدة فقط، يمكنك حذف الأقواس العادية حول قائمة المعلمات.

:::interactive_editor

```js
const cube = x => {
  return x * x * x;
};

console.log(cube(3)); // Output: 27
```

:::

- إذا كان جسم الدالة يتكون من تعبير واحد فقط، يمكنك حذف الأقواس المتعرجة والكلمة المفتاحية `return`.

:::interactive_editor

```js
const square = number => number * number;

console.log(square(5)); // Output: 25
```

:::

## النطاق في البرمجة

- **النطاق العام**: هذا هو أوسع نطاق في JavaScript. المتغيرات المُعلنة في النطاق العام يمكن الوصول إليها من أي مكان في الشفرة وتسمى متغيرات عامة.
- **النطاق المحلي**: يشير إلى المتغيرات المُعلنة داخل دالة. هذه المتغيرات يمكن الوصول إليها فقط داخل الدالة التي أُعلنت فيها وتسمى متغيرات محلية.
- **نطاق الكتلة**: الكتلة هي مجموعة من العبارات محاطة بأقواس متعرجة `{}` مثل عبارات `if` أو الحلقات.
- يوفر نطاق الكتلة باستخدام `let` و `const` تحكمًا أدق في إمكانية الوصول إلى المتغيرات، مما يساعد على منع الأخطاء وجعل الشفرة أكثر توقعًا.
