# --interactive--

JavaScriptでの日付と時間の扱いは、いつも簡単だったわけではありません。実際のコードベースでは、多くの問題を解決してくれるライブラリを使うことが多いでしょう。

しかし、JavaScriptの組み込み`Date`オブジェクトを使わなければならない場合もあります。

このレッスンでは、`Date`オブジェクトの基本と、JavaScriptで日付や時間を扱う際によく使うメソッドについて説明します。

`Date`オブジェクトはJavaScriptの組み込みオブジェクトで、日付と時間を扱うことができます。新しい`Date`オブジェクトを作成する基本的な構文は以下の通りです。

```js
const now = new Date();
```

`new`キーワードは`Date`オブジェクトの新しいインスタンスを作成するために使われ、`Date`オブジェクトが変数`now`に代入されます。もし`now`の値をコンソールに出力すると、コードを実行しているコンピューターのシステムクロックに基づいた現在の日付と時間が表示されます。

`now`の値をコンソールに出力した場合に表示される例は以下の通りです。

:::interactive_editor

```js
const now = new Date();
console.log(now);
```

:::

時間は24時間制で表されており、`14:30`は`2:30 PM`です。`GMT-0700`はタイムゾーンのオフセットで、`Pacific Daylight Time`はタイムゾーン名です。

また、年、月、日、時、分、秒、ミリ秒の値を引数として渡すことで、特定の日付と時間を持つ`Date`オブジェクトを作成することもできます。

特定の日付と時間で新しい`Date`オブジェクトを作成する例は以下の通りです。

:::interactive_editor

```js
const specificDate = new Date("July 4, 1776 12:00:00");

console.log(specificDate);
```

:::

これは、現在の日付と時間ではなく特定の日付と時間を扱う必要がある場合に便利です。入力は常に文字列である必要があり、`Date`オブジェクトが認識できる形式でなければなりません。

現在の日付と時間を取得するには、`Date.now()`メソッドを使います。これは`January 1, 1970, 00:00:00 UTC`からのミリ秒数を返します。これをUnixエポック時間と呼びます。

Unixエポック時間は、整数で表されるためコンピューターシステムで日付と時間を表す一般的な方法です。UTCは世界が時計と時間を調整するための主要な時間基準である協定世界時（Universal Time Coordinated）を表します。

現在の日付に基づいて月の日を取得したい場合は、`getDate`メソッドを使います。`getDate`メソッドを使った例は以下の通りです。

:::interactive_editor

```js
const now = new Date();
const date = now.getDate();
console.log(date); 
```

:::

この例では、`Date`オブジェクトを使って新しい日付インスタンスを作成し、変数`now`に代入しています。次に`getDate`オブジェクトの`now`メソッドを呼び出して月の日を取得し、変数`date`に代入しています。最後に`date`の値をコンソールに出力すると、現在の月の日が表示されます。

`getDate`は月の日に応じて`1`から`31`までの整数値を返します。日付が無効な場合は`NaN`（Not a Number）を返します。

月を取得するには、次のように`getMonth`メソッドを使います。

:::interactive_editor

```js
const now = new Date();
const month = now.getMonth();
console.log(month);
```

:::

月はゼロベースなので、`January`は`0`、`February`は`1`、というようになります。月が無効な場合は`NaN`を返します。

西暦年を取得したい場合は、次のように`getFullYear`メソッドを使います。

:::interactive_editor

```js
const now = new Date();
const year = now.getFullYear();
console.log(year); 
```

:::

年が無効な場合は`NaN`を返します。

`Date`オブジェクトには他にも`getHours`、`getMinutes`、`getSeconds`など多くのメソッドがあります。Mozillaのドキュメントや他のリソースでさらに調べてみることをお勧めします。
