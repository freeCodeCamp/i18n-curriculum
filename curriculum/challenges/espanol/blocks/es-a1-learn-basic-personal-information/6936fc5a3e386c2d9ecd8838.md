# --description--

En la pregunta `¿Quieres que te lo deletree?`, varios elementos trabajan juntos para expresar una oferta.

`Quieres que` se usa para preguntar si alguien quiere algo.

`Te` se refiere a la persona a la que se le habla. En este caso, Camila.

`Lo` se refiere a la información, aquí el correo electrónico de Mateo.

`Deletree` viene de `deletrear`, que significa deletrear una palabra letra por letra.

Juntos, la pregunta se usa para ofrecer deletrear el correo electrónico a Camila.


# --instructions--

Escucha el audio y decide si la siguiente afirmación es verdadera o falsa.
