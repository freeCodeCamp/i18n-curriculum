# --description--

Перша важлива концепція, яку потрібно зрозуміти, щоб успішно працювати з CSS, — це модель блоку. Вона не складна, але пропускати її зараз буде дуже розчаровуюче пізніше.

Кожна річ на вебсторінці — це прямокутний блок. Ці блоки можуть містити інші блоки і розташовуватися поруч один з одним. Ви можете приблизно уявити, як це працює, якщо додати кордон до кожного елемента на сторінці ось так:

```css
* {
  border: 1px solid red;
}
```

# --assignment--
