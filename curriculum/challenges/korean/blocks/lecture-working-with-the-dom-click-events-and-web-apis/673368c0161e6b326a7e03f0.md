# --interactive--

`innerHTML`와 `createElement()`로 노드를 만드는 방법을 살펴보겠습니다.

`innerHTML`부터 시작하겠습니다.

`innerHTML`는 `Element` 객체의 속성으로, HTML 마크업을 설정하는 데 사용할 수 있습니다. `innerHTML`를 사용하면 문자열로 기존 요소의 HTML 구조를 설정하여 필요한 모든 노드를 생성할 수 있습니다.

예를 들어, 빈 `div`가 있고 `innerHTML` 속성을 사용해 그 안에 몇몇 요소를 추가해 보겠습니다:

```html
<div id="container">
  <!-- Add new elements here -->
</div>
```

먼저, ID로 요소를 선택해야 합니다:

```js
const container = document.getElementById("container");
```

그다음, 컨테이너의 `innerHTML` 속성을 문자열로 설정합니다. 이 문자열은 생성하려는 노드와 구조를 나타내는 데 필요한 모든 마크업을 포함해야 합니다. 이것을 문자열 안에 HTML을 작성하는 것으로 생각할 수 있습니다.

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
container.innerHTML = "<ul><li>Cheese</li><li>Tomato</li></ul>";
```

:::

이 요소의 `innerHTML`를 피자 재료(치즈와 토마토)의 순서 없는 목록으로 설정할 것입니다.

마크업을 이해하기 쉽게 여러 줄로 문자열을 작성해도 됩니다. 중요한 것은 마크업이 생성하려는 HTML 구조를 정확히 나타내는 것입니다.

이 코드를 실행한 후 마크업을 검사하면 다음과 같은 HTML 구조가 보일 것입니다.

```html
<div id="container">
  <ul>
    <li>Cheese</li>
    <li>Tomato</li>
  </ul>
</div>
```

새 노드들이 문자열이 파싱된 후 동적으로 DOM에 생성되어 추가되었습니다.

`innerHTML`는 특정 상황에서 매우 유용할 수 있습니다. 하지만 알아야 할 보안 위험도 있습니다. 문자열을 완전히 제어할 수 없다면 사용하지 않는 것이 좋습니다.

예를 들어, 문자열이 사용자가 입력하는 경우, 사용자가 악성 콘텐츠를 웹사이트에 삽입할 수 있으므로 `innerHTML`를 사용하면 안 됩니다. 이런 이유로 일반 텍스트를 삽입할 때는 보통 `textContent`를 사용하는 것이 권장됩니다.

새 노드를 만드는 또 다른 방법은 `createElement()` 메서드를 사용하는 것입니다. 이 메서드를 사용하면 태그 이름을 지정해 새 요소를 만들 수 있습니다.

예를 들어, 이미지 요소를 만들고 싶다면 이 메서드를 호출할 때 `img` 태그를 문자열로 전달합니다:

```js
document.createElement("img");
```

그리고 이 새 객체를 변수에 할당할 수 있습니다:

```js
const img = document.createElement("img");
```

`createElement()` 메서드는 문서가 `HTMLElement`인 경우 새 `HTMLDocument` 객체를 반환합니다. 그렇지 않으면 `Element` 객체를 반환합니다.

이 새 요소를 준비한 후에는 `appendChild()` 메서드를 사용해 다른 기존 요소의 자식으로 DOM에 추가하거나, 다른 메서드를 사용해 특정 위치에 삽입할 수 있습니다. 필요에 가장 적합한 방법을 선택하세요.

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
const img = document.createElement("img");
img.src = "https://cdn.freecodecamp.org/curriculum/cat-photo-app/lasagna.jpg";
img.alt = "A slice of lasagna on a plate.";
container.appendChild(img);
```

:::

`innerHTML`와 `createElement()`로 노드를 만들면 웹사이트의 구조와 콘텐츠를 동적으로 조작할 수 있습니다. 이 기법들을 결합해 인터랙티브한 웹 애플리케이션을 구축할 수 있습니다.
