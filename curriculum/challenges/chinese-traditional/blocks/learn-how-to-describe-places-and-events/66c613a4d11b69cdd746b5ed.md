# --explanation--

當聆聽 Sarah 在對話窗中的零件時，請注意字 `if`。這個字經常介紹問題的原因，說明為了讓其他事情發生，需要發生什麼。 

在找到 `if` 之後，識別句子中描述當該條件不成立時發生什麼的零件（效果）。在 Sarah 的句子中：

- **效果：** `The auto-completion doesn't work`。

- **由** `if` **引起的原因：** `If the project isn't set up correctly`。

理解這種關係有助於你清楚說明問題和解法。
