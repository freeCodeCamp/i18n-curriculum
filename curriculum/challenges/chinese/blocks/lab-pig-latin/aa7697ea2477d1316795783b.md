# --description--

Pig Latin 是一种通过遵循特定转换规则来改变英语单词的方式。

**目标：**实现以下用户需求并通过所有测试以完成实验。

**用户故事：**

1. 你应该创建一个接受一个字串作为参数的 `translatePigLatin` 函数。
1. 如果字串参数以辅音字母开头，你的函数应取第一个辅音字母或辅音字母群，将其移到单词末尾，添加 `ay`，并返回结果。
1. 如果字串参数以元音开头，你的函数应在末尾添加 `way` 并返回结果。
1. 你的函数应处理第一个元音出现在单词中间的字串参数，并返回适当转换的字串。
1. 如果字串参数没有元音，你的函数应在末尾添加 `ay` 并返回结果。

注意：在本实验的上下文中，元音是 `a`、`e`、`i`、`o` 和 `u`。字母 `y` 不被视为元音。

```js
function translatePigLatin(str) {
  if (isVowel(str.charAt(0))) return str + "way";
  let front = [];
  str = str.split('');
  while (str.length && !isVowel(str[0])) {
    front.push(str.shift());
  }
  return [].concat(str, front).join('') + 'ay';
}

function isVowel(c) {
  return ['a', 'e', 'i', 'o', 'u'].indexOf(c.toLowerCase()) !== -1;
}
```
