# --interactive--

Реверс рядка — це загальне завдання програмування, яке можна виконати в JavaScript, використовуючи комбінацію методів рядків і масивів. Процес складається з трьох основних кроків:

- Розбити рядок на масив символів.
- Реверсувати масив.
- З’єднати символи назад у рядок.

Давайте розглянемо кожен із цих кроків, використовуючи методи `split()`, `reverse()` і `join()`.

Перший крок у реверсі рядка — перетворити його на масив окремих символів. Це можна зробити за допомогою методу `split()`. Метод `split()` розбиває рядок на масив підрядків і визначає, де має відбутися розбиття, на основі заданого роздільника. Якщо роздільник не вказано, метод повертає масив, що містить оригінальний рядок як єдиний елемент. Прикладами загальних роздільників є:

- Порожній рядок (`""`), який розбиває рядок на окремі символи.

- Один пробіл (`" "`), який розбиває рядок у місцях пробілів.

- Дефіс (`"-"`), який розбиває рядок на кожному дефісі.

Ось приклад використання методу split для створення масиву символів:

:::interactive_editor

```js
let str = "hello";
let charArray = str.split("");
console.log(charArray); // ["h", "e", "l", "l", "o"]
```

:::

У цьому прикладі ми використовуємо `split("")` (передаючи порожній рядок) для перетворення рядка `hello` у масив його окремих символів. Коли ми маємо масив символів, можемо використати метод `reverse()` для реверсу порядку елементів у масиві.

Метод `reverse()` — це метод масиву, який реверсує масив на місці. Це означає, що він змінює оригінальний масив, а не створює новий. Ось як ми можемо його використати:

:::interactive_editor

```js
let charArray = ["h", "e", "l", "l", "o"];
charArray.reverse();
console.log(charArray); // ["o", "l", "l", "e", "h"]
```

:::

У цьому прикладі `reverse()` змінює порядок елементів у `charArray`, реверсуючи його з `["h", "e", "l", "l", "o"]` до `["o", "l", "l", "e", "h"]`.

Останній крок — перетворити реверсований масив символів назад у рядок. Це можна зробити за допомогою методу `join()`. Метод `join()` створює і повертає новий рядок, об’єднуючи всі елементи масиву, розділені заданим рядком-роздільником. Якщо ви хочете об’єднати символи без роздільника, можна використати порожній рядок як аргумент. Ось приклад:

:::interactive_editor

```js
let reversedArray = ["o", "l", "l", "e", "h"];
let reversedString = reversedArray.join("");
console.log(reversedString); // "olleh"
```

:::

У цьому прикладі `join("")` (передаючи порожній рядок як аргумент) об’єднує всі символи масиву в один рядок без будь-яких роздільників між ними.

Пам’ятайте, що рядки в JavaScript є незмінними, тобто ви не можете безпосередньо реверсувати рядок, змінюючи його. Саме тому потрібно спочатку перетворити його на масив, реверсувати масив, а потім знову перетворити на рядок. Ця комбінація методів рядків і масивів дає потужний і гнучкий спосіб маніпулювати рядками в JavaScript.
