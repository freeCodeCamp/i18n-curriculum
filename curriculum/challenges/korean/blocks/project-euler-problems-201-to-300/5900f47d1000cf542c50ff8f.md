# --description--

양수 $n$에 대해, $C(n)$을 $1 < x < n$ 이고 $x^3 \equiv 1\bmod n$인 정수 $x$의 개수로 정의하세요.

$n = 91$일 때, 가능한 $x$ 값은 9, 16, 22, 29, 53, 74, 79, 81로 8개입니다. 따라서 $C(91) = 8$입니다.

$C(n) = 242$인 $n ≤ {10}^{11}$인 양수들의 합을 구하세요.
