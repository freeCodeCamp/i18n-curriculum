# --explanation--

`Right away` significa hacer algo inmediatamente, sin ningún retraso. Por ejemplo:

`Please call me right away if there are any issues.` - Esto significa que debes hacer la llamada lo antes posible.  

Esta frase se usa comúnmente para enfatizar la prontitud en tomar acción.
