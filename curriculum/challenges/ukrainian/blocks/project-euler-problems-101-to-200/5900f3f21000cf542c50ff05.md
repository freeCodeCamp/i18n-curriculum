# --description--

Розглянемо послідовні прості числа $p_1 = 19$ та $p_2 = 23$. Можна перевірити, що 1219 — це найменше число, у якого останні цифри утворені $p_1$, і яке при цьому ділиться на $p_2$.

Насправді, за винятком $p_1 = 3$ та $p_2 = 5$, для кожної пари послідовних простих чисел, де $p_2 > p_1$, існують значення $n$, для яких останні цифри утворені $p_1$, а $n$ ділиться на $p_2$. Нехай $S$ — найменше з таких значень $n$.

Знайдіть $\sum{S}$ для кожної пари послідовних простих чисел з $5 ≤ p_1 ≤ 1000000$.
