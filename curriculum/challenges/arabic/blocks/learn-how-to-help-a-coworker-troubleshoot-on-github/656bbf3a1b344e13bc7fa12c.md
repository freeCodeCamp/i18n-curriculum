# --description--

`Strange` تعني غير عادي أو غير متوقع. يشبه العثور على شيء مختلف عما تراه عادة. على سبيل المثال: `A strange sound in a car might mean a problem.`

الـ`error` هو خطأ، خاصةً إذا كان يوقف شيئًا عن العمل بشكل صحيح. على سبيل المثال: `An error in a code means there is a mistake that needs fixing.`
