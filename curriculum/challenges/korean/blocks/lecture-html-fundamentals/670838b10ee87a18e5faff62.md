# --interactive--

`script` 요소는 실행 가능한 코드를 임베드하는 데 사용됩니다. 대부분의 개발자는 이를 사용해 JavaScript 코드를 실행합니다. JavaScript는 웹 페이지에 상호작용 기능을 추가하는 데 사용됩니다. JavaScript를 사용하는 일반적인 예로는 인터랙티브 게임, 이미지 슬라이더, 사용자 입력을 실시간으로 검증하는 동적 폼 등이 있습니다.

다음은 HTML 문서에서 `script` 요소를 사용하는 예입니다. `//` 앞에 있는 `alert("Welcome to freeCodeCamp");`를 지우면 경고창이 나타납니다. 미리보기를 보려면 인터랙티브 편집기를 활성화해야 합니다.

:::interactive_editor

```html
<body>
  <script>
    // alert("Welcome to freeCodeCamp");
  </script>
</body>
```

:::

기술적으로는 모든 JavaScript 코드를 `script` 태그 안에 작성할 수 있지만, 외부 JavaScript 파일을 링크하는 것이 모범 사례로 간주됩니다. 다음은 `script` 요소를 사용해 외부 JavaScript 파일을 링크하는 예입니다:

```html
<script src="path-to-javascript-file.js"></script>
```

여기서 `src` 속성은 외부 JavaScript 파일의 위치를 지정하는 데 사용됩니다. `src`는 "source"를 의미합니다. 모든 JavaScript를 HTML 문서 안에 넣는 것이 권장되지 않는 이유는 관심사의 분리 때문입니다. 관심사의 분리는 프로그램을 구분된 섹션으로 나누고 각 섹션이 별도의 관심사를 다루도록 하는 설계 원칙입니다. 이 경우 JavaScript 코드를 HTML 코드와 분리하려는 것입니다.
