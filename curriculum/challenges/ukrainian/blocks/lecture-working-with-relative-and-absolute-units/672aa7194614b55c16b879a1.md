# --interactive--

Під час створення своїх сторінок ви працюватимете з різними властивостями, такими як ширина, висота, відступи, поля та інше. Коли ви визначаєте ці властивості, потрібно вказати одиниці вимірювання довжини, які ви хочете використовувати.

Існують два типи одиниць, які можна використовувати для визначення цих властивостей: відносні одиниці та абсолютні одиниці. У цьому уроці ми зосередимося лише на абсолютних одиницях.

Абсолютні одиниці довжини мають фіксовану довжину і не залежать від чогось іншого. Відносні означає, що довжина залежить від чогось іншого, наприклад, від розміру екрана або розміру батьківського елемента. Про відносні одиниці ми поговоримо в наступних уроках.

Найпоширенішою абсолютною одиницею є піксель (`px`). Пікселі — це одиниця вимірювання фіксованого розміру в CSS, що забезпечує точний контроль над розмірами. Це означає, що 1px завжди дорівнює 1/96 дюйма.

Важливо зазначити, що хоча 1px стандартизований як 1/96 дюйма для цілей макета CSS, фактичний фізичний розмір пікселя може відрізнятися залежно від дисплея.

Давайте розглянемо приклад використання пікселів для встановлення ширини та висоти блоку:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="box"></div>
```

```css
.box {
  width: 100px;
  height: 100px;
  background-color: red;
}
```

:::

У цьому прикладі ми маємо червоний блок із шириною та висотою, встановленими в `100px`. Розміри цього блоку залишатимуться однаковими незалежно від змін розміру екрана.

Отже, якщо ви на пристрої з маленьким екраном, блок все одно буде `100px` завширшки. Якщо ви на великому екрані настільного комп’ютера, блок все одно буде `100px` завширшки.

Отже, коли слід використовувати абсолютні одиниці, такі як пікселі? Це залежить від ситуації. Але загалом ви використовуватимете пікселі там, де потрібен точний контроль над розмірами елементів, відстанями та макетом. Іноді пікселі використовують для полів, відступів і кордонів.

Ось приклад використання пікселів для встановлення поля для двох блоків:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="box"></div>
<div class="box"></div>
```

```css
.box {
  width: 100px;
  height: 100px;
  background-color: red;
  margin: 10px;
}
```

:::

Пам’ятайте, що поле — це простір поза межами блоку. Тож у цьому прикладі блок матиме поле `10px` з усіх боків.

Інші типи абсолютних одиниць включають:

- Одиницю `in` (дюйми), яка дорівнює 96px
- Одиницю `cm` (сантиметри), яка дорівнює 25.2/64 дюйма
- Одиницю `mm` (міліметри), яка дорівнює 1/10 сантиметра
- Одиницю `q` (четверті міліметра), яка дорівнює 1/40 сантиметра
- Одиницю `pc` (піки), яка дорівнює 1/6 дюйма
- Одиницю `pt` (пункти), яка дорівнює 1/72 дюйма

Більшість цих одиниць використовують для друку, а не для екранів.

Хоча найпоширенішою одиницею, яку ви використовуватимете, є піксель, важливо знати, що існують і інші абсолютні одиниці.
