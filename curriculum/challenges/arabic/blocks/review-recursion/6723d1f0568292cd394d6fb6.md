# --description--

- التسكين هو مفهوم برمجي يسمح لك باستدعاء دالة بشكل متكرر حتى تصل إلى حالة أساسية.

إليك مثالًا على دالة مسكنة تحسب المضروب لعدد ما:

```js
function findFactorial(n) {
  if (n === 0) {
    return 1;
  }
  return n * findFactorial(n - 1);
}
```

في المثال أعلاه، تُستدعى الدالة `findFactorial` بشكل متكرر حتى يصل `n` إلى `0`. عندما يكون `n` هو `0`، يتم الوصول إلى الحالة الأساسية وتُرجع الدالة القيمة `1`. ثم تُرجع الدالة حاصل ضرب `n` في نتيجة الاستدعاء التكراري لـ `findFactorial(n - 1)`.

- يسمح التسكين بالتعامل مع شيء ذو عمق غير معروف، مثل الكائنات/المصفوفات المسكنة بعمق، أو شجرة الملفات.
- تُستخدم مكدس الاستدعاءات لتتبع استدعاءات الدالة في دالة مسكنة. في كل مرة تُستدعى دالة، تُضاف إلى مكدس الاستدعاءات. عند الوصول إلى الحالة الأساسية، تُزال استدعاءات الدالة من المكدس.
- يجب تعريف الحالة الأساسية بعناية لأن استدعائها بلا نهاية قد يتسبب في تعطل الشفرة. ذلك لأن التسكين يستمر في تراكم المزيد من استدعاءات الدالة حتى تنفد الذاكرة.
- يجد التسكين استخداماته في حل المشكلات الرياضية مثل المضروب وFibonacci، وتصفح الأشجار والرسوم البيانية، وتوليد التباديل والتوافيق، وأكثر من ذلك.
