# --description--

Il verbo `to contain` significa contenere o avere qualcosa all'interno. Nel contesto della documentazione o di un file README, `to contain` si riferisce alle informazioni incluse all'interno del documento.

Per esempio, una bottiglia può `contain` acqua, proprio come un file README `contains` informazioni su un progetto.

# --questions--

## --text--

Nel contesto della documentazione, cosa significa `to contain`?

## --answers--

Escludere

### --feedback--

Significa tenere qualcosa fuori, che è l'opposto di contenere. `To contain` significa includere o avere qualcosa all'interno.

Eliminare

### --feedback--

Significa rimuovere o cancellare qualcosa, che è diverso da `containing`. `Containing` implica trattenere o includere, non rimuovere.

---

Copiare

### --feedback--

Questa espressione significa fare una copia di qualcosa. `Containing`, invece, significa avere qualcosa all'interno, non replicarlo.
