# --description--

如果 $x$、$y$ 和 $x + y$ 都是正的完全平方數，則稱格點 ($x$, $y$) 為不可接受的。

例如，(9, 16) 是不可接受的，而 (0, 4)、(3, 1) 和 (9, 4) 則不是。

考慮一條從點 ($x_1$, $y_1$) 到點 ($x_2$, $y_2$) 的路徑，僅使用向北或向東的單位步伐。若該路徑的中間點皆非不可接受，則稱該路徑為可接受。

設 $P(n)$ 為從 (0, 0) 到 ($n$, $n$) 的可接受路徑數量。可驗證 $P(5) = 252$、$P(16) = 596\\,994\\,440$ 且 $P(1\\,000)\bmod 1\\,000\\,000\\,007 = 341\\,920\\,854$。

尋找 $P(10\,000\,000)\bmod 1\,000\,000\,007$。
