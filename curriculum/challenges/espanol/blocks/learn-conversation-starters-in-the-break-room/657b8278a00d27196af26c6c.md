# --description--

Al elaborar preguntas sobre una tercera persona del singular (`he`, `she` o `it`) podrías necesitar usar el verbo auxiliar `does`.

Por ejemplo: si Alice tiene curiosidad sobre los pasatiempos de Sophie y quiere preguntarle a Tom, puede decir, `Does Sophie like hiking?` La palabra `does` se usa aquí porque el sujeto es `Sophie`, una tercera persona del singular.

Recuerda, `does` se usa para formar preguntas en el tiempo `present simple` para `he`, `she` o `it`.
