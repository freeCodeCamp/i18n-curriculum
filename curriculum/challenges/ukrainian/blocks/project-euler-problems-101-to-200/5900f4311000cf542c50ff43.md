# --description--

Назвемо трикутник із цілими сторонами, у якого рівно один кут дорівнює 60°, трикутником 60°.

Нехай $r$ — радіус вписаного кола такого трикутника 60°.

Існує 1234 трикутники 60°, для яких $r ≤ 100$.

Нехай $T(n)$ — кількість трикутників 60°, для яких $r ≤ n$, отже $T(100) = 1234$, $T(1000) = 22767$ і $T(10000) = 359912$.

Знайдіть $T(1053779)$.
