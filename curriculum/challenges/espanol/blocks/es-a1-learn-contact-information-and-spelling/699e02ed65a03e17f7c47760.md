# --description--

Como viste, `repetir` significa decir algo de nuevo. Cuando agregas el pronombre `lo`, te refieres a algo ya mencionado. El pronombre reemplaza la cosa específica (una palabra, un número, una dirección de correo electrónico, etc.). Entonces:

- `Repetir` = repetir

- `Repetirlo` = repetirlo

`Repetirlo` se usa cuando "la cosa" ya se conoce por el contexto. El mismo patrón funciona con otros verbos:

- `¿Puedes escribirlo?` - ¿Puedes escribirlo?

- `¿Puedes decirlo otra vez?` - ¿Puedes decirlo otra vez?

# --explanation--

`Lo` es un pronombre de objeto directo usado para reemplazar algo ya conocido en la conversación. Cuando se une a un infinitivo, forma una sola palabra:

- `repetir` + `lo` → `repetirlo`  - repetirlo

- `escribir` + `lo` → `escribirlo` - Escríbelo

- `decir` + `lo` → `decirlo` - decirlo

Esta estructura es común cuando el oyente ya entiende a qué se refiere "it".
