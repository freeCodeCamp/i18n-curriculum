# --description--

`Open-source` projects are when el código fuente se hace disponible gratuitamente y puede ser modificado o compartido. Por ejemplo, el navegador web de Firefox es un proyecto de código abierto, lo que significa que cualquier persona puede ver cómo fue hecho y puede ayudar a mejorarlo.

`To work on` something, especially con la preposición `en`, indica poner esfuerzo en una tarea o proyecto a lo largo del tiempo. Like when you say, `I'm working on a new app`, it means you're in the process of developing it.
