# --description--

Gli insiemi sono una delle strutture dati integrate di Python. Una delle caratteristiche principali degli insiemi è che non archiviano valori duplicati. Se provi ad aggiungere un valore duplicato a un insieme, ne verrà archiviato solo uno.

Gli insiemi sono mutabili e non ordinati, il che significa che i loro elementi non sono archiviati in un ordine specifico, quindi non puoi usare indici o chiavi per consultarli. Possono contenere solo valori di tipi di dati immutabili come numeri, stringhe e tuple. Supportano inoltre operazioni matematiche sugli insiemi, inclusi unione, intersezione, differenza e differenza simmetrica.

Per definire un insieme, devi solo scrivere i suoi elementi tra parentesi graffe separandoli con delle virgole. Questo è un esempio di un insieme di numeri:

```python
my_set = {1, 2, 3, 4, 5} 
```

Una particolarità nel lavorare con gli insiemi è che, se hai bisogno di definire un insieme vuoto, devi usare la funzione `set()`. Se scrivi solo parentesi graffe vuote, come `{}`, Python creerà automaticamente un dizionario.

```python
set() # Set
{}    # Dictionary
```

Puoi aggiungere un elemento a un insieme con il metodo `.add()`, passando il nuovo elemento come argomento:

```python
my_set.add(6)
```

Nel nostro esempio, il nuovo insieme sarà:

```python
{1, 2, 3, 4, 5, 6}
```

Se provi ad aggiungere un elemento che è già presente nell’insieme, ne verrà mantenuto solo uno. In questo caso, abbiamo già il numero 5 nell’insieme:

```python
my_set.add(5)
```

Quindi l’insieme non cambierà:

```python
{1, 2, 3, 4, 5, 6}
```

Per rimuovere un elemento dall’insieme, hai due opzioni. Puoi usare il metodo `.remove()` oppure il metodo `.discard()`, passando come argomento l’elemento che vuoi rimuovere.

Il metodo `.remove()` genera un’eccezione `KeyError` se l’elemento non viene trovato, mentre il metodo `.discard()` no:

```python
my_set.remove(4)
my_set.discard(4)
```

Il metodo `.clear()` rimuove tutti gli elementi dall’insieme:

```python
my_set.clear()
```

Gli insiemi di Python hanno anche metodi potenti che eseguono operazioni matematiche comuni sugli insiemi.

I metodi `.issubset()` e `.issuperset()` verificano rispettivamente se un insieme è un sottoinsieme o un sovrainsieme di un altro insieme.

Qui stiamo verificando se `your_set` è un sottoinsieme di `my_set`, il che non è vero perché non tutti gli elementi di `your_set` sono in `my_set`.

Stiamo anche verificando se `my_set` è un sovrainsieme di `your_set`. Anche questo non è vero perché `my_set` non contiene tutti gli elementi di `your_set`:

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

print(your_set.issubset(my_set)) # False
print(my_set.issuperset(your_set)) # False
```

Il metodo `.isdisjoint()` verifica se due insiemi sono disgiunti, cioè se non hanno elementi in comune. In questo caso, non lo sono perché `my_set` e `your_set` hanno elementi in comune – 2, 3 e 4:

```python
print(my_set.isdisjoint(your_set)) # False
```

L’operatore di unione `|` restituisce un nuovo insieme con tutti gli elementi di entrambi gli insiemi:

```python
my_set | your_set # {1, 2, 3, 4, 5, 6}
```

L’operatore di intersezione `&` restituisce un nuovo insieme con solo gli elementi che gli insiemi hanno in comune:

```python
my_set & your_set # {2, 3, 4}
```

L’operatore di differenza `-` restituisce un nuovo insieme con gli elementi del primo insieme che non sono negli altri insiemi. In questo esempio, i numeri 1 e 5 sono in `my_set` ma NON in `your_set`:

```python
my_set - your_set # {1, 5}
```

L’operatore di differenza simmetrica `^` restituisce un nuovo insieme con gli elementi che sono o nel primo o nel secondo insieme, ma non in entrambi. In questo caso, 1 e 5 sono in `my_set` ma non in `your_set`, quindi sono inclusi. E il numero 6 è in `your_set` ma non in `my_set`, quindi è incluso anche lui:

```python
my_set ^ your_set # {1, 5, 6}
```

Ognuno di questi operatori ha anche il rispettivo operatore di assegnazione composta se aggiungi il segno di uguale accanto. Questi operatori assegnano automaticamente il risultato all’insieme a sinistra nell’espressione:

```python
|= &= -= ^=
```

Per esempio, l’operatore `-=` trova la differenza tra gli insiemi e aggiorna il primo insieme con quel risultato:

```python
my_set -= your_set
```

Dopo questo, `my_set` sarà aggiornato a `{1, 5}`:

```python
print(my_set) # {1, 5}
```

Puoi verificare se un elemento è in un insieme o no con l’operatore `in`. Qui stiamo verificando se 5 è in `my_set`. Il risultato sarà un valore booleano `True` o `False`:

```python
print(5 in my_set)
```

E questi sono i fondamenti degli insiemi. Sono molto utili quando non hai bisogno di archiviare i valori in un ordine specifico e quando devi archiviare solo valori unici.
