# --explanation--

`Have to follow` é usado para perguntar sobre obrigações ou requisitos. Por exemplo:

`You have to follow the safety guidelines when using this machine.` - É necessário seguir as regras de segurança ao operar a máquina.

`Have to` funciona como `must` e indica que algo é necessário.

`Follow` é o verbo correto ao se referir a diretrizes, regras ou instruções.
