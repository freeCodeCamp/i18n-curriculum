# --explanation--

问题 `¿Cuántos años tienes?` 询问某人的年龄。

要找到答案，你需要查看表单中的 `Edad` 字段。

该形式显示 `Edad`：25

所以正确答案是 **25**。

这个问题通常用西班牙语通过以下结构回答：

`Tengo` + 数字 + `años`

请注意，字母 **`ñ`** 不仅仅是带有 `virgulilla`（`˜`）的 `n`，它是西班牙语字母表中的一个完全独立的字母，并且它会改变单词的发音和含义。
