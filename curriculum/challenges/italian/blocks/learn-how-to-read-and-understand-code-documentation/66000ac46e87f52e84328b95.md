# --description--

`Interpret` significa cercare di capire o trovare il significato di qualcosa, come scoprire cosa significa una parola difficile o cosa qualcuno sta cercando di dire. Per esempio, `He tried to interpret the instructions as best as he could.`

`Focus` è quando presti particolare attenzione a una cosa specifica, ignorando le altre. È come guardare attraverso una macchina fotografica e regolarla in modo che solo un oggetto sia nitido, mentre tutto il resto è sfocato. Per esempio, `She decided to focus on her math homework to finish it quickly.`
