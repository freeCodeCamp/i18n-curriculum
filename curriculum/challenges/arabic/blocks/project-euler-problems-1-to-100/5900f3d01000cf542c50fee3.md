# --description--

إذا احتوت علبة على واحد وعشرين قرصًا ملونًا، تتألف من خمسة عشر قرصًا أزرق وستة أقراص حمراء، وأُخذ قرصان عشوائيًا، يمكن ملاحظة أن احتمال أخذ قرصين أزرقين هو.

$${P(BB)} = \frac{15}{21}×\frac{14}{20} = \frac{1}{2}$$

الترتيب التالي الذي يكون فيه احتمال أخذ قرصين أزرقين عشوائيًا بالضبط 50% هو علبة تحتوي على خمسة وثمانين قرصًا أزرق وخمسة وثلاثين قرصًا أحمر.

بإيجاد أول ترتيب يحتوي على أكثر من `limit` قرصًا إجمالًا، حدد عدد الأقراص الزرقاء التي ستحتويها العلبة.

```js
function arrangedProbability(limit) {
  // Based on https://www.mathblog.dk/project-euler-100-blue-discs-two-blue/
  let blue = 15;
  let discs = 21;

  while (discs < limit) {
    const nextBlue = 3 * blue + 2 * discs - 2;
    const nextDiscs = 4 * blue + 3 * discs - 3;

    blue = nextBlue;
    discs = nextDiscs;
  }
  return blue;
}
```
