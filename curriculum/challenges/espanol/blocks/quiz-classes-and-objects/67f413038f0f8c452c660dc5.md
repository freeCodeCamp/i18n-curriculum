¿Cuál de las siguientes es la forma correcta de definir una clase?

#### --distractors--

```py
def className(class):
    pass
```

---

```py
self className:
    pass
```

---

```py
class className
    pass
```

#### --answer--

```py
class className:
    pass
```

### --question--

#### --text--

¿Cuál es la diferencia entre un atributo de instancia y un atributo de clase?

#### --distractors--

Los atributos de instancia definen métodos que pueden ser accedidos desde un objeto; los atributos de clase definen métodos que pueden ser accedidos desde una clase.

---

Los atributos de instancia se establecen dentro del método `__init__`; los atributos de clase se establecen dentro del método `__class__`.

---

Los atributos de instancia pueden ser accedidos directamente desde la clase; los atributos de clase solo pueden ser accedidos desde una instancia real de la clase.

#### --answer--

Los atributos de instancia pertenecen a un objeto específico; los atributos de clase pertenecen a una clase.
