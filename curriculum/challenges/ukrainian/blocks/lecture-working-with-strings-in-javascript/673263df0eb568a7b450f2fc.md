# --interactive--

Працюючи з рядками в JavaScript, іноді потрібно включити спеціальні символи, які движок JavaScript інакше може неправильно інтерпретувати.

Дві поширені задачі — це створення нового рядка всередині рядка та екранування певних символів (наприклад, лапок), щоб вони відображалися правильно.

У багатьох мовах програмування, включно з JavaScript, ви можете створити новий рядок у рядку за допомогою спеціального символу, який називається керованою послідовністю. Найпоширеніша керована послідовність для нового рядка — це `\n`.

Наприклад, якщо ви хочете розбити рядок на кілька ліній, ви використовуєте `\n` там, де хочете почати новий рядок:

:::interactive_editor

```js
let poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

Керована послідовність `\n` каже JavaScript вставити розрив рядка в цій точці, в результаті рядок відображається на кількох лініях.

Ще одна важлива концепція при роботі з рядками — це екранування символів. Іноді потрібно включити в рядок символи, які JavaScript зазвичай використовує для інших цілей, наприклад лапки.

Якщо просто вставити лапки всередину рядка без екранування, це може спричинити помилку, бо JavaScript подумає, що ви намагаєтеся завершити рядок.

Наприклад, це спричинить помилку:

```js
let statement = "She said, "Hello!"";
```

JavaScript плутається, бо думає, що рядок закінчується після слова `"said,"`, але ви хочете, щоб лапки навколо `"Hello!"` були частиною рядка.

Щоб виправити це, можна екранувати внутрішні лапки, поставивши перед ними зворотну скісну риску (`\`):

:::interactive_editor

```js
let statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

Зворотна скісна риска каже JavaScript трактувати лапки як буквальні символи, тому вони відображаються правильно у вихідних даних.

Ви також можете екранувати інші спеціальні символи, наприклад саму зворотну скісну риску (`\\`) або одинарні лапки всередині рядка, укладеного в одинарні лапки (`\'`).

Ось ще один приклад із одинарними лапками:

:::interactive_editor

```js
let quote = 'It\'s a beautiful day!';
console.log(quote); // It's a beautiful day!
```

:::

Екранування одинарної лапки за допомогою `\'` дає JavaScript знати, що її слід включити як частину рядка, а не завершувати рядок раніше.

Екранування та створення нових рядків — це необхідні техніки, коли ви форматуватимете вихідні дані або працюватимете зі спеціальними символами в рядках. Вони допомагають уникнути помилок і гарантують, що ваш текст відображатиметься саме так, як задумано.
