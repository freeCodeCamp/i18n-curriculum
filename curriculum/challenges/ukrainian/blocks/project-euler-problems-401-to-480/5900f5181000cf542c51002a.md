# --description--

Послідовність цілих чисел $S = \\{s_i\\}$ називається $n$-послідовністю, якщо вона має $n$ елементів і кожен елемент $s_i$ задовольняє умову $1 ≤ s_i ≤ n$. Таким чином існує загалом $n^n$ різних $n$-послідовностей.

Наприклад, послідовність $S = \\{1, 5, 5, 10, 7, 7, 7, 2, 3, 7\\}$ є 10-послідовністю.

Для будь-якої послідовності $S$ позначимо $L(S)$ як довжину найдовшої суміжної підпослідовності $S$ з однаковим значенням. Наприклад, для наведеної вище послідовності $S$ маємо $L(S) = 3$, через три послідовні сімки.

Нехай $f(n) = \sum L(S)$ для всіх $n$-послідовностей $S$.

Наприклад, $f(3) = 45$, $f(7) = 1\\,403\\,689$ і $f(11) = 481\\,496\\,895\\,121$.

Знайдіть $f(7\\,500\\,000)\bmod 1\\,000\\,000\\,009$.
