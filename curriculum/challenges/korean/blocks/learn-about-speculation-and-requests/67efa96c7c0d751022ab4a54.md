# --explanation--

소피는 마크에게 신중하게 작업하고 제출하기 전에 검토하라고 말합니다.

`Take your time`는 서두르지 않고 신중하게 무언가를 하는 것을 의미합니다. 예를 들어:

`Take your time to review the code before submitting it.` - 서두를 필요 없으니, 코드를 확인하는 데 필요한 만큼 시간을 쓰세요.

`Double-check`는 무언가가 정확한지 다시 검토하는 것을 의미합니다. 예를 들어:

`I will double-check the data before running the analysis.` - 데이터가 정확한지 확인하기 위해 한 번 더 검토할 것입니다.
