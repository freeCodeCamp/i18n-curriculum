# --description--

`Encourage`는 누군가가 무언가를 할 수 있도록 지지하거나 자신감을 주는 것을 의미합니다. 예를 들어, 친구가 `You can solve this coding problem!`라고 말하면 격려하는 것입니다.

`Discourage`는 그 반대이며, 누군가가 덜 자신감을 느끼거나 무언가를 할 가능성이 줄어드는 것을 의미합니다. 만약 그 친구가 `That programming language is too hard for beginners`라고 말하면, 배우는 것을 단념하게 만들 수 있습니다.

`Discouraging`는 낙담하게 만드는 것을 설명합니다. 예를 들어, `Finding a bug in your code after hours of work can be discouraging.`와 같습니다.

`Encouraging`는 무언가가 용기나 희망을 주는 경우입니다. 예를 들어, `Getting your first program to run is very encouraging.`가 있습니다.
