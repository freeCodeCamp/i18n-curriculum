# --explanation--

`must` 和 `have to` 都表示義務或必要性，但它們的用法有些微差異。

`Must` 用於強烈的內部義務或官方規則。它通常聽起來較為正式或具權威性。例如：

`All sensitive data must be encrypted。`－嚴格的規則或要求。

`Have to` 用於外部義務，通常基於規則或情況，而非個人權限。 例如：

你必須確保加密金鑰被安全地儲存。— 這是基於安全政策的責任。
