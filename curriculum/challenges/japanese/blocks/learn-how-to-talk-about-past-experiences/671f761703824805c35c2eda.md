# --explanation--

`get` に続けて形容詞を使う表現は、状態の変化やある状態になることを表します。例えば：

- `They always get excited when a new Python update is released.` - `Get excited` は熱狂的になる、ワクワクするという意味です。これは新しいPythonのアップデートがあるたびにいつも熱狂していることを意味します。

- `He got tired after working on the project all night.` - これは彼が以前は疲れていなかったが、プロジェクトに一晩中取り組んだ後で疲れていることを意味します。

マリアの文では、彼女はブライアンに一緒に働いていたときに新しい技術に対して感じていた熱狂を覚えているか尋ねています。
