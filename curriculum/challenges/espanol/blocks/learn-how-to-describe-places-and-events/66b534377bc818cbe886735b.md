Esta palabra se refiere a períodos dedicados a una actividad particular, como el entrenamiento.

# --explanation--

Cuando preguntes sobre la duración de un evento, puedes usar la pregunta `How long are...?` o `how long is...?` Por ejemplo:

- `How long are these meetings?` - Preguntando por la duración de varias reuniones.

- `How long is it going to take?` - Preguntando por la duración de un evento o actividad específica.
