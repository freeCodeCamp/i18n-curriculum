# --description--

모든 양의 정수 $n$에 대해, $n$번째 약한 Goodstein 수열 $\\{g1, g2, g3, \ldots\\}$는 다음과 같이 정의됩니다:

- $g_1 = n$
- $k > 1$일 때, $g_k$는 $g_{k - 1}$을 $k$진법으로 표현한 후, 이를 $k + 1$진법 숫자로 해석하고 1을 뺀 값입니다.

수열은 $g_k$가 0이 되면 종료됩니다.

예를 들어, 6번째 약한 Goodstein 수열은 $\\{6, 11, 17, 25, \ldots\\}$입니다:

- $g_1 = 6$.
- $g_2 = 11$인데, $6 = 110_2$, $110_3 = 12$, 그리고 $12 - 1 = 11$이기 때문입니다.
- $g_3 = 17$인데, $11 = 102_3$, $102_4 = 18$, 그리고 $18 - 1 = 17$이기 때문입니다.
- $g_4 = 25$인데, $17 = 101_4$, $101_5 = 26$, 그리고 $26 - 1 = 25$이기 때문입니다.

이와 같이 계속됩니다.

모든 약한 Goodstein 수열이 종료됨을 증명할 수 있습니다.

$n$번째 약한 Goodstein 수열에서 0이 아닌 원소의 개수를 $G(n)$이라 합시다.

$G(2) = 3$, $G(4) = 21$, $G(6) = 381$임을 확인할 수 있습니다.

또한 $1 ≤ n < 8$에 대해 $\sum G(n) = 2517$임을 확인할 수 있습니다.

$1 ≤ n < 16$에 대해 $\sum G(n)$의 마지막 9자리를 구하세요.
