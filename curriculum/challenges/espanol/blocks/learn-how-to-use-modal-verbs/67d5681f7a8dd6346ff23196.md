## --text--

¿Cuándo usarías verbos modales como `should`, `must` y `have to`?

## --answers--

Para describir eventos pasados en detalle.

### --feedback--

Estos verbos modales no se usan para eventos pasados.

---

Para expresar necesidad, obligación o recomendaciones.

---

Para comparar diferentes acciones o ideas.

### --feedback--

Estos verbos modales no expresan comparaciones.

---

Para pedir permiso para hacer algo.

### --feedback--

Para permiso, los verbos modales como `can` o `may` son más apropiados.

# --explanation--

Recuerda:

`Must` expresa una necesidad u obligación fuerte. Por ejemplo:

Debes usar el cinturón de seguridad.

`Have to` también expresa obligación pero a veces es menos estricto que must. Por ejemplo:

Tienes que entregar el informe antes del viernes.

`Should` se usa para recomendaciones o consejos. Por ejemplo:

Debes revisar tus notas antes del test.
