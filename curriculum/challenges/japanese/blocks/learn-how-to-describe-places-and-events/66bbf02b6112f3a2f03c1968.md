# --explanation--

`It`は、すでに言及された名詞を置き換えて、文中の繰り返しを避けるために使われます。例えば：

- `I bought a new laptop. It works really well.` - ここで`it`は`laptop`を置き換えて、同じ単語の繰り返しを避けています。

`The Live Preview isn't showing in Visual Studio Code. Isn't it supposed to support that feature?`では、トムがプログラムとその機能のサポート能力について話しているため、`it`は`Visual Studio Code`を指しています。

`It`はプログラム名の繰り返しを避けるために`Visual Studio Code`を置き換えています。

機能をサポートするものが何かを特定することで、`it`がプログラム自体を指していると判断できます。
