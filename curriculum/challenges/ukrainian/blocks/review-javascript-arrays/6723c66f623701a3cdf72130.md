## Основи масивів JavaScript

- **Визначення**: Масив JavaScript — це впорядкована колекція значень, кожне з яких ідентифікується числовим індексом. Значення в масиві JavaScript можуть бути різних типів даних, включно з числами, рядками, булевими значеннями, об’єктами та навіть іншими масивами. Масиви розташовані безперервно в пам’яті, що означає, що всі елементи зберігаються в одному безперервному блоці пам’яті, що дозволяє ефективно індексувати та швидко отримувати доступ до елементів за їх індексом.

```js
const developers = ["Jessica", "Naomi", "Tom"];
```

- **Доступ до елементів масивів**: Щоб отримати доступ до елементів масиву, потрібно звернутися до масиву, а потім вказати його індекс у квадратних дужках. Масиви JavaScript мають індексацію з нуля, тобто перший елемент знаходиться під індексом 0, другий — під індексом 1 і так далі. Якщо ви спробуєте звернутися до індексу, якого немає в масиві, JavaScript поверне `undefined`.

:::interactive_editor

```js
const developers = ["Jessica", "Naomi", "Tom"];
console.log(developers[0]) // "Jessica"
console.log(developers[1]) // "Naomi"

console.log(developers[10]) // undefined
```

:::

- **Властивість `length`**: Ця властивість використовується, щоб повернути кількість елементів у масиві.

:::interactive_editor

```js
const developers = ["Jessica", "Naomi", "Tom"];
console.log(developers.length) // 3
```

:::

- **Оновлення елементів у масиві**: Щоб оновити елемент у масиві, використовуйте оператор присвоєння (`=`), щоб призначити нове значення елементу за певним індексом.

:::interactive_editor

```js
const fruits = ['apple', 'banana', 'cherry'];
fruits[1] = 'blueberry';

console.log(fruits); // ['apple', 'blueberry', 'cherry']
```

:::

## Двовимірні масиви  

- **Визначення**: Двовимірний масив — це по суті масив масивів. Його використовують для представлення даних, які мають природну сіткову структуру, наприклад шахівницю, електронну таблицю або пікселі зображення. Щоб отримати доступ до елемента в двовимірному масиві, потрібно два індекси: один для рядка, інший — для стовпця.

:::interactive_editor

```js
const chessboard = [
    ['R', 'N', 'B', 'Q', 'K', 'B', 'N', 'R'],
    ['P', 'P', 'P', 'P', 'P', 'P', 'P', 'P'],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    ['p', 'p', 'p', 'p', 'p', 'p', 'p', 'p'],
    ['r', 'n', 'b', 'q', 'k', 'b', 'n', 'r']
];

console.log(chessboard[0][3]); // "Q"
```

:::

## Деструктуризація масивів  

- **Визначення**: Деструктуризація масивів — це функція в JavaScript, яка дозволяє витягувати значення з масивів і призначати їх змінним у більш лаконічний і читабельний спосіб. Вона надає зручний синтаксис для розпакування елементів масиву в окремі змінні.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange"];

const [first, second, third] = fruits;

console.log(first); // "apple"
console.log(second); // "banana"
console.log(third); // "orange"
```

:::

- **Синтаксис rest**: Дозволяє захопити залишок елементів масиву, які не були деструктуризовані, у новий масив.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange", "mango", "kiwi"];
const [first, second, ...rest] = fruits;

console.log(first); // "apple"
console.log(second); // "banana"
console.log(rest); // ["orange", "mango", "kiwi"]
```

:::

## Загальні методи масивів 

- **Метод `push()`**: Цей метод використовується, щоб додати елементи в кінець масиву і повертає нову довжину.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.push("ice cream");

console.log(desserts); // ["cake", "cookies", "pie", "ice cream"];
```

:::

- **Метод `pop()`**: Цей метод використовується, щоб видалити останній елемент з масиву і повертає цей видалений елемент. Якщо масив порожній, повернене значення буде `undefined`.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.pop();

console.log(desserts); // ["cake", "cookies"];
```

:::

- **Метод `shift()`**: Цей метод використовується, щоб видалити перший елемент з масиву і повертає цей видалений елемент. Якщо масив порожній, повернене значення буде `undefined`.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.shift();

console.log(desserts); // ["cookies", "pie"];
```

:::

- **Метод `unshift()`**: Цей метод використовується, щоб додати елементи на початок масиву і повертає нову довжину.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.unshift("ice cream");

console.log(desserts); // ["ice cream", "cake", "cookies", "pie"];
```

:::

- **Метод `indexOf()`**: Цей метод корисний для знаходження першого індексу певного елемента в масиві. Якщо елемент не знайдено, він поверне `-1`.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange", "banana"];
const index = fruits.indexOf("banana");

console.log(index); // 1
console.log(fruits.indexOf("not found")); // -1
```

:::

- **Метод `splice()`**: Цей метод використовується для додавання або видалення елементів з будь-якої позиції в масиві. Повернене значення методу `splice()` буде масивом елементів, видалених з масиву. Якщо нічого не видаляється, повертається порожній масив. Цей метод змінює оригінальний масив, модифікуючи його на місці, а не створюючи новий. Перший аргумент вказує індекс, з якого починається модифікація масиву. Другий аргумент — кількість елементів, які потрібно видалити. Наступні аргументи — елементи, які потрібно додати.

:::interactive_editor

```js
const colors = ["red", "green", "blue"];
colors.splice(1, 0, "yellow", "purple");

console.log(colors); // ["red", "yellow", "purple", "green", "blue"]
```

:::

- **Метод `includes()`**: Цей метод використовується, щоб перевірити, чи містить масив певне значення. Метод повертає `true`, якщо масив містить вказаний елемент, і `false` в іншому випадку.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];

console.log(programmingLanguages.includes("Python")); // true
console.log(programmingLanguages.includes("Perl")); // false
```

:::

- **Метод `concat()`**: Цей метод створює новий масив, об’єднуючи два або більше масивів.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];
const newList = programmingLanguages.concat("Perl");

console.log(newList); // ["JavaScript", "Python", "C++", "Perl"]
```

:::

- **Метод `slice()`**: Цей метод повертає новий масив, що містить поверхневу копію частини оригінального масиву, визначеної початковим і кінцевим індексами. Новий масив містить референси на ті ж елементи, що й оригінальний масив (не дублікати). Це означає, що якщо елементи — примітиви (наприклад, числа або рядки), значення копіюються; але якщо елементи — об’єкти або масиви, копіюються референси, а не самі об’єкти.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];
const newList = programmingLanguages.slice(1);

console.log(newList); // ["Python", "C++"]
```

:::

- **Синтаксис розширення**: Синтаксис розширення використовується для створення поверхневих копій масиву.

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const shallowCopiedArray = [...originalArray];

shallowCopiedArray.push(4);

console.log(originalArray); // [1, 2, 3]
console.log(shallowCopiedArray); // [1, 2, 3, 4]
```

:::

- **Метод `split()`**: Цей метод розбиває рядок на масив підрядків і визначає, де має відбутися кожне розбиття, на основі заданого роздільника. Якщо роздільник не вказано, метод повертає масив, що містить оригінальний рядок як єдиний елемент.

:::interactive_editor

```js
const str = "hello";
const charArray = str.split("");

console.log(charArray); // ["h", "e", "l", "l", "o"]
```

:::

- **Метод `reverse()`**: Цей метод змінює порядок елементів масиву на зворотний.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
console.log(desserts.reverse()); // ["pie", "cookies", "cake"]
```

:::

- **Метод `join()`**: Цей метод об’єднує всі елементи масиву в один рядок, розділяючи кожен елемент вказаним роздільником. Якщо роздільник не вказано або використано порожній рядок (`""`), елементи будуть об’єднані без роздільника.

:::interactive_editor

```js
const reversedArray = ["o", "l", "l", "e", "h"];
const reversedString = reversedArray.join("");

console.log(reversedString); // "olleh"
```

:::
