# --description--

قبل العمل مع متغيرات Python، من المهم فهم أنواع البيانات. نوع البيانات يصف نوع القيمة التي يحملها المتغير. على سبيل المثال، رقم، قطعة نص، أو قائمة من العناصر. تستخدم لغات البرمجة أنواع البيانات لتعرف كيف تخزن وتتعامل مع أنواع مختلفة من المعلومات.

Python لغة ذات نوع ديناميكي مثل JavaScript، مما يعني أنك لا تحتاج إلى إعلان أنواع المتغيرات صراحةً. اللغة تعرف نوع البيانات للمتغير بناءً على ما تعيّنه له.

إليك بعض الأمثلة:

```python
name = 'John Doe' # Python knows this is a string
age = 25 # Python knows this is an integer
```

هذا على عكس بعض اللغات ذات النوع الثابت مثل C# وJava وC++، حيث يجب إعلان أنواع المتغيرات، مثل هذا:

```csharp
string name = 'John Doe'
int age = 25
```

الطبيعة الديناميكية للنوع في Python تجعل الترميز سريعًا ومرنًا أكثر، لكنها قد تؤدي إلى أخطاء برمجية غير متوقعة لأن أخطاء النوع تُكتشف فقط عند تشغيل البرنامج، وليس عند تجميعه.

بما أن Python تحدد أنواع البيانات أثناء **تشغيل** البرنامج، فإن الأخطاء المتعلقة بالنوع تُكتشف فقط في تلك اللحظة. عند تشغيل البرنامج، ينفذ Python الشفرة سطرًا بسطر. إذا وصل إلى سطر يُتوقع فيه أن يتصرف كائن معين بطريقة غير ممكنة له، سيتوقف Python ويعرض خطأ.

على النقيض، بعض اللغات **تُجمّع** برنامجك قبل تشغيله. التجميع يعني أن الحاسوب يتحقق من الشفرة مسبقًا ويجهزها للتنفيذ. خلال هذه الخطوة، يمكن لتلك اللغات اكتشاف أخطاء النوع قبل بدء البرنامج.

لا تحتاج إلى معرفة تلك اللغات الآن. الفكرة المهمة ببساطة هي:

- **في Python يمكن لأخطاء النوع أن تظهر أثناء التنفيذ**، عندما يكون البرنامج قيد التشغيل ويستخدم شفرتك.

- **اللغات المجمعة تكتشف أخطاء النوع أثناء خطوة التجميع**، قبل السماح بتشغيل البرنامج.

لذلك، قد لا تعرف عن خطأ نوع في Python حتى يصل البرنامج إلى ذلك السطر المحدد من الشفرة أثناء التشغيل.

إليك أكثر أنواع البيانات شيوعًا التي ستستخدمها في Python:

- Integer: عدد صحيح بدون كسور عشرية، مثل `10` أو `-5`.

```python
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- Float: عدد يحتوي على فاصلة عشرية، مثل `4.41` أو `-0.4`.

```python
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- String: متتالية من الأحرف محاطة بعلامات اقتباس مفردة أو مزدوجة مثل `'Hello world!'`.

```python
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- Boolean: نوع بقيمة منطقية صحيحة أو خاطئة، مكتوبة كـ `True` أو `False`.

```python
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- Set: مجموعة غير مرتبة من العناصر الفريدة، مثل `{0.5, 4, 'apple'}`.
 
```python
my_set_var = {7, 'hello', 8.5}
print('Set:', my_set_var) # Set: {7, 'hello', 8.5}
```

- Dictionary: مجموعة من أزواج المفتاح-القيمة محاطة بأقواس متعرجة، مثل `{'name': 'John Doe', 'age': 28}`.

```python
my_dictionary_var = {'name': 'Alice', 'age': 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- Tuple: مجموعة مرتبة غير قابلة للتغيير، محاطة بأقواس عادية، مثل `('apple', 4.5, 7)`.

```python
my_tuple_var = (7, 'hello', 8.5)
print('Tuple:', my_tuple_var) # Tuple: (7, 'hello', 8.5)
```

- Range: متتالية من الأعداد، تُستخدم غالبًا في الحلقات، مثل `range(5)`.

```python
my_range_var = range(5)
print('Range:', my_range_var) # Range: range(0, 5)
```

- List: مجموعة مرتبة من العناصر تدعم أنواع بيانات مختلفة.
    
```python
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- None: قيمة خاصة تمثل غياب القيمة.

```python
my_none_var = None
print('None:', my_none_var) # None: None
```

في الدروس القادمة، ستتعلم المزيد عن كيفية العمل مع كل هذه الأنواع من البيانات.
