# --explanation--

`I see.` використовується, щоб коротко підтвердити, що мовник зрозумів, що говорить інша людина. Це не обов’язково означає згоду, але показує визнання інформації. Наприклад:

`I see. So the project deadline has been moved up?` — це означає, що мовник розуміє сказане і обробляє інформацію.
