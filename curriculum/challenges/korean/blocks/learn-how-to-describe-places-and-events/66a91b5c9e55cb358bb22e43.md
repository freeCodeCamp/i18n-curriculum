# --explanation--

`Plan to work on`는 무언가를 시작하거나 계속하려는 의도나 결정을 의미합니다. 미래에 하려는 행동이나 프로젝트에 대해 말할 때 사용합니다.

`work on`라고 하면 특정 작업이나 프로젝트에 시간과 에너지를 쏟는다는 뜻입니다. 예를 들어:

- `I need to work on my presentation for tomorrow.` - 다음 날 발표 준비를 위해 시간을 할애해야 합니다.

- `She plans to work on her novel this summer.` - 그녀는 여름 동안 소설을 쓰거나 다듬는 데 시간을 보내려고 합니다.
