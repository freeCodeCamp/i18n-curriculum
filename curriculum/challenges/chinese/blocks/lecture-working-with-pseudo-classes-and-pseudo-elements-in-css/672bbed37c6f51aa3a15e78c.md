# --interactive--

CSS 最有趣的方面之一是伪元素的使用。在这个上下文中，“pseudo” 意味着“非真实”，所以伪元素是虚拟的或合成的元素，它们不直接匹配任何实际的 HTML 元素。它们允许你为元素的特定部分添加样式或插入内容，而无需添加额外的 HTML。

要应用伪元素，请使用双冒号（`::`）将其附加到原始元素的选择器上。请注意，选择器可以是任何类型，例如类选择器或 ID 选择器。伪元素的基本语法如下所示：

```css
selector::pseudo-element {
  property: value;
}
```

这个双冒号是区分伪元素和使用单冒号的伪类的标志。

伪元素允许你为元素内容的特定部分设置样式或在其前后插入内容，但它们不能独立存在。伪元素所附加的元素称为其起始元素。

让我们先来看一些关于 `::before` 和 `::after` 伪元素的示例。正如它们的名称所示，`::before` 允许你在元素的内容之前插入内容，而 `::after` 允许你在内容之后插入内容。

这是一个按钮元素的示例。在 CSS 中，我们将使用绝对定位和 `::before` 伪元素为按钮的 `Learn More` 文本前添加一个星号。你将在后续课程中学习更多关于绝对定位的内容。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: lightseagreen;
  color: white;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
  position: relative;
}

.cta-button::before {
  content: "⭐";
  position: absolute;
  left: 3px;
  top: 8px;
  font-size: 0.75rem;
}
```

:::

`content` 属性用于表现你希望在按钮文本前添加的内容。在这个例子中，我们添加了一个星星。你会注意到你不仅可以插入内容，还可以为其添加样式。以下是带有相同按钮的 `::after` 伪元素示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: '➡️';
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}
```

:::

在 `::after` 伪元素中，`transition` 属性用于以 0.3 秒的缓动效果动画变化，创建平滑且渐进的转换，而不是突然的变化。你将在后续课程中学习更多关于 `transition` 属性的内容。 

你还可以使用 `::before` 和 `::after` 伪元素，将伪类附加到插入到另一个内容中的内容上。例如，内容的悬停状态。以下是一个示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: "➡️";
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}

.cta-button:hover::after {
  transform: translateX(2px);
}
```

:::

在悬停状态下使用 `transform: translateX(2px)`，内容在用户每次悬停按钮时都会向右移动 `2px`。`::after` 本身的 transition 属性确保该进程耗时 `0.3s`。

这就是 `transform` 属性的作用——它允许你旋转、倾斜、缩放或沿特定方向平移一个元素。你将在后续课程中学习更多内容。

在下一个示例中，我们将查看 `::first-letter` 伪元素。`::first-letter` 伪元素定位元素内容的第一个字母，允许你对其进行样式设置。这里有一段段落文本的示例。如果我们想要设置第一个字母的样式，可以像这样使用 `::first-letter` 伪元素：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>freeCodeCamp lets you learn to code without having to pay.</p>
```

```css
p::first-letter {
  font-size: 4rem;
}
```

:::

在最后一个示例中，我们将查看 `::marker` 伪元素，它允许你选择列表项的标记、项目符号或编号以进行样式设置。`::marker` 伪元素提供了一种通过自定义列表标记以匹配你的配色方案来增强你网站品牌同一性的方法。

这是一个无序列表和有序列表的示例。要更改列表项的标记颜色和大小，你可以像这样使用 `::marker` 伪元素：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<ul>
  <li>Unordered list item 1</li>
  <li>Unordered list item 2</li>
  <li>Unordered list item 3</li>
  <li>Unordered list item 4</li>
</ul>

<ol>
  <li>Ordered list item 1</li>
  <li>Ordered list item 2</li>
  <li>Ordered list item 3</li>
  <li>Ordered list item 4</li>
</ol>
```

```css
li::marker {
  color: crimson;
  font-size: 1.5em;
  font-weight: bold;
}
```

:::

在本课中，我们只介绍了少数几个伪元素。但还有许多其他的伪元素，比如 `::placeholder`、`::spelling-error` 和 `::selection`，我鼓励你自己去探索。
