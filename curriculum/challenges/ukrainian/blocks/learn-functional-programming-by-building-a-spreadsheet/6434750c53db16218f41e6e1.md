# --description--

Пам’ятайте, що `range()` повертає масив, тож ви можете напряму ланцюжити методи масиву до виклику функції.

Викличте `range()` з `1` та `99` як аргументами, і ланцюжте метод `.forEach()`. Передайте методу `.forEach()` порожню функцію зворотного виклику, яка приймає `number` як параметр.
