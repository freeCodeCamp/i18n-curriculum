# --explanation--

`Typically` é usado para descrever algo que acontece de uma maneira usual ou esperada.É semelhante a `usually` e é frequentemente usado para falar sobre o que normalmente acontece.Por exemplo:

- `Typically, we have meetings on Mondays.` - Isso significa que as reuniões geralmente acontecem às segundas-feiras.

- `She typically arrives at work by 9 AM.` - Isso significa que ela geralmente chega ao trabalho até as 9h.
