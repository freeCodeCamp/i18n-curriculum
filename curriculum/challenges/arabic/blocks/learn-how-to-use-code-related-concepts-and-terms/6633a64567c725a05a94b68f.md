# --description--

في هذه المهمة، ستستكشف الاستخدام المجازي للمصطلحات لفهم مفاهيم البرمجة. الاستعارات وسيلة قوية لتصوّر وشرح الأفكار المعقدة من خلال ربطها بأشياء أو أفعال من الحياة اليومية.

يُستخدم مصطلح `Container` بشكل مجازي لوصف المتغيرات كأشياء يمكنها الاحتفاظ وتخزين بيانات مختلفة، تمامًا كما يمكن لـ `container` أن يحمل عناصر متنوعة.

الأفعال `fill and empty` تصف كيف يمكنك إضافة البيانات إلى المتغيرات أو إزالتها، تمامًا كما يمكنك `fill` زجاجة ماء أو تفريغها.

`Locked boxes` تصف الثوابت بأنها `containers` آمنة وغير قابلة للتغيير، وتؤكد أنه لا يمكن تعديلها بعد ضبطها.

تُبرز عبارة `keep the same contents` أن البيانات في الثوابت لا تتغير، مثل صندوق مغلق يحتفظ دائمًا بنفس الأشياء داخله.
