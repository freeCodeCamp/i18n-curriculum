# --instructions--

In questa sfida, devi creare un modello di machine learning che classifichi i messaggi SMS come "ham" o "spam". Un messaggio "ham" è un messaggio normale inviato da un amico. Un messaggio "spam" è una pubblicità o un messaggio inviato da un'azienda.

Devi creare una funzione chiamata `predict_message` che prende una stringa di messaggio come argomento e restituisce una lista. Il primo elemento della lista deve essere un numero tra zero e uno che indica la probabilità di "ham" (0) o "spam" (1). Il secondo elemento della lista deve essere la parola "ham" o "spam", a seconda di quale sia più probabile.

Per questa sfida, userai il dataset SMS Spam Collection. Il dataset è già stato suddiviso in dati di addestramento e dati di test.

Le prime due celle importano le librerie e i dati. L'ultima cella testa il tuo modello e la tua funzione. Aggiungi il tuo codice tra queste celle.
