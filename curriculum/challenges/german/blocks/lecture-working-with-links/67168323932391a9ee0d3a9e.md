# --interactive--

Sie haben vielleicht schon gesehen, dass ein Link auf einer Webseite nach dem Klicken lila wird. Das liegt daran, dass sich der Zustand des Links geändert hat und deshalb eine andere Gestaltung angewendet wird. Es gibt fünf verschiedene Zustände, in denen sich ein Link befinden kann.

Der erste ist der Standardzustand, der `:link` ist. Dieser Zustand stellt einen Link dar, den der Benutzer noch nicht besucht, angeklickt oder mit dem er noch nicht interagiert hat. Sie können sich diesen Zustand als die Basisgestaltung für alle Links auf Ihrer Seite vorstellen. Die anderen Zustände bauen darauf auf.

Aktivieren Sie den interaktiven Editor und schauen Sie sich den Link im Vorschaufenster an. Er sollte die Standardfarbe Blau haben, die den Standardzustand darstellt. Wenn Sie darauf klicken, wird er lila.

:::interactive_editor

```html
<a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
```

:::

Der zweite Zustand ist `:visited`, der gilt, wenn ein Benutzer die verlinkte Seite bereits besucht hat. Standardmäßig wird der Link dann lila – aber Sie können CSS nutzen, um dem Benutzer eine andere visuelle Rückmeldung zu geben. Das ist hilfreich, um jemandem zu zeigen, dass er einen Teil Ihrer Dokumentation schon gelesen hat. Oder dass die Seite vertrauenswürdig ist, weil er sie schon benutzt hat.

Klicken Sie im Vorschaufenster auf den Link, und Sie sollten sehen, dass der besuchte Link braun wird. Um die Vorschauen zu sehen, müssen Sie den interaktiven Editor aktivieren.

**HINWEIS**: Für dieses interaktive Beispiel wurde etwas CSS bereitgestellt, damit Sie die Änderungen in den Link-Stilen sehen können. Machen Sie sich keine Sorgen, wenn Sie den CSS-Code noch nicht verstehen – Sie lernen alles dazu in zukünftigen Modulen.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:visited {
  color: brown;
}
```

:::

Der dritte Zustand ist `:hover`. Dieser Zustand gilt, wenn ein Benutzer mit dem Cursor über einen Link fährt. Dieser Zustand ist hilfreich, um einem Link mehr Aufmerksamkeit zu geben und sicherzustellen, dass der Benutzer tatsächlich vorhat, darauf zu klicken.

Aktivieren Sie den interaktiven Editor und fahren Sie mit der Maus über den Link. Sie werden sehen, dass die Farbe sich zu Rot ändert.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:hover {
  color: red;
}
```

:::

Dann haben wir `:focus`. Dieser Zustand gilt, wenn wir den Fokus auf einen Link setzen.

Klicken Sie auf einen beliebigen Bereich des Leerraums im Vorschaufenster und drücken Sie dann `tab` auf Ihrer Tastatur. Sie sollten sehen, dass der Link grün wird. Um die Vorschauen zu sehen, müssen Sie den interaktiven Editor aktivieren.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:focus {
  color: green;
}
```

:::

Und schließlich gibt es `:active`. Dieser Zustand gilt für Links, die vom Benutzer aktiviert werden. Das bedeutet in den meisten Fällen, dass der Link mit der primären Maustaste durch Linksklick angeklickt wird. Dieser Zustand kann hilfreich sein, um dem Benutzer zu zeigen, dass das Element, auf das er geklickt hat, interaktiv ist.

Aktivieren Sie den interaktiven Editor und klicken Sie auf den Link. Sie sollten sehen, dass der Link schwarz wird. Das passiert ziemlich schnell, deshalb müssen Sie vielleicht mehrmals klicken, um die Farbänderung zu sehen.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:active {
  color: black;
}
```

:::

Wenn Sie diese Zustände verwenden, um Ihre Links zu gestalten, gibt es eine bestimmte Reihenfolge, in der Sie Ihr CSS schreiben müssen: `link`, `visited`, `hover`, `focus`, dann `active`.

Sie wissen jetzt, wie Sie Links auf Ihrer Seite Ihre persönliche Note geben können und gleichzeitig die wichtigen Informationen über den Zustand jedes Links bereitstellen.
