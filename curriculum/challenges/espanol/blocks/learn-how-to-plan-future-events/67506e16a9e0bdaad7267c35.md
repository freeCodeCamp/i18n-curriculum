# --explanation--

`And when does the team perform the final checks?`, usa el tiempo `Present Simple` (`does perform`) para preguntar sobre un evento futuro programado. 

La respuesta correcta, `The final system checks are set for Friday night`, también usa el `Present Simple` para describir un plan fijo. 

`Present Simple` se usa a menudo para eventos futuros que forman parte de un horario o calendario. Por ejemplo:

- `The train departs at 8 PM tonight.` - Esto indica una salida programada.

- `Our meeting starts tomorrow morning.` - Esto muestra un tiempo planeado para un evento futuro.

En este contexto, el `Present Simple` hace que la respuesta sea concisa y adecuada para hablar de un evento fijo.
