# --description--

CSS ya ndani (au CSS iliyojumuishwa) inahusisha kuongeza CSS ndani ya faili la HTML yenyewe badala ya kuunda faili tofauti kabisa. Kwa njia ya ndani, unaweka sheria zote ndani ya jozi ya mabano ya mviringo ya kufungua na kufunga `<style>`, ambazo kisha huwekwa ndani ya mabano ya mviringo ya kufungua na kufunga `<head>` ya faili yako ya HTML. Kwa kuwa mitindo inawekwa moja kwa moja ndani ya mabano ya mviringo ya `<head>`, hautahitaji tena kipengele cha `<link>` ambacho njia ya nje inahitaji.

Mbali na tofauti hizi, sintaksia ni sawa kabisa na njia ya nje (kichaguzi, mabano ya mviringo, tamko):

```html
<head>
  <style>
    div {
      color: white;
      background-color: black;
    }

    p {
      color: red;
    }
  </style>
</head>
<body>...</body>
```

Njia hii inaweza kuwa na manufaa kwa kuongeza mitindo ya kipekee kwa ukurasa mmoja wa tovuti, lakini haifanyi mambo kuwa tofauti kama njia ya nje, na kulingana na idadi ya sheria na tamko, inaweza kusababisha faili la HTML kuwa kubwa sana.

## --text--

Ni ipi kati ya zifuatazo ni tofauti kati ya njia za CSS za ndani na nje?

## --answers--

Njia ya nje inaweka sheria za CSS katika faili tofauti, wakati njia ya ndani inaweka sheria za CSS ndani ya faili la HTML yenyewe.

---

Njia ya ndani huweka CSS tofauti na HTML, wakati njia ya nje huingiza CSS moja kwa moja ndani ya HTML.

---

Njia ya ndani hutumia kipengele cha `<link>` kuunganisha CSS na HTML, wakati njia ya nje huingiza CSS moja kwa moja ndani ya HTML.
