# --explanation--

`Built-in` تشير إلى الميزات أو الأدوات التي تُدرج كجزء من برنامج أو جهاز من البداية، دون الحاجة إلى إضافتها أو تثبيتها بشكل منفصل.

عندما يكون شيء ما `built-in`، فهو موجود بالفعل وجاهز للاستخدام. على سبيل المثال:

- `This laptop has a built-in camera.` - الكاميرا مدمجة كجزء من اللابتوب.

- `The software comes with built-in security features.` - ميزات الأمان مدمجة بالفعل في البرنامج.
