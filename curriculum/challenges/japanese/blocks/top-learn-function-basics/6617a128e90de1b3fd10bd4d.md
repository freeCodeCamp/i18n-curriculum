# --description--

次の例の関数を使って、パラメータと引数について説明します。

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

JavaScriptでは、パラメータは関数宣言の丸括弧 `()` の間にリストされている項目です。関数の引数は、関数に渡す実際の値のことです。

上の例では、関数の定義が1行目に書かれています：`function favoriteAnimal(animal)`。パラメータの`animal`は丸括弧の中にあります。`animal`を`pet`、`x`、または`blah`に置き換えても問題ありません。しかしこの場合、パラメータを`animal`と名付けることで、コードを読む人が`animal`に何が入るのか推測しなくて済むように、少し文脈を与えています。
