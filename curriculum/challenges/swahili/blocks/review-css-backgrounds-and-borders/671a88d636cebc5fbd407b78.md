# --interactive--

## Kupamba orodha

- **Sifa ya `line-height`**: Sifa hii hutumika kuunda nafasi kati ya mistari ya maandishi. Thamani zinazokubalika za `line-height` ni pamoja na neno kuu `normal`, nambari, asilimia na vitengo vya urefu kama kitengo cha `em`.
- **Sifa ya `list-style-type`**: Sifa hii hutumika kubainisha alama ya kipengee cha orodha. Thamani zinazokubalika zinaweza kujumuisha mduara, diski, au nambari za desimali.

Hapa kuna mfano unaotumia `list-style-type` kubadilisha mtindo wa alama:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
ul {
  list-style-type: square;
}
```

:::

- **Sifa ya `list-style-position`**: Sifa hii hutumika kuweka nafasi ya alama ya orodha. Thamani mbili pekee zinazokubalika ni ndani na nje.

Hapa kuna mfano unaoonyesha tofauti kati ya `inside` na `outside`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="inside-list">
  <li>Item with inside position</li>
  <li>Item with inside position</li>
</ul>
<ul class="outside-list">
  <li>Item with outside position</li>
  <li>Item with outside position</li>
</ul>
```

```css
.inside-list {
  list-style-position: inside;
}

.outside-list {
  list-style-position: outside;
}
```

:::

- **Sifa ya `list-style-image`**: Sifa hii hutumika kutumia picha kama alama ya kipengee cha orodha. Matumizi ya kawaida ni kutumia kitendakazi cha `url` na thamani iliyowekwa kwa eneo halali la picha.

## Kuongeza nafasi kati ya vipengee vya orodha kwa kutumia `margin`

- Mbali na `line-height`, kingo pia zinaweza kutumika katika CSS kuboresha nafasi na urahisi wa kusoma vipengee vya orodha.
- Kingo huunda nafasi nje ya kila kipengee cha `li`, kuruhusu udhibiti wa pengo kati ya vipengee vya orodha.
- `margin-bottom` hutumika kuunda nafasi chini ya kila kipengee cha orodha. Kwa mfano, `margin-bottom: 10px;` itaunda pengo la pikseli 10 chini ya kila kipengee cha orodha.

Hapa kuna mfano unaotumia `margin-bottom` kuongeza nafasi kati ya vipengee vya orodha:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="list">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
.list li {
  margin-bottom: 20px;
}
```

## Kupamba viungo

- **`pseudo-class`**: Hii ni neno kuu linaloongezwa kwenye kichaguzi kinachokuruhusu kuchagua vipengele kulingana na hali fulani. Hali za kawaida ni pamoja na hali za `:hover`, `:visited` na `:focus`.
- **`:link pseudo-class`**: Darasa la bandia hili hutumika kupamba viungo ambavyo mtumizi hajatembelea bado.

Hapa kuna mfano unaotumia darasa la bandia la `:link`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:link {
  color: red;
}
```

:::

- **`:visited pseudo-class`**: Darasa la bandia hili hutumika kupamba viungo ambavyo mtumizi tayari amevitembelea.
- **`:hover pseudo-class`**: Darasa la bandia hili hutumika kupamba vipengele ambavyo mtumizi kwa sasa anavihudhuria kwa kuvinjari.

Hapa kuna mfano unaotumia darasa la bandia la `:hover`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Hover over me!</a>
```

```css
a:hover {
  color: green;
  text-decoration: underline;
}
```

:::

- **`:focus pseudo-class`**: Darasa la bandia hili hutumika kupamba kipengee kinapopata makini. Mifano ni pamoja na vipengee vya `input` au `select` ambapo bofya au tabo kwenye kipengee ili kukiweka makini.

Hapa kuna mfano unaotumia darasa la bandia la `:focus`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:focus {
  outline: 2px solid orange;
}
```

:::

- **`:active pseudo-class`**: Darasa la bandia hili hutumika kupamba kipengee kilichowashwa na mtumizi. Mfano wa kawaida ni pale mtumizi anapobofya kitufe.

Hapa kuna mfano unaotumia darasa la bandia la `:active`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Click me!</a>
```

```css
a:active {
  color: pink;
}
```

## Kufanya kazi na mandhari na mipaka

- **Sifa ya `background-size`**: Sifa hii hutumika kuweka ukubwa wa mandhari kwa kipengee. Thamani za kawaida ni pamoja na `cover` kwa picha ya mandhari kufunika kipengee kizima na `contain` kwa picha ya mandhari kufaa ndani ya kipengee.

Hapa kuna mfano unaotumia `background-size: cover`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

Hapa kuna mfano unaotumia `background-size: contain`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **Sifa ya `background-repeat`**: Sifa hii hutumika kuamua jinsi picha za mandhari zinavyorudiwa kando ya mhimili wa usawa na wima. Thamani ya msingi ya `background-repeat` ni `repeat` ikimaanisha picha itarudiwa kwa usawa na wima. Pia unaweza kubainisha kutorudiwa kwa picha kwa kutumia sifa ya `no-repeat`.

Hapa kuna mfano unaotumia `background-repeat: no-repeat`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **Sifa ya `background-position`**: Sifa hii hutumika kubainisha nafasi ya picha ya mandhari. Inaweza kuwekwa kwa urefu maalum, asilimia, au maneno kama `top`, `bottom`, `left`, `right`, na `center`.

Hapa kuna mfano unaotumia `background-position`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

- **Sifa ya `background-attachment`**: Sifa hii hutumika kubainisha kama picha ya mandhari itasogea pamoja na maudhui au itabaki mahali pake. Thamani kuu ni `scroll` (ya msingi), ambapo picha ya mandhari husogea na maudhui, na `fixed`, ambapo picha ya mandhari inabaki mahali pake kwenye skrini.
- **Sifa ya `background-image`**: Sifa hii hutumika kuweka picha ya mandhari ya kipengee. Unaweza kuweka picha nyingi za mandhari kwa wakati mmoja na kutumia kitendakazi cha `url`, `radial-gradient` au `linear-gradient` kama thamani.

Hapa kuna mfano unaotumia `background-image`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **Sifa ya `background`**: Hii ni sifa ya ufupisho ya kuweka sifa zote za mandhari kwa tamko moja. Hapa kuna mfano wa kuweka picha ya mandhari na kuizuia kurudiwa: `background: no-repeat url("example-url-goes-here");`

Hapa kuna mfano unaotumia sifa ya ufupisho ya `background`:

:::interactive_editor

```html
<style>
  body {
    background: center top no-repeat url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **Utofauti mzuri kwa rangi za mandhari na mbele**: Ni muhimu kuhakikisha rangi za mandhari na mbele zina uwiano mzuri wa utofauti ili kufanya maandishi yasomeke. Miongozo ya Ufikikaji wa Maudhui ya Mtandao (WCAG) inapendekeza uwiano wa kiwango cha chini wa utofauti wa 4.5:1 kwa maandishi ya kawaida na 3:1 kwa maandishi makubwa.

## Mipaka

- **Sifa ya `border-top`**: Sifa hii hutumika kuweka mitindo ya mpaka wa juu wa kipengee. `border-top: 3px solid blue;` huweka mpaka wa bluu imara wenye upana wa pikseli 3 upande wa juu wa kipengee.
- **Sifa ya `border-right`**: Sifa hii hutumika kuweka mitindo ya mpaka wa kulia wa kipengee. `border-right: 2px solid red;` huweka mpaka wa nyekundu imara wenye upana wa pikseli 2 upande wa kulia wa kipengee.
- **Sifa ya `border-bottom`**: Sifa hii hutumika kuweka mitindo ya mpaka wa chini wa kipengee. `border-bottom: 1px dashed green;` huweka mpaka wa kijani wenye mistari ya mviringo wenye upana wa pikseli 1 upande wa chini wa kipengee.
- **Sifa ya `border-left`**: Sifa hii hutumika kuweka mitindo ya mpaka wa kushoto wa kipengee. `border-left: 4px dotted orange;` huweka mpaka wa machungwa wenye nukta wenye upana wa pikseli 4 upande wa kushoto wa kipengee.

Hapa kuna mfano unaotumia sifa za mipaka binafsi:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="bordered-box">Box with different borders</div>
```

```css
.bordered-box {
  border-top: 3px solid blue;
  border-right: 2px solid red;
  border-bottom: 1px dashed green;
  border-left: 4px dotted orange;
  padding: 20px;
}
```

:::

- **Sifa ya `border`**: Hii ni sifa ya ufupisho ya kuweka upana, mtindo, na rangi ya mpaka wa kipengee. `border: 1px solid black;` huweka mpaka mweusi imara wenye upana wa pikseli 1.

Hapa kuna mfano unaotumia sifa ya ufupisho ya `border`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

- **Sifa ya `border-radius`**: Sifa hii hutumika kuunda pembe za mviringo kwa mpaka wa kipengee.

Hapa kuna mfano unaotumia `border-radius`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

- **Sifa ya `border-style`**: Sifa hii hutumika kuweka mtindo wa mpaka wa kipengee. Thamani zinazokubalika ni pamoja na `solid`, `dashed`, `dotted`, na `double`.

Hapa kuna mfano unaotumia thamani tofauti za `border-style`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="solid-border">Solid border</div>
<div class="dashed-border">Dashed border</div>
<div class="dotted-border">Dotted border</div>
<div class="double-border">Double border</div>
```

```css
.solid-border {
  border: 3px solid blue;
  margin-bottom: 10px;
  padding: 10px;
}

.dashed-border {
  border: 3px dashed red;
  margin-bottom: 10px;
  padding: 10px;
}

.dotted-border {
  border: 3px dotted green;
  padding: 10px;
}

.double-border {
  border: 3px double blueviolet;
  padding: 10px;
}
```

:::

## Mwinuko wa rangi

- **Kitendakazi cha `linear-gradient()`**: Kitendakazi hiki cha CSS hutumika kuunda mpito kati ya rangi nyingi kando ya mstari wa moja kwa moja.

Hapa kuna mfano unaotumia `linear-gradient()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

- **Kitendakazi cha `radial-gradient()`**: Kitendakazi hiki cha CSS huunda picha inayotoka kwenye sehemu fulani, kama mduara au mviringo, na polepole hupita kati ya rangi nyingi.

Hapa kuna mfano unaotumia `radial-gradient()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```
