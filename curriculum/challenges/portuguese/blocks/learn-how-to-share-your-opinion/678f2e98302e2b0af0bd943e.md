# --explanation--

`Guess` é usado quando você está incerto sobre algo, mas ainda assim forma uma opinião ou faz uma sugestão. Por exemplo:

`I guess we could try a different approach.` - Isso significa que você não tem certeza completa, mas acha que tentar uma abordagem diferente é uma boa ideia.

`Consider` significa pensar cuidadosamente sobre algo, frequentemente antes de tomar uma decisão. Por exemplo:

`We need to consider all the options before choosing a solution.` - Isso significa que você precisa pensar cuidadosamente sobre todas as soluções possíveis antes de fazer uma escolha final.
