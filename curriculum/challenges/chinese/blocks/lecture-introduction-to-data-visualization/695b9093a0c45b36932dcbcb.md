# --description--

在之前的课程中，你已经了解了数据可视化，这是创建清晰、直观格式以表现复杂数据的过程。在本课中，你将学习 D3 库。

D3 库是一个免费且开源的库，你可以用它在 JavaScript 中创建引人注目且有效的数据可视化。

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-1.png" alt="美国县级数据的绿色分级地图，颜色范围从浅绿色（3%）到深绿色（60%+）">


D3 这个名字是 data-driven documents（数据驱动文档）的缩略语，指的是它如何动态地将数据绑定到文档对象模型（DOM）中的元素。这个库非常灵活，自然适用于网页。你无需学习全新的图形表现形式，就可以用 `svg` 和 `canvas` 元素构建数据可视化。

例如，如果你想用 D3 创建一个条形图，你可以在一个 JS 文件中完成以下操作：

* 声明表示图表维度和边距的变量
    
* 声明水平位置比例尺
    
* 声明垂直位置比例尺
    
* 创建包含条形和标签的 SVG 容器
    
<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-2.png" alt="标题为美国 GDP 的条形图，显示 1947 年到 2015 年的国内生产总值，数值从接近零上升到约 18,000">

那么，为什么要学习 D3，而不是其他可视化工具如 Chart.js、Plotly 或 Tableau 呢？

与其他工具不同，D3 为你提供了大量底层控制，你可以用纯 HTML、CSS 和 JavaScript 构建可自定义且交互式的可视化。此外，这个库是用 JavaScript 编写的，所以你不必担心学习全新的语言。

在接下来的工作坊和实验中，你将学习 D3 库的基础知识，比如选择、形状和比例尺。你将通过构建散点图、条形图、仪表盘应用等来学习这些关键概念。
