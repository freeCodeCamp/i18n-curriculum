# --description--

`Patterns` は、何かが起こるまたは行われる規則的で繰り返される方法です。データでは、パターンが傾向や行動を示すことがあります。

例えば、顧客データの `pattern` は、より多くの人がアクセサリーとしてリボンを使うことかもしれません。これは `trend pattern` です。

また、コーヒーショップが朝により多くのコーヒーを販売するような `cycle patterns` もあります。

`Patterns` は、毎月売上が増加するような数値の中や、顧客がオンラインで買い物を好むような行動の中に見つけられます。
