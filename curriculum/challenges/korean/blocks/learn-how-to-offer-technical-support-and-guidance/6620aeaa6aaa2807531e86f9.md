# --description--

동사 `encounter`는 특히 예상치 못한 무언가를 만나거나 마주친다는 뜻입니다. 프로그래밍에서 `encountering` 문제나 버그를 만난다는 것은 코드가 실행되는 동안 문제에 직면한다는 의미입니다.

예를 들어, `You might encounter an error if the user inputs the wrong type of data.` 이 문장은 예상치 못하게 오류를 마주치거나 처리해야 할 수도 있다는 뜻입니다.
