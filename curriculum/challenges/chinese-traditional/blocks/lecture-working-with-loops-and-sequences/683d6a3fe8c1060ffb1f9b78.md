# --description--

在上一課中，你學會了如何使用 tuple 資料型別。在本課中，你將學習一些在操作 tuple 時會使用的常見方法。

我們將涵蓋的第一個方法是 `count()`。此方法用於判斷一個項目在 tuple 中出現了多少次。以下是一個範例，檢查字串 `Rust` 在名為 `programming_languages` 的 tuple 中出現了多少次：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

由於 `Rust` 在元組中出現了兩次，`count()` 方法傳回的數字是 2。如果在元組中完全沒有指定的項目，則 `count()` 函式的返回值是 0：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

如果沒有引數傳入 `count()` 函式，Python 就會引發 `TypeError`：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count()

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: tuple.count() takes exactly one argument (0 given)
"""
```

接下來我們將查看的行為是 `index()` 行為。此行為用於尋找特定項目在元組中的索引。以下是使用 `index()` 行為來尋找字串 `Java` 索引的範例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

如果找不到指定的項目，則 Python 會引發 `ValueError`：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: tuple.index(x): x not in tuple
"""
```

你還可以為 `index()` 方法傳入選用的起始和結束索引引數。以下是傳入選用起始索引的範例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

在這個範例中，我們指定從哪裡開始搜尋字串 `Python`。透過將數字 `3` 作為第二個引數傳入 `index()` 函式，我們指定從索引 `3` 開始搜尋。由於 `Python` 在元組中出現兩次，`index()` 函式會因為使用了選用的起始索引引數而傳回索引 `5`，而非索引 `2`。

你也可以傳入一個選用的停止索引。以下是指定起始和停止索引的修改範例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

現在結果是索引 `2`，因為我們從索引 `2` 開始搜尋，並搜尋到但不包含索引 `5`。

另一個常用於元組的函式是 `sorted()` 函式。在之前的課程中，你已經學過用於列表的 `sort()` 方法。事實上，`sorted()` 函式可以用於任何可迭代物件，包括元組。

以下是使用 `sorted()` 函式（程式）建立新的數字列表的範例：

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

`sorted()` 函式（程式）將永遠創建一個排序後值的新列表。這與 `sort()` 方法不同，後者會就地排序列表的元素，且不會傳回新的列表。

如果你需要自訂可迭代物件的排序行為，可以使用選用的 `reverse` 和 `key` 引數。以下是使用 `key` 引數依長度排序元組中項目的範例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

如果你想要創建一個反向排序的新值列表，則可以像這樣使用 `reverse` 引數：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

Tuple 是 Python 中常見的資料型別。了解如何操作它們，以及一些有用的方法和函式，將幫助你撰寫更高效的程式碼。
