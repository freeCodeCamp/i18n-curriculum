## Farbtheorie

- **Definition der Farbtheorie**: Dies ist die Untersuchung, wie Farben miteinander interagieren und wie sie unsere Wahrnehmung beeinflussen. Sie umfasst Farbbeziehungen, Farbharmonie und die psychologische Wirkung von Farben.
- **Primärfarben**: Diese Farben, nämlich Gelb, Blau und Rot, sind die grundlegenden Farbtöne, aus denen alle anderen Farben abgeleitet werden.
- **Sekundärfarben**: Diese Farben entstehen durch Mischen gleicher Mengen von zwei Primärfarben. Grün, Orange und Lila sind Beispiele für Sekundärfarben.
- **Tertiärfarben**: Diese Farben entstehen durch die Kombination einer Primärfarbe mit einer benachbarten Sekundärfarbe. Gelb-Grün, Blau-Grün und Blau-Violett sind Beispiele für Tertiärfarben.
- **Warme Farben**: Diese Farben, zu denen Rot, Orange und Gelb gehören, rufen Gefühle von Komfort, Wärme und Gemütlichkeit hervor.
- **Kühle Farben**: Diese Farben, zu denen Blau, Grün und Violett gehören, vermitteln Gefühle von Ruhe, Gelassenheit und Professionalität.
- **Farbrad**: Das Farbrad ist ein kreisförmiges Diagramm, das zeigt, wie Farben zueinander in Beziehung stehen. Es ist ein wichtiges Werkzeug für Designer, da es ihnen hilft, Farbkombinationen auszuwählen.
- **Analoge Farbschemata**: Diese Farbschemata erzeugen ein stimmiges und beruhigendes Erlebnis. Sie bestehen aus analogen Farben, die im Farbrad nebeneinanderliegen.
- **Komplementäre Farbschemata**: Diese Farbschemata erzeugen hohen Kontrast und visuelle Wirkung. Ihre Farben liegen sich im Farbrad gegenüber.
- **Triadisches Farbschema**: Dieses Farbschema enthält lebendige Farben. Sie bestehen aus Farben, die ungefähr gleich weit voneinander entfernt sind. Wenn man sie verbindet, bilden sie ein gleichseitiges Dreieck im Farbrad.
- **Monochromatisches Farbschema**: Bei diesem Farbschema stammen alle Farben von derselben Grundfarbe ab, indem deren Helligkeit, Dunkelheit und Sättigung angepasst werden. Dies erzeugt ein Gefühl von Einheit und Harmonie und schafft dennoch Kontrast.

## Verschiedene Möglichkeiten, mit Farben in CSS zu arbeiten

- **Benannte Farben**: Diese Farben sind vordefinierte Farbnamen, die von Browsern erkannt werden. Beispiele sind `blue`, `darkred`, `lightgreen`.
- **`rgb()`-Funktion**: RGB steht für Rot, Grün und Blau – die Primärfarben des Lichts. Diese drei Farben werden in unterschiedlichen Intensitäten kombiniert, um eine große Farbpalette zu erzeugen. Die `rgb()`-Funktion ermöglicht es Ihnen, Farben mit dem RGB-Farbmodell zu definieren.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>RGB Color</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

- **`rgba()`-Funktion**: Diese Funktion fügt einen vierten Wert, Alpha, hinzu, der die Transparenz der Farbe steuert. Wenn kein Wert angegeben wird, ist der Alpha-Wert standardmäßig `1`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>RGBA Background</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

- **`hsl()`-Funktion**: HSL steht für Farbton, Sättigung und Helligkeit – drei wichtige Komponenten, die eine Farbe definieren.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>HSL Color</p>
```

```css
p {
  color: hsl(120, 100%, 50%);
}
```

:::

- **`hsla()`-Funktion**: Diese Funktion fügt einen vierten Wert, Alpha, hinzu, der die Deckkraft der Farbe steuert.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>HSLA Background</div>
```


```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

- **Hexadezimal**: Ein Hex-Code (Kurzform für hexadezimaler Code) ist ein sechsstelliger String, der Farben im RGB-Farbmodell darstellt. „Hex“ bezieht sich auf das Zahlensystem zur Basis 16, das die Ziffern 0 bis 9 und die Buchstaben A bis F verwendet.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1 class="hex-text">Hex Text</h1>
<p class="hex-bg">Hex Background</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}

p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

## Die `box-shadow`-Eigenschaft

- **Definition**: Die `box-shadow`-Eigenschaft wendet einen oder mehrere Schatten um ein Element herum an.
- **Versatzwerte**: Sie müssen horizontale (`offset-x`) und vertikale (`offset-y`) Werte angeben. Positive `offset-x` verschieben den Schatten nach rechts, während negative Werte ihn nach links verschieben. Positive `offset-y` verschieben den Schatten nach unten, während negative Werte ihn nach oben verschieben. Wenn ein Wert `0` ist, müssen Sie keine Einheit angeben.
- **Unschärferadius**: Dieser optionale Wert steuert, wie unscharf der Schatten erscheint. Wenn er nicht angegeben wird, ist der Standardwert `0`, der scharfe Kanten erzeugt. Je höher der Wert, desto weicher der Schatten.
- **Ausbreitungsradius**: Dieser optionale Wert steuert, wie stark der Schatten sich ausdehnt oder zusammenzieht. Wenn er nicht angegeben wird, ist der Standardwert `0`.
- **Schattenfarbe**: Sie können die Farbe mit benannten Farben, Hex-Werten, `rgb()`, `rgba()`, `hsl()` oder `hsla()`-Funktionen angeben.
- **`inset`-Schlüsselwort**: Das Hinzufügen des `inset`-Schlüsselworts platziert den Schatten innerhalb des Elements statt außerhalb.
- **Mehrere Box-Schatten anwenden**: Sie können mehrere Schatten anwenden, indem Sie sie durch Kommas trennen. Die Schatten werden von vorne nach hinten geschichtet.

### Syntax

```css
box-shadow: offset-x offset-y blur-radius spread-radius color;
```

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="shadow-box">Shadow Color Example</div>
```

```css
.shadow-box {
  width: 200px;
  padding: 20px;
  background-color: lightblue;
  box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}
```

:::

## Lineare und radiale Verläufe

- **Lineare Verläufe**: Diese Verläufe erzeugen einen allmählichen Übergang zwischen Farben entlang einer geraden Linie. Sie können die Richtung dieser Linie mit Schlüsselwörtern wie `to top`, `to right`, `to bottom right` oder Winkeln wie `45deg` steuern. Sie können jede gültige CSS-Farbe und beliebig viele Farb-Stopps verwenden.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient">Linear Gradient</div>
```

```css
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

:::

- **Radiale Verläufe**: Diese Verläufe erzeugen kreisförmige oder elliptische Verläufe, die von einem zentralen Punkt ausgehen.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient">Radial Gradient</div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::
