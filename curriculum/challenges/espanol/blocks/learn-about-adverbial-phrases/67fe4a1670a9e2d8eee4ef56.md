# --explanation--

`First off` se usa para introducir la primera cosa que se discutirá o hará. Señala el comienzo de una lista o el orden de cosas en las que enfocarse. Por ejemplo:

- `First off, we need to address the project's scope.` – Esto significa que lo primero que se hará es abordar el alcance del proyecto.

- `First off, let's review the current goals before we start planning.` – Esto significa que revisar los objetivos actuales es lo primero que sucederá antes de pasar a la planificación.
