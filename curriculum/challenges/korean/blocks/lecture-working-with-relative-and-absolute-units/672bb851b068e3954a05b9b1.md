# --interactive--

`calc()` 함수를 사용하면 스타일시트 내에서 직접 계산을 수행하여 속성 값을 동적으로 결정할 수 있습니다. 이는 뷰포트 크기나 다른 요소를 기준으로 치수를 계산하여 유연하고 반응형 사용자 인터페이스를 만들 수 있다는 뜻입니다.

`calc()`은 CSS 함수입니다. JavaScript를 배우기 시작할 때 함수에 대해 더 많이 배우겠지만, 이번 수업에서는 `calc()`가 어떻게 작동하는지 이해하는 데 필요한 기본 사항을 배울 것입니다.

함수는 특정 작업을 수행하는 코드 블록입니다. CSS에는 이미 정의된 함수가 있어, 이를 직접 사용하고 작업 수행 방식을 사용자 지정하기 위해 필요한 값을 전달할 수 있습니다.

프로그래밍 세계에서는 함수가 수행하는 작업을 실행할 때 그 함수를 "호출"한다고 합니다. 함수에 전달하는 값들은 인수라고 부릅니다.

아래 코드에서 볼 수 있듯이, 함수를 호출하려면 함수 이름 뒤에 소괄호 안에 인수를 쉼표로 구분하여 작성합니다. 함수 이름과 여는 소괄호 사이에는 공백이 없어야 합니다:

```css
function(argument1, argument2, argument3)
```

함수는 작업을 수행하는 데 하나의 값만 필요할 수도 있습니다. 이 경우 인수도 하나만 받습니다. `calc()` 함수가 바로 그런 경우입니다. 무엇을 계산할지 알아야 하므로 하나의 인수를 받습니다.

이를 위해 인수로 표현식이라는 것을 전달합니다. 표현식은 결과를 만들어내는 값과 연산자의 조합입니다.

다음은 `calc()` 함수를 호출하는 방법입니다. calc라는 이름을 쓰고 소괄호를 붙인 뒤, 소괄호 안에 표현식을 작성합니다:

```css
calc(expression)
```

표현식은 최종 결과를 계산하기 위해 평가됩니다. "평가된다"는 것은 값과 연산자가 내부적으로 하나의 값으로 변환된다는 뜻입니다. 그 결과가 계산이 이루어진 CSS 속성에 할당됩니다.

길이, 각도, 시간, 백분율, 숫자, 색상 값을 나타내는 값에 대해 계산할 수 있습니다. 픽셀, 백분율, em 같은 서로 다른 단위를 결합할 수도 있습니다.

숫자의 경우, 표현식 내 모든 값(피연산자라고도 함)은 `px`, `em`, 백분율(`%`) 같은 해당 단위를 가져야 합니다. 연산자에 따라 서로 다른 피연산자가 서로 다른 단위를 가질 수 있습니다.

표현식에서는 덧셈(`+`), 뺄셈(`-`), 곱셈(`*`), 나눗셈(`/`) 연산자를 사용할 수 있습니다.

피연산자와 연산자가 여러 개일 경우, `calc()`는 표준 연산자 우선순위 규칙을 따릅니다. 필요하면 소괄호를 추가해 연산 순서를 지정할 수도 있습니다.

아래 예제에서는 `div`에 `Hello, World!`라는 텍스트가 있습니다.

`div`를 선택하는 CSS 타입 선택자를 사용해 흰색 텍스트와 진한 파란색 배경으로 스타일링할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>Hello, World!</div>
```

```css
div {
  color: white;
  background-color: #1b1b32;
  width: calc(50% - 20px);
}
```

:::

여기서 새로워진 점은 너비가 동적으로 계산된다는 것입니다. `calc()` 함수를 호출하고 표현식을 인수로 전달하는 부분을 주목하세요. 표현식에는 서로 다른 단위를 가진 두 피연산자와 하나의 연산자인 뺄셈 연산자가 있습니다.

백분율은 상대 단위입니다. 값(`50%`)은 부모 컨테이너의 너비에 의해 결정됩니다. 그런 다음 `20px`이 그 값에서 뺍니다. 이 표현식의 결과가 `div`의 너비를 결정합니다.

`div`의 너비는 컨테이너 전체 너비의 약 절반이며, 부모 컨테이너 크기를 조절하면 너비가 자동으로 다시 계산됩니다.

이것이 `calc()`를 사용하는 핵심 장점입니다. 애플리케이션이나 뷰포트의 다양한 측면을 기준으로 CSS 속성 값을 동적으로 결정할 수 있습니다.

표현식에는 계산에 필요하다면 CSS 함수와 변수도 포함할 수 있습니다. CSS 변수에 대해서는 다음 수업에서 더 배우게 됩니다.

좋습니다. 이제 `calc()` 함수의 기본을 알았으니, 몇 가지 모범 사례를 살펴보겠습니다.

먼저, 덧셈(`+`)과 뺄셈(`-`) 연산자 주위에는 반드시 공백을 넣어야 합니다.

예를 들어, 아래 표현식은 뺄셈 연산자가 두 번째 피연산자 바로 앞에 붙어 있어 유효하지 않습니다.

```css
calc(100% -30px)
```

뺄셈(`-`) 연산자는 다음과 같이 공백으로 둘러싸여야 합니다. 공백을 추가하면 유효한 표현식이 됩니다.

```css
calc(100% - 30px)
```

곱셈과 나눗셈 연산자에는 필수는 아니지만 권장됩니다.

필요하다면 `calc()` 함수 호출을 중첩하여 계산을 수행하고 그 결과를 다른 계산에 사용할 수도 있습니다.

또한, `calc()` 함수에 전달하는 표현식에서 길이를 나타내는 값으로 0을 사용할 때는 단위를 반드시 포함해야 합니다. 예를 들어, 다음 표현식은 유효하지 않습니다:

```css
calc(100% - 0)
```

단위를 px처럼 추가해야 합니다.

```css
calc(100% - 0px)
```

곱셈이나 나눗셈 연산자를 사용할 때는 현재 피연산자 중 하나가 단위가 없어야 한다는 점도 알아두세요. 특히 나눗셈 연산자에서는 오른쪽 피연산자가 단위가 없어야 합니다. 아래는 두 피연산자 모두 단위를 가지고 있어 유효하지 않은 표현식입니다. 5 또는 50 중 하나는 단위가 없어야 합니다:

```css
calc(5px * 50px)
```

둘 중 하나에서 단위를 생략해야 합니다. 다음 두 대안은 모두 유효합니다:

```css
calc(5 * 50px)
calc(5px * 50)
```

나눗셈 연산자 예시입니다. 두 피연산자 모두 단위를 가지고 있어 유효하지 않습니다:

```css
calc(50% / 5%)
```

나눗셈 연산자에서는 오른쪽 피연산자에서 단위를 제거해야 합니다:

```css
calc(50% / 5)
```

`calc()` 함수는 웹 개발자에게 매우 유용할 수 있습니다. 이 함수를 사용하면 속성 값을 동적으로 설정해 유연하고 반응형 디자인을 만들 수 있습니다.
