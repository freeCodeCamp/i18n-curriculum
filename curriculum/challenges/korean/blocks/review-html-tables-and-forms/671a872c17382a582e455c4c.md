## HTML 폼 요소와 속성

- **`form` 요소**: 사용자 입력을 위한 HTML 폼을 만들 때 사용합니다.
- **`action` 속성**: 폼 데이터를 전송할 URL을 지정할 때 사용합니다.
- **`method` 속성**: 폼 데이터를 전송할 때 사용할 HTTP 메서드를 지정할 때 사용합니다. 가장 일반적인 메서드는 `GET`와 `POST`입니다.

```html
<form method="value-goes-here" action="url-goes-here">
  <!-- inputs go inside here -->
</form>
```

- **`input` 요소**: 사용자 입력을 위한 입력 필드를 만들 때 사용합니다.
- **`type` 속성**: 입력 필드의 유형을 지정할 때 사용합니다. 예: `text`, `email`, `number`, `radio`, `checkbox` 등.
- **`placeholder` 속성**: 입력 필드에 무엇을 입력해야 하는지 사용자에게 힌트를 보여줄 때 사용합니다.
- **`value` 속성**: 입력 값의 값을 지정할 때 사용합니다. 입력 유형이 `button`인 경우, `value` 속성으로 버튼 텍스트를 설정할 수 있습니다.
- **`name` 속성**: 입력 필드에 이름을 할당할 때 사용하며, 폼 데이터가 제출될 때 키 역할을 합니다. 라디오 버튼의 경우, 같은 `name`을 주면 그룹으로 묶여 한 번에 하나의 옵션만 선택할 수 있습니다.
- **`size` 속성**: 사용자가 입력할 때 보이는 문자 수를 정의할 때 사용합니다.
- **`min` 속성**: `number` 같은 입력 유형과 함께 최솟값을 지정할 때 사용할 수 있습니다.
- **`max` 속성**: `number` 같은 입력 유형과 함께 최댓값을 지정할 때 사용할 수 있습니다.
- **`minlength` 속성**: 입력 필드에 필요한 최소 문자 수를 지정할 때 사용합니다.
- **`maxlength` 속성**: 입력 필드에 허용되는 최대 문자 수를 지정할 때 사용합니다.
- **`required` 속성**: 폼 제출 전에 입력 필드를 반드시 채워야 한다고 지정할 때 사용합니다.
- **`disabled` 속성**: 입력 필드를 비활성화할 때 사용합니다.
- **`readonly` 속성**: 입력 필드를 읽기 전용으로 지정할 때 사용합니다.

:::interactive_editor

```html
<!-- Text input -->
<input 
  type="text"
  id="name"
  name="name"
  placeholder="e.g. Quincy Larson" 
  size="20"
  minlength="5"
  maxlength="30"
  required
/>

<!-- Number input -->
<input 
  type="number"
  id="quantity"
  name="quantity"
  min="2"
  max="10"
  disabled
/>

<!-- Button -->
<input type="button" value="Show Alert" />
```

:::

- **`label` 요소**: 입력 필드에 대한 레이블을 만들 때 사용합니다.
- **`for` 속성**: 레이블이 어떤 입력 필드에 해당하는지 지정할 때 사용합니다.
- **암묵적 폼 연결**: 입력 필드를 `label` 요소 안에 감싸면 레이블과 연결할 수 있습니다.

:::interactive_editor

```html
<form action="">
  <label>
    Full Name:
    <input type="text" />
  </label>
</form>
```

:::

- **명시적 폼 연결**: `for` 요소에 `label` 속성을 사용해 입력 필드와 레이블을 연결할 수 있습니다.

:::interactive_editor

```html
<form action="">
  <label for="email">Email Address: </label>
  <input type="email" id="email" />
</form>
```

:::

- **`button` 요소**: 클릭 가능한 버튼을 만들 때 사용합니다. 버튼에는 `type` 속성을 가질 수 있으며, 버튼이 활성화될 때 동작을 제어하는 데 사용합니다. 예: `submit`, `reset`, `button`.

:::interactive_editor

```html
<button type="button">Show Form</button>
<button type="submit">Submit Form</button>
<button type="reset">Reset Form</button>
```

:::

- **`fieldset` 요소**: 관련된 입력들을 그룹화할 때 사용합니다.
- **`legend` 요소**: 입력 그룹을 설명하는 캡션을 추가할 때 사용합니다.

:::interactive_editor

```html
<!-- Radio group -->
<fieldset>
  <legend>Was this your first time at our hotel?</legend>

  <label for="yes-option">Yes</label>
  <input id="yes-option" type="radio" name="hotel-stay" value="yes" />

  <label for="no-option">No</label>
  <input id="no-option" type="radio" name="hotel-stay" value="no" />
</fieldset>

<!-- Checkbox group -->
<fieldset>
  <legend>
    Why did you choose to stay at our hotel? (Check all that apply)
  </legend>

  <label for="location">Location</label>
  <input type="checkbox" id="location" name="location" value="location" />

  <label for="price">Price</label>
  <input type="checkbox" id="price" name="price" value="price" />
</fieldset>
```

:::

- **포커스 상태**: 사용자가 선택한 입력 필드의 상태입니다.

## HTML 표 요소와 속성 다루기

- **표 요소**: HTML 표를 만들 때 사용합니다.
- **표 헤드 (`thead`) 요소**: HTML 표에서 헤더 내용을 그룹화할 때 사용합니다.
- **표 행 (`tr`) 요소**: HTML 표에서 행을 만들 때 사용합니다.
- **표 헤더 (`th`) 요소**: HTML 표에서 헤더 셀을 만들 때 사용합니다.
- **표 본문 (`tbody`) 요소**: HTML 표에서 본문 내용을 그룹화할 때 사용합니다.
- **표 데이터 셀 (`td`) 요소**: HTML 표에서 데이터 셀을 만들 때 사용합니다.
- **표 푸터 (`tfoot`) 요소**: HTML 표에서 푸터 내용을 그룹화할 때 사용합니다.
- **`caption` 요소**: HTML 표의 제목을 추가할 때 사용합니다.
- **`colspan` 속성**: 표 셀이 차지할 열 수를 지정할 때 사용합니다.

:::interactive_editor

```html
<table>
  <caption>Exam Grades</caption>

  <thead>
    <tr>
      <th>Last Name</th>
      <th>First Name</th>
      <th>Grade</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>Davis</td>
      <td>Alex</td>
      <td>54</td>
    </tr>

    <tr>
      <td>Doe</td>
      <td>Samantha</td>
      <td>92</td>
    </tr>

    <tr>
      <td>Rodriguez</td>
      <td>Marcus</td>
      <td>88</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <td colspan="2">Average Grade</td>
      <td>78</td>
    </tr>
  </tfoot>
</table>
```

:::

## HTML 도구 다루기

- **HTML 검증기**: HTML 코드의 구문이 올바른지 확인하는 도구입니다.
- **DOM 검사기**: 웹 페이지의 HTML 구조를 검사하고 수정할 수 있는 도구입니다.
- **개발자 도구**: 브라우저에 내장된 웹 개발자 도구 모음으로, 웹 페이지를 디버그하고 프로파일링하며 분석하는 데 도움을 줍니다.
