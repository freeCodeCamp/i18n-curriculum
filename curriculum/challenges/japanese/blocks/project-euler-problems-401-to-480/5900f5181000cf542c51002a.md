# --description--

整数のシーケンス $S = \\{s_i\\}$ は、$n$ 個の要素を持ち、各要素 $s_i$ が $1 ≤ s_i ≤ n$ を満たす場合、$n$-列と呼ばれます。したがって、異なる $n$-列は合計で $n^n$ 個あります。

例えば、シーケンス $S = \\{1, 5, 5, 10, 7, 7, 7, 2, 3, 7\\}$ は 10-列です。

任意のシーケンス $S$ に対して、$L(S)$ を $S$ の中で同じ値が連続する最長の部分列の長さとします。例えば、上記のシーケンス $S$ では、7 が3回連続しているため、$L(S) = 3$ です。

$f(n) = \sum L(S)$ をすべての $n$-列 $S$ についての和とします。

例えば、$f(3) = 45$、$f(7) = 1\,403\,689$、$f(11) = 481\,496\,895\,121$ です。

$f(7\,500\,000)\bmod 1\,000\,000\,009$ を求めてください。
