# --description--

次の方程式において、$x$、$y$、$n$ は正の整数です。

$$\frac{1}{x} + \frac{1}{y} = \frac{1}{n}$$

ある上限 $L$ に対して、$F(L)$ を $x < y ≤ L$ を満たす解の個数と定義します。

$F(15) = 4$ および $F(1000) = 1069$ であることが確認できます。

$F({10}^{12})$ を求めてください。
