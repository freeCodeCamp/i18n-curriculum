# --description--

Il `CSS` in linea permette di aggiungere stili direttamente agli elementi `HTML`, anche se questo metodo non è molto consigliato:

```html
<body>
  <div style="color: white; background-color: black;">...</div>
</body>
```

La prima cosa da notare è che qui non ci sono selettori, dato che gli stili vengono aggiunti direttamente al tag di apertura `<div>`. Poi c’è l’attributo `style`, con il suo valore racchiuso tra virgolette che rappresenta le dichiarazioni.

Se hai bisogno di aggiungere uno stile unico per un singolo elemento, questo metodo può andare bene. In generale, però, non è proprio un modo consigliato per aggiungere CSS all’HTML per alcune ragioni:

Può diventare rapidamente disordinato una volta che inizi ad aggiungere molte dichiarazioni a un singolo elemento, facendo diventare il file HTML inutilmente gonfio.
Se vuoi che molti elementi abbiano lo stesso stile, dovresti copiare e incollare lo stesso stile in ogni singolo elemento, causando molta ripetizione inutile e ancora più gonfiore.
Qualsiasi CSS in linea sovrascriverà gli altri due metodi, il che può causare risultati inattesi. (Anche se qui non lo approfondirai, questo può effettivamente essere sfruttato).
