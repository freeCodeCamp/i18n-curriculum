# --description--

Agora você sabe que os números de 11 a 19 são formados pelo dígito das unidades (1–9) após `十 (shí)` (10). A mesma lógica se aplica aos números de 21 a 29.

Em chinês, o número 20 é chamado `二十 (èr shí)`.

Para formar os números de 21 a 29, você simplesmente pega `二十 (èr shí)` (20) e adiciona o dígito das unidades (1–9) diretamente após ele.

| Número | Chinês |
| :--- | :--- |
| 20 | `二十 (èr shí)` |
| 21 | `二十一 (èr shí yī)` |
| 22 | `二十二 (èr shí èr)` |
| 23 | `二十三 (èr shí sān)` |
| 24 | `二十四 (èr shí sì)` |
| 25 | `二十五 (èr shí wǔ)` |
| 26 | `二十六 (èr shí liù)` |
| 27 | `二十七 (èr shí qī)` |
| 28 | `二十八 (èr shí bā)` |
| 29 | `二十九 (èr shí jiǔ)` |
