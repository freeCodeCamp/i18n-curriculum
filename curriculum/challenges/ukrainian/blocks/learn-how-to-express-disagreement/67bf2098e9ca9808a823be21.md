# --explanation--

`Do you remember` використовується, щоб перевірити, чи хтось пам’ятає минулу інформацію. Наприклад:

- `Do you remember where we saved that file?` — ви питаєте когось, чи пам’ятає він місце розташування файлу.

- `Do you remember when the issue first occurred?` — ви питаєте когось, чи пам’ятає він, коли почалася проблема.
