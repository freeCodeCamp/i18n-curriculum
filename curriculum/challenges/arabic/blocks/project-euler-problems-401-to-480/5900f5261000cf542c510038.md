# --description--

لعدد صحيح $M$، نُعرّف $R(M)$ كمجموع $\frac{1}{p·q}$ لكل أزواج الأعداد الصحيحة $p$ و $q$ التي تحقق كل هذه الشروط:

- $1 ≤ p &lt; q ≤ M$
- $p + q ≥ M$
- $p$ و $q$ عددان أوليان مشتركان.

نُعرّف أيضًا $S(N)$ كمجموع $R(i)$ لكل $2 ≤ i ≤ N$.

يمكننا التحقق من أن $S(2) = R(2) = \frac{1}{2}$، و $S(10) ≈ 6.9147$، و $S(100) ≈ 58.2962$.

احسب $S({10}^7)$. قدّم إجابتك مقربة لأربعة أرقام عشرية.
