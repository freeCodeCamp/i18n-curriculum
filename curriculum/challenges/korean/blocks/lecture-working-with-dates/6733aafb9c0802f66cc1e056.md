# --interactive--

JavaScript에서 날짜와 시간을 다루는 것은 항상 쉽지 않았습니다. 전문적인 코드베이스에서는 이러한 문제를 많이 해결한 라이브러리를 사용할 가능성이 큽니다.

하지만 때로는 JavaScript의 내장 `Date` 객체를 직접 다뤄야 할 때가 있습니다.

이번 수업에서는 `Date` 객체의 기본 개념과 JavaScript에서 날짜와 시간을 다룰 때 자주 사용하는 메서드 몇 가지를 살펴보겠습니다.

`Date` 객체는 JavaScript 내장 객체로, 날짜와 시간을 다룰 수 있게 해줍니다. 새로운 `Date` 객체를 생성하는 기본 구문은 다음과 같습니다:

```js
const now = new Date();
```

`new` 키워드는 `Date` 객체의 새 인스턴스를 생성하는 데 사용되며, `Date` 객체가 변수 `now`에 할당됩니다. 만약 `now`의 값을 콘솔에 출력하면, 코드를 실행하는 컴퓨터의 시스템 시계를 기준으로 현재 날짜와 시간이 표시됩니다.

콘솔에 `now`의 값을 출력했을 때 볼 수 있는 예시 응답은 다음과 같습니다:

:::interactive_editor

```js
const now = new Date();
console.log(now);
```

:::

시간은 24시간제 형식을 사용하므로 `14:30`은 `2:30 PM`입니다. `GMT-0700`는 시간대 오프셋이며, `Pacific Daylight Time`은 시간대 이름입니다.

또한 연도, 월, 일, 시, 분, 초, 밀리초 값을 인수로 제공하여 `Date` 객체에 특정 날짜와 시간을 전달할 수 있습니다.

특정 날짜와 시간으로 새 `Date` 객체를 생성하는 예시는 다음과 같습니다:

:::interactive_editor

```js
const specificDate = new Date("July 4, 1776 12:00:00");

console.log(specificDate);
```

:::

이는 현재 날짜와 시간 대신 특정 날짜와 시간을 다뤄야 할 때 유용합니다. 입력은 항상 문자열이어야 하며, `Date` 객체가 인식할 수 있는 형식이어야 합니다.

현재 날짜와 시간을 얻으려면 `Date.now()` 메서드를 사용할 수 있는데, 이 메서드는 `January 1, 1970, 00:00:00 UTC` 이후의 밀리초 수를 반환합니다. 이를 유닉스 에포크 시간이라고 합니다.

유닉스 에포크 시간은 정수로 쉽게 저장하고 조작할 수 있기 때문에 컴퓨터 시스템에서 날짜와 시간을 나타내는 일반적인 방법입니다. UTC는 세계가 시계와 시간을 조정하는 기본 시간 표준인 협정 세계시(Universal Time Coordinated)를 의미합니다.

현재 날짜를 기준으로 월의 일을 얻으려면 `getDate` 메서드를 사용할 수 있습니다. `getDate` 메서드를 사용하는 예시는 다음과 같습니다:

:::interactive_editor

```js
const now = new Date();
const date = now.getDate();
console.log(date); 
```

:::

이 예시에서는 `Date` 객체를 사용해 새 날짜 인스턴스를 생성하고 변수 `now`에 할당합니다. 그런 다음 `getDate` 객체에서 `now` 메서드를 호출해 월의 일을 얻고 변수 `date`에 할당합니다. 마지막으로 `date`의 값을 콘솔에 출력하면 현재 월의 일이 표시됩니다.

`getDate`은 월의 일에 따라 `1`부터 `31` 사이의 정수 값을 반환합니다. 날짜가 유효하지 않으면 `NaN` (숫자가 아님)을 반환합니다.

월을 얻으려면 다음과 같이 `getMonth` 메서드를 사용할 수 있습니다:

:::interactive_editor

```js
const now = new Date();
const month = now.getMonth();
console.log(month);
```

:::

월은 0부터 시작하므로 `January`는 `0`, `February`는 `1`이고 계속됩니다. 월이 유효하지 않으면 `NaN`을 반환합니다.

전체 연도를 얻으려면 다음과 같이 `getFullYear` 메서드를 사용할 수 있습니다:

:::interactive_editor

```js
const now = new Date();
const year = now.getFullYear();
console.log(year); 
```

:::

연도가 유효하지 않으면 `NaN`을 반환합니다.

`Date` 객체에는 `getHours`, `getMinutes`, `getSeconds` 등 더 많은 메서드가 있습니다. Mozilla 문서나 다른 자료를 통해 직접 더 탐색해 보시길 권합니다.
