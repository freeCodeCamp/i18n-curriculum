# --explanation--

Марія використовує слово `automate`, щоб запропонувати зробити процес автоматичним або керованим машинами чи системами замість людей. Це допомагає зменшити ймовірність людської помилки, оскільки машини виконують заздалегідь визначені інструкції без помилок, спричинених втомою чи недоглядом. Наприклад:

`We can automate the process of sending email updates.` - Ви можете використовувати програму для автоматичної відправки е-пошти замість ручного виконання.
