# --explanation--

الجملة الصحيحة تستخدم تركيب `First Conditional`، الذي يتكون من:

`if` + `Present Simple`: `If we lose critical data`.

فعل مساعد (`could`) + الفعل الأساسي: `it could set us back weeks`.


يُستخدم تركيب `First Conditional` لوصف حدث مستقبلي محتمل ونتيجته.
