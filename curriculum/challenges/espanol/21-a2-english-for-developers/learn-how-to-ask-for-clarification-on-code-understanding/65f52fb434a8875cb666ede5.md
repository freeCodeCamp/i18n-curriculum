# --description--

`Seem to be` se usa cuando no estás seguro sobre algo y quieres adivinar o sugerir una posibilidad. Por ejemplo, si dices `It seems to be working`, estás adivinando que algo está funcionando pero no estás 100% seguro.

Ejemplos:

Afirmativo: `It seems to be working fine.`

Negativo: `It doesn't seem to be a principal problema.`

Pregunta: `Does it seem to be related to the network?`
