# --description--

Para preguntar a alguien si quiere algo, el español usa el verbo `querer`.

La forma verbal `quieres` se usa con `tú` para preguntar sobre los deseos o preferencias de otra persona. Es común en la conversación diaria. Por ejemplo:

- `¿Quieres ayuda?` - ¿Quieres ayuda?

- `¿Quieres más tiempo?` - ¿Quieres más tiempo?
