# --interactive--

Unapofanya kazi na orodha zilizopangwa katika HTML, sifa ya kipengele `type` inakuwezesha kubainisha mtindo wa nambari unaotumika, kama vile nambari za tarakimu, herufi za alfabeti, au nambari za Kirumi. 

Sifa ya kipengele `type` inaweza kuwekwa kwenye kipengele cha `ol` (orodha iliyopangwa) kudhibiti muundo wa nambari:

- `1` kwa orodha za tarakimu (kama 1, 2, 3, ...),

- `A` kwa orodha za herufi kubwa za alfabeti (kama A, B, C, ...),

- `a` kwa orodha za herufi ndogo za alfabeti (kama a, b, c, ...),

- `I` kwa orodha za nambari kubwa za Kirumi (kama I, II, III, ...),

- `i` kwa orodha za nambari ndogo za Kirumi (kama i, ii, iii, ...).

Hapa ni jinsi unavyoweza kutumia kichaguzi cha sifa ya kipengele kulenga orodha iliyopangwa yenye sifa ya `type` maalum:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="A"] {
  color: purple;
  font-weight: bold;
}
```

:::

Katika mfano huu, sheria ya CSS itatumia rangi ya maandishi ya zambarau na fonti nzito kwa orodha yoyote iliyopangwa inayotumia nambari za herufi kubwa za alfabeti, ambayo imetajwa na `type="A"`.

Vivyo hivyo, unaweza kulenga orodha zilizopangwa zinazotumia nambari ndogo za Kirumi kwa kuchagua zile zilizo na `type="i"`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="i">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="i"] {
  color: green;
}
```

:::

Sheria hii itabadilisha rangi ya maandishi kuwa kijani kwa orodha yoyote iliyopangwa inayotumia nambari ndogo za Kirumi.

Uwezo wa kupangilia orodha zilizopangwa kulingana na sifa ya kipengele `type` unakupa udhibiti mkubwa juu ya muonekano wa orodha katika hati yako, ukiruhusu ubinafsishaji kulingana na jinsi maudhui yanavyopangwa.
