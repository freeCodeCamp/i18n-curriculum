# --description--

一组三个正整数（$a$，$b$，$c$）如果满足以下条件，则称为 Cardano 三元组：

$$\sqrt[3]{a + b \sqrt{c}} + \sqrt[3]{a - b \sqrt{c}} = 1$$

例如，（2,1,5）是Cardano Triplet。

存在 149 个满足 $a + b + c ≤ 1000$ 的 Cardano 三元组。

查找满足 $a + b + c ≤ 110\\,000\\,000$ 的 Cardano 三元组数量。
