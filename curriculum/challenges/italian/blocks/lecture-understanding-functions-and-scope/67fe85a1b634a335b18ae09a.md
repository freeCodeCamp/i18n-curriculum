# --description--

In Python, l'ambito determina il punto in cui puoi consultare una variabile. Controlla la durata di vita di una variabile e come viene risolta in diverse parti del codice.

Per determinare correttamente l'ambito, Python segue la regola **LEGB**, che sta per:

- **Ambito locale (L)**: Variabili definite in funzioni o classi.
    
- **Ambito di contenimento (E)**: Variabili definite in funzioni contenute o annidate.
    
- **Ambito globale (G):** Variabili definite al livello superiore del modulo o file.
    
- **Ambito integrato (B)**: Nomi riservati in Python per funzioni, moduli, parole chiave e oggetti predefiniti.

Python usa la regola LEGB per risolvere l'ambito delle variabili nel tuo programma. Esploreremo ciascuna di queste regole per farti capire meglio il processo.

**Ambito locale** significa che una variabile dichiarata all'interno di una funzione o classe può essere consultata solo all'interno di quella funzione o classe.

Ecco un esempio:

```python
def my_func():
    my_var = 10
    print(my_var)
```

In questo caso, la funzione `my_func` ha un proprio ambito che non può essere consultato dall'esterno della funzione. Chiamare `my_func` stamperà `10`, ma stampare `my_var` fuori dalla funzione causerà un `NameError`:

```python
def my_func():
    my_var = 10 # Locally scoped to my_func
    print(my_var)

my_func() # 10

print(my_var) # NameError: name 'my_var' is not defined
```

**Ambito di contenimento** significa che una funzione annidata dentro un'altra funzione può consultare le variabili della funzione in cui è annidata.

Per esempio:

```python
def outer_func():
    msg = 'Hello there!'

    def inner_func():
        print(msg)

    inner_func()

outer_func() # Hello there!
```

In questo esempio, la funzione interna `inner_func` può liberamente consultare la variabile `msg` definita nella funzione esterna `outer_func`. Tuttavia, nota che le funzioni esterne non possono consultare variabili definite all'interno di funzioni annidate:

```python
def outer_func():
    msg = 'Hello there!'
    print(res)

    def inner_func():
        res = 'How are you?'
        print(msg)

    inner_func()

outer_func() # NameError: name 'res' is not defined
```

Questo perché `res` ha ambito locale a `inner_func`. Inoltre, nota che `outer_func` prova a stampare `res` prima che `inner_func` venga chiamata.

Una soluzione è inizializzare `res` come stringa vuota nell'ambito di contenimento, cioè dentro `outer_func`. Poi, dentro `inner_func`, rendi `res` una variabile non locale con la parola chiave `nonlocal`:

```python
def outer_func():
    msg = 'Hello there!'
    res = ""  # Declare res in the enclosing scope

    def inner_func():
        nonlocal res  # Allow modification of an enclosing variable
        res = 'How are you?'
        print(msg)  # Accessing msg from outer_func()

    inner_func()
    print(res)  # Now res is accessible and modified

outer_func()

# Output:
# Hello there!
# How are you?
```

**Ambito globale** si riferisce a variabili dichiarate fuori da qualsiasi funzione o classe che possono essere consultate da qualsiasi punto del programma. Qui, `my_var` può essere consultata ovunque, anche dentro una funzione in cui non è definita:

```python
my_var = 100

def show_var():
    print(my_var)

show_var() # 100
print(my_var) # 100
```

E se vuoi rendere una variabile con ambito locale definita dentro una funzione accessibile globalmente, puoi usare la parola chiave `global`:

```python
my_var_1 = 7

def show_vars():
    global my_var_2
    my_var_2 = 10
    print(my_var_1)
    print(my_var_2)

show_vars() # 7 10

# my_var_2 is now a global variable and can be accessed anywhere in the program
print(my_var_2) # 10
```

Puoi anche usare la parola chiave `global` per modificare una variabile globale:

```python
my_var = 10  # A global variable

def change_var():
    global my_var  # Allows modification of a global variable
    my_var = 20

change_var()

print(my_var)  # my_var is now modified globally to 20
```

Infine, **l'ambito integrato** si riferisce a tutte le funzioni, moduli e parole chiave integrati di Python, disponibili ovunque nel tuo programma:

```python
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```
