# --interactive--

`grid-template-areas` プロパティを使うと、名前付きラベルを使って視覚的なグリッドレイアウトを設計できます。

その後、grid-area プロパティを使ってラベルを特定のグリッドアイテムに代入します。つまり、これらの名前付きラベルは「グリッドエリア名」とも呼ばれます。

`grid-template-areas` プロパティの基本的な構文は以下の通りです。

```css
grid-template-areas:
 'header header header'
 'left-sidebar main right-sidebar'
 'footer footer footer';
```

基本構文から押さえておくべきポイントは次の通りです：

- `header` や `main` のような値はグリッドエリアの名前です。

- 文字列内のスペースで区切られた各値は列に対応します。

- 各文字列はグリッドの行を表します。

この構文では、3×3 のグリッドコンテナを定義しています。

テンプレートを定義した後、`grid-area` プロパティと名前付きラベル（またはエリア）を値として使い、特定の要素がグリッドのそのエリアに属することをCSSに伝えます。

`grid-area` プロパティはグリッドアイテムを `grid-template-areas` で定義した名前付き領域に結びつけます。

`grid-template-areas` プロパティの機能を示す代表的な例として、クラシックなホーリーグレイルレイアウトの作成があります。

もし知らなければ、ホーリーグレイルレイアウトはヘッダー、フッター、2つのサイドバー、メインコンテンツエリアを持つウェブデザインパターンです。メインコンテンツを優先しつつ、サイドバーや他のセクションがレスポンシブに調整されるようになっています。

ホーリーグレイルレイアウトを実装する方法は多くありますが、`grid-template-areas` と `grid-area` プロパティを使うのが最もシンプルな方法です。

ホーリーグレイルレイアウトの例を示します：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="header">
    <h2>Header</h2>
  </div>
  <div class="sidebar-left">
    <h2>Left Sidebar</h2>
  </div>
  <div class="main"><h2>Main Content</h2></div>
  <div class="sidebar-right">
    <h2>Right Sidebar</h2>
  </div>
  <div class="footer">
    <h2>Footer</h2>
  </div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-areas:
    'header header header'
    'sidebar-left main sidebar-right'
    'footer footer footer';
  grid-gap: 10px;
  background-color: #2196F3;
  padding: 10px;
}

.header {
  grid-area: header;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.sidebar-left {
  grid-area: sidebar-left;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.main {
  grid-area: main;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.sidebar-right {
  grid-area: sidebar-right;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.footer {
  grid-area: footer;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}
```

:::

`grid-template-areas` と `grid-area` の両方のプロパティは、それぞれ独立して使うことができます。

`grid-template-areas` プロパティは、グリッドコンテナ内の名前付きグリッドエリアをマッピングして視覚的なレイアウトを定義するために特化しています。

一方、`grid-area` プロパティは、個々のグリッドアイテムを配置するために使われ、行と列の位置を指定するか、`grid-template-areas` プロパティで定義した名前付きエリアを参照します。
