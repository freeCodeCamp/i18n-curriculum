# --description--

文字列が与えられたとき、以下のルールに従ってキャメルケースのバージョンを返してください。

- 文字列の引数内の単語は、スペース (` `)、ダッシュ (`-`)、またはアンダースコア (`_`) のいずれかの文字が1つ以上連続して区切りとして使われています。これらの連続した文字列を単語の区切りとして扱ってください。
- 最初の単語はすべて小文字にしてください。
- それ以降の各単語は、最初の文字を大文字にし、残りは小文字にしてください。
- すべてのスペースや区切り文字は削除してください。

```js
function toCamelCase(s) {
  const words = s.replace(/[_\- ]+/g, ' ').split(' ');

  return words.map((word, i) => {
    if (i === 0) {
      return word.toLowerCase();
    } else {
      const tempWord = word.split('');
      return tempWord.shift().toUpperCase() + tempWord.join('').toLowerCase();
    }
  }).join('')
}
```
