# --interactive--

JavaScript에서 문자열을 다룰 때, 문자열의 시작이나 끝에 원하지 않는 공백이 있는 경우가 흔합니다. 공백은 비교, 저장, 표시 같은 작업에 방해가 될 수 있으므로 효율적으로 제거하는 방법을 아는 것이 중요합니다.

이 수업에서는 JavaScript의 `trim()`, `trimStart()`, `trimEnd()` 메서드를 사용해 공백을 제거하는 방법을 살펴보겠습니다.

공백은 문자열 내에 있지만 눈에 보이지 않는 문자로, 공백 문자, 탭, 줄 바꿈 등이 포함됩니다. 예를 들어:

```js
let greeting = "   Hello, world!   ";
```

이 경우, 보이는 텍스트 `Hello, world!` 앞뒤에 공백이 있습니다.

`trim()` 메서드는 문자열의 시작과 끝 양쪽에서 공백을 제거하는 데 가장 많이 사용됩니다. 예시는 다음과 같습니다:

:::interactive_editor

```js
let message = "   Hello!   ";
console.log(message); // "   Hello!   "
let trimmedMessage = message.trim();
console.log(trimmedMessage);  // "Hello!"
```

:::

이 경우 `trim()` 메서드는 앞뒤의 모든 공백을 제거하여 `Hello!`만 남깁니다. 문자열 내부(예: 단어 사이)의 공백은 `trim()`에 의해 건드리지 않는다는 점에 유의하세요.

때로는 문자열의 시작이나 끝 중 한쪽에서만 공백을 제거하고 싶을 때가 있습니다. 이럴 때 `trimStart()`과 `trimEnd()` 메서드를 사용합니다.

`trimStart()`은 문자열의 시작 부분에서 공백을 제거합니다.

:::interactive_editor

```js
let greeting = "   Hello!   ";
console.log(greeting);  // "   Hello!   "
let trimmedStart = greeting.trimStart();
console.log(trimmedStart);  // "Hello!   "
```

:::

`trimEnd()`은 문자열의 끝 부분에서 공백을 제거합니다.

:::interactive_editor

```js
let greeting = "   Hello!   ";
console.log(greeting);  // "   Hello!   "
let trimmedEnd = greeting.trimEnd();
console.log(trimmedEnd);  // "   Hello!"
```

:::

이 메서드들은 문자열의 어느 부분을 정리할지 더 정확하게 제어할 수 있게 해줍니다.

요약하자면, 공백 제거는 JavaScript에서 문자열을 다룰 때 필수적인 작업입니다. 입력 데이터를 정리하거나 일관된 문자열 비교를 보장하려면, `trim()`를 사용해 문자열 양쪽의 공백을 제거하거나 `trimStart()`과 `trimEnd()`를 사용해 특정 쪽만 공백을 제거할 수 있습니다.
