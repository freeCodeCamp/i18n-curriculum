# --description--

In der Programmierung steht `int` für „ganze Zahl“. Es ist ein Datentyp, der verwendet wird, um ganze Zahlen zu speichern, also Zahlen ohne Bruchteile oder Dezimalstellen. Ganze Zahlen können positiv, negativ oder null sein.

Die Wahl des Datentyps `int` ist entscheidend, wenn Sie mit ganzen Zahlen arbeiten, wie zum Beispiel bei Zählungen, Indizes in Arrays oder in jedem Szenario, in dem keine Bruchzahlen benötigt werden.

Beispiele:

Elemente zählen: `int itemCount = 5;`

Alter speichern: `int age = 30;`
