# --explanation--

`More robust` 意味著相較於其他東西更強固或更可靠。 例如：

`This software is more robust than the previous version because it can handle larger data sets.`－此處，`more robust` 意指該軟體更強固且更有能力。

`When it comes to` 介紹一個特定主題，意思是「關於」或「有關」。例如：

`When it comes to customer support, this company is very responsive.` - 這裡，`when it comes to` 介紹了客戶支援的主題。
