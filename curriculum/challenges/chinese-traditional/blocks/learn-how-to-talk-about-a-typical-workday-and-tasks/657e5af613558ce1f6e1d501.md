# --description--

`security incident` 是可能對公司或組織的安全造成危害的事件。 例如，`A security incident like a stolen password needs to be reported immediately`。

關於 `harm` 和 `harmful` 的說明：

`Harm` 是指對某人或某物造成的損害或傷害。它指的是造成負面影響或使某人或某物處於風險中。例如：

`Touching hot surfaces can cause harm.`－這表示觸摸熱表面可能導致受傷或損害。

`Harmful` 是用來描述可能造成傷害的形容詞。它表示危險或有風險，可能導致損害或受傷。 例如：

`Smoking is harmful to health.`－這顯示吸菸可能會導致健康問題或損害。

這些術語經常用於討論安全性和風險，特別是在健康、安全和環境等背景關係中。
