# --description--

`Serverless computing` 是一種 `cloud computing` 執行模型，雲端提供者負責執行伺服器並動態管理機器資源的分配。計費是根據應用程式實際消耗的資源量，而非預先購買的容量單位。之所以稱為 `serverless`，是因為開發者不需要處理伺服器－他們只需上傳程式碼，剩下的由雲端提供者負責。

例如，在 `serverless computing` 中，你可能會將一個函式（程式）上傳到雲端服務，並且只有在需要時才執行。
