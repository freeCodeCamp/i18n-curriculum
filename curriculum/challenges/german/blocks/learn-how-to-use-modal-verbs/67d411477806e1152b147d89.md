# --explanation--

`Have to` wird verwendet, um Verpflichtungen auszudrücken, und wird immer von der Grundform des Verbs (`set up`) gefolgt. Zum Beispiel:

- `You have to update the software to use the new features.` – Es ist erforderlich, die Software zu aktualisieren, um auf die neuen Funktionen zugreifen zu können.

- `They have to set up the meeting room for the presentation.` – Sie müssen den Besprechungsraum für die Präsentation vorbereiten.
