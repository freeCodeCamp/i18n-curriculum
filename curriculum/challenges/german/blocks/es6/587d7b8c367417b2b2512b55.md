# --description--

`import` ermöglicht es Ihnen, auszuwählen, welche Teile einer Datei oder eines Moduls geladen werden sollen. In der vorherigen Lektion haben die Beispiele `add` aus der `math_functions.js`-Datei exportiert. So können Sie es importieren, um es in einer anderen Datei zu verwenden:

```js
import { add } from './math_functions.js';
```

Hier findet `import` die `add` in `math_functions.js`, importiert genau diese Funktion für Sie zur Verwendung und ignoriert den Rest. Das `./` sagt dem Import, dass er die `math_functions.js`-Datei im gleichen Ordner wie die aktuelle Datei suchen soll. Der relative Dateipfad (`./`) und die Dateiendung (`.js`) sind erforderlich, wenn Sie import auf diese Weise verwenden.

Sie können mehr als ein Element aus der Datei importieren, indem Sie sie in der `import`-Anweisung so hinzufügen:

```js
import { add, subtract } from './math_functions.js';
```

# --instructions--

Fügen Sie die passende `import`-Anweisung hinzu, die es der aktuellen Datei erlaubt, die Funktionen `uppercaseString` und `lowercaseString` zu verwenden, die Sie in der vorherigen Lektion exportiert haben. Diese Funktionen befinden sich in einer Datei namens `string_functions.js`, die im gleichen Verzeichnis wie die aktuelle Datei liegt.
