# --explanation--

`Encapsulation` é um princípio fundamental na programação orientada a objetos. Significa agrupar dados (variáveis) e métodos (funções) que operam sobre os dados dentro de uma única unidade, frequentemente uma classe.

Este conceito ajuda a ocultar os detalhes internos de como os dados são gerenciados, expondo apenas o que é necessário para o mundo externo. Isso melhora a segurança e torna o código mais organizado e mais fácil de manter.
