# --description--

Злічувані іменники — це ті, які можна порахувати. Їх може бути один, два, три або більше. Ви можете використовувати числа та квантори, такі як `many`, `a few` або `several`, щоб описати злічувані іменники. Наприклад, яблуко: ви можете мати `one apple`, `two apples` або `many apples`.

Незлічувані іменники — це ті, які не можна порахувати по одному. Зазвичай вони не змінюються, щоб показати множину. Ви не використовуєте числа безпосередньо, натомість використовуйте квантори, як `much`, `a little`, `some`, щоб описати незлічувані іменники. Наприклад, вода: ви не можете сказати `one water` або `two waters`. Ви можете сказати `much water`, `some water`.
