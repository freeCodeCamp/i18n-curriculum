# --description--

Neste diálogo, Sarah usa `Have I told you...?` para perguntar a Tom de forma envolvente se ela já mencionou algo sobre si mesma anteriormente. 

`Have I told you...?` é uma forma de conectar o passado com o presente, como se Sarah estivesse tentando lembrar se já compartilhou sua paixão por sci-fi com Tom antes. 

Esta frase é um ótimo exemplo de como o tempo `Present Perfect` é usado para falar sobre ações que ocorreram em um momento não especificado antes de agora.
