# --description--

`Work overtime` significa trabajar más horas que el horario habitual. Por ejemplo, `I usually work overtime during project deadlines` significa trabajar horas extra para cumplir con los plazos del proyecto.

`Unless` se usa para hablar de una excepción a algo o introducir una condición que hace que la sentencia anterior no sea verdadera. Significa `if it's not the case`. Por ejemplo, `I walk to work unless it rains` significa camino al trabajo si no llueve.

`Urgent` significa algo que es muy importante y necesita atención inmediata. Por ejemplo, `The project is urgent` significa que el proyecto necesita completarse de inmediato.

En una sentencia negativa, `don't/doesn't have to` expresa que algo no es un requisito. Por ejemplo, `I don't have to work late` significa que no se requiere trabajar tarde.
