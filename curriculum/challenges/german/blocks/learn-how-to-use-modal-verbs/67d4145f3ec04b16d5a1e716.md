# --explanation--

`Multi-factor authentication` ist eine Sicherheitsfunktion, die von Benutzenden verlangt, ihre Identität auf mehrere Arten zu bestätigen, bevor sie auf ein Konto zugreifen.

`Multi-factor` bedeutet, mehrere Informationen oder Schritte zu verwenden, um etwas zu überprüfen. Es ist wie mehrere Schutzschichten.

`Authentication` ist der Prozess der Überprüfung oder Bestätigung der Identität einer Person. So stellen Systeme oder Websites sicher, dass Sie wirklich die Person sind, die Sie vorgeben zu sein (zum Beispiel beim Einloggen in Ihr Konto).
