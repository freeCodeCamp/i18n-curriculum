# --description--

プログラミングにおける`assignment operator`は、変数に値を代入するために使われます。Pythonを含む多くのプログラミング言語では、`assignment operator`は`=`記号で表されます。これは、`=`の右側にある値を、左側にある変数名にストアするようコンピューターに指示します。例えば、`number = 10`は値`10`を変数`number`に代入します。
