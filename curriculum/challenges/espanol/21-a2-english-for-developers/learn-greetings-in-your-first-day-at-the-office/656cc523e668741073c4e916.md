# --description--

Para hacer una pregunta usando `it is`, cambias el orden. En lugar de `it is`, se dice `is it`. Así es como conviertes una statement en una pregunta en inglés.

# --questions--

## --text--

¿Cuál de las siguientes opciones es un formulario correcto de una pregunta?

## --answers--

`It is lunch time?`

### --feedback--

Esta sentence se puede entender, pero esta no es la manera común de form una pregunta.

---

`Lunch time it is?`

### --feedback--

Este order no se usa comúnmente para preguntas.

---

`Is lunch time it?`

### --feedback--

Este order es incorrecto para hacer preguntas.

---

`Is it lunch time?`
