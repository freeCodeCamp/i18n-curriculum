# --description--

JavaScriptでは、すべての変数名と関数名は大文字と小文字が区別されます。つまり、文字の大文字・小文字が重要です。

`MYVAR` は `MyVar` や `myvar` と同じではありません。同じ名前でも大文字・小文字が異なれば複数の別々の変数を持つことが可能です。ただし、わかりやすさのために、この言語機能は*使わないこと*を強くお勧めします。

**ベストプラクティス**

JavaScriptの変数名は <dfn>キャメルケース</dfn>で書いてください。<dfn>キャメルケース</dfn>では、複数語の変数名は最初の単語が小文字で、続く各単語の最初の文字が大文字になります。

**例：**

```js
var someVariable;
var anotherVariableName;
var thisVariableNameIsSoLong;
```

# --instructions--

既存の宣言と代入を修正して、名前が <dfn>キャメルケース</dfn>になるようにしてください。

新しい変数は作成しないでください。
