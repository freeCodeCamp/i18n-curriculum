# --description--

$s_k$를 0부터 $k$까지의 숫자를 이진수로 썼을 때 1의 개수라고 하겠습니다.

예를 들어, 0부터 5까지 이진수로 쓰면 0, 1, 10, 11, 100, 101입니다. 1의 개수는 일곱 개이므로 $s_5 = 7$입니다.

수열 $S = \\{s_k : k ≥ 0\\}$는 $\\{0, 1, 2, 4, 5, 7, 9, 12, \ldots\\}$로 시작합니다.

두 명의 플레이어가 게임을 합니다. 게임 시작 전에 숫자 $n$이 정해집니다. 카운터 $c$는 0에서 시작합니다. 각 턴마다 플레이어는 1부터 $n$까지(포함) 숫자 하나를 선택해 $c$에 더합니다. 결과로 나온 $c$의 값은 반드시 $S$의 원소여야 합니다. 유효한 수가 더 이상 없으면 그 플레이어가 집니다.

예를 들어, $n = 5$이고 $c = 0$에서 시작하면:

- 플레이어 1이 4를 선택해 $c$가 $0 + 4 = 4$가 됩니다.
- 플레이어 2가 5를 선택해 $c$가 $4 + 5 = 9$가 됩니다.
- 플레이어 1이 3을 선택해 $c$가 $9 + 3 = 12$가 됩니다.
- 계속 진행됩니다.

$c$는 항상 $S$에 속해야 하며, 각 플레이어는 $c$를 최대 $n$만큼만 증가시킬 수 있습니다.

$M(n)$을 첫 번째 플레이어가 첫 턴에 이길 수 있도록 고를 수 있는 가장 큰 수라고 하겠습니다. 그런 수가 없으면 $M(n) = 0$입니다. 예를 들어, $M(2) = 2$, $M(7) = 1$, $M(20) = 4$입니다.

$1 ≤ n ≤ 20$일 때 $\sum M{(n)}^3 = 8150$임을 확인할 수 있습니다.

$1 ≤ n ≤ 1000$일 때 $\sum M{(n)}^3$를 구하세요.
