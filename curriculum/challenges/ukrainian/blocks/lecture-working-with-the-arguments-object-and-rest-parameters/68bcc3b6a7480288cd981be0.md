# --interactive--

Як ви пам’ятаєте з попередніх уроків і практичних занять, ви можете створювати функції з певною кількістю параметрів і викликати цю функцію з аргументами.

Ось нагадування:

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call
getSum(3, 4); // 7
```

Але що, якщо функцію викликають з більшою кількістю аргументів, ніж вона була визначена приймати?

:::interactive_editor

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call with extra argument
console.log(getSum(3, 4, 5)); // 7
```

:::

JavaScript у такому випадку не видасть помилку. Замість цього він ігнорує зайвий аргумент і просто додасть числа `3` і `4`. Функції, які приймають змінну кількість аргументів, називаються <dfn>варіативними функціями</dfn>.

Якщо ви працюєте з варіативними функціями, то можете використовувати об’єкт `arguments`. Цей об’єкт, схожий на масив, містить значення аргументів, переданих у функцію.

Ось приклад:

:::interactive_editor

```js
function logArgs() {
  for (const arg of arguments) {
    console.log(arg);
  }
}

logArgs(1, 2, 3);
// result:
// 1
// 2
// 3

logArgs("example"); // "example"
```

:::

Оскільки об’єкт `arguments` схожий на масив, ви можете отримати доступ до аргументу за певним індексом так:

:::interactive_editor

```js
function getArg() {
  return arguments[1];
}

console.log(getArg(2, 4, 6)); // 4
```

:::

Ви також можете використати властивість `length`, щоб дізнатися кількість аргументів, з якими викликали функцію:

:::interactive_editor

```js
function getArgs() {
  return arguments.length;
}

console.log(getArgs("Example")); // 1
console.log(getArgs("Another", "Example")); // 2
```

:::

Хоча об’єкт `arguments` виглядає як справжній масив, він не має вбудованих методів `Array`, таких як `includes` або `push`. Щоб мати доступ до цих методів, спочатку потрібно перетворити об’єкт `arguments` на справжній масив за допомогою чогось на кшталт `slice`, `Array.from()` або оператора розширення:

:::interactive_editor

```js
function hasCat() {
  return [...arguments].includes("cat");
}

console.log(hasCat("dog", "chicken", "cat")); // true
console.log(hasCat("dog", "chicken", "horse")); // false
```

:::

Хоча можна працювати з об’єктом `arguments` для варіативних функцій, сучасні JavaScript-застосунки зазвичай використовують синтаксис параметрів залишку. Ви дізнаєтеся про це більше в одному з наступних уроків.
