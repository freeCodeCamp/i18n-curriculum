# --description--

`flex` 宣告其實是你可以在彈性容器的彈性項目上設定的 3 個屬性的縮寫。這些屬性會影響彈性項目在其容器內如何調整大小。你之前看過一些縮寫屬性，但你還沒有正式定義它們。

> 縮寫屬性是 CSS 屬性，讓你同時設定多個其他 CSS 屬性的值。使用縮寫屬性，你可以撰寫更簡潔（且通常更易讀）的樣式表，節省時間和精力。

在這種情況下，`flex` 實際上是 `flex-grow`、`flex-shrink` 和 `flex-basis` 的簡寫。

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/flex-box/flexbox-04.png" alt="CSS code setting the flex property to 1 for a div element.">

在上方的截圖中，`flex: 1` 等同於：`flex-grow: 1`、`flex-shrink: 1`、`flex-basis: 0`。

你經常會看到 flex 簡寫只定義一個值。在這種情況下，該值會套用到 `flex-grow`。所以當你在 div 上設定 `flex: 1` 時，你實際上是在指定 `flex: 1 1 0` 的簡寫。
