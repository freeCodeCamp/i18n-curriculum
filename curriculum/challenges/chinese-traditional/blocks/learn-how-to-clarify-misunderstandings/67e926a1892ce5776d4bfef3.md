# --explanation--

「`Reducing complexity`」意指使系統更容易理解和操作。
  
「`Showing only what's necessary`」表示使用者只會看到他們需要互動的零件。
  
「`Keeping the details hidden`」是指隱藏使用者不需要取用的內部程式碼或邏輯。

綜合來說，抽象性創建了一個清晰、簡單的介面，同時隱藏了幕後更複雜的內部運作。
