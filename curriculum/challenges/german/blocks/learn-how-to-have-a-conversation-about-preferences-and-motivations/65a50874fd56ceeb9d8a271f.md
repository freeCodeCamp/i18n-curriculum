# --description--

`Contagious` bedeutet im Allgemeinen etwas, das von einer Person auf eine andere übertragen werden kann, wie eine Erkältung oder ein Lachen. Wenn Sarah sagt, Bobs Leidenschaft für technische Geräte sei `contagious`, meint sie, dass seine Begeisterung so stark ist, dass sie auch andere, einschließlich sie selbst, dazu bringt, sich für technische Geräte zu begeistern, selbst wenn sie es vorher nicht waren. Es ist wie wenn das Lachen einer Person den ganzen Raum zum Lachen bringt.
