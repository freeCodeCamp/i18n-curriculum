# --explanation--

`Does`は三人称単数の主語に対して`Present Simple`時制で疑問文を作るために使われます。

`Yet`は、期待されることがこれまでに起こったかどうかを尋ねる疑問文でよく使われます。これは、その行動やイベントが予想されているが、まだ起こっていないことを示唆します。例えば：

- `Have you finished your homework yet?`（宿題がもうすぐ終わる、あるいはすでに終わっていることを期待している場合）。
- `Has the movie started yet?`（映画がもうすぐ始まる、あるいはすでに始まっていることを期待している場合）。
