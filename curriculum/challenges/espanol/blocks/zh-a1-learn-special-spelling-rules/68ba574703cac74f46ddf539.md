# --explanation--

La primera sílaba es `zì`, pronunciada con el cuarto tono. La segunda sílaba es `sī`, pronunciada con el primer tono.

`自私 (zì sī)` se usa comúnmente para describir a alguien que solo piensa en sí mismo, incluso en entornos de equipo o trabajo.
