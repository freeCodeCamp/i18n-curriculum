`Mentioned`는 무언가를 간단히 언급하거나 이야기하는 것을 의미합니다. 누군가 대화 중에 주제를 꺼낼 때 자주 사용합니다. 예를 들어:

`Polymorphism`는 프로그래밍 개념입니다. 하나의 함수, 메서드 또는 연산자가 다양한 유형의 데이터나 객체와 함께 사용될 수 있음을 의미합니다. 예를 들어:

`In object-oriented programming, polymorphism allows a single method to behave differently based on the object that calls it.` - 여기서 `polymorphism`는 어떤 객체가 사용하는지에 따라 다른 동작을 수행할 수 있는 메서드를 가리킵니다. 예를 들어, 같은 메서드 `draw()`는 `Circle` 객체가 호출할 때와 `Rectangle` 객체가 호출할 때 다르게 동작할 수 있습니다.
