# --description--

Gli elementi `thead` e `tbody` servono a indicare quale parte della tua tabella è l'intestazione e quale parte contiene i dati o il contenuto principale.

Aggiungi un `thead` e un `tbody` al tuo primo `table`, sotto l'elemento `caption`.

# --hints--

Il tuo primo elemento `table` dovrebbe avere un elemento `thead`.

```js
assert(document.querySelectorAll('table')?.[0]?.querySelector('thead'));
```

Il tuo primo elemento `table` dovrebbe avere un elemento `tbody`.

```js
assert(document.querySelectorAll('table')?.[0]?.querySelector('tbody'));
```

Il tuo elemento `thead` dovrebbe trovarsi subito sotto il tuo elemento `caption`.

```js
assert(document.querySelector('caption')?.nextElementSibling?.localName === 'thead');
```

Il tuo elemento `tbody` dovrebbe trovarsi subito sotto il tuo elemento `thead`.

```js
assert(document.querySelector('thead')?.nextElementSibling?.localName === 'tbody');
```
