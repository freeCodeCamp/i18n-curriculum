# --explanation--

`That way`는 아이디어의 결과나 이점을 설명하는 데 사용됩니다. 예를 들어:

- `If we meet online, we can save time and avoid traveling that way.` - 이는 회의를 온라인으로 진행하여 시간을 절약하고 이동할 필요가 없음을 시사합니다.

- `You should save your work frequently. That way, you won't lose any progress if the system crashes.` - 이는 문제가 발생할 경우 데이터를 잃지 않도록 작업을 자주 저장하라고 조언합니다.

Anna는 수당을 나누어 사용하는 것이 직원들에게 도움이 된다고 설명합니다. 왜냐하면 일부는 기술에, 일부는 사무용 가구에 사용할 수 있기 때문입니다.
