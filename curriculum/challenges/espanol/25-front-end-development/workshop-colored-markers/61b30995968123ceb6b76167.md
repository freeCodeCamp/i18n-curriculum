# --description--

Tenga en cuenta que los bordes de la sombra son nítidos. Esto se debe a que existe un valor opcional `blurRadius` para la propiedad `box-shadow`:

```css
box-shadow: offsetX offsetY blurRadius color;
```

Si no se incluye un valor de `blurRadius`, su valor predeterminado es `0` y produce bordes nítidos. Cuanto mayor sea el valor de `blurRadius`, mayor será el efecto de desenfoque.

En la regla CSS `.green`, añade la propiedad `box-shadow` con los valores `5px` para `offsetX`, `5px` para `offsetY`, `5px` para `blurRadius` y `green` para `color`.

# --hints--

La regla CSS `.green` debe tener la propiedad abreviada `box-shadow` con el valor `5px 5px 5px green`.

```js
assert.strictEqual(new __helpers.CSSHelp(document).getStyle('.green')?.boxShadow, 'green 5px 5px 5px');
```
