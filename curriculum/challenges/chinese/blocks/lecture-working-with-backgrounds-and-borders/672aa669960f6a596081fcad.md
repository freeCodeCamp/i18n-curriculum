# --interactive--

在使用 CSS 中的背景图像时，你有多个属性可用来控制这些图像的显示方式。

我们将重点关注的主要属性是 `background-size`、`background-repeat`、`background-position` 和 `background-attachment`。

我们先来看一下 `background-image` 属性：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

上述 CSS 为 `body` 元素设置了一个猫的背景图像。

如果你想设置背景图像的大小，可以使用 `background-size` 属性。

你可以使用 `contain` 将图像缩放到尽可能大而不裁剪或拉伸。

这是一个使用 `background-size: contain` 的示例：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    min-height: 100px;
  }
</style>
```

:::

我们将 `min-height` 设置为 `100px`，以便背景图像可见且版型维护基线高度，确保即使内容最少，设计也显得平衡且视觉上吸引人。

如果我们将 `background-size` 属性更改为使用 `cover` 值，那么背景图像将缩放以覆盖整个 `body` 元素，同时维护其纵横比。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    min-height: 100px;
  }
</style>
```

:::

在前面的示例中，你可能注意到背景图像会不断重复。

默认情况下，背景图像会在水平方向和垂直方向重复以填充整个元素。但是，你可以控件此行为。

你可以使用 `background-repeat` 属性，值设置为 `no-repeat`。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

当 `background-size` 设置为 `contain` 且 `background-repeat` 设置为 `no-repeat` 时，图像将不再在屏幕上重复。

如果你想水平重复背景图像，可以为 `background-repeat` 属性使用 `repeat-x` 值。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-x;
    min-height: 100px;
  }
</style>
```

:::

如果你想垂直重复背景图像，可以为 `background-repeat` 属性使用 `repeat-y` 值。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-y;
    min-height: 100px;
  }
</style>
```

:::

要在屏幕上定位背景图像，你可以使用 `background-position` 属性。

`background-position` 属性允许你设置背景图像在元素中的显示位置。你可以使用关键字如 `top`、`bottom`、`left`、`right` 和 `center`，或者使用具体的像素或百分比值。

这是一个使用 `center` 和 `top` 作为 `background-position` 的示例：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

这段 CSS 代码将背景图像水平居中定位于元素，并垂直定位于顶部。

最后，`background-attachment` 决定背景图像是在内容滚动时随内容滚动还是在页面滚动时保持固定。

主要值是 `scroll`（默认），背景图像随内容滚动，以及 `fixed`，背景图像保持在屏幕上的相同位置。

下面是使用 `fixed` 值作为 `background-attachment` 属性的示例：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-position: center top;
    background-attachment: fixed;
  }
</style>
```

:::

此 CSS 代码设置背景图像在页面滚动时保持固定不动。

如果你想将几个属性合并成一行，可以使用简写的 `background` 属性。

这是一个示例：

:::interactive_editor

```html
<style>
  body {
    background: center top fixed
      url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

上面的示例将 `background-image`、`background-position` 和 `background-attachment` 合并到一行中。

通过掌握这些属性，你将能够很好地控制背景图像在你的网页设计中的显示方式，从而实现更具视觉吸引力和功能性的版型。
