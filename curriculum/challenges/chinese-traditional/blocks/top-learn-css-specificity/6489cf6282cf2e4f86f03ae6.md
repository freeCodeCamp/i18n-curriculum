# --description--

```css
/* rule 1 */
#subsection .list {
  background-color: yellow;
  color: blue;
}

/* rule 2 */
#subsection .main .list {
  color: red;
}
```

在這個最後的範例中，兩個規則都使用 `ID` 和 `class` 選擇器，因此沒有一個規則使用比另一個更具特異性的選擇器。接著層疊會檢查每種選擇器型別的數量。兩個規則都只有一個 `ID` 選擇器，但規則 2 有更多的 `class` 選擇器，所以規則 2 的特異性較高！

雖然 `color: red` 宣告會有較高的優先權，但 `background-color: yellow` 宣告仍會被套用，因為沒有與之衝突的宣告。
