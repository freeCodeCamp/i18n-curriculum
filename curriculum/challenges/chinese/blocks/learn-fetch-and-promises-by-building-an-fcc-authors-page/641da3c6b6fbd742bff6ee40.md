# --description--

在这个项目中，我们想要在 FreeCodeCamp 新闻上提供有关作者的数据。 如果你想要从在线来源获得数据，你需要使用一个 `API` (应用程序编程界面)。 一个 `API` 让来自机构外的人检索其内部数据。 

有一种名为`fetch`的方法，它通过发送`GET`请求，允许代码从`API`中获取数据。 

下面演示了如何使用 `fetch()` 方法发出 `GET` 请求：

```js
fetch("url-goes-here")
```

向此 URL 发出 `GET` 请求：`"https://cdn.freecodecamp.org/curriculum/news-author-page/authors.json"`。 不要用分号终止你的代码。
