# --description--

FRACTRAN — це турінг-повна езотерична мова програмування, винайдена математиком Джоном Гортоном Конвеєм.

Програма FRACTRAN — це впорядкований список додатних дробів $P = (f_1, f_2, \\ldots, f_m)$ разом з ініціальним додатним цілим числом $n$.

Програму запускають, оновлюючи ціле число $n$ таким чином:

<ul>
  <li>для першого дробу $f_i$ у списку, для якого $nf_i$ є цілим числом, замінити $n$ на $nf_i$;</li>
  <li>повторювати це правило, доки жоден дріб у списку не дає цілого числа при множенні на $n$, після чого зупинитися.</li>
</ul>

Конвей навів програму для простих чисел у FRACTRAN:

$\\dfrac{17}{91}$, $\\dfrac{78}{85}$, $\\dfrac{19}{51}$, $\\dfrac{23}{38}$, $\\dfrac{29}{33}$, $\\dfrac{77}{29}$, $\\dfrac{95}{23}$, $\\dfrac{77}{19}$, $\\dfrac{1}{17}$, $\\dfrac{11}{13}$, $\\dfrac{13}{11}$, $\\dfrac{15}{14}$, $\\dfrac{15}{2}$, $\\dfrac{55}{1}$

Починаючи з $n=2$, ця програма FRACTRAN змінить $n$ на $15=2\\times (\\frac{15}{2})$, потім на $825=15\\times (\\frac{55}{1})$, генеруючи таку послідовність цілих чисел:

$2$, $15$, $825$, $725$, $1925$, $2275$, $425$, $390$, $330$, $290$, $770$, $\\ldots$

Після 2 ця послідовність містить такі степені двійки:

$2^2=4$, $2^3=8$, $2^5=32$, $2^7=128$, $2^{11}=2048$, $2^{13}=8192$, $2^{17}=131072$, $2^{19}=524288$, $\\ldots$

які є простими степенями двійки.

function fractran(progStr){
  var num = new Array();
  var den = new Array();
  var val ;
  var out="";
  function compile(prog){
    var regex = /\s*(\d*)\s*\/\s*(\d*)\s*(.*)/m;
    while(regex.test(prog)){
      num.push(regex.exec(prog)[1]);
      den.push(regex.exec(prog)[2]);
      prog = regex.exec(prog)[3];
    }
  }

  function step(val){
    var i=0;
    while(i<den.length && val%den[i] != 0) i++;
    return num[i]*val/den[i];
  }

  var seq=[]

  function exec(val){
    var i = 0;
    while(val && i<limit){
      seq.push(val)
      val = step(val);
      i ++;
    }
  }

  // Main
  compile(progStr);
  var limit = 10;
  exec(2);
  return seq;
}
