`Take your time to review the code before submitting it.` – Es gibt keinen Zeitdruck; nehmen Sie sich so viel Zeit, wie Sie brauchen, um den Code zu überprüfen.

`Double-check` bedeutet, etwas noch einmal zu überprüfen, um sicherzugehen, dass es korrekt ist. Zum Beispiel:

`I will double-check the data before running the analysis.` – Sie werden die Daten noch einmal überprüfen, um sicherzugehen, dass sie korrekt sind.
