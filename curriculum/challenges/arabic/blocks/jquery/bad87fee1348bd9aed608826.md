# --description--

لنجرّب الآن نقل العناصر من `div` إلى آخر.

تحتوي jQuery على دالة تسمى `appendTo()` تتيح لك اختيار عناصر HTML وإلحاقها بعنصر آخر.

على سبيل المثال، إذا أردنا نقل `target4` من البئر الأيمن إلى البئر الأيسر، نستخدم:

```js
$("#target4").appendTo("#left-well");
```

انقل عنصر `target2` من `left-well` إلى `right-well`.

# --hints--

يجب ألا يكون عنصر `target2` داخل `left-well`.

```js
assert($('#left-well').children('#target2').length === 0);
```

يجب أن يكون عنصر `target2` داخل `right-well`.

```js
assert($('#right-well').children('#target2').length > 0);
```

يجب استخدام jQuery فقط لنقل هذه العناصر.

```js
assert(!code.match(/class.*animated/g));
```
