# --interactive--

`Number`型はJavaScriptや他のプログラミング言語で最もよく使われるデータ型の一つです。数値は単純に見えますが、JavaScriptの数値には多くの興味深い側面があります。では、詳しく見ていきましょう。JavaScriptでは、`Number`データ型が数値を表します。

多くの他のプログラミング言語が整数と浮動小数点数を別々の型に分けているのに対し、JavaScriptは数値を扱うために統一された`Number`型を使います。つまり、整数、小数、さらには特別な数値もすべて同じ`Number`データ型の下で扱えます。

以下は、整数、浮動小数点数、負の数がすべてnumber型であることを示す基本的な例です。

:::interactive_editor

```js
const wholeNumber = 50;
const decimalNumber = 4.5;
const negativeNumber = -7;

console.log(typeof wholeNumber); // number
console.log(typeof decimalNumber); // number
console.log(typeof negativeNumber); // number
```

:::

JavaScriptの`Number`型には、単純な整数や浮動小数点数から、`Infinity`や`NaN`（「Not a Number」）のような特別なケースまで、さまざまな数値が含まれます。主な種類を見ていきましょう。整数は分数や小数部分を持たない全体の数です。正の数、負の数、ゼロのいずれも含まれます。以下は例です。

:::interactive_editor

```js
const positiveInteger = 100;
const negativeInteger = -25;
const zero = 0;

console.log(typeof positiveInteger); // number
console.log(typeof negativeInteger); // number
console.log(typeof zero); // number
```

:::

浮動小数点数は小数点を含む数値です。JavaScriptの開発者はこれを単に「float」と呼ぶことが多いです。測定値や通貨のようにより高い精度が必要な場合に役立ちます。以下は例です。

:::interactive_editor

```js
const floatingPointNumber = 4.5;
const anotherFloat = 89.56;
const oneMoreFloat = 16.462;

console.log(typeof floatingPointNumber); // number
console.log(typeof anotherFloat); // number
console.log(typeof oneMoreFloat); // number
```

:::

JavaScriptは、最大値を超える数値を`Infinity`で表現できます。これは、数値をゼロで割ろうとした場合や、まれに`Number`型の上限を超えた場合に遭遇します。以下は例です。

:::interactive_editor

```js
const infiniteNumber = 1 / 0;
console.log(infiniteNumber); // Infinity
console.log(typeof infiniteNumber); // number
```

:::

JavaScriptでは、数学的な操作が有効な数値を返さない場合があります。例えば、数値でないものに対して算術演算を行おうとすると、`NaN`（「Not a Number」）が返されます。

:::interactive_editor

```js
const notANumber = 'hello world' / 2;
console.log(notANumber); // NaN
```

:::

驚くことに、`NaN`の型も`Number`です。

:::interactive_editor

```js
const notANumber = 'hello world' / 2;
console.log(typeof notANumber); // number
```

:::

標準の10進法（基数10）に加えて、JavaScriptは2進数、8進数、16進数のような異なる基数の数値もサポートしています。2進数は1と0の数字だけを使う基数2のシステムです。8進数は0から7の数字だけを使う基数8のシステムです。16進数は0から9の数字とaからfの文字を使う基数16のシステムで、CSSの16進カラーコードなどで見られます。
