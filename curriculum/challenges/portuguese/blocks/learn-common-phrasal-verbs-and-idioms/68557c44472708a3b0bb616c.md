# --explanation--

`Clear` significa fácil de entender ou ver.Sua forma comparativa, `clearer`, significa mais fácil de entender.Por exemplo:

- `The instructions are clear and simple.` – Isso apenas enfatiza a simplicidade das instruções sem compará-las. 

- `This version of the guide is clearer than the last one.` – Isso compara a simplicidade de duas versões, a atual e a última.

`Prominent` significa algo que é fácil de notar ou que se destaca.Por exemplo:

`The error message should be placed in a prominent position on the screen.` – Isso significa que a mensagem deve ser muito visível para que os usuários possam vê-la facilmente.
