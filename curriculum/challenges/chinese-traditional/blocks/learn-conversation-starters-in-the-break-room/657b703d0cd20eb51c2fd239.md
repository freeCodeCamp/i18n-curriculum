# --description--

讓我們練習使用助動詞來造否定句。

# --questions--

## --text--

選擇正確使用 `don't` 來表達某人不做某事的句子。

## --answers--

`I don't coding on weekends.`

### --feedback--

當與 `don't` 一起使用時，`coding` 應該使用基本形式 `code`。

---

`I don't code on weekends.`

---

`I don't codes on weekends.`

### --feedback--

當與 `don't` 一起使用時，動詞結尾不應該有 `-s`。

---

`I does not code on weekends.`

### --feedback--

`Does not` 用於第三人稱單數主詞（`he`、`she`、`it`）；若主詞是 `I`，則應使用 `do not`。
