# --interactive--

Margin 和 padding 是 CSS 中用於建立良好結構、易讀且視覺吸引的網頁的基本屬性。

邊距控制元素外部的空間，有助於將其與其他元素分開並定義版面結構，而內距控制元素內部的空間，提升內容的可讀性和美觀度。

為了更好地理解 `margin` 和 `padding` 之間的差異，讓我們來看看一些範例。

以下是頁面上三個段落元素的 HTML 範例：

:::interactive_editor

```html
<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

:::

要為每個段落元素的頂部應用間距，你可以像這樣使用 `margin-top` 屬性：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  margin-top: 20px;
  border: 2px solid black;
}
```

:::

在此範例中，我們僅將 `20px` 的外距套用到每個段落元素的頂部。

我們在四個邊緣也有實心黑色邊框，讓你能更清楚看到邊界。

四個不同的 `margin` 屬性是 `margin-top`、`margin-right`、`margin-bottom` 和 `margin-left`。

這裡是一個使用所有四個屬性的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 30px;
  margin-left: 40px;
  border: 2px solid black;
}
```

:::

在此範例中，我們正在為段落元素的四個邊指定間距值。也已添加實心黑色邊框，以便你能更清楚地看到邊距。

使用 `margin` 屬性的另一種方式是使用簡寫表示法。你可以一次指定一個、兩個、三個或四個值。讓我們一起探索這些選項。

當在 `margin` 簡寫中使用單一數值時，該數值將套用到標的元素的所有四個邊。

這裡是一個使用單一值於 `margin` 簡寫的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px;
}
```

:::

此程式碼範例將對段落元素的四個邊緣均勻地套用 `10px` 的 `margin`。

當使用兩個值時，第一個值套用到元素的 `top` 和 `bottom`，而第二個值套用到 `left` 和 `right` 側。

這裡是一個使用兩個值作為 `margin` 簡寫的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px;
}
```

:::

這會將段落元素的 `top` 和 `bottom` 邊距設為 `10px`，`left` 和 `right` 邊距設為 `20px`。

如果提供三個值，第一個值套用到 `top` 邊距，第二個值套用到 `left` 和 `right` 邊距，第三個值套用到 `bottom` 邊距。

這裡有一個範例以便更好地理解：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px;
}
```

:::

這會將 `top` 的邊距設為 `10px`，`left` 和 `right` 的邊距設為 `20px`，以及 `bottom` 的邊距設為 `30px`。

當使用四個數值時，這會給你更多控制，因為你可以獨立指定標的元素每一側的邊界值。

第一個值標的 `top`，第二個值標的 `right`，第三個值標的 `bottom`，第四個值標的 `left`。

這裡是一個使用四個值的 margin 簡寫範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px 40px;
}
```

:::

這會將 `top` 的邊距設為 `10px`，`right` 設為 `20px`，`bottom` 設為 `30px`，以及 `left` 設為 `40px`。

`padding` 屬性用於在元素內部、內容與其邊框之間套用間距。

像 `margin` 屬性一樣，四個 `padding` 屬性分別是 `padding-top`、`padding-right`、`padding-bottom` 和 `padding-left`。

以下是如何為段落元素設定 `padding` 的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 30px;
  padding-left: 40px;
  border: 2px solid black;
}
```

:::

這會將 `top` 的內距設為 `10px`，`right` 設為 `20px`，`bottom` 設為 `30px`，以及 `left` 設為 `40px`。

如你所見，`padding` 是套用在邊框內部的內容上，與套用在邊框外部的 `margin` 不同。

就像 `margin` 屬性一樣，你也可以選擇使用 `padding` 屬性的縮寫。

你也可以在 `padding` 簡寫屬性上指定一、二、三或四個值。

以下是先前段落元素使用 `padding` 簡寫的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding: 10px 20px 30px 40px;
  border: 2px solid black;
}
```

:::

在範例中，使用簡寫時，程式碼會將段落元素的 `top` 設為 `10px`、`right` 設為 `20px`、`bottom` 設為 `30px`，以及 `left` 設為 `40px`。
