# --interactive--

Wenn Sie mit HTML und CSS beginnen, werden Sie feststellen, dass einige Stile auf Ihre Webseiten angewendet werden, noch bevor Sie eigenes CSS schreiben. Diese Stile nennt man „Standard-Browser-Stile“ oder „User-Agent-Stile“.

Diese Standard-Browser-Stile sorgen für eine grundlegende Formatierung, damit HTML-Elemente in allen Browsern lesbar dargestellt werden.

Allerdings können diese Stile von Browser zu Browser leicht variieren, weshalb es wichtig ist, sie zu verstehen, wenn Sie ein einheitliches Erscheinungsbild für Ihre Webseite gestalten möchten.

Schauen wir uns einige häufige Standard-Browser-Stile an, die auf verschiedene HTML-Elemente angewendet werden.

Überschriften sind eines der am häufigsten verwendeten HTML-Elemente und werden standardmäßig mit unterschiedlichen Größen und Schriftstärken gestylt.

Zum Beispiel ist `h1` (die Überschrift der höchsten Ebene) normalerweise fett und größer in der Schriftart als niedrigere Überschriften wie `h2`, `h3` und so weiter.

Diese Überschriften-Tags helfen dabei, die Hierarchie und Struktur Ihres Inhalts zu definieren.

Das folgende Beispiel zeigt alle sechs Überschriften-Elemente von `h1` bis `h6`.

:::interactive_editor

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```

:::

Ein weiteres Element mit Standard-Stilen ist das `hr`-Element, das eine horizontale Linie erzeugt, die oft verwendet wird, um Abschnitte visuell zu trennen. Browser wenden in der Regel einen einfachen Linienstil auf dieses Element an.

Um es besser zu verstehen, sehen wir uns dieses Codebeispiel an:

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<hr>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

Die horizontale Linie erscheint als dünne Linie mit Abstand ober- und unterhalb des Textes, um die Abschnitte voneinander zu unterscheiden.

Als Nächstes betrachten wir das `blockquote`-Element.

Blockzitate werden verwendet, um einen Textabschnitt als Zitat aus einer anderen Quelle zu kennzeichnen. Browser fügen typischerweise eine Einrückung hinzu und setzen den Text manchmal kursiv.

Die Einrückung hilft, Blockzitate vom übrigen Text abzuheben, sodass klar wird, dass dieser Inhalt aus einer anderen Quelle zitiert ist.

Schauen wir uns dieses Codebeispiel an:

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<blockquote>I think, therefore I am. (Rene Descartes)</blockquote>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

Im obigen Beispiel erzeugt das `blockquote`-Element eine Einrückung des Textes, die durch die Standard-Browser-Stile angewendet wird.

Ein weiteres Element mit Standard-Stilen ist das Ankerelement (`<a>`), das standardmäßig mit einer blauen Farbe und Unterstreichung gestylt wird, damit es als Link erkennbar ist.

Sehen wir uns folgendes HTML-Beispiel an:

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<a href="https://freecodecamp.org/">Visit the freeCodeCamp learn page</a>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

Der obige Code enthält vier Absatz-Elemente mit einem Ankerelement in der Mitte. Das Ankerelement ist blau und unterstrichen, um den Nutzenden zu zeigen, dass sie zur freeCodeCamp-Lernseite gelangen.

Zum Schluss betrachten wir die ungeordneten und geordneten Listen-Elemente.

Browser fügen Listen eine grundlegende Formatierung hinzu, einschließlich Einrückung und Aufzählungszeichen oder Zahlen, je nachdem, ob Sie eine ungeordnete Liste (`ul`) oder eine geordnete Liste (`ol`) verwenden.

Sehen wir uns ein Beispiel an:

:::interactive_editor

```html
<p>Sample Paragraph</p>
<ol>
  <li>item 1</li>
  <li>item 2</li>
  <li>item 3</li>
</ol>
<ul>
  <li>item</li>
  <li>another item</li>
  <li>yet another item</li>
</ul>
<p>Ending Paragraph</p>
```

:::

Im obigen Codebeispiel verwenden wir eine ungeordnete Liste und eine geordnete Liste. Beide Listen sind standardmäßig leicht nach rechts eingerückt.

Bei all diesen Beispielen haben Sie gesehen, wie der Browser standardmäßige Abstände, unterschiedliche Schriftgrößen und Schriftstärken auf diese HTML-Elemente anwendet.

In späteren Lektionen lernen Sie, wie Sie einige dieser Standard-Stile mit einem CSS-Reset zurücksetzen können.
