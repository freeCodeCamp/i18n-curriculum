# --description--

Bozo sort，不要與稍微不那麼高效的 bogo sort 混淆，其包含檢查輸入序列是否已排序，若未排序則隨機交換兩個元素。此過程會重複進行，直到序列最終排序完成。

如果我們將前 4 個自然數的所有排列組合作為輸入，則交換次數的期望值，對所有 $4!$ 個輸入序列取平均後為 $24.75$。

已排序的序列需要 0 步。

在這個問題中，我們考慮以下 bozo sort 的變體。

如果序列不是有序的，我們會隨機選取三個元素並隨機洗牌這三個元素。

這三個元素的所有 $3! = 6$ 種排列都是同樣可能的。

已排序的序列將花費 0 步驟。

如果我們將前 4 個自然數的所有排列組合作為輸入，則在所有 $4!$ 輸入序列中，洗牌次數的期望值平均為 $27.5$。

將前 11 個自然數的排列組合作為輸入序列。

在所有 $11!$ 輸入序列的平均下，這個排序演算法預期會執行多少次洗牌？請將你的答案四捨五入到最接近的整數。
