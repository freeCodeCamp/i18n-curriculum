# --description--

単語 `super` と `pretty` は、何かの程度を強調するために使われます。

`Super` は「とても」や「非常に」という意味です。例えば、`This book is super interesting` はその本がとても面白いことを意味します。

`Pretty` は「かなり」や「やや」という意味で使われます。これは、何かが大きな程度であるけれども、極端ではないことを表す言い方です。例えば、`It's pretty cold outside` と言うと、かなり寒いけれども極端に寒いわけではないことを意味します。
