# --description--

函式（程式）是可重複使用的程式碼片段，當你呼叫它們時會執行。許多程式設計語言都內建函式，使你更容易開始。Python 也不例外，我們已經在之前的課程中涵蓋了一些內建函式，例如 `print()`。

另一個有用的內建函式（程式）是 `input()`，它讓你提示使用者輸入：

```python
name = input('What is your name?') # User types "Kolade" and presses Enter  
print('Hello', name) # Output: Hello Kolade
```

另一方面，`int()` 將數字、布林值和數字字串轉換為整數：

```python
print(int(3.14)) # 3
print(int('42')) # 42
print(int(True)) # 1
print(int(False)) # 0 
```

你也可以撰寫你自己的訂製函式（程式）。為此，你使用 `def` 關鍵字，接著是你想給函式的名稱、一對小括弧，然後是冒號。接著在新的一行，撰寫函式應該執行的程式碼。函式執行的程式碼也稱為函式的主體。

這是一個名為 `hello` 的訂製函式（程式）範例，它會將字串 `Hello World` 列印到終端機：

```python
def hello():
    print('Hello World')
```

要執行該函式（程式），你需要使用其名稱呼叫它，後面接上一對小括弧：

```python
hello() # Hello World
```

請注意 `print('Hello World')` 前的縮排。正如你可能從先前的課程中回想起來，Python 依賴縮排來判斷哪些述語群組屬於同一個區塊。這些述語群組稱為程式碼區塊。

這裡有另一個簡單的函式（程式），會將兩個數字的總和列印到終端機：

```python
def calculate_sum(a, b):
    print(a + b)
```

你可以看到我們的函式（程式）`calculate_sum` 在它的小括弧中有 `a` 和 `b`，並以逗號分隔。那些稱為參數。將參數視為佔位變數，當你呼叫函式（程式）時，它們充當你傳入值的「槽」。

要使用參數，你必須傳入「引數」。引數是在你呼叫函式時傳遞給它的值。

以下是如何呼叫 `calculate_sum` 函式（程式）來將數字 `3` 和 `1` 相加：

```python
calculate_sum(3, 1) # 4
```

如果你呼叫函式時引數數量不正確，將會得到 `TypeError`：

```python
calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

函式（程式）也使用特殊的 `return` 關鍵字來離開函式（程式）並傳回一個值。如果你沒有明確地使用 `return`，Python 預設會傳回 `None`。

這裡有一個範例：

```python
def calculate_sum(a, b):
    print(a + b)

my_sum = calculate_sum(3, 1) # 4
print(my_sum) # None
```

你可以看到 `calculate_sum` 函式（程式）列印了 `a` 和 `b` 的總和，但它沒有明確地傳回任何東西。所以當我們將它的結果指定給 `my_sum` 時，實際上的值是 `None`。要修正這個問題，你可以使用 `return` 關鍵字來傳回結果：

```python
def calculate_sum(a, b):
    return a + b

my_sum = calculate_sum(3, 1)
print(my_sum) # 4
```

現在，`calculate_sum` 傳回 `a` 和 `b` 的總和，該總和會被儲存在 `my_sum`。
