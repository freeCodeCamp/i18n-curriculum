# --description--

يمكن كتابة قيم <dfn>السلسلة النصية</dfn> في JavaScript بعلامات اقتباس مفردة أو مزدوجة، طالما تبدأ وتنهي بنفس نوع علامات الاقتباس. على عكس بعض لغات البرمجة الأخرى، تعمل علامات الاقتباس المفردة والمزدوجة بنفس الطريقة في JavaScript.

```js
const doubleQuoteStr = "This is a string"; 
const singleQuoteStr = 'This is also a string';
```

السبب الذي قد يجعلك تستخدم نوعًا من علامات الاقتباس بدل الآخر هو إذا أردت استخدام كلا النوعين داخل سلسلة نصية. قد يحدث هذا إذا أردت حفظ محادثة داخل سلسلة نصية وتضمين المحادثة داخل علامات اقتباس. استخدام آخر هو حفظ وسم `<a>` مع خصائص متعددة داخل علامات اقتباس، وكل ذلك داخل سلسلة نصية.

```js
const conversation = 'Finn exclaims to Jake, "Algebraic!"';
```

لكن هذا يصبح مشكلة إذا احتجت لاستخدام علامات الاقتباس الخارجية داخل السلسلة نفسها. تذكر أن السلسلة النصية تبدأ وتنتهي بنفس نوع علامات الاقتباس. لكن إذا وُجدت نفس علامة الاقتباس في المنتصف، ستتوقف السلسلة مبكرًا وتُصدر خطأ.

```js
const goodStr = 'Jake asks Finn, "Hey, let\'s go on an adventure?"'; 
const badStr = 'Finn responds, "Let's go!"';
```

هنا `badStr` سيرمي خطأ.

في <dfn>goodStr</dfn> أعلاه، يمكنك استخدام كلا النوعين من علامات الاقتباس بأمان باستخدام الشرطة المائلة العكسية `\` كحرف هروب.

**ملاحظة:** لا تخلط بين الشرطة المائلة العكسية `\` والشرطة المائلة الأمامية `/`. فهما لا يؤديان نفس الوظيفة.

# --instructions--

غيّر السلسلة النصية المقدمة إلى سلسلة تبدأ وتنتهي بعلامات اقتباس مفردة ولا تحتوي على أي حروف هروب.

حاليًا، يستخدم وسم `<a>` في السلسلة علامات اقتباس مزدوجة في كل مكان. ستحتاج إلى تغيير علامات الاقتباس الخارجية إلى مفردة لتتمكن من إزالة حروف الهروب.

# --hints--

يجب أن تزيل كل الشرطة المائلة العكسية (`\`).

```js
assert(
  !/\\/g.test(__helpers.removeJSComments(code)) &&
    myStr.match(
      '\\s*<a href\\s*=\\s*"http://www.example.com"\\s*target\\s*=\\s*"_blank">\\s*Link\\s*</a>\\s*'
    )
);
```

يجب أن تحتوي على علامتي اقتباس مفردة `'` وأربع علامات اقتباس مزدوجة `"`.

```js
assert(__helpers.removeJSComments(code).match(/"/g).length === 4 && __helpers.removeJSComments(code).match(/'/g).length === 2);
```
