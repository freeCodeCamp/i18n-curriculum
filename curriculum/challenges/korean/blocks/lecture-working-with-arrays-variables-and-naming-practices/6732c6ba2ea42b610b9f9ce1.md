# --description--

이전 수업에서 `length` 속성을 처음 배웠습니다. 이 속성은 배열의 요소 개수를 반환합니다. 작동 방식을 간단히 다시 살펴보겠습니다:

```js
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits.length); // 3
```

빈 슬롯이 있는 배열도 만들 수 있습니다. 빈 슬롯은 아무것도 들어있지 않은 슬롯을 의미합니다. 이는 `undefined` 값을 가진 배열과는 다릅니다. 이런 배열을 희소 배열이라고 합니다. 예를 들어 다음과 같습니다:

```js
const sparseArray = [1, , , 4];
console.log(sparseArray.length); // 4
```

이 경우 정의된 요소는 `1`와 `4` 두 개뿐이지만, 길이는 `4`입니다. 이는 가장 높은 인덱스(`3`)에 `1`을 더한 값이 `4`이기 때문입니다.

이제 고정 길이의 빈 배열을 만드는 방법을 살펴보겠습니다. JavaScript에서는 여러 방법이 있지만, 일반적인 방법 중 하나는 숫자 인수를 가진 `Array()` 생성자를 사용하는 것입니다. `Array()` 생성자는 `new` 키워드와 함께 새 배열을 만들 때 사용할 수 있습니다. 예를 들어 다음과 같습니다:

```js
const emptyArray = new Array(5);
console.log(emptyArray.length); // 5
console.log(emptyArray); // [ , , , , ]
```

이 예제에서는 `Array(5)`을 사용해 새 배열을 만듭니다. 이 배열은 길이가 `5`인 희소 배열로, 모든 슬롯이 비어 있습니다.

고정 길이의 빈 배열을 만드는 또 다른 방법은 길이 인수를 가진 `Array.from()` 메서드를 사용하는 것입니다. `new Array(n)`와 달리 이 메서드는 지정한 길이의 배열을 만들고 모든 요소가 존재하며 값이 `undefined`입니다:

```js
const fixedLengthArray = Array.from({ length: 5 });
console.log(fixedLengthArray.length); // 5
console.log(fixedLengthArray); // [undefined, undefined, undefined, undefined, undefined]
```

특정 길이의 배열을 만들고 기본값으로 채우려면 `Array.fill()` 메서드를 사용할 수 있습니다:

```js
const filledArray = new Array(3).fill(0);
console.log(filledArray); // [0, 0, 0]
```

이 배열은 길이가 3이고 모든 요소가 `0` 값으로 채워집니다. 참고: 객체로 채울 때 모든 슬롯은 같은 객체를 참조합니다. 독립적인 복사본이 필요하면 콜백이나 `Array.from()`을 사용하세요.

배열의 길이를 구하고 고정 길이 배열을 만드는 방법을 이해하는 것은 특정 알고리즘이나 자료 구조를 위해 배열을 초기화해야 할 때 특히 중요합니다.
