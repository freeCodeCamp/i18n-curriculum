# --explanation--

`Could` e `might` são verbos modais usados para descrever possibilidades.

`Could` sugere que algo é possível mas não certo.Por exemplo:

`If we don't fix this bug, it could cause a system crash.` - Há uma possibilidade de que o sistema pare de funcionar.

`Might` sugere que algo é possível mas menos certo do que `could`.Por exemplo:

`We might need to rewrite some parts of the code.` - É possível que mudanças sejam necessárias.
