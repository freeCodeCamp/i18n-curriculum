# --description--

Eine Variante von N.G. de Bruijns Silver-dollar-Spiel lässt sich wie folgt beschreiben:

Auf einem Streifen von Feldern werden mehrere Münzen platziert, höchstens eine Münze pro Feld. Nur eine Münze, die sogenannte Silver-dollar-Münze, hat einen Wert. Zwei Spieler machen abwechselnd Züge. In jedem Zug muss ein Spieler entweder einen regulären oder einen speziellen Zug ausführen.

Ein regulärer Zug besteht darin, eine Münze auszuwählen und sie ein oder mehrere Felder nach links zu bewegen. Die Münze darf den Streifen nicht verlassen und darf nicht auf oder über eine andere Münze springen.

Alternativ kann der Spieler den speziellen Zug wählen, die linkeste Münze einzustecken, anstatt einen regulären Zug zu machen. Wenn keine regulären Züge möglich sind, ist der Spieler gezwungen, die linkeste Münze einzustecken.

Gewonnen hat der Spieler, der die Silver-dollar-Münze einsteckt.

<img alt="silver dollar game" src="https://cdn.freecodecamp.org/curriculum/project-euler/silver-dollar-game.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Eine Gewinnkonfiguration ist eine Anordnung der Münzen auf dem Streifen, bei der der erste Spieler einen Sieg erzwingen kann, egal was der zweite Spieler tut.

Sei $W(n, c)$ die Anzahl der Gewinnkonfigurationen für einen Streifen mit $n$ Feldern, $c$ wertlosen Münzen und einer Silver-dollar-Münze.

Es ist gegeben, dass $W(10, 2) = 324$ und $W(100, 10) = 1\\,514\\,704\\,946\\,113\\,500$.

Finden Sie $W(1\\,000\\,000, 100)$ modulo der Semiprime $1000\\,036\\,000\\,099 (= 1\\,000\\,003 \times 1\\,000\\,033)$.
