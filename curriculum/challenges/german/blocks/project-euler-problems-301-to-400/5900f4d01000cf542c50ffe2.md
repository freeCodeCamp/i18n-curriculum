# --description--

Definiere $Co(n)$ als die maximal mögliche Summe einer Menge von gleichrangigen Elementen aus $\\{1, 2, \ldots, n\\}$. Zum Beispiel ist $Co(10)$ 30 und erreicht dieses Maximum auf der Teilmenge $\\{1, 5, 7, 8, 9\\}$.

Es ist gegeben, dass $Co(30) = 193$ und $Co(100) = 1356$.

Finde $Co(200\\,000)$.
