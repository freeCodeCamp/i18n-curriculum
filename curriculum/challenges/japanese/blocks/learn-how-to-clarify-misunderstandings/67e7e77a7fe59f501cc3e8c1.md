# --questions--

## --text--

どの文が名詞節を正しく使っていますか？

## --answers--

`I don't understand what does this code do.`

### --feedback--

名詞節では疑問文の語順は使いません。

---

`She asked me whether can AI replace programmers.`

### --feedback--

名詞節では疑問文の語順（`AI can`）ではなく、平叙文の語順（`can AI`）を使います。

---

`The developer explained how the algorithm works.`

---

`We need to know when the update release.`

### --feedback--

名詞節の動詞の構造が不完全です。

## --video-solution--

3

# --explanation--

文`the developer explained how the algorithm works`では、動詞`explained`の目的語として名詞節が正しく使われています。この名詞節は開発者が説明した内容の詳細を提供しています。

他の文は名詞節の語順を誤って使っているか、動詞の構造が不完全で文法的に正しくありません。
