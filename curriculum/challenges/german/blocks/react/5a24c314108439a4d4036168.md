# --description--

Jetzt, da Sie die Grundlagen von JSX und React-Komponenten gelernt haben, ist es Zeit, eine Komponente selbst zu schreiben. React-Komponenten sind die zentralen Bausteine von React-Anwendungen, daher ist es wichtig, sich mit dem Schreiben vertraut zu machen. Denken Sie daran, eine typische React-Komponente ist eine ES6 `class`, die `React.Component` erweitert. Sie hat eine render-Methode, die HTML (aus JSX) oder `null` zurückgibt. Dies ist die Grundform einer React-Komponente. Sobald Sie dies gut verstanden haben, sind Sie bereit, komplexere React-Projekte zu erstellen.

# --instructions--

Definieren Sie eine Klasse `MyComponent`, die `React.Component` erweitert. Ihre render-Methode sollte ein `div` zurückgeben, das einen `h1`-Tag mit dem Text `My First React Component!` enthält. Verwenden Sie diesen Text genau, Groß- und Kleinschreibung sowie Zeichensetzung sind wichtig. Stellen Sie außerdem sicher, dass Sie den Konstruktor für Ihre Komponente aufrufen.

Rendern Sie diese Komponente mit `ReactDOM.render()` in den DOM. Es steht Ihnen ein `div` mit `id='challenge-node'` zur Verfügung.

# --solutions--

```jsx
// Change code below this line
class MyComponent extends React.Component {
  constructor(props) {
    super(props);
  }
  render() {
    return (
      <div>
        <h1>My First React Component!</h1>
      </div>
    );
  }
};

ReactDOM.render(<MyComponent />, document.getElementById('challenge-node'));
```
