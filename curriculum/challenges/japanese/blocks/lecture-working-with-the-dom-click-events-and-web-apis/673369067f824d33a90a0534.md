# --interactive--

ユーザーがボタンをクリックしたりフォームに変更があったりすることをイベントと呼びます。プログラムでは、これらのイベントを監視して反応する方法が必要です。

`addEventListener()` メソッドはイベントを監視するために使います。2つの引数を取ります。1つ目は監視したいイベント、2つ目はイベントが発生したときに呼び出される関数です。

基本的な構文は次の通りです。

```js
element.addEventListener("event", listener);
```

element はイベントを監視する HTML 要素で、event は `"click"` のような監視するイベントの種類を指定します。

listener はイベントが発生したときに通知を受け取るオブジェクトです。ほとんどの場合、イベントを処理するために自分で定義した関数になります。

例を示します。

```js
element.addEventListener("click", () => {
  // code to run when the click event occurs
});
```

関数の参照を渡すこともできます。

```js
function handleClick() {
  // code to run when the click event occurs
}

element.addEventListener("click", handleClick);
```

イベントを処理するために別の関数を作成することもあります。こうすることでコードが読みやすく、管理しやすくなります。

listener 引数には `null` を渡すこともできますし、`EventListener` インターフェースを使うオブジェクトを渡すこともできます。`EventListener` インターフェースは `handleEvent()` という単一のメソッドを定義しています。このメソッドは監視しているイベントが発生すると自動的に呼び出されます。このインターフェースを使うと、同じオブジェクトで複数のイベントを処理することも可能です。

ほとんどの場合、このインターフェースを使う必要はなく、イベントが発生したときに呼び出される関数を渡せば十分です。

では、どのように動作するかを理解するために例を見てみましょう。

この例では、id が `button` の `btn` 要素があります。

```html
<button id="btn">Show alert</button>
```

このボタンの `"click"` イベントを監視して、クリックされたらアラートを表示したいです。

まず JavaScript コードでその `button` 要素にアクセスします。次にクリックイベントを監視するイベントリスナーを追加し、ボタンがクリックされたらアラートを表示します。

:::interactive_editor

```html
<button id="btn">Show alert</button>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("btn");

btn.addEventListener("click", () => alert("You clicked the button"));
```

:::

これでボタンがクリックされるたびに、ユーザーは次のようなアラートメッセージを画面で見ることができます。

```md
You clicked the button
```

もう一つの監視できるイベントは `"input"` イベントです。このイベントは入力要素の値が変わったときに発生します。

例を示します。

```html
<input type="text" id="input" placeholder="Type something" />
```

この入力要素の input イベントを監視して、ユーザーが何か入力するたびにその値をコンソールに記録したいです。

まず JavaScript コードでその `input` 要素にアクセスします。次に input イベントを監視するイベントリスナーを追加し、ユーザーが入力するたびに値をコンソールに記録します。

:::interactive_editor

```html
<input type="text" id="input" placeholder="Type something" />
<script src="index.js"></script>
```

```js
const input = document.getElementById("input");

input.addEventListener("input", () => {
  console.log(input.value);
});
```

:::

`addEventListener()` メソッドを使えば、他にも多くのイベントを監視できます。よく使われるイベントには `mouseover`、`mouseout`、`keydown`、`keyup`、`submit` などがあります。

今後のレッスンで、さらに多くのイベントと `addEventListener()` メソッドを使った監視方法を学びます。
