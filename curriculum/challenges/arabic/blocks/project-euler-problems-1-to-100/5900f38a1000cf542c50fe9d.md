# --description--

من المدهش أن هناك ثلاثة أعداد فقط يمكن كتابتها كمجموع قوى رابعة لأرقامها:

<div style='margin-left: 4em;'>
  1634 = 1<sup>4</sup> + 6<sup>4</sup> + 3<sup>4</sup> + 4<sup>4</sup><br>
  8208 = 8<sup>4</sup> + 2<sup>4</sup> + 0<sup>4</sup> + 8<sup>4</sup><br>
  9474 = 9<sup>4</sup> + 4<sup>4</sup> + 7<sup>4</sup> + 4<sup>4</sup><br>
</div>

بما أن 1 = 1<sup>4</sup> ليست مجموعًا فهي غير مدرجة.

مجموع هذه الأعداد هو 1634 + 8208 + 9474 = 19316.

احسب مجموع كل الأعداد التي يمكن كتابتها كمجموع قوى `n` لأرقامها.
