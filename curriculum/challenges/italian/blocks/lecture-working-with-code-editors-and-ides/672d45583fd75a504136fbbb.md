# --description--

Per installare VS Code, visita il loro sito web per scaricare l’installer adatto al tuo sistema operativo. Assicurati di visitare il sito di VS Code, non quello di Visual Studio.

Per Windows, puoi scaricare un installer EXE. Una volta scaricato, esegui il file e la procedura guidata ti accompagnerà nell’installazione.

Per Mac, puoi scaricare un archivio zip che contiene un `.app` eseguibile. Puoi anche installare VS Code tramite homebrew.

Apri l’app Terminale sul tuo Mac ed esegui il seguente comando:

```sh
brew install --cask visual-studio-code
```

Su Linux, puoi scaricare direttamente un file `.deb` o `.rpm`. Una volta installato, puoi usare l’eseguibile appropriato o il comando CLI (di solito `code`) per avviare l’applicazione. Ora sei pronto per programmare.
