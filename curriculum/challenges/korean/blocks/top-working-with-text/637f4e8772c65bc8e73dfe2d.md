# --description--

HTML 주석은 브라우저에 보이지 않습니다. 주석은 다른 개발자나 미래의 자신이 코드를 읽을 때, 코드에서 명확하지 않을 수 있는 부분에 대한 맥락을 제공하기 위해 작성합니다.

HTML 주석을 작성하는 방법은 간단합니다: 주석을 `<!--`와 `-->` 태그로 감싸면 됩니다. 예를 들어:

```html
<h1> View the html to see the hidden comments </h1>

<!-- I am a html comment -->

<p>Some paragraph text</p>

<!-- I am another html comment -->
```

# --assignment--

HTML에서 텍스트 작업을 연습하려면, 다양한 제목을 사용하고 단락을 포함하며 단락 내 일부 텍스트를 굵게 및 기울임꼴로 표시하는 일반 블로그 기사 페이지를 만드세요. 사이트를 만들 때 실제 텍스트 대신 [Lorem Ipsum](https://loremipsum.io)를 사용해 더미 텍스트를 생성할 수 있습니다.
