# --description--

Eine positive ganze Zahl $n$ heißt quadratfrei, wenn kein Quadrat einer Primzahl $n$ teilt. Somit sind 1, 2, 3, 5, 6, 7, 10, 11 quadratfrei, aber nicht 4, 8, 9, 12.

Wie viele quadratfreie Zahlen gibt es unterhalb von $2^{50}$?
