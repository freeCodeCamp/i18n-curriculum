# --description--

此任務專注於檢視使用 `can` 表達能力以及過去進行式中動詞 `try` 的用法。

`Can` 用來表示某人有能力做某事，例如 `I can solve this problem.`。

`Try` 在過去進行式（`was`/`were` + 動詞的 `-ing` 形式）中表示過去持續進行的動作，例如 `I was trying to fix the issue.`

# --questions--
