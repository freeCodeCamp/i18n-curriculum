# --explanation--

У часі `Present Simple` форма дієслів у третій особі однини зазвичай закінчується на `-s` або `-es`. Наприклад:

- `He codes every day.` — дієслово `codes` закінчується на `-s`, бо воно стосується `he`.

- `She fixes bugs in the system.` — дієслово `fixes` закінчується на `-es`, бо воно стосується `she`.

`Concern` означає турбуватися або викликати у когось занепокоєння щодо чогось. Наприклад:

- `The security of the network concerns the IT team.` — ІТ-команда турбується про безпеку мережі.

- `It concerns me that the software isn't updating correctly.` — Ви турбуєтеся, що програмне забезпечення не оновлюється так, як має.
