# --description--

3차 베지에 곡선은 네 점 $P_0$, $P_1$, $P_2$, $P_3$로 정의됩니다.

곡선은 다음과 같이 구성됩니다:

<img alt="construction of Bézier curve" src="https://cdn.freecodecamp.org/curriculum/project-euler/bzier-curves.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

선분 $P_0P_1$, $P_1P_2$, $P_2P_3$ 위에 점 $Q_0$, $Q_1$, $Q_2$가 그려지며, $\frac{P_0Q_0}{P_0P_1} = \frac{P_1Q_1}{P_1P_2} = \frac{P_2Q_2}{P_2P_3} = t$이고, 여기서 $t$는 [0,1] 범위에 있습니다.

선분 $Q_0Q_1$과 $Q_1Q_2$ 위에 점 $R_0$와 $R_1$이 그려지며, $\frac{Q_0R_0}{Q_0Q_1} = \frac{Q_1R_1}{Q_1Q_2} = t$가 같은 $t$ 값에 대해 성립합니다.

선분 $R_0R_1$ 위에 점 $B$가 그려지며, $\frac{R_0B}{R_0R_1} = t$가 같은 $t$ 값에 대해 성립합니다.

점 $P_0$, $P_1$, $P_2$, $P_3$로 정의된 베지에 곡선은 $Q_0$가 선분 $P_0P_1$ 위의 모든 가능한 위치를 취할 때 점 $B$의 궤적입니다. (모든 점에 대해 $t$ 값이 동일하다는 점을 유의하세요.)

구성에서 알 수 있듯이, 베지에 곡선은 $P_0$에서 선분 $P_0P_1$에 접하고, $P_3$에서 선분 $P_2P_3$에 접합니다.

$P_0 = (1, 0)$, $P_1 = (1, v)$, $P_2 = (v, 1)$, $P_3 = (0, 1)$인 3차 베지에 곡선이 4분원의 근사에 사용됩니다. 값 $v > 0$는 선 $OP_0$, $OP_3$와 곡선으로 둘러싸인 영역의 넓이가 $\frac{π}{4}$ (4분원의 넓이)와 같도록 선택됩니다.

곡선의 길이가 4분원의 길이와 몇 퍼센트 차이가 나는지 구하세요. 즉, 곡선의 길이를 $L$이라 할 때, $100 × \displaystyle\frac{L − \frac{π}{2}}{\frac{π}{2}}$ 값을 소수점 아래 10자리까지 반올림하여 구하세요.
