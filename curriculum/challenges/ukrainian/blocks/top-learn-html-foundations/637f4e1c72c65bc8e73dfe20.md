# --description--

Щоб продемонструвати шаблонний код HTML, вам спочатку потрібен HTML-файл для роботи.

Створіть нову папку на своєму комп’ютері та назвіть її `html-boilerplate`. У цій папці створіть новий файл і назвіть його `index.html`.

Ви, мабуть, вже знайомі з багатьма різними типами файлів, наприклад doc, pdf і зображення.

Щоб повідомити комп’ютеру, що ви хочете створити HTML-файл, потрібно додати до імені файлу розширення `.html`, як ви робили при створенні файлу `index.html`.

Варто зазначити, що ви назвали свій HTML-файл index. Завжди слід називати HTML-файл, який міститиме домашню сторінку ваших сайтів, `index.html`. Це тому, що вебсервери за замовчуванням шукають сторінку `index.html`, коли користувачі заходять на ваші сайти — і відсутність такої сторінки спричинить великі проблеми.

## The DOCTYPE

Кожна HTML-сторінка починається з оголошення doctype. Мета doctype — повідомити браузеру, яку версію HTML слід використовувати для відтворення документа. Остання версія HTML — HTML5, і doctype для цієї версії — просто `<!DOCTYPE html>`.

Doctype для старіших версій HTML був трохи складнішим. Наприклад, це оголошення doctype для HTML4:

```html
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
```

Однак, ймовірно, ви ніколи не захочете використовувати старішу версію HTML, тому завжди будете використовувати `<!DOCTYPE html>`.

Відкрийте файл `index.html`, створений раніше, у вашому текстовому редакторі та додайте `<!DOCTYPE html>` у самий перший рядок.
