# --description--

前のステップでは、`h1`、`h2`、コメント、そして`p`要素を`main`要素の中に入れました。これは*ネストする*と呼ばれます。ネストされた要素は、ネストされている要素よりも右に2文字分スペースを空けて配置します。このスペースの空け方をインデントと呼び、HTMLを読みやすくするために使われます。

以下はネストとインデントの例です：

```html
<main>
  <h1>Most important content of the document</h1>
  <p>Some more important content...</p>
</main>
```

下のコードでは、`h1`要素、`h2`要素、そしてコメントが`main`要素よりも2文字分多くインデントされています。キーボードのスペースキーを使って、`p`要素の前に2文字分のスペースを追加し、正しくインデントしてください。

# --hints--

コードには`h2`要素があり、そのテキストは`Cat Photos`である必要があります。誤って削除したか、開始タグと終了タグの両方が欠けているか、テキストが変更されている可能性があります。

```js
assert.exists(document.querySelector('h2'));
assert.match(code, /<\/h2\>/);
assert.equal(document.querySelector('h2')?.innerText?.trim().replace(/\s+/g, ' ').toLowerCase(),'cat photos')
```

例にある`ul`や`li`要素は追加しないでください。

```js
assert.notExists(document.querySelector('ul'));
assert.notExists(document.querySelector('li'));
```

`h2`要素の行のインデントは変更しないでください。開始タグは行の先頭から6文字分の位置にある必要があります。元のインデントに戻すにはステップを再開してください。

```js
assert.match(code.toLowerCase(), /<\/h1\>\s*\n\s{6}<h2>/);
```

```js

```

コメントのテキストは`TODO: Add link to cat photos`である必要があります。コメントのテキストやスペースは変更しないでください。

```js

コメント要素の行のインデントは変更しないでください。開始タグは行の先頭から6文字分の位置にある必要があります。元のインデントに戻すにはステップを再開してください。

    .toLowerCase(),
    /<\/h2>\s*\n\s{6}<!--\s*todo:\s+add\s+link\s+to\s+cat\s+photos\s*-->/

```

コードには`p`要素が必要です。前のステップで`p`要素を削除しています。

```js
assert.exists(document.querySelector('p'));
```

`p`要素のテキストは`Everyone loves cute cats online!`である必要があります。`p`要素のテキスト、スペース、句読点は変更しないでください。

```js

  document
    .querySelector('p')
    ?.innerText?.trim().toLowerCase(),
    /^everyone\s+loves\s+cute\s+cats\s+online!$/

```

開始`p`タグのインデントは、`h2`要素やコメント要素と一致している必要があります。開始タグは行の先頭から6文字分の位置にある必要があります。

```js

```
