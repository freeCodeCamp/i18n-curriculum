# --explanation--

La `Payment integration` es cuando un sistema de pago (como PayPal, Stripe o procesadores de tarjetas de crédito) está conectado a una app o sitio web para permitir que los clientes paguen. Por ejemplo:

- `We need payment integration on our website to accept credit cards.` - El sitio web debe permitir que los usuarios paguen en línea.

- `The developer worked on payment integration for the mobile app.` - Agregaron una forma para que los usuarios paguen a través de la app.
