# --description--

I valori <dfn>stringa</dfn> in JavaScript possono essere scritti con virgolette singole o doppie, purché si inizi e si finisca con lo stesso tipo di virgolette. A differenza di altri linguaggi di programmazione, le virgolette singole e doppie funzionano allo stesso modo in JavaScript.

```js
const doubleQuoteStr = "This is a string"; 
const singleQuoteStr = 'This is also a string';
```

Il motivo per cui potresti voler usare un tipo di virgolette piuttosto che l'altro è se vuoi usare entrambi all'interno di una stringa. Questo può succedere se vuoi salvare una conversazione in una stringa e avere la conversazione tra virgolette. Un altro uso potrebbe essere salvare un tag `<a>` con vari attributi tra virgolette, tutto all'interno di una stringa.

```js
const conversation = 'Finn exclaims to Jake, "Algebraic!"';
```

Tuttavia, questo diventa un problema se devi usare le virgolette più esterne all'interno della stringa stessa. Ricorda, una stringa ha lo stesso tipo di virgolette all'inizio e alla fine. Ma se hai quel tipo di virgolette da qualche parte nel mezzo, la stringa si interromperà prima e lancerà un errore.

```js
const goodStr = 'Jake asks Finn, "Hey, let\'s go on an adventure?"'; 
const badStr = 'Finn responds, "Let's go!"';
```

Qui `badStr` lancerà un errore.

Nella <dfn>goodStr</dfn> sopra, puoi usare entrambe le virgolette in sicurezza usando la barra rovesciata `\` come carattere di escape.

**Nota:** La barra rovesciata `\` non va confusa con la barra normale `/`. Non fanno la stessa cosa.

# --instructions--

Cambia la stringa fornita in una stringa con virgolette singole all'inizio e alla fine e senza caratteri di escape.

Al momento, il tag `<a>` nella stringa usa virgolette doppie ovunque. Dovrai cambiare le virgolette esterne in singole così potrai rimuovere i caratteri di escape.

# --hints--

Dovresti rimuovere tutte le barre rovesciate (`\`).

```js
assert(
  !/\\/g.test(__helpers.removeJSComments(code)) &&
    myStr.match(
      '\\s*<a href\\s*=\\s*"http://www.example.com"\\s*target\\s*=\\s*"_blank">\\s*Link\\s*</a>\\s*'
    )
);
```

Dovresti avere due virgolette singole `'` e quattro virgolette doppie `"`.

```js
assert(__helpers.removeJSComments(code).match(/"/g).length === 4 && __helpers.removeJSComments(code).match(/'/g).length === 2);
```
