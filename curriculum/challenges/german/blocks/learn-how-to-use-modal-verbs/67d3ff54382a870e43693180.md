# --explanation--

Jessica sagt `you must update your software tools`, was bedeutet, dass es notwendig ist.

`Must` drückt eine starke Notwendigkeit oder Verpflichtung aus. Zum Beispiel:

- `You must wear a helmet when riding a bike.` – Erforderlich für die Sicherheit.

- `Employees must complete training before starting work.` – Eine Regel, die die Mitarbeitenden befolgen müssen.
