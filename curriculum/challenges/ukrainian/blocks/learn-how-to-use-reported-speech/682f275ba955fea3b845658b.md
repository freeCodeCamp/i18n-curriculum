# --explanation--

У **косвенній мові** `to explain` використовується, коли хтось дає причину або пояснює щось. Наприклад:

- Пряма мова: `The server went down because of a power issue.`

- Косвена мова: `She explained that the server went down due to a power issue.`

`system component` — це частина або елемент більшої системи, зазвичай у комп’ютерах або мережах. Наприклад:

`The login module is a key system component for user access.` – це означає, що це одна з основних частин системи.
