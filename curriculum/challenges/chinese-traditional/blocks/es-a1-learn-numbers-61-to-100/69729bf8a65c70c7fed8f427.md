# --description--

在前一節中，你學習了西班牙語中的數字 `sesenta` (60)。

現在你將學習 61 到 69 的數字。

這些數字是由使用十位數字的字 `sesenta` (60)，接著是連接詞 `y` ("and")，以及個位數字 (1-9) 組成。

| Number | Spanish |
| :--- | :--- |
| 60 | `Sesenta` |
| 61 | `Sesenta y uno` |
| 62 | `Sesenta y dos` |
| 63 | `Sesenta y tres` |
| 64 | `Sesenta y cuatro` |
| 65 | `Sesenta y cinco` |
| 66 | `Sesenta y seis` |
| 67 | `Sesenta y siete` |
| 68 | `Sesenta y ocho` |
| 69 | `Sesenta y nueve` |

數字 `sesenta y uno` (61) 在此範圍中是獨特的，因為它在陽性名詞前會變成 `sesenta y un`。例如：

`Sesenta y un años` - 六十一年。
