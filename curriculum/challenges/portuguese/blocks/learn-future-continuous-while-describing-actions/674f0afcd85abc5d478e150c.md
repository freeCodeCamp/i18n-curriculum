# --explanation--

`Scheduled` significa que algo foi planejado ou organizado para ocorrer em um momento específico.Por exemplo:  

`The scheduled meeting will start at 2 PM.` - Isso significa que a reunião foi planejada com antecedência para começar no horário especificado.

`Maintenance` refere-se a atividades realizadas para manter algo funcionando bem ou para corrigir quaisquer problemas.Por exemplo:  

`The team performed maintenance on the server to ensure it runs smoothly.` - Isso significa que a equipe trabalhou para manter o servidor em boas condições corrigindo ou atualizando-o.
