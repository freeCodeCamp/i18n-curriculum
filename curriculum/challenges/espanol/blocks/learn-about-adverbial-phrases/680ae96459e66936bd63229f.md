# --explanation--

`What about...` se usa para pedir información o plantear un nuevo tema relacionado con lo que se acaba de discutir. A menudo se usa para cambiar el enfoque o verificar un aspecto diferente de algo. Por ejemplo:

- `We finished the presentation. What about the handouts?` – Esto significa que el hablante está preguntando si los folletos están listos o si han sido considerados.

- `That plan sounds good. What about the timeline?` – Esto significa que el hablante está de acuerdo con la idea pero quiere preguntar cuánto tiempo tomará.
