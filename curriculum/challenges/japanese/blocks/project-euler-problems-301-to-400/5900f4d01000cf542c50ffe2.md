# --description--

$Co(n)$ を $\\{1, 2, \ldots, n\\}$ から選ばれた互いに素な要素の集合の最大の和と定義します。例えば、$Co(10)$ は 30 で、その最大値は部分集合 $\\{1, 5, 7, 8, 9\\}$ で達成されます。

$Co(30) = 193$ および $Co(100) = 1356$ が与えられています。

$Co(200\\,000)$ を求めてください。
