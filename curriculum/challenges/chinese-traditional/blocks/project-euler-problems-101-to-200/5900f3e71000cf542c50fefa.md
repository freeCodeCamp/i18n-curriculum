# --description--

設 $p_n$ 為第 n 個質數：2、3、5、7、11、⋯，且設 $r$ 為 ${(p_n−1)}^n + {(p_n+1)}^n$ 除以 ${p_n}^2$ 的餘數。

例如，當 $n = 3, p_3 = 5$，且 $4^3 + 6^3 = 280 ≡ 5\\ mod\\ 25$。

使餘數首次超過 `10^9` 的最小 $n$ 值是 7037。

找出餘數首次超過 `10^{10}` 的最小值 n。
