# --explanation--

`Might` は可能性について話すときに使います。つまり、起こるかもしれないが確実ではないことです。例えば：

- `It might rain tomorrow.` - 可能性はあるが保証はない。

- `We might need more time to finish the project.` - 可能性はあるが確定していない。
