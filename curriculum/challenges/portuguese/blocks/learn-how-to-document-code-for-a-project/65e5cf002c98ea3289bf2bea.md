# --description--

Quando um substantivo terminando em `y` se torna plural, o `y` frequentemente muda para `ies`. Essa regra é especialmente útil em contextos de tecnologia onde certos termos frequentemente terminam em `y`.

# --questions--

## --text--

Como você forma o plural de substantivos terminados em `y`?

## --answers--

Ao mudar `y` para `ys`

### --feedback--

Essa não é a regra usual. Geralmente, o `y` muda para `ies` no plural, como em `query` para `queries`.

---

Ao adicionar `s`

### --feedback--

Para a maioria dos substantivos terminando em `y`, simplesmente adicionar `s` não está correto. Por exemplo, `directory` vira `directories`, não `directorys`.

---

Alterando `y` para `ies`

---

Removendo `y`

### --feedback--

Remover o `y` não é a forma correta de formar plurais. O `y` geralmente muda para `ies` no plural, como `binary` para `binaries`.
