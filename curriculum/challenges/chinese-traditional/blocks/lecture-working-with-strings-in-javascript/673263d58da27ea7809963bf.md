# --interactive--

在 JavaScript 中，模板字面值是一種強大且靈活的操作字串方式。與使用單引號（`'`）或雙引號（`"`）的普通字串不同，模板字面值是用反引號（```）定義的。

它們允許更輕鬆的字串操作，包括直接在字串中嵌入變數，這種特性稱為字串插值。

模板字串讓你更容易創建跨多行或直接在字串中包含表達式（例如變數甚至 JavaScript 程式碼）的字串。

這裡有一個模板字串的範例：

:::interactive_editor

```js
const name = "Alice";
const greeting = `Hello, ${name}!`;

console.log(greeting);
```

:::

請注意使用反引號而非單引號或雙引號。`${name}` 語法是一個字串插值的範例，其中變數 `name` 的值會直接插入到字串中。

字串插值允許你在字串中嵌入變數和表達式。這比舊有的方法有顯著的改進，舊方法是使用 `+` 運算子串接字串和變數。

這裡是一個使用字串串接和加號（`+`）運算子的範例：

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = "My name is " + name + " and I am " + age + " years old.";
console.log(message); 
```

:::

以下是使用模板字面值和字串插值的範例：

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = `My name is ${name} and I am ${age} years old.`;
console.log(message); 
```

:::

如你所見，使用模板字面值進行字串插值更加簡潔且易於閱讀，特別是當你處理多個變數時。

模板字面值的另一個優秀特性是它們支援多行字串。使用一般字串時，你需要使用跳脫字元（`\n`）來產生換行。使用模板字面值時，你可以直接將字串寫在多行上，且格式會被保留：

:::interactive_editor

```js
let poem = `Roses are red,
Violets are blue,
JavaScript is fun,
And so are you.`;

console.log(poem);
```

:::

模板字面值的另一個特性是它們允許你直接在字串中嵌入 JavaScript 表達式，就像這個範例中一樣：

:::interactive_editor

```js
const song = "Bohemian Rhapsody";
const score = 9.5;
const highestScore = 10;
const output = `One of my favorite songs is "${song}". I rated it ${
  (score / highestScore) * 100
}%.`;
console.log(output); 
```

:::

模板字串在你需要將變數或表達式包含在字串中、格式化複雜字串或處理多行文字時特別有用。與傳統字串串接相比，它們更簡潔且易讀。
