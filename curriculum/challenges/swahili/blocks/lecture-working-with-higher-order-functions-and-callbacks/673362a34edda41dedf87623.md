# --interactive--

Njia ya `map` ni kitendakazi chenye nguvu na kinachotumika sana katika JavaScript kinachofanya kazi kwenye safu za data. Imeundwa kuunda safu mpya ya data kwa kutumia kitendakazi fulani kwenye kila kipengele cha safu ya awali.

Njia hii haibadilishi safu ya awali bali hurudisha safu mpya yenye matokeo ya kitendakazi kilichotumika kwenye kila kipengele.

Hapa kuna mfano wa kutumia njia ya `map` kwenye safu ya nambari:

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map((num) => num * 2);
console.log(numbers); // [1, 2, 3, 4, 5]
console.log(doubled); // [2, 4, 6, 8, 10]
```

:::

Ili kuunda safu mpya ambapo kila nambari inazidishwa mara mbili, tunatumia njia ya `map`. Njia ya `map` inakubali kitendakazi cha mwitiko ambapo kitendakazi kinaitwa kwenye kila kipengele cha safu.

Katika kesi hii, kila nambari katika safu itazidishwa na `2`. Matokeo yatakuwa safu mpya ya nambari `2,4,6,8,10`.

Kitendakazi cha mwitiko kinaweza kupokea hoja tatu hadi tatu.

Hoja ya kwanza ni kipengele kinachotumika kwa sasa.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element) => {
  console.log("Element:", element);
  return element * 2;
});
```

:::

Hoja ya pili ni kielezo cha kipengele kinachotumika kwa sasa.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index) => {
  console.log("Element:", element);
  console.log("Index:", index);
  return element * 2;
});
```

:::

Hoja ya tatu ni safu ambayo `map` inaitwa juu yake.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index, array) => {
  console.log("Element:", element);
  console.log("Index:", index);
  console.log("Array:", array);
  return element * 2;
});
```

:::

Kuelewa na kutumia kwa ufanisi njia ya `map` kunaweza kuboresha sana uwezo wako wa kufanya kazi na safu za data katika JavaScript. Katika mafundisho yajayo, tutaingia zaidi katika matumizi ya hali ya juu ya `map` na kuchunguza jinsi inavyoweza kuwa chombo chenye nguvu kwa kujenga programu zenye nguvu na zenye ufanisi.
