# --description--

6174 是一个非凡的数字；如果我们将它的数字按升序排列并从按降序排列的数字中减去该数字，我们得到 $7641 - 1467 = 6174$。

更值得注意的是，如果我们从任何4位数字开始并重复这个排序和减法过程，我们最终将以6174结束或者如果所有数字相等则立即结束0。

如果我们用前导零填充数字直到我们有4位数，这也适用于少于4位的数字。

例如，让我们从数字 0837 开始：

$$\begin{align}
  & 8730 - 0378 = 8352 \\\\
  & 8532 - 2358 = 6174
\end{align}$$

6174被称为Kaprekar常数。 排序和减去并重复这一过程直到0或达到Kaprekar常数的过程称为Kaprekar例程。

我们可以考虑其他基数和位数的Kaprekar例程。 不幸的是，并不能保证在所有情况下都存在Kaprekar常数;例程可以在某个输入数字的循环中结束，或者例程到达的常数对于不同的输入数字可以是不同的。  However, it can be shown that for 5 digits and a base $b = 6t + 3 ≠ 9$, a Kaprekar constant exists.

例如  
base 15: ${(10, 4, 14, 9, 5)}\_{15}$  
base 21: $(14, 6, 20, 13, 7)\_{21}$

将 $C_b$ 定义为基数 $b$ 下 5 位的 Kaprekar 常量。将函数 $sb(i)$ 定义为：

- 当 $i = C_b$ 或者 $i$ 用基数 $b$ 表示时由 5 个一致数字组成，则为 0
- Kaprekar 例程在基数 $b$ 下达到 $C_b$ 所需的迭代次数，否则

请注意，我们可以为所有整数 $i &lt; b^5$ 定义 $sb(i)$。如果用基数 $b$ 表示的 $i$ 少于 5 位数字，则该数字会用前导零填充，直到有 5 位数字，然后应用 Kaprekar 例程。

将 $S(b)$ 定义为 $sb(i)$ 在 $0 < i < b^5$ 范围内的和。例如，$S(15) = 5\,274\,369$，$S(111) = 400\,668\,930\,299$

查找 $2 ≤ k ≤ 300$ 时 $S(6k + 3)$ 的和。将最后 18 位数字作为你的答案。
