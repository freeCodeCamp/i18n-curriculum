# --explanation--

`Overall` significa considerar toda a situação ou todos os fatores.Por exemplo:

`Overall, the app is easy to use.` - Isso significa que, ao olhar para tudo, o app é geralmente bom.

`User-friendly` significa que algo é fácil de usar para as pessoas, especialmente para aquelas que podem não estar familiarizadas com isso.Por exemplo:

`The new website is very user-friendly, so even beginners can navigate it easily.` - Isso significa que o site é simples de usar, com um design que facilita o entendimento para todos.
