# --explanation--

`Make sure to` 用來表達責任或承諾，表示說話者一定會做某事。 例如：

`Make sure to save your work before closing.` — 不要忘記在關閉程式前先存檔。

「`As needed`」表示只有在變得必要時才執行某事。 例如：

`We can add more team members as needed。` — 他們只有在變得必要時才會添加人員。

在 Alice 的句子中，她表示如果專案的進度需要，她會定期通知 David 並更改預算計劃。
