# --explanation--

`To struggle`는 무언가에 어려움을 겪는다는 뜻입니다. 예를 들어:

- `The program is struggling to load.` - 이 문장은 프로그램이 현재 로딩하는 데 어려움을 겪고 있다는 뜻입니다.

`Present Continuous` 시제는 **지금 바로 일어나고 있는** 동작을 설명할 때 사용합니다.

위 예시에서, 동사 `Present Continuous`로 `struggle` 시제를 만들 때 마지막 `e`가 빠져 `struggling`가 되는 것을 볼 수 있습니다. 이는 `e`로 끝나는 동사에 흔히 적용되는 규칙입니다.

Tom의 문장에서는 IDE(통합 개발 환경)에 대한 지속적인 어려움을 설명하기 위해 `Present Continuous` 시제를 사용했습니다.

지금 이 동작이 일어나고 있으므로 `-ing`를 `struggle`에 추가해야 합니다.
