# --explanation--

Un `draft` es una versión temprana de un documento o plan que puede ser editada o revisada. Por ejemplo:

`I wrote a draft of the email before sending the final version.` – Esto significa que la persona escribió una versión preliminar del correo electrónico.

`To run something by someone` se usa para mostrar o explicar algo a alguien para recibir retroalimentación o aprobación. Por ejemplo:

`Can I run this idea by you before we present it?` – Esto significa que estás pidiendo la opinión de alguien al respecto.

`To roll something out` significa lanzar o introducir oficialmente algo a un grupo más amplio. Por ejemplo:

`They will roll out the new software update next week.` – Esto significa que la nueva versión del software será lanzada la próxima semana.

El sufijo `-wide` significa "a lo largo de todo" algo. En `company-wide`, significa algo que aplica a toda la empresa. Otro ejemplo:

`There was a system-wide outage yesterday.` – Esto significa que el problema afectó a todo el sistema.
