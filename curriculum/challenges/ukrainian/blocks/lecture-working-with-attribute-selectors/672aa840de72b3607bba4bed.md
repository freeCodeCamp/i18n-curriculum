# --interactive--

Селектор атрибутів у CSS — це потужний інструмент, який дозволяє цілитися в HTML-елементи на основі їхніх атрибутів. Це означає, що ви можете застосовувати стилі до елементів, які мають певні атрибути або навіть конкретні значення цих атрибутів.

Це особливо корисно, коли ви хочете стилізувати елементи динамічно або коли одних лише імен класів недостатньо для потрібної специфіки.

Наприклад, ви можете використати селектор атрибутів, щоб цілитися у всі посилання, які мають атрибут `href`. Це особливо зручно, коли потрібно застосувати однаковий стиль до всіх посилань на сторінці.

Ось як можна використати селектор атрибутів, щоб цілитися у посилання з атрибутом `href`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Example link with an href attribute</a>
<a>Example link without an href attribute</a>
```

```css
a {
  display: block;
}

a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

Це правило CSS застосує синій колір і підкреслення до будь-якого посилання, яке містить атрибут `href`, забезпечуючи однакове стилізування всіх активних посилань.

Але селектори атрибутів можуть бути більш точними. Припустімо, ви хочете цілитися лише у посилання, які мають атрибут `title`. Атрибут `title` часто надає додаткову інформацію про посилання, і ви можете захотіти стилізувати такі посилання інакше, щоб показати, що вони містять додаткову інформацію. Ось як це зробити:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

Це правило застосовує жирний текст і прибирає підкреслення з будь-якого посилання з атрибутом `title`. Це чудовий спосіб візуально відрізнити ці посилання від інших на сторінці.

Ви також можете комбінувати селектори атрибутів для ще більшої точності. Наприклад, якщо ви хочете стилізувати посилання, які мають одночасно атрибути `href` і `title`, ви можете зробити так:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[href][title] {
  display:block;
  color: green;
}
```

:::

У цьому випадку стилізуються зеленим текстом лише посилання, які містять одночасно атрибути `href` і `title`. Такий рівень контролю робить селектори атрибутів дуже потужними в CSS.

Ще один приклад селектора атрибутів — це співпадіння одного значення у списку значень, розділених пробілами, в атрибуті.

Ось приклад елемента посилання з кількома класами:

```html
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

Щоб цілитися у цей конкретний елемент посилання, можна використати такий селектор:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

Синтаксис `[attr~=value]` тут використовується, щоб цілитися у всі елементи посилання, де атрибут class містить слово `"primary"`.

Якщо потрібно цілитися у елемент, де значення атрибута починається з певного префікса, можна використати синтаксис `[attr^=value]`. Ось приклад:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

У цьому прикладі селектор `a[href^="https://"]` цілитиметься у всі елементи посилання, де значення атрибута `href` починається з `"https://"`.

Щоб цілитися у елементи, де значення атрибута закінчується на певне значення, можна використати синтаксис `[attr$=value]`. Ось приклад:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Visit Example Site</a>
```

```css
a[href$=".com"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

У цьому прикладі селектор `a[href$=".com"]` цілитиметься у всі елементи посилання, де значення атрибута `href` закінчується на `.com`.

Використання селекторів атрибутів не лише покращує стилізацію вашої сторінки, а й підвищує доступність, роблячи інтерактивні елементи, як-от посилання, більш помітними на основі їхніх атрибутів.
