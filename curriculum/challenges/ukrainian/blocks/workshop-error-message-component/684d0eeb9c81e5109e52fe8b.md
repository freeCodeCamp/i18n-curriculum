# --description--

Елемент `div` починає виглядати краще. Але тепер кнопка dismiss `button` розташована трохи занадто близько до абзацу.

Щоб створити пробіл між гнучкими елементами, ви можете використати такий шаблон: `gap-number`, де кожна одиниця `number` представляє 0.25rem (або 4px) фіксованого простору між рядками та стовпцями в гнучкому макеті.

Ось приклад застосування `gap` до елемента:

```html
<p class="flex gap-8">This is gap.</p>
```

Всередині вашого відкриваючого тегу `div` використайте правильний утилітарний клас для встановлення горизонтального розриву (`gap`) на `16px`.

# --hints--

Ваш відкриваючий тег `div` повинен містити утилітний клас `gap-4`.

```js
const divEl = document.querySelector("div");
assert.isTrue(divEl.classList.contains("gap-4"));
```
