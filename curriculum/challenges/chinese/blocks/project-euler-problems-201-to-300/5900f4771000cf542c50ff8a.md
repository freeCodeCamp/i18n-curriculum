# --description--

您将获得独特的投资机会。

从 £1 资本开始，你可以选择固定比例 $f$ 的资本，在公平的抛硬币中连续下注 1000 次。

你的回报是对你的投注加倍，你失去了对尾巴的赌注。

例如，如果 $f = \frac{1}{4}$，第一次抛掷你下注 £0.25，如果正面朝上你赢得 £0.5，手中就有 £1.5。然后你下注 £0.375，如果第二次抛掷是反面，你就有 £1.125。

选择 $f$ 以最大化你在 1,000 次抛掷后至少拥有 £1,000,000,000 的机会，你成为亿万富翁的概率是多少？

假设所有计算都是精确的（没有舍入），但是将答案四舍五入到小数点后面的12位数，形式为0.abcdefghijkl。
