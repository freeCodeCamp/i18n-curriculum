# --description--

As pessoas se comunicam de maneiras diferentes, dependendo da situação.

Você pode usar o `teléfono` para falar diretamente com alguém, ou um `mensaje de texto`, que é uma mensagem escrita curta enviada por telefone.

Você também pode usar o `correo`, que se refere ao correio tradicional, como cartas enviadas para um endereço residencial. Hoje, muitas mensagens são enviadas online. Nesses casos, é chamado de `correo electrónico`.

`Electrónico` descreve algo que usa tecnologia ou dispositivos eletrônicos.
