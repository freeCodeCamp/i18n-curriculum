# --description--

Dada uma string de palavras, retorne apenas as palavras com um número ímpar de letras.

- As palavras na string dada estarão separadas por um único espaço.
- Retorne as palavras separadas por um único espaço.

```py
def get_odd_words(s):
    return ' '.join(word for word in s.split(' ') if len(word) % 2 == 1)
```
