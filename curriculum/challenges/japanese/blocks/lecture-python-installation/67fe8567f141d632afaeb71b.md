# --description--

すべてのワークショップとラボでは、freeCodeCampのPythonエディターを使用します。しかし、ローカルマシンにPythonをセットアップする方法を学ぶことも重要です。

WindowsとMacにPythonをインストールする最も簡単な方法は、公式Pythonウェブサイトからインストーラーをダウンロードすることです。このレッスンの後半ではLinuxでPythonを実行する方法も説明します。

`https://www.python.org/`にアクセスして「Downloads」にカーソルを合わせてください。モーダルが表示され、あなたのOSに対応した現在のPythonバージョンが表示されます。

まずmacOSでPythonをインストールする方法を説明します：

- 現在のPythonバージョンが表示されているボタンをクリックすると（前述のモーダルから）、`.pkg`のインストールファイルのダウンロードが自動的に始まります。
- `.pkg`インストーラーのダウンロードが完了したら、それを開き、表示されるウィンドウで「Continue」をクリックしてください。
- 「Continue」ボタンをクリックし続け、「Installation Type」セクションに到達したら、「Install」ボタンをクリックします。
- 必要に応じてパスワードを入力し、インストールを開始してください。
- その後、Pythonが正常にインストールされたことを知らせるお祝いのメッセージが表示されます。
- 「Close」ボタンをクリックして完了です！

インストールを確認するには、ターミナルを開いて`python --version`または`python3 --version`を実行してください。

また、ターミナルで`python`または`python3`を実行するとPythonインタープリターを開けます。PythonインタープリターはPythonコードを読み取り、コンピューターが理解し実行できる命令に変換するプログラムです。

<dfn>ターミナル</dfn>は、コマンドを入力してコンピューターとやり取りできるテキストベースのインターフェースです。各OSにはデフォルトのターミナルアプリがあります。macOSではTerminalアプリ、WindowsではCommand PromptやPowerShell、LinuxではGNOME TerminalやKonsoleなどのデスクトップ環境ごとのデフォルトターミナルがあります。

古いmacOSやLinuxシステムでは、`python`がPython 2用に予約されていて、`python3`がPython 3専用の場合があります。`python --version`を実行して`Python 2.7.18`のようなPython 2のバージョンが表示されたら、OSが古いPythonで書かれたソフトウェアに依存している可能性があります。その場合は、今後Pythonコードを実行する際に`python3`を使うべきです。Python 2はサポート終了しており、新しい開発には使用しないでください。

WindowsにPythonをインストールするには、以下の手順に従ってください：

- `https://www.python.org/`にアクセスし、「Downloads」にカーソルを合わせてください。「Download for Windows」と書かれたモーダルと現在のPythonバージョンのダウンロードボタンが表示されます。
- バージョン番号をクリックすると、Windows用の実行可能ファイル（`.exe`）のダウンロードが自動的に始まります。
- Pythonインストーラーのダウンロードが完了したら、ダブルクリックして指示に従ってください。
- `Add python.exe to Path`のオプションが表示されたら、そのオプションにチェックを入れてから`Install Now`をクリックしてください。これにより後で作業が楽になります。

インストールを確認するには、PowerShellなどのコマンドラインシェルを開いて`python --version`を実行してください。Pythonインタープリターは`python`を実行して開けます。

Linuxの主要なディストリビューション（Ubuntu、Debian、Fedoraなど）にはほとんどPythonがプリインストールされています。

ターミナルを開いて`python --version`または`python3 --version`を実行してください：

どちらのコマンドもPythonのバージョンを表示しない場合は、`https://www.python.org`であなたのLinuxディストリビューションに合ったインストールパッケージを探すか、オンラインで推奨されるPythonのインストール方法を検索してください。
