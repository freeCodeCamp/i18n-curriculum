# --description--

`Strange` означає щось незвичайне або неочікуване. Це як знайти щось відмінне від того, що ви зазвичай бачите. Наприклад: `A strange sound in a car might mean a problem.`

`error` — це помилка, особливо така, що заважає чомусь працювати правильно. Наприклад: `An error in a code means there is a mistake that needs fixing.`
