# --description--

Das Verb `encounter` bedeutet, etwas zu treffen oder zu begegnen, besonders etwas Unerwartetem. In der Programmierung bedeutet `encountering` ein Problem oder einen Bug zu begegnen, dass man während der Ausführung des Codes auf ein Problem stößt.

Zum Beispiel bedeutet `You might encounter an error if the user inputs the wrong type of data.` Dieser Satz bedeutet, dass Sie unerwartet mit einem Fehler konfrontiert werden oder sich damit auseinandersetzen müssen.
