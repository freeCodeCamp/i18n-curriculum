# --description--

Verbos modais como `might` e `can` são usados para expressar possibilidade e habilidade. 

`Might` é usado para algo que pode acontecer ou ser verdade, como dizer, `It might rain today`. 

`Can` é usado para falar sobre a habilidade de alguém de fazer algo, como, `She can speak three languages`.Vamos praticar usando essas palavras em frases.
