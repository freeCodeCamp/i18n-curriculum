# --description--

A frase `have to` é usada para falar sobre coisas que são necessárias ou obrigatórias. É semelhante a `must`, mas `must` é mais forte e usado para regras mais importantes ou sentimentos pessoais.

Depois de `have to`, você usa um verbo na forma base para explicar qual ação é necessária. Por exemplo, `You have to wear a helmet` significa que usar um capacete é necessário para a segurança.

`Though` é uma palavra usada para mostrar um pequeno contraste ou surpresa em uma frase. É semelhante a `but`. Por exemplo, `It's cold, though it's sunny` significa que é surpreendente o tempo estar frio porque também está ensolarado.
