# --description--

Eine 3-glatte Zahl ist eine ganze Zahl, die keinen Primfaktor größer als 3 hat. Für eine ganze Zahl $N$ definieren wir $S(N)$ als die Menge der 3-glatten Zahlen, die kleiner oder gleich $N$ sind. Zum Beispiel ist $S(20) = \\{1, 2, 3, 4, 6, 8, 9, 12, 16, 18\\}$.

Wir definieren $F(N)$ als die Anzahl der Permutationen von $S(N)$, bei denen jedes Element nach all seinen echten Teilern kommt.

Dies ist eine der möglichen Permutationen für $N = 20$.

-   1, 2, 4, 3, 9, 8, 16, 6, 18, 12.

Dies ist keine gültige Permutation, weil 12 vor seinem Teiler 6 kommt.

-   1, 2, 4, 3, 9, 8, 12, 16, 6, 18.

Wir können überprüfen, dass $F(6) = 5$, $F(8) = 9$, $F(20) = 450$ und $F(1000) ≈ 8.8521816557e\\,21$ gilt.

Finden Sie $F({10}^{18})$. Geben Sie Ihre Antwort als String in wissenschaftlicher Notation an, gerundet auf zehn Stellen nach dem Komma. Verwenden Sie bei der Angabe Ihrer Antwort ein Kleinbuchstaben-`e` als Trennzeichen zwischen Mantisse und Exponent. Zum Beispiel, wenn die Antwort $112\\,233\\,445\\,566\\,778\\,899$ ist, dann wäre das Antwortformat `1.1223344557e17`.
