# --description--

ウェブサイトはテキストだけを表示できるとかなり退屈になってしまいます。幸いにもHTMLはさまざまなメディアを表示するための多様な要素を提供しています。その中で最も広く使われているのが画像要素です。

HTMLで画像を表示するには`<img>`要素を使います。これまでに出てきた他の要素とは異なり、`<img>`は空要素です。空の、voidなHTML要素は終了タグが不要です。

開くタグと閉じるタグでコンテンツを囲む代わりに、`src`属性を使って画像をページに埋め込みます。この属性はブラウザに画像ファイルの場所を伝えます。`src`属性はアンカーの`href`属性とよく似ています。絶対パスと相対パスの両方で画像を埋め込むことができます。

例えば、絶対パスを使ってThe Odin Projectサイトにある画像を表示できます：

自分のウェブサイトにある画像を使うには相対パスを使います。

- `images`プロジェクト内に`odin-links-and-images`という名前の新しいディレクトリを作成してください。

- 次に、<a href="https://unsplash.com/photos/Mv9hjnEUHR4/download?force=true&w=640" target="_blank">この画像</a>をダウンロードして、先ほど作成したimagesディレクトリに移動してください。

- 画像の名前を`dog.jpg`に変更してください。

最後に、画像を`index.html`ファイルに追加します：

```html
<body>
  <h1>Homepage</h1>
	<a href="https://www.theodinproject.com/about">click me</a>

	<a href="pages/about.html">About</a>

	<img src="images/dog.jpg">
</body>
```

`index.html`ファイルを保存してブラウザで開き、チャールズの姿を確認してください。


## 親ディレクトリ

`about`ページで犬の画像を使いたい場合はどうしますか？まずpagesディレクトリの親ディレクトリに一段階上がってからimagesディレクトリにアクセスする必要があります。

親ディレクトリに移動するには相対ファイルパスで2つのドットを使います：`../.`。これを実際にやってみましょう。`body`ファイルの`about.html`内で、先ほど追加した見出しの下に次の画像を追加してください：

```html
<img src="../images/dog.jpg">
```

説明すると：

- まず、pagesディレクトリの親ディレクトリである`odin-links-and-images`に移動します。

- 次に親ディレクトリから`images`ディレクトリに入ります。

- 最後に`dog.jpg`ファイルにアクセスします。

先ほどの比喩を使うと、ファイルパスで`../`を使うのは、今いる部屋から一歩出てメインの廊下に出て別の部屋に行くようなものです。

## Alt属性

`src`属性のほかに、すべての画像要素には`alt`（代替テキスト）属性も必要です。

`alt`属性は画像を説明するために使います。画像が読み込めない場合に代わりに表示されます。また、視覚障害のあるユーザー向けにスクリーンリーダーで画像の内容を伝えるためにも使われます。
