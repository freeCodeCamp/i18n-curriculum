# --interactive--

الرفع هو سلوك في JavaScript يربك المبتدئين غالبًا، لكن فهمه يساعدك على تجنب الأخطاء البرمجية الدقيقة في شفرتك. ببساطة، الرفع هو السلوك الافتراضي في JavaScript الذي ينقل الإعلانات إلى أعلى نطاقاتها خلال مرحلة التجميع قبل تنفيذ الشفرة.

لفهم الرفع، من المهم معرفة أن JavaScript تعمل في مرحلتين: مرحلة التجميع ومرحلة التنفيذ. خلال مرحلة التجميع، يقوم محرك JavaScript بمراجعة شفرتك ويُعد مساحة ذاكرة للمتغيرات والدوال. هنا يظهر دور الرفع.

لنبدأ برفع المتغيرات، عندما تعلن عن متغير باستخدام الكلمة المفتاحية `var`، يقوم JavaScript برفع الإعلان إلى أعلى نطاقه. لكن من المهم ملاحظة أن الإعلان فقط هو الذي يُرفع، وليس التهيئة. هذا يعني أنه يمكنك استخدام متغير في شفرتك قبل إعلانه، لكن قيمته ستكون `undefined` حتى تعيّن له قيمة فعلية.

:::interactive_editor

```js
console.log(x); // undefined
var x = 5;
console.log(x); // 5
```

:::

في هذا الكود، رغم أننا نستخدم `x` قبل إعلانه، لا نحصل على خطأ، بل نحصل على `undefined`. هذا لأن JavaScript ترفع الإعلان `var x` إلى أعلى نطاقه، لكن لا ترفع التهيئة `x = 5`. كأن الشفرة أُعيدت كتابتها هكذا:

:::interactive_editor

```js
var x;
console.log(x); // undefined
x = 5;
console.log(x); // 5
```

:::

يعمل رفع الدوال بطريقة مختلفة قليلاً. عندما تعلن عن دالة باستخدام صيغة إعلان `function`، يتم رفع اسم الدالة وجسمها معًا. هذا يعني أنه يمكنك استدعاء دالة قبل إعلانها في شفرتك. إليك مثالًا على رفع الدوال:

:::interactive_editor

```js
sayHello(); // "Hello, World!"

function sayHello(){
  console.log("Hello, World!");
}
```

:::

في هذه الحالة، يمكننا استدعاء `sayHello()` قبل إعلانها لأن الدالة كاملة تُرفع إلى أعلى نطاقها.

من المهم ملاحظة أن الرفع يعمل بشكل مختلف مع إعلانات `let` و `const` التي أُدخلت في ES6.

```js
console.log(y); // Throws a ReferenceError
let y = 10;
```

هذه الإعلانات تُرفع لكنها غير مهيأة ولا يمكنك الوصول إليها قبل الإعلان الفعلي في شفرتك. يُشار إلى هذا غالبًا بمنطقة الموت الزمنية.

فهم الرفع يساعدك على كتابة شفرة أنظف وأكثر توقعًا. لكن الاعتماد على الرفع قد يجعل شفرتك أصعب في القراءة والصيانة. كأفضل الممارسات، يُنصح بإعلان متغيراتك في أعلى نطاقها ودوالك قبل استخدامها بغض النظر عن الرفع. هذا يجعل سلوك شفرتك أكثر وضوحًا وأسهل لفهم الآخرين بما فيهم أنت في المستقبل.
