# --explanation--

`Must have happened` は過去のことについて論理的な推測をするために使われます。

`Must` は強い確信を表します。`Have` + `Past Participle`（`happened` のように）は過去のことを指します。例えば：

- `He must have forgotten to update the system.` - 彼がアップデートをするのを覚えていなかった可能性が高い。

- `They must have made some changes to the app.` - アプリが更新または改善された可能性が高い。

ジェイク は証拠（`unusual activity on the servers`）を使って、侵害が `must have happened` 昨夜に起きたと結論づけています。この構文は論理的な推測をするのに役立ちます。
