# --explanation--

`That should work.` é uma forma comum de concordar com uma sugestão.Significa que a ideia soa aceitável ou possível, mesmo que não esteja 100% confirmada.Por exemplo:

- Pessoa 1: `Can we meet at 3 PM tomorrow?` - Esta pessoa está sugerindo um horário para se encontrar no dia seguinte.

- Pessoa 2: `That should work.` - Isso significa que o horário provavelmente está bom para eles se encontrarem.
