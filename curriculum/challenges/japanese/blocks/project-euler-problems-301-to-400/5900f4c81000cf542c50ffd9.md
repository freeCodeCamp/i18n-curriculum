# --description--

素数 2 と 3 の両方で割り切れる $100$ 以下の最大の整数は 96 です。なぜなら $96 = 32 \times 3 = 2^5 \times 3$ だからです。

異なる2つの素数 $p$ と $q$ に対して、$M(p, q, N)$ を $N$ 以下の正の整数のうち、$p$ と $q$ の両方でのみ割り切れる最大の整数と定義します。もしそのような正の整数が存在しなければ $M(p, q, N)=0$ とします。

例えば、$M(2, 3, 100) = 96$ です。

$M(3, 5, 100) = 75$ であり、90 ではありません。なぜなら 90 は 2、3、5 のすべてで割り切れるからです。また、$M(2, 73, 100) = 0$ です。なぜなら 2 と 73 の両方で割り切れる $100$ 以下の正の整数が存在しないからです。

$S(N)$ をすべての異なる $M(p, q, N)$ の和とします。$S(100)=2262$ です。

$S(10\,000\,000)$ を求めてください。
