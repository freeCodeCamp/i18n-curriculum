# --explanation--

Este diálogo entre Florencia y Federico repasa frases introductorias fundamentales:

`Mi nombre es` y `Me llamo` son formas alternativas de decir tu nombre. 

`¿De dónde eres?` significa "Where are you from?". Esta pregunta se responde con **`Soy de` + [country]**.

`¿Cuántos años tienes?` significa "How old are you?". Esta pregunta se responde con **`Tengo` + [number] + `años`**.

`¿Y vos?` significa "¿Y tú?" y se usa para devolver una pregunta. Es una variación regional de `¿Y tú?` común en países de Centroamérica como El Salvador, y algunos países de Sudamérica, como Argentina.

`Encantada` ("Mucho gusto") e `Igualmente` ("Igualmente") son frases corteses usadas durante las presentaciones.
