# --description--

`Work overtime` significa trabalhar mais horas do que o horário de trabalho habitual.Por exemplo, `I usually work overtime during project deadlines` significa trabalhar horas extras para cumprir os prazos do projeto.

`Unless` é usado para falar sobre uma exceção a algo ou introduzir uma condição que torna a afirmação anterior falsa.Significa `if it's not the case`.Por exemplo, `I walk to work unless it rains` significa que eu caminho para o trabalho se não chover.

`Urgent` significa algo que é muito importante e precisa de atenção imediata.Por exemplo, `The project is urgent` significa que o projeto precisa ser concluído imediatamente.

Em uma frase negativa, `don't/doesn't have to` expressa que algo não é um requisito.Por exemplo, `I don't have to work late` significa que trabalhar até tarde não é obrigatório.
