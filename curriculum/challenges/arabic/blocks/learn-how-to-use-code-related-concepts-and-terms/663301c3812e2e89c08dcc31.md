# --description--

يُستخدم التعبير `thanks for` للتعبير عن الامتنان تجاه أفعال محددة أو مساعدة تم تلقيها. غالبًا ما يتبع هذا التركيب صيغة `-ing` من فعل (اسم الفعل) أو اسم لتوضيح سبب الشكر.

أمثلة:

`Thanks for helping me with my homework.`

`Thanks for the advice.`
