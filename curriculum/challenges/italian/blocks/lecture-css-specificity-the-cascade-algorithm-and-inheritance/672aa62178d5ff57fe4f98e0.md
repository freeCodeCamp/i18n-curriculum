# --interactive--

La specificità CSS è un concetto fondamentale che determina quali stili vengono applicati a un elemento quando possono applicarsi più regole.

Capire la specificità aiuta gli sviluppatori a risolvere i conflitti tra diverse regole CSS e garantisce che gli stili desiderati vengano applicati in modo coerente.

La specificità CSS si calcola in base al tipo di selettori usati.

La specificità più alta è attribuita agli stili in linea, che vengono applicati direttamente a un elemento tramite l'attributo `style`.

In questo esempio, il primo paragrafo sarà rosso mentre gli altri elementi `p` saranno blu. Questo perché gli stili in linea hanno una specificità più alta rispetto ai selettori di tipo come il selettore `p` mostrato nel file `styles.css`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p style="color: red;">Red paragraph</p>
<p>Other paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p {
  color: blue;
}
```

:::

Successivamente, i selettori ID hanno un alto livello di specificità.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Red paragraph</p>
<p>Other paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

p {
  color: blue;
}
```

:::

Poi entrano in gioco i selettori di classe, i selettori di attributo e le pseudo-classi.

Gli esempi includono selettori di classe come `.container` e `.button`, selettori di attributo come `[type="text"]` e pseudo-classi come `:hover`. Questi hanno un livello moderato di specificità.

**NOTA**: Imparerai di più sulle pseudo-classi nelle lezioni future.

In questo esempio, il primo paragrafo sarà rosso perché un selettore `id` ha una specificità più alta rispetto a classi e selettori di tipo. Gli elementi `.example-para` avranno un colore verde mentre i restanti elementi paragrafo avranno un colore blu.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p class="example-para">Other paragraph</p>
<p class="example-para">Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

.example-para {
  color: green;
}

p {
  color: blue;
}
```

:::

I selettori di tipo, come `div` e `h1`, e i pseudo-elementi come `::before` e `::after`, hanno una specificità più bassa rispetto ai gruppi precedenti.

**NOTA**: Imparerai di più sui pseudo-elementi nelle lezioni future.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p>Other paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

p {
  color: blue;
}
```

:::

Infine, il selettore universale, rappresentato da un asterisco `*`, ha la specificità più bassa di tutte.

Ecco un esempio di impostazione del colore per tutti gli elementi a rosso usando il selettore `*`. Tuttavia non vedrai elementi rossi perché ci sono selettori ID e di tipo che sovrascrivono quegli stili, evidenziando la bassa specificità del selettore `*`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p>Other paragraph</p>
<p>Yet another paragraph</p>
```

```css
* {
  color: red;
}

#para {
  color: green; 
}

p {
  color: blue;
}
```

:::

I valori di specificità si calcolano come un valore a quattro parti `(a, b, c, d)`:

- `a`: Stili in linea (`1` o `0`).
- `b`: Numero di selettori ID.
- `c`: Numero di selettori di classe, selettori di attributo e pseudo-classi.
- `d`: Numero di selettori di tipo, pseudo-elementi e selettori universali.

Ogni parte del valore di specificità ha un peso diverso:

- **Gli stili in linea** (`a`) hanno il peso più alto, contribuendo con un valore di `1` alla prima parte del valore di specificità.
    
- **I selettori ID** (`b`) hanno il secondo peso più alto, con ogni ID che contribuisce con `1` alla seconda parte del valore di specificità.
    
- **I selettori di classe, selettori di attributo e pseudo-classi** (`c`) hanno un peso moderato, con ciascuno che contribuisce con `1` alla terza parte del valore di specificità.
    
- **I selettori di tipo e i pseudo-elementi** (`d`) hanno il peso più basso, con ciascuno che contribuisce con `1` alla quarta parte del valore di specificità.
    
- **Il selettore universale** (`*`): il selettore universale contribuisce con `0` al calcolo della specificità e non influisce sulla specificità. La sua inclusione in un selettore non cambia il valore di specificità.

Il CSS in linea ha la specificità più alta perché viene applicato direttamente all’elemento. Sovrascrive qualsiasi CSS interno o esterno. Il valore di specificità per gli stili in linea è `(1, 0, 0, 0)`.

Ecco un altro esempio di uno stile in linea:

:::interactive_editor

```html
<p style="color: red;">This text is red.</p>
```

:::

Il CSS interno è definito all’interno di un elemento `style` nella sezione `head` del documento HTML. Ha una specificità più bassa rispetto agli stili in linea. Sia il CSS interno che quello esterno condividono lo stesso livello di specificità — la specificità è determinata dai selettori usati, non da dove il CSS è definito. Quando due regole hanno la stessa specificità, vince quella che appare più tardi nel documento. Quindi se un foglio di stile esterno è collegato dopo un blocco interno `style`, le regole esterne hanno la precedenza. Se è collegato prima, vincono le regole interne.

Il valore di specificità per gli stili interni è determinato dai selettori usati. Per esempio, un selettore ID all’interno del CSS interno ha un valore di specificità di `(0, 1, 0, 0)`.

:::interactive_editor

```html
<head>
  <style>
    #text {
      color: blue;
    }
  </style>
</head>
<body>
  <div id="text">This text is blue.</div>
</body>
```

:::

In questo esempio, il testo sarà blu a meno che non venga applicato uno stile in linea o un selettore più specifico.

Il CSS esterno è collegato tramite un elemento `link` nella sezione `head` ed è scritto in file separati `.css`. Come il CSS interno, la specificità degli stili esterni è determinata dai selettori usati. Il CSS esterno offre la migliore manutenibilità per progetti più grandi.

Il valore di specificità per gli stili esterni è anch’esso determinato dai selettori usati. Per esempio, un selettore di classe all’interno del CSS esterno ha un valore di specificità di `(0, 0, 1, 0)`.

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <div class="text">This text's color is defined in an external CSS file.</div>
</body>
```

```css
.text {
  color: purple;
}
```

:::

In questo esempio, il colore del testo è definito nel file `styles.css` e verrà applicato a meno che non venga sovrascritto da un selettore più specifico o da uno stile in linea.
