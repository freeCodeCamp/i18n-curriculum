`differently`  

### --feedback--

Este es un adverbio que significa que algo se puede hacer de otra manera.

# --explanation--

`To be better` sugiere que algo diferente podría ser una mejora. Por ejemplo:

¿No sería mejor empezar antes y evitar el estrés de último minuto? - Esto significa que empezar antes podría ser una mejora.

`Differently` significa de una manera diferente. Describe **cómo** se hace algo. Esto contrasta con `different`, que se usa para describir cómo son las cosas, personas o situaciones. Otros ejemplos:

- Necesitamos un plan diferente.

- Necesitamos planificar de manera diferente.
