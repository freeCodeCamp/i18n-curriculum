# --interactive--

في الدروس السابقة، تحدثنا قليلاً عن كيفية إنشاء مساحة بين عناصر الشبكة. لكن في هذا الدرس، سنتعمق أكثر في كيفية استخدام خصائص `row-gap` و`column-gap` و`gap` في تخطيط الشبكة.

لكن أولاً، نحتاج إلى مراجعة ما هو المسار في شبكة CSS. المسار هو المسافة بين خطي شبكة متجاورين. تُنشأ هذه الخطوط تلقائيًا عند استخدام CSS Grid. في هذا السياق، تشير المسارات عادةً إلى الصفوف والأعمدة التي تشكل تخطيط الشبكة.

لإنشاء فجوات بين الأعمدة في شبكة CSS، يمكنك استخدام خاصية `column-gap`. القيم المقبولة لهذه الخاصية تشمل البكسل، وحدة `em`، النسب المئوية، أو الكلمة المفتاحية `normal`.

إذا استخدمت القيمة `normal` لخاصية `column-gap`، فستكون النتيجة `0` لتخطيطات الشبكة و`1em` لتخطيطات متعدد الأعمدة.

إليك مثال على الترميز لتخطيط شبكة بأربعة أعمدة:

```html
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

بالنسبة لـ CSS، نحدد العرض إلى grid ونضبط خاصية `column-gap` على 10 بكسل:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  height: 100px;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.grid-container div {
  background-color: darkblue;
}
```

:::

إذا أردنا تعديل المثال ليحتوي على صفين من الصناديق الزرقاء وإنشاء مساحة أكبر بين الصفوف، يمكننا استخدام خاصية `row-gap`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  column-gap: 10px;
  row-gap: 30px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

في هذا المثال المعدل، نضبط خاصية `row-gap` على 30 بكسل ونغير `grid-template-columns` لاستخدام وحدتين `1fr` فقط بدلاً من أربعة لإنشاء صفين من الصناديق.

تمامًا مثل خاصية `column-gap`، القيم المقبولة لخاصية `row-gap` يمكن أن تشمل النسب المئوية، `em`، والبكسل.

إذا أردت استخدام خاصية مختصرة لإنشاء فجوات بين الصفوف والأعمدة، يمكنك استخدام خاصية `gap`. إليك الصيغة النحوية الأساسية:

```css
gap: row-value optional-column-value;
```

إذا حددت قيمة واحدة لخاصية `gap`، فستُطبق هذه القيمة على كل من الصفوف والأعمدة. وإذا حددت قيمتين، فالقيمة الأولى تذهب للصف والقيمة الثانية تُطبق على العمود:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 30px 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

القيم المقبولة للخاصية المختصرة `gap` تشمل النسب المئوية، البكسل، `em` وحتى قيم `calc()`. لكن لا يمكنك استخدام وحدات `fr` هنا.

توفر خصائص `row-gap` و`column-gap` و`gap` طرقًا مرنة للتحكم في المسافات بين العناصر في تخطيط شبكة CSS. باستخدام هذه الخصائص، يمكنك بسهولة إنشاء شبكات جذابة بصريًا مع فجوات متناسقة وقابلة للتعديل بين الصفوف والأعمدة.
