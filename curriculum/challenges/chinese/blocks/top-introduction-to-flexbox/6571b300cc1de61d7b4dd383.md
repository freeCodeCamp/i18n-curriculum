# --description--

`flex` 声明实际上是你可以在一个弹性项上设置的 3 个属性的简写。这些属性影响弹性项在其容器内如何调整自身大小。你之前见过一些简写属性，但你还没有正式定义它们。

> 简写属性是 CSS 属性，允许你同时设置多个其他 CSS 属性的值。使用简写属性，你可以编写更简洁（且通常更易读）的样式表，节省时间和精力。

在这个分支中，`flex` 实际上是 `flex-grow`、`flex-shrink` 和 `flex-basis` 的简写。

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/flex-box/flexbox-04.png" alt="CSS code setting the flex property to 1 for a div element.">

在上图中，`flex: 1` 等同于：`flex-grow: 1`、`flex-shrink: 1`、`flex-basis: 0`。

你经常会看到只用一个值定义的 flex 简写。在这种情况下，该值会应用到 `flex-grow`。所以当你在我们的 div 上设置 `flex: 1` 时，你实际上是在指定 `flex: 1 1 0` 的简写。
