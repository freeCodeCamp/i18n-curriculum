# --explanation--

`Right` no final de uma frase é uma tag de pergunta. É usado para confirmar concordância ou verificar se o ouvinte entende. Por exemplo:

- `It's going to rain today, right?` - O falante está pedindo confirmação sobre o clima.

- `You finished the project, right?` - O falante está verificando se o ouvinte completou a tarefa.

Neste diálogo, Bob está pedindo para Maria confirmar ou concordar com sua afirmação.
