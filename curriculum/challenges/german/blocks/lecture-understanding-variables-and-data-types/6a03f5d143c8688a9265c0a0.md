# --description--

In der vorherigen Lektion haben Sie die gängigen Datentypen in Python kennengelernt, darunter die Typen String, Integer und Float. Während Sie Ihre Programme entwickeln, müssen Sie lernen, wie Sie den Typ einer Variablen anzeigen können.

Hier ist ein Beispiel für eine Variable:

```py
developer = 'Devin'
```

Um zu sehen, welchen Typ `developer` hat, können Sie die Funktion `type()` so verwenden:

```py
developer = 'Devin'

print(type(developer)) # <class 'str'>
```

Die Ausgabe von `<class 'str'>` bedeutet, dass `developer` vom Typ String ist.

Wenn Sie der Funktion `type()` keine Argumente übergeben, erhalten Sie folgende Fehlermeldung:

```md
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: type() takes 1 or 3 arguments
```

Hier sind alle Datentypen, die Sie bisher gelernt haben, zusammen mit ihren Typen im Terminal:

```python
my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 'hello', 8.5}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 'hello', 8.5)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>
```

Es wird Situationen in Ihrem Programm geben, in denen Sie überprüfen müssen, ob eine bestimmte Variable einen bestimmten Typ hat, bevor Sie Operationen darauf ausführen. Hier ist die Funktion `isinstance()` sehr nützlich.

Hier ist ein Beispiel für eine Variable, der ein String zugewiesen wurde:

```py
account_balance = '12'
```

Wenn Sie versuchen, mathematische Ausdrücke wie Division mit der Variablen `account_balance` durchzuführen, erhalten Sie eine Fehlermeldung.

```py
account_balance = '12'

account_balance / 2

# Traceback (most recent call last):
#   File "<stdin>", line 1, in <module>
# TypeError: unsupported operand type(s) for /: 'str' and 'int'
```

Um zu prüfen, ob `account_balance` eine ganze Zahl ist, können Sie die Funktion `isinstance()` so verwenden:

```py
account_balance = '12'

isinstance(account_balance, int) # False
```

Die eingebaute Funktion `isinstance()` ermöglicht es Ihnen zu überprüfen, ob eine Variable einem bestimmten Datentyp entspricht. Sie nimmt ein Objekt und den Typ, gegen den geprüft werden soll, und gibt einen Boolean zurück. In diesem Fall, da `account_balance` ein String ist, wird `False` zurückgegeben.

Die Funktion `isinstance()` erlaubt es Ihnen auch, mehrere Typen gleichzeitig zu überprüfen.

Hier ist ein Beispiel, um zu prüfen, ob `account_balance` ein `int` oder `float` ist:

```py
account_balance = 12
isinstance(account_balance, (int, float)) # True
```

In diesem Beispiel ist `account_balance` eine ganze Zahl, daher gibt `isinstance()` `True` zurück. Wenn `account_balance` ein `12.0` wäre, würde `isinstance()` trotzdem `True` zurückgeben, weil Sie auf ganze Zahlen oder Floats prüfen.

In zukünftigen Workshops und Labs werden Sie die Funktionen `type()` und `isinstance()` verwenden, um sicherzustellen, dass Ihre Variablen die richtigen Datentypen enthalten, bevor Sie Operationen darauf ausführen.
