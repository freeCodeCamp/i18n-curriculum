# --explanation--

Послідовні сполучники, такі як `first`, `then`, `after that` і `finally`, використовуються для опису порядку дій або подій.

- `First` використовується для опису ініціальної дії.

- `Then` позначає наступний безпосередній крок.

- `After that` використовується для введення кроку, що йде після іншої дії з певним розмежуванням між ними.

- `Finally` позначає останній крок у послідовності.
