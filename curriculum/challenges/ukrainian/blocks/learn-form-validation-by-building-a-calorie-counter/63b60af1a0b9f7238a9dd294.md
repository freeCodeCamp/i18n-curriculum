# --description--

Вашому меню вибору потрібні опції для кожного з елементів їжі та вправ `fieldset`, які ви створили на попередніх кроках. Використайте елемент `option`, щоб створити нову опцію для кожного `fieldset`. Атрибут `value` кожної опції має бути `id` `fieldset`, а текст кожної опції має бути текстом `legend`.

Встановіть опцію `Breakfast` як опцію `selected`.
