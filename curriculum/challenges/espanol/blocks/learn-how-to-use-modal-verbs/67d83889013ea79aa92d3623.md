# --explanation--

Un `full scan` es una revisión de seguridad completa que examina todo un sistema o red en busca de amenazas. Por ejemplo:

- `You should run a full scan to check for viruses.` - Es una buena idea escanear todo el sistema en busca de archivos dañinos.

- El equipo de TI realiza un escaneo completo de la red cada mes. - El equipo técnico revisa todos los sistemas de la red en busca de problemas de seguridad una vez al mes.
