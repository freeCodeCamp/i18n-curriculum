# --explanation--

`Third Conditional` 用于谈论过去的不真实情况——未发生的事情——及其可能的结果。它的结构是 `If` + `past perfect`，情态动词（`could/might/should`）+ `have` + `past participle`。使用这个条件式意味着 `if` 从句中的动作没有发生，结果也没有出现，但你想象它是一个可能的结果。例如：

`If we had tested the system earlier, we might have prevented the failure.` – 这意味着没有进行测试，导致了失败，但如果进行了测试，可能就能避免失败。
