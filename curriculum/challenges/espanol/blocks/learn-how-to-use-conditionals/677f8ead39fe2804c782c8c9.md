# --explanation--

Sarah está preguntando si Mark ha notado un problema con algunas computadoras que no `booting properly`.

`Have you noticed` muestra que Sarah está verificando la conciencia de Mark sobre el problema, no pidiéndole que lo arregle o realice alguna acción.

`Booting` significa iniciar una computadora cargando su sistema operativo. Por ejemplo:

- ¿Has notado que la red está funcionando lentamente? - Esto pregunta si alguien ha observado un problema.

- `The laptop isn't booting properly; we need to check the operating system.` - Esto resalta un problema común donde una computadora no arranca correctamente.
