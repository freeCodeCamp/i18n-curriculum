# --description--

수 $a$의 양의 정수 $b$에 대한 초지수 연산 또는 테트레이션은 $a↑↑b$ 또는 ${}^ba$로 표기하며, 다음과 같이 재귀적으로 정의됩니다:

$a↑↑1 = a$,

$a↑↑(k+1) = a^{(a↑↑k)}$.

예를 들어, $3↑↑2 = 3^3 = 27$이고, 따라서 $3↑↑3 = 3^{27} = 7625597484987$이며, $3↑↑4$는 대략 ${10}^{3.6383346400240996 \times {10}^{12}}$입니다. $1777↑↑1855$의 마지막 8자리를 구하세요.
