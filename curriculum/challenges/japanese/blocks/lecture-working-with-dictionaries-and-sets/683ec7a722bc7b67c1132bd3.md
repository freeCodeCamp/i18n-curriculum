# --description--

Pythonの辞書は、キーと値のペアのコレクションをストアする組み込みのデータ構造です。実際の辞書と非常によく似ていて、単語を検索して対応する意味を見つけるように動作します。

Pythonの辞書では、キーを使って対応する値を見つけます。値を一意のキーに関連付ける必要がある場合に辞書を使うべきです。これは、キーに基づいて値を素早く見つけたいときや、構造化されたデータを表現したいときに役立ちます。

Python辞書の一般的な構文は次の通りです：

```python
dictionary = {
    key1: value1,
    key2: value2
}
```

まず、辞書を保持する変数を見つけます。辞書を変数に代入する必要は必ずしもありませんが、メモリに保持して後でコード内で使うために代入することが非常に一般的です。

次に波括弧（中括弧とも呼ばれます）が続きます。そして波括弧の中にはキーと値のペアがあります。

各キーは値に関連付けられているので、キーを使ってその値にアクセスできます。最後の値を除いて、各値の後には異なるキーと値のペアを区切るためのコンマがあります。キーは辞書内で一意でなければならず、変更不可能なデータ型である必要があります。ただし、値は繰り返されてもよく、任意のデータ型でかまいません。

ここにマルゲリータピザのレシピ情報をストアする辞書の例があります：

```python
pizza = {
    'name': 'Margherita Pizza',
    'price': 8.9,
    'calories_per_slice': 250,
    'toppings': ['mozzarella', 'basil']
}
```

辞書は`pizza`変数に代入されています。4つのキーと値のペアがあります：`name`、`price`、`calories_per_slice`、`toppings`です。

もう一つの方法は、`dict()`コンストラクタを使うことで、キーと値のペアのシーケンスから辞書を構築します。

これはピザの例に対する同等の構文です。`dict()`コンストラクタにタプルのリストを引数として渡します。これらのタプルは最初の要素にキー、2番目の要素に値を含みます。

```python
pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
```

キーと値のペアの値にアクセスするには、ブラケット記法として知られるこの構文を使います。辞書を保持する変数名の後に大括弧が続き、その大括弧の中にアクセスしたいキーを書きます：

```python
dictionary[key]
```

ピザの例では、`name`の値にアクセスしたい場合、変数名`pizza`の後に大括弧を書き、その中に引用符付きのキー`name`を入れます：

```python
pizza['name']
```

これにより次のように評価されます：

```python
'Margherita Pizza'
```

値を更新するには、代入演算子を追加し、その後に新しい値を書くだけです。

キーが辞書に存在しない場合、新しいキーと値のペアが作成されます。Pythonの最近のバージョンでは、辞書は挿入順序を保持します。これは辞書を繰り返す必要があるときに役立ちます：

```python
pizza['name'] = 'Margherita'
```

今、キー`name`の値は`'Margherita'`です：

```python
print(pizza['name']) # 'Margherita'
```

辞書には一般的な操作を行うための便利なメソッドもあります。

`.get()`メソッドはキーに関連付けられた値を取得します。これは先ほど使ったブラケット記法に似ていますが、キーが存在しない場合にエラーにならず、デフォルト値を設定できる点が利点です：

```python
dictionary.get(key, default)
```

この例では、`toppings`キーが存在しない場合、ここで2番目の引数として渡している空のリストというデフォルト値を返します。しかし`toppings`が存在する場合は、その値を返します：

```python
pizza.get('toppings', []) # ['mozzarella', 'basil']
```

`.keys()`と`.values()`メソッドは、それぞれ辞書内のすべてのキーと値を含むビューオブジェクトを返します：

```python
pizza.keys()
# dict_keys(['name', 'price', 'calories_per_slice'])

pizza.values()
# dict_values(['Margherita Pizza', 8.9, 250])
```

ビューオブジェクトは、データの別コピーを作成せずに辞書の内容を見るための方法です。

`.items()`メソッドは、キーと値の両方を含む辞書内のすべてのキーと値のペアを含むビューオブジェクトを返します：

```python
pizza.items()
# dict_items([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250)])
```

`.clear()`メソッドは辞書からすべてのキーと値のペアを削除します：

```python
pizza.clear()
```

`.pop()`メソッドは、最初の引数で指定したキーのキーと値のペアを削除し、その値を返します。キーが存在しない場合は、2番目の引数で指定したデフォルト値を返します。キーが存在せず、デフォルト値を渡さなかった場合は`KeyError`がスローされます：

```python
pizza.pop('price', 10)
pizza.pop('total_price') # KeyError
```

Python 3.7以降のバージョンでは、`.popitem()`メソッドは最後に挿入されたアイテムを削除します：

```python
pizza.popitem()
```

最後に、`.update()`メソッドは別の辞書のキーと値のペアで辞書を更新します。共通のキーがある場合、その値は置き換えられます。

この例では、`pizza`辞書を更新しています。`price`キーは両方に存在するので、その値は`15`に置き換えられます。

しかし`total_time`は新しいので、`pizza`辞書に新しいキーと値のペアとして追加されます：

```python
pizza.update({ 'price': 15, 'total_time': 25 })
```

これは更新された`price`と新しい`total_time`を含む新しい辞書です。価格が`15`になり、`total_time`が新しいキーと値のペアになっていることに注目してください：

```python
{
    'name': 'Margherita Pizza', 
    'price': 15, 
    'calories_per_slice': 250, 
    'toppings': ['mozzarella', 'basil'], 
    'total_time': 25
}
```

これらは最もよく使われる辞書メソッドの一部ですが、他にもたくさんあります。適切なメソッドを選ぶことは複雑な操作を効率的に行うのに役立ちます。
