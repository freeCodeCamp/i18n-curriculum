# --description--

`hasattr` 組み込み関数は、最初の引数にオブジェクトを、2番目の引数に属性名を表す文字列を取ります。指定された属性がオブジェクトにあるかどうかを示すブール値を返します。

これから `__init_subclass__` メソッドを使って、子クラスがインスタンス化された時点で `degree` 属性を持っているかどうかを確認します。

`if` 文を作成して、`cls` が `degree` 属性を持っていない場合にチェックしてください。その場合は `AttributeError` を発生させ、文字列 `f"Cannot create '{cls.__name__}' class: missing required attribute 'degree'"` を使ってカスタムメッセージを提供してください。

その後、ターミナルに表示されたエラーを修正するために、`degree` クラス内に `LinearEquation` クラス属性を宣言してください。この属性は方程式の次数、つまり最高次の \\( x \\) の指数を表します。したがって、整数 `1` を `degree` 属性に代入してください。
