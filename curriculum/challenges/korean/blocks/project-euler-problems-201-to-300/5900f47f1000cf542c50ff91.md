# --description--

10과 서로소인 각 정수 $p > 1$에 대해, 다음 함수가 어떤 양의 정수 $n$에 대해서도 $p$로 나누어지는 성질을 보존하는 양의 나눗셈 배수 곱셈자 $m < p$가 존재합니다:

$f(n) = (\text{마지막 자릿수를 제외한} \\; n) + (\text{마지막 자릿수} \\; n) \times m$

즉, $m$이 $p$의 나눗셈 배수 곱셈자라면, $f(n)$이 $p$로 나누어지려면 그리고 오직 그럴 때에만 $n$도 $p$로 나누어집니다.

($n$이 $p$보다 훨씬 클 때, $f(n)$은 $n$보다 작아지고 $f$를 반복 적용하면 $p$에 대한 곱셈 나눗셈 검사가 됩니다.)

예를 들어, 113의 나눗셈 배수 곱셈자는 34입니다.

$f(76275) = 7627 + 5 \times 34 = 7797$: 76275와 7797 모두 113으로 나누어집니다

$f(12345) = 1234 + 5 \times 34 = 1404$: 12345와 1404 모두 113으로 나누어지지 않습니다

10과 서로소이며 1000보다 작은 소수들의 나눗셈 배수 곱셈자들의 합은 39517입니다. 10과 서로소이며 ${10}^7$보다 작은 소수들의 나눗셈 배수 곱셈자들의 합은 얼마인가요?
