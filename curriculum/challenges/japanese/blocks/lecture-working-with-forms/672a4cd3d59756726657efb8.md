# --interactive--

`button` 要素は、起動されたときに特定のアクションを実行するために使われます。ここに、ボタンのテキストが `button` の `Start Game` 要素の例があります。

```html
<button>Start Game</button>
```

`button` 要素の他の使用例には、フォームの送信、モーダルの表示、サイドメニューの開閉の切り替えなどがあります。`button` 要素には、ボタンが起動されたときの動作を制御する `type` 属性があります。`type` 属性の最初の可能な値は `button` タイプです。ここに、`button` タイプでテキストが `button` の `Show Alert` 要素の例があります。

```html
<button type="button">Show Alert</button>
```

デフォルトでは、ボタンは起動されても何も起こりません。ただし、この場合のようにアラートを表示するなど、ボタンをインタラクティブにするためにJavaScriptコードを追加できます。

`Show Alert` ボタンをクリックすると、画面にアラートが表示されます。例とやり取りするには、インタラクティブエディターを有効にしてください。

**注意**：このインタラクティブな例はJavaScriptを使用していますが、JavaScriptコードを理解する必要はありません。JavaScriptについては今後のモジュールで学習します。

:::interactive_editor

```html
<button type="button">Show Alert</button>
<script src="index.js"></script>
```

```js
const btn = document.querySelector("button");
btn.addEventListener("click", () => alert("You clicked on the alert button"));
```

:::

`type` 属性のもう一つの可能な値は `submit` です。ここに、`button` タイプの `submit` 要素の例があります。

:::interactive_editor

```html
<form action="">
  <label for="email">Email address:</label>
  <input type="email" id="email" name="email" />
  <button type="submit">Submit form</button>
</form>
```

:::

この `form` 要素の中には、ユーザーのメールアドレス用の `label` と `input` 要素があります。ユーザーが送信ボタンをクリックすると、データがサーバーに送信され処理されます。`type` 属性の3つ目の可能な値は `reset` です。ここに、リセットボタンと送信ボタンがある `form` 要素の例があります。

インタラクティブエディターを有効にして、プレビューウィンドウのメール入力に架空のメールアドレスを入力してください。その後、リセットボタンをクリックすると、入力したメールがフィールドから消えます。

:::interactive_editor

```html
<form action="">
  <label for="email">Email address:</label>
  <input type="email" id="email" name="email" />
  <button type="reset">Reset form</button>
  <button type="submit">Submit form</button>
</form>
```

:::

この修正された例では、ユーザーのメールアドレスを収集するために `label` と `input` 要素が使われています。ユーザーがリセットボタンをクリックすると、すべての入力データがクリアされます。リセットボタンは、ユーザーが誤ってデータをリセットしてしまう可能性があるため、通常はあまり良いアイデアではないことに注意してください。また、フォームに複数のボタンがあるとユーザーインターフェースが煩雑になる場合があります。

HTMLでボタンを作成するもう一つの方法は、`input` 要素を使うことです。`input` 要素にも `type` 属性があり、`submit`、`reset`、`button` の値が可能です。ここに、`input` が `type` に設定された `button` 要素の例があります。

:::interactive_editor

```html
<input class="start-btn" type="button" value="Start Game" />
<script src="index.js"></script>
```

```js
document.addEventListener("DOMContentLoaded", () => {
  const btn = document.querySelector(".start-btn");
  btn.addEventListener("click", () => {
    const paraEl = document.createElement("p");
    const bodyEl = document.querySelector("body");

    bodyEl.appendChild(paraEl);
    paraEl.textContent = "The game has started!!!";
  });
});
```

:::

`value` 属性はボタンのテキストを表示するために使われます。では、`input` と `button` 要素を使う違いは何でしょうか？`input` 要素は空要素で、子ノード（テキストなど）を持つことができず、開始タグのみを持ちます。一方、`button` 要素はテキストや画像、アイコンをネストできるため、より柔軟です。
