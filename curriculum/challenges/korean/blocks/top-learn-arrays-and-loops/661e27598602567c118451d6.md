# --description--

`for` 반복문은 코드를 여러 번 실행할 때 사용하는 또 다른 유형의 반복문입니다. `for` 반복문은 반복 횟수를 알고 있을 때 사용합니다. `for` 반복문의 구문은 다음과 같습니다:

```javascript
for (initialization; condition; increment/decrement) {
  // code block to be executed
}
```

예를 들어, 다음 코드 스니펫은 1부터 5까지 숫자를 출력합니다:

```javascript
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
```

`for` 반복문을 사용해 배열을 순회하려면 배열의 length 속성을 사용할 수 있습니다. 예를 들어, 다음 코드 스니펫은 배열의 요소를 출력합니다:

```javascript
const arr = [1, 2, 3, 4, 5];
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}
```
