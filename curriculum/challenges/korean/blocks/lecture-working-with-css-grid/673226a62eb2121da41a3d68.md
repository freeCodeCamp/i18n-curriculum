# --interactive--

`minmax()` 함수는 그리드 트랙 크기의 범위를 정의하며, 행이나 열이 차지할 수 있는 공간의 크기를 지정합니다.

트랙 크기는 `px` (픽셀), `rem`, 또는 `em` 같은 단위와 분수 단위(`fr`)로 설정할 수 있다는 점을 기억하세요.

`minmax()` 함수는 그리드 트랙의 최솟값과 최댓값을 설정할 수 있게 하여 한 단계 더 발전된 기능을 제공합니다.

다음은 `minmax()` 함수의 구문입니다:

```css
minmax(min, max)
```

`min`은 그리드 트랙의 최솟값으로, 픽셀, 백분율, 또는 `auto` 단위로 설정할 수 있습니다. `max`은 그리드 트랙의 최댓값으로, 같은 단위로 설정할 수 있습니다.

이 두 값은 다음과 같이 함께 작동합니다:

- `min` 값은 그리드 트랙이 설정한 크기보다 작아지지 않도록 보장합니다.

- `max` 값은 그리드 트랙이 커질 수 있는 최대 크기를 제한합니다.

그리드 트랙 크기는 콘텐츠와 컨테이너 크기에 따라 `min`와 `max` 값 사이에서 동적으로 조절됩니다.

실제 예제를 살펴보겠습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div>
    <h2>Item 1</h2>
  </div>
  <div>
    <h2>Item 2</h2>
  </div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: minmax(150px, 300px) 1fr;
  gap: 20px;
}

.grid-container > div {
  background: crimson;
  padding: 20px;
  text-align: center;
}
```

:::

여기서 무슨 일이 일어나고 있나요?

첫 번째 열인 `minmax(150px, 300px)`는 항상 최소 `150px`에서 최대 `300px`까지 공간에 따라 크기가 조절됩니다.

반면 두 번째 열인 `1fr`는 추가 열이 없기 때문에 그리드 컨테이너 내 남은 모든 공간을 차지합니다.

`minmax()` 함수가 고정 크기나 `fr` 단위보다 더 유연하다는 점이 장점이며, 이는 적응성과 반응형 디자인에 이상적입니다.
