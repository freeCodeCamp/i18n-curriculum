# --description--

Recientemente aprendiste el patrón `¿Quieres que lo deletree?` para ofrecer deletrear algo. Otra pregunta común es:

`¿Cómo se deletrea?`

Esta pregunta se usa cuando alguien quiere saber cómo se deletrea una palabra, nombre o correo electrónico letra por letra.

Es muy común en conversaciones donde las personas están confirmando detalles de contacto.

# --explanation--

Esteban pregunta `¿Cómo se deletrea?`, así que está preguntando cómo se escribe algo letra por letra. La respuesta correcta debe deletrear la información letra por letra.

Solo una opción hace esto: `j-u-l-i-e-t-a punto c-a-c-e-r-e-s arroba a-n-d-e-s punto com.`

Las otras opciones son declaraciones completas (nombre, nacionalidad, profesión) y no proporcionan ortografía.
