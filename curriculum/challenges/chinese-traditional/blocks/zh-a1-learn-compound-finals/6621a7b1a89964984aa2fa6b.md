# --description--

你已經學會了聲調符號總是標記在韻尾，而非聲母。如果一個音節中有多個母音，以下有三條規則用來決定哪個母音會帶有聲調符號：

- 如果有 `a`，聲調符號會標記在 `a` 上。

- 如果沒有 `a`，但有 `o` 或 `e`，聲調符號就標在出現的那個字母上。這兩個字母不會同時出現在同一個韻母中。

- 如果 `i` 和 `u` 一起出現，聲調符號會標在第二個字母上。

因為 `ai` 包含 `a`，聲調符號放在 `a` 上。 例如，`lái`。
