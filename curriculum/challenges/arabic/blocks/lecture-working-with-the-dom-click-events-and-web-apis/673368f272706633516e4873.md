# --interactive--

قد تكون معتادًا على العمل مع الخصائص في مشاريع HTML وCSS الخاصة بك. لكن هل تعلم أنه يمكنك إضافة خصائص إلى عناصر HTML في ملف JavaScript الخاص بك؟

في هذا الدرس، ستتعلم كيفية العمل مع الطريقة `setAttribute()` لإضافة خصائص إلى عناصر HTML الخاصة بك.

إليك الصيغة النحوية الأساسية:

```js
setAttribute(attribute, value);
```

لنلقِ نظرة على بعض الأمثلة لفهم أفضل لكيفية استخدام الطريقة `setAttribute()`.

في هذا المثال الأول، لدينا عنصر `p` داخل HTML:

```html
<p id="para">I am a paragraph</p>
```

لإضافة خاصية `class`، نحتاج أولًا إلى الوصول إلى عنصر `p` هذا باستخدام الطريقة `getElementById()`. ثم يمكننا استخدام الطريقة `setAttribute()` لإضافة خاصية `class` وتعيين القيمة إلى `my-class`.

نستخدم `outerHTML` لتسجيل العنصر HTML بالكامل بما في ذلك خصائصه.

:::interactive_editor

```html
<p id="para">I am a paragraph</p>
<script src="index.js"></script>
```

```js
const para = document.getElementById("para");
para.setAttribute("class", "my-class");

console.log(`${para.outerHTML}`);
```

:::

إذا كان لديك خاصية موجودة على عنصر HTML، يمكنك تحديث قيمتها باستخدام الطريقة `setAttribute()`.

في هذا المثال لدينا عنصر `div` مع خاصية `class` معينة إلى `my-class`:

```html
<div class="my-class"></div>
```

لتحديث قيمة خاصية `class` إلى example، يمكننا استخدام الطريقة `setAttribute()` مرة أخرى:

:::interactive_editor

```html
<div class="my-class"></div>
<script src="index.js"></script>
```

```js
const divEl = document.querySelector(".my-class");
divEl.setAttribute("class", "example");

console.log(`${divEl.outerHTML}`);
```

:::

فما هي بعض الأمثلة الواقعية لاستخدام الطريقة `setAttribute()`؟

إذا كنت تبني معرض صور ديناميكي، قد تحتاج إلى تحديث خاصية `src` لعنصر صورة عندما ينقر المستخدم على صورة مصغرة.

مثال آخر هو إذا كنت تتعامل مع التحقق من صحة النموذج وتحتاج إلى إضافة خصائص معينة مثل `required` أو `minlength` إلى عنصر `input`.
