# --interactive--

行內程式碼元素用於在文字中表述短程式碼片段。`code` 元素的常見使用情境是技術文章和文件頁面。

這裡是一個使用 `code` 元素來顯示 CSS 程式碼片段的範例：

:::interactive_editor

```html
<p>
  To set the text color to blue in CSS, use the following code:
  <code>color: blue;</code>
</p>
```

:::

在此範例中，CSS 的 `color` 屬性用來將文字顏色設定為 `blue`。透過將這段程式碼片段包裹在 `<code>` 頁籤中，它向瀏覽器傳達該文字是一段行內程式碼。

瀏覽器會為 `code` 元素內的內容套用預設樣式。預設樣式為等寬字型。

`code` 元素用於表述單行程式碼。如果你想表述多行程式碼，你需要將 `code` 元素放置在預格式化文字元素內。

預格式化文字元素用於表述預格式化文字。以下是一個使用預格式化文字元素來顯示 CSS 宣告的範例：

:::interactive_editor

```html
<pre>
  <code>
    body {
      color: red;
    }
  </code>
</pre>
```

:::

使用 `pre` 元素時，你需要注意間距，因為它會如實顯示在 HTML 文件內的內容。

在瀏覽器中，你會看到程式碼向右縮排了幾個空格。如果你更改程式碼範例中的縮排，則你會在螢幕上看到縮排的差異。

當你要在你的 HTML 文件中包含程式碼範例時，你應該為短的行內範例使用 `code` 元素。 

如果你需要顯示較長的程式碼片段，則你需要使用 `pre` 和 `code` 元素。
