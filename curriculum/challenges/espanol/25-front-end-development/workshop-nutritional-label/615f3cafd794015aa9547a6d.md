# --description--

Recuerda que el uso de `h1`, `h2` y etiquetas similares determinan la estructura semántica de tu HTML. Sin embargo, puedes ajustar el CSS de estos elementos para controlar el flujo visual y la jerarquía.

Crea una regla `h1` y establece la propiedad `font-weight` en `800`. Esto hará que tu texto `h1` sea más grueso.

# --hints--

Debes crear un selector `h1`.

```js
assert(new __helpers.CSSHelp(document).getStyle('h1'));
```

Tu selector `h1` debe tener una propiedad `font-weight` establecida en `800`.

```js
assert(new __helpers.CSSHelp(document).getStyle('h1')?.fontWeight === '800');
```
