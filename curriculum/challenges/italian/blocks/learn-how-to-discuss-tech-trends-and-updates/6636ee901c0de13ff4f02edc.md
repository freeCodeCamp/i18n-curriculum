# --description--

Il termine `run code` indica l'azione di eseguire o far funzionare un programma o uno script da parte di un computer. `run` in questo contesto significa avviare il processo e portarlo a termine affinché il programma faccia ciò per cui è stato progettato, che si tratti di calcolare numeri, ordinare dati o altro.

L'espressione `run code without managing servers` si riferisce all'esecuzione di programmi o applicazioni informatiche senza la necessità di gestire i server fisici o virtuali su cui il codice viene eseguito. Questo è un aspetto fondamentale del serverless computing, dove il provider cloud gestisce i server, permettendo agli sviluppatori di concentrarsi più sul codice che sulla gestione dei server.
