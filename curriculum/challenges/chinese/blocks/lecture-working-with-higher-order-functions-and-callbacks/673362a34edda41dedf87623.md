# --interactive--

`map` 方法是 JavaScript 中一个强大且广泛使用的函数，它作用于数组。它旨在通过将给定的函数应用到原始数组的每个元素上来创建一个新数组。

此方法不会修改原始数组，而是返回一个包含对每个元素应用函数结果的新建数组。

下面是对数字数组使用 `map` 方法的示例：

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map((num) => num * 2);
console.log(numbers); // [1, 2, 3, 4, 5]
console.log(doubled); // [2, 4, 6, 8, 10]
```

:::

要创建一个新数组，其中每个数字都翻倍，我们使用了 `map` 方法。`map` 方法接受一个回调函数，该函数会对数组中的每个元素进行调用。

在这种分支中，数组中的每个数字将乘以 `2`。结果将是一个包含数字 `2,4,6,8,10` 的新建数组。

回调函数最多可以接受三个参数。

第一个参数是当前正在处理的数组项。

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element) => {
  console.log("Element:", element);
  return element * 2;
});
```

:::

第二个参数是当前正在处理的元素的索引。

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index) => {
  console.log("Element:", element);
  console.log("Index:", index);
  return element * 2;
});
```

:::

第三个参数是调用 `map` 的数组。

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index, array) => {
  console.log("Element:", element);
  console.log("Index:", index);
  console.log("Array:", array);
  return element * 2;
});
```

:::

理解并高效使用 `map` 方法可以显著提升你在 JavaScript 中操作数组的能力。在后续课程中，我们将深入探讨 `map` 的更高级用法，并探索它如何成为创建动态且高效程序的强大工具。
