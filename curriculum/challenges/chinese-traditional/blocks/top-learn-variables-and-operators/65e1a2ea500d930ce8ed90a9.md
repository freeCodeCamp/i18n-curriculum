# --description--

讓我們來認識超越學校算術的 JavaScript 運算子特性。

通常，運算子 `+` 將數字相加。

但是，如果二進位的 `+` 用於字串，它會將它們合併（串接）：

```js
let s = "my" + "string";
console.log(s); // mystring
```

請注意，如果任何一個運算元是字串，則另一個也會被轉換成字串。

例如：

```js
console.log(1 + '2'); // '12'
console.log('1' + 2); // '12'
```

看，第一個運算元是字串還是第二個運算元是字串並不重要。

這裡有一個更複雜的範例：

```js
console.log(2 + 2 + '1') // "41" and not "221"
```

這裡，運算子依序運作。第一個 `+` 將兩個數字相加，所以它傳回 `4`，接著下一個 `+` 將字串 `1` 加到它上面，因此就像是 `4 + '1' = '41'`。

```js
console.log('1' + 2 + 2) // "122" and not "14"
```

這裡，第一個 `+` 將字串和數字串接起來，所以它傳回 `'12'`，然後下一個 `+` 將 `2` 加到它上面，所以就像 `'12' + 2 = '122'`。

二進位的 `+` 是唯一以此方式支援字串的運算子。其他算術運算子僅能處理數字，且總是將其運算元轉換為數字。

這是減法和除法的示範：

```js
console.log( 6 - '2' ); // 4, converts '2' to a number
console.log( '6' / '2' ); // 3, converts both operands to numbers
```
