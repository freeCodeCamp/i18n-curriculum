# --explanation--

En la frase `limit what other parts can see`, `what` introduce una oración sustantiva, que funciona como el objeto de `limit`.

Una cláusula sustantiva es una cláusula que actúa como un sustantivo dentro de una oración. Puede funcionar como el objeto o el sujeto de una oración. Usualmente comienza con palabras como `that`, `what`, `whether`, `who`, `when`, `why`, `how`, etc.

Las oraciones sustantivas usan el orden normal de las palabras en una afirmación (no el orden de una pregunta). Por ejemplo:

- El equipo está verificando por qué el sistema se bloqueó durante la actualización. - `Why the system crashed during the update` es la cláusula sustantiva que actúa como objeto de `checking`.
   
- `What the function returns` depende de los datos de entrada. - `What the function returns` es una cláusula sustantiva que actúa como sujeto de `depends on`.
