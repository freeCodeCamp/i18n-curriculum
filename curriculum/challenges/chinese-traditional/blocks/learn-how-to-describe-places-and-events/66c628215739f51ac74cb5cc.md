# --explanation--

`When` 在此用來描述導致某事發生的特定條件或情況。 

在 Sarah 的句子中，`when` 介紹了整合開發環境變得棘手的條件：`if they're not set up properly`。例如：

- `The program crashes when it runs out of memory。` - 記憶體不足的狀況會導致程式當機。

- `The printer only works when it's plugged in.` - 只有在插入電源時，印表機才會運作。

在這兩種情況下，`when` 用來說明某事發生的情況。
