# --description--

UTF-8，或稱 UCS 轉換格式 8，是一種廣泛用於網路上的標準化字元編碼。字元編碼是電腦用來將字元儲存為資料的方法。基本上，網頁上的所有文字都是以一或多個位元組儲存的字元序列。在計算機中，位元組是由 8 個位元（二進位數字）組成的資料單位。UTF-8 支援 Unicode 字元集中的每個字元——這包括所有書寫系統、語言及技術符號的字元和符號。以下是使用帶有 `charset` 屬性的 `meta` 元素來設定字元編碼為 `UTF-8` 的範例：

```html
<meta charset="UTF-8" />
```

透過將字元編碼設定為 UTF-8，將確保帶有重音的 `"e"` 字元（`é`）能在頁面上正確顯示。以下是使用 UTF-8 字元編碼的擴充程式碼範例：

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Examples of the UTF-8 encoding</title>
  </head>
  <body>
    <p>Café</p>
  </body>
</html>
```

對於你創建的每個新專案，你應該包含此 `meta` 元素，並將 `charset` 屬性設為 `UTF-8`。
