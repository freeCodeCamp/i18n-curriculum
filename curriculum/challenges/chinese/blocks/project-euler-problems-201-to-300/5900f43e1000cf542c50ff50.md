# --description--

考虑满足 $|x| + |y| ≤ r$ 的整数坐标点 ($x$,$y$) 的集合 $S(r)$。

设 $O$ 为点 (0,0)，$C$ 为点 ($\frac{r}{4}$,$\frac{r}{4}$)。

设 $N(r)$ 为 $S(r)$ 中点 $B$ 的数量，使得三角形 $OBC$ 有一个钝角，即最大角 $α$ 满足 $90°<α<180°$。

例如，$N(4)=24$ 且 $N(8)=100$。

什么是 $N(1\,000\,000\,000)$？
