## --answers--

### --feedback--

Comentários no código não afetam a velocidade com que o código é executado.Eles são para o entendimento humano.

---

Para ajudar futuros desenvolvedores a entender a lógica do código

---

Porque comentários são obrigatórios em todas as linguagens de programação

### --feedback--

Embora os comentários sejam úteis, eles não são um requisito obrigatório em todas as linguagens de programação.

---

Para evitar que o código tenha erros

### --feedback--

Comentários ajudam a entender o código mas não previnem diretamente erros no código.
