## Ankerelemente
Um einen Link in HTML zu erstellen, verwenden Sie das Ankerelement. Ein Ankerelement wird definiert, indem Sie den Text oder ein anderes HTML-Element, das als Link fungieren soll, mit einem `<a>`-Tag umschließen.

Fügen Sie Folgendes in das `body` der von Ihnen erstellten `index.html`-Seite ein und öffnen Sie sie im Browser:

```html
<a>click me</a>
```

Sie haben vielleicht bemerkt, dass das Klicken auf diesen Link nichts bewirkt. Das liegt daran, dass ein Ankerelement für sich allein nicht weiß, wohin Sie verknüpfen möchten. Sie müssen ihm ein Ziel angeben. Das machen Sie, indem Sie ein HTML-Attribut verwenden.

Ein HTML-Attribut gibt einem HTML-Element zusätzliche Informationen und steht immer im öffnenden Tag des Elements. Ein Attribut besteht normalerweise aus zwei Teilen: einem Namen und einem Wert; jedoch benötigen nicht alle Attribute einen Wert. In Ihrem Fall müssen Sie dem öffnenden Ankerelement ein `href`-Attribut (hyperlink reference) hinzufügen. Der Wert des `href`-Attributs ist das Ziel, zu dem Ihr Link führen soll.

Fügen Sie dem zuvor erstellten Ankerelement das folgende `href`-Attribut hinzu und versuchen Sie erneut, darauf zu klicken. Vergessen Sie nicht, den Browser zu aktualisieren, damit die neuen Änderungen übernommen werden.

```html
<a href="https://www.theodinproject.com/about">click me</a>
```

Standardmäßig sieht jeder Text, der mit einem Anker-Tag ohne `href`-Attribut umschlossen ist, wie normaler Text aus. Wenn das `href`-Attribut vorhanden ist, zeigt der Browser den Text in blauer Farbe und unterstrichen an, um zu signalisieren, dass es sich um einen Link handelt.

Es ist erwähnenswert, dass Sie Ankerelemente verwenden können, um auf jede Art von Ressource im Internet zu verknüpfen, nicht nur auf andere HTML-Dokumente. Sie können auf Videos, PDF-Dateien, Bilder und so weiter verknüpfen, aber meistens werden Sie auf andere HTML-Dokumente verknüpfen.
