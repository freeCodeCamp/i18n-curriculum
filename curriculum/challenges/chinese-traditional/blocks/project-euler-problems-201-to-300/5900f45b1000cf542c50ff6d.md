# --description--

使用「Blum Blum Shub」偽隨機數產生器創建一組數字序列：

$$
s_0 = 14025256 \\\\
s_{n + 1} = {s_n}^2 \\; mod \\; 20\\,300\\,713
$$

將這些數字 $s_0s_1s_2\ldots$ 串接起來以創建一個無限長度的字串 $w$。然後，$w = 14025256741014958470038053646\ldots$

對於正整數 $k$，如果不存在 $w$ 的子字串，其數字和相等於 $k$，則定義 $p(k)$ 為零。如果存在至少一個 $w$ 的子字串，其數字和相等於 $k$，則定義 $p(k) = z$，其中 $z$ 是最早出現該子字串的起始位置。

例如：

子字串 1、14、1402、… 的各自數字和分別相等於 1、5、7、…，起始於位置 1，因此 $p(1) = p(5) = p(7) = \ldots = 1$。

子字串 4、402、4025、… 的各自數字和分別相等於 4、6、11、…，起始於位置 2，因此 $p(4) = p(6) = p(11) = \ldots = 2$。

子字串 02、0252、… 的各位數字和分別相等於 2、9、…，起始於位置 3，因此 $p(2) = p(9) = \ldots = 3$。

請注意，從位置 3 開始的子字串 025，其數字總和為 7，但之前有一個子字串（從位置 1 開始）其數字總和也為 7，所以 $p(7) = 1$，而不是 3。

我們可以驗證，對於 $0 &lt; k ≤ {10}^3$，$\sum p(k) = 4742$。

求 $\sum p(k)$，其中 $0 < k ≤ 2 \times {10}^{15}$。
