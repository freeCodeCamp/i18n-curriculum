# --description--

`Overwhelming` significa sentir que algo é demais para lidar.

Por exemplo, `The amount of information was overwhelming.` (significando que era muito!)

`Effective` é um adjetivo que descreve algo que produz um resultado desejado. Como, `An effective method`.

`Effectively` é um advérbio, significando de uma maneira `effective`. Por exemplo, `He completed the task effectively.`
