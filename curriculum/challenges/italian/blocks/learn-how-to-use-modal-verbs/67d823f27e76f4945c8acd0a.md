# --explanation--

`Should` è un verbo modale che esprime un consiglio, una raccomandazione o la migliore linea d’azione — ma non un obbligo.

- `You should update your password regularly.` - È una buona idea cambiare spesso la password per motivi di sicurezza.

- `You should check for errors in the code before running it.` - È una buona pratica trovare gli errori prima di eseguire il codice.

Dato che Jake dice `we should focus on the critical systems first`, sta dando una forte raccomandazione, ma il team non è strettamente obbligato a farlo.
