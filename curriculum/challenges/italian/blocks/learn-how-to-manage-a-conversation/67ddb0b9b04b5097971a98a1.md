# --explanation--

Nel linguaggio parlato informale, spesso si omettono parole quando il significato resta chiaro. In questo caso, `Anything else we should discuss?` è una forma abbreviata di `Is there anything else we should discuss?`. Questo tipo di omissione è comune nelle conversazioni. Un altro esempio:

`Need any help?` è un modo più breve per dire `Do you need any help?` – il soggetto e il verbo ausiliare sono omessi, ma il significato rimane lo stesso.
