# --explanation--

在**转述**中，短语 `said that` 用于以不同的方式传达某人所说的话，而不是使用他们的原话。单词 `that` 是可选的，且在日常讲话中经常省略。例如：

- 直接引语：`We need to update the software.`

- 转述：`He said that we need to update the software.` 或 `He said we need to update the software.`

两个版本都是正确的，意思相同。
