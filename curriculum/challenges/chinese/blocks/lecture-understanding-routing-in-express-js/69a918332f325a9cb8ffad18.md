# --description--

在 Express 中，**路由处理程序**（也称为路由回调函数或控制器函数）是定义当客户发送请求到特定路由时会发生什么的函数。

路由处理程序处理传入的请求，执行任何必要的逻辑，比如查询数据库或运行计算，然后将响应发送回客户。

让我们用一个简单的例子来分解说明（实际上你一直都在使用它）：

```javascript
app.get("/menu", (req, res) => {
  res.send("Welcome to the menu!");
});
```

`.get()` 方法是 Express 的路由方法之一。它监听指定路由上的 **GET** 请求。在这个例子中，它监听对 `/menu` 的请求。

函数 `(req, res) => { ... }` 是 **路由处理程序**。每当客户发送 GET 请求到 `/menu` 时，它就会运行。

在处理程序内部：

* `req`（请求）是一个包含有关传入请求的信息的对象，比如路由参数、查询字串、头部和体数据。
    
* `res`（响应）是一个用于向客户发送响应的对象。
    

在这种情况下，`res.send("Welcome to the menu!")` 向客户发送纯文本响应。

当用户在浏览器中导航到 `/menu` ——或者向该路由发送 GET 请求时——服务器执行此处理程序并响应消息 `Welcome to the menu!`。

路由处理程序是 Express 应用的核心。它们允许你定义你的网页应用的不同路由将如何响应请求。你还可以向这些处理程序添加逻辑，比如处理表单数据、与数据库交互或执行验证。
