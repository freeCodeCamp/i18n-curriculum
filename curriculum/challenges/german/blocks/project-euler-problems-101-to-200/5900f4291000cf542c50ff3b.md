# --description--

Die Hyperexponentiation oder Tetration einer Zahl $a$ mit einer positiven ganzen Zahl $b$, bezeichnet durch $a↑↑b$ oder ${}^ba$, ist rekursiv definiert durch:

$a↑↑1 = a$,

$a↑↑(k+1) = a^{(a↑↑k)}$.

So gilt zum Beispiel $3↑↑2 = 3^3 = 27$, daher ist $3↑↑3 = 3^{27} = 7625597484987$ und $3↑↑4$ ist ungefähr ${10}^{3.6383346400240996 \times {10}^{12}}$. Finden Sie die letzten 8 Ziffern von $1777↑↑1855$.
