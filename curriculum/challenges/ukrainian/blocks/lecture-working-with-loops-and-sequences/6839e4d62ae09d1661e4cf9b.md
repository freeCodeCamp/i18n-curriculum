# --description--

Протягом останніх кількох уроків ви звикали працювати з циклами ось так:

```py
even_numbers = []

for num in range(21):
    if num % 2 == 0:
        even_numbers.append(num)

print(even_numbers)
```

Цей приклад створює новий порожній список з назвою `even_numbers` і проходить циклом по послідовності чисел між `0` та `20`. Всередині циклу є умова, яка перевіряє, чи має поточне число остачу `0` при діленні на `2`. Це використовується, щоб визначити, чи є число парним. Якщо умова `True`, тоді поточне `num` додається в кінець списку `even_numbers`. Нарешті, ми виводимо список `even_numbers` у консоль.

Хоча цей код працює, існує більш лаконічний спосіб написати це, використовуючи list comprehension. List comprehension дозволяє створити новий список в одному рядку, поєднуючи цикл і умову безпосередньо в квадратних дужках. Це робить код коротшим і часто легшим для читання.

Ось перероблений приклад з попереднього коду з використанням квадратних дужок:

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

У цьому переробленому прикладі список `even_numbers` створюється за допомогою одного рядка коду. List comprehension проходить по числах від `0` до `20` і включає лише ті, що діляться на `2` без остачі. Такий підхід компактніший і усуває потребу в окремому циклі та блоці умови.

Давайте розглянемо ще один приклад, щоб краще зрозуміти, як працює list comprehension:

```py
numbers = [1, 2, 3, 4, 5]
result = [(num, 'Even') if num % 2 == 0 else (num, 'Odd') for num in numbers]
print(result)
```

У цьому прикладі у нас є список чисел, і ми хочемо створити новий список кортежів, які вказують, чи є числа парними чи непарними. У першій частині list comprehension ми використовуємо інструкцію `if`, щоб перевірити, чи ділиться число на `2` без остачі. Якщо так, результатом є кортеж з цим числом і словом `Even`. Інакше результатом є кортеж з числом і словом `Odd`.

Ось як виглядає результат, виведений у консоль:

```py
[(1, 'Odd'), (2, 'Even'), (3, 'Odd'), (4, 'Even'), (5, 'Odd')]
```

Ще один спосіб створити список на основі існуючого ітерованого об’єкта — це функція `filter()`. Ось приклад створення нового списку лише зі словами довжиною більше чотирьох символів:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

Функція `filter()` використовується для вибору елементів з ітерованого об’єкта, які відповідають певній умові. Функція `filter()` приймає функцію та ітерований об’єкт як аргументи. У цьому прикладі ми передаємо функцію `is_long_word` у функцію `filter()`, щоб перевірити, чи кількість символів у поточному слові більша за `4`. Усі слова, які мають довжину символів більшу за `4`, додаються до нового списку і призначаються змінній `long_words`.

Окрім функції `filter()`, є ще кілька функцій, які корисні при роботі зі списками. Ще одна функція, про яку варто знати, — це функція `map()`, яка приймає ітерований об’єкт і застосовує функцію до кожного його елемента. Ось приклад використання функції `map()` для перетворення списку температур із Цельсія у Фаренгейти:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

Як і функція `filter()`, `map()` приймає функцію та ітерований об’єкт як аргументи. Функція `to_fahrenheit` приймає температуру і конвертує її з Цельсія у Фаренгейти.

Остання функція, яку ми розглянемо, — це функція `sum()`. Ця функція використовується для отримання суми з ітерованого об’єкта, наприклад списку або кортежу. Ось приклад використання функції `sum()`:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

Ви також можете передати необов’язковий аргумент `start`, який встановлює ініціальне значення для сумування. Ось оновлений приклад із використанням аргументу `start` як позиційного аргументу:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

Ви також можете вибрати використання аргументу `start` як ключового слова ось так:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

Обидві версії дадуть однаковий результат, але ключовий аргумент є трохи більш явним.

List comprehension, а також інші функції, як `map()`, `filter()` і `sum()`, можуть спочатку здатися трохи заплутаними. Але з достатньою практикою і часом ви почнете почуватися впевненіше, використовуючи їх у своїх програмах на Python.
