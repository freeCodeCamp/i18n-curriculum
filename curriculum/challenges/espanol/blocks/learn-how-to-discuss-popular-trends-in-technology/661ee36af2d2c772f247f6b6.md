# --description--

Cuando agregas `'s` después de un sustantivo, indica que algo pertenece o está conectado a ese sustantivo. Es como agregar una etiqueta para decir que esta cosa pertenece a esta persona u objeto. `'s` indica la propiedad o posesión.

Ejemplos: `This is Tom's camera.` (esta cámara pertenece a Tom). `That's today's newspaper.` (ese periódico tiene las noticias de hoy).

# --questions--
