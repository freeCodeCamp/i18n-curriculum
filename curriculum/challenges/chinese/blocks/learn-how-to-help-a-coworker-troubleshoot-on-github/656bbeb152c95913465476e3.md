# --description--

编程中的`Compile`是指将人们编写的代码转换成计算机能够理解并使用的格式。

就像把一种语言写的食谱翻译成另一种语言，以便别人可以烹饪这道菜。 

例如，`When we compile Java code, it changes into a form that the computer can run.`
