# --explanation--

Un `targeted attack` è un attacco informatico deliberato contro una persona, un'azienda o un sistema specifico. Per esempio:

`Hackers launched a targeted attack against the financial sector.` - Questo significa che gli aggressori avevano un obiettivo specifico invece di diffondere malware in modo casuale.

Sospetta che qualcuno abbia pianificato l'attacco specificamente invece che si tratti di una minaccia informatica casuale.
