## 사용자 동작 가상 클래스

- **가상 클래스 정의**: 특정 상태나 위치에 따라 요소를 선택할 수 있게 해주는 특별한 CSS 키워드입니다.
- **사용자 동작 가상 클래스**: 사용자 상호작용에 따라 요소의 모양을 변경할 수 있게 해주는 특별한 키워드로, 전반적인 사용자 경험을 향상시킵니다.
- **`:active` 가상 클래스**: 버튼 클릭처럼 요소의 활성 상태를 선택할 수 있게 해줍니다.
- **`:hover` 가상 클래스**: 요소의 호버 상태를 정의합니다.
- **`:focus` 가상 클래스**: 키보드 탐색이나 사용자가 폼 입력 필드에 클릭했을 때 요소가 포커스를 얻으면 스타일을 적용합니다.
- **`:focus-within` 가상 클래스**: 요소나 그 자손 요소 중 하나가 포커스를 가질 때 스타일을 적용합니다.

## 입력 가상 클래스

- **입력 가상 클래스**: 사용자 상호작용 전후 상태에 따라 HTML `input` 요소를 대상으로 스타일을 적용할 때 사용합니다.
- **`:enabled` 가상 클래스**: 현재 활성화된 폼 버튼이나 기타 요소를 대상으로 합니다.
- **`:disabled` 가상 클래스**: 비활성화된 상태의 인터랙티브 요소에 스타일을 적용합니다.
- **`:checked` 가상 클래스**: 체크된 상태임을 사용자에게 표시할 때 사용합니다.
- **`:valid` 가상 클래스**: 유효성 검사 기준을 충족하는 입력 필드를 대상으로 합니다.
- **`:invalid` 가상 클래스**: 유효성 검사 기준을 충족하지 못하는 입력 필드를 대상으로 합니다.
- **`:in-range` 및 `:out-of-range` 가상 클래스**: 값이 지정된 범위 내에 있거나 범위를 벗어난 요소에 스타일을 적용합니다.
- **`:required` 가상 클래스**: `input` 요소 중 `required` 속성이 있는 요소를 대상으로 하며, 사용자가 폼을 제출하려면 반드시 입력해야 함을 알립니다.
- **`:optional` 가상 클래스**: 필수 입력이 아니어서 비워둘 수 있는 입력 요소에 스타일을 적용합니다.
- **`:autofill` 가상 클래스**: 브라우저가 저장된 데이터를 자동으로 채워 넣은 입력 필드에 스타일을 적용합니다.

## 위치 가상 클래스

- **위치 가상 클래스**: 현재 문서 내에서 대상이 되는 링크와 요소를 스타일링할 때 사용합니다.
- **`:any-link` 가상 클래스**: `:link` 및 `:visited` 가상 클래스의 조합으로, 방문 여부와 상관없이 href 속성이 있는 모든 앵커 요소를 선택합니다.
- **`:link` 가상 클래스**: 웹페이지 내 모든 방문하지 않은 링크를 대상으로 하며, 사용자가 클릭하기 전 링크를 다르게 스타일링할 때 사용합니다.
- **`:local-link` 가상 클래스**: 같은 문서를 가리키는 링크를 대상으로 하며, 내부 링크와 외부 링크를 구분할 때 유용합니다.
- **`:visited` 가상 클래스**: 사용자가 방문한 링크를 대상으로 합니다.
- **`:target` 가상 클래스**: URL 조각의 대상이 되는 요소에 스타일을 적용할 때 사용합니다.

## 트리 구조 가상 클래스

- **트리 구조 가상 클래스**: 문서 트리 내 위치에 따라 요소를 선택하고 스타일링할 수 있게 해줍니다.
- **`:root` 가상 클래스**: 보통 문서의 최상위 `html` 요소를 선택하며, 문서 전체에 공통 스타일을 적용할 때 사용합니다.        
- **`:empty` 가상 클래스**: 자식 요소가 공백 문자뿐인 빈 요소도 문서 트리에 포함되므로, 빈 요소를 선택하는 `:empty` 가상 클래스가 있습니다.
- **`:nth-child(n)` 가상 클래스**: 부모 내 위치에 따라 요소를 선택할 수 있습니다.
- **`:nth-last-child(n)` 가상 클래스**: 끝에서부터 요소를 세어 선택할 수 있습니다.
- **`:first-child` 가상 클래스**: 부모 요소나 문서 내 첫 번째 요소를 선택합니다.
- **`:last-child` 가상 클래스**: 부모 요소나 문서 내 마지막 요소를 선택합니다.
- **`:only-child` 가상 클래스**: 부모 요소나 문서 내 유일한 요소를 선택합니다.
- **`:first-of-type` 가상 클래스**: 부모 내 특정 요소 유형 중 첫 번째 발생 요소를 선택합니다.
- **`:last-of-type` 가상 클래스**: 부모 내 특정 요소 유형 중 마지막 발생 요소를 선택합니다.
- **`:nth-of-type(n)` 가상 클래스**: 같은 유형의 형제 요소 중 위치에 따라 특정 요소를 선택할 수 있습니다.
- **`:only-of-type` 가상 클래스**: 부모 내에서 유일한 유형의 요소일 경우 선택합니다.

## 함수형 가상 클래스

- **함수형 가상 클래스**: 일반 가상 클래스가 상태에 따라 요소를 선택하는 것과 달리 (예: `:hover`, `:focus`), 함수형 가상 클래스는 인수를 받아 더 복잡한 조건이나 관계에 따라 요소를 선택합니다.
- **`:is()` 가상 클래스**: 선택자 목록(예: `ol`, `ul`)을 인수로 받아 목록 중 하나와 일치하는 요소를 선택합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<p class="example">This text will change color.</p>
<p>This text will not change color.</p>
<p>This text will not change color.</p>
<p class="this-works-too">This text will change color.</p>
```

```css
p:is(.example, .this-works-too) {
    color: red;
}
```

:::

- **`:where()` 가상 클래스**: 선택자 목록(예: `ol`, `ul`)을 인수로 받아 목록 중 하나와 일치하는 요소를 선택합니다. `:is`와 `:where`의 차이는 후자가 특이성이 0이라는 점입니다.

```css
:where(h1, h2, h3) {
    margin: 0;
    padding: 0;
}
```

- **`:has()` 가상 클래스**: 자식 요소가 선택자 목록에 지정된 요소를 포함하는 경우 스타일을 적용할 수 있어 `"parent"` 선택자라고도 불립니다.

```css
article:has(h2) {
    border: 2px solid hotpink;
}
```

- **`:not()` 가상 클래스**: 제공된 선택자와 일치하지 않는 요소를 선택할 때 사용합니다.

```css
p:not(.example) {
  color: blue;
}
```

## 가상 요소

- **`::before` 가상 요소**: `content` 속성을 사용해 요소 바로 앞에 아이콘 같은 장식용 콘텐츠를 삽입합니다.
- **`::after` 가상 요소**: `content` 속성을 사용해 요소 바로 뒤에 아이콘 같은 장식용 콘텐츠를 삽입합니다.
- **`::first-letter` 가상 요소**: 요소 내용의 첫 글자를 대상으로 스타일링할 수 있습니다.
- **`::marker` 가상 요소**: 목록 항목의 마커(글머리표나 번호)를 선택해 스타일링할 수 있습니다.
