# --description--

CSS tratta ogni elemento HTML come un proprio riquadro, che di solito viene chiamato <dfn>modello a riquadro CSS</dfn>. Gli elementi a livello di blocco iniziano automaticamente su una nuova riga (pensa a intestazioni, paragrafi e div), mentre gli elementi in linea si trovano all’interno del contenuto circostante (come immagini o span). Il layout predefinito degli elementi in questo modo si chiama <dfn>flusso normale</dfn> di un documento, ma CSS offre la proprietà position per sovrascriverlo.

Quando la posizione di un elemento è impostata su `relative`, puoi specificare come CSS dovrebbe spostarlo *relativamente* alla sua posizione attuale nel flusso normale della pagina. Si abbina alle proprietà offset CSS `left` o `right`, e `top` o `bottom`. Queste indicano di quanti pixel, percentuali o em spostare l’elemento *lontano* da dove si trova normalmente. L’esempio seguente sposta il paragrafo di 10 pixel dal basso:

```css
p {
  position: relative;
  bottom: 10px;
}
```

Cambiare la posizione di un elemento in relativa non lo rimuove dal flusso normale: gli altri elementi intorno si comportano ancora come se quell’elemento fosse nella sua posizione predefinita.

**Nota:** Il posizionamento ti dà molta flessibilità e potere sul layout visivo di una pagina. È importante ricordare che, indipendentemente dalla posizione degli elementi, il markup HTML sottostante dovrebbe essere organizzato e avere senso se letto dall’alto verso il basso. Questo è il modo in cui gli utenti con disabilità visiva (che si affidano a dispositivi assistivi come i lettori di schermo) consultano il tuo contenuto.

Cambia la `position` del `h2` in `relative` e usa un offset CSS per spostarlo di 15 pixel lontano dal `top` rispetto a dove si trova nel flusso normale. Nota che non ci sono effetti sulle posizioni degli elementi h1 e p circostanti.

# --hints--

L’elemento `h2` dovrebbe avere la proprietà `position` impostata su `relative`.

```js
const h2Element =document.querySelector('h2')
const h2Style = window.getComputedStyle(h2Element);
assert.equal(h2Style?.position, 'relative');
```

Il tuo codice dovrebbe usare un offset CSS per posizionare relativamente il `h2` a 15px di distanza dal `top` rispetto a dove si trova normalmente.

```js
const h2Element =document.querySelector('h2')
const h2Style = window.getComputedStyle(h2Element);
assert.equal(h2Style?.top, '15px');
```
