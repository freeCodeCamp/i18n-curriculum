`Absolutely, I'll review the code today.` – Das bedeutet, Sie stimmen vollkommen zu, es zu tun.

`The IT team will prioritize server repairs before installing updates.` – Das bedeutet, dass Serverreparaturen zuerst abgeschlossen werden, weil sie dringender sind.
