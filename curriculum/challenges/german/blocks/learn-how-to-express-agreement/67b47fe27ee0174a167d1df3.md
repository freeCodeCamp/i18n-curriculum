# --explanation--

`Tend to` ist ein Ausdruck, der verwendet wird, um etwas zu beschreiben, das regelmäßig passiert oder wahrscheinlich passieren wird. Zum Beispiel:

`She tends to be early for meetings.` – Das bedeutet, dass es für sie eine Gewohnheit oder ein regelmäßiges Ereignis ist, zu Meetings früh zu kommen.

`Lag` bezieht sich auf eine Verzögerung oder langsame Reaktionszeit. Zum Beispiel:

`The video lags when there is a poor internet connection.` – Das bedeutet, dass das Video aufgrund der Verbindung verzögert oder langsam ist.

`Switch` bedeutet, von einer Sache zur anderen zu wechseln. Zum Beispiel:

`Users can switch between different tabs in the browser.` – Das bedeutet, dass Benutzer beim Surfen von einem Tab zum anderen wechseln können.
