# --explanation--

`This way` usa `this` para se referir ao método que Bob acabou de descrever.

`Every new person` usa `every` quando fala sobre todos os indivíduos juntos (como um grupo inteiro).

`No one` significa zero pessoas - nenhuma pessoa sequer.

Tanto `each` quanto `every` descrevem todas as pessoas, mas a sensação é ligeiramente diferente:

- `Each` foca em indivíduos separadamente: `Each developer has a mentor.`

- `Every` foca no grupo como um todo: `Every developer gets support.`
