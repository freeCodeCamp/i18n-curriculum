# --description--

對於正整數 $k$，定義 $d(k)$ 為 $k$ 在其通常十進位表現中的各位數字之和。因此 $d(42) = 4 + 2 = 6$。

對於正整數 $n$，定義 $S(n)$ 為滿足以下條件且小於 ${10}^n$ 的正整數 $k$ 的數量：

- $k$ 可被 23 整除，且，
- $d(k) = 23$。

你已知 $S(9) = 263\\,626$ 和 $S(42) = 6\\,377\\,168\\,878\\,570\\,056$。

求 $S({11}^{12})$ 並給出你的答案 $\bmod {10}^9$。
