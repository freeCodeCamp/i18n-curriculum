# --description--

La expresión `thanks for` se utiliza para mostrar gratitud por acciones específicas o ayuda recibida. La estructura comúnmente va seguida de un formulario `-ing` de un verbo (un gerundio) o un sustantivo para especificar para qué es el agradecimiento.

Ejemplos:

`Thanks for helping me with my homework.`

`Thanks for the advice.`
