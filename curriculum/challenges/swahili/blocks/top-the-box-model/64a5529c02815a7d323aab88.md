# --description--

Kuwa na uwezo wa kukagua na kutafuta makosa katika HTML na CSS yako ni muhimu sana kwa ukuzaji wa mbele (front-end). Funzo hili litatupeleka kupitia Chrome Dev Tools, ambazo zinakuwezesha kuona taarifa za kina kuhusu vipengele vyako na sheria za CSS, pamoja na kusaidia katika kutafuta na kurekebisha matatizo katika msimbo wako.

Ili kufungua kipengele cha ukaguzi, unaweza kubofya kulia kwenye kipengele chochote cha ukurasa wa wavuti na bofya “Inspect” au bonyeza F12. Tafadhali fanya hivyo sasa hivi ili kuona HTML na CSS zinazotumika kwenye ukurasa huu.

Usijisikie kuzidiwa na zana zote unazoziona sasa! Kwa funzo hili, tunataka makini zaidi kwenye sehemu za Elements na Styles.

# --assignment--

Jichezee na Chrome Dev Tools na uone kama unaweza kujibu swali lifuatalo.
