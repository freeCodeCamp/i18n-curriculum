# --explanation--

`Several` bedeutet mehr als ein paar, aber nicht alle. Zum Beispiel:

`Several users reported issues with the new software.` – Das bedeutet, dass mehr als zwei Nutzer Probleme hatten, aber nicht alle.

`To target` bedeutet, etwas auf jemanden oder etwas zu richten oder zu zielen. Zum Beispiel:

`The hackers targeted customers by sending fake emails.` – Das bedeutet, dass sie speziell versucht haben, die Kunden hereinzulegen.
