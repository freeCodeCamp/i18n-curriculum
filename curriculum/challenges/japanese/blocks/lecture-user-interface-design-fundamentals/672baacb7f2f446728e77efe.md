# --description--

ホワイトスペースとは、画像、テキスト、ボタンなどの要素の周りにあるあらゆる種類の空白のことを指します。ホワイトスペースは、ページ上の要素間のバランスを作り出すのに役立つため、デザインにおいて重要です。

ホワイトスペースがデザインで効果的に使われている例を見てみましょう。

例えば、コールトゥアクション（CTA）ボタンを考えてみます。CTAは、ユーザーにニュースレターの登録や購入など特定の行動を促すために使われます。

freeCodeCampのホームページでは、CTAボタンが他の要素から視覚的に分離されています。下の画像は、このボタンの周りに一定のスペースがある様子を示しています。

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-importance-of-whitespace-in-design-1.png" alt="Call-to-action button on the freeCodeCamp homepage with yellow background and black text reading: Get started (it's free). The button is centered on its own line with ample white space above and below.">

ホワイトスペースを効果的に使うことで、CTAボタンをより目立たせ、ユーザーがクリックしやすくなります。

では、ホワイトスペースのさまざまな種類を詳しく見ていきましょう。

最初の例では、マクロホワイトスペースとアクティブホワイトスペースの両方が使われています。マクロホワイトスペースは、画像、テキストブロック、ボタンなどの大きな要素間のスペースです。

アクティブホワイトスペースは、ユーザーの視線を誘導し、ページ上の特定の要素に注意を引くために意図的に作られたスペースです。

アクティブホワイトスペースとは対照的に、パッシブホワイトスペースもあります。パッシブホワイトスペースは、ページ上のすべての要素が配置された後に残るスペースです。

もう一つのホワイトスペースの種類はマイクロホワイトスペースです。これは、テキストの1行内の個々の文字間のスペースです。

下の画像は、freeCodeCampのホームページのよくある質問セクションを示しており、この間隔によって各質問と回答が読みやすくなっています。

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-importance-of-whitespace-in-design-2.png" alt="The Frequently Asked Questions section on the freeCodeCamp homepage, with text spaced sufficiently between each letter.">

マイクロホワイトスペースは、読みやすさと判読性を向上させ、ユーザーがコンテンツをスキャンして理解しやすくするために重要です。

ウェブページをデザインするときは、常に近接の法則を意識してください。この法則は、近くにある要素は関連していると認識され、離れている要素は関連していないと認識されることを示しています。

ホワイトスペースを使って関連する要素をグループ化し、ユーザーがページのコンテンツをナビゲートしやすくすることができます。
