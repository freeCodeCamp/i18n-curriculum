# --description--

Casi todos los elementos de una página HTML no son más que fragmentos de contenido envueltos en etiquetas HTML de apertura y cierre.

Las etiquetas de apertura indican al navegador que este es el inicio de un elemento HTML. Están compuestas por una palabra clave encerrada entre corchetes angulares `<>`. Por ejemplo, una etiqueta de párrafo de apertura se ve así: `<p>`.

Las etiquetas de cierre indican al navegador dónde termina un elemento. Son prácticamente iguales que las etiquetas de apertura; la única diferencia es que llevan una barra inclinada antes de la palabra clave.  For example, a closing paragraph tag looks like this: `</p>`.
