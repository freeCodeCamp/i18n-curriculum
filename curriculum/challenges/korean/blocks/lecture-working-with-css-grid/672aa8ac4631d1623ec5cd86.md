# --interactive--

CSS Grid는 웹 개발자가 복잡하고 반응형인 웹 페이지 레이아웃을 쉽게 만들 수 있게 해주는 강력한 레이아웃 시스템입니다.

방 안에 가구를 배치하는 것을 상상해 보세요 — CSS Grid는 바닥에 보이지 않는 격자가 있어서 원하는 위치에 정확히 배치할 수 있도록 도와줍니다.

웹사이트를 만들 때, 페이지에 다양한 요소를 배치해야 할 때가 많습니다.

CSS Grid가 나오기 전에는 특히 복잡한 레이아웃을 만들 때 어려움이 있었습니다. CSS Grid는 웹 페이지를 행과 열로 나누어 격자 구조를 만들어 이 과정을 단순화합니다.

컨테이너 `div` 안에 여러 항목이 중첩된 상태라고 상상해 보세요:

:::interactive_editor

```html
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

:::

이 요소들을 그리드 형식으로 스타일링하고 싶다면 `display`를 `grid`로 설정하고 다음과 같이 열을 적용할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-gap: 20px;
}

.item {
  background-color: lightgray;
  padding: 20px;
  text-align: center;
  border: 1px solid #ccc;
}
```

:::

이 코드는 브라우저에게 세 개의 같은 너비 열로 구성된 그리드를 만들라고 지시합니다. 이것이 `1fr 1fr 1fr`가 의미하는 바이며, 각 그리드 항목 사이에 20픽셀 간격을 추가합니다.

이제 이렇게 궁금할 수 있습니다: "Flexbox는요? 레이아웃에 사용되지 않나요?"

맞습니다! Flexbox도 또 다른 CSS 레이아웃 모델로 매우 유용합니다. 하지만 몇 가지 중요한 차이가 있습니다.

Flexbox는 1차원이고, Grid는 2차원입니다. 즉, Flexbox는 한 줄이나 한 열로 배치할 때 뛰어나고, Grid는 행과 열 모두를 포함하는 레이아웃을 만들 때 탁월합니다.

Flexbox는 콘텐츠 중심으로, 콘텐츠에 따라 레이아웃을 조정합니다. 반면 Grid는 레이아웃 중심으로, 먼저 레이아웃을 만들고 그 안에 항목을 배치할 수 있습니다. Grid는 배치에 대해 더 정밀한 제어를 제공합니다. 항목이 어느 행과 열에 위치할지 정확히 지정할 수 있습니다.

비교를 위해 Flexbox 예제를 보세요:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
}
```

:::

이 코드는 주축을 따라 항목들이 고르게 간격을 두고 배치된 플렉스 컨테이너를 만듭니다.

Grid와 Flexbox는 각각 장점이 있으며, 종종 가장 좋은 레이아웃은 두 가지를 조합해서 사용합니다. 전체 페이지 레이아웃에는 Grid를, 각 그리드 영역 내 항목 정렬에는 Flexbox를 사용할 수 있습니다.

요약하자면, CSS Grid는 정밀한 2차원 레이아웃을 가능하게 하는 강력한 도구입니다. 처음에는 복잡해 보일 수 있지만, 연습하면 반응형이고 복잡한 웹 레이아웃을 만드는 데 없어서는 안 될 도구가 됩니다.
