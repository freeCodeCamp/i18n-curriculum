# --description--

Щоб попрактикуватися у використанні посилань та зображень протягом цього уроку, вам потрібен HTML проєкт для роботи.

- Створіть нову директорію з назвою `odin-links-and-images`.

- У цій директорії створіть новий файл з назвою `index.html`.

- Заповніть звичайний шаблонний код HTML.

- Нарешті, додайте наступний `h1` до `body`: `<h1>Homepage</h1>`

## Елементи посилання
Щоб створити посилання в HTML, ви використовуєте елемент посилання. Елемент посилання визначається обгортанням тексту або іншого HTML елемента, який ви хочете зробити посиланням, тегом `<a>`.

Додайте наступне до `body` сторінки `index.html`, яку ви створили, і відкрийте її у браузері:

```html
<a>click me</a>
```

Ви могли помітити, що натиснення на це посилання нічого не робить. Це тому, що тег посилання сам по собі не знає, куди ви хочете направити. Ви повинні вказати йому місце призначення. Це робиться за допомогою HTML атрибута.

HTML атрибут надає додаткову інформацію HTML елементу і завжди розміщується у початковому тегу елемента. Атрибут зазвичай складається з двох частин: імені та значення; однак не всі атрибути потребують значення. У вашому випадку потрібно додати атрибут `href` (посилання на гіперпосилання) до початкового тегу елемента посилання. Значення атрибута `href` — це місце призначення, куди ви хочете направити посилання.

Додайте наступний атрибут `href` до елемента посилання, який ви створили раніше, і спробуйте натиснути на нього знову, не забудьте оновити браузер, щоб застосувати нові зміни.

```html
<a href="https://www.theodinproject.com/about">click me</a>
```

За замовчуванням будь-який текст, обгорнутий тегом посилання без атрибута `href`, виглядатиме як звичайний текст. Якщо атрибут `href` присутній, браузер зробить текст синім і підкреслить його, щоб позначити, що це посилання.

Варто зазначити, що ви можете використовувати теги посилання для направлення на будь-який ресурс в інтернеті, а не лише на інші HTML документи. Ви можете направляти на відео, pdf файли, зображення тощо, але здебільшого ви будете направляти на інші HTML документи.
