# --explanation--

`Database` 是一個用來儲存和組織資料的系統。

`Activity` 指的是資料庫中發生的動作或處理程序。

- `The database contains customer information and order details.`－這表示資料庫儲存重要的商務資料。

- `There was unusual activity in the database last night。`－這表示資料庫中發生了某些意外的事件。

總稱為 `database activity` 是指資料庫系統內發生的所有動作、變更或處理程序。
