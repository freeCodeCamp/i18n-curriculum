# --interactive--

Властивості `grid-column` і `grid-row` дозволяють вам вказати горизонтальне та вертикальне розташування елементів сітки в межах макета сітки.

Інакше кажучи, вони обидві дають змогу контролювати, де елемент сітки починається і закінчується, посилаючись на лінії сітки. Ці лінії сітки — це межі, які розділяють рядки та стовпці, які ви вже визначили за допомогою властивостей `grid-template-rows` і `grid-template-columns`.

Давайте розглянемо синтаксис обох `grid-row` і `grid-column`, а також практичний приклад.

Ось синтаксис властивості `grid-row`:

```css
grid-row: <start-line> / <end-line>;
```

А ось `grid-column`:

```css
grid-column: <start-line> / <end-line>;
```

`<start-line>` — це лінія сітки, де елемент починається, а `<end-line>` — лінія сітки, де елемент закінчується. Обидві нумеруються з 1, тобто рахунок починається з 1, а не з 0.

Пам’ятайте, що лінії сітки для рядків генеруються на основі кількості рядків, вказаних у властивості `grid-template-rows`. Те саме стосується стовпців із властивістю `grid-template-columns`.

Для практичного прикладу почнемо з ліній, які генерує браузер, коли ви використовуєте властивості `grid-template-rows` і `grid-template-columns`.

Ось приклад сітки з 4 стовпцями і 3 рядками:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}
```

:::

Перевірка контейнера сітки (клас `grid`) показує, що кожен рядок і стовпець обмежений двома лініями — початковою лінією на початку рядка або стовпця та кінцевою лінією в кінці рядка або стовпця.

Ви можете націлитися на ці лінії, щоб почати використовувати властивості `grid-row` і `grid-column`, щоб визначити, де розмістити елемент.

Ось як можна зробити так, щоб перший елемент сітки займав перші два стовпці:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
}
```

:::

Цим ви вказуєте, що перший елемент сітки має починатися з колонки 1 і закінчуватися безпосередньо перед колонкою 3.

Тепер перший елемент займатиме два стовпці, а четвертий елемент буде переміщений на другий рядок. Круто!

Якщо ви також хочете, щоб перший елемент займав два рядки, ви можете вказати `grid-row` зі значенням `1 / 3`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
  grid-row: 1 / 3;
}
```

:::

Ви також можете використовувати ключове слово `span`, щоб сказати елементу сітки, через які рядки і стовпці він має простягатися. Наприклад, `1 / 3` — це те саме, що `1 / span 2`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / span 2;
  grid-row: 1 / span 2;
}
```

:::

Ви можете продовжувати застосовувати цю техніку до будь-якого елемента сітки і розміщувати їх де завгодно.

Продовжуючи процес, ось як ми зробили макет masonry з елементами.

Ось новий приклад:

```html
  <div class="item4">4</div>
```

```css
.grid {

  grid-template-rows: repeat(4, 100px); /* 4 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;

  grid-column: 1 / span 3;

  grid-column: 1 / span 2;
```
