# --description--

现在，让我们稍微改变一下：

```html
<!-- index.html -->

<div class="main">
  <div class="list" id="subsection"></div>
</div>
```

```css
/* rule 1 */
#subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

在上面的示例中，尽管规则 2 拥有比 ID 选择器更多的类选择器，但规则 1 更具体，因为 ID 优先于类。
