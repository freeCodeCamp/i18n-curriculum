# --description--

Wie Sie gelernt haben, bedeutet `repetir`, etwas noch einmal zu sagen. Wenn Sie das Pronomen `lo` hinzufügen, beziehen Sie sich auf etwas, das bereits erwähnt wurde. Das Pronomen ersetzt die spezifische Sache (ein Wort, eine Zahl, eine E-Mail-Adresse usw.). Also:

- `Repetir` = wiederholen.

- `Repetirlo` = es wiederholen.

`Repetirlo` wird verwendet, wenn „die Sache“ bereits aus dem Kontext bekannt ist. Dasselbe Muster funktioniert mit anderen Verben:

- `¿Puedes escribirlo?` – Können Sie es schreiben?

- `¿Puedes decirlo otra vez?` – Können Sie es noch einmal sagen?

# --explanation--

`Lo` ist ein direktes Objektpronomen, das verwendet wird, um etwas zu ersetzen, das im Gespräch bereits bekannt ist. Wenn es an einen Infinitiv angehängt wird, bildet es ein Wort:

- `repetir` + `lo` → `repetirlo` – Wiederholen Sie es.

- `escribir` + `lo` → `escribirlo` – Schreiben Sie es.

- `decir` + `lo` → `decirlo` – Sagen Sie es.

Diese Struktur ist üblich, wenn der Zuhörer bereits versteht, worauf sich „es“ bezieht.
