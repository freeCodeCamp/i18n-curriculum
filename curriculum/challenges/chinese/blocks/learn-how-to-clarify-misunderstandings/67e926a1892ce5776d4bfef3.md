# --explanation--

`Reducing complexity` 意味着使系统更易于理解和使用。
  
`Showing only what's necessary` 意味着用户只看到他们需要交互的部分。
  
`Keeping the details hidden` 指隐藏用户不需要的内部代码或逻辑。

总的来说，这意味着抽象创建了一个清晰、简单的接口，同时隐藏了背后更复杂的内部工作。
