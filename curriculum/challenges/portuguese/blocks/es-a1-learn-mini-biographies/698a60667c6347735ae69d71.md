# --description--

`Le` é um pronome que representa "para ele", "para ela" ou "para você".

`Le gusta` significa "ele gosta" quando se refere ao Diego, que é um homem. É o mesmo quando se refere a uma mulher. Por exemplo:

- `Le gusta el fútbol.` - Ele gosta de futebol (referindo-se a um homem).

- `Le gusta el fútbol.` - Ela gosta de futebol (referindo-se a uma mulher).
