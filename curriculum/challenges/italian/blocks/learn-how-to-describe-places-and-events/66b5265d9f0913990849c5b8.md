# --explanation--

`To touch on` può significare menzionare brevemente o affrontare un argomento. Spesso si usa quando si tratta un tema senza entrare troppo nei dettagli. Per esempio:

- `The speaker touched on the new software update during the meeting.` - Il parlante ha menzionato brevemente l’aggiornamento del software ma non ha approfondito.
