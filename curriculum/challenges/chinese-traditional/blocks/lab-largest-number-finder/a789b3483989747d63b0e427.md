# --description--

在本實驗中，你將建置一個函式，該函式傳回由每個提供的子陣列中最大數字所組成的陣列。

請記住，你可以用簡單的 `for` 迴圈迭代陣列，並用陣列語法 `arr[i]` 存取每個成員。

**目標：** 完成以下使用者故事並通過所有測試以完成實驗。

**使用者故事：**

1. 你應該創建一個函式（程式）`largestOfAll`，該函式（程式）以陣列的陣列作為引數。
2. 該函式（程式）應該傳回一個包含每個子陣列中最大數字的陣列。

# --solutions--

```js
function largestOfAll(arr) {
  return arr.map(subArr => Math.max.apply(null, subArr));
}

largestOfAll([
  [4, 5, 1, 3],
  [13, 27, 18, 26],
  [32, 35, 37, 39],
  [1000, 1001, 857, 1]
]);
```
