# --explanation--

`Present Perfect Continuous` 시제는 과거에 시작되어 아직도 진행 중인 동작을 설명할 때 사용합니다. 이 시제는 동작의 지속 시간이나 계속됨에 포커스를 둡니다. 이 시제의 구조는 `have/has + been + verb(-ing)`입니다. 예를 들어:

- `We have been developing a new app for accessibility.` - 이는 개발이 과거에 시작되어 아직도 진행 중임을 의미합니다.

- `He has been fixing bugs in the software for several hours.` - 이는 그가 버그 수정을 일찍 시작했고 아직도 작업 중임을 의미합니다.
