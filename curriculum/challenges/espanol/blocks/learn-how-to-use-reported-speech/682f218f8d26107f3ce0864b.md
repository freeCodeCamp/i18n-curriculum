# --explanation--

`Of course` es una forma educada de mostrar acuerdo o decir `yes` sin dudar. Significa que el hablante está contento o dispuesto a hacer algo. Por ejemplo:

`Of course, I'll help you with the report.` – Esto significa que el hablante está encantado de ayudar.

`Will` se usa a menudo para hacer promesas o expresar una decisión en el momento de hablar. Por ejemplo:

`I'll email you the file this afternoon.` – Esto muestra una promesa de hacer algo pronto.
