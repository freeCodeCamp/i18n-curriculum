# --description--

`Much of` は、特に不可算名詞に対して、何かの大部分を表現するために使われます。例えば、`He spent much of the day reading.` は彼がその日をかなりの時間読書に費やしたことを示します。

`Free time` は、仕事や重要な用事をしていない時間で、自分の好きなことを選べる時間を指します。例えば、`She uses her free time to learn guitar.` は彼女が他に義務がないときに何を選ぶかを示します。
