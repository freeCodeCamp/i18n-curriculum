# --explanation--

`They'll`는 `they will`의 축약형입니다. 미래에 일어날 것으로 예상되거나 예측되는 일을 말할 때 사용합니다.

`Likely`는 어떤 일이 일어날 가능성이 높거나 확률이 높다는 뜻입니다. 예를 들어:

- `They'll likely ask for more information.` - 클라이언트의 요청이 있을 가능성이 높다고 예측합니다.

- `The feature will likely need additional testing.` - 테스트가 필요할 가능성이 높다는 것을 나타냅니다.
