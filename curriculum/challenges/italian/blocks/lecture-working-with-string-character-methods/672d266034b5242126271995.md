# --interactive--

In programmazione, capire come i caratteri sono rappresentati come numeri è fondamentale. Qui entra in gioco ASCII. ASCII, abbreviazione di American Standard Code for Information Interchange, è uno standard di codifica dei caratteri usato nei computer per rappresentare il testo. Assegna un valore numerico a ogni carattere, riconosciuto universalmente dalle macchine.

In questa lezione esploreremo cos’è ASCII, come funziona e come i metodi JavaScript come `charCodeAt()` e `fromCharCode()` si collegano alla codifica dei caratteri. Anche se le stringhe JavaScript usano internamente Unicode (UTF-16), i valori ASCII corrispondono ai primi 128 caratteri Unicode, motivo per cui gli esempi basati su ASCII funzionano in JavaScript.

ASCII è un sistema per codificare caratteri come lettere, cifre e simboli in valori numerici. Ogni carattere è associato a un numero specifico.

Per esempio, la lettera maiuscola `A` è rappresentata dal numero `65` in ASCII, mentre la minuscola `a` è rappresentata da `97`. Questa codifica permette ai computer di archiviare e manipolare il testo.

Lo standard ASCII copre 128 caratteri tra cui:

- Lettere inglesi maiuscole e minuscole (A-Z, a-z).
- Numeri (0-9).  
- Segni di punteggiatura e simboli comuni (!, @, # e così via).
- Caratteri di controllo (come newline e tab).  

In JavaScript puoi consultare il codice numerico di un carattere usando il metodo `charCodeAt()`. Questo metodo restituisce l’unità di codice UTF-16 del carattere a un indice specificato. Per i primi 128 caratteri, questo valore corrisponde al codice ASCII.

Diamo un’occhiata a un esempio:

:::interactive_editor

```js
let letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

:::

In questo esempio, `A` è il primo carattere della stringa, e chiamare `charCodeAt(0)` restituisce il suo codice numerico (che corrisponde al valore ASCII per i caratteri latini di base), `65`.

Puoi anche usare questo metodo con altri caratteri per trovare i loro codici numerici:

:::interactive_editor

```js
let symbol = "!";
console.log(symbol.charCodeAt(0));  // 33
```

:::

Qui, il codice numerico per il punto esclamativo `!` è restituito come `33` (che corrisponde al suo valore ASCII).

Mentre `charCodeAt()` ti aiuta a recuperare il codice numerico di un carattere, il metodo `fromCharCode()` ti permette di fare l’opposto: convertire un’unità di codice UTF-16 (che corrisponde ad ASCII per i caratteri base) nel carattere corrispondente.

Vediamo come funziona:

:::interactive_editor

```js
let char = String.fromCharCode(65);
console.log(char);  //  A
```

:::

In questo esempio, `fromCharCode(65)` converte il codice numerico `65` (che corrisponde al valore ASCII per `A`) di nuovo nel carattere `A`.

Un altro esempio è convertire il numero `97` nella sua lettera minuscola corrispondente:

:::interactive_editor

```js
let char = String.fromCharCode(97);
console.log(char);  // a
```

:::

Questi metodi sono particolarmente utili quando devi manipolare o confrontare caratteri basandoti sui loro codici numerici.

Per esempio, potresti usare `charCodeAt()` per verificare se un carattere è maiuscolo, minuscolo o una cifra confrontando il suo valore ASCII.

D’altra parte, `fromCharCode()` può essere usato per generare dinamicamente caratteri dai loro codici ASCII.
