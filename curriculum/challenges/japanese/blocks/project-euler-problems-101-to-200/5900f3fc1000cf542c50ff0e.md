# --description--

ABC をすべての内角が120度未満の三角形とします。X を三角形の内部の任意の点とし、$XA = p$、$XC = q$、$XB = r$ とします。

フェルマーはトリチェリに対し、$p + q + r$ を最小化するような X の位置を見つけるようチャレンジしました。

トリチェリは、三角形 ABC の各辺に正三角形 AOB、BNC、AMC を作図すると、AOB、BNC、AMC の外接円が三角形の内部の一点 T で交わることを証明しました。さらに、T はトリチェリ／フェルマー点と呼ばれ、$p + q + r$ を最小化することも証明しました。さらに驚くべきことに、和が最小化されるとき、$AN = BM = CO = p + q + r$ となり、AN、BM、CO も T で交わることが示されています。

<img alt="三角形 ABC の各辺に作図された正三角形 AOB、BNC、AMC。AOB、BNC、AMC の外接円が三角形内部の一点 T で交わる様子" src="https://cdn.freecodecamp.org/curriculum/project-euler/investigating-the-torricelli-point-of-a-triangle.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

和が最小化され、かつ a、b、c、p、q、r がすべて正の整数である場合、三角形 ABC をトリチェリ三角形と呼びます。例えば、$a = 399$、$b = 455$、$c = 511$ はトリチェリ三角形の例であり、$p + q + r = 784$ となります。$p + q + r ≤ 120000$ となるトリチェリ三角形のすべての異なる値の和を求めてください。
