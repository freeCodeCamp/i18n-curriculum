# --description--

마지막 요소, 최종 결정권자, 결정권자의 결정권자입니다. 모든 다른 요소를 고려한 후에도 여전히 하나의 요소를 대상으로 하는 여러 충돌하는 규칙이 있다면, 계단식 규칙은 어떤 규칙을 적용할지 어떻게 결정할까요?

사실 아주 간단합니다. 가장 나중에 정의된 규칙이 승자가 됩니다.

```css
/* styles.css */

.alert {
  color: red;
}

.warning {
  color: yellow;
}
```

`alert`와 `warning` 클래스를 모두 가진 요소의 경우, 계단식 규칙은 상속(여기서는 없음)과 특이성(어느 규칙도 더 특이하지 않음)을 포함한 모든 다른 요소를 차례로 확인합니다. `.warning` 규칙이 가장 나중에 정의되었고, 다른 어떤 요소도 적용할 규칙을 결정하지 못했기 때문에 이 규칙이 요소에 적용됩니다.
