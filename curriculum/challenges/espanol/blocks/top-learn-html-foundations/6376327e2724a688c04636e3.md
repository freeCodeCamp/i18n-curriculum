# --description--

HTML y CSS son dos lenguajes que trabajan juntos para crear todo lo que ves cuando miras algo en internet. HTML es la información en bruto de la que se construye una página web. Todo el texto, enlaces, tarjetas, listas y botones se crean en HTML. CSS es lo que añade estilo a esos elementos simples. HTML coloca la información en una página web, y CSS posiciona esa información, le da color, cambia la fuente y la hace lucir genial.

# --questions--
    
## --text--

¿Cuáles son las responsabilidades principales de HTML y CSS al crear una página web?

## --answers--

HTML estructura el contenido, mientras que CSS da estilo a los elementos.

---

HTML crea el diseño visual, mientras que CSS organiza el contenido.

---

CSS gestiona la funcionalidad, mientras que HTML controla la estructura.
