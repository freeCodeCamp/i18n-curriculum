# --description--

我们将定义一个 sqube 为形式为 `${p^2}{q^3}` 的数，其中 $p$ 和 $q$ 是不同的素数。

例如，$200 = {5^2}{2^3}$ 或 $120072949 = {{23}^2}{{61}^3}$。

前五个角是72,108,200,392和500。

有趣的是，200 也是第一个无法通过更改任何单个数字来变成素数的数字；我们称这样的数字为素数防护数。下一个包含连续子串 `200` 的素数防护立方数是 1992008。

查找包含连续子字串 `200` 的第 200 个素数证明的 sqube。
