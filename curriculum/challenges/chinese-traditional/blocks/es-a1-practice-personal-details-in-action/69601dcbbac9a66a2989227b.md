# --description--

你最近學會了樣式 `¿Quieres que lo deletree?` 來提出拼寫某物。另一個常見的問題是：

`¿Cómo se deletrea?`

當有人想知道一個字、名稱或電子郵件的拼寫方式時，會使用這個問題。

這在確認聯絡資料的對話中非常常見。

# --explanation--

Esteban 問 `¿Cómo se deletrea?`，所以他在問某個東西怎麼拼寫。正確答案應該要一個字母一個字母地拼出該資訊。

只有一個選項可做到這點：`j-u-l-i-e-t-a punto c-a-c-e-r-e-s arroba a-n-d-e-s punto com.`

其他選項是完整的述語（name、nationality、profession），且不提供拼寫。
