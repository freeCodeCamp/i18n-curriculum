# --description--

HTML y CSS se ocupan principalmente de presentar información. No se usan para programar lógica. JavaScript, que aprenderás en la siguiente sección, se usa para hacer que las páginas web hagan cosas. Hay bastante que puedes hacer solo con HTML y CSS, y definitivamente necesitarás ambos. A lo largo de nuestro plan de estudios, las siguientes lecciones se enfocan en darte las herramientas que necesitas para tener éxito cuando llegues al contenido de JavaScript.

# --questions--
    
## --text--

¿Cuál es una forma en la que JavaScript es diferente de HTML y CSS?

## --answers--

JavaScript se ocupa principalmente de presentar información.

---

JavaScript hace que las páginas web realicen acciones.

---

No necesitas HTML/CSS si conoces JavaScript.
