# --description--

これからウェブフォームを作成しましょう。

`input` 要素はユーザーからの入力を簡単に受け取る方法です。

次のようにテキスト入力を作成できます。

```html
<input type="text">
```

`input` は空要素であることに注意してください。

# --instructions--

リストの下にタイプが `input` の `text` 要素を作成してください。

# --hints--

アプリにはタイプが `input` の `text` 要素が必要です。

```js
assert.isNotEmpty(document.querySelectorAll('input[type=text]'));
```
