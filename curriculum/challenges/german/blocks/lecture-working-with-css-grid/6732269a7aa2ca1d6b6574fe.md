# --interactive--

Implizites Grid bezieht sich auf die Zeilen und Spalten, die vom Browser automatisch erstellt werden, wenn Elemente in einem Grid-Layout platziert werden – jene, die nicht explizit mit `grid-template-rows` oder `grid-template-columns` definiert sind.

Die Eigenschaften, die die vom Browser implizit erstellten Spalten und Zeilen steuern, sind `grid-auto-columns` und `grid-auto-rows`.

Implizites Grid bezieht sich auch auf die zusätzlichen Zeilen und Spalten, die der Browser automatisch generiert, wenn Sie ein Element außerhalb der explizit definierten Zeilen und Spalten platzieren.

Zum Beispiel, nehmen wir an, Sie definieren nur zwei explizite Spalten in einem Grid-Layout auf diese Weise:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
  <div class="grid-item">Item 4</div>
  <div class="grid-item">Item 5</div>
  <div class="grid-item">Item 6</div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 100px 100px; /* Only 2 explicit columns */
}

.grid-item {
  background-color: burlywood;
  border: 1px solid orangered;
  padding: 0.5rem;
  margin: 0.5rem;
}
```

:::

Zwei Elemente füllten die erste Zeile mit den zwei expliziten Spalten – `Item 1` in der ersten Spalte und `Item 2` in der zweiten Spalte.

Die nächsten Elemente beginnen eine neue Zeile mit `Item 3` in der ersten Spalte der zweiten Zeile und `Item 4` in der zweiten Spalte der zweiten Zeile, und so weiter.

Wie Sie bereits gesehen haben, sind explizite Grids die Bereiche des Grids, die Sie absichtlich einrichten. Das heißt, die Zeilen und Spalten, die Sie explizit für ein Grid-Layout mit den Eigenschaften `grid-template-rows` und `grid-template-columns` definieren.

Hier ist eine Tabelle, die die Unterschiede zwischen impliziten und expliziten Grids zusammenfasst:

| **Funktion**          | **Explizites Grid**                                                                  | **Implizites Grid**                                                                          |
|-----------------------|-------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------|
| **Größensteuerung**    | Vollständig anpassbar mit ` grid-template-rows ` und ` grid-template-columns `.                                | Gesteuert durch ` grid-auto-rows ` und ` grid-auto-columns ` oder standardmäßig ` auto `.                    |
| **Standardverhalten**  | Ändert sich nicht, es sei denn, es wird explizit definiert.                         | Passt sich automatisch an Elemente an, die außerhalb des expliziten Grids platziert sind.   |
| **Komplexität**        | Erfordert mehr Planung für die Layout-Struktur.                                    | Einfacher zu implementieren für unstrukturierte oder variable Inhalte.                      |
| **Flexibilität**       | Starre Struktur mit definierten Zeilen und Spalten.                                | Flexibel und passt sich dynamisch platzierten Inhalten an.                                 |
| **Performance**        | Potenziell leistungsfähiger, da das Layout vordefiniert ist.                       | Kann zusätzliche Browser-Berechnungen für die Generierung des impliziten Grids erfordern.  |
| **Anwendungsfall**     | Nützlich, wenn die Grid-Struktur vorhersehbar und im Voraus definiert ist.         | Nützlich für dynamische Layouts, bei denen der Inhalt unbekannt ist oder sich häufig ändert.|
