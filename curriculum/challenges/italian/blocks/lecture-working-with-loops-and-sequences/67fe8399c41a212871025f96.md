# --description--

Nelle prossime lezioni impareremo a conoscere le liste, le tuple e gli intervalli, che sono tre tipi di sequenze di base usate in Python.

Il tipo di dato lista è una sequenza ordinata di elementi che può essere composta da stringhe, numeri o anche altre liste. Le liste sono mutabili e usano un indice a base zero, il che significa che il primo elemento della lista si trova all’indice zero.

Ecco la sintassi di base per una lista:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
```

Per consultare un elemento dalla lista `cities`, puoi fare riferimento al suo numero di indice nella sequenza. Ecco un esempio di come consultare il primo elemento della lista `cities`:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # 'Los Angeles'
```

L’indicizzazione negativa viene usata per consultare gli elementi a partire dalla fine della lista invece che dall’inizio all’indice `0`. Per consultare l’ultimo elemento di qualsiasi lista, puoi usare `-1` in questo modo:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # 'Tokyo'
```

Un altro modo per creare una lista è usare il costruttore `list()`. Il costruttore `list()` serve a convertire un iterabile in una lista in questo modo:

```py
developer = 'Jessica'
list(developer) # ['J', 'e', 's', 's', 'i', 'c', 'a']
```

Un iterabile è un tipo speciale di oggetto che può essere iterato un elemento alla volta. Imparerai di più sui loop in Python più avanti.

Per ottenere il numero totale di elementi in una lista, puoi usare la funzione `len()` in questo modo:

```py
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

Se vuoi aggiornare un valore a un indice particolare, puoi fare qualcosa del genere:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

Poiché le liste sono mutabili, puoi aggiornare qualsiasi elemento nella lista purché fornisca un numero di indice valido. Se fornisci un indice (positivo o negativo) che è fuori dai limiti della lista, riceverai un `IndexError`:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list assignment index out of range
"""
```

Se vuoi rimuovere un elemento da una lista puoi usare la parola chiave `del` in questo modo:

```py
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

A volte è utile verificare se un elemento è presente nella lista. Per farlo, puoi usare la parola chiave `in` in questo modo:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

A volte è comune avere liste annidate dentro altre liste in questo modo:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

In questo esempio, abbiamo una lista annidata che contiene tre linguaggi di programmazione popolari. Per consultare la lista annidata, devi farlo usando l’indice `2` dato che le liste hanno indice a base zero:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

Poi, per consultare il secondo linguaggio da quella lista annidata, devi farlo usando l’indice `1` in questo modo:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # 'Rust'
```

Un’altra tecnica comune usata con le liste è lo scompattamento dei valori.

Scompattare i valori da una lista è una tecnica usata per assegnare valori da una lista a nuove variabili. Ecco un esempio di scompattamento di una lista `developer` in nuove variabili chiamate `name`, `age` e `job`.

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

Qui, `name` ha il valore `'Alice'`, `age` ha il valore `34` e `job` ha il valore `'Rust Developer'`.

Se hai bisogno di raccogliere gli elementi rimanenti da una lista, puoi usare l’operatore asterisco (`*`) in questo modo:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

In questo esempio, `name` avrà ancora il valore `'Alice'` e `rest` è una lista di due elementi: il numero `34` e la stringa `'Rust Developer'`.

Se il numero di variabili sul lato sinistro dell’operatore di assegnazione non corrisponde al numero totale di elementi nella lista, riceverai un `ValueError`:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job, city = developer

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: not enough values to unpack (expected 4, got 3)
"""
```

L’ultimo concetto che vedremo è l’operatore slice (`:`). Simile alle stringhe, puoi consultare porzioni di una lista usando l’operatore slice in questo modo:

```py
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie', 'Brownies']
desserts[1:4] # ['Cookies', 'Ice Cream', 'Pie']
```

In questo esempio, l’indice di partenza è `1` dato che punta al secondo elemento della lista. Poi usiamo l’operatore slice seguito da un indice finale `4`, che include tutto fino a (ma non incluso) l’elemento a quell’indice.

Un’altra cosa che puoi fare con l’operatore slice `:` è specificare un intervallo di passo che determina di quanto incrementare tra gli indici. Supponiamo di avere una lista di numeri come questa:

```py
numbers = [1, 2, 3, 4, 5, 6]
```

Se vuoi estrarre una lista solo dei numeri pari, puoi usare l’operatore slice in questo modo:

```py
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

Il primo numero pari si trova all’indice `1`, quindi quello sarà l’indice di partenza. Poiché vogliamo arrivare fino alla fine della lista, omettiamo l’indice finale. Infine, specifichiamo `2` per l’intervallo di passo opzionale così incrementerà solo di `2` invece del predefinito `1`.

Le liste sono una struttura dati utile e flessibile che userai molto nei tuoi programmi Python. Nella prossima lezione, imparerai i metodi comuni che puoi usare con le liste.
