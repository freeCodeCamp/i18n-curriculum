# --description--

`Probably` використовується, коли ви вважаєте, що щось, ймовірно, станеться, але не впевнені на 100%.

Наприклад, ви могли б сказати: `The code will probably run without errors after the update`, бо ви в це вірите, але все ще існує невелика ймовірність проблеми.
