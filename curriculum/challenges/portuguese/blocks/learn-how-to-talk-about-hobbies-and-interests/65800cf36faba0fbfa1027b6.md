# --description--

Perguntas negativas são uma forma de pedir a alguém para fazer algo ou sugerir uma ação de maneira educada ou encorajadora. Eles frequentemente começam com `why` seguido por uma forma negativa como `don't`. Por exemplo:

`Why don't we go for a walk?` - Isso sugere que dar uma caminhada é uma boa ideia.
