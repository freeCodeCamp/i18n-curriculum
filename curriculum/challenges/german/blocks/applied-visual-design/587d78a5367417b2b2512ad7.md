# --description--

Die `repeating-linear-gradient()`-Funktion ist der `linear-gradient()`-Funktion sehr ähnlich, mit dem wesentlichen Unterschied, dass sie das angegebene Verlaufsmuster wiederholt. `repeating-linear-gradient()` akzeptiert eine Vielzahl von Werten, aber der Einfachheit halber arbeiten Sie in dieser Aufgabe mit einem Winkelwert und Farb-Stopps.

Der Winkelwert gibt die Richtung des Verlaufs an. Farb-Stopps sind wie Breitenwerte, die markieren, wo ein Übergang stattfindet, und werden mit einem Prozentsatz oder einer Anzahl von Pixeln angegeben.

Im Beispiel, das im Code-Editor gezeigt wird, beginnt der Verlauf mit der Farbe `yellow` bei 0 Pixeln, die in die zweite Farbe `blue` übergeht, die 40 Pixel vom Anfang entfernt ist. Da der nächste Farb-Stopp ebenfalls bei 40 Pixeln liegt, ändert sich der Verlauf sofort zur dritten Farbe `green`, die selbst in die vierte Farbwert `red` übergeht, da diese 80 Pixel vom Anfang des Verlaufs entfernt ist.

Für dieses Beispiel hilft es, die Farb-Stopps als Paare zu betrachten, bei denen jeweils zwei Farben ineinander übergehen.

```css
0px [yellow -- blend -- blue] 40px [green -- blend -- red] 80px
```

Wenn jeweils zwei Farb-Stopp-Werte dieselbe Farbe haben, ist der Übergang nicht sichtbar, weil er zwischen derselben Farbe stattfindet, gefolgt von einem harten Übergang zur nächsten Farbe, sodass Sie Streifen erhalten.

# --instructions--

Erzeugen Sie Streifen, indem Sie `repeating-linear-gradient()` so ändern, dass ein Verlauf mit einem Winkel von `45deg` verwendet wird, dann setzen Sie die ersten beiden Farb-Stopps auf `yellow` und schließlich die zweiten beiden Farb-Stopps auf `black`.

# --hints--

Der Winkel des `repeating-linear-gradient()` sollte 45deg betragen.

```js
assert.match(code,/background:\s*?repeating-linear-gradient\(\s*?45deg/gi);
```

Der Winkel des `repeating-linear-gradient()` sollte nicht mehr 90deg sein.

```js
assert.notMatch(code, /90deg/gi);
```

Der Farb-Stopp bei 0 Pixeln sollte `yellow` sein.

```js
assert.match(code, /yellow\s+?0(px)?/gi);
```

Der erste Farb-Stopp bei 40 Pixeln sollte `yellow` sein.

```js
assert.match(code, /yellow\s+?40px/gi);
```

Der zweite Farb-Stopp bei 40 Pixeln sollte `black` sein.

```js
assert.match(code, /yellow\s+?40px,\s*?black\s+?40px/gi);
```

Der letzte Farb-Stopp bei 80 Pixeln sollte `black` sein.

```js
assert.match(code, /black\s+?80px/gi);
```
