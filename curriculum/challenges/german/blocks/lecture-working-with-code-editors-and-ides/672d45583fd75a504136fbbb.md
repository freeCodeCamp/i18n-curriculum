# --description--

Um VS Code zu installieren, sollten Sie die Website besuchen, um den passenden Installer für Ihr Betriebssystem zu erhalten. Achten Sie darauf, die VS Code-Website zu besuchen und nicht die Visual Studio-Website.

Für Windows können Sie einen EXE-Installer herunterladen. Sobald Sie ihn heruntergeladen haben, führen Sie die Datei aus, und der Setup-Assistent führt Sie durch den Installationsprozess.

Für Mac können Sie ein ZIP-Archiv herunterladen, das eine `.app` enthält, die Sie ausführen können. Sie können VS Code auch über Homebrew installieren.

Öffnen Sie die Terminal-App auf Ihrem Mac und führen Sie den folgenden Befehl aus:

```sh
brew install --cask visual-studio-code
```

Unter Linux können Sie eine `.deb` oder `.rpm` Datei direkt herunterladen. Sobald Sie sie installiert haben, können Sie die passende ausführbare Datei oder den CLI-Befehl (typischerweise `code`) verwenden, um die Anwendung zu starten. Und jetzt sind Sie bereit zum Programmieren.
