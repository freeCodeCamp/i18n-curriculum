# --interactive--

가상 클래스는 특정 상태나 위치에 따라 요소를 선택할 수 있게 해주는 특별한 CSS 키워드입니다. 요소의 상태나 위치는 다음과 같을 수 있습니다:

- 활성 상태일 때.
- 마우스가 위에 있을 때.
- 부모의 첫 번째 자식일 때.
- 부모의 마지막 자식일 때.
- 방문한 링크일 때.
- 비활성화 상태일 때.

가상 클래스를 사용하려면 선택자에 콜론(`:`)을 붙이고 가상 클래스 이름을 추가하세요:

```css
selector:pseudo-class {
  /* CSS properties */
}
```

이제 앞서 언급한 상태와 위치 각각을 나타내기 위해 가상 클래스를 어떻게 사용할 수 있는지 살펴보겠습니다.

`:active` 가상 클래스는 버튼을 클릭하는 것처럼 요소의 활성 상태를 선택할 수 있게 해줍니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example Button</button>
```

```css
button:active {
  background: greenyellow;
}
```

:::

`:hover` 가상 클래스는 요소의 호버 상태를 정의합니다. 예를 들어 버튼이나 링크 위에 마우스를 올렸을 때입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="#">Hover over me!</a>  
```

```css
a:hover {
  text-decoration: none;
  color: white;
  background: crimson;
}
```

:::

`:first-child` 가상 클래스는 부모 요소의 첫 번째 자식인 요소를 선택합니다. 다음은 `div` 컨테이너 내 첫 번째 단락 요소를 선택하는 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:first-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

그 `div` 내 첫 번째 단락 요소는 `lightcoral` 배경색과 네 면 모두에 `padding`의 `0.4rem`을 받게 됩니다.

`:last-child` 가상 클래스는 부모의 마지막 자식인 요소를 선택합니다. 다음은 `div` 컨테이너 내 마지막 단락 요소를 선택하는 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:last-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

`:visited` 가상 클래스는 사용자가 이미 방문한 링크를 스타일링할 수 있게 해줍니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="https://www.example.com" target="_blank">Visit Example.com</a>
```

```css
a:visited {
  color: purple;
}
```

:::

`:disabled` 가상 클래스는 비활성화된 인터랙티브 요소를 스타일링할 수 있게 해줍니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button disabled>Disabled Button</button>
```

```css
button:disabled {
  background-color: lightgray;  
}
```

:::

보시다시피, 가상 클래스는 호버나 방문한 링크처럼 사용자 상호작용에 따라 요소를 스타일링할 수 있게 해줍니다. 덕분에 웹사이트가 더 인터랙티브하게 느껴집니다.

앞서 언급한 가상 클래스 외에도 다음과 같은 것들이 있습니다:

- `:focus`
- `:first-of-type`
- `:last-of-type`
- `:nth-of-type`
- `:modal`
- `:enabled`
- `:checked`
- `:required` 등.
