# --description--

おめでとうございます！React と Redux のレッスンを終えました。次に進む前に、最後に指摘しておきたいことがあります。通常、このようなコードエディターで React アプリを書くことはありません。このチャレンジでは、自分のマシン上のファイルシステムで作業するときの構文がどのようなものかを垣間見ることができます。コードは似たようなものになるはずですが、`import` 文の使用が異なります（これらはチャレンジで提供されたすべての依存関係を取り込みます）。

最後に、React と Redux のコードを書くには一般的にいくつかの設定が必要です。これはすぐに複雑になる場合があります。もし自分のマシンで試してみたい場合は、<a href="https://www.freecodecamp.org/news/install-react-with-create-react-app/" target="_blank" rel="noopener noreferrer nofollow">Create React App</a> が設定済みで準備が整っています。

または、CodePen で Babel を JavaScript プリプロセッサとして有効にし、React と ReactDOM を外部の JavaScript リソースとして追加して、そこで作業することもできます。
