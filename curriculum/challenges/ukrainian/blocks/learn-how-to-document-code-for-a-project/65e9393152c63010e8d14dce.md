# --description--

`Ensure` означає переконатися, що щось відбувається або є певним фактом. Наприклад, `Ensure the door is locked before leaving.`

`Clear` означає легкий для розуміння, як `a clear explanation`.

`Concise` означає висловлювати щось кількома словами без зайвих деталей. Прикладом є, `She gave a concise summary of the report.`
