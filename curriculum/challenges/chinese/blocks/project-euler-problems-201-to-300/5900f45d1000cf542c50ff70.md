# --description--

对于正整数 $n$，令 $σ(n)$ 为 $n$ 的所有因数之和，例如 $σ(6) = 1 + 2 + 3 + 6 = 12$。

完美数，正如你可能知道的，是满足 $σ(n) = 2n$ 的数字。

让我们定义正整数的完美商为 $p(n) = \frac{σ(n)}{n}$。

查找所有满足 $n ≤ {10}^{18}$ 且 $p(n)$ 具有形式 $k + \frac{1}{2}$（其中 $k$ 是整数）的正整数 $n$ 的和。
