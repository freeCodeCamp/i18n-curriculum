# --description--

Saper ispezionare ed eseguire il debug del tuo HTML e CSS è fondamentale per lo sviluppo front-end. Questa lezione ti guiderà attraverso i Chrome Dev Tools, che ti permettono di vedere informazioni dettagliate sui tuoi elementi e sulle regole CSS, oltre ad aiutarti a trovare e risolvere problemi nel tuo codice.

Per aprire l’ispezione, puoi cliccare con il tasto destro su qualsiasi elemento di una pagina web e selezionare “Ispeziona” oppure premere F12. Prova subito a farlo per vedere l’HTML e il CSS usati in questa pagina.

Non farti sopraffare da tutti gli strumenti che vedi ora! Per questa lezione, vogliamo concentrarci sui pannelli Elementi e Stili.
