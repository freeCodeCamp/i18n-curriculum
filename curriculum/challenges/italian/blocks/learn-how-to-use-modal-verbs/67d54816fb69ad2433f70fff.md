# --explanation--

`Have to follow` si usa per chiedere di obblighi o requisiti. Per esempio:

`You have to follow the safety guidelines when using this machine.` - È necessario seguire le regole di sicurezza quando si usa la macchina.

`Have to` funziona come `must` e indica che qualcosa è necessario.

`Follow` è il verbo corretto quando si parla di linee guida, regole o istruzioni.
