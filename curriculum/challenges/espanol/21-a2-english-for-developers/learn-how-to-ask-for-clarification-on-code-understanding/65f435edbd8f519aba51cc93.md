# --description--

Un `function` en programación es un conjunto de instrucciones dentro de un programa. Está diseñado para realizar una tarea particular. Defines un `function` una vez, especificando lo que debe hacer, y luego puedes usarlo tantas veces como necesites llamando su nombre.

Si algo está `missing` en el código, significa que una parte necesaria no está presente o ha sido pasada por alto.

Cuando el código `working as expected`, se desempeña exactamente como debería, sin errores o problemas.
