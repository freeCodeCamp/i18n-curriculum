# --explanation--

`UI` significa "interface do usuário", que se refere aos elementos visuais de um produto com os quais os usuários interagem. `设计师 (shè jì shī)` significa "designer".

Juntos, `UI 设计师 (shè jì shī)` significa uma profissão que foca em projetar a aparência e o layout dos produtos para torná-los fáceis e agradáveis de usar. Por exemplo:

`UI` refere-se ao design de Interface do Usuário, que é um papel fundamental no design de produto. É um termo muito comum em empresas de tecnologia e as pessoas usam diretamente "UI" sem traduzi-lo para o chinês.

`我是 (wǒ shì) UI 设计师 (shè jì shī)。` – Eu sou um designer de UI.
