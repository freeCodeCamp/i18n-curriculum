# --explanation--

`User Interface (UI)` はユーザーが見るもの（画面、ボタン、色）です。

`User Experience (UX)` はユーザーが感じるもの（滑らかさ、明快さ、使いやすさ）です。

良い製品は両方にフォーカスしています：明確な視覚と満足のいく体験です。
