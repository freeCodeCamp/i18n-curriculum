# --explanation--

Liu Ming está usando `您好 (nín hǎo)` para dizer "olá" respeitosamente.

`你好 (nǐ hǎo)` também é uma saudação comum em chinês, mas não é tão formal quanto `您好 (nín hǎo)`. Para amigos, colegas ou pares, você geralmente diz `你好 (nǐ hǎo)`.
