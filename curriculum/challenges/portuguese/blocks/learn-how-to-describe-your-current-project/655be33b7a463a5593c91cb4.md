# --description--

Você diz que algo é `definite` quando você tem certeza sobre isso. 

Quando você adiciona o sufixo `-ly`, `definite` se torna um advérbio, `definitely`, que significa estar certo sobre algo ou estar confiante sobre isso. 

É frequentemente usado para enfatizar o que o falante acredita ser verdade ou provavelmente verdade. Por exemplo:

- `I can definitely see why this is important.` - Isso enfatiza que a importância deve ser clara.

- `She will definitely finish the project on time.` - Isso mostra confiança nela para completar o projeto.
