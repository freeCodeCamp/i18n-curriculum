# --description--

Al escribir código en Python, es común encontrarse con errores. Entender estos errores es clave para depurar tu código rápida y eficientemente. Estos mensajes te dicen exactamente qué salió mal, si sabes cómo leerlos.

Los errores comunes en Python incluyen `SyntaxError`, `NameError`, `TypeError`, `IndexError` y `AttributeError`. Estos ocurren cuando Python no entiende tu código o cuando tu lógica no coincide con los datos con los que estás trabajando.

Aquí tienes un ejemplo de un `SyntaxError`:

```py
print("Hello, world!"
# SyntaxError: unexpected EOF while parsing
```

Esta línea carece de un paréntesis de cierre. Python genera un `SyntaxError` porque el código no sigue las reglas de sintaxis adecuadas.

Aquí tienes un ejemplo de un `NameError`:

```py
print(name)
# NameError: name 'name' is not defined
```

Estás intentando imprimir una variable que aún no ha sido definida. Python genera un `NameError` cuando no puede encontrar una variable con ese nombre.

Aquí tienes un ejemplo de un `TypeError`:

```py
5 + "5"
# TypeError: unsupported operand type(s) for +: 'int' and 'str'
```

No puedes sumar un entero y una cadena juntos. Python genera un `TypeError` cuando intentas realizar una operación con tipos de datos incompatibles.

Aquí tienes un ejemplo de un `IndexError`:

```py
my_list = [1, 2, 3]
print(my_list[5])
# IndexError: list index out of range
```

Estás intentando acceder a un índice que no existe en la lista. Python genera un `IndexError` cuando sales de los límites.

Aquí tienes un ejemplo de un `AttributeError`:

```py
num = 42
num.append(5)
# AttributeError: 'int' object has no attribute 'append'
```

El objeto `int` no tiene un método `append()`. Python genera un `AttributeError` cuando intentas usar un método o propiedad que no existe para ese tipo de dato.

Reconocer los mensajes de error comunes de Python te ayuda a solucionar problemas más rápido. En lugar de adivinar, lee el mensaje de error con atención, a menudo te dice exactamente qué salió mal y dónde buscar.
