# --instructions--

在這個專案中，你將使用折線圖、長條圖和箱型圖來視覺化時間序列資料。你將使用 Pandas、Matplotlib 和 Seaborn 來視覺化包含 2016-05-09 到 2019-12-03 期間 freeCodeCamp.org 論壇每日頁面瀏覽次數的資料集。資料視覺化將幫助你了解訪問的樣式並識別年度和每月的成長。

使用資料完成以下任務：

- 使用 Pandas 從 "fcc-forum-pageviews.csv" 匯入資料。將索引設為 `date` 行。
- 透過篩選出頁面瀏覽量位於資料集前 2.5% 或後 2.5% 的天數來清理資料。
- 創建一個 `draw_line_plot` 函式（程式），使用 Matplotlib 繪製類似於 "examples/Figure_1.png" 的折線圖。標題應為 `Daily freeCodeCamp Forum Page Views 5/2016-12/2019`。x 軸的標籤應為 `Date`，y 軸的標籤應為 `Page Views`。
- 創建一個 `draw_bar_plot` 函式（程式），用來繪製類似於 "examples/Figure_2.png" 的長條圖。它應該顯示按年份群組的每個月份的每日平均頁面瀏覽量。圖例應該顯示月份標籤並且標題為 `Months`。在圖表上，x 軸的標籤應該是 `Years`，y 軸的標籤應該是 `Average Page Views`。
- 創建一個 `draw_box_plot` 函式（程式），使用 Seaborn 繪製兩個相鄰的箱型圖，類似於 "examples/Figure_3.png"。這些箱型圖應該顯示在特定年份或月份內數值的分佈情況，以及隨時間的比較。第一張圖表的標題應為 `Year-wise Box Plot (Trend)`，第二張圖表的標題應為 `Month-wise Box Plot (Seasonality)`。確保底部的月份標籤從 `Jan` 開始，且 x 軸和 y 軸的標籤正確。樣板程式碼包含準備資料的指令。

對於每個圖表，請確保使用資料影格的複本。

樣板程式碼也包含用於存檔和傳回影像的指令。
