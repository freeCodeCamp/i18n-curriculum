# --description--

В Python область видимості визначає точку, в якій ви можете отримати доступ до змінної. Вона контролює життєвий цикл змінної та те, як вона розв’язується в різних частинах коду.

Щоб правильно визначити область видимості, Python дотримується правила **LEGB**, що означає:

- **Локальна область видимості (L)**: Змінні, оголошені у функціях або класах.
    
- **Область видимості оточення (E)**: Змінні, оголошені у зовнішніх або вкладених функціях.
    
- **Глобальна область видимості (G):** Змінні, оголошені на верхньому рівні модуля або файлу.
    
- **Вбудована область видимості (B)**: Зарезервовані імена в Python для вбудованих функцій, модулів, ключових слів і об’єктів.

Python використовує правило LEGB для розв’язання області видимості змінних у вашій програмі. Ми розглянемо кожне з цих правил, щоб ви краще зрозуміли цей процес.

**Локальна область видимості** означає, що змінна, оголошена всередині функції або класу, доступна лише в межах цієї функції або класу.

Ось приклад:

```python
def my_func():
    my_var = 10
    print(my_var)
```

У цьому випадку функція `my_func` має власну область видимості, до якої неможливо отримати доступ поза межами функції. Виклик `my_func` виведе `10`, але спроба вивести `my_var` поза функцією призведе до `NameError`:

```python
def my_func():
    my_var = 10 # Locally scoped to my_func
    print(my_var)

my_func() # 10

print(my_var) # NameError: name 'my_var' is not defined
```

**Область видимості оточення** означає, що функція, вкладена всередину іншої функції, може отримувати доступ до змінних функції, в якій вона вкладена.

Наприклад:

```python
def outer_func():
    msg = 'Hello there!'

    def inner_func():
        print(msg)

    inner_func()

outer_func() # Hello there!
```

У цьому прикладі внутрішня функція `inner_func` може вільно отримувати доступ до змінної `msg`, оголошеної у зовнішній функції `outer_func`. Проте зверніть увагу, що зовнішні функції не можуть отримувати доступ до змінних, оголошених у вкладених функціях:

```python
def outer_func():
    msg = 'Hello there!'
    print(res)

    def inner_func():
        res = 'How are you?'
        print(msg)

    inner_func()

outer_func() # NameError: name 'res' is not defined
```

Це тому, що `res` має локальну область видимості у `inner_func`. Також помітно, що `outer_func` намагається вивести `res` до виклику `inner_func`.

Одне з рішень — ініціалізувати `res` як порожній рядок в області видимості оточення, тобто всередині `outer_func`. Потім у `inner_func` зробити `res` нелокальною змінною за допомогою ключового слова `nonlocal`:

```python
def outer_func():
    msg = 'Hello there!'
    res = ""  # Declare res in the enclosing scope

    def inner_func():
        nonlocal res  # Allow modification of an enclosing variable
        res = 'How are you?'
        print(msg)  # Accessing msg from outer_func()

    inner_func()
    print(res)  # Now res is accessible and modified

outer_func()

# Output:
# Hello there!
# How are you?
```

**Глобальна область видимості** стосується змінних, оголошених поза будь-якими функціями або класами, до яких можна отримати доступ з будь-якої частини програми. Тут `my_var` доступна будь-де, навіть всередині функції, де вона не оголошена:

```python
my_var = 100

def show_var():
    print(my_var)

show_var() # 100
print(my_var) # 100
```

Якщо ви хочете зробити локальну змінну, оголошену всередині функції, глобально доступною, можна використати ключове слово `global`:

```python
my_var_1 = 7

def show_vars():
    global my_var_2
    my_var_2 = 10
    print(my_var_1)
    print(my_var_2)

show_vars() # 7 10

# my_var_2 is now a global variable and can be accessed anywhere in the program
print(my_var_2) # 10
```

Також можна використати ключове слово `global` для зміни глобальної змінної:

```python
my_var = 10  # A global variable

def change_var():
    global my_var  # Allows modification of a global variable
    my_var = 20

change_var()

print(my_var)  # my_var is now modified globally to 20
```

Нарешті, **вбудована область видимості** охоплює всі вбудовані функції, модулі та ключові слова Python, які доступні в будь-якій частині вашої програми:

```python
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```
