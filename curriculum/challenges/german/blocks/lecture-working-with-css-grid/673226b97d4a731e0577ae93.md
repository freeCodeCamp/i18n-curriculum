# --interactive--

Die Eigenschaft `grid-template-areas` ermöglicht es Ihnen, ein visuelles Grid-Layout mit benannten Bezeichnungen zu gestalten.

Anschließend weisen Sie die Bezeichnungen mit der Eigenschaft grid-area bestimmten Grid-Elementen zu. Anders gesagt, diese benannten Bezeichnungen werden auch „Grid-Bereichsnamen“ genannt.

Hier ist die grundlegende Syntax der Eigenschaft `grid-template-areas`:

```css
grid-template-areas:
 'header header header'
 'left-sidebar main right-sidebar'
 'footer footer footer';
```

Folgende Punkte sind bei der Grundsyntax wichtig:

- Werte wie `header` und `main` sind die Namen der Grid-Bereiche.

- Jeder durch Leerzeichen getrennte Wert innerhalb eines Strings entspricht einer Spalte.

- Jeder String stellt eine Zeile im Grid dar.

In der Syntax haben wir also einen 3 mal 3 Grid-Container.

Nachdem Sie die Vorlage definiert haben, müssen Sie die Eigenschaft `grid-area` verwenden und die benannten Bezeichnungen (oder Bereiche) als Werte angeben, um CSS mitzuteilen, dass ein bestimmtes Element zu diesem Bereich des Grids gehört.

Die Eigenschaft `grid-area` verbindet das Grid-Element mit dem benannten Bereich, den Sie in `grid-template-areas` definieren.

Eine beliebte Methode, um die Möglichkeiten der Eigenschaft `grid-template-areas` zu demonstrieren, ist die Erstellung des klassischen Holy-Grail-Layouts.

Falls Sie es nicht kennen: Das Holy-Grail-Layout ist ein Webdesign-Muster mit einer Kopfzeile, Fußzeile, zwei Seitenleisten und einem Hauptinhaltsbereich. Es stellt sicher, dass der Hauptinhalt Priorität hat, während Seitenleisten und andere Abschnitte sich responsiv innerhalb des Layouts anpassen.

Es gibt viele Lösungen, um das Holy-Grail-Layout zu implementieren, aber die Verwendung von `grid-template-areas` und der Eigenschaft `grid-area` ist der einfachste Weg, es zu erstellen.

Hier ist ein Beispiel für das Holy-Grail-Layout:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="header">
    <h2>Header</h2>
  </div>
  <div class="sidebar-left">
    <h2>Left Sidebar</h2>
  </div>
  <div class="main"><h2>Main Content</h2></div>
  <div class="sidebar-right">
    <h2>Right Sidebar</h2>
  </div>
  <div class="footer">
    <h2>Footer</h2>
  </div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-areas:
    'header header header'
    'sidebar-left main sidebar-right'
    'footer footer footer';
  gap: 10px;
  background-color: #2196F3;
  padding: 10px;
}

.header {
  grid-area: header;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.sidebar-left {
  grid-area: sidebar-left;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.main {
  grid-area: main;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.sidebar-right {
  grid-area: sidebar-right;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.footer {
  grid-area: footer;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}
```

:::

Bitte beachten Sie, dass sowohl die Eigenschaften `grid-template-areas` als auch `grid-area` unabhängig voneinander verwendet werden können.

Die Eigenschaft `grid-template-areas` wird speziell verwendet, um ein visuelles Layout zu definieren, indem benannte Grid-Bereiche innerhalb des Grid-Containers zugeordnet werden.

Die Eigenschaft `grid-area` hingegen wird verwendet, um einzelne Grid-Elemente zu positionieren, entweder durch Angabe ihrer Zeilen- und Spaltenpositionen oder durch Bezugnahme auf die benannten Bereiche, die mit der Eigenschaft `grid-template-areas` definiert wurden.
