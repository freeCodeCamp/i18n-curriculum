# --description--

Una `statement` en programación es una línea de código que realiza una acción específica. Por ejemplo, `print("Hello World")` es una sentencia que muestra texto.

Una `condition` es una prueba en el código que verifica si algo es verdadero o falso. Por ejemplo, `if x > 5` verifica si `x` es mayor que 5.

Un `conditional` es generalmente un adjetivo que se refiere a algo que depende de una `condition` o `conditions`. En programación, se usa para describir cualquier expresión o característica que involucra `conditions` lógicas. No se limita a la estructura explícita "si-entonces", sino que también puede referirse a cualquier configuración que altere el comportamiento basado en `conditions`, incluyendo bucles o incluso operadores ternarios.

Una `condition` es una sentencia que solo se ejecuta si una cierta `conditional statement` es verdadera.  For example, `if x > 5: print("Large")` is a `conditional statement`. Muestra `Large` solo si `x` es mayor que 5.

## --text--

¿Qué es una `conditional statement` en programación?

## --answers--

Una línea de código que se ejecuta sin verificar ninguna `conditions`

### --feedback--

Una `conditional statement` sí implica verificar `conditions`; no es una línea de código que se ejecuta sin ningún chequeo.

---

Una prueba para ver si un programa está funcionando correctamente

### --feedback--

Mientras que una `conditional statement` puede ayudar a verificar cómo funciona un programa, se usa principalmente para ejecutar código basado en `conditions`, no para probar el programa.

---

Una sentencia que muestra una `condition`

### --feedback--

Una `conditional statement` usa una `condition` para decidir si se ejecuta, pero no muestra una `condition` por sí misma.
