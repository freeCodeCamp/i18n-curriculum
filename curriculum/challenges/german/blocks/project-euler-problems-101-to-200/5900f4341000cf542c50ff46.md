# --description--

Drei Kreise mit gleichem Radius werden innerhalb eines größeren Kreises so platziert, dass jedes Kreis-Paar sich berührt und die inneren Kreise sich nicht überlappen. Es entstehen vier unbedeckte „Lücken“, die iterativ mit weiteren tangierenden Kreisen gefüllt werden sollen.

<img alt="a diagram of non-overlapping circles" src="https://cdn-media-1.freecodecamp.org/project-euler/199-circles-in-circles.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

In jeder Iteration wird in jede Lücke ein maximal großer Kreis gesetzt, wodurch neue Lücken für die nächste Iteration entstehen. Nach 3 Iterationen (im Bild dargestellt) gibt es 108 Lücken, und der Bruch des Bereichs, der nicht von Kreisen bedeckt ist, beträgt 0,06790342, gerundet auf acht Dezimalstellen.

Welcher Bruch des Bereichs ist nach `n` Iterationen nicht von Kreisen bedeckt? Geben Sie Ihre Antwort gerundet auf acht Dezimalstellen im Format x.xxxxxxxx an.

function iterativeCirclePacking(n) {
  let k1 = 1;
  let k0 = k1 * (3 - 2 * Math.sqrt(3));
  let a0 = 1 / (k0 * k0);
  let a1 = 3 / (k1 * k1);
  a1 += 3 * getArea(k0, k1, k1, n);
  a1 += getArea(k1, k1, k1, n);
  let final = ((a0 - a1) / a0).toFixed(8);

  return parseFloat(final);
  function getArea(k1, k2, k3, depth) {
      if (depth == 0) return 0.0;
      let k4 = k1 + k2 + k3 + 2 * Math.sqrt(k1 * k2 + k2 * k3 + k3 * k1);
      let a = 1 / (k4 * k4);
      a += getArea(k1, k2, k4, depth - 1);
      a += getArea(k2, k3, k4, depth - 1);
      a += getArea(k3, k1, k4, depth - 1);
      return a;
  }
}
