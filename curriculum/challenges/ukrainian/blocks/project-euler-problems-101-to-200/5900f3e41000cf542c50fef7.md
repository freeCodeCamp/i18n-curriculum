# --description--

Нехай `r` — це остача від ділення ${(a − 1)}^n + {(a + 1)}^n$ на $a^2$.

Наприклад, якщо $a = 7$ і $n = 3$, тоді $r = 42$: $6^3 + 8^3 = 728 ≡ 42 \\ \text{mod}\\ 49$. І коли `n` змінюється, змінюватиметься і `r`, але для $a = 7$ виявляється, що $r_{max} = 42$.

Для $3 ≤ a ≤ 1000$ знайдіть $\sum{r}_{max}$.
