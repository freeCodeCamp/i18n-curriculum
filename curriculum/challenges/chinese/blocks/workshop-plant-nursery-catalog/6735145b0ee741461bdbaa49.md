# --description--

备选地，你可以使用 `entries` 方法迭代你的 map 对象，该方法返回一个包含密钥-值点对的 map 迭代器。

修改你的 `displayCatalog` 函数以迭代你的 `catalog` 映射中的条目并将每个条目记录到控制台。

# --hints--

你的 `displayCatalog` 函数应调用 `catalog` 上的 `entries` 方法。

```js
const catalogCopy = new Map(catalog);
let flag = false;
const temp = catalog.entries;
try {
    catalog.entries = () => {
        flag = true;
        return catalogCopy.entries()
    };
    displayCatalog();    
    assert.isTrue(flag);
} finally {
    catalog.entries = temp;
}
```

你的 `displayCatalog` 函数应将 `catalog` 中包含的每个值记录到控制台日志。

```js
const testArr = [];
const temp = console.log;
try {
    console.log = obj => {testArr.push(obj)};
    displayCatalog();    
    assert.isNotEmpty(testArr);
    const ents = catalog.entries();
    for (let i = 0; i < catalog.size; i++) {        
        assert.deepEqual(testArr[i], ents.next().value)
    }
} finally {
    console.log = temp;
}
```
