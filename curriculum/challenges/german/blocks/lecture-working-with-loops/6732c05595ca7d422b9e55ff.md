# --interactive--

Eine `for...in`-Schleife eignet sich am besten, wenn Sie über die Eigenschaften eines Objekts iterieren müssen. Diese Schleife durchläuft alle aufzählbaren Eigenschaften eines Objekts, einschließlich geerbter Eigenschaften und nicht-numerischer Eigenschaften.

Eine geerbte Eigenschaft ist eine Eigenschaft, die von der Prototyp-Kette des Objekts geerbt wird. Eine nicht-numerische Eigenschaft ist eine Eigenschaft, die keine Zahl ist oder kein String, der in eine Zahl umgewandelt werden kann.

Hier ist die grundlegende Syntax einer `for...in`-Schleife:

```js
for (variable in object) {
  // code block to be executed
}
```

Die Variable im Beispiel stellt die aktuelle Eigenschaft des Objekts dar, über die gerade iteriert wird.

Schauen wir uns einige Beispiele an, damit Sie besser verstehen, wie die `for...in`-Schleife funktioniert.

Im ersten Beispiel haben wir ein `fruit`-Objekt und möchten über jede Eigenschaft iterieren und den Wert in der Konsole ausgeben.

:::interactive_editor

```js
const fruit = {
  name: 'apple',
  color: 'red',
  price: 0.99
};

for (const prop in fruit) {
  console.log(fruit[prop]);
}
```

:::

Die `prop`-Variable stellt die aktuelle Eigenschaft des Objekts dar. `fruit[prop]` wird verwendet, um auf den Wert jeder Eigenschaft zuzugreifen.

Bei der ersten Iteration ist `prop` `name`. Bei der zweiten Iteration ist `prop` `color` und so weiter.

Die in der Konsole ausgegebenen Ergebnisse sind `apple`, `red` und `0.99`.

Im zweiten Beispiel haben wir ein verschachteltes Objekt und möchten über jede Eigenschaft iterieren und den Wert in der Konsole ausgeben.

:::interactive_editor

```js
const person = {
  name: 'John',
  age: 30,
  address: {
    street: '123 Main St',
    city: 'Anytown',
    state: 'CA'
  }
};

for (const prop in person) {
  console.log(person[prop]);
}
```

:::

Die `address`-Eigenschaft ist selbst ein Objekt. Die `for...in`-Schleife wird auch über die Eigenschaften des `person`-Objekts iterieren und das gesamte `address`-Objekt in der Konsole ausgeben.

So sieht das Ergebnis in der Konsole aus:

```md
John
30
{ street: '123 Main St', city: 'Anytown', state: 'CA' }
```

Wenn Sie über die Eigenschaften des `address`-Objekts iterieren möchten, können Sie eine weitere `for...in`-Schleife innerhalb der ersten verschachteln.

:::interactive_editor

```js
const person = {
  name: 'John',
  age: 30,
  address: {
    street: '123 Main St',
    city: 'Anytown',
    state: 'CA'
  }
};

function isObject(obj) {
  return typeof obj === 'object' && !Array.isArray(obj) && obj !== null;
}

for (const prop in person) {
  if (isObject(person[prop])) {
    for (const nestedProp in person[prop]) {
      console.log(person[prop][nestedProp]);
    }
  } else {
    console.log(person[prop]);
  }
}
```

:::

In diesem Beispiel haben wir eine benutzerdefinierte Funktion `isObject`, die überprüft, ob der Wert ein Objekt ist.

Die `Array.isArray`-Methode wird verwendet, um zu prüfen, ob der Wert ein Array ist. Indem wir den logischen NICHT-Operator (`!`) vor die Methode setzen, prüfen wir, ob der Wert kein Array ist.

Der Grund, warum wir nicht einfach `typeof` gleich `'object'` verwenden können, ist, dass Arrays in JavaScript ebenfalls als Objekte gelten. Wir wollen Arrays von der Prüfung ausschließen.

Außerdem gibt `typeof null` aufgrund eines historischen Bugs in JavaScript `'object'` zurück. Deshalb wollen wir auch `null`-Werte von der Prüfung ausschließen.

Wenn die Bedingung wahr ist, verschachteln wir eine weitere `for...in`-Schleife, die über die Eigenschaften des verschachtelten Objekts iteriert und den Wert in der Konsole ausgibt.

Die `nestedProp`-Variable stellt die aktuelle Eigenschaft des verschachtelten Objekts dar.

So sieht das modifizierte Ergebnis in der Konsole aus:

```md
"John"
30
"123 Main St"
"Anytown"
"CA"
```

Eine `for...in`-Schleife ist nützlich, wenn Sie über die Eigenschaften eines Objekts iterieren müssen.

Es wird nicht empfohlen, eine `for...in`-Schleife zu verwenden, um über die Elemente eines Arrays zu iterieren. Verwenden Sie stattdessen eine `for...of`-Schleife oder andere Array-Methoden wie `forEach`, `map`, `filter` und `reduce`, die Sie in zukünftigen Lektionen kennenlernen werden.
