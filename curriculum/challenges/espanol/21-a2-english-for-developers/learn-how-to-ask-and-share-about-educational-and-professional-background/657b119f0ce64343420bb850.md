# --description--

Una `database` es una colección estructurada de datos, típicamente almacenada electrónicamente en un sistema informático. Se utiliza para organizar, almacenar, y recuperar grandes cantidades de información de manera eficiente. En programación, una `database` es esencial para gestionar los datos que las aplicaciones necesitan para funcionar.

# --questions--

## --text--

¿Qué es una `database` en el contexto de la programación?

## --answers--

Una colección estructurada de datos almacenada electrónicamente

---

Una herramienta para diseñar sitios web

### --feedback--

Aunque es importante en el desarrollo web, una Base de datos no es una herramienta para diseñar sitios web, sino para almacenar datos.

---

Un lenguaje de programación utilizado para crear aplicaciones

### --feedback--

Una Base de datos no es un lenguaje de programación; es un sistema para almacenar y gestionar datos.

---

Un tipo de virus informático

### --feedback--

Una Base de datos es una forma segura y estructurada de almacenar datos, no un virus informático.
