# --description--

Una `constant` nella programmazione è simile a una variabile, ma una volta che le viene assegnato un valore, quel valore non può essere modificato. Le `Constants` si usano quando hai valori che sai non dovranno essere modificati durante tutto il programma. Spesso vengono scritte in lettere maiuscole per distinguerle dalle variabili.

Per esempio, `const PI = 3.14;` In molti linguaggi di programmazione, `const` si usa per dichiarare una costante. Qui, `PI` è una `constant` con valore assegnato `3.14`.
