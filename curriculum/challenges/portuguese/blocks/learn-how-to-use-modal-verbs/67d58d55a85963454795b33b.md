# --explanation--

`Could` e `might` são verbos modais usados para falar sobre possibilidades.

`Could` significa que algo é possível mas não certo.Por exemplo:

`This update could make the app run faster.` - A atualização tem o potencial de melhorar a velocidade do app.

`Might` também expressa possibilidade mas frequentemente sugere uma chance menor do que `could`.Por exemplo:

A nova funcionalidade pode melhorar a experiência do usuário.
