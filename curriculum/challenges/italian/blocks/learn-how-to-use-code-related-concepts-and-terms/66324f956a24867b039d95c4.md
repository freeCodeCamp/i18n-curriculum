# --description--

La parola `should` è un verbo modale usato per dare consigli o suggerimenti. È seguita dalla forma base del verbo. Per esempio, nella frase `You should try that restaurant`, `should` è seguito da `try`, che è la forma base del verbo.

Il verbo `pick` significa scegliere o selezionare qualcosa da una gamma di opzioni. Nel contesto della programmazione, `picking a data type` significa scegliere il tipo di dato appropriato da usare, come un intero, una stringa o un booleano, in base a ciò che è più adatto all'attività da svolgere.

`Match` significa assicurarsi che due cose siano adatte o si combinino bene insieme. Per esempio, `This key matches the lock.` indica che la chiave è adatta alla serratura.
