# --explanation--

Jake kann `It's possible` sagen, weil er denkt, dass es eine Chance gibt, dass die verdächtige E-Mail Teil eines Phishing-Angriffs war.

`Impossible` wird gebildet, indem das Präfix `im-` (was „nicht“ oder „das Gegenteil von“ bedeutet) an das Grundwort `possible` angehängt wird. Zum Beispiel:

- `It's possible.` – Etwas kann passieren.

- `It's impossible.` – Es kann nicht passieren.
