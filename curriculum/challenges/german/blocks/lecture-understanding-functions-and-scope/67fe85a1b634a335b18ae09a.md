# --description--

In Python bestimmt der Gültigkeitsbereich den Punkt, an dem Sie auf eine Variable zugreifen können. Er steuert die Lebensdauer einer Variable und wie sie in verschiedenen Teilen des Codes aufgelöst wird.

Um den Gültigkeitsbereich korrekt zu bestimmen, folgt Python der **LEGB**-Regel, die für Folgendes steht:

- **Lokaler Gültigkeitsbereich (L)**: Variablen, die in Funktionen oder Klassen definiert sind.
    
- **Einschließender Gültigkeitsbereich (E)**: Variablen, die in einschließenden oder verschachtelten Funktionen definiert sind.
    
- **Globaler Gültigkeitsbereich (G):** Variablen, die auf Modulebene oder in der Datei definiert sind.
    
- **Eingebauter Gültigkeitsbereich (B)**: Reservierte Namen in Python für vordefinierte Funktionen, Module, Schlüsselwörter und Objekte.

Python verwendet die LEGB-Regel, um den Gültigkeitsbereich der Variablen in Ihrem Programm aufzulösen. Wir werden jede dieser Regeln genauer betrachten, damit Sie den Prozess besser verstehen.

**Lokaler Gültigkeitsbereich** bedeutet, dass eine Variable, die innerhalb einer Funktion oder Klasse deklariert wird, nur innerhalb dieser Funktion oder Klasse zugänglich ist.

Hier ist ein Beispiel:

```python
def my_func():
    my_var = 10
    print(my_var)
```

In diesem Fall hat die `my_func`-Funktion ihren eigenen Gültigkeitsbereich, auf den von außerhalb der Funktion nicht zugegriffen werden kann. Das Aufrufen von `my_func` gibt `10` aus, aber das Ausgeben von `my_var` außerhalb der Funktion führt zu einem `NameError`:

```python
def my_func():
    my_var = 10 # Locally scoped to my_func
    print(my_var)

my_func() # 10

print(my_var) # NameError: name 'my_var' is not defined
```

**Einschließender Gültigkeitsbereich** bedeutet, dass eine Funktion, die innerhalb einer anderen Funktion verschachtelt ist, auf die Variablen der umschließenden Funktion zugreifen kann.

Zum Beispiel:

```python
def outer_func():
    msg = 'Hello there!'

    def inner_func():
        print(msg)

    inner_func()

outer_func() # Hello there!
```

In diesem Beispiel kann die innere Funktion `inner_func` problemlos auf die Variable `msg` zugreifen, die in der äußeren Funktion `outer_func` definiert ist. Beachten Sie jedoch, dass äußere Funktionen nicht auf Variablen zugreifen können, die in verschachtelten Funktionen definiert sind:

```python
def outer_func():
    msg = 'Hello there!'
    print(res)

    def inner_func():
        res = 'How are you?'
        print(msg)

    inner_func()

outer_func() # NameError: name 'res' is not defined
```

Das liegt daran, dass `res` lokal auf `inner_func` beschränkt ist. Außerdem fällt auf, dass `outer_func` versucht, `res` auszugeben, bevor `inner_func` aufgerufen wird.

Eine Lösung besteht darin, `res` als leeren String im einschließenden Gültigkeitsbereich, also innerhalb von `outer_func`, zu initialisieren. Dann machen Sie innerhalb von `inner_func` `res` mit dem Schlüsselwort `nonlocal` zu einer nicht-lokalen Variable:

```python
def outer_func():
    msg = 'Hello there!'
    res = ""  # Declare res in the enclosing scope

    def inner_func():
        nonlocal res  # Allow modification of an enclosing variable
        res = 'How are you?'
        print(msg)  # Accessing msg from outer_func()

    inner_func()
    print(res)  # Now res is accessible and modified

outer_func()

# Output:
# Hello there!
# How are you?
```

**Globaler Gültigkeitsbereich** bezieht sich auf Variablen, die außerhalb von Funktionen oder Klassen deklariert sind und von überall im Programm zugänglich sind. Hier kann auf `my_var` überall zugegriffen werden, sogar innerhalb einer Funktion, in der sie nicht definiert ist:

```python
my_var = 100

def show_var():
    print(my_var)

show_var() # 100
print(my_var) # 100
```

Und wenn Sie eine lokal definierte Variable innerhalb einer Funktion global zugänglich machen möchten, können Sie das Schlüsselwort `global` verwenden:

```python
my_var_1 = 7

def show_vars():
    global my_var_2
    my_var_2 = 10
    print(my_var_1)
    print(my_var_2)

show_vars() # 7 10

# my_var_2 is now a global variable and can be accessed anywhere in the program
print(my_var_2) # 10
```

Sie können auch das Schlüsselwort `global` verwenden, um eine globale Variable zu ändern:

```python
my_var = 10  # A global variable

def change_var():
    global my_var  # Allows modification of a global variable
    my_var = 20

change_var()

print(my_var)  # my_var is now modified globally to 20
```

Schließlich bezieht sich der **eingebaute Gültigkeitsbereich** auf alle eingebauten Funktionen, Module und Schlüsselwörter von Python, die überall in Ihrem Programm verfügbar sind:

```python
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```
