# --explanation--

`Might` si usa per parlare di possibilità — qualcosa che potrebbe succedere ma non è certo. Per esempio:

- `It might rain tomorrow.` - C'è una possibilità, ma non è garantita.

- `We might need more time to finish the project.` - C'è una possibilità, ma non è confermata.
