# --interactive--

JavaScriptで文字列を扱うとき、大きな文字列から一部の部分文字列を抽出する必要がよくあります。

たとえば、単語の一部や特定の文字の並び、あるいは文の断片だけを取り出したい場合です。

JavaScriptにはこの作業のためのいくつかのメソッドがあり、その中でもよく使われるのが`slice()`メソッドです。

`slice()`メソッドは文字列の一部を抽出し、新しい文字列を返します。元の文字列は変更されません。2つのパラメータを取ります：開始インデックスと省略可能な終了インデックスです。

基本的な構文は次の通りです：

```js
string.slice(startIndex, endIndex);
```

`startIndex`は抽出を開始する位置です。`endIndex`は抽出を終了する位置です。指定しない場合、`slice()`は文字列の終わりまで抽出します。

文字列の一部を抽出する簡単な例を見てみましょう：

:::interactive_editor

```js
let message = "Hello, world!";
let greeting = message.slice(0, 5);

console.log(greeting);  // Hello
```

:::

この例では、`slice(0, 5)`はインデックス`0`から始まり、インデックス`5`の直前までの文字を抽出します。その結果、単語`Hello`が抽出されます。

2番目のパラメータを省略すると、`slice()`は開始インデックスから文字列の終わりまで全てを抽出します：

:::interactive_editor

```js
let message = "Hello, world!";
let world = message.slice(7);

console.log(world);  // world!
```

:::

ここでは、`slice(7)`がインデックス`7`から文字列の終わりまでを抽出し、結果は`world!`になります。

インデックスに負の数を使うこともできます。負の数を使うと、文字列の終わりから逆方向に数えます：

:::interactive_editor

```js
let message = "JavaScript is fun!";
let lastWord = message.slice(-4);

console.log(lastWord);  // fun!
```

:::

この場合、`slice(-4)`は文字列の最後の4文字を抽出し、結果は`fun!`になります。

文字列の途中から一部を抽出したい場合は、開始インデックスと終了インデックスの両方を指定して、抽出したい部分を正確に制御できます：

:::interactive_editor

```js
let message = "I love JavaScript!";
let language = message.slice(7, 17);

console.log(language);  // JavaScript
```

:::

ここでは、`slice(7, 17)`がインデックス7から始まり、インデックス`17`の直前までの部分文字列を抽出し、単語`JavaScript`になります。

`slice()`メソッドはJavaScriptで文字列の一部を抽出する強力なツールです。

開始と終了のインデックスを指定すると、抽出した部分を含む新しい文字列が返されます。

正の数、負の数、省略可能なインデックスのオプションがあり、元の文字列を変更せずにさまざまな状況に対応できます。
