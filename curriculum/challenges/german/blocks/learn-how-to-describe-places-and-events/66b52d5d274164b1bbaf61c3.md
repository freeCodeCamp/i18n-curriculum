# --explanation--

Das `Present Continuous`-Tempus wird verwendet, um Handlungen zu beschreiben, die im Moment des Sprechens stattfinden. Es wird gebildet mit dem Verb `to be` (`am, is, are`) + der Grundform des Verbs + `-ing`. Zum Beispiel:

- `She is coding a new feature right now.` – Sie ist gerade dabei, eine neue Funktion zu programmieren.

- `They are testing the software.` – Sie testen die Software gerade jetzt.
