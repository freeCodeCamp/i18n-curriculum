# --description--

Strings und Zahlen sind vielleicht unsere Bausteine, aber wenn Ihre Skripte komplexer werden, benötigen Sie eine Möglichkeit, mit großen Mengen davon umzugehen. Glücklicherweise hat JavaScript ein paar Datentypen, die genau dafür verwendet werden. Ein Array ist eine geordnete Sammlung von Elementen (Strings, Zahlen oder andere Dinge).

Arrays sind eine Möglichkeit, mehrere Werte in einer einzigen Variablen zu speichern. Sie sind ein spezieller Objekttyp, der eine Eigenschaft namens length und eine Reihe nummerierter Eigenschaften besitzt. Jede nummerierte Eigenschaft wird Element genannt, und jedes Element kann einen Wert beliebigen Typs speichern.

Ein Beispiel für ein Array ist:

```javascript
const fruits = ['apple', 'banana', 'orange'];
```
