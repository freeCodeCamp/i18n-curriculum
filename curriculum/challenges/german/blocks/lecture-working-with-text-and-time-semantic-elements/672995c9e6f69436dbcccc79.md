# --interactive--

In HTML werden zitierte Elemente verwendet, um zitierten Text vom umgebenden Inhalt zu unterscheiden. Dadurch erhält der zitierte Text ein Format, das leicht erkennbar ist.

Sie sollten das Blockzitat-Element verwenden, um einen Abschnitt darzustellen, der aus einer anderen Quelle zitiert wird. Es wird hauptsächlich für längere Zitate genutzt. Wenn die Quelle des Zitats eine Adresse hat, können Sie diese mit dem `cite`-Attribut angeben. Der Wert dieses Attributs sollte eine gültige URL sein. Hier ist ein Beispiel für ein Zitat innerhalb eines Blockzitat-Elements:

:::interactive_editor

```html
<blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
  "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
</blockquote>
```

:::

Dieses Element hat ein `cite`-Attribut. Der Wert des `cite`-Attributs ist die URL der Quelle. Obwohl dieses Attribut die Darstellung des Blockzitats nicht verändert, ist es sehr hilfreich, um Screenreadern und Suchmaschinen mehr Informationen über das Zitat zu geben. Im Browser sehen Sie, dass der Text leicht eingerückt ist.

Wenn Sie das Blockzitat mit Anführungszeichen beginnen und beenden möchten, müssen Sie diese möglicherweise explizit im Text schreiben. Sie können den Text direkt innerhalb des Blockzitat-Elements schreiben, wie ich es gerade getan habe, oder ihn in einen oder mehrere Absatz-Elemente einschließen. Das ist hilfreich, wenn der Text mehrere Absätze hat, die Sie aber im selben Blockzitat behalten möchten. Hier ist ein Beispiel mit vier Absätzen:

:::interactive_editor

```html
<blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
  <p>Build your projects. Show them to your friends. Build projects for your friends.</p>
  <p>Build your network. Help the people you meet along the way. What goes around comes around. You'll get what's coming to you.</p>   
  <p>It is not too late. Life is long.</p>
  <p>You will look back on this moment years from now and be glad you made a move.</p>
</blockquote>
```

:::

Alle Absätze sind im selben Blockzitat-Element enthalten, sodass sie Teil desselben Zitats sind. Sie sehen, dass das Element ein `cite`-Attribut mit der URL der Quelle hat. Im Browser sehen Sie, dass die vier Absätze zueinander ausgerichtet sind, aber gegenüber ihrem Container eingerückt sind.

Bisher habe ich das `cite`-Attribut verwendet, um die Quelle des Zitats anzugeben, aber das Attribut zeigt die Quelle dem Benutzer nicht sichtbar an. Es wirkt nur im Hintergrund.

Wenn Sie die Quelle sichtbar angeben möchten, können Sie ein Zitat-Element, `cite`, außerhalb des Blockzitat-Elements hinzufügen. Dies unterscheidet sich vom `cite`-Attribut. Das Zitat-Element ist ein HTML-Element, mit dem Sie den Titel eines referenzierten kreativen Werks wie eines Buchs, Artikels, Songs, Films, einer Website oder einer Forschungsarbeit markieren können. Hier ist ein Beispiel:

:::interactive_editor

```html
<div>
  <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
    Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?
  </blockquote>
  <p>—Quincy Larson, <cite>How to Learn to Code and Get a Developer Job [Full Book].</cite></p>
</div>
```

:::

Das Blockzitat-Element enthält den zitierten Text. Unterhalb des Elements sehen Sie ein Absatz-Element mit dem Namen des Autors, gefolgt von einem Zitat-Element. Das Zitat-Element enthält den Titel des Buchs, aus dem das Zitat stammt.

Wenn Sie im Browser nachsehen, ist die Quelle jetzt deutlich sichtbar, und Sie sehen, dass das Zitat aus einem Buch von Quincy Larson stammt. Der Titel dieses Buchs ist `How to Learn to Code and Get a Developer Job`.

Sie sollten Blockzitate wie diese für längere Zitate aus anderen Quellen verwenden. Manchmal müssen Sie jedoch nur wenige Wörter innerhalb eines größeren Absatzes zitieren.

Genau dafür ist das Inline-Zitat-Element gedacht. Es ist für kurze inline-Zitate aus anderen Quellen. Die meisten modernen Browser fügen automatisch Anführungszeichen um das Inline-Zitat hinzu, wenn Sie dieses Element verwenden. Hier ist ein Beispiel:

:::interactive_editor

```html
<p>
  As Quincy Larson said,
  <q cite="https://www.freecodecamp.org/news/learn-to-code-book/">
    Momentum is everything.
  </q>
</p>
```

:::

Sie sehen ein Absatz-Element, das Text enthält. Ein Teil des Texts ist ein Inline-Zitat, weil es innerhalb des Inline-Zitat-Elements steht. Sie können auch ein `cite`-Attribut hinzufügen, um die Quelle anzugeben.

Das funktioniert genau wie beim Blockzitat-Element. Es gibt keine sichtbaren Änderungen, aber es liefert Screenreadern und Suchmaschinen mehr Informationen über das Zitat.

Im Browser sehen Sie, dass der zitierte Text Teil des Absatzes ist und von Anführungszeichen umgeben wird. Die meisten modernen Browser fügen diese Anführungszeichen automatisch hinzu.

Was ist der Unterschied zwischen Blockzitaten und Inline-Zitaten? Sie sollten Blockzitate für längere Zitate aus anderen Quellen verwenden und Inline-Zitate für kurze Zitate aus anderen Quellen, die Teil bestehender Absätze sein sollen.
