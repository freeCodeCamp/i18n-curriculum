# --description--

`Step by step` bedeutet, etwas Schritt für Schritt zu tun. Zum Beispiel bedeutet `debugging step by step`, ein Problem in kleinen, überschaubaren Teilen zu lösen.

`Input` bedeutet beim Programmieren oft Informationen oder Ratschläge, die für ein Projekt gegeben werden.

`Work together` bedeutet zusammenzuarbeiten, und `pair up` ist ähnlich und bedeutet, dass zwei Personen eng zusammenarbeiten.

Schließlich bedeutet `to solve a problem`, zu versuchen, eine Lösung dafür zu finden. In diesem Fall versuchen Tom und Sarah, eine Lösung für einen möglichen Bug im Code zu finden.
