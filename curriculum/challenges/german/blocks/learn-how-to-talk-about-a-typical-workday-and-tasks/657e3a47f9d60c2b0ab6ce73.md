# --description--

Um `make sure` auszudrücken, verwendet man es, um Gewissheit oder Bestätigung über etwas auszudrücken. Zum Beispiel:

`I check the list twice to make sure I have everything.` – Das bedeutet, dass Sie die Liste zweimal ansehen, um sicherzugehen, dass nichts fehlt.

Um `work properly` auszudrücken, bedeutet, dass etwas richtig oder wie erwartet funktioniert. Zum Beispiel:

`It's important that all machines work properly for safety.` – Das bedeutet, dass die Maschinen richtig arbeiten müssen, damit die Menschen sicher bleiben.
