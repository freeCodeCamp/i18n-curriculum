# --explanation--

讓我們分解一下：

Mark 說了什麼？他說 BIOS `prevents the computer from starting.` 那表示他在談論 BIOS **可以做** 的事情，當它損壞時。 

如果 Sarah 同意，她會談論 BIOS **是什麼** 還是 BIOS **能做什麼**？是後者，對吧？這就是為什麼 `Yeah, it can` 是正確的選擇。

為什麼 `yeah, it is` 是錯誤的？因為 `it is` 談論的是 BIOS 的狀態，而不是它能做什麼。這裡有一個範例：

**正確**：`Yeah, it can.` – 這同意 Mark 關於 BIOS 能做什麼的觀點。

**錯誤**：`Yeah, it is.` – 這不合適，因為 Mark 並不是在描述 BIOS 是什麼，他是在談論它的效果。
