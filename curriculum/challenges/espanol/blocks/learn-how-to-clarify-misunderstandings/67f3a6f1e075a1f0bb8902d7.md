# --explanation--

`Updated` significa que algo ha sido revisado o cambiado para reflejar la información más reciente. Por ejemplo:

`The document has been updated with new data.` — Esto significa que el documento ahora incluye la información más actual.

`Timeline` se refiere a un plan que describe cuándo ocurrirán las tareas o eventos. Por ejemplo:

`The project timeline shows when each phase will be completed.` — Este es el calendario para completar un proyecto.
