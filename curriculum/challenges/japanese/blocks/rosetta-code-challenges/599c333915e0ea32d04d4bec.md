# --description--

基本的な要素ごとの行列-行列およびスカラー-行列の演算を実装してください。

**実装する内容：**

<ul>
  <li>加算</li>
  <li>減算</li>
  <li>乗算</li>
  <li>除算</li>
  <li>べき乗</li>
</ul>

最初のパラメータは実行する演算で、例えば行列の加算は "m_add"、スカラーの加算は "s_add" です。2番目と3番目のパラメータは演算対象の行列になります。

```js
function operation(op, arr1, arr2) {
  const ops = {
    add: ((a, b) => a + b),
    sub: ((a, b) => a - b),
    mult: ((a, b) => a * b),
    div: ((a, b) => a / b),
    exp: ((a, b) => Math.pow(a, b))
  };
  const ifm = op.startsWith('m');
  const doOp = ops[op.substring(2)];
  for (let i = 0; i < arr1.length; i++) {
    for (let j = 0; j < arr1[0].length; j++) {
      arr1[i][j] = doOp(arr1[i][j], (ifm) ? (arr2[i][j]) : (arr2));
    }
  }
  return arr1;
}
```
