# --interactive--

Le proprietà personalizzate CSS, note anche come variabili CSS, sono entità definite dagli autori CSS che contengono valori specifici da riutilizzare in tutto un documento. Sono una funzionalità potente che permette di creare fogli di stile più efficienti, manutenibili e flessibili.

La sintassi per dichiarare una proprietà personalizzata è semplice. Inizia con due trattini (`--`) seguiti dal nome della proprietà:

```css
:root {
  --main-color: #3498db;
}
```

In questo esempio, dichiariamo una proprietà personalizzata chiamata `--main-color` con un valore di `#3498db`. La pseudo-classe `:root` è comunemente usata per dichiarare proprietà personalizzate globali, poiché rappresenta il genitore di livello più alto nell'albero DOM.

Per usare una proprietà personalizzata, si impiega la funzione `var()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
:root {
  --main-color: #3498db;
}

.button {
  background-color: var(--main-color);
}
```

:::

Questo imposta il colore di sfondo degli elementi con la classe `button` al valore memorizzato in `--main-color`.

Una delle caratteristiche chiave delle proprietà personalizzate è che seguono la cascata CSS. Ciò significa che puoi ridefinirle per elementi o contesti specifici:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="alert">This is an alert message.</div>
```

```css
.alert {
  --main-color: #e74c3c;
  background-color: var(--main-color);
}
```

:::

In questo caso, gli elementi con la classe `alert` useranno un valore `--main-color` diverso, sovrascrivendo la definizione globale.

Le proprietà personalizzate supportano anche valori fallback. Se una proprietà personalizzata non è definita o è invalida, puoi fornire un valore di fallback:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="text">This is some text.</div>
```

```css
:root {
  --text-color: green;
}

.text {
  color: var(--text-color, green);
}
```

:::

Qui, se `--text-color` non è definita, `color` assumerà come valore predefinito `green`.

Le proprietà personalizzate sono particolarmente utili per creare design tematizzabili. Puoi definire un insieme di proprietà per temi diversi:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="light-theme">
  <p>This is some light themed text.</p>
</div>
```

```css
:root {
  --bg-color: black;
}

body {
  background-color: var(--bg-color);
}

.light-theme {
  --bg-color: white;
  --text-color: black;
  background-color: var(--bg-color);
  color: var(--text-color);
  padding: 10px;
}

```

:::

Cambiare tema diventa semplice come aggiungere o rimuovere una classe dall'elemento `body`.

Le proprietà personalizzate possono essere usate anche con le media query per creare design responsivi:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <h2>Responsive Design</h2>
  <p>Resize the window to see the card adapt!</p>
</div>

```

```css
:root {
  --card-width: 90%;
  --card-bg: #f0f0f0;
  --card-padding: 1rem;
  --text-color: #333;
}

/* Tablet screens and up */
@media (min-width: 600px) {
  :root {
    --card-width: 70%;
    --card-bg: #e8f5e9;
    --card-padding: 1.5rem;
  }
}

/* Desktop screens and up */
@media (min-width: 1024px) {
  :root {
    --card-width: 50%;
    --card-bg: #d0f0ff;
    --card-padding: 2rem;
  }
}

body {
  font-family: system-ui, sans-serif;
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
  background-color: #fafafa;
}

.card {
  width: var(--card-width);
  background-color: var(--card-bg);
  padding: var(--card-padding);
  color: var(--text-color);
  border-radius: 8px;
  box-shadow: 0 2px 6px rgba(0,0,0,0.1);
  text-align: center;
  transition: all 0.3s ease;
}

```

:::

Questo approccio permette di avere layout responsivi più manutenibili, poiché puoi modificare i valori in un solo posto invece che in tutto il foglio di stile. Questo può ridurre significativamente la ripetizione nei fogli di stile, specialmente per valori complessi usati frequentemente.

Le proprietà personalizzate possono anche fare riferimento ad altre proprietà personalizzate:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
:root {
  --primary-color: #3498db;
  --button-bg: var(--primary-color);
}

button {
  background-color: var(--button-bg);
}

```

:::

Questo consente di creare relazioni tra diversi aspetti dello stile, facilitando il mantenimento di temi coerenti in tutto il sito.

Anche se le proprietà personalizzate offrono molti vantaggi, è importante essere consapevoli del loro supporto nei browser. Sono ben supportate nei browser moderni, ma quelli più vecchi potrebbero non riconoscerle. Assicurati sempre di avere un piano di fallback quando usi funzionalità all’avanguardia.

In conclusione, le proprietà personalizzate CSS offrono un modo potente per creare fogli di stile più dinamici, flessibili e manutenibili. Permettono di creare design tematizzabili, semplificano i layout responsivi e abilitano la manipolazione degli stili a runtime.

Con l’evoluzione dello sviluppo web, le proprietà personalizzate stanno diventando uno strumento sempre più importante nel kit dello sviluppatore, offrendo nuove possibilità per creare CSS adattabili ed efficienti.
