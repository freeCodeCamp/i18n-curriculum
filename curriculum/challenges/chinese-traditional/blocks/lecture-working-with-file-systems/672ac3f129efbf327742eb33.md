# --description--

讓我們學習如何在 Windows 和 macOS 上搜尋檔案和資料夾。

我們將從 Windows 開始。你可以從工作列或檔案總管進行搜尋。

要從工作列搜尋，請輸入你正在尋找的檔案或資料夾名稱。你也可以搜尋關鍵字。你將會看到來自整個電腦和 OneDrive 的結果。

如果你只想查看文件，可以點擊「Documents」標籤。你也有用於 apps 和 web results 的標籤。

另一個選擇是使用搜尋檔案總管。你會在檔案總管視窗的右上方找到它，那裡可以看到「搜尋文件」。

你有三個選項，可定義搜尋的範圍：

- 你可以從 Home 進行搜尋，以快速找到來自你的 PC 和雲端的檔案。

- 你可以從資料夾中搜尋以找到儲存在其中的檔案。

- 你可以從 This PC 進行緩慢但深入的搜尋。

這是你如何在 Windows 上搜尋檔案和資料夾。現在讓我們看看你如何在 macOS 上做到這一點。

首先，你可以使用 Spotlight，這是一個幫助你在 Mac 上尋找檔案的工具。要開啟 Spotlight，請前往右上角的選單列並找到放大鏡圖示。如果你在選單列看不到此圖示，可以在控制元件中心設定中將它添加進去。

前往你的設定，篩選「控制中心」，點擊此選項，然後向下捲動直到找到「Spotlight」選項。你也可以使用鍵盤快速鍵開啟 Spotlight：`Command + Spacebar`。

當你開啟 Spotlight 時，你會看到 Spotlight 搜尋，在那裡你可以透過輸入名稱來搜尋任何檔案或資料夾。當你開始輸入時，你會看到依類別分組的結果，包括建議、資料夾、簡報、照片、文件和相關搜尋。

你也可以點擊「Show More」來展開特定類別。

另一種搜尋你最近開啟過的檔案的方法是點擊 Dock 中的 Finder 圖示來開啟 Finder。然後，前往 Finder 側邊欄中的「最近使用」。在那裡，你會找到所有你最近查看過的檔案。

這是你如何在 macOS 上搜尋檔案和資料夾的方法。使用這些搜尋工具，你可以最佳化你的工作流程，並且非常快速地找到你需要的檔案和資料夾。
