# --description--

Betrachten Sie den Bereich, der durch $1 ≤ x$ und $0 ≤ y ≤ \frac{1}{x}$ eingeschränkt ist.

Sei $S_1$ das größte Quadrat, das unter die Kurve passt.

Sei $S_2$ das größte Quadrat, das in den verbleibenden Bereich passt, und so weiter.

Der Index von $S_n$ ist das Paar (links, unten), das die Anzahl der Quadrate links von $S_n$ und die Anzahl der Quadrate unter $S_n$ angibt.

<img alt="Diagramm mit Quadraten unter der Hyperbel" src="https://cdn.freecodecamp.org/curriculum/project-euler/squares-under-a-hyperbola.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Das Diagramm zeigt einige solcher Quadrate mit Nummern als Bezeichnung.

$S_2$ hat ein Quadrat links davon und keines darunter, daher ist der Index von $S_2$ (1, 0).

Man sieht, dass der Index von $S_{32}$ (1,1) ist, ebenso wie der Index von $S_{50}$.

50 ist das größte $n$, für das der Index von $S_n$ (1, 1) ist.

Was ist das größte $n$, für das der Index von $S_n$ (3, 3) ist?
