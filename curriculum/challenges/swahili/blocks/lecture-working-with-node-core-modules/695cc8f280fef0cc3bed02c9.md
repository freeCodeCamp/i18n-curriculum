# --description--

Moduli ya OS ni moduli nyingine ya kawaida ambayo imejumuishwa ndani ya Node.js.

Inakuwezesha kuingiliana na mfumo wa uendeshaji unaotumika sasa ambapo Node inafanya kazi ili upate taarifa muhimu kama aina ya OS, maelezo ya prosesa (CPU), kumbukumbu inayopatikana, jumla ya kumbukumbu, miunganisho ya mtandao, na zaidi.

Ili kutumia moduli ya OS, uilete kwa njia hii:

```js
const os = require("os");
```

Moduli ya OS hutoa njia kadhaa unazoweza kutumia kufanya programu yako iwe na ufahamu zaidi wa mfumo. Tuchunguze baadhi ya njia maarufu.

Njia ya `platform()` inapata mfumo wa uendeshaji ambao Node inafanya kazi sasa:

```js
const os = require("os");
console.log(os.platform()); // darwin
```

Hii inaweza kuwa na manufaa kwa kutekeleza maandishi yanayofanya kazi kwenye mifumo mingi (cross-platform scripting):

```js
if (os.platform() === 'win32') {
 // Windows specific code
} else {
 // Non-Windows specific code
}
```

Njia ya `arch()` inaonyesha mfuatano wa herufi unaowakilisha usanifu wa prosesa (CPU) ambao Node.js ilikusudiwa kuendeshwa kwa ajili yake:

```js
const os = require("os");
console.log(os.arch()); // arm64
```

Hii inaweza kusaidia ikiwa unataka watumizi kupakua binaries na utegemezi sahihi kwa usanifu fulani.

Thamani nyingine zinazowezekana ni `'x64'`, `'arm'`, `'arm64'`, `'ia32'`, `'mips'`, `'ppc'`, na zingine.

`type()` hupata jina rasmi la OS, ili uweze kutambua mifumo ya uendeshaji kwa njia ya programu:

```js
const os = require("os");
console.log(os.type()); // Darwin (the core OS for macOS, iOS, and other Apple products)
```

`release()` inaonyesha toleo la kernel ya OS ya mfumo, sehemu kuu ya mfumo wa uendeshaji inayosimamia rasilimali za mfumo na mawasiliano kati ya vifaa na sehemu za programu. Njia hii inaweza kusaidia kufuatilia ulinganifu kati ya matoleo ya kernel ya OS na mahitaji ya seva.

Inarudisha hii kama mfuatano wa herufi kama `'20.6.0'`:

```js
const os = require("os");
console.log(os.release()); // 25.0.0
```

`version()` inarudisha toleo maalum la mfumo wa uendeshaji lenye maelezo zaidi kuliko njia ya `release()`:

```js
const os = require("os");
console.log(os.version());
// Darwin Kernel Version 25.0.0: Wed Sep 17 21:41:39 PDT 2025;
// root:xnu-12377.1.9~141/RELEASE_ARM64_T8103
```

`cpus()` inarudisha safu ya vitu vyenye maelezo kuhusu kila kiini cha prosesa (CPU) cha kimantiki. Hii inaweza kusaidia kufuatilia mzigo wa prosesa:

`uptime()` inaonyesha muda tangu mfumo ulipoanzishwa. Inaweza kusaidia kubaini muda ambao seva zimekuwa zikiendesha:

```js
const os = require("os");
console.log(os.uptime()); // 23047
```

`totalmem()` na `freemem()` zinaonyesha jumla ya kumbukumbu ya mfumo kwa bait na kumbukumbu ya bure ya mfumo kwa bait, mtawalia:

```js
const os = require("os");
console.log(os.totalmem()); // 8589934592 (8 GB)
console.log(os.freemem()); // 93585408 (87 MB)
```

`userInfo()` inarudisha kitu chenye taarifa kuhusu mtumizi wa mfumo wa sasa:

```js

```
