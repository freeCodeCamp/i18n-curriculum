## Grundlagen von Strings

- **Definition**: Ein String ist eine Folge von Zeichen, die entweder in einfache Anführungszeichen, doppelte Anführungszeichen oder Ticks eingeschlossen sind. Strings sind primitive Datentypen und sie sind unveränderlich. Unveränderlichkeit bedeutet, dass ein String, sobald er erstellt wurde, nicht mehr geändert werden kann. 
- **Zugreifen auf Zeichen eines Strings**: Um auf ein Zeichen eines Strings zuzugreifen, können Sie die Klammernotation verwenden und die Indexnummer übergeben. Ein Index ist die Position eines Zeichens innerhalb eines Strings, und er ist nullbasiert.

:::interactive_editor

```js
const developer = "Jessica";
console.log(developer[0]); // J
```

:::

- **`\n` (Zeilenumbruchzeichen)**: Sie können in einem String einen Zeilenumbruch erzeugen, indem Sie das `\n` Zeilenumbruchzeichen verwenden.

:::interactive_editor

```js
const poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

- **Escape-Sequenzen in Strings**: Sie können Zeichen in einem String maskieren, indem Sie Backslashes (`\`) vor die Anführungszeichen setzen.

:::interactive_editor

```js
const statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

## Template-Literale (Template Strings) und String-Interpolation

- **Definition**: Template-Literale werden mit Ticks (`) definiert. Sie ermöglichen eine einfachere String-Manipulation, einschließlich des direkten Einbettens von Variablen in einen String, eine Funktion, die als String-Interpolation bekannt ist.

:::interactive_editor

```js
const name = "Jessica";
const greeting = `Hello, ${name}!`; 
console.log(greeting); // "Hello, Jessica!"
```

:::

## ASCII, die `charCodeAt()`-Methode und die `fromCharCode()`-Methode

- **ASCII**: ASCII (American Standard Code for Information Interchange) ist ein Zeichencodierungsstandard, der verwendet wird, um grundlegende englische Zeichen mit numerischen Werten darzustellen. Frühere Lektionen führen `charCodeAt()` und `fromCharCode()` anhand von ASCII-Beispielen ein.
- **Unicode**: JavaScript-Strings verwenden intern Unicode, speziell die UTF-16-Codierung. Für die ersten 128 Zeichen (Grundlateinische Buchstaben, Ziffern und gängige Symbole) stimmen die Unicode-Werte mit den ASCII-Codes überein. Deshalb funktionieren ASCII-basierte Beispiele weiterhin in JavaScript.
- **Die `charCodeAt()`-Methode**: Diese Methode gibt die UTF-16-Code-Einheit des Zeichens an einem bestimmten Index zurück. Für grundlegende lateinische Zeichen entspricht dieser Wert dem ASCII-Code.

:::interactive_editor

```js
const letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

:::

- **Die `fromCharCode()`-Methode**: Diese Methode wandelt einen ASCII-Code in das entsprechende Zeichen um.

:::interactive_editor

```js
const char = String.fromCharCode(65);
console.log(char);  // A
```

:::

## Weitere gebräuchliche String-Methoden

- **Die `indexOf`-Methode**: Diese Methode wird verwendet, um in einem String nach einer Teilfolge zu suchen. Wenn die Teilfolge gefunden wird, gibt `indexOf` den Index (oder die Position) des ersten Vorkommens dieser Teilfolge zurück. Wenn die Teilfolge nicht gefunden wird, gibt `indexOf` -1 zurück, was anzeigt, dass die Suche erfolglos war.

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.indexOf("fox")); // 16
console.log(text.indexOf("cat")); // -1
```
  
:::  

- **Die `includes()`-Methode**: Diese Methode wird verwendet, um zu überprüfen, ob ein String eine bestimmte Teilfolge enthält. Wenn die Teilfolge im String gefunden wird, gibt die Methode true zurück. Andernfalls gibt sie false zurück.

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.includes("fox")); // true
console.log(text.includes("cat")); // false
```

:::

- **Die `slice()`-Methode**: Diese Methode extrahiert einen Teil eines Strings und gibt einen neuen String zurück, ohne den ursprünglichen String zu verändern. Sie nimmt zwei Parameter: den Startindex und den optionalen Endindex.

:::interactive_editor

```js
const text = "freeCodeCamp";
console.log(text.slice(0, 4));  // "free"
console.log(text.slice(4, 8));  // "Code"
console.log(text.slice(8, 12)); // "Camp"
```

:::

- **Die `toUpperCase()`-Methode**: Diese Methode wandelt alle Zeichen in Großbuchstaben um und gibt einen neuen String mit allen Großbuchstaben zurück.

:::interactive_editor

```js
const text = "Hello, world!";
console.log(text.toUpperCase()); // "HELLO, WORLD!"
```

:::

- **Die `toLowerCase()`-Methode**: Diese Methode wandelt alle Zeichen in einem String in Kleinbuchstaben um.

:::interactive_editor

```js
const text = "HELLO, WORLD!"
console.log(text.toLowerCase()); // "hello, world!"
```

:::

- **Die `replace()`-Methode**: Diese Methode ermöglicht es Ihnen, einen bestimmten Wert (wie ein Wort oder Zeichen) in einem String zu finden und durch einen anderen Wert zu ersetzen. Die Methode gibt einen neuen String mit der Ersetzung zurück und lässt den ursprünglichen unverändert, da JavaScript-Strings unveränderlich sind.

:::interactive_editor

```js
const text = "I like cats";
console.log(text.replace("cats", "dogs")); // "I like dogs"
```

:::

- **Die `replaceAll()`-Methode**: Diese Methode ermöglicht es Ihnen, alle Vorkommen eines bestimmten Werts (eines Wortes, Zeichens oder Musters) in einem String zu finden und durch einen anderen Wert zu ersetzen. Sie funktioniert wie `replace()`, aber anstatt nach dem ersten Treffer zu stoppen, aktualisiert sie jeden gefundenen Treffer im String.

:::interactive_editor

```js
const text = "I love cats and cats are so much fun!";
console.log(text.replaceAll("cats", "dogs")); // "I love dogs and dogs are so much fun!"
```

:::

- **Die `repeat()`-Methode**: Diese Methode wird verwendet, um einen String eine bestimmte Anzahl von Malen zu wiederholen.

:::interactive_editor

```js
const text = "Hello";
console.log(text.repeat(3)); // "HelloHelloHello"
```

:::

- **Die `trim()`-Methode**: Diese Methode wird verwendet, um Leerzeichen sowohl am Anfang als auch am Ende eines Strings zu entfernen.

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trim()); // "Hello, world!"
```

:::

- **Die `trimStart()`-Methode**: Diese Methode entfernt Leerzeichen vom Anfang (oder „Start“) des Strings.

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trimStart()); // "Hello, world!  "
```

:::

- **Die `trimEnd()`-Methode**: Diese Methode entfernt Leerzeichen vom Ende des Strings.

:::interactive_editor

```js
const text = " Hello, world! ";
console.log(text.trimEnd()); // "  Hello, world!"
```

:::

- **Die `prompt()`-Methode**: Diese Methode des `window` wird verwendet, um Informationen von einem Benutzer in Form eines Dialogfelds zu erhalten. Diese Methode nimmt zwei Argumente entgegen. Das erste Argument ist die Nachricht, die im Dialogfeld erscheint und den Benutzer typischerweise auffordert, Informationen einzugeben. Das zweite ist ein Standardwert, der optional ist und das Eingabefeld initial ausfüllt.

```js
const answer = window.prompt("What's your favorite animal?"); // This will change depending on what the user answers
```
