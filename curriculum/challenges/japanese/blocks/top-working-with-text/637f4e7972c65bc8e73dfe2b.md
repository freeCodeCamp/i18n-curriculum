# --description--

このレッスンのすべての例で、他の要素の中にある要素をインデントしていることに気づいたかもしれません。これは要素をネストするという意味です。

要素を他の要素の中にネストすると、それらの間に親子関係ができます。ネストされた要素は子であり、ネストされている要素が親になります。

次の例では、`body` 要素が親で、`p` が子です。

```html
<html>
  <head>
  </head>
  <body>
    <p>Lorem ipsum dolor sit amet.</p>
  </body>
 </html>
```

人間関係と同じように、HTMLの親要素は複数の子を持つことができます。同じネストレベルにある要素は兄弟要素とみなされます。

たとえば、次のコードの2つの `p` 要素は、どちらも `body` タグの子であり、同じネストレベルにあるため兄弟要素です。

```html
<html>
  <head>
  </head>
  <body>
    <p>Lorem ipsum dolor sit amet.</p>
    <p>Ut enim ad minim veniam.</p>
  </body>
 </html>
```

インデントを使うことで、ネストのレベルを自分自身や将来あなたのHTMLを扱う他の開発者にとってわかりやすく読みやすくします。子要素は2スペース分インデントすることをお勧めします。

要素間の親子や兄弟の関係は、後でCSSでHTMLのスタイルを設定したり、JavaScriptで動作を追加したりするときに非常に重要になります。しかし今は、要素がどのように関連しているか、その関係を説明する用語を知っておくことが大切です。
