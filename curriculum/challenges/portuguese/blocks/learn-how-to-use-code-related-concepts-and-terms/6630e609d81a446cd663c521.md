# --description--

O verbo `to store` em programação refere-se ao ato de manter ou salvar dados para que possam ser usados depois.Escolher os tipos de dados corretos é fundamental porque isso influencia como a informação é `stored` e usada em um programa.

Diferentes tipos de dados são adequados para diferentes tipos de informações, como texto, números ou valores booleanos.Por exemplo, para `store` texto, um programador pode usar o tipo de dado `string`.
