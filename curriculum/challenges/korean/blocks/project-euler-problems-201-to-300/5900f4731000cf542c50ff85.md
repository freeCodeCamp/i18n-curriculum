# --description--

다음 방정식은 산악 지역의 연속적인 지형을 나타내며, 임의의 점 ($x$,$y$)에서의 고도 $h$를 제공합니다:

$$h = \left(5000 - \frac{x^2 + y^2 + xy}{200} + \frac{25(x + y)}{2}\right) \times e^{-\left|\frac{x^2 + y^2}{1\\,000\\,000} - \frac{3(x + y)}{2000} + \frac{7}{10}\right|}$$

모기는 $0 ≤ x$, $y ≤ 1600$로 주어진 영역을 벗어나지 않고 A(200,200)에서 B(1400,1400)까지 날아가려고 합니다.

중간에 산이 있기 때문에, 먼저 고도 $f$인 점 A'까지 수직으로 상승합니다. 그런 다음, 같은 고도 $f$를 유지하면서 장애물을 피해 B 바로 위의 점 B'까지 날아갑니다.

먼저, 지정된 영역 내에서 A에서 B까지 이동할 수 있게 하는 최솟값 고도 $f_{min}$을 구하세요. 그다음, 고도 $f_{min}$에서 A'와 B' 사이의 최단 경로 길이를 구하세요.

답으로 그 길이를 소수점 셋째 자리까지 반올림하여 제출하세요.

**참고:** 편의성을 위해 위에 제시된 고도 함수가 대부분의 프로그래밍 언어에 적합한 형태로 아래에 다시 나와 있습니다: `h=( 5000-0.005*(x*x+y*y+x*y)+12.5*(x+y) )* exp( -abs(0.000001*(x*x+y*y)-0.0015*(x+y)+0.7) )`.
