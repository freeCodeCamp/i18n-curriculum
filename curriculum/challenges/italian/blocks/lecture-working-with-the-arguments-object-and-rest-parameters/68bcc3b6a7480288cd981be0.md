# --interactive--

Come ricorderai dalle lezioni e dai workshop precedenti, puoi creare funzioni con un certo numero di parametri e chiamare quella funzione con degli argomenti.

Ecco un promemoria:

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call
getSum(3, 4); // 7
```

Ma cosa succede se hai una funzione chiamata con più argomenti di quanti ne sia stata definita per accettare?

:::interactive_editor

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call with extra argument
console.log(getSum(3, 4, 5)); // 7
```

:::

JavaScript non lancerà un errore in questo caso. Invece ignorerà l'argomento extra e sommerà semplicemente i numeri `3` e `4`. Le funzioni che accettano un numero variabile di argomenti sono conosciute come <dfn>funzioni variadiche</dfn>.

Se lavori con funzioni variadiche, puoi utilizzare l'oggetto `arguments`. Questo oggetto simile a un array contiene i valori degli argomenti passati alla funzione.

Ecco un esempio:

:::interactive_editor

```js
function logArgs() {
  for (const arg of arguments) {
    console.log(arg);
  }
}

logArgs(1, 2, 3);
// result:
// 1
// 2
// 3

logArgs("example"); // "example"
```

:::

Poiché l'oggetto `arguments` è simile a un array, puoi consultare un argomento a un indice specifico così:

:::interactive_editor

```js
function getArg() {
  return arguments[1];
}

console.log(getArg(2, 4, 6)); // 4
```

:::

Puoi anche usare la proprietà `length` in questo modo per ottenere il numero di argomenti con cui la funzione è stata chiamata:

:::interactive_editor

```js
function getArgs() {
  return arguments.length;
}

console.log(getArgs("Example")); // 1
console.log(getArgs("Another", "Example")); // 2
```

:::

Anche se l'oggetto `arguments` sembra comportarsi come un vero array, non possiede metodi `Array` integrati come `includes` o `push`. Per avere accesso a quei metodi, dovresti prima convertire l'oggetto `arguments` in un vero array usando qualcosa come `slice`, `Array.from()` o l'operatore di espansione:

:::interactive_editor

```js
function hasCat() {
  return [...arguments].includes("cat");
}

console.log(hasCat("dog", "chicken", "cat")); // true
console.log(hasCat("dog", "chicken", "horse")); // false
```

:::

Anche se è possibile lavorare con l'oggetto `arguments` per funzioni variadiche, le applicazioni JavaScript moderne normalmente usano la sintassi dei parametri rest. Imparerai di più su questo in una lezione futura.
