# --explanation--

`Of course`는 정중하게 동의를 표현하거나 주저 없이 `yes`라고 말하는 방법입니다. 이는 화자가 기꺼이 또는 기쁘게 무언가를 하겠다는 뜻입니다. 예를 들어:

`Of course, I'll help you with the report.` – 화자가 기꺼이 도와주겠다는 뜻입니다.

`Will`는 약속을 하거나 말하는 순간 결정을 표현할 때 자주 사용합니다. 예를 들어:

`I'll email you the file this afternoon.` – 곧 무언가를 하겠다는 약속을 나타냅니다.
