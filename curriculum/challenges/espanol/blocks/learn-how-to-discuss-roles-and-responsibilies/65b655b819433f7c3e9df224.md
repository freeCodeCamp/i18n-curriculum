# --description--

La frase `word is` se utiliza cuando compartes información o rumores que has escuchado de otros. Es como decir `people say` o `I've heard`.

Por ejemplo, si alguien dice, `Word is that the new update will make the program faster`, quiere decir que ha escuchado que la nueva actualización mejorará la velocidad del programa.
