## Pseudoklassen für Benutzeraktionen

- **Definition von Pseudoklassen**: Dabei handelt es sich um spezielle CSS-Schlüsselwörter, mit denen Sie ein Element basierend auf seinem bestimmten Zustand oder seiner Position auswählen können.
- **Pseudoklassen für Benutzeraktionen**: Diese speziellen Schlüsselwörter ermöglichen es Ihnen, das Aussehen von Elementen basierend auf Benutzerinteraktionen zu ändern und so die Benutzererfahrung zu verbessern.
- **`:active` Pseudoklasse**: Diese Pseudoklasse erlaubt es Ihnen, den aktiven Zustand eines Elements auszuwählen, zum Beispiel beim Klicken auf eine Schaltfläche.
- **`:hover` Pseudoklasse**: Diese Pseudoklasse definiert den Hover-Zustand eines Elements.
- **`:focus` Pseudoklasse**: Diese Pseudoklasse wendet Stile an, wenn ein Element den Fokus erhält, typischerweise durch Tastaturnavigation oder wenn ein Benutzer in ein Formulareingabefeld klickt.
- **`:focus-within` Pseudoklasse**: Diese Pseudoklasse wird verwendet, um Stile auf ein Element anzuwenden, wenn es selbst oder eines seiner Nachfahren den Fokus hat.

## Pseudoklassen für Eingaben

- **Pseudoklassen für Eingaben**: Diese Pseudoklassen werden verwendet, um HTML `input` Elemente basierend auf dem Zustand auszuwählen, in dem sie sich vor und nach der Benutzerinteraktion befinden.
- **`:enabled` Pseudoklasse**: Diese Pseudoklasse wird verwendet, um Formularschaltflächen oder andere Elemente auszuwählen, die derzeit aktiviert sind.
- **`:disabled` Pseudoklasse**: Diese Pseudoklasse ermöglicht es Ihnen, ein interaktives Element im deaktivierten Modus zu gestalten.
- **`:checked` Pseudoklasse**: Diese Pseudoklasse wird verwendet, um dem Benutzer anzuzeigen, dass es aktiviert ist.
- **`:valid` Pseudoklasse**: Diese Pseudoklasse zielt auf Eingabefelder ab, die die Validierungskriterien erfüllen.
- **`:invalid` Pseudoklasse**: Diese Pseudoklasse zielt auf Eingabefelder ab, die die Validierungskriterien nicht erfüllen.
- **`:in-range` und `:out-of-range` Pseudoklassen**: Diese Pseudoklassen wenden Stile auf Elemente an, je nachdem, ob deren Werte innerhalb oder außerhalb der angegebenen Bereichsbeschränkungen liegen.
- **`:required` Pseudoklasse**: Diese Pseudoklasse zielt auf `input` Elemente ab, die das `required` Attribut besitzen. Sie signalisiert dem Benutzer, dass das Feld ausgefüllt werden muss, um das Formular abzusenden.
- **`:optional` Pseudoklasse**: Diese Pseudoklasse wendet Stile auf Eingabeelemente an, die nicht erforderlich sind und leer bleiben können.
- **`:autofill` Pseudoklasse**: Diese Pseudoklasse wendet Stile auf Eingabefelder an, die der Browser automatisch mit gespeicherten Daten ausfüllt.

## Pseudoklassen für Positionen

- **Pseudoklassen für Positionen**: Diese Pseudoklassen werden verwendet, um Links und Elemente zu gestalten, die innerhalb des aktuellen Dokuments angesprochen werden.
- **`:any-link` Pseudoklasse**: Diese Pseudoklasse ist eine Kombination der `:link` und `:visited` Pseudoklassen. Sie trifft also auf jedes Ankerelement mit einem href-Attribut zu, unabhängig davon, ob es besucht wurde oder nicht.
- **`:link` Pseudoklasse**: Diese Pseudoklasse ermöglicht es Ihnen, alle nicht besuchten Links auf einer Webseite auszuwählen. Sie können sie verwenden, um Links vor dem Klicken unterschiedlich zu gestalten.
- **`:local-link` Pseudoklasse**: Diese Pseudoklasse zielt auf Links ab, die auf dasselbe Dokument verweisen. Sie ist nützlich, wenn Sie interne Links von externen unterscheiden möchten.
- **`:visited` Pseudoklasse**: Diese Pseudoklasse zielt auf einen Link ab, den der Benutzer bereits besucht hat.
- **`:target` Pseudoklasse**: Diese Pseudoklasse wird verwendet, um Stile auf ein Element anzuwenden, das Ziel eines URL-Fragments ist.

## Baumstruktur-Pseudoklassen

- **Baumstruktur-Pseudoklassen**: Diese Pseudoklassen ermöglichen es Ihnen, Elemente basierend auf ihrer Position im Dokumentbaum auszuwählen und zu gestalten.
- **`:root` Pseudoklasse**: Diese Pseudoklasse ist normalerweise das Wurzel-`html` Element. Sie hilft Ihnen, die höchste Ebene im Dokument anzusprechen, damit Sie einen gemeinsamen Stil auf das gesamte Dokument anwenden können.
- **`:empty` Pseudoklasse**: Leere Elemente, also Elemente ohne Kinder außer Leerraum, sind ebenfalls im Dokumentbaum enthalten. Deshalb gibt es eine `:empty` Pseudoklasse, um leere Elemente auszuwählen.
- **`:nth-child(n)` Pseudoklasse**: Diese Pseudoklasse ermöglicht es Ihnen, Elemente basierend auf ihrer Position innerhalb eines Elternteils auszuwählen.
- **`:nth-last-child(n)` Pseudoklasse**: Diese Pseudoklasse ermöglicht es Ihnen, Elemente auszuwählen, indem Sie vom Ende zählen.
- **`:first-child` Pseudoklasse**: Diese Pseudoklasse wählt das erste Element in einem Elternelement oder im Dokument aus.
- **`:last-child` Pseudoklasse**: Diese Pseudoklasse wählt das letzte Element in einem Elternelement oder im Dokument aus.
- **`:only-child` Pseudoklasse**: Diese Pseudoklasse wählt das einzige Element in einem Elternelement oder im Dokument aus.
- **`:first-of-type` Pseudoklasse**: Diese Pseudoklasse wählt das erste Vorkommen eines bestimmten Elementtyps innerhalb seines Elternelements aus.
- **`:last-of-type` Pseudoklasse**: Diese Pseudoklasse wählt das letzte Vorkommen eines bestimmten Elementtyps innerhalb seines Elternelements aus.
- **`:nth-of-type(n)` Pseudoklasse**: Diese Pseudoklasse ermöglicht es Ihnen, ein bestimmtes Element innerhalb seines Elternelements basierend auf seiner Position unter Geschwistern desselben Typs auszuwählen.
- **`:only-of-type` Pseudoklasse**: Diese Pseudoklasse wählt ein Element aus, wenn es das einzige seiner Art innerhalb seines Elternelements ist.

## Funktionale Pseudoklassen

- **Funktionale Pseudoklassen**: Funktionale Pseudoklassen ermöglichen es Ihnen, Elemente basierend auf komplexeren Bedingungen oder Beziehungen auszuwählen. Im Gegensatz zu regulären Pseudoklassen, die Elemente basierend auf einem Zustand auswählen (zum Beispiel `:hover`, `:focus`), akzeptieren funktionale Pseudoklassen Argumente.
- **`:is()` Pseudoklasse**: Diese Pseudoklasse nimmt eine Liste von Selektoren (z. B. `ol`, `ul`) und wählt ein Element aus, das einem der Selektoren in der Liste entspricht.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<p class="example">This text will change color.</p>
<p>This text will not change color.</p>
<p>This text will not change color.</p>
<p class="this-works-too">This text will change color.</p>
```

```css
p:is(.example, .this-works-too) {
    color: red;
}
```

:::

- **`:where()` Pseudoklasse**: Diese Pseudoklasse nimmt eine Liste von Selektoren (z. B. `ol`, `ul`) und wählt ein Element aus, das einem der Selektoren in der Liste entspricht. Der Unterschied zwischen `:is` und `:where` besteht darin, dass Letztere eine Spezifität von 0 hat.

```css
:where(h1, h2, h3) {
    margin: 0;
    padding: 0;
}
```

- **`:has()` Pseudoklasse**: Diese Pseudoklasse wird oft als `"parent"` Selektor bezeichnet, weil sie es Ihnen ermöglicht, Elemente zu gestalten, die Kind-Elemente enthalten, die in der Selektorliste angegeben sind.

```css
article:has(h2) {
    border: 2px solid hotpink;
}
```

- **`:not()` Pseudoklasse**: Diese Pseudoklasse wird verwendet, um Elemente auszuwählen, die nicht dem angegebenen Selektor entsprechen.

```css
p:not(.example) {
  color: blue;
}
```

## Pseudoelemente

- **`::before` Pseudoelement**: Dieses Pseudoelement verwendet die `content` Eigenschaft, um kosmetische Inhalte wie Symbole direkt vor dem Element einzufügen.
- **`::after` Pseudoelement**: Dieses Pseudoelement verwendet die `content` Eigenschaft, um kosmetische Inhalte wie Symbole direkt nach dem Element einzufügen.
- **`::first-letter` Pseudoelement**: Dieses Pseudoelement zielt auf den ersten Buchstaben des Inhalts eines Elements ab und ermöglicht es Ihnen, ihn zu gestalten.
- **`::marker` Pseudoelement**: Dieses Pseudoelement ermöglicht es Ihnen, den Marker (Aufzählungszeichen oder Nummerierung) von Listenelementen für die Gestaltung auszuwählen.
