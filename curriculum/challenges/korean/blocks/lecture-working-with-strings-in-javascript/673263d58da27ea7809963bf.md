# --interactive--

JavaScript에서 템플릿 리터럴은 문자열을 다루는 강력하고 유연한 방법입니다. 일반 문자열은 작은따옴표(`'`)나 큰따옴표(`"`)를 사용하지만, 템플릿 리터럴은 눈금(`` ` ``)으로 정의합니다.

템플릿 리터럴은 문자열 안에 변수를 직접 임베드하는 문자열 보간이라는 기능을 포함해 문자열 조작을 더 쉽게 할 수 있게 해줍니다.

템플릿 리터럴은 여러 줄에 걸친 문자열을 만들거나 문자열 안에 표현식(변수나 JavaScript 코드 등)을 직접 포함하기 쉽게 만듭니다.

다음은 템플릿 리터럴의 예입니다:

:::interactive_editor

```js
const name = "Alice";
const greeting = `Hello, ${name}!`;

console.log(greeting);
```

:::

작은따옴표나 큰따옴표 대신 눈금을 사용한 점에 주목하세요. `${name}` 구문은 문자열 보간의 예로, 변수 `name`의 값이 문자열 안에 직접 삽입됩니다.

문자열 보간은 문자열 안에 변수와 표현식을 임베드할 수 있게 해줍니다. 이는 예전 방식인 `+` 연산자를 사용해 문자열과 변수를 연결하는 방법보다 훨씬 개선된 점입니다.

다음은 문자열 연결과 더하기(`+`) 연산자를 사용한 예입니다:

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = "My name is " + name + " and I am " + age + " years old.";
console.log(message); 
```

:::

다음은 템플릿 리터럴과 문자열 보간을 사용한 예입니다:

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = `My name is ${name} and I am ${age} years old.`;
console.log(message); 
```

:::

보시다시피, 템플릿 리터럴을 이용한 문자열 보간은 특히 여러 변수를 다룰 때 훨씬 깔끔하고 읽기 쉽습니다.

템플릿 리터럴의 또 다른 훌륭한 기능은 여러 줄 문자열을 지원한다는 점입니다. 일반 문자열에서는 줄 바꿈을 만들기 위해 이스케이프 문자(`\n`)를 사용해야 하지만, 템플릿 리터럴에서는 문자열을 여러 줄에 걸쳐 작성해도 형식이 그대로 유지됩니다:

:::interactive_editor

```js
let poem = `Roses are red,
Violets are blue,
JavaScript is fun,
And so are you.`;

console.log(poem);
```

:::

템플릿 리터럴의 또 다른 기능은 문자열 안에 JavaScript 표현식을 직접 임베드할 수 있다는 점입니다. 다음 예를 보세요:

:::interactive_editor

```js
const song = "Bohemian Rhapsody";
const score = 9.5;
const highestScore = 10;
const output = `One of my favorite songs is "${song}". I rated it ${
  (score / highestScore) * 100
}%.`;
console.log(output); 
```

:::

템플릿 리터럴은 문자열에 변수나 표현식을 포함하거나 복잡한 문자열을 포맷하거나 여러 줄 텍스트를 다룰 때 특히 유용합니다. 전통적인 문자열 연결보다 더 간결하고 읽기 쉽습니다.
