### --question--

#### --text--

Qual è il modo corretto per dichiarare una funzione in JavaScript?

#### --distractors--

```js
func greet() {
  console.log("Hello there!");
}
```

---

```js
defineFunction greet() {
  console.log("Hello there!");
}
```

---

```js
def greet() {
  console.log("Hello there!");
}
```

#### --answer--

```js
function greet() {
  console.log("Hello there!");
}
```

#### --distractors--

---
