# --description--

前のチャレンジで作成されたオブジェクトインスタンス`constructor`と`duck`には、特別な`beagle`プロパティがあります。

```js
let duck = new Bird();
let beagle = new Dog();

console.log(duck.constructor === Bird); 
console.log(beagle.constructor === Dog);
```

これらの`console.log`呼び出しはどちらもコンソールに`true`を表示します。

`constructor`プロパティは、そのインスタンスを作成したコンストラクタ関数への参照であることに注意してください。`constructor`プロパティの利点は、このプロパティを調べることでオブジェクトの種類を判別できることです。以下はその使い方の例です。

```js
function joinBirdFraternity(candidate) {
  if (candidate.constructor === Bird) {
    return true;
  } else {
    return false;
  }
}
```

**注意：** `constructor`プロパティは上書き可能であるため（これは次の2つのチャレンジで扱います）、オブジェクトの型を調べるには一般的に`instanceof`メソッドを使うことをお勧めします。
