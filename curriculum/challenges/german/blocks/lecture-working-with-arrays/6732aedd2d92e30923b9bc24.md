# --interactive--

Array-Destrukturierung ist eine Funktion in JavaScript, mit der Sie Werte aus Arrays extrahieren und auf eine prägnantere und lesbarere Weise Variablen zuweisen können. Sie bietet eine praktische Syntax, um Array-Elemente in einzelne Variablen zu entpacken.

Diese Technik ist besonders nützlich, wenn Sie mit Arrays und Funktionen arbeiten, die mehrere Werte zurückgeben. Hier ist ein Beispiel für die Verwendung von Array-Destrukturierung:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];

let [first, second, third] = fruits;

console.log(first);  // "apple"
console.log(second); // "banana"
console.log(third);  // "orange"
```

:::

In diesem Beispiel haben wir ein Array namens `fruits` mit drei Elementen. Mithilfe der Array-Destrukturierung weisen wir das erste Element der Variablen `first`, das zweite Element `second` und das dritte Element `third` zu. So können wir einfach auf einzelne Elemente des Arrays zugreifen, ohne die Indexnotation zu verwenden.

So sähe es aus, wenn Sie jedes dieser Elemente über seinen Index anstatt mit Array-Destrukturierung aufrufen würden:

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange"];

const first = fruits[0];
const second = fruits[1];
const third = fruits[2];

console.log(first); // "apple"
console.log(second); // "banana"
console.log(third); // "orange"
```

:::

Mit Array-Destrukturierung können Sie auch Elemente überspringen, an denen Sie nicht interessiert sind, indem Sie Kommas verwenden. Zum Beispiel:

:::interactive_editor

```js
let colors = ["red", "green", "blue", "yellow"];
let [firstColor, , thirdColor] = colors;

console.log(firstColor); // "red"
console.log(thirdColor); // "blue"
```

:::

In diesem Beispiel überspringen wir das zweite Element des `colors`-Arrays durch ein zusätzliches Komma. Dadurch wird `red` `firstColor` zugewiesen und `blue` `thirdColor`, während `green` effektiv ignoriert wird.

Eine weitere leistungsstarke Funktion der Array-Destrukturierung ist die Möglichkeit, Standardwerte zu verwenden. Wenn das Array weniger Elemente hat als die Variablen, denen Sie Werte zuweisen möchten, können Sie Standardwerte angeben:

:::interactive_editor

```js
let numbers = [1, 2];
let [a, b, c = 3] = numbers;

console.log(a); // 1
console.log(b); // 2
console.log(c); // 3
```

:::

Hier weisen wir der Variablen `3` den Standardwert `c` zu, weil das `numbers`-Array kein drittes Element hat.

Kommen wir nun zur Rest-Syntax, dargestellt durch drei Punkte (`...`). Damit können Sie die verbleibenden Elemente eines Arrays, die nicht destrukturiert wurden, in einem neuen Array erfassen. So funktioniert das:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango", "kiwi"];
let [first, second, ...rest] = fruits;

console.log(first);  // "apple"
console.log(second); // "banana"
console.log(rest);   // ["orange", "mango", "kiwi"]
```

:::

In diesem Beispiel erfassen `first` und `second` die ersten beiden Elemente des `fruits`-Arrays, und `rest` erfasst alle übrigen Elemente als neues Array. Die Rest-Syntax muss das letzte Element im Destrukturierungsmuster sein.

Array-Destrukturierung ist eine mächtige Funktion, die Ihren Code prägnanter und leichter lesbar machen kann. Sie ist besonders nützlich, wenn Sie mit Arrays arbeiten und bestimmte Elemente aus einem Array extrahieren müssen.
