# --explanation--

Camila dice `¿Cuál es tu correo electrónico?` La estructura sigue este patrón:

`¿Cuál es tu` + noun?

Esta estructura se usa para pedir información personal específica. En este caso, `correo electrónico` se refiere a una dirección de correo usada para enviar y recibir mensajes en línea.
