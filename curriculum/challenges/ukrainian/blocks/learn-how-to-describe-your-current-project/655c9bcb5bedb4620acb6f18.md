# --description--

Коли ви хочете `make sure` щось зробити, ви намагаєтеся робити це з особливою увагою, щоб у кінці підтвердити результат. Наприклад:

- `We have to make sure the code is running.` — це означає, що ви будете особливо уважні з кодом, щоб підтвердити, що він добре запускається.

- `Please make sure that you order the correct color.` — тут вас просять підтвердити колір, який ви хочете, перед тим як щось замовляти.
