# --description--

Нехай $R(M, N)$ — це кількість вузлів ґратки ($x$, $y$), які задовольняють умови $M \lt x \le N$, $M \lt y \le N$ та $\left\lfloor\frac{y^2}{x^2}\right\rfloor$ є непарним.

Ми можемо перевірити, що $R(0, 100) = 3\\,019$ і $R(100, 10\\,000) = 29\\,750\\,422$.

Знайдіть $R(2 \times {10}^6, {10}^9)$.

**Примітка:** $\lfloor x\rfloor$ означає функцію підлоги.
