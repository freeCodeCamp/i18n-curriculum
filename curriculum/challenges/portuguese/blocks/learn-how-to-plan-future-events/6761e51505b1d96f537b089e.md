# --explanation--

`Going to` é usado para falar sobre ações planejadas ou intenções para o futuro.Por exemplo:

- `We are going to upgrade the system next month.` - Isso significa que a atualização está planejada e acontecerá em breve.
  
- `She is going to learn SQL next year.` - Isso significa que ela pretende aprender SQL no futuro.

Nesta frase:

`We're` é a contração de `we are`.

`Going to` indica que a ação está planejada para o futuro próximo.

`Need` é o verbo base que indica algo necessário ou exigido.
