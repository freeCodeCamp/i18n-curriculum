# --interactive--

CSS 動畫允許你在網頁上創建動態且視覺吸引人的效果，而無需使用 JavaScript 或複雜的程式設計。它們提供了一種方法，可以在指定的持續時間內平滑地將元素從一種樣式過渡到另一種樣式。

在其核心，CSS 動畫由兩個主要組件組成：`@keyframes` 規則和 animation 屬性。

`@keyframes` 規則定義動畫的階段和樣式。它指定元素在動畫過程中各個時間點應該具有的樣式。 

這裡有一個範例：

```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(0);
  }
}
```

這個名為 `slide-in` 的 `@keyframes` 規則定義了一個將元素從左移動到右的動畫。百分比表述動畫的進度，`0%` 為開始，`100%` 為結束。

你在 `@keyframes` 動畫中的 `translateX` 函式（程式）正在控制元素進入視圖時的水平位置。

要將此動畫套用到元素，請使用 `animation` 屬性。此範例也會無限重複動畫，讓你可以看到其作用中（你需要啟用互動式編輯器才能看到預覽）：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="sliding-element">Hello, I slide in!</div>
```

```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(0);
  }
}

.sliding-element {
  animation: slide-in 2s ease-out infinite;
}
```

:::

這會將 `slide-in` 動畫套用到元素上，持續時間為 2 秒，並使用 `ease-out` 時間函式。

`animation` 屬性實際上是多個單獨屬性的縮寫：

`animation-name`，用來指定要使用的 `@keyframes` 規則。

`animation-duration` 用來設定動畫應該花費多長時間完成。

`animation-timing-function` 定義動畫隨時間的進展方式－例如 `ease`、`linear`、`ease-in-out`。

`animation-delay` 指定動畫開始前的延遲時間。

`animation-iteration-count` 用來設定動畫應該重複的次數。

`animation-direction` 用來決定動畫應該向前播放、向後播放，或交替播放。

`animation-fill-mode` 指定元素在動畫開始前和結束後應該如何被樣式化。

`animation-play-state` 允許你暫停和繼續動畫。

你可以單獨使用這些屬性以獲得更精確的控制：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="complex-animation">Watch my colors change!</div>
```

```css
.complex-animation {
  animation-name: color-change;
  animation-duration: 3s;
  animation-timing-function: linear;
  animation-delay: 1s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}

@keyframes color-change {
  0% {
    background-color: red;
  }
  50% {
    background-color: blue;
  }
  100% {
    background-color: green;
  }
}
```

:::

這會創建一個動畫，使元素的背景色在紅色、藍色和綠色之間持續過渡。

CSS 動畫可以由各種事件觸發，例如暫留在元素上：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Hover over me!</button>
```

```css
.button {
  background-color: blue;
  transition: background-color 0.3s;
}

.button:hover {
  background-color: red;
}
```

:::

雖然此範例使用較簡單的 `transition` 屬性來實現基本效果，但它展示了 CSS 如何創建互動式的動畫元素。

重要的是要注意，雖然 CSS 動畫功能強大，但應該適度使用。過度使用動畫可能導致效率不佳，並且可能對具有特定存取需求的使用者造成干擾或問題。始終考慮為偏好較少動態效果的使用者提供減少或停用動畫的選項。

CSS 動畫提供了一種方法，可以在不依賴 JavaScript 的情況下創建引人入勝且互動的網頁體驗。透過理解 `@keyframes` 和 `animation` 屬性的原理，你可以以高效且易於存取的方式讓你的網頁設計栩栩如生。
