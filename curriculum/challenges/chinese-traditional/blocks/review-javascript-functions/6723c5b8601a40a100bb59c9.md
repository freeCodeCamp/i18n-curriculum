## JavaScript 函式（程式）

- 函式（程式）是可重複使用的程式碼區塊，用來執行特定任務。
- 函式（程式）可以使用 `function` 關鍵字定義，後接名稱、一個參數列表，以及執行任務的程式碼區塊。

:::interactive_editor

```js
function addNumbers(x, y, z) {
  return x + y + z;
}

console.log(addNumbers(5, 3, 8)); // Output: 16
```

:::

- 引數是在呼叫函式時傳遞給該函式的值。
- 函式（程式）呼叫是在程式中透過指定函式名稱，後接小括弧，並可選擇性地在小括弧內包含引數，來執行函式（程式）的處理程序。
- 當一個函式（程式）完成執行時，它將始終傳回一個值。
- 預設情況下，函式（程式）的返回值是 `undefined`。
- `return` 關鍵字用於指定從函式中傳回的值，並結束函式執行。
- 預設值參數允許函式（程式）擁有預先定義的值，當呼叫函式時若未提供引數，將使用這些值。這使函式（程式）更具彈性，並防止在某些引數可能被省略的情況下發生錯誤。

:::interactive_editor

```js
const calculateTotal = (amount, taxRate = 0.05) => {
  return amount + (amount * taxRate);
};

console.log(calculateTotal(100)); // Output: 105
```

:::

- 函式表達式是你指定給變數的函式。透過這樣做，你可以在程式碼中任何變數可存取的部分使用該函式。

:::interactive_editor

```js
const multiplyNumbers = function(firstNumber, secondNumber) {
  return firstNumber * secondNumber;
};

console.log(multiplyNumbers(4, 5)); // Output: 20
```

:::

## 箭頭函式

- 箭頭函式是在 JavaScript 中撰寫函式（程式）的更簡潔方式。

:::interactive_editor

```js
const calculateArea = (length, width) => {
  const area = length * width;
  return `The area of the rectangle is ${area} square units.`;
};

console.log(calculateArea(5, 10)); // Output: "The area of the rectangle is 50 square units."
```

:::

- 在定義箭頭函式時，你不需要 `function` 關鍵字。
- 如果你使用單一參數，可以省略參數列周圍的小括弧。

:::interactive_editor

```js
const cube = x => {
  return x * x * x;
};

console.log(cube(3)); // Output: 27
```

:::

- 如果函式（程式）主體只包含單一個表達式、運算式，你可以省略大括號和 `return` 關鍵字。

:::interactive_editor

```js
const square = number => number * number;

console.log(square(5)); // Output: 25
```

:::

## 程式設計中的範圍

- **全域的範圍**：這是 JavaScript 中最外層的範圍。在全域的範圍中宣告的變數可以在程式碼的任何地方存取，並稱為全域變數。
- **區域的作用域**：這是指在函式內宣告的變數。這些變數只能在宣告它們的函式內存取，稱為區域變數。
- **區塊範圍**：區塊是一組用大括號 `{}` 包圍的述語，例如在 `if` 述語或迴圈中。
- 使用 `let` 和 `const` 的區塊作用域提供更細緻的變數存取控制，有助於防止錯誤並使你的程式碼更可預測。
