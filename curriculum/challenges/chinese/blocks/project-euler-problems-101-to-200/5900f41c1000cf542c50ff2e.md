# --description--

定义 $f(0) = 1$，且 $f(n)$ 为将 $n$ 写成不超过两次出现的 2 的幂之和的方式数。

例如，$f(10) = 5$，因为有五种不同的方式表示 10：

$$10 = 8 + 2 = 8 + 1 + 1 = 4 + 4 + 2 = 4 + 2 + 2 + 1 + 1 = 4 + 4 + 1 + 1$$

可以证明，对于每个分数 $\frac{p}{q}\\; (p>0, q>0)$，至少存在一个整数 $n$，使得 $\frac{f(n)}{f(n - 1)} = \frac{p}{q}$。

例如，满足 $\frac{f(n)}{f(n - 1)} = \frac{13}{17}$ 的最小 $n$ 是 241。241 的二进制展开是 11110001。

从最高有效 `bit` 到最低有效 `bit` 读取这个二进制数，有 4 个 1，3 个 0 和 1 个 1。我们将字串 4,3,1 称为 241 的简化二进制展开。

查找最小的 $n$ 的简化二进制展开，其满足

$$\frac{f(n)}{f(n - 1)} = \frac{123456789}{987654321}$$

以字串形式给出答案，整数之间用逗号分隔，且不含任何空白字符。
