# --description--

بما أن لديك فهمًا أساسيًا عن المصفوفات، فلنتحدث عن الحلقات. تُستخدم الحلقات لتنفيذ كتلة من الشفرة عدة مرات. إحدى هذه الحلقات هي حلقة `while`. تنفذ حلقة `while` كتلة الشفرة طالما أن الشرط صحيح. صيغة حلقة `while` هي كما يلي:

```javascript
while (condition) {
  // code block to be executed
}
```

على سبيل المثال، يطبع مقتطف الشفرة التالي الأرقام من 1 إلى 5:

```javascript
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
```

يُهيئ مقتطف الشفرة أعلاه متغيرًا `i` بالقيمة `1`. تنفذ حلقة `while` كتلة الشفرة طالما أن قيمة `i` أقل من أو تساوي `5`. تُزاد قيمة `i` بمقدار `1` في كل تكرار.
