# --description--

关系代词如 `who`、`whom`、`which` 和 `that` 用于将关系从句连接到名词或代词。

`Who` 和 `whom` 参考人。区别在于 `who` 是动词的主语，而 `whom` 用作动词或介词的宾语。

`Which` 用于特定的事物或动物，而 `that` 可用于定义性从句中的人、事物或动物。

示例：

`The programmer who wrote this code is brilliant.` （`who` 指代程序员）

`The artist whom we met at the gallery is internationally recognized.` （`whom` 指代艺术家）

`The tool that you lent me was useful.` （`that` 指代那个工具）

`The book, which I bought yesterday, is already a bestseller.` （`which` 指代那本书）

# --questions--

## --text--

哪个句子正确使用了关系代词？

## --answers--

`The teacher which teaches us math is very kind.`

### --feedback--

`which` 通常用于事物，而非人。这里正确的代词应该是 `who`。

---

`The car, that won the race, is very fast.`

### --feedback--

在这个句子中，`that` 的用法在语法上是正确的，但逗号是不必要的。它应该是：`The car that won the race is very fast.`

---

`She is the one who made the cake.`

---

`The flowers whom are on the table smell lovely.`

### --feedback--

`Whom` 用于动词或介词的对象，指代人，而不是像花这样的事物。这里正确的代词应该是 `which` 或 `that`。
