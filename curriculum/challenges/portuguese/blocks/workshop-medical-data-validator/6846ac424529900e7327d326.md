# --description--

Como você aprendeu em uma lição anterior, um conjunto é uma coleção não ordenada de elementos únicos:

```py
integers = set([3, 5, 1, 2, 1, 3, 4])
print(integers) # {1, 2, 3, 4, 5}
```

Você vai usar um conjunto para garantir que cada dicionário não contenha chaves extras ou com erros de digitação.

Use o construtor `set()` para criar um conjunto a partir da seguinte lista de chaves que cada dicionário deve ter: `['patient_id', 'age', 'gender', 'diagnosis', 'medications', 'last_visit_id']`. Atribua o conjunto a uma variável chamada `key_set`.
