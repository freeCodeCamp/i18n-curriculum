# --interactive--

Існує кілька способів видалити властивості з об’єкта, при цьому оператор `delete` є найпростіший і найпоширеніший метод.

Коли ви використовуєте `delete`, він видаляє вибрану властивість з об’єкта. Ось приклад, як використовувати оператор `delete`:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  job: "Engineer"
};

delete person.job;

console.log(person.job); // undefined
```

:::

У цьому прикладі ми починаємо з об’єкта `person`, який має три властивості: `name`, `age` і `job`. Потім ми використовуємо оператор `delete`, щоб видалити властивість `job`. Після видалення об’єкт `person` більше не має властивості `job`.

Інший спосіб видалити властивості — це використання деструктурованого присвоєння з параметрами залишку. Цей підхід фактично не видаляє властивість, але створює новий об’єкт без зазначених властивостей:

:::interactive_editor

```js
const person = {
  name: "Bob",
  age: 25,
  job: "Designer",
  city: "New York"
};

const { job, city, ...remainingProperties } = person;

// { name: "Bob", age: 25 }
console.log(remainingProperties);
```

:::

У цьому прикладі ми використовуємо деструктуризацію, щоб витягти `job` і `city` з об’єкта `person`, а решту властивостей збираємо в новий об’єкт під назвою `remainingProperties`. Це створює новий об’єкт без властивостей `job` і `city`.

Розуміння того, як видаляти властивості з об’єктів, є важливою навичкою в програмуванні на JavaScript. Це дозволяє динамічно маніпулювати об’єктами та очищати непотрібні дані.
