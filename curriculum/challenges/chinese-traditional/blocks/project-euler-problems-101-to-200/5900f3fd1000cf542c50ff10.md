# --description--

某些正整數 $n$ 具有這樣的特性：和 [ $n + reverse(n)$ ] 的和完全由奇數（十進位）數字組成。例如，$36 + 63 = 99$ 和 $409 + 904 = 1313$。我們將這樣的數字稱為可逆數；因此 36、63、409 和 904 都是可逆數。$n$ 或 $reverse(n)$ 中皆不允許有前導零。

在 1000 以下有 120 個可逆數字。

在 10^9 以下有多少個可逆數？
