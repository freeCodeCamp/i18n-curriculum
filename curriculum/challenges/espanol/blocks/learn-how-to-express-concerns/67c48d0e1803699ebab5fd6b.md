# --explanation--

Cuando uses un adjetivo seguido de un infinitivo, la forma negativa se hace añadiendo `not to` antes del verbo. Por ejemplo:

- Afirmativo: `She was happy to help with the project.` - Esto significa que ayudar con el proyecto la hizo feliz.

- Negativo: `He was careful not to forget the deadline.` - Esto significa que se aseguró de recordarlo.

`To burn out` significa volverse extremadamente cansado o agotado por demasiado trabajo. Por ejemplo:

`If we don't take breaks, we might burn out before the project is finished.` - Esto significa que trabajar sin descansar podría causar agotamiento.
