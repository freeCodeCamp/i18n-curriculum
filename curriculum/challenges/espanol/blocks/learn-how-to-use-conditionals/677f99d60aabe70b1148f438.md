# --explanation--

El tiempo `Present Simple` se usa a menudo para describir estados, hechos o condiciones que generalmente son verdaderos. Este tiempo se utiliza porque la corrupción del BIOS se considera una condición estable o inmutable cuando la computadora intenta arrancar.

`Is corrupted` significa dañado o modificado de una manera que hace que deje de funcionar correctamente. Por ejemplo:

`The file is corrupted and won't open.` - El archivo está dañado y no se puede abrir.
