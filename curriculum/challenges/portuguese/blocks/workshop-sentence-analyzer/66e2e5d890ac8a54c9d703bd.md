# --description--

É hora de contar as consoantes.Crie uma função `getConsonantCount` com um parâmetro `sentence`.

Dentro da função, use um loop para contar o número de consoantes na `sentence` que será passada para a função quando ela for chamada.Uma consoante é qualquer letra que não seja um dos seguintes caracteres: `"aeiou"`. 

Sua função `getConsonantCount` deve retornar um número.

function getVowelCount(sentence) {
  const vowels = "aeiou";
  let count = 0;

  for (const char of sentence.toLowerCase()) {
    if (vowels.includes(char)) {
      count++;
    }
  }
  return count;
}

const vowelCount = getVowelCount("Apples are tasty fruits");
