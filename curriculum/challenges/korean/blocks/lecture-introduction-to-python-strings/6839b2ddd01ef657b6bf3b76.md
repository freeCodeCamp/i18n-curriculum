# --description--

Python은 문자열 작업을 쉽게 해주는 여러 내장 메서드를 제공합니다. 여기에는 다음과 같은 것들이 포함되지만 이에 국한되지는 않습니다:

- `upper()`: 모든 문자를 대문자로 변환한 새 문자열을 반환합니다.
    

```python
my_str = 'hello world'

uppercase_my_str = my_str.upper()
print(uppercase_my_str)  # HELLO WORLD
```

- `lower()`: 모든 문자를 소문자로 변환한 새 문자열을 반환합니다.
    

```python
my_str = 'Hello World'

lowercase_my_str = my_str.lower()
print(lowercase_my_str)  # hello world
```

- `strip()`: 지정한 앞뒤 문자를 제거한 새 문자열을 반환합니다. 인수가 전달되지 않으면 앞뒤 공백을 제거합니다.
    

```python
my_str = '  hello world  '

trimmed_my_str = my_str.strip()
print(trimmed_my_str)  # "hello world"
```

- `replace(old, new)`: `old`가 `new`로 교체된 모든 경우를 포함하는 새 문자열을 반환합니다.
    

```python
my_str = 'hello world'

replaced_my_str = my_str.replace('hello', 'hi')
print(replaced_my_str)  # hi world
```

- `split(separator)`: 지정한 구분자를 기준으로 문자열을 나누어 문자열 리스트를 반환합니다. 구분자가 지정되지 않으면 공백을 기준으로 나눕니다.
    

```python
my_str = 'hello world'

split_words = my_str.split()
print(split_words)  # ['hello', 'world']
```

- `join(iterable)`: 반복 가능한 요소들을 구분자로 연결하여 문자열로 만듭니다.
    

```python
my_list = ['hello', 'world']

joined_my_str = ' '.join(my_list)
print(joined_my_str)  # hello world
```

- `startswith(prefix)`: 문자열이 지정한 접두사로 시작하는지 여부를 불리언으로 반환합니다.
    

```python
my_str = 'hello world'

starts_with_hello = my_str.startswith('hello')
print(starts_with_hello)  # True
```

- `endswith(suffix)`: 문자열이 지정한 접미사로 끝나는지 여부를 불리언으로 반환합니다.
    

```python
my_str = 'hello world'

ends_with_world = my_str.endswith('world')
print(ends_with_world)  # True
```

- `find(substring)`: `substring`가 처음 나타나는 인덱스를 반환하며, 찾지 못하면 `-1`를 반환합니다.
    

```python
my_str = 'hello world'

world_index = my_str.find('world')
print(world_index)  # 6
```

- `count(substring)`: 문자열 내에 부분 문자열이 나타나는 횟수를 반환합니다.
    

```python
my_str = 'hello world'

o_count = my_str.count('o')
print(o_count)  # 2
```

- `capitalize()`: 첫 문자는 대문자로, 나머지 문자는 소문자로 변환한 새 문자열을 반환합니다.
    

```python
my_str = 'hello world'

capitalized_my_str = my_str.capitalize()
print(capitalized_my_str)  # Hello world
```

- `isupper()`: 문자열의 모든 문자가 대문자이면 `True`를, 그렇지 않으면 `False`를 반환합니다.
    

```python
my_str = 'hello world'

is_all_upper = my_str.isupper()
print(is_all_upper)  # False
```

- `islower()`: 문자열의 모든 문자가 소문자이면 `True`를, 그렇지 않으면 `False`를 반환합니다.
    

```python
my_str = 'hello world'

is_all_lower = my_str.islower()
print(is_all_lower)  # True
```

- `title()`: 각 단어의 첫 글자를 대문자로 변환한 새 문자열을 반환합니다.
    

```python
my_str = 'hello world'

title_case_my_str = my_str.title()
print(title_case_my_str)  # Hello World
```
