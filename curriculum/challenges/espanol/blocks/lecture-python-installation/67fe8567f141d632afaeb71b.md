# --description--

Para todos los talleres y laboratorios, usarás el editor de Python de freeCodeCamp. Pero es importante aprender a configurar Python en tu máquina local.

La forma más fácil de instalar Python en Windows y Mac es descargar el instalador desde el sitio oficial de Python. También veremos cómo ejecutar Python en Linux más adelante en esta lección.

Ve a `https://www.python.org/` y pasa el cursor sobre "Downloads". Aparecerá un modal que muestra la versión actual de Python para tu sistema operativo (SO).

Primero veremos cómo instalar Python en una computadora con macOS:

- Haz clic en el botón que muestra la versión actual de Python (del modal anterior) y comenzarás a descargar automáticamente un archivo de instalación `.pkg`.
- Una vez que el instalador `.pkg` termine de descargarse, ábrelo y haz clic en "Continue" en la ventana que se abre.
- Sigue haciendo clic en el botón "Continue" hasta llegar a la sección "Installation Type". Allí, haz clic en el botón "Install".
- Ingresa tu contraseña si es necesario y comienza la instalación.
- Después, deberías recibir un mensaje de felicitación que dice que Python se instaló correctamente.
- Haz clic en el botón "Close" y ¡listo!

Puedes verificar la instalación abriendo tu terminal y ejecutando `python --version` o `python3 --version`.

También puedes abrir el intérprete de Python ejecutando `python` o `python3` en la terminal. Un intérprete de Python es el programa que lee el código Python, lo traduce en instrucciones que la computadora entiende y ejecuta esas instrucciones.

Un <dfn>terminal</dfn> es una interfaz basada en texto que te permite interactuar con tu computadora escribiendo comandos. Cada sistema operativo viene con una app de terminal predeterminada. En macOS, puedes usar la app Terminal. En Windows, puedes usar Command Prompt o PowerShell. En Linux, cada entorno de escritorio tiene su propia app de terminal predeterminada, como GNOME Terminal o Konsole.

Ten en cuenta que, en algunos sistemas macOS y Linux antiguos, `python` puede estar reservado para Python 2, mientras que `python3` es para Python 3 específicamente. Si ejecutas `python --version` y ves una versión de Python 2 como `Python 2.7.18`, entonces es posible que tu SO dependa de algún software escrito en la versión antigua de Python. En ese caso, deberías usar `python3` para ejecutar tu código Python de ahora en adelante. Python 2 está en fin de vida y no debe usarse para ningún desarrollo nuevo.

Para instalar Python en Windows, sigue estos pasos:

- Ve a `https://www.python.org/` y pasa el cursor sobre “Downloads“. Deberías ver un modal que dice "Download for Windows" y un botón para descargar la versión actual de Python.
- Haz clic en el número de versión y comenzarás a descargar automáticamente un archivo ejecutable para Windows (`.exe`).
- Una vez que termines de descargar el instalador de Python para Windows, haz doble clic en él y sigue las instrucciones.
- Cuando veas la opción `Add python.exe to Path`, marca esa opción y luego haz clic en `Install Now`. Hacer eso facilitará las cosas más adelante.

Puedes verificar la instalación abriendo una consola de comandos como PowerShell y ejecutando `python --version`. También puedes abrir el intérprete de Python ejecutando `python`.

Para Python en Linux, la mayoría de las distribuciones principales como Ubuntu, Debian y Fedora ya incluyen Python.

Solo abre una terminal y ejecuta `python --version` o `python3 --version`:

Si alguno de estos comandos no muestra una versión de Python, puedes buscar un paquete de instalación para tu distribución de Linux en `https://www.python.org`, o buscar en línea la forma recomendada de instalar Python para tu distro.
