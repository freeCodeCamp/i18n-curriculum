# --description--

音楽から医療まで、あらゆる業界で大規模かつ複雑なデータセットが扱われています。これには患者の医療記録、時間経過によるプレイリストの傾向、ソーシャルメディアのエンゲージメント指標などが含まれます。

この情報を理解するために、企業はさまざまなツールを使ってチームがデータを分析し、製品やサービスに関する情報に基づいた意思決定を行えるようにしています。

その中でも特に強力なツールの一つが**データの可視化**です。

データの可視化は複雑なデータからわかりやすい視覚的な形式を作り出します。こうした視覚表現の例には円グラフ、散布図、棒グラフなどがあります。優れた可視化は、企業がデータの重要な情報、傾向、関係性を素早く見つけるのに役立ちます。

ビジネスでよく使われるデータの可視化ツールには、Microsoft Excel、Power BI、Google Charts、Tableauなどがあります。

これからのレッスンやワークショップでは、JavaScriptのD3.jsライブラリを使ってインタラクティブな可視化を作成する方法を学びます。

このモジュールの目標は、データの可視化の基本を紹介し、JavaScriptで自分の可視化を作る手順を案内することです。
