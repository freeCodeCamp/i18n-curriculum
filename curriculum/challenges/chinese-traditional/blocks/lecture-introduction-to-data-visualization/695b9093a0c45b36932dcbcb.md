# --description--

在之前的課程中，你已經被介紹到資料視覺化，這是建立清晰、視覺化格式以表述複雜資料的處理程序。在本課程中，你將學習 `D3` 函式庫。

D3 函式庫是一個免費且開放原始碼的函式庫，你可以用來在 JavaScript 中創建引人注目且有效的資料視覺化。

Node B connects to A and D. Node C connects to A. Node D connects to A and B. It's the same image as before.">


名稱 D3 是 data-driven documents 的縮寫，指的是它如何將資料動態綁定到文件物件模型（DOM）中的元素。這個函式庫非常靈活，且能自然地與網頁協同運作。你可以使用 `svg` 和 `canvas` 元素來建置資料視覺化，而不需要學習全新的圖形表現方式。

例如，如果你想使用 D3 創建一個長條圖，你可以在一個 JS 檔案中完成以下所有操作：

* 宣告表述圖表尺寸和邊距的變數
    
* 宣告水平位置刻度
    
* 宣告垂直位置刻度
    
* 創建包含長條和標籤的 SVG 容器
    
<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-2.png" alt="A graph with four nodes labeled A, B, C, and D. Node A connects to B, C, and D. Node B connects to A and D. Node C connects to A. Node D connects to A and B. It's the same image as before.">

那麼，還有什麼其他理由讓你學習 D3，而不是其他像是 Chart.js、Plotly 或 Tableau 的視覺化工具呢？

與其他工具不同，D3 為你提供大量低層次的控制元件，你可以僅使用原生的 HTML、CSS 和 JavaScript 建造訂製且互動的視覺化效果。此外，該函式庫是用 JavaScript 編寫的，所以你不需要擔心學習全新的語言。

在即將舉辦的工作坊和實驗室中，你將學習 D3 函式庫的基本概念，例如選取、形狀和比例尺。你將透過建造散佈圖、長條圖、儀表板應用程式等來學習這些關鍵概念。
