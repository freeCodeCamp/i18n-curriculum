# --explanation--

`Absolutely` показує повну згоду або сильне підтвердження. Наприклад:

`Absolutely, I'll review the code today.` – Це означає, що ви повністю погоджуєтеся це зробити.

`Prioritize` означає організувати завдання так, щоб найважливіші виконувалися першими. Наприклад:

`The IT team will prioritize server repairs before installing updates.` – Це означає, що ремонт серверів буде виконано першим, бо це більш терміново.
