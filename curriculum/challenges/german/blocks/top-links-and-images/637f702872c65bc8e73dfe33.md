# --description--


Im Allgemeinen gibt es zwei Arten von Links, die Sie erstellen werden:

- Links zu Seiten auf anderen Websites im Internet

- Links zu Seiten auf Ihren eigenen Websites


## Absolute Links
Links zu Seiten auf anderen Websites im Internet werden absolute Links genannt. Ein typischer absoluter Link besteht aus den folgenden Teilen: `protocol://domain/path`. Ein absoluter Link enthält immer das Protokoll und die Domain des Ziels.

Sie haben bereits einen absoluten Link in Aktion gesehen. Der Link, den Sie zuvor zur About-Seite von The Odin Project erstellt haben, war ein absoluter Link, da er das Protokoll und die Domain enthält.

`https://www.theodinproject.com/about`

## Relative Links
Links zu anderen Seiten innerhalb Ihrer eigenen Website werden relative Links genannt. Relative Links enthalten nicht den Domainnamen, da es sich um eine andere Seite auf derselben Website handelt, wird angenommen, dass der Domainname derselbe ist wie auf der Seite, auf der Sie den Link erstellt haben.

Relative Links enthalten nur den Dateipfad zur anderen Seite, relativ zu der Seite, auf der Sie den Link erstellen. Das ist ziemlich abstrakt, sehen wir uns das anhand eines Beispiels an.

Erstellen Sie im Verzeichnis `odin-links-and-images` eine weitere HTML-Datei namens `about.html` und fügen Sie den folgenden Code darin ein:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Odin Links and Images</title>
  </head>

  <body>
    <h1>About Page</h1>
  </body>
</html>
```

Gehen Sie zurück zur `index`-Seite und fügen Sie das folgende Ankerelement hinzu, um einen Link zur `about`-Seite zu erstellen:

```html
<body>
  <h1>Homepage</h1>
	<a href="https://www.theodinproject.com/about">click me</a>

	<a href="about.html">About</a>
</body>
```

Öffnen Sie die `index.html`-Datei in einem Browser und klicken Sie auf den about-Link, um sicherzustellen, dass alles richtig verknüpft ist. Ein Klick auf den Link sollte zur `about`-Seite führen, die Sie gerade erstellt haben.

Das funktioniert, weil sich die `index`- und die `about`-Seite im selben Verzeichnis befinden. Das bedeutet, Sie können einfach den Namen (`about.html`) als `href`-Wert des Links verwenden.

Normalerweise möchten Sie Ihre Website-Verzeichnisse jedoch etwas besser organisieren. Normalerweise hätten Sie nur die `index.html` im Stammverzeichnis und alle anderen HTML-Dateien in ihren eigenen Verzeichnissen.

Erstellen Sie ein Verzeichnis namens `pages` im `odin-links-and-images`-Verzeichnis und verschieben Sie die `about.html`-Datei in dieses neue Verzeichnis.

Aktualisieren Sie die `index`-Seite im Browser und klicken Sie dann auf den `about`-Link. Dieser ist jetzt defekt. Das liegt daran, dass sich der Speicherort der `about`-Datei geändert hat.

Um das zu beheben, müssen Sie nur den `about`-Link `href`-Wert aktualisieren, um das `pages/`-Verzeichnis einzuschließen, da dies der neue Speicherort der `about.html`-Datei relativ zur `index.html`-Datei ist.

```html
<body>
  <h1>Homepage</h1>
  <a href="pages/about.html">About</a>
</body>
```

Aktualisieren Sie die `index`-Seite im Browser und versuchen Sie erneut, auf den `about`-Link zu klicken. Er sollte jetzt wieder funktionieren.

In vielen Fällen funktioniert das so einwandfrei; dennoch können bei diesem Ansatz unerwartete Probleme auftreten. Das Voranstellen von `./` vor den Link verhindert in den meisten Fällen solche Probleme. Indem Sie `./` hinzufügen, geben Sie Ihrem Code an, dass er mit der Suche nach der Datei/dem Verzeichnis relativ zum **aktuellen** Verzeichnis beginnen soll.

```html
<body>
  <h1>Homepage</h1>
  <a href="./pages/about.html">About</a>
</body>
```

## --text--

Was ist der Unterschied zwischen einem absoluten und einem relativen Link?

## --answers--

Ein absoluter Link ist ein Link zu einer anderen Seite auf der aktuellen Website. Ein relativer Link ist ein Link zu einer anderen Website.

---

Ein absoluter Link ist ein Link zu einer anderen Website. Ein relativer Link ist ein Link zu einer anderen Seite auf der aktuellen Website.

---

Es gibt keinen Unterschied zwischen absoluten und relativen Links.
