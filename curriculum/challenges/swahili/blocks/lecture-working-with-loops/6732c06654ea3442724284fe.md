# --interactive--

Katika mafundisho yaliyopita, ulijifunza jinsi ya kufanya kazi na mizunguko ya `for`, mizunguko ya `for...in` na mizunguko ya `for...of`. Katika funzo hili, utajifunza kuhusu mzunguko wa `while` na mzunguko wa `do...while`.

Mzunguko wa `while` utaendesha kipande cha msimbo mradi tu hali ni kweli. Hapa kuna sintaksia ya msingi ya mzunguko wa `while`:

```js
while (condition) {
  // code block to be executed
}
```

Hali inakaguliwa kabla ya kipande cha msimbo kutekelezwa. Ikiwa hali ni potofu, kipande cha msimbo hakitatekelezwa.

Mizunguko ya `while` ni muhimu wakati hujui ni mara ngapi unahitaji kuendesha kipande cha msimbo. Hapa kuna mfano wa kutumia mzunguko wa `while`:

:::interactive_editor

```js
let counter = 0;
while(counter < 5) {
  console.log(counter);
  counter++;
}
```

:::

Katika mfano huu, tuna kigezo kinachoitwa `counter` ambacho kimeanzishwa kwa `0`. Mzunguko wa `while` utaendelea kuendesha mradi tu thamani ya `counter` ni ndogo kuliko `5`. Ndani ya mzunguko, tunarekodi thamani ya `counter` kwenye konsoli kisha tunaongeza `counter` kwa `1`.

Mzunguko mwingine unaofanana na mzunguko wa `while` ni mzunguko wa `do...while`. Hapa kuna sintaksia ya msingi:

```js
do {
  // code block to be executed
} while (condition);
```

Tofauti kuu kati ya mzunguko wa `do...while` na mzunguko wa `while` ni kwamba mzunguko wa `do...while` utaendesha kipande cha msimbo angalau mara moja kabla ya kukagua hali.

Ikiwa hali ni kweli, kipande cha msimbo kitaendelea kutekelezwa. Ikiwa hali ni potofu, kipande cha msimbo kitasitishwa.

Hapa kuna mfano wa kutumia mzunguko wa `do...while`:

:::interactive_editor

```js
let counter = 0;
do {
  console.log(counter);
  counter++;
} while (counter < 5);  
```

:::

Katika mfano huu, tuna kigezo kinachoitwa `counter` ambacho kimeanzishwa kwa `0`. Mzunguko wa `do...while` utaandika thamani ya `counter` kwenye konsoli kisha kuongeza `counter` kwa `1`. Baada ya kutekeleza kipande cha msimbo, unakagua kama thamani ya `counter` ni ndogo kuliko `5`. Ikiwa ni kweli, mzunguko utaendelea kuendesha. Ikiwa si hivyo, mzunguko utasimama.

Katika hali nyingi, huenda ukatumia mzunguko wa `while` mara nyingi zaidi kuliko mzunguko wa `do...while`. Hata hivyo, ni vizuri kujua aina zote mbili za mizunguko na wakati wa kuzitumia.
