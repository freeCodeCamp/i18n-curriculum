# --description--

このビデオでは、try/except ブロックを使って例外を処理し、プログラムをより堅牢にする方法を学びます。

# --questions--

## --text--

ユーザーが文字列 `"random"` を入力した場合、次のコードの結果はどうなりますか？

```python
try:
    number = int(input("Enter a number: "))
    print(number)
except:
    print("Invalid Input")
```

## --answers--

コンソールには何も出力されません。

---

コンソールに文字列 `"random"` が出力されます。

---

コンソールに文字列 `"Invalid Input"` が出力されます。

---

プログラムがクラッシュします。
