# --description--

Como um elemento `div` está sendo usado para a nota, leitores de tela não entenderão seu propósito.Você pode corrigir isso adicionando um atributo `aria-label`.Isso ajudará na acessibilidade e fornecerá significado para este elemento `div`. 

No seu elemento `div` de abertura, adicione um atributo `aria-label` e defina seu valor como `"Note editor"`.

# --hints--

Sua tag de abertura `div` deve ter o atributo `aria-label`.

```js
assert.isTrue(document.getElementById("note").hasAttribute("aria-label"));
```

Sua tag de abertura `div` deve ter o atributo `aria-label` definido como `"Note editor"`.

```js
assert.equal(document.getElementById("note").getAttribute("aria-label"), "Note editor");
```
