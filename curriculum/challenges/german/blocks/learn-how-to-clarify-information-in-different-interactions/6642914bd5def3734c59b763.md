# --description--

Ein `tray` ist ein flacher, flacher Container, der zum Tragen oder Halten von Dingen verwendet wird. Zum Beispiel benutzen Sie einen `tray`, um Essen oder Getränke von der Küche zum Tisch zu bringen.

`Enough` bedeutet, so viel zu haben, wie Sie brauchen. Zum Beispiel, wenn Sie `enough water` haben, bedeutet das, dass Sie das ganze Wasser haben, das Sie zum Trinken brauchen, und kein weiteres benötigen. Es kann verwendet werden, um über Menge zu sprechen, wie wenn man `enough food` hat, oder um über Qualität zu sprechen, wie wenn etwas `good enough` ist.

Hören Sie sich den Dialog an und füllen Sie die Lücken aus.
