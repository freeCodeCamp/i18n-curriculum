# --description--

$f(n)$을 $10^9$보다 작은 가장 큰 양의 정수 $x$로 정의합니다. 이때 $n^x$의 마지막 9자리 숫자가 $x$와 같아야 합니다(앞에 오는 0 포함). 그런 정수가 없으면 0으로 합니다.

예를 들어:

$$\begin{align}
  & f(4) = 411\\,728\\,896 (4^{411\\,728\\,896} = ...490\underline{411728896}) \\\\
  & f(10) = 0 \\\\
  & f(157) = 743\\,757 (157^{743\\,757} = ...567\underline{000743757}) \\\\
  & Σf(n), 2 ≤ n ≤ 103 = 442\\,530\\,011\\,399
\end{align}$$

$2 ≤ n ≤ {10}^6$일 때 $\sum f(n)$을 구하세요.
