# --description--

$f(n)$ を、$n^x$ の末尾9桁が（先頭のゼロを含めて）数値 $x$ になるような、$10^9$ 未満の最大の正の整数 $x$ とし、そのような整数が存在しない場合は0とします。

例えば：

$$\begin{align}
  & f(4) = 411\\,728\\,896 (4^{411\\,728\\,896} = ...490\underline{411728896}) \\\\
  & f(10) = 0 \\\\
  & f(157) = 743\\,757 (157^{743\\,757} = ...567\underline{000743757}) \\\\
  & Σf(n), 2 ≤ n ≤ 103 = 442\\,530\\,011\\,399
\end{align}$$

$2 ≤ n ≤ {10}^6$ のときの $\sum f(n)$ を求めてください。
