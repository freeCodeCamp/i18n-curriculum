# --explanation--

`Clarifying` 意味着以消除混淆的方式解释某事。它帮助某人更好地理解一个概念或情况。例如：

`Thanks for clarifying how the API works.`——说话者感激因为解释使 API 更易理解。

`Abstraction` 指通过仅显示必要的特色并隐藏复杂的内部工作来简化某物。例如：

`Using a login function is an example of abstraction - you don't need to know how the password is checked, just that it works.` —— 这展示了用户如何与简单的接口交互，而技术逻辑保持隐藏。
