# --description--

숫자는 프로그래밍 논리의 기본 요소입니다! 사실, 기본적인 수학이 조금도 포함되지 않는 유용한 프로그래밍 작업을 생각하기 어렵기 때문에 숫자가 어떻게 작동하는지 아는 것이 매우 중요합니다. 다행히도, 이것은 꽤 간단합니다.

| 연산자 | 설명                        | 예시               |
|--------|-----------------------------|--------------------|
| +      | 덧셈                        | 5 + 3 = 8          |
| -      | 뺄셈                        | 5 - 3 = 2          |
| *      | 곱셈                        | 5 * 3 = 15         |
| **     | 거듭제곱                    | 5 ** 3 = 125       |
| /      | 나눗셈                      | 6 / 3 = 2          |
| %      | 나머지                      | 5 % 3 = 2          |


# --assignment--
