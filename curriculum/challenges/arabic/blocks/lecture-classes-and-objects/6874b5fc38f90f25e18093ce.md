# --description--

في الدرس السابق، تعلمت عن الأصناف وكيف تعمل كخرائط لإنشاء الكائنات.

هنا، سنغوص أعمق في الخصائص والطرق.

لنلقِ نظرة أقرب على الخصائص أولًا، ثم الطرق.

الخصائص هي متغيرات تنتمي إلى كائن، لذا فهي تخزن بيانات. هناك نوعان من الخصائص: خصائص المثيل وخصائص الصنف.

خصائص المثيل فريدة لكل كائن يُنشأ من صنف، وعادةً ما تُحدد باستخدام الدالة `__init__`. أما خصائص الصنف، فهي تنتمي إلى الصنف نفسه وتُشارك بين جميع مثيلات ذلك الصنف.

للوصول إلى خاصية، تستخدم ترميز النقطة.

إليك أمثلة على كل من خصائص المثيل وخصائص الصنف، وكيفية الوصول إليها من الكائنات:

```python
class Dog:
    species = "French Bulldog" # Class attribute

    def __init__(self, name):
        self.name = name # Instance attribute

print(Dog.species) # French Bulldog

dog1 = Dog("Jack")
print(dog1.name)    # Jack
print(dog1.species) # French Bulldog

dog2 = Dog("Tom")
print(dog2.name)    # Tom
print(dog2.species) # French Bulldog
```

لاحظ أنه يمكنك الوصول إلى خصائص الصنف مباشرة من الصنف نفسه، لكنك تحتاج إلى إنشاء كائن وتمرير بيانات له أولًا قبل أن تتمكن من الوصول إلى خصائص المثيل.

السيارات مثال جيد آخر، إذ أن جميع السيارات لها طراز ولون:

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

car_1 = Car("red", "Toyota Corolla")
car_2 = Car("green", "Lamborghini Revuelto")

print(car_1.model) # Toyota Corolla
print(car_2.model) # Lamborghini Revuelto

print(car_1.color) # red
print(car_2.color) # green
```

الطرق هي دوال تُعرف داخل صنف. من خلالها، يمكن لأي كائن معرف من صنف أن ينفذ إجراءات تعمل على بياناته أو تعدلها. كما تصل إلى الطريقة باستخدام ترميز النقطة.

على سبيل المثال، يمكن للكلاب النباح. لذا يمكن أن يكون لدينا طريقة `bark` في صنف `Dog` كما رأيت في درس سابق:

```python
class Dog:
   species = "French Bulldog"

   def __init__(self, name):
     self.name = name

   def bark(self):
       return f"{self.name} says woof woof!"

jack = Dog("Jack")
jill = Dog("Jill")

print(jack.bark()) # Jack says woof woof!
print(jill.bark()) # Jill says woof woof!
```

يمكن أن يحتوي صنف `Car` أيضًا على طريقة `describe`:

```python
class Car:
    def __init__(self, color, model):
        self.color = color  # Instance attribute
        self.model = model  # Instance attribute

    def describe(self):
        return f"This car is a {self.color} {self.model}"

car_1 = Car("red", "Toyota Corolla")
car_2 = Car("green", "Lamborghini Revuelto")

print(car_1.describe()) # This car is a red Toyota Corolla
print(car_2.describe()) # This car is a green Lamborghini Revuelto
```
