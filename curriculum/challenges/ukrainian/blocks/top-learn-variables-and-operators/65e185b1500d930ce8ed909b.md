# --description--
Найпростіший спосіб почати — створити HTML-файл із JavaScript-кодом всередині. Введіть базовий каркас HTML у файл десь на своєму комп’ютері:

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>
  <script>
    // Your JavaScript goes here!
    console.log("Hello, World!")
  </script>
</body>
</html>
```

Збережіть і відкрийте цей файл у веб-браузері (ви можете використати <a href="https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer" target="_blank" >"Live Server" у Visual Studio Code</a> для цього!) і потім відкрийте консоль браузера, клацнувши правою кнопкою миші на порожній вебсторінці та вибравши "Перевірити" або "Перевірити елемент". У панелі «Інструменти розробника» знайдіть і виберіть вкладку «Консоль», де ви маєте побачити вихідні дані нашої `console.log` інструкції.

> `console.log()` — це команда для виведення чогось у консоль розробника у вашому браузері. Ви можете використовувати її, щоб виводити результати з будь-яких наступних статей і вправ у консоль. Ми заохочуємо вас писати код разом з усіма прикладами в цьому та майбутніх уроках.

Ще один спосіб додати JavaScript на вебсторінку — через зовнішній скрипт. Це дуже схоже на підключення зовнішніх CSS-файлів до вашого сайту.

```html
  <script src="javascript.js"></script>
```

JavaScript-файли мають розширення `.js`, подібно до `.css` для таблиць стилів. Зовнішні JavaScript-файли використовують для більш складних скриптів.

# --questions--

## --text--

Яка інструкція правильно описує, як включити JavaScript у HTML-документ?

## --answers--

JavaScript можна включати лише внутрішньо всередині тегу `<script>` у тілі HTML.

---

JavaScript-код має бути написаний у розділі `<head>` HTML-документа, щоб він виконувався правильно.

---

JavaScript можна включати безпосередньо в HTML-файл за допомогою тегу `<script>` або підключати зовнішньо через тег `<script src="javascript.js">`.

---

Зовнішні JavaScript-файли потребують спеціального атрибута в тегу `<script>`, щоб браузери їх розпізнавали.
