# --interactive--

一元運算子作用於單一運算元，以執行型別轉換、數值操作或檢查特定條件等操作。讓我們來看看一些常見的一元運算子及其運作方式。

一元加號運算子會將其運算元轉換成數字。如果運算元已經是數字，則保持不變。

:::interactive_editor

```js
const str = '42';
const strToNum = +str;

console.log(strToNum); // 42
console.log(typeof str); // string
console.log(typeof strToNum); // number
```

:::

一元加號在你想確保正在處理數值時非常方便。正如你可能猜到的，有一個一元負號運算子。它會將運算元的值取反。它的運作方式類似於一元加號，只是會翻轉符號。

:::interactive_editor

```js
const str = '42';
const strToNegativeNum = -str;

console.log(strToNegativeNum); // -42
console.log(typeof str); // string
console.log(typeof strToNegativeNum); // number
```

:::

邏輯 NOT 運算子，由驚嘆號（`!`）表述，是另一種一元運算子。它會反轉其運算元的布林值。所以，如果運算元是 `true`，它會變成 `false`，如果是 `false`，它會變成 `true`。 

:::interactive_editor

```js
let isOnline = true;
console.log(!isOnline); // false

let isOffline = false;
console.log(!isOffline); // true
```

:::

以 bit 為單元逐一取反運算子是一個較少使用的一元運算子。以波浪號 `~` 表示，它會反轉數字的二進位表述。電腦以二進位格式（1 和 0）儲存數字。`~` 運算子會翻轉每一個位元，意即將所有 1 變成 0，所有 0 變成 1。你將在未來的課程中學習更多關於二進位和位元的內容。

:::interactive_editor

```js
const num = 5; // The binary for 5 is 00000101

console.log(~num); // -6
```

:::

在這個範例中，`5` 變成了 `-6`，因為對 `5` 應用 `~` 運算子會得到 `- (5 + 1)`，由於二補數表述，這等於 `-6`。二補數是電腦用來以二進位表述負數的一種方式。除非你從事像是直接操作位元這類低階程式設計工作，否則你可能不會經常使用以 bit 為單元逐一反轉的 NOT 運算子。

`void` 關鍵字是一個一元運算子，用來評估表達式並傳回 `undefined`。

:::interactive_editor

```js
const result = void (2 + 2);

console.log(result); // undefined
```

:::

`void` 也常用於超連結中以防止導覽：

```js
<a href="javascript:void(0);">Click Me</a>
```

最後，還有你在先前課程中學過的 `typeof` 運算子。這會傳回其運算元的型別字串。 

:::interactive_editor

```js
const value = 'Hello world';

console.log(typeof value); // string
```

:::
