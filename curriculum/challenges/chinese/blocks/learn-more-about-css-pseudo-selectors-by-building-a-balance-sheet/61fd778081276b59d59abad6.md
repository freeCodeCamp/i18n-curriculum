# --description--

在你的第一个 `tr` 中，添加一个 `th` 元素，文本为 `Cash This is the cash we currently have on hand.`。 将除 `Cash` 之外的文本包裹在一个 `class` 属性为`description` 的 `span` 元素中。

接着，添加三个 `td` 元素，按顺序添加以下文本：`$25`、`$30`、`$28`。 给第三个 `td` 元素添加一个 `class` 属性，设置为 `current`。
