# --description--

이번 수업에서는 함수 타입을 다루는 방법을 배우세요. 매개변수 타입 주석부터 시작하겠습니다.

기본 함수의 예는 다음과 같습니다:

```ts
function circleArea(radius) {
  return Math.PI * radius * radius;
}
```

현재 TypeScript는 `Parameter 'radius' implicitly has an 'any' type` 메시지를 표시할 것입니다. `any` 타입은 값이 어떤 타입이든 될 수 있음을 나타냅니다. 이 타입에 대해서는 이후 수업에서 더 배우게 됩니다.

오류를 없애고 매개변수에 명시적으로 타입을 지정하려면 이렇게 타입을 지정할 수 있습니다:

```ts
function circleArea(radius: number) {
  return Math.PI * radius * radius;
}
```

만약 `circleArea` 함수에 숫자가 아닌 다른 타입을 인수로 전달하면, TypeScript는 다음과 같은 오류를 표시합니다:

```ts
function circleArea(radius: number) {
  return Math.PI * radius * radius;
}

// Argument of type 'string' is not assignable to parameter of type 'number'
circleArea("3");
```

함수가 예상하는 것보다 더 많은 인수를 제공하면 TypeScript는 오류를 표시합니다:

```ts
function circleArea(radius: number) {
  return Math.PI * radius * radius;
}

// Expected 1 arguments, but got 2.
circleArea(3, 4);
```

인수가 너무 적으면 TypeScript도 오류를 표시합니다:

```ts
function circleArea(radius: number) {
  return Math.PI * radius * radius;
}

// Expected 1 arguments, but got 0.
// An argument for 'radius' was not provided.
circleArea();
```

매개변수를 선택 사항으로 만들려면 매개변수 옆에 `?` 기호를 사용하세요:

```ts
function area(radius: number, height?: number): number {
  const baseArea = Math.PI * radius * radius;

  if (height !== undefined) {
    const sideArea = 2 * Math.PI * radius * height;
    return baseArea + sideArea;
  }

  return baseArea;
}
```

함수 내부의 조건문은 `height` 인수가 제공되지 않았을 경우를 대비한 것입니다.

반환값에 타입을 지정하고 싶으면 다음과 같이 반환 타입 주석을 추가할 수 있습니다:

```ts
function circleArea(radius: number): number {
  return Math.PI * radius * radius;
}
```

반환 타입 주석은 매개변수 목록 뒤에 옵니다. 변수 타입 주석과 마찬가지로 대부분의 경우 TypeScript가 반환 타입을 똑똑하게 추론합니다. 함수의 의도를 더 명확히 하거나 반환값의 의도치 않은 변경을 막고 싶을 때 명시적인 반환 타입 주석을 사용하세요.

다음 예제는 비동기 함수와 프로미스를 다룹니다. 함수가 프로미스를 반환할 경우, `Promise` 타입을 이렇게 사용할 수 있습니다:

```ts
async function getCityTemperature(city: string): Promise<number> {
  const response = await fetch(`https://api.example.com/weather?city=${city}`);
  const data = await response.json(); 
  return data.temperature;
}
```

이 경우 반환 타입을 제거하면 TypeScript가 반환을 `Promise<any>`로 추론할 수 있습니다. 따라서 반환값에 명시적으로 타입을 지정하는 것이 좋습니다.

지금까지는 이름 있는 함수 예제만 살펴봤습니다. 그렇다면 익명 함수에서는 타입이 어떻게 작동할까요?

다음은 `forEach`에 전달된 콜백으로 익명 함수를 사용하는 예입니다:

```ts
const languages = ["JavaScript", "Python", "TypeScript"];

languages.forEach((lang) => {
  console.log(`${lang.toUpperCase()} has ${lang.length} characters`);
});
```

`lang` 매개변수에는 타입 주석이 없습니다. 대신 TypeScript는 문맥 타입 추론을 사용하여 해당 타입이 나타나는 문맥에서 타입을 추론합니다.
