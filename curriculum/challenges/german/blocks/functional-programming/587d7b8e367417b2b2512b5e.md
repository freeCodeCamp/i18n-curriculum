# --description--

Falls Sie es noch nicht bemerkt haben: Das Problem in der vorherigen Aufgabe lag beim `splice`-Aufruf in der `tabClose()`-Funktion. Leider verändert `splice` das ursprüngliche Array, auf dem es aufgerufen wird, sodass der zweite Aufruf ein verändertes Array verwendete und unerwartete Ergebnisse lieferte.

Dies ist ein kleines Beispiel für ein viel größeres Muster – Sie rufen eine Funktion auf eine Variable, ein Array oder ein Objekt auf, und die Funktion verändert die Variable oder etwas im Objekt.

Einer der Grundsätze der funktionalen Programmierung ist, nichts zu verändern. Veränderungen führen zu Bugs. Es ist einfacher, Bugs zu vermeiden, wenn Sie wissen, dass Ihre Funktionen nichts verändern – weder die Funktionsargumente noch eine globale Variable.

Das vorherige Beispiel hatte keine komplizierten Operationen, aber die `splice`-Methode veränderte das ursprüngliche Array und führte zu einem Bug.

Erinnern Sie sich: In der funktionalen Programmierung nennt man das Verändern oder Ändern von Dingen <dfn>Mutation</dfn>, und das Ergebnis wird <dfn>Nebeneffekt</dfn> genannt. Eine Funktion sollte idealerweise eine <dfn>reine Funktion</dfn> sein, das heißt, sie verursacht keine Nebeneffekte.

Lassen Sie uns diese Disziplin meistern und keine Variable oder kein Objekt in unserem Code verändern.

# --instructions--

Füllen Sie den Code für die Funktion `incrementer` so aus, dass sie den Wert der globalen Variable `fixedValue` um eins erhöht zurückgibt.

# --hints--

Ihre Funktion `incrementer` sollte den Wert von `fixedValue` (das ist `4`) nicht verändern.

```js
incrementer();
assert(fixedValue === 4);
```

Ihre `incrementer`-Funktion sollte einen Wert zurückgeben, der um eins größer ist als der Wert von `fixedValue`.

```js
const __newValue = incrementer();
assert(__newValue === 5);
```

Ihre `incrementer`-Funktion sollte einen Wert basierend auf dem Wert der globalen `fixedValue`-Variable zurückgeben.

```js
(function () {
  fixedValue = 10;
  const newValue = incrementer();
  assert(fixedValue === 10 && newValue === 11);
  fixedValue = 4;
})();
```
