# --description--

Python 提供了一种特殊类型的字符串，称为*原始*字符串。 原始字符串以 `r` 为前缀。 与常规字符串的主要区别在于它们如何处理反斜杠字符：在原始字符串中，反斜杠被视为文字字符而不是转义字符。 编写正则表达式时，使用原始字符串是一种很好的做法，因为它们通常可以包含很多 `\` 字符。

通过在 `pattern` 字符串前添加 `r` 作为前缀，将其转换为原始字符串。
