# --description--

En la siguiente ecuación x, y, y n son enteros positivos.

$$\frac{1}{x} + \frac{1}{y} = \frac{1}{n}$$

Para `n` = 4 hay exactamente tres soluciones distintas:

$$\begin{align}
  & \frac{1}{5} + \frac{1}{20} = \frac{1}{4}\\\\
  \\\\
  & \frac{1}{6} + \frac{1}{12} = \frac{1}{4}\\\\
  \\\\
  & \frac{1}{8} + \frac{1}{8} = \frac{1}{4}
\end{align}$$

¿Cuál es el valor mínimo de `n` para el cual el número de soluciones distintas supera mil?
