# --description--

Cuando quieres preguntar sobre la existencia de lugares o cosas específicas en tu ubicación actual, puedes usar la estructura de pregunta `Are there any ... around here?`. Por ejemplo:

`Are there any Wi-Fi spots around here?` - Estás preguntando si puedes encontrar Wi-Fi cerca de este lugar.

La frase `around here` se refiere al área cercana.
