# --description--

Для цілих чисел $a$ та $b$ визначимо $D(a, b)$ як область, обмежену параболою $y = x^2$ та лінією $y = ax + b$: $D(a, b) = \\{ (x, y) | x^2 ≤ y ≤ ax + b \\}$.

$L(a, b)$ визначається як кількість вузлів ґратки, що містяться в $D(a, b)$. Наприклад, $L(1, 2) = 8$ та $L(2, -1) = 1$.

Також визначимо $S(N)$ як суму $L(a, b)$ для всіх пар ($a$, $b$), таких що площа $D(a, b)$ є раціональним числом і $|a|,|b| ≤ N$.

Ми можемо перевірити, що $S(5) = 344$ та $S(100) = 26\\,709\\,528$.

Знайдіть $S({10}^{12})$. Відповідь подайте за модулем ${10}^8$.
