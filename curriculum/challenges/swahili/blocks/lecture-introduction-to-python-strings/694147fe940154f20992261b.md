# --description--

Katika funzo la awali, ulijifunza jinsi kila herufi katika mfuatano wa herufi inaweza kutambuliwa kwa kielezo chake (kuanzia sifuri), na kufikiliwa kwa kutumia uandishi wa mabano:

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
print(my_str[-1]) # d
```

**Ukatwaji wa mfuatano wa herufi** unakuwezesha kutoa sehemu ya mfuatano wa herufi au kufanya kazi na sehemu maalum tu ya mfuatano huo. Hapa kuna sintaksia ya msingi:

```python
string[start:stop]
```

Ikiwa unataka kutoa herufi kutoka kielezo fulani hadi kingine, unachotakiwa kufanya ni kutenganisha kielezo cha `start` na `stop` kwa kutumia nukta mbili:

```python
my_str = 'Hello world'
print(my_str[1:4]) # ell
```

Kumbuka kuwa kielezo cha `stop` hakijumuishi, kwa hivyo `[1:4]` ilitoa herufi kutoka kielezo cha `1`, hadi, lakini isijumuisha, herufi iliyo kwenye kielezo cha `4`.

Unaweza pia kuacha kielezo cha `start` na `stop`, na Python itatumia chaguo-msingi cha `0` au mwisho wa mfuatano wa herufi, mtawalia. Kwa mfano, hii ndiyo hutokea ukiacha kielezo cha `start`:

```python
my_str = 'Hello world'
print(my_str[:7])  # Hello w
```

Hii hutoa kila kitu kuanzia kielezo cha `0` hadi (lakini isijumuisha), herufi iliyo kwenye kielezo cha `7`. Na hii ndiyo hutokea ukiacha kielezo cha `stop`:

```python
my_str = 'Hello world'
print(my_str[8:])  # rld
```

Hii hutoa kila kitu kuanzia herufi iliyo kwenye kielezo cha `8` hadi mwisho wa mfuatano wa herufi.

Kumbuka kuwa ukatwa wa mfuatano wa herufi haubadili mfuatano wa herufi wa awali:

```py
my_str = 'Hello world'
print(my_str[8:])  # rld
print(my_str)  # Hello world
```

Unaweza pia kuacha kielezo cha `start` na `stop` kwa pamoja, ambacho kitatoa mfuatano mzima wa herufi:

```python
my_str = 'Hello world'
print(my_str[:])  # Hello world
```

Mbali na kielezo cha `start` na `stop`, pia kuna kigezo cha hiari cha `step`, ambacho hutumika kubainisha ongezeko kati ya kila kielezo katika ukatwa.

Hapa kuna sintaksia yake:

```python
string[start:stop:step]
```

Katika mfano hapa chini, ukatwa unaanza kwenye kielezo cha `0`, unasimama kabla ya `11`, na hutoa kila herufi ya pili:

```python
my_str = 'Hello world'
print(my_str[0:11:2])  # Hlowrd
```

Njia moja ya msaada unayoweza kutumia na kigezo cha `step` ni kugeuza mfuatano wa herufi kwa kuweka hatua (step) kuwa `-1`, na kuacha `start` na `stop` wazi:

```python
my_str = 'Hello world'
print(my_str[::-1]) # dlrow olleH
```
