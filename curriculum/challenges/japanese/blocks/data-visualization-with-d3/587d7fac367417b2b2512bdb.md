# --description--

デフォルトでは、スケールは恒等関係を使います。これは、入力値が出力値に対応することを意味します。しかし、スケールはもっと柔軟で面白く設定できます。

例えば、データセットの値が50から480の範囲にあるとします。これはスケールの入力情報で、<dfn>ドメイン</dfn>とも呼ばれます。

これらの点をSVGの`x`軸上で10単位から500単位の間にマッピングしたいとします。これは出力情報で、<dfn>レンジ</dfn>とも呼ばれます。

`domain()`と`range()`メソッドは、スケールのこれらの値を設定します。どちらのメソッドも、少なくとも2つの要素を持つ配列を引数に取ります。例を示します。

```js
scale.domain([50, 480]);
scale.range([10, 500]);
scale(50);
scale(480);
scale(325);
scale(750);
d3.scaleLinear();
```

順に、次の値がコンソールに表示されます：`10`、`500`、`323.37`、および`807.67`。

スケールはドメインとレンジの値の間の線形関係を使って、与えられた数値の出力が何になるかを計算していることに注意してください。ドメインの最小値（50）はレンジの最小値（10）に対応します。

# --instructions--

スケールを作成し、そのドメインを`[250, 500]`に、レンジを`[10, 150]`に設定してください。

**注意:** `domain()`変数に対して`range()`と`scale`メソッドをチェーンして使うことができます。
