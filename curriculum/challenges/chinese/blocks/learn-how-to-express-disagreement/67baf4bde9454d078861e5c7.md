# --explanation--

`True, but` 是一种在表示同意的同时添加不同观点的方式。这个短语有助于保持讨论的开放和平衡。例如：

- 人物 1：`AI can automate a lot of tasks in software development.`

- 第 2 个人：`True, but human oversight is still necessary to catch errors AI might miss.`——此人同意 AI 有帮助，但提醒仍然需要人类。
