# --explanation--

У `Simple Past` питання можуть починатися з питального слова (наприклад, `what`, `where`, `how`, `who`) за такою структурою:

**Питальне слово + `did` + підмет + інфінітив дієслова.**

Питання Лізи, `What did you find there?`, використовує `what` для запиту конкретної інформації про результат дій Джеймса. Інші приклади включають:

- `Where did you go yesterday?` — ви питаєте про місце.

- `How did they solve the problem?` — ви питаєте про метод.

- `Who did you call last night?` — ви питаєте про людину.

У діалозі Ліза питає про відкриття Джеймса під час пошуку помилки.
