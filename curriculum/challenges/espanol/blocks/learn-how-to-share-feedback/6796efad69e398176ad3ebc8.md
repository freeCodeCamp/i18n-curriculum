# --explanation--

El `First Conditional` se usa para hablar de situaciones reales o posibles en el futuro y sus resultados probables. Sigue la estructura `if` + tiempo `Present Simple` para la condición, y `will` o `can` + verbo base para el resultado. Por ejemplo:

- `If you study hard, you will pass the exam.` - Esto describe una situación posible y su resultado.

- `If it rains tomorrow, we can cancel the picnic.` - Esto explica lo que podría pasar si ocurre una condición (lluvia).
