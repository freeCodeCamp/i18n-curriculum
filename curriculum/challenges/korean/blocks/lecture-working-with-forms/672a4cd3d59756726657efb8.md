# --interactive--

`button` 요소는 활성화되었을 때 특정 동작을 수행하는 데 사용됩니다. 다음은 버튼 텍스트가 `button`인 `Start Game` 요소의 예입니다.

```html
<button>Start Game</button>
```

`button` 요소를 사용하는 다른 예로는 폼 제출, 모달 표시, 사이드 메뉴 열기 및 닫기 토글 등이 있습니다. `button` 요소에는 버튼이 활성화될 때 동작을 제어하는 `type` 속성이 있습니다. `type` 속성의 첫 번째 가능한 값은 `button` 유형입니다. 다음은 `button` 요소를 `button` 유형과 텍스트 `Show Alert`으로 사용하는 예입니다:

```html
<button type="button">Show Alert</button>
```

기본적으로 버튼은 활성화해도 아무 동작도 하지 않습니다. 하지만 이 경우처럼 버튼을 인터랙티브하게 만들기 위해 JavaScript 코드를 추가할 수 있습니다.

화면에 알림이 뜨는 것을 보려면 `Show Alert` 버튼을 클릭하세요. 예제와 상호작용하려면 인터랙티브 편집기를 활성화해야 합니다.

**참고**: 이 인터랙티브 예제는 JavaScript를 사용하지만 JavaScript 코드를 이해할 필요는 없습니다. JavaScript는 이후 모듈에서 배우게 됩니다.

:::interactive_editor

```html
<button type="button">Show Alert</button>
<script src="index.js"></script>
```

```js
const btn = document.querySelector("button");
btn.addEventListener("click", () => alert("You clicked on the alert button"));
```

:::

`type` 속성의 또 다른 가능한 값은 `submit` 값입니다. 다음은 `button` 요소를 `submit` 유형으로 사용하는 예입니다.

:::interactive_editor

```html
<form action="">
  <label for="email">Email address:</label>
  <input type="email" id="email" name="email" />
  <button type="submit">Submit form</button>
</form>
```

:::

이 `form` 요소 안에는 사용자의 이메일 주소를 위한 `label`와 `input` 요소가 있습니다. 사용자가 제출 버튼을 클릭하면 데이터가 서버로 전송되어 처리됩니다. `type` 속성의 세 번째 가능한 값은 `reset` 값입니다. 다음은 리셋 버튼과 제출 버튼이 있는 `form` 요소의 예입니다.

인터랙티브 편집기를 활성화하고 미리보기 창에서 가짜 이메일 주소를 입력하여 이메일 입력란과 상호작용하세요. 그런 다음 리셋 버튼을 클릭하면 이메일이 입력란에서 사라지는 것을 볼 수 있습니다.

:::interactive_editor

```html
<form action="">
  <label for="email">Email address:</label>
  <input type="email" id="email" name="email" />
  <button type="reset">Reset form</button>
  <button type="submit">Submit form</button>
</form>
```

:::

수정된 이 예제에서는 사용자의 이메일 주소를 수집하기 위해 `label`와 `input` 요소가 사용됩니다. 사용자가 리셋 버튼을 클릭하면 모든 입력 데이터가 지워집니다. 리셋 버튼은 사용자가 실수로 데이터를 초기화할 수 있으므로 보통 좋은 선택이 아닙니다. 또한 폼에 여러 버튼이 있으면 사용자 인터페이스가 복잡해질 수 있습니다.

HTML에서 버튼을 만드는 또 다른 방법은 `input` 요소를 사용하는 것입니다. `input` 요소에도 `type` 속성이 있으며 가능한 값으로 `submit`, `reset`, `button`가 있습니다. 다음은 `input` 요소를 `type`에 `button`로 설정한 예입니다:

:::interactive_editor

```html
<input class="start-btn" type="button" value="Start Game" />
<script src="index.js"></script>
```

```js
document.addEventListener("DOMContentLoaded", () => {
  const btn = document.querySelector(".start-btn");
  btn.addEventListener("click", () => {
    const paraEl = document.createElement("p");
    const bodyEl = document.querySelector("body");

    bodyEl.appendChild(paraEl);
    paraEl.textContent = "The game has started!!!";
  });
});
```

:::

`value` 속성은 버튼 텍스트를 표시하는 데 사용됩니다. 그렇다면 `input`와 `button` 요소를 사용하는 차이는 무엇일까요? `input` 요소는 void 요소로, 자식 노드(텍스트 등)를 가질 수 없으며 시작 태그만 가질 수 있습니다. 반면 `button` 요소는 텍스트, 이미지, 아이콘 등을 중첩할 수 있어 더 유연합니다.
