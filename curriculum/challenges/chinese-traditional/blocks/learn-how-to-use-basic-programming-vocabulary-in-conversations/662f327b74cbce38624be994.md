# --description--

片語 `no worries` 是一種隨意的表達式，用來安慰某人一切都沒問題，且他們不需要擔心。它可以用來回應感謝、婉拒道歉，或安慰某人問題並不嚴重。

例如，當你的朋友不小心灑了飲料並開始道歉時，你可以說 `no worries! It's just water, and it'll dry up quickly` 來讓朋友感覺好一點。
