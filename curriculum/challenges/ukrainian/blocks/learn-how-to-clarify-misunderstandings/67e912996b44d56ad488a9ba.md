# --explanation--

`Code reuse` означає практику використання існуючого коду в нових частинах програми. Наприклад:

`The "Login" class was used in multiple applications to handle user access, which is a good example of code reuse.` - Замість того, щоб писати логіку входу для кожного застосунку, той самий код використовується повторно, що підвищує ефективність.

`Extending` означає додавання нових функцій до чогось, що вже існує. Наприклад:

`The "AdminUser" class extends the basic "User" class by adding an "approveUser()" method.` - `AdminUser` зберігає все з `User` і розширює це новими можливостями.
