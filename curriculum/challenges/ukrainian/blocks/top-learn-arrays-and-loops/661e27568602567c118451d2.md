# --description--

Щоб отримати доступ до елементів масиву, ви можете використовувати індекс числа. Індекс починається з 0, тому перший елемент масиву має індекс 0, другий елемент — індекс 1 і так далі.

Наприклад, щоб отримати доступ до першого елемента масиву `fruits`, ви можете використати наступний код:

```javascript
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits[0]); // Output: apple
console.log(fruits[2]); // Output: orange
```

Якщо індекс виходить за межі, JavaScript поверне `undefined`. Це означає, що якщо ви спробуєте отримати доступ до `fruits[3]` у наведеному прикладі, він поверне `undefined`.
