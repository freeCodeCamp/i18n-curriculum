# --description--

Fügen Sie innerhalb Ihres `tbody`-Elements vier `tr`-Elemente hinzu. Geben Sie den ersten drei ein `class`-Attribut mit dem Wert `data` und dem vierten ein `class`-Attribut mit dem Wert `total`.

# --hints--

Ihr `tbody`-Element sollte vier `tr`-Elemente enthalten.

```js
const children = [...document.querySelector('tbody')?.children];
assert.lengthOf(children, 4);
children.forEach(child => assert.strictEqual(child?.localName, 'tr'));
```

Die ersten drei `tr`-Elemente sollten das `class`-Attribut mit dem Wert `data` haben.

```js
const children = [...document.querySelector('tbody')?.children];
children.forEach((child, index) => {
  if (index < 3) {
    assert.isTrue(child?.classList?.contains('data'));
  }
});
```

Das vierte `tr`-Element sollte das `class`-Attribut mit dem Wert `total` haben.

```js
const children = [...document.querySelector('tbody')?.children];
assert.isTrue(children?.[3]?.classList?.contains('total'));
```
