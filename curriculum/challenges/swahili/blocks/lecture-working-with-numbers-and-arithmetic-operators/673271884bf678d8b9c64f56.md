# --interactive--

JavaScript hutoa zana za kufanya operesheni za msingi za kihesabu kwa nambari, kama vile kuongeza, kutoa, kuzidisha, na kugawanya. JavaScript pia ina opereta za operesheni za kihesabu tata zaidi, kama mabaki na nguvu.

Zana hizi zote huitwa opereta za operesheni za kihesabu. Tuchunguze opereta hizi kwa undani, jinsi ya kuzitumia na jinsi ya kuzichanganya.

Opereta ya kuongeza ni alama ya kuongeza (`+`). Opereta ya kuongeza inakuwezesha kupata jumla ya nambari mbili au zaidi. Katika operesheni za kuongeza, mpangilio wa nambari hauna umuhimu:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 + num2;
const result2 = num2 + num1;
const result3 = num2 + num1 + num3;

console.log(result1); // 15
console.log(result2); // 15
console.log(result3); // 30
```

:::

Opereta ya kutoa ni alama ya kutoa (`-`). Inakuwezesha kupata tofauti kati ya nambari mbili. Tumia alama ya kutoa unapotaka kutoa nambari moja kutoka kwa nyingine, kawaida nambari ndogo kutoka kwa kubwa:

:::interactive_editor

```js
const difference = 10 - 5;
console.log(difference); // 5
```

:::

Kama nambari ndogo itakuja kwanza, utapata matokeo hasi:

:::interactive_editor

```js
const difference = 5 - 10;
console.log(difference); // -5
```

:::

Unaweza pia kuweka nambari kwenye vigezo na kufanya utoaji kwa kutumia majina ya vigezo:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const result = num1 - num2;

console.log(result); // 5
```

:::

Katika JavaScript, opereta ya kuzidisha inawakilishwa na nyota (`*`) na hutumika kupata bidhaa ya nambari mbili au zaidi. Mpangilio wa nambari unazozidisha hauna umuhimu:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 * num2;
const result2 = num2 * num1;
const result3 = num2 * num1 * num3;

console.log(result1); // 50
console.log(result2); // 50
console.log(result3); // 750
```

:::

Katika JavaScript, opereta ya kugawanya ni mstari wa kosa (`/`), tofauti na alama ya kugawanya inayotumika katika hesabu za kawaida (`÷`). Unafanya operesheni za kugawanya kwa kutumia opereta ya kugawanya. Hapa, mpangilio wa nambari unazogawanya ni muhimu:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 / num2;
const result2 = num2 / num1;
const result3 = num2 / num1 / num3;

console.log(result1); // 2
console.log(result2); // 0.5
console.log(result3); // 0.03333333333333333
```

:::

Ni muhimu kutambua kwamba ukijaribu kugawanya kwa sifuri, JavaScript itarudisha `Infinity`:

:::interactive_editor

```js
const result = 10 / 0; 

console.log(result); // Infinity
```

:::

Hakikisha kuepuka aina hizo za mahesabu ili usipate matokeo yasiyotegemewa katika msimbo wako.

Opereta ya mabaki, inayowakilishwa na alama ya asilimia (`%`), inarudisha mabaki ya mgawanyo. Mabaki katika hesabu ni thamani iliyobaki baada ya kugawanya:

:::interactive_editor

```js
const num1 = 10;
const num2 = 3;
const remainder = num1 % num2;

console.log(remainder); // 1
```

:::

Opereta ya nguvu, inayowakilishwa na nyota mbili (`**`), huinua nambari moja kwa nguvu ya nyingine:

:::interactive_editor

```js
const num1 = 2;
const num2 = 3;

const exponent = num1 ** num2;
console.log(exponent); // 8
```

:::

Inawezekana kuchanganya opereta katika operesheni au usemi wa kihesabu mmoja:

:::interactive_editor

```js
const result = 10 + 5 * 2 - 8 / 4;
console.log(result); // 18
```

:::

Unapochanganya opereta tofauti katika usemi mmoja, injini ya JavaScript hufuata mfumo unaoitwa kipaumbele cha opereta ili kuamua mpangilio wa operesheni. Kipaumbele cha opereta kinaamua mpangilio wa utekelezaji wa operesheni katika usemi wa kihesabu. Utajifunza zaidi kuhusu kipaumbele cha opereta katika mafundisho yajayo.
