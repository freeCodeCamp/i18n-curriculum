# --explanation--

`Present Simple` 시제에서 동사의 3인칭 단수형은 보통 `-s` 또는 `-es`로 끝납니다. 예를 들어:

- `He codes every day.` - 동사 `codes`는 `-s`를 가리키기 때문에 `he`로 끝납니다.

- `She fixes bugs in the system.` - 동사 `fixes`는 `-es`를 가리키기 때문에 `she`로 끝납니다.

`Concern`는 무언가에 대해 걱정하거나 불안하게 만드는 것을 의미합니다. 예를 들어:

- `The security of the network concerns the IT team.` - IT 팀은 네트워크 보안에 대해 걱정하고 있습니다.

- `It concerns me that the software isn't updating correctly.` - 소프트웨어가 제대로 업데이트되지 않는 것에 대해 걱정하고 있습니다.
