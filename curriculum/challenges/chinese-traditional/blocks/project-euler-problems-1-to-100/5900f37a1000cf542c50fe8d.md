# --description--

以下反覆的數列定義於正整數集合：

<div style='padding-left: 4em;'><var>n</var> → <var>n</var>/2 (<var>n</var> is even)</div>

<div style='padding-left: 4em;'><var>n</var> → 3<var>n</var> + 1 (<var>n</var> is odd)</div>

使用上述規則並從 13 開始，我們產生以下序列：

<div style='text-align: center;'>13 → 40 → 20 → 10 → 5 → 16 → 8 → 4 → 2 → 1</div>

可以看出這個序列（從 13 開始，到 1 結束）包含 10 個項目。雖然尚未被證明（Collatz 問題），但普遍認為所有起始數字最終都會結束於 1。

在給定的 `limit` 下，哪個起始數字會產生最長的串鏈？

**注意：** 一旦串鏈開始，條款允許超過 `limit`。

```js
function longestCollatzSequence(limit) {
  let longest = 1;
  let maxLength = 1;
  for (let i = Math.floor(limit / 2); i < limit; i++) {
    let len = colLen(i);
    if (len > maxLength) {
      longest = i;
      maxLength = len;
    }
  }
  return longest;
}

const knownSequence = { '1': 1 };

function colLen(n) {
  if (knownSequence[n]) {
    return knownSequence[n];
  } else {
    const len = n % 2 === 0 ? colLen(n / 2) + 1 : colLen((3 * n + 1) / 2) + 2;
    knownSequence[n] = len;
    return len;
  }
}
```
