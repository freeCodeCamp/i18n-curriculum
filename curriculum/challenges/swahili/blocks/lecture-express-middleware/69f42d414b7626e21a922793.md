# --description--

Middleware ya ngazi ya kipanga-njia ni sawa na middleware ya ngazi ya programu, lakini imefungwa kwa mfano wa `express.Router()` badala ya mfano mkuu wa programu. Inaweza kutumika kugawanya programu yako ya Express kwa kuunda moduli za njia za data zilizotengwa.

Middleware ya ngazi ya kipanga-njia hufanya kazi kwa njia ile ile kama middleware ya ngazi ya programu lakini hutumika kwa mifano maalum ya kipanga-njia. Hii inafanya iwe rahisi kutumia middleware kwa seti fulani ya njia za data badala ya kutumia kwa ujumla.

```js
const express = require('express')
const app = express()
const router = express.Router()

// Router-level middleware to log requests to '/menu' routes
router.use((req, res, next) => {
  console.log('Request made to /menu route')
  next() // Pass control to the next handler
})

// Define routes
router.get('/drinks', (req, res) => {
  res.send('Welcome to the drinks menu!')
})

app.use('/menu', router) // Apply the router

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* `router.use()`: Hii ni middleware ya ngazi ya kipanga-njia, ambayo hutumika tu kwa njia za data zinazoshughulikiwa na kipanga-njia.
* `app.use('/menu', router)`: Middleware hutumika tu kwa njia za data zinazotangulia na `/menu` (mfano, `/menu/drinks`).

Middleware ya ngazi ya kipanga-njia inakuwezesha kugawanya programu yako katika sehemu za moduli na kutumia middleware kwa makusudi kwa kundi la njia za data. Hii husaidia katika upangaji na uwazi wa msimbo.
