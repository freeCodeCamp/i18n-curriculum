# --description--

令 $f(n)$ 為滿足 $x$ 和 $y$ 為正整數、$x ≤ y$ 且 $x$ 和 $y$ 的最小公倍數等於 $n$ 的有序對 ($x$, $y$) 的數量。

令 $g$ 為 $f$ 的累加函式，即：$g(n) = \sum f(i)$，其中 $1 ≤ i ≤ n$。

已知 $g({10}^6) = 37\\,429\\,395$。

求 $g({10}^{12})$。
