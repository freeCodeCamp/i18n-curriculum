# --explanation--

`Brute force attacks` sono quando gli hacker provano più password finché non trovano quella giusta. Per esempio:

`The system detected a brute force attack on the login page.` - L'attacco ha provato molte password per entrare.

`SQL injection attempts` succedono quando gli hacker cercano di inserire codice SQL dannoso per consultare o modificare un database. Per esempio:

`The firewall stopped two SQL injection attempts.` - Qualcuno ha tentato di hackerare il database usando codice dannoso.
