# --description--

値や配列を繰り返す最も一般的な方法を理解したので、配列を繰り返す他の方法もいくつかあります。配列には`map()`という組み込みメソッドがあり、元の配列の各要素に関数を適用して新しい配列を作成するために使われます。`map()`メソッドは元の配列を変更しません。`map()`メソッドの構文は次の通りです。

```javascript
const array = [1, 2, 3, 4, 5];

const newArray = array.map((arrayValue) => {
  return arrayValue * 2;
});

console.log(newArray); // Output: [2, 4, 6, 8, 10]
```

`map()`メソッドは、元の配列の各要素に関数`(arrayValue) => { return arrayValue * 2; }`を適用して新しい配列を作成します。これは元の配列を変更せずに配列の要素を変換したい場合に特に便利です。
