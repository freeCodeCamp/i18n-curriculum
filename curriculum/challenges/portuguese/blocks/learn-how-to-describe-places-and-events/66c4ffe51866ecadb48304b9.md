# --explanation--

Uma `tag question` é uma pergunta curta adicionada ao final de uma frase para confirmar ou verificar algo.É frequentemente usado quando o falante espera que o ouvinte concorde com ele.Por exemplo: 

- `It's a nice day, isn't it?` - O falante espera que o ouvinte concorde que é um dia agradável.

- `They are coming to the meeting, aren't they?` - O falante espera que o ouvinte concorde que eles virão para a reunião.

Quando Tom diz, `Isn't it automatic?`, ele está pedindo confirmação de que a auto-completação deve acontecer automaticamente.
