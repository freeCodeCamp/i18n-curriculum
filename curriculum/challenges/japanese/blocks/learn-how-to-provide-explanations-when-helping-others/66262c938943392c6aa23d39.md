# --description--

`hard drive` は、コンピューターでデータを永久に保存するために使われるコンポーネントです。すべてのデジタルファイルが保管されている大きなファイルキャビネットのようなものです。たとえば、文書や写真、音楽をコンピューターに保存すると、それらは `hard drive` に保存されます。

これは、一時的に使う間だけ物を置いておく机のような一時保存とは異なります。
