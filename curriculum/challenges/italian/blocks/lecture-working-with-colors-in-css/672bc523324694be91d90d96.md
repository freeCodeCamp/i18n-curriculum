# --interactive--

Quando si stilizzano siti web, uno dei modi più versatili e intuitivi per lavorare con i colori è usare il modello di colore HSL. HSL sta per tonalità, saturazione e luminosità — tre componenti chiave che definiscono un colore.

Il modello di colore HSL rappresenta i colori in un modo più vicino a come gli esseri umani percepiscono il colore. In questa lezione, esploreremo cos’è il modello di colore HSL e come si usa la funzione `hsl()` in CSS.

La tonalità è il tipo di colore, rappresentata come un angolo sulla ruota dei colori, che va da `0` a `360` gradi. Per esempio, `0` gradi rappresenta il rosso, `120` gradi è il verde e `240` gradi è il blu. Regolando la tonalità, si passa attraverso colori diversi nello spettro.

La saturazione si riferisce all’intensità o purezza del colore. È misurata in percentuale da `0%` (un colore completamente desaturato, in scala di grigi) a `100%` (la forma più vivida di quel colore). Un colore con saturazione `100%` è completamente vivido, mentre un colore con saturazione `0%` appare come una sfumatura di grigio.

La luminosità determina quanto il colore è chiaro o scuro, sempre misurata in percentuale. Un valore di luminosità `0%` produrrà il nero, `50%` darà il tono normale della tonalità, e `100%` risulterà in bianco.

In CSS, la funzione `hsl()` si usa per definire i colori usando il modello di colore HSL. Ecco la sintassi base:

```css
element {
  color: hsl(hue, saturation, lightness);
}
```

Analizziamo questo con un esempio:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
body {
  background-color: hsla(0, 0%, 1%, 1.00);
}

p {
  color: hsl(120, 100%, 50%);
}
```

:::

In questo caso, la tonalità è `120` gradi, che corrisponde al verde. La saturazione è `100%`, quindi il verde sarà completamente vivido. La luminosità è `50%`, quindi è al suo tono normale — né troppo scuro né troppo chiaro. Di conseguenza, il colore del testo del paragrafo sarà un verde brillante.

Uno dei principali vantaggi del modello di colore HSL è la sua natura intuitiva. Permette di regolare facilmente la vivacità o la luminosità di un colore modificando i valori di saturazione e luminosità senza dover alterare il colore base (tonalità).

Per esempio, se vuoi creare diverse sfumature o tinte dello stesso colore, puoi semplicemente regolare il valore di luminosità.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="light">This is a light blue div.</div>
<div class="dark">This is a dark blue div.</div>
```

```css
div.light {
  background-color: hsl(240, 100%, 80%);
}

div.dark {
  background-color: hsl(240, 100%, 20%);
  color: hsl(0, 0%, 100%);
}
```

:::

Qui, entrambi gli elementi `div` usano la stessa tonalità (`240` gradi, che è blu), ma uno ha una luminosità di `80%` (una sfumatura più chiara di blu), e l’altro ha una luminosità di `20%` (una sfumatura più scura di blu).

Proprio come il modello RGB ha una funzione `rgba()` per includere la trasparenza, il modello HSL ha una funzione `hsla()`. Il quarto parametro in questa funzione rappresenta il valore alfa, che controlla l’opacità del colore. Ecco la sintassi base:

```css
element {
  background-color: hsla(hue, saturation, lightness, alpha);
}
```

Vediamo un esempio:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

Questo codice darebbe al `div` uno sfondo rosso semi-trasparente, dove la tonalità è impostata a `0` gradi (rosso), la saturazione è `100%`, la luminosità è `50%` e l’alfa è `0.5` (50% di opacità). Puoi anche specificare l’alfa come percentuale — per esempio, `hsla(0, 100%, 50%, 50%)` è equivalente a `hsla(0, 100%, 50%, 0.5)`.

Il modello di colore HSL è particolarmente utile quando devi creare schemi di colore e regolare facilmente sfumature o tinte.

Poiché HSL separa la tonalità dalla saturazione e dalla luminosità, puoi mantenere un tema di colore coerente mentre modifichi la luminosità per ombre o luci. Questo livello di flessibilità rende HSL una scelta preferita sia per designer che per sviluppatori.

Inoltre, HSL rende i valori di colore più leggibili e comprensibili rispetto ad altri metodi come i codici esadecimali o RGB. Puoi vedere più chiaramente l’intento dietro il colore e regolare di conseguenza, specialmente quando sperimenti con design diversi.

Con il modello di colore HSL, ottieni più flessibilità nel controllare tonalità, saturazione e luminosità, rendendolo uno strumento potente per creare design ricchi e dinamici in CSS.
