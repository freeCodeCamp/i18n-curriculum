# --description--

لإعلان متغير ثابت (لا يتغير)، استخدم `const` بدلًا من `let`:

```javascript
const myBirthday = '16-09-2003';
```

المتغيرات المُعلنة باستخدام `const` تُسمى "ثوابت". لا يمكن إعادة تعيينها. محاولة ذلك تسبب خطأ:

```javascript
const myBirthday = '16-09-2003';

myBirthday = '01-01-2001'; // error, can't reassign the constant!
```

عندما يتأكد المبرمج أن المتغير لن يتغير أبدًا، يمكنه إعلانه بـ `const` لضمان ذلك وإبلاغ الجميع بهذه الحقيقة.
