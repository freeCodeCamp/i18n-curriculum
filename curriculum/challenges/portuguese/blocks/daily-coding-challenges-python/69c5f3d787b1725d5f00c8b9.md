# --description--

Hoje é o dia da terra. Dado um array de itens que você limpou, retorne sua pontuação total de limpeza com base nas regras abaixo.

Os itens dados serão um dos seguintes:

| Item | Valor base |
| - | - |
| `"bottle"` | 10 |
| `"can"` | 6 |
| `"bag"` | 8 |
| `"tire"` | 35 |
| `"straw"` | 4 |
| `"cardboard"` | 3 |
| `"newspaper"` | 3 |
| `"shoe"` | 12 |
| `"electronics"` | 25 |
| `"battery"` | 18 |
| `"mattress"` | 38 |

Um item raro é representado como `["rare", value]`. Por exemplo, `["rare", 80]`. Itens raros não recebem bônus de sequência.

- Bônus de sequência: se o mesmo item aparecer consecutivamente, ele recebe pontos de bônus crescentes.
  - Primeira ocorrência consecutiva: valor base
  - Segunda: valor base + 1
  - Terceira: valor base + 2
  - etc.
- Multiplicador do quinto item: todo quinto item coletado recebe um multiplicador.
  - Quinto item: *2
  - Décimo item: *3
  - etc.

- Aplique o multiplicador após calcular quaisquer bônus.

```py
def get_cleanup_score(items):
    values = {
        'bottle': 10, 'can': 6, 'bag': 8, 'tire': 35,
        'straw': 4, 'cardboard': 3, 'newspaper': 3,
        'shoe': 12, 'electronics': 25, 'battery': 18, 'mattress': 38
    }
    total = 0
    prev_item = None
    streak = 0
    for i, item in enumerate(items):
        is_rare = isinstance(item, list) and item[0] == 'rare'
        if is_rare:
            base = item[1]
            prev_item = None
            streak = 0
        else:
            base = values[item]
            streak = streak + 1 if item == prev_item else 0
            prev_item = item
        points = base if is_rare else base + streak
        if (i + 1) % 5 == 0:
            multiplier = (i + 1) // 5 + 1
            points *= multiplier
        total += points
    return total
```
