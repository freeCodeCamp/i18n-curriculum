# --interactive--

HTML 物體，或字元引用，是用來表述 HTML 中保留字元的一組字元。 

假設你想在螢幕上顯示文字 `This is an <img/> element`。如果你使用目前編輯器中的程式碼，它不會顯示出預期的結果。即使你為範例添加了 `src` 和 `alt` 屬性，也會在段落中間顯示一張圖片，而不是預期的結果。若要與範例互動，你需要啟用互動式編輯器。


:::interactive_editor

```html
<p>This is an <img /> element</p>
```

:::

當 HTML 解析器看到小於 (`<`) 符號後面接著 HTML 頁籤名稱時，它會將其解釋為 HTML 元素。這就是為什麼你在螢幕上沒有得到預期的 `This is an <img/> element` 結果。

為了解決此問題，你可以使用 HTML 實體。以下是一個使用正確的 less (`<`) than 和 greater than (`>`) 符號的 HTML 實體的更新範例。現在你應該會在螢幕上看到 `This is an <img/> element`。

啟用互動式編輯器，並嘗試在段落元素下方添加 `&lt;p&gt;learning is fun&lt;/p&gt;`。你應該會在螢幕上看到 `<p>learning is fun</p>`。

:::interactive_editor

```html
<p>This is an &lt;img /&gt; element</p>

```

:::

這些類型的字元引用稱為命名字元引用。命名引用以和號 (`&`) 開頭，以分號 (`;`) 結尾。透過使用命名字元引用，HTML 解析器不會將其誤認為實際的 HTML 元素。 

另一種字元引用是十進位數字引用。這種字元引用以一個和號及井字號（`#`）開頭，接著是一個或多個十進位數字，最後以分號結尾。

這裡是一個使用小於符號的十進位數字引用的範例。 

啟用互動式編輯器並更改程式碼以查看不同的符號。你可以使用 `&#169;` 代表版權符號，使用 `&#174;` 代表註冊商標符號。

:::interactive_editor

```html
&#60;
```

:::

最後一種字元引用是十六進位數字引用。這種字元引用以「&」符號、井字號符號和字母 `x` 開頭，接著是一個或多個 ASCII 十六進位數字，並以分號結尾。

這裡是一個使用十六進位數字引用小於符號的範例。

啟用互動式編輯器並更改程式碼以查看不同的符號。你可以使用 `&#x20AC;` 代表歐元符號，使用 `&#x03A9;` 代表希臘大寫字母 Omega 符號。

:::interactive_editor

```html
&#x3C;
```

:::
