# --description--

根据 `isVisible` 状态动态显示单词 `Show Message` 或 `Hide Message` 会很好。

之前，你回顾了如何使用内联条件渲染。 你将对按钮文本应用类似的逻辑。

下面是使用三元运算符有条件地显示单词 `Hide` 和 `Show` 的示例：

```jsx
<button>{isMenuVisible? "Hide" : "Show"} Menu</button>
```

作为研讨会的最后一步，使用三元运算符有条件地显示消息按钮的单词 `Hide` 或 `Show`。 将条件放在 `Message` 文本之前，以便它显示 `Show Message` 或 `Hide Message`。

经过最后的改变，你的工作室就完成了！ 测试你的按钮以查看其运行情况。
