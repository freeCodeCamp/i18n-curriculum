# --description--

Diamo un’occhiata a qualche esempio rapido per visualizzare come funziona la specificità. Considera il seguente codice HTML e CSS:

```html
<!-- index.html -->

<div class="main">
  <div class="list subsection"></div>
</div>
```

```css
/* rule 1 */
.subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

Nell’esempio sopra, entrambe le regole usano solo selettori di classe, ma la regola 2 è più specifica perché usa più selettori di classe.
