# --interactive--

CSS flexbox هو نموذج تخطيط أحادي البعد يسمح لك بترتيب العناصر في صفوف وأعمدة داخل حاوية. يمكنك أيضًا التحكم في ترتيبها واتجاهها. يستخدم مطورو الويب هذا النموذج لإنشاء مواقع وتطبيقات ويب متجاوبة تتكيف مع أحجام وأوضاع الشاشات المختلفة. نُشير إلى flexbox كنموذج تخطيط أحادي البعد لأنه يركز على ترتيب العناصر على محور واحد في كل مرة. يمكن أن يكون المحور أفقيًا أو عموديًا.

هناك مفهومان رئيسيان يجب أن تعرفهما قبل أن تبدأ العمل مع flexbox: الحاوية المرنة والعنصر المرن.

الحاوية المرنة هي عنصر HTML يحتوي على تخطيط مرن. يمكنك ترتيب ومحاذاة العناصر بطرق مختلفة داخل الحاوية المرنة. لجعل عنصر HTML حاوية مرنة، تحتاج إلى إضافة `display: flex` إلى أنماط CSS الخاصة به.

العناصر المرنة هي الأبناء المباشرون للحاوية المرنة. يمكن ترتيب هذه العناصر ومحاذاتها داخل الحاوية المرنة بناءً على خصائصها. كما يمكنها الانكماش أو التوسع لتناسب المساحة المتاحة.

هذا مثال يحتوي على حاوية `main` وثلاثة عناصر `div` أبناء:

```html
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

إذا قمت بتعيين فقط `width` و`height` و`background-color` لهذه العناصر `div` في ملف CSS، فسيتم وضع كل عنصر في صف خاص به لأن الحاوية ليست مرنة افتراضيًا.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

لكن إذا أضفت `display: flex` إلى الحاوية `main`، فسيتم إعادة ترتيب عناصر `div` لتناسب نفس الصف وستنكمش إذا لزم الأمر (لرؤية المعاينات، فعّل المحرر التفاعلي):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

افتراضيًا، تكون الحاوية المرنة عنصرًا على مستوى الكتلة، لذا ستكون الحاوية نفسها في صف خاص بها مقارنة بالعناصر والحاويات الأخرى.

الآن بعد أن عرفت المزيد عن الحاويات المرنة والعناصر المرنة، يجب أن تعرف أيضًا عن خصائص flex. تحدد هذه الخصائص كيفية ترتيب العناصر المرنة وتغيير حجمها وتوزيعها داخل الحاوية المرنة. من الخصائص الأكثر استخدامًا `flex-direction`، و`justify-content`، و`align-items`، و`flex-wrap`.

رائع. الآن دعنا نتحدث قليلاً عن نموذج flex. يحدد هذا النموذج كيفية ترتيب العناصر المرنة داخل الحاوية المرنة. لكل حاوية مرنة محوران:

- المحور الرئيسي.
- المحور المتقاطع.

تحدد اتجاهات هذه المحاور كيف تؤثر الخصائص المختلفة على تخطيط وتوزيع العناصر المرنة. افتراضيًا، يكون المحور الرئيسي للحاوية المرنة أفقيًا والمحور المتقاطع عموديًا. تُرتب العناصر المرنة في اتجاه المحور الرئيسي. المحور المتقاطع عمودي على المحور الرئيسي.

تحدد خاصية `flex-direction` اتجاه المحور الرئيسي. القيمة الافتراضية لـ `flex-direction` هي `row`، التي تضع كل العناصر المرنة في نفس الصف، في اتجاه لغة المتصفح الافتراضية (من اليسار إلى اليمين أو من اليمين إلى اليسار):

```css
flex-direction: row; /* Default */
```

لعكس ترتيب العناصر في الصف، يمكنك استخدام `flex-direction: row-reverse`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: row-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

سيعكس هذا ترتيب العناصر المرنة. إذا أردت محاذاة العناصر المرنة عموديًا بدلًا من ذلك، فقط عيّن `flex-direction` إلى `column` في الحاوية المرنة. سيغير هذا اتجاه المحور الرئيسي:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

الآن ستُحاذى عناصر `div` عموديًا لأن المحور الرئيسي سيكون عموديًا والمحور المتقاطع سيكون أفقيًا.

يمكنك أيضًا عكس ترتيب العناصر المرنة عموديًا باستخدام `flex-direction: column-reverse`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

CSS flexbox هو نموذج تخطيط قوي يوفر طريقة مرنة وفعالة لترتيب العناصر داخل الحاوية. بفهم المفاهيم الأساسية للحاويات المرنة والعناصر المرنة والخصائص المختلفة، يمكنك إنشاء مواقع ويب ديناميكية ومتجاوبة تتكيف مع أحجام وأوضاع الشاشات المختلفة.
