# --explanation--

`To draft` significa crear una versión inicial de algo, como un documento, plan o diseño. Por ejemplo:

`She drafted an email to send to the client before finalizing it.` - Esto significa que ella escribió una versión preliminar del correo electrónico.  

Un `schedule` se refiere a un plan que describe cuándo se realizarán tareas o eventos. Por ejemplo:

`The project schedule shows all the deadlines for the team.` - Esto significa que el cronograma lista cuándo deben completarse las diferentes tareas.
