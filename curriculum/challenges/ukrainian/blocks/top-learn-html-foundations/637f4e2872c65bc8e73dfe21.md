# --description--

Після того, як ви оголосите doctype, потрібно додати елемент `<html>`. Це те, що називають кореневим елементом документа, тобто всі інші елементи в документі будуть його нащадками.

Це стане важливішим пізніше, коли ви навчитеся маніпулювати HTML за допомогою JavaScript. А поки просто знайте, що елемент `html` має бути включений у кожен HTML-документ.

Повертаючись до файлу `index.html`, давайте додамо елемент `<html>`, набравши його відкриваючий і кінцевий теги, ось так:

```html
<!DOCTYPE html>
<html lang="en">
</html>
```

## Що таке атрибут lang?
`lang` вказує мову текстового вмісту в цьому елементі. Цей атрибут головним чином використовується для покращення доступності вебсторінки. Він дозволяє допоміжним технологіям, наприклад читачам екрана, адаптуватися відповідно до мови та забезпечувати правильну вимову.
