# --description--

Nella lezione precedente, hai imparato a lavorare con il tipo di dato tuple. In questa lezione, scoprirai alcuni metodi comuni che userai quando lavori con le tuple.

Il primo metodo che tratteremo è `count()`. Questo metodo serve a determinare quante volte un elemento appare in una tupla. Ecco un esempio di come verificare quante volte la stringa `Rust` appare in una tupla chiamata `programming_languages`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

Poiché `Rust` appare due volte nella tupla, il metodo `count()` restituisce il numero `2`. Se l'elemento specificato nella funzione `count()` non è presente affatto nella tupla, allora il valore di ritorno è `0`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

Se non vengono passati argomenti alla funzione `count()`, Python genera un `TypeError`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count()

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: tuple.count() takes exactly one argument (0 given)
"""
```

Il metodo successivo che vedremo è il metodo `index()`. Questo metodo serve a trovare l'indice in cui un particolare elemento è presente in una tupla. Ecco un esempio di utilizzo del metodo `index()` per trovare l'indice della stringa `Java`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

Se l'elemento specificato non può essere trovato, Python genera un `ValueError`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: tuple.index(x): x not in tuple
"""
```

Un'altra cosa che puoi fare con il metodo `index()` è passare argomenti opzionali di indice di inizio e fine. Ecco un esempio di passaggio di un indice di inizio opzionale:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

In questo esempio, stiamo specificando da dove iniziare a cercare la stringa `Python`. Passando il numero `3` come secondo argomento alla funzione `index()`, stiamo indicando di iniziare la ricerca dall'indice `3`. Poiché `Python` appare due volte nella tupla, la funzione `index()` restituirà l'indice `5` invece dell'indice `2` a causa dell'uso dell'argomento opzionale dell'indice di inizio.

Puoi anche passare un indice di fine opzionale. Ecco un esempio modificato in cui si specificano un indice di inizio e uno di fine:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

Ora il risultato è l'indice `2` perché iniziamo la ricerca dall'indice `2` e cerchiamo fino, ma senza includere, l'indice `5`.

Un'altra funzione comunemente usata con le tuple è la funzione `sorted()`. In una lezione precedente hai imparato il metodo `sort()` per le liste. Bene, la funzione `sorted()` può essere usata su qualsiasi iterabile, comprese le tuple.

Ecco un esempio di creazione di una nuova lista di numeri usando la funzione `sorted()`:

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

La funzione `sorted()` creerà sempre una nuova lista con i valori ordinati. Questo differisce dal metodo `sort()` che ordina gli elementi di una lista sul posto e non restituisce una nuova lista.

Se hai bisogno di personalizzare il comportamento di ordinamento per un iterabile, puoi usare gli argomenti opzionali `reverse` e `key`. Ecco un esempio di utilizzo dell'argomento `key` per ordinare gli elementi di una tupla in base alla lunghezza:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

Se vuoi creare una nuova lista di valori in ordine inverso, puoi usare l'argomento `reverse` così:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

Le tuple sono un tipo di dato comune in Python. Capire come lavorare con esse, insieme ad alcuni metodi e funzioni utili, ti aiuterà a scrivere codice più efficiente.
