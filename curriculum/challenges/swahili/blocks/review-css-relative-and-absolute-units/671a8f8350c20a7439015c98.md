## Vitengo vya moja kwa moja

- **`px` (Pikseli)**: Hiki ni kitengo cha moja kwa moja cha ukubwa thabiti katika CSS. Ni kitengo cha moja kwa moja kinachotumika zaidi na hutoa udhibiti sahihi juu ya vipimo. `1px` daima ni sawa na 1/96 ya inchi.
- **`in` (Inchi)**: Hiki ni kitengo cha moja kwa moja kinacholingana na 96px.
- **`cm` (Santimita)**: Hiki ni kitengo cha moja kwa moja kinacholingana na 25.2/64 ya inchi.
- **`mm` (Milimita)**: Hiki ni kitengo cha moja kwa moja kinacholingana na 1/10 ya santimita.
- **`q` (Robo milimita)**: Hiki ni kitengo cha moja kwa moja kinacholingana na 1/40 ya santimita.
- **`pc` (Pika)**: Hiki ni kitengo cha moja kwa moja kinacholingana na 1/6 ya inchi.
- **`pt` (Pointi)**: Hiki ni kitengo cha moja kwa moja kinacholingana na 1/72 ya inchi.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="units">
  <div class="unit px">px</div>
  <div class="unit inch">in</div>
  <div class="unit cm">cm</div>
  <div class="unit mm">mm</div>
  <div class="unit q">q</div>
  <div class="unit pc">pc</div>
  <div class="unit pt">pt</div>
</div>
```

```css
.units {
  display: flex;
  gap: 12px;
  flex-wrap: wrap;
  align-items: flex-end;
}

.unit {
  background: steelblue;
  color: white;
  text-align: center;
  padding: 4px;
}

.px {
  width: 40px;
  height: 40px;
}

.inch {
  width: 0.5in;
  height: 0.5in;
}

.cm {
  width: 1cm;
  height: 1cm;
}

.mm {
  width: 10mm;
  height: 10mm;
}

.q {
  width: 40q;
  height: 40q;
}

.pc {
  width: 6pc;
  height: 6pc;
}

.pt {
  width: 36pt;
  height: 36pt;
}
```

## Vitengo vinavyohusiana

- **Asilimia**: Vitengo hivi vinavyohusiana vinakuwezesha kuweka ukubwa, vipimo, na sifa nyingine kama sehemu ya kipimo cha kontena mama. Kwa mfano, ukitumia `width: 50%;` kwenye kipengee, kitachukua nusu ya upana wa kontena lake mama.
- **Kitengo cha `em`**: Vitengo hivi vinahusiana na ukubwa wa fonti ya kipengee. Ukitumia `ems` kwa sifa ya `font-size`, ukubwa wa maandishi utakuwa kulingana na ukubwa wa fonti ya kipengee mama.
- **Kitengo cha `rem`**: Vitengo hivi vinahusiana na ukubwa wa fonti ya kipengee mzizi, ambacho ni kipengee cha `html`.
- **Kitengo cha `vh`**: `vh` inasimama kwa `"viewport height"` na `1vh` ni sawa na 1% ya urefu wa eneo la kuona.
- **Kitengo cha `vw`**: `vw` inasimama kwa `"viewport width"` na `1vw` ni sawa na 1% ya upana wa eneo la kuona.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  <div class="box percent">50%</div>
  <div class="box em">2em</div>
  <div class="box rem">2rem</div>
  <div class="box vh">10vh</div>
  <div class="box vw">10vw</div>
</div>
```

```css
html {
  font-size: 16px;
}

.parent {
  width: 200px;
  font-size: 20px;
  border: 2px dashed #555;
  padding: 10px;
  display: flex;
  flex-direction: column;
  gap: 12px;
}

.box {
  background: seagreen;
  color: white;
  text-align: center;
  padding: 6px;
}

.percent {
  width: 50%;
}

.em {
  font-size: 2em;
}

.rem {
  font-size: 2rem;
}

.vh {
  height: 10vh;
}

.vw {
  width: 10vw;
}
```

## Kitendakazi cha `calc`

- **Kitendakazi cha `calc()`**: Kwa kitendakazi cha `calc()`, unaweza kufanya mahesabu moja kwa moja ndani ya mitindo yako ili kubaini thamani za sifa kwa njia ya mabadiliko. Hii inamaanisha unaweza kuunda interface zinazojibadilisha kulingana na kifaa kwa kuhesabu vipimo kulingana na ukubwa wa eneo la kuona au vipengee vingine.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="calc-box">calc()</div>
```

```css
.calc-box {
  width: calc(100% - 40px);
  padding: 10px;
  background: steelblue;
  color: white;
  text-align: center;
  border: 2px solid black;
}
```
