# --explanation--

Die Struktur `it's + adjective + to` wird verwendet, um die Art einer Handlung zu beschreiben. Zum Beispiel:

`It's necessary to prepare before an important presentation.` – Das bedeutet, dass das Vorbereiten eine wesentliche Handlung vor dem Präsentieren ist.

In diesem Satz bedeutet `it's healthy to have`, dass Uneinigkeit vorteilhaft ist.
