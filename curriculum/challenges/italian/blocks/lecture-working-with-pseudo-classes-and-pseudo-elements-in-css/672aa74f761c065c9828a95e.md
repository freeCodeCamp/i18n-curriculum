# --interactive--

Le pseudo-classi sono parole chiave speciali del CSS che ti permettono di selezionare un elemento in base al suo stato o alla sua posizione specifica. Lo stato o la posizione dell'elemento possono includere:

- Quando è attivo.
- Quando viene passato sopra con il mouse.
- Quando è il primo figlio di un genitore.
- Quando è l’ultimo figlio di un genitore.
- Quando un collegamento è stato visitato.
- Quando è disabilitato.

Per usare una pseudo-classe, la aggiungi al selettore usando due punti (`:`) seguiti dal nome della pseudo-classe:

```css
selector:pseudo-class {
  /* CSS properties */
}
```

Vediamo come puoi usare una pseudo-classe per rappresentare ciascuno degli stati e delle posizioni che abbiamo appena menzionato.

La pseudo-classe `:active` ti permette di selezionare lo stato attivo di un elemento, come cliccare su un pulsante:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example Button</button>
```

```css
button:active {
  background: greenyellow;
}
```

:::

La pseudo-classe `:hover` definisce lo stato hover di un elemento. Un esempio è passare sopra con il mouse un pulsante o un collegamento:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="#">Hover over me!</a>  
```

```css
a:hover {
  text-decoration: none;
  color: white;
  background: crimson;
}
```

:::

La pseudo-classe `:first-child` seleziona un elemento che è il primo figlio del suo elemento genitore. Ecco un esempio di come selezionare il primo elemento paragrafo in un contenitore `div`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:first-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

Il primo elemento paragrafo in quel contenitore `div` riceverà il colore di sfondo `lightcoral` e il margine `padding` di `0.4rem` su tutti e quattro i lati.

La pseudo-classe `:last-child` seleziona l’ultimo elemento che è figlio del suo genitore. Ecco un esempio di come selezionare l’ultimo elemento paragrafo nel contenitore `div`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:last-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

La pseudo-classe `:visited` ti permette di definire lo stile di un collegamento che l’utente ha già visitato:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="https://www.example.com" target="_blank">Visit Example.com</a>
```

```css
a:visited {
  color: purple;
}
```

:::

La pseudo-classe `:disabled` ti permette di definire lo stile di un elemento interattivo in modalità disabilitata:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button disabled>Disabled Button</button>
```

```css
button:disabled {
  background-color: lightgray;  
}
```

:::

Come vedi, le pseudo-classi ti permettono di definire lo stile degli elementi in base alle interazioni dell’utente, come passare sopra con il mouse o visitare un collegamento. Questo rende il tuo sito web più interattivo.

Oltre alle pseudo-classi già menzionate, ce ne sono altre come:

- `:focus`
- `:first-of-type`
- `:last-of-type`
- `:nth-of-type`
- `:modal`
- `:enabled`
- `:checked`
- `:required`, e altre.
