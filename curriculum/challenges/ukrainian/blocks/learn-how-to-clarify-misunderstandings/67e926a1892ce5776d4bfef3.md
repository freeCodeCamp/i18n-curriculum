# --explanation--

`Reducing complexity` означає зробити систему легшою для розуміння та роботи з нею.
  
`Showing only what's necessary` означає, що користувач бачить лише ті частини, з якими потрібно взаємодіяти.
  
`Keeping the details hidden` стосується приховування внутрішнього коду або логіки, які не потрібні користувачу.

Разом це означає, що абстракція створює чіткий, простий інтерфейс, приховуючи складніші внутрішні процеси за лаштунками.
