# --explanation--

Cuando te pregunten por qué elegiste Python como tu primer lenguaje, es mejor resaltar las ventajas específicas de Python.

Python es conocido por ser amigable para principiantes, con una sintaxis simple que es fácil de leer y aprender. Además, Python tiene una gran colección de bibliotecas y frameworks, lo que lo convierte en una opción práctica para principiantes.

Por otro lado, decir, `I've always wanted to be a developer` no explica por qué elegiste Python. Se enfoca en el deseo de ser desarrollador pero carece de cualquier razón específica relacionada con Python.
