# --description--

一列中有 $N$ 個座位。$N$ 個人依序來坐，並依照以下規則填滿座位：

1. 沒有人坐在另一個人旁邊。
1. 第一個人選擇任何座位。
1. 每個後續的人會選擇距離已坐下的任何人最遠的座位，只要不違反規則 1。如果有多個選擇符合此條件，則該人會選擇最左邊的選擇。

請注意，由於規則 1，某些座位必定會空著，且最多可入座的人數少於 $N$（當 $N > 1$ 時）。

以下是 $N = 15$ 的可能座位安排：

<img alt="seating arrangements for N = 15" src="https://cdn.freecodecamp.org/curriculum/project-euler/comfortable-distance-ii.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

我們看到如果第一個人選擇正確，15 個座位最多可以坐下 7 個人。我們也看到第一個人有 9 個選擇來最大化可能就座的人數。

令 $f(N)$ 為第一個人為了最大化在一列中 $N$ 個座位的乘客數量所擁有的選擇數。因此，$f(1) = 1$、$f(15) = 9$、$f(20) = 6$，以及 $f(500) = 16$。

此外，$\sum f(N) = 83$ 對於 $1 ≤ N ≤ 20$，且 $\sum f(N) = 13\\,343$ 對於 $1 ≤ N ≤ 500$。

求 $\sum f(N)$，其中 $1 ≤ N ≤ {10}^{12}$。給出答案的最後 8 位數字。
