# --description--

自分の状況を話した後に相手の状況を尋ねたいときは、`你呢 (nǐ ne)`を使います。これは「あなたはどうですか？」や「あなたは？」という意味です。

# --explanation--

`你呢 (nǐ ne)`は代名詞`你 (nǐ)`（「あなた」）と助詞`呢 (ne)`を組み合わせて、前に述べた状況についての質問を作ります。意味は「あなたはどうですか？」や「あなたは？」です。会話を自然で親しみやすく続けるのに役立ちます。例えば：

`我是数据分析师 (wǒ shì shù jù fēn xī shī)。你呢 (nǐ ne)？` - 私はデータアナリストです。あなたは？
