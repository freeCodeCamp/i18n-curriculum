# --explanation--

Im `Present Perfect Continuous`-Tempus wird `since` verwendet, um anzugeben, wann eine Handlung begonnen hat und bis zur Gegenwart andauert. Zum Beispiel:

`I've been learning JavaScript since 2020.` – Es zeigt, dass die Handlung, JavaScript zu lernen, im Jahr 2020 begonnen hat und noch andauert.

Der Begriff `software update` bezieht sich auf Änderungen oder Verbesserungen, die an einem Computerprogramm vorgenommen wurden. Zum Beispiel:

`The software update fixed several bugs and added new features.` – Es bedeutet die neueste Version des Programms, die Fehlerbehebungen und Verbesserungen enthält. Es zeigt an, dass die Verbesserungen Teil der laufenden Entwicklung der Software sind.
