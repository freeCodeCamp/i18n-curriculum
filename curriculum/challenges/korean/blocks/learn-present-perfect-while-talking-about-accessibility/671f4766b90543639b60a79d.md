# --explanation--

`Present Perfect Continuous` 시제의 부정형은 `hasn't/haven't + been + present participle`으로 만들어지며, 예상한 대로 일정 기간 동안 계속되지 않은 동작을 나타냅니다. 예를 들어:

`He hasn't been exercising regularly.` - 이것은 그의 운동 루틴이 불규칙했음을 의미합니다.
  
`Present Perfect` 시제의 부정형은 `hasn't/haven't + past participle`을 사용하며, 현재 시점까지 완료되지 않은 동작을 설명합니다. 예를 들어:

`He hasn't exercised this week.` - 이것은 이번 주에 운동을 하지 않았음을 나타냅니다.
