# --explanation--

動詞 `to be` 在 `Simple Past` 時態的否定形式，單數主詞使用 `was not` 或 `wasn't`，複數主詞使用 `were not` 或 `weren't`。它用來描述某事未發生或不屬實。 例如：  

- 單數：`The code wasn't running as expected.`－程式碼發生了問題。

- 複數：`The changes weren't implemented correctly.` - 實作中的錯誤導致了這些錯誤。

在 James 的句子中，他使用 `wasn't` 來說明系統未能連接到資料庫。帶有 `wasn't` 或 `weren't` 的否定句顯示未發生或未如你所預期般運作的情況。
