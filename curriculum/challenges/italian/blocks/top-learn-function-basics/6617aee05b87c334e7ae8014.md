# --description--

Una variabile dichiarata all'interno di una funzione è visibile solo all'interno della funzione stessa. Questo si chiama ambito della variabile, e questo ambito particolare si chiama ambito locale. Le variabili dichiarate all'esterno di una funzione si chiamano variabili globali e sono visibili in tutto il programma. Per esempio, il codice seguente genererà un errore:

```js
function myFunction() {
  let x = 10;
}

console.log(x);
```

La variabile `x` è dichiarata all'interno della funzione `myFunction` e non è visibile all'esterno della funzione. Il codice genererà un `ReferenceError` perché `x` non è definita.

Se dichiari una variabile all'esterno di una funzione, si chiama variabile globale ed è visibile in tutto il programma. Per esempio, il codice seguente funzionerà:

```js
let x = 10;

function myFunction() {
  console.log(x);
}

myFunction();
```

Se una variabile con lo stesso nome è dichiarata all'interno della funzione, allora nasconde quella esterna.
