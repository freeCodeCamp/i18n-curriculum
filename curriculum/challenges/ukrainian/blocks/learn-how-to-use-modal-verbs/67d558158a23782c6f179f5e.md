# --explanation--

Джессіка каже Марку проводити `regular security audits`, щоб забезпечити безпеку.

`security audit` — це процес перевірки системи на вразливості. Наприклад:

`After the security audit, the team made improvements to the system.` — після повторення були внесені зміни для покращення безпеки.

`Regular` означає, що щось відбувається послідовно протягом часу. Наприклад:

`Regular testing helps find problems before they get worse.` — часте тестування допомагає рано виявляти проблеми.
