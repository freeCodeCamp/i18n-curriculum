# --description--

動詞 `to contain` は、何かを内包する、または持っていることを意味します。ドキュメントやREADMEファイルの文脈では、`to contain` はその文書内に含まれている情報を指します。

例えば、ボトルは水を`contain` できるのと同様に、READMEファイルはプロジェクトに関する情報を`contains` します。

# --questions--

## --text--

ドキュメントの文脈で、`to contain` は何を意味しますか？

## --answers--

除外すること

### --feedback--

何かを外に出すことを意味し、内包することの反対です。`To contain` は何かを含む、または持っていることを意味します。

削除すること

### --feedback--

何かを取り除く、または消すことを意味し、`containing` とは異なります。`Containing` は保持または含むことを意味し、削除することではありません。

---

コピーすること

### --feedback--

これは何かの複製を作ることを意味します。しかし`Containing` は、複製ではなく何かを内包していることを意味します。
