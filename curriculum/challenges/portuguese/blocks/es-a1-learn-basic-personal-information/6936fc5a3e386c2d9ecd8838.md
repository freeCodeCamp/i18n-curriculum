# --description--

Na pergunta `¿Quieres que te lo deletree?`, vários elementos trabalham juntos para expressar uma oferta.

`Quieres que` é usado para perguntar se alguém quer algo.

`Te` refere-se à pessoa com quem se está falando. Neste caso, Camila.

`Lo` refere-se à informação, aqui o e-mail do Mateo.

`Deletree` vem de `deletrear`, que significa soletrar uma palavra letra por letra.

Juntos, a pergunta é usada para oferecer soletrar o e-mail para Camila.


# --instructions--

Ouça o áudio e decida se a seguinte declaração é verdadeira ou falsa.

# --questions--
