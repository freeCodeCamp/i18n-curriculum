# --description--

Für zwei positive ganze Zahlen $a$ und $b$ ist die Ulam-Folge $U(a,b)$ definiert durch ${U{(a,b)}\_1} = a$, ${U{(a,b)}\_2} = b$ und für $k > 2$ ist ${U{(a,b)}\_k}$ die kleinste ganze Zahl größer als ${U{(a,b)}\_{(k-1)}}$, die genau auf eine Weise als Summe zweier verschiedener vorheriger Mitglieder von $U(a,b)$ geschrieben werden kann.

Zum Beispiel beginnt die Folge $U(1,2)$ mit

$$1, 2, 3 = 1 + 2, 4 = 1 + 3, 6 = 2 + 4, 8 = 2 + 6, 11 = 3 + 8$$

Die 5 gehört nicht dazu, weil $5 = 1 + 4 = 2 + 3$ zwei Darstellungen als Summe zweier vorheriger Mitglieder hat, ebenso $7 = 1 + 6 = 3 + 4$.

Finden Sie $\sum {U(2, 2n + 1)_k}$ für $2 ≤ n ≤ 10$, wobei $k = {10}^{11}$ ist.
