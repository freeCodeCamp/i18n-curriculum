# --explanation--

`Could` e `might` sono verbi modali usati per descrivere possibilità.

`Could` suggerisce che qualcosa è possibile ma non certo. Per esempio:

`If we don't fix this bug, it could cause a system crash.` - C'è la possibilità che il sistema smetta di funzionare.

`Might` suggerisce che qualcosa è possibile ma meno certo di `could`. Per esempio:

`We might need to rewrite some parts of the code.` - È possibile che siano necessarie modifiche.
