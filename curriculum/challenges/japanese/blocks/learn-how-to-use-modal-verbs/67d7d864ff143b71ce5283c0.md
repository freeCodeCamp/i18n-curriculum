# --explanation--

ジェイク はセキュリティ侵害の調査が緊急であることを強調しています。

`Need to` + 動詞 は、重要または必要なことを表現するためによく使われる構文です。例えば：

`We need to improve security.` - これはセキュリティの向上が必要であることを意味します。

`Find out` は情報を発見したり知ったりすることを意味します。例えば：

`I need to find out who accessed the system.` - これは話者が調査して情報を得る必要があることを意味します。
