# --description--

Множина вузлів ґратки $S$ називається титановою множиною, якщо існує лінія, що проходить рівно через дві точки з $S$.

Приклад титанової множини: $S = \\{(0, 0), (0, 1), (0, 2), (1, 1), (2, 0), (1, 0)\\}$, де лінія, що проходить через (0, 1) і (2, 0), не проходить через жодну іншу точку з $S$.

З іншого боку, множина {(0, 0), (1, 1), (2, 2), (4, 4)} не є титановою, оскільки лінія, що проходить через будь-які дві точки множини, також проходить через інші дві.

Для будь-якого додатного цілого числа $N$ позначимо $T(N)$ кількість титанових множин $S$, у яких кожна точка ($x$, $y$) задовольняє $0 ≤ x$, $y ≤ N$. Можна перевірити, що $T(1) = 11$, $T(2) = 494$, $T(4) = 33\\,554\\,178$, $T(111)\bmod {10}^8 = 13\\,500\\,401$ та $T({10}^5)\bmod {10}^8 = 63\\,259\\,062$.

Знайдіть $T({10}^{11})\bmod {10}^8$.
