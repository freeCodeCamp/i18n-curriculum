# --description--

In dieser Aufgabe lernen Sie den Unterschied zwischen `specific` und `specify` kennen. `Specific` ist ein Adjektiv, das etwas klar Definiertes oder Detailliertes beschreibt, während `specify` ein Verb ist, das bedeutet, etwas ausführlich darzulegen.

Zum Beispiel könnte eine `specific`-Anweisung `use port 8080` sein, während `specify` bedeuten kann, dieses Detail zu nennen: `specify the port number.`

`Setup` bezieht sich auf die Anordnung oder Vorbereitung von etwas, oft im Zusammenhang mit der Vorbereitung von Software oder Geräten, wie `setup instructions for the software.`
