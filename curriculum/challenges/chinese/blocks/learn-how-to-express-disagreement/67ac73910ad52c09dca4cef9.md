# --explanation--

`Not necessarily` 是一种礼貌表达部分不同意的方式。它不是直接说 `no`，而是暗示可能还有其他因素需要考虑。在对话中，这个短语有助于保持讨论的开放性和尊重。例如：

- 人物 1：`I think upgrading the computer's processor will fix the issue.`

- 第 2 个人：`Not necessarily. The problem could be with the software, not the hardware.`——此人建议升级处理器可能无法解决问题，因为问题可能由其他原因引起，比如软件过时。
