# --description--

동사 `vivir`는 "살다"를 의미합니다.

다른 사람("그"/"그녀")을 가리킬 때, 동사는 `vive` 형태로 활용됩니다. 보통 사람이 사는 곳을 나타내기 위해 전치사 `en`가 뒤따르며, 구조는 다음과 같습니다:

**`Vive en` + 장소**

줄리에타는 마테오의 질문에 답하며 발레리아가 어디에 사는지 알려주고 있습니다.
