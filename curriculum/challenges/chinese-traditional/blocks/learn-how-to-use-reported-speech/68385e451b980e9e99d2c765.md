# --explanation--

`True` 是一個形容詞，表示某事是正確的、真實的或準確的。它用來同意一個述語或事實。它不同於 `the truth`，後者是一個名詞，指的是關於某事的實際事實或現實，通常用於更廣泛或更抽象的意義。 例如：

- `It's true that the server crashed yesterday.`－這表示關於當機的述語是正確的。

- 「`The truth is that we didn't have a backup.`」－這表示實際情況是沒有備份存在。
