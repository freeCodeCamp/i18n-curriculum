# --explanation--

Використання `that` після `I think` є необов’язковим у реченнях такого типу. Також правильно сказати `I think that it's vital we make these updates.`. Однак у повсякденній розмові часто опускають `that` для більш плавного і швидкого висловлювання. Наприклад:

`I think that we need more time to finish the project.` – Ця структура правильна. Проте більшість людей сказали б `I think we need more time…`.
