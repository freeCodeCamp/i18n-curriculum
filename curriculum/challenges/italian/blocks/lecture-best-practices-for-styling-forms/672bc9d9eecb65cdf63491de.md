# --interactive--

Impariamo quando usare `appearance: none` per risolvere problemi di stile con input di ricerca, caselle di spunta e pulsanti di opzione.

I browser applicano uno stile predefinito a molti elementi. Nel caso degli elementi input, la possibilità di definirne lo stile con CSS può sembrare piuttosto limitata. Quindi, potresti voler usare `appearance: none` per nascondere aspetti dell'elemento predefinito e costruire il tuo stile personalizzato. Per esempio, questo nasconderebbe le caselle di spunta predefinite per un input checkbox permettendoti di usare indicatori personalizzati come un segno di spunta verde e una X rossa per mostrare lo stato. Per un input di ricerca, i browser basati su WebKit mostrano un'icona di ricerca predefinita e un pulsante di annullamento. Nascondere questi elementi ti permette di creare i tuoi indicatori che appariranno su tutti i browser.

**NOTA**: Alcuni CSS in questo esempio usano proprietà che non sono ancora state trattate. Questo serve solo a darti un'idea di come creare una casella di spunta personalizzata. Imparerai come funziona tutto questo nelle lezioni e workshop futuri.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form>
  <label>
    <input class="checkbox" type="checkbox" /> Agree
  </label>
</form>
```

```css
.checkbox {
  appearance: none;
  width: 18px;
  height: 18px;
  border: 2px solid #ccc;
  border-radius: 4px;
  display: inline-block;
  position: relative;
  cursor: pointer;
  transition: all 0.25s ease;
  vertical-align: middle; 
}

.checkbox:hover {
  border-color: #888;
}

.checkbox:checked {
  background-color: #4caf50;
  border-color: #4caf50;
}

.checkbox:checked::after {
  content: "";
  position: absolute;
  left: 4px;
  top: 0px;
  width: 5px;
  height: 10px;
  border: solid white;
  border-width: 0 2px 2px 0;
  transform: rotate(45deg);
}

.checkbox:focus {
  outline: 2px solid #90caf9;
  outline-offset: 2px;
}
```

:::

WebKit è un motore software che aiuta i browser web a mostrare i siti web. Browser come Safari usano WebKit per assicurarsi che le pagine web appaiano e funzionino correttamente. Questa proprietà CSS `appearance: none` ti dà il controllo completo sullo stile, ma ci sono alcune cose a cui fare attenzione. I componenti interattivi predefiniti degli elementi input includono funzionalità come indicatori di focus e di errore che devi assicurarti di non perdere.

Creare uno stile coerente su più piattaforme è un ottimo motivo per usare questa proprietà. Puoi anche usarla per garantire che gli obiettivi di tocco su dispositivi mobili siano abbastanza grandi o che i colori di una casella di spunta abbiano sufficiente contrasto.
