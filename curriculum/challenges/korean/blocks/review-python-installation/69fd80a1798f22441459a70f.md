## 로컬 환경에서의 파이썬

- **설치**: Windows, Mac, Linux에 파이썬을 설치하는 가장 좋은 방법은 공식 Python 웹사이트에서 설치 프로그램을 다운로드하는 것입니다(`https://www.python.org/`).
- **터미널**: 명령어를 입력해 컴퓨터와 상호작용할 수 있는 텍스트 기반 인터페이스입니다. 각 운영체제는 기본 터미널 앱을 제공합니다. macOS에서는 Terminal 앱을 사용할 수 있습니다. Windows에서는 Command Prompt 또는 PowerShell을 사용할 수 있습니다. Linux에서는 GNOME Terminal이나 Konsole 같은 각 데스크톱 환경별 기본 터미널 앱이 있습니다.
- **IDE**: IDE는 통합 개발 환경(Integrated Development Environment)의 약자로, 코드 편집기, 테스트 도구, 터미널 등 다양한 기능을 포함합니다.
- **파이썬용 인기 코드 편집기 및 IDE**: VS Code, PyCharm, Spyder
- **로컬에서 코드 실행하기**: 파이썬 코드를 실행하는 방법은 여러 가지가 있습니다. 첫 번째 방법은 VS Code에서 오른쪽 상단에 있는 재생 버튼 모양의 실행 버튼을 클릭하는 것입니다. 그러면 터미널이 열리고 파이썬 스크립트가 실행됩니다.

또 다른 방법은 터미널을 열어 직접 프로그램을 실행하는 것입니다. 예를 들어 `main.py`라는 프로그램을 실행하려면:

```bash
python main.py
```

이 명령은 `main.py`가 저장된 폴더에서 실행해야 합니다. 예를 들어 `main.py`가 `python-projects`라는 폴더 안에 있다면, 먼저 터미널에서 다음을 입력하세요:

```bash
cd python-projects
```

그다음 다음 명령을 실행하세요:

```bash
python main.py
```

## 파이썬 대화형 셸 사용하기

- **정의**: 대화형 셸은 명령어를 한 번에 하나씩 입력하고 결과를 바로 확인할 수 있는 프로그램입니다. 터미널 앱을 열고 `python`를 입력한 뒤 `Enter`를 누르세요. 그러면 파이썬 대화형 셸이 시작됩니다. `>>>` 기호는 파이썬이 명령어 입력을 기다리고 있다는 뜻입니다. 터미널에 `"Hello, world!"`를 출력해 보세요:

```bash
print("Hello, world!")
```

다음과 같이 텍스트가 나타납니다:

```bash
>>> print("Hello, world!")
Hello, world!
```

텍스트가 출력된 후 파이썬은 다시 다음 상태로 돌아갑니다:

```bash
>>>
```

이 상태에서는 또 다른 명령어를 입력할 수 있습니다.

- **읽기, 평가, 출력, 반복 주기(REPL)**: 명령어를 입력할 때마다 인터프리터가 입력을 읽고, 평가하며, 결과를 출력하고, 다시 `>>>`를 보여주어 추가 명령어를 입력할 수 있게 합니다.
