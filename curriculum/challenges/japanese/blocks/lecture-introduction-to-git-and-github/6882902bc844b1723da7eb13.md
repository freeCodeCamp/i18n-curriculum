# --description--

バージョン管理やGitHubのようなサイトを本格的に使い始める前に、Gitをインストールして使える状態にしておく必要があります。

Gitがすでにマシンにインストールされているかどうかを確認するには、ターミナルで次のコマンドを実行してください。

```sh
git --version
```

バージョン番号が表示されればGitがインストールされています。表示されなければ、インストールが必要です。

Linuxシステムでは、多くのディストリビューションにGitがあらかじめインストールされていることが多いです。もしGitがプリインストールされていなければ、`sudo apt-get install git`や`sudo pacman -S git`のようなパッケージマネージャーのコマンドでインストールできるはずです。

Macユーザーは、Homebrewを使って`brew install git`でGitをインストールできます。またはGitのウェブサイトから実行可能なインストーラーをダウンロードすることもできます。

Windowsの場合は、Gitのウェブサイトから実行可能なインストーラーをダウンロードできます。また、Chocolateyを設定している場合は、PowerShellで`choco install git.install`を実行できます。Windowsでは、Unix風のシェル環境を利用できるGitBashもダウンロードすることをお勧めします。

インストールが成功したか確認するには、ターミナルで再度`git --version`コマンドを実行してください。

Gitをインストールしたら、いくつか設定を変更する必要があります。これらの変更はコンピューターで一度だけ行えばよく、アップグレード時に繰り返し変更する必要はありません。

`git config`は、Gitがマシン上でどのように動作するかを決める設定変数を設定するために使います。現在の設定変数とそれがシステムのどこに保存されているかを確認するには、次のコマンドを実行してください。

```sh
git config --list --show-origin
```

Gitを初めてインストールした場合は、システムレベルの設定だけが表示されるはずです。すでにGitをインストールしている場合は、ユーザー名やメールアドレスのユーザー設定が表示されるかもしれません。

ユーザー名を設定するには、次のコマンドを実行してください。

```sh
git config --global user.name "Jane Doe"
```

ここで使われている`--global`フラグは、Gitを使うすべてのプロジェクトに対してユーザー名を設定します。特定のプロジェクトだけでユーザー名をオーバーライドしたい場合は、そのプロジェクトのディレクトリで`--global`フラグなしでコマンドを実行してください。

ユーザーのメールアドレスを設定するには、次のコマンドを実行してください。

```sh
git config --global user.email janedoe@example.com
```

ユーザー名とメールアドレスの設定は重要です。Gitはこれらの情報を使って、プロジェクトで行うすべてのコミットに記録します。コミットについては今後のレッスンで詳しく学びます。

もう一つ設定できるのは、Gitが使う好みのエディターです。例えば、好みのエディターをEmacsに設定する例は次の通りです。

```sh
git config --global core.editor emacs
```

他にはVim、Nano、VS Codeなどがあります。例えばVS Codeをエディターに使う場合は、次のようにGitの設定を行えます。

```sh
git config --global core.editor "code --wait"
```

好みのエディターを設定しなければ、Gitはシステムのデフォルトエディターを使います。

Windowsマシンでエディターを変更したい場合は、実行ファイルの完全なパスを指定する必要があります。例は次の通りです。

```sh
git config --global core.editor "'C:/Program Files/Notepad++/notepad++.exe' -multiInst -notabbar -nosession -noPlugin"
```

設定の一覧を確認するには、次のコマンドを実行してください。

```sh
git config --list
```

これでGitのインストールとセットアップが完了しました。これからプロジェクトにバージョン管理を追加する準備が整いました。
