# --description--

بما أنك تعرف الآن الطرق الأكثر شيوعًا للتكرار على القيم والمصفوفات، هناك بعض الطرق الإضافية للتكرار على المصفوفات. تحتوي المصفوفات على طريقة مدمجة تسمى `map()`، تُستخدم لإنشاء مصفوفة جديدة بتطبيق دالة على كل عنصر من عناصر المصفوفة الأصلية. طريقة `map()` لا تغيّر المصفوفة الأصلية. صيغة `map()` كالتالي:

```javascript
const array = [1, 2, 3, 4, 5];

const newArray = array.map((arrayValue) => {
  return arrayValue * 2;
});

console.log(newArray); // Output: [2, 4, 6, 8, 10]
```

تُنشئ طريقة `map()` مصفوفة جديدة بتطبيق الدالة `(arrayValue) => { return arrayValue * 2; }` على كل عنصر من عناصر المصفوفة الأصلية. هذا مفيد بشكل خاص عندما تريد تحويل عناصر المصفوفة دون تغيير المصفوفة الأصلية.
