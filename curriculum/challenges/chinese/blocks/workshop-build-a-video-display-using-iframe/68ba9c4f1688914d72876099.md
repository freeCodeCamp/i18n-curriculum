# --description--

最后但同样重要的是，你将添加的属性是 `allowfullscreen`。顾名思义，它允许视频以全屏模式观看。

至此，工作坊已完成！

# --hints--

你的 `iframe` 元素应具有 `allowfullscreen` 属性。

```js
const iframeEl = document.querySelector('iframe')
assert.exists(iframeEl?.getAttribute('allowfullscreen'))
```
