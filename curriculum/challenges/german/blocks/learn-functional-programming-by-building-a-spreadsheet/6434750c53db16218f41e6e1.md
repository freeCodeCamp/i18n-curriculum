# --description--

Denken Sie daran, dass `range()` ein Array zurückgibt, sodass Sie Array-Methoden direkt an den Funktionsaufruf anhängen können.

Rufen Sie `range()` mit `1` und `99` als Argumente auf und hängen Sie die `.forEach()`-Methode an. Übergeben Sie der `.forEach()`-Methode eine leere Callback-Funktion, die `number` als Parameter erhält.

# --hints--

Sie sollten Ihre `range()`-Funktion aufrufen.

```js
assert.lengthOf(code.match(/range\(/g), 2);
```

Sie sollten `1` als erstes Argument an Ihren `range()`-Aufruf übergeben.

```js
assert.match(code, /range\(\s*1/);
```

Sie sollten `99` als zweites Argument an Ihren `range()`-Aufruf übergeben.

```js
assert.match(code, /range\(\s*1\s*,\s*99\s*\)/);
```

Sie sollten die `.forEach()`-Methode an Ihren `range()`-Aufruf anhängen.

```js
assert.match(code, /range\(\s*1\s*,\s*99\s*\)\.forEach\(/);
```

Sie sollten eine Callback-Funktion mit Pfeilsyntax an `.forEach()` übergeben.

```js
assert.match(code, /range\(\s*1\s*,\s*99\s*\)\.forEach\(\s*(\([^)]*\)|[^\s()]+)\s*=>/);
```

Ihre Callback-Funktion sollte `number` als einzigen Parameter haben.

```js
assert.match(code, /range\(\s*1\s*,\s*99\s*\)\.forEach\(\s*(\(\s*number\s*\)|number)\s*=>/);
```
