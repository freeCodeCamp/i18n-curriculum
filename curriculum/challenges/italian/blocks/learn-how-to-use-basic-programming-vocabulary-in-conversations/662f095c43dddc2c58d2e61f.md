# --description--

La parola `super` viene spesso usata informalmente prima di un aggettivo per rafforzarne il significato, in modo simile a `very` o `extremely`, per potenziare il significato dell'aggettivo che segue.

Per esempio, dire `this book is super interesting` significa che il libro è estremamente interessante, più che semplicemente interessante.

`Helpful` descrive qualcuno o qualcosa che fornisce un aiuto o supporto prezioso. Insieme, `super helpful` significa fornire un'assistenza eccezionale o essere estremamente utile.
