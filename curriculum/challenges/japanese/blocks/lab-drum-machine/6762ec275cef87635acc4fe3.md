# --description--

**目的:** 以下のストーリーを満たし、すべてのテストに合格してラボを完了してください。

**ストーリー:**

1. `div` 要素があり、その `id` は `drum-machine` で、他のすべての要素を含んでいる必要があります。

2. `#drum-machine` 要素の中に、`div` が `id` の別の `pad-bank` がある必要があります。

3. `#drum-machine` 要素の中に、`p` が `id` の `display` 要素がある必要があります。

4. `#pad-bank` 要素の中に、クラスが `drum-pad` のクリック可能なドラムパッド要素が9つあり、それぞれにドラムパッドがトリガーするオーディオクリップを説明するユニークな `id` があり、キーボードの次のキーのいずれかに対応する内部テキストがある必要があります：`Q`、`W`、`E`、`A`、`S`、`D`、`Z`、`X`、`C`。ドラムパッドはこの順序でなければなりません。

5. 各 `.drum-pad` は、クラスが `audio` の `clip` 要素を持ち、オーディオクリップを指す `src` 属性と、その親 `id` 要素の内部テキストに対応する `.drum-pad` を持つ必要があります（例：`id="Q"`、`id="W"`、`id="E"` など）。

6. `.drum-pad` 要素をクリックすると、その子のオーディオ要素に含まれるオーディオクリップが再生される必要があります。

7. 各 `.drum-pad` に関連付けられたトリガーキーを押すと、その子のオーディオ要素に含まれるオーディオクリップが再生される必要があります（例：`Q` キーを押すと `Q` を含むドラムパッドがトリガーされ、`W` キーを押すと `W` を含むドラムパッドがトリガーされるなど）。

8. `.drum-pad` がトリガーされると、関連するオーディオクリップを説明する文字列を `#display` 要素の内部テキストとして表示する必要があります（各文字列はユニークでなければなりません）。

ドラムマシンで使用できるオーディオサンプルは `https://cdn.freecodecamp.org/curriculum/drum/<fileName>` にあり、`<fileName>` は以下の通りです：

| ドラム名 | `fileName` |
| --- | --- |
| Heater 1 | `Heater-1.mp3` |
| Heater 2 | `Heater-2.mp3` |
| Heater 3 | `Heater-3.mp3` |
| Heater 4 | `Heater-4_1.mp3` |
| Clap | `Heater-6.mp3` |
| Open-HH | `Dsc_Oh.mp3` |
| Kick-n'-Hat | `Kick_n_Hat.mp3` |
| Kick | `RP4_KICK_1.mp3` |
| Closed-HH | `Cev_H2.mp3` |

```js
const letters = [`Q`, `W`, `E`, `A`, `S`, `D`, `Z`, `X`, `C`];
const audios = document.querySelectorAll('#drum-machine .drum-pad audio');

for (let i = 0; i < 9; i++) {
  const audio = audios[i];
  document.dispatchEvent(
    new KeyboardEvent('keydown', { key: letters[i].toLowerCase() })
  );
  assert.isFalse(audio.paused);
  audio.pause();
}
```

```js
const display = document.getElementById('display');

const setDisplay = text => {
  state.display = text;
  display.innerText = text;
};

const adjustVolume = e => {
  if (state.power) {
    state.sliderVal = e.target.value;
    setDisplay(`Volume: ${Math.round(e.target.value * 100)}`);
    setTimeout(() => setDisplay(' '), 1000);
    document.querySelectorAll('.clip').forEach(sound => {
      sound.volume = state.sliderVal;
    });
  }
};

const activatePad = pad => {
  pad.classList.add('active');
  setTimeout(() => pad.classList.remove('active'), 100);
};

const playSound = (id, name) => {
  if (!state.power) return;

  const sound = document.getElementById(id);
  sound.currentTime = 0;
  sound.play();
  activatePad(sound.parentNode);
  setDisplay(name);
};

const renderPadBank = () => {
  const padBank = document.querySelector('#pad-bank');
  padBank.innerHTML = '';

  state.currentPadBank.forEach(drum => {
    const pad = document.createElement('button');
    pad.classList.add('drum-pad');
    pad.id = drum.id;
    pad.innerText = drum.keyTrigger;

    const audio = document.createElement('audio');
    audio.classList.add('clip');
    audio.id = drum.keyTrigger;
    audio.src = drum.url;
    pad.appendChild(audio);

    pad.addEventListener('click', () =>
      playSound(drum.keyTrigger, drum.id.replace(/-/g, ' '))
    );
    document.addEventListener('keydown', e => {
      if (e.key === drum.keyTrigger.toLowerCase())
        playSound(drum.keyTrigger, drum.id.replace(/-/g, ' '));
    });

    padBank.appendChild(pad);
  });
};

const powerControl = () => {
  state.power = !state.power;

    : 'Off';
```
