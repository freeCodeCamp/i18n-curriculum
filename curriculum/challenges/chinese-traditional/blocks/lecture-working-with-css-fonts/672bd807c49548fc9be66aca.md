# --description--

排版是以有效溝通為目的，安排字型的藝術與技術。它包含格式化文字和選擇適當的字型。在本課程中，你將學習使用排版的最佳實務，以提升可讀性、階層體系和存取性。 

讓我們從可讀性開始。你應該選擇清晰且簡單的字型，讓你的設計容易理解。這對於你網站的主要文字尤其重要。如果字型容易閱讀，使用者更有可能與你的內容互動。網頁開發中常用的字型有 Roboto、Open Sans、Poppins、Lato 和 Barlow。但有許多字型可供選擇，你一定能找到符合你需求的字型。

文字也應該足夠大，讓每個人都能閱讀。你應該嘗試保持行數相對較短。每行的具體字元數量實際上取決於你的專案。但保持行數較短會讓內容更容易閱讀。

你可以使用字型大小為標題、副標題、段落和其他元素創建視覺階層體系。例如，網頁上的主標題應該使用較大的字型，接著是使用較小字型大小的副標題。這將為階層體系中的每個元素賦予特定的字型大小，幫助使用者透過視覺瀏覽結構。

你最多應該使用兩到三種字型來創建視覺一致性。使用過多字型會使文字更難閱讀，並削弱你的品牌識別度。這也可能影響使用者體驗，因為會增加網站的載入時間。

一旦你選擇這兩種或三種字型，就在不同頁面和元素中持續使用它們。這些字型應該與你的網頁應用程式、產品或服務的整體設計保持一致。你也應該使用空白來創造視覺平衡並提升可讀性。

為了存取目的，你應該遵循網頁內容存取指引。這些指引設定了文字的建議對比度比率。較高的對比度比率使文字更容易閱讀，特別是對於視覺障礙者。

你也應該提供選項，可調整字型大小並盡可能避免使用複雜字型。一般規則是，應盡可能使用簡單字型。如果你的內容易於閱讀，使用者更有可能與之互動。關於排版的最後一個建議是，確保你的字型能在不同螢幕大小和裝置上正確顯示。

排版是一個強大的工具，用於創建美麗的設計和吸引使用者的體驗。透過理解排版的基本原理並應用這些最佳實踐，你可以創建有效的設計。
