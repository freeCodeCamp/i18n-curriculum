# --description--

A expressão `thanks for` é usada para mostrar gratidão por ações específicas ou ajuda recebida.A estrutura é comumente seguida por uma forma `-ing` de um verbo (um gerúndio) ou um substantivo para especificar pelo que os agradecimentos são feitos.

Exemplos:

`Thanks for helping me with my homework.`

`Thanks for the advice.`
