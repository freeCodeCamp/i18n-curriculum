# --description--

Esistono due modelli di colore principali: il modello <dfn>additivo</dfn> RGB (rosso, verde, blu) usato nei dispositivi elettronici, e il modello <dfn>sottrattivo</dfn> CMYK (ciano, magenta, giallo, nero) usato nella stampa.

In questo progetto lavorerai con il modello RGB. Questo significa che i colori partono dal nero e cambiano man mano che vengono introdotti diversi livelli di rosso, verde e blu. Un modo semplice per vedere questo è con la funzione CSS `rgb`.

Crea una nuova regola CSS che abbia come obiettivo la classe `container` e imposta la sua `background-color` su nero con `rgb(0, 0, 0)`.

# --hints--

Dovresti usare un selettore di classe per prendere di mira la classe `container`.

```js
assert(new __helpers.CSSHelp(document).getStyle('.container'));
```

La tua regola CSS `.container` dovrebbe avere una proprietà `background-color` impostata su `rgb(0, 0, 0)`.

```js
assert(new __helpers.CSSHelp(document).getStyle('.container')?.backgroundColor === 'rgb(0, 0, 0)');
```
