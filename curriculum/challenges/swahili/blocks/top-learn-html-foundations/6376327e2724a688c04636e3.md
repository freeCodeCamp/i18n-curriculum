# --description--

HTML na CSS ni lugha mbili zinazofanya kazi pamoja kuunda kila kitu unachokiona unapotazama kitu kwenye mtandao. HTML ni data ghafi ambayo ukurasa wa mtandao umejengwa nayo. Maneno yote, viungo, kadi, orodha, na vifungo vinaundwa kwa HTML. CSS ndiyo inayoongeza mtindo kwa vipengele hivyo rahisi. HTML huweka taarifa kwenye ukurasa wa mtandao, na CSS huweka taarifa hiyo mahali, huipa rangi, hubadilisha fonti, na kuifanya ionekane nzuri!

# --questions--
    
## --text--

Ni majukumu gani makuu ya HTML na CSS katika kuunda ukurasa wa mtandao?

## --answers--

HTML huunda muundo wa maudhui, wakati CSS huipa mtindo vipengele.

---

HTML huunda muundo wa kuona, wakati CSS huandaa maudhui.

---

CSS hudhibiti utendaji, wakati HTML hudhibiti mpangilio.
