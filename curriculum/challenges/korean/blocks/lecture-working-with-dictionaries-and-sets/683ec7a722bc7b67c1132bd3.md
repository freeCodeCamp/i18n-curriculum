# --description--

Python에서 사전은 키-값 쌍의 모음을 저장하는 내장 자료 구조입니다. 실제 사전에서 단어를 찾아 그에 해당하는 뜻을 찾는 것과 매우 비슷하게 작동합니다.

Python 사전에서는 키를 사용해 해당하는 값을 찾습니다. 고유한 키에 값을 연결해야 할 때 사전을 사용하세요. 키를 기반으로 값을 빠르게 찾아야 하거나 구조화된 데이터를 나타내야 할 때 유용합니다.

Python 사전의 일반적인 구문은 다음과 같습니다:

```python
dictionary = {
    key1: value1,
    key2: value2
}
```

먼저 사전을 담고 있는 변수를 찾습니다. 사전을 변수에 꼭 할당할 필요는 없지만, 메모리에 저장하고 나중에 코드에서 사용하려면 이렇게 하는 것이 일반적입니다.

그다음 중괄호가 나오는데, 중괄호는 중괄호라고도 합니다. 중괄호 안에는 키-값 쌍이 들어 있습니다.

각 키는 값과 연결되어 있어서 키를 사용해 그 값을 접근할 수 있습니다. 마지막 값을 제외한 각 값 뒤에는 서로 다른 키-값 쌍을 구분하기 위한 쉼표가 있습니다. 키는 사전 내에서 고유해야 하며, 변경 불가능한 데이터 타입이어야 합니다. 하지만 값은 중복될 수 있고 어떤 데이터 타입이든 될 수 있습니다.

여기 마르게리타 피자 레시피 정보를 저장하는 사전 예제가 있습니다:

```python
pizza = {
    'name': 'Margherita Pizza',
    'price': 8.9,
    'calories_per_slice': 250,
    'toppings': ['mozzarella', 'basil']
}
```

사전은 `pizza` 변수에 할당되어 있습니다. 네 개의 키-값 쌍이 있습니다: `name`, `price`, `calories_per_slice`, 그리고 `toppings`.

또 다른 방법으로는 `dict()` 생성자를 사용하는 것인데, 이 생성자는 키-값 쌍의 수열에서 사전을 만듭니다.

피자 예제에 해당하는 구문은 다음과 같습니다. `dict()` 생성자에 튜플 리스트를 인수로 전달합니다. 이 튜플들은 첫 번째 요소로 키를, 두 번째 요소로 값을 포함합니다.

```python
pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
```

키-값 쌍의 값을 접근하려면 대괄호 표기법으로 알려진 이 구문을 사용할 수 있습니다. 사전을 담고 있는 변수 이름 다음에 대괄호를 쓰고, 대괄호 안에 접근하려는 키를 넣습니다:

```python
dictionary[key]
```

피자 예제에서 `name`의 값을 접근하려면 변수 이름 `pizza` 다음에 대괄호를 쓰고, 따옴표 안에 키 `name`를 넣으면 됩니다:

```python
pizza['name']
```

이것은 다음과 같이 평가됩니다:

```python
'Margherita Pizza'
```

값을 업데이트하려면 대입 연산자를 추가하고 새 값을 할당하면 됩니다.

키가 사전에 없으면 새 키-값 쌍이 생성됩니다. 최신 Python 버전에서는 사전이 삽입 순서를 유지합니다. 이는 사전을 순회할 때 유용합니다:

```python
pizza['name'] = 'Margherita'
```

이제 키 `name`의 값은 `'Margherita'`입니다:

```python
print(pizza['name']) # 'Margherita'
```

사전에는 일반적인 작업을 수행하는 데 도움이 되는 메서드도 있습니다.

`.get()` 메서드는 키와 연결된 값을 가져옵니다. 방금 사용한 대괄호 표기법과 비슷하지만, 키가 없을 때 오류가 발생하지 않도록 기본값을 설정할 수 있다는 장점이 있습니다:

```python
dictionary.get(key, default)
```

이 예제에서 `toppings` 키가 없으면 두 번째 인수로 전달한 기본값인 빈 리스트를 반환합니다. 하지만 `toppings`가 있으면 그 값을 반환합니다:

```python
pizza.get('toppings', []) # ['mozzarella', 'basil']
```

`.keys()`와 `.values()` 메서드는 각각 사전의 모든 키와 값을 포함하는 뷰 객체를 반환합니다:

```python
pizza.keys()
# dict_keys(['name', 'price', 'calories_per_slice'])

pizza.values()
# dict_values(['Margherita Pizza', 8.9, 250])
```

뷰 객체는 데이터를 별도로 복사하지 않고 사전 내용을 볼 수 있는 방법일 뿐입니다.

`.items()` 메서드는 키와 값 모두를 포함하는 모든 키-값 쌍의 뷰 객체를 반환합니다:

```python
pizza.items()
# dict_items([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250)])
```

`.clear()` 메서드는 사전에서 모든 키-값 쌍을 제거합니다:

```python
pizza.clear()
```

`.pop()` 메서드는 첫 번째 인수로 지정한 키를 가진 키-값 쌍을 제거하고 그 값을 반환합니다. 키가 없으면 두 번째 인수로 지정한 기본값을 반환합니다. 기본값을 지정하지 않고 키가 없으면 `KeyError`가 발생합니다:

```python
pizza.pop('price', 10)
pizza.pop('total_price') # KeyError
```

Python 3.7 이상 버전에서는 `.popitem()` 메서드가 마지막에 삽입된 항목을 제거합니다:

```python
pizza.popitem()
```

마지막으로 `.update()` 메서드는 다른 사전의 키-값 쌍으로 사전을 업데이트합니다. 공통 키가 있으면 그 값이 교체됩니다.

이 예제에서는 `pizza` 사전을 업데이트합니다. `price` 키가 두 사전에 모두 있으므로 그 값이 `15`로 교체됩니다.

하지만 `total_time`는 새 키이므로 `pizza` 사전에 새 키-값 쌍으로 추가됩니다:

```python
pizza.update({ 'price': 15, 'total_time': 25 })
```

업데이트된 `price`와 새 `total_time`가 포함된 새 사전입니다. 가격이 이제 `15`이고 `total_time`가 새 키-값 쌍임을 확인하세요:

```python
{
    'name': 'Margherita Pizza', 
    'price': 15, 
    'calories_per_slice': 250, 
    'toppings': ['mozzarella', 'basil'], 
    'total_time': 25
}
```

이들은 가장 자주 사용되는 사전 메서드 중 일부지만, 더 많은 메서드가 있습니다. 적절한 메서드를 선택하면 복잡한 작업을 효율적으로 수행하는 데 도움이 됩니다.
