# --description--

`split` メソッドは文字列を文字列の配列に分割します。区切り文字として使う文字や正規表現を引数に取ります。例えば、区切り文字がスペースなら単語の配列が得られ、区切り文字が空文字列なら文字列の各文字の配列が得られます。

以下は、1つの文字列をスペースで分割し、もう1つを正規表現で数字で分割する例です。

```js
const str = "Hello World";
const bySpace = str.split(" ");

const otherString = "How9are7you2today";
const byDigits = otherString.split(/\d/);
```

`bySpace` は `["Hello", "World"]` の値を持ち、`byDigits` は `["How", "are", "you", "today"]` の値を持ちます。

文字列は不変なので、`split` メソッドを使うと扱いやすくなります。

# --instructions--

`split` メソッドを `splitify` 関数内で使い、`str` を単語の配列に分割してください。関数はその配列を返す必要があります。単語は必ずしもスペースで区切られているわけではなく、配列に句読点は含めないようにしてください。

# --hints--

コードは `split` メソッドを使う必要があります。

```js
assert(__helpers.removeJSComments(code).match(/\.split/g));
```

`splitify("Hello World,I-am code")` は `["Hello", "World", "I", "am", "code"]` を返すべきです。

```js
assert(
  JSON.stringify(splitify('Hello World,I-am code')) ===
    JSON.stringify(['Hello', 'World', 'I', 'am', 'code'])
);
```

`splitify("Earth-is-our home")` は `["Earth", "is", "our", "home"]` を返すべきです。

```js
assert(
  JSON.stringify(splitify('Earth-is-our home')) ===
    JSON.stringify(['Earth', 'is', 'our', 'home'])
);
```

`splitify("This.is.a-sentence")` は `["This", "is", "a", "sentence"]` を返すべきです。

```js
assert(
  JSON.stringify(splitify('This.is.a-sentence')) ===
    JSON.stringify(['This', 'is', 'a', 'sentence'])
);
```
