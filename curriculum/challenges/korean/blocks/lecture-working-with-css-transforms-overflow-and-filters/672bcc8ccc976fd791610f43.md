# --interactive--

CSS `transform` 속성은 웹페이지의 다른 요소 레이아웃에 영향을 주지 않고 요소의 시각적 표현을 수정할 수 있는 강력한 도구입니다. 이 속성을 사용하면 요소를 2D 또는 3D 공간에서 회전, 크기 조절, 기울이기, 이동(변환)하는 등 다양한 변형을 적용할 수 있습니다.

`transform` 속성은 요소의 좌표계에 수학적 변환을 적용하는 방식으로 작동합니다. 즉, 요소의 원래 위치와 문서 흐름을 유지하면서 모양과 위치를 조작할 수 있습니다.

일부 일반적인 변환 함수를 살펴보겠습니다. 다음은 박스 요소의 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
body {
  border: 2px solid black;
}

.box {
  width: 200px;
  height: 200px;
  background-color: red;
}
```

:::

`body`에 검은색 실선 테두리를 설정하여 `.box` 요소 안에 중첩된 `body` 요소를 볼 수 있게 했습니다.

`translate` 함수는 요소를 현재 위치에서 이동시킵니다. 다음은 `translate` 함수를 사용한 업데이트된 예입니다:  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
body {
  border: 2px solid black;
}

.box {
  width: 200px;
  height: 200px;
  background-color: red;
  transform: translate(50px, 100px);
}
```

:::

이 CSS 규칙은 `box` 클래스를 가진 요소를 원래 위치에서 오른쪽으로 50픽셀, 아래로 100픽셀 이동시킵니다.

`rotate` 함수는 고정된 점을 중심으로 요소를 회전시키며, 다음은 앞서 나온 `rotate` 요소에 `.box` 함수를 적용한 예입니다:  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: rotate(45deg);
}
```

:::

이것은 요소를 시계 방향으로 45도 회전시킵니다.

`scale` 함수는 요소의 크기를 변경할 수 있게 합니다. 다음은 예시입니다:  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: scale(1.5, 2);
}
```

:::

이것은 요소를 원래 크기보다 가로로 1.5배, 세로로 2배 크게 만듭니다.

여러 변환을 하나의 선언에 결합할 수도 있습니다:  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: translate(50px, 50px) rotate(45deg) scale(1.5);
}
```

:::

이것은 요소를 오른쪽과 아래로 50픽셀 이동시키고, 45도 회전시키며, 원래 크기의 1.5배로 크기 조절합니다.

`transform` 속성은 시각적으로 매력적인 디자인을 만드는 데 강력하지만, 사용할 때 접근성을 고려하는 것이 중요합니다. 다음은 기억해야 할 주요 접근성 문제입니다.

화면 읽기 프로그램은 변환된 내용을 정확히 전달하지 못할 수 있습니다. 예를 들어, `transform`을 사용해 요소의 시각적 순서를 재배치해도 화면 읽기 프로그램은 여전히 원래 DOM 순서대로 내용을 읽습니다. 이는 화면 읽기 프로그램에 의존하는 사용자에게 혼란을 줄 수 있습니다.

`scale`을 사용해 텍스트 크기를 조절할 때 너무 작거나 너무 크게 만들지 않도록 주의하세요. 지나치게 작은 텍스트는 읽기 어렵고, 너무 큰 텍스트는 컨테이너를 넘쳐 읽기 어려워질 수 있습니다. 텍스트 크기 조절에는 적절한 폰트 스타일링 기법을 사용하는 것이 일반적으로 더 좋습니다.

`transform`을 애니메이션 효과에 사용할 때는 움직임에 민감한 사용자를 고려하세요. 과도하거나 빠른 애니메이션은 불편함을 주거나 일부 사용자에게 발작을 유발할 수 있습니다. 애니메이션을 줄이거나 끌 수 있는 방법을 제공하는 것을 고려하세요. 3D 변환을 사용할 때는 모든 사용자가 깊이를 동일하게 인지하지 않는다는 점을 기억하세요. 3D 효과로 전달되는 중요한 정보는 2D 형식이나 텍스트로도 제공되어야 합니다.

`transform`을 사용해 콘텐츠를 숨기거나 표시할 때는 콘텐츠가 화면 읽기 프로그램과 키보드 탐색에서 여전히 접근 가능하도록 하세요. 숨겨진 콘텐츠는 단순히 시각적으로 화면 밖으로 이동시키는 것이 아니라 `display: none;` 또는 `visibility: hidden;` 같은 방법으로 실제로 숨겨져야 합니다.

`transform`을 버튼이나 링크 같은 인터랙티브 요소에 적용할 때는 클릭 가능한 영역이 직관적이고 쉽게 타겟팅될 수 있도록 하세요. 크게 변형된 버튼은 시각적으로 혼란스럽거나 운동 능력이 제한된 사용자에게 클릭하기 어려울 수 있습니다.

결론적으로, CSS `transform` 속성은 시각적으로 역동적인 웹 디자인을 만드는 데 강력한 도구지만, 접근성을 염두에 두고 책임감 있게 사용하는 것이 필수입니다. 다양한 보조 기술로 변환된 요소를 항상 테스트하고, 변환으로 영향을 받을 수 있는 정보나 기능에 대한 대체 접근 방법을 제공하는 것을 고려하세요.
