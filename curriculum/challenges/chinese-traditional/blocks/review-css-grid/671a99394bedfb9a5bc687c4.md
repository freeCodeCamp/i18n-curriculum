## CSS Grid 基礎

- **定義**：CSS Grid 是一種用於在網頁中創建複雜版面配置的二維版面系統。格線會有列和行，且它們之間會有間隙。若要定義格線版面配置，你需要將 `display` 屬性設為 `grid`。

```css
.container {
  display: grid;
}
```

- **`fr`（分數）單位**：此單位表述網格容器內空間的一部分。你可以使用 `fr` 單位來創建彈性網格。
- **在軌道之間創建間隙**：在 CSS grid 中有三種方法可以在軌道之間創建間隙。你可以使用 `column-gap` 屬性在行之間創建間隙。你可以使用 `row-gap` 屬性在列之間創建間隙。或者你可以使用 `gap` 簡寫屬性在列和行之間同時創建間隙。
- **`grid-template-columns`**：用於設定網格軌道行的線名稱和大小。

```css
.container {
  display: grid;
  width: 100%;
  grid-template-columns: 30px 1fr;
}
```

- **`grid-template-rows`**：用於設定網格軌道列的線名稱和大小。
- **`grid-auto-flow`**：這決定自動放置的項目如何適應網格。

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-flow: column;
}
```

- **`grid-auto-columns`**：這用於設定隱喻的創建行的大小。

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-columns: auto;
}
```

- **`place-items`**：這用於同時對區塊和行內方向的項目進行對齊。
- **`align-items`**：這用於設定網格容器中項目的對齊方式。
- **`repeat()` 函式（程式）**：此函式用於重複軌道列表中的區段。你可以使用 `repeat()` 函式來取代撰寫 `grid-template-columns: 1fr 1fr 1fr;`。

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}
```

- **明確的網格**：你可以使用 `grid-template-columns` 或 `grid-template-rows` 屬性來指定行數或軌道數。
- **隱式網格**：當項目被放置在網格外時，系統會自動為那些外部元素創建列和行。
- **`minmax()` 函式（程式）**：此函式用於設定軌道的最小和最大尺寸。

```css
.container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-auto-rows: minmax(150px, auto);
}
```

- **基於行的配置**：所有網格都有行。若要指定條款在某行的開始位置，你可以使用 `grid-column-start` 和 `grid-row-start` 屬性。若要指定條款在該行的結束位置，你可以使用 `grid-column-end` 和 `grid-row-end` 屬性。你也可以選擇使用 `grid-column` 或 `grid-row` 簡寫屬性。

這裡是一個使用 `grid-column` 屬性讓元素跨越所有行的範例。

```css
.element {
  grid-column: 1 / -1;
}
```

- **`grid-template-areas`**：該屬性用於為你將要在網格上定位的項目命名。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="header">Header</div>
  <div class="sidebar">Sidebar</div>
  <div class="main">Main Content</div>
  <div class="footer">Footer</div>
</div>

```

```css
.container {
  display: grid;
  grid-template-columns: 200px 1fr; 
  grid-template-rows: auto 1fr auto; 
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer"; 
  gap: 20px; 
}

.header {
  grid-area: header; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}

.sidebar {
  grid-area: sidebar;
  background-color: #f4f4f4;
  padding: 10px;
}

.main {
  grid-area: main; 
  background-color: #e0e0e0;
  padding: 10px;
}

.footer {
  grid-area: footer; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}
```

:::

## CSS 偵錯

- **DevTools (Developer Tools)**：DevTools 允許你即時檢查和修改你的 CSS。Styles 面板顯示套用到所選元素的所有 CSS 規則，包括繼承的樣式。你可以切換單一屬性的開啟與關閉、編輯數值，甚至直接在瀏覽器中新增規則。這種即時回授對於在不修改你的原始碼的情況下嘗試不同樣式非常有用。
- **CSS 驗證器**：驗證器用來檢查你的 CSS 是否符合官方規範，並回報任何錯誤或警告。你可以使用的一個熱門驗證器是 W3C CSS Validator。
- **偵錯響應式設計**：DevTools 有一個選項，可讓你模擬網站在各種螢幕尺寸和裝置上的顯示方式。這可以幫助你識別中斷點問題或在不同視窗大小下無法良好縮放的樣式。
