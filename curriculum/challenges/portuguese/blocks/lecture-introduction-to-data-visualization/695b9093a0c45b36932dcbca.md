# --description--

Existem muitas maneiras de representar dados em um formato visual. Nesta lição, você aprenderá sobre alguns tipos de visualizações de dados comumente usados.

O primeiro tipo que você aprenderá são gráficos que representam mudanças ao longo do tempo, como desempenho do preço das ações ou estatísticas de saúde.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-are-common-chart-types-and-when-should-you-use-them-1.png" alt="example area chart">

Outro tipo comum de visual seria um gráfico de comparação de categorias. Esse tipo de visual mostra comparações entre diferentes categorias, como desempenho de anúncios em canais de marketing ou vendas por categoria de produto para lojas online.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-are-common-chart-types-and-when-should-you-use-them-2.png" alt="example bubble chart for marketing channel performance leads">

Se você precisasse representar a posição de um item em uma lista ordenada, poderia usar um gráfico de classificação. Exemplos desse tipo de visual incluiriam estatísticas de desempenho ou resultados de eleições.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-are-common-chart-types-and-when-should-you-use-them-3.png" alt="example ranking chart for election results">

Outro tipo de visualização de dados comumente usado seria uma parte do todo visual. Isso incluiria gráficos de pizza ou treemaps que poderiam ser usados para representar orçamentos ou categorias de produtos.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-are-common-chart-types-and-when-should-you-use-them-4.png" alt="example pie chart for budget">

Se você está trabalhando com dados que mostram uma correlação entre duas ou mais variáveis, pode usar um gráfico de correlação. Esses tipos de gráficos incluem mapas de calor ou scatterplots que podem representar dados como renda vs. nível de educação, gasto com publicidade vs. vendas ou temperatura vs. consumo de energia.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-are-common-chart-types-and-when-should-you-use-them-5.png" alt="example scatterplot for income vs. education level">

Existem muitos outros tipos de gráficos como fluxogramas, gráficos de distribuição, gráficos de relacionamento e mais. Mas esta lição foi sobre fornecer uma breve introdução a alguns dos tipos de gráficos que você pode considerar ao representar dados complexos.
