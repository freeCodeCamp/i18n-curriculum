## 列表样式

- **`line-height` 属性**：此属性用于创建文本行之间的间距。接受的 `line-height` 值包括关键字 `normal`、数字、百分比以及像 `em` 单位这样的长度单位。
- **`list-style-type` 属性**：此属性用于指定列表项的标记。可接受的值包括 circle、disc 或 decimal。

这是一个使用 `list-style-type` 来更改项目符号样式的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
ul {
  list-style-type: square;
}
```

:::

- **`list-style-position` 属性**：此属性用于设置列表标记的位置。唯一可接受的两个值是 inside 和 outside。

下面是一个展示 `inside` 和 `outside` 区别的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="inside-list">
  <li>Item with inside position</li>
  <li>Item with inside position</li>
</ul>
<ul class="outside-list">
  <li>Item with outside position</li>
  <li>Item with outside position</li>
</ul>
```

```css
.inside-list {
  list-style-position: inside;
}

.outside-list {
  list-style-position: outside;
}
```

:::

- **`list-style-image` 属性**：此属性用于为列表项标记使用图像。一个常见的用例是使用 `url` 函数，其值设置为有效的图像位置。

## 使用 `margin` 调整列表项间距

- 除了 `line-height`，还可以在 CSS 中使用边距来增强列表项的间距和可读性。
- 边距在每个 `li` 元素外部创建空间，从而允许控件列表项之间的间隙。
- `margin-bottom` 用于在每个列表项下方创建空间。例如，`margin-bottom: 10px;` 会在每个列表项下方创建 10 像素的间隙。

这是一个使用 `margin-bottom` 来为列表项添加间距的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="list">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
.list li {
  margin-bottom: 20px;
}
```

:::

## 样式化 `Links`

- **`pseudo-class`**：这是添加到选择器中的关键字，允许你根据特定状态选择元素。常见状态包括 `:hover`、`:visited` 和 `:focus` 状态。
- **`:link 伪类`**：此伪类用于为用户未访问的链接设置样式。

这是一个使用 `:link` 伪类的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:link {
  color: red;
}
```

:::

- **`:visited pseudo-class`**：此伪类用于为用户已访问的链接设置样式。
- **`:hover 伪类`**：此伪类用于为用户正在悬停的元素设置样式。

这是一个使用 `:hover` 伪类的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Hover over me!</a>
```

```css
a:hover {
  color: green;
  text-decoration: underline;
}
```

:::

- **`:focus 伪类`**：当元素获得焦点时，使用此伪类来设置样式。示例包括通过点击或按 Tab 键使 `input` 或 `select` 元素获得焦点。

这是一个使用 `:focus` 伪类的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:focus {
  outline: 2px solid orange;
}
```

:::

- **`:active pseudo-class`**：此伪类用于为被用户激活的元素设置样式。一个常见的例子是用户点击按钮时。

这是一个使用 `:active` 伪类的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Click me!</a>
```

```css
a:active {
  color: pink;
}
```

:::

## 处理背景和边框

- **`background-size` 属性**：此属性用于设置元素的背景大小。一些常见的值包括 `cover`，用于使背景图像覆盖整个元素，以及 `contain`，用于使背景图像适应元素内部。

这是一个使用 `background-size: contain` 的示例：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **`background-repeat` 属性**：此属性用于确定背景图像应如何沿水平和垂直轴重复。`background-repeat` 的默认值是 `repeat`，意味着图像将在水平和垂直方向重复。你也可以使用 `no-repeat` 属性指定不重复。

这是一个使用 `background-repeat: no-repeat` 的示例：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **`background-position` 属性**：此属性用于指定背景图像的位置。它可以设置为特定长度、百分比或关键字值，如 `top`、`bottom`、`left`、`right` 和 `center`。

这是一个使用 `background-position` 的示例：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

- **`background-attachment` 属性**：此属性用于指定背景图像是随内容滚动还是固定不动。主要取值有 `scroll`（默认），背景图像随内容滚动；以及 `fixed`，背景图像保持在屏幕上的相同位置。
- **`background-image` 属性**：此属性用于设置元素的背景图像。你可以同时设置多个背景图像，并使用 `url`、`radial-gradient` 或 `linear-gradient` 函数作为值。

这是一个使用 `background-image` 的示例：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **`background` 属性**：这是用于在一个声明中设置所有背景属性的简写属性。以下是设置背景图像并设置为不重复的示例：`background: no-repeat url("example-url-goes-here");`

这是一个使用 `background` 简写属性的示例：

:::interactive_editor

```html
<style>
  body {
    background: center top no-repeat url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **背景色和前景色的良好对比度**：确保背景色和前景色具有良好的对比度对文字的可读性非常重要。 《网页内容无障碍指南》（WCAG）建议，普通文字的对比度最低为 4.5:1，大文字的对比度最低为 3:1。

## 边框

- **`border-top` 属性**：此属性用于设置元素顶部边框的样式。`border-top: 3px solid blue;` 设置元素顶部边框为 3 像素宽的实线蓝色边框。
- **`border-right` 属性**：此属性用于设置元素右侧边框的样式。`border-right: 2px solid red;` 会在元素的右侧设置一个宽度为 2 像素的实线红色边框。
- **`border-bottom` 属性**：此属性用于设置元素底部边框的样式。`border-bottom: 1px dashed green;` 在元素底部设置了一个宽度为 1 像素的绿色虚线边框。
- **`border-left` 属性**：此属性用于设置元素左侧边框的样式。`border-left: 4px dotted orange;` 将元素左侧边框设置为 4 像素宽的橙色点状边框。

这是一个使用单独边框属性的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="bordered-box">Box with different borders</div>
```

```css
.bordered-box {
  border-top: 3px solid blue;
  border-right: 2px solid red;
  border-bottom: 1px dashed green;
  border-left: 4px dotted orange;
  padding: 20px;
}
```

:::

- **`border` 属性**：这是设置元素边框的宽度、样式和颜色的简写属性。`border: 1px solid black;` 设置一个宽度为 1 像素的实线黑色边框。

这是一个使用 `border` 简写属性的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

- **`border-radius` 属性**：此属性用于为元素的边框创建圆角。

这是一个使用 `border-radius` 的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

- **`border-style` 属性**：此属性用于设置元素的边框样式。一些可接受的值包括 `solid`、`dashed`、`dotted` 和 `double`。

这是一个使用不同 `border-style` 值的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="solid-border">Solid border</div>
<div class="dashed-border">Dashed border</div>
<div class="dotted-border">Dotted border</div>
```

```css
.solid-border {
  border: 3px solid blue;
  margin-bottom: 10px;
  padding: 10px;
}

.dashed-border {
  border: 3px dashed red;
  margin-bottom: 10px;
  padding: 10px;
}

.dotted-border {
  border: 3px dotted green;
  padding: 10px;
}
```

:::

## 梯度

- **`linear-gradient()` 函数**：此 CSS 函数用于创建沿直线的多种颜色过渡。

这是一个使用 `linear-gradient()` 的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

- **`radial-gradient()` 函数**：此 CSS 函数创建一个从特定点辐射的图像，类似圆形或椭圆形，并在多种颜色之间逐渐过渡。

这是一个使用 `radial-gradient()` 的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::
