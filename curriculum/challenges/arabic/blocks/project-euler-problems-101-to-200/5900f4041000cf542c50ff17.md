# --description--

هناك عدة طرق لكتابة العدد $\frac{1}{2}$ كمجموع لمربعات مقلوبة باستخدام أعداد صحيحة مميزة.

على سبيل المثال، يمكن استخدام الأعداد {2,3,4,5,7,12,15,20,28,35}:

$$\frac{1}{2} = \frac{1}{2^2} + \frac{1}{3^2} + \frac{1}{4^2} + \frac{1}{5^2} + \frac{1}{7^2} + \frac{1}{{12}^2} + \frac{1}{{15}^2} + \frac{1}{{20}^2} + \frac{1}{{28}^2} + \frac{1}{{35}^2}$$

في الواقع، باستخدام الأعداد الصحيحة فقط بين 2 و45 شاملة، هناك ثلاث طرق بالضبط للقيام بذلك، والطريقتان المتبقيتان هما: {2,3,4,6,7,9,10,20,28,35,36,45} و{2,3,4,6,7,9,12,15,28,30,35,36,45}.

كم عدد الطرق لكتابة العدد $\frac{1}{2}$ كمجموع لمربعات مقلوبة باستخدام أعداد صحيحة مميزة بين 2 و80 شاملة؟
