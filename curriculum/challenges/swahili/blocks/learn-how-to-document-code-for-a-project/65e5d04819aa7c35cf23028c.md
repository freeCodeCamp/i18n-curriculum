# --description--

Neno `so` ni kiunganishi kinachotumika kuonyesha sababu na matokeo au kueleza sababu ya jambo fulani. Katika teknolojia na programu, `so` mara nyingi huunganisha tatizo au hali na suluhisho au matokeo.

# --questions--

## --text--

Chagua sentensi ambayo `so` imetumika kwa usahihi kuunganisha sababu na matokeo katika muktadha wa teknolojia.

## --answers--

`The program crashed, so we must reboot the computer.`

---

`We wrote the code, so but it didn't compile.`

### --feedback--

Neno `so` limetumika vibaya hapa. `But` haipaswi kutumika baada ya `so`. Inapaswa kuwa `The code didn't compile, so we rewrote it`, au `We wrote the code, but it didn't compile.`

---

`The server is online, so therefore we can't connect.`

### --feedback--

Matumizi ya `so` na `therefore` pamoja ni ya ziada. Inapaswa kuwa `The server is online, so we can connect` au `The server is online; therefore, we can connect.`

---

`We need more data, so we can ignore the results.`

### --feedback--

Sentensi hii haikuunganishi sababu na matokeo kwa mantiki. Matumizi bora ya `so` yangekuwa, `We need more data, so we will perform more tests.`
