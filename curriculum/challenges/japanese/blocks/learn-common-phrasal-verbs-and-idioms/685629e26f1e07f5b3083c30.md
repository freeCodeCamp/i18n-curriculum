# --explanation--

`Real-time alerts` は何かが起こったときにすぐに表示されるメッセージや警告です。例えば：

`The app sends real-time alerts if someone logs in from a new location.` – これは誰かが別の場所からログインするたびに、ユーザーに即座に警告メッセージが届くことを意味します。

`Graphics` は情報を伝えるために使われる画像や視覚的要素です。例えば：

`The dashboard uses graphics to show project progress.` – これはチャートやアイコンのような視覚的ツールを使ってデータを理解しやすくすることを意味します。
