# --description--

Le funzioni freccia sono un modo più moderno di scrivere funzioni in JavaScript. Offrono una sintassi più compatta rispetto alle espressioni di funzione tradizionali. Le funzioni freccia si definiscono usando la sintassi `=>`.

```js
const add = (a, b) => {
  return a + b;
}
```

Nell'esempio sopra, la funzione freccia `add` prende due parametri `a` e `b` e restituisce la loro somma. La parola chiave `return` viene usata per restituire il risultato dell'operazione di addizione.

Se la funzione freccia ha solo una espressione, le parentesi graffe `{}` e la parola chiave `return` possono essere omesse. L'espressione verrà restituita implicitamente.

```js
const add = (a, b) => a + b;
```

Nell'esempio sopra, la funzione freccia `add` prende due parametri `a` e `b` e restituisce la loro somma. La parola chiave `return` e le parentesi graffe `{}` sono omesse, e il risultato dell'operazione di addizione viene restituito implicitamente.
