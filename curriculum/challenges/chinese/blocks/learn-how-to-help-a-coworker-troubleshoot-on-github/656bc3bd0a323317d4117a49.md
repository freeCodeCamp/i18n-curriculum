# --description--

过去进行时用于描述过去正在进行的动作。 

它是由代词 `I`、`he`、`she`、`it` 后接 `was`，或代词 `you`、`we`、`they` 后接 `were`，再加上以 `-ing` 结尾的动词构成。例如：

`I was reading` 意味着工作在过去的一段时间内正在进行。 

过去进行时常用于一个动作被另一个动作中止的情况，如 `I was reading when the phone rang.` 读取被电话铃声中止。
