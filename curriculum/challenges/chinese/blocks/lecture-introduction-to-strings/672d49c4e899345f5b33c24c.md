# --interactive--

之前的课程向你介绍了 `console.log()`，但本课将深入探讨它的用途和用法。

在 JavaScript 中，`console.log()` 是一个简单而强大的工具，用于向浏览器控制台显示消息或输出信息。 开发人员在开发程序时，大多使用它来调试和检查代码。

你可以使用 `console.log()` 将文本或变量输出到控制台，确保代码运行正确。

要使用 `console.log()`，你需要调用该方法，并在括号内输入要输出的值或信息。 这是一些例子：

:::interactive_editor

```js
console.log("Hello, world!");

let num = 5;
console.log(num); // 5
```

:::

第一个示例在浏览器控制台中打印 `Hello, world!` ，而第二个示例打印值 `5`。

下面是另一个使用 `console.log()` 的示例：

:::interactive_editor

```js
let name = "Alice";
console.log("Hello, " + name + "!"); // Hello, Alice!
```

:::

你也可以将多值用逗号分隔传递给 `console.log()`。例如：

:::interactive_editor

```js
let name = "Alice";
let age = 25;
console.log("Name:", name, "Age:", age); // Name: Alice Age: 25
```

:::

这对于一次记录多条信息非常有帮助。

`console.log()` 方法可帮助你在代码运行时对其进行监控，从而更容易发现错误并了解程序的运行情况。
