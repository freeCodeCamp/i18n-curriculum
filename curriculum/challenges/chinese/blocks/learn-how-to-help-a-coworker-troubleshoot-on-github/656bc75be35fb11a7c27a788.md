# --description--

短语 `Got it` 通常在对话中用来表示某人已经理解了某事。这是一种随意且快速的方式，用来确认你已经接收并理解了该信息。例如：

Lucy：`You need to finish the report by tomorrow。`
Bob：`Got it！` 

`Got it` 意味着 Bob 理解了指令。
