# --description--

تعليقات HTML غير مرئية للمتصفح؛ تسمح لنا بالتعليق على شفرتك حتى يتمكن المطوِّرون الآخرون أو أنت في المستقبل من قراءتها وفهم سياق شيء قد لا يكون واضحًا في الشفرة.

كتابة تعليق HTML بسيطة: فقط ضع التعليق بين وسمَي `<!--` و `-->`. على سبيل المثال:

```html
<h1> View the html to see the hidden comments </h1>

<!-- I am a html comment -->

<p>Some paragraph text</p>

<!-- I am another html comment -->
```
