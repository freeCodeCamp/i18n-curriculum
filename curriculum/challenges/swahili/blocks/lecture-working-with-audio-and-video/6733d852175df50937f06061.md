# --description--

Badala ya kuonyesha sauti na video, wakati mwingine unaweza kutaka kukamata sauti au video. API ya Media Capture and Streams, au API ya MediaStream, inakuwezesha kufanya hivyo.

Hata hivyo, ili kutumia API hii, unahitaji kuunda kitu cha `MediaStream`. Unaweza kufanya hivyo kwa kutumia kijenzi, lakini kingehusishwa na vifaa vya mtumizi. Badala yake, sifa ya `mediaDevices` ya kitu cha `navigator` cha ulimwengu ina njia ya `getUserMedia()` ambayo unaweza kutumia.

Njia hii inakubali kitu kimoja cha `constraints` kinachoelezea aina ya vyombo vya habari unavyotaka kupokea. Kitu hiki kina sifa za `audio` na `video` zinazowakilisha mito ya sauti na video. Sifa hizi zinaweza kuwa `false` ikiwa hutaki kupokea aina hiyo ya mto, `true` ikiwa unataka, au vitu vinavyoelezea vizingiti vya ziada.

Kwa mfano, unaweza kuhitaji azimio maalum la video:

```js
window.navigator.mediaDevices.getUserMedia({
  audio: true,
  video: {
    width: {
      min: 1280,
      ideal: 1920,
      max: 3840
    },
    height: {
      min: 720,
      ideal: 1080,
      max: 2160
    }
  }
});
```

Kitu hiki cha vizingiti kinaelezea azimio cha kiwango cha chini na cha juu kwa mtiririko wa video. Sifa ya `ideal` inaelezea azimio unalotaka zaidi – na mto utatoa azimio linalokaribia zaidi azimio lako bora.

Mara tu unapounda MediaStream yako (ikiwa mtumizi atakubali ombi la moja kwa moja la upatikanaji wa vifaa vyao), unaweza kutumia data ya mto kwa njia yoyote unayohitaji.

Kumbuka kwamba `getUserMedia()` inarudisha Ahadi, ambayo ina maana utahitaji kitendakazi cha mwitiko ili kutumia mto huo, au utumie sintaksia ya async/await. Utajifunza zaidi kuhusu Ahadi na programu isiyo ya wakati mmoja katika mafundisho yajayo.

Hapa kuna mfano wa msingi unaoonyesha mtiririko wa kamera ya mtumizi kwenye ukurasa.

```js
const video = document.querySelector("video");
const stream =
  await window.navigator.mediaDevices.getUserMedia({ video: true });
video.srcObject = stream;
await video.play();
```

Inafaa kutambua kuwa API hii haitozi upatikanaji wa kukamata skrini. Utajifunza kuhusu API hiyo katika funzo lijalo.
