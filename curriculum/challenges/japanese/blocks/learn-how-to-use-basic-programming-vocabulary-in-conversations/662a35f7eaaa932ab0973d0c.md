# --description--

プログラミングで`variables`、`data types`、`conditional statements`について学びましょう。

`variable`は情報を保存できる箱のようなものです。この情報は数字、テキスト、または他の種類のデータである場合があります。箱の中身は変えることができるので、`variable`と呼ばれます。例えば、`x = 5`では`x`が一時的に値5を保持しています。

`data type`は`variable`が保持できるデータの種類を指定する分類です。一般的な`data types`には整数（整数）、浮動小数点数（小数を含む数）、文字列（テキスト）、ブール値（真または偽）があります。例えば、`age = 21`の場合、`data type`は整数です。

`conditional statement`は特定の条件が真の場合にのみコンピューターに何かを実行させる命令です。これは現実の状況に基づいて決定を下すことに似ています。一般的な`conditional statements`は`if-else`文です。例えば、`if (temperature > 20) {console.log("It's warm outside!");}`は温度が20度を超えた場合に`conditional statement`を実行する`console.log("It's warm outside!")`です。
