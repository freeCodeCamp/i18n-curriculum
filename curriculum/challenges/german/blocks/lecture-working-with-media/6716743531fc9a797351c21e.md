# --interactive--

Ein ersetztes Element ist ein Element, dessen Inhalt von einer externen Ressource bestimmt wird und nicht direkt durch CSS. CSS, oder Cascading Style Sheets, wird verwendet, um einer Webseite Stil hinzuzufügen. Häufige Beispiele für ersetzte Elemente sind die Elemente image, iframe und video.

Bei ersetzten Elementen können Sie die Position oder das Layout eines Elements steuern. Ihr CSS kann jedoch den Inhalt dieses Elements nicht direkt verändern. Das lässt sich mit einigen Beispielen leichter erklären. Betrachten Sie das image-Element, das ein Bild auf Ihrer Webseite einbettet:

```html
<img src="example-img-url" alt="Descriptive text goes here">
```

Das Element selbst wird durch das externe Objekt ersetzt: das Bild. Ihr CSS kann Dinge wie die Positionierung des Bildes steuern oder einen Filter darauf anwenden, aber Sie können das Bild selbst nicht verändern. Ein robusteres Beispiel ist das `iframe`-Element, das eine externe Webseite auf Ihrer Webseite einbettet.

Hier ist ein Beispiel für die Verwendung des `iframe`-Elements für ein beliebtes YouTube-Video auf dem freeCodeCamp-Kanal. Wenn Sie andere Videos im Vorschaufenster sehen möchten, ändern Sie den Wert des `src`-Attributs zu einem Video Ihrer Wahl. Um die Vorschauen zu sehen, müssen Sie den interaktiven Editor aktivieren.

**HINWEIS**: Machen Sie sich keine Sorgen über die zusätzlichen Attribute wie `referrerpolicy` und `allowfullscreen` im interaktiven Beispiel. Sie lernen später in einem Workshop mehr über die Arbeit mit `iframe`-Elementen.

:::interactive_editor

```html
<iframe width="400" height="200" src="https://www.youtube.com/embed/u43gJJrVa1I?si=BoDW_puFsy8OEr_Z" title="Professional Cloud Architect Certification Course – Pass the Exam! (YouTube video)" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen></iframe>
```

:::

Weitere häufige Beispiele für die Verwendung des `iframe`-Elements sind das Einbetten von Karten auf der Seite. Hier ist ein Beispiel für eine eingebettete Karte.

Aktivieren Sie den interaktiven Editor und probieren Sie aus, mit der Karte selbst zu zoomen.

:::interactive_editor

```html
<iframe
  title="Map of the Royal Observatory, Greenwich, London"
  width="300"
  height="200"
  src="https://www.openstreetmap.org/export/embed.html?bbox=-0.004017949104309083%2C51.47612752641776%2C0.00030577182769775396%2C51.478569861898606&amp;layer=mapnik">
</iframe>
```

:::

Das Element selbst wird durch das externe Objekt ersetzt: die Webseite. Ihr CSS kann die Positionierung der eingebetteten Webseite ändern, aber Sie können den Inhalt der Webseite nicht verändern. Wenn die eingebettete Webseite ein `h1`-Element enthält, könnte Ihr CSS dieses `h1`-Element nicht gestalten. Sie können Größe, Schriftfarbe und so weiter nicht ändern.

Es gibt noch weitere ersetzte Elemente, wie `video` und `embed`. Und einige Elemente verhalten sich unter bestimmten Umständen wie ersetzte Elemente. Hier ist ein Beispiel für ein `input`-Element mit dem `type`-Attribut, das auf `image` gesetzt ist:

```html
<input type="image" alt="Descriptive text goes here" src="example-img-url">
```

Dieser Typ von `input` gilt als ersetztes Element, aber andere `input`-Typen wie `text` oder `email` sind keine ersetzten Elemente.
