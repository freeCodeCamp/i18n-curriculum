# --description--

Die folgende iterative Folge ist für die Menge der positiven ganzen Zahlen definiert:

<div style='padding-left: 4em;'><var>n</var> → <var>n</var>/2 (<var>n</var> ist gerade)</div>

<div style='padding-left: 4em;'><var>n</var> → 3<var>n</var> + 1 (<var>n</var> ist ungerade)</div>

Mit der obigen Regel und beginnend bei 13 erzeugen wir die folgende Folge:

<div style='text-align: center;'>13 → 40 → 20 → 10 → 5 → 16 → 8 → 4 → 2 → 1</div>

Man sieht, dass diese Folge (beginnend bei 13 und endend bei 1) 10 Glieder enthält. Obwohl es noch nicht bewiesen ist (Collatz-Problem), wird angenommen, dass alle Startzahlen bei 1 enden.

Welche Startzahl unter den gegebenen `limit` erzeugt die längste Kette?

**Hinweis:** Sobald die Kette beginnt, dürfen die Glieder über `limit` hinausgehen.

```js
function longestCollatzSequence(limit) {
  let longest = 1;
  let maxLength = 1;
  for (let i = Math.floor(limit / 2); i < limit; i++) {
    let len = colLen(i);
    if (len > maxLength) {
      longest = i;
      maxLength = len;
    }
  }
  return longest;
}

const knownSequence = { '1': 1 };

function colLen(n) {
  if (knownSequence[n]) {
    return knownSequence[n];
  } else {
    const len = n % 2 === 0 ? colLen(n / 2) + 1 : colLen((3 * n + 1) / 2) + 2;
    knownSequence[n] = len;
    return len;
  }
}
```
