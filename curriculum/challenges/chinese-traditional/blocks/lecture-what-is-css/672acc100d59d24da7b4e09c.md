# --interactive--

CSS 組合器用於定義 CSS 中選取器之間的關係。它們有助於根據元素與其他元素的關係來選取元素，從而實現更精確且高效的樣式設定。

我們將討論一些主要的 CSS 組合器及其使用案例，從後代組合器開始。

後代組合器用於標的被第二個選擇器匹配的元素，前提是它們巢狀在匹配第一個選擇器的祖先元素內。祖先可以是父元素或父元素的父元素。

為了更好地理解這個運作方式，讓我們來看一個範例。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<figure>
  <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Relaxing Cat">
  <figcaption>A relaxing cat with a border</figcaption>
</figure>
```

```css
figure img {
  border: 4px solid red;
}
```

:::

在上述範例中，我們使用後代組合器來選取所有位於 `figure` 元素內的影像元素，並在四個邊緣套用 `solid red` 邊框。

請注意，父選擇器和子選擇器之間使用了 `space`。

在這種情況下，`figure` 會是父元素，而 `img` 會是子元素。

如果你在 `figure` 元素內有多個影像，使用後代組合器會是為每個影像套用實心黑色邊框的好方法。

另一種型別的組合器是子組合器。

CSS 中的子代組合器（`>`）用於選取指定父元素的直接子元素。

此組合器僅標的具有特定父元素的元素，使你的 CSS 規則更精確，並防止對更深層巢狀元素產生非預期的樣式。

讓我們來看看以下的 HTML 範例：

:::interactive_editor

```html
<div class="container">
  <p>First</p>
  <div>
    <p>Second</p>
  </div>
  <div>
    <p>Third</p>
  </div>
</div>
```

:::

在上述 HTML 結構中，`container` 類別被套用到一個 `div` 元素。

在這個容器內，有一個直接子元素 `p`（「First」），接著是兩個額外的 `div` 元素，每個都包含一個 `p` 元素（「Second」和「Third」）。

第一個 `p` 元素是 `.container` 元素的直接子元素，而另外兩個 `p` 元素則巢狀在其他 `div` 元素內，使它們成為更深層的後代元素。

要將樣式套用到 `container` 類別的直接子元素，你可以使用子元素組合器，如下所示：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <p>First</p>
  <div>
    <p>Second</p>
  </div>
  <div>
    <p>Third</p>
  </div>
</div>
```

```css
.container > p {
  color: blue;
}
```

:::

在上述範例中，我們只針對 `container` 類別的直接子元素。這將使直接子元素的文字顏色變為 `blue`。

因為另外兩個段落元素是巢狀在 `div` 元素內，所以它們不被視為 `container` 類別的直接子元素，且不會獲得藍色的文字顏色。

另一個常見的組合器是 next-sibling 組合器。

CSS 中的下一個兄弟組合器（`+`）會選取緊接在指定兄弟元素後面的元素。當你想要將樣式套用到直接跟隨另一個元素的元素時，這個組合器非常有用，允許根據元素相對於其兄弟元素的位置進行目標式的樣式設定。

讓我們來看看以下的 HTML 範例：

:::interactive_editor

```html
<figure>
  <img
    src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg"
    alt="A cute orange cat lying on its back."
  />
  <figcaption>A cute orange cat lying on its back.</figcaption>
</figure>
```

:::

這裡，我們有一個包含 `img` 元素的 `figure` 元素，後面緊接著一個 `figcaption` 元素。`figcaption` 元素是 `img` 元素的直接兄弟元素。

如果你想要在 `figcaption` 元素周圍套用黑色邊框，你可以使用下一個兄弟組合器，如下所示：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<figure>
  <img
    src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg"
    alt="A cute orange cat lying on its back."
  />
  <figcaption>A cute orange cat lying on its back.</figcaption>
</figure>
```

```css
img + figcaption {
  border: 4px solid black;
}
```

:::

在此範例中，下一個兄弟組合器（`+`）會選取緊接在 `img` 元素後的 `figcaption` 元素。所套用的 CSS 規則會在 `figcaption` 周圍添加 `4px solid black border`。

另一個常見的組合器是後續兄弟組合器。

CSS 中的後續兄弟組合器（`~`）會選取指定元素之後的所有兄弟元素。

與只標的緊接後方兄弟元素的 next-sibling 組合器不同，subsequent-sibling 組合器（`~`）可以標的指定元素後方的任何兄弟元素，提供更大的彈性來進行樣式設定。

讓我們來看看以下的 HTML 範例：

:::interactive_editor

```html
<div class="container">
  <h2>Subheading</h2>
  <p>First paragraph.</p>
  <p>Second paragraph.</p>
  <p>Third paragraph.</p>
  <p>Another paragraph element</p>
</div>
```

:::

在這個 HTML 結構中，我們有一個 `h2` 元素，後面跟著四個段落元素。這些段落元素是 `h2` 元素的兄弟元素。

如果你想為所有位於 `h2` 元素之後的段落元素設計樣式，則可以使用後續兄弟組合器，如下所示：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <h2>Subheading</h2>
  <p>First paragraph.</p>
  <p>Second paragraph.</p>
  <p>Third paragraph.</p>
  <p>Another paragraph element</p>
</div>
```

```css
h2 ~ p {
  color: green;
}
```

:::

在此範例中，所有位於 `h2` 元素之後的段落元素文字顏色都將是綠色。

後續兄弟組合器（`~`）會標的所有出現在 `h2` 元素之後的段落兄弟元素，無論它們是否為直接兄弟元素。

總之，理解並使用各種 CSS 組合器讓你能夠對你的 HTML 元素套用精確的樣式，提升你設計的控制元件和彈性。

透過精通這些選擇器，你可以創建更複雜且具標的性的樣式規則，從而提升你的網頁外觀與功能。

哪一個 CSS 規則會為所有位於 `div` 元素內的 `span` 元素套用樣式，無論它們巢狀多深？

## --answers--

```css

  color: red;
```
