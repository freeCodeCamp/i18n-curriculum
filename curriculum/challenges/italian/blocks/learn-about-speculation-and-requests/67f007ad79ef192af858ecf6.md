# --explanation--

Jake può dire `It's possible` perché pensa che ci sia la possibilità che l'email sospetta faccia parte di un attacco di phishing.

`Impossible` si forma aggiungendo il prefisso `im-` (che significa "non" o "l'opposto di") alla parola base `possible`. Per esempio:

- `It's possible.` - Qualcosa può succedere.

- `It's impossible.` - Non può succedere.
