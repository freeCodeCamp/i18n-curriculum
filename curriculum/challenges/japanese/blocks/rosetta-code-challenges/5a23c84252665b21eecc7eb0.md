# --description--

**<a href="http://www.rosettacode.org/wiki/I_before_E_except_after_C" target="_blank" rel="noopener noreferrer nofollow">「Cの後を除きIがEの前に来る」</a>** は英語の綴りに関する一般的なルールです。単語が`ei`か`ie`のどちらで綴られているか不確かな場合、この韻文は、前の文字が`c`でない限り正しい順序は`ie`であり、もし前の文字が`c`なら`ei`である可能性があることを示唆しています。

提供された単語を使って、このフレーズの2つの副節がそれぞれ妥当かどうかを確認してください：

<ol>
  <li>
    <i>「Cに先行されない場合はIがEの前に来る」</i>
  </li>
  <li>
    <i>「Cに先行される場合はEがIの前に来る」</i>
  </li>
</ol>

両方の副節が妥当であれば、元のフレーズは妥当と言えます。

# --instructions--

単語を受け取り、その単語がこのルールに従っているかどうかをチェックする関数を書いてください。単語がルールに従っていればtrueを、そうでなければfalseを返す関数にしてください。
