# --explanation--

在 `Present Simple` 时态中，否定形式是使用 `doesn't` 或 `isn't` 为第三人称单数主语（`he, she, it`）创建的。 

- 使用 `doesn't` 和动作动词表示某事未发生，例如 `It doesn't work.`

- 使用 `isn't` 和 `to be` 的形式来表示某事不真实，例如 `isn't set up.`

在这个句子中，你需要同时使用 `doesn't` 和 `isn't` 来正确描述该特色为什么没有按预期工作。
