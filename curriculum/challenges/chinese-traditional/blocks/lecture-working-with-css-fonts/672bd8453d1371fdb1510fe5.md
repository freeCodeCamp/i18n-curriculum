# --interactive--

外部字型是未直接包含在你的專案檔案中的字型檔案。它們通常託管在獨立的伺服器上。伺服器是透過網路向其他電腦提供資料或服務的電腦。你將在未來的課程中學習更多關於伺服器的內容。外部字型提供你更多彈性，因為你可以使用可能未安裝在使用者裝置上的訂製字型。

Google Fonts 和 Font Squirrel 是尋找及使用免費字型、字體以進行網頁開發的熱門線上資源。你可以非常輕鬆地將它們整合到你的專案中。

讓我們從 Google Fonts 開始。這是 Google 提供的一項服務，提供一個字型群集，其中許多字型是專為網頁開發設計的。在 Google Fonts 的使用者介面上，你可以看到許多不同的元素。讓我們逐一介紹。在左側邊欄，你會找到：

- 一個用來查看、尋找和篩選字型的區段。
- 關於 Noto 的特別章節，這是一個包含多種字重、字寬和樣式的高品質字型群集，非常適合用於超過 1,000 種語言和 150 多種書寫系統的溝通。
- 一個你可以為你的網頁專案尋找並下載圖示的區段。
- 一個你可以進一步了解字型、字體及其最佳實務的章節，接著是常見問題。

要為你的網站添加 Google Font，你應該前往第一個區段（Fonts）。你可以在左側邊欄標示為「Preview」的地方自訂預覽文字。只要輸入你想看到的文字，它就會自動更新。這對於預覽你已經為網站構思好的文字字型非常有幫助。你也可以調整字型大小並根據字型的特性篩選字型。 

要查看字型的更多資訊，只要點擊它，你就會看到關於該字型的重要資訊，例如設計者、預覽文字，以及不同的樣式，如 light、regular、medium、bold 和 black。如果你點擊頂部的選項，你可以前往型別測試器、查看單一字形，並了解更多關於字型及其授權的資訊。

你可以使用位於可輸入預覽文字欄位旁邊的藍色滾軸，自訂預覽文字的字型大小。當你準備好將字型添加到你的專案時，請點擊頂部的藍色 `Get font` 按鈕。系統會帶你到一個摘要頁面，在那裡你可以看到目前選取的字型。你可以同時選取多個字型。

接著，你必須選擇是否要下載字型檔案，將它們作為區域的檔案添加到你的專案中，或是選擇將它們作為外部字型，並在使用者進入你的網站時從 Google 的伺服器下載。如果你想下載字型，請點擊「Download all」，但如果你偏好使用 Google 伺服器上的外部字型，請點擊「Get embed code」。如果你點擊「Get embed code」，你將會看到應該遵循的指示，以將這些外部字型添加到你的專案中。

對於網頁開發專案，你有兩個選項，可使用 `link` 元素或 `@import`。如果你選擇 `link` 元素選項，你可以複製並貼上 HTML 片段和 CSS 規則，將它們添加到你的專案中。你應該將程式碼嵌入在 HTML 檔案的 `head` 元素中，並添加符合你需求的 CSS 規則。

每種字型樣式都有一個 CSS 規則。每個規則都會指定訂製字型及備用字型，以防訂製字型未正確載入。

如果你選擇 `@import` 選項，可選方案，則需要將該規則添加到你的 CSS 檔案中。以下是一個範例：

```css
@import url('https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap');
```

CSS 規則完全相同。

如果你只想包含特定的字型樣式，你也有選項可從這些程式碼片段中移除它們。HTML 和 CSS 程式碼片段將會自動更新。 

這就是 Google Fonts。它是尋找免費且開放原始碼字型、字體用於網頁開發專案中最受歡迎的選項之一。

另一個很棒的選項是 Font Squirrel，在那裡你可以找到並下載你為設計所想像的訂製字型。 如果你搜尋字型並點擊結果，你會看到更多關於該字型的詳細資訊，包括樣本、試用、字形和 Webfont 套件。你也可以看到該字型不同風格和變化的範例，例如 thin、light、medium、bold 和 black。

一旦你決定了要在你的專案中使用的字型，請前往「Webfont Kit」標籤。在這裡，你可以檢查該字型的授權是否允許你在 `@font-face` CSS 嵌入中使用。你也可以選擇子集和格式。

點擊「Download @font-face Kit」按鈕將會下載一個壓縮（zip）檔案。如果你解壓縮它，你會找到一個「web fonts」資料夾、以程式本文檔形式的授權文件，以及一個包含如何使用網頁字型說明的 HTML 檔案。web fonts 資料夾中會有你專案所需的所有字型檔案。你會發現它們被分組到不同字型樣式的特定資料夾中。

如果你在瀏覽器中開啟 HTML 檔案，它會說明如何為你的專案添加字型、如何使用 `@font-face` 宣告將它們包含在樣式表中，以及如何在 CSS 規則中使用它們。

這是你如何將訂製 Font Squirrel 字型添加到你的專案。你也可以將它們託管在獨立的伺服端，並將它們用作外部字型。外部字型對於訂製你網站的整體外觀和設計非常有幫助。它們可以讓你的网站真正獨一無二。

但是，你也應該記住，使用多個外部字型會增加網站的載入時間，影響其整體效率。你應該始終努力在風格和效率之間找到平衡。
