# --description--

既然你已經重設了 html 盒模型，你也需要將此套用到其中的元素。為此，你需要將所有其他元素的 `box-sizing` 屬性設定為繼承，這可確保標的元素採用與其父元素相同的值。

你還需要標的偽元素，偽元素是跟隨選擇器的特殊關鍵字。你將使用的兩個偽元素是 `::before` 和 `::after` 偽元素。

`::before` 選擇器創建一個偽元素，該元素是所選元素的第一個子元素，而 `::after` 選擇器創建一個偽元素，該元素是所選元素的最後一個子元素。這些偽元素通常用於創建裝飾性內容，你稍後會在這個專案中看到。

現在，使用 `*` 創建一個 CSS 選擇器來標的所有元素，並包含偽元素 `::before` 和 `::after`。將 `box-sizing` 屬性設為 `inherit`。

```js
assert.exists(new __helpers.CSSHelp(document).getStyle('html'));

```

```js
assert.equal(new __helpers.CSSHelp(document).getStyle('*, ::before, ::after')?.boxSizing, 'inherit');
```
