# --description--

Kuna vikwazo viwili kwa majina ya vigezo katika JavaScript:

1. Jina lazima liwe na herufi tu, nambari, au alama `$` na `_`.
1. Herufi ya kwanza haipaswi kuwa nambari.

Mifano ya majina sahihi:

```js
let userName;
let test123;
```

Kinachovutia – alama ya dola `'$'` na mstari wa chini `'_'` pia zinaweza kutumika katika majina. Ni alama za kawaida, kama herufi, bila maana maalum yoyote.

Majina haya ni halali:

```js
let $ = 1; // declared a variable with the name "$"
let _ = 2; // and now a variable with the name "_"

console.log($ + _); // 3
```

Mifano ya majina ya vigezo yasiyo sahihi:

```js
let 1a; // cannot start with a digit

let my-name; // hyphens '-' aren't allowed in the name
```
