Un giocatore parte dalla casella GO e somma i risultati di due dadi a 6 facce per determinare il numero di caselle da avanzare in senso orario. Senza altre regole ci aspetteremmo di visitare ogni casella con probabilità uguale: 2,5%. Tuttavia, atterrare su G2J (Vai in prigione), CC (cassetta comunità) e CH (imprevisti) modifica questa distribuzione.

Oltre a G2J e a una carta da ciascuno di CC e CH che ordina al giocatore di andare direttamente in prigione, se un giocatore tira tre doppi consecutivi, non avanza il risultato del terzo tiro. Invece procede direttamente in prigione.

All'inizio della partita, le carte CC e CH vengono mescolate. Quando un giocatore atterra su CC o CH prende una carta dalla cima del rispettivo mazzo e, dopo aver seguito le istruzioni, la rimette in fondo al mazzo. Ci sono sedici carte in ogni mazzo, ma per questo problema ci interessano solo le carte che ordinano un movimento; qualsiasi istruzione non riguardante il movimento sarà ignorata e il giocatore rimarrà sulla casella CC/CH.

<ul>
  <li>Cassetta comunità (2/16 carte):</li>
  <ol>
    <li>Avanza a GO</li>
    <li>Vai in prigione</li>
  </ol>

  <li>Imprevisti (10/16 carte):</li>
  <ol>
    <li>Avanza a GO</li>
    <li>Vai in prigione</li>
    <li>Vai a C1</li>
    <li>Vai a E3</li>
    <li>Vai a H2</li>
    <li>Vai a R1</li>
    <li>Vai alla prossima R (compagnia ferroviaria)</li>
    <li>Vai alla prossima R</li>
    <li>Vai alla prossima U (compagnia di servizi)</li>
    <li>Torna indietro di 3 caselle.</li>
  </ol>
</ul>

Il cuore di questo problema riguarda la probabilità di visitare una casella particolare. Cioè, la probabilità di finire su quella casella dopo un tiro. Per questo motivo dovrebbe essere chiaro che, fatta eccezione per G2J per cui la probabilità di finirci sopra è zero, le caselle CH avranno le probabilità più basse, dato che 5/8 richiedono un movimento verso un'altra casella, e ci interessa la casella finale su cui il giocatore si ferma ad ogni tiro. Non faremo distinzione tra "Solo in visita" e essere mandati in prigione, e ignoreremo anche la regola che richiede un doppio per "uscire di prigione", assumendo che paghino per uscire al turno successivo.

Partendo da GO e numerando le caselle sequenzialmente da 00 a 39 possiamo concatenare questi numeri a due cifre per produrre stringhe che corrispondono a insiemi di caselle.

Statisticamente si può dimostrare che le tre caselle più popolari, in ordine, sono JAIL (6,24%) = casella 10, E3 (3,18%) = casella 24, e GO (3,09%) = casella 00. Quindi queste tre caselle più popolari possono essere elencate con la stringa modale a sei cifre `102400`.

Se, invece di usare due dadi a 6 facce, si usano due dadi a `n` facce, trova la stringa modale a sei cifre.

```js
const GO = 0;
const JAIL = 10;
const GO_TO_JAIL = 30;

const C1 = 11;
const E3 = 24;
const H2 = 39;

const R1 = 5;
const R2 = 15;
const R3 = 25;

const U1 = 12;
const U2 = 28;

const SPECIAL_CARDS = 16;
const GAME_SQUARES = 40;

const CC1 = 2;
const CC2 = 17;
const CC3 = 33;
const CHESTS = [CC1, CC2, CC3];
const chestCardsMoves = [GO, JAIL];

const CH1 = 7;
const CH2 = 22;
const CH3 = 36;
const CHANCES = [CH1, CH2, CH3];
const chanceCardsMoves = [GO, JAIL, C1, E3, H2, R1];
const chanceToRailroad = { [CH1]: R2, [CH2]: R3, [CH3]: R1 };
const chanceToUtility = { [CH1]: U1, [CH2]: U2, [CH3]: U1 };

function multiplyMatrix(matrix1, matrix2) {
  const multiplied = [];

  for (let row = 0; row < matrix1.length; row++) {
    const newRow = [];
    for (let col = 0; col < matrix1[row].length; col++) {
      let newCell = 0;
      for (let i = 0; i < matrix1[row].length; i++) {
        const value1 = matrix1[row][i];
        const value2 = matrix2[i][col];
        newCell += value1 * value2;
      }
      newRow.push(newCell);
    }
    multiplied.push(newRow);
  }
  return multiplied;
}

function normalizeRow(row) {
  const sum = row.reduce((total, value) => total + value, 0);
  if (sum > 0) {
    for (let j = 0; j < row.length; j++) {
      const value = row[j];
      row[j] = value / sum;
    }
  }
}

function sortByProbability(board) {
  return board
    .map((probability, squareNo) => [squareNo, probability])
    .sort((a, b) => a[1] - b[1])
}

function getTopThree(board) {
  return sortByProbability(board)
    .slice(-3)
    .reverse()
    .map(([squareNo, _]) => squareNo.toString().padStart(2, '0')
    )
    .join('');
}

function didConverge(matrix1, matrix2, precision) {
  return matrix1.every((row, rowNo) => row.every((value1, colNo) => Math.abs(value1 - matrix2[rowNo][colNo]) <= precision))
}

function monopolyOdds(diceSides) {
  // Based on https://github.com/ByteThisCoding/project-euler/blob/master/problems/0084/0084.ts

  const timesRolled = new Array(diceSides * 2 + 1).fill(0);
  for (let dice1 = 1; dice1 <= diceSides; dice1++) {
    for (let dice2 = 1; dice2 <= diceSides; dice2++) {
      timesRolled[dice1 + dice2]++;
    }
  }

  // Transitions matrix contain probabilities of reaching each square (row values)
  // from each starting square (row no.).
  let transitions = [];
  for (let startSquare = 0; startSquare < GAME_SQUARES; startSquare++) {
    const row = new Array(GAME_SQUARES).fill(0);
    for (let rollResult = 2; rollResult <= diceSides * 2; rollResult++) {
      const rollChance = timesRolled[rollResult]
      const position = (startSquare + rollResult) % GAME_SQUARES;

      if (CHANCES.includes(position)) {
        // Chance cards ordering movement.
        for (let i = 0; i < chanceCardsMoves.length; i++) {
          const nextSquare = chanceCardsMoves[i];
          row[nextSquare] += rollChance / SPECIAL_CARDS;
        }
        row[chanceToRailroad[position]] += 2 * rollChance / SPECIAL_CARDS;
        row[chanceToUtility[position]] += rollChance / SPECIAL_CARDS;
        row[position - 3] += rollChance / SPECIAL_CARDS;

        // Rest non-moving Chance cards.
        row[position] += (SPECIAL_CARDS - chanceCardsMoves.length) * rollChance / SPECIAL_CARDS;
      } else if (CHESTS.includes(position)) {
        // Community Chest cards ordering movement.
        for (let i = 0; i < chestCardsMoves.length; i++) {
          const nextSquare = chestCardsMoves[i];
          row[nextSquare] += rollChance / SPECIAL_CARDS;
        }
        // Rest non-moving Community Chest cards.
        row[position] += (SPECIAL_CARDS - chestCardsMoves.length) * rollChance / SPECIAL_CARDS
      } else if (position === GO_TO_JAIL) {
        row[JAIL] += rollChance;
      } else {
        row[position] += rollChance;
      }
    }
    normalizeRow(row)
    transitions.push(row);
  }

  const precision = 0.000001;
  for (let i = 0; i < GAME_SQUARES; i++) {
    const next = multiplyMatrix(transitions, transitions);
    if (didConverge(transitions, next, precision)) {
      break;
    }
    transitions = next;
  }

  // All rows converge to the same values.
  return getTopThree(transitions[0]);
}
```
