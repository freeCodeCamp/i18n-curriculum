# --explanation--

`To cost` bedeutet, dass eine Zahlung erforderlich ist oder einen Preis hat. Wenn Sie sagen, dass etwas `costs more than expected` ist, bedeutet das, dass Sie mehr Geld bezahlen müssen als geplant. Zum Beispiel:

`The event cost more than we budgeted for because of extra equipment.` – Das bedeutet, dass die endgültigen Ausgaben höher waren als ursprünglich geplant.
