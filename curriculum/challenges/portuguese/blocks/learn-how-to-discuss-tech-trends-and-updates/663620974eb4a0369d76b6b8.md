# --description--

A frase `some of these` é usada para se referir a uma parte específica de um grupo ou a um subconjunto de itens de um conjunto maior. É frequentemente usado quando você quer mencionar algumas coisas de um grupo sem listar cada uma individualmente.

Por exemplo, `Let's try some of these cookies.` O locutor sugere experimentar alguns cookies de um lote maior.

## --blanks--

`some`

### --feedback--

Esta palavra indica uma parte de algo, não o todo, referindo-se a uma quantidade não especificada dos itens discutidos.

---

`of`

### --feedback--

Esta preposição é usada para ligar a quantidade geral com os itens específicos que estão sendo discutidos, formando uma expressão partitiva.

---

`these`

### --feedback--

Esta palavra especifica sobre qual grupo de itens está sendo falado, indicando um grupo específico e presente.
