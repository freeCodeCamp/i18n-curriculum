# --explanation--

`To set aside` significa mettere da parte o conservare qualcosa, specialmente denaro o tempo, per uno scopo futuro specifico. Per esempio:

`It's smart to set aside money for unforeseen expenses.` — È bene tenere un po' di soldi extra per le sorprese.

Una `contingency` è un piano di riserva o di emergenza, specialmente per situazioni che potrebbero non verificarsi ma che potrebbero accadere. Per esempio:

`It's important to have contingency measures during testing.` — Hai bisogno di passaggi o piani extra nel caso in cui le cose non vadano come previsto durante i test.

Alice sta dicendo che hanno riservato il 15% del budget per situazioni impreviste.
