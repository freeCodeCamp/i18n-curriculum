# --explanation--

`Weak passwords`는 추측하거나 해킹하기 쉬운 비밀번호입니다. 예를 들어:

`123456 and qwerty are weak passwords.` - 강력한 비밀번호는 길고 문자, 숫자, 기호를 포함해야 합니다.

`Shared credentials`는 여러 사람이 사용하는 로그인 정보(사용자 이름과 비밀번호)입니다. 예를 들어:

`Using shared credentials increases security risks.` - 비밀번호를 공유하면 잘못된 사람에게 넘어갔을 때 무단 접근으로 이어질 수 있습니다.
