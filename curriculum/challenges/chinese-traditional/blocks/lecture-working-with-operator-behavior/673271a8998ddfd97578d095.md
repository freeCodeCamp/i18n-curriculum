# --interactive--

如果你正在處理數字並且需要將數值增加或減少 1，遞增和遞減運算子會讓工作變得更簡單。讓我們用簡單的方式來說明。

遞增和遞減運算子分別由 `++` 和 `--` 表述。它們都允許你將變數的值調整 `1`。

與其寫類似 `x = x + 1` 或 `x = x - 1`，你可以直接使用 `x++` 來加 `1`，或 `x--` 來減 `1`。這樣更快、更簡潔且更容易閱讀。

累加運算子和遞減運算子的運作方式有一個變化：它們有兩種形式，前置式和後置式，差別在於值何時被更新。對於累加運算子，前置式是 `++x`，後置式是 `x++`。

前置式（`++x`）先增加變數的值，然後傳回新的值。後置式（`x++`）先傳回變數的當前值，然後再增加它：

:::interactive_editor

```js
let x = 5;

console.log(++x); // 6
console.log(x); // 6
```

:::

在上述程式碼中，`++x` 意味著「先遞增 `x`，然後使用它」。所以當你記錄 `++x` 時，你會立即得到遞增後的值，也就是 `6`。

現在，讓我們來看看使用後置式的範例：

:::interactive_editor

```js
let y = 5;

console.log(y++); // 5
console.log(y); // 6
```

:::

在這個範例中，`y++` 意思是「先使用 `y`，然後再遞增它」。當你記錄 `y++` 時，你會得到 `5`，但在那行程式碼之後，`y` 變成 `6`。

遞減運算子與遞增運算子做相同的事情，不同的是它將值減少 `1`。同樣地，有兩種形式：前置式（`--x`）先將變數的值減少，然後傳回新的值。後置式（`x--`）先傳回目前的值，然後再將其減少：

:::interactive_editor

```js
let x = 5;
console.log(--x); // 4
console.log(x); // 4

let y = 5;
console.log(y--); // 5
console.log(y); // 4
```

:::

那麼，你應該使用前置式還是後置式？在許多情況下，使用哪一種並不重要。兩者都能完成工作。然而，如果你立即在一個表達式中使用該值，差異就變得重要了。讓我們來看看這個範例：

:::interactive_editor

```js
let a = 5;
let b = ++a;
console.log(b); // 6 (a was incremented before assignment)

let c = 5;
let d = c++;
console.log(d); // 5 (c was incremented after assignment)
```

:::

所以，如果你需要立即取得更新後的值，請使用前置式。如果你想先取得目前的值，且不在意遞增的結果直到稍後，再使用後置式。
