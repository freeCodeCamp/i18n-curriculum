# --description--

Albert 选择一个正整数 $k$，然后在区间 [0,1] 内以均匀分布随机选择两个实数 $a$、$b$。

然后计算和 ${(ka + 1)}^2 + {(kb + 1)}^2$ 的平方根，并四舍五入到最接近的整数。如果结果与 $k$ 相等，他得 $k$ 分；否则他得零分。

例如，如果 $k = 6$，$a = 0.2$ 且 $b = 0.85$，那么 ${(ka + 1)}^2 + {(kb + 1)}^2 = 42.05$。42.05 的平方根是 6.484……，四舍五入到最接近的整数后，变为 6。这与 $k$ 相等，所以他得 6 分。

可以证明，如果他以 $k = 1, k = 2, \ldots, k = 10$ 玩 10 轮，他的总得分的期望值，四舍五入到小数点后五位，是 10.20914。

如果他玩 ${10}^5$ 回合，且 $k = 1, k = 2, k = 3, \ldots, k = {10}^5$，那么他的总得分的期望值是多少，四舍五入到五位小数？
