# --description--

상속은 특정 CSS 속성이 요소에 적용될 때, 그 요소의 자손 요소들이 명시적으로 규칙을 작성하지 않아도 해당 속성을 물려받는 것을 말합니다. 타이포그래피 관련 속성들(`color`, `font-size`, `font-family` 등)은 보통 상속되지만, 대부분 다른 속성들은 그렇지 않습니다.

예외는 요소를 직접 지정하는 경우로, 이 경우 상속보다 우선합니다:

```html
<!-- index.html -->

<div id="parent">
  <div class="child"></div>
</div>
```

```css
/* styles.css */

#parent {
  color: red;
}

.child {
  color: blue;
}
```

`parent` 요소가 ID로 더 높은 특이성을 가지더라도, `child` 요소는 직접 지정된 `color: blue` 스타일이 적용되며, 부모로부터 상속된 `color: red` 스타일은 적용되지 않습니다.
