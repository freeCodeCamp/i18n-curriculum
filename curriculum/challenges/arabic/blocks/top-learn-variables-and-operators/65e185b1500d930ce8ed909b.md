# --description--
أسهل طريقة للبدء هي إنشاء ملف HTML يحتوي على شفرة JavaScript بداخله. اكتب الهيكل الأساسي لـ HTML في ملف على جهاز الكمبيوتر الخاص بك في مكان ما:

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>
  <script>
    // Your JavaScript goes here!
    console.log("Hello, World!")
  </script>
</body>
</html>
```

احفظ وافتح هذا الملف في متصفح ويب (يمكنك استخدام <a href="https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer" target="_blank" >"Live Server" في Visual Studio Code</a> للقيام بذلك!) ثم افتح وحدة التحكم في المتصفح بالنقر بزر الفأرة الأيمن على صفحة الويب الفارغة واختيار "Inspect" أو "Inspect Element". في لوحة 'أدوات المطوِّر' ابحث واختر تبويب 'وحدة التحكم'، حيث سترى ناتج أمر `console.log` الخاص بنا.

> `console.log()` هو الأمر لطباعة شيء ما في وحدة التحكم الخاصة بالمطوِّر في متصفحك. يمكنك استخدامه لطباعة النتائج من أي من المقالات والتمارين التالية في وحدة التحكم. نشجعك على كتابة الشفرة مع جميع الأمثلة في هذا الدرس والدروس القادمة.

طريقة أخرى لإدراج JavaScript في صفحة ويب هي عبر سكربت خارجي. هذا مشابه جدًا لربط ملفات CSS الخارجية بموقعك.

```html
  <script src="javascript.js"></script>
```

ملفات JavaScript تحمل الامتداد `.js` مثل `.css` لملفات الأنماط. تُستخدم ملفات JavaScript الخارجية للسكريبتات الأكثر تعقيدًا.
