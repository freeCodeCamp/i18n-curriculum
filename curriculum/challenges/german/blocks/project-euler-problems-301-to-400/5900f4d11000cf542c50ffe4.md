# --description--

Betrachten Sie die Teiler von 30: 1, 2, 3, 5, 6, 10, 15, 30.

Man erkennt, dass für jeden Teiler $d$ von 30 gilt: $d + \frac{30}{d}$ ist eine Primzahl.

Finden Sie die Summe aller positiven ganzen Zahlen $n$, die 100\\,000\\,000 nicht überschreiten, so dass für jeden Teiler $d$ von $n$ gilt: $d + \frac{n}{d}$ ist eine Primzahl.
