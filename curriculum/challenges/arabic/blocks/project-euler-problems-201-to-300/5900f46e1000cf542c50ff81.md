# --description--

تعرف المتتالية كما يلي:

- $g_k = 1$، لكل $0 ≤ k ≤ 1999$
- $g_k = g_{k - 2000} + g_{k - 1999}$، لكل $k ≥ 2000$.

احسب $g_k$ باقي القسمة على 20092010 عندما يكون $k = {10}^{18}$.
