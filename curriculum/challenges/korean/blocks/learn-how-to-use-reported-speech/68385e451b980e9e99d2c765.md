# --explanation--

`True`는 어떤 것이 정확하거나 진짜이거나 올바르다는 뜻의 형용사입니다. 진술이나 사실에 동의할 때 사용합니다. `the truth`와는 다르며, `the truth`는 어떤 것에 대한 실제 사실이나 현실을 가리키는 명사로, 더 넓거나 추상적인 의미로 쓰입니다. 예를 들어:

- `It's true that the server crashed yesterday.` – 충돌에 관한 진술이 정확하다는 뜻입니다.

- `The truth is that we didn't have a backup.` – 실제로 백업이 존재하지 않았다는 사실을 의미합니다.
