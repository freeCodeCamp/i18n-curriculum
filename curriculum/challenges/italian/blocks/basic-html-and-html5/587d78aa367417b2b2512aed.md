# --description--

Finora le sfide hanno trattato elementi HTML specifici e i loro usi. Tuttavia, ci sono alcuni elementi che danno la struttura generale alla tua pagina e dovrebbero essere inclusi in ogni documento HTML.

All'inizio del documento devi indicare al browser quale versione di HTML sta usando la tua pagina. HTML è un linguaggio in evoluzione e viene aggiornato regolarmente. La maggior parte dei browser principali supporta la versione più recente, che è HTML5. Tuttavia, le pagine web più vecchie possono usare versioni precedenti del linguaggio.

Comunichi questa informazione al browser aggiungendo il tag `<!DOCTYPE ...>` nella prima riga, dove la parte `...` indica la versione di HTML. Per HTML5, usi `<!DOCTYPE html>`.

Il `!` e il `DOCTYPE` maiuscolo sono importanti, soprattutto per i browser più vecchi. Il `html` non fa distinzione tra maiuscole e minuscole.

Successivamente, il resto del codice HTML deve essere racchiuso tra i tag `html`. Il tag di apertura `<html>` va subito sotto la riga `<!DOCTYPE html>`, e il tag di chiusura `</html>` va alla fine della pagina.

Ecco un esempio della struttura della pagina. Il tuo codice HTML andrà nello spazio tra i due tag `html`.

```html
<!DOCTYPE html>
<html>

</html>
```

# --instructions--

Aggiungi un tag `DOCTYPE` per HTML5 all'inizio del documento HTML vuoto nell'editor di codice. Sotto di esso, aggiungi i tag di apertura e chiusura `html`, che racchiudono un elemento `h1`. L'intestazione può contenere qualsiasi testo.

# --hints--

Il tuo codice dovrebbe includere un tag `<!DOCTYPE html>`.

```js
assert.match(code,/<!DOCTYPE\s+?html\s*?>/gi);
```

Dovrebbe esserci un elemento `html`.

```js
assert.lengthOf(document.querySelectorAll('html'), 1);
```

I tag `html` devono racchiudere un elemento `h1`.

```js
assert.match(code,/<html>\s*?<h1>\s*?.*?\s*?<\/h1>\s*?<\/html>/gi);
```
