# --explanation--

`On schedule` bedeutet, dass die Dinge gemäß dem geplanten Zeitplan voranschreiten. Zum Beispiel:

`The project is still on schedule for delivery next week.` — Es bedeutet, dass es rechtzeitig fertiggestellt wird.

`Miscommunication` bezieht sich auf eine Situation, in der Informationen nicht korrekt geteilt oder verstanden wurden. Zum Beispiel:

`There was a miscommunication about the meeting location.` — Die Leute haben den richtigen Ort nicht verstanden.

`Right away` bedeutet sofort oder so schnell wie möglich. Zum Beispiel:

`I'll send the update right away.` — Sie werden es ohne Verzögerung senden.
