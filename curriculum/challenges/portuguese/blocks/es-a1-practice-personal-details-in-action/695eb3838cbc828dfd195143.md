# --explanation--

A pergunta `¿Cuántos años tienes?` pergunta a idade de alguém.

Para buscar a resposta, você precisa olhar o campo `Edad` no formulário.

O formulário mostra `Edad`: 25

Então a resposta correta é **25**.

Esta pergunta é comumente respondida em espanhol usando a estrutura:

`Tengo` + número + `años`

Observe que a letra **`ñ`** não é apenas um `n` com um `virgulilla` (`˜`), é uma letra completamente separada no alfabeto espanhol, e ela muda tanto a pronúncia e o significado de uma palavra.
