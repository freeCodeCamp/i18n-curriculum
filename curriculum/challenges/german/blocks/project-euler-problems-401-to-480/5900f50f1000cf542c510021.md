# --description--

Sei $n$ eine positive ganze Zahl. Ein ganzzahliges Dreier-Tupel ($a$, $b$, $c$) heißt Faktorisierungs-Dreier von $n$, wenn:

- $1 ≤ a ≤ b ≤ c$
- $a \times b \times c = n$.

Definieren Sie $f(n)$ als $a + b + c$ für das Faktorisierungs-Dreier ($a$, $b$, $c$) von $n$, das $\frac{c}{a}$ minimiert. Man kann zeigen, dass dieses Dreier-Tupel eindeutig ist.

Zum Beispiel gilt $f(165) = 19$, $f(100\\,100) = 142$ und $f(20!) = 4\\,034\\,872$.

Finden Sie $f(43!)$.
