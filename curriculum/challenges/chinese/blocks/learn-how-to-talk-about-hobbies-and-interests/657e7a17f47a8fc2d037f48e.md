# --description--

`One of these days` 是你用来表示你计划很快做某事，但还没有确定具体时间的短语。它类似于说“很快的某个时候”或“在不久的将来”。例如：

`One of these days, I'll visit Paris.`——这意味着你想去巴黎并希望在不久的将来去，但你还不知道确切的日期。
