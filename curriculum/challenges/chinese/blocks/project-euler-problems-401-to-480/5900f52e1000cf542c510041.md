# --description--

假轮线是由一个小圆在一个大圆内部滚动时其上一点描绘的曲线。以原点为中心、并从最右侧点开始的假轮线的参数方程为：

$$x(t) = (R - r) \cos(t) + r \cos(\frac{R - r}{r}t)$$

$$y(t) = (R - r) \sin(t) - r \sin(\frac{R - r}{r} t)$$

其中 $R$ 是大圆的半径，$r$ 是小圆的半径。

设 $C(R, r)$ 为在半径为 $R$ 和 $r$ 的内轮廓线上的具有整数坐标的不同点的集合，且存在对应的 $t$ 值使得 $\sin(t)$ 和 $\cos(t)$ 是有理数。

令 $S(R, r) = \sum_{(x,y) \in C(R, r)} |x| + |y|$ 为集合 $C(R, r)$ 中点的 $x$ 和 $y$ 坐标绝对值之和。

令 $T(N) = \sum_{R = 3}^N \sum_{r=1}^{\left\lfloor \frac{R - 1}{2} \right\rfloor} S(R, r)$ 为 $S(R, r)$ 在 $R$ 和 $r$ 为正整数，且 $R\leq N$ 且 $2r &lt; R$ 时的总和。

你获得了：

$$\begin{align}
  C(3, 1) = & \\{(3, 0), (-1, 2), (-1,0), (-1,-2)\\} \\\\
  C(2500, 1000) = & \\{(2500, 0), (772, 2376), (772, -2376), (516, 1792), (516, -1792), (500, 0), (68, 504), \\\\
                  &(68, -504),(-1356, 1088), (-1356, -1088), (-1500, 1000), (-1500, -1000)\\}
\end{align}$$

**注意：** (-625, 0) 不是 $C(2500, 1000)$ 的元素，因为对应的 $t$ 值下 $\sin(t)$ 不是有理数。

$S(3, 1) = (|3| + |0|) + (|-1| + |2|) + (|-1| + |0|) + (|-1| + |-2|) = 10$

$T(3) = 10$；$T(10) = 524$；$T(100) = 580\\,442$；$T({10}^3) = 583\\,108\\,600$。

查找 $T({10}^6)$。
