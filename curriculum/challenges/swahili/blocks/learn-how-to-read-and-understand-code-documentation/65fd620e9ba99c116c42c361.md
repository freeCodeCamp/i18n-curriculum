# --description--

Wasanidi programu mara nyingi hutumia `once` kuonyesha wakati wa kutekeleza mazoezi ndani ya mchakato wao wa kazi. Inamaanisha `as soon as` au `when`.

Kwa mfano, `Once you receive the client's feedback, update the project documentation.` Hii ina maana kwamba kusasisha nyaraka kunapaswa kuanza mara moja baada ya kupokea maoni, ikionyesha mfululizo wa hatua katika mzunguko wa maisha wa mradi.
