# --description--

ブートストラップには `btn-default` という別のボタンクラスがあります。

`btn` と `btn-default` の両方のクラスを、すべての `button` 要素に適用してください。

# --hints--

すべての `button` 要素に `btn` クラスを適用してください。

```js
assert.lengthOf(document.querySelectorAll('.btn'),6);
```

すべての `button` 要素に `btn-default` クラスを適用してください。

```js
assert.lengthOf(document.querySelectorAll('.btn-default'), 6);
```
