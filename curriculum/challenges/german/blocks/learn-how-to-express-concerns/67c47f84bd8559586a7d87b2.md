# --explanation--

`To work hard` bedeutet, viel Einsatz zu zeigen. Beachten Sie, dass das Adverb für `hard` in diesem Fall ebenfalls `hard` ist, nicht `hardly`. `To hardly work` bedeutet, fast keine Arbeit zu leisten, wenig bis keinen Einsatz zu zeigen, was das Gegenteil von dem ist, was Maria über das Team sagt. Zum Beispiel:

- `She hardly trained for the marathon.` – Das bedeutet, sie hat keinen Einsatz gezeigt, um sich auf den Marathon vorzubereiten.

- `She trained hard for the marathon.` – Das bedeutet, sie hat viel Einsatz gezeigt, um sich vorzubereiten.
