# --description--

Ще один спосіб порівняння умовних інструкцій — це тернарний оператор. Тернарний оператор — це скорочений спосіб запису інструкції `if-else`. Він використовується для оцінки умови та повернення значення залежно від результату цієї умови. Синтаксис тернарного оператора виглядає так:

```javascript
let result = condition ? value1 : value2;
```

У наведеному прикладі, якщо умова істинна, значення `result` буде `value1`. Інакше значення `result` буде `value2`.

Тернарні оператори часто використовують, щоб призначити значення змінній залежно від умови. Їх також використовують, щоб повернути значення залежно від умови.
