## Was ist Python?

- **Einführung**: Python ist eine universelle Programmiersprache, die für ihre Einfachheit und Benutzerfreundlichkeit bekannt ist. Python wird in vielen Bereichen eingesetzt, wie Datenwissenschaft und maschinelles Lernen, Webentwicklung, Skripting und Automatisierung, eingebettete Systeme und IoT und vieles mehr.
- **Häufige Anwendungsfälle**: Python wird in der Datenwissenschaft, im maschinellen Lernen, in der Webentwicklung, in der Cybersicherheit, in der Automatisierung und bei Mikrocomputern wie dem Raspberry Pi und MicroPython-kompatiblen Boards verwendet.

## Variables

- **Variablen deklarieren**: Um eine Variable zu deklarieren, beginnen Sie mit dem Variablennamen, gefolgt vom Zuweisungsoperator (`=`) und dann dem Wert. Dies kann eine Zahl, ein String, ein Boolean usw. sein. Hier sind einige Beispiele:

```py
name = 'John Doe'
age = 25
```

- **Benennungskonventionen für Variablen**: Hier sind die Benennungskonventionen, die Sie für Variablen verwenden sollten:

  - Variablennamen dürfen nur mit einem Buchstaben oder einem Unterstrich (_) beginnen, nicht mit einer Zahl.
  - Variablennamen dürfen nur alphanumerische Zeichen (a-z, A-Z, 0-9) und Unterstriche (_) enthalten.
  - Variablennamen sind case-sensitive — `age`, `Age` und `AGE` werden alle als einzigartig betrachtet.
  - Variablennamen dürfen keine der reservierten Schlüsselwörter von Python sein, wie `if`, `class` oder `def`.
  - Variablennamen mit mehreren Wörtern werden durch Unterstriche getrennt. Zum Beispiel `snake_case`.


## Comments

- **Einzeilige Kommentare**: Diese Art von Kommentaren sollte für kurze Notizen verwendet werden, die Sie in Ihrem Code hinterlassen möchten.

```py
# This is a single line comment
```

- **Mehrzeilige Strings**: Diese Art von Strings kann verwendet werden, um größere Notizen zu hinterlassen oder um Codeabschnitte auszukommentieren.

```py
"""
This is a multi-line string.
Here is some code commented out.

name = 'John Doe'
age = 25
"""
```

- **`print()` Funktion**: Um Daten in der Konsole auszugeben, können Sie die `print()` Funktion wie folgt verwenden:

```py
print('Hello world!') # Hello world!
```

## Häufige Datentypen in Python

- **Einführung**: Python ist eine dynamisch typisierte Sprache wie JavaScript, was bedeutet, dass Sie die Typen für Variablen nicht explizit deklarieren müssen. Die Sprache weiß, welchen Typ eine Variable hat, basierend darauf, was Sie der Variable zuweisen.
- **Integer**: Eine ganze Zahl ohne Dezimalstellen:

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **Float**: Eine Zahl mit Dezimalstellen:

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **String**: Eine Folge von Zeichen, die in Anführungszeichen eingeschlossen sind:

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **Boolean**: Ein Wert, der entweder `True` oder `False` darstellt:

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **Set**: Eine ungeordnete Sammlung einzigartiger Elemente:

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **Dictionary**: Eine Sammlung von Schlüssel-Wert-Paaren, eingeschlossen in geschweifte Klammern:

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **Tuple**: Eine unveränderliche geordnete Sammlung, eingeschlossen in runde Klammern:

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **Range**: Eine Folge von Zahlen, die oft in Schleifen verwendet wird:

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **List**: Eine geordnete Sammlung von Elementen, die verschiedene Datentypen unterstützen:

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**: Ein spezieller Wert, der das Fehlen eines Werts darstellt:

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## Unveränderliche und veränderliche Typen

- **Unveränderliche Typen**: Diese Typen können nach der Deklaration nicht geändert werden, obwohl Sie ihre Variablen auf etwas Neues zeigen lassen können, was als erneute Zuweisung bezeichnet wird. Dazu gehören Integer, Float, Complex, Boolean, String, Tuple, Range und `None`.
- **Veränderliche Typen**: Diese Typen können nach der Deklaration geändert werden. Sie können ihre Elemente hinzufügen, entfernen oder aktualisieren. Dazu gehören Sammlungstypen wie List, Set und Dictionary.
- **`type()` Funktion**: Um den Typ einer Variable zu sehen, können Sie die `type()` Funktion wie folgt verwenden:

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **`isinstance()` Funktion**: Diese wird verwendet, um zu überprüfen, ob eine Variable einem bestimmten Datentyp entspricht:

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## Arbeiten mit Strings

- **Definition**: Wie Sie sich aus JavaScript erinnern, sind Strings unveränderlich, was bedeutet, dass Sie sie nach ihrer Erstellung nicht ändern können. In Python können Sie entweder einfache oder doppelte Anführungszeichen verwenden. Es wird empfohlen, eine Regel zu wählen und dabei zu bleiben:

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **Zugriff auf Zeichen in Strings**: Sie können auf Zeichen in Strings mit der Klammernotation wie folgt zugreifen:

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **Escape-Sequenzen in Strings**: Sie können einen Backslash (`\`) verwenden, wenn Ihr String Anführungszeichen enthält, wie hier:

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **String-Verkettung**: Um Strings zu verketten, können Sie den `+` Operator wie folgt verwenden:

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

Eine andere Möglichkeit, Strings zu verketten, ist die Verwendung des `+=` Operators. Dieser wird verwendet, um Verkettung und Zuweisung in einem Schritt durchzuführen, wie hier:

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**: Dies ist die Kurzform für formatierte String-Literale. Es ermöglicht Ihnen, Interpolation zu handhaben und auch Verkettung mit einer kompakten und lesbaren Syntax durchzuführen:

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **String-Slicing**: Dabei können Sie Teile eines Strings extrahieren. Hier ist die grundlegende Syntax:

```py
str[start:stop:step]
```

Die Startposition gibt den Index an, an dem die Extraktion beginnen soll. Die Stopp-Position ist das Ende des Slices. Diese Position ist nicht inklusive. Die Schritt-Position gibt das Intervall an, um das beim Slicing inkrementiert wird. Hier sind einige Beispiele:

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **Länge eines Strings ermitteln**: Die `len()` Funktion wird verwendet, um die Anzahl der Zeichen im String zurückzugeben:

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## Arbeiten mit dem `in` Operator

- **`in` Operator**: Dieser gibt einen Boolean zurück, der angibt, ob das Zeichen oder die Zeichen im String vorhanden sind oder nicht:

```py
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

## Häufige String-Methoden

- **`str.upper()`**: Diese gibt einen neuen String zurück, bei dem alle Zeichen in Großbuchstaben umgewandelt wurden:

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`str.lower()`**: Diese gibt einen neuen String zurück, bei dem alle Zeichen in Kleinbuchstaben umgewandelt wurden:

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`str.strip()`**: Diese gibt eine Kopie des Strings zurück, bei der angegebene führende und abschließende Zeichen entfernt wurden (wenn kein Argument an die Methode übergeben wird, werden führende und abschließende Leerzeichen entfernt).

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**: Diese gibt einen neuen String zurück, bei dem alle Vorkommen des alten Strings durch einen neuen ersetzt wurden.

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**: Diese wird verwendet, um einen String mit einem angegebenen Trennzeichen in eine Liste zu teilen. Ein Trennzeichen ist ein String, der angibt, wo die Teilung erfolgen soll.

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**: Diese wird verwendet, um Elemente eines iterierbaren Objekts mit einem Trennzeichen zu einem String zusammenzufügen. Ein iterierbares Objekt ist eine Sammlung von Elementen, über die man wie über eine Liste, einen String oder ein Tuple iterieren kann.

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`str.startswith(prefix)`**: Diese gibt einen Boolean zurück, der angibt, ob ein String mit dem angegebenen Präfix beginnt:

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`str.endswith(suffix)`**: Diese gibt einen Boolean zurück, der angibt, ob ein String mit dem angegebenen Suffix endet:

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`str.find()`**: Diese gibt den Index des ersten Vorkommens eines Substrings zurück. Wenn keiner gefunden wird, wird `-1` zurückgegeben:

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`str.count(substring)`**: Diese zählt, wie oft ein Substring in einem String vorkommt:

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`str.capitalize()`**: Diese gibt einen neuen String zurück, bei dem das erste Zeichen großgeschrieben und die anderen Zeichen kleingeschrieben sind:

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`str.isupper()`**: Diese gibt `True` zurück, wenn alle Buchstaben im String Großbuchstaben sind, und `False`, wenn nicht:

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`str.islower()`**: Diese gibt `True` zurück, wenn alle Buchstaben im String Kleinbuchstaben sind, und `False`, wenn nicht:

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`str.title()`**: Diese gibt einen neuen String zurück, bei dem der erste Buchstabe jedes Wortes großgeschrieben ist:

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`str.maketrans()`**: Diese Methode wird verwendet, um eine Tabelle mit 1-zu-1-Zeichenzuordnungen für die Übersetzung zu erstellen. Sie wird oft mit der `translate()` Methode verwendet, die diese Tabelle auf einen String anwendet und das übersetzte Ergebnis zurückgibt.

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## Häufige Operationen mit ganzen Zahlen und Fließkommazahlen

- **Grundlegende mathematische Operationen**: In Python können Sie grundlegende mathematische Operationen mit ganzen Zahlen und Fließkommazahlen durchführen, einschließlich Addition, Subtraktion, Multiplikation und Division:

```py
int_1 = 56
int_2 = 12
float_1 = 5.4
float_2 = 12.0

# Addition

print('Integer Addition:', int_1 + int_2) # Integer Addition: 68
print('Float Addition:', float_1 + float_2) # Float Addition: 17.4

# Subtraction

print('Int Subtraction:', int_1 - int_2) # Int Subtraction: 44
print('Float Subtraction:',  float_2 - float_1) # Float Subtraction: 6.6

# Multiplication

print('Int Multiplication:', int_1 * int_2) # Int Multiplication: 672
print('Float Multiplication:', float_2 * float_1) # Float Multiplication: 64.80000000000001

# Division

print('Division:', int_1 / int_2) # Division: 4.666666666666667
print('Float Division:', float_2 / float_1) # Float Division: 2.222222222222222
```

Wenn Sie eine Fließkommazahl und eine ganze Zahl addieren, wird das Ergebnis in eine Fließkommazahl umgewandelt, wie hier:

```py
int_1 = 56
float_1 = 5.4

print(int_1 + float_1) # 61.4
```

- **Modulo-Operator (`%`)**: Dieser gibt den Rest zurück, wenn eine Zahl durch eine andere Zahl geteilt wird:

```py
int_1 = 56
int_2 = 12

print(int_1 % int_2) # 8
```

- **Ganzzahldivision (`//`)**: Dieser Operator wird verwendet, um zwei Zahlen zu dividieren und das Ergebnis auf die nächste ganze Zahl abzurunden:

```py
int_1 = 56
int_2 = 12

print(int_1 // int_2) # 4
```

- **Exponentiation-Operator (`**`)**: Dieser Operator wird verwendet, um eine Zahl mit der Potenz einer anderen zu potenzieren:

```py
int_1 = 4
int_2 = 2

print(int_1 ** int_2) # 16
```

- **`float()` Funktion**: Sie können diese Funktion verwenden, um eine ganze Zahl in eine Fließkommazahl umzuwandeln.

```py
num = 4

print(float(num)) # 4.0
```

- **`int()` Funktion**: Sie können diese Funktion verwenden, um eine Fließkommazahl in eine ganze Zahl umzuwandeln.

```py
num = 4.0

print(int(num)) # 4
```

- **`round()` Funktion**: Diese wird verwendet, um eine Zahl auf die nächste ganze Zahl zu runden:

```py
num_1 = 3.4
num_2 = 7.7

print(round(num_1)) # 3
print(round(num_2)) # 8
```

- **`abs()` Funktion**: Diese wird verwendet, um den Absolutwert einer Zahl zurückzugeben:

```py
num = -13

print(abs(num)) # 13
```

- **`pow()` Funktion**: Diese wird verwendet, um eine Zahl mit der Potenz einer anderen zu potenzieren:

```py
result = pow(2, 3) 
print(result)  # 8
```

## Augmented Assignments

- **Definition**: Erweiterte Zuweisungen kombinieren eine binäre Operation mit einer Zuweisung in einem Schritt. Sie nehmen eine Variable, wenden eine Operation mit einem anderen Wert darauf an und speichern das Ergebnis wieder in derselben Variable.

```py
# Addition assignment 
my_var = 10
my_var += 5

print(my_var) # 15

# Subtraction assignment
count = 14
count -= 3

print(count) # 11

# Multiplication assignment 
product = 65
product *= 7

print(product) # 455

# Division assignment 
price = 100
price /= 4

print(price) # 25.0

# Floor Division assignment 
total_pages = 23
total_pages //= 5

print(total_pages) # 4

# Modulo assignment 
bits = 35
bits %= 2

print(bits) # 1

# Exponentiation assignment 
power = 2
power **= 3

print(power) # 8
```

Es gibt auch andere erweiterte Zuweisungsoperatoren, wie die für bitweise Operatoren. Dazu gehören `&=`, `^=`, `>>=` und `<<=`.

## Arbeiten mit Funktionen

- **Definition**: Funktionen sind wiederverwendbare Codeabschnitte, die Eingaben (Argumente) entgegennehmen und eine Ausgabe zurückgeben. Um eine Funktion aufzurufen, müssen Sie den Funktionsnamen gefolgt von einem Satz runder Klammern referenzieren:

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call
print(result) # 7
```

Wenn eine Funktion keinen expliziten Rückgabewert hat, ist der Standard-Rückgabewert `None`:

```py
def greet():
    print('hello') 

result = greet() # hello
print(result) # None
```

Sie können auch Standardwerte für Parameter wie folgt angeben:

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

Wenn Sie die Funktion ohne die korrekte Anzahl von Argumenten aufrufen, erhalten Sie einen `TypeError`:

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

## Gültigkeitsbereich in Python

- **Lokaler Gültigkeitsbereich**: Dies ist, wenn eine Variable, die innerhalb einer Funktion oder Klasse deklariert wurde, nur innerhalb dieser Funktion oder Klasse aufgerufen werden kann.

```py
def my_func():
    num = 10
    print(num)
```

- **Einschließender Gültigkeitsbereich**: Dies ist, wenn eine Funktion, die innerhalb einer anderen Funktion verschachtelt ist, auf die Variablen der umschließenden Funktion zugreifen kann.

```py
def outer_func():
    msg = 'Hello there!'
    
    def inner_func():
        print(msg)
    inner_func()

print(outer_func()) # Hello there!
```

- **Globaler Gültigkeitsbereich**: Dies bezieht sich auf Variablen, die außerhalb von Funktionen oder Klassen deklariert wurden und von überall im Programm aufgerufen werden können.

```py
tax = 0.70 

def get_total(subtotal):
    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **Eingebauter Gültigkeitsbereich**: Reservierte Namen in Python für vordefinierte Funktionen, Module, Schlüsselwörter und Objekte.

```py
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```

## Arbeiten mit `if`, `elif` und `else` Anweisungen

- **`if` Anweisungen**: Dies sind Bedingungen, die verwendet werden, um zu bestimmen, ob etwas wahr ist oder nicht. Wenn die Bedingung zu `True` ausgewertet wird, wird dieser Codeblock ausgeführt.

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **`elif` Klausel**: Dies sind Bedingungen, die nach einer `if` Anweisung kommen. Eine `elif` Klausel wird nur ausgeführt, wenn alle vorherigen Bedingungen zu `False` ausgewertet wurden und ihre eigene Bedingung zu `True` ausgewertet wird.

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **`else` Klausel**: Diese wird ausgeführt, wenn keine anderen Bedingungen zu `True` ausgewertet werden.

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

Sie können auch verschachtelte `if` Anweisungen wie folgt verwenden:

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## Truthy- und falsy-Werte

- **Definition**: In Python hat jeder Wert einen inhärenten Boolean-Wert oder ein eingebautes Verständnis dafür, ob er in einem logischen Kontext als `True` oder `False` behandelt werden sollte. Viele Werte gelten als truthy, das heißt, sie werden in einem logischen Kontext zu `True` ausgewertet. Andere sind falsy, was bedeutet, dass sie zu `False` ausgewertet werden. Hier sind einige Beispiele für falsy-Werte:

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

Andere Werte wie Nicht-Null-Zahlen und nicht-leere Strings sind truthy.

## Arbeiten mit der `bool()` Funktion

- **Definition**: Wenn Sie überprüfen möchten, ob ein Wert truthy oder falsy ist, können Sie die eingebaute `bool()` Funktion verwenden. Sie wandelt einen Wert explizit in sein Boolean-Äquivalent um und gibt `True` für truthy-Werte und `False` für falsy-Werte zurück. Hier sind einige Beispiele:

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## Boolean-Operatoren und Kurzschlussauswertung

- **Definition**: Dies sind spezielle Operatoren, mit denen Sie mehrere Ausdrücke kombinieren können, um komplexere Entscheidungslogiken in Ihrem Code zu erstellen. Es gibt drei Boolean-Operatoren in Python: `and`, `or` und `not`.
- **`and` Operator**: Dieser Operator nimmt zwei Operanden und gibt den ersten Operand zurück, wenn er falsy ist, andernfalls gibt er den zweiten Operand zurück. Beide Operanden müssen truthy sein, damit ein Ausdruck zu einem truthy-Wert führt.

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

Sie können den `and` Operator auch in Bedingungen wie folgt verwenden:

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **`or` Operator**: Dieser Operator gibt den ersten Operand zurück, wenn er truthy ist, andernfalls den zweiten Operand. Ein `or` Ausdruck führt zu einem truthy-Wert, wenn mindestens ein Operand truthy ist. Hier ist ein Beispiel:

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

Wie beim `and` Operator können Sie den `or` Operator in Bedingungen wie folgt verwenden:

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **Kurzschlussauswertung**: Die `and` und `or` Operatoren sind als Kurzschlussoperatoren bekannt. Kurzschlussauswertung bedeutet, dass Python die Werte von links nach rechts überprüft und stoppt, sobald das endgültige Ergebnis bestimmt ist.
- **`not` Operator**: Dieser Operator nimmt einen einzelnen Operanden und invertiert dessen Boolean-Wert. Er wandelt truthy-Werte in `False` und falsy-Werte in `True` um. Im Gegensatz zu den vorherigen Operatoren gibt `not` immer `True` oder `False` zurück. Hier sind einige Beispiele:

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

Hier ist ein Beispiel für den `not` Operator in einer Bedingung:

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```
