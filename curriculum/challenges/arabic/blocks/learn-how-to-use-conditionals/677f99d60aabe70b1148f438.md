# --explanation--

غالبًا ما يُستخدم زمن `Present Simple` لوصف الحالات أو الحقائق أو الظروف التي تكون صحيحة بشكل عام. يُستخدم هذا الزمن لأن تلف الـ BIOS يُعتبر حالة مستقرة أو غير متغيرة عندما يحاول الكمبيوتر الإقلاع.

`Is corrupted` تعني تالف أو تغير بطريقة تجعله يتوقف عن العمل بشكل صحيح. على سبيل المثال:

`The file is corrupted and won't open.` - الملف تالف ولا يمكن الوصول إليه.
