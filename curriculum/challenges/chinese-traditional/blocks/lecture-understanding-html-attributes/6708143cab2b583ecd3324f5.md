# --interactive--

屬性是放置在 HTML 元素起始標記內的值。屬性提供關於該元素的額外資訊或指定該元素應如何運作。以下是屬性的基本語法：

```html
<element attribute="value"></element>
```

屬性名稱後面接著一個等號（`=`）和用引號括起來的值。該值可以是字串或數字，取決於屬性。

這個第一個範例使用了 `href` 和 `target` 屬性。`href` 屬性指定聯結的 URL，`target` 屬性指定在哪裡開啟聯結。

啟用互動式編輯器並將 `href="https://www.freecodecamp.org/news/"` 更改為 `href="https://www.freecodecamp.org"`。現在當你在互動式編輯器中點擊該連結時，你將會在新的瀏覽器分頁中看到 freeCodeCamp 首頁。

:::interactive_editor

```html
<a href="https://www.freecodecamp.org/news/" target="_blank">Visit freeCodeCamp</a>
```

:::

如果沒有 `href` 屬性，連結將無法運作，因為沒有標的 URL。所以你必須包含這個 `href` 屬性才能使連結具有功能。`target="_blank"` 會啟用連結在新的瀏覽器分頁中開啟。你將在未來的課程中學習更多關於 `target` 屬性的內容。

其他常見的屬性有 `src` 和 `alt`，或稱替代屬性－分別用於指定影像的來源和提供影像的替代描述文字。 

啟用互動式編輯器，並將 `src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"` 改為 `src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/running-cats.jpg"`。然後將 `alt="Two tabby kittens sleeping together on a couch."` 改為 `alt="Two cats running in the dirt."`。

:::interactive_editor

```html
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" alt="Two tabby kittens sleeping together on a couch." />
```

:::

類似於 `href` 屬性，`src` 屬性是必需的，因為它指定要顯示的圖片檔案。`alt` 屬性不是必需的，但建議為了存取性目的使用。存取性是指確保每個人，包括有障礙者，都能使用並理解像是網站、應用程式和實體空間等事物。你將在接下來的課程中學習更多關於存取性的內容。

有些屬性的語法有點獨特，例如 `checked` 屬性。

啟用互動式編輯器，並嘗試在預覽視窗中點擊核取方塊，以查看其在已勾選和未勾選狀態之間切換。

:::interactive_editor

```html
<input type="checkbox" checked />
```

:::

在以下範例中，我們有一個 `input` 元素，其 `type` 屬性設定為 `checkbox`。`input` 用於從使用者收集資料，而 `type` 屬性指定輸入的型別。在此情況下，此輸入為核取方塊。你將在接下來的課程中學習更多關於 `input` 的運作方式。 

`checked` 屬性用於指定核取方塊預設應該被勾選。`checked` 屬性不需要值。若存在，核取方塊將預設被勾選。若屬性不存在，核取方塊將未被勾選。這稱為布林值屬性。當你進入 JavaScript 部分時，將會學習更多關於布林值的內容。 

啟用互動式編輯器並嘗試移除 `input` 的 `checked` 屬性。你會看到核取方塊不再是預設勾選。 

:::interactive_editor

```html
<input type="checkbox" checked />
```

:::

在 HTML 中，你會遇到幾個常見的布林值屬性，例如 `disabled`、`readonly` 和 `required`。這些屬性用來指定元素的狀態，例如它是否被停用、唯讀或必填。

這是一個預設停用的文字 `input` 元素範例。啟用互動式編輯器並嘗試在預覽視窗中點擊 `input` 元素。現在從 `input` 元素中移除 `disabled` 屬性，你會看到該 `input` 不再是預設停用。你現在應該可以點擊它並在欄位中輸入文字。

:::interactive_editor

```html
<input type="text" disabled>
```

:::

HTML 有許多屬性可以用來自訂網頁上元素的行為和外觀。了解如何使用屬性對於建立互動且可存取的網頁內容至關重要。在接下來的幾堂課中，你將學習更多 HTML 屬性以及如何在你的網頁開發專案中有效地使用它們。
