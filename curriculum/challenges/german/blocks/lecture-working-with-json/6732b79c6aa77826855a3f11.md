# --interactive--

Es gibt zwei leistungsstarke Methoden in JavaScript zum Umgang mit JSON-Daten: `JSON.parse()` und `JSON.stringify()`. Diese Methoden werden häufig verwendet, um zwischen JSON-Strings und JavaScript-Objekten zu konvertieren.

`JSON.stringify()` wird verwendet, um ein JavaScript-Objekt in einen JSON-String umzuwandeln. Das ist nützlich, wenn Sie Daten in einem Format speichern oder übertragen möchten, das leicht geteilt oder zwischen Systemen übertragen werden kann.

So können Sie die `JSON.stringify()`-Methode verwenden:

:::interactive_editor

```js
const user = {
  name: "John",
  age: 30,
  isAdmin: true
};

const jsonString = JSON.stringify(user);
console.log(jsonString);
```

:::

Die `JSON.stringify()`-Methode akzeptiert auch einen optionalen Parameter namens replacer, der eine Funktion oder ein Array sein kann. Hier ist ein Beispiel, wie man ein Array für den optionalen replacer-Parameter verwendet:

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

// result: {"firstName":"Jessica","country":"USA"}
console.log(JSON.stringify(developerObj, ["firstName", "country"]));
```

:::

In diesem Beispiel haben wir ein `developerObj` mit vier Eigenschaften. Wenn wir die `JSON.stringify()`-Methode verwenden, können wir ein Array als zweiten Parameter übergeben und angeben, welche Eigenschaften wir stringifizieren möchten. Das Ergebnis ist ein stringifiziertes Objekt, das nur die `firstName`- und `country`-Eigenschaften enthält.

Ein weiterer optionaler Parameter für die `JSON.stringify()`-Methode ist der spacer-Parameter. Damit können Sie den Abstand für das stringifizierte Ergebnis steuern:

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

console.log(JSON.stringify(developerObj, null, 2));

/* result
{
  "firstName": "Jessica",
  "isAwesome": true,
  "isMusician": true,
  "country": "USA"
}
*/
```

:::

Meistens werden Sie diese optionalen Parameter für die `JSON.stringify()`-Methode nicht verwenden, aber es ist trotzdem hilfreich, sie zu kennen.

Eine weitere Methode, die Sie in Ihrem Programmieren oft verwenden werden, ist die `JSON.parse()`-Methode. `JSON.parse()` wandelt einen JSON-String zurück in ein JavaScript-Objekt um. Das ist nützlich, wenn Sie JSON-Daten von einem Webserver oder von `localStorage` abrufen und die Daten in Ihrer Anwendung bearbeiten müssen. Mehr über `localStorage` lernen Sie in einer zukünftigen Lektion.

Hier ist ein Beispiel, wie Sie mit der `JSON.parse()`-Methode arbeiten:

:::interactive_editor

```js
const jsonString = '{"name":"John","age":30,"isAdmin":true}';
const userObject = JSON.parse(jsonString);
console.log(userObject);

// Result:
// { name: 'John', age: 30, isAdmin: true }
```

:::

So können Sie mit den Daten in Ihrem Programm wie mit einem normalen JavaScript-Objekt arbeiten, was die Bearbeitung und Nutzung erleichtert.

In zukünftigen Modulen werden wir weiterhin mehr darüber lernen, wie man mit JSON und den `JSON.parse()`- und `JSON.stringify()`-Methoden arbeitet.
