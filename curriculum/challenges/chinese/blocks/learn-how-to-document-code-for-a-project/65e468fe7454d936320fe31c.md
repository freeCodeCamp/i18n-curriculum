# --description--

动词 `to contain` 意味着在内部包含或拥有某物。在文档或 README 文件的上下文中，`to contain` 指的是包含在文档内的信息。

例如，瓶子可以 `contain` 水，就像 README `file` `contains` 关于一个 `project` 的信息。
