# --explanation--

Un `for loop` è utile quando sai quante volte il loop deve essere eseguito. Per esempio:

- `A for loop is useful for going through a list.` - Puoi usarlo per controllare ogni elemento in una lista.

- `The for loop stops when it reaches the last item.` - Il loop termina quando non ci sono più elementi.

Un `while loop` è meglio quando il numero di ripetizioni dipende da una condizione. Per esempio:

- `A while loop runs until a condition is false.` - Il loop continua finché la regola è vera.

- `Be careful, a while loop can run forever!` - Se non lo fermi, il loop non finirà.
