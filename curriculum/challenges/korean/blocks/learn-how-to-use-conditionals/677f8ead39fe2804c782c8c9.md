# --explanation--

Sarah는 Mark가 일부 컴퓨터가 `booting properly` 문제를 알아차렸는지 묻고 있습니다.

`Have you noticed`은 Sarah가 Mark가 문제를 인지하고 있는지 확인하는 것이지, 고치거나 조치를 취하라고 하는 것이 아님을 보여줍니다.

`Booting`는 운영체제를 로드하여 컴퓨터를 시작하는 것을 의미합니다. 예를 들어:

- `Have you noticed the network is running slowly?` - 누군가가 문제를 관찰했는지 묻는 표현입니다.

- `The laptop isn't booting properly; we need to check the operating system.` - 컴퓨터가 제대로 시작하지 못하는 흔한 문제를 강조합니다.
