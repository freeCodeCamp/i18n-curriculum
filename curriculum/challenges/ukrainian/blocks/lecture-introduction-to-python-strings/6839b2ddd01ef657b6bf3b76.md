# --description--

Python надає низку вбудованих методів, які значно полегшують роботу з рядками. Вони включають, але не обмежуються, такими:

- `upper()`: Повертає новий рядок, у якому всі символи перетворені на верхній регістр.
    

```python
my_str = 'hello world'

uppercase_my_str = my_str.upper()
print(uppercase_my_str)  # HELLO WORLD
```

- `lower()`: Повертає новий рядок, у якому всі символи перетворені на нижній регістр.
    

```python
my_str = 'Hello World'

lowercase_my_str = my_str.lower()
print(lowercase_my_str)  # hello world
```

- `strip()`: Повертає новий рядок, у якому видалені вказані початкові та кінцеві символи. Якщо аргумент не передано, видаляє початкові та кінцеві пробіли.
    

```python
my_str = '  hello world  '

trimmed_my_str = my_str.strip()
print(trimmed_my_str)  # "hello world"
```

- `replace(old, new)`: Повертає новий рядок, у якому всі входження `old` замінені на `new`.
    

```python
my_str = 'hello world'

replaced_my_str = my_str.replace('hello', 'hi')
print(replaced_my_str)  # hi world
```

- `split(separator)`: Розбиває рядок за вказаним роздільником на список рядків. Якщо роздільник не вказано, розбиває за пробілами.
    

```python
my_str = 'hello world'

split_words = my_str.split()
print(split_words)  # ['hello', 'world']
```

- `join(iterable)`: Об’єднує елементи ітерованого об’єкта в рядок із роздільником.
    

```python
my_list = ['hello', 'world']

joined_my_str = ' '.join(my_list)
print(joined_my_str)  # hello world
```

- `startswith(prefix)`: Повертає булеве значення, яке вказує, чи починається рядок із вказаного префікса.
    

```python
my_str = 'hello world'

starts_with_hello = my_str.startswith('hello')
print(starts_with_hello)  # True
```

- `endswith(suffix)`: Повертає булеве значення, яке вказує, чи закінчується рядок вказаним суфіксом.
    

```python
my_str = 'hello world'

ends_with_world = my_str.endswith('world')
print(ends_with_world)  # True
```

- `find(substring)`: Повертає індекс першого входження `substring`, або `-1`, якщо такого не знайдено.
    

```python
my_str = 'hello world'

world_index = my_str.find('world')
print(world_index)  # 6
```

- `count(substring)`: Повертає кількість разів, скільки підрядок зустрічається в рядку.
    

```python
my_str = 'hello world'

o_count = my_str.count('o')
print(o_count)  # 2
```

- `capitalize()`: Повертає новий рядок, у якому перша літера велика, а всі інші — у нижньому регістрі.
    

```python
my_str = 'hello world'

capitalized_my_str = my_str.capitalize()
print(capitalized_my_str)  # Hello world
```

- `isupper()`: Повертає `True`, якщо всі літери в рядку у верхньому регістрі, і `False`, якщо ні.
    

```python
my_str = 'hello world'

is_all_upper = my_str.isupper()
print(is_all_upper)  # False
```

- `islower()`: Повертає `True`, якщо всі літери в рядку у нижньому регістрі, і `False`, якщо ні.
    

```python
my_str = 'hello world'

is_all_lower = my_str.islower()
print(is_all_lower)  # True
```

- `title()`: Повертає новий рядок, у якому перша літера кожного слова велика.
    

```python
my_str = 'hello world'

title_case_my_str = my_str.title()
print(title_case_my_str)  # Hello World
```
