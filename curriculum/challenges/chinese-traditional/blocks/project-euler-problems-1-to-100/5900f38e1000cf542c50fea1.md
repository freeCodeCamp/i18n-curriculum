# --description--

145 是一個有趣的數字，因為 1! + 4! + 5! = 1 + 24 + 120 = 145。

找出數字以及這些數字的總和，該總和相等於其各位數字階乘的總和。

**注意：** 由於 1! = 1 和 2! = 2 不是總和，因此不包含在內。

# --hints--

`digitFactorial()` 應該傳回一個物件。

```js
assert.typeOf(digitFactorial(), 'object');
```

`digitFactorial()` 應該傳回 { sum: 40730, numbers: [145, 40585] }。

```js
assert.deepEqual(digitFactorial(), { sum: 40730, numbers: [145, 40585] });
```
