# --description--

Cadenas definidas usando comillas invertidas se llaman literales plantilla. Son una nueva manera de definir cadenas en JavaScript. Permiten insertar expresiones dentro de la cadena. Esto se hace envolviendo la expresión en `${}`.

Por ejemplo, el siguiente código:

```javascript
let name = "John";
let age = 25;
let greeting = `Hello, my name is ${name} and I am ${age} years old.`;
```

resultará en la variable `greeting` conteniendo la cadena "Hola, mi nombre es John y tengo 25 años."
