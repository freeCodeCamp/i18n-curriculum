# --interactive--

## L'oggetto `date()` e i metodi comuni

- **Definizione**: L'oggetto `date()` serve a creare, manipolare e formattare date e orari in JavaScript. Nell'esempio seguente, la parola chiave `new` viene usata per creare una nuova istanza dell'oggetto `Date`, e l'oggetto `Date` viene poi assegnato alla variabile `now`. Se registrassi il valore di `now` nella console, vedresti la data e l'ora correnti basate sull'orologio di sistema del computer che esegue il codice.

:::interactive_editor

```js
const now = new Date();
console.log(now); // prints the current date and time
```

:::

- **Metodo `Date.now()`**: Questo metodo serve per ottenere la data e l'ora correnti. `Date.now()` restituisce il numero di millisecondi trascorsi dal 1° gennaio 1970, 00:00:00 UTC. Questo è noto come tempo epoch Unix. Il tempo epoch Unix è un modo comune per rappresentare date e orari nei sistemi informatici perché è un intero che può essere facilmente archiviato e manipolato. UTC sta per Universal Time Coordinated, che è lo standard temporale principale con cui il mondo regola orologi e tempo.
- **Metodo `getDate()`**: Questo metodo serve per ottenere il giorno del mese basato sulla data corrente. `getDate()` restituirà un valore intero tra 1 e 31, a seconda del giorno del mese. Se la data non è valida, restituirà `NaN` (Not a Number).

:::interactive_editor

```js
const now = new Date("2014-10-15");
const date = now.getDate();
console.log(date); // 15
```

:::

- **Metodo `getMonth()`**: Questo metodo serve per ottenere il mese. Il mese è a base zero, quindi gennaio è 0, febbraio è 1 e così via. In questo esempio, l'output è 9, che corrisponde a ottobre. Se il mese non è valido, restituirà `NaN`.

:::interactive_editor

```js
const now = new Date("2014-10-15");
const month = now.getMonth();
console.log(month); // 9
```

:::

- **Metodo `getFullYear()`**: Questo metodo serve per ottenere l'anno completo. Se l'anno non è valido, restituirà `NaN`.

:::interactive_editor

```js
const now = new Date("2014-10-15");
const year = now.getFullYear();
console.log(year); // 2014
```

:::

## Modi diversi per formattare le date

- **Metodo `toISOString()`**: Questo metodo serve per formattare la data in un formato esteso `ISO` (ISO 8601). ISO 8601 è uno standard internazionale per rappresentare date e orari. Il formato è `YYYY-MM-DDTHH:mm:ss.sssZ`.

:::interactive_editor

```js
const date = new Date("2014-10-15");
console.log(date.toISOString()); // "2014-10-15T00:00:00.000Z"
```

:::

- **Metodo `toLocaleDateString()`**: Questo metodo serve per formattare la data in base alla localizzazione dell'utente.

:::interactive_editor

```js
const date = new Date("2014-10-15");
console.log(date.toLocaleDateString("en-US"));  // "10/15/2014"
```

:::

Il metodo `toLocaleDateString()` accetta due parametri opzionali: locales e options.

Il parametro locales è una stringa che rappresenta la localizzazione da usare. Per esempio, puoi passare `"en-US"` per l'inglese (Stati Uniti) o `"fr-FR"` per il francese (Francia). Se non passi un parametro locales, viene usata la localizzazione predefinita. Il secondo parametro opzionale è options. Questo parametro è un oggetto che ti permette di specificare il formato della stringa della data.

:::interactive_editor

```js
const date = new Date("2014-10-15");
const options = {
  weekday: "long",
  year: "numeric",
  month: "long",
  day: "numeric",
};
console.log(date.toLocaleDateString("en-GB", options)); // "Wednesday, 15 October 2014"
```

:::
