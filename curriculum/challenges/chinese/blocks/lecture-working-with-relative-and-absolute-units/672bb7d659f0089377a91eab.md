# --interactive--

CSS 中的百分比是相对单位，允许你将大小、维度和其他属性定义为其父元素的比例。当你使用百分比值时，实际上是在说，“将此设置为其容器的 X%。”

例如，如果你在一个元素上设置了 `width: 50%`，它将占据其父容器宽度的一半。这使得百分比在创建适应不同屏幕尺寸的响应式设计时非常有用。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <div class="box"></div>
</div>
```

```css
.container {
  width: 400px;
  height: 200px;
  background-color: lightgray;
}

.box {
  width: 50%;
  height: 100%;
  background-color: red;
}
```

:::

百分比非常适合创建可根据各种屏幕尺寸调整的流体版型。例如，将容器设置为 `width: 80%` 可确保它占据其父级宽度的 80% ，无论设备如何。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="parent">
  <div class="child"></div>
</div>
```

```css
.parent {
  width: 100%;
  height: 300px;
  background-color: lightblue;
}

.child {
  width: 80%;
  height: 100%;
  background-color: red;
}
```

:::

使用百分比来处理灵活的图像是另一种常见做法。通过为图像应用 `max-width: 100%`，你可以让它们在较小的屏幕上缩小，同时维护其宽高比。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<img src="https://placehold.co/150x150" alt="Example Product Image" />
```

```css
img {
  max-width: 100%;
  height: auto;
}
```

:::

虽然不太常见，百分比也可以用于字体大小，以创建可缩放的排版。例如，`font-size: 120%` 会使文本比其父元素的字体大小大 20%。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="text-container">
  Parent text.
  <p class="text">This is some example text.</p>
</div>
```

```css
.text-container {
  font-size: 16px;
}

.text {
  font-size: 120%;
}
```

:::

百分比在垂直居中时特别有用。以下是如何将百分比与 `transform` 属性结合使用以垂直居中一个元素的示例。

此示例将元素定位在其容器顶部的 50％ 处，然后使用 `transform` 将其向上移动自身高度的一半，从而实现垂直居中。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="centered"></div>
```

```css
.centered {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
  width: 300px;
  height: 300px;
  background-color: red;
}
```

:::

你将在后续课程中更详细地学习绝对定位和 transform 属性的工作原理。 

请记住，百分比总是相对于某个东西。对于像 `width` 这样的水平属性，它们相对于父元素的宽度。对于像 `height` 这样的垂直属性，它们通常相对于父元素的高度（如果已指定）。

但是，当嵌套具有百分比维度的元素时要小心，因为这可能导致意外的结果。同时，请记住，如果父元素没有定义高度，基于百分比的高度可能会很棘手。

总之，CSS 中的百分比是创建灵活响应式设计的强大工具。当你希望元素相对于它们的容器按比例缩放，或需要版型能够无缝适应不同屏幕尺寸时，请使用它们。
