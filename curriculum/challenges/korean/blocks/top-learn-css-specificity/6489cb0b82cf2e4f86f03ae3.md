# --description--

특이성이 어떻게 작동하는지 시각적으로 이해할 수 있도록 몇 가지 간단한 예를 살펴보겠습니다. 다음 HTML과 CSS 코드를 참고하세요:

```html
<!-- index.html -->

<div class="main">
  <div class="list subsection"></div>
</div>
```

```css
/* rule 1 */
.subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

위 예제에서 두 규칙 모두 클래스 선택자만 사용하지만, 규칙 2가 더 많은 클래스 선택자를 사용하기 때문에 더 특이성이 높습니다.
