# --description--

Viele Berufe werden gebildet, indem man `-er` an das Ende eines Wortes anhängt, das beschreibt, was die Person tut.

Zum Beispiel ist ein `programmer` jemand, der `programs`, ein `developer` ist jemand, der `develops` Software oder Systeme. Ein weiteres Beispiel ist ein `designer`, der eine Person ist, die Websites, Grafiken oder Produkte `designs`.

Indem Sie `-er` an ein Verb anhängen, erstellen Sie ein Substantiv, das den Beruf einer Person beschreibt. Das gilt nicht für alle Berufe, aber Sie werden dieses Muster bei vielen sehen.
