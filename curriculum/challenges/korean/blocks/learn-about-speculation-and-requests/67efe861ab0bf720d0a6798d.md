# --explanation--

`Could`는 가능성을 나타냅니다. 확신하지는 않지만 가능한 원인 중 하나를 제시하는 뜻입니다.

`Unpatched`는 무언가가 고쳐지거나 업데이트되지 않았음을 의미합니다. 예를 들어:

`The system was attacked because of an unpatched vulnerability.` – 시스템의 알려진 문제가 수정되지 않아 공격이 발생했다는 뜻입니다.
