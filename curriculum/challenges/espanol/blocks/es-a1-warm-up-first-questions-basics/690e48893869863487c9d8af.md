# --description--

Ángela está preguntando por la edad de alguien. Ella comienza su pregunta con una palabra que literalmente significa "cuántos" en español.

# --instructions--

Escucha el audio y responde a la pregunta de abajo.

# --questions--

## --text--

¿Qué palabra está usando Ángela para comenzar su pregunta?

## --answers--

`¿Qué…?`

### --feedback--

La palabra `Qué` significa "What". No se usa para preguntar la edad de alguien.

---

`¿Cómo…?`

### --feedback--

La palabra `Cómo` significa "How". No se usa para preguntar la edad de alguien.

---

`¿Cuántos…?`

---

`¿Cuál…?`

### --feedback--

La palabra `Cuál` significa "Which" o "What" (cuando se selecciona entre opciones). No se usa para preguntar la edad de alguien.

## --video-solution--

3

# --explanation--

La palabra `Cuántos` se traduce como "How many" y se usa para preguntar sobre una cantidad.

Debe llevar una tilde en la letra `a` cuando forma parte de una pregunta.
