# --description--

Eine Harshad- oder Niven-Zahl ist eine Zahl, die durch die Summe ihrer Ziffern teilbar ist.

201 ist eine Harshad-Zahl, weil sie durch 3 (die Summe ihrer Ziffern) teilbar ist.

Wenn wir die letzte Ziffer von 201 abschneiden, erhalten wir 20, was ebenfalls eine Harshad-Zahl ist.

Wenn wir die letzte Ziffer von 20 abschneiden, erhalten wir 2, was auch eine Harshad-Zahl ist.

Nennen wir eine Harshad-Zahl, die beim rekursiven Abschneiden der letzten Ziffer immer eine Harshad-Zahl ergibt, eine rechts abschneidbare Harshad-Zahl.

Außerdem gilt:

$\frac{201}{3} = 67$, was eine Primzahl ist.

Nennen wir eine Harshad-Zahl, die, wenn man sie durch die Summe ihrer Ziffern teilt, eine Primzahl ergibt, eine starke Harshad-Zahl.

Nehmen wir nun die Zahl 2011, die eine Primzahl ist. Wenn wir die letzte Ziffer abschneiden, erhalten wir 201, eine starke Harshad-Zahl, die außerdem rechts abschneidbar ist. Solche Primzahlen nennen wir starke, rechts abschneidbare Harshad-Primzahlen.

Es ist gegeben, dass die Summe der starken, rechts abschneidbaren Harshad-Primzahlen unter 10000 gleich 90619 ist.

Finden Sie die Summe der starken, rechts abschneidbaren Harshad-Primzahlen unter ${10}^{14}$.
