# --description--

`div` 元素開始看起來比較好。但現在關閉 `button` 有點太靠近段落元素。 

為你的 flex 項目之間添加間距，你可以使用以下樣式：`gap-number`。`number` 表示 flex 佈局中列與行之間的固定間距。

以下是將 `gap` 應用於元素的範例：

```html
<p class="flex gap-8">This is gap.</p>
```

在你的開啟 `div` 頁籤中，使用正確的工具類別來將 `gap` 設定為 `4`。

# --hints--

你的開啟 `div` 頁籤應該有 `gap-4` 實用類別。

```js
const divEl = document.querySelector("div");
assert.isTrue(divEl.classList.contains("gap-4"));
```
