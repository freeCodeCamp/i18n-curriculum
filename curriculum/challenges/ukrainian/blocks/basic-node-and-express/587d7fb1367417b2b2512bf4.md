# --description--

Підпрограмне забезпечення можна підключати до конкретного маршруту за допомогою `app.METHOD(path, middlewareFunction)`. Також підпрограмне забезпечення можна зв’язувати в ланцюжок у визначенні маршруту.

Подивіться на наступний приклад:

```js
app.get('/user', function(req, res, next) {
  req.user = getTheUserSync();  // Hypothetical synchronous operation
  next();
}, function(req, res) {
  res.send(req.user);
});
```

Такий підхід корисний для розбиття операцій сервера на менші частини. Це призводить до кращої структури застосунку та можливості повторно використовувати код у різних місцях. Цей підхід також можна використовувати для виконання валідації даних. На кожному кроці стеку підпрограмного забезпечення ви можете зупинити виконання поточного ланцюжка і передати керування функціям, спеціально призначеним для обробки помилок. Або ви можете передати керування наступному відповідному маршруту, щоб обробити особливі випадки. Ми побачимо, як це робити, у розділі про просунутий Express.

# --instructions--

У маршруті `app.get('/now', ...)` зв’яжіть функцію підпрограмного забезпечення та фінальний обробник. У функції підпрограмного забезпечення додайте поточний час до об’єкта запиту в ключ `req.time`. Ви можете використати `new Date().toString()`. В обробнику відповідайте JSON-об’єктом, маючи структуру `{time: req.time}`.

**Примітка:** Тест не пройде, якщо ви не зв’яжете підпрограмне забезпечення в ланцюжок. Якщо ви підключите функцію в іншому місці, тест не пройде, навіть якщо вихідні дані будуть правильними.

# --hints--

До кінцевої точки /now має бути підключене підпрограмне забезпечення

```js
  const response = await fetch(code + '/_api/chain-middleware-time');
  if (!response.ok) {
    throw new Error(await response.text());
  }
  const data = await response.json();
  assert.equal(
    data.stackLength,
    2,
    '"/now" route has no mounted middleware'
  );
```

Кінцева точка `/now` має повертати поточний час.

```js
  const response = await fetch(code + '/_api/chain-middleware-time');
  if (!response.ok) {
    throw new Error(await response.text());
  }
  const data = await response.json();
  var now = new Date();
  assert.isAtMost(
    Math.abs(new Date(data.time) - now),
    20000,
    'the returned time is not between +- 20 secs from now'
  );
```
