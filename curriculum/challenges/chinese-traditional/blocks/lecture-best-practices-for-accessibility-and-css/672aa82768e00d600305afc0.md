# --interactive--

在設計網站時，確保良好的色彩對比對於存取性和可讀性至關重要。有多種工具可協助開發者和設計師檢查並維護網站上的適當色彩對比比例。其中一個受歡迎的工具是 WebAIM 的 Color Contrast Checker。

這個線上工具允許你輸入設計的前景色和背景色，並立即查看它們是否符合網頁內容存取準則（WCAG）標準。它使用者友善，並即時提供你的顏色選擇是否通過對比度要求的回授。

要使用 WebAIM 的 Color Contrast Checker，只需造訪他們的網站並輸入你的前景色和背景色的十六進位色碼。該工具將計算對比度比率並顯示是否符合 WCAG 2.0 層級 AA 或 AAA 標準。以下是你如何在 CSS 中使用色彩值的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
body {
  background-color: #FFFFFF;
  color: #333333;
}
```

:::

在這個範例中，我們設定了白色背景（`#FFFFFF`）和深灰色文字（`#333333`）。你可以將這些色碼輸入到 WebAIM 工具中，以檢查它們的對比度比例。

另一個有價值的工具是 TPGi Colour Contrast Analyzer。這款桌面應用程式提供比線上工具更進階的特性。它允許你分析整個網頁，而不僅僅是單一的顏色配對。你可以使用它的取色器直接從你的螢幕選取顏色，使檢查現有設計的對比度比率更加容易。

TPGi Colour Contrast Analyzer 也提供不同型別色覺缺陷的模擬，幫助你確保你的設計對各種色盲形式的使用者皆可存取。

要使用 TPGi Colour Contrast Analyzer，請先在你的電腦上下載並安裝該應用程式。安裝完成後，你可以使用它的吸管工具從你的螢幕選取顏色，並立即查看對比度比率。當你處理複雜設計或檢查即時網站的對比度時，這特別有用。

這兩個工具對於建立可存取且易讀的網站非常寶貴。它們有助於確保你的 `text` 在其 `background` 上清晰可見，這對所有使用者都很重要，但對於視覺障礙者尤其重要。請記住，雖然這些工具非常有用，但應該與手動測試和使用者回授一起使用。不同的背景關係和使用者需求可能需要超出自動化工具建議的調整。

透過定期使用像 WebAIM 的 Color Contrast Checker 和 TPGi Colour Contrast Analyzer 這類工具，你可以創建不僅視覺上吸引人，且更具包容性和更易於更廣泛受眾存取的設計。對色彩對比的細節關注能顯著提升你網站所有訪客的使用者體驗。
