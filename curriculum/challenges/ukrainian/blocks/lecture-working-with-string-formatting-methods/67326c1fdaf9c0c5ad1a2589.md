# --interactive--

Працюючи з рядками в JavaScript, часто виникають ситуації, коли потрібно змінити регістр тексту, наприклад, перетворити всі літери на верхній регістр для заголовка або привести текст до нижнього регістру для уніфікації.

На щастя, JavaScript робить це просто за допомогою двох вбудованих методів: `toUpperCase()` та `toLowerCase()`.

Метод `toUpperCase()` перетворює всі символи на великі літери і повертає новий рядок з усіма символами у верхньому регістрі. Це корисно, коли ви хочете виділити текст або створити послідовність у форматі рядків.

Давайте подивимось приклад:

:::interactive_editor

```js
let greeting = "Hello, World!";
let uppercaseGreeting = greeting.toUpperCase();
console.log(uppercaseGreeting);  // "HELLO, WORLD!"
```

:::

У цьому коді метод `toUpperCase()` перетворює весь рядок у верхній регістр.

Оригінальний рядок залишається незмінним, бо `toUpperCase()` повертає новий рядок, а не змінює початковий.

З іншого боку, метод `toLowerCase()` перетворює всі символи рядка на нижній регістр. Це корисно, коли потрібно стандартизувати вхідні дані, наприклад, при порівнянні тексту, введеного користувачем, або при нечутливих до регістру перевірках.

Розглянемо приклад:

:::interactive_editor

```js
let shout = "I AM LEARNING JAVASCRIPT!";
let lowercaseShout = shout.toLowerCase();
console.log(lowercaseShout);  // "i am learning javascript!"
```

:::

Метод `toLowerCase()` перетворює всі символи на нижній регістр, роблячи рядок менш агресивним, при цьому оригінальний рядок залишається незмінним.

Підсумовуючи, методи `toUpperCase()` та `toLowerCase()` у JavaScript — це потужні інструменти для перетворення рядків у всі великі або всі малі літери.

Ці методи особливо корисні для стандартизації текстових вхідних даних, виконання нечутливих до регістру порівнянь і забезпечення послідовності дизайну.

Завдяки цим простим, але ефективним методам ви можете керувати маніпуляціями з текстом більш контрольовано і передбачувано.
