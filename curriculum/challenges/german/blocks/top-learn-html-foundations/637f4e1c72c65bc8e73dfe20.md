# --description--

Um einen HTML-Boilerplate-Code zu demonstrieren, benötigen Sie zuerst eine HTML-Datei, mit der Sie arbeiten können.

Erstellen Sie auf Ihrem Computer einen neuen Ordner und nennen Sie ihn `html-boilerplate`. Erstellen Sie in diesem Ordner eine neue Datei und nennen Sie sie `index.html`.

Sie sind wahrscheinlich schon mit vielen verschiedenen Dateitypen vertraut, zum Beispiel doc-, pdf- und Bilddateien.

Um dem Computer mitzuteilen, dass Sie eine HTML-Datei erstellen möchten, müssen Sie den Dateinamen mit der Erweiterung `.html` versehen, wie Sie es bereits beim Erstellen der `index.html`-Datei getan haben.

Es ist wichtig zu erwähnen, dass Sie Ihre HTML-Datei index genannt haben. Sie sollten die HTML-Datei, die die Startseite Ihrer Websites enthält, immer `index.html` nennen. Das liegt daran, dass Webserver standardmäßig nach einer `index.html`-Seite suchen, wenn Nutzer Ihre Websites aufrufen – und das Fehlen einer solchen Seite große Probleme verursachen kann.

## The DOCTYPE

Jede HTML-Seite beginnt mit einer doctype-Deklaration. Der Zweck des Doctypes ist es, dem Browser mitzuteilen, welche HTML-Version er zum Rendern des Dokuments verwenden soll. Die neueste Version von HTML ist HTML5, und der Doctype für diese Version ist einfach `<!DOCTYPE html>`.

Die Doctypes für ältere HTML-Versionen waren etwas komplizierter. Zum Beispiel ist dies die Doctype-Deklaration für HTML4:

```html
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
```

Sie werden wahrscheinlich nie eine ältere HTML-Version verwenden wollen, daher werden Sie immer `<!DOCTYPE html>` verwenden.

Öffnen Sie die zuvor erstellte `index.html`-Datei in Ihrem Texteditor und fügen Sie `<!DOCTYPE html>` in die allererste Zeile ein.
