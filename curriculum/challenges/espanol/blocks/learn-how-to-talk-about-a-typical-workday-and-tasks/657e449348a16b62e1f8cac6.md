# --description--

`Security sweeps` son inspecciones o búsquedas detalladas realizadas en un área o edificio, generalmente para propósitos de seguridad y protección. Implica revisar por cualquier riesgo potencial, acceso no autorizado, o elementos peligrosos. Por ejemplo:

Los oficiales de seguridad realizan barridos de seguridad para encontrar cualquier peligro oculto. Esto significa que los oficiales de seguridad revisan cuidadosamente el área para ver si hay cosas peligrosas que no son fáciles de ver.
