# --description--

La parola `earlier` si usa per parlare di un tempo precedente al momento attuale o al tempo di cui si discute. È come dire `before now` o `some time ago`.

Per esempio, `If you finished your work earlier, you can relax now` significa che se hai completato il tuo lavoro un po’ di tempo fa, ora hai tempo per rilassarti.
