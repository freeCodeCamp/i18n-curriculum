# --description--

Viele Zahlen können als Summe aus einem Quadrat und einer Kubikzahl ausgedrückt werden. Einige davon auf mehr als eine Weise.

Betrachten Sie die palindromischen Zahlen, die genau auf 4 verschiedene Arten als Summe aus einem Quadrat und einer Kubikzahl, beide größer als 1, ausgedrückt werden können.

Zum Beispiel ist 5229225 eine palindromische Zahl und kann genau auf 4 verschiedene Arten ausgedrückt werden:

$$\begin{align}
  & {2285}^2 + {20}^3 \\\\
  & {2223}^2 + {66}^3 \\\\
  & {1810}^2 + {125}^3 \\\\
  & {1197}^2 + {156}^3
\end{align}$$

Finden Sie die Summe der fünf kleinsten solchen palindromischen Zahlen.
