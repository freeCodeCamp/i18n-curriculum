# --description--

Ángela anauliza swali kwa kutumia muundo `¿Cuál es tu` + nomino.

Anaweza kuacha hapo. Badala yake, anaongeza kipengele kingine cha habari kwa kutumia kiunganishi `y` na nomino ya pili.

`y` ni kiunganishi kinachotumika kuunganisha nomino mbili. Huu ndio muundo anaoutumia:

`¿Cuál es tu` + nomino + `y` + nomino`?`

Hii inaruhusu msemaji kuomba maelezo zaidi ya moja katika swali moja.

# --explanation--

Katika sentensi `¿Cuál es tu nombre y apellido?`, nomino mbili zimeunganishwa kwa `y`:

- `Nombre`

- `Apellido`

Neno `y` linaunganisha vipengele viwili vya habari.

Ángela kwanza anauliza `nombre` ya mtu kisha anaongeza `apellido`.

Muundo huu unaruhusu msemaji kuomba maelezo mengi katika swali moja:

`¿Cuál es tu correo y número?`

Hapa, kipengele cha pili cha habari Ángela anauliza ni `apellido`.
