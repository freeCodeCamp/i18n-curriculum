# --description--

配列 `arr` が与えられたとき、最初の要素（インデックス0）から順に繰り返し処理し、関数 `func` に要素を渡したときに `true` を返すまで各要素を削除してください。

条件が満たされたら残りの配列を返し、そうでなければ `arr` を空の配列として返してください。

# --solutions--

```js
function dropElements(arr, func) {
  while (arr.length && !func(arr[0])) {
    arr.shift();
  }
  return arr;
}
```
