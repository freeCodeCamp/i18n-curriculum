# --description--

Um **`endpoint`** em uma API é um local específico onde você pode solicitar determinados dados ou serviços.Por exemplo, `The API endpoint for weather data retrieves the latest forecasts.` 

 `Offer` significa fornecer ou disponibilizar.Por exemplo, `This service offers real-time updates.`

Uma `route` é um caminho definido na API que determina como as requisições são tratadas.Como, `The login route verifies user credentials.`
