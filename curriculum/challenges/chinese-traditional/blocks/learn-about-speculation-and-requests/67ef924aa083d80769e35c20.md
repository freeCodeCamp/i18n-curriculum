# --explanation--

「`Make it a habit`」是指定期做某件事直到它變成自動化。例如：

- `Try to make it a habit to write clean code.`－你應該定期撰寫易於閱讀的程式碼。

- `She made it a habit to learn one new programming tip every week.`－她定期學習，作為她常式的一部分。
