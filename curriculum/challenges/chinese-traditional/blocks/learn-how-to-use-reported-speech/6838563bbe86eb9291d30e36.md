# --explanation--

`Past Perfect` 時態（`had` + 動詞的 `past participle` 形式）用來描述發生在另一個過去事件之前的動作。它顯示哪個動作先發生。<dfn>例如</dfn>：

`The server had crashed before the backup started。` – 當備份開始時，當機已經發生。

`Third Conditional` 結構（`if` + 動詞的 `past perfect` 形式作為假設原因，情態動詞 + `have` + 動詞的 `past participle` 形式作為假設結果）表達一個不真實的過去條件及其可能但不真實的結果。它說明如果採取其他行動，事情可能會有不同。 例如：

`If they had updated the software, they could have avoided the error.`－這表示更新沒有發生，錯誤也沒有被避免。
