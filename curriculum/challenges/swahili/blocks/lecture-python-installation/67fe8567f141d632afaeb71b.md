# --description--

Kwa warsha zote na maabara, utatumia mhariri wa Python wa freeCodeCamp. Lakini ni muhimu kujifunza jinsi ya kusanidi Python kwenye kompyuta yako ya ndani.

Njia rahisi zaidi ya kusakinisha Python kwenye Windows na Mac ni kupakua faili la kusakinisha kutoka kwenye tovuti rasmi ya Python. Pia tutazungumzia jinsi ya kuendesha Python kwenye Linux baadaye katika funzo hili.

Nenda `https://www.python.org/` na elekeza kipanya juu ya "Downloads". Dirisha la kidirisha litaonekana likionyesha toleo la sasa la Python kwa mfumo wako wa uendeshaji (OS).

Tutaangalia jinsi ya kusakinisha Python kwenye kompyuta inayotumia macOS kwanza:

- Bofya kitufe kinachoonyesha toleo la sasa la Python (kutoka kwenye dirisha la kidirisha lililotangulia), na utaanza kupakua faili la kusakinisha la `.pkg` moja kwa moja.
- Mara faili la `.pkg` limekamilika kupakuliwa, lifungue, kisha bofya "Continue" kwenye dirisha linalofunguka.
- Endelea kubofya kitufe cha "Continue" hadi ufike sehemu ya "Installation Type". Hapo, bofya kitufe cha "Install".
- Ingiza nenosiri lako ikiwa ni lazima, kisha anza usakinishaji.
- Baada ya hapo, unapaswa kupata ujumbe wa pongezi ukisema Python imesakinishwa kwa mafanikio.
- Bofya kitufe cha "Close", na umemaliza!

Unaweza kuthibitisha usakinishaji kwa kufungua terminal yako na kuendesha `python --version` au `python3 --version`.

Pia unaweza kufungua mfasiri wa Python kwa kuendesha `python` au `python3` kwenye terminal. Mfasiri wa Python ni programu inayosoma msimbo wa Python, kutafsiri kuwa maagizo ambayo kompyuta inaelewa na kutekeleza maagizo hayo.

<dfn>Terminal</dfn> ni kiolesura cha maandishi kinachokuwezesha kuingiliana na kompyuta yako kwa kuandika amri. Kila mfumo wa uendeshaji huja na programu ya terminal ya msingi. Katika macOS, unaweza kutumia programu ya Terminal. Katika Windows, unaweza kutumia Command Prompt au PowerShell. Katika Linux, kila mazingira ya desktop yana programu yake ya terminal ya msingi, kama GNOME Terminal au Konsole.

Kumbuka kwamba, katika baadhi ya mifumo ya zamani ya macOS na Linux, `python` inaweza kuwa imehifadhiwa kwa Python 2, wakati `python3` ni kwa Python 3 hasa. Ikiwa unaendesha `python --version` na unaona toleo la Python 2 kama `Python 2.7.18`, basi inawezekana mfumo wako unategemea programu fulani iliyotengenezwa kwa toleo la zamani la Python. Ikiwa ni hivyo, unapaswa kutumia `python3` kuendesha msimbo wako wa Python kuanzia sasa. Python 2 ni mwisho wa maisha yake na haipaswi kutumika kwa ukuzaji mpya wowote.

Kusakinisha Python kwenye Windows, fuata hatua hizi:

- Nenda `https://www.python.org/`, na elekeza kipanya juu ya “Downloads“. Unapaswa kuona dirisha linalosema "Download for Windows" na kitufe cha kupakua chenye toleo la sasa la Python.
- Bofya nambari ya toleo, na utaanza kupakua faili la utekelezaji la Windows (`.exe`) moja kwa moja.
- Mara umemaliza kupakua faili la kusakinisha Python kwa Windows, bofya mara mbili juu yake, na fuata maelekezo.
- Unapoona chaguo la `Add python.exe to Path`, angalia chaguo hilo, kisha bofya `Install Now`. Kufanya hivyo kutakuwezesha baadaye kwa urahisi.

Unaweza kuthibitisha usakinishaji kwa kufungua shell ya mstari wa amri kama PowerShell na kuendesha `python --version`. Pia unaweza kufungua mfasiri wa Python kwa kuendesha `python`.

Kwa Python kwenye Linux, distros kuu kama Ubuntu, Debian, na Fedora huja na Python tayari.
