# --description--

Se você ampliar a janela de visualização, deverá ver o elemento `div` centralizado horizontalmente na página.Mas para dispositivos menores, a largura do elemento `div` abrange toda a largura da página.  

Atualize a classe `mx-auto` para que ela seja aplicada apenas em dispositivos `md` e maiores.

# --hints--

Sua tag de abertura `div` deve ter a classe utilitária `md:mx-auto`.

```js
const divEl = document.querySelector("div");
assert.isTrue(divEl.classList.contains("md:mx-auto"));
```

Você não deve mais ter a classe `mx-auto` na sua tag `div` de abertura.

```js
const divEl = document.querySelector("div");
assert.isFalse(divEl.classList.contains("mx-auto"));
```
