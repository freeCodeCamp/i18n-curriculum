# --description--

`extension` は、より大きなソフトウェアアプリケーションに特定の機能を追加するソフトウェアコンポーネントです。`Extensions` は新しい機能を提供したり、既存の機能を強化したりできます。

たとえば、ブラウザの `extension` は、広告をブロックしたりパスワードを管理したりするなど、ウェブブラウザに特別な機能を追加できます。
