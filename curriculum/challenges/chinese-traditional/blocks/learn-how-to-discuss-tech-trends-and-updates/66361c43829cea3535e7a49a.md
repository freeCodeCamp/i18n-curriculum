# --description--

片語 `why don't` 通常用來提出建議或提供幫助。它後面接主詞（例如 `we`、`you` 或 `they`）和原形動詞。結構是 `why don't + subject + base verb`。這是一種禮貌地建議行動或請求某人做某事的方式。

範例：

`Why don't we go to the cinema tonight?` 講話者建議去看電影。

`Why don't you try restarting the computer?` 講者建議重新啟動電腦作為一種解決方案。

片語 `check out` 意指更仔細地檢查或查看某物，通常用於建議探索或進一步了解某事。例如，`Let's check out the new restaurant` 意味著拜訪並嘗試那家餐廳。
