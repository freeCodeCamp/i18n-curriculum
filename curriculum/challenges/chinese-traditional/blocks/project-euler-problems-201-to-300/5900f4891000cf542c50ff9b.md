# --description--

十進位制中三位數字 376 是具有特殊屬性的數字範例，其平方的末尾與該數字相同：${376}^2 = 141376$。讓我們將具有此屬性的數字稱為穩定平方數。

穩定平方數也可以在其他進位制中觀察到。在 14 進位制中，三位數字 $c37$ 也是一個穩定平方數：$c37^2 = aa0c37$，且其數字和為 $c+3+7=18$，同樣在該進位制中。字母 $a$、$b$、$c$ 和 $d$ 分別用於表示 10、11、12 和 13 的數字，方式類似於十六進位制。

對於 $1 ≤ n ≤ 9$，在 14 進位制中所有 $n$ 位數的穩定平方數的位數和為 $2d8$（十進位為 582）。不允許有前導 0 的穩定平方數。

找出在 14 進位制中所有 $n$ 位穩定平方數的數字總和，對於 $1 ≤ n ≤ 10000$（十進位），並以 14 進位制的字串形式給出你的答案，必要時使用小寫字母。
