# --description--

如果一個整數的十進位展開中不包含任何表述 11 次方的子字串（除了 1），則該整數稱為 eleven-free。

例如，2404 和 13431 是 eleven-free，而 911 和 4121331 則不是。

令 $E(n)$ 為第 $n$ 個正的無十一整數。 例如，$E(3) = 3$、$E(200) = 213$ 和 $E(500\,000) = 531\,563$。

尋找 $E({10}^{18})$。

# --hints--

`elevenFreeIntegers()` 應該傳回 `1295552661530920200`。

assert.strictEqual(elevenFreeIntegers(), 1295552661530920200);
