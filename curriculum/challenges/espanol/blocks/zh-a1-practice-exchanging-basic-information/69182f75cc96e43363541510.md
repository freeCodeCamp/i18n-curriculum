# --explanation--

Si quieres hacer una pregunta, no olvides primero introducirla cortésmente con `请问 (qǐng wèn)` y luego formularla.

Para saber el nombre de una persona, puedes usar `叫什么 (jiào shén me)`.

Si acabas de presentarte y quieres preguntarle a la otra persona lo mismo, simplemente di `你呢 (nǐ ne)?`.

Para expresar que sientes lo mismo que la otra persona, puedes decir `我也是 (wǒ yě shì)`.
