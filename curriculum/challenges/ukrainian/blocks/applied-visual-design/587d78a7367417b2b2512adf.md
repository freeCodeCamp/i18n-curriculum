# --description--

Щоб анімувати елемент, потрібно знати про властивості animation і правило `@keyframes`. Властивості animation контролюють, як має поводитися анімація, а правило `@keyframes` визначає, що відбувається під час цієї анімації. Всього існує вісім властивостей animation. Це завдання спростить і спочатку розгляне дві найважливіші:

`animation-name` задає ім’я анімації, яке пізніше використовується `@keyframes` для того, щоб CSS знав, які правила належать яким анімаціям.

`animation-duration` задає тривалість анімації.

`@keyframes` визначає, що саме відбувається в анімації протягом її тривалості. Це робиться шляхом задання CSS-властивостей для конкретних «кадрів» анімації, з відсотками від 0% до 100%. Якщо порівняти це з фільмом, то CSS-властивості для 0% — це як елемент виглядає в початковій сцені. CSS-властивості для 100% — це як елемент виглядає в кінці, перед титрами. Потім CSS застосовує магію, щоб плавно змінити елемент протягом заданої тривалості і показати сцену. Ось приклад, який ілюструє використання `@keyframes` і властивостей animation:

```css
#anim {
  animation-name: colorful;
  animation-duration: 3s;
}

@keyframes colorful {
  0% {
    background-color: blue;
  }
  100% {
    background-color: yellow;
  }
}
```

Для елемента з ідентифікатором `anim` наведений фрагмент коду задає `animation-name` зі значенням `colorful` і встановлює `animation-duration` на 3 секунди. Потім правило `@keyframes` пов’язує властивості animation з ім’ям `colorful`. Воно задає колір синім на початку анімації (0%), який плавно змінюється на жовтий до кінця анімації (100%). Ви не обмежені лише початково-кінцевими переходами, можна задавати властивості для елемента на будь-який відсоток між 0% і 100%.
