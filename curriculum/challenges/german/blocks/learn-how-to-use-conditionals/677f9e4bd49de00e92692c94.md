# --explanation--

Ein `operating system` ist die Hauptsoftware, die es anderen Programmen ermöglicht, auf einem Computer zu laufen. Sie verwaltet die Hardware- und Software-Ressourcen und sorgt dafür, dass alles reibungslos zusammenarbeitet. Zum Beispiel:

- `The app is compatible with any operating system.` – Die App funktioniert auf allen Arten von Systemen wie Windows oder MacOS.

- `I installed a new operating system on my laptop.` – Sie richten ein anderes System ein, um Ihren Computer zu betreiben.
