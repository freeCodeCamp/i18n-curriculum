# --explanation--

`Figure out` è un phrasal verb che significa risolvere o capire qualcosa di difficile. Per esempio:

`We need to figure out why the website isn't loading.` - Hanno bisogno di capire il motivo per cui il sito web non funziona.

Un `firewall` è una funzionalità di sicurezza che protegge le reti informatiche dagli hacker. Funziona come una barriera digitale, permettendo il passaggio sicuro dei dati e bloccando il traffico pericoloso. Per esempio:

`Make sure the firewall is active before you connect to the internet.` - Assicurati che il sistema di sicurezza sia attivo prima di usare internet.
