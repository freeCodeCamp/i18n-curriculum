# --description--

SQL steht für Structured Query Language. Es wurde in den 1970er-Jahren entwickelt und war ursprünglich als Structured English Query Language (SEQUEL) bekannt. Dieser Begriff wurde später auf SQL verkürzt.

SQL ist eine Programmiersprache, die zum Speichern und Verwalten von Daten in relationalen Datenbanken verwendet wird.

Wie Sie in einer vorherigen Lektion gelernt haben, organisieren relationale Datenbanken Daten in Tabellen. Jede Tabelle besteht aus einer Reihe von Zeilen und Spalten, in denen die Datenattribute und deren Beziehungen dargestellt werden. Jede Zeile stellt einen Datensatz dar, während jede Spalte ein bestimmtes Attribut oder Feld dieses Datensatzes repräsentiert.

Mit SQL können Sie Daten in einer relationalen Datenbank hinzufügen, ändern, löschen, suchen und abrufen. Es bietet eine umfassende Reihe von Befehlen zum Abfragen, Filtern, Sortieren und Aggregieren von Daten.

SQL kann helfen, die Leistung der Datenbank zu optimieren, indem nur die notwendigen Daten abgefragt werden, und es lässt sich leicht mit einer Vielzahl von Programmiersprachen integrieren, sodass Sie direkt aus Ihren Anwendungen mit Datenbanken interagieren können.

Es wurde außerdem als Industriestandard übernommen. SQL gewährleistet die Datenintegrität und enthält viele Sicherheitsfunktionen, wie Unterstützung für Benutzer-Authentifizierung und Datenverschlüsselung.

Es ist zudem skalierbar, portabel und kompatibel mit einer Vielzahl von Datenbankmanagementsystemen.

SQL basiert auf Befehlen, die allgemein als SQL-Anweisungen oder SQL-Abfragen bezeichnet werden.

Als Entwickler schreiben Sie diese Anweisungen mit spezifischen SQL-Sprachbestandteilen oder Schlüsselwörtern. Diese ermöglichen es Ihnen, die notwendigen Operationen auf Ihrer Datenbank auszuführen.

Bevor Sie SQL-Befehle eingeben können, müssen Sie die psql-Shell öffnen, um direkt mit der PostgreSQL-Datenbank zu interagieren. Geben Sie im Eingabeaufforderungsfenster oder Terminal Folgendes ein:

```bash
psql -U <username> -d <database_name>
```

Ersetzen Sie `username` durch Ihren Benutzernamen und `database_name` durch die Datenbank, mit der Sie sich verbinden möchten.

Falls Sie noch keine eigene Datenbank erstellt haben, enthält PostgreSQL standardmäßig eine Datenbank namens `postgres`, die bei der Installation automatisch angelegt wird. Sie können diese `postgres`-Datenbank verwenden, um mitzumachen.

Sobald Sie verbunden sind, ändert sich die Eingabeaufforderung zu:

```bash
postgres=#
```

Sie zeigt den Namen der verbundenen Datenbank an und wartet auf SQL-Befehle.

Lassen Sie uns einige der grundlegendsten SQL-Befehle durchgehen, mit denen Sie vertraut sein sollten. Sie werden feststellen, dass SQL-Befehle konventionell mit einem Semikolon (`;`) enden.

Zuerst können Sie diesen Befehl verwenden, um eine Datenbank namens `my_database` zu erstellen:

```sql
CREATE DATABASE my_database;
```

Sie können `CREATE` gefolgt von `DATABASE` und dem Namen der Datenbank eingeben, die Sie erstellen möchten. Beachten Sie, dass die SQL-Schlüsselwörter des Befehls in Großbuchstaben geschrieben sind.

Sobald Sie Ihre Datenbank haben, können Sie sich mit ihr verbinden.

Wie Sie sich mit der Datenbank verbinden, hängt stark von der Umgebung und dem verwendeten Tool ab.

Im Eingabeaufforderungsfenster oder Terminal können Sie die zuvor gezeigte Methode verwenden. In der interaktiven psql-Shell können Sie mit dem `\c`-Shortcut-Befehl gefolgt vom Datenbanknamen die Datenbank wechseln, so:

```bash
\c my_database
```

Beachten Sie, dass der `\c`-Befehl und andere Shortcut-Befehle (die mit einem Backslash beginnen) Teil der psql-Shell sind und nicht zur SQL-Sprache selbst gehören, weshalb sie kein Semikolon zum Abschließen benötigen.

Nachdem Sie sich mit Ihrer neuen Datenbank verbunden haben, sehen Sie, dass sich die Eingabeaufforderung geändert hat:

```bash
my_database=#
```

Sobald Sie eine Datenbank erstellt und sich mit ihr verbunden haben, können Sie mit den `CREATE TABLE`-Schlüsselwörtern eine Tabelle erstellen:

```sql
CREATE TABLE products (
  id SERIAL,
  name VARCHAR(255)
);
```

In dem Befehl schreiben Sie `CREATE TABLE`, gefolgt vom Namen der Tabelle, die Sie erstellen möchten. In diesem Fall wird eine Tabelle namens `products` erstellt.

Dann schreiben Sie in runde Klammern die Namen der Spalten, die die Tabelle haben soll, und geben den Datentyp der Werte an, die in jeder Spalte gespeichert werden.

Im Beispiel hat die Tabelle zwei Spalten: eine Spalte für die Produkt-ID und eine weitere für den Namen des Produkts.

Die Standard-Konvention für die Benennung von Tabellen- und Spaltennamen ist snake case, also Wörter in Kleinbuchstaben zu schreiben und sie mit einem Unterstrich zu trennen (`_`). Zum Beispiel: `delivery_orders`

Dies sind einige wesentliche SQL-Befehle, die Sie kennen sollten, um loszulegen.

Mit SQL können Sie Daten einfach abfragen, manipulieren und analysieren, um fundierte Entscheidungen zu treffen und reale Probleme zu lösen.
