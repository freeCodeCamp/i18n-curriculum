# --description--

$12n$명의 음악가가 음악 축제에 참가합니다. 첫째 날, 이들은 $3n$개의 4인조를 구성하여 하루 종일 연습합니다.

결과는 참담합니다. 하루가 끝날 무렵, 모든 음악가는 자신이 속한 4인조의 어떤 멤버와도 다시는 함께 연주하지 않기로 결심합니다.

둘째 날, 이들은 $4n$개의 3인조를 구성하는데, 각 음악가는 이전 4인조의 멤버를 피합니다.

$12n$명의 음악가들 사이에서 3인조를 조직하는 방법의 수를 $f(12n)$이라 합시다.

$f(12) = 576$이고 $f(24)\bmod 1\,000\,000\,007 = 509\,089\,824$가 주어졌습니다.

$f(600)\bmod 1\,000\,000\,007$을 구하세요.
