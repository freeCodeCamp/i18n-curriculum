# --explanation--

`The manager addressed the client's concerns during the meeting.` — тут `addressed` означає вжиття заходів для розв’язання проблем клієнта.

У реченні Джейк використовує `Past Continuous` (`was addressing`) для опису тривалої дії розв’язання проблеми, поки відбувалася інша тривала дія (`the team was waiting`).
