# --description--

Um eine Konstante (unveränderliche) Variable zu deklarieren, verwenden Sie `const` anstelle von `let`:

```javascript
const myBirthday = '16-09-2003';
```

Variablen, die mit `const` deklariert werden, nennt man „Konstanten“. Sie können nicht neu zugewiesen werden. Ein Versuch, dies zu tun, würde einen Fehler verursachen:

```javascript
const myBirthday = '16-09-2003';

myBirthday = '01-01-2001'; // error, can't reassign the constant!
```

Wenn eine Programmierkraft sicher ist, dass sich eine Variable niemals ändert, kann sie diese mit `const` deklarieren, um diese Tatsache zu garantieren und allen mitzuteilen.
