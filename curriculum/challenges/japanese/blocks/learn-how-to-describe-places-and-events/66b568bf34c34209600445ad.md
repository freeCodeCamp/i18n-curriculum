# --explanation--

ブライアンがトレーニングについて現在どのように考えているかを理解するには、彼が計画について話している部分をよく聞いてください。

`I'm thinking of using...` と `especially for...` の直後にブライアンが話している内容に注目すると、トレーニングに関する彼の具体的な考えがわかります。
