# --description--

Парне додатне ціле число $N$ називатиметься припустимим, якщо воно є степенем двійки або його різні прості множники є послідовними простими числами.

Перші дванадцять припустимих чисел: 2, 4, 6, 8, 12, 16, 18, 24, 30, 32, 36, 48.

Якщо $N$ припустиме, найменше ціле число $M > 1$, таке що $N + M$ є простим числом, називатиметься псевдо-Фортунатним числом для $N$.

Наприклад, $N = 630$ є припустимим, оскільки воно парне, а його різні прості множники — це послідовні прості числа 2, 3, 5 і 7. Наступне просте число після 631 — 641; отже, псевдо-Фортунатне число для 630 дорівнює $M = 11$. Також можна побачити, що псевдо-Фортунатне число для 16 дорівнює 3.

Знайдіть суму всіх різних псевдо-Фортунатних чисел для припустимих чисел $N$, менших за ${10}^9$.
