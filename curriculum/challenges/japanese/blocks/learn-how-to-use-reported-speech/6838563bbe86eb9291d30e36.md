# --explanation--

`Past Perfect` 時制（`had` + 動詞の`past participle`形）は、ある過去の出来事より前に起きた行動について話すときに使います。どの行動が先に起きたかを示します。例えば：

`The server had crashed before the backup started.` – クラッシュが先に起き、その後バックアップが始まりました。

`Third Conditional` 構文（`if` + 動詞の`past perfect`形を仮定の原因として、助動詞 + `have` + 動詞の`past participle`形を仮定の結果として使う）は、現実とは異なる過去の条件とその可能な結果を表します。別の行動を取っていたらどうなっていたかを話します。例えば：

`If they had updated the software, they could have avoided the error.` – これはアップデートが起きず、エラーが回避されなかったことを意味します。
