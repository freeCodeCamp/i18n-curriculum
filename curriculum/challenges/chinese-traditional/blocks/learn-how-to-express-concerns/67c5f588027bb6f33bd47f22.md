# --explanation--

當你說某事 `works smoothly`，表示該事發生時沒有困難或中斷。`Smoothly` 來自 `smooth`，後者描述某物沒有粗糙或障礙，但它們用於不同的上下文。舉例來說：

- `The table has a smooth surface.`－這表示該桌面沒有粗糙區域。

- `The software update helped the system run more smoothly。`－這表示系統現在執行時問題更少。
