# --description--

このタスクは、`can` を使った能力の表現と、過去進行形の動詞 `try` の使い方のレビューにフォーカスしています。

`Can` は、`I can solve this problem.` のように、誰かが何かをできる能力について話すときに使います。

過去進行形の `Try`（`was`/`were` + 動詞の `-ing` 形）は、`I was trying to fix the issue.` のように、過去の継続的な動作を示します。
