# --explanation--

`Deployment` 是指釋出或啟動軟體，使其能在真實環境中使用。例如：

`We're planning the next deployment for Thursday.` — 團隊將在星期四釋出新變更。

`Documentation` 是用來說明某物如何運作或如何使用的書面資訊。例如：

`The software needs better documentation。` — 指示或使用者指南應該更清楚。

合稱為 `deployment documentation` 是指正確啟動軟體所需的書面步驟和說明。
