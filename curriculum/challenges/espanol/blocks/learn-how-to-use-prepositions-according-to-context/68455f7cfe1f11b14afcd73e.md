# --explanation--

`Layout` se refiere al diseño o estructura de un espacio. Por ejemplo:  

`He shared the layout of the new office.` – Esto significa el plan de dónde van las cosas.

`Draft` es una versión temprana de un documento. Por ejemplo:  

`She wrote the first draft of the proposal.` – Todavía puede ser editado.

`Send it over` significa entregar algo. Por ejemplo:  

`I'll send it over once it's ready.` – Significa compartir un archivo o mensaje.

`To` muestra para quién es la acción. Por ejemplo:  

`She gave the file to her manager.` – Muestra quién lo recibe.
