# --description--

Katika funzo lililopita, ulijifunza kuhusu aina ya data ya orodha na jinsi ya kupata vipengele kutoka kwenye orodha pamoja na kukata orodha. Katika funzo hili, utaendelea kujifunza kuhusu orodha na baadhi ya njia za kawaida zinazohusiana nazo kama `append()`, `pop()`, na `sort()`.

Njia ya kwanza tutakayozungumzia ni njia ya `append()`. Hii hutumika kuongeza kipengele mwishoni mwa orodha. Hapa kuna mfano wa kutumia njia ya `append()` kuongeza nambari `6` kwenye orodha ya nambari:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

Kama unataka kuongeza orodha moja mwishoni mwa nyingine, pia unaweza kutumia njia ya `append()` kama ifuatavyo:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

Angalia jinsi orodha nzima ya `even_numbers` imepangwa ndani ya orodha ya `numbers`.

Lakini kama unataka kuongeza nambari zote binafsi kutoka kwenye orodha ya `even_numbers` mwishoni mwa orodha ya `numbers`, basi unaweza kutumia njia ya `extend()`.

Njia ya `extend()` ni sawa na njia ya `append()`, lakini kwa `extend()` unaweza kuongeza vipengele vingi kutoka kwenye orodha moja hadi nyingine. Hapa kuna mfano wa kuongeza nambari `6`, `8`, na `10` kutoka kwenye orodha moja mwishoni mwa orodha ya `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

Kama unavyoona, orodha iliyopangwa ndani imeondoka na sasa ni orodha ya nambari tu.

Ili kuingiza kipengele mahali fulani katika orodha, unaweza kutumia njia ya `insert()`. Njia hii inakubali hoja mbili: kielezo ambacho unataka kuingiza kipengele kipya na kipengele unachotaka kuingiza.

Hapa kuna mfano wa kutumia njia ya `insert()`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

Msimbo ufuatao utaingiza nambari `2.5` kwenye kielezo `2` katika orodha ya `numbers`.

Kama unataka kuondoa kipengele kutoka kwenye orodha, unaweza kutumia njia ya `remove()`. Njia ya `remove()` inachukua thamani ya kipengele cha kuondoa kama hoja:

```py
numbers = [10, 20, 30, 40, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50]
```

Ni muhimu kutambua kuwa njia hii itaondoa tu mara ya kwanza ya kipengele. Sio zote:

```py
numbers = [10, 20, 30, 40, 50, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50, 50]
```

Ili kuondoa kipengele kilichoko kwenye kielezo fulani katika orodha, unaweza kutumia njia ya `pop()` kama ifuatavyo:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

Kama hutaja kipengele kwa njia ya `pop`, basi kipengele cha mwisho ndicho kinachotolewa.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

Kama unahitaji kuondoa orodha yote, basi unaweza kutumia njia ya `clear()` kama ifuatavyo:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

Njia inayofuata tutakayozungumzia ni njia ya `sort()`. Njia hii hutumika kupanga vipengele mahali pake. Hapa kuna mfano wa kupanga orodha ya nambari kwa mpangilio mahali pake:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

Tofauti na njia ya `sort()`, kuna kitendakazi cha `sorted()` ambacho hufanya kazi kwa kila kitu kinachoweza kurudiwa na hurudisha orodha mpya iliyopangwa badala ya kubadilisha orodha ya awali. Kwa mfano:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(numbers) # [19, 2, 35, 1, 67, 41]
print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
```

Kama ukumbusho, kitu kinachoweza kurudiwa ni aina maalum ya kitu ambacho unaweza kurudia, kukuwezesha kupata kila kipengele moja kwa moja. Utajifunza zaidi kuhusu jinsi mizunguko inavyofanya kazi katika Python katika funzo lijalo.

Njia ya `sort()` na kitendakazi cha `sorted()` zinakubali vigezo vya hiari vya `key` na `reverse`. Utajifunza zaidi kuhusu vigezo hivi vya hiari katika funzo lijalo unapoanza kujifunza kuhusu tuples. Pia utajifunza zaidi kuhusu vitendakazi vingine vilivyojengwa ndani kama `sorted()` katika mafundisho yajayo.

Njia inayofuata tutakayozungumzia ni njia ya `reverse()`. Njia hii, itageuza orodha ya vipengele mahali pake kama ifuatavyo:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

Njia ya mwisho tutakayozungumzia ni njia ya `index`. Hii hutumika kupata kielezo cha kwanza ambacho kipengele kinaweza kupatikana katika orodha. Hapa kuna mfano wa kutumia njia ya `index` kutafuta lugha ya `'Java'` katika orodha ya `programming_languages`:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

Kama kipengele hakiwezi kupatikana, basi Python hutupa `ValueError`:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: 'JavaScript' is not in list
"""
```

Kuna njia chache zaidi za orodha za Python, lakini orodha hii ya awali ya njia ni mahali pazuri pa kuanza.
