# --explanation--

`Every month` significa que algo sucede una vez al mes, de forma regular. Por ejemplo:

Hacemos una copia de seguridad de los datos de nuestro sistema cada mes para prevenir la pérdida de datos.

`Every two weeks` significa que algo sucede una vez cada dos semanas, o dos veces al mes. Por ejemplo:

El equipo de TI revisa las actualizaciones de seguridad cada dos semanas para mantener los sistemas actualizados. Esta frase destaca la importancia de verificar las actualizaciones dos veces al mes para asegurar que los sistemas permanezcan seguros y funcionen correctamente.

`Vigilant` significa estar atento y cuidadoso para evitar riesgos o problemas. Por ejemplo:

`Employees must stay vigilant when opening emails to avoid phishing scams.` - Esta frase enfatiza la necesidad de que los empleados presten atención y sean cautelosos al manejar correos electrónicos para evitar ser víctimas de estafas o ataques.
