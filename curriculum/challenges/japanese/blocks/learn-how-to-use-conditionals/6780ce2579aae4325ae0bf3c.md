# --explanation--

`Critical`は非常に重要または不可欠なものを意味し、`critical data`はチームが効果的に作業するために頼りにしている重要な情報を指します。これを失うと遅れや問題が生じます。例えば：

- `The backup includes all critical data in case of a failure.` - 保存されたコピーには問題から回復するための重要な情報が含まれています。

- `You must encrypt critical data to ensure it's secure.` - 重要な情報を暗号化で保護することはセキュリティ上必要です。
