# --description--

`function` 是一组执行特定任务的指令。`Functions` 可以使代码更有条理、可重用且更易于读取。例如，`function` 可能计算两个数字的和或向用户显示一条消息。

把 `function` 想象成工厂里的一个小机器；你给它一些输入，它对这些输入做些处理，然后给你一个输出。
