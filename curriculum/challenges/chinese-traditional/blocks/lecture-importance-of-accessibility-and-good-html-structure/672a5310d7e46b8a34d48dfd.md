# --description--

螢幕閱讀器是協助盲人和視障人士使用電腦和行動裝置的輔助技術程式。

螢幕閱讀器不僅是盲人和視障者用來存取電腦和行動裝置的工具。

他們使這些個人能夠存取教育、工作機會和社群媒體。這確保了數位包容性並提升他們全面參與社會的能力。

有一個常見的誤解是認為螢幕閱讀器是文字轉語音的裝置。

然而，文字轉語音只是螢幕閱讀器的其中一項特性。有些螢幕閱讀器甚至會將文字呈現為點字輸出，而非語音。

除了文字轉語音和點字輸出之外，螢幕閱讀器的其他顯著特性是導覽輔助和網頁瀏覽協助。

螢幕閱讀程式不僅僅是為盲人和視障者設計的。閱讀障礙者和認知障礙者也會使用螢幕閱讀器。所有流行的作業系統都內建了螢幕閱讀器。

macOS 和 iOS 都內建了 VoiceOver。你可以在你的電腦上按下 `CMD + F5` 來開啟它。你可以透過設定在 iPhone 上存取它。

Windows 電腦內建有 Narrator。你可以按下 `WIN + CTRL + ENTER` 來開啟它。NonVisual Desktop Access（NVDA）和 Job Access With Speech（JAWS）也適用於 Windows 電腦。NVDA 是免費且開放原始碼的，而 JAWS 是付費的。

Linux 在桌面環境中有 *Orca*，在 Linux 終端機中有 *Speakup*。

Android 手機內建了 TalkBack。你可以透過存取 *Settings > Special Function > Accessibility > Talkback* 來開啟它。

某些 Android 裝置也內建有 Ella 和 Select to Speak。

對於螢幕閱讀器使用者來說，一個主要的挑戰是許多軟體開發者在設計產品時沒有考慮到螢幕閱讀器的友善性和無障礙性。

雖然無障礙性是一個廣泛的主題，但每個開發者都需要學習如何讓他們的網頁軟體對盲人和視障者，以及其他殘障群組可存取。

這展現了同理心和對包容性的承諾，確保所有使用者都能從他們的工作中獲得利益。
