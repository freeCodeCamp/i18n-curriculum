# --description--

Der letzte Faktor, das Ende der Reihe, der Entscheidungsfaktor des Entscheidungsfaktors. Angenommen, nachdem alle anderen Faktoren berücksichtigt wurden, gibt es immer noch mehrere widersprüchliche Regeln, die auf ein Element abzielen. Wie bestimmt die Kaskade, welche Regel angewendet wird?

Ganz einfach eigentlich. Die Regel, die zuletzt definiert wurde, gewinnt.

```css
/* styles.css */

.alert {
  color: red;
}

.warning {
  color: yellow;
}
```

Für ein Element, das sowohl die `alert`- als auch die `warning`-Klasse hat, würde die Kaskade alle anderen Faktoren durchlaufen, einschließlich Vererbung (hier keine) und Spezifität (keine der Regeln ist spezifischer als die andere). Da die `.warning`-Regel zuletzt definiert wurde und kein anderer Faktor bestimmen konnte, welche Regel angewendet wird, ist es diejenige, die auf das Element angewendet wird.
