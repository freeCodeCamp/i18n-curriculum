# --interactive--

`aria-describedby` 속성은 화면 읽기 프로그램 사용자가 페이지에 이미 있는 콘텐츠를 참조하여 요소에 대한 추가 정보를 제공할 때 사용합니다. 이 속성은 요소와 콘텐츠 사이에 프로그래밍적으로 연결(기술적으로 접근 가능한 설명이라고 함)을 만들어, 사용자가 요소와 상호작용할 때 화면 읽기 프로그램이 추가 정보를 알릴 수 있게 합니다.

`aria-describedby`의 가장 일반적인 용도는 양식 입력과 관련된 지침과 오류 메시지를 연결하는 것입니다. 화면 읽기 프로그램 사용자가 페이지를 탐색하는 다양한 방법 때문에 입력 필드 사이를 이동할 때 이러한 메시지를 놓칠 수 있습니다. `aria-describedby`를 사용하면 이 메시지를 반드시 들을 수 있도록 도와줍니다.

작동 방식을 이해하기 위해 몇 가지 예제를 살펴보겠습니다. 첫 번째 예제에서는 비밀번호를 입력받는 `form` 요소가 있습니다.

인터랙티브 편집기를 활성화하고 비밀번호 입력 필드에 몇 글자를 입력하세요. 미리보기 창에서 비밀번호가 가려지는 것을 볼 수 있습니다. 또한 `password-help` 텍스트가 입력 필드에 8자 이상 입력할 때까지 빨간색으로 유지되는 것도 확인할 수 있습니다.

참고: 이 인터랙티브 예제는 CSS와 JavaScript를 사용해 `password-help` 텍스트의 색상을 동적으로 업데이트합니다. JavaScript 코드를 이해하려고 걱정하지 마세요. 앞으로 모듈에서 JavaScript를 배우게 됩니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<form>
  <label for="password">Password:</label>
  <input type="password" id="password" aria-describedby="password-help" />
  <p id="password-help">Your password must be at least 8 characters long.</p>
</form>

<script src="index.js"></script>
```

```css
#password-help {
  color: red;
}
```

```js
const passwordEl = document.getElementById("password");
const passwordHelpText = document.getElementById("password-help");

passwordEl.addEventListener("input", (e) => { 
  const userInput = e.target.value;
  passwordHelpText.style.color = userInput.length >= 8 ? "green" : "red";
});
```

:::

`label` 요소를 `Password` 텍스트에 사용하고, 이를 비밀번호 `input` 필드와 연결하고 있습니다.

또한 비밀번호 요구사항을 설명하는 단락 요소가 있습니다. 비밀번호 `aria-describedby` 필드와 단락 요소의 비밀번호 요구사항을 연결하기 위해 `input` 속성을 사용합니다. 화면 읽기 프로그램 사용자가 이 `input`과 상호작용하면, 화면 읽기 프로그램이 `input`의 이름(`Password`)을 알리고, 이어서 비밀번호 요구사항을 알릴 수도 있습니다. 다만, 일부 화면 읽기 프로그램은 추가 콘텐츠를 자동으로 알리지 않거나 특정 상황에서만 알릴 수 있으므로 절대적인 보장은 아닙니다. 이는 흔한 일이며, 각 화면 읽기 프로그램은 ARIA 속성을 다르게 처리합니다. 그렇다고 `aria-describedby` 사용이 무의미한 것은 아니며, 일반적으로 화면 읽기 프로그램 사용자에게 도움이 됩니다.

`aria-describedby` 속성의 또 다른 좋은 활용 사례는 삭제 `button`가 있을 때입니다. 다음은 삭제 `button` 버튼과 버튼을 클릭했을 때 발생할 일을 설명하는 메시지 예제입니다:

:::interactive_editor

```html
<button aria-describedby="delete-message">Delete</button>

<p id="delete-message">Warning! All deletions are permanent.</p>
```

:::

앞선 예제와 마찬가지로, 삭제 버튼과 메시지를 `aria-describedby` 속성으로 연결합니다. `id` 값과 `aria-describedby` 속성의 값은 일치해야 합니다.

`aria-describedby` 속성은 요소에 대한 추가 정보를 화면 읽기 프로그램 사용자에게 제공할 수 있도록 도와주는 강력한 속성입니다. 주로 양식 입력과 관련된 지침과 오류 메시지를 연결해, 화면 읽기 프로그램 사용자가 양식을 탐색할 때 이러한 메시지를 놓칠 가능성을 줄이는 데 사용됩니다.
