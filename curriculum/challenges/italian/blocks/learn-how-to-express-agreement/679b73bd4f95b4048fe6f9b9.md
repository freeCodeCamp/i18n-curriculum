# --explanation--

Un `sketch` è un disegno approssimativo usato per pianificare o visualizzare un'idea. Per esempio:

`She made a quick sketch of the website layout before designing it digitally.` – Questo significa che ha creato un disegno semplice prima di iniziare il progetto vero e proprio.

Un `wireframe` è una semplice struttura o schema di un progetto digitale, che mostra la struttura senza dettagli. Per esempio:

`The team created wireframes to plan the website's navigation before adding colors and images.` – Questo significa che hanno prima progettato una struttura di base prima di lavorare sui dettagli visivi.
