# --explanation--

Ви використовуєте структуру `Second Conditional` (у промові Марії `It'd be helpful if we had...`) для опису гіпотетичної ситуації. Структура другого умовного речення — `It would` + базове дієслово ... `if` + підмет + дієслово у `Past Simple`. Вона використовується для опису нереальних або гіпотетичних ситуацій у теперішньому або майбутньому часі. Наприклад:

`We would finish earlier if we had more team members.` — це означає, що у них немає більше членів команди, але якби були, вони могли б закінчити раніше. Ми говоримо про гіпотезу.
