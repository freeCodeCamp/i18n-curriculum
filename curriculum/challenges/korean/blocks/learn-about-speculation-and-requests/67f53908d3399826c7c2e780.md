# --explanation--

`Want`는 무언가를 하고자 하는 욕구를 표현합니다. 과거에는 `wanted`를 사용해 그 욕구가 이전에 존재했음을 나타냅니다. 그 뒤에는 동사의 원형(부정사, `to` 포함)이 옵니다. 예를 들어:

- `She wants to install the new software.` - 그녀는 시스템에 새 소프트웨어를 설치하거나 추가하기를 원합니다.

- `They wanted to upgrade their system last week.` - 과거에 그들은 업그레이드할 계획이었습니다.
