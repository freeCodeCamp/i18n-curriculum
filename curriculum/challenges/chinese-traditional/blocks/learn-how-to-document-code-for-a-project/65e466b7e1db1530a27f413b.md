# --description--

在電腦檔案系統中，`root directory` 指的是主要或頂層目錄。它就像檔案櫃的最上層抽屜，包含所有其他資料夾和檔案。

`main folder` 是專案或一組檔案所儲存的主要資料夾。

`subfolder` 就像是在另一個資料夾內的較小抽屜。

例如，在名為 `MyProject` 的專案中，`root directory` 將會是 `MyProject` 資料夾，而其內的任何資料夾，例如 `Images`，將會是 `subfolder`。

## --text--

在專案的背景關係中，什麼是 `root directory`？

## --answers--

主要資料夾

---

子資料夾

### --feedback--

子目錄是在另一個目錄內。`root directory` 是主要目錄，不是子目錄。

---

一種檔案型別

### --feedback--

`root directory` 是一個資料夾，而不是檔案型別。它是專案中存放檔案和資料夾的主要位置。

---

一種程式設計語言

### --feedback--

`root directory` 是檔案系統中的一個概念，不是程式語言。它指的是專案的主要資料夾。
