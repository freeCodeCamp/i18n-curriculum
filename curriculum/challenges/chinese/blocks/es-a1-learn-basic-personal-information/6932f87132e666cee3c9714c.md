# --description--

Ángela 使用结构 `¿Cuál es tu` + 名词 提问。

她本可以停在那里。相反，她使用连接词 `y` 和第二个名词添加了另一条信息。

`y` 是用来连接两个名词的连词。她使用的结构如下：

`¿Cuál es tu` + noun + `y` + noun`？`

这允许说话者在同一个问题中请求多个细节。

# --explanation--

在句子 `¿Cuál es tu nombre y apellido?` 中，两个名词由 `y` 连接：

- `nombre`

- `apellido`

单词 `y` 连接两条信息。

Ángela 首先询问该人的 `nombre`，然后添加 `apellido`。 

该结构允许说话者在一个问题中请求多个细节：

`¿Cuál es tu correo y número?`

这里，Ángela 请求的第二条 `apellido` 信息。
