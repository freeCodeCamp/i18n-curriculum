# --explanation--

`To struggle` 意味著在某事上有困難。例如：

- `The program is struggling to load.`－這表示該程式目前在載入時遇到困難。

`Present Continuous` 時態用來描述**正在發生的**動作。

在上述範例中，請注意當使用動詞 `struggle` 形成 `Present Continuous` 時，最後的 `e` 被省略，變成 `struggling`。這是以 `e` 結尾的動詞常見的規則。

在 Tom 的句子中，他使用 `Present Continuous` 時態來描述他在使用整合開發環境（Integrated Development Environments）時持續遇到的困難。

你應該為你的 `struggle` 添加 `-ing`，因為動作正在發生中。
