# --interactive--

Unapofanya kazi na mfuatano wa herufi katika JavaScript, kuna nyakati unahitaji kujumuisha herufi maalum ambazo engine ya JavaScript vinginevyo inaweza kuzitafsiri vibaya.

Zoezi mbili za kawaida ni kuunda mstari mpya ndani ya mfuatano wa herufi na kutoroka baadhi ya herufi (kama nukuu) ili kuhakikisha zinaonekana kwa usahihi.

Katika lugha nyingi za programu, ikiwa ni pamoja na JavaScript, unaweza kuunda mstari mpya katika mfuatano wa herufi kwa kutumia herufi maalum inayoitwa mfuatano wa kutoroka. Mfuatano wa kutoroka wa kawaida kwa mistari mipya ni `\n`.

Kwa mfano, ikiwa unataka kuvunja mfuatano wa herufi kuwa mistari mingi, utatumia `\n` mahali unapotaka mstari mpya uanze:

:::interactive_editor

```js
let poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

Mfuatano wa kutoroka `\n` unaambia JavaScript kuweka mapumziko ya mstari hapo, jambo linalosababisha mfuatano wa herufi kuonyeshwa katika mistari mingi.

Dhana nyingine muhimu unapotumia mfuatano wa herufi ni kutoroka herufi. Wakati mwingine, unahitaji kujumuisha herufi katika mfuatano wako ambazo JavaScript kawaida hutumia kwa kitu kingine, kama nukuu.

Ikiwa utatumia nukuu ndani ya mfuatano wa herufi bila kuzitoroka, inaweza kusababisha kosa kwa sababu JavaScript itadhani unajaribu kumaliza mfuatano wa herufi.

Kwa mfano, hii itasababisha kosa:

```js
let statement = "She said, "Hello!"";
```

JavaScript inachanganyikiwa kwa sababu inadhani mfuatano wa herufi unamalizika baada ya neno `"said,"` lakini, unataka nukuu zilizo karibu na `"Hello!"` ziwe sehemu ya mfuatano wa herufi.

Ili kurekebisha hili, unaweza kutoroka nukuu za ndani kwa kuweka mkwaju wa nyuma (`\`) kabla yake:

:::interactive_editor

```js
let statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

Mkwaju wa nyuma unaambia JavaScript kutambua nukuu kama herufi halisi, hivyo zinaonekana kwa usahihi katika matokeo.

Unaweza pia kutoroka herufi maalum nyingine, kama mkwaju wa nyuma mwenyewe (`\\`), au nukuu moja ndani ya mfuatano wa herufi uliozungukwa na nukuu moja (`\'`).

Hapa kuna mfano mwingine ukitumia nukuu moja:

:::interactive_editor

```js
let quote = 'It\'s a beautiful day!';
console.log(quote); // It's a beautiful day!
```

:::

Kwa kutoroka nukuu moja kwa `\'`, JavaScript inajua kuijumuisha kama sehemu ya mfuatano wa herufi badala ya kumaliza mfuatano mapema.

Kutoroka na kuunda mistari mipya ni muhimu unapopanga matokeo au kushughulikia herufi maalum katika mfuatano wa herufi. Mbinu hizi zinakusaidia kuzuia makosa na kuhakikisha maandishi yako yanaonekana kama ulivyokusudia.
