# --description--

Un numero 3-smooth è un intero che non ha fattori primi maggiori di 3. Per un intero $N$, definiamo $S(N)$ come l’insieme dei numeri 3-smooth minori o uguali a $N$. Per esempio, $S(20) = \\{1, 2, 3, 4, 6, 8, 9, 12, 16, 18\\}$.

Definiamo $F(N)$ come il numero di permutazioni di $S(N)$ in cui ogni elemento viene dopo tutti i suoi divisori propri.

Questa è una delle possibili permutazioni per $N = 20$.

-   1, 2, 4, 3, 9, 8, 16, 6, 18, 12.

Questa non è una permutazione valida perché 12 viene prima del suo divisore 6.

-   1, 2, 4, 3, 9, 8, 12, 16, 6, 18.

Possiamo verificare che $F(6) = 5$, $F(8) = 9$, $F(20) = 450$ e $F(1000) ≈ 8.8521816557e\\,21$.

Trova $F({10}^{18})$. Fornisci la tua risposta come stringa in notazione scientifica arrotondata a dieci cifre decimali. Quando fornisci la risposta, usa una `e` minuscola per separare mantissa ed esponente. Per esempio, se la risposta è $112\\,233\\,445\\,566\\,778\\,899$ il formato della risposta sarà `1.1223344557e17`.
