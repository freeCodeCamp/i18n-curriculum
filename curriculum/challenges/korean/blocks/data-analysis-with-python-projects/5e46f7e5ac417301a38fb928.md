# --instructions--

Numpy를 사용해 3 x 3 행렬의 행, 열, 요소별로 평균, 분산, 표준편차, 최대값, 최소값, 합계를 출력하는 `calculate()` 안에 `mean_var_std.py`라는 이름의 함수를 만드세요.

함수의 입력은 9개의 숫자를 담은 리스트여야 합니다. 함수는 이 리스트를 3 x 3 Numpy 배열로 변환한 뒤, 두 축과 평탄화된 행렬에 대해 평균, 분산, 표준편차, 최대값, 최소값, 합계를 포함하는 사전을 반환해야 합니다.

반환하는 사전은 다음 형식을 따라야 합니다:

```py
{
  'mean': [axis1, axis2, flattened],
  'variance': [axis1, axis2, flattened],
  'standard deviation': [axis1, axis2, flattened],
  'max': [axis1, axis2, flattened],
  'min': [axis1, axis2, flattened],
  'sum': [axis1, axis2, flattened]
}
```

9개 미만의 요소가 들어있는 리스트가 함수에 전달되면, "List must contain nine numbers."라는 메시지와 함께 `ValueError` 예외를 발생시켜야 합니다. 반환하는 사전의 값들은 리스트여야 하며 Numpy 배열이 아니어야 합니다.

예를 들어, `calculate([0,1,2,3,4,5,6,7,8])`는 다음을 반환해야 합니다:

```py
{
  'mean': [[3.0, 4.0, 5.0], [1.0, 4.0, 7.0], 4.0],
  'variance': [[6.0, 6.0, 6.0], [0.6666666666666666, 0.6666666666666666, 0.6666666666666666], 6.666666666666667],
  'standard deviation': [[2.449489742783178, 2.449489742783178, 2.449489742783178], [0.816496580927726, 0.816496580927726, 0.816496580927726], 2.581988897471611],
  'max': [[6, 7, 8], [2, 5, 8], 8],
  'min': [[0, 1, 2], [0, 3, 6], 0],
  'sum': [[9, 12, 15], [3, 12, 21], 36]
}
```
