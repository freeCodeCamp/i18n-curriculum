# --description--

CSS 將每個 HTML 元素視為自己的盒子，通常稱為 <dfn>CSS Box Model</dfn>。區塊層級的項目會自動從新行開始（例如標題、段落和 div），而行內項目則置於周圍內容中（例如影像或 span）。元素以這種方式的預設配置稱為文件的 <dfn>normal flow</dfn>，但 CSS 提供了 position 屬性來改寫它。

當元素的位置設定為 `relative` 時，允許你指定 CSS 如何相對於該元素在頁面正常流程中的當前位置移動它。它與 CSS 偏移屬性 `left` 或 `right`，以及 `top` 或 `bottom` 配對。這些屬性表示將該項目從其正常位置移動多少像素、百分比或 em。以下範例將段落從底部移動 10 像素：

```css
p {
  position: relative;
  bottom: 10px;
}
```

將元素的位置改為 relative 不會將它從正常流程中移除－周圍的其他元素仍會像該條款處於其預設位置一樣運作。

**注意：** 定位讓你對頁面的視覺佈局擁有很大的彈性和控制力。值得記住的是，無論元素的位置如何，底層的 HTML 標記應該要有組織且從上到下閱讀時要有意義。這是視覺障礙使用者（依賴像是 `screen readers` 這類輔助裝置）存取你內容的方式。

# --instructions--

將 `h2` 的 `position` 改為 `relative`，並使用 CSS 偏移將其從正常流程中所在位置的 `top` 移動 15 像素。注意這不會影響周圍的 h1 和 p 元素的位置。

# --hints--

`h2` 元素應該有一個 `position` 屬性設為 `relative`。

```js
const h2Element =document.querySelector('h2')
const h2Style = window.getComputedStyle(h2Element);
assert.equal(h2Style?.position, 'relative');
```

你的程式碼應該使用 CSS 偏移量，將 `h2` 相對定位，使其距離正常位置的 `top` 15px。

```js
const h2Element =document.querySelector('h2')
const h2Style = window.getComputedStyle(h2Element);
assert.equal(h2Style?.top, '15px');
```
