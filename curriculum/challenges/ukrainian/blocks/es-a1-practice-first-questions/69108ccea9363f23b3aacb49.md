# --explanation--

Цей діалог між Матео та Камілою фокусується на професіях.

`¿A qué te dedicas?` означає "Якою є твоя професія?" або "Чим ти займаєшся?".

**`Soy` + [професія]** означає "Я є [професія]".

В іспанській мові більшість професій змінюються залежно від роду. Однак деякі професії, що закінчуються на `-ista` або `-nte`, часто залишаються однаковими для обох родів:

| Професія | Чоловічий рід | Жіночий рід |
| :---: | :---: | :---: |
| Аналітик | `analista` | `analista` |
| Журналіст | `periodista` | `periodista` |
| Рецепціоніст | `recepcionista` | `recepcionista` |
| Співак | `cantante` | `cantante` |
