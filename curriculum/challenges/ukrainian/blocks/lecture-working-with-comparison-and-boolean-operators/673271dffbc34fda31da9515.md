# --interactive--

Оператори порівняння дозволяють порівняти два значення і повернути результат `true` або `false`. Потім ви можете використати цей результат, щоб прийняти рішення або контролювати потік вашої програми. Ви використовуєте порівняння в інструкціях `if`, циклах і багатьох інших ситуаціях, коли потрібно приймати рішення на основі певних умов. Давайте розглянемо найпоширеніші оператори порівняння і подивимося, як вони працюють.

Оператор «більше ніж», представлений правою кутовою дужкою (`>`), перевіряє, чи є значення зліва більшим за значення справа:

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a > b); // false
console.log(b > a); // true
```

:::

Оператор «більше або дорівнює», представлений правою кутовою дужкою та знаком рівності (`>=`), перевіряє, чи є значення зліва більшим або рівним значенню справа:

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a >= b); // false
console.log(b >= a); // true
console.log(a >= c); // true
```

:::

Оператор «менше ніж», представлений лівою кутовою дужкою (`<`), працює подібно до `>`, але навпаки. Він перевіряє, чи є значення зліва меншим за значення справа:

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a < b); // true
console.log(b < a); // false
```

:::

Оператор «менше або дорівнює», представлений лівою кутовою дужкою та знаком рівності (`<=`), перевіряє, чи є значення зліва меншим або рівним значенню справа:

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a <= b); // true
console.log(b <= a); // false
console.log(a <= c); // true
```

:::
