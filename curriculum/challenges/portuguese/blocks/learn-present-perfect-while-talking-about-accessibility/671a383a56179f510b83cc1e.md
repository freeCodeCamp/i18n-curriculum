# --explanation--

A resposta de Tom, `We haven't been receiving as many negative comments`, usa o `Present Perfect Continuous` na forma negativa.  This tense is formed by `haven't/hasn't + been + verb ending in -ing` and is used to describe actions that have not been happening over a period of time. Neste caso, mostra que menos comentários negativos foram recebidos ao longo do tempo, indicando uma possível melhoria.
