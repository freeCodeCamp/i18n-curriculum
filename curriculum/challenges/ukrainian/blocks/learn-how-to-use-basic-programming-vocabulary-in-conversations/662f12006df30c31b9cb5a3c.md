# --description--

`concept` зазвичай означає ідею або принцип, який використовується для планування чи визначення дій і операцій. У різних предметах, включно з програмуванням, `concepts` є фундаментальними ідеями або теоріями.

Наприклад, `Object-oriented programming is a concept in computer programming.`
