## Теорія кольору

- **Визначення теорії кольору**: Це вивчення того, як кольори взаємодіють між собою і як вони впливають на наше сприйняття. Вона охоплює відносини кольорів, гармонію кольорів і психологічний вплив кольору.
- **Основні кольори**: Ці кольори — жовтий, синій і червоний — є фундаментальними відтінками, з яких походять усі інші кольори.
- **Вторинні кольори**: Ці кольори утворюються шляхом змішування рівних частин двох основних кольорів. Прикладами вторинних кольорів є зелений, помаранчевий і фіолетовий.
- **Третинні кольори**: Ці кольори утворюються шляхом поєднання основного кольору з сусіднім вторинним кольором. Прикладами третинних кольорів є жовто-зелений, синьо-зелений і синьо-фіолетовий.
- **Теплі кольори**: Ці кольори, до яких належать червоні, помаранчеві та жовті, викликають відчуття комфорту, тепла і затишку.
- **Холодні кольори**: Ці кольори, до яких належать сині, зелені та фіолетові, викликають відчуття спокою, умиротворення і професіоналізму.
- **Колірне коло**: Колірне коло — це кругова діаграма, яка показує, як кольори пов’язані між собою. Це важливий інструмент для дизайнерів, оскільки допомагає їм вибирати кольорові поєднання.
- **Аналогові кольорові схеми**: Ці схеми створюють гармонійні та заспокійливі враження. Вони містять аналогові кольори, які розташовані поруч один з одним на колірному колі.
- **Доповнюючі кольорові схеми**: Ці схеми створюють високий контраст і візуальний ефект. Їхні кольори розташовані на протилежних кінцях колірного кола.
- **Тріадична кольорова схема**: Ця схема має яскраві кольори. Вона складається з кольорів, які приблизно рівновіддалені один від одного. Якщо їх з’єднати, утвориться рівносторонній трикутник на колірному колі.
- **Монохроматична кольорова схема**: Для цієї схеми всі кольори походять від одного базового кольору шляхом регулювання його світлості, темряви та насиченості. Це викликає відчуття єдності та гармонії, одночасно створюючи контраст.

## Різні способи роботи з кольорами в CSS

- **Іменовані кольори**: Це кольори з попередньо визначеними назвами, які розпізнають браузери. Прикладами є `blue`, `darkred`, `lightgreen`.
- **Функція `rgb()`**: RGB означає Red, Green і Blue — основні кольори світла. Ці три кольори комбінуються з різною інтенсивністю, щоб створити широкий спектр кольорів. Функція `rgb()` дозволяє визначати кольори, використовуючи модель кольору RGB.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>RGB Color</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

- **Функція `rgba()`**: Ця функція додає четверте значення — альфа, яке контролює прозорість кольору. Якщо його не вказано, значення альфа за замовчуванням дорівнює `1`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>RGBA Background</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

- **Функція `hsl()`**: HSL означає Hue, Saturation і Lightness — три ключові компоненти, що визначають колір.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>HSL Color</p>
```

```css
p {
  color: hsl(120, 100%, 50%);
}
```

:::

- **Функція `hsla()`**: Ця функція додає четверте значення — альфа, яке контролює непрозорість кольору.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>HSLA Background</div>
```


```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

- **Шістнадцятковий код**: Шістнадцятковий код (скорочено hex) — це рядок із шести символів, який використовується для представлення кольорів у моделі RGB. «Hex» означає систему числення з основою 16, яка використовує цифри від 0 до 9 і літери від A до F.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1 class="hex-text">Hex Text</h1>
<p class="hex-bg">Hex Background</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}

p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

## Властивість `box-shadow`

- **Визначення**: Властивість `box-shadow` застосовує одну або кілька тіней навколо елемента.
- **Значення зміщення**: Ви повинні вказати горизонтальні (`offset-x`) і вертикальні (`offset-y`) значення. Позитивне `offset-x` зміщує тінь вправо, а від’ємне — вліво. Позитивне `offset-y` зміщує тінь вниз, а від’ємне — вгору. Якщо значення дорівнює `0`, не потрібно вказувати одиниці виміру.
- **Радіус розмиття**: Це необов’язкове значення контролює, наскільки розмитою виглядає тінь. Якщо його не вказано, за замовчуванням використовується `0`, що створює чіткі краї. Чим більше значення, тим м’якша тінь.
- **Радіус розповсюдження**: Це необов’язкове значення контролює, наскільки тінь розширюється або звужується. Якщо його не вказано, за замовчуванням використовується `0`.
- **Колір тіні**: Ви можете вказати колір, використовуючи іменовані кольори, шістнадцяткові значення, `rgb()`, `rgba()`, `hsl()` або `hsla()` функції.
- **Ключове слово `inset`**: Додавання ключового слова `inset` розміщує тінь всередині елемента замість зовні.
- **Застосування кількох тіней**: Ви можете застосувати кілька тіней, розділяючи їх комами. Тіні накладаються одна на одну від переднього плану до заднього.

### Синтаксис

```css
box-shadow: offset-x offset-y blur-radius spread-radius color;
```

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="shadow-box">Shadow Color Example</div>
```

```css
.shadow-box {
  width: 200px;
  padding: 20px;
  background-color: lightblue;
  box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}
```

:::

## Лінійні та радіальні градієнти

- **Лінійні градієнти**: Ці градієнти створюють поступове змішування кольорів уздовж прямої лінії. Ви можете контролювати напрямок цієї лінії за допомогою ключових слів, таких як `to top`, `to right`, `to bottom right`, або кутів, як `45deg`. Ви можете використовувати будь-який дійсний CSS колір і стільки кольорових зупинок, скільки забажаєте.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient">Linear Gradient</div>
```

```css
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

:::

- **Радіальні градієнти**: Ці градієнти створюють кругові або еліптичні градієнти, що розходяться від центральної точки.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient">Radial Gradient</div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::
