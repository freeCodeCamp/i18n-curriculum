# --description--

Katika ukuzaji wa programu, maktaba ni kama sanduku la zana kwa wasanidi programu.

Badala ya kuhitaji kutekeleza kila sehemu ya msimbo mwenyewe kutoka mwanzo, maktaba inakupa msimbo uliotangulia kuandikwa na unaoweza kutumika tena, kama vile vitendakazi, madarasa ya vitu, na miundo ya data ambayo unaweza kutumia katika miradi yako.

Python ina maktaba ya kawaida pana yenye moduli nyingi zilizojengwa ndani. Zote ni suluhisho zilizopangwa na kuthibitishwa vizuri kwa changamoto na mazoezi mengi utakayokutana nayo kila siku kama msanidi programu, kama vile:

* Kuingiliana na mfumo wa uendeshaji.
    
* Kufanya kazi na mafaili.
    
* Mitandao.
    
* Kufanya kazi na tarehe na muda.
    
* Kutekeleza operesheni za kihesabu.
    
* Kutumia usemi wa kawaida.
    
* Kupima na kurekebisha makosa ya msimbo wako.
    
* Na mengi zaidi!
    

Mifano ya moduli maarufu zilizojengwa ndani ni `math`, `random`, `re` (mfupi wa "usemi wa kawaida"), na `datetime`.

Moduli ya `math` ina vitendakazi vinavyosaidia kutekeleza operesheni za kihesabu ngumu zaidi.

Moduli ya `random` ni msaada kwa kuzalisha nambari za bahati nasibu.

Moduli ya `re` hutumika kwa kufanya kazi na usemi wa kawaida.

Na moduli ya `datetime` ni msaada kwa kufanya kazi na tarehe na muda katika Python.

Lakini unaweza vipi kupata upatikanaji wa vigezo, thabiti, vitendakazi, na madarasa ya vitu yaliyoelezwa katika moduli hizi zilizojengwa ndani?

Unatumia tamko la import. Tamko hizi hukuruhusu kuleta moduli ndani ya skiripti yako ya Python. Tamko za import kwa kawaida huandikwa juu ya faili. Pia, unaweza kuzibadilisha kulingana na mahitaji yako. Kwanza, unatumia tamko la `import`, ikifuatiwa na jina la moduli:

```python
import module_name
```

Tuseme unataka kuleta moduli ya `math`. Katika hali hiyo, utaandika hivi juu ya faili yako:

```python
import math
```

Kisha, ikiwa unahitaji kuita kitendakazi kutoka moduli hiyo katika skiripti yako ya Python, utatumia uandishi wa nukta, ukiandika jina la moduli ikifuatiwa na jina la kitendakazi:

```python
module_name.function_name() 
```

Kwa mfano, kupata mzizi wa mraba wa 36, utaandika `math` ikifuatiwa na nukta kisha `sqrt`, kifupi cha mzizi wa mraba, na ndani ya mabano ya kawaida, utapita hoja yoyote inayohitajika. Katika kesi hii, tunahitaji kupita nambari tunayotaka mzizi wa mraba wake:

```python
math.sqrt(36)
```

Hii ni toleo la msingi kabisa la tamko la import, lakini kuna mbadala nyingine.

Ikiwa unahitaji kuleta moduli kwa jina tofauti (linalojulikana pia kama "jina la utani"), unaweza kutumia sintaksia hii, ukiandika `as` ikifuatiwa na jina la utani mwishoni mwa tamko la import:

```python
import module_name as module_alias
```

Hii hutumika mara nyingi kufupisha majina marefu ya moduli, au kuepuka migongano ya majina.

Kwa mfano, kurejelea moduli ya `math` kama `m` katika msimbo wako, unaweza kuweka jina la utani, kama hivi:

```python
import math as m 
```

Kisha, unaweza kupata vipengele vya moduli kwa kutumia jina la utani:

```python
m.sqrt(36)
```

Lakini wakati mwingine huna haja ya kuleta kila kitu kutoka moduli. Labda unahitaji vitendakazi au madarasa ya vitu maalum moja au viwili tu. Python ina kile unachohitaji katika hali hiyo.

Sasa tamko la import linaanza na `from`, ikifuatiwa na jina la moduli, kisha neno kuu `import` ikifuatiwa na majina ya vipengele unavyotaka kuleta:

```python
from module_name import name1, name2
```

Kisha, unaweza kutumia majina haya bila kiambishi awali cha moduli katika skiripti yako ya Python.

Ikiwa unataka kuweka majina haya kuwa na majina ya utani, unaweza kufanya hivyo kwa kutumia neno kuu `as` baada ya kila moja, ikifuatiwa na jina la utani unalotaka kutumia:

```python
from module_name import name1 as alias1, name2 as alias2
```

Tuseme unataka kuleta vitendakazi vya radians, sine, na cosine kutoka moduli ya `math` tu. Utaandika:

```python
from math import radians, sin, cos
```

Sasa unaweza kuita vitendakazi hivi moja kwa moja katika msimbo wako, bila moduli ya `math` kama kiambishi awali.

Hapa tuna mfano wa kina zaidi:

Kupata sine na cosine ya pembe maalum iliyowekwa kwa awali kwa digrii, tunaweza kuita kitendakazi cha `radians` kubadilisha kuwa radians, kisha kuita vitendakazi vya sine na cosine, tukipita pembe hiyo kwa radians:

```python
from math import radians, sin, cos

angle_degrees = 40
angle_radians = radians(angle_degrees)

sine_value = sin(angle_radians)
cos_value = cos(angle_radians)

print(sine_value) # 0.6427876096865393
print(cos_value)  # 0.766044443118978
```

Angalia jinsi tunavyowaita vitendakazi moja kwa moja, bila jina la moduli kama kiambishi awali. Hii ni kwa sababu tuliileta vitendakazi kwa sintaksia hii mbadala.

Hii ni msaada, lakini inaweza kusababisha migongano ya majina ikiwa tayari una vitendakazi au vigezo vyenye majina sawa vilivyoelezwa katika skiripti ya Python yenyewe. Hivyo ni jambo la kuzingatia unapoamua aina gani ya tamko la import unayotaka kutumia.

Na hatimaye, tunapata tamko la import linalomalizika na nyota. Nyota inamwambia Python kwamba unataka kuleta kila kitu katika moduli hiyo, lakini unataka kuileta ili usihitaji kutumia jina la moduli kama kiambishi awali:

```python
from module_name import *
```

Kwa mfano, ukifanya hivi wakati wa kuleta moduli ya `math`, utaweza kuita kitendakazi chochote kilichoelezwa katika moduli hiyo bila kutaja jina la moduli kama kiambishi awali. Hapa kuna mifano:

```python
from math import *
print(sqrt(36))  # 6.0
print(pow(5, 2)) # 25.0
print(exp(1))    # 2.718281828459045
```

Hata hivyo, hii kwa kawaida haipendekezwi kwa sababu inaweza kusababisha migongano ya majina, na kufanya iwe vigumu kujua majina fulani yanatoka wapi.

Tamko za import hufanya kazi sawa kwa vitendakazi, madarasa ya vitu, thabiti, vigezo, na vipengele vingine vyote vilivyoelezwa katika moduli.

Hapa kuna mfano wa thabiti kutoka moduli ya `math`, nambari `pi`:

```python
import math
print(math.pi)
```

Na hapa kuna mfano wa darasa la vitu kutoka moduli ya `datetime`. Tunaunda kitu cha tarehe kinachowakilisha Julai 15, 1959. Kisha, tunaweka kitu hicho cha `date` kwenye kigezo na kupata siku, mwezi, na mwaka kando kwa kutumia uandishi wa nukta:

```python
import datetime
birthday = datetime.date(1959, 7, 15)
print(birthday.day)    # 15
print(birthday.month)  # 7
print(birthday.year)   # 1959
```

Unaweza kupata maelezo zaidi kuhusu yaliyomo katika moduli hiyo katika nyaraka rasmi za Python kwa moduli hiyo.

Nzuri. Sasa unajua zaidi kuhusu moduli, unapaswa pia kujua kuhusu msemo huu muhimu sana katika skiripti za Python, kwa sababu wana uhusiano wa karibu sana:

```python
if __name__ == '__main__': 
    # Code
```

`__name__` ni kigezo maalum kilichojengwa ndani katika Python.

Wakati faili ya Python inatekelezwa moja kwa moja, Python huweka thamani ya kigezo hiki kuwa mfuatano wa herufi `"__main__"`.

Lakini ikiwa faili ya Python inaletwa kama moduli ndani ya skiripti nyingine ya Python, thamani ya kigezo cha `__name__` huwekwa kuwa jina la moduli hiyo (kwa kawaida jina la faili bila nyongeza ya `.py`).

Hii ndiyo sababu mara nyingi utapata masharti haya katika skiripti za Python. Yana msimbo unaotaka kuendesha **tu ikiwa** skiripti ya Python inaendeshwa kama programu kuu:

```python
if __name__ == '__main__': 
    # Code
```

Lakini ikiwa skiripti inaletwa kama moduli, msimbo ndani ya sehemu hiyo hauendi.

Hii ni msaada kwa sababu inaruhusu skiripti za Python kuwa na madhumuni mawili. Zinaweza kuendeshwa moja kwa moja kutekeleza mantiki kuu, au kuletwa ndani ya moduli nyingine bila kutekeleza mantiki kuu.
