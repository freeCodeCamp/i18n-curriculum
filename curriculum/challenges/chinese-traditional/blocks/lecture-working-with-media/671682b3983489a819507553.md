# --description--

首先，究竟什麼是 `iframe` 元素？ `iframe` 代表內聯框架。 它是一種內聯元素，用於在 HTML 頁面中直接嵌入其他 HTML 內容。 這些 HTML 內容可以是視頻、地圖、其他 HTML 元素，甚至是其他網頁。 下面是 `iframe` 元素的語法：

```html
<iframe
  src="video-url"
  width="width-value"
  height="height-value"
  allowfullscreen
></iframe>
```

`src` 屬性指定了要嵌入的頁面的 URL。 `width` 屬性指定 `iframe` 的寬度。 `height` 屬性指定 `iframe` 的高度。 `allowfullscreen` 屬性允許用戶以全屏模式顯示 `iframe` 。 爲 `iframe` 指定 `title` 屬性也是一種好的做法，因爲它對可訪問性非常重要。

要在 `iframe` 中嵌入視頻，你可以直接從 YouTube 和 Vimeo 等流行的視頻服務中複製視頻鏈接，或者使用指向視頻 URL 的 `src` 屬性自定義嵌入方式。 下面是一個嵌入來自 YouTube 的 freeCodeCamp 熱門課程的示例：

```html
<h1>A freeCodeCamp YouTube Video Embedded with the iframe Element</h1>

<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/PkZNo7MFNFg?si=-UBVIUNM3csdeiWF"
  title="YouTube video player"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
  referrerpolicy="strict-origin-when-cross-origin"
  allowfullscreen
></iframe>
```

稍後，你可以根據自己的需要自定義 `iframe` ，甚至可以通過複製 YouTube 視頻鏈接並將其粘貼爲 URL 值來定義自己的視頻。 需要注意的是，你必須將 https://youtu.be 替換爲 https://youtube.com/embed/。

youtu.be 是複製分享鏈接時的域名，而 youtube.com/embed/ 是使用 `iframe` 元素嵌入視頻時所需的域名。 這是一個示例：

```html
<iframe
  src="https://youtube.com/embed/gp5H0Vw39yw?si=Rb_2nDK6abv1iIAM"
  title="freeCodeCamp Typescript Course"
  width="500"
  height="285"
  allowfullscreen
></iframe>
```

注意，視頻可以來自任何地方。 它不一定要來自 YouTube 和 Vimeo 等視頻服務。 下面是一段用 `iframe` 元素嵌入的 Pixabay 視頻：

```html
<h1>A Video from Pixabay Embedded with the iframe Element</h1>

<iframe
  src="https://cdn.pixabay.com/video/2022/07/24/125310-733046613_large.mp4"
  width="500"
  height="285"
></iframe>
```

別忘了，你還可以在 `iframe` 元素中嵌入地圖、其他網頁或直接嵌入 HTML。 下面是我使用 `iframe` 元素嵌入的地圖：

```html
<h1>A Map from Openstreetmap.org Embedded with the iframe Element</h1>

<iframe
  width="425"
  height="350"
  src="https://www.openstreetmap.org/export/embed.html?bbox=3.006134033203125%2C6.150112578753815%2C3.6357879638671875%2C6.749850810550778&amp;layer=mapnik"
  style="border: 1px solid black"
>
</iframe>
<br />
<small>
  <a href="https://www.openstreetmap.org/#map=11/6.4501/3.3210">
    View Larger Map
  </a>
</small>
```

如果你想在 `iframe` 元素中直接嵌入 HTML，則必須使用 `srcdoc` 屬性而不是 `src`。
