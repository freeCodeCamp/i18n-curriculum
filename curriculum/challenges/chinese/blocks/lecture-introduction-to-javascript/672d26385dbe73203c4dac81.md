# --interactive--

JavaScript 是一种功能强大的编程语言，可为网站带来交互性和动态行为。

HTML 和 CSS 是标记语言，用于在页面上构建内容和元素样式，而 JavaScript 则超越了这两种语言，可实现更复杂的功能，如处理用户输入、动画元素，甚至构建完整的网络应用程序。

例如，当你点击按钮、提交表单或将鼠标悬停在菜单上时，JavaScript 会决定页面的行为方式。

下面举例说明这三者如何协同工作：

:::interactive_editor

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: green;
    }
  </style>
</head>
<body>
  <h1>Hello, World!</h1>
  <button onclick="alert('Button clicked!')">Click me</button>
</body>
</html>
```

:::

在此示例中，使用超文本标记语言来定义内容：一个标题（一个 `h1` 元素）和一个按钮（`button` 元素）。

CSS 用于为标题应用样式，例如将文本设置为绿色。 JavaScript 用于在点击按钮时显示提示信息。

总结来说，HTML 提供结构，CSS 增加样式，JavaScript 实现行为，从而创造出完整的交互式网络体验。
