# --description--

Das Muster `be`, gefolgt von der `-ing`-Form eines Verbs, wird verwendet, um über Handlungen zu sprechen, die gerade im Moment stattfinden.

Der Ausdruck `have trouble with` wird verwendet, um Schwierigkeiten im Umgang mit einem bestimmten Problem oder Objekt auszudrücken. Zum Beispiel bedeutet `She is having trouble with her computer`, dass sie Schwierigkeiten bei der Nutzung ihres Computers hat.
