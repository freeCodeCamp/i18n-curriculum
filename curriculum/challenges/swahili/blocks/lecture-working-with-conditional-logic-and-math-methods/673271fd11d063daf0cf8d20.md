# --interactive--

Taarifa za masharti hukuruhusu kufanya maamuzi katika msimbo wako wa JavaScript. Zinakuwezesha programu yako kuendelea kwa njia fulani kulingana na masharti maalum. Hebu tuchunguze jinsi `if`, `else if`, `else`, na opereta ya ternary zinavyofanya kazi kukuwezesha kudhibiti mtiririko wa msimbo wako.

Taarifa ya `if` huchukua hali fulani na kuendesha kifungu cha msimbo ikiwa hali hiyo ni thabiti. Thamani za kweli ni thamani yoyote inayosababisha `true` wakati zinapochunguzwa katika muktadha wa thamani ya kweli au siyo kweli kama katika taarifa ya `if`. Hapa kuna mifano ya thamani za kweli:

- mfuatano wa herufi usio tupu, kwa mfano, `hello`

- nambari yoyote isipokuwa `0` na `-0`, kwa mfano, `4`, `-5`, na nyinginezo

- safu za data

- vitu

- thamani ya kweli au siyo kweli `true`

Kwa upande mwingine, thamani za uongo ni thamani zinazochunguzwa kuwa `false` katika muktadha wa thamani ya kweli au siyo kweli. JavaScript ina thamani chache za uongo, jambo linalowafanya kuwa rahisi kuzikumbuka. Hapa kuna baadhi ya thamani za uongo:

- thamani ya kweli au siyo kweli `false`

- `0` (sifuri)

- `""` (mfuatano wa herufi tupu)

- `null`

- `undefined`

- `NaN` (Sio Nambari)

Sasa, tukiwa na uelewa wa msingi wa thamani za kweli na uongo, tuchunguze jinsi inavyofanya kazi na taarifa za `if`. Katika mfano huu wa kwanza, tunatumia taarifa chache za `if` kuangalia dhidi ya thamani za kweli na uongo:

:::interactive_editor

```js
if (null) {
  console.log("This will not run.");
}

if ("freeCodeCamp") {
  console.log("This will run.");
}
```

:::

Kwa kuwa `null` ni thamani ya uongo, ujumbe ulioko ndani ya kifungu hautawahi kuandikwa kwenye konsoli. Lakini kwa taarifa ya pili ya `if`, mfuatano wa herufi `freeCodeCamp` ni thamani ya kweli, na itachukuliwa kuwa `true` katika muktadha huu wa thamani ya kweli wa taarifa ya `if`. Matokeo yake, ujumbe `This will run.` utaandikwa kwenye konsoli.

Hebu tuchunguze mifano mingine kadhaa jinsi taarifa za `if` zinavyofanya kazi na opereta tofauti za kulinganisha. Hapa kuna mfano wa kutumia taarifa ya `if` kuangalia kama mtumizi anastahili kupiga kura:

:::interactive_editor

```js
const age = 22;

if (age >= 18) {
 console.log("You're eligible to vote"); // You're eligible to vote
}
```

:::

Katika mfano huu, kwa kuwa `age` kwa sasa ni `22`, hii inamaanisha hali itachunguzwa kuwa `true` kwa sababu `22` ni kubwa au sawa na `18`. Hivyo ujumbe `You're eligible to vote` utaandikwa kwenye konsoli. Ikiwa tutabadilisha mfano ili `age` sasa iwe `15`, basi hali itachunguzwa kuwa `false` na ujumbe hautaandikwa kwenye konsoli:

:::interactive_editor

```js
const age = 15;

if (age >= 18) {
 console.log("You're eligible to vote"); // Code not running because age is less than 18
}
```

:::

Wakati hali ni `false`, basi unaweza kutumia kifungu cha `else`:

:::interactive_editor

```js
const age = 15;

if (age >= 18) {
 console.log("You're eligible to vote");
} else {
 console.log("You're not eligible to vote"); // You're not eligible to vote
}
```

:::

Katika mfano huu, `15` si kubwa au sawa na `18`, hivyo hali itakuwa `false`. Msimbo ulioko ndani ya kifungu cha `else` utaendeshwa katika kesi hii.

Ikiwa unataka kuangalia masharti mengi, unaweza kutumia kifungu cha `else if`. Hii inaruhusu programu yako kuchagua kati ya njia zaidi ya mbili.

:::interactive_editor

```js
const score = 87;

if (score >= 90) {
 console.log('You got an A'); 
} else if (score >= 80) {
 console.log('You got a B'); // You got a B
} else if (score >= 70) {
 console.log('You got a C');
} else {
 console.log('You failed! You need to study more!');
}
```

:::

Kwa kuwa `score` kwa sasa ni `87`, basi ujumbe wa `You got a B` utaandikwa kwenye konsoli.

Opereta ya ternary ni njia fupi ya kuandika taarifa rahisi za `if/else`. Ina sehemu tatu: hali, matokeo ikiwa hali ni kweli, na matokeo ikiwa ni uongo. Hapa ni sintaksia ya msingi:

```js
condition ? expressionIfTrue : expressionIfFalse;
```

Hapa kuna mfano unaohusu halijoto ya hali ya hewa kwa Celsius:

:::interactive_editor

```js
const temperature = 30;
const weather = temperature > 25 ? 'sunny' : 'cool';

console.log(`It's a ${weather} day!`);
```

:::

Ikiwa `temperature` ni kubwa kuliko `25`, msimbo hapo juu utaandika `It's a sunny day!`. Ikiwa `temperature` itakuwa ndogo au sawa na `25`, itaandika `It's a cool day!`.

Basi, ni ipi unapaswa kutumia kati ya taarifa ya if na opereta ya ternary? Tumia opereta ya ternary unapotumia hali moja au usemi mmoja, au unapotaka sintaksia fupi kwa mantiki rahisi. Tumia taarifa za `if/else` unapotumia masharti magumu na taarifa nyingi, kwani mambo yanakuwa magumu kusomeka ikiwa unapanga ndani opereta za ternary.
