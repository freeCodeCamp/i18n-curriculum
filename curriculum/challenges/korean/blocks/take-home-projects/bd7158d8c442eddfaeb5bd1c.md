**목표:** 다음과 기능적으로 유사한 앱을 만드세요: <a href="https://codepen.io/freeCodeCamp/full/obYBjE" target="_blank" rel="noopener noreferrer nofollow">https://codepen.io/freeCodeCamp/full/obYBjE</a>.

아래 사용자 스토리를 충족하고 모든 테스트를 통과하세요. 필요한 라이브러리나 API를 자유롭게 사용하세요. 자신만의 스타일을 더해도 좋습니다.

**사용자 스토리:** 무작위 버튼 누름 시퀀스가 제시됩니다.

**사용자 스토리:** 버튼 누름 시퀀스를 올바르게 입력할 때마다, 이전 시퀀스에 한 단계가 추가된 동일한 시퀀스를 볼 수 있습니다.

**사용자 스토리:** 버튼 누름 시퀀스가 재생될 때와 직접 버튼을 누를 때마다 각 버튼에 해당하는 소리가 들립니다.

**사용자 스토리:** 잘못된 버튼을 누르면 알림이 표시되고, 해당 버튼 누름 시퀀스가 다시 시작되어 패턴을 기억할 수 있도록 도와줍니다.

**사용자 스토리:** 현재 버튼 누름 시퀀스에 몇 단계가 있는지 볼 수 있습니다.

**사용자 스토리:** 다시 시작하고 싶으면 버튼을 눌러 게임을 한 단계부터 다시 시작할 수 있습니다.

**사용자 스토리:** 엄격 모드에서 버튼을 잘못 누르면 알림이 표시되고, 게임이 새로운 무작위 버튼 누름 시퀀스로 다시 시작됩니다.

**사용자 스토리:** 20단계 시퀀스를 모두 맞히면 게임에서 승리하며, 승리 알림 후 게임이 다시 시작됩니다.
