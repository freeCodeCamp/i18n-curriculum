# --explanation--

`吗 (ma)` zeigt eine Frageform an. Es wird am Ende eines Aussagesatzes verwendet, um eine Aussage in eine Frage zu verwandeln. Zum Beispiel:

`你是开发者 (nǐ shì kāi fā zhě)。` ist eine Aussage und bedeutet „Sie sind ein Entwickler“. Wenn man `吗 (ma)` am Ende hinzufügt, wird daraus eine Frage: `你是开发者吗 (nǐ shì kāi fā zhě ma)？`, was „Sind Sie ein Entwickler?“ bedeutet.
