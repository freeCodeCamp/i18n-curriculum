# --description--

SEO, oder Suchmaschinenoptimierung, ist eine Praxis, die Webseiten so optimiert, dass sie sichtbarer werden und in Suchmaschinen höher ranken. Eine Möglichkeit, die SEO Ihrer Seite zu verbessern, besteht darin, eine kurze Beschreibung für die Webseite mit dem `meta`-Element bereitzustellen. Hier ist ein Beispiel für die Verwendung des meta-Elements, um eine Seitenbeschreibung für eine Garten-Website festzulegen:

```html
<meta
  name="description"
  content="Discover expert tips and techniques for gardening in small spaces, choosing the right plants, and maintaining a thriving garden."
/>
```

Indem das `name`-Attribut auf `description` gesetzt wird, wird sichergestellt, dass Browser, Suchmaschinen und andere Web-Tools diese Metadaten korrekt interpretieren. Das `content`-Attribut ist der Ort, an dem Sie Ihre Beschreibung einfügen. Es wird empfohlen, Ihre Beschreibungen kurz und prägnant zu halten. Das liegt daran, dass Suchmaschinen die Beschreibung oft basierend auf dem Layout der Ergebnisseite kürzen.

Ähnlich wie bei anderen Arten von `meta`-Elementen ist die `meta`-Beschreibung auf der Webseite selbst nicht sichtbar. Ein Ort, an dem die Seitenbeschreibung zu finden ist, ist im Snippet der Suchmaschinenergebnisseite. Hier sind einige Beispiele für Ergebnis-Snippets der freeCodeCamp-Subreddit und GitHub-Repositories:

```sh
r\FreeCodeCamp: This is the official subreddit for the freeCodeCamp.org community. Learn to
code for free together with millions of other people...
```

```sh
Our full-stack web development and machine learning curriculum is completely free and self-
paced. We have thousands of interactive coding challenges to help you...
```

In den Beispielen befinden sich die Seitenbeschreibungen jeweils direkt unter den Seitenlinks. Innerhalb von wenigen Sekunden können Nutzer einen allgemeinen Eindruck davon gewinnen, worum es auf der Seite geht, und entscheiden, ob sie auf die Links für weitere Informationen klicken.

Auch wenn `meta`-Beschreibungen das Ranking einer Seite in Suchmaschinen nicht direkt beeinflussen, kann eine starke Beschreibung zu mehr Traffic auf Ihrer Webseite führen.
