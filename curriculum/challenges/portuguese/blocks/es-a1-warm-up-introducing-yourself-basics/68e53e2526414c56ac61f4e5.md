# --description--

Em espanhol, o número 20 é `veinte`. Ao formar números de 21 a 29, a palavra muda ligeiramente. `Veinte` se torna `veinti-` (o `e` final é removido), depois se junta com o próximo número. Por exemplo:

- `Veinti` + `dos` (dois) = `veintidós` (22)

- `Veinti` + `cinco` (cinco) = `veinticinco` (25)

- `Veinti` + `ocho` (oito) = `veintiocho` (28)
