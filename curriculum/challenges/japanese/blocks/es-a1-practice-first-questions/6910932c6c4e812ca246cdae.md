# --explanation--

FlorenciaとFedericoのこの会話は基本的な自己紹介のフレーズを復習しています：

`Mi nombre es`と`Me llamo`は名前を言う別の方法です。

`¿De dónde eres?`は「どこから来ましたか？」を意味します。この質問には**`Soy de` + [国名]**で答えます。

`¿Cuántos años tienes?`は「何歳ですか？」を意味します。この質問には**`Tengo` + [数字] + `años`**で答えます。

`¿Y vos?`は「あなたは？」を意味し、質問を返すときに使います。これは`¿Y tú?`（「あなたは？」）の地域的な変種で、エルサルバドルなどの中央アメリカ諸国やアルゼンチンなどの南米の一部で一般的です。

`Encantada`（「はじめまして」）と`Igualmente`（「こちらこそ」）は、自己紹介の際に使う丁寧な表現です。
