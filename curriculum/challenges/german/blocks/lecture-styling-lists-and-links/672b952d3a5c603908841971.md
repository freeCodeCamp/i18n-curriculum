# --interactive--

Standard-Link-Stile spielen eine entscheidende Rolle bei der Verbesserung der Benutzerfreundlichkeit und Barrierefreiheit im Web.

Diese Stile, typischerweise blau für nicht besuchte Links und lila für besuchte Links, sind zu einem Standard geworden, den Nutzer beim Navigieren auf Websites erwarten und auf den sie sich verlassen.

Der Hauptzweck der Standard-Link-Stile besteht darin, klare visuelle Hinweise zu geben, die Nutzern helfen, zwischen interaktiven und nicht interaktiven Elementen auf einer Webseite zu unterscheiden.

Diese Unterscheidung ist grundlegend, um ein intuitives und benutzerfreundliches Surferlebnis zu schaffen.

Betrachten wir die grundlegenden Standardstile für Links:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
a:link {
  color: blue;
  text-decoration: underline;
}

a:visited {
  color: purple;
}
```

:::

Diese Stile erfüllen mehrere wichtige Funktionen.

Erstens hebt sich die blaue Farbe für nicht besuchte Links von den meisten Hintergrundfarben und Texten ab, wodurch Links leicht erkennbar sind. Dieser Kontrast ist entscheidend, damit Nutzer eine Seite schnell überfliegen und Navigations- oder wichtige Informationselemente finden können.

Die Unterstreichung betont zusätzlich, dass der Text anklickbar ist, und bietet einen weiteren visuellen Hinweis. Das ist besonders hilfreich für Nutzer, die farbenblind sind oder Schwierigkeiten haben, Farben zu unterscheiden.

Die Farbänderung bei besuchten Links (typischerweise zu Lila) hilft Nutzern, den Überblick zu behalten, welche Seiten sie bereits besucht haben. Diese Funktion ist unschätzbar beim Navigieren auf großen Websites oder bei der Recherche, da sie verhindert, dass Nutzer versehentlich dieselben Seiten erneut besuchen.

Betrachten Sie dieses HTML-Beispiel:

:::interactive_editor

```html
<p>Learn more about <a href="https://www.example.com/cats">cats</a> and <a href="https://www.example.com/dogs">dogs</a>.</p>
```

:::

Ohne benutzerdefiniertes CSS werden die meisten Browser diese Links blau mit Unterstreichung darstellen. Nach dem Klicken auf einen der Links ändert sich dessen Farbe zu Lila, was dem Nutzer sofortiges Feedback über seinen Surfverlauf gibt.

Obwohl es üblich ist, dass Designer diese Standardstile an das Design einer Website anpassen, ist es entscheidend, die dahinterstehenden Grundsätze beizubehalten.

Wenn Sie die Standardstile ändern, stellen Sie sicher, dass Links weiterhin klar von normalem Text unterscheidbar sind, es einen sichtbaren Unterschied zwischen besuchten und nicht besuchten Links gibt und die gewählten Farben ausreichend Kontrast zum Hintergrund bieten.

Zum Beispiel könnten Sie einen benutzerdefinierten Stil wie diesen verwenden:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
a:link {
  color: blue;
  text-decoration: none;
  border-bottom: 1px solid blue;
}

a:visited {
  color: purple;
  border-bottom: 1px solid purple;
}
```

:::

Dies bewahrt das Blau- und Lila-Farbschema, ersetzt jedoch die Unterstreichung durch einen unteren Rahmen für ein moderneres Aussehen.

Es ist auch wichtig, die verschiedenen Zustände von Links zu berücksichtigen. Zusätzlich zu den Standard- und Besuchtszuständen haben Links typischerweise Hover- und Aktiv-Zustände:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
a:hover {
  color: red;
}

a:active {
  color: darkorange;
}
```

:::

Diese Zustände geben Nutzern sofortiges Feedback, während sie mit Links interagieren, und verbessern so die Gesamtbenutzerfreundlichkeit der Seite.

Zusammenfassend lässt sich sagen, dass es zwar möglich ist, Link-Stile anzupassen, die Grundsätze hinter den Standardstilen jedoch erhalten bleiben sollten.

Sie spielen eine entscheidende Rolle dabei, ein nutzbares und barrierefreies Web-Erlebnis zu schaffen, das Nutzern hilft, Inhalte effektiv zu navigieren und zu nutzen.

Priorisieren Sie stets Klarheit und Benutzererfahrung, wenn Sie Link-Stile für Ihre Websites gestalten.
