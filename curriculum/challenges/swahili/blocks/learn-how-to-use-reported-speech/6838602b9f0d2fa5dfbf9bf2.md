# --explanation--

`Unusually` ni kitenzi cha kivumishi kinachotumika kuelezea jambo linalotokea zaidi au kidogo kuliko kawaida au ilivyotarajiwa. Kwa upande mwingine, `unusual` ni kivumishi kinachoelezea jambo ambalo si la kawaida au la kawaida. Kwa mfano:

- `The server experienced an unusually high load today.` – Katika sentensi hii, `unusually` inahusiana na jinsi mzigo ulivyokuwa mkubwa (zaidi sana kuliko kawaida).  

- `It was unusual for the server to slow down like that.` – Katika sentensi hii, `unusual` inahusu hali hiyo. Haiathiri kitenzi wala kivumishi.

`query` ni swali au ombi lililotumwa kwa hifadhidata kupata taarifa au kufanya kitendo. Kwa mfano:

`The application sent a query to the database to fetch user details.` – Hii ina maana programu ilimuomba hifadhidata taarifa maalum.
