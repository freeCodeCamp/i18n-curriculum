# --description--

배열에 대한 기본적인 이해를 했으니, 이제 반복문에 대해 이야기해 보겠습니다. 반복문은 코드 블록을 여러 번 실행할 때 사용합니다. 그중 하나가 `while` 반복문입니다. `while` 반복문은 조건이 참인 동안 코드 블록을 실행합니다. `while` 반복문의 구문은 다음과 같습니다:

```javascript
while (condition) {
  // code block to be executed
}
```

예를 들어, 다음 코드 스니펫은 1부터 5까지 숫자를 출력합니다:

```javascript
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
```

위 코드 스니펫은 `i` 변수를 `1` 값으로 초기화합니다. `while` 반복문은 `i` 값이 `5` 이하인 동안 코드 블록을 실행합니다. `i` 값은 각 반복마다 `1` 만큼 증가합니다.
