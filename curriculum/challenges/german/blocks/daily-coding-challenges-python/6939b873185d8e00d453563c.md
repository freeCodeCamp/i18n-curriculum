# --description--

Gegeben ist der String eines Links in Markdown. Geben Sie den entsprechenden HTML-String zurück.

Ein Markdown-Bild hat folgendes Format: `"[link_text](link_url)"`. Geben Sie den String des HTML-`a`-Tags zurück, wobei `href` auf `link_url` gesetzt ist und `link_text` als Inhalt des Tags dient.

Zum Beispiel, gegeben `"[freeCodeCamp](https://freecodecamp.org/)"`, geben Sie `'<a href="https://freecodecamp.org/">freeCodeCamp</a>'` zurück;

Hinweis: Die Konsole zeigt HTML-Tags in Strings beim Protokollieren von Nachrichten möglicherweise nicht an – prüfen Sie die Browser-Konsole, um Protokolle mit enthaltenen Tags zu sehen.

# --solutions--

```py
import re
def parse_link(markdown):
    match = re.match(r'\[(.*?)\]\((.*?)\)', markdown)
    text, url = match.groups()
    return f'<a href="{url}">{text}</a>'
```
