# --explanation--

有時候，當你不完全確定某件事，但你相信它應該是真的，你可以使用片語 `It's supposed to` + 動詞。 

這將表示某事預期會是某種方式，即使你尚未確認它。 例如：

- `The update is supposed to fix the bug.`－這表示你預期更新會修正該錯誤，但你尚未確認。
