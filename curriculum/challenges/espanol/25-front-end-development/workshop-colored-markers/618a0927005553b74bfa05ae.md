# --description--

Una rueda de color es un círculo donde colores similares, o <dfn>tonos</dfn>, están cerca entre sí, y los diferentes están más separados. Por ejemplo, el rojo puro está entre los tonos rosa y naranja.

Dos colores que están opuestos entre sí en la rueda de color se llaman <dfn>colores complementarios</dfn>. Si se combinan dos colores complementarios, producen gris. Pero cuando se colocan uno al lado del otro, estos colores producen un fuerte contraste visual y parecen más brillantes.

En la función `rgb` para la regla CSS `.one`, establece el valor rojo al máximo de `255` para producir rojo puro. En la función `rgb` para la regla CSS `.two`, establece los valores para verde y azul al máximo de `255` para producir cian.
