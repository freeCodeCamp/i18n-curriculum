# --description--

Ви можете використовувати `typeof` для перевірки структури даних, або типу, змінної. Це корисно під час налагодження, коли працюєте з кількома типами даних. Якщо ви думаєте, що додаєте два числа, але одне насправді є рядком, результати можуть бути несподіваними. Помилки типів можуть ховатися в обчисленнях або викликах функцій. Будьте особливо уважні, коли отримуєте доступ і працюєте з зовнішніми даними у формі об’єкта JavaScript Object Notation (JSON).

Ось кілька прикладів із використанням `typeof`:

```js
console.log(typeof "");
console.log(typeof 0);
console.log(typeof []);
console.log(typeof {});
```

У порядку, консоль виведе рядки `string`, `number`, `object` та `object`.

JavaScript розпізнає сім примітивних (незмінних) типів даних: `Boolean`, `Null`, `Undefined`, `Number`, `String`, `Symbol` (новий у ES6) та `BigInt` (новий у ES2020), а також один тип для змінних елементів: `Object`. Зверніть увагу, що в JavaScript масиви технічно є типом об’єкта.
