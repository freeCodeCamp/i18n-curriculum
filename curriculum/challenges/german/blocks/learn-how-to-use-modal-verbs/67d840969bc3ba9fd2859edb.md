# --explanation--

Ein `incident response plan` (`IRP`) hilft Organisationen, schnell und effektiv auf Cybersecurity-Bedrohungen zu reagieren. Zum Beispiel:

- `Every company should have an incident response plan for cyberattacks.` – Unternehmen benötigen einen Plan, um Sicherheitsprobleme zu bewältigen.

- `Without an incident response plan, it's hard to recover from a data breach.` – Ohne Plan kann die Behebung von Sicherheitsproblemen schwierig sein.
