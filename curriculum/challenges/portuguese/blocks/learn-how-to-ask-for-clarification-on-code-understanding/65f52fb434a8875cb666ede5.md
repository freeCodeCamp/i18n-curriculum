# --description--

`Seem to be` é usado quando você não tem certeza sobre algo e quer adivinhar ou sugerir uma possibilidade. Por exemplo, se você disser `It seems to be working`, você está supondo que algo está funcionando mas não tem 100% de certeza.

Exemplos:

Afirmativo: `It seems to be working fine.`

Negativo: `It doesn't seem to be a major issue.`

Pergunta: `Does it seem to be related to the network?`
