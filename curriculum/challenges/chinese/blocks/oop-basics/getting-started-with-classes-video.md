## --text--

以下哪种方式是为 `class` 添加 `method` 的正确方法？

## --answers--

```py
class Dog:
    set bark(self):
        print("Woof!")
```

---

```py
class Dog:
    def bark(self):
        print("Woof!")
```

---

```py
class Dog:
    method bark(self):
        print("Woof!")
```

---

```py
class Dog:
    def bark:
        print("Woof!")
```
