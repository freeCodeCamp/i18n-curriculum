# --explanation--

`Build the software`は、プログラムやアプリケーションをゼロから作成し開発することを意味します。コーディング、設計、テストを含みます。例えば：

`The company wants to build the software from the ground up using Python.` – これはPythonを使ってソフトウェアを最初から完全に開発することを意味します。

`In-house`は、外部チームを雇ったりプロジェクトを外注するのではなく、社内で作業が行われることを意味します。例えば：

`Our team decided to do front-end development in-house instead of hiring an external agency.` – これは、フロントエンドの開発を外注せずに会社の従業員が担当することを意味します。
