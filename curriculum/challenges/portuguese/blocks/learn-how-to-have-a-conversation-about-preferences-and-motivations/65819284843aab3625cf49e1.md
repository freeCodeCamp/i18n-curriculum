# --description--

`Working overtime` significa que você está trabalhando mais horas do que o tempo usual que deveria trabalhar. É como se você normalmente trabalhasse até as 17h, mas ficasse até as 19h para terminar algo; isso é trabalhar horas extras.

Quando algo está `draining`, faz você se sentir muito cansado, tanto física quanto mentalmente, como se estivesse tirando toda a sua energia. Imagine se você tivesse que carregar uma mochila pesada o dia todo; no final do dia, você se sentiria exausto.
