# --description--

Nella lezione precedente, hai conosciuto il tipo di dato lista e imparato a consultare gli elementi di una lista oltre al slicing delle liste. In questa lezione continuerai a imparare sulle liste e su alcuni metodi comuni associati a esse come `append()`, `pop()` e `sort()`.

Il primo metodo che vedremo è il metodo `append()`. Questo serve per aggiungere un elemento alla fine della lista. Ecco un esempio di utilizzo del metodo `append()` per aggiungere il numero `6` a una lista di numeri:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

Se vuoi aggiungere una lista alla fine di un'altra, puoi anche usare il metodo `append()` così:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

Nota come l'intera lista `even_numbers` sia annidata dentro la lista `numbers`.

Ma se vuoi aggiungere tutti i singoli numeri dalla lista `even_numbers` alla fine della lista `numbers`, allora puoi usare il metodo `extend()`.

Il metodo `extend()` è simile al metodo `append()`, ma con `extend()` puoi aggiungere più elementi da una lista a un'altra. Ecco un esempio di aggiunta dei numeri `6`, `8` e `10` da una lista alla fine della lista `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

Come vedi, la lista annidata non c’è più ed è solo una lista di numeri.

Per inserire un elemento in un indice specifico di una lista, puoi usare il metodo `insert()`. Questo metodo accetta due argomenti: l’indice dove vuoi inserire il nuovo elemento e l’elemento che vuoi inserire.

Ecco un esempio di utilizzo del metodo `insert()`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

Il codice seguente inserirà il numero `2.5` all’indice `2` nella lista `numbers`.

Se vuoi rimuovere un elemento da una lista, puoi usare il metodo `remove()`. Il metodo `remove()` prende come argomento il valore dell’elemento da rimuovere:

```py
numbers = [10, 20, 30, 40, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50]
```

È importante notare che questo metodo rimuoverà solo la prima occorrenza di un elemento. Non tutte:

```py
numbers = [10, 20, 30, 40, 50, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50, 50]
```

Per rimuovere un elemento a un indice specifico nella lista, puoi usare il metodo `pop()` così:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

Se non specifichi un elemento per il metodo `pop`, allora viene rimosso l’ultimo elemento.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

Se hai bisogno di svuotare la lista, puoi usare il metodo `clear()` così:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

Il prossimo metodo che vedremo è il metodo `sort()`. Questo metodo serve per ordinare gli elementi in loco. Ecco un esempio di ordinamento in loco di una lista casuale di numeri:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

In contrasto con il metodo `sort()`, c’è la funzione `sorted()` che funziona per qualsiasi iterabile e restituisce una nuova lista ordinata invece di modificare la lista originale. Per esempio:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(numbers) # [19, 2, 35, 1, 67, 41]
print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
```

Come promemoria, un iterabile è un tipo speciale di oggetto su cui puoi iterare, permettendoti di consultare ogni elemento uno alla volta. Imparerai di più su come funzionano i loop in Python in una lezione futura.

Sia il metodo `sort()` che la funzione `sorted()` accettano parametri opzionali `key` e `reverse`. Imparerai di più su questi parametri opzionali in una lezione futura quando studierai le tuple. Imparerai anche di più su altre funzioni integrate come `sorted()` nelle lezioni successive.

Il prossimo metodo che vedremo è il metodo `reverse()`. Questo metodo invertirà una lista di elementi in loco così:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

L’ultimo metodo che vedremo è il metodo `index`. Questo serve per trovare il primo indice in cui un elemento può essere trovato in una lista. Ecco un esempio di utilizzo del metodo `index` per trovare la lingua `'Java'` in una lista `programming_languages`:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

Se l’elemento non può essere trovato, Python lancia un `ValueError`:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: 'JavaScript' is not in list
"""
```

Ci sono ancora altri metodi per le liste Python, ma questa lista iniziale di metodi è un buon punto di partenza.
