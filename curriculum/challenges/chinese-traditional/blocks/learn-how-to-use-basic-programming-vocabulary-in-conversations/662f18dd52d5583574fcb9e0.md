# --description--

在詢問和回答有關數量的問題時，重要的是要一致地使用 `much` 和 `many`。`Much` 用於不可數名詞，而 `many` 用於可數名詞。在對話中，問題和答案應該在使用 `much` 或 `many` 上保持一致。

範例：

問題：`How much time do you have?` 答案：`Not much.`

問題：`How many books do you own?` 答案：`Not many.`
