# --description--

すべての整数 $n > 1$ に対して、関数族 $f_{n, a, b}$ は次のように定義されます：

$f_{n, a, b}(x) ≡ ax + b\bmod n$ ただし $a, b, x$ は整数で、$0 \lt a \lt n$、$0 \le b \lt n$、$0 \le x \lt n$。

すべての $0 \le x \lt n$ に対して $f_{n, a, b}(f_{n, a, b}(x)) \equiv f_{n, a, b}(x)\bmod n$ を満たす場合、$f_{n, a, b}$ をリトラクションと呼びます。

$R(n)$ を $n$ に対するリトラクションの数とします。

$F(N) = \displaystyle\sum_{n = 2}^N R(n)$。

$F({10}^7) ≡ 638\\,042\\,271\bmod 1\\,000\\,000\\,007$。

$F({10}^{14})$ を求めてください。答えは $1\\,000\\,000\\,007$ で割った余りで提出してください。
