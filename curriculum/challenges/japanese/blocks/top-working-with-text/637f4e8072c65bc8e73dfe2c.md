# --description--

このレッスンのすべての例で、他の要素の中にある要素をインデントしていることに気づいたかもしれません。これは要素をネストするという意味です。

要素を他の要素の中にネストすると、それらの間に親子関係ができます。ネストされた要素は子要素で、ネストされている側の要素が親要素です。

次の例では、body要素が親で、段落が子です：

```html
<html>
  <head>
  </head>
  <body>
    <p>Lorem ipsum dolor sit amet.</p>
  </body>
 </html>
```

人間関係と同じように、HTMLの親要素は複数の子要素を持つことができます。ネストの同じレベルにある要素は兄弟要素とみなされます。

たとえば、次のコードの2つの段落は兄弟要素です。なぜなら、どちらもbodyタグの子であり、同じネストレベルにあるからです：

```html
<html>
  <head>
  </head>
  <body>
    <p>Lorem ipsum dolor sit amet.</p>
    <p>Ut enim ad minim veniam.</p>
  </body>
 </html>
```

インデントは、ネストのレベルを自分自身や将来あなたのHTMLを扱う他の開発者にとってわかりやすく読みやすくするために使います。子要素は2スペース分インデントすることをお勧めします。

要素間の親子関係や兄弟関係は、後でCSSでHTMLのスタイルを設定したり、JavaScriptで動作を追加したりするときに非常に重要になります。しかし今は、要素の関係の違いとそれを説明する用語を知っておくことが大切です。
