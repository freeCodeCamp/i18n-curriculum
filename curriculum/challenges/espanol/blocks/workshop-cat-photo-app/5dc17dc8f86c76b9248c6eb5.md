# --description--

Los comentarios te permiten dejar mensajes sin afectar lo que se muestra en el navegador. Te permite crear código que no se ejecutará.  A comment in HTML starts with `<!--`, contains any number of lines of text, and ends with `-->`. 

Aquí hay un ejemplo de un comentario con el `TODO: Remove h1`:

```html
<!-- TODO: Remove h1 -->
```

Añade un comentario sobre el elemento `p` con este texto:

`TODO: Add link to cat photos`

# --hints--

Tu comentario debe iniciar con `<!--`. Faltan uno o más de los caracteres que definen el inicio de un comentario.

```js
assert.match(code, /<!--/);
```

Tu comentario debe terminar con `-->`. Faltan uno o más de los caracteres que definen el final de un comentario.

```js
assert.match(code, /-->/);
```

Tu código no debe tener caracteres de apertura/cierre extra. Tienes los caracteres `<!--` o `-->` extra mostrándose en el navegador.

```js
const noSpaces = code.replace(/\s/g, '');
assert.isBelow(noSpaces.match(/<!--/g)?.length, 2)
assert.isBelow(noSpaces.match(/-->/g)?.length, 2);
```

Tu comentario debe estar encima del elemento `p`. Los tienes en el orden incorrecto.

```js
assert.match(
  code.replace(/\s/g, ''),
  /<!--(.*?)--><p>everyonelovescutecatsonline!<\/p>/i
);
```

Tu comentario debe contener el texto `TODO: Add link to cat photos`.

```js
assert.match(code, /<!--\s*todo:\s+add\s+link\s+to\s+cat\s+photos\s*-->/i);
```
