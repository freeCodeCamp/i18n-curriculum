# --description--

$f_5(n)$ を $5^x$ が $n$ を割り切る最大の整数 $x$ とします。

例えば、$f_5(625\\,000) = 7$ です。

$T_5(n)$ を、$f_5((2 \times i - 1)!) < 2 \times f_5(i!)$ を満たす整数 $i$ の個数で、$1 ≤ i ≤ n$ の範囲とします。

$T_5({10}^3) = 68$ および $T_5({10}^9) = 2\\,408\\,210$ であることが確認できます。

$T_5({10}^{18})$ を求めてください。
