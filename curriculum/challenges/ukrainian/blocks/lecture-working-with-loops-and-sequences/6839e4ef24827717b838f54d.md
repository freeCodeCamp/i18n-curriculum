# --description--

Протягом попередніх уроків ви звикли визначати функції за допомогою ключового слова `def` ось так:

```py
def square(num):
    return num ** 2

print(square(4)) # 16
```

Але коли справа доходить до роботи з функціями вищого порядку, як `map()` і `filter()`, ви можете використати анонімну рядкову функцію. Саме тут і з’являються lambda-функції.

Ось як виглядає функція `square()`, переписана у вигляді lambda-функції:

```py
lambda num: num ** 2
```

Як уже згадувалося, lambda-функції анонімні, тому ця функція більше не має імені `square`. Lambda-функції чудово підходять, коли потрібно використати їх у функціях вищого порядку, наприклад так:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

У цьому прикладі у нас є список чисел, і ми хочемо створити новий список парних чисел. Тож ми передаємо lambda-функцію як один із аргументів функції `filter()`, щоб отримати новий список, що містить числа `2` і `4`.

Працюючи з lambda-функціями, важливо дотримуватися найкращих практик. Наприклад, не рекомендується призначати lambda-функцію змінній ось так:

```py
numbers = [1, 2, 3, 4, 5]

square = lambda x: x ** 2
squared_numbers = list(map(square, numbers))
print(squared_numbers) # [1, 4, 9, 16, 25]
```

Це суперечить ідеї використання анонімних функцій. У такому випадку краще використати звичайну функцію, ось так:

```py
numbers = [1, 2, 3, 4, 5]

def square(num):
    return num ** 2

squared_numbers = list(map(square, numbers))
print(squared_numbers) # [1, 4, 9, 16, 25]
```

Також слід уникати створення lambda-функцій, які складно читати або які є надмірно ускладненими, ось так:

```py
result = (lambda x: (x**2 + 2*x - 1) if x > 0 else (x**3 - x + 4))(3)
print(result)  # 14
```

Хоч ця функція і працює правильно, даючи коректний результат `14`, її важко читати і сприймати. У такому випадку краще створити окрему функцію з інструкцією `if/else`, а потім викликати цю функцію:

```py
def calculate_expression(x):
    if x > 0:
        return x**2 + 2*x - 1
    else:
        return x**3 - x + 4

print(calculate_expression(3))  # 14
```

І звичайні функції, і lambda-функції мають свої випадки використання у програмах на Python. Якщо ви маєте справу з одним рядковим виразом, варто розглянути використання lambda-функції. В іншому випадку краще застосувати звичайну функцію.
