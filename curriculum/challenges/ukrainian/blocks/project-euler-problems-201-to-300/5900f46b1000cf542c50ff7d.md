# --description--

Визначте $f(n)$ як суму факторіалів цифр числа $n$. Наприклад, $f(342) = 3! + 4! + 2! = 32$.

Визначте $sf(n)$ як суму цифр числа $f(n)$. Тож $sf(342) = 3 + 2 = 5$.

Визначте $g(i)$ як найменше додатне ціле число $n$, для якого $sf(n) = i$. Хоч $sf(342)$ дорівнює 5, $sf(25)$ також дорівнює 5, і можна перевірити, що $g(5)$ — це 25.

Визначте $sg(i)$ як суму цифр числа $g(i)$. Тож $sg(5) = 2 + 5 = 7$.

Крім того, можна перевірити, що $g(20)$ дорівнює 267, а $\sum sg(i)$ для $1 ≤ i ≤ 20$ дорівнює 156.

Яке значення $\sum sg(i)$ для $1 ≤ i ≤ 150$?
