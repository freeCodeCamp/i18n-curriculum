# --description--

让我们看几个快速示例来直观了解特异性是如何工作的。请考虑以下 `HTML` 和 `CSS` 代码：

```html
<!-- index.html -->

<div class="main">
  <div class="list subsection"></div>
</div>
```

```css
/* rule 1 */
.subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

在上面的示例中，两个规则都只使用了类选择器，但规则 2 更具体，因为它使用了更多的类选择器。
