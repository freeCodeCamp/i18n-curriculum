# --explanation--

ليزا تستخدم `Second Conditional` لمناقشة حالة افتراضية.

`Second Conditional` يُستخدم للحديث عن حالات غير حقيقية أو غير محتملة في الحاضر أو المستقبل. يتبع التركيب `if` + `Present Simple` أو `Past Continuous` (لشرط الحالة) و `might / would / could` + الفعل الأساسي (للناتج). على سبيل المثال:

- `If I had more time, I would learn a new programming language.` - هذا يصف حالة افتراضية حيث وجود المزيد من الوقت يؤدي إلى التعلم.

- `If the server were faster, it could handle more users.` - الخادم ليس سريعًا الآن، لكن لو كان كذلك، لكان بإمكانه دعم المزيد من المستخدمين.
