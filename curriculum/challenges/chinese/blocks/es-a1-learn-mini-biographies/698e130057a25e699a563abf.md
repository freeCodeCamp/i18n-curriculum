# --description--

用西班牙语表达某人喜欢什么，你可以使用以下结构：

**`Le gusta`（“他/她喜欢”）+ 这个人喜欢的事物。**

代词 `le` 指的是“他”或“她”。它明确表示你在谈论另一个人。

要说某人**不**喜欢某事，你需要在 `no` 前添加单词 `le`。

**`No le gusta`（“他/她不喜欢”）+ 这个人不喜欢的事物。**

Sebastián 正在谈论 Marisol 在日常工作中喜欢做的事情。
