# --explanation--

`Parameterized` 来自根词 `parameter`，它是用于控制某事如何工作的值或占位符。  

当人们说一个查询是 `parameterized`，意味着它使用预定义的值（参数），而不是直接插入用户输入。 

这有助于通过防止诸如 SQL 注入之类的攻击来提高数据库安全性。例如：  

- `The developer used parameterized queries to secure the database.`——这意味着他们使用了占位符以确保不会插入有害代码。
