## 색상 이론

- **색상 이론 정의**: 색상이 서로 어떻게 상호작용하는지, 그리고 우리의 인식에 어떤 영향을 미치는지 연구하는 학문입니다. 색상 관계, 색상 조화, 색상의 심리적 영향을 다룹니다.
- **기본 색상**: 노란색, 파란색, 빨간색으로, 모든 다른 색상이 파생되는 기본 색상입니다.
- **보조 색상**: 두 개의 기본 색상을 같은 비율로 섞어 만든 색상입니다. 초록, 주황, 보라색이 보조 색상의 예입니다.
- **삼차 색상**: 기본 색상과 인접한 보조 색상을 결합해 만든 색상입니다. 노랑-초록, 파랑-초록, 파랑-보라가 삼차 색상의 예입니다.
- **따뜻한 색상**: 빨강, 주황, 노랑을 포함하며, 편안함, 따뜻함, 아늑함을 느끼게 합니다.
- **차가운 색상**: 파랑, 초록, 보라를 포함하며, 차분함, 평온함, 전문성을 느끼게 합니다.
- **색상환**: 색상들이 서로 어떻게 관련되는지 보여주는 원형 도표입니다. 디자이너에게 필수 도구로, 색상 조합을 선택하는 데 도움을 줍니다.
- **유사 색상 조합**: 색상환에서 서로 인접한 색상들로 구성되어 일관되고 편안한 느낌을 줍니다.
- **보색 조합**: 색상환에서 서로 반대편에 위치한 색상들로 구성되어 강한 대비와 시각적 효과를 만듭니다.
- **삼원색 조합**: 대략 같은 간격으로 떨어진 세 가지 색상으로 구성되어 생동감 있는 색상 조합을 만듭니다. 연결하면 색상환에서 정삼각형을 형성합니다.
- **단색 조합**: 같은 기본 색상에서 밝기, 어둠, 채도를 조절해 만든 색상들로 구성되어 통일감과 조화를 느끼면서도 대비를 만듭니다.

## CSS에서 색상을 다루는 다양한 방법

- **이름 있는 색상**: 브라우저가 인식하는 미리 정의된 색상 이름입니다. 예로 `blue`, `darkred`, `lightgreen`가 있습니다.
- **`rgb()` 함수**: RGB는 빛의 기본 색상인 빨강, 초록, 파랑을 의미합니다. 이 세 가지 색상을 다양한 강도로 조합해 다양한 색상을 만듭니다. `rgb()` 함수는 RGB 색상 모델을 사용해 색상을 정의할 수 있게 합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>RGB Color</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

- **`rgba()` 함수**: 이 함수는 네 번째 값인 알파를 추가해 색상의 투명도를 제어합니다. 제공하지 않으면 알파 값은 `1`로 기본 설정됩니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>RGBA Background</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

- **`hsl()` 함수**: HSL은 색상, 채도, 밝기의 세 가지 주요 요소를 나타냅니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>HSL Color</p>
```

```css
p {
  color: hsl(120, 100%, 50%);
}
```

:::

- **`hsla()` 함수**: 이 함수는 네 번째 값인 알파를 추가해 색상의 불투명도를 제어합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>HSLA Background</div>
```


```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

- **16진수**: 16진수 코드는 RGB 색상 모델에서 색상을 나타내는 6자리 문자열입니다. "16진수"는 0부터 9까지 숫자와 A부터 F까지 문자를 사용하는 16진법을 의미합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1 class="hex-text">Hex Text</h1>
<p class="hex-bg">Hex Background</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}

p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

## `box-shadow` 속성

- **정의**: `box-shadow` 속성은 요소 주위에 하나 이상의 그림자를 적용합니다.
- **오프셋 값**: 수평(`offset-x`)과 수직(`offset-y`) 값을 지정해야 합니다. 양수 `offset-x`는 그림자를 오른쪽으로 이동시키고, 음수는 왼쪽으로 이동시킵니다. 양수 `offset-y`는 그림자를 아래로 이동시키고, 음수는 위로 이동시킵니다. 값이 `0`이면 단위를 포함하지 않아도 됩니다.
- **블러 반경**: 선택 사항으로, 그림자의 흐림 정도를 제어합니다. 포함하지 않으면 기본값은 `0`로 선명한 가장자리를 만듭니다. 값이 클수록 그림자가 부드러워집니다.
- **확산 반경**: 선택 사항으로, 그림자가 얼마나 확장되거나 축소되는지 제어합니다. 포함하지 않으면 기본값은 `0`입니다.
- **그림자 색상**: 이름 있는 색상, 16진수 값, `rgb()`, `rgba()`, `hsl()`, 또는 `hsla()` 함수를 사용해 지정할 수 있습니다.
- **`inset` 키워드**: `inset` 키워드를 추가하면 그림자가 요소 내부에 위치합니다.
- **여러 개의 박스 그림자 적용**: 쉼표로 구분해 여러 그림자를 적용할 수 있습니다. 그림자는 앞에서 뒤로 겹쳐집니다.

### 구문

```css
box-shadow: offset-x offset-y blur-radius spread-radius color;
```

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="shadow-box">Shadow Color Example</div>
```

```css
.shadow-box {
  width: 200px;
  padding: 20px;
  background-color: lightblue;
  box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}
```

:::

## 선형 및 방사형 그라디언트

- **선형 그라디언트**: 직선을 따라 색상이 점진적으로 섞이는 그라디언트입니다. `to top`, `to right`, `to bottom right` 같은 키워드나 `45deg` 같은 각도를 사용해 방향을 제어할 수 있습니다. 유효한 CSS 색상과 원하는 만큼 많은 색상 정지를 사용할 수 있습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient">Linear Gradient</div>
```

```css
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

:::

- **방사형 그라디언트**: 중심점에서 원형 또는 타원형으로 퍼지는 그라디언트입니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient">Radial Gradient</div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::
