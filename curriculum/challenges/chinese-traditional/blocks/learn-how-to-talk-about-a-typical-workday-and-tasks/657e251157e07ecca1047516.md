# --description--

`log in` 是指輸入你的使用者名稱和密碼以存取電腦或網站。例如：

`Every morning, I log in to my computer to start work。`－這表示每天早上你輸入你的使用者名稱和密碼來開啟你的電腦並開始工作。

`check emails` 是指查看並閱讀你電子郵件帳戶中的新訊息。例如：

`After logging in, I check my emails to see if I have any new messages。`－這表示在你開啟你的電腦或電子郵件帳戶後，你會查看是否有人寄給你新的訊息。
