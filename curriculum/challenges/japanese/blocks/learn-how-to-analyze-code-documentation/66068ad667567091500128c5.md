# --description--

`library` は、プログラマーが使えるたくさんのコードが詰まった大きな本棚のようなものです。さまざまな料理（プロジェクト）を作るためのレシピ（関数）がたくさん載っている料理本のように考えてください。

たとえば、プログラミングの `library` にはチャートを作る関数が含まれていることがあります。コードを一から書く代わりに、この関数を使うことで、ケーキを作るレシピに従うようにプログラムできます。
