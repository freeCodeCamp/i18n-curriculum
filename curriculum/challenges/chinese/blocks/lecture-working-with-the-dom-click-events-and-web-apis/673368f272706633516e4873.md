# --interactive--

你可能熟悉在你的 HTML 和 CSS 项目中使用属性。但你知道你可以在你的 JavaScript 文件中为你的 HTML 元素添加属性吗？

在本课中，你将学习如何使用 `setAttribute()` 方法为你的超文本标记语言元素添加属性。

这是基本语法：

```js
setAttribute(attribute, value);
```

让我们看几个例子，更好地理解如何使用 `setAttribute()` 方法。

在这个第 1 个示例中，我们有一个位于超文本标记语言内的 `p` 元素：

```html
<p id="para">I am a paragraph</p>
```

要为 `class` 属性添加，我们首先需要使用 `getElementById()` 方法访问该 `p` 元素。然后我们可以使用 `setAttribute()` 方法添加 `class` 属性并将值设置为 `my-class`。

我们使用 `outerHTML` 来记录整个超文本标记语言元素及其属性。

:::interactive_editor

```html
<p id="para">I am a paragraph</p>
<script src="index.js"></script>
```

```js
const para = document.getElementById("para");
para.setAttribute("class", "my-class");

console.log(`${para.outerHTML}`);
```

:::

如果你的超文本标记语言元素上已有属性，可以使用 `setAttribute()` 方法更新其值。

在此示例中，我们有一个 `div` 元素，其 `class` 属性设置为 `my-class`：

```html
<div class="my-class"></div>
```

要将 `class` 属性值更新为 example，我们可以再次使用 `setAttribute()` 方法：

:::interactive_editor

```html
<div class="my-class"></div>
<script src="index.js"></script>
```

```js
const divEl = document.querySelector(".my-class");
divEl.setAttribute("class", "example");

console.log(`${divEl.outerHTML}`);
```

:::

那么，什么时候使用 `setAttribute()` 方法的一些实际例子是什么？

如果你正在创建一个动态图像库，当用户点击缩略图时，你可能需要更新图像元素的 `src` 属性。

另一个例子是，如果你正在处理表单验证，并且需要为 `input` 元素添加诸如 `required` 或 `minlength` 之类的属性。
