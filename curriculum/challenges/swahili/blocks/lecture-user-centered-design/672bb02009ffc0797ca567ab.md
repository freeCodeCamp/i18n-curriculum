# --interactive--

**KUMBUKA**: Baadhi ya mifano ya mwingiliano inaweza kutumia CSS na JavaScript ambazo bado hujajifunza. Usijali kuhusu kuelewa msimbo wote. Lengo la mifano ni kukuonyesha mapitio ya dhana hizi za muundo ili uelewe vizuri jinsi mambo yanavyofanya kazi. Ili kuona mapitio, utahitaji kuwezesha mhariri wa mwingiliano.

Kuonyesha maendeleo ni njia ya kuonyesha watumizi walipo katika mchakato. Inaweza kutumika kwenye fomu, usajili, na michakato ya usanidi. Lengo ni kuwasaidia watumizi kuelewa walipo katika mchakato na ni kiasi gani bado wanahitaji kufanya.

Kwa mfano, unaweza kutumia kipimo cha kuonyesha maendeleo kuonyesha watumizi kilichobaki cha kufanya wanapojaza fomu. Hutaki kuunda hali ambapo mtumizi anahitaji kujaza fomu ndefu na hawajui ni hatua ngapi zaidi wanahitaji kukamilisha. Uwazi ni muhimu ili mtumizi ajue kama ana muda wa kukaa na kumaliza fomu au kama anahitaji kurudi baadaye.

```js
const form = document.getElementById('multiStepForm');
const steps = form.querySelectorAll('.form-step');
const progressBar = form.querySelector('.progress-bar');
const progressText = form.querySelector('.progress-text');
const totalSteps = steps.length;

let currentStep = 0;

function updateProgress() {
  const percent = ((currentStep + 1) / totalSteps) * 100;
  progressBar.style.width = percent + '%';
  progressText.textContent = `Step ${currentStep + 1} of ${totalSteps}`;
}

function showStep(index) {
  steps.forEach((step, i) => {
    step.classList.toggle('active', i === index);
  });
  updateProgress();
}

form.querySelectorAll('.next-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    if (currentStep < totalSteps - 1) {
      currentStep++;
      showStep(currentStep);
    }
  });
});

form.querySelectorAll('.prev-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    if (currentStep > 0) {
      currentStep--;
      showStep(currentStep);
    }
  });
});

showStep(currentStep);

form.addEventListener('submit', e => {
  e.preventDefault();
  alert('Form submitted!');
});
```

Unapobuni sehemu ya kuonyesha maendeleo, kuna mbinu bora chache za kuzingatia. Kitu cha kwanza ni kuifanya iwe rahisi. Hutaki kumchosha mtumizi kwa taarifa nyingi sana hadi wachoke na kuondoka kwenye tovuti.

Kitu cha pili ni kuwezesha kurudi kwenye hatua zilizopita. Hii ni muhimu kwa sababu watumizi wanaweza kutaka kurudi na kukagua majibu yao ya awali au kufanya mabadiliko.

Kitu kingine ni kuhakikisha sehemu ya kuonyesha maendeleo ni rahisi kupatikana. Ikiwa mtumizi hawezi kuipata, hawawezi kujua walipo katika mchakato.

Kitu cha mwisho ni kuwa na vichwa vya sehemu, asilimia, au hatua zilizo wazi. Ikiwa una kipimo cha maendeleo tu bila muktadha, mtumizi hatajua maana yake.

```js
const form = document.getElementById('progressForm');
const steps = form.querySelectorAll('.form-step');
const progressBar = form.querySelector('.progress-bar');
const currentStepSpan = document.getElementById('currentStep');
const totalStepsSpan = document.getElementById('totalSteps');
const percentageSpan = document.getElementById('percentage');

const totalSteps = steps.length;
let currentStep = 0;

totalStepsSpan.textContent = totalSteps;

function updateProgress() {
  const percent = Math.round(((currentStep + 1) / totalSteps) * 100);
  progressBar.style.width = percent + '%';
  currentStepSpan.textContent = currentStep + 1;
  percentageSpan.textContent = percent + '%';

  // Update ARIA attribute
  form.querySelector('.progress-container').setAttribute('aria-valuenow', percent);
}

function showStep(index) {
  steps.forEach((step, i) => {
    step.classList.toggle('active', i === index);
  });
  updateProgress();
}

form.querySelectorAll('.next-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    if (currentStep < totalSteps - 1) {
      currentStep++;
      showStep(currentStep);
    }
  });
});

form.querySelectorAll('.prev-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    if (currentStep > 0) {
      currentStep--;
      showStep(currentStep);
    }
  });
});

form.addEventListener('submit', e => {
  e.preventDefault();
  alert('Form submitted! Thanks!');
});

showStep(currentStep);
```
