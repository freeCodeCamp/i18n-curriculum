# --description--

グラフはさまざまな方法で表現できます。ここでは、<dfn>隣接リスト</dfn>と呼ばれる方法を説明します。隣接リストは基本的に箇条書きで、左側がノード、右側にそのノードとつながっている他のすべてのノードがリストされています。以下は隣接リストの表現例です。

<blockquote>Node1: Node2, Node3<br>Node2: Node1<br>Node3: Node1</blockquote>

上記は無向グラフです。なぜなら`Node1`は`Node2`と`Node3`に接続されており、その情報は`Node2`と`Node3`の接続と一致しているからです。有向グラフの隣接リストでは、リストの各行が方向を示します。もし上記が有向グラフなら、`Node2: Node1`は有向辺が`Node2`から`Node1`に向かっていることを意味します。上記の無向グラフはJavaScriptオブジェクト内に入れて隣接リストとして表現できます。

```js
var undirectedG = {
  Node1: ["Node2", "Node3"],
  Node2: ["Node1"],
  Node3: ["Node1"]
};
```

これはノードが文字列ラベルではなく数字だけの場合、配列としてより簡単に表現することもできます。

```js
var undirectedGArr = [
  [1, 2], // Node1
  [0],    // Node2
  [0]     // Node3
];
```
