# --description--

`library` 就像一個有許多程式碼的巨大書架，程式設計師可以使用。把它想像成一本有許多食譜（函式）的食譜書，你可以用來烹飪（撰寫程式）不同的菜餚（專案）。

例如，程式設計中的 `library` 可能有一個用來製作圖表的函式。你可以使用這個函式，而不是從頭撰寫程式碼，就像照著食譜做蛋糕一樣。
