# --explanation--

Wenn ein Satz mit einem Verb endet, das auf `-ing` endet, fungiert es oft als Subjekt des Satzes. Diese Form ist als Gerundium bekannt. Zum Beispiel:

- `Coding is an essential skill for developers.` – `Coding` ist das Subjekt des Satzes.

- `Testing new software helps prevent bugs.` – `Testing` ist das Subjekt.
