# --explanation--

`Enough` означає наявність достатньої кількості чогось. Наприклад:

`We don't have enough resources to complete the project.` - Це означає, що доступних ресурсів недостатньо.

`Actual` означає реальний або суттєвий, на відміну від просто обговорення чи планування. Наприклад:

`The meeting took longer than expected, so we had little time for the actual work.` - Це означає, що реальні завдання були відкладені через обговорення.

`To get something done` означає завершити завдання або виконати те, що потрібно зробити. Наприклад:

`We need to get this report done by Friday.` - Це означає, що звіт має бути завершений до дедлайну.
