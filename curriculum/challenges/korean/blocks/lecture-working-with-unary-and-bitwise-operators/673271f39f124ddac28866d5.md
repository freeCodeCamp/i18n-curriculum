# --interactive--

JavaScript의 비트 연산자는 숫자의 이진 표현에 작동하는 특별한 연산자입니다. 비트 연산자를 이해하려면 먼저 비트와 이진수 개념을 파악해야 합니다. 컴퓨팅에서 비트는 가장 기본적인 정보 단위로, 두 가지 값만 가질 수 있습니다: `0` 또는 `1`. 이진수는 이 두 숫자만 사용하여 모든 숫자를 나타내는 숫자 체계입니다.

예를 들어, 십진수 `10`의 이진 표현은 `1010`입니다. 이 체계에서 각 숫자는 오른쪽 끝 숫자부터 시작해 왼쪽으로 갈수록 `2`의 거듭제곱을 나타냅니다.

위 표에서 첫 번째 행은 이진수 `1010`를 나타내고, 두 번째 행은 각 이진 자리수가 나타내는 `2`의 거듭제곱을 보여주며, 세 번째 행은 각 곱셈 결과입니다. 세 번째 행의 모든 값을 더하면 총합이 `10`이 됩니다.

이제 비트 연산자를 살펴보겠습니다. 이 연산자들은 숫자의 이진 표현에 대해 연산을 수행합니다. JavaScript는 AND (`&`), OR (`|`), XOR (`^`), NOT (`~`), 왼쪽 시프트 (`<<`), 오른쪽 시프트 (`>>`) 등 여러 비트 연산자를 제공합니다.

비트 AND (`&`) 연산자는 두 피연산자의 해당 비트가 모두 `1`일 때 각 비트 위치에 `1`를 반환합니다. 예를 들어:

:::interactive_editor

```js
let a = 5;  // Binary: 101
let b = 3;  // Binary: 011
console.log(a & b);  // 1 (Binary: 001)
```

:::

이 예제에서는 `5` (이진수 `101`)와 `3` (이진수 `011`)에 대해 비트 AND 연산을 수행합니다. 결과는 `1` (이진수 `001`)인데, 두 숫자 모두에서 오른쪽 끝 비트만 `1`이기 때문입니다.

비트 OR (`|`) 연산자는 두 피연산자 중 하나 또는 둘 다 해당 비트가 `1`일 때 각 비트 위치에 `1`를 반환합니다. 예를 들어:

:::interactive_editor

```js
let a = 5;  // Binary: 101
let b = 3;  // Binary: 011
console.log(a | b);  // 7 (Binary: 111)
```

:::

여기서 결과는 `7` (이진수 `111`)인데, 각 위치에서 적어도 하나의 비트가 `1`이기 때문입니다.

비트 XOR (`^`) 연산자는 두 피연산자 중 하나만 해당 비트가 `1`일 때 각 비트 위치에 `1`를 반환합니다. 예를 들어:

:::interactive_editor

```js
let a = 5;  // Binary: 101
let b = 3;  // Binary: 011
console.log(a ^ b);  // 6 (Binary: 110)
```

:::

결과는 `6` (이진수 `110`)인데, 두 숫자에서 오른쪽에서 첫 번째와 두 번째 비트가 다르기 때문입니다.

비트 NOT (`~`) 연산자는 피연산자의 모든 비트를 반전시킵니다. 예를 들어:

:::interactive_editor

```js
let a = 5;  // Binary: 101
console.log(~a);  // -6
```

:::

이것은 놀라울 수 있지만, 음수를 이진수에서 2의 보수로 나타내는 방식 때문입니다.

왼쪽 시프트 (`<<`) 연산자는 모든 비트를 지정한 자리 수만큼 왼쪽으로 이동시킵니다. 예를 들어:

:::interactive_editor

```js
let a = 5;  // Binary: 101
console.log(a << 1);  // 10 (Binary: 1010)
```

:::

여기서는 모든 비트가 한 자리 왼쪽으로 이동하여 숫자를 `2`배 곱한 효과가 있습니다.

오른쪽 시프트 (`>>`) 연산자는 모든 비트를 오른쪽으로 이동시킵니다. 예를 들어:

:::interactive_editor

```js
let a = 5;  // Binary: 101
console.log(a >> 1);  // 2 (Binary: 10)
```

:::

여기서는 모든 비트가 한 자리 오른쪽으로 이동하여 숫자를 `2`로 나누고 내림한 효과가 있습니다.

비트 연산자는 저수준 프로그래밍과 암호학에서 자주 사용됩니다. 일상적인 JavaScript 프로그래밍에서는 자주 쓰이지 않을 수 있지만, 특정 전문 작업에 유용하며 컴퓨터가 근본적으로 어떻게 작동하는지 이해하는 데 도움이 됩니다.
