# --description--

Ti viene fornita la seguente informazione, ma potresti preferire fare qualche ricerca per conto tuo.

<ul>
  <li>Il 1° gennaio 1900 era un lunedì.</li>

  <li>Un anno bisestile si verifica in ogni anno divisibile per 4, ma non in un secolo a meno che non sia divisibile per 400.</li>

Quante domeniche sono cadute il primo giorno del mese durante il ventesimo secolo (dal 1° gennaio 1901 al 31 dicembre 2000)?

```js
function countingSundays(firstYear, lastYear) {
  let sundays = 0;

  for (let year = firstYear; year <= lastYear; year++) {
    for (let month = 0; month <= 11; month++) {
      const thisDate = new Date(year, month, 1);
      if (thisDate.getDay() === 0) {
        sundays++;
      }
    }
  }
  return sundays;
}
```
