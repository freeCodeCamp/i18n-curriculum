# --description--

El elemento `<head>` es donde pones información meta importante sobre tus páginas web y elementos necesarios para que tus páginas web se rendericen correctamente en el navegador. Dentro del `<head>`, no debes usar ningún elemento que muestre contenido en la página web.

## El elemento Meta Charset
Siempre debes tener la etiqueta `meta` para la codificación `charset` de la página web en el elemento `head`: `<meta charset="utf-8">`.

Configurar la codificación es muy importante porque garantiza que la página web mostrará correctamente símbolos especiales y caracteres de diferentes idiomas en el navegador.

## Elemento Title
Otro elemento que siempre debes incluir en el head de un documento HTML es el elemento `title`:

```html
<title>My First Webpage</title>
```

El elemento `title` se usa para darle a las páginas web un título legible para humanos que se muestra en la pestaña del navegador de tu página web.

Si no incluiste un elemento `title`, el título de la página web por defecto sería el nombre del archivo. En tu caso, eso sería `index.html`, lo cual no es muy significativo para los usuarios; esto haría que sea muy difícil encontrar tu página web si el usuario tiene muchas pestañas del navegador abiertas.

Hay muchos más elementos que pueden ir dentro del `head` de un documento HTML. Sin embargo, por ahora solo es crucial conocer los dos elementos que has visto aquí. Introducirás más elementos que van dentro del `head` a lo largo del resto del currículo.

De nuevo en el archivo `index.html`, vamos a agregar un elemento `head` con un elemento `meta` de `charset` y un `title` dentro de él. El elemento head va dentro del elemento HTML y siempre debe ser el primer elemento bajo la etiqueta de apertura `<html>`:


```html
<!DOCTYPE html>

<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Webpage</title>
  </head>
</html>
```

## --text--

¿Cuál es el propósito del elemento `head`?

## --answers--

El elemento `head` se usa para mostrar todos los elementos que se muestran en la página web.

---

El elemento `head` se usa para mostrar información importante sobre tu página web y se utiliza para renderizar correctamente las páginas web con elementos `meta`.

---

El elemento `head` se usa para mostrar el contenido del encabezado en la parte superior de la página web.
