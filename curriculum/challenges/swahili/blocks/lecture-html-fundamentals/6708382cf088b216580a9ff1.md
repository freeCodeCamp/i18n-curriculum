# --interactive--

Sifa ya kipengele `id` inaongeza kitambulisho cha kipekee kwa kipengele cha HTML.

Hapa kuna mfano wa kipengele cha `h1` chenye `id` ya `title`.

Chini ya kipengele cha `h1`, ongeza kipengele cha `h2` chenye `id` imewekwa kuwa `"subtitle"`. Unaweza kuandika maandishi yoyote unayopenda kwa ajili ya `h2` na utaona mabadiliko katika dirisha la onyesho. Ili kuingiliana na mfano huu, utahitaji kuwezesha mhariri wa kuingiliana.

:::interactive_editor

```html
<h1 id="title">Movie Review Page</h1>

```

:::

Unaweza kurejea jina la `id` la `title` ndani ya JavaScript au CSS yako. Hapa kuna mfano wa CSS unaorejea `id` ya `title` kubadilisha rangi ya maandishi ya `color` kuwa `red`.

**KUMBUKA**: Baadhi ya CSS imetolewa kwako katika mfano huu wa kuingiliana. Usijali kuhusu kuelewa msimbo wa CSS kwa sasa kwa sababu utajifunza zaidi kuhusu hilo katika mafundisho yajayo. Lakini ikiwa unataka kuona rangi ya maandishi ikibadilika kuwa buluu, wezesha mhariri wa kuingiliana, bofya kichupo cha `styles.css` na badilisha `color: red;` kuwa `color: blue;`.

:::interactive_editor

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>Review page Example</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
    <h1 id="title">Movie Review Page</h1>
  </body>
</html>
```

```css
#title {
  color: red;
}
```

:::

Alama ya hash (`#`) mbele ya `title` inamwambia kompyuta unataka kulenga `id` yenye thamani hiyo. Majina ya `id` hayapaswi kutumika zaidi ya mara moja, na yanapaswa kuwa ya kipekee kila wakati. Jambo lingine la kuzingatia kuhusu thamani za `id` ni kwamba haziwezi kuwa na nafasi ndani yake. Hapa kuna mfano wa kutumia maneno `main` na `heading` kama thamani ya sifa ya kipengele `id`:

```html
<h1 id="main heading">Main heading</h1>
```

Vivinjari vitaona nafasi hii kama sehemu ya `id` ambayo itasababisha matatizo yasiyotakikana linapokuja suala la mtindo na uandishi wa msimbo. Thamani za sifa ya kipengele `id` zinapaswa kuwa na herufi, nambari, alama za mstari chini, na alama za dashi tu.

Tofauti na sifa ya kipengele `id`, thamani ya sifa ya kipengele `class` haipaswi kuwa ya kipekee na inaweza kuwa na nafasi ndani yake.

Hapa kuna mfano wa kutumia darasa linaloitwa `box` kwa kipengele cha `div`.

```html
<div class="box"></div>
```

Ikiwa ungependa kuongeza majina mengi ya madarasa kwa kipengele, unaweza kufanya hivyo kwa kutenganisha majina kwa nafasi. Hapa kuna mfano uliosasishwa wa kutumia madarasa mengi kwa kipengele cha `div`.

```html
<div class="box red-box"></div>
```

Hapa kuna mfano mwingine wa kutumia madarasa mengi kwa vipengele vingi vya `div`.

**KUMBUKA**: Baadhi ya CSS imetolewa kwako katika mfano huu wa kuingiliana. Usijali kuhusu kuelewa msimbo wa CSS kwa sasa kwa sababu utajifunza zaidi kuhusu hilo katika mafundisho yajayo. Lakini ikiwa ungependa kubadilisha rangi ya masanduku ya kwanza na ya tatu, wezesha mhariri wa kuingiliana na bofya kichupo cha `styles.css` na badilisha `background-color: red;` kuwa `background-color: black;`.

:::interactive_editor

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>Colored boxes example</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
    <div class="box red-box"></div>
    <div class="box blue-box"></div>
    <div class="box red-box"></div>
    <div class="box blue-box"></div>
  </body>
</html>
```

```css
.box {
  width: 100px;
  height: 100px;
}

.red-box {
  background-color: red;
}

.blue-box {
  background-color: blue;
}
```

:::

Kwa hiyo, kwa muhtasari, lini unapaswa kutumia `id` badala ya `class`? Madarasa ni bora kutumika unapotaka kutumia seti ya mitindo kwa vipengele vingi. Ikiwa unataka kulenga kipengele maalum, ni bora kutumia `id` kwa sababu thamani hizo zinapaswa kuwa za kipekee.

## --text--

Lini unapaswa kutumia `id` badala ya `class`?

## --answers--

Tumia `class` unapotaka kitambulisho cha kipekee ambacho kinapaswa kutumika kwa kipengele kimoja tu.

### --feedback--

---
