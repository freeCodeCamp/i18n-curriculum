# --description--

3-光滑数是指没有大于 3 的素因子的整数。对于整数 $N$，我们定义 $S(N)$ 为小于或等于 $N$ 的 3-光滑数集合。例如，$S(20) = \\{1, 2, 3, 4, 6, 8, 9, 12, 16, 18\\}$。

我们定义 $F(N)$ 为 $S(N)$ 的排列数，其中每个元素都出现在其所有真因数之后。

这是 $N = 20$ 的一种可能排列。

-   1,2,4,3,9,8,12,16,6,18。

这不是有效的排列，因为12在它的除数6之前出现。

-   1,2,4,3,9,8,16,6,18,12。

我们可以验证 $F(6) = 5$，$F(8) = 9$，$F(20) = 450$ 和 $F(1000) ≈ 8.8521816557e\\,21$。

查找 $F({10}^{18})$。以科学计数法形式给出你的答案，保留小数点后十位数字。给出答案时，使用小写的 `e` 分隔尾数和指数。例如，如果答案是 $112\\,233\\,445\\,566\\,778\\,899$，则答案格式应为 `1.1223344557e17`。
