# --description--

$a$、$b$、$c$ を正の数とします。

$W$、$X$、$Y$、$Z$ を共線点とし、$|WX| = a$、$|XY| = b$、$|YZ| = c$、$|WZ| = a + b + c$ とします。

$C_{\text{in}}$ を直径が $XY$ の円とします。

$C_{\text{out}}$ を直径が $WZ$ の円とします。

3つ組 ($a$、$b$、$c$) は、$k ≥ 3$ 個の異なる円 $C_1, C_2, \ldots, C_k$ を次の条件で配置できる場合、*ネックレス3つ組* と呼ばれます。

- $1 ≤ i, j ≤ k$ かつ $i ≠ j$ のとき、$C_i$ と $C_j$ は内部の共通部分を持たない、
- $1 ≤ i ≤ k$ のとき、$C_i$ は $C_{\text{in}}$ と $C_{\text{out}}$ の両方に接している、
- $1 ≤ i < k$ のとき、$C_i$ は $C_{i + 1}$ に接している、
- $C_k$ は $C_1$ に接している。

例えば、(5, 5, 5) と (4, 3, 21) はネックレス3つ組ですが、(2, 2, 5) はそうではないことが示せます。

<img alt="a visual representation of a necklace triplet" src="https://cdn.freecodecamp.org/curriculum/project-euler/necklace-of-circles.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$T(n)$ を、$a$、$b$、$c$ が正の整数で $b ≤ n$ を満たすネックレス3つ組 $(a, b, c)$ の個数とします。例えば、$T(1) = 9$、$T(20) = 732$、$T(3\,000) = 438\,106$ です。

$T(1\,000\,000\,000)$ を求めてください。
