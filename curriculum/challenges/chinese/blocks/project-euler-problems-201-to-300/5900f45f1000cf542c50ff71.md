# --description--

给定集合 {1,2,..., $n$}，我们定义 $f(n, k)$ 为其所有元素和为奇数的 $k$-元素子集的数量。例如，$f(5,3) = 4$，因为集合 {1,2,3,4,5} 中有四个 3-元素子集的元素和为奇数，即：{1,2,4}、{1,3,5}、{2,3,4} 和 {2,4,5}。

当三个值 $n$、$k$ 和 $f(n, k)$ 都是奇数时，我们称它们构成一个奇数三元组 $[n, k, f(n, k)]$。

当 $n ≤ 10$ 时，恰好有五个奇数三元组，分别是：$[1, 1, f(1, 1) = 1]$、$[5, 1, f(5, 1) = 3]$、$[5, 5, f(5, 5) = 1]$、$[9, 1, f(9, 1) = 5]$ 和 $[9, 9, f(9, 9) = 1]$。

在  $n ≤ {10}^{12}$  时，有多少个奇数三元组？
