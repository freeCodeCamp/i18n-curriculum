# --explanation--

`Not for now`는 현재는 더 말하거나 할 일이 없지만, 앞으로 더 이야기할 가능성을 닫지 않는다는 뜻입니다. 대화를 잠시 멈추거나 끝낼 때 정중하게 표현하는 방법입니다. 예를 들어:

- **Sarah:** `Do you have any questions?`

- **Shaun:** `Not for now, thanks.` – 이 표현은 지금은 질문이 없지만 나중에 있을 수 있다는 의미입니다.
