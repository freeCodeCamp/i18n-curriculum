# --interactive--

伪类是特殊的 CSS 关键字，允许你根据元素的特定状态或位置选择元素。元素的状态或位置可能包括：

- 当它处于激活状态时。
- 当鼠标悬停时。
- 当它是父元素的第一个子元素时。
- 当它是父元素的最后一个子元素时。
- 当 `link` 已被访问时。
- 当它被禁用时。

要使用伪类，你通过使用冒号（`:`）后跟伪类名称将其添加到选择器中： 

```css
selector:pseudo-class {
  /* CSS properties */
}
```

让我们看看你如何使用伪类来表现我们已经提到的每个状态和位置。

`:active` 伪类让你选择元素的激活状态，比如点击按钮时：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example Button</button>
```

```css
button:active {
  background: greenyellow;
}
```

:::

`:hover` 伪类定义了元素的悬停状态。一个例子是悬停在按钮或链接上：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="#">Hover over me!</a>  
```

```css
a:hover {
  text-decoration: none;
  color: white;
  background: crimson;
}
```

:::

`:first-child` 伪类选择作为其父元素第一个子元素的元素。以下是定位 `div` 容器中第一个段落元素的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:first-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

该 `div` 中的第一个段落元素将获得 `lightcoral` 背景色和四个边的 `0.4rem` `padding`。 

`:last-child` 伪类选择其父元素的最后一个子元素。下面是一个在容器 `div` 元素中定位最后一个段落元素的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:last-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

`:visited` 伪类允许你为用户已经访问过的链接设置样式：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="https://www.example.com" target="_blank">Visit Example.com</a>
```

```css
a:visited {
  color: purple;
}
```

:::

`:disabled` 伪类允许你为处于禁用模式的交互式元素设置样式：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button disabled>Disabled Button</button>
```

```css
button:disabled {
  background-color: lightgray;  
}
```

:::

正如你所见，伪类允许你根据用户交互（如悬停或访问 `link`）来为元素设置样式。这使你的网站感觉更具互动性。

除了已经提到的伪类，还有其他如：

- `:focus`
- `:first-of-type`
- `:last-of-type`
- `:nth-of-type`
- `:modal`
- `:enabled`
- `:checked`
- `:required`，等等。
