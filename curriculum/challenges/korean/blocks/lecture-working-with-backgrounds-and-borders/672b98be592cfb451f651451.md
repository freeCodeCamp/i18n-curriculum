# --interactive--

CSS에서 배경 그라디언트는 두 가지 이상의 색상이 부드럽게 전환되어 요소의 배경에 적용되는 효과입니다. 그라디언트를 사용하면 이미지를 사용하지 않고도 시각적으로 매력적인 배경을 만들 수 있습니다.

CSS에는 두 가지 주요 그라디언트 유형이 있습니다: 선형 그라디언트와 방사형 그라디언트입니다.

선형 그라디언트는 직선을 따라 색상이 전환됩니다. 방향과 포함할 색상을 정의할 수 있습니다.

기본 구문은 다음과 같습니다:

```css
background: linear-gradient(direction, color-stop1, color-stop2, ...);
```

이 예제에서는 `background` CSS 속성에 `linear gradient` 값을 사용하고 있습니다.

방향은 그라디언트의 진행 방향을 지정합니다. 각도(예: `45deg`), 키워드(예: `to right`, `to bottom`), 또는 면/모서리일 수 있습니다.

`color-stop`는 그라디언트가 전환되는 색상과 위치를 지정합니다.

선형 그라디언트가 어떻게 작동하는지 더 잘 이해하려면 다음 예제를 살펴보세요:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient{
  background: linear-gradient(to right, red, yellow);
  height: 40vh;
}
```

:::

이 CSS는 `red` 쪽에서 `left`에서 `yellow` 쪽으로 `right`로 전환되는 선형 그라디언트를 만듭니다. 그라디언트는 뷰포트 높이의 `40%` 크기를 가진 요소에 적용됩니다. `vh` 단위에 대해서는 이후 수업에서 더 배우게 됩니다.

`to right` 방향은 그라디언트가 왼쪽에서 오른쪽으로 수평으로 진행됨을 의미합니다.

또 다른 그라디언트 유형은 `radial` 그라디언트입니다.

방사형 그라디언트는 원점(보통 중앙)에서 바깥쪽으로 원형 또는 타원형 모양으로 색상이 전환됩니다.

기본 구문은 다음과 같습니다:

```css
background: radial-gradient(shape size at position, color-stop1, color-stop2, ...)
```

구문에서 `shape`는 그라디언트의 모양을 지정하며 `circle` 또는 `ellipse`일 수 있습니다.

`size`는 그라디언트의 끝 모양 크기를 결정하며 `closest-side`, `closest-corner`, `farthest-side` 또는 `farthest-corner` 중 하나일 수 있습니다.

`position`는 그라디언트 중심 위치를 지정하며, 키워드(예: `center`, `top left`, `bottom right`) 또는 정확한 값(예: `50% 50%`, `10px 20px`)으로 지정할 수 있습니다.

마지막으로, 색상 정지는 그라디언트가 전환하는 색상 목록입니다. 각 색상 정지는 위치 값(백분율 또는 길이)을 포함하여 색상이 배치될 위치를 선택적으로 지정할 수 있습니다.

예시는 다음과 같습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient{
  background: radial-gradient(circle closest-side at center, red, yellow 50%, green);
  height: 60vh;
}
```

:::

이 CSS는 요소 중앙에 원형 방사형 그라디언트를 만듭니다. 중앙에서 `red`로 시작해 반경의 `yellow` 위치에서 `50%`로 전환되고 `green`로 끝납니다.

`closest-side` 키워드는 그라디언트의 끝 모양이 요소의 가장 가까운 면에 맞게 조정되도록 합니다. 그라디언트는 뷰포트 높이의 `60%` 크기를 가진 요소에 적용됩니다.

CSS 그라디언트를 다루는 방법을 이해하면 이미지를 사용하지 않고도 시각적으로 매력적인 배경을 제공하여 디자인을 크게 향상할 수 있습니다.

부드러운 전환을 위한 선형 그라디언트와 원형 효과를 위한 방사형 그라디언트 같은 옵션을 통해 웹 디자인에서 유연성과 창의성을 모두 제공합니다.
