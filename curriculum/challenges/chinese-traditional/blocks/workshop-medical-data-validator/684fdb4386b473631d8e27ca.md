# --description--

正規表達式可以包含由反斜線（`\`）後接一個字元所組成的特殊序列。這些序列具有特殊意義。例如，`\d` 會匹配一個十進位數字。

```py
import re

book = "Fahrenheit 451"
print(re.search('\d', book))
# <re.Match object; span=(11, 12), match='4'>
```

在字元 `p` 之後，`patient_id` 應該有一串數字。所以，修改你的正則表達式樣式，使其有字元 `p`，後面跟著特殊序列 `\d`。

# --hints--
