# --description--
Der einfachste Weg, um zu starten, ist, eine HTML-Datei mit dem JavaScript-Code darin zu erstellen. Geben Sie das grundlegende HTML-Skelett in eine Datei auf Ihrem Computer ein:

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>
  <script>
    // Your JavaScript goes here!
    console.log("Hello, World!")
  </script>
</body>
</html>
```

Speichern Sie diese Datei und öffnen Sie sie in einem Webbrowser (Sie können dafür <a href="https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer" target="_blank" >„Live Server“ in Visual Studio Code</a> verwenden!). Öffnen Sie dann die Konsole des Browsers, indem Sie mit der rechten Maustaste auf die leere Webseite klicken und „Untersuchen“ oder „Element untersuchen“ auswählen. Im Bereich „Entwicklertools“ finden und wählen Sie den Tab „Konsole“, wo Sie die Ausgabe unserer `console.log`-Anweisung sehen sollten.

> `console.log()` ist der Befehl, um etwas in der Entwicklerkonsole Ihres Browsers auszugeben. Sie können dies verwenden, um die Ergebnisse aus allen folgenden Artikeln und Übungen in der Konsole auszugeben. Wir empfehlen Ihnen, bei allen Beispielen in dieser und zukünftigen Lektionen mitzucodieren.

Eine andere Möglichkeit, JavaScript in eine Webseite einzubinden, ist über ein externes Skript. Das ist sehr ähnlich wie das Verlinken externer CSS-Dateien zu Ihrer Website.

```html
  <script src="javascript.js"></script>
```

JavaScript-Dateien haben die Endung `.js`, ähnlich wie `.css` für Stylesheets. Externe JavaScript-Dateien werden für komplexere Skripte verwendet.

# --questions--

## --text--

Welche Aussage beschreibt korrekt, wie man JavaScript in ein HTML-Dokument einbindet?

## --answers--

JavaScript kann nur intern innerhalb eines `<script>`-Tags im HTML-Body eingebunden werden.

---

JavaScript-Code muss im `<head>`-Abschnitt eines HTML-Dokuments geschrieben werden, damit er richtig ausgeführt wird.

---

JavaScript kann direkt in einer HTML-Datei mit einem `<script>`-Tag eingebunden oder extern über ein `<script src="javascript.js">`-Tag verlinkt werden.

---

Externe JavaScript-Dateien benötigen ein spezielles Attribut im `<script>`-Tag, damit Webbrowser sie erkennen.
