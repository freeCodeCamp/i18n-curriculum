## Aina za Data za Kawaida katika Python

- **Utangulizi**: Python ni lugha inayobadilika aina zake kwa wakati wa utekelezaji kama JavaScript, maana yake huna haja ya kutangaza aina za vigezo waziwazi. Lugha inajua aina ya kigezo kulingana na kile unachoweka kwenye kigezo.
- **Nambari Kamili**: Nambari isiyo na sehemu za desimali:

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **Nambari ya Desimali**: Nambari yenye sehemu za desimali:

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **Mfuatano wa Herufi**: Mfuatano wa herufi uliowekwa ndani ya alama za nukuu:

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **Thamani ya Kweli au Siyo Kweli**: Thamani inayowakilisha `True` au `False`:

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **Seti**: Mkusanyiko usio na mpangilio wa vipengele vya kipekee:

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **Kamusi**: Mkusanyiko wa jozi za ufunguo-thamani, zikiwa zimefungwa ndani ya mabano ya mviringo:

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **Tuple**: Mkusanyiko uliopangwa na usiobadilika, umefungwa ndani ya mabano ya kawaida:

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **Mfululizo wa Nambari**: Mfululizo wa nambari, mara nyingi hutumika katika mizunguko:

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **Orodha**: Mkusanyiko uliopangwa wa vipengele unaounga mkono aina tofauti za data:

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**: Thamani maalum inayowakilisha kutokuwepo kwa thamani:

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## Aina Zisizobadilika na Zinazobadilika

- **Kitendakazi cha `type()`**: Ili kuona aina ya kigezo, unaweza kutumia kitendakazi cha `type()` kama ifuatavyo:

## Kufanya kazi na Mfuatano wa Herufi

- **Ufafanuzi**: Kama unavyokumbuka kutoka JavaScript, mfuatano wa herufi ni usiobadilika, ambayo ina maana huwezi kubadilisha baada ya kuundwa. Katika Python, unaweza kutumia alama za nukuu moja au alama za nukuu maradufu. Inashauriwa kuchagua kanuni moja na kuifuata:

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **Kupata Herufi kutoka kwa Mfuatano wa Herufi**: Unaweza kupata herufi kutoka kwa mfuatano wa herufi kwa kutumia uandishi wa mabano kama ifuatavyo:

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **Kuepuka Mfuatano wa Herufi**: Unaweza kutumia mkwaju wa nyuma (`\`) ikiwa mfuatano wako wa herufi una alama za nukuu kama ifuatavyo:

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **Kuunganisha Mfuatano wa Herufi**: Ili kuunganisha mfuatano wa herufi, unaweza kutumia opereta ya `+` kama ifuatavyo:

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

Njia nyingine ya kuunganisha mfuatano wa herufi ni kwa kutumia opereta ya `+=`. Hii hutumika kufanya kuunganisha na mgawo katika hatua moja kama ifuatavyo:

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**: Hii ni kifupi cha mfuatano wa herufi uliopangwa. Inakuwezesha kufanya uingizaji wa thamani na pia kuunganisha kwa sintaksia fupi na inayosomeka:

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **Kuchukua Sehemu ya Mfuatano wa Herufi**: Hii ni pale unapoweza kutoa sehemu za mfuatano wa herufi. Hapa ni sintaksia ya msingi:

```py
str[start:stop:step]
```

Nambari ya kuanzia inaonyesha kielezo ambako uchimbaji unapaswa kuanza. Nambari ya kuacha ni mahali ambapo sehemu hiyo inapaswa kuisha. Nambari hii haijumuishi thamani hiyo. Nambari ya hatua inaonyesha kipindi cha kuongeza kwa uchimbaji. Hapa kuna mifano kadhaa:

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **Kupata Urefu wa Mfuatano wa Herufi**: Kitendakazi cha `len()` hutumika kurudisha idadi ya herufi katika mfuatano:

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## Kufanya kazi na opereta ya `in`

- **Opereta ya `in`**: Hii hurudisha thamani ya kweli au siyo kweli inayobainisha kama herufi au herufi zipo katika mfuatano wa herufi au la:

my_str = 'Hello world'

## Njia za Kawaida za Mfuatano wa Herufi

- **`upper()`**: Njia hii ya mfuatano wa herufi hurudisha mfuatano mpya wa herufi ambapo herufi zote zimebadilishwa kuwa herufi kubwa:

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`lower()`**: Njia hii ya mfuatano wa herufi hurudisha mfuatano mpya wa herufi ambapo herufi zote zimebadilishwa kuwa herufi ndogo:

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`strip()`**: Njia hii ya mfuatano wa herufi hurudisha nakala ya mfuatano wa herufi ambapo herufi zilizotangulia na kufuatia zimeondolewa (kama hakuna hoja iliyotumwa kwa njia hii, huondoa nafasi za mwanzoni na mwishoni).

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**: Hii hurudisha mfuatano mpya wa herufi ambapo matukio yote ya mfuatano wa zamani yamebadilishwa na mfuatano mpya.

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**: Hii hutumika kugawanya mfuatano wa herufi kuwa orodha kwa kutumia kitenzi kilichobainishwa. Kitenzi ni mfuatano wa herufi unaoelezea mahali pa kugawanya.

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**: Hii hutumika kuunganisha vipengele vya mkusanyiko unaoweza kurudiwa kuwa mfuatano wa herufi kwa kutumia kitenzi. Mkusanyiko unaoweza kurudiwa ni mkusanyiko wa vipengele unaoweza kurudiwa kama orodha, mfuatano wa herufi au tuple.

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`startswith(prefix)`**: Njia hii ya mfuatano wa herufi hurudisha thamani ya kweli au siyo kweli ikionyesha kama mfuatano unaanza na kiambishi awali kilichotajwa:

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`endswith(suffix)`**: Njia hii ya mfuatano wa herufi hurudisha thamani ya kweli au siyo kweli ikionyesha kama mfuatano unaisha na kiambishi cha mwisho kilichotajwa:

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`find()`**: Njia hii ya mfuatano wa herufi hurudisha kielezo cha tukio la kwanza la mfuatano mdogo. Ikiwa haipatikani, basi `-1` hurudishwa:

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`count(substring)`**: Njia hii ya mfuatano wa herufi huhesabu mara ngapi mfuatano mdogo unaonekana ndani ya mfuatano:

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`capitalize()`**: Njia hii ya mfuatano wa herufi hurudisha mfuatano mpya wa herufi ambapo herufi ya kwanza imewekwa herufi kubwa na herufi zingine ziko herufi ndogo:

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`isupper()`**: Njia hii ya mfuatano wa herufi hurudisha `True` ikiwa herufi zote katika mfuatano ni kubwa na `False` ikiwa sivyo:

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`islower()`**: Njia hii ya mfuatano wa herufi hurudisha `True` ikiwa herufi zote katika mfuatano ni ndogo na `False` ikiwa sivyo:

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`title()`**: Njia hii ya mfuatano wa herufi hurudisha mfuatano mpya wa herufi ambapo herufi ya kwanza ya kila neno imewekwa kuwa kubwa:

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`maketrans()`**: Njia hii ya mfuatano wa herufi hutumika kuunda jedwali la ramani za herufi 1 kwa 1 kwa ajili ya tafsiri. Mara nyingi hutumika pamoja na njia ya `translate()` ambayo huweka jedwali hilo kwa mfuatano wa herufi na kurudisha matokeo yaliyotafsiriwa.

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## Operesheni za Kawaida zinazotumika na Nambari Kamili na Nambari za Desimali

- **Operesheni za Hisabati za Msingi**: Katika Python, unaweza kufanya operesheni za hisabati za msingi na nambari kamili na nambari za desimali ikiwa ni pamoja na kuongeza, kutoa, kuzidisha na kugawanya:

## Augmented Assignments

- **Ufafanuzi**: Mgawo wa thamani ulioboreshwa huunganisha operesheni ya binary na mgawo katika hatua moja. Huchukua kigezo, hutumia operesheni kwa thamani nyingine, na kuhifadhi matokeo tena kwenye kigezo hicho hicho.

power = 2
power **= 3

print(power) # 8

Kuna opereta nyingine za mgawo ulioboreshwa pia, kama zile za opereta za bitwise. Zinajumuisha `&=`, `^=`, `>>=`, na `<<=`.

## Kufanya kazi na Vitendakazi

- **Ufafanuzi**: Vitendakazi ni vipande vya msimbo vinavyoweza kutumika tena vinavyopokea ingizo (hoja) na kurudisha matokeo. Ili kuita kitendakazi, unahitaji kurejea jina la kitendakazi ikifuatiwa na mabano:

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call
print(result) # 7
```

Kama kitendakazi hakirudishi thamani waziwazi, basi thamani ya default ya return ni `None`:

```py
def greet():
    print('hello') 

result = greet() # hello
print(result) # None
```

Unaweza pia kutoa thamani za default kwa vigezo kama ifuatavyo:

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

Kama utaita kitendakazi bila idadi sahihi ya hoja, utapata `TypeError`:

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

## Wigo katika Python

- **Wigo wa Ndani**: Hii ni pale kigezo kinapotangazwa ndani ya kitendakazi au darasa na kinaweza kufikiliwa tu ndani ya kitendakazi au darasa hilo.

def my_func():

    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **Wigo wa Iliyojengwa Ndani**: Majina yaliyohifadhiwa katika Python kwa ajili ya vitendakazi vilivyotangazwa awali, moduli, maneno muhimu, na vitu.

print(3 != 4) # True

```py

print(3 < 4) # True

```py
print(3 <= 4) # True
```

## Kufanya kazi na Matamko ya `if`, `elif` na `else`

- **Matamko ya `if`**: Haya ni masharti yanayotumika kuamua kama kitu ni kweli au si kweli. Ikiwa sharti linathibitishwa kuwa `True`, basi sehemu hiyo ya msimbo itaendeshwa.

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **`elif` Clause**: Hizi ni masharti yanayofuata baada ya tamko la `if`. Ikiwa sharti la `elif` linatathminiwa kuwa `True`, basi sehemu hiyo ya msimbo itaendeshwa.

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **Kifungu cha `else`**: Hiki kitaendeshwa ikiwa hakuna masharti mengine yanayothibitishwa kuwa `True`.

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

Unaweza pia kutumia matamko ya `if` yaliyopangwa ndani kama ifuatavyo:

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## Thamani za Kweli na Uongo

- **Ufafanuzi**: Katika Python, kila thamani ina thamani ya kweli au siyo kweli ya asili, au hisia iliyojengwa ndani ya kama inapaswa kutendewa kama `True` au `False` katika muktadha wa mantiki. Thamani nyingi zinachukuliwa kuwa za kweli, yaani, zinathibitishwa kuwa `True` katika muktadha wa mantiki. Nyingine ni za uongo, zinazoangaliwa kuwa `False`. Hapa kuna mifano ya thamani za uongo:

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

Thamani nyingine kama nambari zisizo sifuri, na mfuatano wa herufi usio tupu ni za kweli.

## Kufanya kazi na Kitendakazi cha `bool()`

- **Ufafanuzi**: Ikiwa unataka kuangalia kama thamani ni ya kweli au uongo, unaweza kutumia kitendakazi kilicho ndani cha `bool()`. Hukiibadilisha thamani kuwa sawa na thamani ya kweli au siyo kweli na kurudisha `True` kwa thamani za kweli na `False` kwa thamani za uongo. Hapa kuna mifano michache:

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## Opereta za Thamani ya Kweli au Siyo Kweli na Kukata Njia

- **Ufafanuzi**: Hizi ni opereta maalum zinazokuwezesha kuunganisha usemi wa kihesabu zaidi kuunda mantiki tata ya maamuzi katika msimbo wako. Kuna opereta tatu za thamani ya kweli au siyo kweli katika Python: `and`, `or`, na `not`.
- **Opereta ya `and`**: Opereta hii huchukua operands mbili na hurudisha operand ya kwanza ikiwa ni ya uongo, vinginevyo hurudisha operand ya pili. Operands zote mbili lazima ziwe za kweli ili usemi uzalishwe kuwa wa kweli.

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

Unaweza pia kutumia opereta ya `and` katika masharti kama ifuatavyo:

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **Opereta ya `or`**: Opereta hii hurudisha operand ya kwanza ikiwa ni ya kweli, vinginevyo hurudisha operand ya pili. Usemi wa `or` hutokea kuwa wa kweli ikiwa angalau operand moja ni ya kweli. Hapa kuna mfano:

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

Kama ilivyo kwa opereta ya `and`, unaweza kutumia opereta ya `or` katika masharti kama ifuatavyo:

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **Kukata mzunguko**: Vendeta za `and` na `or` zinajulikana kama vendeta za kukata mzunguko. Kukata mzunguko maana yake ni Python hukagua thamani kutoka kushoto kwenda kulia na kusitisha mara tu inapotambua matokeo ya mwisho.
- **Vendeta ya `not`**: Vendeta hii huchukua kipengele kimoja na kubadilisha thamani yake ya kweli au siyo kweli. Hubadilisha thamani za kweli kuwa `False` na thamani za uongo kuwa `True`. Tofauti na vendeta tulizozitazama awali, `not` daima hurudisha `True` au `False`. Hapa kuna mifano kadhaa:

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

Hapa kuna mfano wa opereta ya `not` katika sharti:

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```

numbers = (1, 2, 3, 4, 5)

- Mizunguko ya `for` na `while` inaweza kuunganishwa na kifungu cha `else`, ambacho hutekelezwa tu wakati mzunguko haujakamilishwa kwa `break`:

            break
    else:
        print(f"'{word}' has no vowels")
```

## Mfululizo na Matumizi Yake Katika Mizunguko

- **Kitendakazi cha `range()`**: Kinatumika kuzalisha mfululizo wa nambari kamili.

```py
range(start, stop, step)
```

- Hoja ya `stop` inayohitajika ni nambari kamili (isiyo jumuishi) inayowakilisha mwisho wa mfululizo wa nambari unaozalishwa. Hapa kuna mfano wa kutumia kitendakazi cha `range()`:

```py
for num in range(3):
    print(num)
```

- Ikiwa hoja ya `start` haijatolewa, basi kwa chaguo-msingi itakuwa `0`. Kwa chaguo-msingi mfululizo wa nambari kamili utaongezeka kwa `1`. Unaweza kutumia hoja ya hiari ya `step` kubadilisha thamani ya ongezeko la chaguo-msingi. Hapa kuna mfano wa kuzalisha mfululizo wa nambari kamili za hata (even) kuanzia 2 hadi lakini si pamoja na 11 (yaani, inajumuisha 10)

```py
for num in range(2, 11, 2):
    print(num)
```

- Ikiwa hutoi hoja yoyote kwa kitendakazi cha `range()`, basi utapata `TypeError`.

- Kitendakazi cha `range()` kinakubali nambari kamili tu kama hoja, si nambari za desimali. Kutumia nambari za desimali pia kutaelekea kuwa na `TypeError`:

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- Unaweza kutumia nambari kamili hasi kwa hoja ya `step` kuzalisha mfululizo wa nambari kamili kwa mpangilio wa kushuka:

```py
for num in range(40, 0, -10):
    print(num)
```

- Kitendakazi cha `range()` pia kinaweza kutumika kuunda orodha ya nambari kamili kwa kukitumia pamoja na kijenzi cha `list`. Kijenzi cha `list` kinatumika kubadilisha inayoweza kurudiwa kuwa orodha. Hapa kuna mfano wa kuzalisha orodha ya nambari kamili za hata kati ya 2 na 10 ikijumuisha:

```py
numbers = list(range(2, 11, 2))

```

## Kitendakazi `enumerate()` na `zip()` katika Python

- **`enumerate()`**: hutumika kurudia juu ya mfululizo na kufuatilia kielezo cha kila kipengee katika mfululizo huo. Kitendakazi `enumerate()` huchukua iterable kama hoja na hurudisha kitu cha `enumerate` kinachojumuisha kielezo na thamani ya kila kipengee katika iterable.

languages = ['Spanish', 'English', 'Russian', 'Chinese']

## Iterable Functions

- **`filter()`**: Inatumika kuchuja vipengele kutoka kwenye inayoweza kurudiwa kulingana na sharti fulani. Inarudisha iterator ambayo ina vipengele tu vinavyokidhi sharti hilo. Hapa kuna mfano wa kuunda orodha mpya ya maneno marefu zaidi ya herufi nne:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**: Inatumika kutumia kitendakazi kwa kila kipengee katika inayoweza kurudiwa na kurudisha inayoweza kurudiwa mpya yenye matokeo. Hapa kuna mfano wa kutumia kitendakazi cha `map()` kubadilisha orodha ya joto la celsius kuwa fahrenheit:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**: Inatumika kupata jumla kutoka kwenye inayoweza kurudiwa kama orodha au tuple. Hapa kuna mfano wa kutumia kitendakazi cha `sum()`:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- Pia unaweza pia kupitisha hoja ya hiari ya `start` ambayo inaweka thamani ya mwanzo kwa jumla. Hapa kuna mfano uliosasishwa ukitumia hoja ya `start` kama hoja ya nafasi:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- Pia unaweza kuchagua kutumia hoja ya `start` kama hoja ya neno kuu kama ifuatavyo badala yake:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

pizza.items()

pizza.update({ 'price': 15, 'total_time': 25 })
```

## Kurudia juu ya kamusi

- **Kurudia juu ya thamani**: Ikiwa unahitaji kurudia juu ya thamani katika kamusi, unaweza kuandika mzunguko wa `for` na `values()` kupata thamani zote za kamusi.

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,
    'Headphones': 70,
}

for price in products.values():
    print(price)
```

Output:

```md
990
600
250
70
```

- **Kurudia juu ya funguo**: Ikiwa unahitaji kurudia juu ya funguo katika kamusi ya `products` hapo juu, unaweza kuandika `products.keys()` au `products` moja kwa moja.

```python
for product in products.keys():
    print(product)
    
# Or

for product in products:
    print(product)
```

Output:

```md
Laptop
Smartphone
Tablet
Headphones
```

- **Kurudia juu ya jozi za ufunguo-thamani**: Ikiwa unahitaji kurudia juu ya funguo na thamani zao sambamba kwa wakati mmoja, unaweza kurudia juu ya `products.items()`. Unapata tuple binafsi zenye funguo na thamani zao sambamba.

```python
for product in products.items():

```

Output:

```md

for product, price in products.items():

Kitendakazi hurejesha kitu cha `enumerate`, ambacho huweka nambari kamili kwa kila kipengee, kama kielezi. Unaweza kuanza kielezi kutoka nambari yoyote, lakini kwa chaguo-msingi, huanza kutoka 0.

Kuweka kielezo na kipengee katika vigezo tofauti vya mzunguko ni njia ya kawaida ya kutumia `enumerate()`. Kwa mfano, kwa `products.items()`, unaweza kupata jozi nzima ya ufunguo-thamani pamoja na kielezo:

```python
for index, product in enumerate(products.items()):

Output:

1 ('Smartphone', 600)
2 ('Tablet', 250)
3 ('Headphones', 70)
```

Ili kubadilisha thamani ya mwanzo ya kuhesabu, unaweza kupitisha hoja ya pili kwa `enumerate()`. Kwa mfano, hapa tunaanza kuhesabu kutoka 1.

```python
for index, product in enumerate(products.items(), 1):
    print(index, product)
```

Output:

```md
1 ('Laptop', 990)
2 ('Smartphone', 600)

my_set = {1, 2, 3, 4, 5}

## Operesheni za seti za kihesabu

- **Njia za `issubset()` na `issuperset()`**: Njia za `issubset()` na `issuperset()` huthibitisha kama seti ni seti ndogo au seti kubwa ya seti nyingine, kwa mtiririko huo.

```python
my_set = {1, 2, 3, 4, 5}

print(your_set.issubset(my_set)) # True

```

- **Kipengele cha muungano (`|`)**: Kipengele cha muungano `|` hurudisha seti mpya yenye vipengele vyote kutoka seti zote mbili.

my_set = {1, 2, 3, 4, 5}

```

```python
my_set = {1, 2, 3, 4, 5}

```

- **Kipengele cha tofauti ya usawa (`^`)**: Kipengele cha tofauti ya usawa `^` hurudisha seti mpya yenye vipengele vilivyoko katika seti ya kwanza au ya pili, lakini si katika zote mbili.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

```

- **Kipengele cha `in`**: Unaweza kuthibitisha kama kipengele kiko katika seti au hakiko kwa kutumia kipengele cha `in`.

## Maktaba ya kawaida ya Python

- **Maktaba ya kawaida ya Python**: Maktaba hukupa msimbo uliotangulia kuandikwa na unaoweza kutumika tena, kama vile vitendakazi, madarasa ya vitu, na miundo ya data, ambayo unaweza kutumia tena katika miradi yako. Python ina maktaba ya kawaida pana yenye moduli zilizojengwa ndani zinazotekeleza suluhisho za kawaida kwa matatizo na zoezi nyingi. Baadhi ya mifano ya moduli maarufu zilizojengwa ndani ni `math`, `random`, `re` (mfupi wa "usemi wa kawaida"), na `datetime`.

## Import Statement

- **Kauli ya import**: Ili kupata upatikanaji wa vipengele vilivyotangazwa katika moduli zilizojengwa ndani, unatumia kauli ya import. Kauli za import kwa kawaida huandikwa juu kabisa ya faili. Kauli za import hufanya kazi sawa kwa vitendakazi, madarasa ya vitu, constants, vigezo, na vipengele vingine vyote vilivyotangazwa katika moduli.

- **Kauli ya msingi ya import**: Unaweza kutumia neno kuu la `import` ikifuatiwa na jina la moduli:

```python
import module_name
```

Kisha, ikiwa unahitaji kuita kitendakazi kutoka moduli hiyo, utatumia uandishi wa nukta, ukiandika jina la moduli ikifuatiwa na jina la kitendakazi.

```python
module_name.function_name()
```

Kwa mfano, utaandika yafuatayo katika msimbo wako kuleta moduli ya `math` na kupata mzizi wa mraba wa 36:

```python
import math

math.sqrt(36)
```

- **Kuleta moduli kwa jina tofauti**: Ikiwa unahitaji kuleta moduli kwa jina tofauti (linalojulikana pia kama "alias"), unaweza kutumia `as` ikifuatiwa na jina la alias mwishoni mwa kauli ya import. Hii hutumika mara nyingi kwa majina marefu ya moduli au kuepuka migongano ya majina.

```python
import module_name as module_alias
```

Kwa mfano, ili kurejelea moduli ya `math` kama `m` katika msimbo wako, unaweza kuweka alias kama ifuatavyo:

```python
import math as m
```

Kisha, unaweza kupata vipengele vya moduli kwa kutumia alias:

```python
m.sqrt(36)
```

- **Kuleta vipengele maalum**: Ikiwa hautaji kila kitu kutoka moduli, unaweza kuleta vipengele maalum kwa kutumia `from`. Katika kesi hii, kauli ya import huanza na `from`, ikifuatiwa na jina la moduli, kisha neno kuu la `import`, na hatimaye majina ya vipengele unavyotaka kuleta.

```python
from module_name import name1, name2
```

Kisha, unaweza kutumia majina haya bila kiambishi awali cha moduli katika script yako ya Python. Kwa mfano:

```python
from math import radians, sin, cos

angle_degrees = 40
angle_radians = radians(angle_degrees)

sine_value = sin(angle_radians)
cos_value = cos(angle_radians)

print(sine_value) # 0.6427876096865393
print(cos_value)  # 0.766044443118978
```
 
Hii ni msaada, lakini inaweza kusababisha migongano ya majina ikiwa tayari una vitendakazi au vigezo vyenye majina sawa. Kumbuka hili unapotafuta aina gani ya kauli ya import unayotaka kutumia.

Ikiwa unahitaji kuweka alias kwa majina haya, unaweza pia kufanya hivyo kwa kutumia neno kuu la `as` ikifuatiwa na alias.

```python
from module_name import name1 as alias1, name2 as alias2
```

- **Kauli ya import yenye nyota (`*`)**: Nyota inaambia Python kwamba unataka kuleta kila kitu katika moduli hiyo, lakini unataka kuileta ili usihitaji kutumia jina la moduli kama kiambishi awali.

```python
from module_name import *
```

Kwa mfano, ukitumia hii kuleta moduli ya `math`, utaweza kuita kitendakazi chochote kilichotangazwa katika moduli hiyo bila kutaja jina la moduli kama kiambishi awali.

```python
from math import *
print(sqrt(36))  # 6.0
```

Hata hivyo, hii kwa kawaida haipendekezwi kwa sababu inaweza kusababisha migongano ya majina na kufanya iwe vigumu kujua majina yanatoka wapi.

Lakini ikiwa faili ya Python inaletwa kama moduli katika script nyingine ya Python, thamani ya variable `__name__` huwekwa kuwa jina la moduli hiyo.

Hii ndiyo sababu mara nyingi utapata masharti haya katika script za Python. Yana msimbo ambao unataka kuendesha **tu** ikiwa script ya Python inaendeshwa kama programu kuu.

if __name__ == '__main__':

## Makosa ya kawaida katika Python

- **SyntaxError**: Kosa ambalo Python hutupa wakati msimbo wako haufuati sheria zake za sintaksia. Kwa mfano, msimbo `print("Hello there"` utasababisha kosa la sintaksia lenye ujumbe, `SyntaxError: '(' was never closed`, kwa sababu msimbo unakosa mkato wa kufunga mabano.
- **NameError**: Python hutupa `NameError` unapojaribu kupata upatikanaji wa kigezo au kitendakazi ambacho hujafafanua. Kwa mfano, ikiwa una mstari `print(username)` katika msimbo wako bila kuwa na kigezo `username` kilichofafanuliwa kwanza, utapata kosa la jina lenye ujumbe `NameError: name 'username' is not defined`.
- **TypeError**: Huu ni kosa ambalo Python hutupa unapofanya operesheni kwa aina mbili au zaidi za data zisizolingana. Kwa mfano, ukijaribu kuongeza mfuatano wa herufi kwenye nambari, utapata kosa `TypeError: can only concatenate str (not "int") to str`.
- **IndexError**: Utapata `IndexError` ikiwa utapata upatikanaji wa kielezo ambacho hakipo katika orodha au mfululizo mwingine kama tuple na mfuatano wa herufi. Kwa mfano, katika mfuatano wa herufi `Hello world`, kielezo cha herufi ya mwisho ni `10`. Ukifuata na kupata herufi kwa njia hii, `greet = "hello world"; print(greet[11])`, utapata kosa lenye ujumbe `IndexError: string index out of range`.
- **AttributeError**: Python hutupa kosa hili unapojaribu kutumia njia au sifa ambayo haipo katika kitu cha aina hiyo. Kwa mfano, kuita `.append()` kwenye mfuatano wa herufi kama `"hello".append("!")` kutasababisha kosa lenye ujumbe `AttributeError: 'str' object has no attribute 'append'`.

## Exception Handling

- **`try...except`**: Hii hutumika kutekeleza kipande cha msimbo ambacho kinaweza kusababisha hitilafu. Kipande cha `try` ndicho unachotarajia hitilafu inaweza kutokea, wakati kipande cha `except` kinachukua hitilafu maalum na kinaendeshwa ikiwa hitilafu hiyo maalum itatupwa. Hapa kuna mfano:

  ```py
  try:
    print(22 / 0)
  except ZeroDivisionError:
    print('You can\'t divide by zero!')
    # You can't divide by zero!
  ```

Pia unaweza kuunganisha vipande vingi vya `except` ili kushughulikia aina zaidi za hitilafu:

  ```py
  try:
    number = int(input('Enter a number: '))
    print(22 / number)
  except ZeroDivisionError:
    print('You cannot divide by zero!')
    # You cannot divide by zero! prints when you enter 0
  except ValueError:
    print('Please enter a valid number!')
    # Please enter a valid number! prints when you don't enter an integer
  ```

- **`else` na `finally`**: Vipande hivi vinaongeza `try...except`. Ikiwa hakuna hitilafu itakayojitokeza, kipande cha `else` kinaendeshwa. Kipande cha `finally` kinaendeshwa kila wakati bila kujali kama kuna hitilafu.

  ```py
  try:
    result = 100 / 4
  except ZeroDivisionError:
    print('You cannot divide by zero!') # This will not run
  else:
    print(f'Result is {result}') # Result is 25.0
  finally:
    print('Execution complete!') # Execution complete!
  ```

- **Kipengele cha hitilafu**: Hii inakuwezesha kupata hitilafu yenyewe kwa ajili ya kurekebisha makosa vizuri zaidi na kuchapisha ujumbe wa moja kwa moja wa hitilafu. Ili kupata kipengele cha hitilafu, unahitaji kutumia neno kuu la `as`. Hapa kuna mfano:

  ```py
  try:
      value = int('This will raise an error')
  except ValueError as e:
      print(f'Caught an error: {e}')
      # Caught an error: invalid literal for int() with base 10: 'This will raise an error'
  ```

- **Kauli ya `raise`**: Hii inakuwezesha kutupa hitilafu kwa hiari yako. Unaweza kuitumia kutupa hitilafu wakati hali fulani inatimizwa. Hapa kuna mfano:

  ```py
  def divide(a, b):
      if b == 0:
          raise ZeroDivisionError('You cannot divide by zero')
      return a / b
  ```
  
## Exception Signaling
  
Sentensi ya `raise` pia ni muhimu unapounda isipokuwa zako za kawaida, kwani unaweza kuitumia kutupa isipokuwa yenye ujumbe wa kawaida. Hapa kuna mfano wa hilo:

  ```py
  class InvalidCredentialsError(Exception):
      def __init__(self, message="Invalid username or password"):
          self.message = message
          super().__init__(self.message)

  def login(username, password):
      stored_username = "admin"
      stored_password = "password123"
      
      if username != stored_username or password != stored_password:
          raise InvalidCredentialsError()
      
      return f"Welcome, {username}!"
  ```  

Hapa ni jinsi unavyoweza kutumia kitendakazi cha `login` pamoja na isipokuwa ya `InvalidCredentialsError`:

  ```py
  # failed login attempt
  try:
      message = login("user", "wrongpassword")
  except InvalidCredentialsError as e:
      print(f"Login failed: {e}")
  else:
      print(message)

  # successful login attempt
  try:
      message = login("admin", "password123")
  except InvalidCredentialsError as e:
      # This block is not executed because the login was successful
      print(f"Login failed: {e}")
  else:
      # The else block runs if the 'try' block completes without an exception
      print(message)
  ```

Sentensi ya `raise` pia inaweza kutumika pamoja na neno kuu la `from` kuunganisha isipokuwa, kuonyesha uhusiano kati ya makosa tofauti:

  ```py
  def parse_config(filename):
    try:
        with open(filename, 'r') as file:
            data = file.read()
            return int(data)
    except FileNotFoundError:
        raise ValueError('Configuration file is missing') from None
    except ValueError as e:
        raise ValueError('Invalid configuration format') from e

  config = parse_config('config.txt')
  ```

## Njia dunder (za kichawi)

- **Maana**: Njia maalum zinazotangulia na kumalizika na alama mbili za mstari chini (mfano, `__init__`, `__len__`, `__str__`, `__eq__`). Python hutumia njia hizi ndani ya mfumo wake kwa operesheni zilizojengwa ndani.

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **Kuitisha njia dunder kwa njia isiyo ya moja kwa moja**: Huna haja ya kuitisha njia dunder moja kwa moja. Badala yake, Python huziita moja kwa moja wakati matendo fulani yanapotokea. Operesheni hizi ni pamoja na:

- **operesheni za kihesabu kama kuongeza, kutoa, kuzidisha, kugawanya, na nyinginezo**. `__add__()` huitwa kwa kuongeza, `__sub__()` kwa kutoa, `__mul__()` kwa kuzidisha, na `__truediv__()` kwa kugawanya.
        
- **operesheni za mfuatano wa herufi kama kuunganisha, kurudia, kupanga, na kubadilisha kuwa maandishi**. `__add__()` huitwa kwa kuunganisha, `__mul__()` kwa kurudia, `__format__()` kwa kupanga, `__str__()` na `__repr__()` kwa kubadilisha kuwa maandishi, na kadhalika.
        
- **operesheni za kulinganisha kama usawa, chini ya, juu ya, na nyinginezo**. `__eq__()` huitwa kwa ukaguzi wa usawa, `__lt__()` kwa chini ya, `__gt__()` kwa juu ya, na kadhalika.
        
- **operesheni za kurudia kama kufanya kitu kiwe kinachoweza kurudiwa na kusonga mbele kupitia vitu**. `__iter__()` huitwa kurudisha kielezi cha kurudia na `__next__()` kuchukua kipengele kinachofuata.

## Mfano Halisi wa Maisha: Kifaa cha Ununuzi

```python
class Cart:
    def __init__(self):
        self.items = []

    def add(self, item):

## Encapsulation ni nini?

- **Encapsulation**: Kuunganisha sifa za kipengele na njia za kitu katika kitengo kimoja. Hii inakuwezesha kuficha hali ya ndani ya kitu nyuma ya seti rahisi ya njia na sifa za umma zinazofanya kazi kama milango. Nyuma ya milango hiyo kuna sifa za kipengele na njia za faragha zinazodhibiti jinsi data inavyobadilika na nani anaweza kuiona.
- **Mfano wa encapsulation**: Ikiwa unataka kufuatilia salio la pochi, utaruhusu kuweka na kutoa fedha, lakini hutaki mtu yeyote kuingilia salio la pochi yenyewe:

```py
class Wallet:
   def __init__(self, balance):
       self.__balance = balance # Private attribute

   def deposit(self, amount):
       if amount > 0:
           self.__balance += amount # Add to the balance safely

   def withdraw(self, amount):
       if 0 < amount <= self.__balance:
           self.__balance -= amount # Remove from the balance safely

account = Wallet(500)
print(account.__balance) # AttributeError: 'Wallet' object has no attribute '__balance'
```

- **Tofauti kati ya kuweka kiambishi awali cha underscore moja na mbili kwa sifa za kipengele**: Kuweka kiambishi awali cha underscore moja kwa sifa za kipengele na njia ina maana kwamba zinakusudiwa kwa matumizi ya ndani. Hii ni mkataba, na haizuizi upatikanaji wa sifa kutoka nje. Kuweka kiambishi awali cha underscore mbili kwa sifa za kipengele na njia kwa ufanisi huzuia upatikanaji wake kutoka nje ya darasa lao.

## Getters na setters ni nini?

- **Getters na setters**: Njia zinazokuwezesha kudhibiti jinsi sifa za darasa la vitu zinavyopatikana na kubadilishwa. Unapata thamani kwa getters na unaweka thamani kwa setters.
- **Sifa**: Zinahusisha getters na setters, na kuruhusu upatikanaji wa data. Zinatumia mantiki ya ziada nyuma ya pazia wakati unapata, kuweka, au kufuta thamani.
- **Kwa nini sifa badala ya njia**: Sifa hutumika badala ya njia kwa ajili ya usomaji bora na msimbo safi zaidi. Zinakuwezesha kupata thamani kwa uandishi wa nukta, kama sifa za kawaida, bila mabano ya kawaida.
- **Kuumba getter**: Ili kuunda getter, unatumia dekoreta `@property`. Hapa kuna getter inayopata radius ya mduara:

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self): # A getter to get the radius
        return self._radius
  
    @property
    def area(self):  # A getter to calculate area
        return 3.14 * (self._radius ** 2)

my_circle = Circle(3)

print(my_circle.radius) # 3
print(my_circle.area) # 28.26
```

Kutumia self.radius ndani ya `__init__()` huhakikisha setter inafanya kazi wakati wa kuunda kitu, hivyo thamani zisizo halali za radius zinagunduliwa mara moja.

- **Kuumba setter**: Ili kuunda setter itakayoweka radius, lazima uunde njia nyingine yenye jina sawa na utumie `@<property_name>.setter` juu yake:

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self):  # A getter to get the radius
        return self._radius

    @radius.setter
    def radius(self, value):  # A setter to set the radius
        if value <= 0:
            raise ValueError('Radius must be positive')
        self._radius = value

my_circle = Circle(3)
print('Initial radius:', my_circle.radius) # Initial radius: 3

my_circle.radius = 8
print('After modifying the radius:', my_circle.radius) # After modifying the radius: 8
```

- **Jinsi Python inavyoshughulikia getters na setters**: Mara tu unapounda getters na setters, Python huwaita moja kwa moja chini ya pazia kila unapotumia sintaksia ya sifa ya kawaida kwa njia hii:

```py
my_circle.radius # This will call the getter
my_circle.radius = 4 # This will call the setter
```

Ndani ya setter, haipaswi kuweka thamani kwa jina la sifa yenyewe kwa sababu hiyo itasababisha `RecursionError`.

- **Deleter**: Baada ya kuweka na kupata thamani kwa setter na getter, unaweza kudhibiti jinsi inavyofutwa kwa `deleter`. Deleter hufanya mantiki maalum unapotumia tamko la `del` kwenye sifa. Ili kuunda deleter, unatumia dekoreta `@<property_name>.deleter`.

```py
  # Deleter
    @radius.deleter
    def radius(self):
        print("Deleting radius...")
        del self._radius
```

## Urithi ni nini na unakuzaje matumizi ya msimbo tena?

- **Urithi**: Ni mchakato ambapo darasa la mtoto hutumia sifa za kipengele na njia za darasa la mzazi. Urithi unakuza matumizi ya msimbo tena, hutoa ngazi wazi za uongozi, na hubadilisha tabia bila kuandika upya kila kitu. Ili kutekeleza urithi, darasa la mtoto huchukua jina la darasa la mzazi:

```py
class Parent:
    # Parent attributes and methods

class Child(Parent):
    # Child inherits, extends, and/or overrides where necessary
```

- **Urithi wa moja kwa moja na wa mara nyingi**: Wakati darasa la mtoto linarithi sifa za kipengele na njia kutoka kwa mzazi mmoja, kama unavyoona hapo juu, mchakato huo huitwa **urithi wa moja kwa moja**. Wakati darasa la mtoto linarithi sifa za kipengele na njia kutoka kwa wazazi zaidi ya mmoja, hiyo ni **urithi wa mara nyingi**. Hapa kuna sintaksia yake:

```py
class Parent:
    # Attributes and methods for Parent

class Child:
    # Attributes and methods for Child

class GrandChild(Parent, Child):
    # GrandChild inherits from both Parent and Child
    # GrandChild can combine or override behavior from each
```

- **`super()` kitendakazi**: Kitendakazi kinachokuwezesha kufanya **wito** wa njia kutoka kwa darasa la mzazi, wakati darasa lina utekelezaji tofauti wa njia hiyo, au linaongeza njia hiyo, bila kuzidisha msimbo.

## Polymorphism ni nini na inavyosaidia upya matumizi ya msimbo?

- **Polymorphism**: Kanuni ya OOP inayoruhusu madarasa tofauti kutumia jina moja la njia, lakini kila darasa hutekeleza njia hiyo kwa namna tofauti inapoitwa. Hapa kuna sintaksia yake:

```py
class A:
   def action(self): ...

class B:
   def action(self): ...

   def action(self): ...

```

## Nini maana ya Name Mangling na Inafanya Kazi Vipi?

- **Name Mangling**: Ni mchakato ambapo Python ndani yake hubadilisha jina la sifa ya kipengele iliyo na kiambishi awali cha alama mbili za mstari chini kwa kuongeza mstari chini na jina la darasa kama kiambishi awali, ikibadilisha `__attribute` kuwa `_ClassName__attribute`.
- **Madhumuni ya Name Mangling**: Madhumuni kuu ya name mangling ni kuzuia upitishaji wa bahati mbaya wa sifa za kipengele na njia wakati unapotumia urithi. Hapa kuna msimbo unaofanya hili kuwa rahisi kuelewa:

```py
class Parent:
    def __init__(self):
        self.__data = 'Parent data'

class Child(Parent):
    def __init__(self):
        super().__init__()

c = Child()

```

## Abstraksheni ni nini na inasaidiaje kuweka mifumo tata kuwa na mpangilio?

- **Abstraksheni**: Ni dhana ya programu ambapo maelezo tata ya utekelezaji wa kitu au mfumo hufichwa na kuonyeshwa tu vipengele muhimu. Katika Python na lugha nyingine za programu, abstraksheni hufanya mifumo tata kuwa rahisi kwa kuongeza uwezo wa kutumia tena.
- **Mfano wa abstraksheni**: Mfano mzuri wa abstraksheni katika maisha ya kila siku ni gari ambalo hukuruhusu kutumia tu gurudumu, pedali, na shifta bila kujua jinsi injini au breki zinavyofanya kazi.
- **Jinsi Python inavyotekeleza abstraksheni**: Python hutekeleza abstraksheni kupitia moduli ya `abc`. Moduli hii hutoa darasa la vitu la `ABC` (darasa la msingi la abstraksheni) na dekoreta wa `@abstractmethod`. Darasa la msingi la abstraksheni (ABC) linafafanua njia na vigezo vya kawaida ambavyo madarasa yanayotokana nayo lazima yatekeleze. Haiwezi kuanzishwa moja kwa moja.
- **Jinsi njia ya abstraksheni inavyofafanuliwa**: Njia ya abstraksheni hufafanuliwa kwa `@abstractmethod` na lazima ibadilishwe katika madarasa yanayotokana nayo, hata kama ina utekelezaji wa msingi. Sintaksia ya msingi ya abstraksheni inaonekana kama ifuatavyo:

```py
from abc import ABC, abstractmethod

# Define an abstract base class
class AbstractClass(ABC):
    @abstractmethod
    def abstract_method(self):
        pass

# Concrete subclass that implements the abstract method
class ConcreteClassOne(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassOne')

# Another concrete subclass
class ConcreteClassTwo(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassTwo')
```

## Algoriti na noti ya Big O

- **Algoriti**: Seti ya maagizo yasiyo na utata kwa kutatua tatizo au kutekeleza zoezi. Algoriti lazima zikamilike kwa idadi ya hatua zisizo na mwisho na kila hatua lazima iwe sahihi na isiyo na utata.

- **Noti ya Big O**: Inaelezea utendaji mbaya zaidi, au kiwango cha ukuaji, cha algoriti kadri ukubwa wa ingizo unavyoongezeka. Inazingatia jinsi matumizi ya rasilimali yanavyokua kwa ukubwa wa ingizo, ikisahau mambo ya kudumu na vipengele vya kiwango cha chini.

### Ugumu wa kawaida wa muda

- **O(1) - Muda wa kudumu**: Algoriti huchukua muda sawa bila kujali ukubwa wa ingizo.

```python
def check_even_or_odd(number):
    if number % 2 == 0:
        return 'Even'
    else:
        return 'Odd'
```

- **O(log n) - Muda wa logaritmiki**: Muda huongezeka polepole kadri ingizo linavyokua. Ni kawaida katika algoriti zinazopunguza mara kwa mara ukubwa wa tatizo kwa sehemu (kama Binary Search).

- **O(n) - Muda wa mstari**: Muda wa utekelezaji huongezeka kwa uwiano na ukubwa wa ingizo.

```python
for grade in grades:
    print(grade)
```

- **O(n log n) - Muda wa mstari-logaritmiki**: Ugumu wa kawaida wa muda wa algoriti za kupanga zenye ufanisi kama Merge Sort na Quick Sort.

- **O(n²) - Muda wa mraba**: Muda wa utekelezaji huongezeka mrabani. Mara nyingi huonekana katika mizunguko zilizopangwa ndani.

```python
for i in range(n):
    for j in range(n):
        print("Hello, World!")
```

### Space Complexity

- **O(1) - Nafasi ya kudumu**: Algoriti hutumia kiasi sawa cha kumbukumbu bila kujali ukubwa wa ingizo.
- **O(n) - Nafasi ya mstari**: Matumizi ya kumbukumbu huongezeka kwa uwiano na ukubwa wa ingizo.

## Searching Algorithms

Algoriti za utafutaji hukuruhusu kutafuta lengo ndani ya orodha fulani ya vitu.

Katika sayansi ya kompyuta, kuna algoriti mbili za utafutaji unazopaswa kujua. Ni algoriti za **utafutaji wa mstari** na **utafutaji wa binary**. Ni muhimu kuelewa tofauti kati ya algoriti hizi mbili na lini kutumia kila moja.

### Linear Search

- Linear search hurudia kupitia orodha ya vitu, ikikagua kila kipengee kuanzia mwanzo hadi kipengee lengwa kipatikane.
- Ikiwa kipengee lengwa kinapatikana, hurejesha kielezo ambacho kiko kwenye orodha.
- Ikiwa lengo halipatikani, hurudisha `-1`, ambayo ina maana ya **kielezo batili** katika lugha nyingi za programu.
- Kwa sababu linear search hukagua kila kipengee hadi apate lengo, si mbinu bora kwa orodha kubwa ya vitu.
- Ugumu wa muda wa linear search ni `O(n)` kwa sababu muda unaochukua kutafuta kupitia orodha huongezeka kwa mstari kulingana na ukubwa wa orodha.
- Ugumu wa nafasi wa linear search ni `O(1)` kwa sababu haitegemei nafasi ya ziada kutafuta kupitia orodha.

### Binary Search

- Utafutaji wa binary hufanya kazi kwa kugawanya orodha ya vitu nusu, na kuangalia kama thamani lengwa iko katikati ya orodha.
- Sharti la utafutaji wa binary kufanya kazi ni kwamba vitu katika orodha lazima viwe katika mpangilio wa kuongezeka.
- Utafutaji wa binary ni algoriti yenye ufanisi zaidi kwa kutafuta ndani ya orodha kubwa ya vitu kwa sababu hugawanya orodha nusu na kupuuza nusu yoyote ambayo lengo halipo.
- Ikiwa kipengee lengwa kinapatikana katikati ya orodha, kielezo cha kipengee lengwa hurudishwa.
- Ikiwa kipengee hakipatikani, algoriti huangalia kama kipengee lengwa kiko nusu ya kushoto au nusu ya kulia ya orodha.
- Inaendelea kugawanya sehemu zilizobaki za orodha nusu hadi kipengee lengwa kipatikane.
- Ikiwa kipengee lengwa hatimaye hakipatikani ndani ya orodha, hurudisha `-1`
- Ugumu wa wakati wa utafutaji wa binary ni `O(log n)` kwa sababu muda unaochukua kutafuta ndani ya orodha huongezeka kwa njia ya logaritmiki kulingana na ukubwa wa orodha.
- Ugumu wa nafasi wa utafutaji wa binary ni `O(1)` kwa sababu hauhitaji nafasi ya ziada kutafuta ndani ya orodha.

### Jinsi utafutaji wa mstari unavyotofautiana na utafutaji wa binary

- Binary search ni bora zaidi kwa orodha kubwa ya vitu ikilinganishwa na linear search.
- Ugumu wa muda wa linear search ni `O(n)` kwa sababu muda unaochukua kutafuta kupitia orodha huongezeka kwa mstari kulingana na ukubwa wa orodha.
- Ugumu wa muda wa binary search ni `O(log n)` kwa sababu muda unaochukua kutafuta kupitia orodha huongezeka kwa njia ya logaritm kwa ukubwa wa orodha.

## Algoriti za upangaji na mbinu ya kugawanya-na-kushinda

Katika sayansi ya kompyuta, kugawanya-na-kushinda ni mbinu inayotumika kugawanya tatizo kuwa matatizo madogo madogo ili yawe rahisi kutatuliwa. Rekursi ni mbinu inayotumika mara nyingi katika kugawanya-na-kushinda, na kugawanya-na-kushinda ni mkakati wenye nguvu unaotumika kutekeleza algoriti nyingi za upangaji zenye ufanisi kama vile merge sort.

### Merge Sort

- Merge sort ni algoriti ya upangaji inayofuata mbinu ya kugawanya-na-kushinda.
- Hufanya kazi kwa kugawanya orodha kwa rekursi kuwa orodha ndogo ndogo hadi kila orodha ndogo iwe na kipengee kimoja tu.
- Kisha hurudia kuunganisha orodha ndogo ndogo hizo pamoja kwa mpangilio uliopangwa.
- Ugumu wa muda wa merge sort ni `O(n log n)` kwa sababu orodha hugawanywa mara kwa mara nusu `(log n)` kisha kuunganishwa pamoja `(O(n))`.
- Ugumu wa nafasi wa merge sort ni `O(n)` kwa sababu si algoriti ya upangaji inayofanyika mahali pake (in-place).

## Graphs Overview

Grafu ni seti ya nodi (vichwa) vinavyounganishwa kwa njia ya makali (muunganisho). Kila nodi inaweza kuunganishwa na nodi nyingi nyingine, ikaunda mtandao. Aina tofauti za grafu ni pamoja na:

- Zenye mwelekeo: makali yana mwelekeo (kutoka nodi moja hadi nyingine), mara nyingi huwakilishwa kwa mistari ya moja kwa moja na mishale.
- Zisizo na mwelekeo: makali hayana mwelekeo, huwakilishwa kwa mistari rahisi.
- Vichwa: kila nodi ina lebo au kitambulisho.
- Zenye mizunguko: zina mizunguko (njia inayotoka na kurudi kwenye nodi ileile).
- Zisizo na mizunguko (DAG): hazina mizunguko.
- Makali yenye lebo: kila mkao una lebo inayochorwa karibu na mkao husika.
- Zenye uzito: makali yana uzito (thamani) unaohusishwa nayo, ambao unaweza kutumika kufanya operesheni za kihesabu.
- Zisizounganishwa: zina nodi mbili au zaidi ambazo hazijaunganishwa na makali yoyote.

Grafu hutumika katika programu mbalimbali kama vile ramani, mitandao, mifumo ya mapendekezo, na utatuzi wa utegemezi.

## Priority Queues

Foleni ya kipaumbele ni aina ya data ya muhtasari ambapo kila kipengele kina kipaumbele.

## Heaps

Ni muundo wa data maalum unaotegemea mti wenye sifa maalum sana inayoitwa sifa ya rundo.

Sifa ya rundo inaamua uhusiano kati ya nodi ya mzazi na nodi ya mtoto. Kuna aina mbili za rundo:

- Rundo kubwa (Max-Heap)
  - Thamani ya kila nodi ya mzazi ni kubwa kuliko au sawa na thamani za nodi zake za watoto.
  - Kipengele kikubwa zaidi kiko kwenye mzizi.

- Rundo ndogo (Min-Heap)
  - Thamani ya kila nodi ya mzazi ni ndogo kuliko au sawa na thamani za nodi zake za watoto.
  - Kipengele kidogo zaidi kiko kwenye mzizi.

### Mfano wa moduli ya Python `heapq`

```py
import heapq

# Create empty heap
my_heap = []

# Insert elements
heapq.heappush(my_heap, 9)
heapq.heappush(my_heap, 3)
heapq.heappush(my_heap, 5)
print(my_heap) # [3, 9, 5]

# Remove smallest element
print(heapq.heappop(my_heap))  # 3
print(my_heap) # [5, 9]

# Push + Pop in one step
print(heapq.heappushpop(my_heap, 7)) # 5
print(my_heap) # [7, 9]

# Transform list into heap
nums = [5, 7, 3, 1]
heapq.heapify(nums)
```

### Using Priorities

```py
my_heap = []
heapq.heappush(my_heap, (3, "A"))
heapq.heappush(my_heap, (2, "B"))
heapq.heappush(my_heap, (1, "C"))

# Removes lowest number = highest priority
print(heapq.heappop(my_heap))  # (1, "C")
```

## Utangulizi wa programu inayobadilika (dynamic programming)

- **Ufafanuzi**: Programu inayobadilika ni mbinu ya algoriti inayotatua matatizo magumu kwa kuyavunja kuwa matatizo madogo madogo na kuhifadhi matokeo ili kuepuka mahesabu yanayojirudia.
- **Matatizo yanayojirudia**: Matatizo madogo madogo yanayofanana hujitokeza mara nyingi wakati wa kutatua tatizo kubwa. Badala ya kuhesabu matatizo haya madogo mara kwa mara, tunahifadhi suluhisho lao.
- **Muundo bora wa sehemu**: Suluhisho bora la tatizo lina suluhisho bora za matatizo madogo madogo. Hii ina maana tunaweza kujenga suluhisho bora kwa kuunganisha suluhisho bora za sehemu ndogo ndogo.

## Suluhisho za programu zinazobadilika (dynamic programming)

- **Kumbukumbu ya matokeo (Mbinu ya kutoka juu hadi chini)**: Kumbukumbu ya matokeo huhifadhi matokeo ya wito wa kitendakazi ghali na hurudisha matokeo yaliyohifadhiwa wakati ingizo sawa linapotokea tena.

```py
def climb_stairs_memo(n, memo={}):
    """Dynamic programming with memoization"""
    # Check if we've already calculated this value
    if n in memo:
        return memo[n]  # Return cached result - O(1) lookup!
    
    # Base cases
    if n <= 2:
        return n
    
    # Calculate once and store in memo for future use
    memo[n] = climb_stairs_memo(n-1, memo) + climb_stairs_memo(n-2, memo)
    return memo[n]
```

- **Jedwali la matokeo (Mbinu ya kutoka chini hadi juu)**: Jedwali la matokeo hujenga suluhisho kutoka chini hadi juu, likijaza jedwali na suluhisho za matatizo madogo madogo.

```py
def climb_stairs_tabulation(n):
    """Dynamic programming with tabulation"""
    if n <= 2:
        return n
    
    # Create array to store results for all steps from 0 to n
    dp = [0] * (n + 1)
    dp[1] = 1  # 1 way to reach step 1
    dp[2] = 2  # 2 ways to reach step 2
    
    # Build up the solution iteratively
    for i in range(3, n + 1):
        # Ways to reach step i = ways to reach (i-1) + ways to reach (i-2)
        dp[i] = dp[i-1] + dp[i-2]
    
    return dp[n]
```
