# --interactive--

在 JavaScript 中，`object` 是一种基本的数据结构，允许你保存和组织相关的数据和功能性。 

你可以将一个对象视为一个容器，里面保存着各种信息，就像文件柜保存着不同的文件夹和文档一样。 

这些信息称为属性，它们由名称（或密钥）和值组成。

```js
const exampleObject = {
  propertyName: value,
}
```

对象极其多功能，是 JavaScript 的核心。事实上，JavaScript 中几乎所有东西都是对象或可以被视为对象。这包括数组、函数，甚至在某些用法中作为对象处理的原语数据类型，如字串和数字。 

JavaScript 以对象为中心的特性是它如此灵活且强大的原因之一。让我们来看一下你如何创建一个对象：

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};
```

在此示例中，我们创建了一个名为 `person` 的对象，该对象有三个属性：`name`、`age` 和 `city`。每个属性都有一个名称和值，二者由冒号分隔。

现在，让我们来探讨你如何访问这些属性。在 JavaScript 中访问对象属性有两种主要方式：点符号和方括弧符号。

点符号是访问对象属性最常见且最直接的方式。以下是点符号的基本语法：

```js
objectName.propertyName
```

下面是如何对我们的 `person` 对象使用点符号的示例：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

console.log(person.name);  // Alice
console.log(person.age);   // 30
```

:::

点符号简洁且易于阅读，当你知道要访问的属性的确切名称且该名称是有效的 JavaScript 标识符（意味着它不以数字开头且不包含特殊字符或空格）时，它是首选。

另一方面，方括弧表示法允许你使用方括弧内的字串来访问对象属性。以下是你如何使用方括弧表示法：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

console.log(person["name"]); // Alice
console.log(person["age"]); //  30
```

:::

方括弧表示法比点表示法更灵活，因为它允许你使用不是有效 JavaScript 标识符的属性名。例如，如果你的属性名包含空格或以数字开头，你就需要使用方括弧表示法：

:::interactive_editor

```js
const oddObject = {
  "1stProperty": "Hello",
  "property with spaces": "World"
};

console.log(oddObject["1stProperty"]);  // Hello
console.log(oddObject["property with spaces"]);  // World
```

:::

方括弧表示法的另一个优点是它允许你使用变量动态访问属性：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "Wonderland"
};

let propertyName = "city";
console.log(person[propertyName]); // Wonderland
```

:::

这种灵活性使得方括弧表示法特别有用，当你在编写代码时不知道确切的属性名称，或者当你处理来自用户输入或其他动态来源的属性名称时。

值得注意的是，JavaScript 中的对象非常强大且多功能。它们不仅可以包含像字串和数字这样的简单值，还可以包含数组或其他对象。

理解对象以及如何使用它们在 JavaScript 中至关重要，因为它们在整个语言以及许多 JavaScript 库和框架中被广泛使用。 

随着你继续学习和使用 JavaScript，你会发现掌握对象为创建复杂且强大的应用打开了无限可能。
