# --description--

`this`, `that`, `these`, `those` 같은 지시 대명사에 대해 이미 배웠습니다. 시간이나 공간에서의 근접성을 고려하여 특정 사물이나 사람을 가리킬 때 사용하세요.

소유격은 보통 아포스트로피와 `s`로 표시되며(`function's`처럼), 사람이나 사물의 소유나 속함을 나타낼 때 사용합니다. 이것은 어떤 것이 다른 것에 속하거나 일부임을 알려줍니다.

`purpose`라는 단어는 무언가가 행해지거나 사용되는 이유, 또는 존재하는 이유를 가리킵니다. 예를 들어 `The purpose of a clock is to tell time.`
