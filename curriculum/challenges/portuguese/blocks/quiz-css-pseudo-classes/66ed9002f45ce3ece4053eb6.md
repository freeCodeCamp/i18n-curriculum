## --quiz--

### --question--

#### --text--

O que são pseudo-classes?

#### --distractors--

São regras CSS que criam conteúdo adicional dentro de um elemento quando necessário.

---

São propriedades que ajustam o tamanho ou o layout de um elemento na página.

---

Eles são seletores usados para adicionar efeitos de movimento a um elemento durante interações.

#### --answer--

São palavras-chave adicionadas a um seletor que estilizam um elemento com base no seu estado.
