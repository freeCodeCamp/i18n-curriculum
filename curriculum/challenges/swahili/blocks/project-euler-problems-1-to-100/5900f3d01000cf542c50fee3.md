# --description--

Ikiwa sanduku lina diski ishirini na moja zenye rangi, zikiwa na diski kumi na tano za buluu na diski sita za nyekundu, na diski mbili zikichaguliwa kwa bahati nasibu, inaonekana kuwa uwezekano wa kuchukua diski mbili za buluu ni:

$${P(BB)} = \frac{15}{21}×\frac{14}{20} = \frac{1}{2}$$

Mpangilio unaofuata wa aina hii, ambao kuna uwezekano wa asilimia 50 kabisa wa kuchukua diski mbili za buluu kwa bahati nasibu, ni sanduku lenye diski themanini na tano za buluu na diski thelathini na tano za nyekundu.

Kwa kupata mpangilio wa kwanza wenye diski zaidi ya `limit` kwa jumla, amua idadi ya diski za buluu ambazo sanduku hilo lingekuwa nazo.

```js
function arrangedProbability(limit) {
  // Based on https://www.mathblog.dk/project-euler-100-blue-discs-two-blue/
  let blue = 15;
  let discs = 21;

  while (discs < limit) {
    const nextBlue = 3 * blue + 2 * discs - 2;
    const nextDiscs = 4 * blue + 3 * discs - 3;

    blue = nextBlue;
    discs = nextDiscs;
  }
  return blue;
}
```
