# --description--

Пройдіться циклом одночасно по кількох масивах і створіть новий масив, $i^{th}$ елемент якого є конкатенацією $i^{th}$ елементів кожного з заданих масивів.

Для цього прикладу, якщо вам дано такий масив масивів:

```js
[ ["a", "b", "c"], ["A", "B", "C"], [1, 2, 3] ]
```

вихідні дані мають бути:

```js
["aA1","bB2","cC3"]
```

# --instructions--

Напишіть функцію, яка приймає масив масивів як параметр і повертає масив рядків, що відповідає наведеному опису.

```js
function loopSimult(A) {
    var res = [], output;
    for (var i = 0; i < A[0].length; i += 1) {
        output = "";
        for (var j = 0; j < A.length; j++)
            output += A[j][i];
        res.push(output);
    }
    return res;
}
```
