# --description--

在 React 中，<dfn>受控输入</dfn> 是处理表单数据的标准方式，其中输入的值与组件的状态同步。 

```jsx
const MyForm = () => {
  const [name, setName] = useState("");

  return (
    <>
      <label htmlFor="name">Your name</label>
      <input type="text" value={name} id="name" />
    </>
  );
}
```

你将在未来的课程中学习更多关于这个模式的内容。

为了跟踪用户在搜索输入字段中输入的内容，创建一个名为 `query` 的状态变量，初始值为空字串（`''`）。同时使用 `useState` Hook 定义其对应的设置函数 `setQuery`。
