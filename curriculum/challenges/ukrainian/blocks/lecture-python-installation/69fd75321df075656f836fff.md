# --description--

У попередньому уроці ви дізналися, як запускати скрипти Python локально. Але бувають випадки, коли не хочеться створювати повноцінні програми на Python, а потрібно просто протестувати якийсь код. Саме для цього існує інтерактивна оболонка Python.

Але спершу давайте повторимо, що таке термінал.

Термінал — це текстовий інтерфейс, який дозволяє взаємодіяти з комп’ютером, вводячи команди. Кожна операційна система має свій застосунок терміналу за замовчуванням. На macOS можна використовувати застосунок Terminal. На Windows — Command Prompt або PowerShell. На Linux у кожного робочого середовища є свій застосунок терміналу за замовчуванням, наприклад GNOME Terminal або Konsole.

Відкрийте застосунок терміналу і введіть `python`, потім натисніть `Enter`. Це запустить інтерактивну оболонку Python. Інтерактивна оболонка — це програма, яка дозволяє вводити команди по одній і одразу бачити результати.

Коли ви починаєте нову сесію, спочатку можете побачити такий вивід:

```bash
Python 3.12.2 (main, Mar 21 2024, 22:48:26) [Clang 14.0.3 (clang-1403.0.22.14.1)] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>> 
```

Символ `>>>` означає, що Python чекає на введення команди. Спробуйте вивести `"Hello, world!"` у термінал:

```bash
print("Hello, world!")
```

Ви побачите текст, який з’явиться так:

```bash
>>> print("Hello, world!")
Hello, world!
```

Після виведення тексту Python повернеться до такого вигляду:

```bash
>>>
```

Це означає, що ви можете ввести ще одну команду.

Інтерпретатор Python працює за циклом, який називають Read, Evaluate, Print, Loop, або скорочено REPL. Коли ви вводите команди, інтерпретатор читає вхідні дані, аналізує їх, виводить результат і повертається до стану `>>>`, щоб ви могли вводити нові команди.

Що станеться, якщо спробувати ввести неправильну команду, наприклад таку?

```bash
>>> something random
```

Інтерпретатор Python все одно виконає той самий процес REPL. У цьому випадку ви отримаєте повідомлення про помилку:

```bash
>>> something random
  File "<stdin>", line 1
    something random
              ^^^^^^
SyntaxError: invalid syntax
```

Щоб вийти з інтерактивної оболонки, можна ввести `exit()` або натиснути `Ctrl + D` (на Mac/Linux) або `Ctrl + Z + Enter` (на Windows).

Використання інтерактивної оболонки Python чудово підходить для невеликих експериментів з кодом або базового дослідження. Під час проходження наступних уроків курсу ви можете вводити деякі нові методи, які вивчили, безпосередньо в інтерактивній оболонці.

Якщо ви плануєте працювати з довшими програмами або з кількома файлами, краще використовувати редактор коду або IDE.
