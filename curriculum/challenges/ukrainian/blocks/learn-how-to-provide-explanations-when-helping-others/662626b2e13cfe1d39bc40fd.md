# --description--

`To act up` означає поводитися неправильно або не функціонувати належним чином. Наприклад, якщо телефон постійно зависає, можна сказати `it's acting up`. Це як щось, що не працює так, як має.

`To open` у цьому контексті означає отримувати доступ або запускати щось, наприклад файл чи програму. Наприклад, `opening a document` означає отримання доступу для перегляду або редагування.
