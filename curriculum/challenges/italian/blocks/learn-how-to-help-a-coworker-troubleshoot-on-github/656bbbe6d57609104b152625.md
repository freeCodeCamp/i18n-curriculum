# --description--

L'istruzione di Bob usa il past continuous `I was checking` per descrivere un'azione che era in corso nel passato.

Puoi creare frasi al past continuous combinando il verbo to be al passato `was` o `were` + `verb + ing`.

Usa anche il verbo modale `should` per suggerire un corso d'azione.

# --questions--
