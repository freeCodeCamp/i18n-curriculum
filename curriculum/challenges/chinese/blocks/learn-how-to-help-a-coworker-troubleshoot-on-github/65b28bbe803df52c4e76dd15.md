# --description--

情态动词如 `might` 和 `can` 用于表达可能性和能力。 

`Might` 用于表示某事可能发生或为真，比如说，`It might rain today`。 

`Can` 用来表示某人做某事的能力，比如，`She can speak three languages`。让我们练习在句子中使用这些词。
