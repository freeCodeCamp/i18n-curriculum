# --description--

Quando você quer confirmar uma informação (para dizer que algo é verdade), você pode usar o verbo `do`. Isso torna sua resposta mais enfática. Por exemplo:

- Pergunta: `Do you like pizza?`

- Resposta: `Yes, I do.` - Esta resposta é mais enfática do que apenas `Yes.`

Você usa `do` para mostrar que você tem certeza.
