# --description--

Express.js, o semplicemente Express, è un framework web minimale e flessibile costruito **sopra** Node.js. Semplifica il processo di creazione di server web e API.

Invece di scrivere una montagna di codice boilerplate con il modulo nativo `http` di Node, Express ti permette di creare applicazioni web robuste con poche righe di codice.

Puoi pensarlo come lo strato veloce e non vincolante che rende Node.js più semplice e potente.

Ecco tre motivi chiave per cui gli sviluppatori amano Express:

1. **Minimale e veloce** – Express fornisce solo l’essenziale, dandoti il pieno controllo e permettendoti di aggiungere funzionalità quando serve tramite middleware e plugin.

2. **Ecosistema enorme** – Ha una vasta collezione di middleware e plugin per logging, autenticazione, sicurezza, sessioni e altro.

3. **Ottimo per le API** – Il suo modello semplice di routing e gestione delle richieste rende Express particolarmente adatto per costruire API RESTful.

Con Express puoi creare:

* App web

* API RESTful

* Backend per Single Page Application (SPA)

* Servizi basati su middleware

* Qualsiasi cosa server-side con HTTP


## Come installare Express

Per prima cosa, assicurati di avere Node.js e npm installati. Ecco come verificare se node e npm sono installati:

```bash
node -v
npm -v
```

Se vedi un numero di versione, significa che sono installati. Altrimenti, dovrai installarli. Puoi trovare le istruzioni di installazione in una lezione precedente o sul sito ufficiale di Node.js.

Poi, nel terminale, scrivi questi comandi:

```bash
mkdir my-app
cd my-app
npm init -y
npm install express
```

Imparerai di più su middleware e plugin nelle lezioni future, ma in breve, il middleware è codice che viene eseguito tra la ricezione di una richiesta e l’invio di una risposta. Viene comunemente usato per la gestione degli errori, la validazione degli input, la modifica dei dati nella richiesta o nella risposta e altro. Puoi concatenare middleware per costruire logiche flessibili e riutilizzabili per gestire le richieste.

I plugin sono tipicamente moduli Node.js esterni di terze parti che estendono le funzionalità di Express. Puoi installarli separatamente con npm, e la maggior parte è implementata come middleware che puoi usare nella tua app con poca configurazione. Invece di scrivere da zero funzionalità comuni, puoi usare un plugin per gestire cose come logging, autenticazione o buone pratiche di sicurezza.

Express è un framework popolare per lo sviluppo server-side in Node.js. Ti fornisce i mattoni per avviare rapidamente un server, gestire le rotte, amministrare il middleware e costruire API—tutto con codice pulito e leggibile.
