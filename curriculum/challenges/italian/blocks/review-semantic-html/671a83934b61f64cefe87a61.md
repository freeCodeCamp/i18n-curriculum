# --interactive--

## Importanza dell'HTML semantico

- **Gerarchia strutturale per gli elementi di intestazione**: è importante usare l'elemento di intestazione corretto per mantenere la gerarchia strutturale del contenuto. L'elemento `h1` è il livello più alto di intestazione e l'elemento `h6` è il livello più basso di intestazione.
- **Elementi HTML presentazionali**: elementi che definiscono l'aspetto del contenuto. Es. gli elementi deprecati `center`, `big` e `font`.
- **Elementi HTML semantici**: elementi che contengono significato e struttura. Es. `header`, `nav`, `figure`.

## Elementi HTML semantici

- **Elemento header**: usato per definire l'intestazione di un documento o di una sezione.

:::interactive_editor

  ```html
  <header>
    <h1>CatPhotoApp</h1>
    <p>Welcome to our cat gallery.</p>
  </header>
  ```

:::

- **Elemento main**: usato per contenere il contenuto principale della pagina web.

:::interactive_editor

  ```html
  <main>
    <section>
      <h2>Cat Photos</h2>
      <p>Browse adorable cat pictures.</p>
    </section>
  </main>
  ```

:::

- **Elemento section**: usato per suddividere il contenuto in sezioni più piccole.

:::interactive_editor

  ```html
  <section>
    <h2>About Me</h2>
    <p>Hi, I am Jane Doe and I am a web developer.</p>
  </section>
  ```

:::

- **Elemento di sezione di navigazione (`nav`)**: rappresenta una sezione con collegamenti di navigazione.

:::interactive_editor

  ```html
  <nav>
    <ul>
      <li><a href="#photos">Photos</a></li>
      <li><a href="#videos">Videos</a></li>
    </ul>
  </nav>
  ```

:::

- **Elemento figure**: usato per contenere illustrazioni e diagrammi.

:::interactive_editor

  ```html
  <figure>
    <img
      src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"
      alt="Two tabby kittens sleeping together on a couch."
    />
    <figcaption>Cats <strong>hate</strong> other cats.</figcaption>
  </figure>
  ```

:::

- **Elemento emphasis (`em`)**: indica un testo con enfasi di stress.

:::interactive_editor

  ```html
  <p>
    Never give up on <em>your</em> dreams.
  </p>
  ```

:::

- **Elemento idiomatic text (`i`)**: usato per evidenziare una voce o un modo alternativo, termini idiomatici da un'altra lingua, termini tecnici e pensieri.

:::interactive_editor

  ```html
  <p>
    There is a certain <i lang="fr">je ne sais quoi</i> in the air.
  </p>
  ```

:::

L'attributo `lang` all'interno del tag aperto `i` serve a specificare la lingua del contenuto. In questo caso, la lingua sarebbe il francese. L'elemento `i` non indica se il testo è importante o meno, mostra solo che è in qualche modo diverso dal testo circostante.

- **Elemento strong importance (`strong`)**: indica un testo con forte importanza.

:::interactive_editor

  ```html
  <p>
    <strong>Warning:</strong> This product may cause allergic reactions.
  </p>
  ```

:::

- **Elemento bring attention to (`b`)**: usato per attirare l'attenzione su un testo che non è importante per il significato del contenuto. È comunemente usato per evidenziare parole chiave nei riassunti o nomi di prodotti nelle recensioni.

:::interactive_editor

  ```html
  <p>
    We tested several products, including the <b>SuperSound 3000</b> for audio quality, the <b>QuickCharge Pro</b> for fast charging, and the <b>Ecoclean Vacuum</b> for cleaning. The first two performed well, but the <b>Ecoclean Vacuum</b> did not meet expectations.
  </p>
  ```

:::

- **Elemento description list (`dl`)**: usato per rappresentare una lista di gruppi termine-descrizione.
- **Elemento description term (`dt`)**: usato per rappresentare il termine definito.
- **Elemento description details (`dd`)**: usato per rappresentare la descrizione del termine.

:::interactive_editor

  ```html
  <dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
  </dl>
  ```

:::

- **Elemento block quotation (`blockquote`)**: usato per rappresentare una sezione citata da un'altra fonte. Questo elemento ha un attributo `cite`. Il valore dell'attributo `cite` è l'URL della fonte.

:::interactive_editor

  ```html
  <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
    "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
  </blockquote>
  ```

:::

- **Elemento citation (`cite`)**: usato per attribuire visivamente la fonte del lavoro citato. Evidenzia il titolo del riferimento.

:::interactive_editor

  ```html
  <div>
    <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
      "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
    </blockquote>
    <p>
      -Quincy Larson, <cite>How to learn to Code and Get a Developer Job [Full Book].</cite>
    </p>
  </div>
  ```

:::

- **Elemento inline quotation (`q`)**: usato per rappresentare una breve citazione in linea.

:::interactive_editor

  ```html
  <p>
    As Quincy Larson said,
    <q cite="https://www.freecodecamp.org/news/learn-to-code-book/">
      Momentum is everything.
    </q>
  </p>
  ```

:::

- **Elemento abbreviation (`abbr`)**: usato per rappresentare un'abbreviazione o acronimo. Per aiutare gli utenti a capire cosa significa l'abbreviazione o acronimo, puoi mostrare la sua forma completa, una descrizione leggibile, con l'attributo `title`.

:::interactive_editor

  ```html
  <p>
    <abbr title="HyperText Markup Language">HTML</abbr> is the foundation of the web.
  </p>
  ```

:::

- **Elemento contact address (`address`)**: usato per rappresentare le informazioni di contatto.
- **Elemento (date) time (`time`)**: usato per rappresentare una data e/o un orario. L'attributo `datetime` serve a tradurre date e orari in un formato leggibile dalla macchina.

:::interactive_editor

  ```html
  <p>
    The reservations are for the <time datetime="20:00">20:00 </time>
  </p>
  ```

:::

- **Attributo ISO 8601 date (`datetime`)**: usato per rappresentare date e orari in un formato leggibile dalla macchina. Il formato standard è `YYYY-MM-DDThh:mm:ss`, con la lettera maiuscola `T` che separa la data dall'orario.
- **Elemento superscript (`sup`)**: usato per rappresentare testo in apice. Gli usi comuni per l'elemento `sup` includono esponenti, lettere superiori e numeri ordinali.

:::interactive_editor

  ```html
  <p>
    2<sup>2</sup> (2 squared) is 4.
  </p>
  ```

:::

- **Elemento subscript (`sub`)**: usato per rappresentare testo in pedice. Gli usi comuni per l'elemento subscript includono formule chimiche, note a piè di pagina e pedici di variabili.

:::interactive_editor

  ```html
  <p>
    CO<sub>2</sub>
  </p>
  ```

:::

- **Elemento inline code (`code`)**: usato per rappresentare un frammento di codice informatico.
- **Elemento preformatted text (`pre`)**: rappresenta testo preformattato

:::interactive_editor

  ```html
  <pre>
    <code>
      body {
        color: red;
      }
    </code>
  </pre>
  ```

:::

- **Elemento unarticulated annotation (`u`)**: usato per rappresentare una porzione di testo in linea che dovrebbe essere renderizzata in modo da indicare che ha un'annotazione non testuale.

:::interactive_editor

  ```html
  <p>
    You can use the unarticulated annotation element to highlight
    <u>inccccort</u> <u>spling</u> <u>issses</u>.
  </p>
  ```

:::

- **Elemento ruby annotation (`ruby`)**: usato per annotare il testo con pronunce o spiegazioni di significato. Un esempio di utilizzo è per la tipografia dell'Asia orientale.
- **Elemento ruby fallback parenthesis (`rp`)**: usato come fallback per i browser che non supportano la visualizzazione delle annotazioni ruby.
- **Elemento ruby text (`rt`)**: usato per indicare il testo per l'annotazione ruby. Di solito usato per la pronuncia o dettagli di traduzione nella tipografia dell'Asia orientale.

:::interactive_editor

  ```html
  <ruby>
    明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp>
  </ruby>
  ```

:::

- **Elemento strikethrough (`s`)**: usato per rappresentare contenuti che non sono più accurati o rilevanti.

:::interactive_editor

  ```html
  <p>
    <s>Tomorrow's hike will be meeting at noon.</s>
  </p>
  <p>
    Due to unforeseen weather conditions, the hike has been canceled.
  </p>
  ```

:::

- **Collegamenti interni**: usati per collegare a un'altra sezione della pagina usando `href="#id"` su un elemento `a` e assegnando all'elemento di destinazione lo stesso `id`. Questo è comunemente usato per collegamenti di salto, sommari o pagine lunghe con più sezioni.

:::interactive_editor

```html
<a href="#about-section">Go to About Section</a>

<section id="about-section">
  <h2>About</h2>
  <p>This is the about section of the page.</p>
</section>
```

:::
