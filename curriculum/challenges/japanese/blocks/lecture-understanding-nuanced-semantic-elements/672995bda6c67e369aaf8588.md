# --interactive--

説明リストは、用語とその定義を整理されて読みやすい形式で提示するのに最適です。例えば、用語集や辞書のように、単語とそれに対応する定義を見つけることができます。

これは、HTMLで作成された説明リストの例で、2つの用語とそれに対応する詳細が示されています。

インタラクティブエディターを有効にして、コードのコメントを外し、新しい詳細項目がプレビューウィンドウに表示されるのを確認してください。

:::interactive_editor

```html
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
  <!-- <dt>JS</dt>
  <dd>JavaScript</dd> -->
</dl>
```

:::

この場合、用語はHTMLとCSSという略語で、詳細はそれらの展開形です。詳細は定義や用語に関連する他の情報でもかまいません。

説明リストを定義するには3つのHTML要素が必要です。まず、説明リスト要素である`dl`はリスト全体のコンテナです。例では説明リストの他のすべての要素を囲んでいるのがわかります。

次に、用語ごとに1つの説明用語要素`dt`があります。この例では説明リストにHTMLとCSSの2つの用語があるので、この要素も2つあります。

最後に、各用語の後に説明詳細要素`dd`があり、その用語に関連する説明や詳細が入ります。この例ではHypertext Markup LanguageとCascading Style Sheetsです。

ブラウザーでは、各用語の後に対応する説明が続いて表示されます。デフォルトでは、説明は視覚的に区別しやすいように少し右にインデントされています。

しかし説明リストは用語と定義だけに限りません。もっと多用途に使えます。ここでは2つの材料があるレシピの例です。

インタラクティブエディターを有効にして、コードのコメントを外し、新しい詳細項目がプレビューウィンドウに表示されるのを確認してください。

:::interactive_editor

```html
<dl>
  <dt>Flour</dt>
  <dd>2 cups</dd>
  <dt>Sugar</dt>
  <dd>1/2 cup</dd>
  <!-- <dt>Vegetable Oil</dt>
  <dd>2 tablespoons</dd> -->
</dl>
```

:::

説明リスト全体は説明リスト要素内にあります。最初の材料`Flour`は説明用語要素内にあります。次に、その材料の必要量`2 cups`が説明詳細要素内にあり、対応する材料のすぐ後に配置されています。

同じ構造が`Sugar`についても繰り返されています。この例では材料は2つだけですが、もっと多くても同じ構造を説明リスト全体で繰り返せます。

ブラウザーでは材料は左揃えで表示され、分量は視覚的に区別できるようにインデントされます。

説明リストの他の使い方には、製品仕様、よくある質問、連絡先情報、メタデータなどがあります。基本的に、2つの関連情報がキーと値のペア形式であり、一方がラベル（キー）として、もう一方が追加の関連情報（値）として機能する場合に説明リストを使えます。
