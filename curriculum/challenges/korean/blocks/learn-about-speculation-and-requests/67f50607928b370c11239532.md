# --explanation--

`Evidence of`는 무언가가 발생했다는 징후나 증거를 의미합니다. 예를 들어:

`The report showed evidence of user error.` - 보고서는 누군가 실수를 했다는 것을 보여주었습니다.

`brute force attack`는 누군가 여러 비밀번호를 시도하다가 맞는 것을 찾는 경우입니다.

`SQL injection`는 누군가 검색 상자나 양식에 악성 코드를 넣어 시스템을 공격하려는 시도입니다.

Jake는 이러한 유형의 공격 징후를 확인하는 것에 대해 이야기하고 있습니다.
