# --description--

배열과 함께 사용되는 더 복잡한 메서드 중 하나는 `splice()`와 `slice()` 메서드입니다. `splice()` 메서드는 배열의 요소를 제거하거나 교체하여 배열의 내용을 변경합니다. `slice()` 메서드는 배열의 일부를 선택해 `begin`부터 `end`까지(`end` 제외) 새로운 배열 객체로 얕은 복사본을 반환합니다. 원래 배열은 변경되지 않습니다.

예를 들어, `characters` 배열에서 두 번째 요소를 제거하려면 다음 코드를 사용할 수 있습니다.

```javascript
const characters = ['Harry', 'Ron', 'Hermione'];
characters.splice(1, 1);
console.log(characters); // Output: ['Harry', 'Hermione']
```

위 코드는 `characters` 배열에서 두 번째 요소를 제거합니다. `splice()` 메서드는 두 개의 인수를 받습니다: 제거할 요소의 인덱스와 제거할 요소의 개수입니다.


`character` 배열에서 두 번째 요소를 포함하는 새 배열을 만들려면 다음 코드를 사용할 수 있습니다.

```javascript
const characters = ['Harry', 'Ron', 'Hermione'];
const newCharacters = characters.slice(1, 2);
console.log(newCharacters); // Output: ['Ron']
```

위 코드는 `newCharacters` 배열에서 두 번째 요소를 포함하는 새 배열 `characters`을 만듭니다. `slice()` 메서드는 두 개의 인수를 받습니다: 슬라이스를 시작할 요소의 인덱스와 슬라이스를 끝낼 요소의 인덱스(포함하지 않음)입니다.
