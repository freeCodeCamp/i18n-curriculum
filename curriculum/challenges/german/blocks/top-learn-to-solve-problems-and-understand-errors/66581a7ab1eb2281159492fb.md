## Verstehen Sie das Problem
Der erste Schritt zur Lösung eines Problems ist, genau zu verstehen, worum es bei dem Problem geht. Wenn Sie das Problem nicht verstehen, wissen Sie nicht, wann Sie es erfolgreich gelöst haben, und könnten viel Zeit mit einer falschen Lösung verschwenden.

Um Klarheit und Verständnis für das Problem zu gewinnen, schreiben Sie es auf Papier, formulieren Sie es in einfachem Deutsch um, bis es für Sie Sinn ergibt, und zeichnen Sie Diagramme, wenn das hilft. Wenn Sie das Problem jemand anderem in einfachem Deutsch erklären können, haben Sie es verstanden.

### Planen
Jetzt, wo Sie wissen, was Sie lösen wollen, stürzen Sie sich noch nicht sofort ins Programmieren. Es ist Zeit, zuerst zu planen, wie Sie das Problem lösen werden. Einige der Fragen, die Sie in diesem Schritt beantworten sollten:

- Hat Ihr Programm eine Benutzeroberfläche? Wie wird sie aussehen? Welche Funktionen wird die Oberfläche haben? Skizzieren Sie dies auf Papier.

- Welche Eingaben wird Ihr Programm haben? Gibt der Benutzer Daten ein oder erhalten Sie Eingaben von einer anderen Quelle?

- Was ist die gewünschte Ausgabe?

- Welche Schritte sind notwendig, um basierend auf Ihren Eingaben die gewünschte Ausgabe zu erzeugen?

Die letzte Frage ist der Punkt, an dem Sie einen Algorithmus zur Lösung des Problems aufschreiben. Sie können sich einen Algorithmus als ein Rezept vorstellen, um ein bestimmtes Problem zu lösen. Er definiert die Schritte, die der Computer in Pseudocode ausführen muss, um das Problem zu lösen.
