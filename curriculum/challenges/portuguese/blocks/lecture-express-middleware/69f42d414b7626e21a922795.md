# --description--

O Express oferece uma variedade de funções de middleware embutidas que ajudam a realizar tarefas comuns, como servir arquivos estáticos, analisar corpos de requisições JSON e muito mais. Você também pode usar middleware de terceiros para adicionar funcionalidades.

**Aqui estão alguns exemplos de middleware embutido**:

1. `express.json()`: Um middleware embutido que analisa payloads JSON recebidos.

```js
app.use(express.json())
```

1. `express.static()`: Um middleware para servir arquivos estáticos como imagens, CSS e arquivos JavaScript.

```js
app.use(express.static('public'))
```

1. `express.urlencoded()`: Um middleware que analisa requisições recebidas com payloads codificados em URL, o formato normalmente enviado por formulários HTML. Ele disponibiliza os dados analisados em `req.body`.

```js
app.use(express.urlencoded({ extended: true }))
```

**Agora, aqui estão alguns exemplos de middleware de terceiros:**

1. `cors`: Um middleware de terceiros popular para habilitar o compartilhamento de recursos entre origens diferentes (Cross-Origin Resource Sharing).

```bash
npm install cors
```

```js
const cors = require('cors')
app.use(cors())
```

1. `morgan`: Um middleware de registro para requisições HTTP.

```bash
npm install morgan
```

```js
const morgan = require('morgan')
app.use(morgan('tiny'))
```

Agora, vamos ver um exemplo de uso tanto de middleware embutido quanto de terceiros:

```js
const express = require('express')
const cors = require('cors')
const morgan = require('morgan')
const app = express()

app.use(express.json()) // Built-in middleware to parse JSON
app.use(cors())       // Third-party middleware for CORS
app.use(morgan('tiny')) // Third-party middleware for logging

app.get('/', (req, res) => {
  res.send('Hello, World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* O middleware embutido `express.json()` analisa payloads JSON recebidos.
* O middleware de terceiros `cors` permite requisições entre origens diferentes, e `morgan` registra as requisições HTTP recebidas.

É importante notar que as funções de middleware são adicionadas na ordem em que são chamadas com `app.use()`.

O Express fornece middleware embutido essencial para lidar com tarefas comuns como análise de JSON e servir arquivos estáticos. Você também pode estender a funcionalidade com middleware de terceiros para capacidades adicionais.
