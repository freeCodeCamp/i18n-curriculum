# --interactive--

절대 위치 지정은 요소를 일반 문서 흐름에서 분리하여 다른 요소와 독립적으로 동작하게 합니다. 요소가 절대 위치 지정되면, 레이아웃 내 다른 모든 것과 완전히 분리된 자체 레이어에 배치됩니다. 이 기능은 모달, 툴팁, 드롭다운 메뉴 같은 떠 있는 UI 기능을 만들 때 유용하며, 페이지의 다른 요소 위에 겹쳐서 표시할 수 있습니다.

기본적으로 절대 위치 지정 요소는 가장 가까운 위치 지정 조상을 기준으로 배치됩니다. 위치 지정 조상이 없으면, 요소는 보통 브라우저 뷰포트를 의미하는 초기 포함 블록을 기준으로 위치가 정해집니다. `top`, `bottom`, `left`, `right` 속성을 사용해 포함 요소의 가장자리에서 얼마나 떨어져 있어야 하는지 지정하여 절대 위치 지정 요소를 이동할 수 있습니다.

예를 들어, `top: 30px`와 `left: 30px`를 설정하면 요소가 포함 블록의 위쪽과 왼쪽 가장자리에서 `30px`만큼 떨어져 이동합니다. 다음은 CSS에서 절대 위치 지정을 요소에 적용하는 예시입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="positioned">Absolutely Positioned</div>
```

```css
body {
  background-color: #eeeeee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

이 코드를 적용하면 요소가 일반 문서 흐름에서 제거되고 포함 블록의 위쪽과 왼쪽에서 `30px`만큼 떨어진 위치에 배치됩니다.

절대 위치 지정은 오버레이나 모달 창처럼 나머지 콘텐츠 위에 떠 있어야 하는 요소를 만들 때 특히 유용합니다. 하지만 요소가 문서 흐름에서 제거되므로 적절히 처리하지 않으면 빈 공간이 생기거나 다른 요소들이 서로 겹쳐질 수 있습니다.
