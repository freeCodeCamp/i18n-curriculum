# --description--

Ora vedremo come comporre insieme piÃ¹ componenti React. Immagina di costruire un'app e di aver creato tre componenti: un `Navbar`, `Dashboard` e `Footer`.

Per comporre questi componenti insieme, potresti creare un componente *genitore* `App` che renderizza ciascuno di questi tre componenti come *figli*. Per renderizzare un componente come figlio in un componente React, includi il nome del componente scritto come un tag HTML personalizzato nel JSX. Per esempio, nel metodo `render` potresti scrivere:

```jsx
return (
 <App>
  <Navbar />
  <Dashboard />
  <Footer />
 </App>
)
```

Quando React incontra un tag HTML personalizzato che fa riferimento a un altro componente (un nome di componente racchiuso in `< />` come in questo esempio), renderizza il markup per quel componente nella posizione del tag. Questo illustra la relazione genitore/figlio tra il componente `App` e i componenti `Navbar`, `Dashboard` e `Footer`.
