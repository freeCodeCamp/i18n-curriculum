# --interactive--

توفر JavaScript أدوات لأداء العمليات الحسابية الأساسية على الأعداد، مثل الجمع والطرح والضرب والقسمة. كما تتضمن JavaScript عوامل لعمليات حسابية أكثر تعقيدًا، مثل الباقي والأس.

تُسمى كل هذه الأدوات عوامل العمليات الحسابية. دعنا نلقي نظرة على هذه العوامل بالتفصيل، وكيفية استخدامها وكيفية دمجها.

عامل الجمع هو علامة زائد (`+`). يسمح عامل الجمع بحساب مجموع رقمين أو أكثر. في عمليات الجمع، لا يهم ترتيب الأرقام:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 + num2;
const result2 = num2 + num1;
const result3 = num2 + num1 + num3;

console.log(result1); // 15
console.log(result2); // 15
console.log(result3); // 30
```

:::

عامل الطرح هو علامة ناقص (`-`). يسمح لك بحساب الفرق بين رقمين. استخدم علامة الناقص عندما تريد طرح رقم من رقم آخر، عادةً رقم أصغر من رقم أكبر:

:::interactive_editor

```js
const difference = 10 - 5;
console.log(difference); // 5
```

:::

إذا جاء الرقم الأصغر أولًا، ستحصل على نتيجة سالبة:

:::interactive_editor

```js
const difference = 5 - 10;
console.log(difference); // -5
```

:::

يمكنك أيضًا تعيين الأرقام إلى متغيرات وإجراء الطرح باستخدام أسماء المتغيرات:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const result = num1 - num2;

console.log(result); // 5
```

:::

في JavaScript، يُمثل عامل الضرب بنجمة (`*`) ويُستخدم لإيجاد حاصل ضرب رقمين أو أكثر. لا يهم ترتيب الأرقام التي تضربها:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 * num2;
const result2 = num2 * num1;
const result3 = num2 * num1 * num3;

console.log(result1); // 50
console.log(result2); // 50
console.log(result3); // 750
```

:::

في JavaScript، عامل القسمة هو شرطة مائلة (`/`)، وهو يختلف عن رمز القسمة المستخدم في الرياضيات التقليدية (`÷`). تُجرى عمليات القسمة باستخدام عامل القسمة. في هذه الحالة، يهم ترتيب الأرقام التي تقسمها:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 / num2;
const result2 = num2 / num1;
const result3 = num2 / num1 / num3;

console.log(result1); // 2
console.log(result2); // 0.5
console.log(result3); // 0.03333333333333333
```

:::

من المهم ملاحظة أنه إذا حاولت القسمة على صفر، سترجع JavaScript القيمة `Infinity`:

:::interactive_editor

```js
const result = 10 / 0; 

console.log(result); // Infinity
```

:::

تأكد من تجنب هذه الأنواع من العمليات حتى لا تحصل على نتائج غير متوقعة في شفرتك.

عامل الباقي، الممثل بعلامة النسبة المئوية (`%`)، يرجع الباقي من عملية القسمة. الباقي في الرياضيات هو القيمة المتبقية بعد إجراء القسمة:

:::interactive_editor

```js
const num1 = 10;
const num2 = 3;
const remainder = num1 % num2;

console.log(remainder); // 1
```

:::

عامل الأس، الممثل بنجمتين مزدوجتين (`**`)، يرفع رقمًا إلى قوة رقم آخر:

:::interactive_editor

```js
const num1 = 2;
const num2 = 3;

const exponent = num1 ** num2;
console.log(exponent); // 8
```

:::

من الممكن دمج عوامل مختلفة في عملية أو تعبير واحد:

:::interactive_editor

```js
const result = 10 + 5 * 2 - 8 / 4;
console.log(result); // 18
```

:::

عندما تدمج عوامل مختلفة في تعبير واحد، يتبع محرك JavaScript نظامًا يسمى أسبقية العوامل لتحديد ترتيب العمليات. تحدد أسبقية العوامل ترتيب تنفيذ العمليات في التعبيرات. ستتعلم المزيد عن أسبقية العوامل في الدروس القادمة.
