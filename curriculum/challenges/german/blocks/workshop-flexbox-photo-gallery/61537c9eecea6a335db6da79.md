# --description--

Unter Ihrem `.header`-Element erstellen Sie ein neues `div`-Element und weisen ihm eine `class` von `gallery` zu. Dieses `div` fungiert als Container für die Galerie-Bilder.

Innerhalb dieses `.gallery`-Elements erstellen Sie neun `img`-Elemente.

```js
assert.isAtLeast(document.querySelector('body')?.querySelectorAll('div')?.length, 1);
```

