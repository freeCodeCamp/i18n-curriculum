# --explanation--

Ángela diz `¿Navarro con dos erres?` Esta é uma pergunta de confirmação.

`Con` introduz informações extras sobre a ortografia.

`Dos` é o número dois.

`Erres` é a forma plural de `erre`, o nome da letra `r`.

Ángela está confirmando que o sobrenome Navarro é escrito com duas letras `r`.
