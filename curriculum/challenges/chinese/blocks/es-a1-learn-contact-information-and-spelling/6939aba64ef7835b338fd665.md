# --description--

你还记得动词是根据它们所参考的人称进行变位的吗？如果不记得，现在你记得了！

几 个 任务 之前，你 学会 了 如何 使用 动词 `poder` 来 询问 某人 是否 能够 为 你 做 某事。

现在你将学习如何变位这个相同的动词来询问你是否能做某事。你只需要使用以下结构：

`Puedo` + 不定式动词 + `...?`

让我们来看一些例子：

- `¿Puedo usar la computadora?` - 我可以使用计算机吗？

- `¿Puedo usar mi apodo?` - 我可以使用我的昵称吗？

在这两个示例中，你分别是在请求使用计算机和你的昵称的权限。

# --explanation--

Basti 说 `¿Puedo usar mi apodo?`。

这里，`puedo` 是动词 `poder` 的第一人称单数形式，用于当某人请求许可时。

结构遵循该模式：

`¿Puedo` + 不定式动词 + `...?`

在这种分支中：

- `Usar` 意思是使用。

- `Mi apodo` 参考他的昵称。

所以 Basti 在问他是否被允许使用他的昵称。
