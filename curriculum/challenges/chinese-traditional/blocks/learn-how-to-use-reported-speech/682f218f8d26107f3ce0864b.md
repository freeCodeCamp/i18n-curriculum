# --explanation--

`Of course` 是一種禮貌的方式，用來表示同意或毫不猶豫地說 `yes`。這表示說話者很樂意或願意做某事。 例如：

`Of course, I'll help you with the report.`－這表示說話者很樂意幫助你。

`Will` 經常用來做出承諾或表達當下的決定。 例如：

`I'll email you the file this afternoon.`－這表示很快會做某事的承諾。
