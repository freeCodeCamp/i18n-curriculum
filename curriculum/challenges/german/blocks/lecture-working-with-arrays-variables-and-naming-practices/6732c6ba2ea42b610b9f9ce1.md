# --description--

In früheren Lektionen wurden Sie bereits mit der `length`-Eigenschaft vertraut gemacht, diese Eigenschaft gibt die Anzahl der Elemente in einem Array zurück. Hier eine kurze Erinnerung, wie sie funktioniert:

```js
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits.length); // 3
```

Es ist möglich, Arrays mit leeren Slots zu haben. Leere Slots sind Slots, in denen sich nichts befindet. Das unterscheidet sich von einem Array mit dem Wert `undefined`. Diese Arten von Arrays werden als Sparse-Arrays bezeichnet. Hier ein Beispiel:

```js
const sparseArray = [1, , , 4];
console.log(sparseArray.length); // 4
```

In diesem Fall haben wir zwar nur zwei definierte Elemente, `1` und `4`, aber die Länge ist `4`, weil der höchste Index (`3`) plus `1` uns eine Länge von `4` ergibt.

Kommen wir nun dazu, wie man ein leeres Array mit fester Länge erstellt. Es gibt mehrere Möglichkeiten, dies in JavaScript zu tun, aber eine gängige Methode ist die Verwendung des `Array()`-Konstruktors mit einem numerischen Argument. Der `Array()`-Konstruktor kann mit dem Schlüsselwort `new` verwendet werden, um ein neues Array zu erstellen. Hier ein Beispiel:

```js
const emptyArray = new Array(5);
console.log(emptyArray.length); // 5
console.log(emptyArray); // [ , , , , ]
```

In diesem Beispiel erstellen wir ein neues Array mit `Array(5)`. Dies erzeugt ein Sparse-Array mit einer Länge von `5`, bei dem alle Slots leer sind.

Eine weitere Möglichkeit, ein leeres Array mit fester Länge zu erstellen, ist die Verwendung der `Array.from()`-Methode mit einem Längenargument. Im Gegensatz zu `new Array(n)` erzeugt diese Methode ein Array der angegebenen Länge, bei dem alle Elemente existieren und den Wert `undefined` haben:

```js
const fixedLengthArray = Array.from({ length: 5 });
console.log(fixedLengthArray.length); // 5
console.log(fixedLengthArray); // [undefined, undefined, undefined, undefined, undefined]
```

Wenn Sie ein Array mit einer bestimmten Länge erstellen und es mit einem Standardwert füllen möchten, können Sie die `Array.fill()`-Methode verwenden:

```js
const filledArray = new Array(3).fill(0);
console.log(filledArray); // [0, 0, 0]
```

Dies erzeugt ein Array der Länge drei und füllt alle Elemente mit dem Wert `0`. Hinweis: Wenn Sie mit Objekten füllen, verweisen alle Slots auf dasselbe Objekt; wenn Sie unabhängige Kopien benötigen, verwenden Sie stattdessen einen Callback oder `Array.from()`.

Zu verstehen, wie man die Länge eines Arrays ermittelt und Arrays mit fester Länge erstellt, ist für viele Programmieraufgaben wichtig, insbesondere wenn Sie Arrays für bestimmte Algorithmen oder Datenstrukturen initialisieren müssen.
