# --description--

La frase `to keep in mind` significa recordar o considerar algo. A menudo se usa cuando alguien da un consejo o información importante que debe ser recordada para uso futuro.

Por ejemplo, si alguien dice, "Recuerda revisar el clima antes de hacer senderismo." Podrías responder con `I'll keep that in mind` lo que significa que recordarás considerar este consejo.
