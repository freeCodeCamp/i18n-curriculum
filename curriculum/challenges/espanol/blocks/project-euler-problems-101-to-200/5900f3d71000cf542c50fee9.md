# --description--

Sea $S(A)$ la suma de elementos en el conjunto A del tamaño n. Lo llamaremos un conjunto de suma especial si para dos subconjuntos disjuntos no vacíos cualesquiera, B y C, las siguientes propiedades son verdaderas:

1. $S(B) ≠ S(C)$; es decir, las sumas de subconjuntos no pueden ser iguales.
2. Si B contiene más elementos que C entonces $S(B) > S(C)$.

Para este problema asumiremos que un conjunto dado contiene n elementos estrictamente crecientes y que ya cumple la segunda regla.

Sorprendentemente, de los 25 pares de subconjuntos posibles que se pueden obtener de un conjunto para el cual n = 4, solo 1 de estos pares necesita ser probado para igualdad (primera regla). De manera similar, cuando n = 7, solo 70 de los 966 pares de subconjuntos necesitan ser probados.

Para n = 12, ¿cuántos de los 261625 pares de subconjuntos que se pueden obtener necesitan ser probados para igualdad?

**Nota:** Este problema está relacionado con el problema 103 y el problema 105.
