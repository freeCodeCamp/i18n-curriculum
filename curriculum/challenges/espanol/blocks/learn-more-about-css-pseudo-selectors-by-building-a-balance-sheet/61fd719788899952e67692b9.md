# --description--

Los elementos `thead` y `tbody` se utilizan para indicar qué parte de su tabla es el encabezado, y qué parte contiene los datos o contenido principal.

Agregar un `thead` y un `tbody` a su primer `table`, por debajo del elemento `caption`.

# --hints--

El primer elemento `table` debe tener un elemento `thead`.

```js
assert(document.querySelectorAll('table')?.[0]?.querySelector('thead'));
```

El primer elemento `table` debe tener un elemento `tbody`.

```js
assert(document.querySelectorAll('table')?.[0]?.querySelector('tbody'));
```

El elemento `thead` debe estar inmediatamente debajo del elemento `caption`.

```js
assert(document.querySelector('caption')?.nextElementSibling?.localName === 'thead');
```

El elemento `tbody` debe estar inmediatamente debajo del elemento `thead`.

```js
assert(document.querySelector('thead')?.nextElementSibling?.localName === 'tbody');
```
