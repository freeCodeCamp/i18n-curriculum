# --explanation--

句動詞は、動詞と前置詞や副詞が組み合わさって新しい意味を作るものです。この会話の部分では、`to send something` と `to send something over` の違いがわかります。

`To send something` は何かをある場所や人から別の場所や人へ移動させることを意味しますが、`to send something over` は人や場所への配達の意味を加え、しばしば非公式またはデジタルな形で使われます。以下にそれぞれの例を示します：

- `I'll send the letter tomorrow.` - 何かを移動させる基本的な表現。

- `Can you send over the file by email?` - 何かを*誰かに*送ることを強調し、通常は短距離やデジタルでの送信を指します。`over` の追加で、カジュアルな会話でより自然に聞こえます。
