# --description--

`Make sure`는 무언가가 올바르게 수행되었거나 조건이 충족되었는지 확인하거나 검증하는 것을 의미합니다. 정확성이나 완료를 보장하기 위해 필요한 단계를 밟는 것을 포함합니다. 예를 들어, `Please make sure you turn off the lights before you leave.`

`Follow instructions`는 작업을 완료하기 위해 제공된 지침이나 가이드라인을 따르는 것을 의미합니다. 지정된 순서와 방식대로 각 단계를 수행하는 것을 포함합니다. 예시는 `follow the instructions on the box to assemble the table`입니다.
