# --explanation--

Às vezes, quando você não tem certeza completa sobre algo, mas acredita que deve ser verdade, você pode usar a frase `It's supposed to` + verbo. 

Isso expressará que algo é esperado ser de uma certa forma, mesmo que você ainda não tenha confirmado.Por exemplo:

- `The update is supposed to fix the bug.` - Isso significa que você espera que a atualização corrija o bug, mas ainda não confirmou isso.
