# --interactive--

## أساسيات CSS

- **ما هو CSS؟**: أوراق الأنماط المتتالية (CSS) هي لغة أنماط تُستخدم لتطبيق الأنماط على عناصر HTML. تُستخدم CSS للألوان، وصور الخلفية، والتخطيطات، والمزيد.
- **التركيب الأساسي لقاعدة CSS**: تتكون قاعدة CSS من جزأين رئيسيين: محدِّد وكتلة إعلان. المحدِّد هو نمط يُستخدم في CSS لتحديد واستهداف عناصر HTML معينة لتطبيق الأنماط عليها. كتلة الإعلان تطبق مجموعة من الأنماط لمحدد أو محددات معينة.

إليك الصيغة النحوية العامة لقاعدة CSS:

```css
selector {
    property: value;
}
```

- **عنصر `meta name="viewport"`**: هذا العنصر `meta` يعطي المتصفح تعليمات حول كيفية التحكم في أبعاد الصفحة وتكبيرها على أجهزة مختلفة، خصوصًا الهواتف المحمولة والأجهزة اللوحية.
- **أنماط المتصفح الافتراضية**: كل عنصر HTML له أنماط متصفح افتراضية تُطبق عليه. عادةً تشمل هذه العناصر هوامش وحشوات افتراضية.

## CSS المضمَّن، الداخلي، والخارجي

- **CSS المضمَّن**: تُكتب هذه الأنماط مباشرة داخل عنصر HTML باستخدام الخاصية `style`. في معظم الأحيان لن تستخدم CSS المضمَّن بسبب مبدأ فصل الاهتمامات.

إليك مثالًا على CSS المضمَّن:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p style="color: red;">This is a red paragraph.</p>
```

:::

- **CSS الداخلي**: تُكتب هذه الأنماط داخل وسم `<style>` في قسم `head` من وثيقة HTML. قد يكون هذا مفيدًا لإنشاء أمثلة شفرة قصيرة، لكن عادةً لن تحتاج لاستخدام CSS الداخلي.
- **CSS الخارجي**: تُكتب هذه الأنماط في ملف CSS منفصل ويرتبط بملف HTML باستخدام عنصر `link` في قسم `head`. في معظم المشاريع، ستستخدم ملف CSS خارجي بدلًا من الداخلي أو المضمَّن.

## العمل مع خاصيتي `width` و`height`

- **خاصية `width`**: تحدد هذه الخاصية عرض العنصر. إذا لم تحدد عرضًا، يكون الافتراضي `auto`، مما يسمح للمتصفح بتحديد عرض العنصر بناءً على محتواه، والعنصر الأب، ونوع العرض.
- **خاصية `min-width`**: تحدد الحد الأدنى لعرض العنصر.
- **خاصية `max-width`**: تحدد الحد الأقصى لعرض العنصر.
- **خاصية `height`**: تحدد ارتفاع العنصر. بالمثل، يكون الارتفاع `auto` افتراضيًا، مما يعني أنه يتكيف مع المحتوى الداخلي.
- **خاصية `min-height`**: تحدد الحد الأدنى لارتفاع العنصر.
- **خاصية `max-height`**: تحدد الحد الأقصى لارتفاع العنصر.

## أنواع مختلفة من مركبات CSS

- **مركب العنصر التابع**: يُستخدم هذا المركب لاستهداف العناصر التابعة لعنصر أب محدد. المثال التالي يستهدف كل عناصر `li` داخل عناصر `ul`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul>
    <li>Example item one</li>
    <li>Example item two</li>
    <li>Example item three</li>
</ul>
```

```css
ul li {
    background-color: yellow;
}
```

:::

- **مركب الطفل (`>`)**: يُستخدم هذا المركب لاختيار العناصر التي هي أبناء مباشرين لعنصر أب محدد. المثال التالي يستهدف كل عناصر `p` التي هي أبناء مباشرة لفئة `container`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <p>This will get styled.</p>

  <div>
    <p>This will not get styled.</p>
  </div>
</div>
```

```css
.container > p {
  background-color: black;
  color: white;
}
```

:::

- **مركب الأخ التالي (`+`)**: يختار هذا المركب العنصر الذي يلي مباشرة عنصر أخ محدد. المثال التالي يختار عنصر الفقرة الذي يلي مباشرة عنصر `h2`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h2>I am a sub heading</h2>

<p>This paragraph element will get a red background.</p>
```

```css
h2 + p {
  background-color: red;
}
```

:::

- **مركب الأخ اللاحق (`~`)**: يختار هذا المركب كل الإخوة لعنصر محدد الذين يأتون بعده. المثال التالي يطبق النمط فقط على عنصر الفقرة الثاني لأنه الوحيد الذي هو أخ لعنصر `ul` ويشترك معه في نفس العنصر الأب.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <p>This will not get styled.</p>
  <ul>
    <li>Example item one</li>
    <li>Example item two</li>
    <li>Example item three</li>
  </ul>
  <p>This will get styled.</p>
</div>
<p>This will not get styled.</p>
```

```css
ul ~ p {
  background-color: green;
}
```

:::

## عناصر المستوى المضمَّن، الكتلي، والمضمَّن-كتلي

- **عناصر المستوى المضمَّن**: تأخذ العناصر المضمَّنة فقط العرض الذي تحتاجه ولا تبدأ في سطر جديد. تتدفق هذه العناصر ضمن المحتوى، مما يسمح للنص والعناصر المضمَّنة الأخرى بالظهور بجانبها. من العناصر المضمَّنة الشائعة `span`، `a`، و`img`.
- **عناصر المستوى الكتلي**: تبدأ عناصر المستوى الكتلي في سطر جديد وتأخذ العرض الكامل المتاح لها افتراضيًا، وتمتد عبر عرض الحاوية الخاصة بها. من العناصر الكتلية الشائعة `div`، `p`، و`section`.
- **عناصر المستوى المضمَّن-كتلي**: يمكنك تعيين عنصر إلى `inline-block` باستخدام خاصية `display`. تتصرف هذه العناصر كعناصر مضمَّنة لكنها يمكن أن تحدد لها `width` و`height` مثل العناصر الكتلية.

## الهامش والحشو

- **خاصية `margin`**: تُستخدم هذه الخاصية لتطبيق مساحة خارج العنصر، بين حدود العنصر والعناصر المحيطة به.
- **خاصية `padding`**: تُستخدم هذه الخاصية لتطبيق مساحة داخل العنصر، بين المحتوى وحدوده.
- **اختصار `margin`**: يمكنك تحديد من 1 إلى 4 قيم لتعيين جوانب الهامش. قيمة واحدة تُطبق على الجوانب الأربعة كلها؛ قيمتان تُعيّنان `top` و`bottom`، ثم `right` و`left`؛ ثلاث قيم تُعيّن `top`، والأفقي (`right` و`left`)، ثم `bottom`؛ أربع قيم تُعيّن `top`، `right`، `bottom`، `left`.
- **اختصار `padding`**: يمكنك تحديد من 1 إلى 4 قيم لتعيين جوانب الحشو. قيمة واحدة تُطبق على الجوانب الأربعة كلها؛ قيمتان تُعيّنان `top` و`bottom`، ثم `right` و`left`؛ ثلاث قيم تُعيّن `top`، والأفقي (`right` و`left`)، ثم `bottom`؛ أربع قيم تُعيّن `top`، `right`، `bottom`، `left`.

## خصوصية CSS

- **خصوصية CSS المضمَّن**: CSS المضمَّن له أعلى خصوصية لأنه يُطبق مباشرة على العنصر. يتجاوز أي CSS داخلي أو خارجي. قيمة الخصوصية لأنماط المضمَّن هي (1, 0, 0, 0).
- **خصوصية CSS الداخلي**: يُعرف CSS الداخلي داخل عنصر `style` في قسم `head` من وثيقة HTML. له خصوصية أقل من أنماط المضمَّن. يشترك كل من CSS الداخلي والخارجي في نفس مستوى الخصوصية، الذي يُحدد بواسطة المحددات، وليس الموقع.
- **خصوصية CSS الخارجي**: يرتبط CSS الخارجي عبر عنصر `link` في قسم `head` ويُكتب في ملفات `.css` منفصلة. مثل CSS الداخلي، تُحدد خصوصيته بواسطة المحددات المستخدمة. عندما يكون لمجموعتين من القواعد نفس الخصوصية، يحدد ترتيب المصدر الفائز: القاعدة التي تظهر لاحقًا في الوثيقة لها الأولوية. يوفر CSS الخارجي أفضل قابلية للصيانة للمشاريع الأكبر.
- **المحدد العام (`*`)**: نوع خاص من محددات CSS يطابق أي عنصر في الوثيقة. يُستخدم غالبًا لتطبيق نمط على كل العناصر في الصفحة، وهو مفيد لإعادة تعيين أو توحيد الأنماط عبر المتصفحات المختلفة. للمحدد العام أقل قيمة خصوصية بين كل المحددات. يساهم بصفر في كل أجزاء قيمة الخصوصية (0, 0, 0, 0).
- **محددات النوع**: تستهدف هذه المحددات العناصر بناءً على اسم الوسم الخاص بها. لمحددات النوع خصوصية منخفضة نسبيًا مقارنة بالمحددات الأخرى. قيمة الخصوصية لمحدد النوع هي (0, 0, 0, 1).
- **محددات الصنف**: تُعرف هذه المحددات بنقطة (`.`) تليها اسم الصنف. قيمة الخصوصية لمحدد الصنف هي (0, 0, 1, 0). هذا يعني أن محددات الصنف يمكنها تجاوز محددات النوع، لكنها يمكن أن تُجاوز بواسطة محددات المعرف وأنماط المضمَّن.
- **محددات المعرف**: تُعرف محددات المعرف برمز الهاش (`#`) يليه اسم المعرف. لمحددات المعرف خصوصية عالية جدًا، أعلى من محددات النوع والصنف، لكنها أقل من أنماط المضمَّن. قيمة الخصوصية لمحدد المعرف هي (0, 1, 0, 0).
- **الكلمة المفتاحية `!important`**: تُستخدم لإعطاء قاعدة نمط أعلى أولوية، مما يسمح لها بتجاوز أي إعلانات أخرى لخاصية معينة. عند استخدامها، تجبر المتصفح على تطبيق النمط المحدد بغض النظر عن خصوصية المحددات الأخرى. يجب الحذر عند استخدام `!important` لأنه قد يجعل CSS أصعب في الصيانة ومعالجة الأخطاء.
- **خوارزمية التتابع (Cascade Algorithm)**: خوارزمية تُستخدم لتحديد أي قواعد CSS تُطبق عندما تستهدف أنماط متعددة نفس العنصر. تضمن استخدام الأنماط الأنسب بناءً على مجموعة قواعد محددة جيدًا.
- **وراثة CSS**: العملية التي تنتقل بها الأنماط من العناصر الأب إلى أبنائها. تسمح الوراثة بتعريف أنماط على مستوى أعلى في شجرة الوثيقة وتطبيقها على عدة عناصر دون الحاجة لتحديدها صراحة لكل عنصر.
