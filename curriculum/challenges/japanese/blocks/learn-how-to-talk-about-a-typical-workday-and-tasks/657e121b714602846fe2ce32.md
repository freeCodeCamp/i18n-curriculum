# --description--

`present simple` 時制は、定期的な行動やルーティンについて話すときに使います。

`present simple` を作るには、`I`、`you`、`we`、`they`（`I code`、`they test`）の基本動詞形を使い、`-s`、`-es`、`he`（`she`、`it`）には `he codes` または `she tests` を付けます。

仕事の文脈では、日々の作業を説明するときは通常、これらが習慣的な行動であるため、`present simple` 時制が使われます。
