# --description--

Du kannst das Browserverhalten einstellen, indem du `meta`-Elemente im `head` hinzufügst. Hier ist ein Beispiel:

```html
<meta attribute="value">
```

Inside the `head` element, nest a `meta` element with a `charset` attribute set to the value of `utf-8`. This tells the browser how to encode characters for the page.

Denk daran, dass ein `meta`-Element ein leeres Element ist.

With that last change, you have completed the Cat Photo App workshop. Glückwunsch!

# --hints--

Du solltest ein `meta`-Element innerhalb des `head`-Elements erstellen.

```js
assert.exists(document.querySelector('head > meta'));
```

You should give the `meta` element a `charset` of `UTF-8`.

```js
assert.equal(document.querySelector('head > meta')?.getAttribute('charset')?.toLowerCase(), 'utf-8');
```

Dein `meta` Element soll ein leeres Element sein, du brauchst kein schließendes Tag `</meta>`.

```js
assert.notMatch(code, /<\/meta\s*>?/i);
```
