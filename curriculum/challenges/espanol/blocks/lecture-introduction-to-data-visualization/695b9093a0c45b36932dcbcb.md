# --description--

En las lecciones anteriores, se te introdujo a la visualización de datos, que es el proceso de crear formatos visuales claros para representar datos complejos. En esta lección, aprenderás sobre la biblioteca D3.

La biblioteca D3 es una biblioteca gratuita y de código abierto que puedes usar para crear visualizaciones de datos atractivas y efectivas en JavaScript.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-1.png" alt="Un mapa coroplético verde de Estados Unidos que muestra datos a nivel de condado, con una escala de color que va de verde claro (3%) a verde oscuro (60%+)">


El nombre D3 es una abreviatura de documentos dirigidos por datos, refiriéndose a cómo enlaza dinámicamente datos con elementos en el Modelo de Objetos del Documento (DOM). Esta biblioteca es flexible y funciona de forma natural con la web. En lugar de tener que aprender una representación gráfica completamente nueva, puedes construir visualizaciones de datos con elementos `svg` y `canvas`.

Por ejemplo, si quieres crear un gráfico de barras usando D3, puedes hacer lo siguiente todo en un solo archivo JS:

* Declarar variables que representen las dimensiones y márgenes del gráfico
    
* Declarar la escala de posición horizontal
    
* Declarar la escala de posición vertical
    
* Crear el contenedor SVG con las barras y etiquetas
    
<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-2.png" alt="Un gráfico de barras titulado Producto Interno Bruto de Estados Unidos mostrando el Producto Interno Bruto desde 1947 hasta 2015, con valores que suben desde casi cero hasta aproximadamente 18,000">

Entonces, ¿cuáles son algunas otras razones para aprender D3 en lugar de otras herramientas de visualización como Chart.js, Plotly o Tableau?

A diferencia de otras herramientas, D3 te ofrece mucho control a bajo nivel para que puedas construir visualizaciones personalizables e interactivas usando solo HTML, CSS y JavaScript puros. Además, la biblioteca está en JavaScript, así que no necesitas preocuparte por aprender un lenguaje completamente nuevo.

En próximos talleres y laboratorios, aprenderás los fundamentos de la biblioteca D3, como selecciones, formas y escalas. Aprenderás estos conceptos clave construyendo gráficos de dispersión, gráficos de barras, aplicaciones de paneles y más.
