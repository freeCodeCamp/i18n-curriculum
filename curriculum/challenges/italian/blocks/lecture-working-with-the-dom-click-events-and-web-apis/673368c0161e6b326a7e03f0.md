# --interactive--

Vediamo come puoi creare nodi con `innerHTML` e `createElement()`.

Inizieremo con `innerHTML`.

`innerHTML` è una proprietà degli oggetti `Element` che puoi usare per impostare il loro markup HTML. Con `innerHTML` puoi impostare la struttura HTML di un elemento esistente con una stringa, creando tutti i nodi necessari.

Ecco un esempio. Abbiamo un `div` vuoto e aggiungeremo alcuni elementi al suo interno usando la proprietà `innerHTML`:

```html
<div id="container">
  <!-- Add new elements here -->
</div>
```

Per prima cosa, dobbiamo selezionare l’elemento tramite il suo ID:

```js
const container = document.getElementById("container");
```

Poi, impostiamo la proprietà `innerHTML` del contenitore su una stringa. Questa stringa deve contenere tutto il markup necessario per rappresentare i nodi e la struttura che vuoi creare. Puoi pensarlo come scrivere HTML all’interno di una stringa.

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
container.innerHTML = "<ul><li>Cheese</li><li>Tomato</li></ul>";
```

:::

Imposteremo la `innerHTML` di questo elemento su una lista non ordinata di ingredienti per la pizza (formaggio e pomodoro).

Puoi anche scrivere la stringa su più righe se rende il markup più facile da capire. L’importante è che il markup rappresenti accuratamente la struttura HTML che vuoi creare.

Dopo aver eseguito questo codice, vedrai la seguente struttura HTML se ispezioni il markup.

```html
<div id="container">
  <ul>
    <li>Cheese</li>
    <li>Tomato</li>
  </ul>
</div>
```

I nuovi nodi sono stati creati e aggiunti dinamicamente al DOM dopo che la stringa è stata analizzata.

`innerHTML` può essere molto utile in certi scenari. Tuttavia, presenta alcuni rischi di sicurezza che dovresti conoscere. Non dovresti usarlo se non hai il pieno controllo sulla stringa.

Per esempio, se la stringa viene inserita dall’utente, non dovresti usare `innerHTML` perché l’utente potrebbe inserire contenuti dannosi nel tuo sito. Per questo motivo, di solito si consiglia di usare `textContent` invece, per inserire testo semplice.

Un altro modo per creare un nuovo nodo è usare il metodo `createElement()`. Con questo nuovo metodo puoi creare un nuovo elemento specificando il nome del tag.

Per esempio, se vuoi creare un elemento immagine, passeresti il tag `img` come stringa quando chiami questo metodo:

```js
document.createElement("img");
```

E puoi assegnare questo nuovo oggetto a una variabile:

```js
const img = document.createElement("img");
```

Il metodo `createElement()` restituisce un nuovo oggetto `HTMLElement` se il documento è un `HTMLDocument`. Altrimenti, restituisce un oggetto `Element`.

Una volta che hai questo nuovo elemento pronto, puoi aggiungerlo al DOM come figlio di un altro elemento esistente usando il metodo `appendChild()`, oppure puoi inserirlo in una posizione specifica usando altri metodi. Scegli quello che meglio si adatta alle tue esigenze.

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
const img = document.createElement("img");
img.src = "https://cdn.freecodecamp.org/curriculum/cat-photo-app/lasagna.jpg";
img.alt = "A slice of lasagna on a plate.";
container.appendChild(img);
```

:::

Creare nodi con `innerHTML` e `createElement()` ti permette di manipolare dinamicamente la struttura e il contenuto dei tuoi siti web. Combinando queste tecniche puoi costruire applicazioni web interattive.
