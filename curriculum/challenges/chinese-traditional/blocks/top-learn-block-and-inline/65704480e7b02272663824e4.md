## --text--

CSS 版面配置中的「正常流程」是什麼？

## --answers--

正常流程是指在 CSS 佈局中為元素製作動畫的處理程序。

---

Normal flow 是一個用來控制版面配置中元素可見性的 CSS 屬性。

---

正常流程是一個 CSS 規則，用來定義響應式設計中媒體查詢的順序。

---

CSS 版面配置中的正常流程是指網頁上元素的預設定位。
