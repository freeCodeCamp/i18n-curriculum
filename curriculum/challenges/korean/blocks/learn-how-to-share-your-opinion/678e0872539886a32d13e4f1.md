# --explanation--

`Lightweight`는 자원이나 전력을 많이 필요로 하지 않아 실행이 쉽고 빠르며 효율적인 것을 설명합니다. 예를 들어:

`The app is lightweight, so it runs smoothly even on older smartphones.` - 이 말은 앱이 매우 적은 메모리나 처리 능력을 사용해 용량이 적은 기기에서도 잘 작동한다는 뜻입니다.

`Features`는 제품이나 소프트웨어가 사용자에게 제공하는 특정 기능이나 도구를 의미합니다. 예를 들어:

`The software has many useful features like file sharing and cloud storage.` - 이 말은 소프트웨어가 여러 유용한 기능을 포함해 사용자에게 더 가치 있고 효율적이라는 뜻입니다.
