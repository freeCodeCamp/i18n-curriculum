# --description--

**Objective:** Build an app that is functionally similar to this: <a href="https://build-a-pinterest-clone.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://build-a-pinterest-clone.freecodecamp.rocks/</a>.

完成以下需求，並且通過所有測試。 你可以使用你需要的任何庫或 API。 可以用你自己的樣式來完成。

**用戶故事：** 作爲未經身份驗證的用戶，我可以使用 GitHub 登錄。

**用戶故事：** 作爲經過身份驗證的用戶，我可以鏈接到圖像。

**用戶故事：** 作爲經過身份驗證的用戶，我可以刪除已鏈接的圖像。

**用戶故事：** 作爲一個認證用戶，我可以看到一個 Pinterest 風格的牆，上面有我鏈接的所有圖片

**用戶故事：** 作爲未經身份驗證的用戶，我可以瀏覽其他用戶的圖像牆。

**用戶故事：** 作爲經過身份驗證的用戶，如果我上傳的圖片已損壞，則會被佔位圖片替換。 （可以使用 jQuery 檢測已損壞的圖像）

**提示：** Masonry.js 是一個庫，可以實現 Pinterest 風格的照片牆。

一旦您實現了這些用戶故事，輸入您在運行的應用 URL 和可選的 GitHub 倉庫。 然後點擊"我已完成這項挑戰"按鈕。

可以分享你的項目到 <a href="https://forum.freecodecamp.org/c/project-feedback/409" target="_blank" rel="noopener noreferrer nofollow">freeCodeCamp 論壇</a>上獲得反饋。
