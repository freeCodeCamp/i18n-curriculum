# --description--

对于两个正整数a和b，Ulam序列U（a，b）由U（a，b）1 = a，U（a，b）2 = b定义，对于k> 2，U（a，b） ）k是大于U（a，b）（k-1）的最小整数，它可以用一种方式写成U（a，b）的两个不同的先前成员的总和。

例如，序列 $U(1,2)$ 开头以

$$1, 2, 3 = 1 + 2, 4 = 1 + 3, 6 = 2 + 4, 8 = 2 + 6, 11 = 3 + 8$$

5 不属于它，因为5=1+4=2+3有两个方法，是两个前成员的总和。 同样，$7=$1+6=$3+4$。

查找 $\sum {U(2, 2n + 1)_k}$，其中 $2 ≤ n ≤ 10$，且 $k = {10}^{11}$。
