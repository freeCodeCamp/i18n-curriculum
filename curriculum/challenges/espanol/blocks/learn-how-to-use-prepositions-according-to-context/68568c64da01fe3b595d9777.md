# --explanation--

`By tomorrow` significa que algo ocurrirá antes o, a más tardar, el día siguiente. Establece una fecha límite clara. Por ejemplo:

`Please finish the report by tomorrow.` – Esto significa que el informe debe estar terminado antes del final del día siguiente.
