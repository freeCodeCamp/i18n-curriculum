# --description--

`Palm vein recognition`는 손바닥의 독특한 정맥 패턴을 기반으로 개인을 식별하는 기술입니다. 각 사람마다 고유한 정맥 패턴을 캡처하고 분석하여 안전한 인증 방법을 제공합니다.

`Behavioral biometrics`는 개인을 식별하고 검증하는 방법으로 고유한 행동을 연구하고 사용하는 것을 의미합니다. 여기에는 타이핑 방식, 움직임, 또는 특정 개인에게만 해당되는 다른 독특한 행동들이 포함됩니다.
