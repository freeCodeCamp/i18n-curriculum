# --explanation--

`Past Continuous` 時態（`was/were` + 動詞的 `-ing` 形式）描述過去一段時間內正在發生的動作。它經常與 `Simple Past` 事件一起使用，以顯示持續的動作被打斷或發生在另一個事件之前。<dfn>例如</dfn>：

`She was working on the report when the computer crashed.` – 工作的動作正在進行時，電腦當機（`Simple Past`）發生了。
