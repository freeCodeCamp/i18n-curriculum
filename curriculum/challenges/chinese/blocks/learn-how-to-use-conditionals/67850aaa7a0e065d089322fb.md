# --explanation--

该句使用了 `Second Conditional`，其结构为：

`If` + `Past Simple` 或 `Past Continuous` 时态，`might / would / could` + 动词原形。在此示例中：

`If they closed` 是条件（假设关闭）。

`It would affect` 是结果（作为后果会发生什么）。

这种类型的条件式用于讨论不真实或假设的情况。
