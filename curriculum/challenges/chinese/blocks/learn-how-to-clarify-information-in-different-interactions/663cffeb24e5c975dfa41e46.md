# --description--

`Double check` 意味着再次查看某物以确保其正确。例如，你可以说 `I wanted to be sure I locked the door, so I double checked it`。这意味着你正在小心避免错误，所以你再次检查了你所做的事情。

`schedule` 是一个显示事件发生时间的计划。例如，`school schedule` 显示课程开始和结束的时间。它帮助你了解一天中不同时间该做什么。在一个项目中，你可以用它来估算项目某些部分开始和结束的天数。

`Accurate` 意味着某事是正确的且没有错误。例如，如果时钟显示正确时间，它就是 `accurate`。这意味着信息是真实且精确的。
