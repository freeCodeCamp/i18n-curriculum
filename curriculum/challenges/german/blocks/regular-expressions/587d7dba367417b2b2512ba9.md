# --description--

<dfn>Vorausschau</dfn> sind Muster, die JavaScript anweisen, im String vorauszuschauen, um weiter hinten liegende Muster zu überprüfen. Das ist nützlich, wenn Sie mehrere Muster im selben String suchen möchten.

Es gibt zwei Arten von Vorausschau: <dfn>positive Vorausschau</dfn> und <dfn>negative Vorausschau</dfn>.

Eine positive Vorausschau prüft, ob das Element im Suchmuster vorhanden ist, stimmt es aber nicht tatsächlich ab. Eine positive Vorausschau wird als `(?=...)` verwendet, wobei `...` der erforderliche Teil ist, der nicht abgeglichen wird.

Eine negative Vorausschau hingegen prüft, ob das Element im Suchmuster nicht vorhanden ist. Eine negative Vorausschau wird als `(?!...)` verwendet, wobei `...` das Muster ist, das nicht vorhanden sein soll. Der Rest des Musters wird zurückgegeben, wenn der Teil der negativen Vorausschau nicht vorhanden ist.

Vorausschau kann etwas verwirrend sein, aber einige Beispiele helfen weiter.

```js
let quit = "qu";
let noquit = "qt";
let quRegex= /q(?=u)/;
let qRegex = /q(?!u)/;
quit.match(quRegex);
noquit.match(qRegex);
```

Beide `match`-Aufrufe würden `["q"]` zurückgeben.

Eine praktischere Anwendung von Vorausschau ist, zwei oder mehr Muster in einem String zu überprüfen. Hier ist ein (naiv) einfacher Passwortprüfer, der nach 3 bis 6 Zeichen und mindestens einer Zahl sucht:

```js
let password = "abc123";
let checkPass = /(?=\w{3,6})(?=\D*\d)/;
checkPass.test(password);
```
