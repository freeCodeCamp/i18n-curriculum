# --interactive--

JavaScriptは、ウェブサイトにインタラクティブ性や動的な動作をもたらす強力なプログラミング言語です。

HTMLとCSSはページ上のコンテンツの構造を作り、要素のスタイルを設定するためのマークアップ言語ですが、JavaScriptはそれを超えて、ユーザーの入力を処理したり、要素をアニメーションさせたり、さらには完全なウェブアプリケーションを構築したりするなど、より複雑な機能を可能にします。

たとえば、ボタンをクリックしたり、フォームを提出したり、メニューにカーソルを合わせたりすると、JavaScriptがページの動作を決定します。

これら3つがどのように連携するかの例を示します：

:::interactive_editor

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: green;
    }
  </style>
</head>
<body>
  <h1>Hello, World!</h1>
  <button onclick="alert('Button clicked!')">Click me</button>
</body>
</html>
```

:::

この例では、HTMLはコンテンツを定義するために使われています：見出し（`h1`要素）とボタン（`button`要素）です。

CSSは見出しのテキストを緑色にするなどのスタイルを適用するために使われています。JavaScriptはボタンがクリックされたときにアラートメッセージを表示するために使われています。

まとめると、HTMLは構造を提供し、CSSはスタイルを追加し、JavaScriptは動作を可能にして、完全でインタラクティブなウェブ体験を作り出します。
