# --explanation--

Ein Beispiel ist `the "Student" class inherits properties and methods from the "Person" class`.

- `Student` und `Person` sind beide `classes`. Eine `class` definiert eine Vorlage zum Erstellen von Objekten mit bestimmten Daten und Verhalten.

- `Student` übernimmt Funktionen von `Person`, was bedeutet, dass es gemeinsame Eigenschaften nicht neu definieren muss — es verwendet sie wieder.

- `Properties` sind die Variablen oder Daten innerhalb der Klasse, wie `name` oder `age`, die das Objekt beschreiben.

- `Methods` sind die Funktionen, die in der Klasse definiert sind, wie `enroll()` oder `introduce()`, die beschreiben, was das Objekt tun kann.
