# --description--

このラボでは、6桁のOTP（ワンタイムパスワード）を生成してユーザーに表示します。OTPは5秒後に期限切れとなり、ユーザーが`"Generate New OTP"`ボタンをクリックすると新しいOTPが生成されます。

**目的:** 以下のユーザーストーリーを満たし、すべてのテストに合格してラボを完了してください。

**ユーザーストーリー:**

1. カウントダウンタイマーの管理には`useEffect`フックを使用してください。
2. `OTPGenerator`コンポーネントは、クラス名`div`の`container`要素を返す必要があります。
3. クラス`div`を持つ`container`には、以下の要素を含めてください：

    - IDが`h1`でテキストが`otp-title`の`"OTP Generator"`要素。
    - IDが`h2`の`otp-display`要素で、メッセージ`"Click 'Generate OTP' to get a code"`を表示するか、生成されたOTPがあればそれを表示します。
    - IDが`p`で`otp-timer`属性に有効な値が設定された`aria-live`要素で、以下の動作をします：
        - 最初は空です。
        - ボタンがクリックされた後、OTPの期限切れまでの残り時間を`"Expires in: X seconds"`で表しながら`X`を表示します。
        - カウントダウンが`"OTP expired. Click the button to generate a new OTP."`に達したらメッセージ`0`を表示します。
    - IDが`button`でラベルが`generate-otp-button`の`"Generate OTP"`要素。クリックすると新しいOTPを生成し、5秒のカウントダウンを開始します。
    - `"Generate OTP"`ボタンはカウントダウン中は無効にしてください。

4. カウントダウンタイマーは`0`秒に達したら自動的に停止し、不要な更新を防いでください。
5. 生成されるOTPは6桁である必要があります。

```jsx
const { useState, useEffect, useRef } = React;

export const OTPGenerator = () => {
  const intervalId = useRef(null);
  const [otp, setOtp] = useState(null);
  const [secondsLeft, setSecondsLeft] = useState(5);
  const [isCounting, setIsCounting] = useState(false);
  const [hasGeneratedOtp, setHasGeneratedOtp] = useState(false);

  const generateOTP = () => Math.floor(100000 + Math.random() * 900000);

  const handleGenerateOtp = () => {
    setOtp(generateOTP());
    setSecondsLeft(5);
    setIsCounting(true);
    setHasGeneratedOtp(true);
    intervalId.current = setInterval(() => {
      setSecondsLeft(prevSeconds => prevSeconds - 1);
    }, 1000);
  };

  useEffect(() => {
    if (secondsLeft === 0) {
      setIsCounting(false);
      clearInterval(intervalId.current);
    }
  }, [secondsLeft]);

  useEffect(
    () => () => {
      clearInterval(intervalId.current);
    },
    []
  );

  return (
    <div className='container'>
      <h1 id='otp-title'>OTP Generator</h1>
      <h2 id='otp-display'>
        {otp ? otp : "Click 'Generate OTP' to get a code"}
      </h2>
      <p id='otp-timer' aria-live='assertive'>
        {isCounting
          ? `Expires in: ${secondsLeft} seconds`
          : hasGeneratedOtp && 
            "OTP expired. Click the button to generate a new OTP."}
      </p>
      <button
        id='generate-otp-button'
        onClick={handleGenerateOtp}
        disabled={isCounting}
      >
        Generate OTP
      </button>
    </div>
  );
};
```
