# --description--

字串是由单引号或双引号包围的一系列字符。在某些编程语言中，被单引号包围的字符与被双引号包围的字符处理方式不同，但在 Python 中，它们被视为相等。因此，在处理字串时，你可以使用任意一种。以下是一些字串的示例：

```python
my_str_1 = 'Hello'
my_str_2 = "World"
```

如果你需要多行字串，可以使用三重双引号或单引号：

```python
my_str_3 = """Multiline
string"""
my_str_4 = '''Another
multiline
string'''
```

如果你的字串包含单引号或双引号，那么你有两个选项：

- 使用相反类型的引用。也就是说，如果你的字串包含单引号，则使用双引号来包裹字串，反之亦然：
    

```python
msg = "It's a sunny day"
quote = 'She said, "Hello World!"'
```

- 用反斜线（`\`）转义字串中的单引号或双引号。通过这种方法，你可以使用单引号或双引号来包裹字串本身：
    

```python
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

有时，你可能需要查看一个字串是否包含一个或多个字符。为此，Python 提供了 `in` 操作符，它返回一个布尔值，指定该字符或字符是否存在于字串中。

这是一些例子：

```python
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

现在，让我们看看你如何获取字串的长度并处理字串中的单个字符，这个过程称为**索引**。要获取字串的长度，你可以使用内置的 `len()` 函数。下面是一个示例：

```python
my_str = 'Hello world'
print(len(my_str))  # 11
```

字串中的每个字符都有一个称为索引的位置。索引是从零开始的，这意味着字串中第一个字符的索引是 `0`，第二个字符的索引是 `1`，依此类推。要通过索引访问字符，你使用带有你想访问的字符索引的方括弧（`[]`）。以下是一些示例：

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
```

也允许负索引，因此你可以用 `-1` 获取任何字串的最后一个字符，用 `-2` 获取倒数第二个字符，依此类推：

```python
my_str = 'Hello world'
print(my_str[-1])  # d
print(my_str[-2]) # l
```

许多其他编程语言将数据类型大致分为原生类型或引用类型。原生类型简单且不可变的，意味着一旦声明就不能更改。引用类型可以包含多值，并且可以是可变的或不可变的。但 Python 并不在这两组之间划清界限。相反，所有数据都被视为对象，有些对象是不可变的，而其他对象是可变的。

不可变的数据类型一旦声明后就不能被修改或更改。你可以将它们的变量指向新的内容，这称为赋值，但你不能通过添加、删除或替换任何元素来更改原始对象本身。 

字串是 Python 中的不可变数据类型。这意味着你可以为变量赋值不同的字串：

```python
greeting = 'hi'
greeting = 'hello'
print(greeting) # hello
```

但不允许直接修改字串：

```python
greeting = 'hi'
greeting[0] = 'H' # TypeError: 'str' object does not support item assignment
```

Python 中其他不可变的数据类型示例包括整数、浮点数、布尔、元组和范围。你将在接下来的课程中了解每种类型。
