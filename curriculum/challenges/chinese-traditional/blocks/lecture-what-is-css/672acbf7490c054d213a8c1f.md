# --interactive--

CSS 可以透過三種主要方式套用到網頁：行內、內部或外部。

每個方法都有其使用情境、優點和限制，並且知道何時使用每個方法對於撰寫乾淨、高效且易於維護的程式碼至關重要。

讓我們拆解三種 CSS 型別，以及你應該何時使用它們。

行內 CSS 是直接在 HTML 元素中使用 `style` 屬性撰寫的。它將樣式套用到特定元素。

以下是使用行內 CSS 的範例：

:::interactive_editor

```html
<p style="color: green;">This is an inline-styled paragraph.</p>
```

:::

在這個範例中，我們使用 `style` 屬性將段落文字設為 `green`。

行內 CSS 通常用於快速、一次性的樣式或為特定元素改寫其他樣式。

然而，在大多數情況下應避免使用，因為它會使 HTML 混亂並且讓程式碼更難維護。

大多數情況下，使用內部或外部 CSS 來保持你的樣式有條理且易於維護會比較好。

內部 CSS 是寫在 HTML 文件中 `head` 區段內的 `style` 頁籤中。它會套用樣式到整個頁面，當你需要為單一文件設定樣式時非常有用。

以下是內部 CSS 的範例：

:::interactive_editor

```html
<head>
  <style>
    p {
      color: blue;
    }
  </style>
</head>
<body>
  <p>This paragraph is styled using internal CSS.</p>
</body>
```

:::

在上述範例中，內部 CSS 對頁面上的所有 `p` 元素套用藍色文字。

當你需要將樣式套用到特定頁面而非多個頁面時，內部 CSS 是最佳選擇。它適用於單頁網站或樣式不需要在其他地方重複使用的情況。

然而，也有一些缺點，例如無法促進跨多個頁面的重複使用。此外，像行內 CSS 一樣，它將 HTML 和 CSS 混合，使得在較大型專案中程式碼較難維護。

外部 CSS 寫在獨立的 `.css` 檔案中，並使用 `head` 區段的 `link` 元素將其連結到 HTML 文件。

它允許你一致地為多個頁面設計樣式，並且是在專業網頁開發中首選的方法。

以下是為所有段落元素設定樣式的範例：

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <p>This paragraph is styled using external CSS.</p>
</body>
```

```css
p {
  color: red;
}
```

:::

在先前的課程中，你學到 `link` 元素具有一個 `rel` 屬性，用來指定目前文件與連結資源之間的關係，例如連結到樣式表或外部資源。

另一方面，`href` 屬性指定聯結資源的 URL，表示應該從哪裡擷取該資源。

外部 CSS 非常適合大型專案，當你想在多個頁面中維護一致的樣式時。

它透過讓 HTML 控制結構、CSS 控制樣式來促進關注點分離，這使得程式碼更易於維護和擴展。

了解何時使用每種 CSS 型別對於高效且有效的網頁開發至關重要。

在大多數情況下，外部 CSS 應該是你的首選方法，特別是對於較大且較複雜的專案。
