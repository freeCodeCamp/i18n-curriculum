# --interactive--

Wenn der Benutzer auf eine Schaltfläche klickt oder sich ein Formular ändert, spricht man von einem Ereignis. In Ihren Programmen müssen Sie eine Möglichkeit haben, auf diese Ereignisse zu hören und darauf zu reagieren.

Die `addEventListener()`-Methode wird verwendet, um auf Ereignisse zu hören. Sie nimmt zwei Argumente: das Ereignis, auf das Sie hören möchten, und eine Funktion, die aufgerufen wird, wenn das Ereignis eintritt.

Hier ist die grundlegende Syntax:

```js
element.addEventListener("event", listener);
```

Das Element ist das HTML-Element, das auf Ereignisse überwacht wird, und event gibt die Art des Ereignisses an, auf das gehört werden soll, zum Beispiel `"click"`.

Der Listener ist ein Objekt, das die Benachrichtigung erhält, wenn das Ereignis eintritt. Meistens ist dies eine Funktion, die Sie definieren, um das Ereignis zu behandeln.

Hier ist ein Beispiel:

```js
element.addEventListener("click", () => {
  // code to run when the click event occurs
});
```

Sie können auch eine Funktionsreferenz so übergeben:

```js
function handleClick() {
  // code to run when the click event occurs
}

element.addEventListener("click", handleClick);
```

Manchmal möchten Sie vielleicht eine separate Funktion erstellen, um das Ereignis zu behandeln. Das kann Ihren Code leichter lesbar und wartbar machen.

Das Listener-Argument kann auch `null` sein oder ein Objekt, das das `EventListener`-Interface verwendet. Das `EventListener`-Interface definiert eine einzelne Methode namens `handleEvent()`. Diese Methode wird automatisch aufgerufen, wann immer das Ereignis, auf das Sie hören, eintritt. Die Verwendung dieses Interfaces ermöglicht es, dass dasselbe Objekt bei Bedarf mehrere Ereignisse behandelt.

Meistens müssen Sie dieses Interface nicht verwenden und können stattdessen eine Funktion übergeben, die aufgerufen wird, wenn das Ereignis eintritt.

Schauen wir uns nun ein Beispiel an, um besser zu verstehen, wie das funktioniert:

In diesem Beispiel haben wir ein `button`-Element mit der ID `btn`:

```html
<button id="btn">Show alert</button>
```

Wir möchten auf das `"click"`-Ereignis bei dieser Schaltfläche hören und eine Warnmeldung anzeigen, wenn die Schaltfläche angeklickt wird.

Zuerst müssen wir in unserem JavaScript-Code auf dieses `button`-Element zugreifen. Dann fügen wir einen Event Listener hinzu, um auf das Klick-Ereignis zu hören und eine Warnmeldung anzuzeigen, wenn die Schaltfläche angeklickt wird:

:::interactive_editor

```html
<button id="btn">Show alert</button>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("btn");

btn.addEventListener("click", () => alert("You clicked the button"));
```

:::

Jetzt sieht der Benutzer jedes Mal, wenn die Schaltfläche angeklickt wird, eine Warnmeldung auf dem Bildschirm wie diese:

```md
You clicked the button
```

Ein weiterer Ereignistyp, auf den Sie hören können, ist das `"input"`-Ereignis. Dieses Ereignis wird ausgelöst, wenn sich der Wert eines Eingabeelements ändert.

Hier ist ein Beispiel:

```html
<input type="text" id="input" placeholder="Type something" />
```

Wir möchten auf das input-Ereignis bei diesem Eingabeelement hören und den Wert der Eingabe jedes Mal in der Konsole protokollieren, wenn der Benutzer etwas eingibt.

Zuerst müssen wir in unserem JavaScript-Code auf dieses `input`-Element zugreifen. Dann fügen wir einen Event Listener hinzu, um auf das input-Ereignis zu hören und den Wert der Eingabe jedes Mal in der Konsole zu protokollieren, wenn der Benutzer etwas eingibt:

:::interactive_editor

```html
<input type="text" id="input" placeholder="Type something" />
<script src="index.js"></script>
```

```js
const input = document.getElementById("input");

input.addEventListener("input", () => {
  console.log(input.value);
});
```

:::

Es gibt viele weitere Ereignisse, auf die Sie mit der `addEventListener()`-Methode hören können. Einige häufige Ereignisse sind `mouseover`, `mouseout`, `keydown`, `keyup` und `submit`.

In zukünftigen Lektionen werden wir weitere Ereignisse behandeln und wie man die `addEventListener()`-Methode verwendet, um auf sie zu hören.
