# --description--

Das Verb `jog` bedeutet, in einem langsamen, gleichmäßigen Tempo zu laufen. Das Wort `start` kann von einem Verb gefolgt werden, das auf `-ing` endet (wie `jogging`), oder von `to` gefolgt von der Grundform des Verbs (wie `to jog`). Dies zeigt den Beginn einer Handlung an.

Adverbien beschreiben, wie Handlungen ausgeführt werden. Sie werden oft gebildet, indem man `-ly` an Adjektive anhängt. Zum Beispiel ist `regular` ein Adjektiv, das „gewöhnlich“ oder „oft gemacht“ bedeutet. Wenn Sie `-ly` hinzufügen, wird daraus `regularly`, ein Adverb, das bedeutet, etwas auf übliche oder häufige Weise zu tun.

Beispiele:

`He has a regular exercise schedule.`, oder `He exercises regularly.`
