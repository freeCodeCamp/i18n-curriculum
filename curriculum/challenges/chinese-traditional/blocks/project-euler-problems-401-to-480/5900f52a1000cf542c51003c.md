# --description--

對於每個整數 $n > 1$，函式族 $f_{n, a, b}$ 定義為：

$f_{n, a, b}(x) ≡ ax + b\bmod n$，其中 $a, b, x$ 為整數且 $0 \lt a \lt n$、$0 \le b \lt n$、$0 \le x \lt n$。

如果對每個 $0 \le x \lt n$ 都有 $f_{n, a, b}(f_{n, a, b}(x)) \equiv f_{n, a, b}(x)\bmod n$，我們將稱 $f_{n, a, b}$ 為一個縮回。

令 $R(n)$ 為 $n$ 的縮回數量。

你被給予了

$$\sum_{k = 1}^{99\\,999} R(\displaystyle\binom{100\\,000}{k}) \equiv 628\\,701\\,600\bmod 1\\,000\\,000\\,007$$

求 $$\sum_{k = 1}^{9\,999\,999} R(\displaystyle\binom{10\,000\,000}{k})$$，並將你的答案對 $1\,000\,000\,007$ 取模。
