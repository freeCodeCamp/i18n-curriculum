# --description--

术语 `run code` 指的是计算机执行或操作程序或脚本的动作。在此上下文中，`run` 意味着启动进程并持续执行，直到程序完成其设计的功能，无论是计算数字、排序数据还是其他任何操作。

短语 `run code without managing servers` 指的是执行计算机程序或应用，而无需处理代码运行所依赖的物理或虚拟服务器。这是无服务器计算的一个关键方面，在这种模式下，云提供商管理服务器，使开发者能够更多地专注于编码，而不是服务器管理。
