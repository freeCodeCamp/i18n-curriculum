# --description--

Давайте познайомимося з функціями операторів JavaScript, які виходять за межі шкільної арифметики.

Зазвичай оператор плюс `+` додає числа.

Але якщо двійковий `+` застосовується до рядків, він об’єднує (конкатенує) їх:

```js
let s = "my" + "string";
console.log(s); // mystring
```

Зверніть увагу, що якщо будь-який із операндів є рядком, то інший також перетворюється на рядок.

Наприклад:

```js
console.log(1 + '2'); // '12'
console.log('1' + 2); // '12'
```

Бачите, не має значення, чи перший операнд є рядком, чи другий.

Ось більш складний приклад:

```js
console.log(2 + 2 + '1') // "41" and not "221"
```

Тут оператори працюють один за одним. Перший `+` додає два числа, тому повертає `4`, потім наступний `+` додає до нього рядок `1`, тож це як `4 + '1' = '41'`.

```js
console.log('1' + 2 + 2) // "122" and not "14"
```

Тут перший `+` конкатенує рядок і число, тому повертає `'12'`, потім наступний `+` додає `2`, тож це як `'12' + 2 = '122'`.

Двійковий `+` — єдиний оператор, який підтримує рядки таким чином. Інші арифметичні оператори працюють лише з числами і завжди перетворюють свої операнди на числа.

Ось демонстрація для віднімання та ділення:

```js
console.log( 6 - '2' ); // 4, converts '2' to a number
console.log( '6' / '2' ); // 3, converts both operands to numbers
```
