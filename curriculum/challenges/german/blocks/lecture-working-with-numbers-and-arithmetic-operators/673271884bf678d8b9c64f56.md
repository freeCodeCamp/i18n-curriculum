# --interactive--

JavaScript bietet Werkzeuge, um grundlegende arithmetische Operationen mit Zahlen durchzuführen, wie Addition, Subtraktion, Multiplikation und Division. JavaScript enthält auch Operatoren für komplexere arithmetische Operationen, wie Restwert und Exponentiation.

All diese Werkzeuge nennt man arithmetische Operatoren. Schauen wir uns diese Operatoren im Detail an, wie man sie verwendet und wie man sie kombiniert.

Der Additionsoperator ist ein Pluszeichen (`+`). Der Additionsoperator ermöglicht es Ihnen, die Summe von zwei oder mehr Zahlen zu ermitteln. Bei Additionsoperationen spielt die Reihenfolge der Zahlen keine Rolle:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 + num2;
const result2 = num2 + num1;
const result3 = num2 + num1 + num3;

console.log(result1); // 15
console.log(result2); // 15
console.log(result3); // 30
```

:::

Der Subtraktionsoperator ist ein Minuszeichen (`-`). Er ermöglicht es Ihnen, die Differenz zwischen zwei Zahlen zu finden. Verwenden Sie das Minuszeichen, wenn Sie eine Zahl von einer anderen Zahl subtrahieren möchten, normalerweise eine kleinere von einer größeren:

:::interactive_editor

```js
const difference = 10 - 5;
console.log(difference); // 5
```

:::

Wenn eine kleinere Zahl zuerst kommt, erhalten Sie ein negatives Ergebnis:

:::interactive_editor

```js
const difference = 5 - 10;
console.log(difference); // -5
```

:::

Sie können die Zahlen auch Variablen zuweisen und die Subtraktion mit den Variablennamen durchführen:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const result = num1 - num2;

console.log(result); // 5
```

:::

In JavaScript wird der Multiplikationsoperator durch ein Sternchen (`*`) dargestellt und dient dazu, das Produkt von zwei oder mehr Zahlen zu ermitteln. Die Reihenfolge der Zahlen, die Sie multiplizieren, spielt keine Rolle:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 * num2;
const result2 = num2 * num1;
const result3 = num2 * num1 * num3;

console.log(result1); // 50
console.log(result2); // 50
console.log(result3); // 750
```

:::

In JavaScript ist der Divisionsoperator ein Schrägstrich (`/`), der sich vom in der traditionellen Mathematik verwendeten Divisionszeichen (`÷`) unterscheidet. Divisionen führen Sie mit dem Divisionsoperator durch. Die Reihenfolge der Zahlen, die Sie dividieren, ist in diesem Fall wichtig:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 / num2;
const result2 = num2 / num1;
const result3 = num2 / num1 / num3;

console.log(result1); // 2
console.log(result2); // 0.5
console.log(result3); // 0.03333333333333333
```

:::

Es ist wichtig zu beachten, dass JavaScript `Infinity` zurückgibt, wenn Sie versuchen, durch Null zu dividieren:

:::interactive_editor

```js
const result = 10 / 0; 

console.log(result); // Infinity
```

:::

Vermeiden Sie solche Berechnungen, damit Sie keine unerwarteten Ergebnisse in Ihrem Code erhalten.

Der Restwertoperator, dargestellt durch ein Prozentzeichen (`%`), gibt den Rest einer Division zurück. Der Rest in der Mathematik ist der verbleibende Wert nach der Division:

:::interactive_editor

```js
const num1 = 10;
const num2 = 3;
const remainder = num1 % num2;

console.log(remainder); // 1
```

:::

Der Exponentationsoperator, dargestellt durch zwei Sternchen (`**`), hebt eine Zahl auf die Potenz einer anderen:

:::interactive_editor

```js
const num1 = 2;
const num2 = 3;

const exponent = num1 ** num2;
console.log(exponent); // 8
```

:::

Es ist möglich, Operatoren in einem einzigen Ausdruck zu mischen:

:::interactive_editor

```js
const result = 10 + 5 * 2 - 8 / 4;
console.log(result); // 18
```

:::

Wenn Sie verschiedene Operatoren in einem Ausdruck mischen, folgt die JavaScript-Engine einem System namens Operatorpräzedenz, um die Reihenfolge der Operationen zu bestimmen. Die Operatorpräzedenz legt fest, in welcher Reihenfolge Operationen in Ausdrücken ausgeführt werden. Mehr über die Operatorpräzedenz lernen Sie in zukünftigen Lektionen.
