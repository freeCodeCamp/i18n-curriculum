# --description--

`r` の文字はスペイン語で `erre` と呼ばれます。

この子音には2つの異なる発音があります。

単一の `r` は柔らかい音です。例えば、`cero`（「ゼロ」）のように。

二重の `r`（`rr`）または単語の最初の `r` は強い音になります。例えば、`perro`（「犬」）や `ruido`（「騒音」）のように。
