# --description--

Impariamo a conoscere il metodo `querySelectorAll()`. Puoi usare questo metodo per ottenere una lista di tutti gli elementi del DOM che corrispondono a un selettore CSS specifico.

Ecco come puoi chiamare il metodo `querySelectorAll()`:

```js
document.querySelectorAll(selectors);
```

Chiamalo sull’oggetto `document` e passa una stringa con i selettori CSS come argomento. L’argomento deve essere una stringa di selettore CSS valida. Altrimenti, verrà lanciata un’eccezione `SyntaxError`.

`querySelectorAll()` restituisce un oggetto `NodeList`, una collezione di nodi che corrispondono al selettore CSS specificato.

La lista conterrà un oggetto `Element` per ogni elemento che corrisponde ai selettori CSS. Se non ci sono corrispondenze, la lista sarà vuota.

Gli elementi saranno nell’ordine in cui appaiono nel documento HTML. Per esempio, puoi selezionare tutti gli elementi di un tipo specifico:

```js
document.querySelectorAll("div");
```

Tutti gli elementi con una classe specifica:

```js
document.querySelectorAll(".rounded");
```

Tutti gli elementi con un ID specifico:

```js
document.querySelectorAll("#logo");
```

Oppure tutti gli elementi con un attributo specifico, come tutti i link che portano gli utenti a un URL specifico:

```js
document.querySelectorAll("a[href='https://www.freecodecamp.org/']");
```

Puoi anche usare selettori CSS più complessi, come questo, che corrisponde a tutti gli elementi lista all’interno di un elemento lista non ordinata con la classe `ingredients`:

```js
document.querySelectorAll("ul.ingredients li");
```

Poiché il metodo restituisce una collezione di nodi, puoi assegnare questa collezione a una variabile e usarla nei tuoi programmi per lavorare con gli elementi singoli, come puoi vedere in questo esempio con la sintassi generale:

```js
const matches = document.querySelectorAll(selectors); 
```

Per esempio, se hai questa lista non ordinata con tre elementi lista:

```html
<ul class="ingredients">
  <li>Flour</li>
  <li>Cheese</li>
  <li>Water</li>
</ul>
```

Puoi chiamare il metodo `querySelectorAll()` per selezionare tutti gli elementi lista all’interno della lista non ordinata con la classe `ingredients` e assegnare il valore di ritorno alla costante `matches`:

```js
const matches = document.querySelectorAll("ul.ingredients li");
```

Questo restituirà una collezione di tre nodi. Ogni nodo rappresenta un elemento lista:

```js
// NodeList(3)
{
  0: `<li>Flour</li>`,
  1: `<li>Cheese</li>`,
  2: `<li>Water</li>`,
  length: 3,
}
```

Puoi lavorare con questa collezione esattamente come faresti con qualsiasi altro array JavaScript.

Per esempio, puoi mostrarla nella console con `console.log()`:

```js
console.log(matches);
```

Puoi anche controllare la lunghezza della collezione con la proprietà `length`:

```js
console.log(matches.length);
```

L’output sarà un numero intero che rappresenta il numero di nodi nella collezione:

```md
3
```

Puoi anche consultare gli elementi singoli della collezione con i loro indici corrispondenti. Il primo elemento sarà all’indice `0`:

```js
console.log(matches[0]);
console.log(matches[1]);
console.log(matches[2]);
```

Questo è l’output:

```html
<li>Flour</li>
<li>Cheese</li>
<li>Water</li>
```

Gli elementi vengono stampati uno per uno sulla console.

Puoi anche iterare sugli elementi con un loop `for`:

```js
for (let i = 0; i < matches.length; i++) {
  console.log(matches[i]);    
}
```

L’output sarà esattamente lo stesso in questo caso. Ma questo può essere molto potente per lavorare con tutti gli elementi della collezione.

Il metodo `querySelectorAll()` in JavaScript è uno strumento potente per selezionare più elementi basandosi sui loro selettori CSS. Con questo metodo puoi selezionare elementi specifici all’interno delle tue pagine web e manipolarli come necessario.
