# --description--

在之前的课程中，你学习了如何在 JavaScript 中使用导入和导出。在本课中，我们将了解如何在 React 中导入和导出组件。

在这个示例中，我们有一个属于名为 `Cat.jsx` 的文件的 `Cat` 组件。对于文件扩展名，我们使用 `.jsx` 扩展名，因为该文件主要处理 JSX。

这个 `Cat` 组件 `return` 一个包含标题和名为 Mr. Whiskers 的猫的图像的 JSX 标记：

```jsx
function Cat() {
  return (
    <div className="card">
      <h2>Mr. Whiskers</h2>
      <img
        src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/running-cats.jpg"
        alt="Tuxedo cats running on dirt ground."
      />
    </div>
  );
}
```

如果我们想在另一个文件中使用我们的 `Cat` 组件，首先需要像这样导出它：

```jsx
function Cat() {
  return (
    <div className="card">
      <h2>Mr. Whiskers</h2>
      <img
        src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/running-cats.jpg"
        alt="Tuxedo cats running on dirt ground."
      />
    </div>
  );
}

export default Cat;
```

我们使用 `default` 关键字，因为这将是该模块的默认导出。你也可以选择在组件定义的同一行导出该组件，如下所示：

```jsx
export default function Cat() {
  return (
    <div className="card">
      <h2>Mr. Whiskers</h2>
      <img
        src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/running-cats.jpg"
        alt="Tuxedo cats running on dirt ground."
      />
    </div>
  );
}
```

你可以选择在其他父组件的文件中导入子组件，或者在根组件的文件中导入它们。对于此示例，我们将在根组件的文件中导入 `Cat` 组件。

每个 React 项目都会有一个顶层组件，通常称为 `App.jsx`：

```jsx
export default function App() {
  return (
    // return component here
  );
}
```

此文件通常位于你的项目的 `src` 目录中。你将在后续课程中了解更多关于常见项目版型的内容。

要在根 `App` 组件内使用 `Cat` 组件，你需要这样导入它：

```jsx
import Cat from "./Cat";

export default function App() {
  return (
    // return component here
  );
}
```

现在，你可以像这样在 `App` 组件内 `return` `Cat` 组件：

```jsx
import Cat from "./Cat";

export default function App() {
  return (
    <Cat />
  );
}
```

随着你继续创建你自己的 React 项目，你会更加熟悉组织组件、在需要的地方导入它们，以及通过组合这些模块化组件来创建复杂的用户界面。
