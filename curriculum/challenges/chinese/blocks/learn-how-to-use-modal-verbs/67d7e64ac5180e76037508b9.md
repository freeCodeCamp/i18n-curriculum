# --explanation--

在疑问句中，情态动词如 `can`、`could` 和 `should` 位于主语之前。紧随其后的动词必须使用原形。这就是为什么你说 `Could you help?` 而不是 `Do you can help?` 或 `Can helping?`。

例如，在 `Should we report this issue?` 和 `Might he know the password?` 中，情态动词位于前面，主要动词保持原形。

在情态动词前使用 `do` 或 `does` 是错误的。你不会说 `Do you can reset the system?`，而应该说 `Can you reset the system?`

英语疑问句中词序很重要。说 `You can help me?` 听起来不自然，因为主语和情态动词的顺序不正确。正确的形式是 `Can you help me?`
