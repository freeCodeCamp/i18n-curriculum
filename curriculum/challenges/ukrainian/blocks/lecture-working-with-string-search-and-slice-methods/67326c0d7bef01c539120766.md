# --interactive--

Працюючи з рядками в JavaScript, часто потрібно перевірити, чи містить рядок певний підрядок — меншу частину цього рядка.

Наприклад, ви можете захотіти перевірити, чи містить вхідні дані користувача конкретне слово або символ перед виконанням певної дії. Один зі способів зробити це — використати метод `includes()`.

Метод `includes()` використовується для перевірки, чи містить рядок певний підрядок. Якщо підрядок знайдено в рядку, метод повертає `true`, інакше повертає `false`.

Ось базовий синтаксис:

```js
string.includes(searchValue);
```

У цьому синтаксисі `searchValue` — це підрядок, який ви хочете знайти в рядку. Ось приклад:

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("awesome");

console.log(result);  // true
```

:::

У цьому прикладі слово `awesome` міститься в рядку `JavaScript is awesome!`, тому метод `includes()` повертає `true`.

Важливо зауважити, що метод `includes()` чутливий до регістру. Це означає, що потрібне точне співпадіння символів, включно з їх регістром.

Наприклад:

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("Awesome");

console.log(result);  // false
```

:::

Оскільки `Awesome` (з верхнім регістром `A`) не співпадає з `awesome` (з нижнім регістром `a`), результатом буде `false`.

Ви також можете використати метод `includes()` для перевірки підрядка, починаючи з певного індексу в рядку, передавши другий параметр:

:::interactive_editor

```js
let text = "Hello, JavaScript world!";
let result = text.includes("JavaScript", 7);

console.log(result);  // true
```

:::

Тут пошук підрядка `JavaScript` починається з 7-ї позиції в рядку, пропускаючи всі символи перед цією позицією.

Метод `includes()` повертає лише результат `true` або `false`. Він не надає інформації про те, де саме знаходиться підрядок у рядку або скільки разів він зустрічається. Якщо вам потрібна така детальна інформація, інші методи, наприклад метод `indexOf()`, можуть бути більш доречними.
