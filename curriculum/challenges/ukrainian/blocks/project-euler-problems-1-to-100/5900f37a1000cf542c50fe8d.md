# --description--

Наступна ітеративна послідовність визначена для множини натуральних чисел:

<div style='padding-left: 4em;'><var>n</var> → <var>n</var>/2 (<var>n</var> парне)</div>

<div style='padding-left: 4em;'><var>n</var> → 3<var>n</var> + 1 (<var>n</var> непарне)</div>

Використовуючи наведене правило і починаючи з 13, ми отримуємо таку послідовність:

<div style='text-align: center;'>13 → 40 → 20 → 10 → 5 → 16 → 8 → 4 → 2 → 1</div>

Можна побачити, що ця послідовність (починаючи з 13 і закінчуючи 1) містить 10 членів. Хоча це ще не доведено (проблема Коллатца), вважається, що всі початкові числа закінчуються на 1.

Яке початкове число, менше за `limit`, породжує найдовший ланцюг?

**Примітка:** Після початку ланцюга члени можуть перевищувати `limit`.

```js
function longestCollatzSequence(limit) {
  let longest = 1;
  let maxLength = 1;
  for (let i = Math.floor(limit / 2); i < limit; i++) {
    let len = colLen(i);
    if (len > maxLength) {
      longest = i;
      maxLength = len;
    }
  }
  return longest;
}

const knownSequence = { '1': 1 };

function colLen(n) {
  if (knownSequence[n]) {
    return knownSequence[n];
  } else {
    const len = n % 2 === 0 ? colLen(n / 2) + 1 : colLen((3 * n + 1) / 2) + 2;
    knownSequence[n] = len;
    return len;
  }
}
```
