# --explanation--

`log` 是由電腦系統或軟體創建的事件、動作或訊息的記錄。例如：

`The system logs showed several errors before the crash。`－這表示記錄中有關於發生問題的資訊。

`crash` 是指電腦系統或程式突然停止正常運作。 例如：

`The application crash caused all unsaved data to be lost。`－這表示程式意外失敗，且資料未被存檔。
