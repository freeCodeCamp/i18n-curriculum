# --description--

スペイン語では、出身地を表す一般的な方法が2つあります。

1つ目は **`Es de` + 国名** を使う方法です。例えば：

`Es de Venezuela` - 彼はベネズエラ出身です。

2つ目は国籍の形容詞を使う方法です。例えば：

`Es venezolano` - 彼はベネズエラ人です。

国籍は大文字で始めません。

# --explanation--

エステバンは `Sí, es brasileño` と言っています。これは男性形で「はい、彼はブラジル人です」という意味です。

単語 `i` の `Sí` の文字にアクセント記号を付けることが非常に重要です。アクセント記号があると、この単語は「はい」という意味になります。アクセント記号がない場合、文脈によっては「もし」という条件を表すことがあります。

スペイン語では国籍は大文字で始めないことを覚えておいてください。
