# --description--

Kwa kila nambari kamili $n > 1$, familia ya vitendakazi $f_{n, a, b}$ inafafanuliwa kama ifuatavyo:

$f_{n, a, b}(x) ≡ ax + b\bmod n$ kwa $a, b, x$ nambari kamili na $0 \lt a \lt n$, $0 \le b \lt n$, $0 \le x \lt n$.

Tutaita $f_{n, a, b}$ kurudisha ikiwa $f_{n, a, b}(f_{n, a, b}(x)) \equiv f_{n, a, b}(x)\bmod n$ kwa kila $0 \le x \lt n$.

Acha $R(n)$ iwe idadi ya kurudisha kwa $n$.

$F(N) = \displaystyle\sum_{n = 2}^N R(n)$.

$F({10}^7) ≡ 638\\,042\\,271\bmod 1\\,000\\,000\\,007$.

Tafuta $F({10}^{14})$. Toa jibu lako modulo $1\\,000\\,000\\,007$.
