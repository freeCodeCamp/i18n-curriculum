# --interactive--

Nella lezione precedente, hai conosciuto la griglia CSS che puoi usare per creare layout complessi e fluidi nelle tue pagine web. In questa lezione, esploreremo come creare layout a griglia flessibili usando l’unità `fr`.

Iniziamo con questo markup HTML che rappresenterà il nostro contenitore griglia:

```html
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

Nel CSS, impostiamo la proprietà display su grid per il contenitore.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 25% 25% 25% 25%;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

`grid-template-columns` viene usato per impostare la dimensione di ogni colonna. In questo caso, la dimensione di ogni colonna sarà `25%` del contenitore. Poi la proprietà gap viene usata per creare spazio intorno a ogni colonna.

Finora abbiamo usato percentuali per la dimensione delle colonne ma possiamo anche usare l’unità `fr`.

L’unità `fr` è un’unità frazionaria che rappresenta una frazione dello spazio del contenitore griglia.

Ecco come sarà il codice quando viene rifattorizzato per usare unità `fr` invece delle percentuali.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

Ogni colonna occuperà una frazione dello spazio disponibile. Poiché ci sono quattro colonne, ognuna avrà una quota uguale dello spazio disponibile nel contenitore griglia.

Man mano che inizi a costruire i tuoi layout a griglia, ti accorgerai di voler usare più spesso le unità `fr` perché offrono un modo flessibile e proporzionale per distribuire lo spazio, permettendoti di creare layout responsivi che si adattano a diverse dimensioni dello schermo senza dover regolare manualmente i valori in pixel.
