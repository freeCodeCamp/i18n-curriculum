# --interactive--

Potresti aver visto un collegamento su una pagina web diventare viola dopo averci cliccato sopra. Questo succede perché lo stato del collegamento è cambiato, quindi viene applicato uno stile diverso. Ci sono cinque stati diversi in cui un collegamento può trovarsi.

Il primo è lo stato predefinito, che è `:link`. Questo stato rappresenta un collegamento che l’utente non ha ancora visitato, cliccato o con cui non ha ancora interagito. Puoi pensare a questo stato come a quello che fornisce gli stili base per tutti i collegamenti sulla tua pagina. Gli altri stati si basano su questo.

Abilita l’editor interattivo e dai un’occhiata al collegamento nella finestra di anteprima. Dovrebbe essere del colore blu predefinito che rappresenta lo stato predefinito. Se ci clicchi sopra, diventerà viola.

:::interactive_editor

```html
<a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
```

:::

Il secondo stato è `:visited`, che si applica quando un utente ha già visitato la pagina a cui il collegamento punta. Per impostazione predefinita, questo fa diventare il collegamento viola, ma puoi sfruttare il CSS per fornire un’indicazione visiva diversa all’utente. Questo è utile per far sapere a qualcuno che ha già letto una parte della tua documentazione. Oppure, che il sito è uno di cui si può fidare perché lo ha già usato in precedenza.

Clicca sul collegamento nella finestra di anteprima e vedrai che il collegamento visitato cambia colore diventando marrone. Per vedere le anteprime, devi abilitare l’editor interattivo.

**NOTA**: Per questo esempio interattivo è stato fornito del CSS così puoi vedere i cambiamenti negli stili dei collegamenti. Non preoccuparti di capire il codice CSS perché imparerai tutto nei moduli futuri.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:visited {
  color: brown;
}
```

:::

Il terzo stato è `:hover`. Questo stato si applica quando un utente passa il cursore sopra un collegamento. Questo stato è utile per attirare maggiore attenzione su un collegamento, per assicurarsi che l’utente abbia davvero intenzione di cliccarlo.

Abilita l’editor interattivo e prova a passare il mouse sopra il collegamento. Vedrai il colore cambiare in rosso.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:hover {
  color: red;
}
```

:::

Poi abbiamo `:focus`. Questo stato si applica quando mettiamo il focus su un collegamento.

Clicca in qualsiasi punto dello spazio vuoto nella finestra di anteprima e poi premi `tab` sulla tastiera. Dovresti vedere il collegamento diventare verde. Per vedere le anteprime, devi abilitare l’editor interattivo.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:focus {
  color: green;
}
```

:::

E infine, abbiamo `:active`. Questo stato si applica ai collegamenti che vengono attivati dall’utente. Questo di solito significa cliccare sul collegamento con il pulsante principale del mouse, cioè il clic sinistro nella maggior parte dei casi. Questo stato può essere utile per mostrare all’utente che l’elemento su cui ha cliccato è interattivo.

Abilita l’editor interattivo e clicca sul collegamento. Dovresti vedere il collegamento diventare nero. Succede molto rapidamente quindi potresti dover cliccare più volte per vedere il cambiamento di colore.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:active {
  color: black;
}
```

:::

Quando usi questi stati per definire lo stile dei tuoi collegamenti, c’è un ordine specifico in cui devi scrivere il CSS: `link`, `visited`, `hover`, `focus`, poi `active`.

Ora dovresti sapere come dare ai collegamenti della tua pagina un tocco personale, fornendo comunque le informazioni importanti che un utente deve conoscere sullo stato di ogni collegamento.
