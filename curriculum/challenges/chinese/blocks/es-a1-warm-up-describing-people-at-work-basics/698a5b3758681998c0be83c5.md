# --description--

有些符号在日常工作活动中变得必不可少。

以下是西班牙语中最常用的四个符号：

| 符号 | 西班牙语 | 含义 |
| :---: | :--- | :--- |
| `.` | `Punto` | 句号 / 点 |
| `@` | `Arroba` | 电子邮件符号（@） |
| `-` | `Guión` | 连字符 / 破折号 |
| `_` | `Guión bajo` | 下划线 |

注意单词 `Guión` 在字母 `o` 上带有重音符号。

Camila 会帮助你练习这些符号。重复它们的发音，直到你感觉更熟悉，然后将每个符号与其西班牙语名称匹配。
