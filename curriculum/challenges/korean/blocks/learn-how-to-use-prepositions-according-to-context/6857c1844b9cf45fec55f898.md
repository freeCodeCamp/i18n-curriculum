# --explanation--

`Lastly`는 마지막 항목이나 요점을 소개합니다. 예를 들어:

`Lastly, please remember to lock the door.` – 이것은 마지막 지시임을 보여줍니다.

`Ethernet cables`는 컴퓨터와 다른 장치를 네트워크에 연결하는 데 사용하는 선입니다. 예를 들어:

`We need more Ethernet cables for the new desks.` – 이것은 더 많은 네트워크 케이블이 필요함을 의미합니다.

`Ended up`는 누군가나 무언가가 결국 어디에 있었는지를 의미합니다. 예를 들어:

`The documents ended up on my desk.` – 이것은 그것들이 결국 발견된 곳임을 의미합니다.
