# --description--

Nas lições anteriores, você foi apresentado à visualização de dados, que é o processo de criar formatos visuais claros para representar dados complexos. Nesta lição, você vai aprender sobre a biblioteca D3.

A biblioteca D3 é uma biblioteca gratuita e de código aberto que você pode usar para criar visualizações de dados atraentes e eficazes em JavaScript.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-1.png" alt="Um mapa coroplético verde dos Estados Unidos mostrando dados em nível de condado, com uma escala de cores que vai do verde claro (3%) ao verde escuro (60%+)">


O nome D3 é uma abreviação para data-driven documents, referindo-se a como ela vincula dinamicamente dados a elementos no Modelo de Objeto do Documento (DOM). Essa biblioteca é flexível e funciona naturalmente com a web. Em vez de ter que aprender uma representação gráfica totalmente nova, você pode construir visualizações de dados com elementos `svg` e `canvas`.

Por exemplo, se você quiser criar um gráfico de barras usando D3, pode fazer o seguinte tudo em um arquivo JS:

* Declarar variáveis que representam as dimensões e margens do gráfico
    
* Declarar a escala de posição horizontal
    
* Declarar a escala de posição vertical
    
* Criar o container SVG com as barras e rótulos
    
<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-2.png" alt="Um gráfico de barras intitulado Produto Interno Bruto dos Estados Unidos mostrando o Produto Interno Bruto de 1947 a 2015, com valores subindo de quase zero para aproximadamente 18.000">

Então, quais são algumas outras razões para aprender D3 em vez de outras ferramentas de visualização como Chart.js, Plotly ou Tableau?

Diferente de outras ferramentas, D3 oferece muito controle de baixo nível, onde você pode construir visualizações personalizáveis e interativas usando apenas HTML, CSS e JavaScript puros. Além disso, a biblioteca é em JavaScript, então você não precisa se preocupar em aprender uma linguagem totalmente nova.

Nas próximas oficinas e laboratórios, você vai aprender os fundamentos da biblioteca D3, como seleções, formas e escalas. Você vai aprender esses conceitos-chave construindo gráficos de dispersão, gráficos de barras, apps de painel e muito mais.
