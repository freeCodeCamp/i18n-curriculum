# --description--

Vitendakazi ni vipande vya msimbo vinavyoweza kutumika tena vinavyotekelezwa unapovitoa wito. Lugha nyingi za programu zina vitendakazi vilivyojengwa ndani vinavyorahisisha kuanza. Python si tofauti, na tayari tumeshazungumzia baadhi ya vitendakazi vilivyojengwa ndani kama `print()` katika mafundisho yaliyopita.

Kitendakazi kingine cha msaada kilicho jengwa ndani ni `input()`, ambacho hukuruhusu kuomba mtumizi kutoa ingizo:

```python
name = input('What is your name?') # User types "Kolade" and presses Enter  
print('Hello', name) # Output: Hello Kolade
```

Kwa upande mwingine, `int()` hubadilisha nambari, thamani ya kweli au siyo kweli, na mfuatano wa herufi wa nambari kuwa nambari kamili:

```python
print(int(3.14)) # 3
print(int('42')) # 42
print(int(True)) # 1
print(int(False)) # 0 
```

Pia unaweza kuandika vitendakazi vyako maalum. Ili kufanya hivyo, tumia neno kuu la `def`, ikifuatiwa na jina unalotaka kumpa kitendakazi chako, mabano ya kawaida, na nukta. Kisha kwenye mstari mpya, andika msimbo ambao kitendakazi chako kinapaswa kuutekeleza. Msimbo unaotekelezwa na kitendakazi pia huitwa mwili wa kitendakazi.

Hapa kuna mfano wa kitendakazi maalum kinachoitwa `hello` kinachochapisha mfuatano wa herufi `Hello World` kwenye terminali:

```python
def hello():
    print('Hello World')
```

Ili kuendesha kitendakazi, unahitaji kukitoa wito kwa jina lake likifuatiwa na mabano ya kawaida:

```python
hello() # Hello World
```

Angalia kivunjiko cha mstari kabla ya `print('Hello World')`. Kama unavyoweza kukumbuka kutoka kwa mafundisho yaliyopita, Python hutegemea kivunjiko cha mstari kuamua ni makundi gani ya tamko yanayohusiana. Makundi haya ya tamko huitwa blokki za msimbo.

Hapa kuna kitendakazi kingine rahisi kinachochapisha jumla ya nambari mbili kwenye terminali:

```python
def calculate_sum(a, b):
    print(a + b)
```

Unaweza kuona kuwa kitendakazi chetu, `calculate_sum`, kina `a` na `b` ndani ya mabano yake, yamegawanywa kwa koma. Hivi huitwa vigezo. Fikiria vigezo kama vigezo vya kiwakilishi vinavyofanya kazi kama "nafasi" kwa thamani unazozipitisha kwenye vitendakazi unapovitoa wito.

Ili kutumia vigezo, lazima upitishe "hoja". Hoja ni thamani unazozipitisha kwenye kitendakazi unapokitoa wito.

Hivi ndivyo unavyotoa wito kwa kitendakazi cha `calculate_sum` ili kujumlisha nambari `3` na `1`:

```python
calculate_sum(3, 1) # 4
```

Kama utatoa wito kwa kitendakazi bila idadi sahihi ya hoja, utapata `TypeError`:

```python
calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

Vitendakazi pia hutumia neno kuu maalum la `return` kuondoka kwenye kitendakazi na kurudisha thamani. Kama hutumiwi waziwazi `return`, Python itarudisha `None` kwa chaguo-msingi.

Hapa kuna mfano:

```python
def calculate_sum(a, b):
    print(a + b)

my_sum = calculate_sum(3, 1) # 4
print(my_sum) # None
```

Unaweza kuona kuwa kitendakazi cha `calculate_sum` kinachapisha jumla ya `a` na `b`, lakini hakirudishi chochote waziwazi. Hivyo tunapoweka matokeo yake kwenye `my_sum`, thamani ni `None`. Ili kurekebisha hilo, unaweza kutumia neno kuu la `return` kurudisha matokeo:

```python
def calculate_sum(a, b):
    return a + b

my_sum = calculate_sum(3, 1)
print(my_sum) # 4
```

Sasa, `calculate_sum` inarudisha jumla ya `a` na `b`, ambayo huhifadhiwa katika `my_sum`.
