# --description--

`It depends on` se usa para decir que la respuesta a una pregunta puede cambiar basándose en diferentes cosas. Por ejemplo:

Cuánto tiempo lleva hacer un sitio web depende de qué tan grande o pequeño sea el trabajo. Elegir el lenguaje de codificación depende del ordenador o sistema que usemos.

Nota que después de `depends` siempre está la preposición `on`.
