## 의사 코드

의사 코드는 코드를 쓰는 대신 자연어로 프로그램의 논리를 작성하는 것입니다. 문제를 해결하기 위해 프로그램이 거쳐야 할 단계를 천천히 생각하는 데 도움이 됩니다.

입력한 숫자까지 모든 숫자를 출력하는 프로그램의 의사 코드 예시는 다음과 같습니다:

```md
When the user inputs a number
Initialize a counter variable and set its value to zero
While counter is smaller than user inputted number increment the counter by one
Print the value of the counter variable
```

이것은 의사 코드가 어떻게 생겼는지 보여주기 위한 기본 프로그램입니다. 과제에 더 많은 의사 코드 예시가 포함될 예정입니다.
