# --explanation--

`To work hard` は多くの努力を注ぐことを意味します。この場合、`hard` の副詞も `hard` であり、`hardly` ではありません。`To hardly work` はほとんど努力しない、ほとんど働かないことを意味し、マリアがチームについて言っていることとは正反対です。例えば：

- `She hardly trained for the marathon.` - これは彼女がマラソンの準備に全く努力しなかったことを意味します。

- `She trained hard for the marathon.` - これは彼女が準備に多くの努力を注いだことを意味します。
