# --description--

你为超文本标记语言中的某些元素添加了 `center` 类。现在，你应该定义该类以使这些元素居中。

定位 `center` 类，并将具有该类且 `text-align` 属性设置为 `center` 的元素居中。

# --hints--

你应该将目标定位到具有 `center` 类的元素。

```js
assert(new __helpers.CSSHelp(document).getStyle('.center'));
```

你应该将 `text-align` 属性设置为 `center`。

```js
const style = new __helpers.CSSHelp(document).getStyle('.center')
assert.strictEqual(style?.textAlign, 'center')
```
