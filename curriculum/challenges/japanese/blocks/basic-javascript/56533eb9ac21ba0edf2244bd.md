# --description--

<dfn>パラメータ</dfn>は、関数が呼び出されたときに入力される値のプレースホルダーとして機能する変数です。関数を定義するときは、通常1つ以上のパラメータとともに定義されます。関数が呼び出されたときに実際に入力（または<dfn>「渡される」</dfn>）される値は、<dfn>引数</dfn>と呼ばれます。

以下は2つのパラメータ、`param1` と `param2` を持つ関数です：

```js
function testFun(param1, param2) {
  console.log(param1, param2);
}
```

次に、`testFun` をこのように呼び出せます：`testFun("Hello", "World");`。2つの文字列引数、`Hello` と `World` を渡しました。関数内では、`param1` は文字列 `Hello` と等しくなり、`param2` は文字列 `World` と等しくなります。異なる引数で `testFun` を再度呼び出すこともでき、その場合パラメータは新しい引数の値を取ります。
