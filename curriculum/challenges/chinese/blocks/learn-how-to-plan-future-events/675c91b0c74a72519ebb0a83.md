# --explanation--

`They'll` 是 `they will` 的缩写。它用于谈论预期或预测将来发生的事情。 

`Likely` 意味着某事很可能发生或有很高的发生几率。例如：

- `They'll likely ask for more information.` - 这预测了客户可能的请求。

- `The feature will likely need additional testing.` - 这表示很可能需要进行额外的测试。
