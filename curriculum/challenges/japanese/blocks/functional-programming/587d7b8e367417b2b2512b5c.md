# --description--

FCCチームは気分が変わり、今は緑茶と紅茶の2種類のお茶を欲しがっています。一般的な事実として、クライアントの気分の変動はよくあります。

この情報を踏まえて、前回のチャレンジで使った`getTea`関数を見直し、さまざまなお茶のリクエストに対応できるようにしましょう。`getTea`を修正して、準備するお茶の種類を変えられるように関数をパラメータとして受け取れるようにします。これにより`getTea`がより柔軟になり、クライアントのリクエストが変わったときにプログラマーがよりコントロールしやすくなります。

まずは、関数型プログラミングの用語を説明します。

<dfn>コールバック</dfn>は、別の関数に渡されて、その関数の呼び出しを決定するために使われる関数です。例えば`filter`のように、配列をフィルターする基準をJavaScriptに伝えるためにコールバック関数が他のメソッドに渡されることがあります。

変数に代入したり、別の関数に渡したり、戻り値として返したりできる関数は、<dfn>ファーストクラス</dfn>関数と呼ばれます。JavaScriptではすべての関数がファーストクラス関数です。

関数を引数として受け取ったり、戻り値として関数を返す関数は、<dfn>高階関数</dfn>と呼ばれます。

関数が別の関数に渡されたり、戻り値として返されたりするとき、その渡されたり返されたりした関数は<dfn>ラムダ</dfn>と呼ばれます。
