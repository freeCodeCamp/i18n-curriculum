# --interactive--

عند العمل مع السلاسل النصية في JavaScript، هناك العديد من الحالات التي قد تحتاج فيها إلى التحقق مما إذا كانت السلسلة تحتوي على سلسلة فرعية محددة، وهي جزء أصغر من تلك السلسلة.

على سبيل المثال، قد ترغب في التحقق مما إذا كان مُدخَل المستخدم يتضمن كلمة أو حرفًا معينًا قبل تنفيذ إجراء ما. إحدى الطرق لتحقيق ذلك هي استخدام الطريقة `includes()`.

تُستخدم الطريقة `includes()` للتحقق مما إذا كانت السلسلة تحتوي على سلسلة فرعية محددة. إذا وُجدت السلسلة الفرعية داخل السلسلة، تُرجع الطريقة `true`، وإلا تُرجع `false`.

إليك الصيغة النحوية الأساسية:

```js
string.includes(searchValue);
```

في الصيغة، `searchValue` هي السلسلة الفرعية التي تريد البحث عنها داخل السلسلة. وإليك مثالًا:

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("awesome");

console.log(result);  // true
```

:::

في هذا المثال، الكلمة `awesome` موجودة داخل السلسلة `JavaScript is awesome!`، لذلك تُرجع الطريقة `includes()` القيمة `true`.

من المهم ملاحظة أن الطريقة `includes()` تميز بين الأحرف الكبيرة والصغيرة. هذا يعني أن التطابق الدقيق للأحرف مطلوب، بما في ذلك حالة الأحرف.

على سبيل المثال:

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("Awesome");

console.log(result);  // false
```

:::

بما أن `Awesome` (بحرف `A` كبير) لا يطابق `awesome` (بحرف `a` صغير)، فالنتيجة هي `false`.

يمكنك أيضًا استخدام الطريقة `includes()` للتحقق من وجود سلسلة فرعية بدءًا من فهرس معين في السلسلة عبر توفير معلمة ثانية:

:::interactive_editor

```js
let text = "Hello, JavaScript world!";
let result = text.includes("JavaScript", 7);

console.log(result);  // true
```

:::

هنا، يبدأ البحث عن السلسلة الفرعية `JavaScript` من الموضع السابع في السلسلة، مما يضمن تخطي أي أحرف قبل هذا الموضع.

تُرجع الطريقة `includes()` فقط نتيجة `true` أو `false`. لا توفر معلومات عن مكان وجود السلسلة الفرعية في السلسلة أو عدد مرات ظهورها. إذا كنت تحتاج إلى هذا المستوى من التفاصيل، قد تكون طرق أخرى مثل الطريقة `indexOf()` أكثر ملاءمة.
