# --explanation--

이 문장은 **수동태**를 사용하여, 누가 행동을 했는지보다 행동을 받는 대상에 초점을 맞춥니다. 예를 들어:  

`The navigation menu was appreciated.` – 사람들이 그것을 좋아했다는 의미지만, 누가 좋아했는지는 말하지 않습니다.  

**능동태**에서는 `The users appreciated the navigation menu.` – 누가 행동을 했는지 명확히 나타냅니다.

**수동태**는 행동을 한 사람이 알려지지 않았거나 중요하지 않을 때 자주 사용합니다. 또 다른 예:

`The report was completed on time.` – 보고서가 완료되었음을 알지만, 누가 완료했는지는 알 필요도 없고 말할 필요도 없습니다.
