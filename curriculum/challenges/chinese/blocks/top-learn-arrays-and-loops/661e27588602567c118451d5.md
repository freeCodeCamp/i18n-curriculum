# --description--

既然你已经对数组有了基本的了解，现在让我们来谈谈循环。循环用于多次执行一段代码块。其中一种循环是 `while` 循环。`while` 循环只要条件为真就执行一段代码块。`while` 循环的语法如下：

```javascript
while (condition) {
  // code block to be executed
}
```

例如，以下代码截图打印从 1 到 5 的数字：

```javascript
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
```

上面的代码截图初始化了一个变量 `i`，其值为 `1`。只要 `i` 的值小于或相等于 `5`，`while` 循环就会执行该代码块。每次迭代中，`i` 的值都会增加 `1`。
