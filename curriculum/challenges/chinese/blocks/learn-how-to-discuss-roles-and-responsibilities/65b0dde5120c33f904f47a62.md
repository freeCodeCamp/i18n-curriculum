# --description--

*现在完成时* 由 `have` 或 `has` 加动词的过去分词构成。例如，对于不规则动词 `meet`，它的过去分词是 `met`。这个时态用于谈论过去的经历或现在仍然重要的动作。

对代词 `I`、`you`、`we` 和 `they` 使用 `have`，对 `he`、`she` 和 `it` 使用 `has`。

单词 `yet` 经常用于 `Present Perfect Tense` 的疑问句中，以询问某事是否发生到现在为止。
