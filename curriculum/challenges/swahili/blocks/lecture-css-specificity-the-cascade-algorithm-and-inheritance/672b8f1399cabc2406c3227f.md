# --interactive--

Neno muhimu la `!important` katika CSS linatumika kutoa kipaumbele cha juu zaidi kwa kanuni ya mtindo, na hivyo kuweza ghairi tamko lolote lingine la sifa.

Linapotumika, linamfanya kivinjari kutekeleza mtindo uliobainishwa, bila kujali uhalisia wa vichaguzi vingine.

Tuseme una kipengele cha aya cha HTML chenye mitindo ndani ya mstari kama ifuatavyo:

:::interactive_editor

```html
<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

:::

Katika mfano huu, kipengele cha aya kina rangi ya nyuma ya `lightblue` na rangi ya maandishi ya `black`.

Faili ya CSS inatumia mitindo ifuatayo kwa kipengele cha aya:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black;
  color: white;
}
```

:::

Kwa kuwa mitindo ndani ya mstari ina kipaumbele cha juu zaidi kuliko vichaguzi vya darasa la vitu, kitambulisho na aina, rangi ya nyuma nyeusi na rangi ya maandishi meupe haitatumika kwa kipengele hicho cha aya.

Ili ghairi mitindo hiyo ndani ya mstari, unaweza kutumia neno muhimu la `!important` kama ifuatavyo:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black !important;
  color: white !important;
}
```

:::

Neno muhimu la `!important` hutumika baada ya thamani ya CSS na kabla ya alama ya nukta ya nusu.

Sasa kipengele cha aya kitakuwa na rangi hizo nyeusi na nyeupe kwa sababu mitindo ndani ya mstari imeghairiwa kwa kutumia neno muhimu la `!important`.

Neno muhimu la `!important` katika CSS linatumika kutoa kipaumbele cha juu zaidi kwa kanuni ya mtindo, na kwa ufanisi kughairi tamko lingine lolote, ikiwemo zile zenye uhalisia wa juu zaidi na mitindo ndani ya mstari.

Hata hivyo, neno muhimu la `!important` halibadilishi uhalisia wa kichaguzi cha CSS chenyewe. Linahakikisha tu kwamba kanuni yenye `!important` inatekelezwa, hata kama kuna kanuni zingine zinazopingana zenye uhalisia wa juu zaidi.

Matumizi mengine yanayofaa ya neno muhimu la `!important` ni kughairi mitindo kutoka maktaba za watu wengine au mifumo ya kazi wakati huna udhibiti wa CSS ya awali.

Hata hivyo, kutumia neno muhimu la `!important` kupita kiasi kunaweza kusababisha ugumu katika kudumisha na kurekebisha makosa ya CSS yako, kwa sababu huvunja mtiririko wa kawaida wa mitindo na inaweza kusababisha matokeo yasiyotarajiwa.

Kwa hivyo, ni bora kutumia neno muhimu la `!important` kwa kiasi kidogo.
