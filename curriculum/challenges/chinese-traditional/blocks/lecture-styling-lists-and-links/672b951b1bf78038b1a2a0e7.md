# --interactive--

在 CSS 中，`list-style` 屬性用於控制網頁上列表的外觀。

無論你是在處理有序列表（`ol`）或無序列表（`ul`），`list-style` 屬性都允許你自訂列表項目的顯示方式。

`list-style` 屬性實際上是三個其他屬性的縮寫：

- `list-style-type`
- `list-style-position`
- `list-style-image`

每一個在定義列表外觀時扮演不同的角色。

`list-style-type` 屬性允許你定義列表中使用的項目符號或數字的型別。

對於無序列表，你可以從多種項目符號樣式中選擇，例如實心圓、空心圓或方塊。

對於有序列表，你可以使用不同的編號系統，例如十進位、羅馬數字，甚至字母字元。

以下是使用 `list-style-type` 的範例：

:::interactive_editor

```html
<ul style="list-style-type: square;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

在此範例中，無序列表的項目符號被更改為方形。

`list-style-type` 屬性是這三個中最常使用的，因為它直接影響你的列表中項目符號或編號樣式的外觀。

`list-style-position` 屬性控制項目符號或編號相對於列表項目內容的位置。你可以使用兩個值：`inside` 和 `outside`。

當你使用值 `outside` 時，項目符號或編號會出現在內容外側，這是預設行為。

而且，當你使用值 `inside` 時，項目符號或數字會出現在內容內部，這可能會導致文字換行並與項目符號或數字對齊。

以下是使用 `list-style-position` 的範例：

:::interactive_editor

```html
<ul style="list-style-position: inside;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
<ul style="list-style-position: outside;">
  <li>Item 4</li>
  <li>Item 5</li>
  <li>Item 6</li>
</ul>
```

:::

在此範例中，為兩個不同的無序列表頁籤提供了 `inside` 和 `outside` 兩個值。

當你想要控制列表內容的對齊方式時，`list-style-position` 屬性會很有用，特別是在單一列表項目中有多行文字的情況下。

`list-style-image` 屬性允許你將影像作為列表項目的項目符號。這對於為你的列表添加獨特的視覺風格非常有用。

以下是使用 `list-style-image` 的範例：

:::interactive_editor

```html
<head>
  <style>
    ul {
      list-style-image: url('https://cdn.freecodecamp.org/platform/universal/freecodecamp-org-gravatar.jpeg');
      list-style-position: inside;
    }
  </style>
</head>
<body>
  <ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
  </ul>
</body>
```

:::

在此範例中，項目符號被訂製的 freeCodeCamp 標誌取代，為列表添加了個人化的風格。

使用 `list-style-image` 時，請確保你選擇的影像是小且適合你的網頁設計。如果影像太大或太複雜，會使列表較難閱讀。

你可以將三個屬性 – `list-style-type`、`list-style-position` 和 `list-style-image` – 組合成單一的 `list-style` 簡寫屬性。

簡寫中值的順序不重要，但三者都可以一起指定。

以下是使用縮寫屬性的範例：

:::interactive_editor

```html
<ul style="list-style: square inside url('https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg');">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

在此範例中，列表項目使用方形符號，位置在內容內部，並以訂製影像作為符號。

但是，如果影像不可用或未呈現，將使用方形項目符號作為備用。
