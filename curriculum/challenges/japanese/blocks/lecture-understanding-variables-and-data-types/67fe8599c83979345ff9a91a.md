# --description--

すべてのプログラミング言語には、組み込みのメソッド、関数、プロパティ、またはキーワードを使ってターミナルにデータを出力する方法があります。Pythonでは、`print`関数を使ってデータをターミナルに出力できます。`print`関数を詳しく見て、安心して使い始められるようにしましょう。

どのプログラミング言語を学ぶときも、最初にすることの一つはシンプルな`Hello world!`プログラムを書くことです。Pythonでは`print`関数だけでそれが簡単にできます。

そのためには、`Hello world!`関数を呼び出すために使う開き丸括弧と閉じ丸括弧の間に文字列`print`を入れるだけです：

```python
print('Hello world!') # Hello world!
```

Pythonの文字列や関数については今後のレッスンでさらに学びます。今は文字列を、シングルクォーテーション（`'`）またはダブルクォーテーション（`"`）で囲まれた文字のシーケンスと考えてください。

`print('Hello world!')`の例では、文字列`'Hello world!'`が`print`関数に渡された**引数**です。複数の値や引数を一度に表示したい場合は、カンマで区切って`print`関数を使うこともできます。例えば：

```python
print('My favorite colors are', 'blue', 'green', 'red')

# Output: My favorite colors are blue green red
```

Pythonはカンマで区切られた各項目の間に自動的にスペースを追加します。これは複数の情報を一緒に出力したいときに便利です。
