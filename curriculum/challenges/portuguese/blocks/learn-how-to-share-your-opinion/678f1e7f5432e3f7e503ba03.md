# --explanation--

`I see` é usado para mostrar que você entende o que alguém está dizendo, mas não significa necessariamente que você concorda.Por exemplo:  

`I see why you prefer that framework, but I still think we should compare other options.` – Isso significa que você reconhece a opinião da outra pessoa enquanto mantém uma discussão aberta.  

`Sure` é usado quando alguém está confiante ou certo sobre algo.Por exemplo:

`I'm sure this new programming language will improve our development speed.` - Isso significa que você está confiante de que a nova linguagem tornará o desenvolvimento mais rápido, com base no seu conhecimento ou experiência.
