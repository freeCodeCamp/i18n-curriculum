# --description--

La frase `crazy part` se usa para expresar que algo es sorprendente o inesperado. Es como decir `the surprising thing is` o `the unexpected thing is`.

Por ejemplo, `The crazy part is that he finished all the work in one day.` Esto significa que es sorprendente que pudiera hacerlo tan rápidamente.
