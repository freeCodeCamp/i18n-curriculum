# --description--

對所有大於或等於 `1` 的數字，創建一個 `else` 區塊，並在其中呼叫 `countup` 函式（程式）並傳入 `number - 1`，將結果指定給 `countArray`。

然後使用 `.push()` 將 `number` 添加到 `countArray`，最後傳回 `countArray`。

這確保遞迴最終會達到基底案例並終止。

assert.match(countup.toString(), /countArray\s*=\s*countup\(\s*number\s*-\s*1\s*\)/);

```js
assert.deepStrictEqual(countup(3), [1, 2, 3]);
```
