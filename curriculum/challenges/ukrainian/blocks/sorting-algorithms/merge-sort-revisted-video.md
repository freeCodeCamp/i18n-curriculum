# --questions--

## --text--

Яка основна відмінність між алгоритмами сортування злиттям і швидким сортуванням?

## --answers--

Сортування злиттям багаторазово міняє місцями сусідні елементи, доки не відсортує.

---

Швидке сортування розбиває список навколо опорного елемента і рекурсивно сортує підсписки, тоді як сортування злиттям розбиває список навпіл і об’єднує відсортовані половини.

---

Сортування злиттям використовує хеш-таблицю для підрахунку елементів, тоді як швидке сортування використовує двонаправлену чергу для перестановки елементів.

---

Швидке сортування завжди використовує цикл while для сортування, тоді як сортування злиттям використовує цикл for.
