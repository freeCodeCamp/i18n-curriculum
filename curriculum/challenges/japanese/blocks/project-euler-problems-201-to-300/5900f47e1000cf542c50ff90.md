# --description--

次の形の方程式を考えます：$a^2 + b^2 = N$、$0 ≤ a ≤ b$、$a$、$b$、$N$ は整数。

$N = 65$ の場合、2つの解があります：

$a = 1, b = 8$ と $a = 4, b = 7$。

すべての解 $a^2 + b^2 = N$、$0 ≤ a ≤ b$、$a$、$b$、$N$ が整数であるときの $a$ の値の合計を $S(N)$ と呼びます。

したがって $S(65) = 1 + 4 = 5$ です。

$4k + 1$ の形の素数で割り切れ、かつ $4k + 1 < 150$ である平方因子なしのすべての $N$ について、$\sum S(N)$ を求めてください。
