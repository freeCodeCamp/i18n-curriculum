## Python in Ihrer lokalen Umgebung

- **Installation**: Der beste Weg, Python auf Windows, Mac und Linux zu installieren, ist das Herunterladen des Installationsprogramms von der offiziellen Python-Website (`https://www.python.org/`).
- **Terminal**: Eine textbasierte Schnittstelle, mit der Sie durch Eingabe von Befehlen mit Ihrem Computer interagieren können. Jedes Betriebssystem verfügt über eine Standard-Terminal-App. Auf macOS können Sie die Terminal-App verwenden. Unter Windows können Sie die Eingabeaufforderung oder PowerShell nutzen. Unter Linux hat jede Desktop-Umgebung ihre eigene Standard-Terminal-App, wie GNOME Terminal oder Konsole.
- **IDE**: IDE steht für Integrierte Entwicklungsumgebung und bietet Funktionen wie einen Code-Editor, Testwerkzeuge, ein Terminal und mehr.
- **Beliebte Code-Editoren und IDEs für Python**: VS Code, PyCharm und Spyder
- **Code lokal ausführen**: Um Python-Code auszuführen, haben Sie mehrere Möglichkeiten. Die erste Möglichkeit ist, auf die Schaltfläche „Ausführen“ in VS Code zu klicken, die wie ein Play-Button aussieht und sich oben rechts befindet. Dadurch wird ein Terminal geöffnet und Ihr Python-Skript dort ausgeführt.

Eine weitere Möglichkeit ist, ein Terminal zu öffnen und das Programm manuell auszuführen. Hier ist ein Beispiel, um ein Programm namens `main.py` auszuführen:

```bash
python main.py
```

Sie müssen diesen Befehl aus dem Ordner ausführen, in dem `main.py` gespeichert ist. Wenn sich `main.py` beispielsweise in einem Ordner namens `python-projects` befindet, verwenden Sie zuerst Folgendes im Terminal:

```bash
cd python-projects
```

Dann führen Sie aus:

```bash
python main.py
```

## Verwendung der Python-Interactive-Shell

- **Definition**: Eine interaktive Shell ist ein Programm, mit dem Sie Befehle einzeln eingeben und die Ergebnisse sehen können. Öffnen Sie eine Terminal-App, geben Sie `python` ein und drücken Sie `Enter`. Dadurch wird eine Python-Interactive-Shell gestartet. Das Symbol `>>>` bedeutet, dass Python darauf wartet, dass Sie einen Befehl eingeben. Versuchen Sie, `"Hello, world!"` im Terminal auszugeben:

```bash
print("Hello, world!")
```

Sie sollten den Text wie folgt sehen:

```bash
>>> print("Hello, world!")
Hello, world!
```

Nachdem der Text ausgegeben wurde, kehrt Python zu folgendem zurück:

```bash
>>>
```

Das bedeutet, Sie können einen weiteren Befehl eingeben.

- **Read, Evaluate, Print, Loop-Zyklus (REPL)**: Immer wenn Sie Befehle eingeben, liest der Interpreter die Eingabe, wertet sie aus, gibt das Ergebnis aus und kehrt zurück, um das `>>>` anzuzeigen, damit Sie weitere Befehle eingeben können.
