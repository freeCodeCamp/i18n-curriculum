# --description--

Pythonでは、クラスとオブジェクトは連携してデータを整理・管理します。クラスを作成して共通の振る舞いを定義し、その振る舞いを使うオブジェクトを作成します。

つまり、クラスはオブジェクトを作るための設計図やテンプレートのようなものです。

クラスとは何か、そしてそれを使ってオブジェクトを作成する方法を見ていきましょう。

クラスを作成するには、`class` キーワードの後にクラス名とコロンを続けて書きます。クラス内には初期化メソッドや属性、メソッドを追加できます。

属性はクラス内の変数のようなもので、データをストアするために使います。メソッドはクラス内で定義された関数で、クラスから作成されたオブジェクトが実行できる動作です。

クラスの基本的な構文は以下の通りです。

```python
class ClassName:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def sample_method(self):               
        print(self.name.upper())
```

* `class ClassName` は、クラスを作成するための `class` キーワードと、ここでは `ClassName` と呼ばれるクラス名で構成されています。Pythonではクラス名に **PascalCase** の慣例を使うことが一般的です。
    
* `def __init__(self, name, age)` は新しいオブジェクトが作成されると自動的に呼び出される特別なメソッドです。クラスから作成されるオブジェクトの属性を初期化します。
    
    さらに、`__init__` の最初のパラメータは常に作成または使用される特定のオブジェクトへの参照です。慣例としてこのパラメータは `self` と名付けられていますが、技術的には任意の名前を使えます。`self` を使うとオブジェクト自身の属性やメソッドにアクセスできます。
    
* `self.name = name` と `self.age = age` はオブジェクトが持つ属性です。
    
* `def sample_method(self):` は作成された各オブジェクトが呼び出せるメソッドです。
    
* `print(self.name.upper())` は `sample_method` メソッドが行う処理で、この場合は名前を大文字で出力します。
    

もし多く感じても心配いりません。`Dog` クラスの似た例を見て、そこからオブジェクトを作成する方法を確認しましょう。

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name.upper()} says woof woof!")
```

この `Dog` クラスを使ってオブジェクトを作成できます。クラスからオブジェクトを作成する基本的な構文は以下の通りです。

```python
object_1 = ClassName(attribute_1, attribute_2)
object_2 = ClassName(attribute_1, attribute_2)
```

また、各オブジェクトからクラスで定義されたメソッドを呼び出すこともできます。

```python
object_1.method_name()
object_2.method_name()
```

では、`Dog` クラスを設計図として使い、2匹の犬を作成してみましょう。

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name.upper()} says woof woof! I'm {self.age} years old!")

dog_1 = Dog("Jack", 3)
dog_2 = Dog("Thatcher", 5)

# Call the bark method
dog_1.bark()  # JACK says woof woof! I'm 3 years old!
dog_2.bark()  # THATCHER says woof woof! I'm 5 years old!
```

ご覧の通り、`Dog` クラスを使って2つの犬オブジェクトを作成しています。`dog_1` を初期化するときに文字列の `Jack` と数値の `3` が渡され、それぞれのインスタンスの `name` と `age` 属性が設定されます。そして `dog_2` は文字列の `Thatcher` と数値の `5` を `name` と `age` として初期化しています。

その後、`.bark()` と `dog_1` の `dog_2` メソッドを呼び出すと、両者の出力が異なり、オブジェクト作成時に渡した固有の `name` と `age` 属性を使っていることがわかります。

まとめると、クラスとオブジェクトの違いは、クラスがテンプレートや設計図であり、オブジェクトはそのテンプレートを使って作られたものだということです。

また、クラスはオブジェクトが持つべきデータと振る舞いを定義し、オブジェクトは実際のデータを保持し、その振る舞いを使います。クラスは一度書けば、異なるデータを持つ多くのオブジェクトを作成できます。
