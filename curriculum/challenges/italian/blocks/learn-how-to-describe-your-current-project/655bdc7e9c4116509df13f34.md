# --description--

`use` significa destinare qualcosa a uno scopo particolare. Per esempio:

`She uses a notebook to take notes in class.` - Questo mostra come lei utilizza il quaderno per scrivere.

Un `tool` è qualcosa che ti aiuta a svolgere un lavoro. Nel lavoro, gli strumenti sono spesso software o cose sul computer. Per esempio:

`Zoom is a tool for online meetings.` - In questo caso, lo strumento è un software che aiuta le persone a connettersi e parlare.
