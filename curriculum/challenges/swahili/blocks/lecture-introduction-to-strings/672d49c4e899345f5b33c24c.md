# --interactive--

Mafunzo yaliyopita yalikuanzisha kwa `console.log()` lakini funzo hili litaingia zaidi katika kusudi na matumizi yake.

Katika JavaScript, `console.log()` ni chombo rahisi lakini chenye nguvu kinachotumika kuonyesha ujumbe au kutoa taarifa kwenye konsoli ya kivinjari. Kawaida hutumika na wasanidi programu kutafuta makosa na kuchunguza msimbo wakati wa kufanya kazi kwenye programu zao.

Unaweza kutumia `console.log()` kuandika maandishi au vigezo kwenye konsoli na kuhakikisha msimbo wako unafanya kazi ipasavyo.

Ili kutumia `console.log()`, unaita njia hiyo ukiweka thamani au ujumbe unaotaka kuutoa ndani ya mabano ya kawaida. Hapa kuna mifano kadhaa:

:::interactive_editor

```js
console.log("Hello, world!");

let num = 5;
console.log(num); // 5
```

:::

Mfano wa kwanza unachapisha `Hello, world!` kwenye konsoli ya kivinjari, wakati mfano wa pili unachapisha thamani `5`.

Hapa kuna mfano mwingine wa kufanya kazi na `console.log()`:

:::interactive_editor

```js
let name = "Alice";
console.log("Hello, " + name + "!"); // Hello, Alice!
```

:::

Unaweza pia kupitisha thamani nyingi kwa `console.log()` zikitenganishwa kwa koma. Kwa mfano:

:::interactive_editor

```js
let name = "Alice";
let age = 25;
console.log("Name:", name, "Age:", age); // Name: Alice Age: 25
```

:::

Hii ni msaada kwa kuandika taarifa nyingi kwa wakati mmoja.

Njia ya `console.log()` inakusaidia kufuatilia msimbo wako unavyotekelezwa, na kufanya iwe rahisi kugundua makosa na kuelewa jinsi programu yako inavyotenda.
