# --description--

次の方程式は山岳地帯の連続した地形を表しており、任意の点 ($x$,$y$) における標高 $h$ を示しています：

$$h = \left(5000 - \frac{x^2 + y^2 + xy}{200} + \frac{25(x + y)}{2}\right) \times e^{-\left|\frac{x^2 + y^2}{1\\,000\\,000} - \frac{3(x + y)}{2000} + \frac{7}{10}\right|}$$

蚊は A(200,200) から B(1400,1400) まで、$0 ≤ x$, $y ≤ 1600$ の範囲を出ることなく飛ぼうとしています。

途中に山があるため、まず標高 $f$ の点 A' まで真っ直ぐ上昇します。次に、同じ標高 $f$ を保ちながら障害物を回避し、B の真上にある点 B' に到達します。

まず、指定された範囲内で A から B までの移動を可能にする最小の一定標高 $f_{min}$ を求めてください。次に、その一定標高 $f_{min}$ で A' から B' までの最短経路の長さを求めてください。

答えはその長さを小数点以下3桁に丸めてください。

**注意:** 利便性のため、上記の標高関数は以下に多くのプログラミング言語で使いやすい形で繰り返しています：`h=( 5000-0.005*(x*x+y*y+x*y)+12.5*(x+y) )* exp( -abs(0.000001*(x*x+y*y)-0.0015*(x+y)+0.7) )`。
