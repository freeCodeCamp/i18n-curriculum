# --interactive--

الهامش والحشو خاصيتان أساسيتان في CSS لإنشاء صفحات ويب منظمة، قابلة للقراءة، وجذابة بصريًا.

تتحكم الهوامش في المساحة خارج العنصر، مما يساعد على فصله عن العناصر الأخرى وتحديد هيكل التخطيط، بينما يتحكم الحشو في المساحة داخل العنصر، مما يحسن من قابلية قراءة المحتوى وجاذبيته الجمالية.

لفهم الفرق بين `margin` و`padding` بشكل أفضل، دعنا نلقي نظرة على بعض الأمثلة.

إليك مثال HTML لثلاثة عناصر فقرة على الصفحة:

:::interactive_editor

```html
<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

:::

لتطبيق مسافة على الجزء العلوي من كل عنصر فقرة، يمكنك استخدام الخاصية `margin-top` بهذه الطريقة:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  margin-top: 20px;
  border: 2px solid black;
}
```

:::

في هذا المثال، نُعيّن هامشًا بمقدار `20px` فقط على الجزء العلوي من كل عنصر فقرة.

كما أضفنا حدودًا سوداء صلبة على جميع الجوانب الأربعة لتتمكن من رؤية الهامش بشكل أفضل.

الخصائص الأربعة المختلفة لـ `margin` هي `margin-top`، `margin-right`، `margin-bottom` و`margin-left`.

إليك مثالًا على استخدام جميع الخصائص الأربعة:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 30px;
  margin-left: 40px;
  border: 2px solid black;
}
```

:::

في هذا المثال، نُعيّن قيم المسافات على جميع الجوانب الأربعة لعنصر الفقرة. كما أُضيفت حدود سوداء صلبة لتتمكن من رؤية الهوامش بشكل أفضل.

طريقة أخرى لاستخدام خاصية `margin` هي باستخدام التدوين المختصر. يمكنك تحديد قيمة واحدة أو اثنتين أو ثلاث أو أربع قيم مرة واحدة. دعنا نستكشف هذه الخيارات معًا.

عند استخدام قيمة واحدة في التدوين المختصر لـ `margin`، تُطبّق تلك القيمة نفسها على جميع الجوانب الأربعة للعنصر المستهدف.

إليك مثالًا على استخدام قيمة واحدة في التدوين المختصر لـ `margin`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px;
}
```

:::

هذا المثال من الشفرة سيطبق `10px` من `margin` بالتساوي على جميع جوانب عنصر الفقرة الأربعة.

عند استخدام قيمتين، تُطبّق القيمة الأولى على الجانبين `top` و`bottom`، بينما تُطبّق القيمة الثانية على الجانبين `left` و`right` للعنصر.

إليك مثالًا على استخدام قيمتين في التدوين المختصر لـ `margin`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px;
}
```

:::

هذا يضبط هوامش `top` و`bottom` إلى `10px`، و`left` و`right` إلى `20px` لعنصر الفقرة.

إذا وُفّرت ثلاث قيم، تُطبّق القيمة الأولى على هامش `top`، والثانية على هامش `left` و`right`، والثالثة على هامش `bottom`.

إليك مثالًا لفهم ذلك بشكل أفضل:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px;
}
```

:::

هذا يحدد الهامش ليكون `10px` في الأعلى `top`، و`20px` في الجانبين الأيسر والأيمن `left` و`right`، و`30px` في الأسفل `bottom`.

عند استخدام أربع قيم، يمنحك ذلك تحكمًا أكبر، إذ يمكنك تحديد كل قيمة هامش بشكل مستقل لكل جانب من جوانب العنصر المستهدف.

القيمة الأولى تستهدف الجزء العلوي `top`، والثانية تستهدف الجانب الأيمن `right`، والثالثة تستهدف الجزء السفلي `bottom`، والرابعة تستهدف الجانب الأيسر `left`.

إليك مثالًا على استخدام التدوين المختصر للهامش بأربع قيم:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px 40px;
}
```

:::

يُحدد هذا الهامش بقيمة `10px` للـ `top`، و`20px` للـ `right`، و`30px` للـ `bottom`، و`40px` للـ `left`.

تُستخدم خاصية `padding` لتعيين مساحة داخل العنصر، بين المحتوى وحدوده.

مثل خاصية `margin`، هناك أربع خصائص لـ `padding` وهي `padding-top`، `padding-right`، `padding-bottom` و`padding-left`.

إليك مثالًا على كيفية تعيين `padding` لعنصر فقرة:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 30px;
  padding-left: 40px;
  border: 2px solid black;
}
```

:::

هذا يحدد الحشو ليكون `10px` للـ `top`، و`20px` للـ `right`، و`30px` للـ `bottom`، و`40px` للـ `left`.

كما ترى، يُطبَّق `padding` على المحتوى داخل الحدود، على عكس `margin` الذي يُطبَّق خارج الحدود.

تمامًا مثل خاصية `margin`، يمكنك أيضًا استخدام التدوين المختصر لخاصية `padding`.

يمكنك تحديد قيمة واحدة أو اثنتين أو ثلاث أو أربع قيم في التدوين المختصر لـ `padding` أيضًا.

إليك مثالًا على استخدام التدوين المختصر لـ `padding` لعنصر الفقرة السابق:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding: 10px 20px 30px 40px;
  border: 2px solid black;
}
```

:::

في المثال، باستخدام الاختصار، سيُعيّن الحشو إلى `10px` في الـ `top`، و`20px` في الـ `right`، و`30px` في الـ `bottom`، و`40px` في الـ `left` لعنصر الفقرة.
