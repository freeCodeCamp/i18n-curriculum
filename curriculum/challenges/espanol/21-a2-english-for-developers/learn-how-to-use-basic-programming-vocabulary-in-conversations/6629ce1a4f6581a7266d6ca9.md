# --description--

El **tiempo presente perfecto continuo** se utiliza para hablar sobre acciones o situaciones que comenzaron en el pasado y todavía están sucediendo ahora. Es muy útil cuando quieres enfatizar la duración de una actividad o que es un proceso continuo.

Aquí está cómo formar el **tiempo presente perfecto continuo**: Sujeto + `has/have been` + formulario `-ing` del verbo. Este tiempo se usa a menudo con expresiones de tiempo como `for` y `since`. `For` se utiliza con un período de tiempo como dos años o seis meses, mientras que `since` se utiliza con un punto específico en el tiempo, como 2018 o esta mañana.

Ejemplos:

`I have been studying English for two years.` Esto significa que comencé a estudiar inglés hace dos años y todavía lo estoy estudiando ahora.

`She has been working here since 2018.` Significa que comenzó a trabajar aquí en 2018 y todavía está trabajando aquí.
