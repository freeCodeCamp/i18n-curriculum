# --interactive--

型別選擇器，也稱為元素選擇器，根據元素的頁籤名稱來標的元素。

這些選擇器是 CSS 中的基礎，允許你將樣式套用到特定 HTML 元素的所有實體。

型別選擇器使用簡單，直接寫成你想要設定樣式的元素的頁籤名稱。

以下是一個針對頁面上所有段落元素的型別選擇器範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  color: blue;
}
```

:::

在這個範例中，所有 `p` 元素的文字顏色都會被設定為 `blue`。

型別選擇器的特異性相較於其他選擇器較低。型別選擇器的特異性值為 `(0, 0, 0, 1)`。

這表示型別選擇器會被類別選擇器、ID 選擇器和行內樣式覆寫，但除非存在那些較高特異性的規則，否則仍然可以套用樣式。

讓我們來看看一個範例，其中類別選擇器會改寫型別選擇器的樣式。

這裡有一個包含兩個段落元素的範例：

:::interactive_editor

```html
<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

:::

兩個段落元素都有一個名為 `para` 的類別。

在 CSS 檔案中，型別選擇器標的為段落，而類別選擇器標的為具有 `para` 類別的元素。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

```css
p {
  color: blue;
}

.para {
  color: red;
}
```

:::

所有具有 `para` 類別的段落將會將文字顏色設定為 `red`，而非 `blue`，因為類別選擇器的特異性高於型別選擇器。
