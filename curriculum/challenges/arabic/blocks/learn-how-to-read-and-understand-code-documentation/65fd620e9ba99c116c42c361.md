# --description--

غالبًا ما يستخدم المطوِّرون `once` للدلالة على توقيت المهام ضمن سير عملهم. تعني `as soon as` أو `when`.

على سبيل المثال، `Once you receive the client's feedback, update the project documentation.` يعني أن تحديث الوثائق يجب أن يبدأ فور استلام الملاحظات، مما يوضح متتالية الإجراءات في دورة حياة المشروع.
