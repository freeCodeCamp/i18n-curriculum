# --explanation--

`Real-world` означає щось, що відбувається в реальному житті, а не в тесті чи вигаданій ситуації. Наприклад:  

`We tested the new app in real-world conditions to see how it works.` — це означає, що застосунок тестували в нормальних, повсякденних ситуаціях, щоб переконатися, що він добре працює.

`Feedback` означає пораду або коментар, щоб допомогти комусь покращитися. Наприклад:  

`The project manager gave feedback on my work so I can do better next time.` — це означає, що менеджер проєкту дає пораду, щоб допомогти вам покращити вашу роботу.

`Roll it out` означає запуск або випуск продукту чи послуги для ширшої аудиторії. Наприклад:

`We will roll out the new feature to all users next month.` — це вказує, що нова функція буде випущена для публічного використання, переходячи від тестування до доступності для всіх.
