# --description--

Il polimorfismo è il prossimo concetto chiave della programmazione orientata agli oggetti (OOP) di cui parleremo.

Con il polimorfismo, hai accesso a un’interfaccia con cui puoi interagire con molti oggetti dello stesso tipo.

Esaminiamo più a fondo il polimorfismo e come ti permette di riusare il codice.

Il polimorfismo consente ai metodi in classi diverse di condividere lo stesso nome ma di svolgere compiti diversi. Chiami lo stesso nome di metodo su oggetti diversi, e ciascuno risponde a modo suo.

Ecco l’esempio base di polimorfismo:

```py
class A:
   def action(self): ...

class B:
   def action(self): ...

class C:
   def action(self): ...

Class().method()  # Works for A, B, or C
```

Ecco un esempio che usa suoni di animali diversi per rappresentare il polimorfismo:

```py
class Cat:
   def speak(self):
       return "A cat meow"

class Bird:
   def speak(self):
       return "A bird tweet"
  
class Monkey:
   def speak(self):
       return "A monkey ooh ooh aah aah ooh ooh aah aah"

def animal_sound(animal):
   print(animal.speak())

animal_sound(Cat())
animal_sound(Bird())
animal_sound(Monkey())
```

In questo esempio, `animal_sound()` è una funzione che prende qualsiasi oggetto con un metodo `speak()`.

Quando passi un `Cat`, `Bird` o `Monkey`, chiama il metodo `speak()` dell’oggetto e stampa il risultato. Poiché ogni classe definisce `speak()` in modo diverso, ottieni output diversi dalla stessa funzione. Questo è il polimorfismo in azione.

Ecco un altro esempio, questa volta con istanze e un attributo:

```py
class Twitter:
   def __init__(self, content):
       self.content = content

   def post(self):
       return f"🐦 Tweet: '{self.content}' (280 chars max)"

class Instagram:
   def __init__(self, content):
       self.content = content

   def post(self):
       return f"📸 Instagram Post: '{self.content}' + ✨ filters"

class LinkedIn:
   def __init__(self, content):
       self.content = content

   def post(self):
       return f"💼 LinkedIn Article: '{self.content}' (Professional Mode)"

def start(social_media):
   print(social_media.post())  # Calls .post() on any object

# Instances
tweet = Twitter('Just learned Python polymorphism!')
photo = Instagram('Sunset vibes 🌅')
article = LinkedIn('Why OOP matters in 2024')

# The polymorphic calls - same function, different outputs
start(tweet) # 🐦 Tweet: 'Just learned Python polymorphism!' (280 chars max)
start(photo) # 📸 Instagram Post: 'Sunset vibes 🌅' + ✨ filters
start(article) # 💼 LinkedIn Article: 'Why OOP matters in 2024' (Professional Mode)
```

Esiste anche un tipo di polimorfismo chiamato **polimorfismo basato sull’ereditarietà**.

Nel polimorfismo basato sull’ereditarietà, una classe genitore definisce un metodo e più classi figlie sovrascrivono quel metodo a modo loro. Puoi quindi chiamare lo stesso metodo su qualsiasi oggetto figlio, e si comporta in modo diverso a seconda della classe figlia a cui appartiene.

Ecco un esempio:

```py
class Animal:
   def speak(self):
       return 'Some generic sound'

class Cat(Animal):
   def speak(self):
       return 'A cat meow'

class Dog(Animal):
   def speak(self):
       return 'A dog barks woof woof'

class Monkey(Animal):
   def speak(self):
       return 'A monkey ooh ooh aah aah ooh ooh aah aah'
  
print(Cat().speak()) # A cat meow
print(Dog().speak()) # A dog barks woof woof
print(Monkey().speak()) # A monkey ooh ooh aah aah ooh ooh aah aah
print(Animal().speak()) # Some generic sound
```

Puoi vedere che ogni classe figlia della classe genitore `Animal` sovrascrive il metodo `speak()` per fornire la propria implementazione. Quindi, quando chiami il metodo `speak()` su un’istanza di ciascuna sottoclasse, restituisce il suono specifico associato a quell’animale.

Puoi anche fare un passo avanti e chiamare i metodi in una lista, poi iterare la lista per mostrare cosa restituisce il metodo `speak()` per ciascuno:

```py
animals = [Cat(), Dog(), Monkey()]

for animal in animals:
   print(animal.speak())

# Output:
# A cat meow
# A dog barks woof woof
# A monkey ooh ooh aah aah ooh ooh aah aah
```
