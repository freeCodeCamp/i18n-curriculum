## --text--

Welcher Satz ist grammatikalisch korrekt?

## --answers--

`If we lose critical data, it do set us back weeks.`

### --feedback--

Das Wort `do` wird in diesem Satz fälschlicherweise anstelle von `could` verwendet.

---

`If we lose critical data, it could set us back weeks.`

---

`If we lose critical data, it could sets us back weeks.`

### --feedback--

Das Verb `sets` steht in diesem Satz nicht in der richtigen Form.

---

`If we losing critical data, it could set us back weeks.`

### --feedback--

Das Wort `losing` wird in diesem Satz fälschlicherweise anstelle von `lose` verwendet.

## --video-solution--

2

# --explanation--

Der korrekte Satz verwendet eine `First Conditional`-Struktur, die gebildet wird aus:

`if` + `Present Simple`: `If we lose critical data`.

Modalverb (`could`) + Grundform des Verbs: `it could set us back weeks`.


Die `First Conditional` wird verwendet, um ein mögliches zukünftiges Ereignis und dessen Ergebnis zu beschreiben.
