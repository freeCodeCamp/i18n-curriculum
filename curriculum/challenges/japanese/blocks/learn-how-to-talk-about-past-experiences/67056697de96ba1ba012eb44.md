# --explanation--

`first`、`then`、`after that`、`finally`のような順序を示す接続詞は、行動やイベントの順番を説明するために使われます。

- `First`は最初の行動を表すために使われます。

- `Then`は次の直後のステップを指します。

- `After that`は、ある行動の後に区別をつけて次のステップを導入するために使われます。

- `Finally`はシーケンスの最後のステップを示します。
