# --description--

在西班牙語中，有兩種常見的說法用來表達來源。

第一種是使用 **`Es de` + 國家**。例如：

`Es de Venezuela` - 他來自委內瑞拉。

第二種是使用國籍形容詞。例如：

`Es venezolano` - 他是委內瑞拉人。

國籍形容詞**不會**大寫。

# --explanation--

Esteban 說 `Sí, es brasileño`，意思是「是的，他是巴西人」（男性形式）。

在單字 `i` 的 `Sí` 字母上加上重音符號非常重要。加上重音符號，這個字的意思是「是」。如果沒有重音符號，根據上下文，這個字可能表示「如果」，作為條件句使用。

記得西班牙語中的國籍形容詞不會大寫。
