# --description--

증강 할당은 이항 연산과 할당을 한 단계로 결합합니다. 변수에 다른 값과 연산을 적용한 후 결과를 같은 변수에 다시 저장합니다.

JavaScript 같은 언어에 익숙하다면 덧셈 대입 연산자(`+=`)나 뺄셈 대입 연산자(`-=`) 등을 들어봤을 겁니다. Python에도 이런 연산자가 있습니다. 다만 Python에서는 이를 **증강 할당**이라고 부릅니다.

증강 할당의 기본 구문은 다음과 같습니다:

```python
variable <operator>= value
```

이는 다음과 같이 작성하는 것보다 더 효율적인 방법입니다:

```python
variable = variable <operator> value
```

예를 들어, 기존 변수에 `5`를 더할 때 증강 할당을 사용하는 예는 다음과 같습니다:

```python
my_var = 10
my_var += 5

print(my_var) # 15
```

그리고 증강 할당 없이 같은 작업을 하는 예는 다음과 같습니다:

```python
my_var = 10
my_var = my_var + 5

print(my_var) # 15
```

증강 할당의 장점은 변수 이름을 반복하지 않고도 변수 값을 간결하고 읽기 쉽게 업데이트할 수 있다는 점입니다. 이로 인해 중복이 줄고 오타 같은 실수로 인한 오류 가능성이 감소합니다.

모든 연산자는 증강 할당으로 사용할 수 있습니다. 덧셈 대입 연산자(`+=`)를 살펴봤으니, 다른 연산자들도 살펴보겠습니다.

- 뺄셈 대입 연산자(`-=`)는 오른쪽 피연산자를 왼쪽 변수에서 빼고 그 차이를 왼쪽 변수에 저장합니다:

```python
count = 14
count -= 3

print(count) # 11
```

- 곱셈 대입 연산자(`*=`)는 왼쪽 변수를 오른쪽 피연산자와 곱한 후 곱셈 결과를 왼쪽 변수에 저장합니다:
    

```python
product = 65
product *= 7

print(product) # 455
```

- 나눗셈 대입 연산자(`/=`)는 왼쪽 변수를 오른쪽으로 나눈 후 결과를 왼쪽 변수에 저장합니다:
    

```python
price = 100
price /= 4

print(price) # 25.0
```

- 몫 나눗셈 대입 연산자(`//=`)는 왼쪽 변수를 오른쪽으로 나눈 후 몫을 왼쪽 변수에 저장합니다:
    

```python
total_pages = 23
total_pages //= 5

print(total_pages) # 4
```

- 나머지 대입 연산자(`%=`)는 왼쪽 변수를 오른쪽으로 나눈 나머지를 계산해 왼쪽 변수에 저장합니다:
    

```python
bits = 35
bits %= 2

print(bits) # 1
```

- 거듭제곱 대입 연산자(`**=`)는 왼쪽 변수를 오른쪽 거듭제곱한 결과를 왼쪽 변수에 저장합니다:
    

```python
power = 2
power **= 3

print(power) # 8
```

문자열에도 일부 증강 할당 연산자를 사용할 수 있습니다. 예를 들어, 덧셈 대입 연산자는 문자열을 쉽게 연결할 수 있게 합니다:

```python
greet = 'Hello'
greet += ' World'

print(greet) # Hello World
```

그리고 곱셈 대입 연산자는 문자열을 반복하는 데 사용할 수 있습니다:

```python
greet = 'Hello'
greet *= 3

print(greet) # HelloHelloHello
```

다른 증강 할당 연산자를 문자열에 사용하면 `TypeError`가 발생합니다:

```python
greet = 'Hello'
greet -= ' World'

print(greet) # TypeError: unsupported operand type(s) for -=: 'str' and 'str'


greet = 'Hello'
greet /= 'World'

print(greet) # TypeError: unsupported operand type(s) for /=: 'str' and 'str' 
```

Python에서 증가 연산자(`++`)와 감소 연산자(`--`)가 작동하는지 궁금하다면, 작동하지 않습니다. Python은 언어를 명확하고 명시적으로 유지하기 위해 C 스타일 증가 및 감소 단축 구문을 의도적으로 피합니다.

`x++` 대신 `x += 1`를 쓰면 `x` 값을 `1`만큼 증가시키는 것이 명확해집니다.

Python에서 `++x`를 작성하면 단항 더하기 연산자가 두 번 적용될 뿐, 아무것도 증가하지 않습니다:

```python
my_var = 5

print(+my_var)   # 5
print(++my_var)  # 5
print(+++my_var) # 5

my_var += 1

print(my_var) # 6
```
