# --interactive--

## Lavorare con diversi selettori di attributi e collegamenti

- **Definizione**: Il selettore `attribute` ti permette di selezionare elementi HTML in base ai loro attributi come gli attributi `href` o `title`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Link with href</a>
<a>No href</a>
```

```css
a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

- **Attributo `title`**: Questo attributo fornisce informazioni aggiuntive su un elemento. Ecco come puoi selezionare i collegamenti con l'attributo `title`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Tooltip">Link with title</a>
<a href="#">Normal link</a>
```

```css
a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

- **Combina selettori per selezionare collegamenti che hanno sia gli attributi `href` che `title`**: Combina più selettori di attributi.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Info">Href + Title</a>
<a href="#">Only href</a>
```

```css
a[href][title] {
  color: green;
}
```

:::

- **Seleziona una singola parola all'interno di una lista separata da spazi**: Seleziona i collegamenti che hanno una specifica parola di classe.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a class="link primary">Primary link</a>
<a class="link secondary">Secondary link</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

- **Seleziona valori che iniziano con un prefisso specifico**: Seleziona i collegamenti che iniziano con `https://`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">HTTPS link</a>
<a href="http://www.freecodecamp.org">HTTP link</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

- **Seleziona valori che finiscono con un suffisso specifico**: Seleziona i collegamenti che finiscono con `.jpg`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="photo.jpg">Image link</a>
<a href="index.html">HTML link</a>
```

```css
a[href$=".jpg"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

- **Seleziona valori che contengono una sottostringa ovunque**: Seleziona i collegamenti che contengono `https` in qualsiasi punto del valore.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Secure link</a>
<a href="page.html">Local link</a>
```

```css
a[href*="https"] {
  color: teal;
}
```

:::

- **Riepilogo**: Questi pattern rendono facile definire lo stile in modo coerente per i collegamenti in base ai loro attributi e valori.

## Selezionare elementi con l'attributo `lang` e `data-lang`

- **Attributo `lang`**: Questo attributo si usa in HTML per specificare la lingua del contenuto all'interno di un elemento. Potresti voler definire lo stile degli elementi in modo diverso in base alla lingua in cui sono scritti, specialmente su un sito web multilingue.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">English text</p>
<p lang="fr">French text</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

- **Attributo `data-lang`**: Gli attributi dati personalizzati come l'attributo `data-lang` sono comunemente usati per archiviare informazioni aggiuntive negli elementi, ad esempio per specificare la lingua usata in una sezione specifica di testo. Ecco come puoi definire lo stile degli elementi in base all'attributo `data-lang`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div data-lang="fr">French content</div>
<div data-lang="en">English content</div>
```

```css
div[data-lang="fr"] {
  color: blue;
}
```

:::

## Lavorare con il selettore di attributi, gli elementi di lista ordinata e l'attributo `type`

- **Attributo `type`**: Quando lavori con liste ordinate in HTML, l'attributo `type` ti permette di specificare lo stile della numerazione usata, come numeri, lettere o numeri romani.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Alpha</li>
  <li>Beta</li>
</ol>

<ol type="i">
  <li>One</li>
  <li>Two</li>
</ol>
```

```css
/*Example targeting uppercase alphabetical numbering*/
ol[type="A"] {
  color: purple;
  font-weight: bold;
}

/*Example targeting lowercase Roman numerals*/
ol[type="i"] {
  color: green;
}
```
