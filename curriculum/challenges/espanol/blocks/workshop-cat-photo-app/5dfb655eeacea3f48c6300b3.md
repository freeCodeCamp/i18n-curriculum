# --description--

El elemento `figure` representa contenido autónomo y te permitirá asociar una imagen con un título.

Introduce la imagen que acabas de añadir dentro de un elemento `figure`.

# --hints--

El elemento `figure` debe tener una etiqueta de apertura. Las etiquetas de apertura tienen la siguiente sintaxis: `<elementName>`.

```js
assert.exists(document.querySelector('figure'));
```

El elemento `figure` debe tener una etiqueta de cierre. Las etiquetas de cierre tienen un `/` justo después del carácter `<`.

```js
assert.match(code, /<\/figure\>/);
```

Debe haber un elemento `figure` justo encima de la etiqueta de cierre del segundo elemento `section`.

```js
assert.equal(document.querySelectorAll('section')[1]?.lastElementChild.nodeName, 'FIGURE');
```

El elemento `img` de lasaña debe estar anidado en el elemento `figure`.

```js
assert.exists(document.querySelector('figure > img'));
assert.equal(
    document.querySelector('figure > img').getAttribute('src').toLowerCase(),
      'https://cdn.freecodecamp.org/curriculum/cat-photo-app/lasagna.jpg'
);
```
