# --explanation--  

`That way` 用于解释决策或行动的结果或好处。例如：

`We should automate testing. That way, we can catch bugs faster.` – `That way` 表示自动化将帮助更快地查找错误。  

`While` 用于表示两件事同时发生。例如：  

`The team is fixing bugs while the designers work on the UI.` – 这意味着两个任务同时进行。
