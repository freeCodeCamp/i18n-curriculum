# --description--

UTF-8, oder UCS Transformation Format 8, ist eine standardisierte Zeichenkodierung, die im Web weit verbreitet ist. Zeichenkodierung ist die Methode, mit der Computer Zeichen als Daten speichern. Im Grunde ist jeder Text auf einer Webseite eine Folge von Zeichen, die als ein oder mehrere Bytes gespeichert sind. In der Informatik ist ein Byte eine Dateneinheit, die aus 8 Bits oder Binärziffern besteht. UTF-8 unterstützt jedes Zeichen im Unicode-Zeichensatz, einschließlich Zeichen und Symbole aus allen Schriftsystemen, Sprachen und technischen Symbolen. Hier ist ein Beispiel für die Verwendung des `meta`-Elements mit dem `charset`-Attribut, um die Zeichenkodierung auf `UTF-8` festzulegen:

```html
<meta charset="UTF-8" />
```

Indem Sie die Zeichenkodierung auf UTF-8 setzen, stellen Sie sicher, dass das akzentuierte `"e"`-Zeichen (`é`) auf der Seite korrekt angezeigt wird. Hier ist ein erweitertes Codebeispiel zur Verwendung der UTF-8-Zeichenkodierung:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Examples of the UTF-8 encoding</title>
  </head>
  <body>
    <p>Café</p>
  </body>
</html>
```

Für jedes neue Projekt, das Sie erstellen, sollten Sie dieses `meta`-Element mit dem `charset`-Attribut, das auf `UTF-8` gesetzt ist, einfügen.
