# --explanation--

Manchmal, wenn Sie sich bei etwas nicht ganz sicher sind, aber glauben, dass es so sein sollte, können Sie die Phrase `It's supposed to` + Verb verwenden.

Damit drücken Sie aus, dass etwas voraussichtlich so ist, auch wenn Sie es noch nicht bestätigt haben. Zum Beispiel:

- `The update is supposed to fix the bug.` – Das bedeutet, Sie erwarten, dass das Update den Bug behebt, haben es aber noch nicht bestätigt.
