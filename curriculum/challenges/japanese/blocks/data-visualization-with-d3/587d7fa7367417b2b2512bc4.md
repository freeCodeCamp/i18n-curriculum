# --description--

D3ライブラリはデータ駆動型のアプローチにフォーカスしています。データのセットがあるとき、D3のメソッドを使ってページに表示できます。データはさまざまな形式がありますが、このチャレンジではシンプルな数字の配列を使います。

最初のステップはD3にデータを認識させることです。`data()`メソッドはDOM要素のセレクションに対して使い、その要素にデータを紐付けます。データセットはメソッドの引数として渡されます。

よくあるワークフローパターンは、データセットの各データに対してドキュメント内に新しい要素を作成することです。この目的のためにD3には`enter()`メソッドがあります。

`enter()`が`data()`メソッドと組み合わさると、ページ上の選択された要素とデータセット内のデータ数を比較します。要素の数がデータ数より少なければ、不足している要素を作成します。

以下は`ul`要素を選択し、配列のエントリー数に基づいて新しいリストアイテムを作成する例です：

```html
<body>
  <ul></ul>
  <script>
    const dataset = ['a', 'b', 'c'];
    d3.select('ul')
      .selectAll('li')
      .data(dataset)
      .enter()
      .append('li')
      .text('New item');
  </script>
</body>
```

まだ存在しない要素を選択するのは混乱するかもしれません。このコードはまずページ上の`ul`をD3に選択させています。次にすべてのリストアイテムを選択しますが、これは空のセレクションを返します。次に`data()`メソッドがデータセットを確認し、配列の各アイテムに対して3回このコードを実行します。`enter()`メソッドはページ上に`li`要素がないことを確認しますが、`dataset`の各データに対して3つ必要です。新しい`li`要素が`ul`に追加され、テキストは`New item`になります。
