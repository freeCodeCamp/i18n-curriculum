# --explanation--

`Simple Past` 시제의 질문은 `did` + 주어 + 기본 동사 형태를 사용합니다. 예를 들어:

`What did Sophie say?` – `Did`는 과거임을 나타내고, `say`는 기본 동사입니다.

반면, 긍정문은 동사의 과거형을 직접 사용합니다. 예를 들어:

`Sophie said that the movie was great.` – `say`의 과거형은 `said`입니다.
