# --description--

في الدرس السابق، تعرّفت على نوع بيانات القائمة وتعلمت كيفية الوصول إلى عناصر القائمة وكذلك تقطيع القائمة. في هذا الدرس، ستستمر في التعرف على القوائم وبعض الطرق الشائعة المرتبطة بها مثل `append()` و`pop()` و`sort()`.

أول طريقة سننظر إليها هي الطريقة `append()`. تُستخدم هذه الطريقة لإضافة عنصر إلى نهاية القائمة. إليك مثالًا على استخدام الطريقة `append()` لإضافة الرقم `6` إلى قائمة الأرقام:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

إذا أردت إضافة قائمة واحدة إلى نهاية قائمة أخرى، يمكنك أيضًا استخدام الطريقة `append()` بهذه الطريقة:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

لاحظ كيف أن قائمة `even_numbers` بأكملها مسكنة داخل قائمة `numbers`.

لكن إذا أردت إضافة كل الأرقام الفردية من قائمة `even_numbers` إلى نهاية قائمة `numbers`، فيمكنك استخدام الطريقة `extend()`.

الطريقة `extend()` مشابهة للطريقة `append()`، لكن مع `extend()` يمكنك إضافة عدة عناصر من قائمة إلى أخرى. إليك مثالًا على إضافة الأرقام `6` و`8` و`10` من قائمة إلى نهاية قائمة `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

كما ترى، القائمة المسكنة اختفت وأصبحت مجرد قائمة أرقام.

لإدخال عنصر في فهرس محدد داخل القائمة، يمكنك استخدام الطريقة `insert()`. تقبل هذه الطريقة معلمتين: الفهرس الذي تريد إدخال العنصر الجديد فيه والعنصر الذي تريد إدخاله.

إليك مثالًا على استخدام الطريقة `insert()`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

الكود التالي سيدخل الرقم `2.5` في الفهرس `2` داخل قائمة `numbers`.

إذا أردت إزالة عنصر من قائمة، يمكنك استخدام الطريقة `remove()`. تأخذ الطريقة `remove()` قيمة العنصر الذي تريد إزالته كمعلمة:

```py
numbers = [10, 20, 30, 40, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50]
```

من المهم ملاحظة أن هذه الطريقة تزيل فقط أول ظهور للعنصر، وليس كلها:

```py
numbers = [10, 20, 30, 40, 50, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50, 50]
```

لإزالة عنصر في فهرس محدد داخل القائمة، يمكنك استخدام الطريقة `pop()` بهذه الطريقة:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

إذا لم تحدد عنصرًا لطريقة `pop`، فسيتم إزالة العنصر الأخير.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

إذا كنت بحاجة إلى تفريغ القائمة، يمكنك استخدام الطريقة `clear()` بهذه الطريقة:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

الطريقة التالية التي سننظر إليها هي الطريقة `sort()`. تُستخدم هذه الطريقة لترتيب العناصر في مكانها. إليك مثالًا على ترتيب قائمة أرقام عشوائية في مكانها:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

على عكس الطريقة `sort()`، هناك الدالة `sorted()` التي تعمل مع أي كائن قابل للتكرار وتُرجع قائمة مرتبة جديدة بدلاً من تعديل القائمة الأصلية. على سبيل المثال:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(numbers) # [19, 2, 35, 1, 67, 41]
print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
```

للتذكير، الكائن القابل للتكرار هو نوع خاص من الكائنات يمكنك التكرار عليه، مما يسمح لك بالوصول إلى كل عنصر على حدة. ستتعلم المزيد عن كيفية عمل الحلقات في Python في درس مستقبلي.

كل من الطريقة `sort()` والدالة `sorted()` تقبلان معلمتين اختياريتين هما `key` و`reverse`. ستتعلم المزيد عن هذه المعلمات الاختيارية في درس مستقبلي عندما تتعلم عن الصفوف المرتبة. ستتعلم أيضًا المزيد عن الدوال المدمجة الأخرى مثل `sorted()` في دروس قادمة.

الطريقة التالية التي سننظر إليها هي الطريقة `reverse()`. هذه الطريقة تعكس قائمة العناصر في مكانها بهذه الطريقة:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

آخر طريقة سننظر إليها هي الطريقة `index`. تُستخدم هذه الطريقة لإيجاد أول فهرس يمكن العثور فيه على عنصر داخل القائمة. إليك مثالًا على استخدام الطريقة `index` لإيجاد اللغة `'Java'` في قائمة `programming_languages`:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

إذا لم يُعثر على العنصر، فإن Python ترمي خطأ `ValueError`:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: 'JavaScript' is not in list
"""
```

هناك المزيد من الطرق لقوائم Python، لكن هذه القائمة الأولية من الطرق مكان جيد للبدء.
