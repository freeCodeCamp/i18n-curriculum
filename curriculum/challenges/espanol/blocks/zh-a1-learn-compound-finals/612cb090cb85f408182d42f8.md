# --description--

Para pronunciar `uai`, comienzas con la lengua alta y cerca del paladar duro para `u`, luego haces una transición suave a la posición para `a`, y finalmente a la posición para `i`. El movimiento debe ser fluido, sin interrupción entre los tres sonidos.

Cuando `uai` se combina con las iniciales, obtienes las sílabas como `kuai`, `guai`, `huai`, `chuai`, `shuai`.

Practiquemos la pronunciación de estas sílabas, y luego usa `guai` en diferentes tonos como ejemplo.

PinyinCarácter
|--------|----------|
| `guāi`  | `乖`|
| `guǎi`   | `拐`|
| `guài`  | `怪`|
