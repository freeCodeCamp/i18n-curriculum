# --description--

En los videos de la lección anterior, aprendiste cómo llamar (invocar) a una función. Llamar a una función significa ejecutar el código dentro de la función.

Aquí tienes un recordatorio de cómo llamar a una función:

```js
function myFunction() {
  return "Hello World";
}

// function call
myFunction();
```

Agrega una declaración `console.log`, y dentro de ella, llama a la función `addTwoAndSeven`.

Ahora deberías ver la suma de `2` y `7` en el registro de la consola.
