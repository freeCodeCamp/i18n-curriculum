# --description--

이번 작업에서는 Ángela가 Esteban과의 대화를 마무리하는 것을 들으실 겁니다. Ángela는 `Todo listo por hoy`라는 표현으로 대화를 끝냅니다.

- `Todo`은 "모든 것"을 의미합니다.

- `Listo`는 "준비된"을 의미합니다.

- `Por hoy`는 오늘, 즉 "오늘을 위해"라는 의미를 가리킵니다.

이 세 가지가 합쳐져 `Todo listo por hoy`는 오늘의 작업이나 활동이 완료되었음을 나타냅니다.
