# --interactive--

在 JavaScript 中，`indexOf()` 方法可用于查找某个特定元素在数组中首次出现的索引。 如果找不到该元素，则返回 `-1` 。 这是基本语法：

```js
array.indexOf(element, fromIndex)
```

`element` 表示要在数组中搜索的值，`fromIndex` 参数则是搜索应从哪个位置开始。 `fromIndex` 参数是可选的。 如果没有提供`fromIndex`，则从数组的开头开始搜索。 来看一个示例：

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "banana"];
let index = fruits.indexOf("banana");
console.log(index); // 1
```

:::

在这个示例中，我们有一个包含各种水果名称的数组 `fruits`。 我们使用 `indexOf()` 方法查找字符串 `banana` 在 `fruits` 数组中的索引。 由于 `banana` 出现在索引 `1` 处，因此方法返回 `1`，结果存储在变量 `index` 中，并输出到控制台。

如果在数组中找不到要搜索的元素，`indexOf()` 返回 `-1`。 例如：

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let index = fruits.indexOf("grape");
console.log(index); // -1
```

:::

在这里，我们使用 `indexOf()` 查找字符串 `grape` 是否存在于 fruits 数组中。 于 `grape` 不存在，方法返回 `-1`，结果存储在变量 `index` 中，并记录到控制台。

如果要在特定索引号之后开始查找项目，可以像本例一样传递第二个参数： 

:::interactive_editor

```js
let colors = ["red", "green", "blue", "yellow", "green"];
let index = colors.indexOf("green", 3);
console.log(index); // 4
```

:::

在此示例中，搜索不是从数组的起始位置开始，而是从索引 `3` 即`yellow` 开始，因此最终结果为 `4`。
