# --description--

Der Ausdruck `there is` wird verwendet, um die Existenz von etwas an einem bestimmten Ort anzugeben. Zum Beispiel:

`There is a problem in the code.` – Ein Problem besteht mit dem Code.

Maria verwendet diese Struktur, um Tom über den Standort der Bank zu informieren.

# --questions--

## --text--
