# --description--

Im Elements-Bereich können Sie die gesamte HTML-Struktur Ihrer Seite sehen. Sie können auf eines der Elemente in diesem Bereich klicken, um genau dieses Element auszuwählen. Alternativ können Sie auf das unten links blau hervorgehobene Symbol klicken und mit der Maus über ein beliebiges Element auf der Seite fahren.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/the-box-model/inspector-icon.png" alt="Eine Webentwicklungsumgebung mit geöffneten Entwicklerwerkzeugen, die ein blaues Inspektor-Symbol oben links hervorhebt, um Webseiten-Elemente auszuwählen und die HTML-Struktur sowie Styles anzuzeigen.">

Wenn ein Element ausgewählt ist, zeigt der Styles-Reiter alle aktuell angewendeten Styles sowie alle Styles, die überschrieben werden (gekennzeichnet durch einen durchgestrichenen Text). Wenn Sie zum Beispiel mit dem Inspektor auf die Kopfzeile „Your Career in Web Development Starts Here“ auf der <a href="https://www.theodinproject.com/" target="_blank">Startseite von The Odin Project</a> klicken, sehen Sie auf der rechten Seite alle Styles, die das Element gerade beeinflussen, wie unten dargestellt:

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/the-box-model/overwritten-style.png" alt="CSS-Code-Ausschnitt in der Entwicklerkonsole, der .hero_main-heading mit angewendetem margin-bottom: 100px, padding-bottom: 1.875rem, .accent color #ce973e zeigt. h1 Style-Regeln umfassen font-size, weight, letter-spacing, wobei margin-bottom: 2rem überschrieben ist.">

# --questions--    

## --text--

Was bedeutet es, wenn beim Untersuchen eines Elements in den Chrome Dev Tools eine Style-Eigenschaft durchgestrichen angezeigt wird?

## --answers--

Die Style-Eigenschaft ist derzeit deaktiviert und wird nicht auf das Element angewendet.

---

Die Style-Eigenschaft wird durch eine spezifischere CSS-Regel überschrieben.

---

Die Style-Eigenschaft ist veraltet und sollte nicht verwendet werden.

---

Die Style-Eigenschaft ist experimentell und wird möglicherweise nicht von allen Browsern unterstützt.
