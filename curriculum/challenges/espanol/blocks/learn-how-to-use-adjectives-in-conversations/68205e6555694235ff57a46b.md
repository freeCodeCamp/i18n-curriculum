# --explanation--

`Minimal` significa solo una cantidad muy pequeña. Justo lo suficiente para cubrir una necesidad. Por ejemplo:

`The task required minimal effort.` – Esto significa que fue muy fácil y no requirió mucha energía.

El `Setup time` es el tiempo que toma preparar algo para su uso. Por ejemplo:

`The software has a short setup time.` – Esto significa que no toma mucho tiempo prepararlo para usar.
