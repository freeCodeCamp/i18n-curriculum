# --description--

Залежно від того, якою роботою ви займаєтеся, ви можете більше працювати з текстовими фрагментами, а не з числами. **Рядок** — це фрагмент тексту… і він є фундаментальним будівельним блоком мови.

HTML надає тексту структуру та значення, CSS дозволяє точно його стилізувати, а JavaScript пропонує багато функцій для маніпулювання рядками. Це включає створення кастомних привітань і підказок, показ правильних текстових міток за потреби, сортування термінів у потрібному порядку та багато іншого.

Рядки — це базовий тип даних у JavaScript. Вони використовуються для представлення тексту і беруться в одинарні, подвійні лапки або зворотні апострофи:

```javascript
let greeting = "Hello World!";

let greeting2 = 'I am learning JavaScript!';

let greeting3 = `with The Odin Project!`;
```

Рядки, оголошені в одинарних лапках, і рядки, оголошені в подвійних лапках, однакові, і який стиль ви оберете — це справа особистих уподобань, хоча хорошою практикою є вибрати один стиль і послідовно його використовувати у своєму коді.

## --answers--

Рядками в JavaScript можна маніпулювати лише за допомогою CSS для дизайну вебсторінки.

---

Рядки JavaScript виключно використовуються для математичних обчислень і не можуть взаємодіяти з HTML або CSS.

---

Рядки дозволяють маніпулювати текстом у JavaScript, включно, але не обмежуючись створенням кастомних повідомлень, сортуванням термінів і динамічним оновленням вмісту HTML із стилізацією CSS.
