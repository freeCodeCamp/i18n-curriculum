# --explanation--

`Next to`는 공간이나 위치상 무언가 옆이나 아주 가까운 곳에 있음을 의미합니다.

정답 문장에서는 프린터가 책상 옆에 물리적으로 있습니다.

다른 문장들은 `next to`를 시간이나 부적절한 문맥에서 잘못 사용했습니다.
