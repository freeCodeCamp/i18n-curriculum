# --description--

على عكس العمل مع مشاريع HTML وCSS وJavaScript البسيطة، يتطلب بدء مشروع React جديد بعض الخطوات الإضافية لضمان تشغيل كل شيء بشكل صحيح. بدلًا من محاولة إعداد كل التكوينات اللازمة بنفسك، هناك أدوات تقوم بذلك نيابة عنك.

واحدة من أشهر الأدوات لإعداد المشاريع هي Vite. تعني كلمة Vite "سريع" بالفرنسية، وهي أداة بناء تهدف إلى توفير تجربة تطوير أسرع لمشاريع الويب الحديثة. يمكن استخدام Vite مع React، وكذلك مع مكتبات وأُطُر عمل أخرى مثل Vue أو Svelte. يمكنك حتى استخدامها مع مشاريع JavaScript العادية.

لإنشاء مشروع جديد باستخدام Vite، ستحتاج إلى استخدام سطر الأوامر. إذا كنت تستخدم جهاز ويندوز، يمكنك استخدام موجه الأوامر أو Windows PowerShell. وإذا كنت تستخدم جهاز ماك، فاستخدم تطبيق Terminal.

في حال لم يكن Node.js مثبتًا لديك، وهو ضروري لعمل npm، إليك رابط الصفحة الرسمية لـ Node.js: https://nodejs.org. اتبع تعليمات التثبيت ثم عد هنا لاستكمال تثبيت React.

بمجرد فتح سطر الأوامر، يمكنك استخدام الأمر التالي:

```bash
npm create vite@latest my-react-app -- --template react
```

ينشئ هذا الأمر مشروع React جديد باسم `my-react-app` باستخدام قالب React الخاص بـ Vite.

يمكنك بعد ذلك فتح المشروع الجديد ورؤية شفرة قالب React الجاهز التي وفرها Vite لك.

الميزة الرائعة في Vite هي أنه يوفر فقط الملفات الضرورية تمامًا للبدء في مشروع React الخاص بك.

لتشغيل المشروع كما هو، ستحتاج إلى تثبيت التبعيات باستخدام الأوامر التالية في سطر الأوامر:

```bash
cd my-react-app
npm install
```

يضمن أمر `cd my-react-app` تغيير الدليل إلى المكان الصحيح حيث يوجد مشروعك.

يُستخدم أمر `npm install` لتثبيت التبعيات اللازمة لتشغيل مشروع React الخاص بك بشكل صحيح. التبعيات هي مكتبات أو حزم يحتاجها مشروع React ليعمل بشكل صحيح، مثل React نفسه، ReactDOM، أو حزم طرف ثالث توفر وظائف إضافية.

عند تشغيل `npm install`، سيقرأ ملف `package.json` في دليل مشروعك ويثبت كل التبعيات المدرجة هناك، مما يسمح لك ببناء وتشغيل تطبيق React دون فقدان أي مكونات ضرورية.

ملف `package.json` هو ملف تكوين رئيسي في المشاريع يحتوي على بيانات وصفية عن مشروعك، بما في ذلك اسمه، وإصداره، وتبعياته. كما يحدد السكربتات، ومعلومات الترخيص، وإعدادات أخرى تساعد في إدارة المشروع وتبعياته.

بمجرد تثبيت التبعيات، يجب أن تلاحظ وجود مجلد جديد في مشروعك يسمى `node_modules`.

مجلد `node_modules` هو المكان الذي تُخزن فيه كل الحزم والمكتبات التي يحتاجها مشروعك. يحتوي هذا المجلد على الشفرة الفعلية للتبعيات المدرجة في ملف `package.json` الخاص بك، بما في ذلك التبعيات المباشرة لمشروعك وأي تبعيات لتلك التبعيات.

لتشغيل مشروعك، نفذ أمر `npm run dev` وافتح تبويب جديد في المتصفح على العنوان `http://localhost:5173/`.

يجب أن ترى قالب البداية الذي وفره لك Vite.

إذا احتجت إلى إيقاف الخادم المحلي، اضغط `CTRL + C` في سطر الأوامر.

لمشاهدة شفرة هذا القالب المبدئي، يمكنك الدخول إلى مشروعك داخل مجلد `src` وستجد ملف `App.jsx`.

من هناك يمكنك البدء بتعديل الملف، حفظ التغييرات، ورؤية التحديثات الجديدة تظهر في المتصفح.

وبذلك، أنت جاهز لبدء بناء تطبيق React الخاص بك!
