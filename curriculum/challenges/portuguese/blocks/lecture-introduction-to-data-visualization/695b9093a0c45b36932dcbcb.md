# --description--

Nas lições anteriores, você foi apresentado à visualização de dados que é o processo de criar formatos visuais claros para representar dados complexos. Nesta lição, você aprenderá sobre a biblioteca D3.

A biblioteca D3 é uma biblioteca gratuita e de código aberto que você pode usar para criar visualizações de dados atraentes e eficazes em JavaScript.

O nome D3 é uma abreviação para data-driven documents, referindo-se a como ele vincula dinamicamente dados a elementos no Document Object Model (DOM). Esta biblioteca é flexível e funciona naturalmente com a web. Em vez de ter que aprender uma representação gráfica totalmente nova, você pode construir visualizações de dados com os elementos `svg` e `canvas`.

Por exemplo, se você quiser criar um gráfico de barras usando D3, você pode fazer o seguinte tudo em um arquivo JS:

* Declare variáveis que representam as dimensões e margens do gráfico
    
* Declare a escala de posição horizontal
    
* Declare a escala de posição vertical
    
* Crie o contêiner SVG com as barras e os rótulos

Quais são algumas outras razões para aprender D3 em vez de outras ferramentas de visualização como Chart.js, Plotly e Tableau?

Ao contrário de outras ferramentas, o D3 oferece muito controle de baixo nível onde você pode criar visualizações personalizáveis e interativas usando apenas HTML, CSS e JavaScript puros. Além disso, a biblioteca é em JavaScript, então você não precisa se preocupar em aprender uma linguagem totalmente nova.

Nos próximos workshops e laboratórios, você aprenderá os fundamentos da biblioteca D3, como seleções, formas e escalas. Você aprenderá esses conceitos-chave construindo gráficos de dispersão, gráficos de barras, aplicativos de painel e mais.
