# --description--

Визначимо піфагоровий полігон як опуклий полігон із такими властивостями:

- має щонайменше три вершини,
- жодні три вершини не лежать на одній прямій,
- кожна вершина має цілі координати,
- кожна сторона має цілу довжину.

Для заданого цілого числа $n$ визначимо $P(n)$ як кількість різних піфагорових полігонів, для яких периметр $≤ n$.

Піфагорові полігони вважаються різними, якщо жоден із них не є зсувом іншого.

Відомо, що $P(4) = 1$, $P(30) = 3655$ та $P(60) = 891045$.

Знайдіть $P(120)$.
