# --description--

`plugin` は既存のコンピューター・プログラムに特定の機能を追加するコンポーネントです。プログラムが`plugins`をサポートしている場合、カスタマイズが可能になります。  
IDEの文脈では、`plugins`はコーディング、デバッグ、ユーザーインターフェースの強化などの追加機能を提供できます。

例えば、IDEのコードフォーマッターはしばしば`plugin`です。
