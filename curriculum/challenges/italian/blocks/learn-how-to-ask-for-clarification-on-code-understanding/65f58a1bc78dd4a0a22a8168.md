# --description--

`Encounter`, specialmente nella sua forma passata `encountered`, significa imbattersi in qualcosa, di solito inaspettatamente. Per esempio, `I encountered an error while testing the software.` Implica trovare qualcosa senza cercarla.

`Pinpoint` significa identificare qualcosa con grande precisione. Per esempio, `After reviewing the code, I was able to pinpoint the error.` Questo vuol dire trovare esattamente dove e qual è il problema.

Nel coding, trovare problemi spesso significa `encountering` problemi inaspettati e cercare di `pinpoint` le loro cause.
