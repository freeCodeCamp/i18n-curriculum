# --description--

`quarter` ділить рік на чотири рівні частини, кожна з яких складається з трьох місяців.

`first quarter` (Q1) охоплює січень по березень, `second quarter` (Q2) триває з квітня по червень, `third quarter` (Q3) включає липень по вересень, а `fourth quarter` (Q4) охоплює жовтень по грудень.

Наприклад, у бізнесі компанія може робити повторення своїх продажів наприкінці кожного `quarter`. Технічна команда може ставити за мету випустити нове оновлення програмного забезпечення до кінця Q2.
