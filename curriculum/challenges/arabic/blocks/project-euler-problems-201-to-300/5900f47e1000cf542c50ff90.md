# --description--

اعتبر المعادلات من الشكل: $a^2 + b^2 = N$، حيث $0 ≤ a ≤ b$، و$a$ و$b$ و$N$ أعداد صحيحة.

بالنسبة لـ $N = 65$ هناك حلّان:

$a = 1, b = 8$ و$a = 4, b = 7$.

نسمي $S(N)$ مجموع قيم $a$ لكل الحلول للمعادلة $a^2 + b^2 = N$، حيث $0 ≤ a ≤ b$، و$a$ و$b$ و$N$ أعداد صحيحة.

إذاً $S(65) = 1 + 4 = 5$.

احسب $\sum S(N)$ لكل $N$ خالٍ من المربعات وقابل للقسمة فقط على أعداد أولية من الشكل $4k + 1$ حيث $4k + 1 < 150$.
