# --description--

Розглянемо число 45656.

Можна побачити, що різниця між кожною парою послідовних цифр числа 45656 дорівнює одному.

Число, у якого різниця між кожною парою послідовних цифр дорівнює одному, називається кроковим числом.

Пандигітальне число містить кожен десятковий знак від 0 до 9 принаймні один раз.

Скільки існує пандигітальних крокових чисел, менших за ${10}^{40}$?
