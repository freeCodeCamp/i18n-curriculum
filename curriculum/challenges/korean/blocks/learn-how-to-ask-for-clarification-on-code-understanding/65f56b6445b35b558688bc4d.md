# --description--

`Step by step`는 한 단계씩 차근차근 하는 것을 의미합니다. 예를 들어, `debugging step by step`는 문제를 작고 관리하기 쉬운 부분으로 나누어 해결하는 것을 뜻합니다.

코딩에서 `Input`는 종종 프로젝트에 대한 정보나 조언을 의미합니다.

`Work together`는 협력하는 것을 뜻하며, `pair up`도 비슷하게 두 사람이 긴밀히 함께 일하는 것을 의미합니다.
