# --explanation--

`First Conditional` は未来の現実的または可能な状況とその結果について話すために使われます。条件節は `if` + `Present Simple` 時制の構造で、結果節は `will` または `can` + 動詞の原形で表されます。例えば：

- `If you study hard, you will pass the exam.` - これは可能な状況とその結果を説明します。

- `If it rains tomorrow, we can cancel the picnic.` - これは条件（雨）が起こった場合に何が起こるかを説明します。
