# --explanation--

当你同意某人的想法时，可以简短地对他说 `Sure`。例如：

- `Would you like to visit another country some day? Sure.`－这意味着你想这么做。

- `How about going to the movies this week? Sure.` - 这意味着你接受了邀请。
