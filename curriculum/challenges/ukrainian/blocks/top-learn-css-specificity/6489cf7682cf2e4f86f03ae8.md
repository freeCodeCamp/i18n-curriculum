# --description--

Успадкування означає певні CSS-властивості, які, коли застосовані до елемента, успадковуються нащадками цього елемента, навіть якщо ми явно не пишемо правило для цих нащадків. Властивості, пов’язані з типографікою (`color`, `font-size`, `font-family` тощо), зазвичай успадковуються, тоді як більшість інших властивостей — ні.

Винятком є пряме націлювання на елемент, оскільки воно завжди має пріоритет над успадкуванням:

```html
<!-- index.html -->

<div id="parent">
  <div class="child"></div>
</div>
```

```css
/* styles.css */

#parent {
  color: red;
}

.child {
  color: blue;
}
```

Незважаючи на те, що елемент `parent` має вищу специфічність через ідентифікатор, стиль `child` буде застосований до елемента `color: blue`, оскільки це оголошення безпосередньо націлене на нього, тоді як `color: red` від батьківського елемента лише успадковується.
