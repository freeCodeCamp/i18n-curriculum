# --description--

다음 반복 수열은 양의 정수 집합에 대해 정의됩니다:

<div style='padding-left: 4em;'><var>n</var> → <var>n</var>/2 (<var>n</var>이 짝수일 때)</div>

<div style='padding-left: 4em;'><var>n</var> → 3<var>n</var> + 1 (<var>n</var>이 홀수일 때)</div>

위 규칙을 사용해 13부터 시작하면 다음 수열이 생성됩니다:

<div style='text-align: center;'>13 → 40 → 20 → 10 → 5 → 16 → 8 → 4 → 2 → 1</div>

이 수열(13에서 시작해 1에서 끝남)은 10개의 항을 포함하는 것을 알 수 있습니다. 아직 증명되지는 않았지만(Collatz 문제), 모든 시작 숫자가 1에서 끝난다고 생각됩니다.

주어진 `limit` 이하의 시작 숫자 중 가장 긴 수열을 만드는 숫자는 무엇인가요?

**참고:** 수열이 시작된 후에는 항이 `limit`를 초과할 수 있습니다.

```js
function longestCollatzSequence(limit) {
  let longest = 1;
  let maxLength = 1;
  for (let i = Math.floor(limit / 2); i < limit; i++) {
    let len = colLen(i);
    if (len > maxLength) {
      longest = i;
      maxLength = len;
    }
  }
  return longest;
}

const knownSequence = { '1': 1 };

function colLen(n) {
  if (knownSequence[n]) {
    return knownSequence[n];
  } else {
    const len = n % 2 === 0 ? colLen(n / 2) + 1 : colLen((3 * n + 1) / 2) + 2;
    knownSequence[n] = len;
    return len;
  }
}
```
