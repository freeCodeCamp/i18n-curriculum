# --explanation--

При формуванні питання з модальними дієсловами структура така: модальне дієслово + підмет + дієслово у початковій формі + (об’єкт)? Наприклад:

- `Can we complete the project by Friday?` — запитання, чи можливо завершити проєкт до п’ятниці.

- `Should we update the software to the latest version?` — прохання поради або пропозиції, чи варто оновити програмне забезпечення.

У правильній відповіді `can` стоїть перед підметом (`you`), а дієслово (`explain`) — у початковій формі.
