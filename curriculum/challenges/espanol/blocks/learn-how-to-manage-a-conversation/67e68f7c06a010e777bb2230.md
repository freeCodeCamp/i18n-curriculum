# --explanation--

`It seems to be` + verbo en forma `-ing` se usa para describir algo que aparentemente está ocurriendo, aunque puede no estar confirmado. Por ejemplo:

`The app seems to be running slower today.` – Esto significa que parece que la app está más lenta de lo habitual.

`To cause something to` + verbo significa hacer que algo suceda o desencadenar una acción. Por ejemplo:

`Low memory can cause the program to crash.` – Esto significa que la baja memoria es la razón por la que el programa deja de funcionar.
