# --description--

Сортування Bozo, яке не слід плутати з трохи менш ефективним сортуванням bogo, полягає у перевірці, чи відсортована послідовність вхідних даних, і якщо ні — випадковому обміні двох елементів. Цей процес повторюється, доки послідовність не буде відсортована.

Якщо розглядати всі перестановки перших 4 натуральних чисел як вхідні дані, очікуване значення кількості обмінів, усереднене по всіх $4!$ вхідних послідовностях, дорівнює $24.75$.

Вже відсортована послідовність потребує 0 кроків.

У цьому завданні ми розглядаємо наступний варіант сортування Bozo.

Якщо послідовність не впорядкована, ми випадковим чином вибираємо три елементи і випадково перемішуємо ці три елементи.

Всі $3! = 6$ перестановок цих трьох елементів однаково ймовірні.

Вже відсортована послідовність потребує 0 кроків.

Якщо розглядати всі перестановки перших 4 натуральних чисел як вхідні дані, очікуване значення кількості перемішувань, усереднене по всіх $4!$ вхідних послідовностях, дорівнює $27.5$.

Розгляньте як вхідні послідовності перестановки перших 11 натуральних чисел.

Усереднене по всіх $11!$ вхідних послідовностях, яке очікуване число перемішувань виконає цей алгоритм сортування? Наведіть відповідь, округлену до найближчого цілого числа.
