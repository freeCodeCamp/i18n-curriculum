# --interactive--

Функції вищого порядку — це потужна концепція в JavaScript, яка може значно покращити ваші навички кодування та зробити ваш код більш гнучким і багаторазовим.

По суті, функція вищого порядку — це функція, яка приймає одну або кілька функцій як аргументи, повертає функцію або робить і те, й інше.

Щоб зрозуміти функції вищого порядку, спершу розглянемо функції як об’єкти першого класу в JavaScript. Це означає, що функції можна розглядати як будь-яке інше значення — їх можна призначати змінним, передавати як аргументи іншим функціям і повертати з функцій. Саме ця гнучкість дає змогу створювати та використовувати функції вищого порядку.

Одне з поширених застосувань функцій вищого порядку — це абстрагування складних операцій.

Наприклад, у вас може бути функція, яка виконує певну операцію над кожним елементом масиву. Замість того, щоб писати окремі функції для різних операцій, ви можете створити функцію вищого порядку, яка прийматиме операцію як аргумент. Це дозволяє повторно використовувати одну й ту саму структуру функції з різною поведінкою.

Ось приклад, щоб проілюструвати цю концепцію:

:::interactive_editor

```js
function operateOnArray(arr, operation) {
  let result = [];
  for (let i = 0; i < arr.length; i++) {
    result.push(operation(arr[i]));
}
  return result;
}

function double(x) {
    return x * 2;
}

let numbers = [1, 2, 3, 4, 5];
let doubledNumbers = operateOnArray(numbers, double);
console.log(doubledNumbers); // [2, 4, 6, 8, 10]
```

:::

У цьому прикладі `operateOnArray` — це функція вищого порядку. Вона приймає масив і функцію (`operation`) як аргументи. Потім вона застосовує операцію до кожного елемента масиву. Функція double передається як аргумент до `operateOnArray`, що демонструє, як функції можна використовувати як значення.

Функції вищого порядку також можуть повертати функції. Це особливо корисно для створення спеціалізованих функцій на основі більш загальних. Цю концепцію часто називають фабриками функцій. Ось приклад:

:::interactive_editor

```js
function multiplyBy(factor) {
  return function(number) {
    return number * factor;
  }
}

let double = multiplyBy(2);
let triple = multiplyBy(3);

console.log(double(5)); // 10
console.log(triple(5)); // 15
```

:::

У цьому випадку `multiplyBy` — це функція вищого порядку, яка повертає нову функцію. Ця нова функція спеціалізована на основі множника, переданого до `multiplyBy`. Це дозволяє легко створювати кастомні функції множення.

Функції вищого порядку — це не просто теоретична концепція, вони широко використовуються в JavaScript.

Багато вбудованих методів для масивів у JavaScript, таких як `map()`, `filter()` і `reduce()`, є функціями вищого порядку. Ці методи приймають функцію як аргумент і застосовують її до елементів масиву різними способами. Ви дізнаєтеся більше про ці методи в майбутніх уроках.

Використання функцій вищого порядку може призвести до більш декларативного та легшого для розуміння коду.

Замість того, щоб описувати крок за кроком, як виконати завдання (імперативне програмування), функції вищого порядку дозволяють описати, що ви хочете зробити (декларативне програмування). Це може зробити ваш код більш читабельним і простим у підтримці.

Продовжуючи працювати з JavaScript, ви часто зустрічатимете та використовуватимете функції вищого порядку. Вони є ключовою частиною функціонального програмування в JavaScript і необхідні для написання чистого, ефективного та гнучкого коду.

Розуміння та використання функцій вищого порядку значно покращить вашу здатність писати складні та елегантні програми на JavaScript.
