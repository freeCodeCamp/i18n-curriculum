## --text--

`Console.Write` と `Console.WriteLine` の違いは何ですか？

## --answers--

`Console.Write` は出力を新しい行に出力します。

---

`Console.WriteLine` は出力を新しい行に出力します。

---

`Console.WriteLine` は出力の後に改行を追加します。
