# --description--

Per `make sure` si usa per esprimere certezza o conferma su qualcosa. Per esempio:

`I check the list twice to make sure I have everything.` - Questo significa che guardi la lista due volte per essere sicuro che non manchi nulla.

Per `work properly` significa funzionare correttamente o come previsto. Per esempio:

`It's important that all machines work properly for safety.` - Questo significa che le macchine devono funzionare nel modo giusto affinché le persone restino al sicuro.
