# --description--

定数（変更されない）変数を宣言するには、`const` の代わりに `let` を使ってください。

```javascript
const myBirthday = '16-09-2003';
```

`const` を使って宣言された変数は「定数」と呼ばれます。これらは再代入できません。再代入しようとするとエラーになります。

```javascript
const myBirthday = '16-09-2003';

myBirthday = '01-01-2001'; // error, can't reassign the constant!
```

プログラマーが変数が絶対に変わらないと確信している場合、`const` を使って宣言することで、その事実を保証し、みんなに伝えることができます。
