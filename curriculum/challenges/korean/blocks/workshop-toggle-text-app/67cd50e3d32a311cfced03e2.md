# --description--

이 방법도 가능하지만, 단락 텍스트의 가시성을 토글하기 위해 `useState` 훅의 초기 값을 수동으로 업데이트하는 것은 그리 실용적이지 않습니다.

`useState` 훅의 초기 값을 `true`에서 `false`으로 업데이트하세요.

다음 몇 단계에서 단락 텍스트의 가시성 토글을 처리할 버튼 기능을 구현할 것입니다.
