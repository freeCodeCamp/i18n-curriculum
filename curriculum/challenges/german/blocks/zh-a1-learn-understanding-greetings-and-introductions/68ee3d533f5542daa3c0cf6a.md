# --description--

Wenn Sie jemandem auf Chinesisch Ihren Namen sagen möchten, können Sie eine sehr gebräuchliche und natürliche Struktur verwenden:

`​我是 (wǒ shì)​​` + Name.

Das bedeutet „Ich bin...“ oder „Mein Name ist...“. `我 (wǒ)` bedeutet „Ich“. Denken Sie an `是 (shì)` als das Verb „sein“ im Englischen.

# --explanation--

`我是 (wǒ shì)​​` bedeutet „Ich bin...“ oder „Mein Name ist...“. Es verwendet das Verb `是 (shì)` nach dem Pronomen `我 (wǒ)`, das „Ich“ bedeutet, um die Identität auszudrücken. Zum Beispiel:

`我是刘明 (wǒ shì liú míng)。` — Ich bin Liu Ming.
