# --description--

Добре, ви вже пройшли чимало матеріалу. Залишилось лише розглянути, як додати весь цей CSS до вашого HTML. Існує три способи зробити це.

Зовнішній CSS — це найпоширеніший метод, з яким ви стикнетеся, і він полягає у створенні окремого файлу для CSS та підключенні його всередині відкриваючого і кінцевого `<head>` тегів HTML за допомогою елемента `<link>` (який є порожнім елементом і не потребує кінцевого тегу):

Спочатку ви додаєте порожній тег `<link>` всередині відкриваючих і кінцевих `<head>` тегів HTML-файлу. Атрибут `href` — це розташування CSS-файлу, яке може бути абсолютним URL або, що ви будете використовувати, URL відносно розташування HTML-файлу. У наведеному вище прикладі припускається, що обидва файли знаходяться в одній теці. Атрибут `rel` є обов’язковим і вказує на відношення між HTML-файлом і підключеним файлом.

Потім у новоствореному `styles.css` файлі у вас є селектор (`div` і `p`), за яким слідує пара відкриваючих і закриваючих фігурних дужок, що створюють «блок оголошень». Нарешті, ви розміщуєте будь-які оголошення всередині блоку оголошень. `color: white; ` — це одне оголошення, де `color` — це властивість, а `white` — значення, а `background-color: black;` — інше оголошення.

Примітка щодо назв файлів: `styles.css` — це просто назва файлу, яку ви обрали тут. Ви можете назвати файл як завгодно, доки тип файлу — `.css`, хоча найчастіше використовують «style» або «styles».

Декілька переваг цього методу:

1. Він розділяє ваш HTML і CSS, що робить HTML-файл меншим і виглядає охайніше.
2. Вам потрібно редагувати CSS лише в одному місці, що особливо зручно для сайтів із багатьма сторінками, які всі використовують схожі стилі.
