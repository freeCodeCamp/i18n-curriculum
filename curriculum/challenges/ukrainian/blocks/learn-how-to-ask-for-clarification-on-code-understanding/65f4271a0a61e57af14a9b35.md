# --description--

Це завдання зосереджене на повторенні використання `can` для вираження здатності та використання дієслова `try` у минулому безперервному часі.

`Can` використовується, щоб говорити про чиюсь здатність щось робити, як `I can solve this problem.`

`Try` у минулому безперервному часі (`was`/`were` + дієслово у `-ing` формі) вказує на безперервну дію в минулому, як `I was trying to fix the issue.`
