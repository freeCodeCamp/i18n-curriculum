# --description--

O termo `run code` refere-se à ação de executar ou operar um programa ou script por um computador. Executar neste contexto significa iniciar o processo e acompanhá-lo até que o programa faça o que foi projetado para fazer, seja calcular números, ordenar dados ou qualquer outra coisa.

A frase `run code without managing servers` refere-se a executar programas de computador ou aplicações sem a necessidade de gerenciar os servidores físicos ou virtuais nos quais o código é executado. Este é um aspecto fundamental da computação serverless, onde o provedor de nuvem gerencia os servidores, permitindo que os desenvolvedores se concentrem mais em codificar do que na gestão dos servidores.
