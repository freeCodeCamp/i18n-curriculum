# --description--

正規表現を使って文字のグループをチェックしたい場合があり、そのために丸括弧 `()` を使います。

文字列の中で `Penguin` または `Pumpkin` のどちらかを見つけたい場合、次の正規表現を使えます：`/P(engu|umpk)in/g`

そして、`test()` メソッドを使って、テスト文字列に目的の文字列グループが含まれているかどうかを確認します。

```js
let testStr = "Pumpkin";
let testRegex = /P(engu|umpk)in/;
testRegex.test(testStr);
```

ここでの `test` メソッドは `true` を返します。

# --instructions--

正規表現を修正して、`Franklin Roosevelt` または `Eleanor Roosevelt` の名前を大文字小文字を区別してチェックし、ミドルネームに対応できるようにしてください。

次に、作成した正規表現が `myString` に対してチェックされ、正規表現にマッチすれば `true`、そうでなければ `false` が返されるようにコードを修正してください。
