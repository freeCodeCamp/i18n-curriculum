# --description--

Створіть послідовність чисел за допомогою генератора псевдовипадкових чисел "Blum Blum Shub":

$$
s_0 = 14025256 \\\\
s_{n + 1} = {s_n}^2 \\; mod \\; 20\\,300\\,713
$$

Об’єднайте ці числа $s_0s_1s_2\ldots$ у рядок $w$ нескінченної довжини. Тоді $w = 14025256741014958470038053646\ldots$

Для додатного цілого числа $k$, якщо у рядку $w$ не існує підрядка з сумою цифр, що дорівнює $k$, визначаємо $p(k)$ як нуль. Якщо існує принаймні один підрядок $w$ із сумою цифр, що дорівнює $k$, тоді $p(k) = z$, де $z$ — початкова позиція найранішого такого підрядка.

Наприклад:

Підрядки 1, 14, 1402, … із відповідними сумами цифр 1, 5, 7, … починаються з позиції 1, отже $p(1) = p(5) = p(7) = \ldots = 1$.

Підрядки 4, 402, 4025, … із відповідними сумами цифр 4, 6, 11, … починаються з позиції 2, отже $p(4) = p(6) = p(11) = \ldots = 2$.

Підрядки 02, 0252, … із відповідними сумами цифр 2, 9, … починаються з позиції 3, отже $p(2) = p(9) = \ldots = 3$.

Зверніть увагу, що підрядок 025, який починається з позиції 3, має суму цифр 7, але існував раніший підрядок (починаючи з позиції 1) із сумою цифр 7, тому $p(7) = 1$, а не 3.

Ми можемо перевірити, що для $0 < k ≤ {10}^3$, $\sum p(k) = 4742$.

Знайдіть $\sum p(k)$ для $0 < k ≤ 2 \times {10}^{15}$.
