# --description--

길이가 `n` 단위인 한 줄에 최소 길이가 `m` 단위인 빨간 블록들이 배치되어 있습니다. 서로 다른 길이일 수 있는 두 빨간 블록 사이에는 최소한 한 칸의 검은색 칸이 반드시 있습니다.

채우기 개수 함수 $F(m, n)$는 한 줄을 채우는 방법의 수를 나타냅니다.

예를 들어, $F(3, 29) = 673135$이고 $F(3, 30) = 1089155$입니다.

즉, m = 3일 때, 채우기 개수 함수가 처음으로 백만을 넘는 최솟값이 n = 30임을 알 수 있습니다.

마찬가지로, m = 10일 때 $F(10, 56) = 880711$이고 $F(10, 57) = 1148904$임을 확인할 수 있으므로, n = 57이 채우기 개수 함수가 처음으로 백만을 넘는 최솟값입니다.

m = 50일 때, 채우기 개수 함수가 처음으로 백만을 넘는 최솟값 `n`를 구하세요.

**참고:** 이 문제는 문제 114의 더 어려운 버전입니다.
