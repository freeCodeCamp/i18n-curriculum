# --interactive--

Gli array in JavaScript sono dinamici, il che significa che puoi facilmente aggiungere o rimuovere elementi da essi. Ci sono quattro metodi principali per aggiungere e rimuovere elementi dall'inizio e dalla fine di un array: `push()`, `pop()`, `shift()` e `unshift()`. Esploriamo ciascuno di questi metodi nel dettaglio.

Il metodo `push()` serve per aggiungere uno o più elementi alla fine di un array. Il valore di ritorno del metodo `push()` è la nuova lunghezza dell'array. Ecco un esempio di aggiunta di un nuovo frutto all'array `fruits` esistente:

:::interactive_editor

```js
const fruits = ["apple", "banana"];
const newLength = fruits.push("orange");
console.log(newLength); // 3
console.log(fruits); // ["apple", "banana", "orange"]
```

:::

In questo esempio, partiamo da un array chiamato `fruits` che contiene due elementi. Poi usiamo il metodo `push()` per aggiungere la stringa `orange` alla fine dell'array.

Avrai notato che usiamo `const` quando dichiariamo l'array `fruits`. Ma perché è possibile aggiungere più elementi a questo array `fruits` se `fruits` è una costante? Questo è possibile perché dichiarare un array con la parola chiave `const` crea un riferimento all'array. Mentre l'array stesso è modificabile e può essere cambiato, non puoi riassegnare un nuovo valore alla costante `fruits`, come in questo esempio:

```js
const fruits = ["apple", "banana"];
fruits = ["This", "will", "not", "work"];
console.log(fruits); // Uncaught TypeError: Assignment to constant variable. 
```

Il prossimo metodo che vedremo è il metodo `pop()`. Il metodo `pop()` rimuove l'ultimo elemento da un array e restituisce quell'elemento. Modifica anche l'array originale. Ecco come funziona:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let lastFruit = fruits.pop();
console.log(fruits); // ["apple", "banana"]
console.log(lastFruit); // "orange"
```

:::

In questo esempio, partiamo da un array di tre frutti. Il metodo `pop()` rimuove l'ultimo elemento (`orange`) dall'array e lo restituisce. L'array originale `fruits` viene modificato e contiene solo due elementi.

Il metodo `unshift()` aggiunge uno o più elementi all'inizio di un array e restituisce la sua nuova lunghezza. Funziona in modo simile a `push()`, ma modifica l'inizio dell'array invece della fine. Ecco un esempio:

:::interactive_editor

```js
let numbers = [2, 3];
let newLength = numbers.unshift(1);
console.log(numbers); // [1, 2, 3]
console.log(newLength); // 3
```

:::

In questo esempio, usiamo `unshift()` per aggiungere il numero `1` all'inizio dell'array `numbers`. Il metodo restituisce la nuova lunghezza dell'array, che è `3`.

Infine, il metodo `shift()` rimuove il primo elemento da un array e restituisce quell'elemento. È simile a `pop()`, ma agisce all'inizio dell'array invece che alla fine. Ecco come funziona:

:::interactive_editor

```js
let colors = ["red", "green", "blue"];
let firstColor = colors.shift();
console.log(colors); // ["green", "blue"]
console.log(firstColor); // "red"
```

:::

In questo esempio, partiamo da un array di tre colori. Il metodo `shift()` rimuove il primo elemento (`red`) dall'array e lo restituisce. L'array originale `colors` viene modificato per contenere solo due elementi.

Nota che mentre `push()` e `unshift()` possono aggiungere più elementi contemporaneamente, `pop()` e `shift()` rimuovono solo un elemento alla volta.
