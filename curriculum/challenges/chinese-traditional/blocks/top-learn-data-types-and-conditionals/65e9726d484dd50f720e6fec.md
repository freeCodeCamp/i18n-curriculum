# --description--

JavaScript 也能使用嚴格等號運算子 `===` 比較型別和值。這個運算子會檢查兩個值是否相等且型別相同。<dfn>例如</dfn>：

```javascript
let x = 5;
let y = "5";

let result = x === y;

console.log(result); // false
```

在這個範例中，`result` 變數會包含 `false`，因為 `x` 是數字而 `y` 是字串。但如果你使用 `==` 運算子，結果會是 `true`，因為 JavaScript 會將字串轉換成數字並比較這些值。

這個運算子稱為嚴格等號運算子，因為它會檢查值和型別的相等性。通常建議使用 `===` 運算子，以避免在比較值時出現意外結果。

還有一個嚴格不相等運算子 `!==`，用來檢查兩個值是否不相等且型別相同。

有很多方法可以使用嚴格的等號運算子，並且了解它的運作方式非常重要。
