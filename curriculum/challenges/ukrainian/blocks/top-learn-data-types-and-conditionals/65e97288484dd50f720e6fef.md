# --description--

Більш складні умовні інструкції можуть містити кілька умов. Саме тому використовують логічні оператори для поєднання кількох умов. Логічні оператори — це `&&`, `||` і `!`, які використовують для представлення `and`, `or` і `not` відповідно.

Логічний оператор `||` використовується для поєднання двох булевих умов. Він повертає `true`, якщо принаймні одна з умов є `true`. Інакше повертає `false`:

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a > b || a > c) {
  console.log("At least one of the conditions is true");
} else {
  console.log("Both of the conditions are false");
}
```

Логічний оператор `&&` використовується для поєднання двох булевих умов. Він повертає `true` лише якщо обидві умови є `true`. Інакше повертає `false`:

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a < b && a < c) {
  console.log("Both of the conditions are true");
} else {
  console.log("At least one of the conditions is false");
}
```

Логічний оператор `!` використовується для заперечення булевої умови. Він повертає `true`, якщо умова є `false`. Інакше повертає `false`:

```javascript
let a = 5;
let b = 10;

if (!(a > b)) {
  console.log("The condition is false");
} else {
  console.log("The condition is true");
}
```

У наведеному вище прикладі додаткова пара круглих дужок використовується для кращої читабельності коду. Використовувати їх не обов’язково.

# --questions--

## --text--

Вам потрібно написати інструкцію `if`, яка перевіряє такі умови у веб-застосунку, щоб показати користувачу `Welcome!`:

1. Користувач повинен або мати преміум-акаунт (`isPremium`), або бути учасником більше року (`membershipDuration` > 12 місяців).

1. Користувач не повинен бути наразі заблокований у сервісі (`!isBlocked`).

Який оператор if правильно перевіряє ці умови?

## --answers--

```javascript
if (isPremium && membershipDuration > 12 && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (isPremium || (membershipDuration > 12 && !isBlocked)) {
  console.log("Welcome!");
}
```

---

```javascript
if ((isPremium || membershipDuration > 12) && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (!isPremium || membershipDuration <= 12 || isBlocked) {
  console.log("Welcome!");
}
```
