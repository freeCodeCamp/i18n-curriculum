# --description--

Hacer `make sure` significa verificar algo cuidadosamente para estar seguro de que es correcto o que se ha hecho. Por ejemplo, puedes decir `I wanted to make sure the door is locked`, cuando verificas la cerradura para confirmar que está segura.

`Choices` son las diferentes opciones de las que puedes escoger. Por ejemplo, si estás eligiendo un sabor de helado, las `choices` pueden ser chocolate, vainilla o fresa. Significa que tienes más de una posibilidad para seleccionar.
