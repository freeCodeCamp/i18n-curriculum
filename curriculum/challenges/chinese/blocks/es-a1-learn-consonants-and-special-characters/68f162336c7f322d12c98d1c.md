# --description--

字母 `q` 在西班牙语中称为 `cu`。

它的发音类似于一个硬的 `k` 音，就像 `casa` 中的字母 `c`。

它总是出现在一个无声的 `u` 之前，后面跟着 `e` 或 `i`，例如在 `que` 或 `qui` 中。

一个例子是 `Queso`（"Cheese"）。
