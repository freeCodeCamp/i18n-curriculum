# --description--

$S_n$ を正$n$角形、つまり頂点 $v_k (k = 1, 2, \ldots, n)$ の座標が次のようになっている図形とします：

$$\begin{align}
  & x_k = cos(\frac{2k - 1}{n} × 180°) \\\\
  & y_k = sin(\frac{2k - 1}{n} × 180°)
\end{align}$$

各 $S_n$ は、周囲の点と内部のすべての点からなる塗りつぶされた図形として解釈されます。

2つの図形 $S$ と $T$ のミンコフスキー和 $S + T$ は、$S$ のすべての点と $T$ のすべての点を座標ごとに加算して得られる図形です：$(u, v) + (x, y) = (u + x, v + y)$。

例えば、$S_3$ と $S_4$ の和は、下のピンク色で示された6角形の図形になります：

<img alt="image showing S_3, S_4 and S_3 + S_4" src="https://cdn.freecodecamp.org/curriculum/project-euler/minkowski-sums.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$S_{1864} + S_{1865} + \ldots + S_{1909}$ は何角形になりますか？
