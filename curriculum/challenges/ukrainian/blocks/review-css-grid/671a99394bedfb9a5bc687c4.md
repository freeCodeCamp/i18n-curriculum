## Основи CSS Grid

- **Визначення**: CSS Grid — це двовимірна система макету, яка використовується для створення складних макетів на вебсторінках. Сітки мають рядки та колонки з пробілами між ними. Щоб задати макет сітки, потрібно встановити властивість `display` зі значенням `grid`.

```css
.container {
  display: grid;
}
```

- **Одиниця `fr` (дрібна)**: Ця одиниця представляє частку простору всередині контейнера сітки. Ви можете використовувати одиницю `fr` для створення гнучких сіток.
- **Створення пробілів між треками**: Існує три способи створити пробіли між треками в CSS Grid. Ви можете використати властивість `column-gap` для створення пробілів між колонками. Ви можете використати властивість `row-gap` для створення пробілів між рядками. Або ви можете використати скорочену властивість `gap` для створення пробілів і між рядками, і між колонками.
- **`grid-template-columns`**: Використовується для задання імен ліній і розмірів для колонок треків сітки.

```css
.container {
  display: grid;
  width: 100%;
  grid-template-columns: 30px 1fr;
}
```

- **`grid-template-rows`**: Використовується для задання імен ліній і розмірів для рядків треків сітки.
- **`grid-auto-flow`**: Визначає, як елементи, розміщені автоматично, вписуються в сітку.

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-flow: column;
}
```

- **`grid-auto-columns`**: Використовується для встановлення розміру колонок, створених неявно.

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-columns: auto;
}
```

- **`place-items`**: Використовується для вирівнювання елементів у блоковому та рядковому напрямках.
- **`align-items`**: Використовується для встановлення вирівнювання елементів у контейнері сітки.
- **Функція `repeat()`**: Ця функція використовується для повторення секцій у списку треків. Замість того, щоб писати `grid-template-columns: 1fr 1fr 1fr;`, можна використати функцію `repeat()`.

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}
```

- **Явна сітка**: Ви можете вказати кількість ліній або треків за допомогою властивостей `grid-template-columns` або `grid-template-rows`.
- **Неявна сітка**: Коли елементи розміщуються поза межами сітки, автоматично створюються рядки та колонки для цих зовнішніх елементів.
- **Функція `minmax()`**: Ця функція використовується для встановлення мінімальних і максимальних розмірів трека.

```css
.container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-auto-rows: minmax(150px, auto);
}
```

- **Розміщення на основі ліній**: Усі сітки мають лінії. Щоб вказати, де елемент починається на лінії, можна використати властивості `grid-column-start` і `grid-row-start`. Щоб вказати, де елемент закінчується на лінії, можна використати властивості `grid-column-end` і `grid-row-end`. Також можна вибрати скорочені властивості `grid-column` або `grid-row`.

Ось приклад використання властивості `grid-column`, щоб елемент розтягнувся на всі колонки.

```css
.element {
  grid-column: 1 / -1;
}
```

- **`grid-template-areas`**: Властивість використовується для надання імені елементам, які ви збираєтеся позиціонувати на сітці.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="header">Header</div>
  <div class="sidebar">Sidebar</div>
  <div class="main">Main Content</div>
  <div class="footer">Footer</div>
</div>

```

```css
.container {
  display: grid;
  grid-template-columns: 200px 1fr; 
  grid-template-rows: auto 1fr auto; 
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer"; 
  gap: 20px; 
}

.header {
  grid-area: header; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}

.sidebar {
  grid-area: sidebar;
  background-color: #f4f4f4;
  padding: 10px;
}

.main {
  grid-area: main; 
  background-color: #e0e0e0;
  padding: 10px;
}

.footer {
  grid-area: footer; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}
```

:::

## Налагодження CSS

- **Інструменти розробника (DevTools)**: Інструменти розробника дозволяють переглядати та змінювати ваш CSS у реальному часі. Панель стилів показує всі CSS-правила, застосовані до вибраного елемента, включно з успадкованими стилями. Ви можете вмикати та вимикати окремі властивості, редагувати значення і навіть додавати нові правила безпосередньо в браузері. Такий миттєвий зворотний зв’язок дуже корисний для експериментів зі стилями без зміни вихідного коду.
- **Валідатори CSS**: Валідатори використовуються для перевірки вашого CSS на відповідність офіційним специфікаціям і повідомляють про помилки або попередження. Популярним валідатором, який ви можете використати, є W3C CSS Validator.
- **Налагодження адаптивних дизайнів**: У DevTools є опція, яка дозволяє симулювати, як ваш сайт виглядає на різних розмірах екранів і пристроях. Це допомагає виявити проблеми з точками зупинки або стилі, які погано масштабуються на різних вікнах перегляду.
