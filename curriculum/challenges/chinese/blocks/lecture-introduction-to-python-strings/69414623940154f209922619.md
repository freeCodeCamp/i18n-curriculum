# --description--

在处理字串时，将不同的文本片段组合在一起是你经常会遇到的常见操作。

在 Python 中，你可以使用加号（`+`）操作符将多个字串串接在一起。这个过程称为**字串串接**。以下是如何使用加号操作符串接两个字串：

```python
my_str_1 = 'Hello'
my_str_2 = "World"

str_plus_str = my_str_1 + ' ' + my_str_2
print(str_plus_str) # Hello World
```

但请注意，这仅适用于字串。如果你尝试将字串与数字连接，你将得到一个 `TypeError`：

```python
name = 'John Doe'
age = 26

name_and_age = name + age
print(name_and_age) # TypeError: can only concatenate str (not "int") to str
```

这是因为 Python 在连接时不会自动将其他数据类型如整数转换为字串。Python 要求所有元素必须是字串才能进行连接。为了解决这个问题，你可以使用内置的 `str()` 函数将数字转换为字串，该函数返回给定对象的字串表现形式，而不修改原始对象：

```python
name = 'John Doe'
age = 26

name_and_age = name + str(age)
print(name_and_age) # John Doe26
```

你也可以使用扩充赋值操作符来进行串接。它表现为加号和等号（`+=`），并在一步中执行串接和赋值。下面是它的示例：

```py
name = 'John Doe'
age = 26

name_and_age = name  # Start with the name
name_and_age += str(age)  # Append the age as string

print(name_and_age)  # John Doe26
```

将变量和表达式插入到字串中的过程称为**字符串插值**。Python 有一类称为**f-strings**（格式化字面值的缩写）的字串，它允许你用简洁且可读的语法来处理插值。

F-strings 以 `f`（小写或大写）开头，紧跟引号，允许你在由花括弧（`{}`）表示的替换字段中嵌入变量或表达式。示例如下：

```python
name = 'John Doe'
age = 26
name_and_age = f'My name is {name} and I am {age} years old'
print(name_and_age) # My name is John Doe and I am 26 years old

num1 = 5
num2 = 10
print(f'The sum of {num1} and {num2} is {num1 + num2}') # The sum of 5 and 10 is 15
```

注意你不需要使用 `str()` 函数转换非字串类型。在上面的示例中，`age`、`num1` 和 `num2` 变量的值在插值进程中在底层被转换为字串。
