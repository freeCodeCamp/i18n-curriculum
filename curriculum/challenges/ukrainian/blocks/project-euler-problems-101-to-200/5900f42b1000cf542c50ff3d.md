# --description--

Нехай $S_m = (x_1, x_2, \ldots, x_m)$ — це $m$-кортеж додатних дійсних чисел, для якого $x_1 + x_2 + \cdots + x_m = m$, і для якого $P_m = x_1 \times {x_2}^2 \times \cdots \times {x_m}^m$ максимальний.

Наприклад, можна перевірити, що $[P_{10}] = 4112$ (де [ ] — це функція цілої частини).

Знайдіть $\sum {[P_m]}$ для $2 ≤ m ≤ 15$.
