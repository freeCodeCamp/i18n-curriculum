# --description--

El método `.catch()` es otro método asincrónico de JavaScript que puedes usar para manejar errores. Esto es útil en caso de que el `Promise` sea rechazado.

Encadena `.catch()` al último `.then()`. Pasa una función de retorno con `err` como parámetro. Dentro de la función de retorno, usa `console.error()` para registrar posibles errores en la consola con el texto `Hubo un error: ${err}`. Como estás usando `err` en el texto, no olvides usar una cadena literal de plantilla con comillas invertidas (``) en lugar de comillas simples o dobles.

**Nota**: `catch` es la última llamada encadenada a fetch, por lo que puedes terminar tu código con un punto y coma.
