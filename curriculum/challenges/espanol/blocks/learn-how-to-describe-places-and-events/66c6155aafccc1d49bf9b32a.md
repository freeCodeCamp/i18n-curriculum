Cuando alguien hace una pregunta como `Have you` + verbo ?, está tratando de averiguar si hiciste algo específico. 

En este caso, Sarah pregunta, `Have you checked the settings?` porque quiere saber si Tom ya ha tomado una acción específica que podría resolver el problema.
