# --description--

単一の Ramvok のゲームを考えます：

$t$ をゲームが続く最大ターン数とします。$t = 0$ の場合、ゲームは即座に終了します。そうでなければ、各ターン $i$ にプレイヤーはサイコロを振ります。振った後、もし $i &lt; t$ なら、プレイヤーはゲームを止めて現在の出目の値と同じ賞金を受け取るか、出目を破棄して次のターンに再挑戦するか選べます。$i = t$ の場合は出目を破棄できず、賞金を受け取らなければなりません。ゲーム開始前に $t$ はプレイヤーによって選ばれ、定数 $c$ に対して前払い費用 $ct$ を支払う必要があります。$c = 0$ の場合、$t$ は無限大に選べます（前払い費用は0）。$R(d, c)$ を、公平な $d$ 面サイコロと費用定数 $c$ のもとで最適にプレイした単一の Ramvok ゲームからプレイヤーが得る期待利益（純利益）とします。例えば、$R(4, 0.2) = 2.65$ です。プレイヤーは任意の前払い費用を支払う十分な資金があると仮定します。

次に、スーパー Ramvok のゲームを考えます：

スーパー Ramvok では、Ramvok のゲームが繰り返し行われますが、少し変更があります。各ゲームの後、サイコロが変化します。変化の手順は次の通りです：サイコロを1回振り、出た面の目が見えている場合、その面は空白に変わります。すでに空白の場合は元の値に戻ります。変化が行われた後、別の Ramvok のゲームが始まります（そのゲーム中、各ターンでサイコロを振り、値のある面が出るまで振り続けます）。プレイヤーは常にどの面が空白でどの面がそうでないかを知っています。スーパー Ramvok のゲームは、サイコロの全ての面が空白になった時点で終了します。

$S(d, c)$ を、公平な $d$ 面サイコロ（開始時は全ての面が見えている）と費用定数 $c$ のもとで最適にプレイしたスーパー Ramvok のゲームからプレイヤーが得る期待利益とします。例えば、$S(6, 1) = 208.3$ です。

$F(n) = \sum_{4 ≤ d ≤ n} \sum_{0 ≤ c ≤ n} S(d, c)$ を計算してください。

$F(20)$ を最も近い整数に丸めて求めてください。
