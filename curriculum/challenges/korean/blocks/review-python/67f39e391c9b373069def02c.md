## 파이썬이란?

- **소개**: 파이썬은 단순하고 사용하기 쉬운 범용 프로그래밍 언어입니다. 파이썬은 데이터 과학과 머신러닝, 웹 개발, 스크립팅과 자동화, 임베디드 시스템과 IoT 등 다양한 분야에서 사용됩니다.
- **주요 활용 사례**: 파이썬은 데이터 과학, 머신러닝, 웹 개발, 사이버보안, 자동화, 그리고 Raspberry Pi나 MicroPython 호환 보드 같은 마이크로컴퓨터에서 사용됩니다.

## 로컬 환경에서의 Python

- **설치**: Windows, Mac, Linux에 Python을 설치하는 가장 좋은 방법은 공식 Python 웹사이트(`https://www.python.org/`)에서 설치 프로그램을 다운로드하는 것입니다.

## 변수

- **변수 선언하기**: 변수를 선언하려면 변수 이름 다음에 대입 연산자(`=`)를 쓰고 값을 할당하세요. 값은 숫자, 문자열, 불리언 등일 수 있습니다. 다음은 몇 가지 예입니다:

```py
name = 'John Doe'
age = 25
```

- **변수 이름 관례**: 변수 이름에 사용할 관례는 다음과 같습니다:

  - 변수 이름은 숫자가 아니라 문자나 밑줄(_)로 시작해야 합니다.
  - 변수 이름에는 영숫자(a-z, A-Z, 0-9)와 밑줄(_)만 포함할 수 있습니다.
  - 변수 이름은 대소문자를 구분합니다 — `age`, `Age`, `AGE`는 모두 서로 다른 이름으로 간주됩니다.
  - 변수 이름은 `if`, `class`, `def` 같은 Python 예약 키워드일 수 없습니다.
  - 여러 단어로 된 변수 이름은 밑줄로 구분합니다. 예: `snake_case`.

## 주석

- **한 줄 주석**: 코드에 남기고 싶은 짧은 메모에 사용하세요.

```py
# This is a single line comment
```

- **여러 줄 문자열**: 더 긴 메모를 남기거나 코드 섹션을 주석 처리할 때 사용할 수 있습니다.

```py
"""
This is a multi-line string.
Here is some code commented out.

name = 'John Doe'
age = 25
"""
```

- **`print()` 함수**: 데이터를 콘솔에 출력하려면 `print()` 함수를 이렇게 사용할 수 있습니다:

```py
print('Hello world!') # Hello world!
```

## 파이썬의 일반적인 데이터 타입

- **소개**: 파이썬은 JavaScript처럼 동적 타입 언어로, 변수의 타입을 명시적으로 선언할 필요가 없습니다. 변수에 할당하는 값에 따라 언어가 변수의 타입을 자동으로 인식합니다.
- **정수**: 소수점이 없는 정수입니다:

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **실수**: 소수점을 포함한 숫자입니다:

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **문자열**: 따옴표로 감싼 문자들의 수열입니다:

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **불리언**: `true` 또는 `false`를 나타내는 값입니다:

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **집합**: 중복되지 않는 요소들의 순서 없는 모음입니다:

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **사전**: 중괄호로 감싼 키-값 쌍들의 모음입니다:

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **튜플**: 소괄호로 감싼 변경 불가능한 순서 있는 모음입니다:

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **범위**: 주로 반복문에서 사용되는 숫자의 수열입니다:

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **리스트**: 다양한 데이터 타입을 지원하는 순서 있는 요소들의 모음입니다:

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**: 값이 없음을 나타내는 특별한 값입니다:

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## 불변형과 가변형

- **불변형**: 선언 후 변경할 수 없지만, 변수에 새 값을 가리키도록 할당할 수 있습니다. 여기에는 정수, 부동소수점, 복소수, 불리언, 문자열, 튜플, 범위, 그리고 `None`가 포함됩니다.
- **가변형**: 선언 후 변경할 수 있으며, 항목을 추가, 삭제, 또는 수정할 수 있습니다. 여기에는 리스트, 집합, 사전과 같은 컬렉션 타입이 포함됩니다.
- **`type()` 함수**: 변수의 타입을 확인하려면 다음과 같이 `type()` 함수를 사용할 수 있습니다.

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **`isinstance()` 함수**: 변수의 데이터 타입이 특정 타입과 일치하는지 확인할 때 사용합니다.

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## 문자열 다루기

- **정의**: JavaScript에서 기억하듯이, 문자열은 불변이어서 한 번 생성된 후에는 변경할 수 없습니다. Python에서는 작은따옴표나 큰따옴표를 모두 사용할 수 있습니다. 규칙을 정해서 일관되게 사용하는 것이 좋습니다.

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **문자열에서 문자 접근하기**: 대괄호 표기법을 사용해 문자열에서 문자를 접근할 수 있습니다.

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **문자열 이스케이프 처리하기**: 문자열에 따옴표가 포함되어 있을 때는 역슬래시(`\`)를 사용할 수 있습니다.

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **문자열 연결하기**: 문자열을 연결하려면 `+` 연산자를 사용할 수 있습니다.

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

문자열을 연결하는 또 다른 방법은 `+=` 연산자를 사용하는 것입니다. 이 연산자는 연결과 할당을 한 번에 수행할 때 사용합니다.

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**: 포맷된 문자열 리터럴의 줄임말입니다. 보간을 처리하고 간결하고 읽기 쉬운 구문으로 문자열 연결도 할 수 있습니다.

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **문자열 슬라이싱**: 문자열의 일부를 추출할 때 사용합니다. 기본 구문은 다음과 같습니다.

```py
str[start:stop:step]
```

시작 위치는 추출을 시작할 인덱스를 나타냅니다. 종료 위치는 슬라이스가 끝나는 인덱스이며, 이 위치는 포함되지 않습니다. 단계 위치는 슬라이싱할 때 증가할 간격을 나타냅니다. 몇 가지 예는 다음과 같습니다.

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **문자열 길이 구하기**: `len()` 함수는 문자열에 포함된 문자 수를 반환하는 데 사용합니다.

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## `in` 연산자 사용하기

- **`in` 연산자**: 문자열에 특정 문자나 문자가 존재하는지 여부를 나타내는 불리언 값을 반환합니다.

## 자주 쓰이는 문자열 메서드

- **`upper()`**: 이 문자열 메서드는 모든 문자를 대문자로 변환한 새 문자열을 반환합니다.

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`lower()`**: 이 문자열 메서드는 모든 문자를 소문자로 변환한 새 문자열을 반환합니다.

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`strip()`**: 이 문자열 메서드는 지정한 앞뒤 문자를 제거한 문자열 복사본을 반환합니다(인수를 전달하지 않으면 앞뒤 공백을 제거합니다).

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**: 이 메서드는 문자열 내 모든 이전 문자열을 새 문자열로 교체한 새 문자열을 반환합니다.

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**: 이 메서드는 지정한 구분자를 사용해 문자열을 리스트로 나눕니다. 구분자는 어디서 나눌지 지정하는 문자열입니다.

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**: 이 메서드는 반복 가능한 요소들을 구분자로 연결해 문자열로 만듭니다. 반복 가능한 것은 리스트, 문자열, 튜플처럼 반복할 수 있는 요소 모음입니다.

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`startswith(prefix)`**: 이 문자열 메서드는 문자열이 지정한 접두사로 시작하는지 여부를 불리언으로 반환합니다.

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`endswith(suffix)`**: 이 문자열 메서드는 문자열이 지정한 접미사로 끝나는지 여부를 불리언으로 반환합니다.

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`find()`**: 이 문자열 메서드는 부분 문자열이 처음 나타나는 인덱스를 반환합니다. 찾지 못하면 `-1`를 반환합니다.

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`count(substring)`**: 이 문자열 메서드는 문자열 내 부분 문자열이 나타나는 횟수를 셉니다.

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`capitalize()`**: 이 문자열 메서드는 첫 문자는 대문자로, 나머지는 소문자로 변환한 새 문자열을 반환합니다.

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`isupper()`**: 이 문자열 메서드는 문자열의 모든 문자가 대문자면 `True`를, 그렇지 않으면 `False`를 반환합니다.

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`islower()`**: 이 문자열 메서드는 문자열의 모든 문자가 소문자면 `True`를, 그렇지 않으면 `False`를 반환합니다.

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`title()`**: 이 문자열 메서드는 각 단어의 첫 글자를 대문자로 변환한 새 문자열을 반환합니다.

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`maketrans()`**: 이 문자열 메서드는 1대1 문자 매핑 테이블을 만들어냅니다. 보통 `translate()` 메서드와 함께 사용해 그 테이블을 문자열에 적용하고 번역 결과를 반환합니다.

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## 정수와 부동소수점에 자주 사용하는 연산

- **기본 산술 연산**: Python에서는 정수와 부동소수점에 대해 덧셈, 뺄셈, 곱셈, 나눗셈 같은 기본 산술 연산을 할 수 있습니다.

```py
int_1 = 56
int_2 = 12
float_1 = 5.4
float_2 = 12.0

# Addition

print('Integer Addition:', int_1 + int_2) # Integer Addition: 68
print('Float Addition:', float_1 + float_2) # Float Addition: 17.4

# Subtraction

print('Int Subtraction:', int_1 - int_2) # Int Subtraction: 44
print('Float Subtraction:',  float_2 - float_1) # Float Subtraction: 6.6

# Multiplication

print('Int Multiplication:', int_1 * int_2) # Int Multiplication: 672
print('Float Multiplication:', float_2 * float_1) # Float Multiplication: 64.80000000000001

# Division

print('Int Division:', int_1 / int_2) # Int Division: 4.666666666666667
print('Float Division:', float_2 / float_1) # Float Division: 2.222222222222222
```

부동소수점과 정수를 더하면 결과가 부동소수점으로 변환됩니다.

```py
int_1 = 56
float_1 = 5.4

print(int_1 + float_1) # 61.4
```

- **나머지 연산자 (`%`)**: 이 연산자는 한 수를 다른 수로 나눈 나머지를 반환합니다.

```py
int_1 = 56
int_2 = 12

print(int_1 % int_2) # 8
```

- **몫 연산자 (`//`)**: 이 연산자는 두 수를 나눈 후 결과를 가장 가까운 정수로 내림합니다.

```py
int_1 = 56
int_2 = 12

print(int_1 // int_2) # 4
```

- **거듭제곱 연산자 (`**`)**: 이 연산자는 한 수를 다른 수의 거듭제곱으로 올립니다.

```py
int_1 = 4
int_2 = 2

print(int_1 ** int_2) # 16
```

- **`float()` 함수**: 이 함수는 정수를 부동소수점으로 변환할 때 사용합니다.

```py
num = 4

print(float(num)) # 4.0
```

- **`int()` 함수**: 이 함수는 부동소수점을 정수로 변환할 때 사용합니다.

```py
num = 4.0

print(int(num)) # 4
```

- **`round()` 함수**: 이 함수는 수를 가장 가까운 정수로 반올림할 때 사용합니다.

```py
num_1 = 3.4
num_2 = 7.7

print(round(num_1)) # 3
print(round(num_2)) # 8
```

- **`abs()` 함수**: 이 함수는 수의 절댓값을 반환할 때 사용합니다.

```py
num = -13

print(abs(num)) # 13
```

- **`bin()` 함수**: 이 함수는 정수를 문자열 형태의 이진수로 변환할 때 사용합니다.

```py
num = 56

print(bin(num))  # 0b111000
```

- **`oct()` 함수**: 이 함수는 정수를 문자열 형태의 8진수로 변환할 때 사용합니다.

```py
num = 56

print(oct(num))  # 0o70
```

- **`hex()` 함수**: 이 함수는 정수를 문자열 형태의 16진수로 변환할 때 사용합니다.

```py
num = 56

print(hex(num))  # 0x38
```

- **`pow()` 함수**: 이 함수는 한 수를 다른 수의 거듭제곱으로 올릴 때 사용합니다.

```py
result = pow(2, 3) 
print(result)  # 8
```

## 복합 할당

- **정의**: 복합 할당은 이항 연산과 할당을 한 단계로 결합한 것입니다. 변수에 다른 값과 연산을 적용한 후 결과를 같은 변수에 다시 저장합니다.

```py
# Addition assignment 
my_var = 10
my_var += 5

print(my_var) # 15

# Subtraction assignment
count = 14
count -= 3

print(count) # 11

# Multiplication assignment 
product = 65
product *= 7

print(product) # 455

# Division assignment 
price = 100
price /= 4

print(price) # 25.0

# Floor Division assignment 
total_pages = 23
total_pages //= 5

print(total_pages) # 4

# Modulo assignment 
bits = 35
bits %= 2

print(bits) # 1

# Exponentiation assignment 
power = 2
power **= 3

print(power) # 8
```

비트 연산자용 복합 할당 연산자도 있습니다. 여기에는 `&=`, `^=`, `>>=`, 그리고 `<<=`가 포함됩니다.

## 함수 다루기

- **정의**: 함수는 입력(인수)을 받아 출력을 반환하는 재사용 가능한 코드 조각입니다. 함수를 호출하려면 함수 이름 뒤에 괄호를 붙여 참조하세요.

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call
print(result) # 7
```

함수가 명시적으로 값을 반환하지 않으면 기본 반환값은 `None`입니다.

```py
def greet():
    print('hello') 

result = greet() # hello
print(result) # None
```

매개변수에 기본값을 이렇게 지정할 수도 있습니다.

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

함수를 올바른 인수 개수 없이 호출하면 `TypeError`가 발생합니다.

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

name = input('What is your name?') # User types 'Kolade' and presses Enter

## 파이썬의 범위

- **지역 범위**: 함수나 클래스 내부에서 선언된 변수가 그 함수나 클래스 내에서만 접근될 수 있는 경우입니다.

```py
def my_func():
    num = 10
    print(num)
```

- **포함 범위**: 한 함수 안에 중첩된 함수가 중첩된 함수가 속한 함수의 변수를 접근할 수 있는 경우입니다.

```py
def outer_func():
    msg = 'Hello there!'
    
    def inner_func():
        print(msg)
    inner_func()

print(outer_func()) # Hello there!
```

- **전역 범위**: 함수나 클래스 외부에서 선언되어 프로그램 어디서나 접근할 수 있는 변수를 의미합니다.

```py
tax = 0.70 

def get_total(subtotal):
    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **내장 범위**: 파이썬에서 미리 정의된 함수, 모듈, 키워드, 객체에 예약된 이름들입니다.

```py
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```

print(3 == 4) # False

- **Not equal (`!=`)**: 두 값이 같지 않은지 확인합니다.

print(3 != 4) # True

print(3 > 4) # False

- **Strictly less than (`<`)**: 한 값이 다른 값보다 작은지 확인합니다.

print(3 < 4) # True
```

print(3 >= 4) # False
```

- **Less than or equal(`<=`)**: 한 값이 다른 값보다 작거나 같은지 확인합니다.

```py
print(3 <= 4) # True
```

## `if`, `elif` 그리고 `else` 문장 다루기

- **`if` 문장**: 어떤 것이 참인지 아닌지를 판단하는 조건입니다. 조건이 `True`로 평가되면 해당 코드 블록이 실행됩니다.

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **`elif` 절**: `if` 문장 뒤에 오는 조건입니다. `elif` 조건이 `True`로 평가되면 해당 코드 블록이 실행됩니다.

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **`else` 절**: 다른 어떤 조건도 `True`로 평가되지 않을 때 실행됩니다.

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

다음과 같이 중첩된 `if` 문장도 사용할 수 있습니다:

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## 참 같은 값과 거짓 같은 값

- **정의**: Python에서 모든 값은 고유한 불리언 값을 가지며, 논리적 맥락에서 `True` 또는 `False`으로 처리되어야 하는지를 내장된 감각으로 판단합니다. 많은 값이 참 같은 값으로 간주되어 논리적 맥락에서 `True`로 평가됩니다. 반면 거짓 같은 값은 `False`로 평가됩니다. 다음은 거짓 같은 값의 몇 가지 예입니다:

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

0이 아닌 숫자나 비어 있지 않은 문자열 같은 다른 값들은 참 같은 값입니다.

## `bool()` 함수 사용하기

- **정의**: 값이 참 같은 값인지 거짓 같은 값인지 확인하려면 내장 `bool()` 함수를 사용할 수 있습니다. 이 함수는 값을 불리언 등가물로 명시적으로 변환하며, 참 같은 값에는 `True`을, 거짓 같은 값에는 `False`를 반환합니다. 다음은 몇 가지 예시입니다:

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## 불리언 연산자와 단락 평가

- **정의**: 여러 표현식을 결합해 코드에서 더 복잡한 의사결정 논리를 만들 수 있게 해주는 특별한 연산자입니다. Python에는 세 가지 불리언 연산자가 있습니다: `and`, `or`, 그리고 `not`.
- **`and` 연산자**: 이 연산자는 두 피연산자를 받아 첫 번째 피연산자가 거짓 같은 값이면 그 값을 반환하고, 그렇지 않으면 두 번째 피연산자를 반환합니다. 표현식이 참 같은 값을 가지려면 두 피연산자 모두 참 같은 값이어야 합니다.

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

조건문에서 `and` 연산자를 이렇게 사용할 수도 있습니다:

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **`or` 연산자**: 이 연산자는 첫 번째 피연산자가 참 같은 값이면 그 값을 반환하고, 그렇지 않으면 두 번째 피연산자를 반환합니다. `or` 표현식은 적어도 하나의 피연산자가 참 같은 값일 때 참 같은 값을 결과로 냅니다. 예시는 다음과 같습니다:

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

`and` 연산자와 마찬가지로, 조건문에서 `or` 연산자를 이렇게 사용할 수 있습니다:

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **단락 평가**: `and`와 `or` 연산자는 단락 평가 연산자로 알려져 있습니다. 단락 평가는 Python이 왼쪽에서 오른쪽으로 값을 검사하며 최종 결과가 확정되는 즉시 정지하는 것을 의미합니다.
- **`not` 연산자**: 이 연산자는 하나의 피연산자를 받아 그 불리언 값을 반전시킵니다. 참 같은 값은 `False`로, 거짓 같은 값은 `True`로 변환합니다. 이전에 살펴본 연산자들과 달리 `not`는 항상 `True` 또는 `False`를 반환합니다. 몇 가지 예시는 다음과 같습니다:

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

조건문에서 `not` 연산자의 예시는 다음과 같습니다:

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```

## Python 리스트

- **소개**: Python에서 리스트 데이터 타입은 문자열, 숫자 또는 다른 리스트로 구성될 수 있는 순서가 있는 요소들의 수열입니다. 리스트는 변경 가능하며 0부터 시작하는 인덱스를 가집니다.

```python
cities = ['Los Angeles', 'London', 'Tokyo']
```

- **리스트에서 요소 접근하기**: `cities` 리스트에서 요소에 접근하려면 수열 내 인덱스 번호를 참조하면 됩니다:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # Los Angeles
```

- **음수 인덱싱을 사용해 요소 접근하기**: 어떤 리스트의 마지막 요소에 접근하려면 `-1`를 인덱스 번호로 사용할 수 있습니다:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # Tokyo
```

- 음수 인덱싱은 리스트의 시작이 아닌 끝에서부터 요소에 접근할 때 사용합니다. 인덱스는 `0`부터 시작합니다.

- **`list()` 생성자를 사용해 리스트 만들기**: 리스트는 `list()` 생성자를 사용해 만들 수도 있습니다. `list()` 생성자는 반복 가능한 객체를 리스트로 변환하는 데 사용됩니다:

```python
developer = 'Jessica'

print(list(developer)) 
# Result: ['J', 'e', 's', 's', 'i', 'c', 'a']
```

- **리스트 길이 구하기**: `len()` 함수를 사용해 리스트의 길이를 구할 수 있습니다:

```python
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

- **리스트 변경 가능성**: 리스트는 변경 가능하므로 유효한 인덱스 번호를 전달하는 한 리스트 내 어떤 요소든 업데이트할 수 있습니다. 특정 인덱스의 리스트를 업데이트하려면 해당 인덱스에 새 값을 할당하면 됩니다:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

- **인덱스 범위 초과 오류**: 리스트의 범위를 벗어난 인덱스(양수 또는 음수)를 전달하면 `IndexError` 오류가 발생합니다:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
"""
```

- **리스트에서 요소 제거하기**: `del` 키워드를 사용해 리스트에서 요소를 제거할 수 있습니다:

```python
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

- **리스트에 요소가 존재하는지 확인하기**: `in` 키워드를 사용해 리스트에 특정 요소가 있는지 확인할 수 있습니다:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **리스트 중첩하기**: 리스트는 다른 리스트 안에 중첩될 수 있습니다:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

- 중첩된 리스트에 접근하려면 리스트가 0부터 시작하는 인덱스를 가지므로 `2` 인덱스를 사용해 접근해야 합니다.

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

- 그 중첩된 리스트에서 두 번째 언어에 더 접근하려면 `1` 인덱스를 사용해야 합니다:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # Rust
```

- **리스트에서 값 풀기**: 리스트에서 값을 풀어내는 것은 리스트의 값을 새 변수에 할당하는 기법입니다. 다음은 `developer` 리스트를 `name`, `age`, `job`이라는 새 변수로 풀어내는 예입니다:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer
```

- 할당 연산자 왼쪽의 변수 개수가 리스트 내 아이템 총 개수와 일치하지 않으면 `ValueError` 오류가 발생합니다.

- **리스트에서 나머지 아이템 모으기**: 리스트에서 남은 요소들을 모으려면 별표(`*`) 연산자를 이렇게 사용할 수 있습니다:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer
```

- **리스트 슬라이싱**: 슬라이싱은 슬라이스 연산자 `:`를 사용해 리스트의 일부를 접근하는 개념입니다. 인덱스 `1`에서 시작해 `3` 이전까지 슬라이싱하려면 다음 구문을 사용하세요:

```python
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie']
desserts[1:3] # ['Cookies', 'Ice Cream']
```

- **단계 간격**: 인덱스 사이의 증가량을 결정하는 단계 간격을 지정할 수도 있습니다. 슬라이싱을 사용해 짝수만 추출하는 예는 다음과 같습니다:

```python
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

## 리스트 메서드

- **append()**: 리스트 끝에 항목을 추가할 때 사용합니다. 다음은 `append()` 메서드를 사용해 `numbers` 리스트에 숫자 `6`을 추가하는 예시입니다:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **리스트 추가하기**: `append()` 메서드는 한 리스트를 다른 리스트 끝에 추가할 때도 사용할 수 있습니다:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

- **extend()**: 리스트 끝에 여러 항목을 추가할 때 사용합니다. 다음은 `numbers` 리스트 끝에 숫자 `6`, `8`, `10`를 추가하는 예시입니다:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

- **insert()**: 리스트의 특정 인덱스에 항목을 삽입할 때 사용합니다. 다음은 `insert()` 메서드를 사용하는 예시입니다:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

- **remove()**: 리스트에서 항목을 제거할 때 사용합니다. `remove()` 메서드는 리스트에서 해당 항목의 첫 번째 발생만 제거합니다:

```py
numbers = [1, 2, 3, 4, 5, 5, 5]
numbers.remove(5)

print(numbers) # [1, 2, 3, 4, 5, 5]

```

- **pop()**: 리스트에서 특정 항목을 제거하고 그 값을 반환할 때 사용합니다:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

- `pop` 메서드에 요소를 지정하지 않으면 마지막 요소가 제거됩니다.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

- **clear()**: 리스트의 모든 항목을 제거할 때 사용합니다:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

- **sort()**: `sort()` 메서드는 리스트 내 요소를 제자리에서 정렬할 때 사용합니다. 다음은 무작위 `numbers` 리스트를 제자리에서 정렬하는 예시입니다:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

- **sorted()**: 원본 리스트를 수정하지 않고 새로 정렬된 리스트를 반환하는 내장 함수입니다:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
print(numbers) # [19, 2, 35, 1, 67, 41]
```

- **reverse()**: 리스트 내 요소의 순서를 뒤집을 때 사용합니다:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **index()**: 리스트에서 특정 요소가 처음 나타나는 인덱스를 찾을 때 사용합니다:

```python
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

- `index()` 메서드로 요소를 찾을 수 없으면 결과는 `ValueError`가 됩니다.

## Python의 튜플

- **정의**: 튜플은 값을 순서대로 나열한 수열을 만들 때 사용하는 Python 데이터 타입입니다. 튜플은 다양한 데이터 타입을 혼합해서 포함할 수 있습니다:

```py
developer = ('Alice', 34, 'Rust Developer')
```

- 튜플은 불변(immutable)이므로, 한 번 생성된 튜플의 요소는 변경할 수 없습니다. 튜플의 항목 중 하나를 수정하려 하면 `TypeError`가 발생합니다:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **튜플 요소 접근**: 튜플에서 요소를 접근하려면 대괄호 표기법과 인덱스 번호를 사용하세요:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

- 음수 인덱스를 사용하면 튜플의 끝에서부터 요소를 접근할 수 있습니다:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- 튜플 길이보다 크거나 같은 인덱스 번호를 전달하면 `IndexError`가 발생합니다:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- 튜플은 `tuple()` 생성자를 사용해 만들 수도 있습니다. 생성자 안에는 문자열, 리스트, 다른 튜플 같은 반복 가능한(iterable) 객체를 전달할 수 있습니다.

```python
developer = 'Jessica'

print(tuple(developer)) 
# Result: ('J', 'e', 's', 's', 'i', 'c', 'a')
```

- **튜플에 항목이 있는지 확인하기**: 튜플에 특정 항목이 있는지 확인하려면 `in` 키워드를 이렇게 사용하세요:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **튜플 풀기(unpacking)**: 튜플에서 항목을 이렇게 풀 수 있습니다:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

- 튜플에서 남은 요소들을 모두 모아야 할 때는 별표(`*`) 연산자를 이렇게 사용하세요:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer
```

- **튜플 슬라이싱**: 슬라이싱을 사용해 튜플의 일부를 추출할 수 있습니다. 예를 들어 `pie`와 `cookies` 항목을 별도의 튜플로 슬라이스할 수 있습니다:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

- **튜플에서 항목 제거하기**: 튜플은 불변이므로 항목을 제거하려 하면 `TypeError`가 발생합니다:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

- **튜플과 리스트 중 언제 사용할까?**: 요소를 추가, 제거, 수정할 수 있는 동적인 컬렉션이 필요하면 리스트를 사용하세요. 고정되고 불변인 데이터 집합을 다룰 때는 튜플을 사용하세요.

## 일반적인 튜플 메서드

- **count()**: 튜플에서 특정 항목이 몇 번 나타나는지 확인하는 데 사용합니다. 예를 들어, 튜플에서 언어 `Python`이 몇 번 나타나는지 확인할 수 있습니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

- 튜플에 `count()` 메서드에 지정한 항목이 전혀 없으면 반환값은 `0`이 됩니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

- `count()` 메서드에 인수를 전달하지 않으면 Python은 `TypeError`를 반환합니다.

- **index()**: 튜플에서 특정 항목이 있는 인덱스를 찾는 데 사용합니다. 다음은 `index()` 메서드를 사용해 언어 `Python`의 인덱스를 찾는 예입니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

- 지정한 항목을 찾을 수 없으면 Python은 `ValueError`를 반환합니다.

- `index()` 메서드에 선택적 시작 인덱스를 전달해 튜플에서 항목 검색을 시작할 위치를 지정할 수 있습니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

- 선택적 종료 인덱스를 `index()` 메서드에 전달해 튜플에서 항목 검색을 멈출 위치를 지정할 수도 있습니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

- **sorted()**: 반복 가능한 객체의 요소를 정렬하고 새 정렬된 리스트를 반환하는 데 사용합니다. 다음은 `sorted()` 함수를 사용해 숫자 리스트를 새로 만드는 예입니다:

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

- **정렬 동작 수정하기**: 선택적 `key`와 `reverse` 인수를 사용해 반복 가능한 객체의 정렬 방식을 사용자 지정할 수 있습니다. 다음은 `key` 인수를 사용해 튜플 항목을 길이 기준으로 정렬하는 예입니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

- `reverse` 인수를 사용해 값을 역순으로 정렬한 새 리스트를 만들 수 있습니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

## 파이썬의 반복문

- **정의**: 반복문은 코드 블록을 정해진 횟수만큼 반복 실행할 때 사용합니다.

- **`for` 반복문**: 리스트, 튜플, 문자열 같은 수열을 순회하며 각 항목에 대해 코드 블록을 실행할 때 사용합니다. 다음은 리스트를 순회하며 각 언어를 콘솔에 출력하는 `for` 반복문 예제입니다:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)

"""
Result 

Rust
Java
Python
C++
"""
```

- 다음은 문자열 `code`을 순차적으로 반복하며 각 문자를 출력하는 `for` 반복문 예제입니다:

```py
for char in 'code':
    print(char)

"""
Result 

c
o
d
e
"""
```

- `for` 반복문은 중첩할 수 있습니다. 다음은 중첩된 `for` 반복문 예제입니다:

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)

"""
Result

Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
"""

```

- **`while` 반복문**: 조건이 거짓이 될 때까지 코드 블록을 반복 실행합니다. 다음은 추측 게임에 `while` 반복문을 사용하는 예제입니다:

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')

"""
Result

Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
"""

```

- **`break`와 `continue` 문**: 반복문 내에서 반복 실행 방식을 조절할 때 사용합니다.

- `break` 문은 특정 조건이 충족되면 즉시 반복문을 종료할 때 사용합니다. 다음은 `developer_names` 리스트에 대해 `break` 문을 사용하는 예제입니다:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

- `continue` 문은 현재 반복을 건너뛰고 다음 반복으로 넘어갈 때 사용합니다. 다음은 `continue` 문을 `if` 문 대신 사용하는 예제입니다:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

- `for`와 `while` 반복문 모두 `else` 절과 함께 사용할 수 있으며, 이 절은 반복문이 `break`로 종료되지 않았을 때만 실행됩니다:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

## 수열과 반복문에서의 사용

- **`range` 함수**: 정수 수열을 생성하는 데 사용합니다.

```py
range(start, stop, step)
```

- 필수 `stop` 인수는 생성할 숫자 수열의 끝점을 나타내는 정수(포함하지 않음)입니다. 다음은 `range` 함수를 사용하는 예입니다:

```py
for num in range(3):
    print(num)
```

- `start` 인수를 지정하지 않으면 기본값은 0입니다. 기본적으로 정수 수열은 1씩 증가합니다. 선택적 `step` 인수를 사용해 기본 증가 값을 변경할 수 있습니다. 다음은 2부터 11 미만(즉, 10 포함)까지의 짝수 수열을 생성하는 예입니다.

```py
for num in range(2, 11, 2):
    print(num)
```

- `range` 함수에 인수를 전혀 제공하지 않으면 빈 수열이 생성됩니다.

- `range` 함수는 인수로 정수만 허용하며, 실수를 사용하면 오류가 발생합니다.

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- `step` 인수에 음수를 사용하면 감소하는 정수 수열을 생성할 수 있습니다.

```py
for num in range(40, 0, -10):
    print(num)
```

- `range` 함수는 `list` 생성자와 함께 사용해 정수 리스트를 만들 수도 있습니다. `list` 생성자는 반복 가능한 객체를 리스트로 변환하는 데 사용됩니다. 다음은 2부터 10까지의 짝수 정수 리스트를 생성하는 예입니다.

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

## Python의 `enumerate()`와 `zip()` 함수

- **`enumerate()`**: 수열을 순회하면서 각 항목의 인덱스를 추적하는 데 사용합니다. `enumerate()` 함수는 반복 가능한 객체를 인수로 받아, 그 안의 각 항목의 인덱스와 값을 포함하는 `enumerate` 객체를 반환합니다.

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')

# Result
# Index 0 and language Spanish
# Index 1 and language English
# Index 2 and language Russian
# Index 3 and language Chinese

```

- `enumerate()` 함수는 `for` 반복문 밖에서도 사용할 수 있습니다:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

print(list(enumerate(languages)))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

- `enumerate()` 함수는 선택적 `start` 인수를 받아서 카운트 시작 값을 지정할 수 있습니다. 이 인수를 생략하면 카운트는 `0`에서 시작합니다.

- **`zip()`**: 여러 반복 가능한 객체를 병렬로 순회하는 데 사용합니다. 다음은 `zip()` 함수를 사용해 `developers`과 `ids`을 순회하는 예입니다:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')


"""
Result

Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
"""

```

## 파이썬의 리스트 내포

- **정의**: 리스트 내포는 대괄호 안에 반복문과 조건문을 직접 결합하여 한 줄로 새로운 리스트를 만드는 방법입니다. 이렇게 하면 코드가 더 짧아지고 종종 읽기 쉬워집니다.

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

## 반복 가능한 함수

- **`filter()`**: 조건에 따라 반복 가능한 객체에서 요소를 필터링하는 데 사용합니다. 조건을 만족하는 요소만 포함하는 이터레이터를 반환합니다. 다음은 네 글자보다 긴 단어만 새 리스트로 만드는 예시입니다:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**: 반복 가능한 객체의 각 항목에 함수를 적용하고 결과로 새 반복 가능한 객체를 반환하는 데 사용합니다. 다음은 `map()` 함수를 사용해 섭씨 온도 리스트를 화씨로 변환하는 예시입니다:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**: 리스트나 튜플 같은 반복 가능한 객체에서 합계를 구하는 데 사용합니다. 다음은 `sum()` 함수를 사용하는 예시입니다:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- 선택적으로 `start` 인수를 전달해 합계의 초기값을 설정할 수도 있습니다. 다음은 위치 인수로 `start` 인수를 사용한 업데이트된 예시입니다:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- 대신 다음과 같이 키워드 인수로 `start` 인수를 사용할 수도 있습니다:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## 람다 함수

- **정의**: Python에서 람다 함수는 이름이 없는 함수를 간결하게 만드는 방법입니다(익명 함수).
- 람다 함수는 종종 다른 함수에 인수로 사용됩니다. 다음은 람다 함수의 예입니다:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

- 람다 함수를 사용할 때 모범 사례는 변수에 할당하지 않고, 간단하고 읽기 쉽게 유지하며, 짧고 일회성 함수에 사용하는 것입니다.

## 딕셔너리

- **딕셔너리**: 딕셔너리는 키-값 쌍의 모음을 저장하는 내장 자료 구조입니다. 키는 변경 불가능한 데이터 유형이어야 합니다. 파이썬 딕셔너리의 일반적인 구문은 다음과 같습니다:

```python
dictionary = {
    key1: value1,
    key2: value2
}
```

- **`dict()` 생성자**: `dict()` 생성자는 딕셔너리를 만드는 또 다른 방법입니다. `dict()` 생성자에 튜플 목록을 인수로 전달합니다. 이 튜플들은 첫 번째 요소로 키를, 두 번째 요소로 값을 포함합니다.

```python
pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
```

- **대괄호 표기법**: 키-값 쌍의 값을 접근하려면 대괄호 표기법으로 알려진 구문을 사용할 수 있습니다.

```python
dictionary[key]
```

## 자주 사용하는 딕셔너리 메서드

- **`get()` 메서드**: `get()` 메서드는 키와 연결된 값을 가져옵니다. 대괄호 표기법과 비슷하지만, 기본값을 설정할 수 있어 키가 없을 때 발생하는 오류를 방지합니다.

```python
dictionary.get(key, default)
```

- **`keys()`와 `values()` 메서드**: `keys()`와 `values()` 메서드는 각각 딕셔너리의 모든 키와 값을 포함하는 뷰 객체를 반환합니다. 뷰 객체는 데이터를 별도로 복사하지 않고 딕셔너리 내용을 볼 수 있는 방법입니다.

```python
pizza = {
    'name': 'Margherita Pizza',
    'price': 8.9,
    'calories_per_slice': 250
}

pizza.keys()
# dict_keys(['name', 'price', 'calories_per_slice'])

pizza.values()
# dict_values(['Margherita Pizza', 8.9, 250])
```

- **`items()` 메서드**: `items()` 메서드는 키와 값을 모두 포함하는 딕셔너리의 모든 키-값 쌍을 담은 뷰 객체를 반환합니다.

```python
pizza.items()
# dict_items([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250)])
```

- **`clear()` 메서드**: `clear()` 메서드는 딕셔너리의 모든 키-값 쌍을 제거합니다.

```python
pizza.clear()
```

- **`pop()` 메서드**: `pop()` 메서드는 첫 번째 인수로 지정한 키를 가진 키-값 쌍을 제거하고 그 값을 반환합니다. 키가 없으면 두 번째 인수로 지정한 기본값을 반환합니다. 기본값이 지정되지 않았고 키가 없으면 `KeyError`가 발생합니다.

```python
pizza.pop('price', 10)
pizza.pop('total_price') # KeyError
```

- **`popitem()` 메서드**: Python 3.7 이상에서 `popitem()` 메서드는 마지막에 삽입된 항목을 제거합니다.

```python
pizza.popitem()
```

- **`update()` 메서드**: `update()` 메서드는 다른 딕셔너리의 키-값 쌍으로 현재 딕셔너리를 업데이트합니다. 공통 키가 있으면 값이 덮어쓰이고, 새로운 키는 새 키-값 쌍으로 추가됩니다.

```python
pizza.update({ 'price': 15, 'total_time': 25 })
```

## 딕셔너리 순회하기

- **값 순회하기**: 딕셔너리의 값들을 순회해야 할 때는 `for` 루프를 `values()`와 함께 작성하여 딕셔너리의 모든 값을 가져올 수 있습니다.

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,
    'Headphones': 70,
}

for price in products.values():
    print(price)
```

출력:

```md
990
600
250
70
```

- **키 순회하기**: 위 `products` 딕셔너리의 키를 순회해야 할 때는 `products.keys()` 또는 `products`를 직접 작성할 수 있습니다.

```python
for product in products.keys():
    print(product)
    
# Or

for product in products:
    print(product)
```

출력:

```md
Laptop
Smartphone
Tablet
Headphones
```

- **키-값 쌍 순회하기**: 키와 해당하는 값을 동시에 순회해야 할 때는 `products.items()`를 순회할 수 있습니다. 키와 해당 값이 포함된 개별 튜플을 얻습니다.

```python
for product in products.items():
    print(product)
```

출력:

```md
('Laptop', 990)
('Smartphone', 600)
('Tablet', 250)
('Headphones', 70)
```

키와 값을 별도의 루프 변수에 저장하려면 쉼표로 구분해야 합니다. 첫 번째 변수는 키를 저장하고, 두 번째 변수는 값을 저장합니다.

```python
for product, price in products.items():
    print(product, price)
```

출력:

```md
Laptop 990
Smartphone 600
Tablet 250
Headphones 70
```

- **`enumerate()` 함수**: 카운터를 추적하면서 딕셔너리를 순회해야 할 때는 `enumerate()` 함수를 호출할 수 있습니다. 이 함수는 `enumerate` 객체를 반환하는데, 각 항목에 정수를 할당하여 카운터처럼 작동합니다. 카운터는 어떤 숫자부터 시작할 수 있지만 기본적으로 0부터 시작합니다.

인덱스와 항목을 별도의 루프 변수에 할당하는 것이 `enumerate()`를 사용하는 일반적인 방법입니다. 예를 들어 `products.items()`를 사용하면 인덱스뿐 아니라 전체 키-값 쌍도 얻을 수 있습니다:

```python
for index, product in enumerate(products.items()):
    print(index, product)
```

출력:

```md
0 ('Laptop', 990)
1 ('Smartphone', 600)
2 ('Tablet', 250)
3 ('Headphones', 70)
```

카운트의 초기 값을 사용자 지정하려면 `enumerate()`에 두 번째 인수를 전달하세요. 예를 들어, 여기서는 카운트를 1부터 시작합니다.

```python
for index, product in enumerate(products.items(), 1):
    print(index, product)
```

출력:

```md
1 ('Laptop', 990)
2 ('Smartphone', 600)
3 ('Tablet', 250)
4 ('Headphones', 70)
```

## 집합

- **집합**: 집합은 중복 값을 허용하지 않는 Python의 내장 자료 구조입니다. 집합은 변경 가능하고 순서가 없기 때문에 요소들이 특정한 순서로 저장되지 않아 인덱스나 키로 접근할 수 없습니다. 또한 집합에는 숫자, 문자열, 튜플과 같은 변경 불가능한 데이터 타입의 값만 포함할 수 있습니다.

- **집합 정의하기**: 집합을 정의하려면 중괄호 안에 요소들을 작성하고 쉼표로 구분하세요.

```python
my_set = {1, 2, 3, 4, 5}
```

- **빈 집합 정의하기**: 빈 집합을 정의해야 할 때는 `set()` 함수를 사용해야 합니다. 빈 중괄호만 작성하면 자동으로 사전이 생성됩니다.

```python
set() # Set
{}    # Dictionary
```

## 자주 사용하는 집합 메서드

- **`add()` 메서드**: 새로운 요소를 인수로 전달하여 `add()` 메서드로 집합에 요소를 추가할 수 있습니다.

```python
my_set.add(6)
```

- **`remove()`와 `discard()` 메서드**: 집합에서 요소를 제거하려면, 제거할 요소를 인수로 전달하여 `remove()` 메서드나 `discard()` 메서드를 사용할 수 있습니다. 요소가 없으면 `remove()` 메서드는 `KeyError` 예외를 발생시키지만 `discard()` 메서드는 발생시키지 않습니다.

```python
my_set.remove(4)
my_set.discard(4)
```

- **`clear()` 메서드**: `clear()` 메서드는 집합의 모든 요소를 제거합니다.

```python
my_set.clear()
```

## 수학적 집합 연산

- **`issubset()`와 `issuperset()` 메서드**: `issubset()` 메서드는 한 집합이 다른 집합의 부분집합인지, `issuperset()` 메서드는 한 집합이 다른 집합의 상위집합인지 확인합니다.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 5}

print(your_set.issubset(my_set)) # True
print(my_set.issuperset(your_set)) # True
```

- **`isdisjoint()` 메서드**: `isdisjoint()` 메서드는 두 집합이 서로 공통 원소가 없는지, 즉 서로소인지 확인합니다.

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

print(my_set.isdisjoint(your_set)) # True
```

- **합집합 연산자 (`|`)**: 합집합 연산자 `|` 는 두 집합의 모든 원소를 포함하는 새 집합을 반환합니다.

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

my_set | your_set # {1, 2, 3, 4, 5, 6}
```

- **교집합 연산자 (`&`)**: 교집합 연산자 `&` 는 두 집합이 공통으로 가진 원소만 포함하는 새 집합을 반환합니다.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set & your_set # {2, 3, 4}
```

- **차집합 연산자 (`-`)**: 차집합 연산자 `-` 는 첫 번째 집합에는 있지만 두 번째 집합에는 없는 원소들로 이루어진 새 집합을 반환합니다.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set - your_set # {1, 5}
```

- **대칭차집합 연산자 (`^`)**: 대칭차집합 연산자 `^` 는 첫 번째 집합 또는 두 번째 집합 중 한 쪽에만 있는 원소들로 이루어진 새 집합을 반환합니다.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set ^ your_set # {1, 5, 6}
```

- **`in` 연산자**: `in` 연산자를 사용해 원소가 집합에 포함되어 있는지 확인할 수 있습니다.

```python
print(5 in my_set)
```

## Python 표준 라이브러리

- **Python 표준 라이브러리**: 라이브러리는 함수, 클래스, 자료 구조처럼 미리 작성되어 재사용할 수 있는 코드를 제공하여 프로젝트에서 다시 사용할 수 있게 합니다. Python은 많은 문제와 작업에 대해 표준화된 해결책을 구현한 내장 모듈을 포함하는 방대한 표준 라이브러리를 갖추고 있습니다. 인기 있는 내장 모듈의 예로는 `math`, `random`, `re` ("정규 표현식"의 약자), 그리고 `datetime`이 있습니다.

## 가져오기 문장

- **가져오기 문장**: 내장 모듈에 정의된 요소에 접근하려면 가져오기 문장을 사용하세요. 가져오기 문장은 일반적으로 파일 상단에 작성합니다. 가져오기 문장은 함수, 클래스, 상수, 변수 및 모듈에 정의된 다른 모든 요소에 대해 동일하게 작동합니다.

- **기본 가져오기 문장**: `import` 키워드 다음에 모듈 이름을 써서 사용할 수 있습니다.

```python
import module_name
```

그런 다음, 해당 모듈의 함수를 호출해야 할 경우, 점 표기법을 사용하여 모듈 이름 다음에 함수 이름을 씁니다.

```python
module_name.function_name()
```

예를 들어, `math` 모듈을 가져오고 36의 제곱근을 구하려면 코드에 다음과 같이 작성합니다.

```python
import math

math.sqrt(36)
```

- **다른 이름으로 모듈 가져오기**: 모듈을 다른 이름(별칭이라고도 함)으로 가져와야 할 경우, 가져오기 문장 끝에 `as` 다음에 별칭을 쓸 수 있습니다. 이는 긴 모듈 이름을 다루거나 이름 충돌을 피할 때 자주 사용합니다.

```python
import module_name as module_alias
```

예를 들어, 코드에서 `math` 모듈을 `m` 라고 부르려면 이렇게 별칭을 할당할 수 있습니다.

```python
import math as m
```

그런 다음 별칭을 사용해 모듈의 요소에 접근할 수 있습니다.

```python
m.sqrt(36)
```

- **특정 요소만 가져오기**: 모듈의 모든 것이 필요하지 않을 때는 `from` 를 사용해 특정 요소만 가져올 수 있습니다. 이 경우 가져오기 문장은 `from` 로 시작하고, 모듈 이름, `import` 키워드, 그리고 가져올 요소 이름들이 뒤따릅니다.

```python
from module_name import name1, name2
```

그런 다음 Python 스크립트에서 모듈 접두사 없이 이 이름들을 사용할 수 있습니다. 예를 들어:

```python
from math import radians, sin, cos

angle_degrees = 40
angle_radians = radians(angle_degrees)

sine_value = sin(angle_radians)
cos_value = cos(angle_radians)

print(sine_value) # 0.6427876096865393
print(cos_value)  # 0.766044443118978
```

이 방법은 유용하지만, 이미 같은 이름의 함수나 변수가 있을 경우 이름 충돌이 발생할 수 있으니 주의하세요. 어떤 유형의 가져오기 문장을 사용할지 결정할 때 참고하세요.

이 이름들에 별칭을 할당해야 한다면, `as` 키워드 다음에 별칭을 써서 할당할 수도 있습니다.

```python
from module_name import name1 as alias1, name2 as alias2
```

- **별표(*)를 사용한 가져오기 문장 (`*`)**: 별표는 해당 모듈의 모든 것을 가져오되, 모듈 이름을 접두사로 사용하지 않고 가져오겠다는 뜻입니다.

```python
from module_name import *
```

예를 들어, `math` 모듈을 이렇게 가져오면, 모듈 이름을 접두사로 지정하지 않고도 그 모듈에 정의된 모든 함수를 호출할 수 있습니다.

```python
from math import *
print(sqrt(36))  # 6.0
```

하지만 일반적으로 권장하지 않습니다. 이름 공간 충돌이 발생할 수 있고, 이름이 어디서 왔는지 알기 어려워지기 때문입니다.

## `if __name__ == '__main__'`

- **`__name__` 변수**: `__name__`는 Python에서 특별한 내장 변수입니다. Python 파일을 직접 실행할 때, Python은 이 변수의 값을 문자열 `"__main__"`로 설정합니다. 하지만 Python 파일이 다른 Python 스크립트에 모듈로 가져와질 경우, `__name__` 변수의 값은 그 모듈의 이름으로 설정됩니다.

이 때문에 Python 스크립트에서 이 조건문을 자주 볼 수 있습니다. 이 조건문은 Python 스크립트가 메인 프로그램으로 실행될 때만 실행하고 싶은 코드를 포함합니다.

```python
if __name__ == '__main__': 
    # Code
```

## 파이썬에서 자주 발생하는 오류

- **SyntaxError**: 코드가 파이썬 구문 규칙을 따르지 않을 때 발생하는 오류입니다. 예를 들어, `print("Hello there"` 코드는 닫는 괄호가 없기 때문에 `SyntaxError: '(' was never closed` 메시지와 함께 구문 오류가 발생합니다.
- **NameError**: 정의하지 않은 변수나 함수를 접근하려 할 때 파이썬이 발생시키는 오류입니다. 예를 들어, 코드에 `print(username)` 라인이 있지만 `username` 변수가 먼저 정의되어 있지 않으면 `NameError: name 'username' is not defined` 메시지와 함께 이름 오류가 발생합니다.
- **TypeError**: 서로 호환되지 않는 데이터 타입끼리 연산을 수행할 때 파이썬이 발생시키는 오류입니다. 예를 들어, 문자열과 숫자를 더하려 하면 `TypeError: can only concatenate str (not "int") to str` 오류가 발생합니다.
- **IndexError**: 리스트나 튜플, 문자열 같은 수열에서 존재하지 않는 인덱스에 접근할 때 발생하는 오류입니다. 예를 들어, `Hello world` 문자열에서 마지막 문자의 인덱스는 `10`입니다. 그런데 `greet = "hello world"; print(greet[11])` 방식으로 문자를 접근하면 `IndexError: string index out of range` 메시지와 함께 오류가 발생합니다.
- **AttributeError**: 해당 타입의 객체에 존재하지 않는 메서드나 속성을 사용하려 할 때 파이썬이 발생시키는 오류입니다. 예를 들어, `"hello".append("!")` 같은 문자열에서 `.append()` 메서드를 호출하면 `AttributeError: 'str' object has no attribute 'append'` 메시지와 함께 오류가 발생합니다.

## 파이썬에서 좋은 디버깅 기법

- **`print` 함수 사용하기**: 디버깅할 때 코드의 여러 지점에 `print()` 문을 삽입하면 변수 값과 코드 흐름을 확인하는 데 도움이 됩니다.

## 예외 처리

- **try**: 예외가 발생할 수 있는 코드 블록을 실행할 때 사용합니다. `try` 블록은 오류가 발생할 가능성이 있는 부분이고, `except` 블록은 지정한 예외가 발생했을 때 실행됩니다. 예를 들어:

  ```py
  try:
    print(22 / 0)
  except ZeroDivisionError:
    print('You can\'t divide by zero!')
    # You can't divide by zero!
  ```

  여러 개의 `except` 블록을 연결해서 다양한 예외를 처리할 수도 있습니다:

  ```py
  try:
    number = int(input('Enter a number: '))
    print(22 / number)
  except ZeroDivisionError:
    print('You cannot divide by zero!')
    # You cannot divide by zero! prints when you enter 0
  except ValueError:
    print('Please enter a valid number!')
    # Please enter a valid number! prints when you don't enter an integer
  ```

- **else**와 **finally**: 이 블록들은 `try`를 확장합니다. 예외가 발생하지 않으면 `else` 블록이 실행되고, `finally` 블록은 오류 여부와 상관없이 항상 실행됩니다.

  ```py
  try:
    result = 100 / 4
  except ZeroDivisionError:
    print('You cannot divide by zero!') # This will not run
  else:
    print(f'Result is {result}') # Result is 25.0
  finally:
    print('Execution complete!') # Execution complete!
  ```

- **예외 객체**: 예외 자체에 접근해서 디버깅하거나 직접 오류 메시지를 출력할 수 있습니다. 예외 객체에 접근하려면 `as` 키워드를 사용해야 합니다. 예를 들어:

  ```py
  try:
      value = int('This will raise an error')
  except ValueError as e:
      print(f'Caught an error: {e}')
      # Caught an error: invalid literal for int() with base 10: 'This will raise an error'
  ```

- **raise 문**: 특정 조건이 충족될 때 수동으로 예외를 발생시킬 수 있습니다. 예를 들어:

  ```py
  def divide(a, b):
      if b == 0:
          raise ZeroDivisionError('You cannot divide by zero')
      return a / b
  ```

## 예외 신호 보내기

`raise` 문장은 사용자 정의 예외를 만들 때도 유용합니다. 이 문장을 사용해 사용자 정의 메시지와 함께 예외를 발생시킬 수 있습니다. 다음은 그 예입니다:

```py
class InvalidCredentialsError(Exception):
    def __init__(self, message="Invalid username or password"):
        self.message = message
        super().__init__(self.message)

def login(username, password):
    stored_username = "admin"
    stored_password = "password123"
    
    if username != stored_username or password != stored_password:
        raise InvalidCredentialsError()
    
    return f"Welcome, {username}!"
```  

다음은 `login` 함수와 `InvalidCredentialsError` 예외를 함께 사용하는 방법입니다:

```py
# 로그인 실패 시도
try:
    message = login("user", "wrongpassword")
except InvalidCredentialsError as e:
    print(f"Login failed: {e}")
else:
    print(message)

# 로그인 성공 시도
try:
    message = login("admin", "password123")
except InvalidCredentialsError as e:
    # 로그인 성공했으므로 이 블록은 실행되지 않습니다
    print(f"Login failed: {e}")
else:
    # 'try' 블록이 예외 없이 완료되면 else 블록이 실행됩니다
    print(message)
```

`raise` 문장은 `from` 키워드와 함께 사용해 예외를 연결할 수도 있습니다. 이렇게 하면 서로 다른 오류 간의 관계를 보여줄 수 있습니다:

```py
def parse_config(filename):
  try:
      with open(filename, 'r') as file:
          data = file.read()
          return int(data)
  except FileNotFoundError:
      raise ValueError('Configuration file is missing') from None
  except ValueError as e:
      raise ValueError('Invalid configuration format') from e

config = parse_config('config.txt')
```

## Python 클래스와 객체

- **클래스 정의**: 클래스는 객체를 생성하기 위한 청사진입니다. 클래스는 속성과 메서드를 통해 객체가 가질 동작을 정의합니다. 다음은 Python에서 클래스 정의의 기본 예시입니다.

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f'{self.name.upper()} says woof woof!')
```

- **객체 생성**: 객체는 클래스의 인스턴스입니다. 필요한 인수를 사용해 클래스를 호출하여 생성합니다.

```python
dog1 = Dog('Jack', 3)
dog2 = Dog('Thatcher', 5)

dog1.bark()  # JACK says woof woof!
dog2.bark()  # THATCHER says woof woof!
```

- **객체에서 메서드 호출**: 객체에서 메서드를 호출해 동작을 수행하거나 정보를 가져올 수 있습니다.

```python
object_name1.method_name()
object_name2.method_name()
```

- **클래스와 객체의 차이점**: 클래스는 재사용 가능한 템플릿이고, 객체는 실제 데이터를 가진 그 클래스의 특정 인스턴스입니다.

## 속성

- **인스턴스 속성**: `__init__()`에서 `self`를 사용해 정의하며, 각 객체마다 고유합니다.
- **클래스 속성**: 클래스 내부에 직접 정의되며 모든 인스턴스가 공유합니다.

```python
class Dog:
    species = 'French Bulldog'  # Class attribute

    def __init__(self, name):
        self.name = name  # Instance attribute

print(Dog.species) # French Bulldog

jack = Dog('Jack')
print(jack.name)     # Jack
print(jack.species)  # French Bulldog
```

## 메서드

- **메서드**: 객체의 속성에 작용하는 클래스 내부에 정의된 함수입니다.

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
print(my_car_1.describe())  # This car is a red Tesla Model S
```

- **메서드 접근**: 점 표기법을 사용해 객체의 메서드를 호출합니다. 다음은 두 개의 서로 다른 자동차 객체에서 `describe` 메서드를 호출하는 예입니다:

```python
class Car:
    def __init__(self, color, model):
        self.color = color  
        self.model = model  

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
my_car_2 = Car('green', 'Lamborghini Revuelto')

print(my_car_1.describe()) # Calling method using the dot notation

print(my_car_2.describe()) # Calling method using the dot notation
```

## Dunder (매직) 메서드

- **정의**: 앞뒤에 이중 밑줄이 붙은 특별한 메서드(e.g., `__init__`, `__len__`, `__str__`, `__eq__`)입니다. Python은 내장 연산을 위해 내부적으로 이 메서드를 사용합니다.

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **dunder 메서드를 간접적으로 호출하기**: dunder 메서드를 직접 호출할 필요가 없습니다. 특정 동작이 발생할 때 Python이 자동으로 호출합니다. 이러한 동작에는 다음이 포함됩니다:

  - **덧셈, 뺄셈, 곱셈, 나눗셈 등 산술 연산**. 덧셈에는 `__add__()`, 뺄셈에는 `__sub__()`, 곱셈에는 `__mul__()`, 나눗셈에는 `__truediv__()`가 호출됩니다.
        
  - **문자열 연결, 반복, 포맷팅, 텍스트 변환 같은 문자열 연산**. 연결에는 `__add__()`, 반복에는 `__mul__()`, 포맷팅에는 `__format__()`, 텍스트 변환에는 `__str__()`와 `__repr__()` 등이 호출됩니다.
        
  - **동등 비교, 작음, 큼 등 비교 연산**. 동등 비교에는 `__eq__()`, 작음에는 `__lt__()`, 큼에는 `__gt__()` 등이 호출됩니다.
        
  - **객체를 반복 가능하게 만들고 항목을 순회하는 반복 연산**. 반복자를 반환할 때 `__iter__()`, 다음 항목을 가져올 때 `__next__()`가 호출됩니다.

## 실제 예제: 쇼핑 카트

- **더블 언더스코어 메서드를 가진 Cart 클래스**: 내장 동작으로 항목을 추가, 제거, 반복하고 내용을 확인할 수 있습니다.

```python
class Cart:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

    def remove(self, item):
        if item in self.items:
            self.items.remove(item)
        else:
            print(f'{item} is not in cart')

    def list_items(self):
        return self.items

    def __len__(self):
        return len(self.items)

    def __getitem__(self, index):
        return self.items[index]

    def __contains__(self, item):
        return item in self.items

    def __iter__(self):
        return iter(self.items)

cart = Cart()
cart.add('Laptop')
print(len(cart))        # 1
print('Laptop' in cart) # True
```

## 객체지향 프로그래밍이란?

- **객체지향 프로그래밍**: 개발자가 코드 내 모든 것을 실제 세계의 객체처럼 다루는 프로그래밍 방식입니다. 흔히 OOP라고 부릅니다. 코드를 효과적으로 구성하고 관리하는 데 도움이 되는 네 가지 핵심 원칙은 **캡슐화**, **상속**, **다형성**, 그리고 **추상화**입니다.
- **클래스**: 객체를 생성하기 위한 청사진입니다. 클래스에서 생성된 모든 객체는 데이터를 정의하는 속성과 객체의 동작을 결정하는 메서드를 가집니다.

## 캡슐화란 무엇인가요?

- **캡슐화**: 객체의 속성과 메서드를 하나의 단위로 묶는 것입니다. 객체의 내부 상태를 단순한 공개 메서드와 속성 뒤에 숨겨서, 이들이 문처럼 작동하도록 합니다. 그 문 뒤에는 데이터가 어떻게 변경되고 누가 볼 수 있는지 제어하는 비공개 속성과 메서드가 있습니다.
- **캡슐화 예시**: 지갑 잔액을 추적하려면 입금과 출금을 허용하되, 지갑 잔액 자체를 아무나 조작하지 못하게 해야 합니다.

```py
class Wallet:
   def __init__(self, balance):
       self.__balance = balance # Private attribute

   def deposit(self, amount):
       if amount > 0:
           self.__balance += amount # Add to the balance safely

   def withdraw(self, amount):
       if 0 < amount <= self.__balance:
           self.__balance -= amount # Remove from the balance safely

account = Wallet(500)
print(account.__balance) # AttributeError: 'Wallet' object has no attribute '__balance'
```

- **속성에 단일 밑줄과 이중 밑줄 접두사를 붙이는 차이점**: 단일 밑줄을 붙이면 내부 용도로 사용한다는 관례를 의미하며, 외부에서 속성 접근을 막지는 않습니다. 이중 밑줄을 붙이면 클래스 외부에서 속성이나 메서드에 접근하는 것을 효과적으로 막습니다.

## 게터와 세터란 무엇인가요?

- **게터와 세터**: 클래스의 속성에 어떻게 접근하고 수정할지 제어할 수 있게 해주는 메서드입니다. 게터로 값을 가져오고 세터로 값을 설정합니다.
- **속성**: 게터와 세터를 연결해 주며, 데이터에 접근할 수 있게 합니다. 값을 가져오거나 설정하거나 삭제할 때 내부에서 추가 로직을 실행합니다.
- **메서드 대신 속성을 사용하는 이유**: 속성은 메서드보다 가독성이 좋고 코드가 깔끔해집니다. 소괄호 없이 점 표기법으로 일반 속성처럼 값을 접근할 수 있게 해줍니다.
- **게터 만들기**: 게터를 만들려면 `@property` 데코레이터를 사용합니다. 다음은 원의 반지름을 가져오는 게터입니다:

```py
class Circle:
    def __init__(self, radius):
        self._radius = radius

    @property
    def radius(self): # A getter to get the radius
        return self._radius
  
    @property
    def area(self):  # A getter to calculate area
        return 3.14 * (self._radius ** 2)

my_circle = Circle(3)

print(my_circle.radius) # 3
print(my_circle.area) # 28.26
```

- **세터 만들기**: 반지름을 설정하는 세터를 만들려면 같은 이름의 메서드를 하나 더 정의하고 그 위에 `@<property_name>.setter`를 사용해야 합니다:

```py
class Circle:
    def __init__(self, radius):
        self._radius = radius

    @property
    def radius(self):  # A getter to get the radius
        return self._radius

    @radius.setter
    def radius(self, value):  # A setter to set the radius
        if value <= 0:
            raise ValueError('Radius must be positive')
        self._radius = value

my_circle = Circle(3)
print('Initial radius:', my_circle.radius) # Initial radius: 3

my_circle.radius = 8
print('After modifying the radius:', my_circle.radius) # After modifying the radius: 8
```

- **파이썬이 게터와 세터를 처리하는 방법**: 게터와 세터를 정의하면, 파이썬은 일반 속성 구문을 사용할 때 자동으로 내부에서 이들을 호출합니다:

```py
my_circle.radius # This will call the getter
my_circle.radius = 4 # This will call the setter
```

 값을 설정할 때는 속성 이름 자체에 할당하면 `RecursionError`가 발생하므로, 보통 밑줄이 붙은 별도의 내부 이름을 사용해 값을 저장하세요.

- **딜리터**: 세터와 게터로 값을 설정하고 가져온 후, `deleter`로 삭제하는 방식을 제어할 수 있습니다. 딜리터는 속성에 `del` 문을 사용할 때 사용자 정의 로직을 실행합니다. 딜리터를 만들려면 `@<property_name>.deleter` 데코레이터를 사용하세요.

```py
  # Deleter
    @radius.deleter
    def radius(self):
        print("Deleting radius...")
        del self._radius
```

## 상속이란 무엇이며 어떻게 코드 재사용을 촉진하나요?

- **상속**: 자식 클래스가 부모 클래스의 속성과 메서드를 사용하는 과정입니다. 상속은 코드 재사용을 촉진하고, 명확한 계층 구조를 제공하며, 모든 것을 다시 작성하지 않고도 동작을 맞춤화할 수 있게 합니다. 상속을 구현하려면 자식 클래스가 부모 클래스의 이름을 받습니다:

```py
class Parent:
    # Parent attributes and methods

class Child(Parent):
    # Child inherits, extends, and/or overrides where necessary
```

- **단일 상속과 다중 상속**: 자식 클래스가 하나의 부모로부터 속성과 메서드를 상속받으면 위에서 본 것처럼 이를 **단일 상속**이라고 합니다. 자식 클래스가 둘 이상의 부모로부터 속성과 메서드를 상속받으면 이를 **다중 상속**이라고 합니다. 구문은 다음과 같습니다:

```py
class Parent:
    # Attributes and methods for Parent

class Child:
    # Attributes and methods for Child

class GrandChild(Parent, Child):
    # GrandChild inherits from both Parent and Child
    # GrandChild can combine or override behavior from each
```

- **`super()` 함수**: 클래스가 해당 메서드를 다르게 구현하거나 확장할 때, 코드를 중복하지 않고 부모 클래스의 메서드를 **호출**할 수 있게 해주는 함수입니다.

## 다형성이란 무엇이며 어떻게 코드 재사용을 촉진하나요?

- **다형성**: 서로 다른 클래스가 같은 메서드 이름을 사용하지만, 호출될 때 각 클래스가 다르게 구현하는 객체지향 원리입니다. 구문은 다음과 같습니다:

```py
class A:
   def action(self): ...

class B:
   def action(self): ...

class C:
   def action(self): ...

Class().method()  # Works for A, B, or C
```

- **상속 기반 다형성**: 부모가 메서드를 설정하고, 각 자식 클래스가 이를 자신에게 맞게 변형합니다.

## 이름 맹글링이란 무엇이며 어떻게 작동하나요?

- **이름 맹글링**: Python이 내부적으로 이중 밑줄로 시작하는 속성 이름에 밑줄과 클래스 이름을 접두사로 붙여 이름을 바꾸는 과정입니다. 예를 들어 `__attribute`가 `_ClassName__attribute`으로 바뀝니다.
- **이름 맹글링의 목적**: 이름 맹글링의 주요 목적은 상속을 사용할 때 속성과 메서드가 실수로 덮어쓰이는 것을 방지하는 것입니다. 다음 코드는 이를 더 쉽게 이해할 수 있게 합니다:

```py
class Parent:
    def __init__(self):
        self.__data = 'Parent data'

class Child(Parent):
    def __init__(self):
        super().__init__()
        self.__data = 'Child data'

c = Child()
print(c.__dict__) # {'_Parent__data': 'Parent data', '_Child__data': 'Child data'}
```

## 추상화란 무엇이며 복잡한 시스템을 체계적으로 유지하는 데 어떻게 도움이 될까요?

- **추상화**: 객체나 시스템의 복잡한 구현 세부사항을 숨기고 필수 기능만 보여 주는 프로그래밍 개념입니다. Python과 다른 프로그래밍 언어에서 추상화는 재사용성을 높여 복잡한 시스템을 단순화합니다.
- **추상화의 예**: 일상생활에서 추상화의 좋은 예는 자동차입니다. 엔진이나 브레이크가 어떻게 작동하는지 몰라도 핸들, 페달, 변속기만 사용하면 됩니다.
- **Python에서 추상화 구현 방법**: Python은 `abc` 모듈을 통해 추상화를 구현합니다. 이 모듈은 `ABC` 클래스(추상 기본 클래스)와 `@abstractmethod` 데코레이터를 제공합니다. 추상 기본 클래스(ABC)는 하위 클래스가 반드시 구현해야 하는 공통 메서드와 프로프를 정의하며, 인스턴스화할 수 없습니다.
- **추상 메서드 정의 방법**: 추상 메서드는 `@abstractmethod`로 정의하며, 기본 구현이 있더라도 하위 클래스에서 반드시 재정의해야 합니다. 추상화의 기본 구문은 다음과 같습니다:

```py
from abc import ABC, abstractmethod

# Define an abstract base class
class AbstractClass(ABC):
    @abstractmethod
    def abstract_method(self):
        pass

# Concrete subclass that implements the abstract method
class ConcreteClassOne(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassOne')

# Another concrete subclass
class ConcreteClassTwo(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassTwo')
```

## 알고리즘과 빅오 표기법

- **알고리즘**: 문제를 해결하거나 작업을 수행하기 위한 명확한 지침 집합입니다. 알고리즘은 유한한 단계 내에 종료되어야 하며, 각 단계는 정확하고 명확해야 합니다.

- **빅오 표기법**: 입력 크기가 증가할 때 알고리즘의 최악의 성능 또는 성장률을 나타냅니다. 상수 요소와 하위 차수 항은 무시하고, 입력 크기에 따른 자원 사용량의 증가에 초점을 맞춥니다.

#
## 일반적인 시간 복잡도

- **O(1) - 상수 시간**: 입력 크기에 관계없이 알고리즘이 동일한 시간 내에 실행됩니다.

```python
def check_even_or_odd(number):
    if number % 2 == 0:
        return 'Even'
    else:
        return 'Odd'
```

- **O(log n) - 로그 시간**: 입력이 커질수록 시간이 천천히 증가합니다. 문제 크기를 일정 비율로 반복해서 줄이는 알고리즘(예: 이진 탐색)에서 흔히 나타납니다.

- **O(n) - 선형 시간**: 실행 시간이 입력 크기에 비례하여 증가합니다.

```python
for grade in grades:
    print(grade)
```

- **O(n log n) - 로그-선형 시간**: 병합 정렬과 퀵 정렬 같은 효율적인 정렬 알고리즘의 일반적인 시간 복잡도입니다.

- **O(n²) - 이차 시간**: 실행 시간이 제곱에 비례하여 증가합니다. 중첩된 반복문에서 자주 나타납니다.

```python
for i in range(n):
    for j in range(n):
        print("Hello, World!")
```

#
## 공간 복잡도

- **O(1) - 상수 공간**: 입력 크기에 관계없이 동일한 메모리를 사용합니다.
- **O(n) - 선형 공간**: 메모리 사용량이 입력 크기에 비례하여 증가합니다.
- **O(n²) - 이차 공간**: 메모리 사용량이 입력 크기의 제곱에 비례하여 증가합니다.

## 문제 해결 기법

- **문제 이해하기**: 문제 설명을 여러 번 읽으세요. 입력, 예상 출력, 그리고 입력을 출력으로 변환하는 방법을 파악하세요.

- **의사 코드**: 언어에 구애받지 않는 알고리즘 논리를 고수준으로 설명하는 방법입니다. 일반적인 문장과 `IF`, `ELSE`, `FOR`, `WHILE` 같은 프로그래밍 구문을 혼합해 사용합니다.

```md
GET original_string
SET reversed_string = ""
FOR EACH character IN original_string:
  ADD character TO THE BEGINNING OF reversed_string
DISPLAY reversed_string
```

- **경계 사례**: 알고리즘이 처리해야 하는 범위의 경계에 해당하는 특정하고 유효한 입력입니다. 항상 경계 사례를 고려하고 테스트하세요.

## 배열

- **정적 배열**: 초기화 시 고정된 크기를 가집니다. 요소들이 인접한 메모리 위치에 저장됩니다. 프로그램 실행 중 크기를 변경할 수 없습니다.

- **동적 배열**: 프로그램 실행 중 자동으로 크기가 커지거나 작아질 수 있습니다. 필요할 때 더 큰 배열로 자동 복사를 통해 크기 조절을 처리합니다.

#
## 파이썬 리스트 (동적 배열)

```python
numbers = [3, 4, 5, 6]

# Access elements
numbers[0]  # 3

# Update elements
numbers[2] = 16

# Add elements
numbers.append(7)
numbers.insert(3, 15)  # Insert at specific index

# Remove elements
numbers.pop(2)  # Remove at specific index
numbers.pop()   # Remove last element
```

#
## 동적 배열의 시간 복잡도

- **접근**: O(1)
- **끝에 삽입**: 평균 O(1), 크기 조절 시 O(n)
- **중간에 삽입**: O(n)
- **삭제**: 중간은 O(n), 끝은 O(1)

## 스택

- **스택**: 후입선출(LIFO) 자료 구조입니다. 요소는 오직 맨 위에서만 추가되고 제거됩니다.

- **푸시 연산**: 스택 맨 위에 요소를 추가하는 연산입니다. 시간 복잡도: O(1).

- **팝 연산**: 스택 맨 위에서 요소를 제거하는 연산입니다. 시간 복잡도: O(1).

```python
# Using Python list as stack
stack = []

# Push operations
stack.append(1)
stack.append(2)
stack.append(3)

# Pop operations
top_element = stack.pop()  # Returns 3
```

## 큐

- **큐**: 선입선출(FIFO) 자료 구조입니다. 요소는 뒤에 추가되고 앞에서 제거됩니다.

- **인큐 연산**: 큐의 뒤에 요소를 추가하는 연산입니다. 시간 복잡도: O(1).

- **디큐 연산**: 큐의 앞에서 요소를 제거하는 연산입니다. 시간 복잡도: O(1).

```python
from collections import deque

# Using deque for efficient queue operations
queue = deque()

# Enqueue operations
queue.append(1)
queue.append(2)
queue.append(3)

# Dequeue operations
first_element = queue.popleft()  # Returns 1
```

## 연결 리스트

- **연결 리스트**: 각 노드가 데이터와 다음 노드에 대한 참조를 포함하는 선형 자료 구조입니다. 노드들이 사슬처럼 연결되어 있습니다.

#
## 단일 연결 리스트

- **구조**: 각 노드는 데이터와 다음 노드에 대한 하나의 참조를 가집니다.
- **순회**: 머리 노드에서 꼬리 노드 방향으로만 이동할 수 있습니다.
- **머리 노드**: 리스트의 첫 번째 노드로, 보통 직접 접근할 수 있는 유일한 노드입니다.
- **꼬리 노드**: 리스트의 마지막 노드로, `None`를 가리킵니다.

#
## 연산과 시간 복잡도

- **처음에 삽입**: O(1)
- **끝에 삽입**: O(n) - 끝까지 순회해야 합니다
- **중간에 삽입**: O(n) - 위치까지 순회해야 합니다
- **처음에서 삭제**: O(1)
- **끝에서 삭제**: O(n) - 이전 노드를 찾기 위해 순회해야 합니다
- **중간에서 삭제**: O(n) - 노드를 찾기 위해 순회해야 합니다

#
## 이중 연결 리스트

- **구조**: 각 노드는 데이터와 두 개의 참조를 가집니다: 다음 노드와 이전 노드.
- **순회**: 양방향으로 이동할 수 있습니다.
- **메모리**: 추가 참조 때문에 단일 연결 리스트보다 더 많은 메모리가 필요합니다.

## 해시 맵과 세트

#
## 맵과 해시 맵

- **맵 (추상 자료형)**: 키-값 쌍 컬렉션을 관리합니다. 모든 키는 고유해야 하지만 값은 중복될 수 있습니다.

- **해시 맵**: 해싱 기법을 사용해 맵 ADT를 구체적으로 구현한 것입니다. 해시 함수로 키의 해시 값을 생성하고, 이 값이 기본 배열 내 저장 위치를 결정합니다.

#
## 파이썬 딕셔너리(해시 맵)

```python
# Creating dictionaries
my_dictionary = {
    "A": 1,
    "B": 2, 
    "C": 3
}

# Alternative creation
my_dictionary = dict(A=1, B=2, C=3)

# Access and modify
value = my_dictionary["A"]  # 1
my_dictionary["A"] = 4      # Update value
del my_dictionary["A"]      # Remove key-value pair

# Check membership
"C" in my_dictionary

# Get keys, values, items
my_dictionary.keys()
my_dictionary.values()
my_dictionary.items()
```

#
## 해시 맵의 시간 복잡도

- **평균 경우**: 삽입, 조회, 삭제 모두 O(1)
- **최악 경우**: 해시 충돌이 많이 발생하면 O(n)

#
## 세트

- **세트**: 중복 없는 고유 요소들의 순서 없는 집합입니다. 중복을 허용하지 않고 특정 순서를 유지하지 않습니다.

- **불변 요소만 가능**: 세트는 숫자, 문자열, 튜플 같은 불변 데이터 타입만 포함할 수 있습니다. 해시 값이 변하지 않아야 하기 때문입니다.

```python
# Creating sets
numbers = {1, 2, 3, 4}
empty_set = set()  # Must use set(), not {}

# Add and remove elements
numbers.add(5)
numbers.remove(4)      # Raises KeyError if not found
numbers.discard(4)     # No error if not found

# Set operations
set_a = {1, 2, 3, 4}
set_b = {2, 3, 4, 5, 6}

# Union, intersection, difference, symmetric difference
set_a.union(set_b)                    # or set_a | set_b
set_a.intersection(set_b)             # or set_a & set_b
set_a.difference(set_b)               # or set_a - set_b
set_a.symmetric_difference(set_b)     # or set_a ^ set_b

# Subset and superset checks
set_a.issubset(set_b)
set_a.issuperset(set_b)
set_a.isdisjoint(set_b)

# Membership testing
5 in numbers
```

#
## 세트의 시간 복잡도

- **평균 경우**: 추가, 제거, 멤버십 검사 모두 O(1)
- **최악 경우**: 해시 충돌로 인해 O(n)

## 해시 충돌

- **해시 충돌**: 서로 다른 두 키가 같은 해시 값을 생성할 때 발생합니다.

- **충돌 해결 전략**:
  - **체이닝**: 각 배열 인덱스가 같은 해시 값을 가진 모든 요소를 저장하는 연결 리스트를 가리킵니다.
  - **오픈 어드레싱**: 미리 정해진 수열을 사용해 다음 사용 가능한 인덱스를 찾습니다.

## 각 자료 구조를 언제 사용해야 할까요

- **리스트**: 순서가 있고 인덱스로 접근해야 하며 크기를 미리 모를 때
- **스택**: LIFO 작업에 (실행 취소 기능, 표현식 평가, 백트래킹)
- **큐**: FIFO 작업에 (작업 스케줄링, 너비 우선 탐색)
- **연결 리스트**: 시작 부분에 자주 삽입/삭제가 필요하고 크기를 모를 때, 임의 접근이 필요 없을 때
- **해시 맵**: 빠른 키-값 조회, 발생 횟수 세기, 캐싱에
- **셋**: 고유성 검사, 수학적 집합 연산, 중복 제거에

## 검색 알고리즘

검색 알고리즘은 특정 항목 목록 내에서 목표를 찾을 수 있게 해줍니다.

컴퓨터 과학에서 알아야 할 검색 알고리즘은 두 가지가 있습니다. 바로 **선형 검색**과 **이진 검색** 알고리즘입니다. 두 알고리즘의 차이점과 각각 언제 사용하는지 이해하는 것이 중요합니다.

#
## 선형 검색

- 선형 검색은 항목 목록을 처음부터 끝까지 반복하며 각 항목을 확인하다가 목표 항목을 찾으면 멈춥니다.
- 목표 항목을 찾으면 목록에서 해당 항목이 위치한 인덱스를 반환합니다.
- 목표를 찾지 못하면 `-1`를 반환하는데, 이는 대부분의 프로그래밍 언어에서 **유효하지 않은 인덱스**를 의미합니다.
- 선형 검색은 목표를 찾을 때까지 모든 항목을 확인하기 때문에 항목이 많은 목록에서는 효율적이지 않습니다.
- 선형 검색의 시간 복잡도는 `O(n)`인데, 목록 크기에 따라 검색 시간이 선형적으로 증가하기 때문입니다.
- 선형 검색의 공간 복잡도는 `O(1)`인데, 목록을 검색하는 데 추가 공간이 필요하지 않기 때문입니다.

#
## 이진 검색

- 이진 검색은 항목 목록을 반으로 나누고, 목록 중간에 목표 값이 있는지 확인하는 방식으로 작동합니다.
- 이진 검색이 작동하려면 목록의 항목들이 오름차순으로 정렬되어 있어야 합니다.
- 이진 검색은 목록을 반으로 나누고 목표가 없는 쪽 절반을 무시하기 때문에 항목이 많은 목록을 검색할 때 더 효율적인 알고리즘입니다.
- 목표 항목이 목록 중간에 있으면 해당 항목의 인덱스를 반환합니다.
- 항목을 찾지 못하면 알고리즘은 목표 항목이 목록의 왼쪽 절반에 있는지 오른쪽 절반에 있는지 확인합니다.
- 목표 항목을 찾을 때까지 남은 목록 부분을 계속 반으로 나눕니다.
- 최종적으로 목록에서 목표 항목을 찾지 못하면 `-1`를 반환합니다.
- 이진 검색의 시간 복잡도는 `O(log n)`인데, 목록 크기에 따라 검색 시간이 로그 함수적으로 증가하기 때문입니다.
- 이진 검색의 공간 복잡도는 `O(1)`인데, 목록을 검색하는 데 추가 공간이 필요하지 않기 때문입니다.

#
## 선형 검색과 이진 검색의 차이점

- 이진 검색은 선형 검색에 비해 항목이 많은 목록에 더 적합합니다.
- 선형 검색의 시간 복잡도는 `O(n)`인데, 목록 크기에 따라 검색 시간이 선형적으로 증가하기 때문입니다.
- 이진 검색의 시간 복잡도는 `O(log n)`인데, 목록 크기에 따라 검색 시간이 로그 함수적으로 증가하기 때문입니다.

## 정렬 알고리즘과 분할 정복

컴퓨터 과학에서 분할 정복은 문제를 더 작은 하위 문제로 나누어 해결하기 쉽게 만드는 기법입니다. 재귀는 분할 정복에서 자주 사용하는 기법이며, 분할 정복은 병합 정렬과 같은 효율적인 정렬 알고리즘을 구현하는 데 강력한 전략입니다.

#
## 병합 정렬

- 병합 정렬은 분할 정복 방식을 따르는 정렬 알고리즘입니다.
- 리스트를 재귀적으로 더 작은 하위 리스트로 나누어 각 하위 리스트가 하나의 요소만 포함할 때까지 나눕니다.
- 그런 다음 하위 리스트들을 정렬된 순서로 반복해서 병합합니다.
- 병합 정렬의 시간 복잡도는 `O(n log n)`인데, 이는 리스트가 계속 반으로 나누어지고 `(log n)` 다시 병합되기 때문입니다 `(O(n))`.
- 병합 정렬의 공간 복잡도는 `O(n)`인데, 이는 제자리 정렬 알고리즘이 아니기 때문입니다.

## 그래프 개요

그래프는 엣지(연결선)로 연결된 노드(정점) 집합입니다. 각 노드는 여러 다른 노드와 연결되어 네트워크를 형성할 수 있습니다. 그래프의 다양한 유형은 다음과 같습니다:

- 방향 그래프: 엣지에 방향이 있으며(한 노드에서 다른 노드로), 보통 직선과 화살표로 나타냅니다.
- 무방향 그래프: 엣지에 방향이 없으며, 단순한 선으로 나타냅니다.
- 정점 그래프: 각 노드에 레이블이나 식별자가 연결되어 있습니다.
- 순환 그래프: 사이클(시작 노드와 끝 노드가 같은 경로)을 포함합니다.
- 비순환 그래프(DAG): 사이클을 포함하지 않습니다.
- 엣지 레이블 그래프: 각 엣지에 레이블이 있으며 보통 해당 엣지 옆에 그려집니다.
- 가중치 그래프: 엣지에 산술 연산에 사용할 수 있는 가중치(값)가 연결되어 있습니다.
- 비연결 그래프: 두 개 이상의 노드가 어떤 엣지로도 연결되어 있지 않습니다.

그래프는 지도, 네트워크, 추천 시스템, 의존성 해상도 등 다양한 애플리케이션에서 사용됩니다.

## 그래프 순회

그래프의 모든 노드를 방문하는 작업입니다. 주요 알고리즘 두 가지는 다음과 같습니다:

- **너비 우선 탐색 (BFS)**
  - 큐를 사용합니다.
  - 레벨별로 탐색합니다.
  - 가중치 없는 그래프에서 최단 경로를 찾습니다.

- **깊이 우선 탐색 (DFS)**
  - 스택(또는 재귀)을 사용합니다.
  - 한 갈래를 완전히 탐색한 후 되돌아갑니다.
  - 사이클 탐지와 경로 찾기에 유용합니다.

## 그래프 표현

그래프는 두 가지 주요 방식으로 나타낼 수 있습니다:

- **인접 리스트**
  - 각 노드는 이웃 노드들의 목록을 가집니다.
  - 희소 그래프에 공간 효율적입니다.
  - 이웃 노드를 순회하기 쉽습니다.

- **인접 행렬**
  - 행과 열이 노드를 나타내는 2차원 배열입니다.
  - 큰 그래프에서는 공간을 많이 차지합니다.
  - 두 노드 사이에 간선이 있는지 빠르게 확인할 수 있습니다.

## 트리

트리는 사이클이 없고 연결된 특별한 유형의 그래프입니다. 주요 특징은 다음과 같습니다:

- 루프나 사이클이 없습니다(시작 노드와 끝 노드가 같은 경로).
- 반드시 연결되어 있어야 합니다(모든 노드가 서로 도달 가능).

#
## 일반적인 트리 유형

가장 흔한 트리 유형은 다음과 같습니다:

- 이진 트리
  - 각 노드는 최대 두 개의 자식, 왼쪽 자식과 오른쪽 자식을 가집니다.

- 이진 탐색 트리 (BST)
  - 모든 왼쪽 자식이 부모보다 작고, 모든 오른쪽 자식이 부모보다 큰 이진 트리입니다.

## 트라이

접두사 트리라고도 하며, 각 노드가 문자를 나타내는 문자열 집합을 저장하는 데 사용합니다.

공유하는 접두사는 한 번만 저장되어 자동 완성이나 맞춤법 검사 같은 작업에 효율적입니다.

검색과 삽입 연산의 시간 복잡도는 문자열 길이 L에 대해 O(L)입니다.

## 우선순위 큐

우선순위 큐는 각 요소가 우선순위를 가지는 추상 자료형입니다.

큐와 스택은 삽입 순서만 고려하지만, 우선순위 큐는 요소의 우선순위를 고려합니다.

일반 큐는 FIFO(선입선출)를 따르고 스택은 LIFO(후입선출)를 따릅니다. 그러나 우선순위 큐에서는 삽입 순서와 상관없이 우선순위가 높은 요소가 낮은 요소보다 먼저 처리됩니다.

## 힙

힙은 힙 속성이라는 매우 특정한 속성을 가진 특수한 트리 기반 자료 구조입니다.

힙 속성은 부모 노드와 자식 노드 간의 관계를 결정합니다. 힙에는 두 가지 유형이 있습니다:

- 최대 힙(Max-Heap)
  - 각 부모 노드의 값이 자식 노드들의 값보다 크거나 같습니다.
  - 가장 큰 요소가 루트에 있습니다.

- 최소 힙(Min-Heap)
  - 각 부모 노드의 값이 자식 노드들의 값보다 작거나 같습니다.
  - 가장 작은 요소가 루트에 있습니다.

## Python `heapq` 모듈 예제

```py
import heapq

# Create empty heap
my_heap = []

# Insert elements
heapq.heappush(my_heap, 9)
heapq.heappush(my_heap, 3)
heapq.heappush(my_heap, 5)
print(my_heap) # [3, 9, 5]

# Remove smallest element
print(heapq.heappop(my_heap))  # 3
print(my_heap) # [5, 9]

# Push + Pop in one step
print(heapq.heappushpop(my_heap, 7)) # 5
print(my_heap) # [7, 9]

# Transform list into heap
nums = [5, 7, 3, 1]
heapq.heapify(nums)
```

## 우선순위 사용하기

```py
my_heap = []
heapq.heappush(my_heap, (3, "A"))
heapq.heappush(my_heap, (2, "B"))
heapq.heappush(my_heap, (1, "C"))

# Removes lowest number = highest priority
print(heapq.heappop(my_heap))  # (1, "C")
```

## 동적 프로그래밍 소개

- **정의**: 동적 프로그래밍은 복잡한 문제를 더 간단한 하위 문제로 나누고, 중복 계산을 피하기 위해 결과를 저장하는 알고리즘 기법입니다.  
- **중복되는 하위 문제**: 더 큰 문제를 해결할 때 동일한 작은 문제가 여러 번 나타납니다. 이 하위 문제들을 반복해서 다시 계산하는 대신, 그 해답을 저장합니다.  
- **최적 부분 구조**: 문제의 최적 해답은 하위 문제들의 최적 해답을 포함합니다. 즉, 더 작은 부분들의 최적 해답을 결합하여 최상의 해답을 만들 수 있습니다.

## 동적 프로그래밍 해법

- **메모이제이션(상향식 접근법)**: 메모이제이션은 비용이 많이 드는 함수 호출 결과를 저장소에 저장하고, 동일한 입력이 다시 발생하면 저장된 결과를 반환합니다.

```py
def climb_stairs_memo(n, memo={}):
    """Dynamic programming with memoization"""
    # Check if we've already calculated this value
    if n in memo:
        return memo[n]  # Return cached result - O(1) lookup!
    
    # Base cases
    if n <= 2:
        return n
    
    # Calculate once and store in memo for future use
    memo[n] = climb_stairs_memo(n-1, memo) + climb_stairs_memo(n-2, memo)
    return memo[n]
```

- **테이블화(하향식 접근법)**: 테이블화는 하위 문제들의 해답으로 표를 채우면서 바닥부터 해결책을 만들어 갑니다.

```py
def climb_stairs_tabulation(n):
    """Dynamic programming with tabulation"""
    if n <= 2:
        return n
    
    # Create array to store results for all steps from 0 to n
    dp = [0] * (n + 1)
    dp[1] = 1  # 1 way to reach step 1
    dp[2] = 2  # 2 ways to reach step 2
    
    # Build up the solution iteratively
    for i in range(3, n + 1):
        # Ways to reach step i = ways to reach (i-1) + ways to reach (i-2)
        dp[i] = dp[i-1] + dp[i-2]
    
    return dp[n]
```
