# --description--

考虑由 $1 ≤ x$ 和 $0 ≤ y ≤ \frac{1}{x}$ 限制的区域。

令 $S_1$ 为能放入曲线下方的最大 `square`。

令 $S_2$ 为适合剩余区域的最大 `square`，依此类推。

令 $S_n$ 的索引为点对 (left, below)，表示 $S_n$ 左侧的平方数和 $S_n$ 下方的平方数的数量。

<img alt="diagram with squares under the hyperbola" src="https://cdn.freecodecamp.org/curriculum/project-euler/squares-under-a-hyperbola.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

该图显示了一些标有数字的此类 `square`。

$S_2$ 左边有一个平方，下面没有，所以 $S_2$ 的索引是 (1, 0)。

可以看出，$S_{32}$ 的索引是 (1,1)，$S_{50}$ 的索引也是 (1,1)。

50 是 $S_n$ 的指标为 (1, 1) 的最大 $n$。

对于哪个最大的 $n$，$S_n$ 的指标是 (3, 3)？
