# --explanation--

`Database`는 데이터가 저장되고 조직되는 시스템을 의미합니다.

`Activity`는 데이터베이스에서 일어나는 동작이나 과정을 의미합니다.

- `The database contains customer information and order details.` - 이는 데이터베이스가 중요한 비즈니스 데이터를 저장한다는 뜻입니다.

- `There was unusual activity in the database last night.` - 이는 데이터베이스에서 예상치 못한 일이 발생했다는 뜻입니다.

이 둘을 합쳐 `database activity`는 데이터베이스 시스템 내에서 일어나는 모든 동작, 변경 또는 과정을 의미합니다.
