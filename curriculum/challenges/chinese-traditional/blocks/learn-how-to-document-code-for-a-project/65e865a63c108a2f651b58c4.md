# --description--

在這個任務中，你將學習 `specific` 和 `specify` 之間的差異。`Specific` 是形容詞，用來描述明確定義或詳細的事物，而 `specify` 是動詞，表示詳細說明某事。

例如，`specific` 指令可能是 `use port 8080`，而 `specify` 則可以表示說明該細節：`specify the port number.`

`Setup` 指的是某物的配置或準備，通常用於準備軟體或設備的背景關係，例如 `setup instructions for the software.`
