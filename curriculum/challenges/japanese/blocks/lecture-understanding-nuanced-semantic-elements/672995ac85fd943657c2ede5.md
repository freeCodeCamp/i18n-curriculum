# --interactive--

「bring attention to」要素、`b` は、要約のキーワードやレビューの製品名を強調するためによく使われます。通常、ブラウザーはこのテキストを太字で表示します。以下は、レビュー内の製品名を`b`要素で強調した例です。

:::interactive_editor

```html
<p>
  We tested several products, including the <b>SuperSound 3000</b> for audio
  quality, the <b>QuickCharge Pro</b> for fast charging, and the
  <b>EcoClean Vacuum</b> for cleaning. The first two performed well, but the
  <b>EcoClean Vacuum</b> did not meet expectations.
</p>
```

:::

ブラウザーはこれらのテキスト部分を太字としてレンダリングします。この視覚的な強調は読者の注意を製品名に引きつけます。

テキストの重要性を強調する必要がある場合は、`b`要素ではなく`strong`要素を使うべきです。

`strong`は意味論的なHTML要素で、重要または緊急なテキストを強調します。以下は、顧客が製品に対して起こす可能性のあるアレルギー反応に関する非常に重要な警告を`strong`要素でラベル付けした例です。

:::interactive_editor

```html
<p>
  <strong>Warning:</strong> This product may cause allergic reactions.
</p>
```

:::

`strong`要素はその緊急性を伝えます。

見た目はどちらもデフォルトで太字としてレンダリングされるため非常に似ていますが、意味はかなり異なります。「bring attention to」要素はテキストに注意を引くだけで、より高い重要度を示しませんが、`strong`要素はそれ以上のことをします。重要性や緊急性を伝えます。これが主な違いです。

どちらを使うかは、テキストの目的と周囲のコンテンツ内での重要性を考慮してください。
