# --description--

`plugin` 是一个为现有计算机程序添加特定特色的组件。当一个程序支持 `plugins` 时，它启用自定义功能。 在集成开发环境的上下文中，`plugins` 可以提供额外的功能性，比如用于编码、调试或增强用户界面的新工具。

例如，集成开发环境中的代码格式化器通常是一个 `plugin`。
