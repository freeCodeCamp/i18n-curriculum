# --description--

Pythonの特殊メソッドは、「マジックメソッド」や「ダンダーメソッド」とも呼ばれ、両端にダブルアンダースコアが付く特別なPythonのメソッドです（`__`）。「ダンダー」という言葉自体は、ダブル（**d**はdoubleの意味）とアンダースコア（**under**）から来ています。

おそらく、あなたはすでに知らずに特殊メソッドを使ったことがあるでしょう。`3 + 4`のようなコードを書くたびに、Pythonは裏で静かに`3.__add__(4)`を実行しています。これが特殊メソッドの働きです。ですから、特殊メソッドを直接呼び出すこともできますが、ほとんどの場合はしません。`3 + 4`のように書くほうが、`3.__add__(4)`を自分で呼び出すよりもずっとわかりやすく読みやすいです。

`__add__`のほかに、`__init__()`もよく使う特殊メソッドで、クラスの初期化子です。ほかにも`__len__()`や`__str__()`のようなものがあります。

特殊メソッドは、プログラミングする人とPython言語インタプリタの間のやり取りを取り仕切る監督のようなものだと考えてください。

特殊メソッドを直接呼び出す必要はありません。Pythonは特定の操作が起きたときに自動的に呼び出します。これらの操作には以下が含まれます：

- **加算、減算、乗算、除算などの算術演算**。加算には`__add__()`、減算には`__sub__()`、乗算には`__mul__()`、除算には`__truediv__()`が呼び出されます。
    
- **連結、繰り返し、フォーマット、文字列への変換などの文字列操作**。連結には`__add__()`、繰り返しには`__mul__()`、フォーマットには`__format__()`、文字列変換には`__str__()`や`__repr__()`などが呼び出されます。
    
- **等価比較、小なり、大なりなどの比較操作**。等価比較には`__eq__()`、小なりには`__lt__()`、大なりには`__gt__()`などが呼び出されます。
    
- **イテラブルにしてアイテムを順に処理するイテレーション操作**。イテレータを返すには`__iter__()`、次のアイテムを取得するには`__next__()`が呼び出されます。
    

通常、文字列や数値のようなPythonの組み込みデータ型は、加算や連結、等価比較、ループ処理などの方法をすでに知っています。

しかし、自分でクラスを作成すると、Pythonは自動的にそれらを扱う方法を知りません。

そこで特殊メソッドの出番です。Pythonの組み込みの動作をカスタマイズできるようにします。

たとえば、以下のクラスで作成したbookオブジェクトのページ数を取得したり、比較したり、オブジェクトの読みやすい文字列を得たいとします。特殊メソッドがない場合は次のようになります：

```python
class Book:
   def __init__(self, title, pages):
       self.title = title
       self.pages = pages

book1 = Book("Built Wealth Like a Boss", 420)
book2 = Book("Be Your Own Start", 420)

print(len(book1)) # TypeError: object of type 'Book' has no len()
print(str(book1)) # <__main__.Book object at 0x102ed2900>
print(book1 == book2) # False even though they have the same number of pages
```

この例では：

- `len(book1)`は失敗しました。なぜなら、Pythonは`__len__()`なしでbookオブジェクトの長さを取得する方法を知らないからです。
    
- `str(book1)`は`<__main__.Book object at 0x102ed2900>`のようなものを出力しました。これは`__str__()`を使わないときのデフォルトの表現です。
    
- `book1 == book2`は`False`になりました。Pythonは内容ではなく、メモリ上で同じオブジェクトかどうかをチェックするだけだからです。
    

次のように、`__len__()`クラスから作成したオブジェクトを扱いやすくするために、`__str__()`、`__eq__()`、`Book`という特殊メソッドを自分で定義できます：

```python
class Book:
   def __init__(self, title, pages):
       self.title = title
       self.pages = pages

   def __len__(self):
       return self.pages

   def __str__(self):
       return f"'{self.title}' has {self.pages} pages"

   def __eq__(self, other):
       return self.pages == other.pages
  
book1 = Book("Built Wealth Like a Boss", 420)
book2 = Book("Be Your Own Start", 420)

print(len(book1)) # 420
print(len(book2)) # 420
print(str(book1)) # 'Built Wealth Like a Boss' has 420 pages
print(str(book2)) # 'Be Your Own Start' has 420 pages
print(book1 == book2) # True
```

もう一つの例はショッピングカートで、以下の操作をします：

- カートにアイテムを追加する
    
- カートからアイテムを削除する
    
- カート内のアイテム数を取得する
    
- カート内のアイテムを確認する
    
- 特定のアイテムがカートにあるかチェックする
    
- カート内の特定のインデックスのアイテムを返すまたは表示する
    

カートにアイテムを追加したり削除したりするメソッドはあるかもしれませんが、ほかの機能には特殊メソッドを作成できます：

- カート内のアイテム数を取得するには`__len__()`
    
- カート内のアイテムをループ処理して見るには`__iter__()`
    
- 特定のアイテムがカートにあるかチェックするには`__contains__()`
    
- 特定のインデックスのアイテムを返すまたは表示するには`__getitem__()`
    

これらのユーザー定義メソッドと特殊メソッドを持つ`Cart`クラスの例がこちらです：

```python
class Cart:
   def __init__(self):
       self.items = []

   def add(self, item):
       self.items.append(item)

   def remove(self, item):
       if item in self.items:
           self.items.remove(item)
       else:
           print(f'{item} is not in cart')

   def list_items(self):
       return self.items

   def __len__(self):
       return len(self.items)

   def __getitem__(self, index):
       return self.items[index]

   def __contains__(self, item):
       return item in self.items

   def __iter__(self):
       return iter(self.items)
```

そして、こう使います：

```python
cart = Cart()
cart.add('Laptop')
cart.add('Wireless mouse')
cart.add('Ergo keyboard')
cart.add('Monitor')

for item in cart:
   print(item, end=' ') # Laptop Wireless mouse Ergo keyboard Monitor

print(len(cart)) # 4
print(cart[3]) # Monitor

print('Monitor' in cart) # True
print('banana' in cart) # False

cart.remove('Ergo keyboard')

print(cart.list_items()) # ['Laptop', 'Wireless mouse', 'Monitor']

cart.remove('banana') # banana is not in cart
```

これらはPythonで特殊メソッドを実際に使ういくつかの例です。
