# --description--

Para un número positivo $n$, define $S(n)$ como la suma de los enteros $x$, para los cuales $1 < x < n$ y $x^3 \equiv 1\bmod n$.

Cuando $n = 91$, hay 8 valores posibles para $x$, a saber: 9, 16, 22, 29, 53, 74, 79, 81. Por lo tanto, $S(91) = 9 + 16 + 22 + 29 + 53 + 74 + 79 + 81 = 363$.

Encuentra $S(13\\,082\\,761\\,331\\,670\\,030)$.
