# --description--

Agora é hora de aprender os números na casa dos 50.

Em chinês, o número 50 é chamado de `五十 (wǔ shí)`.

Os números de 51 a 59 são formados adicionando o dígito das unidades (1–9) após `五十 (wǔ shí)` (50).

| Número | Chinês |
| :--- | :--- |
| 50 | `五十 (wǔ shí)` |
| 51 | `五十一 (wǔ shí yī)` |
| 52 | `五十二 (wǔ shí èr)` |
| 53 | `五十三 (wǔ shí sān)` |
| 54 | `五十四 (wǔ shí sì)` |
| 55 | `五十五 (wǔ shí wǔ)` |
| 56 | `五十六 (wǔ shí liù)` |
| 57 | `五十七 (wǔ shí qī)` |
| 58 | `五十八 (wǔ shí bā)` |
| 59 | `五十九 (wǔ shí jiǔ)` |
