# --description--

Il verbo modale `can` si usa per indicare possibilità o capacità. Per esempio, nella frase `You can see the stars at night`, `can` mostra che è possibile vedere le stelle durante la notte.

`Change` significa rendere o diventare diverso. Nella vita quotidiana, potresti `change` i tuoi vestiti dopo il lavoro, indicando che cambi dall’abbigliamento da lavoro a quello casual.

`Stay the same` significa rimanere invariato o costante. Per esempio, `The rules of the game stay the same every year.` Questo significa che le regole non cambiano da un anno all’altro.
