# --explanation--

Критична `critical bug` — це серйозна помилка в програмному забезпеченні, яка впливає на функціональність і має бути виправлена негайно. Наприклад:

- `We need to fix the critical bugs before launching the app.` — Найсерйозніші проблеми потрібно вирішити перед випуском.

- `A critical bug caused the website to crash.` — Серйозна помилка спричинила зупинку роботи сайту.
