# --description--

Компоненти — це основа React. Усе в React — це компонент, і тут ви навчитеся створювати один.

Існує два способи створення компонента React. Перший спосіб — це використання JavaScript-функції. Визначення компонента таким чином створює *компонент без стану на основі функції*. Концепцію стану у застосунку буде розглянуто в наступних завданнях. Наразі уявіть компонент без стану як такий, що може отримувати дані та відтворювати їх, але не керує і не відстежує зміни цих даних. (Другий спосіб створення компонента React ми розглянемо в наступному завданні.)

Щоб створити компонент за допомогою функції, достатньо написати JavaScript-функцію, яка повертає JSX або `null`. Важливо пам’ятати, що React вимагає, щоб ім’я вашої функції починалося з великої літери. Ось приклад компонента без стану на основі функції, який призначає HTML-клас у JSX:

```jsx
const DemoComponent = function() {
  return (
    <div className='customClass' />
  );
};
```

Після трансляції `<div>` матиме CSS-клас `customClass`.

Оскільки JSX-компонент представляє HTML, ви можете об’єднати кілька компонентів, щоб створити більш складну HTML-сторінку. Це одна з ключових переваг компонентної архітектури React. Вона дозволяє складати інтерфейс користувача з багатьох окремих, ізольованих компонентів. Це полегшує створення та підтримку складних інтерфейсів.

# --instructions--

У редакторі коду є функція з назвою `MyComponent`. Завершіть цю функцію так, щоб вона повертала один елемент `div`, який містить деякий рядок тексту.

**Примітка:** Текст вважається дочірнім елементом `div`, тому ви не зможете використати самозакривний тег.

# --hints--

`MyComponent` має повертати JSX.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.length === 1;
  })()
);
```

`MyComponent` має повертати елемент `div`.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.children().type() === 'div';
  })()
);
```

Елемент `div` має містити рядок тексту.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.find('div').text() !== '';
  })()
);
```
