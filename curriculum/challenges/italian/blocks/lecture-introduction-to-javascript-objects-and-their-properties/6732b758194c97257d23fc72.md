# --interactive--

In JavaScript, un costruttore è un tipo speciale di funzione usata per creare e inizializzare oggetti. Viene invocato con la parola chiave `new` e può inizializzare proprietà e metodi sul nuovo oggetto creato.

In questa lezione, vedremo come lavorare con il costruttore `Object()`. Il costruttore `Object()` crea un nuovo oggetto vuoto. Ecco un esempio:

```js
new Object()
```

Quando chiami `new Object()`, restituisce un nuovo oggetto che può essere usato per archiviare valori.

Il costruttore `Object()` può essere usato con o senza la parola chiave `new`. Quando viene chiamato come funzione senza `new`, si comporta in modo diverso a seconda del tipo di valore passato. Ecco un esempio di utilizzo del costruttore `Object()` senza la parola chiave `new`:

:::interactive_editor

```js
const num = 42;
const numObj = Object(num); // Creates an object wrapper for the number

console.log(numObj);
console.log(typeof numObj); // "object"
```

:::

Come puoi vedere nel secondo `console.log`, `numObj` è un oggetto. Questo succede perché abbiamo usato il costruttore `Object()` per trasformare quell’input di tipo numero in un oggetto.

Cosa succede se proviamo a passare `null` o `undefined` al costruttore `Object()`?

:::interactive_editor

```js
const newObj = new Object(undefined);
console.log(newObj); // {}
```

:::

Beh, il risultato sarà un oggetto vuoto. Un altro caso d’uso del costruttore `Object()` è quando lavori con un valore di tipo sconosciuto e devi assicurarti che sia un oggetto. Diamo un’occhiata al seguente esempio:

:::interactive_editor

```js
function toObject(value) {
  if (value === null || value === undefined) {
    return {};
  }

  if (typeof value === "object") {
    return value;
  }

  return Object(value);
}

console.log(toObject(null));

console.log(toObject(true));

console.log(toObject([1, 2, 3]));
```

:::

In questo esempio, abbiamo una funzione chiamata `toObject`. La seconda condizione controllerà se il valore è di tipo oggetto e restituirà il valore se la condizione è `true`. Questa condizione controllerà sia gli oggetti sia gli array, dato che gli array sono tipi speciali di oggetti.

Se nessuna delle condizioni è vera, la funzione restituisce `Object(value)`, che converte l’input in un oggetto. Questo funziona per valori come numeri, stringhe e booleani.

La maggior parte delle volte non userai il costruttore `Object()` per creare nuovi oggetti perché userai invece la sintassi letterale degli oggetti (ad esempio, `const objectLiteral = { name: "Beau" }`). Ma è comunque utile capire le basi del lavoro con il costruttore Object.
