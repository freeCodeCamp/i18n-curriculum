# --description--

عبارة `to keep in mind` تعني تذكر أو أخذ شيء بعين الاعتبار. تُستخدم غالبًا عندما يقدم شخص نصيحة أو معلومات مهمة يجب تذكرها للاستخدام لاحقًا.

على سبيل المثال، إذا قال شخص ما، "تذكر التحقق من الطقس قبل التنزه." قد ترد بعبارة `I'll keep that in mind` بمعنى أنك ستتذكر أخذ هذه النصيحة بعين الاعتبار.
