# --interactive--

对话框让你向用户显示重要的信息或操作。借助超文本标记语言的内置 `dialog` 元素，你可以轻松地在你的网络应用中创建这些对话框（包括模态和非模态对话框）。

模态对话框是一种对话框类型，它迫使用户在访问应用或网页的其余部分之前与其交互。它有效地阻止与其他内容的交互，直到用户完成某个操作，例如关闭对话框或提交表单。

相比之下，非模态对话框允许用户在对话框打开时继续与页面或应用的其他部分交互。它不会阻止访问其余内容。

在本课中，你将学习如何使用 `showModal()` 和 `close()` 方法打开和关闭模态框。

当你想确保用户专注于模态的特定操作或消息时，可以使用 `showModal()` 方法打开模态对话框。这将为页面上的其他项添加一个背景并禁用它们。这对于显示形式、确认和需要用户操作的关键信息的模态非常理想。 

这是模态对话框的超文本标记语言：

:::interactive_editor

```html
<dialog id="my-modal">
  <p>This is a modal dialog.</p>
</dialog>
```

:::

目前，你看不到页面上的任何内容，因为模态框在初始 `Render` 时是关闭的。你可以通过使用 `showModal()` 方法自动打开模态框：

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
dialog.showModal();
```

:::

浏览器中的结果将显示一个带有文本 `This is a modal dialog.` 的模态窗口。

最好将控件交给用户。为此，你可以为按钮添加点击事件监听器并使用 `showModal()` 方法：

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.showModal();
});
```

:::

如果你需要显示一个对话框，同时仍允许与对话框外的内容交互，那么你可以使用 `show()` 方法：

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});
```

:::

要关闭模态窗口，你可以在 `dialog` 元素内为模态窗口添加一个按钮，并使用 `close()` 方法：

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
  <button id="close-modal-btn">Close Modal</button>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");
const closeButton = document.getElementById("close-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});

closeButton.addEventListener("click", () => {
  dialog.close();
});
```

:::

模态对话框和非模态对话框有什么区别？

## --answers--
