# --explanation--

`Could` und `might` sind Modalverben, die verwendet werden, um über Möglichkeiten zu sprechen.

`Could` bedeutet, dass etwas möglich, aber nicht sicher ist. Zum Beispiel:

`This update could make the app run faster.` – Das Update hat das Potenzial, die Geschwindigkeit der App zu verbessern.

`Might` drückt ebenfalls Möglichkeit aus, deutet aber oft auf eine geringere Wahrscheinlichkeit als `could` hin. Zum Beispiel:

`The new feature might improve the user experience.` – Es ist möglich, dass die Funktion das Erlebnis verbessert.
