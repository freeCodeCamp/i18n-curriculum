# --interactive--

CSS는 Cascading Style Sheets의 약자로, 현대 웹 개발에서 매우 중요한 컴포넌트입니다. HTML에 스타일을 적용하는 데 사용하는 마크업 언어입니다. 쉽게 말해, HTML이 웹 페이지의 구조라면 CSS는 그 구조를 보기 좋게 만드는 역할을 합니다.

여기 내비게이션 바를 스타일링한 예시가 있습니다.

**참고**: 아직 CSS 코드를 이해하려고 애쓰지 마세요. 앞으로 수업과 워크숍에서 이 모든 것이 어떻게 작동하는지 배우게 됩니다. 이 예시는 CSS로 할 수 있는 일을 간단히 보여주기 위한 것입니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<nav class="navbar">
    <ul class="nav-links">
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Services</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>
```

```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: 'Segoe UI', sans-serif;
  background-color: #f4f4f4;
}

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  background-color: #333;
  padding: 1rem 2rem;
  color: white;
}

.navbar .logo {
  font-size: 1.5rem;
  font-weight: bold;
}

.nav-links {
  list-style: none;
  display: flex;
  gap: 1.5rem;
}

.nav-links a {
  color: white;
  text-decoration: none;
  font-weight: 500;
  transition: color 0.3s ease;
}

.nav-links a:hover {
  color: #ff9800;
}
```

:::

CSS의 주요 역할은 웹 페이지의 콘텐츠와 시각적 표현을 분리하는 것입니다. 이 분리는 웹 개발자가 더 유연하고 유지 관리하기 쉬운 웹사이트를 만들 수 있게 합니다.

CSS를 사용하면 HTML 구조를 변경하지 않고도 웹 페이지의 레이아웃, 색상, 폰트, 전반적인 시각적 모습을 제어할 수 있습니다.

간단한 비유를 들어 보겠습니다. 웹사이트를 집이라고 생각하면, HTML은 집의 기초와 뼈대이고 CSS는 집을 시각적으로 매력적이고 독특하게 만드는 페인트, 벽지, 장식입니다.

CSS는 HTML 요소를 선택하고 스타일을 적용하는 방식으로 작동합니다.

여기 스타일이 적용되지 않은 `button` 요소와 스타일이 적용된 요소를 보여주는 예시가 있습니다. 이 예시와 상호작용하려면 인터랙티브 에디터를 활성화해야 합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<button class="regular-btn">Unstyled Button</button>
<button class="round-btn">Round Button</button>
```

```css

.round-btn {
  font-size: 1rem;
  font-family: 'Segoe UI', sans-serif;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease, transform 0.2s ease;
  background-color: #2ecc71;
  color: white;
  border-radius: 50px;
  padding: 0.6rem 1.6rem;
}

.round-btn:hover {
  background-color: #27ae60;
  transform: translateY(-2px);
}
```

:::

이 스타일에는 색상, 폰트 크기 등 다양한 속성이 포함될 수 있습니다. 이러한 속성을 변경하면 콘텐츠를 바꾸지 않고도 웹 페이지의 모양을 크게 바꿀 수 있습니다.

CSS의 가장 강력한 특징 중 하나는 반응형 디자인을 만들 수 있다는 점입니다.

즉, CSS를 사용하면 데스크톱 컴퓨터, 태블릿, 스마트폰 등 어떤 기기에서든 웹사이트가 멋지게 보이도록 만들 수 있습니다.

CSS는 웹사이트를 보는 기기의 화면 크기에 따라 레이아웃, 폰트 크기, 기타 시각적 요소를 조정할 수 있게 합니다.

인터랙티브 에디터를 활성화하고 미리보기 창 크기를 조절해 화면 크기에 따라 레이아웃이 어떻게 적응하는지 확인해 보세요.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<header class="header">
    <h1>Welcome to My Responsive Site</h1>
    <p>This layout adapts to your screen size!</p>
</header>

<main class="content">
    <section class="box">Box 1</section>
    <section class="box">Box 2</section>
    <section class="box">Box 3</section>
</main>
```

```css
body {
  margin: 0;
  font-family: 'Segoe UI', sans-serif;
  background-color: #f9f9f9;
  color: #333;
}

.header {
  background-color: #4CAF50;
  color: white;
  padding: 2rem;
  text-align: center;
}

.content {
  display: flex;
  gap: 1rem;
  padding: 1rem;
  justify-content: center;
}

.box {
  flex: 1;
  min-width: 200px;
  background-color: #ddd;
  padding: 2rem;
  text-align: center;
  font-size: 1.2rem;
  border-radius: 8px;
  transition: background-color 0.3s;
}

.box:hover {
  background-color: #ccc;
}

/* Responsive: for tablets and below (≤ 768px) */
@media (max-width: 768px) {
  .content {
    flex-direction: column;
    align-items: center;
  }

  .box {
    width: 90%;
    font-size: 1.1rem;
  }
}

/* Responsive: for phones (≤ 480px) */
@media (max-width: 480px) {
  .header {
    padding: 1.5rem 1rem;
  }

  .header h1 {
    font-size: 1.5rem;
  }

  .header p {
    font-size: 1rem;
  }

  .box {
    font-size: 1rem;
    padding: 1.5rem;
  }
}
```

:::

CSS의 또 다른 중요한 기능은 이름에 있는 "cascading" 즉, 계단식 특성입니다.

이는 스타일이 상속되고 재정의될 수 있어 계층적인 스타일 구조를 만들 수 있다는 뜻입니다. 이 작동 방식은 앞으로 수업에서 더 자세히 배우게 됩니다.

CSS는 외부 스타일시트 사용도 지원합니다.

즉, 모든 스타일 규칙을 별도의 파일에 보관하고 이를 여러 HTML 페이지에 연결할 수 있습니다.

이 기능은 특히 큰 웹사이트에서 유지 관리를 크게 향상시킵니다. 각 페이지마다 스타일을 바꾸는 대신 하나의 CSS 파일만 수정하면 연결된 모든 페이지에 변경 사항이 적용됩니다.

웹 개발 세계에서 CSS는 시각적으로 매력적이고 반응형이며 사용자 친화적인 웹사이트를 만드는 데 필수적인 역할을 합니다.

개발자가 단순한 HTML 문서를 사용자 관심을 끌고 웹 콘텐츠와의 상호작용을 향상시키는 매력적인 웹 경험으로 바꾸도록 돕습니다.
