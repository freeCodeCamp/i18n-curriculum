# --explanation--

У **косвенній мові** фраза `said that` використовується, щоб передати те, що хтось сказав, іншими словами, не дослівно. Слово `that` є необов’язковим і часто опускається в повсякденній мові. Наприклад:

- Пряма мова: `We need to update the software.`

- Косвена мова: `He said that we need to update the software.` або `He said we need to update the software.`

Обидва варіанти правильні і мають однакове значення.
