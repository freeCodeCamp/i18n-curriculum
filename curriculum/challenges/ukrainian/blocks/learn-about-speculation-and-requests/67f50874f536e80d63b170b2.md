# --explanation--

`Brute force attacks` — це коли хакери пробують багато паролів, доки не знайдуть правильний. Наприклад:

`The system detected a brute force attack on the login page.` — атака намагалася зламати, перебираючи багато паролів.

`SQL injection attempts` трапляються, коли хакери намагаються вставити шкідливий SQL-код, щоб отримати доступ або змінити базу даних. Наприклад:

`The firewall stopped two SQL injection attempts.` — хтось намагався зламати базу даних, використовуючи шкідливий код.
