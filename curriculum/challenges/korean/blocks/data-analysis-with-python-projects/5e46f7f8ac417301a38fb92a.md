이 프로젝트에서는 `matplotlib`, `seaborn`, `pandas`를 사용해 의료 검사 데이터에서 시각화하고 계산할 것입니다. 데이터셋 값은 의료 검사 중에 수집되었습니다.

## 데이터 설명

데이터셋의 행은 환자를 나타내고 열은 신체 측정치, 다양한 혈액 검사 결과, 생활 습관 정보 등을 나타냅니다. 이 데이터셋을 사용해 심장 질환, 신체 측정치, 혈액 지표, 생활 습관 간의 관계를 탐구할 것입니다.

파일 이름: medical_examination.csv

| 기능 | 변수 유형 | 변수 | 값 유형 |
|:-------:|:------------:|:-------------:|:----------:|
| 나이 | 객관적 기능 | `age` | 정수 (일) |
| 키 | 객관적 기능 | `height` | 정수 (cm) |
| 몸무게 | 객관적 기능 | `weight` | 실수 (kg) |
| 성별 | 객관적 기능 | `gender` | 범주형 코드 |
| 수축기 혈압 | 검사 기능 | `ap_hi` | 정수 |
| 이완기 혈압 | 검사 기능 | `ap_lo` | 정수 |
| 콜레스테롤 | 검사 기능 | `cholesterol` | 1: 정상, 2: 정상 이상, 3: 매우 높음 |

| 심혈관 질환 유무 | 목표 변수 | `cardio` | 이진 |


## 지침

`examples/Figure_1.png`와 유사한 차트를 만드세요. 여기서는 `cholesterol`와 `gluc` 환자에 대해 `alco`, `active`, `smoke`, `cardio=1`, `cardio=0` 변수의 좋은 결과와 나쁜 결과 수를 서로 다른 패널에 표시합니다.

`medical_data_visualizer.py` 파일의 각 번호에 대해 아래 해당 지침 번호의 코드를 추가하세요.

1. `medical_examination.csv`에서 데이터를 가져와 `df` 변수에 할당하세요.
2. 데이터에 `overweight` 열을 추가하세요. 과체중 여부를 판단하려면 먼저 몸무게(kg)를 키(m)의 제곱으로 나누어 BMI를 계산하세요. 그 값이 25보다 크면 과체중입니다. 과체중이 아니면 `0` 값을, 과체중이면 `1` 값을 사용하세요.
3. `0`는 항상 좋은 상태, `1`은 항상 나쁜 상태가 되도록 데이터를 표준화하세요. `cholesterol` 또는 `gluc` 값이 1이면 `0` 값으로 설정하세요. 값이 `1`보다 크면 `1` 값으로 설정하세요.
4. `draw_cat_plot` 함수에서 범주형 차트를 그리세요.
5. `pd.melt`를 사용해 `cholesterol`, `gluc`, `smoke`, `alco`, `active`, `overweight` 값을 포함하는 데이터프레임을 `df_cat` 변수에 만드세요.
6. `df_cat`에서 데이터를 그룹화하고 재구성해 `cardio`별로 나누세요. 각 기능의 개수를 표시하세요. `catplot`가 제대로 작동하려면 열 이름 하나를 바꿔야 합니다.
7. 데이터를 `long` 형식으로 변환하고 seaborn 라이브러리에서 제공하는 `sns.catplot()` 메서드를 사용해 범주형 기능의 값 개수를 보여주는 차트를 만드세요.
8. 출력용 그림을 가져와 `fig` 변수에 저장하세요.
9. 다음 두 줄은 수정하지 마세요.
10. `draw_heat_map` 함수에서 히트맵을 그리세요.
11. `df_heat` 변수에서 다음과 같은 잘못된 데이터를 나타내는 환자 구간을 필터링해 데이터를 정리하세요:
    - 이완기 혈압이 수축기 혈압보다 높음 (`(df['ap_lo'] <= df['ap_hi'])`로 올바른 데이터 유지)
    - 키가 2.5 백분위수 미만 (`(df['height'] >= df['height'].quantile(0.025))`로 올바른 데이터 유지)
    - 키가 97.5 백분위수 초과
    - 몸무게가 2.5 백분위수 미만
    - 몸무게가 97.5 백분위수 초과
12. 상관관계 행렬을 계산해 `corr` 변수에 저장하세요.
13. 상삼각형 마스크를 생성해 `mask` 변수에 저장하세요.
14. `matplotlib` 그림을 설정하세요.
15. `seaborn` 라이브러리에서 제공하는 `sns.heatmap()` 메서드를 사용해 상관관계 행렬을 그리세요.
16. 다음 두 줄은 수정하지 마세요.
