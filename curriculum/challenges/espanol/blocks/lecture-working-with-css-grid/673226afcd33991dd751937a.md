# --interactive--

Las propiedades `grid-column` y `grid-row` te permiten especificar la colocación horizontal y vertical de los elementos de la cuadrícula dentro de un diseño de cuadrícula.

En otras palabras, ambos te permiten controlar dónde comienza y termina un elemento de cuadrícula haciendo referencia a las líneas de cuadrícula. Estas líneas de cuadrícula son los límites que separan las filas y columnas que ya has definido usando las propiedades `grid-template-rows` y `grid-template-columns`.

Echemos un vistazo a las sintaxis de ambas propiedades `grid-row` y `grid-column`, y también a un ejemplo práctico.

Esta es la sintaxis de la propiedad `grid-row`:

```css
grid-row: <start-line> / <end-line>;
```

Y de `grid-column`:

```css
grid-column: <start-line> / <end-line>;
```

`<start-line>` es la línea de la cuadrícula donde comienza el ítem y `<end-line>` es la línea de la cuadrícula donde termina el ítem. Ambas están indexadas desde 1, es decir, comienzas a contarlas desde 1, no desde 0.

Recuerda, las líneas de cuadrícula para filas se generan en base al número de filas especificadas en la propiedad `grid-template-rows`. Lo mismo se aplica a las columnas con la propiedad `grid-template-columns`.

Para el ejemplo práctico, comencemos con las líneas generadas por el navegador cuando usas ambas propiedades `grid-template-rows` y `grid-template-columns`.

Aquí tienes una estructura de cuadrícula con 4 columnas y 3 filas:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}
```

:::

Inspeccionar el contenedor de la cuadrícula (la clase `grid`) muestra que cada fila y columna está delimitada por dos líneas: una línea de inicio al comienzo de la fila o columna, y una línea de fin al final de la fila o columna.

Puedes apuntar a estas líneas para usar las propiedades `grid-row` y `grid-column` para determinar dónde debes colocar un elemento.

Así es como puedes hacer que el primer elemento de la cuadrícula ocupe las dos primeras columnas:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
}
```

:::

Con esto, estás diciendo que el primer elemento de la cuadrícula debe comenzar en la columna 1 y terminar justo antes de la columna 3.

El primer elemento ahora ocupará dos columnas, y el cuarto elemento será trasladado a la segunda fila. ¡Genial!

Si también quieres que el primer elemento ocupe dos filas, puedes especificar un `grid-row` de ` 1 / 3`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
  grid-row: 1 / 3;
}
```

:::

También puedes usar la palabra clave `span` para decirle al elemento de la cuadrícula qué filas y columnas debe abarcar. Por ejemplo, `1 / 3` es lo mismo que `1 / span 2`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / span 2;
  grid-row: 1 / span 2;
}
```

:::

Puedes continuar aplicando esta técnica a cualquier elemento en la cuadrícula y colocarlos donde quieras. 

Continuando con el proceso, así es como hicimos un diseño tipo masonería con los elementos.

```css
}

}

  grid-column: 3 / span 2;
}
```
