# --description--

Нехай $S_n$ — послідовність цілих чисел, утворена за допомогою такого псевдовипадкового генератора чисел:

$$\begin{align}
        S_0 & = 290\\,797 \\\\
  S_{n + 1} & = {S_n}^2\bmod 50\\,515\\,093
\end{align}$$

Нехай $A(i, j)$ — мінімум чисел $S_i, S_{i + 1}, \ldots, S_j$ для $i ≤ j$. Нехай $M(N) = \sum A(i, j)$ для $1 ≤ i ≤ j ≤ N$.

Ми можемо перевірити, що $M(10) = 432\\,256\\,955$ і $M(10\\,000) = 3\\,264\\,567\\,774\\,119$.

Знайдіть $M(2\\,000\\,000\\,000)$.
