# --description--

La frase `have to` se usa para hablar de cosas que son necesarias o requeridas. Es similar a `must`, pero `must` es más fuerte y se usa para reglas más importantes o sentimientos personales.

Después de `have to`, se usa un verbo base para explicar qué acción es necesaria. Por ejemplo, `You have to wear a helmet` significa que usar un casco es necesario para la seguridad.

`Though` es una palabra que se usa para mostrar un pequeño contraste o sorpresa en una oración. Es similar a `but`. Por ejemplo, `It's cold, though it's sunny` significa que es sorprendente que el clima sea frío porque también está soleado.
