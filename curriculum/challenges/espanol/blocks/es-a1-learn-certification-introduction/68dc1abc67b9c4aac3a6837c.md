# --description--

Si estás usando tu dispositivo móvil para revisar el currículo de español, necesitarás asegurarte de que tu dispositivo esté listo para escribir marcas de acento y caracteres especiales, como `á`, `é` o `ñ`.

## El truco rápido de presionar y mantener

La forma más sencilla de escribir letras acentuadas es presionando y manteniendo pulsada la letra que necesitas.

1. Cuando se muestra el teclado, presiona y mantén presionada la tecla de letra, por ejemplo `a` o `n`.

2. Aparecerá un menú emergente con versiones acentuadas como `á` o `ñ`.

3. Desliza tu dedo hasta el carácter que quieres y levántalo para seleccionarlo.

Esto es perfecto cuando solo necesitas acentos ocasionalmente. 

## Agregar teclados de idioma

Otra alternativa sería agregar una distribución de teclado en español, lo cual puede ser útil para tener acceso más rápido a los acentos y otros caracteres.

### En Android

**NOTA:** La ruta de Android no es universal y puede variar según el dispositivo.

1. Abre Settings.

2. Ve a Sistema (o Administración General, dependiendo del dispositivo).

3. Configuración del teclado táctil (o Idiomas y entrada).

4. Toca Languages and types (o Add keyboard).

5. Selecciona Spanish (US), o toca Manage input languages y elige Spanish (US).

6. Guarda o toca Done, si se te solicita.

### En iPhone

1. Abre Configuración.

2. Ve a General > Keyboard.

3. Teclados táctiles.

4. Toca Añadir nuevo teclado.

5. Selecciona Español (Latinoamérica).

6. Usa el ícono del globo en el teclado para cambiar de idioma.

## Teclados de terceros

También puedes instalar diferentes aplicaciones para tu teléfono o tableta para usar el teclado en español con facilidad. Estas aplicaciones a menudo ofrecen funciones útiles como sugerencias inteligentes, soporte para varios idiomas y formas más rápidas de escribir las tildes.

Ahora estás listo para seguir el currículo de español y completar las tareas, especialmente aquellas que requieren tildes.
