# --description--

`present simple` 時態用來描述定期的動作或常式。

請記住，為了構成 `present simple`，你對 `I`、`you`、`we`、`they` 使用動詞原形（`I code`、`they test`），而對 `he`、`she`、`it` 則加上 `-s` 或 `-es`（`he codes`、`she tests`）。

在工作背景關係中，描述日常任務通常使用 `present simple` 時態，因為這些是習慣性動作。
