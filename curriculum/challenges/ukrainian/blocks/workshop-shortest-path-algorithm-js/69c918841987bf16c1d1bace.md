# --description--

Тепер, коли ви знаєте, що вузол досяжний і не є початковим, відформатуйте його шлях у читабельний рядок.

Масив `paths[nodeNo]` містить послідовність індексів вузлів, які відвідувалися, щоб дістатися до `nodeNo`. Ви можете перетворити це на зрозумілий рядок, як `0 -> 2 -> 3`, використовуючи метод `.join()`.

Всередині циклу `for...of`, після інструкції `if` з `continue`, оголосіть змінну з ім’ям `path` і призначте їй значення `paths[nodeNo].join(' -> ')`.
