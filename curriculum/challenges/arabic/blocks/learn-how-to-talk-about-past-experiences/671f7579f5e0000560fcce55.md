# --explanation--

التركيب `what` + أداة تنكير يُستخدم لتأكيد مشاعر قوية أو ردود فعل تجاه شيء ما. غالبًا ما يعبر عن الدهشة أو الإعجاب أو العاطفة. على سبيل المثال:

`What a beautiful day it is!` - هنا، يبرز التركيب إعجاب المتحدث باليوم.

`I attended PyCon last year and learned about the latest Python frameworks.` - هذا يعني أن الشخص حضر PyCon العام الماضي وتعلم الكثير.
