## Utangulizi wa CSS Flexbox na Mfano wa Kubadilika

- **Ufafanuzi**: CSS flexbox ni mfano wa mpangilio wa kipimo kimoja unaokuwezesha kupanga vipengele katika safu na nguzo ndani ya kontena. 
- **Mfano wa Kubadilika**: Mfano huu unaeleza jinsi vipande vya kubadilika vinavyopangwa ndani ya chombo cha kubadilika. Kila chombo cha kubadilika kina mihimili miwili: mhimili mkuu na mhimili wa msalaba.

## Sifa ya `flex-direction`

- **Ufafanuzi**: Sifa hii inaweka mwelekeo wa mhimili mkuu. Thamani ya msingi ya `flex-direction` ni `row`, ambayo huweka vipande vyote vya kubadilika kwenye safu moja, kwa mwelekeo wa lugha ya kivinjari chako (kutoka kushoto kwenda kulia au kutoka kulia kwenda kushoto). 
- **`flex-direction: row-reverse;`**: Hii hubadilisha mpangilio wa vipande kwenye safu.
- **`flex-direction: column;`**: Hii itaweka vipande vya kubadilika wima badala ya usawa.
- **`flex-direction: column-reverse;`**: Hii hubadilisha mpangilio wa vipande vya kubadilika wima.

:::interactive_editor

```html
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: row;
  gap: 10px;
}

.box {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}
```

:::

## Sifa ya `flex-wrap`

- **Ufafanuzi**: Sifa hii inaamua jinsi vipande vya kubadilika vinavyojifunga ndani ya chombo cha kubadilika ili kufaa nafasi iliyopo. `flex-wrap` inaweza kuchukua thamani tatu zinazowezekana: `nowrap`, `wrap`, na `wrap-reverse`.
- **`flex-wrap: nowrap;`**: Hii ni thamani ya msingi. Vipande vya kubadilika havitajifunga kwenye mstari mpya, hata kama upana wao unazidi upana wa kontena.
- **`flex-wrap: wrap;`**: Sifa hii itajifunga vipande wakati vinapozidi upana wa kontena lao.
- **`flex-wrap: wrap-reverse;`**: Sifa hii itajifunga vipande vya kubadilika kwa mpangilio wa mwelekeo wa kinyume.
- **Sifa ya `flex-flow`**: Sifa hii ni sifa ya ufupisho kwa `flex-direction` na `flex-wrap`.

:::interactive_editor

```html
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
  <div class="box">4</div>
  <div class="box">5</div>
</div>
```

```css
.container {
  display: flex;
  flex-wrap: wrap;
  width: 150px;
  background: #f0f0f0;
}

.box {
  width: 60px;
  padding: 10px;
  margin: 5px;
  background: skyblue;
  text-align: center;
}
```

:::

## Sifa ya `justify-content`

- **Ufafanuzi**: Sifa hii huoanisha vipengele vidogo kando ya mhimili mkuu wa chombo cha kubadilika.
- **`justify-content: flex-start;`**: Katika kesi hii, vipande vya kubadilika vitaoanishwa mwanzoni mwa mhimili mkuu. Hii inaweza kuwa usawa au wima.
- **`justify-content: flex-end;`**: Katika kesi hii, vipande vya kubadilika vinaoanishwa mwishoni mwa mhimili mkuu, usawa au wima.
- **`justify-content: center;`**: Hii huweka vipande vya kubadilika katikati kando ya mhimili mkuu.
- **`justify-content: space-between;`**: Hii itasambaza vipengele kwa usawa kando ya mhimili mkuu.
- **`justify-content: space-around;`**: Hii itasambaza vipande vya kubadilika kwa usawa ndani ya mhimili mkuu, ikiongeza nafasi kabla ya kipande cha kwanza na baada ya kipande cha mwisho.
- **`justify-content: space-evenly;`**: Hii itasambaza vipande kwa usawa kando ya mhimili mkuu.

:::interactive_editor

```html
<div class="container">
  <div class="box">A</div>
  <div class="box">B</div>
  <div class="box">C</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
  background: #eee;
}

.box {
  padding: 20px;
  background: salmon;
}
```

:::
  
## Sifa ya `align-items`

- **Ufafanuzi**: Sifa hii hutumika kusambaza vipande kando ya mhimili wa msalaba. Kumbuka kwamba mhimili wa msalaba ni wima kwa mhimili mkuu.
- **`align-items: center;`**: Hii hutumika kuweka vipande katikati kando ya mhimili wa msalaba.
- **`align-items: flex-start;`**: Hii huoanisha vipande mwanzoni mwa mhimili wa msalaba.
- **`align-items: stretch;`**: Hii hutumika kunyoosha vipande vya kubadilika kando ya mhimili wa msalaba.

:::interactive_editor

```html
<div class="container">
  <div class="box tall">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  align-items: center;
  height: 150px;
  background: #ddd;
}

.box {
  background: lightgreen;
  padding: 10px;
}

.tall {
  height: 100px;
}
```
