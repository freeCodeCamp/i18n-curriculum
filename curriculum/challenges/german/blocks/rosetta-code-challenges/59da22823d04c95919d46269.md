# --description--

Fünf Matrosen sind auf einer Insel Schiffbruch erlitten und sammeln tagsüber einen großen Haufen Kokosnüsse.

In der Nacht wacht der erste Matrose auf und beschließt, seinen ersten Anteil frühzeitig zu nehmen. Er versucht, den Haufen Kokosnüsse gleichmäßig in fünf Stapel zu teilen, stellt aber fest, dass eine Kokosnuss übrig bleibt. Diese wirft er einem Affen zu, versteckt dann „seinen“ von fünf gleich großen Kokosnussstapeln und schiebt die anderen vier Stapel wieder zusammen, sodass erneut ein einzelner sichtbarer Kokosnussstapel entsteht, und geht schlafen.

Um es kurz zu machen: Jeder der Matrosen steht nacheinander einmal in der Nacht auf und führt dieselben Schritte aus: Er teilt den Kokosnusshaufen in fünf, stellt fest, dass eine Kokosnuss übrig bleibt, gibt diese einzelne Kokosnuss dem Affen und schiebt dann die anderen vier Stapel wieder zusammen, sodass ein einzelner Stapel entsteht.

Am Morgen (nach der heimlichen und getrennten Aktion jedes der fünf Matrosen in der Nacht) werden die verbleibenden Kokosnüsse in fünf gleiche Stapel für jeden Matrosen aufgeteilt, wobei festgestellt wird, dass sich der Kokosnusshaufen ohne Rest gleichmäßig unter den Matrosen aufteilen lässt. (Am Morgen bleibt nichts für den Affen übrig.)

# --instructions--

Erstellen Sie eine Funktion, die die kleinste mögliche Größe des ursprünglichen Kokosnusshaufens zurückgibt, der tagsüber von `N` Matrosen gesammelt wurde. **Hinweis:** Natürlich spielt die Geschichte in einer Welt, in der das Sammeln beliebiger Kokosnussmengen an einem Tag und mehrfache Teilungen des Haufens usw. zeitlich so möglich sind, dass die Handlung nicht die Mathematik beeinflusst.

```js
function splitCoconuts(intSailors) {
  let intNuts = intSailors;
  let result = splitCoconutsHelper(intNuts, intSailors);
  while (!result) {
    intNuts += 1;
    result = splitCoconutsHelper(intNuts, intSailors);
  }

  return intNuts;
}

function splitCoconutsHelper(intNuts, intSailors, intDepth) {
  const nDepth = intDepth !== undefined ? intDepth : intSailors;
  const portion = Math.floor(intNuts / intSailors);
  const remain = intNuts % intSailors;

  if (portion <= 0 || remain !== (nDepth ? 1 : 0)) {
    return null;
  }

  if (nDepth) {
    return splitCoconutsHelper(
      intNuts - portion - remain, intSailors, nDepth - 1
    );
  }

  return intNuts;
}
```
