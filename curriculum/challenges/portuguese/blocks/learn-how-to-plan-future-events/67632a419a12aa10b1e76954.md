# --explanation--

`Database` refere-se a um sistema onde os dados são armazenados e organizados.

`Activity` refere-se às ações ou processos que acontecem no banco de dados.

- `The database contains customer information and order details.` - Isso significa que o banco de dados armazena dados importantes do negócio.  

- `There was unusual activity in the database last night.` - Isso significa que algo inesperado aconteceu no banco de dados.

Juntos, `database activity` significa todas as ações, mudanças ou processos acontecendo dentro do sistema de banco de dados.
