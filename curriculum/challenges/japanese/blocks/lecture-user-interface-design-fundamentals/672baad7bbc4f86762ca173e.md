# --interactive--

**注意**: いくつかのインタラクティブな例では、まだ学習していないCSSが使われている場合があります。コードのすべてを理解しようと心配しないでください。これらの例の目的は、デザインの概念をプレビューで示し、仕組みをよりよく理解してもらうことです。

ウェブサイトに画像を追加することは、ユーザーの関心を引きつけ、サイトの視覚的な魅力を高める素晴らしい方法です。しかし、デザインで画像を扱う際にはいくつか考慮すべき点があります。

まず考慮すべきはレスポンシブ画像の作成です。レスポンシブ画像とは、表示される画面のサイズに合わせて拡大縮小される画像のことです。これは、デスクトップからスマートフォンまで、すべてのデバイスで画像が美しく見えるようにするために重要です。

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    padding: 20px;
    background-color: #fefefe;
    color: #333;
    text-align: center;
  }

  img {
    max-width: 100%;
    height: auto;
    border-radius: 8px;
  }

  p {
    font-size: 16px;
    max-width: 600px;
    margin: 20px auto;
    line-height: 1.6;
  }
</style>

<h1>Responsive Cat Image</h1>

<img 
  src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" 
  alt="Two cats peacefully sleeping together."
/>

<p>
  This image automatically scales based on the screen size. Whether you're viewing on a desktop or a mobile phone,
  it adjusts its size without losing proportions, making the design clean and user-friendly on all devices.
</p>
```

:::

次に考慮すべきは画像の解像度です。解像度が高い画像は、1インチあたりのピクセル数が多くなります。ピクセルは画像を構成する小さな正方形です。

PPI（Pixels Per Inch）は、画像の1インチあたりのピクセル数を表します。PPIが高いほど、画像の品質が良くなります。

画像が高品質で、すべてのデバイスで美しく見えるようにする必要があります。つまり、ウェブ用に最適化された高解像度の画像を使うべきです。

また、画像のサイズとレイアウト内のスペースへの収まり方も考慮してください。画像が適切なサイズで、大きすぎたり小さすぎたりしないことが重要です。

小さいスペースに合わせるために大きな画像を使うと、ウェブサイトの読み込みが遅くなり、ユーザーがサイトを開くのが難しくなります。画像は適切なサイズで、ウェブ用に最適化されていることを確認してください。

画像の配置に関しては、バランス、階層、整列を考慮して、画像がウェブ用に最適化されるようにしましょう。

バランスとは、デザイン内の視覚的な重みの分布のことです。テキストと画像のバランスが良いことで、調和のとれたデザインになります。

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    margin: 0;
    padding: 40px 20px;
    background-color: #f9f9f9;
    color: #333;
  }

  .container {
    display: flex;
    flex-wrap: wrap;
    align-items: center;
    justify-content: space-between;
    gap: 30px;
    max-width: 1000px;
    margin: 0 auto;
  }

  .text {
    flex: 1 1 400px;
  }

  .text h2 {
    font-size: 28px;
    margin-bottom: 10px;
  }

  .text p {
    font-size: 16px;
    line-height: 1.6;
  }

  .image {
    flex: 1 1 400px;
  }

  .image img {
    width: 100%;
    height: auto;
    border-radius: 8px;
  }
</style>

<div class="container">
  <div class="text">
    <h2>Balanced Layout</h2>
    <p>
      Balance is essential in web design. By evenly distributing visual weight—such as pairing this block of text
      with a complementary image—you create a layout that feels calm, structured, and easy to navigate.
    </p>
  </div>

  <div class="image">
    <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" alt="Two cats peacefully sleeping together.">
  </div>
</div>
```

:::

階層とは、ページ上で要素が閲覧される順序のことです。重要なコンテンツに関連する画像は、重要度の低い画像よりも上位に配置する必要があります。

整列とは、要素同士の位置関係のことです。画像がテキストや他の要素と整列していることで、一体感のあるデザインになります。

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    padding: 40px 20px;
    background-color: #ffffff;
    color: #222;
    max-width: 900px;
    margin: 0 auto;
  }

  .aligned-content {
    display: flex;
    flex-wrap: wrap;
    align-items: center;
    gap: 20px;
  }

  .aligned-content img {
    width: 300px;
    height: auto;
    border-radius: 8px;
    flex-shrink: 0;
  }

  .aligned-text {
    flex: 1;
    min-width: 250px;
  }

  .aligned-text h2 {
    font-size: 24px;
    margin-bottom: 10px;
  }

  .aligned-text p {
    font-size: 16px;
    line-height: 1.6;
  }
</style>

<div class="aligned-content">
  <img 
    src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" 
    alt="Two cats peacefully sleeping together."
  >

  <div class="aligned-text">
    <h2>Aligned Elements</h2>
    <p>
      In this example, the image is aligned horizontally next to the text block. This creates a visually cohesive layout, 
      where both elements feel like they belong together. Proper alignment like this improves readability and reinforces 
      structure in your design.
    </p>
  </div>
</div>
```

:::

最後に考慮すべきは画像のアクセシビリティです。視覚障害のあるユーザーを含むすべてのユーザーが画像にアクセスできるようにする必要があります。つまり、スクリーンリーダーが視覚障害のあるユーザーに読み上げられるように、画像にはaltテキストを設定してください。
