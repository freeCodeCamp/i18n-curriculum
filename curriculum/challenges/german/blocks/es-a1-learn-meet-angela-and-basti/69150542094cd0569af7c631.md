# --description--

`¿Y tú?` bedeutet „Und du?“

Diese kurze Frage wird verwendet, um dem Sprecher schnell eine Frage zurückzugeben, nachdem Sie sie beantwortet haben, ohne sie zu wiederholen.

# --instructions--

Hören Sie sich das Audio an und beantworten Sie die folgende Frage.
