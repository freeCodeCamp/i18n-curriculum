# --description--

数列ゲームは、$N$ 個の数字からなる数列 $S$ が横一列に書かれているところから始まります。

2人のプレイヤーが交互に手番を取ります。手番では、プレイヤーは残っている数列の最初か最後のどちらかの数字を選んで取り除かなければなりません。

プレイヤーの得点は、彼が取ったすべての数字の合計です。各プレイヤーは自分の得点を最大化しようとします。

$N = 4$ で $S = \\{1, 2, 10, 3\\}$ の場合、各プレイヤーは次のように得点を最大化します：

- プレイヤー1：最初の数字（1）を取り除く
- プレイヤー2：残った数列の最後の数字（3）を取り除く
- プレイヤー1：残った数列の最後の数字（10）を取り除く
- プレイヤー2：残った数字（2）を取り除く

プレイヤー1の得点は $1 + 10 = 11$ です。

$F(N)$ を、両プレイヤーが最適戦略に従った場合のプレイヤー1の得点とします。数列 $S = \\{s_1, s_2, \ldots, s_N\\}$ は次のように定義されます：

- $s_1 = 0$
- $s_{i + 1} = ({s_i}^2 + 45)$ を $1\,000\,000\,007$ で割った余り（モジュラス）

数列は $S = \\{0, 45, 2\,070, 4\,284\,945, 753\,524\,550, 478\,107\,844, 894\,218\,625, \ldots\\}$ から始まります。

$F(2) = 45$、$F(4) = 4\,284\,990$、$F(100) = 26\,365\,463\,243$、$F(104) = 2\,495\,838\,522\,951$ が与えられています。

$F({10}^8)$ を求めてください。
