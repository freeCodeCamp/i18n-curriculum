# --description--

На кожній із шести граней куба написана різна цифра (від 0 до 9); те саме зроблено з другим кубом. Розміщуючи два куби поруч у різних положеннях, ми можемо утворювати різноманітні двозначні числа.

Наприклад, квадратне число 64 можна утворити так:

<img alt="two cubes, one with the number 6 and the other with number 4" src="https://cdn-media-1.freecodecamp.org/project-euler/cube-digit-pairs.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Насправді, ретельно вибравши цифри на обох кубах, можна відобразити всі квадратні числа менші за сто: 01, 04, 09, 16, 25, 36, 49, 64 і 81.

Наприклад, один із способів досягти цього — розмістити {0, 5, 6, 7, 8, 9} на одному кубі і {1, 2, 3, 4, 8, 9} на іншому.

Однак для цього завдання ми дозволимо повертати 6 або 9 догори дригом, тому розташування {0, 5, 6, 7, 8, 9} і {1, 2, 3, 4, 6, 7} дозволяє відобразити всі дев’ять квадратних чисел; інакше було б неможливо отримати 09.

При визначенні унікального розташування нас цікавлять цифри на кожному кубі, а не порядок.

<div style="margin-left: 4em;">
  {1, 2, 3, 4, 5, 6} еквівалентна {3, 6, 4, 1, 2, 5}<br>
  {1, 2, 3, 4, 5, 6} відрізняється від {1, 2, 3, 4, 5, 9}
</div>

Але оскільки ми дозволяємо перевертати 6 і 9, дві різні множини в останньому прикладі обидві представляють розширену множину {1, 2, 3, 4, 5, 6, 9} для формування двозначних чисел.

Скільки унікальних розташувань двох кубів дозволяють відобразити всі квадратні числа?
