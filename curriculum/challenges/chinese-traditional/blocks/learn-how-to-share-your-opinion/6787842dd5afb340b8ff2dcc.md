# --explanation--

當被問到為什麼選擇 Python 作為你的第一個語言時，最好凸顯標示 Python 的特定優勢。

Python 以對初學者友善著稱，具有簡單且易於閱讀和學習的語法。此外，Python 擁有豐富的函式庫和框架，使其成為初學者的實用選擇。

另一方面，說「`I've always wanted to be a developer`」並沒有解釋你為什麼選擇 Python。它著重於成為開發者的渴望，但缺乏任何與 Python 相關的理由。
