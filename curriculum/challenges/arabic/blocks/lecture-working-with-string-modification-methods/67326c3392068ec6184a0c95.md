# --interactive--

عند العمل مع JavaScript، قد تواجه مواقف تحتاج فيها إلى تكرار سلسلة نصية عددًا محددًا من المرات.

سواء كنت تُنشئ أنماطًا متكررة أو ببساطة تكرر نصًا، توفر طريقة `repeat()` وسيلة بسيطة وفعالة لتحقيق ذلك.

طريقة `repeat()` هي دالة مدمجة في JavaScript تسمح لك بتكرار سلسلة نصية عددًا محددًا من المرات. إليك الصيغة النحوية الأساسية:

```js
string.repeat(count);
```

`string` هي السلسلة النصية التي تريد تكرارها، و`count` هو عدد المرات التي تريد تكرار السلسلة فيها. إليك مثالًا:

:::interactive_editor

```js
let word = "Hello!";
let repeatedWord = word.repeat(3);
console.log(repeatedWord);  // "Hello!Hello!Hello!"
```

:::

في هذه الحالة، تتكرر السلسلة النصية `Hello!` ثلاث مرات، مما ينتج عنه `Hello!Hello!Hello!`.

بينما طريقة `repeat()` مفيدة، هناك بعض الاستثناءات والقيود التي يجب الانتباه إليها.

يجب أن تكون معلمة `count` رقمًا غير سالب. إذا مررت برقم سالب، ستُرمى في JavaScript استثناء `RangeError`.

```js
let word = "Test";
console.log(word.repeat(-1));  // Throws RangeError: Invalid count value
```

يجب أن يكون `count` رقمًا محدودًا. إذا حاولت تكرار سلسلة عددًا لا نهائيًا من المرات أو استخدمت `Infinity` كعدد التكرار، ستحصل أيضًا على `RangeError`.

في JavaScript، `Infinity` قيمة خاصة تمثل كمية لا نهائية. تُستخدم للدلالة على أرقام أكبر من أي رقم محدود.

```js
let word = "Test";
console.log(word.repeat(Infinity));  // Throws RangeError: Invalid count value
```

إذا لم يكن العدد صحيحًا (مثل رقم عشري مثل `2.5`)، ستقوم طريقة `repeat()` بتقريبه إلى أقرب عدد صحيح أقل.

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(2.5));  // "TestTest"
```

:::

إذا مررت بالعدد `0` كعدد التكرار، ستُرجع طريقة `repeat()` سلسلة نصية فارغة.

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(0));  // ""
```

:::

يمكن لطريقة `repeat()` تبسيط المهام التي تتطلب تكرار السلاسل النصية، مما يجعل شفرتك أكثر إيجازًا وقابلية للقراءة.

سواء كنت تُنشئ أنماط نصية متكررة أو تملأ مساحة بأحرف، يمكن لـ `repeat()` أن توفر عليك كتابة الحلقات أو الشفرات الأكثر تعقيدًا.

لست مقيدًا بتمرير رقم مباشرة إلى طريقة `repeat()`. يمكنك أيضًا تمرير متغير يخزن قيمة رقمية.

:::interactive_editor

```js
let count = 4;
let word = "Test";
let repeatedWord = word.repeat(count);
console.log(repeatedWord); // TestTestTestTest
```

:::

في هذا المثال، يخزن المتغير `count` عدد التكرارات. هذا مفيد عندما يعتمد عدد التكرارات على مُدخل المستخدم أو قيم ديناميكية أخرى في برنامجك.
