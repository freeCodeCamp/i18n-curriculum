# --description--

Програмуючи на JavaScript, ви будете стикатись з повідомленнями про помилки. Розуміння поширених повідомлень про помилки допоможе ефективніше налагоджувати код і стати кращим програмістом. Чотири поширені типи повідомлень про помилки — це `SyntaxError`, `ReferenceError`, `TypeError` та `RangeError`.

`SyntaxError` виникає, коли ви пишете щось у коді неправильно (наприклад, пропускаєте круглу або квадратну дужку). Думайте про це як граматичну помилку в реченні. Ось поширена помилка, яку роблять розробники при створенні масивів:

```js
const arr = ["Beau", "Quincy" "Tom"]
```

Кожен елемент масиву має бути відокремлений комою, інакше виникне повідомлення про помилку.

Друга поширена помилка в JavaScript — `ReferenceError`. Існує кілька типів `ReferenceError`, які виникають різними способами. Один з типів `ReferenceError` — це не визначені змінні.

```js
console.log(price);
```

У цьому прикладі ми намагаємося вивести змінну `price` на консоль, але вона не була визначена. Це призведе до `ReferenceError`.

Ще один приклад `ReferenceError` — це спроба отримати доступ до змінної, оголошеної за допомогою `let` або `const`, до того, як вона була визначена:

```js
console.log(b);
const b = 50;
```

Вищенаведений приклад призведе до помилки `Cannot access 'b' before initialization`. 

Третя поширена помилка — `TypeError`. Вона виникає, коли ви намагаєтесь виконати операцію над неправильним типом. Ось спроба використати метод `map` на об’єкті:

```js
const developerObj = {
  name: "Jessica",
  country: "USA",
  isEmployed: true
};

developerObj.map()
```

Цей приклад призведе до помилки `developerObj.map is not a function`, тому що метод `map` використовується для масивів, а не об’єктів. 

Остання поширена помилка, яку ми розглянемо, — `RangeError`.

`RangeError` виникає, коли код намагається використати значення, яке знаходиться поза межами діапазону, що може обробити JavaScript. Ось приклад призначення недійсного значення довжині масиву:

```js
const arr = [];
arr.length = -1; 
```

Значення `length` масиву має бути невід’ємним цілим числом, тому `-1` викликає `RangeError`.

Продовжуючи програмувати на JavaScript, просто майте на увазі ці типи помилок, з якими ви, ймовірно, зустрінетесь.
