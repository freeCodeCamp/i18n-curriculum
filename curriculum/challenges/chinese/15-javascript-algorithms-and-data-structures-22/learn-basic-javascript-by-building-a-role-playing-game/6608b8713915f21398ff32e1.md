# --description--

如果对象的属性名称（键）中包含空格，则需要在名称周围使用单引号或双引号。

以下是属性名称中有空格的对象的示例：

```js
const spaceObj = {
  "Space Name": "Kirk",
};
```

如果你尝试写入不带引号的键，则会引发错误：

```js
const spaceObj = {
  // Throws an error
  Space Name: "Kirk",
}; 

```

向 `cat` 对象添加一个新属性，其键为 `"Number of legs"`，值为 `4`。

打开控制台查看输出。

# --hints--

你的 `cat` 对象应该有一个名为 `"Number of legs"` 的键。

```js
assert.property(cat, "Number of legs");
```

你的 `cat` 对象应该具有一个 `"Number of legs"` 属性，其值为 `4`。

```js
assert.propertyVal(cat, "Number of legs", 4);
```

你的 `cat` 对象应该具有两个属性。

```js
assert.lengthOf(Object.keys(cat), 2);
```
