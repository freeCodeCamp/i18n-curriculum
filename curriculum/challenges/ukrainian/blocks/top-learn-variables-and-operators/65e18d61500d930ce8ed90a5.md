# --description--

Ви можете уявити змінні як «контейнери для зберігання» даних у вашому коді.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/variables-and-operators/top-learn-variables-and-operators.png" alt='концепція змінних у програмуванні як блоки з мітками "birthday", які зберігають і надають доступ до даних, зокрема дати "16-09-2003"'>

Змінна — це «іменований контейнер» для даних. Ви можете використовувати змінні, щоб зберігати різні речі, відвідувачів та інші дані.

Щоб створити змінну в JavaScript, використовуйте ключове слово `let`.

Інструкція нижче створює (іншими словами: оголошує) змінну з ім’ям "message":

```js
let message;
```

Тепер ви можете покласти в неї деякі дані, використовуючи оператор присвоєння `=`:

```js
let message;

// store the string 'Hello' in the variable named message
message = 'Hello'; 
```

Щоб бути лаконічним, ви можете поєднати оголошення змінної та присвоєння в одному рядку:

```js
let message = 'Hello';
```
