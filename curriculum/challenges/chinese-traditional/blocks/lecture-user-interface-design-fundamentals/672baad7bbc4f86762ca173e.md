# --interactive--

**注意**：有些互動範例可能會使用你還沒學過的 CSS。不要擔心要理解所有的程式碼。這些範例的目標是為你展示這些設計概念的預覽，讓你更了解事情的運作方式。

為你的網站添加影像是吸引你的使用者並提升網站視覺吸引力的絕佳方式。但在設計中使用影像時，有幾件事需要注意。

首先要考慮的是建立響應式影像。響應式影像是會根據所觀看的螢幕大小進行縮放的影像。這很重要，因為它確保你的影像在所有裝置上都能呈現良好，從桌上型電腦到手機。

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    padding: 20px;
    background-color: #fefefe;
    color: #333;
    text-align: center;
  }

  img {
    max-width: 100%;
    height: auto;
    border-radius: 8px;
  }

  p {
    font-size: 16px;
    max-width: 600px;
    margin: 20px auto;
    line-height: 1.6;
  }
</style>

<h1>Responsive Cat Image</h1>

<img 
  src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" 
  alt="Two cats peacefully sleeping together."
/>

<p>
  This image automatically scales based on the screen size. Whether you're viewing on a desktop or a mobile phone,
  it adjusts its size without losing proportions, making the design clean and user-friendly on all devices.
</p>
```

:::

另一件需要考慮的是影像的決議程序。具有較高品質且決議程序較佳的影像每英吋擁有更多像素。像素是組成影像的小方格。

每英吋像素數（Pixels per inch，簡稱 PPI）是指一英吋影像中的像素數量。PPI 越高，影像品質越好。

你想確保你的影像品質高且在所有裝置上看起來都很好。這表示你應該使用為網路最佳化的高解析度影像。

另一件需要考慮的是你的影像大小以及它們如何適合版面中的空間。你要確保你的影像大小適中，不會太大或太小。

使用原本要放在較小空間中的大型影像，會降低你網站的速度並讓使用者更難載入你的網站。你要確保你的影像尺寸正確且已為網路最佳化。

在影像擺放方面，你需要考慮平衡、階層體系和對齊，以協助確保你的影像為網路最佳化。

平衡是設計中視覺重量的分佈。你要確保網站上的文字和影像之間有良好的平衡，才能創造出和諧的設計。

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    margin: 0;
    padding: 40px 20px;
    background-color: #f9f9f9;
    color: #333;
  }

  .container {
    display: flex;
    flex-wrap: wrap;
    align-items: center;
    justify-content: space-between;
    gap: 30px;
    max-width: 1000px;
    margin: 0 auto;
  }

  .text {
    flex: 1 1 400px;
  }

  .text h2 {
    font-size: 28px;
    margin-bottom: 10px;
  }

  .text p {
    font-size: 16px;
    line-height: 1.6;
  }

  .image {
    flex: 1 1 400px;
  }

  .image img {
    width: 100%;
    height: auto;
    border-radius: 8px;
  }
</style>

<div class="container">
  <div class="text">
    <h2>Balanced Layout</h2>
    <p>
      Balance is essential in web design. By evenly distributing visual weight—such as pairing this block of text
      with a complementary image—you create a layout that feels calm, structured, and easy to navigate.
    </p>
  </div>

  <div class="image">
    <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" alt="Two cats peacefully sleeping together.">
  </div>
</div>
```

:::

階層體系是元素在頁面上被觀看的順序。你要確保與重要內容對齊的影像被放置在比不那麼重要的影像更高的位置。

:::interactive_editor

  .section {

:::

對齊是元素彼此之間的排列方式。你要確保你的影像與文字及網站上的其他元素對齊，以便創造出一致的設計。

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    padding: 40px 20px;
    background-color: #ffffff;
    color: #222;
    max-width: 900px;
    margin: 0 auto;
  }

  .aligned-content {
    display: flex;
    flex-wrap: wrap;
    align-items: center;
    gap: 20px;
  }

  .aligned-content img {
    width: 300px;
    height: auto;
    border-radius: 8px;
    flex-shrink: 0;
  }

  .aligned-text {
    flex: 1;
    min-width: 250px;
  }

  .aligned-text h2 {
    font-size: 24px;
    margin-bottom: 10px;
  }

  .aligned-text p {
    font-size: 16px;
    line-height: 1.6;
  }
</style>

<div class="aligned-content">
  <img 
    src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" 
    alt="Two cats peacefully sleeping together."
  >

  <div class="aligned-text">
    <h2>Aligned Elements</h2>
    <p>
      In this example, the image is aligned horizontally next to the text block. This creates a visually cohesive layout, 
      where both elements feel like they belong together. Proper alignment like this improves readability and reinforces 
      structure in your design.
    </p>
  </div>
</div>
```

:::

最後要考慮的是影像的存取性。你要確保你的影像對所有使用者都能存取，包括視覺障礙者。這表示你應該為你的影像使用替代文字，以便螢幕閱讀器能將文字讀給視覺障礙的使用者聽。
