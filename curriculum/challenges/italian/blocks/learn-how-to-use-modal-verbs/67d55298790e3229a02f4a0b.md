# --explanation--

Una `data breach` si verifica quando informazioni sensibili vengono consultate da persone non autorizzate. Per esempio:

- `A data breach can happen if hackers get access to private information.` - Se i criminali informatici rubano informazioni, si chiama violazione dei dati.

- `Companies must act quickly if there is a data breach.` - Quando avviene una violazione dei dati, le aziende devono rispondere rapidamente per risolverla.

Jessica dice che criptare i dati e conservare in modo sicuro le chiavi di crittografia è essenziale per prevenire questo rischio.
