# --interactive--

HTMLやCSSのプロジェクトで属性を扱ったことがあるかもしれません。でも、JavaScriptのファイル内でHTML要素に属性を追加できることをご存知でしたか？

このレッスンでは、`setAttribute()` メソッドを使ってHTML要素に属性を追加する方法を学びます。

基本的な構文は次の通りです：

```js
setAttribute(attribute, value);
```

`setAttribute()` メソッドの使い方をよりよく理解するために、いくつか例を見てみましょう。

最初の例では、HTMLの中に`p` 要素があります：

```html
<p id="para">I am a paragraph</p>
```

`class` 属性を追加するには、まず`p` メソッドを使ってその`getElementById()` 要素にアクセスする必要があります。次に`setAttribute()` メソッドを使って`class` 属性を追加し、その値を`my-class` に設定します。

`outerHTML` を使って、属性を含むHTML要素全体をログに出力しています。

:::interactive_editor

```html
<p id="para">I am a paragraph</p>
<script src="index.js"></script>
```

```js
const para = document.getElementById("para");
para.setAttribute("class", "my-class");

console.log(`${para.outerHTML}`);
```

:::

HTML要素に既に属性がある場合は、`setAttribute()` メソッドを使ってその値を更新できます。

この例では、`div` 要素に`class` 属性が`my-class` に設定されています：

```html
<div class="my-class"></div>
```

`class` 属性の値を example に更新するには、再び`setAttribute()` メソッドを使います：

:::interactive_editor

```html
<div class="my-class"></div>
<script src="index.js"></script>
```

```js
const divEl = document.querySelector(".my-class");
divEl.setAttribute("class", "example");

console.log(`${divEl.outerHTML}`);
```

:::

では、実際の場面で`setAttribute()` メソッドを使う例はどんなものがあるでしょうか？

動的な画像ギャラリーを作っている場合、ユーザーがサムネイルをクリックしたときに画像要素の`src` 属性を更新する必要があるかもしれません。

また、フォームのバリデーションを扱う場合、`required` や`minlength` のような属性を`input` 要素に追加する必要があることもあります。
