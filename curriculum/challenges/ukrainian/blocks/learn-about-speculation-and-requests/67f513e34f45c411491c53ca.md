# --explanation--

`Incident response team` — це спеціалізована група, яка займається інцидентами безпеки. Наприклад:

- `The incident response team handled the security breach.` — команда вирішила проблему, коли систему зламали.

- `We called the incident response team when we noticed suspicious activity.` — вони звернулися до команди, коли помітили щось підозріле.
