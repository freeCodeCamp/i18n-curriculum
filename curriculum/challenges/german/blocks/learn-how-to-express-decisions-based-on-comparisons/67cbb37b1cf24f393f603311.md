# --explanation--

`Crucial` bedeutet, dass etwas äußerst wichtig oder lebenswichtig ist. Zum Beispiel:  

`Security is a crucial aspect of the system.` – Das bedeutet, dass Sicherheit im System sehr wichtig ist und nicht übersehen werden darf.

`Extra` bezieht sich auf etwas, das über die übliche Menge hinausgeht. Es impliziert, dass mehr Arbeit oder Aufwand als normal erforderlich ist. Zum Beispiel:  

`The extra work on this project took longer than expected.` – Das bedeutet, dass mehr Arbeit nötig war als ursprünglich geplant und zusätzlicher Aufwand erforderlich ist.
