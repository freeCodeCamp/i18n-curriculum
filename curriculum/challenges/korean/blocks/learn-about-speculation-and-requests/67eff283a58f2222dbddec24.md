# --explanation--

`Have you checked...?`는 누군가가 지금까지 언제든지 어떤 일을 했는지 물을 때 사용합니다. 예를 들어:

- `Have you checked your email?` - 이는 상대방이 지금까지 이메일을 확인했는지 묻는 것입니다.

- `Have you checked if the server is running?` - 누군가가 서버 상태를 확인했는지 묻고 있습니다.
