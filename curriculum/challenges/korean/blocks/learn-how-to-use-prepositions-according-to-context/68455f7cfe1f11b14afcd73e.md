# --explanation--

`Layout`는 공간의 디자인이나 구조를 의미합니다. 예를 들어:  

`He shared the layout of the new office.` – 이것은 물건이 어디에 배치되는지에 대한 계획입니다.

`Draft`는 문서의 초기 버전입니다. 예를 들어:  

`She wrote the first draft of the proposal.` – 아직 편집할 수 있습니다.

`Send it over`는 무언가를 전달하는 것을 의미합니다. 예를 들어:  

`I'll send it over once it's ready.` – 파일이나 메시지를 공유하는 것을 뜻합니다.

`To`는 행동이 누구를 위한 것인지 보여줍니다. 예를 들어:  

`She gave the file to her manager.` – 누가 그것을 받는지 나타냅니다.
