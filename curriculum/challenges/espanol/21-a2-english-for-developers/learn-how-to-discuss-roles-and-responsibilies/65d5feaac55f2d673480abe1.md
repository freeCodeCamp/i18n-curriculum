# --description--

Las palabras `super` y `pretty` se utilizan para enfatizar el grado de algo.

`Super` significa muy o extremadamente. Por ejemplo, `This book is super interesting` significa que el libro es muy interesante.

`Pretty` se utiliza para significar "bastante" o "más bien". Es una manera de decir que algo es en gran medida, pero no extremadamente. Por ejemplo, si dices `It's pretty cold outside`, quieres decir que hace bastante frío, pero no extremadamente frío.
