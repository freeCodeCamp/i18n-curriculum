# --description--

일반적으로 `Past Simple` 시제는 규칙 동사에 `-ed`를 추가하여 만듭니다. 예를 들어, 동사 `play`는 `played`가 됩니다.

`play around`는 보통 무언가를 가볍거나 실험적으로 시도해 보는 것을 의미하며, 종종 결과를 보기 위해서입니다. 보통 `with`가 뒤따릅니다. 예를 들어:

`He played around with the new software to see how it works.` - 이는 그가 진지한 계획 없이 소프트웨어를 사용해 보았다는 뜻입니다.
