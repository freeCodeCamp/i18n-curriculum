# --interactive--

## 語意化 HTML 的重要性

- **標題元素的結構階層體系**：使用正確的標題元素以維護內容的結構階層體系非常重要。`h1` 元素是最高層級的標題，而 `h6` 元素是最低層級的標題。
- **呈現型 HTML 元素**：定義內容外觀的元素。例如已棄用的 `center`、`big` 和 `font` 元素。
- **語意化 HTML 元素**：具有意義和結構的元素。例如 `header`、`nav`、`figure`。

## 語意化 HTML 元素

- **Header element**：用於定義文件或區段的標頭。

:::interactive_editor

  ```html
  <header>
    <h1>CatPhotoApp</h1>
    <p>Welcome to our cat gallery.</p>
  </header>
  ```

:::

- **Main element**：用於包含網頁的主要內容。

:::interactive_editor

  ```html
  <main>
    <section>
      <h2>Cat Photos</h2>
      <p>Browse adorable cat pictures.</p>
    </section>
  </main>
  ```

:::

- **Section 元素**：用於將內容劃分為較小的區段。

:::interactive_editor

  ```html
  <section>
    <h2>About Me</h2>
    <p>Hi, I am Jane Doe and I am a web developer.</p>
  </section>
  ```

:::

- **導航區段（`nav`）元素**：表述一個包含導航聯結的區段。

:::interactive_editor

  ```html
  <nav>
    <ul>
      <li><a href="#photos">Photos</a></li>
      <li><a href="#videos">Videos</a></li>
    </ul>
  </nav>
  ```

:::

- **Figure element**：用於包含插圖和圖表。

:::interactive_editor

  ```html
  <figure>
    <img
      src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"
      alt="Two tabby kittens sleeping together on a couch."
    />
    <figcaption>Cats <strong>hate</strong> other cats.</figcaption>
  </figure>
  ```

:::

- **強調（`em`）元素**：標示具有強調語氣的文字。

:::interactive_editor

  ```html
  <p>
    Never give up on <em>your</em> dreams.
  </p>
  ```

:::

- **慣用語文字（`i`）元素**：用於凸顯標示替代語氣或語調、來自其他語言的慣用語、技術術語及想法。

:::interactive_editor

  ```html
  <p>
    There is a certain <i lang="fr">je ne sais quoi</i> in the air.
  </p>
  ```

:::

開啟的 `i` 標記內的 `lang` 屬性用來指定內容的語言。在這個例子中，語言會是法語。`i` 元素不會顯示文字是否重要，它只顯示該文字與周圍文字有所不同。

- **強調重要性（`strong`）元素**：標示具有強烈重要性的文字。

:::interactive_editor

  ```html
  <p>
    <strong>Warning:</strong> This product may cause allergic reactions.
  </p>
  ```

:::

- **為 `b` 元素凸顯標示**：用於凸顯標示對內容意義不重要的文字。它通常用於在摘要中凸顯標示關鍵字或在評論中凸顯標示產品名稱。

:::interactive_editor

  ```html
  <p>
    We tested several products, including the <b>SuperSound 3000</b> for audio quality, the <b>QuickCharge Pro</b> for fast charging, and the <b>Ecoclean Vacuum</b> for cleaning. The first two performed well, but the <b>Ecoclean Vacuum</b> did not meet expectations.
  </p>
  ```

:::

- **描述列表（`dl`）元素**：用於表述一組術語與描述的列表。
- **Description Term (`dt`) 元素**：用於表述被定義的術語。
- **Description Details (`dd`) 元素**：用於表述該術語的描述。

:::interactive_editor

  ```html
  <dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
  </dl>
  ```

:::

- **區塊引用 (`blockquote`) 元素**：用於表述從其他來源引用的區段。此元素具有 `cite` 屬性。`cite` 屬性的值是來源的 URL。

:::interactive_editor

  ```html
  <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
    "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
  </blockquote>
  ```

:::

- **引用 (`cite`) 元素**：用於視覺上標示被引用作品的來源。標記引用的標題。

:::interactive_editor

  ```html
  <div>
    <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
      "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
    </blockquote>
    <p>
      -Quincy Larson, <cite>How to learn to Code and Get a Developer Job [Full Book].</cite>
    </p>
  </div>
  ```

:::

- **行內引用 (`q`) 元素**：用於表述短的行內引用。

:::interactive_editor

  ```html
  <p>
    As Quincy Larson said,
    <q cite="https://www.freecodecamp.org/news/learn-to-code-book/">
      Momentum is everything.
    </q>
  </p>
  ```

:::

- **縮寫（`abbr`）元素**：用於表述縮寫或首字母縮略詞。為了幫助使用者理解該縮寫或首字母縮略詞是什麼，你可以使用 `title` 屬性顯示其完整形式或可讀的描述。

:::interactive_editor

  ```html
  <p>
    <abbr title="HyperText Markup Language">HTML</abbr> is the foundation of the web.
  </p>
  ```

:::

- **聯絡位址 (`address`) 元素**：用於表述聯絡資訊。
- **(Date) Time (`time`) 元素**：用於表述日期和／或時間。`datetime` 屬性用於將日期和時間轉換成機器可讀格式。

:::interactive_editor

  ```html
  <p>
    The reservations are for the <time datetime="20:00">20:00 </time>
  </p>
  ```

:::

- **ISO 8601 日期 (`datetime`) 屬性**：用於以機器可讀格式表述日期和時間。標準格式為 `YYYY-MM-DDThh:mm:ss`，其中大寫的 `T` 是日期和時間之間的分隔符。
- **上標 (`sup`) 元素**：用於表述上標文字。`sup` 元素的常見使用情境包括指數、上標字母和序數字。

:::interactive_editor

  ```html
  <p>
    2<sup>2</sup> (2 squared) is 4.
  </p>
  ```

:::

- **下標（`sub`）元素**：用於表述下標文字。下標元素的常見使用情境包括化學式、腳註和變數下標。

:::interactive_editor

  ```html
  <p>
    CO<sub>2</sub>
  </p>
  ```

:::

- **行內程式碼（`code`）元素**：用於表述一段電腦程式碼。
- **預格式化文字（`pre`）元素**：表述預格式化文字

:::interactive_editor

  ```html
  <pre>
    <code>
      body {
        color: red;
      }
    </code>
  </pre>
  ```

:::

- **未明確標示註解（`u`）元素**：用於表述一段行內文字，該文字應以顯示其具有非文字註解的方式呈現。

:::interactive_editor

  ```html
  <p>
    You can use the unarticulated annotation element to highlight
    <u>inccccort</u> <u>spling</u> <u>issses</u>.
  </p>
  ```

:::

- **Ruby Annotation (`ruby`) 元素**：用於為文字添加注音或意義說明。範例用法是東亞排版。
- **Ruby 後備括號（`rp`）元素**：用於瀏覽器不支援顯示 ruby 註解時的後備方案。
- **Ruby 文字（`rt`）元素**：用於顯示 ruby 註解的文字。通常用於東亞排版中的發音或翻譯細節。

:::interactive_editor

  ```html
  <ruby>
    明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp>
  </ruby>
  ```

:::

- **刪除線（`s`）元素**：用於表述不再準確或相關的內容。

:::interactive_editor

  ```html
  <p>
    <s>Tomorrow's hike will be meeting at noon.</s>
  </p>
  <p>
    Due to unforeseen weather conditions, the hike has been canceled.
  </p>
  ```

:::

- **內部聯結**：用於透過在 `a` 元素上使用 `href="#id"` 並給予目標元素相同的 `id`，來聯結頁面中的另一個區段。這通常用於跳過聯結、目錄或具有多個區段的長頁面。

:::interactive_editor

```html
<a href="#about-section">Go to About Section</a>

<section id="about-section">
  <h2>About</h2>
  <p>This is the about section of the page.</p>
</section>
```

:::
