# --explanation--

`Exactly` は、誰かの発言に対してそれが完全に正しいことを確認したり、強く同意したりする時に使います。例えば：

- **QAテスター:** `Did you mean we should start testing today?`

- **マネージャー:** `Exactly.` - この人はテストを今日始めるべきだと確信しています。
