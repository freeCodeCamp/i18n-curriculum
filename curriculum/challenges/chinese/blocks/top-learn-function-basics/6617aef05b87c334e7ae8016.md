# --description--

正如你之前所见，函数可以使用 `return` 关键字返回一个值。`return` 关键字用于从函数返回一个值。当使用 `return` 关键字时，函数将停止执行并返回 `return` 关键字后指定的值。

```js
function add(a, b) {
  return a + b
}

console.log(add(2, 3)); // Output: 5
```

但是如果在函数结束之前使用了 `return` 关键字，会发生什么？  
为了解答这个问题，请考虑以下示例：

```js
function add(a, b) {
  if(a > 2){
    return b;
  }

  return a + b;
}

console.log(add(3, 7)); // Output: 7
``` 

在上面的示例中，函数 `add` 有一个条件式语句，用于查看 `a` 的值是否大于 `2`。如果满足条件，函数将返回 `b` 的值并停止执行。如果不满足条件，函数将返回 `a` 和 `b` 的和。
