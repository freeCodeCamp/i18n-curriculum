# --description--

十进制数系中的三位数 376 就是一个具有特殊性质的例子，其平方以相同的数字结尾：${376}^2 = 141376$。 让我们将具有此属性的数字称为稳定平方。

在其他编号系统中也可以看到稳定的正方形。 在 14 进制数系中，三位数$c37$也是一个稳定平方：$c37^2 = aa0c37$，其各位数字之和在相同数系中为$c+3+7=18$。 字母$a$、$b$、$c$和$d$分别代表 10、11、12 和 13，类似于十六进制数系。

对于$1 ≤ n ≤ 9$，14 进制数系中所有$n$位稳定平方的各位数字之和为$2d8$（十进制 582）。 不允许以 0 开头的稳定正方形。

求 14 进制数系中所有$1 ≤ n ≤ 10000$（十进制）位的稳定平方的各位数字之和，并将答案以 14 进制数系的字符串形式给出，必要时使用小写字母。
