# --description--

給定整數的值 $1 < a_1 < a_2 < \ldots < a_n$，考慮線性組合 $q_1a_1 + q_2a_2 + \ldots + q_na_n = b$，其中僅使用整數值 $q_k ≥ 0$。

請注意，對於給定的一組 $a_k$，可能並非所有的 $b$ 值都是可能的。例如，如果 $a_1 = 5$ 且 $a_2 = 7$，則不存在 $q_1 ≥ 0$ 和 $q_2 ≥ 0$ 使得 $b$ 可以是 1、2、3、4、6、8、9、11、13、16、18 或 23。

事實上，23 是在 $a_1 = 5$ 和 $a_2 = 7$ 時最大的無法達成的 $b$ 值。因此我們稱 $f(5, 7) = 23$。同樣地，可以證明 $f(6, 10, 15)=29$ 和 $f(14, 22, 77) = 195$。

求 $\sum f(pq,pr,qr)$，其中 $p$、$q$ 和 $r$ 是質數，且 $p &lt; q &lt; r &lt; 5000$。
