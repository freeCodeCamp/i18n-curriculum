# --explanation--

マークはソフトウェアのクラッシュの原因として`incompatible drivers`と`outdated software`を挙げています。

`Incompatible drivers`はハードウェアと正しく動作しないドライバーのことで、`outdated software`はソフトウェアが最新のバージョンに更新されていないことを意味します。どちらもクラッシュの原因になります。例えば：

- `The graphics card isn't working because of incompatible drivers.` - 間違ったドライバーがグラフィックカードの誤動作を引き起こしています。

- `Outdated software can lead to security vulnerabilities.` - 古いプログラムは更新がないため安全に使えない場合があります。
