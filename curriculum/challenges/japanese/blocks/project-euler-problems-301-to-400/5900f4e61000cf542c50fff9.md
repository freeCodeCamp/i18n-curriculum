# --description--

$T(n)$ を $n$ 番目の三角数とします。つまり $T(n) = \frac{n(n + 1)}{2}$ です。

$dT(n)$ を $T(n)$ の約数の個数とします。例：$T(7) = 28$ で $dT(7) = 6$ です。

$Tr(n)$ を、$1 ≤ i < j < k ≤ n$ を満たし、かつ $dT(i) > dT(j) > dT(k)$ となる三つ組 ($i$, $j$, $k$) の個数とします。$Tr(20) = 14$、$Tr(100) = 5\,772$、$Tr(1000) = 11\,174\,776$ です。

$Tr(60\,000\,000)$ を求めてください。答えの下18桁を答えてください。
