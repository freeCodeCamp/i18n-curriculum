# --instructions--

このチャレンジでは、Pythonを使ってデータグラフエクスプローラーを作成し、以下のことを行います。

- .csvファイルを3つの方法で取得する
  - ローカルコンピューターからアップロードする
  - ユーザーの入力からURLを取得する
  - コード内にURLを直接記述する
- Pandasライブラリを使って.csvをデータフレームとして保存する
- 見出しと最初の2行を出力する
- 列名をリストとして保存する
- 1つまたは2つの列を選び、データをNumpy配列に変換する
- 散布図または折れ線グラフとしてデータを表示する
- 異なる列の組み合わせでこれらを実行し、グラフを解釈できるようにする

完了したら、Googleドライブ上の公開ColabノートブックのURLを提出してください。
