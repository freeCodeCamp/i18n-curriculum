# --description--

Nella lezione precedente, hai imparato come eseguire script Python in locale. Ma ci possono essere momenti in cui non vuoi creare programmi Python completi e ti serve solo testare un po’ di codice Python. Qui entra in gioco la shell interattiva di Python.

Ma prima, rivediamo cos’è un terminale.

Un terminale è un’interfaccia testuale che ti permette di interagire con il computer digitando comandi. Ogni sistema operativo ha un’app terminale predefinita. Su macOS puoi usare l’app Terminal. Su Windows puoi usare Command Prompt o PowerShell. Su Linux, ogni ambiente desktop ha la sua app terminale predefinita, come GNOME Terminal o Konsole.

Apri un’app terminale e digita `python` poi premi `Enter`. Questo avvierà una shell interattiva di Python. Una shell interattiva è un programma che ti permette di digitare comandi uno alla volta e vedere i risultati.

Quando inizi una nuova sessione, potresti vedere inizialmente questo tipo di output:

```bash
Python 3.12.2 (main, Mar 21 2024, 22:48:26) [Clang 14.0.3 (clang-1403.0.22.14.1)] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>> 
```

Il simbolo `>>>` significa che Python sta aspettando che tu digiti un comando. Prova a stampare `"Hello, world!"` nel terminale:

```bash
print("Hello, world!")
```

Dovresti vedere il testo apparire così:

```bash
>>> print("Hello, world!")
Hello, world!
```

Dopo che il testo è stato stampato, Python torna a questo:

```bash
>>>
```

Questo significa che puoi digitare un altro comando.

L’interprete Python segue quello che si chiama ciclo Read, Evaluate, Print, Loop. O REPL per abbreviare. Ogni volta che digiti comandi, l’interprete legge l’input, lo valuta, stampa il risultato e torna indietro per mostrare il `>>>` così puoi digitare altri comandi.

Cosa succede se provi a digitare un comando non valido come questo?

```bash
>>> something random
```

Beh, l’interprete Python seguirà comunque lo stesso processo REPL. In questo caso, riceverai un messaggio di errore:

```bash
>>> something random
  File "<stdin>", line 1
    something random
              ^^^^^^
SyntaxError: invalid syntax
```

Se vuoi uscire dalla shell interattiva puoi digitare `exit()` oppure premere `Ctrl + D` su Mac/Linux o `Ctrl + Z + Enter` su Windows.

Usare la shell interattiva di Python è ottimo per piccoli esperimenti con il codice o per esplorazioni di base. Mentre segui le lezioni rimanenti del corso, puoi digitare alcuni dei nuovi metodi che impari dentro una shell interattiva.

Se invece prevedi di lavorare con programmi più lunghi o su più file, è meglio usare un editor di codice o un IDE.
