# --interactive--

Es gibt verschiedene Zustände eines Links, darunter `link`, `visited`, `hover`, `focus` und `active`. Diese Zustände sind wichtig, um Nutzern zu helfen, Links zu erkennen und nach Interaktionen klares Feedback zu geben, was sowohl die Benutzerfreundlichkeit als auch die Barrierefreiheit verbessert.

Das Stylen dieser verschiedenen Link-Zustände ist entscheidend für die Benutzerfreundlichkeit und Barrierefreiheit, da es visuelle Hinweise über den aktuellen Zustand des Links liefert. Das hilft Nutzern zu verstehen, welche Links sie bereits besucht haben, mit welchem Link sie gerade interagieren und was passiert, wenn sie klicken.

Für Nutzer mit Sehbeeinträchtigungen oder kognitiven Einschränkungen können diese unterschiedlichen Stile die Navigation deutlich einfacher und intuitiver machen.

Außerdem verbessern klare Link-Zustände das gesamte Nutzererlebnis, indem sie sofortiges Feedback auf Nutzerinteraktionen geben, Verwirrung reduzieren und die Navigation auf der Seite erleichtern.

Diese Zustände können mit etwas gestylt werden, das in CSS `pseudo-classes` genannt wird.

Eine Pseudoklasse ist ein Schlüsselwort, das einem Selektor hinzugefügt wird und einen speziellen Zustand des ausgewählten Elements angibt.

Zum Beispiel kann `:hover` die Farbe eines Buttons ändern, wenn der Mauszeiger darüber schwebt, während `:visited` die Farbe eines Links ändert, der bereits besucht wurde.

Pseudoklassen ermöglichen es Ihnen, Elemente basierend auf ihrem Zustand oder der Interaktion des Nutzers mit ihnen zu stylen, ohne dass zusätzlicher Auszeichnungscode in Ihrem HTML nötig ist.

Die Syntax einer `pseudo-class` sieht ungefähr so aus, wobei `A` der Selektor und `:B` die `pseudo-class` ist:

```css
A:B {
  property: value;
}
```

Um besser zu verstehen, wie man die verschiedenen Link-Zustände stylt, schauen wir uns einige Beispiele an.

Die `:link`-Pseudoklasse stylt nicht besuchte Links und zeigt an, dass sie anklickbar sind.

Hier ist ein Beispiel, wie man ein Ankerelement anspricht und die `:link`-Pseudoklasse verwendet:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Normal state (unvisited link) */
a:link { 
  color: red;
}
```

:::

Das obige Beispiel ändert die Standardfarbe Blau des Links in `red`, wenn er nicht besucht wurde.

`:visited` stylt Links, die bereits besucht oder angeklickt wurden, und hilft Nutzern, nachzuverfolgen, welche Links sie schon angeklickt haben. Hier ein Beispiel für die Verwendung der `:visited`-Pseudoklasse:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="https://www.freecodecamp.org/learn/" target="_blank">freeCodeCamp</a>
```

```css
/* Visited link */
a:visited {
  color: green;
}
```

:::

Dieser Code färbt den Link in `green`, wenn er angeklickt wurde.

`:hover` ändert den Stil des Links, wenn der Nutzer mit der Maus darüber schwebt, und gibt so einen visuellen Hinweis, dass der Link interaktiv ist. Hier ein Beispiel für die Verwendung der `:hover`-Pseudoklasse:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Hover state */
a:hover {
  color: green;
}
```

:::

Dieser Code färbt die Links in `green`, wenn die Maus darüber schwebt.

`:focus` fügt Stile um den Link hinzu, wenn er den Fokus hat, zum Beispiel bei der Navigation mit der Tastatur, und verbessert so die Barrierefreiheit.

Hier ein Beispiel, das die `outline`-Eigenschaft verwendet, um eine durchgezogene orangefarbene Umrandung anzuzeigen, wenn der Link den Fokus hat.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Focus state */
a:focus {
  outline: 2px solid orange;
}
```

:::

`:active` ändert die Link-Stile, während der Link angeklickt wird, und gibt dem Nutzer sofortiges Feedback, dass seine Aktion registriert wird. Hier ein Beispiel für die Verwendung der `:active`-Pseudoklasse:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Active state */
a:active {
  color: pink;
}
```

:::

Dieses Codebeispiel färbt den Link sofort in `pink`, wenn er angeklickt wird.
