# --description--

L’espressione `only if` viene usata per impostare una condizione affinché qualcosa accada. È come dire "questo succederà se quello succede". È un modo per fare un accordo in cui un’azione dipende da un’altra.

Per esempio, il tuo amico potrebbe dire `I'll go out only if it doesn't rain`. Significa che uscirà ma il tempo deve essere bello — niente pioggia.
