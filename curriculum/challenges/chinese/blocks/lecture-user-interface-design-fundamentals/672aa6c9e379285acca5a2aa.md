# --interactive--

**注意**：某些交互示例可能使用你尚未学习的 CSS。不要担心试图理解所有的代码。示例的目的是向你展示这些设计概念的预览，以便你更好地理解其工作原理。

作为开发人员，你可能需要与设计师密切合作。 如果你了解基本的设计术语，你们就会有一个共同的词汇表，你就能更积极地参与设计过程。 这可以提高工作流程的效率，带来更好的用户体验。

那么，让我们开始吧。 我们先从布局开始。 布局是指如何在页面或屏幕上安排视觉元素以传达信息。 这些元素可能包括文字、图像和留白。 布局就像是设计的蓝图。 设计师必须考虑每个元素的位置、大小和层次。

与布局密切相关的术语是对齐。 对齐是指元素之间的位置关系。 正确使用对齐方式有助于使设计看起来整洁有序。 设计师通过沿着假想的线条、边缘或中心点排列元素来创造视觉秩序。

很好。 现在我们来谈谈构图。 构图是一种排列元素以创造和谐设计的艺术。 它决定了图像、文字和形状等元素如何相互关联，并以艺术的方式为设计做出贡献。 布局主要侧重于元素的摆放，而构图还要考虑元素的摆放在整体设计中产生的艺术效果。

构图与平衡概念密切相关。 平衡是指视觉权重在构图中的分布。 设计师的目标是通过对称或不对称的安排来创造平衡。 平衡的设计让人感觉和谐。

:::interactive_editor

另一个有用的术语是留白。 留白，又称 "负空间"，是指设计中的空白空间。 这是元素周围的区域。 你可能会惊讶地发现，留白不一定是白色的。 事实上，它可以是任何颜色或纹理的空间。 其目的是提高设计的可读性和视觉层次感。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

</div>
```

这些都是你会经常发现的一般设计概念，但你也可能会发现与软件开发更密切相关的设计术语。

用户界面，又称 UI，是人类与计算机进行交互的方式。 用户界面包括用户可以在屏幕上看到的视觉和交互元素，如图标、图像、文本、菜单、链接和按钮。

用户体验，又称UX，是指用户在使用产品或服务时的感受。 一个具有精心设计的用户体验的应用程序是直观的、易于使用的、高效的、可访问的和令人愉悦的。 用户界面在使用户体验尽可能轻松愉悦方面发挥着关键作用，因此二者关系密切。

这些都是你在与设计师交流时应该知道的一些常用术语。 有了这些知识，你现在可以在设计过程中发挥更积极的作用。
