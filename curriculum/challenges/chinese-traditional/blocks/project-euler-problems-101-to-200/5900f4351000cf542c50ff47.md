# --description--

我們將定義 sqube 為形如 ${p^2}{q^3}$ 的數字，其中 $p$ 和 $q$ 是不同的質數。

例如，$200 = {5^2}{2^3}$ 或 $120072949 = {{23}^2}{{61}^3}$。

前五個 squbes 是 72、108、200、392 和 500。

有趣的是，200 也是第一個無法透過更改任何單一數字來變成質數的數字；我們將這類數字稱為質數防護數。下一個包含連續子字串 `200` 的質數防護平方立方數是 1992008。

尋找包含連續子字串 `200` 的第 200 個質數證明的 sqube。
