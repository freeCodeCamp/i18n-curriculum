# --description--

一個遊戲由三堆石頭和兩個玩家進行。

在每位玩家的回合中，玩家從堆中移除一顆或多顆石頭。然而，如果玩家從多於一個堆中取石頭，則必須從每個被選中的堆中移除相同數量的石頭。

換句話說，玩家選擇某個 $N > 0$ 並移除：

- 從任何單一堆中取出 $N$ 顆石頭；或
- 從任意兩堆中各取 $N$ 顆石頭（共 $2N$ 顆）；或者
- 每個三堆中各有 $N$ 顆石頭（共 $3N$ 顆）。

拿最後一顆石頭的玩家贏得遊戲。

一個獲勝的設定是指第一位玩家能夠強制獲勝的情況。

例如，(0,0,13)、(0,11,11) 和 (5,5,5) 是獲勝的組態，因為第一個玩家可以立即移除所有石頭。

輸掉的設定是指第二位玩家無論第一位玩家做什麼，都能強制獲勝的情況。

例如，(0,1,2) 和 (1,3,3) 是輸的設定，任何合法的移動都會讓第二位玩家取得贏的設定。

考慮所有輸掉的組態 (`$x_i$`、`$y_i$`、`$z_i$`) ，其中 `$x_i ≤ y_i ≤ z_i ≤ 100`。我們可以驗證這些組態的總和為 `$\sum (x_i + y_i + z_i) = 173\\,895$`。

求 $\sum (x_i + y_i + z_i)$，其中 ($x_i$,$y_i$,$z_i$) 在 $x_i ≤ y_i ≤ z_i ≤ 1000$ 的輸掉組態中取值。
