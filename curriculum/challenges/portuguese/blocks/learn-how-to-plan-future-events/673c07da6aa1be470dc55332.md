# --explanation--

`On call` significa estar disponível online para fornecer ajuda ou suporte se necessário.

Esta frase usa a estrutura `Will + subject + verb` para perguntar sobre um evento planejado no futuro.Essa estrutura é comum em perguntas formais sobre arranjos futuros.Por exemplo:

- `Will the meeting start at 10 AM?` - Isso está perguntando se a reunião está agendada para começar naquele horário específico.

- `Will they join us for dinner?` - Isso está perguntando se eles têm planos de vir para o jantar.
