## Einführung in die Typografie

- **Definition**: Typografie ist die Kunst, die richtigen Schriftarten und das Format auszuwählen, um Text optisch ansprechend und gut lesbar zu machen. „Type“ bezieht sich darauf, wie die einzelnen Zeichen gestaltet und angeordnet sind.
- **Definition der Schriftart**: Eine Schriftart ist das Gesamtdesign und der Stil einer Gruppe von Zeichen, Zahlen und Symbolen. Sie ist wie ein Bauplan für eine Schriftfamilie.
- **Definition der Schrift**: Eine Schrift ist eine spezifische Variante einer Schriftart mit bestimmten Merkmalen wie Größe, Gewicht, Stil und Breite.
- **Serifen-Schriftart**: Diese Schriftart hat einen klassischen Stil mit kleinen Linien am Ende der Zeichen. Serifen-Schriftarten werden häufig für gedruckte Materialien wie Bücher verwendet.
- **Sans-Serif-Schriftart**: Diese Schriftart wirkt moderner, da sie keine kleinen Linien am Ende der Zeichen hat. Sans-Serif-Schriftarten werden oft im digitalen Design verwendet, weil sie auf Bildschirmen gut lesbar sind. Beispiele sind Helvetica, Arial und Roboto.
- **Schriftgewicht**: Dies ist die Dicke der Zeichen, einschließlich light, regular, bold und black.
- **Schriftstil**: Dies ist die Neigung und Ausrichtung der Zeichen, wie italic und oblique.
- **Schriftbreite**: Dies ist der horizontale Platz, den Zeichen einnehmen, wie condensed und expanded.
- **Grundlinie**: Dies ist die imaginäre Linie, auf der die meisten Zeichen ruhen.
- **Versalhöhe**: Dies ist die Höhe der Großbuchstaben, gemessen von der Grundlinie bis zur Oberkante.
- **x-Höhe**: Dies ist die durchschnittliche Höhe der Kleinbuchstaben, ohne Auf- und Abstriche.
- **Aufstriche**: Das sind die Teile von Kleinbuchstaben, die über die x-Höhe hinausragen, wie die Oberteile der Buchstaben „h“, „b“, „d“ und „f“.
- **Abstriche**: Das sind die Teile von Kleinbuchstaben, die unter die Grundlinie reichen, wie die Schwänze von „y“, „g“, „p“ und „q“.
- **Kerning**: Dies ist die Anpassung des Abstands zwischen bestimmten Zeichenpaaren, um die Lesbarkeit und Ästhetik zu verbessern. Zum Beispiel wird der Abstand zwischen den Buchstaben A und V verringert.
- **Tracking**: Dies ist die Anpassung des Abstands zwischen allen Zeichen in einem Textblock. Es bestimmt, wie dicht oder luftig der Text wirkt.
- **Zeilenabstand (Leading)**: Dies ist der vertikale Abstand zwischen Textzeilen. Er wird von der Grundlinie einer Zeile bis zur Grundlinie der nächsten gemessen.
- **Bewährte Vorgehensweisen bei der Typografie**: Sie sollten klare und einfache Schriftarten wählen, damit Ihre Designs leicht verständlich sind. Das ist besonders wichtig für den Haupttext Ihrer Website. Nutzer beschäftigen sich eher mit Ihrem Inhalt, wenn die Schrift gut lesbar ist. Verwenden Sie höchstens zwei oder drei Schriftarten, um eine visuelle Konsistenz zu schaffen. Zu viele Schriftarten erschweren das Lesen und schwächen die Identität Ihrer Marke. Außerdem kann das die Nutzererfahrung beeinträchtigen, da die Ladezeit der Website steigt. Sie können die Schriftgröße nutzen, um eine visuelle Hierarchie für Überschriften, Unterüberschriften, Absätze und andere Elemente zu schaffen. Zum Beispiel sollte die Hauptüberschrift auf einer Webseite eine größere Schrift haben, gefolgt von Unterüberschriften mit kleineren Schriftgrößen. So erhält jedes Element in der Hierarchie eine spezifische Schriftgröße, die den Nutzern hilft, die Struktur visuell zu erfassen.

  
## CSS `font-family` Eigenschaft

- **Definition**: Eine Schriftfamilie ist eine Gruppe von Schriften, die ein gemeinsames Design teilen. Alle Schriften derselben Familie basieren auf derselben Grundschriftart, haben aber Variationen in Stil, Gewicht und Breite. Sie können mehrere Schriftfamilien in der Reihenfolge ihrer Priorität angeben, von der höchsten bis zur niedrigsten, getrennt durch Kommas. Diese alternativen Schriften dienen als Fallback-Optionen. Am Ende der Liste sollten Sie immer eine generische Schriftfamilie angeben.

```css
font-family: Arial, Lato;
```

- **Gängige Schriftfamilien**: Hier sind einige gängige Schriftfamilien, die in CSS verwendet werden: serif, sans-serif, monospace, cursive, fantasy 

## Web-sichere Schriften

- **Definition**: Diese Schriften sind eine Teilmenge von Schriften, die sehr wahrscheinlich auf einem Computer oder Gerät installiert sind. Wenn der Browser eine Schrift rendern muss, versucht er, die Schriftdatei auf dem System des Nutzers zu finden. Wird die Schrift nicht gefunden, greift er normalerweise auf eine Standardsystemschrift zurück.

- **Web-sichere Schriften**:
  - Sans-Serif-Schriften werden häufig für Webentwicklung verwendet, weil sie keine kleinen „Füßchen“ oder Linien am Ende der Zeichen haben und daher auf Bildschirmen gut lesbar sind. Beispiele für web-sichere Sans-Serif-Schriften sind: Arial, Verdana und Trebuchet MS.
  - Im Gegensatz dazu haben Serifen-Schriften kleine „Füßchen“ am Ende der Zeichen und werden daher oft für traditionelle Druckerzeugnisse verwendet. Web-sichere Serifen-Schriften sind: Times New Roman und Georgia.

## At-Regeln und die `@font-face` At-Regel

- **Definition**: At-Regeln sind Anweisungen, die dem Browser bestimmte Vorgaben machen. Sie können damit verschiedene Aspekte des Stylesheets definieren, wie Media Queries, Keyframes, Font Faces und mehr.  
- **`@font-face`**: Damit können Sie eine benutzerdefinierte Schrift definieren, indem Sie die Schriftdatei, das Format und andere wichtige Eigenschaften wie Gewicht und Stil angeben. Damit die `@font-face` At-Regel gültig ist, müssen Sie auch die `src` Eigenschaft angeben. Diese Eigenschaft enthält Referenzen zu den Schriftressourcen.

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2"),
       url("path/to/font.woff"),
       url("path/to/font.otf");
}
```

- **Schriftformate**: Für jede Schriftressource können Sie optional auch das Format angeben. Das ist ein Hinweis für den Browser zum Schriftformat. Wird das Format weggelassen, wird die Ressource heruntergeladen und das Format danach erkannt. Ist das Format ungültig, wird die Ressource nicht heruntergeladen. Mögliche Schriftformate sind `collection`, `embedded-opentype`, `opentype`, `svg`, `truetype`, `woff` und `woff2`.

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
       url("path/to/font.otf") format("opentype"),
       url("path/to/font.woff") format("woff");
}
```

- **`woff` und `woff2`**: `woff` steht für „Web Open Font Format“. Es ist ein weit verbreitetes Schriftformat, das von Mozilla in Zusammenarbeit mit Type Supply, LettError und anderen Organisationen entwickelt wurde. Der Unterschied zwischen `woff` und `woff2` liegt im Algorithmus zur Datenkompression.
- **OpenType**: Dies ist ein Format für skalierbare Computerschriften, das von Microsoft und Adobe entwickelt wurde und es Nutzern ermöglicht, zusätzliche Funktionen einer Schrift zu nutzen. Es ist auf den wichtigsten Betriebssystemen weit verbreitet.
- **`tech()`**: Dies wird verwendet, um die Technologie der Schriftressource anzugeben. Dies ist optional.

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
       url("path/to/font.otf") format("opentype") tech(color-COLRv1),
       url("path/to/font.woff") format("woff");
}
```

## Arbeiten mit externen Schriften

- **Definition**: Eine externe Schrift ist eine Schriftdatei, die nicht direkt in Ihren Projektdateien enthalten ist. Sie wird meist auf einem separaten Server gehostet. Um diese externen Schriften in Ihr Projekt einzubinden, können Sie ein `link`-Element oder eine `@import`-Anweisung in Ihrem CSS verwenden. 
- **Google Fonts**: Dies ist ein Google-Dienst, der eine Sammlung von Schriften anbietet, von denen viele speziell für die Webentwicklung entworfen wurden.

Hier ein Beispiel mit dem `link`-Element:

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap" rel="stylesheet">
```

```css
.roboto-thin {
  font-family: "Roboto", sans-serif;
  font-weight: 100;
  font-style: normal;
}
```

Hier ein Beispiel mit der `@import`-Anweisung:

```css
@import url('https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap');
```

- **Font Squirrel**: Dies ist eine weitere beliebte Ressource, die Sie nutzen können, um benutzerdefinierte externe Schriften in Ihre Projekte einzubinden.

## `text-shadow` Eigenschaft

- **Definition**: Diese Eigenschaft wird verwendet, um Schatten auf Text anzuwenden. Sie müssen den X- und Y-Versatz angeben, die den horizontalen und vertikalen Abstand des Schattens vom Text darstellen. Diese Werte sind erforderlich. Positive Werte für X- und Y-Versatz verschieben den Schatten nach rechts bzw. unten, negative Werte nach links bzw. oben.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Typography Shadow</p>
```

```css
  p {
    text-shadow: 3px 2px;
  }
```

:::

- **Schattenfarbe**: Sie können die Farbe des Schattens auch anpassen, indem Sie diesen Wert vor oder nach dem Versatz angeben. Wenn keine Farbe angegeben wird, bestimmt der Browser die Farbe automatisch, daher ist es meist besser, sie explizit zu setzen.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Colored Shadow</p>
```

```css
p {
  text-shadow: 3px 2px #00ffc3;
}
```

:::

- **Unschärferadius**: Der Unschärferadius ist optional, macht den Schatten aber viel weicher und subtiler. Der Standardwert des Unschärferadius ist null. Je höher der Wert, desto größer die Unschärfe, was bedeutet, dass der Schatten heller wird.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Blurred Shadow</p>
```

```css
p {
  text-shadow: 3px 2px 3px #00ffc3;
}
```

:::

- **Mehrere Textschatten anwenden**: Ein Text kann mehr als einen Schatten haben. Die Schatten werden in Schichten angewendet, von vorne nach hinten, wobei der erste Schatten oben liegt.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Multiple Shadows</p>
```

```css
p {
  text-shadow:
    3px 2px 3px #00ffc3,
    -3px -2px 3px #0077ff,
    5px 4px 3px #dee7e5;
}
```

:::
