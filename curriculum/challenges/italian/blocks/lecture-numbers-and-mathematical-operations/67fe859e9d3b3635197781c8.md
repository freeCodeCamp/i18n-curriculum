# --description--

I numeri interi e i numeri a virgola mobile sono i principali tipi di dati numerici in Python. Con essi puoi archiviare dati numerici ed eseguire operazioni matematiche.

Vediamo cosa sono i numeri interi e i numeri a virgola mobile, come eseguire calcoli aritmetici con essi e alcuni metodi che Python offre per lavorare con entrambi.

I numeri interi sono numeri interi senza punti decimali, positivi o negativi:

```python
my_int_1 = 56
my_int_2 = -4

print(type(my_int_1)) # <class 'int'>
print(type(my_int_2)) # <class 'int'>
```

Ecco come eseguire un’operazione di addizione con numeri interi:

```python
my_int_1 = 56
my_int_2 = 12

sum_ints = my_int_1 + my_int_2
print('Integer Addition:', sum_ints) # Integer Addition: 68
```

Ecco come eseguire una sottrazione con numeri interi:

```python
my_int_1 = 56
my_int_2 = 12

# Subtraction
diff_ints = my_int_1 - my_int_2
print('Integer Subtraction:', diff_ints) # Integer Subtraction: 44
```

Ecco come eseguire un’operazione di moltiplicazione con numeri interi:

```python
my_int_1 = 12
my_int_2 = 4

# Multiplication
product_ints = my_int_1 * my_int_2
print('Integer Multiplication:', product_ints) # Integer Multiplication: 48
```

Ecco come eseguire un’operazione di divisione con numeri interi:

```python
my_int_1 = 56
my_int_2 = 12

# Division
div_ints = my_int_1 / my_int_2
print('Integer Division:', div_ints) # Integer Division: 4.666666666666667
```

I numeri a virgola mobile sono numeri positivi o negativi con punti decimali, come `3.14`, `-0.5` o `0.0`.

```python
my_float_1 = -12.0
my_float_2 = 4.9

print(type(my_float_1)) # <class 'float'>
print(type(my_float_2)) # <class 'float'>
```

Ecco un’operazione di addizione con numeri a virgola mobile:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_addition = my_float_1 + my_float_2
print('Float Addition:', float_addition) # Float Addition: 17.4
```

Ecco un’operazione di sottrazione con numeri a virgola mobile:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_subtraction = my_float_2 - my_float_1
print('Float Subtraction:', float_subtraction) # Float Subtraction: 6.6
```

Ecco un’operazione di moltiplicazione con numeri a virgola mobile:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_multiplication = my_float_2 * my_float_1
print('Float Multiplication:', float_multiplication) # Float Multiplication: 64.80000000000001
```

Ecco un’operazione di divisione con numeri a virgola mobile:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_division = my_float_2 / my_float_1
print('Float Division:', float_division) # Float Division: 2.222222222222222
```

Se aggiungi un numero intero e un numero a virgola mobile, il risultato viene automaticamente convertito in un numero a virgola mobile:

```python
my_int = 56
my_float = 5.4

sum_int_and_float = my_int + my_float

print(sum_int_and_float) # 61.4
print(type(sum_int_and_float)) # <class 'float'>
```

Lo stesso vale per altre operazioni aritmetiche di base, come sottrazione, moltiplicazione e divisione. Se mescoli numeri interi e numeri a virgola mobile, Python restituirà un numero a virgola mobile come risultato.

Puoi anche eseguire calcoli aritmetici più complessi come ottenere il resto di due numeri con l’operatore modulo, la divisione intera e l’elevamento a potenza con entrambi i tipi.

L’operatore modulo (`%`) restituisce il resto quando il valore a sinistra viene diviso per il valore a destra:

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

mod_ints = my_int_1 % my_int_2
mod_floats = my_float_2 % my_float_1

print('Integer Modulo:', mod_ints) # Integer Modulo: 8
print('Float Modulo:', mod_floats) # Float Modulo: 1.1999999999999993
```

La divisione intera divide due numeri e restituisce il numero intero più grande minore o uguale al risultato. Questo si fa con l’operatore doppio slash (`//`):

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

floor_div_ints = my_int_1 // my_int_2
floor_div_floats = my_float_2 // my_float_1

print('Integer Floor Division:', floor_div_ints) # Integer Floor Division: 4
print('Float Floor Division:', floor_div_floats) # Float Floor Division: 2.0
```

L’elevamento a potenza eleva un numero alla potenza di un altro, e si fa con l’operatore doppio asterisco (`**`):

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

exp_ints = my_int_1 ** my_int_2
exp_floats = my_float_1 ** my_float_2

print('Integer Exponentiation:', exp_ints) # Integer Exponentiation: 951166013805414055936
print('Float Exponentiation:',  exp_floats) # Float Exponentiation: 614787626.1765089
```

A volte potresti notare che il risultato di un’operazione che coinvolge numeri a virgola mobile ha più cifre decimali del previsto. Per esempio, la somma `0.1 + 0.2` è uguale a `0.30000000000000004` invece che a `0.3`.

Questo succede perché i numeri sono archiviati in formato binario, e alcune frazioni non possono essere rappresentate esattamente in binario. Di conseguenza, vengono archiviate come approssimazioni finite, nello stesso modo in cui la frazione `1/3` non può essere rappresentata con un numero finito di cifre decimali ed è troncata dopo un certo numero delle sue infinite cifre (`0.33333...`).

Questo porta a piccoli errori di arrotondamento.

Python fornisce anche funzioni integrate per convertire dati numerici o stringhe in numeri interi o a virgola mobile.

La funzione `float()` restituisce un numero a virgola mobile costruito dal numero dato:

```python
my_int_1 = 56
my_float_1 = float(my_int_1)

print(my_float_1)  # 56.0
print(type(my_float_1))  # <class 'float'>
```

La funzione `int()` restituisce un numero intero costruito dal numero dato:

```python
my_float = 12.92563
my_int = int(my_float)

print(my_int)  # 12
print(type(my_int))  # <class 'int'>
```

Inoltre, puoi usare le stesse funzioni integrate per convertire una stringa in un numero a virgola mobile o intero:

```python
my_str_int = '45'
my_str_float = '7.8'

converted_int = int(my_str_int)
converted_float = float(my_str_float)

print(converted_int, type(converted_int))  # 45 <class 'int'>
print(converted_float, type(converted_float))  # 7.8 <class 'float'>
```

Ecco alcuni altri metodi che Python offre per lavorare con numeri interi e numeri a virgola mobile.

- `round()`: arrotonda un numero al numero specificato di cifre decimali. Per impostazione predefinita questa funzione arrotonda al numero intero più vicino e restituisce un numero intero senza cifre decimali:
    

```python
my_int_1 = 4.798
my_int_2 = 4.253

rounded_int_1 = round(my_int_1)
rounded_int_2 = round(my_int_2, 1)

print(rounded_int_1) # 5
print(rounded_int_2) # 4.3
```

- `abs()`: restituisce il valore assoluto di un numero,
    

```python
num = -15

absolute_value = abs(num)
print(absolute_value) # 15
```

- `pow()`: eleva un numero alla potenza di un altro o esegue l’esponenziazione modulare.
    

```python
result_1 = pow(2, 3)  # Equivalent to 2 ** 3
print(result_1)  # 8

result_2 = pow(2, 3, 5)  # (2 ** 3) % 5
print(result_2)  # 3
```
