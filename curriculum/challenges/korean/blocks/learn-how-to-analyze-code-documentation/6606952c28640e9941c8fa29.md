# --description--

보통 `with`는 연결이나 사용을 나타낼 때 쓰입니다. 예를 들어, `I write with a pen.` 그러나 어떤 문장에서는 `with`가 사용된 단어 뒤에 옵니다. 이는 주로 누군가가 사용하거나 다루는 대상이나 사물을 강조하고 싶을 때, 특히 기술이나 업무에 관한 질문이나 진술에서 발생합니다.

예를 들어, 기술 분야에서는 `This program I work with is challenging.`라고 말할 수 있습니다. 여기서 `with`는 작업하기 어려운 프로그램을 강조하기 위해 문장 끝에 사용됩니다. 또 다른 예로는 `What else is he working with?`와 같은 질문이 있을 수 있는데, 여기서 `with`는 사용 중인 다른 도구나 기술에 대해 묻기 위해 문장 끝에 옵니다.
