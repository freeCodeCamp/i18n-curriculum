# --description--

`Configuration` nel software e nella programmazione indica la configurazione o la disposizione di un sistema o dei componenti di un'applicazione. Spesso riguarda le impostazioni che determinano come un programma funziona.

Per esempio, modificare il `configuration` di un software può regolare come esso si comporta o interagisce con altri sistemi.
