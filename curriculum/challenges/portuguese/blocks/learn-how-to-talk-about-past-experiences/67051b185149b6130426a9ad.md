# --explanation--

No `Simple Past`, perguntas podem começar com uma palavra interrogativa (por exemplo, `what`, `where`, `how`, `who`) usando a seguinte estrutura:

**Palavra interrogativa + `did` + sujeito + verbo base.**

A pergunta de Lisa, `What did you find there?`, usa `what` para pedir informações específicas sobre o resultado das ações de James.Outros exemplos incluem:

- `Where did you go yesterday?` - Você está perguntando sobre um local.

- `How did they solve the problem?` - Você está perguntando sobre um método.

- `Who did you call last night?` - Você está perguntando sobre uma pessoa.

No diálogo, Lisa está perguntando sobre a descoberta de James enquanto ele procurava pelo bug.
