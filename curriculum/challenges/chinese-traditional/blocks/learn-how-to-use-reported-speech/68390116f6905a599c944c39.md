# --explanation--

`If only` 用於句首，表達對過去未發生事情的願望或遺憾。它表示說話者希望情況本來不同。 例如：

`If only we had checked the system earlier, we could have avoided the crash.`－這表示他們沒有及早檢查，現在感到後悔。
