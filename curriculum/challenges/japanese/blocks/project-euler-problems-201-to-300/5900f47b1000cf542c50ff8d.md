# --description--

正の数 $n$ に対して、$1 < x < n$ かつ $x^3 \equiv 1\bmod n$ を満たす整数 $x$ の和を $S(n)$ と定義します。

$n = 91$ の場合、$x$ の取り得る値は 9、16、22、29、53、74、79、81 の8つです。したがって、$S(91) = 9 + 16 + 22 + 29 + 53 + 74 + 79 + 81 = 363$ となります。

$S(13\\,082\\,761\\,331\\,670\\,030)$ を求めてください。
