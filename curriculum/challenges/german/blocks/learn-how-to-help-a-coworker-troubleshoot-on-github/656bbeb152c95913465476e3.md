# --description--

`Compile` bedeutet in der Programmierung, Code, den Menschen schreiben, in eine Form zu verwandeln, die Computer verstehen und verwenden können.

Es ist wie ein Rezept, das in einer Sprache geschrieben ist, in eine andere Sprache zu übersetzen, damit jemand anderes das Gericht zubereiten kann.

Zum Beispiel `When we compile Java code, it changes into a form that the computer can run.`
