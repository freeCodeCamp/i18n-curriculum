# --description--

10進数の3桁の数376は、その平方の末尾が同じ数字で終わるという特別な性質を持つ数の例です：${376}^2 = 141376$。この性質を持つ数をsteady square（定常平方数）と呼びます。

steady squareは他の進数体系でも見られます。14進数体系では、3桁の数$c37$もsteady squareです：$c37^2 = aa0c37$で、その桁の和は同じ進数体系で$c+3+7=18$となります。文字$a$、$b$、$c$、$d$はそれぞれ16進数と同様に10、11、12、13の数字を表します。

$1 ≤ n ≤ 9$の場合、14進数体系におけるすべての$n$桁のsteady squareの桁の和は$2d8$（10進数で582）です。先頭に0があるsteady squareは許されません。

$1 ≤ n ≤ 10000$（10進数）について、14進数体系におけるすべての$n$桁のsteady squareの桁の和を求め、必要に応じて小文字のアルファベットを使った14進数の文字列として答えてください。
