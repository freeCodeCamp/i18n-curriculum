# --description--

正の整数 $n$ の丸めた平方根を、$n$ の平方根を最も近い整数に丸めたものと定義します。

次の手順（整数演算に適応したヘロンの方法にほぼ相当します）で $n$ の丸めた平方根を求めます：

$n$ の桁数を $d$ とします。

$d$ が奇数の場合、$x_0 = 2 × {10}^{\frac{d - 1}{2}}$ と設定します。

$d$ が偶数の場合、$x_0 = 7 × {10}^{\frac{d - 2}{2}}$ と設定します。

次を繰り返します：

$$x_{k + 1} = \left\lfloor\frac{x_k + \left\lceil\frac{n}{x_k}\right\rceil}{2}\right\rfloor$$

ただし $x_{k + 1} = x_k$ になるまで続けます。

例として、$n = 4321$ の丸めた平方根を求めてみましょう。

$n$ は4桁なので、$x_0 = 7 × {10}^{\frac{4-2}{2}} = 70$ となります。

$$x_1 = \left\lfloor\frac{70 + \left\lceil\frac{4321}{70}\right\rceil}{2}\right\rfloor = 66 \\\\
x_2 = \left\lfloor\frac{66 + \left\lceil\frac{4321}{66}\right\rceil}{2}\right\rfloor = 66$$

$x_2 = x_1$ なのでここで終了します。わずか2回の反復で、4321 の丸めた平方根が 66（実際の平方根は 65.7343137…）であることがわかりました。

この方法で必要な反復回数は驚くほど少ないです。例えば、5桁の整数 ($10\,000 ≤ n ≤ 99\,999$) の丸めた平方根は平均で3.2102888889回の反復で求められます（平均値は小数点以下10桁に丸めています）。

上記の手順を使って、14桁の数 (${10}^{13} ≤ n < {10}^{14}$) の丸めた平方根を求めるのに必要な平均反復回数はいくつでしょうか？答えは小数点以下10桁に丸めてください。

**注意：** 記号 $⌊x⌋$ と $⌈x⌉$ はそれぞれ床関数と天井関数を表します。
