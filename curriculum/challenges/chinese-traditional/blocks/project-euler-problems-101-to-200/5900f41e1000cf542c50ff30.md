# --description--

設 ABCD 為一個凸四邊形，其對角線為 AC 和 BD。在每個頂點，對角線與兩條邊各形成一個角，產生八個角落角。

<img alt="convex quadrilateral ABCD, with diagonals AC and BD" src="https://cdn.freecodecamp.org/curriculum/project-euler/integer-angled-quadrilaterals.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

例如，在頂點 A，兩個角分別是 CAD、CAB。

我們將所有八個角度以度數測量皆為整數值的四邊形稱為「整數角四邊形」。整數角四邊形的一個例子是正方形，其所有八個角度皆為 45°。另一個例子為 DAC = 20°、BAC = 60°、ABD = 50°、CBD = 30°、BCA = 40°、DCA = 30°、CDB = 80°、ADB = 50°。

非相似整數角四邊形的總數是多少？

**注意：** 在你的計算中，如果計算出的角度在距離整數值 ${10}^{-9}$ 的容差範圍內，你可以假設該角度為整數。
