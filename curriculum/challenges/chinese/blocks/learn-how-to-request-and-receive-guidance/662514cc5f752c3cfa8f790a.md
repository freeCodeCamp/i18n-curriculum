# --description--

`时间复杂度` 是计算机科学中的一个术语。它描述了算法（一组指令）完成所需的时间，基于输入的大小。一个简单的例子是洗碗：洗一个碗很快，但洗很多碗需要更长时间。这就像编程中的时间复杂度。
