## CSS 오버플로우 속성

- **정의**: 오버플로우는 요소가 포함하는 영역 크기를 초과하는 콘텐츠를 처리하는 방식을 말합니다. 오버플로우는 2차원입니다.
- **`overflow-x`**: x축은 수평 오버플로우를 결정합니다.
- **`overflow-y`**: y축은 수직 오버플로우를 결정합니다.
- **`overflow`**: `overflow-x`와 `overflow-y`의 단축 속성입니다. 하나의 값이 주어지면 두 오버플로우 모두 그 값을 사용합니다. 두 값이 주어지면 `overflow-x`는 첫 번째 값을, `overflow-y`는 두 번째 값을 사용합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="overflow-box">
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
</div>
```

```css
.overflow-box {
  width: 140px;
  height: 70px;
  border: 2px solid black;
  overflow-x: auto;
  overflow-y: auto;
  white-space:nowrap;
}
```

## CSS 변형 속성

- **정의**: 이 속성은 요소에 회전, 크기 조정, 기울이기, 이동(2D 또는 3D 공간에서)과 같은 다양한 변형을 적용할 수 있게 합니다.
- **`translate()` 함수**: 이 함수는 요소를 현재 위치에서 이동시키는 데 사용됩니다.
- **`scale()` 함수**: 이 함수는 요소의 크기를 변경할 수 있게 합니다.
- **`rotate()` 함수**: 이 함수는 요소를 회전시킬 수 있게 합니다.
- **`skew()` 함수**: 이 함수는 요소를 기울일 수 있게 합니다.
- **변형과 접근성**: 변형을 사용해 콘텐츠를 숨기거나 표시할 때는 콘텐츠가 화면 읽기 프로그램과 키보드 탐색에서 여전히 접근 가능하도록 하세요. 숨긴 콘텐츠는 단순히 시각적으로 화면 밖으로 이동시키는 대신 `display: none` 또는 `visibility: hidden` 같은 방법으로 실제로 숨겨야 합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="transform-box translate">Translate</div>
  <div class="transform-box scale">Scale</div>
  <div class="transform-box rotate">Rotate</div>
  <div class="transform-box skew">Skew</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: column;
  align-items: center;
  gap: 16px;
  flex-wrap: wrap;
}

.transform-box {
  width: 100px;
  height: 100px;
  background: coral;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

/* translate() */
.translate {
  transform: translateX(20px);
}

/* scale() */
.scale {
  transform: scale(1.2);
}

/* rotate() */
.rotate {
  transform: rotate(15deg);
}

/* skew() */
.skew {
  transform: skew(10deg, 5deg);
}
```

## 박스 모델

- **정의**: CSS 박스 모델에서 모든 요소는 박스로 둘러싸여 있습니다. 이 박스는 콘텐츠 영역, `padding`, `border`, `margin` 네 가지 구성 요소로 이루어져 있습니다.
- **콘텐츠 영역**: 콘텐츠 영역은 박스의 가장 안쪽 부분입니다. 텍스트나 이미지 같은 실제 콘텐츠가 들어 있는 공간입니다.
- **`padding`**: 패딩은 콘텐츠 영역 바로 다음에 오는 영역입니다. 콘텐츠 영역과 요소의 테두리 사이의 공간입니다.
- **`border`**: 테두리는 CSS 박스 모델에서 요소의 바깥 가장자리 또는 윤곽선입니다. 요소의 시각적 경계입니다.
- **`margin`**: 여백은 요소의 테두리 바깥 공간입니다. 요소와 주변 다른 요소 사이의 거리를 결정합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="box-model">Content</div>
```

```css
.box-model {
  width: 120px;
  background: lightblue;
  padding: 16px;
  border: 4px solid teal;
  margin: 20px;
  text-align: center;
}
```

:::

## 여백 병합

- **정의**: 이 동작은 인접한 요소들의 수직 여백이 겹쳐서 두 여백 중 더 큰 값과 같은 하나의 여백으로 합쳐지는 현상입니다. 이 동작은 수직 여백(위쪽과 아래쪽)에만 적용되며, 수평 여백(왼쪽과 오른쪽)에는 적용되지 않습니다.

## `content-box`와 `border-box` 속성 값

- **`box-sizing` 속성**: 이 속성은 HTML 요소의 최종 `width`와 `height`가 어떻게 계산되는지 결정하는 데 사용됩니다.
- **`content-box` 값**: `content-box` 모델에서는 요소에 설정한 `width`와 `height`가 콘텐츠 영역의 크기를 결정하지만 `padding`, `border`, `margin`는 포함하지 않습니다.
- **`border-box` 값**: `border-box`에서는 요소의 `width`와 `height`가 콘텐츠 영역, `padding`, `border`를 포함하지만 `margin`는 포함하지 않습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="sizing-box content-box">Content-box</div>
<div class="sizing-box border-box">Border-box</div>
```

```css
.sizing-box {
  width: 150px;
  padding: 10px;
  border: 4px solid teal;
  margin-bottom: 10px;
  background: peachpuff;
  text-align: center;
}

.content-box {
  box-sizing: content-box;  
}

.border-box {
  box-sizing: border-box;
  background: lightgreen;
}
```

## CSS 리셋

- **정의**: CSS 리셋은 웹 브라우저가 HTML 요소에 기본적으로 적용하는 서식을 모두 또는 일부 제거하는 스타일시트입니다. CSS 리셋의 서드파티 옵션으로는 `sanitize.css`와 `normalize.css`가 있습니다.

## CSS 필터 속성

- **정의**: 이 속성은 블러, 색상 변화, 대비 조정 등 다양한 효과를 만드는 데 사용됩니다.
- **`blur()` 함수**: 이 함수는 요소에 가우시안 블러를 적용합니다. 양은 픽셀 단위로 정의되며 블러 반경을 나타냅니다.
- **`brightness()` 함수**: 이 함수는 요소의 밝기를 조절합니다. 0%는 요소를 완전히 검게 만들고, 100% 이상은 밝기를 증가시킵니다.
- **`contrast()` 함수**: 이 함수는 요소의 대비를 조절합니다. 0%는 요소를 완전히 회색으로 만들고, 100%는 정상적으로 보이게 하며, 100% 이상은 대비를 증가시킵니다.
- **`grayscale()` 함수**: 이 함수는 요소를 그레이스케일로 변환합니다. 양은 백분율로 정의되며 100%는 완전한 그레이스케일, 0%는 원본 이미지를 유지합니다.
- **`sepia()` 함수**: 이 함수는 요소에 세피아 톤을 적용합니다. 그레이스케일과 마찬가지로 백분율 값을 사용합니다.
- **`hue-rotate()` 함수**: 이 함수는 요소에 색상 회전을 적용합니다. 값은 도 단위로 정의되며 색상 원을 기준으로 회전하는 각도를 나타냅니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <img class="filter blur" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Blur" />
  <img class="filter brightness" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Brightness" />
  <img class="filter contrast" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Contrast" />
  <img class="filter grayscale" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Grayscale" />
  <img class="filter sepia" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Sepia" />
  <img class="filter hue" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Hue Rotate" />
</div>
```

```css
.container {
  display: flex;
  gap: 12px;
  flex-wrap: wrap;
}

.filter {
  width: 120px;
}

/* blur() */
.blur {
  filter: blur(3px);
}

/* brightness() */
.brightness {
  filter: brightness(130%);
}

/* contrast() */
.contrast {
  filter: contrast(150%);
}

/* grayscale() */
.grayscale {
  filter: grayscale(100%);
}

/* sepia() */
.sepia {
  filter: sepia(100%);
}

/* hue-rotate() */
.hue {
  filter: hue-rotate(90deg);
}
```
