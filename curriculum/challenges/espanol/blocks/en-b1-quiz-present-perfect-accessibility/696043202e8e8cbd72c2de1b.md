# --description--

Este cuestionario verifica tu comprensión del `Present Perfect` y `Present Perfect Continuous` hablando sobre características y mejoras de accesibilidad.

Para aprobar el cuestionario, debe responder correctamente al menos 18 de las 20 preguntas a continuación.

Lee cada pregunta y elige la respuesta correcta. Solo hay una respuesta correcta para cada pregunta.

Para accesibilidad, hay transcripciones disponibles para estudiantes con discapacidad auditiva; recomendamos usarlas solo si es necesario.
