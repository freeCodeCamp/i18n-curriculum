# --explanation--

**간접화법**에서는 누군가 이유를 설명하거나 명확히 할 때 `to explain`를 사용합니다. 예를 들어:

- 직접화법: `The server went down because of a power issue.`

- 간접화법: `She explained that the server went down due to a power issue.`

`system component`는 보통 컴퓨터나 네트워크에서 더 큰 시스템의 일부나 조각을 의미합니다. 예를 들어:

`The login module is a key system component for user access.` – 이것은 시스템의 필수적인 부분 중 하나임을 뜻합니다.
