# --description--

這裡是來自一個擁有一百萬使用者的繁忙電話系統的記錄：

| RecNr | Caller | Called |
|-------|--------|--------|
|   1   | 200007 | 100053 |
|   2   | 600183 | 500439 |
|   3   | 600863 | 701497 |
|  ...  |  ...   |  ...   |

記錄 $n$ 中的呼叫者電話號碼和被呼叫號碼分別為 $Caller(n) = S_{2n - 1}$ 和 $Called(n) = S_{2n}$，其中 ${S}_{1,2,3,\ldots}$ 來自「Lagged Fibonacci Generator」：

對於 $1 ≤ k ≤ 55$，$S_k = [100003 - 200003k + 300007{k}^3]\\;(\text{模除}\\;1000000)$

對於 $56 ≤ k$，$S_k = [S_{k - 24} + S_{k - 55}]\\;(\text{取模}\\;1000000)$

如果 $Caller(n) = Called(n)$，則假設使用者撥錯號碼且呼叫失敗；否則呼叫成功。

從記錄開始，我們說任何使用者對組 $X$ 和 $Y$ 是朋友，如果 $X$ 呼叫 $Y$ 或反之亦然。同樣地，如果 $X$ 是 $Y$ 的朋友，而 $Y$ 是 $Z$ 的朋友，則 $X$ 是 $Z$ 的朋友的朋友；對於更長的串鏈亦然。

總理的電話號碼是 524287。在不計算撥錯號碼的情況下，經過多少次成功的呼叫後，99％ 的使用者（包括總理）會成為總理的朋友，或是朋友的朋友等？
