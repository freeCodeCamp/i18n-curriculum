# --description--

세 번째 `li` 요소 안에 계속해서 팝업용 `ul` 요소를 만드세요. `className`의 `sub-menu`와 `aria-label`의 `Apps`를 지정하세요.

# --hints--

세 번째 `ul` 요소 안에 `li` 요소를 만들어야 합니다.

```js
assert.exists(document.querySelector('li ul'));
```

`className` 요소의 `ul` 속성을 `sub-menu`으로 설정해야 합니다.

```js
assert.equal(document.querySelector('li ul')?.getAttribute('class'), 'sub-menu');
```

`aria-label` 요소의 `ul` 속성을 `Apps`으로 설정해야 합니다.

```js
assert.equal(document.querySelector('li ul')?.getAttribute('aria-label'), 'Apps');
```
