# --interactive--

ウェブサイトのスタイリングで、色を扱う最も多用途で直感的な方法の一つがHSLカラーモデルの使用です。HSLはHue（色相）、Saturation（彩度）、Lightness（明度）の略で、色を定義する3つの重要な要素です。

HSLカラーモデルは、人間が色を認識する方法により近い形で色を表します。このレッスンでは、HSLカラーモデルとは何か、そしてCSSで`hsl()`関数がどのように使われるかを探ります。

色相は色の種類を表し、カラーホイール上の角度で表され、`0`度から`360`度までの範囲です。例えば、`0`度は赤、`120`度は緑、`240`度は青を表します。色相を調整することで、スペクトル上の異なる色に移動できます。

彩度は色の強さや純度を指し、`0%`（完全に彩度がなくグレースケールの色）から`100%`（その色の最も鮮やかな形）までのパーセンテージで測定されます。`100%`の彩度の色は完全に鮮やかで、`0%`の彩度の色は灰色のように見えます。

明度は色の明るさや暗さを決定し、これもパーセンテージで測定されます。明度が`0%`の場合は黒、`50%`は色相の通常のトーン、`100%`は白になります。

CSSでは、`hsl()`関数を使ってHSLカラーモデルで色を定義します。基本的な構文は以下の通りです。

```css
element {
  color: hsl(hue, saturation, lightness);
}
```

例を使って説明しましょう。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
body {
  background-color: hsla(0, 0%, 1%, 1.00);
}

p {
  color: hsl(120, 100%, 50%);
}
```

:::

この場合、色相は`120`度で緑に対応します。彩度は`100%`なので緑は完全に鮮やかです。明度は`50%`で、通常のトーンなので暗すぎず明るすぎません。その結果、段落のテキスト色は明るい緑になります。

HSLカラーモデルの主な利点の一つは、その直感的な性質です。彩度と明度の値を調整するだけで、色相（コアの色）を変えずに色の鮮やかさや明るさを簡単に調整できます。

例えば、同じ色の異なるシェードやティントを作りたい場合は、明度の値を調整するだけで済みます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="light">This is a light blue div.</div>
<div class="dark">This is a dark blue div.</div>
```

```css
div.light {
  background-color: hsl(240, 100%, 80%);
}

div.dark {
  background-color: hsl(240, 100%, 20%);
  color: hsl(0, 0%, 100%);
}
```

:::

ここでは、両方の`div`要素が同じ色相（`240`度、青）を使っていますが、一方は明度が`80%`（明るい青）、もう一方は明度が`20%`（暗い青）です。

RGBモデルに透明度を含める`rgba()`関数があるのと同様に、HSLモデルには`hsla()`関数があります。この関数の4番目のパラメータはアルファ値を表し、色の不透明度を制御します。基本的な構文は以下の通りです。

```css
element {
  background-color: hsla(hue, saturation, lightness, alpha);
}
```

例を見てみましょう。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

このコードは、色相が`div`度（赤）、彩度が`0`、明度が`100%`、アルファが`50%`（50%の不透明度）に設定された半透明の赤い背景を`0.5`に与えます。アルファはパーセンテージでも指定でき、例えば`hsla(0, 100%, 50%, 50%)`は`hsla(0, 100%, 50%, 0.5)`と同じです。

HSLカラーモデルは、カラースキームを作成したり、シェードやティントを簡単に調整したりする場合に特に便利です。

色相を彩度や明度から分離しているため、影やハイライトのために明度を調整しながら、一貫したカラーテーマを保つことができます。この柔軟性の高さが、デザイナーや開発者にHSLが好まれる理由です。

さらに、HSLは16進コードやRGBなどの他の方法と比べて色の値が読みやすく理解しやすいです。色の意図がより明確に見え、異なるデザインを試す際に調整しやすくなります。

HSLカラーモデルを使うことで、色調、彩度、明度のコントロールにより柔軟性が増し、CSSで豊かでダイナミックなデザインを作成する強力なツールになります。
