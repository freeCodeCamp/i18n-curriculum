# --description--

要使複選框輸入默認爲選中，可以添加 `checked` 屬性。

下面是使用 `checked` 屬性的示例：

```html
<input checked type="checkbox" id="checked" name="checked">
```

給 `id` 爲 `"reputation"` 的複選框輸入添加 `checked` 屬性，使其默認爲選中狀態。

# --hints--

你的 `id` 爲 `"reputation"` 的複選框輸入應具有 `checked` 屬性。

```js
assert.exists(document.querySelector("input#reputation[checked]"));
```
