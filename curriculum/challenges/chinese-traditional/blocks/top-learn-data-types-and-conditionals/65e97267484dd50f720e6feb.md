# --description--

你知道 JavaScript 有各種資料型別，包括字串、數字和布林值。你可以使用比較運算子來比較這些資料型別，並根據結果做出決策。

JavaScript 中可用的比較運算子如下：

| Operator | Description                              | Example      |
|----------|------------------------------------------|--------------|
| ==       | Equal (value comparison)                 | x == y       |
| !=       | Not equal                                | x != y       |
| >        | Greater than                             | x > y        |
| <        | Less than                                | x < y        |
| >=       | Greater than or equal to                 | x >= y       |
| <=       | Less than or equal to                    | x <= y       |

要在變數指派中使用比較，你可以使用 `==` 運算子。 例如：

```javascript
let x = 5;
let y = 5;

let result = x == y;

console.log(result); // true
```
