# --description--

El código Gray es una forma de codificación binaria donde las transiciones entre números consecutivos difieren en solo un bit.

Esta es una codificación útil para reducir los riesgos de datos en hardware con valores que cambian rápidamente y/o se conectan a hardware más lento como entradas.

También es útil para generar entradas para mapas de Karnaugh en orden de izquierda a derecha o de arriba hacia abajo.

# --instructions--

Crea una función para codificar un número a y decodificar un número desde el código Gray. La función tendrá 2 parámetros.

El primero sería un booleano. La función debería codificar para true y decodificar para false. El segundo parámetro sería el número que se va a codificar/decodificar.

Muestra las representaciones binarias normales, las representaciones en código Gray y los valores decodificados del código Gray para todos los números binarios de 5 bits (0-31 inclusive, no es necesario incluir ceros a la izquierda).

Hay muchos códigos Gray posibles. El siguiente codifica lo que se llama "binary reflected Gray code."

Codificación (MSB es bit 0, b es binario, g es código Gray):

<pre>if b[i-1] = 1
  g[i] = not b[i]
else
  g[i] = b[i]
</pre>

O:

<pre>g = b xor (b logically right shifted 1 time)
</pre>

Decodificación (MSB es el bit 0, b es binario, g es código Gray):

<pre>b[0] = g[0]<br>
for other bits:
b[i] = g[i] xor b[i-1]
</pre>

function gray(enc, number){
  if(enc){
      return number ^ (number >> 1);
  }else{
      let n = number;

      while (number >>= 1) {
          n ^= number;
      }
      return n;
  }
