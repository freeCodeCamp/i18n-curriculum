# --description--

一組格點 $S$ 稱為巨集集，若存在一條直線恰好通過 $S$ 中的兩個點。

一個巨集集的範例是 $S = \\{(0, 0), (0, 1), (0, 2), (1, 1), (2, 0), (1, 0)\\}$，其中通過 (0, 1) 和 (2, 0) 的直線不會通過 $S$ 中的任何其他點。

另一方面，集合 {(0, 0)、(1, 1)、(2, 2)、(4, 4)} 不是一個 titanic 集合，因為通過集合中任意兩點的直線也會通過其他兩點。

對於任何正整數 $N$，令 $T(N)$ 為所有點 ($x$, $y$) 均滿足 $0 ≤ x$, $y ≤ N$ 的巨集集 $S$ 的數量。可驗證 $T(1) = 11$、$T(2) = 494$、$T(4) = 33\\,554\\,178$、$T(111)\bmod {10}^8 = 13\\,500\\,401$ 以及 $T({10}^5)\bmod {10}^8 = 63\\,259\\,062$。

尋找 $T({10}^{11})\bmod {10}^8$。
