# --description--

助動詞を使った否定文の作り方を練習しましょう。

# --questions--

## --text--

誰かがしないことを表すために`don't`を正しく使っている文を選んでください。

## --answers--

`I don't coding on weekends.`

### --feedback--

`don't`と一緒に使うときは、`coding`は底辺にある形の`code`でなければなりません。

---

`I don't code on weekends.`

---

`I don't codes on weekends.`

### --feedback--

`don't`と一緒に使うときは、動詞の最後に`-s`があってはいけません。

---

`I does not code on weekends.`

### --feedback--

`Does not`は三人称単数の主語（`he`、`she`、`it`）と一緒に使われます。`I`の場合は`do not`でなければなりません。
