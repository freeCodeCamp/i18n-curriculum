# --explanation--

`Proactive` significa agir antecipadamente para lidar com uma situação antes que ela se torne um problema.Por exemplo:

- `Being proactive means fixing small issues before they become big problems.` - Aqui, `proactive` é sobre agir cedo.

`Approach` é uma forma de lidar com algo.Por exemplo:

- `Her approach to solving the problem was very effective.` - Aqui, `approach` refere-se ao método que ela usou para resolver o problema.
