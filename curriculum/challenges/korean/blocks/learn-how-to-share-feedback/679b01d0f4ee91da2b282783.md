# --explanation--

`That's a good point`라는 표현은 누군가가 타당하거나 중요한 인수를 했음을 인정할 때 사용합니다. 예를 들어:

- **직원:** `We should consider remote work options to improve productivity.`

- **관리자:** `That's a good point. It could help employees stay more focused.` - 이 사람은 원격 근무가 직원들이 포커스를 유지하게 하여 생산성을 높이는 좋은 방법이라고 생각합니다.

이 대화에서 Lisa는 사람들의 의견을 소중히 여기지 않는 것의 영향에 대해 Brian의 말에 동의하고 있습니다.
