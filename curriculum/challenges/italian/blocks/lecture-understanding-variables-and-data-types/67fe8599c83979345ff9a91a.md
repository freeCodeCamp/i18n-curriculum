# --description--

Ogni linguaggio di programmazione ha un modo per mostrare dati nel terminale tramite un metodo, una funzione, una proprietà o una parola chiave integrata. In Python, puoi usare la funzione `print` per stampare dati nel terminale. Esaminiamo più da vicino la funzione `print` così potrai iniziare a usarla con sicurezza.

Una delle prime cose che fai quando impari un linguaggio di programmazione è scrivere un semplice programma `Hello world!`. Puoi farlo molto facilmente in Python usando solo la funzione `print`.

Per farlo, devi solo mettere la stringa `Hello world!` tra le parentesi tonde di apertura e chiusura che usi per chiamare la funzione `print`:

```python
print('Hello world!') # Hello world!
```

Imparerai di più sulle stringhe e sulle funzioni in Python nelle lezioni future. Per ora, considera le stringhe come una sequenza di caratteri racchiusa tra virgolette singole (`'`) o doppie (`"`).

Nell'esempio `print('Hello world!')`, la stringa `'Hello world!'` è un **argomento** passato alla funzione `print`. Puoi anche usare la funzione `print` per mostrare più valori, o argomenti, contemporaneamente separandoli con delle virgole. Per esempio:

```python
print('My favorite colors are', 'blue', 'green', 'red')

# Output: My favorite colors are blue green red
```

Python aggiunge automaticamente uno spazio tra ogni elemento quando li separi con le virgole. Questo è utile quando vuoi stampare insieme più informazioni.
