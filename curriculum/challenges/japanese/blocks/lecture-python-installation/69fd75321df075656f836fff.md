# --description--

前のレッスンでは、Pythonスクリプトをローカルで実行する方法を学びました。しかし、フルのPythonプログラムを作成せずに、ちょっとしたPythonコードを試したい場合もあります。そんなときに役立つのがPythonインタラクティブシェルです。

まずは、ターミナルとは何かを復習しましょう。

ターミナルは、コマンドを入力してコンピューターとやり取りできるテキストベースのインターフェースです。各オペレーティングシステムにはデフォルトのターミナルアプリが付属しています。macOSではTerminalアプリを使えます。WindowsではCommand PromptやPowerShellが使えます。Linuxでは、GNOME TerminalやKonsoleなど、デスクトップ環境ごとにデフォルトのターミナルアプリがあります。

ターミナルアプリを開いて`python`と入力し、`Enter`を押してください。これでPythonインタラクティブシェルが起動します。インタラクティブシェルは、コマンドを1つずつ入力して結果を確認できるプログラムです。

新しいセッションを始めると、最初にこのような出力が表示されることがあります。

```bash
Python 3.12.2 (main, Mar 21 2024, 22:48:26) [Clang 14.0.3 (clang-1403.0.22.14.1)] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>> 
```

`>>>`の記号は、Pythonがコマンドの入力を待っていることを意味します。ターミナルに`"Hello, world!"`を出力してみましょう。

```bash
print("Hello, world!")
```

以下のようにテキストが表示されるはずです。

```bash
>>> print("Hello, world!")
Hello, world!
```

テキストが表示された後、Pythonは以下に戻ります。

```bash
>>>
```

これはさらにコマンドを入力できる状態です。

Pythonインタプリタは「読み込み、評価、出力、ループ」のサイクル、略してREPLを実行しています。コマンドを入力すると、インタプリタは入力を読み込み、評価し、結果を出力し、再び`>>>`を表示して次のコマンド入力を待ちます。

もし無効なコマンドを入力したらどうなるでしょうか？

```bash
>>> something random
```

Pythonインタプリタは同じREPLプロセスを実行しますが、この場合はエラーメッセージが表示されます。

```bash
>>> something random
  File "<stdin>", line 1
    something random
              ^^^^^^
SyntaxError: invalid syntax
```

インタラクティブシェルを終了したい場合は、`exit()`と入力するか、Mac/Linuxなら`Ctrl + D`、Windowsなら`Ctrl + Z + Enter`を押してください。

Pythonのインタラクティブシェルは、コードの小さな実験や基本的な探索に最適です。コースの残りのレッスンを進める際に、新しく学んだメソッドをインタラクティブシェルで試してみてください。

より長いプログラムを扱ったり、複数のファイルにまたがって作業する場合は、コードエディタやIDEで作業することをお勧めします。
