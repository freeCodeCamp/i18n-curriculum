# --explanation--

`Could` і `might` — це модальні дієслова, які використовують для вираження можливостей.

`Could` означає, що щось можливо, але не точно. Наприклад:

`This update could make the app run faster.` - The update has the potential to improve the app's speed.

`Might` також виражає можливість, але часто припускає меншу ймовірність, ніж `could`. Наприклад:

`The new feature might improve the user experience.` - It is possible that the feature will make the experience better.
