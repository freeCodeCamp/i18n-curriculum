# --interactive--

你可能看過網頁上的聯結在點擊後變成紫色。這是因為聯結的狀態已改變，因此會套用不同的樣式。聯結可以有五種不同的狀態。

第一個是預設狀態，即 `:link`。此狀態表述使用者尚未造訪、點擊或互動過的聯結。你可以將此狀態視為為你頁面上所有聯結提供基礎樣式。其他狀態則是在此基礎上建置。

啟用互動式編輯器並查看預覽視窗中的聯結。它應該是預設的藍色，表現預設狀態。如果你點擊它，則會變成紫色。 

:::interactive_editor

```html
<a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
```

:::

第二個狀態是 `:visited`，當使用者已經造訪過被連結的頁面時會套用此狀態。預設值會將連結變成紫色－但你可以利用 CSS 為使用者提供不同的視覺提示。這有助於讓某人知道他們已經閱讀過你文件的一部分。或者，該網站是他們可以信任的，因為他們之前使用過。

點擊預覽視窗中的聯結，你應該會看到已造訪的聯結變成棕色。要看到預覽，你需要啟用互動式編輯器。

**注意**：為了讓你看到聯結樣式的變化，這個互動範例已提供部分 CSS。不要擔心嘗試理解 CSS 程式碼，因為你會在未來的模組中學到它的全部意義。 

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:visited {
  color: brown;
}
```

:::

第三種狀態是 `:hover`。當使用者將游標暫留在聯結上時，會套用此狀態。此狀態有助於為聯結提供額外的注意，確保使用者確實打算點擊它。

啟用互動式編輯器，並嘗試將你的滑鼠暫留在聯結上。你將看到顏色變為紅色。 

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:hover {
  color: red;
}
```

:::

接著我們有 `:focus`。當我們聚焦於一個聯結時，會套用此狀態。

點擊預覽視窗中空白區域的任何部分，然後按下鍵盤上的 `tab`。你應該會看到聯結變成綠色。若要查看預覽，你需要啟用互動式編輯器。

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:focus {
  color: green;
}
```

:::

最後，我們有 `:active`。此狀態適用於正在被使用者作用的連結。這通常表示在大多數情況下，使用主要滑鼠按鈕左鍵點擊該連結。此狀態有助於向使用者顯示他們點擊的元素是可互動的。

啟用互動式編輯器並點擊該聯結。你應該會看到該聯結變成黑色。這個過程非常快速，所以你可能需要點擊幾次才能看到顏色變化。 

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:active {
  color: black;
}
```

:::

當你使用這些狀態來為你的聯結設計樣式時，你需要以特定順序撰寫你的 CSS：`link`、`visited`、`hover`、`focus`，然後是 `active`。

你現在應該知道如何為你的頁面中的聯結賦予個人風格，同時仍提供使用者所需的每個聯結狀態的重要資訊。
