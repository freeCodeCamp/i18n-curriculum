# --description--

前のレッスンでは、リストのデータ型について紹介し、リストから要素をアクセスする方法やリストのスライスについて学びました。このレッスンでは、リストとそれに関連するよく使われるメソッド、例えば `append()`、`pop()`、`sort()` についてさらに学びます。

最初に見ていくメソッドは `append()` メソッドです。これはリストの末尾にアイテムを追加するために使います。以下は `append()` メソッドを使って数値 `6` を数値のリストに追加する例です。

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

もし別のリストの末尾に1つのリストを追加したい場合は、次のように `append()` メソッドを使うこともできます。

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

`even_numbers` リスト全体が `numbers` リストの中にネストされていることに注目してください。

しかし、`even_numbers` リストの個々の数値すべてを `numbers` リストの末尾に追加したい場合は、`extend()` メソッドを使うことができます。

`extend()` メソッドは `append()` メソッドに似ていますが、`extend()` を使うと1つのリストから別のリストに複数の要素を追加できます。以下は `6`、`8`、`10` の数値を1つのリストから `numbers` リストの末尾に追加する例です。

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

ご覧の通り、ネストされたリストはなくなり、単なる数値のリストになっています。

リストの特定のインデックスに要素を挿入したい場合は、`insert()` メソッドを使えます。このメソッドは2つの引数を受け取ります：新しいアイテムを挿入したいインデックスと挿入したいアイテムです。

以下は `insert()` メソッドを使った例です。

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

このコードは `2.5` リストのインデックス `2` に数値 `numbers` を挿入します。

リストから要素を削除したい場合は、`remove()` メソッドを使えます。`remove()` メソッドは削除したい要素の値を引数として受け取ります。

```py
numbers = [10, 20, 30, 40, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50]
```

このメソッドは最初に見つかった1つの要素だけを削除することに注意してください。すべての要素を削除するわけではありません。

```py
numbers = [10, 20, 30, 40, 50, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50, 50]
```

リストの特定のインデックスにある要素を削除したい場合は、次のように `pop()` メソッドを使えます。

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

`pop` メソッドで要素を指定しなければ、最後の要素が削除されます。

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

リストを空にしたい場合は、次のように `clear()` メソッドを使えます。

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

次に見ていくメソッドは `sort()` メソッドです。このメソッドは要素をその場でソートします。以下はランダムな数値のリストをその場でソートする例です。

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

`sort()` メソッドとは対照的に、`sorted()` 関数は任意のイテラブルに対して動作し、元のリストを変更するのではなく新しいソート済みリストを返します。例えば：

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(numbers) # [19, 2, 35, 1, 67, 41]
print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
```

イテラブルとは、1つずつアイテムにアクセスできる特別なタイプのオブジェクトです。Pythonのループの仕組みについては今後のレッスンでさらに学びます。

`sort()` メソッドと `sorted()` 関数はどちらもオプションの `key` と `reverse` パラメータを受け取ります。これらのオプションのパラメータについては、タプルについて学ぶ将来のレッスンで詳しく学びます。また、`sorted()` のような他の組み込み関数についても今後のレッスンで学びます。

次に見ていくメソッドは `reverse()` メソッドです。このメソッドはリストの要素をその場で逆順にします。

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

最後に見ていくメソッドは `index` メソッドです。これはリストの中で要素が最初に見つかるインデックスを探すために使います。以下は `index` メソッドを使って `'Java'` リストの中から言語 `programming_languages` を探す例です。

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

要素が見つからない場合、Pythonは `ValueError` をスローします。

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: 'JavaScript' is not in list
"""
```

Pythonのリストには他にもいくつかメソッドがありますが、この最初のメソッドのリストは学び始めるのに良い出発点です。
