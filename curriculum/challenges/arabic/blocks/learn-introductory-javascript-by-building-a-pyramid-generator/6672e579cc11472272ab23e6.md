# --description--


JavaScript هي لغة البرمجة التي تشغل الويب. بخلاف HTML وCSS التي تعلمتها سابقًا، تُستخدم JavaScript عادةً لكتابة المنطق بدلًا من الترميز.

في هذا المشروع، ستتعلم أساسيات JavaScript وتطبق تلك المفاهيم لبناء مولد هرم.

مولد الهرم هو برنامج يمكنك من خلاله تحديد نوع الحرف، وعدد طبقات الهرم، واتجاه الهرم. ثم يُولد البرنامج هرمًا بناءً على تلك المُدخلات.

انقر على زر "Check your code" للانتقال إلى الخطوة التالية وبدء المشروع.
