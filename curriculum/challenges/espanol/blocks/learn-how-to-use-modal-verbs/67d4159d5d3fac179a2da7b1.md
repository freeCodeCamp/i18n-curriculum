# --explanation--

`Multi-factor authentication` se refiere a un proceso de seguridad que requiere múltiples formas de confirmar la identidad de un usuario. Por ejemplo:

- Muchos sitios web requieren autenticación multifactor para proteger la información personal. Muchos sitios web piden medidas de seguridad adicionales para mantener los datos personales seguros.

- Si quieres más seguridad, debes activar la autenticación multifactor en tu cuenta. Activar autenticaciones adicionales aumenta la seguridad de la cuenta.
