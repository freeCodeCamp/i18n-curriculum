# --description--

为你的输入添加一个 `onChange` 属性以捕捉用户输入的内容。将其设置为一个以 `e` 作为参数的箭头函数。在函数内部，通过将 `e.target.value` 传递给设置函数来更新 `query` 状态，该函数保存输入的当前值。示例如下：

```js
<input
  onChange={(e) => setExample(e.target.value)}
/>
```
