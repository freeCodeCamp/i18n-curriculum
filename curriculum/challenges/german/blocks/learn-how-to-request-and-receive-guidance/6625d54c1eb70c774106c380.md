# --description--

Ein `installer` ist ein Programm, das zum Installieren von Software verwendet wird. Zum Beispiel `Download the installer to get the app on your computer`.

`Setup` als Substantiv bezeichnet die Anordnung oder Vorbereitung von etwas. Zum Beispiel `Check the setup instructions for your new phone`. `Set up` als Verb bedeutet anordnen oder vorbereiten, wie `Please set up the chairs for the meeting`.

`Installation` bezeichnet den Prozess des Installierens von etwas, insbesondere Software oder Ausrüstung. Zum Beispiel `Follow the steps for the software installation`.
