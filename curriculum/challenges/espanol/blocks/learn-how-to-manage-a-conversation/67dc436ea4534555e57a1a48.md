# --explanation--

`As I was saying` se usa para volver a un tema anterior después de una interrupción. Señala que el hablante está continuando su punto previo. Por ejemplo:  

`As I was saying, we should finalize the report by Friday.` - Esto significa que el hablante ya estaba discutiendo el informe antes de ser interrumpido.
