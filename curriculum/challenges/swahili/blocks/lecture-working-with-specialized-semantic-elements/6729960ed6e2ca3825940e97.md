# --interactive--

Kipengele cha msimbo ndani ya mstari hutumika kuwakilisha vipande vifupi vya msimbo ndani ya maandishi. Matumizi ya kawaida ya kipengele cha msimbo ni kwa makala za kiufundi na kurasa za nyaraka.

Hapa kuna mfano wa kutumia kipengele cha `code` kuonyesha kipande cha msimbo cha CSS:

:::interactive_editor

```html
<p>
  To set the text color to blue in CSS, use the following code:
  <code>color: blue;</code>
</p>
```

:::

Katika mfano huu, sifa ya CSS `color` inatumika kuweka rangi ya maandishi kuwa `blue`. Kwa kuzungusha kipande hiki cha msimbo ndani ya lebo za `<code>`, inawasilisha kwa kivinjari kwamba maandishi ni kipande cha msimbo ndani ya mstari.

Kivinjari kitaweka mitindo ya msingi kwa yaliyomo ndani ya kipengele cha `code`. Mtindo wa msingi ni fonti ya monospaced.

Kipengele cha `code` kimekusudiwa kuwakilisha mstari mmoja wa msimbo. Ikiwa unataka kuwakilisha mistari mingi ya msimbo, utahitaji kuweka kipengele cha `code` ndani ya kipengele cha maandishi yaliyopangwa awali.

Kipengele cha maandishi yaliyopangwa awali hutumika kuwakilisha maandishi yaliyopangwa awali. Hapa kuna mfano wa kutumia kipengele cha maandishi yaliyopangwa awali kuonyesha tamko la CSS:

:::interactive_editor

```html
<pre>
  <code>
    body {
      color: red;
    }
  </code>
</pre>
```

:::

Unapotumia kipengele cha `pre`, utahitaji kuzingatia nafasi kwa sababu kitaonyesha kama ilivyoandikwa ndani ya hati ya HTML.

Katika kivinjari, utaona kwamba msimbo umeingizwa nafasi kadhaa kulia. Ikiwa utabadilisha uingizaji nafasi katika mfano wa msimbo, basi utaona tofauti ya uingizaji nafasi kwenye skrini.

Linapokuja suala la kujumuisha mifano ya msimbo ndani ya hati yako ya HTML, unapaswa kutumia kipengele cha `code` kwa mifano mifupi ndani ya mstari.

Ikiwa unahitaji kuonyesha vipande virefu vya msimbo, basi utahitaji kutumia vipengele vya `pre` na `code`.
