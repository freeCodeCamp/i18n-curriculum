# --description--

格子点 ($x$, $y$) が不許容であるとは、$x$、$y$、および $x + y$ のすべてが正の完全平方数である場合と呼びます。

例えば、(9, 16) は不許容ですが、(0, 4)、(3, 1)、(9, 4) は不許容ではありません。

単位ステップで北または東にのみ進む経路を、点 ($x_1$, $y_1$) から点 ($x_2$, $y_2$) まで考えます。このような経路を、その中間のどの点も不許容でない場合に許容可能と呼びます。

$P(n)$ を (0, 0) から ($n$, $n$) までの許容可能な経路の数とします。$P(5) = 252$、$P(16) = 596\,994\,440$、および $P(1\,000)\bmod 1\,000\,000\,007 = 341\,920\,854$ であることが確認できます。

$P(10\,000\,000)\bmod 1\,000\,000\,007$ を求めてください。
