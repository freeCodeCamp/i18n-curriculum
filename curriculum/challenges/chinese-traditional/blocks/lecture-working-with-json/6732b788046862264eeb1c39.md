# --description--

什麼是 JSON 以及你如何使用括號和點符號表示法存取值？

JSON 代表 JavaScript Object Notation。它是一種輕量級、基於文字的資料格式，常用於伺服器與網頁應用程式之間交換資料。 

JSON 在網頁開發中如此受歡迎的原因之一是因為它既能被機器解析，也易於人類閱讀。   

由於 JSON 是語言獨立的，你可以輕鬆地將 JSON 資料從 Java 應用程式傳送到 Python 應用程式，或從 JavaScript 應用程式傳送到 C# 應用程式。

JSON 支援多種資料型別，包括物件、陣列、字串、布林值、null 和數字。 

以下是一個 JSON 物件的範例：

```js
{
  "name": "Alice",
  "age": 30,
  "isStudent": false,
  "list of courses": ["Mathematics", "Physics", "Computer Science"]
}
```

如你所見，JSON 使用鍵值配對來儲存資訊，且每個配對以逗號分隔。每個鍵必須用雙引號包住，否則你會收到錯誤。

要從 JSON 物件中存取資料，你可以使用點記法或括號記法。在此範例中，我們使用點記法來存取 JSON 物件中的 `age`：

```js
import data from "./example.json" with { type: "json" };

console.log(data.age);
```

這個特定範例使用了所謂的 `import` 述語，將 JSON 物件匯入到這個檔案中，讓我們可以存取它。你將在未來的課程中學習更多關於 `import` 述語的內容。

你也可以使用括號表示法來存取 JSON 物件中的資訊。以下是存取 `list of courses` 陣列的範例：

```js
import data from "./example.json" with { type: "json" };

console.log(data["list of courses"]);
```

在這裡使用括號表示法特別有用，因為鍵包含多個以空格分隔的字。若嘗試使用點表示法，將會導致錯誤。

總結來說，JSON 是一種多功能格式，可以儲存許多資料型別，包括陣列和巢狀物件。透過使用點記法或括號記法，你可以輕鬆存取儲存在 JSON 物件中的值。
