# --description--

動詞は、それが指す人に応じて活用されることを覚えていますか？もし忘れていたら、今覚えましょう！

数チャレンジ前に、誰かに何かをしてもらえるかどうか尋ねるために動詞 `poder` を使う方法を学びました。

今回は、同じ動詞を使って**自分が**何かをできるかどうか尋ねるための活用を学びます。使う構文は次の通りです：

`Puedo` + 不定詞動詞 + `...?`

いくつか例を見てみましょう：

- `¿Puedo usar la computadora?` - コンピューターを使ってもいいですか？

- `¿Puedo usar mi apodo?` - 私のニックネームを使ってもいいですか？

どちらの例でも、コンピューターやニックネームを使う許可を求めています。

# --explanation--

バスティは `¿Puedo usar mi apodo?` と言っています。

ここで `puedo` は動詞 `poder` の一人称単数形で、許可を求めるときに使います。

構文は次のパターンに従います：

`¿Puedo` + 不定詞動詞 + `...?`

この場合：

- `Usar` は「使う」という意味です。

- `Mi apodo` は彼のニックネームを指します。

つまり、バスティは自分のニックネームを使ってもいいかどうか尋ねています。
