## 语义化 HTML 的重要性

- **标题元素的结构层次**：使用正确的标题元素来保持内容的结构层次非常重要。 `h1` 元素是最高级别的标题，`h6` 元素是最低级别的标题。
- **表示性 HTML 元素**：定义内容外观的元素。 例如：已废弃的 `center`、`big` 和 `font` 元素。
- **语义化 HTML 元素**：具有意义和结构的元素。 例如： `header`、`nav`、`figure`。

## 语义化 HTML 元素

- **header 元素**：用于定义文档或某个部分的头部内容。
- **main 元素**：用于包含网页的主要内容。
- **section 元素**：用于将内容划分为较小的部分。
- **导航（`nav`）元素**：代表带有导航链接的部分。
- **figure 元素**：用于包含插图和图表。
- **强调（`em`）元素**：用于标记需要强调的文本。

```html
<p>
  Never give up on <em>your</em> dreams.
</p>
```

- **惯用文本元素（`i`）**：用于突出显示另一种语调或语境下的词语，比如外语术语、技术术语和思想。

```html
<p>
  There is a certain <i lang="fr">je ne sais quoi</i> in the air.
</p>
```

`i` 开始标签中的 `lang` 属性用于指定内容的语言。 本例中，语言就是法语。 `i` 元素并不表示文本是否重要，它只是表示文本与周围的文本有某种不同。

- **强烈强调（`strong`）元素**：用于标记非常重要的文本。

```html
<p>
  <strong>Warning:</strong> This product may cause allergic reactions.
</p>
```

- **引起注意（`b`）元素**：用于突出显示对内容含义不重要但需要引起注意的文本。 它常用于总结中的关键词或评论中的产品名。

```html
<p>
  We tested several products, including the <b>SuperSound 3000</b> for audio quality, the <b>QuickCharge Pro</b> for fast charging, and the <b>Ecoclean Vacuum</b> for cleaning. The first two performed well, but the <b>Ecoclean Vacuum</b> did not meet expectations.
</p>
```

- **描述列表（`dl`）元素**：用于表示术语-描述的配对列表。
- **描述术语（`dt`）元素**：表示要定义的术语。
- **描述细节（`dd`）元素**：表示该术语的解释。

```html
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
```

- **块级引用（`blockquote`）元素**：用于表示引用自其他来源的较长内容。 该元素有一个 `cite` 属性。 `cite` 属性的值是源文件的 URL。

```html
<blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
  "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
</blockquote>
```

- **引用（`cite`）元素**：用于可视化地标记引用来源。 通常用于标记作品的标题。

```html
<div>
  <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
    "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
  </blockquote>
  <p>
    -Quincy Larson, <cite>How to learn to Code and Get a Developer Job [Full Book].</cite>
  </p>
</div>
```

- **内联引用（`q`）元素**：用于表示短的内联引用。

```html
<p>
  As Quincy Larson said,
  <q cite="https://www.freecodecamp.org/news/learn-to-code-book/">
    Momentum is everything.
  </q>
</p>
```

- **缩写（`abbr`）元素**：用于表示缩写或首字母缩略词。 为帮助用户理解缩写或首字母缩略词的含义，可以使用 `title` 属性显示其完整形式，即人类可读的描述。

```html
<p>
  <abbr title="HyperText Markup Language">HTML</abbr> is the foundation of the web.
</p>
```

- **联系地址（`address`）元素**：用于表示联系信息。
- **（日期）时间（`time`）元素**：用于表示日期或时间。 `datetime` 属性用于将日期和时间转换为机器可读格式。

```html
<p>
  The reservations are for the <time datetime="20:00">20:00 </time>
</p>
```

- **ISO 8601 日期（`datetime`）属性**：用于以机器可读格式表示日期和时间。 标准格式是 `YYYY-MM-DDThh:mm:ss`，其中大写 `T` 用于分隔日期和时间。
- **上标（`sup`）元素**：用于表示上标文本。 `sup` 元素的常见用例包括指数、上位字母和序数词。

```html
<p>
  2<sup>2</sup> (2 squared) is 4.
</p>
```

- **下标（`sub`）元素**：用于表示下标文本。 下标元素的常见用例包括化学式、脚注和变量下标。

```html
<p>
  CO<sub>2</sub>
</p>
```

- **内联代码（`code`）元素**：用于表示一段代码。
- **预格式化文本（`pre`）元素**：用于表示预先格式化的文本。

```html
<pre>
  <code>
    body {
      color: red;
    }
  </code>
</pre>
```

- **非文本注释（`u`）元素**：用于表示一段内联文本，该文本以表明其具有非文本注释的方式呈现。

```html
<p>
  You can use the unarticulated annotation element to highlight
  <u>inccccort</u> <u>spling</u> <u>issses</u>.
</p>
```

- **Ruby 注释（`ruby`）元素**：用于注释带有发音或含义解释的文本。 一个示例用法是东亚文字注音排版。
- **Ruby 回退括号（`rp`）元素**：用作不支持显示 ruby 注释的浏览器的回退显示。
- **Ruby 文本（`rt`）元素**：用于表示 ruby 注释的文本。 通常用于东亚排版中的发音或翻译细节。

```html
<ruby>
  明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp>
</ruby>
```

- **删除线（`s`）元素**：用于表示不再准确或相关的内容。

```html
<p>
  <s>Tomorrow's hike will be meeting at noon.</s>
</p>
<p>
  Due to unforeseen weather conditions, the hike has been canceled.
</p>
```
