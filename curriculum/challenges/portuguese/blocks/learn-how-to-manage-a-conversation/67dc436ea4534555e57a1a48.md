# --explanation--

`As I was saying` é usado para retornar a um tópico anterior após uma interrupção.Isso indica que o falante está continuando seu ponto anterior.Por exemplo:  

`As I was saying, we should finalize the report by Friday.` - Isso significa que o falante já estava discutindo o relatório antes de ser interrompido.
