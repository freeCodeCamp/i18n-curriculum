# --explanation--

`Improve` は、何かをより良くしたり、品質を高めたり、問題を修正したりすることを意味します。例えば：

`She wants to improve her coding skills.` - これは、彼女がもっと学んだり練習したりしてコーディングスキルを向上させたいという意味です。

`Improve on` は、以前よりも何かを良くすることについて話すときに使われます。特定の側面やエリアの改善にフォーカスしています。例えば：

`He wants to improve on his previous performance in the competition.` - これは、彼が競技で以前よりも良い成績を出したいという意味です。
