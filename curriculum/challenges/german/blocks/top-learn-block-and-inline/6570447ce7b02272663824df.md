# --description--

Die meisten der Elemente, die Sie bisher kennengelernt haben, sind Blockelemente. Mit anderen Worten: Ihr Standardstil ist `display: block`. Standardmäßig erscheinen Blockelemente auf der Seite übereinander gestapelt, wobei jedes neue Element in einer neuen Zeile beginnt.
