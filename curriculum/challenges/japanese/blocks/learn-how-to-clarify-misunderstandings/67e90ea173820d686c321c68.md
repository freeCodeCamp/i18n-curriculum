# --explanation--

例として`the "Student" class inherits properties and methods from the "Person" class`があります。

- `Student`と`Person`はどちらも`classes`です。`class`は特定のデータと振る舞いを持つオブジェクトを作成するための設計図を定義します。

- `Student`は`Person`から機能を引き継ぎます。つまり、共通の特性を再定義する必要がなく、それらを再利用します。

- `Properties`はクラス内の変数やデータで、`name`や`age`のようにオブジェクトを説明します。

- `Methods`はクラス内で定義された関数で、`enroll()`や`introduce()`のようにオブジェクトができることを説明します。
