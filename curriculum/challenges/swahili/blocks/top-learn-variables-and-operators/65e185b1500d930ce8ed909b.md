# --description--
Njia rahisi kabisa ya kuanza ni kuunda faili la HTML lenye msimbo wa JavaScript ndani yake. Andika mifupa ya msingi ya HTML katika faili kwenye kompyuta yako mahali fulani:

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>
  <script>
    // Your JavaScript goes here!
    console.log("Hello, World!")
  </script>
</body>
</html>
```

Hifadhi na fungua faili hili katika kivinjari cha mtandao (unaweza kutumia <a href="https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer" target="_blank" >"Live Server" kwenye Visual Studio Code</a> kufanya hivi!) kisha fungua konsoli ya kivinjari kwa kubofya kulia kwenye ukurasa tupu wa mtandao na kuchagua "Inspect" au "Inspect Element". Katika dirisha la ‘Vifaa vya Msanidi Programu’ tafuta na chagua kichupo cha ‘Console’, ambapo unapaswa kuona matokeo ya usemi wetu wa `console.log`.

> `console.log()` ni amri ya kuchapisha kitu kwenye konsoli ya msanidi programu katika kivinjari chako. Unaweza kutumia hii kuchapisha matokeo kutoka kwa makala na mazoezi yoyote yaliyo hapa. Tunakuhimiza uandike msimbo sambamba na mifano yote katika funzo hili na funzo zijazo.

Njia nyingine ya kujumuisha JavaScript kwenye ukurasa wa mtandao ni kupitia script ya nje. Hii ni sawa sana na kuunganisha faili za CSS za nje kwenye tovuti yako.

```html
  <script src="javascript.js"></script>
```

Mafaili ya JavaScript yana kiambatisho cha `.js` kama vile `.css` kwa ajili ya majalada ya mitindo. Mafaili ya JavaScript ya nje hutumika kwa script ngumu zaidi.

# --questions--

## --text--

Ni usemi gani unaelezea kwa usahihi jinsi ya kujumuisha JavaScript katika hati ya HTML?

## --answers--

JavaScript inaweza kujumuishwa tu ndani ya lebo ya `<script>` ndani ya mwili wa HTML.

---

Msimbo wa JavaScript lazima uandikwe katika sehemu ya `<head>` ya hati ya HTML ili utekelezwe ipasavyo.

---

JavaScript inaweza kujumuishwa moja kwa moja katika faili la HTML kwa kutumia lebo ya `<script>` au kuunganishwa kwa nje kupitia lebo ya `<script src="javascript.js">`.

---

Mafaili ya JavaScript ya nje yanahitaji sifa maalum katika lebo ya `<script>` ili yatambuliwe na vivinjari vya mtandao.
