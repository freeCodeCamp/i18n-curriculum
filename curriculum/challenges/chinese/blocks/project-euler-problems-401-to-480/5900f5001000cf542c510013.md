# --description--

对于整数 $a$ 和 $b$，我们定义 $D(a, b)$ 为由抛物线 $y = x^2$ 和直线 $y = ax + b$ 围成的域：$D(a, b) = \\{ (x, y) | x^2 ≤ y ≤ ax + b \\}$。

$L(a, b)$ 被定义为包含在 $D(a, b)$ 中的格点数。例如，$L(1, 2) = 8$ 和 $L(2, -1) = 1$。

我们还定义 $S(N)$ 为所有点对 ($a$, $b$) 的 $L(a, b)$ 之和，条件是 $D(a, b)$ 的面积为有理数且 $|a|,|b| ≤ N$。

我们可以验证 $S(5) = 344$ 和 $S(100) = 26\\,709\\,528$。

查找 $S({10}^{12})$。给出你的答案 $\bmod {10}^8$。
