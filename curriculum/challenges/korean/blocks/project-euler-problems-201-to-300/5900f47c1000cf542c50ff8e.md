# --description--

한 변의 길이가 정수인 $N×N$ 크기의 정사각형 종이가 원점에 한 꼭짓점을 두고, 두 변이 $x$축과 $y$축을 따라 놓여 있습니다. 다음 규칙을 지키며 이 종이를 자릅니다:

- 정사각형의 서로 다른 변 위에 있고 정수 좌표를 가진 두 점 사이에만 직선으로 자릅니다.
- 두 자르는 선은 교차할 수 없지만, 여러 선이 같은 테두리 점에서 만날 수 있습니다.
- 더 이상 합법적인 자르기가 불가능할 때까지 진행합니다.

반사나 회전을 구분하여, $C(N)$을 $N×N$ 정사각형을 자르는 방법의 수라고 합니다. 예를 들어, $C(1) = 2$이고 $C(2) = 30$입니다(아래 그림 참조).

$C(30)\bmod {10}^8$ 의 값은 얼마인가요?
