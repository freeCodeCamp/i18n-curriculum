# --description--

Для цілого числа $M$ визначимо $R(M)$ як суму $\frac{1}{p·q}$ для всіх пар цілих чисел $p$ та $q$, які задовольняють усі ці умови:

- $1 ≤ p &lt; q ≤ M$
- $p + q ≥ M$
- $p$ та $q$ є взаємно простими.

Також визначимо $S(N)$ як суму $R(i)$ для $2 ≤ i ≤ N$.

Ми можемо перевірити, що $S(2) = R(2) = \frac{1}{2}$, $S(10) ≈ 6.9147$ та $S(100) ≈ 58.2962$.

Знайдіть $S({10}^7)$. Надайте відповідь, округлену до чотирьох десяткових знаків.
