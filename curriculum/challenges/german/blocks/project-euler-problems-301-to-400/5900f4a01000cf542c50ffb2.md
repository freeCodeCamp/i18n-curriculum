# --description--

$k$ Fehler sind zufällig unter $n$ integrierten Schaltkreisen verteilt, die von einer Fabrik hergestellt wurden (eine beliebige Anzahl von Fehlern kann auf einem Chip gefunden werden, und jeder Fehler ist unabhängig von den anderen Fehlern).

Sei $p(k,n)$ die Wahrscheinlichkeit, dass es einen Chip mit mindestens 3 Fehlern gibt. Zum Beispiel ist $p(3,7) ≈ 0.0204081633$.

Berechnen Sie $p(20\\,000, 1\\,000\\,000)$ und geben Sie Ihre Antwort auf 10 Dezimalstellen gerundet in der Form 0.abcdefghij an.
