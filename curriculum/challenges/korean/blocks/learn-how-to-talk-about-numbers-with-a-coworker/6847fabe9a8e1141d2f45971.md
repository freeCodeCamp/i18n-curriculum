# --explanation--

83%를 읽을 때는 `Eighty-three percent`가 올바른 방법입니다.

정확한 숫자 뒤에는 항상 `percent`를 사용합니다 (`percentages`가 아니라). 차이가 무엇일까요?

숫자 뒤에는 `percent`를 사용하세요:

- 83%는 `eighty-three percent`입니다.

- 20%는 `twenty percent`입니다.

일반적으로 숫자 뒤가 아닌 경우에는 `percentage`나 `percentages`를 사용하세요:

- `Eighty-three percentages`는 틀렸습니다.

- `We saw an increase in percentages across departments.`는 올바릅니다 (특정 숫자 없이 사용됨).
