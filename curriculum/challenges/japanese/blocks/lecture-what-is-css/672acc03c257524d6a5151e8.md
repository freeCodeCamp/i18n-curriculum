# --interactive--

CSSでは、`width`と`height`のプロパティを使ってウェブページ上の要素の寸法を制御します。

これらのプロパティは、ピクセル（`px`）、パーセンテージ（`%`）、ビューポート単位（`vw`、`vh`）など、さまざまな単位で指定できます。

`width`プロパティは要素の幅を指定します。幅を指定しない場合、デフォルトでは`auto`に設定されます。これにより、ブラウザーは要素の内容、親要素、表示タイプに基づいて幅を決定します。`div`要素の場合、`width: auto`は親コンテナの幅いっぱいに広がるようにします。

`height`プロパティは要素の高さを指定します。同様に、高さはデフォルトで`auto`に設定されており、内部の内容に合わせて調整されます。

以下は`width`と`height`のプロパティを使った例です：

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 100px;
      height: 100px;
      background-color: skyblue
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

この例では、`div`要素に`box`というクラスが付いています。この要素は高さと幅が`100px`を占め、背景色は`skyblue`になります。

ピクセルはCSSで固定サイズの単位であり、寸法を正確に制御できます。

`min-width`プロパティは要素の最小幅を指定します。内部の内容が小さくても、この値より小さくはなりません。

`min-height`は要素の最小高さを指定します。設定した値より短くならないようにします。

以下は例です：

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 50px;
      min-width: 100px;
      height: 50px;
      min-height: 100px;
      background-color: lightcoral;
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

上の例は`min-width`と`min-height`の動作を示しています。

ボックスの`width`と`height`が50pxに設定されていても、実際には`100px`×`100px`になります。これは`min-width`と`min-height`が`100px`に設定されており、指定された`width`と`height`より大きいためです。

覚えておいてください。`min-width`や`min-height`が`width`や`height`より大きい場合、それらが小さい値をオーバーライドします。これにより、要素が小さくなりすぎず、一貫性のある使いやすいデザインを保てます。

`max-width`は要素が成長できる最大幅を指定します。十分なスペースがあってもこれ以上は広がりません。

`max-height`は要素が成長できる最大高さを指定します。内容のサイズに関係なく制限されます。

以下は例です：

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 200px;
      max-width: 150px;
      height: 200px;
      max-height: 150px;
      background-color: lightgreen;
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

上の例は`max-width`と`max-height`が`width`と`height`をオーバーライドする様子を示しています。ボックスは`200px`×`200px`に設定されていますが、実際には`150px`×`150px`になります。これは`max-width`と`max-height`が`150px`に設定されており、より小さいためです。

覚えておいてください。`max-width`や`max-height`が`width`や`height`より小さい場合、それらが優先されます。これはレイアウト内の要素の最大サイズを制御するのに重要です。

CSSは`min-width`と`min-height`を`width`と`height`より優先します。`max-width`と`max-height`は値が制限を超えた場合に寸法を制限します。

例えば、`width`を`600px`に、`max-width`を`500px`に設定すると、要素は`500px`の幅に制限されます。`max-width`がより広い設定をオーバーライドし、要素を指定された最大サイズ内に保ちます。

これにより、標準の幅と高さの値に関係なく、要素が望ましいサイズ範囲内に収まります。
