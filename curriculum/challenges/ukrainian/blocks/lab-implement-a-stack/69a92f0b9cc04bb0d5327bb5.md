# --description--

У цій лабораторній роботі ви реалізуєте структуру даних стек за допомогою функцій. Стек — це структура даних типу «останній зайшов — перший вийшов» (LIFO), де елементи додаються та видаляються з вершини.

**Мета:** Виконайте наведені нижче історії користувача та пройдіть усі тести, щоб завершити лабораторну роботу.

**Історії користувача:**

1. Ви повинні мати функцію `initStack`, яка повертає об’єкт із властивістю `collection`, встановленою в порожній масив.
1. Ви повинні мати функцію `push`, яка додає елемент на вершину стеку.
1. Ви повинні мати функцію `pop`, яка видаляє та повертає верхній елемент стеку, або `undefined`, якщо такого немає.
1. Ви повинні мати функцію `peek`, яка повертає верхній елемент стеку без видалення, або `undefined`, якщо такого немає.
1. Ви повинні мати функцію `isEmpty`, яка повертає `true`, якщо стек не містить елементів, інакше `false`.
1. Ви повинні мати функцію `clear`, яка видаляє всі елементи зі стеку.

**Примітка:** Більшість тестів залежать від функцій `initStack` і `push`. Спочатку реалізуйте їх, оскільки тести для `pop`, `peek`, `isEmpty` і `clear` вимагають додавання елементів у стек.

```js
const stack = initStack();
push(stack, 'first');

```

```js

```

Функція `pop` має видаляти та повертати верхній елемент стеку.

```js
const stack = initStack();
push(stack, 'first');
push(stack, 'second');
push(stack, 'third');
assert.strictEqual(pop(stack), 'third');
assert.strictEqual(pop(stack), 'second');
```

```js

push(stack, '');
assert.strictEqual(pop(stack), '');
assert.lengthOf(stack.collection, 0);
```

# --solutions--

```js
function initStack() {
  return {
    collection: []
  };
}

function print(stack) {
  console.log(stack.collection);
}

function push(stack, element) {
  stack.collection.push(element);
}

function pop(stack) {
  return stack.collection.pop();
}

function peek(stack) {
  return stack.collection[stack.collection.length - 1];
}

function isEmpty(stack) {
  return stack.collection.length === 0;
}

function clear(stack) {
  stack.collection = [];
}
```
