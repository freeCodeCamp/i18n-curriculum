# --description--

你要构建的第一种密码称为*Caesar*密码  具体来说，你将取出消息中的每个字母，找到其在字母表中的位置，取出字母表中 3 个位置之后的字母，然后用新字母替换原始字母。

要实现此目的，你将使用上一步中讨论的 `.find()` 方法 修改现有的 `.find()` 调用，将其传递为 `text[0]` 作为参数，而不是 `'z'`。
