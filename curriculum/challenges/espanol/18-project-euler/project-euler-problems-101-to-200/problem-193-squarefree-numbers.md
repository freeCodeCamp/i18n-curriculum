# --description--

Un entero positivo $n$ se llama squarefree, si ningún cuadrado de un primo divide a $n$, así 1, 2, 3, 5, 6, 7, 10, 11 son squarefree, pero no 4, 8, 9, 12.

¿Cuántos números squarefree existen por debajo de $2^{50}$?
