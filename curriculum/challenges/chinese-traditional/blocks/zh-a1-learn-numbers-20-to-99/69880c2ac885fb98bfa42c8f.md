# --description--

我們來學習 60 幾的數字。

中文中，數字 60 稱為 `六十 (liù shí)`。

61 到 69 的數字是由 `六十 (liù shí)`（60）後面加上個位數（1–9）的中文組成。

| 數字 | 中文 |
| :--- | :--- |
| 60 | `六十 (liù shí)` |
| 61 | `六十一 (liù shí yī)` |
| 62 | `六十二 (liù shí èr)` |
| 63 | `六十三 (liù shí sān)` |
| 64 | `六十四 (liù shí sì)` |
| 65 | `六十五 (liù shí wǔ)` |
| 66 | `六十六 (liù shí liù)` |
| 67 | `六十七 (liù shí qī)` |
| 68 | `六十八 (liù shí bā)` |
| 69 | `六十九 (liù shí jiǔ)` |
