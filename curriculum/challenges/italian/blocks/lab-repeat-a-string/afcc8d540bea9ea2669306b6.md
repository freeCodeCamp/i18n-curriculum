# --description--

In questo laboratorio, creerai una funzione che ripete una stringa data un numero specifico di volte. Per questo laboratorio, non usare il metodo integrato `.repeat()`.

**Obiettivo:** Soddisfa le storie utente qui sotto e fai superare tutti i test per completare il laboratorio.

**Storie utente:**

1. Dovresti creare una funzione chiamata `repeatStringNumTimes` che prende due parametri: una stringa e un numero.
2. La funzione dovrebbe restituire la stringa ripetuta il numero specificato di volte.
3. Se il numero è minore o uguale a zero, la funzione dovrebbe restituire una stringa vuota.

# --solutions--

```js
function repeatStringNumTimes(str, num) {
  if (num < 1) return '';
  return num === 1 ? str : str + repeatStringNumTimes(str, num - 1);
}

repeatStringNumTimes('abc', 3);
```
