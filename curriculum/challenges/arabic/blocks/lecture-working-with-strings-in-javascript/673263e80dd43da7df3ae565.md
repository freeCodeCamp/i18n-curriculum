# --interactive--

عند العمل مع السلاسل النصية في JavaScript، قد تحتاج أحيانًا إلى تحديد موقع سلسلة نصية فرعية معينة داخل سلسلة نصية أكبر.

السلسلة النصية الفرعية هي متتالية من الأحرف تظهر داخل سلسلة نصية أكبر. على سبيل المثال، في السلسلة النصية `hello world`، تعتبر `hello` و`world` سلاسل نصية فرعية.

لتحديد موقع سلسلة نصية فرعية داخل سلسلة نصية، يمكنك استخدام طريقة `indexOf()`.
تسمح طريقة `indexOf()` في JavaScript بالبحث عن سلسلة نصية فرعية داخل سلسلة نصية.

إذا وُجدت السلسلة النصية الفرعية، تُرجع `indexOf()` الفهرس (أو الموقع) لأول ظهور لتلك السلسلة. إذا لم تُوجد السلسلة النصية الفرعية، تُرجع `indexOf()` القيمة `-1`، مما يشير إلى أن البحث لم ينجح.

تأخذ طريقة `indexOf()` معلمتين: الأولى هي السلسلة النصية الفرعية التي تريد إيجادها داخل السلسلة الأكبر، والثانية هي موقع بداية اختياري للبحث. إذا لم تحدد موقع بداية، يبدأ البحث من بداية السلسلة.

في هذا السياق، **المعلمة** هي قيمة تعطيها إلى دالة أو طريقة عند استدعائها، مما يمكّن تلك الدالة أو الطريقة من أداء مهمتها باستخدام المعلومات المحددة التي تقدمها. ستتعلم المزيد عن المعلمات في الدروس القادمة.

إليك مثالًا على استخدام طريقة `indexOf()` لإيجاد موقع السلسلة النصية `awesome`:

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("awesome!");
console.log(position); // 14
```

:::

في هذا المثال، تبدأ كلمة `awesome` عند الفهرس `14` في السلسلة النصية `JavaScript is awesome!`، لذا تُرجع طريقة `indexOf()` القيمة `14`.

الآن، لنرَ ماذا يحدث عندما لا تُوجد السلسلة النصية الفرعية:

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("fantastic");
console.log(position); // -1
```

:::

بما أن كلمة `fantastic` لا تظهر في السلسلة النصية، تُرجع الطريقة القيمة `-1`.

يمكنك أيضًا تحديد مكان بدء البحث داخل السلسلة النصية بتوفير معلمة ثانية إلى `indexOf()`. إليك مثالًا:

:::interactive_editor

```js
let sentence = "JavaScript is awesome, and JavaScript is powerful!";
let position = sentence.indexOf("JavaScript", 10);
console.log(position); // 27
```

:::

في هذه الحالة، يبدأ البحث عن `JavaScript` بعد الحرف العاشر، لذا يُوجد الظهور الثاني لـ `JavaScript` عند الفهرس `27`.

من المهم ملاحظة أن طريقة `indexOf()` تميز بين الحروف الكبيرة والصغيرة.

في هذا المثال، ستُرجع القيمة `-1` لأن الحرف الكبير `F` غير موجود في السلسلة النصية `freeCodeCamp`.

:::interactive_editor

```js
console.log("freeCodeCamp".indexOf("F")) // -1
```

:::

يمكن أن يكون استخدام `indexOf()` مفيدًا جدًا عندما تحتاج إلى التحقق من وجود سلسلة نصية فرعية داخل سلسلة نصية وتحديد موقعها لإجراء عمليات لاحقة.
