# --explanation--

`Make sure` bedeutet, Maßnahmen zu ergreifen, um zu bestätigen, dass etwas korrekt oder abgeschlossen ist. Zum Beispiel:  

`Please make sure the report is submitted before the deadline.` – Das bedeutet, zu überprüfen und sicherzustellen, dass der Bericht rechtzeitig abgegeben wird.  

`Reliable` bedeutet, dass etwas oder jemand vertrauenswürdig ist und über die Zeit gut oder beständig funktioniert. Zum Beispiel:

`The server is reliable; it has been running without any issues for months.` – Das bedeutet, der Server funktioniert zuverlässig und verursacht keine Probleme.

`More reliable` bedeutet, dass etwas noch vertrauenswürdiger oder zuverlässiger geworden ist. Zum Beispiel:

`The new software update makes the app more reliable.` – Das bedeutet, die App ist nach dem Update vertrauenswürdiger oder stabiler geworden.
