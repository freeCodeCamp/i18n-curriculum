# --description--

さて、ここまでかなりの内容を学びました。残っているのは、これらのCSSをHTMLに追加する方法を説明することだけです。方法は3つあります。

外部CSSは最も一般的な方法で、CSS用の別ファイルを作成し、HTMLの開閉`<head>`タグ内に`<link>`要素（これはvoid要素で終了タグは不要）でリンクします。

まず、HTMLファイルの開閉`<link>`タグ内にvoid要素の`<head>`タグを追加します。`href`属性はCSSファイルの場所を指定し、絶対URLか、ここで使うようにHTMLファイルの場所に対する相対URLです。上の例では両方のファイルが同じディレクトリにあると仮定しています。`rel`属性は必須で、HTMLファイルとリンクされたファイルの関係を指定します。

次に、新しく作成した`styles.css`ファイル内には、セレクター（`div`と`p`）があり、その後に開閉の波括弧で囲まれた「宣言ブロック」が続きます。最後に宣言ブロック内に宣言を記述します。`color: white; `は1つの宣言で、`color`がプロパティ、`white`が値、`background-color: black;`は別の宣言です。

ファイル名についての注意：`styles.css`はここで使ったファイル名です。ファイルタイプが`.css`であれば好きな名前を付けられますが、「style」や「styles」が最もよく使われます。

この方法の利点は以下の通りです：

1. HTMLとCSSが分離されるため、HTMLファイルが小さくなり見た目もすっきりします。
2. CSSを1か所だけ編集すればよく、似たスタイルを共有する複数ページのウェブサイトで特に便利です。
