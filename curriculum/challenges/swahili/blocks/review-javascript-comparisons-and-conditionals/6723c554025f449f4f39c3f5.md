## Kulinganisha na aina za data `null` na `undefined`

- **Kulinganisha na `undefined`**: Kigezo ni `undefined` wakati kime tangazwa lakini hakijapewa thamani. Hii ni thamani ya msingi ya vigezo visivyotangazwa na vigezo vya kitendakazi ambavyo havikupatiwa hoja. `undefined` hubadilika kuwa `NaN` katika muktadha wa nambari, jambo linalofanya kulinganisha nambari zote na `undefined` kurudisha `false`.

:::interactive_editor

```js
console.log(undefined < 0); // false (NaN < 0 is false)
console.log(undefined >= 0); // false (NaN >= 0 is false)
```

:::

- **Kulinganisha na `null`**: Aina ya `null` inawakilisha kutokuwepo kwa thamani kwa makusudi. `null` hubadilika kuwa `0` katika muktadha wa nambari, jambo linaloweza kusababisha tabia zisizotarajiwa katika kulinganisha nambari:

:::interactive_editor

```js
console.log(null < 0); // false (0 < 0 is false)
console.log(null >= 0); // true (0 >= 0 is true)
```

:::

- Unapotumia opereta ya usawa (`==`), `null` na `undefined` ni sawa tu na kila mmoja na nafsi zao:

:::interactive_editor

```js
console.log(null == undefined); // true
console.log(null == 0); // false
console.log(undefined == NaN); // false
```

:::

- Hata hivyo, unapotumia opereta ya usawa mkali (`===`), inayokagua thamani na aina bila kulazimisha aina, `null` na `undefined` si sawa:

:::interactive_editor

```js
console.log(null === undefined); // false
```

:::

## Kifungu cha `switch`

- **Ufafanuzi**: Kifungu cha `switch` kinakagua usemi wa kihesabu na kulinganisha thamani yake na mfululizo wa vifungu vya `case`. Wakati patakapopatikana mlingano, sehemu ya msimbo inayohusiana na kesi hiyo hufanyika. Kifungu cha `break` kinapaswa kuwekwa mwishoni mwa kila kesi, ili kumaliza utekelezaji wake na kuendelea na inayofuata. Kesi ya `default` ni kesi ya hiari na hufanyika tu ikiwa hakuna kesi nyingine inayolingana. Kesi ya `default` huwekwa mwishoni mwa kifungu cha `switch`.

:::interactive_editor

```js
const dayOfWeek = 3; 

switch (dayOfWeek) {
  case 1:
    console.log("It's Monday! Time to start the week strong.");
    break;
  case 2:
    console.log("It's Tuesday! Keep the momentum going.");
    break;
  case 3:
    console.log("It's Wednesday! We're halfway there.");
    break;
  case 4:
    console.log("It's Thursday! Almost the weekend.");
    break;
  case 5:
    console.log("It's Friday! The weekend is near.");
    break;
  case 6:
    console.log("It's Saturday! Enjoy your weekend.");
    break;
  case 7:
    console.log("It's Sunday! Rest and recharge.");
    break;
  default:
    console.log("Invalid day! Please enter a number between 1 and 7.");
}
```

:::
