# --description--

在深入讲解前，你需要先了解 CSS 中的 @规则。 @规则是向浏览器提供指令的语句。 你可以将其用于定义样式表的各个方面，例如媒体查询、关键帧、字体面等。  You'll learn more about concepts like media queries and keyframes in future lessons. 本次课程，你将学习 `@font-face` 规则。

通过 `@font-face`，你可以通过指定字体文件、格式以及其他重要属性来定义自定义字体，如粗细和样式。 这是基本语法。 你可以看到 `@font-face` 之后是一个用大括号括起来的区块：

```css
@font-face {
  /* Descriptors */
}
```

在大括号内，你需要加入描述符来定制字体。 让我们来看看最常用的几种方法。  The font-family descriptor specifies the name that you will use throughout the stylesheet to refer to that font. 例如，假设你定义了下面的 `@font-face` 规则。 它定义了 `font-family` 描述符，其值为 `MyCustomFont`：

```css
@font-face {
  font-family: "MyCustomFont";
}
```

在这种情况下，你需要在样式表中使用该名称，以便为 CSS 选择器匹配的所有元素指定自定义字体族。 例如，你可以将其用作 `font-family` 的值：

```css
body { 
  font-family: "MyCustomFont"; 
}
```

但是，要使 `@font-face` 规则有效，你还需要指定 `src`。 其中包含对字体资源的引用。 它基本上就是一个外部引用或本地安装的字体面列表，用逗号分隔。 它还可以包括有关字体资源格式和技术的提示。

在下面的代码中，定义了 `src` 。 该值是一个 URL 列表，用逗号分隔，分行排列，以提高可读性：

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2"),
    url("path/to/font.woff"),
    url("path/to/font.otf");
}
```

你可以调用 `url()` 函数在样式表中包含一个文件。 在这种情况下，我们包含的是字体文件。 你需要在括号和引号内写入文件路径，包括文件扩展名。 为提高可读性，可将每个资源写在不同的一行，但最后一行应以分号结束。

你还可以为每个字体资源指定格式。 这是可选项。 这是浏览器对字体格式的提示。 如果省略格式，将下载资源，并在下载后检测格式。 如果格式无效，将无法下载资源。

可能的字体格式包括 `collection`、`embedded-opentype`、`opentype`、`svg`、`truetype`、`woff` 和 `woff2`。

下面是一个字体格式示例。 注意我们是如何在括号和引号中写出具体格式的：

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
    url("path/to/font.otf") format("opentype"),
    url("path/to/font.woff") format("woff");
}
```

在这个例子中，你可以看到我们指定了 WOFF2 格式、OpenType 格式和 WOFF 格式。

woff stands for "Web Open Font Format." The difference between WOFF and WOFF2 is the algorithm used to compress the data. OpenType 是微软和 Adobe 开发的一种可扩展计算机字体格式，允许用户访问字体中的其他功能。 它广泛应用于各大操作系统。

除了指定格式外，还可以指定字体资源的技术。 这也是可选项。 下面是一个例子，我们在其中指定了第二种字体资源的技术。

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
    url("path/to/font.otf") format("opentype") tech(color-COLRv1),
    url("path/to/font.woff") format("woff");
}
```

这些就是 `@font-face` 规则的基本内容。 使用此@规则，你可以指定字体文件，为你的独特设计定义自定义字体。
