# --description--

让我们在以下示例函数的上下文中讨论参数和参数。

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

在 JavaScript 中，参数是函数声明中括号 `()` 之间列出的项。函数参数是你决定传递给函数的实际值。

在上面的例子中，函数定义写在第一行：`function favoriteAnimal(animal)`。参数 `animal` 位于括号内。你也可以轻松地将 `animal` 替换为 `pet`、`x` 或 `blah`。但在这种情况下，将参数命名为 `animal` 会为阅读你代码的人提供一些上下文，这样他们就不必猜测 `animal` 最终可能包含什么。
