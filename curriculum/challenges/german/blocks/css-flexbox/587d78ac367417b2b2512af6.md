# --description--

Manchmal füllen die Flex-Elemente innerhalb eines Flex-Containers nicht den gesamten Platz im Container aus. Es ist üblich, CSS anzuweisen, wie die Flex-Elemente auf eine bestimmte Weise ausgerichtet und verteilt werden sollen. Glücklicherweise bietet die `justify-content`-Eigenschaft mehrere Optionen dafür. Zuvor gibt es jedoch einige wichtige Begriffe, die Sie verstehen sollten, bevor Sie diese Optionen überprüfen.

<a href="https://www.freecodecamp.org/news/flexbox-the-ultimate-css-flex-cheatsheet/" target="_blank" rel="noopener noreferrer nofollow">Für weitere Informationen zu Flexbox-Eigenschaften lesen Sie hier</a>

Erinnern Sie sich, dass das Festlegen eines Flex-Containers als Zeile die Flex-Elemente nebeneinander von links nach rechts anordnet. Ein als Spalte eingestellter Flex-Container stapelt die Flex-Elemente vertikal von oben nach unten. In beiden Fällen wird die Richtung, in der die Flex-Elemente angeordnet sind, als **Hauptachse** bezeichnet. Bei einer Zeile ist dies eine horizontale Linie, die durch jedes Element verläuft. Bei einer Spalte ist die Hauptachse eine vertikale Linie durch die Elemente.

Es gibt mehrere Möglichkeiten, wie die Flex-Elemente entlang der Linie, die die Hauptachse ist, verteilt werden können. Eine der am häufigsten verwendeten ist `justify-content: center;`, die alle Flex-Elemente im Flex-Container zentriert ausrichtet. Weitere Optionen sind:

<ul><li><code>flex-start</code>: richtet die Elemente am Anfang des Flex-Containers aus. Bei einer Zeile schiebt dies die Elemente nach links im Container. Bei einer Spalte schiebt es die Elemente nach oben im Container. Dies ist die Standardausrichtung, wenn kein <code>justify-content</code> angegeben ist.</li><li><code>flex-end</code>: richtet die Elemente am Ende des Flex-Containers aus. Bei einer Zeile schiebt dies die Elemente nach rechts im Container. Bei einer Spalte schiebt es die Elemente nach unten im Container.</li><li><code>space-between</code>: richtet die Elemente in der Mitte der Hauptachse aus, wobei zusätzlicher Platz zwischen den Elementen verteilt wird. Das erste und letzte Element werden an den äußersten Rand des Flex-Containers geschoben. Zum Beispiel steht bei einer Zeile das erste Element an der linken Seite des Containers, das letzte Element an der rechten Seite, und der verbleibende Platz wird gleichmäßig unter den anderen Elementen verteilt.</li><li><code>space-around</code>: ähnlich wie <code>space-between</code>, aber das erste und letzte Element sind nicht an den Rändern des Containers fixiert, der Platz wird um alle Elemente herum verteilt, mit halbem Abstand an beiden Enden des Flex-Containers.</li><li><code>space-evenly</code>: verteilt den Platz gleichmäßig zwischen den Flex-Elementen mit vollem Abstand an beiden Enden des Flex-Containers.</li></ul>

# --instructions--

Ein Beispiel zeigt diese Eigenschaft in Aktion. Fügen Sie die CSS-Eigenschaft `justify-content` zum `#box-container`-Element hinzu und geben Sie ihr den Wert `center`.

**Bonus**  
Probieren Sie die anderen Optionen für die `justify-content`-Eigenschaft im Code-Editor aus, um deren Unterschiede zu sehen. Beachten Sie jedoch, dass nur der Wert `center` diese Aufgabe besteht.

# --hints--

Das `#box-container`-Element sollte eine `justify-content`-Eigenschaft mit dem Wert `center` haben.

```js
const boxContainer = document.querySelector('#box-container');
const justifyDirection =
  window.getComputedStyle(boxContainer)['justify-content'];
assert.strictEqual(justifyDirection, 'center');
```
