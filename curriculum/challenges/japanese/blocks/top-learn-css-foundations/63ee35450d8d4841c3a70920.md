# --description--

内部CSS（または埋め込みCSS）は、完全に別のファイルを作成するのではなく、CSSをHTMLファイル内に直接追加する方法です。内部メソッドでは、すべてのルールを開く`<style>`タグと閉じる`<style>`タグのペアの中に配置し、それをHTMLファイルの開く`<head>`タグと閉じる`<head>`タグの間に置きます。スタイルが`<head>`タグの中に直接配置されるため、外部メソッドで必要な`<link>`要素はもう必要ありません。

これらの違い以外は、構文は外部メソッドとまったく同じです（セレクター、波括弧、中括弧、宣言）：

```html
<head>
  <style>
    div {
      color: white;
      background-color: black;
    }

    p {
      color: red;
    }
  </style>
</head>
<body>...</body>
```

この方法はウェブサイトの単一ページにユニークなスタイルを追加するのに便利ですが、外部メソッドのように分離されていないため、ルールや宣言が多いとHTMLファイルがかなり大きくなる場合があります。

# --questions--    

## --text--

内部CSSメソッドと外部CSSメソッドの違いはどれですか？

## --answers--

外部メソッドはCSSルールを別のファイルに配置し、内部メソッドはCSSルールをHTMLファイル内に配置します。

---

内部メソッドはCSSをHTMLから分離し、外部メソッドはCSSをHTMLに直接埋め込みます。

---

内部メソッドはCSSをHTMLにリンクするために`<link>`要素を使い、外部メソッドはCSSをHTMLに直接埋め込みます。
