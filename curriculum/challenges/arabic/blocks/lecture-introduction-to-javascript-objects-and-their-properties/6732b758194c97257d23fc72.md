# --interactive--

في JavaScript، دالة البناء هي نوع خاص من الدوال تُستخدم لإنشاء الكائنات وتهيئتها. تُستدعى باستخدام الكلمة المفتاحية `new` ويمكنها تهيئة الخصائص والطرق على الكائن الذي تم إنشاؤه حديثًا.

في هذا الدرس، سنلقي نظرة على كيفية العمل مع دالة البناء `Object()`. دالة البناء `Object()` تُنشئ كائنًا جديدًا فارغًا. إليك مثالًا:

```js
new Object()
```

عندما تستدعي `new Object()`، تُعيد كائنًا جديدًا يمكن استخدامه لتخزين القيم.

يمكن استخدام دالة البناء `Object()` مع أو بدون الكلمة المفتاحية `new`. عند استدعائها كدالة بدون `new`، تتصرف بشكل مختلف حسب نوع القيمة التي تُمرر إليها. إليك مثالًا على استخدام دالة البناء `Object()` بدون الكلمة المفتاحية `new`:

:::interactive_editor

```js
const num = 42;
const numObj = Object(num); // Creates an object wrapper for the number

console.log(numObj);
console.log(typeof numObj); // "object"
```

:::

كما ترى في السطر الثاني من `console.log`، فإن `numObj` هو كائن. يحدث هذا لأننا استخدمنا دالة البناء `Object()` لتحويل ذلك المُدخَل الرقمي إلى كائن.

ماذا يحدث إذا حاولنا تمرير `null` أو `undefined` إلى دالة البناء `Object()`؟

:::interactive_editor

```js
const newObj = new Object(undefined);
console.log(newObj); // {}
```

:::

النتيجة ستكون كائنًا فارغًا. حالة استخدام أخرى لدالة البناء `Object()` هي عندما تعمل مع قيمة من نوع غير معروف وتحتاج إلى التأكد من أنها كائن. لنلقِ نظرة على المثال التالي:

:::interactive_editor

```js
function toObject(value) {
  if (value === null || value === undefined) {
    return {};
  }

  if (typeof value === "object") {
    return value;
  }

  return Object(value);
}

console.log(toObject(null));

console.log(toObject(true));

console.log(toObject([1, 2, 3]));
```

:::

في هذا المثال، لدينا دالة تُسمى `toObject`. الشرط الثاني سيتحقق مما إذا كانت القيمة من نوع كائن وسيُعيد القيمة إذا كان الشرط `true`. هذا الشرط يتحقق من الكائنات وكذلك المصفوفات لأن المصفوفات هي أنواع خاصة من الكائنات.

إذا لم يكن أي من الشرطين صحيحًا، تُعيد الدالة `Object(value)`، التي تحول المُدخَل إلى كائن. هذا يعمل مع القيم مثل الأرقام، والسلاسل النصية، والقيم المنطقية.

في معظم الأحيان لن تستخدم دالة البناء `Object()` لإنشاء كائنات جديدة لأنك ستستخدم صيغة الكائن الحرفي بدلاً من ذلك (مثلًا، `const objectLiteral = { name: "Beau" }`). لكن من الجيد فهم أساسيات العمل مع دالة البناء Object.
