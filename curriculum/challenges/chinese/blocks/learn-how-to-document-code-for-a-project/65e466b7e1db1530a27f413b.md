# --description--

在计算机文件系统中，`root directory` 指的是主目录或顶层目录。它就像文件柜的最上层抽屉，包含所有其他文件夹和文件。

`main folder` 是存储项目或一组文件的主要文件夹。

`subfolder` 就像顶层文件夹内的一个更小的抽屉，位于另一个文件夹内。

例如，在名为 `MyProject` 的项目中，`root directory` 将是 `MyProject` 文件夹，而其内的任何文件夹，如 `Images`，将是一个 `subfolder`。
