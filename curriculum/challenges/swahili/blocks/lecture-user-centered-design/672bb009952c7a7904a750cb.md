# --interactive--

**KUMBUKA**: Baadhi ya mifano ya mwingiliano inaweza kutumia CSS na JavaScript ambazo bado hujajifunza. Usijali kuhusu kuelewa msimbo wote. Lengo la mifano ni kukuonyesha mapitio ya dhana hizi za kubuni ili uelewe vyema jinsi mambo yanavyofanya kazi. Ili kuona mapitio, utahitaji kuwezesha mhariri wa mwingiliano.

Kuvinjari bila mwisho ni mtindo wa kubuni unaopakia maudhui zaidi wakati mtumizi anapovinjari chini kwenye ukurasa. Mara nyingi, hii hutumika kwenye tovuti za mitandao ya kijamii kama Twitter. Kwa mfano, ikiwa umeingia na unataka kuona tweet zaidi, unaweza kuvinjari chini na tweet zaidi zitapakiwa. Hii ni mfano wa kuvinjari bila mwisho.

Kuvinjari bila mwisho pia hutumika badala ya kugawanya maudhui katika kurasa. Kugawanya kurasa ni mtindo wa kubuni unaogawanya maudhui katika kurasa tofauti. Hii hutumika mara nyingi wakati kuna maudhui mengi ya kuonyesha. Mfano wa kugawanya kurasa ni unapotafuta kitu Google na kuona matokeo ya utafutaji kwenye kurasa nyingi. Kwa kugawanya kurasa, unalazimika kubofya kitufe ili kwenda kwenye ukurasa unaofuata. Kwa kuvinjari bila mwisho, unaendelea tu kuvinjari chini na maudhui zaidi yatapakiwa.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="pagination">
  <div class="post">Post 1</div>
  <div class="post">Post 2</div>
  <div class="post">Post 3</div>
  <!-- More posts will be on the next pages -->
</div>
<div class="pagination-controls">
  <button class="prev" disabled>Previous</button>
  <button class="next">Next</button>
</div>
<script src="index.js"></script>
```

```css
.pagination {
  display: flex;
  flex-direction: column;
  gap: 12px;
}

.post {
  padding: 12px;
  border: 1px solid #ccc;
  border-radius: 4px;
}

.pagination-controls {
  margin-top: 12px;
}

.pagination-controls button {
  padding: 8px 16px;
  margin-right: 8px;
}
```

```js
let currentPage = 1;
const postsPerPage = 3;
const totalPosts = 9; 
const totalPages = Math.ceil(totalPosts / postsPerPage);
const container = document.querySelector('.pagination');
const prevButton = document.querySelector('.prev');
const nextButton = document.querySelector('.next'); 

function renderPosts() {
  container.innerHTML = '';
  const start = (currentPage - 1) * postsPerPage;
  const end = start + postsPerPage;
  for (let i = start; i < end && i < totalPosts; i++) {
    const post = document.createElement('div');
    post.className = 'post';
    post.textContent = `Post ${i + 1}`;
    container.appendChild(post);
  }
  prevButton.disabled = currentPage === 1;
  nextButton.disabled = currentPage === totalPages;
}

prevButton.addEventListener('click', () => {
  if (currentPage > 1) {
    currentPage--;
    renderPosts();
  }
});

nextButton.addEventListener('click', () => {
  if (currentPage < totalPages) {
    currentPage++;
    renderPosts();
  }
});

renderPosts();
```

Jukumu lingine ni kuongeza kitufe cha "Rudi". Hii inampa mtumizi uwezo wa kurudi kwenye ukurasa uliopita bila kuvinjari tena kutoka juu kabisa. Hii huunda uzoefu bora wa mtumizi na huwapa udhibiti zaidi juu ya jinsi wanavyovinjari.

Wakati mwingine utaona miundo yenye kitufe cha "Rudi juu" kinachoelekeza watumizi kurudi juu kabisa ya ukurasa wa matokeo. Jukumu lingine ni kutoa kiashiria cha upakiaji. Watumizi wanapaswa kuwa na dalili wazi kwamba maudhui zaidi yanapakiwa; vinginevyo, wanaweza kufikiri ukurasa umevunjika.

```js
const container = document.querySelector('.infinite-scroll');
const loadingIndicator = document.querySelector('.loading-indicator');
const backToTopBtn = document.getElementById('back-to-top');

window.addEventListener('scroll', () => {
  if (window.scrollY > 400) {
    backToTopBtn.style.display = 'block';
  } else {
    backToTopBtn.style.display = 'none';
  }

  if (window.innerHeight + window.scrollY >= document.body.offsetHeight) {
    loadMorePosts();
  }
});

backToTopBtn.addEventListener('click', () => {
  window.scrollTo({ top: 0, behavior: 'smooth' });
});

function loadMorePosts() {
  loadingIndicator.style.display = 'block';
  setTimeout(() => {
    for (let i = 0; i < 3; i++) {
      const post = document.createElement('div');
      post.className = 'post';
      post.textContent = `Post ${container.children.length + 1}`;
      container.appendChild(post);
    }
    loadingIndicator.style.display = 'none';
  }, 1000);
}
```

Jukumu la mwisho ni kuhakikisha kijachini kinapatikana kwa mtumizi. Ikiwa kijachini kina taarifa muhimu, basi kinapaswa kupatikana kwa mtumizi wakati wote.

Kwa kumalizia, kuvinjari bila mwisho ni njia nzuri ya kuonyesha maudhui kwenye tovuti yako. Hata hivyo, unapaswa kuzingatia mbinu bora wakati wa kubuni kuvinjari bila mwisho ili uweze kutoa uzoefu bora zaidi kwa mtumizi.
