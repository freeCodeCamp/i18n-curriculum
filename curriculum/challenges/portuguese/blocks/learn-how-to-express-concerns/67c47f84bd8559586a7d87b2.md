# --explanation--

`To work hard` significa dedicar muito esforço.Note que o advérbio para `hard` neste caso também é `hard`, não `hardly`. `To hardly work` significa fazer quase nenhum trabalho, fazer pouco ou nenhum esforço, o que é o oposto do que Maria está dizendo sobre a equipe.Por exemplo:

- `She hardly trained for the marathon.` - Isso significa que ela não fez nenhum esforço para estar preparada para a maratona.

- `She trained hard for the marathon.` - Isso significa que ela se esforçou muito para se preparar.
