# --explanation--

**분수**는 전체 중 일부를 나타낼 때 사용합니다.

`One-fourth` (¼)는 네 개의 같은 부분 중 한 부분입니다. 다른 예시는 다음과 같습니다:

- ½ = `one-half`

- ⅓ = `one-third`

- ¾ = `three-fourths`
