# --description--

Auf der Parabel $y = \frac{x^2}{k}$ sind drei Punkte $A(a, \frac{a^2}{k})$, $B(b, \frac{b^2}{k})$ und $C(c, \frac{c^2}{k})$ gewählt.

Sei $F(K, X)$ die Anzahl der ganzzahligen Vierlinge $(k, a, b, c)$, so dass mindestens ein Winkel des Dreiecks $ABC$ 45° beträgt, mit $1 ≤ k ≤ K$ und $-X ≤ a &lt; b &lt; c ≤ X$.

Zum Beispiel gilt $F(1, 10) = 41$ und $F(10, 100) = 12\\,492$.

Finden Sie $F({10}^6, {10}^9)$.
