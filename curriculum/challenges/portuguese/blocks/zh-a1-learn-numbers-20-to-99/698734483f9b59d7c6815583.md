### type 24 — numbers (40–49)

# --description--

Agora vamos para os números na casa dos **40**.

Em chinês, o número **40** é chamado `四十 (sì shí)`.

Os números de **41 a 49** são formados adicionando o dígito das unidades (1–9) diretamente após  
`四十 (sì shí)` (40). O padrão é o mesmo dos números nas casas dos 20 e 30.

| Number | Chinese |
| :--- | :--- |
| 40 | `四十 (sì shí)` |
| 41 | `四十一 (sì shí yī)` |
| 42 | `四十二 (sì shí èr)` |
| 43 | `四十三 (sì shí sān)` |
| 44 | `四十四 (sì shí sì)` |
| 45 | `四十五 (sì shí wǔ)` |
| 46 | `四十六 (sì shí liù)` |
| 47 | `四十七 (sì shí qī)` |
| 48 | `四十八 (sì shí bā)` |
| 49 | `四十九 (sì shí jiǔ)` |
