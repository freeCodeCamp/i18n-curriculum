# --interactive--

Klassenselektoren sind ein wichtiger Bestandteil von CSS, da sie Entwicklern ermöglichen, mehrere Elemente mit demselben Klassenattribut anzusprechen und einheitliche Formatierungen anzuwenden.

Das macht sie sehr vielseitig und effizient, um Stile auf einer gesamten Website anzuwenden.

Klassenselektoren werden durch einen Punkt (`.`) gefolgt vom Klassennamen definiert. Sie können auf jedes Element im HTML-Dokument angewendet werden.

Hier ist ein Beispiel mit einem Klassenselektor:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="highlight">Example paragraph</p>
```

```css
.highlight {
  color: green;
}
```

:::

In diesem Beispiel wird bei jedem Element mit der Klasse `highlight` die Textfarbe auf `green` gesetzt.

Klassenselektoren haben eine höhere Spezifität als Typselektoren, aber eine niedrigere als ID-Selektoren und inline Styles.

Der Spezifitätswert für einen Klassenselektor ist `(0, 0, 1, 0)`. Das bedeutet, dass Klassenselektoren Typselektoren überschreiben können, aber von ID-Selektoren und inline Styles überschrieben werden können.

Klassenselektoren können mit anderen Selektoren kombiniert werden, um spezifischere Regeln zu erstellen.

Hier ist ein Beispiel, das einen Absatz-Typselektor mit einem Klassenselektor kombiniert:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="bold-text">Example paragraph</p>
<p class="bold-text">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p.bold-text {
  font-weight: bold;
}
```

:::

Diese Regel gilt nur für `p`-Elemente, die auch die Klasse `bold-text` haben, und macht deren Text fett.
