# --explanation--

何かについて完全に確信が持てないけれど、そうであるはずだと思う場合、`It's supposed to` + 動詞 の表現を使うことがあります。

これは、まだ確認していなくても、ある状態であることが期待されていることを表します。例えば：

- `The update is supposed to fix the bug.` - これは、アップデートがバグを修正するはずだと期待しているけれど、まだ確認していないことを意味します。
