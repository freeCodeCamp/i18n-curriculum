# --description--

正の整数 $N$ を $k$ 等分し、$r = \frac{N}{k}$ とすると、$N = r + r + \cdots + r$ となります。

これらの部分の積を $P$ とすると、$P = r × r × \cdots × r = r^k$ です。

例えば、11 を5等分すると、11 = 2.2 + 2.2 + 2.2 + 2.2 + 2.2 となり、このとき $P = {2.2}^5 = 51.53632$ となります。

$M(N)$ を、与えられた $N$ に対して $P$ の最大値とします。

実際、$N = 11$ の最大値は11を4等分することで得られ、$P_{max} = {(\frac{11}{4})}^4$ となります。つまり、$M(11) = \frac{14641}{256} = 57.19140625$ で、これは有限小数です。

しかし、$N = 8$ の場合は3等分することで最大値が得られ、$M(8) = \frac{512}{27}$ となり、これは無限小数です。

$M(N)$ が無限小数の場合は $D(N) = N$、有限小数の場合は $D(N) = -N$ と定義します。

例えば、$5 ≤ N ≤ 100$ の範囲での $\sum D(N)$ は2438です。

$5 ≤ N ≤ 10000$ の範囲での $\sum D(N)$ を求めてください。
