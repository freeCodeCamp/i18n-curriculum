# --interactive--

Ci sono due potenti metodi in JavaScript per gestire i dati JSON: `JSON.parse()` e `JSON.stringify()`. Questi metodi sono comunemente usati per convertire tra stringhe JSON e oggetti JavaScript.

`JSON.stringify()` serve a convertire un oggetto JavaScript in una stringa JSON. Questo è utile quando vuoi archiviare o trasmettere dati in un formato che può essere facilmente condiviso o trasferito tra sistemi.

Ecco come puoi usare il metodo `JSON.stringify()`:

:::interactive_editor

```js
const user = {
  name: "John",
  age: 30,
  isAdmin: true
};

const jsonString = JSON.stringify(user);
console.log(jsonString);
```

:::

Il metodo `JSON.stringify()` accetta anche un parametro opzionale chiamato replacer, che può essere una funzione o un array. Ecco un esempio di utilizzo di un array per il parametro opzionale replacer:

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

// result: {"firstName":"Jessica","country":"USA"}
console.log(JSON.stringify(developerObj, ["firstName", "country"]));
```

:::

In questo esempio, abbiamo un `developerObj` con quattro proprietà. Quando usiamo il metodo `JSON.stringify()`, possiamo passare un array come secondo parametro e specificare quali proprietà vogliamo convertire in stringa. Il risultato sarà un oggetto convertito in stringa contenente solo le proprietà `firstName` e `country`.

Un altro parametro opzionale per il metodo `JSON.stringify()` è il parametro spacer. Questo ti permette di controllare la spaziatura del risultato convertito in stringa:

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

console.log(JSON.stringify(developerObj, null, 2));

/* result
{
  "firstName": "Jessica",
  "isAwesome": true,
  "isMusician": true,
  "country": "USA"
}
*/
```

:::

La maggior parte delle volte non userai nessuno di questi parametri opzionali per il metodo `JSON.stringify()`, ma è comunque utile conoscerli.

Un altro metodo che userai molto nella programmazione è il metodo `JSON.parse()`. `JSON.parse()` converte una stringa JSON di nuovo in un oggetto JavaScript. Questo è utile quando recuperi dati JSON da un server web o da `localStorage` e hai bisogno di manipolare i dati nella tua applicazione. Imparerai di più su `localStorage` in una lezione futura.

Ecco un esempio di come lavorare con il metodo `JSON.parse()`:

:::interactive_editor

```js
const jsonString = '{"name":"John","age":30,"isAdmin":true}';
const userObject = JSON.parse(jsonString);
console.log(userObject);

// Result:
// { name: 'John', age: 30, isAdmin: true }
```

:::

Questo ti permette di lavorare con i dati nel tuo programma come un normale oggetto JavaScript, rendendo più facile manipolarli e usarli.
