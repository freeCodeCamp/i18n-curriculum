# --explanation--

Na frase `limit what other parts can see`, `what` introduz uma oração substantiva, que serve como objeto de `limit`.

Uma oração substantiva é uma oração que atua como um substantivo dentro de uma frase.Ele pode funcionar como o objeto ou o sujeito de uma frase.Normalmente começa com palavras como `that`, `what`, `whether`, `who`, `when`, `why`, `how`, etc.

Orações substantivas usam a ordem normal das palavras da frase (não a ordem de pergunta).Por exemplo:

- A equipe está verificando por que o sistema travou durante a atualização. - `Why the system crashed during the update` é a oração substantiva que atua como objeto de `checking`.
   
- `What the function returns` depende dos dados de entrada. - `What the function returns` é uma oração substantiva que atua como sujeito de `depends on`.
