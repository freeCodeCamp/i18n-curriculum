# --description--

`this`, `that`, `these`, `those` 같은 지시 대명사는 특정 사물이나 사람을 가리킬 때 사용합니다. `This`와 `these`는 화자 가까이에 있는 대상을 가리킵니다. `This`는 단수 항목에, `these`는 복수 항목에 사용합니다. `That`와 `those`는 더 멀리 있는 대상을 가리키며, `that`는 단수, `those`는 복수에 사용합니다.

예시:

`This is my book.` (화자 가까이에 있는 책을 가리킴)

`That is a car.` (화자에서 더 멀리 있는 차를 가리킴)

`These are my friends.` (화자 가까이에 있는 친구들을 가리킴)

`Those are the mountains.` (화자에서 먼 산들을 가리킴)
