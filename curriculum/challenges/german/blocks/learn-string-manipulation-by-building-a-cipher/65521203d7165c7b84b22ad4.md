# --description--

Eine Schleife ermöglicht es Ihnen, systematisch eine Folge von Elementen durchzulaufen und auf jedes einzelne Aktionen auszuführen.

In diesem Fall verwenden Sie eine `for`-Schleife. So können Sie `text` durchlaufen:

```py
for i in text:
```

`for` ist das Schlüsselwort, das den Schleifentyp angibt. `i` ist eine Variable, die nacheinander die Werte der Elemente in `text` annimmt. Die Anweisung endet mit einem Doppelpunkt, `:`.

Schreiben Sie unter der Zeile, in der Sie `alphabet` deklariert haben, eine `for`-Schleife, um `text` durchzulaufen. Verwenden Sie `i` als Schleifenvariable.

Dabei tritt ein Fehler im Terminal auf. Sie werden im nächsten Schritt mehr darüber erfahren.
