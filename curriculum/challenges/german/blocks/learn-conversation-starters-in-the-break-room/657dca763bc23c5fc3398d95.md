# --description--

`Probably` wird verwendet, wenn Sie denken, dass etwas wahrscheinlich passieren wird, aber nicht zu 100 % sicher sind.

Zum Beispiel könnten Sie sagen: `The code will probably run without errors after the update`, weil Sie daran glauben, aber es noch eine kleine Chance auf ein Problem gibt.
