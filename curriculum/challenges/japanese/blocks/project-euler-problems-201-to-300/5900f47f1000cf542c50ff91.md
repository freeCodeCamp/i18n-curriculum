# --description--

$1$より大きく10と互いに素な各整数$p$に対して、任意の正の整数$n$に対し次の関数で除算可能性を保つ正の除算乗数$m < p$が存在します。

$f(n) = (\text{$n$の末尾の1桁を除いた部分}) + (\text{$n$の末尾の1桁}) \times m$

つまり、$m$が$p$の除算乗数であれば、$f(n)$が$p$で割り切れるのは$n$が$p$で割り切れる場合に限ります。

（$n$が$p$よりずっと大きい場合、$f(n)$は$n$より小さくなり、$f$を繰り返し適用することで$p$の乗法的除算テストができます。）

例えば、113の除算乗数は34です。

$f(76275) = 7627 + 5 \times 34 = 7797$：76275も7797も113で割り切れます

$f(12345) = 1234 + 5 \times 34 = 1404$：12345も1404も113で割り切れません

10と互いに素で1000未満の素数の除算乗数の合計は39517です。10の7乗未満の10と互いに素な素数の除算乗数の合計はいくつでしょうか？
