# --explanation--

`Present Simple`時制では、否定形は`doesn't`または三人称単数主語（`isn't`）の場合は`he, she, it`を使って作られます。

- 動作動詞には`doesn't`を使い、何かが起きていないことを示します。例：`It doesn't work.`

- `isn't`の形には`to be`を使い、何かが真実でないことを示します。例：`isn't set up.`

この文では、機能が期待通りに動作しない理由を正しく説明するために、`doesn't`と`isn't`の両方が必要です。
