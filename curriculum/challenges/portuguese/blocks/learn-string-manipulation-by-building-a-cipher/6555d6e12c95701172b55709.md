# --description--

Funções podem ser chamadas com argumentos padrão.Um argumento padrão indica o valor que a função deve assumir se o argumento não for passado.Por exemplo, a função abaixo aceita três argumentos mas você pode chamá-la passando dois argumentos.O terceiro assumirá o valor especificado por padrão:

```py
def foo(a, b, c=value):
    <code>
```
