# --description--

All’interno di ciascuno dei tuoi nuovi elementi `th`, annida un elemento `span` con l’insieme `class` impostato su `sr-only year`. Assegna loro il seguente testo (in ordine): `2019`, `2020` e `2021`.

Dai al tuo terzo elemento `th` l’attributo `class` impostato su `current`.

Lascia vuoto l’elemento `td`. Questo elemento serve solo a garantire che la tua tabella abbia un layout a quattro colonne e a collegare le intestazioni alle colonne corrette.

# --hints--

Ognuno dei tuoi elementi `th` dovrebbe avere un elemento `span`.

```js
const ths = [...document.querySelectorAll('th')];
ths?.forEach(th => {
  assert.lengthOf(th?.children, 1);
  assert.strictEqual(th?.children?.[0]?.localName, 'span');
});
```

Ognuno dei tuoi nuovi elementi `span` dovrebbe avere l’attributo `class` impostato su `sr-only year`.

```js
const ths = [...document.querySelectorAll('th')];
ths?.forEach(th => {
  assert.isTrue(th?.children?.[0]?.classList?.contains('sr-only'));
  assert.isTrue(th?.children?.[0]?.classList?.contains('year'));
});
```

```js
assert.strictEqual(
  document.querySelectorAll('th')?.[0]?.children?.[0]?.textContent.trim(),
  '2019'
);
```

Il tuo secondo elemento `span` dovrebbe avere il testo `2020`.

```js
assert.strictEqual(
  document.querySelectorAll('th')?.[1]?.children?.[0]?.textContent.trim(),
  '2020'
);
```

Il tuo terzo elemento `span` dovrebbe avere il testo `2021`.

```js
assert.strictEqual(
  document.querySelectorAll('th')?.[2]?.children?.[0]?.textContent.trim(),
  '2021'
);
```

Il tuo terzo elemento `th` dovrebbe avere l’insieme `class` impostato su `current`.

```js
assert.isTrue(
  document
    .querySelector('table')
    ?.querySelectorAll('th')?.[2]
    ?.classList?.contains('current')
);
```

Il tuo elemento `td` dovrebbe essere vuoto.

```js
const table = document.querySelector('table');
const tableData = table.querySelectorAll('td')?.[0];
assert.strictEqual(tableData?.textContent.trim(), '');
assert.isEmpty(tableData?.children);
```
