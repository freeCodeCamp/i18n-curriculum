# --description--

Für eine positive ganze Zahl $n$ sei $σ_2(n)$ die Summe der Quadrate ihrer Teiler. Zum Beispiel,

$$σ_2(10) = 1 + 4 + 25 + 100 = 130$$

Finden Sie die Summe aller $n$, $0 &lt; n &lt; 64\\,000\\,000$, für die $σ_2(n)$ eine perfekte Quadratzahl ist.
