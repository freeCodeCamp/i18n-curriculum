<dfn>Fetch API</dfn> 是一个内置的 JavaScript 接口，用于向服务器发出网络请求。 它有一个 `fetch()` 方法，你可以使用它来发出 `GET`、`POST`、`PUT` 或 `PATCH` 请求。 在这个项目中，你将向一个 URL 发出 `GET` 请求，以获取一个包含 freeCodeCamp News 作者信息的 JSON 文件。

下面演示了如何使用 `fetch()` 方法发出 `GET` 请求：

```js
fetch("url-goes-here")
```

向此 URL 发出 `GET` 请求：`"https://cdn.freecodecamp.org/curriculum/news-author-page/authors.json"`。 不要用分号终止你的代码。
