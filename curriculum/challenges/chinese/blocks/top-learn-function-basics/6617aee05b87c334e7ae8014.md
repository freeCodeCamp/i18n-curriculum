# --description--

在函数内部声明的变量仅在函数内部可见。这称为变量的作用域，这种特定的作用域称为局部作用域。在函数外部声明的变量称为全局变量，并且在整个程序中可见。例如，以下代码将引起误差：

```js
function myFunction() {
  let x = 10;
}

console.log(x);
```

变量 `x` 在函数 `myFunction` 内部声明，函数外部不可见。代码将引起 `ReferenceError`，因为未定义 `x`。

如果你在函数外声明一个变量，它被称为全局变量，并且在整个程序中可见。例如，以下代码将起作用：

```js
let x = 10;

function myFunction() {
  console.log(x);
}

myFunction();
```

如果在函数内部声明了同名变量，则它会遮蔽外部的变量。
