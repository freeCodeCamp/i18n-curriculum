# --explanation--

`Might` 用來表達可能性或不確定性。它表示某事是可能的，但不確定。例如：

- `The system might crash if too many users log in at the same time。` - 有系統當機的可能性，但不一定會發生。

- `He might finish the coding task by the end of the day.`－他可能會在一天結束前完成這項任務，但並不確定。
