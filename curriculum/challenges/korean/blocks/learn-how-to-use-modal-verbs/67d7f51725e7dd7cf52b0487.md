# --explanation--

축약형 `can't`는 `cannot`의 줄임말로, 무언가를 할 수 없다는 뜻입니다.  

`Can't` (`cannot`) + 기본 동사는 무언가가 불가능하거나 이해하기 어려울 때 사용합니다. 예를 들어:

- `I can't believe this happened!` - 믿기 어렵다.

- `She can't access the system.` - 그녀는 할 수 없다.
