# --description--

给定一个边长为整数的三角形 $ABC$，满足 $BC ≤ AC ≤ AB$。$k$ 是角 $ACB$ 的角平分线。$m$ 是 $C$ 点处三角形 $ABC$ 外接圆的切线。$n$ 是通过 $B$ 且与 $m$ 平行的直线。

$n$ 和 $k$ 的交集称为 $E$。

存在多少个三角形 $ABC$，其周长不超过 $100\,000$，且 $BE$ 的长度为整数？
