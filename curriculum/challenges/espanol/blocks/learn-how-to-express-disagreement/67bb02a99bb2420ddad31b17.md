# --explanation--

`That's a good suggestion, but` es una forma de reconocer una idea mientras introduces una preocupación. Esta frase permite una discusión respetuosa manteniendo las preocupaciones abiertas para una conversación posterior. Por ejemplo:

- Persona 1: `How about we use this new software to improve productivity?`

- Persona 2: `That's a good suggestion, but we should check if it's compatible with our current systems.` - Esta persona reconoce la idea pero destaca un posible problema de compatibilidad.
