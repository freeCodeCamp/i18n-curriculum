# --interactive--

DOM にノードを追加したり削除したりする必要がある場合があり、使えるウェブ API がいくつかあります。

このレッスンでは、`appendChild()` と `removeChild()` メソッドについて説明します。

`appendChild()` メソッドは、指定した親ノードの子ノードのリストの最後にノードを追加するために使われます。

`appendChild()` メソッドの基本的な構文は次の通りです：

```js
parentNode.appendChild(newNode);
```

`appendChild()` メソッドの使い方をよりよく理解するために、例を見てみましょう。

この例では、HTML 内に ID が `desserts` の番号なしリスト要素があります。このリストには `Cake` と `Pie` の2つのリスト項目が含まれています：

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
```

JavaScript ファイル内でそのリストにアクセスするには、`getElementById()` メソッドを使います：

```js
const dessertsList = document.getElementById("desserts");
```

`dessertsList` という変数に、ID が `ul` の `desserts` 要素の参照を格納しています。

次に、`createElement()` メソッドを使って新しいリスト項目要素を作成する必要があります：

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");
```

以下のコードは新しいリスト項目要素を作成し、`listItem` という変数に格納します。

`ul` 要素の子ノードのリストの最後にそのノードを追加するには、`appendChild()` メソッドを使います：

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

dessertsList.appendChild(listItem);
```

このコードを実行すると、新しいリスト項目要素が `ul` 要素の子ノードのリストの最後に追加されるのがわかります。

:::interactive_editor

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
<script src="index.js"></script>
```

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

dessertsList.appendChild(listItem);
```

:::

問題は、新しいリスト項目要素が空であることです。新しいリスト項目要素にテキストコンテンツを追加するには、`textContent` プロパティを使います：

:::interactive_editor

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
<script src="index.js"></script>
```

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

listItem.textContent = "Cookies";
dessertsList.appendChild(listItem);
```

:::

これでリストには `Cake`、`Pie`、`Cookies` が表示されます。

DOM からノードを削除するには、`removeChild()` メソッドを使います。

`removeChild()` メソッドの基本的な構文は次の通りです：

```js
parentNode.removeChild(childNode);
```

この `section` 要素から最後の段落要素を削除したい例を見てみましょう：

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
```

JavaScript ファイル内で `section` 要素にアクセスするには、`getElementById()` メソッドを使います：

```js
const sectionEl = document.getElementById("example-section");
```

`section` 要素の参照を取得したら、`querySelector()` メソッドを使って最後の段落にアクセスできます：

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");
```

`:last-of-type` 疑似クラスを使って `section` 要素内の最後の段落要素を選択しています。

親ノードと子ノードの参照が揃ったので、`section` 要素から最後の段落要素を `removeChild()` メソッドで削除できます：

:::interactive_editor

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
<script src="index.js"></script>
```

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");

sectionEl.removeChild(lastParagraph);
```

:::

では、いつ DOM にノードを追加したり削除したりしたいでしょうか？

動的なコンテンツを扱う場合、DOM にノードを追加・削除する必要があります。実際の例としては、ショッピングカート、やることリスト、ソーシャルメディアのフィードなどがあります。
