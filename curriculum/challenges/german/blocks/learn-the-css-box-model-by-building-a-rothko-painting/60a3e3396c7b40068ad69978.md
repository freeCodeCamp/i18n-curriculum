# --description--

Verwenden Sie Außenabstände, um den Abstand außerhalb eines Elements anzupassen.

Verwenden Sie die `margin`-Eigenschaft, um dem `.frame`-Element einen vertikalen Außenabstand von `20px` und einen horizontalen Außenabstand von `auto` zu geben. Dadurch wird der Rahmen um 20 Pixel nach unten verschoben und horizontal auf der Seite zentriert.

# --hints--

Sie sollten die `margin`-Eigenschaft auf `20px auto` setzen.

```js
const hasMargin = new __helpers.CSSHelp(document).getCSSRules().some(x => x.style.margin === '20px auto');
assert(hasMargin);
```

Ihr `.frame`-Element sollte einen `margin`-Wert von `20px auto` haben.

```js
const frameMargin = new __helpers.CSSHelp(document).getStyle('.frame')?.getPropertyValue('margin');
assert(frameMargin === '20px auto');
```
