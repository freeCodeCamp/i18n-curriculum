# --interactive--

في CSS، تُعد وحدتا `vh` و `vw` وحدات نسبية لإطار العرض تسمح لك بتحديد حجم العناصر بناءً على أبعاد نافذة المتصفح. هذه الوحدات مفيدة بشكل خاص لإنشاء تصميمات متجاوبة تتكيف مع أحجام الشاشات المختلفة.

تشير `vh` إلى "ارتفاع إطار العرض"، ويمثل `1vh` نسبة 1% من ارتفاع إطار العرض.

وبالمثل، تشير `vw` إلى "عرض إطار العرض"، ويمثل `1vw` نسبة 1% من عرض إطار العرض.

هذا يعني أنه إذا ضبطت ارتفاع عنصر على `100vh`، فسيشغل كامل ارتفاع إطار العرض، بغض النظر عن الأبعاد الفعلية بالبكسل للجهاز.

تكون هذه الوحدات مفيدة جدًا عندما تريد إنشاء تخطيطات ملء الشاشة أو عناصر تحافظ على نسبة محددة من الشاشة.

على سبيل المثال، قد ترغب في استخدامها لإنشاء قسم رئيسي يملأ الشاشة بالكامل دائمًا (للتفاعل مع المثال، ستحتاج إلى تفعيل المحرر التفاعلي).

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<section class="hero">
  <h1>100vh / 100vw Example</h1>
  <p>This section fills the entire browser window.</p>
</section>
```

```css
body {
  margin: 0;
  font-family: sans-serif;
  border: 5px dashed #333;
}

.hero {
  height: 100vh;
  width: 100vw;
  background-color: #add8e6; 
  padding: 2em;
}

.hero h1 {
  font-size: 2em;
  margin-bottom: 0.5em;
}

.hero p {
  font-size: 1em;
}
```

:::

يضمن هذا الـ CSS أن يكون قسم البطل دائمًا بحجم إطار العرض بالضبط، بغض النظر عن حجم شاشة الجهاز.

يمكن أيضًا استخدام وحدات `vh` و `vw` في الطباعة لإنشاء أحجام نصوص متجاوبة.

فعّل المحرر التفاعلي وجرب تعديل حجم نافذة المعاينة لترى كيف يتغير حجم النص مع حجم إطار العرض:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h1>Responsive Heading</h1>
<p>The h1 heading scales with the viewport width.</p>
```

```css
h1 {
  font-size: 5vw;
}
```

:::

سيضبط هذا حجم خط عناصر `h1` إلى 5% من عرض إطار العرض، مما يسمح للنص بالتدرج بسلاسة مع حجم نافذة المتصفح.

من مزايا وحدات `vh` و `vw` أنها تستجيب لتغيرات حجم إطار العرض في الوقت الحقيقي. هذا يعني أنه إذا قام المستخدم بتغيير حجم نافذة المتصفح، ستتغير أحجام العناصر التي تستخدم هذه الوحدات تلقائيًا دون الحاجة لإعادة تحميل الصفحة. مع ذلك، من المهم استخدام هذه الوحدات بحكمة. على سبيل المثال، تحديد أحجام الخطوط فقط بوحدات `vw` قد يؤدي إلى صغر النص جدًا على الشاشات الضيقة أو كبره جدًا على الشاشات العريضة.

اعتبار آخر هو أنه على الأجهزة المحمولة، قد يتغير ارتفاع إطار العرض عندما تظهر أو تختفي شريط عنوان المتصفح، مما قد يسبب تغييرات غير متوقعة في التخطيط إذا استخدمت وحدات `vh` بشكل مكثف.

باختصار، تعد وحدات `vh` و `vw` أدوات قوية لإنشاء تخطيطات وعناصر متجاوبة تتكيف مع حجم إطار العرض. هي مفيدة بشكل خاص للأقسام التي تملأ الشاشة بالكامل، والحفاظ على نسب الأبعاد، وإنشاء تصميمات تتدرج بسلاسة. مع ذلك، يجب استخدامها بعناية وغالبًا مع تقنيات CSS أخرى لضمان أفضل تجربة مستخدم عبر جميع الأجهزة.
