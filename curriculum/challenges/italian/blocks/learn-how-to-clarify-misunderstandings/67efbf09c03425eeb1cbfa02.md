# --explanation--

`Have you + Past Participle + recently?` è una struttura interrogativa usata per chiedere se qualcosa è stato fatto o è accaduto nel passato recente. Per esempio:

`Have you written articles recently?` – Questa domanda chiede se la persona ha scritto articoli nelle ultime ore o giorni.
