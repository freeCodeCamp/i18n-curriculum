# --description--

学んだように、`repetir` は何かをもう一度言うことを意味します。代名詞 `lo` を加えると、すでに言及されたものを指します。この代名詞は特定のもの（単語、数字、メールアドレスなど）を置き換えます。つまり：

- `Repetir` = 繰り返す。

- `Repetirlo` = それを繰り返す。

`Repetirlo` は「そのもの」が文脈からすでにわかっている場合に使われます。同じパターンは他の動詞でも使えます：

- `¿Puedes escribirlo?` - それを書けますか？

- `¿Puedes decirlo otra vez?` - もう一度それを言えますか？

# --explanation--

`Lo` は会話ですでにわかっているものを置き換える直接目的語の代名詞です。不定詞に付けると一語になります：

- `repetir` + `lo` → `repetirlo`  - それを繰り返す。

- `escribir` + `lo` → `escribirlo` - それを書く。

- `decir` + `lo` → `decirlo` - それを言う。

この構造は、聞き手が「それ」が何を指すかすでに理解している場合によく使われます。
