# --description--

前のチャレンジでは、すべての動物に共通する振る舞いを定義した`supertype`という`Animal`を作成しました。

```js
function Animal() { }
Animal.prototype.eat = function() {
  console.log("nom nom nom");
};
```

このチャレンジと次のチャレンジでは、`Animal`のメソッドを`Bird`や`Dog`の中で再定義せずに再利用する方法を扱います。これは継承という手法を使います。このチャレンジでは最初のステップを扱います：`supertype`（親）のインスタンスを作成します。`Animal`演算子を使って`new`のインスタンスを作成する方法はすでに知っています。

```js
let animal = new Animal();
```

この構文を継承に使うといくつかの欠点がありますが、それはこのチャレンジの範囲では複雑すぎます。代わりに、これらの欠点がない別の方法を紹介します。

```js
let animal = Object.create(Animal.prototype);
```

`Object.create(obj)`は新しいオブジェクトを作成し、その新しいオブジェクトの`obj`を`prototype`に設定します。`prototype`はオブジェクトを作るための「レシピ」のようなものです。`prototype`の`animal`を`prototype`の`Animal`に設定することで、`animal`のインスタンスに`Animal`の他のインスタンスと同じ「レシピ」を与えることになります。

```js
animal.eat();
animal instanceof Animal;
```

ここでの`instanceof`メソッドは`true`を返します。

# --instructions--

`Object.create`を使って、`Animal`のインスタンスを2つ作り、それぞれ`duck`と`beagle`と名付けてください。
