# --explanation--

과거 특정 시점에 일어나지 않았던 행동을 설명할 때 부정형 `Past Continuous`가 사용됩니다. 구조는 다음과 같습니다:  
**`wasn't` / `weren't` + `-ing`로 끝나는 동사**. 예를 들어: 

- `He wasn't listening during the meeting.` - 이 문장은 단수형 `was`을 사용합니다.

- `They weren't following the instructions carefully.` - 이 문장은 복수형 `were`을 사용합니다.

이 문맥에서 `weren't considering`는 과거 어느 시점에 팀이 업데이트를 수행하면서 보안 영향을 고려하지 않았다는 뜻입니다.  

동사 `consider`는 결정을 내리거나 행동하기 전에 무언가를 신중하게 생각하는 것을 의미합니다. 예를 들어: 

`We need to consider all options before finalizing the project plan.` - 선택지를 신중히 생각해야 한다는 의미입니다.

Jake의 문장에서 `considering`는 팀이 보안 영향을 평가하거나 생각하는 것을 가리킵니다.
