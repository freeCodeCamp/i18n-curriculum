# --description--

このラボでは、文字列が指定されたターゲットの文字列で終わっているかどうかをチェックする関数を実装します。

**目的:** 以下のユーザーストーリーを満たし、すべてのテストに合格してラボを完了してください。

**ユーザーストーリー:**

1. `confirmEnding` という名前の関数を作成し、チェックする文字列と比較対象の文字列の2つのパラメータを受け取る必要があります。
2. 最初の文字列が2番目の文字列で終わっていれば `true` を返し、そうでなければ `false` を返す必要があります。
3. `.endsWith()` メソッドは使用せず、代わりに JavaScript の substring メソッドのいずれかを使って実装してください。

# --solutions--

```js
function confirmEnding(str, target) {
  return str.substring(str.length - target.length) === target;
}

confirmEnding('Bastian', 'n');
```
