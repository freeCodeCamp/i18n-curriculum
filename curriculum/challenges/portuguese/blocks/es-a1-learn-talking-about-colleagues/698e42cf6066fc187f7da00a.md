# --description--

O verbo `vivir` significa "morar". 

Quando se refere a outra pessoa ("ele"/"ela"), o verbo é conjugado como `vive`. Geralmente é seguido pela preposição `en` para indicar onde a pessoa mora, com esta estrutura:

**`Vive en` + Localização**

Julieta está respondendo à pergunta de Mateo, dizendo onde Valeria mora.
