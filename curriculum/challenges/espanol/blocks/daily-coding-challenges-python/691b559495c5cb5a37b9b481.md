# --description--

Dado un arreglo de elementos, devuelve el elemento que aparece con mayor frecuencia.

# --solutions--

```py
def most_frequent(arr):
    freq = {}
    max_count = 0
    most_elem = None

    for el in arr:
        freq[el] = freq.get(el, 0) + 1
        if freq[el] > max_count:
            max_count = freq[el]
            most_elem = el

    return most_elem
```
