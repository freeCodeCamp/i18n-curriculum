Essas duas palavras significam que algo é relevante ou afeta um grupo específico.

# --explanation--

`Does that apply to...` é usado para perguntar se uma regra ou requisito é relevante para um grupo específico.Por exemplo:

- `Does that apply to remote workers too?` - Isso é o mesmo para pessoas que trabalham de casa?

- `Does that apply to projects outside of our department?` - Essa regra também cobre projetos de outros departamentos?
