# --questions--

## --text--

どの文が比較級を正しく使っていますか？

## --answers--

`This task is more easy than the previous one.`  

### --feedback--

`more` は不要です。なぜなら比較級では `easy` が `easier` に変わるからです。

---

`This code runs more faster than before.` 

### --feedback--

`More faster` は間違いです。なぜなら `faster` はすでに比較級だからです。

---

`This algorithm is more efficient than the old one.` 

---

`The new update is gooder than the last one.`

### --feedback--

`good` の正しい比較級は `better` です。

## --video-solution--

3

# --explanation--

比較級を作るには、1音節の形容詞や副詞の語尾に `-er` を付けます。より長い単語の場合は、語頭に `more` を付けます。

`More efficient` が正しいのは、`efficient` がより長い形容詞だからです。

各選択肢に魔法の言葉 `than` が含まれていることに気づくかもしれません。これは比較に使われる接続詞です。通常、比較級の形容詞や副詞の後に続き、比較の第2要素を導入して、程度や量、好みの違いを示します。
