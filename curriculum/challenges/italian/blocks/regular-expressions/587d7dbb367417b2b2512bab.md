# --description--

La ricerca è utile. Tuttavia, puoi rendere la ricerca ancora più potente quando modifica (o sostituisce) anche il testo che corrisponde.

Puoi cercare e sostituire testo in una stringa usando `.replace()` su una stringa. Gli input per `.replace()` sono prima il pattern di espressione regolare che vuoi cercare. Il secondo parametro è la stringa per sostituire la corrispondenza o una funzione per fare qualcosa.

```js
let wrongText = "The sky is silver.";
let silverRegex = /silver/;
wrongText.replace(silverRegex, "blue");
```

La chiamata `replace` restituirebbe la stringa `The sky is blue.`.

Puoi anche consultare i gruppi di cattura nella stringa di sostituzione con i simboli del dollaro (`$`).

```js
"Code Camp".replace(/(\w+)\s(\w+)/, '$2 $1');
```

La chiamata `replace` restituirebbe la stringa `Camp Code`.

# --instructions--

Scrivi un pattern di espressione regolare `fixRegex` usando tre gruppi di cattura che cercherà ogni parola nella stringa `one two three`. Poi aggiorna la variabile `replaceText` per sostituire `one two three` con la stringa `three two one` e assegna il risultato alla variabile `result`. Assicurati di usare i gruppi di cattura nella stringa di sostituzione usando la sintassi del simbolo del dollaro (`$`).

# --hints--

Dovresti usare `.replace()` per cercare e sostituire.

```js
assert(__helpers.removeJSComments(code).match(/\.replace\(.*\)/));
```

La tua espressione regolare dovrebbe cambiare la stringa `one two three` nella stringa `three two one`

```js
assert(result === 'three two one');
```

Non dovresti modificare l’ultima riga.

```js
assert(__helpers.removeJSComments(code).match(/result\s*=\s*str\.replace\(.*?\)/));
```

`fixRegex` dovrebbe usare almeno tre gruppi di cattura.

```js
assert(new RegExp(fixRegex.source + '|').exec('').length - 1 >= 3);
```

`replaceText` dovrebbe usare stringhe di sottocorrispondenza racchiuse tra parentesi (cioè la n-esima sottocorrispondenza racchiusa tra parentesi, $n, corrisponde al n-esimo gruppo di cattura).

```js
{
  const re = /(\$\d{1,2})+(?:[\D]|\b)/g;
  assert(replaceText.match(re).length >= 3);
}
```
