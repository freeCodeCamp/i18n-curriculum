# --description--

`r` を ${(a − 1)}^n + {(a + 1)}^n$ を $a^2$ で割ったときの余りとします。

例えば、$a = 7$、$n = 3$ の場合、$r = 42$ です。$6^3 + 8^3 = 728 ≡ 42 \\ \text{mod}\\ 49$ となります。そして `n` が変わると `r` も変わりますが、$a = 7$ のときは $r_{max} = 42$ となります。

$3 ≤ a ≤ 1000$ の範囲で、$\sum{r}_{max}$ を求めてください。
