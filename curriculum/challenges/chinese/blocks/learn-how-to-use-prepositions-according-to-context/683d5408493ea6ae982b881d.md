# --questions--

## --text--

哪个句子正确使用了单词 `near`？

## --answers--

`The developer near fixed the bug this morning.`

### --feedback--

`Near` 不应以这种方式用在动词前。

---

`He near completed the project.`

### --feedback--

这不是正确的结构。在此上下文中使用 `almost` 或其他词，而不是 `near`。

---

`The printer is near the manager's office.`

---

`She is near finishing the user guide.`

### --feedback--

该短语应为 `almost finished` 或 `close to finishing`，而不是 `near finishing`。

## --video-solution--

3

# --explanation--

`Near` 用于描述对象或地点之间的物理接近或短距离。

在正确的句子中，它清楚地显示了打印机相对于经理办公室的位置。这是将 `near` 作为介词来描述位置的预期且正确的用法。
