# --explanation--

`user interface` (보통 `UI`로 줄여 부릅니다)는 디지털 제품의 시각적이고 상호작용하는 부분으로, 사용자가 클릭하거나 보고, 터치하는 부분입니다. 예를 들어:

`The user interface includes the navigation bar and search field.` — 사용자가 보는 앱 부분에는 메뉴와 검색 상자 같은 것이 있습니다.

`consume`는 시간, 에너지, 예산 같은 것을 사용한다는 뜻입니다. 예를 들어:

`Testing often consumes a large part of the schedule.` — 테스트는 보통 프로젝트 계획에서 많은 시간을 차지합니다.
