# --description--

다른 유용한 배열 메서드 중 하나는 `filter()` 메서드입니다. `filter()` 메서드는 제공된 함수로 구현된 테스트를 통과하는 모든 요소로 새 배열을 만듭니다. `filter()` 메서드의 구문은 다음과 같습니다:

```javascript
const numbers = [2, 5, 6, 1, 9, -1]

const newNumbers = numbers.filter((number) => {
  return number > 2;
});

console.log(newNumbers); // Output: [5, 6, 9]
```

`filter()` 메서드는 2보다 큰 모든 요소로 새 배열을 만듭니다. `filter()` 메서드는 원래 배열을 변경하지 않습니다.
