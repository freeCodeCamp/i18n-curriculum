# --description--

Sea $f_5(n)$ el entero más grande $x$ para el cual $5^x$ divide a $n$.

Por ejemplo, $f_5(625\,000) = 7$.

Let $T_5(n)$ be the number of integers $i$ which satisfy $f_5((2 \times i - 1)!) &lt; 2 \times f_5(i!)$ and $1 ≤ i ≤ n$.

Se puede verificar que $T_5({10}^3) = 68$ y $T_5({10}^9) = 2\,408\,210$.

Encuentra $T_5({10}^{18})$.
