# --description--

`Double check` 是指再次查看某物以確保其正確。 例如，你可以說 `I wanted to be sure I locked the door, so I double checked it`。 這表示你正在小心避免錯誤，因此你正在再次檢查你所做的事情。

`schedule` 是一個顯示事情何時發生的計畫。 例如，`school schedule` 顯示課程何時開始和結束。 它幫助你知道一天中不同時間該做什麼。 在專案中，你可以用它來估算專案某些零件開始和結束的天數。

`Accurate` 意味著某事是正確且沒有錯誤。 例如，如果時鐘顯示正確時間，它就是 `accurate`。 這表示資訊是真實且精確的。
