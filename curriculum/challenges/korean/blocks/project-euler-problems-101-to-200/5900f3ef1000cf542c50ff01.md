# --description--

모두 1로만 이루어진 수를 반복 단위수라고 합니다. $R(k)$를 길이가 $k$인 반복 단위수로 정의하겠습니다; 예를 들어, $R(6) = 111111$입니다.

$n$이 양의 정수이고 $GCD(n, 10) = 1$일 때, 항상 $R(k)$가 $n$으로 나누어지는 어떤 값 $k$가 존재함을 증명할 수 있으며, $A(n)$을 그러한 $k$ 중 가장 작은 값으로 정의합니다; 예를 들어, $A(7) = 6$이고 $A(41) = 5$입니다.

$A(n)$이 처음으로 10을 초과하는 가장 작은 $n$의 값은 17입니다.

$A(n)$이 처음으로 백만을 초과하는 가장 작은 $n$의 값을 찾으세요.
