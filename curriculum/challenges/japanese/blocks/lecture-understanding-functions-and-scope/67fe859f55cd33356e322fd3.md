# --description--

関数は呼び出すと実行される再利用可能なコードのかたまりです。多くのプログラミング言語には、始めやすくするための組み込み関数が用意されています。Pythonも例外ではなく、これまでのレッスンで`print()`のような組み込み関数を扱いました。

もう一つ便利な組み込み関数が`input()`で、ユーザーに入力を促すことができます。

```python
name = input('What is your name?') # User types "Kolade" and presses Enter  
print('Hello', name) # Output: Hello Kolade
```

一方、`int()`は数値、ブール値、数値の文字列を整数に変換します。

```python
print(int(3.14)) # 3
print(int('42')) # 42
print(int(True)) # 1
print(int(False)) # 0 
```

また、自分でカスタム関数を書くこともできます。そのためには、`def`キーワードの後に関数名、丸括弧、コロンを続けて書きます。次の行に関数が実行するコードを書きます。関数が実行するコードは関数の本体とも呼ばれます。

以下は、`hello`という名前のカスタム関数の例で、文字列`Hello World`をターミナルに出力します。

```python
def hello():
    print('Hello World')
```

関数を実行するには、関数名の後に丸括弧を付けて呼び出します。

```python
hello() # Hello World
```

`print('Hello World')`の前のインデントに注目してください。これまでのレッスンで学んだように、Pythonはどの文のグループが一緒に属するかをインデントで判断します。これらの文のグループはコードブロックと呼ばれます。

次は、2つの数の合計をターミナルに出力するシンプルな関数です。

```python
def calculate_sum(a, b):
    print(a + b)
```

関数`calculate_sum`の丸括弧内に`a`と`b`がカンマで区切られているのがわかります。これらはパラメータと呼ばれます。パラメータは、関数を呼び出すときに渡す値の「スロット」として機能するプレースホルダー変数のようなものです。

パラメータを使うには、呼び出し時に「引数」を渡す必要があります。引数は関数に渡す値のことです。

`calculate_sum`関数を呼び出して、`3`と`1`の数を合計する方法は以下の通りです。

```python
calculate_sum(3, 1) # 4
```

正しい数の引数を渡さずに関数を呼び出すと、`TypeError`が発生します。

```python
calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

関数は特別な`return`キーワードを使って関数を終了し、値を返すこともできます。明示的に`return`を使わない場合、Pythonはデフォルトで`None`を返します。

例を示します。

```python
def calculate_sum(a, b):
    print(a + b)

my_sum = calculate_sum(3, 1) # 4
print(my_sum) # None
```

`calculate_sum`関数は`a`と`b`の合計を出力しますが、明示的に何も返していません。そのため、結果を`my_sum`に代入すると、実際には`None`になります。これを修正するには、結果を返すために`return`キーワードを使います。

```python
def calculate_sum(a, b):
    return a + b

my_sum = calculate_sum(3, 1)
print(my_sum) # 4
```

これで`calculate_sum`は`a`と`b`の合計を返し、その値が`my_sum`にストアされます。
