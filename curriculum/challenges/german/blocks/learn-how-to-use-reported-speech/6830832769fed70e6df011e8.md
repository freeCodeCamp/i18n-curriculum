# --explanation--

`To bring up` bedeutet, ein Thema im Gespräch zu erwähnen oder einzuführen. Die Vergangenheitsform des Verbs ist die unregelmäßige Form `brought`. Zum Beispiel:

`She brought up the budget during the meeting.` – Das bedeutet, sie hat das Thema Budget angesprochen.

Die Struktur `was brought up` steht im **Passiv**. Das bedeutet, jemand hat das Thema erwähnt, aber es wird nicht gesagt, wer es erwähnt hat. Der Fokus liegt auf der Handlung, nicht auf der Person. Zum Beispiel:

`The issue was brought up by several users.` – Das bedeutet, Nutzer haben das Problem erwähnt, aber der Fokus liegt auf dem Problem selbst.
