# --description--

音楽アルバムのコレクションのメンテナンスを助ける関数を作成しています。コレクションは複数のアルバムを含むオブジェクトとして整理されており、各アルバムもオブジェクトです。各アルバムはコレクション内で一意の`id`をプロパティ名として表しています。各アルバムオブジェクトの中には、アルバムに関するさまざまな情報を表すプロパティがあります。すべてのアルバムが完全な情報を持っているわけではありません。

`updateRecords`関数は、以下の関数パラメータで表される4つの引数を取ります：

- `records` - 複数の個別アルバムを含むオブジェクト
- `id` - `records`オブジェクト内の特定のアルバムを表す数値
- `prop` - 更新するアルバムのプロパティ名を表す文字列
- `value` - アルバムのプロパティを更新するための情報を含む文字列

**目的：** 以下のユーザーストーリーを満たし、すべてのテストを通過してラボを完了してください。

**ユーザーストーリー：**

1. 関数は常に`records`オブジェクト全体を返す必要があります。
2. `value`が空文字列の場合、指定された`prop`プロパティをアルバムから削除してください。
3. `prop`が`tracks`でなく、かつ`value`が空文字列でない場合、そのアルバムの`value`に`prop`を代入してください。
4. `prop`が`tracks`で、`value`が空文字列でないが、アルバムに`tracks`プロパティがない場合、空の配列を作成し`value`を追加してください。
5. propが`tracks`で、`value`が空文字列でない場合、アルバムの既存の`value`配列の末尾に`tracks`を追加してください。

**注意：** テストには`recordCollection`オブジェクトのコピーが使用されます。関数は`recordCollection`オブジェクトを直接参照せず、関数パラメータのみを使用してください。

# --solutions--

```js
const recordCollection = {
  2548: {
    albumTitle: 'Slippery When Wet',
    artist: 'Bon Jovi',
    tracks: ['Let It Rock', 'You Give Love a Bad Name']
  },
  2468: {
    albumTitle: '1999',
    artist: 'Prince',
    tracks: ['1999', 'Little Red Corvette']
  },
  1245: {
    artist: 'Robert Palmer',
    tracks: []
  },
  5439: {
    albumTitle: 'ABBA Gold'
  }
};

function updateRecords(records, id, prop, value) {
  if (value === '') delete records[id][prop];
  else if (prop === 'tracks') {
    records[id][prop] = records[id][prop] || [];
    records[id][prop].push(value);
  } else {
    records[id][prop] = value;
  }

  return records;
}
```
