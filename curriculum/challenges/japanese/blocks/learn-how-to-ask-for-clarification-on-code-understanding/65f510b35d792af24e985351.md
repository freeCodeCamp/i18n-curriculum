# --description--

プログラミングでは、`persistent` という用語は、特に解決が難しい問題やバグのように、継続して存在したり発生したりするものを指すことが多いです。

`Persistence` は名詞形で、持続している状態を指します。

例えば、コード内の`persistent error`は、繰り返し発生し修正が難しいエラーのことです。
