### --question--

#### --text--

Was sind Pseudoklassen?

#### --distractors--

Sie sind CSS-Regeln, die bei Bedarf zusätzlichen Inhalt innerhalb eines Elements erzeugen.

---

Sie sind Eigenschaften, die die Größe oder das Layout eines Elements auf der Seite anpassen.

Sie sind Schlüsselwörter, die einem Selektor hinzugefügt werden und ein Element basierend auf seinem Zustand stylen.
