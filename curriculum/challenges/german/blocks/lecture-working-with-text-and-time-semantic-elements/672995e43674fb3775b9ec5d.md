# --interactive--

Das Kontaktadress-Element wird verwendet, um Kontaktinformationen für einen Abschnitt auf einer Webseite darzustellen. Das `address`-Element ist vielseitig und kann für Unternehmensseiten, Autoren-Seiten, persönliche Websites und mehr verwendet werden.

Wenn Sie die Kontaktabschnitte Ihrer Website erstellen, sollten Sie das semantische `address`-Element anstelle eines generischen Elements wie eines `div` verwenden.

Hier ist ein Beispiel für die Verwendung des `address`-Elements auf einer Unternehmens-Kontaktseite:

:::interactive_editor

```html
<address>
  <h2>Company Name</h2>
  <p>
    1234 Elm Street<br />
    Springfield, IL 62701<br />
    United States
  </p>
  <p>Phone: <a href="tel:+15555555555">+1 (555) 555-5555</a></p>
  <p>Email: <a href="mailto:contact@company.com">contact@company.com</a></p>
</address>
```

:::

In diesem Beispiel gibt es den Firmennamen, die physische Adresse, Telefon- und E-Mail-Informationen. Für die physische Adresse wird das Zeilenumbruch-Element `br` verwendet, um die Trennung zwischen Straßenname, Stadt und Land anzuzeigen.

Für die Telefonnummer haben wir ein Ankerelement mit dem `href`-Wert, der für Telefonnummern festgelegt ist. Der `tel:+`-Wert innerhalb des `href`-Attributs erstellt einen anklickbaren Link, um auf bestimmten Geräten, die das unterstützen, einen Anruf zu starten.

Für die E-Mail-Adresse wird ein weiteres Ankerelement verwendet, bei dem der `href`-Wert auf einen `mailto`-Link gesetzt ist. `mailto`-Links werden in HTML-Dokumenten verwendet, damit Benutzer eine neue E-Mail in ihrem bevorzugten E-Mail-Client öffnen können.

Ein Nachteil der Verwendung eines `mailto`-Links ist, dass Benutzer ihn oft als Spam wahrnehmen. Leider nutzen viele Spammer diese Option, um E-Mails an Benutzer zu senden. Behalten Sie das also im Hinterkopf, wenn Sie ihn verwenden.
