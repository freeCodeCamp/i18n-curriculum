# --description--

类型选择器（或元素选择器）将选择给定元素类型的所有元素，其语法是：元素名 { css 代码 }：

```html
<!-- index.html -->

<div>Hello, World!</div>
<div>Hello again!</div>
<p>Hi...</p>
<div>Okay, bye.</div>
```

```css
/* styles.css */

div {
  color: white;
}
```

此处所有3个<div>`<div>`元素都将被选中，而`<p>`元素不被选中。
