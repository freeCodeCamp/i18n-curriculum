# --interactive--

Sehen wir uns an, wie Sie Knoten mit `innerHTML` und `createElement()` erstellen können.

Wir beginnen mit `innerHTML`.

`innerHTML` ist eine Eigenschaft von `Element`-Objekten, mit der Sie deren HTML-Auszeichnung festlegen können. Mit `innerHTML` können Sie die HTML-Struktur eines vorhandenen Elements mit einem String setzen und dabei alle notwendigen Knoten erstellen.

Dies ist ein Beispiel. Wir haben einen leeren `div` und fügen darin einige Elemente mit der `innerHTML`-Eigenschaft hinzu:

```html
<div id="container">
  <!-- Add new elements here -->
</div>
```

Zuerst müssen wir das Element anhand seiner ID auswählen:

```js
const container = document.getElementById("container");
```

Dann setzen wir die `innerHTML`-Eigenschaft des Containers auf einen String. Dieser String muss alle Auszeichnungen enthalten, die notwendig sind, um die Knoten und die Struktur darzustellen, die Sie erstellen möchten. Sie können es sich vorstellen wie das Schreiben von HTML innerhalb eines Strings.

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
container.innerHTML = "<ul><li>Cheese</li><li>Tomato</li></ul>";
```

:::

Wir setzen die `innerHTML` dieses Elements auf eine ungeordnete Liste von Pizzazutaten (Käse und Tomate).

Sie können den String auch über mehrere Zeilen schreiben, wenn dies die Auszeichnung leichter verständlich macht. Wichtig ist, dass die Auszeichnung die HTML-Struktur, die Sie erstellen möchten, genau darstellt.

Nachdem Sie diesen Code ausgeführt haben, sehen Sie beim Inspizieren Ihrer Auszeichnung die folgende HTML-Struktur.

```html
<div id="container">
  <ul>
    <li>Cheese</li>
    <li>Tomato</li>
  </ul>
</div>
```

Die neuen Knoten wurden erstellt und dynamisch zum DOM hinzugefügt, nachdem der String geparst wurde.

`innerHTML` kann in bestimmten Szenarien sehr hilfreich sein. Allerdings gibt es einige Sicherheitsrisiken, mit denen Sie vertraut sein sollten. Sie sollten es nicht verwenden, wenn Sie keine vollständige Kontrolle über den String haben.

Zum Beispiel sollten Sie `innerHTML` nicht verwenden, wenn der String vom Benutzer eingegeben wird, da der Benutzer möglicherweise schädlichen Inhalt auf Ihrer Website einfügen könnte. Deshalb wird normalerweise empfohlen, stattdessen `textContent` zu verwenden, um reinen Text einzufügen.

Eine andere Möglichkeit, einen neuen Knoten zu erstellen, ist die Verwendung der `createElement()`-Methode. Mit dieser Methode können Sie ein neues Element erstellen, indem Sie seinen Tag-Namen angeben.

Wenn Sie zum Beispiel ein Bild-Element erstellen möchten, übergeben Sie den `img`-Tag als String beim Aufrufen dieser Methode:

```js
document.createElement("img");
```

Und Sie können dieses neue Objekt einer Variablen zuweisen:

```js
const img = document.createElement("img");
```

Die `createElement()`-Methode gibt ein neues `HTMLElement`-Objekt zurück, wenn das Dokument ein `HTMLDocument` ist. Andernfalls gibt sie ein `Element`-Objekt zurück.

Sobald Sie dieses neue Element bereit haben, können Sie es mit der `appendChild()`-Methode als Kind eines anderen vorhandenen Elements zum DOM hinzufügen oder es mit anderen Methoden an einer bestimmten Stelle einfügen. Wählen Sie die Methode, die am besten zu Ihren Anforderungen passt.

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
const img = document.createElement("img");
img.src = "https://cdn.freecodecamp.org/curriculum/cat-photo-app/lasagna.jpg";
img.alt = "A slice of lasagna on a plate.";
container.appendChild(img);
```

:::

Das Erstellen von Knoten mit `innerHTML` und `createElement()` ermöglicht es Ihnen, die Struktur und den Inhalt Ihrer Websites dynamisch zu manipulieren. Durch die Kombination dieser Techniken können Sie interaktive Webanwendungen erstellen.
