# --description--

**Мета:** Виконайте наведені нижче історії користувача та пройдіть усі тести, щоб завершити лабораторну роботу.

**Історії користувача:**

1. У вас має бути функція з назвою `pyramid`, яка приймає три аргументи.
1. Перший аргумент має бути рядком, що представляє символ шаблону для повторення у вашій піраміді.
1. Другий аргумент має бути цілим числом, що представляє кількість рядків у піраміді.
1. Третій аргумент має бути булевим значенням.
1. Функція `pyramid` має повертати рядок, у якому символ шаблону повторюється та розташовується так, щоб утворити піраміду з вершиною, спрямованою вгору, коли третій аргумент дорівнює `false`.
1. Коли третій аргумент дорівнює `true`, піраміда має мати вершину, спрямовану вниз.
1. Рядок із вершиною має містити один символ шаблону, а кожен наступний рядок має містити на два символи шаблону більше, ніж попередній.
1. Кожен рядок має починатися з кількості пробілів, достатньої, щоб центр символів кожного рядка був у тій самій колонці, і в кінці рядка не має бути пробілів.
1. Піраміда має починатися і закінчуватися символом нового рядка.

Наприклад, виклик `pyramid("o", 4, false)` має дати такий вихід:

```js

   o
  ooo
 ooooo
ooooooo

```

function pyramid(char, count, isInverted) {
    const rows = []
    for (let i = 1; i <= count; i++) {
        if (isInverted) {
            rows.unshift(" ".repeat(count - i) + char.repeat(2 * i - 1))
        } else {
            rows.push(" ".repeat(count - i) + char.repeat(2 * i - 1))
        }        
    }
    return "\n" + rows.join("\n") + "\n";
}

console.log(pyramid("#", 10, false))
