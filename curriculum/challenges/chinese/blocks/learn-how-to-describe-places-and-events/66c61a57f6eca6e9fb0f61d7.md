# --explanation--

`Built-in` 指的是作为软件程序或设备的一部分从一开始就包含的特色或工具，无需单独添加或安装。 

当某物是 `built-in` 时，它已经存在且可供使用。例如：

- 这台笔记本电脑有一个内置摄像头。

- `The software comes with built-in security features.` - 这些安全性特色已包含在该软件中。
