# --explanation--

`Then` am Ende eines Satzes wird verwendet, um eine Schlussfolgerung, einen Vorschlag oder ein Ergebnis zu zeigen, das sich aus etwas zuvor Erwähntem ergibt. Es vermittelt oft die Idee eines Ergebnisses oder einer Konsequenz. Zum Beispiel:

`If it's too expensive, we'll wait until next month then.` – Das bedeutet, dass der Sprecher eine Entscheidung basierend auf dem zuvor Gesagten getroffen hat.
