# --description--

La palabra `what` se puede usar para introducir una cláusula. Típicamente introduce cláusulas sustantivas, proporcionando más información sobre un sustantivo o actuando como parte de una sentencia.

Una cláusula sustantiva es una cláusula que funciona como un sustantivo dentro de una oración más grande. Típicamente incluye un sujeto y un verbo, y puede actuar como sujeto, objeto o complemento en una oración.

Por ejemplo, en la oración `She doesn't know what to buy`, `what to buy` es una cláusula donde `what` representa la cosa que ella necesita decidir comprar.

Otro ejemplo es `Tell me what you heard.` `What you heard` es la cláusula, con `what` explicando las cosas específicas que se escucharon, actuando como el objeto del verbo `heard`.

`These` es un pronombre demostrativo plural que se refiere a elementos específicos mencionados previamente o entendidos del contexto.
