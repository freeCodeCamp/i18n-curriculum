# --explanation--

`If only` se usa al comienzo de una oración para expresar un deseo o arrepentimiento sobre algo que no ocurrió en el pasado. Indica que el hablante desearía que la situación hubiera sido diferente. Por ejemplo:

`If only we had checked the system earlier, we could have avoided the crash.` – Esto significa que no revisaron lo suficientemente temprano y ahora lo lamentan.
