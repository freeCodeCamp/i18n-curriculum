# --description--

Ora che conosci i colori secondari, imparerai come creare i <dfn>colori terziari</dfn>. I colori terziari si ottengono combinando un colore primario con un colore secondario vicino.

Per creare il colore terziario arancione, aggiorna la funzione `rgb` nella regola CSS `.one` in modo che il rosso sia al valore massimo e imposta il verde su `127`.

# --hints--

La tua regola CSS `.one` dovrebbe avere una proprietà `background-color` impostata su `rgb(255, 127, 0)`.

```js
assert(new __helpers.CSSHelp(document).getStyle('.one')?.backgroundColor === 'rgb(255, 127, 0)');
```
