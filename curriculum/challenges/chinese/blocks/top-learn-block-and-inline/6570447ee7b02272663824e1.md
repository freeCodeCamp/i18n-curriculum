# --description--

内边距用于在元素内容周围创建空间，位于任何定义的边框内部。对于内联元素，内边距会在内容的左侧、右侧、顶部和底部添加空间。然而，重要的是要注意，虽然内边距确实影响内容周围的空间，但它不会影响内联元素周围其他元素的版型。这意味着水平内边距（左侧和右侧）会被尊重，但垂直内边距（顶部和底部）不会导致其他元素移动。

`padding` 如何影响 `inline` 元素？

内联元素的填充仅影响水平间距。

内联元素的填充仅影响垂直间距。
