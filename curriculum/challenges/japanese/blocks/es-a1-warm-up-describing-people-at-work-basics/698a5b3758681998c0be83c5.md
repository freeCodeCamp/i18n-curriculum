# --description--

いくつかの記号は、日常の職場での活動に欠かせないものになっています。

スペイン語で最もよく使われる4つの記号は次の通りです：

| 記号 | スペイン語 | 意味 |
| :---: | :--- | :--- |
| `.` | `Punto` | ピリオド / ドット |
| `@` | `Arroba` | アットマーク |
| `-` | `Guión` | ハイフン / ダッシュ |
| `_` | `Guión bajo` | アンダースコア |

`Guión` という単語には、`o` の文字にアクセント記号が付いていることに注意してください。

カミラがこれらの記号の練習を手伝います。音を繰り返し聞いて慣れてから、それぞれの記号をスペイン語の名前と結びつけてください。

# --instructions--

音声を聞いて、記号と名前をマッチさせてください。
