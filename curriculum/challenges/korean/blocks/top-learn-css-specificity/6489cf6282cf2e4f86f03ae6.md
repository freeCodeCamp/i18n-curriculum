# --description--

```css
/* rule 1 */
#subsection .list {
  background-color: yellow;
  color: blue;
}

/* rule 2 */
#subsection .main .list {
  color: red;
}
```

이 마지막 예제에서는 두 규칙 모두 ID와 클래스 선택자를 사용하므로 어느 규칙도 더 구체적인 선택자를 사용하지 않습니다. 계단식 규칙은 각 선택자 유형의 개수를 확인합니다. 두 규칙 모두 ID 선택자가 하나뿐이지만, 규칙 2가 더 많은 클래스 선택자를 가지고 있어 규칙 2가 더 높은 특이성을 가집니다!

`color: red` 선언이 우선 적용되지만, `background-color: yellow` 선언은 충돌하는 선언이 없으므로 여전히 적용됩니다.
