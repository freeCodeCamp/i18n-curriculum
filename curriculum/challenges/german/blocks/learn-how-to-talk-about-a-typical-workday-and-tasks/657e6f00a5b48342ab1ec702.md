# --description--

Ein `QA (Quality Assurance) tester` ist jemand, der Software überprüft, um sicherzustellen, dass sie gut funktioniert und keine Probleme hat.

`Quality` bedeutet, wie gut oder schlecht etwas ist, und `assurance` bedeutet, sich über etwas zu vergewissern. Ein `QA tester` muss Software testen, um sicherzustellen, dass ihre Qualität gut ist und sie so funktioniert, wie sie soll.
