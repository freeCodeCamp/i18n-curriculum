# --interactive--

マージンとパディングは、CSSでよく構造化され、読みやすく、視覚的に魅力的なウェブページを作成するために欠かせないプロパティです。

マージンは要素の外側のスペースを制御し、他の要素との間隔を作ってレイアウト構造を定義します。一方、パディングは要素の内側のスペースを制御し、コンテンツの読みやすさと見た目の美しさを向上させます。

`margin` と `padding` の違いをよりよく理解するために、いくつかの例を見てみましょう。

以下はページ上の3つの段落要素のHTML例です：

:::interactive_editor

```html
<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

:::

各段落要素の上部にスペースを適用するには、`margin-top` プロパティをこのように使います：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  margin-top: 20px;
  border: 2px solid black;
}
```

:::

この例では、各段落要素の上部にだけ `20px` のマージンを適用しています。

また、マージンが見やすいように4辺すべてに黒の実線ボーダーを付けています。

4つの異なる `margin` プロパティは `margin-top`、`margin-right`、`margin-bottom`、`margin-left` です。

これら4つのプロパティを使った例がこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 30px;
  margin-left: 40px;
  border: 2px solid black;
}
```

:::

この例では、段落要素の4辺すべてにスペースの値を代入しています。マージンが見やすいように黒の実線ボーダーも追加しています。

`margin` プロパティの別の使い方は省略形表記です。1つ、2つ、3つ、または4つの値を一度に指定できます。これらのオプションを一緒に見ていきましょう。

`margin` の省略形で1つの値を使う場合、その値が対象要素の4辺すべてに適用されます。

`margin` 省略形で1つの値を使った例はこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px;
}
```

:::

このコード例は、段落要素の4辺すべてに `10px` の `margin` を均等に適用します。

2つの値を使う場合、最初の値は `top` と `bottom` に適用され、2つ目の値は `left` と `right` に適用されます。

`margin` 省略形で2つの値を使った例はこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px;
}
```

:::

この例では、段落要素の `top` と `bottom` のマージンを `10px` に、`20px` は `left` と `right` のマージンに設定しています。

3つの値が指定された場合、最初の値は `top` のマージンに、2つ目の値は `left` と `right` のマージンに、3つ目の値は `bottom` のマージンに適用されます。

理解を深めるための例はこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px;
}
```

:::

この例では、`10px` に `top` のマージン、`20px` と `left` に `right` のマージン、`30px` に `bottom` のマージンを設定しています。

4つの値を使う場合は、より細かく制御でき、対象要素の各辺に独立してマージンの値を指定できます。

最初の値は `top` に、2つ目は `right` に、3つ目は `bottom` に、4つ目は `left` に適用されます。

4つの値を使ったマージン省略形の例はこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px 40px;
}
```

:::

この例では、段落要素の `10px` に `top` のマージン、`20px` に `right`、`30px` に `bottom`、`40px` に `left` のマージンを設定しています。

`padding` プロパティは、要素の内側、コンテンツとボーダーの間のスペースを設定するために使います。

`margin` プロパティと同様に、4つの `padding` プロパティは `padding-top`、`padding-right`、`padding-bottom`、`padding-left` です。

段落要素の `padding` を設定する例はこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 30px;
  padding-left: 40px;
  border: 2px solid black;
}
```

:::

この例では、段落要素の `10px` に `top` のパディング、`20px` に `right`、`30px` に `bottom`、`40px` に `left` のパディングを設定しています。

ご覧の通り、`padding` はボーダーの内側にあるコンテンツに適用されます。これは、ボーダーの外側に適用される `margin` とは異なります。

`margin` プロパティと同様に、`padding` プロパティにも省略形を使うことができます。

`padding` 省略形プロパティにも、1つ、2つ、3つ、または4つの値を指定できます。

先ほどの段落要素に対して `padding` 省略形を使った例はこちらです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding: 10px 20px 30px 40px;
  border: 2px solid black;
}
```

:::

この例では、省略形を使って段落要素の `10px` に `top` のパディング、`20px` に `right`、`30px` に `bottom`、`40px` に `left` のパディングを設定しています。
