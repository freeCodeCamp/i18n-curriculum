# --description--

Pratique escolher a forma correta do verbo para um departamento.

# --instructions--

Ouça o áudio e responda à pergunta abaixo.

# --questions--

## --text--

Qual frase usa o verbo correto?

## --answers--

`El departamento de control de calidad verifica el software.`

---

`El departamento de control de calidad verifican el software.`

### --feedback--

`Verifican` é a forma do verbo para `ellos/ellas` (plural), mas `el departamento` é singular.

---

`El departamento de control de calidad verificar el software.`

### --feedback--

`Verificar` é a forma infinitiva, então não completa a frase corretamente aqui.

---

`El departamento de control de calidad verificas el software.`

### --feedback--

`Verificas` é a forma para `tú`, não para `el departamento`.

## --video-solution--

1

# --explanation--

A resposta correta é `el departamento de control de calidad verifica el software`, que significa "o departamento de controle de qualidade verifica o software".

O sujeito `el departamento` é singular, então precisa da forma verbal singular `verifica`.
