# --explanation--

`Currently` bedeutet „zurzeit“ oder „jetzt“. Es zeigt an, dass etwas in diesem Moment passiert. Zum Beispiel:

- `She is currently debugging the software.` – Sie debuggt gerade die Software.

- `We are currently testing the new app.` – Sie und Ihr Team testen gerade die neue App.
