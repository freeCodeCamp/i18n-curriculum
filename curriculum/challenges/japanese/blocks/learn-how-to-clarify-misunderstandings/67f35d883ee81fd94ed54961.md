# --explanation--

`On schedule` は、物事が計画されたスケジュール通りに進んでいることを意味します。例えば：

`The project is still on schedule for delivery next week.` — 予定通りに終わることを意味します。

`Miscommunication` は、情報が正しく共有されなかったり理解されなかった状況を指します。例えば：

`There was a miscommunication about the meeting location.` — 正しい場所が理解されていませんでした。

`Right away` は、すぐに、できるだけ早くという意味です。例えば：

`I'll send the update right away.` — 遅れずに送ることを意味します。
