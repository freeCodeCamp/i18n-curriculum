針對此挑戰，你需要使用 Python 創建一個資料圖形瀏覽器，該瀏覽器接受輸入並執行以下操作：

- 以三種方式取得 .csv 檔案
  - 從區域的電腦上傳它
  - 從使用者輸入取得 url
  - 將 url 放入程式碼中
- 使用 Pandas 函式庫將 .csv 存檔為 dataframe
- 列印標題和前兩列
- 將行名稱儲存為列表
- 選擇一個或兩個行並將資料轉換為 Numpy 陣列
- 將資料顯示為散佈圖或折線圖
- 能夠針對不同的行組合執行此操作，並解讀圖表

完成後，提交你 Google 雲端硬碟中公開 Colab 筆記本的 URL。
