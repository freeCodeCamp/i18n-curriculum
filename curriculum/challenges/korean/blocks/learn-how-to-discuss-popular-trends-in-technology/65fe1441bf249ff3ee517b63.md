# --description--

전치사 `about`는 동사 `-ing`로 끝나는 단어 뒤에 와서 행동의 주제나 포커스를 나타냅니다.
예를 들어, 문장 `It's all about teaching machines to think and learn just like humans`에서 `about teaching`는 특정 주제나 과목을 가르치는 진행 중인 행동을 나타내며 기계에게 가르치는 지속적인 과정을 강조합니다.
