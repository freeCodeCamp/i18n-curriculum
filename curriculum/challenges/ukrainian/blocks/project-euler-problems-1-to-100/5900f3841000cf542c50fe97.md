# --description--

Перестановка — це впорядковане розташування об’єктів. Наприклад, 3124 — це одна з можливих перестановок цифр 1, 2, 3 і 4. Якщо всі перестановки впорядкувати чисельно або алфавітно, це називається лексикографічним порядком. Лексикографічні перестановки цифр 0, 1 і 2 такі:

<div style='text-align: center;'>012   021   102   120   201   210</div>

Яка `n`-та лексикографічна перестановка цифр 0, 1, 2, 3, 4, 5, 6, 7, 8 і 9?
