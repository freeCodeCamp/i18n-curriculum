# --interactive--

`.toFixed()` メソッドは、数値を固定小数点表記でフォーマットする組み込みの JavaScript 関数です。特に、小数点以下の桁数を制御したい場合に便利で、通貨の表示や正確な測定値を扱うときに役立ちます。

`.toFixed()` メソッドは数値に対して呼び出され、1つのオプションの引数を取ります。これは小数点以下に表示する桁数です。指定した小数点以下の桁数で数値を文字列として返します。`.toFixed()` の基本的な使い方は次の通りです。

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed(2)); // "3.14"
```

:::

この場合、小数点以下の桁数を2桁に制限しています。つまり、`3.14159` は `3.14` になります。重要なのは、`.toFixed()` は数値ではなく文字列を返すことです。これは、このメソッドが主に表示用に数値をフォーマットするためのもので、計算に使うためではないからです。

`.toFixed()` メソッドは、指定した小数点以下の桁数で表現できる最も近い値に数値を丸めます。この丸めの動作を理解することが重要です。

:::interactive_editor

```js
console.log((3.14159).toFixed(3));  // "3.142"
console.log((3.14449).toFixed(3));  // "3.144"
console.log((3.14550).toFixed(3));  // "3.146"
```

:::

ご覧の通り、`.toFixed()` は次の桁が `5` 以上の場合は切り上げ、それ以外は切り捨てます。引数なしで `.toFixed()` を呼び出すと、小数点以下 `0` 桁がデフォルトになります。

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed()); // "3"
```

:::

`.toFixed()` メソッドは、特に金融計算や価格表示で役立ちます。

:::interactive_editor

```js
let price = 19.99;
let taxRate = 0.08;
let total = price + (price * taxRate);

console.log("Total: $" + total.toFixed(2)); // "Total: $21.59"
```

:::

この例では、`.toFixed(2)` によって合計が常に2桁の小数点以下で表示されます。これは多くの国で通貨の標準です。

まとめると、`.toFixed()` メソッドは、JavaScript で数値の小数点以下の表示を制御したいときに強力なツールです。主に出力のフォーマットに使われますが、正確な計算が必要な場合はその動作を覚えておいてください。
