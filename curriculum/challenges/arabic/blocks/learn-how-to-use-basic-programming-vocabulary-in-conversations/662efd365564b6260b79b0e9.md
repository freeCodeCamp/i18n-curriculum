# --description--

`loop` هو متتالية من التعليمات تُكرر باستمرار حتى يتحقق شرط معين. تُستخدم `Loops` لتشغيل نفس الشفرة عدة مرات بكفاءة. من الأنواع الشائعة لـ `loops`: `for` loops و `while` loops.

على سبيل المثال، `for loop`: `for (int i = 0; i < 5; i++) { /* code to be executed */ }`

في المثال، تُنفذ الشفرة داخل الأقواس `{ }` بشكل متكرر حتى يصبح الشرط غير صحيح.
