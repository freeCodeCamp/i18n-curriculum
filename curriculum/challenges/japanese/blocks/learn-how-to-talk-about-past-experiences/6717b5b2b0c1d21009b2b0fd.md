# --explanation--

`used to`を使った否定文では、過去を示すために助動詞`didn't`が使われ、主動詞`use`は原形のまま（活用されません）です。

この構造は、過去にその行動が習慣的でなかったことを強調します。

**重要:** 助動詞`didn't`が文を否定にします。動詞`use`は過去時制がすでに`didn't`で表されているため、原形のままです。例えば：

`We didn't use to have weekly meetings, but now we do.` - これは当時、週次ミーティングが一般的でなかったことを意味します。
