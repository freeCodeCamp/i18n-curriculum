# --description--

Adicione `encrypted-media`, `gyroscope` e `web-share` aos valores existentes no atributo `allow`.

Esses três permitirão o uso de extensões de mídia criptografada para proteger o vídeo, permitir que o iframe seja destacado no modo picture-in-picture quando necessário e permitir o compartilhamento do conteúdo do iframe através dos diálogos nativos de compartilhamento do dispositivo. 

# --hints--

O atributo `allow` do seu elemento `iframe` deve ter `encrypted-media` como um dos seus valores.

```js
const iframeEl = document.querySelector('iframe')
const iframeElAllowAttr = iframeEl?.getAttribute('allow')
const encryptedMedia = iframeElAllowAttr.trim().split(' ')[3]
assert.strictEqual(encryptedMedia, 'encrypted-media')
```

O atributo `allow` do seu elemento `iframe` deve ter `gyroscope` como um dos seus valores.

```js
const iframeEl = document.querySelector('iframe')
const iframeElAllowAttr = iframeEl?.getAttribute('allow')
const gyroscope = iframeElAllowAttr.trim().split(' ')[4]
assert.strictEqual(gyroscope, 'gyroscope')
```

O atributo `allow` do seu elemento `iframe` deve ter `web-share` como um dos seus valores.

```js
const iframeEl = document.querySelector('iframe')
const iframeElAllowAttr = iframeEl?.getAttribute('allow')
const webShare = iframeElAllowAttr.trim().split(' ')[5]
assert.strictEqual(webShare, 'web-share')
```
