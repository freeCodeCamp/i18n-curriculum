# --description--

Um das Aussehen von jemandem zu beschreiben, verwendet man oft Adjektive wie `tall` (überdurchschnittlich) oder `short` (unterdurchschnittlich) für die Körpergröße.

Sie können `have` verwenden, um auszudrücken, dass jemand ein Merkmal besitzt. Zum Beispiel `She has blue eyes.`. Das bedeutet, dass die Person das Merkmal blaue Augen besitzt.

`Curly hair` bedeutet, dass das Haar nicht glatt oder platt ist, sondern viele Locken oder Wellen hat.
