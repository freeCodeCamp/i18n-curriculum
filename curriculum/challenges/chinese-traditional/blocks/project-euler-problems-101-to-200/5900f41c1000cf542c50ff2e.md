# --description--

定義 $f(0) = 1$，且 $f(n)$ 為將 $n$ 寫成 2 的冪次和且每個冪次不超過兩次的方式數量。

例如，$f(10) = 5$，因為有五種不同的方法來表示 10：

$$10 = 8 + 2 = 8 + 1 + 1 = 4 + 4 + 2 = 4 + 2 + 2 + 1 + 1 = 4 + 4 + 1 + 1$$

可以證明，對於每一個分數 $\frac{p}{q}\\; (p>0, q>0)$，至少存在一個整數 $n$，使得 $\frac{f(n)}{f(n - 1)} = \frac{p}{q}$。

例如，最小的 $n$ 使得 $\frac{f(n)}{f(n - 1)} = \frac{13}{17}$ 是 241。241 的二進位展開是 11110001。

從最高有效位元讀取這個二進位數到最低有效位元，有 4 個 1、3 個 0 和 1 個 1。我們將字串 4,3,1 稱為 241 的縮短二進位展開。

尋找最小的 $n$ 的縮短二進位展開，該 $n$ 滿足

$$\frac{f(n)}{f(n - 1)} = \frac{123456789}{987654321}$$

以逗號分隔的整數字串，且不含任何空白。
