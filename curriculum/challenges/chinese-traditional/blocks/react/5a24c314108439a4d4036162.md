# --description--

組件是 React 的核心。React 中的所有東西都是組件，在這裡你將學習如何創建一個組件。

有兩種方法可以創建 React 組件。第一種方法是使用 JavaScript 函式（程式）。以這種方式定義組件會創建一個*無狀態函式組件*。應用程式中狀態的概念將在後續挑戰中涵蓋。現在，將無狀態組件視為一個可以接收資料並呈現它，但不管理或追蹤該資料變化的組件。（我們會在下一個挑戰中涵蓋創建 React 組件的第二種方法。）

要用函式創建組件，你只需撰寫一個 JavaScript 函式，該函式傳回 JSX 或 `null`。一個重要的注意事項是 React 要求你的函式名稱必須以大寫字母開頭。以下是一個無狀態函式組件的範例，它在 JSX 中指定了一個 HTML 類別：

```jsx
const DemoComponent = function() {
  return (
    <div className='customClass' />
  );
};
```

經過轉譯後，`<div>` 將會有一個 CSS 類別為 `customClass`。

因為 JSX 組件表述 HTML，你可以將多個組件組合在一起，創建更複雜的 HTML 頁面。這是 React 所提供的組件架構的主要優勢之一。它允許你從許多獨立且隔離的組件中組合你的使用者介面。這使得建置和維護複雜的使用者介面變得更容易。

# --instructions--

程式碼編輯器有一個名為 `MyComponent` 的函式。完成此函式，使其傳回一個包含某些字串文字的單一 `div` 元素。

**注意：**該文字被視為 `div` 元素的子元素，因此你無法使用自我關閉標籤。

# --hints--

`MyComponent` 應該傳回 JSX。

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.length === 1;
  })()
);
```

`MyComponent` 應該傳回一個 `div` 元素。

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.children().type() === 'div';
  })()
);
```

`div` 元素應該包含一個字串的文字。

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.find('div').text() !== '';
  })()
);
```
