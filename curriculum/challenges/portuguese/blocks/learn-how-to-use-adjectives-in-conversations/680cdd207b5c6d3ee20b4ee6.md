# --explanation--

`Could` é usado para pedir educadamente que alguém faça algo.Por exemplo:

`Could you update the software today?` – Isso mostra um pedido educado para uma ação.

`Ensure` significa garantir que algo aconteça ou seja concluído corretamente.Por exemplo:

`We must ensure all servers are properly backed up.` – Isso significa garantir que os backups aconteçam.

`By the end of` significa antes que um limite de tempo termine.Por exemplo:

`We need to submit the project by the end of the week.` – Isso significa que o projeto deve ser concluído antes do final da semana.
