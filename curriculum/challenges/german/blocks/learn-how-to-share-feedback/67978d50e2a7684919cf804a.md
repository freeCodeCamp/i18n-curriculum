# --explanation--

`To set goals` bedeutet, spezifische Ziele oder Vorgaben zu definieren, die erreicht werden sollen. Zum Beispiel:

`The team set goals to improve customer satisfaction this year.` – Das bedeutet, dass das Team Ziele in Bezug auf die Kundenzufriedenheit festgelegt hat.

`Quarter` bezeichnet ein Vierteljahr oder einen Zeitraum von drei Monaten. Im geschäftlichen Kontext bedeutet `the next quarter` den kommenden Dreimonatszeitraum im Kalender- oder Geschäftsjahr. Zum Beispiel:

- `Our sales target for the next quarter is $1 million.` – Das bedeutet, dass das Ziel für den nächsten Dreimonatszeitraum gilt.

- `We review our performance at the end of each quarter.` – Das bezieht sich auf die Bewertung des Fortschritts alle drei Monate.
