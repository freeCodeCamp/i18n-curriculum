# --description--

Підпрограмне забезпечення рівня маршрутизатора схоже на підпрограмне забезпечення рівня застосунку, але воно прив’язане до екземпляра `express.Router()` замість головного екземпляра застосунку. Його можна використовувати для модульного побудови вашого Express-застосунку, створюючи ізольовані модулі маршрутів.

Підпрограмне забезпечення рівня маршрутизатора працює так само, як і підпрограмне забезпечення рівня застосунку, але застосовується до конкретних екземплярів маршрутизатора. Це полегшує застосування підпрограмного забезпечення лише до певної множини маршрутів, а не глобально.

```js
const express = require('express')
const app = express()
const router = express.Router()

// Router-level middleware to log requests to '/menu' routes
router.use((req, res, next) => {
  console.log('Request made to /menu route')
  next() // Pass control to the next handler
})

// Define routes
router.get('/drinks', (req, res) => {
  res.send('Welcome to the drinks menu!')
})

app.use('/menu', router) // Apply the router

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* `router.use()`: Це підпрограмне забезпечення рівня маршрутизатора, яке застосовується лише до маршрутів, оброблених маршрутизатором.
* `app.use('/menu', router)`: Підпрограмне забезпечення застосовується лише до маршрутів, що починаються з `/menu` (наприклад, `/menu/drinks`).

Підпрограмне забезпечення рівня маршрутизатора дозволяє розбити ваш застосунок на модульні компоненти та вибірково застосовувати підпрограмне забезпечення до групи маршрутів. Це допомагає з організацією та зрозумілістю коду.
