# --explanation--

`Will` використовується, щоб відповісти на запит, виражаючи готовність або намір щось зробити. Наприклад:

- Особа 1: `Can you send me the report?`

- Особа 2: `Sure, I'll send it right away.` - Тут `I'll send it` показує, що мовник погоджується з проханням.

У цьому діалозі Джеймс використовує `I'll check it out`, щоб підтвердити, що він розслідує проблему.
