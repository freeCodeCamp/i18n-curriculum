# --interactive--

Унарні оператори діють на один операнд, виконуючи операції, такі як перетворення типів, маніпуляції зі значеннями або перевірка певних умов. Розглянемо деякі загальні унарні оператори та як вони працюють.

Унарний оператор плюс перетворює свій операнд у число. Якщо операнд уже є числом, він залишається без змін.

:::interactive_editor

```js
const str = '42';
const strToNum = +str;

console.log(strToNum); // 42
console.log(typeof str); // string
console.log(typeof strToNum); // number
```

:::

Унарний плюс зручний, коли потрібно переконатися, що ви працюєте з числовим значенням. Як ви можете здогадатися, існує унарний оператор від’ємності. Він змінює знак операнда на протилежний. Працює подібно до унарного плюса, але змінює знак.

:::interactive_editor

```js
const str = '42';
const strToNegativeNum = -str;

console.log(strToNegativeNum); // -42
console.log(typeof str); // string
console.log(typeof strToNegativeNum); // number
```

:::

Логічний оператор NOT, представлений знаком оклику (`!`), — це ще один унарний оператор. Він змінює булеве значення свого операнда на протилежне. Тож, якщо операнд `true`, він стає `false`, а якщо `false`, то стає `true`. 

:::interactive_editor

```js
let isOnline = true;
console.log(!isOnline); // false

let isOffline = false;
console.log(!isOffline); // true
```

:::

Бітовий оператор NOT — менш поширений унарний оператор. Представлений тильдою, `~`, він інвертує двійкове представлення числа. Комп’ютери зберігають числа у двійковому форматі (1 та 0). Оператор `~` змінює кожен біт, тобто всі 1 перетворює на 0, а всі 0 — на 1. Ви дізнаєтеся більше про двійкову систему та біти в одному з майбутніх уроків.

:::interactive_editor

```js
const num = 5; // The binary for 5 is 00000101

console.log(~num); // -6
```

:::

У цьому прикладі `5` став `-6`, тому що застосування оператора `~` до `5` дає `- (5 + 1)`, що дорівнює `-6` через подання у двійковому додатковому коді (two's complement). Додатковий код — це спосіб, яким комп’ютери представляють від’ємні числа у двійковому вигляді. Ви, ймовірно, не часто використовуватимете бітовий NOT, якщо не працюєте з низькорівневим програмуванням, наприклад, безпосередньою маніпуляцією бітами.

Ключове слово `void` — це унарний оператор, який оцінює вираз і повертає `undefined`.

:::interactive_editor

```js
const result = void (2 + 2);

console.log(result); // undefined
```

:::

`void` також часто використовується у посиланнях, щоб запобігти переходу:

```js
<a href="javascript:void(0);">Click Me</a>
```

Нарешті, є оператор `typeof`, про який ви вже дізналися в попередніх уроках. Він повертає тип свого операнда у вигляді рядка.

:::interactive_editor

```js
const value = 'Hello world';

console.log(typeof value); // string
```

:::
