# --description--

$x$를 실수라고 하겠습니다.

분모 제한 $d$에 대한 $x$의 최적 근사값은 기약분수 형태의 유리수 $\frac{r}{s}$로, $s ≤ d$를 만족하며, $\frac{r}{s}$보다 $x$에 더 가까운 유리수는 모두 분모가 $d$보다 크다는 조건을 만족합니다:

$$|\frac{p}{q} - x| &lt; |\frac{r}{s} - x| ⇒ q > d$$

예를 들어, 분모 제한이 $20$일 때 $\sqrt{13}$의 최적 근사값은 $\frac{18}{5}$이고, 분모 제한이 $30$일 때 $\sqrt{13}$의 최적 근사값은 $\frac{101}{28}$입니다.

$n$이 완전제곱수가 아니고 $1 &lt; n ≤ 100000$인 경우, 분모 제한이 ${10}^{12}$일 때 $\sqrt{n}$의 모든 최적 근사값의 분모 합을 구하세요.
