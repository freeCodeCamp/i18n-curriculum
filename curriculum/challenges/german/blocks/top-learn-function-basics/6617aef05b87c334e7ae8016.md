# --description--

Wie Sie bereits gesehen haben, können Funktionen mit dem Schlüsselwort `return` einen Wert zurückgeben. Das Schlüsselwort `return` wird verwendet, um einen Wert aus einer Funktion zurückzugeben. Wenn das Schlüsselwort `return` verwendet wird, wird die Funktion beendet und gibt den Wert zurück, der nach dem Schlüsselwort `return` angegeben ist.

```js
function add(a, b) {
  return a + b
}

console.log(add(2, 3)); // Output: 5
```

Aber was passiert, wenn das Schlüsselwort `return` vor dem Ende der Funktion verwendet wird?
Um diese Frage zu beantworten, betrachten Sie folgendes Beispiel:

```js
function add(a, b) {
  if(a > 2){
    return b;
  }

  return a + b;
}

console.log(add(3, 7)); // Output: 7
```

Im obigen Beispiel hat die Funktion `add` eine Bedingung, die überprüft, ob der Wert von `a` größer als `2` ist. Wenn die Bedingung erfüllt ist, gibt die Funktion den Wert von `b` zurück und beendet die Ausführung. Wenn die Bedingung nicht erfüllt ist, gibt die Funktion die Summe von `a` und `b` zurück.
