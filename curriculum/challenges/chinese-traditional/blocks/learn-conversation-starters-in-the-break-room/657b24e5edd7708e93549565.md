# --description--

`Present Perfect` 是一種用來顯示現在與過去之間聯結的時態。動作發生的時間是在現在之前但未具體說明，而使用者通常對結果比動作本身更感興趣。 例如：

- `I played this game yesterday.` 是 `past simple` 時態。它表示玩這款遊戲的動作發生在過去的特定時間，也就是昨天。

- `I have played this game before.` - 這是 `present perfect` 時態。它表示在過去某個未指定的時間發生過玩這款遊戲的動作。

- `I've played this game before.`－這句話也使用了 `present perfect` 時態，並使用縮寫 `I've` 代表 `I have`。和前面的例子一樣，它表示玩這個遊戲的動作發生在過去未指定的時間。
