# --interactive--

Il posizionamento assoluto ti permette di estrarre un elemento dal flusso normale del documento, facendolo comportare in modo indipendente dagli altri elementi. Quando un elemento è posizionato in modo assoluto, viene collocato in un proprio livello, completamente separato da tutto il resto nel layout. Questo lo rende utile per creare funzionalità UI flottanti come modali, tooltip o menu a discesa, che possono sovrapporsi ad altri elementi nella pagina.

Per impostazione predefinita, gli elementi posizionati in modo assoluto sono collocati in relazione all’antenato posizionato più vicino. Se non viene trovato nessun antenato posizionato, l’elemento sarà posizionato in relazione al blocco contenitore iniziale, che di solito è il viewport del browser. Puoi spostare l’elemento posizionato in modo assoluto usando le proprietà `top`, `bottom`, `left` e `right` per specificare a che distanza deve trovarsi dai bordi del suo elemento contenitore.

Per esempio, impostare `top: 30px` e `left: 30px` sposterà l’elemento `30px` lontano dai bordi superiore e sinistro del blocco contenitore. Ecco un esempio di come applicare il posizionamento assoluto a un elemento in CSS:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="positioned">Absolutely Positioned</div>
```

```css
body {
  background-color: #eeeeee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

Quando questo codice viene applicato, l’elemento verrà rimosso dal flusso normale del documento e posizionato `30px` lontano dal bordo superiore e sinistro del suo blocco contenitore.

Il posizionamento assoluto può essere particolarmente utile per creare elementi che devono fluttuare sopra il resto del contenuto, come overlay o finestre modali. Tuttavia, poiché l’elemento viene rimosso dal flusso del documento, può anche creare spazi vuoti o far collassare insieme altri elementi se non gestito correttamente.
