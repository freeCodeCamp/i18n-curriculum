# --description--

Chukulia pembetatu zote zenye:

- Vichwa vyote viko kwenye nukta za kimiani.
- Kituo cha mzunguko (circumcenter) kiko kwenye asili O.
- Kituo cha mstatili (orthocenter) kiko kwenye nukta H(5, 0).

Kuna pembetatu tisa kama hizo zenye $\text{ukingo} ≤ 50$.
