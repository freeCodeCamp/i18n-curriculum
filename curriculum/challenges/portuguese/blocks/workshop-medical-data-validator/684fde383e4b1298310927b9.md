# --description--

Quantificadores são usados em expressões regulares para especificar quantas vezes um caractere pode ser repetido.Por exemplo, o caractere `+` corresponde ao caractere anterior uma ou mais vezes:

```py
import re

book = "Fahrenheit 451"
print(re.search('\d', book))
# <re.Match object; span=(11, 12), match='4'>

print(re.search('\d+', book))
# <re.Match object; span=(11, 14), match='451'>
```

Então adicione um quantificador `+` ao seu padrão regex para corresponder a um ou mais dígitos.
