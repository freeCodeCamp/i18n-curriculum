# --explanation--

`Must have happened` використовується для логічного припущення про щось у минулому.  

`Must` виражає сильну впевненість. `Have` + `Past Participle` (наприклад, `happened`) стосується чогось у минулому. Наприклад:

- `He must have forgotten to update the system.` — ймовірно, він не згадав зробити оновлення.

- `They must have made some changes to the app.` — ймовірно, застосунок було оновлено або покращено.

Джейк використовує докази (`unusual activity on the servers`), щоб зробити висновок, що порушення `must have happened` минулої ночі. Ця структура корисна для логічних припущень.
