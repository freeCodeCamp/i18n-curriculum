`support`

### --feedback--

La palabra describe la capacidad del programa para manejar diferentes tareas o funciones.

`feature`

### --feedback--

Esta palabra describe una función o capacidad específica que un programa de software ofrece a sus usuarios.

# --explanation--

En el desarrollo de software, `support` significa que un programa o herramienta es capaz de trabajar con una cierta característica o funcionalidad. Un `feature` se refiere a una función o capacidad específica que un programa de software ofrece a sus usuarios. Por ejemplo:

- `VS Code supports multiple programming languages.` - VS Code puede trabajar con diferentes lenguajes.

- `One feature of VS Code is the ability to integrate with Git.` - Esto se refiere a una capacidad específica que VS Code ofrece a quienes lo usan.
