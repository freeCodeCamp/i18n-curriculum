# --description--

Neste projeto, você vai construir um conversor decimal e binário e aprender sobre ambos os sistemas numéricos.Você também aprenderá sobre recursão usando-a para realizar as conversões.

Todo o HTML e o CSS para este projeto foram fornecidos para você.

Quando estiver pronto para começar, use o método `.getElementById()` para obter o elemento `input` com o id `"number-input"` e armazene-o em uma variável chamada `numberInput`.Use o mesmo método para obter o elemento `button` com o id `"convert-btn"` e armazená-lo em uma variável chamada `convertBtn` e o elemento `output` com o id `"result"` e armazená-lo em uma variável chamada `result`.


**NOTA**: Este projeto irá converter apenas números positivos em binário.
