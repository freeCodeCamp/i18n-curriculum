# --explanation--

Sophie explica herança como um conceito de programação onde uma classe (uma filha ou subclasse) pode acessar as propriedades e métodos de outra classe (a pai ou superclasse).

Isso permite que os desenvolvedores reutilizem código e estendam a funcionalidade sem reescrever tudo.É uma parte fundamental da programação orientada a objetos que promove eficiência e um design mais limpo.
