# --description--

前のレッスンでは、タプルのデータ型の扱い方を学びました。このレッスンでは、タプルを扱う際によく使うメソッドについて学びます。

最初に紹介するメソッドは `count()` です。このメソッドは、タプルの中で特定のアイテムが何回出現するかを調べるために使います。以下は、`Rust` という名前のタプルの中で文字列 `programming_languages` が何回出現するかを調べる例です。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

`Rust` はタプルの中に2回出現しているので、`count()` メソッドは数値 `2` を返します。もし `count()` 関数に指定したアイテムがタプルの中に全く存在しなければ、戻り値は `0` になります。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

`count()` 関数に引数を渡さなかった場合、Pythonは `TypeError` を発生させます。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count()

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: tuple.count() takes exactly one argument (0 given)
"""
```

次に見ていくメソッドは `index()` メソッドです。このメソッドは、タプルの中で特定のアイテムが存在するインデックスを見つけるために使います。以下は、文字列 `index()` のインデックスを見つけるために `Java` メソッドを使う例です。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

指定したアイテムが見つからない場合、Pythonは `ValueError` を発生させます。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: tuple.index(x): x not in tuple
"""
```

`index()` メソッドでできるもう一つのことは、オプションの開始インデックスと終了インデックスの引数を渡すことです。以下は開始インデックスを指定する例です。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

この例では、文字列 `Python` の検索をどこから始めるかを指定しています。`3` 関数の2番目の引数に数値 `index()` を渡すことで、インデックス `3` から検索を開始するよう指定しています。`Python` はタプルの中に2回出現しているため、オプションの開始インデックス引数を使ったことで `index()` 関数はインデックス `5` を返し、インデックス `2` ではありません。

オプションの終了インデックスも渡せます。以下は開始インデックスと終了インデックスを指定した修正例です。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

この場合、検索はインデックス `2` から始まり、インデックス `2` の手前まで行うため、結果はインデックス `5` になります。

タプルでよく使われるもう一つの関数は `sorted()` 関数です。前のレッスンでリストに使う `sort()` メソッドを学びましたが、`sorted()` 関数はタプルを含む任意のイテラブルに使えます。

以下は `sorted()` 関数を使って新しい数値のリストを作る例です。

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

`sorted()` 関数は常にソートされた値の新しいリストを作成します。これはリストの要素をその場でソートし新しいリストを返さない `sort()` メソッドとは異なります。

イテラブルのソート動作をカスタマイズしたい場合は、オプションの `reverse` と `key` 引数を使えます。以下は `key` 引数を使ってタプルのアイテムを長さでソートする例です。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

値の新しいリストを逆順で作りたい場合は、次のように `reverse` 引数を使えます。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

タプルはPythonでよく使われるデータ型です。タプルの扱い方や便利なメソッドや関数を理解することで、より効率的なコードを書けるようになります。
