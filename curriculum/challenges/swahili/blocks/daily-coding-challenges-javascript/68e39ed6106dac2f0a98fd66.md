# --description--

Mnamo Novemba 2, 1988, mdudu wa kwanza mkubwa wa intaneti uliachiliwa, ukiacha takriban asilimia 10 ya kompyuta zilizounganishwa na intaneti kuambukizwa baada ya siku moja tu.

Katika changamoto hii, unapewa idadi ya siku zilizopita tangu mdudu wa intaneti kuachiliwa, na unahitaji kubaini ni kompyuta ngapi zimeambukizwa kwa kutumia sheria zifuatazo:

- Siku ya 0, kompyuta ya kwanza inaambukizwa.
- Kila siku inayofuata, idadi ya kompyuta zilizoambukizwa huongezeka mara mbili.
- Kila siku ya tatu, kinga hutumika baada ya virusi kuenea na kupunguza idadi ya kompyuta zilizoambukizwa kwa asilimia 20%. Zungusha idadi ya kompyuta zilizopatiwa kinga hadi nambari nzima inayofuata.

Kwa mfano, siku:

- Siku ya 0: kompyuta 1 jumla imeambukizwa.
- Siku ya 1: kompyuta 2 jumla zimeambukizwa.
- Siku ya 2: kompyuta 4 jumla zimeambukizwa.
- Siku ya 3: kompyuta 8 jumla zimeambukizwa. Kisha, tumia kinga: kompyuta 8 zilizoambukizwa * asilimia 20 = 1.6 zilipatiwa kinga. Zungusha 1.6 hadi 2. Kompyuta 8 zilizoambukizwa - 2 zilipatiwa kinga = kompyuta 6 jumla zimeambukizwa baada ya siku ya 3.

Rudisha idadi ya kompyuta jumla zilizambukizwa baada ya idadi ya siku zilizotolewa kupita.

function infected(days) {
  let infected = 1;

  for (let day = 1; day <= days; day++) {
    infected *= 2;

    if (day % 3 === 0) {
      let patched = Math.ceil(infected * 0.2);
      infected -= patched;
    }
  }

  return infected;
}
