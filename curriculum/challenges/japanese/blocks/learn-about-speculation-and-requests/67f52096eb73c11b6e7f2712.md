# --explanation--

`Physical access` は、オフィスやサーバールームなどの場所に入ることを意味します。

`physical access breach` は、許可なく安全な場所に入ることが起きた場合のことです。例えば：

- `To avoid a physical access breach, we should require ID checks at the entrance.` - 会社は、入り口でIDを確認することで不正な侵入を防ぐことができます。

- `A physical access breach can happen if someone enters the server room without permission.` - 誰かが許可なくサーバールームに入ると、物理的なセキュリティ問題が発生します。
