# --explanation--

각 작별 인사는 상대를 다시 만날 시기에 따라 약간씩 다른 의미를 가집니다.

| Expresión          | Meaning         | When to use                                                   |
|--------------------|-----------------|----------------------------------------------------------------|
| `Adiós`            | Goodbye         | 일반적이거나 약간 격식을 차린 작별 인사입니다. |
| `Hasta luego`      | See you later   | 상대를 다시 만날 계획이 있지만 즉시가 아닐 때 사용합니다.    |
| `Hasta mañana`     | See you tomorrow| 다음 날 상대를 만날 것으로 예상될 때 사용합니다.                |
| `Hasta pronto`     | See you soon    | 아주 곧 상대를 만날 것으로 예상될 때 사용합니다.                   |
| `Chao`             | Bye             | 비공식적이고 친근한 인사입니다. |
