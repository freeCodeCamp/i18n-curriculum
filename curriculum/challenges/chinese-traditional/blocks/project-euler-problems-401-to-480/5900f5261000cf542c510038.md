# --description--

對於整數 $M$，我們定義 $R(M)$ 為所有滿足以下所有條件的整數配對 $p$ 和 $q$ 的 $\frac{1}{p·q}$ 之和：

- $1 ≤ p &lt; q ≤ M$
- $p + q ≥ M$
- $p$ 和 $q$ 是互質的。

我們也定義 $S(N)$ 為 $2 ≤ i ≤ N$ 時 $R(i)$ 的總和。

我們可以驗證 $S(2) = R(2) = \frac{1}{2}$，$S(10) ≈ 6.9147$ 和 $S(100) ≈ 58.2962$。

求 $S({10}^7)$。將你的答案四捨五入到小數點後四位。
