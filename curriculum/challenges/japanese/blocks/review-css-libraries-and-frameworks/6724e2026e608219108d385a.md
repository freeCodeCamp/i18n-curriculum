## CSSフレームワーク

- **CSSフレームワーク**：CSSフレームワークは作業効率を上げ、ウェブサイト全体で統一されたビジュアルスタイルを作り、複数のブラウザでデザインの一貫性を保ち、CSSコードをより整理された状態にできます。
- **人気のあるCSSフレームワーク**：人気のあるCSSフレームワークにはTailwind CSS、Bootstrap、Materialize、Foundationがあります。
- **考えられるデメリット**：
  - フレームワークが提供するCSSがカスタムCSSと競合する場合があります。
  - 同じフレームワークを使う他のウェブサイトと似た見た目になることがあります。
  - 大きなフレームワークはパフォーマンスの問題を引き起こす場合があります。

## 2種類のCSSフレームワーク

- **ユーティリティファーストCSSフレームワーク**：これらのフレームワークは、マージン、パディング、背景色など特定の目的を持つ小さなクラスを持っています。必要に応じてこれらの小さなクラスをHTML要素に直接代入できます。Tailwind CSSはユーティリティファーストCSSフレームワークに分類されます。

Tailwind CSSを使ってボタンをスタイリングする例です。

```html
<button class="bg-blue-500 text-white font-bold py-2 px-4 rounded-full hover:bg-blue-700">
  Button
</button>
```

- **コンポーネントベースのCSSフレームワーク**：これらのフレームワークは、事前に定義されたスタイルを持つコンポーネントを提供し、ウェブサイトに追加できます。コンポーネントはCSSフレームワークの公式ドキュメントにあり、プロジェクトにコピー＆ペーストして使います。BootstrapはコンポーネントベースのCSSフレームワークに分類されます。

Bootstrapを使ってリストグループを作成する例です。HTML要素に小さなクラスを代入する代わりに、HTML構造を含むコンポーネント全体を追加します。

```html
<div class="card" style="width: 25rem;">
  <ul class="list-group list-group-flush">
    <li class="list-group-item">HTML</li>
    <li class="list-group-item">CSS</li>
    <li class="list-group-item">JavaScript</li>
  </ul>
</div>
```

## Tailwind CSS

TailwindはユーティリティファーストCSSフレームワークです。カスタムCSSルールを書く代わりに、小さなユーティリティクラスをHTMLに直接組み合わせてデザインを作ります。

### レスポンシブデザイン用ユーティリティ

Tailwindは`sm:`、`md:`、`lg:`のような接頭辞を使い、異なる画面サイズでスタイルを適用します。

```html
<div class="w-full md:w-1/2 lg:flex-row">Responsive layout</div>
```

### Flexbox用ユーティリティ

`flex`、`flex-col`、`justify-around`、`items-center`のようなクラスで柔軟なレイアウトを簡単に作れます。

```html
<div class="flex flex-col md:flex-row justify-around items-center">
  <p>Column on small screens</p>
  <p>Row on medium and larger screens</p>
</div>
```

### グリッド用ユーティリティ

Tailwindは`grid`、`grid-cols-1`、`md:grid-cols-3`のようなCSS Grid用ユーティリティを含みます。

```html
<div class="grid grid-cols-1 md:grid-cols-3 gap-8">
  <div class="bg-gray-100 p-4">Column 1</div>
  <div class="bg-gray-100 p-4">Column 2</div>
  <div class="bg-gray-100 p-4">Column 3</div>
</div>
```

### スペーシング用ユーティリティ

`mt-8`、`mx-auto`、`p-4`、`gap-4`のようなユーティリティで、CSSを書かずに一貫したスペースを作れます。

```html
<div class="mt-8 p-4 bg-indigo-600 text-white">Spaced content</div>
```

### タイポグラフィ用ユーティリティ

`uppercase`、`font-bold`、`font-semibold`、`text-4xl`のようなユーティリティでテキストの見た目を制御します。

`text-3xl` `md:text-5xl`のように、ブレークポイントで調整されるフォントサイズを設定できます。

```html
<h1 class="text-3xl md:text-5xl font-semibold text-center">Responsive Heading
</h1>
```

### カラーとホバー状態

Tailwindは`text-red-700`、`bg-indigo-600`、`bg-gray-100`のような幅広いカラーパレットを提供します。

`hover:bg-pink-600`のようなクラスでインタラクティブな効果を簡単に作れます。

```html
<a href="#" class="bg-pink-500 hover:bg-pink-600 text-white px-4 py-2 rounded-md">
  Hover Me
</a>
```

### ボーダー、リング、エフェクト

- **ボーダー**：`border-2 border-red-300`は指定した太さと色のボーダーを追加します。
- **リング**：`ring-1 ring-gray-300`はフォーカスやカードでよく使われるアウトラインのような効果を作ります。
- **角丸とスケーリング**：`rounded-md`、`rounded-xl`、`scale-105`のようなクラスで仕上げを加えます。

```html
<div class="p-6 rounded-xl ring-2 ring-fuchsia-500 scale-105">
  Highlighted card
</div>
```

### グラデーション

Tailwindは`bg-gradient-to-r from-fuchsia-500 to-indigo-600`のようなグラデーション用ユーティリティをサポートします。

```html
<div class="p-4 text-white bg-gradient-to-r from-fuchsia-500 to-indigo-600">
  Gradient background
</div>
```

## CSSプリプロセッサ

- **CSSプリプロセッサ**：CSSプリプロセッサは標準CSSを拡張するツールです。拡張構文のコードをネイティブCSSファイルにコンパイルします。複雑なプロジェクトでよりきれいで再利用可能、繰り返しが少なくスケーラブルなCSSを書くのに役立ちます。
- **提供される機能**：CSSプリプロセッサが提供する機能には変数、ミックスイン、ネスト、セレクターの継承などがあります。
- **人気のあるCSSプリプロセッサ**：人気のあるCSSプリプロセッサにはSass、Less、Stylusがあります。  
- **考えられるデメリット**：
  - CSSルールを標準CSSにコンパイルする際にオーバーヘッドが発生する場合があります。
  - コンパイルされたコードはデバッグが難しい場合があります。

## Sass

- **Sass**：最も人気のあるCSSプリプロセッサの1つです。Sassは「Syntactically Awesome Style Sheets」の略です。
- **Sassがサポートする機能**：変数、ネストされたCSSルール、モジュール、ミックスイン、継承、基本的な数学演算のための演算子などの機能をサポートします。

## Sassがサポートする2つの構文

- **SCSS構文**：SCSS（Sassy CSS）はCSSの基本構文を拡張したものです。Sassで最も広く使われている構文です。SCSSファイルは`.scss`拡張子を持ちます。

SCSSで変数を定義し使う例です。

```scss
$primary-color: #3498eb;

header {
  background-color: $primary-color;
}
```

- **インデント構文**：インデント構文はSassの元々の構文で、「Sass構文」とも呼ばれます。

インデント構文で変数を定義し使う例です。

```sass
$primary-color: #3498eb

header
  background-color: $primary-color
```

### ミックスイン

- **ミックスイン**：ミックスインは複数のCSSプロパティと値を名前の下にまとめ、スタイルシート全体でそのCSSコードのブロックを再利用できます。

SCSS構文でミックスインを定義する例です。この場合、ミックスインの名前は`center-flex`で、flexboxを使って要素を中央に配置する3つのCSSプロパティがあります。

```scss
@mixin center-flex {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

定義したミックスインを使う例です。

```scss
section {
  @include center-flex;
  height: 500px;
  background-color: #3289a8;
}
```
