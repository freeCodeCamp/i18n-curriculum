# --description--

في هذه المهمة، ستتعلم الفرق بين `specific` و `specify`. `Specific` صفة تصف شيئًا محددًا أو مفصلًا بوضوح، بينما `specify` هو فعل يعني ذكر شيء بتفصيل.

على سبيل المثال، قد تكون التعليمات `specific` هي `use port 8080`، بينما تعني `specify` ذكر هذا التفصيل: `specify the port number.`

`Setup` تشير إلى ترتيب أو إعداد شيء ما، وغالبًا ما تُستخدم في سياق تجهيز البرمجيات أو المعدات، مثل `setup instructions for the software.`
