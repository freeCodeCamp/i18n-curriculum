# --interactive--

كما تتذكر من الدروس وورش العمل السابقة، يمكنك إنشاء دوال بعدد من المعلمات واستدعاء تلك الدالة مع معلمات.

إليك تذكير:

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call
getSum(3, 4); // 7
```

لكن ماذا لو كانت هناك دالة تُستدعى بعدد معلمات أكثر مما تم تعريفها لقبوله؟

:::interactive_editor

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call with extra argument
console.log(getSum(3, 4, 5)); // 7
```

:::

لن يرمي JavaScript خطأ في هذه الحالة. بدلاً من ذلك، سيتجاهل المعلمة الزائدة ويجمع فقط الرقمين `3` و `4`. تُعرف الدوال التي تقبل عددًا متغيرًا من المعلمات باسم <dfn>الدوال المتعددة المعلمات</dfn>.

إذا كنت تعمل مع الدوال المتعددة المعلمات، فيمكنك استخدام كائن `arguments`. هذا الكائن الشبيه بالمصفوفة يحتوي على قيم المعلمات التي تم تمريرها إلى الدالة.

إليك مثالاً:

:::interactive_editor

```js
function logArgs() {
  for (const arg of arguments) {
    console.log(arg);
  }
}

logArgs(1, 2, 3);
// result:
// 1
// 2
// 3

logArgs("example"); // "example"
```

:::

بما أن كائن `arguments` يشبه المصفوفة، يمكنك الوصول إلى معلمة عند فهرس معين هكذا:

:::interactive_editor

```js
function getArg() {
  return arguments[1];
}

console.log(getArg(2, 4, 6)); // 4
```

:::

يمكنك أيضًا استخدام خاصية `length` هكذا للحصول على عدد المعلمات التي تم استدعاء الدالة بها:

:::interactive_editor

```js
function getArgs() {
  return arguments.length;
}

console.log(getArgs("Example")); // 1
console.log(getArgs("Another", "Example")); // 2
```

:::

على الرغم من أن كائن `arguments` يبدو كأنه مصفوفة حقيقية، إلا أنه لا يحتوي على طرق `Array` المدمجة مثل `includes` أو `push`. للوصول إلى تلك الطرق، تحتاج أولاً إلى تحويل كائن `arguments` إلى مصفوفة حقيقية باستخدام شيء مثل `slice` أو `Array.from()` أو معامل النشر:

:::interactive_editor

```js
function hasCat() {
  return [...arguments].includes("cat");
}

console.log(hasCat("dog", "chicken", "cat")); // true
console.log(hasCat("dog", "chicken", "horse")); // false
```

:::

بينما يمكن العمل مع كائن `arguments` للدوال المتعددة المعلمات، فإن تطبيقات JavaScript الحديثة عادةً ما تستخدم صيغة معلمات الباقي. ستتعلم المزيد عن ذلك في درس لاحق.
