# --description--

Pig Latin — це спосіб змінювати англійські слова, дотримуючись певних правил трансформації.

**Мета:** Виконайте наведені нижче історії користувача та пройдіть усі тести, щоб завершити лабораторну роботу.

**Історія користувача:**

1. Ви повинні створити `translatePigLatin` функцію, яка приймає один рядок як аргумент.
1. Якщо рядок-аргумент починається з приголосної, ваша функція має взяти першу приголосну або кластер приголосних, перемістити її в кінець слова, додати `ay` і повернути результат.
1. Якщо рядок-аргумент починається з голосної, ваша функція має додати `way` у кінець і повернути результат.
1. Ваша функція має обробляти рядки, де перша голосна знаходиться посередині слова, і повертати відповідно трансформований рядок.
1. Якщо рядок-аргумент не містить голосних, ваша функція має додати `ay` у кінець і повернути результат.

Примітка: Для контексту цієї лабораторної роботи голосними вважаються `a`, `e`, `i`, `o` і `u`. Літера `y` не вважається голосною.

```js
function translatePigLatin(str) {
  if (isVowel(str.charAt(0))) return str + "way";
  let front = [];
  str = str.split('');
  while (str.length && !isVowel(str[0])) {
    front.push(str.shift());
  }
  return [].concat(str, front).join('') + 'ay';
}

function isVowel(c) {
  return ['a', 'e', 'i', 'o', 'u'].indexOf(c.toLowerCase()) !== -1;
}
```
