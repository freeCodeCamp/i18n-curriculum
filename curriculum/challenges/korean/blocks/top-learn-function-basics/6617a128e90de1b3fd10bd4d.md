# --description--

다음 예제 함수에서 매개변수와 인수에 대해 알아보겠습니다:

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

JavaScript에서 매개변수는 함수 선언의 소괄호 `()` 사이에 나열된 항목입니다. 함수 인수는 함수에 전달하기로 결정한 실제 값입니다.

위 예제에서 함수 정의는 첫 번째 줄에 작성되어 있습니다: `function favoriteAnimal(animal)`. 매개변수 `animal`는 소괄호 안에 있습니다. `animal` 대신 `pet`, `x`, 또는 `blah`을 사용해도 무방합니다. 하지만 이 경우 매개변수 이름을 `animal`로 지정하면 코드를 읽는 사람이 `animal`에 무엇이 들어갈지 추측하지 않아도 되도록 약간의 맥락을 제공합니다.
