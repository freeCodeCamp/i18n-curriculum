# --description--

تعرف أن JavaScript تحتوي على أنواع بيانات متنوعة، منها السلاسل النصية، الأعداد، والقيم المنطقية. يمكنك استخدام عوامل المقارنة لمقارنة هذه الأنواع واتخاذ قرارات بناءً على النتائج.

العوامل التالية للمقارنة متاحة في JavaScript:

| العامل | الوصف                                  | المثال      |
|----------|------------------------------------------|--------------|
| ==       | يساوي (مقارنة القيم)                    | x == y       |
| !=       | لا يساوي                                | x != y       |
| >        | أكبر من                                | x > y        |
| <        | أصغر من                                | x < y        |
| >=       | أكبر من أو يساوي                       | x >= y       |
| <=       | أصغر من أو يساوي                       | x <= y       |

لاستخدام مقارنة في تعيين متغير، يمكنك استخدام العامل `==`. على سبيل المثال:

```javascript
let x = 5;
let y = 5;

let result = x == y;

console.log(result); // true
```
