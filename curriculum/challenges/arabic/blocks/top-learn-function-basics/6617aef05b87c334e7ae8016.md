# --description--

كما رأيت سابقًا، يمكن للدوال أن تُرجع قيمة باستخدام الكلمة المفتاحية `return`. تُستخدم الكلمة المفتاحية `return` لإرجاع قيمة من دالة. عند استخدام `return`، تتوقف الدالة عن التنفيذ وتُرجع القيمة المحددة بعد كلمة `return`.

```js
function add(a, b) {
  return a + b
}

console.log(add(2, 3)); // Output: 5
```

لكن ماذا يحدث إذا استُخدمت الكلمة المفتاحية `return` قبل نهاية الدالة؟
للإجابة عن هذا السؤال، انظر المثال التالي:

```js
function add(a, b) {
  if(a > 2){
    return b;
  }

  return a + b;
}

console.log(add(3, 7)); // Output: 7
``` 

في المثال أعلاه، تحتوي الدالة `add` على شرط يتحقق مما إذا كانت قيمة `a` أكبر من `2`. إذا تحقق الشرط، تُرجع الدالة قيمة `b` وتتوقف عن التنفيذ. إذا لم يتحقق الشرط، تُرجع الدالة مجموع `a` و `b`.
