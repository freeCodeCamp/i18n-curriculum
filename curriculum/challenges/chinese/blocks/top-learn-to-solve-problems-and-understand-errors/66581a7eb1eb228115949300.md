# --description--

假设你有两个字串，想要将它们组合以创建一条消息，如下所示：

```js
const str1 = "Hello";
const str2 = "World!";
const message = str1.push(str2);
```

<img src="https://cdn.statically.io/gh/TheOdinProject/curriculum/4ed59981b4ce2c60b5b83bf7415d3127b61821f5/foundations/javascript_basics/understanding_errors/imgs/03.png" style="width:100%" alt="an example of a type error in the Chrome developer console">

这里，你会收到一个 `TypeError`，错误信息指出 `str1.push is not a function`。这是一个常见的错误信息，会让学习者感到困惑，因为你可能知道 `.push()` 确实是一个函数（例如，如果你之前用它为数组添加过项）。

但这就是关键——`.push()` 不是一个字串方法，它是一个数组方法。因此，你无法将它查找为字串方法中的“不是一个函数”。如果你将 `.push()` 改为 `.concat()`，一个合适的字串方法，我们的代码就能按预期运行！

当遇到 `TypeError` 时，一个好的注意事项是考虑你尝试运行方法或操作的数据类型。你很可能会发现它并不是你想象的那样，或者该操作或方法与该类型不兼容。
