# --description--

لكل عدد صحيح $n > 1$، تُعرّف عائلة الدوال $f_{n, a, b}$ كما يلي:

$f_{n, a, b}(x) ≡ ax + b\bmod n$ حيث $a, b, x$ أعداد صحيحة و $0 \lt a \lt n$، $0 \le b \lt n$، $0 \le x \lt n$.

نسمي $f_{n, a, b}$ استدعاءً عكسيًا إذا كان $f_{n, a, b}(f_{n, a, b}(x)) \equiv f_{n, a, b}(x)\bmod n$ لكل $0 \le x \lt n$.

لتكن $R(n)$ عدد الاستدعاءات العكسية لـ $n$.

معطى لك أن

$$\sum_{k = 1}^{99\\,999} R(\displaystyle\binom{100\\,000}{k}) \equiv 628\\,701\\,600\bmod 1\\,000\\,000\\,007$$

احسب $$\sum_{k = 1}^{9\\,999\\,999} R(\displaystyle\binom{10\\,000\\,000}{k})$$ وقدم إجابتك بالباقي عند القسمة على $1\\,000\\,000\\,007$.
