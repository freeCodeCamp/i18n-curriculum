# --interactive--

Тип `Number` є одним із найпоширеніших типів даних у JavaScript та інших мовах програмування. Числа можуть здаватися простими, але в JavaScript є багато цікавого, що стосується чисел. Тож давайте розглянемо це детальніше. У JavaScript тип даних `Number` представляє числове значення.

На відміну від багатьох інших мов програмування, які розділяють цілі числа та числа з плаваючою комою на різні типи, JavaScript використовує один об’єднаний тип `Number` для чисел. Це означає, що ви можете працювати з цілими числами, десятковими дробами та навіть спеціальними числовими значеннями — усе це під одним типом даних `Number`.

Ось базовий приклад, який показує, що цілі числа, числа з плаваючою комою та від’ємні числа — усі мають тип number:

:::interactive_editor

```js
const wholeNumber = 50;
const decimalNumber = 4.5;
const negativeNumber = -7;

console.log(typeof wholeNumber); // number
console.log(typeof decimalNumber); // number
console.log(typeof negativeNumber); // number
```

:::

Тип `Number` у JavaScript включає різні види числових значень — від простих цілих чисел і чисел з плаваючою комою до спеціальних випадків, таких як `Infinity` і `NaN`, або «Not a Number». Розглянемо основні типи, з якими ви зіткнетеся. Цілі числа — це числа без дробової або десяткової частини. Вони можуть бути додатними, від’ємними або нулем. Ось кілька прикладів:

:::interactive_editor

```js
const positiveInteger = 100;
const negativeInteger = -25;
const zero = 0;

console.log(typeof positiveInteger); // number
console.log(typeof negativeInteger); // number
console.log(typeof zero); // number
```

:::

Числа з плаваючою комою — це числа з десятковою крапкою. Розробники JavaScript часто називають їх просто «floats». Floats корисні, коли потрібна більша точність, наприклад, при роботі з вимірами або валютами. Ось кілька прикладів:

:::interactive_editor

```js
const floatingPointNumber = 4.5;
const anotherFloat = 89.56;
const oneMoreFloat = 16.462;

console.log(typeof floatingPointNumber); // number
console.log(typeof anotherFloat); // number
console.log(typeof oneMoreFloat); // number
```

:::

JavaScript може представляти числа, що перевищують максимальний ліміт, за допомогою `Infinity`. Ви зіткнетеся з цим, коли спробуєте поділити число на нуль або в рідкісних випадках перевищите верхню межу типу `Number`. Ось приклад:

:::interactive_editor

```js
const infiniteNumber = 1 / 0;
console.log(infiniteNumber); // Infinity
console.log(typeof infiniteNumber); // number
```

:::

Іноді в JavaScript деякі математичні операції не дають дійсного числа. Наприклад, якщо ви спробуєте виконати арифметичну дію над чимось, що не є числом, ви отримаєте `NaN`, що означає «Not a Number»:

:::interactive_editor

```js
const notANumber = 'hello world' / 2;
console.log(notANumber); // NaN
```

:::

Дивно, але тип `NaN` також є `Number`:

:::interactive_editor

```js
const notANumber = 'hello world' / 2;
console.log(typeof notANumber); // number
```

:::

Окрім стандартної десяткової системи (основа 10), JavaScript підтримує числа в інших системах числення, таких як двійкова, вісімкова та шістнадцяткова. Двійкова — це система з основою 2, яка використовує лише цифри 1 і 0. Вісімкова — система з основою 8, що використовує цифри від 0 до 7. Шістнадцяткова — система з основою 16, яка використовує цифри від 0 до 9 і літери a до f, як у кольорах CSS у шістнадцятковому форматі.
