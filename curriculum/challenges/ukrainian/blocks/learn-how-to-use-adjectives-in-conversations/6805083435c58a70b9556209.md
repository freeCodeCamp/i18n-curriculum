`Let's wrap up this task before lunch.` – Це означає завершити завдання до обіду.

`We ordered all the components for the new smartphone.` – Це стосується всіх окремих частин, необхідних для складання телефону.

`The factory shipped the first product batch last week.` – Це стосується групи продуктів, виготовлених і відправлених разом.
