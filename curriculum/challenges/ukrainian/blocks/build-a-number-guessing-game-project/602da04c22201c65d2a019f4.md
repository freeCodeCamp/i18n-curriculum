# --description--

Це один із необхідних проєктів для отримання вашої сертифікації. Для цього проєкту ви будете використовувати Bash-скрипти, PostgreSQL і Git, щоб створити гру вгадування числа, яка запускається в терміналі та зберігає інформацію про користувача.

# --instructions--

**Важливо:** Після того, як ви пройдете всі тести проєкту, збережіть дамп вашої бази даних у файл `number_guess.sql`, а також ваш файл `number_guess.sh`, щоб ви могли виконати крок 2. Інструкції, як це зробити, будуть у віртуальній машині.
