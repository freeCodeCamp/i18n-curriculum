# --description--

内联 `CSS` 使得可以直接为 `HTML` 元素添加样式，尽管这种方法不太推荐：

```html
<body>
  <div style="color: white; background-color: black;">...</div>
</body>
```

首先需要注意的是这里没有任何选择器，因为样式是直接添加到打开的 `<div>` 标签本身的。接下来，你有 `style` 属性，其值位于引号对内，是声明。

如果你需要为单个元素添加唯一样式，这种方法完全可行。不过，通常来说，出于几个原因，这并不是为超文本标记语言添加 CSS 的推荐方法：

一旦你开始为单个元素添加大量声明，文件很快就会变得相当混乱，导致你的 `HTML` 文件变得不必要地臃肿。  
如果你想让许多元素拥有相同的样式，你必须将相同的样式拷贝并粘贴到每个单独的元素上，导致大量不必要的重复和更多的臃肿。  
任何内联 `CSS` 都会覆写另外两种方法，这可能导致意想不到的结果。（虽然你这里不会深入，但实际上可以利用这一点）。
