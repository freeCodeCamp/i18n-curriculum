# --interactive--

Der Attributselektor in CSS ist ein mächtiges Werkzeug, mit dem Sie HTML-Elemente basierend auf ihren Attributen ansprechen können. Das bedeutet, Sie können Stile auf Elemente anwenden, die bestimmte Attribute oder sogar bestimmte Werte für diese Attribute besitzen.

Er ist besonders nützlich, wenn Sie Elemente dynamisch gestalten möchten oder wenn Klassennamen allein nicht genug Spezifität bieten.

Zum Beispiel können Sie den Attributselektor verwenden, um alle Links anzusprechen, die ein `href` Attribut besitzen. Das ist besonders praktisch, wenn Sie allen Links auf einer Seite ein einheitliches Styling geben möchten.

So können Sie den Attributselektor verwenden, um Links mit dem `href` Attribut anzusprechen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Example link with an href attribute</a>
<a>Example link without an href attribute</a>
```

```css
a {
  display: block;
}

a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

Diese CSS-Regel wendet eine blaue Farbe und eine Unterstreichung auf jeden Link an, der ein `href` Attribut enthält, sodass alle anklickbaren Links einheitlich gestylt sind.

Aber Attributselektoren können noch spezifischer sein. Angenommen, Sie möchten nur die Links ansprechen, die ein `title` Attribut besitzen. Das `title` Attribut liefert oft zusätzliche Informationen über den Link, und Sie möchten diese Links vielleicht anders gestalten, um anzuzeigen, dass sie zusätzliche Informationen enthalten. So geht das:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

Diese Regel macht den Text fett und entfernt die Unterstreichung bei jedem Link mit einem `title` Attribut. Das ist eine gute Möglichkeit, diese Links optisch von anderen auf der Seite zu unterscheiden.

Sie können auch Attributselektoren kombinieren, um noch präziser zu sein. Wenn Sie zum Beispiel Links stylen möchten, die sowohl `href` als auch `title` Attribute besitzen, können Sie das so machen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[href][title] {
  display:block;
  color: green;
}
```

:::

In diesem Fall werden nur Links, die sowohl `href` als auch `title` Attribute enthalten, mit grünem Text gestylt. Diese Kontrolle macht Attributselektoren in CSS so mächtig.

Ein weiteres Beispiel für einen Attributselektor ist das Abgleichen eines einzelnen Werts innerhalb einer durch Leerzeichen getrennten Liste von Werten in einem Attribut.

Hier ist ein Beispiel für ein Ankerelement mit mehreren Klassen:

```html
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

Um genau dieses Ankerelement anzusprechen, können Sie folgenden Selektor verwenden:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

Die `[attr~=value]` Syntax wird hier verwendet, um alle Ankerelemente anzusprechen, bei denen das class-Attribut das Wort `"primary"` enthält.

Wenn Sie ein Element ansprechen müssen, bei dem der Attributwert mit einem bestimmten Wert beginnt, können Sie die `[attr^=value]` Syntax verwenden. Hier ein Beispiel:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

In diesem Beispiel spricht der `a[href^="https://"]` Selektor alle Ankerelemente an, bei denen der Wert des `href` Attributs mit `"https://"` beginnt.

Um Elemente anzusprechen, bei denen der Attributwert mit einem bestimmten Wert endet, können Sie die `[attr$=value]` Syntax verwenden. Hier ein Beispiel:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Visit Example Site</a>
```

```css
a[href$=".com"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

In diesem Beispiel spricht der `a[href$=".com"]` Selektor alle Ankerelemente an, bei denen der Wert des `href` Attributs mit `.com` endet.

Die Verwendung von Attributselektoren verbessert nicht nur das Styling Ihrer Webseite, sondern erhöht auch die Barrierefreiheit, indem interaktive Elemente wie Links anhand ihrer Attribute besser unterscheidbar gemacht werden.
