# --description--

У цьому завданні ви дізнаєтеся різницю між `specific` та `specify`. `Specific` — це прикметник, який описує щось чітко визначене або детальне, тоді як `specify` — це дієслово, що означає викласти щось докладно.

Наприклад, `specific` інструкція може бути `use port 8080`, а `specify` означає викласти цю деталь: `specify the port number.`

`Setup` стосується організації або підготовки чогось, часто використовується у контексті підготовки програмного забезпечення або обладнання, як-от `setup instructions for the software.`
