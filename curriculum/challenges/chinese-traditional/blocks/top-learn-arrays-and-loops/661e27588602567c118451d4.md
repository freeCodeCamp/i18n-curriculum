# --description--

陣列中較複雜的方法之一是 `splice()` 和 `slice()` 方法。`splice()` 方法透過移除或取代陣列中的元素來改變陣列的內容。`slice()` 方法會從 `begin` 到 `end`（不包含 `end`）選取陣列的一部分，並傳回該部分的淺層複本到新的陣列物件。原始陣列不會被修改。

例如，要從 `characters` 陣列中移除第二個元素，你可以使用以下程式碼：

```javascript
const characters = ['Harry', 'Ron', 'Hermione'];
characters.splice(1, 1);
console.log(characters); // Output: ['Harry', 'Hermione']
```

上述元素會從 `characters` 陣列中移除第二個元素。`splice()` 方法需要兩個引數：要移除元素的索引以及要移除的元素數量。


要使用以下程式碼，為你的 `character` 陣列創建一個包含第二個元素的新陣列：

```javascript
const characters = ['Harry', 'Ron', 'Hermione'];
const newCharacters = characters.slice(1, 2);
console.log(newCharacters); // Output: ['Ron']
```

上述程式碼創建了一個新的陣列 `newCharacters`，其中包含來自 `characters` 陣列的第二個元素。`slice()` 方法接受兩個引數：開始切割的元素索引和結束切割的元素索引（不包含該元素）。

## --text--

以下 JavaScript 程式碼片段的輸出會是什麼？


```javascript
const numbers = [10, 20, 30, 40, 50];
numbers.splice(3, 1);
const slicedNumbers = numbers.slice(2, 4);

console.log(numbers);
console.log(slicedNumbers);
```

## --answers--

`numbers` 輸出：`[10, 20, 30, 50]` 和 `slicedNumbers` 輸出：`[30, 50]`
