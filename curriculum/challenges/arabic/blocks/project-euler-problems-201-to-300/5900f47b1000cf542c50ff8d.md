# --description--

لعدد موجب $n$، عرّف $S(n)$ كمجموع الأعداد الصحيحة $x$ التي تحقق $1 < x < n$ و$x^3 \equiv 1\bmod n$.

عندما يكون $n = 91$، هناك 8 قيم ممكنة لـ $x$، وهي: 9، 16، 22، 29، 53، 74، 79، 81. إذن، $S(91) = 9 + 16 + 22 + 29 + 53 + 74 + 79 + 81 = 363$.

احسب $S(13\\,082\\,761\\,331\\,670\\,030)$.
