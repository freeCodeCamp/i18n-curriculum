# --description--

`tech recruiter` 是指專注於招募科技相關職位候選人的專業角色。例如：

`The tech recruiter is looking for a Python developer.`－這表示招募者的工作是尋找具備 Python 技能的人。

這個角色涉及了解科技產業、識別經過資格修飾、合格的候選人，並將他們與合適的科技職缺配對。
