# --description--

Normalmente, o tempo `Past Simple` é formado adicionando `-ed` a verbos regulares. Por exemplo, o verbo `play` se torna `played`.

Brincar de `play around` significa tentar algo de forma casual ou experimental, muitas vezes apenas para ver o que acontece. Geralmente é seguido por `with`. Por exemplo:

`He played around with the new software to see how it works.` - Isso significa que ele tentou usar o software sem um plano sério, apenas para explorá-lo.
