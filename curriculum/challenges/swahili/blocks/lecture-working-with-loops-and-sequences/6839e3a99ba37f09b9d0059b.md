# --description--

Tuple ni aina ya data ya Python inayotumika kuunda mfululizo wa thamani ulio pangwa. Tuple zinaweza kuwa na mchanganyiko wa aina za data kama hizi:

```py
developer = ('Alice', 34, 'Rust Developer')
```

Tuple zinafanana na orodha, lakini wakati orodha ni aina ya data inayobadilika, tuple ni isiyobadilika. Hii inamaanisha kuwa vipengele ndani ya tuple haviwezi kubadilishwa mara tu inapoundwa.

Kama utajaribu kusahihisha kipengee kimoja ndani ya tuple, utapata `TypeError`:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: 'tuple' object does not support item assignment
"""
```

Ili kupata kipengee kutoka kwenye tuple, unaweza kutumia uandishi wa mabano na nambari ya kielezo:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

Kama unahitaji kupata vipengee kuanzia mwisho wa tuple, basi unaweza kutumia uandishi wa kielezo hasi. Hapa kuna mfano wa kutumia kielezo hasi kupata kipengee cha pili kutoka mwisho kwenye tuple:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

Kama utajaribu kuingiza nambari ya kielezo inayozidi au sawa na urefu wa tuple, basi utapokea `IndexError` kama hii:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list index out of range
"""
```

Njia nyingine ya kuunda tuple ni kwa kutumia kijenzi cha `tuple()` kama hii:

```py
developer = 'Jessica'
tuple(developer) # ('J', 'e', 's', 's', 'i', 'c', 'a')
```

Kwa kijenzi cha `tuple()`, unaweza kuingiza vitu mbalimbali vinavyoweza kurudiwa kama mfuatano wa herufi, orodha na hata tuple nyingine.

Ili kuangalia kama kipengee kiko ndani ya tuple, unaweza kutumia neno kuu la `in` kama hii:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

Pia unaweza kutoa vipengee kutoka kwenye tuple kama ulivyofanya na orodha:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

Katika mfano huu, `name` ina thamani ya `'Alice'`, `age` ina thamani ya `34`, na `job` ina thamani ya `'Rust Developer'`.

Kama unahitaji kukusanya vipengee vyote vilivyobaki kutoka kwenye tuple, unaweza kutumia opereta ya nyota (`*`) kama hii:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

Hapa, `name` ina thamani ya `'Alice'`, na `rest` ni orodha iliyojumuisha nambari `34` na mfuatano wa herufi `'Rust Developer'`.

Kama ilivyo kwa orodha, unaweza kutumia opereta ya kukata (slice) kwenye tuple ili kutoa sehemu yake. Hapa kuna mfano wa kutoa vipengee `'pie'` na `'cookies'` katika tuple tofauti:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

Kumbuka kuwa nambari ya kwanza inaonyesha kielezo cha kuanzia kwa kutoa sehemu hiyo wakati nambari ya pili inaonyesha kielezo cha mwisho. Lakini kumbuka kuwa kipengee kilicho kwenye kielezo cha mwisho hakijajumuishwa katika tuple iliyotolewa.

Kama unahitaji kufuta kipengee kutoka kwenye tuple, haiwezekani kwa sababu tuple ni isiyobadilika. Kwa hivyo mfano huu, utasababisha kosa:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

Basi ni lini utatumia tuple badala ya orodha?

Kama unahitaji mkusanyiko wa vipengee unaobadilika ambapo unaweza kuongeza, kufuta na kusahihisha vipengee, basi unapaswa kutumia orodha. Kama unajua kuwa unafanya kazi na mkusanyiko wa data ulio thabiti na usiobadilika, basi unapaswa kutumia tuple.

Katika funzo lijalo, tutaangalia baadhi ya njia za kawaida utakazotumia unapotumia tuple.
