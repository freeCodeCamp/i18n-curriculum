## --text--

Wann wird eine `ArrayTypeMismatchException`-Ausnahme geworfen?

## --answers--

Eine `ArrayTypeMismatchException`-Ausnahme wird geworfen, wenn versucht wird, einen Wert eines Typs in ein Array eines anderen Typs zu speichern.

---

Eine `ArrayTypeMismatchException`-Ausnahme wird geworfen, wenn versucht wird, auf ein Mitglied eines Typs zuzugreifen, dessen Wert null ist.

---

Eine `ArrayTypeMismatchException`-Ausnahme wird geworfen, wenn versucht wird, ein Array außerhalb seiner Grenzen zu indizieren.
