# --description--

In der vorherigen Lektion haben Sie gelernt, wie man mit dem Datentyp Tuple arbeitet. In dieser Lektion lernen Sie einige gängige Methoden kennen, die Sie beim Arbeiten mit Tuples verwenden.

Die erste Methode, die wir behandeln, ist `count()`. Diese Methode wird verwendet, um zu bestimmen, wie oft ein Element in einem Tuple vorkommt. Hier ist ein Beispiel, wie man überprüft, wie oft der String `Rust` in einem Tuple namens `programming_languages` vorkommt:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

Da `Rust` zweimal im Tuple vorkommt, gibt die `count()`-Methode die Zahl `2` zurück. Wenn das angegebene Element in der `count()`-Funktion überhaupt nicht im Tuple vorhanden ist, dann ist der Rückgabewert `0`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

Wenn keine Argumente an die `count()`-Funktion übergeben werden, löst Python einen `TypeError` aus:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count()

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: tuple.count() takes exactly one argument (0 given)
"""
```

Die nächste Methode, die wir uns ansehen, ist die `index()`-Methode. Diese Methode wird verwendet, um den Index zu finden, an dem ein bestimmtes Element in einem Tuple vorhanden ist. Hier ist ein Beispiel für die Verwendung der `index()`-Methode, um den Index für den String `Java` zu finden:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

Wenn das angegebene Element nicht gefunden werden kann, löst Python einen `ValueError` aus:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: tuple.index(x): x not in tuple
"""
```

Eine weitere Möglichkeit, die Sie mit der `index()`-Methode haben, ist das Übergeben optionaler Start- und Stopp-Index-Argumente. Hier ist ein Beispiel für das Übergeben eines optionalen Start-Index:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

In diesem Beispiel geben wir an, wo die Suche nach dem String `Python` beginnen soll. Indem wir die Zahl `3` als zweites Argument an die `index()`-Funktion übergeben, geben wir an, dass die Suche bei Index `3` beginnen soll. Da `Python` zweimal im Tuple vorkommt, gibt die `index()`-Funktion den Index `5` zurück und nicht den Index `2`, wegen der Verwendung des optionalen Start-Index-Arguments.

Sie können auch einen optionalen Stopp-Index übergeben. Hier ist ein modifiziertes Beispiel, bei dem ein Start- und Stopp-Index angegeben wird:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

Jetzt ist das Ergebnis der Index `2`, weil wir die Suche bei Index `2` starten und bis, aber nicht einschließlich, Index `5` suchen.

Eine weitere häufig verwendete Funktion bei Tuples ist die `sorted()`-Funktion. In einer früheren Lektion haben Sie die `sort()`-Methode für Listen kennengelernt. Nun, die `sorted()`-Funktion kann für jedes iterierbare Objekt verwendet werden, einschließlich Tuples.

Hier ist ein Beispiel, wie man mit der `sorted()`-Funktion eine neue Liste von Zahlen erstellt:

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

Die `sorted()`-Funktion erstellt immer eine neue Liste mit den sortierten Werten. Das unterscheidet sich von der `sort()`-Methode, die die Elemente einer Liste an Ort und Stelle sortiert und keine neue Liste zurückgibt.

Wenn Sie das Sortierverhalten für ein iterierbares Objekt anpassen müssen, können Sie die optionalen Argumente `reverse` und `key` verwenden. Hier ist ein Beispiel für die Verwendung des `key`-Arguments, um Elemente in einem Tuple nach Länge zu sortieren:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

Wenn Sie eine neue Liste von Werten in umgekehrter Reihenfolge erstellen möchten, können Sie das `reverse`-Argument so verwenden:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

Tuples sind ein gängiger Datentyp in Python. Zu verstehen, wie man mit ihnen arbeitet, zusammen mit einigen hilfreichen Methoden und Funktionen, wird Ihnen helfen, effizienteren Code zu schreiben.
