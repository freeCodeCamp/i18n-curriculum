# --description--

定義 $Co(n)$ 為從 $\\{1, 2, \ldots, n\\}$ 中互質元素集合的最大可能總和。 例如 $Co(10)$ 是 30，並且在子集合 $\\{1, 5, 7, 8, 9\\}$ 上達到該最大值。

你已知 $Co(30) = 193$ 和 $Co(100) = 1356$。

尋找 $Co(200\,000)$。
