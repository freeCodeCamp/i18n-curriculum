# --description--

このラボでは、ソートアルゴリズムであるクイックソートを実装します。

クイックソートは、配列をソートする効率的な再帰的分割統治法です。この方法では、元の配列からピボット値を選びます。配列はピボット値より小さい値と大きい値の2つのサブ配列に分割されます。その後、両方のサブ配列に対してクイックソートアルゴリズムを再帰的に呼び出した結果を結合します。これを空の配列または要素が1つの配列という基本ケースに達するまで繰り返し、その配列を返します。再帰呼び出しの巻き戻しによってソート済みの配列が返されます。

ピボット値の選択は重要ですが、ここではどのピボットでも問題ありません。簡単のため、最初または最後の要素を使うことができます。

クイックソートは非常に効率的なソート方法で、平均して*O(nlog(n))*の性能を発揮します。また比較的簡単に実装できます。これらの特徴により、人気があり有用なソート方法です。

**目的:** 以下のユーザーストーリーを満たし、すべてのテストに合格してラボを完了してください。

**ユーザーストーリー:**

1. `quicksort` 関数は整数の配列を入力として受け取る必要があります。
1. `quicksort` 関数は入力と同じ整数を含み、最小から最大の順に並んだ配列を返す必要があります。
1. `quicksort` 関数は配列をソートするために再帰的に自分自身を呼び出す必要があります。

# --hints--

`quicksort` 関数を用意してください。

```js
assert.isFunction(quicksort);
```

`quicksort([1, 4, 2, 8, 345, 123, 43, 32, 5643, 63, 123, 43, 2, 55, 1, 234, 92])` は順序以外は変更されていない配列を返す必要があります。

```js
assert.sameMembers(
  quicksort([1, 4, 2, 8, 345, 123, 43, 32, 5643, 63, 123, 43, 2, 55, 1, 234, 92]),
  [1, 4, 2, 8, 345, 123, 43, 32, 5643, 63, 123, 43, 2, 55, 1, 234, 92]
);
```

`quicksort` は昇順にソートされた配列を返す必要があります。

```js
assert.sameOrderedMembers(
  quicksort([1, 4, 2, 8, 345, 123, 43, 32, 5643, 63, 123, 43, 2, 55, 1, 234, 92]),
  [1, 1, 2, 2, 4, 8, 32, 43, 43, 55, 63, 92, 123, 123, 234, 345, 5643]
);
```

`quicksort` は組み込みの `.sort()` メソッドを使わないでください。

```js
function isBuiltInSortUsed(){
  let sortUsed = false;
  const temp = Array.prototype.sort;
  Array.prototype.sort = () => sortUsed = true;
  try {
    quicksort([0, 1]);
  } finally {
    Array.prototype.sort = temp;
  }
  return sortUsed;
}
assert.isFalse(isBuiltInSortUsed());
```

```js
function quicksort(array) {
  if (array.length === 0) {
    return [];
  } else {
    const pivotValue = array[0];

    // Sort elements into three piles
    let lesser = [];
    let equal = [];
    let greater = [];
    for (let e of array) {
      if (e < pivotValue) {
        lesser.push(e);
      } else if (e > pivotValue) {
        greater.push(e);
      } else {
        equal.push(e);
      }
    }

    return [...quicksort(lesser), ...equal, ...quicksort(greater)];
  }
}
```
