# --explanation--

`User experience`（或 `UX`）是關於使用者在使用產品時的感受——是否簡單、愉快且順暢。例如：

`Improving page load time boosts user experience。` — 讓頁面載入更快有助於使用者更享受應用程式。

`Enhancements` 是為了改善產品或處理程序所做的變更。例如：

`The latest update includes performance enhancements。` — 新版本讓應用程式運作得更好、更快。

在 Alice 的句子中，一半的設計預算花在讓產品更容易且更好使用上。
