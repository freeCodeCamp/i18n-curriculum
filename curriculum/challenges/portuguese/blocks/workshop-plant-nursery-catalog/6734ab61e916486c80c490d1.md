# --description--

Quando o valor associado a uma chave é um objeto, o método `get` retornará uma referência para esse objeto.Você pode usar essa referência para modificar o objeto dentro do `map`.

Para testar, defina a propriedade `small` de `getBallerina` para um inteiro diferente de `20`.Então, olhe no console do navegador para verificar se o valor foi alterado no seu catálogo.
