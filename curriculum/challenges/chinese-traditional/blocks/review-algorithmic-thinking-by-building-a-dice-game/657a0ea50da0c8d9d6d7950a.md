# --description--

在這個專案中，你將透過建置一個骰子遊戲來學習演算法思維。共有 6 輪，每一輪中，玩家最多可以擲骰子 3 次並累積分數。 

HTML 和 CSS 已經為你提供。隨意探索它們。當你準備好時，你需要設定你的 HTML 變數。取得所有的 `.die` 元素並將它們指定給 `listOfAllDice` 變數。取得你的分數輸入（`#score-options` 中的 `input` 元素）和分數區段，並將它們指定給 `scoreInputs` 和 `scoreSpans`。將 `#current-round` 元素指定給 `roundElement`，將 `#current-round-rolls` 元素指定給 `rollsElement`，然後對你的 `#total-score` 和 `#score-history` 元素做相同的操作。將你的 `#roll-dice-btn`、`#keep-score-btn`、`#rules-btn` 和 `.rules-container` 指定給格式正確的變數名稱。

當使用者點擊 `Show rules` 按鈕時，他們應該能夠切換顯示和隱藏遊戲規則。創建一個變數 `isModalShowing` 來追蹤遊戲規則的狀態。

每次使用者擲骰子時，你都需要追蹤所有骰子的數值。創建一個變數 `diceValuesArr` 來追蹤這些數值。

在整個遊戲過程中，你需要追蹤當前分數、總分數、擲骰次數以及玩家所處的回合。為此目的宣告 `rolls`、`score` 和 `round` 變數。

思考這些變數中每一個的起始值應該是什麼。所有這些變數都應該能夠被重新指定。
