# --description--

API 中的 `endpoint` 是你可以請求特定資料或服務的特定位置。例如，`The API endpoint for weather data retrieves the latest forecasts.` 

 `offer` 意味著提供或使可用。例如，`This service offers real-time updates.`

`route` 是 API 中定義的一條路徑，用來決定請求如何被處理。例如，`The login route verifies user credentials.`
