# --explanation--

Un `server` es una computadora o un sistema que administra y almacena información, y proporciona servicios, datos o recursos a otras computadoras o dispositivos, usualmente a través de una red. Por ejemplo:

`The company's website is hosted on secure servers.` – Esto significa que los datos del sitio web están almacenados en computadoras diseñadas para gestionar y servir esos datos.
