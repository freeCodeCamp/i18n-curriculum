# --description--

Os estilos para o elemento `div` estão começando a se formar.Mas seria bom se houvesse algum espaço ao redor do texto. 

Para aplicar `padding` a um elemento usando Tailwind CSS, você pode usar o seguinte padrão: `p-number`, onde `number` representa os pixels que você deseja aplicar em todos os quatro lados. 

Aqui está um exemplo de aplicação de `padding` de `8px` a um elemento:

```html
<p class="p-8">This is padding.</p>
```

Dentro da sua tag de abertura `div`, use a classe utilitária correta para aplicar `padding` de `4px`.

# --hints--

Sua tag de abertura `div` deve ter a classe utilitária `p-4`.

```js
const divEl = document.querySelector("div");
assert.isTrue(divEl.classList.contains("p-4"));
```
