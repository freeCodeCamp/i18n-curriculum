# --description--

خلال الدروس السابقة، اعتدت على تعريف الدوال باستخدام الكلمة المفتاحية `def` بهذه الطريقة:

```py
def square(num):
    return num ** 2

print(square(4)) # 16
```

لكن عند العمل مع الدوال عالية الرتبة مثل `map()` و `filter()`، يمكنك استخدام دالة مجهولة مضمَّنة. هنا يأتي دور دوال lambda.

إليك كيف تبدو الدالة `square()` عند إعادة صياغتها كدالة lambda:

```py
lambda num: num ** 2
```

كما ذُكر سابقًا، دوال lambda مجهولة، لذا لم تعد هذه الدالة مرتبطة بالاسم `square`. دوال lambda رائعة عندما تحتاج لاستخدامها في دوال عالية الرتبة مثل هذه:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

في هذا المثال، لدينا قائمة أرقام ونريد إنشاء قائمة جديدة بالأرقام الزوجية. لذلك نمرر دالة lambda كواحدة من المعلمات إلى دالة `filter()` للحصول على قائمة جديدة تحتوي الأرقام `2` و `4`.

عند العمل مع دوال lambda، من المهم أن تكون على دراية بأفضل الممارسات. على سبيل المثال، ليس من الجيد تعيين دالة lambda إلى متغير بهذه الطريقة:

```py
numbers = [1, 2, 3, 4, 5]

square = lambda x: x ** 2
squared_numbers = list(map(square, numbers))
print(squared_numbers) # [1, 4, 9, 16, 25]
```

هذا يُفقد الهدف من استخدام الدوال المجهولة. في هذه الحالة، يجب استخدام دالة عادية، مثل هذه:

```py
numbers = [1, 2, 3, 4, 5]

def square(num):
    return num ** 2

squared_numbers = list(map(square, numbers))
print(squared_numbers) # [1, 4, 9, 16, 25]
```

أيضًا، يجب تجنب إنشاء دوال lambda يصعب قراءتها أو معقدة بلا داعٍ، مثل هذه:

```py
result = (lambda x: (x**2 + 2*x - 1) if x > 0 else (x**3 - x + 4))(3)
print(result)  # 14
```

بينما تعمل هذه الدالة بشكل جيد وتنتج النتيجة الصحيحة `14`، إلا أنها ليست سهلة القراءة أو الفهم. في هذه الحالة، من الأفضل إنشاء دالة منفصلة تحتوي على جملة `if/else`، ثم استدعاء تلك الدالة:

```py
def calculate_expression(x):
    if x > 0:
        return x**2 + 2*x - 1
    else:
        return x**3 - x + 4

print(calculate_expression(3))  # 14
```

لكل من الدوال العادية ودوال lambda استخداماتها في برامج Python. إذا كنت تتعامل مع تعبير مضمَّن واحد، فقد تفكر في استخدام دالة lambda. وإلا، فإن استخدام دالة عادية هو الخيار الأفضل.
