# --description--

`Touch`는 종종 무언가와 물리적으로 접촉하는 것을 의미합니다. 프로그래밍과 같은 비유적인 맥락에서는 무언가를 변경하거나 영향을 미친다는 뜻으로도 사용됩니다.

`These`는 화자와 가까이 있거나 현재 활발히 논의되고 있는 특정 항목을 가리킬 때 사용됩니다.

이번 도전 과제에서는 `can`와 `can't`의 차이를 검토하여 사람의 능력 또는 무능력을 표현하는 방법을 살펴보세요.

Tom이 말하는 내용을 듣고 빈칸을 채우세요.
