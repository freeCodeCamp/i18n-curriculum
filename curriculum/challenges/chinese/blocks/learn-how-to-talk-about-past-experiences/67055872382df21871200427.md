# --explanation--

`Properly` 是形容词 `proper` 的副词形式。这两个词都可以表示以正确或合适的方式做某事，但它们的用法取决于句子结构。例如：  

- `The application didn't function properly after the update.`——这意味着更新给代码带来了误差。副词 `properly` 修饰动词 `function`。

- `This wasn't a proper configuration of the database.` - 形容词 `proper` 描述名词 `configuration`。

这里，James 使用 `properly` 来描述系统没有以正确的方式连接到数据库。
