# --interactive--

JavaScript에서는 모든 산술 연산자가 복합 대입 형태를 가지고 있습니다. 복합 대입 연산자는 변수에 대한 연산을 수행한 후 그 결과를 같은 변수에 저장하는 간결한 축약형을 제공합니다. 이들은 연산과 대입을 `x += y`처럼 더 짧은 형태로 결합하는데, 이는 변수 이름을 반복하지 않고 `x = x + y`을 작성하는 것과 같습니다. 예를 들어 이렇게 작성하는 대신:

:::interactive_editor

```js
let num = 5;
num = num + 2;

console.log(num); // 7
```

:::

다음과 같이 작성할 수 있습니다:

:::interactive_editor

```js
let num = 5;
num += 2;

console.log(num); // 7
```

:::

`num += 2`가 덧셈과 대입 단계를 하나로 결합하는 방식을 주목하세요. 이렇게 하면 시간을 절약하고 코드의 복잡함을 줄일 수 있습니다. 이제 JavaScript에서 가장 흔한 복합 대입 연산자들을 더 자세히 살펴보겠습니다.

이미 보았듯이 `+=` 연산자는 기존 변수에 값을 더할 수 있게 해줍니다. 이것은 덧셈 대입 연산자로 알려져 있습니다. 덧셈 대입 연산자는 변수의 현재 값에 지정된 수를 더한 후 그 결과를 다시 변수에 할당합니다:

:::interactive_editor

```js
let total = 10;
total += 5;

console.log(total); // 15
```

:::

추측할 수 있듯이 `-=`로 표시되는 뺄셈 대입 연산자도 있습니다. 뺄셈 대입 연산자는 변수의 현재 값에서 지정된 값을 빼고 새 값을 다시 변수에 할당합니다:

:::interactive_editor

```js
let score = 20;
score -= 7;

console.log(score); // 13
```

:::

뺄셈 대입을 사용하지 않았다면 다음과 같이 했을 것입니다:

:::interactive_editor

```js
let score = 20;
score = score - 7;

console.log(score); // 13
```

:::

곱셈 대입 연산자는 `*=`로 나타냅니다. 변수의 현재 값에 지정된 수를 곱하고 그 결과를 다시 변수에 할당합니다:

:::interactive_editor

```js
let points = 5;
points *= 3;

console.log(points); // 15
```

:::

마지막으로 `/=`로 표시되는 나눗셈 대입 연산자가 있습니다. 앞서 언급한 연산자들과 마찬가지로 변수의 현재 값을 지정한 수로 나눈 후 결과를 다시 변수에 할당합니다:

:::interactive_editor

```js
let balance = 100;
balance /= 4;

console.log(balance); // 25
```

:::

JavaScript의 모든 연산자마다 복합 대입 연산자가 있다는 점을 기억하세요. 앞서 언급한 네 가지 외에도 다음과 같은 연산자가 있습니다:

- 나머지 대입 연산자(`%=`)는 변수를 지정된 수로 나눈 후 나머지를 변수에 할당합니다.

- 거듭제곱 대입 연산자(`**=`)는 변수를 지정된 수의 거듭제곱으로 올리고 결과를 다시 변수에 할당합니다.

- 비트 AND 대입 연산자(`&=`)는 지정된 수와 비트 단위 AND 연산을 수행하고 결과를 변수에 다시 할당합니다.

- 비트 OR 대입 연산자(`|=`)는 지정된 수와 비트 단위 OR 연산을 수행하고 결과를 변수에 다시 할당합니다.
