# --explanation--

`To be based on something`는 아이디어, 계획, 결정이 그것을 기반이나 출발점으로 삼는다는 뜻입니다. 최종 선택에 영향을 주거나 형성하는 것을 보여줍니다. 예를 들어:

- `Their new pricing model is based on customer feedback.` – 고객의 의견과 제안을 바탕으로 가격을 책정했다는 의미입니다.

- `The movie is based on a true story.` – 실제 이야기가 영화에 나온 사건들에 영감을 주었다는 의미입니다.
