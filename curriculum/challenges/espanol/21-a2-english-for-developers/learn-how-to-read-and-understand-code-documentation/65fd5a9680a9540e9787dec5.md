# --description--

`Overwhelming` significa sentirse como que algo es demasiado para manejar.

Por ejemplo, `The amount of information was overwhelming.` (significa que era mucho!)

`Effective` es un adjetivo que describe algo que produce un resultado deseado. Como, `An effective method`.

`Effectively` es un adverbio, que significa de una manera `effective`. Por ejemplo, `He completed the task effectively.`
