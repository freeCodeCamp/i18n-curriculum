# --interactive--

Kipengele cha `script` kinatumika kuingiza msimbo unaotekelezwa. Wasanidi programu wengi hutumia hili kutekeleza msimbo wa JavaScript. JavaScript hutumika kuongeza mwingiliano kwenye kurasa zako za mtandao. Mifano ya kawaida ya kutumia JavaScript ni pamoja na michezo ya mwingiliano, sliders za picha, na fomu zinazobadilika ambazo zinathibitisha ingizo la mtumizi kwa wakati halisi.

Hapa kuna mfano wa kutumia kipengele cha `script` katika hati ya HTML. Futa `//` iliyoko mbele ya `alert("Welcome to freeCodeCamp");` na utapaswa kuona onyo likitokea. Ili kuona mapitio, utahitaji kuwezesha mhariri wa mwingiliano.

:::interactive_editor

```html
<body>
  <script>
    // alert("Welcome to freeCodeCamp");
  </script>
</body>
```

:::

Ingawa kiufundi unaweza kuandika msimbo wako wote wa JavaScript ndani ya lebo za `script`, inachukuliwa kuwa mbinu bora kuunganisha na faili ya JavaScript ya nje badala yake. Hapa kuna mfano wa kutumia kipengele cha `script` kuunganisha na faili ya JavaScript ya nje:

```html
<script src="path-to-javascript-file.js"></script>
```

Sifa ya `src` hutumika hapa kubainisha mahali pa faili hiyo ya JavaScript ya nje. `src` inasimama kwa "source" (chanzo). Sababu ya kutopendekeza kuweka msimbo wako wote wa JavaScript ndani ya hati ya HTML ni kwa sababu ya mgawanyo wa majukumu. Mgawanyo wa majukumu ni kanuni ya muundo ambapo unagawa programu zako katika sehemu tofauti na kila sehemu inashughulikia jukumu tofauti. Katika kesi hii, tunataka kutenganisha msimbo wetu wa JavaScript na msimbo wetu wa HTML.
