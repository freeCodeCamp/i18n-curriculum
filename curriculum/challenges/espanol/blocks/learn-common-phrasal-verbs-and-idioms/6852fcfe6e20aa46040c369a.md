# --explanation--

`Perhaps` significa "quizás" o "posiblemente". Se usa cuando alguien no está completamente seguro pero quiere sugerir algo. Por ejemplo:

`Perhaps we can try a new tool for project tracking.` – Esto significa que el hablante está sugiriendo la idea como una posibilidad.

El prefijo `un-` se añade a los adjetivos para crear el significado opuesto. En `unnecessary`, la palabra base es `necessary` (que significa necesario), y `un-` hace que signifique "no necesario". Otro ejemplo:

`It was an unfair decision.` – `Unfair` es lo opuesto a `fair`, que significa no justo o no igual.
