# --description--

令 $x_1, x_2, \ldots, x_n$ 為長度為 $n$ 的序列，使得：

- $x_1 = 2$
- 對所有 $1 < i ≤ n : x_{i - 1} < x_i$
- 對所有 $i$ 和 $j$，其中 $1 ≤ i, j ≤ n$：${(x_i)}^j < {(x_j + 1)}^i$

只有五個長度為 2 的此類序列，分別是：{2,4}、{2,5}、{2,6}、{2,7} 和 {2,8}。有 293 個長度為 5 的此類序列；以下給出三個範例：{2,5,11,25,55}、{2,6,14,36,88}、{2,8,22,64,181}。

令 $t(n)$ 表示長度為 $n$ 的此類序列數量。已知 $t(10) = 86195$ 且 $t(20) = 5227991891$。

求出 $t({10}^{10})$ 並將你的答案對 $10^9$ 取模。
