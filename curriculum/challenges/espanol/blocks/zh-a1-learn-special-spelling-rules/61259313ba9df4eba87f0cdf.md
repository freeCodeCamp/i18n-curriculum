# --description--

`w` también es un símbolo ortográfico. Se usa cuando `u` es el sonido vocálico principal en una sílaba. Ayuda a clarificar los límites de las sílabas y hace que las sílabas sean más fáciles de leer.

- Si `u` se usa como una sílaba, añade `w` delante de ella: `u` → `wu`.

- Si una sílaba comienza con `u`, cámbiala a `w`. Ejemplos: `ua` → `wa`, `uo` → `wo`, `uai` → `wai`, `uei` → `wei`, `uan` → `wan`, `ueng` → `weng`.

Practiquemos pronunciar las sílabas y luego usa `wan` en diferentes tonos como ejemplo.

PinyinCarácter
|--------|----------|
| `wān`  | `弯`|
| `wán`   | `完`|
| `wǎn`  | `晚`|
| `wàn`  | `万`|
