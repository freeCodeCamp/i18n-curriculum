# --description--

Nachdem Sie alle Aufgaben zum Thema „Schreiben Sie Ihren ersten Code mit C#“ abgeschlossen haben, sollten Sie die Trophäe dafür auf der Microsoft Learn-Plattform erhalten haben. Wenn Sie diese Trophäe nicht auf <a href="https://learn.microsoft.com/users/me/achievements#trophies-section" target="_blank" rel="noreferrer">Ihrer Erfolge-Seite</a> sehen, haben Sie nicht alle Aufgaben abgeschlossen und Ihre Trophäe kann nicht bestätigt werden.

Verknüpfen Sie Ihren Microsoft-Benutzernamen mit Ihrem freeCodeCamp-Konto und klicken Sie auf die Schaltfläche „Trophäe bestätigen“ unten, um die Aufgabe abzuschließen. Diese Trophäe ist **erforderlich**, um sich für die Zertifizierungsprüfung zu qualifizieren.
