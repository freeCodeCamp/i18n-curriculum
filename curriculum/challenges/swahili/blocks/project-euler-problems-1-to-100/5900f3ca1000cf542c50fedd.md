# --description--

Ni rahisi kuthibitisha kwamba hakuna mviringo sawa upande tatu wenye urefu wa pande zote ni nambari kamili na eneo ni nambari kamili. Hata hivyo, mviringo karibu sawa upande tatu 5-5-6 una eneo la vitengo 12 vya mraba.

Tutaelezea <dfn>mviringo karibu sawa upande tatu</dfn> kuwa ni mviringo ambao pande mbili ni sawa na upande wa tatu unatofautiana kwa kiwango kisizozidi kitengo kimoja.

Tafuta jumla ya mizunguko ya mviringo vyote karibu sawa upande tatu vyenye urefu wa pande ni nambari kamili na eneo ni nambari kamili na mizunguko yao haizidi `limit`.

```js
function almostEquilateralTriangles(limit) {
  // Based on https://blog.dreamshire.com/project-euler-94-solution/
  let perimetersSum = 0;

  let sidesAB = 1;
  let sideC = 1;
  let perimeter = 0;
  let perimeterOffset = 1;

  while (perimeter <= limit) {
    [sidesAB, sideC] = [4 * sidesAB - sideC + 2 * perimeterOffset, sidesAB];
    perimeterOffset = -perimeterOffset;

    perimetersSum += perimeter;
    perimeter = 3 * sidesAB - perimeterOffset;
  }

  return perimetersSum;
}
```
