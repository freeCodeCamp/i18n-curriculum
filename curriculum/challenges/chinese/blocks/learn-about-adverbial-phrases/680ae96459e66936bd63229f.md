# --explanation--

`What about...` 用于询问信息或引起与刚才讨论内容相关的新话题。它通常用于转移注意力或查看某事的不同方面。例如：

- `We finished the presentation. What about the handouts?` – 这意味着说话者在询问讲义是否准备好了或者是否已经考虑过。

- `That plan sounds good. What about the timeline?`——这意味着说话者同意这个想法，但想询问需要多长时间。
