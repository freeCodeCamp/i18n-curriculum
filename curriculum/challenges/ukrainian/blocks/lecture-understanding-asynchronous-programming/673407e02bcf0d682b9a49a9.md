# --description--

Движок JavaScript має здатність читати, розуміти та виконувати ваш код. Він працює як конвертер, який бере ваш код, перетворює його на інструкції, зрозумілі комп’ютеру, і виконує їх відповідно.

Один із найвідоміших движків JavaScript — V8, розроблений Google, який використовується в Chrome і Node.js. Движок JavaScript працює у кілька кроків. Спочатку він аналізує ваш код, читаючи його рядок за рядком, щоб переконатися, що в коді JavaScript немає помилок. Потім він перетворює цей код у байткод — простішу, проміжну версію вашого коду, яку комп’ютеру легше зрозуміти та виконати. Нарешті, він запускає цей байткод, щоб виконати інструкції вашої програми. Ось приклад коду JavaScript:

```js
const greeting = "Hello, World!";
console.log(greeting);
```

Коли ви запускаєте цей код, движок JavaScript спочатку аналізує його, щоб перевірити наявність синтаксичних помилок. Аналіз означає, що движок читає код і розбиває його на структуру, яку може зрозуміти, перевіряючи помилки по ходу.

Потім він компілює код у проміжний формат (зазвичай байткод або машинний код, залежно від движка). Компіляція — це процес перетворення коду, зрозумілого людині, у більш ефективний формат, який комп’ютер може виконувати швидше.

Нарешті, движок виконує код, виводячи `Hello, World!` у консоль.

Тепер поговоримо про середовище виконання JavaScript. Середовище виконання JavaScript — це середовище, у якому виконується ваш код JavaScript. Воно включає движок JavaScript (наприклад, V8 у Chrome або SpiderMonkey у Firefox), який обробляє та виконує код, а також додаткові функції, які надає середовище (наприклад, веббраузер або Node.js, про які ви дізнаєтеся більше в майбутніх уроках).

У той час як основна мова JavaScript опрацьовує такі речі, як змінні, цикли та функції, середовище виконання надає додаткові інструменти, які дозволяють JavaScript взаємодіяти з речами поза межами самої мови, наприклад, з DOM (для вебсторінок) або Fetch API (для мережевих запитів).

Коротко кажучи, середовище виконання дозволяє JavaScript робити більше, ніж просто базові програмні завдання — наприклад, взаємодіяти з вебсторінками або обробляти дії, що залежать від часу — надаючи ці додаткові функції поза межами самої мови.

Хоча вам не потрібно знати всі деталі про движки та середовища виконання, щоб писати JavaScript, базове розуміння допоможе писати більш ефективний код і краще налагоджувати проблеми.
