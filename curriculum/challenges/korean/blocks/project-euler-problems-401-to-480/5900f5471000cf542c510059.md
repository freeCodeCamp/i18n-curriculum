# --description--

양의 정수 $n$과 숫자 $d$에 대해, $F(n, d)$를 $n$의 약수 중 마지막 자리 숫자가 $d$인 약수의 개수로 정의합니다.

예를 들어, $F(84, 4) = 3$입니다. 84의 약수들(1, 2, 3, 4, 6, 7, 12, 14, 21, 28, 42, 84) 중 세 개(4, 14, 84)가 마지막 자리 숫자 4를 가집니다.

또한 $F(12!, 12) = 11$이고 $F(50!, 123) = 17\,888$임을 확인할 수 있습니다.

$F({10}^6!, 65\,432) \text{ modulo } ({10}^{16} + 61)$를 구하세요.
