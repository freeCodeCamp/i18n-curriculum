# --interactive--

Der Universalselektor (`*`) ist eine spezielle Art von CSS-Selektor, der auf jedes Element im Dokument zutrifft.

Er wird häufig verwendet, um einen Stil auf alle Elemente der Seite anzuwenden, was nützlich sein kann, um Stile in verschiedenen Browsern zurückzusetzen oder zu normalisieren.

Der Universalselektor kann verwendet werden, um alle Elemente innerhalb eines bestimmten Kontexts oder global im gesamten Dokument auszuwählen.

Hier ist ein Beispiel für die Verwendung des Universalselektors, um den `margin` und `padding` für das gesamte HTML-Dokument festzulegen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1>Heading element</h1>
<p>example paragraph element</p>
```

```css
* {
  margin: 0;
  padding: 0;
}
```

:::

In diesem Codebeispiel setzt der `*`-Selektor den Außenabstand und Innenabstand aller Elemente auf null zurück, was eine gängige Technik bei CSS-Resets ist.

Der Universalselektor hat den niedrigsten Spezifitätswert aller Selektoren. Er trägt 0 zu allen Teilen des Spezifitätswerts `(0, 0, 0, 0)` bei.

Das bedeutet, dass jeder andere Selektor, einschließlich Typselektoren, Klassenselektoren, ID-Selektoren und inline-Stile, die vom Universalselektor gesetzten Stile überschreibt.

Schauen wir uns das folgende HTML- und CSS-Beispiel an:

:::interactive_editor

```html
<head>
  <style>
    * {
      color: blue;
    }
    p {
      color: red;
    }
    .highlight {
      color: green;
    }
    #unique {
      color: purple;
    }
  </style>
</head>
<body>
  <p id="unique" class="highlight">This text has multiple styles applied.</p>
</body>
```

:::

Der universelle `*`-Selektor setzt die Textfarbe für alle Elemente auf `blue`.

Der `p`-Typselektor überschreibt dies jedoch, indem er die Textfarbe speziell für `red`-Elemente auf `p` ändert.

Hat ein Element die `highlight`-Klasse, hat der Klassenselektor Vorrang und ändert die Textfarbe auf `green`.

Schließlich überschreibt der `#unique`-ID-Selektor mit der höchsten Spezifität alle anderen und setzt die Textfarbe auf `purple`.
