# --explanation--

`To draft` bedeutet, eine erste Version von etwas zu erstellen, wie zum Beispiel ein Dokument, einen Plan oder ein Design. Zum Beispiel:

`She drafted an email to send to the client before finalizing it.` – Das bedeutet, sie hat eine frühe Version der E-Mail geschrieben.  

Ein `schedule` bezeichnet einen Plan, der darlegt, wann Aufgaben oder Ereignisse stattfinden. Zum Beispiel:

`The project schedule shows all the deadlines for the team.` – Das bedeutet, der Zeitplan listet auf, wann verschiedene Aufgaben erledigt werden müssen.
