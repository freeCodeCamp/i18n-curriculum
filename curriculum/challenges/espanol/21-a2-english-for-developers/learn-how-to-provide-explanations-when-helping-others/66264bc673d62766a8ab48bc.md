# --description--

La palabra `which` se usa para pedir información específica entre un conjunto de opciones conocidas. Por ejemplo, cuando alguien pregunta, `Which book did you read?` Quieren saber el libro específico de entre varias opciones posibles.

En un contexto técnico, preguntar `Which software are you using?` es pedir el nombre específico del software entre opciones conocidas.
