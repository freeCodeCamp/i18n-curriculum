# --description--

`Occasionally` 意味着某事偶尔发生。它不是一直发生，也不是从不发生，而是在两者之间。例如：

`I occasionally get a bug in my code.` - 有时你的代码中会有错误，但不是很频繁。

看看这些副词，以及它们通常表现的频率。

| Adverb       | Frequency | Adverb    | Frequency |
| ------------ | --------- | --------- | --------- |
| Never        | 0%        | Often     | 70%       |
| Rarely       | 5%        | Generally | 80%       |
| Seldom       | 15%       | Usually   | 90%       |
| Occasionally | 30%       | Always    | 100%      |
| Sometimes    | 50%       |           |           |
