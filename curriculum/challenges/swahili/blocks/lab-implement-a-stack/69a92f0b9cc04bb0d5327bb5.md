# --description--

Katika maabara hii, uta-tekeleza muundo wa data wa stack kwa kutumia vitendakazi. Stack ni muundo wa data wa Last-In-First-Out (LIFO) ambapo vipengele huongezwa na kuondolewa kutoka juu.

**Lengo:** Timiza hadithi za watumizi zilizo hapa chini na upite vipimo vyote ili kumaliza maabara.

**Hadithi za watumizi:**

1. Unapaswa kuwa na kitendakazi cha `initStack` kinachorudisha kitu chenye sifa ya `collection` iliyowekwa kuwa safu ya data tupu.
1. Unapaswa kuwa na kitendakazi cha `push` kinachoongeza kipengele kwenye juu ya stack.
1. Unapaswa kuwa na kitendakazi cha `pop` kinachotoa na kurudisha kipengele cha juu cha stack, au `undefined` ikiwa hakipo.
1. Unapaswa kuwa na kitendakazi cha `peek` kinachorudisha kipengele cha juu cha stack bila kuondoa, au `undefined` ikiwa hakipo.
1. Unapaswa kuwa na kitendakazi cha `isEmpty` kinachorudisha `true` ikiwa stack haina vipengele, na `false` vinginevyo.
1. Unapaswa kuwa na kitendakazi cha `clear` kinachoondoa vipengele vyote kutoka stack.

**Kumbuka:** Vipimo vingi vinategemea vitendakazi vya `initStack` na `push`. Tekeleza kwanza, kwani vipimo vya `pop`, `peek`, `isEmpty`, na `clear` vinahitaji kuongeza vipengele kwenye stack.

# --solutions--

```js
function initStack() {
  return {
    collection: []
  };
}

function print(stack) {
  console.log(stack.collection);
}

function push(stack, element) {
  stack.collection.push(element);
}

function pop(stack) {
  return stack.collection.pop();
}

function peek(stack) {
  return stack.collection[stack.collection.length - 1];
}

function isEmpty(stack) {
  return stack.collection.length === 0;
}

function clear(stack) {
  stack.collection = [];
}
```
