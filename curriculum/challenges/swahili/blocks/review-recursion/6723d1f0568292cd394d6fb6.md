# --description--

- Recursion ni dhana ya programu inayokuwezesha kuita kitendakazi mara kwa mara hadi kufikia kesi ya msingi.

Hapa kuna mfano wa kitendakazi cha recursion kinachohesabu faktoria ya nambari:

```js
function findFactorial(n) {
  if (n === 0) {
    return 1;
  }
  return n * findFactorial(n - 1);
}
```

Katika mfano huu wa juu, kitendakazi cha `findFactorial` kinaitwa kwa recursion hadi `n` ifikie `0`. Wakati `n` ni `0`, kesi ya msingi inafikiwa na kitendakazi kinarejesha `1`. Kisha kitendakazi kinarejesha bidhaa ya `n` na matokeo ya wito la recursion kwa `findFactorial(n - 1)`.

- Recursion inakuwezesha kushughulikia kitu chenye kina kisichojulikana, kama vitu/vitu vilivyopangwa ndani kwa kina, au mpangilio wa faili. 
- Stack ya wito hutumika kufuatilia miito ya kitendakazi katika recursion. Kila wakati kitendakazi kinapoitwa, huongezwa kwenye stack ya wito. Wakati kesi ya msingi inafikiwa, miito ya kitendakazi huondolewa kwenye stack.
- Unapaswa kufafanua kwa makini kesi ya msingi kwa sababu kuitwa bila kikomo kunaweza kusababisha msimbo wako kuanguka. Hii ni kwa sababu recursion inaendelea kuongeza miito ya kitendakazi zaidi na zaidi hadi mfumo utakapoisha kumbukumbu.
- Recursion hutumika kutatua matatizo ya kihesabu kama faktoria na Fibonacci, kupita kwenye miti na grafu, kuzalisha mchanganyiko na mchanganyiko wa vitu na mengine mengi.
