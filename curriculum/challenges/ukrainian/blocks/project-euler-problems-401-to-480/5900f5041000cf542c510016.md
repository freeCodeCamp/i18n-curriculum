# --description--

Якщо обчислити $a^2\bmod 6$ для $0 ≤ a ≤ 5$, отримаємо: 0, 1, 4, 3, 4, 1.

Найбільше значення a, для якого $a^2 ≡ a\bmod 6$, дорівнює $4$.

Позначимо $M(n)$ як найбільше значення $a < n$, для якого $a^2 ≡ a (\text{mod } n)$. Тож $M(6) = 4$.

Знайдіть $\sum M(n)$ для $1 ≤ n ≤ {10}^7$.
