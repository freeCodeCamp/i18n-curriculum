# --interactive--

类选择器是 CSS 的一个关键部分，允许开发者定位具有相同 class 属性的多个元素并应用一致的样式。

这使它们在整个网站上应用样式时非常多功能且高效。

类选择器由一个点号（`.`）后跟类名定义。它们可以应用于超文本标记语言文档中的任何元素。

这是一个使用类选择器的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="highlight">Example paragraph</p>
```

```css
.highlight {
  color: green;
}
```

:::

在此示例中，任何具有类 `highlight` 的元素的文本颜色将被设置为 `green`。

类选择器的优先级高于类型选择器，但低于 ID 选择器和内联样式。

类选择器的特异性值为 `(0, 0, 1, 0)`。这意味着类选择器可以覆写类型选择器，但它们可以被 ID 选择器和内联样式覆写。

类选择器可以与其他选择器组合，以创建更具体的规则。

这是一个将段落类型选择器与类选择器结合的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="bold-text">Example paragraph</p>
<p class="bold-text">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p.bold-text {
  font-weight: bold;
}
```

:::

此规则仅适用于同时具有 `bold-text` 类的 `p` 元素，使其文本加粗。
