# --description--

`Whole numbers` sono tipi di dati numerici di base nella programmazione e nella matematica. Sono numeri senza parti frazionarie o decimali, inclusi tutti i numeri interi positivi (come 1, 2, 3), numeri negativi (come -4, -9) e lo zero (0).

Cosa sono `whole numbers` nel contesto della programmazione?
