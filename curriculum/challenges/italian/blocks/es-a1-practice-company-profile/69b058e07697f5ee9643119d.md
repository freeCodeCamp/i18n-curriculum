# --explanation--

Questa attività rafforza le strutture della terza persona singolare usate per descrivere un'azienda.

`Se llama` si usa per dare nomi. Per esempio:

- `La empresa se llama Andes Commerce`. - L'azienda si chiama Andes Commerce.
- `El proyecto se llama Aurora`. - Il progetto si chiama Aurora.

`Es` si usa per descrivere cos'è qualcosa. Per esempio:

- `Es una empresa internacional`. - È un'azienda internazionale.
- `Es un curso avanzado`. - È un corso avanzato.

`Está` si usa per la posizione. Per esempio:

- `La oficina está en Lima`. - L'ufficio è a Lima.
- `El edificio está en el centro`. - L'edificio è nel centro.

`Tiene` esprime possesso o quantità. Per esempio:

- `La empresa tiene veinte empleados`. - L'azienda ha venti dipendenti.
- `El equipo tiene cinco miembros`. - Il team ha cinque membri.

`Setenta y tres` - settantatré

`Sitio web` - sito web
