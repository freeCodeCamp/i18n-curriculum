# --explanation--

Uma `data breach` acontece quando informações sensíveis são acessadas por pessoas não autorizadas. Por exemplo:

- `A data breach can happen if hackers get access to private information.` - Se cibercriminosos roubarem informações, isso é chamado de violação de dados.

- `Companies must act quickly if there is a data breach.` - Quando ocorre uma violação de dados, as empresas precisam responder rapidamente para corrigir o problema.

Jessica diz que criptografar dados e proteger as chaves de criptografia é essencial para prevenir esse risco.
