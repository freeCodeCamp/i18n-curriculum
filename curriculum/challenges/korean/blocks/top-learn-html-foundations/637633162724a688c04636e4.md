# --description--

HTML과 CSS는 주로 정보를 보여주는 데 집중합니다. 이들은 논리를 프로그래밍하는 데 사용되지 않습니다. 다음 섹션에서 배우게 될 JavaScript는 웹페이지가 동작하도록 만드는 데 사용됩니다. HTML과 CSS만으로도 할 수 있는 일이 꽤 많으며, 두 가지 모두 반드시 필요합니다. 커리큘럼 전반에 걸쳐, 다음 수업들은 JavaScript 내용을 배우기 전에 성공하는 데 필요한 도구를 제공하는 데 포커스를 둡니다.
