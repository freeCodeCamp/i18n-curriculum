# --description--

在上一課中，你已經被介紹了列表資料型別，並學會了如何存取列表中的元素以及列表切片。在本課中，你將繼續學習列表以及一些相關的常用方法，例如 `append()`、`pop()` 和 `sort()`。

我們將首先查看的行為是 `append()` 行為。這用於將一個項目添加到列表的末尾。以下是使用 `append()` 行為將數字 `6` 添加到數字列表的範例：

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

如果你想將一個列表添加到另一個列表的末尾，你也可以像這樣使用 `append()` 方法：

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

注意整個 `even_numbers` 列表是巢狀在 `numbers` 列表內。

但是如果你想將 `even_numbers` 列表中所有的個別數字添加到 `numbers` 列表的末尾，則你可以使用 `extend()` 方法。

`extend()` 方法類似於 `append()` 方法，但使用 `extend()` 你可以將多個元素從一個列表添加到另一個列表。以下是將數字 `6`、`8` 和 `10` 從一個列表添加到 `numbers` 列表末尾的範例：

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

如你所見，巢狀列表已消失，現在只剩下一個數字列表。

要在列表中的特定索引插入一個元素，你可以使用 `insert()` 方法。此方法接受兩個引數：你希望插入新條款的索引以及你想插入的條款。

以下是使用 `insert()` 方法的範例：

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

以下程式碼會在 `numbers` 列表的索引 `2` 插入數字 `2.5`。

如果你想從列表中移除一個元素，你可以使用 `remove()` 方法。`remove()` 方法將要移除的元素值作為引數：

```py
numbers = [10, 20, 30, 40, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50]
```

重要的是要注意，這個方法只會移除第一個出現的項目，而不是全部。

```py
numbers = [10, 20, 30, 40, 50, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50, 50]
```

要移除列表中指定索引的元素，你可以像這樣使用 `pop()` 方法：

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

如果你沒有為 `pop` 方法指定元素，則會移除最後一個元素。

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

如果你需要清空列表，則可以像這樣使用 `clear()` 方法：

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

接下來我們將查看的行為是 `sort()` 方法。此方法用於就地排序元素。以下是一個就地排序隨機數字列表的範例：

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

與 `sort()` 方法相反，`sorted()` 函式適用於任何可迭代物件，並傳回一個新的排序後列表，而不是修改原始列表。 例如：

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(numbers) # [19, 2, 35, 1, 67, 41]
print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
```

提醒你，iterable 是一種特殊型別的物件，你可以對它進行迴圈，讓你一次存取一個項目。你將在未來的課程中學習更多關於 Python 中迴圈的運作方式。

`sort()` 方法和 `sorted()` 函式都接受選用的 `key` 和 `reverse` 參數。當你在未來的課程中學習元組時，會進一步了解這些選用參數。你也會在未來的課程中學習更多像 `sorted()` 這樣的內建函式。

接下來我們將查看的行為是 `reverse()` 方法。此方法會就地反轉元素的列表，如下所示：

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

我們將要看的最後一個方法是 `index` 方法。這用於尋找元素在列表中第一次出現的位置索引。以下是使用 `index` 方法在 `programming_languages` 列表中尋找語言 `'Java'` 的範例：

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

如果找不到該元素，則 Python 會丟出 `ValueError`：

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: 'JavaScript' is not in list
"""
```

Python 列表還有一些其他的方法，但這個初始的方法列表是個不錯的起點。
