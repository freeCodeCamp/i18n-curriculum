# --explanation--

`Have to` é usado para expressar que algo é necessário ou obrigatório.Indica que não há outra opção a não ser fazer algo.Por exemplo:

- `You have to save your changes before closing the program.` - Isso significa que é necessário salvar as alterações antes de fechar o programa.

Quando Tom diz, `I'll have to check again`, ele está afirmando que é necessário para ele verificar as configurações mais uma vez, sugerindo que ele não tem certeza sobre isso.
