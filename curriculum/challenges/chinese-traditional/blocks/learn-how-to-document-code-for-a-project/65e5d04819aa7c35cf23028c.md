# --description--

字 `so` 是一個連接詞，用來表示因果關係或解釋某事的原因。在科技和程式設計中，`so` 常用來連接問題或情況與解法或結果。

# --questions--

## --text--

選擇在技術上下文中正確使用 `so` 來連接原因與結果的句子。

## --answers--

`The program crashed, so we must reboot the computer.`

---

`We wrote the code, so but it didn't compile.`

### --feedback--

這裡錯誤地使用了字 `so`。`But` 不應該用在 `so` 之後。應該是 `The code didn't compile, so we rewrote it`，或者是 `We wrote the code, but it didn't compile.`。

---

`The server is online, so therefore we can't connect.`

### --feedback--

同時使用 `so` 和 `therefore` 是多餘的。應該是 `The server is online, so we can connect` 或 `The server is online; therefore, we can connect.`。

---

`We need more data, so we can ignore the results.`

### --feedback--

這句話在邏輯上沒有連接因果關係。更好的 `so` 用法是，`We need more data, so we will perform more tests.`
