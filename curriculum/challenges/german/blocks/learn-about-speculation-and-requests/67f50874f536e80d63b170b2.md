# --explanation--

`Brute force attacks` sind Angriffe, bei denen Hacker mehrere Passwörter ausprobieren, bis sie das richtige finden. Zum Beispiel:

`The system detected a brute force attack on the login page.` – Der Angriff versuchte viele Passwortversuche, um einzubrechen.

`SQL injection attempts` treten auf, wenn Hacker versuchen, schädlichen SQL-Code einzuschleusen, um auf eine Datenbank zuzugreifen oder sie zu verändern. Zum Beispiel:

`The firewall stopped two SQL injection attempts.` – Jemand versuchte, die Datenbank mit schädlichem Code zu hacken.
