# --explanation--

표현식 `stuck on`는 누군가가 작업을 진행하지 못하거나 문제를 해결하지 못할 때 사용합니다. 갇히거나 진전을 이루지 못하는 상황을 나타냅니다. 예를 들어:

- `We were stuck on this code error for hours.` - 코드의 그 오류를 찾을 때까지 진행할 수 없었습니다.

- `She got stuck on the final question of the exam.` - 그녀는 마지막 문제에 오랜 시간을 보냈습니다.

대화에서 Lisa는 `stuck on`을 사용해 작업 중 버그를 해결하지 못해 진전을 이루지 못한 상황을 설명합니다.
