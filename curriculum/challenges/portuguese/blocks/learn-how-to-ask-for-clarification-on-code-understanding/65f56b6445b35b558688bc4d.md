# --description--

`Step by step` significa fazer algo uma etapa de cada vez.Por exemplo, `debugging step by step` significa resolver um problema em partes pequenas e gerenciáveis. 

`Input` em programação geralmente significa informação ou orientação fornecida para um projeto.

`Work together` significa colaborar e `pair up` é semelhante, significando duas pessoas trabalhando juntas de perto.
