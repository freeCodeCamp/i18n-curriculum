# --interactive--

У попередньому уроці ви дізналися, як працювати з об’єктом `Date` у JavaScript. Але існує кілька різних способів форматування дат у JavaScript.

У цьому уроці ми розглянемо, як працювати з методами `toISOString()`, `toString()` та `toLocaleDateString()` для форматування дат у JavaScript.

Перед тим як розглянути різні методи, давайте спочатку повторимо, як виглядає об’єкт `Date`:

:::interactive_editor

```js
const date = new Date();
console.log(date);
```

:::

Коли ви виводите об’єкт `date` у консоль, ви побачите поточну дату і час відповідно до налаштувань системи користувача. Ось приклад вихідних даних, які ви можете побачити:

```js
Sun Sep 29 2024 19:45:37 GMT-0700 (Pacific Daylight Time)
```

Якщо ви використаєте метод `toString()` для об’єкта `date`, ви побачите той самий результат, що й вище.

Щоб відформатувати дату у розширеному форматі ISO (ISO 8601), ви можете використати метод `toISOString()` так:

:::interactive_editor

```js
const date = new Date();
console.log(date.toISOString());
```

:::

ISO 8601 — це міжнародний стандарт для представлення дат і часу. Формат виглядає як `YYYY-MM-DDTHH:mm:ss.sssZ`.

Ось приклад вихідних даних, які ви можете побачити:

```js
2024-09-30T02:47:20.292Z
```

Ще один спосіб форматування дати — це використання методу `toLocaleDateString()`. Цей метод дозволяє форматувати дату залежно від локалі користувача. Ось базовий синтаксис використання методу `toLocaleDateString()`:

:::interactive_editor

```js
const date = new Date();
console.log(date.toLocaleDateString());
```

:::

Ось приклад вихідних даних, які ви можете побачити:

```js
9/29/2024
```

Метод `toLocaleDateString()` приймає два необов’язкові параметри: `locales` і `options`.

Параметр `locales` — це рядок, який представляє локаль для використання. Наприклад, ви можете передати `en-US` для англійської (США) або `fr-FR` для французької (Франція). Якщо ви не передасте параметр `locales`, буде використана локаль за замовчуванням. Ось приклад використання параметра `locales`:

:::interactive_editor

```js
const date = new Date();
console.log(date.toLocaleDateString("fr-FR"));
```

:::

Другий необов’язковий параметр — це параметр `options`. Цей параметр є об’єктом, який дозволяє вказати формат рядка дати. Ось приклад використання параметра `options`:

:::interactive_editor

```js
const date = new Date();
const options = {
  weekday: "long",
  year: "numeric",
  month: "long",
  day: "numeric",
};
console.log(date.toLocaleDateString("en-GB", options));
```

:::

У наведеному вище прикладі ми вказали опції для англійської (Велика Британія) з повною назвою дня тижня, року, місяця та дня.

Ось приклад вихідних даних, які ви можете побачити:

```js
Sunday, September 29, 2024
```

У цьому уроці ми розглянули лише кілька способів форматування дат у JavaScript. Існує багато інших методів і бібліотек, які допоможуть вам форматувати дати у JavaScript. Але методи `toISOString()` та `toLocaleDateString()` — це гарна відправна точка для форматування дат у JavaScript.
