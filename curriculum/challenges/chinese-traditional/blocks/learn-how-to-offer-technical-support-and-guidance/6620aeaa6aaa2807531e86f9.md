# --description--

動詞 `encounter` 意思是遇到或碰到某事，特別是意料之外的事。在程式設計中，`encountering` 問題或錯誤表示在程式碼執行時碰到問題。

例如，`You might encounter an error if the user inputs the wrong type of data.` 這句話的意思是你可能會意外地遇到或必須處理錯誤。
