# --description--

Unajua kuwa JavaScript ina aina mbalimbali za data, ikiwa ni pamoja na mfuatano wa herufi, nambari na thamani za kweli au siyo kweli. Unaweza kutumia vigezo vya kulinganisha kulinganisha aina hizi za data na kufanya maamuzi kulingana na matokeo.

Vigezo vifuatavyo vya kulinganisha vinapatikana katika JavaScript:

| Kigezo | Maelezo                              | Mfano      |
|----------|------------------------------------------|--------------|
| ==       | Sawa (kulinganisha thamani)                 | x == y       |
| !=       | Sio sawa                                | x != y       |
| >        | Kubwa kuliko                             | x > y        |
| <        | Ndogo kuliko                                | x < y        |
| >=       | Kubwa kuliko au sawa na                 | x >= y       |
| <=       | Ndogo kuliko au sawa na                    | x <= y       |

Ili kutumia kulinganisha katika mgawo wa thamani wa variable, unaweza kutumia kigezo `==`. Kwa mfano:

```javascript
let x = 5;
let y = 5;

let result = x == y;

console.log(result); // true
```
