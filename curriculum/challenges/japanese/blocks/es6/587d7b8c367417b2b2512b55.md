# --description--

`import` を使うと、ファイルやモジュールのどの部分を読み込むか選べます。前のレッスンでは、例として `add` ファイルから `math_functions.js` をエクスポートしました。別のファイルで使うためにインポートする方法は次の通りです。

```js
import { add } from './math_functions.js';
```

ここで、`import` は `add` の中から `math_functions.js` を見つけて、その関数だけをインポートして使い、他は無視します。`./` は現在のファイルと同じフォルダーにある `math_functions.js` ファイルを探すように指示しています。相対ファイルパス（`./`）とファイル拡張子（`.js`）は、この方法で import を使う場合に必要です。

複数の項目をファイルからインポートしたい場合は、`import` 文の中に追加して次のように書けます。

```js
import { add, subtract } from './math_functions.js';
```

# --instructions--

前のレッスンでエクスポートした `import` と `uppercaseString` 関数を、現在のファイルで使えるようにする適切な `lowercaseString` 文を追加してください。これらの関数は、現在のファイルと同じディレクトリにある `string_functions.js` というファイルにあります。
