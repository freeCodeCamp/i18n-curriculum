# --description--

設 $C$ 為半徑為 $r$ 的圓，方程式為 $x^2 + y^2 = r^2$。我們選擇兩點 $P(a, b)$ 和 $Q(-a, c)$，使通過 $P$ 和 $Q$ 的直線與 $C$ 相切。

例如，四元組 $(r, a, b, c) = (2, 6, 2, -7)$ 滿足此屬性。

令 $F(R, X)$ 為具有此屬性的整數四元組 $(r, a, b, c)$ 的數量，且滿足 $0 < r ≤ R$ 及 $0 < a ≤ X$。

我們可以驗證 $F(1, 5) = 10$、$F(2, 10) = 52$ 和 $F(10, 100) = 3384$。

尋找 $F({10}^8, {10}^9) + F({10}^9, {10}^8)$。
