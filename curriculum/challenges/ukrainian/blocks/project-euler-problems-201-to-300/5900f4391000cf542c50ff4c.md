# --description--

Пітер має дев’ять чотиригранних (пірамідальних) кубиків, кожен із граннями, пронумерованими 1, 2, 3, 4.

Колін має шість шестигранних (кубічних) кубиків, кожен із граннями, пронумерованими 1, 2, 3, 4, 5, 6.

Пітер і Колін кидають свої кубики та порівнюють суми: виграє той, у кого сума більша. Результат вважається нічиєю, якщо суми рівні.

Яка ймовірність, що пірамідальний Пітер переможе кубічного Коліна? Наведіть відповідь, округлену до семи десяткових знаків у форматі 0.abcdefg

# --hints--
