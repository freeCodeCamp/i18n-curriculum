# --description--

Sei $y_0, y_1, y_2, \ldots$ eine Folge von zufälligen, vorzeichenlosen 32-Bit-ganzen Zahlen (d.h. $0 ≤ y_i &lt; 2^{32}$, jeder Wert ist gleich wahrscheinlich).

Für die Folge $x_i$ gilt die folgende Rekursion:

- $x_0 = 0$ und
- $x_i = x_{i - 1} \mathbf{|} y_{i - 1}$, für $i > 0$. ($\mathbf{|}$ ist der bitweise-OR-Operator)

Es ist ersichtlich, dass es schließlich einen Index $N$ geben wird, so dass $x_i = 2^{32} - 1$ (ein Bitmuster aus lauter Einsen) für alle $i ≥ N$ gilt.

Bestimmen Sie den Erwartungswert von $N$. Geben Sie Ihre Antwort auf 10 Nachkommastellen gerundet an.
