# --explanation--

Un ejemplo es que `the "Student" class inherits properties and methods from the "Person" class`.

- `Student` y `Person` son ambas `classes`. Una `class` define una estructura para crear objetos con datos y comportamientos específicos.

- `Student` hereda características de `Person`, lo que significa que no necesita redefinir rasgos compartidos — los reutiliza.

- `Properties` son las variables o datos dentro de la clase, como `name` o `age`, que describen el objeto.

- Los `Methods` son las funciones definidas en la clase, como `enroll()` o `introduce()`, que describen lo que el objeto puede hacer.
