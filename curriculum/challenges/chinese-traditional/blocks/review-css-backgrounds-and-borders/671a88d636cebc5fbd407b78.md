## 列表樣式

- **`line-height` 屬性**：此屬性用於在文字行之間創建間距。可接受的 `line-height` 值包括關鍵字 `normal`、數字、百分比以及像 `em` 單位這樣的長度單位。
- **`list-style-type` 屬性**：此屬性用於指定列表項目的標記。可接受的值可以包括 circle、disc 或 decimal。

以下是使用 `list-style-type` 來更改項目符號樣式的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
ul {
  list-style-type: square;
}
```

:::

- **`list-style-position` 屬性**：此屬性用於設定列表標記的位置。唯一可接受的兩個值是 inside 和 outside。

以下範例顯示 `inside` 和 `outside` 之間的差異：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="inside-list">
  <li>Item with inside position</li>
  <li>Item with inside position</li>
</ul>
<ul class="outside-list">
  <li>Item with outside position</li>
  <li>Item with outside position</li>
</ul>
```

```css
.inside-list {
  list-style-position: inside;
}

.outside-list {
  list-style-position: outside;
}
```

:::

- **`list-style-image` 屬性**：此屬性用於為列表項目標記使用影像。常見的使用情況是使用 `url` 函式，並將值設為有效的影像位置。

## 使用 `margin` 調整列表項目的間距

- 除了 `line-height`，在 CSS 中也可以使用邊距來增強列表項目的間距和可讀性。
- 邊距會在每個 `li` 元素外部產生空間，允許控制列表項目之間的間距。
- `margin-bottom` 用於在每個列表項目下方創建空間。 例如，`margin-bottom: 10px;` 將在每個列表項目下方創建 10 像素的間隙。

以下是使用 `margin-bottom` 來調整列表項目間距的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="list">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
.list li {
  margin-bottom: 20px;
}
```

:::

## 為聯結設定樣式

- **`pseudo-class`**：這是一個添加到選擇器的關鍵字，允許你根據特定狀態選取元素。常見的狀態包括 `:hover`、`:visited` 和 `:focus` 狀態。
- **`:link 偽類別`**：此偽類別用於設定使用者尚未造訪的聯結樣式。

以下是使用 `:link` 偽類的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:link {
  color: red;
}
```

:::

- **`:visited pseudo-class`**：此偽類用於為使用者已造訪的連結設定樣式。
- **`:hover pseudo-class`**：此偽類別用於為使用者作用中暫留其上的元素設定樣式。

以下是使用 `:hover` 偽類的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Hover over me!</a>
```

```css
a:hover {
  color: green;
  text-decoration: underline;
}
```

:::

- **`:focus pseudo-class`**：此偽類別用於當元素獲得焦點時為其設定樣式。範例包括 `input` 或 `select` 元素，當點擊或使用 Tab 鍵聚焦該元素時。

以下是使用 `:focus` 偽類的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:focus {
  outline: 2px solid orange;
}
```

:::

- **`:active pseudo-class`**：此偽類用於為被使用者作用的元素設定樣式。一個常見的例子是使用者點擊按鈕時。

以下是使用 `:active` 偽類的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Click me!</a>
```

```css
a:active {
  color: pink;
}
```

:::

## 處理背景和邊框

- **`background-size` 屬性**：此屬性用於設定元素的背景大小。一些常見的值包括 `cover`，使背景影像涵蓋整個元素，以及 `contain`，使背景影像適合元素範圍內。

以下是使用 `background-size: contain` 的範例：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **`background-repeat` 屬性**：此屬性用於決定背景影像應如何沿水平和垂直軸重複。`background-repeat` 的預設值為 `repeat`，表示影像會在水平和垂直方向重複。你也可以使用 `no-repeat` 屬性來指定不重複。

以下是使用 `background-repeat: no-repeat` 的範例：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **`background-position` 屬性**：此屬性用於指定背景影像的位置。它可以設定為特定長度、百分比，或關鍵字值如 `top`、`bottom`、`left`、`right` 和 `center`。

以下是使用 `background-position` 的範例：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

- **`background-attachment` 屬性**：此屬性用於指定背景影像是否應隨內容捲動或保持固定位置。主要值為 `scroll`（預設值），背景影像會隨內容捲動，以及 `fixed`，背景影像會保持在螢幕上的相同位置。
- **`background-image` 屬性**：此屬性用於設定元素的背景影像。你可以同時設定多個背景影像，並使用 `url`、`radial-gradient` 或 `linear-gradient` 函式（程式）作為值。

以下是使用 `background-image` 的範例：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **`background` 屬性**：這是用於在一個宣告中設定所有背景屬性的縮寫屬性。以下是一個設定背景圖片並設定為不重複的範例：`background: no-repeat url("example-url-goes-here");`

以下是使用 `background` 簡寫屬性的範例：

:::interactive_editor

```html
<style>
  body {
    background: center top no-repeat url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **背景和前景顏色的良好對比**：確保背景和前景顏色具有良好對比以使 `text` 可讀非常重要。網頁內容無障礙指引（WCAG）建議一般 `text` 的最低對比度為 4.5:1，大字 `text` 則為 3:1。

## 邊框

- **`border-top` 屬性**：此屬性用於設定元素上方邊框的樣式。`border-top: 3px solid blue;` 將在元素的上方邊框設定寬度為 3 像素的實心藍色邊框。
- **`border-right` 屬性**：此屬性用於設定元素右側邊框的樣式。`border-right: 2px solid red;` 將在元素的右側設定一個 2 像素寬的實心紅色邊框。
- **`border-bottom` 屬性**：此屬性用於設定元素底部邊框的樣式。`border-bottom: 1px dashed green;` 將在元素底部設定一個寬度為 1 像素的綠色虛線邊框。
- **`border-left` 屬性**：此屬性用於設定元素左側邊框的樣式。`border-left: 4px dotted orange;` 將元素左側設定為 4 像素寬的橘色點狀邊框。

以下是使用個別邊框屬性的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="bordered-box">Box with different borders</div>
```

```css
.bordered-box {
  border-top: 3px solid blue;
  border-right: 2px solid red;
  border-bottom: 1px dashed green;
  border-left: 4px dotted orange;
  padding: 20px;
}
```

:::

- **`border` 屬性**：這是設定元素邊框寬度、樣式和顏色的縮寫屬性。`border: 1px solid black;` 設定一個寬度為 1 像素的實心黑色邊框。

以下是使用 `border` 簡寫屬性的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

- **`border-radius` 屬性**：此屬性用於為元素的邊框創建圓角。

以下是使用 `border-radius` 的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

- **`border-style` 屬性**：此屬性用於設定元素邊框的樣式。部分可接受的值包括 `solid`、`dashed`、`dotted` 和 `double`。

以下是使用不同 `border-style` 值的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="solid-border">Solid border</div>
<div class="dashed-border">Dashed border</div>
<div class="dotted-border">Dotted border</div>
```

```css
.solid-border {
  border: 3px solid blue;
  margin-bottom: 10px;
  padding: 10px;
}

.dashed-border {
  border: 3px dashed red;
  margin-bottom: 10px;
  padding: 10px;
}

.dotted-border {
  border: 3px dotted green;
  padding: 10px;
}
```

:::

## 漸層

- **`linear-gradient()` 函式（程式）**：此 CSS 函式用於沿直線創建多個顏色之間的過渡。

以下是使用 `linear-gradient()` 的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

- **`radial-gradient()` 函式（程式）**：此 CSS 函式（程式）創建一個從特定點輻射的影像，類似圓形或簡略號，並在多種顏色之間逐漸過渡。

以下是使用 `radial-gradient()` 的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::
