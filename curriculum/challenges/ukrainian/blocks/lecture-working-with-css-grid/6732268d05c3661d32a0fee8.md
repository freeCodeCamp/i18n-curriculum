# --interactive--

У попередніх уроках ми працювали з властивістю `grid-template-columns` і задавали їй значення у кілька дробових одиниць.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

Хоча наведений код є цілком коректним, існує простіший спосіб повторити розділ або всі ваші списки треків.

Функція `repeat()` використовується для повторення розділу або всіх треків для стовпців чи рядків. Ця функція приймає кількість повторів і треки, які ви хочете повторити.

Ось оновлена версія попереднього прикладу з використанням функції repeat:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

Стилі у браузері не зміняться, але це більш лаконічний спосіб записати повторювані значення для стовпців.

Функція `repeat()` приймає будь-який дійсний шаблон, який ви можете використовувати для рядків або стовпців.

Ось приклад із використанням функції `repeat()`, щоб встановити перший і третій стовпець шириною 20 пікселів, а другий і четвертий — у одну дробову одиницю.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(2, 20px 1fr);
  column-gap: 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

Іноді ви можете записати кожне окреме значення замість використання функції `repeat()`. Але бувають випадки, коли ця функція дуже зручна, особливо коли потрібно повторити певний шаблон для списку треків.
