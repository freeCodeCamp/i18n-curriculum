# --description--

집합 {1,2,..., $n$}이 주어졌을 때, $f(n, k)$를 원소들의 합이 홀수인 $k$-원소 부분집합의 개수로 정의합니다. 예를 들어, $f(5,3) = 4$인데, 집합 {1,2,3,4,5}에는 원소 합이 홀수인 3-원소 부분집합이 네 개 있습니다. 즉, {1,2,4}, {1,3,5}, {2,3,4}, {2,4,5}입니다.

세 값 $n$, $k$, $f(n, k)$가 모두 홀수일 때, 이를 홀수 삼중항 $[n, k, f(n, k)]$이라고 합니다.

$n ≤ 10$인 경우 홀수 삼중항은 정확히 다섯 개가 있습니다. 즉, $[1, 1, f(1, 1) = 1]$, $[5, 1, f(5, 1) = 3]$, $[5, 5, f(5, 5) = 1]$, $[9, 1, f(9, 1) = 5]$, $[9, 9, f(9, 9) = 1]$입니다.

$n ≤ {10}^{12}$인 경우 홀수 삼중항은 몇 개일까요?
