# --interactive--

邊距和行高對於間隔列表項目以提升可讀性和視覺吸引力至關重要。

首先，讓我們開始使用邊距來調整列表項目的間距！

可以透過對 `li` 元素套用邊距屬性來為列表項目之間創建空間。此方法允許你控制每個列表項目外部的間距，有效地增加或減少它們之間的間隔。

讓我們來看看一個包含三個列表項目的無序列表範例。

:::interactive_editor

```html
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

預設情況下，HTML 不會在列表項目之間套用那麼多間距。

若要為每個列表項目底部添加一些間距，你可以像這樣使用 `margin-bottom` 屬性：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  margin-bottom: 40px;
}
```

:::

在此範例中，`40px` 的邊距將套用到無序列表中每個列表項目的底部。

另一種調整列表項目間距的方法是使用 `line-height` 屬性。

`line-height` 屬性調整單一列表項目內文字行與行之間的垂直間距。

雖然它主要影響每個項目中文字行間的間距，但如果項目僅包含單行文字，它也可能間接影響項目之間的整體間距。

如果列表項目有多行文字，`line-height` 會影響這些行之間的間距，但它不會直接調整不同列表項目之間的間距。

若要控制各個列表項目之間的間距，則應使用 `margin` 或 `padding` 屬性。

使用先前相同的無序列表，這裡是一個將 `line-height` 應用到列表項目的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  line-height: 2; 
}
```

:::

在此範例中，`line-height: 2;` 將行高設定為字型大小的兩倍，為每個列表項目創造更多垂直空間。

## --text--

你會使用哪個 CSS 屬性，透過調整每個 `li` 元素外部的空間來為列表項目創建間距？

## --answers--

`padding`

### --feedback--

此屬性會影響元素 `border` 內部的空間。

---

`line-height`

### --feedback--

此屬性會影響文字行間距，而非元素外部的間距。

---

`margin`

---

`border`

### --feedback--

此屬性會影響元素的 `border`，而非其外部空間。

## --text--

給定 CSS 規則 `li { margin-bottom: 15px; }`，這對列表項目有什麼效果？

## --answers--

為每個列表項目內部添加 `15px` 的間距。

### --feedback--

此規則影響列表項目的外部空間，而非內部。

---

在每個列表項目的內容和 `border` 之間添加 `15px` 的間距。

### --feedback--

此規則會影響每個列表項目的 `border` 外側空間。

---

在每個列表項目下方添加 `15px` 的間距。

---

在每個列表項目上方添加 `15px` 的間距。

### --feedback--

此規則會在每個列表項目下方添加間距，而非上方。
