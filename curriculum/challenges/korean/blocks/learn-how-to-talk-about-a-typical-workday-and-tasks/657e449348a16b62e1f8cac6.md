# --description--

`Security sweeps`는 일반적으로 안전과 보안을 위해 특정 영역이나 건물에서 수행하는 상세한 점검이나 수색입니다. 잠재적인 위험, 무단 접근, 위험한 물건이 있는지 확인하는 작업을 포함합니다. 예를 들어:

`Security officers perform security sweeps to find any hidden dangers.` - 이는 보안 담당자가 쉽게 보이지 않는 위험한 물건이 있는지 영역을 꼼꼼히 점검하는 것을 의미합니다.
