# --description--

`Every` 和 `any` 经常在句子中使用，但含义不同。

`Every` 用于表示一组或类别中的所有项。例如，`Every student must complete the assignment` 意味着所有学生。

另一方面，`any` 用于疑问句或否定句中，表示一个、一些或全部。例如，`Can any student answer the question?` 是在询问是否至少有一个学生能回答这个问题。
