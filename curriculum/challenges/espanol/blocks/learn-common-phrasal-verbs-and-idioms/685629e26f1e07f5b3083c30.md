# --explanation--

Las `Real-time alerts` son mensajes o advertencias que aparecen inmediatamente cuando ocurre algo. Por ejemplo:

`The app sends real-time alerts if someone logs in from a new location.` – Esto significa que cada vez que alguien inicia sesión desde un lugar diferente, el usuario recibe un mensaje de advertencia inmediato.

`Graphics` son imágenes o elementos visuales usados para comunicar información. Por ejemplo:

`The dashboard uses graphics to show project progress.` – Esto significa que se usan herramientas visuales como gráficos o íconos para facilitar la comprensión de los datos.
