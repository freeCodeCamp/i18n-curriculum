# --description--

단어 `super`는 형용사 앞에서 그 의미를 강화하기 위해 비공식적으로 자주 사용되며, `very` 또는 `extremely`와 비슷하게 뒤따르는 형용사의 의미를 더 강하게 만듭니다.

예를 들어, `this book is super interesting`라고 말하면 그 책이 단순히 흥미로운 것보다 훨씬 더 흥미롭다는 뜻입니다.

`Helpful`는 귀중한 도움이나 지원을 제공하는 사람이나 사물을 묘사합니다. 이 둘을 합치면 `super helpful`는 뛰어난 도움을 제공하거나 매우 유용하다는 의미가 됩니다.
