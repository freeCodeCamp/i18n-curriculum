# --interactive--

Unapofanya kazi na mfuatano wa herufi katika JavaScript, kuna visa vingi ambapo unaweza kuhitaji kuangalia kama mfuatano wa herufi unajumuisha mfuatano mdogo wa herufi, ambao ni sehemu ndogo ya mfuatano huo.

Kwa mfano, unaweza kutaka kuangalia kama ingizo la mtumizi linajumuisha neno au herufi fulani kabla ya kufanya kitendo fulani. Njia moja ya kufanikisha hili ni kwa kutumia njia ya `includes()`.

Njia ya `includes()` hutumika kuangalia kama mfuatano wa herufi unajumuisha mfuatano mdogo wa herufi. Ikiwa mfuatano mdogo unapatikana ndani ya mfuatano, njia hiyo hurudisha `true` vinginevyo, hurudisha `false`.

Hapa kuna sintaksia ya msingi:

```js
string.includes(searchValue);
```

Kwa sintaksia, `searchValue` ni mfuatano mdogo wa herufi unaotaka kutafuta ndani ya mfuatano. Na hapa kuna mfano:

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("awesome");

console.log(result);  // true
```

:::

Katika mfano huu, neno `awesome` linapatikana ndani ya mfuatano `JavaScript is awesome!`, hivyo njia ya `includes()` hurudisha `true`.

Ni muhimu kutambua kuwa njia ya `includes()` ina makini na herufi kubwa na ndogo. Hii ina maana kuwa inahitaji mlingano kamili wa herufi, ikijumuisha aina ya herufi.

Kwa mfano:

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("Awesome");

console.log(result);  // false
```

:::

Kwa kuwa `Awesome` (ikiwa na herufi kubwa `A`) haifananiki na `awesome` (ikiwa na herufi ndogo `a`), matokeo ni `false`.

Unaweza pia kutumia njia ya `includes()` kuangalia mfuatano mdogo wa herufi unaoanza katika kielezo fulani cha mfuatano kwa kutoa kigezo cha pili:

:::interactive_editor

```js
let text = "Hello, JavaScript world!";
let result = text.includes("JavaScript", 7);

console.log(result);  // true
```

:::

Hapa, utafutaji wa mfuatano mdogo wa herufi `JavaScript` unaanza kutoka nafasi ya 7 ndani ya mfuatano, kuhakikisha unaruka herufi zozote kabla ya nafasi hii.

Njia ya `includes()` hurudisha tu matokeo ya `true` au `false`. Haiwezi kutoa taarifa kuhusu mahali mfuatano mdogo unapatikana ndani ya mfuatano au mara ngapi unajitokeza. Ikiwa unahitaji kiwango hicho cha undani, njia nyingine kama njia ya `indexOf()` inaweza kuwa bora zaidi.
