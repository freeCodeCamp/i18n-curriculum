# --description--

ABCD を凸四角形とし、対角線を AC と BD とします。各頂点で対角線はそれぞれの隣接する2辺と角度をなしており、合計で8つの角ができます。

<img alt="convex quadrilateral ABCD, with diagonals AC and BD" src="https://cdn.freecodecamp.org/curriculum/project-euler/integer-angled-quadrilaterals.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

例えば、頂点 A では、2つの角は CAD と CAB です。

このように、8つの角すべてが度数法で整数値となる四角形を「整数角の四角形」と呼びます。整数角の四角形の例としては、8つの角すべてが45°の正方形があります。別の例として、DAC = 20°、BAC = 60°、ABD = 50°、CBD = 30°、BCA = 40°、DCA = 30°、CDB = 80°、ADB = 50° の場合があります。

相似でない整数角の四角形は全部でいくつあるでしょうか？

**注意:** 計算において、角度が整数値から ${10}^{-9}$ の範囲内であれば整数とみなして構いません。
