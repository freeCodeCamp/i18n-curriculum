# --description--

Ahora que tienes un marcador centrado con color, es hora de añadir los otros marcadores.

En el elemento `.container`, añade dos elementos más `div` y dales a cada uno una clase de `marker`.

# --hints--

Tu primer nuevo elemento `div` debe tener una etiqueta de apertura.

```js
assert.exists([...code.matchAll(/<div.*?>/gi)][2]);
```

Tu primer nuevo elemento `div` debe tener una etiqueta de cierre.

```js
assert.exists([...code.matchAll(/<\/div\s*>/gi)][2]);
```

Tu segundo nuevo elemento `div` debe tener una etiqueta de apertura.

```js
assert.exists([...code.matchAll(/<div.*?>/gi)][3]);
```

Tu segundo nuevo elemento `div` debe tener una etiqueta de cierre.

```js
assert.exists([...code.matchAll(/<\/div\s*>/gi)][3]);
```

Tus nuevos elementos `div` deben estar dentro del `div` con la clase `container`.

```js
const containerChildren = [...document.querySelector('.container')?.children];
containerChildren.forEach(child => assert.strictEqual(child?.localName, 'div'));
assert.lengthOf(containerChildren, 3);
```

Tus nuevos elementos `div` deben ambos tener una clase de `marker`.

```js
const containerChildren = [...document.querySelector('.container')?.children];
assert.isNotEmpty(containerChildren);
containerChildren.forEach(child => assert.isTrue(child?.classList?.contains('marker')));
```
