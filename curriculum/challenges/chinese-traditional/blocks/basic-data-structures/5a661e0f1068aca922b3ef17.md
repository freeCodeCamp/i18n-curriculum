# --description--

任何資料結構的基本特性當然是能夠不僅儲存資料，還能夠在指令下取用該資料。所以，既然我們已經學會如何創建一個陣列，現在就開始思考如何存取該陣列的資訊。

當我們定義一個簡單的陣列，如下所示，其中有 3 個條款：

```js
let ourArray = ["a", "b", "c"];
```

在陣列中，每個陣列項目都有一個 <dfn>index</dfn>。這個索引同時代表該項目在陣列中的位置，以及你如何取用它。然而，重要的是要注意，JavaScript 陣列是 <dfn>zero-indexed</dfn>，這表示陣列的第一個元素實際上位於 ***第零*** 個位置，而不是第一個。為了從陣列中取回一個元素，我們可以將索引放在括號中並附加到陣列的末尾，或更常見的是，附加到引用陣列物件的變數後面。這稱為 <dfn>bracket notation</dfn>。例如，如果我們想從 `ourArray` 取回 `a` 並將它指定給一個變數，我們可以使用以下程式碼：

```js
let ourVariable = ourArray[0];
```

現在 `ourVariable` 的值是 `a`。

除了存取與索引相關的值之外，你也可以使用相同的符號*設定*索引的值：

```js
ourArray[1] = "not b anymore";
```

使用括號表示法，我們現在已將索引 1 的項目從字串 `b` 重設為 `not b anymore`。現在 `ourArray` 是 `["a", "not b anymore", "c"]`。

# --instructions--

為了完成此挑戰，將 `myArray` 的第 2 個位置（索引 `1`）設為你想要的任何值，但不能是字母 `b`。

# --hints--

`myArray[0]` 應該要和字母 `a` 相等

```js
assert.strictEqual(myArray[0], 'a');
```

`myArray[1]` 不應該與字母 `b` 相等

```js
assert.notStrictEqual(myArray[1], 'b');
```

`myArray[2]` 應該要和字母 `c` 相等

```js
assert.strictEqual(myArray[2], 'c');
```

`myArray[3]` 應該要和字母 `d` 相等

```js
assert.strictEqual(myArray[3], 'd');
```
