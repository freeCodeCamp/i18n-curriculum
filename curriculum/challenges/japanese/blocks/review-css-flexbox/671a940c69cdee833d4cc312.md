## CSS Flexboxとフレックスモデルの紹介

- **定義**: CSS flexboxは、コンテナ内の要素を行や列に並べることができる一次元のレイアウトモデルです。 
- **フレックスモデル**: このモデルは、フレックスコンテナ内でフレックスアイテムがどのように配置されるかを定義します。すべてのフレックスコンテナには主軸と交差軸の2つの軸があります。

## `flex-direction` プロパティ

- **定義**: このプロパティは主軸の方向を設定します。`flex-direction`のデフォルト値は`row`で、すべてのフレックスアイテムをブラウザのデフォルト言語の方向（左から右、または右から左）に同じ行に配置します。 
- **`flex-direction: row-reverse;`**: これは行内のアイテムの順序を逆にします。
- **`flex-direction: column;`**: これはフレックスアイテムを水平方向ではなく垂直方向に揃えます。
- **`flex-direction: column-reverse;`**: これはフレックスアイテムの垂直方向の順序を逆にします。

:::interactive_editor

```html
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: row;
  gap: 10px;
}

.box {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}
```

:::

## `flex-wrap` プロパティ

- **定義**: このプロパティは、フレックスコンテナ内のフレックスアイテムが利用可能なスペースに収まるように折り返す方法を決定します。`flex-wrap`は3つの値を取ることができます：`nowrap`、`wrap`、および`wrap-reverse`。
- **`flex-wrap: nowrap;`**: これはデフォルト値です。フレックスアイテムはコンテナの幅を超えても新しい行に折り返されません。
- **`flex-wrap: wrap;`**: このプロパティは、アイテムがコンテナの幅を超えた場合に折り返します。
- **`flex-wrap: wrap-reverse;`**: このプロパティはフレックスアイテムを逆順で折り返します。
- **`flex-flow` プロパティ**: このプロパティは`flex-direction`と`flex-wrap`の省略形プロパティです。

:::interactive_editor

```html
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
  <div class="box">4</div>
  <div class="box">5</div>
</div>
```

```css
.container {
  display: flex;
  flex-wrap: wrap;
  width: 150px;
  background: #f0f0f0;
}

.box {
  width: 60px;
  padding: 10px;
  margin: 5px;
  background: skyblue;
  text-align: center;
}
```

:::

## `justify-content` プロパティ

- **定義**: このプロパティはフレックスコンテナの主軸に沿って子要素を揃えます。
- **`justify-content: flex-start;`**: この場合、フレックスアイテムは主軸の開始位置に揃えられます。これは水平方向または垂直方向の場合があります。
- **`justify-content: flex-end;`**: この場合、フレックスアイテムは主軸の終了位置に揃えられます（水平方向または垂直方向）。
- **`justify-content: center;`**: これはフレックスアイテムを主軸に沿って中央に配置します。
- **`justify-content: space-between;`**: これは主軸に沿って要素を均等に分布させます。
- **`justify-content: space-around;`**: これは主軸内でフレックスアイテムを均等に分布させ、最初のアイテムの前と最後のアイテムの後にスペースを追加します。
- **`justify-content: space-evenly;`**: これは主軸に沿ってアイテムを均等に分布させます。

:::interactive_editor

```html
<div class="container">
  <div class="box">A</div>
  <div class="box">B</div>
  <div class="box">C</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
  background: #eee;
}

.box {
  padding: 20px;
  background: salmon;
}
```

:::
  
## `align-items` プロパティ

- **定義**: このプロパティは交差軸に沿ってアイテムを分布させるために使います。交差軸は主軸に対して垂直であることを覚えておいてください。
- **`align-items: center;`**: これは交差軸に沿ってアイテムを中央に配置します。
- **`align-items: flex-start;`**: これは交差軸の開始位置にアイテムを揃えます。
- **`align-items: stretch;`**: これは交差軸に沿ってフレックスアイテムを伸ばします。

:::interactive_editor

```html
<div class="container">
  <div class="box tall">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  align-items: center;
  height: 150px;
  background: #ddd;
}

.box {
  background: lightgreen;
  padding: 10px;
}

.tall {
  height: 100px;
}
```
