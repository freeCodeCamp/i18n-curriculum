# --description--

In den vorherigen Lektionen haben Sie Funktionen gewohnt definiert, indem Sie das Schlüsselwort `def` so verwendet haben:

```py
def square(num):
    return num ** 2

print(square(4)) # 16
```

Wenn es jedoch darum geht, mit höherwertigen Funktionen wie `map()` und `filter()` zu arbeiten, können Sie eine anonyme inline-Funktion verwenden. Hier kommen Lambda-Funktionen ins Spiel.

So sieht die `square()`-Funktion aus, wenn sie in eine Lambda-Funktion umgeschrieben wird:

```py
lambda num: num ** 2
```

Wie bereits erwähnt, sind Lambda-Funktionen anonym, daher hat diese Funktion keinen Namen `square` mehr. Lambda-Funktionen sind ideal, wenn Sie sie in höherwertigen Funktionen wie dieser verwenden möchten:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

In diesem Beispiel haben wir eine Liste von Zahlen und möchten eine neue Liste mit geraden Zahlen erstellen. Also übergeben wir eine Lambda-Funktion als eines der Argumente an die `filter()`-Funktion, um eine neue Liste mit den Zahlen `2` und `4` zu erhalten.

Beim Arbeiten mit Lambda-Funktionen ist es wichtig, bewährte Vorgehensweisen zu beachten. Zum Beispiel ist es keine gute Praxis, eine Lambda-Funktion einer Variablen wie folgt zuzuweisen:

```py
numbers = [1, 2, 3, 4, 5]

square = lambda x: x ** 2
squared_numbers = list(map(square, numbers))
print(squared_numbers) # [1, 4, 9, 16, 25]
```

Das widerspricht dem Zweck, anonyme Funktionen zu verwenden. In diesem Fall sollten Sie eine reguläre Funktion verwenden, so wie hier:

```py
numbers = [1, 2, 3, 4, 5]

def square(num):
    return num ** 2

squared_numbers = list(map(square, numbers))
print(squared_numbers) # [1, 4, 9, 16, 25]
```

Außerdem sollten Sie vermeiden, Lambda-Funktionen zu erstellen, die schwer zu lesen oder unnötig kompliziert sind, wie diese:

```py
result = (lambda x: (x**2 + 2*x - 1) if x > 0 else (x**3 - x + 4))(3)
print(result)  # 14
```

Obwohl diese Funktion einwandfrei läuft und das korrekte Ergebnis `14` liefert, ist sie nicht leicht zu lesen oder anzuschauen. In diesem Fall wäre es besser, eine separate Funktion mit einer `if/else`-Anweisung zu erstellen und dann diese Funktion aufzurufen:

```py
def calculate_expression(x):
    if x > 0:
        return x**2 + 2*x - 1
    else:
        return x**3 - x + 4

print(calculate_expression(3))  # 14
```

Sowohl reguläre Funktionen als auch Lambda-Funktionen haben ihre Anwendungsfälle in Python-Programmen. Wenn Sie mit einzelnen inline-Ausdrücken arbeiten, könnten Sie eine Lambda-Funktion in Betracht ziehen. Andernfalls ist die Verwendung einer regulären Funktion der richtige Weg.
