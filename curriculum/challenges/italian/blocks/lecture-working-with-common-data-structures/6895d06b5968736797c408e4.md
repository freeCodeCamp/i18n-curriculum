# --description--

Gli array sono una struttura dati fondamentale nell'informatica. Tutti gli array archiviano collezioni ordinate di dati, ma a seconda del loro tipo, possono funzionare in modo diverso dietro le quinte.

Il loro comportamento sottostante può avere un effetto importante sull'efficienza del programma, quindi impariamo a conoscere gli array dinamici e statici e le loro differenze, così potrai scegliere quello più efficiente per il tuo programma.

Inizieremo con gli array statici.

Gli **array statici** hanno una dimensione fissa. Archiviano gli elementi in posizioni di memoria adiacenti.

La dimensione di un array statico viene determinata quando l'array viene inizializzato. Una volta allocato quel blocco specifico di memoria, è fisso e non può essere modificato mentre il programma è in esecuzione. Questa è una caratteristica chiave degli array statici.

Archiviare gli elementi in posizioni di memoria adiacenti rende il processo di recupero dei dati più efficiente perché il programma può memorizzare la posizione del primo elemento e poi usare gli indici per fare semplici calcoli e trovare gli altri elementi in memoria.

Grazie a questo, consultare i valori di un array statico richiede un tempo costante `O(1)`, che è molto efficiente.

Puoi usare un array statico quando conosci in anticipo il numero di elementi che verranno archiviati. È anche utile quando i valori verranno consultati molto frequentemente, dato che l'operazione di consultazione è molto efficiente.

Tuttavia, questa struttura dati non può crescere o ridursi, quindi se il numero di elementi da archiviare può variare, dovresti usare invece un array dinamico.

Cercare di aumentare la dimensione di un array statico comporterebbe la creazione di un nuovo array e la copia di tutti gli elementi dal vecchio array a quello nuovo, il che è inefficiente. In quel caso, un array dinamico sarebbe molto meglio perché gestisce automaticamente questo processo.

Python non include array statici tradizionali come strutture dati integrate.

Ma altri linguaggi di programmazione, come Java, li supportano. Questo è un esempio di array statico in Java che può archiviare tre numeri interi:

```java
int[] numbers = new int[3]; 
```

Gli array in Python sono dinamici, quindi diamo un’occhiata a quelli.

Gli **array dinamici** sono più flessibili perché possono crescere o ridursi automaticamente mentre il programma è in esecuzione.

Funzionano tramite un meccanismo di ridimensionamento automatico che copia gli elementi in un nuovo array quando l'array originale è pieno. Il processo è fatto in modo efficiente perché la dimensione del nuovo array viene scelta in modo da rendere queste operazioni computazionalmente costose meno frequenti.

Consultare gli elementi di un array dinamico richiede un tempo costante `O(1)`, quindi questa operazione è molto efficiente.

Inserire un elemento nel mezzo dell'array richiede un tempo lineare `O(n)` perché gli elementi successivi devono essere spostati.

Inserire un elemento alla fine dell'array richiede un tempo costante `O(1)` se c’è ancora spazio disponibile nell'array dinamico, ma se l'array è pieno e deve essere ridimensionato, questa operazione ha una complessità `O(n)`.

Dovresti usare gli array dinamici quando non conosci in anticipo il numero di valori che dovrai archiviare nell'array. Sono anche utili quando inserirai e rimuoverai elementi frequentemente.

La struttura dati integrata `list` di Python funziona come un array dinamico. Puoi creare una lista scrivendo gli elementi tra parentesi quadre, separati da virgole.

```python
numbers = [3, 4, 5, 6]
```

Puoi consultare un elemento scrivendo il nome della variabile che contiene la lista, seguito da parentesi quadre, e all’interno delle parentesi quadre l’indice corrispondente.

Gli indici partono da 0 per il primo elemento e aumentano di 1 per ogni elemento successivo:

```python
numbers[0]  # 3
numbers[1]  # 4
numbers[2]  # 5
numbers[3]  # 6
```

Per aggiornare un valore, devi solo riassegnarlo:

```python
numbers[2] = 16
```

Puoi aggiungere elementi alla lista con il metodo `.append()`:

```python
numbers.append(7)
```

Puoi inserire elementi in un indice specifico con il metodo `.insert()`, passando l’indice come primo argomento e l’elemento stesso come secondo argomento.

```python
numbers.insert(3, 15)
```

Puoi rimuovere un elemento in un indice specifico con il metodo `.pop()`:

```python
numbers.pop(2)
```

Se non specifichi l’indice, `.pop()` rimuoverà l’ultimo elemento.

Ci sono altri metodi integrati per le liste che puoi consultare nella documentazione per aggiungere e rimuovere elementi in modo abbastanza semplice.

Questa è la potenza degli array dinamici, o liste in questo caso.

In generale, dovresti usare gli array statici quando conosci in anticipo il numero di elementi e hai bisogno di consultarli frequentemente, e usare gli array dinamici quando il numero di elementi è sconosciuto o variabile nel tempo.

Dovresti sempre considerare il compromesso tra la semplicità degli array statici e la flessibilità degli array dinamici. Sono entrambi utili per casi d’uso e scenari specifici. Saper scegliere il migliore per un dato problema fa parte delle capacità di problem solving che svilupperai gradualmente con la pratica.
