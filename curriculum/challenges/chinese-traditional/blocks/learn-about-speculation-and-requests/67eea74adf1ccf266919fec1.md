# --explanation--

`Should have been` 後面接形容詞，用來表示某事本應該在過去發生，但實際上沒有發生。 例如：

- `I should have been more careful.`－你不夠小心，但如果你更小心會更好。

- `He should have been more prepared for the meeting.`－他準備得不夠充分。

在這種情況下，Mark 的意思是他應該在註解中更詳細。
