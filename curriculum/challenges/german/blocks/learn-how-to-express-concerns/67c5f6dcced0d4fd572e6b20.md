# --explanation--

`Also` wird als Verbindung verwendet, um eine weitere Anfrage, Idee oder Information hinzuzufügen. Es verbindet zwei zusammenhängende Gedanken, während sie als separate Sätze erhalten bleiben. Zum Beispiel:

`We should review the budget. Also, let's check if we have enough resources.` – Das bedeutet, dass die Überprüfung der Ressourcen ein zusätzlicher Schritt nach der Überprüfung des Budgets ist.
