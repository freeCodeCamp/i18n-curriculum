# --description--

在這個任務中，你將學習像 `Did you get it?`、`Got it` 和 `What else?` 這樣的片語。這些片語在英語對話中很常見。`Did you get it?` 是用來詢問某人是否理解某事的問題。`Got it` 是表示「我理解了」的簡短說法。`What else?` 用來詢問額外的資訊或條款。

例如，在會議中，有人可能會說，`Did you get it?`，而你可以回覆，`Got it, what else do we need to discuss?`
