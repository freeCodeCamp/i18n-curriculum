# --description--

Dado um array de inteiros, retorne um array de inteiros que aparecem mais de uma vez no array inicial, ordenados em ordem crescente. Se nenhum valor aparecer mais de uma vez, retorne um array vazio.

```js
function findDuplicates(arr) {
  const duplicates = [];
  
  for (let i = 0; i < arr.length - 1; i++) {
    for (let j = i + 1; j < arr.length; j++) {
      if (arr[i] === arr[j] && !duplicates.includes(arr[i])) {
        duplicates.push(arr[i]);
      }
    }
  }

  return duplicates.sort((a, b) => a - b);
}
```
