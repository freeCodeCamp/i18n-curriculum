# --description--

In der vorherigen Lektion wurden Sie mit dem Listendatentyp vertraut gemacht und haben gelernt, wie Sie auf Elemente einer Liste zugreifen sowie Listen ausschneiden. In dieser Lektion lernen Sie weiterhin Listen kennen und einige gebräuchliche Methoden, die mit ihnen verwendet werden, wie `append()`, `pop()` und `sort()`.

Die erste Methode, die wir uns ansehen, ist die `append()`-Methode. Diese wird verwendet, um ein Element am Ende der Liste hinzuzufügen. Hier ist ein Beispiel für die Verwendung der `append()`-Methode, um die Zahl `6` zu einer Liste von Zahlen hinzuzufügen:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

Wenn Sie eine Liste am Ende einer anderen hinzufügen möchten, können Sie auch die `append()`-Methode so verwenden:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

Beachten Sie, wie die gesamte `even_numbers`-Liste innerhalb der `numbers`-Liste verschachtelt ist.

Wenn Sie jedoch alle einzelnen Zahlen aus der `even_numbers`-Liste am Ende der `numbers`-Liste hinzufügen möchten, können Sie die `extend()`-Methode verwenden.

Die `extend()`-Methode ähnelt der `append()`-Methode, aber mit `extend()` können Sie mehrere Elemente von einer Liste zu einer anderen hinzufügen. Hier ist ein Beispiel, wie die Zahlen `6`, `8` und `10` von einer Liste ans Ende der `numbers`-Liste hinzugefügt werden:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

Wie Sie sehen, ist die verschachtelte Liste verschwunden und es ist nur noch eine Liste von Zahlen.

Um ein Element an einem bestimmten Index in einer Liste einzufügen, können Sie die `insert()`-Methode verwenden. Diese Methode akzeptiert zwei Argumente: den Index, an dem Sie das neue Element einfügen möchten, und das Element, das Sie einfügen wollen.

Hier ist ein Beispiel für die Verwendung der `insert()`-Methode:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

Der folgende Code fügt die Zahl `2.5` an Index `2` in der `numbers`-Liste ein.

Wenn Sie ein Element aus einer Liste entfernen möchten, können Sie die `remove()`-Methode verwenden. Die `remove()`-Methode nimmt den Wert des zu entfernenden Elements als Argument:

```py
numbers = [10, 20, 30, 40, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50]
```

Es ist wichtig zu beachten, dass diese Methode nur das erste Vorkommen eines Elements entfernt. Nicht alle:

```py
numbers = [10, 20, 30, 40, 50, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50, 50]
```

Um ein Element an einem bestimmten Index in der Liste zu entfernen, können Sie die `pop()`-Methode so verwenden:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

Wenn Sie kein Element für die `pop`-Methode angeben, wird das letzte Element entfernt.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

Wenn Sie die Liste leeren müssen, können Sie die `clear()`-Methode so verwenden:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

Die nächste Methode, die wir uns ansehen, ist die `sort()`-Methode. Diese Methode wird verwendet, um die Elemente an Ort und Stelle zu sortieren. Hier ist ein Beispiel, wie eine zufällige Liste von Zahlen an Ort und Stelle sortiert wird:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

Im Gegensatz zur `sort()`-Methode gibt es die `sorted()`-Funktion, die für jedes iterierbare Objekt funktioniert und eine neue sortierte Liste zurückgibt, anstatt die ursprüngliche Liste zu verändern. Zum Beispiel:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(numbers) # [19, 2, 35, 1, 67, 41]
print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
```

Zur Erinnerung: Ein iterierbares Objekt ist ein spezieller Objekttyp, über den Sie iterieren können, sodass Sie auf jedes Element einzeln zugreifen können. Sie werden in einer zukünftigen Lektion mehr darüber lernen, wie Schleifen in Python funktionieren.

Sowohl die `sort()`-Methode als auch die `sorted()`-Funktion akzeptieren optionale `key`- und `reverse`-Parameter. Mehr über diese optionalen Parameter lernen Sie in einer zukünftigen Lektion, wenn Sie etwas über Tuples erfahren. Sie werden auch mehr über andere eingebaute Funktionen wie `sorted()` in zukünftigen Lektionen lernen.

Die nächste Methode, die wir uns ansehen, ist die `reverse()`-Methode. Diese Methode kehrt eine Liste von Elementen an Ort und Stelle um, so:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

Die letzte Methode, die wir uns ansehen, ist die `index`-Methode. Diese wird verwendet, um den ersten Index zu finden, an dem ein Element in einer Liste gefunden werden kann. Hier ist ein Beispiel für die Verwendung der `index`-Methode, um die Sprache `'Java'` in einer `programming_languages`-Liste zu finden:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

Wenn das Element nicht gefunden werden kann, wirft Python eine `ValueError`:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: 'JavaScript' is not in list
"""
```

Es gibt noch einige weitere Methoden für Python-Listen, aber diese erste Liste von Methoden ist ein guter Anfang.
