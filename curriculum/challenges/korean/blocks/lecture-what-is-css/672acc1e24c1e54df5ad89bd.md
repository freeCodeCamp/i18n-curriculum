# --interactive--

HTML과 CSS에서 요소는 **인라인 요소** 또는 **블록 레벨 요소**로 분류되며, 이 분류는 문서 레이아웃에서 요소가 어떻게 동작하는지를 결정합니다.

이 차이를 이해하는 것은 웹페이지에서 콘텐츠가 어떻게 표시되는지 제어하는 데 매우 중요합니다.

블록 레벨 요소는 기본적으로 컨테이너의 가용 너비 전체를 차지하며, 컨테이너의 너비에 맞게 늘어납니다.

이 요소들은 항상 새로운 줄에서 시작하며 다른 콘텐츠를 다음 줄로 밀어내어 하나의 "블록"을 만듭니다.

블록 레벨 요소는 기본적으로 CSS 속성 `display: block;`가 적용되어 있습니다. 이 속성은 요소가 컨테이너의 너비를 채우고 새로운 줄에 나타나도록 보장합니다.

일반적인 블록 레벨 요소로는 `div` 요소, 문단, 제목, 순서 있는 목록, 순서 없는 목록, 섹션 요소가 있습니다.

블록 레벨 요소의 코드 예시는 다음과 같습니다:

:::interactive_editor

```html
<p style="border: 2px solid red;">
  First paragraph
</p>
<p>Second paragraph</p>
```

:::

이 예시에서는 두 개의 문단 요소가 있고, 첫 번째 문단에는 빨간 테두리가 있습니다.

두 문단 요소는 기본적으로 블록 레벨 요소이기 때문에 같은 줄에 있지 않습니다.

따라서 두 문단 요소 모두 이 경우 `body` 요소인 컨테이너의 전체 너비를 차지합니다.

블록 레벨 요소는 문단, 섹션 또는 더 큰 콘텐츠 블록처럼 콘텐츠를 수직으로 쌓고 싶을 때 이상적입니다. 웹페이지의 레이아웃과 구조를 정의하는 데 자주 사용됩니다.

인라인 요소는 블록 레벨 요소와 달리 필요한 만큼의 너비만 차지하며 새로운 줄에서 시작하지 않습니다. 이 요소들은 콘텐츠 내에서 흐르며 텍스트나 다른 인라인 요소가 그 옆에 함께 나타날 수 있게 합니다.

인라인 요소는 기본적으로 CSS 속성 `display: inline;`가 적용되어 있습니다. 이 속성은 요소가 콘텐츠 흐름 내에 머무르고 새로운 줄로 넘어가지 않도록 보장합니다.

일반적인 인라인 요소로는 `span`, `anchor`, `img` 요소가 있습니다.

인라인 요소를 더 잘 이해할 수 있는 예시는 다음과 같습니다:

:::interactive_editor

```html
<p>This is a
  <span style="color: red; border: 2px solid green;">red</span>
  word inside a paragraph.
</p>
```

:::

이 예시에서는 문단 요소 안에 `span` 요소가 중첩되어 있습니다. `span` 요소는 `red` 텍스트 색상과 `green` 테두리가 있어 강조된 단어를 더 잘 볼 수 있습니다.

보시다시피 `span` 요소는 "red"라는 단어만큼의 공간만 차지하며 뒤따르는 콘텐츠를 새로운 줄로 밀어내지 않습니다.

인라인 요소는 한 줄 내에서 텍스트나 콘텐츠의 작은 부분을 스타일링할 때, 예를 들어 단어를 강조하거나 하이퍼링크를 만들거나 문단의 일부에 특정 스타일을 적용할 때 가장 적합합니다.
