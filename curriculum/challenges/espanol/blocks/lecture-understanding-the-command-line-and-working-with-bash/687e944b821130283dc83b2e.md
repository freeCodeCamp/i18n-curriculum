# --description--

Aprendamos sobre Bash y algunos de sus comandos básicos.

Bash significa Bourne Again SHell, y es posiblemente el shell más común que encontrarás en entornos similares a Unix. Por lo tanto, puede ser muy beneficioso entender cómo navegar en este shell y usar los comandos comunes.

El primer comando es `pwd`, que imprime el directorio de trabajo actual en el terminal. El "directorio de trabajo" se refiere al directorio al que el terminal está apuntando actualmente, lo cual es algo importante de saber para muchos de estos otros comandos.

Es totalmente posible que tu terminal esté apuntando al directorio incorrecto. El comando `cd` te permite cambiar de directorio. Puedes especificar una ruta absoluta, precedida por una barra diagonal (`/`), una ruta relativa sin prefijo, y usar la sintaxis de doble punto (`..`) para subir al directorio padre.

El siguiente comando es `ls`, que lista el contenido de tu directorio de trabajo actual. Esto es útil para ver si un archivo o carpeta existe en tu directorio. El comando `ls` también acepta flags para mostrar archivos ocultos (el flag `-a`), permisos de archivos (el flag `-l`) y otras funciones. Aprenderás más sobre los flags de comandos en la siguiente lección. También puedes usar `less` o `cat` para ver el contenido de un archivo. 

¿Qué pasa si el directorio no contiene el archivo o carpeta que necesitas? Hay comandos llamados `mkdir` y touch que te permiten crearlos. `mkdir` crea un nuevo directorio, o carpeta, y `touch` crea un nuevo archivo. Ambos comandos aceptan el nombre del archivo o carpeta que quieres crear. Por ejemplo, `touch readme.md` crea un nuevo archivo Markdown llamado `readme.md`.

Quizás tu nuevo archivo no debería haberse llamado `readme.md`, sino que debería haberse llamado `readthis.md`. Afortunadamente, el comando `mv` te permite mover (o renombrar) un archivo. Toma el nombre del archivo antiguo y el nombre del archivo nuevo.

El comando `rm` te permite eliminar (o borrar) un archivo, y con la bandera `-r` eliminará un directorio. A veces un archivo puede estar protegido, y necesitarás incluir la bandera `-f`.

El comando `cp` se puede usar para copiar un archivo o directorio a una nueva ubicación. A diferencia del comando `mv`, el comando `cp` no elimina el archivo original, y para copiar un directorio, necesitarás pasar la bandera `-r`.

El comando `echo` puede considerarse el equivalente en Bash de una función `console.log()` o `print()`. `echo` toma un argumento de cadena, entre comillas, y lo imprime en el terminal. Esto puede parecer tonto de ejecutar en el terminal, solo para obtener texto de vuelta en el terminal, pero en realidad puedes encadenar `echo` a un símbolo de control. El símbolo `>` te permite especificar un nombre de archivo para crear o sobrescribir con la nueva cadena, como `echo "Naomi was here." > readme.md`, y el símbolo `>>` añadirá al archivo.

Hay muchos más comandos de Bash disponibles para ti, como `head`, `tail`, `chown` y `chmod`. Sería imposible cubrirlos todos en esta lección. Pero puedes usar el comando man para ver el manual o la página de ayuda de casi cualquier comando.

Probablemente recordarás los comandos que usas con más frecuencia. Y para el resto, siempre puedes consultar la página del manual.
