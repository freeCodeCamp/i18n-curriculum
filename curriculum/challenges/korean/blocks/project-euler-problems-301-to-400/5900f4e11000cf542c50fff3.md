# --description--

$R(M, N)$을 $M \lt x \le N$, $M \lt y \le N$를 만족하고 $\left\lfloor\frac{y^2}{x^2}\right\rfloor$가 홀수인 격자점 ($x$, $y$)의 개수라고 하겠습니다.

$R(0, 100) = 3\,019$이고 $R(100, 10\,000) = 29\,750\,422$임을 확인할 수 있습니다.

$R(2 \times {10}^6, {10}^9)$을 구하세요.

**참고:** $\lfloor x\rfloor$는 바닥 함수(floor function)를 나타냅니다.
