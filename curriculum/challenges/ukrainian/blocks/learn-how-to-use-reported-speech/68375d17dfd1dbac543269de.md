# --explanation--

`Which` можна використовувати, щоб додати додаткову інформацію про попередню умову. Це називається **недефінуючим відносним реченням** — воно додає деталі, але не є суттєвим для основного значення речення. Наприклад:

`The system crashed during the update, which caused a delay.` – Частина після `which` дає більше інформації про наслідок аварії, але її можна опустити без зміни основного змісту.
