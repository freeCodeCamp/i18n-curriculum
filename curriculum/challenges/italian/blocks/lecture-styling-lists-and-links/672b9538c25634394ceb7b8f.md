# --interactive--

Ci sono diversi stati di un collegamento, inclusi `link`, `visited`, `hover`, `focus` e `active`. Questi stati sono importanti per aiutare gli utenti a riconoscere i collegamenti e fornire un feedback chiaro dopo le interazioni, migliorando sia l’usabilità sia l’accessibilità.

Definire lo stile di questi diversi stati di collegamento è fondamentale per l’usabilità e l’accessibilità, perché fornisce segnali visivi sullo stato attuale del collegamento. Questo aiuta gli utenti a capire quali collegamenti hanno già visitato, con quale collegamento stanno interagendo e cosa succederà quando cliccheranno.

Per gli utenti con disabilità visiva o cognitive, questi stili distinti possono rendere la navigazione molto più semplice e intuitiva.

Inoltre, stati di collegamento chiari migliorano l’esperienza complessiva dell’utente fornendo un feedback immediato sulle interazioni, riducendo la confusione e migliorando la navigabilità del sito.

Questi stati possono essere stilizzati usando qualcosa chiamato `pseudo-classes` in CSS.

Una pseudo-classe è una parola chiave aggiunta a un selettore che specifica uno stato speciale dell’elemento selezionato.

Per esempio, `:hover` può cambiare il colore di un pulsante quando il puntatore dell’utente ci passa sopra, mentre `:visited` può cambiare il colore di un collegamento già visitato.

Le pseudo-classi ti permettono di definire lo stile degli elementi in base al loro stato o all’interazione dell’utente con essi, senza bisogno di markup aggiuntivo nel tuo HTML.

La sintassi di una `pseudo-class` è qualcosa del genere, dove `A` è il selettore e `:B` è la `pseudo-class`:

```css
A:B {
  property: value;
}
```

Per capire meglio come definire lo stile dei diversi stati di un collegamento, diamo un’occhiata ad alcuni esempi.

La pseudo-classe `:link` definisce lo stile dei collegamenti non visitati, indicando che sono cliccabili.

Ecco un esempio di come selezionare un elemento àncora e usare la pseudo-classe `:link`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Normal state (unvisited link) */
a:link { 
  color: red;
}
```

:::

L’esempio sopra cambierà il colore blu predefinito del collegamento in `red` quando non è stato visitato.

`:visited` definisce lo stile dei collegamenti già visitati o cliccati, aiutando gli utenti a tenere traccia di quali collegamenti hanno già cliccato. Ecco un esempio di utilizzo della pseudo-classe `:visited`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="https://www.freecodecamp.org/learn/" target="_blank">freeCodeCamp</a>
```

```css
/* Visited link */
a:visited {
  color: green;
}
```

:::

Questo codice colorerà il collegamento in `green` quando è stato cliccato.

`:hover` cambia lo stile del collegamento quando l’utente ci passa sopra con il mouse, fornendo un segnale visivo che il collegamento è interattivo. Ecco un esempio di utilizzo della pseudo-classe `:hover`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Hover state */
a:hover {
  color: green;
}
```

:::

Questo codice colorerà i collegamenti in `green` quando il mouse ci passa sopra.

`:focus` aggiunge uno stile attorno al collegamento quando è in focus, ad esempio durante la navigazione con la tastiera, o per migliorare l’accessibilità.

Ecco un esempio che usa la proprietà `outline` per applicare un contorno arancione solido quando il collegamento è in focus.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Focus state */
a:focus {
  outline: 2px solid orange;
}
```

:::

`:active` cambia lo stile del collegamento mentre viene cliccato, fornendo un feedback immediato all’utente che l’azione è stata registrata. Ecco un esempio di utilizzo della pseudo-classe `:active`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Active state */
a:active {
  color: pink;
}
```

:::

Questo esempio di codice farà diventare il collegamento di colore `pink` immediatamente quando viene cliccato.
