# --explanation--

Ángela가 `¿Navarro con dos erres?`라고 말합니다. 이것은 확인 질문입니다.

`Con`는 철자에 관한 추가 정보를 제공합니다.

`Dos`는 숫자 2입니다.

`Erres`는 `erre`의 복수형이며, `r`라는 글자의 이름입니다.

Ángela는 Navarro라는 성이 r 문자 두 개로 쓰였는지 확인하고 있습니다 `r`.
