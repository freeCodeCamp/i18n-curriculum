# --description--

Mateo 正在用西班牙语通过以下结构，使用动词 `tener`（“有”）来说明他的年龄：

`Tengo` + 数字 + `años`。

请记住，西班牙语中 21 到 29 的数字以 `veinti-` 开头，后面跟相应的数字。例如：

25 是 `veinticinco`。这是通过去掉 `e` 并在数字 `cinco` 前添加 `i` 得到的结果。

数字 1 到 9 的西班牙语是：

| Number | Spanish Word |
| :---: | :---: |
| 1 | `uno` |
| 2 | `dos` |
| 3 | `tres` |
| 4 | `cuatro` |
| 5 | `cinco` |
| 6 | `seis` |
| 7 | `siete` |
| 8 | `ocho` |
| 9 | `nueve` |
