# --interactive--

CSSの属性セレクターは、属性に基づいてHTML要素をターゲットにできる強力なツールです。これにより、特定の属性を持つ要素や、その属性の特定の値を持つ要素にスタイルを適用できます。

これは、動的に要素のスタイルを設定したい場合や、クラス名だけでは十分な特異性が得られない場合に特に役立ちます。

例えば、属性セレクターを使って`href`属性を持つすべてのリンクをターゲットにできます。これは、ページ上のすべてのリンクに統一されたスタイルを適用したいときに便利です。

以下は、`href`属性を持つリンクをターゲットにする属性セレクターの使い方です。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Example link with an href attribute</a>
<a>Example link without an href attribute</a>
```

```css
a {
  display: block;
}

a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

このCSSルールは、`href`属性を含むリンクに青色の文字色と下線を適用し、すべてのクリック可能なリンクが一貫したスタイルになるようにします。

しかし、属性セレクターはもっと具体的に指定できます。例えば、`title`属性を持つリンクだけをターゲットにしたい場合です。`title`属性はリンクに関する追加情報を提供することが多く、これらのリンクを別のスタイルにして、追加情報があることを示したいかもしれません。以下のように指定できます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

このルールは、`title`属性を持つリンクに太字を適用し、下線を削除します。これにより、ページ上の他のリンクと視覚的に区別できます。

属性セレクターは組み合わせて、さらに正確に指定することも可能です。例えば、`href`属性と`title`属性の両方を持つリンクにスタイルを適用したい場合は、次のようにします。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[href][title] {
  display:block;
  color: green;
}
```

:::

この場合、`href`属性と`title`属性の両方を含むリンクだけが緑色の文字色でスタイルされます。このような細かい制御が属性セレクターの強みです。

属性セレクターのもう一つの例は、属性の値が空白で区切られたリストの中に特定の値が含まれているかをマッチさせることです。

以下は複数のクラスを持つアンカー要素の例です。

```html
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

この特定のアンカー要素をターゲットにするには、次のセレクターを使います。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

ここでは、`[attr~=value]`構文を使って、class属性に`"primary"`という単語が含まれるすべてのアンカー要素をターゲットにしています。

もし属性値が特定の値で始まる要素をターゲットにしたい場合は、`[attr^=value]`構文を使います。例を示します。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

この例では、`a[href^="https://"]`セレクターが、`href`属性の値が`"https://"`で始まるすべてのアンカー要素をターゲットにします。

属性値が特定の値で終わる要素をターゲットにしたい場合は、`[attr$=value]`構文を使います。例を示します。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Visit Example Site</a>
```

```css
a[href$=".com"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

この例では、`a[href$=".com"]`セレクターが、`href`属性の値が`.com`で終わるすべてのアンカー要素をターゲットにします。

属性セレクターを使うことで、ウェブページのスタイルを強化できるだけでなく、リンクのようなインタラクティブな要素を属性に基づいて区別しやすくすることでアクセシビリティも向上します。
