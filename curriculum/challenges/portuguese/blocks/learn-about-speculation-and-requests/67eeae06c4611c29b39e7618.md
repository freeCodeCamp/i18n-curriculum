### --feedback--

Essas três palavras são usadas para falar sobre algo que teria sido uma boa ideia no passado mas não foi feito. A terceira palavra é o `Past Participle` de `run`.

# --explanation--

`Should have` + `Past Participle` é usado para expressar arrependimento ou conselho sobre o passado. Por exemplo:

- `You should have run the program on a different machine to check for compatibility.` - Testar em uma máquina diferente teria sido útil, mas não foi feito.

- `I should have run the tests before deploying the app.` - Executar os testes era a coisa certa a fazer, mas não foi feito na época.

Neste caso, Sophie está dizendo a Mark que ele deveria ter executado a suíte de testes, o que significa que era importante, mas ele não fez isso.
