# --explanation--

스페인어는 나이를 표현할 때 동사 `tener`("가지다")를 사용합니다. 예를 들어:

`Él tiene veinte años` - 그는 스무 살입니다.

직업을 설명할 때는 보통 직업을 나타내는 동사 `ser`("이다")와 나이를 나타내는 동사 `tener`("가지다")를 함께 사용합니다. 예를 들어:

`Es diseñador gráfico` (직업) + `y tiene veintiséis años` (나이).
