# --description--

Cuando lanzas los dados y haces una selección, no puedes mantener la puntuación que seleccionaste y avanzar a la siguiente ronda.

Crea una función `updateScore` para añadir esta funcionalidad. Tu función necesitará dos parámetros para la opción de puntuación seleccionada por el usuario. Al primer parámetro se le pasará el `value` del botón de radio, recuerda que esto es una cadena de texto, y al segundo parámetro se le pasará el valor `id` del botón de radio, que es el tipo de puntuación que lograron.

La función necesitará añadir el valor seleccionado por el usuario a la puntuación, actualizar el texto de la puntuación total en la página y añadir un nuevo elemento `li` al elemento de historial de puntuaciones `ul`, usando el formato `${achieved} : ${selectedValue}` para el contenido del elemento `li`.

# --hints--

```js
assert.isFunction(updateScore);
```

```js
score = 0;
updateScore("10", "hi");
assert.strictEqual(score, 10);
```

```js

```

```js
scoreHistory.innerHTML = "";

assert.strictEqual(scoreHistory.children.length, 1);
```

```js
scoreHistory.innerHTML = "";
updateScore(10, "hi");
assert.strictEqual(scoreHistory.innerHTML, "<li>hi : 10</li>");
```

```js
const getHighestDuplicates = (arr) => {
  const counts = {};

  for (const num of arr) {
    if (counts[num]) {
      counts[num]++;
    } else {
      counts[num] = 1;
    }
  }

  let highestCount = 0;

  for (const num of arr) {
    const count = counts[num];
    if (count >= 3 && count > highestCount) {
      highestCount = count;
    }
    if (count >= 4 && count > highestCount) {
      highestCount = count;
    }
  }

  const sumOfAllDice = arr.reduce((a, b) => a + b, 0);

  if (highestCount >= 4) {
    updateRadioOption(1, sumOfAllDice);
  }

  if (highestCount >= 3) {
    updateRadioOption(0, sumOfAllDice);
  }

  updateRadioOption(5, 0);
};

const resetRadioOptions = () => {
  scoreInputs.forEach((input) => {
    input.disabled = true;
    input.checked = false;
  });

  scoreSpans.forEach((span) => {
    span.textContent = "";
  });
};

rollDiceBtn.addEventListener("click", () => {
  if (rolls === 3) {
    alert("You have made three rolls this round. Please select a score.");
  } else {
    rolls++;
    resetRadioOptions();
    rollDice();
    updateStats();
    getHighestDuplicates(diceValuesArr);
  }
});
```
