# --description--

Ausgehend von der oberen linken Ecke eines 2×2-Gitters und nur mit der Möglichkeit, sich nach rechts und nach unten zu bewegen, gibt es genau 6 Wege zur unteren rechten Ecke.

Wie viele solcher Wege gibt es durch ein gegebenes `gridSize`?

# --solutions--

```js
function latticePaths(gridSize) {
  let paths = 1;

  for (let i = 0; i < gridSize; i++) {
    paths *= (2 * gridSize) - i;
    paths /= i + 1;
  }
  return paths;
}
```
