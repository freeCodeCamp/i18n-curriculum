# --explanation--

`Longer` significa ter mais distância de ponta a ponta.Por exemplo:

`We need longer cables to reach the new server racks.` – Isso significa cabos com mais comprimento.

`Twice` significa duas vezes a quantidade.Por exemplo:

`We ordered twice the number of processors this time.` – Isso significa o dobro da quantidade.

`As long as` é usado para comparação direta de comprimento.Por exemplo:

`The new cable is as long as the room.` – Isso mostra que o comprimento do cabo corresponde ao tamanho da sala.
