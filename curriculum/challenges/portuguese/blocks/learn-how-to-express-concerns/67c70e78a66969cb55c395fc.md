# --explanation--

`Team` refere-se a várias pessoas, é um substantivo singular. Embora existam algumas exceções a esta regra, substantivos coletivos, como `team`, `company` ou `group` usam verbo no singular, então usamos `feels` em vez de `feel`. Outros exemplos:

- `The company values its employees.` - Note que o verbo singular `values` é usado porque `company` é um substantivo singular.

- `The group decides where to meet.` - Note que o verbo singular `decides` é usado porque `group` é um substantivo singular.
