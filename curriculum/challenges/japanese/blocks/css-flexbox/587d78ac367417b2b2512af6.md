# --description--

フレックスコンテナ内のフレックスアイテムがコンテナの全スペースを埋めない場合があります。フレックスアイテムを特定の方法で整列させたり間隔を空けたりしたいことはよくあります。幸い、`justify-content` プロパティにはそれを実現するためのいくつかのオプションがあります。ただし、そのオプションを確認する前に理解しておくべき重要な用語があります。

<a href="https://www.freecodecamp.org/news/flexbox-the-ultimate-css-flex-cheatsheet/" target="_blank" rel="noopener noreferrer nofollow">フレックスボックスのプロパティについて詳しくはこちらをお読みください</a>

フレックスコンテナを row に設定すると、フレックスアイテムは左から右へ横並びになります。column に設定すると、フレックスアイテムは上から下へ縦に積み重なります。それぞれの場合、フレックスアイテムが配置される方向を **主軸** と呼びます。row の場合は各アイテムを横切る水平線で、column の場合はアイテムを縦に通る線です。

主軸に沿ってフレックスアイテムの間隔をどう配置するかにはいくつかのオプションがあります。最もよく使われるのは `justify-content: center;` で、フレックスアイテムをすべてフレックスコンテナの中央に整列させます。他のオプションには以下があります：

<ul><li><code>flex-start</code>: アイテムをフレックスコンテナの開始位置に整列させます。row の場合はアイテムがコンテナの左側に寄せられ、column の場合は上側に寄せられます。<code>justify-content</code> が指定されていない場合のデフォルトの整列です。</li><li><code>flex-end</code>: アイテムをフレックスコンテナの終了位置に整列させます。row の場合は右側に、column の場合は下側に寄せられます。</li><li><code>space-between</code>: アイテムを主軸の中央に整列させ、アイテム間に余白を配置します。最初と最後のアイテムはフレックスコンテナの端に寄せられます。例えば row の場合、最初のアイテムはコンテナの左端に、最後のアイテムは右端に寄せられ、残りのスペースが他のアイテム間に均等に分配されます。</li><li><code>space-around</code>: <code>space-between</code> に似ていますが、最初と最後のアイテムはコンテナの端に固定されず、すべてのアイテムの周りにスペースが分配され、コンテナの両端には半分のスペースが配置されます。</li><li><code>space-evenly</code>: フレックスアイテム間に均等にスペースを分配し、コンテナの両端にも同じ量のスペースが配置されます。</li></ul>

# --instructions--

このプロパティの動作を示す例です。`justify-content` 要素に CSS プロパティ `#box-container` を追加し、値を `center` にしてください。

**ボーナス**  
コードエディターで `justify-content` プロパティの他のオプションも試して違いを確認してください。ただし、このチャレンジをクリアできるのは `center` の値だけです。

# --hints--

`#box-container` 要素に `justify-content` プロパティを `center` の値で設定してください。

```js
const boxContainer = document.querySelector('#box-container');
const justifyDirection =
  window.getComputedStyle(boxContainer)['justify-content'];
assert.strictEqual(justifyDirection, 'center');
```
