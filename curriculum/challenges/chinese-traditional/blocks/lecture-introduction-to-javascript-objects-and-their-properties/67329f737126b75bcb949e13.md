# --interactive--

在 JavaScript 中，物件是基本的資料結構，允許你儲存並組織相關的資料和功能。 

你可以將物件視為一個容器，裡面裝有各種資訊，就像檔案櫃裡放著不同的資料夾和文件一樣。 

這些資訊稱為屬性，並且由名稱（或鍵）和值組成。

```js
const exampleObject = {
  propertyName: value,
}
```

物件非常多功能，並且構成了 JavaScript 的核心。事實上，JavaScript 中幾乎所有東西都是物件或可以被當作物件來處理。這包括陣列、函式（程式）、甚至在某些用法下的原始資料型別，如字串和數字。 

JavaScript 以物件為中心的特性是它成為如此靈活且強大語言的原因之一。讓我們來看看你如何創建一個物件：

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};
```

在這個範例中，我們已經創建了一個名為 `person` 的物件，該物件有三個屬性：`name`、`age` 和 `city`。每個屬性都有一個名稱和一個值，兩者以冒號分隔。

現在，讓我們來探討你如何存取這些屬性。在 JavaScript 中，有兩種主要方式存取物件屬性：點記法和括號記法。

點記法是存取物件屬性最常見且最直接的方式。以下是點記法的基本語法：

```js
objectName.propertyName
```

以下是你如何對我們的 `person` 物件使用點記法：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

console.log(person.name);  // Alice
console.log(person.age);   // 30
```

:::

點記法簡潔且易於閱讀，當你知道要存取的屬性名稱確切且該名稱是有效的 JavaScript 識別字（意指它不以數字開頭且不包含特殊字元或空格）時，點記法是首選。

另一方面，括號表示法允許你使用方括號內的字串來存取物件屬性。以下是你如何使用括號表示法：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

console.log(person["name"]); // Alice
console.log(person["age"]); //  30
```

:::

括號表示法比點表示法更靈活，因為它允許你使用不是有效 JavaScript 識別字的屬性名稱。 例如，如果你有一個包含空格或以數字開頭的屬性名稱，你就需要使用括號表示法：

:::interactive_editor

```js
const oddObject = {
  "1stProperty": "Hello",
  "property with spaces": "World"
};

console.log(oddObject["1stProperty"]);  // Hello
console.log(oddObject["property with spaces"]);  // World
```

:::

方括號表示法的另一個優點是它允許你使用變數來動態存取屬性：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "Wonderland"
};

let propertyName = "city";
console.log(person[propertyName]); // Wonderland
```

:::

這種彈性使得括號表示法特別有用，當你在撰寫程式碼時不知道確切的屬性名稱，或當你處理來自使用者輸入或其他動態來源的屬性名稱時。

值得注意的是，JavaScript 中的物件非常強大且多功能。它們不僅可以包含像字串和數字這樣的簡單值，還可以包含陣列或其他物件。

理解物件以及如何操作它們在 JavaScript 中至關重要，因為它們在整個語言以及許多 JavaScript 函式庫和框架中被廣泛使用。 

當你持續學習並使用 JavaScript 時，你會發現掌握物件為創建複雜且強大的應用程式開啟了無限可能的世界。
