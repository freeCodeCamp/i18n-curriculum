# --description--

情報理論やコンピューターサイエンスにおいて、**レーベンシュタイン距離**は2つのシーケンス間の差異の量を測る指標（編集距離）です。2つの文字列間のレーベンシュタイン距離は、一方の文字列をもう一方に変換するために必要な編集の最小数として定義され、許される編集操作は単一文字の挿入、削除、または置換です。

例：

「**kitten**」と「**sitting**」のレーベンシュタイン距離は3です。以下の3つの編集で一方をもう一方に変換でき、3回未満の編集で変換する方法はありません。

<ul>
  <li><strong>k</strong>itten   <strong>s</strong>itten   （'k'を's'に置換）</li>
  <li>sitt<strong>e</strong>n   sitt<strong>i</strong>n   （'e'を'i'に置換）</li>
  <li>sittin   sittin<strong>g</strong>   （末尾に'g'を挿入）</li>
</ul>

*「**rosettacode**」と「**raisethysword**」のレーベンシュタイン距離は**8**です。*

*2つの文字列の距離は、両方の文字列を逆順にしても同じです。*

# --instructions--

2つの文字列をパラメータとして受け取り、それらのレーベンシュタイン距離を返す関数を書いてください。

```js
function levenshtein(a, b) {
  var t = [], u, i, j, m = a.length, n = b.length;
  if (!m) { return n; }
  if (!n) { return m; }
  for (j = 0; j <= n; j++) { t[j] = j; }
  for (i = 1; i <= m; i++) {
    for (u = [i], j = 1; j <= n; j++) {
      u[j] = a[i - 1] === b[j - 1] ? t[j - 1] : Math.min(t[j - 1], t[j], u[j - 1]) + 1;
    } t = u;
  } return u[n];
}
```
