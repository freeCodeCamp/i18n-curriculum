# --explanation--

`Not that I know of` se usa para decir que, hasta donde tú sabes, algo no es cierto o no ha ocurrido. A menudo se usa para decir `no` de manera educada, dejando abierta la posibilidad de que quien habla pueda estar equivocado. Por ejemplo:

- Persona 1: `Has the update been installed?`

- Persona 2: `Not that I know of.` - Esto significa que no crees que la actualización se haya instalado, pero no estás completamente seguro.
