# --explanation--

`Built-in` bezieht sich auf Funktionen oder Werkzeuge, die von Anfang an als Teil eines Softwareprogramms oder Geräts enthalten sind, ohne dass sie separat hinzugefügt oder installiert werden müssen.

Wenn etwas `built-in` ist, ist es bereits vorhanden und einsatzbereit. Zum Beispiel:

- `This laptop has a built-in camera.` – Die Kamera ist als Teil des Laptops eingebaut.

- `The software comes with built-in security features.` – Die Sicherheitsfunktionen sind bereits in der Software eingebaut.
