# --description--

Щоб запитати когось, чи він хоче щось, іспанська використовує дієслово `querer`.

Форма дієслова `quieres` використовується з `tú` (ти) для запитання про бажання або вподобання іншої людини. Це поширено в повсякденному спілкуванні. Наприклад:

- `¿Quieres ayuda?` - Ти хочеш допомогу?

- `¿Quieres más tiempo?` - Ти хочеш більше часу?
