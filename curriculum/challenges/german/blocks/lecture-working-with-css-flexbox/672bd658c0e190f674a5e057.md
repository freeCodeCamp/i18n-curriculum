# --interactive--

Flex-Eigenschaften sind Eigenschaften, die Sie auf Flex-Container anwenden können, um die Verteilung der Kindelemente zu bestimmen. Wir behandeln einige der am häufigsten verwendeten: `flex-wrap`, `justify-content` und `align-items`.

Beginnen wir mit `flex-wrap`. Diese Eigenschaft bestimmt, wie Flex-Elemente innerhalb eines Flex-Containers umgebrochen werden, um in den verfügbaren Platz zu passen. `flex-wrap` kann drei mögliche Werte annehmen: `nowrap`, `wrap` und `wrap-reverse`. `nowrap` ist der Standardwert – Flex-Elemente werden nicht in eine neue Zeile umgebrochen, selbst wenn ihre Breite die Breite des Containers überschreitet.

Im folgenden Code haben wir drei `div`-Elemente. Konzentrieren wir uns auf den `width`. Der `main`-Container mit rotem Rahmen hat eine `width` von `200px`, während seine drei Kind-`div`-Elemente zusammen eine `width` von `240px` haben (jeweils `80px`):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::


Die Breite der `div`-Elemente überschreitet die Breite ihres Containers, aber standardmäßig werden sie verkleinert, um in den verfügbaren Platz zu passen. Wenn Sie möchten, dass sie umgebrochen werden, wenn sie die Breite ihres Containers überschreiten, können Sie `flex-wrap: wrap` auf dem Flex-Container setzen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  flex-wrap: wrap;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Die `div`-Elemente werden in Reihen neu angeordnet, wenn sie die Breite ihres Containers überschreiten. Sie können Flex-Elemente auch in umgekehrter Reihenfolge umbrechen mit `flex-wrap: wrap-reverse`. Die `flex-flow`-Eigenschaft ist eine Kurzschreibweise-Eigenschaft für `flex-direction` und `flex-wrap`. In diesem Beispiel setzen wir `flex-direction` auf `column` und `flex-wrap` auf `wrap-reverse`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  flex-flow: column wrap-reverse;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Super. Nun sprechen wir über `justify-content`. `justify-content` richtet die Kindelemente entlang der Hauptachse des Flex-Containers aus. Wenn Sie den Wert `flex-start` der `justify-content` zuweisen, werden die Flex-Elemente am Anfang der Hauptachse ausgerichtet. Das kann horizontal oder vertikal sein:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: flex-start;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Mit `justify-content: flex-end` werden Flex-Elemente am Ende der Hauptachse ausgerichtet, horizontal oder vertikal.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: flex-end;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Um die Flex-Elemente entlang der Hauptachse zu zentrieren, können Sie `justify-content: center` verwenden.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: center;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Um die Elemente gleichmäßig entlang der Hauptachse zu verteilen, können Sie `justify-content: space-between` verwenden. Dadurch wird bei Bedarf etwas Platz zwischen den Flex-Elementen eingefügt.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: space-between;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

`justify-content: space-around` verteilt Flex-Elemente gleichmäßig innerhalb der Hauptachse und fügt einen Abstand vor dem ersten und nach dem letzten Element hinzu. Dieser zusätzliche Abstand entspricht der Hälfte des Abstands zwischen jeweils zwei benachbarten Elementen. Wenn nur ein Element verteilt wird, wird es zentriert.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: space-around;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::


Und zu guter Letzt haben wir `justify-content: space-evenly`, das die Elemente gleichmäßig entlang der Hauptachse verteilt. Der Abstand zwischen den Elementen sowie der Abstand vor dem ersten und nach dem letzten Element sind genau gleich:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: space-evenly;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;

#third-div {
  background-color: #4da3b2;
```

Super. Jetzt wissen Sie, wie Sie Flex-Elemente entlang der Hauptachse verteilen. Aber Sie möchten sie vielleicht auch entlang der Querachse verteilen. Denken Sie daran, dass die Querachse senkrecht zur Hauptachse steht. Das können Sie mit der `align-items`-Eigenschaft tun. Um die Elemente entlang der Querachse zu zentrieren, müssen Sie nur `align-items: center` zum Flex-Container hinzufügen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: center;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

In diesem Beispiel sind die Flex-Elemente entlang der Querachse zentriert, die standardmäßig vertikal ist. Wenn die Querachse horizontal ist, werden sie stattdessen horizontal zentriert. Im Gegensatz dazu richtet `align-items: flex-start` die Elemente am Anfang der Querachse aus:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: flex-start;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Das Gegenteil ist `align-items: flex-end`. Damit werden Flex-Elemente am Ende der Querachse ausgerichtet, vertikal oder horizontal.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: flex-end;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Um die Flex-Elemente entlang der Querachse zu strecken, können Sie `align-items: stretch` verwenden. Das funktioniert auch bei Elementen, die automatisch dimensioniert sind, wie solche ohne festgelegte `width` oder `height` oder mit nur einem Minimum an `width` oder `height`. Die Flex-Elemente werden sich in Richtung der Querachse strecken, um den Container auszufüllen.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box">Three</div>
</div>

```

```css
.container {
  display: flex;
  align-items: stretch; 
  border: 2px solid #444;
  height: 200px; 
}

.box {
  width: 100px;           
  border: 1px solid #888;
  background-color: lightblue;
}
```

:::

Und schließlich können Sie mit der `align-self`-Eigenschaft einem einzelnen Flex-Element eine andere Ausrichtung auf der Querachse zuweisen. Zum Beispiel können Sie es mit `align-self: stretch` strecken.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box special">Three</div>
</div>

```

```css
.container {
  display: flex;
  align-items: flex-start; 
  border: 2px solid #444;
  height: 200px; 
}

.box {
  width: 100px;
  border: 1px solid #888;
  background-color: lightblue;
  margin: 4px;
}

.special {
  align-self: stretch; 
  background-color: lightcoral;
}
```

:::

Sie können es mit `align-self: center` zentrieren.

:::interactive_editor

Es gibt noch weitere Flex-Eigenschaften und Werte, aus denen Sie wählen können, um das responsive Layout zu erstellen, das Sie sich vorstellen, aber dies sind die am häufigsten verwendeten.
