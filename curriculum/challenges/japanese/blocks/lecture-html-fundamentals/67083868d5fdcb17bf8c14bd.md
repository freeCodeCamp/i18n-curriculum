# --interactive--

HTMLエンティティ、または文字参照とは、HTMLで予約された文字を表すために使われる文字のセットです。

画面に`This is an <img/> element`というテキストを表示したいとしましょう。現在エディターにあるコードを使うと、期待した結果は表示されません。例に`src`や`alt`属性を追加しても、段落の途中に画像が表示されるだけで、望んだ結果にはなりません。例を操作するには、インタラクティブエディターを有効にしてください。


:::interactive_editor

```html
<p>This is an <img /> element</p>
```

:::

HTMLパーサーが小なり記号（`<`）の後にHTMLタグ名を見つけると、それをHTML要素として解釈します。だから画面に`This is an <img/> element`が表示されないのです。

この問題を解決するには、HTMLエンティティを使います。こちらは小なり（`<`）と大なり（`>`）の記号に正しいHTMLエンティティを使った更新例です。これで画面に`This is an <img/> element`が表示されるはずです。

インタラクティブエディターを有効にして、段落要素の下に`&lt;p&gt;learning is fun&lt;/p&gt;`を追加してみてください。画面に`<p>learning is fun</p>`が表示されるはずです。

:::interactive_editor

```html
<p>This is an &lt;img /&gt; element</p>

```

:::

このような文字参照は名前付き文字参照と呼ばれます。名前付き参照はアンパサンド記号（`&`）で始まり、セミコロン（`;`）で終わります。名前付き文字参照を使うことで、HTMLパーサーはこれを実際のHTML要素と混同しません。

もう一つの文字参照の種類は10進数の数値参照です。この文字参照はアンパサンド記号とシャープ記号（`#`）で始まり、1つ以上の10進数の桁が続き、セミコロンで終わります。

こちらは小なり記号の10進数数値参照を使った例です。

インタラクティブエディターを有効にしてコードを変えて、異なる記号を表示してみてください。著作権記号には`&#169;`、登録商標記号には`&#174;`を使えます。

:::interactive_editor

```html
&#60;
```

:::

最後の文字参照の種類は16進数の数値参照です。この文字参照はアンパサンド記号、シャープ記号、そして文字`x`で始まります。その後に1つ以上のASCIIの16進数の桁が続き、セミコロンで終わります。

こちらは小なり記号の16進数数値参照を使った例です。

インタラクティブエディターを有効にしてコードを変えて、異なる記号を表示してみてください。ユーロ記号には`&#x20AC;`、ギリシャ文字の大文字オメガ記号には`&#x03A9;`を使えます。

:::interactive_editor

```html
&#x3C;
```

:::
