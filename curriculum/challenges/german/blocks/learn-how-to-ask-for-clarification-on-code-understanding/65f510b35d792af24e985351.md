# --description--

Im Programmieren bezeichnet der Begriff `persistent` oft etwas, das weiterhin besteht oder auftritt, insbesondere ein Problem oder einen Bug, der schwer zu lösen ist.

`Persistence` ist die Substantivform und bezeichnet den Zustand des Andauernden.

Zum Beispiel ist ein `persistent error` im Code ein Fehler, der immer wieder auftritt und schwer zu beheben ist.
