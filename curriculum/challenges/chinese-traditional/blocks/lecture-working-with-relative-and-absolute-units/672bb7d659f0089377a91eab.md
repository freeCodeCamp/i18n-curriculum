# --interactive--

CSS 中的百分比是相對單位，允許你將大小、尺寸和其他屬性定義為其父元素的比例。當你使用百分比值時，實際上是在說：「將此設為其容器的 X%。」

例如，如果你在一個元素上設定 `width: 50%`，它將佔據其父容器寬度的一半。這使得百分比在建立可適應不同螢幕尺寸的響應式設計時非常有用。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <div class="box"></div>
</div>
```

```css
.container {
  width: 400px;
  height: 200px;
  background-color: lightgray;
}

.box {
  width: 50%;
  height: 100%;
  background-color: red;
}
```

:::

百分比非常適合用來建立可隨不同螢幕尺寸調整的流動版面配置。例如，將容器設定為 `width: 80%` 可確保它佔據其父元素寬度的 80％，無論裝置為何。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="parent">
  <div class="child"></div>
</div>
```

```css
.parent {
  width: 100%;
  height: 300px;
  background-color: lightblue;
}

.child {
  width: 80%;
  height: 100%;
  background-color: red;
}
```

:::

使用百分比來設定彈性影像是另一種常見做法。透過對影像套用 `max-width: 100%`，你可以讓它們在較小的螢幕上縮放，同時維護其長寬比。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<img src="https://placehold.co/150x150" alt="Example Product Image" />
```

```css
img {
  max-width: 100%;
  height: auto;
}
```

:::

雖然較少見，百分比也可以用於字型大小，以創建可縮放的排版。例如，`font-size: 120%` 會使文字比其父元素的字型大小大 20%。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="text-container">
  Parent text.
  <p class="text">This is some example text.</p>
</div>
```

```css
.text-container {
  font-size: 16px;
}

.text {
  font-size: 120%;
}
```

:::

百分比在垂直置中時特別有用。以下是一個範例，說明你如何使用百分比搭配 `transform` 屬性來垂直置中一個元素。

此範例將元素定位於其容器頂部的 50％，然後使用 `transform` 將其向上移動自身高度的一半，有效地垂直置中。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="centered"></div>
```

```css
.centered {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
  width: 300px;
  height: 300px;
  background-color: red;
}
```

:::

你將在未來的課程中更詳細地了解絕對定位和 transform 屬性的運作方式。 

請記住，百分比總是相對於某些東西。對於像 `width` 這樣的水平屬性，它們是相對於父元素的寬度。對於像 `height` 這樣的垂直屬性，它們通常是相對於父元素的高度（如果有指定）。

但是，當巢狀包含具有百分比尺寸的元素時要小心，因為這可能導致意外的結果。此外，如果父元素沒有定義高度，百分比高度可能會很棘手。

總結來說，CSS 中的百分比是用來建立彈性且響應式設計的強大工具。當你希望元素相對於其容器按比例縮放，或需要版面配置能無縫適應不同螢幕尺寸時，請使用它們。
