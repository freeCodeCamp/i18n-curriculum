# --description--

ハイポサイクロイドとは、大きな円の内側を小さな円が転がるときに、小さな円上の一点が描く曲線です。原点を中心とし、右端の点から始まるハイポサイクロイドの媒介変数表示は次の通りです。

$$x(t) = (R - r) \cos(t) + r \cos(\frac{R - r}{r}t)$$

$$y(t) = (R - r) \sin(t) - r \sin(\frac{R - r}{r} t)$$

ここで、$R$ は大きな円の半径、$r$ は小さな円の半径です。

$C(R, r)$ を、半径 $R$ と $r$ のハイポサイクロイド上の整数座標を持つ異なる点の集合とし、対応する $t$ の値に対して $\sin(t)$ と $\cos(t)$ が有理数となるものとします。

$S(R, r) = \sum_{(x,y) \in C(R, r)} |x| + |y|$ を、$C(R, r)$ に含まれる点の $x$ 座標と $y$ 座標の絶対値の和の合計とします。

$T(N) = \sum_{R = 3}^N \sum_{r=1}^{\left\lfloor \frac{R - 1}{2} \right\rfloor} S(R, r)$ を、$R$ と $r$ が正の整数で $R \leq N$ かつ $2r < R$ の条件を満たすときの $S(R, r)$ の合計とします。

次の値が与えられています：

$$\begin{align}
  C(3, 1) = & \\{(3, 0), (-1, 2), (-1,0), (-1,-2)\\} \\\\
  C(2500, 1000) = & \\{(2500, 0), (772, 2376), (772, -2376), (516, 1792), (516, -1792), (500, 0), (68, 504), \\\\
                  &(68, -504),(-1356, 1088), (-1356, -1088), (-1500, 1000), (-1500, -1000)\\}
\end{align}$$

**注意：** $(-625, 0)$ は $C(2500, 1000)$ の要素ではありません。対応する $t$ の値に対して $\sin(t)$ が有理数ではないためです。

$S(3, 1) = (|3| + |0|) + (|-1| + |2|) + (|-1| + |0|) + (|-1| + |-2|) = 10$

$T(3) = 10$; $T(10) = 524$; $T(100) = 580\,442$; $T({10}^3) = 583\,108\,600$.

$T({10}^6)$ を求めてください。
