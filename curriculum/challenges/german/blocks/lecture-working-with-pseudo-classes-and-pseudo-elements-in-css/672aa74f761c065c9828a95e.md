# --interactive--

Pseudoklassen sind spezielle CSS-Schlüsselwörter, mit denen Sie ein Element basierend auf seinem bestimmten Zustand oder seiner Position auswählen können. Der Zustand oder die Position des Elements kann Folgendes umfassen:

- Wenn es aktiv ist.
- Wenn es von einer Maus überfahren wird.
- Wenn es das erste Kind eines Elternteils ist.
- Wenn es das letzte Kind eines Elternteils ist.
- Wenn ein Link besucht wurde.
- Wenn es deaktiviert ist.

Um eine Pseudoklasse zu verwenden, fügen Sie sie dem Selektor mit einem Doppelpunkt (`:`) gefolgt vom Namen der Pseudoklasse hinzu:

```css
selector:pseudo-class {
  /* CSS properties */
}
```

Sehen wir uns an, wie Sie eine Pseudoklasse verwenden können, um jeden der bereits erwähnten Zustände und Positionen darzustellen.

Die `:active`-Pseudoklasse ermöglicht es Ihnen, den aktiven Zustand eines Elements auszuwählen, zum Beispiel beim Klicken auf einen Button:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example Button</button>
```

```css
button:active {
  background: greenyellow;
}
```

:::

Die `:hover`-Pseudoklasse definiert den Hover-Zustand eines Elements. Ein Beispiel wäre das Überfahren eines Buttons oder Links mit der Maus:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="#">Hover over me!</a>  
```

```css
a:hover {
  text-decoration: none;
  color: white;
  background: crimson;
}
```

:::

Die `:first-child`-Pseudoklasse wählt ein Element aus, das das erste Kind seines Elternelements ist. Hier ist ein Beispiel, wie Sie das erste Absatz-Element in einem `div`-Container ansprechen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:first-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

Das erste Absatz-Element in diesem `div` erhält die `lightcoral`-Hintergrundfarbe und `padding` von `0.4rem` an allen vier Seiten.

Die `:last-child`-Pseudoklasse zielt auf das letzte Element ab, das Kind seines Elternelements ist. Hier ein Beispiel, wie Sie das letzte Absatz-Element im `div`-Container ansprechen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:last-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

Die `:visited`-Pseudoklasse ermöglicht es Ihnen, einen Link zu stylen, den der Benutzer bereits besucht hat:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="https://www.example.com" target="_blank">Visit Example.com</a>
```

```css
a:visited {
  color: purple;
}
```

:::

Die `:disabled`-Pseudoklasse erlaubt es Ihnen, ein interaktives Element im deaktivierten Zustand zu stylen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button disabled>Disabled Button</button>
```

```css
button:disabled {
  background-color: lightgray;  
}
```

:::

Wie Sie sehen, erlauben Pseudoklassen das Stylen von Elementen basierend auf Benutzerinteraktionen wie Hover oder besuchten Links. Dadurch wirkt Ihre Website interaktiver.

Neben den bereits erwähnten Pseudoklassen gibt es noch weitere wie:

- `:focus`
- `:first-of-type`
- `:last-of-type`
- `:nth-of-type`
- `:modal`
- `:enabled`
- `:checked`
- `:required` und mehr.
