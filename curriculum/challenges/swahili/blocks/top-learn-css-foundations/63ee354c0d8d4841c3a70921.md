# --description--

Njia ya ndani ya mstari ya `CSS` inaruhusu kuongeza mitindo moja kwa moja kwa vipengele vya `HTML`, ingawa njia hii haipendekezwi sana:

```html
<body>
  <div style="color: white; background-color: black;">...</div>
</body>
```

Kitu cha kwanza cha kuzingatia ni kwamba hakuna vichaguzi hapa, kwa sababu mitindo inaongezwa moja kwa moja kwenye lebo ya ufunguzi ya `<div>` yenyewe. Kisha, una sifa ya `style`, na thamani yake ndani ya nukta za nukuu ni tamko.

Ikiwa unahitaji kuongeza mtindo wa kipekee kwa kipengele kimoja tu, njia hii inaweza kufanya kazi vizuri. Kwa ujumla, hata hivyo, hii siyo njia inayopendekezwa kwa kuongeza CSS kwenye HTML kwa sababu chache:

Inaweza kuwa machafu haraka unapoanza kuongeza tamko nyingi kwa kipengele kimoja, na kusababisha faili yako ya HTML kuwa kubwa isiyo ya lazima.
Ikiwa unataka vipengele vingi kuwa na mtindo mmoja, utalazimika kunakili + kubandika mtindo huo kwa kila kipengele, na kusababisha kurudia nyingi zisizo za lazima na kuongeza ukubwa.
CSS yoyote ndani ya mstari itaghairi njia nyingine mbili, ambayo inaweza kusababisha matokeo yasiyotegemewa. (Ingawa hutachunguza hili hapa, hii inaweza kutumika kwa faida).
