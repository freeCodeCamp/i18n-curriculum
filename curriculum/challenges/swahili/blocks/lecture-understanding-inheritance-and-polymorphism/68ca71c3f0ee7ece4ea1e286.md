# --description--

Katika funzo la awali, ulijifunza kuhusu kuweka kiambishi awali cha underscore moja na underscore mbili kwenye sifa za kipengele.

Ili kukukumbusha tofauti kati yao, underscore moja ni mkataba unaomaanisha sifa ya kipengele ni kwa matumizi ya ndani ndani ya darasa la vitu na haipaswi kufikiliwa moja kwa moja kutoka nje ya darasa hilo. Underscore mbili, kwa upande mwingine, huzuia sifa hiyo kufikiliwa moja kwa moja kutoka nje ya darasa.

Hapa kuna mfano unaoonyesha jinsi hizi mbili zinavyofanya kazi:

```py
class Example:
    def __init__(self):
        self._internal = 'I can be accessed from outside the class, but should not'
        self.__private = 'You cannot access me directly from outside the class'

obj = Example()

print(obj._internal) # I can be accessed from outside the class, but should not
print(obj.__private)  # AttributeError: 'Example' object has no attribute '__private'
```

Kuweka kiambishi awali cha underscore mbili kwenye sifa ya kipengele husababisha mchakato wa name mangling wa Python, ambapo Python kwa siri hubadilisha jina la sifa hiyo kwa kuongeza underscore na jina la darasa la vitu kama kiambishi awali, ikibadilisha `__attribute` kuwa `_ClassName__attribute`.

Ili kuona hili likifanyika, unaunda mfano wa darasa la vitu na kutumia sifa maalum ya `__dict__` ya mfano huo, ambayo ni kamusi yenye sifa za kipengele za kitu hicho:

```py
class Example:
    def __init__(self, internal, private):
        self._internal = internal
        self.__private = private

example1 = Example(
    'I can be accessed from outside the class, but should not',
    'I cannot be accessed directly from outside the class'
)

print(example1.__dict__)
```

Matokeo yatakuwa:

```py
{
  '_internal': 'I can be accessed from outside the class, but should not',
  '_Example__private': 'I cannot be accessed directly from outside the class'
}
```

Kama unavyoona, sifa ya `__private` imehifadhiwa kama `_Example__private`. Hii ina maana bado unaweza kufikilia sifa hiyo kutoka nje ya darasa kwa njia hii:

```py
class Example:
    def __init__(self, internal, private):
        self._internal = internal
        self.__private = private

example1 = Example(
    'I can be accessed from outside the class, but should not',
    'I cannot be accessed directly from outside the class'
)
example2 = Example(
    'I should not be accessed from outside the class',
    'But I can be accessed from outside the class with name mangling'
)

print(example1._Example__private) # I cannot be accessed directly from outside the class
print(example2._Example__private) # But I can be accessed from outside the class with name mangling
```

Kwa nini basi Python hufanya name mangling?

Madhumuni kuu ya name mangling ni kuzuia kuingiliwa kwa bahati mbaya kwa sifa za kipengele na njia wakati unapotumia urithi. Hapa kuna mfano unaoeleweka wazi:

```py
class Parent:
    def __init__(self):
        self.__data = 'Parent data'

class Child(Parent):
    def __init__(self):
        super().__init__()
        self.__data = 'Child data'

c = Child()
print(c.__dict__) # {'_Parent__data': 'Parent data', '_Child__data': 'Child data'}
```

Unaweza kuona kuwa darasa la `Parent` na `Child` linalorithi kutoka kwake kila moja lina sifa zake za `_class__data` tofauti. Hii inawezekana kwa name mangling. Vinginevyo, `Child` ingekuwa imetandika data ya Parent kwa bahati mbaya.

Hii ndicho kingetokea kama Python isingeruhusu name mangling, yaani kama hutumii kiambishi awali cha underscore mbili kwenye sifa zote za madarasa yote mawili:

```py
class Parent:
   def __init__(self):
       self.data = 'Parent data'

class Child(Parent):
   def __init__(self):
       super().__init__()
       self.data = 'Child data'

c = Child()
print(c.__dict__)  # {'data': 'Child data'}
```

Basi, ni ipi unapaswa kutumia kuweka kiambishi awali cha sifa kati ya underscore moja (`_`) na underscore mbili (`__`)? Inategemea. Ikiwa sifa ni kwa matumizi ya ndani tu ndani ya darasa, tumia underscore moja.

Lakini ikiwa unafanya kazi na darasa litakalorithiwa, unapaswa kutumia underscore mbili ili sifa kutoka kwa mzazi isibadilishwe kwa bahati mbaya.
