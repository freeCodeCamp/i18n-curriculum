# --description--

在这个 `Task` 中，你将探索术语的隐喻用法以理解编程概念。隐喻是一种强大的方式，通过将复杂的概念与日常对象或动作联系起来，来形象化和解释复杂的想法。

`Container` 用作比喻，描述变量为可以保存和存储不同数据的东西，就像 `container` 可以保存不同的项一样。

动作 `fill and empty` 描述了你如何向变量添加或移除数据，类似于你如何 `fill` 一个水瓶或倒出水。

`Locked boxes` 描述常量为安全、不可更改的 `containers`，强调它们一旦设置就无法更改。

短语 `keep the same contents` 强调常量中的数据不会改变，就像一个被锁定的盒子，里面始终装着相同的东西。
