# --description--

パターンの中の正確な文字がわからなかったり、知る必要がなかったりする場合があります。例えば、スペルミスにマッチするすべての単語を考えるのは時間がかかります。幸いなことに、ワイルドカード文字を使えば時間を節約できます：`.`

ワイルドカード文字`.`は任意の1文字にマッチします。ワイルドカードは`dot`や`period`とも呼ばれます。ワイルドカード文字は正規表現の他の文字と同じように使えます。例えば、`hug`、`huh`、`hut`、`hum`にマッチさせたい場合は、正規表現`/hu./`を使えば4つの単語すべてにマッチします。

```js
let humStr = "I'll hum a song";
let hugStr = "Bear hug";
let huRegex = /hu./;
huRegex.test(humStr);
huRegex.test(hugStr);
```

これら2つの`test`の呼び出しはどちらも`true`を返します。

# --instructions--

正規表現`unRegex`を完成させて、文字列`run`、`sun`、`fun`、`pun`、`nun`、`bun`にマッチさせてください。正規表現にはワイルドカード文字を使う必要があります。
