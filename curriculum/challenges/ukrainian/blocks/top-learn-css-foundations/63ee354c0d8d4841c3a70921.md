# --description--

Рядковий `CSS` дає змогу додавати стилі безпосередньо до `HTML` елементів, хоча цей метод не є найкращим:

```html
<body>
  <div style="color: white; background-color: black;">...</div>
</body>
```

Перше, що варто зауважити, — тут немає селекторів, оскільки стилі додаються безпосередньо до відкриваючого `<div>` тегу. Далі йде атрибут `style`, значення якого в лапках — це оголошення.

Якщо потрібно додати унікальний стиль для одного елемента, цей метод цілком підходить. Проте загалом це не найкращий спосіб додавання CSS до HTML з кількох причин:

Це може швидко стати досить заплутаним, коли починаєте додавати багато оголошень до одного елемента, через що ваш HTML-файл стає зайво громіздким.
Якщо багато елементів мають мати однаковий стиль, доведеться копіювати й вставляти той самий стиль у кожен окремий елемент, що призводить до зайвого дублювання і ще більшої громіздкості.
Будь-який рядковий CSS заміщатиме два інші методи, що може спричинити несподівані результати. (Хоча тут ви не заглиблюватиметесь у це, насправді це можна використати на свою користь).
