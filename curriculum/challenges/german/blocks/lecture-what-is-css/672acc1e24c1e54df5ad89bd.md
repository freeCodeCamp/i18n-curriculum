# --interactive--

In HTML und CSS werden Elemente entweder als **Inline-Elemente** oder als **Block-Elemente** klassifiziert, und diese Klassifizierung bestimmt, wie sie sich im Dokumentlayout verhalten.

Dieses Verständnis ist entscheidend, um zu steuern, wie Ihr Inhalt auf einer Webseite angezeigt wird.

Block-Elemente sind Elemente, die standardmäßig die volle verfügbare Breite einnehmen und sich über die Breite ihres Containers erstrecken.

Diese Elemente beginnen immer auf einer neuen Zeile und schieben anderen Inhalt in die nächste Zeile, wodurch ein „Block“ von Inhalt entsteht.

Block-Elemente haben standardmäßig die CSS-Eigenschaft `display: block;` angewendet. Diese Eigenschaft sorgt dafür, dass das Element die Breite des Containers ausfüllt und auf einer neuen Zeile erscheint.

Einige häufige Block-Elemente sind `div`-Elemente, Absätze, Überschriften, geordnete Listen, ungeordnete Listen und Abschnittselemente.

Hier ist ein Codebeispiel für ein Block-Element:

:::interactive_editor

```html
<p style="border: 2px solid red;">
  First paragraph
</p>
<p>Second paragraph</p>
```

:::

In diesem Beispiel haben wir zwei Absatz-Elemente, wobei das erste einen roten Rahmen hat.

Die beiden Absatz-Elemente teilen sich nicht dieselbe Zeile, weil sie standardmäßig Block-Elemente sind.

Beide Absatz-Elemente nehmen also die volle Breite ihres Containers ein, der in diesem Fall das `body`-Element ist.

Block-Elemente sind ideal, wenn Sie Inhalte vertikal stapeln möchten, wie Absätze, Abschnitte oder größere Inhaltsblöcke. Sie werden häufig verwendet, um das Layout und die Struktur einer Webseite zu definieren.

Inline-Elemente hingegen nehmen nur so viel Breite ein, wie sie benötigen, und beginnen nicht auf einer neuen Zeile. Diese Elemente fließen innerhalb des Inhalts, sodass Text und andere Inline-Elemente neben ihnen erscheinen können.

Inline-Elemente haben standardmäßig die CSS-Eigenschaft `display: inline;` angewendet. Diese Eigenschaft sorgt dafür, dass das Element im Fluss des Inhalts bleibt und nicht auf eine neue Zeile umbricht.

Gängige Inline-Elemente sind `span`, `a` und `img` Elemente.

Hier ist ein Beispiel, um Inline-Elemente besser zu verstehen:

:::interactive_editor

```html
<p>This is a
  <span style="color: red; border: 2px solid green;">red</span>
  word inside a paragraph.
</p>
```

:::

In diesem Beispiel haben wir ein `span`-Element, das innerhalb eines Absatz-Elements verschachtelt ist. Das `span`-Element hat eine `red`-Textfarbe mit einem `green`-Rahmen, damit Sie das hervorgehobene Wort besser sehen können.

Wie Sie sehen, nimmt das `span`-Element nur so viel Platz ein wie das Wort „red“ und schiebt den folgenden Inhalt nicht auf eine neue Zeile.

Inline-Elemente eignen sich am besten, um kleinere Textabschnitte oder Inhalte innerhalb einer Zeile zu gestalten, wie zum Beispiel zur Hervorhebung eines Wortes, zum Erstellen von Hyperlinks oder zum Anwenden spezieller Stile auf Teile eines Absatzes.
