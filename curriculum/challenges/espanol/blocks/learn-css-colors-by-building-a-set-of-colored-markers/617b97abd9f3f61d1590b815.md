# --description--

En el modelo de color aditivo RGB, los <dfn>colores primarios</dfn> son colores que, cuando se combinan, crean un blanco puro. Pero para que esto ocurra, cada color necesita estar en su máxima intensidad.

Antes de combinar colores, establece tu marcador verde de nuevo al verde puro. Para la función `rgb` en la regla CSS `.two`, establece el verde de nuevo al valor máximo de `255`.

# --hints--

Tu regla CSS `.two` debe tener una propiedad `background-color` establecida en `rgb(0, 255, 0)`.

```js
assert(new __helpers.CSSHelp(document).getStyle('.two')?.backgroundColor === 'rgb(0, 255, 0)');
```
