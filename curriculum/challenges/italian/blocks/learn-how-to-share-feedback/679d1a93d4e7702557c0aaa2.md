# --explanation--

Nelle domande `Past Simple`, il verbo principale resta nella sua forma base perché l'ausiliare `did` indica già il passato. Per esempio:

`What time did you arrive?` (Non `did you arrived`)  

`Coding` come aggettivo si riferisce a cose legate alla scrittura di software. Un `coding issue` indica un problema con il codice che influisce sulla funzionalità di un programma. Per esempio:

`The development team is working on fixing a coding issue that caused system crashes.` - Questo significa che un problema nel codice stava influenzando il sistema.
