# --explanation--

在**間接引語**中，片語 `said that` 用來以不同的方式分享某人所說的話，而不是使用他們的原話。字 `that` 是選用的，且在日常語言中常常省略。例如：

- 直接引語：`We need to update the software.`

- 轉述語氣：`He said that we need to update the software.` 或 `He said we need to update the software.`

兩個版本都是正確的，且意思相同。
