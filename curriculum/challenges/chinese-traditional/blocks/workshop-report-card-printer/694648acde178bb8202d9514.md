# --description--

現在你需要將學生的年齡添加到成績單。為此你將使用整數，這是 Python 中的數值資料型別之一。

宣告一個名為 `age` 的變數並指定其整數值為 `20`。

然後，將 `age` 的值和資料型別以逗號分隔一起列印。檢查終端機中的輸出，該輸出會顯示 `age` 的值及其型別為 `int`（整數）。
