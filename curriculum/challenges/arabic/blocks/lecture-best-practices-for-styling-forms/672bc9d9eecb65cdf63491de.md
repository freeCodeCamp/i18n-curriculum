# --interactive--

دعنا نتعلم متى تستخدم `appearance: none` لمعالجة مشاكل التنسيق في مُدخَلات البحث، مربعات الاختيار، وأزرار الاختيار.

تُطبّق المتصفحات تنسيقًا افتراضيًا على الكثير من العناصر. في حالة عناصر المُدخَل، قد تشعر بأن قدرتك على تنسيقها باستخدام CSS محدودة إلى حد ما. لذلك، قد ترغب في استخدام `appearance: none` لإخفاء جوانب من العنصر الافتراضي وبناء تنسيقك الخاص بدلاً منه. على سبيل المثال، هذا سيخفي مربعات الاختيار الافتراضية لعناصر مربع الاختيار، مما يسمح لك باستخدام مؤشرات مخصصة مثل علامة صح خضراء وعلامة إكس حمراء لإظهار الحالة. بالنسبة لمُدخَل البحث، تعرض المتصفحات المبنية على WebKit أيقونة بحث افتراضية وزر إلغاء. إخفاء هذه العناصر يسمح لك بإنشاء مؤشرات خاصة بك تظهر على جميع المتصفحات.

إليك مثال على مربع اختيار مخصص:

**ملاحظة**: بعض خصائص CSS في هذا المثال لم تُغطَّ بعد. هذا فقط ليعطيك فكرة عن كيفية إنشاء مربع اختيار مخصص. ستتعلم كيف يعمل كل هذا في الدروس وورش العمل القادمة.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form>
  <label>
    <input class="checkbox" type="checkbox" /> Agree
  </label>
</form>
```

```css
.checkbox {
  appearance: none;
  width: 18px;
  height: 18px;
  border: 2px solid #ccc;
  border-radius: 4px;
  display: inline-block;
  position: relative;
  cursor: pointer;
  transition: all 0.25s ease;
  vertical-align: middle; 
}

.checkbox:hover {
  border-color: #888;
}

.checkbox:checked {
  background-color: #4caf50;
  border-color: #4caf50;
}

.checkbox:checked::after {
  content: "";
  position: absolute;
  left: 4px;
  top: 0px;
  width: 5px;
  height: 10px;
  border: solid white;
  border-width: 0 2px 2px 0;
  transform: rotate(45deg);
}

.checkbox:focus {
  outline: 2px solid #90caf9;
  outline-offset: 2px;
}
```

:::

WebKit هو محرك برمجي يساعد متصفحات الويب على عرض المواقع. تستخدم متصفحات مثل Safari محرك WebKit لضمان ظهور صفحات الويب وعملها بشكل صحيح. خاصية CSS `appearance: none` تمنحك تحكمًا كاملاً في التنسيق، لكنها تأتي مع بعض الأمور التي يجب الانتباه لها. المكونات التفاعلية الافتراضية لعناصر المُدخَل تشمل ميزات مثل مؤشرات التركيز والأخطاء التي يجب التأكد من عدم فقدانها.

إنشاء تنسيق متناسق عبر المنصات المختلفة سبب ممتاز لاستخدام هذه الخاصية. يمكنك أيضًا استخدامها لضمان أن تكون أهداف النقر على الأجهزة المحمولة كبيرة بما يكفي أو أن ألوان مربع الاختيار تتمتع بتباين كافٍ.
