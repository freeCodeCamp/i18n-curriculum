# --interactive--

I loop nella programmazione servono a ripetere un blocco di codice più volte.

Un esempio di loop è quando progetti un programma che deve stampare una lista di elementi. Puoi usare un loop per stampare ciascuno degli elementi della lista.

Un altro esempio è quando progetti un gioco e vuoi muovere un personaggio sullo schermo. Puoi usare un loop per spostare il personaggio di un certo numero di pixel ogni volta che il loop viene eseguito.

In JavaScript, ci sono diversi tipi di loop che puoi usare. In questa lezione, copriremo il loop `for`. Ecco la sintassi base per un loop `for`:

```js
for (initialization; condition; increment or decrement) {
  // code block to be executed
}
```

L’istruzione di inizializzazione viene eseguita prima che il loop inizi. Viene tipicamente usata per inizializzare una variabile contatore. Una variabile contatore è una variabile usata per tenere traccia di quante volte il loop è stato eseguito.

L’istruzione condizionale viene valutata prima di ogni iterazione del loop. Un’iterazione è un singolo passaggio attraverso il loop.

Se la condizione è vera, il blocco di codice all’interno del loop viene eseguito. Se la condizione è falsa, il loop si ferma e si passa al blocco di codice successivo.

L’ultima parte del loop è l’istruzione di incremento/decremento. Questa istruzione viene eseguita dopo ogni iterazione del loop. Viene tipicamente usata per incrementare o decrementare la variabile contatore.

:::interactive_editor

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

:::

Nella prima parte dell’esempio sopra, inizializziamo una variabile contatore `i` a `0`. È una convenzione comune usare `i` come variabile contatore in un loop `for`.

La parte successiva è controllare la condizione. In questo caso, la condizione verifica se `i` è minore di `5`. Poiché `i` è `0`, la condizione è vera e il blocco di codice all’interno del loop viene eseguito.

Il blocco di codice all’interno del loop serve a stampare il valore di `i` nella console. Il valore di `i` è `0`, quindi la console mostrerà il valore di `0`.

Poi viene eseguita l’istruzione di incremento. In questo caso, stiamo incrementando `i` di `1`. Quindi `i` è ora `1`.

Poi controlliamo di nuovo la condizione, che verifica se `i` è minore di `5`. Poiché `i` è ora `1`, la condizione è ancora vera e il blocco di codice all’interno del loop viene eseguito di nuovo.

Continuiamo a ripetere questo processo finché la condizione non diventa falsa. In questo caso, quando `i` è `5`, la condizione è falsa e il loop si ferma.

Quando lavori con i loop devi fare attenzione a non creare una condizione che sia sempre vera. Se succede, il loop girerà all’infinito e il tuo programma andrà in crash. Questo è noto come loop infinito.

È possibile creare loop `for` annidati. Un loop annidato è quando metti un loop dentro un altro. Vedremo esempi di quando potresti voler fare questo più avanti.

I loop possono essere molto utili nella programmazione quando devi ripetere un blocco di codice un certo numero di volte. Anche se lavorare con i loop `for` può sembrare complicato all’inizio, con la pratica prenderai la mano.
