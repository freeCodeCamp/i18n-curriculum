Sarah 正在询问 Mark 是否注意到有些计算机无法 `booting properly` 的问题。

`Booting` 是通过装载操作系统来启动计算机。例如：

- `Have you noticed the network is running slowly?` - 这是在询问某人是否观察到问题。

- `The laptop isn't booting properly; we need to check the operating system。` - 这突出了计算机无法正确启动的常见问题。
