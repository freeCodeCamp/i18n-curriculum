# --description--

Для $n$-кортежу цілих чисел $t = (a_1, \ldots, a_n)$ нехай $(x_1, \ldots, x_n)$ будуть розв’язками поліноміального рівняння $x^n + a_1x^{n - 1} + a_2x^{n - 2} + \ldots + a_{n - 1}x + a_n = 0$.

Розглянемо такі дві умови:

- $x_1, \ldots, x_n$ усі є дійсними числами.
- Якщо $x_1, ..., x_n$ відсортовані, то $⌊x_i⌋ = i$ для $1 ≤ i ≤ n$. ($⌊·⌋:$ функція підлоги.)

У випадку $n = 4$ існує 12 $n$-кортежів цілих чисел, які задовольняють обидві умови.

Визначимо $S(t)$ як суму абсолютних значень цілих чисел у $t$.

Для $n = 4$ можна перевірити, що $\sum S(t) = 2087$ для всіх $n$-кортежів $t$, які задовольняють обидві умови.

Знайдіть $\sum S(t)$ для $n = 7$.
