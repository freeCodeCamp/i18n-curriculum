# --description--

Considera el conjunto $S(r)$ de puntos ($x$,$y$) con coordenadas enteras que cumplen $|x| + |y| ≤ r$.

Sea $O$ el punto (0,0) y $C$ el punto ($\frac{r}{4}$,$\frac{r}{4}$).

Sea $N(r)$ el número de puntos $B$ en $S(r)$, tal que el triángulo $OBC$ tiene un ángulo obtuso, es decir, el ángulo más grande $α$ satisface $90°<α<180°$.

Entonces, por ejemplo, $N(4)=24$ y $N(8)=100$.

¿Qué es $N(1\\,000\\,000\\,000)$?
