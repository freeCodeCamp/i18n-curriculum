# --description--

Зазвичай `with` використовується, щоб показати зв’язок або використання. Наприклад, `I write with a pen.` Проте в деяких реченнях `with` стоїть після слова, з яким його використовують. Це зазвичай трапляється, коли хочуть підкреслити об’єкт або річ, якою хтось користується або з якою працює, особливо в питаннях або твердженнях про технології чи роботу.

Наприклад, у техніці можна сказати `This program I work with is challenging.` Тут `with` використовується в кінці, щоб підкреслити програму, з якою складно працювати. Іншим прикладом може бути питання на кшталт `What else is he working with?`, де `with` в кінці запитує про інші інструменти чи технології, які використовують.
