# --description--

세 개의 방이 자동문으로 서로 연결되어 있습니다.

<img alt="자동문으로 서로 연결된 세 개의 방" src="https://cdn.freecodecamp.org/curriculum/project-euler/rooms-of-doom.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

각 문은 보안 카드로 작동합니다. 방에 들어가면 문이 자동으로 닫히고, 그 보안 카드는 다시 사용할 수 없습니다. 시작 시 기계가 무제한으로 카드를 배급하지만, 각 방(시작 방 포함)에는 스캐너가 있습니다. 스캐너가 세 장 이상의 보안 카드를 소지했거나 바닥에 방치된 보안 카드를 감지하면 모든 문이 영구적으로 잠깁니다. 하지만 각 방에는 나중에 사용할 보안 카드를 안전하게 보관할 수 있는 상자가 있습니다.

만약 단순히 방을 한 번에 하나씩 지나가려고 하면, 3번 방에 들어갈 때 세 장의 카드를 모두 사용해 그 방에 영원히 갇히게 됩니다!

하지만 보관 상자를 이용하면 탈출이 가능합니다. 예를 들어, 첫 번째 카드를 사용해 1번 방에 들어가 한 장을 상자에 넣고, 세 번째 카드를 사용해 시작 지점으로 돌아갈 수 있습니다. 그런 다음 배급기에서 세 장의 카드를 더 받아 한 장을 사용해 1번 방에 들어가 방금 상자에 넣은 카드를 꺼낼 수 있습니다. 이제 다시 세 장의 카드를 가지고 남은 세 개의 문을 통과할 수 있습니다. 이 방법으로 총 여섯 장의 보안 카드로 세 개의 방을 모두 통과할 수 있습니다.

최대 3장의 카드를 소지하면서 총 123장의 카드로 여섯 개의 방을 통과할 수 있습니다.

$C$를 한 번에 소지할 수 있는 카드의 최댓값이라고 합시다.

$R$을 통과해야 할 방의 수라고 합시다.

$M(C, R)$을 한 번에 최대 $C$장의 카드를 소지하면서 $R$개의 방을 통과하는 데 필요한 배급기에서의 카드 최솟값이라고 합시다.

예를 들어, $M(3, 6) = 123$이고 $M(4, 6) = 23$입니다.

그리고 $3 ≤ C ≤ 4$일 때, $\sum M(C, 6) = 146$입니다.

$3 ≤ C ≤ 10$일 때, $\sum M(C, 10) = 10382$임이 주어졌습니다.

$3 ≤ C ≤ 40$일 때, $\sum M(C, 30)$을 구하세요.
