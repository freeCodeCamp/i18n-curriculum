# --explanation--

En preguntas con etiquetas como `wouldn't they?`, el hablante típicamente busca confirmación o acuerdo. Una respuesta como `That's right` proporciona una afirmación clara y mantiene la conversación alineada.

Otros ejemplos de confirmar respuestas:

- `Yes, exactly.`

- `Absolutely, you're correct.`

- `I agree completely.`

Estas respuestas reconocen y validan la afirmación del hablante.
