# --description--

Vamos avançar para os números na casa dos 60.

Em chinês, o número 60 é chamado `六十 (liù shí)`.

Os números de 61 a 69 são formados adicionando o chinês para o dígito das unidades (1–9) após `六十 (liù shí)` (60).

| Número | Chinês |
| :--- | :--- |
| 60 | `六十 (liù shí)` |
| 61 | `六十一 (liù shí yī)` |
| 62 | `六十二 (liù shí èr)` |
| 63 | `六十三 (liù shí sān)` |
| 64 | `六十四 (liù shí sì)` |
| 65 | `六十五 (liù shí wǔ)` |
| 66 | `六十六 (liù shí liù)` |
| 67 | `六十七 (liù shí qī)` |
| 68 | `六十八 (liù shí bā)` |
| 69 | `六十九 (liù shí jiǔ)` |
