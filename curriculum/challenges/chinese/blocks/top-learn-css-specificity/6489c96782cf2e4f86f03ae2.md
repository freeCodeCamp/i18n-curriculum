# --description--

更具体的 CSS 声明将优先于不那么具体的声明。内联样式（你在之前的课程中已经学习过）相比选择器具有最高的特异性，而每种类型的选择器都有其自身的特异性级别，这些级别共同决定了声明的具体程度。其他选择器也会影响特异性，但你目前只关注已经学习过的那些：

1. ID 选择器（最具体的选择器）
2. 类选择器
3. 类型选择器

只有当一个元素有多个相互冲突的声明针对它时，特异性才会被考虑，类似于一个决胜负的规则。`ID` 选择器总是胜过任意数量的 `class` 选择器，`class` 选择器总是胜过任意数量的 `type` 选择器，而 `type` 选择器总是胜过任意数量比它特异性更低的选择器。当没有声明的选择器具有更高的特异性时，单个选择器的较大量将胜过同一选择器的较小量。
