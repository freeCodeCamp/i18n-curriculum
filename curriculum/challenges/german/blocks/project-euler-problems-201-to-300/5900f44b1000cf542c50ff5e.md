# --description--

Nennen wir ein ganzzahliges Dreieck mit den Seitenlängen $a ≤ b ≤ c$ knapp spitzwinklig, wenn die Seiten die Gleichung $a^2 + b^2 = c^2 + 1$ erfüllen.

Wie viele knapp spitzwinklige Dreiecke gibt es mit einem Umfang von höchstens $25\\,000\\,000$?
