## Python en tu entorno local

- **Instalación**: La mejor manera de instalar Python en Windows, Mac y Linux es descargar el instalador desde el sitio web oficial de Python (`https://www.python.org/`).
- **Terminal**: Una interfaz basada en texto que te permite interactuar con tu computadora escribiendo comandos. Cada sistema operativo incluye una app de terminal predeterminada. En macOS, puedes usar la app Terminal. En Windows, puedes usar Command Prompt o PowerShell. En Linux, cada entorno de escritorio tiene su propia app de terminal predeterminada, como GNOME Terminal o Konsole.
- **IDE**: IDE significa entorno de desarrollo integrado y cuenta con funcionalidades como un editor de código, herramientas de prueba, una terminal y más.
- **Editores de código e IDEs populares para Python**: VS Code, PyCharm y Spyder
- **Ejecutar código localmente**: Para ejecutar código Python, tienes varias opciones. La primera es hacer clic en el botón de ejecutar en VS Code, que parece un botón de reproducir ubicado en la esquina superior derecha. Eso abrirá una terminal y ejecutará tu script de Python allí.

Otra opción es abrir una terminal y ejecutar el programa manualmente. Aquí tienes un ejemplo para ejecutar un programa llamado `main.py`:

```bash
python main.py
```

Debes ejecutar este comando desde la carpeta donde está guardado `main.py`. Por ejemplo, si `main.py` está dentro de una carpeta llamada `python-projects`, primero usa lo siguiente en la terminal:

```bash
cd python-projects
```

Luego ejecuta:

```bash
python main.py
```

## Usar el intérprete interactivo de Python

- **Definición**: Un intérprete interactivo es un programa que te permite escribir comandos uno a la vez y ver los resultados. Abre una app de terminal y escribe `python` y presiona `Enter`. Esto iniciará un intérprete interactivo de Python. El símbolo `>>>` significa que Python está esperando que escribas un comando. Prueba imprimir `"Hello, world!"` en la terminal:

```bash
print("Hello, world!")
```

Deberías ver que el texto aparece así:

```bash
>>> print("Hello, world!")
Hello, world!
```

Después de que el texto se imprime, Python vuelve a mostrar lo siguiente:

```bash
>>>
```

Esto significa que puedes escribir otro comando.

- **Ciclo leer, evaluar, imprimir, repetir (REPL)**: Cada vez que escribes comandos, el intérprete lee la entrada, la evalúa, imprime el resultado y vuelve a mostrar el `>>>` para que puedas escribir más comandos.
