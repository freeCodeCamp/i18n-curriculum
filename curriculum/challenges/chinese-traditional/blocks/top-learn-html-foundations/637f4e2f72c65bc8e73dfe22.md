# --description--

`<head>` 元素是你放置關於你的網頁的重要超-資訊，以及網頁在瀏覽器中正確呈現所需的內容的地方。在 `<head>` 內，你不應該使用任何會在網頁上顯示內容的元素。

## 字元集 Meta 元素
你應該在 `head` 元素中，為網頁的 `charset` 編碼始終包含 `meta` 頁籤：`<meta charset="utf-8">`。

設定編碼非常重要，因為它確保網頁能在瀏覽器中正確顯示來自不同語言的特殊符號和字元。

## 標題元素
你應該始終在 HTML 文件的 head 中包含的另一個元素是 `title` 元素：

```html
<title>My First Webpage</title>
```

`title` 元素用於為網頁提供一個人類可讀的標題，該標題會顯示在你網頁的瀏覽器分頁中。

如果你沒有包含 `title` 元素，網頁的標題將會預設為其檔案名稱。在你的情況下，那會是 `index.html`，這對使用者來說意義不大；如果使用者開啟許多瀏覽器分頁，這會讓他們很難找到你的網頁。

在 HTML `head` 中可以放入更多元素。然而，目前你只需要知道這裡涵蓋的兩個元素。你將在課程的其餘部分介紹更多放入 `head` 的元素。

回到 `index.html` 檔案，讓我們添加一個包含 `charset` `meta` 元素和 `title` 的 `head` 元素。`head` 元素位於 HTML 元素內，且應該永遠是開啟 `<html>` 標籤下的第一個元素：


```html
<!DOCTYPE html>

<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Webpage</title>
  </head>
</html>
```
