# --description--

47を取り、逆にして足すと、47 + 74 = 121となり、これは回文です。

すべての数字がすぐに回文を作るわけではありません。例えば、

<div style="margin-left: 4em;">
  349 + 943 = 1292,<br>
  1292 + 2921 = 4213<br>
  4213 + 3124 = 7337<br>
</div>

つまり、349は回文にたどり着くまでに3回の繰り返しが必要でした。

まだ証明されていませんが、196のように回文を作らない数字があると考えられています。逆にして足す過程で回文を一度も作らない数字をライクリル数と呼びます。これらの数字は理論的な性質を持つため、この問題では、証明されるまでは数字はライクリル数であると仮定します。さらに、1万未満のすべての数字については、(i) 50回未満の繰り返しで回文になるか、(ii) これまでのすべての計算能力をもってしても回文にたどり着けていないかのどちらかです。実際、10677は回文を作るまでに50回以上の繰り返しが必要であることが示された最初の数字で、4668731596684224866951378664（53回の繰り返し、28桁）です。

驚くことに、回文でありながらライクリル数である数字も存在し、最初の例は4994です。

`num`未満のライクリル数はいくつありますか？

**注意:** ライクリル数の理論的な性質を強調するため、2007年4月24日に文言が少し修正されました。

```js
const countLychrelNumbers = (size) => {
  const numReverse = (num) => {
    return Number(num.toString().split('').reverse().join(''));
  };
  const isPalin = (num) => {
    if (numReverse(num) === num) {
      return true;
    }
    return false;
  };
  let total = 0;
  for (let i = 1; i < size; i++) {
    let loopCount = 1;
    let sum = i;
    while (loopCount < 50) {
      sum = sum + numReverse(sum);
      if (isPalin(sum)) {
        break;
      } else {
        loopCount++;
      }
    }
    if (loopCount === 50) {
      total++;
    }
  }
  return total;
}
```
