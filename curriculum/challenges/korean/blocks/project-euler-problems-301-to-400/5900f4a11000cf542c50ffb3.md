# --description--

프랙트란(Fractran) 프로그래밍 언어로 작성된 프로그램은 분수 목록으로 구성됩니다.

프랙트란 가상 머신의 내부 상태는 양의 정수이며, 초기에는 시드 값으로 설정됩니다. 프랙트란 프로그램의 각 반복은 상태 정수에 목록에서 정수를 유지하게 하는 첫 번째 분수를 곱합니다.

예를 들어, John Horton Conway가 소수 생성을 위해 작성한 프랙트란 프로그램 중 하나는 다음 14개의 분수로 이루어져 있습니다:

$$\frac{17}{91}, \frac{78}{85}, \frac{19}{51}, \frac{23}{38}, \frac{29}{33}, \frac{77}{29}, \frac{95}{23}, \frac{77}{19}, \frac{1}{17}, \frac{11}{13}, \frac{13}{11}, \frac{15}{2}, \frac{1}{7}, \frac{55}{1}$$

시드 정수 2부터 시작하여 프로그램의 연속 반복은 다음 수열을 생성합니다:

$$15, 825, 725, 1925, 2275, 425, \ldots, 68, \mathbf{4}, 30, \ldots, 136, \mathbf{8}, 60, \ldots, 544, \mathbf{32}, 240, \ldots$$

이 수열에 나타나는 2의 거듭제곱은 $2^2, 2^3, 2^5, \ldots$입니다.

이 수열의 모든 2의 거듭제곱 지수는 소수이며, 모든 소수가 올바른 순서로 2의 거듭제곱 지수로 나타난다는 것을 증명할 수 있습니다!

위 프랙트란 프로그램을 사용해 프로젝트 오일러 문제 7 (${10001}^{\text{st}}$ 소수를 찾는 문제)을 해결할 때, 프로그램이 $2^{10001^{\text{st}}\text{ 소수}}$를 생성할 때까지 몇 번의 반복이 필요할까요?
