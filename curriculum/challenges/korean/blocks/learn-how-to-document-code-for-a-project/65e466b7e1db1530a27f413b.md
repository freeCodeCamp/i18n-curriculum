# --description--

컴퓨터 파일 시스템에서 `root directory`는 최상위 디렉터리를 의미합니다. 이는 모든 다른 폴더와 파일을 포함하는 서류 캐비닛의 가장 윗서랍과 같습니다.

`main folder`는 프로젝트나 파일 집합이 저장되는 기본 폴더입니다.

`subfolder`는 최상위 폴더 안에 위치한 작은 서랍과 같으며, 다른 폴더 안에 있습니다.

예를 들어, `MyProject`라는 프로젝트에서 `root directory`는 `MyProject` 폴더이고, 그 안에 있는 `Images` 같은 폴더는 `subfolder`입니다.

# --questions--

## --text--

프로젝트 맥락에서 `root directory`는 무엇인가요?

## --answers--

주요 폴더

---

하위 폴더

### --feedback--

하위 폴더는 다른 폴더 안에 있습니다. `root directory`는 하위 폴더가 아니라 주요 폴더입니다.

---

파일 유형

### --feedback--

`root directory`는 파일 유형이 아니라 폴더입니다. 프로젝트의 파일과 폴더가 저장되는 기본 장소입니다.

---

프로그래밍 언어

### --feedback--

`root directory`는 프로그래밍 언어가 아니라 파일 시스템 개념입니다. 프로젝트의 주요 폴더를 의미합니다.
