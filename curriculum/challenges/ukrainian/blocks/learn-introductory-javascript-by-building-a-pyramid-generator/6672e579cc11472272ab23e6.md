# --description--


JavaScript — це мова програмування, яка керує вебом. На відміну від HTML і CSS, які ви вивчали раніше, JavaScript найчастіше використовується для написання логіки, а не розмітки.

У цьому проєкті ви вивчите основи JavaScript і застосуєте ці концепції для створення генератора піраміди.

Генератор піраміди — це програма, де ви можете задати тип символу, кількість для піраміди та напрямок піраміди. Програма потім згенерує піраміду на основі цих вхідних даних.

Натисніть на кнопку «Перевірити свій код», щоб перейти до наступного кроку і почати проєкт.
