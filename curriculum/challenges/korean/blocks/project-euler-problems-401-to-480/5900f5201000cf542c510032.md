# --description--

피보나치 수 $\\{f_n, n ≥ 0\\}$는 $f_n = f_{n - 1} + f_{n - 2}$로 재귀적으로 정의되며, 초기값은 $f_0 = 0$과 $f_1 = 1$입니다.

다항식 $\\{F_n, n ≥ 0\\}$를 $F_n(x) = \displaystyle\sum_{i = 0}^n f_ix^i$로 정의하세요.

예를 들어, $F_7(x) = x + x^2 + 2x^3 + 3x^4 + 5x^5 + 8x^6 + 13x^7$이고, $F_7(11) = 268\\,357\\,683$입니다.

$n = {10}^{15}$일 때, $\displaystyle\sum_{x = 0}^{100} F_n(x)$의 합을 구하고, 답을 $1\\,307\\,674\\,368\\,000 \\, (= 15!)$로 나눈 나머지를 구하세요.
