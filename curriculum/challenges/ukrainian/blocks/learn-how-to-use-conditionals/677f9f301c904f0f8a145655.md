### --feedback--

# --explanation--

`software crash` трапляється, коли програма раптово перестає працювати, часто через помилки, баги або несумісність. Наприклад:

- `A bug in the code led to the software crash.` — помилка в програмі, через яку вона перестала функціонувати.

- `We need to fix the issue that caused the software crash.` — нам потрібно вирішити проблему, через яку програма перестала працювати.
