# --description--

In questo laboratorio costruirai una `rangeOfNumbers` funzione che genera un array di numeri all'interno di un intervallo specificato.

Esempi:

- `rangeOfNumbers(3, 9)` restituisce `[3, 4, 5, 6, 7, 8, 9]`
- `rangeOfNumbers(5, 5)` restituisce `[5]`

Requisiti:

- Usa la ricorsione (la funzione deve chiamare se stessa)
- Niente loop o metodi di array (`for`, `while`, `Array.from`, `forEach`, `map`, `filter`, `reduce`)

**Obiettivo:** Soddisfa le storie utente qui sotto e fai superare tutti i test per completare il laboratorio.

**Storie utente:**

1. Devi creare una funzione chiamata `rangeOfNumbers` che prende due parametri: `startNum` e `endNum`.
1. La funzione deve restituire un array di numeri interi che inizia con il numero rappresentato dal parametro `startNum` e termina con il numero rappresentato dal parametro `endNum` (incluso).
1. `startNum` sarà sempre minore o uguale a `endNum`.
1. La tua funzione deve usare la ricorsione chiamando se stessa. Non deve usare alcuna sintassi di loop (`for`, `while`, `Array.from()` o funzioni di ordine superiore come `forEach`, `map`, `filter` o `reduce`).
1. La funzione deve gestire il caso base in cui `startNum` è uguale a `endNum` restituendo un array contenente solo quel singolo numero.
1. Per il caso ricorsivo, la funzione deve chiamare se stessa con parametri modificati per costruire l'array, quindi aggiungere il numero corrente al risultato.
1. La funzione non deve fare affidamento su variabili globali per memorizzare o costruire l'array.

```js
function rangeOfNumbers(startNum, endNum) {
  if (endNum - startNum === 0) {
    return [startNum];
  } else {
    const numbers = rangeOfNumbers(startNum, endNum - 1);
    numbers.push(endNum);
    return numbers;
  }
}
```
