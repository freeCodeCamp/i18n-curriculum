# --description--

在西班牙语中，有两种常见的表达来源的方式。

第一种是使用 **`Es de` + 国家**。例如：

`Es de Venezuela` - 他来自委内瑞拉。

第二种是使用国籍形容词。例如：

`Es venezolano` - 他是委内瑞拉人。

国籍词不大写。

# --explanation--

Esteban 说了 `Sí, es brasileño`，意思是“是的，他是巴西人”（男性形式）。

在单词 `i` 的字母 `Sí` 上加上重音符号非常重要。带重音符号时，这个词的意思是“是”。没有重音符号时，根据上下文，这个词可以表示条件式“如果”。

记住，西班牙语中的国籍词不大写。
