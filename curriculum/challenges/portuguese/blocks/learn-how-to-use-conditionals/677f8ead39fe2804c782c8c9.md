# --explanation--

Sarah está perguntando se Mark percebeu um problema com alguns computadores que não estão `booting properly`.

`Have you noticed` mostra que Sarah está verificando a consciência de Mark sobre o problema, não pedindo para ele consertá-lo ou realizar qualquer ação.

`Booting` significa iniciar um computador carregando seu sistema operacional. Por exemplo:

- `Have you noticed the network is running slowly?` - Isso pergunta se alguém observou um problema.

- `The laptop isn't booting properly; we need to check the operating system.` - Isso destaca um problema comum onde um computador não consegue iniciar corretamente.
