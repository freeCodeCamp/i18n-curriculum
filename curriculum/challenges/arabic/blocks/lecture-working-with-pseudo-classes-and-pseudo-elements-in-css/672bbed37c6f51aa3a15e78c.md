# --interactive--

واحدة من أكثر الجوانب إثارة في CSS هي استخدام العناصر المزيفة. في هذا السياق، تعني كلمة "pseudo" "غير حقيقي"، لذا فإن العناصر المزيفة هي عناصر افتراضية أو تركيبية لا تطابق مباشرة أي عناصر HTML فعلية. تسمح لك بتنسيق أجزاء محددة من العنصر أو إدخال محتوى دون إضافة HTML إضافي.

لتطبيق عنصر مزيف، اربطه بمحدد العنصر الأصلي باستخدام نقطتين رأسيتين (`::`). لاحظ أن المحدد يمكن أن يكون أي نوع، مثل محدد صنف أو معرف. إليك كيف تبدو الصيغة النحوية الأساسية للعناصر المزيفة:

```css
selector::pseudo-element {
  property: value;
}
```

النقطتان الرأسيتان هما ما يميزان العناصر المزيفة عن الفئات المزيفة (pseudo-classes)، التي تستخدم نقطة رأسية واحدة.

تسمح العناصر المزيفة بتنسيق أجزاء محددة من محتوى العنصر أو إدخال محتوى قبله أو بعده، لكنها لا يمكن أن توجد بشكل مستقل. العنصر الذي يُرفق به العنصر المزيف يُسمى العنصر المنشأ.

لنبدأ بالنظر إلى أمثلة على العناصر المزيفة `::before` و `::after`. كما توحي أسماؤها، يسمح لك `::before` بإدخال محتوى قبل محتوى العنصر مباشرة، بينما يسمح لك `::after` بإدخال محتوى بعده.

إليك مثالًا على عنصر زر. في CSS، سنستخدم التمركز المطلق والعنصر المزيف `::before` لإضافة نجمة قبل نص الزر `Learn More`. ستتعلم المزيد عن التمركز المطلق في دروس لاحقة.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: lightseagreen;
  color: white;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
  position: relative;
}

.cta-button::before {
  content: "⭐";
  position: absolute;
  left: 3px;
  top: 8px;
  font-size: 0.75rem;
}
```

:::

تُستخدم خاصية `content` لتمثيل المحتوى الذي ترغب في إضافته قبل نص الزر. في هذا المثال، نضيف نجمة. ستلاحظ أنه يمكنك ليس فقط إدخال المحتوى بل وتنسيقه أيضًا. إليك مثالًا على العنصر المزيف `::after` مع نفس الزر:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: '➡️';
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}
```

:::

في العنصر المزيف `::after`، تُستخدم خاصية `transition` لتحريك التغييرات خلال 0.3 ثانية مع تأثير تدرج، مما يخلق تحولًا سلسًا وتدريجيًا بدلاً من تحول مفاجئ. ستتعلم المزيد عن خاصية `transition` في دروس لاحقة.

يمكنك أيضًا إرفاق فئة مزيفة (pseudo-class) بالمحتوى الذي تدخله داخل محتوى آخر باستخدام العناصر المزيفة `::before` و `::after`. على سبيل المثال، حالة التمرير فوق المحتوى. إليك مثالًا:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: "➡️";
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}

.cta-button:hover::after {
  transform: translateX(2px);
}
```

:::

مع `transform: translateX(2px)` في حالة التمرير فوق، يتم دفع المحتوى إلى اليمين بمقدار `2px` في كل مرة يمرر المستخدم فوق الزر. خاصية الانتقال في `::after` نفسها تضمن أن تستغرق العملية `0.3s`.

هذا ما تفعله خاصية `transform` – تسمح لك بتدوير أو إمالة أو تغيير مقياس أو تحريك العنصر في اتجاه معين. ستتعلم المزيد عن ذلك في دروس لاحقة.

في المثال التالي، سننظر إلى العنصر المزيف `::first-letter`. يستهدف العنصر المزيف `::first-letter` الحرف الأول من محتوى العنصر، مما يسمح لك بتنسيقه. إليك مثالًا على نص فقرة. إذا أردنا تنسيق الحرف الأول، يمكننا استخدام العنصر المزيف `::first-letter` هكذا:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>freeCodeCamp lets you learn to code without having to pay.</p>
```

```css
p::first-letter {
  font-size: 4rem;
}
```

:::

في المثال الأخير، سننظر إلى العنصر المزيف `::marker`، الذي يسمح لك باختيار العلامة أو الرصاصة أو الترقيم لعناصر القائمة لتنسيقها. يوفر العنصر المزيف `::marker` طريقة لتعزيز هوية علامتك التجارية على الموقع من خلال تخصيص علامات القائمة لتتناسب مع نظام الألوان الخاص بك.

إليك مثالًا على قائمة غير مرتبة وقائمة مرتبة. لتغيير لون وحجم علامة عنصر القائمة، يمكنك استخدام العنصر المزيف `::marker` هكذا:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<ul>
  <li>Unordered list item 1</li>
  <li>Unordered list item 2</li>
  <li>Unordered list item 3</li>
  <li>Unordered list item 4</li>
</ul>

<ol>
  <li>Ordered list item 1</li>
  <li>Ordered list item 2</li>
  <li>Ordered list item 3</li>
  <li>Ordered list item 4</li>
</ol>
```

```css
li::marker {
  color: crimson;
  font-size: 1.5em;
  font-weight: bold;
}
```

:::

في هذا الدرس، غطينا فقط بعض العناصر المزيفة. لكن هناك المزيد مثل `::placeholder`، `::spelling-error` و `::selection` التي أشجعك على استكشافها بنفسك.
