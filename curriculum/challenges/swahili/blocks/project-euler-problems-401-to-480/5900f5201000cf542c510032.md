# --description--

Nambari za Fibonacci $\\{f_n, n ≥ 0\\}$ zimetangazwa kwa mfululizo kama $f_n = f_{n - 1} + f_{n - 2}$ kwa misingi ya $f_0 = 0$ na $f_1 = 1$.

Tangaza polinomia $\\{F_n, n ≥ 0\\}$ kama $F_n(x) = \displaystyle\sum_{i = 0}^n f_ix^i$.

Kwa mfano, $F_7(x) = x + x^2 + 2x^3 + 3x^4 + 5x^5 + 8x^6 + 13x^7$, na $F_7(11) = 268\\,357\\,683$.

Acha $n = {10}^{15}$. Tafuta jumla $\displaystyle\sum_{x = 0}^{100} F_n(x)$ na toa jibu lako modulo $1\\,307\\,674\\,368\\,000 \\, (= 15!)$.
