# --description--

Ви вже вивчили вказівні займенники, такі як `this`, `that`, `these` і `those`. Пам’ятайте, що їх слід використовувати, щоб посилатися на конкретні речі або людей, враховуючи їхню близькість у часі чи просторі.

Родовий регістр, який часто позначається апострофом і `s` (наприклад, `function's`), використовується для позначення приналежності або належності, зазвичай для людей або об’єктів. Він показує, що щось належить або є частиною чогось іншого.

Слово `purpose` означає причину, чому щось робиться або використовується, або чому щось існує. Наприклад, `The purpose of a clock is to tell time.`
