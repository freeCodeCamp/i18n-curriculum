# --description--

- Rekursion ist ein Programmierkonzept, das es Ihnen ermöglicht, eine Funktion wiederholt aufzurufen, bis ein Basisfall erreicht ist.

Hier ist ein Beispiel für eine rekursive Funktion, die die Fakultät einer Zahl berechnet:

```js
function findFactorial(n) {
  if (n === 0) {
    return 1;
  }
  return n * findFactorial(n - 1);
}
```

Im obigen Beispiel wird die `findFactorial`-Funktion rekursiv aufgerufen, bis `n` `0` erreicht. Wenn `n` `0` ist, wird der Basisfall erreicht und die Funktion gibt `1` zurück. Die Funktion gibt dann das Produkt von `n` und dem Ergebnis des rekursiven Aufrufs von `findFactorial(n - 1)` zurück.

- Rekursion ermöglicht es Ihnen, etwas mit unbekannter Tiefe zu verarbeiten, wie zum Beispiel tief verschachtelte Objekte/Arrays oder einen Dateibaum.
- Ein Aufrufstapel wird verwendet, um die Funktionsaufrufe in einer rekursiven Funktion zu verfolgen. Jedes Mal, wenn eine Funktion aufgerufen wird, wird sie dem Aufrufstapel hinzugefügt. Wenn der Basisfall erreicht ist, werden die Funktionsaufrufe aus dem Stapel entfernt.
- Sie sollten den Basisfall sorgfältig definieren, da ein unendliches Aufrufen dazu führen kann, dass Ihr Code abstürzt. Das liegt daran, dass die Rekursion immer mehr Funktionsaufrufe anhäuft, bis das System keinen Speicher mehr hat.
- Rekursionen werden verwendet, um mathematische Probleme wie Fakultät und Fibonacci zu lösen, Bäume und Graphen zu durchlaufen, Permutationen und Kombinationen zu erzeugen und vieles mehr.
