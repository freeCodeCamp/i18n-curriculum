# --interactive--

Sifa za kawaida za CSS, zinazojulikana pia kama vigezo vya CSS, ni vitu vinavyotangazwa na waandishi wa CSS vinavyoshikilia thamani maalum za kutumika tena katika hati nzima. Ni kipengele chenye nguvu kinachoruhusu majedwali ya mitindo kuwa na ufanisi zaidi, rahisi kudumishwa, na yenye kubadilika.

Sintaksia ya kutangaza sifa ya kawaida ni rahisi. Huanza na alama mbili za mkato (`--`) ikifuatiwa na jina la sifa:

```css
:root {
  --main-color: #3498db;
}
```

Katika mfano huu, tunatangaza sifa ya kawaida yenye jina `--main-color` na thamani ya `#3498db`. Darasa la bandia la `:root` hutumika sana kutangaza sifa za kawaida za kimataifa, kwani linawakilisha mzazi wa ngazi ya juu kabisa katika mti wa DOM.

Ili kutumia sifa ya kawaida, unatumia kitendakazi cha `var()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
:root {
  --main-color: #3498db;
}

.button {
  background-color: var(--main-color);
}
```

:::

Hii inaweka rangi ya nyuma ya vipengele vyenye darasa la `button` kuwa thamani iliyohifadhiwa katika `--main-color`.

Moja ya vipengele muhimu vya sifa za kawaida ni kwamba zinafuata mtiririko wa CSS. Hii ina maana unaweza kuzitangaza upya kwa vipengele maalum au muktadha fulani:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="alert">This is an alert message.</div>
```

```css
.alert {
  --main-color: #e74c3c;
  background-color: var(--main-color);
}
```

:::

Katika kesi hii, vipengele vyenye darasa la `alert` vitatumia thamani tofauti ya `--main-color`, ikizusha tamko la kimataifa.

Sifa za kawaida pia zinaunga mkono thamani za njia mbadala. Ikiwa sifa ya kawaida haijatangazwa au ni batili, unaweza kutoa thamani ya njia mbadala:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="text">This is some text.</div>
```

```css
:root {
  --text-color: green;
}

.text {
  color: var(--text-color, green);
}
```

:::

Hapa, ikiwa `--text-color` haijatangazwa, `color` itatumia thamani ya `green` kama chaguo la njia mbadala.

Sifa za kawaida ni muhimu sana katika kuunda miundo inayoweza kubadilika kwa mandhari tofauti. Unaweza kutangaza seti ya sifa kwa mandhari tofauti:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="light-theme">
  <p>This is some light themed text.</p>
</div>
```

```css
:root {
  --bg-color: black;
}

body {
  background-color: var(--bg-color);
}

.light-theme {
  --bg-color: white;
  --text-color: black;
  background-color: var(--bg-color);
  color: var(--text-color);
  padding: 10px;
}

```

:::

Kubadilisha mandhari ni rahisi kama kuongeza au kuondoa darasa kutoka kwa kipengele cha `body`.

Sifa za kawaida pia zinaweza kutumika pamoja na maswali ya vyombo vya habari (media queries) kuunda miundo inayojibadilisha kulingana na kifaa:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <h2>Responsive Design</h2>
  <p>Resize the window to see the card adapt!</p>
</div>

```

```css
:root {
  --card-width: 90%;
  --card-bg: #f0f0f0;
  --card-padding: 1rem;
  --text-color: #333;
}

/* Tablet screens and up */
@media (min-width: 600px) {
  :root {
    --card-width: 70%;
    --card-bg: #e8f5e9;
    --card-padding: 1.5rem;
  }
}

/* Desktop screens and up */
@media (min-width: 1024px) {
  :root {
    --card-width: 50%;
    --card-bg: #d0f0ff;
    --card-padding: 2rem;
  }
}

body {
  font-family: system-ui, sans-serif;
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
  background-color: #fafafa;
}

.card {
  width: var(--card-width);
  background-color: var(--card-bg);
  padding: var(--card-padding);
  color: var(--text-color);
  border-radius: 8px;
  box-shadow: 0 2px 6px rgba(0,0,0,0.1);
  text-align: center;
  transition: all 0.3s ease;
}

```

:::

Njia hii inaruhusu miundo inayojibadilisha kulingana na kifaa kuwa rahisi kudumishwa, kwani unaweza kubadilisha thamani mahali pamoja badala ya katika jedwali lako lote la mitindo. Hii inaweza kupunguza sana kurudia mara kwa mara katika majedwali yako ya mitindo, hasa kwa thamani ngumu zinazotumika mara kwa mara.

Sifa za kawaida pia zinaweza kurejea sifa nyingine za kawaida:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
:root {
  --primary-color: #3498db;
  --button-bg: var(--primary-color);
}

button {
  background-color: var(--button-bg);
}

```

:::

Hii inaruhusu kuunda uhusiano kati ya vipengele tofauti vya mtindo, na kufanya iwe rahisi kudumisha mandhari thabiti katika tovuti yako.

Ingawa sifa za kawaida zina faida nyingi, ni muhimu kujua kuhusu msaada wa vivinjari. Zinasaidiwa vizuri katika vivinjari vya kisasa, lakini vivinjari vya zamani huenda visivitambue. Daima kuwa na mpango wa njia mbadala unapotumia vipengele vipya kabisa.

Kwa kumalizia, sifa za kawaida za CSS hutoa njia yenye nguvu ya kuunda majedwali ya mitindo yenye nguvu zaidi, yenye kubadilika, na rahisi kudumishwa. Zinakuwezesha kuunda miundo inayoweza kubadilika kwa mandhari, kurahisisha miundo inayojibadilisha kulingana na kifaa, na kuwezesha mabadiliko ya mitindo wakati wa utekelezaji.

Kadiri ukuzaji wa mtandao unavyoendelea, sifa za kawaida zinakuwa chombo muhimu zaidi katika mkoba wa msanidi programu, zikitoa fursa mpya za kuunda CSS inayobadilika na yenye ufanisi.
