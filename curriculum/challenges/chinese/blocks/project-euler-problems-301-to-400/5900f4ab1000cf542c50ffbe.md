# --description--

设 $x_1, x_2, \ldots, x_n$ 为长度为 $n$ 的序列，使得：

- $x_1 = 2$
- 对于所有 $1 &lt; i ≤ n : x_{i - 1} &lt; x_i$
- 对于所有 $i$ 和 $j$，满足 $1 ≤ i, j ≤ n$ ：${(x_i)}^j < {(x_j + 1)}^i$

这样的长度为2的序列只有五个，即： {2,4}，{2,5}，{2,6}，{2,7}和{2,8}。 有293个这样的长度为5的序列。 以下是三个示例： {2,5,11,25,55}，{2,6,14,36,88}，{2,8,22,64,181}。

令 $t(n)$ 表示长度为 $n$ 的此类序列的数量。已知 $t(10) = 86195$ 且 $t(20) = 5227991891$。

查找 $t({10}^{10})$ 并将你的答案对 $10^9$ 取模。
