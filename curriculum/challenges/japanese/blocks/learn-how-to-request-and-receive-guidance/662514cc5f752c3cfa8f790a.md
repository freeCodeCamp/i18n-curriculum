# --description--

`Time complexity` はコンピューターサイエンスの用語です。これは、入力の大きさに基づいて、アルゴリズム（命令のセット）が完了するまでにかかる時間を表します。簡単な例としては、皿洗いがあります：皿を1枚洗うのは早いですが、多くの皿を洗うには時間がかかります。これはプログラミングにおける時間計算量に似ています。
