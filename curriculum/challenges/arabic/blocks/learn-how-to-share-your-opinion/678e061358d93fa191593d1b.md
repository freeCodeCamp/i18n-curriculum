# --explanation--

`Visual Studio Code` هو محرر شفرة مجاني ومفتوح المصدر طورته شركة Microsoft. يستخدمه المطوّرون على نطاق واسع. يقدم مجموعة متنوعة من الميزات التي تساعد في البرمجة، مثل تمييز الصيغة النحوية، وتصحيح الأخطاء، والتحكم في الإصدارات. على سبيل المثال:

`Many developers use Visual Studio Code for its flexibility and support for multiple programming languages.` - هذا يعني أن المطوّرين يفضلون Visual Studio Code لأنه يمكن استخدامه مع لغات ومهام مختلفة.

`Because` تُستخدم لتقديم سبب أو مبرر لشيء ما. تربط بين بيان وشرح. على سبيل المثال:

`I prefer coding in Python because it's simple and easy to learn.` - يعني أن سبب تفضيلك للغة Python هو سهولة فهمها واستخدامها.
