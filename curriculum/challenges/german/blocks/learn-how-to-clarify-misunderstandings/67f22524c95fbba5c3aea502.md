# --explanation--

`On track` bedeutet, dass ein Projekt oder eine Aufgabe planmäßig voranschreitet. Zum Beispiel:

`We're on track to finish the website next month.` — Das bedeutet, dass das Team das Ziel wahrscheinlich erreichen wird.

`Release` bedeutet, etwas zu starten oder verfügbar zu machen. Zum Beispiel:

`They plan to release the new version tomorrow.` — Das bedeutet, dass es öffentlich gemacht wird.

`By the end of` legt einen Zeitrahmen fest und bedeutet, dass etwas vor einem bestimmten Zeitpunkt erledigt sein muss. Zum Beispiel:

`The report is due by the end of the day.` — Das bedeutet, dass es vor Tagesende fertig sein muss.
