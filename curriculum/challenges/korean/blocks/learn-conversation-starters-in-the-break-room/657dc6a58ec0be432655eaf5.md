# --description--

`sometimes`, `always`, `never` 같은 단어들은 행동의 빈도를 설명할 때 사용합니다.

`Sometimes`는 어떤 경우에만 행동이 일어나지만 매번 일어나지는 않을 때 사용합니다. 예를 들어, `I sometimes have coffee for breakfast`. 이는 가능성을 나타내지만 확실성을 의미하지는 않습니다.

`Always`는 행동이 매번 빠짐없이 일어날 때 사용합니다. 예를 들어, `The sun always rises in the east`. 이는 일관되거나 습관적인 행동을 보여줍니다.

`Never`는 행동이 전혀 일어나지 않을 때 의미합니다. 예를 들어, `She never eats meat`. 이는 행동이 완전히 부재함을 나타냅니다.
