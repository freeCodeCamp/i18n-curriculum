# --interactive--

**KUMBUKA**: Baadhi ya mifano ya mwingiliano inaweza kutumia CSS ambayo bado hujajifunza. Usijali kuhusu kuelewa msimbo wote. Lengo la mifano ni kukuonyesha mapitio ya dhana hizi za kubuni ili uelewe vyema jinsi vitu vinavyofanya kazi. Ili kuona mapitio, utahitaji kuwezesha mhariri wa mwingiliano.

Linapokuja suala la kubuni mtandao, kuna aina nyingi za msaada wa urambazaji unaweza kutumia. Mifano ni pamoja na bar za urambazaji za juu, bar za pembeni, na vijachini. Lakini ikiwa tovuti yako ni ngumu zaidi na ina ngazi za urambazaji za kina, unaweza kutaka kuzingatia kutumia breadcrumbs.

Breadcrumbs ni msaada wa urambazaji unaomuonyesha mtumizi anapopatikana katika muktadha wa muundo wa tovuti. Hapa kuna mfano wa jinsi breadcrumbs zinavyoonekana kwa tovuti ya mfano ya vifaa vya elektroniki:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home / Electronics / Phones / Smartphone XYZ
</div>
```

```css
.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

Katika tovuti nyingi, breadcrumbs huonyeshwa juu ya ukurasa, zikimuonyesha mtumizi njia aliyopita kufikia ukurasa wa sasa. Kuanzia na `Homepage`, mtumizi alirambazia hadi kwenye kategoria ya `Electronics`, kisha hadi kategoria ya `Phones`, na hatimaye hadi bidhaa ya `Smartphone XYZ`. Huenda umewahi kuingiliana na breadcrumbs kwenye tovuti wakati ulipotafuta bidhaa au kipande maalum cha taarifa.

Matumizi ya breadcrumbs ni msaada kwa sababu yanaweza kusaidia watumizi kuelewa walipo katika muundo wa tovuti na jinsi ya kurudi nyuma kwenye kurasa zilizopita. Hii ni muhimu hasa wakati mtumizi amekuja kutoka matokeo ya utafutaji au kiungo cha nje na anahitaji kuelewa muktadha wa ukurasa alio nao.

Linapokuja suala la kubuni breadcrumbs, kuna mambo machache ya kuzingatia. Kwanza ni kuamua ni alama gani itakuwa separator. Separator ni alama inayotenganisha ngazi tofauti za muundo. Separator zinazotumika kawaida ni alama ya zaidi kuliko (`>`), alama za nukuu za pembe ya kulia (`»`), na alama ya mstari wa mbele (`/`).

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
```

```css
.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

Mambo ya pili ni mahali pa kuweka breadcrumbs. Breadcrumbs kawaida huwekwa juu ya ukurasa, iwe juu au chini ya bar kuu ya urambazaji. Watumizi hawapaswi kuhitaji shida kupata breadcrumbs, hivyo hakikisha zinaonekana na rahisi kupatikana.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<nav class="main-nav">
  <a href="#">Home</a>
  <a href="#">Electronics</a>
  <a href="#">Phones</a>
</nav>
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
```

```css
.main-nav {
  background-color: #333;
  padding: 1rem;
}

.main-nav a {
  color: white;
  margin-right: 1rem;
  text-decoration: none;
}

.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

Mambo mengine ni ukubwa wa breadcrumbs. Unataka kuhakikisha breadcrumbs ni kubwa vya kutosha kusomeka kwa urahisi, lakini si kubwa sana kiasi cha kuchukua nafasi nyingi kwenye ukurasa. Kumbuka, breadcrumbs hazikusudiwi kutumika kama chombo kikuu cha urambazaji, bali kama msaada wa urambazaji wa sekondari. Katika tovuti zilizo na taarifa nyingi kwenye ukurasa, watumizi wanaweza kuona kwa urahisi walipo katika muundo na kurudi nyuma kwenye kurasa zilizopita kwa kutumia breadcrumbs.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
<p>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
</p>
```

```css
.breadcrumbs {
  font-size: 1rem;
  color: #555;
  margin: 1rem 0;
}   

p {
  font-size: 1.2rem;
  line-height: 1.6;
}
```

:::

Basi ni lini unapaswa kutumia breadcrumbs kwenye tovuti yako? Ikiwa tovuti yako ni rahisi na ina muundo wa ngazi za chini, breadcrumbs huenda hazihitajiki. Muundo wa ngazi za chini maana yake ni kwamba tovuti ina ngazi chache za urambazaji, hivyo breadcrumbs huenda hazileti thamani kubwa. Kawaida, utaona breadcrumbs kwenye tovuti za biashara mtandaoni, tovuti za habari, na tovuti nyingine zenye muundo wa ngazi za kina kama tovuti za nyaraka za kiufundi.
