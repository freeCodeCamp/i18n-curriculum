# --description--

To `use` significa poner algo a un propósito particular. Por ejemplo:

`She uses a notebook to take notes in class.` - Esto muestra cómo ella utiliza el cuaderno para escribir.

A `tool` es algo que te ayuda a hacer un trabajo. En el trabajo, las herramientas son a menudo software o cosas en la computadora. Por ejemplo:

`Zoom` es una herramienta para reuniones en línea. - En este caso, la herramienta es un software que ayuda a las personas a conectarse y hablar.
