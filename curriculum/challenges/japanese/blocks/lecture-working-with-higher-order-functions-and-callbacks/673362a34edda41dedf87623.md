# --interactive--

`map` メソッドは、JavaScriptで配列に対して動作する強力で広く使われている関数です。このメソッドは、元の配列の各要素に指定された関数を適用して新しい配列を作成するために設計されています。

このメソッドは元の配列を変更せず、代わりに各要素に適用された関数の結果を含む新しい配列を返します。

以下は、数値の配列に`map` メソッドを使った例です。

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map((num) => num * 2);
console.log(numbers); // [1, 2, 3, 4, 5]
console.log(doubled); // [2, 4, 6, 8, 10]
```

:::

各数値を2倍にした新しい配列を作成するために、`map` メソッドを使っています。`map` メソッドはコールバック関数を受け取り、その関数が配列のすべての要素に対して呼び出されます。

この場合、配列の各数値は`2` 倍されます。結果は`2,4,6,8,10` 数値の新しい配列になります。

コールバック関数は最大で3つの引数を受け取ることができます。

最初の引数は現在処理されている要素です。

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element) => {
  console.log("Element:", element);
  return element * 2;
});
```

:::

2番目の引数は現在処理されている要素のインデックスです。

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index) => {
  console.log("Element:", element);
  console.log("Index:", index);
  return element * 2;
});
```

:::

3番目の引数は`map` が呼び出されている配列です。

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index, array) => {
  console.log("Element:", element);
  console.log("Index:", index);
  console.log("Array:", array);
  return element * 2;
});
```

:::

`map` メソッドを理解し効果的に使うことは、JavaScriptで配列を扱う能力を大きく向上させます。今後のレッスンでは、`map` のより高度な使い方に深く入り込み、動的で効率的なプログラムを作るための強力なツールとしての活用法を探っていきます。
