# --explanation--

**間接話法**では、`said that` という表現を使って、誰かが言ったことをそのままの言葉ではなく別の言い方で伝えます。`that` は任意で、日常会話では省略されることが多いです。例えば：

- 直接話法：`We need to update the software.`

- 間接話法：`He said that we need to update the software.` または `He said we need to update the software.`

どちらの表現も正しく、同じ意味です。
