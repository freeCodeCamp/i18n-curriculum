# --explanation--

`Really` と `too` はどちらも**強調語**ですが、意味が異なります。`Really` は修飾する形容詞を強めますが、過剰を示すわけではありません。一方、`Too` は過剰を示し、許容できる範囲を超えていることを意味します。例えば：

- `The project is really important.` - これはプロジェクトが非常に重要であることを意味しますが、必ずしも過剰ではありません。

- `This task is too complicated.` - これはそれが本来よりも複雑すぎることを意味します。
