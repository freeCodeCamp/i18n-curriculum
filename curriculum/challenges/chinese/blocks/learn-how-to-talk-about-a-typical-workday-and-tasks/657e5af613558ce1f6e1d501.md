# --description--

`security incident` 是可能对公司或组织的安全性造成危害的事件。例如，`A security incident like a stolen password needs to be reported immediately`。

关于 `harm` 和 `harmful` 的说明：

`Harm` 指对某人或某物造成的损害或伤害。它指的是造成负面效果或使某人或某物处于风险之中。例如：

`Touching hot surfaces can cause harm.` - 这意味着触摸热表面可能导致受伤或损害。

`Harmful` 是一个形容词，用来描述可能造成伤害的事物。它的意思是危险的或有风险的，可能导致损害或受伤。例如：

`Smoking is harmful to health.`——这表明吸烟会导致健康问题或损害。

这些术语通常用于讨论安全性和风险，特别是在健康、安全性和环境等上下文中。
