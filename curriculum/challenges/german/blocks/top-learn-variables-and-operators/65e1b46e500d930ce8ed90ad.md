# --description--

Eine Zahl um eins zu erhöhen oder zu verringern gehört zu den häufigsten numerischen Operationen.

Dafür gibt es spezielle Operatoren:

- Der Inkrement-Operator `++` erhöht eine Variable um 1:

```js
let counter = 2;

// works the same as counter = counter + 1, but is shorter
counter++;      

console.log(counter); // 3
```

- Der Dekrement-Operator `--` verringert eine Variable um 1:

```js
let counter = 2;

// works the same as counter = counter - 1, but is shorter
counter--;

console.log(counter); // 1
```

Die Operatoren `++` und `--` können entweder vor oder nach einer Variable stehen.

- Wenn der Operator nach der Variable steht, spricht man von der „Postfix-Form“: `counter++`.

- Die „Präfix-Form“ ist, wenn der Operator vor der Variable steht: `++counter`.

Beide Anweisungen bewirken dasselbe: `counter` wird um `1` erhöht.

Gibt es einen Unterschied? Ja, aber Sie sehen ihn nur, wenn Sie den zurückgegebenen Wert von `++/--` verwenden.

Lassen Sie uns das klären. Wie Sie wissen, geben alle Operatoren einen Wert zurück. Inkrementieren/Dekrementieren bildet da keine Ausnahme. Die Präfix-Form gibt den neuen Wert zurück, während die Postfix-Form den alten Wert (vor der Erhöhung/Verringerung) zurückgibt.

Um den Unterschied zu sehen, hier ein Beispiel:

```js
let counter = 1;
let a = ++counter; // (*)

console.log(a); // 2
```

Zusammengefasst:

- Wenn das Ergebnis des Inkrementierens/Dekrementierens nicht verwendet wird, macht es keinen Unterschied, welche Form Sie verwenden:

```js
let counter = 0;
counter++;
++counter;

console.log( counter ); // 2, the lines above did the same
```

- Wenn Sie einen Wert erhöhen und das Ergebnis des Operators sofort verwenden möchten, brauchen Sie die Präfix-Form:

```js
let counter = 0;
console.log( ++counter ); // 1
```

- Wenn Sie einen Wert erhöhen, aber den vorherigen Wert verwenden möchten, brauchen Sie die Postfix-Form:

```js 
let counter = 0;
console.log( counter++ ); // 0
```

# --questions--

## --text--

Was sind die Ausgaben der beiden `console.log`-Anweisungen im folgenden JavaScript-Code?

```js
let counter = 1;
console.log(2 * ++counter); // Statement A
```

```js
let counter = 1;
console.log(2 * counter++); // Statement B
```

## --answers--

Es gibt keinen Unterschied; beide `console.log`-Anweisungen zeigen dasselbe Ergebnis.

---

Anweisung A gibt `4` aus, weil `++counter` den Wert vor der Multiplikation erhöht. Anweisung B gibt `2` aus, weil `counter++` den Wert nach der Multiplikation erhöht.

---

Anweisung A gibt `2` aus, weil `++counter` ein Syntaxfehler ist. Anweisung B gibt `4` aus, weil `counter++` die korrekte Art ist, einen Zähler zu erhöhen.

---

Beide Anweisungen geben `3` aus, weil der Zähler in beiden Fällen vor der `console.log` erhöht wird.
