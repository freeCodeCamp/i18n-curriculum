# --description--

令 $T(n)$ 為第 $n^{\text{}}$ 個三角形數，因此 $T(n) = \frac{n(n + 1)}{2}$。

設 $dT(n)$ 為 $T(n)$ 的因數個數。例如：$T(7) = 28$ 且 $dT(7) = 6$。

令 $Tr(n)$ 為滿足 $1 ≤ i &lt; j &lt; k ≤ n$ 且 $dT(i) > dT(j) > dT(k)$ 的三元組 ($i$, $j$, $k$) 數量。$Tr(20) = 14$、$Tr(100) = 5\\,772$ 及 $Tr(1000) = 11\\,174\\,776$。

求 $Tr(60\,000\,000)$。給出你答案的最後 18 位數字。
