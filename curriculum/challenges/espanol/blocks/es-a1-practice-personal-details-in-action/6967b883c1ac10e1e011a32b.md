# --explanation--

`Soy uruguaya` significa "I am Uruguayan".

La palabra `uruguaya` es la forma femenina para alguien de Uruguay.

El español usa términos de nacionalidad con género, así que como Julieta es mujer, usa `uruguaya` (no `uruguayo`).
