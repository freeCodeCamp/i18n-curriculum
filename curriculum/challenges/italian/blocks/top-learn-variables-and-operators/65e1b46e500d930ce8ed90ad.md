# --description--

Incrementare o decrementare un numero di uno è una delle operazioni numeriche più comuni.

Per questo esistono operatori speciali:

- L'incremento `++` aumenta una variabile di 1:

```js
let counter = 2;

// works the same as counter = counter + 1, but is shorter
counter++;      

console.log(counter); // 3
```

- Il decremento `--` diminuisce una variabile di 1:

```js
let counter = 2;

// works the same as counter = counter - 1, but is shorter
counter--;

console.log(counter); // 1
```

Gli operatori `++` e `--` possono essere posizionati sia prima che dopo una variabile.

- Quando l'operatore segue la variabile, si parla di "forma postfix": `counter++`.

- La "forma prefix" è quando l'operatore precede la variabile: `++counter`.

Entrambe queste istruzioni fanno la stessa cosa: aumentano `counter` di `1`.

C'è qualche differenza? Sì, ma la puoi notare solo se usi il valore restituito da `++/--`.

Facciamo chiarezza. Come sai, tutti gli operatori restituiscono un valore. Incremento/decremento non fa eccezione. La forma prefix restituisce il nuovo valore mentre la forma postfix restituisce il valore precedente (prima dell'incremento/decremento).

Per vedere la differenza, ecco un esempio:

```js
let counter = 1;
let a = ++counter; // (*)

console.log(a); // 2
```

Per riassumere:

- Se il risultato dell'incremento/decremento non viene usato, non c'è differenza su quale forma usare:

```js
let counter = 0;
counter++;
++counter;

console.log( counter ); // 2, the lines above did the same
```

- Se vuoi aumentare un valore e usare subito il risultato dell'operatore, ti serve la forma prefix:

```js
let counter = 0;
console.log( ++counter ); // 1
```

- Se vuoi incrementare un valore ma usare il suo valore precedente, ti serve la forma postfix:

```js 
let counter = 0;
console.log( counter++ ); // 0
```

## --text--

Quali sono gli output delle due istruzioni `console.log` nel codice JavaScript qui sotto?

```js
let counter = 1;
console.log(2 * ++counter); // Statement A
```

```js
let counter = 1;
console.log(2 * counter++); // Statement B
```

## --answers--

Non c'è differenza; entrambe le istruzioni `console.log` mostreranno lo stesso risultato.

---

L'istruzione A stamperà `4` perché `++counter` incrementa il valore prima della moltiplicazione. L'istruzione B stamperà `2` perché `counter++` incrementa il valore dopo la moltiplicazione.

---

L'istruzione A stamperà `2` perché `++counter` è un errore di sintassi. L'istruzione B stamperà `4` perché `counter++` è il modo corretto per incrementare un contatore.

---

Entrambe le istruzioni stamperanno `3` perché il contatore viene incrementato in entrambi i casi prima del `console.log`.
