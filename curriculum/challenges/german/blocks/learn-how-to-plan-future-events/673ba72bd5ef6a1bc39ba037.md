# --explanation--

`Downtime` ist ein Zeitraum, in dem ein System oder Dienst nicht verfügbar oder nicht betriebsbereit ist. Er wird oft geplant, um Wartungen, Updates oder Reparaturen durchzuführen.

`Server downtime` bezeichnet speziell die Zeit, in der ein Server – ein Computersystem, das anderen Geräten Daten, Dienste oder Ressourcen bereitstellt – vorübergehend nicht verfügbar ist. Zum Beispiel:

- `The server downtime was scheduled at midnight to minimize the impact on users.` – Das bedeutet, der Server wurde um Mitternacht abgeschaltet, um den Komfort der Benutzer zu verringern.

- `Unexpected downtime occurred due to a power failure.` – Das bedeutet, das System wurde aufgrund eines Stromausfalls ohne vorherige Planung nicht verfügbar.
