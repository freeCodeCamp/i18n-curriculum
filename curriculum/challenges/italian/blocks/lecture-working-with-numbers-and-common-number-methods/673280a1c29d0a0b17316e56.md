# --interactive--

Il metodo `.toFixed()` è una funzione integrata di JavaScript che formatta un numero usando la notazione a punto fisso. È particolarmente utile quando devi controllare il numero di cifre decimali in un numero, specialmente per mostrare valori monetari o quando lavori con misurazioni precise.

Il metodo `.toFixed()` viene chiamato su un numero e accetta un argomento opzionale, che indica il numero di cifre da mostrare dopo il punto decimale. Restituisce una rappresentazione in stringa del numero con il numero specificato di cifre decimali. Ecco un esempio base di come funziona `.toFixed()`:

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed(2)); // "3.14"
```

:::

In questo caso, limitiamo il numero di cifre decimali a due. Quindi, `3.14159` diventa `3.14`. È importante notare che `.toFixed()` restituisce una stringa, non un numero. Questo perché il metodo è pensato principalmente per formattare i numeri per la visualizzazione, non per ulteriori calcoli.

Il metodo `.toFixed()` arrotonda il numero al valore più vicino che può essere rappresentato con il numero specificato di cifre decimali. Questo comportamento di arrotondamento è importante da capire:

:::interactive_editor

```js
console.log((3.14159).toFixed(3));  // "3.142"
console.log((3.14449).toFixed(3));  // "3.144"
console.log((3.14550).toFixed(3));  // "3.146"
```

:::

Come puoi vedere, `.toFixed()` arrotonda per eccesso quando la cifra successiva è `5` o maggiore, e per difetto altrimenti. Se chiami `.toFixed()` senza argomenti, il valore predefinito è `0` cifre decimali:

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed()); // "3"
```

:::

Il metodo `.toFixed()` può essere particolarmente utile quando lavori con calcoli finanziari o mostri prezzi:

:::interactive_editor

```js
let price = 19.99;
let taxRate = 0.08;
let total = price + (price * taxRate);

console.log("Total: $" + total.toFixed(2)); // "Total: $21.59"
```

:::

In questo esempio, `.toFixed(2)` garantisce che il totale venga sempre mostrato con due cifre decimali, cosa standard per le valute in molti paesi.

In conclusione, il metodo `.toFixed()` è uno strumento potente per formattare numeri in JavaScript, soprattutto quando devi controllare la visualizzazione delle cifre decimali. Anche se è usato principalmente per formattare l'output, ricorda il suo comportamento, specialmente quando servono calcoli precisi.
