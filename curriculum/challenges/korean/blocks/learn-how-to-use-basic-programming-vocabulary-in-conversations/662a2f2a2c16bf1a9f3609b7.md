# --description--

`Much of`는 특히 셀 수 없는 명사와 함께 어떤 것의 큰 부분을 표현할 때 사용합니다. 예를 들어, `He spent much of the day reading.`은 그가 하루의 상당 부분을 독서에 사용했다는 뜻입니다.

`Free time`는 일하거나 필수적인 일을 하지 않는 시간을 가리키며, 원하는 일을 선택할 수 있는 시간입니다. 예를 들어, `She uses her free time to learn guitar.`은 그녀가 다른 의무가 없을 때 무엇을 선택하는지를 나타냅니다.
