# --description--

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

بوضع `animal` داخل الأقواس العادية للدالة `favoriteAnimal()`، تخبر JavaScript بأنك سترسل قيمة ما إلى دالتك `favoriteAnimal`. هذا يعني أن animal هو مجرد عنصر نائب لقيمة مستقبلية. لكن ما القيمة التي سترسلها؟

السطر الأخير، `favoriteAnimal('Goat')`، هو المكان الذي تستدعي فيه دالة `favoriteAnimal` وتمرر القيمة `'Goat'` داخل تلك الدالة. هنا، `'Goat'` هي المعلمة الخاصة بك. أنت تخبر دالة `favoriteAnimal`، "يرجى إرسال `'Goat'` إلى دالة `favoriteAnimal` واستخدام `'Goat'` في أي مكان يوجد فيه عنصر نائب `'animal'`." بسبب المرونة التي يوفرها استخدام المعلمة، يمكنك إعلان أي حيوان كحيوانك المفضل.

إليك رسمًا توضيحيًا يساعدك على تصور كيفية تمرير المعلمات إلى الدالة، وكيف تُعاد القيم منها.
