# --description--

繼承是指某些 CSS 屬性，當套用到一個元素時，即使我們沒有為該元素的後代明確地撰寫規則，這些屬性仍會被該元素的後代繼承。排版相關的屬性（如 `color`、`font-size`、`font-family` 等）通常會被繼承，而大多數其他屬性則不會。

唯一的例外是當直接針對元素時，因為這總是勝過繼承：

```html
<!-- index.html -->

<div id="parent">
  <div class="child"></div>
</div>
```

```css
/* styles.css */

#parent {
  color: red;
}

.child {
  color: blue;
}
```

儘管 `parent` 元素具有較高特異性的 ID，`child` 元素仍會套用 `color: blue` 樣式，因為該宣告是直接標的它，而來自父元素的 `color: red` 只是繼承而來。
