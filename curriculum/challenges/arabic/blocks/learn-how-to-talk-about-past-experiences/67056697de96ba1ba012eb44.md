# --explanation--

تُستخدم الروابط المتتالية مثل `first`، `then`، `after that`، و `finally` لوصف ترتيب الأفعال أو الأحداث.

- تُستخدم `First` لوصف الخطوة الأولى.

- تشير `Then` إلى الخطوة التالية المباشرة.

- تُستخدم `After that` لتقديم خطوة تلي فعلًا آخر مع تمييز بينهما.

- تشير `Finally` إلى الخطوة الأخيرة في المتتالية.
