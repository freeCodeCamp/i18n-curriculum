# --description--

$f(0) = 1$ と定義し、$f(n)$ を、$n$ を2の累乗の和として表す方法のうち、どの累乗も2回を超えて現れない場合の数とします。

例えば、$f(10) = 5$ です。10を表す異なる方法は5通りあります：

$$10 = 8 + 2 = 8 + 1 + 1 = 4 + 4 + 2 = 4 + 2 + 2 + 1 + 1 = 4 + 4 + 1 + 1$$

任意の分数 $\frac{p}{q}\\; (p>0, q>0)$ に対して、少なくとも1つの整数 $n$ が存在し、$\frac{f(n)}{f(n - 1)} = \frac{p}{q}$ となることが示されています。

例えば、$\frac{f(n)}{f(n - 1)} = \frac{13}{17}$ となる最小の $n$ は241です。241の2進展開は11110001です。

この2進数を最上位ビットから最下位ビットまで読むと、1が4回、0が3回、1が1回あります。この文字列「4,3,1」を241の短縮2進展開と呼びます。

次の条件を満たす最小の $n$ の短縮2進展開を求めてください。

$$\frac{f(n)}{f(n - 1)} = \frac{123456789}{987654321}$$

答えは、カンマ区切りの整数の文字列で、空白は入れずに提出してください。
