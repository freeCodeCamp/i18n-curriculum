# --description--

A frase `no worries` é uma expressão casual usada para tranquilizar alguém de que está tudo bem e que não deve se preocupar. Pode ser usado para responder a um agradecimento, para dispensar um pedido de desculpas ou para tranquilizar alguém de que um problema não é sério.

Por exemplo, quando seu amigo acidentalmente derruba uma bebida e começa a pedir desculpas, você pode dizer `no worries!` É apenas água e vai secar rápido` para fazer o amigo se sentir melhor.
