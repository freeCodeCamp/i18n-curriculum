# --interactive--

Успадкування — це ключова концепція в CSS, яка визначає, як стилі передаються від батьківських елементів до їхніх дочірніх елементів.

Як і в реальному світі, де діти часто успадковують риси від батьків, у CSS певні властивості можуть успадковуватися дочірніми елементами від батьківських.

Це дозволяє ефективніше застосовувати послідовне стилювання по всьому документу.

У CSS не всі властивості успадковуються за замовчуванням. Наприклад, властивості `color`, `font-family` і `line-height` успадковуються. Це означає, що якщо ви встановите колір тексту на батьківському елементі, усі його дочірні елементи успадкують цей колір, якщо ви явно не замістите його.

Наприклад, розглянемо приклад, де батьківський `div` елемент має встановлений колір за допомогою рядкового стилю, а дочірній `p` елемент успадковує цей колір:

:::interactive_editor

```html
<div style="color: blue;">
  This is the parent element.
  <p>This is the child element inheriting the color.</p>
</div>
```

:::

У цьому випадку і батьківський `div`, і дочірній `p` відобразять текст синім кольором, оскільки колір успадковується.

З іншого боку, властивості `margin`, `padding`, `border` і `background` за замовчуванням не успадковуються. Якщо ви хочете, щоб дочірній елемент успадкував ці стилі, потрібно явно їх встановити, або безпосередньо на дочірньому елементі, або використовуючи ключове слово `inherit`.

Ключове слово `inherit` можна використати, щоб примусово успадкувати властивість від батьківського елемента, навіть якщо ця властивість зазвичай не успадковується.

Наприклад, якщо ви хочете, щоб конкретний дочірній елемент мав той самий `padding`, що й його батько, ви можете встановити `padding: inherit` на дочірньому елементі:

:::interactive_editor

```html
<div style="padding: 20px;">
  This is the parent element with padding.
  <p style="padding: inherit;">This is the child element inheriting the padding.</p>
</div>
```

:::

У цьому випадку дочірній `p` елемент успадкує `20px` відступу від свого батьківського `div` елемента.

Успадкування особливо корисне для підтримки послідовності та зменшення дублювання у ваших таблицях стилів.

Замість того, щоб писати одне й те саме правило стилю для кількох елементів, ви можете визначити його один раз на батьківському елементі, і дочірні елементи його успадкують. Це робить ваш CSS більш лаконічним і легшим для керування.

Однак важливо пам’ятати, що успадкування працює лише в одному напрямку — від батька до дитини. Якщо ви замістите стиль на дочірньому елементі, це не вплине на батьківський елемент.
