# --description--

Un fattoriale è il prodotto di un numero intero e di tutti i numeri interi inferiori a esso. Per esempio, il fattoriale di `5` è `5 * 4 * 3 * 2 * 1 = 120`. In questo laboratorio, creerai un calcolatore di fattoriali che prende un numero dall'utente e calcola il fattoriale di quel numero.

**Obiettivo:** Completa le storie utente qui sotto e fai passare tutti i test per completare il laboratorio.

**Storie utente:**

1. Dovresti dichiarare una variabile `num` e assegnarle un numero a tua scelta. Il numero assegnato dovrebbe essere compreso tra `1` e `20` inclusi.
1. Crea una funzione chiamata `factorialCalculator` che prende un numero come argomento e restituisce il fattoriale di quel numero.
1. All'interno della funzione, dichiara una variabile `result` e assegnale il valore di `1`. Usando un loop, itera su tutti i numeri da `1` fino al numero di input (incluso) e per ogni numero moltiplica la variabile `result` per il numero corrente e assegna il risultato alla variabile `result`. Puoi scegliere di usare un loop `for`, `while` o `do...while` qui.
1. Dovresti chiamare la funzione `factorialCalculator` con `num` come argomento e assegnare il risultato alla variabile `factorial`.
1. Dovresti archiviare l'output finale nel formato `Factorial of [num] is [factorial]` e assegnarlo alla variabile `resultMsg`.
1. Dovresti mostrare il valore di `resultMsg` nella console.

# --solutions--

```js
const num = 5;

function factorialCalculator(n) {
  let result = 1;

  for (let i = 2; i <= n; i++) {
    result *= i;
  }

  return result;
}

const factorial = factorialCalculator(num);
const resultMsg = `Factorial of ${num} is ${factorial}`;
console.log(resultMsg);
```
