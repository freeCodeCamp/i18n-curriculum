# --interactive--

Давайте дізнаємося, коли слід використовувати `appearance: none` для вирішення проблем зі стилізацією пошукових полів, прапорців і радіокнопок.

Браузери застосовують стилі за замовчуванням до багатьох елементів. У випадку з елементами input ваша можливість стилізувати їх за допомогою CSS може здаватися досить обмеженою. Тож ви можете захотіти використати `appearance: none`, щоб приховати частини стандартного елемента і створити власні замість них. Наприклад, це приховає стандартні прапорці для input типу checkbox, дозволяючи вам використовувати кастомні індикатори, як-от зелена галочка і червоний хрестик, щоб показати стан. Для пошукового поля браузери на базі WebKit відображають стандартну іконку пошуку та кнопку скасування. Приховання цих елементів дозволяє створити власні індикатори, які будуть однаково відображатися у всіх браузерах.

Ось приклад кастомного прапорця:

**ПРИМІТКА**: Частина CSS у цьому прикладі використовує властивості, які ще не були розглянуті. Це зроблено лише для того, щоб дати уявлення про те, як створити кастомний прапорець. Ви дізнаєтеся, як це все працює, у майбутніх уроках і практичних заняттях.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form>
  <label>
    <input class="checkbox" type="checkbox" /> Agree
  </label>
</form>
```

```css
.checkbox {
  appearance: none;
  width: 18px;
  height: 18px;
  border: 2px solid #ccc;
  border-radius: 4px;
  display: inline-block;
  position: relative;
  cursor: pointer;
  transition: all 0.25s ease;
  vertical-align: middle; 
}

.checkbox:hover {
  border-color: #888;
}

.checkbox:checked {
  background-color: #4caf50;
  border-color: #4caf50;
}

.checkbox:checked::after {
  content: "";
  position: absolute;
  left: 4px;
  top: 0px;
  width: 5px;
  height: 10px;
  border: solid white;
  border-width: 0 2px 2px 0;
  transform: rotate(45deg);
}

.checkbox:focus {
  outline: 2px solid #90caf9;
  outline-offset: 2px;
}
```

:::

WebKit — це програмний движок, який допомагає веббраузерам відображати вебсторінки. Браузери, як Safari, використовують WebKit, щоб вебсторінки виглядали і працювали правильно. Це `appearance: none` CSS-властивість дає вам повний контроль над стилізацією, але має деякі нюанси, на які варто звернути увагу. Стандартні інтерактивні компоненти input-елементів включають функції, як-от індикатори фокусу та помилок, які потрібно зберегти.

Створення послідовної стилізації на різних платформах — чудова причина для використання цієї властивості. Ви також можете використати її, щоб переконатися, що цілі для натискання на мобільних пристроях достатньо великі, або що кольори прапорця мають достатній контраст.
