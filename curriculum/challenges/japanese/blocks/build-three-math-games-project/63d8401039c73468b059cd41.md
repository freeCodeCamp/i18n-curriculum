# --instructions--

このチャレンジでは、Pythonを使って以下の3つの数学ゲームを作成してください。

- 散布図ゲーム
  - グラフ上にランダムに点を生成し、プレイヤーが(x,y)座標を入力します
  - 難易度を上げるために、グラフを大きくしてください
- 代数練習ゲーム
  - 1ステップおよび2ステップの問題をランダムな整数値で生成し、プレイヤーが答えを入力します
  - 正の値と負の値を使います。難易度を上げるために、数値を大きくしてください
- 投射物ゲーム
  - ランダムな高さと位置の「壁」を表示します。プレイヤーはスライダーを動かして放物線の軌道を調整し、壁をクリアします
  - 難易度を上げるために、スライダーなしでa、b、cを入力する第2レベルを作ってください
