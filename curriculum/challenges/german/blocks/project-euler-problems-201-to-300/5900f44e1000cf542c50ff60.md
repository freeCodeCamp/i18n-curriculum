# --description--

Die Folge 1, 1, 1, 3, 5, 9, 17, 31, 57, 105, 193, 355, 653, 1201 ...

ist definiert durch $T_1 = T_2 = T_3 = 1$ und $T_n = T_{n - 1} + T_{n - 2} + T_{n - 3}$.

Es lässt sich zeigen, dass 27 kein Glied dieser Folge teilt. Tatsächlich ist 27 die erste ungerade Zahl mit dieser Eigenschaft.

Finden Sie die ${124}^{\text{te}}$ ungerade Zahl, die kein Glied der obigen Folge teilt.
