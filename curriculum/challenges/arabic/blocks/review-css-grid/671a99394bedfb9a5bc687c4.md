## أساسيات CSS Grid

- **التعريف**: CSS Grid هو نظام تخطيط ثنائي الأبعاد يُستخدم لإنشاء تخطيطات معقدة في صفحات الويب. تحتوي الشبكات على صفوف وأعمدة مع فراغات بينها. لتعريف تخطيط شبكة، تضبط خاصية `display` على القيمة `grid`.

```css
.container {
  display: grid;
}
```

- **وحدة `fr` (كسر)**: تمثل هذه الوحدة جزءًا من المساحة داخل حاوية الشبكة. يمكنك استخدام وحدة `fr` لإنشاء شبكات مرنة.
- **إنشاء فراغات بين المسارات**: هناك ثلاث طرق لإنشاء فراغات بين المسارات في شبكة CSS. يمكنك استخدام خاصية `column-gap` لإنشاء فراغات بين الأعمدة. يمكنك استخدام خاصية `row-gap` لإنشاء فراغات بين الصفوف. أو يمكنك استخدام الخاصية المختصرة `gap` لإنشاء فراغات بين الصفوف والأعمدة معًا.
- **`grid-template-columns`**: تُستخدم لتعيين أسماء الخطوط وحجم أعمدة مسارات الشبكة.

```css
.container {
  display: grid;
  width: 100%;
  grid-template-columns: 30px 1fr;
}
```

- **`grid-template-rows`**: تُستخدم لتعيين أسماء الخطوط وحجم صفوف مسارات الشبكة.
- **`grid-auto-flow`**: تحدد كيفية ملاءمة العناصر التي تُوضع تلقائيًا في الشبكة.

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-flow: column;
}
```

- **`grid-auto-columns`**: تُستخدم لتعيين حجم الأعمدة التي تُنشأ ضمنيًا.

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-columns: auto;
}
```

- **`place-items`**: تُستخدم لمحاذاة العناصر في الاتجاهين العمودي والمضمّن.
- **`align-items`**: تُستخدم لضبط المحاذاة للعناصر داخل حاوية الشبكة.
- **دالة `repeat()`**: تُستخدم هذه الدالة لتكرار أقسام في قائمة المسارات. بدلًا من كتابة `grid-template-columns: 1fr 1fr 1fr;` يمكنك استخدام دالة `repeat()` بدلاً من ذلك.

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}
```

- **الشبكة الصريحة**: يمكنك تحديد عدد الخطوط أو المسارات باستخدام خاصيتي `grid-template-columns` أو `grid-template-rows`.
- **الشبكة الضمنية**: عندما توضع العناصر خارج الشبكة، تُنشأ الصفوف والأعمدة تلقائيًا لتلك العناصر الخارجية.
- **دالة `minmax()`**: تُستخدم هذه الدالة لتعيين الحد الأدنى والحد الأقصى لحجم المسار.

```css
.container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-auto-rows: minmax(150px, auto);
}
```

- **التحديد بناءً على الخطوط**: كل الشبكات تحتوي على خطوط. لتحديد مكان بدء العنصر على خط، يمكنك استخدام خاصيتي `grid-column-start` و`grid-row-start`. لتحديد مكان انتهاء العنصر على الخط، يمكنك استخدام خاصيتي `grid-column-end` و`grid-row-end`. يمكنك أيضًا اختيار استخدام الخاصيتين المختصرتين `grid-column` أو `grid-row` بدلاً من ذلك.

إليك مثالًا على استخدام خاصية `grid-column` لجعل عنصر يمتد عبر جميع الأعمدة.

```css
.element {
  grid-column: 1 / -1;
}
```

- **`grid-template-areas`**: تُستخدم هذه الخاصية لتوفير أسماء للعناصر التي ستوضع على الشبكة.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="header">Header</div>
  <div class="sidebar">Sidebar</div>
  <div class="main">Main Content</div>
  <div class="footer">Footer</div>
</div>

```

```css
.container {
  display: grid;
  grid-template-columns: 200px 1fr; 
  grid-template-rows: auto 1fr auto; 
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer"; 
  gap: 20px; 
}

.header {
  grid-area: header; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}

.sidebar {
  grid-area: sidebar;
  background-color: #f4f4f4;
  padding: 10px;
}

.main {
  grid-area: main; 
  background-color: #e0e0e0;
  padding: 10px;
}

.footer {
  grid-area: footer; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}
```

:::

## تصحيح أخطاء CSS

- **أدوات المطوِّر (DevTools)**: تتيح لك أدوات المطوِّر فحص وتعديل CSS في الوقت الفعلي. يعرض جزء الأنماط جميع قواعد CSS المطبقة على العنصر المحدد، بما في ذلك الأنماط الموروثة. يمكنك تشغيل وإيقاف خاصيات فردية، تعديل القيم، وحتى إضافة قواعد جديدة مباشرة في المتصفح. هذه الملاحظات الفورية مفيدة جدًا لتجربة أنماط مختلفة دون تعديل الكود المصدري.
- **مدققات CSS**: تُستخدم المدققات لفحص CSS مقابل المواصفات الرسمية وتُبلغ عن أي أخطاء أو تحذيرات. من أشهر المدققات التي يمكنك استخدامها هو مدقق W3C CSS.
- **تصحيح أخطاء التصميمات المتجاوبة**: تحتوي أدوات المطوِّر على خيار يسمح بمحاكاة شكل موقعك على أحجام شاشات وأجهزة مختلفة. هذا يساعدك على تحديد مشاكل نقاط التوقف أو الأنماط التي لا تتناسب جيدًا مع أحجام إطارات العرض المختلفة.
