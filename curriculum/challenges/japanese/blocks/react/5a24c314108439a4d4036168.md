# --description--

JSXとReactコンポーネントの基本を学んだので、今度は自分でコンポーネントを書いてみましょう。ReactコンポーネントはReactアプリケーションの核となる構成要素なので、書き方に慣れることがとても重要です。典型的なReactコンポーネントはES6の`class`で、`React.Component`を継承しています。renderメソッドがあり、HTML（JSXから）または`null`を返します。これがReactコンポーネントの基本形です。これをよく理解すれば、より複雑なReactプロジェクトを作り始める準備ができます。

# --instructions--

`MyComponent`というクラスを定義し、`React.Component`を継承してください。そのrenderメソッドは、`div`を返し、その中にテキスト`h1`を含む`My First React Component!`タグがあるようにしてください。このテキストは正確に使い、大文字・小文字や句読点も重要です。コンポーネントのコンストラクタも必ず呼び出してください。

このコンポーネントを`ReactDOM.render()`を使ってDOMにレンダリングしてください。`div`と`id='challenge-node'`が利用可能です。

# --solutions--

```jsx
// Change code below this line
class MyComponent extends React.Component {
  constructor(props) {
    super(props);
  }
  render() {
    return (
      <div>
        <h1>My First React Component!</h1>
      </div>
    );
  }
};

ReactDOM.render(<MyComponent />, document.getElementById('challenge-node'));
```
