# --description--

`Many` y `much` se usan para hablar sobre cantidades, pero se usan con diferentes tipos de sustantivos. `Many` se usa con sustantivos contables, que son cosas que puedes contar, como `apples`, `cars` o `books`. Por ejemplo, puedes decir `There are many books on the shelf.`

`Much` se usa con sustantivos incontables, que son cosas que no puedes contar individualmente, como `water`, `sand`, o `information`. Por ejemplo, podr√≠as decir `There is not much water left in the bottle.`

### --feedback--

`Water` es un sustantivo incontable, por lo que debe usar `much` en lugar de `many` para describir la cantidad.
