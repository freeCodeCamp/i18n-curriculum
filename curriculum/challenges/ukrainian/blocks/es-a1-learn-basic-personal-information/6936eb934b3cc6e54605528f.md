# --explanation--

Каміла каже `¿Cuál es tu correo electrónico?` Структура відповідає такому шаблону:

`¿Cuál es tu` + іменник?

Ця структура використовується для запиту конкретної особистої інформації. У цьому випадку `correo electrónico` означає адресу е-пошти, яку використовують для надсилання та отримання повідомлень онлайн.
