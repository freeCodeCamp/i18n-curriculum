# --description--

你们中的许多人已经使用计算机有一段时间了。但你们可能没有考虑过登录计算机的最安全或最可靠的方法。

无论你使用的是 Mac 还是 PC，都有多种方法可以安全地登录你的计算机。

你首先应该确保你的计算机已设置密码保护。

对于 Windows 用户，首先打开开始菜单并选择 设置。然后，进入 账户，再进入 登录选项。接着，你可以为你的计算机 设置密码。

*注意：如果你遇到任何困难或使用不同版本的 Windows，请参考微软官方文档或支持资源，获取针对你具体版本的详细说明。*

创建密码时，确保它是一个长度较长且复杂的密码。你可以使用字母、数字和特殊字符的组合来增强安全性。

选择像 `12345` 或 `password` 这样简单的密码不是一个好主意。这些密码容易被猜到，且很容易被破解。

此外，不要使用基于个人信息如你的姓名、生日或地址的密码。这些也很容易被黑客猜到。

当你创建密码时，设置双因素认证（2FA）也是一个好主意。2FA 作为额外的安全性层，要求第二种验证形式，以确保尝试访问账户的人确实是授权用户。

如果你是 Mac 用户，你可以点击苹果菜单，然后进入系统设置。从那里你可以进入用户与群组，为你的计算机设置密码。

一些 Mac 计算机随附触控 ID 特色，这通常被认为比常规密码更安全。该特色允许你使用指纹登录你的计算机。

对于 Windows 用户，Windows Hello 特色提供了一种更安全的登录方式。它使用面部识别或指纹等生物识别方法进行认证，提供了传统密码的备选方案。

完成本课后，我强烈建议你研究这些额外的安全性特色，并将它们添加到你的计算机上，以便你能保持你的计算机安全可靠。
