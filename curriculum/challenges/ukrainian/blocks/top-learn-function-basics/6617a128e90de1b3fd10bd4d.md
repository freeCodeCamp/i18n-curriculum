# --description--

Давайте обговоримо параметри та аргументи на прикладі наступної функції:

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

У JavaScript параметри — це елементи, які перераховані між круглими дужками `()` в оголошенні функції. Аргументи функції — це фактичні значення, які ви вирішуєте передати функції.

У наведеному прикладі визначення функції записане в першій лінії: `function favoriteAnimal(animal)`. Параметр, `animal`, знаходиться всередині круглих дужок. Ви так само легко могли б замінити `animal` на `pet`, `x` або `blah`. Але в цьому випадку назва параметра `animal` дає тому, хто читає ваш код, трохи контексту, щоб не доводилося здогадуватися, що `animal` може в кінцевому результаті містити.
