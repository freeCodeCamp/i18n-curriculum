# --description--

`transform-origin` 属性用于设置应用 CSS 变换所围绕的点。 例如，当你应用 `rotate` 时（正如你在本项目后续步骤中将要做的那样），`transform-origin` 会决定元素围绕哪个点进行旋转。

给 `.line` 选择器一个 `0% 0%` 的 `transform-origin` 属性。 这将使原点从左边偏移 `0%`，从上面偏移 `0%`，即将其设定为元素的左上角。

# --hints--

你的 `.line` 选择器应该有一个 `transform-origin` 属性，设置为 `0% 0%`。

```js
const transformOrigin = new __helpers.CSSHelp(document).getStyle('.line')?.transformOrigin;
assert.oneOf(transformOrigin, ['0% 0%', '0% 0% 0px']);
```
