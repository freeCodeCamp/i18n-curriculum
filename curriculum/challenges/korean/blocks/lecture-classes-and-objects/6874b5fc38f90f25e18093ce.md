# --description--

지난 수업에서 클래스와 클래스가 객체를 만드는 청사진 역할을 하는 방법을 배웠습니다.

이번에는 속성과 메서드에 대해 더 깊이 알아보겠습니다.

먼저 속성을 자세히 살펴보고, 그다음 메서드를 살펴보겠습니다.

속성은 객체에 속하는 변수로 데이터를 저장합니다. 속성에는 인스턴스 속성과 클래스 속성 두 가지가 있습니다.

인스턴스 속성은 클래스에서 생성된 각 객체마다 고유하며, 보통 `__init__` 메서드로 설정합니다. 반면 클래스 속성은 클래스 자체에 속하며 해당 클래스의 모든 인스턴스가 공유합니다.

속성에 접근하려면 점 표기법을 사용합니다.

다음은 인스턴스 속성과 클래스 속성의 예시와 객체에서 이를 접근하는 방법입니다:

```python
class Dog:
    species = "French Bulldog" # Class attribute

    def __init__(self, name):
        self.name = name # Instance attribute

print(Dog.species) # French Bulldog

dog1 = Dog("Jack")
print(dog1.name)    # Jack
print(dog1.species) # French Bulldog

dog2 = Dog("Tom")
print(dog2.name)    # Tom
print(dog2.species) # French Bulldog
```

클래스 속성은 클래스 자체에서 직접 접근할 수 있지만, 인스턴스 속성은 객체를 생성하고 데이터를 전달한 후에야 접근할 수 있다는 점을 기억하세요.

자동차는 좋은 예입니다. 모든 자동차는 모델과 색상이 있습니다:

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

car_1 = Car("red", "Toyota Corolla")
car_2 = Car("green", "Lamborghini Revuelto")

print(car_1.model) # Toyota Corolla
print(car_2.model) # Lamborghini Revuelto

print(car_1.color) # red
print(car_2.color) # green
```

메서드는 클래스 내부에 정의된 함수입니다. 이를 통해 클래스에서 정의된 어떤 객체든 자신의 데이터를 조작하거나 작동하는 동작을 수행할 수 있습니다. 메서드에도 점 표기법으로 접근합니다.

예를 들어, 개는 짖을 수 있습니다. 그래서 이전 수업에서 본 것처럼 `bark` 메서드를 `Dog` 클래스에 둘 수 있습니다:

```python
class Dog:
   species = "French Bulldog"

   def __init__(self, name):
     self.name = name

   def bark(self):
       return f"{self.name} says woof woof!"

jack = Dog("Jack")
jill = Dog("Jill")

print(jack.bark()) # Jack says woof woof!
print(jill.bark()) # Jill says woof woof!
```

`Car` 클래스도 `describe` 메서드를 가질 수 있습니다:

```python
class Car:
    def __init__(self, color, model):
        self.color = color  # Instance attribute
        self.model = model  # Instance attribute

    def describe(self):
        return f"This car is a {self.color} {self.model}"

car_1 = Car("red", "Toyota Corolla")
car_2 = Car("green", "Lamborghini Revuelto")

print(car_1.describe()) # This car is a red Toyota Corolla
print(car_2.describe()) # This car is a green Lamborghini Revuelto
```
