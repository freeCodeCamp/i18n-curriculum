# --description--

更复杂的条件式语句可能包含多个条件。这就是为什么使用逻辑操作符来组合多个条件。逻辑操作符是 `&&`、`||` 和 `!`，它们分别用来表现 `and`、`or` 和 `not`。

逻辑操作符 `||` 用于组合两个布尔条件。如果至少有一个条件为 `true`，它就返回 `true`。否则，它返回 `false`：

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a > b || a > c) {
  console.log("At least one of the conditions is true");
} else {
  console.log("Both of the conditions are false");
}
```

逻辑操作符 `&&` 用于组合两个布尔条件。只有当两个条件都为 `true` 时，它才返回 `true`。否则，它返回 `false`：

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a < b && a < c) {
  console.log("Both of the conditions are true");
} else {
  console.log("At least one of the conditions is false");
}
```

逻辑操作符 `!` 用于取反布尔条件。如果条件为 `false`，它返回 `true`。否则，它返回 `false`：

```javascript
let a = 5;
let b = 10;

if (!(a > b)) {
  console.log("The condition is false");
} else {
  console.log("The condition is true");
}
```

在上述示例中，额外的括号用于使代码更易读。使用它们不是必须的。

# --questions--

## --text--

你的任务是编写一个 `if` 语句，用于检查网页应用中的以下条件，以向用户显示 `Welcome!`：

1. 用户必须拥有高级账户（`isPremium`）或成为成员超过 12 个月（`membershipDuration` > 12 months）。

1. 用户当前不得被服务阻止（`!isBlocked`）。

哪个 if 语句正确地查看这些条件？

## --answers--

```javascript
if (isPremium && membershipDuration > 12 && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (isPremium || (membershipDuration > 12 && !isBlocked)) {
  console.log("Welcome!");
}
```

---

```javascript
if ((isPremium || membershipDuration > 12) && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (!isPremium || membershipDuration <= 12 || isBlocked) {
  console.log("Welcome!");
}
```
