# --explanation--

`Healthy` у цьому контексті означає корисний або позитивний для ситуації, не пов’язаний із фізичним здоров’ям. Наприклад:

`It's healthy to take breaks while working.` — це означає, що робити перерви корисно для продуктивності та добробуту.

`Disagreement` означає ситуацію, коли люди мають різні думки або погляди. Наприклад:

`They had a disagreement about the best way to solve the problem.` — це означає, що вони не поділяли однакову думку щодо рішення.
