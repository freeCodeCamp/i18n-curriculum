# --description--

`Strange`는 평소와 다르거나 예상치 못한 것을 의미합니다. 보통 보는 것과 다른 무언가를 발견하는 것과 같습니다. 예를 들어: `A strange sound in a car might mean a problem.`

`error`는 특히 어떤 기능이 제대로 작동하지 못하게 하는 실수를 뜻합니다. 예를 들어: `An error in a code means there is a mistake that needs fixing.`
