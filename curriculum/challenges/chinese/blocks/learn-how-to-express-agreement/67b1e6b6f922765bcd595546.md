# --explanation--

`Make sure` 意味着采取行动确认某事是正确的或已完成。例如：  

`Please make sure the report is submitted before the deadline.` – 这意味着查看并确保报告按时提交。  

`Reliable` 意味着某物或某人可以被信任在一段时间内表现良好或保持一致。例如：

`The server is reliable; it has been running without any issues for months.` - 这意味着服务器工作一致且不会引起问题。

`More reliable` 意味着某物变得更加值得信赖或可靠。例如：

`The new software update makes the app more reliable.`——这意味着应用在更新后变得更加可信或稳定。
