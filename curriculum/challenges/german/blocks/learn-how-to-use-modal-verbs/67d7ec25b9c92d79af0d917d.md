# --explanation--

`Must have happened` wird verwendet, um eine logische Vermutung über etwas in der Vergangenheit anzustellen.

`Must` drückt starke Gewissheit aus. `Have` + `Past Participle` (wie `happened`) bezieht sich auf etwas in der Vergangenheit. Zum Beispiel:

- `He must have forgotten to update the system.` – Es ist wahrscheinlich, dass er vergessen hat, das Update durchzuführen.

- `They must have made some changes to the app.` – Es ist wahrscheinlich, dass die App aktualisiert oder verbessert wurde.

Jake nutzt Beweise (`unusual activity on the servers`), um zu schließen, dass die Sicherheitslücke `must have happened` letzte Nacht aufgetreten ist. Diese Struktur ist nützlich, um logische Annahmen zu treffen.
