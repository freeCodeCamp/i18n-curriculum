# --description--

동사는 지칭하는 사람에 따라 활용된다는 것을 기억하나요? 기억하지 못했다면, 이제 기억하세요!

몇 개의 도전 과제 전에, 누군가가 당신을 위해 무언가를 할 수 있는지 묻는 동사 `poder`를 사용하는 방법을 배웠습니다.

이제 같은 동사를 활용하여 **내가** 무언가를 할 수 있는지 묻는 방법을 배울 차례입니다. 다음 구조만 사용하면 됩니다:

`Puedo` + 동사 원형 + `...?`

몇 가지 예를 살펴보겠습니다:

- `¿Puedo usar la computadora?` - 컴퓨터를 사용할 수 있나요?

- `¿Puedo usar mi apodo?` - 내 별명을 사용할 수 있나요?

두 예 모두 각각 컴퓨터와 별명을 사용할 수 있는지 허락을 구하는 질문입니다.

# --explanation--

Basti는 `¿Puedo usar mi apodo?`라고 말합니다.

여기서 `puedo`는 동사 `poder`의 1인칭 단수형으로, 누군가 허락을 구할 때 사용합니다.

구조는 다음 패턴을 따릅니다:

`¿Puedo` + 동사 원형 + `...?`

이 경우:

- `Usar`는 '사용하다'를 의미합니다.

- `Mi apodo`는 그의 별명을 가리킵니다.

따라서 Basti는 자신의 별명을 사용해도 되는지 묻고 있습니다.
