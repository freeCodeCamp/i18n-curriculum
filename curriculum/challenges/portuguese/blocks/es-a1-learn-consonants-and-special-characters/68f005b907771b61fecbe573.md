# --description--

A letra `h` é muito interessante. É chamada de `hache` (pronunciada `ah-cheh`).

Acredite se quiser, esta letra é sempre silenciosa em espanhol!

Sim, isso mesmo. Em espanhol, nunca é pronunciado. Faz parte da ortografia mas não afeta a pronúncia de forma alguma.

Por exemplo, a palavra `Hola` ("Hello") soa como `ola`, como se a letra `h` nem estivesse lá porque é completamente silenciosa.
