# --explanation--

`Which` використовується для введення неозначеної відносної умови — речення, яке дає додаткову інформацію про всю ідею перед ним. Воно додає деталі, але не є суттєвим для основного значення речення. Наприклад:

`She missed the deadline, which surprised everyone.` – Умова додає додаткову інформацію про те, як люди відреагували на те, що вона пропустила дедлайн (частина, яка була найважливішою в реченні).

У цьому контексті Райкер використовує `which`, щоб висловити свою думку про додаткові витрати, додаткову інформацію щодо суми перевищення бюджету — те, про що насправді питала Марія.
