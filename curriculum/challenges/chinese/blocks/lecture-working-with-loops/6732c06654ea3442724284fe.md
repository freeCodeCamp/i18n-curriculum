# --interactive--

在之前的课程中，你学习了如何使用 `for` 循环、`for...in` 循环和 `for...of` 循环。在本课中，你将学习 `while` 循环和 `do...while` 循环。

只要条件为真，`while` 循环就会运行一块代码。以下是 `while` 循环的基本语法：

```js
while (condition) {
  // code block to be executed
}
```

条件在执行代码块之前被检查。如果条件为假，代码块将不会被执行。

`while` 循环在你不知道需要运行多少次代码块时非常有用。以下是使用 `while` 循环的示例：

:::interactive_editor

```js
let counter = 0;
while(counter < 5) {
  console.log(counter);
  counter++;
}
```

:::

在这个示例中，我们有一个名为 `counter` 的变量，初始化为 `0`。只要 `counter` 的值小于 `5`，`while` 循环将继续运行。在循环内部，我们将 `counter` 的值记录到控制台，然后将 `counter` 增加 `1`。

另一个类似于 `while` 循环的循环是 `do...while` 循环。以下是基本语法：

```js
do {
  // code block to be executed
} while (condition);
```

`do...while` 循环和 `while` 循环之间的一个关键区别是，`do...while` 循环会在检查条件之前至少执行一次代码块。

如果条件为真，代码块将继续执行。如果条件为假，代码块将停止执行。

下面是使用 `do...while` 循环的示例：

:::interactive_editor

```js
let counter = 0;
do {
  console.log(counter);
  counter++;
} while (counter < 5);  
```

:::

在这个例子中，我们有一个名为 `counter` 的变量，初始化为 `0`。`do...while` 循环将把 `counter` 的值记录到控制台，然后将 `counter` 增加 `1`。执行完这段代码块后，它会查看 `counter` 的值是否小于 `5`。如果是，循环将继续运行。如果不是，循环将停止。

在大多数分支中，你可能更常使用 `while` 循环而不是 `do...while` 循环。然而，了解这两种循环类型及其使用时机是很有必要的。

## --text--

`while` 循环和 `do...while` 循环的主要区别是什么？

## --answers--

`while` 循环至少执行一次代码块，而 `do...while` 循环则不然。
