# --description--

这个任务侧重于关系代词 `that`。它用于引入一个从句，提供关于名词的更多细节。

从句是句子的一部分，至少包含一个主语和一个动词。它可以表达一个完整的意思或为主句添加额外的信息。

代词 `that` 有助于更清楚地指定或定义某物，将其与描述或必要的细节连接起来。

例如，`She read the book that won the Pulitzer Prize last year.` 在这个句子中，`that won the Pulitzer Prize last year` 是由 `that` 引导的关系从句。该从句描述或限定了这本书，帮助区分它与其他书籍。这里，`that` 作为关系代词，将从句与名词 `book` 连接起来，描述了她读的是哪一本特定的书。
