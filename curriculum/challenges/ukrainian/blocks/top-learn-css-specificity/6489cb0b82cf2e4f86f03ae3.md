# --description--

Давайте розглянемо кілька швидких прикладів, щоб уявити, як працює специфічність. Розглянемо наступний HTML і CSS код:

```html
<!-- index.html -->

<div class="main">
  <div class="list subsection"></div>
</div>
```

```css
/* rule 1 */
.subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

У наведеному прикладі обидва правила використовують лише селектори класів, але правило 2 є більш специфічним, оскільки воно використовує більше селекторів класів.
