# --hints--

L'elemento `ul` dovrebbe avere un tag di apertura. Opening tags have this syntax: `<elementName>`.

```js
assert.exists(document.querySelector('ul'));
```

L'elemento `ul` dovrebbe avere un tag di chiusura. Closing tags have a `/` just after the `<` character.

```js
assert.match(code, /<\/ul>/);
```

L'elemento `ul` dovrebbe trovarsi sopra il tag di chiusura del secondo elemento `section`.

```js
const secondSectionLastElemNode =
  document.querySelectorAll('main > section')[1]?.lastElementChild;
assert.equal(secondSectionLastElemNode?.nodeName, 'UL');
```
