# --explanation--

`Really`와 `too`는 둘 다 **강조어**이지만, 의미가 다릅니다. `Really`는 수식하는 형용사를 강화하지만 과도함을 나타내지 않습니다. 반면 `Too`는 과도함을 나타내어 허용되거나 감당할 수 있는 범위를 넘는다는 뜻입니다. 예를 들어:

- `The project is really important.` - 이 표현은 프로젝트가 매우 중요하다는 뜻이지만, 반드시 너무 중요하다는 뜻은 아닙니다.
 
- `This task is too complicated.` - 이 표현은 그것이 적절해야 할 것보다 더 복잡하다는 뜻입니다.
