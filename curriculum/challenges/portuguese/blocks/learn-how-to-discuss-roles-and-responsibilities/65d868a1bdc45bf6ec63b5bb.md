# --description--

`To conduct` significa fazer ou realizar algo, como um projeto ou teste.Por exemplo, `The teacher conducts a test` significa que o professor realiza ou executa um teste.

Uma `investigation` é quando você examina algo muito cuidadosamente para descobrir mais sobre isso.Por exemplo, `The school does investigations to find lost books` significa que a escola procura cuidadosamente para encontrar livros que estão desaparecidos.
