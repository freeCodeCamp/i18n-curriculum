# --description--

素数 $p$ に対して、$S(p) = (\sum (p - k)!)\bmod (p)$ を $1 ≤ k ≤ 5$ の範囲で定義します。

例えば、$p = 7$ の場合、

$$(7 - 1)! + (7 - 2)! + (7 - 3)! + (7 - 4)! + (7 - 5)! = 6! + 5! + 4! + 3! + 2! = 720 + 120 + 24 + 6 + 2 = 872$$

$872\bmod (7) = 4$ なので、$S(7) = 4$ となります。

$5 ≤ p < 100$ の範囲で $\sum S(p) = 480$ であることが確認できます。

$5 ≤ p < {10}^8$ の範囲で $\sum S(p)$ を求めてください。
