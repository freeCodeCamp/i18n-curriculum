# --description--

要存取陣列的元素，你可以使用索引編號。索引編號從 0 開始，所以陣列的第一個元素位於索引 0，第二個元素位於索引 1，依此類推。

例如，要存取 `fruits` 陣列的第一個元素，你可以使用以下程式碼：

```javascript
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits[0]); // Output: apple
console.log(fruits[2]); // Output: orange
```

如果索引超出範圍，JavaScript 將傳回 `undefined`。這表示如果你嘗試存取上述範例中的 `fruits[3]`，它將傳回 `undefined`。
