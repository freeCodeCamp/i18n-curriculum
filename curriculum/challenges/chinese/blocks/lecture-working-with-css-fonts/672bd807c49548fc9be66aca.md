# --description--

排版是排列文字以有效传达信息的艺术与技术。 它涉及文本格式设置和字体选择。  In this lesson, you will learn best practices for using typography in a way that enhances readability, hierarchy, and accessibility. 

我们从可读性开始。 你应选择清晰且简洁的字体，以使设计易于理解。 这对于网站正文尤为重要。 如果字体易于阅读，用户更可能与你内容互动。 网页开发中常用的字体包括 Roboto、Open Sans、Poppins、Lato 和 Barlow。 但还有很多选项，你肯定能找到符合需求的字体。

文本还应足够大，以便所有人阅读。 你应尽量保持行长较短。 每行的具体字符数要取决于你的项目。 但保持行长较短会使内容更易阅读。

你可以利用字体大小，为标题、副标题、段落和其他元素创建视觉层次。 例如，网页上的主标题应使用更大的字体，然后是较小字体的副标题。 这将为层次结构中的每个元素赋予特定的字体大小，帮助用户通过视觉层次浏览内容。

为了创造视觉一致性，你应该最多使用两到三种字体。 使用过多字体会使文本更难阅读，并削弱你的品牌识别度。 这还可能因增加网站加载时间而影响用户体验。

一旦选定了这两种或三种字体，就要在不同页面和元素中保持一致。 这些字体应与你的网页应用、产品或服务的整体设计保持一致。 你还应使用留白来创建视觉平衡并增强可读性。

出于可访问性的目的，你应该遵循《网络内容无障碍指南》。 这些指南规定了文本的推荐对比度。 更高的对比度使文本更易阅读，尤其是对视觉障碍者而言。

你还应提供调整字体大小的选项，并尽可能避免使用复杂字体。 作为一般规则，应尽可能使用简单字体。 如果你的内容易于阅读，用户就更可能与之互动。 关于运用排版的最后一条建议是，确保你的字体在不同屏幕尺寸和设备上正确显示。

排版是创造美观设计和吸引用户参与体验的强大工具。 通过理解排版的基础知识并应用这些最佳实践，你能创造出有效的设计。
