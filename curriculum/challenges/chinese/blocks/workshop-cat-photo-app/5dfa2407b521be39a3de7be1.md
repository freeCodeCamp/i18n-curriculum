# --description--

要在新标签页中打开链接，你可以在锚元素（`a`）上使用 `target` 属性。

`target` 属性指定打开链接文档的位置。 `target="_blank"` 是在新标签页或窗口中打开链接的文档。

这是一个带有 `target` 属性的 `a` 元素的基本语法：

```html
<a href="https://www.freecodecamp.org" target="_blank">freeCodeCamp</a>
```

向锚元素（`a`）的开始标签添加一个值为 `_blank` 的 `target` 属性，用以在新的标签页中打开链接。

# --hints--

你的 `p` 元素应该嵌套一个文本为 `cat photos` 的锚元素（`a`）。 你可能删除了它或者拼写有误。

```js
const anchor = document.querySelectorAll('p > a')[1];
assert.equal(
    anchor?.innerText?.toLowerCase().trim().replace(/\s+/g, ' '), 'cat photos'
);
```

你的锚元素（`a`）没有 `target` 属性。 检查开始标签的名称后面是否有空格和/或所有属性名称之前是否有空格。

```js
assert.isTrue(document.querySelectorAll('a')[1]?.hasAttribute('target'));
```

`target` 属性的值应为 `_blank`。 你要么遗漏了属性值，要么拼写有误。 记住，属性值应该放在引号中。

```js
assert.equal(document.querySelectorAll('a')[1]?.getAttribute('target'), '_blank');
```
