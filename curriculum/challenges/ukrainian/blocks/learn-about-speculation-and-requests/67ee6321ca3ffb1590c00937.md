# --explanation--

`edge case` — це рідкісна або крайня ситуація, яка може спричинити проблеми в програмі. Наприклад:

`We need to test the app with edge cases to make sure it works in all situations.` — потрібно тестувати незвичайні ситуації, щоб переконатися, що застосунок працює правильно.

`Unexpected behavior` трапляється, коли програма робить щось, чого програміст не очікував. Наприклад:

`The app showed unexpected behavior when I tried to open it.` — застосунок не працював так, як очікувалося, коли ви намагалися його відкрити.
