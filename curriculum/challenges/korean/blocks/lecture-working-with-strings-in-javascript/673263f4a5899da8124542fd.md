# --interactive--

`prompt()` 메서드는 JavaScript가 사용자와 상호작용하는 데 중요한 부분입니다. 작은 팝업 대화 상자를 통해 사용자로부터 입력을 받는 가장 간단한 방법 중 하나입니다.

웹페이지가 사용자로부터 이름이나 다른 형태의 텍스트 입력 같은 정보를 필요로 할 때 자주 사용되는 것을 볼 수 있습니다.

그렇다면 `prompt()` 메서드는 정확히 무엇을 할까요? 사용자에게 입력을 요청하는 대화 상자를 열고, 사용자가 입력한 텍스트를 문자열로 반환합니다.

`prompt()` 메서드는 두 개의 인수를 받습니다: 첫 번째는 대화 상자 안에 표시될 메시지로, 보통 사용자가 정보를 입력하도록 유도합니다. 두 번째는 선택 사항인 기본값으로, 입력 필드를 처음에 채워 넣습니다.

```js
prompt(message, default);
```

작동 방식을 보여주는 예시입니다.

**참고**: 이 예시는 아직 배우지 않은 코드를 포함하고 있습니다. 코드를 모두 이해하려고 걱정하지 마세요. 이 예시는 `prompt()` 메서드가 어떻게 작동하는지 보여주고, 페이지가 로드될 때 즉시 프롬프트가 나타나지 않도록 하여 방해가 되지 않게 하기 위한 것입니다. 인터랙티브 편집기가 활성화되어 있다면 직접 시도해 보세요.

:::interactive_editor

```html
<button id="prompt-btn">Show Prompt</button>
<p id="output"></p>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("prompt-btn");
const output = document.getElementById("output");
btn.addEventListener("click", () => {
  const userName = prompt("What is your name?", "Guest");
  output.textContent = "Hello, " + userName + "!";
});
```

:::

이 예시에서 사용자가 버튼을 클릭하면 `prompt()` 메서드가 `What is your name?` 메시지와 처음에 `Guest` 값을 가진 입력 필드가 있는 대화 상자를 표시합니다.

사용자가 이름을 입력하고 "확인"을 누르면 `userName` 변수에 입력한 값이 저장됩니다. 사용자가 "취소"를 누르면 `userName` 변수는 `null`로 설정됩니다. `null`은 사용자가 아무 입력도 하지 않았음을 의미합니다. 출력 단락은 제공된 이름이나 사용자가 취소했을 경우 `null`을 사용해 인사 메시지를 표시합니다. 앞으로의 수업에서 사용자가 프롬프트를 취소할 때 `null`가 표시되지 않도록 하는 기법을 배우게 됩니다.

`prompt()` 메서드는 사용자가 대화 상자와 상호작용할 때까지 스크립트 실행을 멈춘다는 점을 기억하세요.

즉, 사용자가 입력을 제공하고 "확인"을 누르거나 프롬프트를 취소할 때까지 나머지 JavaScript 코드는 실행되지 않습니다.

또 한 가지 고려할 점은 `prompt()`가 빠른 테스트나 작은 애플리케이션에는 유용하지만, 방해가 되고 브라우저마다 동작이 일관되지 않아 현대의 복잡한 웹 애플리케이션에서는 일반적으로 피한다는 것입니다.

`prompt()` 메서드를 이해하면, 비록 현대 웹 애플리케이션에서 널리 사용되지는 않지만, 브라우저를 통해 사용자와 간단히 상호작용하고 정보를 직접 받아오는 방법을 알게 됩니다.
