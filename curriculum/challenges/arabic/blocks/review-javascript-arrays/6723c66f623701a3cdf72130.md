# --interactive--

## أساسيات مصفوفات JavaScript

- **التعريف**: المصفوفة في JavaScript هي مجموعة مرتبة من القيم، يُحدد كل منها بواسطة فهرس رقمي. يمكن أن تكون القيم في مصفوفة JavaScript من أنواع بيانات مختلفة، بما في ذلك الأرقام، والسلاسل النصية، والقيم المنطقية، والكائنات، وحتى مصفوفات أخرى. المصفوفات متجاورة في الذاكرة، مما يعني أن جميع العناصر مخزنة في كتلة واحدة متصلة من مواقع الذاكرة، مما يسمح بفهرسة فعالة ووصول سريع للعناصر عبر الفهرس.

```js
const developers = ["Jessica", "Naomi", "Tom"];
```

- **الوصول إلى عناصر المصفوفات**: للوصول إلى عناصر من مصفوفة، تحتاج إلى الإشارة إلى المصفوفة متبوعة برقم الفهرس داخل أقواس مربعة. مصفوفات JavaScript تبدأ فهارسها من الصفر، مما يعني أن العنصر الأول في الفهرس 0، والعنصر الثاني في الفهرس 1، وهكذا. إذا حاولت الوصول إلى فهرس غير موجود في المصفوفة، فستُرجع JavaScript القيمة `undefined`.

:::interactive_editor

```js
const developers = ["Jessica", "Naomi", "Tom"];
console.log(developers[0]) // "Jessica"
console.log(developers[1]) // "Naomi"

console.log(developers[10]) // undefined
```

:::

- **خاصية `length`**: تُستخدم هذه الخاصية لإرجاع عدد العناصر في المصفوفة.

:::interactive_editor

```js
const developers = ["Jessica", "Naomi", "Tom"];
console.log(developers.length) // 3
```

:::

- **تحديث عناصر في المصفوفة**: لتحديث عنصر في مصفوفة، استخدم معامل التعيين (`=`) لتعيين قيمة جديدة للعنصر عند فهرس معين.

:::interactive_editor

```js
const fruits = ['apple', 'banana', 'cherry'];
fruits[1] = 'blueberry';

console.log(fruits); // ['apple', 'blueberry', 'cherry']
```

:::

## المصفوفات ثنائية الأبعاد

- **التعريف**: المصفوفة ثنائية الأبعاد هي في الأساس مصفوفة من مصفوفات. تُستخدم لتمثيل بيانات ذات هيكل شبكي طبيعي، مثل رقعة الشطرنج، أو جدول بيانات، أو بكسلات في صورة. للوصول إلى عنصر في مصفوفة ثنائية الأبعاد، تحتاج إلى فهرسين: واحد للصف وآخر للعمود.

:::interactive_editor

```js
const chessboard = [
    ['R', 'N', 'B', 'Q', 'K', 'B', 'N', 'R'],
    ['P', 'P', 'P', 'P', 'P', 'P', 'P', 'P'],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    ['p', 'p', 'p', 'p', 'p', 'p', 'p', 'p'],
    ['r', 'n', 'b', 'q', 'k', 'b', 'n', 'r']
];

console.log(chessboard[0][3]); // "Q"
```

## التفكيك في المصفوفات

- **التعريف**: التفكيك في المصفوفات هو ميزة في JavaScript تسمح لك باستخراج القيم من المصفوفات وتعيينها إلى متغيرات بطريقة أكثر إيجازًا ووضوحًا. توفر صيغة نحوية مريحة لفك عناصر المصفوفة إلى متغيرات منفصلة.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange"];

const [first, second, third] = fruits;

console.log(first); // "apple"
console.log(second); // "banana"
console.log(third); // "orange"
```

:::

- **صيغة rest**: تتيح لك هذه الصيغة التقاط العناصر المتبقية من المصفوفة التي لم تُفكك في مصفوفة جديدة.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange", "mango", "kiwi"];
const [first, second, ...rest] = fruits;

console.log(first); // "apple"
console.log(second); // "banana"
console.log(rest); // ["orange", "mango", "kiwi"]
```

:::

## طرق المصفوفة الشائعة

- **طريقة `push()`**: تُستخدم هذه الطريقة لإضافة عناصر إلى نهاية المصفوفة وستُرجع الطول الجديد.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.push("ice cream");

console.log(desserts); // ["cake", "cookies", "pie", "ice cream"];
```

:::

- **طريقة `pop()`**: تُستخدم هذه الطريقة لإزالة العنصر الأخير من المصفوفة وستُرجع هذا العنصر المُزال. إذا كانت المصفوفة فارغة، فستكون قيمة الإرجاع `undefined`.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.pop();

console.log(desserts); // ["cake", "cookies"];
```

:::

- **طريقة `shift()`**: تُستخدم هذه الطريقة لإزالة العنصر الأول من المصفوفة وإرجاع هذا العنصر المُزال. إذا كانت المصفوفة فارغة، فستكون قيمة الإرجاع `undefined`.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.shift();

console.log(desserts); // ["cookies", "pie"];
```

:::

- **طريقة `unshift()`**: تُستخدم هذه الطريقة لإضافة عناصر إلى بداية المصفوفة وستُرجع الطول الجديد.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.unshift("ice cream");

console.log(desserts); // ["ice cream", "cake", "cookies", "pie"];
```

:::

- **طريقة `indexOf()`**: تُستخدم هذه الطريقة للعثور على أول فهرس لعنصر معين داخل المصفوفة. إذا لم يُعثر على العنصر، فستُرجع `-1`.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange", "banana"];
const index = fruits.indexOf("banana");

console.log(index); // 1
console.log(fruits.indexOf("not found")); // -1
```

:::

- **طريقة `splice()`**: تُستخدم هذه الطريقة لإضافة أو إزالة عناصر من أي موقع في المصفوفة. قيمة الإرجاع لطريقة `splice()` ستكون مصفوفة بالعناصر التي أُزيلت من المصفوفة. إذا لم يُزل شيء، فستُرجع مصفوفة فارغة. هذه الطريقة تُغير المصفوفة الأصلية في مكانها بدلاً من إنشاء مصفوفة جديدة. المعلمة الأولى تحدد الفهرس الذي يبدأ عنده تعديل المصفوفة. المعلمة الثانية هي عدد العناصر التي تريد إزالتها. المعلمات التالية هي العناصر التي تريد إضافتها.

:::interactive_editor

```js
const colors = ["red", "green", "blue"];
colors.splice(1, 0, "yellow", "purple");

console.log(colors); // ["red", "yellow", "purple", "green", "blue"]
```

:::

- **طريقة `includes()`**: تُستخدم هذه الطريقة للتحقق مما إذا كانت المصفوفة تحتوي على قيمة معينة. تُرجع هذه الطريقة `true` إذا كانت المصفوفة تحتوي على العنصر المحدد، و`false` خلاف ذلك.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];

console.log(programmingLanguages.includes("Python")); // true
console.log(programmingLanguages.includes("Perl")); // false
```

:::

- **طريقة `concat()`**: تُنشئ هذه الطريقة مصفوفة جديدة بدمج مصفوفتين أو أكثر.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];
const newList = programmingLanguages.concat("Perl");

console.log(newList); // ["JavaScript", "Python", "C++", "Perl"]
```

:::

- **طريقة `slice()`**: تُرجع هذه الطريقة مصفوفة جديدة تحتوي على نسخة سطحية من جزء من المصفوفة الأصلية، محددة بفهارس البداية والنهاية. تحتوي المصفوفة الجديدة على مراجع لنفس العناصر في المصفوفة الأصلية (وليس نسخًا مكررة). هذا يعني أنه إذا كانت العناصر بدائية (مثل الأرقام أو السلاسل النصية)، تُنسخ القيم؛ أما إذا كانت العناصر كائنات أو مصفوفات، فتُنسخ المراجع وليس الكائنات نفسها.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];
const newList = programmingLanguages.slice(1);

console.log(newList); // ["Python", "C++"]
```

:::

- **صيغة النشر**: تُستخدم صيغة النشر لإنشاء نسخ سطحية من المصفوفة.

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const shallowCopiedArray = [...originalArray];

shallowCopiedArray.push(4);

console.log(originalArray); // [1, 2, 3]
console.log(shallowCopiedArray); // [1, 2, 3, 4]
```

:::

- **طريقة `split()`**: تُقسّم هذه الطريقة سلسلة نصية إلى مصفوفة من السلاسل الفرعية وتحدد مكان كل تقسيم بناءً على فاصل معين. إذا لم يُحدد فاصل، تُرجع الطريقة مصفوفة تحتوي على السلسلة النصية الأصلية كعنصر واحد.

:::interactive_editor

```js
const str = "hello";
const charArray = str.split("");

console.log(charArray); // ["h", "e", "l", "l", "o"]
```

:::

- **طريقة `reverse()`**: تُعكس هذه الطريقة ترتيب عناصر المصفوفة في مكانها.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
console.log(desserts.reverse()); // ["pie", "cookies", "cake"]
```

:::

- **طريقة `join()`**: تُدمج هذه الطريقة جميع عناصر المصفوفة في سلسلة نصية واحدة، مع فصل كل عنصر بفاصل محدد. إذا لم يُحدد فاصل، أو استُخدم سلسلة فارغة (`""`)، تُدمج العناصر بدون أي فاصل.

:::interactive_editor

```js
const reversedArray = ["o", "l", "l", "e", "h"];
const reversedString = reversedArray.join("");

console.log(reversedString); // "olleh"
```

:::
