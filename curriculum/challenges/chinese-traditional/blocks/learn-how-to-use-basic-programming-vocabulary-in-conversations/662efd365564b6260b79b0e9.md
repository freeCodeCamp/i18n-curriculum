# --description--

`loop` 是一連串指令，會持續重複直到達到某個條件。`loops` 用來高效地多次執行相同的程式碼。常見的 `loops` 型別包括 `for` 迴圈和 `while` 迴圈。 

例如，`for loop`：`for (int i = 0; i < 5; i++) { /* code to be executed */ }`

在範例中，大括號 `{ }` 內的程式碼會重複執行，直到條件不再為真。
