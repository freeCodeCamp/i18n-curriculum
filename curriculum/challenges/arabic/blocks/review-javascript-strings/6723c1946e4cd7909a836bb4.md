# --interactive--

## أساسيات السلاسل النصية

- **التعريف**: السلسلة النصية هي متتالية من الأحرف محاطة إما بعلامات اقتباس مفردة، أو علامات اقتباس مزدوجة، أو علامات دقيقة. السلاسل النصية هي أنواع بيانات بدائية ولا يمكن تغييرها. عدم القابلية للتغيير يعني أنه بمجرد إنشاء السلسلة النصية، لا يمكن تعديلها.
- **الوصول إلى الأحرف من السلسلة النصية**: للوصول إلى حرف من سلسلة نصية يمكنك استخدام ترميز الأقواس وتمرير رقم الفهرس. الفهرس هو موقع الحرف داخل السلسلة النصية، وهو يبدأ من الصفر.

:::interactive_editor

```js
const developer = "Jessica";
console.log(developer[0]); // J
```

:::

- **`\n` (حرف السطر الجديد)**: يمكنك إنشاء سطر جديد في السلسلة النصية باستخدام حرف السطر الجديد `\n`.

:::interactive_editor

```js
const poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

- **الهروب من الأحرف في السلاسل النصية**: يمكنك الهروب من الأحرف في السلسلة النصية بوضع شرطات مائلة عكسية (`\`) أمام علامات الاقتباس.

:::interactive_editor

```js
const statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

## القوالب الحرفية (قوالب السلاسل النصية) واستيفاء السلاسل النصية

- **التعريف**: تُعرّف القوالب الحرفية باستخدام العلامات الدقيقة (`). تسمح بتسهيل التعامل مع السلاسل النصية، بما في ذلك تضمين المتغيرات مباشرة داخل السلسلة النصية، وهي ميزة تعرف باسم استيفاء السلاسل النصية.

:::interactive_editor

```js
const name = "Jessica";
const greeting = `Hello, ${name}!`; 
console.log(greeting); // "Hello, Jessica!"
```

:::

## ASCII، طريقة `charCodeAt()` وطريقة `fromCharCode()`

- **ASCII**: ASCII (الرمز القياسي الأمريكي لتبادل المعلومات) هو معيار ترميز الأحرف يُستخدم لتمثيل الأحرف الإنجليزية الأساسية باستخدام قيم رقمية. الدروس السابقة قدمت `charCodeAt()` و`fromCharCode()` باستخدام أمثلة ASCII.
- **Unicode**: تستخدم سلاسل JavaScript الترميز الداخلي Unicode، وتحديدًا ترميز UTF-16. بالنسبة لأول 128 حرفًا (الحروف اللاتينية الأساسية، الأرقام، والرموز الشائعة)، تتطابق قيم Unicode مع رموز ASCII. لهذا السبب تستمر أمثلة ASCII في العمل في JavaScript.
- **طريقة `charCodeAt()`**: تعيد هذه الطريقة وحدة الترميز UTF-16 للحرف عند فهرس محدد. بالنسبة للحروف اللاتينية الأساسية، تتطابق هذه القيمة مع رمز ASCII.

:::interactive_editor

```js
const letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

:::

- **طريقة `fromCharCode()`**: تحول هذه الطريقة رمز ASCII إلى الحرف المقابل له.

:::interactive_editor

```js
const char = String.fromCharCode(65);
console.log(char);  // A
```

:::

## طرق أخرى شائعة للسلاسل النصية

- **طريقة `indexOf`**: تُستخدم هذه الطريقة للبحث عن سلسلة فرعية داخل سلسلة نصية. إذا وُجدت السلسلة الفرعية، تعيد `indexOf` الفهرس (أو الموقع) لأول ظهور لتلك السلسلة الفرعية. إذا لم تُعثر على السلسلة الفرعية، تعيد `indexOf` القيمة -1، مما يشير إلى فشل البحث.

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.indexOf("fox")); // 16
console.log(text.indexOf("cat")); // -1
```
  
:::  

- **طريقة `includes()`**: تُستخدم هذه الطريقة للتحقق مما إذا كانت السلسلة النصية تحتوي على سلسلة فرعية محددة. إذا وُجدت السلسلة الفرعية داخل السلسلة النصية، تعيد الطريقة القيمة true. وإلا، تعيد false.

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.includes("fox")); // true
console.log(text.includes("cat")); // false
```

:::

- **طريقة `slice()`**: تستخرج هذه الطريقة جزءًا من السلسلة النصية وتُعيد سلسلة جديدة، دون تعديل السلسلة الأصلية. تأخذ معلمتين: فهرس البداية والفهرس الاختياري للنهاية.

:::interactive_editor

```js
const text = "freeCodeCamp";
console.log(text.slice(0, 4));  // "free"
console.log(text.slice(4, 8));  // "Code"
console.log(text.slice(8, 12)); // "Camp"
```

:::

- **طريقة `toUpperCase()`**: تحول هذه الطريقة جميع الأحرف إلى أحرف كبيرة وتُعيد سلسلة جديدة تحتوي على جميع الأحرف الكبيرة.

:::interactive_editor

```js
const text = "Hello, world!";
console.log(text.toUpperCase()); // "HELLO, WORLD!"
```

:::

- **طريقة `toLowerCase()`**: تحول هذه الطريقة جميع الأحرف في السلسلة النصية إلى أحرف صغيرة.

:::interactive_editor

```js
const text = "HELLO, WORLD!"
console.log(text.toLowerCase()); // "hello, world!"
```

:::

- **طريقة `replace()`**: تتيح لك هذه الطريقة العثور على قيمة محددة (مثل كلمة أو حرف) في السلسلة النصية واستبدالها بقيمة أخرى. تعيد الطريقة سلسلة جديدة مع الاستبدال وتترك الأصلية دون تغيير لأن سلاسل JavaScript غير قابلة للتغيير.

:::interactive_editor

```js
const text = "I like cats";
console.log(text.replace("cats", "dogs")); // "I like dogs"
```

:::

- **طريقة `replaceAll()`**: تتيح لك هذه الطريقة العثور على كل تكرارات قيمة محددة (كلمة، حرف، أو نمط) في السلسلة النصية واستبدالها بقيمة أخرى. تعمل مثل `replace()`، لكنها لا تتوقف عند أول تطابق، بل تحدث كل التطابقات الموجودة في السلسلة النصية.

:::interactive_editor

```js
const text = "I love cats and cats are so much fun!";
console.log(text.replaceAll("cats", "dogs")); // "I love dogs and dogs are so much fun!"
```

:::

- **طريقة `repeat()`**: تُستخدم هذه الطريقة لتكرار السلسلة النصية عددًا محددًا من المرات.

:::interactive_editor

```js
const text = "Hello";
console.log(text.repeat(3)); // "HelloHelloHello"
```

:::

- **طريقة `trim()`**: تُستخدم هذه الطريقة لإزالة الفراغات من بداية ونهاية السلسلة النصية.

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trim()); // "Hello, world!"
```

:::

- **طريقة `trimStart()`**: تزيل هذه الطريقة الفراغات من بداية (أو "بداية") السلسلة النصية.

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trimStart()); // "Hello, world!  "
```

:::

- **طريقة `trimEnd()`**: تزيل هذه الطريقة الفراغات من نهاية السلسلة النصية.

:::interactive_editor

```js
const text = " Hello, world! ";
console.log(text.trimEnd()); // "  Hello, world!"
```

:::

- **طريقة `prompt()`**: تُستخدم هذه الطريقة من كائن `window` للحصول على معلومات من المستخدم عبر مربع حوار. تأخذ هذه الطريقة معلمتين. المعلمة الأولى هي الرسالة التي تظهر داخل مربع الحوار، عادةً لتوجيه المستخدم لإدخال معلومات. المعلمة الثانية هي قيمة افتراضية اختيارية تملأ حقل الإدخال مبدئيًا.

```js
const answer = window.prompt("What's your favorite animal?"); // This will change depending on what the user answers
```
