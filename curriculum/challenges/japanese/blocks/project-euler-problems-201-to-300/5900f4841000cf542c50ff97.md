# --description--

勤勉なアリが5x5のグリッド上をランダムに歩きます。歩行は中央のマスから始まります。各ステップで、アリはグリッドを出ることなく隣接するマスにランダムに移動します。そのため、アリの位置によっては2、3、または4つの移動先が可能です。

歩行の開始時に、下段の各マスに種が置かれます。アリが種を持っていない状態で、種がある下段のマスに到達すると、その種を持ち始めます。アリは最終的に到達する上段の最初の空いているマスに種を落とします。

すべての種が上段に落とされるまでの期待ステップ数はいくつでしょうか？答えは小数点以下6桁に丸めてください。
