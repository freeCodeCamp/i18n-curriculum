# --description--

표현식 `That/it sounds`는 무언가가 어떻게 보이거나 나타나는지를 설명하는 데 사용되며, 주로 들은 정보나 주는 인상에 기반합니다. 예를 들어, `That sounds like a great idea!`는 들은 내용을 바탕으로 그 아이디어가 좋아 보인다는 것을 나타냅니다.

마찬가지로, `That/it looks`는 `It looks beautiful.`처럼 외관에 기반해 무언가가 어떻게 보이는지를 설명하는 데 사용할 수 있습니다.

단어 `convenient`는 자신의 필요에 적합하고 사용하기 쉽거나 일을 더 쉽고 빠르게 할 수 있도록 돕는 것을 설명하는 형용사입니다. 주로 작업을 덜 어렵게 하거나 시간을 절약하는 것을 의미합니다.

예를 들어, `Online shopping is convenient because you can do it from home.`
