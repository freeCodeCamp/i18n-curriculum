# --description--

Давайте повторимо теперішній доконаний час у цьому завданні. Його використовують, щоб описати дії або ситуації, які відбулися в невизначений час до теперішнього моменту. Він утворюється за допомогою `have/has` + дієприкметник минулого часу. Наприклад, `She has visited Paris.`

Фраза `spend time on` використовується, щоб описати, скільки часу витрачено на певну діяльність, завдання або предмет. Приклад: `She spends a lot of time on her homework.` Дієприкметник минулого часу від `spend` — це `spent`.

Послухайте Брайана і заповніть пропуск.
