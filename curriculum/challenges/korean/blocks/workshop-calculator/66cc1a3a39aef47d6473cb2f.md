# --description--

다음 단계는 계산기에 나누기 기능을 추가하는 것입니다.

두 개의 매개변수 `calculateQuotient`과 `num1`를 받는 `num2`라는 함수를 선언하세요.

함수 내부에서 두 매개변수의 몫을 반환하세요.

그다음, `console.log`와 `calculateQuotient` 인수를 사용하여 `7` 함수를 호출하는 `11`를 추가하세요.
