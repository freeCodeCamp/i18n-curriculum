# --description--

`automate` 意味着使一个进程自动运行，无需人为帮助。在编程中，这可能涉及使用软件编写代码、测试应用程序或查找并修复误差。例如，早晨自动开启制作咖啡的咖啡机就是 `automated`。

单词 `certain` 用于指特定的事物或人，但不是所有的事物或人。例如，如果你说 `certain books are on sale`，意思是只有部分书籍在促销，而不是全部。
