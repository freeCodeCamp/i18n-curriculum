# --description--

`introductory` 零件是文件的開始，介紹主要主題。範例：`The introductory chapter of the book outlines the key concepts.`

`paragraph` 是關於特定主題的寫作零件。比如，`Each paragraph in the blog post focuses on a different technology trend.` 

`To cover` 意思是處理或涵蓋特定內容。 例如，在技術指南中，`This section covers the steps to install software.`
