# --description--

Множини — це одна з вбудованих структур даних у Python. Однією з основних характеристик множин є те, що вони не зберігають дублікати значень. Якщо ви спробуєте додати дублікат до множини, буде збережено лише одне таке значення.

Множини змінні та неупорядковані, тобто їхні елементи не зберігаються у певному порядку, тому ви не можете використовувати індекси або ключі для доступу до них. Вони можуть містити лише значення незмінних типів даних, таких як числа, рядки та кортежі. Також множини підтримують математичні операції над множинами, включно з об’єднанням, перетином, різницею та симетричною різницею.

Щоб визначити множину, достатньо записати її елементи у фігурних дужках, розділяючи їх комами. Ось приклад множини чисел:

```python
my_set = {1, 2, 3, 4, 5} 
```

Особливість роботи з множинами полягає в тому, що якщо вам потрібно визначити порожню множину, слід використати функцію `set()`. Якщо ж просто написати порожні фігурні дужки, як `{}`, Python автоматично створить словник.

```python
set() # Set
{}    # Dictionary
```

Ви можете додати елемент до множини за допомогою методу `.add()`, передавши новий елемент як аргумент:

```python
my_set.add(6)
```

У нашому прикладі нова множина буде такою:

```python
{1, 2, 3, 4, 5, 6}
```

Якщо ви спробуєте додати елемент, який уже є в множині, буде збережено лише один екземпляр. У цьому випадку число 5 вже є в множині:

```python
my_set.add(5)
```

Отже, множина не зміниться:

```python
{1, 2, 3, 4, 5, 6}
```

Щоб видалити елемент із множини, у вас є два варіанти. Ви можете використати метод `.remove()` або метод `.discard()`, передавши елемент, який хочете видалити, як аргумент.

Метод `.remove()` викличе помилку `KeyError`, якщо елемент не знайдено, тоді як метод `.discard()` — ні:

```python
my_set.remove(4)
my_set.discard(4)
```

Метод `.clear()` видаляє всі елементи з множини:

```python
my_set.clear()
```

Множини Python також мають потужні методи для виконання загальних математичних операцій над множинами.

Методи `.issubset()` і `.issuperset()` перевіряють, чи є множина підмножиною або надмножиною іншої множини відповідно.

Тут ми перевіряємо, чи є `your_set` підмножиною `my_set`, що є `False`, бо не всі елементи `your_set` містяться в `my_set`.

Також ми перевіряємо, чи є `my_set` надмножиною `your_set`. Це також `False`, бо `my_set` не містить усіх елементів `your_set`:

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

print(your_set.issubset(my_set)) # False
print(my_set.issuperset(your_set)) # False
```

Метод `.isdisjoint()` перевіряє, чи є дві множини диз’юнктними, тобто чи не мають вони спільних елементів. У цьому випадку це `False`, бо `my_set` і `your_set` мають спільні елементи – 2, 3 і 4:

```python
print(my_set.isdisjoint(your_set)) # False
```

Оператор об’єднання `|` повертає нову множину з усіма елементами обох множин:

```python
my_set | your_set # {1, 2, 3, 4, 5, 6}
```

Оператор перетину `&` повертає нову множину лише з тими елементами, які є в обох множинах:

```python
my_set & your_set # {2, 3, 4}
```

Оператор різниці `-` повертає нову множину з елементами першої множини, яких немає в інших множинах. У цьому прикладі числа 1 і 5 є в `my_set`, але НЕ в `your_set`:

```python
my_set - your_set # {1, 5}
```

Оператор симетричної різниці `^` повертає нову множину з елементами, які є або в першій, або в другій множині, але не в обох одночасно. У цьому випадку 1 і 5 є в `my_set`, але не в `your_set`, тому вони включені. А число 6 є в `your_set`, але не в `my_set`, тому воно також включене:

```python
my_set ^ your_set # {1, 5, 6}
```

Кожен із цих операторів має відповідний оператор складеного присвоєння, якщо додати поруч знак «дорівнює». Ці оператори автоматично оновлюють першу множину в виразі результатом операції:

```python
|= &= -= ^=
```

Наприклад, оператор `-=` знаходить різницю між множинами і оновлює першу множину цим результатом:

```python
my_set -= your_set
```

Після цього `my_set` буде оновлено до `{1, 5}`:

```python
print(my_set) # {1, 5}
```

Ви можете перевірити, чи є елемент у множині, за допомогою оператора `in`. Тут ми перевіряємо, чи є 5 у `my_set`. Результатом буде булеве значення `True` або `False`:

```python
print(5 in my_set)
```

Ось і основи множин. Вони дуже корисні, коли не потрібно зберігати значення у певному порядку, а також коли потрібно зберігати лише унікальні значення.
