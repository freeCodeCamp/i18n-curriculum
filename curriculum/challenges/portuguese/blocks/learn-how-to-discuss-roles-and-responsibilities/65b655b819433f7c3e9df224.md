# --description--

A frase `word is` é usada quando se compartilha informações ou rumores que você ouviu de outras pessoas. É como dizer `people say` ou `I've heard`.

Por exemplo, se alguém diz, `Word is that the new update will make the program faster`, eles querem dizer que ouviram que a nova atualização vai melhorar a velocidade do programa.
