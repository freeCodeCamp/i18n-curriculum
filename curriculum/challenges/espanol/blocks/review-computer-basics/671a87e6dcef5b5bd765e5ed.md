## Comprender los conceptos básicos de computación, internet y herramientas para desarrolladores

- **Placa base**: contiene toda la memoria, conectores y discos duros necesarios para que funcione la computadora. Sirve como la placa de circuito principal del equipo.
- **Unidad central de procesamiento (CPU)**: un procesador responsable de ejecutar instrucciones y realizar cálculos.
- **Memoria de acceso aleatorio (RAM)**: un lugar de almacenamiento temporal para la CPU de la computadora.
- **Disco duro (HDD)**: un lugar de almacenamiento permanente que se usa para guardar datos incluso cuando la computadora está apagada.
- **Unidad de estado sólido (SSD)**: memoria flash no volátil que puede usarse en lugar de un disco duro.
- **Unidad de fuente de poder (PSU)**: responsable de convertir la electricidad del enchufe en una forma que la computadora pueda usar.
- **Unidad de procesamiento gráfico (GPU)**: responsable de renderizar los elementos visuales en la pantalla de la computadora.
- **Diferentes tipos de proveedores de servicios de internet**: un proveedor de servicios de internet (ISP) es una empresa que ofrece acceso a internet. Hay diferentes tipos de ISP, incluyendo dial-up, DSL, cable, fibra óptica y satélite.
- **Formas seguras de iniciar sesión en tu computadora**: ejemplos de formas seguras para iniciar sesión incluyen usar una contraseña fuerte, habilitar la autenticación de dos factores y usar un administrador de contraseñas.
- **Entorno de desarrollo integrado (IDE)**: una herramienta que ayuda a los desarrolladores a escribir, probar y depurar código de manera eficiente.
- **Editor de código**: una herramienta que los desarrolladores usan para escribir y depurar código.
- **Git**: un sistema de control de versiones popular que permite a los desarrolladores rastrear cambios en su código y colaborar con otros.
- **Servicios de hosting en la nube para repositorios**: un repositorio es un lugar de almacenamiento para archivos de proyectos e historial de versiones. Los servicios de hosting en la nube populares para repositorios incluyen GitHub, GitLab y Bitbucket.
- **Gestores de paquetes**: herramientas que ayudan a los desarrolladores a simplificar el proceso de agregar, actualizar y eliminar bibliotecas y dependencias de proyectos. Ejemplos incluyen npm, pip y Maven.
- **Bibliotecas y frameworks de pruebas**: las pruebas se realizan en el software para asegurar que el código funcione como se espera. Ejemplos de bibliotecas y frameworks de pruebas incluyen Jest, PHPUnit y JUnit.

## Trabajar con archivos, sistemas de archivos y formatos de medios

- **Buenas prácticas para nombrar archivos**: querrás nombrar tus archivos de una manera que sea fácil de entender y mantener. Por ejemplo, `about-us.html` es un nombre más descriptivo que `page1.html`.
- **Directorio raíz**: directorio de nivel superior en un sistema de archivos. Directorio es otro nombre para carpeta.
- **Markdown**: un lenguaje de marcado comúnmente usado para documentación y archivos `README`. Un archivo `README` es un archivo que contiene información sobre un proyecto, como cómo instalarlo y usarlo.
- **`index.html`**: representa la página predeterminada que se muestra cuando un usuario visita un sitio web.
- **Crear, mover y eliminar archivos y carpetas usando Explorer/Finder**: Explorer es el administrador de archivos en Windows, y Finder es el administrador de archivos en macOS. Puedes usar estas herramientas para crear, mover y eliminar archivos y carpetas.
- **Buscar archivos y carpetas**: puedes usar la función de búsqueda en Explorer o Finder para encontrar archivos y carpetas en tu computadora.
- **Tipos de archivos HTML, CSS y JS**: la extensión `.html` se usa para archivos HTML, `.css` para archivos CSS y `.js` para archivos JavaScript.
- **Formatos comunes de imágenes y gráficos**: `JPEG` y `PNG` son formatos comunes de archivos de imagen. `GIF` es otro formato común de imagen que soporta animación. `SVG` es un formato de archivo para gráficos vectoriales.
- **Formatos comunes de audio y video**: el formato `MP3` se usa comúnmente para archivos de audio. El formato `MP4` se usa comúnmente para archivos de video. El formato `MOV` fue desarrollado por Apple y se usa comúnmente para archivos de video.
- **Formatos comunes de fuentes**: el formato `TTF` se usa comúnmente para fuentes TrueType. El formato `WOFF` se usa comúnmente para fuentes web. El sucesor de `WOFF` es `WOFF2`, que ofrece mejor compresión.
- **ZIP**: un formato de archivo que se usa para comprimir archivos y carpetas.

## Navegar por la web de manera efectiva

- **¿Qué es un navegador web?**: una aplicación de software que permite a los usuarios acceder y ver sitios web en internet.
- **¿Qué es un motor de búsqueda?**: una herramienta que permite a los usuarios buscar información en internet. Ejemplos incluyen Google, Bing y Yahoo.
- **Navegadores web comunes**: algunos ejemplos de navegadores web comunes incluyen Google Chrome, Mozilla Firefox y Microsoft Edge.
- **Estrategias comunes de búsqueda**: puedes usar `site:` seguido de la URL de un sitio web para buscar contenido en ese sitio. Puedes usar `filetype:` seguido de una extensión de archivo para buscar archivos de ese tipo. Puedes anteponer un signo menos a un término de búsqueda para excluir resultados que contengan ese término. Puedes anteponer un signo más a un término de búsqueda para incluir resultados que contengan ese término.
