# --interactive--

تُستخدم حلقة `for...of` عندما تحتاج إلى التكرار على قيم من قابل للتكرار. من أمثلة القابل للتكرار المصفوفات والسلاسل النصية.

إليك الصيغة النحوية الأساسية لحلقة `for...of`:

```js
for (variable of iterable) {
  // code block to be executed
}
```

المتغير في المثال يُمثل القيمة الحالية من القابل للتكرار الذي يتم التكرار عليه.

إذا كان لديك مصفوفة من الأرقام، فالمتغير سيكون الرقم الحالي في المصفوفة. وإذا كان لديك سلسلة نصية، فالمتغير سيكون الحرف الحالي في السلسلة.

لنلقِ نظرة على بعض الأمثلة لتفهم كيف تعمل حلقة `for...of` بشكل أفضل.

في هذا المثال الأول لدينا مصفوفة من الأرقام ونريد التكرار على كل رقم وتسجيله في وحدة التحكم.

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

:::

أنشأنا متغيرًا يسمى `num` سيمثل الرقم الحالي في المصفوفة. في التكرار الأول، سيكون `num` هو `1`، وفي التكرار الثاني سيكون `num` هو `2`، وهكذا.

داخل الحلقة، نسجل الرقم الحالي في وحدة التحكم.

إليك مثال آخر حيث لدينا سلسلة نصية ونريد التكرار على كل حرف وتسجيله في وحدة التحكم.

:::interactive_editor

```js
const str = 'freeCodeCamp';

for (let char of str) {
  console.log(char);
}
```

:::

في هذا المثال، أنشأنا متغيرًا يسمى `char` سيمثل الحرف الحالي في السلسلة النصية.

في كل تكرار، ستسجل الحلقة الحرف الحالي في وحدة التحكم.

من المهم أن تعرف أنه يمكنك استخدام `let` أو `const` عند إعلان المتغير في حلقة `for...of`.

إذا كنت ستستخدم `const`، فتأكد من أن قيمة المتغير لا تتغير داخل الحلقة. إذا تغيرت، ستحصل على خطأ.

إليك مثالًا لاستخدام `const` يؤدي إلى حدوث خطأ:

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
  num = num + 1; // This will cause an error
}
```

في هذا المثال، نحاول تغيير قيمة `num` داخل الحلقة. وبما أننا أعلنّا `num` باستخدام `const`، فسنحصل على خطأ. لذلك، إذا كنت بحاجة إلى تغيير قيمة المتغير داخل الحلقة، استخدم `let` بدلاً من ذلك.

لنلقِ نظرة على مثال أخير يتعامل مع مصفوفة من الكائنات.

:::interactive_editor

```js
const people = [
  { name: 'John', age: 30 },
  { name: 'Jane', age: 25 },
  { name: 'Jim', age: 40 }
];

for (const person of people) {
  console.log(`${person.name} is ${person.age} years old`);
}
```

:::

في هذا المثال، لدينا مصفوفة من الكائنات تسمى `people`. كل كائن يحتوي على خاصيتي `name` و `age`.

عندما نكرر عبر المصفوفة، ننشئ متغيرًا يسمى `person` سيمثل الكائن الحالي في المصفوفة.

داخل الحلقة، نُخرج رسالة إلى وحدة التحكم.

الرسالة الأولى ستكون `John is 30 years old`، والرسالة الثانية ستكون `Jane is 25 years old`، والرسالة الثالثة ستكون `Jim is 40 years old`.

حلقات `for...of` مفيدة جدًا عندما تحتاج إلى التكرار على قيم من قابل للتكرار مثل المصفوفة أو السلسلة النصية. كما أنها سهلة القراءة ويمكن أن تجعل شفرتك أكثر إيجازًا.
