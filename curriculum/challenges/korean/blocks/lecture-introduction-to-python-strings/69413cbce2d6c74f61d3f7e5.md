# --description--

문자열은 작은따옴표나 큰따옴표로 둘러싸인 문자 수열입니다. 일부 프로그래밍 언어에서는 작은따옴표로 둘러싸인 문자와 큰따옴표로 둘러싸인 문자를 다르게 취급하지만, Python에서는 둘 다 동일하게 취급합니다. 따라서 문자열을 다룰 때 어느 쪽이든 사용할 수 있습니다. 다음은 문자열의 예시입니다:

```python
my_str_1 = 'Hello'
my_str_2 = "World"
```

여러 줄 문자열이 필요하다면, 세 개의 큰따옴표나 작은따옴표를 사용할 수 있습니다:

```python
my_str_3 = """Multiline
string"""
my_str_4 = '''Another
multiline
string'''
```

문자열에 작은따옴표나 큰따옴표가 포함되어 있다면 두 가지 방법이 있습니다:

- 반대 종류의 따옴표를 사용하세요. 즉, 문자열에 작은따옴표가 포함되어 있다면 큰따옴표로 문자열을 감싸고, 그 반대도 마찬가지입니다:
    

```python
msg = "It's a sunny day"
quote = 'She said, "Hello World!"'
```

- 문자열 내의 작은따옴표나 큰따옴표 앞에 역슬래시(`\`)를 붙여 이스케이프하세요. 이 방법을 사용하면 문자열 자체를 감쌀 때 작은따옴표나 큰따옴표 어느 쪽이든 사용할 수 있습니다:
    

```python
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

때때로 문자열에 특정 문자나 문자가 포함되어 있는지 확인해야 할 때가 있습니다. 이를 위해 Python은 `in` 연산자를 제공하며, 이 연산자는 문자열에 해당 문자나 문자가 존재하는지 여부를 나타내는 불리언 값을 반환합니다.

다음은 몇 가지 예시입니다:

```python
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

이제 문자열의 길이를 구하고 문자열 내 개별 문자를 다루는 방법인 **인덱싱**을 살펴보겠습니다. 문자열의 길이를 구하려면 내장 `len()` 함수를 사용할 수 있습니다. 예시는 다음과 같습니다:

```python
my_str = 'Hello world'
print(len(my_str))  # 11
```

문자열의 각 문자는 인덱스라는 위치를 가집니다. 인덱스는 0부터 시작하는데, 이는 문자열의 첫 번째 문자의 인덱스가 `0`이고, 두 번째 문자의 인덱스가 `1`라는 뜻입니다. 인덱스로 문자를 접근하려면 대괄호(`[]`) 안에 접근하려는 문자의 인덱스를 넣으면 됩니다. 다음은 몇 가지 예시입니다:

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
```

음수 인덱싱도 허용되므로, `-1`를 사용해 문자열의 마지막 문자를, `-2`를 사용해 끝에서 두 번째 문자를 얻을 수 있습니다:

```python
my_str = 'Hello world'
print(my_str[-1])  # d
print(my_str[-2]) # l
```

많은 다른 프로그래밍 언어는 데이터 타입을 원시 타입과 참조 타입으로 크게 나눕니다. 원시 타입은 단순하고 불변이며, 선언 후 변경할 수 없습니다. 참조 타입은 여러 값을 담을 수 있고, 변경 가능하거나 불변일 수 있습니다. 하지만 Python은 이 두 그룹 사이에 명확한 경계를 두지 않습니다. 대신 모든 데이터를 객체로 취급하며, 일부 객체는 불변이고 다른 일부는 변경 가능합니다.

불변 데이터 타입은 선언 후 수정하거나 변경할 수 없습니다. 변수에 새로운 값을 다시 할당하는 것은 가능하지만, 원래 객체 자체를 추가, 삭제, 교체하는 방식으로 변경할 수는 없습니다.

문자열은 Python에서 불변 데이터 타입입니다. 즉, 변수에 다른 문자열을 다시 할당할 수는 있지만:

```python
greeting = 'hi'
greeting = 'hello'
print(greeting) # hello
```

문자열을 직접 수정하는 것은 허용되지 않습니다:

```python
greeting = 'hi'
greeting[0] = 'H' # TypeError: 'str' object does not support item assignment
```

Python에서 다른 불변 데이터 타입의 예로는 정수, 부동소수점, 불리언, 튜플, 범위가 있습니다. 앞으로 수업에서 각각 자세히 배우게 됩니다.
