# --description--

Mateo ist 25 Jahre alt, und Luna ist 28 Jahre alt.

Verwenden Sie die korrekte Kombination aus `veinti` und der ausgeschriebenen Zahl, um die Lücken zu füllen.

Zur Referenz:

| Zahl | Spanisches Wort |
| :---: | :---: |
| 1 | `uno` |
| 2 | `dos` |
| 3 | `tres` |
| 4 | `cuatro` |
| 5 | `cinco` |
| 6 | `seis` |
| 7 | `siete` |
| 8 | `ocho` |
| 9 | `nueve` | 

# --fillInTheBlank--

## --sentence--

`Luna tiene BLANK años.`

`Mateo tiene BLANK años.`

## --blanks--

`veintiocho`

`veinticinco`

### --feedback--

Denken Sie daran, dass beim Verbinden von `veinte` (20) mit `cinco` (5) das abschließende „e“ in `veinte` entfällt.

# --explanation--

Die richtigen Antworten sind `veintiocho` und `veinticinco`.

Im Spanischen werden Zahlen von 21 bis 29 als ein einziges Wort geschrieben. Zum Beispiel:

- `Luna tiene veintiocho años` – Luna ist 28 Jahre alt.

- `Mateo tiene veinticinco años` – Mateo ist 25 Jahre alt.
