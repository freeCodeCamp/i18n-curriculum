# --explanation--

`Won't`는 `will not`의 축약형으로, 미래에 대한 부정문을 만들 때 사용합니다. 예를 들어:

- `They won't finish the project on time if they don't start now.` - 행동을 취하지 않으면 부정적인 결과가 예상된다는 뜻입니다.

- `He won't attend the meeting because he is sick.` - 미래에 일어나지 않을 행동을 나타냅니다.
