# --description--

오각수는 공식 P<sub>n</sub>=`n`(3`n`−1)/2로 생성됩니다. 처음 열 개의 오각수는 다음과 같습니다:

1, 5, 12, 22, 35, 51, 70, 92, 117, 145, ...

P<sub>4</sub> + P<sub>7</sub> = 22 + 70 = 92 = P<sub>8</sub>임을 알 수 있습니다. 하지만 두 수의 차이인 70 − 22 = 48은 오각수가 아닙니다.

합과 차가 모두 오각수인 오각수 쌍 P<sub>j</sub>와 P<sub>k</sub>를 찾아서, D = |P<sub>k</sub> − P<sub>j</sub>|가 최소가 되는 값을 구하세요; D의 값은 얼마인가요?
