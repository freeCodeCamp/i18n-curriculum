# --description--

CSS box model — це базова концепція для веброзробки. Вона визначає, як структуровані та розташовані HTML-елементи. Якщо ви розумієте цю модель, то зможете контролювати розмір, відступи та зовнішній вигляд елементів на своєму сайті.

У CSS box model кожен елемент оточений блоком. Цей блок складається з чотирьох частин: області вмісту, відступу, кордону та поля.

Область вмісту — це найвнутрішніша частина блока. Це простір, який містить фактичний вміст елемента, наприклад текст або зображення.

Відступ — це область безпосередньо після області вмісту. Це простір між областю вмісту та кордоном елемента. За допомогою відступу ви можете додати простір навколо вмісту, щоб покращити його читабельність. Ви можете призначити різні значення для верхнього, правого, нижнього та лівого відступу за допомогою властивості `padding`.

Ось приклад зі скороченою властивістю `padding`, де ми встановлюємо верхній відступ у п’ятнадцять пікселів, правий — у п’ять, нижній — у два, а лівий — у вісім пікселів:

```css
padding: 15px 5px 2px 8px;
```

Кордон — це зовнішній край або обриси елемента в CSS box model. Це візуальна межа елемента. Ви можете налаштувати стиль, ширину, колір та інші властивості кордону за допомогою властивості `border`. Ось приклад, де ми встановлюємо ширину кордону п’ять пікселів, стиль — solid, а колір — синій:

```css
border: 5px solid blue;
```

Якщо ви пропустите значення, буде використано властивість за замовчуванням для цього значення. Це `medium` для ширини, `none` для стилю та поточний колір для кольору.

Ви можете встановити ці три властивості безпосередньо у скороченій властивості `border`, якщо хочете, щоб усі сторони були однаковими. Але якщо хочете призначити різний стиль для кожної сторони, можете використати властивості `border-width`, `border-style` та `border-color`.

```css
border-width: 2px 4px 7px 12px;
border-style: dashed solid solid dashed;
border-color: blue red green black;
```

Ви можете записати до чотирьох значень для кожної з цих властивостей. Вони застосовуються за годинниковою стрілкою, починаючи зверху. Якщо вказати лише одне значення, воно застосовується до всіх чотирьох сторін.

Нарешті, поле — це простір поза межами кордону елемента. Воно визначає відстань між елементом та іншими елементами навколо нього. Ви можете призначити різні значення поля для верхньої, правої, нижньої та лівої сторін елемента за допомогою властивості `margin`.

Отже, у цьому прикладі верхнє поле становить три пікселі, праве — дванадцять, нижнє — дев’ять, а ліве — сім пікселів:

```css
margin: 3px 12px 9px 7px;
```

Ці чотири компоненти є необхідними для обчислення загальної ширини та висоти елемента.

У наступних кількох уроках ви дізнаєтеся більше про те, як це обробляє браузер і як ви можете це налаштувати. CSS box model — це базова концепція для веброзробки.

Розуміння того, як ці компоненти взаємодіють і впливають на розміри елемента, є ключовим для реалізації вебдизайнів.
