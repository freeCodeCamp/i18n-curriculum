# --description--

`Encourage` تعني أن تمنح شخصًا الدعم أو الثقة للقيام بشيء ما. على سبيل المثال، عندما يقول صديقك: `You can solve this coding problem!` فهو يشجعك.

`Discourage` هي العكس؛ تعني أن تجعل شخصًا يشعر بثقة أقل أو يصبح أقل رغبة في القيام بشيء ما. إذا قال ذلك الصديق: `That programming language is too hard for beginners` فقد يثنيك ذلك عن تعلمها.

`Discouraging` تصف شيئًا يجعلك تشعر بالإحباط أو قلة الحماس. مثل: `Finding a bug in your code after hours of work can be discouraging.`

`Encouraging` هو عندما يمنحك شيء ما الشجاعة أو الأمل. على سبيل المثال: `Getting your first program to run is very encouraging.`
