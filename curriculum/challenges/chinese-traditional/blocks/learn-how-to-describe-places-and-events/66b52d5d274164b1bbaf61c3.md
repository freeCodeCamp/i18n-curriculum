# --explanation--

`Present Continuous` 時態用來描述說話當下正在發生的動作。它是由動詞 `to be`（`am, is, are`）＋動詞原形＋`-ing` 所組成。 例如：

- `She is coding a new feature right now.` - 她此刻正在編寫一個新的特性。

- `They are testing the software.`－他們現在正在測試軟體。
