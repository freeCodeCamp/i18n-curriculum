# --explanation--

تُستخدم عبارة `First off` لتقديم أول شيء سيتم مناقشته أو تنفيذه. تشير إلى بداية قائمة أو ترتيب للأشياء التي يجب التركيز عليها. على سبيل المثال:

- `First off, we need to address the project's scope.` – هذا يعني أن معالجة نطاق المشروع هي أول شيء سيتم القيام به.

- `First off, let's review the current goals before we start planning.` – هذا يعني أن مراجعة الأهداف الحالية هي أول شيء سيحدث قبل الانتقال إلى التخطيط.
