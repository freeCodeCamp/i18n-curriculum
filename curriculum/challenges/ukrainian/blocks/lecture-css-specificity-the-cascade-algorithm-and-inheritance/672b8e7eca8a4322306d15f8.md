# --interactive--

Універсальний селектор (`*`) — це особливий тип CSS-селектора, який відповідає будь-якому елементу в документі.

Його часто використовують, щоб застосувати стиль до всіх елементів на сторінці, що може бути корисним для скидання або нормалізації стилів у різних браузерах.

Універсальний селектор можна застосовувати для вибору всіх елементів у певному контексті або глобально по всьому документу.

Ось приклад використання універсального селектора для встановлення `margin` і `padding` для всього HTML-документа:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1>Heading element</h1>
<p>example paragraph element</p>
```

```css
* {
  margin: 0;
  padding: 0;
}
```

:::

У цьому прикладі коду селектор `*` скидає поле і відступ усіх елементів до нуля, що є загальною технікою, яку застосовують у CSS-скиданнях.

Універсальний селектор має найнижче значення специфічності серед усіх селекторів. Він додає 0 до всіх частин значення специфічності `(0, 0, 0, 0)`.

Це означає, що будь-який інший селектор, включно з селекторами типу, класу, ідентифікатора та рядковими стилями, заміщатиме стилі, встановлені універсальним селектором.

Розглянемо наступний приклад HTML і CSS:

:::interactive_editor

```html
<head>
  <style>
    * {
      color: blue;
    }
    p {
      color: red;
    }
    .highlight {
      color: green;
    }
    #unique {
      color: purple;
    }
  </style>
</head>
<body>
  <p id="unique" class="highlight">This text has multiple styles applied.</p>
</body>
```

:::

Універсальний `*` селектор встановлює колір тексту `blue` для всіх елементів.

Однак селектор типу `p` заміщує це, встановлюючи колір тексту `red` спеціально для елементів `p`.

Якщо елемент має клас `highlight`, селектор класу має пріоритет, змінюючи колір тексту на `green`.

Нарешті, селектор ідентифікатора `#unique`, який має найвищу специфічність, заміщує всі інші, встановлюючи колір тексту `purple`.
