# --description--

Die bisherigen Aufgaben haben sich auf bestimmte HTML-Elemente und deren Verwendung konzentriert. Es gibt jedoch einige Elemente, die der Seite eine Gesamtstruktur geben und in jedem HTML-Dokument enthalten sein sollten.

Ganz oben im Dokument müssen Sie dem Browser mitteilen, welche HTML-Version Ihre Seite verwendet. HTML ist eine sich weiterentwickelnde Sprache und wird regelmäßig aktualisiert. Die meisten großen Browser unterstützen die neueste Spezifikation, nämlich HTML5. Ältere Webseiten verwenden jedoch möglicherweise frühere Versionen der Sprache.

Sie geben dem Browser diese Information, indem Sie die `<!DOCTYPE ...>`-Tag in der ersten Zeile hinzufügen, wobei der `...`-Teil die HTML-Version angibt. Für HTML5 verwenden Sie `<!DOCTYPE html>`.

Das `!` und das Großbuchstaben-`DOCTYPE` sind wichtig, besonders für ältere Browser. Das `html` ist nicht case-sensitiv.

Anschließend muss der restliche HTML-Code in `html`-Tags eingeschlossen werden. Das öffnende `<html>` steht direkt unter der `<!DOCTYPE html>`-Zeile, und das schließende `</html>` steht am Ende der Seite.

Hier ist ein Beispiel für die Seitenstruktur. Ihr HTML-Code würde in den Bereich zwischen den beiden `html`-Tags eingefügt.

```html
<!DOCTYPE html>
<html>

</html>
```

# --instructions--

Fügen Sie dem leeren HTML-Dokument im Code-Editor oben ein `DOCTYPE`-Tag für HTML5 hinzu. Darunter fügen Sie öffnende und schließende `html`-Tags ein, die ein `h1`-Element umschließen. Die Überschrift kann beliebigen Text enthalten.

# --hints--

Ihr Code sollte ein `<!DOCTYPE html>`-Tag enthalten.

```js
assert.match(code,/<!DOCTYPE\s+?html\s*?>/gi);
```

Es sollte ein `html`-Element vorhanden sein.

```js
assert.lengthOf(document.querySelectorAll('html'), 1);
```

Die `html`-Tags sollten ein `h1`-Element umschließen.

```js
assert.match(code,/<html>\s*?<h1>\s*?.*?\s*?<\/h1>\s*?<\/html>/gi);
```
