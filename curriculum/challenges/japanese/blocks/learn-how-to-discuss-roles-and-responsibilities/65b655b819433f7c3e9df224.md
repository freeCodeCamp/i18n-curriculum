# --description--

`word is` というフレーズは、他の人から聞いた情報や噂を伝えるときに使います。これは `people say` や `I've heard` と言うのと同じような意味です。

例えば、誰かが `Word is that the new update will make the program faster` と言った場合、それは新しいアップデートがプログラムの速度を向上させると聞いたという意味です。
