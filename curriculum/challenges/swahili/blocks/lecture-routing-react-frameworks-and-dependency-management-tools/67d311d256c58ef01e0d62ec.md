# --description--

Hadi sasa, umekuwa ukitumia React kujenga interface za watumizi. Ikiwa unahitaji vipengele zaidi kama kupanga-njia, basi ulilazimika kuleta maktaba ya mtu wa tatu kama React Router ili uweze kubadilisha kati ya maoni tofauti.

Lakini nini hutokea unapotaka kujenga programu kamili ya mtandao? Unaweza kutumia React kwa upande wa mbele na kutumia Node na Express kwa mantiki ya nyuma ikiwa unataka kubaki tu na JavaScript. Au unaweza kutumia lugha nyingine kama Go, Python, au Java kwa upande wa nyuma.

Ingawa hizi zote ni chaguzi zinazowezekana, kuna nyakati ambapo unaweza kutaka kutumia mfumo wa kazi wa React badala yake. Mifumo ya kazi ya React hutoa vipengele kama kupanga-njia, uboreshaji wa picha, upokeaji wa data, uthibitisho wa utambulisho na zaidi. Hii inamaanisha kwamba huenda usihitaji kuanzisha programu tofauti za mbele na nyuma kwa matumizi fulani.

Tuchunguze kwa karibu Next.js ambayo ni mfumo maarufu wa kazi wa React. Moja ya vipengele vikuu vya Next.js ni kipanga-njia kinachotegemea mfumo wa faili. Mfumo huu wa kupanga-njia unajumuisha msaada kwa njia za data zinazobadilika, njia za data sambamba, washughulikiaji wa njia za data, elekezo upya, utafsiri wa lugha na zaidi.

Hapa kuna mfano wa kuunda mshughulikiaji wa ombi maalum:

```js
export async function GET() {
  const res = await fetch("https://example-api.com");
  const data = await res.json();

  return Response.json({ data });
}
```

Unaweza kufafanua washughulikiaji wa njia za data kama maombi ya GET au POST katika faili inayoitwa `route.js` ndani ya saraka ya `app/api`.

Kipengele kingine cha Next.js ni uboreshaji wa picha na fonti kiotomatiki.

Hapa kuna mfano wa kufanya kazi na sehemu ya `Image` ndani ya faili ya `page.js`:

```jsx
import Image from "next/image";

export default function Page() {
  return (
    <Image src="link-to-image-goes-here" alt="descriptive-title-goes-here" />
  );
}
```

Sehemu ya `Image` inaongeza kipengele cha asili cha HTML `img` na kuruhusu upakiaji wa kurasa kwa kasi zaidi na uboreshaji wa ukubwa. Hii inamaanisha kwamba picha zitapakia tu zinapoingia katika eneo la kuona na sehemu ya `Image` itahudumia picha zilizo na ukubwa unaofaa kwa kila kifaa kiotomatiki.

Ingawa haya ni baadhi tu ya vipengele vya Next.js, kuna vipengele vingi zaidi ambavyo unaweza kutumia kujenga programu imara za mtandao kamili. Pia kuna mifumo mingine ya kazi ya React kama Remix ambayo itatoa uwezo sawa wa kujenga programu za mtandao za kisasa kamili.

Ingawa maktaba na mifumo ya kazi ya JavaScript hubadilika mara kwa mara, ni muhimu kuwa na ufahamu wa chaguzi zinazopatikana kwako na kujifunza kuhusu faida na hasara za kila mfumo wa kazi.
