# --explanation--

Estas preguntas son útiles para conocer a alguien en un contexto informal, usando el pronombre `tú` ("you"):

`¿Cómo te llamas?` se usa para preguntar el nombre de una persona. Es equivalente a "What is your name?"

`¿De dónde eres?` se usa para preguntar sobre el origen o la nacionalidad de una persona. Es equivalente a "Where are you from?"

`¿A qué te dedicas?` se usa para preguntar sobre la profesión o trabajo de una persona. Es equivalente a "What is your profession?"

`¿Cuántos años tienes?` se usa para preguntar la edad de una persona. Es equivalente a "How old are you?"
