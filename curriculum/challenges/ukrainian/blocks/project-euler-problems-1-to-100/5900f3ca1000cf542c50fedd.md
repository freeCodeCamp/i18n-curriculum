# --description--

Легко довести, що не існує рівностороннього трикутника з цілими сторонами та цілою площею. Однак майже рівносторонній трикутник 5-5-6 має площу 12 квадратних одиниць.

Ми визначимо <dfn>майже рівносторонній трикутник</dfn> як трикутник, у якого дві сторони рівні, а третя відрізняється не більше ніж на одну одиницю.

Знайдіть суму периметрів усіх майже рівносторонніх трикутників з цілими довжинами сторін і площею, периметри яких не перевищують `limit`.

```js
function almostEquilateralTriangles(limit) {
  // Based on https://blog.dreamshire.com/project-euler-94-solution/
  let perimetersSum = 0;

  let sidesAB = 1;
  let sideC = 1;
  let perimeter = 0;
  let perimeterOffset = 1;

  while (perimeter <= limit) {
    [sidesAB, sideC] = [4 * sidesAB - sideC + 2 * perimeterOffset, sidesAB];
    perimeterOffset = -perimeterOffset;

    perimetersSum += perimeter;
    perimeter = 3 * sidesAB - perimeterOffset;
  }

  return perimetersSum;
}
```
