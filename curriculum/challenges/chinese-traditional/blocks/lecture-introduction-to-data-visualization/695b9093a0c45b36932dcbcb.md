# --description--

在之前的課程中，你已經接觸過資料視覺化，這是將複雜資料以清晰、視覺化的格式呈現的過程。在本課程中，你將學習 D3 函式庫。

D3 函式庫是一個免費且開源的函式庫，你可以用它在 JavaScript 中創建引人注目且有效的資料視覺化。

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-1.png" alt="A green choropleth map of the United States showing county-level data, with a color scale ranging from light green (3%) to dark green (60%+)">


D3 這個名稱是 data-driven documents（資料驅動文件）的縮寫，指的是它如何動態地將資料綁定到文件物件模型（DOM）中的元素。這個函式庫非常靈活，且與網頁自然結合。你不需要學習全新的圖形表示法，就可以用 `svg` 和 `canvas` 元素來建置資料視覺化。

舉例來說，如果你想用 D3 創建長條圖，你可以在一個 JS 檔案中完成以下步驟：

* 宣告代表圖表尺寸和邊界的變數
    
* 宣告水平位置的比例尺
    
* 宣告垂直位置的比例尺
    
* 創建包含長條和標籤的 SVG 容器
    
<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-2.png" alt="A bar chart titled United States GDP showing Gross Domestic Product from 1947 to 2015, with values rising from near zero to approximately 18,000">

那麼，為什麼要學習 D3，而不是其他像 Chart.js、Plotly 或 Tableau 這樣的視覺化工具呢？

與其他工具不同，D3 提供你大量低階的控制權，讓你能用純 HTML、CSS 和 JavaScript 建置可訂製且互動的視覺化。此外，這個函式庫是用 JavaScript 寫成的，所以你不必擔心要學習全新的語言。

在接下來的工作坊和實作課中，你將學習 D3 函式庫的基礎知識，例如選取、形狀和比例尺。你會透過建置散佈圖、長條圖、儀表板應用程式等來學習這些重要概念。
