# --interactive--

Ви, мабуть, бачили, як посилання на вебсторінці ставало фіолетовим після того, як ви на нього натиснули. Це відбувається тому, що стан посилання змінився, і до нього застосовуються інші стилі. Існує п’ять різних станів, у яких може перебувати посилання.

Перший — це стан за замовчуванням, який є `:link`. Цей стан представляє посилання, яке користувач ще не відвідував, не натискав і не взаємодіяв із ним. Можна уявити цей стан як базові стилі для всіх посилань на вашій сторінці. Інші стани будуються поверх нього.

Увімкніть інтерактивний редактор і подивіться на посилання у вікні попереднього перегляду. Воно має бути синього кольору, що представляє стан за замовчуванням. Якщо ви натиснете на нього, воно стане фіолетовим.

:::interactive_editor

```html
<a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
```

:::

Другий стан — це `:visited`, який застосовується, коли користувач уже відвідував сторінку, на яку веде посилання. За замовчуванням це робить посилання фіолетовим — але ви можете використати CSS, щоб надати користувачу інший візуальний сигнал. Це допомагає повідомити, що користувач уже прочитав частину вашої документації або що сайт є надійним, бо він уже ним користувався.

Натисніть на посилання у вікні попереднього перегляду, і ви побачите, що відвідане посилання змінює колір на коричневий. Щоб побачити попередній перегляд, потрібно увімкнути інтерактивний редактор.

**ПРИМІТКА**: Для цього інтерактивного прикладу надано CSS, щоб ви могли побачити зміни в стилях посилань. Не хвилюйтеся, якщо не розумієте CSS — ви дізнаєтеся про це в майбутніх модулях.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:visited {
  color: brown;
}
```

:::

Третій стан — це `:hover`. Цей стан застосовується, коли користувач наводить курсор на посилання. Цей стан допомагає привернути додаткову увагу до посилання, щоб переконатися, що користувач справді має намір натиснути на нього.

Увімкніть інтерактивний редактор і наведіть мишку на посилання. Ви побачите, що колір зміниться на червоний.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:hover {
  color: red;
}
```

:::

Далі йде `:focus`. Цей стан застосовується, коли ми фокусуємося на посиланні.

Натисніть на будь-яку частину пробілу у вікні попереднього перегляду, а потім натисніть `tab` на клавіатурі. Ви побачите, що посилання змінить колір на зелений. Щоб побачити попередній перегляд, потрібно увімкнути інтерактивний редактор.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:focus {
  color: green;
}
```

:::

І нарешті, є `:active`. Цей стан застосовується до посилань, які активуються користувачем. Зазвичай це означає натискання на посилання лівою кнопкою миші. Цей стан допомагає показати користувачу, що елемент, на який він натиснув, є інтерактивним.

Увімкніть інтерактивний редактор і натисніть на посилання. Ви побачите, що посилання стане чорним. Це відбувається дуже швидко, тому можливо доведеться натиснути кілька разів, щоб помітити зміну кольору.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:active {
  color: black;
}
```

:::

Коли ви використовуєте ці стани для стилізації посилань, існує певний порядок, у якому потрібно писати CSS: `link`, `visited`, `hover`, `focus`, а потім `active`.

Тепер ви знаєте, як надати посиланням на вашій сторінці власний стиль, одночасно надаючи користувачу важливу інформацію про стан кожного посилання.
