# --description--

片語 `Got it` 常用於對話中，顯示某人已經理解某件事。這是一種隨意且快速的方式，表示你已經收到並理解該資訊。 例如：

Lucy：`You need to finish the report by tomorrow。`  
Bob：`Got it！` 

`Got it` 意味著 Bob 理解該指令。
