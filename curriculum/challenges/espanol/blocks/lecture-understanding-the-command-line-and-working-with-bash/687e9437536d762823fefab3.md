# --description--

Aprendamos sobre los atajos de línea de comandos que puedes usar para acelerar la productividad.

Los atajos de terminal y línea de comandos pueden ayudarte a optimizar tus flujos de trabajo y maximizar tu productividad. Para Linux y macOS, que ambos pueden rastrear sus raíces hasta Unix, muchos de estos atajos serán los mismos. Pero para Windows, habrá algunas diferencias.

El primer atajo es la tecla de flecha hacia arriba. Presionar esta tecla te permitirá recorrer hacia atrás el historial de comandos, un comando a la vez. La tecla de flecha hacia abajo, entonces, se puede usar para recorrer hacia adelante el historial de comandos. Estas dos teclas te permiten navegar rápidamente por los comandos que has ejecutado anteriormente para encontrar uno que necesites ejecutar de nuevo.

Muchos shells también ofrecen una función de autocompletado, proponiendo comandos basados en lo que has empezado a escribir. La tecla tab puede usarse para completar el resto de la sugerencia, llenando rápidamente tu línea de comandos con la sintaxis completa. Cómo se generan estas sugerencias varía de un shell a otro. Por ejemplo, en zsh estas sugerencias se basan en el historial de comandos más reciente. Pero en PowerShell, se basan en comandos, variables y argumentos disponibles (con menos peso en qué tan recientemente fueron usados).

A veces un comando puede generar mucha salida en tu terminal. Este ruido extra puede interferir o volverse disruptivo para tu flujo de trabajo. En terminales basados en *nix que se encuentran en dispositivos Linux y macOS, puedes presionar Control + L para limpiar toda la pantalla y comenzar con un prompt limpio y fresco. En PowerShell, necesitarías ejecutar el comando `cls` (que puedes asignar a una combinación de teclas como Control + L).

Si necesitas interrumpir un comando para detener su ejecución, puedes usar Control + C para terminar el proceso. Esto finalizará el comando y creará un nuevo prompt, permitiéndote continuar con lo que necesites hacer. Para los usuarios de PowerShell, Control + C también se usa para copiar texto, y solo funcionará para terminar un proceso cuando el contexto no sea ambiguo (por ejemplo, cuando no hay texto seleccionado para copiar).

Los siguientes atajos están disponibles solo en terminales basados en *nix, y no tienen una alternativa en PowerShell.

Puede haber ocasiones en las que necesites realizar múltiples tareas, permitiendo que un proceso o comando se ejecute en segundo plano mientras trabajas en otro. Presionar Control + Z coloca el proceso actual en una tarea en segundo plano y te devuelve a la línea de comandos, donde puedes continuar tu trabajo. Cuando necesites cambiar el enfoque de nuevo a la tarea en segundo plano, puedes usar `fg` para restaurarla.

Y finalmente, puede que haya comandos que necesites ejecutar dos veces seguidas. Puedes presionar la tecla de flecha hacia arriba para retroceder al comando ejecutado anteriormente, pero también puedes escribir dos signos de exclamación (`!!`) en la línea de comandos y presionar enter; esto ejecutará automáticamente el último comando ejecutado.

Muchos terminales ofrecen aún más atajos, y te animo a leer la documentación de tu configuración particular para encontrar los atajos que mejor funcionen para ti. Pero estos atajos universales básicos deberían servir como un excelente punto de partida para maximizar tu productividad y convertirte en un experto en terminal.
