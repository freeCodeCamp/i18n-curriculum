# --explanation--

`Might happen` se usa para hablar de cosas que son posibles pero no seguras. Por ejemplo:

`There's a chance that a data breach might happen if we don't follow security rules.` - Podría ocurrir una violación de datos si no siguen las pautas de seguridad correctas.

Recuerda, los verbos modales siempre van seguidos de la forma base del verbo (sin `to`).
