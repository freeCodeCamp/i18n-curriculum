# --description--

`There is` y `there are` se usan para indicar la existencia o presencia de algo. `There is` se usa con sustantivos singulares o incontables, mientras que `there are` se usa con sustantivos plurales.

Ejemplos:

- `There is a book on the table.` (Sustantivo singular)

- `There is some water in the glass.` (Sustantivo incontable)

- `There are three apples in the basket.` (Sustantivo plural)
