# --description--

你可以把元素看成是内容的容器。 开始和结束标签告诉浏览器元素包含什么内容。 然后，浏览器可以使用该信息来确定如何解释和格式化内容。

有一些 HTML 元素没有结束标签。  These elements often look like this: `<br />` or `<img/>`, but some can also be used without the closing forward slash such as `<br>` or `<img>`.  These are known as void elements or empty elements because they don’t wrap any content. 你将在之后的课程中遇到其中一些，但在大多数情况下，元素都有开始和结束标签。

HTML 有大量预定义的标签，你可以用它们来创建各种不同的元素。 重要的是使用正确的标签来处理内容。 使用正确的标签可以对你的网站的两个方面产生很大影响：它们在搜索引擎中的排名情况；以及依靠辅助技术（如屏幕阅读器）使用互联网的用户如何能够访问这些网站。

对内容使用正确的元素叫作语义化 HTML。 你稍后将在课程中更深入地探索这一点。
