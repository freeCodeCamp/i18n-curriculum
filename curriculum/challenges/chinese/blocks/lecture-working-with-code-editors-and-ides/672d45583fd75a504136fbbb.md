# --description--

要安装 VS Code，你需要访问他们的网站，获取适合你操作系统的安装程序。 请确保你访问的是 VS Code 网站，而不是 Visual Studio 网站。

对于 Windows，你可以下载 EXE 安装程序。 下载完成后，运行文件，安装向导将带你完成安装过程。

对于 Mac，你可以下载一个压缩包，其中包含一个 `.app`，你可以运行它。 你还可以通过 homebrew 安装 VS Code。

打开 Mac 上的终端应用程序，运行以下命令：

```sh
brew install --cask visual-studio-code
```

在 Linux 上，你可以直接下载 `.deb` 或 `.rpm` 文件。 安装完成后，你可以使用相应的可执行文件或 CLI 命令（通常为 `code` ）启动应用程序。 现在你就可以开始编码了。
