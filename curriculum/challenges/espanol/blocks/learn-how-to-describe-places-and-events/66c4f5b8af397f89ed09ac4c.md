# --explanation--

Imagina que estás escribiendo código y quieres encontrar rápidamente la función o variable correcta sin escribir todo. 

`Auto-completion` ayuda sugiriendo lo que probablemente estás intentando escribir basado en lo que ya has tecleado. Esto hace que programar sea más rápido y reduce errores.
