# --description--

Camperbotはプロフィールページを作ろうとしています。友達にコードを見てもらったところ、いくつかエラーがあると言われました。

あなたの仕事は、Camperbotのすべてのエラーを修正して、プロフィールページの作成を続けられるようにすることです。以下のユーザーストーリーの項目を完了し、「Run the Tests」をクリックしてすべてのエラーが修正されたか確認してください。

**ユーザーストーリー:**

1. Camperbotは`heading2`要素を使おうとしていますが、その要素は存在しません。正しい2番目のレベルの見出し要素を使うようにタグを修正してください。
2. Camperbotは`pp`を使って2つの段落を追加しようとしていますが、それらも存在しません。正しい段落タグを使うように修正してください。
3. Camperbotは`h3`要素を`Background and Interests`のサブ見出しに使っていますが、構文エラーがあります。問題を見つけて解決してください。

# --hints--

ページには正確に1つの`h1`要素が必要です。

```js
assert.lengthOf(document.querySelectorAll("h1"), 1);
```

ページから`<h1>Hello from Camperbot!</h1>`要素を削除してはいけません。

```js
assert.match(code, /\<h1\s*\>\s*Hello\s*from\s*Camperbot\s*!\s*\<\/h1\s*\>/i);
```

ページには正確に1つの`h2`要素が必要です。

```js
assert.lengthOf(document.querySelectorAll("h2"), 1);
```

テキスト`h2`を含む`About`要素が必要です。例はこちらです: `<h2>Text here</h2>`

```js
assert.match(code, /\<h2\s*\>\s*About\s*\<\/h2\s*\>/i);
```

テキスト`My name is Camperbot and I love learning new things.`を含む段落要素が必要です。例はこちらです: `<p>Text here</p>`

```js
assert.match(code, /\<p\s*\>\s*My\s*name\s*is\s*camperbot\s*and\s*I\s*love\s*learning\s*new\s*things\s*\.\s*\<\/p\s*\>/i);
```

テキスト`I enjoy solving puzzles.`を含む段落要素が必要です。例はこちらです: `<p>Text here</p>`

```js
assert.match(code, /\<p\s*\>\s*I\s*enjoy\s*solving\s*puzzles\s*\.\s*\<\/p\s*\>/i);
```

ページには正確に2つの段落要素が必要です。

```js
assert.lengthOf(document.querySelectorAll("p"), 2);
```

ページには正確に1つの`h3`要素が必要です。

```js
assert.lengthOf(document.querySelectorAll("h3"), 1);
```

テキスト`h3`を含む`Background and Interests`要素が必要です。例はこちらです: `<h3>Text here</h3>`

```js
assert.match(code, /\<h3\s*\>\s*Background\s*and\s*Interests\s*\<\/h3\s*\>/i);
```

ページには合計で正確に5つの要素が必要です。

```js
assert.lengthOf(document.querySelectorAll("body *:not(script)"), 5);
```
