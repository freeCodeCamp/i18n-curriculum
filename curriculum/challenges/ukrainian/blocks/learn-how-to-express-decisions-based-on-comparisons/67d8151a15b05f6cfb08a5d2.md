# --explanation--  

`That way` використовується, щоб пояснити результат або користь рішення чи дії. Наприклад:

`We should automate testing. That way, we can catch bugs faster.` – `That way` вказує, що автоматизація допоможе швидше знайти помилки.  

`While` використовується, щоб вказати, що дві речі відбуваються одночасно. Наприклад:  

`The team is fixing bugs while the designers work on the UI.` – Це означає, що обидва завдання відбуваються одночасно.
