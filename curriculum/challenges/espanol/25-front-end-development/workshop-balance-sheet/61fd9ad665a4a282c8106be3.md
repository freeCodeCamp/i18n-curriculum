# --description--

Dentro de tu `thead`, crea un elemento `tr`. En él, añade un `td` y tres elementos `th`. Dentro de cada uno de los elementos `th`, añade un elemento `span` con la `class` establecida en `sr-only` y el siguiente texto, en orden: `2019`, `2020` y `2021`.

# --hints--

```js
const table = document.querySelectorAll('table')?.[2];
assert.strictEqual(
  table?.querySelector('thead')?.children?.[0]?.localName,
  'tr'
);
```

```js
const table = document.querySelectorAll('table')?.[2];
```

```js
const table = document.querySelectorAll('table')?.[2];
```

```js
const ths = [
  ...document.querySelectorAll('table')?.[2]?.querySelectorAll('th')

  assert.lengthOf(th?.children, 1);
  assert.strictEqual(th?.children?.[0]?.localName, 'span');

```

```js
const ths = [
  ...document.querySelectorAll('table')?.[2]?.querySelectorAll('th')

  assert.isTrue(th?.children?.[0]?.classList?.contains('sr-only'));
});
```

