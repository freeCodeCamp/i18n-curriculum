# --description--

時々、ルール同士が競合してしまい、予期しない結果になることがあります。「この段落は青にしたかったのに、なぜ他の段落と同じように赤くなっているの？」といった感じです。こうした状況はイライラしますが、CSSは勝手に動いているわけではないことを理解するのが大切です。CSSは私たちが指示したことだけを実行します。例外はブラウザが提供するデフォルトスタイルです。これらのデフォルトスタイルはブラウザごとに異なり、要素同士の間に大きな「隙間」ができたり、ボタンの見た目が特にCSSルールを書かなくても決まっている理由となっています。

ですので、予期しない動作が起きた場合、それはデフォルトスタイルのせいだったり、プロパティの動きを理解していなかったり、カスケードという仕組みを理解していなかったりすることが原因です。

カスケードは、どのルールが実際にHTMLに適用されるかを決める仕組みです。カスケードが判断に使う要素はいくつかあります。ここではそのうちの3つを見ていき、CSSに対する「もう嫌だ！」というフラストレーションを減らせるようにしましょう。
