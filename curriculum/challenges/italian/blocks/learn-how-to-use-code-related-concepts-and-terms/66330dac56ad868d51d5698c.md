# --description--

Un `comment` è una nota o spiegazione aggiunta al codice dal programmatore. I commenti non vengono eseguiti come parte del programma; invece, aiutano chiunque legga il codice a capire il suo scopo, la sua funzionalità o eventuali dettagli specifici. Sono essenziali per rendere il codice leggibile e manutenibile.

Per esempio, `// This is a single-line comment in JavaScript.`
