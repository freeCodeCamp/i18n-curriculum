# --description--

短语 `to keep in mind` 意味着记住或考虑某事。它通常在有人给出建议或应记住以备后用的重要信息时使用。

例如，如果有人说，“记得在徒步前查看天气。”你可能会用 `I'll keep that in mind` 来回应，意思是你会记住考虑这个建议。
