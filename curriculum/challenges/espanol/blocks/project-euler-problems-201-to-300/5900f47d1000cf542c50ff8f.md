# --description--

Para un número positivo $n$, define $C(n)$ como la cantidad de enteros $x$, para los cuales $1 < x < n$ y $x^3 \equiv 1\bmod n$.

Cuando $n = 91$, hay 8 valores posibles para $x$, a saber: 9, 16, 22, 29, 53, 74, 79, 81. Por lo tanto, $C(91) = 8$.

Encuentra la suma de los números positivos $n ≤ {10}^{11}$ para los cuales $C(n)=242$.
