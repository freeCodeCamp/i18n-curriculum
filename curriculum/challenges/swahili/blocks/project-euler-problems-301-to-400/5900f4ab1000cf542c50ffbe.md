# --description--

Acha $x_1, x_2, \ldots, x_n$ iwe mfululizo wa urefu $n$ unaokidhi:

- $x_1 = 2$
- kwa kila $1 &lt; i ≤ n : x_{i - 1} &lt; x_i$
- kwa kila $i$ na $j$ zenye $1 ≤ i, j ≤ n : {(x_i)}^j &lt; {(x_j + 1)}^i$

Kuna mfululizo mitano tu ya urefu 2, yaani: {2,4}, {2,5}, {2,6}, {2,7} na {2,8}. Kuna mfululizo 293 wa urefu 5; mifano mitatu ni: {2,5,11,25,55}, {2,6,14,36,88}, {2,8,22,64,181}.

Acha $t(n)$ iwe nambari ya mfululizo kama huo wa urefu $n$. Umepata kuwa $t(10) = 86195$ na $t(20) = 5227991891$.

Tafuta $t({10}^{10})$ na toa jibu lako modulo $10^9$.
