# --description--

이전 수업에서 `for` 루프를 사용해 코드를 정해진 횟수만큼 반복하는 방법을 배웠습니다. 다음은 `for` 루프를 사용해 `languages` 리스트의 각 언어를 콘솔에 출력하는 예시입니다:

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for language in languages:
    print(language)
```

그런데 각 요소의 인덱스를 추적하고 싶다면 어떻게 해야 할까요? 한 가지 방법은 `index` 변수를 만들고 루프가 반복될 때마다 `1`씩 증가시키는 것입니다. 예를 들어 다음과 같습니다:

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

index = 0

for language in languages:
    print(f'Index {index} and language {language}')
    index += 1
```

이 방법도 작동하지만, 더 쉬운 방법은 `enumerate()` 함수를 사용하는 것입니다. `enumerate()` 함수는 반복 가능한 객체의 인덱스를 추적하고 enumerate 객체를 반환합니다.

만약 `languages` 리스트를 `enumerate()` 함수에 전달하고, 반환된 값을 `list()` 함수로 리스트로 변환하면 다음과 같습니다:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

list(enumerate(languages))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

enumerate 객체(현재는 리스트)의 각 항목은 튜플이며, 첫 번째는 카운트, 두 번째는 `enumerate()` 함수에 전달된 반복 가능한 객체의 값입니다.

이제 앞서 예시를 `enumerate()` 함수를 사용하도록 리팩터링해 봅시다:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')
```

enumerate 객체의 각 튜플에서 카운트와 값을 각각 `index`와 `language`라는 변수로 풀어냅니다. 마지막으로, 이 두 변수를 f-문자열에 넣어 루프가 반복될 때마다 콘솔에 출력합니다.

```md
Index 0 and language Spanish
Index 1 and language English
Index 2 and language Russian
Index 3 and language Chinese
```

이렇게 하면 `index` 변수를 직접 만들고 업데이트할 필요가 없어집니다.

`enumerate()` 함수는 선택적 `start` 인수를 받아 카운트의 시작 값을 지정할 수 있습니다. 이 인수를 생략하면 카운트는 `0`부터 시작합니다. 다음은 선택적 `start` 인수를 사용하는 예시입니다:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages, 1):
    print(f'Index {index} and language {language}')
```

콘솔에서 결과는 다음과 같이 나타납니다:

```md
Index 1 and language Spanish
Index 2 and language English
Index 3 and language Russian
Index 4 and language Chinese
```

지금까지는 한 리스트만 순회했습니다. 그런데 여러 반복 가능한 객체를 병렬로 순회해야 한다면 어떻게 할까요? 이럴 때는 `zip()` 함수를 사용할 수 있는데, 이 함수는 리스트들을 요소 쌍으로 묶어 튜플 반복자를 반환합니다.

만약 `developers`와 `ids` 리스트를 `zip()` 함수에 전달하고, 반환된 값을 `list()` 함수로 리스트로 변환하면 다음과 같습니다:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

list(zip(developers, ids))
# [('Naomi', 1), ('Dario', 2), ('Jessica', 3), ('Tom', 4)]
```

다음은 `zip()` 함수를 `for` 루프와 함께 사용해 `developers`와 `ids`을 순회하는 예시입니다:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')
```

이 예시에서 `zip()`은 두 리스트를 요소 쌍으로 묶어 튜플 반복자를 반환합니다. `for` 루프는 각 튜플을 `name`과 `id`로 풀어냅니다. 마지막으로 각 출력문에서 `name`와 `id` 리스트의 `ids`와 `developers`를 각각 출력합니다. 콘솔 결과는 다음과 같습니다:

```md
Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
```

`enumerate()`와 `zip()` 함수는 매우 강력하며, 루프와 결합하면 코드를 훨씬 간결하게 만들 수 있습니다.
