# --explanation--

`Present Perfect continuous` 시제의 부정형은 `hasn't/haven't + been + present participle`로 만듭니다. 이 시제는 과거에 시작되어 계속되거나 여전히 관련이 있지만 예상대로 발생하지 않은 진행 중인 동작을 설명합니다.
