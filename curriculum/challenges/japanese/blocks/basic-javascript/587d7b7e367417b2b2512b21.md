# --description--

前のチャレンジでは、単一の条件演算子を使いました。複数の条件をチェックするために、それらを連結することもできます。

次の関数は `if`、`else if`、`else` 文を使って複数の条件をチェックしています：

```js
function findGreaterOrEqual(a, b) {
  if (a === b) {
    return "a and b are equal";
  }
  else if (a > b) {
    return "a is greater";
  }
  else {
    return "b is greater";
  }
}
```

上記の関数は、複数の条件（三項）演算子を使って書き直すことができます：

```js
function findGreaterOrEqual(a, b) {
  return (a === b) ? "a and b are equal" 
    : (a > b) ? "a is greater" 
    : "b is greater";
}
```

複数の条件（三項）演算子を使う場合は、上記のように各条件を別の行に書くフォーマットがベストプラクティスとされています。適切なインデントなしに複数の条件（三項）演算子を使うと、コードが読みづらくなる場合があります。例えば：

```js
function findGreaterOrEqual(a, b) {
  return (a === b) ? "a and b are equal" : (a > b) ? "a is greater" : "b is greater";
}
```

# --instructions--

`checkSign` 関数では、`findGreaterOrEqual` で使われている推奨フォーマットに従って複数の条件（三項）演算子を使い、数値が正か負かゼロかをチェックしてください。関数は `positive`、`negative`、または `zero` を返す必要があります。
