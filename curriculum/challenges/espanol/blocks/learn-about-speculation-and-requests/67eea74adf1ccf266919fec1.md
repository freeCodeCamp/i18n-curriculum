# --explanation--

`Should have been` va seguido de un adjetivo y se usa para decir que algo debería haber ocurrido en el pasado pero no sucedió. Por ejemplo:

- `I should have been more careful.` - No fuiste cuidadoso, pero habría sido mejor si hubieras sido.

- `He should have been more prepared for the meeting.` - No estuvo lo suficientemente preparado.

En este caso, Mark quiere decir que debería haber sido más detallado con sus comentarios.
