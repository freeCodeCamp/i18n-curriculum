# --interactive--

在 JavaScript 中，处理文本是编码的重要组成部分，而且你经常需要将文本片段组合或连接在一起。 这一过程称为字符串连接。

在本课中，我们将重点介绍字串串接的工作原理，特别是使用 `+` 操作符、`+=` 操作符和 `concat()` 方法。

`+` 操作符是最简单、最常用的字符串连接方法之一。 它允许你连接多个字符串，或将字符串与包含文本的变量相结合。

这是一个示例：

:::interactive_editor

```js
let firstName = "John";
let lastName = "Doe";

let fullName = firstName + " " + lastName; 
console.log(fullName); // John Doe
```

:::

在这个示例中，我们使用 `+` 操作符将 `firstName` 和 `lastName` 变量连同空格（`" "`）连接起来，以创建全名。

使用 `+` 操作符进行字符串连接的一个缺点是，如果不仔细管理连接字符串之间的间隔，可能会导致间隔问题。

下面是一个缺少空格的例子：

:::interactive_editor

```js
let firstName = "John";
let lastName = "Doe";

let fullName = firstName + lastName; 
console.log(fullName); // "JohnDoe"
```

:::

无论何时使用 `+` 操作符来连接字符串，都必须仔细检查是否存在任何潜在的间隔问题。

如果需要添加或追加到现有字符串，则可以使用 `+=` 操作符。 当你想在一个字符串的基础上逐渐添加更多文字时，这很有帮助。

下面是一个使用 `+=` 操作符将一个字符串追加到另一个字符串的示例：

:::interactive_editor

```js
let greeting = 'Hello';
greeting += ', John!';

console.log(greeting); // "Hello, John!"
```

:::

重要的是要记住，字符串是不可变的，这意味着一旦创建了字符串，就不能更改它。

在这种情况下，`Hello` 的原始字符串不会被修改。 取而代之的是，问候语现在引用了新字符串 `Hello, John!`。

另一种连接字符串的方式是使用 `concat()` 方法。

在开始学习 `concat()` 方法之前，我们有必要先从更高的层面了解一下什么是方法和函数。

在程序设计中，函数是可重复使用的代码块，可执行特定任务，并可以以不同的输入进行调用。 而方法是一种与对象相关联的函数，这意味着它对该对象中包含的数据进行操作。

在以后的课程中，我们将深入探讨函数、对象和方法在 JavaScript 中的工作原理。但现在，理解 JavaScript 有数十种你可以使用的方法，比如 `concat()` 方法，是很重要的。

下面是一个使用 `concat()` 方法将两个字符串连接在一起的示例：

:::interactive_editor

```js
let str1 = 'Hello';
let str2 = 'World';

let result = str1.concat(' ', str2); 
console.log(result); // Hello World
```

:::

在本例中，我们使用 `concat()` 方法将 `str1`、一个空格（ `' '`）和 `str2` 连接成一个字符串。

总结一下，`+` 操作符最适合用于简单的连接，尤其是当你需要合并几个字符串或变量时。

`+=` 操作符在逐步建立字符串或向现有字符串变量添加新内容时非常有用。

最后，当你需要将多个字符串连接在一起时，`concat()` 方法非常有用。
