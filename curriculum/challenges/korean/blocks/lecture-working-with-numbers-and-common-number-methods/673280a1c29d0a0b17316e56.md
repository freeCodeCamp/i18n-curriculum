# --interactive--

`.toFixed()` 메서드는 숫자를 고정 소수점 표기법으로 포맷하는 내장 JavaScript 함수입니다. 특히 숫자의 소수점 자릿수를 제어해야 할 때 유용하며, 통화 값을 표시하거나 정밀한 측정을 다룰 때 자주 사용됩니다.

`.toFixed()` 메서드는 숫자에서 호출하며, 선택적 인수 하나를 받는데 이는 소수점 뒤에 표시할 자릿수입니다. 지정한 소수점 자릿수로 숫자를 문자열로 반환합니다. `.toFixed()`가 작동하는 기본 예시는 다음과 같습니다:

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed(2)); // "3.14"
```

:::

이 경우 소수점 자릿수를 두 자리로 제한했습니다. 그래서 `3.14159`는 `3.14`가 됩니다. `.toFixed()`가 숫자가 아닌 문자열을 반환한다는 점을 기억하세요. 이 메서드는 주로 숫자를 표시용으로 포맷하기 위해 설계되었기 때문입니다.

`.toFixed()` 메서드는 지정한 소수점 자릿수로 나타낼 수 있는 가장 가까운 값으로 숫자를 반올림합니다. 이 반올림 동작을 이해하는 것이 중요합니다:

:::interactive_editor

```js
console.log((3.14159).toFixed(3));  // "3.142"
console.log((3.14449).toFixed(3));  // "3.144"
console.log((3.14550).toFixed(3));  // "3.146"
```

:::

보시다시피 `.toFixed()`는 다음 숫자가 `5` 이상일 때 올림하고, 그렇지 않으면 내림합니다. `.toFixed()`를 인수 없이 호출하면 기본값으로 `0` 자릿수를 사용합니다:

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed()); // "3"
```

:::

`.toFixed()` 메서드는 금융 계산이나 가격 표시에서 특히 유용할 수 있습니다:

:::interactive_editor

```js
let price = 19.99;
let taxRate = 0.08;
let total = price + (price * taxRate);

console.log("Total: $" + total.toFixed(2)); // "Total: $21.59"
```

:::

이 예시에서 `.toFixed(2)`는 총액이 항상 두 자리 소수점으로 표시되도록 보장하는데, 이는 많은 국가에서 통화 표준입니다.

결론적으로 `.toFixed()` 메서드는 JavaScript에서 숫자를 포맷할 때 강력한 도구이며, 특히 소수점 자릿수 표시를 제어해야 할 때 유용합니다. 주로 출력 포맷용으로 사용되지만, 정밀한 계산이 필요할 때 동작 방식을 기억하세요.
