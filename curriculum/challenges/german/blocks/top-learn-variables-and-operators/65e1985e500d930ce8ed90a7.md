# --description--

Es gibt zwei Einschränkungen für Variablennamen in JavaScript:

1. Der Name darf nur Buchstaben, Ziffern oder die Symbole `$` und `_` enthalten.
1. Das erste Zeichen darf keine Ziffer sein.

Beispiele für gültige Namen:

```js
let userName;
let test123;
```

Interessant ist, dass das Dollarzeichen `'$'` und der Unterstrich `'_'` ebenfalls in Namen verwendet werden können. Sie sind reguläre Symbole, genau wie Buchstaben, ohne besondere Bedeutung.

Diese Namen sind gültig:

```js
let $ = 1; // declared a variable with the name "$"
let _ = 2; // and now a variable with the name "_"

console.log($ + _); // 3
```

Beispiele für ungültige Variablennamen:

```js
let 1a; // cannot start with a digit

let my-name; // hyphens '-' aren't allowed in the name
```
