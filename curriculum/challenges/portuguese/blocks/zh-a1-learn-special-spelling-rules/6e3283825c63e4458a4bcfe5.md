# --description--

Às vezes você vê `y` e `w` no início de uma sílaba. Eles não são iniciais. Pense neles como símbolos especiais de ortografia que esclarecem os limites das sílabas e tornam as sílabas mais fáceis de ler.

As regras para `y` são:

- Se `i` for usado como uma sílaba, adicione `y` na frente dele: `i` → `yi`.

- Se uma sílaba começa com `i`, mude para `y`. Exemplos: `ia` → `ya`, `ie` → `ye`, `iao` → `yao`, `iou` → `you`, `iang` → `yang`.

- Se `ü` for usado como uma sílaba, adicione `y` na frente dele e remova os pontos: `ü` → `yu`.

Vamos praticar a pronúncia das sílabas e depois usar `yang` em diferentes tons como exemplo.

| Pinyin   | Character|
|--------|----------|
| `yāng`  | `央`|
| `yáng`   | `羊`|
| `yǎng`  | `养`|
| `yàng`  | `样`|
