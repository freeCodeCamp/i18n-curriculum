# --description--

このプロジェクトでは、ワールドカップの試合情報をPostgreSQLに入力するBashスクリプトを作成し、その後データベースから役立つ統計情報をクエリで取得します。

# --instructions--

**重要：** すべてのプロジェクトテストに合格したら、データベースのダンプを`worldcup.sql`ファイルに保存し、さらに`insert_data.sh`と`queries.sh`のファイルも保存してください。これによりステップ2を完了できます。仮想マシン内でその方法の指示があります。

# --notes--

必要なファイル：`worldcup.sql`、`insert_data.sh`、`queries.sh`
