# --description--

무언가가 제대로 작동하는지 `make sure` 하고 싶을 때, 결과를 최종적으로 확인하기 위해 특별히 신경 써서 시도합니다. 예를 들어:

- `We have to make sure the code is running.` - 이는 코드가 잘 실행되는지 특별히 주의한다는 뜻입니다.

- `Please make sure that you order the correct color.` - 여기서는 주문하기 전에 원하는 색상을 확인하라는 요청입니다.
