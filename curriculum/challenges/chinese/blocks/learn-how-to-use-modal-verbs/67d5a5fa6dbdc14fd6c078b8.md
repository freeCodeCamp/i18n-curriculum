## --answers--

`What happen would if the system crashed?`

### --feedback--

正确的词序是 `What would happen...`。

---

`Would what happen if the system crashed?`

### --feedback--

在疑问句中，`would` 位于 `what` 之后，而不是之前。

---

`What would happens if the system crashed?`

### --feedback--

`Would` 后面跟动词的原形，所以 `happens` 应该是 `happen`。

---

`What would happen if the system crashed?`

# --explanation--

如果 `wh-` 疑问词（`when`、`who`、`why`、`how`）使用情态动词，情态动词位于主语之前。主要动词保持原形。例如：

- `What can you do to fix this?` - 询问可能的解决方法。

- `Why should we start early?` - 请求建议。

- `How must we prepare?` - 询问要求。

- `When will you finish?` - 询问未来的动作。

- `Where could I find help?` - 询问有关选项。

这样，你就是在询问什么是可能的、必要的或预期的。
