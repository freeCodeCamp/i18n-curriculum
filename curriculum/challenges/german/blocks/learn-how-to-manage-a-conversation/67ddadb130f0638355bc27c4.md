# --explanation--

Phrasal Verbs sind Verben, die mit Präpositionen oder Adverbien kombiniert werden und eine neue Bedeutung ergeben. In diesem Teil des Dialogs sehen Sie den Unterschied zwischen `to send something` und `to send something over`.

`To send something` bedeutet, etwas von einem Ort oder einer Person zu einer anderen zu bringen, während `to send something over` die Idee der Zustellung an eine Person oder einen Ort hinzufügt, oft informell oder digital. Schauen Sie sich ein Beispiel von jedem an:

- `I'll send the letter tomorrow.` – Eine einfache Aussage über das Übertragen von etwas.

- `Can you send over the file by email?` – Dies betont, etwas *an jemanden* zu senden, normalerweise über eine kurze Distanz oder digital. Die Hinzufügung von `over` lässt die Phrase in der lockeren Unterhaltung natürlicher klingen.
