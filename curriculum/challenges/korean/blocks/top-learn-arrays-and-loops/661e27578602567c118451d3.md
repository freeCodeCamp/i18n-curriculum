# --description--

배열에 새 요소를 추가하는 가장 일반적인 방법 중 하나는 `push()` 메서드를 사용하는 것입니다. `push()` 메서드는 배열 끝에 하나 이상의 요소를 추가하고 배열의 새 길이를 반환합니다.

예를 들어, `pet` 배열에 새 요소를 추가하려면 다음 코드를 사용할 수 있습니다:

```javascript
const pet = ['cat', 'dog', 'bunny'];
pet.push('parrot');
console.log(pet); // Output: ['cat', 'dog', 'bunny', 'parrot']
```

배열의 마지막 요소를 제거하려면 `pop()` 메서드를 사용할 수 있습니다. `pop()` 메서드는 배열에서 마지막 요소를 제거하고 그 요소를 반환합니다.

예를 들어, `pet` 배열에서 마지막 요소를 제거하려면 다음 코드를 사용할 수 있습니다:

```javascript
const pet = ['cat', 'dog', 'tiger'];
pet.pop();
console.log(pet); // Output: ['cat', 'dog']
```
