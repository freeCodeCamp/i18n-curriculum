# --interactive--

**ПРИМІТКА**: Деякі інтерактивні приклади можуть використовувати CSS, який ви ще не вивчали. Не хвилюйтеся, якщо не розумієте весь код. Мета прикладів — показати вам попередній перегляд цих концепцій дизайну, щоб ви краще розуміли, як все працює. Щоб побачити попередній перегляд, потрібно увімкнути інтерактивний редактор.

Компоненти карток дуже поширені в електронній комерції, соціальних мережах і новинних сайтах. Вони допомагають відображати інформацію у структурованому вигляді. Коли ви розробляєте свої картки, важливо розуміти найкращі практики, щоб користувачі могли легко сприймати інформацію, яку ви хочете донести.

Першим критерієм дизайну карток має бути простота. Не варто робити картки візуально перевантаженими або показувати надто багато інформації. Наприклад, якщо дизайн картки перевантажений, користувачеві буде складно опрацювати всю інформацію одразу.

Ось приклад перевантаженого дизайну картки:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2><strong><em>Product Name: Ultimate Edition</em></strong></h2>
  <p>
    This is a great product that you will love! It has many features and benefits that will make your life easier and more enjoyable. Whether you're at home or on the go, this product is perfect for all situations. Don’t miss out on this incredible offer, buy now and experience the difference!
  </p>
  <button>Buy Now</button>
  <button>Learn More</button>
  <button>Add to Wishlist</button>
  <p><strong>Only 3 left in stock!</strong></p>
  <p><em>Rated 4.8 stars by 2,391 customers</em></p>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}
.card p {
  font-size: 1em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

Менша кількість інформації та хороший інтервал між елементами на картці полегшують користувачеві сприйняття інформації і дозволяють розмістити на сторінці кілька карток.

Ось приклад простого дизайну картки:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2>Product Name</h2>
  <button>Buy Now</button>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 200px;
  text-align: center;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

Ще один момент, який варто врахувати — де користувач може натиснути на картку. Деякі дизайни карток мають одну кнопку, що робить очевидним місце для натискання. Інші дизайни роблять всю картку активною. Коли користувач наводить курсор на будь-яку частину картки, вона змінює колір або має тінь, що вказує на її активність. Який би дизайн ви не обрали, він має бути послідовним на всьому сайті і зрозумілим для користувача.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <a href="#">
    <img src="https://placehold.co/150x150" alt="Example Product Image">
    <h2>Product Name</h2>
    <button>Buy Now</button>
  </a>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
  cursor: pointer;
}

.card:hover {
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
}

.card a {
  text-decoration: none;
  color: inherit;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

Ще один аспект — використання медіаконтенту на ваших картках. Вибір якісного медіаконтенту значно покращує користувацький досвід. Якщо ви використовуєте зображення або відео, наприклад, для картки продукту, чим вища якість, тим більше користувач зацікавиться цим продуктом. Але якщо медіаконтент низької якості, користувач може не довіряти якості ваших товарів і послуг.

Останнє, що варто врахувати — це використання кольорової ієрархії. Ви хочете, щоб найважливіша інформація на картці була найпомітнішою. Можна використовувати яскраві кольори для важливих елементів, наприклад, кнопки заклику до дії, і світліші кольори для менш важливих елементів.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2>Product Name</h2>
  <button>Buy Now</button>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
  text-align: center;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #28a745; 
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::
