# --description--

考慮無限多項式級數 $A_G(x) = xG_1 + x^2G_2 + x^3G_3 + \cdots$，其中 $G_k$ 是二階遞迴關係式的第 k 項，定義為 $G_k = G_{k − 1} + G_{k − 2}, G_1 = 1$ 且 $G_2 = 4$；也就是說，序列為 $1, 4, 5, 9, 14, 23, \ldots$。

對於此問題，我們將關注使得 $A_G(x)$ 為正整數的 $x$ 值。

前五個自然數對應的 $x$ 值如下所示。

| $x$                         | $A_G(x)$ |
|-----------------------------|----------|
| $\frac{\sqrt{5} − 1}{4}$    | $1$      |
| $\frac{2}{5}$               | $2$      |
| $\frac{\sqrt{22} − 2}{6}$   | $3$      |
| $\frac{\sqrt{137} − 5}{14}$ | $4$      |
| $\frac{1}{2}$               | $5$      |

如果 $x$ 是有理數，我們將稱 $A_G(x)$ 為黃金礦粒，因為它們變得越來越稀有；例如，第 20 個黃金礦粒是 211345365。求前 30 個黃金礦粒的總和。
