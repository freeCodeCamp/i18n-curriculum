# --interactive--

## Unità assolute

- **`px` (Pixel)**: Questa unità assoluta è un'unità di misura a dimensione fissa in CSS. È l'unità assoluta più comune e offre un controllo preciso sulle dimensioni. `1px` è sempre uguale a 1/96 di pollice.
- **`in` (Pollice)**: Questa unità assoluta è uguale a 96px.
- **`cm` (Centimetri)**: Questa unità assoluta è uguale a 25,2/64 di pollice.
- **`mm` (Millimetri)**: Questa unità assoluta è uguale a 1/10 di centimetro.
- **`q` (Quarti di millimetro)**: Questa unità assoluta è uguale a 1/40 di centimetro.
- **`pc` (Pica)**: Questa unità assoluta è uguale a 1/6 di pollice.
- **`pt` (Punti)**: Questa unità assoluta è uguale a 1/72 di pollice.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="units">
  <div class="unit px">px</div>
  <div class="unit inch">in</div>
  <div class="unit cm">cm</div>
  <div class="unit mm">mm</div>
  <div class="unit q">q</div>
  <div class="unit pc">pc</div>
  <div class="unit pt">pt</div>
</div>
```

```css
.units {
  display: flex;
  gap: 12px;
  flex-wrap: wrap;
  align-items: flex-end;
}

.unit {
  background: steelblue;

  padding: 4px;
}

  width: 40px;
  height: 40px;
}

.inch {
  width: 0.5in;
  height: 0.5in;
}

  width: 1cm;
  height: 1cm;
}

.mm {
  width: 10mm;
  height: 10mm;
}

  width: 40q;
  height: 40q;
}

.pc {
  width: 6pc;
  height: 6pc;
}

.pt {
  width: 36pt;
  height: 36pt;
}
```

## Unità relative

- **Percentuali**: Queste unità relative ti permettono di definire dimensioni e altre proprietà come proporzione rispetto all'elemento genitore. Per esempio, se imposti `width: 50%;` su un elemento, occuperà metà della larghezza del suo contenitore genitore.
- **Unità `em`**: Queste unità sono relative alla dimensione del carattere tipografico dell'elemento. Se usi `ems` per la proprietà `font-size`, la dimensione del testo sarà relativa alla dimensione del carattere tipografico dell'elemento genitore.
- **Unità `rem`**: Queste unità sono relative alla dimensione del carattere tipografico dell'elemento radice, che è l'elemento `html`.
- **Unità `vh`**: `vh` indica `"viewport height"` e `1vh` è uguale all'1% dell'altezza del viewport.
- **Unità `vw`**: `vw` indica `"viewport width"` e `1vw` è uguale all'1% della larghezza del viewport.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  <div class="box percent">50%</div>
  <div class="box em">2em</div>
  <div class="box rem">2rem</div>
  <div class="box vh">10vh</div>
  <div class="box vw">10vw</div>
</div>
```

```css
html {
  font-size: 16px;
}

.parent {
  width: 200px;
  font-size: 20px;
  border: 2px dashed #555;
  padding: 10px;
  display: flex;
  flex-direction: column;
  gap: 12px;
}

.box {
  background: seagreen;
  color: white;
  text-align: center;
  padding: 6px;
}

.percent {
  width: 50%;
}

.em {
  font-size: 2em;
}

.rem {
  font-size: 2rem;
}

.vh {
  height: 10vh;
}

.vw {
  width: 10vw;
}
```

## Funzione `calc`

- **Funzione `calc()`**: Con la funzione `calc()` puoi eseguire calcoli direttamente all'interno dei tuoi fogli di stile per determinare dinamicamente i valori delle proprietà. Questo significa che puoi creare interfacce utente flessibili e responsivi calcolando le dimensioni in base alla dimensione del viewport o ad altri elementi.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="calc-box">calc()</div>
```

```css
.calc-box {
  width: calc(100% - 40px);
  padding: 10px;
  background: steelblue;
  color: white;
  text-align: center;
  border: 2px solid black;
}
```
