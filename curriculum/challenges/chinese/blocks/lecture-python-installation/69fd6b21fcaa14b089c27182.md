# --description--

在上一课中，你学习了如何在本地设备上安装 Python。在本课中，你将学习如何在本地运行 Python 代码。

在你开始本地创建 Python 脚本之前，你需要安装一个代码编辑器或集成开发环境（IDE）。IDE 是集成开发环境的缩写，它包含代码编辑器、测试工具、终端等多种特色。

许多开发者常用的一个流行代码编辑器是 VS Code。访问 `https://code.visualstudio.com/download` 下载支持 Mac、Windows 和 Linux 环境的 VS Code。这就是本课中将使用的编辑器。

其他常见的 Python 开发选择包括：

- PyCharm
- Spyder

下载编辑器后，你需要打开它。打开后，你需要打开一个文件夹。你可以点击左上角的 `File` 菜单选项，然后点击 `Open Folder`。从那里，你可以选择或创建一个用于你的项目的文件夹。为了课外额外练习，你可能想要一个名为 `python-projects` 的文件夹。

设置好文件夹后，你可以打开资源管理器，它是 VS Code 左侧的侧边栏。点击看起来像一张带有加号的纸张的 `New File` 图标。然后创建一个名为 `main.py` 的文件并按回车。`.py` 是文件扩展名，表示这是一个 Python 文件。

打开你的 `main.py` 文件并输入以下代码：

```py
print("Hello, world!")
```

要运行这段代码，你有几个选项。第一个选项是点击位于右上角看起来像播放按钮的运行按钮。它会打开一个终端并在那里运行你的 Python 脚本。你应该会看到文本 `"Hello, world!"`。

另一个选项是打开终端并手动运行程序，使用：

```bash
python main.py
```

你必须从保存 `main.py` 的文件夹中运行此命令。例如，如果 `main.py` 在名为 `python-projects` 的文件夹内，先在终端中使用以下命令：

```bash
cd python-projects
```

然后运行：

```bash
python main.py
```

在某些环境中，尤其是在同时安装了 Python 2 和 Python 3 的 macOS 和 Linux 系统上，你可能需要使用 `python3` 代替 `python`。当 `python` 命令不存在或指向旧版本的 Python 时，这种情况很常见。

```bash
python3 main.py
```

随着课程的进展，你应该尝试在本地运行课程中的示例并查看结果。你也应该尝试自己创建示例来测试你从课程中学到的内容。
