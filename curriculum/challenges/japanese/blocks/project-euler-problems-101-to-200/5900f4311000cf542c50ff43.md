# --description--

整数辺の三角形で、ちょうど1つの角が60°であるものを60°三角形と呼びます。

このような60°三角形の内接円の半径を$r$とします。

$r ≤ 100$となる60°三角形は1234個あります。

$r ≤ n$となる60°三角形の個数を$T(n)$とすると、$T(100) = 1234$、$T(1000) = 22767$、$T(10000) = 359912$です。

$T(1053779)$を求めてください。
