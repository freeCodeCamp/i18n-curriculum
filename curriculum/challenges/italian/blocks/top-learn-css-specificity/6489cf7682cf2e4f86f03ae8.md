# --description--

L’ereditarietà si riferisce ad alcune proprietà CSS che, quando vengono applicate a un elemento, vengono ereditate dai discendenti di quell’elemento, anche se non scriviamo esplicitamente una regola per quei discendenti. Le proprietà tipografiche basate su `color`, `font-size`, `font-family` ecc. sono di solito ereditate, mentre la maggior parte delle altre proprietà non lo sono.

L’eccezione a questo è quando si definisce direttamente uno stile per un elemento, perché questo ha sempre la precedenza sull’ereditarietà:

```html
<!-- index.html -->

<div id="parent">
  <div class="child"></div>
</div>
```

```css
/* styles.css */

#parent {
  color: red;
}

.child {
  color: blue;
}
```

Nonostante l’elemento `parent` abbia una specificità più alta con un ID, l’elemento `child` avrebbe applicato lo stile `color: blue` poiché quella dichiarazione lo definisce direttamente, mentre `color: red` del genitore viene solo ereditato.
