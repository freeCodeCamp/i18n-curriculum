# --questions--

## --text--

¿Qué oración usa correctamente `across` para describir una ubicación?

## --answers--

`He walked across the code to find the error.`

### --feedback--

`Across` se usa para espacio físico, no para cosas abstractas como código.

---

`She sent the report across her thoughts.`

### --feedback--

Esto no es un uso natural ni correcto de `across` para describir una ubicación.

---

`The conference room is across the hall from the kitchen.`

---

`They discussed the topic across the lunch break.`

### --feedback--

Este uso de `across` se relaciona con el tiempo, no con la ubicación física.

## --video-solution--

3

# --explanation--

`Across` se usa para describir algo al otro lado de un espacio o área.

En la oración correcta, muestra que la sala de conferencias está ubicada en el lado opuesto del pasillo respecto a la cocina.
