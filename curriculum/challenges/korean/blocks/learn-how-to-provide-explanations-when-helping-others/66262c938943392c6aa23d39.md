# --description--

`hard drive`는 컴퓨터에서 데이터를 영구적으로 저장하는 컴포넌트입니다. 모든 디지털 파일이 보관되는 큰 서류 캐비닛과 같습니다. 예를 들어, 문서, 사진, 음악을 컴퓨터에 저장할 때 `hard drive`에 저장됩니다.

이는 임시 저장소와는 다릅니다. 임시 저장소는 사용 중에 잠시 보관하는 책상과 같습니다.
