# --description--

Um ein Element zu animieren, müssen Sie die Animations-Eigenschaften und die `@keyframes`-Regel kennen. Die Animations-Eigenschaften steuern, wie sich die Animation verhalten soll, und die `@keyframes`-Regel steuert, was während dieser Animation passiert. Insgesamt gibt es acht Animations-Eigenschaften. Diese Aufgabe hält es einfach und behandelt zuerst die zwei wichtigsten:

`animation-name` legt den Namen der Animation fest, der später von `@keyframes` verwendet wird, um CSS mitzuteilen, welche Regeln zu welchen Animationen gehören.

`animation-duration` legt die Dauer der Animation fest.

`@keyframes` gibt genau an, was während der Animation über die Dauer passiert. Dies geschieht, indem CSS-Eigenschaften für bestimmte „Frames“ während der Animation mit Prozentangaben von 0 % bis 100 % angegeben werden. Wenn Sie das mit einem Film vergleichen, sind die CSS-Eigenschaften für 0 % die Darstellung des Elements in der Eröffnungsszene. Die CSS-Eigenschaften für 100 % zeigen, wie das Element am Ende erscheint, kurz bevor die Abspannrolle beginnt. Dann wendet CSS die Magie an, um das Element über die angegebene Dauer zu überblenden und die Szene darzustellen. Hier ist ein Beispiel, das die Verwendung von `@keyframes` und den Animations-Eigenschaften veranschaulicht:

```css
#anim {
  animation-name: colorful;
  animation-duration: 3s;
}

@keyframes colorful {
  0% {
    background-color: blue;
  }
  100% {
    background-color: yellow;
  }
}
```

Für das Element mit der `anim`-ID setzt der obige Code-Ausschnitt die `animation-name` auf `colorful` und die `animation-duration` auf 3 Sekunden. Dann verknüpft die `@keyframes`-Regel die Animations-Eigenschaften mit dem Namen `colorful`. Sie setzt die Farbe zu Beginn der Animation (0 %) auf Blau, die bis zum Ende der Animation (100 %) zu Gelb übergeht. Sie sind nicht auf Übergänge nur von Anfang bis Ende beschränkt, Sie können Eigenschaften für das Element für jeden Prozentsatz zwischen 0 % und 100 % festlegen.
