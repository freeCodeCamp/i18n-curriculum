# --description--

Der Ausdruck `a lot of` kann sowohl mit zählbaren als auch mit unzählbaren Nomen verwendet werden, um eine große Anzahl oder Menge anzugeben.

Bei zählbaren Nomen, wie `cars`, könnten Sie `a lot of cars` sagen. Bei unzählbaren Nomen, wie `water`, würden Sie `a lot of water` sagen.
