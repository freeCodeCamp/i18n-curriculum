# --explanation--

`After that` 用于显示事件的顺序，表示下一个动作或事件紧随之前发生的某事。它有助于按时间顺序组织动作。例如：  

- `We finished the code review. After that, we started debugging.` - 首先，他们审查了代码。然后，他们进行了调试。

- `He submitted the report. After that, he took a short break.` - 首先，他提交了报告。然后，他休息了一会儿。

`After that` 的含义类似于 `then` 或 `next` 这样的表达式，但它特别强调一个动作在时间上紧随另一个动作之后。 

这里，James 使用 `after that` 来表示在调试进程中的另一个操作之后才意识到凭证问题所在。
