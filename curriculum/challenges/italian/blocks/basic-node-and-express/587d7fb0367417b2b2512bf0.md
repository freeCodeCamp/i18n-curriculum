# --description--

Un server HTML di solito ha una o più directory consultabili dall’utente. Puoi mettere lì le risorse statiche necessarie alla tua applicazione (fogli di stile, script, immagini).

In Express, puoi attivare questa funzionalità usando il middleware `express.static(path)`, dove il parametro `path` è il percorso assoluto della cartella che contiene le risorse.

Se non sai cos’è un middleware... non preoccuparti, ne parleremo in dettaglio più avanti. In pratica, i middleware sono funzioni che intercettano i gestori di rotta, aggiungendo qualche tipo di informazione. Un middleware deve essere montato usando il metodo `app.use(path, middlewareFunction)`. Il primo argomento `path` è opzionale. Se non lo passi, il middleware verrà eseguito per tutte le richieste.

# --instructions--

Monta il middleware `express.static()` sul percorso `/public` con `app.use()`. Il percorso assoluto alla cartella delle risorse è `__dirname + /public`.

Ora la tua app dovrebbe essere in grado di servire un foglio di stile CSS. Nota che il file `/public/style.css` è referenziato nel `/views/index.html` nel codice boilerplate del progetto. La tua pagina iniziale dovrebbe ora apparire un po’ migliore!

# --hints--

La tua app dovrebbe servire i file delle risorse dalla directory `/public` al percorso `/public`

```js
  const response = await fetch(code + '/public/style.css');
  if (!response.ok) {
    throw new Error(await response.text());
  }
  const data = await response.text();
  assert.match(
    data,
    /body\s*\{[^\}]*\}/,
    'Your app does not serve static assets'
  );
```

La tua app non dovrebbe servire file da altre cartelle diverse dalla directory `/public`

```js
  const response = await fetch(code + '/server.js');
  if (response.ok) {
    const data = await response.text();
    assert.equal(
      response.status + '',
      404 + '',
      'Your app must serve files only from "public" directory'
    );
  } else {
    assert.equal(
      response.status + '',
      404 + '',
      'Your app must serve files only from "public" directory'
    );
  }
```
