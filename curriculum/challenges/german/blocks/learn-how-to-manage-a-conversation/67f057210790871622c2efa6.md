# --explanation--

`Though` wird verwendet, um zwei Ideen gegenüberzustellen. Es zeigt oft, dass etwas trotz einer Schwierigkeit oder unerwarteten Situation geschieht. In diesem Fall zeigt es, dass das Team trotz der Probleme weiterhin daran arbeitet. Zum Beispiel:  

`The task is difficult. I'm going to finish it, though.` – Das bedeutet, die Person wird die Aufgabe trotz der Schwierigkeit erledigen.
