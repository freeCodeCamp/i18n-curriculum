# --explanation--

A `user interface` (UI) refere-se à parte de um sistema ou software com a qual os usuários interagem diretamente, como botões, menus e telas.Por exemplo: 

`The user interface was redesigned to be more intuitive.` - Significando que a parte do software com a qual o usuário interage foi melhorada.

`User feedback` refere-se a opiniões, sugestões ou problemas relatados pelos usuários sobre a experiência deles com um produto ou serviço.Por exemplo: 

`The company improved the app based on user feedback.` - Significa que os comentários e sugestões vindos dos usuários foram lidos e usados para melhorar o aplicativo.

Neste contexto, Linda explica que a equipe contou com o feedback dos usuários para fazer melhorias constantes na interface do usuário, garantindo que ela atendesse às necessidades dos usuários.
