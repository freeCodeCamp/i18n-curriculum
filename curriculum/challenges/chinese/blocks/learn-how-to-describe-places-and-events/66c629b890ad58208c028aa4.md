# --explanation--

`Walkthrough` 由两部分组成： 

- `Walk` 指的是一步一步移动，通常是步行。

- `Through` 意味着从一侧移动到另一侧或从开始移动到结束。

当这些词组合成 `walkthrough` 时，它表示一个逐步的解释或指南，用于帮助某人理解如何做某事。例如：

- `The software includes a walkthrough to help new users get started.` - 有一个指南解释使用该软件的每一步。
