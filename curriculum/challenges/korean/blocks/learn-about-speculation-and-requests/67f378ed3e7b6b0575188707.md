# --explanation--

피싱 공격은 종종 사용자를 속여 `malicious links`를 클릭하게 하여 계정이나 시스템에 대한 `unauthorized access`로 이어집니다.

`Unauthorized access`는 누군가가 허가 없이 시스템에 들어가는 것을 의미합니다. 예를 들어:

`The system blocked unauthorized access.` - 시스템이 승인되지 않은 사용자의 접근을 막았습니다.

`Malicious link`는 정보를 훔치거나 시스템을 감염시킬 수 있는 해로운 링크를 의미합니다. 예를 들어:

`The email had a malicious link inside.` - 이메일에 당신을 속이기 위해 만들어진 나쁜 링크가 포함되어 있었습니다.
