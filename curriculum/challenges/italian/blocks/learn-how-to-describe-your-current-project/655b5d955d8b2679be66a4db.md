# --description--

`What` viene spesso usato per chiedere informazioni specifiche. Può chiedere di cose, azioni o dettagli che qualcuno vuole sapere. Per esempio:

- `What time is the meeting?` - Questa domanda chiede l'orario esatto della riunione.

- `What do you usually have for lunch?` - Questa domanda chiede dettagli sulla routine di qualcuno.

Quindi, `what` ti aiuta a ottenere risposte chiare quando vuoi sapere qualcosa di specifico.
