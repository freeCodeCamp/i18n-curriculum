# --explanation--

`On another note` se usa para introducir un nuevo tema en una conversación. Por ejemplo:

`On another note, have you seen the new project updates?` – Esto significa que el hablante está cambiando de tema.

`To touch base on something` significa ponerse en contacto o tener una breve discusión sobre un tema. Por ejemplo: 

`We should touch base on the client's feedback before the meeting.` – Esto significa tener una breve charla sobre lo que dijo el cliente.
