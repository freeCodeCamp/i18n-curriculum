# --explanation--

Ikiwa swali la `wh-` (`when`, `who`, `why`, `how`) linatumia kitenzi cha modal, kitenzi cha modal huja kabla ya kiambatisho. Kitenzi kikuu hubaki katika umbo lake la msingi. Kwa mfano:

- `What can you do to fix this?` - Kuuliza kuhusu uwezekano.

- `Why should we start early?` - Kuomba ushauri.

- `How must we prepare?` - Kuuliza kuhusu mahitaji.

- `When will you finish?` - Kuuliza kuhusu matendo ya baadaye.

- `Where could I find help?` - Kuuliza kuhusu chaguzi.

Kwa njia hii, unauliza kuhusu kile kinachowezekana, kinachohitajika, au kinachotarajiwa.
