# --description--

WindowsとmacOSでファイルやフォルダーを検索する方法を学びましょう。

まずはWindowsから始めます。タスクバーまたはファイルエクスプローラーから検索できます。

タスクバーから検索するには、探しているファイルやフォルダーの名前を入力してください。キーワードで検索することもできます。PC全体とOneDriveからの結果が表示されます。

ドキュメントだけを表示したい場合は、「Documents」タブをクリックしてください。アプリやウェブの結果を見るためのタブもあります。

もう一つの方法は、ファイルエクスプローラーの検索を使うことです。エクスプローラーウィンドウの右上に「Search Documents」と表示されている場所があります。

検索のスコープを定義するには3つの選択肢があります：

- Homeから検索すると、PCとクラウドのファイルを素早く見つけられます。

- フォルダーから検索すると、その中に保存されているファイルを探せます。

- This PCから検索すると、遅いですが詳細な検索ができます。

これがWindowsでファイルやフォルダーを検索する方法です。次にmacOSでのやり方を見てみましょう。

まず、Spotlightを使えます。これはMac上のファイルを見つけるのに役立つツールです。Spotlightを開くには、画面右上のメニューバーにある虫眼鏡アイコンを探してください。メニューバーにこのアイコンがない場合は、コントロールセンターの設定で追加できます。

設定を開き、「Control Center」でフィルターをかけ、このオプションをクリックしてから「Spotlight」オプションが見つかるまでスクロールしてください。キーボードショートカット `Command + Spacebar` でもSpotlightを開けます。

Spotlightを開くとSpotlight Searchが表示され、名前を入力してファイルやフォルダーを検索できます。入力を始めると、提案、フォルダー、プレゼンテーション、写真、ドキュメント、関連検索などのカテゴリー別に結果が表示されます。

「Show More」をクリックすると特定のカテゴリーを展開できます。

最近開いたファイルを検索する別の方法は、DockのFinderアイコンをクリックしてFinderを開き、サイドバーの「Recents」に移動することです。そこに最近閲覧したすべてのファイルが表示されます。

これがmacOSでファイルやフォルダーを検索する方法です。これらの検索ツールを使えば、作業効率を高め、必要なファイルやフォルダーを素早く見つけられます。
