# --explanation--

Wenn Brian sagt, `The team is working on it as we speak`, verwendet er die `Present Continuous`-Zeitform, um zu zeigen, dass das Team gerade aktiv am Projekt arbeitet.

`As we speak` bedeutet, dass die Arbeit genau in diesem Moment stattfindet. Das erklärt, warum er zuversichtlich ist, dass das Projekt bis zum Ende der Woche fertig sein wird.

Achten Sie auf diesen Ausdruck und darauf, wie er zeigt, dass Fortschritte gemacht werden, weshalb Brian glaubt, dass das Projekt bald abgeschlossen sein wird.
