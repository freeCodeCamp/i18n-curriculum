## 반복문 다루기

- **`for` 반복문**: 이 반복문은 코드 블록을 정해진 횟수만큼 반복할 때 사용합니다. 이 반복문은 초기화 문장, 조건, 증감 문장 세 부분으로 나뉩니다. 초기화 문장은 반복문이 시작되기 전에 실행됩니다. 보통 카운터 변수를 초기화하는 데 사용합니다. 조건은 반복문이 한 번 실행될 때마다 평가됩니다. 한 번 실행되는 것을 한 번의 반복이라고 합니다. 조건이 `true`이면 반복문 내부의 코드 블록이 실행됩니다. 조건이 `false`이면 반복문이 정지하고 다음 코드 블록으로 넘어갑니다. 증감 문장은 반복문이 한 번 실행된 후 실행됩니다. 보통 카운터 변수를 증가시키거나 감소시키는 데 사용합니다.

:::interactive_editor

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

:::

- **`for...of` 반복문**: 이 반복문은 반복 가능한 값들을 순회할 때 사용합니다. 반복 가능한 예로는 배열과 문자열이 있습니다.

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

:::

- **`for...in` 반복문**: 이 반복문은 객체의 프로프를 순회할 때 가장 적합합니다. 이 반복문은 상속된 프로프와 숫자가 아닌 프로프를 포함해 객체의 모든 열거 가능한 프로프를 순회합니다.

:::interactive_editor

```js
const fruit = {
  name: 'apple',
  color: 'red',
  price: 0.99
};

for (const prop in fruit) {
  console.log(fruit[prop]);
}
```

:::

- **`while` 반복문**: 이 반복문은 조건이 `true`인 동안 코드 블록을 실행합니다.

:::interactive_editor

```js
let i = 5;

while (i > 0) {
  console.log(i);
  i--;
}
```

:::

- **`do...while` 반복문**: 이 반복문은 조건을 확인하기 전에 코드 블록을 최소 한 번 실행합니다.

```js
let userInput;

do {
  userInput = prompt("Please enter a number between 1 and 10");
} while (Number(userInput) < 1 || Number(userInput) > 10);

alert("You entered a valid number!");
```

## `break`와 `continue` 문장

- **정의**: `break` 문장은 반복문을 조기에 종료할 때 사용하며, `continue` 문장은 현재 반복을 건너뛰고 다음 반복으로 넘어갈 때 사용합니다.

:::interactive_editor

```js
// Example of break statement
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, and 4

// Example of continue statement 
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, 4, 6, 7, 8, and 9
```

:::
