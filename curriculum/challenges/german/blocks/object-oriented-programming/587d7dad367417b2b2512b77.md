# --description--

<dfn>Konstruktoren</dfn> sind Funktionen, die neue Objekte erstellen. Sie definieren Eigenschaften und Verhaltensweisen, die zum neuen Objekt gehören werden. Man kann sie sich wie eine Vorlage für die Erstellung neuer Objekte vorstellen.

Hier ist ein Beispiel für einen Konstruktor:

```js
function Bird() {
  this.name = "Albert";
  this.color = "blue";
  this.numLegs = 2;
}
```

Dieser Konstruktor definiert ein `Bird`-Objekt mit den Eigenschaften `name`, `color` und `numLegs`, die jeweils auf Albert, blau und 2 gesetzt sind. Konstruktoren folgen einigen Konventionen:

<ul><li>Konstruktoren werden mit einem großgeschriebenen Namen definiert, um sie von anderen Funktionen zu unterscheiden, die keine <code>constructors</code> sind.</li><li>Konstruktoren verwenden das Schlüsselwort <code>this</code>, um Eigenschaften des Objekts zu setzen, das sie erstellen werden. Innerhalb des Konstruktors bezieht sich <code>this</code> auf das neue Objekt, das erstellt wird.</li><li>Konstruktoren definieren Eigenschaften und Verhaltensweisen, anstatt wie andere Funktionen einen Wert zurückzugeben.</li></ul>
