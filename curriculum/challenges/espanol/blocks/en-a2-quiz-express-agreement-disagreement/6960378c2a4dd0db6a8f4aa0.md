### --question--

#### --text--

¿Qué expresan `while` y `still` en esta oración: `While I understand your point, I still think we should wait`?

#### --distractors--

`While` se usa para tiempo, y `still` se usa para secuencia.

---

`While` se usa para dar una razón, y `still` se usa para mostrar un resultado.

---

`While` se usa para acuerdo, y `still` se usa para terminar la discusión.

#### --answer--

`While` se usa para mostrar contraste, y `still` se usa para mostrar que la misma opinión continúa.
