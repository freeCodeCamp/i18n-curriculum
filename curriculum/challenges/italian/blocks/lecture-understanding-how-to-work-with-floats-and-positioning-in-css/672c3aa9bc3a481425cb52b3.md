# --interactive--

La proprietà `z-index` in CSS serve a controllare l’ordine di sovrapposizione verticale degli elementi posizionati che si sovrappongono sulla pagina. Quando più elementi sono impilati uno sopra l’altro, il valore `z-index` determina quale elemento appare in cima. Più alto è il valore `z-index`, più l’elemento è vicino all’osservatore, mentre valori più bassi posizionano l’elemento più indietro nella pila.

Tuttavia, la proprietà `z-index` funziona solo sugli elementi posizionati, cioè l’elemento deve avere un valore di position diverso da `static`, come `relative`, `absolute` o `fixed`. Il valore `z-index` predefinito è `auto`, che posiziona l’elemento nell’ordine di sovrapposizione predefinito. Vediamo un esempio con tre riquadri:

```html
<div class="container">
  <div class="box1">Box 1</div>
  <div class="box2">Box 2</div>
  <div class="box3">Box 3</div>
</div>
```

Ora possiamo applicare alcuni stili per posizionare i riquadri in modo che si sovrappongano così:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box1">Box 1</div>
  <div class="box2">Box 2</div>
  <div class="box3">Box 3</div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

.box1 {
  position: absolute;
  z-index: 1;
  background: lightcoral;
  top: 20px;
  left: 20px;
  width: 100px;
  height: 100px;
}

.box2 {
  position: absolute;
  z-index: 3;
  background: gold;
  top: 40px;
  left: 40px;
  width: 100px;
  height: 100px;
}

.box3 {
  position: absolute;
  z-index: 2;
  background: lightgreen;
  top: 60px;
  left: 60px;
  width: 100px;
  height: 100px;
}
```

:::

Per il contenitore, il posizionamento sarà impostato su `relative` e tutti i riquadri annidati all’interno avranno un posizionamento `absolute`. Ogni riquadro ha un valore diverso per la proprietà `z-index`, il che fa sì che i riquadri si sovrappongano uno sopra l’altro.

Puoi pensare a `z-index` come a un modo per creare livelli su una pagina web, e gli elementi con valori `z-index` più alti saranno posizionati sopra quelli con valori più bassi. Questo è particolarmente utile per controllare il comportamento degli elementi sovrapposti in layout complessi, come modali, pop-up o tooltip.
