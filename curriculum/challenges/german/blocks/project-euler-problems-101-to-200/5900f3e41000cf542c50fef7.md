# --description--

Sei `r` der Rest, wenn ${(a − 1)}^n + {(a + 1)}^n$ durch $a^2$ geteilt wird.

Zum Beispiel, wenn $a = 7$ und $n = 3$, dann ist $r = 42$: $6^3 + 8^3 = 728 ≡ 42 \\ \text{mod}\\ 49$. Und während `n` variiert, wird auch `r` variieren, aber für $a = 7$ stellt sich heraus, dass $r_{max} = 42$ ist.

Für $3 ≤ a ≤ 1000$ finden Sie $\sum{r}_{max}$.
