# --description--

このクイズは基本的なプログラミングの概念の理解度をチェックします。

クイズに合格するには、以下の20問中少なくとも18問に正解する必要があります。

各質問を読み、正しい答えを選んでください。各質問には正しい答えが1つだけあります。

アクセシビリティのために、聴覚に障害のある学習者向けにトランスクリプトが用意されています。必要な場合のみ使用することをお勧めします。
