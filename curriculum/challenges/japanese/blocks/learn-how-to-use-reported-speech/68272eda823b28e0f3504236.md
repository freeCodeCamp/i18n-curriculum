# --explanation--

`That sounds` + 形容詞は、誰かの発言に対して反応する一般的な方法です。話者がその状況についてどう感じているかを示します。例えば：

`That sounds interesting.` – これは、その話題や考えが興味深い、もっと知りたいと思ったことを意味します。
