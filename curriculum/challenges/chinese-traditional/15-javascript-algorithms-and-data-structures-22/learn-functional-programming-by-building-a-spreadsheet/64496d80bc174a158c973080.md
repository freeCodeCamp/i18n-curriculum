# --description--

使用三元語法，使用 `isEven` 函數檢查 `length` 是否均勻。 如果是，則返回 `middle` 索引處的數字與其後一個數字的平均值。 如果是奇數，則返回 `middle` 索引處的數字 - 你需要將 `middle` 值向上舍入。
