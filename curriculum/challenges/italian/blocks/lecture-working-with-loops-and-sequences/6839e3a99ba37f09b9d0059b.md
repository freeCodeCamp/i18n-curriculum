# --description--

Una tupla è un tipo di dato Python usato per creare una sequenza ordinata di valori. Le tuple possono contenere un insieme misto di tipi di dati come questo:

```py
developer = ('Alice', 34, 'Rust Developer')
```

Le tuple sono simili alle liste, ma mentre le liste sono un tipo di dato mutabile, le tuple sono immutabili. Questo significa che gli elementi in una tupla non possono essere modificati una volta creata.

Se provi ad aggiornare uno degli elementi nella tupla, otterrai un `TypeError`:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: 'tuple' object does not support item assignment
"""
```

Per consultare un elemento da una tupla, puoi usare la notazione a parentesi e il numero dell’indice:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

Se hai bisogno di consultare elementi partendo dalla fine di una tupla, puoi usare l’indicizzazione negativa. Ecco un esempio di utilizzo di un indice negativo per consultare il penultimo elemento in una tupla:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

Se provi a passare un numero di indice che supera o è uguale alla lunghezza della tupla, riceverai un `IndexError` come questo:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list index out of range
"""
```

Un altro modo per creare una tupla è usando il costruttore `tuple()` così:

```py
developer = 'Jessica'
tuple(developer) # ('J', 'e', 's', 's', 'i', 'c', 'a')
```

Per il costruttore `tuple()` puoi passare diversi iterabili come stringhe, liste e anche altre tuple.

Per verificare se un elemento è in una tupla, puoi usare la parola chiave `in` così:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

Puoi anche decomprimere elementi da una tupla proprio come hai fatto con le liste:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

In questo esempio, `name` ha il valore `'Alice'`, `age` ha il valore `34` e `job` ha il valore `'Rust Developer'`.

Se hai bisogno di raccogliere gli elementi rimanenti da una tupla, puoi usare l’operatore asterisco (`*`) così:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

Qui, `name` ha il valore `'Alice'` e `rest` è una lista composta dal numero `34` e dalla stringa `'Rust Developer'`.

Proprio come con una lista, puoi usare l’operatore slice su una tupla per estrarne una porzione. Ecco un esempio di estrazione degli elementi `'pie'` e `'cookies'` in una tupla separata:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

Ricorda che il primo numero rappresenta l’indice di partenza per l’estrazione mentre il secondo numero rappresenta l’indice finale. Ma nota che l’elemento all’indice finale non è incluso nella tupla estratta.

Se hai bisogno di rimuovere un elemento da una tupla, non è possibile perché le tuple sono immutabili. Quindi questo esempio produrrà un errore:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

Quando potresti quindi usare una tupla invece di una lista?

Se ti serve una raccolta dinamica di elementi dove puoi aggiungere, rimuovere e aggiornare elementi, allora dovresti usare una lista. Se sai di lavorare con una raccolta fissa e immutabile di dati, allora dovresti usare una tupla.

Nella prossima lezione, vedremo alcuni metodi comuni che userai quando lavori con le tuple.
