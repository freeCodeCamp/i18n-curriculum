# --description--

العبارة `why don't` تُستخدم غالبًا لتقديم اقتراح أو عرض. يتبعها فاعل (مثل `we` أو `you` أو `they`) وفعل في صورته الأساسية. البنية هي `why don't + subject + base verb`. هذه طريقة مهذبة لاقتراح القيام بفعل أو طلب من شخص أن يفعل شيئًا.

أمثلة:

`Why don't we go to the cinema tonight?` المتحدث يقترح الذهاب إلى السينما.

`Why don't you try restarting the computer?` المتحدث يقترح إعادة تشغيل الكمبيوتر كحل.

العبارة `check out` تعني فحص أو إلقاء نظرة عن قرب على شيء ما، وتُستخدم عادة عند اقتراح استكشاف أو معرفة المزيد عن شيء. على سبيل المثال، `Let's check out the new restaurant` تعني زيارة وتجربة المطعم.
