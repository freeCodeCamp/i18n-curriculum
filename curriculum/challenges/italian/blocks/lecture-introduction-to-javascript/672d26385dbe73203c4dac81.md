# --interactive--

JavaScript è un potente linguaggio di programmazione che porta interattività e comportamento dinamico ai siti web.

Mentre HTML e CSS sono linguaggi di markup usati per strutturare il contenuto e definire lo stile degli elementi su una pagina, JavaScript va oltre permettendo funzionalità più complesse, come gestire l’input dell’utente, animare elementi e persino costruire applicazioni web complete.

Per esempio, quando clicchi un pulsante, invii un modulo o passi il mouse su un menu, JavaScript determina come si comporta la pagina.

Ecco un esempio di come questi tre lavorano insieme:

:::interactive_editor

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: green;
    }
  </style>
</head>
<body>
  <h1>Hello, World!</h1>
  <button onclick="alert('Button clicked!')">Click me</button>
</body>
</html>
```

:::

In questo esempio, HTML viene usato per definire il contenuto: un’intestazione (un elemento `h1`) e un pulsante (l’elemento `button`).

CSS viene usato per applicare lo stile all’intestazione, come rendere il testo verde. JavaScript viene usato per mostrare un messaggio di avviso quando il pulsante viene cliccato.

In sintesi, HTML fornisce la struttura, CSS aggiunge lo stile e JavaScript abilita il comportamento, creando un’esperienza web completa e interattiva.
