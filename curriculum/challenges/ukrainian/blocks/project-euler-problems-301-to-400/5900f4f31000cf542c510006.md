# --description--

Нехай $s_k$ — кількість одиниць при записі чисел від 0 до $k$ у двійковій системі.

Наприклад, записуючи числа від 0 до 5 у двійковому вигляді, маємо 0, 1, 10, 11, 100, 101. Всього сім одиниць, отже $s_5 = 7$.

Послідовність $S = \\{s_k : k ≥ 0\\}$ починається як $\\{0, 1, 2, 4, 5, 7, 9, 12, \ldots\\}$.

Гру грають двоє гравців. Перед початком гри вибирають число $n$. Фішка $c$ починає з 0. Кожен хід гравець вибирає число від 1 до $n$ (включно) і збільшує $c$ на це число. Отримане значення $c$ має належати послідовності $S$. Якщо ходів більше немає, гравець програє.

Наприклад, при $n = 5$ і початковому $c = 0$:

- Гравець 1 вибирає 4, отже $c$ стає $0 + 4 = 4$.
- Гравець 2 вибирає 5, отже $c$ стає $4 + 5 = 9$.
- Гравець 1 вибирає 3, отже $c$ стає $9 + 3 = 12$.
- і так далі.

Зверніть увагу, що $c$ завжди має належати $S$, і кожен гравець може збільшити $c$ не більше ніж на $n$.

Нехай $M(n)$ — найбільше число, яке перший гравець може вибрати у свій перший хід, щоб гарантувати перемогу, і $M(n) = 0$, якщо такого ходу немає. Наприклад, $M(2) = 2$, $M(7) = 1$ і $M(20) = 4$.

Перевірено, що $\sum M{(n)}^3 = 8150$ для $1 ≤ n ≤ 20$.

Знайдіть $\sum M{(n)}^3$ для $1 ≤ n ≤ 1000$.
