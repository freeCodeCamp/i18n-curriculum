# --explanation--

Maria sagt `at the end of each day`, was eindeutig zeigt, dass die Sicherungskopien täglich geplant werden. Das bedeutet, die Aufgabe ist so vorgesehen, dass sie einmal am Tag ausgeführt wird. Zum Beispiel:

- `At the end of each day, the system runs a data backup.` – Das System speichert jeden Abend alle Daten, um Datenverlust zu verhindern.

- `We ensure the servers are updated at the end of each day.` – Updates werden nachts auf den Servern angewendet, um Unterbrechungen während der Arbeitszeit zu vermeiden.
