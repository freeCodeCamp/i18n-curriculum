# --description--

可算名詞は数えられるものです。1つ、2つ、3つ以上と数えられます。数字や量指定子の`many`、`a few`、`several`を使って可算名詞を表現できます。例えば、appleの場合は`one apple`、`two apples`、`many apples`のように言えます。

不可算名詞は1つずつ数えられないものです。通常、複数形にはなりません。数字を直接使わず、代わりに量指定子の`much`、`a little`、`some`を使って表現します。例えば、waterの場合は`one water`や`two waters`とは言いません。`much water`や`some water`と言います。
