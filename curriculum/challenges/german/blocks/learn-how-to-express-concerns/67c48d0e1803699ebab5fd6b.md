# --explanation--

Wenn Sie ein Adjektiv gefolgt von einem Infinitiv verwenden, wird die Verneinung gebildet, indem `not to` vor das Verb gesetzt wird. Zum Beispiel:

- Bejahend: `She was happy to help with the project.` – Das bedeutet, dass das Helfen bei dem Projekt sie glücklich gemacht hat.

- Verneinend: `He was careful not to forget the deadline.` – Das bedeutet, dass er dafür gesorgt hat, dass er sich daran erinnert.

`To burn out` bedeutet, durch zu viel Arbeit extrem müde oder erschöpft zu werden. Zum Beispiel:

`If we don't take breaks, we might burn out before the project is finished.` – Das bedeutet, dass Arbeiten ohne Pause zu Erschöpfung führen kann.
