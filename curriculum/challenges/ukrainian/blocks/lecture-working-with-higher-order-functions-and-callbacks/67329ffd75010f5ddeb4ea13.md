# --interactive--

У JavaScript функція зворотного виклику — це функція, яка передається як аргумент іншій функції, щоб зовнішня функція могла викликати її у певний момент.

Ця концепція є фундаментальною для розуміння багатьох аспектів JavaScript, зокрема того, як працює метод `forEach`.

Почнемо з розуміння, що таке функція зворотного виклику у простому контексті.

Уявіть, що у вас є функція, і всередині неї ви хочете виконати якусь дію кілька разів. Замість того, щоб писати весь код у одній великій функції, ви можете передати другу функцію (функцію зворотного виклику), яку потрібно виконати щоразу, коли потрібно зробити цю конкретну дію. Це дозволяє створювати більш гнучкий і модульний код.

Тепер давайте познайомимося з методом `forEach`. `forEach` — це вбудований метод для масивів у JavaScript. Він дозволяє ітерувати над кожним елементом масиву та виконувати операцію над кожним елементом. Операція, яку ви хочете виконати, визначається у функції зворотного виклику, яку ви передаєте `forEach`.

Ось приклад того, як `forEach` працює з функцією зворотного виклику:

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];

numbers.forEach(function(number) {
  console.log(number * 2);
});
```

:::

У цьому прикладі у нас є масив чисел. Ми використовуємо метод `forEach` для цього масиву і передаємо функцію зворотного виклику як аргумент `forEach`.

Ця функція зворотного виклику приймає один параметр, який представляє кожен елемент масиву. Метод `forEach` викликає цю функцію зворотного виклику один раз для кожного елемента масиву.

Функція зворотного виклику в цьому випадку множить кожне число на `2` і виводить результат у консоль. Тож, коли цей код запускається, він виведе:

```md
2
4
6
8
10
```

Важливо розуміти, що функція зворотного виклику викликається один раз для кожного елемента масиву, послідовно. `forEach` бере на себе цикл за вас, тому вам не потрібно писати цикл `for` самостійно.

Ви також можете використовувати стрілкову функцію як функцію зворотного виклику, що зробить ваш код ще більш лаконічним:

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];
numbers.forEach(number => console.log(number * 2));
```

:::

Це робить те саме, що й попередній приклад, але з меншим обсягом коду.

Функція зворотного виклику в `forEach` може фактично приймати до трьох аргументів: поточний елемент, індекс поточного елемента та масив, на якому був викликаний `forEach`.

Ось приклад, який використовує всі три:

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];
numbers.forEach((number, index, array) => {
  console.log(`Element ${number} is at index ${index} in array ${array}`);
});
```

:::

Це виведе інформацію про кожен елемент, його індекс і початковий масив.

Розуміння функцій зворотного виклику та методів, таких як `forEach`, є важливим на вашому шляху вивчення JavaScript. Вони становлять основу для багатьох більш складних концепцій у мові, особливо в асинхронному програмуванні, про яке ви дізнаєтеся у майбутніх уроках.
