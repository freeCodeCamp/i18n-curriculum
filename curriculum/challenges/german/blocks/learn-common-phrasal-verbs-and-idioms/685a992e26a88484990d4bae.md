# --explanation--

`Let's do that` ist eine lockere Art, einer Idee zuzustimmen und sie zu unterstützen. Es zeigt, dass Sie die Idee gut finden und bereit sind, sie umzusetzen. Zum Beispiel:

- **James:** `Should we order pizza for lunch?` – Diese Person schlägt das Menü für das Mittagessen vor.

- **Phoebe:** `Yes, let's do that.` – Das bedeutet, dass die sprechende Person zustimmt und den Vorschlag umsetzen möchte.
