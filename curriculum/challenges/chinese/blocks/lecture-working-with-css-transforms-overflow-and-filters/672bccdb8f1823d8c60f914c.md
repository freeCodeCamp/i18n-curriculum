# --interactive--

CSS 重置是一种样式表，可移除网络浏览器应用于 HTML 元素的全部或部分默认格式。 例如，你可能已经注意到，某些元素，如段落和标题，已经默认带有页边距，即使你没有在自定义样式表中明确定义它们。 

你还可以在各种 HTML 元素的各种属性中看到这种情况。 不同的浏览器也会对 HTML 元素应用不同的默认样式。 Google Chrome 浏览器中的默认样式可能与 Mozilla Firefox 或 Microsoft Edge 中的不完全相同。 这会导致不同浏览器的样式不一致，应尽量避免。

为此，CSS 重置会删除默认样式。 通过移除全部或部分默认样式，可以为设计提供一致的基线，最大限度地减少不同浏览器和设备之间可能出现的不一致。 移除默认样式还能让样式设计过程变得更简单，因为你只能看到在自定义样式表中明确应用的样式。 CSS 重置主要有两种方法：你可以定义自定义 CSS 重置或使用第三方 CSS 重置。

自定义 CSS 重置是根据项目需要从头开始创建的样式表。 这样，你就可以控制重置的具体样式，并有很大的灵活性空间。 不过，你也需要投入时间来开发和维护样式表。 下面是一个非常常见的 CSS 规则示例，用于重置所有 HTML 元素的外边距和内边距：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Example Heading</h1>
<p>This is a paragraph.</p>
```

```css
* {
  margin: 0;
  padding: 0;
}
```

:::

通常写在 CSS 样式表的顶部。  The asterisk selector is a wildcard selector that matches all HTML elements, so they will have a default margin of zero and a default padding of zero on all four sides. 这将为你提供一个起点，然后你就可以在样式表中进一步使用更具体的 CSS 选择器来定制它们。

你可以使用这种方法选择任何 `HTML` 元素并重置其默认样式。只需创建一个自定义样式表并使用适当的 `CSS` 选择器来匹配元素并设置样式。 

但这可能是一个耗时的过程。如果你想节省时间，你也可以使用第三方 CSS 重置。这些样式表已经预先构建好，因此你只需下载它们并将其添加到你的项目中即可。一个很好的第三方 CSS 重置示例是 `Normalize.css`。该样式表对广泛的 HTML 元素的样式进行规范化，同时仍保留一些有用的默认样式，特别是那些对无障碍非常重要的样式。它还修正了常见的错误和样式不一致。

另一个选项是 `sanitize.css` 。 这是一个 CSS 库，可用于确保默认样式在所有主流现代浏览器中保持一致。  This library is developed alongside `Normalize.css`, so they evolve together. 你还可以下载单独的样式表，用于规范表格和排版等特定用途。

可供选择的方案很多，但你应该选择最适合项目需要的方案。 你还可以将这两种方法结合起来，同时使用第三方 CSS 重置和自定义重置。

在使用 CSS 重置时，考虑到可访问性也很重要。 每个人都应能访问你的网络应用程序。 因此，你不应该重置可能有助于屏幕阅读器或其他辅助技术的样式。 你还应考虑这些额外的样式表可能对应用程序性能造成的影响，因为在应用自定义样式之前，必须先下载这些样式表。 

通过删除默认样式，CSS 重置为你提供了一个实现设计的空白起点。 这样，不同浏览器和设备之间的用户体验就会更加统一和一致。
