# --explanation--

在日常口语中，当意思仍然清楚时，你经常会省略某些词。在这种情况下，`Anything else we should discuss?` 是 `Is there anything else we should discuss?` 的简略形式。这种省略在对话中很常见。另一个例子：

`Need any help?` 是 `Do you need any help?` 的简短表达——省略了主语和辅助动词，但意思保持不变。
