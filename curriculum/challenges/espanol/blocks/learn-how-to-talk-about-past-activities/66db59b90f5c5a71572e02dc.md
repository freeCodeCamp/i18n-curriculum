# --questions--

## --text--

¿Qué opción muestra que algo **ha ocurrido** antes de lo esperado?

## --answers--

`I have already completed the assignment.`

---

`I will already complete the assignment tomorrow.`

### --feedback--

`Already` no se usa con el tiempo futuro. Indica algo que ocurrió antes de ahora.

---

`I already complete the assignment every day.`

### --feedback--

`Already` no se usa con el presente simple para hablar de acciones regulares.

---

`I had already completed the assignment yesterday.`

### --feedback--

Esta oración usa el `Past Perfect` (`had completed`), no el `Present Perfect`. El `Past Perfect` indica una acción antes de otra acción pasada, no algo que sucede antes de lo esperado en el presente.

# --explanation--

`Already` se usa con el tiempo `Present Perfect` para indicar que algo ha ocurrido antes de lo esperado o antes de ahora. Por ejemplo:

- `She has already finished her work.` - La acción se completó antes de lo esperado.

- `They have already left for the meeting.` - Se fueron antes de lo esperado.
