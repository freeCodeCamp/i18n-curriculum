# --description--

JavaScript 中对变量名有两个限制：

1. 名称必须仅包含字母、数字或符号 `$` 和 `_`。
1. 第一个字符不能是数字。

有效名称示例：

```js
let userName;
let test123;
```

有趣的是——美元符号 `'$'` 和下划线 `'_'` 也可以用于名称。它们是普通符号，就像字母一样，没有任何特殊含义。

这些名称是有效的：

```js
let $ = 1; // declared a variable with the name "$"
let _ = 2; // and now a variable with the name "_"

console.log($ + _); // 3
```

错误变量名示例：

```js
let 1a; // cannot start with a digit

let my-name; // hyphens '-' aren't allowed in the name
```
