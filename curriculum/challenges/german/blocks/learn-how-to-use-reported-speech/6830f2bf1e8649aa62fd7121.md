# --explanation--

Ein `image` ist eine visuelle Darstellung, wie ein Foto oder Bild, die hilft, etwas klar zu zeigen. Zum Beispiel:

`The homepage banner image was updated to reflect the new campaign.` – Das bedeutet, dass ein neues Bild zur optischen Aufwertung hinzugefügt wurde.

`Graphics` sind visuelle Elemente wie Illustrationen, Symbole oder Designs, die die Kommunikation oder das Design unterstützen. Zum Beispiel:

`The infographic used clear graphics to show the survey results.` – Das bedeutet, dass visuelle Hilfsmittel verwendet wurden, um die Daten leichter verständlich zu machen.
