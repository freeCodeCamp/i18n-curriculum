# --explanation--

`Support` означає обробляти або дозволяти чомусь працювати правильно. Наприклад:  

`The app can support 1,000 users at the same time.` – Воно може обробити цю кількість без збоїв.

`At once` означає виконувати кілька речей одночасно. Наприклад:

`The software can open many files at once.` – Всі файли відкриваються разом, а не по одному.
