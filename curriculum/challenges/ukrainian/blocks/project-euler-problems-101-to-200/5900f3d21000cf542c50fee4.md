# --description--

Якщо нам дано перші k членів послідовності, неможливо з певністю сказати значення наступного члена, оскільки існує нескінченно багато многочленів, які можуть моделювати цю послідовність.

Як приклад розглянемо послідовність кубічних чисел. Вона визначається твірною функцією $u_n = n^3$: 1, 8, 27, 64, 125, 216, \ldots

Припустимо, що нам дали лише перші два члени цієї послідовності. Виходячи з принципу «простіше — краще», слід припустити лінійну залежність і передбачити наступний член рівним 15 (загальна різниця 7). Навіть якщо нам дали перші три члени, за тим самим принципом простоти слід припустити квадратичну залежність.

Позначимо $OP(k, n)$ як $n^{й}$ член оптимальної многочленної твірної функції для перших k членів послідовності. Має бути зрозуміло, що $OP(k, n)$ точно відтворюватиме члени послідовності для $n ≤ k$, а потенційно перший неправильний член (FIT) буде $OP(k, k+1)$; у такому випадку ми назвемо його поганим OP (BOP).

Як основу, якщо нам дали лише перший член послідовності, найрозумніше припустити сталість; тобто для $n ≥ 2, OP(1, n) = u_1$.

Отже, отримуємо такі OP для кубічної послідовності:

$$\begin{array}{ll}
  OP(1, n) = 1          & 1, {\color{red}1}, 1, 1, \ldots     \\\\
  OP(2, n) = 7n−6       & 1, 8, {\color{red}{15}}, \ldots     \\\\
  OP(3, n) = 6n^2−11n+6 & 1, 8, 27, {\color{red}{58}}, \ldots \\\\
  OP(4, n) = n^3        & 1, 8, 27, 64, 125, \ldots
\end{array}$$

Очевидно, що для k ≥ 4 поганих OP не існує. Розглядаючи суму FIT, згенерованих BOP (позначених вище $\color{red}{червоним}$), отримуємо 1 + 15 + 58 = 74. Розглянемо наступну многочленну твірну функцію десятого степеня:

$$u_n = 1 − n + n^2 − n^3 + n^4 − n^5 + n^6 − n^7 + n^8 − n^9 + n^{10}$$

Знайдіть суму FIT для BOP.
