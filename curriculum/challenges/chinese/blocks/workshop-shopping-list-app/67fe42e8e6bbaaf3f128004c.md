# --description--

如果你检查控制台，你会看到有关缺少 `key` 属性的警告。 这是因为 React 需要为列表中的每个元素提供一个唯一的键，以帮助它识别哪些项目已更改、添加或删除。 出于性能原因，这也很重要。

要解决此问题，请在 `map()` 方法内的 `li` 元素中添加 `key` 属性。 将 `key` 设置为 `item` 的值。 一般来说，你应该使用像 id 这样的唯一标识符作为 `key`，但由于项目都是唯一的字符串，因此在这种情况下你可以使用字符串本身。
