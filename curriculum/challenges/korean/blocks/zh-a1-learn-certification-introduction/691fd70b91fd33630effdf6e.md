# --description--

중국어는 여러 지역 방언으로 사용됩니다. 이 커리큘럼에서는 가장 널리 쓰이는 표준 중국어인 **만다린**을 배우게 됩니다.

중국어는 간체와 번체 두 가지 주요 문자 체계로 쓰입니다. 이 커리큘럼은 **간체 중국어**에 집중합니다.

중국어 문자는 음성적이지 않기 때문에, 발음을 나타내기 위해 **병음**이라는 체계가 사용됩니다. 병음은 라틴 알파벳을 이용해 중국어 발음을 표시하는 로마자 표기법입니다. 이는 초보자가 발음을 배우고 디지털 기기에서 중국어를 입력하는 데 필수적인 도구입니다.

이제 두 글자씩 이루어진 첫 두 개의 중국어 단어를 입력할 차례입니다. 믿기 어려울 수도 있겠지만, freeCodeCamp는 이를 쉽게 할 수 있도록 **병음 성조 입력**과 **병음-한자 입력**을 개발했습니다.

이번 과제에서는 병음 성조 입력을 사용해 병음을 쓰는 연습을 하게 됩니다.

오디오에서 Wang Hua가 중국어로 "안녕하세요, 세계"라고 말합니다. `ni3 hao3`와 `shi4 jie4`을 입력해 보세요. 실수하면 잘못된 부분을 삭제하고 다시 입력하면 됩니다.

아래 설명 섹션을 펼쳐서 내부에서 어떤 일이 일어나는지 확인하세요.

# --explanation--
  
대부분의 경우, 각 중국어 글자는 하나의 병음 음절로 나타내며, 각 음절은 초성(자음), 종성(항상 모음 포함), 그리고 음의 높낮이 변화를 나타내는 성조 부호로 구성됩니다.

컴퓨터나 모바일 기기의 대부분 내장 중국어 입력기와 서드파티 앱에서는 보통 성조 부호를 입력하지 않습니다. 초성과 종성만 입력한 뒤 목록에서 올바른 한자를 선택합니다.

하지만 성조 부호를 입력하고 보는 것은 초보자가 중국어 글자를 기억하는 데 도움이 됩니다.

병음 성조 입력을 사용하면, 글자와 성조 번호를 함께 입력해 자동으로 성조 부호가 붙은 병음으로 변환할 수 있습니다. 예를 들어, 여기서 `ni3 hao3`는 `nǐ hǎo`로, `shi4 jie4`는 `shì jiè`로 변환됩니다.

이 입력 방식은 이 커리큘럼 2장 병음 학습에서 자주 사용하게 됩니다.
