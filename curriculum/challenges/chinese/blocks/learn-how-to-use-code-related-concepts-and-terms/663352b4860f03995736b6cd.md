# --description--

`Store information` 意味着将数据保存在特定位置，通常是在变量中。变量就像你代码中的容器，保存信息，例如数字、文本或其他类型的数据。你可以将它们视为命名的储存点，稍后可以在代码中访问 `stored information`。

例如，`int age = 30;` 这里，变量 `age` 保存数字 30。

单词 `like` 用作介词，用于举例或比较相似的事物。

例如，如果你说，`I love fruits like apples and oranges.` 你正在使用 `like` 来举例说明你喜欢的水果类型，暗示该列表中可能还有更多水果。
