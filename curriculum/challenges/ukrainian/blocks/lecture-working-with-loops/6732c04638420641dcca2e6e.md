# --interactive--

Цикл `for...of` використовується, коли потрібно перебрати значення з ітерованого об’єкта. Прикладами ітерованих об’єктів є масиви та рядки.

Ось базовий синтаксис циклу `for...of`:

```js
for (variable of iterable) {
  // code block to be executed
}
```

Змінна в прикладі представляє поточне значення ітерованого об’єкта, по якому відбувається перебір.

Якщо у вас є масив чисел, змінна буде поточним числом у масиві. Якщо у вас є рядок, змінна буде поточним символом у рядку.

Давайте розглянемо кілька прикладів, щоб краще зрозуміти, як працює цикл `for...of`.

У цьому першому прикладі у нас є масив чисел, і ми хочемо перебрати кожне число та вивести його в консоль.

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

:::

Ми створили змінну з назвою `num`, яка представлятиме поточне число в масиві. Для першої ітерації `num` буде `1`, для другої ітерації `num` буде `2`, і так далі.

Всередині циклу ми виводимо поточне число в консоль.

Ось ще один приклад, де у нас є рядок, і ми хочемо перебрати кожен символ і вивести його в консоль.

:::interactive_editor

```js
const str = 'freeCodeCamp';

for (let char of str) {
  console.log(char);
}
```

:::

У цьому прикладі ми створили змінну з назвою `char`, яка представлятиме поточний символ у рядку.

Для кожної ітерації цикл виводитиме поточний символ у консоль.

Важливо зазначити, що при оголошенні змінної в циклі `let` можна використовувати `const` або `for...of`.

Якщо ви збираєтеся використовувати `const`, переконайтеся, що значення змінної не змінюється всередині циклу. Якщо змінюється, виникне помилка.

Ось приклад використання `const`, що призводить до помилки:

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
  num = num + 1; // This will cause an error
}
```

У цьому прикладі ми намагаємося змінити значення `num` всередині циклу. Оскільки ми оголосили `num` за допомогою `const`, отримаємо помилку. Тож, якщо потрібно змінювати значення змінної всередині циклу, використовуйте `let`.

Розглянемо останній приклад із масивом об’єктів.

:::interactive_editor

```js
const people = [
  { name: 'John', age: 30 },
  { name: 'Jane', age: 25 },
  { name: 'Jim', age: 40 }
];

for (const person of people) {
  console.log(`${person.name} is ${person.age} years old`);
}
```

:::

У цьому прикладі у нас є масив об’єктів з назвою `people`. Кожен об’єкт має властивості `name` і `age`.

Коли ми перебираємо масив, створюємо змінну з назвою `person`, яка представлятиме поточний об’єкт у масиві.

Всередині циклу ми виводимо повідомлення в консоль.

Перше повідомлення буде `John is 30 years old`, друге — `Jane is 25 years old`, а третє — `Jim is 40 years old`.

Цикли `for...of` дуже корисні, коли потрібно перебрати значення з ітерованого об’єкта, такого як масив або рядок. Вони також легко читаються і можуть зробити ваш код більш лаконічним.
