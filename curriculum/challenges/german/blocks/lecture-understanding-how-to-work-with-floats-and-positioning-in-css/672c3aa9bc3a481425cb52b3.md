# --interactive--

Die `z-index`-Eigenschaft in CSS wird verwendet, um die vertikale Stapelreihenfolge von positionierten Elementen zu steuern, die sich auf der Seite überlappen. Wenn mehrere Elemente übereinander gestapelt sind, bestimmt der `z-index`-Wert, welches Element oben angezeigt wird. Je höher der `z-index`-Wert ist, desto näher ist das Element am Betrachter, während niedrigere Werte das Element weiter hinten im Stapel platzieren.

Die `z-index` funktioniert jedoch nur bei Elementen, die positioniert sind, das heißt, das Element muss einen Positionswert ungleich `static` haben, wie zum Beispiel `relative`, `absolute` oder `fixed`. Der Standard-`z-index`-Wert ist `auto`, der das Element in der Standard-Stapelreihenfolge platziert. Schauen wir uns ein Beispiel mit drei Kästchen an:

```html
<div class="container">
  <div class="box1">Box 1</div>
  <div class="box2">Box 2</div>
  <div class="box3">Box 3</div>
</div>
```

Jetzt können wir einige Styles anwenden, um die Kästchen so zu positionieren, dass sie sich überlappen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box1">Box 1</div>
  <div class="box2">Box 2</div>
  <div class="box3">Box 3</div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

.box1 {
  position: absolute;
  z-index: 1;
  background: lightcoral;
  top: 20px;
  left: 20px;
  width: 100px;
  height: 100px;
}

.box2 {
  position: absolute;
  z-index: 3;
  background: gold;
  top: 40px;
  left: 40px;
  width: 100px;
  height: 100px;
}

.box3 {
  position: absolute;
  z-index: 2;
  background: lightgreen;
  top: 60px;
  left: 60px;
  width: 100px;
  height: 100px;
}
```

:::

Für den Container wird die Positionierung auf `relative` gesetzt, und alle darin verschachtelten Kästchen erhalten eine `absolute`-Positionierung. Jedes Kästchen hat einen anderen Wert für den `z-index`, was dazu führt, dass die Kästchen übereinander geschichtet werden.

Sie können sich `z-index` als eine Möglichkeit vorstellen, auf einer Webseite Ebenen zu erstellen, wobei Elemente mit höheren `z-index`-Werten über denen mit niedrigeren Werten liegen. Das ist besonders nützlich, um zu steuern, wie sich überlappende Elemente in komplexen Layouts verhalten, wie zum Beispiel bei Modals, Pop-ups oder Tooltips.
