# --interactive--

CSS에서 가장 흥미로운 부분 중 하나는 가상 요소의 사용입니다. 여기서 "가상"은 "실제가 아닌"을 의미하므로, 가상 요소는 실제 HTML 요소와 직접 일치하지 않는 가상 또는 합성 요소입니다. 가상 요소를 사용하면 추가 HTML을 넣지 않고도 요소의 특정 부분을 스타일링하거나 콘텐츠를 삽입할 수 있습니다.

가상 요소를 적용하려면 원래 요소의 선택자에 이중 콜론(`::`)을 붙이세요. 선택자는 클래스나 ID 선택자 등 어떤 유형도 될 수 있습니다. 가상 요소의 기본 구문은 다음과 같습니다:

```css
selector::pseudo-element {
  property: value;
}
```

이 이중 콜론이 가상 클래스와 구분하는 점인데, 가상 클래스는 단일 콜론을 사용합니다.

가상 요소는 요소 콘텐츠의 특정 부분을 스타일링하거나 그 앞뒤에 콘텐츠를 삽입할 수 있지만 독립적으로 존재할 수는 없습니다. 가상 요소가 붙는 요소를 원본 요소라고 합니다.

먼저 `::before`과 `::after` 가상 요소 예제를 살펴보겠습니다. 이름에서 알 수 있듯이 `::before`은 요소 콘텐츠 바로 앞에 콘텐츠를 삽입하고 `::after`는 그 뒤에 콘텐츠를 삽입합니다.

다음은 버튼 요소 예제입니다. CSS에서 절대 위치 지정과 `::before` 가상 요소를 사용해 버튼의 `Learn More` 텍스트 앞에 별표를 추가할 것입니다. 절대 위치 지정에 대해서는 앞으로 수업에서 더 배우게 됩니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: lightseagreen;
  color: white;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
  position: relative;
}

.cta-button::before {
  content: "⭐";
  position: absolute;
  left: 3px;
  top: 8px;
  font-size: 0.75rem;
}
```

:::

`content` 속성은 버튼 텍스트 앞에 추가할 콘텐츠를 나타내는 데 사용됩니다. 이 예에서는 별표를 추가했습니다. 콘텐츠를 삽입할 뿐 아니라 스타일링도 할 수 있다는 점을 알게 될 것입니다. 다음은 같은 버튼에 `::after` 가상 요소를 적용한 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: '➡️';
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}
```

:::

`::after` 가상 요소에서는 `transition` 속성을 사용해 0.3초 동안 완화 효과와 함께 변화를 애니메이션으로 만들어 갑작스러운 변화가 아니라 부드럽고 점진적인 변형을 만듭니다. `transition` 속성에 대해서는 앞으로 수업에서 더 배우게 됩니다.

또한 `::before`와 `::after` 가상 요소를 사용해 삽입한 콘텐츠에 가상 클래스를 붙일 수 있습니다. 예를 들어, 콘텐츠에 호버 상태를 적용하는 경우입니다. 다음은 예제입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: "➡️";
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}

.cta-button:hover::after {
  transform: translateX(2px);
}
```

:::

호버 상태에서 `transform: translateX(2px)`를 사용하면 사용자가 버튼 위에 마우스를 올릴 때마다 콘텐츠가 `2px`만큼 오른쪽으로 밀립니다. `::after` 자체의 전환 속성은 이 과정이 `0.3s` 동안 진행되도록 보장합니다.

`transform` 속성은 요소를 특정 방향으로 회전, 기울이기, 크기 조절 또는 이동할 수 있게 합니다. 이에 대해서도 앞으로 수업에서 더 배우게 됩니다.

다음 예제에서는 `::first-letter` 가상 요소를 살펴보겠습니다. `::first-letter` 가상 요소는 요소 콘텐츠의 첫 글자를 대상으로 하여 스타일링할 수 있게 합니다. 다음은 단락 텍스트 예제입니다. 첫 글자를 스타일링하려면 `::first-letter` 가상 요소를 이렇게 사용할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>freeCodeCamp lets you learn to code without having to pay.</p>
```

```css
p::first-letter {
  font-size: 4rem;
}
```

:::

마지막 예제에서는 `::marker` 가상 요소를 살펴보겠습니다. 이 가상 요소는 목록 항목의 마커, 불릿 또는 번호 매기기를 선택해 스타일링할 수 있게 합니다. `::marker` 가상 요소는 목록 마커를 색상 구성에 맞게 맞춤 설정해 웹사이트 브랜드 정체성을 강화하는 방법을 제공합니다.

다음은 순서 없는 목록과 순서 있는 목록 예제입니다. 목록 항목의 마커 색상과 크기를 변경하려면 `::marker` 가상 요소를 이렇게 사용할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<ul>
  <li>Unordered list item 1</li>
  <li>Unordered list item 2</li>
  <li>Unordered list item 3</li>
  <li>Unordered list item 4</li>
</ul>

<ol>
  <li>Ordered list item 1</li>
  <li>Ordered list item 2</li>
  <li>Ordered list item 3</li>
  <li>Ordered list item 4</li>
</ol>
```

```css
li::marker {
  color: crimson;
  font-size: 1.5em;
  font-weight: bold;
}
```

:::

이번 수업에서는 몇 가지 가상 요소만 다뤘습니다. 하지만 `::placeholder`, `::spelling-error`, `::selection` 같은 더 많은 가상 요소가 있으니 직접 탐색해 보시길 권합니다.
