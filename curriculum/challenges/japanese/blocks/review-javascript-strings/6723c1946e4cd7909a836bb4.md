# --interactive--

## 文字列の基本

- **定義**: 文字列は、シングルクォート、ダブルクォート、またはバックティックで囲まれた文字のシーケンスです。文字列はプリミティブなデータ型であり、不変です。不変性とは、一度文字列が作成されると変更できないことを意味します。 
- **文字列から文字をアクセスする方法**: 文字列から文字をアクセスするには、ブラケット記法を使い、インデックス番号を渡します。インデックスは文字列内の文字の位置を表し、ゼロベースです。

:::interactive_editor

```js
const developer = "Jessica";
console.log(developer[0]); // J
```

:::

- **`\n`（改行文字）**: 文字列内で改行を作成するには、`\n` 改行文字を使います。

:::interactive_editor

```js
const poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

- **文字列のエスケープ**: 文字列内の文字をエスケープするには、クォートの前にバックスラッシュ（`\`）を置きます。

:::interactive_editor

```js
const statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

## テンプレートリテラル（テンプレート文字列）と文字列補間

- **定義**: テンプレートリテラルはバックティック（`）で定義されます。変数を文字列内に直接埋め込むことができる文字列補間という機能を含み、文字列操作が簡単になります。

:::interactive_editor

```js
const name = "Jessica";
const greeting = `Hello, ${name}!`; 
console.log(greeting); // "Hello, Jessica!"
```

:::

## ASCII、`charCodeAt()` メソッドと `fromCharCode()` メソッド

- **ASCII**: ASCII（American Standard Code for Information Interchange）は、基本的な英字を数値で表す文字コードの標準です。以前のレッスンでは、ASCIIの例を使って`charCodeAt()` と `fromCharCode()` を紹介しました。
- **Unicode**: JavaScriptの文字列は内部的にUnicode、特にUTF-16エンコーディングを使っています。最初の128文字（基本ラテン文字、数字、一般的な記号）については、Unicodeの値はASCIIコードと一致します。だからASCIIベースの例がJavaScriptでも動作します。
- **`charCodeAt()` メソッド**: このメソッドは指定したインデックスの文字のUTF-16コードユニットを返します。基本ラテン文字の場合、この値はASCIIコードと一致します。

:::interactive_editor

```js
const letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

:::

- **`fromCharCode()` メソッド**: このメソッドはASCIIコードを対応する文字に変換します。

:::interactive_editor

```js
const char = String.fromCharCode(65);
console.log(char);  // A
```

:::

## その他の一般的な文字列メソッド

- **`indexOf` メソッド**: このメソッドは文字列内で部分文字列を検索するために使います。部分文字列が見つかると、`indexOf` はその部分文字列の最初の出現位置のインデックスを返します。見つからない場合は、`indexOf` は -1 を返し、検索が失敗したことを示します。

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.indexOf("fox")); // 16
console.log(text.indexOf("cat")); // -1
```
  
:::  

- **`includes()` メソッド**: このメソッドは文字列が特定の部分文字列を含むかどうかをチェックします。部分文字列が見つかれば true を返し、そうでなければ false を返します。

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.includes("fox")); // true
console.log(text.includes("cat")); // false
```

:::

- **`slice()` メソッド**: このメソッドは文字列の一部を抽出して新しい文字列を返します。元の文字列は変更されません。開始インデックスと省略可能な終了インデックスの2つのパラメータを取ります。

:::interactive_editor

```js
const text = "freeCodeCamp";
console.log(text.slice(0, 4));  // "free"
console.log(text.slice(4, 8));  // "Code"
console.log(text.slice(8, 12)); // "Camp"
```

:::

- **`toUpperCase()` メソッド**: このメソッドはすべての文字を大文字に変換し、大文字の新しい文字列を返します。

:::interactive_editor

```js
const text = "Hello, world!";
console.log(text.toUpperCase()); // "HELLO, WORLD!"
```

:::

- **`toLowerCase()` メソッド**: このメソッドは文字列内のすべての文字を小文字に変換します。

:::interactive_editor

```js
const text = "HELLO, WORLD!"
console.log(text.toLowerCase()); // "hello, world!"
```

:::

- **`replace()` メソッド**: このメソッドは文字列内の指定した値（単語や文字など）を見つけて別の値に置き換えます。元の文字列は変更されず、置き換えた新しい文字列を返します。これはJavaScriptの文字列が不変であるためです。

:::interactive_editor

```js
const text = "I like cats";
console.log(text.replace("cats", "dogs")); // "I like dogs"
```

:::

- **`replaceAll()` メソッド**: このメソッドは文字列内の指定した値（単語、文字、またはパターン）のすべての出現箇所を見つけて別の値に置き換えます。`replace()` と似ていますが、最初の一致で止まらず、文字列内のすべての一致を更新します。

:::interactive_editor

```js
const text = "I love cats and cats are so much fun!";
console.log(text.replaceAll("cats", "dogs")); // "I love dogs and dogs are so much fun!"
```

:::

- **`repeat()` メソッド**: このメソッドは文字列を指定した回数だけ繰り返します。

:::interactive_editor

```js
const text = "Hello";
console.log(text.repeat(3)); // "HelloHelloHello"
```

:::

- **`trim()` メソッド**: このメソッドは文字列の先頭と末尾の空白を削除します。

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trim()); // "Hello, world!"
```

:::

- **`trimStart()` メソッド**: このメソッドは文字列の先頭（または「開始」）の空白を削除します。

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trimStart()); // "Hello, world!  "
```

:::

- **`trimEnd()` メソッド**: このメソッドは文字列の末尾の空白を削除します。

:::interactive_editor

```js
const text = " Hello, world! ";
console.log(text.trimEnd()); // "  Hello, world!"
```

:::

- **`prompt()` メソッド**: `window` のこのメソッドは、ダイアログボックスの形でユーザーから情報を取得するために使います。このメソッドは2つの引数を取ります。最初の引数はダイアログボックス内に表示されるメッセージで、通常はユーザーに情報の入力を促します。2つ目は省略可能なデフォルト値で、入力フィールドに初期値として設定されます。

```js
const answer = window.prompt("What's your favorite animal?"); // This will change depending on what the user answers
```
