# --description--

عند العمل مع السلاسل النصية، دمج أجزاء مختلفة من النص معًا هو عملية شائعة ستتعامل معها كثيرًا.

في Python، يمكنك دمج عدة سلاسل نصية معًا باستخدام معامل الجمع (`+`). تُسمى هذه العملية **دمج السلاسل النصية**. إليك كيفية دمج سلسلتين نصيتين باستخدام معامل الجمع:

```python
my_str_1 = 'Hello'
my_str_2 = "World"

str_plus_str = my_str_1 + ' ' + my_str_2
print(str_plus_str) # Hello World
```

لكن لاحظ أن هذا يعمل فقط مع السلاسل النصية. إذا حاولت دمج سلسلة نصية مع رقم، ستحصل على `TypeError`:

```python
name = 'John Doe'
age = 26

name_and_age = name + age
print(name_and_age) # TypeError: can only concatenate str (not "int") to str
```

يحدث هذا لأن Python لا يحول تلقائيًا أنواع البيانات الأخرى مثل الأعداد الصحيحة إلى سلاسل نصية عند الدمج. يتطلب Python أن تكون كل العناصر سلاسل نصية قبل أن يتمكن من دمجها. لإصلاح ذلك، يمكنك تحويل الرقم إلى سلسلة نصية باستخدام الدالة المدمجة `str()`، التي تُرجع التمثيل النصي للكائن المعطى دون تعديل الكائن الأصلي:

```python
name = 'John Doe'
age = 26

name_and_age = name + str(age)
print(name_and_age) # John Doe26
```

يمكنك أيضًا استخدام معامل التعيين المعزز للدمج. يُمثّل هذا بعلامة الجمع وعلامة يساوي (`+=`)، ويؤدي كلًا من الدمج والتعيين في خطوة واحدة. إليك مثالًا على ذلك:

```py
name = 'John Doe'
age = 26

name_and_age = name  # Start with the name
name_and_age += str(age)  # Append the age as string

print(name_and_age)  # John Doe26
```

عملية إدخال المتغيرات والتعابير داخل سلسلة نصية تُسمى **استيفاء السلاسل النصية**. لدى Python نوع من السلاسل النصية يُسمى **f-strings** (اختصار لـ formatted string literals)، والذي يسمح لك بالتعامل مع الاستيفاء بصيغة نحوية مضغوطة وسهلة القراءة.

تبدأ f-strings بـ `f` (سواء صغيرة أو كبيرة) قبل علامات الاقتباس، وتسمح لك بتضمين المتغيرات أو التعابير داخل حقول الاستبدال المحددة بأقواس متعرجة (`{}`). إليك مثالًا:

```python
name = 'John Doe'
age = 26
name_and_age = f'My name is {name} and I am {age} years old'
print(name_and_age) # My name is John Doe and I am 26 years old

num1 = 5
num2 = 10
print(f'The sum of {num1} and {num2} is {num1 + num2}') # The sum of 5 and 10 is 15
```

لاحظ أنك لا تحتاج إلى تحويل الأنواع غير النصية باستخدام الدالة `str()`. في المثال أعلاه، يتم تحويل قيمة المتغيرات `age` و `num1` و `num2` ضمنيًا إلى سلسلة نصية أثناء عملية الاستيفاء.
