# --interactive--

عند تصميم مواقع الويب، تُعد واحدة من أكثر الطرق تنوعًا وبديهية للعمل مع الألوان هي استخدام نموذج ألوان HSL. HSL تعني اللون (Hue)، التشبع (Saturation)، والسطوع (Lightness) — وهي ثلاثة مكونات رئيسية تحدد اللون.

يمثل نموذج ألوان HSL الألوان بطريقة تتماشى أكثر مع كيفية إدراك البشر للون. في هذا الدرس، سنستكشف ما هو نموذج ألوان HSL وكيف تُستخدم الدالة `hsl()` في CSS.

اللون (Hue) هو نوع اللون، يُمثّل كزاوية على عجلة الألوان، تتراوح من `0` إلى `360` درجة. على سبيل المثال، `0` درجة تمثل الأحمر، و`120` درجة تمثل الأخضر، و`240` درجة تمثل الأزرق. بتعديل اللون، تنتقل عبر ألوان مختلفة على الطيف.

التشبع يشير إلى شدة أو نقاء اللون. يُقاس كنسبة مئوية من `0%` (لون رمادي تمامًا بدون تشبع) إلى `100%` (أكثر أشكال اللون حيوية). اللون الذي له تشبع `100%` يكون نابضًا بالكامل، بينما اللون الذي له تشبع `0%` يظهر كلون رمادي.

السطوع يحدد مدى فاتح أو داكن اللون، ويُقاس أيضًا كنسبة مئوية. قيمة السطوع `0%` تعطي اللون الأسود، و`50%` تعطي الدرجة العادية للون، و`100%` تعطي اللون الأبيض.

في CSS، تُستخدم الدالة `hsl()` لتعريف الألوان باستخدام نموذج ألوان HSL. الصيغة الحديثة تستخدم قيمًا مفصولة بمسافات:

```css
element {
  color: hsl(hue saturation lightness);
}
```

قد تواجه أيضًا الصيغة القديمة التي تفصل القيم بفواصل في الشفرة الموجودة، والتي لا تزال مدعومة من المتصفحات:

```css
element {
  color: hsl(hue, saturation, lightness);
}
```

دعنا نوضح ذلك بمثال:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
body {
  background-color: hsl(0 0% 1% / 1);
}

p {
  color: hsl(120 100% 50%);
}
```

:::

في هذه الحالة، اللون هو `120` درجة، وهو ما يعادل الأخضر. التشبع هو `100%`، لذا سيكون الأخضر نابضًا بالكامل. السطوع هو `50%`، لذا هو بدرجته العادية — لا داكن جدًا ولا فاتح جدًا. ونتيجة لذلك، سيكون لون نص الفقرة أخضرًا ساطعًا.

واحدة من المزايا الرئيسية لنموذج ألوان HSL هي طبيعته البديهية. يجعل من السهل تعديل حيوية اللون أو سطوعه بتغيير قيم التشبع والسطوع دون الحاجة إلى تغيير اللون الأساسي (Hue).

على سبيل المثال، إذا أردت إنشاء درجات أو ظلال مختلفة من نفس اللون، يمكنك ببساطة تعديل قيمة السطوع.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="light">This is a light blue div.</div>
<div class="dark">This is a dark blue div.</div>
```

```css
div.light {
  background-color: hsl(240 100% 80%);
}

div.dark {
  background-color: hsl(240 100% 20%);
  color: hsl(0 0% 100%);
}
```

:::

هنا، كلا العنصرين `div` يستخدمان نفس اللون (`240` درجة، وهو الأزرق)، لكن أحدهما له سطوع `80%` (ظل أفتح من الأزرق)، والآخر له سطوع `20%` (ظل أغمق من الأزرق).

تمامًا مثل نموذج RGB، تدعم الدالة `hsl()` أيضًا قيمة ألفا اختيارية للتحكم في الشفافية. تضيفها بعد فاصل `/` كمعلمة رابعة. هذه هي الصيغة الأساسية:

```css
element {
  background-color: hsl(hue saturation lightness / alpha);
}
```

لنلق نظرة على مثال:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: hsl(0 100% 50% / 0.5);
}
```

:::

هذه الشفرة ستعطي العنصر `div` خلفية حمراء نصف شفافة، حيث اللون مضبوط على `0` درجة (أحمر)، التشبع `100%`، السطوع `50%`، والألفا `0.5` (شفافية 50%). يمكنك أيضًا تحديد الألفا كنسبة مئوية — على سبيل المثال، `hsl(0 100% 50% / 50%)` تعادل `hsl(0 100% 50% / 0.5)`.

سترى أيضًا غالبًا الدالة `hsla()` في CSS الموجود. كانت هذه هي الطريقة الأصلية لإضافة الشفافية في ألوان HSL، باستخدام قيم مفصولة بفواصل:

```css
element {
  background-color: hsla(hue, saturation, lightness, alpha);
}
```

على سبيل المثال، `hsla(0, 100%, 50%, 0.5)` هو المكافئ الموروث لـ `hsl(0 100% 50% / 0.5)`. كلاهما مدعوم من المتصفحات، لكن صيغة `hsl()` الحديثة مع فاصل `/` هي المفضلة الآن.

نموذج ألوان HSL مفيد بشكل خاص عندما تحتاج إلى إنشاء مخططات ألوان وتعديل الظلال أو الدرجات بسهولة.

بما أن HSL يفصل اللون عن التشبع والسطوع، يمكنك الحفاظ على ثيم لوني متناسق مع تعديل السطوع للظلال أو الإضاءات. هذا المستوى من المرونة يجعل HSL مفضلًا لدى المصممين والمطورين على حد سواء.

بالإضافة إلى ذلك، يجعل HSL قيم الألوان أكثر قابلية للقراءة والفهم مقارنة بطرق أخرى مثل رموز hex أو RGB. يمكنك رؤية الهدف من اللون بوضوح وتعديله بسهولة، خاصة عند تجربة تصاميم مختلفة.

مع نموذج ألوان HSL، تحصل على مرونة أكبر في التحكم في درجات اللون، التشبع، والسطوع، مما يجعله أداة قوية لإنشاء تصاميم غنية وديناميكية في CSS.
