# --interactive--

사용자가 버튼을 클릭하거나 폼에 변경이 생기면 이를 이벤트라고 합니다. 프로그램에서는 이러한 이벤트를 듣고 반응할 방법이 필요합니다.

`addEventListener()` 메서드는 이벤트를 듣는 데 사용됩니다. 이 메서드는 두 개의 인수를 받는데, 하나는 듣고자 하는 이벤트이고 다른 하나는 이벤트가 발생했을 때 호출될 함수입니다.

기본 구문은 다음과 같습니다:

```js
element.addEventListener("event", listener);
```

element는 이벤트를 감시할 HTML 요소이고, event는 `"click"` 같은 듣고자 하는 이벤트 유형을 지정합니다.

listener는 이벤트가 발생했을 때 알림을 받을 객체입니다. 대부분은 이벤트를 처리하기 위해 정의한 함수가 됩니다.

예를 들어 보겠습니다:

```js
element.addEventListener("click", () => {
  // code to run when the click event occurs
});
```

함수 참조를 이렇게 전달할 수도 있습니다:

```js
function handleClick() {
  // code to run when the click event occurs
}

element.addEventListener("click", handleClick);
```

때로는 이벤트를 처리할 별도의 함수를 만드는 것이 코드를 더 읽기 쉽고 유지보수하기 좋게 만듭니다.

listener 인수는 `null`일 수도 있고, `EventListener` 인터페이스를 사용하는 객체일 수도 있습니다. `EventListener` 인터페이스는 `handleEvent()`라는 단일 메서드를 정의합니다. 이 메서드는 듣고 있는 이벤트가 발생할 때마다 자동으로 호출됩니다. 이 인터페이스를 사용하면 필요에 따라 동일한 객체가 여러 이벤트를 처리할 수 있습니다.

대부분의 경우 이 인터페이스를 사용할 필요 없이, 이벤트가 발생했을 때 호출될 함수를 전달하면 됩니다.

이제 이 작동 방식을 더 잘 이해할 수 있도록 예제를 살펴보겠습니다:

이 예제에서는 id가 `button`인 `btn` 요소가 있습니다:

```html
<button id="btn">Show alert</button>
```

이 버튼에서 `"click"` 이벤트를 듣고 버튼이 클릭되면 알림을 표시하려고 합니다.

먼저 JavaScript 코드에서 `button` 요소에 접근해야 합니다. 그런 다음 클릭 이벤트를 듣도록 이벤트 리스너를 추가하고 버튼이 클릭될 때 알림을 표시합니다:

:::interactive_editor

```html
<button id="btn">Show alert</button>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("btn");

btn.addEventListener("click", () => alert("You clicked the button"));
```

:::

이제 버튼이 클릭될 때마다 사용자 화면에 다음과 같은 알림 메시지가 표시됩니다:

```md
You clicked the button
```

또 다른 이벤트 유형으로 `"input"` 이벤트가 있습니다. 이 이벤트는 입력 요소의 값이 변경될 때 발생합니다.

예를 들어 보겠습니다:

```html
<input type="text" id="input" placeholder="Type something" />
```

이 입력 요소에서 input 이벤트를 듣고 사용자가 입력할 때마다 입력 값이 콘솔에 기록되도록 하려 합니다.

먼저 JavaScript 코드에서 `input` 요소에 접근해야 합니다. 그런 다음 input 이벤트를 듣도록 이벤트 리스너를 추가하고 사용자가 입력할 때마다 입력 값을 콘솔에 기록합니다:

:::interactive_editor

```html
<input type="text" id="input" placeholder="Type something" />
<script src="index.js"></script>
```

```js
const input = document.getElementById("input");

input.addEventListener("input", () => {
  console.log(input.value);
});
```

:::

`addEventListener()` 메서드를 사용해 들을 수 있는 이벤트는 훨씬 더 많습니다. 흔한 이벤트로는 `mouseover`, `mouseout`, `keydown`, `keyup`, `submit` 등이 있습니다.

앞으로의 수업에서 더 많은 이벤트와 `addEventListener()` 메서드를 사용해 이벤트를 듣는 방법을 다룰 예정입니다.
