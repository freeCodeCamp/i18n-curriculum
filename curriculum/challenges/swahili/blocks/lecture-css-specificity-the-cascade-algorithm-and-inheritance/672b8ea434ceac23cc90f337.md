# --interactive--

Vichaguzi vya ID ni miongoni mwa vichaguzi vyenye nguvu zaidi katika CSS, vinavyomruhusu msanidi programu kutumia mitindo kwa vipengele maalum vyenye vitambulisho vya kipekee.

Hii inawafanya kuwa na ufanisi mkubwa kwa kulenga vipengele binafsi vinavyohitaji mitindo ya kipekee.

Vichaguzi vya ID huainishwa kwa alama ya hash (`#`) ikifuatiwa na jina la ID. Zinapaswa kuwa za kipekee ndani ya hati ya HTML, ikimaanisha hakuna vipengele viwili vinavyopaswa kushiriki ID moja.

Hapa kuna mfano wa kutumia kichaguzi cha ID:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="unique">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#unique {
  color: purple;
}
```

:::

Katika mfano huu, kipengele chenye ID `unique` kitakuwa na rangi ya maandishi `purple`.

Vichaguzi vya ID vina sifa ya juu sana, zaidi kuliko vichaguzi vya aina na vichaguzi vya darasa la vitu, lakini chini kuliko mitindo ndani ya mstari. Thamani ya sifa kwa kichaguzi cha ID ni `(0, 1, 0, 0)`.

Hii inamaanisha kuwa vichaguzi vya ID vinaweza ghairi vichaguzi vya darasa la vitu na vichaguzi vya aina lakini vinaweza kughairiwa na mitindo ndani ya mstari.
