# --description--

Селектор типу (або селектор елемента) вибирає всі елементи заданого типу, а синтаксис — це просто назва елемента:

```html
<!-- index.html -->

<div>Hello, World!</div>
<div>Hello again!</div>
<p>Hi...</p>
<div>Okay, bye.</div>
```

```css
/* styles.css */

div {
  color: white;
}
```

Тут будуть вибрані всі три елементи `<div>`, а елемент `<p>` — ні.

## --text--

Який із наведених варіантів найкраще описує наведений вище CSS код?

## --answers--

Код застосовує колір `white` до всіх елементів у HTML файлі.

Код застосовує колір `white` до всіх елементів `div` у HTML файлі.

---

Код застосовує колір `white` до всіх елементів `p` у HTML файлі.
