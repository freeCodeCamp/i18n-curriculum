## Python 클래스와 객체

- **클래스 정의**: 클래스는 객체를 만들기 위한 청사진입니다. 속성과 메서드를 통해 객체가 가질 동작을 정의합니다. 다음은 Python에서 클래스 정의의 기본 예시입니다.

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f'{self.name.upper()} says woof woof!')
```

- **객체 생성**: 객체는 클래스의 인스턴스입니다. 필요한 인수를 사용해 클래스를 호출하여 생성합니다.

```python
dog1 = Dog('Jack', 3)
dog2 = Dog('Thatcher', 5)

dog1.bark()  # JACK says woof woof!
dog2.bark()  # THATCHER says woof woof!
```

- **객체에서 메서드 호출**: 객체에서 메서드를 호출해 동작을 수행하거나 정보를 가져올 수 있습니다.

```python
object_name1.method_name()
object_name2.method_name()
```

- **클래스와 객체의 차이**: 클래스는 재사용 가능한 템플릿이고, 객체는 실제 데이터를 가진 그 클래스의 특정 인스턴스입니다.

## 속성

- **인스턴스 속성**: `__init__()`에서 `self`를 사용해 정의하며, 각 객체마다 고유합니다.
- **클래스 속성**: 클래스 내부에 직접 정의하며 모든 인스턴스가 공유합니다.

```python
class Dog:
    species = 'French Bulldog'  # Class attribute

    def __init__(self, name):
        self.name = name  # Instance attribute

print(Dog.species) # French Bulldog

jack = Dog('Jack')
print(jack.name)     # Jack
print(jack.species)  # French Bulldog
```

## 메서드

- **메서드**: 클래스 내부에 정의된 함수로, 객체의 속성에 작용합니다.

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
print(my_car_1.describe())  # This car is a red Tesla Model S
```

- **메서드 접근**: 점 표기법을 사용해 객체에서 메서드를 호출합니다. 다음은 두 개의 서로 다른 자동차 객체에서 `describe` 메서드를 호출하는 예시입니다.

```python
class Car:
    def __init__(self, color, model):
        self.color = color  
        self.model = model  

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
my_car_2 = Car('green', 'Lamborghini Revuelto')

print(my_car_1.describe()) # Calling method using the dot notation

print(my_car_2.describe()) # Calling method using the dot notation
```

## 던더(매직) 메서드

- **정의**: 앞뒤에 이중 밑줄이 붙은 특별한 메서드(e.g., `__init__`, `__len__`, `__str__`, `__eq__`)입니다. Python이 내장 연산을 위해 내부적으로 사용합니다.

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **던더 메서드를 간접적으로 호출하기**: 던더 메서드를 직접 호출할 필요가 없습니다. 특정 동작이 발생할 때 Python이 자동으로 호출합니다. 이 동작에는 다음이 포함됩니다:

  - **덧셈, 뺄셈, 곱셈, 나눗셈 등 산술 연산**. 덧셈에는 `__add__()`, 뺄셈에는 `__sub__()`, 곱셈에는 `__mul__()`, 나눗셈에는 `__truediv__()`이 호출됩니다.
        
  - **문자열 연산인 연결, 반복, 포맷팅, 텍스트 변환**. 연결에는 `__add__()`, 반복에는 `__mul__()`, 포맷팅에는 `__format__()`, 텍스트 변환에는 `__str__()`와 `__repr__()` 등이 호출됩니다.
        
  - **동등, 미만, 초과 등 비교 연산**. 동등 비교에는 `__eq__()`, 미만 비교에는 `__lt__()`, 초과 비교에는 `__gt__()` 등이 호출됩니다.
        
  - **객체를 반복 가능한 상태로 만들고 항목을 순회하는 반복 연산**. 반복자를 반환할 때 `__iter__()`, 다음 항목을 가져올 때 `__next__()`이 호출됩니다.

## 실제 예시: 쇼핑 카트

- **던더 메서드를 가진 Cart 클래스**: 내장 동작으로 항목 추가, 제거, 반복, 내용 확인을 할 수 있습니다.

```python
class Cart:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

    def remove(self, item):
        if item in self.items:
            self.items.remove(item)
        else:
            print(f'{item} is not in cart')

    def list_items(self):
        return self.items

    def __len__(self):
        return len(self.items)

    def __getitem__(self, index):
        return self.items[index]

    def __contains__(self, item):
        return item in self.items

    def __iter__(self):
        return iter(self.items)

cart = Cart()
cart.add('Laptop')
print(len(cart))        # 1
print('Laptop' in cart) # True
```
