# --interactive--

عند العمل مع السلاسل النصية في JavaScript، توجد العديد من الحالات التي قد تحتاج فيها إلى تعديل حالة الأحرف، مثل تحويل جميع الحروف إلى أحرف كبيرة لعنوان أو تحويل النص إلى أحرف صغيرة لتحقيق التناسق.

لحسن الحظ، يجعل JavaScript هذا الأمر سهلاً باستخدام طريقتين مدمجتين: `toUpperCase()` و `toLowerCase()`.

تُحوّل الطريقة `toUpperCase()` جميع الأحرف إلى أحرف كبيرة وتُعيد سلسلة نصية جديدة تحتوي على جميع الأحرف الكبيرة. هذا مفيد عندما تريد التأكيد على النص أو إنشاء تناسق في تنسيق السلاسل النصية.

لنرَ مثالاً:

:::interactive_editor

```js
let greeting = "Hello, World!";
let uppercaseGreeting = greeting.toUpperCase();
console.log(uppercaseGreeting);  // "HELLO, WORLD!"
```

:::

في هذا الشفرة، تُحوّل الطريقة `toUpperCase()` السلسلة النصية بأكملها إلى أحرف كبيرة.

تظل السلسلة النصية الأصلية دون تغيير لأن `toUpperCase()` تُعيد سلسلة جديدة بدلاً من تعديل الأصلية.

على الجانب الآخر، تُحوّل الطريقة `toLowerCase()` جميع الأحرف في السلسلة النصية إلى أحرف صغيرة. هذا مفيد عندما تحتاج إلى توحيد المُدخلات، مثل مقارنة النصوص التي يقدمها المستخدم أو إجراء فحوصات غير حساسة لحالة الأحرف.

لننظر إلى مثال:

:::interactive_editor

```js
let shout = "I AM LEARNING JAVASCRIPT!";
let lowercaseShout = shout.toLowerCase();
console.log(lowercaseShout);  // "i am learning javascript!"
```

:::

تُحوّل الطريقة `toLowerCase()` جميع الأحرف إلى أحرف صغيرة، مما يجعل السلسلة أقل حدة، مع بقاء السلسلة الأصلية دون تغيير.

باختصار، تُعد الطريقتان `toUpperCase()` و `toLowerCase()` في JavaScript أدوات قوية لتحويل السلاسل النصية إلى أحرف كبيرة أو صغيرة بالكامل.

تُستخدم هذه الطرق بشكل خاص لتوحيد مُدخلات النص، وإجراء مقارنات غير حساسة لحالة الأحرف، وضمان تناسق التصميم.

بهذه الطرق البسيطة والفعالة، يمكنك التعامل مع معالجة النصوص بطريقة أكثر تحكماً وتوقعاً.
