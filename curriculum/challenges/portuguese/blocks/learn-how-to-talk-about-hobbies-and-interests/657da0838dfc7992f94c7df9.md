# --description--

Recomendar significa dizer que algo é uma boa escolha para alguém. É quando você dá um conselho sobre o que outra pessoa deve fazer, usar ou experimentar. Por exemplo:

`They recommended testing the app on different devices.` - Isso sugere a melhor ação para qualidade.

Quando você tiver ações seguindo o verbo `recommend`, use-as na forma `-ing`. Por exemplo, `I recommend reading this book if you want to understand this topic better`.
