# --explanation--

`When` é usado para perguntar sobre tempo.A palavra `available` significa livre para fazer algo ou não ocupado.Por exemplo:

- `When does the meeting start?` - Esta pergunta está perguntando sobre o horário em que a reunião começa.

- `Are you available to help me tomorrow?` - Esta pergunta está perguntando se a pessoa está disponível para ajudar amanhã.
