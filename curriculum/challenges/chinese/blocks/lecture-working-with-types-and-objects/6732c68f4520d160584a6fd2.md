# --interactive--

`toString()` 方法是 JavaScript 中的一个基本特色，用于将一个值转换为其字串表现。你可以对数字、布尔、数组和对象使用这个方法。`toString()` 最常见的用途之一是将数字转换为其字串表现。下面是一个示例：

:::interactive_editor

```js
const num = 10;
console.log(num.toString()); // "10"
```

:::

此方法接受一个可选的基数，该基数是从 `2` 到 `36` 的数字。该基数表现为进制，例如二进制的基数为 2，八进制的基数为 8。如果未指定基数，则默认为基数 10，即十进制。以下是将 `2` 作为参数传递给 `toString()` 方法的示例：

:::interactive_editor

```js
const num = 10;
console.log(num.toString(2)); // "1010"
```

:::

结果将是 `1010`，它是十进制数 `10` 的二进制表现。

你也可以使用 `toString()` 方法将数组和对象转换为字串。数组有一个自定义的 `toString()` 实现，它将每个元素转换为字串并用逗号连接。下面是一个示例：

:::interactive_editor

```js
const arr = [1, 2, 3];
console.log(arr.toString()); // "1,2,3"
```

:::

在此示例中，数组的所有元素被连接在一起形成字串 `1,2,3`。 

当 `toString()` 方法用于对象时，结果不会是对象属性的字串化版本。 

:::interactive_editor

```js
const person = {
  name: "John",
  age: 30,
  isStudent: true
};

console.log(person.toString()); // "[object Object]"
```

:::

在这个例子中，结果将是对象的默认字串表现，即 `[object Object]`。要获取 `person` 对象属性的字串化版本，你需要使用 `JSON.stringify()`，你将在后续课程中详细学习。

总之，`toString()` 方法用于将值转换为字串。理解它如何与不同的数据类型协作以及如何为你自己的对象自定义它，可以极大地提升你在 JavaScript 应用中操作和显示数据的能力。
