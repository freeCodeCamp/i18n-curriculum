# --description--

`Fingerprint scanning` ist eine Technik, die die einzigartigen Muster auf Ihren Fingern betrachtet, um zu identifizieren und zu überprüfen, wer Sie sind. Es ist, als hätten Ihre Finger besondere Merkmale, die sich von denen aller anderen unterscheiden. Wenn Sie `fingerprint scanning` verwenden, überprüft das System diese Merkmale, um sicherzustellen, dass Sie es wirklich sind. Genau wie `facial recognition` ist es eine wichtige Technik, die von `biometrics` verwendet wird.
