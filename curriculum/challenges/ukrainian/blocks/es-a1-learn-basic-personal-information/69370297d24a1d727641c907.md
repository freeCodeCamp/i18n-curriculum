# --description--

У цьому діалозі Каміла повторює частину адреси е-пошти, щоб підтвердити її.

Вона використовує висхідну інтонацію, щоб показати, що це питання для підтвердження.

Коли хтось повторює інформацію з питальною інтонацією, це зазвичай робиться, щоб перевірити, чи вона правильна.

У такій ситуації інша людина може або підтвердити, або спростувати інформацію. Наприклад:

- `Sí, correcto.` (Те, що сказала людина, є правильним).

- `No, es ...` + виправлення (Те, що сказала людина, є неправильним).
