# --explanation--

`Parameterized` は、`parameter` という、動作を制御するために使われる値やプレースホルダーを語源としています。

クエリが `parameterized` であると言うとき、それはユーザーの入力を直接挿入するのではなく、あらかじめ定義された値（パラメータ）を使っていることを意味します。

これにより、SQLインジェクションのような攻撃を防ぎ、データベースのセキュリティが向上します。例えば：

- `The developer used parameterized queries to secure the database.` - これは、悪意のあるコードが挿入されないようにプレースホルダーを使ったことを意味します。
