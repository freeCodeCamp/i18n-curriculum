# --questions-- 

## --text--

Was versteht man unter dem „Normalfluss“ in CSS-Layouts?

## --answers--

Normalfluss bezeichnet den Prozess, Elemente in einem CSS-Layout zu animieren.

---

Normalfluss ist eine CSS-Eigenschaft, mit der die Sichtbarkeit von Elementen in einem Layout gesteuert wird.

---

Normalfluss ist eine CSS-Regel, die die Reihenfolge von Media Queries für responsives Design definiert.

---

Normalfluss in CSS-Layouts bezeichnet die standardmäßige Positionierung von Elementen auf einer Webseite.
