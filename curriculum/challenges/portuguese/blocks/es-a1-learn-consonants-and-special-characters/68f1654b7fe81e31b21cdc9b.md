# --description--

A letra `r` é chamada de `erre` em espanhol.

Esta consoante tem duas pronúncias diferentes.

Um único `r` tem um som suave. Por exemplo, `cero` ("zero").

Um `r` duplo (`rr`) ou um `r` no início de uma palavra produz um som mais forte. Por exemplo, `perro` ("dog") ou `ruido` ("noise").
