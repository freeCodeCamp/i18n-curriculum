# --explanation--

`How many` é usado para perguntar sobre a quantidade de algo. `Usually` indica algo que acontece regularmente. 

Perguntas no tempo `Present Simple` são usadas para perguntar sobre hábitos, fatos ou verdades gerais.Por exemplo:

- `How many books do you read in a month?` - Pergunta sobre o número regular de livros que alguém lê.

- `How many people usually attend the meeting?` - Pergunta sobre o número típico de pessoas em uma reunião regular.
