# --description--

Cuando encuentres un camino más corto hacia un vecino, también necesitas actualizar el camino registrado para llegar a él.

En JavaScript, puedes crear un nuevo arreglo combinando un arreglo existente y un nuevo elemento usando el operador spread:

```js
const newPath = [...existingArray, newElement];
```

Dentro del bloque anidado `if`, actualiza `paths[nodeNo]` para que sea el camino hacia el nodo actual con `nodeNo` añadido al final usando la sintaxis spread así: `[...paths[current], nodeNo]`.
