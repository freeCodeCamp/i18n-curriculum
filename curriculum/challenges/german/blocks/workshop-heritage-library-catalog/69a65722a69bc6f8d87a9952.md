# --description--

JSON (JavaScript Object Notation) ist ein Standardformat zum Austausch von Daten zwischen Systemen. `JSON.stringify` wandelt einen JavaScript-Wert in einen JSON-formatierten String um.

Beginnen Sie damit, die beiden `validateEntry` console.log-Aufrufe zu entfernen, da diese nur zum Testen benötigt wurden. Erstellen Sie dann eine Funktion namens `exportToJSON`, die `catalog` als Parameter entgegennimmt. Geben Sie `JSON.stringify(catalog, null, 2)` zurück. Das erste Argument ist die zu konvertierenden Daten. Das zweite Argument (`null`) bedeutet keine benutzerdefinierte Filterung. Das dritte (`2`) fügt eine Einrückung von zwei Leerzeichen hinzu, wodurch die Ausgabe lesbar wird.

Protokollieren Sie unterhalb der Funktion `exportToJSON(catalog.slice(0, 2))`, um die JSON-Ausgabe für die ersten beiden Einträge zu überprüfen.
