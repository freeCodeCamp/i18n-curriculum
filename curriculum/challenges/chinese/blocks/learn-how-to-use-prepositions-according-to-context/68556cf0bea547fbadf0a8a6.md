# --explanation--

`Around` 意味着大约或接近某个时间或地点。例如：

`We plan to deploy the update around Friday.` – 这意味着在那天附近，而不是确切时间。

`Mid-quarter` 指三个月业务周期的中点。例如：

`The review will happen in mid-quarter.` – 这意味着它将在当前季度的中间进行。
