## --text--

哪个句子语法正确？

## --answers--

`Everyone must to follow the security guidelines.`

### --feedback--

`Must` 后面跟动词原形，不带 `to`。

---

`Everyone must following the security guidelines.`

### --feedback--

`Must` 后应跟动词原形，而非 `-ing` 形式。

---

`Everyone must followed the security guidelines.`

### --feedback--

`Must` 后面应跟动词原形，而不是过去式形式。

## --video-solution--

4

# --explanation--

情态动词后面总是跟动词的原形。情态动词不带 `to`、`-ing` 或过去式形式。以下是错误动词形式的示例： 

- `Must to follow` 是错误的，因为 `must` 接 `to`。

- `Must following` 是错误的，因为 `must` 后面跟的是一个 `-ing` 形式。

- `Must followed` 是错误的，因为 `followed` 是过去式。
