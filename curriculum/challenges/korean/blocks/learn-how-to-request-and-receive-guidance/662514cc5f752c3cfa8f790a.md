# --description--

`Time complexity`는 컴퓨터 과학 용어입니다. 입력 크기에 따라 알고리즘(명령어 집합)이 완료되는 데 걸리는 시간을 설명합니다. 간단한 예로 설거지를 들 수 있습니다: 접시 한 개를 씻는 것은 빠르지만, 많은 접시를 씻는 데는 더 오래 걸립니다. 이것이 프로그래밍에서 시간 복잡도와 비슷합니다.
