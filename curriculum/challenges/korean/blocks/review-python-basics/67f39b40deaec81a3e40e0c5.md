## 파이썬이란?

- **소개**: 파이썬은 단순하고 사용하기 쉬운 범용 프로그래밍 언어입니다. 파이썬은 데이터 과학과 머신러닝, 웹 개발, 스크립팅과 자동화, 임베디드 시스템과 IoT 등 다양한 분야에서 사용됩니다.
- **일반적인 사용 사례**: 파이썬은 데이터 과학, 머신러닝, 웹 개발, 사이버보안, 자동화, 라즈베리 파이 및 MicroPython 호환 보드 같은 마이크로컴퓨터에서 사용됩니다.

## 로컬 환경에서의 파이썬

- **설치**: Windows, Mac, Linux에서 파이썬을 설치하는 가장 좋은 방법은 공식 파이썬 웹사이트에서 설치 프로그램을 다운로드하는 것입니다 (`https://www.python.org/`).

## 변수

- **변수 선언**: 변수를 선언하려면 변수 이름 다음에 대입 연산자(`=`)를 쓰고 값을 할당하세요. 값은 숫자, 문자열, 불리언 등일 수 있습니다. 예시는 다음과 같습니다:

```py
name = 'John Doe'
age = 25
```

- **변수 이름 규칙**: 변수 이름에 적용할 규칙은 다음과 같습니다:

  - 변수 이름은 숫자가 아닌 문자나 밑줄(_)로 시작해야 합니다.
  - 변수 이름은 영숫자(a-z, A-Z, 0-9)와 밑줄(_)만 포함할 수 있습니다.
  - 변수 이름은 대소문자를 구분합니다 — `age`, `Age`, `AGE`는 모두 별개의 이름입니다.
  - 변수 이름은 파이썬 예약어인 `if`, `class`, `def` 같은 단어를 사용할 수 없습니다.
  - 여러 단어로 된 변수 이름은 밑줄로 구분합니다. 예: `snake_case`.

## 주석

- **한 줄 주석**: 코드에 짧은 메모를 남길 때 사용하세요.

```py
# This is a single line comment
```

- **여러 줄 문자열**: 더 긴 메모를 남기거나 코드 일부를 주석 처리할 때 사용할 수 있습니다.

```py
"""
This is a multi-line string.
Here is some code commented out.

name = 'John Doe'
age = 25
"""
```

- **`print()` 함수**: 콘솔에 데이터를 출력하려면 `print()` 함수를 이렇게 사용하세요:

```py
print('Hello world!') # Hello world!
```

## 파이썬의 주요 데이터 타입

- **소개**: 파이썬은 JavaScript처럼 동적 타이핑 언어로, 변수 타입을 명시적으로 선언할 필요가 없습니다. 변수에 할당된 값에 따라 타입이 결정됩니다.
- **정수**: 소수점 없는 정수입니다:

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **실수**: 소수점이 있는 숫자입니다:

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **문자열**: 따옴표로 감싼 문자들의 나열입니다:

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **불리언**: `True` 또는 `False` 값을 나타냅니다:

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **집합**: 중복 없는 요소들의 순서 없는 모음입니다:

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **딕셔너리**: 중괄호로 감싼 키-값 쌍들의 모음입니다:

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **튜플**: 소괄호로 감싼 변경 불가능한 순서 있는 모음입니다:

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **범위**: 반복문에서 자주 쓰이는 숫자 수열입니다:

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **리스트**: 다양한 데이터 타입을 담을 수 있는 순서 있는 모음입니다:

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**: 값이 없음을 나타내는 특별한 값입니다:

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## 불변 타입과 가변 타입

- **불변 타입**: 선언 후 변경할 수 없지만, 변수에 다른 값을 다시 할당하는 것은 가능합니다. 정수, 실수, 복소수, 불리언, 문자열, 튜플, 범위, 그리고 `None`이 포함됩니다.
- **가변 타입**: 선언 후 변경할 수 있으며, 항목을 추가, 삭제, 수정할 수 있습니다. 리스트, 집합, 딕셔너리 같은 컬렉션 타입이 포함됩니다.
- **`type()` 함수**: 변수의 타입을 확인하려면 `type()` 함수를 이렇게 사용하세요:

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **`isinstance()` 함수**: 변수의 타입이 특정 타입인지 확인할 때 사용합니다:

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## 문자열 다루기

- **정의**: JavaScript에서 기억하듯, 문자열은 불변이므로 생성 후 변경할 수 없습니다. 파이썬에서는 작은따옴표나 큰따옴표를 사용할 수 있으며, 한 가지 규칙을 정해 일관되게 사용하는 것이 좋습니다:

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **문자열에서 문자 접근하기**: 대괄호 표기법을 사용해 문자열의 문자를 접근할 수 있습니다:

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **문자열 이스케이프**: 문자열에 따옴표가 포함될 경우 역슬래시(`\`)를 사용하세요:

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **문자열 연결**: `+` 연산자를 사용해 문자열을 연결할 수 있습니다:

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

또 다른 문자열 연결 방법은 `+=` 연산자를 사용하는 것입니다. 연결과 할당을 동시에 수행합니다:

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**: 포맷된 문자열 리터럴의 줄임말로, 보간과 연결을 간결하고 읽기 쉽게 처리할 수 있습니다:

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **문자열 슬라이싱**: 문자열의 일부를 추출하는 방법입니다. 기본 구문은 다음과 같습니다:

```py
str[start:stop:step]
```

시작 위치는 추출을 시작할 인덱스이고, 정지 위치는 추출을 끝낼 인덱스이며 포함하지 않습니다. 단계는 슬라이싱 간격을 나타냅니다. 예시는 다음과 같습니다:

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **문자열 길이 구하기**: `len()` 함수는 문자열의 문자 수를 반환합니다:

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## `in` 연산자 다루기

- **`in` 연산자**: 문자열에 특정 문자나 문자가 존재하는지 여부를 불리언으로 반환합니다:

```py
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

## 자주 쓰이는 문자열 메서드

- **`str.upper()`**: 모든 문자를 대문자로 변환한 새 문자열을 반환합니다:

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`str.lower()`**: 모든 문자를 소문자로 변환한 새 문자열을 반환합니다:

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`str.strip()`**: 지정한 앞뒤 문자를 제거한 문자열 복사본을 반환합니다(인수가 없으면 앞뒤 공백을 제거).

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**: 문자열 내 모든 옛 문자열을 새 문자열로 교체한 새 문자열을 반환합니다.

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**: 지정한 구분자를 사용해 문자열을 리스트로 분할합니다. 구분자는 분할 위치를 지정하는 문자열입니다.

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**: 반복 가능한 요소들을 구분자로 연결해 문자열로 만듭니다. 반복 가능한 것은 리스트, 문자열, 튜플처럼 반복할 수 있는 컬렉션입니다.

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`str.startswith(prefix)`**: 문자열이 지정한 접두사로 시작하는지 여부를 불리언으로 반환합니다:

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`str.endswith(suffix)`**: 문자열이 지정한 접미사로 끝나는지 여부를 불리언으로 반환합니다:

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`str.find()`**: 부분 문자열이 처음 나타나는 인덱스를 반환합니다. 없으면 `-1`을 반환합니다:

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`str.count(substring)`**: 부분 문자열이 문자열 내에 몇 번 나타나는지 셉니다:

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`str.capitalize()`**: 첫 문자는 대문자로, 나머지는 소문자로 변환한 새 문자열을 반환합니다:

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`str.isupper()`**: 문자열의 모든 문자가 대문자면 `True`을, 아니면 `False`을 반환합니다:

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`str.islower()`**: 문자열의 모든 문자가 소문자면 `True`을, 아니면 `False`을 반환합니다:

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`str.title()`**: 각 단어의 첫 글자를 대문자로 변환한 새 문자열을 반환합니다:

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`str.maketrans()`**: 1:1 문자 매핑 테이블을 생성하는 메서드입니다. 보통 `translate()` 메서드와 함께 사용해 문자열에 적용하고 번역 결과를 반환합니다.

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## 정수와 실수에 자주 쓰이는 연산

- **기본 산술 연산**: 파이썬에서 정수와 실수로 덧셈, 뺄셈, 곱셈, 나눗셈을 할 수 있습니다:

```py
int_1 = 56
int_2 = 12
float_1 = 5.4
float_2 = 12.0

# Addition

print('Integer Addition:', int_1 + int_2) # Integer Addition: 68
print('Float Addition:', float_1 + float_2) # Float Addition: 17.4

# Subtraction

print('Int Subtraction:', int_1 - int_2) # Int Subtraction: 44
print('Float Subtraction:',  float_2 - float_1) # Float Subtraction: 6.6

# Multiplication

print('Int Multiplication:', int_1 * int_2) # Int Multiplication: 672
print('Float Multiplication:', float_2 * float_1) # Float Multiplication: 64.80000000000001

# Division

print('Int Division:', int_1 / int_2) # Int Division: 4.666666666666667
print('Float Division:', float_2 / float_1) # Float Division: 2.222222222222222
```

실수와 정수를 더하면 결과가 실수로 변환됩니다:

```py
int_1 = 56
float_1 = 5.4

print(int_1 + float_1) # 61.4
```

- **나머지 연산자(`%`)**: 두 수를 나눈 나머지를 반환합니다:

```py
int_1 = 56
int_2 = 12

print(int_1 % int_2) # 8
```

- **몫 연산자(`//`)**: 두 수를 나누고 결과를 내림하여 정수로 반환합니다:

```py
int_1 = 56
int_2 = 12

print(int_1 // int_2) # 4
```

- **거듭제곱 연산자(`**`)**: 한 수를 다른 수의 거듭제곱으로 올립니다:

```py
int_1 = 4
int_2 = 2

print(int_1 ** int_2) # 16
```

- **`float()` 함수**: 정수를 실수로 변환할 때 사용합니다.

```py
num = 4

print(float(num)) # 4.0
```

- **`int()` 함수**: 실수를 정수로 변환할 때 사용합니다.

```py
num = 4.0

print(int(num)) # 4
```

- **`round()` 함수**: 수를 가장 가까운 정수로 반올림할 때 사용합니다:

```py
num_1 = 3.4
num_2 = 7.7

print(round(num_1)) # 3
print(round(num_2)) # 8
```

- **`abs()` 함수**: 수의 절댓값을 반환할 때 사용합니다:

```py
num = -13

print(abs(num)) # 13
```

- **`bin()` 함수**: 정수를 이진 문자열로 변환할 때 사용합니다:

```py
num = 56

print(bin(num))  # 0b111000
```

- **`oct()` 함수**: 정수를 8진수 문자열로 변환할 때 사용합니다:

```py
num = 56

print(oct(num))  # 0o70
```

- **`hex()` 함수**: 정수를 16진수 문자열로 변환할 때 사용합니다:

```py
num = 56

print(hex(num))  # 0x38
```

- **`pow()` 함수**: 수를 거듭제곱할 때 사용합니다:

```py
result = pow(2, 3) 
print(result)  # 8
```

## 복합 대입 연산

- **정의**: 복합 대입은 이항 연산과 대입을 한 번에 수행합니다. 변수에 연산을 적용하고 결과를 같은 변수에 저장합니다.

```py
# Addition assignment 
my_var = 10
my_var += 5

print(my_var) # 15

# Subtraction assignment
count = 14
count -= 3

print(count) # 11

# Multiplication assignment 
product = 65
product *= 7

print(product) # 455

# Division assignment 
price = 100
price /= 4

print(price) # 25.0

# Floor Division assignment 
total_pages = 23
total_pages //= 5

print(total_pages) # 4

# Modulo assignment 
bits = 35
bits %= 2

print(bits) # 1

# Exponentiation assignment 
power = 2
power **= 3

print(power) # 8
```

비트 연산자용 복합 대입 연산자도 있습니다. `&=`, `^=`, `>>=`, `<<=` 등이 포함됩니다.

## 함수 다루기

- **정의**: 함수는 입력(인수)을 받아 출력을 반환하는 재사용 가능한 코드 조각입니다. 함수를 호출하려면 함수 이름 뒤에 소괄호를 붙이세요:

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call
print(result) # 7
```

함수가 명시적으로 값을 반환하지 않으면 기본 반환값은 `None`입니다:

```py
def greet():
    print('hello') 

result = greet() # hello
print(result) # None
```

매개변수에 기본값을 지정할 수도 있습니다:

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

인수가 올바른 개수가 아니면 `TypeError`가 발생합니다:

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

## 내장 함수

- **`input()` 함수**: 사용자 입력을 받을 때 사용합니다:

```py
name = input('What is your name?') # User types 'Kolade' and presses Enter  
print('Hello', name) # Hello Kolade
```

- **`int()` 함수**: 숫자, 불리언, 숫자 문자열을 정수로 변환할 때 사용합니다:

```py
print(int(3.14)) # 3
print(int('42')) # 42
print(int(True)) # 1
print(int(False)) # 0 
```

## 파이썬의 범위

- **지역 범위**: 함수나 클래스 내부에서 선언된 변수는 그 내부에서만 접근할 수 있습니다.

```py
def my_func():
    num = 10
    print(num)
```

- **포함 범위**: 함수 안에 중첩된 함수가 바깥 함수의 변수를 접근할 수 있습니다.

```py
def outer_func():
    msg = 'Hello there!'
    
    def inner_func():
        print(msg)
    inner_func()

print(outer_func()) # Hello there!
```

- **전역 범위**: 함수나 클래스 밖에서 선언된 변수로 프로그램 어디서든 접근할 수 있습니다.

```py
tax = 0.70 

def get_total(subtotal):
    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **내장 범위**: 파이썬 내장 함수, 모듈, 키워드, 객체에 예약된 이름입니다.

```py
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```

## 비교 연산자

- **같음(`==`)**: 두 값이 같은지 확인합니다:

```py
print(3 == 4) # False
```

- **같지 않음(`!=`)**: 두 값이 같지 않은지 확인합니다:

```py
print(3 != 4) # True
```

- **엄격히 큼(`>`)**: 한 값이 다른 값보다 큰지 확인합니다:

```py
print(3 > 4) # False
```

- **엄격히 작음(`<`)**: 한 값이 다른 값보다 작은지 확인합니다:

```py
print(3 < 4) # True
```

- **크거나 같음(`>=`)**: 한 값이 다른 값보다 크거나 같은지 확인합니다:

```py
print(3 >= 4) # False
```

- **작거나 같음(`<=`)**: 한 값이 다른 값보다 작거나 같은지 확인합니다:

```py
print(3 <= 4) # True
```

## `if`, `elif`, `else` 문 다루기

- **`if` 문**: 참인지 여부를 판단하는 조건문입니다. 조건이 `True`로 평가되면 해당 코드 블록이 실행됩니다.

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **`elif` 절**: `if` 문 뒤에 오는 조건입니다. `elif` 절은 이전 조건이 모두 `False`이고 자신의 조건이 `True`일 때만 실행됩니다.

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **`else` 절**: 다른 조건이 모두 `True`일 때 실행됩니다.

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

중첩된 `if` 문도 이렇게 사용할 수 있습니다:

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## 참 같은 값과 거짓 같은 값

- **정의**: 파이썬에서 모든 값은 논리적 맥락에서 `True` 또는 `False`로 간주되는 내재된 불리언 값을 가집니다. 많은 값이 참 같은 값으로 평가되어 `True`를 반환합니다. 거짓 같은 값은 `False`를 반환합니다. 거짓 같은 값의 예시는 다음과 같습니다:

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

0이 아닌 숫자나 비어 있지 않은 문자열 같은 값은 참 같은 값입니다.

## `bool()` 함수 다루기

- **정의**: 값이 참인지 거짓인지 확인하려면 내장 `bool()` 함수를 사용하세요. 이 함수는 값을 불리언으로 명시적 변환하며, 참 같은 값에 `True`를, 거짓 같은 값에 `False`를 반환합니다. 예시는 다음과 같습니다:

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## 불리언 연산자와 단락 평가

- **정의**: 여러 표현식을 결합해 복잡한 조건 논리를 만드는 특수 연산자입니다. 파이썬에는 `and`, `or`, `not` 세 가지 불리언 연산자가 있습니다.
- **`and` 연산자**: 두 피연산자를 받아 첫 번째가 거짓 같으면 반환하고, 그렇지 않으면 두 번째를 반환합니다. 두 피연산자가 모두 참 같아야 전체 표현식이 참 같습니다.

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

조건문에서 `and` 연산자를 이렇게 사용할 수 있습니다:

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **`or` 연산자**: 첫 번째 피연산자가 참 같으면 반환하고, 아니면 두 번째를 반환합니다. `or` 표현식은 피연산자 중 하나라도 참 같으면 참 같습니다. 예시는 다음과 같습니다:

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

`and` 연산자처럼 조건문에서 `or` 연산자를 이렇게 사용할 수 있습니다:

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **단락 평가**: `and`과 `or` 연산자는 단락 평가 연산자로, 왼쪽에서 오른쪽으로 값을 검사하며 최종 결과가 결정되면 평가를 멈춥니다.
- **`not` 연산자**: 단일 피연산자를 받아 불리언 값을 반전시킵니다. 참 같은 값을 `False`로, 거짓 같은 값을 `True`로 변환합니다. 이전 연산자와 달리 `not`는 항상 `True` 또는 `False`를 반환합니다. 예시는 다음과 같습니다:

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

조건문에서 `not` 연산자를 사용하는 예시는 다음과 같습니다:

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```
