# --interactive--

`break` 语句用于提前退出循环，而 `continue` 语句用于跳过循环的当前迭代并进入下一次迭代。

下面是一个在 `for` 循环中使用 `break` 语句的示例：

:::interactive_editor

```js
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}
```

:::

在上面的示例中，循环从 `0` 开始计数，当 `i` 小于 `10` 时，循环将继续运行。

在 `loop` 内部，我们查看 `i` 是否相等于 `5`。如果是，我们使用 `break` 语句提前退出 `loop`。如果不是，我们将 `i` 的值记录到控制台日志中。所以代码的输出将打印数字 `0`、`1`、`2`、`3` 和 `4`。

当你想根据某个条件提前退出循环时，`break` 语句非常有用。例如，如果你正在查找数组中的特定值，一旦找到该值，你可以使用 `break` 语句退出循环。

有时你可能想跳过循环的某次迭代而不完全退出循环。这时就用到了 `continue` 语句。下面是一个在 `for` 循环中使用 `continue` 语句的示例：

:::interactive_editor

```js
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}
```

:::

就像之前一样，我们已经初始化了 `i` 为 `0`，并且有一个条件，只要 `i` 小于 `10` 就会运行循环。

在循环内，当 `i` 相等 于 `5` 时，我们使用 `continue` 语句跳过当前迭代并进入下一次迭代。

这段代码的输出将打印数字 `0`、`1`、`2`、`3`、`4`、`6`、`7`、`8` 和 `9`。数字 `5` 因为 `continue` 语句而被跳过。

`break` 和 `continue` 语句的另一个用法是使用标签来指定你想要中断或继续的循环。

当你有嵌套循环并且想从内层循环中控制外层循环的流程时，这非常有用。

下面是使用标签和 `break` 语句的示例：

:::interactive_editor

```js
outerLoop: for (let i = 0; i < 3; i++) {
  innerLoop: for (let j = 0; j < 3; j++) {
    if (i === 1 && j === 1) {
      break outerLoop;
    }
    console.log(`i: ${i}, j: ${j}`);
  }
}
```

:::

在此示例中，我们有一个外部的 `for` 循环，标签为 `outerLoop`，以及一个内部的 `for` 循环，标签为 `innerLoop`。

当 `i` 相等于 `1` 且 `j`` 相等于 `1` 时，我们使用带有 `outerLoop` 标签的 `break` 语句来提前退出外层循环。这将退出内层和外层循环。

此代码的输出将向控制台记录以下内容：

```md
"i: 0, j: 0"
"i: 0, j: 1"
"i: 0, j: 2"
"i: 1, j: 0"
```

大多数情况下，你不会发现需要在 `break` 和 `continue` 语句中使用标签，但如果你需要，知道你有这个选项是很好的。
