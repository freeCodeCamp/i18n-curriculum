# --description--

Cuando quieres preguntar a alguien sobre su situación después de haber compartido la tuya, puedes usar `你呢 (nǐ ne)`, que significa "¿Y tú?" o "¿Qué hay de ti?".

La partícula interrogativa `呢 (ne)` se pronuncia con un tono neutral. A menudo se usa después de un pronombre para formar una pregunta sobre la situación mencionada previamente.

# --explanation--

`你呢 (nǐ ne)` combina el pronombre `你 (nǐ)` que significa "tú" con la partícula `呢 (ne)`, para formar una pregunta sobre la situación mencionada anteriormente. Significa "¿Y tú?" o "¿Qué hay de ti?" Ayuda a mantener la conversación de manera amigable y natural. Por ejemplo:

`我是数据分析师，你呢？` - Soy analista de datos. ¿Y tú?
