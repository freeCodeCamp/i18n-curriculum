# --description--

ここ数回のチャレンジでは、関数型プログラミングの原則に従った便利な配列や文字列のメソッドをいくつか学びました。また、問題をより単純な形に還元するために使われる強力なメソッドである`reduce`についても学びました。平均の計算からソートまで、あらゆる配列操作はこれを適用することで実現できます。`map`と`filter`は`reduce`の特別なケースであることを思い出してください。

学んだことを組み合わせて実用的な問題を解決しましょう。

多くのコンテンツ管理サイト（CMS）では、投稿のタイトルがURLの一部に追加されて簡単にブックマークできるようになっています。例えば、`Stop Using Reduce`というタイトルのMedium投稿を書くと、そのURLにはタイトル文字列の何らかの形（`.../stop-using-reduce`）が含まれていることが多いです。freeCodeCampのサイトでもすでに気づいているかもしれません。

# --instructions--

`urlSlug`関数を完成させて、文字列`title`を変換し、URL用のハイフン付きのバージョンを返してください。このセクションで扱ったメソッドを自由に使ってかまいませんが、`replace`は使わないでください。要件は以下の通りです：

入力はスペースを含み、各単語がタイトルケースになっている文字列です

出力は単語間のスペースがハイフン（`-`）に置き換えられた文字列です

出力はすべて小文字である必要があります

出力にスペースは含まれていません
