# --description--

Este áudio usa o tempo present perfect com `for + duration` para falar sobre há quanto tempo alguém está fazendo algo ou faz parte de algo.

A frase `be with` é usada para descrever ser membro de um grupo ou organização.Por exemplo, `She's been with the software development team for three years` significa que ela é membro da equipe que desenvolve software há três anos.
