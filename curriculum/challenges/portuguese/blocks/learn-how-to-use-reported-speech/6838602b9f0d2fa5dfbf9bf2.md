# --explanation--

`Unusually` é um advérbio usado para descrever algo que acontece mais ou menos do que o normal ou esperado.Por outro lado, `unusual` é um adjetivo que descreve algo que não é comum ou normal.Por exemplo:

- `The server experienced an unusually high load today.` – Nesta frase, `unusually` se refere a quão alta foi a carga (muito além do normal).  

- `It was unusual for the server to slow down like that.` – Nesta frase, `unusual` refere-se à situação. Não modifica um verbo ou um adjetivo.

Uma `query` é uma pergunta ou solicitação enviada a um banco de dados para obter informações ou executar uma ação.Por exemplo:

`The application sent a query to the database to fetch user details.` – Isso significa que o app solicitou ao banco de dados informações específicas.
