# --interactive--

Ci sono diversi modi per rimuovere proprietà da un oggetto, con l'operatore `delete` che è il metodo più semplice e comunemente usato.

Quando usi `delete`, rimuove la proprietà selezionata dall'oggetto. Ecco un esempio di come usare l'operatore `delete`:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  job: "Engineer"
};

delete person.job;

console.log(person.job); // undefined
```

:::

In questo esempio, partiamo da un oggetto `person` che ha tre proprietà: `name`, `age` e `job`. Poi, usiamo l'operatore `delete` per rimuovere la proprietà `job`. Dopo l'eliminazione, l'oggetto `person` non ha più la proprietà `job`.

Un altro modo per rimuovere proprietà è usare l'assegnazione destrutturante con i parametri rest. Questo approccio non elimina realmente la proprietà, ma crea un nuovo oggetto senza le proprietà specificate:

:::interactive_editor

```js
const person = {
  name: "Bob",
  age: 25,
  job: "Designer",
  city: "New York"
};

const { job, city, ...remainingProperties } = person;

// { name: "Bob", age: 25 }
console.log(remainingProperties);
```

:::

In questo esempio, usiamo la destrutturazione per estrarre `job` e `city` dall'oggetto `person`, e raccogliere le proprietà rimanenti in un nuovo oggetto chiamato `remainingProperties`. Questo crea un nuovo oggetto senza le proprietà `job` e `city`.

Capire come rimuovere proprietà dagli oggetti è una competenza importante nella programmazione JavaScript. Ti permette di manipolare gli oggetti in modo dinamico e di pulire dati non necessari.
