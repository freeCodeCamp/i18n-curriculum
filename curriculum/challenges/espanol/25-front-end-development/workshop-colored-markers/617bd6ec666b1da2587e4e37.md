# --description--

Hay tres colores terciarios más: verde chartreuse (verde + amarillo), azul (azul + cian), y rosa (rojo + magenta).

Para crear verde chartreuse, actualiza la función `rgb` en la regla CSS `.one` para que el rojo esté en `127`, y pon el verde al valor máximo.

Para el azul, actualiza la función `rgb` en la regla CSS `.two` de manera que el verde esté en `127` y el azul al valor máximo.

Y para el rosa, que a veces se llama rosa brillante, actualiza la función `rgb` en la regla CSS `.three` de modo que el azul esté en `127` y el rojo al valor máximo.
