# --description--

连接词 `so` 用于引入与所述内容相关的结果或效果。它通常显示句子两部分之间的因果关系。

例如，`It was raining, so we canceled the picnic.` 这表明野餐是因为下雨而取消的。

# --questions--

## --text--

哪个句子正确使用了 `so`？

## --answers--

`I was hungry so I am eating.`

### --feedback--

这句话几乎正确使用了 `so`，但第二部分的时态应与第一部分匹配。正确的说法是：`I was hungry, so I ate.`

---

`She is late to work, so she misses the meeting.`

### --feedback--

使用 `so` 表示结果，但时态不匹配。应为：`She is late to work, so she will miss the meeting.`

---

`It started raining, so we took shelter.`

---

`He likes to read books, so he buys a lamp.`

### --feedback--

这个句子没有正确使用 `so`，因为买灯不是喜欢阅读书籍的直接结果。一个更好的用法可能是：`He likes to read at night, so he bought a lamp.`
