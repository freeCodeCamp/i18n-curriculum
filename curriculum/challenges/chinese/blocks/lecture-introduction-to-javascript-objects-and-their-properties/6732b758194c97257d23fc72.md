# --interactive--

在 JavaScript 中，构造函数是一种用于创建和初始化对象的特殊类型函数。它通过 `new` 关键字调用，并且可以初始化新创建对象上的属性和方法。

在本课中，我们将了解如何使用 `Object()` 构造函数。`Object()` 构造函数创建一个新的空对象。下面是一个示例：

```js
new Object()
```

当你调用 `new Object()` 时，它会返回一个可用于保存值的新建对象。

`Object()` 构造函数可以带或不带 `new` 关键字使用。当作为函数调用且不带 `new` 时，其行为取决于传入值的类型。以下是不带 `new` 关键字使用 `Object()` 构造函数的示例：

:::interactive_editor

```js
const num = 42;
const numObj = Object(num); // Creates an object wrapper for the number

console.log(numObj);
console.log(typeof numObj); // "object"
```

:::

正如你在第二个 `console.log` 中看到的，`numObj` 是一个对象。这是因为我们使用了 `Object()` 构造函数将那个数字的输入转换成了对象。 

如果我们尝试将 `null` 或 `undefined` 传递给 `Object()` 构造函数，会发生什么？

:::interactive_editor

```js
const newObj = new Object(undefined);
console.log(newObj); // {}
```

:::

结果将是一个空对象。`Object()` 构造函数的另一个使用场景是当你处理一个未知类型的值并且需要确保它是一个对象时。让我们来看以下示例：

:::interactive_editor

```js
function toObject(value) {
  if (value === null || value === undefined) {
    return {};
  }

  if (typeof value === "object") {
    return value;
  }

  return Object(value);
}

console.log(toObject(null));

console.log(toObject(true));

console.log(toObject([1, 2, 3]));
```

:::

在此示例中，我们有一个名为 `toObject` 的函数。第二个条件将查看值是否为对象类型，如果条件为 `true`，则返回该值。此条件将检查对象以及数组，因为数组是对象的特殊类型。

如果两个条件都不为真，函数将返回 `Object(value)`，它会将输入转换为对象。这适用于数字、字串和布尔等值。

大多数情况下你不会使用 `Object()` 构造函数来新建对象，因为你会使用对象字面语法（例如，`const objectLiteral = { name: "Beau" }`）。但理解使用 Object 构造函数的基础知识仍然很重要。
