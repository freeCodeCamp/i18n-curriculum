# --description--

Um `database` é uma coleção estruturada de dados, tipicamente armazenada eletronicamente em um sistema de computador. É usado para organizar, armazenar e recuperar grandes quantidades de informações de forma eficiente.

Em programação, um `database` é essencial para gerenciar os dados que as aplicações precisam para funcionar.

# --questions--

## --text--

O que é um `database` no contexto da programação?

## --answers--

Uma coleção estruturada de dados armazenados eletronicamente.

---

Uma ferramenta para criar sites.

### --feedback--

Embora importante no desenvolvimento web, um `database` não é uma ferramenta para projetar sites, mas para armazenar dados.

---

Uma linguagem de programação usada para criar aplicações.

### --feedback--

Um `database` não é uma linguagem de programação; é um sistema para armazenar e gerenciar dados.

---

### --feedback--

Um `database` é uma forma segura e estruturada de armazenar dados, não um vírus de computador.
