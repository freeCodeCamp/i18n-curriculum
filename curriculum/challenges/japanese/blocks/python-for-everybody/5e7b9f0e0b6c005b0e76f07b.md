# --questions--

## --text--

次のXMLのどこが間違っていますか？:

```xml
<person>
  <name>Chuck</name>
  <phone type="intl">
    +1 734 303 4456
  <email hide="yes" />
</person>
```

## --answers--

Emailタグに終了タグがありません。

---

スペースの使い方でXMLが無効になります。

---

Phoneタグに終了タグがありません。

---

プレーンテキストはUTF-8でエンコードする必要があります。
