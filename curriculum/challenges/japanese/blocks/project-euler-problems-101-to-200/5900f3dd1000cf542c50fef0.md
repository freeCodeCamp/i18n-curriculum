# --description--

左から右に向かって、ある桁の数字がその左の数字を超えない場合、それは増加数と呼ばれます。例えば、134468です。

同様に、ある桁の数字がその右の数字を超えない場合、それは減少数と呼ばれます。例えば、66420です。

増加数でも減少数でもない正の整数を「バウンシー数」と呼びます。例えば、155349です。

nが大きくなるにつれて、n未満のバウンシー数の割合は増加し、100万未満の非バウンシー数は12951個しかなく、${10}^{10}$未満の非バウンシー数は277032個しかありません。

グーゴル（${10}^{100}$）未満の非バウンシー数はいくつありますか？
