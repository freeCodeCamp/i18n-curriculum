# --explanation--

Ángela sagt `¿Y tu profesión?`, eine kurze Anschlussfrage. `Y` verbindet diese Frage mit dem vorherigen Thema.

Obwohl sie `¿Cuál es...?` nicht sagt, wird die vollständige Frage aus dem Kontext verstanden:

`¿Cuál es tu profesión?`

Im Gespräch lässt das Spanische oft wiederholte Wörter weg, wenn die Bedeutung klar ist.
