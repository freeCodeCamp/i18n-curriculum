# --description--

Normalmente, el tiempo `Past Simple` se forma añadiendo `-ed` a los verbos regulares. Por ejemplo, el verbo `play` se convierte en `played`.

`To play around` significa probar algo de manera casual o experimental, a menudo solo para ver qué pasa. Usualmente va seguido de `with`. Por ejemplo:

`He played around with the new software to see how it works.` - Esto significa que él probó el software sin un plan serio, solo para explorarlo.
