# --explanation--

`Really` und `too` sind beide **Intensivierer**, aber sie haben unterschiedliche Bedeutungen. `Really` verstärkt das Adjektiv, das es modifiziert, ohne Übermaß anzuzeigen. `Too` hingegen zeigt Übermaß an, also mehr als akzeptabel oder handhabbar ist. Zum Beispiel:

- `The project is really important.` – Das bedeutet, dass das Projekt sehr wichtig ist, aber nicht unbedingt zu viel.

- `This task is too complicated.` – Das bedeutet, dass es komplizierter ist, als es sein sollte.
