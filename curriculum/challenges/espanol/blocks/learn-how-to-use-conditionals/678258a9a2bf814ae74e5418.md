# --explanation--

María sugiere la automatización porque minimiza las posibilidades de `human error`. Ella argumenta que esto sería una mejora significativa en comparación con depender únicamente de recordatorios, ya que la automatización asegura que las copias de seguridad se realicen incluso si alguien olvida. Esto está alineado con el objetivo del equipo de crear un proceso de respaldo más confiable.

Destaca el razonamiento de Maria de que la automatización no solo complementa los recordatorios actuales, sino que también fortalece el sistema al eliminar la dependencia manual.
