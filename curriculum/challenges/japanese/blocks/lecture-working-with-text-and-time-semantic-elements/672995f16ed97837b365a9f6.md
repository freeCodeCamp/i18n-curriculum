# --interactive--

`time` 要素は特定の瞬間を表すために使われます。

ここでは、`time` 要素を使って午後8時、つまり20時を表す例を示します。

:::interactive_editor

```html
<p>The reservations are for <time datetime="20:00">20:00 </time></p>
```

:::

`datetime` 属性は、日付や時刻を機械が読み取れる形式に変換するために使われます。

これは重要で、検索エンジンの結果に役立ち、ブラウザが日付や時刻の情報をより効果的に処理できるようにします。

`datetime` 属性の値は、有効な年、有効な月、有効な時刻、ローカル日付、グローバル日付、または有効な期間の文字列である必要があります。

こちらは、time 要素を使って特定の日付を表す別の例です：

:::interactive_editor

```html
<p>
  The graduation will be on <time datetime="2024-06-15T15:00">June 15</time>
</p>
```

:::

`datetime` 属性の値はISO 8601形式です。ISO 8601は日付と時刻を表す国際標準です。

その値の最初の部分は年、月、日を表します。値の中の大文字のTは日付と時刻の間の区切りです。

15時は午後3時にあたります。

イベント、公開日、予定を表す必要がある場合は、`time` 要素を使うのが最適です。
