# --explanation--

`We should` — це загальний спосіб запропонувати ідею або дати пораду. Це означає, що мовник пропонує дію або ідею, яку вважає хорошим вибором. Наприклад:

`We should start the meeting earlier to give everyone more time.` – це означає, що мовник пропонує раніший час початку як корисну зміну.
