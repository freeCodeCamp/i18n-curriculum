# --description--

`Not right away`는 무언가가 즉시 또는 가장 빠른 기회에 일어나지 않았음을 나타낼 때 사용합니다. 예를 들어:

`He didn't understand the code fix right away, but later it became clear.` - 그는 이해하기 전에 시간이 좀 필요했다.

이 표현은 행동이나 변화가 일어나기 전에 지연이나 기간이 있었음을 암시합니다. 사건이나 행동의 시간 순서를 설정하는 데 도움이 됩니다.
