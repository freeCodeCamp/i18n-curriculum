# --description--

Wenn Sie jemanden zum ersten Mal treffen, können Sie einen gebräuchlichen Ausdruck verwenden, um auszudrücken, dass Sie sich freuen, ihn kennenzulernen.

# --instructions--

Hören Sie sich die Audioaufnahme an und vervollständigen Sie den folgenden Satz.

# --fillInTheBlank--

## --sentence--

`BLANK BLANK 你 (nǐ)，王华 (wáng huá)。`

## --blanks--

`很高兴 (hěn gāo xìng)`

### --feedback--

Dieser Ausdruck bedeutet „sehr glücklich“.

---

`认识 (rèn shì)`

### --feedback--

Das bedeutet „kennen“ oder „treffen“.

# --explanation--

`很高兴认识你 (hěn gāo xìng rèn shì nǐ)` bedeutet „Freut mich, Sie kennenzulernen“. Es ist ein gebräuchlicher Ausdruck, der verwendet wird, wenn man jemanden zum ersten Mal trifft. Er zeigt Höflichkeit und Freundlichkeit.

Ein interessanter Hinweis ist, dass das Subjekt „Ich“ in diesem Ausdruck oft weggelassen wird, da aus dem Kontext klar ist, dass der Sprecher von sich selbst spricht. Sie können auch `我很高兴认识你 (wǒ hěn gāo xìng rèn shì nǐ)` sagen, was dieselbe Bedeutung hat wie die Version ohne `我 (wǒ)`.
