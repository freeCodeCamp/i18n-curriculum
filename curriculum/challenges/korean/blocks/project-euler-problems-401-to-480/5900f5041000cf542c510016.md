# --description--

$0 ≤ a ≤ 5$에 대해 $a^2\bmod 6$를 계산하면 0, 1, 4, 3, 4, 1이 나옵니다.

$a^2 ≡ a\bmod 6$를 만족하는 가장 큰 $a$ 값은 $4$입니다.

$M(n)$을 $a^2 ≡ a (\text{mod } n)$를 만족하는 $a < n$ 중 가장 큰 값이라고 합시다. 따라서 $M(6) = 4$입니다.

$1 ≤ n ≤ {10}^7$에 대해 $\sum M(n)$을 구하세요.
