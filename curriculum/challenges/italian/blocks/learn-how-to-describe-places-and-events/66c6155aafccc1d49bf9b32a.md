# --explanation--

Quando qualcuno fa una domanda del tipo `Have you` + verbo ?, sta cercando di scoprire se hai fatto qualcosa di specifico.

In questo caso, Sarah chiede `Have you checked the settings?` perché vuole sapere se Tom ha già compiuto un'azione specifica che potrebbe risolvere il problema.
