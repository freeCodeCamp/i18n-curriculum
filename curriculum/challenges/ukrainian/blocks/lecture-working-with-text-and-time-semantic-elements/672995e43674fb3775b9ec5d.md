# --interactive--

Елемент контактної адреси використовується для представлення контактної інформації для розділу на вебсторінці. Елемент `address` універсальний і може використовуватися для бізнес-сторінок, сторінок авторів, особистих сайтів і не тільки.

Коли ви створюєте розділи контактів на своєму сайті, слід використовувати семантичний елемент `address` замість загального елемента, такого як `div`.

Ось приклад використання елемента `address` для сторінки контактів компанії:

:::interactive_editor

```html
<address>
  <h2>Company Name</h2>
  <p>
    1234 Elm Street<br />
    Springfield, IL 62701<br />
    United States
  </p>
  <p>Phone: <a href="tel:+15555555555">+1 (555) 555-5555</a></p>
  <p>Email: <a href="mailto:contact@company.com">contact@company.com</a></p>
</address>
```

:::

У цьому прикладі є назва компанії, фізична адреса, телефон і інформація про е-пошту. Для фізичної адреси використовується елемент переносу рядка `br`, щоб показати поділ між назвою вулиці, містом і країною.

Для номера телефону ми маємо елемент посилання з атрибутом `href`, встановленим для телефонних номерів. Значення `tel:+` всередині атрибута `href` створює активне посилання для ініціації дзвінка на певних пристроях, які це підтримують.

Для адреси е-пошти використовується інший елемент посилання з атрибутом `href`, встановленим на посилання `mailto`. Посилання `mailto` використовуються в HTML-документах, щоб дозволити користувачам відкрити новий лист у своєму улюбленому клієнті е-пошти.

Одним із недоліків використання посилання `mailto` є те, що користувачі часто сприймають його як спам. На жаль, багато спамерів використовують цей варіант, щоб надсилати листи користувачам. Тож майте це на увазі, коли використовуєте його.
