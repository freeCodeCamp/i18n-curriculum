# --explanation--

`Lastly` вводить останній елемент або пункт. Наприклад:

`Lastly, please remember to lock the door.` – Це показує, що це остання інструкція.

`Ethernet cables` — це дроти, які використовують для підключення комп’ютерів та інших пристроїв до мережі. Наприклад:

`We need more Ethernet cables for the new desks.` – Це означає, що потрібні додаткові мережеві кабелі.

`Ended up` означає, де хтось або щось врешті-решт опинилося. Наприклад:

`The documents ended up on my desk.` – Це означає, що саме там їх зрештою знайшли.
