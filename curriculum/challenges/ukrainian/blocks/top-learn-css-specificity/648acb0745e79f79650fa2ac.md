# --description--

Останній фактор, фіналь, вирішальний аргумент у суперечці. Припустімо, що після врахування всіх інших факторів залишається кілька конфліктних правил, які застосовуються до одного елемента. Як каскад визначає, яке правило застосувати?

Насправді дуже просто. Перемагає те правило, яке було визначене останнім.

```css
/* styles.css */

.alert {
  color: red;
}

.warning {
  color: yellow;
}
```

Для елемента, який має класи `alert` і `warning`, каскад пройде через усі інші фактори, включно з успадкуванням (тут його немає) і специфічністю (жодне правило не є більш специфічним за інше). Оскільки правило `.warning` було визначене останнім і жоден інший фактор не зміг визначити, яке правило застосувати, саме воно застосовується до елемента.
