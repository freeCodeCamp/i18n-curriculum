# --description--

Unapojenga miradi kwa JavaScript, mara nyingi unahitaji zana za ziada kusaidia kukamilisha kazi. Hata programu rahisi zinaweza kuhitaji vipengele kama kuhifadhi data, kuingia (login), kuhamia kati ya kurasa (routing), na kadhalika.

Badala ya kuandika msimbo huu wote kutoka mwanzo, unaweza kutumia vifurushi vya chanzo wazi ambavyo wasanidi programu wengine tayari wamejenga na kushiriki.

Hapa ndipo npm inapoingia. npm ni rejista kubwa mtandaoni ya programu za chanzo wazi. Fikiria kama duka la programu kwa msimbo — unaweza kutafuta unachohitaji, kuisakinisha, na kuanza kuitumia katika mradi wako.

Watu wengi hufikiri npm inamaanisha "Node Package Manager," lakini hiyo si maana rasmi yake. Katika nyaraka za zamani za npm, ilielezewa kama "mufano wa mnyororo wa maneno unaojirudia wa 'npm is not an acronym.'" Kwa maneno mengine, npm haina maana rasmi yoyote.

Leo, npm inahusu sehemu tatu zinazohusiana:

* Tovuti npmjs.com, ambapo unaweza kutafuta vifurushi katika rejista.
    
* Rejista ya npm, ambayo ni hifadhidata kubwa ya umma inayohifadhi vifurushi vyote.
    
* Zana ya kiolesura cha mstari wa amri ya npm inayosakinishwa pamoja na Node.js na inakuwezesha kusakinisha na kusimamia vifurushi.
    

Kwa mfano, ikiwa unahitaji kusimamia hali (state), kushughulikia tarehe, kutuma barua pepe, au kufanya maombi ya HTTP katika mradi wako, kuna uwezekano kuna kifurushi cha npm ambacho tayari kinafanya hayo. Kwa amri rahisi ya `npm install`, unaweza kuanza kutumia programu hii tayari badala ya kufanya kazi yote mwenyewe.

npm pia inakusaidia kusimamia utegemezi wa mradi wako. Hii ina maana kwamba inafuatilia vifurushi unavyotumia katika miradi yako, na toleo la kila kifurushi. Hii hufanyika kwa kutumia faili za `package.json` na `package-lock.json`, ambazo utajifunza kuhusu hizo katika masomo yajayo.

Yarn, PNPM, na Bun ni baadhi ya mbadala za npm ambazo zimekuwa maarufu kwa muda. Ingawa kila moja yao hutoa faida katika maeneo maalum kama kasi ya usakinishaji na ufanisi wa hifadhi, npm bado ni inayotumika zaidi.

Katika masomo yajayo, utajifunza amri mbalimbali za npm, jinsi ya kusakinisha vifurushi, kuhusu faili za `package.json` na `package-lock.json`, na hata jinsi ya kuchapisha kifurushi chako mwenyewe kwenye rejista ya npm.
