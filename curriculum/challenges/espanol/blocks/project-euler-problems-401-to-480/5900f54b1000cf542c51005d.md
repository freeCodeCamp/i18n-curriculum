# --description--

Sea $a_k$, $b_k$ y $c_k$ las tres soluciones (números reales o complejos) de la expresión $\frac{1}{x} = {\left(\frac{k}{x} \right)}^2 (k + x^2) - kx$.

Por ejemplo, para $k = 5$, vemos que $\\{a_5, b_5, c_5\\}$ es aproximadamente $\\{5.727244, -0.363622 + 2.057397i, -0.363622 - 2.057397i\\}$.

Sea $S(n) = \displaystyle\sum_{p = 1}^n \sum_{k = 1}^n {(a_k + b_k)}^p {(b_k + c_k)}^p {(c_k + a_k)}^p$ para todos los enteros $p$, $k$ tales que $1 ≤ p, k ≤ n$.

Curiosamente, $S(n)$ siempre es un entero. Por ejemplo, $S(4) = 51\\,160$.

Encuentra $S({10}^6) \text{ módulo } 1\\,000\\,000\\,007$.
