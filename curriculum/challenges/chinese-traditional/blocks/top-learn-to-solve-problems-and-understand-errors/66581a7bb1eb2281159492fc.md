## 偽程式碼

偽程式是用自然語言而非程式碼來寫出你程式的邏輯。它幫助你放慢速度，並思考你的程式必須經過哪些步驟來解決問題。

以下是列印所有小於或等於輸入數字的程式的偽程式碼範例：

```md
When the user inputs a number
Initialize a counter variable and set its value to zero
While counter is smaller than user inputted number increment the counter by one
Print the value of the counter variable
```

這是一個基本的程式，用來示範偽程式碼的樣子。作業中會包含更多偽程式碼的範例。

根據該聯結文章，使用偽程式碼的利益是什麼？

## --answers--

偽程式碼主要對初學者有益，因為它幫助他們透過跳過複雜的語法，更快學習程式語言。

---

偽程式碼簡化了與非程式設計師的通訊，有助於程式碼構件，提供良好的文件基礎，並協助偵錯。

---

偽程式碼完全消除了程式設計師理解或使用程式語言的需求，使其成為專業程式員的主要工具。

---

偽程式碼專門用於建立詳細的流程圖，以取代複雜軟體開發環境中的傳統程式碼。
