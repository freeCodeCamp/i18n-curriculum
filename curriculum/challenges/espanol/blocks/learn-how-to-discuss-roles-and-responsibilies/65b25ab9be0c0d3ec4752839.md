# --description--

Cuando se describe la apariencia de alguien, a menudo se usa `with` seguido de una característica.

Ejemplos:

`with glasses` para alguien que usa gafas

`with curly hair` para alguien que tiene el cabello rizado, o

`with a blue jacket` para alguien que lleva una chaqueta azul
