# --interactive--

طريقة `splice()` في JavaScript هي وسيلة قوية لتعديل المصفوفات. تتيح لك إضافة أو حذف عناصر من أي موقع في المصفوفة، بما في ذلك المنتصف. قيمة الإرجاع لطريقة `splice()` ستكون مصفوفة بالعناصر التي حُذفت من المصفوفة. إذا لم يُحذف شيء، فستُرجع مصفوفة فارغة.

من المهم ملاحظة أن هذه الطريقة تُغير المصفوفة الأصلية، فتُعدلها في مكانها بدلاً من إنشاء مصفوفة جديدة. يجب أن تأخذ هذا في الاعتبار عند العمل مع `splice()`. إليك الصيغة النحوية الأساسية:

```js
array.splice(startIndex, itemsToRemove, item1, item2)
```

`startIndex` يحدد الفهرس الذي تبدأ عنده تعديل المصفوفة، بينما `itemsToRemove` هي معلمة اختيارية تحدد عدد العناصر التي تُحذف. إذا حُذفت `itemsToRemove`، فستُزيل `splice()` كل العناصر من `startIndex` حتى نهاية المصفوفة. المعلمات التالية (`item1`، `item2`، وهكذا) هي العناصر التي تُضاف إلى المصفوفة بدءًا من فهرس البداية.

لنبدأ بمثال على حذف عناصر من منتصف المصفوفة:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango", "kiwi"];
let removed = fruits.splice(2, 2);

console.log(fruits);  // ["apple", "banana", "kiwi"]
console.log(removed); // ["orange", "mango"]
```

:::

في هذا المثال، يبدأ `splice(2, 2)` عند الفهرس `2` ويزيل `2` عنصرين. ستتكون المصفوفة المعدلة الآن من `apple` و`banana` و`kiwi` فقط. الآن، لنلقِ نظرة على كيفية إضافة عناصر إلى منتصف المصفوفة:

:::interactive_editor

```js
let colors = ["red", "green", "blue"];
colors.splice(1, 0, "yellow", "purple");

console.log(colors); // ["red", "yellow", "purple", "green", "blue"]
```

:::

هنا، تبدأ `splice(1, 0, "yellow", "purple")` عند الفهرس `1`، وتزيل `0` عناصر، وتُدرج `yellow` و`purple`. المعلمة الثانية (`0`) تعني أنه لا تتم إزالة أي عناصر قبل الإدراج. يمكنك أيضًا استخدام `splice()` لإزالة عناصر وإضافة أخرى في الوقت نفسه:

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];
numbers.splice(1, 2, 6, 7, 8);

console.log(numbers); // [1, 6, 7, 8, 4, 5]
```

:::

في هذه الحالة، تبدأ الدالة `splice(1, 2, 6, 7, 8)` عند الفهرس `1`، وتزيل `2` عنصرين (`2` و `3`)، ثم تُدرج العناصر `6` و `7` و `8`. إذا كنت تريد الحفاظ على المصفوفة الأصلية دون تغيير، يجب عليك إنشاء نسخة منها قبل استخدام `splice()`:

:::interactive_editor

```js
let original = [1, 2, 3, 4, 5];
let copy = [...original];
copy.splice(2, 1, 6);

console.log(original); // [1, 2, 3, 4, 5]
console.log(copy);     // [1, 2, 6, 4, 5]
```

:::

في هذا المثال، لإنشاء نسخة من المصفوفة `original` دون تعديلها، نستخدم معامل النشر (`...`). معامل النشر يُنشئ نسخة سطحية من عناصر المصفوفة `original` في مصفوفة جديدة. ستتعلم المزيد عن هذا في الدروس القادمة.

عندما نستخدم `copy.splice(2, 1, 6)`، فإنه يعدل مصفوفة النسخة بحذف العنصر عند الفهرس `2` (وهو `3`) وإدراج العنصر الجديد `6` في ذلك الموضع.

استخدام شائع لطريقة `splice()` هو حذف عنصر واحد من المصفوفة عندما تعرف فهرسه:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango"];
let indexToRemove = fruits.indexOf("orange");
if (indexToRemove !== -1) {
    fruits.splice(indexToRemove, 1);
}

console.log(fruits); // ["apple", "banana", "mango"]
```

:::

في هذا المثال، نستخدم أولاً طريقة `indexOf()` لإيجاد فهرس العنصر `orange` في مصفوفة `fruits`. طريقة `indexOf()` تُرجع فهرس أول ظهور للعنصر المعطى أو `-1` إذا لم يُعثر على العنصر في المصفوفة.

نقارن بعد ذلك `indexToRemove` مع `-1` للتأكد من وجود العنصر في المصفوفة قبل محاولة حذفه. إذا لم يكن `indexToRemove` يساوي `-1` (أي تم العثور على العنصر)، نستخدم `splice()` لحذف عنصر واحد بدءًا من موقع `indexToRemove`.

يمكنك أيضًا استخدام `splice()` لمسح المصفوفة بحذف كل العناصر:

:::interactive_editor

```js
let array = [1, 2, 3, 4, 5];
array.splice(0);

console.log(array); // []
```

:::

بينما `splice()` قوية، من الجدير بالذكر أنه بالنسبة للمصفوفات الكبيرة جدًا، قد تكون أقل كفاءة من طرق أخرى، خاصة عند تعديل بداية المصفوفة. ذلك لأن `splice()` قد يحتاج إلى تحريك كل العناصر التالية. في هذه الحالات، إذا كنت تضيف أو تحذف عناصر فقط من نهاية المصفوفة، قد تكون الطرق مثل `push()`، `pop()`، `unshift()`، و`shift()` أكثر ملاءمة.

في الختام، طريقة `splice()` هي وسيلة متعددة الاستخدامات لتعديل المصفوفات في JavaScript. تتيح تحكمًا دقيقًا في إضافة وحذف العناصر من أي موقع في المصفوفة. فهم كيفية استخدام `splice()` بفعالية يعزز كثيرًا قدرتك على التعامل مع المصفوفات في شفرتك البرمجية.
