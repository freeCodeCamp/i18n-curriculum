## Що таке Python?

- **Вступ**: Python — це мова програмування загального призначення, відома своєю простотою та легкістю у використанні. Python застосовується у багатьох сферах, таких як наука про дані та машинне навчання, веб-розробка, скрипти та автоматизація, вбудовані системи та Інтернет речей, і багато іншого.
- **Поширені випадки використання**: Python використовується у науці про дані, машинному навчанні, веб-розробці, кібербезпеці, автоматизації та на мікрокомп’ютерах, таких як Raspberry Pi та плати, сумісні з MicroPython.

## Python у вашому локальному середовищі

- **Встановлення**: Найкращий спосіб встановити Python на Windows, Mac і Linux — скачати інсталятор з офіційного сайту Python (`https://www.python.org/`).
- **Термінал**: Текстовий інтерфейс, який дозволяє взаємодіяти з комп’ютером, вводячи команди. Кожна операційна система має свій стандартний термінал. На macOS можна використовувати застосунок Terminal. На Windows — Command Prompt або PowerShell. На Linux кожне робоче середовище має свій стандартний термінал, наприклад GNOME Terminal або Konsole.
- **IDE**: IDE означає інтегроване середовище розробки, яке має такі функції, як редактор коду, інструменти тестування, термінал та інше.
- **Популярні редактори коду та IDE для Python**: VS Code, PyCharm і Spyder
- **Запуск коду локально**: Щоб запустити код Python, у вас є кілька варіантів. Перший — натиснути кнопку запуску у VS Code, яка виглядає як кнопка відтворення у верхньому правому куті. Це відкриє термінал і запустить ваш скрипт Python там.

Інший варіант — відкрити термінал і вручну запустити програму. Ось приклад запуску програми з назвою `main.py`:

```bash
python main.py
```

Цю команду потрібно запускати з папки, де збережено `main.py`. Наприклад, якщо `main.py` знаходиться у папці `python-projects`, спочатку використайте у терміналі:

```bash
cd python-projects
```

Потім запустіть:

```bash
python main.py
```

## Використання інтерактивної оболонки Python

- **Визначення**: Інтерактивна оболонка — це програма, яка дозволяє вводити команди по одній і бачити результати. Відкрийте застосунок терміналу і введіть `python`, потім натисніть `Enter`. Це запустить інтерактивну оболонку Python. Символ `>>>` означає, що Python чекає на введення команди. Спробуйте вивести `"Hello, world!"` у термінал:

```bash
print("Hello, world!")
```

Ви побачите текст, який з’явиться так:

```bash
>>> print("Hello, world!")
Hello, world!
```

Після виведення тексту Python повернеться до такого вигляду:

```bash
>>>
```

Це означає, що ви можете ввести ще одну команду.

- **Цикл читання, оцінки, виведення, повторення (REPL)**: Коли ви вводите команди, інтерпретатор читає вхідні дані, оцінює їх, виводить результат і повертається, щоб показати `>>>`, щоб ви могли вводити нові команди.

## Змінні

- **Оголошення змінних**: Щоб оголосити змінну, почніть із імені змінної, потім оператор присвоєння (`=`) і значення. Це може бути число, рядок, булевий тип тощо. Ось кілька прикладів:

```py
name = 'John Doe'
age = 25
```

- **Правила іменування змінних**: Ось правила, які слід використовувати для імен змінних:

  - Імена змінних можуть починатися лише з літери або підкреслення (_), а не з цифри.
  - Імена змінних можуть містити лише алфавітно-цифрові символи (a-z, A-Z, 0-9) і підкреслення (_).
  - Імена змінних чутливі до регістру — `age`, `Age` і `AGE` вважаються різними.
  - Імена змінних не можуть бути зарезервованими ключовими словами Python, такими як `if`, `class` або `def`.
  - Імена змінних із кількома словами розділяються підкресленнями. Наприклад, `snake_case`.


## Коментарі

- **Однорядкові коментарі**: Ці коментарі використовуються для коротких нотаток у коді.

```py
# This is a single line comment
```

- **Багаторядкові рядки**: Ці рядки можна використовувати для більших нотаток або щоб перетворити частини коду на коментарі.

```py
"""

Here is some code commented out.

"""
```

- **Функція `print()`**: Щоб вивести дані у консоль, можна використовувати функцію `print()` так:

```py
print('Hello world!') # Hello world!
```

## Поширені типи даних у Python

- **Вступ**: Python — це мова з динамічною типізацією, як і JavaScript, тобто вам не потрібно явно оголошувати типи змінних. Мова визначає тип змінної на основі того, що ви їй присвоюєте.
- **Ціле число**: Ціле число без десяткових знаків:

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **Число з плаваючою крапкою**: Число з десятковими знаками:

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **Рядок**: Послідовність символів, взятих у лапки:

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **Булевий тип**: Значення, що представляє або `True`, або `False`:

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **Множина**: Несортована колекція унікальних елементів:

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **Словник**: Колекція пар ключ-значення, взята у фігурні дужки:

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **Кортеж**: Незмінна впорядкована колекція, взята у круглі дужки:

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **Діапазон**: Послідовність чисел, часто використовується у циклах:

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **Список**: Впорядкована колекція елементів, що підтримує різні типи даних:

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**: Спеціальне значення, що означає відсутність значення:

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## Незмінні та змінні типи

- **Незмінні типи**: Ці типи не можна змінювати після оголошення, хоча ви можете призначити змінній нове значення, що називається переназначенням. До них належать ціле число, число з плаваючою крапкою, комплексне число, булевий тип, рядок, кортеж, діапазон і `None`.
- **Змінні типи**: Ці типи можна змінювати після оголошення. Ви можете додавати, видаляти або оновлювати їхні елементи. До них належать колекції, такі як список, множина та словник.
- **Функція `type()`**: Щоб побачити тип змінної, можна використовувати функцію `type()` так:

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **Функція `isinstance()`**: Використовується для перевірки, чи відповідає змінна певному типу даних:

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## Робота з рядками

- **Визначення**: Як ви пам’ятаєте з JavaScript, рядки незмінні, тобто їх не можна змінити після створення. У Python можна використовувати як одинарні, так і подвійні лапки. Рекомендується вибрати один стиль і дотримуватися його:

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **Доступ до символів рядка**: Ви можете отримати доступ до символів рядка, використовуючи дужкову нотацію так:

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **Екранізація рядків**: Якщо рядок містить лапки, можна використовувати зворотну скісну риску (`\`) так:

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **Конкатенація рядків**: Щоб об’єднати рядки, можна використовувати оператор `+` так:

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

Інший спосіб об’єднання рядків — використання оператора `+=`, який виконує конкатенацію та присвоєння одночасно так:

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**: Це скорочення для форматованих рядкових літералів. Вони дозволяють виконувати інтерполяцію та деяку конкатенацію з компактним і зрозумілим синтаксисом:

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **Вирізання рядків (slicing)**: Це коли можна витягти частини рядка. Ось базовий синтаксис:

```py
str[start:stop:step]
```

Початкова позиція — це індекс, з якого починається вирізання. Кінцева позиція — де вирізання закінчується, ця позиція не включається. Крок — інтервал, з яким відбувається вирізання. Ось кілька прикладів:

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **Отримання довжини рядка**: Функція `len()` повертає кількість символів у рядку:

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## Робота з оператором `in`

- **Оператор `in`**: Повертає булеве значення, яке вказує, чи існує символ або символи у рядку:

```py
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

## Поширені методи рядків

- **`upper()`**: Цей метод рядка повертає новий рядок, у якому всі символи перетворені на верхній регістр:

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`lower()`**: Цей метод рядка повертає новий рядок, у якому всі символи перетворені на нижній регістр:

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`strip()`**: Цей метод повертає копію рядка з видаленими вказаними початковими та кінцевими символами (якщо аргумент не передано, видаляє пробіли на початку і в кінці).

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**: Повертає новий рядок, у якому всі входження старого рядка замінені на новий.

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**: Використовується для розбиття рядка на список за вказаним роздільником. Роздільник — це рядок, який вказує, де має відбутися розбиття.

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**: Використовується для об’єднання елементів ітерованого об’єкта у рядок із роздільником. Ітерований об’єкт — це колекція елементів, по якій можна пройтися циклом, наприклад список, рядок або кортеж.

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`startswith(prefix)`**: Цей метод рядка повертає булеве значення, яке вказує, чи починається рядок із вказаного префікса:

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`endswith(suffix)`**: Цей метод рядка повертає булеве значення, яке вказує, чи закінчується рядок вказаним суфіксом:

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`find()`**: Цей метод рядка повертає індекс першого входження підрядка. Якщо підрядок не знайдено, повертається `-1`:

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`count(substring)`**: Цей метод рядка рахує, скільки разів підрядок зустрічається у рядку:

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`capitalize()`**: Цей метод рядка повертає новий рядок, у якому перший символ великий, а всі інші — у нижньому регістрі:

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`isupper()`**: Цей метод рядка повертає `True`, якщо всі літери у рядку великі, і `False`, якщо ні:

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`islower()`**: Цей метод рядка повертає `True`, якщо всі літери у рядку малі, і `False`, якщо ні:

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`title()`**: Цей метод рядка повертає новий рядок, у якому перша літера кожного слова велика:

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`maketrans()`**: Цей метод рядка створює таблицю відповідностей символів для трансляції. Часто використовується разом із методом `translate()`, який застосовує цю таблицю до рядка і повертає трансформований результат.

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## Поширені операції з цілими числами та числами з плаваючою крапкою

- **Основні математичні операції**: У Python можна виконувати базові математичні операції з цілими числами та числами з плаваючою крапкою, включно з додаванням, відніманням, множенням і діленням:

```py
int_1 = 56
int_2 = 12
float_1 = 5.4
float_2 = 12.0

# Addition

print('Integer Addition:', int_1 + int_2) # Integer Addition: 68
print('Float Addition:', float_1 + float_2) # Float Addition: 17.4

# Subtraction

print('Int Subtraction:', int_1 - int_2) # Int Subtraction: 44
print('Float Subtraction:',  float_2 - float_1) # Float Subtraction: 6.6

# Multiplication

print('Int Multiplication:', int_1 * int_2) # Int Multiplication: 672
print('Float Multiplication:', float_2 * float_1) # Float Multiplication: 64.80000000000001

# Division

print('Division:', int_1 / int_2) # Division: 4.666666666666667
print('Float Division:', float_2 / float_1) # Float Division: 2.222222222222222
```

Якщо додати число з плаваючою крапкою і ціле число, результат буде перетворений у число з плаваючою крапкою так:

```py
int_1 = 56
float_1 = 5.4

print(int_1 + float_1) # 61.4
```

- **Оператор модуля (`%`)**: Повертає остачу від ділення одного числа на інше:

```py
int_1 = 56
int_2 = 12

print(int_1 % int_2) # 8
```

- **Цілочисельне ділення (`//`)**: Оператор, який ділить два числа і округлює результат вниз до найближчого цілого числа:

```py
int_1 = 56
int_2 = 12

print(int_1 // int_2) # 4
```

- **Оператор піднесення до степеня (`**`)**: Використовується для піднесення числа до степеня іншого числа:

```py
int_1 = 4
int_2 = 2

print(int_1 ** int_2) # 16
```

- **Функція `float()`**: Використовується для перетворення цілого числа у число з плаваючою крапкою.

```py
num = 4

print(float(num)) # 4.0
```

- **Функція `int()`**: Використовується для перетворення числа з плаваючою крапкою у ціле число.

```py
num = 4.0

print(int(num)) # 4
```

- **Функція `round()`**: Використовується для округлення числа до найближчого цілого:

```py
num_1 = 3.4
num_2 = 7.7

print(round(num_1)) # 3
print(round(num_2)) # 8
```

- **Функція `abs()`**: Використовується для отримання абсолютного значення числа:

```py
num = -13

print(abs(num)) # 13
```

- **Функція `pow()`**: Використовується для піднесення числа до степеня іншого числа:

```py
result = pow(2, 3) 
print(result)  # 8
```

## Розширені присвоєння

- **Визначення**: Розширене присвоєння поєднує бінарну операцію з присвоєнням в одному кроці. Воно бере змінну, застосовує до неї операцію з іншим значенням і зберігає результат у ту ж змінну.

```py
# Addition assignment 
my_var = 10
my_var += 5

print(my_var) # 15

# Subtraction assignment
count = 14
count -= 3

print(count) # 11

# Multiplication assignment 
product = 65
product *= 7

print(product) # 455

# Division assignment 
price = 100
price /= 4

print(price) # 25.0

# Floor Division assignment 
total_pages = 23
total_pages //= 5

print(total_pages) # 4

# Modulo assignment 
bits = 35
bits %= 2

print(bits) # 1

# Exponentiation assignment 
power = 2
power **= 3

print(power) # 8
```

Існують й інші оператори розширеного присвоєння, наприклад для побітових операцій. До них належать `&=`, `^=`, `>>=` і `<<=`.

## Робота з функціями

- **Визначення**: Функції — це повторно використовувані фрагменти коду, які приймають вхідні дані (аргументи) і повертають результат. Щоб викликати функцію, потрібно звернутися до її імені, після якого йдуть круглі дужки:

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call
print(result) # 7
```

Якщо функція явно не повертає значення, за замовчуванням вона повертає `None`:

```py
def greet():
    print('hello') 

result = greet() # hello
print(result) # None
```

Також можна задавати значення параметрів за замовчуванням так:

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

Якщо викликати функцію з неправильним числом аргументів, отримаєте `TypeError`:

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

name = input('What is your name?') # User types 'Kolade' and presses Enter  
print('Hello', name) # Hello Kolade
```

- **Функція `int()`**: Використовується для перетворення числа, булевого значення або числового рядка у ціле число:

print(int(3.14)) # 3
print(int('42')) # 42
print(int(True)) # 1
print(int(False)) # 0 
```

## Область видимості у Python

- **Локальна область видимості**: Змінна, оголошена всередині функції або класу, доступна лише в межах цієї функції або класу.

```py
def my_func():
    num = 10
    print(num)
```

- **Огортаюча область видимості**: Функція, вкладена в іншу функцію, може отримувати доступ до змінних зовнішньої функції.

```py
def outer_func():
    msg = 'Hello there!'
    
    def inner_func():
        print(msg)
    inner_func()

print(outer_func()) # Hello there!
```

- **Глобальна область видимості**: Змінні, оголошені поза будь-якими функціями чи класами, доступні з будь-якої частини програми.

```py
tax = 0.70 

def get_total(subtotal):
    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **Вбудована область видимості**: Зарезервовані імена Python для попередньо визначених функцій, модулів, ключових слів і об’єктів.

```py
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```

## Оператори порівняння

- **Рівність (`==`)**: Перевіряє, чи два значення рівні:

```py
print(3 == 4) # False
```

- **Нерівність (`!=`)**: Перевіряє, чи два значення не рівні:

```py
print(3 != 4) # True
```

- **Строго більше (`>`)**: Перевіряє, чи одне значення більше за інше:

```py
print(3 > 4) # False
```

- **Строго менше (`<`)**: Перевіряє, чи одне значення менше за інше:

```py
print(3 < 4) # True
```

- **Більше або дорівнює (`>=`)**: Перевіряє, чи одне значення більше або дорівнює іншому:

```py
print(3 >= 4) # False
```

- **Менше або дорівнює (`<=`)**: Перевіряє, чи одне значення менше або дорівнює іншому:

```py
print(3 <= 4) # True
```

## Робота з інструкціями `if`, `elif` і `else`

- **Інструкції `if`**: Умови, які визначають, чи є щось правдою. Якщо умова оцінюється як `True`, виконується відповідний блок коду.

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **Умова `elif`**: Умови, які йдуть після інструкції `if`. Якщо умова `elif` оцінюється як `True`, виконується відповідний блок коду.

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **Умова `else`**: Виконується, якщо жодна інша умова не оцінюється як `True`.

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

Також можна використовувати вкладені інструкції `if` так:

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## Правдиві та хибні значення

- **Визначення**: У Python кожне значення має вбудоване булеве значення, яке визначає, чи слід його вважати `True` або `False` у логічному контексті. Багато значень вважаються правдивими, тобто вони оцінюються як `True` у логічному контексті. Інші — хибними, тобто оцінюються як `False`. Ось кілька прикладів хибних значень:

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

Інші значення, як-от ненульові числа та непорожні рядки, є правдивими.

## Робота з функцією `bool()`

- **Визначення**: Щоб перевірити, чи є значення правдивим чи хибним, можна використовувати вбудовану функцію `bool()`. Вона явно перетворює значення у булевий еквівалент і повертає `True` для правдивих значень і `False` для хибних. Ось кілька прикладів:

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## Булеві оператори та коротке замикання

- **Визначення**: Це спеціальні оператори, які дозволяють поєднувати кілька виразів для створення складнішої логіки прийняття рішень у коді. У Python є три булеві оператори: `and`, `or` і `not`.
- **Оператор `and`**: Цей оператор приймає два операнди і повертає перший, якщо він хибний, інакше повертає другий. Обидва операнди мають бути правдивими, щоб вираз був правдивим.

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

Оператор `and` також можна використовувати в умовах так:

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **Оператор `or`**: Повертає перший операнд, якщо він правдивий, інакше другий. Вираз `or` є правдивим, якщо хоча б один операнд правдивий. Ось приклад:

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

Як і з оператором `and`, оператор `or` можна використовувати в умовах так:

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **Коротке замикання**: Оператори `and` і `or` називають операторами короткого замикання. Це означає, що Python перевіряє значення зліва направо і зупиняється, щойно визначить кінцевий результат.
- **Оператор `not`**: Приймає один операнд і інвертує його булеве значення. Перетворює правдиві значення на `False` і хибні — на `True`. На відміну від попередніх операторів, `not` завжди повертає `True` або `False`. Ось кілька прикладів:

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

Ось приклад використання оператора `not` в умові:

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```


## Списки в Python

- **Вступ**: У Python тип даних список — це впорядкована послідовність елементів, які можуть бути рядками, числами або навіть іншими списками. Списки змінні і мають індексацію, починаючи з нуля.

```python
cities = ['Los Angeles', 'London', 'Tokyo']
```

- **Доступ до елементів списку**: Щоб отримати елемент зі списку `cities`, зверніться до його індексу у послідовності:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # Los Angeles
```

- **Доступ до елементів за від’ємною індексацією**: Щоб отримати останній елемент будь-якого списку, можна використати `-1` як індекс:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # Tokyo
```

- Від’ємна індексація використовується для доступу до елементів, починаючи з кінця списку, а не з початку з індексу `0`.

- **Створення списків за допомогою конструктора `list()`**: Списки також можна створювати за допомогою конструктора `list()`. Конструктор `list()` використовується для перетворення ітерованого об’єкта у список:

```python
developer = 'Jessica'

print(list(developer)) 
# Result: ['J', 'e', 's', 's', 'i', 'c', 'a']
```

- **Знаходження довжини списку**: Для отримання довжини списку можна використовувати функцію `len()`:

```python
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

- **Змінність списків**: Списки змінні, тобто ви можете оновити будь-який елемент списку, якщо вказати дійсний індекс. Щоб оновити список за певним індексом, призначте нове значення цьому індексу:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

- **Помилка виходу за межі індексу**: Якщо вказати індекс (позитивний або від’ємний), який виходить за межі списку, отримаєте `IndexError`:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
"""
```

- **Видалення елементів зі списку**: Елементи можна видалити зі списку за допомогою ключового слова `del`:

```python
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

- **Перевірка наявності елемента у списку**: Ключове слово `in` використовується для перевірки, чи існує елемент у списку:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **Вкладені списки**: Списки можуть бути вкладені у інші списки:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

- Щоб отримати доступ до вкладеного списку, потрібно звернутися до нього за індексом `2`, оскільки списки індексуються з нуля.

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

- Щоб далі отримати доступ до другої мови з цього вкладеного списку, зверніться до неї за індексом `1`:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # Rust
```

- **Розпакування значень зі списку**: Розпакування — це техніка призначення значень зі списку новим змінним. Ось приклад розпакування списку `developer` у змінні `name`, `age` і `job` так:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer
```

- Якщо кількість змінних зліва від оператора присвоєння не співпадає з кількістю елементів у списку, отримаєте `ValueError`.

- **Збір залишкових елементів зі списку**: Щоб зібрати залишкові елементи зі списку, можна використовувати оператор зірочки (`*`) так:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer
```

- **Вирізання списків (slicing)**: Вирізання — це доступ до частини списку за допомогою оператора вирізання `:`. Щоб вирізати список, починаючи з індексу `1` і закінчуючи перед індексом `3`, використовуйте такий синтаксис:

```python
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie']
desserts[1:3] # ['Cookies', 'Ice Cream']
```

- **Кроки інтервалу**: Можна також вказати крок, який визначає, на скільки індексів збільшуватися між елементами. Ось приклад, якщо потрібно отримати список лише парних чисел за допомогою вирізання:

```python
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

## Методи списків

- **append()**: Використовується для додавання елемента в кінець списку. Ось приклад використання методу `append()` для додавання числа `6` до списку `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **Додавання списків**: Метод `append()` також можна використовувати для додавання одного списку в кінець іншого:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

- **extend()**: Використовується для додавання кількох елементів у кінець списку. Ось приклад додавання чисел `6`, `8` і `10` у кінець списку `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

- **insert()**: Використовується для вставки елемента за певним індексом у списку. Ось приклад використання методу `insert()`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

- **remove()**: Використовується для видалення елемента зі списку. Метод `remove()` видаляє лише перше входження елемента у списку:

```py
numbers = [1, 2, 3, 4, 5, 5, 5]
numbers.remove(5)

print(numbers) # [1, 2, 3, 4, 5, 5]

```

- **pop()**: Використовується для видалення конкретного елемента зі списку з одночасним його поверненням:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

- Якщо не вказати елемент для методу `pop`, буде видалено останній елемент.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

- **clear()**: Використовується для видалення всіх елементів зі списку:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

- **sort()**: Метод `sort()` використовується для сортування елементів на місці. Ось приклад сортування випадкового списку `numbers` на місці:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

- **sorted()**: Вбудована функція, яка повертає новий відсортований список замість зміни оригінального:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
print(numbers) # [19, 2, 35, 1, 67, 41]
```

- **reverse()**: Використовується для зміни порядку елементів у списку на зворотний:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **index()**: Використовується для знаходження першого індексу, де можна знайти елемент у списку:

```python
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

- Якщо елемент не знайдено за допомогою методу `index()`, результатом буде `ValueError`.

## Кортежі в Python

- **Визначення**: Кортеж — це тип даних Python, який використовується для створення впорядкованої послідовності значень. Кортежі можуть містити змішані типи даних:

```py
developer = ('Alice', 34, 'Rust Developer')
```

- Кортежі незмінні, тобто елементи кортежу не можна змінювати після створення. Якщо спробувати оновити один із елементів кортежу, отримаєте `TypeError`:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **Доступ до елементів кортежу**: Щоб отримати елемент із кортежу, використовуйте дужкову нотацію та індекс:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

- Від’ємна індексація може використовуватися для доступу до елементів, починаючи з кінця кортежу:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- Якщо вказати індекс, що перевищує або дорівнює довжині кортежу, отримаєте `IndexError`:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- Кортеж також можна створити за допомогою конструктора `tuple()`. У конструктор можна передати різні ітеровані об’єкти, такі як рядки, списки або інші кортежі.

```python
developer = 'Jessica'

print(tuple(developer)) 
# Result: ('J', 'e', 's', 's', 'i', 'c', 'a')
```

- **Перевірка наявності елемента в кортежі**: Щоб перевірити, чи є елемент у кортежі, можна використовувати ключове слово `in` так:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **Розпакування кортежів**: Елементи можна розпаковувати з кортежу так:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

- Якщо потрібно зібрати залишкові елементи з кортежу, можна використовувати оператор зірочки (`*`) так:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer
```

- **Вирізання кортежів**: Вирізання можна використовувати для отримання частини кортежу. Наприклад, елементи `pie` і `cookies` можна вирізати в окремий кортеж:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

- **Видалення елементів з кортежів**: Видалення елемента з кортежу викличе `TypeError`, оскільки кортежі незмінні:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

- **Коли використовувати кортеж замість списку?**: Якщо потрібна динамічна колекція елементів, де можна додавати, видаляти та оновлювати елементи, слід використовувати список. Якщо ви працюєте з фіксованою та незмінною колекцією даних, слід використовувати кортеж.

## Загальні методи кортежів

- **`count()`**: Використовується, щоб визначити, скільки разів елемент зустрічається в кортежі. Наприклад, ви можете перевірити, скільки разів мова `'Rust'` зустрічається в кортежі:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

- Якщо вказаний елемент у методі `count()` зовсім відсутній у кортежі, тоді повернене значення буде `0`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

- Якщо до методу `count()` не передати жодних аргументів, Python поверне `TypeError`.

- **index()**: Використовується, щоб знайти індекс, де певний елемент присутній у кортежі. Ось приклад використання методу `index()` для пошуку індексу мови `'Java'`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

- Якщо вказаний елемент не знайдено, Python поверне `ValueError`.

- Ви можете передати необов’язковий початковий індекс у метод `index()`, щоб вказати, з якого місця починати пошук елемента в кортежі:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

- Також можна передати необов’язковий кінцевий індекс у метод `index()`, щоб вказати, де припинити пошук елемента в кортежі:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

- **`sorted()`**: Використовується для сортування елементів будь-якого ітерованого об’єкта та повертає новий відсортований список. Ось приклад створення нового списку чисел за допомогою функції `sorted()`:

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

- **Зміна поведінки сортування**: Ви можете налаштувати поведінку сортування для ітерованого об’єкта, використовуючи необов’язкові аргументи `reverse` і `key`. Ось приклад використання аргументу `key` для сортування елементів кортежу за довжиною:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

- Ви можете створити новий список значень у зворотному порядку, використовуючи аргумент `reverse` так:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

## Цикли в Python

- **Визначення**: Цикли використовуються для повторення блоку коду певну кількість разів.

- **`for` цикл**: Використовується для ітерації по послідовності (наприклад, списку, кортежу або рядку) і виконання блоку коду для кожного елемента цієї послідовності. Ось приклад використання `for` циклу для ітерації по списку та виведення кожної мови в консоль:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)

"""
Result 

Rust
Java
Python
C++
"""
```

- Ось приклад використання `for` циклу для проходження по рядку `code` і виведення кожного символу:

```py
for char in 'code':
    print(char)

"""
Result 

c
o
d
e
"""
```

- `for` цикли можуть бути вкладеними. Ось приклад використання вкладеного `for` циклу:

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)

"""
Result

Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
"""

```

- **`while` цикл**: Повторює блок коду, доки умова `False` не стане істинною. Ось приклад використання `while` циклу для гри в угадайку:

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')

"""
Result

Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
"""

```

- **`break` і `continue` інструкції**: Використовуються в циклах для зміни виконання циклу.

- Інструкція `break` використовується для негайного виходу з циклу, коли виконується певна умова. Ось приклад використання інструкції `break` для списку `developer_names`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

- Інструкція `continue` використовується, щоб пропустити поточну ітерацію і перейти до наступної. Ось приклад використання інструкції `continue` замість інструкції `break`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

- Обидва цикли `for` і `while` можна поєднувати з умовою `else`, яка виконується лише тоді, коли цикл не був завершений інструкцією `break`:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

## Діапазони та їх використання в циклах

- **Функція `range()`**: Використовується для генерації послідовності цілих чисел.

```py
range(start, stop, step)
```

- Обов’язковий аргумент `stop` — це ціле число (не включно), яке представляє кінцеву точку для генерованої послідовності чисел. Ось приклад використання функції `range()`:

```py
for num in range(3):
    print(num)
```

- Якщо аргумент `start` не вказано, за замовчуванням буде `0`. За замовчуванням послідовність цілих чисел збільшується на `1`. Ви можете використати необов’язковий аргумент `step`, щоб змінити значення кроку за замовчуванням. Ось приклад генерації послідовності парних чисел від 2 до 10 включно (не включаючи 11):

```py
for num in range(2, 11, 2):
    print(num)
```

- Якщо не передати жодних аргументів функції `range()`, ви отримаєте `TypeError`.

- Функція `range()` приймає лише цілі числа як аргументи, а не числа з плаваючою комою. Використання чисел з плаваючою комою також призведе до `TypeError`:

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- Ви можете використати від’ємне ціле число для аргументу `step`, щоб згенерувати послідовність цілих чисел у спадному порядку:

```py
for num in range(40, 0, -10):
    print(num)
```

- Функцію `range()` також можна використати для створення списку цілих чисел, застосувавши її разом із конструктором `list`. Конструктор `list` використовується для перетворення ітерованого об’єкта на список. Ось приклад генерації списку парних чисел від 2 до 10 включно:

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

## Функції `enumerate()` і `zip()` у Python

- **`enumerate()`**: використовується для ітерації по послідовності з одночасним відстеженням індексу кожного елемента. Функція `enumerate()` приймає ітерований об’єкт як аргумент і повертає об’єкт `enumerate`, який складається з індексу та значення кожного елемента ітерованого об’єкта.

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')

# Result
# Index 0 and language Spanish
# Index 1 and language English
# Index 2 and language Russian
# Index 3 and language Chinese

```

- Функцію `enumerate()` також можна використовувати поза циклом `for`:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

print(list(enumerate(languages)))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

- Функція `enumerate()` також приймає необов’язковий аргумент `start`, який задає початкове значення лічильника. Якщо цей аргумент опущено, лічильник починається з `0`.

- **`zip()`**: Використовується для ітерації одночасно по кількох ітерованих об’єктах. Ось приклад використання функції `zip()` для ітерації по `developers` і `ids`:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')


"""
Result

Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
"""

```

## Генератори списків у Python

- **Визначення**: Генератор списку дозволяє створити новий список в одному рядку, поєднуючи цикл і умову безпосередньо в квадратних дужках. Це робить код коротшим і часто легшим для читання.

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

## Функції для ітерованих об’єктів

- **`filter()`**: Використовується для фільтрації елементів ітерованого об’єкта на основі умови. Повертає ітератор, який містить лише елементи, що задовольняють умову. Ось приклад створення нового списку лише зі словами довжиною більше чотирьох символів:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**: Використовується для застосування функції до кожного елемента ітерованого об’єкта і повертає новий ітерований об’єкт з результатами. Ось приклад використання функції `map()` для перетворення списку температур у Цельсіях у Фаренгейти:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**: Використовується для отримання суми з ітерованого об’єкта, наприклад списку або кортежу. Ось приклад використання функції `sum()`:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- Ви також можете передати необов’язковий аргумент `start`, який задає початкове значення для сумування. Ось оновлений приклад із використанням аргументу `start` як позиційного аргументу:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- Ви також можете використати аргумент `start` як ключовий аргумент ось так:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## Лямбда-функції

- **Визначення**: Лямбда-функція в Python — це короткий спосіб створити функцію без імені (анонімну функцію).
- Лямбда-функції часто використовують як аргумент для іншої функції. Ось приклад лямбда-функції:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

- Найкращі практики використання лямбда-функцій включають: не призначати їх змінним, тримати їх простими та читабельними, а також використовувати для коротких одноразових функцій.

## Словники

- **Словники**: Словники — це вбудовані структури даних, які зберігають колекції пар ключ-значення. Ключі мають бути незмінними типами даних. Ось загальний синтаксис словника в Python:

```python
dictionary = {
    key1: value1,
    key2: value2
}
```

- **Конструктор `dict()`**: Конструктор `dict()` — це альтернативний спосіб створення словника. Ви передаєте список кортежів як аргумент конструктору `dict()`. Ці кортежі містять ключ як перший елемент і значення як другий.

```python
pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
```

- **Дужкова нотація**: Щоб отримати значення за ключем, можна використати синтаксис, відомий як дужкова нотація.

```python
dictionary[key]
```

## Загальні методи словників

- **Метод `get()`**: Метод `get()` отримує значення, пов’язане з ключем. Це схоже на дужкову нотацію, але дозволяє встановити значення за замовчуванням, щоб уникнути помилок, якщо ключ не існує.

```python
dictionary.get(key, default)
```

- **Методи `keys()` і `values()`**: Методи `keys()` і `values()` повертають об’єкт перегляду з усіма ключами та значеннями словника відповідно. Об’єкт перегляду — це спосіб побачити вміст словника без створення окремої копії даних.

```python
pizza = {
    'name': 'Margherita Pizza',
    'price': 8.9,
    'calories_per_slice': 250
}

pizza.keys()
# dict_keys(['name', 'price', 'calories_per_slice'])

pizza.values()
# dict_values(['Margherita Pizza', 8.9, 250])
```

- **Метод `items()`**: Метод `items()` повертає об’єкт перегляду з усіма парами ключ-значення словника, включно з ключами та значеннями.

```python
pizza.items()
# dict_items([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250)])
```

- **Метод `clear()`**: Метод `clear()` видаляє всі пари ключ-значення зі словника.

```python
pizza.clear()
```

- **Метод `pop()`**: Метод `pop()` видаляє пару ключ-значення з ключем, вказаним як перший аргумент, і повертає її значення. Якщо ключ не існує, повертає значення за замовчуванням, вказане другим аргументом. Якщо ключ не існує і значення за замовчуванням не вказано, виникає `KeyError`.

```python
pizza.pop('price', 10)
pizza.pop('total_price') # KeyError
```

- **Метод `popitem()`**: У Python 3.7 і вище метод `popitem()` видаляє останній доданий елемент.

```python
pizza.popitem()
```

- **Метод `update()`**: Метод `update()` оновлює пари ключ-значення словника парами з іншого словника. Якщо ключі збігаються, їхні значення перезаписуються. Нові ключі додаються як нові пари ключ-значення.

```python
pizza.update({ 'price': 15, 'total_time': 25 })
```

## Ітерація по словнику

- **Ітерація по значеннях**: Якщо потрібно ітеруватися по значеннях словника, можна написати цикл `for` з `values()`, щоб отримати всі значення словника.

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,
    'Headphones': 70,
}

for price in products.values():
    print(price)
```

Вивід:

```md
990
600
250
70
```

- **Ітерація по ключах**: Якщо потрібно ітеруватися по ключах словника `products` вище, можна написати `products.keys()` або `products` напряму.

```python
for product in products.keys():
    print(product)
    
# Or

for product in products:
    print(product)
```

Вивід:

```md
Laptop
Smartphone
Tablet
Headphones
```

- **Ітерація по парах ключ-значення**: Якщо потрібно ітеруватися одночасно по ключах і відповідних значеннях, можна ітеруватися по `products.items()`. Ви отримуєте окремі кортежі з ключами та відповідними значеннями.

```python
for product in products.items():
    print(product)
```

Вивід:

```md
('Laptop', 990)
('Smartphone', 600)
('Tablet', 250)
('Headphones', 70)
```

Щоб зберегти ключ і значення в окремих змінних циклу, потрібно розділити їх комою. Перша змінна зберігає ключ, друга — значення.

```python
for product, price in products.items():
    print(product, price)
```

Вивід:

```md
Laptop 990
Smartphone 600
Tablet 250
Headphones 70
```

- **Функція `enumerate()`**: Якщо потрібно ітеруватися по словнику з одночасним відстеженням лічильника, можна викликати функцію `enumerate()`. Вона повертає об’єкт `enumerate`, який призначає ціле число кожному елементу, як лічильник. Лічильник можна почати з будь-якого числа, але за замовчуванням він починається з 0.

Призначення індексу і елемента в окремі змінні циклу — це загальний спосіб використання `enumerate()`. Наприклад, з `products.items()` ви можете отримати повну пару ключ-значення разом з індексом:

```python
for index, product in enumerate(products.items()):
    print(index, product)
```

Вивід:

```md
0 ('Laptop', 990)
1 ('Smartphone', 600)
2 ('Tablet', 250)
3 ('Headphones', 70)
```

Щоб налаштувати початкове значення лічильника, можна передати другий аргумент у `enumerate()`. Наприклад, тут ми починаємо лічильник з 1.

```python
for index, product in enumerate(products.items(), 1):
    print(index, product)
```

Вивід:

```md
1 ('Laptop', 990)
2 ('Smartphone', 600)
3 ('Tablet', 250)
4 ('Headphones', 70)
```

## Множини

- **Множини**: Множини — це вбудовані структури даних у Python, які не допускають дублікати значень. Множини змінні і не впорядковані, тобто їхні елементи не зберігаються у певному порядку, тому до них не можна звертатися за індексами чи ключами. Також множини можуть містити лише значення незмінних типів даних, таких як числа, рядки та кортежі.

- **Визначення множини**: Щоб визначити множину, потрібно записати її елементи в фігурних дужках, розділяючи їх комами.

```python
my_set = {1, 2, 3, 4, 5}
```

- **Визначення порожньої множини**: Якщо потрібно визначити порожню множину, слід використати функцію `set()`. Просто написання порожніх фігурних дужок автоматично створить словник.

```python
set() # Set
{}    # Dictionary
```

## Загальні методи множин

- **Метод `add()`**: Ви можете додати елемент до множини за допомогою методу `add()`, передавши новий елемент як аргумент.

```python
my_set.add(6)
```

- **Методи `remove()` і `discard()`**: Щоб видалити елемент з множини, можна використати метод `remove()` або метод `discard()`, передавши елемент, який хочете видалити, як аргумент. Метод `remove()` викличе `KeyError`, якщо елемент не знайдено, тоді як метод `discard()` — ні.

```python
my_set.remove(4)
my_set.discard(4)
```

- **Метод `clear()`**: Метод `clear()` видаляє всі елементи з множини.

```python
my_set.clear()
```

## Математичні операції над множинами

- **Методи `issubset()` і `issuperset()`**: Методи `issubset()` і `issuperset()` перевіряють, чи є множина підмножиною або надмножиною іншої множини відповідно.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 5}

print(your_set.issubset(my_set)) # True
print(my_set.issuperset(your_set)) # True
```

- **Метод `isdisjoint()`**: Метод `isdisjoint()` перевіряє, чи є дві множини диз’юнктними, тобто чи не мають вони спільних елементів.

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

print(my_set.isdisjoint(your_set)) # True
```

- **Оператор об’єднання (`|`)**: Оператор об’єднання `|` повертає нову множину з усіма елементами з обох множин.

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

my_set | your_set # {1, 2, 3, 4, 5, 6}
```

- **Оператор перетину (`&`)**: Оператор перетину `&` повертає нову множину лише з тими елементами, які є в обох множинах.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set & your_set # {2, 3, 4}
```

- **Оператор різниці (`-`)**: Оператор різниці `-` повертає нову множину з елементами першої множини, яких немає в другій.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set - your_set # {1, 5}
```

- **Оператор симетричної різниці (`^`)**: Оператор симетричної різниці `^` повертає нову множину з елементами, які є або в першій, або в другій множині, але не в обох одночасно.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set ^ your_set # {1, 5, 6}
```

- **Оператор `in`**: Ви можете перевірити, чи є елемент у множині, за допомогою оператора `in`.

```python
print(5 in my_set)
```

Python має велику стандартну бібліотеку з вбудованими модулями, які реалізують стандартизовані рішення для багатьох задач. Деякі популярні вбудовані модулі — це `math`, `random`, `re` (скорочення від "регулярні вирази") і `datetime`.

## Інструкція імпорту

- **Інструкція імпорту**: Щоб отримати доступ до елементів, визначених у вбудованих модулях, використовуйте інструкцію імпорту. Зазвичай інструкції імпорту пишуться на початку файлу. Вони працюють однаково для функцій, класів, констант, змінних та інших елементів модуля.

- **Базова інструкція імпорту**: Ви можете використати ключове слово `import`, за яким йде ім’я модуля:

```python
import module_name
```

Потім, якщо потрібно викликати функцію з цього модуля, використовуйте точкову нотацію: ім’я модуля, потім ім’я функції.

```python
module_name.function_name()
```

Наприклад, у коді ви напишете так, щоб імпортувати модуль `math` і отримати квадратний корінь з 36:

```python
import math

math.sqrt(36)
```

- **Імпорт модуля з іншим ім’ям**: Якщо потрібно імпортувати модуль під іншим ім’ям (так званий "аліас"), можна використати `as` з аліасом наприкінці інструкції імпорту. Це часто застосовується для довгих імен модулів або щоб уникнути конфліктів імен.

```python
import module_name as module_alias
```

Наприклад, щоб звертатися до модуля `math` як `m` у коді, можна призначити аліас так:

```python
import math as m
```

Потім можна звертатися до елементів модуля через аліас:

```python
m.sqrt(36)
```

- **Імпорт конкретних елементів**: Якщо вам не потрібні всі елементи модуля, можна імпортувати конкретні за допомогою `from`. У цьому випадку інструкція імпорту починається з `from`, потім ім’я модуля, далі ключове слово `import` і, нарешті, імена елементів, які хочете імпортувати.

```python
from module_name import name1, name2
```

Потім у вашому скрипті Python можна використовувати ці імена без префікса модуля. Наприклад:

```python
from math import radians, sin, cos

angle_degrees = 40
angle_radians = radians(angle_degrees)

sine_value = sin(angle_radians)
cos_value = cos(angle_radians)

print(sine_value) # 0.6427876096865393
print(cos_value)  # 0.766044443118978
```

Це зручно, але може призвести до конфліктів імен, якщо у вас вже є функції або змінні з такими ж іменами. Враховуйте це при виборі типу інструкції імпорту.

Якщо потрібно призначити аліаси цим іменам, це також можна зробити, використовуючи ключове слово `as` з аліасом.

```python
from module_name import name1 as alias1, name2 as alias2
```

- **Інструкція імпорту з зірочкою (`*`)**: Зірочка каже Python, що ви хочете імпортувати все з модуля, але без префікса імені модуля.

```python
from module_name import *
```

Наприклад, якщо ви імпортуєте модуль `math` таким чином, зможете викликати будь-яку функцію з цього модуля без вказування імені модуля.

```python
from math import *
print(sqrt(36))  # 6.0
```

Однак це зазвичай не рекомендується, бо може призвести до конфліктів імен і ускладнити розуміння, звідки беруться імена.

## `if __name__ == '__main__'`

- **Змінна `__name__`**: `__name__` — це спеціальна вбудована змінна в Python. Коли файл Python виконується напряму, Python встановлює значення цієї змінної в рядок `"__main__"`. Але якщо файл імпортується як модуль в інший скрипт Python, значення змінної `__name__` встановлюється як ім’я цього модуля.

Саме тому ви часто побачите таку умову в скриптах Python. Вона містить код, який ви хочете виконувати **тільки** якщо скрипт запускається як головна програма.

```python
if __name__ == '__main__': 
    # Code
```

## Загальні помилки в Python

- **SyntaxError**: Помилка, яку викликає Python, коли ваш код не відповідає його правилам синтаксису. Наприклад, код `print("Hello there"` призведе до синтаксичної помилки з повідомленням `SyntaxError: '(' was never closed`, бо у коді бракує закриваючої дужки.
- **NameError**: Python викликає `NameError`, коли ви намагаєтеся звернутися до змінної або функції, яку не визначили. Наприклад, якщо у вашому коді є рядок `print(username)` без попереднього визначення змінної `username`, ви отримаєте помилку імені з повідомленням `NameError: name 'username' is not defined`.
- **TypeError**: Цю помилку Python викликає, коли ви виконуєте операцію над двома або більше несумісними типами даних. Наприклад, якщо ви спробуєте додати рядок до числа, отримаєте помилку `TypeError: can only concatenate str (not "int") to str`.
- **IndexError**: Ви отримаєте `IndexError`, якщо звернетеся до індексу, якого немає у списку або інших послідовностях, як-от кортеж чи рядок. Наприклад, у рядку `Hello world` індекс останнього символу — `10`. Якщо ви спробуєте звернутися до символу таким чином `greet = "hello world"; print(greet[11])`, отримаєте помилку з повідомленням `IndexError: string index out of range`.
- **AttributeError**: Python викликає цю помилку, коли ви намагаєтеся використати метод або властивість, якої немає в об’єкті такого типу. Наприклад, виклик `.append()` для рядка `"hello".append("!")` призведе до помилки з повідомленням `AttributeError: 'str' object has no attribute 'append'`.

За допомогою `pdb` можна встановити трасування функцією `set_trace()`, щоб почати покрокове виконання коду і перевірку змінних у інтерактивному режимі.
- **Використання інструментів налагодження IDE**: Багато інтегрованих середовищ розробки (IDE) і редакторів коду, як Pycharm і VS Code, пропонують інструменти налагодження з точками зупинки, покроковим виконанням, перевіркою змінних та іншими функціями.

## Обробка виключень

- **`try...except`**: Використовується для виконання блоку коду, який може викликати виключення. Блок `try` — це місце, де ви очікуєте, що може виникнути помилка, а блок `except` обробляє вказане виключення, якщо воно виникає. Ось приклад:

  ```py
  try:
    print(22 / 0)
  except ZeroDivisionError:
    print('You can\'t divide by zero!')
    # You can't divide by zero!
  ```

  Ви також можете ланцюжити кілька блоків `except`, щоб обробляти різні типи виключень:

  ```py
  try:
    number = int(input('Enter a number: '))
    print(22 / number)
  except ZeroDivisionError:
    print('You cannot divide by zero!')
    # You cannot divide by zero! prints when you enter 0
  except ValueError:
    print('Please enter a valid number!')
    # Please enter a valid number! prints when you don't enter an integer
  ```

- **`else` і `finally`**: Ці блоки розширюють `try...except`. Якщо виключення не виникає, виконується блок `else`. Блок `finally` виконується завжди, незалежно від помилок.

  ```py
  try:
    result = 100 / 4
  except ZeroDivisionError:
    print('You cannot divide by zero!') # This will not run
  else:
    print(f'Result is {result}') # Result is 25.0
  finally:
    print('Execution complete!') # Execution complete!
  ```

- **Об’єкт виключення**: Дозволяє отримати доступ до самого виключення для кращого налагодження і виведення прямого повідомлення про помилку. Щоб отримати об’єкт виключення, потрібно використати ключове слово `as`. Ось приклад:

  ```py
  try:
      value = int('This will raise an error')
  except ValueError as e:
      print(f'Caught an error: {e}')
      # Caught an error: invalid literal for int() with base 10: 'This will raise an error'
  ```

- **Інструкція `raise`**: Дозволяє вручну викликати виключення. Її можна використати, щоб кинути виключення, коли виконується певна умова. Ось приклад:

  ```py
  def divide(a, b):
      if b == 0:
          raise ZeroDivisionError('You cannot divide by zero')
      return a / b
  ```
  
## Сигналізація виключень
  
  Інструкція `raise` також корисна, коли ви створюєте власні кастомні виключення, бо дозволяє кинути виключення з власним повідомленням. Ось приклад:

  ```py
  class InvalidCredentialsError(Exception):
      def __init__(self, message="Invalid username or password"):
          self.message = message
          super().__init__(self.message)

  def login(username, password):
      stored_username = "admin"
      stored_password = "password123"
      
      if username != stored_username or password != stored_password:
          raise InvalidCredentialsError()
      
      return f"Welcome, {username}!"
  ```  

  Ось як можна використати функцію `login` з виключенням `InvalidCredentialsError`:

  ```py
  # failed login attempt
  try:
      message = login("user", "wrongpassword")
  except InvalidCredentialsError as e:
      print(f"Login failed: {e}")
  else:
      print(message)

  # successful login attempt
  try:
      message = login("admin", "password123")
  except InvalidCredentialsError as e:
      # This block is not executed because the login was successful
      print(f"Login failed: {e}")
  else:
      # The else block runs if the 'try' block completes without an exception
      print(message)
  ```

  Інструкцію `raise` також можна використовувати з ключовим словом `from`, щоб ланцюжити виключення, показуючи зв’язок між різними помилками:

  ```py
  def parse_config(filename):
    try:
        with open(filename, 'r') as file:
            data = file.read()
            return int(data)
    except FileNotFoundError:
        raise ValueError('Configuration file is missing') from None
    except ValueError as e:
        raise ValueError('Invalid configuration format') from e

  config = parse_config('config.txt')
  ```

## Класи та об’єкти в Python

- **Визначення класу**: Клас — це шаблон для створення об’єктів. Він визначає поведінку об’єкта через його атрибути та методи. Ось базовий приклад визначення класу в Python:

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f'{self.name.upper()} says woof woof!')
```

- **Створення об’єктів**: Об’єкти — це екземпляри класу. Вони створюються викликом класу з необхідними аргументами.

```python
dog1 = Dog('Jack', 3)
dog2 = Dog('Thatcher', 5)

dog1.bark()  # JACK says woof woof!
dog2.bark()  # THATCHER says woof woof!
```

- **Виклик методів об’єктів**: Ви можете викликати методи об’єктів, щоб виконувати дії або отримувати інформацію.

```python
object_name1.method_name()
object_name2.method_name()
```

- **Відмінність між класом і об’єктом**: Клас — це багаторазовий шаблон, а об’єкт — конкретний екземпляр цього класу з реальними даними.

## Атрибути

- **Атрибути екземпляра**: Визначаються в `__init__()` за допомогою `self` і є унікальними для кожного об’єкта.
- **Атрибути класу**: Визначаються безпосередньо всередині класу і спільні для всіх екземплярів.

```python
class Dog:
    species = 'French Bulldog'  # Class attribute

    def __init__(self, name):
        self.name = name  # Instance attribute

print(Dog.species) # French Bulldog

jack = Dog('Jack')
print(jack.name)     # Jack
print(jack.species)  # French Bulldog
```

## Методи

- **Методи**: Функції, визначені всередині класу, які працюють з атрибутами об’єкта.

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
print(my_car_1.describe())  # This car is a red Tesla Model S
```

- **Доступ до методів**: Викликайте методи об’єктів за допомогою точкової нотації. Ось приклад виклику методу `describe` на двох різних об’єктах car:

```python
class Car:
    def __init__(self, color, model):
        self.color = color  
        self.model = model  

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
my_car_2 = Car('green', 'Lamborghini Revuelto')

print(my_car_1.describe()) # Calling method using the dot notation

print(my_car_2.describe()) # Calling method using the dot notation
```

## Dunder (магічні) методи

- **Визначення**: Спеціальні методи, які починаються і закінчуються подвійним підкресленням (наприклад, `__init__`, `__len__`, `__str__`, `__eq__`). Python використовує їх внутрішньо для вбудованих операцій.

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **Непрямий виклик dunder-методів**: Вам не потрібно викликати dunder-методи напряму. Замість цього Python автоматично викликає їх, коли відбуваються певні дії. Ці операції включають:

  - **арифметичні дії, такі як додавання, віднімання, множення, ділення та інші**. `__add__()` викликається для додавання, `__sub__()` для віднімання, `__mul__()` для множення, а `__truediv__()` для ділення.
        
  - **операції з рядками, такі як конкатенація, повторення, форматування та перетворення в текст**. `__add__()` викликається для конкатенації, `__mul__()` для повторення, `__format__()` для форматування, `__str__()` і `__repr__()` для перетворення в текст тощо.
        
  - **операції порівняння, такі як рівність, менше, більше та інші**. `__eq__()` викликається для перевірки рівності, `__lt__()` для менше, `__gt__()` для більше тощо.
        
  - **операції ітерації, такі як зробити об’єкт ітерованим і переходити по елементах**. `__iter__()` викликається для повернення ітератора, а `__next__()` — для отримання наступного елемента.

## Приклад із реального життя: кошик для покупок

- **Клас Cart з dunder-методами**: Дозволяє додавати, видаляти, ітерувати та перевіряти вміст із вбудованою поведінкою.

```python
class Cart:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

    def remove(self, item):
        if item in self.items:
            self.items.remove(item)
        else:
            print(f'{item} is not in cart')

    def list_items(self):
        return self.items

    def __len__(self):
        return len(self.items)

    def __getitem__(self, index):
        return self.items[index]

    def __contains__(self, item):
        return item in self.items

    def __iter__(self):
        return iter(self.items)

cart = Cart()
cart.add('Laptop')
print(len(cart))        # 1
print('Laptop' in cart) # True
```

## Що таке об’єктно-орієнтоване програмування?

- **Об’єктно-орієнтоване програмування**: Стиль програмування, у якому розробники розглядають усе у своєму коді як реальні об’єкти. Його часто називають OOP. Чотири ключові принципи, які допомагають організувати та ефективно керувати кодом, — це **інкапсуляція**, **успадкування**, **поліморфізм** та **абстракція**.
- **Класи**: Шаблон для створення об’єктів. Кожен об’єкт, створений із класу, має атрибути, які визначають дані, і методи, які визначають поведінку об’єкта.

## Що таке інкапсуляція?

- **Інкапсуляція**: Об’єднання атрибутів і методів об’єкта в єдину одиницю. Вона дозволяє приховати внутрішній стан об’єкта за простим набором публічних методів і атрибутів, які діють як двері. За цими дверима — приватні атрибути і методи, які контролюють, як змінюються дані і хто може їх бачити.
- **Приклад інкапсуляції**: Якщо ви хочете відстежувати баланс гаманця, ви дозволите внесення і зняття коштів, але не дозволите нікому змінювати сам баланс гаманця:

```py
class Wallet:
   def __init__(self, balance):
       self.__balance = balance # Private attribute

   def deposit(self, amount):
       if amount > 0:
           self.__balance += amount # Add to the balance safely

   def withdraw(self, amount):
       if 0 < amount <= self.__balance:
           self.__balance -= amount # Remove from the balance safely

account = Wallet(500)
print(account.__balance) # AttributeError: 'Wallet' object has no attribute '__balance'
```

- **Відмінність між префіксом атрибутів з одним і двома підкресленнями**: Префікс з одним підкресленням означає, що атрибути і методи призначені для внутрішнього використання. Це конвенція, яка не забороняє доступ до атрибутів ззовні. Префікс з двома підкресленнями ефективно забороняє доступ до них ззовні класу.

## Що таке геттери і сеттери?

- **Геттери і сеттери**: Методи, які дозволяють контролювати, як отримують і змінюють атрибути класу. Ви отримуєте значення за допомогою геттерів і встановлюєте значення за допомогою сеттерів.
- **Властивості**: Вони поєднують геттери і сеттери та дозволяють отримувати доступ до даних. Вони виконують додаткову логіку за лаштунками, коли ви отримуєте, встановлюєте або видаляєте значення.
- **Чому властивості замість методів**: Властивості використовують замість методів для кращої читабельності і чистішого коду. Вони дозволяють отримувати доступ до значень через точкову нотацію, як до звичайних атрибутів, без дужок.
- **Створення геттера**: Щоб створити геттер, використовуйте декоратор `@property`. Ось геттер, який отримує радіус кола:

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self): # A getter to get the radius
        return self._radius
  
    @property
    def area(self):  # A getter to calculate area
        return 3.14 * (self._radius ** 2)

my_circle = Circle(3)

print(my_circle.radius) # 3
print(my_circle.area) # 28.26
```

Використання self.radius всередині `__init__()` гарантує, що сеттер виконається під час створення об’єкта, тому недійсні значення радіуса будуть відразу виявлені.

- **Створення сеттера**: Щоб створити сеттер, який встановлюватиме радіус, потрібно визначити інший метод з таким самим ім’ям і використати `@<property_name>.setter` над ним:

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self):  # A getter to get the radius
        return self._radius

    @radius.setter
    def radius(self, value):  # A setter to set the radius
        if value <= 0:
            raise ValueError('Radius must be positive')
        self._radius = value

my_circle = Circle(3)
print('Initial radius:', my_circle.radius) # Initial radius: 3

my_circle.radius = 8
print('After modifying the radius:', my_circle.radius) # After modifying the radius: 8
```

- **Як Python обробляє геттери і сеттери**: Після визначення геттерів і сеттерів Python автоматично викликає їх під капотом щоразу, коли ви використовуєте звичайний синтаксис атрибутів ось так:

```py
my_circle.radius # This will call the getter
my_circle.radius = 4 # This will call the setter
```

Всередині сеттера не слід призначати значення властивості безпосередньо, бо це спричинить `RecursionError`.

- **Делетер**: Після встановлення і отримання значення за допомогою сеттера і геттера ви можете контролювати, як воно видаляється, за допомогою `deleter`. Делетер виконує кастомну логіку, коли ви використовуєте інструкцію `del` для властивості. Щоб створити делетер, використовуйте декоратор `@<property_name>.deleter`.

```py
  # Deleter
    @radius.deleter
    def radius(self):
        print("Deleting radius...")
        del self._radius
```

## Що таке успадкування і як воно сприяє повторному використанню коду?

- **Успадкування**: Процес, за якого дочірній клас використовує атрибути і методи батьківського класу. Успадкування сприяє повторному використанню коду, забезпечує чітку ієрархію і дозволяє налаштовувати поведінку без переписування всього. Щоб реалізувати успадкування, дочірній клас приймає ім’я батьківського класу:

```py
class Parent:
    # Parent attributes and methods

class Child(Parent):
    # Child inherits, extends, and/or overrides where necessary
```

- **Одинарне і множинне успадкування**: Коли дочірній клас успадковує властивості і методи від одного батька, як показано вище, це називається **одинарним успадкуванням**. Коли дочірній клас успадковує властивості і методи від кількох батьків, це **множинне успадкування**. Ось синтаксис для цього:

```py
class Parent:
    # Attributes and methods for Parent

class Child:
    # Attributes and methods for Child

class GrandChild(Parent, Child):
    # GrandChild inherits from both Parent and Child
    # GrandChild can combine or override behavior from each
```

- **Функція `super()`**: Функція, яка дозволяє **викликати** метод батьківського класу, коли клас має іншу реалізацію цього методу або розширює його, без дублювання коду.

## Що таке поліморфізм і як він сприяє повторному використанню коду?

- **Поліморфізм**: Принцип ООП, який дозволяє різним класам використовувати однакову назву методу, але кожен клас реалізує його по-своєму при виклику. Ось синтаксис для цього:

```py
class A:
   def action(self): ...

class B:
   def action(self): ...

class C:
   def action(self): ...

Class().method()  # Works for A, B, or C
```

- **Поліморфізм на основі успадкування**: Батьківський клас задає метод, а кожен дочірній клас адаптує його під свої потреби.

## Що таке name mangling і як він працює?

- **Name mangling**: Процес, у якому Python внутрішньо перейменовує атрибут з префіксом з подвійним підкресленням, додаючи підкреслення і ім’я класу як префікс, перетворюючи `__attribute` у `_ClassName__attribute`.
- **Мета name mangling**: Основна мета — запобігти випадковому перекриттю атрибутів і методів при використанні успадкування. Ось код, який допоможе краще це зрозуміти:

```py
class Parent:
    def __init__(self):
        self.__data = 'Parent data'

class Child(Parent):
    def __init__(self):
        super().__init__()
        self.__data = 'Child data'

c = Child()
print(c.__dict__) # {'_Parent__data': 'Parent data', '_Child__data': 'Child data'}
```

## Що таке абстракція і як вона допомагає тримати складні системи організованими?

- **Абстракція**: Концепція програмування, у якій складні деталі реалізації об’єкта або системи приховані, а показані лише суттєві функції. В Python та інших мовах абстракція спрощує складні системи, підвищуючи повторне використання.
- **Приклад абстракції**: Хороший приклад абстракції в повсякденному житті — це автомобіль, який дозволяє просто користуватися кермом, педалями і важелем перемикання, не знаючи, як працює двигун або гальма.
- **Як Python реалізує абстракцію**: Python реалізує абстракцію через модуль `abc`. Модуль надає клас `ABC` (абстрактний базовий клас) і декоратор `@abstractmethod`. Абстрактний базовий клас (ABC) визначає спільні методи і властивості, які мають реалізувати підкласи. Його не можна інстанціювати.
- **Як визначається абстрактний метод**: Абстрактний метод визначається за допомогою `@abstractmethod` і має бути перевизначений у підкласах, навіть якщо має стандартну реалізацію. Базовий синтаксис абстракції виглядає так:

```py
from abc import ABC, abstractmethod

# Define an abstract base class
class AbstractClass(ABC):
    @abstractmethod
    def abstract_method(self):
        pass

# Concrete subclass that implements the abstract method
class ConcreteClassOne(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassOne')

# Another concrete subclass
class ConcreteClassTwo(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassTwo')
```

## Алгоритми та нотація Big O

- **Алгоритми**: Набір однозначних інструкцій для розв’язання задачі або виконання завдання. Алгоритми мають завершуватися за скінченну кількість кроків, і кожен крок має бути точним і однозначним.

- **Нотація Big O**: Описує найгірший випадок продуктивності або швидкість зростання алгоритму залежно від розміру вхідних даних. Вона фокусується на тому, як зростає використання ресурсів із збільшенням розміру вхідних даних, ігноруючи константні множники та члени нижчого порядку.

### Загальні часові складності

- **O(1) — константний час**: Алгоритм виконується за однаковий час незалежно від розміру вхідних даних.

```python
def check_even_or_odd(number):
    if number % 2 == 0:
        return 'Even'
    else:
        return 'Odd'
```

- **O(log n) — логарифмічний час**: Час зростає повільно зі збільшенням вхідних даних. Зустрічається в алгоритмах, які багаторазово зменшують розмір задачі (наприклад, бінарний пошук).

- **O(n) — лінійний час**: Час виконання пропорційний розміру вхідних даних.

```python
for grade in grades:
    print(grade)
```

- **O(n log n) — лог-лінійний час**: Типова складність ефективних алгоритмів сортування, таких як merge sort і quick sort.

- **O(n²) — квадратичний час**: Час виконання зростає квадратично. Часто зустрічається у вкладених циклах.

```python
for i in range(n):
    for j in range(n):
        print("Hello, World!")
```

### Просторова складність

- **O(1) — константний простір**: Алгоритм використовує однакову кількість пам’яті незалежно від розміру вхідних даних.
- **O(n) — лінійний простір**: Використання пам’яті зростає пропорційно розміру вхідних даних.
- **O(n²) — квадратичний простір**: Використання пам’яті зростає квадратично з розміром вхідних даних.

## Техніки розв’язання задач

- **Розуміння задачі**: Прочитайте умову задачі кілька разів. Визначте вхідні дані, очікуваний результат і як перетворити вхідні дані у вихідні.

- **Псевдокод**: Опис логіки алгоритму на високому рівні, незалежний від мови програмування. Використовує загальновживану мову, змішану з конструкціями програмування, такими як `IF`, `ELSE`, `FOR`, `WHILE`.

```md
GET original_string
SET reversed_string = ""
FOR EACH character IN original_string:
  ADD character TO THE BEGINNING OF reversed_string
DISPLAY reversed_string
```

- **Крайні випадки**: Конкретні, дійсні вхідні дані, які трапляються на межах того, що алгоритм повинен обробляти. Завжди враховуйте і тестуйте крайні випадки.

## Масиви

- **Статичні масиви**: Мають фіксований розмір, визначений під час ініціалізації. Елементи зберігаються у суміжних ділянках пам’яті. Розмір не можна змінити під час виконання програми.

- **Динамічні масиви**: Можуть автоматично збільшуватися або зменшуватися під час виконання програми. Зміна розміру відбувається через автоматичне копіювання у більші масиви за потреби.

### Списки Python (динамічні масиви)

```python
numbers = [3, 4, 5, 6]

# Access elements
numbers[0]  # 3

# Update elements
numbers[2] = 16

# Add elements
numbers.append(7)
numbers.insert(3, 15)  # Insert at specific index

# Remove elements
numbers.pop(2)  # Remove at specific index
numbers.pop()   # Remove last element
```

### Часові складності для динамічних масивів

- **Доступ**: O(1)
- **Вставка в кінець**: O(1) у середньому, O(n) при зміні розміру
- **Вставка посередині**: O(n)
- **Видалення**: O(n) посередині, O(1) в кінці

## Стек

- **Стек**: Структура даних типу LIFO (останній зайшов — перший вийшов). Елементи додаються і видаляються лише з вершини.

- **Операція push**: Додавання елемента на вершину стеку. Часова складність: O(1).

- **Операція pop**: Видалення елемента з вершини стеку. Часова складність: O(1).

```python
# Using Python list as stack
stack = []

# Push operations
stack.append(1)
stack.append(2)
stack.append(3)

# Pop operations
top_element = stack.pop()  # Returns 3
```

## Черги

- **Черги**: Структура даних типу FIFO (перший зайшов — перший вийшов). Елементи додаються в кінець і видаляються з початку.

- **Операція enqueue**: Додавання елемента в кінець черги. Часова складність: O(1).

- **Операція dequeue**: Видалення елемента з початку черги. Часова складність: O(1).

```python
from collections import deque

# Using deque for efficient queue operations
queue = deque()

# Enqueue operations
queue.append(1)
queue.append(2)
queue.append(3)

# Dequeue operations
first_element = queue.popleft()  # Returns 1
```

## Зв’язані списки

- **Зв’язані списки**: Лінійна структура даних, де кожен вузол містить дані і посилання на наступний вузол. Вузли з’єднані як ланцюг.

### Однозв’язні списки

- **Структура**: Кожен вузол має дані і одне посилання на наступний вузол.
- **Перебір**: Можна рухатися лише вперед від голови до хвоста.
- **Головний вузол**: Перший вузол у списку, зазвичай єдиний безпосередньо доступний вузол.
- **Хвостовий вузол**: Останній вузол у списку, вказує на `None`.

### Операції та часові складності

- **Вставка на початок**: O(1)
- **Вставка в кінець**: O(n) — потрібно пройти до кінця
- **Вставка посередині**: O(n) — потрібно пройти до позиції
- **Видалення з початку**: O(1)
- **Видалення з кінця**: O(n) — потрібно знайти попередній вузол
- **Видалення посередині**: O(n) — потрібно знайти вузол

### Двозв’язні списки

- **Структура**: Кожен вузол має дані і два посилання: на наступний і попередній вузол.
- **Перебір**: Можна рухатися в обох напрямках.
- **Пам’ять**: Потребує більше пам’яті, ніж однозв’язні списки через додаткове посилання.

## Хеш-мапи та множини

### Мапи та хеш-мапи

- **Мапа (абстрактний тип даних)**: Керує колекціями пар ключ-значення. Кожен ключ має бути унікальним, але значення можуть повторюватися.

- **Хеш-мапа**: Конкретна реалізація мапи з використанням хешування. Використовує хеш-функцію для генерації хеш-значень ключів, які визначають місце зберігання у внутрішньому масиві.

### Словники Python (хеш-мапи)

```python
# Creating dictionaries
my_dictionary = {
    "A": 1,
    "B": 2, 
    "C": 3
}

# Alternative creation
my_dictionary = dict(A=1, B=2, C=3)

# Access and modify
value = my_dictionary["A"]  # 1
my_dictionary["A"] = 4      # Update value
del my_dictionary["A"]      # Remove key-value pair

# Check membership
"C" in my_dictionary

# Get keys, values, items
my_dictionary.keys()
my_dictionary.values()
my_dictionary.items()
```

### Часові складності для хеш-мап

- **У середньому**: O(1) для вставки, отримання, видалення
- **У найгіршому випадку**: O(n) при великій кількості колізій

### Множини

- **Множини**: Неупорядковані колекції унікальних елементів. Дублікати не допускаються, порядок не зберігається.

- **Тільки незмінні елементи**: Множини можуть містити лише незмінні типи даних (числа, рядки, кортежі), оскільки хеш-значення мають залишатися сталими.

```python
# Creating sets
numbers = {1, 2, 3, 4}
empty_set = set()  # Must use set(), not {}

# Add and remove elements
numbers.add(5)
numbers.remove(4)      # Raises KeyError if not found
numbers.discard(4)     # No error if not found

# Set operations
set_a = {1, 2, 3, 4}
set_b = {2, 3, 4, 5, 6}

# Union, intersection, difference, symmetric difference
set_a.union(set_b)                    # or set_a | set_b
set_a.intersection(set_b)             # or set_a & set_b
set_a.difference(set_b)               # or set_a - set_b
set_a.symmetric_difference(set_b)     # or set_a ^ set_b

# Subset and superset checks
set_a.issubset(set_b)
set_a.issuperset(set_b)
set_a.isdisjoint(set_b)

# Membership testing
5 in numbers
```

### Часові складності для множин

- **У середньому**: O(1) для додавання, видалення, перевірки належності
- **У найгіршому випадку**: O(n) через колізії хешів

## Колізії хешів

- **Колізія хешів**: Виникає, коли два різні ключі дають однакове хеш-значення.

- **Стратегії розв’язання колізій**:
  - **Зв’язування (chaining)**: Кожен індекс масиву вказує на зв’язаний список, що зберігає всі елементи з однаковим хеш-значенням.
  - **Відкрите адресування (open addressing)**: Пошук наступного доступного індексу за заздалегідь визначеною послідовністю.

## Коли використовувати кожну структуру даних

- **Списки**: Коли потрібен впорядкований, індексований доступ і розмір невідомий заздалегідь.
- **Стек**: Для операцій LIFO (функція скасування, оцінка виразів, повернення назад).
- **Черги**: Для операцій FIFO (планування завдань, пошук у ширину).
- **Зв’язані списки**: Коли часто вставляють/видаляють на початку, розмір невідомий, випадковий доступ не потрібен.
- **Хеш-мапи**: Для швидкого пошуку за ключем, підрахунку, кешування.
- **Множини**: Для перевірки унікальності, математичних операцій над множинами, видалення дублікатів.

## Алгоритми пошуку

Алгоритми пошуку дозволяють шукати цільовий елемент у певному списку елементів.

У комп’ютерних науках існують два алгоритми пошуку, які ви повинні знати: **лінійний пошук** і **бінарний пошук**. Важливо розуміти відмінності між цими алгоритмами і коли використовувати кожен.

### Лінійний пошук

- Лінійний пошук ітерує список елементів, перевіряючи кожен елемент з початку, доки не знайде цільовий.
- Якщо ціль знайдена, повертається індекс, де вона розташована у списку.
- Якщо ціль не знайдена, повертається `-1`, що означає **недійсний індекс** у більшості мов програмування.
- Оскільки лінійний пошук перевіряє кожен елемент, він неефективний для великих списків.
- Часова складність лінійного пошуку — `O(n)`, бо час пошуку зростає лінійно з розміром списку.
- Просторова складність лінійного пошуку — `O(1)`, бо додатковий простір не потрібен.

### Бінарний пошук

- Бінарний пошук працює, ділячи список навпіл і перевіряючи, чи знаходиться ціль у середині списку.
- Умова для роботи бінарного пошуку — елементи списку мають бути впорядковані за зростанням.
- Бінарний пошук ефективніший для великих списків, бо ділить список навпіл і ігнорує половину, де ціль не знайдена.
- Якщо ціль знайдена в середині списку, повертається індекс цілі.
- Якщо ні, алгоритм перевіряє, чи ціль у лівій чи правій половині списку.
- Він продовжує ділити залишки списку навпіл, доки ціль не буде знайдена.
- Якщо ціль так і не знайдена, повертається `-1`.
- Часова складність бінарного пошуку — `O(log n)`, бо час пошуку зростає логарифмічно з розміром списку.
- Просторова складність бінарного пошуку — `O(1)`, бо додатковий простір не потрібен.

### Відмінності лінійного пошуку від бінарного

- Бінарний пошук краще підходить для великих списків, ніж лінійний.
- Часова складність лінійного пошуку — `O(n)`, бо час зростає лінійно з розміром списку.
- Часова складність бінарного пошуку — `O(log n)`, бо час зростає логарифмічно з розміром списку.

## Алгоритми сортування та метод розділяй і володарюй

У комп’ютерних науках метод розділяй і володарюй використовується для розбиття задачі на менші підзадачі, які легше розв’язати. Рекурсія часто застосовується в цьому методі, і він є потужною стратегією для реалізації багатьох ефективних алгоритмів сортування, таких як merge sort.

### Merge sort

- Merge sort — це алгоритм сортування, який слідує підходу розділяй і володарюй.
- Він працює, рекурсивно ділячи список на менші підсписки, доки кожен підсписок не містить одного елемента.
- Потім підсписки послідовно об’єднуються назад у відсортованому порядку.
- Часова складність merge sort — `O(n log n)`, бо список постійно ділиться навпіл `(log n)` і потім об’єднується `(O(n))`.
- Просторова складність merge sort — `O(n)`, бо це не in-place алгоритм сортування.

## Огляд графів

Граф — це множина вузлів (вершин), з’єднаних ребрами (зв’язками). Кожен вузол може бути з’єднаний з кількома іншими вузлами, утворюючи мережу. Різновиди графів включають:

- Орієнтовані: ребра мають напрямок (від одного вузла до іншого), часто зображуються прямими лініями зі стрілками.
- Неорієнтовані: ребра не мають напрямку, зображуються простими лініями.
- Вершина: кожен вузол має мітку або ідентифікатор.
- Циклічні: містять цикли (шлях, що починається і закінчується в одному вузлі).
- Ациклічні (DAG): не містять циклів.
- З мітками ребер: кожне ребро має мітку, зазвичай намальовану поруч.
- З вагами: ребра мають ваги (значення), які можна використовувати для арифметичних операцій.
- Роз’єднані: містять два або більше вузлів, які не з’єднані ребрами.

Графи використовуються в різних застосунках, таких як карти, мережі, системи рекомендацій, розв’язання залежностей.

## Обхід графів

Це процес відвідування всіх вузлів графа. Основні алгоритми:

- **Пошук у ширину (BFS)**
  - Використовує чергу.
  - Досліджує рівень за рівнем.
  - Знаходить найкоротший шлях у неважених графах.

- **Пошук у глибину (DFS)**
  - Використовує стек (або рекурсію).
  - Повністю досліджує гілку перед поверненням назад.
  - Корисний для виявлення циклів і пошуку шляхів.

## Представлення графів

Графи можна представити двома основними способами:

- **Список суміжності**
  - Кожен вузол має список своїх сусідів.
  - Економить пам’ять для розріджених графів.
  - Легко ітерувати сусідів.

- **Матриця суміжності**
  - Двовимірний масив, де рядки і стовпці представляють вузли.
  - Вимагає багато пам’яті для великих графів.
  - Швидко перевіряє наявність ребра між двома вузлами.

## Дерева

Дерево — це особливий тип графа, який є ациклічним і зв’язаним. Основні властивості:

- Не містять циклів або петель (шляхів, де початок і кінець однакові).
- Має бути зв’язаним (кожен вузол доступний з будь-якого іншого).

### Поширені типи дерев

Найпоширеніші типи дерев:

- Двійкові дерева
  - Кожен вузол має не більше двох дочірніх: лівого і правого.

- Двійкові дерева пошуку (BST)
  - Двійкове дерево, у якому кожен лівий дочірній вузол менший за батьківський, а кожен правий — більший.


## Префіксні дерева (Trie)

Відомі також як префіксні дерева, використовуються для зберігання множин рядків, де кожен вузол представляє символ.

Спільні префікси зберігаються лише один раз, що робить їх ефективними для автозаповнення і перевірки орфографії.

Операції пошуку і вставки мають часову складність O(L), де L — довжина рядка.

## Пріоритетні черги

Пріоритетна черга — це абстрактний тип даних, у якому кожен елемент має пріоритет.

Черги і стеки враховують лише порядок вставки, а пріоритетні черги — пріоритет елементів.

Стандартні черги працюють за принципом FIFO, а стеки — LIFO. У пріоритетній черзі елементи з вищим пріоритетом обслуговуються раніше, незалежно від порядку вставки.

## Купи

Спеціалізована структура даних на основі дерева з особливою властивістю, що називається властивістю купи.

Властивість купи визначає відношення між батьківськими і дочірніми вузлами. Існують два типи куп:

- Макс-купа
  - Значення кожного батьківського вузла більше або дорівнює значенням дочірніх.
  - Найбільший елемент знаходиться в корені.

- Мін-купа
  - Значення кожного батьківського вузла менше або дорівнює значенням дочірніх.
  - Найменший елемент знаходиться в корені.

### Приклад використання модуля `heapq` у Python

```py
import heapq

# Create empty heap
my_heap = []

# Insert elements
heapq.heappush(my_heap, 9)
heapq.heappush(my_heap, 3)
heapq.heappush(my_heap, 5)
print(my_heap) # [3, 9, 5]

# Remove smallest element
print(heapq.heappop(my_heap))  # 3
print(my_heap) # [5, 9]

# Push + Pop in one step
print(heapq.heappushpop(my_heap, 7)) # 5
print(my_heap) # [7, 9]

# Transform list into heap
nums = [5, 7, 3, 1]
heapq.heapify(nums)
```

### Використання пріоритетів

```py
my_heap = []
heapq.heappush(my_heap, (3, "A"))
heapq.heappush(my_heap, (2, "B"))
heapq.heappush(my_heap, (1, "C"))

# Removes lowest number = highest priority
print(heapq.heappop(my_heap))  # (1, "C")
```

## Вступ до динамічного програмування

- **Визначення**: Динамічне програмування — це алгоритмічна техніка, яка розв’язує складні задачі, розбиваючи їх на простіші підзадачі і зберігаючи результати, щоб уникнути повторних обчислень.
- **Перекривні підзадачі**: Одні й ті ж менші задачі з’являються кілька разів при розв’язанні більшої задачі. Замість повторного обчислення ми зберігаємо їхні рішення.
- **Оптимальна підструктура**: Оптимальне рішення задачі містить оптимальні рішення її підзадач. Це означає, що можна побудувати найкраще рішення, комбінуючи найкращі рішення менших частин.

## Рішення динамічного програмування

- **Мемоізація (підхід зверху вниз)**: Мемоізація зберігає результати дорогих викликів функцій і повертає кешований результат, коли ті самі вхідні дані з’являються знову.

```py
def climb_stairs_memo(n, memo={}):
    """Dynamic programming with memoization"""
    # Check if we've already calculated this value
    if n in memo:
        return memo[n]  # Return cached result - O(1) lookup!
    
    # Base cases
    if n <= 2:
        return n
    
    # Calculate once and store in memo for future use
    memo[n] = climb_stairs_memo(n-1, memo) + climb_stairs_memo(n-2, memo)
    return memo[n]
```

- **Табуляція (підхід знизу вгору)**: Табуляція будує рішення знизу вгору, заповнюючи таблицю рішеннями підзадач.

```py
def climb_stairs_tabulation(n):
    """Dynamic programming with tabulation"""
    if n <= 2:
        return n
    
    # Create array to store results for all steps from 0 to n
    dp = [0] * (n + 1)
    dp[1] = 1  # 1 way to reach step 1
    dp[2] = 2  # 2 ways to reach step 2
    
    # Build up the solution iteratively
    for i in range(3, n + 1):
        # Ways to reach step i = ways to reach (i-1) + ways to reach (i-2)
        dp[i] = dp[i-1] + dp[i-2]
    
    return dp[n]
```

- **Оптимізація маршруту**: GPS-системи використовують алгоритми динамічного програмування, щоб знайти найкоротші шляхи між локаціями.
- **Обробка тексту**: Перевірка орфографії та функції автозаповнення часто покладаються на динамічне програмування для обчислення відстаней редагування між словами.
- **Фінансове моделювання**: Інвестиційні стратегії та оптимізація портфоліо часто застосовують техніки динамічного програмування.
- **Розподіл ресурсів**: Задача рюкзака та її варіанти зустрічаються у плануванні, бюджетуванні та управлінні ресурсами.
