# --description--

UTF-8、またはUCS変換形式8は、ウェブで広く使われている標準化された文字エンコーディングです。文字エンコーディングとは、コンピューターが文字をデータとして保存する方法のことです。基本的に、ウェブページ上のすべてのテキストは、1つ以上のバイトとして保存された文字のシーケンスです。コンピューターでは、バイトは8ビット（2進数の数字）からなるデータの単位です。UTF-8はUnicode文字セットのすべての文字をサポートしており、これにはすべての書記体系、言語、技術記号の文字や記号が含まれます。以下は、`meta`要素に`charset`属性を使って文字エンコーディングを`UTF-8`に設定する例です。

```html
<meta charset="UTF-8" />
```

文字エンコーディングをUTF-8に設定することで、アクセント付きの`"e"`文字（`é`）がページ上で正しく表示されるようになります。UTF-8文字エンコーディングを使った拡張コード例はこちらです。

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Examples of the UTF-8 encoding</title>
  </head>
  <body>
    <p>Café</p>
  </body>
</html>
```

新しいプロジェクトを作成するたびに、この`meta`要素を`charset`属性に`UTF-8`を設定して含めることをお勧めします。
