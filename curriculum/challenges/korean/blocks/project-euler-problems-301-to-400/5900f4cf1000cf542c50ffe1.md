# --description--

각 변의 길이가 1인 완벽한 정육각형으로 이루어진 꿀벌의 벌집을 생각하세요.

<img alt="변 길이가 1인 육각형 벌집" src="https://cdn.freecodecamp.org/curriculum/project-euler/distances-in-a-bees-honeycomb.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

특정한 한 칸은 여왕벌이 차지하고 있습니다. 양의 실수 $L$에 대해, $B(L)$은 여왕벌 칸으로부터 거리 $L$인 칸의 개수를 셉니다(모든 거리는 중심에서 중심까지 측정합니다). 벌집은 우리가 고려하려는 어떤 거리도 수용할 만큼 충분히 크다고 가정하세요.

예를 들어, $B(\sqrt{3}) = 6$, $B(\sqrt{21}) = 12$, 그리고 $B(111\\,111\\,111) = 54$입니다.

$B(L) = 450$인 $L ≤ 5 \times {10}^{11}$인 값의 개수를 구하세요.
