# --interactive--

الخصائص المخصصة في CSS، المعروفة أيضًا بمتغيرات CSS، هي كيانات يُعلن عنها مؤلفو CSS تحتوي على قيم محددة لإعادة استخدامها في جميع أنحاء المستند. إنها ميزة قوية تسمح بإنشاء أوراق أنماط أكثر كفاءة وقابلية للصيانة ومرونة.

صيغة الإعلان عن خاصية مخصصة بسيطة. تبدأ بشرطتين (`--`) متبوعتين باسم الخاصية:

```css
:root {
  --main-color: #3498db;
}
```

في هذا المثال، نعلن عن خاصية مخصصة باسم `--main-color` بقيمة `#3498db`. تُستخدم الفئة المزيفة `:root` عادةً لإعلان الخصائص المخصصة العالمية، إذ تمثل أعلى مستوى أب في شجرة DOM.

لاستخدام خاصية مخصصة، تستعمل الدالة `var()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
:root {
  --main-color: #3498db;
}

.button {
  background-color: var(--main-color);
}
```

:::

هذا يحدد لون خلفية العناصر التي تحمل الصنف `button` إلى القيمة المخزنة في `--main-color`.

من الميزات الرئيسية للخصائص المخصصة أنها تتبع تتابع CSS. هذا يعني أنه يمكنك إعادة تعريفها لعناصر أو سياقات محددة:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="alert">This is an alert message.</div>
```

```css
.alert {
  --main-color: #e74c3c;
  background-color: var(--main-color);
}
```

:::

في هذه الحالة، ستستخدم العناصر التي تحمل الصنف `alert` قيمة مختلفة لـ `--main-color`، متجاوزة التعريف العالمي.

تدعم الخصائص المخصصة أيضًا قيم النظام البديل. إذا لم تُعرّف خاصية مخصصة أو كانت غير صالحة، يمكنك توفير قيمة نظام بديل:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="text">This is some text.</div>
```

```css
:root {
  --text-color: green;
}

.text {
  color: var(--text-color, green);
}
```

:::

هنا، إذا لم يُعرّف `--text-color`، فسيكون `color` افتراضيًا على `green`.

تُستخدم الخصائص المخصصة بشكل خاص في إنشاء تصاميم قابلة للتخصيص حسب الموضوع. يمكنك تعريف مجموعة من الخصائص لمواضيع مختلفة:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="light-theme">
  <p>This is some light themed text.</p>
</div>
```

```css
:root {
  --bg-color: black;
}

body {
  background-color: var(--bg-color);
}

.light-theme {
  --bg-color: white;
  --text-color: black;
  background-color: var(--bg-color);
  color: var(--text-color);
  padding: 10px;
}

```

:::

يصبح تبديل المواضيع بسيطًا مثل إضافة أو إزالة صنف من عنصر `body`.

يمكن أيضًا استخدام الخصائص المخصصة مع استعلامات الوسائط لإنشاء تصميمات متجاوبة:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <h2>Responsive Design</h2>
  <p>Resize the window to see the card adapt!</p>
</div>

```

```css
:root {
  --card-width: 90%;
  --card-bg: #f0f0f0;
  --card-padding: 1rem;
  --text-color: #333;
}

/* Tablet screens and up */
@media (min-width: 600px) {
  :root {
    --card-width: 70%;
    --card-bg: #e8f5e9;
    --card-padding: 1.5rem;
  }
}

/* Desktop screens and up */
@media (min-width: 1024px) {
  :root {
    --card-width: 50%;
    --card-bg: #d0f0ff;
    --card-padding: 2rem;
  }
}

body {
  font-family: system-ui, sans-serif;
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
  background-color: #fafafa;
}

.card {
  width: var(--card-width);
  background-color: var(--card-bg);
  padding: var(--card-padding);
  color: var(--text-color);
  border-radius: 8px;
  box-shadow: 0 2px 6px rgba(0,0,0,0.1);
  text-align: center;
  transition: all 0.3s ease;
}

```

:::

تسمح هذه الطريقة بإنشاء تخطيطات متجاوبة أكثر قابلية للصيانة، إذ يمكنك تغيير القيم في مكان واحد بدلاً من جميع أنحاء ورقة الأنماط. هذا يقلل بشكل كبير من التكرار في أوراق الأنماط، خاصة للقيم المعقدة المستخدمة كثيرًا.

يمكن للخصائص المخصصة أيضًا أن تشير إلى خصائص مخصصة أخرى:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
:root {
  --primary-color: #3498db;
  --button-bg: var(--primary-color);
}

button {
  background-color: var(--button-bg);
}

```

:::

هذا يسمح بإنشاء علاقات بين جوانب مختلفة من الأنماط، مما يسهل الحفاظ على ثيمات متناسقة عبر موقعك.

بينما تقدم الخصائص المخصصة العديد من المزايا، من المهم أن تكون على دراية بدعم المتصفحات لها. فهي مدعومة جيدًا في المتصفحات الحديثة، لكن المتصفحات القديمة قد لا تتعرف عليها. دائمًا جهز نظامًا بديلاً عند استخدام ميزات متقدمة.

في الختام، توفر الخصائص المخصصة في CSS طريقة قوية لإنشاء أوراق أنماط أكثر ديناميكية ومرونة وقابلية للصيانة. تسمح بإنشاء تصاميم قابلة للتخصيص، تبسط التخطيطات المتجاوبة، وتمكّن من تعديل الأنماط أثناء وقت التشغيل.

مع استمرار تطور تطوير الويب، تصبح الخصائص المخصصة أداة مهمة بشكل متزايد في مجموعة أدوات المطوِّر، مقدمة إمكانيات جديدة لإنشاء CSS قابلة للتكيف وفعالة.
