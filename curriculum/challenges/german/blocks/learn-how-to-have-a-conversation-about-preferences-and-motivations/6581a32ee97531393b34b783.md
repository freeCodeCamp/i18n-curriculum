# --description--

`Encourage` bedeutet, jemandem Unterstützung oder Zuversicht zu geben, etwas zu tun. Zum Beispiel, wenn Ihr Freund sagt, `You can solve this coding problem!`, ermutigt er Sie.

`Discourage` ist das Gegenteil; es bedeutet, jemandem das Gefühl zu geben, weniger zuversichtlich oder weniger geneigt zu sein, etwas zu tun. Wenn dieser Freund sagt, `That programming language is too hard for beginners`, könnte das Sie davon abhalten, es zu lernen.

`Discouraging` beschreibt etwas, das Sie entmutigt. Zum Beispiel, `Finding a bug in your code after hours of work can be discouraging.`

`Encouraging` ist, wenn etwas Ihnen Mut oder Hoffnung gibt. Zum Beispiel, `Getting your first program to run is very encouraging.`
