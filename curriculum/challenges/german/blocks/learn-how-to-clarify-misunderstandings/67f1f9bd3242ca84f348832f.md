`user interface`

### --feedback--

Dieser zweiteilige Ausdruck bezeichnet den Teil eines Programms oder einer App, den Benutzer sehen und mit dem sie interagieren, wie Schaltflächen, Menüs oder Bildschirme.

# --explanation--

`Minor bugs` sind kleine Fehler oder Probleme in einem System oder Programm. Zum Beispiel:

`There are some minor bugs in the app, but it still runs.` — Das bedeutet, die Probleme sind nicht schwerwiegend, müssen aber behoben werden.

`User interface` (oft `UI` genannt) ist der visuelle Teil einer Software, mit dem Menschen interagieren. Zum Beispiel:

`The user interface was updated to include a dark mode.` — Das bezieht sich auf Änderungen, die Benutzer direkt sehen und nutzen können.
