# --explanation--

`Make sure to` significa assicurarsi o prendere provvedimenti per fare qualcosa con cura o correttamente. Si usa per esprimere l'idea che qualcuno sta facendo passi per garantire che qualcosa accada. Per esempio:

`I made sure to finish the project on time.` - Questo significa che hai preso delle azioni per assicurarti che il progetto fosse completato in tempo.
