# --explanation--

`True` 是一个形容词，表示某事是正确的、真实的或准确的。它用于同意某个语句或事实。它不同于 `the truth`，后者是一个名词，指某事的实际事实或现实，通常用于更广泛或更抽象的意义。例如：

- `It's true that the server crashed yesterday.` – 这意味着关于崩溃的语句是正确的。

- `The truth is that we didn't have a backup.` – 这意味着实际情况是没有备份存在。
