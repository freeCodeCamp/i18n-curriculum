# --description--

现在我们来学习**40**年代的数字。

在中文里，数字**40**叫做 `四十 (sì shí)`。

从**41 到 49**的数字是通过在 `四十 (sì shí)`（40）后直接添加个位数字（1–9）形成的。这个模式和 20 多和 30 多的数字一样。

| Number | Chinese |
| :--- | :--- |
| 40 | `四十 (sì shí)` |
| 41 | `四十一 (sì shí yī)` |
| 42 | `四十二 (sì shí èr)` |
| 43 | `四十三 (sì shí sān)` |
| 44 | `四十四 (sì shí sì)` |
| 45 | `四十五 (sì shí wǔ)` |
| 46 | `四十六 (sì shí liù)` |
| 47 | `四十七 (sì shí qī)` |
| 48 | `四十八 (sì shí bā)` |
| 49 | `四十九 (sì shí jiǔ)` |
