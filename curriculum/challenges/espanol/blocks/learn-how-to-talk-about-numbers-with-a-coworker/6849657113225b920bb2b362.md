# --explanation--

La estructura "number + unit + `per` + time period" se usa comúnmente para expresar el tiempo dedicado a una actividad. Por ejemplo:

- El proceso de respaldo toma 30 minutos diarios.

- `The system runs for 20 hours per week.` - El sistema funciona 20 horas cada semana.

Tom dice que la plataforma se usa alrededor de `60 hours per week`, lo que significa que el equipo pasa esa cantidad de tiempo usándola cada semana.
