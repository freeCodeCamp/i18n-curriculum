# --explanation--

`Sure` es una palabra usada para mostrar acuerdo o permiso. Por ejemplo: 

`Sure, I can help you with that.` – Esto significa que el hablante está de acuerdo o dispuesto a ayudar.

`What` es una palabra interrogativa usada para pedir información. Por ejemplo:  

`What is your name?` – Esto significa que el hablante quiere saber un detalle específico.

`Choices` se refiere a las diferentes opciones disponibles. Por ejemplo:  

`We have three choices for the new logo design.` – Esto significa que hay tres opciones diferentes para elegir.
