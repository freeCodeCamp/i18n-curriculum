# --description--

正の数 $n$ に対して、$C(n)$ を $1 < x < n$ かつ $x^3 \equiv 1\bmod n$ を満たす整数 $x$ の個数と定義します。

$n = 91$ のとき、$x$ の可能な値は 9、16、22、29、53、74、79、81 の8つあります。したがって、$C(91) = 8$ です。

$C(n) = 242$ となる正の数 $n \leq {10}^{11}$ の合計を求めてください。
