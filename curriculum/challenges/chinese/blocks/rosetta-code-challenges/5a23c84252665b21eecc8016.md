# --description--

编写一个函数，按长度降序对字符串数组（或列表）进行排序，对相等长度的字符串按字典序升序排序。

```js
function lengthSorter(arr) {
  arr.sort(function(a, b) {
    var result = b.length - a.length;
    if (result == 0) result = a.localeCompare(b);
    return result;
  });
  return arr;
}
```
