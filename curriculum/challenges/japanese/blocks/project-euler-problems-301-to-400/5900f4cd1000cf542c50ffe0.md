# --description--

月は中心が (0, 0, 0)、半径が $r$ の球面 $C(r)$ で表せます。

月の表面上の整数座標の点にステーションがあります。座標 (0, 0, $r$) のステーションは北極ステーション、座標 (0, 0, $-r$) のステーションは南極ステーションと呼ばれます。

すべてのステーションは、ステーション間の大円弧上の最短道路で互いに接続されています。2つのステーション間の移動は危険です。2つのステーション間の道路の長さを $d$ とすると、$\{\left(\frac{d}{πr}\right)}^2$ がその移動のリスクの尺度（これを道路のリスクと呼びます）です。移動に2つ以上のステーションが含まれる場合、その移動のリスクは使用した道路のリスクの合計になります。

北極ステーションから南極ステーションへの直接の移動は長さが $πr$ で、リスクは1です。北極ステーションから (0, $r$, 0) を経由して南極ステーションへ行く移動は同じ長さですが、リスクは小さくなります：

$${\left(\frac{\frac{1}{2}πr}{πr}\right)}^2+{\left(\frac{\frac{1}{2}πr}{πr}\right)}^2 = 0.5$$

$C(r)$ 上の北極ステーションから南極ステーションへの移動の最小リスクを $M(r)$ とします。

$M(7) = 0.178\\,494\\,399\\,8$（小数点以下10桁に丸めた値）が与えられています。

$\displaystyle\sum_{n = 1}^{15} M(2^n - 1)$ を求めてください。

答えは小数点以下10桁に丸めて、a.bcdefghijk の形式で提出してください。
