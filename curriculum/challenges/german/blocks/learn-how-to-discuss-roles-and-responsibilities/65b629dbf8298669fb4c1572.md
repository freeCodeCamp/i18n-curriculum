# --description--

Das Wort `everyone` bedeutet jede Person in einer Gruppe. `Every-` wird verwendet, wenn man über alle Dinge oder Personen in einer Gruppe einzeln spricht.

Es ist wichtig zu beachten, dass `everyone` so klingt, als würde es über viele Personen sprechen, aber Sie verwenden es mit einem Singular-Verb. Das ist ähnlich wie bei anderen Wörtern, die mit `every-` beginnen, wie `everything` und `everybody`.

Zum Beispiel sagen Sie `Everyone goes to the party`, `Everything is right` oder `Everybody works hard`.

`Make sure` bedeutet, etwas zu überprüfen oder sicherzustellen, dass es korrekt ist, bevor man weitermacht. Zum Beispiel bedeutet `You make sure the code runs`, dass Sie den Code überprüfen, um sicherzugehen, dass er keine Fehler hat und richtig ausgeführt werden kann.

`Follow the rules` bedeutet, das zu tun, was die Regeln sagen. Wenn es eine Regel gibt, die sagt `use camelCase for variable names`, bedeutet `follow the rules`, dass Sie Ihre Variablen so benennen wie `myVariable` statt `myBLANKvariable`.
