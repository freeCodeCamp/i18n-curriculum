# --explanation--

El tiempo `Past Continuous` (`was/were` + verbo en la forma `-ing`) describe una acción que estaba ocurriendo durante un período de tiempo en el pasado. A menudo se usa junto con un evento en `Simple Past` para mostrar que la acción continua fue interrumpida o sucedió antes de otro evento. Por ejemplo:

`She was working on the report when the computer crashed.` – La acción de trabajar estaba en curso cuando ocurrió el fallo (`Simple Past`).
