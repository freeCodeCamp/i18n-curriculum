# --description--

小さな子どもが「数字のイモムシ」を持っています。これは40個のジグソーピースでできており、それぞれに数字が1つ書かれています。これらを一列につなげると、1から40までの数字が順番に現れます。

毎晩、子どもの父親は遊び部屋に散らばったイモムシのピースを拾い集めます。彼はピースをランダムに拾い、正しい順番に並べます。

こうしてイモムシが組み立てられていくと、いくつかの区間ができ、それらが徐々にマージされていきます。区間の数は最初はゼロ（ピースが置かれていない状態）から始まり、だいたい11か12くらいまで増え、その後また減っていき、最後には1つの区間（すべてのピースが置かれた状態）になります。

例えば：

| 置かれたピース | これまでの区間数 |
| -------------|-----------------|
|       12     |        1        |
|        4     |        2        |
|       29     |        3        |
|        6     |        4        |
|       34     |        5        |
|        5     |        4        |
|       35     |        4        |
|       …      |        …        |

$M$ をイモムシをランダムにかたづけたときに出現する区間の最大数とします。10ピースのイモムシの場合、各 $M$ に対する可能性の数は以下の通りです。

| M | 可能性の数 |
|---|---------------|
| 1 |      512      |
| 2 |     250912    |
| 3 |    1815264    |
| 4 |    1418112    |
| 5 |     144000    |

したがって、$M$ の最も可能性の高い値は3で、平均値は $\frac{385\\,643}{113\\,400} = 3.400732$（小数点以下6桁に丸め）です。

40ピースのイモムシの場合、$M$ の最も可能性の高い値は11ですが、$M$ の平均値はいくつでしょうか？答えは小数点以下6桁に丸めてください。
