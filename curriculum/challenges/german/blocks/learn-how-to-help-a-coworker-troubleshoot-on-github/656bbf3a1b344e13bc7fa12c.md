# --description--

`Strange` bedeutet ungewöhnlich oder nicht erwartet. Es ist wie etwas zu finden, das anders ist als das, was Sie normalerweise sehen. Zum Beispiel: `A strange sound in a car might mean a problem.`

Ein `error` ist ein Fehler, besonders einer, der etwas daran hindert, richtig zu funktionieren. Zum Beispiel: `An error in a code means there is a mistake that needs fixing.`
