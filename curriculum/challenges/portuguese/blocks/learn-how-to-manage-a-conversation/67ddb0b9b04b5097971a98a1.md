# --explanation--

Na linguagem falada casualmente, você frequentemente omite palavras quando o significado ainda está claro.Neste caso, `Anything else we should discuss?` é uma forma abreviada de `Is there anything else we should discuss?`.Esse tipo de omissão é comum em conversas.Outro exemplo:

`Need any help?` é uma forma mais curta de dizer `Do you need any help?` – o sujeito e o verbo auxiliar são omitidos, mas o significado permanece o mesmo.
