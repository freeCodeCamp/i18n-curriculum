# --description--

`r`를 ${(a − 1)}^n + {(a + 1)}^n$를 $a^2$로 나눈 나머지라고 하겠습니다.

예를 들어, $a = 7$이고 $n = 3$일 때, $r = 42$입니다: $6^3 + 8^3 = 728 ≡ 42 \\ \text{mod}\\ 49$. 그리고 `n`가 변함에 따라 `r`도 변하지만, $a = 7$일 때는 $r_{max} = 42$임을 알 수 있습니다.

$3 ≤ a ≤ 1000$에 대해 $\sum{r}_{max}$를 구하세요.
