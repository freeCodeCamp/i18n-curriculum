既然你已完成所有「使用 C# 撰寫你的第一個程式碼」挑戰，你應該已在 Microsoft Learn 平台上獲得該獎盃。如果你在檢視 <a href="https://learn.microsoft.com/users/me/achievements#trophies-section" target="_blank" rel="noreferrer">你的成就頁面</a> 時看不到此獎盃，表示你尚未完成所有任務，且無法驗證你的獎盃。

將你的 Microsoft 使用者名稱連結到你的 freeCodeCamp 帳號，然後點擊下方的「Verify Trophy」按鈕以完成挑戰。此獎盃為通過認證考試的**必要**條件。
