# --description--

Щоб змінити масштаб елемента, у CSS є властивість `transform` разом із її функцією `scale()`. У наведеному прикладі коду розмір усіх абзаців на сторінці подвоюється:

```css
p {
  transform: scale(2);
}
```

# --instructions--

Збільшіть розмір елемента з ідентифікатором `ball2` у 1,5 раза від його початкового розміру.

# --hints--

Властивість `transform` для `#ball2` має бути встановлена так, щоб масштабувати його до 1,5 раза від початкового розміру.

```js
assert.match(code,
    /#ball2\s*?{\s*?left:\s*?65%;\s*?transform:\s*?scale\(1\.5\);\s*?}|#ball2\s*?{\s*?transform:\s*?scale\(1\.5\);\s*?left:\s*?65%;\s*?}/gi
);
```
