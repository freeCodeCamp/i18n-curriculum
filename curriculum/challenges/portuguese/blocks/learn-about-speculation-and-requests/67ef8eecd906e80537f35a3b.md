# --explanation--

`Should have` é usado para expressar arrependimento ou uma responsabilidade perdida sobre algo no passado. Mark significa que executar os testes foi a coisa certa a fazer, mas ele não fez isso.

`But` introduz a razão pela qual ele não agiu: `I was in a hurry`. Outro exemplo:

`We should have fixed the bug, but we didn't notice it.` - Eles gostariam de ter corrigido o bug, mas não perceberam.
