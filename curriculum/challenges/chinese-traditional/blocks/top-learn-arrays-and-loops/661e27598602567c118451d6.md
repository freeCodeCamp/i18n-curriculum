# --description--

`for` 迴圈是另一種用來多次執行一個程式碼區塊的迴圈。當迭代次數已知時，會使用 `for` 迴圈。`for` 迴圈的語法如下：

```javascript
for (initialization; condition; increment/decrement) {
  // code block to be executed
}
```

例如，以下程式碼片段會列印從 1 到 5 的數字：

```javascript
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
```

要使用 `for` 迴圈迭代陣列，你可以使用陣列的長度屬性。 例如，以下程式碼片段會列印陣列的元素：

```javascript
const arr = [1, 2, 3, 4, 5];
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}
```
