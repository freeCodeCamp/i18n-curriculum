# --interactive--

Quando costruisci siti web multilingue o gestisci attributi dati personalizzati, spesso devi definire lo stile degli elementi in base alla lingua che contengono o a valori dati specifici.

Gli attributi `lang` e `data-lang` sono comunemente usati per questi scopi, e il selettore di attributi in CSS ti permette di applicare stili basati efficacemente su questi attributi.

L'attributo `lang` viene usato in HTML per specificare la lingua del contenuto all'interno di un elemento. Potresti voler definire lo stile degli elementi in modo diverso in base alla lingua in cui sono scritti, specialmente su un sito web multilingue.

Ecco un esempio di come puoi usare il selettore di attributi per consultare elementi con un attributo `lang` specifico:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">This is an English paragraph.</p>
<p lang="fr">Ceci est un paragraphe en français.</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

Questa regola CSS applica uno stile corsivo a qualsiasi elemento paragrafo in cui l'attributo `lang` è impostato su inglese (`en`). Questo può essere utile per enfatizzare il testo in inglese in un documento che include più lingue.

Allo stesso modo, puoi usare il selettore di attributi per consultare elementi con un attributo `data-lang`.

Gli attributi dati personalizzati come `data-lang` sono comunemente usati per archiviare informazioni aggiuntive negli elementi, come specificare la lingua usata all'interno di una sezione specifica di testo.

Ecco come puoi usare il CSS per consultare e definire lo stile degli elementi basandoti sull'attributo `data-lang`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p data-lang="fr">Ceci est un paragraphe en français.</p>
<p data-lang="en">This is a paragraph in English.</p>
```

```css
p[data-lang="fr"] {
  color: blue;
}
```

:::

In questo caso, qualsiasi elemento `p` con un attributo `data-lang` impostato su francese (`fr`) avrà il colore del testo cambiato in blu. Questo ti permette di identificare rapidamente e definire lo stile delle sezioni di contenuto basandoti sulle informazioni linguistiche archiviate nell'attributo `data-lang`.

I selettori di attributi come questi offrono un modo potente per applicare stili condizionali basati sui metadati incorporati nel tuo HTML, rendendo le tue pagine web più dinamiche e consapevoli del contesto.
