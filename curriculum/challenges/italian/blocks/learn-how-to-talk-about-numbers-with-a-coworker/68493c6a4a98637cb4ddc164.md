# --explanation--

Tom dice `in the Backup Licenses section`, il che significa che c'è un'area o una cartella specifica all'interno dell'archivio cloud dove quei file sono salvati.

Qui, `in` indica che si trovano all'interno di quella sezione. Un altro esempio:

`The answer is in the FAQ section.` — Puoi trovare la risposta nella parte FAQ.
