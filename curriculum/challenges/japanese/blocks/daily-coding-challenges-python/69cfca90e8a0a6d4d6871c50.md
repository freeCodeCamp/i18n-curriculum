# --description--

`"HH:MM"` 形式（24時間制）の時刻文字列が与えられたとき、次を返してください：

- `"Good morning"` から `05:00` までの時刻には `11:59`
- `"Good afternoon"` から `12:00` までの時刻には `17:59`
- `"Good evening"` から `18:00` までの時刻には `21:59`
- `"Good night"` から `22:00` までの時刻には `04:59`

# --solutions--

```py
def get_greeting(s):
    hours, minutes = map(int, s.split(':'))
    total = hours * 60 + minutes

    if 300 <= total < 720:
        return 'Good morning'
    if 720 <= total < 1080:
        return 'Good afternoon'
    if 1080 <= total < 1320:
        return 'Good evening'
    return 'Good night'
```
