# --description--

Listas Python são mutáveis o que significa que o valor dos itens da lista pode ser alterado.Você pode alterar o valor de um elemento usando a notação de colchetes.

```py
example_list = [4, 5, 6, 7]
example_list[1] = 'oh'
```

Isso fará com que `example_list` tenha o valor `[4, 'oh', 6, 7]`.

Altere o primeiro elemento de `my_list` para `0`, depois imprima a lista para verificar o valor.
