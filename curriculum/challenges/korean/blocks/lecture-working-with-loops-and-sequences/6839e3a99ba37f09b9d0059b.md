# --description--

튜플은 값을 순서대로 나열한 수열을 만들 때 사용하는 Python 데이터 유형입니다. 튜플은 다음과 같이 다양한 데이터 유형을 혼합해서 담을 수 있습니다:

```py
developer = ('Alice', 34, 'Rust Developer')
```

튜플은 리스트와 비슷하지만, 리스트가 변경 가능한 데이터 유형인 반면 튜플은 변경 불가능합니다. 즉, 튜플이 생성된 후에는 그 안의 요소를 변경할 수 없습니다.

튜플의 항목 중 하나를 수정하려고 하면 `TypeError`가 발생합니다:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: 'tuple' object does not support item assignment
"""
```

튜플의 요소에 접근하려면 대괄호 표기법과 인덱스 번호를 사용할 수 있습니다:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

튜플의 끝에서부터 요소에 접근해야 할 경우 음수 인덱싱을 사용할 수 있습니다. 다음은 튜플에서 끝에서 두 번째 요소에 음수 인덱스를 사용해 접근하는 예입니다:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

튜플 길이보다 크거나 같은 인덱스 번호를 전달하면 다음과 같이 `IndexError`가 발생합니다:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list index out of range
"""
```

튜플을 만드는 또 다른 방법은 `tuple()` 생성자를 사용하는 것입니다:

```py
developer = 'Jessica'
tuple(developer) # ('J', 'e', 's', 's', 'i', 'c', 'a')
```

`tuple()` 생성자에는 문자열, 리스트, 심지어 다른 튜플 같은 다양한 반복 가능한(iterable) 객체를 전달할 수 있습니다.

튜플에 특정 항목이 있는지 확인하려면 `in` 키워드를 사용할 수 있습니다:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

리스트에서 했던 것처럼 튜플에서도 항목을 언팩할 수 있습니다:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

이 예에서 `name`는 `'Alice'` 값을 가지며, `age`는 `34` 값을, `job`는 `'Rust Developer'` 값을 가집니다.

튜플에서 나머지 요소들을 모두 수집해야 할 경우 별표(`*`) 연산자를 사용할 수 있습니다:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

여기서 `name`는 `'Alice'` 값을 가지며, `rest`는 숫자 `34`와 문자열 `'Rust Developer'`로 이루어진 리스트입니다.

리스트와 마찬가지로 튜플에서도 슬라이스 연산자를 사용해 일부를 추출할 수 있습니다. 다음은 `'pie'`와 `'cookies'` 항목을 별도의 튜플로 추출하는 예입니다:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

첫 번째 숫자는 추출 시작 인덱스를 나타내고 두 번째 숫자는 종료 인덱스를 나타냅니다. 단, 종료 인덱스에 해당하는 항목은 추출된 튜플에 포함되지 않는다는 점을 기억하세요.

튜플은 변경 불가능하므로 항목을 삭제할 수 없습니다. 따라서 다음 예제는 오류를 발생시킵니다:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

그렇다면 언제 리스트 대신 튜플을 사용해야 할까요?

요소를 추가, 삭제, 수정할 수 있는 동적인 컬렉션이 필요하면 리스트를 사용하세요. 고정되고 변경 불가능한 데이터 집합을 다룰 때는 튜플을 사용하세요.

다음 수업에서는 튜플을 다룰 때 자주 사용하는 메서드를 살펴보겠습니다.
