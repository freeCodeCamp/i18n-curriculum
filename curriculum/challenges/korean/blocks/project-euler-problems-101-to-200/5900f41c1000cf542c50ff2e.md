# --description--

$f(0) = 1$로 정의하고, $f(n)$은 각 거듭제곱이 두 번을 넘지 않도록 2의 거듭제곱 합으로 $n$을 표현하는 방법의 수로 정의하세요.

예를 들어, $f(10) = 5$입니다. 10을 표현하는 다섯 가지 방법은 다음과 같습니다:

$$10 = 8 + 2 = 8 + 1 + 1 = 4 + 4 + 2 = 4 + 2 + 2 + 1 + 1 = 4 + 4 + 1 + 1$$

모든 분수 $\frac{p}{q}\\; (p>0, q>0)$에 대해 적어도 하나의 정수 $n$이 존재하여 $\frac{f(n)}{f(n - 1)} = \frac{p}{q}$임을 증명할 수 있습니다.

예를 들어, $\frac{f(n)}{f(n - 1)} = \frac{13}{17}$이 되는 가장 작은 $n$은 241입니다. 241의 이진 표현은 11110001입니다.

이 이진수를 가장 큰 자리수부터 가장 작은 자리수까지 읽으면 1이 4개, 0이 3개, 1이 1개 있습니다. 이 문자열 4,3,1을 241의 단축 이진 표현(Shortened Binary Expansion)이라고 부릅니다.

다음 조건을 만족하는 가장 작은 $n$의 단축 이진 표현을 구하세요:

$$\frac{f(n)}{f(n - 1)} = \frac{123456789}{987654321}$$

답은 공백 없이 쉼표로 구분된 정수 문자열로 제출하세요.
