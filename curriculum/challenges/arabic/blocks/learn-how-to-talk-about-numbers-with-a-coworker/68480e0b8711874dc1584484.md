# --explanation--

`User Interface (UI)` هو ما يراه المستخدم (الشاشات، الأزرار، الألوان).

`User Experience (UX)` هو كيف يشعر المستخدم (السلاسة، الوضوح، سهولة الاستخدام).

المنتجات الجيدة تركز على كلاهما: مرئيات واضحة وتجربة مرضية.
