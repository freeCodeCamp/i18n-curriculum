Para isso, dentro do bloco `else`, crie um loop `for` que execute até `max_iterations` vezes.

Para seu loop, use a função `range`, que gera uma sequência de números sobre a qual você pode iterar.A sintaxe é `range(start, stop, step)`, onde `start` é o inteiro inicial (inclusivo), `stop` é o último inteiro (não inclusivo) e `step` é a diferença entre um número e o anterior na sequência.

Além disso, use `_` como uma variável de loop.O `_` atua como um espaço reservado e é útil quando você precisa usar uma variável mas não precisa realmente do seu valor.
