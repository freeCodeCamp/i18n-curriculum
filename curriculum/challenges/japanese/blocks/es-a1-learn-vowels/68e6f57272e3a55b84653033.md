# --description--

スペイン語の母音の学習を始める準備はできましたか？

母音は、口を開けて空気の流れを妨げずに発音する音です。

スペイン語には5つの母音があります：**`a`**、**`e`**、**`i`**、**`o`**、**`u`**。

これらは大文字の母音です：**`A`**、**`E`**、**`I`**、**`O`**、**`U`**。

母音は一貫して同じ音で発音されます。これは覚えやすく練習しやすいのでとても良いことです。スペイン語の単語で母音を見かけたら、学んだ通りに正確に発音されることがわかります。

それでは、耳のトレーニングを始めましょう。

5つの母音を聞いて、その音に慣れるまで繰り返し練習してください。

# --assignment--

母音を聞いて繰り返し練習しました。
