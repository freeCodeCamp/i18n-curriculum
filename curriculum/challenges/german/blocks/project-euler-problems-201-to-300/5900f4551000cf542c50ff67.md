# --description--

Zwei Spieler teilen sich eine faire Münze und spielen abwechselnd „Das Rennen“.

In seinem Zug wirft Spieler 1 die Münze einmal: Kommt Kopf, erhält er einen Punkt; kommt Zahl, erhält er keinen Punkt.

Im Zug von Spieler 2 wählt sie eine positive ganze Zahl $T$ und wirft die Münze $T$-mal: Kommt jedes Mal Kopf, erhält sie $2^{T - 1}$ Punkte; andernfalls erhält sie keine Punkte.

Spieler 1 beginnt. Gewinner ist derjenige, der zuerst 100 oder mehr Punkte erreicht.

In jedem Zug wählt Spieler 2 die Anzahl $T$ der Münzwürfe so, dass die Wahrscheinlichkeit ihres Gewinns maximiert wird.

Wie groß ist die Wahrscheinlichkeit, dass Spieler 2 gewinnt?

Geben Sie Ihre Antwort auf acht Dezimalstellen gerundet in der Form 0.abcdefgh an.
