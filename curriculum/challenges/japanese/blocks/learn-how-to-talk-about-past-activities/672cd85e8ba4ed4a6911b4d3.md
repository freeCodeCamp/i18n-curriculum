# --explanation--

`We've` は `we have` の短縮形です。

`Made` は不規則動詞 `make` の過去分詞形です。`Have made` は過去に作られたり完了したことが、現在に影響を与えていることを表します。

この文では、`have had` は現在完了形です。この構造 `have` + `had` は、過去に始まったことが現在も影響を及ぼしていることを示します。

- `Have` は現在完了形で使われる助動詞です。

- `Had` は「経験する」または「所有する」という意味の `have` の過去分詞形です。

これらを合わせて `have had` は、変化が以前に始まり、今も影響を与え続けていることを意味します。

まとめると、これらの不規則動詞（`make` は `made`、`have` は `had`）は現在完了形を作り、過去に始まって今も関連している動作を示します。
