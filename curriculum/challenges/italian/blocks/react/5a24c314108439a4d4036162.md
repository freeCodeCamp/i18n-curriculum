# --description--

I componenti sono il cuore di React. Tutto in React è un componente e qui imparerai come crearne uno.

Ci sono due modi per creare un componente React. Il primo è usare una funzione JavaScript. Definire un componente in questo modo crea un *componente funzionale senza stato*. Il concetto di stato in un'applicazione sarà trattato nelle sfide successive. Per ora, pensa a un componente senza stato come a uno che può ricevere dati e renderizzarli, ma non gestisce né traccia le modifiche a quei dati. (Il secondo modo per creare un componente React lo vedremo nella sfida successiva.)

Per creare un componente con una funzione, basta scrivere una funzione JavaScript che restituisce JSX o `null`. Una cosa importante da notare è che React richiede che il nome della funzione inizi con una lettera maiuscola. Ecco un esempio di componente funzionale senza stato che assegna una classe HTML in JSX:

```jsx
const DemoComponent = function() {
  return (
    <div className='customClass' />
  );
};
```

Dopo la transpilation, il `<div>` avrà una classe CSS `customClass`.

Poiché un componente JSX rappresenta HTML, puoi mettere insieme diversi componenti per creare una pagina HTML più complessa. Questo è uno dei vantaggi chiave dell'architettura a componenti che React offre. Ti permette di comporre la tua interfaccia utente da molti componenti separati e isolati. Questo rende più facile costruire e mantenere interfacce utente complesse.

# --instructions--

L'editor del codice contiene una funzione chiamata `MyComponent`. Completa questa funzione in modo che restituisca un singolo elemento `div` che contenga una stringa di testo.

**Nota:** Il testo è considerato un figlio dell'elemento `div`, quindi non potrai usare un tag a chiusura automatica.

# --hints--

`MyComponent` dovrebbe restituire JSX.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.length === 1;
  })()
);
```

`MyComponent` dovrebbe restituire un elemento `div`.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.children().type() === 'div';
  })()
);
```

L'elemento `div` dovrebbe contenere una stringa di testo.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.find('div').text() !== '';
  })()
);
```
