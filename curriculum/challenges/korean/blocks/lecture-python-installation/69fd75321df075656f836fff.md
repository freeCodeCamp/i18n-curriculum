# --description--

이전 수업에서는 로컬에서 파이썬 스크립트를 실행하는 방법을 배웠습니다. 하지만 때로는 전체 파이썬 프로그램을 만들지 않고 간단히 파이썬 코드를 테스트하고 싶을 때가 있습니다. 이럴 때 파이썬 대화형 셸이 유용합니다.

먼저 터미널이 무엇인지 복습해 보겠습니다.

터미널은 명령어를 입력해 컴퓨터와 상호작용할 수 있는 텍스트 기반 인터페이스입니다. 각 운영체제는 기본 터미널 앱을 제공합니다. macOS에서는 Terminal 앱을 사용할 수 있고, Windows에서는 Command Prompt나 PowerShell을 사용할 수 있습니다. Linux에서는 GNOME Terminal이나 Konsole 같은 각 데스크톱 환경별 기본 터미널 앱이 있습니다.

터미널 앱을 열고 `python`를 입력한 후 `Enter`를 누르세요. 그러면 파이썬 대화형 셸이 시작됩니다. 대화형 셸은 명령어를 한 번에 하나씩 입력하고 결과를 바로 확인할 수 있는 프로그램입니다.

새 세션을 시작하면 처음에 다음과 같은 출력이 보일 수 있습니다:

```bash
Python 3.12.2 (main, Mar 21 2024, 22:48:26) [Clang 14.0.3 (clang-1403.0.22.14.1)] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>> 
```

`>>>` 기호는 파이썬이 명령어 입력을 기다리고 있다는 뜻입니다. 터미널에 `"Hello, world!"`를 출력해 보세요:

```bash
print("Hello, world!")
```

다음과 같이 텍스트가 나타날 것입니다:

```bash
>>> print("Hello, world!")
Hello, world!
```

텍스트가 출력된 후 파이썬은 다시 다음과 같이 돌아갑니다:

```bash
>>>
```

이것은 또 다른 명령어를 입력할 수 있다는 뜻입니다.

파이썬 인터프리터는 읽기(Read), 평가(Evaluate), 출력(Print), 반복(Loop) 주기, 즉 REPL을 따릅니다. 명령어를 입력할 때마다 인터프리터는 입력을 읽고, 평가하고, 결과를 출력한 뒤 `>>>`를 보여주기 위해 다시 반복합니다. 그래서 더 많은 명령어를 입력할 수 있습니다.

만약 다음과 같이 잘못된 명령어를 입력하면 어떻게 될까요?

```bash
>>> something random
```

파이썬 인터프리터는 여전히 같은 REPL 과정을 따릅니다. 이 경우 오류 메시지가 나타납니다:

```bash
>>> something random
  File "<stdin>", line 1
    something random
              ^^^^^^
SyntaxError: invalid syntax
```

대화형 셸을 종료하려면 `exit()`를 입력하거나 (Mac/Linux에서는) `Ctrl + D`를 누르거나 (Windows에서는) `Ctrl + Z + Enter`를 누르세요.

파이썬 대화형 셸은 간단한 코드 실험이나 기본 탐색에 매우 좋습니다. 이 과정의 나머지 수업을 진행하면서 배운 새로운 메서드를 대화형 셸에서 직접 입력해 볼 수 있습니다.

더 긴 프로그램을 작성하거나 여러 파일을 다룰 계획이라면 코드 편집기나 IDE에서 작업하는 것이 더 좋습니다.
