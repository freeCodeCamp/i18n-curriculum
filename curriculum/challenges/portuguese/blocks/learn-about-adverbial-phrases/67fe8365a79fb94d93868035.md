# --explanation--

`Let's think about...` é usado para sugerir considerar uma ideia ou dedicar algum tempo para refletir sobre uma possível ação.

Ao usar `about`, o verbo que o segue está na forma `-ing`.Isso ocorre porque `about` é uma preposição e depois de uma preposição, você usa a forma `-ing` do verbo.Por exemplo:

`Let's think about improving our workflow.` – Aqui, `improving` está na forma `-ing` porque é o verbo que segue `about`.
