# --description--

`One of these days` 是你用來描述你計畫很快要做某事，但還沒確定確切時間的片語。這就像是在說「不久的將來」或「在近期內」。例如：

`One of these days, I'll visit Paris.`－這表示你想造訪巴黎並希望在不久的將來完成，但你還不知道確切的日期。
