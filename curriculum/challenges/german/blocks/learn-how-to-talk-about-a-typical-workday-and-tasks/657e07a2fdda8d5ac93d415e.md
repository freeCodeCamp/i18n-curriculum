# --description--

Sie können Wörter wie `then` verwenden, um über die Reihenfolge zu sprechen, in der Dinge geschehen.

`Then` wird oft verwendet, um den nächsten Schritt oder die nächste Aktion anzuzeigen, nachdem etwas anderes erledigt wurde. Zum Beispiel:

`First, I check my emails, then I start coding.` – Das bedeutet, die Person schaut sich ihre E-Mails an, bevor sie mit dem Schreiben von Code beginnt.

Das hilft Menschen zu verstehen, wie eine Reihe von Aktionen oder Ereignissen zeitlich miteinander verbunden sind.
