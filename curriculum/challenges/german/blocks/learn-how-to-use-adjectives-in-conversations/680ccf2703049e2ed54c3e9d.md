### --feedback--

Diese drei Wörter werden verwendet, um die Länge von zwei Dingen zu vergleichen und zu zeigen, dass sie gleich groß sind.

# --explanation--

`Longer` bedeutet, dass etwas eine größere Entfernung von einem Ende zum anderen hat. Zum Beispiel:

`We need longer cables to reach the new server racks.` – Das bedeutet Kabel mit mehr Länge.

`Twice` bedeutet die doppelte Menge. Zum Beispiel:

`We ordered twice the number of processors this time.` – Das bedeutet die doppelte Menge.

`As long as` wird für den direkten Vergleich der Länge verwendet. Zum Beispiel:

`The new cable is as long as the room.` – Das zeigt, dass die Länge des Kabels der Größe des Raums entspricht.
