# --description--

L'espressione `first things first` viene usata per sottolineare che le cose importanti devono essere fatte prima di qualsiasi altra cosa. Spesso si dice all'inizio di un'attività per mettere il focus sui passaggi più importanti per primi. Per esempio, `First things first, let's review the project plan` significa iniziare con la revisione del piano prima di passare ad altre attività.

Un `environment`, in un contesto tecnologico, si riferisce spesso a un insieme di condizioni software e hardware in cui un sistema opera. Per esempio, un `development environment` è il luogo dove i programmatori scrivono e testano il loro codice.
