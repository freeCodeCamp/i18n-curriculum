# --description--

HTML5 представив кілька нових елементів, які дають розробникам більше варіантів і водночас включають функції доступності. Серед цих тегів є `main`, `header`, `footer`, `nav`, `article` та `section`.

За замовчуванням браузер відтворює ці елементи подібно до скромного `div`. Однак використання їх у відповідних місцях додає додаткове значення вашій розмітці. Саме ім’я тегу може вказувати на тип інформації, яку він містить, що додає семантичне значення цьому вмісту. Допоміжні технології можуть отримувати доступ до цієї інформації, щоб надавати кращі варіанти підсумку сторінки або навігації для своїх користувачів.

Елемент `main` використовується для обгортання (як ви здогадалися) основного вмісту, і на сторінці має бути лише один такий елемент. Він призначений для охоплення інформації, пов’язаної з центральною темою вашої сторінки. Він не повинен включати елементи, що повторюються на сторінках, як-от навігаційні посилання або банери.

Тег `main` також має вбудовану функцію орієнтира, яку допоміжні технології можуть використовувати для швидкої навігації до основного вмісту. Якщо ви коли-небудь бачили посилання «Перейти до основного вмісту» у верхній частині сторінки, використання тегу `main` автоматично надає цю функціональність допоміжним пристроям.

# --instructions--

Кіт Кемпер має великі ідеї для сторінки зі своїми ніндзя-зброями. Допоможіть йому налаштувати розмітку, додавши відкриваючий і закриваючий теги `main` між `header` та `footer` (розглянуті в інших завданнях). Поки що залиште теги `main` порожніми.

# --hints--

Ваш код має містити один тег `main`.

```js
assert.lengthOf(document.querySelectorAll('main'),1);
```

Теги `main` мають бути між закриваючим тегом `header` та відкриваючим тегом `footer`.

```js
assert.match(code,/<\/header>\s*?<main>\s*?<\/main>/gi);
```
