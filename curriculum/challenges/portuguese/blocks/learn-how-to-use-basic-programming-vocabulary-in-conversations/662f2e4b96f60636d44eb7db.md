# --description--

Uma `class` é um conceito fundamental, especialmente em `object-oriented programming`.Ele funciona como um modelo para criar objetos.Uma `class` define as propriedades (atributos) e comportamentos (`methods`) que os objetos criados a partir da `class` podem ter.

Por exemplo, uma classe `Person` pode ter propriedades como `name` e `age` e comportamentos como `talk()` e `walk()`.

Um `method` é uma função que está associada a um objeto. `Methods` definem os comportamentos ou ações que um objeto criado a partir de uma `class` pode executar.Eles são como instruções que dizem ao objeto o que fazer.

Por exemplo, em uma `Calculator class`, um `method` pode ser `add(number1, number2)`, que soma dois números.
