# --explanation--

`Could` 和 `might` 是用來描述可能性的情態動詞。

`Could` 表示某事是可能的，但不確定。例如：

`If we don't fix this bug, it could cause a system crash.`－系統有可能會停止運作。

`Might` 表示某事有可能，但比 `could` 的確定性低。 例如：

`We might need to rewrite some parts of the code。`－可能需要進行修改。
