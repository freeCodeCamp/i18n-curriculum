# --explanation--

`To touch on` puede significar mencionar brevemente o tratar un tema. A menudo se usa cuando se aborda un asunto sin entrar en muchos detalles. Por ejemplo:

- `The speaker touched on the new software update during the meeting.` - El orador mencionó brevemente la actualización del software pero no entró en detalles.
