# --explanation--

Toms Antwort, `We haven't been receiving as many negative comments`, verwendet das `Present Perfect Continuous` in der Verneinung. Diese Zeitform wird mit `haven't/hasn't + been + verb ending in -ing` gebildet und beschreibt Handlungen, die über einen Zeitraum hinweg nicht stattgefunden haben. In diesem Fall zeigt sie, dass im Laufe der Zeit weniger negative Kommentare eingegangen sind, was auf eine mögliche Verbesserung hinweist.
