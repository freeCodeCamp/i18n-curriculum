# --interactive--

略語とは、単語やフレーズを短縮した形のことです。例えば、「Dr.」は「doctor（医師）」の略語です。

略語には一般的に2つの形があります：頭字語とイニシャリズムです。

頭字語は、フレーズの各単語の最初の文字を取って作られた単語です。

GUIは頭字語の例です。これはGraphical User Interfaceの略で、各単語の最初の文字G、U、Iを取ってGUIという頭字語になります。

イニシャリズムは、フレーズの各単語の最初の文字を取って作られますが、それぞれの文字を個別に発音します。

例えば、HTMLはイニシャリズムで、HyperText Markup Languageの略であり、H、T、M、Lと一文字ずつ発音します。

頭字語とイニシャリズムはどちらも略語の一種です。違いは、頭字語は単語として発音され、イニシャリズムは個々の文字として発音されることです。

これらは特に、よく知られていて文脈で理解しやすい場合に、より簡潔な文章を書くのに非常に役立ちます。

HTMLで頭字語やイニシャリズムのような略語を表示する必要がある場合、`abbreviation`要素がまさに役立ちます。最初に使うときは必ずその完全な意味を説明してください。その後、`abbreviation`要素を使って強調し、詳細を提供できます。

以下は、`HTML is the foundation of the web`という文を含む段落の例です。

:::interactive_editor

```html
<p><abbr>HTML</abbr> is the foundation of the web.</p>
```

:::

イニシャリズムのHTMLは`abbreviation`要素内にあります。ブラウザでは見た目はほとんど変わりません。通常のテキストのように見えます。`abbreviation`要素は裏で役立つコンテキストを提供していますが、ユーザーにはイニシャリズムが普通のテキストとして表示されます。

このイニシャリズムの意味をユーザーに理解してもらいたい場合は、`title`属性で完全な形を表示できます。

`title`属性は任意ですが、含める場合は略語、頭字語、イニシャリズムの人間が読める説明でなければなりません。

先ほどの例に`title`属性を追加してみましょう。これはイニシャリズムの展開形である`HyperText Markup Language`になります。

:::interactive_editor

```html
<p><abbr title="HyperText Markup Language">HTML</abbr> is the foundation of the web.</p>
```

:::

通常、この属性を追加すると`abbreviation`要素のスタイルが変わります。具体的なスタイルはブラウザによって異なります。点線の下線を表示するブラウザもあれば、内容を小文字の大文字（スモールキャップ）に変えるブラウザもあります。ユーザーが略語にカーソルを合わせると、完全な形がツールチップとして表示されます。

ウェブページ上のすべての略語に必ずしも`abbreviation`要素を使う必要はありませんが、不明瞭なものや追加の説明が必要なものには使うことをお勧めします。

情報と見た目のバランスをうまく取り、テキストがごちゃごちゃしないようにしながら、明確で簡潔にすることが大切です。
