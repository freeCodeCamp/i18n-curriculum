# --explanation--

La frase creada usando `get` seguida de un adjetivo describe un cambio de estado o volverse de cierta manera. Por ejemplo:

- `They always get excited when a new Python update is released.` - `Get excited` significa volverse entusiasta o emocionado. Esto quiere decir que siempre están llenos de entusiasmo cuando aparece una nueva actualización de Python.

- `He got tired after working on the project all night.` - Esto significa que no estaba cansado antes, pero ahora está cansado, después de toda una noche trabajando en el proyecto.

En la oración de Maria, ella está preguntando si Brian recuerda el entusiasmo que solían sentir por las nuevas tecnologías mientras trabajaban juntos.
