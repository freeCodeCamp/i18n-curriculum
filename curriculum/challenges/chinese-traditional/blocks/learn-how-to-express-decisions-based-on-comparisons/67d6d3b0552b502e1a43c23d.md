# --explanation--

`Build the software` 是指從零開始創建並開發一個程式或應用程式。它包含編碼、設計和測試軟體。為例：

`The company wants to build the software from the ground up using Python。`－這表示他們將完全從頭開始使用 Python 來開發軟體。  

`In-house` 是指工作在公司內部完成，而不是聘請外部團隊或將專案外包。例如：

`Our team decided to do front-end development in-house instead of hiring an external agency.`－這表示公司的員工將負責前端開發，而不是外包給外部機構。
