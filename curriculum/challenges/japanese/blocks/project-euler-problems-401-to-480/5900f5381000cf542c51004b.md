# --description--

ユークリッド平面上で、アリは点 $A(0, 1)$ から整数 $d$ に対して点 $B(d, 1)$ まで移動します。

各ステップで、点 ($x_0$, $y_0$) にいるアリは、$x_1 ≥ 0$ かつ $y_1 ≥ 1$ を満たす格子点 ($x_1$, $y_1$) のうちの1つを選び、一定の速度 $v$ で ($x_1$, $y_1$) へ直進します。$v$ の値は $y_0$ と $y_1$ によって次のように決まります：

- $y_0 = y_1$ の場合、$v$ の値は $y_0$ と等しくなります。
- $y_0 ≠ y_1$ の場合、$v$ の値は $\frac{y_1 - y_0}{\ln y_1 - \ln y_0}$ と等しくなります。

左の画像は $d = 4$ の場合の可能な経路の一つです。まずアリは $A(0, 1)$ から速度 $\frac{3 - 1}{\ln 3 - \ln 1} ≈ 1.8205$ で $P_1(1, 3)$ へ移動します。必要な時間は $\frac{\sqrt{5}}{1.820} ≈ 1.2283$ です。

$P_1(1, 3)$ から $P_2(3, 3)$ へは速度 3 で移動するので、必要な時間は $\frac{2}{3} ≈ 0.6667$ です。$P_2(3, 3)$ から $B(4, 1)$ へは速度 $\frac{1 - 3}{\ln 1 - \ln 3} ≈ 1.8205$ で移動するため、必要な時間は $\frac{\sqrt{5}}{1.8205} ≈ 1.2283$ です。

したがって合計の必要時間は $1.2283 + 0.6667 + 1.2283 = 3.1233$ となります。

右の画像は別の経路です。合計の必要時間は $0.98026 + 1 + 0.98026 = 2.96052$ と計算されます。これは $d = 4$ の場合の最速経路であることが示されています。

<img alt="two possible paths for d = 4" src="https://cdn.freecodecamp.org/curriculum/project-euler/an-ant-on-the-move.jpg" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$F(d)$ をアリが最速経路を選んだ場合の合計必要時間とします。例えば、$F(4) ≈ 2.960\\,516\\,287$ です。$F(10) ≈ 4.668\\,187\\,834$ および $F(100) ≈ 9.217\\,221\\,972$ であることも確認できます。

$F(10\\,000)$ を求めてください。答えは小数点以下9桁に丸めてください。
