# --interactive--

طريقة `toString()` هي ميزة أساسية في JavaScript تحول القيمة إلى تمثيلها كسلسلة نصية. يمكنك استخدام هذه الطريقة للأرقام والقيم المنطقية والمصفوفات والكائنات. من أكثر استخدامات `toString()` شيوعًا تحويل الرقم إلى تمثيله كسلسلة نصية. إليك مثالًا:

:::interactive_editor

```js
const num = 10;
console.log(num.toString()); // "10"
```

:::

تقبل هذه الطريقة أساس عد اختياري وهو رقم من `2` إلى `36`. هذا الأساس يمثل النظام العددي، مثل الأساس 2 للثنائي أو الأساس 8 للثماني. إذا لم يُحدد الأساس، فإنه يكون افتراضيًا 10، وهو النظام العشري. إليك مثالًا على تمرير `2` كمعلمة إلى طريقة `toString()`:

:::interactive_editor

```js
const num = 10;
console.log(num.toString(2)); // "1010"
```

:::

النتيجة ستكون `1010` وهو التمثيل الثنائي للرقم العشري `10`.

يمكنك أيضًا استخدام طريقة `toString()` لتحويل المصفوفات والكائنات إلى سلاسل نصية. للمصفوفات تنفيذ مخصص لطريقة `toString()` يحول كل عنصر إلى سلسلة نصية ويربطهم بفواصل. إليك مثالًا:

:::interactive_editor

```js
const arr = [1, 2, 3];
console.log(arr.toString()); // "1,2,3"
```

:::

في هذا المثال، تم ربط كل عناصر المصفوفة معًا لتكوين السلسلة النصية `1,2,3`.

عند استخدام طريقة `toString()` مع الكائنات، لن تكون النتيجة نسخة نصية من خصائص الكائن.

:::interactive_editor

```js
const person = {
  name: "John",
  age: 30,
  isStudent: true
};

console.log(person.toString()); // "[object Object]"
```

:::

في هذا المثال، ستكون النتيجة التمثيل النصي الافتراضي للكائن وهو `[object Object]`. للحصول على نسخة نصية من خصائص كائن `person`، ستحتاج إلى استخدام `JSON.stringify()` التي ستتعلم عنها أكثر في الدروس القادمة.

في الختام، تُستخدم طريقة `toString()` لتحويل القيم إلى سلاسل نصية. فهم كيفية عملها مع أنواع البيانات المختلفة وكيفية تخصيصها لكائناتك الخاصة يمكن أن يعزز بشكل كبير قدرتك على معالجة البيانات وعرضها في تطبيقات JavaScript الخاصة بك.
