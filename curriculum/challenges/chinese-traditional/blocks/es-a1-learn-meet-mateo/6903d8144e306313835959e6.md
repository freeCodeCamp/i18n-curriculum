# --description--

Mateo 正在用西班牙語使用以下結構，搭配動詞 `tener`（「有」）來說明他的年齡：

`Tengo` + 數字 字、單字 + `años`。

請記住，西班牙語中 21 到 29 的數字以 `veinti-` 開頭，後面接對應的數字。例如：

25 是 `veinticinco`。這是透過移除 `e` 並在數字 `cinco` 前添加 `i` 所得到的結果。

西班牙語中 1 到 9 的數字是：

| Number | Spanish Word |
| :---: | :---: |
| 1 | `uno` |
| 2 | `dos` |
| 3 | `tres` |
| 4 | `cuatro` |
| 5 | `cinco` |
| 6 | `seis` |
| 7 | `siete` |
| 8 | `ocho` |
| 9 | `nueve` |
