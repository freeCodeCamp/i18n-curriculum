# --interactive--

Il selettore universale (`*`) è un tipo speciale di selettore CSS che corrisponde a qualsiasi elemento nel documento.

Spesso viene usato per definire lo stile di tutti gli elementi nella pagina, cosa utile per resettare o normalizzare gli stili tra diversi browser.

Il selettore universale può essere usato per selezionare tutti gli elementi all’interno di un contesto specifico o globalmente in tutto il documento.

Ecco un esempio di utilizzo del selettore universale per impostare il `margin` e la `padding` per l’intero documento HTML:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1>Heading element</h1>
<p>example paragraph element</p>
```

```css
* {
  margin: 0;
  padding: 0;
}
```

:::

In questo esempio di codice, il selettore `*` resetta il margine e la spaziatura interna di tutti gli elementi a zero, una tecnica comune usata nei reset CSS.

Il selettore universale ha il valore di specificità più basso di qualsiasi selettore. Contribuisce con 0 a tutte le parti del valore di specificità `(0, 0, 0, 0)`.

Questo significa che qualsiasi altro selettore, inclusi selettori di tipo, di classe, ID e stili in linea, sovrascriverà gli stili impostati dal selettore universale.

Diamo un’occhiata al seguente esempio HTML e CSS:

:::interactive_editor

```html
<head>
  <style>
    * {
      color: blue;
    }
    p {
      color: red;
    }
    .highlight {
      color: green;
    }
    #unique {
      color: purple;
    }
  </style>
</head>
<body>
  <p id="unique" class="highlight">This text has multiple styles applied.</p>
</body>
```

:::

Il selettore universale `*` imposta il colore del testo su `blue` per tutti gli elementi.

Tuttavia, il selettore di tipo `p` sovrascrive questo impostando il colore del testo su `red` specificamente per gli elementi `p`.

Se un elemento ha la classe `highlight`, il selettore di classe ha la precedenza, cambiando il colore del testo in `green`.

Infine, il selettore ID `#unique`, che ha la specificità più alta, sovrascriverà tutti gli altri, impostando il colore del testo su `purple`.
