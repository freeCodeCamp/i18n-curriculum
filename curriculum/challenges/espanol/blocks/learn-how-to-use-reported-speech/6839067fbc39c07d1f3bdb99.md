# --explanation--

`Forward` se usa para hablar sobre acciones futuras o el tiempo a partir de ahora. Por ejemplo:

`Going forward, we will hold weekly meetings.` – Esto significa que a partir de ahora, las reuniones semanales se llevarán a cabo.

`To happen` significa que un evento o situación ocurra o tenga lugar. Por ejemplo:

`If problems happen, please report them immediately.` – Esto significa que si ocurren problemas, debes informarlo de inmediato.
