# --interactive--

In den vorherigen Lektionen haben Sie gelernt, wie man mit Ereignissen arbeitet, indem Sie die `addEventListener()` Methode verwenden. Aber es gibt eine andere, nicht empfohlene Möglichkeit, mit Ereignissen in JavaScript zu arbeiten.

Inline event handler sind spezielle Attribute an einem HTML-Element, die verwendet werden, um JavaScript-Code auszuführen, wenn ein Ereignis eintritt.

Hier ist ein Beispiel für ein `button` Element mit einem inline click event handler:

:::interactive_editor

```html
<button onclick="alert('Hello World!')">Show alert</button>
```

:::

Wenn der Benutzer auf die Schaltfläche klickt, wird die `alert` Funktion aufgerufen und ein Warnfenster mit der Nachricht `Hello World!` angezeigt.

Eine andere Möglichkeit, inline event handler zu verwenden, besteht darin, eine Funktion aufzurufen, die in einem `script` Tag im HTML-Dokument definiert ist.

Hier ist ein Beispiel, wie eine Funktion namens `changeBgColor` definiert und von einem inline click event handler aufgerufen wird:

:::interactive_editor

```html
<script>
  function changeBgColor() {
    document.body.style.backgroundColor = "lightblue";
  }
</script>

<button onclick="changeBgColor()">Change background color</button>
```

:::

Wenn der Benutzer auf die Schaltfläche klickt, wird die `changeBgColor` Funktion aufgerufen und die Hintergrundfarbe der Seite wird hellblau geändert. Obwohl es möglich ist, inline event handler auf diese Weise zu verwenden, gilt dies nicht als bewährte Vorgehensweise.

Ein Grund dafür ist, dass inline event handler nur verwendet werden können, um einen einzelnen Ereignis-Listener an ein Element anzuhängen. Wenn Sie mehrere Ereignis-Listener an dasselbe Element anhängen möchten, müssen Sie `addEventListener()` verwenden. Ein weiterer Grund ist, dass inline event handler HTML- und JavaScript-Code vermischen, was Ihren Code schwerer lesbar und wartbar machen kann. Es ist besser, Ihren HTML-Code und JavaScript-Code getrennt zu halten, indem Sie `addEventListener()` verwenden, um Ereignis-Listener an Elemente anzuhängen.

Inline event handler werden in modernem JavaScript nicht empfohlen. Daher ist es eine bewährte Vorgehensweise, bei der Arbeit mit Ereignissen in JavaScript bei der `addEventListener()` Methode zu bleiben.
