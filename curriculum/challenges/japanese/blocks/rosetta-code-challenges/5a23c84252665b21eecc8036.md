# --description--

この課題は、文字列から制御コードと拡張文字を取り除くことです。解決策では、以下の各結果を達成する方法を示してください。制御コードと拡張文字が取り除かれた文字列。ASCIIでは、制御コードは10進数で0から31および127のコードを持ちます。ASCIIベースのシステムでは、制御コードが取り除かれると、結果の文字列はASCIIテーブルの10進数32から126の範囲内のすべての文字を含みます。非ASCIIベースのシステムでは、この課題の目的上、ASCIIテーブルの10進数32から126の範囲内に対応するグリフがない文字を拡張文字とみなします。

# --solutions--

```js
function strip(s) {
  return s
    .split('')
    .filter(function(x) {
      var n = x.charCodeAt(0);

      return 31 < n && 127 > n;
    })
    .join('');
}
```
