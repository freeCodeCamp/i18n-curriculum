# --explanation--

`Should` використовується, щоб запропонувати або порекомендувати щось, але це не так категорично, як `must` або `have to`.

Джессіка також каже `it's not mandatory`, що означає, що для Марка не є суворою вимогою проводити регулярні аудити безпеки. Вона просто рекомендує це, бо це добра ідея — виявляти вразливості на ранньому етапі, але це не те, що він мусить робити.
