# --description--

일상생활에서 `right`와 `left`는 방향이나 면을 가리킵니다. 예를 들어, `right`할 때 `right` 쪽으로 움직입니다.

컴퓨터 마우스에서는 `right click`와 `left click`가 보통 다른 동작을 위해 `right` 또는 `left` 버튼을 누르는 것을 의미합니다. `right click`는 추가 옵션이나 메뉴를 여는 경우가 많고, `left click`는 항목을 선택하거나 탐색하는 데 사용됩니다.
