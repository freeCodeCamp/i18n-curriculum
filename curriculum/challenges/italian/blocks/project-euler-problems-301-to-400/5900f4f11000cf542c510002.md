# --description--

Considera tutti i punti reticolari ($a$, $b$, $c$) con $0 ≤ a, b, c ≤ N$.

Dall'origine $O(0, 0, 0)$ vengono tracciate tutte le righe verso gli altri punti reticolari. Sia $D(N)$ il numero di righe distinte di questo tipo.

Ti viene dato che $D(1\\,000\\,000) = 831\\,909\\,254\\,469\\,114\\,121$.

Trova $D({10}^{10})$. Fornisci come risposta le prime nove cifre seguite dalle ultime nove cifre.
