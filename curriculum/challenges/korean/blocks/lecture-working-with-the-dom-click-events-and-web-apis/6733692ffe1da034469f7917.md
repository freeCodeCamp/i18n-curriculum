# --interactive--

JavaScript 파일 내에서 HTML 요소의 스타일을 조작해야 할 때가 있습니다. 예를 들어, 사용자가 버튼을 클릭할 때 메뉴를 숨기거나 표시해야 할 경우가 그렇습니다.

이 수업에서는 `Element.style`와 `Element.classList` 속성을 사용해 스타일을 조작하는 방법을 배우세요.

`Element.style` 속성은 요소의 인라인 스타일을 나타내는 읽기 전용 속성입니다. 이 속성을 사용해 요소의 스타일을 가져오거나 설정할 수 있습니다.

다음은 ID가 `para`인 단락 요소의 예입니다:

```html
<p id="para">This is a paragraph</p>
```

`style` 속성을 사용해 단락 요소의 색상을 빨간색으로 변경할 수 있습니다:

:::interactive_editor

```html
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```js
const paraEl = document.getElementById("para");
paraEl.style.color = "red";
```

:::

`style` 속성은 `color`, `background-color`, `font-size`, `font-weight` 등 다양한 CSS 속성을 설정하는 데 사용할 수 있습니다.

스타일을 조작하는 또 다른 방법은 `Element.classList` 속성을 사용하는 것입니다. `classList` 속성은 읽기 전용이며, 요소에 클래스를 추가, 제거 또는 토글하는 데 사용할 수 있습니다. 몇 가지 예제를 살펴보겠습니다.

이 예제에서는 ID가 `highlight`인 단락 요소에 `para`라는 클래스를 추가할 것입니다. CSS에서는 `highlight` 클래스를 설정해 단락 요소의 배경색을 노란색으로 변경합니다. JavaScript에서는 `highlight` 메서드를 사용해 단락 요소에 `classList.add()` 클래스를 추가합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```css
.highlight {
  background-color: yellow;
}
```

```js
const paraEl = document.getElementById("para");
paraEl.classList.add("highlight");
```

:::

여러 클래스를 요소에 추가하려면 `classList.add()` 메서드에 인수로 전달하세요:

```js
classList.add("class1", "class2", "class3");
```

요소에서 클래스를 제거해야 할 경우, 다음과 같이 `classList.remove()` 메서드를 사용할 수 있습니다:

```js
classList.remove("highlight");
```

요소에서 클래스를 토글하려면 `classList.toggle()` 메서드를 사용하세요. 이 예제에는 ID가 `button`인 `toggle-btn` 요소와 ID가 `nav`인 `menu` 요소가 있습니다. CSS에는 메뉴에 대한 스타일과 `show`라는 클래스가 추가되어 있는데, 이 클래스는 `display`의 `.menu` 속성을 `block`로 변경합니다. JavaScript에서는 `button` 메서드를 사용해 메뉴 요소와 `getElementById()` 요소에 접근합니다. 그리고 `button` 요소에 이벤트 리스너를 추가해 버튼이 클릭될 때 `show` 요소에 `.menu` 클래스를 토글합니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button id="toggle-btn">Toggle Menu</button>

<nav id="menu" class="menu">
  <ul>
    <li>Home</li>
    <li>About</li>
    <li>Products</li>
  </ul>
</nav>
<script src="index.js"></script>
```

```css
.menu {
  display: none;
  background-color: lightgray;
  width: 50%;
  padding: 10px;
}

.menu.show {
  display: block;
}
```

```js
const menu = document.getElementById("menu");
const toggleBtn = document.getElementById("toggle-btn");

toggleBtn.addEventListener("click", () => menu.classList.toggle("show"));
```

:::

사용자가 버튼을 클릭하면 `show` 요소에 `.menu` 클래스가 추가되어 메뉴가 표시됩니다. 버튼을 다시 클릭하면 `show` 요소에서 `.menu` 클래스가 제거되어 메뉴가 숨겨집니다.

`style`, `add()`, `remove()`, `toggle()` 같은 메서드와 속성을 사용하면 JavaScript로 요소의 스타일을 쉽게 조작해 동적이고 인터랙티브한 웹 페이지를 만들 수 있습니다.
