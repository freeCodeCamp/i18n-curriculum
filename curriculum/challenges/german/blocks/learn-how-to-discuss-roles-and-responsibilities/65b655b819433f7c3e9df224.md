# --description--

Der Ausdruck `word is` wird verwendet, wenn Sie Informationen oder Gerüchte weitergeben, die Sie von anderen gehört haben. Es ist so, als würden Sie sagen `people say` oder `I've heard`.

Zum Beispiel, wenn jemand sagt, `Word is that the new update will make the program faster`, meint er, dass er gehört hat, das neue Update werde die Geschwindigkeit des Programms verbessern.
