# --description--
Mchezo wako utaendeshwa dhidi ya kompyuta. Utaandika kitendakazi kinachorudisha kwa bahati `"rock"`, `"paper"` au `"scissors"`.

Huna haja ya kuwa na wasiwasi kuhusu sehemu ya mbele ya mchezo. Utaandika tu mantiki ya mchezo. Fungua `script.js` na anza kufuata maelekezo.

**Hadithi za mtumizi:**

1. Unapaswa kuwa na kitendakazi kinachoitwa `getComputerChoice`.

1. Kitendakazi chako cha `getComputerChoice` kinapaswa kurudisha `"rock"`, `"paper"`, au `"scissors"` kwa bahati.

**Kidokezo:** Njia ya [Math.random](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/random) inarudisha nambari ya bahati ambayo ni kubwa au sawa na 0 na ndogo kuliko 1. Fikiria jinsi unavyoweza kutumia hii kurudisha mojawapo ya chaguzi nyingi kwa masharti.

Mchezo wako utaendeshwa na mchezaji wa binadamu. Utaandika kitendakazi kinachochukua chaguo la mtumizi na kurudisha hilo.

1. Tengeneza kitendakazi kinachoitwa `getHumanChoice`.

1. Andika msimbo ili `getHumanChoice` irudishe mojawapo ya chaguzi halali kulingana na kile mtumizi anachochagua.

**Kidokezo:** Tumia njia ya [prompt](https://developer.mozilla.org/en-US/docs/Web/API/Window/prompt) kupata ingizo la mtumizi.

Mchezo wako utafuatilia alama za mchezaji. Utaandika vigezo vya kuhesabu alama za mchezaji.

1. Tengeneza vigezo viwili vipya vinavyoitwa `humanScore` na `computerScore` katika wigo wa ulimwengu.

1. Anzisha vigezo hivyo kwa thamani ya `0`.

Mchezo wako utaendeshwa raundi kwa raundi. Utaandika kitendakazi kinachochukua chaguzi za mchezaji wa binadamu na kompyuta kama hoja, kucheza raundi moja, kuongeza alama ya mshindi wa raundi na kuandika tangazo la mshindi.

1. Tengeneza kitendakazi kipya kinachoitwa `playRound`.

1. Tambua vigezo viwili kwa `playRound`. Kigezo cha kwanza `humanChoice` na kigezo cha pili `computerChoice`. Tumia vigezo hivi viwili kuchukua chaguzi za binadamu na kompyuta kama hoja.

1. Fanya kigezo cha `humanChoice` cha kitendakazi chako kisizingatie herufi kubwa au ndogo ili wachezaji waweze kuingiza `"rock"`, `"ROCK"`, `"RocK"`, au mabadiliko mengine.

1. Andika msimbo wa kitendakazi chako cha `playRound` kinachorudisha thamani ya mfuatano wa herufi unaowakilisha mshindi wa raundi.

- Ikiwa ni sare, inapaswa kurudisha `"It's a tie!"`.

- Ikiwa mchezaji anashinda, inapaswa kurudisha `"You win! [player choice] beats [computer choice]"`.

- Ikiwa kompyuta inashinda, inapaswa kurudisha `"You lose! [computer choice] beats [player choice]"`.

1. Ongeza thamani ya vigezo vya `humanScore` au `computerScore` kulingana na mshindi wa raundi.

Mchezo wako utaendeshwa raundi 3. Utaandika kitendakazi kinachoitwa `playGame` kinachoitisha `playRound` kucheza raundi 3, kufuatilia alama, na kutangaza mshindi mwishoni.

1. Tengeneza kitendakazi kipya kinachoitwa `playGame`.

1. Tengeneza mzunguko unaocheza raundi 3 na kuitisha kitendakazi cha `playRound` kila mara kwa chaguo la binadamu na chaguo la kompyuta kama hoja.

1. Mwishoni mwa mchezo, rudisha mshindi wa mchezo kulingana na aliyeshinda raundi nyingi zaidi.

- Ikiwa mchezaji wa binadamu anashinda raundi nyingi zaidi kuliko kompyuta, rudisha ujumbe unaosema `"You win the game!"`.

- Ikiwa mchezaji wa kompyuta anashinda raundi nyingi zaidi kuliko binadamu, rudisha ujumbe unaosema `"You lose the game!"`.

```js
const hand = ['rock', 'paper', 'scissors'];


let computerScore = 0;
let humanScore = 0;

const getComputerChoice = () => {
  return hand[Math.floor(Math.random() * hand.length)];
}

const getHumanChoice = () => {
  return prompt();
}

const playRound = (humanChoice, computerChoice) => {
  humanChoice = humanChoice.toLowerCase()

  const tie = "It's a tie!"
  const win = `You win! ${humanChoice} beats ${computerChoice}`
  const lose = `You lose! ${computerChoice} beats ${humanChoice}`


  if (humanChoice == 'rock') {
    if (computerChoice == 'rock') {
      computerScore++;
      humanScore++;
      return tie;
    } else if (computerChoice == 'paper') {
      computerScore++;
      return lose;
    } else {
      humanScore++;
      return win;
    }
  } else if (humanChoice == 'paper') {
    if (computerChoice == 'rock') {
      humanScore++;
      return win;
    } else if (computerChoice == 'paper') {
      computerScore++;
      humanScore++;
      return tie;
    } else {
      computerScore++;
      return lose;
    }
  } else if (humanChoice == 'scissors') {
    if (computerChoice == 'rock') {
      computerScore++;
      return lose;
    } else if (computerChoice == 'paper') {
      humanScore++;
      return win;
    } else {
      computerScore++;
      humanScore++;
      return tie;
    }
  }
}

const playGame = () => {
  for(let i = 0; i < 3; i++){
    playRound(getHumanChoice(), getComputerChoice())
  }

  return humanScore > computerScore ? "You win the game!" : "You lose the game!"
}
```
