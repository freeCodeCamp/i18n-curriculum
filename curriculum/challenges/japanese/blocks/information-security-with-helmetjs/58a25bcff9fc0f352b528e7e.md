# --description--

このプロジェクトは、<a href="https://github.com/freeCodeCamp/boilerplate-bcrypt/" target="_blank" rel="noopener noreferrer nofollow">GitHub</a>からクローンした以下のスタータープロジェクトを基に構築されています。

同期的にハッシュ化するのも簡単ですが、サーバー側でコストが高い場合や頻繁にハッシュ化を行う場合は遅延が発生することがあります。この方法でのハッシュ化は、次のように呼び出すだけで簡単にできます。

```js
var hash = bcrypt.hashSync(myPlaintextPassword, saltRounds);
```

このハッシュ化の方法をコードに追加し、結果をコンソールにログ出力してください。変数はすでにサーバー側で定義されているので調整は不要です。同じパスワードを非同期関数でハッシュ化しても、コンソールに表示される結果が異なることに気づくかもしれません。これは、ハッシュの3番目の文字列の最初の22文字に見られるように、ソルトが毎回ランダムに生成されるためです。次に、新しい同期ハッシュとパスワードの入力を比較するには、compareSyncメソッドを使います。

```js
var result = bcrypt.compareSync(myPlaintextPassword, hash);
```

結果はブール値のtrueまたはfalseになります。
