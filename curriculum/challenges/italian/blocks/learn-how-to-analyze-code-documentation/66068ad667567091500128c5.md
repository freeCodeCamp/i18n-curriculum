# --description--

Una `library` è come una grande libreria con tanto codice che i programmatori possono usare. Pensala come un ricettario con molte ricette (funzioni) che puoi usare per cucinare (programmare) piatti diversi (progetti).

Per esempio, una `library` nella programmazione potrebbe avere una funzione per creare grafici. Invece di scrivere codice da zero, puoi usare questa funzione, come seguire una ricetta per fare una torta.
