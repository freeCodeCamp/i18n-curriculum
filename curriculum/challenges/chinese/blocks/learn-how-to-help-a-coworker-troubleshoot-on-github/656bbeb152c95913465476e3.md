# --description--

编程中的`编译`是指将人们编写的代码转换成计算机能够理解并使用的格式。 

就像把一种语言写的食谱翻译成另一种语言，以便别人可以烹饪这道菜。 

例如，`当我们编译 Java 代码时，它会转换成计算机可以运行的形式。`
