# --description--

你可能已經注意到，在本課程的所有範例中，你都會縮排任何位於其他元素內的元素。這稱為巢狀元素。

當你將元素巢狀於其他元素中時，你會在它們之間創建父元素和子元素的關係。巢狀的元素是子元素，而它們所巢狀的元素是父元素。

在以下範例中，`body` 元素是父元素，而段落是子元素：

```html
<html>
  <head>
  </head>
  <body>
    <p>Lorem ipsum dolor sit amet.</p>
  </body>
 </html>
```

就像在人際關係中，HTML 父元素可以有許多子元素。處於相同巢狀層級的元素被視為兄弟元素。

例如，以下程式碼中的兩個段落是兄弟節點，因為它們都是 body 頁籤的子節點，且彼此處於相同的巢狀層級：

```html
<html>
  <head>
  </head>
  <body>
    <p>Lorem ipsum dolor sit amet.</p>
    <p>Ut enim ad minim veniam.</p>
  </body>
 </html>
```

你使用縮排來讓巢狀的層級對你自己以及未來將會處理你 HTML 的其他開發者清楚且易讀。建議將任何子元素縮排兩個空格。

元素之間的父元素、子元素和兄弟元素關係，在你開始使用 CSS 為你的 HTML 設計樣式以及用 JavaScript 添加行為時，將變得更加重要。不過，目前只需了解元素之間關係的區別以及用來描述這些關係的術語即可。
