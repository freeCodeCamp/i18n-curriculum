Ela usa uma entonação ascendente para mostrar que é uma pergunta de confirmação.

Quando alguém repete uma informação com entonação de pergunta, geralmente é para verificar se está correta.

Nessa situação, a outra pessoa pode confirmar ou negar a informação. Por exemplo:

- `Sí, correcto.` (O que a pessoa disse está correto).

- `No, es ...` + correção (O que a pessoa disse está errado).
