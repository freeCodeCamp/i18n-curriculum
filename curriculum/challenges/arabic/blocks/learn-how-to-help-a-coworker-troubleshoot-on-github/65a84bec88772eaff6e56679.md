# --description--

تخيل أنك تعمل على مشروع وجربت حلولًا مختلفة مع مرور الوقت. للتعبير عن هذا الجهد المستمر، يمكنك استخدام زمن المضارع التام. هذا الزمن يربط بين الأفعال الماضية واللحظة الحالية.

يتكون زمن المضارع التام باستخدام الفعل المساعد `have` أو `has` متبوعًا بالتصريف الثالث للفعل الرئيسي. على سبيل المثال، `I have been working on this for a long time` أو `he has worked on this for a long time`.

لنتمرن على استخدام زمن المضارع التام للحديث عن الأفعال التي بدأت في الماضي وما زالت ذات صلة الآن.
