# --description--

To `make sure` significa verificar algo cuidadosamente para ter certeza de que está correto ou que algo foi feito.Por exemplo, você pode dizer `I wanted to make sure the door is locked`, quando você verifica a trava para confirmar que está segura.

`Choices` são as diferentes opções que você pode escolher.Por exemplo, se você estiver escolhendo um sabor de sorvete, as `choices` podem ser chocolate, baunilha e morango.Isso significa que você tem mais de uma possibilidade para selecionar.
