### --question--

#### --text--

以下哪种方式是创建 `promise` 的正确方法？

#### --distractors--

```js

  setTimeout(() => {
    resolve('Data received successfully');
  }, 2000);
});
```

---

```js
const promise = set Promise((resolve, reject) => {
  setTimeout(() => {
    resolve('Data received successfully');
  }, 2000);
});
```

---

```js
const promise = put Promise((resolve, reject) => {
  setTimeout(() => {
    resolve('Data received successfully');
  }, 2000);
});
```

#### --answer--

```js

  setTimeout(() => {
    resolve('Data received successfully');
  }, 2000);
});
```
