# --interactive--

JavaScriptでは、コールバック関数とは、別の関数に引数として渡され、その外側の関数が特定のタイミングで呼び出す関数のことです。

この概念は、JavaScriptの多くの側面を理解する上で基本となり、`forEach` メソッドの動作を理解するのにも重要です。

まずは、シンプルな文脈でコールバック関数とは何かを理解しましょう。

関数があり、その中で何かを複数回行いたいとします。すべてのコードを一つの大きな関数に書く代わりに、特定の処理を行いたいときに実行される第二の関数（コールバック）を渡すことができます。これにより、より柔軟でモジュール化されたコードが可能になります。

次に、`forEach` メソッドを紹介します。`forEach` はJavaScriptの配列に組み込まれているメソッドです。配列の各要素を繰り返し処理し、各要素に対して操作を行うことができます。実行したい操作は、`forEach` に渡すコールバック関数で定義します。

以下は、`forEach` がコールバック関数とどのように動作するかの例です。

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];

numbers.forEach(function(number) {
  console.log(number * 2);
});
```

:::

この例では、数値の配列があります。この配列に対して`forEach` メソッドを使い、引数としてコールバック関数を`forEach` に渡しています。

このコールバック関数は1つのパラメータを受け取り、それが配列の各要素を表します。`forEach` メソッドは、配列の各要素に対してこのコールバック関数を一度ずつ呼び出します。

この場合のコールバック関数は、各数値に`2` を掛けて結果をログに出力します。したがって、このコードを実行すると以下のように出力されます。

```md
2
4
6
8
10
```

コールバック関数は配列の各要素に対して順番に一度ずつ呼び出されることを理解することが重要です。`forEach` はループ処理を自動で行うため、自分で`for` ループを書く必要はありません。

また、コールバックとしてアロー関数を使うこともでき、コードをさらに簡潔にできます。

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];
numbers.forEach(number => console.log(number * 2));
```

:::

これは前の例とまったく同じ動作をしますが、コードが短くなっています。

`forEach` のコールバック関数は実際には最大3つの引数を取ることができます：現在の要素、現在の要素のインデックス、そして`forEach` が呼び出された配列です。

以下は3つすべてを使った例です。

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];
numbers.forEach((number, index, array) => {
  console.log(`Element ${number} is at index ${index} in array ${array}`);
});
```

:::

これにより、各要素、そのインデックス、元の配列に関する情報がログに出力されます。

コールバック関数や`forEach` のようなメソッドを理解することは、JavaScriptの学習を進める上で重要です。これらは特に今後のレッスンで学ぶ非同期プログラミングなど、より高度な概念の基礎となります。
