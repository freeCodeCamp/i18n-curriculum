# --explanation--

问题 `Did you check that?` 是在询问某人是否核实或确认了某事。 

在这个上下文中，Mark 想知道 Sarah 是否查看了电源线的连接以确保其牢固。其他示例：

- `Did you check that the windows are closed?`——这意味着询问某人是否确保窗口已安全关闭。

- `Did you check that the light is off?` - 这意味着询问某人是否确认灯已关闭。
