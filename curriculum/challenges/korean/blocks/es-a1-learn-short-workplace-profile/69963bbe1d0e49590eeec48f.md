# --explanation--

`Su número es`는 "그의 번호는"이라는 뜻입니다.

스페인어에서는 전화번호를 말할 때 숫자를 묶어서 말하는 것이 일반적입니다. 사람마다 숫자를 묶는 방식에 따라 같은 전화번호를 다르게 말할 수 있습니다.

예를 들어, `8890` 같은 숫자를 공식적인 자리에서 말할 때는 개별 숫자 또는 다음과 같이 묶어서 들을 수 있습니다:

- `Ochenta y ocho, noventa` — "팔십팔, 구십".
- `Ocho, ocho, nueve, cero` — "팔, 팔, 구, 영".

이번 경우에는 Ángela가 첫 번째 방식인 숫자를 묶어서 말하는 방식을 선택했습니다.
