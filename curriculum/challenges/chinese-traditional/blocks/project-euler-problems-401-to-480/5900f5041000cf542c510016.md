# --description--

如果我們計算 $0 ≤ a ≤ 5$ 時的 $a^2\bmod 6$，結果為：0、1、4、3、4、1。

滿足 $a^2 ≡ a\bmod 6$ 的最大值為 $4$。

我們稱 $M(n)$ 為最大的 $a < n$ 值，使得 $a^2 ≡ a (\text{mod} n)$。所以 $M(6) = 4$。

求 $\sum M(n)$，其中 $1 ≤ n ≤ {10}^7$。
