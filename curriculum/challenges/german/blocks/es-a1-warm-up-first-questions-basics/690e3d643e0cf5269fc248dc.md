# --explanation--

Das Wort `Salvadoreña` ist die weibliche Form der Nationalität einer Person aus El Salvador.

Nationalitäten müssen im Geschlecht mit der Person übereinstimmen, die sie beschreiben. Zum Beispiel:

| Geschlecht | Nationalität | Beispiel |
| :--- | :--- | :--- |
| **Weiblich** | `Salvadoreña` | `Ella es salvadoreña.` („Sie ist Salvadorianerin“.) |
| **Männlich** | `Salvadoreño` | `Él es salvadoreño.` („Er ist Salvadorianer“.) |
