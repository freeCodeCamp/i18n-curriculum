# --description--

$ABCD$ هو رباعي أضلاع محدب بأضلاع صحيحة حيث $1 ≤ AB < BC < CD < AD$.

طول $BD$ عدد صحيح. $O$ هو منتصف $BD$. طول $AO$ عدد صحيح.

نسمي $ABCD$ رباعي أضلاع متكامل ثنائي الميل إذا كان $AO = CO ≤ BO = DO$.

على سبيل المثال، الرباعي التالي هو رباعي أضلاع متكامل ثنائي الميل: $AB = 19$، $BC = 29$، $CD = 37$، $AD = 43$، $BD = 48$ و $AO = CO = 23$.

<img alt="quadrilateral ABCD, with point O, a midpoint of BD" src="https://cdn.freecodecamp.org/curriculum/project-euler/biclinic-integral-quadrilaterals.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

لتكن $B(N)$ عدد رباعيات الأضلاع المتكاملة ثنائية الميل المختلفة $ABCD$ التي تحقق ${AB}^2 + {BC}^2 + {CD}^2 + {AD}^2 ≤ N$. يمكننا التحقق من أن $B(10\\,000) = 49$ و $B(1\\,000\\,000) = 38239$.

احسب $B(10\\,000\\,000\\,000)$.
