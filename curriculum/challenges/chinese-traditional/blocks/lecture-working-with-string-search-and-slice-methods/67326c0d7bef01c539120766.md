# --interactive--

在 JavaScript 中處理字串時，常常會需要檢查字串是否包含特定的子字串，子字串是該字串中較小的零件。

例如，你可能想在執行某些動作之前，檢查使用者的輸入是否包含特定的字或字元。一種達成此目的的方法是使用 `includes()` 方法。

`includes()` 方法用於檢查字串是否包含特定的子字串。如果在字串中找到該子字串，該方法會傳回 `true`，否則會傳回 `false`。

以下是基本語法：

```js
string.includes(searchValue);
```

對於語法，`searchValue` 是你想在字串中搜尋的子字串。以下是一個範例：

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("awesome");

console.log(result);  // true
```

:::

在此範例中，字串 `JavaScript is awesome!` 中找到字 `awesome`，所以 `includes()` 方法傳回 `true`。

重要的是要注意 `includes()` 方法是區分大小寫的。這表示需要字元的完全匹配，包括其大小寫。

例如：

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("Awesome");

console.log(result);  // false
```

:::

由於 `Awesome`（大寫的 `A`）與 `awesome`（小寫的 `a`）不相符，結果為 `false`。

你也可以使用 `includes()` 方法，透過提供第二個參數來檢查字串中從特定索引開始的子字串：

:::interactive_editor

```js
let text = "Hello, JavaScript world!";
let result = text.includes("JavaScript", 7);

console.log(result);  // true
```

:::

在這裡，對子字串 `JavaScript` 的搜尋從字串中的第 7 個位置開始，確保跳過此位置之前的任何字元。

`includes()` 方法只會傳回 `true` 或 `false` 的結果。它不會提供子字串在字串中位置或出現次數的資訊。如果你需要那種層級的細節，其他方法，例如 `indexOf()` 方法，可能會比較適合。
