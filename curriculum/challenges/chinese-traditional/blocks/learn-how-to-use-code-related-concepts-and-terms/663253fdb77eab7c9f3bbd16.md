# --description--

這個任務專注於關係代名詞 `that`。它用來介紹提供關於名詞更多細節的語句。

語句是句子中至少包含主詞和動詞的部分。它可以表達完整的想法或為主要句子添加額外的資訊。

代名詞 `that` 有助於更清楚地指定或定義某事，將其連接到描述或必要的細節。

例如，`She read the book that won the Pulitzer Prize last year.` 在這個語句中，`that won the Pulitzer Prize last year` 是由 `that` 介紹的關係語句。該語句描述或指定這本書，幫助區分它與其他書籍。在這裡，`that` 作為關係代名詞，將語句連接到名詞 `book`，並描述她讀的是哪一本特定的書。
