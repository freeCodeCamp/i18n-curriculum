## HTMLフォームの要素と属性

- **`form` 要素**：ユーザー入力のためのHTMLフォームを作成するために使います。
- **`action` 属性**：フォームデータを送信するURLを指定するために使います。
- **`method` 属性**：フォームデータを送信する際に使うHTTPメソッドを指定するために使います。最も一般的なメソッドは`GET`と`POST`です。

```html
<form method="value-goes-here" action="url-goes-here">
  <!-- inputs go inside here -->
</form>
```

- **`input` 要素**：ユーザー入力のための入力フィールドを作成するために使います。
- **`type` 属性**：入力フィールドの種類を指定するために使います。例：`text`、`email`、`number`、`radio`、`checkbox`など。
- **`placeholder` 属性**：ユーザーに入力内容のヒントを表示するために使います。
- **`value` 属性**：入力の値を指定するために使います。入力が`button`タイプの場合、`value`属性でボタンのテキストを設定できます。
- **`name` 属性**：入力フィールドに名前を代入するために使います。これはフォームデータが提出される際のキーとして機能します。ラジオボタンの場合、同じ`name`を付けるとグループ化され、そのグループ内で一つだけ選択可能になります。
- **`size` 属性**：ユーザーが入力する際に表示される文字数を定義するために使います。
- **`min` 属性**：`number`のような入力タイプで、入力フィールドに許可される最小値を指定するために使います。
- **`max` 属性**：`number`のような入力タイプで、入力フィールドに許可される最大値を指定するために使います。
- **`minlength` 属性**：入力フィールドに必要な最小文字数を指定するために使います。
- **`maxlength` 属性**：入力フィールドに許可される最大文字数を指定するために使います。
- **`required` 属性**：フォームを提出する前に入力フィールドの入力が必須であることを指定するために使います。
- **`disabled` 属性**：入力フィールドを無効にするために使います。
- **`readonly` 属性**：入力フィールドを読み取り専用にするために使います。

:::interactive_editor

```html
<!-- Text input -->
<input 
  type="text"
  id="name"
  name="name"
  placeholder="e.g. Quincy Larson" 
  size="20"
  minlength="5"
  maxlength="30"
  required
/>

<!-- Number input -->
<input 
  type="number"
  id="quantity"
  name="quantity"
  min="2"
  max="10"
  disabled
/>

<!-- Button -->
<input type="button" value="Show Alert" />
```

:::

- **`label` 要素**：入力フィールドのラベルを作成するために使います。
- **`for` 属性**：ラベルがどの入力フィールドに対応しているかを指定するために使います。
- **暗黙のフォーム関連付け**：入力フィールドを`label`要素で囲むことで、ラベルと関連付けることができます。

:::interactive_editor

```html
<form action="">
  <label>
    Full Name:
    <input type="text" />
  </label>
</form>
```

:::

- **明示的なフォーム関連付け**：`for`要素に`label`属性を使うことで、入力とラベルを関連付けることができます。

:::interactive_editor

```html
<form action="">
  <label for="email">Email Address: </label>
  <input type="email" id="email" />
</form>
```

:::

- **`button` 要素**：クリック可能なボタンを作成するために使います。ボタンには`type`属性を付けることもでき、ボタンがアクティブになったときの動作を制御します。例：`submit`、`reset`、`button`。

:::interactive_editor

```html
<button type="button">Show Form</button>
<button type="submit">Submit Form</button>
<button type="reset">Reset Form</button>
```

:::

- **`fieldset` 要素**：関連する入力をグループ化するために使います。
- **`legend` 要素**：入力グループを説明するキャプションを追加するために使います。

:::interactive_editor

```html
<!-- Radio group -->
<fieldset>
  <legend>Was this your first time at our hotel?</legend>

  <label for="yes-option">Yes</label>
  <input id="yes-option" type="radio" name="hotel-stay" value="yes" />

  <label for="no-option">No</label>
  <input id="no-option" type="radio" name="hotel-stay" value="no" />
</fieldset>

<!-- Checkbox group -->
<fieldset>
  <legend>
    Why did you choose to stay at our hotel? (Check all that apply)
  </legend>

  <label for="location">Location</label>
  <input type="checkbox" id="location" name="location" value="location" />

  <label for="price">Price</label>
  <input type="checkbox" id="price" name="price" value="price" />
</fieldset>
```

:::

- **フォーカス状態**：ユーザーが選択した入力フィールドの状態です。

## HTMLテーブルの要素と属性の操作

- **テーブル要素**：HTMLテーブルを作成するために使います。
- **テーブルヘッド（`thead`）要素**：HTMLテーブルのヘッダー部分をグループ化するために使います。
- **テーブル行（`tr`）要素**：HTMLテーブルの行を作成するために使います。
- **テーブルヘッダー（`th`）要素**：HTMLテーブルのヘッダーセルを作成するために使います。
- **テーブルボディ（`tbody`）要素**：HTMLテーブルの本文部分をグループ化するために使います。
- **テーブルデータセル（`td`）要素**：HTMLテーブルのデータセルを作成するために使います。
- **テーブルフット（`tfoot`）要素**：HTMLテーブルのフッター部分をグループ化するために使います。
- **`caption` 要素**：HTMLテーブルのタイトルを追加するために使います。
- **`colspan` 属性**：テーブルセルがまたがる列数を指定するために使います。

:::interactive_editor

```html
<table>
  <caption>Exam Grades</caption>

  <thead>
    <tr>
      <th>Last Name</th>
      <th>First Name</th>
      <th>Grade</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>Davis</td>
      <td>Alex</td>
      <td>54</td>
    </tr>

    <tr>
      <td>Doe</td>
      <td>Samantha</td>
      <td>92</td>
    </tr>

    <tr>
      <td>Rodriguez</td>
      <td>Marcus</td>
      <td>88</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <td colspan="2">Average Grade</td>
      <td>78</td>
    </tr>
  </tfoot>
</table>
```

:::

## HTMLツールの操作

- **HTMLバリデーター**：HTMLコードの構文が正しいかどうかをチェックするツールです。
- **DOMインスペクター**：ウェブページのHTML構造を調査・変更できるツールです。
- **開発者ツール**：ブラウザに組み込まれたウェブ開発者向けツールのセットで、ウェブページのデバッグ、プロファイル作成、解析を支援します。
