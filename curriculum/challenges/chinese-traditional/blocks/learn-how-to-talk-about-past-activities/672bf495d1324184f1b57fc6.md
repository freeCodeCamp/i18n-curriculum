# --explanation--

`Came in` 是過去簡單式，意思是某物「到達」或「被接收」。它常用來描述訊息、結果或回授被傳送或變得可用。 例如：

`The test results came in yesterday.`－這表示結果是在昨天收到的。
  
在這句話中，`since that first feedback came in` 意味著已收到最初的回授，標誌著他們進展的開始。
