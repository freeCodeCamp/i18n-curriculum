# --description--

`Occasionally` は、何かが時々起こることを意味します。常に起こるわけでもなく、全く起こらないわけでもなく、その中間の頻度です。例えば：

`I occasionally get a bug in my code.` - 時々コードにミスがありますが、あまり頻繁ではありません。

これらの副詞と、それが通常表す頻度を見てみましょう。

| 副詞         | 頻度      | 副詞       | 頻度      |
| ------------ | --------- | ---------- | --------- |
| Never        | 0%        | Often      | 70%       |
| Rarely       | 5%        | Generally  | 80%       |
| Seldom       | 15%       | Usually    | 90%       |
| Occasionally | 30%       | Always     | 100%      |
| Sometimes    | 50%       |            |           |
