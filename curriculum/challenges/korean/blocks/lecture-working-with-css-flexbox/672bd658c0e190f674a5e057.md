# --interactive--

플렉스 속성은 플렉스 컨테이너에 적용하여 자식 요소들의 분포를 결정하는 속성입니다. 가장 자주 사용하는 `flex-wrap`, `justify-content`, 그리고 `align-items`를 살펴보겠습니다.

먼저 `flex-wrap`부터 시작하겠습니다. 이 속성은 플렉스 항목들이 플렉스 컨테이너 내에서 사용 가능한 공간에 맞게 어떻게 감싸지는지를 결정합니다. `flex-wrap`는 세 가지 값, `nowrap`, `wrap`, 그리고 `wrap-reverse`을 가질 수 있습니다. `nowrap`가 기본값으로, 플렉스 항목들이 컨테이너 너비를 초과해도 새 줄로 감싸지지 않습니다.

아래 코드에는 세 개의 `div` 요소가 있습니다. `width`에 집중해 봅시다. 빨간 테두리로 표시된 `main` 컨테이너는 `width`가 `200px`이고, 세 개의 자식 `div` 요소들의 너비 합은 `width`가 `240px` (각각 `80px`)입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::


`div` 요소들의 너비가 컨테이너 너비를 초과하지만, 기본적으로는 사용 가능한 공간에 맞게 축소됩니다. 만약 컨테이너 너비를 초과할 때 감싸기를 원한다면, 플렉스 컨테이너에 `flex-wrap: wrap`를 설정하세요:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  flex-wrap: wrap;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

`div` 요소들은 컨테이너 너비를 초과하면 행으로 재배치됩니다. `flex-wrap: wrap-reverse`을 사용하면 플렉스 항목들을 역순으로 감쌀 수 있습니다. `flex-flow` 속성은 `flex-direction`과 `flex-wrap`의 단축 속성입니다. 이 예제에서는 `flex-direction`를 `column`으로, `flex-wrap`을 `wrap-reverse`으로 설정했습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  flex-flow: column wrap-reverse;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

좋습니다. 이제 `justify-content`에 대해 이야기해 보겠습니다. `justify-content`은 플렉스 컨테이너의 주축을 따라 자식 요소들을 정렬합니다. `flex-start`에 `justify-content` 값을 할당하면 플렉스 항목들이 주축의 시작 부분에 정렬됩니다. 이 주축은 수평 또는 수직일 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: flex-start;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

`justify-content: flex-end`을 사용하면 플렉스 항목들이 주축의 끝 부분에 정렬됩니다, 수평이든 수직이든 상관없습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: flex-end;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

주축을 따라 플렉스 항목들을 가운데 정렬하려면 `justify-content: center`를 사용하세요.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: center;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

주축을 따라 요소들을 고르게 분배하려면 `justify-content: space-between`를 사용하세요. 필요하면 플렉스 항목들 사이에 공간이 추가됩니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: space-between;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

`justify-content: space-around`은 주축 내에서 플렉스 항목들을 고르게 분배하며, 첫 번째 항목 앞과 마지막 항목 뒤에 공간을 추가합니다. 이 추가 공간은 인접한 항목들 사이 공간의 절반입니다. 만약 분배할 항목이 하나뿐이라면 가운데 정렬됩니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: space-around;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::


마지막으로 `justify-content: space-evenly`이 있습니다. 이 속성은 주축을 따라 항목들을 고르게 분배하며, 항목들 사이 공간과 첫 번째 및 마지막 요소 앞뒤 공간이 정확히 같습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: space-evenly;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

좋습니다. 이제 플렉스 항목들을 주축을 따라 분배하는 방법을 알았습니다. 하지만 교차 축을 따라 분배하고 싶을 수도 있습니다. 교차 축은 주축에 수직임을 기억하세요. `align-items` 속성으로 할 수 있습니다. 교차 축을 따라 항목들을 가운데 정렬하려면 플렉스 컨테이너에 `align-items: center`를 추가하세요:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: center;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

이 예제에서 플렉스 항목들은 기본적으로 수직인 교차 축을 따라 가운데 정렬됩니다. 교차 축이 수평이라면 수평으로 가운데 정렬됩니다. 반면 `align-items: flex-start`은 항목들을 교차 축의 시작 부분에 정렬합니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: flex-start;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

반대는 `align-items: flex-end`입니다. 이 속성은 플렉스 항목들을 교차 축의 끝 부분에 정렬합니다, 수직이든 수평이든 상관없습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: flex-end;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

교차 축을 따라 플렉스 항목들을 늘리려면 `align-items: stretch`를 사용하세요. 이 속성은 `width`이나 `height`가 설정되지 않았거나, 최솟값 `width`나 `height`만 있는 자동 크기 조절 요소에도 작동합니다. 플렉스 항목들은 교차 축 방향으로 컨테이너를 채우도록 늘어납니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box">Three</div>
</div>

```

```css
.container {
  display: flex;
  align-items: stretch; 
  border: 2px solid #444;
  height: 200px; 
}

.box {
  width: 100px;           
  border: 1px solid #888;
  background-color: lightblue;
}
```

:::

마지막으로, 개별 플렉스 항목에 교차 축에서 다른 정렬을 할당하려면 `align-self` 속성을 사용하세요. 예를 들어 `align-self: stretch`로 늘릴 수 있습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box special">Three</div>
</div>

```

```css
.container {
  display: flex;
  align-items: flex-start; 
  border: 2px solid #444;
  height: 200px; 
}

.box {
  width: 100px;
  border: 1px solid #888;
  background-color: lightblue;
  margin: 4px;
}

.special {
  align-self: stretch; 
  background-color: lightcoral;
}
```

:::
