## Елементи і атрибути HTML-форми

- **Елемент `form`** створює HTML-форму для введення даних користувачем.
- **Атрибут `action`** вказує посилання, на яке буде надіслано дані форми.
- **Атрибут `method`** вказує HTTP-метод, який потрібно використати під час надсилання даних форми. Найчастіше використовують методи `GET` та `POST`.

```html
<form method="value-goes-here" action="url-goes-here">
  <!-- inputs go inside here -->
</form>
```

- **Елемент `input`** створює текстове поле для введення даних користувачем.
- **Атрибут `type`** вказує тип введених даних. Наприклад, `text`, `email`, `number`, `radio`, `checkbox` тощо.
- **Атрибут `placeholder`** підказує користувачу, що потрібно ввести в полі.
- **Атрибут `value`** вказує значення вхідних даних. Якщо вхідні дані мають тип `button`, то атрибут `value` можна використати, щоб встановити текст кнопки.
- **Атрибут `name`** надає назву текстового поля, яка служить як ключ при надсиланні форми. Для радіокнопок важливо надавати однакове значення `name`, оскільки це групує їх і користувач може зробити лише один вибір.
- **Атрибут `size`** вказує кількість символів, які мають бути видимими під час введення даних.
- **Атрибут `min`** вказує мінімальне значення для таких типів вхідних даних, як `number`.
- **Атрибут `max`** вказує максимальне значення для таких типів вхідних даних, як `number`.
- **Атрибут `minlength`** вказує мінімальну кількість символів, які можна ввести в текстовому полі.
- **Атрибут `maxlength`** вказує максимальну кількість символів, які можна ввести в текстовому полі.
- **Атрибут `required`** вказує, що поле обов’язково потрібно заповнити перед надсиланням форми.
- **Атрибут `disabled`** вказує, що текстове поле має бути неактивним.
- **Атрибут `readonly`** вказує, що текстове поле доступне лише для читання.

```html
<!-- Text input -->
<input 
  type="text"
  id="name"
  name="name"
  placeholder="e.g. Quincy Larson" 
  size="20"
  minlength="5"
  maxlength="30"
  required
/>

<!-- Number input -->
<input 
  type="number"
  id="quantity"
  name="quantity"
  min="2"
  max="10"
  disabled
/>

<!-- Button -->
<input type="button" value="Show Alert" />
```

- **Елемент `label`** створює мітку для текстового поля.
- **Атрибут `for`** вказує, якого поля стосується мітка.
- **Неявний зв’язок** — це зв’язок, за якого текстові поля пов’язано з мітками, якщо текстове поле вкладено в елементі `label`.

```html
<form action="">
  <label>
    Full Name:
    <input type="text" />
  </label>
</form>
```

- **Явний зв’язок** — це зв’язок, за якого текстові поля пов’язано з мітками, якщо до елемента `label` використано атрибут `for`.

```html
<form action="">
  <label for="email">Email Address: </label>
  <input type="email" id="email" />
</form>
```

- **Елемент `button`** створює активну кнопку. Вона також може мати атрибут `type`, який контролює поведінку кнопки, якщо її активовано. Наприклад, `submit`, `reset`, `button`.

```html
<button type="button">Show Form</button>
<button type="submit">Submit Form</button>
<button type="reset">Reset Form</button>
```

- **Елемент `fieldset`** групує пов’язані поля.
- **Елемент `legend`** додає заголовок, який описує групу елементів форми.

```html
<!-- Radio group -->
<fieldset>
  <legend>Was this your first time at our hotel?</legend>

  <label for="yes-option">Yes</label>
  <input id="yes-option" type="radio" name="hotel-stay" value="yes" />

  <label for="no-option">No</label>
  <input id="no-option" type="radio" name="hotel-stay" value="no" />
</fieldset>

<!-- Checkbox group -->
<fieldset>
  <legend>
    Why did you choose to stay at our hotel? (Check all that apply)
  </legend>

  <label for="location">Location</label>
  <input type="checkbox" id="location" name="location" value="location" />

  <label for="price">Price</label>
  <input type="checkbox" id="price" name="price" value="price" />
</fieldset>
```

- **Сфокусований стан** — це стан елементу форми, якщо він вибраний користувачем.

## Елементи і атрибути HTML-таблиці

- **Елемент `table`** створює HTML-таблицю.
- **Елемент `thead`** групує заголовки HTML-таблиці.
- **Елемент `tr`** створює рядок в HTML-таблиці.
- **Елемент `th`** створює комірку із заголовком в HTML-таблиці.
- **Елемент `tbody`** групує вміст тіла HTML-таблиці.
- **Елемент `td`** створює комірку з даними в HTML-таблиці.
- **Елемент `tfoot`** групує підсумкову частину HTML-таблиці.
- **Елемент `caption`** додає заголовок HTML-таблиці.
- **Атрибут `colspan`** вказує кількість стовпців, які має займати комірка.

```html
<table>
  <caption>Exam Grades</caption>

  <thead>
    <tr>
      <th>Last Name</th>
      <th>First Name</th>
      <th>Grade</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>Davis</td>
      <td>Alex</td>
      <td>54</td>
    </tr>

    <tr>
      <td>Doe</td>
      <td>Samantha</td>
      <td>92</td>
    </tr>

    <tr>
      <td>Rodriguez</td>
      <td>Marcus</td>
      <td>88</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <td colspan="2">Average Grade</td>
      <td>78</td>
    </tr>
  </tfoot>
</table>
```

## Робота з інструментами в HTML

- **Валідатор HTML** — це інструмент, який перевіряє синтаксис HTML, щоб переконатися в його валідності.
- **Інспектор DOM** — це інструмент, що дозволяє переглядати та змінювати HTML-структуру вебсторінки.
- **Інструменти розробника** — це набір інструментів, вбудованих в браузері, що допомагає налагоджувати, відстежувати та аналізувати вебсторінки.
