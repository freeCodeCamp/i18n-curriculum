# --description--

El tiempo `present simple` se usa para hablar de acciones o rutinas regulares.

Recuerda, para formar el `present simple`, usas el verbo base para `I`, `you`, `we`, `they` (`I code`, `they test`), y agregas `-s` o `-es` para `he`, `she`, `it` (`he codes`, `she tests`).

En un contexto laboral, describir tareas diarias se hace típicamente usando el tiempo `present simple`, ya que son acciones habituales.
