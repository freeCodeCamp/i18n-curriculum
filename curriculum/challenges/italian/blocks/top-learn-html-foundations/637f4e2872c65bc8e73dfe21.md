# --description--

Dopo aver dichiarato il doctype, devi fornire un elemento `<html>`. Questo è ciò che si chiama elemento radice del documento, il che significa che ogni altro elemento nel documento sarà un discendente di questo.

Questo diventa più importante più avanti, quando imparerai a manipolare l'HTML con JavaScript. Per ora, sappi solo che l'elemento `html` dovrebbe essere incluso in ogni documento HTML.

Nel file `index.html`, aggiungiamo l'elemento `<html>` digitando i suoi tag di apertura e chiusura, così:

```html
<!DOCTYPE html>
<html lang="en">
</html>
```

## Cos'è l'attributo lang?
`lang` specifica la lingua del contenuto testuale in quell'elemento. Questo attributo viene usato principalmente per migliorare l'accessibilità della pagina web. Permette alle tecnologie assistive, per esempio ai lettori di schermo, di adattarsi in base alla lingua e di attivare la pronuncia corretta.
