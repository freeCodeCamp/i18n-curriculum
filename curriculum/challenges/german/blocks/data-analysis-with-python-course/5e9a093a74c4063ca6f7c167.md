## --text--

Wie durchlaufen Sie ein Dictionary namens `user` und geben die Schlüssel und Werte aus?

## --answers--

```python
for key in user.items():
    print(key)
```

---

```python
for key, value in user.all():
    print(key, value)
    print(value)
```

---

```python
for key, value in user.items():
    print(key, value)
```

---

```python
for key, value in user
    print(key, value)
```
