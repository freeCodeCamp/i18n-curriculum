# --interactive--

边距和行高对于间隔列表项以增强可读性和视觉吸引力至关重要。

首先，让我们开始使用边距为列表项设置间距！

可以通过为 `li` 元素应用 margin 属性来创建列表项之间的间距。此方法允许你控件每个列表项外部的间距，有效地增加或减少它们之间的空隙。

让我们来看一个包含三个列表项的无序列表示例。

:::interactive_editor

```html
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

默认情况下，超文本标记语言不会在列表项之间应用那么多间距。

要为每个列表项的底部应用一些间距，你可以像这样使用 `margin-bottom` 属性：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  margin-bottom: 40px;
}
```

:::

在此示例中，`40px` 的外边距将应用于无序列表中每个列表项的底部。

另一种为列表项设置间距的方法是使用 `line-height` 属性。

`line-height` 属性调整单个列表项内文本行之间的垂直间距。

虽然它主要影响每个项内文本行之间的间距，但如果项仅包含单行文本，它也可以间接影响项之间的整体间距。

如果列表项有多行文本，`line-height` 会影响这些行之间的间距，但它不会直接调整各个列表项之间的间距。

要控制各个列表项之间的间距，应使用 `margin` 或 `padding` 属性。

使用之前相同的无序列表，下面是将 `line-height` 应用于列表项的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  line-height: 2; 
}
```

:::

在此示例中，`line-height: 2;` 将行高设置为字体大小的两倍，从而在每个列表项内创建更多垂直空间。

## --text--

你会使用哪个 CSS `property` 通过调整每个 `li` 元素外部的空间来为列表项创建间距？

## --answers--

`padding`

### --feedback--

此属性影响元素 `border` 内部的空间。

---

`line-height`

### --feedback--

此属性影响文本行间距，而不是元素外部的间距。

---

`margin`

---

`border`

### --feedback--

此属性影响元素的 `border`，而不是其外部空间。

## --text--

给定 CSS 规则 `li { margin-bottom: 15px; }`，这对列表项有什么效果？

## --answers--

为每个列表项内部添加 `15px` 的间距。

### --feedback--

此规则影响列表项外部的空白，而非内部。

---

为每个列表项的内容和 `border` 之间添加 `15px` 的间距。

### --feedback--

此规则影响每个列表项的 `border` 外部的空间。

---

为每个列表项下方添加 `15px` 的间距。

---

为每个列表项上方添加 `15px` 的空间。

### --feedback--

此规则在每个列表项下方添加空白，而不是上方。
