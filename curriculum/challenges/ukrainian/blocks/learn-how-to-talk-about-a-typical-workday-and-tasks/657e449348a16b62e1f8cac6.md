# --description--

`Security sweeps` — це детальні огляди або обшуки, які проводяться в області чи будівлі, зазвичай з метою безпеки та охорони. Вони включають перевірку на наявність потенційних ризиків, несанкціонованого доступу або небезпечних предметів. Наприклад:

`Security officers perform security sweeps to find any hidden dangers.` — це означає, що охоронці ретельно перевіряють область, щоб виявити будь-які небезпечні речі, які важко помітити.
