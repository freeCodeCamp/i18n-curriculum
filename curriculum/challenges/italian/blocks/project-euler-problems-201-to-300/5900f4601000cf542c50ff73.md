# --description--

Una frazione positiva il cui numeratore è minore del denominatore si chiama frazione propria.

Per ogni denominatore, $d$, ci saranno $d−1$ frazioni proprie; per esempio, con $d = 12$:

$$\frac{1}{12}, \frac{2}{12}, \frac{3}{12}, \frac{4}{12}, \frac{5}{12}, \frac{6}{12}, \frac{7}{12}, \frac{8}{12}, \frac{9}{12}, \frac{10}{12}, \frac{11}{12}$$

Chiameremo frazione resiliente una frazione che non può essere semplificata.

Inoltre definiremo la resilienza di un denominatore, $R(d)$, come il rapporto tra le sue frazioni proprie che sono resilienti; per esempio, $R(12) = \frac{4}{11}$.

Infatti, $d = 12$ è il più piccolo denominatore che ha una resilienza $R(d) < \frac{4}{10}$.

Trova il più piccolo denominatore $d$ che ha una resilienza $R(d) < \frac{15\,499}{94\,744}$.
