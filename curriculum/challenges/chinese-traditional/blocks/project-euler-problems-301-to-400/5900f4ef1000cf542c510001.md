# --description--

令 $n$ 為整數，且 $S(n)$ 為 $n$ 的因數集合。

若子集合 $A$ 為 $S(n)$ 的子集合，且 $A$ 僅包含一個元素，或 $A$ 中的任一元素皆不會整除 $A$ 中的其他元素，則稱 $A$ 為 $S(n)$ 的反串鏈。

例如：$S(30) = \\{1, 2, 3, 5, 6, 10, 15, 30\\}$

$\\{2, 5, 6\\}$ 不是 $S(30)$ 的反串鏈。

$\\{2, 3, 5\\}$ 是 $S(30)$ 的一個反串鏈。

令 $N(n)$ 為 $S(n)$ 的反鏈的最大長度。

求 $\sum N(n)$，其中 $1 ≤ n ≤ {10}^8$
