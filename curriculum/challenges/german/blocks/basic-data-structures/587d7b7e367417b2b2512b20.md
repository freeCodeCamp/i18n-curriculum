# --description--

Das Folgende ist ein Beispiel für die einfachste Implementierung einer Array-Datenstruktur. Dies wird als <dfn>eindimensionales Array</dfn> bezeichnet, was bedeutet, dass es nur eine Ebene hat oder dass keine anderen Arrays darin verschachtelt sind. Beachten Sie, dass es <dfn>Booleans</dfn>, <dfn>Strings</dfn> und <dfn>Zahlen</dfn> sowie andere gültige JavaScript-Datentypen enthält:

```js
let simpleArray = ['one', 2, 'three', true, false, undefined, null];
console.log(simpleArray.length);
```

Der `console.log`-Aufruf zeigt `7` an.

Alle Arrays haben eine Eigenschaft namens length, die, wie oben gezeigt, sehr einfach mit der Syntax `Array.length` aufgerufen werden kann. Eine komplexere Implementierung eines Arrays ist unten zu sehen. Dies wird als <dfn>mehrdimensionales Array</dfn> bezeichnet, also ein Array, das andere Arrays enthält. Beachten Sie, dass dieses Array auch JavaScript-<dfn>Objekte</dfn> enthält, die wir im nächsten Abschnitt sehr genau untersuchen werden. Für den Moment müssen Sie nur wissen, dass Arrays auch komplexe Objekte speichern können.

```js
let complexArray = [
  [
    {
      one: 1,
      two: 2
    },
    {
      three: 3,
      four: 4
    }
  ],
  [
    {
      a: "a",
      b: "b"
    },
    {
      c: "c",
      d: "d"
    }
  ]
];
```

# --instructions--

Wir haben eine Variable namens `yourArray` definiert. Vervollständigen Sie die Anweisung, indem Sie der Variablen `yourArray` ein Array mit mindestens 5 Elementen zuweisen. Ihr Array sollte mindestens einen <dfn>String</dfn>, eine <dfn>Zahl</dfn> und einen <dfn>Boolean</dfn> enthalten.

# --hints--

`yourArray` sollte ein Array sein.

```js
assert.strictEqual(Array.isArray(yourArray), true);
```

`yourArray` sollte mindestens 5 Elemente lang sein.

```js
assert.isAtLeast(yourArray.length, 5);
```

`yourArray` sollte mindestens einen `boolean` enthalten.

```js
assert(yourArray.filter((el) => typeof el === 'boolean').length >= 1);
```

`yourArray` sollte mindestens einen `number` enthalten.

```js
assert(yourArray.filter((el) => typeof el === 'number').length >= 1);
```

`yourArray` sollte mindestens einen `string` enthalten.

```js
assert(yourArray.filter((el) => typeof el === 'string').length >= 1);
```
