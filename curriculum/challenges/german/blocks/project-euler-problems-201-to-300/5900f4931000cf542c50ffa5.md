# --description--

Für eine positive ganze Zahl $k$ definieren Sie $d(k)$ als die Summe der Ziffern von $k$ in seiner üblichen Dezimaldarstellung. Somit gilt $d(42) = 4 + 2 = 6$.

Für eine positive ganze Zahl $n$ definieren Sie $S(n)$ als die Anzahl der positiven ganzen Zahlen $k &lt; {10}^n$ mit den folgenden Eigenschaften:

- $k$ ist durch 23 teilbar und,
- $d(k) = 23$.

Es ist gegeben, dass $S(9) = 263\\,626$ und $S(42) = 6\\,377\\,168\\,878\\,570\\,056$.

Finden Sie $S({11}^{12})$ und geben Sie Ihre Antwort $\bmod {10}^9$ an.
