# --explanation--

`Could` y `might` son verbos modales usados para describir posibilidades.

`Could` sugiere que algo es posible pero no seguro. Por ejemplo:

`If we don't fix this bug, it could cause a system crash.` - Existe la posibilidad de que el sistema deje de funcionar.

`Might` sugiere que algo es posible pero menos seguro que `could`. Por ejemplo:

`We might need to rewrite some parts of the code.` - Es posible que sean necesarios cambios.
