# --interactive--

Замінений елемент — це елемент, вміст якого визначається зовнішнім ресурсом, а не самим CSS. CSS, або каскадні таблиці стилів, використовуються для додавання стилів до вебсторінки. Загальними прикладами замінених елементів є елементи image, iframe і video.

За допомогою замінених елементів ви можете контролювати позицію або макет елемента. Але ваш CSS не може безпосередньо змінювати вміст цього елемента. Це легше пояснити на прикладах. Розглянемо елемент image, який вставляє зображення на вашу вебсторінку:

```html
<img src="example-img-url" alt="Descriptive text goes here">
```

Сам елемент замінений зовнішнім об’єктом: зображенням. Ваш CSS може керувати такими речами, як позиціонування зображення або застосування до нього фільтра, але ви не можете змінити саме зображення. Більш складним прикладом може бути елемент `iframe`, який вставляє зовнішній сайт на вашу вебсторінку.

Ось приклад використання елемента `iframe` для популярного відео на каналі freeCodeCamp на YouTube. Якщо хочете побачити інші відео у вікні попереднього перегляду, змініть значення атрибута `src` на відео на ваш вибір. Щоб побачити попередній перегляд, потрібно увімкнути інтерактивний редактор.

**ПРИМІТКА**: Не хвилюйтеся через додаткові атрибути, згадані в інтерактивному прикладі, як `referrerpolicy` і `allowfullscreen`. Ви дізнаєтеся більше про роботу з елементами `iframe` на майбутньому практичному занятті.

:::interactive_editor

```html
<iframe width="400" height="200" src="https://www.youtube.com/embed/u43gJJrVa1I?si=BoDW_puFsy8OEr_Z" title="Professional Cloud Architect Certification Course – Pass the Exam! (YouTube video)" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen></iframe>
```

:::

Інші поширені приклади використання елемента `iframe` — це вставлення карт на сторінку. Ось приклад вставленої карти.

Увімкніть інтерактивний редактор і спробуйте попрацювати з картою, збільшуючи або зменшуючи масштаб.

:::interactive_editor

```html
<iframe
  title="Map of the Royal Observatory, Greenwich, London"
  width="300"
  height="200"
  src="https://www.openstreetmap.org/export/embed.html?bbox=-0.004017949104309083%2C51.47612752641776%2C0.00030577182769775396%2C51.478569861898606&amp;layer=mapnik">
</iframe>
```

:::

Сам елемент замінений зовнішнім об’єктом: сайтом. Ваш CSS може змінювати позицію вставленого сайту, але не може змінювати вміст сайту. Якщо заглибитися, то якщо вставлений сайт має елемент `h1`, ваш CSS не зможе стилізувати цей елемент `h1`. Ви не можете змінити розмір, колір шрифту тощо.

Існують й інші замінені елементи, такі як `video` і `embed`. Деякі елементи поводяться як замінені за певних обставин. Ось приклад елемента `input` з атрибутом `type`, встановленим у `image`:

```html
<input type="image" alt="Descriptive text goes here" src="example-img-url">
```

Цей тип `input` вважається заміненим елементом, але інші типи `input`, як `text` або `email`, не є заміненими елементами.
