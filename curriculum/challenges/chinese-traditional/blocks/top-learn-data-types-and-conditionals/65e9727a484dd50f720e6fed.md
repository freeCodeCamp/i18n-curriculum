# --description--

在任何程式設計語言中，你需要比較數值來做出決策。例如，如果天氣晴朗，你會出門，否則你會待在家裡。另一個例子是檢查使用者是否有足夠的點數來在遊戲中升級。

`if-else` 條件述語用於在 JavaScript 中做出決策。它經常與比較運算子一起使用，以比較數值並根據結果做出決策。

if 述語的範例是：

```javascript
let x = 5;

if (x > 3) {
  console.log("x is greater than 3");
} else {
  console.log("x is less than or equal to 3");
}
```

在這個範例中，`x` 變數使用 `>` 運算子與數字 `3` 進行比較。如果 `x` 大於 `3`，將執行第一個程式碼區塊。否則，將執行第二個程式碼區塊。
