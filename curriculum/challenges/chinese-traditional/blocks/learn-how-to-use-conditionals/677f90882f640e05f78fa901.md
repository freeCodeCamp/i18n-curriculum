# --explanation--

`Present Continuous` 時態通常強調暫時的狀態或在說話當下發生的問題。 例如：

- `The server isn't running diagnostics.`－凸顯標示診斷處理程序目前沒有進行。

- `Developers aren't debugging the application.` - 顯示開發人員目前沒有在進行該活動。

`Properly` 意味著以正確或預期的方式執行某事。

在這句話中，`aren't booting properly` 指的是目前無法正確啟動的電腦。
