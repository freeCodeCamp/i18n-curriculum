# --description--

Просте число $p$ називають простим числом Панайтіпола, якщо $p = \frac{x^4 - y^4}{x^3 + y^3}$ для деяких натуральних чисел $x$ та $y$.

Знайдіть, скільки простих чисел Панайтіпола менше за $5 × {10}^{15}$.
