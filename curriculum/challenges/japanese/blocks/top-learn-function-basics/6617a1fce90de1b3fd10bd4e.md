# --description--

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

`animal`を`favoriteAnimal()`関数の丸括弧内に入れることで、JavaScriptに`favoriteAnimal`関数に何らかの値を送ることを伝えています。つまり、animalは将来の値のプレースホルダーにすぎません。でも、どんな値を送っているのでしょうか？

最後の行、`favoriteAnimal('Goat')`は`favoriteAnimal`関数を呼び出し、その関数内に`'Goat'`という値を渡しています。ここで`'Goat'`が引数です。`favoriteAnimal`関数に「`'Goat'`を`favoriteAnimal`関数に送って、`'Goat'`プレースホルダーがあるところでは`'animal'`を使ってください」と伝えています。パラメータを使うことで柔軟性が生まれ、どんなanimalでもお気に入りとして宣言できます。

パラメータが関数に渡され、値が戻り値として返される仕組みを視覚的に理解するための図を示します。

`favoriteAnimal()`の中で`console.log()`を`'Goat'`という引数で呼び出すことで、`"Goat is my favorite animal!"`という文字列の戻り値がコンソールに出力されることに注目してください。関数呼び出し`favoriteAnimal('Goat')`を別の関数呼び出し`log()`の引数として渡しています。
