# --explanation--

フレーズ `in real time` は、遅延なく出来事が起こると同時に何かが起こることを指します。これは監視、通信、データ処理の文脈でよく使われます。例えば：

`The system updates the dashboard in real time, so you can see changes instantly.` - 更新が適用された直後にダッシュボードが変わることを意味します。

ジェイクの文では、セキュリティの脆弱性を確認したものの、更新が行われた直後に常にチェックしていたわけではないという意味です。
