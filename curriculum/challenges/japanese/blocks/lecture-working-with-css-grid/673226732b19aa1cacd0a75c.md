# --interactive--

前のレッスンでは、ウェブページで複雑かつ流動的なレイアウトを作成できるCSSグリッドを紹介しました。このレッスンでは、`fr`単位を使って柔軟なグリッドレイアウトを作成する方法を探ります。

まずは、このHTMLマークアップから始めましょう。これはグリッドコンテナを表します。

```html
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

CSS内では、コンテナのdisplayプロパティをgridに設定しています。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 25% 25% 25% 25%;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

`grid-template-columns`は各カラムのサイズを設定するために使われます。この場合、各カラムのサイズはコンテナの`25%`になります。次にgapプロパティで各カラムの間にスペースを作っています。

これまではカラムサイズにパーセンテージを使ってきましたが、`fr`単位も使えます。

`fr`単位は分数単位で、グリッドコンテナのスペースの一部を表します。

パーセンテージの代わりに`fr`単位を使うようにコードをリファクタリングすると、次のようになります。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

各カラムは利用可能なスペースの1分数を占めます。4つのカラムがあるので、各カラムはグリッドコンテナ内のスペースを均等に分け合います。

グリッドレイアウトを作成し始めると、`fr`単位をより頻繁に使いたくなるでしょう。なぜなら、これらは柔軟で比例的にスペースを分配できるため、画面サイズの変化に応じて自動的に調整されるレスポンシブなレイアウトを作成できるからです。ピクセル値を手動で調整する必要がなくなります。
