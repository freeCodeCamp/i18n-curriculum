# --description--

값과 배열을 반복하는 가장 일반적인 방법을 알게 되었으니, 배열을 반복하는 몇 가지 추가 방법이 있습니다. 배열에는 `map()`라는 내장 메서드가 있는데, 이 메서드는 원본 배열의 각 요소에 함수를 적용하여 새 배열을 만듭니다. `map()` 메서드는 원본 배열을 변경하지 않습니다. `map()` 메서드의 구문은 다음과 같습니다:

```javascript
const array = [1, 2, 3, 4, 5];

const newArray = array.map((arrayValue) => {
  return arrayValue * 2;
});

console.log(newArray); // Output: [2, 4, 6, 8, 10]
```

`map()` 메서드는 원본 배열의 각 요소에 `(arrayValue) => { return arrayValue * 2; }` 함수를 적용하여 새 배열을 만듭니다. 이는 원본 배열을 변경하지 않고 배열의 요소를 변환하려 할 때 특히 유용합니다.
