# --description--

「追跡」は、2つのサイコロと偶数の人数で遊ぶゲームです。

プレイヤーはテーブルの周りに座り、ゲームは向かい合った2人のプレイヤーがそれぞれ1つずつサイコロを持つところから始まります。各ターンで、サイコロを持っている2人のプレイヤーがそれを振ります。

もしプレイヤーが1を出したら、左隣のプレイヤーにサイコロを渡します。

もしプレイヤーが6を出したら、右隣のプレイヤーにサイコロを渡します。

それ以外の場合は、次のターンまでサイコロを保持します。

ゲームは、サイコロが振られて渡された後に1人のプレイヤーが両方のサイコロを持った時点で終了し、そのプレイヤーが負けとなります。

100人のプレイヤーで行うゲームで、ゲームが続くターンの期待値はどれくらいでしょうか？答えは有効数字10桁で四捨五入してください。
