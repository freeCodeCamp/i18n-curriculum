# --interactive--

In CSS wird die `list-style`-Eigenschaft verwendet, um das Aussehen von Listen auf einer Webseite zu steuern.

Egal, ob Sie mit geordneten Listen (`ol`) oder ungeordneten Listen (`ul`) arbeiten, die `list-style`-Eigenschaft ermöglicht es Ihnen, anzupassen, wie Listenelemente dargestellt werden.

Die `list-style`-Eigenschaft ist tatsächlich eine Kurzschreibweise-Eigenschaft für drei andere Eigenschaften:

- `list-style-type`
- `list-style-position`
- `list-style-image`

Jede davon spielt eine andere Rolle bei der Definition des Aussehens von Listen.

Die `list-style-type`-Eigenschaft erlaubt es Ihnen, die Art des Aufzählungszeichens oder der Nummerierung in einer Liste festzulegen.

Für ungeordnete Listen können Sie aus verschiedenen Aufzählungszeichen wählen, wie Scheiben, Kreisen oder Quadraten.

Für geordnete Listen können Sie unterschiedliche Nummerierungssysteme verwenden, wie Dezimalzahlen, römische Zahlen oder sogar alphabetische Zeichen.

Hier ist ein Beispiel für die Verwendung von `list-style-type`:

:::interactive_editor

```html
<ul style="list-style-type: square;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

In diesem Beispiel werden die Aufzählungszeichen der ungeordneten Liste in Quadrate geändert.

Die `list-style-type`-Eigenschaft ist die am häufigsten verwendete der drei, da sie direkt das Aussehen des Aufzählungszeichens oder der Nummerierung in Ihren Listen beeinflusst.

Die `list-style-position`-Eigenschaft steuert die Position des Aufzählungszeichens oder der Nummer in Bezug auf den Inhalt des Listenelements. Es gibt zwei Werte, die Sie verwenden können: `inside` und `outside`.

Wenn Sie den Wert `outside` verwenden, erscheint das Aufzählungszeichen oder die Nummer außerhalb des Inhalts, was das Standardverhalten ist.

Und wenn Sie den Wert `inside` verwenden, erscheint das Aufzählungszeichen oder die Nummer innerhalb des Inhalts, was dazu führen kann, dass der Text umbricht und mit dem Aufzählungszeichen oder der Nummer ausgerichtet wird.

Hier ist ein Beispiel für die Verwendung von `list-style-position`:

:::interactive_editor

```html
<ul style="list-style-position: inside;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
<ul style="list-style-position: outside;">
  <li>Item 4</li>
  <li>Item 5</li>
  <li>Item 6</li>
</ul>
```

:::

In diesem Beispiel werden für zwei verschiedene ungeordnete Listentags sowohl `inside`- als auch `outside`-Werte angegeben.

Die `list-style-position`-Eigenschaft kann nützlich sein, wenn Sie die Ausrichtung des Listeninhalts steuern möchten, besonders in Situationen, in denen Sie mehrere Textzeilen in einem einzelnen Listenelement haben.

Die `list-style-image`-Eigenschaft erlaubt es Ihnen, ein Bild als Aufzählungszeichen für Ihre Listenelemente zu verwenden. Dies kann nützlich sein, um Ihren Listen einen einzigartigen visuellen Stil zu verleihen.

Hier ist ein Beispiel für die Verwendung von `list-style-image`:

:::interactive_editor

```html
<head>
  <style>
    ul {
      list-style-image: url('https://cdn.freecodecamp.org/platform/universal/freecodecamp-org-gravatar.jpeg');
      list-style-position: inside;
    }
  </style>
</head>
<body>
  <ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
  </ul>
</body>
```

:::

In diesem Beispiel werden die Aufzählungszeichen durch ein individuelles freeCodeCamp-Logo ersetzt, was der Liste eine persönliche Note verleiht.

Wenn Sie `list-style-image` verwenden, stellen Sie sicher, dass das von Ihnen gewählte Bild klein und passend für das Design Ihrer Webseite ist. Wenn das Bild zu groß oder zu komplex ist, kann es die Lesbarkeit der Liste erschweren.

Sie können die drei Eigenschaften – `list-style-type`, `list-style-position` und `list-style-image` – in einer einzigen Kurzschreibweise-Eigenschaft `list-style` kombinieren.

Die Reihenfolge der Werte in der Kurzschreibweise spielt keine Rolle, aber alle drei können zusammen angegeben werden.

Hier ist ein Beispiel mit der Kurzschreibweise-Eigenschaft:

:::interactive_editor

```html
<ul style="list-style: square inside url('https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg');">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

In diesem Beispiel verwenden die Listenelemente quadratische Aufzählungszeichen, die innerhalb des Inhalts positioniert sind, mit einem benutzerdefinierten Bild als Aufzählungszeichen.

Wenn das Bild jedoch nicht verfügbar ist oder nicht gerendert wird, werden die quadratischen Aufzählungszeichen als Fallback verwendet.
