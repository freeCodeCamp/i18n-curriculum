# --description--

Число Хеммінга — це додатне число, яке не має простих множників більших за 5.

Отже, перші кілька чисел Хеммінга: 1, 2, 3, 4, 5, 6, 8, 9, 10, 12, 15.

Існує 1105 чисел Хеммінга, що не перевищують ${10}^8$.

Ми назвемо додатне число узагальненим числом Хеммінга типу $n$, якщо воно не має простих множників більших за $n$. Таким чином, числа Хеммінга — це узагальнені числа Хеммінга типу 5.

Скільки існує узагальнених чисел Хеммінга типу 100, які не перевищують ${10}^9$?
