# --description--

Este cuestionario verifica tu comprensión de los saludos, despedidas, nacionalidades, profesiones y expresiones de edad en español.

Para aprobar el cuestionario, debes responder correctamente **al menos 9 de las 10** preguntas a continuación.

Lee cada pregunta y elige la respuesta correcta. Solo hay una respuesta correcta para cada pregunta.
