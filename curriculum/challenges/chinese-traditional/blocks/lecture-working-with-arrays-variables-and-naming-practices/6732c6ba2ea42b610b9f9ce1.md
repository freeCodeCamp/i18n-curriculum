# --description--

在之前的課程中，你首先被介紹了 `length` 屬性，這個屬性會傳回陣列中元素的數量。以下是它如何運作的快速提醒：

```js
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits.length); // 3
```

陣列中可能會有空槽。空槽定義為槽中沒有任何內容。這與陣列中值為 `undefined` 不同。這種類型的陣列稱為稀疏陣列。以下是一個範例：

```js
const sparseArray = [1, , , 4];
console.log(sparseArray.length); // 4
```

在這種情況下，即使我們只定義了兩個元素，`1` 和 `4`，長度仍然是 `4`，因為最高索引（`3`）加上 `1` 給了我們長度為 `4`。 

現在讓我們討論如何創建一個固定長度的空陣列。在 JavaScript 中有幾種方法可以做到這一點，但一個常見的方法是使用帶有數字引數的 `Array()` 建構式。`Array()` 建構式可以與 `new` 關鍵字一起使用來創建一個新的陣列。以下是一個範例：

```js
const emptyArray = new Array(5);
console.log(emptyArray.length); // 5
console.log(emptyArray); // [ , , , , ]
```

在這個範例中，我們使用 `Array(5)` 創建一個新的陣列。這會創建一個長度為 `5` 的稀疏陣列，所有的槽都是空的。 

另一種創建固定長度空陣列的方法是使用帶有長度引數的 `Array.from()` 方法。與 `new Array(n)` 不同，此方法會創建指定長度的陣列，所有元素皆存在且值為 `undefined`： 

```js
const fixedLengthArray = Array.from({ length: 5 });
console.log(fixedLengthArray.length); // 5
console.log(fixedLengthArray); // [undefined, undefined, undefined, undefined, undefined]
```

如果你想創建一個特定長度的陣列並用預設值填充它，你可以使用 `Array.fill()` 方法： 

```js
const filledArray = new Array(3).fill(0);
console.log(filledArray); // [0, 0, 0]
```

這會創建一個長度為 3 的陣列，並將所有元素填入值 `0`。注意：當填入物件時，所有槽都會參考相同的物件；如果你需要獨立的複本，請改用回呼或 `Array.from()`。

了解如何取得陣列長度以及創建固定長度的陣列對於許多程式設計任務非常重要，特別是當你需要為特定演算法或資料結構初始化陣列時。
