`Cuál`

### --feedback--

Esta palavra é usada ao pedir informações específicas, como o sobrenome de alguém. Não esqueça o acento no `a`.

---

`Puedes`

### --feedback--

Esta palavra é usada ao perguntar se a pessoa pode fazer algo.

# --explanation--

Cada palavra interrogativa é usada para um propósito comunicativo diferente em espanhol.

`¿Cómo te llamas?` é usado para perguntar o nome de alguém. Aqui, `cómo` introduz a pergunta usando o verbo `llamarse`.

`¿Cuál es tu apellido?` é usado para pedir informações específicas. `cuál` é comumente usado ao solicitar detalhes identificadores como nomes, números ou e-mails.

`¿Puedes repetirlo?` é um pedido educado. `puedes` vem do verbo `poder` e é usado na estrutura:

`¿Puedes` + infinitivo + `...?`

Este padrão é comumente usado para pedir a alguém que faça algo educadamente.
