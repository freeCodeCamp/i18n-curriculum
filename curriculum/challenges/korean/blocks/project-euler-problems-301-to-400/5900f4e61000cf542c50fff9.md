# --description--

$T(n)$을 $n^{\text{th}}$ 삼각수라고 할 때, $T(n) = \frac{n(n + 1)}{2}$입니다.

$dT(n)$은 $T(n)$의 약수 개수입니다. 예를 들어: $T(7) = 28$이고 $dT(7) = 6$입니다.

$Tr(n)$은 $1 ≤ i < j < k ≤ n$를 만족하고 $dT(i) > dT(j) > dT(k)$인 삼중항 ($i$, $j$, $k$)의 개수입니다. $Tr(20) = 14$, $Tr(100) = 5\,772$, 그리고 $Tr(1000) = 11\,174\,776$입니다.

$Tr(60\,000\,000)$을 구하세요. 답의 마지막 18자리를 제출하세요.
