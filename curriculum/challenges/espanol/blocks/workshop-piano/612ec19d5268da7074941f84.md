# --description--

Los operadores lógicos se pueden usar para construir consultas de medios más complejas. El operador lógico `and` se utiliza para consultar dos condiciones de medios.

Por ejemplo, una consulta de medios que apunta a un ancho de pantalla entre 500px y 1000px sería:

```css
@media (min-width: 500px) and (max-width: 1000px){

}
```

Agregue otra regla `@media` para aplicar si la ventana del navegador es más ancha que `769px` pero más pequeña que `1199px`.

# --hints--

Debe agregar una nueva consulta `@media`.

```js
assert.lengthOf(new __helpers.CSSHelp(document).getCSSRules('media'), 2);
```

Su consulta `@media` debe tener un `min-width` de `769px` y un `max-width` de `1199px`.

```js
const mediaText = new __helpers.CSSHelp(document).getCSSRules('media')[1]?.media?.mediaText;
assert.oneOf(mediaText, ['(max-width: 1199px) and (min-width: 769px)', '(min-width: 769px) and (max-width: 1199px)']);
```
