## CSS 기본

- **CSS란 무엇인가요?**: CSS(Cascading Style Sheets)는 HTML 요소에 스타일을 적용하는 마크업 언어입니다. CSS는 색상, 배경 이미지, 레이아웃 등 다양한 스타일을 지정하는 데 사용됩니다.
- **CSS 규칙의 기본 구조**: CSS 규칙은 선택자와 선언 블록 두 부분으로 구성됩니다. 선택자는 특정 HTML 요소를 식별하고 스타일을 적용하기 위한 패턴입니다. 선언 블록은 해당 선택자에 적용할 스타일 집합을 지정합니다.

다음은 CSS 규칙의 일반적인 구문입니다:

```css
selector {
    property: value;
}
```

- **`meta name="viewport"` 요소**: 이 `meta` 요소는 브라우저에 페이지의 크기와 다양한 기기, 특히 모바일 폰과 태블릿에서의 확대/축소 방식을 제어하는 방법을 지시합니다.
- **기본 브라우저 스타일**: 각 HTML 요소에는 기본적으로 브라우저 스타일이 적용됩니다. 보통 기본 여백과 패딩 같은 항목이 포함됩니다.

## 인라인, 내부, 외부 CSS

- **인라인 CSS**: 이 스타일은 HTML 요소 내에서 `style` 속성을 사용해 직접 작성합니다. 보통 관심사의 분리를 위해 인라인 CSS는 잘 사용하지 않습니다.

다음은 인라인 CSS 예시입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p style="color: red;">This is a red paragraph.</p>
```

:::

- **내부 CSS**: 이 스타일은 HTML 문서의 `<style>` 섹션 내 `head` 태그 안에 작성됩니다. 짧은 코드 예제를 만들 때 유용하지만, 보통 내부 CSS는 자주 사용하지 않습니다.
- **외부 CSS**: 이 스타일은 별도의 CSS 파일에 작성되고, `link` 섹션의 `head` 요소를 통해 HTML 문서에 연결됩니다. 대부분의 프로젝트에서는 내부나 인라인 CSS보다 외부 CSS 파일을 사용합니다.

## `width` 및 `height` 속성 다루기

- **`width` 속성**: 이 속성은 요소의 너비를 지정합니다. 너비를 지정하지 않으면 기본값은 `auto`으로 설정됩니다. 이 경우 브라우저가 콘텐츠, 부모 요소, 디스플레이 유형에 따라 너비를 결정합니다.
- **`min-width` 속성**: 이 속성은 요소의 최솟값 너비를 지정합니다.
- **`max-width` 속성**: 이 속성은 요소의 최대 너비를 지정합니다.
- **`height` 속성**: 이 속성은 요소의 높이를 지정합니다. 기본적으로 높이는 `auto`이며, 내부 콘텐츠에 맞춰 조절됩니다.
- **`min-height` 속성**: 이 속성은 요소의 최솟값 높이를 지정합니다.
- **`max-height` 속성**: 이 속성은 요소의 최대 높이를 지정합니다.

## 다양한 CSS 결합자 유형

- **자손 결합자**: 이 결합자는 지정한 부모 요소의 자손 요소를 선택하는 데 사용됩니다. 다음 예시는 `li` 요소 안의 모든 `ul` 항목을 선택합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul>
    <li>Example item one</li>
    <li>Example item two</li>
    <li>Example item three</li>
</ul>
```

```css
ul li {
    background-color: yellow;
}
```

:::

- **자식 결합자 (`>`)**: 이 결합자는 지정한 부모 요소의 직접 자식 요소를 선택합니다. 다음 예시는 `p` 클래스의 직접 자식인 모든 `container` 요소를 선택합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <p>This will get styled.</p>

  <div>
    <p>This will not get styled.</p>
  </div>
</div>
```

```css
.container > p {
  background-color: black;
  color: white;
}
```

:::

- **다음 형제 결합자 (`+`)**: 이 결합자는 지정한 형제 요소 바로 다음에 오는 요소를 선택합니다. 다음 예시는 `h2` 요소 바로 다음에 오는 단락 요소를 선택합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h2>I am a sub heading</h2>

<p>This paragraph element will get a red background.</p>
```

```css
h2 + p {
  background-color: red;
}
```

:::

- **이후 형제 결합자 (`~`)**: 이 결합자는 지정한 요소 이후에 나오는 모든 형제 요소를 선택합니다. 다음 예시는 `ul` 요소와 같은 부모를 공유하는 두 번째 단락 요소만 스타일을 지정합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <p>This will not get styled.</p>
  <ul>
    <li>Example item one</li>
    <li>Example item two</li>
    <li>Example item three</li>
  </ul>
  <p>This will get styled.</p>
</div>
<p>This will not get styled.</p>
```

```css
ul ~ p {
  background-color: green;
}
```

:::

## 인라인, 블록, 인라인-블록 레벨 요소

- **인라인 레벨 요소**: 인라인 요소는 필요한 너비만 차지하며 새 줄에서 시작하지 않습니다. 이 요소들은 콘텐츠 내에서 흐르며 텍스트나 다른 인라인 요소와 함께 나타납니다. 일반적인 인라인 요소로는 `span`, `anchor`, `img` 요소가 있습니다.
- **블록 레벨 요소**: 블록 요소는 새 줄에서 시작하며 기본적으로 컨테이너의 전체 너비를 차지합니다. 일반적인 블록 요소로는 `div`, `paragraph`, `section` 요소가 있습니다.
- **인라인-블록 레벨 요소**: `inline-block` 속성을 사용해 요소를 `display`로 설정할 수 있습니다. 이 요소들은 인라인 요소처럼 동작하지만 블록 요소처럼 `width`와 `height`를 설정할 수 있습니다.

## 여백과 패딩

- **`margin` 속성**: 이 속성은 요소 바깥쪽, 즉 요소의 테두리와 주변 요소 사이에 공간을 적용하는 데 사용됩니다.
- **`padding` 속성**: 이 속성은 요소 안쪽, 즉 콘텐츠와 테두리 사이에 공간을 적용하는 데 사용됩니다.
- **`margin` 단축 속성**: 1~4개의 값을 지정해 여백의 각 면을 설정할 수 있습니다. 한 값은 네 면 모두에 적용되고, 두 값은 `top`와 `bottom`, 그리고 `right`와 `left`에 적용됩니다. 세 값은 `top`, 수평(`right`와 `left`), 그리고 `bottom`에 적용됩니다. 네 값은 `top`, `right`, `bottom`, `left`에 각각 적용됩니다.
- **`padding` 단축 속성**: 1~4개의 값을 지정해 패딩의 각 면을 설정할 수 있습니다. 한 값은 네 면 모두에 적용되고, 두 값은 `top`와 `bottom`, 그리고 `right`와 `left`에 적용됩니다. 세 값은 `top`, 수평(`right`와 `left`), 그리고 `bottom`에 적용됩니다. 네 값은 `top`, `right`, `bottom`, `left`에 각각 적용됩니다.

## CSS 특이성

- **인라인 CSS 특이성**: 인라인 CSS는 요소에 직접 적용되므로 가장 높은 특이성을 가집니다. 내부나 외부 CSS를 재정의합니다. 인라인 스타일의 특이성 값은 (1, 0, 0, 0)입니다.
- **내부 CSS 특이성**: 내부 CSS는 HTML 문서의 `style` 섹션 내 `head` 요소 안에 정의됩니다. 인라인 스타일보다 낮은 특이성을 가집니다. 내부와 외부 CSS는 위치가 아니라 선택자에 따라 같은 특이성 수준을 가집니다.
- **외부 CSS 특이성**: 외부 CSS는 `link` 섹션의 `head` 요소를 통해 연결되고 별도의 `.css` 파일에 작성됩니다. 내부 CSS와 마찬가지로 선택자에 의해 특이성이 결정됩니다. 두 규칙의 특이성이 같으면 문서 내에서 나중에 나오는 규칙이 우선합니다. 외부 CSS는 큰 프로젝트에서 유지보수에 가장 적합합니다.
- **범용 선택자 (`*`)**: 문서 내 모든 요소와 일치하는 특별한 CSS 선택자입니다. 페이지의 모든 요소에 스타일을 적용할 때 유용하며, 브라우저 간 스타일 초기화나 정규화에 자주 사용됩니다. 범용 선택자는 가장 낮은 특이성 값을 가지며, 특이성 값에 0을 기여합니다 (0, 0, 0, 0).
- **타입 선택자**: 태그 이름을 기준으로 요소를 선택합니다. 타입 선택자는 다른 선택자에 비해 상대적으로 낮은 특이성을 가집니다. 타입 선택자의 특이성 값은 (0, 0, 0, 1)입니다.
- **클래스 선택자**: 점(`.`) 뒤에 클래스 이름을 붙여 정의합니다. 클래스 선택자의 특이성 값은 (0, 0, 1, 0)입니다. 클래스 선택자는 타입 선택자를 재정의할 수 있지만, ID 선택자와 인라인 스타일에는 재정의됩니다.
- **ID 선택자**: 해시 기호(`#`) 뒤에 ID 이름을 붙여 정의합니다. ID 선택자는 타입 선택자와 클래스 선택자보다 훨씬 높은 특이성을 가지지만 인라인 스타일보다는 낮습니다. ID 선택자의 특이성 값은 (0, 1, 0, 0)입니다.
- **`!important` 키워드**: 스타일 규칙에 가장 높은 우선순위를 부여해 다른 선언을 재정의할 수 있게 합니다. 사용 시 브라우저가 지정한 스타일을 강제로 적용하며, 다른 선택자의 특이성에 상관없이 적용됩니다. `!important` 사용 시 CSS 유지보수와 디버그가 어려워질 수 있으니 주의하세요.
- **캐스케이드 알고리즘**: 여러 스타일이 같은 요소를 대상으로 할 때 어떤 CSS 규칙을 적용할지 결정하는 알고리즘입니다. 명확한 규칙에 따라 가장 적절한 스타일을 선택합니다.
- **CSS 상속**: 스타일이 부모 요소에서 자식 요소로 전달되는 과정입니다. 상속을 통해 문서 트리 상위에서 스타일을 정의하면 여러 요소에 명시적으로 지정하지 않아도 적용할 수 있습니다.
