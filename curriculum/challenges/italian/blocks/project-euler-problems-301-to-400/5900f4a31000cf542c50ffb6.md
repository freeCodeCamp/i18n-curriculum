# --description--

$ABCD$ è un quadrilatero convesso con lati interi e con $1 ≤ AB &lt; BC &lt; CD &lt; AD$.

$BD$ ha lunghezza intera. $O$ è il punto medio di $BD$. $AO$ ha lunghezza intera.

Chiameremo $ABCD$ un quadrilatero integrale biclinico se $AO = CO ≤ BO = DO$.

Ad esempio, il seguente quadrilatero è un quadrilatero integrale biclinico: $AB = 19$, $BC = 29$, $CD = 37$, $AD = 43$, $BD = 48$ e $AO = CO = 23$.

<img alt="quadrilatero ABCD, con punto O, punto medio di BD" src="https://cdn.freecodecamp.org/curriculum/project-euler/biclinic-integral-quadrilaterals.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Sia $B(N)$ il numero di quadrilateri integrali biclinici distinti $ABCD$ che soddisfano ${AB}^2 + {BC}^2 + {CD}^2 + {AD}^2 ≤ N$. Possiamo verificare che $B(10\\,000) = 49$ e $B(1\\,000\\,000) = 38239$.

Trova $B(10\\,000\\,000\\,000)$.
