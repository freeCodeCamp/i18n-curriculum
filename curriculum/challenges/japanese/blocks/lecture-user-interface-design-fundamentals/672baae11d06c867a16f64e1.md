# --description--

プログレッシブエンハンスメントは、ブラウザーやデバイスに関係なく、すべてのユーザーがアプリケーションの基本的なコンテンツと機能にアクセスできるようにするデザイン手法です。

これは、すべての人に対応するコア体験を提供しつつ、より高度なブラウザーや高速なインターネット接続を持つユーザーには追加の機能や改善を提供することに重点を置いています。

プログレッシブエンハンスメントのアプローチは、以下の基本的な原則に基づいています：

- すべてのブラウザーでコアコンテンツと基本機能にアクセスできること
- すべての高度なレイアウトは外部のCSSスタイルシートで提供すること
- すべての高度な機能は外部のJavaScriptファイルで提供すること
- ユーザーのブラウザー設定を尊重すること

プログレッシブエンハンスメントのアプローチを使うことで、サポートされていない環境でコアコンテンツや機能がブロックされることがないため、アプリケーションのアクセシビリティが向上します。

速度の面でも、プログレッシブエンハンスメントはアプリケーションのパフォーマンス改善に役立ちます。

遅いインターネット接続のユーザーでも、ブラウザーが必要なリソースを最初にダウンロードするため、コンテンツにアクセスできます。

SEOの観点からも、プログレッシブエンハンスメントはアプリケーションの可視性向上に役立ちます。

検索エンジンは、初期のHTMLレスポンスにコアコンテンツが含まれているため、アプリケーションのコンテンツをクロールできます。

このアプローチは、機能がJavaScriptに大きく依存するアプリケーションには必ずしも現実的でないと批判されることもありますが、アプリケーションを構築する際に従うべき良い実践です。
