# --description--

`potential` という言葉は、確実ではないものの将来起こるかもしれないことや存在するかもしれないことを指します。プログラミングでは、`potential issue` について話すとき、それは特定の条件下でコードに問題が起こる可能性があることを意味します。確定した問題ではなく、問題になるかもしれないものです。

例えば、`There is a potential issue if the user enters a very large number.` これは問題が今起きているわけではないが、状況が起これば起こる可能性があることを意味します。
