# --description--

SEO, oder Suchmaschinenoptimierung, ist eine Praxis, die Webseiten so optimiert, dass sie sichtbarer werden und in Suchmaschinen höher ranken. Eine Möglichkeit, die SEO Ihrer Seite zu verbessern, besteht darin, eine kurze Beschreibung für die Webseite mit dem `meta`-Element bereitzustellen. Hier ist ein Beispiel, wie das meta-Element verwendet wird, um eine Seitenbeschreibung für eine Garten-Website festzulegen:

```html
<meta
  name="description"
  content="Discover expert tips and techniques for gardening in small spaces, choosing the right plants, and maintaining a thriving garden."
/>
```

Das Setzen des `name`-Attributs auf `description` stellt sicher, dass Browser, Suchmaschinen und andere Web-Tools diese Metadaten korrekt interpretieren. Das `content`-Attribut ist der Ort, an dem Sie Ihre Beschreibung einfügen. Es wird empfohlen, Ihre Beschreibungen kurz und prägnant zu halten. Das liegt daran, dass Suchmaschinen die Beschreibung oft basierend auf dem Layout der Ergebnisseite abschneiden.

Ähnlich wie bei anderen Arten von `meta`-Elementen ist die `meta`-Beschreibung auf der Webseite selbst nicht sichtbar. Ein Ort, an dem die Seitenbeschreibung zu finden ist, ist im Snippet der Suchmaschinenergebnisseite. Hier sind einige Beispiele für Ergebnis-Snippets der freeCodeCamp-Subreddit und GitHub-Repositories:

```md
r/freeCodeCamp: This is the official subreddit for the freeCodeCamp.org community. Learn to
code for free together with millions of other people...
```

```md
Our full-stack web development and machine learning curriculum is completely free and self-
paced. We have thousands of interactive coding challenges to help you...
```

In den Beispielen befindet sich jede der Seitenbeschreibungen direkt unter den Seitenlinks. Innerhalb weniger Sekunden können Nutzer einen allgemeinen Eindruck davon gewinnen, worum es auf der Seite geht, und entscheiden, ob sie auf die Links für weitere Informationen klicken.

Auch wenn `meta`-Beschreibungen das Ranking einer Seite in Suchmaschinen nicht direkt beeinflussen, kann eine starke Beschreibung zu mehr Traffic auf Ihrer Website führen.
