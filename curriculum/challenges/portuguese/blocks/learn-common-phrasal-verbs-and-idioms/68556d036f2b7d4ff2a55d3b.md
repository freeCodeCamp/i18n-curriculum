# --explanation--

Usar `that` depois de `I think` é opcional em frases como esta.Dizer `I think that it's vital we make these updates.` também está correto.Em conversas do dia a dia, no entanto, é comum omitir `that` para uma frase mais fluida e rápida.Por exemplo: 

`I think that we need more time to finish the project.` – Esta estrutura está correta.A maioria das pessoas diria `I think we need more time…`, no entanto.
