# --description--

グラフとは、頂点の集合と頂点同士をつなぐ辺の集合であり、辺でつながれた2つの頂点は隣接していると呼ばれます。

グラフは、各頂点をユークリッド空間の点に対応させることでユークリッド空間に埋め込むことができます。

柔軟なグラフとは、1つ以上の頂点を連続的に動かすことができ、その際に隣接していない少なくとも2つの頂点間の距離が変わる一方で、隣接する各頂点ペア間の距離は一定に保たれるようなグラフの埋め込みです。

剛体グラフとは、柔軟でないグラフの埋め込みを指します。

非公式には、頂点を完全に回転可能なヒンジに置き換え、辺を曲がらず伸縮しない棒に置き換えたときに、グラフの一部を他の部分から独立して動かせないグラフが剛体グラフです。

ユークリッド平面に埋め込まれた格子グラフは剛体ではありません。以下のアニメーションがそれを示しています：

<img alt="animation showing grid graphs are not rigid in Euclidean plane" src="https://cdn.freecodecamp.org/curriculum/project-euler/rigid-graphs-1.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

しかし、セルに対角辺を追加することで剛体にすることができます。例えば、2×3の格子グラフの場合、グラフを剛体にする方法は19通りあります：

<img alt="19 ways to make 2x3 grid graph rigid" src="https://cdn.freecodecamp.org/curriculum/project-euler/rigid-graphs-2.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

この問題の目的上、対角辺の向きを変えたり、セルに両方の対角辺を追加することは、格子グラフを剛体にする異なる方法とはみなしません。

$R(m, n)$ を $m × n$ の格子グラフを剛体にする方法の数とします。

例えば、$R(2, 3) = 19$、$R(5, 5) = 23\,679\,901$ です。

$S(N)$ を $1 ≤ i, j ≤ N$ の範囲での $\sum R(i, j)$ と定義します。

例えば、$S(5) = 25\,021\,721$ です。

$S(100)$ を求めてください。答えは $1\,000\,000\,033$ で割った余りで提出してください。
