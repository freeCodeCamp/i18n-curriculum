# --description--

$D_0$ を2文字の文字列 "Fa" とします。$n ≥ 1$ のとき、文字列書き換えルールによって $D_{n - 1}$ から $D_n$ を導出します：

- "a" → "aRbFR"
- "b" → "LFaLb"

したがって、$D_0$ = "Fa"、$D_1$ = "FaRbFR"、$D_2$ = "FaRbFRRLFaLbFR" となり、以下同様です。

これらの文字列はコンピューターグラフィックスプログラムへの指示として解釈できます。"F" は「1単位前進して描画」、"L" は「左に90度回転」、"R" は「右に90度回転」、"a" と "b" は無視されます。コンピューターのカーソルの初期位置は (0,0) で、(0,1) の方向を向いています。

すると、$D_n$ は $n$ 次の Heighway Dragon として知られる特殊な図形になります。例えば、$D_{10}$ は下図のようになります；"F" を1ステップとして数えると、(18,16) のハイライトされた地点は500ステップ後の位置です。

<img alt="drawing of the Heighway Dragon after 500 steps" src="https://cdn.freecodecamp.org/curriculum/project-euler/heighway-dragon.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$D_{50}$ における ${10}^{12}$ ステップ後のカーソルの位置はどこですか？答えはスペースなしで `x,y` 形式の文字列で示してください。
