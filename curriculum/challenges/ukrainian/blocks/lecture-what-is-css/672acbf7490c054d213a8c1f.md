# --interactive--

CSS можна застосовувати до вебсторінки трьома основними способами: рядковим, внутрішнім або зовнішнім.

Кожен метод має своє призначення, переваги та обмеження, і знати, коли використовувати кожен із них, важливо для написання чистого, ефективного та підтримуваного коду.

Давайте розглянемо три типи CSS і коли слід їх застосовувати.

Рядковий CSS пишеться безпосередньо всередині HTML-елемента за допомогою атрибута `style`. Він застосовує стилі до конкретного елемента.

Ось приклад із рядковим CSS:

:::interactive_editor

```html
<p style="color: green;">This is an inline-styled paragraph.</p>
```

:::

У цьому прикладі ми використовуємо атрибут `style`, щоб встановити текст абзацу `green`.

Рядковий CSS зазвичай використовується для швидких одноразових стилів або щоб замістити інші стилі для конкретного елемента.

Однак його слід уникати в більшості випадків, бо він може захаращувати HTML і ускладнювати підтримку коду.

Зазвичай краще використовувати внутрішній або зовнішній CSS, щоб тримати стилі організованими та зручними для підтримки.

Внутрішній CSS пишеться всередині тегів `style` у розділі `head` HTML-документа. Він застосовує стилі до всієї сторінки і корисний, коли потрібно стилізувати один документ.

Ось приклад внутрішнього CSS:

:::interactive_editor

```html
<head>
  <style>
    p {
      color: blue;
    }
  </style>
</head>
<body>
  <p>This paragraph is styled using internal CSS.</p>
</body>
```

:::

У наведеному прикладі внутрішній CSS задає синій колір тексту для всіх елементів `p` на сторінці.

Внутрішній CSS найкраще використовувати, коли потрібно застосувати стилі до конкретної сторінки, а не до кількох сторінок. Це корисно для односторінкових сайтів або коли стилі не потрібно повторно використовувати в інших місцях.

Однак є й недоліки, наприклад, відсутність повторного використання стилів на кількох сторінках. Крім того, як і рядковий CSS, він змішує HTML і CSS, що ускладнює підтримку коду у великих проєктах.

Зовнішній CSS пишеться у окремому файлі `.css` і підключається до HTML-документа за допомогою елемента `link` у розділі `head`.

Це дозволяє стилізувати кілька сторінок послідовно і є рекомендованим методом у професійній веброзробці.

Ось приклад стилізації всіх елементів абзацу:

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <p>This paragraph is styled using external CSS.</p>
</body>
```

```css
p {
  color: red;
}
```

:::

У попередньому уроці ви дізналися, що елемент `link` має атрибут `rel`, який визначає зв’язок між поточним документом і пов’язаним ресурсом, наприклад, посилання на таблицю стилів або зовнішній ресурс.

З іншого боку, атрибут `href` вказує URL пов’язаного ресурсу, тобто звідки слід отримати цей ресурс.

Зовнішній CSS ідеально підходить для великих проєктів, де потрібно підтримувати послідовний стиль на кількох сторінках.

Він сприяє розділенню обов’язків: HTML відповідає за структуру, а CSS — за стилі, що робить код більш підтримуваним і масштабованим.

Розуміння, коли використовувати кожен тип CSS, є ключовим для ефективної та продуктивної веброзробки.

У більшості випадків зовнішній CSS має бути вашим основним вибором, особливо для більших і складніших проєктів.
