# --description--

السلسلة النصية هي متتالية من الأحرف محاطة إما بعلامات اقتباس مفردة أو مزدوجة. في بعض لغات البرمجة، تُعامل الأحرف المحاطة بعلامات اقتباس مفردة بشكل مختلف عن تلك المحاطة بعلامات اقتباس مزدوجة، لكن في Python تُعامل على قدم المساواة. لذا، يمكنك استخدام أي منهما عند العمل مع السلاسل النصية. إليك بعض الأمثلة على السلاسل النصية:

```python
my_str_1 = 'Hello'
my_str_2 = "World"
```

إذا كنت بحاجة إلى سلسلة نصية متعددة الأسطر، يمكنك استخدام علامات اقتباس مزدوجة ثلاثية أو علامات اقتباس مفردة ثلاثية:

```python
my_str_3 = """Multiline
string"""
my_str_4 = '''Another
multiline
string'''
```

إذا كانت سلسلتك النصية تحتوي على علامات اقتباس مفردة أو مزدوجة، فهناك خياران:

- استخدم نوع علامات الاقتباس المعاكسة. أي، إذا كانت سلسلتك تحتوي على علامات اقتباس مفردة، فاستخدم علامات اقتباس مزدوجة لتغليف السلسلة، والعكس صحيح:
    

```python
msg = "It's a sunny day"
quote = 'She said, "Hello World!"'
```

- اهرب علامة الاقتباس المفردة أو المزدوجة في السلسلة باستخدام الشرطة المائلة العكسية (`\`). باستخدام هذه الطريقة، يمكنك استخدام أي من علامات الاقتباس المفردة أو المزدوجة لتغليف السلسلة نفسها:
    

```python
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

أحيانًا قد تحتاج إلى التحقق مما إذا كانت السلسلة النصية تحتوي على حرف أو أكثر. لهذا، توفر Python عامل `in`، الذي يُرجع قيمة منطقية تحدد ما إذا كان الحرف أو الأحرف موجودة في السلسلة أم لا.

إليك بعض الأمثلة:

```python
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

الآن، لننظر كيف يمكنك الحصول على طول السلسلة النصية والعمل مع الأحرف الفردية في السلسلة، وهي عملية تسمى **الفهرسة**. للحصول على طول السلسلة النصية، يمكنك استخدام الدالة المدمجة `len()`. إليك مثالًا:

```python
my_str = 'Hello world'
print(len(my_str))  # 11
```

كل حرف في السلسلة النصية له موقع يسمى فهرس. الفهرس يبدأ من الصفر، مما يعني أن فهرس أول حرف في السلسلة هو `0`، وفهرس الحرف الثاني هو `1`، وهكذا. للوصول إلى حرف حسب فهرسه، تستخدم الأقواس المربعة (`[]`) مع فهرس الحرف الذي تريد الوصول إليه بداخلها. إليك بعض الأمثلة:

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
```

الفهرسة السالبة مسموحة أيضًا، لذا يمكنك الحصول على آخر حرف في أي سلسلة باستخدام `-1`، والحرف قبل الأخير باستخدام `-2`، وهكذا:

```python
my_str = 'Hello world'
print(my_str[-1])  # d
print(my_str[-2]) # l
```

تُصنف العديد من لغات البرمجة الأخرى أنواع البيانات بشكل عام إلى أنواع بدائية أو أنواع مرجعية. الأنواع البدائية بسيطة وغير قابلة للتغيير، مما يعني أنه لا يمكن تعديلها بعد إعلانها. الأنواع المرجعية يمكن أن تحتوي على قيم متعددة، وتكون إما قابلة للتغيير أو غير قابلة له. لكن Python لا ترسم خطًا فاصلاً واضحًا بين هاتين المجموعتين. بدلًا من ذلك، تُعامل كل البيانات ككائنات، وبعض الكائنات غير قابلة للتغيير بينما البعض الآخر قابل للتغيير.

الأنواع غير القابلة للتغيير لا يمكن تعديلها أو تغييرها بعد إعلانها. يمكنك توجيه متغيراتها إلى شيء جديد، وهو ما يسمى إعادة التعيين، لكن لا يمكنك تغيير الكائن الأصلي نفسه بإضافة أو إزالة أو استبدال أي من عناصره.

السلاسل النصية هي أنواع بيانات غير قابلة للتغيير في Python. هذا يعني أنه يمكنك إعادة تعيين سلسلة نصية مختلفة إلى متغير:

```python
greeting = 'hi'
greeting = 'hello'
print(greeting) # hello
```

لكن التعديل المباشر على السلسلة النصية غير مسموح به:

```python
greeting = 'hi'
greeting[0] = 'H' # TypeError: 'str' object does not support item assignment
```

أمثلة على أنواع بيانات أخرى غير قابلة للتغيير في Python هي الأعداد الصحيحة، الأعداد العشرية، القيمة المنطقية، الصف المرتب، والمدى. ستتعرف على كل من هذه الأنواع في الدروس القادمة.
