## Teoria del colore

- **Definizione di teoria del colore**: è lo studio di come i colori interagiscono tra loro e di come influenzano la nostra percezione. Copre le relazioni tra i colori, l'armonia cromatica e l'impatto psicologico del colore.
- **Colori primari**: questi colori, che sono giallo, blu e rosso, sono le tonalità fondamentali da cui derivano tutti gli altri colori.
- **Colori secondari**: questi colori derivano dalla miscelazione in parti uguali di due colori primari. Verde, arancione e viola sono esempi di colori secondari.
- **Colori terziari**: questi colori derivano dalla combinazione di un colore primario con un colore secondario adiacente. Giallo-verde, blu-verde e blu-viola sono esempi di colori terziari.
- **Colori caldi**: questi colori, che includono rossi, arancioni e gialli, evocano sensazioni di comfort, calore e accoglienza.
- **Colori freddi**: questi colori, che includono blu, verdi e viola, evocano sensazioni di calma, serenità e professionalità.
- **Ruota dei colori**: la ruota dei colori è un diagramma circolare che mostra come i colori sono correlati tra loro. È uno strumento essenziale per i designer perché li aiuta a scegliere combinazioni di colori.
- **Schemi di colori analoghi**: questi schemi creano esperienze coerenti e rilassanti. Comprendono colori analoghi, cioè adiacenti tra loro sulla ruota dei colori.
- **Schemi di colori complementari**: questi schemi creano un alto contrasto e un impatto visivo forte. I loro colori si trovano agli estremi opposti della ruota dei colori, rispetto l'uno all'altro.
- **Schema di colori triadico**: questo schema ha colori vivaci. Sono composti da colori approssimativamente equidistanti tra loro. Se collegati, formano un triangolo equilatero sulla ruota dei colori.
- **Schema di colori monocromatico**: in questo schema tutti i colori derivano dallo stesso colore base modificandone la luminosità, l'oscurità e la saturazione. Questo evoca un senso di unità e armonia pur creando contrasto.

## Modi diversi per lavorare con i colori in CSS

- **Colori nominati**: sono nomi di colori predefiniti riconosciuti dai browser. Esempi includono `blue`, `darkred`, `lightgreen`.
- **Funzione `rgb()`**: RGB sta per Red, Green e Blue — i colori primari della luce. Questi tre colori si combinano in diverse intensità per creare un'ampia gamma di colori. La funzione `rgb()` permette di definire i colori usando il modello RGB.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>RGB Color</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

- **Funzione `rgba()`**: Questa funzione aggiunge un quarto valore, alpha, che controlla la trasparenza del colore. Se non specificato, il valore alpha predefinito è `1`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>RGBA Background</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

- **Funzione `hsl()`**: HSL sta per Hue, Saturation e Lightness — tre componenti chiave che definiscono un colore. La sintassi moderna di `hsl()` supporta anche un valore alpha opzionale usando un separatore `/`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>HSL Color</p>
<p class="alpha">HSL Color with Alpha</p>
```

```css
p {
  color: hsl(120 100% 50%);
}

.alpha {
  color: hsl(120 100% 50% / 0.8);
}
```

:::

- **Funzione `hsla()`**: la funzione `hsla()` è un metodo legacy per aggiungere trasparenza ai colori HSL. Il CSS moderno preferisce `hsl()` con un separatore `/` per i valori alpha.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>HSLA Background</div>
```


```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

- **Esadecimale**: un codice esadecimale (abbreviazione di codice hexadecimal) viene usato per rappresentare i colori nel modello RGB. "Hex" si riferisce al sistema di numerazione base 16, che usa le cifre da `0` a `9` e le lettere da `A` a `F`. I codici esadecimali possono essere lunghi 6 caratteri, come `#F498A2`, oppure una forma abbreviata di 3 caratteri, che funziona quando ogni coppia nella forma completa usa lo stesso carattere due volte. Per esempio, `#2FB` è equivalente a `#22FFBB`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1 class="hex-text">Hex Text</h1>
<p class="hex-bg">Hex Background</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}

p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

## La proprietà `box-shadow`

- **Definizione**: La proprietà `box-shadow` applica una o più ombre attorno a un elemento.
- **Valori di offset**: Devi specificare i valori orizzontali (`offset-x`) e verticali (`offset-y`). Valori `offset-x` positivi spostano l'ombra verso destra, mentre valori negativi la spostano verso sinistra. Valori `offset-y` positivi spostano l'ombra verso il basso, mentre valori negativi la spostano verso l'alto. Se un valore è `0`, non è necessario includere un'unità.
- **Raggio di sfocatura**: Questo valore opzionale controlla quanto l'ombra appare sfocata. Se non incluso, il valore predefinito è `0`, che crea bordi netti. Più alto è il valore, più morbida è l'ombra.
- **Raggio di diffusione**: Questo valore opzionale controlla quanto l'ombra si espande o si restringe. Se non incluso, il valore predefinito è `0`.
- **Colore dell'ombra**: Puoi specificare il colore usando colori nominati, valori esadecimali, `rgb()`, `rgba()`, `hsl()` o funzioni `hsla()`.
- **Parola chiave `inset`**: Aggiungendo la parola chiave `inset` l'ombra viene posizionata all'interno dell'elemento invece che all'esterno.
- **Applicare più ombre al riquadro**: Puoi applicare più ombre separandole con delle virgole. Le ombre si sovrappongono dall'interno verso l'esterno.

### Sintassi

```css
box-shadow: offset-x offset-y blur-radius spread-radius color;
```

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="shadow-box">Shadow Color Example</div>
```

```css
.shadow-box {
  width: 200px;
  padding: 20px;
  background-color: lightblue;
  box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}
```

## Gradienti lineari e radiali

- **Gradienti lineari**: Questi gradienti creano una sfumatura graduale tra colori lungo una linea retta. Puoi controllare la direzione di questa linea usando parole chiave come `to top`, `to right`, `to bottom right` o angoli come `45deg`. Puoi usare qualsiasi colore CSS valido e quante fermate di colore vuoi.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient">Linear Gradient</div>
```

```css
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

:::

- **Gradienti radiali**: Questi gradienti creano sfumature circolari o ellittiche che si irradiano da un punto centrale.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient">Radial Gradient</div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```
