# --description--

`Data` означає інформацію, особливо факти або числа, зібрані для аналізу, розгляду та використання для допомоги у прийнятті рішень.

Існують різні типи даних, наприклад `quantitative data`, які стосуються чисел (наприклад, скільки людей відвідує магазин), та `qualitative data`, які стосуються описів (наприклад, що люди думають про продукт).

`Customer data` — це конкретна інформація про клієнтів, наприклад їхні звички покупок або вподобання.

Наприклад, магазин може збирати `data` про те, в який час доби клієнти найчастіше приходять, або які продукти є найпопулярнішими.
