# --explanation--

Un `server` è un computer o un sistema che gestisce e archivia informazioni, e fornisce servizi, dati o risorse ad altri computer o dispositivi, di solito tramite una rete. Per esempio:

`The company's website is hosted on secure servers.` – Questo significa che i dati del sito web sono archiviati su computer progettati per gestire e fornire quei dati.
