# --interactive--

連絡先住所要素は、ウェブページのセクションの連絡先情報を表すために使われます。`address` 要素は多用途で、ビジネスページ、著者ページ、個人サイトなどに使えます。

ウェブサイトの連絡先セクションを作成するときは、`address` 要素のような意味論的な要素を、`div` のような一般的な要素より使うべきです。

以下は会社の連絡先ページで `address` 要素を使った例です：

:::interactive_editor

```html
<address>
  <h2>Company Name</h2>
  <p>
    1234 Elm Street<br />
    Springfield, IL 62701<br />
    United States
  </p>
  <p>Phone: <a href="tel:+15555555555">+1 (555) 555-5555</a></p>
  <p>Email: <a href="mailto:contact@company.com">contact@company.com</a></p>
</address>
```

:::

この例では、会社名、住所、電話番号、メール情報があります。住所の部分では、改行要素の `br` を使って、通り名、市、国の区切りを示しています。

電話番号には、電話番号用の `href` 値が設定されたアンカー要素があります。`tel:+` 属性内の `href` 値は、対応するデバイスで電話をかけるためのクリック可能なリンクを作成します。

メールアドレスには、別のアンカー要素が使われており、`href` リンクに設定された `mailto` 値があります。`mailto` リンクは、ユーザーが好みのメールクライアントで新しいメールを開けるようにHTML文書で使われます。

`mailto` リンクを使う欠点の一つは、ユーザーがスパムと認識しやすいことです。残念ながら、多くのスパマーがこの方法を使ってユーザーにメールを送ります。使う際はその点を覚えておいてください。
