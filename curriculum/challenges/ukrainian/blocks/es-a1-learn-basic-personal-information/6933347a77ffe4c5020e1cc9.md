# --description--

У цьому завданні ви почуєте реакцію Ángela на те, що вона дізналася про країну походження Esteban. Зверніть увагу на такі вирази:

- `Ah` — це природний звук реакції. Він показує розуміння або усвідомлення.

- `de` використовується, щоб говорити про походження.

- `Muy bien` — це позитивна реакція. Вона виражає схвалення або задоволення.
