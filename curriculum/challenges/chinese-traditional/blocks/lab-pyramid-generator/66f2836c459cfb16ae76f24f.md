# --description--

**目標：** 完成以下使用者故事並通過所有測試以完成實驗。

**使用者故事：**

1. 你應該有一個名為 `pyramid` 的函式（程式），它接受三個引數。
1. 第一個引數應該是表述你金字塔中要重複的樣式字元的字串。
1. 第二個引數應該是一個整數，用來表述金字塔中的列數。
1. 第三個引數應該是布林值。
1. `pyramid` 函式（程式）應該傳回一個字串，其中樣式字元被重複並排列成一個頂點朝上的金字塔，當第三個引數為 `false` 時。
1. 當第三個引數是 `true` 時，金字塔的頂點應該朝下。
1. 頂點列應該只有一個樣式字元，而每個其他列應該比前一列多兩個樣式字元。
1. 每列應以足夠的空格開始，以將每列的中心字元置於相同的行，且每列結尾不應有任何空格。
1. 金字塔應該以換行字元開始和結束。

例如，呼叫 `pyramid("o", 4, false)` 應該會產生以下輸出：

```js

   o
  ooo
 ooooo
ooooooo

```

function pyramid(char, count, isInverted) {
    const rows = []
    for (let i = 1; i <= count; i++) {
        if (isInverted) {
            rows.unshift(" ".repeat(count - i) + char.repeat(2 * i - 1))
        } else {
            rows.push(" ".repeat(count - i) + char.repeat(2 * i - 1))
        }        
    }
    return "\n" + rows.join("\n") + "\n";
}

console.log(pyramid("#", 10, false))
