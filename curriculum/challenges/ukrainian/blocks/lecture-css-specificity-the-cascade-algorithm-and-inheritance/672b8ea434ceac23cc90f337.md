# --interactive--

Селектори ідентифікатора — одні з найпотужніших селекторів у CSS, що дозволяють розробникам застосовувати стилі до конкретних елементів з унікальними ідентифікаторами.

Це робить їх дуже ефективними для націлювання на окремі елементи, які потребують унікального стилю.

Селектори ідентифікатора визначаються символом решітки (`#`) з наступним іменем ідентифікатора. Вони мають бути унікальними в межах HTML-документа, тобто жодні два елементи не повинні мати однаковий ідентифікатор.

Ось приклад використання селектора ідентифікатора:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="unique">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#unique {
  color: purple;
}
```

:::

У цьому прикладі елемент з ідентифікатором `unique` матиме колір тексту, встановлений у `purple`.

Селектори ідентифікатора мають дуже високу специфічність, вищу за селектори типу та селектори класу, але нижчу за рядкові стилі. Значення специфічності для селектора ідентифікатора — `(0, 1, 0, 0)`.

Це означає, що селектори ідентифікатора можуть заміщати селектори класу та типу, але можуть бути заміщені рядковими стилями.
