# --description--

En esta tarea, explorarás el uso metafórico de términos para entender conceptos de programación. Las metáforas son una forma poderosa de visualizar y explicar ideas complejas al relacionarlas con objetos o acciones cotidianas.

`Container` se usa metafóricamente para describir variables como cosas que pueden contener y almacenar diferentes datos, al igual que un `container` puede contener diferentes elementos.

Las acciones `fill and empty` describen cómo puedes agregar o quitar datos de variables, similar a como podrías `fill` una botella de agua o verter el agua.

`Locked boxes` describe las constantes como `containers` seguros e inmutables, enfatizando que no pueden ser cambiadas una vez establecidas.

La frase `keep the same contents` resalta que los datos en las constantes no cambian, como una caja que está cerrada y siempre contiene las mismas cosas adentro.
