# --interactive--

Ci saranno momenti in cui dovrai manipolare gli stili di un elemento HTML all’interno del tuo file JavaScript. Un esempio è quando devi nascondere o mostrare un menu quando un utente clicca su un pulsante.

In questa lezione imparerai come manipolare gli stili usando le proprietà `Element.style` e `Element.classList`.

La proprietà `Element.style` è una proprietà di sola lettura che rappresenta lo stile in linea di un elemento. Puoi usare questa proprietà per ottenere o impostare lo stile di un elemento.

Ecco un esempio di un elemento paragrafo con un ID `para`:

```html
<p id="para">This is a paragraph</p>
```

Puoi usare la proprietà `style` per cambiare il colore del paragrafo in rosso:

:::interactive_editor

```html
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```js
const paraEl = document.getElementById("para");
paraEl.style.color = "red";
```

:::

La proprietà `style` può essere usata per impostare molte proprietà CSS, come `color`, `background-color`, `font-size`, `font-weight` e così via.

Un altro modo per manipolare gli stili è usando la proprietà `Element.classList`. La proprietà `classList` è una proprietà di sola lettura che può essere usata per aggiungere, rimuovere o alternare classi su un elemento. Vediamo un paio di esempi.

In questo esempio aggiungerai una classe chiamata `highlight` a un elemento paragrafo con un ID `para`. Nel CSS, imposteremo la classe `highlight` per cambiare il colore di sfondo del paragrafo in giallo. Nel JavaScript aggiungeremo la classe `highlight` all’elemento paragrafo usando il metodo `classList.add()`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```css
.highlight {
  background-color: yellow;
}
```

```js
const paraEl = document.getElementById("para");
paraEl.classList.add("highlight");
```

:::

Puoi anche aggiungere classi multiple a un elemento passandole come argomenti al metodo `classList.add()`:

```js
classList.add("class1", "class2", "class3");
```

Se hai bisogno di rimuovere una classe da un elemento, puoi usare il metodo `classList.remove()` così:

```js
classList.remove("highlight");
```

Per alternare una classe su un elemento, puoi usare il metodo `classList.toggle()`. In questo esempio abbiamo un elemento `button` con un ID `toggle-btn` e un elemento `nav` con un ID `menu`. Nel CSS abbiamo aggiunto alcuni stili per il menu e una classe chiamata `show` che cambierà la proprietà `display` del `.menu` in `block`. Nel JavaScript stiamo consultando l’elemento menu e l’elemento `button` usando il metodo `getElementById()`. Poi aggiungiamo un listener per l’evento all’elemento `button` che alterna la classe `show` sull’elemento `.menu` quando il pulsante viene cliccato:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button id="toggle-btn">Toggle Menu</button>

<nav id="menu" class="menu">
  <ul>
    <li>Home</li>
    <li>About</li>
    <li>Products</li>
  </ul>
</nav>
<script src="index.js"></script>
```

```css
.menu {
  display: none;
  background-color: lightgray;
  width: 50%;
  padding: 10px;
}

.menu.show {
  display: block;
}
```

```js
const menu = document.getElementById("menu");
const toggleBtn = document.getElementById("toggle-btn");

toggleBtn.addEventListener("click", () => menu.classList.toggle("show"));
```

:::

Quando un utente clicca sul pulsante, la classe `show` verrà aggiunta all’elemento `.menu` e il menu verrà mostrato. Se l’utente clicca di nuovo sul pulsante, la classe `show` verrà rimossa dall’elemento `.menu` e il menu verrà nascosto.

Usando metodi e proprietà come `style`, `add()`, `remove()` e `toggle()`, puoi facilmente manipolare gli stili di un elemento con JavaScript, creando pagine web dinamiche e interattive.
