# --description--

In den nächsten Lektionen werden wir Listen, Tuple und Ranges kennenlernen, drei grundlegende Sequenztypen, die in Python verwendet werden.

Der List-Datentyp ist eine geordnete Folge von Elementen, die aus Strings, Zahlen oder sogar anderen Listen bestehen kann. Listen sind veränderbar und verwenden nullbasiertes Indexieren, was bedeutet, dass das erste Element der Liste den Index null hat.

Hier ist die grundlegende Syntax für eine Liste:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
```

Um auf ein Element der `cities`-Liste zuzugreifen, können Sie seine Indexnummer in der Folge referenzieren. Hier ist ein Beispiel, wie Sie auf das erste Element der `cities`-Liste zugreifen:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # 'Los Angeles'
```

Negatives Indexieren wird verwendet, um Elemente vom Ende der Liste aus anstatt vom Anfang bei Index `0` zuzugreifen. Um auf das letzte Element einer Liste zuzugreifen, können Sie `-1` so verwenden:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # 'Tokyo'
```

Eine weitere Möglichkeit, eine Liste zu erstellen, ist die Verwendung des `list()`-Konstruktors. Der `list()`-Konstruktor wird verwendet, um ein iterierbares Objekt in eine Liste umzuwandeln, so:

```py
developer = 'Jessica'
list(developer) # ['J', 'e', 's', 's', 'i', 'c', 'a']
```

Ein iterierbares Objekt ist ein spezieller Objekttyp, über den man ein Element nach dem anderen durchlaufen kann. Mehr über Schleifen in Python lernen Sie später.

Um die Gesamtanzahl der Elemente in einer Liste zu erhalten, können Sie die `len()`-Funktion so verwenden:

```py
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

Wenn Sie einen Wert an einem bestimmten Index aktualisieren möchten, können Sie so vorgehen:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

Da Listen veränderbar sind, können Sie jedes Element in der Liste aktualisieren, solange Sie eine gültige Indexnummer angeben. Wenn Sie einen Index (positiv oder negativ) angeben, der außerhalb der Grenzen der Liste liegt, erhalten Sie eine `IndexError`:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list assignment index out of range
"""
```

Wenn Sie ein Element aus einer Liste entfernen möchten, können Sie das Schlüsselwort `del` so verwenden:

```py
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

Manchmal ist es hilfreich zu prüfen, ob ein Element in der Liste enthalten ist. Dafür können Sie das Schlüsselwort `in` so verwenden:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

Es ist auch üblich, Listen innerhalb anderer Listen zu verschachteln, so:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

In diesem Beispiel haben wir eine verschachtelte Liste, die drei beliebte Programmiersprachen enthält. Um auf die verschachtelte Liste zuzugreifen, müssen Sie sie mit dem Index `2` ansprechen, da Listen nullbasiert indexiert sind:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

Um dann auf die zweite Sprache aus dieser verschachtelten Liste zuzugreifen, müssen Sie den Index `1` so verwenden:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # 'Rust'
```

Eine weitere gängige Technik bei Listen ist das Entpacken von Werten.

Das Entpacken von Werten aus einer Liste ist eine Technik, um Werte aus einer Liste neuen Variablen zuzuweisen. Hier ist ein Beispiel, wie eine `developer`-Liste in neue Variablen namens `name`, `age` und `job` entpackt wird.

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

Hier hat `name` den Wert `'Alice'`, `age` hat den Wert `34` und `job` hat den Wert `'Rust Developer'`.

Wenn Sie die verbleibenden Elemente einer Liste sammeln möchten, können Sie den Sternoperator (`*`) so verwenden:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

In diesem Beispiel hat `name` weiterhin den Wert `'Alice'`, und `rest` ist eine Liste mit zwei Elementen: der Zahl `34` und dem String `'Rust Developer'`.

Wenn die Anzahl der Variablen auf der linken Seite des Zuweisungsoperators nicht mit der Gesamtanzahl der Elemente in der Liste übereinstimmt, erhalten Sie eine `ValueError`:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job, city = developer

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: not enough values to unpack (expected 4, got 3)
"""
```

Das letzte Konzept, das wir betrachten, ist der Slice-Operator (`:`). Ähnlich wie bei Strings können Sie mit dem Slice-Operator Teile einer Liste so zugreifen:

```py
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie', 'Brownies']
desserts[1:4] # ['Cookies', 'Ice Cream', 'Pie']
```

In diesem Beispiel ist der Startindex `1`, da dieser auf das zweite Element der Liste zeigt. Dann verwenden wir den Slice-Operator gefolgt von einem Endindex von `4`, der alles bis zu (aber nicht einschließlich) des Elements an diesem Index einschließt.

Eine weitere Möglichkeit mit dem Slice-Operator `:` ist, ein Schrittintervall anzugeben, das bestimmt, wie viel zwischen den Indizes inkrementiert wird. Angenommen, Sie haben eine Liste von Zahlen wie diese:

```py
numbers = [1, 2, 3, 4, 5, 6]
```

Wenn Sie nur eine Liste der geraden Zahlen extrahieren möchten, können Sie den Slice-Operator so verwenden:

```py
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

Die erste gerade Zahl befindet sich am Index `1`, also ist das der Startindex. Da wir bis zum Ende der Liste gehen wollen, lassen wir den Endindex weg. Schließlich geben wir `2` für das optionale Schrittintervall an, sodass nur um `2` inkrementiert wird statt des Standardwerts `1`.

Listen sind eine nützliche und flexible Datenstruktur, die Sie in Ihren Python-Programmen häufig verwenden werden. In der nächsten Lektion lernen Sie gängige Methoden kennen, die Sie mit Listen verwenden können.
