# --interactive--

## Msingi wa mfuatano wa herufi

- **Ufafanuzi**: Mfuatano wa herufi ni mfuatano wa herufi uliofungwa kwenye alama za nukuu moja, alama za nukuu maradufu au alama za kupigia. Mfuatano wa herufi ni aina ya data ya msingi na hauwezi kubadilishwa. Kutobadilika kunamaanisha kwamba mara tu mfuatano wa herufi umeundwa, hauwezi kubadilishwa. 
- **Kupata herufi kutoka kwenye mfuatano wa herufi**: Ili kupata herufi kutoka kwenye mfuatano wa herufi unaweza kutumia uandishi wa mabano na kupitisha nambari ya kielezo. Kielezo ni nafasi ya herufi ndani ya mfuatano wa herufi, na huanza na sufuri.

:::interactive_editor

```js
const developer = "Jessica";
console.log(developer[0]); // J
```

:::

- **`\n` (Herufi ya mstari mpya)**: Unaweza kuunda mstari mpya kwenye mfuatano wa herufi kwa kutumia herufi ya mstari mpya ya `\n`.

:::interactive_editor

```js
const poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

- **Kukimbia mfuatano wa herufi**: Unaweza kukimbia herufi katika mfuatano wa herufi kwa kuweka mikwaju wa nyuma (`\`) mbele ya nukuu.

:::interactive_editor

```js
const statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

## Herufi za kiolezo (Template Literals) na uingizaji wa thamani kwenye mfuatano wa herufi

- **Ufafanuzi**: Herufi za kiolezo hufafanuliwa kwa kutumia alama za kupigia (`). Zinakuwezesha kurahisisha uendeshaji wa mfuatano wa herufi, ikiwa ni pamoja na kuingiza vigezo moja kwa moja ndani ya mfuatano wa herufi, kipengele kinachojulikana kama uingizaji wa thamani kwenye mfuatano wa herufi.

:::interactive_editor

```js
const name = "Jessica";
const greeting = `Hello, ${name}!`; 
console.log(greeting); // "Hello, Jessica!"
```

:::

## ASCII, njia ya `charCodeAt()` na njia ya `fromCharCode()`

- **ASCII**: ASCII (American Standard Code for Information Interchange) ni kiwango cha usimbaji wa herufi kinachotumika kuwakilisha herufi za Kiingereza za msingi kwa kutumia thamani za nambari. Mafunzo ya awali yanaanzisha `charCodeAt()` na `fromCharCode()` kwa kutumia mifano ya ASCII.
- **Unicode**: Mfuatano wa herufi za JavaScript hutumia Unicode ndani yake, hasa usimbaji wa UTF-16. Kwa herufi 128 za kwanza (herufi za Kilatini za msingi, nambari, na alama za kawaida), thamani za Unicode zinafanana na msimbo wa ASCII. Hii ndiyo sababu mifano inayotegemea ASCII inaendelea kufanya kazi katika JavaScript.
- **Njia ya `charCodeAt()`**: Njia hii hurudisha kitengo cha msimbo wa UTF-16 cha herufi katika kielezo kilichotolewa. Kwa herufi za Kilatini za msingi, thamani hii inaendana na msimbo wa ASCII.

:::interactive_editor

```js
const letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

:::

- **Njia ya `fromCharCode()`**: Njia hii hubadilisha msimbo wa ASCII kuwa herufi inayolingana.

:::interactive_editor

```js
const char = String.fromCharCode(65);
console.log(char);  // A
```

:::

## Njia nyingine za kawaida za mfuatano wa herufi

- **Njia ya `indexOf`**: Njia hii hutumika kutafuta mfuatano wa herufi ndani ya mfuatano mwingine. Ikiwa mfuatano huo utapatikana, `indexOf` inarudisha kielezo (au nafasi) ya tukio la kwanza la mfuatano huo. Ikiwa mfuatano haukupatikana, `indexOf` inarudisha -1, ambayo inaonyesha kwamba utafutaji haukufanikiwa.

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.indexOf("fox")); // 16
console.log(text.indexOf("cat")); // -1
```
  
:::  

- **Njia ya `includes()`**: Njia hii hutumika kuangalia kama mfuatano wa herufi una mfuatano maalum ndani yake. Ikiwa mfuatano huo utapatikana, njia inarudisha true. Vinginevyo, inarudisha false.

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.includes("fox")); // true
console.log(text.includes("cat")); // false
```

:::

- **Njia ya `slice()`**: Njia hii huchukua sehemu ya mfuatano wa herufi na hurudisha mfuatano mpya wa herufi, bila kubadilisha mfuatano wa asili. Inachukua vigezo viwili: kielezo cha kuanzia na kielezo cha kumalizia kinachoweza kutolewa.

:::interactive_editor

```js
const text = "freeCodeCamp";
console.log(text.slice(0, 4));  // "free"
console.log(text.slice(4, 8));  // "Code"
console.log(text.slice(8, 12)); // "Camp"
```

:::

- **Njia ya `toUpperCase()`**: Njia hii hubadilisha herufi zote kuwa herufi kubwa na inarudisha mfuatano mpya wa herufi zote kubwa.

:::interactive_editor

```js
const text = "Hello, world!";
console.log(text.toUpperCase()); // "HELLO, WORLD!"
```

:::

- **Njia ya `toLowerCase()`**: Njia hii hubadilisha herufi zote kwenye mfuatano wa herufi kuwa herufi ndogo.

:::interactive_editor

```js
const text = "HELLO, WORLD!"
console.log(text.toLowerCase()); // "hello, world!"
```

:::

- **Njia ya `replace()`**: Njia hii inakuwezesha kupata thamani maalum (kama neno au herufi) kwenye mfuatano wa herufi na kuibadilisha na thamani nyingine. Njia inarudisha mfuatano mpya wenye mabadiliko na huacha wa awali bila kubadilika kwa sababu mfuatano wa herufi wa JavaScript hauwezi kubadilishwa.

:::interactive_editor

```js
const text = "I like cats";
console.log(text.replace("cats", "dogs")); // "I like dogs"
```

:::

- **Njia ya `replaceAll()`**: Njia hii inaruhusu kutafuta matukio yote ya thamani fulani (neno, herufi, au muundo) katika mfuatano wa herufi na kuibadilisha na thamani nyingine. Inafanya kazi kama `replace()`, lakini badala ya kusimama baada ya mechi ya kwanza, inasasisha kila mechi inayopatikana katika mfuatano wa herufi.

:::interactive_editor

```js
const text = "I love cats and cats are so much fun!";
console.log(text.replaceAll("cats", "dogs")); // "I love dogs and dogs are so much fun!"
```

:::

- **Njia ya `repeat()`**: Njia hii hutumika kurudia mfuatano wa herufi mara kadhaa kama ilivyobainishwa.

:::interactive_editor

```js
const text = "Hello";
console.log(text.repeat(3)); // "HelloHelloHello"
```

:::

- **Njia ya `trim()`**: Njia hii hutumika kuondoa nafasi tupu mwanzoni na mwishoni mwa mfuatano wa herufi.

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trim()); // "Hello, world!"
```

:::

- **Njia ya `trimStart()`**: Njia hii huondoa nafasi tupu mwanzoni (au "mwanzo") wa mfuatano wa herufi.

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trimStart()); // "Hello, world!  "
```

:::

- **Njia ya `trimEnd()`**: Njia hii huondoa nafasi tupu mwishoni mwa mfuatano wa herufi.

:::interactive_editor

```js
const text = " Hello, world! ";
console.log(text.trimEnd()); // "  Hello, world!"
```

:::

- **Njia ya `prompt()`**: Njia hii ya `window` hutumika kupata taarifa kutoka kwa mtumiaji kupitia kisanduku cha mazungumzo. Njia hii inachukua hoja mbili. Hoja ya kwanza ni ujumbe utakaotokea ndani ya kisanduku cha mazungumzo, mara nyingi ukimwomba mtumiaji aingize taarifa. Ya pili ni thamani chaguo-msingi ambayo ni hiari na itajaza sehemu ya ingizo mwanzoni.

```js
const answer = window.prompt("What's your favorite animal?"); // This will change depending on what the user answers
```
