# --questions--

## --text--

Ikiwa kuna faili liitwalo `data.txt` lenye yaliyomo kama haya:

<pre>
29,97,32,100,45
15,88,5,75,22
</pre>

Ni msimbo gani ungezalisha safu ya data ifuatayo?

```py
[29. 32. 45. 15.  5. 22.]
```

## --answers--

```py
filedata = np.genfromtxt('data.txt', delimiter=',')
output = np.any(filedata < 50)

print(output)
```

---

```py
filedata = np.genfromtxt('data.txt', delimiter=',')
output = np.all(filedata < 50, axis=1)

print(output)
```

---

```py
filedata = np.genfromtxt('data.txt', delimiter=',')
output = filedata[filedata < 50]

print(output)
```
