# --description--

文字列とは、シングルクォーテーションまたはダブルクォーテーションで囲まれた文字のシーケンスです。いくつかのプログラミング言語では、シングルクォーテーションで囲まれた文字とダブルクォーテーションで囲まれた文字は異なる扱いをされますが、Pythonでは同じ扱いです。したがって、文字列を扱う際にはどちらを使ってもかまいません。以下は文字列の例です：

```python
my_str_1 = 'Hello'
my_str_2 = "World"
```

複数行の文字列が必要な場合は、三重のダブルクォーテーションまたはシングルクォーテーションを使うことができます：

```python
my_str_3 = """Multiline
string"""
my_str_4 = '''Another
multiline
string'''
```

文字列にシングルクォーテーションまたはダブルクォーテーションが含まれている場合、2つの方法があります：

- 反対の種類の引用符を使う。つまり、文字列にシングルクォーテーションが含まれている場合はダブルクォーテーションで文字列を囲み、その逆も同様です：
    

```python
msg = "It's a sunny day"
quote = 'She said, "Hello World!"'
```

- 文字列内のシングルクォーテーションまたはダブルクォーテーションをバックスラッシュ (`\`) でエスケープする。この方法なら、文字列自体を囲むのにシングルクォーテーションでもダブルクォーテーションでも使えます：
    

```python
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

文字列に1つ以上の文字が含まれているかどうかを確認したい場合があります。そのために、Pythonは `in` 演算子を提供しており、文字列にその文字が存在するかどうかを示すブール値を返します。

以下はその例です：

```python
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

次に、文字列の長さを取得し、文字列内の個々の文字を扱う方法、つまり**インデックス指定**について見ていきましょう。文字列の長さを取得するには、組み込みの `len()` 関数を使います。例を示します：

```python
my_str = 'Hello world'
print(len(my_str))  # 11
```

文字列の各文字にはインデックスと呼ばれる位置があります。インデックスはゼロベースで、つまり文字列の最初の文字のインデックスは `0`、2番目の文字のインデックスは `1` ということです。文字をインデックスでアクセスするには、大括弧 (`[]`) を使い、アクセスしたい文字のインデックスを中に書きます。以下は例です：

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
```

負のインデックスも使えます。これにより、任意の文字列の最後の文字を `-1` で、最後から2番目の文字を `-2` で取得できます：

```python
my_str = 'Hello world'
print(my_str[-1])  # d
print(my_str[-2]) # l
```

多くの他のプログラミング言語では、データ型を大きくプリミティブ型か参照型に分類します。プリミティブ型は単純で不変であり、一度宣言されると変更できません。参照型は複数の値を保持でき、変更可能なものと不変なものがあります。しかしPythonはこれらの2つのグループを厳密に区別しません。代わりに、すべてのデータはオブジェクトとして扱われ、一部のオブジェクトは不変であり、他は変更可能です。

不変のデータ型は、一度宣言されると変更や修正ができません。変数に新しい値を指すようにすることはできます（これを再代入と呼びます）が、元のオブジェクト自体の要素を追加、削除、置換することはできません。

文字列はPythonの不変データ型です。つまり、変数に別の文字列を再代入することはできます：

```python
greeting = 'hi'
greeting = 'hello'
print(greeting) # hello
```

しかし、文字列を直接変更することはできません：

```python
greeting = 'hi'
greeting[0] = 'H' # TypeError: 'str' object does not support item assignment
```

Pythonの他の不変データ型の例には、整数、浮動小数点数、ブール値、タプル、レンジがあります。これらの型は今後のレッスンで詳しく学びます。
