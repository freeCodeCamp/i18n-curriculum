# --description--

Crea una secuencia de números usando el generador de números pseudoaleatorios "Blum Blum Shub":

$$
s_0 = 14025256 \\\\
s_{n + 1} = {s_n}^2 \\; mod \\; 20\\,300\\,713
$$

Concatena estos números $s_0s_1s_2\ldots$ para crear una cadena $w$ de longitud infinita. Luego, $w = 14025256741014958470038053646\ldots$

Para un entero positivo $k$, si no existe ningún substring de $w$ con una suma de dígitos igual a $k$, $p(k)$ se define como cero. Si existe al menos un substring de $w$ con una suma de dígitos igual a $k$, definimos $p(k) = z$, donde $z$ es la posición inicial del primer substring de ese tipo.

Por ejemplo:

Las subcadenas 1, 14, 1402, … con sumas de dígitos respectivas iguales a 1, 5, 7, … comienzan en la posición 1, por lo tanto $p(1) = p(5) = p(7) = \ldots = 1$.

Las subcadenas 4, 402, 4025, … con sumas de dígitos respectivas iguales a 4, 6, 11, … comienzan en la posición 2, por lo tanto $p(4) = p(6) = p(11) = \ldots = 2$.

Las subcadenas 02, 0252, … con sumas de dígitos respectivas iguales a 2, 9, … comienzan en la posición 3, por lo tanto $p(2) = p(9) = \ldots = 3$.

Ten en cuenta que la subcadena 025 que comienza en la posición 3, tiene una suma de dígitos igual a 7, pero había una subcadena anterior (que comienza en la posición 1) con una suma de dígitos igual a 7, por lo tanto $p(7) = 1$, no 3.

We can verify that, for $0 &lt; k ≤ {10}^3$, $\sum p(k) = 4742$.

Find $\sum p(k)$, for $0 &lt; k ≤ 2 \times {10}^{15}$.
