# --description--

より具体的なCSS宣言は、より具体性の低いものより優先されます。前のレッスンで学んだインラインスタイルは、セレクターと比べて最も高い具体性を持ちます。一方で、各種セレクターはそれぞれ独自の具体性レベルを持ち、宣言の具体性に影響します。その他のセレクターも具体性に寄与しますが、ここではこれまでに学んだ以下のセレクターにのみフォーカスします：

1. IDセレクター（最も具体的なセレクター）
2. クラスセレクター
3. 型セレクター

具体性は、要素に複数の競合する宣言がある場合にのみ考慮され、いわばタイブレーカーの役割を果たします。IDセレクターはどんな数のクラスセレクターよりも常に優先され、クラスセレクターはどんな数の型セレクターよりも常に優先されます。型セレクターは、それより具体性の低いどんな数のセレクターよりも常に優先されます。より高い具体性のセレクターを持つ宣言がない場合は、同じセレクターの数が多い方が優先されます。
