## --text--

Dies ist eine E-Mail von einem Benutzer, der einige Probleme meldet:

`Hi there!`

`I'm frustrated with some problems I've been having with the app recently. Every time I try to enter values for a different month, the app shows the right month at the top, but the numbers stay the same as the month before. For example, when I select March, it still shows the numbers from February. This is really confusing.`

`Can you please fix this? This bug is really frustrating, and I use the app every day to manage my money.`

Was bittet der Benutzer das Support-Team zu tun?

# --explanation--

Um zu verstehen, was der Benutzer vom Support-Team möchte, suchen Sie nach direkten Bitten oder Fragen in der E-Mail.

Achten Sie auf Formulierungen wie `Can you please... ?`, die klar angeben, was der Benutzer verlangt.

Die Verwendung von Wörtern wie `fix` und `bug` hilft dabei, die Aktion zu erkennen, die das Support-Team durchführen soll.
