# --interactive--

疑似クラスは、特定の状態や位置に基づいて要素を選択できる特別なCSSのキーワードです。要素の状態や位置には、以下のようなものがあります。

- アクティブなとき。
- マウスでホバーされているとき。
- 親の最初の子要素であるとき。
- 親の最後の子要素であるとき。
- リンクが訪問済みであるとき。
- 無効化されているとき。

疑似クラスを使うには、セレクターにコロン（`:`）を付け、その後に疑似クラスの名前を続けます。

```css
selector:pseudo-class {
  /* CSS properties */
}
```

それでは、先ほど挙げた状態や位置を表すために疑似クラスをどのように使うか見てみましょう。

`:active` 疑似クラスは、ボタンをクリックするなどの要素のアクティブ状態を選択できます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example Button</button>
```

```css
button:active {
  background: greenyellow;
}
```

:::

`:hover` 疑似クラスは、要素のホバー状態を定義します。ボタンやリンクにマウスをホバーした例です。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="#">Hover over me!</a>  
```

```css
a:hover {
  text-decoration: none;
  color: white;
  background: crimson;
}
```

:::

`:first-child` 疑似クラスは、親要素の最初の子要素である要素を選択します。`div` コンテナ内の最初の段落要素をターゲットにする例です。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:first-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

その`div` 内の最初の段落要素は、`lightcoral` の背景色と`padding` の`0.4rem` が四辺すべてに適用されます。

`:last-child` 疑似クラスは、親の最後の子要素である要素をターゲットにします。`div` コンテナ内の最後の段落要素をターゲットにする例です。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:last-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

`:visited` 疑似クラスは、ユーザーがすでに訪問したリンクのスタイルを設定できます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="https://www.example.com" target="_blank">Visit Example.com</a>
```

```css
a:visited {
  color: purple;
}
```

:::

`:disabled` 疑似クラスは、無効化モードのインタラクティブな要素のスタイルを設定できます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button disabled>Disabled Button</button>
```

```css
button:disabled {
  background-color: lightgray;  
}
```

:::

ご覧の通り、疑似クラスを使うと、ホバーやリンクの訪問などユーザーの操作に基づいて要素のスタイルを変えられます。これにより、ウェブサイトがよりインタラクティブに感じられます。

先に挙げた疑似クラス以外にも、以下のようなものがあります。

- `:focus`
- `:first-of-type`
- `:last-of-type`
- `:nth-of-type`
- `:modal`
- `:enabled`
- `:checked`
- `:required` など。
