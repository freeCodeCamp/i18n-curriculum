# --interactive--

`.toFixed()` 方法是內建的 JavaScript 函式（程式），用於使用定點表示法格式化數字。當你需要控制數字的小數位數時，它特別有用，尤其是在顯示貨幣數值或處理精確測量時。 

`.toFixed()` 方法是在數字上呼叫的，並接受一個選用引數，該引數是小數點後要顯示的位數。它傳回具有指定小數位數的數字字串表述。以下是 `.toFixed()` 如何運作的基本範例：

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed(2)); // "3.14"
```

:::

在這種情況下，我們將小數位數限制為兩位。因此，`3.14159` 變成 `3.14`。重要的是要注意 `.toFixed()` 傳回的是字串，而不是數字。這是因為該方法主要用於格式化數字以供顯示，而非進行進一步計算。

`.toFixed()` 方法會將數字四捨五入到可用指定小數位數表述的最接近值。理解此四捨五入行為非常重要：

:::interactive_editor

```js
console.log((3.14159).toFixed(3));  // "3.142"
console.log((3.14449).toFixed(3));  // "3.144"
console.log((3.14550).toFixed(3));  // "3.146"
```

:::

如你所見，當下一位數字為 `5` 或更大時，`.toFixed()` 會進位，否則會捨去。如果你在呼叫 `.toFixed()` 時不帶引數，則預設為 `0` 位小數：

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed()); // "3"
```

:::

`.toFixed()` 方法在處理財務計算或顯示價格時特別有用：

:::interactive_editor

```js
let price = 19.99;
let taxRate = 0.08;
let total = price + (price * taxRate);

console.log("Total: $" + total.toFixed(2)); // "Total: $21.59"
```

:::

在此範例中，`.toFixed(2)` 確保總數始終以兩位小數顯示，這是許多國家貨幣的標準。

總結來說，`.toFixed()` 方法是 JavaScript 中用於格式化數字的強大工具，特別是在你需要控制小數位數顯示時。雖然它主要用於格式化輸出，但當需要精確計算時，請記得它的行為。
