# --explanation--

`Downtime`는 시스템이나 서비스가 사용 불가능하거나 작동하지 않는 기간입니다. 보통 유지 관리, 업데이트 또는 수리를 위해 예정됩니다.

`Server downtime`는 특히 서버가 일시적으로 사용 불가능한 시간을 가리킵니다. 서버는 다른 장치에 데이터, 서비스 또는 자원을 제공하는 컴퓨터 시스템입니다. 예를 들어:

- `The server downtime was scheduled at midnight to minimize the impact on users.` - 이는 서버가 사용자 편의성을 위해 자정에 사용 불가능하게 되었음을 의미합니다.

- `Unexpected downtime occurred due to a power failure.` - 이는 정전으로 인해 사전 계획 없이 시스템이 사용 불가능해졌음을 의미합니다.
