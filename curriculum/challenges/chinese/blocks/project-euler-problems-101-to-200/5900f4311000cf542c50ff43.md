# --description--

我们称一个有且只有一个角为 60° 的整数边三角形为 60° 三角形。

设 $r$ 为该 60° 三角形的内切圆半径。

有 1234 个 60° 三角形满足 $r ≤ 100$。

设 $T(n)$ 为满足 $r ≤ n$ 的 60° 三角形数量，因此 $T(100) = 1234$，$T(1000) = 22767$，且 $T(10000) = 359912$。

查找 $T(1053779)$。
