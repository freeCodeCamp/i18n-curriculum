# --explanation--

當說話者假設某物存在時，`Something` 可以用於疑問句中。

在 `Is it something serious?` 中，James 假設有問題－因為 Maria 告訴他客戶抱怨這件事－並詢問是否嚴重。相反地，`anything` 用於提問時，當說話者甚至不知道某事是否存在。為例：

- `Is there something wrong with the software?`－這裡，說話者假設有問題並詢問具體是什麼。

- `Is there anything wrong with the software?`－這裡，說話者只是好奇是否有任何問題。他們並不是在尋求確認。
