# --description--

La dichiarazione `flex` è in realtà una proprietà abbreviata per 3 proprietà che puoi impostare su un elemento flex. Queste proprietà influenzano come gli elementi flex si dimensionano all’interno del loro contenitore. Hai già visto alcune proprietà abbreviate, ma non le hai ancora definite ufficialmente.

> Le proprietà abbreviate sono proprietà CSS che ti permettono di impostare contemporaneamente i valori di più altre proprietà CSS. Usando una proprietà abbreviata, puoi scrivere fogli di stile più concisi (e spesso più leggibili), risparmiando tempo ed energia.

In questo caso, `flex` è in realtà una proprietà abbreviata per `flex-grow`, `flex-shrink` e `flex-basis`.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/flex-box/flexbox-04.png" alt="CSS code setting the flex property to 1 for a div element.">

Nello screenshot sopra, `flex: 1` equivale a: `flex-grow: 1`, `flex-shrink: 1`, `flex-basis: 0`.

Molto spesso, vedi la proprietà abbreviata flex definita con un solo valore. In quel caso, quel valore viene applicato a `flex-grow`. Quindi quando hai messo `flex: 1` sui nostri div, in realtà stavi specificando un’abbreviazione di `flex: 1 1 0`.
