#### --text--

Welche der folgenden Möglichkeiten ist die korrekte Art, eine Funktion in JavaScript zu deklarieren?

#### --distractors--

```js
func greet() {
  console.log("Hello there!");
}
```

---

```js
defineFunction greet() {
  console.log("Hello there!");
}
```

---

```js
def greet() {
  console.log("Hello there!");
}
```

#### --answer--

```js
function greet() {
  console.log("Hello there!");
}
```
