# --description--

単語 `clear` は「理解しやすい」または「見やすい」という意味です。何かを比較して、それがより `clear` であることを示したい場合は、最後に `er` を付けて `clearer` を作ります。

例えば：

- この説明は `clear` です。
- この説明はもう一方より `clearer` です。

`clearer` が2つのものを比較するために使われ、一方がもう一方より理解しやすいことを示していることに注目してください。
