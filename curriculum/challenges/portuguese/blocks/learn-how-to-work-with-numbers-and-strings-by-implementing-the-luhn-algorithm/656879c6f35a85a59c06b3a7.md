# --description--

Você já acessou elementos (caracteres) de uma string antes, usando o operador de índice `[]`.Você também pode usar o operador de índice para acessar um intervalo de caracteres em uma string com `string[start:stop:step]`:

```python
my_string = 'camperbot'
my_string[0:6] == 'camper' # True
my_string[0:6:3] == 'cp' # True
```

Onde `start` é o índice inicial (inclusivo), `stop` é o índice final (exclusivo) e `step` é a quantidade de caracteres a serem pulados.Se não especificado, `step` tem valor padrão 1.

Crie uma variável chamada `card_number_reversed` e atribua a ela o valor dos primeiros 4 caracteres de `card_number`.
