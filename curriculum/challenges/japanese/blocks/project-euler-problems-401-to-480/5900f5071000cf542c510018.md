# --description--

半径 $r$ の円 $C$ を $x^2 + y^2 = r^2$ とします。点 $P(a, b)$ と $Q(-a, c)$ を選び、$P$ と $Q$ を通る直線が $C$ に接するようにします。

例えば、四つ組 $(r, a, b, c) = (2, 6, 2, -7)$ はこの条件を満たします。

$F(R, X)$ を、$0 < r ≤ R$ かつ $0 < a ≤ X$ の範囲でこの条件を満たす整数の四つ組 $(r, a, b, c)$ の個数とします。

$F(1, 5) = 10$、$F(2, 10) = 52$、$F(10, 100) = 3384$ であることが確認できます。

$F({10}^8, {10}^9) + F({10}^9, {10}^8)$ を求めてください。
