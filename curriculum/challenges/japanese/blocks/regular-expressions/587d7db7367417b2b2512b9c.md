# --description--

ここで一旦立ち止まり、新しい正規表現のスキルを試してみましょう。犯罪者のグループが刑務所から逃げ出しましたが、何人いるかはわかりません。ただし、他の人の近くにいるときは、犯罪者たちは固まっていることがわかっています。あなたは一度にすべての犯罪者を見つける責任があります。

やり方を復習するための例を示します：

正規表現 `/z+/` は、文字 `z` が1回以上連続して現れる場合にマッチします。次のすべての文字列でマッチを見つけます：

```js
"z"
"zzzzzz"
"ABCzzzz"
"zzzzABC"
"abczzzzzzzzzzzzzzzzzzzzzabc"
```

しかし、文字 `z` が含まれていないため、次の文字列ではマッチしません：

```js
""
"ABC"
"abcabc"
```

# --instructions--

他の人のグループの中で1人以上の犯罪者を見つける貪欲な正規表現を書いてください。犯罪者は大文字の `C` で表されます。
