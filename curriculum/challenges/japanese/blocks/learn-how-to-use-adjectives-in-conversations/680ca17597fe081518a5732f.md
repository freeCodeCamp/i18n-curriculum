# --explanation--

`Last time`は直近の出来事を指します。例えば：

`Last time we updated the server, we had some downtime.` – これは前回の更新時を示します。

`Order`は正式に商品や機器を注文することを意味します。その過去形は`ordered`です。例えば：

`The IT team ordered new laptops for the developers.` – これは彼らがノートパソコンを注文したことを意味します。

`Double that`は前回の数量の2倍を意味します。例えば：

`Last time we ordered 10 routers; this time we need double that, so 20 routers.` – これは数量が2倍に増えたことを示します。
