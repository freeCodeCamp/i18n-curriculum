# --description--

應用程式層級中介層是綁定到 Express 應用程式實體的中介層。它通常用於處理需要跨多個路由進行處理的請求。它可以全域套用，也可以套用到特定路由。

這種中介層是使用 `app.use()` 或特定路由方法如 `app.get()`、`app.post()` 等註冊的。除非限制在特定路由，否則它會對每個進來的請求執行。

```js
const express = require('express')
const app = express()

// Application-level middleware that logs all incoming requests
app.use((req, res, next) => {
  console.log('Request received at:', new Date())
  next(); // Passes control to the next middleware
})

app.get('/', (req, res) => {
  res.send('Hello World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* `app.use()`：這是應用程式層級中介層。它會記錄所有對伺服器的請求，無論路由為何。
* 這個中介層會對每個進來的請求執行，提供伺服器像是當前時間戳記等細節。

應用程式層級中介層適用於需要在應用程式中多數或所有路由套用的任務。這可能包括記錄、驗證和錯誤處理。
