# --interactive--

绝对定位允许你将一个元素从正常的文档流程中分离出来，使其行为独立于其他元素。 当一个元素被绝对定位时，它会被放置在自己的图层中，与布局中的其他元素完全分离。 这使得它在创建模态、提示或下拉菜单等浮动 UI 功能时非常有用，因为这些功能可能会与页面上的其他元素重叠。

默认情况下，绝对定位元素是相对于最近的已定位的祖先放置的。 如果找不到已定位的祖先，元素将相对于初始包含块定位，通常是浏览器的视口。 你可以使用 `top`、`bottom`、`left` 和 `right` 属性来移动绝对定位的元素，指定它距离包含元素的边缘有多远。

例如，设置 `top: 30px` 和 `left: 30px` 将使元素距离包含块的顶部和左侧各 `30px`。 这是一个如何在 CSS 中对元素应用绝对定位的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="positioned">Absolutely Positioned</div>
```

```css
body {
  background-color: #eeeeee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

应用此代码时，该元素将从正常文档流中移除，并放置在距离其包含块的顶部和左侧 `30px` 的位置。

绝对定位对于创建悬浮于其余内容之上的元素，如覆盖层或模态窗口，特别有用。 不过，由于该元素被从文档流中移除，如果处理不当，也会产生间隙或导致其他元素重叠在一起。
