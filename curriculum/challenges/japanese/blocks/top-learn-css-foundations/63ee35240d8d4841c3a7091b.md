# --description--

クラスセレクターは、指定された`class`を持つすべての要素を選択します。これはHTML要素に付ける属性の一種です。HTMLタグにクラスを追加し、CSSで選択する方法は次の通りです。

```html
<!-- index.html -->

<div class="alert-text">
  Please agree to our terms of service.
</div>
```

```css
/* styles.css */

.alert-text {
  color: red;
}
```

`class`セレクターの構文に注目してください：ピリオドの直後にクラス属性の大文字小文字を区別する値が続きます。クラスは一意である必要はないため、同じ`class`を複数の要素に使うことができます。

`class`属性でできるもう一つのことは、複数のクラスをスペースで区切ったリストとして1つの要素に追加することです。例えば`class="alert-text severe-alert"`のようにです。スペースは`class`名を区切るために使われるので、複数単語の名前にはスペースを使わず、代わりにハイフンを使うべきです。

## IDセレクター
IDセレクターは`class`セレクターに似ています。HTML要素に付ける別の属性である指定された`id`を持つ要素を選択します。

```html
<!-- index.html -->

<div id="title">My Awesome 90's Page</div>
```

```css
/* styles.css */

#title {
  background-color: red;
}
```

ピリオドの代わりに、ハッシュタグの直後に`id`属性の大文字小文字を区別する値を続けて使います。よくある間違いは、クラスで十分な場合でも`id`属性を過剰に使ってしまうことです。特異性を利用したり、リンクが現在のページのセクションにリダイレクトする場合など、`id`を使う意味がある場合もありますが、`id`は控えめに使うべきです（使わない方がよい場合もあります）。

クラスとIDの大きな違いは、要素が持てる`id`は1つだけということです。`id`は1ページ内で繰り返すことができず、`id`属性には空白を含めてはいけません。
