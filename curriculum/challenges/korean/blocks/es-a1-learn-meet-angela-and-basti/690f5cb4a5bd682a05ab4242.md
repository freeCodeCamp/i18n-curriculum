# --description--

스페인어에서 자신의 국적을 말할 때 가장 흔히 쓰는 표현은 **`Soy` + [국적]**입니다. 이는 "I am + [국적]"과 같은 의미입니다.

이 표현은 `¿Cuál es tu nacionalidad?`라는 질문에 대한 직접적인 답변으로 소개할 때 사용합니다. 여기서 `nacionalidad`는 "국적"을 뜻합니다.

스페인어에서는 문장 처음에 쓰이는 경우를 제외하고 국적을 대문자로 쓰지 않습니다. 또한 화자의 성별에 맞게 일치시켜야 합니다. 예를 들어:

| 국가 | 여성형 국적 | 남성형 국적 |
| :---: | :---: | :---: |
| **`México`** | `mexicana` | `mexicano` |
| **`Argentina`** | `argentina` | `argentino` |
| **`Perú`** | `peruana` | `peruano` |
| **`Uruguay`** | `uruguaya` | `uruguayo` |
| **`El Salvador`** | `salvadoreña` | `salvadoreño` |
