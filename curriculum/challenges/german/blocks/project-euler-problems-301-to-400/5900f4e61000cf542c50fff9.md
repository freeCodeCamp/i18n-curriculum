# --description--

Sei $T(n)$ die $n^{\text{te}}$ Dreieckszahl, also $T(n) = \frac{n(n + 1)}{2}$.

Sei $dT(n)$ die Anzahl der Teiler von $T(n)$. Zum Beispiel: $T(7) = 28$ und $dT(7) = 6$.

Sei $Tr(n)$ die Anzahl der Tripel ($i$, $j$, $k$), so dass $1 ≤ i < j < k ≤ n$ gilt und $dT(i) > dT(j) > dT(k)$. Es gilt $Tr(20) = 14$, $Tr(100) = 5\,772$ und $Tr(1000) = 11\,174\,776$.

Finden Sie $Tr(60\,000\,000)$. Geben Sie die letzten 18 Ziffern Ihrer Antwort an.
