# --description--

为了在本课中练习使用链接和图像，你需要一个 `HTML` 项目来操作。

- 创建一个名为 `odin-links-and-images` 的新建目录。

- 在该目录中，创建一个名为 `index.html` 的新建文件。

- 填写常用的超文本标记语言 公式化。

- 最后，为你的 `body` 添加以下 `h1`：`<h1>Homepage</h1>`

## 锚点元素
要在超文本标记语言中创建一个链接，你需要使用锚点元素。锚点元素是通过用 `<a>` 标签包裹你想作为链接的文本或其他超文本标记语言元素来定义的。

将以下内容添加到你创建的 `index.html` 页面的 `body` 中并在浏览器中打开：

```html
<a>click me</a>
```

你可能已经注意到点击这个链接没有任何反应。这是因为单独的锚点标签不会知道你想链接到哪里。你必须告诉它一个目标地址。你可以通过使用超文本标记语言属性来实现这一点。

HTML 属性为 HTML 元素提供额外的信息，并且总是放在元素的起始标签中。属性通常由两部分组成：名称和值；但是，并非所有属性都需要值。在你的情况下，你需要为起始锚点标签添加一个 `href`（超链接引用）属性。`href` 属性的值是你希望链接跳转的目标。

为你之前创建的锚点元素添加以下 `href` 属性，然后再次尝试点击，别忘了刷新浏览器以应用新更改。

```html
<a href="https://www.theodinproject.com/about">click me</a>
```

默认情况下，任何用锚点标签包裹但没有 `href` 属性的文本看起来都像普通文本。如果存在 `href` 属性，浏览器会将文本显示为蓝色并加下划线，以表示它是一个链接。

值得注意的是，你可以使用锚点 `tag` 来链接互联网中的任何类型资源，而不仅仅是其他超文本标记语言文档。你可以链接到视频、pdf 文件、图像等等，但大多数情况下，你将链接到其他超文本标记语言文档。
