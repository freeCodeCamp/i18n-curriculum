# --explanation--

Maria dice `at the end of each day`, il che indica chiaramente che i backup sono programmati giornalmente. Questo significa che l'attività è pianificata per accadere una volta ogni giorno. Per esempio:

- `At the end of each day, the system runs a data backup.` - Il sistema salva tutti i dati ogni sera per prevenire perdite.

- `We ensure the servers are updated at the end of each day.` - Gli aggiornamenti vengono applicati ai server ogni notte per evitare interruzioni durante l'orario di lavoro.
