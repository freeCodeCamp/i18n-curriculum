# --description--

Considera la siguiente configuración de 64 triángulos:

<img alt="64 triangles arranged to create larger triangle with side length of 8 triangles" src="https://cdn.freecodecamp.org/curriculum/project-euler/tri-colouring-a-triangular-grid-1.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Queremos colorear el interior de cada triángulo con uno de tres colores: rojo, verde o azul, de modo que no haya dos triángulos vecinos con el mismo color. A esta coloración se le llamará válida. Aquí, se dice que dos triángulos son vecinos si comparten una arista. Nota: si solo comparten un vértice, entonces no son vecinos.

Por ejemplo, aquí tienes una coloración válida de la cuadrícula anterior:

<img alt="colored grid of 64 triangles" src="https://cdn.freecodecamp.org/curriculum/project-euler/tri-colouring-a-triangular-grid-2.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Una coloración C' que se obtiene de una coloración C por rotación o reflexión se considera distinta de C a menos que ambas sean idénticas.

¿Cuántas coloraciones válidas distintas existen para la configuración anterior?
