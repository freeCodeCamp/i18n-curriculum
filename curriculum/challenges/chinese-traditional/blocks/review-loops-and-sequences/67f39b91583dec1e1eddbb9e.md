## Python 列表

- **介紹**：Python 中的列表是有序元素序列，可以包含字串、數字甚至其他列表。列表是可變且以 0 為起始索引。

```python
cities = ['Los Angeles', 'London', 'Tokyo']
```

- **存取列表元素**：要存取 `cities` 列表中的元素，可以用索引號碼：

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # Los Angeles
```

- **使用負索引存取元素**：要存取列表最後一個元素，可以用 `-1` 作為索引：

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # Tokyo
```

- 負索引是從列表尾端開始存取元素，而非從起始索引 `0` 開始。

- **使用 `list()` 建構式建立列表**：列表也可以用 `list()` 建構式建立。`list()` 建構式用來將可迭代物件轉成列表：

```python
developer = 'Jessica'

print(list(developer)) 
# Result: ['J', 'e', 's', 's', 'i', 'c', 'a']
```

- **取得列表長度**：用 `len()` 函式取得列表長度：

```python
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

- **列表的可變性**：列表是可變的，只要索引有效，就能更新列表中任一元素。更新列表元素可將新值指定給該索引：

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

- **索引超出範圍錯誤**：若索引（正或負）超出列表範圍，會得到 `IndexError`：

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
"""
```

- **從列表移除元素**：可用 `del` 關鍵字移除列表元素：

```python
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

- **檢查元素是否存在列表中**：可用 `in` 關鍵字檢查元素是否存在列表：

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **列表巢狀**：列表可以巢狀在其他列表中：

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

- 要存取巢狀列表，需用索引 `2`，因為列表是從 0 開始索引。

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

- 若要進一步存取該巢狀列表中的第二個語言，需用索引 `1`：

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # Rust
```

- **從列表解包值**：解包是將列表中的值指派給新變數的技巧。以下範例將 `developer` 列表解包到名為 `name`、`age` 和 `job` 的變數：

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer
```

- 若指派運算子左側變數數量與列表項目數不符，會得到 `ValueError`。

- **收集列表剩餘項目**：可用星號（`*`）運算子收集列表剩餘元素，如下：

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer
```

- **列表切片**：切片是用切片運算子 `:` 存取列表部分元素。要切片從索引 `1` 開始到索引 `3`（不包含）的元素，可用以下語法：

```python
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie']
desserts[1:3] # ['Cookies', 'Ice Cream']
```

- **步進間隔**：也可以指定步進間隔，決定索引間隔。以下範例示範如何用切片擷取偶數列表：

```python
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

## 列表方法

- **append()**：用來在列表末尾新增項目。以下範例用 `append()` 方法將數字 `6` 新增到 `numbers` 列表：

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **串接列表**：`append()` 方法可將一個列表新增到另一個列表末尾：

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

- **extend()**：用來在列表末尾新增多個項目。以下範例將數字 `6`、`8` 和 `10` 新增到 `numbers` 列表：

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

- **insert()**：用來在列表特定索引插入項目。以下範例示範 `insert()` 方法：

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

- **remove()**：用來從列表移除項目。`remove()` 方法只會移除列表中第一個符合的項目：

```py
numbers = [1, 2, 3, 4, 5, 5, 5]
numbers.remove(5)

print(numbers) # [1, 2, 3, 4, 5, 5]

```

- **pop()**：用來移除並回傳列表中特定項目：

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

- 若未指定元素，`pop` 方法會移除最後一個元素。

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

- **clear()**：用來移除列表所有項目：

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

- **sort()**：`sort()` 方法用來就地排序列表元素。以下範例示範就地排序隨機 `numbers` 列表：

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

- **sorted()**：內建函式，回傳排序後的新列表，不會修改原列表：

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
print(numbers) # [19, 2, 35, 1, 67, 41]
```

- **reverse()**：用來反轉列表元素順序：

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **index()**：用來尋找元素在列表中首次出現的索引：

```python
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

- 若用 `index()` 方法找不到元素，結果會是 `ValueError`。

## Python 元組

- **定義**：元組是 Python 資料型別，用來建立有序值序列。元組可包含混合型別資料：

```py
developer = ('Alice', 34, 'Rust Developer')
```

- 元組是不可變的，建立後無法更改元素。若嘗試更新元組中的項目，會得到 `TypeError`：

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **存取元組元素**：用中括號表示法和索引號碼存取元組元素：

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

- 可用負索引從元組尾端存取元素：

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- 若索引號碼超過或等於元組長度，會得到 `IndexError`：

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- 元組也可用 `tuple()` 建構式建立。建構式中可傳入字串、列表甚至其他元組等可迭代物件。

```python
developer = 'Jessica'

print(tuple(developer)) 
# Result: ('J', 'e', 's', 's', 'i', 'c', 'a')
```

- **檢查元組中項目**：用 `in` 關鍵字檢查項目是否在元組中，如下：

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **解包元組**：可用以下方式解包元組項目：

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

- 若要收集元組剩餘元素，可用星號（`*`）運算子，如下：

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer
```

- **切片元組**：切片可擷取元組部分元素。例如，將 `pie` 和 `cookies` 項目切片成新元組：

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

- **移除元組項目**：移除元組項目會引發 `TypeError`，因為元組是不可變的：

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

- **何時使用元組或列表？**：若需要動態集合，可新增、刪除和更新元素，應使用列表。若資料是固定且不可變，則應使用元組。

## 常用的 Tuple 方法

- **`count()`**：用來判斷某個項目在 tuple 中出現了多少次。例如，你可以檢查語言 `'Rust'` 在 tuple 中出現了幾次：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

- 如果 `count()` 方法中指定的項目在 tuple 中完全不存在，則回傳值會是 `0`：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

- 如果沒有傳入任何引數給 `count()` 方法，Python 會回傳 `TypeError`。

- **index()**：用來尋找特定項目在 tuple 中出現的索引。以下是使用 `index()` 方法尋找語言 `'Java'` 索引的範例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

- 如果找不到指定的項目，Python 會回傳 `ValueError`。

- 你可以傳入選用的起始索引給 `index()` 方法，指定從 tuple 中哪裡開始搜尋該項目：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

- 你也可以傳入選用的結束索引給 `index()` 方法，指定在哪裡停止搜尋該項目：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

- **`sorted()`**：用來對任何可迭代物件的元素進行排序，並回傳一個新的排序後列表。以下是使用 `sorted()` 函式建立一個新的數字列表的範例：

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

- **修改排序行為**：你可以使用選用的 `reverse` 和 `key` 引數來自訂可迭代物件的排序行為。以下是使用 `key` 引數依照長度排序 tuple 項目的範例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

- 你可以使用 `reverse` 引數來建立一個反向排序的新列表，如下所示：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

## Python 中的迴圈

- **定義**：迴圈用來重複執行一段程式碼區塊指定的次數。

- **`for` 迴圈**：用來對序列（例如列表、tuple 或字串）進行迭代，並對序列中的每個項目執行一段程式碼。以下是使用 `for` 迴圈遍歷列表並將每個語言列印到主控台的範例：

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)

"""
Result 

Rust
Java
Python
C++
"""
```

- 以下是使用 `for` 迴圈遍歷字串 `code` 並列印每個字元的範例：

```py
for char in 'code':
    print(char)

"""
Result 

c
o
d
e
"""
```

- `for` 迴圈可以巢狀使用。以下是使用巢狀 `for` 迴圈的範例：

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)

"""
Result

Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
"""

```

- **`while` 迴圈**：重複執行一段程式碼直到條件為 `False`。以下是使用 `while` 迴圈進行猜數字遊戲的範例：

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')

"""
Result

Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
"""

```

- **`break` 和 `continue` 陳述式**：用於迴圈中修改迴圈的執行流程。

- `break` 陳述式用來在特定條件成立時立即離開迴圈。以下是使用 `break` 陳述式處理 `developer_names` 列表的範例：

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

- `continue` 陳述式用來跳過當前迭代並繼續下一次迭代。以下是使用 `continue` 陳述式取代 `break` 陳述式的範例：

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

- `for` 和 `while` 迴圈都可以搭配 `else` 子句使用，該子句只會在迴圈沒有被 `break` 終止時執行：

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

## 範圍（Ranges）及其在迴圈中的使用

- **`range()` 函式**：用來產生一連串整數。

```py
range(start, stop, step)
```

- 必要的 `stop` 引數是一個整數（不包含該值），用來表述所產生數字序列的終點。以下是使用 `range()` 函式的範例：

```py
for num in range(3):
    print(num)
```

- 如果沒有指定 `start` 引數，預設值會是 `0`。預設情況下，整數序列會以 `1` 遞增。你可以使用選用的 `step` 引數來更改預設的遞增值。以下是產生從 2 到不包含 11（即包含 10）的偶數序列的範例：

```py
for num in range(2, 11, 2):
    print(num)
```

- 如果你不傳入任何引數給 `range()` 函式，則會得到 `TypeError`。

- `range()` 函式只接受整數作為引數，不接受浮點數。使用浮點數會導致 `TypeError`：

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- 你可以對 `step` 引數使用負整數，來產生遞減的整數序列：

```py
for num in range(40, 0, -10):
    print(num)
```

- `range()` 函式也可以搭配 `list` 建構式使用，來建立整數列表。`list` 建構式用來將可迭代物件轉換成列表。以下是產生包含 2 到 10（含）的偶數列表的範例：

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

## Python 中的 `enumerate()` 和 `zip()` 函式

- **`enumerate()`**：用來對序列進行迭代，並追蹤序列中每個項目的索引。`enumerate()` 函式接受一個可迭代物件作為引數，並回傳一個 `enumerate` 物件，該物件包含可迭代物件中每個項目的索引和值。

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')

# Result
# Index 0 and language Spanish
# Index 1 and language English
# Index 2 and language Russian
# Index 3 and language Chinese

```

- `enumerate()` 函式也可以在 `for` 迴圈外使用：

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

print(list(enumerate(languages)))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

- `enumerate()` 函式接受一個選用的 `start` 引數，指定計數的起始值。如果省略此引數，計數將從 `0` 開始。

- **`zip()`**：用來平行迭代多個可迭代物件。以下是使用 `zip()` 函式同時迭代 `developers` 和 `ids` 的範例：

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, dev_id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {dev_id}')


"""
Result

Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
"""

```

## Python 中的列表推導式

- **定義**：列表推導式允許你在一行內結合迴圈和條件，直接在中括號內建立新的列表。這讓程式碼更簡潔且通常更易讀。

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

## 可疊代方法

- **`filter()`**：用來根據條件過濾可迭代物件中的元素。它回傳一個只包含符合條件元素的迭代器。以下是建立一個只包含字元數超過四個字的單字新列表的範例：

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**：用來對可迭代物件中的每個項目套用函式，並回傳一個包含結果的新可迭代物件。以下是使用 `map()` 函式將攝氏溫度列表轉換為華氏溫度的範例：

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**：用來取得可迭代物件（如列表或 tuple）的總和。以下是使用 `sum()` 函式的範例：

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- 你也可以傳入選用的 `start` 引數，設定加總的初始值。以下是使用 `start` 引數作為位置引數的更新範例：

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- 你也可以選擇使用 `start` 引數作為關鍵字引數，如下所示：

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## Lambda 函式（程式）

- **定義**：Python 中的 lambda 函式是一種簡潔的方式來建立匿名函式（沒有名稱的函式）。
- Lambda 函式常用作為其他函式的引數。以下是 lambda 函式的範例：

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

- 使用 lambda 函式的最佳實踐包括不將它們指定給變數，保持簡潔易讀，並用於短小且一次性的函式。
