# --explanation--

Ein `log` ist eine Aufzeichnung von Ereignissen, Aktionen oder Nachrichten, die von einem Computersystem oder einer Software erstellt wird. Zum Beispiel:

`The system logs showed several errors before the crash.` – Das bedeutet, die Aufzeichnungen enthielten Informationen über aufgetretene Probleme.

Ein `crash` ist, wenn ein Computersystem oder Programm plötzlich nicht mehr richtig funktioniert. Zum Beispiel:

`The application crash caused all unsaved data to be lost.` – Das bedeutet, das Programm ist unerwartet abgestürzt, und Daten wurden nicht gespeichert.
