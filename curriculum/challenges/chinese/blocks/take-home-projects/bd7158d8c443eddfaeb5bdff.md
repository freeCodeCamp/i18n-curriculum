# --description--

构建一个功能上类似于此的全栈 JavaScript 应用程序：<a href="https://yoyo44.herokuapp.com/" target="_blank" rel="noopener noreferrer nofollow">https://yoyo44.herokuapp.com/</a>。使用你选择的网站构建工具完成该项目。

以下是你应该为此项目实现的用户故事：

**用户故事：** 作为一个未经验证的用户，你可以与浏览我所在区域的所有酒吧。

**用户故事：** 作为经过身份验证的用户，可以将自己添加到酒吧中，表示你今晚会去那里。

**用户故事：** 作为一名经过身份验证的用户，如果我不想再去那里，可以将自己从酒吧中删除。

**用户故事：** 作为未经身份验证的用户，当我登录时，我不需要再次搜索。

**提示：** 尝试使用 Yelp API 在用户搜索的城市中找到相关地点。 如果使用 Yelp 的 API，请务必在应用中提及。

当你完成后，请将一个确保正常运行的 demo（项目演示）托管在可以公开访问的平台上。 然后将 URL 提交到 Solution Link 字段中。 也可以将项目的源码链接提交到 GitHub Link 字段中。
