## HTML 表單元素和屬性

- **`form` element**：用於創建用戶輸入的 HTML 表單。
- **`action` 屬性**：用來指定表單資料應該送出的 URL。
- **`method` 屬性**：用來指定在傳送表單資料時要使用的 HTTP 方法。最常見的方法是 `GET` 和 `POST`。

```html
<form method="value-goes-here" action="url-goes-here">
  <!-- inputs go inside here -->
</form>
```

- **`input` 元素**：用於創建用戶輸入的輸入欄。
- **`type` 屬性**：用於指定輸入欄的型別。範例：`text`、`email`、`number`、`radio`、`checkbox` 等。
- **`placeholder` 屬性**：用來向使用者顯示提示，告訴他們在輸入欄中應輸入什麼內容。
- **`value` 屬性**：用於指定輸入的值。如果輸入的 `type` 是 `button`，`value` 屬性可以用來設定按鈕文字。
- **`name` 屬性**：用於為輸入欄指定名稱，該名稱在提交表單資料時作為鍵。對於圓鈕，將它們指定相同的 `name` 可將它們分組，因此群組中一次只能選擇一個選項。
- **`size` 屬性**：用來定義使用者在輸入時應該可見的字元數量。
- **`min` 屬性**：可用於像是 `number` 這類輸入型別，以指定輸入欄位允許的最小值。
- **`max` 屬性**：可用於像是 `number` 的輸入型別，以指定輸入欄位允許的最大值。
- **`minlength` 屬性**：用於指定輸入欄位中所需的最少字元數。
- **`maxlength` 屬性**：用於指定輸入欄位中允許的最大字元數。
- **`required` 屬性**：用於指定在提交表單之前必須填寫輸入欄。
- **`disabled` 屬性**：用於指定輸入欄應該被停用。
- **`readonly` 屬性**：用於指定輸入欄為唯讀。

:::interactive_editor

```html
<!-- Text input -->
<input 
  type="text"
  id="name"
  name="name"
  placeholder="e.g. Quincy Larson" 
  size="20"
  minlength="5"
  maxlength="30"
  required
/>

<!-- Number input -->
<input 
  type="number"
  id="quantity"
  name="quantity"
  min="2"
  max="10"
  disabled
/>

<!-- Button -->
<input type="button" value="Show Alert" />
```

:::

- **`label` 元素**：用於為輸入欄位創建標籤。
- **`for` 屬性**：用來指定標籤所對應的輸入欄。
- **隱式的表單相關**：輸入可以透過將輸入欄位包裹在 `label` 元素內來與標籤相關聯。

:::interactive_editor

```html
<form action="">
  <label>
    Full Name:
    <input type="text" />
  </label>
</form>
```

:::

- **明確的表單相關**：輸入欄位可以透過在 `label` 元素上使用 `for` 屬性來與標籤相關聯。

:::interactive_editor

```html
<form action="">
  <label for="email">Email Address: </label>
  <input type="email" id="email" />
</form>
```

:::

- **`button` 元素**：用於創建可點擊的按鈕。按鈕也可以有一個 `type` 屬性，用於控制按鈕被作用時的行為。例如 `submit`、`reset`、`button`。

:::interactive_editor

```html
<button type="button">Show Form</button>
<button type="submit">Submit Form</button>
<button type="reset">Reset Form</button>
```

:::

- **`fieldset` 元素**：用於將相關輸入欄位群組在一起。
- **`legend` 元素**：用於為輸入群組添加說明標題。

:::interactive_editor

```html
<!-- Radio group -->
<fieldset>
  <legend>Was this your first time at our hotel?</legend>

  <label for="yes-option">Yes</label>
  <input id="yes-option" type="radio" name="hotel-stay" value="yes" />

  <label for="no-option">No</label>
  <input id="no-option" type="radio" name="hotel-stay" value="no" />
</fieldset>

<!-- Checkbox group -->
<fieldset>
  <legend>
    Why did you choose to stay at our hotel? (Check all that apply)
  </legend>

  <label for="location">Location</label>
  <input type="checkbox" id="location" name="location" value="location" />

  <label for="price">Price</label>
  <input type="checkbox" id="price" name="price" value="price" />
</fieldset>
```

:::

- **聚焦狀態**：這是使用者選取輸入欄位時的狀態。

## 操作 HTML 表格元素和屬性

- **Table element**：用於創建 HTML 表格。
- **表格標頭（`thead`）元素**：用於將 HTML 表格中的標頭內容分組。
- **表格列 (`tr`) 元素**：用於在 HTML 表格中創建一列。
- **表格標題（`th`）元素**：用於在 HTML 表格中創建標題儲存格。
- **表格主體（`tbody`）元素**：用於將 HTML 表格中的主體內容分組。
- **表格資料儲存格（`td`）元素**：用於在 HTML 表格中創建資料儲存格。
- **Table Foot (`tfoot`) 元素**：用於將 HTML 表格中的頁腳內容分組。
- **`caption` 元素**：用於為 HTML 表格添加標題。
- **`colspan` 屬性**：用來指定表格儲存格應該跨越的行數。

:::interactive_editor

```html
<table>
  <caption>Exam Grades</caption>

  <thead>
    <tr>
      <th>Last Name</th>
      <th>First Name</th>
      <th>Grade</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>Davis</td>
      <td>Alex</td>
      <td>54</td>
    </tr>

    <tr>
      <td>Doe</td>
      <td>Samantha</td>
      <td>92</td>
    </tr>

    <tr>
      <td>Rodriguez</td>
      <td>Marcus</td>
      <td>88</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <td colspan="2">Average Grade</td>
      <td>78</td>
    </tr>
  </tfoot>
</table>
```

:::

## 使用 HTML 工具

- **HTML validator**：一種檢查 HTML 程式碼語法以確保其有效的工具。
- **DOM inspector**：一個允許你檢查和修改網頁 HTML 結構的工具。
- **Devtools**：一組直接內建於瀏覽器中的網頁開發工具，幫助你偵錯、評測及分析網頁。
