# --description--

I numeri sono i mattoni della logica di programmazione! In effetti, è difficile pensare a un’attività di programmazione utile che non coinvolga almeno un po’ di matematica di base… quindi sapere come funzionano i numeri è ovviamente molto importante. Per fortuna, è anche abbastanza semplice.

| Operatore | Descrizione                  | Esempio            |
|----------|------------------------------|--------------------|
| +        | Addizione                    | 5 + 3 = 8          |
| -        | Sottrazione                  | 5 - 3 = 2          |
| *        | Moltiplicazione              | 5 * 3 = 15         |
| **       | Esponenziazione              | 5 ** 3 = 125       |
| /        | Divisione                   | 6 / 3 = 2          |
| %        | Modulo                      | 5 % 3 = 2          |


# --assignment--

Data la tabella degli operatori sopra, scrivi un semplice frammento di programma che dimostri l’uso di ogni operatore con la variabile `x` inizializzata a `10`.

# --questions--
