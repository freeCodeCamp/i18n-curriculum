# --description--

`Whole numbers` — це базові числові типи даних у програмуванні та математиці. Це числа без дробової або десяткової частини, включно з усіма натуральними числами (наприклад, 1, 2, 3), від’ємними числами (наприклад, -4, -9) та нулем (0).

# --questions--

## --text--

Що таке `whole numbers` у контексті програмування?

## --answers--

Числа з десятковими частинами

### --feedback--

`Whole numbers` не мають десяткових частин; це цілі числа, як-от 0, 1, 2 тощо.

---

Від’ємні числа

### --feedback--

`Whole numbers` включають усі натуральні числа, від’ємні числа та нуль.

---

Числа без дробів або десяткових частин

---

Літери та символи

### --feedback--

`Whole numbers` позначають стандартні числові цифри, а не літери чи символи.
