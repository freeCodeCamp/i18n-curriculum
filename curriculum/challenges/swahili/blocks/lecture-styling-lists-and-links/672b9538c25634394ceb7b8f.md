# --interactive--

Kuna hali tofauti za kiungo, ikiwa ni pamoja na `link`, `visited`, `hover`, `focus`, na `active`. Hali hizi ni muhimu kusaidia watumizi kutambua viungo na kutoa mrejesho wazi baada ya mwingiliano, jambo ambalo huboresha matumizi na Ufikikaji.

Kupamba hali hizi tofauti za viungo ni muhimu kwa matumizi na Ufikikaji, kwani hutoa dalili za kuona kuhusu hali ya sasa ya kiungo. Hii huwasaidia watumizi kuelewa viungo ambavyo wamevitembelea, kiungo wanachokifanya mwingiliano nacho, na kinachotokea wanapobofya.

Kwa watumizi wenye udhaifu wa kuona au ulemavu wa akili, mitindo hii tofauti inaweza kufanya urambazaji kuwa rahisi zaidi na wa kueleweka.

Zaidi ya hayo, hali wazi za viungo huboresha uzoefu wa mtumiaji kwa kutoa mrejesho wa papo hapo juu ya mwingiliano wa mtumiaji, kupunguza mkanganyiko na kuboresha urahisi wa kutumia tovuti.

Hali hizi zinaweza kupambwa kwa kutumia kitu kinachoitwa `pseudo-classes` katika CSS.

Darasa la bandia ni neno la kiufundi linaloongezwa kwenye kichaguzi kinachobainisha hali maalum ya kipengele kilichochaguliwa.

Kwa mfano, `:hover` inaweza kubadilisha rangi ya kitufe wakati kidokezo cha mtumiaji kinapobeba juu yake, wakati `:visited` inaweza kubadilisha rangi ya kiungo ambacho tayari kimevitembelewa.

Darasa la bandia hukuruhusu kupamba vipengele kulingana na hali yao au mwingiliano wa mtumiaji bila haja ya alama za ziada katika HTML yako.

Sintaksia ya `pseudo-class` inaonekana kama ifuatavyo ambapo `A` ni kichaguzi na `:B` ni `pseudo-class`:

```css
A:B {
  property: value;
}
```

Ili kuelewa vyema jinsi ya kupamba hali tofauti za viungo, tuchunguze mifano kadhaa.

Darasa la bandia la `:link` linapamba viungo visivyotembelewa, kuonyesha kuwa vinaweza kubofywa.

Hapa kuna mfano wa kulenga kipengele cha nanga na kutumia darasa la bandia la `:link`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Normal state (unvisited link) */
a:link { 
  color: red;
}
```

:::

Mfano huu hapo juu utabadilisha rangi ya kiungo kutoka buluu ya kawaida kuwa `red` wakati hakijatembelukwa.

`:visited` inapamba viungo ambavyo tayari vimetembelewa au kubofiwa, kusaidia watumizi kufuatilia viungo walivyovibofya awali. Hapa kuna mfano wa matumizi ya darasa la bandia la `:visited`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="https://www.freecodecamp.org/learn/" target="_blank">freeCodeCamp</a>
```

```css
/* Visited link */
a:visited {
  color: green;
}
```

:::

Msimbo huu utabadilisha rangi ya kiungo kuwa `green` wakati kimebofiwa.

`:hover` hubadilisha mtindo wa kiungo wakati mtumiaji anapobeba kidokezo juu yake, kutoa dalili ya kuona kuwa kiungo kina mwingiliano. Hapa kuna mfano wa matumizi ya darasa la bandia la `:hover`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Hover state */
a:hover {
  color: green;
}
```

:::

Msimbo huu utabadilisha rangi za viungo kuwa `green` wakati panya anapobeba kidokezo juu yake.

`:focus` huongeza mitindo kuzunguka kiungo wakati kimekuwa makini, kama wakati unapotumia kibodi kuvinjari, au kuboresha Ufikikaji.

Hapa kuna mfano wa kutumia sifa ya `outline` kuonyesha mstari wa rangi ya chungwa ulio imara wakati kiungo kimekuwa makini.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Focus state */
a:focus {
  outline: 2px solid orange;
}
```

:::

`:active` hubadilisha mitindo ya kiungo wakati kiungo kinabofiwa, kutoa mrejesho wa papo hapo kwa mtumiaji kuwa kitendo chake kinarekodiwa. Hapa kuna mfano wa matumizi ya darasa la bandia la `:active`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Active state */
a:active {
  color: pink;
}
```

:::

Mfano huu wa msimbo utabadilisha rangi ya kiungo kuwa `pink` mara moja kiungo kinapobofiwa.
