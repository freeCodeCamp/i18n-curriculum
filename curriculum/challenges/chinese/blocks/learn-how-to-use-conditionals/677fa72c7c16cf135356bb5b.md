# --explanation--

Mark 说，`If the application has memory leaks, it can lead to crashes`。

这里的关键术语是 `memory leaks`。当程序为临时任务分配内存但在任务完成后未能释放时，就会发生这种情况。例如：

- `If the application uses too much memory, it could crash。` - 这表明内存使用是导致崩溃的问题。

- `Memory leaks` 会导致性能问题和崩溃。
