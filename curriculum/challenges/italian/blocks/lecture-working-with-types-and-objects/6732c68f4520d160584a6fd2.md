# --interactive--

Il metodo `toString()` è una funzionalità fondamentale in JavaScript che converte un valore nella sua rappresentazione come stringa. È un metodo che puoi usare per numeri, booleani, array e oggetti. Uno degli usi più comuni di `toString()` è convertire un numero nella sua rappresentazione come stringa. Ecco un esempio:

:::interactive_editor

```js
const num = 10;
console.log(num.toString()); // "10"
```

:::

Questo metodo accetta una base opzionale che è un numero da `2` a `36`. Questa base rappresenta la base numerica, come base 2 per il binario o base 8 per l’ottale. Se la base non è specificata, per impostazione predefinita è base 10, cioè decimale. Ecco un esempio di passaggio di `2` come argomento al metodo `toString()`:

:::interactive_editor

```js
const num = 10;
console.log(num.toString(2)); // "1010"
```

:::

Il risultato sarà `1010` che è la rappresentazione binaria del numero decimale `10`.

Puoi anche usare il metodo `toString()` per convertire array e oggetti in stringhe. Gli array hanno un’implementazione personalizzata di `toString()` che converte ogni elemento in stringa e li unisce con delle virgole. Ecco un esempio:

:::interactive_editor

```js
const arr = [1, 2, 3];
console.log(arr.toString()); // "1,2,3"
```

:::

In questo esempio tutti gli elementi dell’array sono uniti insieme per formare la stringa `1,2,3`.

Quando il metodo `toString()` è usato con gli oggetti, il risultato non sarà una versione stringificata delle proprietà dell’oggetto.

:::interactive_editor

```js
const person = {
  name: "John",
  age: 30,
  isStudent: true
};

console.log(person.toString()); // "[object Object]"
```

:::

In questo esempio, il risultato sarà la rappresentazione stringa predefinita per l’oggetto che è `[object Object]`. Per ottenere una versione stringificata delle proprietà dell’oggetto `person` dovrai usare `JSON.stringify()` che imparerai meglio nelle lezioni future.

In conclusione, il metodo `toString()` è usato per convertire valori in stringhe. Capire come funziona con diversi tipi di dati e come personalizzarlo per i tuoi oggetti può migliorare molto la tua capacità di manipolare e mostrare dati nelle tue applicazioni JavaScript.
