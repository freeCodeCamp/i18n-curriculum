# --interactive--

عند بناء برامجك، غالبًا ما تريد حدوث شيء ما بعد تأخير، أو تكراره على فترات منتظمة. هنا تأتي فائدة طريقتي `setTimeout()` و `setInterval()`. كلتا الطريقتين تقبلان معلمتين: دالة وتأخير.

لنلقِ نظرة على كل طريقة بمزيد من التفصيل حتى تبدأ باستخدامها في مشاريع الويب الخاصة بك.

تسمح لك `setTimeout()` بتأخير تنفيذ إجراء لمدة محددة. إليك الصيغة النحوية الأساسية لطريقة `setTimeout()`:

```js
setTimeout(functionToRun, delay);
```

`functionToRun` هي الشفرة التي تريد تنفيذها بعد تأخير محدد، و`delay` هو الوقت (بالملي ثانية) الذي تنتظره قبل تشغيل الدالة. إليك مثالًا:

:::interactive_editor

```js
setTimeout(function () {
 console.log("This runs after 3 seconds");
}, 3000);
```

:::

يمكنك أيضًا استخدام الدالة السهمية كاستدعاء راجع، مثل هذا:

:::interactive_editor

```js
setTimeout(() => {
 console.log("This runs after 3 seconds"); 
}, 3000);
```

:::

تستمر `setInterval()` في تكرار تنفيذ دالة على فاصل زمني محدد. تشبه وجود منبه يرن كل بضع ثوانٍ. `setInterval()` مثالية عندما تريد حدوث شيء مرارًا وتكرارًا، مثل تحديث البيانات أو إنشاء حركة متواصلة في الرسوم المتحركة.

تمامًا مثل `setTimeout()`، تأخذ دالة وتأخيرًا:

```js
setInterval(functionToRun, delay);
```

`functionToRun` هي الشفرة التي تريد تشغيلها بشكل متكرر، و`delay` هو الوقت بالملي ثانية بين كل تنفيذ لتلك الدالة. إليك مثالًا:

:::interactive_editor

```js
setInterval(() => {
 console.log("This runs every 2 seconds");
}, 2000);

// Result:
// This runs every 2 seconds
// This runs every 2 seconds
// This runs every 2 seconds
// ...
```

:::

بما أن `setInterval()` تستمر في تنفيذ الدالة المقدمة على الفاصل الزمني المحدد، قد ترغب في إيقافها. للقيام بذلك، عليك استخدام طريقة `clearInterval()`.

لتنفيذ ذلك، تأخذ `clearInterval()` معرف الـ `setInterval()` الذي تريد إيقافه. قد يكون هذا متغيرًا قمت بتعيينه للفاصل الزمني. إحدى طرق إيقاف الفاصل هي داخل `setTimeout()`، إذ ستوقف الفاصل بعد وقت معين:

:::interactive_editor

```js
const intervalID = setInterval(() => {
 console.log("This will stop after 5 seconds");
}, 1000);

setTimeout(() => {
 clearInterval(intervalID);
}, 5000);
```

:::

يمكنك أيضًا إيقاف مهلة باستخدام طريقة `clearTimeout()`:

:::interactive_editor

```js
let timeoutID = setTimeout(() => {
 console.log("This will not run");
}, 5000);

clearTimeout(timeoutID);
```

:::

مع هذا الكود، لن يتم تشغيل `setTimeout()` على الإطلاق. لذلك، طريقة أكثر أناقة هي عبر DOM، بإضافة مستمع حدث نقر إلى زر لإيقاف المهلة:

:::interactive_editor

```html
<h1>Cancel Timeout Example</h1>
<button id="cancelButton">Cancel Timeout</button>
<script src="index.js"></script>
```

```js
let timeoutID = setTimeout(() => {
  console.log("This will run if not canceled");
}, 5000);

document
.querySelector("#cancelButton")
.addEventListener("click", () => {
  clearTimeout(timeoutID);
  console.log("Timeout canceled!");
});
```

:::
