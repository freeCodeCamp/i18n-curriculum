# --explanation--

`Present Perfect Continuous` 時態用來描述從過去開始且仍在進行的動作。它著重於動作的持續時間或延續性。此時態的結構是 `have/has + been + verb(-ing)`。例如：

- `We have been developing a new app for accessibility.`－這表示開發是在過去開始且仍在進行中。

- `He has been fixing bugs in the software for several hours。`－這表示他早先開始修正軟體中的錯誤，並且仍在進行中。
