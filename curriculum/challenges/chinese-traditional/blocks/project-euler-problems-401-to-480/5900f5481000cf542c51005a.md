# --description--

$12n$ 位音樂家參加音樂節。在第一天，他們組成 $3n$ 個四重奏並練習一整天。

這是一場災難。最終，所有音樂家都決定他們再也不會同意與他們四重奏的任何成員一起演奏。

在第二天，他們組成 $4n$ 個三重奏，每位音樂家都避免與他之前的四重奏夥伴同組。

令 $f(12n)$ 為在 $12n$ 位音樂家中組織三重奏的方法數。

你得到 $f(12) = 576$ 和 $f(24)\bmod 1\\,000\\,000\\,007 = 509\\,089\\,824$。

求 $f(600)\bmod 1\\,000\\,000\\,007$。
