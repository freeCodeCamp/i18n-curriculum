# --interactive--

تتحمل CSS مسؤولية أنماط صفحة الويب. تتكون كل هذه الأنماط من قواعد CSS مختلفة.

تتكون قاعدة CSS من جزأين رئيسيين: **محدِّد** وكتلة **إعلان**.

لنلقِ نظرة على الصيغة النحوية الأساسية:

```css
selector {
  property: value;
}
```

المحدِّد هو نمط يُستخدم في CSS لتحديد واستهداف عناصر HTML معينة لتنسيقها.

تشمل أمثلة المحدِّدات محدِّدات النوع، ومحدِّدات الصنف، ومحدِّدات المعرف.

الأقواس المتعرجة الموجودة في الصيغة النحوية الأساسية تُعرف بكتلة الإعلان. تطبق كتلة الإعلان مجموعة من الأنماط لمحدِّد معين أو لمحدِّدات متعددة.

داخل كتلة الإعلان، ستجد سلسلة من الإعلانات. يتكون كل إعلان من خاصية وقيمة.

الخاصية هي معرف CSS الذي يحدد الميزة التي يتم تنسيقها. مثال على الخاصية هو خاصية `background-color`.

القيمة هي الإعداد المحدد الذي يُطبق على تلك الخاصية. على سبيل المثال، إذا كانت الخاصية هي `background-color`، فقد تكون القيمة `purple`، التي تضبط لون الخلفية إلى اللون البنفسجي.

بعد كل اسم خاصية، يجب وضع نقطتين رأسيتين، وبعد كل قيمة يجب وضع فاصلة منقوطة.

الآن بعد أن تعرفت على صيغة قاعدة CSS، لنلقِ نظرة على مثال. فعّل المحرر التفاعلي وانقر على تبويب `styles.css` لرؤية شفرة CSS.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Learning about CSS</h1>

<p>Example paragraph element</p>
<p>Another example paragraph element</p>
```

```css
p {
  color: blue;
}
```

:::

في قاعدة CSS هذه، يستهدف محدِّد النوع جميع عناصر الفقرة في الصفحة.

داخل كتلة الإعلان، يوجد إعلان واحد.

يتكون الإعلان من خاصية `color` مع قيمة مضبوطة على `blue`. هذا سيغير لون النص لجميع عناصر الفقرة إلى `blue`.

إذا أردت تطبيق نفس مجموعة الأنماط على محدِّدات متعددة، يمكنك إنشاء قائمة محدِّدات يفصل بينها فاصلة.

إليك مثالًا على تنسيق محدِّدات متعددة (لتفاعل مع المثال، فعّل المحرر التفاعلي):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1 id="title">Example heading</h1>
<h2 class="subheading">Example subheading</h2>
<p>This paragraph is not affected by the selector.</p>
```

```css
#title,
.subheading {
  color: navy;
}
```

:::

في قائمة المحددات هذه، يوجد محدد `id` يستهدف عنصر HTML الذي يحمل قيمة `id` وهي `title`. يجب أن تبدأ جميع محددات `id` برمز الهاش `#`.

ثم هناك فاصلة تليها محدِّد صنف `class` يستهدف جميع عناصر HTML التي تحمل قيمة `class` هي `subheading`. يجب أن تبدأ كل محدِّدات الصنف بنقطة `.`

ستتلقى قائمة المحدِّدات بأكملها نفس التنسيق، مع ضبط لون النص إلى `navy`.
