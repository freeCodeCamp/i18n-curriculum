# --description--

고정된 정수 $a$, $b$, $c$에 대해, 미친 함수 $F(n)$을 다음과 같이 정의하세요:

$$\begin{align}
  & F(n) = n - c \\;\text{ 모든 } n > b \text{에 대해} \\\\
  & F(n) = F(a + F(a + F(a + F(a + n)))) \\;\text{ 모든 } n ≤ b \text{에 대해}.
\end{align}$$

또한, $S(a, b, c) = \displaystyle\sum_{n = 0}^b F(n)$로 정의하세요.

예를 들어, $a = 50$, $b = 2000$, $c = 40$일 때, $F(0) = 3240$이고 $F(2000) = 2040$입니다. 또한, $S(50, 2000, 40) = 5\\,204\\,240$입니다.

$S({21}^7, 7^{21}, {12}^7)$의 마지막 9자리를 구하세요.
