# --description--

軸に沿った直方体は、パラメータ $\{ (x_0,y_0,z_0), (dx,dy,dz) \}$ で指定され、$x_0 ≤ X ≤ x_0 + dx$、$y_0 ≤ Y ≤ y_0 + dy$、$z_0 ≤ Z ≤ z_0 + dz$ を満たすすべての点 ($X$,$Y$,$Z$) から構成されます。直方体の体積は積 $dx × dy × dz$ です。複数の直方体の合計体積はそれらの和集合の体積であり、直方体が重なっている場合は個々の体積の合計より小さくなります。

$C_1, \ldots, C_{50000}$ を50000個の軸に沿った直方体の集合とし、$C_n$ のパラメータは次のように定義されます。

$$\begin{align}
  & x_0 = S_{6n - 5} \\; \text{モジュラス} \\; 10000    \\\\
  & y_0 = S_{6n - 4} \\; \text{モジュラス} \\; 10000    \\\\
  & z_0 = S_{6n - 3} \\; \text{モジュラス} \\; 10000    \\\\
  & dx = 1 + (S_{6n - 2} \\; \text{モジュラス} \\; 399) \\\\
  & dy = 1 + (S_{6n - 1} \\; \text{モジュラス} \\; 399) \\\\
  & dz = 1 + (S_{6n} \\; \text{モジュラス} \\; 399)     \\\\
\end{align}$$

ここで $S_1, \ldots, S_{300000}$ は「遅延フィボナッチ生成器」から得られます。

$1 ≤ k ≤ 55$ のとき、$S_k = [100003 - 200003k + 300007k^3] \\; (モジュラス \\; 1000000)$

$56 ≤ k$ のとき、$S_k = [S_{k - 24} + S_{k - 55}] \\; (モジュラス \\; 1000000)$

したがって、$C_1$ のパラメータは $\{(7,53,183), (94,369,56)\}$、$C_2$ は $\{(2383,3563,5079), (42,212,344)\}$ となります。

最初の100個の直方体 $C_1, \ldots, C_{100}$ の合計体積は723581599です。

すべての50000個の直方体 $C_1, \ldots, C_{50000}$ の合計体積はいくつですか？
