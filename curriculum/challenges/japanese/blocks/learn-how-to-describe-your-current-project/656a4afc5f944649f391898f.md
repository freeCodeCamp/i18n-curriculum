# --description--

`Different from` は、2つのものが同じではないことを示す表現です。例えば：

- `My phone is different from yours.` - これは2台の電話が同じではないことを意味します。

- `Her job is different from mine.` - これは彼女の仕事とあなたの仕事が同じではないことを意味します。

`different` は形容詞で、この構文では通常 `from` が続くことに注意してください。
