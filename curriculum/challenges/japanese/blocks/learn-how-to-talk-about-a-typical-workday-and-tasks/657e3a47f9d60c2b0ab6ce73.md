# --description--

`make sure` は、何かについて確信や確認を表すために使われます。例えば：

`I check the list twice to make sure I have everything.` - これは、リストを2回見て、何も抜けていないことを確かめるという意味です。

`work properly` は、正しく機能する、または期待通りに動作することを意味します。例えば：

`It's important that all machines work properly for safety.` - これは、人々の安全を守るために機械が正しく動かなければならないという意味です。
