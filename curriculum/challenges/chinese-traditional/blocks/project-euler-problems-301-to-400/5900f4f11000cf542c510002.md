# --description--

考慮所有格點 ($a$, $b$, $c$)，其中 $0 ≤ a, b, c ≤ N$。

從原點 $O(0, 0, 0)$ 向其他格點繪製所有線段。令 $D(N)$ 為此類不同線段的數量。

你被給予 $D(1\,000\,000) = 831\,909\,254\,469\,114\,121$。

求 $D({10}^{10})$。請將你的答案寫成前九位數字，接著是後九位數字。
