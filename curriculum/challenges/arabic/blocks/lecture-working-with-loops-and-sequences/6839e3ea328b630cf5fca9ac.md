# --description--

كما تعلمت في الوحدات السابقة، تُستخدم الحلقات لتكرار كتلة من الشفرة لعدد محدد من المرات. في هذا الدرس ستتعلم كيفية العمل مع أنواع مختلفة من الحلقات في Python.

أول حلقة سنستعرضها هي حلقة `for`. إليك مثالًا على استخدام حلقة `for` للتكرار خلال قائمة وطباعة كل عنصر في وحدة التحكم:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)
```

النتيجة ستكون:

```md
Rust
Java
Python
C++
```

لاحظ أن `print(language)` مُزاحة إلى الداخل داخل الحلقة. بدون هذه الإزاحة، ستحصل على `IndentationError`:

```python
"""
Traceback (most recent call last):
  File "<stdin>", line 4, in <module>
IndentationError: expected an indented block after 'for' statement on line 3
"""
```

يمكنك أيضًا استخدام حلقة `for` للتكرار خلال عناصر قابلة للتكرار أخرى مثل سلسلة نصية. إليك مثالًا على استخدام حلقة `for` للتكرار خلال السلسلة النصية `code` وطباعة كل حرف:

```py
for char in 'code':
    print(char)
```

النتيجة ستكون:

```md
c
o
d
e
```

تمامًا كما في JavaScript، يمكنك أيضًا التسكين باستخدام حلقات `for` في Python. إليك مثالًا على استخدام حلقة `for` مسكنة:

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)
```

الحلقة الخارجية ستكرر خلال كل `category` في قائمة `categories`. لكل `category`، الحلقة الداخلية ستكرر خلال كل `food` في قائمة `foods`. هذه هي النتيجة التي ستُطبع في وحدة التحكم:

```md
Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
```

نوع آخر من الحلقات يمكنك استخدامه في Python هو حلقة `while`. هذا النوع من الحلقات يكرر كتلة من الشفرة حتى يصبح الشرط `False`. إليك مثالًا على استخدام حلقة `while` للعبة تخمين:

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')
```

في هذا المثال لدينا متغير `secret_number` بقيمة `3` وتخمين أولي بقيمة `0`. ثم نستخدم الدالة `input` للحصول على مُدخَل من المستخدم، ثم نحول السلسلة النصية المُدخلة إلى عدد صحيح باستخدام الدالة `int()`، ونعينها إلى المتغير `guess`. إذا خمن المستخدم بشكل صحيح بإدخال `3`، تُنهي حلقة `while` ويُطبع في وحدة التحكم الرسالة `You got it!`. وإلا، تُطبع الرسالة `Wrong! Try again.` في وحدة التحكم، وتتكرر الحلقة، مما يدفع المستخدم لتخمين مرة أخرى.

إليك مثالًا على النتيجة:

```md
Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
```

تمامًا كما في JavaScript، يدعم Python عبارات `break` و `continue`.

تُستخدم العبارة `break` لإيقاف تنفيذ الحلقة. إليك مثالًا على استخدام العبارة `break` لقائمة `developer_names`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

في هذا المثال، نكرر خلال قائمة `developer_names` ونطبع كل اسم في وحدة التحكم. إذا كان الاسم يساوي `Naomi`، نخرج من الحلقة. هذا يؤدي إلى طباعة الاسم `Jess` فقط في وحدة التحكم.

تُستخدم العبارة `continue` لتخطي التكرار الحالي للحلقة والانتقال إلى التكرار التالي. دعنا نُعدّل المثال السابق لاستخدام العبارة `continue` بدلًا من `break`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

الآن ستكون النتيجة في وحدة التحكم مختلفة. تُطبع الأسماء `Jess` و `Tom` لأن العبارة `continue` تتخطى التكرار الثاني للحلقة عندما يكون `developer` مساويًا لـ `Naomi`، ولا تطبع ذلك الاسم في وحدة التحكم.

يمكن دمج كل من حلقات `for` و `while` مع بند `else`، الذي يُنفذ فقط عندما لا تُنهى الحلقة بواسطة عبارة `break`. إليك مثالًا على استخدام عدة حلقات `for`:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

في هذا المثال لدينا قائمة كلمات عشوائية، وتُستخدم حلقة `for` للتكرار خلال كل كلمة. داخل الحلقة الخارجية `for`، لدينا حلقة `for` أخرى للتكرار خلال كل حرف من كل كلمة. إذا كانت النسخة بالأحرف الصغيرة من الحرف هي حرف متحرك، نطبع الكلمة متبوعة بالحروف المتحركة التي تحتويها، ثم نخرج من الحلقة الداخلية. إذا لم تحتوي الكلمة على حروف متحركة، نطبع رسالة تشير إلى ذلك.

إليك كيف تبدو النتيجة في وحدة التحكم:

```md
'sky' has no vowels
'apple' contains the vowel 'a'
'rhythm' has no vowels
'fly' has no vowels
'orange' contains the vowel 'o'
```

الحلقات شائعة جدًا في Python، لذا من المهم أن تتقنها. في الدروس القادمة، ستتعلم كيفية العمل مع الدالتين `enumerate()` و `range()` في الحلقات.
