### --feedback--

Ці три слова використовують, щоб говорити про те, що було б добре зробити в минулому, але цього не зробили. Третє слово є `Past Participle` від `run`.

# --explanation--

`Should have` + `Past Participle` використовується, щоб висловити жаль або пораду щодо минулого. Наприклад:

- `You should have run the program on a different machine to check for compatibility.` — тестування на іншій машині було б корисним, але цього не зробили.

- `I should have run the tests before deploying the app.` — запуск тестів був правильним кроком, але його не зробили вчасно.

У цьому випадку Софі каже Марку, що йому слід було запустити тестовий набір, тобто це було важливо, але він цього не зробив.
