# --description--

بشكل مربك بعض الشيء، يمكن لأي عنصر أن يكون حاوية مرنة وعنصرًا مرنًا في الوقت نفسه. بعبارة أخرى، يمكنك أيضًا وضع `display: flex` على عنصر مرن ثم استخدام flexbox لترتيب أطفاله.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/flex-box/flexbox-03.png" style="width: 95%; height: 95%;" alt="a flex container that has multiple flex items, within those flex items are nested flex items as well. Making the parent of those nested flex items also a flex container." />

إن إنشاء وتسكين عدة حاويات مرنة وعناصر مرنة هو الطريقة الأساسية لبناء تخطيطات معقدة. الصورة التالية تم تحقيقها باستخدام flexbox فقط لترتيب وتحديد حجم ووضع العناصر المختلفة. flexbox أداة قوية جدًا.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/flex-box/flexbox-02.png" alt="a complex layout of flex items and flex containers. There are multiple flex containers nested into each other, thus making them flex items as well"/>

بالتأكيد، تعرض الصورة تمثيلًا لتخطيط CSS Flexbox مع حاويات مرنة مسكنة. الحاوية الخارجية معلمة بأنها "أيضًا حاوية مرنة" ومظللة باللون الأزرق، وداخلها حاوية أخرى معنونة بـ "مع عناصر مرنة" باللون الأحمر. داخل الحاوية الحمراء، هناك ثلاثة عناصر معنونة بـ "عناصر مرنة" باللون الخوخي. هذا يوضح أن الحاوية المرنة يمكن أن تكون مسكنة داخل حاوية مرنة أخرى وتحتوي على عناصر مرنة خاصة بها، مما يبرز الطبيعة التكرارية لهياكل تخطيط Flexbox.
