# --description--

在每個 `li` 元素中，都需要有一個錨元素。

第一個錨元素的文本應爲 `About`，`href` 屬性值應爲 `"#about"`。 `about` 前面的哈希符號代表一個 `id` 名稱，稍後將添加到項目中。

第二個錨元素的文本應爲 `Posts`，`href` 屬性值應爲 `"#posts"`。

第三個錨元素的文本應爲 `Contact`，`href` 屬性值應爲 `"#contact"`。

```js
assert.strictEqual(anchorElement?.innerText.trim(), "About");
```

你的第一個錨元素應具有 `href` 屬性，設置爲 `"#about"`。

```js
assert.strictEqual(hrefAttribute, "#about");
```
