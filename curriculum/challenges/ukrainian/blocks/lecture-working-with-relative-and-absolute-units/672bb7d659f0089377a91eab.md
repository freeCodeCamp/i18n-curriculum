# --interactive--

Відсотки в CSS — це відносні одиниці, які дозволяють визначати розміри, виміри та інші властивості як пропорцію від батьківського елемента. Коли ви використовуєте відсоткове значення, ви фактично кажете: «Зроби це X% від свого контейнера».

Наприклад, якщо ви встановите `width: 50%` для елемента, він займе половину ширини свого батьківського контейнера. Це робить відсотки надзвичайно корисними для створення адаптивних дизайнів, які підлаштовуються під різні розміри екранів.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <div class="box"></div>
</div>
```

```css
.container {
  width: 400px;
  height: 200px;
  background-color: lightgray;
}

.box {
  width: 50%;
  height: 100%;
  background-color: red;
}
```

:::

Відсотки ідеально підходять для створення плавних макетів, які підлаштовуються під різні розміри екранів. Наприклад, встановлення контейнера на `width: 80%` гарантує, що він займе 80% ширини свого батьківського елемента, незалежно від пристрою.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="parent">
  <div class="child"></div>
</div>
```

```css
.parent {
  width: 100%;
  height: 300px;
  background-color: lightblue;
}

.child {
  width: 80%;
  height: 100%;
  background-color: red;
}
```

:::

Використання відсотків для гнучких зображень — ще одна поширена практика. Застосовуючи `max-width: 100%` до зображень, ви дозволяєте їм масштабуватися на менших екранах, зберігаючи при цьому співвідношення сторін.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<img src="https://placehold.co/150x150" alt="Example Product Image" />
```

```css
img {
  max-width: 100%;
  height: auto;
}
```

:::

Хоча це менш поширено, відсотки також можна використовувати для розмірів шрифтів, щоб створити масштабовану типографіку. Наприклад, `font-size: 120%` зробить текст на 20% більшим за розмір шрифту батьківського елемента.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="text-container">
  Parent text.
  <p class="text">This is some example text.</p>
</div>
```

```css
.text-container {
  font-size: 16px;
}

.text {
  font-size: 120%;
}
```

:::

Відсотки можуть бути особливо корисними для вертикального центрування. Ось приклад, як можна використати відсотки з властивістю `transform` для вертикального центрування елемента.

Цей приклад позиціонує елемент на 50% від верхньої межі контейнера, а потім використовує `transform`, щоб підняти його вгору на половину власної висоти, ефективно центруючи його по вертикалі.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="centered"></div>
```

```css
.centered {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
  width: 300px;
  height: 300px;
  background-color: red;
}
```

:::

Ви дізнаєтеся більше про те, як працюють абсолютне позиціювання та властивості transform, у майбутніх уроках.

Пам’ятайте, що відсотки завжди відносні до чогось. Для горизонтальних властивостей, як `width`, вони відносні до ширини батьківського елемента. Для вертикальних властивостей, як `height`, зазвичай відносні до висоти батьківського елемента (якщо вона задана).

Однак будьте обережні при вкладенні елементів із відсотковими розмірами, оскільки це може призвести до несподіваних результатів. Також майте на увазі, що відсоткові висоти можуть бути складними, якщо у батьківського елемента не визначена висота.

Підсумовуючи, відсотки в CSS — це потужний інструмент для створення гнучких, адаптивних дизайнів. Використовуйте їх, коли хочете, щоб елементи масштабувалися пропорційно до своїх контейнерів або коли потрібні макети, які бездоганно підлаштовуються під різні розміри екранів.
