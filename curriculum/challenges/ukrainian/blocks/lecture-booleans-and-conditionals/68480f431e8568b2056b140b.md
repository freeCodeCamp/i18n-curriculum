# --description--

У попередньому уроці ви дізналися, як використовувати оператори порівняння та умовні інструкції для керування потоком ваших програм.

Хоча вони дуже потужні, часто виникають ситуації, коли потрібно порівнювати кілька значень одночасно. Це може призвести до вкладених умовних інструкцій, наприклад:

```python
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

У наведеному прикладі спочатку перевіряється, чи є `is_citizen` `True`. Якщо так, тоді виконується вкладена інструкція `if` і перевіряється, чи `age` більше або дорівнює `18`. Оскільки `age` більше або дорівнює `18`, у термінал буде виведено повідомлення `You are eligible to vote`. Якби `is_citizen` було `False`, тоді в термінал було б виведено повідомлення `You are not eligible to vote`.

Якщо ви працюєте з більш складними умовними інструкціями, то можете використовувати вбудовані в Python оператори `and`, `or` і `not`.

Але перед тим, як заглибитися в ці оператори, давайте розглянемо, що таке правдиві та хибні значення.

У Python кожне значення має вбудоване булеве значення, або вбудоване відчуття того, чи слід його розглядати як `True` чи `False` у логічному контексті. Багато значень вважаються **правдивими**, тобто вони оцінюються як `True` у логічному контексті. Інші — **хибними**, тобто оцінюються як `False`.

Ось кілька хибних значень:

- `None`
- `False`
- Ціле число `0`
- Число з плаваючою комою `0.0`
- Порожні рядки `""`

Інші значення, як-от ненульові числа та непорожні рядки, є правдивими.

Якщо ви хочете перевірити, чи є значення правдивим чи хибним, ви можете використати вбудовану функцію `bool()`. Вона явно перетворює значення на його булевий еквівалент і повертає `True` для правдивих значень і `False` для хибних. Ось кілька прикладів:

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

Тепер, коли ви розумієте правдиві та хибні значення, розглянемо булеві оператори, які також відомі як логічні оператори. Це спеціальні оператори, які дозволяють поєднувати кілька виразів для створення більш складної логіки прийняття рішень у вашому коді.

У Python є три булеві оператори: `and`, `or` і `not`.

Спершу розглянемо оператор `and`.

Оператор `and` приймає два операнди і повертає перший операнд, якщо він хибний, інакше повертає другий операнд. Обидва операнди мають бути правдивими, щоб вираз дав правдиве значення.

Ось приклад:

```python
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

У наведеному прикладі число 25 виводиться в термінал, тому що оператор `and` оцінить другий операнд, якщо перший операнд є `True`. Оператор `and` відомий як оператор короткого замикання. Коротке замикання означає, що Python перевіряє значення зліва направо і зупиняється, щойно визначає фінальний результат.

Ви часто використовуватимете `and` у умовних інструкціях `if`, щоб перевірити, чи виконуються кілька умов. Ось як можна переписати попередній приклад, використовуючи оператор `and` замість вкладених інструкцій `if`:

```python
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

У наведеному прикладі `is_citizen` є `True`, а `age >= 18` оцінюється як `True`. Оскільки обидва операнди оператора `and` є правдивими, умова `is_citizen and age >= 18` оцінюється як `True`, і виконується виклик `print` у блоці `if`.

Тепер розглянемо оператор `or`. Цей оператор повертає перший операнд, якщо він правдивий, інакше повертає другий операнд. Вираз `or` дає правдиве значення, якщо принаймні один операнд є правдивим. Оператор `or` також відомий як оператор короткого замикання. Ось приклад:

```python
age = 19
is_employed = False

print(age or is_employed) # 19
```

Наступний код виведе число 19, тому що перший операнд `age` є `True`.

Якщо потрібно перевірити, чи є один або кілька виразів `True`, ви можете використати оператор `or` в умові так:

```python
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

У цьому випадку `age < 18` є `False`, але `is_student` є `True`. Оскільки принаймні одна умова істинна, весь вираз `or` оцінюється як `True`, і виводиться повідомлення про знижку в блоці `if`.

Останній оператор, який ми розглянемо, — це оператор `not`, який приймає один операнд і інвертує його булеве значення. Він перетворює правдиві значення на `False` і хибні — на `True`. На відміну від попередніх операторів, які ми розглядали, `not` завжди повертає `True` або `False`.

Ось кілька прикладів:

```python
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

Зазвичай оператор `not` використовують в умовах, щоб перевірити, чи щось не є `True` або `False`, ось так:

```python
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```

Оскільки `is_admin` є `False`, то `not is_admin` означає не `False`, що є `True`. Тож буде виведено повідомлення `Access denied for non-administrators.`.

Тепер, коли ви розумієте правдиві та хибні значення, оператори `and`, `or` і `not`, а також принцип короткого замикання, ви можете писати більш гнучку та зрозумілу умовну логіку.
