# --description--

Una variable declarada dentro de una función solo es visible dentro de esa función. Esto se llama el alcance de la variable, y este alcance particular se llama alcance local. Las variables declaradas fuera de una función se llaman variables globales y son visibles en todo el programa. Por ejemplo, el siguiente código generará un error:

```js
function myFunction() {
  let x = 10;
}

console.log(x);
```

La variable `x` está declarada dentro de la función `myFunction` y no es visible fuera de la función. El código generará un `ReferenceError` porque `x` no está definido.

Si declaras una variable fuera de una función, se llama variable global y es visible en todo el programa. Por ejemplo, el siguiente código funcionará:

```js
let x = 10;

function myFunction() {
  console.log(x);
}

myFunction();
```

Si se declara una variable con el mismo nombre dentro de la función, entonces oculta a la externa.
