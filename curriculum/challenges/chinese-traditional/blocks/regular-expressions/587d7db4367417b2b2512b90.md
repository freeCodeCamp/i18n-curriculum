# --description--

使用像 `/coding/` 這樣的正規表達式，你可以在另一個字串中尋找樣式 `coding`。

這對搜尋單一字串非常強大，但僅限於一個樣式。你可以使用 `alternation` 或 `OR` 運算子：`|` 來搜尋多個樣式。

這個運算子會比對它之前或之後的樣式。例如，如果你想比對字串 `yes` 或 `no`，你想要的正規表達式是 `/yes|no/`。

你也可以搜尋不只兩個樣式。你可以透過為你的樣式添加更多樣式，並用更多 `OR` 運算子將它們分開來做到這點，例如 `/yes|no|maybe/`。

# --instructions--

完成正規表達式 `petRegex`，以匹配寵物 `dog`、`cat`、`bird` 或 `fish`。
