# --description--

`Cloud computing` 是一種技術，允許人們透過網際網路存取和使用電腦資源，例如儲存空間和處理能力。這表示資料和程式不再儲存在你的個人電腦或伺服器上，而是託管在透過雲端（網際網路）存取的虛擬伺服器網路上。只要有網際網路連接，就能從任何地方存取資料和應用程式。

`cloud computing` 服務的範例包括用於儲存的 Google Drive 以及用於各種運算服務的 Amazon Web Services。

比較級 `more convenient` 用來描述某物更容易使用或更適合特定需求，與其他選項相比使事情更簡單或節省時間。 例如，線上購物通常被視為比去實體店面 `more convenient`，因為你可以在家完成。
