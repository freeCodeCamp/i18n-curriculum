# --description--

Neste contexto, estar `out` significa estar disponível ou publicado. Por exemplo:

`The new book is out next week.` - Isso significa que o livro estará disponível então.

`Looking forward to` + um verbo que termina em `-ing` é uma forma comum de expressar entusiasmo sobre um evento futuro.Por exemplo:

- `I'm looking forward to meeting you.` - Você está animado para conhecer alguém.

- `She's looking forward to traveling next month.` - Ela está animada para viajar.
