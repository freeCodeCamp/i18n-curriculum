# --explanation--

`Crucial` は何かが非常に重要または不可欠であることを意味します。例えば：

`Security is a crucial aspect of the system.` – これはシステムのセキュリティが非常に重要であり、見落としてはいけないことを意味します。

`Extra` は通常の量を超えたものを指します。通常より多くの作業や努力が必要であることを示します。例えば：

`The extra work on this project took longer than expected.` – これは当初の計画よりも多くの作業が必要で、追加の努力が求められたことを意味します。
