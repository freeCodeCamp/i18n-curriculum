이 영상에서는 for 루프를 사용해 리스트나 range 같은 수열을 순회하는 방법을 배우세요.

# --questions--

## --text--

다음 중 문자열 `"Giraffe Academy"`의 각 글자를 순회하는 올바른 방법은 무엇인가요?

## --answers--

```python
for (letter) in "Giraffe Academy":
    print(letter)
```

---

```python
for key, letter in "Giraffe Academy":
    print(letter)
```

---

```python
for letter in "Giraffe Academy":
    print(letter)
```

---

```python
for letter, value in "Giraffe Academy":
    print(letter)
```
