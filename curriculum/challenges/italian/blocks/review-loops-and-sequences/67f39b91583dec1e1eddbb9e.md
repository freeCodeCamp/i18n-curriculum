## Liste in Python

- **Introduzione**: In Python, il tipo di dato lista è una sequenza ordinata di elementi che può essere composta da stringhe, numeri o anche altre liste. Le liste sono mutabili e indicizzate a base zero.

```python
cities = ['Los Angeles', 'London', 'Tokyo']
```

- **Consultare elementi in una lista**: Per consultare un elemento dalla lista `cities`, puoi fare riferimento al suo numero di indice nella sequenza:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # Los Angeles
```

- **Consultare elementi usando l'indicizzazione negativa**: Per consultare l'ultimo elemento di qualsiasi lista, puoi usare `-1` come numero di indice:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # Tokyo
```

- L'indicizzazione negativa serve a consultare gli elementi partendo dalla fine della lista invece che dall'inizio all'indice `0`.

- **Creare liste usando il costruttore `list()`**: Le liste possono anche essere create usando il costruttore `list()`. Il costruttore `list()` serve a convertire un iterabile in una lista:

```python
developer = 'Jessica'

print(list(developer)) 
# Result: ['J', 'e', 's', 's', 'i', 'c', 'a']
```

- **Trovare la lunghezza di una lista**: Puoi usare la funzione `len()` per ottenere la lunghezza di una lista:

```python
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

- **Mutabilità delle liste**: Le liste sono mutabili, cioè puoi aggiornare qualsiasi elemento nella lista purché fornisca un numero di indice valido. Per aggiornare una lista a un indice specifico, puoi assegnare un nuovo valore a quell'indice:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

- **Errore indice fuori intervallo**: Se fornisci un indice (positivo o negativo) che è fuori dai limiti della lista, riceverai un `IndexError`:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
"""
```

- **Rimuovere elementi da una lista**: Gli elementi possono essere rimossi da una lista usando la parola chiave `del`:

```python
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

- **Verificare se un elemento esiste in una lista**: La parola chiave `in` può essere usata per verificare se un elemento esiste in una lista:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **Liste annidate**: Le liste possono essere annidate dentro altre liste:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

- Per consultare la lista annidata, devi accedervi usando l'indice `2` dato che le liste sono indicizzate a base zero.

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

- Per accedere ulteriormente alla seconda lingua di quella lista annidata, devi usare l'indice `1`:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # Rust
```

- **Scomporre valori da una lista:** Scomporre valori da una lista è una tecnica usata per assegnare valori da una lista a nuove variabili. Ecco un esempio per scomporre la lista `developer` in nuove variabili chiamate `name`, `age` e `job` così:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer
```

- Se il numero di variabili sul lato sinistro dell'operatore di assegnazione non corrisponde al numero totale di elementi nella lista, riceverai un `ValueError`.

- **Raccogliere gli elementi rimanenti da una lista**: Per raccogliere gli elementi rimanenti da una lista, puoi usare l'operatore asterisco (`*`) così:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer
```

- **Slicing delle liste**: Lo slicing è il concetto di consultare una porzione di una lista usando l'operatore di slicing `:`. Per estrarre una lista che inizia all'indice `1` e termina prima dell'indice `3`, puoi usare questa sintassi:

```python
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie']
desserts[1:3] # ['Cookies', 'Ice Cream']
```

- **Intervalli di passo**: È anche possibile specificare un intervallo di passo che determina di quanto incrementare tra gli indici. Ecco un esempio se vuoi estrarre una lista solo di numeri pari usando lo slicing:

```python
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

## Metodi delle liste

- **append()**: Usato per aggiungere un elemento alla fine della lista. Ecco un esempio di utilizzo del metodo `append()` per aggiungere il numero `6` a questa lista `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **Aggiungere liste**: Il metodo `append()` può anche essere usato per aggiungere una lista alla fine di un'altra:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

- **extend()**: Usato per aggiungere più elementi alla fine di una lista. Ecco un esempio di aggiunta dei numeri `6`, `8` e `10` alla fine della lista `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

- **insert()**: Usato per inserire un elemento a un indice specifico nella lista. Ecco un esempio di utilizzo del metodo `insert()`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

- **remove()**: Usato per rimuovere un elemento dalla lista. Il metodo `remove()` rimuove solo la prima occorrenza di un elemento nella lista:

```py
numbers = [1, 2, 3, 4, 5, 5, 5]
numbers.remove(5)

print(numbers) # [1, 2, 3, 4, 5, 5]

```

- **pop()**: Usato per rimuovere un elemento specifico dalla lista e restituirlo:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

- Se non specifichi un elemento per il metodo `pop`, viene rimosso l'ultimo elemento.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

- **clear()**: Usato per rimuovere tutti gli elementi dalla lista:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

- **sort()**: Il metodo `sort()` serve per ordinare gli elementi in loco. Ecco un esempio di ordinamento in loco di una lista casuale di `numbers`:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

- **sorted()**: Una funzione integrata che restituisce una nuova lista ordinata invece di modificare la lista originale:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
print(numbers) # [19, 2, 35, 1, 67, 41]
```

- **reverse()**: Usato per invertire l'ordine degli elementi in una lista:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **index()**: Usato per trovare il primo indice in cui un elemento può essere trovato in una lista:

```python
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

- Se l'elemento non può essere trovato usando il metodo `index()`, il risultato sarà un `ValueError`.

## Tuple in Python

- **Definizione**: Una tupla è un tipo di dato Python usato per creare una sequenza ordinata di valori. Le tuple possono contenere un insieme misto di tipi di dati:

```py
developer = ('Alice', 34, 'Rust Developer')
```

- Le tuple sono immutabili, cioè gli elementi nella tupla non possono essere cambiati una volta create. Se provi ad aggiornare uno degli elementi nella tupla, otterrai un `TypeError`:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **Consultare elementi da una tupla**: Per consultare un elemento da una tupla, usa la notazione a parentesi e il numero di indice:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

- L'indicizzazione negativa può essere usata per consultare gli elementi partendo dalla fine della tupla:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- Se provi a fornire un numero di indice che supera o è uguale alla lunghezza della tupla, riceverai un `IndexError`:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- Una tupla può anche essere creata usando il costruttore `tuple()`. All'interno del costruttore puoi passare diversi iterabili come stringhe, liste e anche altre tuple.

```python
developer = 'Jessica'

print(tuple(developer)) 
# Result: ('J', 'e', 's', 's', 'i', 'c', 'a')
```

- **Verificare elementi in una tupla**: Per verificare se un elemento è in una tupla, puoi usare la parola chiave `in` così:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **Scomporre tuple**: Gli elementi possono essere scomposti da una tupla così:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

- Se hai bisogno di raccogliere gli elementi rimanenti da una tupla, puoi usare l'operatore asterisco (`*`) così:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer
```

- **Slicing delle tuple**: Lo slicing può essere usato per estrarre una porzione di una tupla. Per esempio, gli elementi `pie` e `cookies` possono essere estratti in una tupla separata:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

- **Rimuovere elementi dalle tuple**: Rimuovere un elemento da una tupla genera un `TypeError` perché le tuple sono immutabili:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

- **Quando usare una tupla invece di una lista?**: Se ti serve una collezione dinamica di elementi dove puoi aggiungere, rimuovere e aggiornare elementi, allora dovresti usare una lista. Se sai di lavorare con una collezione fissa e immutabile di dati, allora dovresti usare una tupla.

## Metodi comuni delle tuple

- **`count()`**: Usato per determinare quante volte un elemento appare in una tupla. Per esempio, puoi verificare quante volte la lingua `'Rust'` appare nella tupla:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

- Se l'elemento specificato nel metodo `count()` non è presente nella tupla, il valore di ritorno sarà `0`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

- Se non vengono passati argomenti al metodo `count()`, Python restituirà un `TypeError`.

- **index()**: Usato per trovare l'indice in cui un elemento specifico è presente nella tupla. Ecco un esempio di utilizzo del metodo `index()` per trovare l'indice della lingua `'Java'`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

- Se l'elemento specificato non può essere trovato, Python restituirà un `ValueError`.

- Puoi passare un indice di inizio opzionale al metodo `index()` per specificare da dove iniziare la ricerca dell'elemento nella tupla:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

- Puoi anche passare un indice di fine opzionale al metodo `index()` per specificare dove terminare la ricerca dell'elemento nella tupla:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

- **`sorted()`**: Usato per ordinare gli elementi in qualsiasi iterabile e restituire una nuova lista ordinata. Ecco un esempio di creazione di una nuova lista di numeri usando la funzione `sorted()`:

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

- **Modificare il comportamento dell'ordinamento**: Puoi personalizzare il comportamento di ordinamento per un iterabile usando gli argomenti opzionali `reverse` e `key`. Ecco un esempio di utilizzo dell'argomento `key` per ordinare gli elementi di una tupla in base alla lunghezza:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

- Puoi creare una nuova lista di valori in ordine inverso, usando l'argomento `reverse` così:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

## Loop in Python

- **Definizione**: I loop servono a ripetere un blocco di codice per un numero prestabilito di volte.

- **Loop `for`**: Usato per iterare su una sequenza (come una lista, tupla o stringa) ed eseguire un blocco di codice per ogni elemento in quella sequenza. Ecco un esempio di utilizzo di un loop `for` per iterare su una lista e stampare ogni lingua sulla console:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)

"""
Result 

Rust
Java
Python
C++
"""
```

- Ecco un esempio di utilizzo di un loop `for` per iterare sulla stringa `code` e stampare ogni carattere:

```py
for char in 'code':
    print(char)

"""
Result 

c
o
d
e
"""
```

- I loop `for` possono essere annidati. Ecco un esempio di utilizzo di un loop `for` annidato:

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)

"""
Result

Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
"""

```

- **Loop `while`**: Ripete un blocco di codice finché la condizione è `False`. Ecco un esempio di utilizzo di un loop `while` per un gioco di indovinelli:

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')

"""
Result

Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
"""

```

- **Istruzioni `break` e `continue`**: Usate nei loop per modificare l'esecuzione di un loop.

- L'istruzione `break` serve per uscire immediatamente dal loop quando una certa condizione è soddisfatta. Ecco un esempio di utilizzo dell'istruzione `break` per una lista di `developer_names`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

- L'istruzione `continue` serve per saltare l'iterazione corrente e passare a quella successiva del loop. Ecco un esempio di utilizzo dell'istruzione `continue` invece di un'istruzione `break`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

- Sia i loop `for` che `while` possono essere combinati con una clausola `else`, che viene eseguita solo quando il loop non è stato terminato da un `break`:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

## Range e il loro uso nei loop

- **La funzione `range()`**: Usata per generare una sequenza di numeri interi.

```py
range(start, stop, step)
```

- L'argomento obbligatorio `stop` è un numero intero (non incluso) che rappresenta il punto finale della sequenza di numeri generata. Ecco un esempio di utilizzo della funzione `range()`:

```py
for num in range(3):
    print(num)
```

- Se non viene specificato un argomento `start`, il valore predefinito sarà `0`. Per impostazione predefinita la sequenza di numeri interi incrementa di `1`. Puoi usare l'argomento opzionale `step` per cambiare il valore di incremento predefinito. Ecco un esempio di generazione di una sequenza di numeri interi pari da 2 fino a 11 escluso (cioè include il 10)

```py
for num in range(2, 11, 2):
    print(num)
```

- Se non fornisci argomenti alla funzione `range()`, otterrai un `TypeError`.

- La funzione `range()` accetta solo numeri interi come argomenti e non numeri decimali. Usare numeri decimali genera un `TypeError`:

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- Puoi usare un numero intero negativo per l'argomento `step` per generare una sequenza di numeri interi in ordine decrescente:

```py
for num in range(40, 0, -10):
    print(num)
```

- La funzione `range()` può anche essere usata per creare una lista di numeri interi usandola con il costruttore `list`. Il costruttore `list` serve a convertire un iterabile in una lista. Ecco un esempio di generazione di una lista di numeri interi pari tra 2 e 10 inclusi:

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

## Funzioni `enumerate()` e `zip()` in Python

- **`enumerate()`**: usata per iterare su una sequenza e tenere traccia dell'indice per ogni elemento in quella sequenza. La funzione `enumerate()` prende un iterabile come argomento e restituisce un oggetto `enumerate` che consiste nell'indice e nel valore di ogni elemento nell'iterabile.

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')

# Result
# Index 0 and language Spanish
# Index 1 and language English
# Index 2 and language Russian
# Index 3 and language Chinese

```

- La funzione `enumerate()` può anche essere usata al di fuori di un loop `for`:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

print(list(enumerate(languages)))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

- La funzione `enumerate()` accetta anche un argomento opzionale `start` che specifica il valore iniziale per il conteggio. Se questo argomento è omesso, il conteggio inizia da `0`.

-  **`zip()`** : Usata per iterare su più iterabili in parallelo. Ecco un esempio di utilizzo della funzione `zip()` per iterare su `developers` e `ids`:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')


"""
Result

Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
"""

```

## Comprensioni di lista in Python

- **Definizione**: La comprensione di lista ti permette di creare una nuova lista in una sola riga combinando il loop e la condizione direttamente dentro parentesi quadre. Questo rende il codice più corto e spesso più facile da leggere.

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

## Metodi per iterabili

- **`filter()`**: Usato per filtrare elementi da un iterabile basandosi su una condizione. Restituisce un iteratore che contiene solo gli elementi che soddisfano la condizione. Ecco un esempio di creazione di una nuova lista con solo parole più lunghe di quattro caratteri:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**: Usato per applicare una funzione a ogni elemento di un iterabile e restituire un nuovo iterabile con i risultati. Ecco un esempio di utilizzo della funzione `map()` per convertire una lista di temperature in gradi celsius in fahrenheit:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**: Usato per ottenere la somma da un iterabile come una lista o tupla. Ecco un esempio di utilizzo della funzione `sum()`:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- Puoi anche passare un argomento opzionale `start` che imposta il valore iniziale per la somma. Ecco un esempio aggiornato che usa l'argomento `start` come argomento posizionale:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- Puoi anche scegliere di usare l'argomento `start` come argomento con nome così:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## Funzioni lambda

- **Definizione**: Una funzione lambda in Python è un modo conciso per creare una funzione senza nome (una funzione anonima). 
- Le funzioni lambda sono spesso usate come argomento per un'altra funzione. Ecco un esempio di funzione lambda:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

- Le buone pratiche per usare le funzioni lambda includono non assegnarle a una variabile, mantenerle semplici e leggibili e usarle per funzioni brevi e usa e getta.
