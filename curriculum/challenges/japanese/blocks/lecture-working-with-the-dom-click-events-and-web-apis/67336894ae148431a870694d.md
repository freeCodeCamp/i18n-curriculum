# --interactive--

DOMとそのウェブ開発における重要性について学びましょう。DOMはDocument Object Modelの略です。HTMLドキュメントとやり取りするためのプログラミングインターフェースです。

DOMを使うと、ウェブページの要素を追加、変更、削除できます。さらに、要素がイベントを聞いて反応するようにして、ウェブサイトをインタラクティブにすることも可能です。

DOMでは、HTMLドキュメントはノードのツリー構造として表されます。各ノードはHTMLドキュメントのHTML要素を表しています。

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>DOM Example</title>
  </head>
  <body>
    <h1>What is the DOM?</h1>
    <h2>Let's learn about the DOM</h2>
  </body>
</html>
```

これは、例の基本的なDOM構造を表す図です。

```md
Document
========

HTML
-----

- Head
  - Title
- Body
  - H1
  - H2   
```

もちろん、これはウェブページのHTMLマークアップの構造によって、より詳細で複雑になる場合があります。

HTMLドキュメントはDOM階層のルートノードです。1つの子ノード、`html`要素を持っています。これはHTMLドキュメントのルート要素で、他のすべてのノードはここから派生しています。

`html`要素は2つの子ノード、`head`と`body`を持っています。

`head`要素はドキュメントのメタデータを含んでいます。ウェブページに関する重要な情報を提供します。

`body`要素はウェブページの可視コンテンツを含んでいます。

これらの要素にはJavaScriptでアクセスでき、コード内で操作したり動的に変更したりできます。これがDOMとウェブAPIの力です。

JavaScriptでこれらの要素にアクセスするには、`getElementById()`や`querySelector()`メソッドを使います。これらのメソッドはウェブAPIで、JavaScriptを使ってDOMと標準的にやり取りする方法を提供します。

`getElementById()`を使うと、指定した`id`を持つHTML要素を表すオブジェクトを取得できます。`id`はすべてのHTMLドキュメントで一意である必要があるため、このメソッドは1つの`Element`オブジェクトだけを返します。例を見てみましょう。

```js
const container = document.getElementById("container");
```

このJavaScriptコードの行は、`id`の値が`container`の要素を取得し、そのオブジェクトをJavaScriptの定数に代入しています。`id`は引用符で囲んで引数として渡す必要があります。このオブジェクトをコンソールにログ出力すると、出力に表示されます。

:::interactive_editor

```html
<div id="container">
  <h1>Hello, World!</h1>
  <p>Welcome to learning about the DOM.</p>
</div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
console.log(container); 
```

:::

`querySelector()`は`getElementById()`よりも広範囲です。`querySelector()`を使うと、引数として渡したCSSセレクターにマッチするHTMLドキュメント内の最初の要素を取得できます。

この例では、`section`セレクターにマッチする要素を取得します。

:::interactive_editor

```html
<section>
  <h2>Section Title</h2>
  <p>This is a section of the webpage.</p>
</section>
<script src="index.js"></script>  
```

```js
const sectionEl = document.querySelector("section");
console.log(sectionEl);
```

:::

`querySelector()`でクラス名で要素を選択したい場合は、クラス名の前にドット（`.`）を付ける必要があります。例を示します。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="highlight">
  <p>This content is highlighted.</p>
</div>
<script src="index.js"></script>
```

```css
body {
  background-color: lightgray;
}

.highlight {
  background-color: yellow;
}
```

```js
const highlightEl = document.querySelector(".highlight");
console.log(highlightEl);
```

:::

複数の要素にマッチさせる他のメソッド、例えば`getElementsByClassName()`や`querySelectorAll()`もあります。これらについては今後のレッスンで学びます。

ウェブAPIを使ってDOMを効率的に操作する方法を理解すれば、強力でインタラクティブなウェブアプリケーションを作成できます。
