# --interactive--

En JavaScript, un tipo de dato es el tipo de valor que almacenas, como un número o un fragmento de texto.

Una variable es un contenedor nombrado que almacena un valor de un tipo de dato específico, permitiéndote referenciarlo y manipularlo a lo largo de tu código. 

Quizás recuerdes en la clase de matemáticas trabajar con variables así:

```md
x = 2
y = 4

x + y
```

Pudiste crear variables como `x` y `y` y luego referenciarlas a lo largo de tu programa y hacer operaciones matemáticas como la suma. Es un concepto similar en programación. Puedes crear tus propios nombres de variables y asignarles valores. Estos valores serán de diferentes tipos de datos.

Los tipos de datos ayudan al programa a entender el tipo de datos con los que está trabajando, ya sea un número, texto, o algo más.

JavaScript tiene varios tipos de datos básicos que utilizarás en tus programas.  We'll explore each data type in greater detail in future lessons. Por ahora, aquí hay una breve introducción de los diferentes tipos de datos en JavaScript.

El primer tipo de dato que veremos es el tipo `Number`.

Un `Number` representa tanto enteros como valores de punto flotante. Ejemplos de números enteros incluyen `7`, `19` y `90`. 

**NOTA**: `console.log()` es una función que muestra información en la consola, que es una parte de tu navegador web usada para depurar código. Aprenderás más sobre `console.log()` en lecciones futuras. Además, los símbolos `//` se usan para agregar comentarios en tu código. Los comentarios son notas para ti o para otros programadores que se ignoran cuando el código se ejecuta.

Activa el editor interactivo y prueba cambiar algunos de los enteros para ver cómo se actualiza en la consola.

:::interactive_editor

```js
// Examples of integers
console.log(3);
console.log(5);
console.log(-67);
```

:::

Un número de punto flotante es un número con un punto decimal. Ejemplos de números de punto flotante incluyen `3.14` y `5.2`. 

:::interactive_editor

```js
// Examples of floating point numbers
console.log(3.14);
console.log(7.2);
console.log(-14.5);
```

:::

El siguiente tipo de dato es un `String`.

Un `String` es una secuencia de caracteres, o texto, encerrado entre comillas. Aquí tienes un ejemplo de `String` usando comillas dobles: 

:::interactive_editor

```js
console.log("I love to code!");
```

:::

Aquí tienes un ejemplo usando comillas simples:

:::interactive_editor

```js
console.log('I love to code!');
```

:::

A menudo usarás cadenas para representar nombres, etiquetas, mensajes de alerta, etc.

Otro tipo de dato utilizado en JavaScript es el tipo `Boolean`.

Un `Boolean` representa uno de dos valores: `true` o `false`. Por ejemplo, un programa podría verificar si un usuario ha iniciado sesión (`true`) o no (`false`) y cambiar la página en función de eso. Si el usuario ha iniciado sesión, probablemente quieras mostrarle la página del panel de control. De lo contrario, querrás mostrarle la página de inicio de sesión.

Los siguientes dos tipos de datos en JavaScript son `undefined` y `null`.

`undefined` significa que una variable ha sido declarada pero aún no se le ha asignado un valor. Aprenderás más sobre esto en la próxima lección.

`null` significa que la variable ha sido intencionalmente establecida en "nada" y no contiene ningún valor. Exploraremos más sobre cómo funciona esto en lecciones futuras.

Los últimos tres tipos de datos son más complejos por naturaleza. Estos son objetos, símbolos y `BigInt`. 

Un `Object` es una colección de pares clave-valor. 

```js
{
  name: "Alice",
  age: 30
};
```

Los objetos son excelentes para agrupar información relacionada. Aprenderás más sobre cómo trabajar con objetos en un módulo futuro.

Un `Symbol` es un tipo especial de valor en JavaScript que siempre es único y no se puede cambiar. A menudo se utiliza para crear etiquetas o identificadores únicos para propiedades:

```js
Symbol('mySymbol');
```

`BigInt` se utiliza para números muy grandes que exceden el límite del tipo `Number`:

```js
1234567890123456789012345678901234567890n;
```

En este ejemplo, creamos un `BigInt` agregando `n` al final de un número muy grande.

`Symbol` y `BigInt` son dos tipos que se usan con menos frecuencia, pero que aún es importante conocer.

Entender estos tipos de datos te ayuda a manejar y trabajar con varios tipos de datos en tus programas, ya que cada tipo tiene sus propias características y comportamientos.
