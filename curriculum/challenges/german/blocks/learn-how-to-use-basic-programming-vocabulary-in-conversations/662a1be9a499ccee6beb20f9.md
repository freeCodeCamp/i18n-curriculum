# --description--

Zählbare Nomen sind Dinge, die Sie zählen können. Es können eins, zwei, drei oder mehr sein. Sie können Zahlen und Quantifizierer wie `many`, `a few` oder `several` verwenden, um zählbare Nomen zu beschreiben. Zum Beispiel Apfel: Sie können `one apple`, `two apples` oder `many apples` haben.

Unzählbare Nomen sind Dinge, die Sie nicht einzeln zählen können. Sie ändern sich normalerweise nicht, um Mehrzahl anzuzeigen. Sie verwenden keine Zahlen direkt, sondern Quantifizierer wie `much`, `a little`, `some`, um unzählbare Nomen zu beschreiben. Zum Beispiel Wasser: Sie können nicht sagen `one water` oder `two waters`. Sie können sagen `much water`, `some water`.
