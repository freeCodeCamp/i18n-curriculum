# --description--

令 d(`n`) 定義為 `n` 的適當因數之和（小於 `n` 且能整除 `n` 的數字）。

如果 d(`a`) = `b` 且 d(`b`) = `a`，其中 `a` ≠ `b`，則 `a` 和 `b` 是一對友好對組，且 `a` 和 `b` 各自被稱為友好數。

例如，220 的真因數是 1、2、4、5、10、11、20、22、44、55 和 110；因此 d(220) = 284。284 的真因數是 1、2、4、71 和 142；所以 d(284) = 220。

計算所有小於 `n` 的親和數之和。

```js
const sumAmicableNum = (n) => {
  const fsum = (n) => {
    let sum = 1;
    for (let i = 2; i <= Math.floor(Math.sqrt(n)); i++)
      if (Math.floor(n % i) === 0)
        sum += i + Math.floor(n / i);
    return sum;
  };
  let d = [];
  let amicableSum = 0;
  for (let i=2; i<n; i++) d[i] = fsum(i);
  for (let i=2; i<n; i++) {
    let dsum = d[i];
    if (d[dsum]===i && i!==dsum) amicableSum += i+dsum;
  }
  return amicableSum/2;
};
```
