# --explanation--

動詞 `to be` を `Simple Past` で使い、`Past Participle` 形の動詞（`were well received` と `was mentioned`）と組み合わせた構文は、どちらも**受動態**です。これらの構文では、行為者ではなく、行為や結果にフォーカスしています。別の例：

`The update was released last week.` – これは誰が行ったかではなく、イベント（リリース）に焦点を当てています。
