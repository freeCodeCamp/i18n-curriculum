# --description--

In dieser Aufgabe erkunden Sie die metaphorische Verwendung von Begriffen, um Programmierkonzepte zu verstehen. Metaphern sind ein kraftvolles Mittel, um komplexe Ideen zu visualisieren und zu erklären, indem man sie mit alltäglichen Gegenständen oder Handlungen vergleicht.

`Container` wird metaphorisch verwendet, um Variablen als Dinge zu beschreiben, die verschiedene Daten aufnehmen und speichern können, ähnlich wie ein `container` verschiedene Gegenstände aufnehmen kann.

Die Aktionen `fill and empty` beschreiben, wie Sie Daten zu Variablen hinzufügen oder daraus entfernen können, ähnlich wie Sie eine Wasserflasche `fill` oder das Wasser daraus ausgießen.

`Locked boxes` beschreibt Konstanten als sichere, unveränderliche `containers`, wobei betont wird, dass sie nach der Festlegung nicht mehr geändert werden können.

Der Ausdruck `keep the same contents` hebt hervor, dass die Daten in Konstanten sich nicht ändern, wie eine verschlossene Box, die immer denselben Inhalt enthält.
