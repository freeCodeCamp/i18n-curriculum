# --description--

لإكمال دالة `initializeThroughput` الخاصة بك:

- كرر التكرار خلال كل بوابة في مصفوفة `gates` المُدخَلة. لكل بوابة، أضف خاصية إلى `summary` حيث:
  - المفتاح هو `id` الخاص بالبوابة.
  - القيمة هي `0`.
- بعد التكرار، أرجع كائن `summary`.

يجب ألا تُغيّر دالة `initializeThroughput` مصفوفة `gates` المصدرية المُدخلة.

# --hints--

استدعاء `initializeThroughput()` مع `[]` يجب أن يُرجع `{}` دون تغيير المصفوفة المصدرية.

```js
const input = [];
const original = JSON.parse(JSON.stringify(input));
const result = initializeThroughput(input);

assert.deepEqual(
  result,
  {},
  "Calling `initializeThroughput([])` should return an empty object."
);

assert.deepEqual(
  input,
  original,
  "`initializeThroughput` should not mutate the source array when given an empty array."
);
```

استدعاء `initializeThroughput()` مع `[{ id: "North", capacity: 5, queue: [1, 2, 3] }]` يجب أن يُرجع `{ North: 0 }` دون تغيير المصفوفة المصدرية.

```js
const input = [{ id: "North", capacity: 5, queue: [1, 2, 3] }];
const original = JSON.parse(JSON.stringify(input));
const result = initializeThroughput(input);

assert.deepEqual(
  result,
  { North: 0 },
  "Calling `initializeThroughput([{ id: 'North', ... }])` should return `{ North: 0 }`."
);

assert.deepEqual(
  input,
  original,
  "`initializeThroughput` should not mutate the source array when given one gate."
);
```

استدعاء `initializeThroughput()` مع `[{ id: "North", capacity: 5, queue: [] }, { id: "East", capacity: 3, queue: [] }, { id: "South", capacity: 4, queue: [] }]` يجب أن يُرجع `{ North: 0, East: 0, South: 0}` دون تغيير المصفوفة المصدرية.

```js
const input = [

  { id: "South", capacity: 4, queue: [] }

  result,
  { North: 0, East: 0, South: 0 },

);

assert.deepEqual(
  input,
  original,
  "`initializeThroughput` should not mutate the source array when given multiple gates."
);
```
