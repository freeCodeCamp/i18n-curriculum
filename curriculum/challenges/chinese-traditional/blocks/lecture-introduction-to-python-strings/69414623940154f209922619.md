# --description--

在處理字串時，將不同的文字片段組合在一起是你經常會遇到的常見操作。

在 Python 中，你可以使用加號 (`+`) 運算子將多個字串串接在一起。這個處理程序稱為**字串串接**。以下是如何使用加號運算子串接兩個字串：

```python
my_str_1 = 'Hello'
my_str_2 = "World"

str_plus_str = my_str_1 + ' ' + my_str_2
print(str_plus_str) # Hello World
```

但請注意，這只適用於字串。如果你嘗試將字串與數字串接，將會得到 `TypeError`：

```python
name = 'John Doe'
age = 26

name_and_age = name + age
print(name_and_age) # TypeError: can only concatenate str (not "int") to str
```

這是因為 Python 不會在你串接時自動將其他資料型別（例如整數）轉換成字串。Python 需要所有元素都是字串才能進行串接。為了解決這個問題，你可以使用內建的 `str()` 函式，將數字轉換成字串，該函式會傳回給定物件的字串表述，且不會修改原始物件：

```python
name = 'John Doe'
age = 26

name_and_age = name + str(age)
print(name_and_age) # John Doe26
```

你也可以使用增強指派運算子來進行串接。這是由加號和相等號（`+=`）所表述，並在一步驟中同時執行串接和指派。以下是範例：

```py
name = 'John Doe'
age = 26

name_and_age = name  # Start with the name
name_and_age += str(age)  # Append the age as string

print(name_and_age)  # John Doe26
```

將變數和表達式插入字串的過程稱為**字串插值**。Python 有一種稱為**f-strings**（格式化字串字面值縮寫）的字串類別，允許你使用簡潔且易讀的語法來控制插值。

F-string 以引號前的 `f`（小寫或大寫皆可）開始，並允許你在由大括號（`{}`）顯示的取代欄中嵌入變數或表達式。以下是一個範例：

```python
name = 'John Doe'
age = 26
name_and_age = f'My name is {name} and I am {age} years old'
print(name_and_age) # My name is John Doe and I am 26 years old

num1 = 5
num2 = 10
print(f'The sum of {num1} and {num2} is {num1 + num2}') # The sum of 5 and 10 is 15
```

請注意，你不需要使用 `str()` 函式將非字串型別轉換。在上述範例中，`age`、`num1` 和 `num2` 變數的值會在插值處理程序中自動轉換成字串。
