# --description--

Komplexere bedingte Anweisungen können mehrere Bedingungen enthalten. Deshalb werden logische Operatoren verwendet, um mehrere Bedingungen zu kombinieren. Die logischen Operatoren sind `&&`, `||` und `!`, die jeweils `and`, `or` und `not` darstellen.

Der logische Operator `||` wird verwendet, um zwei Boolean-Bedingungen zu kombinieren. Er gibt `true` zurück, wenn mindestens eine der Bedingungen `true` ist. Andernfalls gibt er `false` zurück:

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a > b || a > c) {
  console.log("At least one of the conditions is true");
} else {
  console.log("Both of the conditions are false");
}
```

Der logische Operator `&&` wird verwendet, um zwei Boolean-Bedingungen zu kombinieren. Er gibt nur dann `true` zurück, wenn beide Bedingungen `true` sind. Andernfalls gibt er `false` zurück:

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a < b && a < c) {
  console.log("Both of the conditions are true");
} else {
  console.log("At least one of the conditions is false");
}
```

Der logische Operator `!` wird verwendet, um eine Boolean-Bedingung zu negieren. Er gibt `true` zurück, wenn die Bedingung `false` ist. Andernfalls gibt er `false` zurück:

```javascript
let a = 5;
let b = 10;

if (!(a > b)) {
  console.log("The condition is false");
} else {
  console.log("The condition is true");
}
```

Im obigen Beispiel wird das zusätzliche Paar runder Klammern verwendet, um den Code lesbarer zu machen. Es ist nicht notwendig, sie zu verwenden.

# --questions--

## --text--

Sie haben die Aufgabe, eine `if`-Anweisung zu schreiben, die in einer Web-App folgende Bedingungen überprüft, um `Welcome!` für den Benutzer anzuzeigen:

1. Der Benutzer muss entweder ein Premium-Konto haben (`isPremium`) oder seit mehr als einem Jahr Mitglied sein (`membershipDuration` > 12 Monate).

1. Der Benutzer darf derzeit nicht vom Dienst gesperrt sein (`!isBlocked`).

Welche if-Anweisung überprüft diese Bedingungen korrekt?

## --answers--

```javascript
if (isPremium && membershipDuration > 12 && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (isPremium || (membershipDuration > 12 && !isBlocked)) {
  console.log("Welcome!");
}
```

---

```javascript
if ((isPremium || membershipDuration > 12) && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (!isPremium || membershipDuration <= 12 || isBlocked) {
  console.log("Welcome!");
}
```
