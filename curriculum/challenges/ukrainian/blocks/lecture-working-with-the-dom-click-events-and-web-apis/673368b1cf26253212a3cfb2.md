# --description--

Давайте дізнаємося про метод `querySelectorAll()`. Ви можете використовувати цей метод, щоб отримати список усіх елементів DOM, які відповідають певному CSS-селектору.

Ось як можна викликати метод `querySelectorAll()`:

```js
document.querySelectorAll(selectors);
```

Викликайте його на об’єкті `document` і передайте рядок із CSS-селекторами як аргумент. Аргумент має бути дійсним рядком CSS-селектора. Інакше буде викинуто виключення `SyntaxError`.

`querySelectorAll()` повертає об’єкт `NodeList`, колекцію вузлів, які відповідають заданому CSS-селектору.

Список міститиме по одному об’єкту `Element` для кожного елемента, що відповідає селекторам. Якщо відповідностей не знайдено, список буде порожнім.

Елементи будуть у тому порядку, в якому вони з’являються в HTML-документі. Наприклад, ви можете знайти всі елементи певного типу:

```js
document.querySelectorAll("div");
```

Всі елементи з певним класом:

```js
document.querySelectorAll(".rounded");
```

Всі елементи з певним ідентифікатором:

```js
document.querySelectorAll("#logo");
```

Або всі елементи з певним атрибутом, наприклад всі посилання, які ведуть користувачів на конкретну URL-адресу:

```js
document.querySelectorAll("a[href='https://www.freecodecamp.org/']");
```

Ви також можете використовувати складніші CSS-селектори, як цей, який знаходить усі елементи списку всередині невпорядкованого списку з класом `ingredients`:

```js
document.querySelectorAll("ul.ingredients li");
```

Оскільки метод повертає колекцію вузлів, ви можете призначити цю колекцію змінній і використовувати її у своїх програмах для роботи з окремими елементами, як показано в цьому прикладі з загальним синтаксисом:

```js
const matches = document.querySelectorAll(selectors); 
```

Наприклад, якщо у вас є цей невпорядкований список з трьома елементами списку:

```html
<ul class="ingredients">
  <li>Flour</li>
  <li>Cheese</li>
  <li>Water</li>
</ul>
```

Ви можете викликати метод `querySelectorAll()`, щоб знайти всі елементи списку всередині невпорядкованого списку з класом `ingredients` і призначити повернене значення константі `matches`:

```js
const matches = document.querySelectorAll("ul.ingredients li");
```

Це поверне колекцію з трьох вузлів. Кожен вузол представляє елемент списку:

```js
// NodeList(3)
{
  0: `<li>Flour</li>`,
  1: `<li>Cheese</li>`,
  2: `<li>Water</li>`,
  length: 3,
}
```

Ви можете працювати з цією колекцією так само, як і з будь-яким іншим масивом JavaScript.

Наприклад, ви можете вивести її в консоль за допомогою `console.log()`:

```js
console.log(matches);
```

Також можна перевірити довжину колекції за допомогою властивості `length`:

```js
console.log(matches.length);
```

Вихідні дані будуть цілим числом, що представляє кількість вузлів у колекції:

```md
3
```

Ви також можете отримати доступ до окремих елементів колекції за їхніми відповідними індексами. Перший елемент буде за індексом `0`:

```js
console.log(matches[0]);
console.log(matches[1]);
console.log(matches[2]);
```

Ось вихідні дані:

```html
<li>Flour</li>
<li>Cheese</li>
<li>Water</li>
```

Елементи виводяться по одному в консолі.

Ви також можете ітерувати над елементами за допомогою циклу `for`:

```js
for (let i = 0; i < matches.length; i++) {
  console.log(matches[i]);    
}
```

У цьому випадку вихідні дані будуть точно такими ж. Але це дуже зручно для роботи з усіма елементами колекції.

Метод `querySelectorAll()` у JavaScript — це потужний інструмент для вибору кількох елементів на основі їх CSS-селекторів. За допомогою цього методу ви можете вибирати конкретні елементи на своїх вебсторінках і маніпулювати ними за потреби.
