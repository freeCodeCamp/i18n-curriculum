# --description--

Neste projeto, você vai criar uma Binary Search Tree (BST).Uma BST é uma estrutura de dados na qual cada nó tem no máximo dois filhos, com o filho esquerdo contendo valores menores que o nó pai e o filho direito contendo valores maiores que o nó pai, permitindo operações eficientes de busca e ordenação.
