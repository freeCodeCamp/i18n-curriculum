# --interactive--

الفئات المزيفة هي كلمات مفتاحية خاصة في CSS تتيح لك اختيار عنصر بناءً على حالته أو موقعه المحدد. قد تشمل حالة العنصر أو موقعه ما يلي:

- عندما يكون نشطًا.
- عندما يتم التمرير فوقه بواسطة الماوس.
- عندما يكون الطفل الأول لعنصر أب.
- عندما يكون الطفل الأخير لعنصر أب.
- عندما تم زيارة رابط.
- عندما يكون معطلاً.

لاستخدام فئة مزيفة، تضيفها إلى المحدد باستخدام نقطتين رأسيتين (`:`) متبوعة باسم الفئة المزيفة:

```css
selector:pseudo-class {
  /* CSS properties */
}
```

لنرَ كيف يمكنك استخدام فئة مزيفة لتمثيل كل حالة وموقع ذكرناها سابقًا.

تتيح لك الفئة المزيفة `:active` اختيار الحالة النشطة لعنصر، مثل النقر على زر:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example Button</button>
```

```css
button:active {
  background: greenyellow;
}
```

:::

تعرف الفئة المزيفة `:hover` حالة التمرير فوق عنصر. مثال على ذلك التمرير فوق زر أو رابط:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="#">Hover over me!</a>  
```

```css
a:hover {
  text-decoration: none;
  color: white;
  background: crimson;
}
```

:::

تختار الفئة المزيفة `:first-child` عنصرًا يكون الطفل الأول لعنصر الأب. إليك مثالًا على استهداف أول فقرة داخل حاوية `div`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:first-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

سيحصل أول عنصر فقرة في ذلك الـ `div` على لون خلفية `lightcoral` و`padding` بمقدار `0.4rem` على جميع الجوانب الأربعة.

تستهدف الفئة المزيفة `:last-child` آخر عنصر يكون طفلًا لعنصر الأب. إليك مثالًا على استهداف آخر فقرة داخل حاوية `div`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:last-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

تتيح لك الفئة المزيفة `:visited` تنسيق رابط زاره المستخدم بالفعل:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="https://www.example.com" target="_blank">Visit Example.com</a>
```

```css
a:visited {
  color: purple;
}
```

:::

تتيح لك الفئة المزيفة `:disabled` تنسيق عنصر تفاعلي في وضع معطل:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button disabled>Disabled Button</button>
```

```css
button:disabled {
  background-color: lightgray;  
}
```

:::

كما ترى، تتيح لك الفئات المزيفة تنسيق العناصر بناءً على تفاعلات المستخدم، مثل التمرير فوقها أو زيارة رابط. هذا يجعل موقعك أكثر تفاعلية.

بجانب الفئات المزيفة التي ذكرناها، هناك أخرى مثل:

- `:focus`
- `:first-of-type`
- `:last-of-type`
- `:nth-of-type`
- `:modal`
- `:enabled`
- `:checked`
- `:required`، والمزيد.
