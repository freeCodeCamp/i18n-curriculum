# --description--

前のレッスンでは、コードのブロックを決まった回数だけ繰り返すために使う`for`ループの使い方を学びました。ここに、`for`ループを使って`languages`リストの各言語をコンソールに出力する例があります。

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for language in languages:
    print(language)
```

では、各要素のインデックスを追跡したい場合はどうでしょう？一つの方法は、`index`変数を作成し、ループの各反復で`1`ずつ増やすことです。例えば次のようにします。

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

index = 0

for language in languages:
    print(f'Index {index} and language {language}')
    index += 1
```

これでも動作しますが、もっと簡単な方法が`enumerate()`関数を使うことです。`enumerate()`関数はイテラブルのインデックスを追跡し、enumerateオブジェクトを返します。

`languages`リストを`enumerate()`関数に渡し、その戻り値を`list()`関数でリストに変換すると、次のようになります。

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

list(enumerate(languages))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

enumerateオブジェクト（現在はリスト）の各エントリーは、カウントと、その後に`enumerate()`関数に渡されたイテラブルの値を含むタプルです。

では、先ほどの例を`enumerate()`関数を使うようにリファクタリングしましょう。

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')
```

enumerateオブジェクトの各タプルのカウントと値を、それぞれ`index`と`language`という名前の変数にアンパックしています。最後に、これらの変数を使ったf文字列をループの各反復でコンソールに出力しています。

```md
Index 0 and language Spanish
Index 1 and language English
Index 2 and language Russian
Index 3 and language Chinese
```

これにより、手動で`index`変数を作成して更新する必要がなくなります。

`enumerate()`関数は、カウントの開始値を指定するオプションの`start`引数も受け付けます。この引数を省略すると、カウントは`0`から始まります。オプションの`start`引数を使った例は次の通りです。

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages, 1):
    print(f'Index {index} and language {language}')
```

コンソールでの結果は次のようになります。

```md
Index 1 and language Spanish
Index 2 and language English
Index 3 and language Russian
Index 4 and language Chinese
```

これまでは1つのリストだけを繰り返してきました。しかし、複数のイテラブルを並行して繰り返す必要がある場合はどうでしょう？その場合は`zip()`関数を使います。これはリストを要素のペアにまとめ、タプルのイテレータを返します。

`developers`と`ids`のリストを`zip()`関数に渡し、その戻り値を`list()`関数でリストに変換すると、次のようになります。

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

list(zip(developers, ids))
# [('Naomi', 1), ('Dario', 2), ('Jessica', 3), ('Tom', 4)]
```

そして、`zip()`関数を`for`ループと組み合わせて`developers`と`ids`を繰り返す例は次の通りです。

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')
```

この例では、`zip()`が2つのリストを要素のペアにまとめ、タプルのイテレータを返します。`for`ループは各タプルを`name`と`id`にアンパックします。最後に、各print文で`name`と`id`リストのそれぞれの`ids`と`developers`を出力しています。コンソールでの結果は次のようになります。

```md
Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
```

`enumerate()`と`zip()`関数は非常に強力で、ループと組み合わせることでコードをずっと簡潔にできます。
