# --description--

將數字增加或減少 1 是最常見的數值操作行為之一。

所以，為此有特殊的運算子：

- 遞增運算子 `++` 將變數增加 1：

```js
let counter = 2;

// works the same as counter = counter + 1, but is shorter
counter++;      

console.log(counter); // 3
```

- 遞減 `--` 將變數減少 1：

```js
let counter = 2;

// works the same as counter = counter - 1, but is shorter
counter--;

console.log(counter); // 1
```

運算子 `++` 和 `--` 可以放在變數的前面或後面。

- 當運算子位於變數之後時，稱為「後置式」：`counter++`。

- 「前置式」是指運算子位於變數之前：`++counter`。

這兩個述語都做相同的事情：將 `counter` 增加 `1`。

有什麼差別嗎？有，但你只有在使用 `++/--` 的傳回值時才能看到。

讓我們釐清。正如你所知，所有運算子都會傳回一個值。遞增／遞減也不例外。前置式會傳回新值，而後置式會傳回遞增／遞減前的舊值。

要查看差異，這裡有一個範例：

```js
let counter = 1;
let a = ++counter; // (*)

console.log(a); // 2
```

總結：

- 如果遞增/遞減的結果未被使用，使用哪種表單沒有差別：

```js
let counter = 0;
counter++;
++counter;

console.log( counter ); // 2, the lines above did the same
```

- 如果你想增加一個值並立即使用該運算子的結果，你需要使用前置式。

```js
let counter = 0;
console.log( ++counter ); // 1
```

- 如果你想遞增一個值但使用它之前的值，你需要後置式：

```js 
let counter = 0;
console.log( counter++ ); // 0
```

# --questions--

## --text--

下面 JavaScript 程式碼中兩個 `console.log` 述語的輸出是什麼？

```js
let counter = 1;
console.log(2 * ++counter); // Statement A
```

```js
let counter = 1;
console.log(2 * counter++); // Statement B
```

## --answers--

沒有差別；兩個 `console.log` 述語都會顯示相同的結果。

---

述語 A 會記錄 `4`，因為 `++counter` 在乘法之前遞增值。述語 B 會記錄 `2`，因為 `counter++` 在乘法之後遞增值。

---

述語 A 會記錄 `2`，因為 `++counter` 是語法錯誤。述語 B 會記錄 `4`，因為 `counter++` 是遞增計數器的正確方式。

---

兩個述語都會記錄 `3`，因為計數器在兩種情況下都會在 `console.log` 之前遞增。
