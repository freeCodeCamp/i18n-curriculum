## CSS 上限溢位(相對於 underflow) 屬性

- **定義**：上限溢位指的是元素控制代碼超出或「溢出」包含元素大小的內容的方式。上限溢位是二維的。
- **`overflow-x`**：x 軸決定水平上限溢位(相對於 underflow)。
- **`overflow-y`**：y 軸決定垂直上限溢位(相對於 underflow)。
- **`overflow`**：`overflow-x` 和 `overflow-y` 的簡寫屬性。如果只給一個值，兩個上限溢位(相對於 underflow) 都會使用該值。如果給兩個值，`overflow-x` 會使用第一個，`overflow-y` 會使用第二個。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="overflow-box">
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
</div>
```

```css
.overflow-box {
  width: 140px;
  height: 70px;
  border: 2px solid black;
  overflow-x: auto;
  overflow-y: auto;
  white-space:nowrap;
}
```

## CSS 轉換屬性

- **定義**：此屬性可讓你對元素套用各種轉換，例如旋轉、縮放、傾斜或在 2D 或 3D 空間中平移（移動）它們。
- **`translate()` 函式（程式）**：此函式用於將元素從其目前位置移動。
- **`scale()` 函式（程式）**：此函式允許你改變元素的大小。
- **`rotate()` 函式（程式）**：此函式（程式）允許你旋轉一個元素。
- **`skew()` 函式（程式）**：此函式（程式）允許你傾斜一個元素。
- **轉換與存取性**：如果你使用 transform 來隱藏或顯示內容，請確保內容仍可被螢幕閱讀器和鍵盤導覽存取。隱藏的內容應該是真正隱藏，例如使用 `display: none` 或 `visibility: hidden`，而不是僅僅視覺上移出螢幕。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="transform-box translate">Translate</div>
  <div class="transform-box scale">Scale</div>
  <div class="transform-box rotate">Rotate</div>
  <div class="transform-box skew">Skew</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: column;
  align-items: center;
  gap: 16px;
  flex-wrap: wrap;
}

.transform-box {
  width: 100px;
  height: 100px;
  background: coral;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

/* translate() */
.translate {
  transform: translateX(20px);
}

/* scale() */
.scale {
  transform: scale(1.2);
}

/* rotate() */
.rotate {
  transform: rotate(15deg);
}

/* skew() */
.skew {
  transform: skew(10deg, 5deg);
}
```

## 盒模型

- **定義**：在 CSS 盒模型中，每個元素都被一個盒子包圍。這個盒子由四個組件組成：內容區域、`padding`、`border`、`margin`。
- **內容區域**：內容區域是盒子最內層的部分。它是包含元素實際內容（例如文字或影像）的空間。
- **`padding`**：`padding` 是緊接內容區域之後的區域。它是內容區域與元素邊框之間的空間。
- **`border`**：`border` 是 CSS 盒模型中元素的外緣或輪廓。它是元素的視覺邊界。
- **`margin`**：`margin` 是元素邊框外的空間。它決定元素與周圍其他元素之間的距離。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="box-model">Content</div>
```

```css
.box-model {
  width: 120px;
  background: lightblue;
  padding: 16px;
  border: 4px solid teal;
  margin: 20px;
  text-align: center;
}
```

## 外邊距合併

- **定義**：當相鄰元素的垂直外邊距重疊時，會發生此行為，導致單一外邊距等於兩者中較大的那一個。此行為僅適用於垂直外邊距（上方和下方），不適用於水平外邊距（左方和右方）。

## `content-box` 和 `border-box` 屬性值

- **`box-sizing` 屬性**：此屬性用於決定 HTML 元素的最終 `width` 和 `height` 如何計算。
- **`content-box` 值**：在 `content-box` 模型中，你為元素設定的 `width` 和 `height` 決定內容區域的尺寸，但不包含 `padding`、`border` 或 `margin`。
- **`border-box` 值**：使用 `border-box` 時，元素的 `width` 和 `height` 包含內容區域、`padding` 和 `border`，但不包含 `margin`。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="sizing-box content-box">Content-box</div>
<div class="sizing-box border-box">Border-box</div>
```

```css
.sizing-box {
  width: 150px;
  padding: 10px;
  border: 4px solid teal;
  margin-bottom: 10px;
  background: peachpuff;
  text-align: center;
}

.content-box {
  box-sizing: content-box;  
}

.border-box {
  box-sizing: border-box;
  background: lightgreen;
}
```

## CSS 重置

- **定義**：CSS 重置是一種樣式表，用來移除網頁瀏覽器套用於 HTML 元素的全部或部分預設格式。第三方的 CSS 重置選項包括 `sanitize.css` 和 `normalize.css`。

## CSS 濾鏡屬性

- **定義**：此屬性可用於創建各種效果，例如模糊、色彩偏移和對比度調整。
- **`blur()` 函式（程式）**：此函式（程式）對元素套用高斯模糊。數值以像素為單位，表述模糊的半徑。
- **`brightness()` 函式（程式）**：此函式調整元素的亮度。值為 0% 時，元素會變成完全黑色，而超過 100% 的值會增加亮度。
- **`contrast()` 函式（程式）**：此函式（程式）會調整元素的對比度。值為 0% 時，元素會變成完全灰色，100% 時，元素會呈現正常狀態，超過 100% 的值會增加對比度。
- **`grayscale()` 函式（程式）**：此函式（程式）將元素轉換為灰階。數值以百分比定義，100％ 為完全灰階，0％ 則保持影像不變。
- **`sepia()` 函式（程式）**：此函式（程式）會將元素套用棕褐色調。與灰階類似，它使用百分比數值。
- **`hue-rotate()` 函式（程式）**：此函式（程式）對元素套用色相旋轉。該值以度數定義，並表述繞色彩圓的旋轉。

:::interactive_editor

```html
  <img class="filter contrast" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Contrast" />
  <img class="filter grayscale" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Grayscale" />
  <img class="filter sepia" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Sepia" />
  <img class="filter hue" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Hue Rotate" />

```

```css
  filter: sepia(100%);

}
```
