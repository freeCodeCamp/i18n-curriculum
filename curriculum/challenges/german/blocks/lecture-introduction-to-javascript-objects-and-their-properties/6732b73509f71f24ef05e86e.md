# --interactive--

Beim Arbeiten mit JavaScript stoßen Sie häufig auf komplexe Datenstrukturen, die verschachtelte Objekte und Arrays innerhalb von Objekten enthalten. Diese Strukturen können reichhaltige, hierarchische Daten darstellen, erfordern aber auch ein klares Verständnis dafür, wie Sie auf die darin enthaltenen Daten zugreifen und sie bearbeiten. Lassen Sie uns erkunden, wie Sie diese verschachtelten Strukturen effektiv navigieren.

Das Zugreifen auf Eigenschaften von verschachtelten Objekten erfolgt mit der Punktnotation oder der Klammernotation, ähnlich wie beim Zugreifen auf Eigenschaften einfacher Objekte. Allerdings müssen Sie diese Zugriffsmethoden verketten, um in die verschachtelte Struktur hineinzudringen.

Zum Beispiel betrachten wir ein verschachteltes Objekt, das eine Person mit Kontaktinformationen darstellt:

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};
```

Um auf die geschäftliche Telefonnummer von `Alice` zuzugreifen, verketten Sie die Eigenschaftszugriffe so:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person.contact.phone.work); // "098-765-4321"
```

:::

Sie können auch die Klammernotation verwenden, die besonders nützlich ist, wenn Eigenschaftsnamen Leerzeichen oder Sonderzeichen enthalten oder wenn Sie Variablen zum Zugreifen auf Eigenschaften verwenden:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person['contact']['phone']['work']); // "098-765-4321"
```

:::

Schauen wir uns nun an, wie wir auf Daten zugreifen können, bei denen eine der Objekteigenschaften den Wert eines Arrays hat. Hier ist ein modifiziertes `person`-Objekt, das ein Array von Adressen enthält:

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};
```

Hier ein Beispiel, wie Sie auf die Stadt der Arbeitsadresse von `Alice` zugreifen:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};

console.log(person.addresses[1].city); // "Workville"
```

:::

In diesem Beispiel bezieht sich `person.addresses` auf das Array der Adressen. Um auf die zweite Adresse in diesem Array zuzugreifen, verwenden wir die Klammernotation mit dem Index `1`. Dann verwenden wir die Punktnotation, um die `city` aus diesem Adressobjekt zuzugreifen.

Das Verständnis, wie man auf Eigenschaften in verschachtelten Objekten und Arrays zugreift, ist essenziell beim Arbeiten mit komplexen Datenstrukturen. In zukünftigen Workshops und Labs haben Sie die Gelegenheit, mit diesen Arten von Datenstrukturen zu üben.
