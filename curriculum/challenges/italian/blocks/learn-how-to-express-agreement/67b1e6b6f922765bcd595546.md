# --explanation--

`Make sure` significa agire per confermare che qualcosa sia corretto o completato. Per esempio:  

`Please make sure the report is submitted before the deadline.` – Questo significa controllare e assicurarsi che il rapporto venga consegnato in tempo.  

`Reliable` indica che qualcosa o qualcuno può essere considerato affidabile nel funzionamento o costante nel tempo. Per esempio:

`The server is reliable; it has been running without any issues for months.` - Questo significa che il server funziona in modo costante e non crea problemi.

`More reliable` indica che qualcosa è diventato ancora più affidabile o stabile. Per esempio:

`The new software update makes the app more reliable.` - Questo significa che l'app è diventata più affidabile o stabile dopo l'aggiornamento.
