## Вступ до flexbox і моделі flex

- **Визначення**: CSS flexbox — це одномірна модель макета, яка дозволяє розташовувати елементи в рядках і стовпцях усередині контейнера.  
- **Модель flex**: ця модель визначає, як гнучкі елементи розташовуються всередині гнучкого контейнера. Кожен гнучкий контейнер має дві осі: головну і поперечну.

## Властивість `flex-direction`

- **Визначення**: ця властивість встановлює напрямок головної осі. Значення за замовчуванням `flex-direction` — `row`, що розміщує всі гнучкі елементи в одному рядку у напрямку мови браузера (зліва направо або справа наліво).
- **`flex-direction: row-reverse;`**: це змінює порядок елементів у рядку на зворотний.
- **`flex-direction: column;`**: це вирівнює гнучкі елементи вертикально замість горизонтально.
- **`flex-direction: column-reverse;`**: це змінює порядок гнучких елементів вертикально на зворотний.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: row;
  gap: 10px;
}

.box {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}
```

:::

## Властивість `flex-wrap`

- **Визначення**: ця властивість визначає, як гнучкі елементи обгортаються всередині гнучкого контейнера, щоб поміститися у доступний простір. `flex-wrap` може приймати три можливі значення: `nowrap`, `wrap` і `wrap-reverse`.
- **`flex-wrap: nowrap;`**: це значення за замовчуванням. Гнучкі елементи не будуть переноситися на новий рядок, навіть якщо їхня ширина перевищує ширину контейнера.
- **`flex-wrap: wrap;`**: ця властивість переносить елементи, коли вони перевищують ширину свого контейнера.
- **`flex-wrap: wrap-reverse;`**: ця властивість переносить гнучкі елементи у зворотному порядку.
- **Властивість `flex-flow`**: ця властивість є скороченою властивістю для `flex-direction` і `flex-wrap`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
  <div class="box">4</div>
  <div class="box">5</div>
</div>
```

```css
.container {
  display: flex;
  flex-wrap: wrap;
  width: 150px;
  background: #f0f0f0;
}

.box {
  width: 60px;
  padding: 10px;
  margin: 5px;
  background: skyblue;
  text-align: center;
}
```

:::

## Властивість `justify-content`

- **Визначення**: Ця властивість вирівнює дочірні елементи вздовж головної осі гнучкого контейнера.
- **`justify-content: flex-start;`**: У цьому випадку гнучкі елементи будуть вирівняні на початку головної осі. Це може бути горизонтально або вертикально.
- **`justify-content: flex-end;`**: У цьому випадку гнучкі елементи вирівняні в кінці головної осі, горизонтально або вертикально.
- **`justify-content: center;`**: Це центрує гнучкі елементи вздовж головної осі.
- **`justify-content: space-between;`**: Це рівномірно розподіляє елементи вздовж головної осі.
- **`justify-content: space-around;`**: Це рівномірно розподіляє гнучкі елементи всередині головної осі, додаючи пробіл перед першим і після останнього елемента.
- **`justify-content: space-evenly;`**: Це рівномірно розподіляє елементи вздовж головної осі.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box">A</div>
  <div class="box">B</div>
  <div class="box">C</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
  background: #eee;
}

.box {
  padding: 20px;
  background: salmon;
}
```

:::
  
## Властивість `align-items`

- **Визначення**: ця властивість використовується для розподілу елементів вздовж поперечної осі. Пам’ятайте, що поперечна вісь перпендикулярна до головної осі.
- **`align-items: center;`**: використовується для центрування елементів вздовж поперечної осі.
- **`align-items: flex-start;`**: вирівнює елементи на початку поперечної осі.
- **`align-items: stretch;`**: використовується для розтягування гнучких елементів вздовж поперечної осі.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box tall">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  align-items: center;
  height: 150px;
  background: #ddd;
}

.box {
  background: lightgreen;
  padding: 10px;
}

.tall {
  height: 100px;
}
```
