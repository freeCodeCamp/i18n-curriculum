# --description--

Wir nennen eine positive ganze Zahl $A$ eine „alexandrische ganze Zahl“, wenn es ganze Zahlen $p$, $q$, $r$ gibt, so dass:

$$A = p \times q \times r$$

und

$$\frac{1}{A} = \frac{1}{p} + \frac{1}{q} + \frac{1}{r}$$


Zum Beispiel ist 630 eine alexandrische ganze Zahl ($p = 5$, $q = −7$, $r = −18$). Tatsächlich ist 630 die 6. alexandrische ganze Zahl, die ersten 6 alexandrischen ganzen Zahlen sind: 6, 42, 120, 156, 420 und 630.

Finden Sie die 150000. alexandrische ganze Zahl.
