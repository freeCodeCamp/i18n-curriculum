# --explanation--

`Absolutely` se usa para expresar acuerdo completo o certeza. Por ejemplo: 

Persona 1: `Do you think this solution will work?`

Persona 2: `Absolutely!` - Significa que estás completamente seguro y apoyas la solución.

En este diálogo, Lisa, por lo tanto, le está mostrando a James que está completamente de acuerdo con lo que dijo.
