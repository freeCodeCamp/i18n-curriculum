# --interactive--

Diese Elemente stehen in engem Zusammenhang mit den Konzepten von präsentationalem und semantischem HTML. Das idiomatische Textelement, `i`, wurde ursprünglich für präsentationale Zwecke verwendet, um den Text kursiv darzustellen. Heute wird es jedoch häufig verwendet, um alternative Stimme oder Stimmung, idiomatische Ausdrücke aus einer anderen Sprache, Fachbegriffe und Gedanken hervorzuheben.

Hier ist ein Beispiel aus der offiziellen HTML-Spezifikation, das das `i`-Element verwendet, um einen idiomatischen Ausdruck auf Französisch zu zeigen.

:::interactive_editor

```html
<p>There is a certain <i lang="fr">je ne sais quoi</i> in the air.</p>
```

:::

Das `lang`-Attribut im öffnenden `<i>`-Tag wird verwendet, um die Sprache des Inhalts anzugeben. In diesem Fall wäre die Sprache Französisch. Das `i`-Element zeigt nicht an, ob der Text wichtig ist oder nicht, sondern nur, dass er sich irgendwie vom umgebenden Text unterscheidet.

Wenn Sie jedoch die Wichtigkeit des Texts betonen müssen, können Sie ein ähnliches semantisches Element namens Hervorhebungselement, `em`, verwenden. Dies wird normalerweise empfohlen, wenn Sie mehr Kontext bereitstellen müssen. Sie sollten dieses Element für Textteile verwenden, die im Vergleich zum umgebenden Text eine besondere Betonung benötigen. Es ist meist auf nur wenige Wörter beschränkt, da es die Bedeutung des Satzes verändern kann.

Hier ist ein Beispiel für das Hervorhebungselement innerhalb eines Absatzes.

:::interactive_editor

```html
<p>
  Never give up on <em>your</em> dreams.
</p>
```

:::

Sie sehen den Satz `Never give up on your dreams`. Beachten Sie, dass das Wort `your` hervorgehoben wird, weil es innerhalb dieses Elements steht. Im Browser würde das Wort `your` kursiv dargestellt, um den Lesenden zu zeigen, dass dies ein wichtiges Wort im Satz ist.

Auch wenn es optisch gleich aussieht wie beim idiomatischen Textelement, vermittelt das semantische Hervorhebungselement seine Bedeutung und Wichtigkeit im Hintergrund.

Es ist wichtig zu wissen, dass diese Elemente nicht nur für präsentationale Zwecke verwendet werden sollten. Wenn Sie den Text kursiv darstellen möchten, der Text aber keine besondere Funktion, Stil oder Bedeutung im Absatz hat, sollten Sie stattdessen CSS verwenden.
