# --description--

Тепер, коли ви маєте базове розуміння масивів, давайте поговоримо про цикли. Цикли використовують для багаторазового виконання блоку коду. Один із таких циклів — це цикл `while`. Цикл `while` виконує блок коду доти, поки умова є істинною. Синтаксис циклу `while` виглядає так:

```javascript
while (condition) {
  // code block to be executed
}
```

Наприклад, наступний фрагмент коду виводить числа від 1 до 5:

```javascript
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
```

Вищенаведений фрагмент коду ініціалізує змінну `i` значенням `1`. Цикл `while` виконує блок коду доти, поки значення `i` менше або дорівнює `5`. Значення `i` збільшується на `1` у кожній ітерації.
