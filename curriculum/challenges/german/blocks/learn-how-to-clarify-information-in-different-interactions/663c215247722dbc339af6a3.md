# --description--

Wenn Ihnen jemand etwas erklärt und Sie ihm sagen möchten, dass Sie verstanden haben, was gesagt wurde, können Sie einfach `got it` sagen. Das bedeutet, dass Sie die Erklärung verstanden haben.

`Your turn` bedeutet, dass Sie an der Reihe sind, etwas in einem Spiel oder einer Aktivität zu tun. Zum Beispiel, wenn Sie ein Brettspiel spielen und jemand sagt, `it's your turn`, bedeutet das, dass Sie als Nächstes spielen sollten. `Turns` sind in diesem Zusammenhang die Zeiten, in denen verschiedene Personen nacheinander etwas tun.

Hören Sie sich den Dialog an und füllen Sie die Lücken aus.
