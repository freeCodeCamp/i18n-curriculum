# --description--

تعامل CSS كل عنصر HTML كصندوق خاص به، والذي يُشار إليه عادةً بـ <dfn>نموذج صندوق CSS</dfn>. تبدأ العناصر على مستوى الكتلة تلقائيًا في سطر جديد (مثل العناوين، الفقرات، وdivs) بينما تجلس العناصر المضمَّنة ضمن المحتوى المحيط بها (مثل الصور أو spans). يُسمى التخطيط الافتراضي للعناصر بهذه الطريقة <dfn>التدفق الطبيعي</dfn> للوثيقة، لكن CSS توفر خاصية position لتجاوز ذلك.

عندما يُضبط موضع عنصر على `relative`، يسمح لك بتحديد كيف يجب على CSS تحريكه *نسبيًا* إلى موقعه الحالي في التدفق الطبيعي للصفحة. يترافق ذلك مع خصائص إزاحة CSS مثل `left` أو `right`، و`top` أو `bottom`. تحدد هذه الخصائص عدد البكسلات أو النسب المئوية أو ems التي تُبعد العنصر *عن* موقعه الطبيعي. المثال التالي يحرك الفقرة 10 بكسلات بعيدًا عن الأسفل:

```css
p {
  position: relative;
  bottom: 10px;
}
```

تغيير موضع عنصر إلى نسبي لا يخرجه من التدفق الطبيعي - تظل العناصر المحيطة تتصرف كما لو أن ذلك العنصر في موقعه الافتراضي.

**ملاحظة:** يتيح لك تحديد الموضع مرونة وقوة كبيرة في تخطيط الصفحة بصريًا. من الجيد أن تتذكر أنه مهما كان موضع العناصر، يجب أن يكون ترميز HTML الأساسي منظمًا ومنطقيًا عند قراءته من الأعلى إلى الأسفل. هذا هو الأسلوب الذي يصل به المستخدمون ذوو الإعاقة البصرية (الذين يعتمدون على أجهزة مساعدة مثل قارئ الشاشة) إلى محتواك.

```js
const h2Element =document.querySelector('h2')
const h2Style = window.getComputedStyle(h2Element);
assert.equal(h2Style?.position, 'relative');
```

يجب أن يستخدم شفرتك إزاحة CSS لوضع `h2` نسبيًا على بعد 15 بكسل من `top` حيث يجلس عادةً.

```js
const h2Element =document.querySelector('h2')

```
