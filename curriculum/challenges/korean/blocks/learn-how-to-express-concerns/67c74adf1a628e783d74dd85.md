# --explanation--

`To be better`는 다른 무언가가 개선이 될 수 있음을 암시합니다. 예를 들어:

`Wouldn't it be better to start earlier and avoid last-minute stress?` - 이는 더 일찍 시작하는 것이 개선일 수 있음을 의미합니다.

`Differently`는 다른 방식으로 라는 뜻입니다. 이것은 **어떻게** 무언가가 이루어지는지를 설명합니다. 이는 사물, 사람 또는 상황이 어떠한지를 설명하는 `different`와 대조됩니다. 다른 예:

- `We need a different plan.` - 여기서 `different`는 계획을 설명합니다.  

- `We need to plan differently.` - 여기서는 다른 방식으로 계획하는 방법에 대한 설명입니다.
