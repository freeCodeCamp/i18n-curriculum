# --description--

<ol>
  <li>Візьміть ціле число <code>n₀</code>, більше за нуль.</li>
  <li>Сформуйте наступне число <code>n</code> послідовності, розвернувши <code>n₀</code> і додавши його до <code>n₀</code>.</li>
  <li>Зупиніться, коли <code>n</code> стане паліндромним — тобто цифри <code>n</code> у зворотному порядку дорівнюють <code>n</code>.</li>
</ol>

Вищезазначене рекурентне співвідношення, застосоване до більшості початкових чисел `n` = 1, 2, ... досить швидко завершується паліндромом.

Наприклад, якщо `n₀` = 12, отримаємо:

```md
12
12 + 21 = 33,  a palindrome!
```

А якщо `n₀` = 55, отримаємо:

```md
55
55 + 55 = 110
110 + 011 = 121,  a palindrome!
```

Зверніть увагу, що перевірка на паліндром відбувається *після* додавання.

Деякі початкові числа, здається, тривають вічно; рекурентне співвідношення для 196 було обчислено мільйони разів, утворюючи числа з мільйонами цифр, не утворюючи паліндром. Ці числа, які не закінчуються паліндромом, називають **числами Лічрел**.

Для цілей цього завдання число Лічрел — це будь-яке початкове число, яке не утворює паліндром протягом 500 (або більше) ітерацій.

**Початкове та пов’язані числа Лічрел:**

Будь-яке ціле число, отримане в послідовності числа Лічрел, також є числом Лічрел.

Загалом, будь-яка послідовність від одного числа Лічрел *може* збігатися з послідовністю від попереднього кандидата числа Лічрел; наприклад, послідовності для чисел 196 і 689 починаються так:

```md
    196
    196 + 691 = 887
    887 + 788 = 1675
    1675 + 5761 = 7436
    7436 + 6347 = 13783
    13783 + 38731 = 52514
    52514 + 41525 = 94039
    ...
    689
    689 + 986 = 1675
    1675 + 5761 = 7436
    ...
```

Отже, бачимо, що послідовність, що починається з 689, збігається і продовжується тими ж числами, що й для 196.

Через це ми можемо додатково розділити числа Лічрел на справжні кандидати **початкових** чисел Лічрел і **пов’язані** числа, які не утворюють паліндромів, але мають числа у своїй послідовності, які зустрічаються в послідовності, згенерованій від меншого числа Лічрел.

```js
function isLychrel(n) {
  function reverse(num) {
    return parseInt(
      num
        .toString()
        .split('')
        .reverse()
        .join('')
    );
  }

  var i;
  for (i = 0; i < 500; i++) {
    n = n + reverse(n);
    if (n == reverse(n)) break;
  }

  return i == 500;
}
```
