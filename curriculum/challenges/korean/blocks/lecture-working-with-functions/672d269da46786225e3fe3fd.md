# --interactive--

함수는 특정 작업을 수행하거나 값을 계산하는 재사용 가능한 코드 조각입니다. 함수를 입력을 받아서 어떤 작업을 수행한 후 출력을 내는 기계로 생각하세요. 다음은 함수를 선언하는 예입니다:

```js
function greet() {
  console.log("Hello, Jessica!");
}
```

이 예제에서는 `greet`라는 함수를 선언했습니다. 그 함수 안에는 `console.log`가 있고, 이 함수는 `Hello, Jessica!`라는 메시지를 로그에 출력합니다. 이 코드를 실행해도 콘솔에 메시지가 나타나지 않습니다. 함수가 호출되어야 하기 때문입니다.

함수 호출 또는 실행은 실제로 함수를 사용하거나 실행하는 것을 의미합니다. 함수를 호출하려면 함수 이름 뒤에 소괄호를 붙여 참조해야 합니다:

:::interactive_editor

```js
function greet() {
  console.log("Hello, Jessica!");
}

greet(); // "Hello, Jessica!"
```

:::

이제 `Hello, Jessica!` 메시지가 콘솔에 출력됩니다. 그런데 메시지가 `Hello, Nick!` 또는 `Hello, Anna!`라고 나오게 하려면 어떻게 해야 할까요? 다른 사용자를 인사할 때마다 새 함수를 작성하고 싶지 않습니다. 대신, 함수 매개변수와 인수를 사용하는 재사용 가능한 함수를 만들 수 있습니다.

매개변수는 함수가 호출될 때 전달받을 값의 자리 표시자 역할을 합니다. 매개변수를 통해 함수는 입력을 받아서 그 입력으로 작업할 수 있습니다. 인수는 함수가 호출될 때 실제로 전달되는 값입니다. 다음은 매개변수와 인수를 사용하는 `greet` 함수의 업데이트된 버전입니다:

:::interactive_editor

```js
function greet(name) {
  console.log("Hello, " + name + "!");
}

greet("Alice"); // Hello, Alice!
greet("Nick"); // Hello, Nick!
```

:::

`name`는 매개변수 역할을 하고, `Alice`와 `Nick` 문자열이 인수 역할을 합니다. 이제 우리는 다양한 인수로 코드 전반에서 여러 번 사용할 수 있는 재사용 가능한 함수를 갖게 되었습니다.

함수가 실행을 마치면 항상 값을 반환합니다. 기본적으로 반환값은 `undefined`입니다. 다음은 예시입니다:

:::interactive_editor

```js
function doSomething() {
  console.log("Doing something...");
}

let result = doSomething();
console.log(result); // undefined
```

:::

함수가 특정 값을 반환하도록 하려면 `return` 문을 사용해야 합니다. 다음은 두 값을 더한 결과를 반환하는 `return` 문 사용 예입니다:

:::interactive_editor

```js
function calculateSum(num1, num2) {
  return num1 + num2;
}

console.log(calculateSum(3, 4)); // 7
```

:::

대부분 `return` 문을 사용하게 될 텐데, 함수에서 출력된 값을 코드에서 나중에 사용할 수 있기 때문입니다.

지금까지는 이름이 있는 함수를 다뤘지만, 익명 함수라고 하는 이름 없는 함수도 만들 수 있습니다. 익명 함수는 다음과 같이 변수에 할당할 수 있습니다:

:::interactive_editor

```js
const sum = function (num1, num2) {
  return num1 + num2;
};

console.log(sum(3, 4)); // 7
```

:::

이 예제에서는 `const` 변수 `sum`에 `num1`와 `num2`의 합을 반환하는 익명 함수를 할당했습니다. 그런 다음 `sum`을 호출하고 `3`와 `4` 숫자를 전달하여 `7` 결과를 얻을 수 있습니다.

함수는 기본 매개변수를 지원하여 매개변수에 기본값을 설정할 수 있습니다. 함수가 해당 매개변수에 대한 인수 없이 호출되면 이 기본값이 사용됩니다. 예시는 다음과 같습니다:

:::interactive_editor

```js
function greetings(name = "Guest") {
  console.log("Hello, " + name + "!");
}

greetings(); // Hello, Guest!
greetings("Anna"); // Hello, Anna!
```

:::

이 예제에서 `name`에 인수가 제공되지 않으면 기본값 `Guest`이 사용됩니다.

요약하면, 함수는 재사용 가능하고 체계적인 코드를 작성할 수 있게 해줍니다. 입력(매개변수)을 받아 작업을 수행하고 출력을 반환할 수 있습니다.
