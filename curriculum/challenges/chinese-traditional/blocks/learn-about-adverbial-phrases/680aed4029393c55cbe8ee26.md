# --explanation--

`Which` 用來介紹非限定性關係語句－一個提供關於前面整體想法更多資訊的語句。它增加額外細節，但對句子的主要意義並非必要。 例如：

`She missed the deadline, which surprised everyone.` – 該語句添加了關於人們如何反應她錯過截止日期的額外資訊（句子中最重要的零件）。

在背景關係中，Riker 使用 `which` 來表達他對額外支出的看法，這是與超出預算金額相關的額外一則資訊——也是 Maria 真正詢問的內容。
