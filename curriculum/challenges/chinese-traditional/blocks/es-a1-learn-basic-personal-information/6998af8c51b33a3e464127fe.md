# --explanation--

`Navarro con dos erres.` 遵循結構：`con` + 數字 + 複數名詞。

`Con` 添加有關拼寫的資訊。

`Dos` 給出數量。  

`Erres` 取用兩個字母 `r`。

當說明一個字是如何書寫時，這種結構體很常見。
