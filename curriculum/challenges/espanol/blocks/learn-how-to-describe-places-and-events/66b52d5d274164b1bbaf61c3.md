# --explanation--

El tiempo `Present Continuous` se usa para describir acciones que están ocurriendo en el momento de hablar. Se forma usando el verbo `to be` (`am, is, are`) + la forma base del verbo + `-ing`. Por ejemplo:

- Ella está codificando una nueva característica en este momento.

- `They are testing the software.` - Ellos están en medio de probar el software ahora mismo.
