# --description--

Katika maabara hii, utajenga programu inayobadilisha kati ya mandhari tofauti. Unapobadilisha kwenda mandhari tofauti, rangi ya nyuma itabadilika na ujumbe tofauti utaonyeshwa.

**Lengo:** Timiza hadithi za watumizi zilizo hapa chini na upite vipimo vyote ili kumaliza maabara.

**Hadithi za watumizi:**

1. Unapaswa kuwa na kipengele cha `button` chenye maandishi ya `Switch Theme`.
2. Kipengele chako cha `button` kinapaswa kuwa na sifa za kipengele zifuatazo:
   - Sifa ya `id` ikiwa imewekwa kuwa `"theme-switcher-button"`.
   - Sifa ya `aria-haspopup` ikiwa imewekwa kuwa `"true"`.
   - Sifa ya `aria-expanded` ikiwa imewekwa kuwa `"false"`.
   - Sifa ya `aria-controls` ikiwa imewekwa kuwa `"theme-dropdown"`.
3. Unapaswa kuwa na kipengele cha `ul` chenye sifa za kipengele zifuatazo:
   - Sifa ya `id` ikiwa imewekwa kuwa `"theme-dropdown"`.
   - Sifa ya `role` ikiwa imewekwa kuwa `"menu"`.
   - Sifa ya `aria-labelledby` ikiwa imewekwa kuwa `"theme-switcher-button"`.
   - Sifa ya `hidden`.
4. Kipengele chako cha `ul` kinapaswa kuwa na angalau vipengele viwili vya `li` vyenye sifa ya kipengele ya `role` iliyowekwa kuwa `"menuitem"` na maandishi ya uchaguzi wako yanayowakilisha mandhari tofauti.
5. Kila kipengele chako cha `li` kinapaswa kuwa na sifa ya kipengele ya `id` inayotangulia na `theme-` na kumalizika na mandhari uliyoiweka kwa maandishi ya kipengele cha `li`. Kwa mfano, ikiwa moja ya mandhari zako ni `Light` basi `id` yako inapaswa kuwa `theme-light`.
6. Unapaswa kuwa na kipengele chenye sifa ya kipengele ya `aria-live` iliyowekwa kuwa `"polite"`.
7. Unapaswa kuwa na safu ya data ya `themes` yenye angalau vitu viwili ambavyo kila kimoja kina sifa za `name` na `message`. `name` itawakilisha mandhari tofauti na `message` itaonyeshwa wakati mandhari inabadilika. Una uhuru wa kuja na thamani za `name` na `message` za uchaguzi wako lakini thamani za `name` zinapaswa kufanana na maudhui ya maandishi ya vipengele vyako vya `li`.
8. Mtumizi anapobofya `#theme-switcher-button`, kipengele cha `#theme-dropdown` kinapaswa kuonyesha mandhari zinazopatikana.
9. Kipengele cha `#theme-dropdown` kinapoonyeshwa, sifa ya kipengele ya `aria-expanded` inapaswa kuwekwa kuwa `"true"` kwa kipengele cha `button`.
10. Watumizi wanapobofya `#theme-switcher-button` wakati kipengele cha `#theme-dropdown` kikiwa kinaonyeshwa, menyu inapaswa kufungwa na sifa ya kipengele ya `hidden` inapaswa kuongezwa na sifa ya kipengele ya `aria-expanded` iwekwa kuwa `"false"`.
11. Mtumizi anapobofya `#theme-switcher-button` na kuchagua mandhari, darasa la vitu la `theme-<name>` linalohusiana linapaswa kuongezwa kwa kipengele cha `body` na ujumbe wa mandhari unaohusiana ulioweka katika safu ya data ya `themes` utaonyeshwa katika kipengele cha `aria-live="polite"`.
12. Mtumizi anapobofya `#theme-switcher-button` na kuchagua mandhari tofauti na ile ya sasa, darasa la mandhari la `body` na ujumbe wa hali vinapaswa kusasishwa ili kuendana na uchaguzi mpya.

```js
const themes = [
  {
    name: "light",
    message: "Hello sunshine — Light theme is on!"
  },
  {
    name: "dark",
    message: "The night is yours — Dark theme is on!"
  },
  {
    name: "nord",
    message: "The frost has settled - Nord theme is on!"
  },
  {
    name: "ocean",
    message: "Blue skies and high tides — Ocean theme is on!"
  },
];

const setTheme = (theme) => {
  document.body.className = "";
  document.body.classList.add(`theme-${theme.name}`);
};

const updateStatus = (theme) => {
  const status = document.getElementById("status");
  status.textContent = theme.message;
};

document.addEventListener("DOMContentLoaded", () => {
  themes.forEach((theme) => {
    const button = document.getElementById(`theme-${theme.name}`);
    
    if (button) {
      button.addEventListener("click", () => {
        setTheme(theme);
        updateStatus(theme);
        closeMenu();
      });
    }
  });

  const toggleButton = document.getElementById("theme-switcher-button");
  const menu = document.getElementById("theme-dropdown");

  toggleButton.addEventListener("click", () => {
    const isExpanded = toggleButton.getAttribute("aria-expanded") === "true";
    if (isExpanded) {
      closeMenu();
    } else {
      openMenu();
    }
  });

  function closeMenu() {
    menu.setAttribute("hidden", "");
    toggleButton.setAttribute("aria-expanded", "false");
  }

  function openMenu() {
    menu.removeAttribute("hidden");
    toggleButton.setAttribute("aria-expanded", "true");
  }

  // Close the menu when clicking outside
  document.addEventListener("click", (e) => {
    if (!toggleButton.contains(e.target) && !menu.contains(e.target)) {
      closeMenu();
    }
  });
});
```
