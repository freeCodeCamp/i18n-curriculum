# --explanation--

Eine `data breach` tritt auf, wenn sensible Informationen von unbefugten Personen aufgerufen werden. Zum Beispiel:

- `A data breach can happen if hackers get access to private information.` – Wenn Cyberkriminelle Informationen stehlen, spricht man von einer Datenpanne.

- `Companies must act quickly if there is a data breach.` – Wenn eine Datenpanne passiert, müssen Unternehmen schnell reagieren, um sie zu beheben.

Jessica sagt, dass das Verschlüsseln von Daten und das sichere Speichern von Verschlüsselungsschlüsseln entscheidend ist, um dieses Risiko zu verhindern.
