# --description--

Ikiwa umepewa mfuatano wa herufi wa wakati katika muundo wa `"HH:MM"` (saa 24), rudisha:

- `"Good morning"` kwa nyakati `05:00` hadi `11:59`
- `"Good afternoon"` kwa nyakati `12:00` hadi `17:59`
- `"Good evening"` kwa nyakati `18:00` hadi `21:59`
- `"Good night"` kwa nyakati `22:00` hadi `04:59`

```js
function getGreeting(time) {
  const [hours, minutes] = time.split(':').map(Number);
  const total = hours * 60 + minutes;

  if (total >= 300 && total < 720) return 'Good morning';
  if (total >= 720 && total < 1080) return 'Good afternoon';
  if (total >= 1080 && total < 1320) return 'Good evening';
  return 'Good night';
}
```
