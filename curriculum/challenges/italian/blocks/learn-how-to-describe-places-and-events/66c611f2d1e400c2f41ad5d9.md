# --explanation--

Nel tempo `Present Simple`, la forma negativa si crea usando `doesn't` o `isn't` per i soggetti singolari di terza persona (`he, she, it`).

- Usa `doesn't` con i verbi d'azione per indicare che qualcosa non sta succedendo, come `It doesn't work.`

- Usa `isn't` con le forme di `to be` per indicare che qualcosa non è vero, come `isn't set up.`

In questa frase, serviranno sia `doesn't` sia `isn't` per descrivere correttamente perché la funzionalità non funziona come previsto.
