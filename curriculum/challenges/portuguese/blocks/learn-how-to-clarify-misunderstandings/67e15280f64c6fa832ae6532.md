# --explanation--

`Sit in on` significa participar de uma reunião ou sessão como observador.

Mark diz `thanks for letting me sit in on the code review`. Isso mostra que ele recebeu permissão para estar lá e observar. Ele não está liderando a sessão nem fazendo qualquer codificação — ele está apenas participando para ver como funciona.
