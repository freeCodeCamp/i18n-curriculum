# --explanation--

Jessica dice a Mark di eseguire `regular security audits` per garantire la sicurezza.

Un `security audit` è un processo di controllo di un sistema per individuare vulnerabilità. Per esempio:

`After the security audit, the team made improvements to the system.` - Sono state apportate modifiche per migliorare la sicurezza dopo la revisione.

`Regular` significa che avviene in modo costante nel tempo. Per esempio:

`Regular testing helps find problems before they get worse.` - Testare spesso aiuta a individuare i problemi precocemente.
