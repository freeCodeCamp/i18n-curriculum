# --explanation--

`User experience` (ou `UX`) é sobre como um usuário se sente ao usar um produto — se é fácil, agradável e fluido.Por exemplo:

`Improving page load time boosts user experience.` — Fazer as páginas carregarem mais rápido ajuda os usuários a aproveitarem mais o aplicativo.

`Enhancements` são mudanças feitas para melhorar um produto ou processo.Por exemplo:

`The latest update includes performance enhancements.` — A nova versão faz o aplicativo funcionar melhor e mais rápido.

Na frase de Alice, metade do orçamento de design é gasto para tornar o produto mais fácil e melhor de usar.
