# --interactive--

محدِّد الخاصية في CSS هو أداة قوية تتيح لك استهداف عناصر HTML بناءً على خصائصها. هذا يعني أنه يمكنك تطبيق تنسيقات على العناصر التي تحتوي على خصائص معينة أو حتى قيم محددة لتلك الخصائص.

يكون مفيدًا بشكل خاص عندما تريد تنسيق العناصر بشكل ديناميكي أو عندما لا توفر أسماء الأصناف وحدها دقة كافية.

على سبيل المثال، يمكنك استخدام محدِّد الخاصية لاستهداف كل الروابط التي تحتوي على خاصية `href`. هذا مفيد جدًا عندما تريد تطبيق تنسيق موحد على كل الروابط في الصفحة.

إليك كيف يمكنك استخدام محدِّد الخاصية لاستهداف الروابط التي تحتوي على خاصية `href`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Example link with an href attribute</a>
<a>Example link without an href attribute</a>
```

```css
a {
  display: block;
}

a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

تطبق قاعدة CSS هذه لونًا أزرقًا وتسطيرًا على أي رابط يحتوي على خاصية `href`، مما يضمن تنسيق كل الروابط القابلة للنقر بشكل متناسق.

لكن محددات الخصائص يمكن أن تكون أكثر تحديدًا. افترض أنك تريد استهداف الروابط التي تحتوي فقط على خاصية `title`. غالبًا ما توفر خاصية `title` معلومات إضافية عن الرابط، وقد ترغب في تنسيق هذه الروابط بشكل مختلف لتوضيح وجود معلومات إضافية. إليك كيف تفعل ذلك:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

تطبق هذه القاعدة نصًا عريضًا وتزيل التسطير من أي رابط يحتوي على خاصية `title`. هذه طريقة رائعة لتمييز هذه الروابط بصريًا عن غيرها في الصفحة.

يمكنك أيضًا دمج محددات الخصائص لتصبح أكثر دقة. على سبيل المثال، إذا أردت تنسيق الروابط التي تحتوي على كل من خاصيتي `href` و`title`، يمكنك فعل ذلك:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[href][title] {
  display:block;
  color: green;
}
```

:::

في هذه الحالة، فقط الروابط التي تحتوي على كل من خاصيتي `href` و`title` ستُنسَّق بنص أخضر. هذا المستوى من التحكم هو ما يجعل محددات الخصائص قوية جدًا في CSS.

مثال آخر لمحدد الخاصية هو مطابقة قيمة واحدة ضمن قائمة قيم مفصولة بمسافات في خاصية ما.

إليك مثالًا على عنصر وسم الرابط مع عدة أصناف:

```html
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

لاستهداف هذا العنصر المحدد، يمكنك استخدام المحدد التالي:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

تُستخدم صيغة `[attr~=value]` هنا لاستهداف جميع عناصر وسم الرابط التي تحتوي خاصية الصنف فيها على الكلمة `"primary"`.

إذا كنت تحتاج لاستهداف عنصر تكون قيمة خاصيته مسبوقة بقيمة محددة، يمكنك استخدام صيغة `[attr^=value]`. إليك مثالًا:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

في هذا المثال، سيستهدف المحدد `a[href^="https://"]` جميع عناصر وسم الرابط التي تبدأ قيمة الخاصية `href` فيها بـ`"https://"`.

لاستهداف العناصر التي تنتهي قيمة خاصيتها بقيمة محددة، يمكنك استخدام صيغة `[attr$=value]`. إليك مثالًا:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Visit Example Site</a>
```

```css
a[href$=".com"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

في هذا المثال، سيستهدف المحدد `a[href$=".com"]` كل عناصر وسم الرابط التي تنتهي قيمة خاصية `href` فيها بـ `.com`.

استخدام محددات الخصائص لا يعزز فقط تنسيق صفحة الويب بل يحسن أيضًا إمكانية الوصول بجعل العناصر التفاعلية مثل الروابط أكثر تمييزًا بناءً على خصائصها.
