# --description--

前のレッスンでは、依存関係とは何か、そしてプロジェクトに適した依存関係の選び方を学びました。

このレッスンでは、npmを使って依存関係をインストールおよび削除する方法を学びます。

依存関係をインストールするには、次のように`npm install`コマンドを使います。

```bash
npm install dependencyName
```

また、`install`を`i`のように短縮することもできます。

```bash
npm i dependencyName
```

パッケージをインストールすると、npmは`dependencies`ファイルに新しい`package.json`セクションを追加し（まだなければ）、インストールしたパッケージをリストに載せます。例えば、`chalk`パッケージをインストールしたときは次のようになります。

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "chalk": "^5.6.2"
  }
}
```

依存関係をインストールすると、プロジェクトのルートフォルダーに2つの新しいものが現れます。

* `node_modules/`：インストールしたパッケージの実際のコードが入ったフォルダー
    
* `package-lock.json`：それらのパッケージの正確なバージョンや依存関係の詳細情報を保存するファイルです。（このファイルについては後のレッスンで詳しく学びます）
    

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/how-can-you-install-and-remove-dependencies-1.png" alt="">

ほとんどのプロジェクトでは複数の依存関係が必要です。1つずつインストールする代わりに、npmではスペースで区切った複数のパッケージ名を一度に入力してインストールできます。

```bash
npm install express lodash bcrypt
```

前と同様に、複数の依存関係を一度にインストールすると、npmは`package.json`と`package-lock.json`ファイルおよび`node_modules`を更新してそれらを含めます。`package.json`は次のようになります。

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "bcrypt": "^6.0.0",
    "chalk": "^5.6.2",
    "express": "^5.1.0",
    "lodash": "^4.17.21"
  }
}
```

`devDependency`をインストールするには、`--save-dev`コマンドに`install`フラグを追加します。短縮版の`-D`も使えます。

```bash
npm i nodemon -D
```

このコマンドを実行すると、npmは`package.json`ファイルを更新し、パッケージを`devDependencies`セクションに追加します。例は次の通りです。

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "bcrypt": "^6.0.0",
    "chalk": "^5.6.2",
    "express": "^5.1.0",
    "lodash": "^4.17.21"
  },
  "devDependencies": {
    "nodemon": "^3.1.10"
  }
}
```

`npm install`コマンドを実行すると、npmはインストールが成功したかどうかのメッセージを表示します。成功メッセージは通常次のようになります。


パッケージをアンインストールまたは削除するには、`npm uninstall`コマンドを使います。

```bash
npm uninstall dependencyName
```

例えば、プロジェクトから`chalk`を削除するには、次のコマンドを実行します。

```bash
npm uninstall chalk
```

複数のパッケージを一度にアンインストールする場合は、パッケージ名をスペースで区切ってリストにします。

```bash
npm uninstall express lodash bcrypt
```

アンインストールコマンドが成功したか失敗したかのメッセージも表示されます。

```bash
user@Kolade ~/Desktop/fCC/script-code/node/npm/install-uninstall-deps  % npm uninstall chalk

added 1 package, removed 1 package, and audited 101 packages in 1s

20 packages are looking for funding
  run `npm fund` for details

found 0 vulnerabilities

user@Kolade ~/Desktop/fCC/script-code/node/npm/install-uninstall-deps  % npm uninstall express lodash bcrypt

removed 70 packages, and audited 31 in 1s

4 packages are looking for funding
  run `npm fund` for details

found 0 vulnerabilities
```

そして`package.json`ファイルは自動的に更新されます。

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "devDependencies": {
    "nodemon": "^3.1.10"
  }
}
```

`devDependency`をアンインストールするには、単に次のコマンドを実行すればよいです。

```bash
npm uninstall nodemon
```

アンインストール時に`-D`フラグは必要ありません。npmは自動的に正しいセクション（この場合は`devDependencies`）からパッケージを削除します。

`devDependency`をアンインストールした後、`package.json`は再び更新されます。

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC"
}
```
