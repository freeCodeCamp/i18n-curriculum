# --interactive--

Opereta za kulinganisha hukuruhusu kulinganisha thamani mbili na kurudisha matokeo ya `true` au `false`. Kisha unaweza kutumia matokeo hayo kufanya uamuzi au kudhibiti mtiririko wa programu yako. Unatumia kulinganisha katika taarifa za `if`, mizunguko, na hali nyingine nyingi ambapo unahitaji kufanya maamuzi kulingana na masharti fulani. Hebu tuchunguze opereta za kulinganisha zinazotumika zaidi na kuona jinsi zinavyofanya kazi.

Opereta ya kubwa kuliko, inayowakilishwa na alama ya pembetatu ya kulia (`>`), huangalia kama thamani upande wa kushoto ni kubwa kuliko ile upande wa kulia:

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a > b); // false
console.log(b > a); // true
```

:::

Opereta ya kubwa kuliko au sawa, inayowakilishwa na alama ya pembetatu ya kulia na alama ya sawa (`>=`), huangalia kama thamani upande wa kushoto ni kubwa kuliko au sawa na ile upande wa kulia:

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a >= b); // false
console.log(b >= a); // true
console.log(a >= c); // true
```

:::

Opereta ya ndogo kuliko, inayowakilishwa na alama ya pembetatu ya kushoto (`<`) hufanya kazi kama `>`, lakini kwa mwelekeo wa kinyume. Huangalia kama thamani upande wa kushoto ni ndogo kuliko ile upande wa kulia:

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a < b); // true
console.log(b < a); // false
```

:::

Opereta ya ndogo kuliko au sawa, inayowakilishwa na alama ya pembetatu ya kushoto na alama ya sawa (`<=`) huangalia kama thamani upande wa kushoto ni ndogo kuliko au sawa na ile upande wa kulia:

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a <= b); // true
console.log(b <= a); // false
console.log(a <= c); // true
```

:::
