# --description--

Se vuoi avere una lista di elementi in cui l’ordine non conta, come una lista della spesa con elementi che possono essere acquistati in qualsiasi ordine, allora puoi usare una lista non ordinata.

Le liste non ordinate si creano usando l’elemento `<ul>`, e ogni elemento all’interno della lista si crea usando l’elemento lista `<li>`.

Ogni elemento di una lista non ordinata inizia con un punto elenco:

Se invece vuoi creare una lista di elementi in cui l’ordine conta, come istruzioni passo dopo passo per una ricetta, o la tua top 10 delle serie TV preferite, allora puoi usare una lista ordinata.

Le liste ordinate si creano usando l’elemento `<ol>`. Ogni singolo elemento in esse si crea di nuovo usando l’elemento lista `<li>`. Tuttavia, ogni elemento di una lista ordinata inizia con un numero:
