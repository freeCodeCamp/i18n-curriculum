# --explanation--

Un `suspicious email` es un correo electrónico que parece poco confiable o peligroso. Podría ser parte de un `phishing attack`, donde alguien intenta robar información. Por ejemplo:

`This looks like a suspicious email. Can you check it?` - Piensas que el correo podría ser peligroso y le pides a alguien que lo revise.
