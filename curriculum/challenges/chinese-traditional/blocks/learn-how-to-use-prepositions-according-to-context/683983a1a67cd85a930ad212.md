# --explanation--

`Looking at the error logs` 顯示專注於錯誤日誌以識別程式碼中的問題。這與建立程式直接相關。

其他選項要麼專注於位置，要麼錯誤地使用 `at`，使它們在此情況下較不相關或不正確。
