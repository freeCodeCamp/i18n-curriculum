# --explanation--

`They'll` は `they will` の短縮形です。将来起こると予想または予測されることについて話すときに使います。

`Likely` は、何かが起こる可能性が高いことを意味します。例えば：

- `They'll likely ask for more information.` - これはクライアントからのリクエストが起こる可能性が高いことを予測しています。

- `The feature will likely need additional testing.` - これは試験が必要になる可能性が高いことを示しています。
