# --description--

SEO，或稱搜尋引擎最佳化，是一種最佳化網頁，使其在搜尋引擎上更容易被看見並獲得更高等級的做法。提升你網站 SEO 的一種方法，是使用 `meta` 元素為網頁提供簡短描述。以下是一個使用 `meta` 元素為園藝網站設定頁面描述的範例：

```html
<meta
  name="description"
  content="Discover expert tips and techniques for gardening in small spaces, choosing the right plants, and maintaining a thriving garden."
/>
```

透過將 `name` 屬性設定為 `description`，可確保瀏覽器、搜尋引擎及其他網頁工具正確解讀此詮釋資料。`content` 屬性是你放置描述的地方。建議你保持描述簡短且精確，因為搜尋引擎通常會根據結果頁面版面截斷描述。

類似其他型別的 `meta` 元素，`meta` 描述不會在網頁本身顯示。頁面描述可以在搜尋引擎結果頁的摘要中找到。以下是 freeCodeCamp 的 subreddit 和 GitHub 儲存庫的頁面結果摘要範例：

```sh
r\FreeCodeCamp: This is the official subreddit for the freeCodeCamp.org community. Learn to
code for free together with millions of other people...
```

```sh
Our full-stack web development and machine learning curriculum is completely free and self-
paced. We have thousands of interactive coding challenges to help you...
```

在範例中，每個頁面描述都位於網站聯結正下方。在幾秒鐘內，使用者就能大致了解該頁面的內容，並決定是否點擊聯結以獲取更多資訊。

即使 `meta` 描述不會直接影響網站在搜尋引擎上的等級，擁有強而有力的描述仍可能為你的網站帶來更多流量。
