# --description--

La palabra `should` es un verbo modal usado para dar consejos o sugerencias. Va seguido de la forma base del verbo. Por ejemplo, en la oración `You should try that restaurant`, `should` es seguido por `try`, que es la forma base del verbo.

El verbo `pick` significa elegir o seleccionar algo de un rango de opciones. En el contexto de la programación, `picking a data type` significa elegir el tipo de datos apropiado para usar, como un entero, cadena o booleano, basado en lo que sea más adecuado para la tarea en cuestión.

`Coincidir` significa asegurarse de que dos cosas sean adecuadas o encajen bien juntas. Por ejemplo, `Esta clave coincide con la cerradura.` Indica que la clave es adecuada para la cerradura.
