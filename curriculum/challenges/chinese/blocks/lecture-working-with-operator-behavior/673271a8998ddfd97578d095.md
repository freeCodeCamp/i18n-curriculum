# --interactive--

如果你在处理数字时需要增加或减少一个数值，自增和自减运算符会让工作变得更容易。 让我们来简单地分析一下。

自增和自减运算符分别用 `++` 和 `--` 表示。 它们都可以让一个变量的值增加或减少 `1`。

像 `x = x + 1` 或 `x = x - 1`，你只需用 `x++` 来加 `1`，或 `x--` 来减 `1`。 它更快、更干净，也更容易阅读。

自增和自减运算符的工作方式有一个小玄机：它们有两种形式，前缀和后缀，区别在于何时更新值。 自增运算符的前缀形式是 `++x`，后缀形式是 `x++`。

前缀形式（`++x`）是先增加变量的值，然后返回新值。 后缀形式（`x++`）是先返回变量的当前值，然后增加该值：

:::interactive_editor

```js
let x = 5;

console.log(++x); // 6
console.log(x); // 6
```

:::

在上面的代码中，`++x` 表示“先增加 `x`，然后使用它”。 因此，当你记录 `++x` 时，就会立即得到增加后的值，即 `6` 。

现在，我们来看一个使用后缀的例子：

:::interactive_editor

```js
let y = 5;

console.log(y++); // 5
console.log(y); // 6
```

:::

在本例中，`y++` 表示“先使用 `y`，然后增加它的值”。 当你记录 `y++` 时，会得到 `5`，但当该行代码执行完，`y` 会变成 `6`。

自减运算符的作用与自增运算符相同，只是将数值减少 `1` 。 同样，有两种形式：前缀形式（`--x`）先减少变量的值，然后返回新值。 后缀形式（`x--`）先返回当前值，然后减少它：

:::interactive_editor

```js
let x = 5;
console.log(--x); // 4
console.log(x); // 4

let y = 5;
console.log(y--); // 5
console.log(y); // 4
```

:::

那么，应该使用前缀还是后缀？ 在很多情况下，使用哪种并不重要。 结果都是一样的。 但是，如果要在表达式中立即使用更新后的值，两者的区别就变得非常重要。 让我们来看看这个例子：

:::interactive_editor

```js
let a = 5;
let b = ++a;
console.log(b); // 6 (a was incremented before assignment)

let c = 5;
let d = c++;
console.log(d); // 5 (c was incremented after assignment)
```

:::

因此，如果你需要立即获得更新值，请使用前缀。 如果你想先得到当前值，然后再考虑自增，那么就使用后缀。
