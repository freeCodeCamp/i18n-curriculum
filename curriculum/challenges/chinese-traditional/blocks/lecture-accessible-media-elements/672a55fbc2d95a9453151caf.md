# --interactive--

許多使用者因為實體上的障礙、重複性勞損傷害或個人偏好，而依賴鍵盤而非滑鼠。這包括螢幕閱讀器的使用者以及可能沒有可用滑鼠的使用者。鍵盤存取確保這些使用者能有效地瀏覽網頁應用程式，且不受阻礙。

讓我們來看看你可以使用的一些實用技巧，以使網頁應用程式具備鍵盤存取性。

許多使用者依賴 `Tab` 鍵在網頁上的互動元素間移動。瀏覽器預設會讓使用者依照 HTML 中出現的順序，使用 `Tab` 鍵瀏覽像是聯結、按鈕和表單欄位等元素。這稱為自然的分頁順序。

有時候，你可能想調整哪些元素可以聚焦或改變它們的聚焦順序。`tabindex` 屬性允許你這麼做。

這是基本語法：

```html
<element tabindex="number">Element Text</element>
```

`tabindex` 的值決定元素在鍵盤導覽中的行為：

`tabindex="0"` 將元素加入自然的切換鍵順序。 

按 Tab 鍵會將焦點從 `button` 移動到 `div`，然後移動到聯結，依照它們在 HTML 中的順序。

點擊預覽視窗的空白處任意位置。然後使用 `Tab` 鍵查看焦點在不同元素之間移動。若要與範例互動，你需要啟用互動式編輯器。

:::interactive_editor

```html
<button>First</button>
<div tabindex="0">Second</div>
<a href="#">Third</a>
```

:::

`tabindex="-1"` 使元素能夠以程式方式取得焦點。這對於管理通常無法取得焦點的元素（例如標題、容器、對話窗或錯誤訊息）非常有用：

```html
<p tabindex="-1">Sorry, there was an error with your submission.</p>
```

在此範例中，段落不屬於正常的標籤鍵順序，因此使用者無法透過按下 `Tab` 鍵來到達它。然而，如果你透過腳本將焦點設定到此元素，該訊息將會引起使用者的注意。你將在 JavaScript 課程中學習更多關於此技巧。

當 `tabindex` 大於 `0` 時，它會設定訂製的標籤順序。因此，具有較低正值的元素會先被聚焦。

在此範例中，按 Tab 鍵會先聚焦具有 `tabindex="1"` 的 `input`，接著是 `2`，然後是 `3`，不論它們在 HTML 中的順序為何。

```html
<input tabindex="2">
<input tabindex="1">
<input tabindex="3">
```

訂製的正值有時會用於複雜的元件，例如你想要特定導覽順序的工具列。然而，不建議使用這種方法，因為它可能會使導覽變得混亂且難以維護，尤其是在頁面擴充或變更時。

`accesskey` 是你可以用來讓你的網頁專案鍵盤可存取的另一個屬性。它允許你定義一個按鍵，將焦點設置到特定元素或作用該元素。

以下是一個互動範例，你可以依照以下建議嘗試（請先啟用互動式編輯器）：

- `accesskey="s"` 將按鍵 `S` 指定給 `Save` 按鈕。在大多數瀏覽器中，按下 `ALT + S`（在 Windows）和 `CTRL + Option + S`（在 Mac）會作用此按鈕。
- `accesskey="c"` 將按鍵 `C` 指定給 `Cancel` 按鈕，允許使用者使用 `ALT + C`（Windows）和 `CTRL + Option + C`（Mac）來作用它。
- `accesskey="h"` 將按鍵 `H` 指定給 `Home` 連結，讓使用者可以使用 `ALT + H`（Windows）和 `CTRL + Option + H`（Mac）來導覽到首頁。

請注意，啟動 `accesskey` 的確切按鍵組合可能會因瀏覽器和作業系統而異。在 Windows 上通常是 `ALT + Specified Key`，而在 Mac 上則是 `CTRL + Option + Specified Key`。

:::interactive_editor

```html
<button accesskey="s">Save</button>
<button accesskey="c">Cancel</button>
<a href="index.html" accesskey="h">Home</a>
```

:::

另一種讓鍵盤在你的應用程式中可存取的方法是確保你提供清晰的焦點指示器。如果你覺得瀏覽器的預設焦點指示器不夠，你可以透過標的元素的焦點狀態來改寫它。

這裡是一個為 `button` 元素設定焦點狀態樣式的範例。點擊預覽視窗的空白處，然後按下 `Tab` 鍵以聚焦該按鈕。若要與範例互動，你需要啟用互動式編輯器。

:::interactive_editor

```html
<link href="styles.css" rel="stylesheet">

<button>Example button</button>
```

```css
button:focus {
  outline: 2px solid #005fcc;
}
```

:::

`outline` 屬性用於定義元素周圍的輪廓。此範例將輪廓設定為實心藍色線條，厚度為 2 像素。焦點指示器應以明顯的方式設計，使使用者能清楚知道目前哪個元素具有焦點。為了確保可存取性，指示器與其涵蓋的背景色之間的最低色彩對比度必須至少為 3:1。 

你也應該避免鍵盤陷阱，這種情況發生在使用者無法將焦點從像是模態視窗和快顯視窗等組件中的某個元素移開時。
