# --explanation--

`More robust` significa più forte o più affidabile rispetto a qualcos'altro. Per esempio:

`This software is more robust than the previous version because it can handle larger data sets.` - Qui, `more robust` indica che il software è più forte e più capace.

`When it comes to` introduce un argomento specifico, con il significato di "riguardo a" o "a proposito di". Per esempio:

`When it comes to customer support, this company is very responsive.` - Qui, `when it comes to` introduce l'argomento del supporto clienti.
