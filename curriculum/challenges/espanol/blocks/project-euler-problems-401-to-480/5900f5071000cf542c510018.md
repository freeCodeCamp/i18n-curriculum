# --description--

Sea $C$ el círculo con radio $r$, $x^2 + y^2 = r^2$. Elegimos dos puntos $P(a, b)$ y $Q(-a, c)$ de modo que la línea que pasa por $P$ y $Q$ sea tangente a $C$.

Por ejemplo, el cuádruple $(r, a, b, c) = (2, 6, 2, -7)$ satisface esta propiedad.

Sea $F(R, X)$ el número de cuádruples enteros $(r, a, b, c)$ con esta propiedad, y con $0 < r ≤ R$ y $0 < a ≤ X$.

Podemos verificar que $F(1, 5) = 10$, $F(2, 10) = 52$ y $F(10, 100) = 3384$.

Encuentra $F({10}^8, {10}^9) + F({10}^9, {10}^8)$.
