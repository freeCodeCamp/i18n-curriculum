# --description--

配列の要素にアクセスするには、インデックス番号を使います。インデックス番号は0から始まるため、配列の最初の要素はインデックス0、2番目の要素はインデックス1、というように続きます。

例えば、`fruits` 配列の最初の要素にアクセスするには、次のコードを使います。

```javascript
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits[0]); // Output: apple
console.log(fruits[2]); // Output: orange
```

もしインデックスが範囲外の場合、JavaScriptは `undefined` を返します。つまり、上の例で `fruits[3]` にアクセスしようとすると、`undefined` が返されます。
