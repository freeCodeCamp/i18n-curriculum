# --description--

ピーターは4面体（ピラミッド型）のサイコロを9個持っており、それぞれの面には1、2、3、4の数字が書かれています。

コリンは6面体（立方体）のサイコロを6個持っており、それぞれの面には1、2、3、4、5、6の数字が書かれています。

ピーターとコリンはサイコロを振り、その合計を比べます。合計が高い方が勝ちです。合計が同じ場合は引き分けとなります。

ピラミッド型のピーターが立方体のコリンに勝つ確率はどれくらいでしょうか？答えは小数点以下7桁までで、0.abcdefg の形式で答えてください。
