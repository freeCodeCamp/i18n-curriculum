# --interactive--

이전 수업들에서 `grid-template-columns` 속성을 사용해 몇 개의 분수 단위 값을 설정하는 작업을 했습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

다음 코드는 완전히 유효하지만, 트랙 목록의 일부 또는 전체를 반복하는 더 쉬운 방법이 있습니다.

`repeat()` 함수는 열이나 행의 트랙 일부 또는 전체를 반복하는 데 사용됩니다. 이 함수는 반복 횟수와 반복하려는 트랙을 인수로 받습니다.

다음은 repeat 함수를 사용한 이전 예제의 수정된 버전입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

브라우저에서 스타일에는 변화가 없지만, 열에 반복 값을 더 간결하게 작성하는 방법입니다.

`repeat()` 함수는 행이나 열에 사용할 수 있는 모든 유효한 패턴을 허용합니다.

다음은 `repeat()` 함수를 사용해 첫 번째와 세 번째 열을 20픽셀로, 두 번째와 네 번째 열을 1분수 단위로 설정한 예제입니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(2, 20px 1fr);
  column-gap: 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

때로는 `repeat()` 함수를 사용하지 않고 각 값을 직접 작성하는 것을 선택할 수도 있습니다. 하지만 특정 패턴을 트랙 목록에 반복하려 할 때 이 함수가 특히 유용합니다.
