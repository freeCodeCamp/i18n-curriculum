# --description--

A frase `to keep in mind` significa lembrar ou considerar algo. É frequentemente usado quando alguém dá um conselho ou uma informação importante que deve ser lembrada para uso posterior.

Por exemplo, se alguém disser, "Lembre-se de verificar o tempo antes de fazer trilha." Você pode responder com `I'll keep that in mind` querendo dizer que você vai lembrar de considerar esse conselho.
