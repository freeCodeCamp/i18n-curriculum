# --description--

Internes CSS (oder eingebettetes CSS) bedeutet, dass das CSS direkt in der HTML-Datei selbst hinzugefügt wird, anstatt eine komplett separate Datei zu erstellen. Bei der internen Methode platzieren Sie alle Regeln innerhalb eines Paares von öffnenden und schließenden `<style>`-Tags, die dann innerhalb der öffnenden und schließenden `<head>`-Tags Ihrer HTML-Datei eingefügt werden. Da die Styles direkt innerhalb der `<head>`-Tags platziert werden, benötigen Sie kein `<link>`-Element, das die externe Methode erfordert.

Abgesehen von diesen Unterschieden ist die Syntax genau dieselbe wie bei der externen Methode (Selektor, geschweifte Klammern, Deklarationen):

```html
<head>
  <style>
    div {
      color: white;
      background-color: black;
    }

    p {
      color: red;
    }
  </style>
</head>
<body>...</body>
```

Diese Methode kann nützlich sein, um einer einzelnen Seite einer Website einzigartige Styles hinzuzufügen, aber sie hält die Dinge nicht so getrennt wie die externe Methode, und je nachdem, wie viele Regeln und Deklarationen vorhanden sind, kann die HTML-Datei ziemlich groß werden.

# --questions--    

## --text--

Welcher der folgenden Punkte ist ein Unterschied zwischen internen und externen CSS-Methoden?

## --answers--

Die externe Methode platziert CSS-Regeln in einer separaten Datei, während die interne Methode CSS-Regeln direkt in der HTML-Datei platziert.

---

Die interne Methode hält CSS vom HTML getrennt, während die externe Methode CSS direkt in HTML einbettet.

---

Die interne Methode verwendet das `<link>`-Element, um CSS mit HTML zu verknüpfen, während die externe Methode CSS direkt in HTML einbettet.
