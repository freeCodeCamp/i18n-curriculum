# --interactive--

이 요소들은 표현 목적과 의미론적 HTML 개념과 매우 밀접하게 관련되어 있습니다. 관용구 텍스트 요소인 `i`는 원래 텍스트를 이탤릭체로 표시하는 표현 목적에 사용되었습니다. 하지만 이제는 대체 음성이나 분위기, 다른 언어의 관용구, 기술 용어, 생각을 강조하는 데 자주 사용됩니다.

다음은 공식 HTML 명세에서 `i` 요소를 사용해 프랑스어 관용구를 보여주는 예입니다.

:::interactive_editor

```html
<p>There is a certain <i lang="fr">je ne sais quoi</i> in the air.</p>
```

:::

열린 `lang` 태그 안의 `<i>` 속성은 콘텐츠의 언어를 지정하는 데 사용됩니다. 이 경우 언어는 프랑스어입니다. `i` 요소는 텍스트가 중요한지 여부를 나타내지 않고, 단지 주변 텍스트와 다르다는 것만 보여줍니다.

텍스트의 중요성을 강조해야 한다면, 비슷한 의미론적 요소인 강조 요소 `em`를 사용할 수 있습니다. 이는 보통 더 많은 문맥을 제공해야 할 때 권장됩니다. 주변 텍스트와 비교해 특별한 강조가 필요한 부분에 이 요소를 사용하세요. 문장의 의미를 바꿀 수 있기 때문에 보통 몇 단어로 제한됩니다.

다음은 단락 내에서 강조 요소를 사용한 예입니다.

:::interactive_editor

```html
<p>
  Never give up on <em>your</em> dreams.
</p>
```

:::

문장 `Never give up on your dreams`를 볼 수 있습니다. `your` 단어가 이 요소 안에 있으므로 강조됩니다. 브라우저에서는 이 단어 `your`가 이탤릭체로 표시되어 독자에게 문장에서 중요한 단어임을 알려줍니다.

관용구 텍스트 요소 안에 있을 때와 시각적으로는 같아 보여도, 의미론적 강조 요소는 그 의미와 중요성을 내부적으로 전달합니다.

이 요소들은 표현 목적만으로 사용해서는 안 된다는 점을 아는 것이 중요합니다. 텍스트를 이탤릭체로 표시해야 하지만 단락에서 특별한 목적, 스타일, 의미가 없다면 CSS를 사용하세요.
