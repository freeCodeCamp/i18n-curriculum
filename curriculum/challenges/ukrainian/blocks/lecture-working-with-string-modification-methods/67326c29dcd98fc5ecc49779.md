# --interactive--

У JavaScript існує багато ситуацій, коли потрібно замінити частину рядка на інший рядок.

Наприклад, вам може знадобитися оновити інформацію користувача в URL, змінити форматування дат або виправити помилки у вмісті, створеному користувачем.

Метод `replace()` у JavaScript дозволяє знайти задане значення (наприклад, слово або символ) у рядку та замінити його на інше значення. Метод повертає новий рядок із заміною і залишає оригінал без змін, оскільки рядки в JavaScript є незмінними.

Ось базовий синтаксис:

```js
string.replace(searchValue, newValue);
```

`searchValue` — це значення, яке ви хочете знайти у рядку. Воно може бути рядком або регулярним виразом (регулярним виразом), який описує шаблони в тексті. Це дає змогу гнучко і потужно шукати та обробляти рядки. Ви дізнаєтеся більше про регулярні вирази в майбутніх уроках.

`newValue` — це значення, яким буде замінено `searchValue`. Ось простий приклад:

:::interactive_editor

```js
let text = "I love JavaScript!";
console.log(text); // "I love JavaScript!"
let newText = text.replace("JavaScript", "coding");
console.log(newText);  // "I love coding!"
```

:::

У цьому прикладі слово `JavaScript` знайдено в рядку і замінено на `coding`.

Метод `replace()` чутливий до регістру, тобто він знайде лише точні збіги `searchValue`. Наприклад:

:::interactive_editor

```js
let sentence = "I enjoy working with JavaScript.";
console.log(sentence);  // "I enjoy working with JavaScript."
let updatedSentence = sentence.replace("javascript", "coding");
console.log(updatedSentence);  // "I enjoy working with JavaScript."
```

:::

Тут, оскільки `javascript` (з нижнім регістром `j`) не збігається з `JavaScript` (з верхнім регістром `J`), заміна не відбувається.

За замовчуванням метод `replace()` замінює лише перше входження `searchValue`. Якщо значення зустрічається кілька разів у рядку, заміниться лише перше:

:::interactive_editor

```js
let phrase = "Hello, world! Welcome to the world of coding.";
console.log(phrase);  // "Hello, world! Welcome to the world of coding."
let updatedPhrase = phrase.replace("world", "universe");
console.log(updatedPhrase);  // "Hello, universe! Welcome to the world of coding."
```

:::

Зверніть увагу, що замінено лише перше входження `world` на `universe`.

Метод `replace()` у JavaScript — це потужний і гнучкий інструмент для роботи з рядками.

Він дозволяє замінювати конкретні частини рядка, незалежно від того, чи це окремі символи, слова чи складні шаблони за допомогою регулярних виразів.

Хоча він ідеально підходить для простих замін, розуміння його чутливості до регістру та поведінки за замовчуванням (наприклад, заміна лише першого входження) допоможе вам використовувати його ефективніше.
