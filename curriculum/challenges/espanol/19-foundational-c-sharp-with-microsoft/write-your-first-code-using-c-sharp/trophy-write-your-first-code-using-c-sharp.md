Ahora que ha completado todos los desafíos "Escriba su primer código usando C#", debería haber ganado el trofeo en la plataforma Microsoft Learn. Si no ves este trofeo cuando ves tu <a href="https://learn.microsoft.com/users/me/achievements#trophies-section" target="_blank" rel="noreferrer">página de logros</a>, no has completado todas las tareas y tu trofeo no se puede verificar.
