# --description--

有时我们可能会遇到相互冲突的规则，导致一些意想不到的结果。“但我想让这些段落变成蓝色，为什么它们像其他段落那样是红色的？！”虽然这令人沮丧，但理解 CSS 并不会违背你的意愿去做事情非常重要。CSS 只会执行我们告诉它做的事情。唯一的例外是浏览器提供的默认样式。这些默认样式因浏览器而异，这也是为什么某些元素会在自己和其他元素之间产生较大的“间隙”，或者按钮看起来是那样，尽管我们没有编写任何 CSS 规则来为它们设置样式。

所以如果你遇到像这样的一些意外行为，要么是因为这些默认样式，要么是不理解某个属性的工作原理，或者是不理解这个叫做层叠的小东西。

层叠决定了哪些规则实际上会应用到你的 `HTML`。层叠使用不同的因素来确定这一点。我们将研究其中的三个因素，希望能帮助你避免那些令人沮丧的“我讨厌 CSS”的时刻。
