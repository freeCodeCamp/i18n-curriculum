# --explanation--

`tweak`는 약간 조정하거나 수정하는 것을 의미합니다. 예를 들어:

`The developer tweaked the code to improve the app's performance.` - 개발자가 앱의 기능을 향상시키기 위해 코드를 약간 변경했다는 뜻입니다.

`Layouts`는 페이지나 인터페이스에서 요소들의 배치나 구조를 말합니다. 예를 들어:

`The layout of the homepage was changed to make it more user-friendly.` - 홈페이지의 요소 배치를 사용자가 더 쉽게 탐색할 수 있도록 조정했다는 뜻입니다.

`Guidelines`는 방향을 제시하는 규칙이나 원칙입니다. 예를 들어:

`The team followed the design guidelines to maintain consistency across pages.` - 팀이 여러 페이지에서 일관된 외관과 구조를 유지하기 위해 특정 규칙을 따랐다는 뜻입니다.
