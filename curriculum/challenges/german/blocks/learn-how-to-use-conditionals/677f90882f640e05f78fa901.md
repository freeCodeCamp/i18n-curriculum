# --explanation--

Das `Present Continuous`-Tempus betont oft einen vorübergehenden Zustand oder ein Problem, das im Moment des Sprechens auftritt. Zum Beispiel:

- `The server isn't running diagnostics.` – Hebt hervor, dass der Diagnoseprozess gerade nicht stattfindet.

- `Developers aren't debugging the application.` – Zeigt, dass die Entwickler gerade nicht an dieser Tätigkeit arbeiten.

`Properly` bedeutet, etwas auf die richtige oder erwartete Weise zu tun.

In diesem Satz bezieht sich `aren't booting properly` darauf, dass die Computer momentan nicht richtig starten.
