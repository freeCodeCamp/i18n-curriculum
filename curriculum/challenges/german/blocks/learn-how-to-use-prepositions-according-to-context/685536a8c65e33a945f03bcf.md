# --explanation--

`Currently` bedeutet „zurzeit“ oder „jetzt“. Zum Beispiel:  

`We are currently working on a new app.` – Das bedeutet, die Arbeit findet jetzt statt.

`Three designers` ist eine Möglichkeit, die Anzahl der Personen in einer Design-Rolle anzugeben. Zum Beispiel:

`The company employs three designers for this product.` – Das bedeutet, dass drei Personen für Design-Aufgaben verantwortlich sind.
