## HTML-Formularelemente und Attribute

- **`form`-Element**: wird verwendet, um ein HTML-Formular für Benutzereingaben zu erstellen.
- **`action`-Attribut**: wird verwendet, um die URL anzugeben, an die die Formulardaten gesendet werden sollen.
- **`method`-Attribut**: wird verwendet, um die HTTP-Methode anzugeben, die beim Senden der Formulardaten verwendet werden soll. Die gebräuchlichsten Methoden sind `GET` und `POST`.

```html
<form method="value-goes-here" action="url-goes-here">
  <!-- inputs go inside here -->
</form>
```

- **`input`-Element**: wird verwendet, um ein Eingabefeld für Benutzereingaben zu erstellen.
- **`type`-Attribut**: wird verwendet, um den Typ des Eingabefelds anzugeben. Zum Beispiel `text`, `email`, `number`, `radio`, `checkbox` usw.
- **`placeholder`-Attribut**: wird verwendet, um dem Benutzer einen Hinweis anzuzeigen, was er in das Eingabefeld eingeben soll.
- **`value`-Attribut**: wird verwendet, um den Wert der Eingabe anzugeben. Wenn die Eingabe einen `button`-Typ hat, kann das `value`-Attribut verwendet werden, um den Text der Schaltfläche festzulegen.
- **`name`-Attribut**: wird verwendet, um einem Eingabefeld einen Namen zuzuweisen, der als Schlüssel dient, wenn Formulardaten abgesendet werden. Bei Optionsschaltflächen führt die Vergabe desselben `name` dazu, dass sie gruppiert werden, sodass jeweils nur eine Option in der Gruppe ausgewählt werden kann.
- **`size`-Attribut**: wird verwendet, um die Anzahl der Zeichen zu definieren, die sichtbar sein sollen, während der Benutzer in das Eingabefeld tippt.
- **`min`-Attribut**: kann mit Eingabetypen wie `number` verwendet werden, um den minimal zulässigen Wert im Eingabefeld anzugeben.
- **`max`-Attribut**: kann mit Eingabetypen wie `number` verwendet werden, um den maximal zulässigen Wert im Eingabefeld anzugeben.
- **`minlength`-Attribut**: wird verwendet, um die minimale Anzahl von Zeichen anzugeben, die in einem Eingabefeld erforderlich sind.
- **`maxlength`-Attribut**: wird verwendet, um die maximale Anzahl von Zeichen anzugeben, die in einem Eingabefeld erlaubt sind.
- **`required`-Attribut**: wird verwendet, um anzugeben, dass ein Eingabefeld ausgefüllt werden muss, bevor das Formular abgesendet wird.
- **`disabled`-Attribut**: wird verwendet, um anzugeben, dass ein Eingabefeld deaktiviert sein soll.
- **`readonly`-Attribut**: wird verwendet, um anzugeben, dass ein Eingabefeld nur lesbar ist.

:::interactive_editor

```html
<!-- Text input -->
<input 
  type="text"
  id="name"
  name="name"
  placeholder="e.g. Quincy Larson" 
  size="20"
  minlength="5"
  maxlength="30"
  required
/>

<!-- Number input -->
<input 
  type="number"
  id="quantity"
  name="quantity"
  min="2"
  max="10"
  disabled
/>

<!-- Button -->
<input type="button" value="Show Alert" />
```

:::

- **`label`-Element**: wird verwendet, um eine Bezeichnung für ein Eingabefeld zu erstellen.
- **`for`-Attribut**: wird verwendet, um anzugeben, für welches Eingabefeld die Bezeichnung gilt.
- **Implizite Formularzuordnung**: Eingaben können mit Bezeichnungen verknüpft werden, indem das Eingabefeld innerhalb des `label`-Elements eingeschlossen wird.

:::interactive_editor

```html
<form action="">
  <label>
    Full Name:
    <input type="text" />
  </label>
</form>
```

:::

- **Explizite Formularzuordnung**: Eingaben können mit Bezeichnungen verknüpft werden, indem das `for`-Attribut am `label`-Element verwendet wird.

:::interactive_editor

```html
<form action="">
  <label for="email">Email Address: </label>
  <input type="email" id="email" />
</form>
```

:::

- **`button`-Element**: wird verwendet, um eine anklickbare Schaltfläche zu erstellen. Eine Schaltfläche kann auch ein `type`-Attribut haben, das verwendet wird, um das Verhalten der Schaltfläche bei Aktivierung zu steuern. Zum Beispiel `submit`, `reset`, `button`.

:::interactive_editor

```html
<button type="button">Show Form</button>
<button type="submit">Submit Form</button>
<button type="reset">Reset Form</button>
```

:::

- **`fieldset`-Element**: wird verwendet, um zusammengehörige Eingaben zu gruppieren.
- **`legend`-Element**: wird verwendet, um eine Bildunterschrift hinzuzufügen, die die Gruppe von Eingaben beschreibt.

:::interactive_editor

```html
<!-- Radio group -->
<fieldset>
  <legend>Was this your first time at our hotel?</legend>

  <label for="yes-option">Yes</label>
  <input id="yes-option" type="radio" name="hotel-stay" value="yes" />

  <label for="no-option">No</label>
  <input id="no-option" type="radio" name="hotel-stay" value="no" />
</fieldset>

<!-- Checkbox group -->
<fieldset>
  <legend>
    Why did you choose to stay at our hotel? (Check all that apply)
  </legend>

  <label for="location">Location</label>
  <input type="checkbox" id="location" name="location" value="location" />

  <label for="price">Price</label>
  <input type="checkbox" id="price" name="price" value="price" />
</fieldset>
```

:::

- **Fokussierter Zustand**: dies ist der Zustand eines Eingabefelds, wenn es vom Benutzer ausgewählt ist.

## Arbeiten mit HTML-Tabellenelementen und Attributen

- **Tabellen-Element**: wird verwendet, um eine HTML-Tabelle zu erstellen.
- **Tabellenkopf (`thead`)-Element**: wird verwendet, um den Kopfbereich in einer HTML-Tabelle zu gruppieren.
- **Tabellenzeile (`tr`)-Element**: wird verwendet, um eine Zeile in einer HTML-Tabelle zu erstellen.
- **Tabellenkopfzelle (`th`)-Element**: wird verwendet, um eine Kopfzelle in einer HTML-Tabelle zu erstellen.
- **Tabellenkörper (`tbody`)-Element**: wird verwendet, um den Körperbereich in einer HTML-Tabelle zu gruppieren.
- **Tabellendatenzelle (`td`)-Element**: wird verwendet, um eine Datenzelle in einer HTML-Tabelle zu erstellen.
- **Tabellenfuß (`tfoot`)-Element**: wird verwendet, um den Fußbereich in einer HTML-Tabelle zu gruppieren.
- **`caption`-Element**: wird verwendet, um einen Titel für eine HTML-Tabelle hinzuzufügen.
- **`colspan`-Attribut**: wird verwendet, um die Anzahl der Spalten anzugeben, die eine Tabellenzelle überspannen soll.

:::interactive_editor

```html
<table>
  <caption>Exam Grades</caption>

  <thead>
    <tr>
      <th>Last Name</th>
      <th>First Name</th>
      <th>Grade</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>Davis</td>
      <td>Alex</td>
      <td>54</td>
    </tr>

    <tr>
      <td>Doe</td>
      <td>Samantha</td>
      <td>92</td>
    </tr>

    <tr>
      <td>Rodriguez</td>
      <td>Marcus</td>
      <td>88</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <td colspan="2">Average Grade</td>
      <td>78</td>
    </tr>
  </tfoot>
</table>
```

:::

## Arbeiten mit HTML-Werkzeugen

- **HTML-Validator**: ein Werkzeug, das die Syntax von HTML-Code überprüft, um sicherzustellen, dass sie gültig ist.
- **DOM-Inspektor**: ein Werkzeug, mit dem Sie die HTML-Struktur einer Webseite inspizieren und ändern können.
- **Entwicklerwerkzeuge**: eine Sammlung von Webentwicklerwerkzeugen, die direkt im Browser integriert sind und Ihnen helfen, Webseiten zu debuggen, zu profilieren und zu analysieren.
