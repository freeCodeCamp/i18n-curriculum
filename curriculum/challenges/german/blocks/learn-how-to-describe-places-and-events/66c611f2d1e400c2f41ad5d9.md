# --explanation--

Im `Present Simple` Tempus wird die Verneinung mit `doesn't` oder `isn't` für Subjekte der dritten Person Singular (`he, she, it`) gebildet.

- Verwenden Sie `doesn't` mit Tätigkeitsverben, um zu zeigen, dass etwas nicht passiert, wie `It doesn't work.`

- Verwenden Sie `isn't` mit Formen von `to be`, um zu zeigen, dass etwas nicht wahr ist, wie `isn't set up.`

In diesem Satz benötigen Sie sowohl `doesn't` als auch `isn't`, um korrekt zu beschreiben, warum die Funktion nicht wie erwartet funktioniert.
