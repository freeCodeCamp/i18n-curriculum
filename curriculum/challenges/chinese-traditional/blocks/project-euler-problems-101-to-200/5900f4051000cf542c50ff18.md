# --description--

眾所周知，方程式 $x^2 = -1$ 對實數 $x$ 沒有解。

然而，如果我們介紹虛數 $i$，這個方程式有兩個解：$x = i$ 和 $x = -i$。

如果我們更進一步，方程式 ${(x - 3)}^2 = -4$ 有兩個複數解：$x = 3 + 2i$ 和 $x = 3 - 2i$，它們彼此稱為複共軛。

形如 $a + bi$ 的數字稱為複數。

一般來說，$a + bi$ 和 $a − bi$ 是彼此的共軛複數。高斯整數是指複數 $a + bi$，其中 $a$ 和 $b$ 都是整數。

一般整數也是 Gaussian 整數（其中 $b = 0$）。

為了將它們與 $b ≠ 0$ 的高斯整數區分開來，我們稱這類整數為「有理整數」。

如果結果也是 Gaussian 整數，則 Gaussian 整數稱為有理整數 $n$ 的因數。

如果例如我們將 5 除以 $1 + 2i$，我們可以用以下方式簡化：

將分子和分母同乘以 $1 + 2i$ 的複共軛：$1 − 2i$。

結果是：

$$\frac{5}{1 + 2i} = \frac{5}{1 + 2i} \frac{1 - 2i}{1 - 2i} = \frac{5(1 - 2i)}{1 - {(2i)}^2} = \frac{5(1 - 2i)}{1 - (-4)} = \frac{5(1 - 2i)}{5} = 1 - 2i$$

所以 $1 + 2i$ 是 5 的一個因數。

請注意，$1 + i$ 不是 5 的因數，因為：

$$\frac{5}{1 + i} = \frac{5}{2} - \frac{5}{2}i$$

請注意，如果高斯整數（$a + bi$）是有理整數 $n$ 的因數，則其複共軛（$a − bi$）也是 $n$ 的因數。事實上，5 有六個實部為正的因數：{1、1 + 2i、1 − 2i、2 + i、2 − i、5}。

以下是前 5 個正有理整數的所有除數表：

| n | Gaussian integer divisors with positive real part | Sum s(n) of these divisors |
|---|---------------------------------------------------|----------------------------|
| 1 | 1                                                 | 1                          |
| 2 | 1, 1 + i, 1 - i, 2                                | 5                          |
| 3 | 1, 3                                              | 4                          |
| 4 | 1, 1 + i, 1 - i, 2, 2 + 2i, 2 - 2i, 4             | 13                         |
| 5 | 1, 1 + 2i, 1 - 2i, 2 + i, 2 - i, 5                | 12                         |

對於實部為正的除數，我們有：$\displaystyle\sum_{n=1}^5 s(n) = 35$。

對於 $1 ≤ n ≤ {10}^5$，$\displaystyle\sum_{n = 1}^{{10}^5} s(n) = 17924657155$。

$\displaystyle\sum_{n=1}^{{10}^8} s(n)$ 是什麼？
