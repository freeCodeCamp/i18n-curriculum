### --feedback--

# --explanation--

`Built-in` refere-se a recursos ou ferramentas que estão incluídos como parte de um programa de software ou dispositivo desde o início, sem a necessidade de serem adicionados ou instalados separadamente. 

Quando algo é `built-in`, já está lá e pronto para usar.Por exemplo:

- `This laptop has a built-in camera.` - A câmera está incluída como parte do laptop.

- `The software comes with built-in security features.` - Os recursos de segurança já estão incluídos no software.
