# --interactive--

CSS 中的屬性選擇器是一個強大的工具，允許你根據 HTML 元素的屬性來標的這些元素。這表示你可以將樣式套用到具有特定屬性或甚至該屬性某些特定值的元素。

當你想要動態地為元素設計樣式，或是單靠類別名稱無法提供足夠的特異性時，這特別有用。

例如，你可以使用屬性選擇器來標的所有具有 `href` 屬性的連結。當你想要為頁面上的所有連結套用統一樣式時，這特別方便。

以下是如何使用屬性選擇器來標的具有 `href` 屬性的聯結：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Example link with an href attribute</a>
<a>Example link without an href attribute</a>
```

```css
a {
  display: block;
}

a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

此 CSS 規則將對任何包含 `href` 屬性的聯結套用藍色和底線，確保所有可點擊的聯結樣式一致。

但屬性選擇器可以更具體。假設你只想標的具有 `title` 屬性的聯結。`title` 屬性通常會提供關於聯結的額外資訊，而你可能想要以不同的樣式來顯示這些聯結，以顯示它們有額外的資訊可用。以下是你可以這樣做的方法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

此規則會套用粗體文字並移除任何具有 `title` 屬性的聯結底線。這是視覺上區分這些聯結與頁面上其他聯結的絕佳方式。

你也可以結合屬性選擇器以獲得更精確的結果。<dfn>例如</dfn>，如果你想要為同時具有 `href` 和 `title` 屬性的聯結設定樣式，你可以這樣做：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[href][title] {
  display:block;
  color: green;
}
```

:::

在這種情況下，只有同時包含 `href` 和 `title` 屬性的聯結才會以綠色文字呈現。這種層級的控制正是屬性選擇器在 CSS 中如此強大的原因。

另一個屬性選擇器的範例是匹配屬性中以空格分隔的值列表中的單一值。

這是一個具有多個類別的錨點元素範例：

```html
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

要定位這個特定的錨點元素，你可以使用以下選擇器：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

此處使用 `[attr~=value]` 語法來標的所有其 class 屬性包含字、單字 `"primary"` 的錨點元素。

如果你需要標的元素的屬性值以特定值為前置式，則你可以使用 `[attr^=value]` 語法。以下是一個範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

在此範例中，`a[href^="https://"]` 選擇器將標的所有 `href` 屬性值以 `"https://"` 開頭的錨點元素。

若要標的屬性值以特定值結尾的元素，你可以使用 `[attr$=value]` 語法。以下是一個範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Visit Example Site</a>
```

```css
a[href$=".com"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

在此範例中，`a[href$=".com"]` 選擇器將標的所有 `href` 屬性值以 `.com` 結尾的錨點元素。

使用屬性選擇器不僅能增強你的網頁樣式，還能透過根據屬性使像 `links` 這類互動元素更易於區分，從而提升存取性。
