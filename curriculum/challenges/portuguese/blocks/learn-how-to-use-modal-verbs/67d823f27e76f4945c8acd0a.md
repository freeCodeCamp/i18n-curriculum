# --explanation--

`Should` é um verbo modal que expressa conselho, recomendação ou o melhor curso de ação — mas não uma obrigação.

- `You should update your password regularly.` - É uma boa ideia mudar sua senha com frequência para segurança.

- `You should check for errors in the code before running it.` - É uma boa prática encontrar erros antes de executar o código.

Como Jake diz `we should focus on the critical systems first`, ele está dando uma recomendação forte, mas a equipe não é estritamente obrigada a fazê-lo.
