# --explanation--

Час `Present Simple` часто використовується для опису станів, фактів або умов, які зазвичай є істинними. Цей час використовується, тому що пошкодження BIOS вважається стабільним або незмінним станом, коли комп’ютер намагається завантажитися.

`Is corrupted` означає пошкоджений або змінений таким чином, що він перестає працювати правильно. Наприклад:

`The file is corrupted and won't open.` - Файл пошкоджений і до нього неможливо отримати доступ.
