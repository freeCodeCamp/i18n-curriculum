# --interactive--

Іноді вам потрібно буде додавати або видаляти вузли з DOM, і для цього існує кілька Web API.

У цьому уроці ми розглянемо методи `appendChild()` та `removeChild()`.

Метод `appendChild()` використовується для додавання вузла в кінець списку дочірніх вузлів певного батьківського вузла.

Ось базовий синтаксис для методу `appendChild()`:

```js
parentNode.appendChild(newNode);
```

Давайте розглянемо приклад, щоб краще зрозуміти, як використовувати метод `appendChild()`.

У цьому прикладі у нас є елемент невпорядкованого списку з ідентифікатором `desserts` у HTML. Цей список містить два елементи списку `Cake` та `Pie`:

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
```

Щоб отримати доступ до цього списку у JavaScript-файлі, ми можемо використати метод `getElementById()`:

```js
const dessertsList = document.getElementById("desserts");
```

У нас є змінна `dessertsList`, яка зберігає референс на елемент `ul` з ідентифікатором `desserts`.

Потім потрібно створити новий елемент списку за допомогою методу `createElement()`:

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");
```

Наступний код створить новий елемент списку і збереже його у змінній `listItem`.

Щоб додати цей вузол у кінець списку дочірніх вузлів елемента `ul`, ми можемо використати метод `appendChild()`:

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

dessertsList.appendChild(listItem);
```

Якщо запустити цей код, ви побачите, що новий елемент списку додано в кінець списку дочірніх вузлів елемента `ul`.

:::interactive_editor

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
<script src="index.js"></script>
```

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

dessertsList.appendChild(listItem);
```

:::

Проблема в тому, що новий елемент списку порожній. Щоб додати текстовий вміст до нового елемента списку, можна використати властивість `textContent`:

:::interactive_editor

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
<script src="index.js"></script>
```

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

listItem.textContent = "Cookies";
dessertsList.appendChild(listItem);
```

:::

Тепер список показуватиме `Cake`, `Pie` та `Cookies`.

Щоб видалити вузол з DOM, можна використати метод `removeChild()`.

Ось базовий синтаксис для методу `removeChild()`:

```js
parentNode.removeChild(childNode);
```

Розглянемо приклад, де ми хочемо видалити останній елемент абзацу з цього елемента `section`:

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
```

Ми можемо отримати доступ до елемента `section` у JavaScript-файлі за допомогою методу `getElementById()`:

```js
const sectionEl = document.getElementById("example-section");
```

Коли ми маємо референс на елемент `section`, можемо отримати доступ до останнього абзацу за допомогою методу `querySelector()`:

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");
```

Ми використовуємо псевдоклас `:last-of-type`, щоб вибрати останній елемент абзацу всередині елемента `section`.

Тепер, коли у нас є батьківський і дочірній вузли, ми можемо видалити останній елемент абзацу з елемента `section` за допомогою методу `removeChild()`:

:::interactive_editor

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
<script src="index.js"></script>
```

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");

sectionEl.removeChild(lastParagraph);
```

:::

Отже, коли може знадобитися додавати або видаляти вузли з DOM?

Якщо ви працюєте з динамічним вмістом, можливо, доведеться додавати або видаляти вузли з DOM. Реальні приклади включають кошики для покупок, списки справ і стрічки соціальних мереж.
