# --description--

如果网站只能显示文本，那将非常无聊。幸运的是，超文本标记语言 提供了各种各样的元素来显示各种不同的媒体。其中使用最广泛的是图像元素。

要在超文本标记语言中显示图像，你使用 `<img>` 元素。与之前遇到的其他元素不同，`<img>` 是一个空元素。空的、无内容的超文本标记语言元素不需要闭合标签。

它不是用开始和结束标签包裹内容，而是使用一个 `src` 属性将图像嵌入页面，该属性告诉浏览器图像文件的位置。`src` 属性的工作方式类似于锚点标签的 `href` 属性。它可以使用绝对路径和相对路径嵌入图像。

例如，使用绝对路径你可以显示位于 The Odin Project 网站上的图像：

要使用你自己网站上的图像，你可以使用相对路径。

- 在 `odin-links-and-images` 项目中创建一个名为 `images` 的新目录。

- 接下来，下载<a href="https://unsplash.com/photos/Mv9hjnEUHR4/download?force=true&w=640" target="_blank">这张图像</a>并将其移动到你刚刚创建的 images 目录中。

- 将图像重命名为 `dog.jpg`。

最后将图像添加到 `index.html` 文件中：

```html
<body>
  <h1>Homepage</h1>
	<a href="https://www.theodinproject.com/about">click me</a>

	<a href="pages/about.html">About</a>

	<img src="images/dog.jpg">
</body>
```

储存 `index.html` 文件并在浏览器中打开它，以全貌查看 Charles。


## 父目录

如果你想在 `about` 页面使用狗的图像，你首先必须从 pages 目录向上一级进入其父目录，然后才能访问 images 目录。

要进入上级目录，你需要在相对文件路径中使用两个点，如：`../.`。让我们来看一个实际操作，在 `about.html` 文件的 `body` 中，在你之前添加的标题下方添加以下图像：

```html
<img src="../images/dog.jpg">
```

分解如下：

- 首先，你将进入 `odin-links-and-images` 的 pages 目录的上级目录。

- 然后，从父目录进入 `images` 目录。

- 最后，你可以访问 `dog.jpg` 文件。

使用我们之前使用的比喻，在文件路径中使用 `../` 有点像从你当前所在的房间走到主走廊，这样你就可以去另一个房间。

## Alt 属性

除了 `src` 属性，每个图像元素还应具有 `alt`（备选文本）属性。

`alt` 属性用于描述图像。如果图像无法装载，它将作为图像的替代。它还与屏幕阅读器一起使用，以向视障用户描述图像内容。
