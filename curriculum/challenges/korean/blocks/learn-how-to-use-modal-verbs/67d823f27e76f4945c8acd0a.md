# --explanation--

`Should`는 조언, 권고, 최선의 행동 방침을 나타내는 조동사로, 의무를 표현하지 않습니다.

- `You should update your password regularly.` - 보안을 위해 비밀번호를 자주 변경하는 것이 좋습니다.

- `You should check for errors in the code before running it.` - 코드를 실행하기 전에 실수를 찾는 것이 좋은 습관입니다.

Jake가 `we should focus on the critical systems first`라고 말했으므로 강한 권고를 하고 있지만, 팀이 반드시 그렇게 해야 하는 것은 아닙니다.
