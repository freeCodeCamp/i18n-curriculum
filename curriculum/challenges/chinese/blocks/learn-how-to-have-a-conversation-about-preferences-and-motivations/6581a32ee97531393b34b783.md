# --description--

`Encourage` 意味着给予某人支持或信心去做某事。例如，当你的朋友说 `You can solve this coding problem!` 时，他们是在鼓励你。 

`Discourage` 是相反的意思；它指使某人感觉不那么自信或不太可能去做某事。如果那个朋友说，`That programming language is too hard for beginners`，这可能会让你对学习它感到气馁。

`Discouraging` 描述让你感到气馁的事物。比如，`Finding a bug in your code after hours of work can be discouraging.`

`Encouraging` 是指某事给你勇气或希望。例如，`Getting your first program to run is very encouraging.`
