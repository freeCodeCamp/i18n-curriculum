# --explanation--

在短语 `limit what other parts can see` 中，`what` 引导一个名词性从句，该从句作为 `limit` 的对象。

名词性从句是在句子中充当名词的从句。它可以作为句子的宾语或主语。它通常以 `that`、`what`、`whether`、`who`、`when`、`why`、`how` 等词开头。

名词从句使用正常的语句词序（不是疑问词序）。例如：

- 团队正在查看系统在更新期间崩溃的原因。 - `Why the system crashed during the update` 是作为 `checking` 的宾语的名词从句。
   
- `What the function returns` 取决于 `input data`。- `What the function returns` 是作为 `depends on` 主语的名词性从句。
