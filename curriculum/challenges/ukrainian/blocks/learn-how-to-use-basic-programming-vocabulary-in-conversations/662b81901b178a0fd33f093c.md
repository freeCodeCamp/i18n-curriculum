# --questions--

## --text--

Як оголосити змінну в Python за словами Софі?

## --answers--

Ви оголошуєте змінну, назвавши її на честь оператора.

### --feedback--

Цей варіант неправильний, бо змінна не називається на честь оператора, а розташовується перед оператором.

---

Змінні оголошуються за допомогою команди `declare`.

### --feedback--

Python не потребує спеціальної команди на кшталт `declare` для створення змінних; він використовує присвоєння безпосередньо.

---

Ви використовуєте ім’я змінної, за яким іде оператор присвоєння та значення.

---

Змінні потрібно оголошувати, набравши `variable` перед ім’ям.

### --feedback--

Python не використовує ключових слів на кшталт `variable` для оголошення змінних. Просто використовується ім’я змінної, оператор присвоєння та значення.
