# --description--

Bis jetzt hatten Sie nur begrenzte Kontrolle über die Darstellung und das Aussehen Ihres Inhalts. Um das zu ändern, fügen Sie ein `style`-Element innerhalb des `head`-Elements hinzu.

# --hints--

Ihr Code sollte einen öffnenden `<style>`-Tag enthalten.

```js
assert.match(code, /<style\s*>/i);
```

Ihr Code sollte einen schließenden `</style>`-Tag enthalten.

```js
assert.match(code, /<\/style\s*>/);
```

Ihr `style`-Element sollte in Ihrem `head`-Element verschachtelt sein.

```js
assert.match(code, /<head\s*>[\w\W\s]*<style\s*>[\w\W\s]*<\/style\s*>[\w\W\s]*<\/head\s*>/i)
```
