# --explanation--

No tempo `Present Perfect Continuous`, `since` é usado para indicar quando uma ação começou e continua até o presente. Por exemplo:

`I've been learning JavaScript since 2020.` - Isso mostra que a ação de aprender JavaScript começou em 2020 e ainda está em andamento.

O termo `software update` refere-se a mudanças ou melhorias feitas em um programa de computador. Por exemplo:

`The software update fixed several bugs and added new features.` - Isso significa a versão mais recente do programa que inclui correções e melhorias. Indica que as melhorias fazem parte do desenvolvimento contínuo do software.
