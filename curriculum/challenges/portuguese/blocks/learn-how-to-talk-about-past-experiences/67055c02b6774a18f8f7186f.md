# --explanation--

`Credentials` refere-se a informações como nomes de usuário, senhas ou chaves que permitem que um usuário ou sistema acesse outro sistema ou recurso.Por exemplo:

- `Please enter your credentials to log in to the server.` - significa que você deve inserir suas informações para acessar o servidor.

- `The credentials were incorrect, so the login failed.` - significa que a informação que você inseriu é diferente da que o sistema possui em seu banco de dados.

Um `patch` é uma atualização ou correção aplicada a software ou sistemas para corrigir problemas, melhorar a funcionalidade ou aumentar a segurança.Por exemplo:

- Os desenvolvedores lançaram um patch para corrigir o bug.

- `Make sure to apply the latest security patch to protect your system.` - significa que a última atualização de segurança poderá proteger melhor o seu sistema.
