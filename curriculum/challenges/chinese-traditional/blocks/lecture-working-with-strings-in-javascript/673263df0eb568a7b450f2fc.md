# --interactive--

在使用 JavaScript 處理字串時，有時你需要包含 JavaScript 引擎可能會誤解的特殊字元。

兩個常見的任務包括在字串中建立換行以及跳脫某些字元（例如引號），以確保它們正確顯示。

在許多程式語言中，包括 JavaScript，你可以使用稱為跳脫序列的特殊字元在字串中創建換行。最常見的換行跳脫序列是 `\n`。

例如，如果你想將字串分成多行，你會在你想要新行開始的地方使用 `\n`：

:::interactive_editor

```js
let poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

`\n` 跳脫序列告訴 JavaScript 在該處插入換行，導致字串顯示為多行。

在處理字串時，另一個重要的概念是跳脫字元。有時候，你需要在字串中包含 JavaScript 通常用於其他用途的字元，例如引號。

如果你在字串中直接使用引號而不進行跳脫，可能會導致錯誤，因為 JavaScript 會認為你試圖結束字串。

例如，這將導致錯誤：

```js
let statement = "She said, "Hello!"";
```

JavaScript 會混淆，因為它認為字串在字 `"said,"` 後就結束了，但你想要將 `"Hello!"` 兩邊的引號當作字串的一部分。

為了解決這個問題，你可以在內部引號前加上反斜線（`\`）來跳脫它們：

:::interactive_editor

```js
let statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

反斜線告訴 JavaScript 將引號視為字元，因此它們會正確顯示在輸出中。

你也可以跳脫其他特殊字元，例如反斜線本身（`\\`），或是在以單引號包圍的字串中跳脫單引號（`\'`）。

這裡是另一個使用單引號的範例：

:::interactive_editor

```js
let quote = 'It\'s a beautiful day!';
console.log(quote); // It's a beautiful day!
```

:::

透過使用 `\'` 來跳脫單引號，JavaScript 就知道要將它包含為字串的一部分，而不是提前結束字串。

當你格式化輸出或處理字串中的特殊字元時，跳脫和建立換行是必不可少的。這些技巧有助於你避免錯誤並確保你的文字精確呈現。
