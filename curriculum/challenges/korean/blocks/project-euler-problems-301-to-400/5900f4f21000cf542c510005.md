# --description--

변의 길이가 $\sqrt{5}$, $\sqrt{65}$, $\sqrt{68}$인 삼각형을 생각하세요. 이 삼각형의 영역이 9임을 증명할 수 있습니다.

$S(n)$은 변의 길이가 $\sqrt{1 + b^2}$, $\sqrt{1 + c^2}$, $\sqrt{b^2 + c^2}$인 모든 삼각형(양의 정수 $b$, $c$에 대해) 중 영역이 정수이고 $n$을 넘지 않는 삼각형들의 영역 합입니다.

예시 삼각형은 $b = 2$이고 $c = 8$입니다.

$S({10}^6) = 18\\,018\\,206$입니다.

$S({10}^{10})$을 구하세요.
