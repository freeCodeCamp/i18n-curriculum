## 色彩理論

- **色彩理論定義**：這是研究顏色如何相互作用以及如何影響我們感知的學問。它涵蓋色彩關係、色彩和諧以及色彩的心理影響。
- **原色**：這些顏色是黃色、藍色和紅色，是所有其他顏色的基本色調來源。
- **次級色**：這些顏色是由兩種原色以相等比例混合而成。綠色、橙色和紫色是次級色的例子。
- **三級色**：這些顏色是由一種原色與相鄰的次級色結合而成。黃綠色、藍綠色和藍紫色是三級色的例子。
- **暖色系**：這些顏色包括紅色、橙色和黃色，能喚起舒適、溫暖和溫馨的感覺。
- **冷色系**：這些顏色包括藍色、綠色和紫色，能喚起冷靜、寧靜和專業的感覺。
- **色輪**：色輪是一個圓形圖表，展示顏色之間的關係。它是設計師的重要工具，幫助他們選擇顏色組合。
- **類似色配色方案**：這些配色方案創造出和諧且舒緩的體驗。它們包含在色輪上彼此相鄰的類似色。
- **互補色配色方案**：這些配色方案創造出高對比和視覺衝擊。它們的顏色位於色輪的相對兩端。
- **三色配色方案**：這種配色方案色彩鮮明，由色輪上大約等距的三種顏色組成。若將它們連接，會形成一個等邊三角形。
- **單色配色方案**：這種配色方案的所有顏色都源自同一基色，通過調整其明度、暗度和飽和度來產生。這帶來統一與和諧的感覺，同時仍能創造對比。

## 在 CSS 中使用顏色的不同方式

- **命名顏色**：這些顏色是瀏覽器預定義的顏色名稱。例子包括 `blue`、`darkred`、`lightgreen`。
- **`rgb()` 函式**：RGB 代表紅、綠、藍——光的三原色。這三種顏色以不同強度組合，創造出廣泛的顏色。`rgb()` 函式允許你使用 RGB 色彩模型定義顏色。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>RGB Color</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

- **`rgba()` 函式（程式）**：此函式（程式）會添加第四個值 alpha，用來控制顏色的透明度。如果未提供，alpha 值預設為 `1`。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>RGBA Background</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

- **`hsl()` 函式**：HSL 代表色相、飽和度和明度——定義顏色的三個關鍵組件。現代的 `hsl()` 語法也支援使用 `/` 分隔符的選用 alpha 值。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>HSL Color</p>
<p class="alpha">HSL Color with Alpha</p>
```

```css
p {
  color: hsl(120 100% 50%);
}

.alpha {
  color: hsl(120 100% 50% / 0.8);
}
```

:::

- **`hsla()` 函式**：`hsla()` 函式是為 HSL 顏色添加透明度的舊式方法。現代 CSS 偏好使用帶有 `/` 分隔符的 `hsl()` 來表示 alpha 值。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>HSLA Background</div>
```


```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

- **十六進位**：十六進位碼（hex code）用於表示 RGB 色彩模型中的顏色。"hex" 指的是使用數字 `0` 到 `9` 和字母 `A` 到 `F` 的 16 進位制系統。十六進位碼可以是 6 個字元長，例如 `#F498A2`，也可以是 3 個字元的簡寫，當完整形式中每對字元都是相同字元時可用。例如，`#2FB` 等價於 `#22FFBB`。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1 class="hex-text">Hex Text</h1>
<p class="hex-bg">Hex Background</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}

p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

## `box-shadow` 屬性

- **定義**：`box-shadow` 屬性會在元素周圍套用一個或多個陰影。
- **位移值**：你必須指定水平 (`offset-x`) 和垂直 (`offset-y`) 值。正的 `offset-x` 會將陰影向右移動，負值則向左移動。正的 `offset-y` 會將陰影向下移動，負值則向上移動。如果值為 `0`，則不需要包含單位。
- **模糊半徑**：此選用值控制陰影的模糊程度。如果未包含，預設為 `0`，會產生銳利的邊緣。數值越高，陰影越柔和。
- **Spread Radius**：這個選用值控制陰影擴展或縮小的程度。如果未包含，則預設為 `0`。
- **陰影顏色**：你可以使用命名顏色、十六進位值、`rgb()`、`rgba()`、`hsl()` 或 `hsla()` 函式指定顏色。
- **`inset` 關鍵字**：添加 `inset` 關鍵字會將陰影放置在元素內部，而非外部。
- **套用多重盒子陰影**：你可以用逗號分隔來套用多個陰影。陰影會從前到後分層。

### 語法

```css
box-shadow: offset-x offset-y blur-radius spread-radius color;
```

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="shadow-box">Shadow Color Example</div>
```

```css
.shadow-box {
  width: 200px;
  padding: 20px;
  background-color: lightblue;
  box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}
```

:::

## 線性與放射漸層

- **線性漸層**：這些漸層會沿著直線在顏色之間產生漸進混合。你可以使用像 `to top`、`to right`、`to bottom right` 這類關鍵字或 `45deg` 這類角度來控制這條線的方向。你可以使用任何有效的 CSS 顏色以及任意數量的顏色停點。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient">Linear Gradient</div>
```

```css
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

:::

- **放射狀漸層**：這些漸層會從中心點輻射出圓形或橢圓形的漸層。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient">Radial Gradient</div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::
