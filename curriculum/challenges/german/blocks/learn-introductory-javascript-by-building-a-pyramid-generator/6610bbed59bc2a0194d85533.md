# --description--

Beachten Sie, wie der Wert innerhalb Ihres `rows`-Arrays direkt geändert wurde? Das nennt man <dfn>Mutation</dfn>. Wenn Sie mehr über Arrays lernen, erfahren Sie, wann Sie ein Array mutieren sollten und wann nicht.

Bevor Sie weitermachen, ist dies eine großartige Gelegenheit, eine häufige Verwendung von Arrays kennenzulernen. Derzeit greift Ihr Code mit `rows[2]` auf das letzte Element im Array zu. Aber Sie wissen möglicherweise nicht, wie viele Elemente ein Array enthält, wenn Sie das letzte Element möchten.

Sie können die `.length`-Eigenschaft eines Arrays verwenden – diese gibt die Anzahl der Elemente im Array zurück. Um das letzte Element eines beliebigen Arrays zu erhalten, können Sie folgende Syntax verwenden:

```js
array[array.length - 1]
```

`array.length` gibt die Anzahl der Elemente im Array zurück. Wenn Sie `1` subtrahieren, erhalten Sie den Index des letzten Elements im Array. Sie können dieses Konzept auch auf Ihr `rows`-Array anwenden.

Aktualisieren Sie Ihr `rows[2]`, um dynamisch auf das letzte Element im `rows`-Array zuzugreifen. Orientieren Sie sich am obigen Beispiel.

In Ihrer Konsole sollte sich nichts ändern.
