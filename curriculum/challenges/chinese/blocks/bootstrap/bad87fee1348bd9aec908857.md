# --description--

使用 jQuery 时，修改 HTML 元素时并不需要直接修改 HTML 代码。

必须确保让每个人都知道不应该直接修改此页面上的这些代码。

请记住，你可以用 `<!--` 开始一个评论，并用 `-->` 结束一个评论。

请你在 HTML 顶部加如下一段注释： `Code below this line should not be changed` 。

# --hints--

你应该在你的超文本标记语言顶部用 `<!--` 开始一个评论。

```js
assert.match(code,(/^\s*<!--/));
```

注释内容应该为 `Code below this line should not be changed`。

```js
assert.match(code,(/<!--(?!(>|->|.*-->.*this line))\s*.*this line.*\s*-->/gi));
```

你应该用 `-->` 关闭你的评论。

```js
assert.match(code,(/-->.*\n+.+/g));
```

注意，注释的开始标签和闭合标签数量应该一一对应，保持数量一致。

```js
assert.equal(code.match(/<!--/g).length,code.match(/-->/g).length);
```
