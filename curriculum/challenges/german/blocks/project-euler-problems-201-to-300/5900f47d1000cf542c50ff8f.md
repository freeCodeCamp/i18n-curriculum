# --description--

Für eine positive Zahl $n$ definieren Sie $C(n)$ als die Anzahl der ganzen Zahlen $x$, für die $1 < x < n$ gilt und $x^3 \equiv 1\bmod n$.

Wenn $n = 91$ ist, gibt es 8 mögliche Werte für $x$, nämlich: 9, 16, 22, 29, 53, 74, 79, 81. Somit ist $C(91) = 8$.

Finden Sie die Summe der positiven Zahlen $n ≤ {10}^{11}$, für die $C(n)=242$ gilt.
