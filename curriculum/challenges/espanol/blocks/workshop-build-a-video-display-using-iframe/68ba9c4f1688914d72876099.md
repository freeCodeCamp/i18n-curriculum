# --description--

Por último, pero no menos importante, el atributo que agregarás es `allowfullscreen`. Como implica, permite que el video se vea en modo de pantalla completa.

¡Con eso, el taller está completado!

# --hints--

Tu elemento `iframe` debe tener un atributo `allowfullscreen`.

```js
const iframeEl = document.querySelector('iframe')
assert.exists(iframeEl?.getAttribute('allowfullscreen'))
```
