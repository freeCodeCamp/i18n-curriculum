# --interactive--

## 다양한 속성 선택자와 링크 다루기

- **정의**: `attribute` 선택자는 `href` 또는 `title` 같은 속성을 기준으로 HTML 요소를 선택할 수 있게 해줍니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Link with href</a>
<a>No href</a>
```

```css
a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

- **`title` 속성**: 이 속성은 요소에 대한 추가 정보를 제공합니다. `title` 속성을 가진 링크를 선택하는 방법은 다음과 같습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Tooltip">Link with title</a>
<a href="#">Normal link</a>
```

```css
a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

- **`href`와 `title` 속성을 모두 가진 링크를 선택하려면 선택자를 결합하세요**: 여러 속성 선택자를 결합합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Info">Href + Title</a>
<a href="#">Only href</a>
```

```css
a[href][title] {
  color: green;
}
```

:::

- **공백으로 구분된 목록 내 단어 하나와 일치시키기**: 특정 클래스 단어를 가진 링크를 선택합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a class="link primary">Primary link</a>
<a class="link secondary">Secondary link</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

- **특정 접두사로 시작하는 값과 일치시키기**: `https://`로 시작하는 링크를 선택합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">HTTPS link</a>
<a href="http://www.freecodecamp.org">HTTP link</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

- **특정 접미사로 끝나는 값과 일치시키기**: `.jpg`로 끝나는 링크를 선택합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="photo.jpg">Image link</a>
<a href="index.html">HTML link</a>
```

```css
a[href$=".jpg"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

- **값 안에 특정 부분 문자열이 포함된 경우 일치시키기**: 값 어디에나 `https`가 포함된 링크를 선택합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Secure link</a>
<a href="page.html">Local link</a>
```

```css
a[href*="https"] {
  color: teal;
}
```

:::

- **요약**: 이러한 패턴을 사용하면 속성과 값에 따라 링크를 일관되게 스타일링하기 쉽습니다.

## `lang`와 `data-lang` 속성을 가진 요소 선택하기

- **`lang` 속성**: 이 속성은 HTML에서 요소 내 콘텐츠의 언어를 지정하는 데 사용됩니다. 다국어 웹사이트에서는 작성된 언어에 따라 요소를 다르게 스타일링할 수 있습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">English text</p>
<p lang="fr">French text</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

- **`data-lang` 속성**: `data-lang` 같은 사용자 정의 데이터 속성은 요소에 추가 정보를 저장하는 데 자주 사용되며, 특정 텍스트 섹션에서 사용된 언어를 지정할 때도 활용됩니다. `data-lang` 속성을 기준으로 요소를 스타일링하는 방법은 다음과 같습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div data-lang="fr">French content</div>
<div data-lang="en">English content</div>
```

```css
div[data-lang="fr"] {
  color: blue;
}
```

:::

## 속성 선택자, 순서 있는 목록 요소, 그리고 `type` 속성 다루기

- **`type` 속성**: HTML에서 순서 있는 목록을 다룰 때 `type` 속성은 숫자, 알파벳, 로마 숫자 등 번호 매기기 스타일을 지정할 수 있게 해줍니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Alpha</li>
  <li>Beta</li>
</ol>

<ol type="i">
  <li>One</li>
  <li>Two</li>
</ol>
```

```css
/*Example targeting uppercase alphabetical numbering*/
ol[type="A"] {
  color: purple;
  font-weight: bold;
}

/*Example targeting lowercase Roman numerals*/
ol[type="i"] {
  color: green;
}
```

:::
