# --explanation--

`First Conditional` 用于谈论未来真实或可能发生的情况及其可能的结果。它遵循结构 `if` + `Present Simple` 时态表示条件，`will` 或 `can` + 动词原形表示结果。例如：

- `If you study hard, you will pass the exam.`——这描述了一个可能的情况及其结果。

- `If it rains tomorrow, we can cancel the picnic.` - 这解释了如果发生某个条件（下雨），可能会发生什么。
