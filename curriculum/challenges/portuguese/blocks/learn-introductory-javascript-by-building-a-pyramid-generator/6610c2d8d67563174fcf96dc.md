# --description--

Quando você tem um valor que está explicitamente escrito no seu código, como a string `"Hello!"` na sua função, ele é considerado <dfn>hard-coded</dfn>.Hard-coding um valor dentro de uma função pode não torná-la tão reutilizável quanto você gostaria.

Em vez disso, você pode definir <dfn>parameters</dfn> para a função.Parâmetros são variáveis especiais que recebem um valor quando você chama a função e podem ser usados na sua função para alterar dinamicamente o resultado do código da função.

Para adicionar um parâmetro à sua função, você precisa adicionar um nome de variável dentro dos parênteses.Por exemplo, esta função `demo` tem um parâmetro `name`:

```js
function demo(name) {

}
```

`name` parece um parâmetro útil, então vá em frente e adicione-o à sua função `padRow`.
