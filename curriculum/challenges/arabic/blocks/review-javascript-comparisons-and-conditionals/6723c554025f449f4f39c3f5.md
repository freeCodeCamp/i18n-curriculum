## المقارنات وأنواع البيانات `null` و `undefined`

- **المقارنات و`undefined`**: المتغير يكون `undefined` عندما يُعلن عنه دون تعيين قيمة له. هذه هي القيمة الافتراضية للمتغيرات غير المهيأة والمعلمات التي لم تُزوَّد بمعلمة. يتحول `undefined` إلى `NaN` في السياقات الرقمية، مما يجعل جميع المقارنات الرقمية مع `undefined` تُرجع `false`.

:::interactive_editor

```js
console.log(undefined < 0); // false (NaN < 0 is false)
console.log(undefined >= 0); // false (NaN >= 0 is false)
```

:::

- **المقارنات و`null`**: نوع `null` يُمثل الغياب المقصود للقيمة. يتحول `null` إلى `0` في السياقات الرقمية، مما قد يؤدي إلى سلوك غير متوقع في المقارنات الرقمية:

:::interactive_editor

```js
console.log(null < 0); // false (0 < 0 is false)
console.log(null >= 0); // true (0 >= 0 is true)
```

:::

- عند استخدام معامل المساواة (`==`)، يكون `null` و `undefined` متساويين فقط مع بعضهما البعض ومع نفسيهما:

:::interactive_editor

```js
console.log(null == undefined); // true
console.log(null == 0); // false
console.log(undefined == NaN); // false
```

:::

- مع ذلك، عند استخدام معامل المساواة الصارمة (`===`)، الذي يتحقق من القيمة والنوع دون إجراء إجبار النوع، لا يكون `null` و `undefined` متساويين:

:::interactive_editor

```js
console.log(null === undefined); // false
```

:::

## عبارات `switch`

- **التعريف**: تقوم عبارة `switch` بتقييم تعبير وتطابق قيمته مع سلسلة من بنود `case`. عند العثور على تطابق، تُنفذ كتلة الشفرة المرتبطة بذلك البند. يجب وضع عبارة `break` في نهاية كل بند لإنهاء تنفيذه والمتابعة إلى التالي. بند `default` هو بند اختياري وينفذ فقط إذا لم يتطابق أي من البنود الأخرى. يُوضع بند `default` في نهاية عبارة `switch`.

:::interactive_editor

```js
const dayOfWeek = 3; 

switch (dayOfWeek) {
  case 1:
    console.log("It's Monday! Time to start the week strong.");
    break;
  case 2:
    console.log("It's Tuesday! Keep the momentum going.");
    break;
  case 3:
    console.log("It's Wednesday! We're halfway there.");
    break;
  case 4:
    console.log("It's Thursday! Almost the weekend.");
    break;
  case 5:
    console.log("It's Friday! The weekend is near.");
    break;
  case 6:
    console.log("It's Saturday! Enjoy your weekend.");
    break;
  case 7:
    console.log("It's Sunday! Rest and recharge.");
    break;
  default:
    console.log("Invalid day! Please enter a number between 1 and 7.");
}
```

:::
