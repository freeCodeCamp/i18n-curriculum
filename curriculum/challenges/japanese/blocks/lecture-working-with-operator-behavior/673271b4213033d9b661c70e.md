# --interactive--

JavaScriptでは、すべての算術演算子に複合代入の形があります。複合代入演算子は、変数に対する操作を行い、その結果を同じ変数に代入する処理を簡潔に書くための省略形です。操作と代入を組み合わせて、`x += y`のような短い形で書けます。これは変数名を繰り返さずに`x = x + y`と同じ意味です。例えば、次のように書く代わりに：

:::interactive_editor

```js
let num = 5;
num = num + 2;

console.log(num); // 7
```

:::

次のように書けます：

:::interactive_editor

```js
let num = 5;
num += 2;

console.log(num); // 7
```

:::

`num += 2`が加算と代入の両方のステップを1つにまとめているのがわかりますね。これにより時間を節約し、コードの見た目もすっきりします。では、JavaScriptで最もよく使われる複合代入演算子について詳しく見ていきましょう。

すでに見たように、`+=`演算子は既存の変数に値を加えることができます。これは加算代入演算子と呼ばれます。加算代入演算子は変数の現在の値に指定した数を加え、その結果を変数に代入します：

:::interactive_editor

```js
let total = 10;
total += 5;

console.log(total); // 15
```

:::

想像できるように、`-=`で表される減算代入演算子もあります。減算代入演算子は変数の現在の値から指定した値を引き、その新しい値を変数に代入します：

:::interactive_editor

```js
let score = 20;
score -= 7;

console.log(score); // 13
```

:::

もし減算代入を使わなければ、次のように書くことになります：

:::interactive_editor

```js
let score = 20;
score = score - 7;

console.log(score); // 13
```

:::

乗算代入演算子は`*=`で表されます。これは変数の現在の値に指定した数を掛け、その結果を変数に再代入します：

:::interactive_editor

```js
let points = 5;
points *= 3;

console.log(points); // 15
```

:::

最後に、`/=`で表される除算代入演算子があります。これも他の演算子と同様に、変数の現在の値を指定した数で割り、その結果を変数に代入します：

:::interactive_editor

```js
let balance = 100;
balance /= 4;

console.log(balance); // 25
```

:::

JavaScriptにはすべての演算子に対応する複合代入演算子があることを覚えておいてください。先に挙げた4つ以外にも以下があります：

- 余り代入演算子（`%=`）は、変数を指定した数で割り、その余りを変数に代入します。

- べき乗代入演算子（`**=`）は、変数を指定した数のべき乗にして、その結果を変数に代入します。

- ビット単位AND代入演算子（`&=`）は、指定した数とのビット単位AND演算を行い、その結果を変数に代入します。

- ビット単位OR代入演算子（`|=`）は、指定した数とのビット単位OR演算を行い、その結果を変数に代入します。
