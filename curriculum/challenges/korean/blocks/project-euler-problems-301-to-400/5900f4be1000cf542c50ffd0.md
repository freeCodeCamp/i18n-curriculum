# --description--

길이 $n$인 정수 수열 $\\{a_1, a_2, \ldots, a_n\\}$이 다음 조건을 만족한다고 하겠습니다:

- $a_1 = 6$
- 모든 $1 ≤ i &lt; n$에 대해: $φ(a_i) &lt; φ(a_{i + 1}) &lt; a_i &lt; a_{i + 1}$

여기서 $φ$는 오일러 피 함수를 나타냅니다.

$S(N)$은 $a_n ≤ N$인 그러한 수열의 개수입니다.

예를 들어, $S(10) = 4$입니다: {6}, {6, 8}, {6, 8, 9} 그리고 {6, 10}.

$S(100) = 482\\,073\\,668$이고 $S(10\\,000)\bmod {10}^8 = 73\\,808\\,307$임을 확인할 수 있습니다.

$S(20\\,000\\,000)\bmod {10}^8$를 구하세요.
