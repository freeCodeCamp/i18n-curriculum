# --description--

Katika funzo lililopita, ulijifunza kuhusu aina za kawaida za data katika Python ikiwa ni pamoja na aina za mfuatano wa herufi, nambari kamili, na float. Unapojenga programu zako, utahitaji kujifunza jinsi ya kuona aina ya variable.

Hapa kuna mfano wa variable:

```py
developer = 'Devin'
```

Ili kuona ni aina gani `developer`, unaweza kutumia kitendakazi cha `type()` kama ifuatavyo:

```py
developer = 'Devin'

print(type(developer)) # <class 'str'>
```

Matokeo ya `<class 'str'>` yanamaanisha kuwa `developer` ni aina ya mfuatano wa herufi.

Kama hutoi hoja yoyote kwa kitendakazi cha `type()`, basi utapokea ujumbe wa kosa kama ifuatavyo:

```md
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: type() takes 1 or 3 arguments
```

Hapa kuna aina zote za data ulizojifunza hadi sasa pamoja na aina zao kwenye terminali:

```python
my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 'hello', 8.5}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 'hello', 8.5)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>
```

Kutakuwa na nyakati katika programu yako ambapo utahitaji kuthibitisha kuwa variable fulani ni aina maalum kabla ya kufanya operesheni juu yake. Hapa ndipo kitendakazi cha `isinstance()` kinapokuja kuwa cha msaada.

Hapa kuna mfano wa variable yenye mfuatano wa herufi umepewa thamani:

```py
account_balance = '12'
```

Kama utajaribu kufanya usemi wa kihesabu kama mgawanyo ukitumia variable ya `account_balance`, basi utapokea ujumbe wa kosa.

```py
account_balance = '12'

account_balance / 2

# Traceback (most recent call last):
#   File "<stdin>", line 1, in <module>
# TypeError: unsupported operand type(s) for /: 'str' and 'int'
```

Ili kuona kama `account_balance` ni nambari kamili, unaweza kuangalia kwa kutumia kitendakazi cha `isinstance()` kama ifuatavyo:

```py
account_balance = '12'

isinstance(account_balance, int) # False
```

Kitendakazi kilichojengwa ndani cha `isinstance()` hukuruhusu kuangalia kama variable inalingana na aina maalum ya data. Kinachukua kitu na aina unayotaka kuikagua, kisha hurudisha thamani ya kweli au siyo kweli. Katika kesi hii, kwa kuwa `account_balance` ni mfuatano wa herufi, kitarejesha `False`.

Kitendakazi cha `isinstance()` pia hukuruhusu kuangalia aina nyingi kwa wakati mmoja.

Hapa kuna mfano wa kuangalia kama `account_balance` ni `int` au `float`:

```py
account_balance = 12
isinstance(account_balance, (int, float)) # True
```

Katika mfano huu, `account_balance` ni nambari kamili hivyo `isinstance()` hurudisha `True`. Kama `account_balance` ingekuwa `12.0`, `isinstance()` bado ingetumia `True` kwa sababu unakagua nambari kamili au float.

Katika warsha na maabara zijazo, utatumia vitendakazi vya `type()` na `isinstance()` kuhakikisha variables zako zina aina sahihi za data kabla ya kufanya operesheni juu yao.
