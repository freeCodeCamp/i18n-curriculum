# --description--

Usemi `Got it` hutumika mara nyingi katika mazungumzo kuonyesha kuwa mtu amelewa jambo fulani. Ni njia ya kawaida na ya haraka ya kuthibitisha kuwa umepokea na kuelewa taarifa. Kwa mfano:

Lucy: `You need to finish the report by tomorrow.`
Bob: `Got it!` 

`Got it` ina maana kwamba Bob anaelewa maelekezo.
