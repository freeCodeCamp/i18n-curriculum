# --description--

$f(n)$ を、$x$ と $y$ が正の整数で $x ≤ y$ かつ $x$ と $y$ の最小公倍数が $n$ である組み合わせ ($x$, $y$) の個数とします。

$g$ を $f$ の総和関数とし、すなわち $g(n) = \sum f(i)$ （$1 ≤ i ≤ n$）とします。

$g({10}^6) = 37\\,429\\,395$ であることが与えられています。

$g({10}^{12})$ を求めてください。
