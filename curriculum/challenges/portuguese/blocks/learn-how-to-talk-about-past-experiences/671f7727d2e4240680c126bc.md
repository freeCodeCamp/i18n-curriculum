# --explanation--

A frase `so that` é usada para expressar propósito ou razão, mostrando por que uma ação é realizada. Por exemplo: 

`She explained the process clearly so that everyone could understand it.` - Isso significa que a intenção dela ao explicar o processo claramente foi fazer com que todos entendessem. 

No diálogo, Brian usa isso para mostrar que seu objetivo ao fornecer treinamento é capacitar outros a desempenhar no mesmo nível que ele quando ajustou o código ele mesmo.
