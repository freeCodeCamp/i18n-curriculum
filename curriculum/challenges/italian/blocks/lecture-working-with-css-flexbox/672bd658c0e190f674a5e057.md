# --interactive--

Le proprietà flex sono proprietà che puoi applicare ai contenitori flex per determinare la distribuzione degli elementi figli. Vedremo alcune delle più comunemente usate: `flex-wrap`, `justify-content` e `align-items`.

Iniziamo con `flex-wrap`. Questa proprietà determina come gli elementi flex vengono avvolti all'interno di un contenitore flex per adattarsi allo spazio disponibile. `flex-wrap` può assumere tre valori possibili: `nowrap`, `wrap` e `wrap-reverse`. `nowrap` è il valore predefinito: gli elementi flex non verranno avvolti su una nuova riga, anche se la loro larghezza supera quella del contenitore.

Nel codice qui sotto, abbiamo tre elementi `div`. Concentrati sul `width`. Il contenitore `main` bordato di rosso ha un `width` di `200px`, mentre i suoi tre elementi figli `div` combinati hanno un `width` di `240px` (`80px` ciascuno):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::


La larghezza degli elementi `div` supera quella del loro contenitore, ma per impostazione predefinita verranno ridotti per adattarsi allo spazio disponibile. Se vuoi invece avvolgerli quando superano la larghezza del contenitore, puoi impostare `flex-wrap: wrap` sul contenitore flex:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  flex-wrap: wrap;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Gli elementi `div` verranno riorganizzati in righe quando superano la larghezza del contenitore. Puoi avvolgere gli elementi flex in ordine inverso con `flex-wrap: wrap-reverse`. La proprietà `flex-flow` è una proprietà abbreviata per `flex-direction` e `flex-wrap`. In questo esempio, impostiamo `flex-direction` su `column` e `flex-wrap` su `wrap-reverse`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  flex-flow: column wrap-reverse;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Ottimo. Ora parliamo di `justify-content`. `justify-content` allinea gli elementi figli lungo l'asse principale del contenitore flex. Se assegni il valore `flex-start` a `justify-content`, gli elementi flex saranno allineati all'inizio dell'asse principale. Questo può essere orizzontale o verticale:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: flex-start;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Con `justify-content: flex-end`, gli elementi flex sono allineati alla fine dell'asse principale, orizzontalmente o verticalmente.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: flex-end;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Per centrare gli elementi flex lungo l'asse principale, puoi usare `justify-content: center`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: center;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Per distribuire gli elementi in modo uniforme lungo l'asse principale, puoi usare `justify-content: space-between`. Questo aggiungerà dello spazio tra gli elementi flex se necessario.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: space-between;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

`justify-content: space-around` distribuisce gli elementi flex in modo uniforme all'interno dell'asse principale, aggiungendo uno spazio prima del primo elemento e dopo l'ultimo. Questo spazio aggiuntivo è la metà dello spazio tra ogni coppia di elementi adiacenti. Se c'è un solo elemento da distribuire, sarà centrato.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: space-around;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::


E infine, ma non meno importante, abbiamo `justify-content: space-evenly`, che distribuisce gli elementi in modo uniforme lungo l'asse principale. Lo spazio tra gli elementi e lo spazio prima e dopo il primo e l'ultimo elemento sono esattamente uguali:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: space-evenly;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Ottimo. Ora sai come distribuire gli elementi flex lungo l'asse principale. Ma potresti anche volerli distribuire lungo l'asse trasversale. Ricorda che l'asse trasversale è perpendicolare all'asse principale. Puoi farlo con la proprietà `align-items`. Per centrare gli elementi lungo l'asse trasversale, devi solo aggiungere `align-items: center` al contenitore flex:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: center;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

In questo esempio, gli elementi flex sono centrati lungo l'asse trasversale, che per impostazione predefinita è verticale. Se l'asse trasversale è orizzontale, saranno centrati orizzontalmente invece. Al contrario, `align-items: flex-start` allinea gli elementi all'inizio dell'asse trasversale:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: flex-start;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

L'opposto è `align-items: flex-end`. Questo allineerà gli elementi flex alla fine dell'asse trasversale, verticalmente o orizzontalmente.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: flex-end;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Per allungare gli elementi flex lungo l'asse trasversale, puoi usare `align-items: stretch`. Questo funziona anche con elementi che hanno dimensioni automatiche, come quelli senza `width` o `height` impostati, o con solo un `width` o `height` minimo. Gli elementi flex si allungheranno per riempire il contenitore nella direzione dell'asse trasversale.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box">Three</div>
</div>

```

```css
.container {
  display: flex;
  align-items: stretch; 
  border: 2px solid #444;
  height: 200px; 
}

.box {
  width: 100px;           
  border: 1px solid #888;
  background-color: lightblue;
}
```

:::

E infine, puoi usare la proprietà `align-self` per assegnare un allineamento diverso sull'asse trasversale a un singolo elemento flex. Per esempio, puoi allungarlo con `align-self: stretch`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box special">Three</div>
</div>

```

```css
.container {
  display: flex;
  align-items: flex-start; 
  border: 2px solid #444;
  height: 200px; 
}

.box {
  width: 100px;
  border: 1px solid #888;
  background-color: lightblue;
  margin: 4px;
}

.special {
  align-self: stretch; 
  background-color: lightcoral;
}
```

:::

Puoi centrarlo con `align-self: center`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box special">Three</div>
</div>

```

```css
.container {
  display: flex;
  align-items: flex-start; 
  border: 2px solid #444;
  height: 200px; 
}

.box {
  width: 100px;
  border: 1px solid #888;
  background-color: lightblue;
  margin: 4px;
}

.special {
  align-self: center; 
  background-color: lightcoral;
}
```

:::

Puoi allinearlo all'inizio dell'asse trasversale con `align-self: flex-start`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box special">Three</div>
</div>

```

```css

  display: flex;
  align-items: flex-start; 
  border: 2px solid #444;
  height: 200px; 
}

.box {
  width: 100px;
  border: 1px solid #888;
  background-color: lightblue;
  margin: 4px;
}

.special {
  align-self: flex-start; 
  background-color: lightcoral;
}
```

:::

Oppure puoi allinearlo alla fine dell'asse trasversale con `align-self: flex-end`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box special">Three</div>
</div>

```

```css
.container {
  display: flex;
  align-items: flex-start; 
  border: 2px solid #444;
  height: 200px; 
}

.box {
  width: 100px;
  border: 1px solid #888;
  background-color: lightblue;
  margin: 4px;
}

.special {
  align-self: flex-end; 
  background-color: lightcoral;
}
```

:::

Ci sono altre proprietà e valori flex tra cui scegliere per creare il layout responsivo che immagini, ma queste sono le più comunemente usate. Con queste proprietà CSS flex e la tua nuova conoscenza del modello flex CSS, puoi iniziare a creare layout responsivi per offrire un'esperienza utente fluida e inclusiva su tutti i dispositivi.
