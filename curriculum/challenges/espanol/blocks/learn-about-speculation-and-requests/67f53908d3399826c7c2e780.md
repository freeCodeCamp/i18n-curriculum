# --explanation--

`Want` expresa un deseo de hacer algo. En el pasado, dices `wanted` para mostrar que el deseo existía en un momento anterior. Va seguido del infinitivo de un verbo (con `to`). Por ejemplo:

- `She wants to install the new software.` - Ella desea configurar o agregar el nuevo software al sistema.

- `They wanted to upgrade their system last week.` - En el pasado, planearon actualizar.
