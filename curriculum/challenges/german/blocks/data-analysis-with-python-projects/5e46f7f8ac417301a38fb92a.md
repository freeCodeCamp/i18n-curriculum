# --instructions--

In diesem Projekt visualisieren Sie medizinische Untersuchungsdaten und führen Berechnungen mit `matplotlib`, `seaborn` und `pandas` durch. Die Datensatzwerte wurden während medizinischer Untersuchungen erhoben.

## Datenbeschreibung

Die Zeilen im Datensatz repräsentieren Patienten, und die Spalten enthalten Informationen wie Körpermaße, Ergebnisse verschiedener Bluttests und Lebensstilentscheidungen. Sie verwenden den Datensatz, um die Beziehung zwischen Herz-Kreislauf-Erkrankungen, Körpermaßen, Blutwerten und Lebensstil zu untersuchen.

Dateiname: medical_examination.csv

| Funktion | Variablentyp | Variable      | Werttyp |
|:-------:|:------------:|:-------------:|:----------:|
| Alter | Objektives Merkmal | `age` | int (Tage) |
| Größe | Objektives Merkmal | `height` | int (cm) |
| Gewicht | Objektives Merkmal | `weight` | float (kg) |
| Geschlecht | Objektives Merkmal | `gender` | kategorialer Code |
| Systolischer Blutdruck | Untersuchungsmerkmal | `ap_hi` | int |
| Diastolischer Blutdruck | Untersuchungsmerkmal | `ap_lo` | int |
| Cholesterin | Untersuchungsmerkmal | `cholesterol` | 1: normal, 2: über normal, 3: deutlich über normal |
| Glukose | Untersuchungsmerkmal | `gluc` | 1: normal, 2: über normal, 3: deutlich über normal |
| Rauchen | Subjektives Merkmal | `smoke` | binär |
| Alkoholkonsum | Subjektives Merkmal | `alco` | binär |
| Körperliche Aktivität | Subjektives Merkmal | `active` | binär |
| Vorhandensein oder Fehlen von Herz-Kreislauf-Erkrankungen | Zielvariable | `cardio` | binär |


## Anweisungen

Erstellen Sie ein Diagramm ähnlich wie `examples/Figure_1.png`, in dem wir die Anzahl der guten und schlechten Ergebnisse für die Variablen `cholesterol`, `gluc`, `alco`, `active` und `smoke` für Patienten mit `cardio=1` und `cardio=0` in verschiedenen Panels anzeigen.

Fügen Sie zu jeder Nummer in der `medical_data_visualizer.py` Datei den Code aus der zugehörigen Anweisungsnummer unten hinzu.

1. Importieren Sie die Daten aus `medical_examination.csv` und weisen Sie sie der Variablen `df` zu.
2. Fügen Sie den Daten eine `overweight` Spalte hinzu. Um zu bestimmen, ob eine Person übergewichtig ist, berechnen Sie zuerst ihren BMI, indem Sie ihr Gewicht in Kilogramm durch das Quadrat ihrer Größe in Metern teilen. Wenn dieser Wert > 25 ist, dann ist die Person übergewichtig. Verwenden Sie den Wert `0` für NICHT übergewichtig und den Wert `1` für übergewichtig.
3. Normalisieren Sie die Daten, indem Sie `0` immer als gut und `1` immer als schlecht festlegen. Wenn der Wert von `cholesterol` oder `gluc` 1 ist, setzen Sie den Wert auf `0`. Wenn der Wert größer als `1` ist, setzen Sie den Wert auf `1`.
4. Zeichnen Sie das kategorische Diagramm in der `draw_cat_plot` Funktion.
5. Erstellen Sie ein DataFrame für das Kategoriediagramm mit `pd.melt` mit Werten von `cholesterol`, `gluc`, `smoke`, `alco`, `active` und `overweight` in der Variablen `df_cat`.
6. Gruppieren und formatieren Sie die Daten in `df_cat`, um sie nach `cardio` aufzuteilen. Zeigen Sie die Anzahl jedes Merkmals an. Sie müssen eine der Spalten umbenennen, damit `catplot` korrekt funktioniert.
7. Konvertieren Sie die Daten in das `long` Format und erstellen Sie ein Diagramm, das die Wertanzahlen der kategorialen Merkmale mit der folgenden Methode aus der seaborn-Bibliothek zeigt: `sns.catplot()`.
8. Holen Sie die Abbildung für die Ausgabe und speichern Sie sie in der Variablen `fig`.
9. Ändern Sie die nächsten zwei Zeilen nicht.
10. Zeichnen Sie die Heatmap in der `draw_heat_map` Funktion.
11. Bereinigen Sie die Daten in der Variablen `df_heat`, indem Sie die folgenden Patientensegmente herausfiltern, die inkorrekte Daten darstellen:
    - diastolischer Druck ist höher als systolischer (Behalten Sie die korrekten Daten mit `(df['ap_lo'] <= df['ap_hi'])`)
    - Größe ist kleiner als das 2,5. Perzentil (Behalten Sie die korrekten Daten mit `(df['height'] >= df['height'].quantile(0.025))`)
    - Größe ist größer als das 97,5. Perzentil
    - Gewicht ist kleiner als das 2,5. Perzentil
    - Gewicht ist größer als das 97,5. Perzentil
12. Berechnen Sie die Korrelationsmatrix und speichern Sie sie in der Variablen `corr`.
13. Erzeugen Sie eine Maske für das obere Dreieck und speichern Sie sie in der Variablen `mask`.
14. Richten Sie die `matplotlib` Abbildung ein.
15. Zeichnen Sie die Korrelationsmatrix mit der Methode aus der `seaborn` Bibliothek: `sns.heatmap()`.
16. Ändern Sie die nächsten zwei Zeilen nicht.
