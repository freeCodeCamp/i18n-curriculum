### --feedback--

Esta é uma palavra composta, formada pela combinação de duas palavras com um `-`. Este termo significa que acontece instantaneamente, sem qualquer atraso.

# --explanation--

`Real-time` significa acontecer instantaneamente, sem qualquer atraso. Por exemplo:

- `We are having a real-time video chat.` - A conversa está acontecendo instantaneamente, sem atraso entre as respostas.

`Collaboration` significa trabalhar junto com outras pessoas para alcançar um objetivo comum. Por exemplo:

- A equipe colaborou para terminar o projeto no prazo. - A equipe trabalhou junto para completar o projeto.

`Real-time project collaboration` significa trabalhar juntos em um projeto instantaneamente, mesmo que os membros da equipe estejam em locais diferentes. Por exemplo:

- `Our team uses a shared platform for real-time project collaboration.` - A equipe trabalha junta no projeto com atualizações e comunicação imediatas.
