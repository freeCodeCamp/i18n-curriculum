# --description--

Python offre diversi metodi integrati che rendono semplice lavorare con le stringhe. Tra questi, ma non solo, ci sono i seguenti:

- `upper()`: Restituisce una nuova stringa con tutti i caratteri convertiti in maiuscolo.
    

```python
my_str = 'hello world'

uppercase_my_str = my_str.upper()
print(uppercase_my_str)  # HELLO WORLD
```

- `lower()`: Restituisce una nuova stringa con tutti i caratteri convertiti in minuscolo.
    

```python
my_str = 'Hello World'

lowercase_my_str = my_str.lower()
print(lowercase_my_str)  # hello world
```

- `strip()`: Restituisce una nuova stringa con i caratteri iniziali e finali specificati rimossi. Se non viene passato alcun argomento, rimuove gli spazi bianchi iniziali e finali.
    

```python
my_str = '  hello world  '

trimmed_my_str = my_str.strip()
print(trimmed_my_str)  # "hello world"
```

- `replace(old, new)`: Restituisce una nuova stringa con tutte le occorrenze di `old` sostituite da `new`.
    

```python
my_str = 'hello world'

replaced_my_str = my_str.replace('hello', 'hi')
print(replaced_my_str)  # hi world
```

- `split(separator)`: Divide una stringa su un separatore specificato in una lista di stringhe. Se non viene specificato alcun separatore, divide sugli spazi bianchi.
    

```python
my_str = 'hello world'

split_words = my_str.split()
print(split_words)  # ['hello', 'world']
```

- `join(iterable)`: Unisce gli elementi di un iterabile in una stringa con un separatore.
    

```python
my_list = ['hello', 'world']

joined_my_str = ' '.join(my_list)
print(joined_my_str)  # hello world
```

- `startswith(prefix)`: Restituisce un booleano che indica se una stringa inizia con il prefisso specificato.
    

```python
my_str = 'hello world'

starts_with_hello = my_str.startswith('hello')
print(starts_with_hello)  # True
```

- `endswith(suffix)`: Restituisce un booleano che indica se una stringa termina con il suffisso specificato.
    

```python
my_str = 'hello world'

ends_with_world = my_str.endswith('world')
print(ends_with_world)  # True
```

- `find(substring)`: Restituisce l’indice della prima occorrenza di `substring`, o `-1` se non ne trova nessuna.
    

```python
my_str = 'hello world'

world_index = my_str.find('world')
print(world_index)  # 6
```

- `count(substring)`: Restituisce il numero di volte in cui una sottostringa appare in una stringa.
    

```python
my_str = 'hello world'

o_count = my_str.count('o')
print(o_count)  # 2
```

- `capitalize()`: Restituisce una nuova stringa con il primo carattere maiuscolo e gli altri caratteri minuscoli.
    

```python
my_str = 'hello world'

capitalized_my_str = my_str.capitalize()
print(capitalized_my_str)  # Hello world
```

- `isupper()`: Restituisce `True` se tutte le lettere nella stringa sono maiuscole e `False` se non lo sono.
    

```python
my_str = 'hello world'

is_all_upper = my_str.isupper()
print(is_all_upper)  # False
```

- `islower()`: Restituisce `True` se tutte le lettere nella stringa sono minuscole e `False` se non lo sono.
    

```python
my_str = 'hello world'

is_all_lower = my_str.islower()
print(is_all_lower)  # True
```

- `title()`: Restituisce una nuova stringa con la prima lettera di ogni parola maiuscola.
    

```python
my_str = 'hello world'

title_case_my_str = my_str.title()
print(title_case_my_str)  # Hello World
```
