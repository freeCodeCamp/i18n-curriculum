# --explanation--

기억하세요:

`May`는 가능성이나 허가를 나타냅니다. 예를 들어:

`May I use your phone?`

`Might`는 `may`보다 가능성이 낮으며, 덜 확실한 상황에 자주 사용됩니다. 예를 들어:

`She might come to the meeting.`

`Can`는 능력이나 허가를 나타냅니다. 예를 들어:

`Can I leave early?`

`Could`는 과거 능력, 정중한 요청, 또는 `can`보다 덜 확실한 가능성에 사용됩니다. 예를 들어:

`Could you help me?`
