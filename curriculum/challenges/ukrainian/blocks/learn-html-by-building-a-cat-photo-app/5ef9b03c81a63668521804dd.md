# --description--

Елементи `label` використовуються, щоб допомогти асоціювати текст для елемента `input` з самим елементом `input` (особливо для допоміжних технологій, таких як читач екрана).

Ось приклад елемента `label` з кнопкою `radio`:

```html
<label><input type="radio"> cat</label>
```

У прикладі, натиснення на слово `"cat"` також вибере кнопку `radio`.

Вкладіть вашу кнопку `radio` всередину елемента `label`.

# --hints--

Ви повинні переконатися, що радіокнопка все ще присутня.

```js
assert(document.querySelector('input[type="radio"]'));
```

Текст ` Indoor` має розташовуватися безпосередньо праворуч від вашої кнопки `radio`. Переконайтеся, що між елементом і текстом є пробіл. Ви або пропустили текст, або зробили помилку.

```js
const radioInputElem = document.querySelector('input');
assert.match(
  radioInputElem?.nextSibling?.nodeValue?.replace(/\s+/g, ' '),
  / Indoor/i
);
```

Ваш елемент `label` має мати початковий тег. Початкові теги мають такий синтаксис: `<elementName>`.

```js
assert(document.querySelector('label'));
```

Ваш елемент `label` має мати кінцевий тег. Кінцеві теги мають `/` одразу після символу `<`.

```js
assert(code.match(/<\/label\>/));
```

Ваша радіокнопка та її текст мають бути розташовані між початковим і кінцевим тегами елемента `label`.

```js
const labelChildNodes = [...document.querySelector('form > label').childNodes];
assert(
  labelChildNodes.filter((childNode) => childNode.nodeName === 'INPUT').length
);
```
