# --description--

У професійному контексті дієслово `to document` означає записувати інформацію для подальшого використання. Це включає фіксацію деталей, кроків або важливих моментів.

`Future reference` означає використання цієї записаної інформації пізніше, часто для розв’язання проблем або згадування процедур. Наприклад:

`They are documenting their studies for future reference.`
