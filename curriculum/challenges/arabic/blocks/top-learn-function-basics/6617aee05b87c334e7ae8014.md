# --description--

المتغير المُعلن داخل دالة يكون مرئيًا فقط داخل تلك الدالة. يُسمى هذا نطاق المتغير، وهذا النطاق المحدد يُسمى النطاق المحلي. المتغيرات المُعلنة خارج الدالة تُسمى متغيرات عامة وتكون مرئية في جميع أنحاء البرنامج. على سبيل المثال، الشفرة التالية ستُثير خطأ:

```js
function myFunction() {
  let x = 10;
}

console.log(x);
```

المتغير `x` مُعلن داخل الدالة `myFunction` وليس مرئيًا خارج الدالة. الشفرة ستُثير خطأ `ReferenceError` لأن `x` غير معرف.

إذا أعلنت متغيرًا خارج دالة، يُسمى متغيرًا عامًا ويكون مرئيًا في جميع أنحاء البرنامج. على سبيل المثال، الشفرة التالية ستعمل:

```js
let x = 10;

function myFunction() {
  console.log(x);
}

myFunction();
```

إذا أُعلن متغير بنفس الاسم داخل الدالة، فإنه يحجب المتغير الخارجي.
