## --text--

Como você declara uma variável em Python de acordo com Sophie?

## --answers--

Você declara uma variável nomeando-a após um operador.

### --feedback--

Esta escolha está incorreta, porque a variável não é nomeada depois, mas antes, de um operador.

---

Variáveis são declaradas usando um comando chamado `declare`.

### --feedback--

Python não requer um comando especial como `declare` para criar variáveis; ele usa atribuição diretamente.

---

Você usa o nome da variável seguido pelo operador de atribuição e um valor.

---

Variáveis devem ser declaradas digitando `variable` antes do nome.

### --feedback--

Python não usa nenhuma palavra-chave como `variable` para declarar variáveis.Ele simplesmente usa o nome da variável, o operador de atribuição e o valor.
