# --description--

Definamos un triángulo geométrico como un triángulo con lados enteros y lados $a ≤ b ≤ c$ de modo que sus lados formen una progresión geométrica, es decir, $b^2 = a \times c$.

Un ejemplo de tal triángulo geométrico es el triángulo con lados $a = 144$, $b = 156$ y $c = 169$.

Hay $861\\,805$ triángulos geométricos con $\text{perímetro} ≤ {10}^6$.

¿Cuántos triángulos geométricos existen con $\text{perímetro} \leq 2.5 \times {10}^{13}$?
