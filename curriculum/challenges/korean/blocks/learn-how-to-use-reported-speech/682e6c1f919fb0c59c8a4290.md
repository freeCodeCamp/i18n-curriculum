# --explanation--

**간접화법**에서는 `said that` 구문을 사용해 누군가의 말을 정확한 표현 대신 다른 방식으로 전달합니다. `that` 단어는 선택 사항이며 일상 대화에서는 자주 생략됩니다. 예를 들어:

- 직접화법: `We need to update the software.`

- 간접화법: `He said that we need to update the software.` 또는 `He said we need to update the software.`

두 표현 모두 올바르며 같은 의미입니다.
