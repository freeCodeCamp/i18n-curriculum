# --interactive--

Mzunguko wa `for...of` hutumika unapohitaji kurudia juu ya thamani kutoka kwa kitu inayoweza kurudiwa. Mifano ya vitu vinavyoweza kurudiwa ni safu za data na mfuatano wa herufi.

Hapa kuna sintaksia ya msingi ya mzunguko wa `for...of`:

```js
for (variable of iterable) {
  // code block to be executed
}
```

Kigezo katika mfano kinawakilisha thamani ya sasa ya kitu kinachorudiwa.

Ikiwa una safu ya nambari, kigezo kitakuwa nambari ya sasa katika safu. Ikiwa una mfuatano wa herufi, kigezo kitakuwa herufi ya sasa katika mfuatano.

Tuchunguze mifano michache ili uelewe vyema jinsi mzunguko wa `for...of` unavyofanya kazi.

Katika mfano huu wa kwanza tuna safu ya nambari na tunataka kurudia kila nambari na kuiandikisha kwenye konsoli.

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

:::

Tumeunda kigezo kinachoitwa `num` kitakachowakilisha nambari ya sasa katika safu. Kwa kurudia kwa mara ya 1, `num` itakuwa `1`, kwa kurudia kwa mara ya 2, `num` itakuwa `2`, na kadhalika.

Ndani ya mzunguko, tunaandika nambari ya sasa kwenye konsoli.

Hapa kuna mfano mwingine ambapo tuna mfuatano wa herufi na tunataka kurudia kila herufi na kuiandikisha kwenye konsoli.

:::interactive_editor

```js
const str = 'freeCodeCamp';

for (let char of str) {
  console.log(char);
}
```

:::

Katika mfano huu, tumeunda kigezo kinachoitwa `char` kitakachowakilisha herufi ya sasa katika mfuatano.

Kwa kila kurudia, mzunguko utaandika herufi ya sasa kwenye konsoli.

Ni muhimu kutambua kuwa unaweza kutumia `let`, au `const` wakati wa kutangaza kigezo katika mzunguko wa `for...of`.

Hata hivyo, ikiwa utatumia `const`, hakikisha thamani ya kigezo haibadiliki ndani ya mzunguko. Ikiwa itabadilika, utapata kosa.

Hapa kuna mfano wa kutumia `const` unaosababisha kosa:

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
  num = num + 1; // This will cause an error
}
```

Katika mfano huu, tunajaribu kubadilisha thamani ya `num` ndani ya mzunguko. Kwa kuwa tulitangaza `num` kwa `const`, tutapata kosa. Kwa hivyo, ikiwa unahitaji kubadilisha thamani ya kigezo ndani ya mzunguko, tumia `let` badala yake.

Tuchunguze mfano wa mwisho unaohusiana na safu ya vitu.

:::interactive_editor

```js
const people = [
  { name: 'John', age: 30 },
  { name: 'Jane', age: 25 },
  { name: 'Jim', age: 40 }
];

for (const person of people) {
  console.log(`${person.name} is ${person.age} years old`);
}
```

:::

Katika mfano huu, tuna safu ya vitu inayoitwa `people`. Kila kitu kina sifa za `name` na `age`.

Tunaporudia kwenye safu, tunaunda kigezo kinachoitwa `person` kitakachowakilisha kitu cha sasa katika safu.

Ndani ya mzunguko, tunaandika ujumbe kwenye konsoli.

Ujumbe wa kwanza utakuwa `John is 30 years old`, ujumbe wa pili utakuwa `Jane is 25 years old`, na ujumbe wa tatu utakuwa `Jim is 40 years old`.

Mizunguko ya `for...of` ni muhimu sana unapohitaji kurudia juu ya thamani kutoka kwa kitu kinachoweza kurudiwa kama safu au mfuatano wa herufi. Pia ni rahisi kusoma na inaweza kufanya msimbo wako kuwa mfupi zaidi.
