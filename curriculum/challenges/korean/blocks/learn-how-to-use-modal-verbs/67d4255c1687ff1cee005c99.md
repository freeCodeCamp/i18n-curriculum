# --explanation--

조동사 다음에는 항상 기본 동사 형태가 옵니다. 조동사는 `to`, `-ing`, 또는 과거형을 취하지 않습니다. 다음은 올바르지 않은 동사 형태의 예입니다:

- `Must to follow`은 `must`가 `to`을 취하기 때문에 올바르지 않습니다.

- `Must following`은 `must` 다음에 `-ing` 형태가 오기 때문에 올바르지 않습니다.

- `Must followed`은 `followed`가 과거형이기 때문에 올바르지 않습니다.
