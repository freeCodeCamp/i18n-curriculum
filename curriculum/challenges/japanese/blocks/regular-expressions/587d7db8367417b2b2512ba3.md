# --description--

これまでのチャレンジでは、アルファベットの文字や数字のマッチングを扱ってきました。文字の間の空白やスペースもマッチングできます。

空白文字は`\s`を使って検索できます。これは小文字の`s`です。このパターンは空白だけでなく、キャリッジリターン、タブ、フォームフィード、改行文字にもマッチします。文字クラス`[ \r\t\f\n\v]`と似ていると考えてください。

```js
let whiteSpace = "Whitespace. Whitespace everywhere!"
let spaceRegex = /\s/g;
whiteSpace.match(spaceRegex);
```

この`match`の呼び出しは`[" ", " "]`を返します。
# --instructions--

正規表現`countWhiteSpace`を変更して、文字列内の複数の空白文字を探してください。

# --hints--

正規表現にはグローバルフラグを使う必要があります。

```js
assert(countWhiteSpace.global);
```

すべての空白文字にマッチするために、ショートハンド文字`\s`を使う必要があります。

```js
assert(/\\s/.test(countWhiteSpace.source));
```

正規表現は文字列`Men are from Mars and women are from Venus.`の中の8つのスペースを見つける必要があります。

```js
assert(
  'Men are from Mars and women are from Venus.'.match(countWhiteSpace).length ==
    8
);
```

正規表現は文字列`Space: the final frontier.`の中の3つのスペースを見つける必要があります。

```js
assert('Space: the final frontier.'.match(countWhiteSpace).length == 3);
```

正規表現は文字列`MindYourPersonalSpace`の中にスペースがないことを見つける必要があります。

```js
assert('MindYourPersonalSpace'.match(countWhiteSpace) == null);
```
