# --explanation--

`Supposed to be` bedeutet, dass etwas erwartet oder geplant war. Zum Beispiel:

`The new feature was supposed to be ready by last week.` — Das bedeutet, es war geplant, letzte Woche fertig zu sein.

`Coming later` bedeutet, dass etwas in der Zukunft geplant ist. Zum Beispiel:

`The update is coming later this month.` — Das bedeutet, das Update wird zu einem späteren Zeitpunkt veröffentlicht, nicht jetzt.
