# --description--

Sie wissen, dass JavaScript verschiedene Datenarten hat, darunter Strings, Zahlen und Booleans. Sie können Vergleichsoperatoren verwenden, um diese Datenarten zu vergleichen und basierend auf den Ergebnissen Entscheidungen zu treffen.

Die folgenden Vergleichsoperatoren sind in JavaScript verfügbar:

| Operator | Beschreibung                            | Beispiel    |
|----------|----------------------------------------|-------------|
| ==       | Gleich (Wertvergleich)                  | x == y      |
| !=       | Ungleich                              | x != y      |
| >        | Größer als                           | x > y       |
| <        | Kleiner als                          | x < y       |
| >=       | Größer oder gleich                   | x >= y      |
| <=       | Kleiner oder gleich                  | x <= y      |

Um einen Vergleich in einer Variablenzuweisung zu verwenden, können Sie den `==` Operator nutzen. Zum Beispiel:

```javascript
let x = 5;
let y = 5;

let result = x == y;

console.log(result); // true
```
