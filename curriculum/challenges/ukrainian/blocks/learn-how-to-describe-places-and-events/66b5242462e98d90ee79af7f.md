# --explanation--

`Might` використовується, щоб висловити можливість або невпевненість. Це означає, що щось можливо, але не точно. Наприклад:

- `The system might crash if too many users log in at the same time.` - Існує ймовірність збою системи, але це не точно.

- `He might finish the coding task by the end of the day.` - Він міг би виконати завдання, але це не визначено.
