# --description--

Die `dispatch`-Methode verwenden Sie, um Aktionen an das Redux-Speichern zu dispatchen. Wenn Sie `store.dispatch()` aufrufen und den Wert übergeben, der von einem Action Creator zurückgegeben wird, wird eine Aktion an das Speichern gesendet.

Erinnern Sie sich, dass Action Creators ein Objekt mit einer Eigenschaft vom Typ type zurückgeben, die angibt, welche Art von Aktion stattgefunden hat. Dann dispatcht die Methode ein Aktionsobjekt an das Redux-Speichern. Basierend auf dem Beispiel der vorherigen Aufgabe sind die folgenden Zeilen äquivalent und dispatchen beide die Aktion vom Typ `LOGIN`:

```js
store.dispatch(actionCreator());
store.dispatch({ type: 'LOGIN' });
```

# --instructions--

Das Redux-Speichern im Code-Editor hat einen initialisierten Zustand, der ein Objekt mit einer `login`-Eigenschaft ist, die derzeit auf `false` gesetzt ist. Es gibt auch einen Action Creator namens `loginAction()`, der eine Aktion vom Typ `LOGIN` zurückgibt. Dispatchen Sie die `LOGIN`-Aktion an das Redux-Speichern, indem Sie die `dispatch`-Methode aufrufen und die Aktion übergeben, die von `loginAction()` erstellt wurde.
