# --explanation--

`Honestly` al comienzo de una oración se usa para expresar una opinión sincera, verdadera o directa. A menudo indica que la persona que habla está compartiendo sus pensamientos reales, incluso si no son positivos. Por ejemplo:

`Honestly, I don't think this plan will work.` - Esto significa que el hablante está dando su verdadera opinión sobre el plan.
