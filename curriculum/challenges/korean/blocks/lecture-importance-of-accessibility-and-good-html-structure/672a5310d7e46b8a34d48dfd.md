# --description--

화면 읽기 프로그램은 시각장애인과 저시력자가 컴퓨터와 모바일 기기를 사용할 수 있도록 돕는 보조 기술 프로그램입니다.

화면 읽기 프로그램은 단순히 시각장애인과 저시력자가 컴퓨터와 모바일 기기에 접근할 수 있게 하는 도구만은 아닙니다.

이들은 교육, 취업 기회, 소셜 미디어에 접근할 수 있도록 지원합니다. 이는 디지털 포용성을 보장하고 사회에 온전히 참여할 수 있는 능력을 향상시킵니다.

화면 읽기 프로그램이 텍스트 음성 변환 장치라는 오해가 흔히 있습니다.

하지만 텍스트 음성 변환은 화면 읽기 프로그램의 기능 중 하나일 뿐입니다. 일부 화면 읽기 프로그램은 음성 대신 점자 출력으로 텍스트를 렌더링하기도 합니다.

텍스트 음성 변환과 점자 출력 외에도 화면 읽기 프로그램의 주요 기능으로는 내비게이션 지원과 웹 브라우징 지원이 있습니다.

화면 읽기 프로그램은 시각장애인과 저시력자만을 위해 만들어진 것도 아닙니다. 난독증이 있는 사람과 인지 장애가 있는 사람들도 화면 읽기 프로그램을 사용합니다. 모든 주요 운영체제에는 화면 읽기 프로그램이 내장되어 있습니다.

macOS와 iOS에는 VoiceOver가 내장되어 있습니다. 컴퓨터에서 `CMD + F5`를 눌러 켤 수 있습니다. 아이폰에서는 설정에서 접근할 수 있습니다.

Windows 컴퓨터에는 Narrator가 내장되어 있습니다. `WIN + CTRL + ENTER`를 눌러 켤 수 있습니다. NonVisual Desktop Access (NVDA)와 Job Access With Speech (JAWS)도 Windows 컴퓨터에서 사용할 수 있습니다. NVDA는 무료 오픈소스이며, JAWS는 유료입니다.

Linux에는 데스크톱 환경용 _Orca_와 터미널용 _Speakup_이 있습니다.

Android 휴대폰에는 TalkBack이 내장되어 있습니다. _설정 > 특수 기능 > 접근성 > TalkBack_에서 켤 수 있습니다.

일부 Android 기기에는 Ella와 Select to Speak도 내장되어 있습니다.

화면 읽기 프로그램 사용자가 겪는 주요 도전 과제 중 하나는 많은 소프트웨어 개발자가 화면 읽기 프로그램 친화성과 접근성을 고려하지 않고 제품을 설계한다는 점입니다.

접근성은 광범위한 주제이지만, 모든 개발자는 시각장애인과 저시력자뿐 아니라 다른 장애인 그룹도 사용할 수 있도록 웹 소프트웨어를 만드는 방법을 배워야 합니다.

이는 공감을 보여주고 포용성을 약속하는 것으로, 모든 사용자가 자신의 작업에서 혜택을 누릴 수 있게 합니다.
