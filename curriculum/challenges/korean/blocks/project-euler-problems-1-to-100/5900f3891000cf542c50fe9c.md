# --description--

2 ≤ a ≤ 5 그리고 2 ≤ b ≤ 5인 모든 정수 조합의 $a^b$를 고려하세요:

<div style='padding-left: 4em;'>
  2<sup>2</sup>=4, 2<sup>3</sup>=8, 2<sup>4</sup>=16, 2<sup>5</sup>=32 <br>
  3<sup>2</sup>=9, 3<sup>3</sup>=27, 3<sup>4</sup>=81, 3<sup>5</sup>=243 <br>
  4<sup>2</sup>=16, 4<sup>3</sup>=64, 4<sup>4</sup>=256, 4<sup>5</sup>=1024 <br>
  5<sup>2</sup>=25, 5<sup>3</sup>=125, 5<sup>4</sup>=625, 5<sup>5</sup>=3125 <br>
</div>

이들을 숫자 순서대로 정렬하고 중복을 제거하면, 다음과 같은 15개의 서로 다른 항으로 이루어진 수열을 얻습니다:

<div style='padding-left: 4em;'>
  4, 8, 9, 16, 25, 27, 32, 64, 81, 125, 243, 256, 625, 1024, 3125
</div>

2 ≤ `a` ≤ `n` 그리고 2 ≤ `b` ≤ `n`인 $a^b$로 생성된 수열에 몇 개의 서로 다른 항이 있나요?

# --solutions--

```js
const distinctPowers = (n) => {
  let list = [];
  for (let a=2; a<=n; a++) {
    for (let b=2; b<=n; b++) {
      let term = Math.pow(a, b);
      if (list.indexOf(term)===-1) list.push(term);
    }
  }
  return list.length;
};
```
