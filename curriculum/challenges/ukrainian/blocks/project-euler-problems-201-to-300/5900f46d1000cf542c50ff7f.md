# --description--

Ми визначаємо округлений квадратний корінь додатного цілого числа $n$ як квадратний корінь з $n$, округлений до найближчого цілого числа.

Наступна процедура (фактично метод Герона, адаптований до цілочисельної арифметики) знаходить округлений квадратний корінь числа $n$:

Нехай $d$ — кількість цифр числа $n$.

Якщо $d$ непарне, встановіть $x_0 = 2 × {10}^{\frac{d - 1}{2}}$.

Якщо $d$ парне, встановіть $x_0 = 7 × {10}^{\frac{d - 2}{2}}$.

Повторюйте:

$$x_{k + 1} = \left\lfloor\frac{x_k + \left\lceil\frac{n}{x_k}\right\rceil}{2}\right\rfloor$$

доки $x_{k + 1} = x_k$.

Як приклад, знайдемо округлений квадратний корінь числа $n = 4321$.

У $n$ 4 цифри, отже $x_0 = 7 × {10}^{\frac{4-2}{2}} = 70$.

$$x_1 = \left\lfloor\frac{70 + \left\lceil\frac{4321}{70}\right\rceil}{2}\right\rfloor = 66 \\\\
x_2 = \left\lfloor\frac{66 + \left\lceil\frac{4321}{66}\right\rceil}{2}\right\rfloor = 66$$

Оскільки $x_2 = x_1$, зупиняємося. Таким чином, після лише двох ітерацій ми знайшли, що округлений квадратний корінь з 4321 дорівнює 66 (фактичний квадратний корінь — 65.7343137…).

Кількість ітерацій, необхідних при використанні цього методу, вражаюче мала. Наприклад, ми можемо знайти округлений квадратний корінь п’ятицифрового цілого числа ($10\,000 ≤ n ≤ 99\,999$) у середньому за 3.2102888889 ітерацій (середнє значення округлено до 10 десяткових знаків).

Використовуючи описану вище процедуру, яка середня кількість ітерацій потрібна, щоб знайти округлений квадратний корінь 14-значного числа (${10}^{13} ≤ n < {10}^{14}$)? Наведіть відповідь, округлену до 10 десяткових знаків.

**Примітка:** Символи $⌊x⌋$ та $⌈x⌉$ позначають функції підлоги та стелі відповідно.
