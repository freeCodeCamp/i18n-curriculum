# --description--

Come hai imparato nei moduli precedenti, i loop servono a ripetere un blocco di codice per un numero prestabilito di volte. In questa lezione imparerai a lavorare con diversi tipi di loop in Python.

Il primo loop che vedremo è il loop `for`. Ecco un esempio di utilizzo di un loop `for` per iterare su una lista e stampare ogni elemento nella console:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)
```

Il risultato sarà:

```md
Rust
Java
Python
C++
```

Nota che il `print(language)` è indentato all’interno del loop. Senza quell’indentazione, otterresti un `IndentationError`:

```python
"""
Traceback (most recent call last):
  File "<stdin>", line 4, in <module>
IndentationError: expected an indented block after 'for' statement on line 3
"""
```

Puoi anche usare un loop `for` per iterare su altri iterabili come una stringa. Ecco un esempio di utilizzo di un loop `for` per iterare sulla stringa `code` e stampare ogni carattere:

```py
for char in 'code':
    print(char)
```

Il risultato sarà:

```md
c
o
d
e
```

Proprio come in JavaScript, puoi anche annidare loop `for` in Python. Ecco un esempio di utilizzo di un loop `for` annidato:

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)
```

Il loop esterno itera su ogni `category` nella lista `categories`. Per ogni `category`, il loop interno itera su ogni `food` nella lista `foods`. Ecco il risultato che verrà stampato nella console:

```md
Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
```

Un altro tipo di loop che puoi usare in Python è il loop `while`. Questo tipo di loop ripete un blocco di codice finché la condizione è `False`. Ecco un esempio di utilizzo di un loop `while` per un gioco di indovinelli:

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')
```

In questo esempio abbiamo una variabile `secret_number` con valore `3` e un primo tentativo di `0`. Poi usiamo la funzione `input` per ottenere l’input dall’utente, quindi convertiamo la stringa di input in un intero con la funzione `int()` e la assegniamo alla variabile `guess`. Se l’utente indovina correttamente inserendo `3`, il loop `while` viene interrotto e il messaggio `You got it!` viene stampato nella console. Altrimenti, viene stampato il messaggio `Wrong! Try again.` e il loop si ripete, chiedendo all’utente di indovinare di nuovo.

Ecco un esempio di risultato:

```md
Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
```

Proprio come in JavaScript, Python supporta le istruzioni `break` e `continue`.

L’istruzione `break` serve a terminare l’esecuzione di un loop. Ecco un esempio di utilizzo dell’istruzione `break` per una lista di `developer_names`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

In questo esempio, iteriamo su una lista di `developer_names` e stampiamo ogni nome nella console. Se il nome è uguale a `Naomi`, allora usciamo dal loop. Questo fa sì che venga stampato nella console solo il nome `Jess`.

L’istruzione `continue` serve a saltare l’iterazione corrente di un loop e passare a quella successiva. Modifichiamo l’esempio precedente per usare l’istruzione `continue` invece di `break`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

Ora il risultato nella console sarà diverso. I nomi `Jess` e `Tom` vengono stampati perché l’istruzione `continue` salta la seconda iterazione del loop quando `developer` è uguale a `Naomi`, e non stampa quel nome nella console.

Sia i loop `for` che `while` possono essere combinati con una clausola `else`, che viene eseguita solo quando il loop non viene terminato da un’istruzione `break`. Ecco un esempio di utilizzo di più loop `for`:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

In questo esempio abbiamo una lista di parole casuali, e un loop `for` viene usato per iterare su ogni parola. All’interno del loop esterno `for`, abbiamo un altro loop `for` per iterare su ogni lettera di ogni parola. Se la versione minuscola della lettera è una vocale, stampiamo la parola seguita dalle vocali che contiene, poi usciamo dal loop interno. Se la parola non contiene vocali, stampiamo un messaggio che lo indica.

Ecco come appare il risultato nella console:

```md
'sky' has no vowels
'apple' contains the vowel 'a'
'rhythm' has no vowels
'fly' has no vowels
'orange' contains the vowel 'o'
```

I loop sono molto comuni in Python, quindi è importante prenderci confidenza. Nelle prossime lezioni imparerai a lavorare con le funzioni `enumerate()` e `range()` nei loop.
