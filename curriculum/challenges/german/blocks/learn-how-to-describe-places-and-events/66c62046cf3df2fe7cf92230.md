# --explanation--

Um die Antwort zu finden, achten Sie auf das Wort `but`, das einen Gegensatz oder eine Ausnahme signalisiert.

Nach `but` erwähnt Sarah `they're not visible`, gefolgt von der Bedingung `if the project isn't linked to Git.`.

Das zeigt Ihnen, dass die Tools verborgen sind, es sei denn, das Projekt ist mit Git verbunden.

Der Ausdruck `if the project isn't` ist ein wichtiger Hinweis, der die Bedingung einleitet, unter der die Git-Tools nicht erscheinen.

In Sarahs Satz verwendet sie `they're not`, um zu erklären, dass die Git-Tools nicht sichtbar sind, und `isn't`, um zu sagen, dass das Projekt nicht mit Git verknüpft ist.
