# --description--

Una forma muy común de aplicar color a un elemento con CSS es usando valores <dfn>hexadecimales</dfn> o hex. Aunque los valores hexadecimales parecen complicados, en realidad son solo otra forma de valores RGB.

Los valores de color hexadecimales comienzan con un carácter `#` y toman seis caracteres del 0-9 y A-F. El primer par de caracteres representa el rojo, el segundo par representa el verde y el tercer par representa el azul. Por ejemplo, `#4B5320`.

En el selector de clase `.green`, establece la propiedad `background-color` a un código de color hex con los valores `00` para rojo, `FF` para verde y `00` para azul.

# --hints--

La regla CSS `.green` debe tener la propiedad `background-color` establecida en `#00FF00`.

```js
assert(new __helpers.CSSHelp(document).getStyle('.green')?.backgroundColor === 'rgb(0, 255, 0)');
```
