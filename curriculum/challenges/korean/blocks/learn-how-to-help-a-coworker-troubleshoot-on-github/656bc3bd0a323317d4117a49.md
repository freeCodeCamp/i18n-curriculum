# --description--

과거 진행형은 과거에 진행 중이었던 동작에 대해 이야기할 때 사용합니다.

`was`를 사용해 `I`, `he`, `she`, `it`, 또는 `were` 대명사에 대해, `you`, `we`, `they` 대명사에 대해 `-ing`로 끝나는 동사를 뒤에 붙여 만듭니다. 예를 들어:

`I was reading`는 과거에 일정 기간 동안 일이 진행되고 있었음을 의미합니다.

과거 진행형은 종종 다른 동작에 의해 중단된 동작을 표현할 때 사용됩니다. 예를 들어 `I was reading when the phone rang.` 전화벨이 울려서 독서가 중단되었습니다.
