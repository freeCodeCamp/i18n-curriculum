# --explanation--

`Must`은 조동사이므로 다음 규칙을 따릅니다:

- 항상 동사의 기본형( `to` 없이 )이 뒤따릅니다.

- 3인칭 단수형에서 `-s`를 사용하지 않습니다.

- 의문문이나 부정문에서 `do/does`를 사용하지 않습니다.
