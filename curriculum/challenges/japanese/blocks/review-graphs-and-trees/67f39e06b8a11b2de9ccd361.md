## グラフの概要

グラフは、エッジ（接続）でつながれたノード（頂点）の集合です。各ノードは複数の他のノードと接続でき、ネットワークを形成します。グラフの種類には以下があります：

- 有向グラフ：エッジに方向があり（あるノードから別のノードへ）、直線と矢印で表されることが多いです。
- 無向グラフ：エッジに方向がなく、単純な線で表されます。
- 頂点グラフ：各ノードにラベルや識別子が関連付けられています。
- 循環グラフ：サイクル（同じノードで始まり終わる経路）を含みます。
- 非循環グラフ（DAG）：サイクルを含みません。
- エッジラベル付きグラフ：各エッジにラベルがあり、通常は対応するエッジの隣に描かれます。
- 重み付きグラフ：エッジに重み（値）が関連付けられており、算術演算に使えます。
- 非連結グラフ：エッジでつながれていない2つ以上のノードを含みます。

グラフは地図、ネットワーク、推薦システム、依存関係の解決など様々なアプリケーションで使われます。

## グラフの探索

グラフ内のすべてのノードを訪問する操作です。主なアルゴリズムは次の2つです：

- **幅優先探索（BFS）**
  - キューを使います。
  - レベルごとに探索します。
  - 重みなしグラフで最短経路を見つけます。

- **深さ優先探索（DFS）**
  - スタック（または再帰）を使います。
  - ある枝を完全に探索してから戻ります。
  - サイクル検出や経路探索に役立ちます。

## グラフの表現方法

グラフは主に2つの方法で表現できます：

- **隣接リスト**
  - 各ノードに隣接ノードのリストがあります。
  - 疎なグラフに対して空間効率が良いです。
  - 隣接ノードを繰り返すのが簡単です。

- **隣接行列**
  - 行と列がノードを表す2次元配列です。
  - 大きなグラフでは空間を多く使います。
  - 2つのノード間にエッジがあるかを高速に確認できます。

## 木

木は非循環かつ連結な特別な種類のグラフです。主な特徴は：

- ループやサイクル（開始ノードと終了ノードが同じ経路）がありません。
- 連結でなければならず（すべてのノードが他のすべてのノードから到達可能です）。

### 一般的な木の種類

最も一般的な木の種類は：

- 二分木
  - 各ノードは最大2つの子ノード（左と右）を持ちます。

- 二分探索木（BST）
  - 左の子ノードは親ノードより小さく、右の子ノードは親ノードより大きい二分木です。

## トライ木

接頭辞木とも呼ばれ、文字列の集合を格納するために使われます。各ノードは1文字を表します。

共通の接頭辞は一度だけ格納されるため、オートコンプリートやスペルチェックなどに効率的です。

検索や挿入の時間計算量は文字列の長さLに対してO(L)です。

## 優先キュー

優先キューは各要素に優先度がある抽象データ型です。

キューやスタックは挿入順序のみを考慮しますが、優先キューは要素の優先度を考慮します。

標準のキューはFIFO（先入れ先出し）、スタックはLIFO（後入れ先出し）ですが、優先キューでは優先度の高い要素が挿入順に関係なく先に処理されます。

## ヒープ

ヒープは特定のヒープ特性を持つ特殊な木構造のデータ構造です。

ヒープ特性は親ノードと子ノードの関係を決定します。ヒープには2種類あります：

- 最大ヒープ
  - 各親ノードの値は子ノードの値以上です。
  - 最大の要素がルートにあります。

- 最小ヒープ
  - 各親ノードの値は子ノードの値以下です。
  - 最小の要素がルートにあります。

### Python `heapq` モジュールの例

```py
import heapq

# Create empty heap
my_heap = []

# Insert elements
heapq.heappush(my_heap, 9)
heapq.heappush(my_heap, 3)
heapq.heappush(my_heap, 5)
print(my_heap) # [3, 9, 5]

# Remove smallest element
print(heapq.heappop(my_heap))  # 3
print(my_heap) # [5, 9]

# Push + Pop in one step
print(heapq.heappushpop(my_heap, 7)) # 5
print(my_heap) # [7, 9]

# Transform list into heap
nums = [5, 7, 3, 1]
heapq.heapify(nums)
```

### 優先度の使用例

```py
my_heap = []
heapq.heappush(my_heap, (3, "A"))
heapq.heappush(my_heap, (2, "B"))
heapq.heappush(my_heap, (1, "C"))

# Removes lowest number = highest priority
print(heapq.heappop(my_heap))  # (1, "C")
```
