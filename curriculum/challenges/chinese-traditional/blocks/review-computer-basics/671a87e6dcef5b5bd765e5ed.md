## 理解電腦、網際網路與開發工具基礎知識

- **主機板**：容納所有需要用來執行電腦的記憶體、連接器和硬碟。它作為電腦的主要電路板。
- **中央處理器(CPU)**：負責執行指令和進行計算的處理器。
- **隨機存取記憶體(RAM)**：電腦 CPU 的臨時儲存位置。
- **硬碟驅動器(HDD)**：一個永久的儲存位置，用來在電腦關機時仍然儲存資料。
- **固態硬碟(SSD)**：非易揮發性的快閃記憶體，可用來取代硬碟。
- **電源供應器(PSU)**：負責將牆壁插座的電力轉換成電腦可以使用的形式。
- **圖形處理器(GPU)**：負責在電腦螢幕上呈現視覺效果。
- **不同類型的網際網路服務提供者**：網際網路服務提供者（ISP）是提供網際網路存取的公司。ISP 有不同的類型，包括撥號、DSL、有線、光纖和衛星。
- **安全登入你的電腦的方法**：安全登入你的電腦的方法範例包括使用強密碼、啟用雙重驗證，以及使用密碼管理工具。
- **整合開發環境 (IDE)**：一種幫助開發者以高效方式撰寫、測試和偵錯程式碼的工具。
- **程式碼編輯器**：開發者用來撰寫和偵錯程式碼的工具。
- **Git**：一個流行的版本控制系統，讓開發者能追蹤他們程式碼中的變更並與他人協作。
- **用於儲存庫的雲端託管服務**：儲存庫是專案檔案和版本歷史的儲存位置。常見的用於儲存庫的雲端託管服務包括 GitHub、GitLab 和 Bitbucket。
- **軟體包管理工具**：幫助開發者簡化為專案添加、更新和移除函式庫及相依性的處理程序的工具。範例包括 npm、pip 和 Maven。
- **測試函式庫和框架**：在軟體中進行測試，以確保程式碼如預期般運作。測試函式庫和框架的範例包括 Jest、PHPUnit 和 JUnit。

## 處理檔案、檔案系統和媒體格式

- **命名檔案的最佳實務**：你會想以易於理解和維護的方式命名你的檔案。舉例來說，`about-us.html` 比 `page1.html` 更具描述性。
- **根目錄**：檔案系統中的最高層目錄。目錄是資料夾的另一種稱呼。
- **Markdown**：一種常用於文件和 `README` 檔案的標記語言。`README` 檔案是一種包含專案相關資訊的檔案，例如如何安裝和使用它。
- **`index.html`**：表述使用者造訪網站時所顯示的預設頁面。
- **使用 Explorer/Finder 創建、移動和刪除檔案和資料夾**：Explorer 是 Windows 中的檔案管理員，Finder 是 macOS 中的檔案管理員。你可以使用這些工具來創建、移動和刪除檔案和資料夾。
- **搜尋檔案和資料夾**：你可以使用 Explorer 或 Finder 中的搜尋功能來尋找你電腦上的檔案和資料夾。
- **HTML、CSS 和 JS 檔案類型**：`.html` 副檔名用於 HTML 檔案，`.css` 用於 CSS 檔案，`.js` 用於 JavaScript 檔案。
- **常見的影像與圖形格式**：`JPEG` 和 `PNG` 是常見的影像檔案格式。`GIF` 是另一種支援動畫的常見影像檔案格式。`SVG` 是向量圖形的檔案格式。
- **常見的音訊和影片格式**：`MP3` 格式常用於音訊檔案。`MP4` 格式常用於影片檔案。`MOV` 格式由 Apple 開發，常用於影片檔案。
- **常見字型格式**：`TTF` 格式常用於 TrueType 字型。`WOFF` 格式常用於網頁字型。`WOFF` 的後繼者是 `WOFF2`，提供更佳的壓縮效果。
- **ZIP**：一種用來壓縮檔案和資料夾的檔案格式。

## 有效瀏覽網頁

- **什麼是網頁瀏覽器？**：一種軟體應用程式，允許使用者在網際網路上存取並瀏覽網站。
- **什麼是 Search Engine**：一種讓使用者能在網際網路上搜尋資訊的工具。範例包括 Google、Bing 和 Yahoo。
- **常見的網頁瀏覽器**：一些常見的網頁瀏覽器範例包括 Google Chrome、Mozilla Firefox 和 Microsoft Edge。
- **常見搜尋策略**：你可以使用 `site:` 後接網站的 URL 來搜尋該網站上的內容。你可以使用 `filetype:` 後接副檔名來搜尋該類型的檔案。你可以在搜尋詞前置式加上減號，以排除包含該詞的結果。你可以在搜尋詞前置式加上加號，以包含包含該詞的結果。
