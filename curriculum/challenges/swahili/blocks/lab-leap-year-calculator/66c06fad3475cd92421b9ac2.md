# --description--

Mwaka wa kuruka ni mwaka unaogawika kwa `4`, isipokuwa kwa miaka inayogawika kwa `100` na isiyogawika kwa `400`. Kwa mfano, `2000` ni mwaka wa kuruka, lakini `1900` siyo. Pia, mwaka wa kuruka una siku ya ziada mwezi wa Februari, ambayo ni siku ya 29 ya mwezi huo.

**Lengo:** Timiza hadithi za watumizi zilizo hapa chini na upite vipimo vyote ili kumaliza maabara.

**Hadithi za watumizi:**

1. Tangaza kitendakazi kinachoitwa `isLeapYear` kinachopokea nambari kama hoja.
2. Nje ya kitendakazi, tangaza kigezo `year` kinachohifadhi thamani ya mwaka unaotaka kukagua.
3. Ndani ya kitendakazi, tumia tamko la `if/ else` au opereta ya ternary kukagua kama mwaka ni mwaka wa kuruka.
4. Ili kukagua kama mwaka ni mwaka wa kuruka, timiza masharti yafuatayo:

- Ikiwa mwaka unagawika kwa `4`, basi ni mwaka wa kuruka.
   - Isipokuwa mwaka pia unagawika kwa `100`, basi siyo mwaka wa kuruka.
   - Isipokuwa mwaka pia unagawika kwa `400`, basi ni mwaka wa kuruka.

5. Ikiwa mwaka ni mwaka wa kuruka, rudisha `[year] is a leap year.`. Vinginevyo, rudisha `[year] is not a leap year.`. Utabadilisha `[year]` na kigezo kilichotangazwa katika kitendakazi cha `isLeapYear`.
6. Unapaswa kuita kitendakazi cha `isLeapYear` na kutoa `year` kama hoja na weka matokeo katika kigezo kinachoitwa `result`.
7. Unapaswa toa thamani ya kigezo `result` kwenye konsoli kwa kutumia `console.log()`.

function isLeapYear(year) {
  if ((year % 4 === 0 && year % 100 !== 0) || year % 400 === 0) {
    return(year + " is a leap year.");
  } else {
    return(year + " is not a leap year.");
  }
}

const year = 2024;
const result = isLeapYear(year);
console.log(result);
