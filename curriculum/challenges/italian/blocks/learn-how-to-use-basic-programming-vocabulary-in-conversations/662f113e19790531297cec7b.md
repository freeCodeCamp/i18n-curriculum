# --description--

`Object-oriented programming` (OOP) è un paradigma di programmazione basato sul concetto di `objects`, che può contenere dati e codice: dati sotto forma di campi (spesso noti come attributi o proprietà) e codice sotto forma di procedure (spesso noti come metodi).

Un esempio comune di `OOP` è creare una classe in un linguaggio di programmazione come Java o Python. Questa classe definisce la struttura degli oggetti, inclusi i loro attributi e metodi. Per esempio, una classe `Car` potrebbe includere attributi come `color` e `model`, e metodi come `drive()` e `brake()`.

Il termine `oriented` in questo contesto significa allineato o diretto verso uno scopo o principio particolare.

Per esempio, il curriculum di design web responsivo di freeCodeCamp è descritto come `project-oriented` perché è strutturato attorno al completamento di progetti, il che significa che il metodo di insegnamento è focalizzato sull'applicazione pratica piuttosto che solo sulla conoscenza teorica.
