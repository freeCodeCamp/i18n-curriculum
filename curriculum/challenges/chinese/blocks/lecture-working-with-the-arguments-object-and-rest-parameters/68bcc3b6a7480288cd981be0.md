# --interactive--

正如你从之前的课程和研讨会中回忆的那样，你可以创建带有多个参数的函数，并使用参数调用该函数。

这是一个提醒：

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call
getSum(3, 4); // 7
```

但是如果你有一个函数被调用时传入的参数比它定义时接受的参数更多，该怎么办？

:::interactive_editor

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call with extra argument
console.log(getSum(3, 4, 5)); // 7
```

:::

JavaScript 在这种分支中不会抛出误差。它会忽略多余的参数，只将数字 `3` 和 `4` 相加。接受可变数量参数的函数称为 <dfn>variadic functions</dfn>。

如果你正在使用可变参数函数，那么你可以利用 `arguments` 对象。这个类似数组的对象包含传入函数的参数值。

这是一个示例：

:::interactive_editor

```js
function logArgs() {
  for (const arg of arguments) {
    console.log(arg);
  }
}

logArgs(1, 2, 3);
// result:
// 1
// 2
// 3

logArgs("example"); // "example"
```

:::

由于 `arguments` 对象类似数组，你可以这样访问特定索引处的参数：

:::interactive_editor

```js
function getArg() {
  return arguments[1];
}

console.log(getArg(2, 4, 6)); // 4
```

:::

你也可以像这样使用 `length` 属性来获取函数被调用时的参数数量：

:::interactive_editor

```js
function getArgs() {
  return arguments.length;
}

console.log(getArgs("Example")); // 1
console.log(getArgs("Another", "Example")); // 2
```

:::

尽管 `arguments` 对象看起来像一个真正的数组，但它没有内置的 `Array` 方法，比如 `includes` 或 `push`。要访问这些方法，你需要先使用类似 `slice`、`Array.from()` 或扩展操作符将 `arguments` 对象转换为真正的数组：

:::interactive_editor

```js
function hasCat() {
  return [...arguments].includes("cat");
}

console.log(hasCat("dog", "chicken", "cat")); // true
console.log(hasCat("dog", "chicken", "horse")); // false
```

:::

虽然可以使用 `arguments` 对象来处理可变参数函数，但现代 JavaScript 应用通常会使用剩余参数语法。你将在后续课程中学习更多内容。
