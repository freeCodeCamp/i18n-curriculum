# --explanation--

質問`Did you check that?`は、誰かが何かを確認または検証したかどうかを尋ねています。

この文脈では、マークはサラが電源ケーブルの接続が確実かどうかを確認したか知りたいのです。ほかの例：

- `Did you check that the windows are closed?` - これは誰かが窓がしっかり閉まっていることを確認したか尋ねています。

- `Did you check that the light is off?` - これは誰かがライトが消えていることを確認したか尋ねています。
