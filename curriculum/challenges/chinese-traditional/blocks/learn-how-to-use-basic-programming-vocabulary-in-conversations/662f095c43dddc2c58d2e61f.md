# --description--

字 `super` 通常非正式地用在形容詞前，以加強其意義，類似於 `very` 或 `extremely`，用來強化後方形容詞的意義。

例如，說 `this book is super interesting` 意味著這本書非常有趣，比單純有趣還要多。

`Helpful` 描述某人或某物提供有價值的協助或支援。合起來，`super helpful` 意味著提供卓越的協助或非常有用。
