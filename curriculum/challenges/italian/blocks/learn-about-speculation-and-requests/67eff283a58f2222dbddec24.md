# --explanation--

`Have you checked...?` si usa quando si chiede se qualcuno ha fatto qualcosa in qualsiasi momento prima di ora. Per esempio:

- `Have you checked your email?` - Qui si chiede se la persona ha guardato l'email in qualsiasi momento prima di ora.

- `Have you checked if the server is running?` - Qualcuno chiede se hanno verificato lo stato del server.
