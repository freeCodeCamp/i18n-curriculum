# --description--

インライン`CSS`を使うと、`HTML`要素に直接スタイルを追加できますが、この方法はあまり推奨されていません。

```html
<body>
  <div style="color: white; background-color: black;">...</div>
</body>
```

まず注目すべきは、ここにはセレクターが一切ないことです。スタイルは開く`<div>`タグ自体に直接追加されているからです。次に、`style`属性があり、その値は引用符で囲まれた中に宣言が入っています。

もし単一の要素にユニークなスタイルを追加する必要がある場合、この方法は十分に機能します。ただし、一般的にはいくつかの理由でHTMLにCSSを追加する推奨方法ではありません。

- 1つの要素に多くの宣言を追加し始めると、すぐにかなり散らかってしまい、HTMLファイルが不必要に膨れ上がってしまいます。
- 多くの要素に同じスタイルを適用したい場合、同じスタイルをそれぞれの要素にコピー＆ペーストしなければならず、無駄な繰り返しや膨張が増えます。
- インラインCSSは他の2つの方法（内部および外部）をオーバーライドするため、予期しない結果を引き起こす場合があります。（ここでは詳しく扱いませんが、実際にはこれを利用することもできます）。
