# --description--

现在 `div` 元素被放置在预览屏幕的左上角。但为元素顶部添加一些间距会更好。要使用 Tailwind CSS 为元素应用 `margin`，你可以使用以下模式：`m-number`，其中 `number` 表示应用于四个边的固定像素。

如果你只想为一侧应用边距，可以使用以下任意模式：`mt-number`（上边距）、`mb-number`（下边距）、`mr-number`（右边距）和 `ml-number`（左边距）。

这是一个将 `margin-bottom` 应用到元素的示例：

```html
<p class="mb-8">This is margin.</p>
```

在你的开头 `div` 标签内，使用正确的实用类来应用 `margin-top` 为 `4px`。

# --hints--

你的开头 `div` 标签应具有 `mt-4` 实用类。

```js
const divEl = document.querySelector("div");
assert.isTrue(divEl.classList.contains("mt-4"));
```
