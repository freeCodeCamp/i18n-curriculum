# --interactive--

## 반응형 웹 디자인

- **정의**: 반응형 디자인의 핵심 원칙은 적응성입니다 – 웹사이트가 보는 기기의 화면 크기와 기능에 따라 레이아웃과 콘텐츠를 조정하는 능력입니다.
- **유동 그리드**: 픽셀 같은 고정 단위 대신 백분율 같은 상대 단위를 사용하여 화면 크기에 따라 콘텐츠가 크기 조절되고 재배치되도록 합니다.
- **유연한 이미지**: 컨테이너 내에서 크기 조절되도록 설정되어 작은 화면에서 컨테이너를 넘치지 않도록 보장합니다.

## 미디어 쿼리

- **정의**: 개발자가 주로 뷰포트 너비를 기준으로 기기의 특성에 따라 다른 스타일을 적용할 수 있게 합니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="mq-example">The background of this paragraph will change when the viewport is at least 768px wide.</p>
```

```css
.mq-example {
  padding: 10px;
  border: 1px solid #ccc;
  background-color: lightyellow;
}

@media screen and (min-width: 768px) {
  /* Styles for screens at least 768px wide */
  .mq-example {
    background-color: lightblue;
  }
}
```

:::

- **`all` 미디어 타입**: 모든 기기에 적합합니다. 미디어 타입이 지정되지 않으면 기본값입니다.
- **`print` 미디어 타입**: 페이지 매체와 인쇄 미리보기 모드에서 화면에 표시되는 문서에 적합합니다.
- **`screen` 미디어 타입**: 주로 화면용입니다.
- **`aspect-ratio`**: 뷰포트의 너비와 높이 비율을 설명합니다.

```css
@media screen and (aspect-ratio: 16/9) {
  /* Styles for screens with a 16:9 aspect ratio */
}
```

- **`orientation`**: 기기가 가로 또는 세로 방향인지 나타내는 데 사용됩니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="orientation-example">This paragraph's background will change in landscape orientation.</p>
```

```css
.orientation-example {
  padding: 10px;
  border: 1px solid #ccc;
  background-color: lightyellow;
}

@media screen and (orientation: landscape) {
  /* Styles for landscape orientation */
  .orientation-example {
    background-color: lightgreen;
  }
}
```

:::

- **`resolution`**: 출력 기기의 해상도를 인치당 도트(dpi) 또는 센티미터당 도트(dpcm)로 설명하는 데 사용됩니다.

```css
@media screen and (min-resolution: 300dpi) {
  /* Styles for high-resolution screens */
}
```

- **`hover`**: 기본 입력 장치가 요소 위에 마우스를 올릴 수 있는지 테스트하는 데 사용됩니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button>Hover over me</button>
```

```css
button {
  padding: 10px 20px;
  font-size: 1rem;
  background-color: lightgray;
  border: 1px solid #999;
  cursor: pointer;
}

@media (hover: hover) {
  /* Styles for devices that support hover */
  button:hover {
    background-color: yellow;
  }
}
```

:::

- **`prefers-color-scheme`**: 사용자가 밝은 테마 또는 어두운 테마를 요청했는지 감지하는 데 사용됩니다.
- **미디어 쿼리와 논리 연산자**: `and` 연산자는 여러 미디어 기능을 결합하는 데 사용되며, `not`와 `only`은 미디어 쿼리를 부정하거나 분리하는 데 사용할 수 있습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="logical-example">This paragraph's background will change when the screen is at least 768px wide AND in landscape orientation.</p>
```

```css
.logical-example {
  padding: 10px;
  border: 1px solid #ccc;
  background-color: lightyellow;
}

@media screen and (min-width: 768px) and (orientation: landscape) {
  /* Styles for landscape screens at least 768px wide */
  .logical-example {
    background-color: lightcoral;
  }
}
```

:::

## 일반적인 미디어 브레이크포인트

- **정의**: 미디어 브레이크포인트는 웹사이트 디자인에서 레이아웃과 콘텐츠가 다양한 화면 크기에 맞게 조정되는 특정 지점입니다. 휴대폰, 태블릿, 데스크톱 화면을 대상으로 하는 일반적인 브레이크포인트가 있지만, 모든 기기의 모든 화면 크기를 다 쫓아가는 것은 현명하지 않습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="breakpoint-example">This text will change size when the viewport width is at least 768px.</p>
```

```css
.breakpoint-example {
  font-size: 1rem;
  padding: 10px;
  border: 1px solid #ccc;
}

/* Styles for screens wider than 768px */
@media screen and (min-width: 768px) {
  .breakpoint-example {
    font-size: 1.125rem;
    background-color: lightblue;
  }
}
```

:::

- **소형 기기(스마트폰)**: 최대 640px
- **중간 기기(태블릿)**: 641px에서 1024px
- **대형 기기(데스크톱)**: 1025px 이상

## 모바일 우선 접근법

- **정의**: 모바일 우선 접근법은 반응형 웹 디자인에서 모바일 기기를 우선으로 웹사이트를 만들고, 그 다음에 더 큰 화면을 위한 디자인을 하는 철학이자 개발 전략입니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <p>The width of this container changes based on the viewport size.</p>
</div>
```

```css
/* Base styles for mobile */
.container {
  width: 100%;
  padding: 10px;
  background-color: lightblue;
}

/* Styles for larger screens */
@media screen and (min-width: 768px) {
  .container {
    width: 750px;
    margin: 0 auto;
    padding: 20px;
  }
}

@media screen and (min-width: 1024px) {
  .container {
    width: 960px;
  }
}
```
