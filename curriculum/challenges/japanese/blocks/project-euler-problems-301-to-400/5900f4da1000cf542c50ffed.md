# --description--

2人のプレイヤー、Anton と Bernhard が次のゲームをしています。

石の山が1つあり、石の数は $n$ 個です。

最初のプレイヤーは正の数の石を取り除けますが、山全体を取ることはできません。

その後は、各プレイヤーが前の手で相手が取った石の数の最大2倍まで取り除けます。

最後の石を取ったプレイヤーが勝ちです。

例：$n = 5$

最初のプレイヤーが1個より多く取ると、次のプレイヤーは残りの石をすべて取れます。

最初のプレイヤーが1個取って4個残すと、相手も1個取り、3個残ります。

最初のプレイヤーは3個すべて取れません。なぜなら最大で $2 \times 1 = 2$ 個までしか取れないからです。たとえば1個取って2個残します。

次のプレイヤーは残りの2個を取り勝ちます。

したがって、5は最初のプレイヤーにとって負けの状態です。

勝ちの状態では、最初のプレイヤーが複数の手を選べる場合があります。

例：$n = 17$ のとき、最初のプレイヤーは1個または4個の石を取れます。

$M(n)$ を、最初のプレイヤーが勝ちの状態で最初の手に取れる最大の石の数とし、その他の状態では $M(n) = 0$ とします。

$n ≤ 100$ のときの $\sum M(n)$ は 728 です。

$n ≤ {10}^{18}$ のときの $\sum M(n)$ を求めてください。答えは ${10}^8$ でモジュラスを取ってください。
