# --description--

値を正しくソートする必要があります。`.sort()` の呼び出しに `Object.keys()` メソッドをチェーンしてください。

コールバックには、各キーの値を比較するために `counts` オブジェクトを使う必要があります。キーにアクセスするには `a` と `b` のパラメータを使えます。次に、`counts[b]` から `counts[a]` を引いた値を返してください。

最後に、ブラケット記法を使って配列の最初の要素にアクセスし、`highest` 変数を完成させてください。

const getMean = (array) => array.reduce((acc, el) => acc + el, 0) / array.length;

const getMedian = (array) => {
  const sorted = array.toSorted((a, b) => a - b);
  const median =
    sorted.length % 2 === 0
      ? getMean([sorted[sorted.length / 2], sorted[sorted.length / 2 - 1]])
      : sorted[Math.floor(sorted.length / 2)];
  return median;
}
