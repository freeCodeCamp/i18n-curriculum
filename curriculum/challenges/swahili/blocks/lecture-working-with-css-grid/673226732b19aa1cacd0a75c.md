# --interactive--

Katika funzo lililopita, ulijifunza kuhusu gridi ya CSS ambayo inaweza kutumika kuunda mipangilio tata na inayotiririka kwenye kurasa zako za mtandao. Katika funzo hili, tutaangalia jinsi ya kuunda mipangilio ya gridi inayojibadilisha kwa kutumia kitengo cha `fr`.

Tuanze na alama hizi za HTML ambazo zitawakilisha chombo chetu cha gridi:

```html
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

Ndani ya CSS, tumeweka sifa ya display kuwa grid kwa kontena.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 25% 25% 25% 25%;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

`grid-template-columns` hutumika kuweka ukubwa wa kila safu. Katika kesi hii, ukubwa wa kila safu utakuwa `25%` ya kontena. Kisha sifa ya gap hutumika kuunda nafasi kati ya kila safu.

Hadi sasa tumekuwa tukitumia asilimia kwa ukubwa wa safu lakini pia tunaweza kutumia kitengo cha `fr`.

Kitengo cha `fr` ni kitengo cha sehemu kinachowakilisha sehemu ya nafasi ya chombo cha gridi.

Hivi ndivyo msimbo utakavyoonekana ukirekebishwa kutumia vitengo vya `fr` badala ya asilimia.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

Kila safu itachukua sehemu moja ya nafasi iliyopo. Kwa kuwa kuna safu nne, kila safu itakuwa na sehemu sawa ya nafasi iliyopo ndani ya chombo cha gridi.

Unapoanza kujenga mipangilio yako ya gridi, utagundua unataka kutumia vitengo vya `fr` mara nyingi zaidi kwa sababu vinatoa njia inayojibadilisha, ya uwiano wa kugawanya nafasi, ikikuruhusu kuunda mipangilio inayojibadilisha kulingana na kifaa inayobadilika kulingana na ukubwa wa skrini bila hitaji la kurekebisha thamani za pikseli kwa mkono.
