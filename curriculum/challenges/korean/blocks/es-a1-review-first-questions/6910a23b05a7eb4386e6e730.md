# --description--

축하합니다! 이 모듈을 거의 완료했습니다.

아래 글은 과제에서 다룬 주요 문법 사항을 검토하는 데 도움이 되어, 모듈 퀴즈를 보기 전에 자신감을 가질 수 있게 합니다.

## 출신지 말하기

출신지를 말할 때는 **`Soy` + `de` + [country]**를 사용하세요. 예를 들어:

- `Soy de El Salvador.`

- `Soy de Guatemala.`

누군가의 출신지를 알고 싶으면 `¿De dónde eres?`라고 물어보세요.

<br>

## 국적 말하기

자신의 국적을 말할 때는 **`Soy` + [nationality]**를 사용하세요.

- `Soy salvadoreña.`

- `Soy guatemalteco.`

지금까지 배운 국적은 다음과 같습니다:

| 국가 | 남성형 | 여성형 |
| :--- | :--- | :--- |
| **`El Salvador`** | `salvadoreño` | `salvadoreña` |
| **`Guatemala`** | `guatemalteco` | `guatemalteca` |
| **`Colombia`** | `colombiano` | `colombiana` |
| **`Argentina`** | `argentino` | `argentina` |
| **`Uruguay`** | `uruguayo` | `uruguaya` |
| **`Puerto Rico`** | `puertorriqueño` | `puertorriqueña` |
| **`Chile`** | `chileno` | `chilena` |
| **`Perú`** | `peruano` | `peruana` |

스페인어에서 국적은 보통 성별에 따라 달라집니다. 국적이 `-o`로 끝나면, 여성형을 만들 때 보통 이 마지막 글자를 `-a`로 교체합니다.

하지만 자음으로 끝나는 국적은 여성형을 만들 때 `-a`를 추가하는 경우가 많습니다. 예를 들어, `España`("스페인") 출신 남성형은 `español`이고, 여성형은 `española`입니다.

`-e` 또는 `-í`로 끝나는 국적은 보통 남녀 구분 없이 같습니다. 예를 들어, `Estados Unidos`("미국") 출신 남성형은 `estadounidense`이고, 여성형도 `estadounidense`입니다.

<br>

## 직업 말하기

자신의 직업을 말할 때는 **`Soy` + [profession]**를 사용하세요. 예를 들어:

- `Soy representante de recursos humanos.`

- `Soy analista junior.`

누군가의 직업을 물어볼 때는 `¿A qué te dedicas?`라고 질문하세요.

<br>

## 나이 말하기

스페인어로 나이를 말할 때는 **`Tengo` + [number] + `años`**를 사용하세요. 예를 들어:

- `Tengo veintiún años.`

- `Tengo treinta y cinco años.`

누군가의 나이를 물어볼 때는 `¿Cuántos años tienes?`라고 질문하세요.

<br>

## 소개할 때 공손하게 말하기

스페인어에는 세 가지 중요한 공손 표현이 있습니다.

**`Mucho gusto`**는 "만나서 반갑습니다"라는 뜻입니다. 누군가를 만나서 기쁘다는 공손한 표현이며, 성별에 상관없습니다.

**`Encantada`**도 "만나서 반갑습니다"라는 뜻이지만, 더 감정을 담아 만나서 매우 기쁘다는 뜻입니다. 말하는 사람의 성별에 맞춰야 하며, 여성형은 `Encantada`, 남성형은 `Encantado`입니다.

**`Igualmente`**는 "저도 마찬가지입니다"라는 뜻입니다. 상대방이 표현한 말을 반복하지 않고 공손하게 되돌려 줄 때 사용합니다.
