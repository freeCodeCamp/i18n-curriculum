# --explanation--

`Trying to` significa hacer un esfuerzo para hacer algo. Por ejemplo:

`She's trying to finish the report before noon.` – Esto muestra que está trabajando duro para hacerlo.

`Monitor` es un dispositivo que muestra imágenes y se usa comúnmente con computadoras. Por ejemplo:

`He connected a second monitor to his laptop.` – Esto significa que añadió otra pantalla.

`They've` es la contracción de `they have`. Es el tiempo Presente Perfecto que muestra una acción realizada antes de ahora o una situación que comenzó en el pasado y continúa.​ Por ejemplo:

`They've moved the files to a new folder.` – Esto significa que ellos trasladaron los archivos a una nueva carpeta.
