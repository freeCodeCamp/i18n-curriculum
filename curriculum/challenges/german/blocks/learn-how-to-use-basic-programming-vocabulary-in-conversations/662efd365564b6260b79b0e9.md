# --description--

Eine `loop` ist eine Folge von Anweisungen, die kontinuierlich wiederholt wird, bis eine bestimmte Bedingung erreicht ist. `Loops` werden verwendet, um denselben Code effizient mehrfach auszuführen. Häufige Typen von `loops` sind `for`-Schleifen und `while`-Schleifen.

Zum Beispiel `for loop`: `for (int i = 0; i < 5; i++) { /* code to be executed */ }`

In dem Beispiel wird der Code innerhalb der geschweiften Klammern `{ }` wiederholt ausgeführt, bis die Bedingung nicht mehr wahr ist.
