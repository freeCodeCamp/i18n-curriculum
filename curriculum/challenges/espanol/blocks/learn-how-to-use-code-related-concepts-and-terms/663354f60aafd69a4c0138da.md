# --description--

El `símbolo de igualdad` (`=`) se usa comúnmente para la asignación. Se usa para asignar un valor a una variable. El nombre de la variable se escribe a la izquierda del símbolo de igualdad, y el valor que deseas almacenar en la variable está a la derecha.

Por ejemplo, `name = "Sophie";` asigna la cadena `"Sophie"` a la variable `name`.

`Asignar un valor` en programación significa dar un valor específico a una variable. El `símbolo de igualdad` (`=`) se usa comúnmente para este propósito. Cuando asignas un valor a una variable, estás diciendo al programa que recuerde ese valor para su uso posterior.

Por ejemplo, `age = 25` asigna el valor `25` a la variable `age`.
