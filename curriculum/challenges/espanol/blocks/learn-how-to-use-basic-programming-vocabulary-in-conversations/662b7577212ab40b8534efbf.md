# --description--

`Pretty well` significa un alto grado o bastante bien pero no perfecto. Es una forma de decirle a alguien que tú o alguien más puede hacer algo bastante bien, pero todavía hay espacio para mejorar.

Por ejemplo, `I did pretty well in the English test. I got most of the answers right.`
