# --explanation--

يمكن أن يعني `To touch on` ذكر موضوع أو التعامل معه بشكل موجز. يُستخدم غالبًا عند تغطية موضوع دون الخوض في تفاصيل كثيرة. على سبيل المثال:

- `The speaker touched on the new software update during the meeting.` - ذكر المتحدث تحديث البرنامج بشكل موجز لكنه لم يدخل في التفاصيل.
