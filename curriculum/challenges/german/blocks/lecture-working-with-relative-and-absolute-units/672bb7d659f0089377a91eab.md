# --interactive--

Prozentangaben in CSS sind relative Einheiten, mit denen Sie Größen, Abmessungen und andere Eigenschaften als Anteil ihres übergeordneten Elements definieren können. Wenn Sie einen Prozentwert verwenden, sagen Sie im Grunde: „Machen Sie das zu X % seines Containers.“

Wenn Sie zum Beispiel `width: 50%` für ein Element festlegen, nimmt es die halbe Breite seines übergeordneten Containers ein. Das macht Prozentangaben besonders nützlich, um responsives Design zu erstellen, das sich an verschiedene Bildschirmgrößen anpasst.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <div class="box"></div>
</div>
```

```css
.container {
  width: 400px;
  height: 200px;
  background-color: lightgray;
}

.box {
  width: 50%;
  height: 100%;
  background-color: red;
}
```

:::

Prozentangaben eignen sich ideal für flüssige Layouts, die sich an verschiedene Bildschirmgrößen anpassen. Wenn Sie beispielsweise einen Container auf `width: 80%` setzen, nimmt er 80 % der Breite seines Elternteils ein, unabhängig vom Gerät.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="parent">
  <div class="child"></div>
</div>
```

```css
.parent {
  width: 100%;
  height: 300px;
  background-color: lightblue;
}

.child {
  width: 80%;
  height: 100%;
  background-color: red;
}
```

:::

Ein weiterer häufiger Anwendungsfall für Prozentangaben sind flexible Bilder. Wenn Sie `max-width: 100%` auf Bilder anwenden, können diese auf kleineren Bildschirmen verkleinert werden, während ihr Seitenverhältnis erhalten bleibt.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<img src="https://placehold.co/150x150" alt="Example Product Image" />
```

```css
img {
  max-width: 100%;
  height: auto;
}
```

:::

Weniger häufig, aber möglich, ist die Verwendung von Prozentangaben für Schriftgrößen, um skalierbare Typografie zu erzeugen. Zum Beispiel würde `font-size: 120%` den Text 20 % größer als die Schriftgröße des Elternteils machen.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="text-container">
  Parent text.
  <p class="text">This is some example text.</p>
</div>
```

```css
.text-container {
  font-size: 16px;
}

.text {
  font-size: 120%;
}
```

:::

Prozentangaben sind besonders praktisch für vertikale Zentrierung. Hier ist ein Beispiel, wie Sie Prozentangaben mit der Eigenschaft `transform` verwenden können, um ein Element vertikal zu zentrieren.

Dieses Beispiel positioniert das Element 50 % vom oberen Rand seines Containers entfernt und verwendet dann `transform`, um es um die Hälfte seiner eigenen Höhe nach oben zu verschieben, wodurch es effektiv vertikal zentriert wird.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="centered"></div>
```

```css
.centered {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
  width: 300px;
  height: 300px;
  background-color: red;
}
```

:::

Sie werden in zukünftigen Lektionen mehr darüber lernen, wie absolute Positionierung und die Transform-Eigenschaften im Detail funktionieren.

Denken Sie daran, dass Prozentangaben immer relativ zu etwas sind. Für horizontale Eigenschaften wie `width` beziehen sie sich auf die Breite des Elternteils. Für vertikale Eigenschaften wie `height` beziehen sie sich normalerweise auf die Höhe des Elternteils (sofern angegeben).

Seien Sie jedoch vorsichtig, wenn Sie Elemente mit prozentualen Abmessungen verschachteln, da dies zu unerwarteten Ergebnissen führen kann. Beachten Sie auch, dass prozentuale Höhen schwierig sein können, wenn der Elternteil keine definierte Höhe hat.

Zusammenfassend sind Prozentangaben in CSS mächtige Werkzeuge, um flexible, responsive Designs zu erstellen. Verwenden Sie sie, wenn Elemente proportional zu ihren Containern skalieren sollen oder wenn Sie Layouts benötigen, die sich nahtlos an verschiedene Bildschirmgrößen anpassen.
