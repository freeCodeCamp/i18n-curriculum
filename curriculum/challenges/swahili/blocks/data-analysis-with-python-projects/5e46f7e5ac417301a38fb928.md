Kitendakazi kinapaswa kubadilisha orodha hiyo kuwa safu ya data ya Numpy ya 3 x 3, kisha kirudishe kamusi yenye wastani, mabadiliko, upotofu wa kawaida, thamani kubwa zaidi, ndogo zaidi, na jumla kwa pande zote mbili na kwa matriki iliyopindishwa.

Kamusi inayorudishwa inapaswa kufuata muundo huu:

```py
{
  'mean': [axis1, axis2, flattened],
  'variance': [axis1, axis2, flattened],
  'standard deviation': [axis1, axis2, flattened],
  'max': [axis1, axis2, flattened],
  'min': [axis1, axis2, flattened],
  'sum': [axis1, axis2, flattened]

```
