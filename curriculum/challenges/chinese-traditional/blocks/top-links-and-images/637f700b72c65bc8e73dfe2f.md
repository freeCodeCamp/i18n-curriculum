# --description--

如果網站只能顯示文字，那會相當無聊。幸運的是，HTML 提供了各種元素來顯示各種不同的媒體。其中使用最廣泛的是影像元素。

要在 HTML 中顯示影像，請使用 `<img>` 元素。與你遇過的其他元素不同，`<img>` 是一個空元素。空的、無內容的 HTML 元素不需要結束標籤。

它不是用開啟和關閉的標記包裹內容，而是使用 `src` 屬性將影像嵌入頁面中，該屬性告訴瀏覽器影像檔案的位置。`src` 屬性的運作方式類似於錨點標籤的 `href` 屬性。它可以使用絕對路徑和相對路徑來嵌入影像。

例如，使用絕對路徑你可以顯示位於 The Odin Project 網站上的影像：

若要使用你自己網站上的影像，可以使用相對路徑。

- 在 `odin-links-and-images` 專案中創建一個名為 `images` 的新目錄。

- 接著，下載<a href="https://unsplash.com/photos/Mv9hjnEUHR4/download?force=true&w=640" target="_blank">這張影像</a>，並將它移動到你剛剛創建的 images 目錄中。

- 將影像重新命名為 `dog.jpg`。

最後將影像添加到 `index.html` 檔案：

```html
<body>
  <h1>Homepage</h1>
	<a href="https://www.theodinproject.com/about">click me</a>

	<a href="pages/about.html">About</a>

	<img src="images/dog.jpg">
</body>
```

將 `index.html` 檔案存檔，並在瀏覽器中開啟以查看 Charles 的全部風采。


## 父目錄

如果你想在 `about` 頁面使用狗的影像，你必須先從 pages 目錄往上一層到它的父目錄，這樣你才能存取 images 目錄。

要前往上層目錄，你需要在相對路徑中使用兩個點，如此：`../.` 讓我們來實際操作，在 `about.html` 檔案的 `body` 中，於你先前添加的標題下方加入以下影像：

```html
<img src="../images/dog.jpg">
```

分解如下：

- 首先，你將前往 `odin-links-and-images` 的 pages 目錄的上層目錄。

- 然後，從父目錄，你可以進入 `images` 目錄。

- 最後，你可以存取 `dog.jpg` 檔案。

使用我們之前使用的比喻，在檔案路徑中使用 `../` 有點像是從你目前所在的房間走到主走廊，這樣你才能前往另一個房間。

## Alt 屬性

除了 `src` 屬性外，每個影像元素也應該有一個 `alt`（替代文字）屬性。

`alt` 屬性用於描述影像。若影像無法載入，將會使用它來取代影像。它也用於螢幕閱讀器，向視障使用者描述影像內容。
