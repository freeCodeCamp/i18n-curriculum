# --description--

Ein `extension` ist eine Softwarekomponente, die einer größeren Softwareanwendung bestimmte Fähigkeiten hinzufügt. `Extensions` kann neue Funktionen bereitstellen oder bestehende Funktionen erweitern.

Zum Beispiel kann ein Browser-`extension` Ihrem Webbrowser spezielle Funktionen hinzufügen, wie das Blockieren von Werbung oder das Verwalten von Passwörtern.

# --questions--

## --text--

Was bedeutet `extension` im Programmierkontext?

## --answers--

Es ist ein Befehl, der verwendet wird, um ein Programm zu beenden.

### --feedback--

Ein `extension` ist kein Befehl, sondern ein Software-Add-on, das die Funktionalität erweitert.

---

Es ist ein Software-Add-on, das die Fähigkeiten eines bestehenden Programms erweitert.

---

Es bezieht sich auf die zusätzliche Zeit, die zur Fertigstellung eines Projekts gegeben wird.

### --feedback--

Während `extension` in anderen Kontexten zusätzliche Zeit bedeuten kann, bezieht es sich im Programmieren speziell auf eine Softwarekomponente.

---

Es ist eine Art Datenformat, das für die Speicherung verwendet wird.

### --feedback--

Ein `extension` erweitert die Softwarefunktionalität; es bezeichnet kein Datenformat zur Speicherung.
