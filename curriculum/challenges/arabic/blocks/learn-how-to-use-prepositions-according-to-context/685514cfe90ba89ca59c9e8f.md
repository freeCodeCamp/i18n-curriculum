# --explanation--

في `by next month`، تشير `by` إلى موعد نهائي، مما يعني أن شيئًا ما يجب أن يحدث قبل أو في موعد لا يتجاوز ذلك الوقت.

الجملة `We need to finish the report by Friday` تستخدم `by` بنفس الطريقة، محددة حدًا زمنيًا واضحًا.
