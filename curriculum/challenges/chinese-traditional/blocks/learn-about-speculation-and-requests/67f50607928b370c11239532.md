# --explanation--

`Evidence of` 意味著某事已經發生的跡象或證明。例如：

`The report showed evidence of user error.` - 報告顯示有人犯了錯誤。

`brute force attack` 是指有人嘗試許多不同的密碼直到其中一個成功。

`SQL injection` 是指有人試圖透過在搜尋框或表單中輸入有害程式碼來入侵系統。

Jake 正在討論檢查這些型別攻擊的跡象。
