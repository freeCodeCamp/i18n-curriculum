# --description--

在上一课中，你学习了如何使用元组数据类型。在本课中，你将学习一些在使用元组时常用的方法。

我们将介绍的第一个方法是 `count()`。此方法用于确定一项在元组中出现了多少次。下面是一个示例，展示如何查看字串 `Rust` 在名为 `programming_languages` 的元组中出现了多少次：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

由于 `Rust` 在元组中出现了两次，`count()` 方法返回数字 `2`。如果 `count()` 函数中指定的项在元组中根本不存在，则返回值为 `0`：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

如果没有参数传递给 `count()` 函数，那么 Python 会引起一个 `TypeError`：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count()

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: tuple.count() takes exactly one argument (0 given)
"""
```

下一个我们将要查看的方法是 `index()` 方法。此方法用于查找元组中某个特定项所在的索引。以下是使用 `index()` 方法查找字串 `Java` 索引的示例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

如果找不到指定的项，则 Python 会引起 `ValueError`：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: tuple.index(x): x not in tuple
"""
```

你还可以使用 `index()` 方法传入可选的起始和结束索引参数。以下是传入可选起始索引的示例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

在此示例中，我们指定从哪里开始查找字串 `Python`。通过将数字 `3` 作为第二个参数传递给 `index()` 函数，我们指定从索引 `3` 开始查找。由于 `Python` 在元组中出现了两次，`index()` 函数将因为使用了可选的起始索引参数而返回索引 `5`，而不是索引 `2`。

你也可以传入一个可选的停止索引。以下是指定起始和停止索引的修改示例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

现在结果是索引 `2`，因为我们从索引 `2` 开始查找，并查找直到但不包括索引 `5`。

另一个常用的用于元组的函数是 `sorted()` 函数。在之前的课程中你学习了列表的 `sort()` 方法。实际上，`sorted()` 函数可以用于任何可迭代对象，包括元组。

下面是使用 `sorted()` 函数创建数字新建列表的示例：

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

`sorted()` 函数总是创建一个排序值的新列表。这与 `sort()` 方法不同，后者会就地排序列表的元素且不返回新列表。

如果你需要为可迭代对象自定义排序行为，可以使用可选的 `reverse` 和 `key` 参数。下面是使用 `key` 参数按长度对元组中的项进行排序的示例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

如果你想创建一个按相反顺序排列的新值列表，那么你可以像这样使用 `reverse` 参数：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

元组是 Python 中常见的数据类型。了解如何使用它们，以及一些有用的方法和函数，将帮助你编写更高效的代码。
