# --interactive--

有時你需要在你的 JavaScript 檔案中操作 HTML 元素的樣式。舉例來說，當使用者點擊按鈕時，你需要隱藏或顯示一個選單。

在本課程中，你將學習如何使用 `Element.style` 和 `Element.classList` 屬性操作樣式。

`Element.style` 屬性是一個唯讀屬性，用來表述元素的行內樣式。你可以使用這個屬性來取得或設定元素的樣式。

這是一個具有 ID 為 `para` 的段落元素範例：

```html
<p id="para">This is a paragraph</p>
```

你可以使用 `style` 屬性將段落元素的顏色改為紅色：

:::interactive_editor

```html
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```js
const paraEl = document.getElementById("para");
paraEl.style.color = "red";
```

:::

`style` 屬性可用來設定許多 CSS 屬性，例如 `color`、`background-color`、`font-size`、`font-weight` 等等。

操作樣式的另一種方法是使用 `Element.classList` 屬性。`classList` 屬性是一個唯讀屬性，可用來為元素添加、移除或切換類別。讓我們來看看幾個範例。

在這個範例中，你將為一個 ID 為 `para` 的段落元素添加一個名為 `highlight` 的類別。在 CSS 中，我們會設定 `highlight` 類別，使段落元素的背景顏色變為黃色。在 JavaScript 中，我們會使用 `classList.add()` 方法將 `highlight` 類別添加到段落元素。 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```css
.highlight {
  background-color: yellow;
}
```

```js
const paraEl = document.getElementById("para");
paraEl.classList.add("highlight");
```

:::

你也可以將多個類別作為引數傳遞給 `classList.add()` 方法，從而為元素添加多個類別：

```js
classList.add("class1", "class2", "class3");
```

如果你需要從元素中移除一個類別，你可以像這樣使用 `classList.remove()` 方法：

```js
classList.remove("highlight");
```

要在元素上切換類別，你可以使用 `classList.toggle()` 方法。在此範例中，我們有一個 ID 為 `toggle-btn` 的 `button` 元素和一個 ID 為 `menu` 的 `nav` 元素。在 CSS 中，我們為選單添加了一些樣式，以及一個名為 `show` 的類別，該類別會將 `.menu` 的 `display` 屬性改為 `block`。在 JavaScript 中，我們使用 `getElementById()` 方法存取選單元素和 `button` 元素。接著，我們為 `button` 元素添加一個事件監聽器，當按鈕被點擊時，會在 `.menu` 元素上切換 `show` 類別：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button id="toggle-btn">Toggle Menu</button>

<nav id="menu" class="menu">
  <ul>
    <li>Home</li>
    <li>About</li>
    <li>Products</li>
  </ul>
</nav>
<script src="index.js"></script>
```

```css
.menu {
  display: none;
  background-color: lightgray;
  width: 50%;
  padding: 10px;
}

.menu.show {
  display: block;
}
```

```js
const menu = document.getElementById("menu");
const toggleBtn = document.getElementById("toggle-btn");

toggleBtn.addEventListener("click", () => menu.classList.toggle("show"));
```

:::

當使用者點擊按鈕時，`show` 類別將會被添加到 `.menu` 元素，選單將會顯示。若使用者再次點擊按鈕，`show` 類別將會從 `.menu` 元素中移除，選單將會隱藏。

透過使用像是 `style`、`add()`、`remove()` 和 `toggle()` 等方法與屬性，你可以輕鬆地用 JavaScript 操作元素的樣式，打造動態且互動的網頁。
