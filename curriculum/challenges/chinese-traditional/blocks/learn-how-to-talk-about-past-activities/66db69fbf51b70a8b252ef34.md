# --explanation--

這句話使用了 `Present Perfect` 和 `Present Continuous` 時態的混合來描述進展和持續的問題：

- `We've` 是 `We have` 的縮寫，用於 `Present Perfect` 時態以表示進展已經發生。

- `Made` 是 `make` 的過去分詞，這裡用來描述已經達成的進度。

- `Aren't` 是 `are not` 的縮寫，屬於現在進行式的零件。

- `Responding` 是表示該動作（圖示和按鈕無法正確運作）正在發生中。
