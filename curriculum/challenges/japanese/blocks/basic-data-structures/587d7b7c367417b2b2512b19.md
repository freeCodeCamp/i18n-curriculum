# --description--

少し複雑なオブジェクトを見てみましょう。オブジェクトのプロパティは任意の深さまでネストでき、その値は配列や他のオブジェクトを含む、JavaScriptでサポートされている任意のデータ型にできます。次の例を考えてみてください。

```js
let nestedObject = {
  id: 28802695164,
  date: 'December 31, 2016',
  data: {
    totalUsers: 99,
    online: 80,
    onlineStatus: {
      active: 67,
      away: 13,
      busy: 8
    }
  }
};
```

`nestedObject` は3つのプロパティを持っています：`id`（値は数値）、`date`（値は文字列）、そして`data`（ネストされた構造を持つオブジェクト）。構造はすぐに複雑になりますが、必要な情報にアクセスするために同じ記法を使えます。ネストされた`10`オブジェクトの`busy`プロパティに`onlineStatus`の値を代入するには、ドット記法を使ってプロパティを参照します。

```js
nestedObject.data.onlineStatus.busy = 10;
```

# --instructions--

ここでは`userActivity`というオブジェクトを定義しており、その中に別のオブジェクトがネストされています。`online`キーの値を`45`に設定してください。

# --hints--

`userActivity`は`id`、`date`、`data`のプロパティを持つべきです。

```js
assert(
  'id' in userActivity && 'date' in userActivity && 'data' in userActivity
);
```

`userActivity`は`data`キーに、`totalUsers`と`online`キーを持つオブジェクトを設定するべきです。

```js
assert('totalUsers' in userActivity.data && 'online' in userActivity.data);
```

`online`プロパティは`data`の`userActivity`キーにネストされており、`45`に設定するべきです。

```js
assert(userActivity.data.online === 45);
```

`online`プロパティはドット記法かブラケット記法で設定してください。

```js
assert.strictEqual(__helpers.removeJSComments(code).search(/online: 45/), -1);
```
