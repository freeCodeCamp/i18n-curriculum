# --interactive--

У JavaScript функції та методи об’єктів — це обидва способи інкапсуляції повторно використовуваного коду, але вони мають ключові відмінності у тому, як їх визначають, використовують і в контексті, у якому вони працюють. Розуміння цих відмінностей є важливим для написання ефективного та організованого JavaScript-коду.

Як ви дізналися в попередніх модулях, функції — це повторно використовувані блоки коду, які виконують конкретне завдання:

:::interactive_editor

```js
function greet(name) {
    return "Hello, " + name + "!";
}
console.log(greet("Alice")); // "Hello, Alice!"
```

:::

Методи об’єктів, натомість, — це функції, які пов’язані з об’єктом. Вони визначаються як властивості об’єкта і можуть отримувати доступ до даних об’єкта та змінювати їх. Ось приклад об’єкта з методом:

:::interactive_editor

```js
const person = {
    name: "Bob",
    age: 30,
    sayHello: function() {
        return "Hello, my name is " + this.name;
    }
};

console.log(person.sayHello()); // "Hello, my name is Bob"
```

:::

У цьому прикладі `sayHello` є методом об’єкта `person`. Ключове слово `this` дозволяє методу `sayHello` отримувати доступ до властивостей об’єкта з ідентифікатором `person`. Ви дізнаєтеся більше про ключове слово `this` у майбутніх уроках.

Відмінність між функціями та методами полягає у способі їх виклику. Функції викликають за їхнім іменем, а методи — за допомогою точкової нотації на об’єкті, якому вони належать. Наприклад, ми викликаємо функцію `greet` як `greet("Alice")`, а метод `sayHello` — як `person.sayHello()`.

Ще одна важлива відмінність — це контекст, у якому вони працюють. Звичайні функції мають власну область видимості, але не мають вбудованого референса на якийсь конкретний об’єкт. Методи ж прив’язані до свого об’єкта і можуть отримувати доступ до його властивостей та інших методів за допомогою ключового слова `this`.

Важливо пам’ятати, що методи допомагають організувати код у логічні об’єкти, тоді як функції використовують для більш загального, повторно використовуваного коду.

Розуміння різниці між функціями та методами об’єктів є важливим на вашому шляху вивчення JavaScript. Хоч вони можуть здаватися схожими на перший погляд, розпізнавання, коли використовувати кожен із них, допоможе писати більш організований, ефективний і кращий код.

```js

        return "Hi there!";
    }
};
```
