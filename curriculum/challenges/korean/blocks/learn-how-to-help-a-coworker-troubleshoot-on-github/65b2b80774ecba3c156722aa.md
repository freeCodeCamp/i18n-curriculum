# --description--

`To run into`는 종종 누군가를 예상치 못하게 만나거나 문제에 부딪히는 것을 의미합니다. 예를 들어, `I ran into my friend at the store`는 계획하지 않고 친구를 만난 경우를 뜻합니다.

표현 `run into the same problem`는 특히 같은 어려움이나 문제를 다시 마주하는 것을 가리킵니다. 이전에 경험했고 피하고 싶은 반복되는 도전을 이야기할 때 쓰입니다.

예를 들어, 소프트웨어가 특정 버그 때문에 충돌했는데 이를 고쳤다면, 앞으로 `run into the same problem`지 않도록 해결책을 문서화할 것입니다.
