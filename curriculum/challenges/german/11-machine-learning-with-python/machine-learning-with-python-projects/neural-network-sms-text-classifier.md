# --instructions--

In dieser Herausforderung musst du ein maschinelles Lernmodell erstellen, das SMS-Nachrichten entweder als "Ham" oder "Spam" klassifiziert. Eine "Ham"-Nachricht ist eine normale Nachricht, die von einem Freund gesendet wurde. Eine "Spam"-Nachricht ist eine Werbung oder eine Nachricht, die von einem Unternehmen gesendet wurde.

Du solltest eine Funktion namens `predict_message` erstellen, die eine Zeichenfolge von Nachrichten als ein Argument und eine Liste zurückgibt. Das erste Element in der Liste sollte eine Zahl zwischen 0 und 1, die die Wahrscheinlichkeit von "Ham" (0) oder "Spam" (1) anzeigt. Das zweite Element in der Liste sollte das Wort "Ham" oder "Spam" sein, je nachdem, was am wahrscheinlichsten ist.

Für diese Herausforderung, verwende den <a href="http://www.dt.fee.unicamp.br/~tiago/smsspamcollection/" target="_blank" rel="noopener noreferrer nofollow">SMS Spam Collection</a> Datensatz. Der Datensatz wurde bereits in Trainingsdaten und Testdaten unterteilt.

Die ersten beiden Zellen importieren Bibliotheken und Daten. Die letzte Zelle testet dein Modell und deine Funktion. Füge deinen Code zwischen diesen Zellen hinzu.
