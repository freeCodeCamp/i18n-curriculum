# --explanation--

单词 `some` 意思是 `a part of` 或 `not all`。例如：

- `Some of the team members are working remotely.` - 不是所有成员，只是部分成员。

- 她买了一些新书来准备考试。

使用 `some` 突出表明，虽然 TechCorp 对较小的公司很重要，但它并不资助它们所有的项目，只资助其中一部分。这就是为什么失去 TechCorp 仍会产生重大影响。
