# --interactive--

JavaScript에서는 문자열의 일부를 다른 문자열로 교체해야 하는 상황이 많습니다.

예를 들어, URL에서 사용자 정보를 업데이트하거나, 날짜 형식을 변경하거나, 사용자가 생성한 콘텐츠의 오류를 수정해야 할 수 있습니다.

JavaScript의 `replace()` 메서드는 문자열에서 지정한 값(단어 또는 문자 등)을 찾아 다른 값으로 교체할 수 있게 해줍니다. 이 메서드는 교체된 새 문자열을 반환하며, JavaScript 문자열은 불변이기 때문에 원본 문자열은 변경되지 않습니다.

기본 구문은 다음과 같습니다:

```js
string.replace(searchValue, newValue);
```

`searchValue`은 문자열에서 찾고자 하는 값입니다. 문자열 또는 텍스트 패턴을 설명하는 정규식(regex)일 수 있습니다. 이를 통해 문자열을 유연하고 강력하게 검색하고 조작할 수 있습니다. 정규 표현식에 대해서는 이후 수업에서 더 배우게 됩니다.

`newValue`는 `searchValue`을 교체할 값입니다. 간단한 예는 다음과 같습니다:

:::interactive_editor

```js
let text = "I love JavaScript!";
console.log(text); // "I love JavaScript!"
let newText = text.replace("JavaScript", "coding");
console.log(newText);  // "I love coding!"
```

:::

이 예제에서는 문자열 내에서 `JavaScript`라는 단어를 찾아 `coding`로 교체합니다.

`replace()` 메서드는 대소문자를 구분하므로 `searchValue`과 정확히 일치하는 경우에만 찾습니다. 예를 들어:

:::interactive_editor

```js
let sentence = "I enjoy working with JavaScript.";
console.log(sentence);  // "I enjoy working with JavaScript."
let updatedSentence = sentence.replace("javascript", "coding");
console.log(updatedSentence);  // "I enjoy working with JavaScript."
```

:::

여기서는 `javascript`(소문자 `j`)가 `JavaScript`(대문자 `J`)와 일치하지 않아 교체가 이루어지지 않습니다.

기본적으로 `replace()` 메서드는 `searchValue`의 첫 번째 발생만 교체합니다. 문자열에 값이 여러 번 나타나도 첫 번째 것만 교체됩니다:

:::interactive_editor

```js
let phrase = "Hello, world! Welcome to the world of coding.";
console.log(phrase);  // "Hello, world! Welcome to the world of coding."
let updatedPhrase = phrase.replace("world", "universe");
console.log(updatedPhrase);  // "Hello, universe! Welcome to the world of coding."
```

:::

`world`의 첫 번째 발생만 `universe`로 교체된 것을 확인하세요.

JavaScript의 `replace()` 메서드는 문자열 조작에 강력하고 유연한 도구입니다.

개별 문자, 단어 또는 정규 표현식을 사용한 복잡한 패턴 등 문자열의 특정 부분을 교체할 수 있습니다.

간단한 교체에 이상적이지만, 대소문자 구분과 기본 동작(첫 번째 발생만 교체하는 것)을 이해하면 더 효과적으로 사용할 수 있습니다.
