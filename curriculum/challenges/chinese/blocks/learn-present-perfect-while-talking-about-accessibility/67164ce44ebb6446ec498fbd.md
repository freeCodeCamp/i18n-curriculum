# --explanation--

`Present Perfect Continuous` 的疑问形式是通过使用 `Have/Has + subject + been + verb(-ing)` 创建的。它用于询问从过去开始并仍在进行的动作。例如：

`Have they been testing the new software features?` - 有人查看测试是否一直在进行。
