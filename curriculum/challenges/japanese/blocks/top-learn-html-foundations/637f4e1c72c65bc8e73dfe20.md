# --description--

HTMLのボイラープレートを示すには、まず作業用のHTMLファイルが必要です。

コンピューターに新しいフォルダーを作成し、名前を`html-boilerplate`にしてください。そのフォルダー内に新しいファイルを作成し、名前を`index.html`にしてください。

おそらくdoc、pdf、画像ファイルなど、さまざまな種類のファイルにはすでに慣れているでしょう。

コンピューターにHTMLファイルを作成したいことを知らせるには、ファイル名に`.html`拡張子を付ける必要があります。これは`index.html`ファイルを作成したときと同じです。

HTMLファイルの名前をindexにしたことは重要です。ウェブサイトのホームページを含むHTMLファイルは常に`index.html`と名付けるべきです。これは、ウェブサーバーがユーザーがウェブサイトにアクセスしたときにデフォルトでは`index.html`ページを探すためで、これがないと大きな問題が起きるからです。

## The DOCTYPE

すべてのHTMLページはdoctype宣言から始まります。doctypeの目的は、ブラウザーにどのHTMLバージョンを使ってドキュメントをレンダリングするかを伝えることです。最新のHTMLバージョンはHTML5で、そのバージョンのdoctypeは単に`<!DOCTYPE html>`です。

古いバージョンのHTMLのdoctypeはもう少し複雑でした。例えば、これはHTML4のdoctype宣言です：

```html
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
```

しかし、おそらく古いバージョンのHTMLを使うことはほとんどないので、常に`<!DOCTYPE html>`を使うことになります。

先ほど作成した`index.html`ファイルをテキストエディターで開き、最初の行に`<!DOCTYPE html>`を追加してください。
