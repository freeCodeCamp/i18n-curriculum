# --description--

L'elemento `<head>` è dove inserisci informazioni meta importanti sulle tue pagine web e tutto ciò che serve affinché le tue pagine web vengano renderizzate correttamente nel browser. All'interno del `<head>` non dovresti usare alcun elemento che mostri contenuti sulla pagina web.

## L'elemento meta charset
Dovresti sempre avere il tag `meta` per la codifica `charset` della pagina web nell'elemento `head`: `<meta charset="utf-8">`.

Impostare la codifica è molto importante perché garantisce che la pagina web mostri correttamente simboli speciali e caratteri di lingue diverse nel browser.

## Elemento title
Un altro elemento che dovresti sempre includere nell'intestazione di un documento HTML è l'elemento `title`:

```html
<title>My First Webpage</title>
```

L'elemento `title` serve a dare alle pagine web un titolo leggibile dall'utente che viene mostrato nella scheda del browser della tua pagina web.

Se non includessi un elemento `title`, il titolo della pagina web sarebbe predefinito al nome del file. Nel tuo caso sarebbe `index.html`, che non è molto significativo per gli utenti; questo renderebbe molto difficile trovare la tua pagina web se l'utente ha molte schede del browser aperte.

Ci sono molti altri elementi che possono andare all'interno del `head` di un documento HTML. Tuttavia, per ora è cruciale conoscere solo i due elementi che hai visto qui. Introdurrai altri elementi che vanno nel `head` nel resto del curriculum.

Nel file `index.html` aggiungiamo un elemento `head` con un elemento `charset` `meta` e un `title` al suo interno. L'elemento head va all'interno dell'elemento HTML e dovrebbe essere sempre il primo elemento sotto il tag di apertura `<html>`:


```html
<!DOCTYPE html>

<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Webpage</title>
  </head>
</html>
```

## --text--

Qual è lo scopo dell'elemento `head`?

## --answers--

L'elemento `head` serve a mostrare tutti gli elementi che vengono visualizzati sulla pagina web.

---

L'elemento `head` serve a mostrare informazioni importanti sulla tua pagina web ed è usato per renderizzare correttamente le pagine web con gli elementi `meta`.

---

L'elemento `head` serve a mostrare il contenuto dell'intestazione in cima alla pagina web.
