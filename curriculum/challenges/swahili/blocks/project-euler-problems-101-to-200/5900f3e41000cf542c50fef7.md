# --description--

Acha `r` awe mabaki yanapotengenezwa mgawanyo wa ${(a − 1)}^n + {(a + 1)}^n$ kwa $a^2$.

Kwa mfano, ikiwa $a = 7$ na $n = 3$, basi $r = 42: 6^3 + 8^3 = 728 ≡ 42 \\ \text{mod}\\ 49$. Na kadri `n` inavyobadilika, ndivyo `r` itakavyobadilika, lakini kwa $a = 7$ inajitokeza kuwa $r_{max} = 42$.

Kwa $3 ≤ a ≤ 1000$, tafuta $\sum{r}_{max}$.
