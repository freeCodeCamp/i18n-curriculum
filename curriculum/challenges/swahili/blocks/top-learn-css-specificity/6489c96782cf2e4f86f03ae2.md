# --description--

Tamko la CSS lenye makini zaidi litapewa kipaumbele kuliko yale yenye makini kidogo. Mitindo ndani ya mstari, ambayo umejifunza katika funzo lililopita, ina makini zaidi ikilinganishwa na vichaguzi, huku kila aina ya kichaguzi ikiwa na kiwango chake cha makini kinachochangia jinsi tamko linavyokuwa na makini. Vichaguzi vingine vinachangia makini, lakini wewe unazingatia tu vile ulivyovijifunza hadi sasa:

1. Vichaguzi vya ID (kichaguzi chenye makini zaidi)
2. Vichaguzi vya darasa la vitu
3. Vichaguzi cha aina

Makini yatazingatiwa tu wakati kipengee kina tamko nyingi zinazopingana zinazolenga kipengee hicho, kama vile kisheria cha kuamua mshindi. Kichaguzi cha ID kitaendelea kushinda idadi yoyote ya vichaguzi vya darasa la vitu, kichaguzi cha darasa la vitu kitaendelea kushinda idadi yoyote ya vichaguzi cha aina, na kichaguzi cha aina kitaendelea kushinda idadi yoyote ya vichaguzi vyovyote vyenye makini kidogo kuliko hicho. Wakati hakuna tamko lenye kichaguzi chenye makini zaidi, idadi kubwa ya kichaguzi kimoja itashinda idadi ndogo ya kichaguzi hicho hicho.
