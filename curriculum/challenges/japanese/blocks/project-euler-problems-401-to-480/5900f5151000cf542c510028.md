# --description--

2つの正の数 $A$ と $B$ は、次のいずれかの条件を満たすとき「つながっている」（"$A ↔ B$" と表記）と言います。

1. $A$ と $B$ は同じ桁数で、ちょうど1つの数字だけ異なる。例えば、$123 ↔ 173$。
2. $A$ の左に1桁の数字を追加すると $B$ になる（またはその逆）。例えば、$23 ↔ 223$ および $123 ↔ 23$。

素数 $P$ が「2の親戚」と呼ばれるのは、2と $P$ の間に、つながった素数の連鎖が存在し、その連鎖の中のどの素数も $P$ を超えない場合です。

例えば、127 は2の親戚です。可能な連鎖の一例は以下の通りです。

$$2 ↔ 3 ↔ 13 ↔ 113 ↔ 103 ↔ 107 ↔ 127$$

しかし、11 と 103 は2の親戚ではありません。

$F(N)$ を、$N$ 以下の素数のうち2の親戚でない素数の和とします。$F({10}^3) = 431$ および $F({10}^4) = 78\,728$ であることが確認できます。

$F({10}^7)$ を求めてください。
