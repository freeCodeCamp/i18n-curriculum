# --description--

既然你已經知道了迭代值和陣列的最常見方法，還有一些其他方法可以迭代陣列。陣列有一個內建的行為叫做 `map()`，用來透過對原始陣列的每個元素應用函式來創建一個新的陣列。`map()` 行為不會改變原始陣列。`map()` 行為的語法如下：

```javascript
const array = [1, 2, 3, 4, 5];

const newArray = array.map((arrayValue) => {
  return arrayValue * 2;
});

console.log(newArray); // Output: [2, 4, 6, 8, 10]
```

`map()` 方法透過將函式 `(arrayValue) => { return arrayValue * 2; }` 套用到原始陣列的每個元素，來創建一個新的陣列。當你想要轉換陣列的元素而不改變原始陣列時，這特別有用。
