# --description--

المجموعات هي واحدة من هياكل البيانات المدمجة في Python. من الخصائص الأساسية للمجموعات أنها لا تخزن القيم المكررة. إذا حاولت إضافة قيمة مكررة إلى مجموعة، فسيتم تخزين واحدة فقط منها.

المجموعات قابلة للتغيير وغير مرتبة، مما يعني أن عناصرها لا تُخزن بأي ترتيب محدد، لذلك لا يمكنك استخدام الفهارس أو المفاتيح للوصول إليها. يمكن أن تحتوي فقط على قيم من أنواع بيانات غير قابلة للتغيير مثل الأعداد، والسلاسل النصية، والصفوف المرتبة. كما تدعم العمليات الرياضية على المجموعات، بما في ذلك الاتحاد، والتقاطع، والفرق، والفرق المتماثل.

لتعريف مجموعة، تحتاج فقط إلى كتابة عناصرها داخل أقواس متعرجة وفصلها بفواصل. هذا مثال على مجموعة من الأعداد:

```python
my_set = {1, 2, 3, 4, 5} 
```

من الخصائص الغريبة للعمل مع المجموعات أنه إذا احتجت لتعريف مجموعة فارغة، يجب عليك استخدام الدالة `set()`. إذا كتبت فقط أقواس متعرجة فارغة مثل `{}`، فسيقوم Python تلقائيًا بإنشاء قاموس.

```python
set() # Set
{}    # Dictionary
```

يمكنك إضافة عنصر إلى مجموعة باستخدام الطريقة `.add()`، وتمرير العنصر الجديد كمعلمة:

```python
my_set.add(6)
```

في مثالنا، ستكون المجموعة الجديدة:

```python
{1, 2, 3, 4, 5, 6}
```

إذا حاولت إضافة عنصر موجود بالفعل في المجموعة، فسيتم الاحتفاظ بواحد فقط. في هذه الحالة، لدينا بالفعل الرقم 5 في المجموعة:

```python
my_set.add(5)
```

لذا لن تتغير المجموعة:

```python
{1, 2, 3, 4, 5, 6}
```

لإزالة عنصر من المجموعة، لديك خياران. يمكنك استخدام إما الطريقة `.remove()` أو الطريقة `.discard()`، وتمرير العنصر الذي تريد إزالته كمعلمة.

الطريقة `.remove()` سترفع خطأ `KeyError` إذا لم يُعثر على العنصر، بينما الطريقة `.discard()` لن تفعل ذلك:

```python
my_set.remove(4)
my_set.discard(4)
```

الطريقة `.clear()` تزيل كل العناصر من المجموعة:

```python
my_set.clear()
```

للمجموعات في Python أيضًا طرق قوية تؤدي عمليات رياضية شائعة على المجموعات.

الطريقتان `.issubset()` و `.issuperset()` تتحققان مما إذا كانت مجموعة ما مجموعة جزئية أو مجموعة شاملة لمجموعة أخرى، على التوالي.

هنا، نتحقق مما إذا كانت `your_set` مجموعة جزئية من `my_set`، والنتيجة `False` لأن ليس كل عناصر `your_set` موجودة في `my_set`.

نحن نتحقق أيضًا مما إذا كانت `my_set` مجموعة شاملة لـ `your_set`. هذه أيضًا `False` لأن `my_set` لا تحتوي على كل عناصر `your_set`:

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

print(your_set.issubset(my_set)) # False
print(my_set.issuperset(your_set)) # False
```

الطريقة `.isdisjoint()` تتحقق مما إذا كانت مجموعتان منفصلتان، أي لا تشتركان في أي عناصر. في هذه الحالة، النتيجة `False` لأن `my_set` و `your_set` تشتركان في العناصر 2 و3 و4:

```python
print(my_set.isdisjoint(your_set)) # False
```

عامل الاتحاد `|` يعيد مجموعة جديدة تحتوي على كل العناصر من المجموعتين:

```python
my_set | your_set # {1, 2, 3, 4, 5, 6}
```

عامل التقاطع `&` يعيد مجموعة جديدة تحتوي فقط على العناصر المشتركة بين المجموعتين:

```python
my_set & your_set # {2, 3, 4}
```

عامل الفرق `-` يعيد مجموعة جديدة تحتوي على عناصر المجموعة الأولى التي ليست في المجموعات الأخرى. في هذا المثال، الأعداد 1 و5 موجودة في `my_set` لكنها ليست في `your_set`:

```python
my_set - your_set # {1, 5}
```

عامل الفرق المتماثل `^` يعيد مجموعة جديدة تحتوي على العناصر الموجودة في المجموعة الأولى أو الثانية، ولكن ليس في كلتيهما. في هذه الحالة، 1 و5 موجودان في `my_set` وليسا في `your_set`، لذا تم تضمينهما. والعدد 6 موجود في `your_set` وليس في `my_set`، لذا تم تضمينه أيضًا:

```python
my_set ^ your_set # {1, 5, 6}
```

لكل واحد من هذه العوامل أيضًا معامل تعيين مركب مقابله إذا أضفت علامة المساواة بجانبه. هذه المعاملات تعين تلقائيًا المجموعة الناتجة إلى المجموعة الأولى في التعبير:

```python
|= &= -= ^=
```

على سبيل المثال، المعامل `-=` يجد الفرق بين المجموعتين ويحدث المجموعة الأولى بالنتيجة:

```python
my_set -= your_set
```

بعد هذا، سيتم تحديث `my_set` إلى `{1, 5}`:

```python
print(my_set) # {1, 5}
```

يمكنك التحقق مما إذا كان عنصر موجودًا في مجموعة أم لا باستخدام عامل `in`. هنا، نتحقق مما إذا كان الرقم 5 موجودًا في `my_set`. ستكون النتيجة قيمة منطقية `True` أو `False`:

```python
print(5 in my_set)
```

وهذه هي أساسيات المجموعات. هي مفيدة جدًا عندما لا تحتاج إلى تخزين القيم بترتيب محدد، وعندما تحتاج فقط إلى تخزين قيم فريدة.
