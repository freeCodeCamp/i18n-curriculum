# --interactive--

## Робота з плаваючими елементами

- **Визначення**: Плаваючі елементи використовуються, щоб вивести елемент із його нормального потоку на сторінці та розмістити його або зліва, або справа від контейнера. Коли це відбувається, текст обтікає цей плаваючий вміст.

```css
float: left;
float: right;
```

- **Очищення плаваючих елементів**: Властивість `clear` використовується, щоб визначити, чи потрібно перемістити елемент під плаваючий вміст. Коли у вас є кілька плаваючих елементів, розташованих поруч, можуть виникати проблеми з накладанням і зсувом у макетах. Тому був створений хак `clearfix`, щоб виправити цю проблему.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="clearfix">
  <div class="box left">Left</div>
  <div class="box right">Right</div>
</div>
```

```css
.box {
  width: 100px;
  height: 60px;
  color: white;
  text-align: center;
  line-height: 60px;
}

.left {
  float: left;
  background: teal;
}

.right {
  float: right;
  background: purple;
}

.clearfix::after {
  content: "";
  display: block;
  clear: both;
}
```

:::

## Статичне, відносне та абсолютне позиціонування

- **Статичне позиціонування**: Це нормальний потік документа. Елементи розташовуються зверху вниз і зліва направо один за одним.
- **Відносне позиціонування**: Дозволяє використовувати властивості `top`, `left`, `right` і `bottom` для позиціонування елемента в межах нормального потоку документа. Ви також можете використовувати відносне позиціонування, щоб елементи накладалися один на одного на сторінці.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="relative">Relative Element</div>
```

```css
.relative {
    position: relative;
    top: 30px;
    left: 30px;
}
```

:::

- **Абсолютне позиціонування**: Дозволяє вивести елемент із нормального потоку документа, роблячи його незалежним від інших елементів.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  Parent
  <div class="positioned">Absolute</div>
</div>
```

```css
.parent {
  position: relative;
  height: 120px;
  background: #eee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

## Фіксоване та липке позиціонування

- **Фіксоване позиціонування**: Коли елемент позиціонується за допомогою `position: fixed`, він виводиться з нормального потоку документа і розміщується відносно вікна перегляду, тобто залишається на одному місці навіть під час прокручування користувачем. Це часто використовується для елементів, таких як заголовки або панелі навігації, які повинні бути завжди видимими.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="navbar">Fixed Navbar</div>
<div style="height: 600px;"></div>
```

```css
.navbar {
  position: fixed; 
  top: 0; 
  width: 100%;
  background: #0077ff;
  color: white;
  padding: 8px; 
}
```

:::

- **Липке позиціонування**: Цей тип позиціонування є гібридом між відносним і фіксованим позиціонуванням. Спочатку елемент поводиться так, ніби він позиціонується відносно, залишаючись у потоці документа. Однак, коли користувач прокручує сторінку і елемент проходить певну точку, він «прилипає» до вікна перегляду (зазвичай зверху) і поводиться так, ніби він фіксований. Це чудово підходить для створення липких панелей навігації, які стають фіксованими лише після прокручування до певної позиції.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Scroll down</p>
<div class="positioned">Sticky Element</div>
<div style="height: 500px;"></div>
```

```css
.positioned {
  position: sticky;
  top: 30px;
  left: 30px;
  background: #4caf50;
  color: white;
  padding: 6px;
}
```

:::

## Робота з властивістю `z-index`

- **Визначення**: Властивість `z-index` у CSS використовується для керування вертикальним порядком накладання позиційованих елементів, які перекриваються на сторінці.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box box1">1</div>
  <div class="box box2">2</div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

.box {
  position: absolute;
  width: 100px;
  height: 100px;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

.box1 {
  background: lightcoral;
  top: 20px;
  left: 20px;
  z-index: 1;
}

.box2 {
  background: steelblue;
  top: 40px;
  left: 40px;
  z-index: 2;
}
```

:::
