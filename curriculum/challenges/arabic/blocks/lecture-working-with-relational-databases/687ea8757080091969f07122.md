# --description--

SQL تعني لغة الاستعلامات الهيكلية. تم تطويرها في السبعينيات، وكانت تُعرف في البداية باسم لغة الاستعلامات الإنجليزية الهيكلية (SEQUEL). ثم تم اختصار هذا المصطلح لاحقًا إلى SQL.

SQL هي لغة برمجة تُستخدم لتخزين وإدارة البيانات في قواعد البيانات العلائقية.

كما تعلمت في درس سابق، تنظم قواعد البيانات العلائقية البيانات في جداول. يحتوي كل جدول على مجموعة من الصفوف والأعمدة، حيث تُمثّل خصائص البيانات وعلاقاتها. كل صف يمثل سجلًا، وكل عمود يمثل خاصية أو حقلًا محددًا لذلك السجل.

باستخدام SQL، يمكنك إضافة البيانات وتغييرها وحذفها والعثور عليها واسترجاعها من قاعدة بيانات علائقية. توفر مجموعة شاملة من الأوامر للاستعلام، والترشيح، والفرز، وتجميع البيانات.

تساعد SQL في تحسين أداء قاعدة البيانات من خلال استعلام البيانات الضرورية فقط، ويمكن دمجها بسهولة مع مجموعة واسعة من لغات البرمجة، لتتمكن من التفاعل مع قواعد البيانات مباشرة من تطبيقاتك.

تم اعتمادها أيضًا كمعيار صناعي. تفرض سلامة البيانات وتشمل العديد من ميزات الأمان، مثل دعم توثيق المستخدم وتشفير البيانات.

كما أنها قابلة للتوسع، ومحمولة، ومتوافقة مع مجموعة واسعة من أنظمة إدارة قواعد البيانات.

تعتمد SQL على الأوامر، المعروفة عادةً بتصريحات SQL أو استعلامات SQL.

بصفتك مطوِّرًا، ستكتب هذه التصريحات باستخدام عناصر لغة SQL أو الكلمات المفتاحية المحددة. تسمح لك بأداء العمليات اللازمة على قاعدة بياناتك.

قبل أن تدخل أوامر SQL، تحتاج إلى فتح واجهة psql لتتفاعل مباشرة مع قاعدة بيانات PostgreSQL. في موجه الأوامر أو الطرفية، أدخل:

```bash
psql -U <username> -d <database_name>
```

استبدل `username` باسم المستخدم الخاص بك و`database_name` باسم قاعدة البيانات التي تريد الاتصال بها.

إذا لم تنشئ قاعدة بيانات خاصة بك بعد، يتضمن PostgreSQL تلقائيًا قاعدة بيانات افتراضية تسمى `postgres` عند التثبيت. يمكنك استخدام قاعدة البيانات `postgres` هذه للمتابعة.

بمجرد الاتصال، سترى تغير الموجه إلى:

```bash
postgres=#
```

يعرض اسم قاعدة البيانات التي اتصلت بها وينتظر أوامر SQL.

دعنا نستعرض بعض أوامر SQL الأساسية التي يجب أن تكون على دراية بها. ستلاحظ أنه، حسب الاصطلاح، تنتهي أوامر SQL بفاصلة منقوطة (`;`).

أولًا، يمكنك استخدام هذا الأمر لإنشاء قاعدة بيانات باسم `my_database`:

```sql
CREATE DATABASE my_database;
```

يمكنك كتابة `CREATE` متبوعة بـ `DATABASE`، واسم قاعدة البيانات التي تريد إنشاؤها. لاحظ أن الكلمات المفتاحية في SQL مكتوبة بأحرف كبيرة.

ثم، بمجرد أن تمتلك قاعدة بياناتك، يمكنك الاتصال بها.

كيفية الاتصال بقاعدة البيانات تعتمد حقًا على بيئة العمل والأداة التي تستخدمها.

في موجه الأوامر أو الطرفية، يمكنك استخدام الطريقة التي عرضناها سابقًا. في واجهة psql التفاعلية، يمكنك التبديل بين قواعد البيانات باستخدام الأمر المختصر `\c` متبوعًا باسم قاعدة البيانات هكذا:

```bash
\c my_database
```

لاحظ أن الأمر `\c` والأوامر المختصرة الأخرى (التي تبدأ بشرطة مائلة عكسية) هي جزء من واجهة psql وليست جزءًا من لغة SQL نفسها، لذلك لا تحتاج إلى فاصلة منقوطة لإكمال الأمر.

الآن بعد أن اتصلت بقاعدة بياناتك الجديدة، سترى أن الموجه قد تغير:

```bash
my_database=#
```

بمجرد إنشاء قاعدة البيانات والاتصال بها، يمكنك إنشاء جدول باستخدام الكلمات المفتاحية `CREATE TABLE`:

```sql
CREATE TABLE products (
  id SERIAL,
  name VARCHAR(255)
);
```

في الأمر، تكتب `CREATE TABLE`، متبوعًا باسم الجدول الذي تريد إنشاؤه. في هذه الحالة، سينشئ جدولًا باسم `products`.

ثم، داخل أقواس عادية، تكتب أسماء الأعمدة التي يجب أن يحتويها الجدول وتحدد نوع البيانات للقيم التي ستُخزن في كل عمود.

في المثال، سيكون للجدول عمودان، عمود لمعرف المنتج وعمود آخر لاسم المنتج.

الاصطلاح القياسي لتسمية الجداول والأعمدة هو snake case، بكتابة الكلمات بأحرف صغيرة وفصلها بشرطة سفلية (`_`). على سبيل المثال: `delivery_orders`

هذه بعض أوامر SQL الأساسية التي يجب أن تعرفها للبدء.

باستخدام SQL، يمكنك بسهولة استعلام البيانات والتلاعب بها وتحليلها لاتخاذ قرارات مستنيرة وحل مشكلات العالم الحقيقي.
