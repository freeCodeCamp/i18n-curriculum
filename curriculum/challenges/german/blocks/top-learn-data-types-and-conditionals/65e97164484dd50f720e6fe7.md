# --description--

Nachdem Sie die Punkte aus der Zuweisung der vorherigen Aufgabe durchgegangen sind, sollten Sie nun ein gutes Verständnis dafür haben, wie Strings in JavaScript manipuliert werden können.

Derzeit enthält dieser gegebene String am Ende nach dem Ausrufezeichen ein Leerzeichen. Um dies zu beheben, müssen Sie das nachgestellte Leerzeichen mit einer vordefinierten Methode aus dem String entfernen.

```javascript
let greeting = "Hello World!  ";

console.log(greeting);
```

## --text--

Welche Methode sollte verwendet werden, um nur das nachgestellte Leerzeichen aus dem String zu entfernen?
