# --interactive--

二進位邏輯運算子幫助你評估兩個表達式並根據它們的真值傳回結果。讓我們來看看三個最常見的二進位邏輯運算子：邏輯 AND、邏輯 OR 和 nullish 合併運算子。

邏輯 AND 運算子由雙重和號（`&&`）表述。它會檢查兩個運算元是否皆為 true 並傳回結果。如果兩個運算元皆為真值，它會傳回第二個值，也就是右邊的那個：

:::interactive_editor

```js
const result = true && 'hello';

console.log(result); // hello
```

:::

在上述範例中，因為兩個運算元都是 `true`，所以 `hello` 這段文字會被記錄到主控台。如果任一運算元是 falsy，則會傳回該 falsy 值：

:::interactive_editor

```js
const result = 0 && 3;

console.log(result); // 0
```

:::

由於 `0` 是一個假值，數字 `0` 被記錄到主控台。如果兩個運算元都是假值，則傳回第一個假值：

:::interactive_editor

```js
const result = false && 0;

console.log(result); // false
```

:::

由於 `false` 是一個假值，因此 `false` 被記錄到主控台。邏輯 AND 運算子在你想要檢查多個條件並確保全部為真時非常有用。以下是一個範例：

:::interactive_editor

```js
if (2 < 3 && 3 < 4) {
 console.log('The if block runs'); 
} else {
 console.log('The else block runs');
} 
```

:::

在條件中，因為 `2` 小於 `3` 且 `3` 小於 `4`，所以句子 `The if block runs` 將會被記錄到主控台。

邏輯 OR 運算子會檢查是否至少有一個運算元為真值。如果第一個運算元為真值，它會傳回該值：

:::interactive_editor

```js
const result = 'This is truthy' || false;

console.log(result); // This is truthy
```

:::

如果第一個運算元是假值但第二個是真值，第二個值將會被記錄到主控台：

:::interactive_editor

```js
const result = 0 || 'This is truthy';

console.log(result); // This is truthy
```

:::

在 `if/else` 述語中使用邏輯 OR 運算子是很常見的：

:::interactive_editor

```js
let userInput;

if (userInput || 'Guest') {
 console.log('A user is present');
} else {
 console.log('No user detected');
}
```

:::

由於我們沒有為 `userInput` 變數指定值，它目前是 `undefined`。`if` 述語中的條件會檢查 `userInput` 變數或字串 `Guest` 是否為真值。由於字串 `Guest` 在這種布林值背景關係中為真，字串 `A user is present` 將會被記錄到主控台。

空值合併運算子比邏輯 OR 和邏輯 AND 更為複雜。由雙問號（`??`）表述，它有助於在你只想在第一個值為 `null` 或 `undefined` 時傳回該值的情況下使用。以下是使用空值合併運算子的範例：

:::interactive_editor

```ts
const result = null ?? 'default';

console.log(result); // default
```

:::

由於 `null` 是一個 nullish 值，字串 `default` 將會被記錄到主控台。nullish 合併運算子在只有 `null` 或 `undefined` 會觸發備援或預設值的情況下非常有用。以下是一個處理使用者偏好設定的範例：


:::interactive_editor

```ts
const userSettings = {
 theme: null,
 volume: 0,
 notifications: false,
};

let theme = userSettings.theme ?? 'light';
console.log(theme); // light
```

:::

在上述範例中，我們有一個名為 `userSettings` 的物件，包含 `theme`、`volume` 和 `notifications` 屬性。我們使用點記法像 `userSettings.theme` 來存取 `theme`。你將在未來的課程中學習更多如何操作物件。由於使用者的 `theme` 目前設定為 `null`，因此字串 `light` 將會被記錄到主控台。
