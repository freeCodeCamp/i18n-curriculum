# --explanation--

Le **frazioni** si usano per descrivere parti di un intero.

`One-fourth` (¼) indica una parte su quattro parti uguali. Altri esempi:

- ½ = `one-half`

- ⅓ = `one-third`

- ¾ = `three-fourths`

Nel caso di Alice, un quarto del budget per lo sviluppo è destinato allo sviluppo back-end, il che mostra come il denaro viene suddiviso all’interno di una categoria.
