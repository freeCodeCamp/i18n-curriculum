# --description--

Ora sai che i numeri dall'11 al 19 si formano mettendo la cifra delle unità (1–9) dopo `十 (shí)` (10). La stessa logica vale per i numeri dal 21 al 29.

In cinese, il numero 20 si chiama `二十 (èr shí)`.

Per formare i numeri dal 21 al 29, prendi semplicemente `二十 (èr shí)` (20) e aggiungi direttamente dopo la cifra delle unità (1–9).

| Numero | Cinese |
| :--- | :--- |
| 20 | `二十 (èr shí)` |
| 21 | `二十一 (èr shí yī)` |
| 22 | `二十二 (èr shí èr)` |
| 23 | `二十三 (èr shí sān)` |
| 24 | `二十四 (èr shí sì)` |
| 25 | `二十五 (èr shí wǔ)` |
| 26 | `二十六 (èr shí liù)` |
| 27 | `二十七 (èr shí qī)` |
| 28 | `二十八 (èr shí bā)` |
| 29 | `二十九 (èr shí jiǔ)` |
