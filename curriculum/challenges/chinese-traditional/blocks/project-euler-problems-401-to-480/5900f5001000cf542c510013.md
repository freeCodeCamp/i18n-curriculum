# --description--

對於整數 $a$ 和 $b$，我們定義 $D(a, b)$ 為由拋物線 $y = x^2$ 與直線 $y = ax + b$ 所包圍的區域：$D(a, b) = \\{ (x, y) | x^2 ≤ y ≤ ax + b \\}$。

$L(a, b)$ 定義為包含在 $D(a, b)$ 中的格點數量。例如，$L(1, 2) = 8$ 和 $L(2, -1) = 1$。

我們也定義 $S(N)$ 為所有配對 ($a$, $b$) 的 $L(a, b)$ 之和，條件是 $D(a, b)$ 的面積為有理數且 $|a|,|b| ≤ N$。

我們可以驗證 $S(5) = 344$ 和 $S(100) = 26\\,709\\,528$。

求 $S({10}^{12})$。請將你的答案取模 $\bmod {10}^8$。
