# --description--

`Which` wird verwendet, wenn nach einem oder mehreren Elementen aus einer bekannten Auswahl gefragt wird. Zum Beispiel fragt `Which computer do you prefer?` nach einer Wahl aus bekannten Computern.

`What` ist offener und wird verwendet, wenn jede Antwort möglich ist, wie `What do you like to do?`

`Absolute` bedeutet vollständig oder total. Es wird verwendet, um zu betonen, dass etwas so sehr wie möglich ist. Zum Beispiel bedeutet `My absolute favorite game is chess`, dass Schach ohne Zweifel der absolute Favorit ist.
