# --explanation--

`Currently` significa en el momento presente. Por ejemplo:

Actualmente tenemos tres posiciones abiertas. – Esto significa que hay tres vacantes de trabajo en este momento.

`Working on` significa estar ocupado con una tarea o proyecto específico. Por ejemplo:

`She is working on the design for the new homepage.` – Esto significa que ella está diseñándolo activamente.
