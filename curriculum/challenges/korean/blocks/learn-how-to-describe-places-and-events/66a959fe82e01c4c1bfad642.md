# --explanation--

`Might`는 가능성을 표현하는 데 사용됩니다. 무언가가 가능하지만 확실하지 않다는 뜻입니다. 예를 들어:

- `I might go to the party if I finish my work.` - 내가 갈 수도 있지만 확실하지는 않다.

- `She might call you later.` - 그녀가 전화할 수도 있지만 확실하지는 않다.
