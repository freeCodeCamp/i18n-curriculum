## --answers--

`What happen would if the system crashed?`

### --feedback--

正確的字序是 `What would happen...`。

---

`Would what happen if the system crashed?`

### --feedback--

在疑問句中，`would` 位於 `what` 之後，而非之前。

---

`What would happens if the system crashed?`

### --feedback--

`Would` 後面接動詞原形，所以 `happens` 應該是 `happen`。

---

`What would happen if the system crashed?`

# --explanation--

如果 `wh-` 疑問詞（`when`、`who`、`why`、`how`）使用情態動詞，情態動詞會放在主詞之前。主要動詞保持原形。例如：

- `What can you do to fix this?`－詢問可能的解決方法。

- `Why should we start early?`－尋求建議。

- `How must we prepare?`－詢問需求。

- `When will you finish?`－詢問未來的行動。

- `Where could I find help?`－詢問關於選項，可選方案。

這樣，你就是在詢問什麼是可能的、必要的或預期的。
