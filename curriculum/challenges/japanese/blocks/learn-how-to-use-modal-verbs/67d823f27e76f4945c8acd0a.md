# --explanation--

`Should` は助動詞で、助言や推奨、最善の行動を表しますが、義務を示すものではありません。

- `You should update your password regularly.` - セキュリティのためにパスワードを頻繁に変更するのは良い考えです。

- `You should check for errors in the code before running it.` - コードを実行する前にミスを見つけるのは良い習慣です。

ジェイクが`we should focus on the critical systems first`と言っているので、強い推奨をしているだけで、チームに厳密な義務はありません。
