# --description--

`Would you like...?` se usa a menudo cuando propones o sugieres una actividad. Por ejemplo:

`Would you like to test the new app with me?` - Es una invitación amable para trabajar juntos en las pruebas.

Cuando Linda dice `Would you like to go cycling with me one of these days?`, es similar a decir "Oye, vamos a andar en bicicleta juntos."
