# --description--

在这个任务中，你将学习 `specific` 和 `specify` 之间的区别。`Specific` 是一个形容词，用来描述某事物被清晰定义或详细说明，而 `specify` 是一个动词，意思是详细说明某事。

例如，`specific` 指令可能是 `use port 8080`，而 `specify` 则可以表示说明该细节：`specify the port number.`

`Setup` 指的是某物的布置或准备，通常用于准备软件或设备的上下文中，例如 `setup instructions for the software.`
