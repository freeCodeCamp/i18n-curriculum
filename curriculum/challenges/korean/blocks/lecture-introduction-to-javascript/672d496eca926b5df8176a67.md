# --interactive--

JavaScript에서 데이터 타입은 숫자나 텍스트 같은 저장하는 값의 종류입니다.

변수는 특정 데이터 타입의 값을 저장하는 이름 있는 컨테이너로, 코드 전반에서 참조하고 조작할 수 있게 해줍니다.

수학 시간에 다음과 같은 변수들을 다뤘던 기억이 있을 겁니다:

```md
x = 2
y = 4

x + y
```

`x`와 `y` 같은 변수를 만들고 프로그램 전반에서 참조하며 덧셈 같은 수학 연산을 할 수 있었습니다. 프로그래밍에서도 비슷한 개념입니다. 자신만의 변수 이름을 만들고 값들을 할당할 수 있습니다. 이 값들은 서로 다른 데이터 타입일 수 있습니다.

데이터 타입은 프로그램이 숫자인지, 텍스트인지, 혹은 다른 종류의 데이터인지 이해하도록 돕습니다.

JavaScript에는 프로그램에서 사용할 여러 기본 데이터 타입이 있습니다. 앞으로 수업에서 각 데이터 타입을 더 자세히 살펴볼 것입니다. 지금은 JavaScript의 다양한 데이터 타입에 대한 간단한 소개를 하겠습니다.

먼저 살펴볼 데이터 타입은 `Number` 타입입니다.

`Number`는 정수와 부동 소수점 값을 모두 나타냅니다. 정수의 예로는 `7`, `19`, `90`가 있습니다.

**참고**: `console.log()`는 웹 브라우저의 일부인 콘솔에 정보를 출력하는 함수입니다. 콘솔은 코드를 디버깅할 때 사용합니다. `console.log()`에 대해서는 앞으로 수업에서 더 배울 것입니다. 또한 `//` 기호는 코드에 주석을 추가할 때 사용합니다. 주석은 코드 실행 시 무시되는 자신이나 다른 프로그래머를 위한 메모입니다.

인터랙티브 편집기를 활성화하고 정수 중 일부를 변경해 콘솔에서 업데이트되는 모습을 확인해 보세요.

:::interactive_editor

```js
// Examples of integers
console.log(3);
console.log(5);
console.log(-67);
```

:::

부동 소수점 숫자는 소수점이 있는 숫자입니다. 부동 소수점 숫자의 예로는 `3.14`과 `5.2`이 있습니다.

:::interactive_editor

```js
// Examples of floating point numbers
console.log(3.14);
console.log(7.2);
console.log(-14.5);
```

:::

다음 데이터 타입은 `String`입니다.

`String`는 따옴표로 감싼 문자들의 수열, 즉 텍스트입니다. 큰따옴표를 사용한 문자열 예시는 다음과 같습니다:

:::interactive_editor

```js
console.log("I love to code!");
```

:::

다음은 작은따옴표를 사용한 예시입니다:

:::interactive_editor

```js
console.log('I love to code!');
```

:::

문자열은 이름, 레이블, 알림 메시지 등을 나타낼 때 자주 사용합니다.

JavaScript에서 사용되는 또 다른 데이터 타입은 `Boolean` 타입입니다.

`Boolean`는 `true` 또는 `false` 중 하나의 값을 나타냅니다. 예를 들어, 프로그램이 사용자가 로그인했는지(`true`) 아닌지(`false`)를 확인하고 그에 따라 페이지를 변경할 수 있습니다. 사용자가 로그인했다면 대시보드 페이지를 보여주고, 그렇지 않으면 로그인 페이지를 보여주고 싶을 것입니다.

JavaScript에서 사용되는 다음 두 데이터 타입은 `undefined`와 `null`입니다.

`undefined`는 변수가 선언되었지만 아직 값이 할당되지 않았음을 의미합니다. 다음 수업에서 더 배울 것입니다.

`null`는 변수가 의도적으로 "없음"으로 설정되어 어떤 값도 가지지 않음을 의미합니다. 이 부분도 앞으로 수업에서 더 자세히 다룰 것입니다.

마지막 세 가지 데이터 타입은 좀 더 복잡합니다. 객체, 심볼, 그리고 `BigInt`입니다.

`Object`는 키-값 쌍들의 모음입니다.

```js
{
  name: "Alice",
  age: 30
};
```

객체는 관련 정보를 함께 묶기에 좋습니다. 객체를 다루는 방법은 앞으로 모듈에서 더 배울 것입니다.

`Symbol`는 JavaScript에서 항상 고유하며 변경할 수 없는 특별한 값 타입입니다. 주로 프로퍼티의 고유 레이블이나 식별자를 만들 때 사용합니다:

```js
Symbol('mySymbol');
```

`BigInt`는 `Number` 타입의 한계를 넘는 매우 큰 숫자에 사용됩니다:

```js
1234567890123456789012345678901234567890n;
```

이 예시에서는 매우 큰 숫자 끝에 `BigInt`을 붙여 `n`를 만듭니다.

`Symbol`와 `BigInt`는 덜 자주 사용되지만 여전히 알아두면 좋은 두 가지 타입입니다.

이 데이터 타입들을 이해하면 각 타입이 가진 특성과 동작 방식을 바탕으로 프로그램에서 다양한 종류의 데이터를 다루고 처리할 수 있습니다.
