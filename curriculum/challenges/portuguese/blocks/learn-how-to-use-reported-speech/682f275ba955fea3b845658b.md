# --explanation--

No **discurso indireto**, `to explain` é usado quando alguém dá uma razão ou esclarece algo.Por exemplo:

- Discurso direto: `The server went down because of a power issue.`

- Discurso indireto: `She explained that the server went down due to a power issue.`

Um `system component` é uma parte ou peça de um sistema maior, geralmente em computadores ou redes.Por exemplo:

`The login module is a key system component for user access.` – Isso significa que é uma das partes essenciais do sistema.
