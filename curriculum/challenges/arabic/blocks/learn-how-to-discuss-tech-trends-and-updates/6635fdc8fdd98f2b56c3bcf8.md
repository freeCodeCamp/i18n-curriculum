# --description--

تُستخدم تعبيرات مثل `do you know`، و`could you tell me`، و`do you have any idea` في اللغة الإنجليزية عادةً للاستفسار أو التأكد من المعلومات بطريقة لطيفة. تستخدمها متبوعة بسؤال، لكن بنية السؤال تتحول إلى جملة مثبتة.

أمثلة:

السؤال الأصلي `What time is it?` تغير إلى `Do you know what time it is?`

السؤال الأصلي `How long does it take to read this book?` تغير إلى `Could you tell me how long it takes to read this book?`
