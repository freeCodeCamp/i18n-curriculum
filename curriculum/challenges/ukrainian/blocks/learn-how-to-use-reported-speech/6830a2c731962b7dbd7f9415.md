# --explanation--

`User-friendly` означає, що щось є простим, легким у використанні і не вимагає багато зусиль для розуміння. Це часто використовується для опису програмного забезпечення, вебсайтів або форм, які розроблені з урахуванням користувача. Наприклад:

`The new dashboard is much more user-friendly than the old one.` – Це означає, що людям легше і приємніше користуватися.
