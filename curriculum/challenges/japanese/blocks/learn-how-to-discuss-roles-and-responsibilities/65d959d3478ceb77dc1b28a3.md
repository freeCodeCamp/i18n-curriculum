# --description--

`Work overtime` は通常の勤務時間よりも多く働くことを意味します。例えば、`I usually work overtime during project deadlines` はプロジェクトの締め切りに間に合わせるために追加で働くことを指します。

`Unless` は何かの例外を話したり、前の文が当てはまらなくなる条件を示すときに使います。これは `if it's not the case` を意味します。例えば、`I walk to work unless it rains` は雨が降らなければ歩いて通勤するという意味です。

`Urgent` は非常に重要で、すぐに対応が必要なことを意味します。例えば、`The project is urgent` はプロジェクトをすぐに完成させる必要があることを示します。

否定文では、`don't/doesn't have to` は何かが必須ではないことを表します。例えば、`I don't have to work late` は残業が必要ないことを意味します。
