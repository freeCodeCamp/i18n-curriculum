# --description--

Fügen Sie außerdem einen `box-sizing` von `border-box` zu allem hinzu. Dadurch wird verhindert, dass der von Ihnen hinzugefügte Rahmen die Größe Ihrer Elemente verändert.

# --hints--

Sie sollten die `box-sizing`-Eigenschaft verwenden.

```js
assert.exists(new __helpers.CSSHelp(document).isPropertyUsed('box-sizing'));
```

Sie sollten den vorhandenen `*`-Selektor nutzen.

```js
// Two selectors create two CSSStyleRule objects
assert.lengthOf(new __helpers.CSSHelp(document).getStyleDeclarations('*'), 1);
```

Alle Elemente sollten eine `box-sizing` von `border-box` haben.

```js
const astStyles = new __helpers.CSSHelp(document).getStyle('*');
assert.equal(astStyles.boxSizing, 'border-box');
```
