# --explanation--

`Back end` refere-se à parte do software no lado do servidor — a parte que gerencia dados, lógica e processos nos bastidores. No diálogo, atua como um adjetivo composto e usa `-`. Outro exemplo:

`She's working on the back-end code today.` — Ela está escrevendo ou corrigindo a parte do código que roda nos bastidores.

`Scalability` significa o quão bem um sistema pode lidar com o crescimento, como mais usuários ou maiores quantidades de dados. Por exemplo:

`They tested the system's scalability under high traffic.` — Eles verificaram se o sistema funciona bem quando muitas pessoas o utilizam.

Na frase, Alice está explicando que o back end é essencial para suportar o crescimento, por isso é uma área chave do orçamento.
