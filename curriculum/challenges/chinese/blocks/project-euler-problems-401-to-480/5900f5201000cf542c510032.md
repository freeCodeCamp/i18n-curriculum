# --description--

斐波那契数列 $\\{f_n, n ≥ 0\\}$ 递归地定义为 $f_n = f_{n - 1} + f_{n - 2}$，基本情形为 $f_0 = 0$ 和 $f_1 = 1$。

定义多项式的 $\\{F_n, n ≥ 0\\}$ 为 $F_n(x) = \displaystyle\sum_{i = 0}^n f_ix^i$。

例如，$F_7(x) = x + x^2 + 2x^3 + 3x^4 + 5x^5 + 8x^6 + 13x^7$，且 $F_7(11) = 268\\,357\\,683$。

令 $n = {10}^{15}$。查找和 $\displaystyle\sum_{x = 0}^{100} F_n(x)$ 并将你的答案对 $1\\,307\\,674\\,368\\,000 \\, (= 15!)$ 取模。
