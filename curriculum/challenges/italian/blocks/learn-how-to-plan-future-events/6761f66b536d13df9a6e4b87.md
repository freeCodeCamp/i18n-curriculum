# --explanation--

`Parameterized` deriva dalla parola radice `parameter`, che è un valore o segnaposto usato per controllare come funziona qualcosa.

Quando si dice che una query è `parameterized`, significa che usa valori predefiniti (parametri) invece di inserire direttamente l'input dell'utente.

Questo aiuta a migliorare la sicurezza del database prevenendo certi attacchi come l'iniezione SQL. Per esempio:

- `The developer used parameterized queries to secure the database.` - Questo significa che hanno usato segnaposto per assicurarsi che non potesse essere inserito codice dannoso.
