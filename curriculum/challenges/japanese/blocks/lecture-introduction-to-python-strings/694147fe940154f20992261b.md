# --description--

前のレッスンでは、文字列の各文字がインデックス（ゼロから始まる）で識別でき、ブラケット記法を使ってアクセスできることを学びました。

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
print(my_str[-1]) # d
```

**文字列スライス**は、文字列の一部を抽出したり、特定の部分だけを扱ったりすることができます。基本的な構文は次の通りです。

```python
string[start:stop]
```

あるインデックスから別のインデックスまでの文字を抽出したい場合は、`start` と `stop` のインデックスをコロンで区切るだけです。

```python
my_str = 'Hello world'
print(my_str[1:4]) # ell
```

`stop` のインデックスは含まれないことに注意してください。つまり、`[1:4]` はインデックス `1` から、インデックス `4` の文字は含まずに抽出しています。

`start` と `stop` のインデックスを省略することもでき、その場合Pythonはそれぞれ `0` または文字列の終わりをデフォルトにします。例えば、`start` のインデックスを省略するとどうなるか見てみましょう。

```python
my_str = 'Hello world'
print(my_str[:7])  # Hello w
```

これはインデックス `0` からインデックス `7` の文字まで（ただし含まない）を抽出します。次に、`stop` のインデックスを省略した場合はどうなるかです。

```python
my_str = 'Hello world'
print(my_str[8:])  # rld
```

これはインデックス `8` の文字から文字列の終わりまでを抽出します。

文字列をスライスしても元の文字列は変更されないことに注意してください。

```py
my_str = 'Hello world'
print(my_str[8:])  # rld
print(my_str)  # Hello world
```

`start` と `stop` の両方のインデックスを省略すると、文字列全体を抽出します。

```python
my_str = 'Hello world'
print(my_str[:])  # Hello world
```

`start` と `stop` のインデックスのほかに、スライス内の各インデックス間の増分を指定するオプションの `step` パラメータもあります。

その構文は次の通りです。

```python
string[start:stop:step]
```

以下の例では、スライスはインデックス `0` から始まり、`11` の前で止まり、2文字ごとに抽出しています。

```python
my_str = 'Hello world'
print(my_str[0:11:2])  # Hlowrd
```

`step` パラメータを使った便利なテクニックとして、step を `-1` に設定し、`start` と `stop` を空白にすると文字列を逆順にできます。

```python
my_str = 'Hello world'
print(my_str[::-1]) # dlrow olleH
```
