# --description--

Sei $R(M, N)$ die Anzahl der Gitterpunkte ($x$, $y$), die die Bedingungen $M \lt x \le N$, $M \lt y \le N$ erfüllen und für die $\left\lfloor\frac{y^2}{x^2}\right\rfloor$ ungerade ist.

Wir können überprüfen, dass $R(0, 100) = 3\\,019$ und $R(100, 10\\,000) = 29\\,750\\,422$ gilt.

Bestimmen Sie $R(2 \times {10}^6, {10}^9)$.

**Hinweis:** $\lfloor x\rfloor$ steht für die Abrundungsfunktion (floor function).
