# --description--

Щоб оголосити константу (незмінну) змінну, використовуйте `const` замість `let`:

```javascript
const myBirthday = '16-09-2003';
```

Змінні, оголошені за допомогою `const`, називають «константами». Їм не можна призначати нові значення. Спроба це зробити спричинить помилку:

```javascript
const myBirthday = '16-09-2003';

myBirthday = '01-01-2001'; // error, can't reassign the constant!
```

Коли програміст впевнений, що змінна ніколи не зміниться, він може оголосити її з `const`, щоб гарантувати це і донести цю інформацію до всіх.

---

---

Тип змінної автоматично змінюється на `let`.
