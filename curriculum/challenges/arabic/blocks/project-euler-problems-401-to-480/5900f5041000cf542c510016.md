# --description--

إذا حسبنا $a^2\bmod 6$ لكل $0 ≤ a ≤ 5$ نحصل على: 0، 1، 4، 3، 4، 1.

أكبر قيمة لـ a بحيث $a^2 ≡ a\bmod 6$ هي $4$.

دعنا نُسمّي $M(n)$ أكبر قيمة لـ $a < n$ بحيث $a^2 ≡ a (\text{mod } n)$. إذًا $M(6) = 4$.

احسب $\sum M(n)$ لكل $1 ≤ n ≤ {10}^7$.
