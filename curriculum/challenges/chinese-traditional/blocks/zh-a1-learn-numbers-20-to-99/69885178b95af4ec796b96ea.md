# --description--

在中文裡，數字 80 稱為 `八十 (bā shí)`。

從 81 到 89 的數字，是在 `八十 (bā shí)`（80）後面加上個位數字（1–9）所組成。

| 數字 | 中文 |
| :--- | :--- |
| 80 | `八十 (bā shí)` |
| 81 | `八十一 (bā shí yī)` |
| 82 | `八十二 (bā shí èr)` |
| 83 | `八十三 (bā shí sān)` |
| 84 | `八十四 (bā shí sì)` |
| 85 | `八十五 (bā shí wǔ)` |
| 86 | `八十六 (bā shí liù)` |
| 87 | `八十七 (bā shí qī)` |
| 88 | `八十八 (bā shí bā)` |
| 89 | `八十九 (bā shí jiǔ)` |
