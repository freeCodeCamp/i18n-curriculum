# --description--

‘`Indentar`’ significa mover el texto o `código` hacia adentro desde el `margen` del documento. En programación, ‘`identar`’ se usa para hacer la estructura del `código` más clara y ayudar a entender cómo los bloques de `código` se relacionan entre sí.

Por ejemplo, `en Python, el cuerpo de un bucle o una función se indenta para mostrar que es parte de ese bucle o función.`
