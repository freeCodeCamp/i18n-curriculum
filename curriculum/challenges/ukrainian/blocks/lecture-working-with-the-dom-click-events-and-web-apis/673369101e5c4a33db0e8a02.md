# --interactive--

У попередньому уроці ви дізналися, як працювати з подіями `"click"` і `"input"` за допомогою методу `addEventListener()`. У цьому уроці ви навчитеся видаляти слухачів подій за допомогою методу `removeEventListener()`.

Метод `removeEventListener()` використовується для видалення слухача подій, який раніше було додано до елемента за допомогою методу `addEventListener()`. Це корисно, коли ви хочете припинити слухати певну подію на елементі.

Ось базовий синтаксис для методу `removeEventListener()`:

```js
element.removeEventListener("event", listener);
```

Як і метод `addEventListener()`, метод `removeEventListener()` приймає два аргументи: подію, яку ви хочете видалити, і функцію-слухача, яку було додано раніше.

Також існує додатковий необов’язковий третій аргумент, який можна передати методу `removeEventListener()`. Цей аргумент може бути або `options`, або `useCapture`.

Аргумент `options` — це об’єкт, який визначає опції для слухача подій, наприклад, чи має слухач бути пасивним або виконуватися лише один раз.

Аргумент `useCapture` — це булеве значення, яке вказує, чи має подія захоплюватися під час фази поширення події.

Здебільшого вам потрібно передавати лише аргументи події та слухача методу `removeEventListener()`.

Якщо ми хочемо перемикати колір фону між сірим і синім, то можемо використати слухача подій для цього.

Спершу потрібно отримати доступ до елемента `button` разом з елементом `body`:

```js
const bodyEl = document.querySelector("body");
const btn = document.getElementById("btn");
```

Потім потрібно створити функцію, яка відповідатиме за перемикання між сірим і синім кольорами:

```js
let isBgColorGrey = true;

function toggleBgColor() {
  bodyEl.style.backgroundColor = isBgColorGrey ? "blue" : "grey";
  isBgColorGrey = !isBgColorGrey;
}
```

Ми використовуємо булеву змінну `isBgColorGrey`, щоб відстежувати поточний колір фону. Якщо фон сірий, то змінюємо його на синій, і навпаки.

Потім потрібно додати слухача подій до елемента `button`, щоб викликати функцію `toggleBgColor` при натисканні на кнопку:

```js
btn.addEventListener("click", toggleBgColor);
```

Кожного разу, коли натискають кнопку, колір фону body змінюватиметься між сірим і синім.

Ми можемо оновити HTML, додавши елемент абзацу, який буде використовуватися для видалення слухача подій при наведенні курсору:

```html
<link rel="stylesheet" href="styles.css" />
<h1>Event Listener examples</h1>
<p id="para">MouseOver this text to remove the event listener</p>
<button id="btn">Change background color</button>
<script src="index.js"></script>
```

Щоб видалити слухача подій при наведенні на абзац, ми можемо додати слухача подій до елемента абзацу. Ми використовуємо подію `mouseover` і передаємо функцію, яка видаляє слухача подій з елемента `button`:

```js
const para = document.getElementById("para");
const btn = document.getElementById("btn");

para.addEventListener("mouseover", () => {
  btn.removeEventListener("click", toggleBgColor);
});
```

Коли курсор наведено на абзац, слухач подій для кліку по кнопці видаляється, і колір фону більше не змінюватиметься при натисканні на кнопку.
