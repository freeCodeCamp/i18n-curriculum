# --description--

Вираз `only if` використовується, щоб встановити умову для чогось, що має статися. Це як сказати «це станеться, якщо те станеться». Це спосіб укласти домовленість, де одна дія залежить від іншої.

Наприклад, ваш друг може сказати `I'll go out only if it doesn't rain`. Це означає, що він вийде, але погода має бути гарною — без дощу.
