# --description--

Ora, cambiamo un po' le cose:

```html
<!-- index.html -->

<div class="main">
  <div class="list" id="subsection"></div>
</div>
```

```css
/* rule 1 */
#subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

Nell'esempio sopra, nonostante la regola 2 abbia più selettori di classe rispetto ai selettori ID, la regola 1 è più specifica perché l'ID ha la precedenza sulla classe.
