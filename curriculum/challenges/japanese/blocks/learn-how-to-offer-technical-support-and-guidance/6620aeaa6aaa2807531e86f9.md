# --description--

動詞 `encounter` は、特に予期しないものに出会う、遭遇することを意味します。プログラミングでは、`encountering` 問題やバグに遭遇することは、コードが実行されている間に問題に直面することを意味します。

例えば、`You might encounter an error if the user inputs the wrong type of data.` この文は、予期せずエラーに直面したり対処しなければならない場合があることを意味します。
