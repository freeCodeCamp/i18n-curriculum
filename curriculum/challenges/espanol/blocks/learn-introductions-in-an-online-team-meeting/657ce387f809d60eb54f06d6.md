# --description--

`Go-to person` se usa para describir a una persona que es experta o la mejor persona a la que acudir en un campo específico o para un problema particular.

Por ejemplo, `Él es el chico de referencia para problemas con computadoras` significa que es la persona más adecuada para ayudar con problemas de computadoras.
