# --explanation--

`To set goals` означає визначити конкретні цілі або завдання для досягнення. Наприклад:

`The team set goals to improve customer satisfaction this year.` — це означає, що команда встановила цілі, пов’язані з задоволенням клієнтів.

`Quarter` означає одну четверту року, або тримісячний період. У бізнес-контексті `the next quarter` означає наступний тримісячний період у календарному або фінансовому році. Наприклад:

- `Our sales target for the next quarter is $1 million.` — це означає, що ціль стосується наступного тримісячного періоду.

- `We review our performance at the end of each quarter.` — це стосується оцінки прогресу кожні три місяці.
