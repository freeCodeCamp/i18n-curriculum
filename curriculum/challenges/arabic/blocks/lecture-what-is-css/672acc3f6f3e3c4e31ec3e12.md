# --interactive--

عند العمل مع CSS، غالبًا ما تواجه ثلاثة أنواع مختلفة من سلوك العرض للعناصر: `inline`، و`block`، و`inline-block`.

كل من هذه الخاصيات تؤثر على كيفية تموضع العناصر وكيفية تفاعلها مع العناصر الأخرى في الصفحة.

في هذا الدرس، سنركز على كيفية عمل الخاصية `inline-block` وكيف تختلف عن العناصر المضمَّنة والعناصر الكتلية.

العناصر الكتلية تتصرف كحاويات كبيرة أو "كتل" تشغل العرض الكامل لحاويتها الأصلية. تبدأ دائمًا في سطر جديد، ويمكن تعديل ارتفاعها وعرضها.

العناصر المضمَّنة تشغل فقط المساحة التي تحتاجها. تتدفق ضمن المحتوى المحيط ولا تنتقل إلى سطر جديد.

الخاصية `inline-block` هي مزيج من هذين السلوكين. مثل العناصر المضمَّنة، تبقى عناصر `inline-block` ضمن تدفق النص دون أن تبدأ في سطر جديد.

لكن، على عكس العناصر المضمَّنة، يمكنك تعديل عرض وارتفاع عنصر `inline-block`، تمامًا كما تفعل مع العناصر الكتلية.

باختصار، الفرق الرئيسي بين `inline` و`inline-block` هو أن عناصر `inline` لا يمكن التحكم في حجمها، بينما تسمح عناصر `inline-block` بالتحكم الكامل في الأبعاد مع البقاء ضمن تدفق النص.

لنلق نظرة على مثال.

:::interactive_editor

```html
<link href="styles.css" rel="stylesheet">

<div class="container">
  <span class="inline-block-element element1">Inline-Block</span>
  <span class="inline-block-element element2">Inline-Block</span>
</div>
```

```css
.inline-block-element {
  display: inline-block;
  width: 150px;
  height: 100px;
}

.element1 {
  background-color: lightblue;
}

.element2 {
  background-color: lightgreen;
}
```

:::

في المثال أعلاه، لدينا عنصر `div` يحمل صنف `container`. داخل هذا العنصر `div`، لدينا عنصران `span`.

تم تعيين خاصية `display:inline-block` لكل عنصر من عناصر الـ span مع تحديد العرض والارتفاع أيضًا.

ستظهر عناصر inline-block جنبًا إلى جنب لأنها تتصرف كعناصر مضمَّنة، لكنها أيضًا تملك عرضًا وارتفاعًا محددين، ما يمنحها خصائص العناصر الكتلية.

لكن، إذا أزلت الخاصية `display: inline-block`، فلن يتم تطبيق لا `height` ولا `width` حتى لو عرّفتها بوضوح.

إليك CSS المعدلة:

:::interactive_editor

```html
<link href="styles.css" rel="stylesheet">

<div class="container">
  <span class="inline-block-element element1">Span element</span>
  <span class="inline-block-element element2">Span element</span>
</div>
```

```css
.inline-block-element {
  width: 150px;
  height: 100px;
}

.element1 {
  background-color: lightblue;
}

.element2 {
  background-color: lightgreen;
}
```

:::

في هذا الكود، أزلنا الخاصية `display: inline-block;` لكن أبقينا كل شيء آخر كما هو. هنا، تعود عناصر `span` إلى سلوكها الافتراضي كعناصر مضمَّنة.

نتيجة لذلك، يتم تجاهل العرض والارتفاع المحددين، وتحتل العناصر فقط المساحة اللازمة لمحتواها.

فهم كيفية عمل `inline-block` مفيد لأنك تستطيع استخدامه لإنشاء تخطيطات تتطلب كلًا من المحاذاة والتحكم في الأبعاد ضمن تدفق نص مستمر.
