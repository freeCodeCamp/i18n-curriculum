# --description--

Acha $f(n) = n^2 - 3n - 1$.

Acha $p$ iwe namba msingi.

Acha $R(p)$ iwe nambari chanya ndogo kabisa $n$ kama $f(n)\bmod p^2 = 0$ ikiwa nambari kama hiyo $n$ ipo, vinginevyo $R(p) = 0$.

Acha $SR(L)$ iwe $\sum R(p)$ kwa namba msingi zote zisizozidi $L$.

Tafuta $SR({10}^7)$.
