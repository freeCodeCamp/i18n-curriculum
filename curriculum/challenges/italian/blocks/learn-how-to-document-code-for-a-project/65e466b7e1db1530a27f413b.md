# --description--

Nei sistemi di file dei computer, `root directory` indica la directory principale o di livello superiore. È come il cassetto più alto di un archivio, che contiene tutte le altre cartelle e file.

Un `main folder` è la cartella principale in cui è archiviato un progetto o un insieme di file.

Un `subfolder` è come un cassetto più piccolo all'interno di quello superiore, situato dentro un'altra cartella.

Per esempio, in un progetto chiamato `MyProject`, il `root directory` sarebbe la cartella `MyProject`, e qualsiasi cartella al suo interno, come `Images`, sarebbe un `subfolder`.

## --answers--

Una cartella principale

---

Una sottocartella

### --feedback--

Una sottocartella si trova dentro un'altra cartella. Il `root directory` è la cartella principale, non una sottocartella.

---

Un tipo di file

### --feedback--

Il `root directory` è una cartella, non un tipo di file. È il luogo principale dove sono archiviati i file e le cartelle del progetto.

---

Un linguaggio di programmazione

### --feedback--

Un `root directory` è un concetto nei sistemi di file, non un linguaggio di programmazione. Indica la cartella principale di un progetto.
