# --description--

`plugin` هو مكوِّن يضيف ميزة محددة إلى برنامج حاسوبي موجود. عندما يدعم البرنامج `plugins`، فإنه يتيح التخصيص. 
في سياق بيئة تطوير متكاملة (IDE)، يمكن لـ `plugins` توفير وظائف إضافية، مثل أدوات جديدة للترميز، وتصحيح الأخطاء، أو تحسين واجهة المستخدم.

على سبيل المثال، غالبًا ما يكون منسق الشفرة في بيئة تطوير متكاملة `plugin`.
