# --interactive--

JavaScriptを使っていると、文字列を特定の回数繰り返す必要がある場面に出くわすことがあります。

繰り返しパターンを生成したり、単にテキストを複製したりする場合でも、`repeat()` メソッドは簡単かつ効果的な方法を提供します。

`repeat()` メソッドはJavaScriptの組み込み関数で、文字列を指定した回数繰り返すことができます。基本的な構文は以下の通りです。

```js
string.repeat(count);
```

`string` は繰り返したい文字列で、`count` は繰り返す回数です。例を見てみましょう。

:::interactive_editor

```js
let word = "Hello!";
let repeatedWord = word.repeat(3);
console.log(repeatedWord);  // "Hello!Hello!Hello!"
```

:::

この場合、文字列 `Hello!` が3回繰り返され、`Hello!Hello!Hello!` になります。

`repeat()` メソッドは便利ですが、いくつかの例外や制限事項があります。

`count` パラメータは0以上の数値である必要があります。負の数を渡すと、JavaScriptは `RangeError` をスローします。

```js
let word = "Test";
console.log(word.repeat(-1));  // Throws RangeError: Invalid count value
```

`count` は有限の数値でなければなりません。文字列を無限回繰り返そうとしたり、`Infinity` を回数として使うと、同様に `RangeError` が発生します。

JavaScriptでは、`Infinity` は無限の量を表す特別な値です。有限のどんな数よりも大きい数を示すために使われます。

```js
let word = "Test";
console.log(word.repeat(Infinity));  // Throws RangeError: Invalid count value
```

回数が整数でない場合（例えば`2.5`のような小数）、`repeat()` メソッドは最も近い整数に切り捨てます。

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(2.5));  // "TestTest"
```

:::

回数に`0` を渡すと、`repeat()` メソッドは空の文字列を返します。

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(0));  // ""
```

:::

`repeat()` メソッドは文字列の複製を伴う作業を簡単にし、コードをより簡潔で読みやすくします。

繰り返しのテキストパターンを生成したり、文字でスペースを埋めたりするときに、`repeat()` はループやより複雑なコードを書く手間を省けます。

`repeat()` メソッドに直接数値を渡すだけでなく、数値を格納した変数を渡すこともできます。

:::interactive_editor

```js
let count = 4;
let word = "Test";
let repeatedWord = word.repeat(count);
console.log(repeatedWord); // TestTestTestTest
```

:::

この例では、`count` 変数が繰り返し回数を格納しています。これは繰り返し回数がユーザーの入力やプログラム内の他の動的な値に依存する場合に便利です。
