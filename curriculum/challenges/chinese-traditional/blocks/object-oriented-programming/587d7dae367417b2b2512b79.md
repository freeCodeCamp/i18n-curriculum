# --description--

上一個挑戰中的 `Bird` 和 `Dog` 建構式運作良好。然而，請注意所有使用 `Bird` 建構式創建的 `Birds` 都會自動命名為 Albert、顏色為藍色，且有兩條腿。如果你想要具有不同 name 和 color 值的鳥呢？雖然可以手動更改每隻鳥的屬性，但那將會是很多工作：

```js
let swan = new Bird();
swan.name = "Carlos";
swan.color = "white";
```

假設你正在撰寫一個程式來追蹤飛禽園中數百甚至數千種不同的鳥。創建所有鳥類然後為每一隻更改屬性值會花費大量時間。為了更輕鬆地創建不同的 `Bird` 物件，你可以設計你的 Bird 建構式以接受參數：

```js
function Bird(name, color) {
  this.name = name;
  this.color = color;
  this.numLegs = 2;
}
```

然後將值作為引數傳入 `Bird` 建構式以定義每個獨特的鳥類：`let cardinal = new Bird("Bruce", "red");` 這會產生一個新的 `Bird` 實體，其 `name` 和 `color` 屬性分別設定為 `Bruce` 和 `red`。`numLegs` 屬性仍然設定為 2。`cardinal` 具有這些屬性：

```js
cardinal.name
cardinal.color
cardinal.numLegs
```

建構式更具彈性。現在可以在每個 `Bird` 被產生時定義其屬性，這是 JavaScript 建構式非常有用的一種方式。它們根據共用的特性和行為將物件群組在一起，並定義一個自動化產生它們的藍圖。

# --instructions--

創建另一個 `Dog` 建構式。這次，將它設置為接受參數 `name` 和 `color`，並將屬性 `numLegs` 固定為 4。然後創建一個新的 `Dog`，並將其存入變數 `terrier`。為 `name` 和 `color` 屬性傳遞兩個字串作為引數。

# --hints--

`Dog` 應該接收一個 `name` 的引數。

```js
assert(new Dog('Clifford').name === 'Clifford');
```

`Dog` 應該接收一個 `color` 的引數。

```js
assert(new Dog('Clifford', 'yellow').color === 'yellow');
```

`Dog` 應該有屬性 `numLegs` 設定為 4。

```js
assert(new Dog('Clifford').numLegs === 4);
```

`terrier` 應該使用 `Dog` 建構式來建立。

```js
assert(terrier instanceof Dog);
```
