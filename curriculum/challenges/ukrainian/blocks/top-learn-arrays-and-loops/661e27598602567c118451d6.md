# --description--

Цикл `for` — це інший тип циклу, який використовується для багаторазового виконання блоку коду. Цикл `for` застосовується, коли відома кількість ітерацій. Синтаксис циклу `for` виглядає так:

```javascript
for (initialization; condition; increment/decrement) {
  // code block to be executed
}
```

Наприклад, наступний фрагмент коду виводить числа від 1 до 5:

```javascript
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
```

Щоб ітерувати над масивом за допомогою циклу `for`, можна використати властивість довжина масиву. Наприклад, наступний фрагмент коду виводить елементи масиву:

```javascript
const arr = [1, 2, 3, 4, 5];
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}
```
