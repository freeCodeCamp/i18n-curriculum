# --explanation--

`Either` é usado para introduzir uma escolha entre duas opções. Neste caso, usar contratados ou distribuir o trabalho.

`Contractors` são trabalhadores temporários ou freelancers, frequentemente contratados para um trabalho específico. Por exemplo:

`We hired contractors to help with the new app design.` - Trabalhadores externos estão ajudando no design do aplicativo.

Neste diálogo, você não usa um artigo antes de `part-time contractors` porque é uma categoria geral, não um grupo específico.
