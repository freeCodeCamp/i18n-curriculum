# --explanation--

Sarah의 대화 부분을 들을 때 `if`라는 단어에 주목하세요. 이 단어는 문제의 원인을 소개하며, 어떤 일이 일어나기 위해 필요한 조건을 설명할 때 자주 사용됩니다.

`if`를 찾은 후, 그 조건이 충족되지 않을 때 발생하는 결과(효과)를 문장에서 찾아보세요. Sarah의 문장에서:

- **효과:** `The auto-completion doesn't work`.

- `if`가 소개하는 **원인**: `If the project isn't set up correctly`.

이 관계를 이해하면 문제와 해결책을 명확하게 설명할 수 있습니다.
