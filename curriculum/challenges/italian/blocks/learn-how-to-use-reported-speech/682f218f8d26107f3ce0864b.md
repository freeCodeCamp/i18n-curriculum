# --explanation--

`Of course` è un modo cortese per mostrare accordo o dire `yes` senza esitazione. Significa che chi parla è felice o disposto a fare qualcosa. Per esempio:

`Of course, I'll help you with the report.` – Questo significa che chi parla è contento di aiutare.

`Will` è spesso usato per fare promesse o esprimere una decisione nel momento in cui si parla. Per esempio:

`I'll email you the file this afternoon.` – Questo mostra una promessa di fare qualcosa presto.
