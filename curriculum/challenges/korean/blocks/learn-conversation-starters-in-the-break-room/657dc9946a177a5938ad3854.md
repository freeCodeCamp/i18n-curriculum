# --description--

`depend on`는 어떤 것이 여러 가지에 따라 달라질 수 있음을 말할 때 사용합니다. 예를 들어:

- `The project depends on good teamwork.` - 프로젝트는 팀이 잘 협력할 때만 성공할 것입니다.

- `Developers depend on tests to find errors.` - 프로그래머는 코드에 오류가 있는지 확인하기 위해 테스트가 필요합니다.

`depend` 다음에는 항상 전치사 `on`가 온다는 점을 주목하세요.
