# --interactive--

문자열을 뒤집는 것은 자주 하는 프로그래밍 작업으로, JavaScript에서 문자열과 배열 메서드를 조합해 수행할 수 있습니다. 이 과정은 세 가지 주요 단계로 이루어집니다:

- 문자열을 문자 배열로 나누기
- 배열을 뒤집기
- 문자를 다시 문자열로 합치기

`split()`, `reverse()`, `join()` 메서드를 사용해 각 단계를 살펴보겠습니다.

문자열을 뒤집는 첫 번째 단계는 문자열을 개별 문자 배열로 변환하는 것입니다. `split()` 메서드를 사용해 할 수 있습니다. `split()` 메서드는 문자열을 배열로 나누며, 주어진 구분자를 기준으로 어디서 나눌지 지정합니다. 구분자가 없으면 원래 문자열을 단일 요소로 가진 배열을 반환합니다. 일반적인 구분자 예시는 다음과 같습니다:

- 빈 문자열(`""`) — 문자열을 개별 문자로 나눕니다.

- 공백 한 칸(`" "`) — 공백이 있는 곳마다 문자열을 나눕니다.

- 대시(`"-"`) — 대시가 있는 곳마다 문자열을 나눕니다.

다음은 split 메서드를 사용해 문자 배열을 만드는 예시입니다:

:::interactive_editor

```js
let str = "hello";
let charArray = str.split("");
console.log(charArray); // ["h", "e", "l", "l", "o"]
```

:::

이 예시에서는 `split("")` (빈 문자열을 인수로 전달)로 `hello` 문자열을 개별 문자 배열로 변환합니다. 문자 배열을 얻으면 `reverse()` 메서드를 사용해 배열 요소의 순서를 뒤집을 수 있습니다.

`reverse()` 메서드는 배열을 제자리에서 뒤집는 배열 메서드입니다. 즉, 새 배열을 만들지 않고 원래 배열을 수정합니다. 사용 방법은 다음과 같습니다:

:::interactive_editor

```js
let charArray = ["h", "e", "l", "l", "o"];
charArray.reverse();
console.log(charArray); // ["o", "l", "l", "e", "h"]
```

:::

이 예시에서 `reverse()`는 `charArray` 배열의 요소 순서를 `["h", "e", "l", "l", "o"]`에서 `["o", "l", "l", "e", "h"]`로 뒤집습니다.

마지막 단계는 뒤집힌 문자 배열을 다시 문자열로 변환하는 것입니다. `join()` 메서드를 사용해 할 수 있습니다. `join()` 메서드는 배열의 모든 요소를 지정한 구분자 문자열로 연결해 새 문자열을 만듭니다. 구분자 없이 문자를 합치려면 빈 문자열을 인수로 사용하세요. 예시는 다음과 같습니다:

:::interactive_editor

```js
let reversedArray = ["o", "l", "l", "e", "h"];
let reversedString = reversedArray.join("");
console.log(reversedString); // "olleh"
```

:::

이 예시에서 `join("")` (빈 문자열을 인수로 전달)은 배열의 모든 문자를 구분자 없이 하나의 문자열로 합칩니다.

JavaScript의 문자열은 불변이므로 직접 수정해 뒤집을 수 없습니다. 그래서 배열로 변환하고, 배열을 뒤집은 뒤, 다시 문자열로 변환하는 과정을 거칩니다. 문자열과 배열 메서드를 조합하면 JavaScript에서 문자열을 강력하고 유연하게 다룰 수 있습니다.
