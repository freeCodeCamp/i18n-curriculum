# --interactive--

Die `Canvas`-API ist ein leistungsstarkes Werkzeug, mit dem Sie Grafiken direkt in Ihrer JavaScript-Datei manipulieren können. Alles beginnt mit einem `canvas`-Element in HTML. Dieses Element dient als Zeichenfläche, die Sie mit den Instanzmethoden und Eigenschaften der `Canvas`-API bearbeiten können.

Die `Canvas`-API bietet alles, was Sie brauchen, um beeindruckende Visualisierungen zu erstellen, einschließlich Formen, Text, Animationen und sogar komplexen Spielen. Sie verfügt über Schnittstellen wie `HTMLCanvasElement`, `CanvasRenderingContext2D`, `CanvasGradient`, `CanvasPattern`, `TextMetrics`, die Methoden und Eigenschaften bereitstellen, mit denen Sie Grafiken in Ihrer JavaScript-Datei erstellen können.

Sehen wir uns an, wie die `Canvas`-API funktioniert, damit Sie mit JavaScript Grafiken erstellen können.

Zuerst müssen Sie in Ihrer HTML-Datei ein `canvas`-Element erstellen:

```html
<canvas id="my-canvas"></canvas>
```

Das `canvas`-Element wird durch die `HTMLCanvasElement`-Schnittstelle dargestellt, die Methoden und Eigenschaften zur Manipulation bereitstellt. Zusätzlich können Sie Methoden und Eigenschaften anderer Schnittstellen der `Canvas`-API verwenden.

Sie können Ihrem `canvas` in HTML eine `width` und `height` geben:

```html
<canvas id="my-canvas" width="400" height="400"></canvas>
```

Oder Sie verwenden die `width`- und `height`-Eigenschaften der `HTMLCanvasElement`-Schnittstelle:

```js
const canvas = document.getElementById("my-canvas");
canvas.width = 400;
canvas.height = 400;
```

Im Moment sehen Sie noch nichts auf dem Bildschirm. Nachdem Sie Ihr `canvas`-Element erstellt haben, ist der nächste Schritt, mit der `getContext()`-Methode der `HTMLCanvasElement`-Schnittstelle auf den Zeichenkontext des Canvas zuzugreifen.

Der gebräuchlichste Kontext ist `2d`, mit dem Sie in zwei Dimensionen zeichnen können:

```js
const canvas = document.getElementById("my-canvas");
const ctx = canvas.getContext('2d');
```

Wenn Sie die `ctx`-Variable in der Konsole ausgeben, sehen Sie die Methoden und Eigenschaften von `CanvasRenderingContext2D`, die Sie verwenden können, um Formen, Farben, Linien und mehr zu erstellen, zusammen mit ihren Standardwerten:

```js
console.log(ctx);
```

Sobald Sie den 2D-Kontext haben, können Sie mit dem Zeichnen auf dem Canvas beginnen.

Die `Canvas`-API stellt mehrere Methoden und Eigenschaften zum Zeichnen von Formen, Linien und Text bereit. Eine davon ist die `fillStyle`-Eigenschaft, die Sie mit der `fillRect()`-Methode kombinieren können, um ein Rechteck oder Quadrat zu zeichnen:

:::interactive_editor

```html
<html>
  <head>
  </head>
  <body>
    <canvas id="my-canvas" width="400" height="400"></canvas>
    <script src="index.js"></script>
  </body>
</html>
```

```js
const canvas = document.getElementById("my-canvas");

const ctx = canvas.getContext("2d");

// Set the background color
ctx.fillStyle = "crimson";

// Draw a rectangle
ctx.fillRect(1, 1, 150, 100);
```

:::

`fillRect` nimmt 4 Zahlenwerte entgegen, die jeweils die x-Achse, y-Achse, Breite und Höhe darstellen.

Jetzt ist etwas auf dem Bildschirm zu sehen. Sie können auch Text zeichnen oder sogar eine Animation erstellen. Hier ist ein Canvas, um Text darzustellen:

```html
<canvas id="my-text-canvas" width="300" height="70"></canvas>
```

Um den Text schließlich zu zeichnen, übergeben Sie den Text als erstes Argument an die `fillText()`-Methode, gefolgt von den Werten für die x- und y-Achse:

:::interactive_editor

```html
<html>
  <head>
  </head>
  <body>
    <canvas id="my-text-canvas" width="300" height="70"></canvas>
    <script src="index.js"></script>
  </body>
</html>
```

```js
const textCanvas = document.getElementById("my-text-canvas");

const textCanvasCtx = textCanvas.getContext("2d");

// Set font family and size
textCanvasCtx.font = "30px Arial";

// Set text color
textCanvasCtx.fillStyle = "crimson";

// Draw the text
textCanvasCtx.fillText("Hello HTML Canvas!", 1, 50);
```

:::

Das Ergebnis im Browser wird der rote Text `Hello HTML Canvas!` sein.

Mit der `Canvas`-API können Sie noch viel mehr machen. Zum Beispiel können Sie sie mit `requestAnimationFrame` kombinieren, um benutzerdefinierte Animationen, Visualisierungen, Spiele und mehr zu erstellen.
