# --explanation--

이 문장은 `Second Conditional`를 사용하며, 구조는 다음과 같습니다:

`If` + `Past Simple` 또는 `Past Continuous` 시제, `might / would / could` + 기본 동사. 이 예에서는:

`If they closed`이 조건(폐쇄를 상상하는 것)입니다.

`It would affect`이 결과(그 결과로 일어날 일)입니다.

이 유형의 조건문은 현실과 다른 가상 상황에 대해 이야기할 때 사용합니다.
