# --description--

JavaScript kann auch Typen und Werte mit dem strikten Gleichheitsoperator `===` vergleichen. Dieser Operator prüft, ob die beiden Werte gleich und vom gleichen Typ sind. Zum Beispiel:

```javascript
let x = 5;
let y = "5";

let result = x === y;

console.log(result); // false
```

In diesem Beispiel enthält die Variable `result` den Wert `false`, weil `x` eine Zahl und `y` ein String ist. Wenn Sie jedoch den Operator `==` verwenden, ist das Ergebnis `true`, weil JavaScript den String in eine Zahl umwandelt und die Werte vergleicht.

Dieser Operator heißt strikter Gleichheitsoperator, weil er sowohl Wert- als auch Typgleichheit prüft. Es wird oft empfohlen, den Operator `===` zu verwenden, um unerwartete Ergebnisse beim Vergleichen von Werten zu vermeiden.

Es gibt auch einen strikten Ungleichheitsoperator `!==`, der prüft, ob die beiden Werte ungleich und vom gleichen Typ sind.

Es gibt viele Möglichkeiten, den strikten Gleichheitsoperator zu verwenden, und es ist wichtig zu verstehen, wie er funktioniert.
