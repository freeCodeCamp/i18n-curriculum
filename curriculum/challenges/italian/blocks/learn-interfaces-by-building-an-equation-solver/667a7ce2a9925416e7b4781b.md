# --description--

Il structural pattern matching ti permette di verificare che il soggetto abbia una struttura specifica. Inoltre, associa nomi nel pattern agli elementi del soggetto. Per esempio:

```py
match my_list:
    case [a]:
        print(a)
    case [a, b]:
        print(a, b)
```

Modifica la tua costruzione `match`/`case` per farla corrispondere a `results` invece di `len(results)`. Poi, modifica ogni `case` per usare una lista con il numero appropriato di elementi. Usa `x` per il caso in cui la lista contenga un solo elemento, e `x1` e `x2` per il caso in cui la lista contenga due elementi.

Infine, modifica le f-string per usare i nomi delle variabili usate in ciascun `case`.

import re


class Equation(ABC):

    type: str

                f"'Equation' object takes {self.degree + 1} positional arguments but {len(args)} were given"
            )

            raise TypeError("Coefficients must be of type 'int' or 'float'")

        slope, intercept = self.coefficients.values()
        return {'slope': slope, 'intercept': intercept}
