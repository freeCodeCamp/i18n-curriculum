# --description--

소수 $p$에 대해 $S(p) = (\sum (p - k)!)\bmod (p)$를 $1 ≤ k ≤ 5$에 대해 정의합니다.

예를 들어, $p = 7$일 때,

$$(7 - 1)! + (7 - 2)! + (7 - 3)! + (7 - 4)! + (7 - 5)! = 6! + 5! + 4! + 3! + 2! = 720 + 120 + 24 + 6 + 2 = 872$$

$872\bmod (7) = 4$이므로, $S(7) = 4$입니다.

$5 ≤ p < 100$에 대해 $\sum S(p) = 480$임을 확인할 수 있습니다.

$5 ≤ p < {10}^8$에 대해 $\sum S(p)$를 구하세요.
