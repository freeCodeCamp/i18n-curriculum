# --description--

Ukizingatia nambari kamili mbili, par kwa shimo la gofu na idadi ya michoro aliyochukua mchezaji wa gofu kwenye shimo hilo, rudisha alama ya mchezaji kwa kutumia maneno ya gofu.

Rudisha:

- `"Hole in one!"` ikiwa ilichukua mchoro mmoja.
- `"Eagle"` ikiwa ilichukua michoro miwili chini ya par.
- `"Birdie"` ikiwa ilichukua mchoro mmoja chini ya par.
- `"Par"` ikiwa ilichukua idadi sawa ya michoro na par.
- `"Bogey"` ikiwa ilichukua mchoro mmoja zaidi ya par.
- `"Double bogey"` ikiwa ilichukua michoro miwili zaidi ya par.

```js
function golfScore(par, strokes) {
  if (strokes === 1) return "Hole in one!";

  const diff = strokes - par;

  if (diff === -2) return "Eagle";
  if (diff === -1) return "Birdie";
  if (diff === 0) return "Par";
  if (diff === 1) return "Bogey";
  if (diff === 2) return "Double bogey";
}
```
