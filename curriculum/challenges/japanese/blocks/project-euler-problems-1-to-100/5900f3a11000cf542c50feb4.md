# --description--

5つの中から3つを選ぶ方法はちょうど10通りあります。12345の場合：

<div style='text-align: center;'>123、124、125、134、135、145、234、235、245、345</div>

組み合わせでは、記法として $\\displaystyle \\binom 5 3 = 10$ を使います。

一般に、$\\displaystyle \\binom n r = \\dfrac{n!}{r!(n-r)!}$ で、ここで $r \\le n$、$n! = n \\times (n-1) \\times ... \\times 3 \\times 2 \\times 1$、そして $0! = 1$ です。

$ n = 23 $ になるまで、値が100万を超えることはありません：$\\displaystyle \\binom {23} {10} = 1144066$。

$1 \\le n \\le 100$ の範囲で、$\\displaystyle \\binom n r$ の値が100万を超えるものは、重複を含めていくつありますか？

```js
function combinatoricSelections(limit) {
    const factorial = n =>
        Array.apply(null, { length: n })
            .map((_, i) => i + 1)
            .reduce((p, c) => p * c, 1);

    let result = 0;
    const nMax = 100;

    for (let n = 1; n <= nMax; n++) {
        for (let r = 0; r <= n; r++) {
            if (factorial(n) / (factorial(r) * factorial(n - r)) >= limit)
                result++;
        }
    }

    return result;
}
```
