# --explanation--

`Unsecured`는 보호되지 않았거나 안전하지 않은 상태를 의미하며, 비밀번호나 암호화 같은 보안 조치가 없는 경우를 말합니다. 예를 들어:

`Using an unsecured network can put your personal data at risk.` - 안전하지 않은 네트워크를 설명하며, 해커가 쉽게 데이터를 접근할 수 있음을 뜻합니다.

`Networks`는 컴퓨터나 장치를 연결해 데이터, 자원, 인터넷 접근을 공유하는 시스템입니다. 예를 들어:

`Many businesses rely on secure networks to protect their sensitive information.` - 여기서 `networks`는 기업들이 통신과 데이터 공유를 위해 사용하는 연결된 시스템을 의미합니다.
