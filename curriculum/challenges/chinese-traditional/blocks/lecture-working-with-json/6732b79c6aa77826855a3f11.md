# --interactive--

JavaScript 中有兩個強大的方法用於處理 JSON 資料：`JSON.parse()` 和 `JSON.stringify()`。這些方法通常用於在 JSON 字串和 JavaScript 物件之間轉換。

`JSON.stringify()` 用於將 JavaScript 物件轉換成 JSON 字串。當你想以一種可以輕鬆共享或在系統之間傳輸的格式來儲存或傳送資料時，這非常有用。 

以下是你如何使用 `JSON.stringify()` 方法的說明：

:::interactive_editor

```js
const user = {
  name: "John",
  age: 30,
  isAdmin: true
};

const jsonString = JSON.stringify(user);
console.log(jsonString);
```

:::

`JSON.stringify()` 方法也接受一個稱為 replacer 的選用參數，該參數可以是函式或陣列。以下是使用陣列作為選用 replacer 參數的範例：

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

// result: {"firstName":"Jessica","country":"USA"}
console.log(JSON.stringify(developerObj, ["firstName", "country"]));
```

:::

在這個範例中，我們有一個具有四個屬性的 `developerObj`。當我們使用 `JSON.stringify()` 方法時，我們可以為第二個參數傳入一個陣列，並指定我們想要字串化的屬性。結果將會是一個只包含 `firstName` 和 `country` 屬性的字串化物件。 

另一個 `JSON.stringify()` 方法的選用參數是 spacer 參數。這讓你可以控制字串化結果的縮排：

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

console.log(JSON.stringify(developerObj, null, 2));

/* result
{
  "firstName": "Jessica",
  "isAwesome": true,
  "isMusician": true,
  "country": "USA"
}
*/
```

:::

大多數時間你不會使用 `JSON.stringify()` 方法的這兩個選用參數，但了解它們仍然很有幫助。 

你在程式設計中會經常使用的另一個方法是 `JSON.parse()` 方法。`JSON.parse()` 將 JSON 字串轉換回 JavaScript 物件。當你從網頁伺服器或 `localStorage` 取得 JSON 資料並且需要在你的應用程式中操作這些資料時，這非常有用。你將在未來的課程中學習更多關於 `localStorage` 的內容。 

以下是如何使用 `JSON.parse()` 方法的範例：

:::interactive_editor

```js
const jsonString = '{"name":"John","age":30,"isAdmin":true}';
const userObject = JSON.parse(jsonString);
console.log(userObject);

// Result:
// { name: 'John', age: 30, isAdmin: true }
```

:::

這讓你可以將你的程式中的資料當作一般的 JavaScript 物件來操作，使其更容易操作和使用。
