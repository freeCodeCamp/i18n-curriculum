# --description--

一排中有 $N$ 个座位。$N$ 个人依次按照以下规则坐满这些座位：

1. 以下是N = 15的可能座位安排：
1. 第一个人选择任何座位。
1. 只要不违反规则1，每个后续人员选择离已经就座的人最远的座位。 如果满足该条件的选择不止一个，则该人选择最左边的选择。

请注意，由于规则 1，某些座位肯定会空着，且最多可就座人数少于 $N$（当 $N > 1$ 时）。

以下是 $N = 15$ 的可能座位安排：

<img alt="seating arrangements for N = 15" src="https://cdn.freecodecamp.org/curriculum/project-euler/comfortable-distance-ii.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

我们看到，如果第一个人选择正确，15个座位最多可容纳7人。 我们还可以看到第一个人有9个选择来最大化可能坐着的人数。

设 $f(N)$ 为第一人为了最大化一排中 $N$ 个座位的乘客数所拥有的选择数。因此，$f(1) = 1$，$f(15) = 9$，$f(20) = 6$，且 $f(500) = 16$。

另外，$\sum f(N) = 83$ 对于 $1 ≤ N ≤ 20$，且 $\sum f(N) = 13\\,343$ 对于 $1 ≤ N ≤ 500$。

查找 $\sum f(N)$，其中 $1 ≤ N ≤ {10}^{12}$。给出你的答案的最后 8 位数字。
