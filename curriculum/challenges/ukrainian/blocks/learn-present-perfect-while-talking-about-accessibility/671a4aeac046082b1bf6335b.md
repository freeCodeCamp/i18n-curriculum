# --explanation--

У розробці програмного забезпечення `accessibility` означає створення застосунків таким чином, щоб кожен, включно з людьми з інвалідністю, міг легко ними користуватися. Прикладами є додавання голосових команд, читачів екрана або функцій, як-от навігація за допомогою клавіатури для користувачів з обмеженою рухливістю.
