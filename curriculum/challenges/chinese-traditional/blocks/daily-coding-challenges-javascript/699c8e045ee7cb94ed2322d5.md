# --description--

給定一個表述多米諾骨牌序列的二維陣列，判斷它是否形成有效的串鏈。

- 陣列中的每個元素表述一個多米諾骨牌，且會是由兩個介於 1 到 6（含）之間的數字所組成的陣列。
- 為了使串鏈有效，每個多米諾骨牌的第二個數字必須與下一個多米諾骨牌的第一個數字相符。
- 第一個多米諾骨牌的第一個數字和最後一個多米諾骨牌的最後一個數字不需要匹配任何東西。

function isValidDominoChain(dominoes) {
  for (let i = 0; i < dominoes.length - 1; i++) {
    if (dominoes[i][1] !== dominoes[i+1][0]) return false
  }

  return true;
}
