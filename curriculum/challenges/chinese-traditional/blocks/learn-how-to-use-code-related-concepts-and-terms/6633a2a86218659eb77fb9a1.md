# --description--

助動詞 `can` 用來顯示可能性或能力。例如，在句子 `You can see the stars at night` 中，`can` 顯示在夜間可以看到星星。

`Change` 意味著使某物不同或變得不同。在日常生活中，你可能會在下班後 `change` 衣服，表示你從工作服換成休閒服。

`Stay the same` 意味著保持不變或恆定。例如，`The rules of the game stay the same every year.` 意味著規則每年都不會改變。
