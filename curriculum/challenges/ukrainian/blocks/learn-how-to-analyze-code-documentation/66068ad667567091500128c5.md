# --description--

`library` — це як велика книжкова полиця з безліччю коду, який програмісти можуть використовувати. Уявіть це як кулінарну книгу з багатьма рецептами (функціями), які ви можете застосувати, щоб приготувати (запрограмувати) різні страви (проєкти).

Наприклад, `library` у програмуванні може містити функцію для створення діаграм. Замість того, щоб писати код з нуля, ви можете використати цю функцію, як слідувати рецепту для приготування торта.
