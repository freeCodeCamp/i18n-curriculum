# --interactive--

`id` 属性はHTML要素に一意の識別子を追加します。

こちらは`h1` 要素に`id` が`title` の例です。

`h1` 要素の下に、`h2` 要素を`id` が`"subtitle"` に設定された状態で追加してください。`h2` に好きなテキストを書いても構いません。プレビューウィンドウで変更が反映されるのが見えます。例を操作するには、インタラクティブエディターを有効にしてください。

:::interactive_editor

```html
<h1 id="title">Movie Review Page</h1>

```

:::

JavaScriptやCSS内で`id` 名の`title` を参照できます。こちらは`id` の`title` を参照してテキストの`color` を`red` に変更するCSSの例です。

**注意**：このインタラクティブな例には一部CSSが用意されています。CSSコードを理解しようと心配しなくて大丈夫です。今後のレッスンで詳しく学びます。もしテキストの色を青に変えたい場合は、インタラクティブエディターを有効にして`styles.css` タブをクリックし、`color: red;` を`color: blue;` に変更してください。

:::interactive_editor

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>Review page Example</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
    <h1 id="title">Movie Review Page</h1>
  </body>
</html>
```

```css
#title {
  color: red;
}
```

:::

ハッシュ記号（`#`）が`title` の前にあると、その値を持つ`id` をターゲットにしたいことをコンピューターに伝えます。`id` 名は複数回使ってはいけませんし、常に一意である必要があります。`id` の値についてもう一つ注意点があります。それはスペースを含めてはいけないことです。以下は`main` と`heading` という単語を`id` 属性値に適用した例です。

```html
<h1 id="main heading">Main heading</h1>
```

ブラウザーはこのスペースを`id` の一部とみなすため、スタイリングやスクリプトで望ましくない問題が発生します。`id` 属性値には文字、数字、アンダースコア、ダッシュのみを含めるべきです。

`id` 属性とは対照的に、`class` 属性値は一意である必要はなく、スペースを含めることができます。

こちらは`box` というクラスを`div` 要素に適用した例です。

```html
<div class="box"></div>
```

要素に複数のクラス名を追加したい場合は、名前をスペースで区切ってください。こちらは複数のクラスを`div` 要素に適用した更新例です。

```html
<div class="box red-box"></div>
```

こちらは複数のクラスを複数の`div` 要素に適用した別の例です。

**注意**：このインタラクティブな例には一部CSSが用意されています。CSSコードを理解しようと心配しなくて大丈夫です。今後のレッスンで詳しく学びます。もし最初と3番目のボックスの色を変えたい場合は、インタラクティブエディターを有効にして`styles.css` タブをクリックし、`background-color: red;` を`background-color: black;` に変更してください。

:::interactive_editor

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>Colored boxes example</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
    <div class="box red-box"></div>
    <div class="box blue-box"></div>
    <div class="box red-box"></div>
    <div class="box blue-box"></div>
  </body>
</html>
```

```css
.box {
  width: 100px;
  height: 100px;
}

.red-box {
  background-color: red;
}

.blue-box {
  background-color: blue;
}
```

:::

まとめると、いつ`id` を使い、いつ`class` を使うべきでしょうか？クラスは多くの要素に一連のスタイルを適用したい場合に最適です。特定の要素をターゲットにしたい場合は、値が一意である必要があるため`id` を使うのが最適です。

# --questions--

## --text--

いつ`id` を使い、いつ`class` を使うべきでしょうか？

## --answers--

一つの要素にのみ適用される一意の識別子が必要な場合は`class` を使ってください。

### --feedback--

`id` 名は一意であり、HTMLドキュメント内で一度だけ使うべきです。

---

特定の要素に一意の識別子が必要な場合は`id` を使い、複数の要素にスタイルや動作を適用したい場合は`class` を使ってください。

---

ウェブページの異なる部分で複数の要素を一貫してスタイルしたい場合は`id` を使ってください。
