`They installed new software to prevent viruses.` – 이 소프트웨어가 문제가 생기기 전에 바이러스를 막는다는 뜻입니다.

`incident`는 예상치 못한, 보통 부정적이거나 문제가 되는 이벤트입니다. 예를 들어:

`There was a security incident last night that shut down the server.` – 시스템에 문제가 생겼다는 뜻입니다.
