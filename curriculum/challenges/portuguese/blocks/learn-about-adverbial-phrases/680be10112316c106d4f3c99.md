# --explanation--

`To spread` neste contexto significa distribuir algo por uma área maior ou por diferentes partes. Por exemplo:

`The team spread the project tasks evenly among all members.` – Isso significa que o trabalho foi dividido de forma justa para que cada pessoa tivesse uma quantidade semelhante para fazer.

`Funds` refere-se ao dinheiro disponível para um propósito específico, como um projeto, campanha ou evento. Por exemplo:

`The company set aside special funds for emergency expenses.` – Isso significa que eles reservaram dinheiro para ser usado somente se acontecerem problemas inesperados.
