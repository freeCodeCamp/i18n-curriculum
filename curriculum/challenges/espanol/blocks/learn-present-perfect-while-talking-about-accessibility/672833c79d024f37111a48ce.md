# --questions--

## --text--

¿Qué oración usa correctamente `since` en el tiempo `Present Perfect Continuous`?

## --answers--

`I have worked on the project since last week.`

### --feedback--

Este es el tiempo `Present Perfect Simple`. La forma verbal debe estar en el tiempo `Present Perfect Continuous` (`have been working`) para indicar una acción en curso.

---

`I was testing the program since yesterday.`

### --feedback--

Este es el tiempo `Past Simple`. El tiempo `Present Perfect Continuous` debe usar `have been` en lugar de `was`.

---

`I have been testing them since the latest software update.`

---

`I will have been working on the project since next month.`

### --feedback--

El tiempo `Future Perfect` no es apropiado aquí; `since` se usa con el tiempo `Present Perfect Continuous`.
