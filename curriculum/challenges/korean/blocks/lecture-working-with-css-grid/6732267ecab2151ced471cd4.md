# --interactive--

이전 수업에서는 그리드 항목 사이에 공간을 만드는 방법에 대해 간단히 이야기했습니다. 하지만 이번 수업에서는 그리드 레이아웃에서 `row-gap`, `column-gap` 그리고 `gap` 속성을 사용하는 방법을 더 자세히 살펴보겠습니다.

먼저 CSS 그리드에서 트랙이 무엇인지 복습해야 합니다. 트랙은 인접한 두 그리드 선 사이의 공간입니다. 이 선들은 CSS Grid를 사용할 때 자동으로 생성됩니다. 여기서 트랙은 일반적으로 그리드 레이아웃을 구성하는 행과 열을 의미합니다.

CSS Grid에서 열 사이에 간격을 만들려면 `column-gap` 속성을 사용할 수 있습니다. 이 속성에 허용되는 값은 픽셀, `em` 단위, 백분율 또는 `normal` 키워드가 있습니다.

`normal` 속성에 `column-gap` 값을 사용하면 그리드 레이아웃에서는 `0`이, 다중 열 레이아웃에서는 `1em`이 결과로 나타납니다.

다음은 네 개 열 그리드 레이아웃의 마크업 예시입니다:

```html
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

CSS에서는 display를 grid로 설정하고 `column-gap` 속성을 10픽셀로 지정했습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  height: 100px;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.grid-container div {
  background-color: darkblue;
}
```

:::

예제를 파란색 박스 두 줄로 바꾸고 행 사이에 더 많은 공간을 만들고 싶다면 `row-gap` 속성을 사용할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  column-gap: 10px;
  row-gap: 30px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

이 수정된 예제에서는 `row-gap` 속성을 30픽셀로 설정하고 `grid-template-columns`을 네 개 대신 두 개의 `1fr` 단위만 사용하여 두 줄의 박스를 만듭니다.

`column-gap` 속성과 마찬가지로 `row-gap` 속성에 허용되는 값은 백분율, `em`, 픽셀 등이 있습니다.

행과 열 사이의 간격을 만드는 단축 속성을 사용하려면 `gap` 속성을 사용할 수 있습니다. 기본 구문은 다음과 같습니다:

```css
gap: row-value optional-column-value;
```

`gap` 속성에 하나의 값을 지정하면 그 값이 행과 열 모두에 적용됩니다. 두 개의 값을 지정하면 첫 번째 값은 행에, 두 번째 값은 열에 적용됩니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 30px 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

`gap` 단축 속성에 허용되는 값은 백분율, 픽셀, `em` 그리고 `calc()` 값도 포함됩니다. 하지만 `fr` 단위는 사용할 수 없습니다.

`row-gap`, `column-gap`, `gap` 속성은 CSS Grid 레이아웃에서 항목 사이의 간격을 유연하게 제어하는 방법을 제공합니다. 이 속성들을 사용하면 행과 열 사이에 일관되고 조절 가능한 간격을 가진 시각적으로 매력적인 그리드를 쉽게 만들 수 있습니다.
