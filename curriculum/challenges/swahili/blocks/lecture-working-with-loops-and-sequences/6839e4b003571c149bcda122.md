# --description--

Katika mafundisho ya awali ulijifunza jinsi ya kufanya kazi na mzunguko wa `for`, ambao hutumika kurudia kipande cha msimbo mara fulani. Hapa kuna mfano wa kutumia mzunguko wa `for` kuchapisha kila lugha kutoka kwenye orodha ya `languages` kwenye konsoli:

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for language in languages:
    print(language)
```

Lakini vipi kama ungetaka kufuatilia kielezo cha kila kipengele? Naam, chaguo moja ni kuunda kigezo cha `index` na kuongezeka kwa `1` kwa kila kurudia kwa mzunguko, kama hii:

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

index = 0

for language in languages:
    print(f'Index {index} and language {language}')
    index += 1
```

Ingawa hiyo inafanya kazi, njia rahisi zaidi ya kufanya hivyo ni kwa kutumia kitendakazi cha `enumerate()`. Kitendakazi cha `enumerate()` hufuatilia kielezo cha kitu kinachoweza kurudiwa na hurudisha kitu cha enumerate.

Ikiwa tutapita orodha ya `languages` kwa kitendakazi cha `enumerate()` na kubadilisha thamani yake iliyorejeshwa kuwa orodha kwa kitendakazi cha `list()`, itaonekana hivi:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

list(enumerate(languages))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

Kila kipengele katika kitu cha enumerate (sasa ni orodha) ni tuple inayojumuisha hesabu, ikifuatiwa na thamani kutoka kwa kitu kinachoweza kurudiwa kilichopitishwa kwa kitendakazi cha `enumerate()`.

Sasa, hebu tubadilishe mfano wa awali kutumia kitendakazi cha `enumerate()`:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')
```

Tunafungua hesabu na thamani kwa kila tuple katika kitu cha enumerate kuwa vigezo vinavyoitwa `index` na `language`, mtawalia. Mwisho, vigezo hivyo vyote vinatumika katika f-string inayochapishwa kwenye konsoli kila kurudia kwa mzunguko.

```md
Index 0 and language Spanish
Index 1 and language English
Index 2 and language Russian
Index 3 and language Chinese
```

Hii inafuta haja ya kuunda na kusasisha kigezo cha `index` kwa mkono.

Kitendakazi cha `enumerate()` pia kinakubali hoja ya hiari ya `start` inayobainisha thamani ya kuanzia kwa hesabu. Ikiwa hoja hii itakosekana, basi hesabu itaanzia `0`. Hapa kuna mfano wa kutumia hoja ya hiari ya `start`:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages, 1):
    print(f'Index {index} and language {language}')
```

Na hapa ni jinsi matokeo yatakavyoonekana kwenye konsoli:

```md
Index 1 and language Spanish
Index 2 and language English
Index 3 and language Russian
Index 4 and language Chinese
```

Hadi sasa tumekuwa tukipita juu ya orodha moja tu. Lakini vipi kama unahitaji kurudia juu ya vitu vinavyoweza kurudiwa vingi sambamba? Naam, unaweza kutumia kitendakazi cha `zip()` kwa hilo, ambacho huunganisha orodha kuwa jozi za vipengele na hurudisha kielezo cha tuples.

Ikiwa tutapita orodha ya `developers` na `ids` kwa kitendakazi cha `zip()` na kubadilisha thamani yake iliyorejeshwa kuwa orodha kwa kitendakazi cha `list()`, itaonekana hivi:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

list(zip(developers, ids))
# [('Naomi', 1), ('Dario', 2), ('Jessica', 3), ('Tom', 4)]
```

Na hapa ni mfano wa kutumia kitendakazi cha `zip()` pamoja na mzunguko wa `for` kurudia juu ya `developers` na `ids`:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, dev_id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {dev_id}')
```

Katika mfano huu, `zip()` huunganisha orodha mbili kuwa jozi za vipengele na kurudisha iterator ya tuple. Kisha `for` loop hutenganisha kila tuple kuwa `name` na `dev_id`. Mwishowe, kwa kila kauli ya chapisho, tunachapisha kila `name` na `dev_id` kutoka kwa orodha za `ids` na `developers` mtawalia. Hapa ni jinsi matokeo yanavyoonekana kwenye konsoli:

```md
Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
```

Kitendakazi cha `enumerate()` na `zip()` ni zenye nguvu sana, na zinapochanganywa na mizunguko, zinaweza kufanya msimbo wako kuwa mfupi zaidi.
