# --description--

Katika mafundisho yaliyopita, umezoea kufafanua kitendakazi kwa kutumia neno kuu la `def` kama hii:

```py
def square(num):
    return num ** 2

print(square(4)) # 16
```

Lakini unaposhughulika na kitendakazi cha ngazi ya juu kama `map()` na `filter()`, unaweza kutumia kitendakazi kisicho na jina ndani ya mstari. Hapa ndipo kitendakazi za lambda zinapoingia.

Hivi ndivyo kitendakazi cha `square()` kinavyoonekana kinapobadilishwa kuwa kitendakazi cha lambda:

```py
lambda num: num ** 2
```

Kama ilivyotajwa awali, kitendakazi za lambda hazina majina, hivyo kitendakazi hiki hakina tena jina la `square` linalohusishwa nalo. Kitendakazi za lambda ni nzuri unapohitaji kuzitumia katika kitendakazi cha ngazi ya juu kama hivi:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

Katika mfano huu, tuna orodha ya nambari na tunataka kuunda orodha mpya ya nambari kamili chanya. Hivyo tunapitia kitendakazi cha lambda kama moja ya hoja za kitendakazi cha `filter()` kupata orodha mpya yenye nambari `2` na `4`.

Unapofanya kazi na kitendakazi za lambda ni muhimu kuwa na ufahamu wa mbinu bora. Kwa mfano, si mbinu nzuri kuweka kitendakazi cha lambda kwenye kigezo kama hiki:

```py
numbers = [1, 2, 3, 4, 5]

square = lambda x: x ** 2
squared_numbers = list(map(square, numbers))
print(squared_numbers) # [1, 4, 9, 16, 25]
```

Hii inavunja madhumuni ya kutumia kitendakazi kisicho na jina. Katika kesi hii, unapaswa kutumia kitendakazi cha kawaida, kama hiki:

```py
numbers = [1, 2, 3, 4, 5]

def square(num):
    return num ** 2

squared_numbers = list(map(square, numbers))
print(squared_numbers) # [1, 4, 9, 16, 25]
```

Pia, unapaswa kuepuka kuunda kitendakazi za lambda ambazo ni ngumu kusoma au tata bila sababu, kama hiki:

```py
result = (lambda x: (x**2 + 2*x - 1) if x > 0 else (x**3 - x + 4))(3)
print(result)  # 14
```

Ingawa kitendakazi hiki kinafanya kazi vizuri na kinatoa matokeo sahihi ya `14`, si rahisi kusoma au kuangalia. Katika kesi hii, itakuwa bora kuunda kitendakazi tofauti chenye tamko la `if/else`, kisha kuita kitendakazi hicho:

```py
def calculate_expression(x):
    if x > 0:
        return x**2 + 2*x - 1
    else:
        return x**3 - x + 4

print(calculate_expression(3))  # 14
```

Kitendakazi cha kawaida na kitendakazi cha lambda vyote vina matumizi yao katika programu za Python. Ikiwa unashughulika na usemi wa kihesabu ndani ya mstari mmoja, basi unaweza kuzingatia kutumia kitendakazi cha lambda. Vinginevyo, kutumia kitendakazi cha kawaida ndicho njia sahihi.
