# --description--

Il loop `for` è un altro tipo di loop usato per eseguire un blocco di codice più volte. Il loop `for` si usa quando si conosce il numero di iterazioni. La sintassi del loop `for` è la seguente:

```javascript
for (initialization; condition; increment/decrement) {
  // code block to be executed
}
```

Ad esempio, il seguente frammento di codice stampa i numeri da 1 a 5:

```javascript
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
```

Per iterare su un array usando un loop `for`, puoi usare la proprietà lunghezza dell'array. Ad esempio, il seguente frammento di codice stampa gli elementi di un array:

```javascript
const arr = [1, 2, 3, 4, 5];
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}
```
