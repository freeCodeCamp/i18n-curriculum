# --description--

Дотепер завдання охоплювали конкретні HTML-елементи та їхнє використання. Однак є кілька елементів, які задають загальну структуру вашої сторінки і мають бути включені в кожен HTML-документ.

На початку документа потрібно повідомити браузеру, яку версію HTML використовує ваша сторінка. HTML — це мова, що розвивається, і її регулярно оновлюють. Більшість основних браузерів підтримують останню специфікацію, якою є HTML5. Проте старіші вебсторінки можуть використовувати попередні версії мови.

Ви повідомляєте браузеру цю інформацію, додавши тег `<!DOCTYPE ...>` у першому рядку, де частина `...` — це версія HTML. Для HTML5 ви використовуєте `<!DOCTYPE html>`.

`!` і верхній регістр `DOCTYPE` важливі, особливо для старіших браузерів. `html` не чутливий до регістру.

Далі весь ваш HTML-код має бути обгорнутий у теги `html`. Відкриваючий тег `<html>` йде безпосередньо під рядком `<!DOCTYPE html>`, а закриваючий тег `</html>` — у кінці сторінки.

Ось приклад структури сторінки. Ваш HTML-код має розміщуватися в проміжку між двома тегами `html`.

```html
<!DOCTYPE html>
<html>

</html>
```

# --instructions--

Додайте тег `DOCTYPE` для HTML5 на початок порожнього HTML-документа в редакторі коду. Під ним додайте відкриваючий і закриваючий теги `html`, які обгортають елемент `h1`. Заголовок може містити будь-який текст.

# --hints--

Ваш код має містити тег `<!DOCTYPE html>`.

```js
assert.match(code,/<!DOCTYPE\s+?html\s*?>/gi);
```

Повинен бути один елемент `html`.

```js
assert.lengthOf(document.querySelectorAll('html'), 1);
```

Теги `html` мають обгортати один елемент `h1`.

```js
assert.match(code,/<html>\s*?<h1>\s*?.*?\s*?<\/h1>\s*?<\/html>/gi);
```
