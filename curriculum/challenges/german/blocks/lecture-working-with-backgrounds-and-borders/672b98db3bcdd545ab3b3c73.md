# --interactive--

In CSS gibt es mehrere Möglichkeiten, Rahmen um Bilder hinzuzufügen, die jeweils unterschiedliche Styling-Optionen und Kontrollmöglichkeiten bieten.

Lassen Sie uns einige der gebräuchlichsten und vielseitigsten Techniken erkunden.

Die einfachste Methode, um einem Bild einen Rahmen hinzuzufügen, ist die Verwendung der `border`-Eigenschaft. Diese Eigenschaft ist eine Kurzschreibweise, mit der Sie die Breite, den Stil und die Farbe des Rahmens auf einmal festlegen können.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

Diese CSS-Regel fügt allen `img`-Elementen einen 2 Pixel breiten, durchgezogenen roten Rahmen hinzu. Sie können die Breite, den Stil (wie `dashed`, `dotted` oder `double`) und die Farbe an Ihre Designanforderungen anpassen.

Wenn Sie mehr Kontrolle über einzelne Seiten des Rahmens benötigen, können Sie die spezifischen Rahmen-Eigenschaften für jede Seite verwenden:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border-top: 10px solid red;
  border-right: 10px dashed green;
  border-bottom: 10px dotted blue;
  border-left: 10px double purple;
}
```

:::

Damit können Sie für jede Seite des Bildes individuelle Rahmenstile erstellen.

Eine weitere Möglichkeit, einen Rahmeneffekt zu erzeugen, ist die Verwendung der `outline`-Eigenschaft. Obwohl sie dem Rahmen ähnlich ist, beeinflusst die Outline die Abmessungen oder das Layout des Elements nicht:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  outline: 3px solid gold;
}
```

:::

Dies erzeugt eine goldene Umrandung um das Bild, ohne dessen Größe oder Position zu verändern.

Wenn Sie abgerundete Ecken an Ihrem Rahmen erstellen möchten, können Sie die `border-radius`-Eigenschaft zusammen mit border verwenden:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

Denken Sie daran, dass diese Techniken kombiniert und individuell angepasst werden können, um einzigartige Rahmeneffekte zu erzeugen. Die Wahl der Methode hängt von Ihren spezifischen Designanforderungen und dem gewünschten Komplexitätsgrad ab.
