# --explanation--

Заперечна форма `Present Perfect Continuous` часу (`hasn't/haven't + been + present participle`) використовується, щоб описати тривалу дію, яка не відбувалася, як очікувалося, протягом певного часу. У цьому контексті це означає, що проблема впливала лише на кілька відео, а не на всі.
