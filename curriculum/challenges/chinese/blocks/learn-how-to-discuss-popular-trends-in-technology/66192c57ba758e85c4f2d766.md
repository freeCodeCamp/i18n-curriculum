# --description--

当你说某事 `is likely to`，意味着某事预计或很可能发生。它表示某个特定动作在未来发生的可能性很高。例如，如果某种编程语言易于使用，你可能会说它 `is likely to` 保持流行，意思是它很可能在未来继续流行。
 
在英语中，你可以在动词前使用 `to` 来表示不同的动作。例如，`to eat`、`to run` 或 `to learn`。当你谈论需要发生或可能在将来发生的事情时，你会使用这种结构。例如，`I want to eat`、`She likes to run` 或 `They need to learn`。
 
所以，`to + infinitive` 是你表达可能或应该发生的动作的一种方式，通常表示计划、愿望或可能性。
