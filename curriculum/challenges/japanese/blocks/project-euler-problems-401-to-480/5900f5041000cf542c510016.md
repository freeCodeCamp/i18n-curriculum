# --description--

$0 ≤ a ≤ 5$ の範囲で $a^2\bmod 6$ を計算すると、結果は 0、1、4、3、4、1 となります。

$a^2 ≡ a\bmod 6$ を満たす最大の $a$ の値は $4$ です。

ここで、$M(n)$ を $a^2 ≡ a (\text{mod } n)$ を満たす $a < n$ の最大値と定義します。つまり $M(6) = 4$ です。

$1 ≤ n ≤ {10}^7$ の範囲で $\sum M(n)$ を求めてください。
