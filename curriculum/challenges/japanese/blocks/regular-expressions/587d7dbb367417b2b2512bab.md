# --description--

検索は便利です。しかし、検索したテキストを変更（または置換）もできると、さらに強力になります。

文字列の中で`.replace()`を使って検索と置換ができます。`.replace()`の入力は、まず検索したい正規表現パターンです。2番目のパラメータは、マッチした部分を置換する文字列か、何か処理をする関数です。

```js
let wrongText = "The sky is silver.";
let silverRegex = /silver/;
wrongText.replace(silverRegex, "blue");
```

`replace`の呼び出しは文字列`The sky is blue.`を返します。

置換文字列の中でドル記号（`$`）を使うと、キャプチャグループにアクセスできます。

```js
"Code Camp".replace(/(\w+)\s(\w+)/, '$2 $1');
```

`replace`の呼び出しは文字列`Camp Code`を返します。

# --instructions--

3つのキャプチャグループを使った正規表現`fixRegex`を書いて、文字列`one two three`の中の各単語を検索してください。それから`replaceText`変数を更新して、`one two three`を文字列`three two one`に置換し、その結果を`result`変数に代入してください。置換文字列の中でドル記号（`$`）構文を使ってキャプチャグループを活用することを忘れないでください。

# --hints--

検索と置換には`.replace()`を使うべきです。

```js
assert(__helpers.removeJSComments(code).match(/\.replace\(.*\)/));
```

正規表現は文字列`one two three`を文字列`three two one`に変えるべきです。

```js
assert(result === 'three two one');
```

最後の行は変更しないでください。

```js
assert(__helpers.removeJSComments(code).match(/result\s*=\s*str\.replace\(.*?\)/));
```

`fixRegex`は少なくとも3つのキャプチャグループを使うべきです。

```js
assert(new RegExp(fixRegex.source + '|').exec('').length - 1 >= 3);
```

`replaceText`は括弧で囲まれた部分一致文字列（つまりn番目の括弧で囲まれた部分一致文字列、$nはn番目のキャプチャグループに対応）を使うべきです。

```js
{
  const re = /(\$\d{1,2})+(?:[\D]|\b)/g;
  assert(replaceText.match(re).length >= 3);
}
```
