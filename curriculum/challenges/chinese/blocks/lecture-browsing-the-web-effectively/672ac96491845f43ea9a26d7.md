# --description--

网页浏览器是允许用户访问和浏览万维网的软件应用。它们解释并显示网页内容，包括文本、图像、视频和交互元素。

网页浏览器的主要特色包括地址栏，你可以在其中输入 URL 或查找查询，以及将 HTML、CSS 和 JavaScript 转换为可视网页的渲染引擎。

浏览器还提供书签以储存和组织收藏的网站，扩展以增强功能性，以及各种隐私和安全性特色，如弹出窗口阻止程序和隐私浏览模式。

流行的网页浏览器包括 Google Chrome、Mozilla Firefox、Microsoft Edge、Safari 和 Opera。

网站则是由网页和相关内容组成的集合，这些内容通过一个共同的域名标识，并发布在至少一个网络服务器上。它们是你使用网页浏览器访问的目标。

网站的组件包括域名，它是网站的唯一地址（如 freecodecamp.org），以及网页，它们是构成网站的单个文档。

网站通常还包含多媒体内容，如图像、视频和音频文件，以增强用户体验，以及允许用户交互的表单和按钮等交互元素。

网站可以是静态的，内容很少变化，或者是动态的，内容经常更新或为用户个性化。

搜索引擎是基于网络的工具，旨在帮助用户查找互联网信息。它们使用复杂的算法对网页进行索引和排名，基于与用户查询的相关性。

搜索引擎的关键方面包括网络爬虫，也称为蜘蛛或机器人，它们是系统地浏览网络以发现和索引新内容的程序。

索引进程涉及在庞大的数据库中组织和存储有关网页的信息。

搜索引擎还使用排名算法，这些算法是复杂的系统，根据相关性、权威性和用户体验等因素确定搜索结果的顺序。流行的搜索引擎包括 Google、Bing、Yahoo 和 DuckDuckGo。

浏览器、网站和 `search` 引擎之间的关系是相互关联的。

网页浏览器是用于访问网站和查找引擎的工具。

网站是用户通过浏览器访问或通过搜索引擎查找的目的地。

搜索引擎通过提供有组织的访问，帮助用户发现网络上大量的信息。

理解这些区别和关系对于有效使用互联网和基本的网页素养至关重要。
