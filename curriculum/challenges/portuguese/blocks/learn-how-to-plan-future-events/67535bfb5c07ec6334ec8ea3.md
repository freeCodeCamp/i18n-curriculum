# --explanation--

Preste atenção nas palavras `will affect` e `deadlines` no áudio.Esses são essenciais para entender a questão.

Um `project deadline` é a data final até a qual um projeto deve ser concluído.

`Will affect` é usado para perguntar se algo causará uma mudança ou impacto.Por exemplo:

`The new policy will affect the project deadline.` - Isso significa que a política vai mudar o cronograma do projeto.
