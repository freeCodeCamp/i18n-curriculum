# --explanation--

`To come up` يعني أن شيئًا سيحدث قريبًا أو يتم التخطيط له. على سبيل المثال:

- `We have a meeting coming up next week.` - تم التخطيط لاجتماع الأسبوع المقبل.

- `There are some events coming up this month.` - هناك بعض الأحداث التي ستحدث هذا الشهر.
