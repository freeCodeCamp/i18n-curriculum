# --instructions--

パスワードは決してプレーンテキストで保存してはいけません。パスワードリストが発見された場合に備えて、ハッシュとして保存する必要があります。ただし、すべてのハッシュが同じではありません。

このプロジェクトでは、SHA-1を使ってハッシュ化されたパスワードを解読するパスワードクラッカーを作成し、良いセキュリティの重要性を学びます。

SHA-1でハッシュ化されたパスワードを受け取り、そのパスワードが上位10,000のパスワードのいずれかであればパスワードを返す関数を作成してください。もしSHA-1ハッシュがデータベース内のパスワードのものではなければ、"PASSWORD NOT IN DATABASE" を返してください。

関数は`top-10000-passwords.txt`から各パスワードをハッシュ化し、関数に渡されたハッシュと比較する必要があります。

関数は`use_salts`という名前のオプションの第2引数を取るべきです。これがtrueに設定されている場合、`known-salts.txt`のファイルからの各ソルト文字列を、`top-10000-passwords.txt`の各パスワードの前後に追加してからハッシュ化し、関数に渡されたハッシュと比較してください。

以下は関数のテストに使えるハッシュ化されたパスワードです：

- `b305921a3723cd5d70a375cd21a61e60aabb84ec` は "sammy123" を返すべきです
- `c7ab388a5ebefbf4d550652f1eb4d833e5316e3e` は "abacab" を返すべきです
- `5baa61e4c9b93f3f0682250b6cf8331b7ee68fd8` は "password" を返すべきです

`use_salts` が `True` に設定されている場合にテストできるハッシュ化されたパスワードは以下です：

- `53d8b3dc9d39f0184144674e310185e41a87ffd5` は "superman" を返すべきです
- `da5a4e8cf89539e66097acd2f8af128acae2f8ae` は "q1w2e3r4t5" を返すべきです
- `ea3f62d498e3b98557f9f9cd0d905028b3b019e1` は "bubbles1" を返すべきです

`hashlib` ライブラリはすでにインポートされています。コード内での使用を検討してください。<a href="https://docs.python.org/3/library/hashlib.html" target="_blank" rel="noopener noreferrer nofollow">hashlibについてはこちらをご覧ください</a>。
