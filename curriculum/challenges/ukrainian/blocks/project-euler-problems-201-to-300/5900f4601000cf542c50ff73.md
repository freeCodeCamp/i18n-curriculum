# --description--

Позитивний дріб, чисельник якого менший за знаменник, називають правильним дробом.

Для будь-якого знаменника $d$ існуватиме $d−1$ правильних дробів; наприклад, для $d = 12$:

$$\frac{1}{12}, \frac{2}{12}, \frac{3}{12}, \frac{4}{12}, \frac{5}{12}, \frac{6}{12}, \frac{7}{12}, \frac{8}{12}, \frac{9}{12}, \frac{10}{12}, \frac{11}{12}$$

Ми назвемо дріб, який не можна скоротити, стійким дробом.

Крім того, визначимо стійкість знаменника, $R(d)$, як відношення його правильних дробів, що є стійкими; наприклад, $R(12) = \frac{4}{11}$.

Насправді, $d = 12$ — це найменший знаменник, що має стійкість $R(d) < \frac{4}{10}$.

Знайдіть найменший знаменник $d$, що має стійкість $R(d) < \frac{15\\,499}{94\\,744}$.
