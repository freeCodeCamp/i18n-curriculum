# --description--

让我们定义一个几何三角形为一个边长为整数且满足 $a ≤ b ≤ c$ 的三角形，使其边长形成等比数列，即 $b^2 = a \times c$。

这样一个几何三角形的例子是边长为 $a = 144$、$b = 156$ 和 $c = 169$ 的三角形。

有 $861\,805$ 个几何三角形，其 `\text{perimeter}` ≤ ${10}^6$。

存在多少个几何三角形，其 $\text{perimeter} ≤ 2.5 \times {10}^{13}$？
