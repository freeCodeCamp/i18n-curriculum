# --description--


JavaScript 是驅動網路的程式設計語言。與你之前學過的 HTML 和 CSS 不同，JavaScript 最常用於撰寫邏輯，而非標記。

在這個專案中，你將學習 JavaScript 的基礎知識，並將這些概念應用於建造金字塔產生器。 

金字塔產生器是一個程式，你可以設定字元的型別、金字塔的數量以及金字塔的方向。程式將根據這些輸入來產生金字塔。

點擊「Check your code」按鈕以進行下一步並啟動專案。
