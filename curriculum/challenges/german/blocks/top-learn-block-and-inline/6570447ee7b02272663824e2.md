# --description--

Jetzt, wo Sie ein grundlegendes Verständnis dafür haben, wie das Hinzufügen von Breite und Höhe zu `inline`-Elementen funktioniert, was passiert, wenn Sie einem `inline`-Element einen oberen Außenabstand hinzufügen?

# --questions--    

## --text--

Wie wirkt sich `margin-top` auf `inline`-Elemente aus?

## --answers--

`margin-top` hat keinen Einfluss auf das Layout von `inline`-Elementen

---

`margin-top` erhöht die Höhe von `inline`-Elementen.

---

`margin-top` fügt oberhalb des `inline`-Elements Platz hinzu.

---

`margin-top` beeinflusst nur den horizontalen Abstand von `inline`-Elementen.
