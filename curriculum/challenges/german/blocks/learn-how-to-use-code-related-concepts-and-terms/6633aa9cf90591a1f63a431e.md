# --description--

Der Konnektor `so` wird verwendet, um ein Ergebnis oder eine Auswirkung in Bezug auf das Gesagte einzuführen. Er zeigt oft eine Ursache-Wirkungs-Beziehung zwischen zwei Satzteilen.

Zum Beispiel `It was raining, so we canceled the picnic.` Dies zeigt, dass das Picknick wegen des Regens abgesagt wurde.

# --questions--

## --text--

Welcher Satz verwendet `so` korrekt?

## --answers--

`I was hungry so I am eating.`

### --feedback--

Dieser Satz verwendet `so` fast korrekt, aber die Zeitform im zweiten Teil sollte mit der im ersten übereinstimmen. Korrekt wäre: `I was hungry, so I ate.`

---

`She is late to work, so she misses the meeting.`

### --feedback--

Die Verwendung von `so` deutet auf eine Folge hin, aber die Zeitformen stimmen nicht überein. Es sollte heißen: `She is late to work, so she will miss the meeting.`

---

`It started raining, so we took shelter.`

---

`He likes to read books, so he buys a lamp.`

### --feedback--

Dieser Satz verwendet `so` nicht korrekt, weil der Kauf einer Lampe keine direkte Folge davon ist, dass man gerne Bücher liest. Eine bessere Verwendung wäre: `He likes to read at night, so he bought a lamp.`
