# --description--

لعدد صحيح موجب $n$، عرّف $f(n)$ على أنه أصغر مضاعف موجب لـ $n$ مكتوب بالقاعدة 10 ويستخدم فقط الأرقام $≤ 2$.

بالتالي $f(2) = 2$، $f(3) = 12$، $f(7) = 21$، $f(42) = 210$، $f(89) = 1\\,121\\,222$.

أيضًا، $\displaystyle\sum_{n = 1}^{100} \frac{f(n)}{n} = 11\\,363\\,107$.

احسب $\displaystyle\sum_{n = 1}^{10\\,000} \frac{f(n)}{n}$.
