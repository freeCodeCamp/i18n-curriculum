# --description--

`Occasionally`는 어떤 일이 가끔 일어난다는 뜻입니다. 항상 일어나는 것도 아니고 전혀 일어나지 않는 것도 아니며, 그 중간쯤입니다. 예를 들어:

`I occasionally get a bug in my code.` - 때때로 코드에 실수가 있지만 자주 그런 것은 아닙니다.

이 부사들과 보통 나타내는 빈도를 살펴보세요.

| 부사         | 빈도      | 부사       | 빈도      |
| ------------ | --------- | ---------- | --------- |
| Never        | 0%        | Often      | 70%       |
| Rarely       | 5%        | Generally  | 80%       |
| Seldom       | 15%       | Usually    | 90%       |
| Occasionally | 30%       | Always     | 100%      |
| Sometimes    | 50%       |            |           |
