# --description--

CSSは各HTML要素をそれぞれのボックスとして扱います。これは通常<dfn>CSSボックスモデル</dfn>と呼ばれます。ブロックレベルの要素は自動的に新しい行から始まります（見出し、段落、divなどを想像してください）。一方、インライン要素は周囲のコンテンツ内に配置されます（画像やspanのように）。このような要素のデフォルトの配置は文書の<dfn>通常のフロー</dfn>と呼ばれますが、CSSのpositionプロパティを使うとこれをオーバーライドできます。

要素のpositionが`relative`に設定されると、ページの通常のフロー内での現在の位置に対して*相対的に*どのように移動させるかを指定できます。これはCSSのオフセットプロパティである`left`または`right`、および`top`または`bottom`と組み合わせて使います。これらは要素を通常の位置から何ピクセル、パーセント、またはem単位で*離して*移動させるかを指定します。以下の例では段落を下から10ピクセル離して移動させています。

```css
p {
  position: relative;
  bottom: 10px;
}
```

要素のpositionをrelativeに変更しても通常のフローからは外れません。周囲の他の要素は、その要素がデフォルトの位置にあるかのように振る舞います。

**注意:** ポジショニングはページの視覚的なレイアウトに対して多くの柔軟性と強力な制御を与えます。ただし、要素の位置に関わらず、基盤となるHTMLマークアップは上から下に読んだときに整理されていて意味が通るようにしておくことが重要です。これはスクリーンリーダーのような支援技術に頼る視覚障害のあるユーザーがコンテンツにアクセスするために必要です。

# --instructions--

`position`の`h2`を`relative`に変更し、CSSオフセットを使って通常のフロー内での位置の`top`から15ピクセル離して移動させてください。周囲のh1やp要素の位置には影響がないことに注意してください。

# --hints--

`h2`要素には`position`プロパティを`relative`に設定してください。

```js
const h2Element =document.querySelector('h2')
const h2Style = window.getComputedStyle(h2Element);
assert.equal(h2Style?.position, 'relative');
```

コードではCSSオフセットを使って`h2`を通常の位置の`top`から15px離して相対的に配置してください。

```js
const h2Element =document.querySelector('h2')
const h2Style = window.getComputedStyle(h2Element);
assert.equal(h2Style?.top, '15px');
```
