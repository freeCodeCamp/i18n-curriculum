# --interactive--

## المجموعات في JavaScript

- `Set` هو خيار مدمج لإدارة تجميع البيانات.
- تضمن المجموعات أن كل قيمة تظهر مرة واحدة فقط، مما يجعلها مفيدة لإزالة التكرارات من مصفوفة أو التعامل مع مجموعات من القيم المميزة.
- يمكنك إنشاء `Set` باستخدام دالة البناء `Set()`:

:::interactive_editor

```js
const set = new Set([1, 2, 3, 4, 5]);
console.log(set); // Set { 1, 2, 3, 4, 5 }
```

:::

- يمكن التلاعب بالمجموعات باستخدام هذه الطرق:

- `add()`: يضيف عنصرًا جديدًا إلى الـ `Set`.
  - `delete()`: يحذف عنصرًا من الـ `Set`.
  - `has()`: يتحقق مما إذا كان عنصر موجودًا في الـ `Set`.
  - `clear()`: يمسح كل العناصر من الـ `Set`.
  - `keys()` و `values()`: كلاهما يعيد `SetIterator` يحتوي على قيم الـ `Set`. هما نفس الشيء لأن `keys()` هو اسم مستعار لـ `values()`.
  - `forEach()`: للتكرار على قيم الـ `Set`.

## المجموعات الضعيفة في JavaScript

- `WeakSet` هو تجميع للكائنات يسمح لك بتخزين كائنات مضمونة بشكل ضعيف.

## المجموعات مقابل المجموعات الضعيفة

- على عكس المجموعات، لا يدعم `WeakSet` الأنواع الأولية مثل الأرقام أو السلاسل النصية.
- يخزن `WeakSet` الكائنات فقط، والمرجع لتلك الكائنات "ضعيف"، مما يعني أنه إذا لم يكن الكائن مستخدمًا في أي مكان آخر في شفرتك، يُزال تلقائيًا لتحرير الذاكرة.

## الخرائط في JavaScript

- `Map` هو كائن مدمج يحتفظ بأزواج المفتاح-القيمة تمامًا مثل الكائن.
- تختلف الخرائط عن كائنات JavaScript القياسية بقدرتها على السماح بمفاتيح من أي نوع، بما في ذلك الكائنات والدوال.
- توفر `Map` أداءً أفضل من الكائن القياسي عند الإضافة والحذف المتكرر لأزواج المفتاح-القيمة.
- يمكنك إنشاء `Map` باستخدام دالة البناء `Map()`:

:::interactive_editor

```js
const map = new Map([
  ['flower', 'rose'],
  ['fruit', 'apple'],
  ['vegetable', 'carrot']
]);
console.log(map); // Map(3) { 'flower' => 'rose', 'fruit' => 'apple', 'vegetable' => 'carrot' }
```

:::

- يمكن التلاعب بالخرائط باستخدام هذه الطرق:

- `set()`: يضيف زوج مفتاح-قيمة جديد إلى الـ `Map`.
  - `get()`: يسترجع قيمة مفتاح من الـ `Map`.
  - `delete()`: يحذف زوج مفتاح-قيمة من الـ `Map`.
  - `has()`: يتحقق مما إذا كان مفتاح موجودًا في الـ `Map`.
  - `clear()`: يمسح كل أزواج المفتاح-القيمة من الـ `Map`.

لاحظ أن كلًا من الخرائط والمجموعات تحتوي على خاصية `size` التي تعيد عدد العناصر الفريدة فيها.

## الخرائط الضعيفة في JavaScript

- `WeakMap` هو تجميع لأزواج المفتاح-القيمة مثل `Map`، لكنه يحتوي على مراجع ضعيفة للمفاتيح. يجب أن تكون المفاتيح كائنات والقيم يمكن أن تكون أي شيء تريده.

## الخرائط مقابل الخرائط الضعيفة

- الخرائط الضعيفة تشبه المجموعات الضعيفة في أنها تخزن الكائنات فقط والمرجع لتلك الكائنات "ضعيف".
