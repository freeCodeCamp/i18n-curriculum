# --explanation--

`incident response plan` (`IRP`) допомагає організаціям швидко та ефективно реагувати на кіберзагрози. Наприклад:

- `Every company should have an incident response plan for cyberattacks.` — компаніям потрібен план для вирішення проблем безпеки.

- `Without an incident response plan, it's hard to recover from a data breach.` — без плану усунення проблем безпеки може бути складним.
