# --description--

Las bases de datos relacionales tienen una amplia gama de aplicaciones, incluyendo desarrollo web, sistemas de gestión de inventarios, sistemas de salud, aplicaciones de comercio electrónico y más.

Algunas de las bases de datos relacionales más comunes son MySQL, PostgreSQL, SQLite y Microsoft SQL Server. Vamos a aprender un poco sobre cada una de ellas.

MySQL es un sistema de gestión de bases de datos relacional de código abierto. Es muy popular para el desarrollo web porque es fácil de usar, confiable y eficiente para construir aplicaciones del mundo real. También cuenta con una comunidad activa de desarrolladores que lo apoyan y mantienen.

PostgreSQL se describe oficialmente como "The World's Most Advanced Open Source Relational Database." Es un sistema de base de datos objeto-relacional gratuito y de código abierto conocido por su fiabilidad, integridad de datos y extensibilidad. Por ejemplo, puedes definir tus propios tipos de datos, funciones personalizadas y escribir código en diferentes lenguajes de programación sin necesidad de recompilar tu base de datos.

SQLite es un motor de base de datos SQL gratuito, de código abierto y ligero basado en archivos. Es uno de los motores de base de datos más utilizados en el mundo. SQLite es autónomo y sin servidor. Es una base de datos de configuración cero, lo que significa que no requiere un proceso de configuración inicial, lo cual es ideal para comenzar rápidamente.

Estas tres bases de datos son gratuitas y de código abierto. También hay otras opciones propietarias populares, incluyendo Microsoft SQL Server y Oracle database.

Microsoft SQL server es un sistema de gestión de bases de datos relacional popular desarrollado por Microsoft y utilizado en diversas aplicaciones.

Oracle database es un sistema de gestión de bases de datos relacional escalable que se usa frecuentemente para aplicaciones empresariales que requieren alto rendimiento.

Genial. Ahora que sabes más sobre las diferentes opciones disponibles, veamos cómo puedes instalar PostgreSQL.

Primero, necesitas ir al sitio web oficial: <a href="https://www.postgresql.org/" target="_blank">https://www.postgresql.org/</a>.

Una vez que estés allí, haz clic en el botón "Download".

Esto te llevará a una página donde puedes seleccionar tu sistema operativo. Si haces clic en uno, serás llevado a las instrucciones para ese OS.

Vamos a repasar el proceso de instalación para el sistema operativo Windows, pero el proceso es similar en todos los sistemas operativos.

Desde esta página, puedes descargar el instalador haciendo clic en el enlace "Download the installer".

Esto te llevará a una página donde puedes elegir la versión de PostgreSQL que quieres instalar. Si haces clic en el ícono correspondiente, la descarga debería comenzar.

Luego, en tu carpeta de Descargas o en la carpeta donde guardes tus descargas por defecto, deberías ver el instalador.

Si haces doble clic en él, comenzarás el proceso de configuración. Haz clic en Next.

Elige el directorio donde se instalará PostgreSQL. Verás uno por defecto, pero puedes personalizarlo.

Luego, selecciona el componente que quieres instalar. Hay cuatro componentes posibles:

- PostgreSQL Server, el motor de base de datos en sí.
- pgAdmin 4, una herramienta fácil de usar para la gestión de bases de datos.
- Stack Builder, una herramienta para descargar e instalar software adicional y extensiones relacionadas con PostgreSQL
- Y herramientas de línea de comandos, para interactuar con el servidor PostgreSQL directamente desde tu terminal. Esto incluye psql, un terminal interactivo para ejecutar consultas SQL.

Cuando estés listo, haz clic en Next.

Luego, selecciona un directorio para almacenar tus datos. Verás una ruta predeterminada, pero puedes personalizarla. Cuando estés listo, haz clic en Next.

Ahora necesitarás ingresar una contraseña para el superusuario de la base de datos, el usuario con acceso sin restricciones a todas las bases de datos y objetos. Escribe tu contraseña, vuelve a escribirla y luego haz clic en Next.

Luego, selecciona el número de puerto en el que el servidor debe escuchar. Por defecto, es 5432. Luego, haz clic en Next.

Necesitas elegir una configuración regional para el nuevo clúster de base de datos. Esto es importante para que la base de datos interprete los tipos de datos sensibles a la configuración regional. Luego, haz clic en Next.

Finalmente, verás un resumen de preinstalación. Tómate un momento para revisarlo y haz clic en Next.

Necesitarás confirmar que quieres iniciar el proceso de instalación. Si haces clic en Next, el proceso comenzará. Debería tomar solo unos minutos.

Cuando el proceso haya finalizado, según los componentes que instalaste, puede que tengas que elegir si quieres iniciar Stack Builder para descargar e instalar herramientas, controladores y aplicaciones adicionales.

Si decides iniciar Stack Builder, verás una pantalla donde puedes comenzar la instalación. Alternativamente, puedes hacer clic en Cancelar para hacerlo más tarde.

¡Felicidades! Ahora tienes PostgreSQL instalado en tu computadora. El proceso debería ser similar para otros sistemas operativos. Después de esto, puedes comenzar a usar la base de datos.

Ve a tus aplicaciones y busca pgAdmin, una plataforma de administración y desarrollo de código abierto para PostgreSQL que debería haberse instalado si la seleccionaste en los componentes. Haz clic en pgAdmin 4 para abrirlo.

Verás una pantalla de carga inicial cuando la aplicación se inicie. Luego, deberías ver un panel donde puedes iniciar un nuevo servidor, revisar tus bases de datos, configurar pgAdmin y más. También encontrarás enlaces útiles a la documentación.

Si haces clic en "Servers" en la parte superior izquierda, necesitarás ingresar la contraseña que elegiste durante la instalación.

Después de ingresar tu contraseña, deberías ver una pantalla donde tendrás acceso a tus bases de datos, incluyendo sus tablas.

También encontrarás otra herramienta muy útil, psql, un terminal interactivo para consultas SQL. Para abrirlo, busca psql en tu sistema. Haz clic en él.

Se abrirá el SQL Shell, donde puedes comenzar a ejecutar comandos.

Y así es como puedes instalar PostgreSQL en Windows. El proceso es muy similar para macOS y Linux.

Alternativamente, si lo estás instalando en macOS, puedes usar Homebrew, un gestor de paquetes. Una vez que tengas Homebrew instalado, ejecuta este comando para instalar PostgreSQL:

```bash
brew install postgresql@<version>
```

Puedes especificar la versión que quieres instalar escribiendo el número después del símbolo arroba (`@`).
