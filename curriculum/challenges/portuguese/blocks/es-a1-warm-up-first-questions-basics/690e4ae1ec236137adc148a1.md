# --questions--

## --text--

Qual frase Ángela está usando para responder?

## --answers--

`Estoy bien`

### --feedback--

Esta frase significa "I am okay", não uma resposta a uma introdução.

---

`¿Cómo estás?`

### --feedback--

Esta frase significa "How are you?". É comumente usada para iniciar uma conversa, não como uma resposta a uma apresentação.

---

`Gracias`

### --feedback--

Esta palavra significa "Thank you". Embora seja educado, não é usada para responder a uma apresentação.

---

`Mucho gusto`

## --video-solution--

4

# --explanation--

A frase `Mucho gusto` significa "Prazer em conhecê-lo". 

É a forma educada e mais comum de responder a alguém que se apresenta.

É usado em ambientes formais e informais.
