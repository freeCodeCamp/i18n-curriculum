# --interactive--

CSSのパーセンテージは相対単位で、親要素に対する比率としてサイズや寸法、その他のプロパティを定義できます。パーセンテージの値を使うと、「この要素を親コンテナのX％の大きさにする」という意味になります。

例えば、要素に`width: 50%`を設定すると、その要素は親コンテナの幅の半分を占めます。これにより、パーセンテージは異なる画面サイズに対応するレスポンシブデザインを作るのに非常に便利です。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <div class="box"></div>
</div>
```

```css
.container {
  width: 400px;
  height: 200px;
  background-color: lightgray;
}

.box {
  width: 50%;
  height: 100%;
  background-color: red;
}
```

:::

パーセンテージは、さまざまな画面サイズに合わせて調整される流動的なレイアウトを作るのに最適です。例えば、コンテナに`width: 80%`を設定すると、デバイスに関係なく親の幅の80％を占めるようになります。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="parent">
  <div class="child"></div>
</div>
```

```css
.parent {
  width: 100%;
  height: 300px;
  background-color: lightblue;
}

.child {
  width: 80%;
  height: 100%;
  background-color: red;
}
```

:::

画像にパーセンテージを使うのもよくある方法です。画像に`max-width: 100%`を適用すると、小さい画面でもアスペクト比を保ちながら縮小されます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<img src="https://placehold.co/150x150" alt="Example Product Image" />
```

```css
img {
  max-width: 100%;
  height: auto;
}
```

:::

あまり一般的ではありませんが、フォントサイズにパーセンテージを使ってスケーラブルなタイポグラフィを作ることもできます。例えば、`font-size: 120%`は親のフォントサイズより20％大きくなります。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="text-container">
  Parent text.
  <p class="text">This is some example text.</p>
</div>
```

```css
.text-container {
  font-size: 16px;
}

.text {
  font-size: 120%;
}
```

:::

パーセンテージは垂直方向の中央揃えにも便利です。以下は、`transform`プロパティとパーセンテージを使って要素を垂直方向に中央に配置する例です。

この例では、要素をコンテナの上から50％の位置に配置し、`transform`で自身の高さの半分だけ上に移動させて、垂直方向に中央揃えしています。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="centered"></div>
```

```css
.centered {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
  width: 300px;
  height: 300px;
  background-color: red;
}
```

:::

絶対配置やtransformプロパティの詳細については、今後のレッスンでさらに学びます。

パーセンテージは常に何かに対して相対的であることを覚えておいてください。水平方向のプロパティ（`width`など）は親の幅に対して相対的です。垂直方向のプロパティ（`height`など）は、通常親の高さ（指定されている場合）に対して相対的です。

ただし、パーセンテージベースの寸法を持つ要素をネストすると予期しない結果になる場合があるので注意してください。また、親に高さが定義されていない場合、パーセンテージベースの高さは扱いが難しいことがあります。

まとめると、CSSのパーセンテージは柔軟でレスポンシブなデザインを作るための強力なツールです。要素を親コンテナに対して比例的にスケールさせたい場合や、異なる画面サイズにシームレスに対応するレイアウトが必要な場合に使ってください。
