# --description--

Para ver la lista de compras actualizada, deberá registrar la lista de compras actual y un mensaje corto en la consola.

Dado que este mensaje se utilizará repetidamente durante el taller, es mejor crear una función reutilizable.

Cree una función llamada `getShoppingListMsg` que tome un array como parámetro y devuelva la cadena `"Lista de Compras Actual: "` seguida por el contenido del array proporcionado.

Puede usar literales de plantilla o concatenación de cadenas con el operador `+` para combinar la cadena y el array `shoppingList`.

# --hints--

Debería tener una función llamada `getShoppingListMsg` en su código.

```js
assert.isFunction(getShoppingListMsg);
```

Su función `getShoppingListMsg` debería devolver una cadena.

```js
assert.isString(getShoppingListMsg(shoppingList));
```

Su función `getShoppingListMsg` debería aceptar un array como argumento.


```js
assert.strictEqual(getShoppingListMsg(["Anchovies", "Olives"]), "Current Shopping List: Anchovies,Olives");
```

Su función `getShoppingListMsg` debería aceptar un array y devolver el mensaje `"Lista de Compras Actual: "` seguido por el contenido de ese array. Verifique dos veces los espacios y los errores ortográficos.

```js
assert.strictEqual(getShoppingListMsg(["Apples"]), "Current Shopping List: Apples");
```
