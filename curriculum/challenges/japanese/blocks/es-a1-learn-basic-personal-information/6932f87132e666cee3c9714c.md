# --description--

アンヘラは`¿Cuál es tu` + 名詞の構文を使って質問しています。

そこで終わってもよいのですが、彼女は接続詞`y`と2つ目の名詞を使ってもう一つの情報を加えています。

`y`は2つの名詞をつなぐ接続詞です。彼女が使っている構造は次の通りです：

`¿Cuál es tu` + 名詞 + `y` + 名詞`?`

これにより、話者は同じ質問の中で複数の詳細を尋ねることができます。

# --explanation--

文`¿Cuál es tu nombre y apellido?`では、2つの名詞が`y`でつながれています：

- `Nombre`

- `Apellido`

2つの情報をつなぐのは`y`という言葉です。

アンヘラは最初にその人の`nombre`を尋ね、次に`apellido`を加えています。

この構造により、話者は1つの質問で複数の詳細を尋ねることができます：

`¿Cuál es tu correo y número?`

ここでアンヘラが尋ねている2つ目の情報は`apellido`です。
