# --explanation--

Ein `tooltip` ist eine kleine Nachricht, die erscheint, wenn ein Nutzender mit der Maus über ein Element fährt oder darauf klickt, um zu erklären, was es bewirkt. Zum Beispiel:

`The tooltip tells users what to enter in each field.` – Das bedeutet, eine kleine Nachricht informiert die Nutzenden, was sie tun sollen.

Ein `indicator` ist ein visuelles Zeichen, Symbol oder eine Funktion, die Status, Niveau oder Zustand anzeigt. Zum Beispiel:

`The color indicator turns red when the connection is lost.` – Das bedeutet, das visuelle Zeichen hilft den Nutzenden, den Status des Systems zu verstehen.
