# --interactive--

## Ubunifu wa tovuti unaojibadilisha kulingana na kifaa

- **Maana**: Kanuni kuu ya muundo unaobadilika ni uwezo wa kubadilika – uwezo wa tovuti kurekebisha mpangilio na maudhui yake kulingana na ukubwa wa skrini na uwezo wa kifaa kinachotazamwa.
- **Mifumo ya gridi inayoyeyuka**: Hizi hutumia vipimo vinavyohusiana kama asilimia badala ya vipimo thabiti kama pikseli, kuruhusu maudhui kubadilika ukubwa na kuendelea upya kulingana na ukubwa wa skrini.
- **Picha zinazobadilika**: Hizi zimewekwa kubadilika ukubwa ndani ya vipengele vinavyowazunguka, kuhakikisha hazizidi kontena zao kwenye skrini ndogo.

## Maswali ya vyombo vya habari

- **Maana**: Hii inaruhusu wasanidi programu kutumia mitindo tofauti kulingana na sifa za kifaa, hasa upana wa eneo la kuona.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="mq-example">The background of this paragraph will change when the viewport is at least 768px wide.</p>
```

```css
.mq-example {
  padding: 10px;
  border: 1px solid #ccc;
  background-color: lightyellow;
}

@media screen and (min-width: 768px) {
  /* Styles for screens at least 768px wide */
  .mq-example {
    background-color: lightblue;
  }
}
```

:::

- **`all` Aina ya vyombo vya habari**: Hii inafaa kwa vifaa vyote. Hii ni chaguo-msingi ikiwa hakuna aina ya vyombo vya habari iliyoainishwa.
- **`print` Aina za vyombo vya habari**: Hii inalenga nyenzo zilizo na kurasa na nyaraka zinazotazamwa kwenye skrini katika hali ya onyesho la uchapishaji.
- **`screen` Aina za vyombo vya habari**: Hii inalenga hasa skrini.
- **`aspect-ratio`**: Hii inaelezea uwiano kati ya upana na urefu wa eneo la kuona.

```css
@media screen and (aspect-ratio: 16/9) {
  /* Styles for screens with a 16:9 aspect ratio */
}
```

- **`orientation`**: Hii hutumika kuonyesha kama kifaa kiko katika mwelekeo wa usawa au wima.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="orientation-example">This paragraph's background will change in landscape orientation.</p>
```

```css
.orientation-example {
  padding: 10px;
  border: 1px solid #ccc;
  background-color: lightyellow;
}

@media screen and (orientation: landscape) {
  /* Styles for landscape orientation */
  .orientation-example {
    background-color: lightgreen;
  }
}
```

:::

- **`resolution`**: Hii hutumika kuelezea azimio la kifaa cha pato kwa nukta kwa inchi (dpi) au nukta kwa sentimita (dpcm).

```css
@media screen and (min-resolution: 300dpi) {
  /* Styles for high-resolution screens */
}
```

- **`hover`**: Hii hutumika kujaribu kama njia kuu ya ingizo inaweza kuangalia vitu kwa kuvinjari juu yao.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button>Hover over me</button>
```

```css
button {
  padding: 10px 20px;
  font-size: 1rem;
  background-color: lightgray;
  border: 1px solid #999;
  cursor: pointer;
}

@media (hover: hover) {
  /* Styles for devices that support hover */
  button:hover {
    background-color: yellow;
  }
}
```

:::

- **`prefers-color-scheme`**: Hii hutumika kugundua kama mtumizi ameomba mandhari ya rangi nyepesi au giza.
- **Maswali ya vyombo vya habari na opereta za kimantiki**: Opereta `and` hutumika kuunganisha vipengele vingi vya vyombo vya habari, wakati `not` na `only` zinaweza kutumika kupinga au kutenganisha maswali ya vyombo vya habari.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="logical-example">This paragraph's background will change when the screen is at least 768px wide AND in landscape orientation.</p>
```

```css
.logical-example {
  padding: 10px;
  border: 1px solid #ccc;
  background-color: lightyellow;
}

@media screen and (min-width: 768px) and (orientation: landscape) {
  /* Styles for landscape screens at least 768px wide */
  .logical-example {
    background-color: lightcoral;
  }
}
```

:::

## Mipaka ya kawaida ya vyombo vya habari

- **Maana**: Mipaka ya vyombo vya habari ni pointi maalum katika muundo wa tovuti ambapo mpangilio na maudhui hubadilika ili kuendana na ukubwa tofauti wa skrini. Kuna mipaka ya jumla unayoweza kutumia kulenga simu, vidonge na skrini za kompyuta za mezani. Lakini si busara kujaribu kufuata kila ukubwa wa skrini unaowezekana kwa vifaa tofauti.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="breakpoint-example">This text will change size when the viewport width is at least 768px.</p>
```

```css
.breakpoint-example {
  font-size: 1rem;
  padding: 10px;
  border: 1px solid #ccc;
}

/* Styles for screens wider than 768px */
@media screen and (min-width: 768px) {
  .breakpoint-example {
    font-size: 1.125rem;
    background-color: lightblue;
  }
}
```

:::

- **Vifaa vidogo (simu za mkononi)**: hadi 640px
- **Vifaa vya wastani (vidonge)**: 641px hadi 1024px
- **Vifaa vikubwa (kompyuta za mezani)**: 1025px na zaidi

## Mbinu ya kwanza simu

- **Maana**: Mbinu ya kwanza simu ni falsafa ya muundo na mkakati wa ukuzaji katika ubunifu wa tovuti unaojibadilisha kulingana na kifaa inayoweka kipaumbele kuunda tovuti kwa vifaa vya simu kabla ya kubuni kwa skrini kubwa zaidi.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <p>The width of this container changes based on the viewport size.</p>
</div>
```

```css
/* Base styles for mobile */
.container {
  width: 100%;
  padding: 10px;
  background-color: lightblue;
}

/* Styles for larger screens */
@media screen and (min-width: 768px) {
  .container {
    width: 750px;
    margin: 0 auto;
    padding: 20px;
  }
}

@media screen and (min-width: 1024px) {
  .container {
    width: 960px;
  }
}
```
