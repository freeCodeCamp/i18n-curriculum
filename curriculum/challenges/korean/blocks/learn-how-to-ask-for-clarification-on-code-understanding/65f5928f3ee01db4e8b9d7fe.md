# --description--

이 작업에서는 `to be willing` 구문에 대해 배우세요.

누군가가 무언가를 하겠다고 `willing`다고 말할 때, 이는 친절이나 협력의 마음으로 준비되어 있고 기꺼이 하겠다는 뜻입니다. 예를 들어, `Brian is willing to help Sophie`는 Brian이 Sophie를 도울 준비가 되어 있고 기쁘다는 의미입니다.

이 구문은 무언가를 하거나 누군가를 돕는 데 긍정적인 태도를 나타내기 위해 직장이나 개인적인 상황에서 자주 사용됩니다.

## --answers--

### --feedback--

Brian의 대답은 바쁘다는 뜻이 아니라 도울 준비가 되어 있음을 나타냅니다.

### --feedback--

Brian은 다른 사람을 제안하지 않고 직접 도와주겠다고 했습니다.
