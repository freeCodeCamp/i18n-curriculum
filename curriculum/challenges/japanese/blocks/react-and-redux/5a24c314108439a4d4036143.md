# --description--

React コンポーネントを完成させたので、`state` 内でローカルに実行しているロジックを Redux に移す必要があります。これはシンプルな React アプリを Redux に接続するための最初のステップです。アプリの唯一の機能は、ユーザーからの新しいメッセージを番号なしリストに追加することです。この例は、React と Redux がどのように連携するかを示すためにシンプルにしています。

# --instructions--

まず、アクションタイプ `ADD` を定義し、const `ADD` に設定してください。次に、メッセージを追加するアクションを作成するアクションクリエーター `addMessage()` を定義します。このアクションクリエーターには `message` を渡し、返される `action` にメッセージを含める必要があります。

次に、メッセージの状態を管理するリデューサー `messageReducer()` を作成してください。初期状態は空の配列とします。このリデューサーは、状態に保持されているメッセージの配列にメッセージを追加するか、現在の状態を返すようにします。最後に、リデューサーを渡して Redux ストアを作成してください。

# --hints--

const `ADD` は存在し、その値は文字列 `ADD` と等しくなければなりません。

```js
assert(ADD === 'ADD');
```

アクションクリエーター `addMessage` は、`type` が `ADD` と等しく、`message` が渡されたメッセージと等しいオブジェクトを返す必要があります。

```js
assert(
  (function () {
    const addAction = addMessage('__TEST__MESSAGE__');
    return addAction.type === ADD && addAction.message === '__TEST__MESSAGE__';
  })()
);
```

`messageReducer` は関数でなければなりません。

```js
assert(typeof messageReducer === 'function');
```

ストアは存在し、初期状態は空の配列に設定されている必要があります。

```js
assert(
  (function () {
    const initialState = store.getState();
    return typeof store === 'object' && initialState.length === 0;
  })()
);
```

ストアに対して `addMessage` をディスパッチすると、状態に保持されているメッセージの配列に新しいメッセージがイミュータブルに追加される必要があります。

```js
assert(
  (function () {
    const initialState = store.getState();
    const isFrozen = DeepFreeze(initialState);
    store.dispatch(addMessage('__A__TEST__MESSAGE'));
    const addState = store.getState();
    return isFrozen && addState[0] === '__A__TEST__MESSAGE';
  })()
);
```

`messageReducer` は、他のアクションで呼び出された場合は現在の状態を返す必要があります。

```js
assert(
  (function () {
    const addState = store.getState();
    store.dispatch({ type: 'FAKE_ACTION' });
    const testState = store.getState();
    return addState === testState;
  })()
);
```

# --solutions--

```jsx
const ADD = 'ADD';

const addMessage = (message) => {
  return {
    type: ADD,
    message
  }
};

const messageReducer = (state = [], action) => {
  switch (action.type) {
    case ADD:
      return [
        ...state,
        action.message
      ];
    default:
      return state;
  }
};

const store = Redux.createStore(messageReducer);
```
