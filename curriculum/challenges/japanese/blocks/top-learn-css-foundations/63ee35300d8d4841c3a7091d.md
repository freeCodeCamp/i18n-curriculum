# --description--

セレクターを使う別の方法として、区切りなしでリストのようにチェーンする方法があります。例えば、次のようなHTMLがあったとします。

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection preview">This is where a preview for a post might go.</p>
</div>
```

`subsection`クラスを持つ2つの要素があり、それぞれに独自のスタイルがありますが、もし`header`という2つ目のクラスも持つ要素だけに別のルールを適用したい場合はどうしますか？その場合、CSSで両方の`class`セレクターを次のようにチェーンできます。

```css
.subsection.header {
  color: red;
}
```

`.subsection.header`がすることは、`subsection`と`header`の両方のクラスを持つ要素を選択することです。`.subsection`と`.header` `class`セレクターの間にスペースがないことに注目してください。この構文は、複数のセレクターをチェーンする際に基本的にどんな組み合わせでも機能しますが、複数の型セレクターをチェーンすることはできません。

これはクラスとIDをチェーンする場合にも使えます。以下の例をご覧ください。

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection" id="preview">This is where a preview for a post might go.</p>
</div>
```

上記の2つの要素を次のように組み合わせることができます。

```css
.subsection.header {
  color: red;
}

.subsection#preview {
  color: blue;
}
```

一般的に、複数の型セレクターをチェーンすることはできません。なぜなら、要素は同時に2つの異なる型であることはできないからです。例えば、`div`と`p`のように2つの型セレクターをチェーンすると、`divp`というセレクターになりますが、これは存在しない`<divp>`要素を探そうとするため機能しません。

  ...
