# --explanation--

`Present Continuous` 时态通常强调说话时刻发生的临时状态或问题。例如：

- `The server isn't running diagnostics.`——强调诊断进程当前未在进行。

- `Developers aren't debugging the application.` —— 表示开发者当前没有进行该活动。

`Properly` 意味着以正确或预期的方式做某事。

在这句话中，`aren't booting properly` 指的是当前未能正确启动的计算机。
