# --explanation--

`Not only ... but also ...` 是一个常用结构，用于强调两个相关的语句都为真，且第二个语句通常更令人惊讶或更重要。例如：

`She is not only skilled but also very creative.` – 这意味着她有两个强项，而且两者都很重要。

`Adaptable` 意味着某物可以根据情况轻松改变或调整。例如：

`This program is adaptable to different types of teams.` – 这意味着它在多种设置中都能良好运行。
