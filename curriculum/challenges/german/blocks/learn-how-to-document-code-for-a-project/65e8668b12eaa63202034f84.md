# --description--

Eine `convention` ist wie eine Regel oder eine Art, wie die meisten Menschen zustimmen, Dinge zu tun, besonders bei der Arbeit. Zum Beispiel ist es beim Programmieren eine übliche Regel, Code in einem bestimmten Stil zu schreiben.

`To follow` bedeutet, sich an diese Regeln zu halten oder das zu tun, was diese Regeln vorgeben. Das Befolgen von Programmierregeln bedeutet also, Code so zu schreiben, wie die meisten Menschen es für am besten halten.
