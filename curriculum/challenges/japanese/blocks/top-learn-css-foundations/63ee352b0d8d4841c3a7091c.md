# --description--

もし2つの要素グループがいくつかのスタイル宣言を共有していたらどうしますか？

```css
.read {
  color: white;
  background-color: black;
  /* several unique declarations */
}

.unread {
  color: white;
  background-color: black;
  /* several unique declarations */
}
```

`.read`と`.unread`の両方のセレクターは`color: white;`と`background-color: black;`の宣言を共有していますが、それ以外はそれぞれ独自の宣言をいくつか持っています。繰り返しを減らすために、これら2つのセレクターをカンマ区切りのリストとしてまとめることができます。

```css
.read,
.unread {
  color: white;
  background-color: black;
}

.read {
  /* several unique declarations */
}

.unread {
  /* several unique declarations */
}
```

上記の例（グループ化ありとなし）はどちらも同じ結果になりますが、2番目の例は宣言の繰り返しを減らし、`color`または`background-color`のどちらかを両方のクラスに対して一度に編集しやすくなります。
