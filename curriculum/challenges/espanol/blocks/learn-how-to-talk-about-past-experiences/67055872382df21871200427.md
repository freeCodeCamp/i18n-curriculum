# --explanation--

`Properly` es la forma adverbial del adjetivo `proper`. Ambas palabras pueden expresar hacer algo de manera correcta o adecuada, pero su uso varía según la estructura de la oración. Por ejemplo:  

- `The application didn't function properly after the update.` - Esto significa que las actualizaciones introdujeron errores en el código. El adverbio `properly` modifica al verbo `function`.

- `This wasn't a proper configuration of the database.` - El adjetivo `proper` describe al sustantivo `configuration`.

Aquí, James usa `properly` para describir que el sistema no se estaba conectando a la base de datos de la manera correcta.
