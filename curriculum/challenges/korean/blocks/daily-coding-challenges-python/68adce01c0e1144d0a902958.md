# --description--

문자열이 주어졌을 때, 유효한 IPv4 주소인지 확인하세요. 유효한 IPv4 주소는 점(`.`)으로 구분된 네 개의 정수로 구성됩니다. 각 숫자는 다음 조건을 만족해야 합니다:

- 0부터 255 사이여야 합니다(포함).
- 선행 0이 없어야 합니다(예: 0은 허용되지만 01은 허용되지 않습니다).
- 숫자 문자만 허용됩니다.

# --solutions--

```py
def is_valid_ipv4(ipv4):
    parts = ipv4.split(".")    

    if len(parts) != 4:
        return False

    for part in parts:
        if not part.isdigit():
            return False

        num = int(part)
        if num < 0 or num > 255:
            return False

        if len(part) > 1 and part.startswith("0"):
            return False

    return True
```
