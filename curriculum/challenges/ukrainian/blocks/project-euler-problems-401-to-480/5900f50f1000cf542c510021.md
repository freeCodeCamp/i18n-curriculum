# --description--

Нехай $n$ — додатне ціле число. Цілою трійкою ($a$, $b$, $c$) називають трійку факторизації числа $n$, якщо:

- $1 ≤ a ≤ b ≤ c$
- $a \times b \times c = n$.

Визначимо $f(n)$ як $a + b + c$ для трійки факторизації ($a$, $b$, $c$) числа $n$, яка мінімізує $\frac{c}{a}$. Можна довести, що ця трійка є унікальною.

Наприклад, $f(165) = 19$, $f(100\\,100) = 142$ та $f(20!) = 4\\,034\\,872$.

Знайдіть $f(43!)$.
