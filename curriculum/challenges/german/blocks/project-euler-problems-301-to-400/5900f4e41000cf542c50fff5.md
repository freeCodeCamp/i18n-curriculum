# --description--

Sei $S_n$ eine Ganzzahlfolge, die mit folgendem Pseudozufallszahlengenerator erzeugt wird:

$$\begin{align}
        S_0 & = 290\\,797 \\\\
  S_{n + 1} & = {S_n}^2\bmod 50\\,515\\,093
\end{align}$$

Sei $A(i, j)$ das Minimum der Zahlen $S_i, S_{i + 1}, \ldots, S_j$ für $i ≤ j$. Sei $M(N) = \sum A(i, j)$ für $1 ≤ i ≤ j ≤ N$.

Wir können überprüfen, dass $M(10) = 432\\,256\\,955$ und $M(10\\,000) = 3\\,264\\,567\\,774\\,119$ gilt.

Finden Sie $M(2\\,000\\,000\\,000)$.
