# --description--

Para hacer una pregunta usando `there is`, hay que invertir el orden de la fórmula, anteponiendo el verbo `to be` al adverbio (is there), como se muestra a continuación:

| Affirmative | Interrogative |
| --- | --- |
| `There is an ATM nearby.` | `Is there an ATM nearby?` |

Esta fórmula se usa para preguntar si en un área concreta existe el lugar u objeto en el cual estamos interesados.

Completa la siguiente oración.
