# --instructions--

在這個挑戰中，你將使用迴歸演算法來預測醫療費用。

你會得到一個包含不同人員及其醫療費用資訊的資料集。使用這些資料來根據新資料預測醫療費用。

這個筆記本的前兩個儲存格匯入函式庫和資料。

請確保將類別資料轉換為數字。使用 80% 的資料作為 `train_dataset`，並使用 20% 的資料作為 `test_dataset`。

從這些資料集中將 `pop` 出 "expenses" 行，以創建名為 `train_labels` 和 `test_labels` 的新資料集。訓練模型時使用這些標籤。

創建一個模型並使用 `train_dataset` 訓練它。執行此筆記本中的最後一個儲存格以檢查你的模型。最後一個儲存格將使用未見過的 `test_dataset` 來檢查模型的泛化能力。

要通過挑戰，`model.evaluate` 必須傳回低於 3500 的平均絕對誤差。這表示它能正確預測醫療費用，誤差在 3500 美元以內。

最後一個儲存格也會使用 `test_dataset` 預測支出並繪製結果圖。
