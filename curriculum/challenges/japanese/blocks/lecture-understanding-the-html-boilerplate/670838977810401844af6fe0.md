# --description--

`link` 要素について学び、外部スタイルシートにリンクする方法を見ていきましょう。

`link` 要素は、スタイルシートやサイトアイコンなどの外部リソースにリンクするために使われます。外部CSSファイルに対して`link` 要素を使う基本的な構文は以下の通りです。

```html
<link rel="stylesheet" href="./styles.css" />
```

`rel` 属性は、リンクされたリソースとHTMLドキュメントの関係を指定するために使われます。この場合、リンクされたリソースが`stylesheet` であることを指定する必要があります。

HTMLとCSSを別々のファイルに分けることはベストプラクティスとされています。開発者はすべてをHTMLドキュメント内に書く代わりに、外部CSSファイルに対して`link` 要素を使います。

`href` 属性は、外部リソースのURLの場所を指定するために使われます。

例の中の`dot` に続くスラッシュは、コンピューターに現在のフォルダーまたはディレクトリ内の`styles.css` ファイルを探すよう指示しています。

`link` 要素は、以下の例のように`head` 要素の中に配置する必要があります。

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Examples of the link element</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

プロのコードベースでは、複数の`link` 要素が異なるスタイルシート、フォント、アイコンにリンクするために使われているのをよく見かけます。以下は、外部のGoogleフォント「Playwright Cuba」にリンクするために`link` 要素を使った例です。

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Playwrite+CU:wght@100..400&display=swap"
  rel="stylesheet"
/>
```

Google Fontsは、どのプロジェクトでも使える無料のオープンソースのカスタムフォントセットです。使いたいフォントを選ぶと、Googleが必要なHTMLとCSSのコードを提供してくれます。この例では、`preconnect` の値が`rel` 属性に指定されており、ブラウザーに`href` 属性で指定された値への早期接続を作成させています。これは外部リソースの読み込み時間を短縮するためです。

`link` 要素のもう一つの一般的な使い方は、アイコンへのリンクです。以下はfaviconにリンクする例です。

```html
<link rel="icon" href="favicon.ico" />
```

faviconは「favorite icon」の略で、通常ブラウザーのタブにサイトタイトルの隣に表示される小さなアイコンです。多くのウェブサイトはブランドアイコンを表示するためにfaviconを使います。
