# --description--

Jake fornece uma resposta usando `it isn't`, que é uma contração de `it is not`.É uma forma de negar ou dizer "não" a uma afirmação ou pergunta.Contrações são uma forma de combinar palavras para uma fala mais natural.Por exemplo:

- Pergunta: `Is the software updated?`

- Resposta: `No, it isn't updated.` ou `No, it is not updated.`

Jake também usa `it's`, a contração de `it is`, ao falar sobre o sistema de cartão com deslize.Vamos ver o quanto você entende deles no contexto do diálogo.
