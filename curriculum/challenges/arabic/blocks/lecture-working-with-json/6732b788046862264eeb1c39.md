# --description--

ما هو JSON وكيف تصل إلى القيم باستخدام ترميز الأقواس وترميز النقطة؟

JSON هو اختصار لـ JavaScript Object Notation. هو تنسيق بيانات خفيف الوزن يعتمد على النص ويُستخدم عادة لتبادل البيانات بين خادم وتطبيق ويب.

أحد أسباب شعبية JSON في تطوير الويب هو أنه قابل للتحليل آليًا وسهل القراءة للبشر.

بما أن JSON مستقل عن اللغة، يمكنك بسهولة إرسال بيانات JSON من تطبيق Java إلى تطبيق Python، أو من تطبيق JavaScript إلى تطبيق C#.

يدعم JSON العديد من أنواع البيانات بما في ذلك الكائنات، والمصفوفات، والسلاسل النصية، والقيم المنطقية، والقيمة null، والأرقام.

إليك مثال على كائن JSON:

```js
{
  "name": "Alice",
  "age": 30,
  "isStudent": false,
  "list of courses": ["Mathematics", "Physics", "Computer Science"]
}
```

كما ترى، يستخدم JSON أزواج المفتاح-القيمة لتخزين المعلومات، ويفصل بين كل زوج وآخر بفاصلة. يجب أن يكون كل مفتاح محاطًا بعلامات اقتباس مزدوجة، وإلا ستظهر لك رسالة خطأ.

للوصول إلى البيانات من كائن JSON، يمكنك استخدام ترميز النقطة أو ترميز الأقواس. في هذا المثال، نستخدم ترميز النقطة للوصول إلى `age` من كائن JSON:

```js
import data from "./example.json" with { type: "json" };

console.log(data.age);
```

يستخدم هذا المثال تحديدًا ما يُعرف بعبارة `import`، التي تستورد كائن JSON إلى هذا الملف لنتمكن من الوصول إليه. ستتعلم المزيد عن عبارة `import` في درس لاحق.

يمكنك أيضًا استخدام ترميز الأقواس للوصول إلى المعلومات من كائنات JSON. إليك مثالًا للوصول إلى مصفوفة `list of courses`:

```js
import data from "./example.json" with { type: "json" };

console.log(data["list of courses"]);
```

يكون استخدام ترميز الأقواس مفيدًا هنا بشكل خاص لأن المفتاح يحتوي على عدة كلمات مفصولة بمسافات. لو حاولنا استخدام ترميز النقطة، سينتج عن ذلك خطأ.

باختصار، JSON هو تنسيق متعدد الاستخدامات يمكنه تخزين العديد من أنواع البيانات، بما في ذلك المصفوفات والكائنات المسكنة. باستخدام ترميز النقطة أو ترميز الأقواس، يمكنك بسهولة الوصول إلى القيم المخزنة داخل كائن JSON.
