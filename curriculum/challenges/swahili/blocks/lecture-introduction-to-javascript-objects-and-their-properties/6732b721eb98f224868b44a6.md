# --interactive--

Kuna njia kadhaa za kufuta sifa kutoka kwa kitu, ambapo opereta `delete` ni njia rahisi na inayotumika zaidi.

Unapotumia `delete`, inafuta sifa iliyochaguliwa kutoka kwa kitu. Hapa kuna mfano wa jinsi ya kutumia opereta `delete`:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  job: "Engineer"
};

delete person.job;

console.log(person.job); // undefined
```

:::

Katika mfano huu, tunaanzia na kitu cha `person` ambacho kina sifa tatu: `name`, `age`, na `job`. Kisha, tunatumia opereta `delete` kufuta sifa ya `job`. Baada ya kufuta, kitu cha `person` hakina tena sifa ya `job`.

Njia nyingine ya kufuta sifa ni kwa kutumia mgawo wa ufumbuzi wa muundo pamoja na vigezo vya ziada. Njia hii haifuti sifa kwa kweli, bali huunda kitu kipya kisicho na sifa zilizotajwa:

:::interactive_editor

```js
const person = {
  name: "Bob",
  age: 25,
  job: "Designer",
  city: "New York"
};

const { job, city, ...remainingProperties } = person;

// { name: "Bob", age: 25 }
console.log(remainingProperties);
```

:::

Katika mfano huu, tunatumia ufumbuzi wa muundo kutoa `job` na `city` kutoka kwa kitu cha `person`, na kukusanya sifa zilizobaki katika kitu kipya kinachoitwa `remainingProperties`. Hii huunda kitu kipya kisicho na sifa za `job` na `city`.

Kuelewa jinsi ya kufuta sifa kutoka kwa vitu ni ujuzi muhimu katika programu za JavaScript. Inakuwezesha kudhibiti vitu kwa njia ya mabadiliko, na kusafisha data zisizohitajika.
