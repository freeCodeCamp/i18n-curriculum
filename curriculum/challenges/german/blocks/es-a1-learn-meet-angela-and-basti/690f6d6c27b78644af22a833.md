# --description--

Der Ausdruck **`Tengo` + [Alter]+ `años`** ist die gebräuchlichste Art, im Spanischen sein Alter anzugeben. Er entspricht dem deutschen „Ich bin [Alter] Jahre alt“.

Er ist die Antwort auf die Frage `¿Cuántos años tienes?` („Wie alt bist du?“).

# --explanation--

Sebastián sagt `Tengo veintiún años`, was „Ich bin einundzwanzig Jahre alt“ bedeutet.

Das Wort `veintiuno` bedeutet 21.

Wenn man das Alter angibt, wird das Wort `veintiún` verwendet, wenn es direkt vor dem Wort `años` („Jahre“) steht. Diese verkürzte Form wird unmittelbar vor maskulinen Pluralnomen benutzt.

Nachdem er sein Alter genannt hat, gibt Sebastián die Frage mit `¿Y tú?` an Ángela zurück.
