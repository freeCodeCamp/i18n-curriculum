## --text--

Який із наведених способів є правильним для додавання методу до класу?

## --answers--

```py
class Dog:
    set bark(self):
        print("Woof!")
```

---

```py
class Dog:
    def bark(self):
        print("Woof!")
```

---

```py
class Dog:
    method bark(self):
        print("Woof!")
```

---

```py
class Dog:
    def bark:
        print("Woof!")
```
