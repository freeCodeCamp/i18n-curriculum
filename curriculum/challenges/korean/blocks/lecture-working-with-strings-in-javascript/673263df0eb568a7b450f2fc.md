# --interactive--

JavaScript에서 문자열을 다룰 때, JavaScript 엔진이 잘못 해석할 수 있는 특수 문자를 포함해야 할 때가 있습니다.

두 가지 흔한 작업은 문자열 내에서 줄바꿈을 만들고, 따옴표 같은 특정 문자를 올바르게 표시되도록 이스케이프하는 것입니다.

JavaScript를 포함한 많은 프로그래밍 언어에서는 이스케이프 시퀀스라는 특수 문자를 사용해 문자열에 줄바꿈을 만들 수 있습니다. 줄바꿈을 위한 가장 흔한 이스케이프 시퀀스는 `\n`입니다.

예를 들어, 문자열을 여러 줄로 나누고 싶다면 줄바꿈을 시작할 위치에 `\n`를 사용합니다:

:::interactive_editor

```js
let poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

`\n` 이스케이프 시퀀스는 JavaScript에 그 지점에 줄바꿈을 삽입하라고 알려주어, 문자열이 여러 줄에 걸쳐 표시되도록 만듭니다.

문자열을 다룰 때 또 다른 중요한 개념은 문자를 이스케이프하는 것입니다. 때로는 JavaScript가 다른 용도로 사용하는 문자를 문자열에 포함해야 할 때가 있습니다. 예를 들어 따옴표가 그렇습니다.

이스케이프하지 않고 문자열 안에 따옴표를 그냥 사용하면, JavaScript가 문자열이 끝난 것으로 오해해 오류가 발생할 수 있습니다.

예를 들어, 다음은 오류를 일으킵니다:

```js
let statement = "She said, "Hello!"";
```

JavaScript는 `"said,"` 단어 뒤에서 문자열이 끝난다고 생각해 혼란스러워하지만, `"Hello!"` 주위의 따옴표는 문자열의 일부여야 합니다.

이 문제를 고치려면, 따옴표 앞에 역슬래시(`\`)를 넣어 이스케이프할 수 있습니다:

:::interactive_editor

```js
let statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

역슬래시는 JavaScript에 따옴표를 문자 그대로 처리하라고 알려주어, 출력에 올바르게 나타나게 합니다.

역슬래시 자체(`\\`)나, 작은따옴표로 감싼 문자열 안의 작은따옴표(`\'`) 같은 다른 특수 문자도 이스케이프할 수 있습니다.

작은따옴표를 사용하는 또 다른 예는 다음과 같습니다:

:::interactive_editor

```js
let quote = 'It\'s a beautiful day!';
console.log(quote); // It's a beautiful day!
```

:::

작은따옴표 앞에 `\'`를 넣어 이스케이프하면, JavaScript는 문자열을 일찍 끝내지 않고 그 문자를 문자열의 일부로 포함한다는 것을 알게 됩니다.

이스케이프와 줄바꿈 만들기는 출력 형식을 지정하거나 문자열 내 특수 문자를 처리할 때 필수적입니다. 이 기법들은 오류를 방지하고 텍스트가 의도한 대로 정확히 나타나도록 도와줍니다.
