# --description--

집합 A의 크기가 n일 때, 집합 A의 원소 합을 $S(A)$라고 나타내겠습니다. 두 개의 공집합이 아닌 서로소 부분집합 B와 C에 대해 다음 성질이 모두 참이면 이를 특별한 합 집합이라고 부르겠습니다:

1. $S(B) ≠ S(C)$; 즉, 부분집합들의 합이 같을 수 없습니다.
2. B가 C보다 원소가 더 많으면 $S(B) > S(C)$입니다.

주어진 n에 대해 $S(A)$가 최소가 되면 이를 최적 특별 합 집합이라고 부르겠습니다. 처음 다섯 개의 최적 특별 합 집합은 아래와 같습니다.

$$\begin{align}
  & n = 1: \\{1\\} \\\\
  & n = 2: \\{1, 2\\} \\\\
  & n = 3: \\{2, 3, 4\\} \\\\
  & n = 4: \\{3, 5, 6, 7\\} \\\\
  & n = 5: \\{6, 9, 11, 12, 13\\} \\\\
\end{align}$$

주어진 최적 집합 $A = \\{a_1, a_2, \ldots, a_n\\}$에 대해, 다음 최적 집합은 $B = \\{b, a_1 + b, a_2 + b, \ldots, a_n + b\\}$ 형태인 것 같습니다. 여기서 b는 이전 행의 "중간" 원소입니다.

이 "규칙"을 적용하면 $n = 6$에 대한 최적 집합은 $A = \\{11, 17, 20, 22, 23, 24\\}$이고, $S(A) = 117$일 것으로 예상됩니다. 하지만 이것은 단지 근사 최적 집합을 제공하는 알고리즘을 적용한 결과일 뿐, 최적 집합은 아닙니다. $n = 6$에 대한 최적 집합은 $A = \\{11, 18, 19, 20, 22, 25\\}$이고, $S(A) = 115$이며, 해당 집합 문자열은 `111819202225`입니다.

$A$가 $n = 7$에 대한 최적 특별 합 집합일 때, 그 집합 문자열을 구하세요.

**참고:** 이 문제는 문제 105와 문제 106과 관련이 있습니다.
