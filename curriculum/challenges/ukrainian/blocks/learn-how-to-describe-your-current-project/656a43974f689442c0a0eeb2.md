# --description--

`Currently` походить від `current`, що означає дію, яка відбувається зараз. Додавання `-ly` перетворює прикметник на прислівник. Наприклад:

`She is currently reading a book.` - Вона зараз читає книгу.

`Trends` — це популярні стилі або ідеї, про які багато людей говорять або які вони використовують. Слово `trend` в однині стає `trends` у множині. Наприклад:

- `One current trend is using eco-friendly products.` - Тут `trend` (однина) означає одну популярну ідею.

- `She writes about fashion trends.` - Тут `trends` (множина) означає багато різних популярних стилів.

`Tech trends` означають популярні тенденції в технологіях.
