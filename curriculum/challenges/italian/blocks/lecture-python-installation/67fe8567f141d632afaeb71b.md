# --description--

Per tutti i workshop e i laboratori, userai l'editor Python di freeCodeCamp. Ma è importante imparare a configurare Python sulla tua macchina locale.

Il modo più semplice per installare Python su Windows e Mac è scaricare il programma di installazione dal sito ufficiale di Python. Più avanti in questa lezione vedremo anche come eseguire Python su Linux.

Vai su `https://www.python.org/` e passa con il mouse su "Downloads". Apparirà una finestra modale che mostra la versione attuale di Python per il tuo sistema operativo (OS).

Vediamo prima come installare Python su un computer con macOS:

- Clicca sul pulsante che mostra la versione attuale di Python (dalla finestra modale precedente), e inizierai automaticamente a scaricare un file di installazione `.pkg`.
- Una volta terminato il download del programma di installazione `.pkg`, aprilo, poi clicca su "Continua" nella finestra che si apre.
- Continua a cliccare il pulsante "Continua" finché non arrivi alla sezione "Tipo di installazione". Lì, clicca sul pulsante "Installa".
- Inserisci la tua password se necessario, poi avvia l'installazione.
- Dopo di che, dovresti ricevere un messaggio di congratulazioni che dice che Python è stato installato con successo.
- Clicca sul pulsante "Chiudi" e hai finito!

Puoi verificare l'installazione aprendo il terminale ed eseguendo `python --version` o `python3 --version`.

Puoi anche aprire l'interprete Python eseguendo `python` o `python3` nel terminale. Un interprete Python è il programma che legge il codice Python, lo traduce in istruzioni che il computer capisce ed esegue quelle istruzioni.

Un <dfn>terminal</dfn> è un'interfaccia testuale che ti permette di interagire con il computer digitando comandi. Ogni sistema operativo ha un'app terminale predefinita. Su macOS puoi usare l'app Terminal. Su Windows puoi usare Prompt dei comandi o PowerShell. Su Linux ogni ambiente desktop ha la sua app terminale predefinita, come GNOME Terminal o Konsole.

Nota che, su alcuni sistemi macOS e Linux più vecchi, `python` può essere riservato a Python 2, mentre `python3` è specifico per Python 3. Se esegui `python --version` e vedi una versione di Python 2 come `Python 2.7.18`, allora è possibile che il tuo OS dipenda da qualche software scritto nella versione più vecchia di Python. In questo caso, dovresti usare `python3` per eseguire il tuo codice Python d'ora in poi. Python 2 è a fine ciclo di vita e non dovrebbe essere usato per nessun nuovo sviluppo.

Per installare Python su Windows, segui questi passaggi:

- Vai su `https://www.python.org/` e passa con il mouse su “Downloads“. Dovresti vedere una finestra modale che dice "Download per Windows" e un pulsante di download con la versione attuale di Python.
- Clicca sul numero di versione, e inizierai automaticamente a scaricare un file eseguibile per Windows (`.exe`).
- Una volta terminato il download del programma di installazione Python per Windows, fai doppio clic su di esso e segui le istruzioni.
- Quando vedi l'opzione `Add python.exe to Path`, seleziona quell'opzione, poi clicca `Install Now`. Così sarà tutto più semplice per te in seguito.

Puoi verificare l'installazione aprendo una shell a riga di comando come PowerShell ed eseguendo `python --version`. Puoi anche aprire l'interprete Python eseguendo `python`.

Per Python su Linux, la maggior parte delle distribuzioni principali come Ubuntu, Debian e Fedora include Python.
