# --description--

Quando vuoi chiedere qualcosa di specifico, puoi usare la parola interrogativa `what`. `What` si usa comunemente per parlare di cose.

Di solito metti `what` all’inizio di una domanda. Poi segui con il verbo `to be` (come `is` o `are`) o altri verbi, e poi il soggetto (come `you` o `the team`). Per esempio:

- `What is your favorite color?` – Qualcuno vuole sapere qual è il colore che ti piace di più.

- `What does this code do?` – Qualcuno vuole sapere il lavoro o il risultato del codice.
