# --description--

Ao elaborar perguntas sobre uma terceira pessoa do singular (`he`, `she` ou `it`) você pode precisar usar o verbo auxiliar `does`.

Por exemplo: se Alice está curiosa sobre os hobbies de Sophie e quer perguntar para Tom, ela pode dizer, `Does Sophie like hiking?` A palavra `does` é usada aqui porque o sujeito é `Sophie`, uma terceira pessoa do singular.

Lembre-se, `does` é usado para formar perguntas no tempo `present simple` para `he`, `she` ou `it`.
