# --description--

`function` у програмуванні — це множина інструкцій у межах програми. Вона призначена для виконання конкретного завдання. Ви визначаєте `function` один раз, вказуючи, що вона має робити, а потім можете використовувати її стільки разів, скільки потрібно, викликаючи її ім’я.

Якщо щось `missing` у коді, це означає, що необхідна частина відсутня або пропущена.

Коли код `working as expected`, він виконує саме те, що має, без помилок чи проблем.
