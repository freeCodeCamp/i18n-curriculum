# --description--

令 $S(A)$ 表示大小為 n 的集合 A 中元素的總和。若對於任意兩個非空且不相交的子集合 B 和 C，以下性質皆成立，則我們稱之為特殊總和集合：

1. $S(B) ≠ S(C)$；也就是說，子集合的總和不能相等。
2. 如果 B 包含比 C 更多的元素，則 $S(B) > S(C)$。

如果在給定 n 的情況下 $S(A)$ 被最小化，我們將稱之為最佳特殊和集合。以下是前五個最佳特殊和集合。

$$\begin{align}
  & n = 1: \\{1\\} \\\\
  & n = 2: \\{1, 2\\} \\\\
  & n = 3: \\{2, 3, 4\\} \\\\
  & n = 4: \\{3, 5, 6, 7\\} \\\\
  & n = 5: \\{6, 9, 11, 12, 13\\} \\\\
\end{align}$$

看起來對於給定的最佳集合，$A = \\{a_1, a_2, \ldots, a_n\\}$，下一個最佳集合的形式為 $B = \\{b, a_1 + b, a_2 + b, \ldots, a_n + b\\}$，其中 b 是前一列的「中間」元素。

透過套用此「rule」，我們預期當 $n = 6$ 時的最佳集合為 $A = \\{11, 17, 20, 22, 23, 24\\}$，且 $S(A) = 117$。然而，這並非最佳集合，因為我們只是套用了一個演算法來提供一個近似最佳集合。當 $n = 6$ 時的最佳集合為 $A = \\{11, 18, 19, 20, 22, 25\\}$，且 $S(A) = 115$，對應的集合字串為：`111819202225`。

假設 A 是 $n = 7$ 的最佳特殊和集合，找出它的集合字串。

**注意：** 此問題與問題 105 和問題 106 有關。
