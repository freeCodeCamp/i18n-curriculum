# --explanation--

`Code reuse` indica la pratica di usare codice esistente in nuove parti di un programma. Per esempio:

`The "Login" class was used in multiple applications to handle user access, which is a good example of code reuse.` - Invece di scrivere la logica di login per ogni app, si riusa lo stesso codice, migliorando l'efficienza.

`Extending` significa aggiungere nuove funzionalità a qualcosa che esiste già. Per esempio:

`The "AdminUser" class extends the basic "User" class by adding an "approveUser()" method.` - La `AdminUser` mantiene tutto da `User` e costruisce sopra con nuove capacità.
