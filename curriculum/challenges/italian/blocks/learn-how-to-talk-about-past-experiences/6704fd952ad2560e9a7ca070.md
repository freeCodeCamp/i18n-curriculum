# --explanation--

L'espressione `stuck on` si usa quando qualcuno non riesce ad andare avanti con un'attività o a risolvere un problema. Suggerisce di essere bloccati o incapaci di progredire. Per esempio:

- `We were stuck on this code error for hours.` - Non potevamo procedere finché non è stato trovato quell'errore nel codice.

- `She got stuck on the final question of the exam.` - Ha passato molto tempo sull'ultima domanda.

Nel dialogo, Lisa usa `stuck on` per descrivere come non riuscivano a fare progressi nel risolvere un bug nel loro lavoro.
