# --explanation--

`I wanted to` ist eine höfliche Art, einen Wunsch oder eine Absicht auszudrücken. Wenn Sie eine Anfrage beginnen, zeigt dies, dass Sie darüber nachdenken, was die andere Person tun oder antworten könnte. Zum Beispiel:

`I wanted to ask if you could help me with this task.` – In diesem Beispiel drückt die sprechende Person höflich ihren Wunsch nach Hilfe in der Vergangenheit aus, aber die Anfrage ist weiterhin gültig.
