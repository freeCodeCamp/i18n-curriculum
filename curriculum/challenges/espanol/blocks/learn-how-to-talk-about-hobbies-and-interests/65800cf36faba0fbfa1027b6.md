# --description--

Las preguntas negativas son una forma de pedir a alguien que haga algo o sugerir una acción de manera cortés o alentadora. A menudo comienzan con `why` seguido de un formulario negativo como `don't`. For example:

`Why don't we go for a walk?` - This suggests going for a walk as a good idea.
