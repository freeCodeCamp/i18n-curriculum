# --description--

Символ `equals symbol` (`=`) загально використовується для присвоєння. Його застосовують, щоб призначити значення змінній. Ім'я змінної пишеться ліворуч від знака рівності, а значення, яке ви хочете зберегти у змінній, — праворуч.

Наприклад, `name = "Sophie";` присвоює рядок `"Sophie"` змінній `name`.

Присвоїти (`assign a value`) у програмуванні означає надати змінній конкретне значення. Для цього зазвичай використовують знак рівності (`equals symbol`, `=`). Коли ви присвоюєте значення змінній, ви кажете програмі запам’ятати це значення для подальшого використання.

Наприклад, `age = 25` присвоює значення `25` змінній `age`.
