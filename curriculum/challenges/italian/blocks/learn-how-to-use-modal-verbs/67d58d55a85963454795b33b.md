# --explanation--

`Could` e `might` sono verbi modali usati per parlare di possibilità.

`Could` indica che qualcosa è possibile ma non certo. Per esempio:

`This update could make the app run faster.` - L'aggiornamento ha il potenziale per migliorare la velocità dell'app.

`Might` esprime anch'esso possibilità ma spesso suggerisce una probabilità più bassa rispetto a `could`. Per esempio:

`The new feature might improve the user experience.` - È possibile che la funzionalità migliori l'esperienza.
