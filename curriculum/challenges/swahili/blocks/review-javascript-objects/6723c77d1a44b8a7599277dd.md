## Misingi ya Vitu

- **Ufafanuzi**: Kitu ni muundo wa data unaoundwa na sifa. Sifa inajumuisha ufunguo na thamani. Ili kupata data kutoka kwenye kitu unaweza kutumia uandishi wa nukta au uandishi wa mabano.

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

console.log(person.name);  // Alice
console.log(person["name"]); // Alice
```

:::

Ili kuweka sifa ya kitu kilichopo unaweza kutumia uandishi wa nukta au uandishi wa mabano pamoja na opereta ya mgawo.

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30
};

person.job = "Engineer"
person["hobby"] = "Knitting"
console.log(person);  // {name: 'Alice', age: 30, job: 'Engineer', hobby: 'Knitting'}
```

:::

## Kuondoa Sifa Kutoka kwenye Kitu

- **Opereta ya `delete`**: Opereta hii hutumika kuondoa sifa kutoka kwenye kitu.

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  job: "Engineer"
};

delete person.job;

console.log(person.job); // undefined
```

:::

## Kukagua kama Kitu Kina Sifa Fulani

- **Njia ya `hasOwnProperty()`**: Njia hii inarudisha thamani ya kweli au siyo kweli ikionyesha kama kitu kina sifa iliyobainishwa kama sifa yake yenyewe.

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30
};

console.log(person.hasOwnProperty("name")); // true
console.log(person.hasOwnProperty("job")); // false
```

:::

- **Njia ya `Object.hasOwn()`**: Hii ni njia ya kisasa, inayopendekezwa ya kuangalia kama kitu kina sifa kama yake mwenyewe (si ya kurithiwa). Sintaksia ni `Object.hasOwn(object, propertyName)`. Inarudisha `true` ikiwa sifa ipo kwenye kitu na `false` ikiwa haipo, bila kujali thamani ya sifa hiyo. Hii inafanya iwe salama zaidi kuliko `hasOwnProperty()` na kuaminika zaidi kuliko `if (obj.prop)` wakati thamani zinaweza kuwa `0`, `false`, `null`, au `undefined`.

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30
};

console.log(Object.hasOwn(person, "name")); // true
console.log(Object.hasOwn(person, "job")); // false
```

:::

Hapa kuna mfano wa kina unaoonyesha kwa nini `Object.hasOwn()` ni ya kuaminika zaidi kuliko kukagua thamani moja kwa moja:

:::interactive_editor

```js
const settings = {
  darkMode: false,
  fontSize: 0,
  language: null
};

// Object.hasOwn() correctly sees these properties exist
console.log(Object.hasOwn(settings, "darkMode")); // true (value is false, but exists)
console.log(Object.hasOwn(settings, "fontSize")); // true (value is 0, but exists)
console.log(Object.hasOwn(settings, "theme"));    // false (property was never added)

// Using if() directly is unsafe for falsy values!
if (settings.darkMode) {
  console.log("Dark mode on"); // Does NOT print — misleading!
}

// Object.hasOwn() is the safe way
if (Object.hasOwn(settings, "darkMode")) {
  console.log("darkMode exists, value is:", settings.darkMode); // darkMode exists, value is: false
}
```

:::

## Kupata Sifa Kutoka kwenye Vitu Vilivyopangwa Ndani

- **Kupata Data**: Kupata sifa kutoka kwenye vitu vilivyopangwa ndani kunahusisha kutumia uandishi wa nukta au uandishi wa mabano, sawa na kupata sifa kutoka kwenye vitu rahisi. Hata hivyo, utahitaji kuunganisha viashiria hivi ili kufikia muundo wa ndani zaidi.

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person.contact.phone.work); // "098-765-4321"
```

:::

## Aina za Data za Kimsingi na Zisizo za Kimsingi

- **Aina za Data za Kimsingi**: Aina hizi za data ni pamoja na nambari, mfuatano wa herufi, thamani ya kweli au siyo kweli, `null`, `undefined`, na alama. Aina hizi huitwa "kimsingi" kwa sababu zinawakilisha thamani moja na siyo vitu. Thamani za kimsingi hazibadiliki, maana yake mara zinapotengenezwa, thamani yake haiwezi kubadilishwa.
- **Aina za Data Zisizo za Kimsingi**: Katika JavaScript, hizi ni vitu, ambavyo ni pamoja na vitu vya kawaida, safu za data, na vitendakazi. Tofauti na kimsingi, aina zisizo za kimsingi zinaweza kushikilia thamani nyingi kama sifa au vipengele.

## Njia za Vitu

- **Ufafanuzi**: Njia za vitu ni vitendakazi vinavyohusishwa na kitu. Hutangazwa kama sifa za kitu na vinaweza kupata na kubadilisha data ya kitu. Neno kuu `this` ndani ya njia humaanisha kitu chenyewe, hivyo kuruhusu kupata sifa zake.

:::interactive_editor

```js
const person = {
  name: "Bob",
  age: 30,
  sayHello: function() {
    return "Hello, my name is " + this.name;
  }
};

console.log(person.sayHello()); // "Hello, my name is Bob"
```

:::

## Kijenzi cha Kitu

- **Ufafanuzi**: Katika JavaScript, kijenzi ni aina maalum ya kitendakazi kinachotumika kuunda na kuanzisha vitu. Huitwa kwa kutumia neno kuu `new` na kinaweza kuanzisha sifa na njia kwenye kitu kipya kilichoundwa. Kijenzi cha `Object()` huunda kitu kipya tupu.

```js
new Object()
```

## Kufanya Kazi na Opereta ya Optional Chaining (`?.`)

- **Ufafanuzi**: Opereta hii inakuwezesha kupata sifa za kitu au kuita njia bila kuwa na wasiwasi kama zipo au la.

:::interactive_editor

```ts
const user = {
  name: "John",
  profile: {
    email: "john@example.com",
    address: {
      street: "123 Main St",
      city: "Somewhere"
    }
  }
};

console.log(user.profile?.address?.street); // "123 Main St"
console.log(user.profile?.phone?.number);   // undefined
```

:::

## Ufumbuzi wa Muundo wa Kitu

- **Ufafanuzi**: Ufumbuzi wa muundo wa kitu unakuwezesha kutoa thamani kutoka kwenye vitu na kuzipa vigezo kwa njia fupi na inayoeleweka zaidi.

:::interactive_editor

```js
const person = { name: "Alice", age: 30, city: "New York" };

const { name, age } = person;

console.log(name); // Alice
console.log(age);  // 30
```

## Kufanya Kazi na JSON

- **Ufafanuzi**: JSON inasimama kwa JavaScript Object Notation. Ni muundo wa data wa maandishi mwepesi na hutumika sana kubadilishana data kati ya seva na programu ya mtandao.

```js
{
  "name": "Alice",
  "age": 30,
  "isStudent": false,
  "list of courses": ["Mathematics", "Physics", "Computer Science"]
}
```

- **`JSON.stringify()`**: Njia hii hutumika kubadilisha kitu cha JavaScript kuwa mfuatano wa herufi wa JSON. Hii ni muhimu unapohitaji kuhifadhi au kutuma data katika muundo unaoweza kushirikiwa au kuhamishwa kwa urahisi kati ya mifumo.

:::interactive_editor

```js
const user = {
  name: "John",
  age: 30,
  isAdmin: true
};

const jsonString = JSON.stringify(user);
console.log(jsonString); // '{"name":"John","age":30,"isAdmin":true}'
```

:::

- **`JSON.parse()`**: Njia hii hubadilisha mfuatano wa herufi wa JSON kuwa kitu cha JavaScript. Hii ni muhimu unapopata data ya JSON kutoka seva ya mtandao au localStorage na unahitaji kubadilisha data hiyo kwenye programu yako.

:::interactive_editor

```js
const jsonString = '{"name":"John","age":30,"isAdmin":true}';
const userObject = JSON.parse(jsonString);

// result: { name: 'John', age: 30, isAdmin: true }
console.log(userObject);
```
