# --description--

正如你所见，flexbox 不仅仅是一个单一的 CSS 属性，而是一整套属性工具箱，你可以用它们将内容放置到你需要的位置。其中一些属性属于 flex 容器，而另一些则属于 flex 项。这是一个简单但重要的概念。

弹性容器是任何具有 `display: flex` 的元素。弹性项是直接位于弹性容器内的任何元素。

# --questions--

## --text--

flex 容器和 flex 项之间有什么区别？

## --answers--

Flex 容器是具有 `display: flex` 的元素，而 flex 项是位于 flex 容器内的元素。

---

Flex 容器是位于 flex 项内的元素，而 flex 项定义整体版型。

---

Flex 容器是使用 flexbox 的元素，flex 项是内联元素。

---

Flex 容器是定义版型结构的元素，而 flex 项控制整体页面设计。
