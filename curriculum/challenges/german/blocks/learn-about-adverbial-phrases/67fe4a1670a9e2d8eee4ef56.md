# --explanation--

`First off` wird verwendet, um das erste Thema einzuführen, das besprochen oder erledigt wird. Es signalisiert den Beginn einer Liste oder Reihenfolge von Punkten, auf die man sich konzentriert. Zum Beispiel:

- `First off, we need to address the project's scope.` – Das bedeutet, dass die Behandlung des Gültigkeitsbereichs des Projekts als erstes erfolgt.

- `First off, let's review the current goals before we start planning.` – Das bedeutet, dass die Überprüfung der aktuellen Ziele als erstes geschieht, bevor mit der Planung begonnen wird.
