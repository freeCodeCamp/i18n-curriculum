# --interactive--

Katika JavaScript, kijenzi ni aina maalum ya kitendakazi kinachotumika kuunda na kuanzisha vitu. Kinaitwa kwa kutumia neno kuu la `new` na kinaweza kuanzisha vigezo na njia kwenye kitu kipya kilichoundwa.

Katika funzo hili, tutaangalia jinsi ya kufanya kazi na kijenzi cha `Object()`. Kijenzi cha `Object()` huunda kitu kipya tupu. Hapa kuna mfano:

```js
new Object()
```

Unapofanya wito wa `new Object()`, hurejesha kitu kipya ambacho kinaweza kutumika kuhifadhi thamani.

Kijenzi cha `Object()` kinaweza kutumika kwa au bila neno kuu la `new`. Kinapoitwa kama kitendakazi bila `new`, hufanya tofauti kulingana na aina ya thamani inayopitishwa. Hapa kuna mfano wa kutumia kijenzi cha `Object()` bila neno kuu la `new`:

:::interactive_editor

```js
const num = 42;
const numObj = Object(num); // Creates an object wrapper for the number

console.log(numObj);
console.log(typeof numObj); // "object"
```

:::

Kama unavyoona katika `console.log` ya pili, `numObj` ni kitu. Hii inatokea kwa sababu tumetumia kijenzi cha `Object()` kubadilisha ingizo hilo la nambari kuwa kitu.

Nini kinatokea ikiwa tutajaribu kupitisha `null` au `undefined` kwa kijenzi cha `Object()`?

:::interactive_editor

```js
const newObj = new Object(undefined);
console.log(newObj); // {}
```

:::

Hasa, matokeo yatakuwa kitu tupu. Matumizi mengine ya kijenzi cha `Object()` ni wakati unafanya kazi na thamani ya aina isiyojulikana na unahitaji kuhakikisha ni kitu. Tuchunguze mfano ufuatao:

:::interactive_editor

```js
function toObject(value) {
  if (value === null || value === undefined) {
    return {};
  }

  if (typeof value === "object") {
    return value;
  }

  return Object(value);
}

console.log(toObject(null));

console.log(toObject(true));

console.log(toObject([1, 2, 3]));
```

:::

Katika mfano huu, tuna kitendakazi kinachoitwa `toObject`. Hali ya pili itakagua ikiwa thamani ni aina ya kitu na itarejesha thamani ikiwa hali ni `true`. Hali hii itakagua vitu pamoja na safu za data kwa kuwa safu ni aina maalum za vitu.

Ikiwa hakuna hali yoyote kati ya hizo ni kweli, kitendakazi kitarudisha `Object(value)`, ambayo hubadilisha ingizo kuwa kitu. Hii hufanya kazi kwa thamani kama nambari, mfuatano wa herufi, na thamani za kweli au siyo kweli.

Mara nyingi hutatumia kijenzi cha `Object()` kuunda vitu vipya kwa sababu utatumia sintaksia ya vitu vya methali badala yake (mfano, `const objectLiteral = { name: "Beau" }`). Lakini bado ni vizuri kuelewa misingi ya kufanya kazi na kijenzi cha Object.
