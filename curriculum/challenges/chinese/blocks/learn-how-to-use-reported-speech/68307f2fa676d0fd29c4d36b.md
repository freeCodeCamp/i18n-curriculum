# --explanation--

`Was there` 是 `There was` 的疑问（提问）形式，用于询问某事是否在过去存在或发生。它通过颠倒主语和动词来构成疑问句。例如：

- `There was a problem with the layout.`

- `Was there a problem with the layout?` 有问题吗？

这有助于询问过去某事的存在或发生。
