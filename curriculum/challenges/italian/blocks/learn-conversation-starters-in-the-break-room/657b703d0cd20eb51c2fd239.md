# --description--

Esercitiamoci a creare frasi negative con un verbo ausiliare.

# --questions--

## --text--

Scegli la frase che usa correttamente `don't` per esprimere ciò che qualcuno non fa.

## --answers--

`I don't coding on weekends.`

### --feedback--

Quando si usa con `don't`, `coding` dovrebbe essere nella base `code`.

---

`I don't code on weekends.`

---

`I don't codes on weekends.`

### --feedback--

Quando si usa con `don't`, il verbo non dovrebbe avere un `-s` alla fine.

---

`I does not code on weekends.`

### --feedback--

`Does not` si usa con soggetti singolari di terza persona (`he`, `she`, `it`); con `I` dovrebbe essere `do not`.
