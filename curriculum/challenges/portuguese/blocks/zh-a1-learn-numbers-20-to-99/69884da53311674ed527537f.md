# --description--

Vamos praticar escrever alguns números entre 70 e 79. Para completar esta tarefa, lembre-se de que em chinês, o número 70 é chamado `七十 (qī shí)`.

Os números de 0 a 9 são:

| Número | Chinês |
| :--- | :--- |
| 0 | `零 (líng)` |
| 1 | `一 (yī)` |
| 2 | `二 (èr)` |
| 3 | `三 (sān)` |
| 4 | `四 (sì)` |
| 5 | `五 (wǔ)` |
| 6 | `六 (liù)` |
| 7 | `七 (qī)` |
| 8 | `八 (bā)` |
| 9 | `九 (jiǔ)` |

# --instructions--

Esta tarefa não tem áudio. Complete os espaços em branco abaixo com os números correspondentes em chinês.

`72 - BLANK`

# --explanation--

Estes números são:

 `七十二 (qī shí èr)` - 72
 
 `七十七 (qī shí qī)` - 77
 
 `七十九 (qī shí jiǔ)` - 79
 
 Eles são formados adicionando o chinês para 2, 7 e 9 após `七十 (qī shí)` (70).
