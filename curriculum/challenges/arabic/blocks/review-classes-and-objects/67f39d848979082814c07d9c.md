## أصناف وكائنات بايثون

- **تعريف الصنف**: الصنف هو نموذج لإنشاء الكائنات. يحدد السلوك الذي سيملكه الكائن من خلال خصائصه وطرقه. إليك مثالًا أساسيًا لتعريف صنف في بايثون:

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f'{self.name.upper()} says woof woof!')
```

- **إنشاء الكائنات**: الكائنات هي نسخ من الصنف. تُنشأ باستدعاء الصنف مع المعلمات اللازمة.

```python
dog1 = Dog('Jack', 3)
dog2 = Dog('Thatcher', 5)

dog1.bark()  # JACK says woof woof!
dog2.bark()  # THATCHER says woof woof!
```

- **استدعاء الطرق على الكائنات**: يمكنك استدعاء الطرق على الكائنات لأداء إجراءات أو استرجاع معلومات.

```python
object_name1.method_name()
object_name2.method_name()
```

- **الفرق بين الصنف والكائن**: الصنف هو قالب قابل لإعادة الاستخدام، بينما الكائن هو نسخة محددة من ذلك الصنف تحتوي على بيانات فعلية.

## الخصائص

- **خصائص النسخة**: تُعرّف في `__init__()` باستخدام `self`، وتكون فريدة لكل كائن.
- **خصائص الصنف**: تُعرّف مباشرة داخل الصنف وتُشترك بين كل النسخ.

```python
class Dog:
    species = 'French Bulldog'  # Class attribute

    def __init__(self, name):
        self.name = name  # Instance attribute

print(Dog.species) # French Bulldog

jack = Dog('Jack')
print(jack.name)     # Jack
print(jack.species)  # French Bulldog
```

## الطرق

- **الطرق**: دوال تُعرّف داخل الصنف وتعمل على خصائص الكائن.

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
print(my_car_1.describe())  # This car is a red Tesla Model S
```

- **الوصول إلى الطرق**: استدعِ الطرق على الكائنات باستخدام ترميز النقطة. إليك مثالًا لاستدعاء الطريقة `describe` على كائنين مختلفين من نوع سيارة:

```python
class Car:
    def __init__(self, color, model):
        self.color = color  
        self.model = model  

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
my_car_2 = Car('green', 'Lamborghini Revuelto')

print(my_car_1.describe()) # Calling method using the dot notation

print(my_car_2.describe()) # Calling method using the dot notation
```

## الطرق الخاصة (Dunder) أو السحرية

- **التعريف**: طرق خاصة تبدأ وتنتهي بشرطتين سفليتين (مثل `__init__`, `__len__`, `__str__`, `__eq__`). يستخدمها بايثون داخليًا للعمليات المدمجة.

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **استدعاء الطرق الخاصة بشكل غير مباشر**: لا تحتاج لاستدعاء الطرق الخاصة مباشرة. بدلاً من ذلك، يستدعيها بايثون تلقائيًا عند حدوث إجراءات معينة. تشمل هذه العمليات:

- **العمليات الحسابية مثل الجمع والطرح والضرب والقسمة وغيرها**. تُستدعى `__add__()` للجمع، و`__sub__()` للطرح، و`__mul__()` للضرب، و`__truediv__()` للقسمة.
        
- **عمليات السلاسل النصية مثل الربط والتكرار والتنسيق والتحويل إلى نص**. تُستدعى `__add__()` للربط، و`__mul__()` للتكرار، و`__format__()` للتنسيق، و`__str__()` و`__repr__()` للتحويل إلى نص، وهكذا.
        
- **عمليات المقارنة مثل التساوي، والأقل من، والأكبر من، وغيرها**. تُستدعى `__eq__()` لفحص التساوي، و`__lt__()` للأقل من، و`__gt__()` للأكبر من، وهكذا.
        
- **عمليات التكرار مثل جعل الكائن قابلًا للتكرار والتنقل بين العناصر**. تُستدعى `__iter__()` لإرجاع مكرر و`__next__()` لجلب العنصر التالي.

## مثال من العالم الحقيقي: عربة التسوق

- **صنف العربة مع الطرق الخاصة**: يسمح بإضافة وإزالة العناصر، والتكرار، والتحقق من المحتويات بسلوك مدمج.

```python
class Cart:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

    def remove(self, item):
        if item in self.items:
            self.items.remove(item)
        else:
            print(f'{item} is not in cart')

    def list_items(self):
        return self.items

    def __len__(self):
        return len(self.items)

    def __getitem__(self, index):
        return self.items[index]

    def __contains__(self, item):
        return item in self.items

    def __iter__(self):
        return iter(self.items)

cart = Cart()
cart.add('Laptop')
print(len(cart))        # 1
print('Laptop' in cart) # True
```
