# --interactive--

JavaScript 中的数组是动态的，这意味着你可以轻松添加或删除其中的元素。 有四个主要方法可以用于从数组的开头和结尾添加或删除元素：`push()`、`pop()`、`shift()` 和 `unshift()`。 让我们分别详细了解一下这些方法。

`push()` 方法用于在数组末尾添加一个或多个元素。 `push()` 方法的返回值是数组的新长度。 下面是一个在现有 `fruits` 数组中添加新水果的示例：

:::interactive_editor

```js
const fruits = ["apple", "banana"];
const newLength = fruits.push("orange");
console.log(newLength); // 3
console.log(fruits); // ["apple", "banana", "orange"]
```

:::

在这个示例中，我们从一个名为 `fruits` 的数组开始，该数组包含两个元素。 然后我们使用 `push()` 方法将字符串 `orange` 添加到数组的末尾。

你可能注意到我们使用 `const` 来声明 `fruits` 数组。 但为什么当 `fruits` 是一个常量时仍然可以向该 `fruits` 数组中添加新元素呢？ 这是因为使用 `const` 关键字声明数组会创建对该数组的引用。 虽然数组本身是可变的，但你不能把 `fruits` 常量重新赋值为其他值，例如：

```js
const fruits = ["apple", "banana"];
fruits = ["This", "will", "not", "work"];
console.log(fruits); // Uncaught TypeError: Assignment to constant variable. 
```

下一个方法是 `pop()` 方法。 `pop()` 方法从数组中删除最后一个元素并返回该元素。 它还会修改原始数组。 具体操作如下：

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let lastFruit = fruits.pop();
console.log(fruits); // ["apple", "banana"]
console.log(lastFruit); // "orange"
```

:::

在这个示例中，我们从一个包含三个水果的数组开始。 `pop()` 方法删除了最后一个元素（`orange`）并返回它。 原 `fruits` 数组被修改，只剩下两个元素。

`unshift()` 方法将一个或多个元素添加到数组的开头，并返回其新长度。 其工作原理与 `push()` 类似，但修改的是数组的起始位置而不是结束位置。 这是一个示例：

:::interactive_editor

```js
let numbers = [2, 3];
let newLength = numbers.unshift(1);
console.log(numbers); // [1, 2, 3]
console.log(newLength); // 3
```

:::

在这个示例中，我们使用 `unshift()` 将数字 `1` 添加到 `numbers` 数组的开头。 该方法返回数组的新长度 `3`。

最后，`shift()` 方法从数组中移除第一个元素并返回该元素。 它与 `pop()` 类似，但它作用于数组的开头而不是结尾。 具体操作如下：

:::interactive_editor

```js
let colors = ["red", "green", "blue"];
let firstColor = colors.shift();
console.log(colors); // ["green", "blue"]
console.log(firstColor); // "red"
```

:::

在这个示例中，我们从一个包含三种颜色的数组开始。 `shift()` 方法删除了数组的第一个元素（`red`）并返回它。 原 `colors` 数组被修改，只剩下两个元素。

需要注意的是，`push()` 和 `unshift()` 可以一次添加多个元素，而 `pop()` 和 `shift()` 每次只能删除一个元素。
