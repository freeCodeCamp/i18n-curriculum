# --explanation--

`Alright` é uma forma casual de começar a falar e chamar a atenção de alguém. Por exemplo:

`Alright, let's get started.` - Isso mostra que você está pronto para começar.

`Here's` é a contração de `here is`. Você pode usá-lo para introduzir ou apresentar algo. Por exemplo:

`Here's the report you asked for.` - Você está mostrando o relatório para alguém.

`Pitch` geralmente significa uma apresentação curta para promover ou explicar algo. Por exemplo:

`She gave a pitch to investors.` - Isso significa que ela fez uma apresentação de negócios.
