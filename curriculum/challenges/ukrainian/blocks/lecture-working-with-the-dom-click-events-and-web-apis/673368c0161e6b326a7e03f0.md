# --interactive--

Давайте подивимося, як ви можете створювати вузли за допомогою `innerHTML` і `createElement()`.

Почнемо з `innerHTML`.

`innerHTML` — це властивість об’єктів `Element`, яку можна використовувати для встановлення їхньої HTML-розмітки. За допомогою `innerHTML` ви можете встановити HTML-структуру існуючого елемента рядком, створюючи всі необхідні вузли.

Ось приклад. У нас є порожній `div`, і ми додамо всередину нього кілька елементів, використовуючи властивість `innerHTML`:

```html
<div id="container">
  <!-- Add new elements here -->
</div>
```

Спочатку потрібно вибрати елемент за його ідентифікатором:

```js
const container = document.getElementById("container");
```

Потім ми встановлюємо властивість `innerHTML` контейнера у рядок. Цей рядок має містити всю розмітку, необхідну для представлення вузлів і структури, яку ви хочете створити. Можна уявити це як написання HTML всередині рядка.

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
container.innerHTML = "<ul><li>Cheese</li><li>Tomato</li></ul>";
```

:::

Ми встановимо `innerHTML` цього елемента у невпорядкований список інгредієнтів для піци (сир і помідор).

Ви також можете писати рядок на кількох лініях, якщо це робить розмітку легшою для розуміння. Головне, щоб розмітка точно відображала HTML-структуру, яку ви хочете створити.

Після запуску цього коду ви побачите таку HTML-структуру, якщо перевірите розмітку.

```html
<div id="container">
  <ul>
    <li>Cheese</li>
    <li>Tomato</li>
  </ul>
</div>
```

Нові вузли були створені та динамічно додані до DOM після того, як рядок було проаналізовано.

`innerHTML` може бути дуже корисним у певних сценаріях. Однак він має деякі ризики безпеки, з якими варто ознайомитися. Не слід його використовувати, якщо ви не маєте повного контролю над рядком.

Наприклад, якщо рядок буде введений користувачем, не слід використовувати `innerHTML`, бо користувач може вставити шкідливий вміст на ваш сайт. Через це зазвичай рекомендують використовувати `textContent` замість нього, щоб вставляти простий текст.

Інший спосіб створити новий вузол — це використати метод `createElement()`. За допомогою цього методу ви можете створити новий елемент, вказавши його тег.

Наприклад, якщо ви хочете створити елемент зображення, ви передаєте тег `img` як рядок при виклику цього методу:

```js
document.createElement("img");
```

І ви можете призначити цей новий об’єкт змінній:

```js
const img = document.createElement("img");
```

Метод `createElement()` повертає новий об’єкт `HTMLElement`, якщо документ є `HTMLDocument`. Інакше він повертає об’єкт `Element`.

Коли цей новий елемент готовий, ви можете додати його до DOM як дочірній елемент іншого існуючого елемента за допомогою методу `appendChild()`, або вставити його у певне місце за допомогою інших методів. Оберіть той, що найкраще підходить для ваших потреб.

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
const img = document.createElement("img");
img.src = "https://cdn.freecodecamp.org/curriculum/cat-photo-app/lasagna.jpg";
img.alt = "A slice of lasagna on a plate.";
container.appendChild(img);
```

:::

Створення вузлів за допомогою `innerHTML` і `createElement()` дозволяє динамічно маніпулювати структурою та вмістом ваших вебсайтів. Поєднуючи ці техніки, ви можете будувати інтерактивні вебзастосунки.
