# --description--

الفعل `to contain` يعني الاحتواء أو وجود شيء ما بداخل. في سياق التوثيق أو ملف README، يشير `to contain` إلى المعلومات التي يتضمنها الملف.

على سبيل المثال، يمكن للزجاجة أن `contain` الماء، تمامًا كما أن ملف README `contains` معلومات عن مشروع.
