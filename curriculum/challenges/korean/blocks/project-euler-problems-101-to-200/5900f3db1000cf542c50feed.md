# --description--

다음 방정식에서 x, y, n은 양의 정수입니다.

$$\frac{1}{x} + \frac{1}{y} = \frac{1}{n}$$

서로 다른 해의 개수가 400만 개를 넘는 가장 작은 `n` 값은 무엇인가요?
