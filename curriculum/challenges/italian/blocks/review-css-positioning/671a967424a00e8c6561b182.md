# --interactive--

## Lavorare con i float

- **Definizione**: I float si usano per rimuovere un elemento dal suo flusso normale nella pagina e posizionarlo sul lato sinistro o destro del suo contenitore. Quando succede questo, il testo si avvolge attorno a quel contenuto flottante.

```css
float: left;
float: right;
```

- **Cancellare i float**: La proprietà `clear` serve a stabilire se un elemento deve essere spostato sotto il contenuto flottante. Quando hai più elementi flottanti affiancati, possono sorgere problemi di sovrapposizione e collasso nei layout. Perciò è stato creato un trucco `clearfix` per risolvere questo problema.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="clearfix">
  <div class="box left">Left</div>
  <div class="box right">Right</div>
</div>
```

```css
.box {
  width: 100px;
  height: 60px;
  color: white;
  text-align: center;
  line-height: 60px;
}

.left {
  float: left;
  background: teal;
}

.right {
  float: right;
  background: purple;
}

.clearfix::after {
  content: "";
  display: block;
  clear: both;
}
```

:::

## Posizionamento statico, relativo e assoluto

- **Posizionamento statico**: Questo è il flusso normale del documento. Gli elementi sono posizionati dall’alto verso il basso e da sinistra a destra uno dopo l’altro.
- **Posizionamento relativo**: Ti permette di usare le proprietà `top`, `left`, `right` e `bottom` per posizionare l’elemento all’interno del flusso normale del documento. Puoi anche usare il posizionamento relativo per far sovrapporre elementi con altri elementi nella pagina.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="relative">Relative Element</div>
```

```css
.relative {
    position: relative;
    top: 30px;
    left: 30px;
}
```

:::

- **Posizionamento assoluto**: Ti permette di togliere un elemento dal flusso normale del documento, facendolo comportare in modo indipendente dagli altri elementi.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  Parent
  <div class="positioned">Absolute</div>
</div>
```

```css
.parent {
  position: relative;
  height: 120px;
  background: #eee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

## Posizionamento fisso e sticky

- **Posizionamento fisso**: Quando un elemento è posizionato con `position: fixed`, viene rimosso dal flusso normale del documento e posizionato rispetto alla viewport, cioè resta nella stessa posizione anche quando l’utente scorre. Questo si usa spesso per elementi come intestazioni o barre di navigazione che devono rimanere sempre visibili.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="navbar">Fixed Navbar</div>
<div style="height: 600px;"></div>
```

```css
.navbar {
  position: fixed; 
  top: 0; 
  width: 100%;
  background: #0077ff;
  color: white;
  padding: 8px; 
}
```

:::

- **Posizionamento sticky**: Questo tipo di posizionamento è un ibrido tra relativo e fisso. Inizialmente, l’elemento si comporta come se fosse posizionato relativamente, restando nel flusso del documento. Però, una volta che l’utente scorre e l’elemento supera un certo punto, si “attacca” alla viewport (di solito in alto) e si comporta come se fosse fisso. È ottimo per creare elementi come barre di navigazione sticky, che diventano fisse solo quando l’utente scorre fino a una certa posizione.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Scroll down</p>
<div class="positioned">Sticky Element</div>
<div style="height: 500px;"></div>
```

```css
.positioned {
  position: sticky;
  top: 30px;
  left: 30px;
  background: #4caf50;
  color: white;
  padding: 6px;
}
```

:::

## Lavorare con la proprietà `z-index`

- **Definizione**: La proprietà `z-index` in CSS serve a controllare l’ordine di sovrapposizione verticale degli elementi posizionati che si sovrappongono nella pagina.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box box1">1</div>
  <div class="box box2">2</div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

.box {
  position: absolute;
  width: 100px;
  height: 100px;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

.box1 {
  background: lightcoral;
  top: 20px;
  left: 20px;
  z-index: 1;
}

.box2 {
  background: steelblue;
  top: 40px;
  left: 40px;
  z-index: 2;
}
```
