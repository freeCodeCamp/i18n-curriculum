# --explanation--

Джейк просить Джесіку повідомити його одразу, якщо вона побачить щось підозріле.

Він використовує речення `First Conditional`, яке виражає реальну можливість у майбутньому. Воно має структуру `If` + `Present Simple`, `will` + інфінітив (або наказовий спосіб). У цьому діалозі:

- `If you find anything unusual` — це умова.

- `Let me know immediately` — це результат.
