# --description--

Нехай $S$ — (нескінченний) рядок, який утворюється шляхом послідовного запису натуральних чисел (починаючи з 1) у десятковій системі.

Отже, $S = 1234567891011121314151617181920212223242\ldots$

Легко помітити, що будь-яке число з’явиться у $S$ нескінченну кількість разів.

Нехай $f(n)$ — початкова позиція $n^{\text{го}}$ входження числа $n$ у $S$. Наприклад, $f(1) = 1$, $f(5) = 81$, $f(12) = 271$ і $f(7780) = 111\\,111\\,365$.

Знайдіть $\sum f(3^k)$ для $1 \leq k \leq 13$.
