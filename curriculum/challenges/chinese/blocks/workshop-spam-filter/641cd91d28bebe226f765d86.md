# --description--

将事件监听器附加到你的 `checkMessageButton`，监听 `click` 事件。

如果 `messageInput` 为空，则向用户显示带有消息 `"Please enter a message."` 的警告。

然后，退出函数执行。


# --hints--

当你单击 `checkMessageButton` 并且 `messageInput` 为空时，你应该使用消息 `"Please enter a message."` 提醒用户。

```js
const testArr = [];
const temp = alert;
alert = (msg) => {testArr.push(msg)};
try {
  messageInput.value = "";
  checkMessageButton.dispatchEvent(new Event("click"));
  assert.lengthOf(testArr, 1);
  assert.strictEqual(testArr[0], "Please enter a message.")
} finally {
  alert = temp;
}
```

当你单击 `checkMessageButton` 时，仅当 `messageInput` 为空时才应提醒用户。

```js
const testArr = [];
const temp = alert;
alert = (msg) => {testArr.push(msg)};
try {
  messageInput.value = "some message";
  checkMessageButton.dispatchEvent(new Event("click"));
  assert.isEmpty(testArr);
} finally {
  alert = temp;
}
```
