# --description--

لعدد أولي $p$، لنفرض $S(p) = (\sum (p - k)!)\bmod (p)$ حيث $1 ≤ k ≤ 5$.

على سبيل المثال، إذا كان $p = 7$،

$$(7 - 1)! + (7 - 2)! + (7 - 3)! + (7 - 4)! + (7 - 5)! = 6! + 5! + 4! + 3! + 2! = 720 + 120 + 24 + 6 + 2 = 872$$

بما أن $872\bmod (7) = 4$، إذن $S(7) = 4$.

يمكن التحقق من أن $\sum S(p) = 480$ عندما يكون $5 ≤ p &lt; 100$.

احسب $\sum S(p)$ عندما يكون $5 ≤ p &lt; {10}^8$.
