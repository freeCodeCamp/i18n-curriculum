# --description--

前のチャレンジでは、指定された幅と高さを持つ`svg`要素を作成しました。これは`background-color`タグ内で`style`が適用されていたため表示されていました。このコードは指定された幅と高さのスペースを確保していました。

次のステップは、`svg`エリアに配置する図形を作成することです。SVGには長方形や円など、いくつかのサポートされている図形があります。これらはデータを表示するために使われます。例えば、長方形（`<rect>`）のSVG図形は棒グラフのバーを作成するのに使えます。

図形を`svg`エリアに配置するときは、`x`と`y`の座標で位置を指定できます。原点の(0, 0)は左上隅にあります。`x`の正の値は図形を右に押し、`y`の正の値は原点から下に押します。

前のチャレンジの500（幅）×100（高さ）の`svg`の中央に図形を配置するには、`x`座標は250、`y`座標は50になります。

SVGの`rect`には4つの属性があります。`x`エリア内の配置場所を示す`y`と`svg`の座標です。また、サイズを指定する`height`と`width`もあります。

# --instructions--

`rect`に`svg`図形を`append()`を使って追加し、`width`属性に`25`、`height`属性に`100`を設定してください。また、`rect`には`x`と`y`属性をそれぞれ`0`に設定してください。

