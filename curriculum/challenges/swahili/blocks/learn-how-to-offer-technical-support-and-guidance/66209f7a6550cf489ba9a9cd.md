# --description--

`To handle` kitu fulani maana yake ni kukishughulikia au kukisimamia. Katika programu, `handling` inahusisha kusimamia data au shughuli. Kwa mfano, `to handle user input means to manage what happens when a user enters data`.

`Error handling` ni kushughulikia makosa yanayotokea wakati programu inapoendesha. Inahusiana na kuandika msimbo unaoweza kusimamia makosa haya bila kuanguka. Kwa mfano, `using try-except blocks in Python to handle errors that might occur when a program runs`.
