# --explanation--

O tempo `Present Simple` é frequentemente usado para descrever estados, fatos ou condições que são geralmente verdadeiros.Este tempo verbal é usado porque a corrupção do BIOS é considerada uma condição estável ou imutável quando o computador tenta inicializar.

`Is corrupted` significa danificado ou alterado de uma forma que faz com que pare de funcionar corretamente.Por exemplo:

`The file is corrupted and won't open.` - O arquivo está danificado e não pode ser acessado.
