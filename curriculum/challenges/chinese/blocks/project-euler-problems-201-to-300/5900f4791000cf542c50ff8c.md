# --description--

多项式的 $P(x)$ 的根或零点是方程 $P(x) = 0$ 的方案。

定义 $P_n$ 为系数是 $n$ 的数字的多项式。

例如，$P_{5703}(x) = 5x^3 + 7x^2 + 3$。

我们可以看到：

- $P_n(0)$ 是 $n$ 的最后一位数字，
- $P_n(1)$ 是 $n$ 的各位数字之和，
- $Pn(10)$ 是 $n$ 本身。

定义 $Z(k)$ 为不超过 $k$ 的正整数 $n$ 的数量，其中多项式 $P_n$ 至少有一个整数根。

可以验证 $Z(100\,000)$ 是 14696。

什么是 $Z({10}^{16})$？
