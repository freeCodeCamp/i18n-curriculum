# --interactive--

JavaScript에서 객체는 관련된 데이터와 기능을 저장하고 조직할 수 있는 기본적인 자료 구조입니다.

객체를 여러 폴더와 문서를 담는 서류 캐비닛처럼 다양한 정보를 담는 컨테이너로 생각할 수 있습니다.

이 정보들은 속성이라고 하며, 이름(또는 키)과 값으로 구성됩니다.

```js
const exampleObject = {
  propertyName: value,
}
```

객체는 매우 다재다능하며 JavaScript의 근간을 이룹니다. 사실 JavaScript에서 거의 모든 것은 객체이거나 객체로 취급할 수 있습니다. 여기에는 배열, 함수, 그리고 특정 방식으로 사용될 때 문자열이나 숫자 같은 원시 데이터 타입도 포함됩니다.

이러한 객체 중심의 특성 덕분에 JavaScript는 매우 유연하고 강력한 언어가 됩니다. 이제 객체를 어떻게 생성하는지 살펴보겠습니다:

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};
```

이 예제에서는 `person`라는 객체를 만들었고, 세 개의 속성 `name`, `age`, `city`를 포함합니다. 각 속성은 이름과 값이 콜론으로 구분되어 있습니다.

이제 이 속성들에 어떻게 접근할 수 있는지 알아보겠습니다. JavaScript에서 객체 속성에 접근하는 주요 방법은 점 표기법과 대괄호 표기법 두 가지입니다.

점 표기법은 가장 일반적이고 직관적인 객체 속성 접근 방법입니다. 점 표기법의 기본 구문은 다음과 같습니다:

```js
objectName.propertyName
```

우리 `person` 객체에 점 표기법을 사용하는 방법은 다음과 같습니다:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

console.log(person.name);  // Alice
console.log(person.age);   // 30
```

:::

점 표기법은 간결하고 읽기 쉬워서, 접근하려는 속성 이름을 정확히 알고 있고 그 이름이 유효한 JavaScript 식별자(숫자로 시작하지 않고 특수 문자나 공백이 없는 경우)일 때 선호됩니다.

반면 대괄호 표기법은 대괄호 안에 문자열을 넣어 객체 속성에 접근할 수 있게 해줍니다. 대괄호 표기법을 사용하는 방법은 다음과 같습니다:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

console.log(person["name"]); // Alice
console.log(person["age"]); //  30
```

:::

대괄호 표기법은 점 표기법보다 더 유연해서, 유효한 JavaScript 식별자가 아닌 속성 이름도 사용할 수 있습니다. 예를 들어, 속성 이름에 공백이 있거나 숫자로 시작한다면 대괄호 표기법을 사용해야 합니다:

:::interactive_editor

```js
const oddObject = {
  "1stProperty": "Hello",
  "property with spaces": "World"
};

console.log(oddObject["1stProperty"]);  // Hello
console.log(oddObject["property with spaces"]);  // World
```

:::

또한 대괄호 표기법은 변수를 사용해 동적으로 속성에 접근할 수 있다는 장점도 있습니다:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "Wonderland"
};

let propertyName = "city";
console.log(person[propertyName]); // Wonderland
```

:::

이러한 유연성 덕분에 대괄호 표기법은 코드 작성 시점에 정확한 속성 이름을 모를 때나 사용자 입력 또는 다른 동적 소스에서 속성 이름이 올 때 특히 유용합니다.

JavaScript의 객체는 매우 강력하고 다재다능하다는 점도 기억하세요. 객체는 문자열이나 숫자 같은 단순 값뿐 아니라 배열이나 다른 객체도 포함할 수 있습니다.

객체와 객체를 다루는 방법을 이해하는 것은 JavaScript에서 매우 중요합니다. 객체는 언어 전반과 많은 JavaScript 라이브러리 및 프레임워크에서 광범위하게 사용되기 때문입니다.

JavaScript를 계속 배우고 사용하면서 객체를 마스터하는 것이 복잡하고 강력한 애플리케이션을 만드는 데 큰 가능성을 열어준다는 것을 알게 될 것입니다.
