# --description--

양의 정수 $n$에 대해, $σ(n)$을 $n$의 모든 약수의 합으로 정의합니다. 예를 들어 $σ(6) = 1 + 2 + 3 + 6 = 12$입니다.

완전수는 아시다시피 $σ(n) = 2n$인 수입니다.

양의 정수의 완전수 몫을 $p(n) = \frac{σ(n)}{n}$으로 정의합니다.

$p(n)$이 정수 $k$에 대해 $k + \frac{1}{2}$ 형태인 모든 양의 정수 $n ≤ {10}^{18}$의 합을 구하세요.
