# --explanation--

Фраза `in real time` означає, що щось відбувається негайно, у міру того, як події трапляються, без затримки. Її часто використовують у контексті моніторингу, комунікації або обробки даних. Наприклад:

`The system updates the dashboard in real time, so you can see changes instantly.` — означає, що панель керування змінюється одразу після застосування оновлень.

У реченні Джейка він має на увазі, що хоча він і перевіряв наявність вразливостей безпеки, це не завжди робилося негайно, у міру появи оновлень.
