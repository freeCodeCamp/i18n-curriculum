# --description--

البرمجيات الوسيطة مفهوم أساسي في Express، وتسمح لنا بالتعامل مع الطلبات والاستجابات بين العميل والخادم. دوال البرمجيات الوسيطة تصل إلى كائن الطلب، وكائن الاستجابة، والدالة التالية في دورة الطلب والاستجابة للتطبيق.

لنشرح الأمر:

## ما هي البرمجيات الوسيطة؟

البرمجيات الوسيطة هي دالة تُنفذ خلال دورة حياة الطلب إلى الخادم. تقوم بمهام مختلفة مثل تسجيل الطلبات، معالجة الأخطاء، تحليل الطلبات الواردة، أو تقديم الملفات الثابتة.

يوفر Express طريقة سهلة لتعريف البرمجيات الوسيطة التي تعالج الطلبات الواردة قبل أن تصل إلى معالجي المسارات.

كيف تعمل البرمجيات الوسيطة:

عندما يُرسل طلب إلى تطبيق Express، تُنفذ البرمجيات الوسيطة بالترتيب الذي أُضيفت به. التسلسل يكون كالتالي:

* يُرسل الطلب إلى الخادم.
* تعالج دوال البرمجيات الوسيطة الطلب.
* إذا استدعت البرمجيات الوسيطة `next()`، تنتقل السيطرة إلى البرمجيات الوسيطة التالية أو معالج المسار.
* إذا لم تستدع البرمجيات الوسيطة `next()`، تتوقف دورة الطلب والاستجابة.

```js
const express = require('express')
const app = express()

// Simple middleware that logs request info
app.use((req, res, next) => {
  console.log('Request URL:', req.url)
  console.log('Request Method:', req.method)
  next() // Passes control to the next middleware
})

app.get('/', (req, res) => {
  res.send('Hello World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

`app.use()`: تضيف البرمجيات الوسيطة على مستوى التطبيق بأكمله.

في هذا المثال، تسجل البرمجيات الوسيطة تفاصيل الطلب وتستدعي `next()` للانتقال إلى البرمجيات الوسيطة التالية أو معالج المسار.

البرمجيات الوسيطة ميزة قوية في Express تتيح لك اعتراض الطلبات وتعديلها في نقاط مختلفة. يمكنك استخدامها لمعالجة أشياء مثل التسجيل، التوثيق، أو حتى تعديل كائنات الطلب والاستجابة قبل أن تصل إلى معالجي المسارات.
