# --explanation--

`Might happen` é usado para falar sobre coisas que são possíveis mas não certas. Por exemplo:

`There's a chance that a data breach might happen if we don't follow security rules.` - Uma violação de dados pode ocorrer se eles não seguirem as diretrizes corretas de segurança.

Lembre-se, verbos modais são sempre seguidos pela forma base do verbo (sem `to`).
