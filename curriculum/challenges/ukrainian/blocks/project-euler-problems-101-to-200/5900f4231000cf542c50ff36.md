# --description--

Нехай $N$ — додатне ціле число, і нехай $N$ розбите на $k$ рівних частин, $r = \frac{N}{k}$, так що $N = r + r + \cdots + r$.

Нехай $P$ — добуток цих частин, $P = r × r × \cdots × r = r^k$.

Наприклад, якщо 11 розбити на п’ять рівних частин, 11 = 2.2 + 2.2 + 2.2 + 2.2 + 2.2, тоді $P = {2.2}^5 = 51.53632$.

Нехай $M(N) = P_{max}$ для заданого значення $N$.

Виявляється, що максимум для $N = 11$ знаходиться, якщо розбити одинадцять на чотири рівні частини, що дає $P_{max} = {(\frac{11}{4})}^4$; тобто, $M(11) = \frac{14641}{256} = 57.19140625$, що є кінцевим десятковим числом.

Однак для $N = 8$ максимум досягається, якщо розбити його на три рівні частини, тому $M(8) = \frac{512}{27}$, що є нескінченним десятковим дробом.

Нехай $D(N) = N$, якщо $M(N)$ є нескінченним десятковим дробом, і $D(N) = -N$, якщо $M(N)$ є кінцевим десятковим дробом.

Наприклад, $\sum D(N)$ для $5 ≤ N ≤ 100$ дорівнює 2438.

Знайдіть $\sum D(N)$ для $5 ≤ N ≤ 10000$.
