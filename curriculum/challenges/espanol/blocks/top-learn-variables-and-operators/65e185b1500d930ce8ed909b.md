# --description--
La forma más sencilla de comenzar es crear un archivo HTML con el código JavaScript dentro. Escribe la estructura básica de HTML en un archivo en alguna parte de tu computadora:

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>
  <script>
    // Your JavaScript goes here!
    console.log("Hello, World!")
  </script>
</body>
</html>
```

Guarda y abre este archivo en un navegador web (¡puedes usar <a href="https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer" target="_blank" >"Live Server" en Visual Studio Code</a> para hacerlo!) y luego abre la consola del navegador haciendo clic derecho en la página en blanco y seleccionando "Inspect" o "Inspect Element". En el panel de ‘Developer Tools’ encuentra y selecciona la pestaña ‘Console’, donde deberías ver la salida de nuestra instrucción `console.log`.

> `console.log()` es el comando para imprimir algo en la consola de desarrollador de tu navegador. Puedes usar esto para imprimir los resultados de cualquiera de los siguientes artículos y ejercicios en la consola. Te animamos a que codifiques junto con todos los ejemplos en esta y futuras lecciones.

Otra forma de incluir JavaScript en una página web es a través de un script externo. Esto es muy similar a enlazar documentos CSS externos a tu sitio web.

```html
  <script src="javascript.js"></script>
```

Los archivos de JavaScript tienen la extensión `.js`, similar a `.css` para las hojas de estilo. Los archivos externos de JavaScript se usan para scripts más complejos.

# --questions--

## --text--

¿Qué afirmación describe con precisión cómo incluir JavaScript en un documento HTML?

## --answers--

JavaScript solo puede incluirse internamente dentro de una etiqueta `<script>` en el cuerpo del HTML.

---

El código JavaScript debe escribirse en la sección `<head>` de un documento HTML para que se ejecute correctamente.

---

JavaScript puede incluirse directamente en un archivo HTML usando una etiqueta `<script>` o vincularse externamente mediante una etiqueta `<script src="javascript.js">`. 

---

Los archivos JavaScript externos requieren un atributo especial en la etiqueta `<script>` para ser reconocidos por los navegadores web.
