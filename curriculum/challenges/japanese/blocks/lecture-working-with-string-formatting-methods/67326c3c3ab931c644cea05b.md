# --interactive--

JavaScriptで文字列を扱うとき、文字列の先頭や末尾に不要な空白があることがよくあります。空白は比較や保存、表示などの操作に支障をきたすことがあるため、効率的に取り除く方法を知っておくことが重要です。

このレッスンでは、JavaScriptの`trim()`、`trimStart()`、`trimEnd()`メソッドを使って空白をトリムする方法を見ていきます。

空白とは、文字列内にあるスペース、タブ、改行などの目に見えない文字のことです。例えば：

```js
let greeting = "   Hello, world!   ";
```

この場合、見えるテキストの前後にスペースがあります、`Hello, world!`。

`trim()`メソッドは、文字列の先頭と末尾の空白を取り除く最も一般的な方法です。例を見てみましょう：

:::interactive_editor

```js
let message = "   Hello!   ";
console.log(message); // "   Hello!   "
let trimmedMessage = message.trim();
console.log(trimmedMessage);  // "Hello!"
```

:::

この場合、`trim()`メソッドは先頭と末尾のすべてのスペースを取り除き、`Hello!`だけを残します。文字列内の空白（例えば単語間のスペース）は`trim()`によってそのまま残されることに注意してください。

時には、文字列の先頭か末尾のどちらか一方だけの空白を取り除きたい場合もあります。そんなときに`trimStart()`と`trimEnd()`が役立ちます。

`trimStart()`は文字列の先頭（開始位置）の空白を取り除きます。

:::interactive_editor

```js
let greeting = "   Hello!   ";
console.log(greeting);  // "   Hello!   "
let trimmedStart = greeting.trimStart();
console.log(trimmedStart);  // "Hello!   "
```

:::

`trimEnd()`は文字列の末尾の空白を取り除きます。

:::interactive_editor

```js
let greeting = "   Hello!   ";
console.log(greeting);  // "   Hello!   "
let trimmedEnd = greeting.trimEnd();
console.log(trimmedEnd);  // "   Hello!"
```

:::

これらのメソッドを使うことで、文字列のどの部分の空白を取り除くかをより細かくコントロールできます。

まとめると、空白のトリムはJavaScriptで文字列を扱う際に欠かせない操作です。入力データをきれいにしたり、一貫した文字列比較を行いたい場合は、`trim()`を使って両端の空白を取り除くか、`trimStart()`と`trimEnd()`を使って特定の側だけを対象にしてください。
