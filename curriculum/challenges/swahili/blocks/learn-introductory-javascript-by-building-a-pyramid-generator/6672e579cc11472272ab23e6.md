# --description--


JavaScript ni lugha ya programu inayotumia mtandao. Tofauti na HTML na CSS ulizojifunza awali, JavaScript hutumika zaidi kuandika mantiki badala ya alama za uandishi.

Katika mradi huu, utajifunza misingi ya JavaScript na kutumia dhana hizo kujenga kizalishaji cha piramidi.

Kizalishaji cha piramidi ni programu ambapo unaweza kuweka aina ya tabia, idadi ya piramidi, na mwelekeo wa piramidi. Programu hiyo itazalisha piramidi kulingana na ingizo hizo.

Bofya kitufe cha "Check your code" ili uende kwenye hatua inayofuata na kuanza mradi.
