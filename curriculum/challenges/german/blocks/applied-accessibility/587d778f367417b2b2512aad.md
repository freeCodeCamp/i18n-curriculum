# --description--

Es gibt verschiedene Formen der Farbenblindheit. Diese reichen von einer verminderten Empfindlichkeit gegenüber einer bestimmten Lichtwellenlänge bis hin zur völligen Unfähigkeit, Farben zu sehen. Die häufigste Form ist eine verminderte Empfindlichkeit für Grüntöne.

Wenn zum Beispiel zwei ähnliche Grüntöne als Vordergrund- und Hintergrundfarbe Ihres Inhalts verwendet werden, kann ein farbenblinder Nutzer diese möglicherweise nicht unterscheiden. Ähnliche Farben können als Nachbarn auf dem Farbrad betrachtet werden, und solche Kombinationen sollten vermieden werden, wenn wichtige Informationen vermittelt werden.

**Hinweis:** Einige Online-Farbwähler enthalten visuelle Simulationen, wie Farben für verschiedene Arten der Farbenblindheit erscheinen. Diese sind großartige Hilfsmittel zusätzlich zu Online-Rechnern für die Überprüfung des Kontrastverhältnisses.

# --instructions--

Camper Cat testet verschiedene Stile für einen wichtigen Button, aber das Gelb (`#FFFF33`) `background-color` und der grüne (`#33FF33`) Text `color` sind benachbarte Farbtöne auf dem Farbrad und für einige farbenblinde Nutzer praktisch nicht unterscheidbar. (Ihre ähnliche Helligkeit besteht auch den Kontrastverhältnis-Test nicht). Ändern Sie den Text `color` in ein dunkles Blau (`#003366`), um beide Probleme zu lösen.

# --hints--

Ihr Code sollte den Text `color` für das `button` in das dunkle Blau ändern.

```js
const button = document.querySelector('button');
const buttonColor = window.getComputedStyle(button).color; 
assert.equal(buttonColor, 'rgb(0, 51, 102)');
```
