# --description--

`Whoa` ist ein Wort, das Sie sagen, wenn Sie von etwas überrascht oder beeindruckt sind. Zum Beispiel, wenn Sie einen großen, schönen Regenbogen sehen, könnten Sie `Whoa!` sagen, um zu zeigen, dass Sie beeindruckt sind. Es kann auch verwendet werden, um jemandem zu sagen, er soll aufhören oder langsamer machen.

Wenn Sie sagen, dass etwas `a mouthful` ist, meinen Sie, dass etwas (wie ein Wort oder Satz) lang oder schwer auszusprechen ist. Zum Beispiel, wenn ein Wort oder Name viele Buchstaben und Laute hat, können Sie `it's a mouthful` sagen. Das bedeutet, dass es schwer ist, es auf einmal auszusprechen.

Hören Sie sich den Dialog an und füllen Sie die Lücken aus.
