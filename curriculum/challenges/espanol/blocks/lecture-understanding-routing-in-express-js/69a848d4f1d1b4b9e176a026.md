# --description--

Ahora que has creado una aplicación básica con Express, es momento de examinar más de cerca el enrutamiento y cómo funciona.

En Express, el enrutamiento es la forma en que defines los puntos finales de una aplicación web. Puedes pensar en las rutas como reglas que indican a la app cómo responder cuando un usuario visita una URL específica.

Veamos de nuevo nuestra sencilla aplicación Express:

```javascript
const express = require("express");
const app = express();
const port = 3000;

app.get("/", (req, res) => {
  res.send("Hello World!");
});

app.listen(port, () => {
  console.log(`Example app listening on port ${port}`);
});
```

En este ejemplo, la app define una ruta para la URL raíz (`/`). Cuando un cliente envía una solicitud GET a esta ruta, el servidor responde con el texto `Hello World!`. Luego, la app comienza a escuchar las solicitudes entrantes en el puerto `3000`.

Pero existen otros métodos de ruta.

Express te permite manejar diferentes tipos de solicitudes HTTP usando distintos métodos de enrutamiento. Los más comunes son:

* `app.get()` – recupera datos
    
* `app.post()` – crea nuevos datos
    
* `app.put()` – actualiza datos existentes
    
* `app.delete()` – elimina datos
    

Cada método define cómo debe responder tu aplicación a una solicitud HTTP específica en una ruta determinada.

Aquí tienes algunos ejemplos simples:

```javascript
app.post("/submit", (req, res) => {
  res.send("Form submitted!");
});

app.put("/update", (req, res) => {
  res.send("Item updated!");
});

app.delete("/delete", (req, res) => {
  res.send("Item deleted!");
});
```

En cada caso, Express escucha una solicitud con un método HTTP y una ruta URL particulares, luego ejecuta la función manejadora correspondiente para enviar una respuesta al cliente.

Puedes probar estas rutas usando herramientas como Postman o `curl`, que te permiten enviar diferentes tipos de solicitudes HTTP a tu servidor.

Y eso es el enrutamiento en Express: simplemente se trata de definir cómo responde tu aplicación a distintas solicitudes.
