# --explanation--

`Clarifying` يعني شرح شيء بطريقة تزيل الالتباس. يساعد ذلك على فهم أفضل لمفهوم أو موقف. على سبيل المثال:

`Thanks for clarifying how the API works.` - المتحدث ممتن لأن الشرح جعل API أسهل للفهم.

`Abstraction` يشير إلى تبسيط شيء ما عن طريق إظهار الميزات الأساسية فقط وإخفاء التفاصيل التقنية المعقدة. على سبيل المثال:

`Using a login function is an example of abstraction - you don't need to know how the password is checked, just that it works.` - هذا يوضح كيف يتفاعل المستخدمون مع واجهات بسيطة بينما تبقى المنطق التقنية مخفية.
