# --description--

Sea $S(A)$ la suma de elementos en el conjunto A del tamaño n. Lo llamaremos un conjunto de suma especial si para dos subconjuntos disjuntos no vacíos cualesquiera, B y C, las siguientes propiedades son verdaderas:

1. $S(B) ≠ S(C)$; es decir, las sumas de subconjuntos no pueden ser iguales.
2. Si B contiene más elementos que C, entonces $S(B) > S(C)$.

Si el $S(A)$ se minimiza para un determinado n, lo llamaremos un conjunto de suma especial óptima. Los primeros cinco conjuntos de sumas especiales óptimas se dan a continuación.

$$\begin{align}   & n = 1: \\{1\\} \\\\
  & n = 2: \\{1, 2\\} \\\\   & n = 3: \\{2, 3, 4\\} \\\\
  & n = 4: \\{3, 5, 6, 7\\} \\\\   & n = 5: \\{6, 9, 11, 12, 13\\} \\\\
\end{align}$$

Parece que para un conjunto óptimo dado, $A = {a_1, a_2, \ldots, a_n}$, el siguiente conjunto óptimo es de la forma $B = {b, a_1 + b, a_2 + b, \ldots, a_n + b}$, donde b es el elemento "medio" de la fila anterior.

Aplicando esta "regla" esperaríamos que el conjunto óptimo para $n = 6 sea $A = {11, 17, 20, 22, 23, 24}, con $S(A) = 117. Sin embargo, este no es el conjunto óptimo, ya que simplemente hemos aplicado un algoritmo para obtener un conjunto casi óptimo. El conjunto óptimo para $n = 6 es $A = {11, 18, 19, 20, 22, 25}, con $S(A) = 115 y la cadena correspondiente del conjunto: `111819202225`.

Dado que A es un conjunto especial óptimo para $n = 7, encuentra su cadena de conjunto.

**Nota:** Este problema está relacionado con el Problema 105 y el Problema 106.
