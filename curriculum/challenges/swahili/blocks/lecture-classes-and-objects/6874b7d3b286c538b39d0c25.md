# --description--

Njia maalum katika Python, pia zinajulikana kama "njia za kichawi" au "njia dunder", ni njia maalum za Python zinazotangulia na kumalizika na alama mbili za mstari chini (`__`). Neno "dunder" lenyewe linatokana na alama mbili za mstari chini (**d** kwa double, **under** kwa underscores).

Huenda tayari umewahi kutumia njia maalum bila kujua. Kila unapouandika kitu kama `3 + 4`, Python hufanya wito wa `3.__add__(4)` kimya kimya nyuma ya pazia. Hiyo ni njia maalum ikifanya kazi. Hivyo ingawa unaweza kuita njia maalum moja kwa moja, mara nyingi hufanyi hivyo. Kitu kama `3 + 4` ni wazi zaidi na rahisi kusoma kuliko kuita `3.__add__(4)` mwenyewe.

Mbali na `__add__`, `__init__()` ni njia nyingine maalum utakayoiwona na kuitumia zaidi, kwani ni manzilishi wa darasa la vitu. Pia kuna nyingine kama `__len__()` na `__str__()`.

Fikiria njia maalum kama wakurugenzi wa shughuli kati ya mtu anayeandika programu na tafsiri ya lugha ya Python yenyewe.

Kumbuka, huna haja ya kuita njia maalum moja kwa moja. Badala yake, Python huwaita moja kwa moja wakati matendo fulani yanapotokea. Operesheni hizi ni pamoja na:

- **Operesheni za kihesabu kama kuongeza, kutoa, kuzidisha, kugawanya, na nyinginezo**. Kwa kuongeza, `__add__()` huitwa, `__sub__()` kwa kutoa, `__mul__()` kwa kuzidisha, na `__truediv__()` kwa kugawanya.
    
- **Operesheni za mfuatano wa herufi kama kuunganisha, kurudia, kuunda muundo, na kubadilisha kuwa maandishi**. `__add__()` huitwa kwa kuunganisha, `__mul__()` kwa kurudia, `__format__()` kwa kuunda muundo, `__str__()` na `__repr__()` kwa kubadilisha kuwa maandishi, na kadhalika.
    
- **Operesheni za kulinganisha kama usawa, chini ya, juu ya, na nyinginezo**. `__eq__()` huitwa kwa ukaguzi wa usawa, `__lt__()` kwa chini ya, `__gt__()` kwa juu ya, na kadhalika.
    
- **Operesheni za kurudia kama kufanya kitu kiwe kinachoweza kurudiwa na kusonga mbele kupitia vitu**. `__iter__()` huitwa kurudisha kielezi na `__next__()` kuchukua kipengele kinachofuata.
    

Kwa kawaida, aina za data za Python kama mfuatano wa herufi na nambari tayari zinafahamu jinsi ya kuongeza vitu, kuunganisha, kulinganisha usawa, kutumika katika mizunguko, na nyinginezo.

Lakini unapounda darasa lako mwenyewe, Python haitajua jinsi ya kushughulikia mambo moja kwa moja.

Hapa ndipo njia maalum zinapokuja — zinakuwezesha kubinafsisha tabia iliyojengwa ndani ya Python.

Tuseme unataka kupata idadi ya kurasa katika vitu vya kitabu vilivyoundwa na darasa lililopo hapa chini, au kulinganisha na kupata mfuatano wa herufi unaosomeka wa vitu hivyo. Hapa ni kinachotokea bila njia maalum:

```python
class Book:
   def __init__(self, title, pages):
       self.title = title
       self.pages = pages

book1 = Book("Built Wealth Like a Boss", 420)
book2 = Book("Be Your Own Start", 420)

print(len(book1)) # TypeError: object of type 'Book' has no len()
print(str(book1)) # <__main__.Book object at 0x102ed2900>
print(book1 == book2) # False even though they have the same number of pages
```

Katika mfano huu:

- `len(book1)` ilishindwa kwa sababu Python haitajua jinsi ya kupata urefu wa kitu chako cha kitabu bila `__len__()`
    
- `str(book1)` ilichapisha kitu kama `<__main__.Book object at 0x102ed2900>` kwa sababu hiyo ndiyo rejea ya kawaida wakati hujatumia `__str__()`
    
- `book1 == book2` ilisababisha `False` kwa sababu Python huangalia tu kama vitu vyote viwili ni sawa katika kumbukumbu, si kwa maudhui.
    

Hapa ni jinsi unavyoweza kufafanua njia maalum zako za `__len__()`, `__str__()`, na `__eq__()` ili kufanya kazi na vitu vilivyotengenezwa kutoka darasa la `Book` iwe rahisi:

```python
class Book:
   def __init__(self, title, pages):
       self.title = title
       self.pages = pages

   def __len__(self):
       return self.pages

   def __str__(self):
       return f"'{self.title}' has {self.pages} pages"

   def __eq__(self, other):
       return self.pages == other.pages
  
book1 = Book("Built Wealth Like a Boss", 420)
book2 = Book("Be Your Own Start", 420)

print(len(book1)) # 420
print(len(book2)) # 420
print(str(book1)) # 'Built Wealth Like a Boss' has 420 pages
print(str(book2)) # 'Be Your Own Start' has 420 pages
print(book1 == book2) # True
```

Mfano mwingine ni gari la ununuzi ambapo unafanya yafuatayo:

- Ongeza vitu kwenye gari
    
- Ondoa vitu kutoka kwenye gari
    
- Pata idadi ya vitu vilivyomo kwenye gari
    
- Angalia ni vitu gani vilivyomo kwenye gari
    
- Angalia kama kitu fulani kiko kwenye gari
    
- Rudisha au onyesha kitu kilicho kwenye kielezo fulani cha gari
    

Ingawa unaweza kuwa na njia inayoongeza vitu kwenye gari na kuondoa vitu fulani kutoka kwenye gari, unaweza kuunda njia maalum kwa ajili ya kazi zingine zote:

- `__len__()` kupata urefu wa vitu vilivyomo kwenye gari
    
- `__iter__()` kurudia vitu vilivyomo kwenye gari ili uweze kuviangalia
    
- `__contains__()` kuangalia kama kitu fulani kiko kwenye gari
    
- `__getitem__()` kurudisha au kuonyesha kitu kilicho kwenye kielezo fulani cha gari
    

Hapa kuna mfano wa darasa la `Cart` lenye njia hizi za mtumiaji na njia maalum:

```python
class Cart:
   def __init__(self):
       self.items = []

   def add(self, item):
       self.items.append(item)

   def remove(self, item):
       if item in self.items:
           self.items.remove(item)
       else:
           print(f'{item} is not in cart')

   def list_items(self):
       return self.items

   def __len__(self):
       return len(self.items)

   def __getitem__(self, index):
       return self.items[index]

   def __contains__(self, item):
       return item in self.items

   def __iter__(self):
       return iter(self.items)
```

Na hapa ni jinsi unavyoweza kuvitumia:

```python
cart = Cart()
cart.add('Laptop')
cart.add('Wireless mouse')
cart.add('Ergo keyboard')
cart.add('Monitor')

for item in cart:
   print(item, end=' ') # Laptop Wireless mouse Ergo keyboard Monitor

print(len(cart)) # 4
print(cart[3]) # Monitor

print('Monitor' in cart) # True
print('banana' in cart) # False

cart.remove('Ergo keyboard')

print(cart.list_items()) # ['Laptop', 'Wireless mouse', 'Monitor']

cart.remove('banana') # banana is not in cart
```
