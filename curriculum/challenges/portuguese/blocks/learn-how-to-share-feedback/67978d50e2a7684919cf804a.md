# --explanation--

`To set goals` significa definir objetivos ou metas específicas para alcançar.Por exemplo:

`The team set goals to improve customer satisfaction this year.` - Isso significa que a equipe estabeleceu objetivos relacionados à satisfação do cliente.

`Quarter` refere-se a um quarto de um ano ou a um período de três meses.No contexto empresarial, `the next quarter` significa o próximo período de três meses no ano civil ou fiscal.Por exemplo:

- `Our sales target for the next quarter is $1 million.` - Isso significa que a meta se aplica ao próximo período de três meses.

- `We review our performance at the end of each quarter.` - Isso se refere a avaliar o progresso a cada três meses.
