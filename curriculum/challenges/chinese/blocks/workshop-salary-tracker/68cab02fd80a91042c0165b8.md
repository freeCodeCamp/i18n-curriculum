# --description--

在你现有的 `if` 语句之后，创建另一个用于当新工资低于当前级别的基本工资时的语句。使用 `hasattr(self, '_level')` 来避免初始化期间的 `AttributeError`。

在 `if` 语句内，引起一个带有消息 `ValueError`，后跟当前级别的基本工资和句号的 `Salary must be higher than minimum salary $`。

```js
({ test: () => runPython(`

  built_in_print = print
  print = lambda x: x

  try:
    emp.salary = minimum_salary - 1
  except ValueError as e:

```
