# --explanation--

`Going to` si usa per parlare di azioni pianificate o intenzioni per il futuro. Per esempio:

- `We are going to upgrade the system next month.` - Questo significa che l'aggiornamento è pianificato e avverrà presto.
  
- `She is going to learn SQL next year.` - Questo significa che lei intende imparare SQL in futuro.

In questa frase:

`We're` è la contrazione di `we are`.

`Going to` indica che l'azione è pianificata per un futuro prossimo.

`Need` è il verbo base che indica qualcosa di necessario o richiesto.
