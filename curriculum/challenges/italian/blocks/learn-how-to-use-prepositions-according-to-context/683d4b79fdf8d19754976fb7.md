# --explanation--

`Should` si usa per chiedere un consiglio o suggerire qual è la cosa migliore. Per esempio:

`Where should we sit?` – Questa frase chiede qual è il posto migliore dove sedersi.

`Put` significa mettere qualcosa da qualche parte. Per esempio:

`Please put the laptop on the table.` – Questo significa spostare il portatile in quel luogo.
