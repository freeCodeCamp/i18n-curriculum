# --interactive--

Katika JavaScript, variables hufanya kazi kama kontena za kuhifadhi data ambayo unaweza kupata na kubadilisha katika programu yako yote.

Unaweza kufikiria variables kama masanduku yanayoshikilia thamani. Kwa variables, unaweza kufuatilia vitu kama nambari au maandishi na kurejelea thamani hizi wakati wowote unazihitaji katika programu yako.

Njia moja ya kutangaza variable katika JavaScript ni kutumia neno muhimu la `let`. Utajifunza zaidi kuhusu neno muhimu la `let` pamoja na njia nyingine za kutangaza variables katika mafundisho yajayo.

Hapa kuna mfano wa kutumia `let` kutangaza variable inayoitwa `age`:

```js
let age;
```

Kwa sasa, variable ya `age` haina thamani imepewa. Ukijaribu kuitumia, itarudisha `undefined`, ambayo ina maana haina thamani.

Hapa kuna mfano.

**KUMBUKA**: `console.log()` ni kitendakazi kinachotoa taarifa kwenye konsoli, sehemu ya kivinjari chako cha mtandao inayotumika kutafuta makosa katika msimbo. Utajifunza zaidi kuhusu `console.log()` katika mafundisho yajayo. Pia, alama za `//` hutumika kuongeza maoni katika msimbo wako. Maoni ni vidokezo kwa ajili yako mwenyewe au waprogramu wengine ambavyo havizingatiwi wakati msimbo unapoendeshwa.

:::interactive_editor
 
```js
let age;
console.log(age); // undefined
```

:::

Ili kuweka thamani kwa variable utahitaji kutumia opereta ya mgawo kama ifuatavyo:

```js
let age = 25;
```

Sasa unapotumia variable ya `age`, itarudisha thamani ya `25`.

:::interactive_editor

```js
let age = 25;
console.log(age); // 25
```

:::

Opereta ya mgawo inaonekana kama alama ya sawa (`=`) lakini haikagui usawa. Utajifunza kuhusu opereta sahihi za kukagua usawa katika mafundisho yajayo.

Opereta ya mgawo hutumika kuweka thamani kwa variable. Mchakato huu wa kuweka thamani kwa variable unajulikana kama uanzishaji.

Faida moja ya kutumia neno muhimu la `let` kutangaza variables ni kwamba unaweza kuweka tena thamani kwao. Katika programu, kuweka tena thamani ina maana ya kutoa thamani mpya kwa variable ambayo tayari ina thamani.

Hapa kuna mfano wa kuweka tena thamani kwa variable ya `age`.

:::interactive_editor

```js
let age = 25;
console.log(age); // 25
age = 30;
console.log(age); // 30
```

:::

Sasa variable ya `age` ina thamani ya `30`. Angalia kwamba neno muhimu la `let` halikutakiwa tena kwa sababu variable ya `age` tayari ilitangazwa, hivyo hakuna haja ya kuitangaza mara ya pili.

Unapotumia kuweka tena thamani, unahitaji tu kurejelea jina la variable. Kuweka tena thamani ni muhimu kwa sababu kunakuwezesha kusasisha na kubadilisha thamani iliyohifadhiwa katika variable wakati programu yako inapoendelea. Mfano mzuri wa hili ni kusasisha pointi katika mchezo.

Kuitwa kwa variables kunaweza kuonekana rahisi, lakini kuna baadhi ya sheria na mbinu bora kuhakikisha msimbo wako unasomeka na kufanya kazi vizuri.

Majina ya variables yako yanapaswa kuelezea kile data inawakilisha. Kwa mfano, badala ya kutumia jina kama `x`, jina linaloelezea zaidi kama `age` au `points` linafanya msimbo wako kuwa rahisi kuelewa.

```js
// Bad variable names
let x = 10;
let y = "John";

// Good variable names
let age = 10;
```

Variables katika JavaScript lazima ziweze kuanza na herufi, mstari chini (`_`), au alama ya dola (`$`). Haziwezi kuanza na nambari.

```js
// Valid variable names
let age;
let _score;
let $total;

// Invalid variable names
let 1stPlace; // starts with a number
```

Majina ya variables yanazingatia herufi ndogo na kubwa, ikimaanisha neno `age` kwa herufi ndogo zote na neno `Age` lenye herufi kubwa ya `A` yanachukuliwa kuwa variables tofauti.

:::interactive_editor

```js
let age = 25;
let Age = 30;
console.log(age); // 25
console.log(Age); // 30
```

:::

Hii ndiyo sababu ni muhimu kufuata mkataba wa kuitwa kwa majina kama camelCase. camelCase ni pale neno la kwanza likiwa herufi ndogo zote na kila neno linalofuata kuanza na herufi kubwa.

Hapa kuna mfano wa kutumia mkataba wa camelCase kwa variable:

```js
let thisIsCamelCase;
let anotherExampleVariable;
let freeCodeCampStudents;
```

Kuna maneno muhimu fulani katika JavaScript ambayo huwezi kutumia kama majina ya variables, kama `let`, `const`, `function`, au `return`, kwa sababu yamehifadhiwa kwa lugha yenyewe.

Pia unapaswa kuepuka kutumia herufi maalum kama alama za mshangao (`!`) au alama ya at (`@`) katika majina ya variables. Ni bora kuweka majina ya variables yasomeke kwa kutumia herufi, nambari, mistari chini, au alama za dola.

Kwa kufuata miongozo hii, msimbo wako utakuwa safi zaidi na rahisi kusimamia kadri unavyoongezeka kwa ugumu.
