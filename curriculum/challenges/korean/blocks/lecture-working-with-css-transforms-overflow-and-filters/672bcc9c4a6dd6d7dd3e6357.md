# --description--

CSS 박스 모델은 웹 개발의 기본 개념입니다. HTML 요소가 어떻게 구조화되고 배치되는지 정의합니다. 이 모델을 이해하면 웹사이트의 요소 크기, 간격, 그리고 외관을 제어할 수 있습니다.

CSS 박스 모델에서 모든 요소는 박스로 둘러싸여 있습니다. 이 박스는 네 가지 요소로 구성됩니다: 콘텐츠 영역, 패딩, 테두리, 그리고 여백입니다.

콘텐츠 영역은 박스의 가장 안쪽 부분입니다. 텍스트나 이미지 같은 요소의 실제 콘텐츠가 들어 있는 공간입니다.

패딩은 콘텐츠 영역 바로 다음에 오는 영역입니다. 콘텐츠 영역과 요소의 테두리 사이의 공간입니다. 패딩을 사용하면 콘텐츠 주변에 공간을 추가해 가독성을 높일 수 있습니다. `padding` 속성으로 위, 오른쪽, 아래, 왼쪽 패딩에 각각 다른 값을 설정할 수 있습니다.

다음은 `padding` 단축 속성을 사용한 예시로, 위쪽 패딩을 15픽셀, 오른쪽 패딩을 5픽셀, 아래쪽 패딩을 2픽셀, 왼쪽 패딩을 8픽셀로 설정했습니다:

```css
padding: 15px 5px 2px 8px;
```

테두리는 CSS 박스 모델에서 요소의 바깥 가장자리 또는 윤곽선입니다. 요소의 시각적 경계입니다. `border` 속성을 사용해 테두리 스타일, 너비, 색상 등 다른 속성을 사용자 지정할 수 있습니다. 다음은 테두리 너비를 5픽셀, 스타일을 실선, 색상을 파란색으로 설정한 예시입니다:

```css
border: 5px solid blue;
```

값을 생략하면 해당 값의 기본 속성이 사용됩니다. 너비는 `medium`, 스타일은 `none`, 색상은 현재 색상이 기본값입니다.

모든 면에 동일한 스타일을 적용하려면 `border` 단축 속성에 이 세 가지 속성을 직접 설정할 수 있습니다. 하지만 각 면에 다른 스타일을 할당하려면 `border-width`, `border-style`, `border-color` 속성을 사용할 수 있습니다.

```css
border-width: 2px 4px 7px 12px;
border-style: dashed solid solid dashed;
border-color: blue red green black;
```

각 속성에 최대 네 개의 값을 작성할 수 있습니다. 값은 시계 방향으로 위쪽부터 적용됩니다. 값이 하나만 있으면 네 면 모두에 적용됩니다.

마지막으로 여백은 요소 테두리 바깥의 공간입니다. 요소와 주변 다른 요소 사이의 거리를 결정합니다. `margin` 속성으로 요소의 위, 오른쪽, 아래, 왼쪽 여백에 각각 다른 값을 설정할 수 있습니다.

다음 예시에서 위쪽 여백은 3픽셀, 오른쪽 여백은 12픽셀, 아래쪽 여백은 9픽셀, 왼쪽 여백은 7픽셀입니다:

```css
margin: 3px 12px 9px 7px;
```

이 네 가지 컴포넌트는 요소의 전체 너비와 높이를 계산하는 데 필수적입니다.

다음 수업들에서 브라우저가 이를 어떻게 처리하는지, 그리고 어떻게 사용자 지정할 수 있는지 더 배우게 됩니다. CSS 박스 모델은 웹 개발의 기본 개념입니다.

이 컴포넌트들이 어떻게 상호작용하고 요소 크기에 기여하는지 이해하는 것은 웹 디자인을 구현하는 데 필수적입니다.
