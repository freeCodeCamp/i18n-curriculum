# --description--

`What are you up to?` 是一个随意且友好的问题，人们用它来了解某人当前的活动或计划。它也可以表示对某人此刻可能正在做什么的好奇。例如：

`What are you up to this weekend?` - 这是在询问某人的周末计划。

类似地，`up to` 意味着正在做某事。例如：

`He is up to something.` — 这意味着他正在做某件事，通常带有好奇或怀疑的意味。
