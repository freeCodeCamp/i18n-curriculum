# --description--

`Seriously` se usa cuando quieres mostrar que algo es importante o verdadero. Ayuda a hacer más fuerte lo que estás diciendo.

Por ejemplo, si dices, `I seriously love programming`, significa que realmente amas mucho programar. O, si dices, `Take this project seriously`, significa que piensas que el proyecto es muy importante y debe ser tratado con cuidado.
