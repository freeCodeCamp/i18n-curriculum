# --interactive--

외부 폰트는 프로젝트 파일에 직접 포함되지 않은 폰트 파일입니다. 보통 별도의 서버에 호스트되어 있습니다. 서버는 네트워크를 통해 다른 컴퓨터에 데이터나 서비스를 제공하는 컴퓨터입니다. 서버에 대해서는 앞으로 수업에서 더 배우게 됩니다. 외부 폰트를 사용하면 사용자의 기기에 설치되어 있지 않은 맞춤 폰트를 사용할 수 있어 더 많은 유연성을 제공합니다.

Google Fonts와 Font Squirrel은 웹 개발용 무료 폰트를 찾고 사용할 수 있는 인기 있는 온라인 자원입니다. 프로젝트에 아주 쉽게 통합할 수 있습니다.

먼저 Google Fonts부터 시작해 보겠습니다. Google Fonts는 웹 개발용으로 특별히 디자인된 폰트를 많이 제공하는 Google 서비스입니다. Google Fonts 사용자 인터페이스에는 여러 가지 요소가 있습니다. 하나씩 살펴보겠습니다. 왼쪽 사이드바에는 다음이 있습니다:

- 폰트를 보고, 찾고, 필터링할 수 있는 섹션
- Noto라는 고품질 폰트 모음에 관한 특별 섹션으로, 1,000개 이상의 언어와 150개 이상의 문자 체계에 적합한 다양한 굵기, 너비, 스타일을 제공합니다
- 웹 프로젝트용 아이콘을 찾고 다운로드할 수 있는 섹션
- 폰트와 모범 사례에 대해 더 배우고 자주 묻는 질문을 볼 수 있는 섹션

웹사이트에 Google Font를 추가하려면 첫 번째 섹션(Fonts)으로 가세요. 왼쪽 사이드바의 "Preview"라고 적힌 곳에서 미리보기 텍스트를 맞춤 설정할 수 있습니다. 보고 싶은 텍스트를 입력하면 자동으로 업데이트됩니다. 이는 웹사이트에 사용할 텍스트로 폰트를 미리 보는 데 유용합니다. 폰트 크기를 조절하거나 폰트 특성에 따라 필터링할 수도 있습니다.

폰트에 대한 자세한 정보를 보려면 폰트를 클릭하세요. 폰트를 디자인한 사람, 미리보기 텍스트, 라이트, 레귤러, 미디엄, 굵게, 블랙 같은 다양한 스타일 등 중요한 정보를 볼 수 있습니다. 상단 옵션을 클릭하면 타입 테스터로 이동하거나 개별 글리프를 보고, 폰트와 라이선스에 대해 더 배울 수 있습니다.

미리보기 텍스트 크기는 미리보기 텍스트 입력란 옆에 있는 파란색 슬라이더로 조절할 수 있습니다. 프로젝트에 폰트를 추가할 준비가 되면 상단의 파란색 "Get font" 버튼을 클릭하세요. 현재 선택한 폰트가 요약 페이지에 표시됩니다. 여러 폰트를 동시에 선택할 수 있습니다.

다음으로, 폰트 파일을 다운로드하여 프로젝트에 로컬 파일로 추가할지, 아니면 외부 폰트로 사용하여 사용자가 웹사이트에 접속할 때 Google 서버에서 다운로드할지 선택해야 합니다. 다운로드하려면 "Download all"을 클릭하고, Google 서버에서 외부 폰트로 사용하려면 "Get embed code"를 클릭하세요. "Get embed code"를 클릭하면 프로젝트에 외부 폰트를 추가하는 방법에 대한 지침이 표시됩니다.

웹 개발 프로젝트에서는 `link` 요소 또는 `@import` 두 가지 옵션이 있습니다. `link` 요소 옵션을 선택하면 HTML 코드 조각과 CSS 규칙을 복사해 프로젝트에 붙여넣을 수 있습니다. HTML 파일의 `head` 요소에 코드를 임베드하고 필요에 맞는 CSS 규칙을 추가하세요.

각 폰트 스타일마다 CSS 규칙이 있습니다. 각 규칙은 커스텀 폰트가 제대로 로드되지 않을 경우를 대비해 대체 폰트를 할당합니다.

`@import` 옵션을 선택하면 해당 규칙을 CSS 파일에 추가해야 합니다. 예시는 다음과 같습니다:

```css
@import url('https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap');
```

CSS 규칙은 완전히 동일합니다.

특정 폰트 스타일만 포함하고 싶으면 이 코드 조각에서 제거할 수도 있습니다. HTML과 CSS 코드 조각은 자동으로 업데이트됩니다.

이것이 Google Fonts입니다. 웹 개발 프로젝트에 무료이면서 오픈 소스인 폰트를 찾는 가장 인기 있는 옵션 중 하나입니다.

또 다른 훌륭한 옵션은 Font Squirrel입니다. 여기서 원하는 맞춤 폰트를 찾아 다운로드할 수 있습니다. 폰트를 검색하고 결과를 클릭하면 폰트에 대한 자세한 정보, 샘플, 테스트 드라이브, 글리프, Webfont 키트를 볼 수 있습니다. 얇은, 라이트, 미디엄, 굵게, 블랙 같은 다양한 스타일과 변형 샘플도 확인할 수 있습니다.

프로젝트에 사용할 폰트를 결정했으면 "Webfont Kit" 탭으로 가세요. 여기서 폰트 라이선스가 `@font-face` CSS 임베딩에 사용할 수 있는지 확인할 수 있습니다. 부분집합과 포맷도 선택할 수 있습니다.

"Download @font-face Kit" 버튼을 클릭하면 압축(zip) 파일이 다운로드됩니다. 압축을 풀면 "web fonts" 폴더, 텍스트 파일 형태의 라이선스, 웹 폰트 사용법이 담긴 HTML 파일이 있습니다. web fonts 폴더에는 프로젝트에 필요한 모든 폰트 파일이 들어 있습니다. 서로 다른 폰트 스타일별로 특정 폴더에 그룹화되어 있습니다.

HTML 파일을 브라우저에서 열면 프로젝트에 폰트를 추가하는 방법, `@font-face` 선언으로 스타일시트에 포함하는 방법, CSS 규칙에서 사용하는 방법에 대한 지침이 있습니다.

이렇게 Font Squirrel 맞춤 폰트를 프로젝트에 추가할 수 있습니다. 별도의 서버에 호스트하여 외부 폰트로 사용할 수도 있습니다. 외부 폰트는 웹사이트의 전체적인 모습과 디자인을 맞춤화하는 데 매우 유용합니다. 웹사이트를 진정으로 독특하게 만들 수 있습니다.

하지만 여러 외부 폰트를 사용하면 웹사이트 로드 시간이 늘어나 전체 성능에 영향을 줄 수 있다는 점도 기억하세요. 스타일과 효율성 사이에서 균형을 찾는 것이 중요합니다.
