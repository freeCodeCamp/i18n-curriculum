# --explanation--

`Scheduled` bedeutet, dass etwas offiziell geplant ist, zu einem bestimmten Zeitpunkt stattzufinden. Zum Beispiel:

`The meeting is scheduled for next Monday.` – Das bedeutet, es ist geplant, an diesem Tag stattzufinden.

`Quarter` bezeichnet einen von vier Dreimonatszeiträumen in einem Geschäfts- oder Finanzjahr. Zum Beispiel:

`Sales increased in the second quarter.` – Das bedeutet während des zweiten Dreimonatszeitraums des Jahres.
