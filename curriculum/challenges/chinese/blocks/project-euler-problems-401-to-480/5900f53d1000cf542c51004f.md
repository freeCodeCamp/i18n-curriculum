# --description--

莫比乌斯函数，记作 $μ(n)$，定义为：

- 当 $n$ 是无平方因子时，$μ(n) = (-1)^{ω(n)}$（其中 $ω(n)$ 是 $n$ 的不同素数因子的数量）
- 当 $n$ 不是无平方因子时，$μ(n) = 0$。

设 $P(a, b)$ 为区间 $[a, b]$ 中满足 $μ(n) = 1$ 的整数个数。

设 $N(a, b)$ 为区间 $[a, b]$ 中满足 $μ(n) = -1$ 的整数 $n$ 的数量。

例如，$P(2, 10) = 2$ 且 $N(2, 10) = 4$。

令 $C(n)$ 为满足条件的整数点对 $(a, b)$ 的数量：

- $1 ≤ a ≤ b ≤ n$，
- $99 \times N(a, b) ≤ 100 \times P(a, b)$，且
- $99 \times P(a, b) ≤ 100 \times N(a, b)$。

例如，$C(10) = 13$，$C(500) = 16\,676$ 和 $C(10\,000) = 20\,155\,319$。

查找 $C(20\,000\,000)$。
