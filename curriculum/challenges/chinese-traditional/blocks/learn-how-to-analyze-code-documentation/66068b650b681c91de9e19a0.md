# --description--

所有格 `'s` 表示某物屬於某人或某物，例如 `Brian's book`，表示這本書屬於 Brian。

`Dealing` 是指管理或處理某事。例如，`When dealing with homework, it's important to be focused.`

程式設計中，`function's signature` 就像函式的身分證。它告訴你函式的名稱、所需的參數，以及它所傳回的 `return type`。

函式的 `return type` 是函式執行後所傳回的結果，就像自動販賣機在你投入錢後傳回一罐汽水。
