# --explanation--

`Might` використовується, щоб висловити можливість. Це означає, що щось можливо, але не точно. Наприклад:

- `I might go to the party if I finish my work.` — можливо, я піду, але я не впевнений.

- `She might call you later.` — можливо, вона подзвонить, але це не точно.
