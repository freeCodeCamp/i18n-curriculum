# --description--

Hay exactamente diez formas de seleccionar tres de cinco, 12345:

<div style='text-align: center;'>123, 124, 125, 134, 135, 145, 234, 235, 245, and 345</div>

En combinatoria, usamos la notación, $\\displaystyle \\binom 5 3 = 10$

En general, $\\displaystyle \\binom n r = \\dfrac{n!}{r!(n-r)!}$, donde $r \\le n$, $n! = n \\times (n-1) \\times ... \\times 3 \\times 2 \\times 1$, y $0! = 1$.

No es hasta $n = 23$ que un valor supera el millón: $\\displaystyle \\binom {23} {10} = 1144066$.

¿Cuántos valores, no necesariamente distintos, de $\\displaystyle \\binom n r$ para $1 \\le n \\le 100$, son mayores que un millón?

```js
function combinatoricSelections(limit) {
    const factorial = n =>
        Array.apply(null, { length: n })
            .map((_, i) => i + 1)
            .reduce((p, c) => p * c, 1);

    let result = 0;
    const nMax = 100;

    for (let n = 1; n <= nMax; n++) {
        for (let r = 0; r <= n; r++) {
            if (factorial(n) / (factorial(r) * factorial(n - r)) >= limit)
                result++;
        }
    }

    return result;
}
```
