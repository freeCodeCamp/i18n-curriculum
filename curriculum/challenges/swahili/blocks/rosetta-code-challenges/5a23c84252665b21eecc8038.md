# --description--

Subleq ni mfano wa Kompyuta yenye Seti ya Maelekezo Moja (OISC).

Imepewa jina lake kutokana na maelekezo yake pekee, ambayo ni **SU**btract na **B**ranch ikiwa **L**ess kuliko au **EQ**ual kwa sifuri.

Zoezi lako ni kuunda tafsiri inayofanana na mashine kama hiyo.

Kumbukumbu ya mashine ina safu ya data ya nambari kamili zilizo na alama. Ukubwa wowote wa neno unaofaa ni sawa, lakini kumbukumbu lazima iweze kuhifadhi nambari hasi pamoja na nambari chanya.

Utekelezaji huanza na kiashiria cha maelekezo kikiwa kimeelekezwa kwenye neno la kwanza, ambalo ni anwani 0. Hufuatwa kama ifuatavyo:

<ol>
  <li>Acha A, B, na C wawe thamani zilizohifadhiwa katika maneno matatu mfululizo ya kumbukumbu kuanzia kiashiria cha maelekezo.</li>
  <li>Endesha kiashiria cha maelekezo maneno 3 mbele ili kiweze kuelekeza anwani inayofuata ile yenye C.</li>
  <li>Kama A ni -1, basi herufi inasomwa kutoka kwenye ingizo la kawaida na nambari yake ya msimbo huhifadhiwa katika anwani inayotolewa na B. C haifanyi kazi.</li>
  <li>Kama B ni -1, basi nambari iliyoko katika anwani inayotolewa na A hutafsiriwa kama nambari ya msimbo na herufi inayolingana hutolewa. C tena haifanyi kazi.</li>
  <li>Vinginevyo, A na B zote hutumiwa kama anwani za maeneo ya kumbukumbu. Nambari iliyoko katika anwani inayotolewa na A inachukuliwa kutoka kwa nambari iliyoko katika anwani inayotolewa na B (na matokeo kuhifadhiwa tena katika anwani B). Kama matokeo ni sifuri au hasi, thamani ya C inakuwa kiashiria kipya cha maelekezo.</li>
  <li>Kama kiashiria cha maelekezo kinakuwa hasi, utekelezaji unasimama.</li>
</ol>

Anwani nyingine hasi isipokuwa -1 zinaweza kutendewa sawa na -1, au kusababisha kosa, kulingana na unavyopendelea.

Suluhisho lako linapaswa kupokea programu itakayotekelezwa kwenye mashine, tofauti na ingizo linalotolewa kwa programu yenyewe.

Programu hii inapaswa kuwa msimbo wa mashine wa subleq wa asili - nambari za desimali zilizoachwa kwa nafasi, bila majina ya alama au nyongeza nyingine za ngazi ya mkusanyiko, zitakazopakiwa kwenye kumbukumbu kuanzia anwani 0. Onyesha matokeo ya suluhisho lako ukipatia programu hii ya "Hello, world!" ingizo. (Kumbuka kuwa mfano huu unadhani ASCII au toleo lake, kama seti yoyote ya herufi za Latin-N au Unicode. Unaweza kuibadilisha kuwa seti nyingine ya herufi ikiwa utekelezaji wako uko katika mazingira yasiyoendana na ASCII.)

<pre>15 17 -1 17 -1 -1 16 1 -1 16 3 -1 15 15 0 0 -1 72 101 108 108 111 44 32 119 111 114 108 100 33 10 0</pre>

Ambayo inalingana na kitu kama hiki katika lugha ya mkusanyiko ya mfano:

    zero, message, -1

    neg1, start+1, -1

    zero, zero, start
zero: 0
neg1: -1

function Subleq(mem) {
  var out = '';
  var instructionPointer = 0;
  do {
    var a = mem[instructionPointer];
    var b = mem[instructionPointer + 1];
    if (a === -1) {
    } else if (b === -1) {
      out += String.fromCharCode(mem[a]);
    } else {
      mem[b] -= mem[a];
      if (mem[b] < 1) {
        instructionPointer = mem[instructionPointer + 2];
        continue;
      }
    }
    instructionPointer += 3;
  } while (instructionPointer >= 0);

  return out;
}
