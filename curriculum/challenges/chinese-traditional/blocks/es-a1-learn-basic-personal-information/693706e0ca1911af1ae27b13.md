# --description--

在此任務中，你將聽到 Mateo 在被感謝後禮貌地回應。他的用語是在西班牙語中承認他人感謝的標準方式。

還有其他說法可以表達「不客氣」。以下是一些範例：

- `De nada`－不客氣。（正式與非正式）
- `No hay de qué`－不必客氣。（較正式或傳統）
- `Con gusto`－我的榮幸。（拉丁美洲常用）
