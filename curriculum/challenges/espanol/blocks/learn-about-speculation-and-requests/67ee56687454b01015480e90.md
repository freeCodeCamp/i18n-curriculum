# --explanation--

Un `for loop` es útil cuando sabes cuántas veces debe ejecutarse el ciclo. Por ejemplo:

- `A for loop is useful for going through a list.` - Puedes usarlo para revisar cada elemento en una lista.

- `The for loop stops when it reaches the last item.` - El ciclo termina cuando no hay más elementos.

Un `while loop` es mejor cuando el número de repeticiones depende de una condición. Por ejemplo:

- Un ciclo while se ejecuta hasta que una condición es falsa. - El ciclo continúa mientras la regla sea verdadera.

- `Be careful, a while loop can run forever!` - Si no la detienes, el ciclo no terminará.
