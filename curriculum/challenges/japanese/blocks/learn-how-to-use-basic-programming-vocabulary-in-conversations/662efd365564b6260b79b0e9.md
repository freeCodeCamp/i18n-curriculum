# --description--

`loop` は、ある条件が満たされるまで繰り返し実行される命令のシーケンスです。`Loops` は同じコードを効率的に複数回実行するために使われます。一般的な`loops`の種類には`for`ループや`while`ループがあります。

例えば、`for loop`: `for (int i = 0; i < 5; i++) { /* code to be executed */ }`

この例では、中括弧内のコード`{ }`が条件が真である限り繰り返し実行されます。
