# --description--

`Object.keys()` 메서드를 사용해 객체에 저장된 모든 키를 포함하는 배열을 생성할 수도 있습니다. 이 메서드는 객체를 인수로 받아 객체의 각 속성을 나타내는 문자열 배열을 반환합니다. 배열의 항목 순서는 특정하지 않습니다.

# --instructions--

`getArrayOfUsers` 함수를 완성하여 인수로 받은 객체의 모든 속성을 포함하는 배열을 반환하게 하세요.

# --hints--

`users` 객체에는 `Alan`, `Jeff`, `Sarah`, `Ryan` 키만 포함되어야 합니다.

```js
assert(
  'Alan' in users &&
    'Jeff' in users &&
    'Sarah' in users &&
    'Ryan' in users &&
    Object.keys(users).length === 4
);
```

`getArrayOfUsers` 함수는 `users` 객체의 모든 키를 포함하는 배열을 반환해야 합니다.

```js
assert(
  (function () {
    users.Sam = {};
    users.Lewis = {};
    let R = getArrayOfUsers(users);
    return (
      R.indexOf('Alan') !== -1 &&
      R.indexOf('Jeff') !== -1 &&
      R.indexOf('Sarah') !== -1 &&
      R.indexOf('Ryan') !== -1 &&
      R.indexOf('Sam') !== -1 &&
      R.indexOf('Lewis') !== -1
    );
  })() === true
);
```
