# --description--

$12n$ 名音乐家参加音乐节。第一天，他们组成 $3n$ 个四重奏并全天练习。

这是一场灾难。 在一天结束时，所有音乐家都决定再也不会同意与他们四重奏的任何成员一起玩。

在第 2 天，他们组成了 $4n$ 个三人组，每个音乐家都避免与之前的四重奏伙伴同组。

设 $f(12n)$ 为在 $12n$ 名音乐家中组织三重奏的方式数。

给定 $f(12) = 576$ 和 $f(24)\bmod 1\\,000\\,000\\,007 = 509\\,089\\,824$。

查找 $f(600)\bmod 1\\,000\\,000\\,007$。
