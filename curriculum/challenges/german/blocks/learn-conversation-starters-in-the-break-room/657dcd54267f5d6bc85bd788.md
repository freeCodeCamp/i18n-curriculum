# --description--

Wenn Sie Geld abheben möchten und nach einem Automaten suchen, der Bargeld ausgibt, können Sie fragen, `Is there an ATM nearby?`

Ein `ATM`, was für `Automated Teller Machine` steht, ist ein Gerät, an dem Sie mit einer Bankkarte Bargeld abheben können.

Das Wort `nearby` bedeutet, dass es nicht weit von Ihnen entfernt ist, vielleicht nur ein kurzer Fußweg. Es ist ein praktisches Wort, wenn Sie an einem neuen Ort schnell etwas finden müssen.
