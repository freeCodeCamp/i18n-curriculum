# --description--

Njia nyingine muhimu ya safu ya data ni njia ya `filter()`. Njia ya `filter()` huunda safu mpya yenye vipengele vyote vinavyopita mtihani uliofanywa na kitendakazi kilichotolewa. Sintaksia ya njia ya `filter()` ni kama ifuatavyo:

```javascript
const numbers = [2, 5, 6, 1, 9, -1]

const newNumbers = numbers.filter((number) => {
  return number > 2;
});

console.log(newNumbers); // Output: [5, 6, 9]
```

Njia ya `filter()` huunda safu mpya yenye vipengele vyote vinavyozidi 2. Njia ya `filter()` haibadilishi safu ya data ya awali.
