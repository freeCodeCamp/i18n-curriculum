# --description--

`promover la ética` significa fomentar y apoyar principios morales y estándares. Por ejemplo:

`The manager promotes ethics by enforcing transparent reporting.` - Esto significa que el manager apoya un comportamiento bueno y honesto asegurándose de que los reportes sean claros y veraces.
