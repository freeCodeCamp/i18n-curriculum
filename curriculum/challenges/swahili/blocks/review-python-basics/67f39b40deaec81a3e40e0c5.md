- **Kutangaza Vigezo**: Ili kutangaza kigezo, unaanza na jina la kigezo likifuatiwa na opereta ya mgawo (`=`) kisha thamani. Hii inaweza kuwa nambari, mfuatano wa herufi, thamani ya kweli au siyo kweli, n.k. Hapa kuna mifano kadhaa:

```py
name = 'John Doe'
```

## Aina za Data za Kawaida katika Python

- **Utangulizi**: Python ni lugha inayobadilika aina zake kwa wakati wa utekelezaji kama JavaScript, maana yake huna haja ya kutangaza aina za vigezo waziwazi. Lugha inajua aina ya kigezo kulingana na kile unachoweka kwenye kigezo.
- **Nambari Kamili**: Nambari isiyo na sehemu za desimali:

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **Nambari ya Desimali**: Nambari yenye sehemu za desimali:

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **Mfuatano wa Herufi**: Mfuatano wa herufi uliowekwa ndani ya alama za nukuu:

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **Thamani ya Kweli au Siyo Kweli**: Thamani inayowakilisha `True` au `False`:

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **Seti**: Mkusanyiko usio na mpangilio wa vipengele vya kipekee:

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **Kamusi**: Mkusanyiko wa jozi za ufunguo-thamani, zikiwa zimefungwa ndani ya mabano ya mviringo:

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **Tuple**: Mkusanyiko uliopangwa na usiobadilika, umefungwa ndani ya mabano ya kawaida:

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **Mfululizo wa Nambari**: Mfululizo wa nambari, mara nyingi hutumika katika mizunguko:

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **Orodha**: Mkusanyiko uliopangwa wa vipengele unaounga mkono aina tofauti za data:

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**: Thamani maalum inayowakilisha kutokuwepo kwa thamani:

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## Aina Zisizobadilika na Zinazobadilika

- **Aina Zisizobadilika**: Aina hizi haziwezi kubadilika mara baada ya kutangazwa, ingawa unaweza kuelekeza vigezo vyake kwa kitu kipya, kinachoitwa mgawo upya. Zinajumuisha nambari kamili, nambari ya desimali, nambari tata, thamani ya kweli au siyo kweli, mfuatano wa herufi, tuple, mfululizo, na `None`.
- **Aina Zinazobadilika**: Aina hizi zinaweza kubadilika mara baada ya kutangazwa. Unaweza kuongeza, kuondoa, au kusahihisha vipengele vyake. Zinajumuisha aina za makusanyo kama orodha, seti, na kamusi.
- **Kitendakazi cha `type()`**: Ili kuona aina ya kigezo, unaweza kutumia kitendakazi cha `type()` kama ifuatavyo:

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **Kitendakazi cha `isinstance()`**: Hiki hutumika kuangalia kama kigezo kinaendana na aina fulani ya data:

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## Kufanya kazi na Mfuatano wa Herufi

- **Ufafanuzi**: Kama unavyokumbuka kutoka JavaScript, mfuatano wa herufi ni zisizobadilika, maana huwezi kuzibadilisha baada ya kuundwa. Katika Python, unaweza kutumia alama za nukuu moja au maradufu. Inashauriwa kuchagua kanuni moja na kuifuata:

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **Kupata Herufi kutoka kwa Mfuatano wa Herufi**: Unaweza kupata herufi kutoka kwa mfuatano wa herufi kwa kutumia uandishi wa mabano kama ifuatavyo:

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **Kuepuka Mfuatano wa Herufi**: Unaweza kutumia mkwaju wa nyuma (`\`) ikiwa mfuatano wako wa herufi una alama za nukuu kama ifuatavyo:

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **Kuunganisha Mfuatano wa Herufi**: Ili kuunganisha mfuatano wa herufi, unaweza kutumia opereta ya `+` kama ifuatavyo:

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

Njia nyingine ya kuunganisha mfuatano wa herufi ni kwa kutumia opereta ya `+=`. Hii hutumika kufanya kuunganisha na mgawo katika hatua moja kama ifuatavyo:

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**: Hii ni kifupi cha mfuatano wa herufi uliopangwa. Inakuwezesha kufanya uingizaji wa thamani na pia kuunganisha kwa sintaksia fupi na inayosomeka:

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **Kuchukua Sehemu ya Mfuatano wa Herufi**: Hii ni pale unapoweza kutoa sehemu za mfuatano wa herufi. Hapa ni sintaksia ya msingi:

```py
str[start:stop:step]
```

Nambari ya kuanzia inaonyesha kielezo ambacho uchimbaji unapaswa kuanza. Nambari ya kusimama ni mahali ambapo sehemu inapaswa kuisha. Nambari hii haijumuishi. Nambari ya hatua inaonyesha kipindi cha kuongezeka kwa uchimbaji. Hapa kuna mifano:

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **Kupata Urefu wa Mfuatano wa Herufi**: Kitendakazi cha `len()` hutumika kurudisha idadi ya herufi katika mfuatano:

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## Kufanya kazi na opereta ya `in`

- **Opereta ya `in`**: Hii hurudisha thamani ya kweli au siyo kweli inayobainisha kama herufi au herufi zipo katika mfuatano wa herufi au la:

```py
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

## Njia za Kawaida za Mfuatano wa Herufi

- **`str.upper()`**: Hii hurudisha mfuatano mpya wa herufi wenye herufi zote kubadilishwa kuwa herufi kubwa:

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`str.lower()`**: Hii hurudisha mfuatano mpya wa herufi wenye herufi zote kubadilishwa kuwa herufi ndogo:

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`str.strip()`**: Hii hurudisha nakala ya mfuatano wa herufi yenye kuondoa herufi za mbele na nyuma zilizobainishwa (kama hakuna hoja imetolewa, huondoa nafasi za tupu za mbele na nyuma).

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**: Hii hurudisha mfuatano mpya wa herufi ambapo matukio yote ya mfuatano wa zamani yamebadilishwa na mfuatano mpya.

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**: Hii hutumika kugawanya mfuatano wa herufi kuwa orodha kwa kutumia kitenzi kilichobainishwa. Kitenzi ni mfuatano wa herufi unaoelezea mahali pa kugawanya.

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**: Hii hutumika kuunganisha vipengele vya mkusanyiko unaoweza kurudiwa kuwa mfuatano wa herufi kwa kutumia kitenzi. Mkusanyiko unaoweza kurudiwa ni mkusanyiko wa vipengele unaoweza kurudiwa kama orodha, mfuatano wa herufi au tuple.

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`str.startswith(prefix)`**: Hii hurudisha thamani ya kweli au siyo kweli ikionyesha kama mfuatano wa herufi unaanza na kiambishi awali kilichobainishwa:

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`str.endswith(suffix)`**: Hii hurudisha thamani ya kweli au siyo kweli ikionyesha kama mfuatano wa herufi unaisha na kiambishi cha mwisho kilichobainishwa:

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`str.find()`**: Hii hurudisha kielezo cha tukio la kwanza la mfuatano mdogo. Ikiwa haipatikani, `-1` hurudishwa:

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`str.count(substring)`**: Hii huhesabu mara ngapi mfuatano mdogo unaonekana katika mfuatano wa herufi:

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`str.capitalize()`**: Hii hurudisha mfuatano mpya wa herufi wenye herufi ya kwanza kuwa kubwa na zingine kuwa ndogo:

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`str.isupper()`**: Hii hurudisha `True` ikiwa herufi zote katika mfuatano ni kubwa na `False` ikiwa sivyo:

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`str.islower()`**: Hii hurudisha `True` ikiwa herufi zote katika mfuatano ni ndogo na `False` ikiwa sivyo:

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`str.title()`**: Hii hurudisha mfuatano mpya wa herufi wenye herufi ya kwanza ya kila neno kuwa kubwa:

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`str.maketrans()`**: Njia hii hutumika kuunda jedwali la ulinganifu wa herufi 1 kwa 1 kwa ajili ya tafsiri. Mara nyingi hutumika pamoja na njia ya `translate()` ambayo inatumia jedwali hilo kwenye mfuatano wa herufi na kurudisha matokeo yaliyotafsiriwa.

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## Operesheni za Kawaida zinazotumika na Nambari Kamili na Nambari za Desimali

- **Operesheni za Hisabati za Msingi**: Katika Python, unaweza kufanya operesheni za hisabati za msingi na nambari kamili na nambari za desimali ikiwa ni pamoja na kuongeza, kutoa, kuzidisha na kugawanya:

```py
int_1 = 56
int_2 = 12
float_1 = 5.4
float_2 = 12.0

# Addition

print('Integer Addition:', int_1 + int_2) # Integer Addition: 68
print('Float Addition:', float_1 + float_2) # Float Addition: 17.4

# Subtraction

print('Int Subtraction:', int_1 - int_2) # Int Subtraction: 44
print('Float Subtraction:',  float_2 - float_1) # Float Subtraction: 6.6

# Multiplication

print('Int Multiplication:', int_1 * int_2) # Int Multiplication: 672
print('Float Multiplication:', float_2 * float_1) # Float Multiplication: 64.80000000000001

# Division

print('Division:', int_1 / int_2) # Division: 4.666666666666667
print('Float Division:', float_2 / float_1) # Float Division: 2.222222222222222
```

Unapoongeza nambari ya desimali na nambari kamili, matokeo yatabadilishwa kuwa nambari ya desimali kama ifuatavyo:

```py
int_1 = 56
float_1 = 5.4

print(int_1 + float_1) # 61.4
```

- **Opereta ya Modulo (`%`)**: Hii hurudisha mabaki wakati nambari ikigawanywa na nambari nyingine:

```py
int_1 = 56
int_2 = 12

print(int_1 % int_2) # 8
```

- **Mgawanyo wa Sakafu (`//`)**: Opereta hii hutumika kugawanya nambari mbili na kuzungusha matokeo chini hadi nambari kamili inayokaribu:

```py

```

- **Opereta ya Nguvu (`**`)**: Opereta hii hutumika kuinua nambari kwa nguvu ya nyingine:

```py
int_1 = 4
int_2 = 2

print(int_1 ** int_2) # 16
```

- **Kitendakazi cha `float()`**: Unaweza kutumia kitendakazi hiki kubadilisha nambari kamili kuwa nambari ya desimali.

```py
num = 4

print(float(num)) # 4.0
```

- **Kitendakazi cha `int()`**: Unaweza kutumia kitendakazi hiki kubadilisha nambari ya desimali kuwa nambari kamili.

```py
num = 4.0

print(int(num)) # 4
```

- **Kitendakazi cha `round()`**: Hii hutumika kuzungusha nambari hadi nambari kamili inayokaribu:

```py
num_1 = 3.4
num_2 = 7.7

print(round(num_1)) # 3
print(round(num_2)) # 8
```

- **Kitendakazi cha `abs()`**: Hii hutumika kurudisha thamani kamili ya nambari:

```py
num = -13

print(abs(num)) # 13
```

- **Kitendakazi cha `pow()`**: Hii hutumika kuinua nambari kwa nguvu ya nyingine:


## Augmented Assignments

- **Ufafanuzi**: Mgawo wa thamani ulioboreshwa huunganisha operesheni ya binary na mgawo katika hatua moja. Huchukua kigezo, hutumia operesheni kwa thamani nyingine, na kuhifadhi matokeo tena kwenye kigezo hicho hicho.

```py
count = 14

product = 65

print(product) # 455

price = 100

print(price) # 25.0

total_pages = 23

# Modulo assignment 
bits = 35
bits %= 2

print(bits) # 1

# Exponentiation assignment 
power = 2
power **= 3

print(power) # 8
```

Kuna opereta nyingine za mgawo ulioboreshwa pia, kama zile za opereta za bitwise. Zinajumuisha `&=`, `^=`, `>>=`, na `<<=`.

## Kufanya kazi na Vitendakazi

- **Ufafanuzi**: Vitendakazi ni vipande vya msimbo vinavyoweza kutumika tena vinavyopokea ingizo (hoja) na kurudisha matokeo. Ili kuita kitendakazi, unahitaji kurejea jina la kitendakazi likifuatiwa na mabano ya kawaida:

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call
print(result) # 7
```

Kama kitendakazi hakirudishi thamani waziwazi, basi thamani ya default ya return ni `None`:

```py
def greet():
    print('hello') 

result = greet() # hello
print(result) # None
```

Unaweza pia kutoa thamani za default kwa vigezo kama ifuatavyo:

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

Kama utaita kitendakazi bila idadi sahihi ya hoja, utapata `TypeError`:

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

## Wigo katika Python

- **Wigo wa Ndani**: Hii ni pale kigezo kinapotangazwa ndani ya kitendakazi au darasa na kinaweza kufikiliwa tu ndani ya kitendakazi au darasa hilo.

```py
def my_func():
    num = 10
    print(num)
```

- **Wigo wa Kuambatana**: Hii ni pale kitendakazi kilichopangwa ndani ya kitendakazi kingine kinapoweza kufikilia vigezo vya kitendakazi cha juu.

```py
def outer_func():
    msg = 'Hello there!'
    
    def inner_func():
        print(msg)
    inner_func()

print(outer_func()) # Hello there!
```

- **Wigo wa Ulimwengu**: Hii inahusu vigezo vilivyotangazwa nje ya vitendakazi au madarasa vyenye upatikanaji kutoka sehemu yoyote ya programu.

```py
tax = 0.70 

def get_total(subtotal):
    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **Wigo wa Iliyojengwa Ndani**: Majina yaliyohifadhiwa katika Python kwa ajili ya vitendakazi vilivyotangazwa awali, moduli, maneno muhimu, na vitu.

```py
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```

## Comparison Operators

```py
print(3 == 4) # False
```

```py
print(3 != 4) # True
```

```py
print(3 > 4) # False
```

- **Ndogo zaidi kwa ukali (`<`)**: Huu huangalia kama thamani moja ni ndogo zaidi kuliko nyingine:

```py
print(3 < 4) # True
```

```py
print(3 >= 4) # False
```

```py
print(3 <= 4) # True
```

## Kufanya kazi na Matamko ya `if`, `elif` na `else`

- **Matamko ya `if`**: Haya ni masharti yanayotumika kuamua kama kitu ni kweli au si kweli. Ikiwa sharti linathibitishwa kuwa `True`, basi sehemu hiyo ya msimbo itaendeshwa.

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **Kifungu cha `elif`**: Haya ni masharti yanayofuata tamko la `if`. Kifungu cha `elif` hutaenda tu ikiwa masharti yote ya awali yanathibitishwa kuwa `False` na sharti lake linathibitishwa kuwa `True`.

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **Kifungu cha `else`**: Hiki kitaendeshwa ikiwa hakuna masharti mengine yanayothibitishwa kuwa `True`.

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

Unaweza pia kutumia matamko ya `if` yaliyopangwa ndani kama ifuatavyo:

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## Thamani za Kweli na Uongo

- **Ufafanuzi**: Katika Python, kila thamani ina thamani ya kweli au siyo kweli ya asili, au hisia iliyojengwa ndani ya kama inapaswa kutendewa kama `True` au `False` katika muktadha wa mantiki. Thamani nyingi zinachukuliwa kuwa za kweli, yaani, zinathibitishwa kuwa `True` katika muktadha wa mantiki. Nyingine ni za uongo, zinazoangaliwa kuwa `False`. Hapa kuna mifano ya thamani za uongo:

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

Thamani nyingine kama nambari zisizo sifuri, na mfuatano wa herufi usio tupu ni za kweli.

## Kufanya kazi na Kitendakazi cha `bool()`

- **Ufafanuzi**: Ikiwa unataka kuangalia kama thamani ni ya kweli au uongo, unaweza kutumia kitendakazi kilicho ndani cha `bool()`. Hukiibadilisha thamani kuwa sawa na thamani ya kweli au siyo kweli na kurudisha `True` kwa thamani za kweli na `False` kwa thamani za uongo. Hapa kuna mifano michache:

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## Opereta za Thamani ya Kweli au Siyo Kweli na Kukata Njia

- **Ufafanuzi**: Hizi ni opereta maalum zinazokuwezesha kuunganisha usemi wa kihesabu zaidi kuunda mantiki tata ya maamuzi katika msimbo wako. Kuna opereta tatu za thamani ya kweli au siyo kweli katika Python: `and`, `or`, na `not`.
- **Opereta ya `and`**: Opereta hii huchukua operands mbili na hurudisha operand ya kwanza ikiwa ni ya uongo, vinginevyo hurudisha operand ya pili. Operands zote mbili lazima ziwe za kweli ili usemi uzalishwe kuwa wa kweli.

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

Unaweza pia kutumia opereta ya `and` katika masharti kama ifuatavyo:

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **Opereta ya `or`**: Opereta hii hurudisha operand ya kwanza ikiwa ni ya kweli, vinginevyo hurudisha operand ya pili. Usemi wa `or` hutokea kuwa wa kweli ikiwa angalau operand moja ni ya kweli. Hapa kuna mfano:

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

Kama ilivyo kwa opereta ya `and`, unaweza kutumia opereta ya `or` katika masharti kama ifuatavyo:

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **Kukata Njia**: Opereta za `and` na `or` zinajulikana kama opereta za kukata njia. Kukata njia maana yake ni Python huangalia thamani kutoka kushoto kwenda kulia na kusitisha mara tu inapotambua matokeo ya mwisho.
- **Opereta ya `not`**: Opereta hii huchukua operand moja na kubadilisha thamani yake ya kweli au siyo kweli. Hubadilisha thamani za kweli kuwa `False` na thamani za uongo kuwa `True`. Tofauti na opereta zilizopita tulizozitazama, `not` daima hurudisha `True` au `False`. Hapa kuna mifano:

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

Hapa kuna mfano wa opereta ya `not` katika sharti:

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```
