# --description--

`class` 是一個基本的概念，特別是在 `object-oriented programming` 中。它就像是用來創建物件的藍圖。`class` 定義了從該 `class` 創建的物件可以擁有的屬性（attributes）和行為（`methods`）。

例如，`Person` 類別可能有像 `name` 和 `age` 這樣的屬性，以及像 `talk()` 和 `walk()` 這樣的行為。

`method` 是與物件相關的函式（程式）。`Methods` 定義了從 `class` 產生的物件可以執行的行為或動作。它們就像告訴物件該做什麼的指令。

例如，在 `Calculator class` 中，`method` 可能是 `add(number1, number2)`，它會將兩個數字相加。
