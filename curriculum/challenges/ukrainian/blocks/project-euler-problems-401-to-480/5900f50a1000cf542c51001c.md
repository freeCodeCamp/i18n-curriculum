# --description--

Говорять, що додатне число з $d$ цифр (без провідних нулів) є числом з одним дочірнім елементом, якщо рівно один із його підрядків ділиться на $d$.

Наприклад, 5671 — це 4-цифрове число з одним дочірнім елементом. Серед усіх його підрядків 5, 6, 7, 1, 56, 67, 71, 567, 671 і 5671 лише 56 ділиться на 4.

Аналогічно, 104 — це 3-цифрове число з одним дочірнім елементом, бо лише 0 ділиться на 3. 1132451 — це 7-цифрове число з одним дочірнім елементом, бо лише 245 ділиться на 7.

Нехай $F(N)$ — кількість чисел з одним дочірнім елементом, менших за $N$. Можна перевірити, що $F(10) = 9$, $F({10}^3) = 389$ і $F({10}^7) = 277\\,674$.

Знайдіть $F({10}^{19})$.
