# --description--

Маніпулюючи різними селекторами та властивостями, ви можете створювати цікаві форми. Однією з простіших для спроби є форма місяця. Для цього завдання потрібно працювати з властивістю `box-shadow`, яка задає тінь елемента, а також з властивістю `border-radius`, що контролює округлість кутів елемента.

Ви створите круглий прозорий об’єкт із чіткою тінню, яка трохи зміщена вбік — тінь фактично буде формою місяця, яку ви бачите.

Щоб створити круглий об’єкт, властивість `border-radius` має бути встановлена на значення 50%.

Ви, можливо, пам’ятаєте з попереднього завдання, що властивість `box-shadow` приймає значення для `offset-x`, `offset-y`, `blur-radius`, `spread-radius` і `color` у такому порядку. Значення `blur-radius` і `spread-radius` є необов’язковими.

# --instructions--

Маніпулюйте квадратним елементом у редакторі, щоб створити форму місяця. Спочатку змініть `background-color` на `transparent`, потім встановіть властивість `border-radius` на 50%, щоб зробити круглу форму. Нарешті, змініть властивість `box-shadow`, щоб задати `offset-x` на 25px, `offset-y` на 10px, `blur-radius` на 0, `spread-radius` на 0 і `color` на `blue`.

# --hints--

Значення властивості `background-color` має бути встановлене на `transparent`.

```js
assert.match(code,/background-color:\s*?transparent;/gi);
```

Значення властивості `border-radius` має бути встановлене на `50%`.

```js
assert.match(code,/border-radius:\s*?50%;/gi);
```

Значення властивості `box-shadow` має бути встановлене на 25px для `offset-x`, 10px для `offset-y`, 0 для `blur-radius`, 0 для `spread-radius` і нарешті `blue` для `color`.

```js
assert.match(code,/box-shadow:\s*?25px\s+?10px\s+?0(px)?\s+?0(px)?\s+?blue\s*?;/gi);
```
