# --description--

Agora que você conhece os números dos 30, vamos para os 40.

Em espanhol, o número 40 é chamado de `cuarenta`.

Números de 41 a 49 são formados usando a palavra das dezenas `cuarenta` (40), seguida da conjunção `y` ("and") e depois o dígito das unidades (1-9).

| Number | Spanish |
| :--- | :--- |
| 40 | `Cuarenta` |
| 41 | `Cuarenta y uno` |
| 42 | `Cuarenta y dos` |
| 43 | `Cuarenta y tres` |
| 44 | `Cuarenta y cuatro` |
| 45 | `Cuarenta y cinco` |
| 46 | `Cuarenta y seis` |
| 47 | `Cuarenta y siete` |
| 48 | `Cuarenta y ocho` |
| 49 | `Cuarenta y nueve` |

O número `cuarenta y uno` (41) é único nesta faixa porque se torna `cuarenta y un` antes de um substantivo masculino. Por exemplo:

`Cuarenta y un años` - Quarenta e um anos.
