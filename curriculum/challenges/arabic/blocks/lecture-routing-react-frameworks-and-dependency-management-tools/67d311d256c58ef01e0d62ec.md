# --description--

حتى هذه اللحظة، كنت تستخدم React لبناء واجهات المستخدم. إذا احتجت إلى ميزات إضافية مثل التوجيه، كان عليك استيراد مكتبة طرف ثالث مثل React Router لتتمكن من التنقل بين العروض المختلفة.

لكن ماذا يحدث عندما تحتاج إلى بناء تطبيق ويب متكامل؟ يمكنك استخدام React للواجهة الأمامية واستخدام Node وExpress للمنطق الخلفي إذا أردت الالتزام بجافاسكريبت فقط. أو يمكنك استخدام لغات أخرى مثل Go أو Python أو Java للواجهة الخلفية.

بينما كل هذه خيارات صالحة، هناك أوقات قد تفضل فيها استخدام إطار عمل React بدلاً من ذلك. توفر أُطُر عمل React ميزات مثل التوجيه، تحسين الصور، جلب البيانات، التوثيق والمزيد. هذا يعني أنك قد لا تحتاج إلى إعداد تطبيقات منفصلة للواجهة الأمامية والخلفية في بعض الحالات.

لنلق نظرة أقرب على Next.js، وهو إطار عمل React شائع. من الميزات الرئيسية لـ Next.js هو الموجِّه المعتمد على نظام الملفات. يشمل هذا النظام دعم المسارات الديناميكية، المسارات المتوازية، معالجات المسارات، إعادة التوجيه، التعريب والمزيد.

إليك مثال على إنشاء معالِج طلب مخصص:

```js
export async function GET() {
  const res = await fetch("https://example-api.com");
  const data = await res.json();

  return Response.json({ data });
}
```

يمكنك تعريف معالجات المسارات مثل طلبات GET أو POST في ملف يسمى `route.js` داخل مجلد `app/api`.

ميزة أخرى في Next.js هي تحسينات الصور والخطوط التلقائية.

إليك مثال على العمل مع مكوِّن `Image` داخل ملف `page.js`:

```jsx
import Image from "next/image";

export default function Page() {
  return (
    <Image src="link-to-image-goes-here" alt="descriptive-title-goes-here" />
  );
}
```

يمتد مكوِّن `Image` عنصر HTML الأصلي `img` ويسمح بتحميل صفحات أسرع وتحسينات في الحجم. هذا يعني أن الصور ستُحمّل فقط عند دخولها إطار العرض، وسيقوم مكوِّن `Image` تلقائيًا بتقديم الصور بالحجم المناسب لكل جهاز.

بينما هذه مجرد بعض ميزات Next.js، هناك العديد من الميزات الأخرى التي يمكنك استخدامها لبناء تطبيقات ويب متكاملة قوية. هناك أيضًا أُطُر عمل React أخرى مثل Remix توفر نفس القدرة على بناء تطبيقات ويب متكاملة حديثة.

على الرغم من أن مشهد مكتبات وأُطُر عمل جافاسكريبت يتغير باستمرار، من المهم أن تكون على دراية بالخيارات المتاحة لك وتتعلم مزايا وعيوب كل إطار عمل.
