# --description--

JavaScriptには多くの<dfn>比較演算子</dfn>があります。これらの演算子はすべてブール値`true`または`false`を返します。

最も基本的な演算子は等値演算子`==`です。等値演算子は2つの値を比較し、それらが等しい場合は`true`を、そうでない場合は`false`を返します。等値は代入（`=`）とは異なり、代入は演算子の右側の値を左側の変数に代入します。

```js
function equalityTest(myVal) {
  if (myVal == 10) {
    return "Equal";
  }
  return "Not Equal";
}
```

もし`myVal`が`10`と等しい場合、等値演算子は`true`を返すため、波括弧内のコードが実行され、関数は`Equal`を返します。そうでなければ、関数は`Not Equal`を返します。JavaScriptが異なる<dfn>データ型</dfn>（例えば`numbers`と`strings`）を比較する場合、片方の型をもう片方に変換する必要があります。これを型変換と呼びます。しかし一度変換されると、次のように比較できます。

```js
1   ==  1  // true
1   ==  2  // false
1   == '1' // true
"3" ==  3  // true
```
