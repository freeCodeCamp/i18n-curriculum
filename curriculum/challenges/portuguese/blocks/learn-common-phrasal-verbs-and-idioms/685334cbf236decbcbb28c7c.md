# --explanation--

Um `draft` é uma versão inicial de um documento ou plano que pode ser editada ou revisada.Por exemplo:

`I wrote a draft of the email before sending the final version.` – Isso significa que a pessoa escreveu uma versão inicial do email.

`To run something by someone` é usado para mostrar ou explicar algo a alguém para obter feedback ou aprovação.Por exemplo:

`Can I run this idea by you before we present it?` – Isso significa que você está pedindo a opinião de alguém sobre isso.

`To roll something out` significa lançar ou introduzir oficialmente algo para um grupo maior.Por exemplo:

`They will roll out the new software update next week.` – Isso significa que a nova versão do software será lançada na próxima semana.

O sufixo `-wide` significa "através de todo" algo.Em `company-wide`, significa algo que se aplica a toda a empresa.Outro exemplo:

`There was a system-wide outage yesterday.` – Isso significa que o problema afetou todo o sistema.
