# --description--

Python bietet eine Reihe eingebauter Methoden, die die Arbeit mit Strings sehr einfach machen. Dazu gehören unter anderem die folgenden:

- `upper()`: Gibt einen neuen String zurück, bei dem alle Zeichen in Großbuchstaben umgewandelt sind.
    

```python
my_str = 'hello world'

uppercase_my_str = my_str.upper()
print(uppercase_my_str)  # HELLO WORLD
```

- `lower()`: Gibt einen neuen String zurück, bei dem alle Zeichen in Kleinbuchstaben umgewandelt sind.
    

```python
my_str = 'Hello World'

lowercase_my_str = my_str.lower()
print(lowercase_my_str)  # hello world
```

- `strip()`: Gibt einen neuen String zurück, bei dem die angegebenen führenden und abschließenden Zeichen entfernt wurden. Wenn kein Argument übergeben wird, entfernt es führende und abschließende Leerzeichen.
    

```python
my_str = '  hello world  '

trimmed_my_str = my_str.strip()
print(trimmed_my_str)  # "hello world"
```

- `replace(old, new)`: Gibt einen neuen String zurück, bei dem alle Vorkommen von `old` durch `new` ersetzt wurden.
    

```python
my_str = 'hello world'

replaced_my_str = my_str.replace('hello', 'hi')
print(replaced_my_str)  # hi world
```

- `split(separator)`: Teilt einen String an einem angegebenen Trennzeichen in eine Liste von Strings auf. Wenn kein Trennzeichen angegeben ist, wird an Leerzeichen geteilt.
    

```python
my_str = 'hello world'

split_words = my_str.split()
print(split_words)  # ['hello', 'world']
```

- `join(iterable)`: Verbindet Elemente eines iterierbaren Objekts mit einem Trennzeichen zu einem String.
    

```python
my_list = ['hello', 'world']

joined_my_str = ' '.join(my_list)
print(joined_my_str)  # hello world
```

- `startswith(prefix)`: Gibt einen Boolean zurück, der angibt, ob ein String mit dem angegebenen Präfix beginnt.
    

```python
my_str = 'hello world'

starts_with_hello = my_str.startswith('hello')
print(starts_with_hello)  # True
```

- `endswith(suffix)`: Gibt einen Boolean zurück, der angibt, ob ein String mit dem angegebenen Suffix endet.
    

```python
my_str = 'hello world'

ends_with_world = my_str.endswith('world')
print(ends_with_world)  # True
```

- `find(substring)`: Gibt den Index des ersten Vorkommens von `substring` zurück oder `-1`, wenn keines gefunden wird.
    

```python
my_str = 'hello world'

world_index = my_str.find('world')
print(world_index)  # 6
```

- `count(substring)`: Gibt die Anzahl der Vorkommen eines Teilstrings in einem String zurück.
    

```python
my_str = 'hello world'

o_count = my_str.count('o')
print(o_count)  # 2
```

- `capitalize()`: Gibt einen neuen String zurück, bei dem das erste Zeichen großgeschrieben und alle anderen Zeichen in Kleinbuchstaben umgewandelt sind.
    

```python
my_str = 'hello world'

capitalized_my_str = my_str.capitalize()
print(capitalized_my_str)  # Hello world
```

- `isupper()`: Gibt `True` zurück, wenn alle Buchstaben im String Großbuchstaben sind, und `False`, wenn nicht.
    

```python
my_str = 'hello world'

is_all_upper = my_str.isupper()
print(is_all_upper)  # False
```

- `islower()`: Gibt `True` zurück, wenn alle Buchstaben im String Kleinbuchstaben sind, und `False`, wenn nicht.
    

```python
my_str = 'hello world'

is_all_lower = my_str.islower()
print(is_all_lower)  # True
```

- `title()`: Gibt einen neuen String zurück, bei dem der erste Buchstabe jedes Wortes großgeschrieben ist.
    

```python
my_str = 'hello world'

title_case_my_str = my_str.title()
print(title_case_my_str)  # Hello World
```
