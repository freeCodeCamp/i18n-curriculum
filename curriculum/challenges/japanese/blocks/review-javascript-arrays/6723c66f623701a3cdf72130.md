## JavaScript 配列の基本

- **定義**: JavaScript の配列は、数値のインデックスで識別される値の順序付きコレクションです。JavaScript の配列内の値は、数値、文字列、ブール値、オブジェクト、さらには他の配列など、さまざまなデータ型を含むことができます。配列はメモリ上で連続しているため、すべての要素が単一の連続したメモリ領域に格納されており、効率的なインデックス指定と高速な要素アクセスが可能です。

```js
const developers = ["Jessica", "Naomi", "Tom"];
```

- **配列から要素にアクセスする方法**: 配列から要素にアクセスするには、配列名の後に大括弧内にインデックス番号を指定します。JavaScript の配列はゼロベースのインデックスなので、最初の要素はインデックス 0、2番目の要素はインデックス 1 となります。存在しないインデックスにアクセスしようとすると、JavaScript は `undefined` を返します。

:::interactive_editor

```js
const developers = ["Jessica", "Naomi", "Tom"];
console.log(developers[0]) // "Jessica"
console.log(developers[1]) // "Naomi"

console.log(developers[10]) // undefined
```

:::

- **`length` プロパティ**: このプロパティは、配列内のアイテム数を返すために使われます。

:::interactive_editor

```js
const developers = ["Jessica", "Naomi", "Tom"];
console.log(developers.length) // 3
```

:::

- **配列内の要素を更新する方法**: 配列内の要素を更新するには、代入演算子 (`=`) を使って特定のインデックスの要素に新しい値を代入します。

:::interactive_editor

```js
const fruits = ['apple', 'banana', 'cherry'];
fruits[1] = 'blueberry';

console.log(fruits); // ['apple', 'blueberry', 'cherry']
```

:::

## 2次元配列  

- **定義**: 2次元配列は基本的に配列の配列です。チェスボード、スプレッドシート、画像のピクセルなど、自然なグリッド状の構造を表すデータに使われます。2次元配列の要素にアクセスするには、行のインデックスと列のインデックスの2つが必要です。

:::interactive_editor

```js
const chessboard = [
    ['R', 'N', 'B', 'Q', 'K', 'B', 'N', 'R'],
    ['P', 'P', 'P', 'P', 'P', 'P', 'P', 'P'],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    ['p', 'p', 'p', 'p', 'p', 'p', 'p', 'p'],
    ['r', 'n', 'b', 'q', 'k', 'b', 'n', 'r']
];

console.log(chessboard[0][3]); // "Q"
```

:::

## 配列の分割代入  

- **定義**: 配列の分割代入は、JavaScript の機能で、配列から値を取り出して変数により簡潔で読みやすい方法で代入できます。配列の要素を個別の変数に展開する便利な構文を提供します。

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange"];

const [first, second, third] = fruits;

console.log(first); // "apple"
console.log(second); // "banana"
console.log(third); // "orange"
```

:::

- **rest構文**: 分割代入されなかった配列の残りの要素を新しい配列にまとめることができます。

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange", "mango", "kiwi"];
const [first, second, ...rest] = fruits;

console.log(first); // "apple"
console.log(second); // "banana"
console.log(rest); // ["orange", "mango", "kiwi"]
```

:::

## よく使う配列メソッド 

- **`push()` メソッド**: このメソッドは配列の末尾に要素を追加し、新しい長さを返します。

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.push("ice cream");

console.log(desserts); // ["cake", "cookies", "pie", "ice cream"];
```

:::

- **`pop()` メソッド**: このメソッドは配列の最後の要素を削除し、その削除した要素を返します。配列が空の場合は戻り値が `undefined` になります。

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.pop();

console.log(desserts); // ["cake", "cookies"];
```

:::

- **`shift()` メソッド**: このメソッドは配列の最初の要素を削除し、その削除した要素を返します。配列が空の場合は戻り値が `undefined` になります。

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.shift();

console.log(desserts); // ["cookies", "pie"];
```

:::

- **`unshift()` メソッド**: このメソッドは配列の先頭に要素を追加し、新しい長さを返します。

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.unshift("ice cream");

console.log(desserts); // ["ice cream", "cake", "cookies", "pie"];
```

:::

- **`indexOf()` メソッド**: このメソッドは配列内の特定の要素の最初のインデックスを見つけるのに便利です。要素が見つからない場合は `-1` を返します。

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange", "banana"];
const index = fruits.indexOf("banana");

console.log(index); // 1
console.log(fruits.indexOf("not found")); // -1
```

:::

- **`splice()` メソッド**: このメソッドは配列の任意の位置から要素を追加または削除するために使われます。`splice()` メソッドの戻り値は、配列から削除されたアイテムの配列です。何も削除されなければ空の配列が返されます。このメソッドは元の配列を変更し、新しい配列を作成しません。最初の引数は配列の変更を開始するインデックスを指定し、2番目の引数は削除する要素数を指定します。続く引数は追加したい要素です。

:::interactive_editor

```js
const colors = ["red", "green", "blue"];
colors.splice(1, 0, "yellow", "purple");

console.log(colors); // ["red", "yellow", "purple", "green", "blue"]
```

:::

- **`includes()` メソッド**: このメソッドは配列が特定の値を含むかどうかをチェックします。配列に指定した要素が含まれていれば `true` を返し、そうでなければ `false` を返します。

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];

console.log(programmingLanguages.includes("Python")); // true
console.log(programmingLanguages.includes("Perl")); // false
```

:::

- **`concat()` メソッド**: このメソッドは2つ以上の配列を結合して新しい配列を作成します。

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];
const newList = programmingLanguages.concat("Perl");

console.log(newList); // ["JavaScript", "Python", "C++", "Perl"]
```

:::

- **`slice()` メソッド**: このメソッドは元の配列の一部を指定した開始・終了インデックスで浅くコピーした新しい配列を返します。新しい配列は元の配列と同じ要素の参照を含みます（複製ではありません）。つまり、要素がプリミティブ（数値や文字列など）なら値がコピーされますが、要素がオブジェクトや配列の場合は参照がコピーされ、オブジェクト自体はコピーされません。

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];
const newList = programmingLanguages.slice(1);

console.log(newList); // ["Python", "C++"]
```

:::

- **スプレッド構文**: スプレッド構文は配列の浅いコピーを作成するために使われます。

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const shallowCopiedArray = [...originalArray];

shallowCopiedArray.push(4);

console.log(originalArray); // [1, 2, 3]
console.log(shallowCopiedArray); // [1, 2, 3, 4]
```

:::

- **`split()` メソッド**: このメソッドは文字列を指定した区切り文字で分割し、部分文字列の配列を作成します。区切り文字が指定されていない場合は、元の文字列を単一要素とする配列を返します。

:::interactive_editor

```js
const str = "hello";
const charArray = str.split("");

console.log(charArray); // ["h", "e", "l", "l", "o"]
```

:::

- **`reverse()` メソッド**: このメソッドは配列をその場で逆順に並べ替えます。

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
console.log(desserts.reverse()); // ["pie", "cookies", "cake"]
```

:::

- **`join()` メソッド**: このメソッドは配列のすべての要素を指定した区切り文字で連結して1つの文字列にします。区切り文字が指定されていないか空文字列 (`""`) の場合は、区切りなしで連結されます。

:::interactive_editor

```js
const reversedArray = ["o", "l", "l", "e", "h"];
const reversedString = reversedArray.join("");

console.log(reversedString); // "olleh"
```

:::
