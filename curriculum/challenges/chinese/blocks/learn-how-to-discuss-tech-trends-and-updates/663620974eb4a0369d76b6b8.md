# --description--

短语 `some of these` 用于指代一组中的特定部分或来自更大集合的项的子集。当你想提及一组中的几项而不逐一列出时，通常会使用它。

例如，`Let's try some of these cookies.` 说话者建议尝试来自更大批次的几个饼干。
