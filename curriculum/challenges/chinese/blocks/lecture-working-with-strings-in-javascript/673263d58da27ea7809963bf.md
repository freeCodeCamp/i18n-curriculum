# --interactive--

在 JavaScript 中，模板字面量是处理字符串的一种强大而灵活的方法。 与普通字符串使用单引号（`'`）或双引号（`"`）不同，模板字面量使用反引号（```）定义。

它们可以让字符串操作变得更简单，包括直接在字符串中嵌入变量，这种特性被称为字符串插值。

模板字面量可以更轻松地创建跨多行的字符串，或直接在字符串中包含表达式（如变量甚至 JavaScript 代码）。

这是一个模板字面示例：

:::interactive_editor

```js
const name = "Alice";
const greeting = `Hello, ${name}!`;

console.log(greeting);
```

:::

注意使用反引号而不是单引号或双引号。 `${name}` 语法是字符串插值的一个示例，其中变量 `name` 的值会直接插入字符串中。

字符串插值可以将变量和表达式嵌入字符串中。 与使用 `+` 操作符连接字符串和变量的旧方法相比，这是一项重大改进。

下面是一个使用字符串连接和加号（`+`）操作符的示例：

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = "My name is " + name + " and I am " + age + " years old.";
console.log(message); 
```

:::

下面是一个使用模板字面量和字符串插值的示例：

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = `My name is ${name} and I am ${age} years old.`;
console.log(message); 
```

:::

正如你所看到的，使用模板字面量进行字符串插值更加简洁易读，尤其是在处理多个变量时。

模板字面量的另一大特点是支持多行字符串。 对于常规字符串，你需要使用转义字符（`\n`）来创建新行。 使用模板字面量时，只需跨行写入字符串，格式就会得到保留：

:::interactive_editor

```js
let poem = `Roses are red,
Violets are blue,
JavaScript is fun,
And so are you.`;

console.log(poem);
```

:::

模板字面值的另一个特色是它们允许你直接在字串中嵌入 JavaScript 表达式，就像这个例子中一样：

:::interactive_editor

```js
const song = "Bohemian Rhapsody";
const score = 9.5;
const highestScore = 10;
const output = `One of my favorite songs is "${song}". I rated it ${
  (score / highestScore) * 100
}%.`;
console.log(output); 
```

:::

当需要在字符串中包含变量或表达式、格式化复杂字符串或处理多行文本时，模板字面量尤其有用。 与传统的字符串连接相比，它们更加简洁易读。
