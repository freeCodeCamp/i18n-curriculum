# --explanation--

`Minor bugs` são pequenos erros ou problemas em um sistema ou programa.Por exemplo:

`There are some minor bugs in the app, but it still runs.` — Isso significa que os problemas não são graves mas precisam ser corrigidos.

`User interface` (frequentemente chamado de `UI`) é a parte visual do software com a qual as pessoas interagem.Por exemplo: 

`The user interface was updated to include a dark mode.` — Isso se refere a mudanças que os usuários podem ver e usar diretamente.
