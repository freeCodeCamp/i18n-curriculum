# --description--

Pig Latin es una forma de alterar palabras en inglés siguiendo reglas específicas de transformación.

**Objetivo:** Cumplir con las historias de usuario a continuación y pasar todas las pruebas para completar el laboratorio.

**Historias de usuario:**

1. Debes crear una función `translatePigLatin` que acepte una cadena como argumento.
1. Si el argumento de tipo cadena comienza con una consonante, tu función debe tomar la primera consonante o grupo consonántico, moverlo al final de la palabra, agregar `ay` y devolver el resultado.
1. Si el argumento de tipo cadena comienza con una vocal, tu función debe agregar `way` al final y devolver el resultado.
1. Tu función debe manejar argumentos de cadena donde la primera vocal aparece en medio de la palabra y devolver la cadena transformada apropiadamente.
1. Si el argumento de tipo cadena no tiene vocales, tu función debe agregar `ay` al final y devolver el resultado.

Nota: Para el contexto de este laboratorio, las vocales son `a`, `e`, `i`, `o` y `u`. La letra `y` no se considera una vocal.

```js
function translatePigLatin(str) {
  if (isVowel(str.charAt(0))) return str + "way";
  let front = [];
  str = str.split('');
  while (str.length && !isVowel(str[0])) {
    front.push(str.shift());
  }
  return [].concat(str, front).join('') + 'ay';
}

function isVowel(c) {
  return ['a', 'e', 'i', 'o', 'u'].indexOf(c.toLowerCase()) !== -1;
}
```
