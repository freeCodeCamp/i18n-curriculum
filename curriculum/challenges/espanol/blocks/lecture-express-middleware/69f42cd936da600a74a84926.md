# --description--

Middleware es un concepto esencial en Express que nos permite manejar solicitudes y respuestas entre el cliente y el servidor. Las funciones middleware tienen acceso al objeto solicitud, al objeto respuesta y a la función next en el ciclo de solicitud-respuesta de la aplicación.

Vamos a desglosarlo:

## Qué es middleware

Middleware es esencialmente una función que se ejecuta durante el ciclo de vida de una solicitud al servidor. Realiza diversas tareas como registrar solicitudes, manejar errores, analizar solicitudes entrantes o servir archivos estáticos.

Express ofrece una forma conveniente de definir middleware que procesa las solicitudes entrantes antes de que lleguen a los manejadores de ruta.

Cómo funciona middleware:

Cuando se realiza una solicitud a una aplicación Express, el middleware se ejecuta en el orden en que se agrega. El flujo es así:

* La solicitud se envía al servidor.
* Las funciones middleware procesan la solicitud.
* Si el middleware llama a `next()`, pasa el control al siguiente middleware o manejador de ruta.
* Si el middleware no llama a `next()`, el ciclo de solicitud-respuesta se detiene.

```js
const express = require('express')
const app = express()

// Simple middleware that logs request info
app.use((req, res, next) => {
  console.log('Request URL:', req.url)
  console.log('Request Method:', req.method)
  next() // Passes control to the next middleware
})

app.get('/', (req, res) => {
  res.send('Hello World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

`app.use()`: Agrega middleware globalmente a tu aplicación.

En este ejemplo, el middleware registra detalles de la solicitud y llama a `next()` para continuar con el siguiente middleware o manejador de ruta.

Middleware es una funcionalidad poderosa en Express que te permite interceptar y modificar solicitudes en varios puntos. Puedes usar middleware para manejar cosas como registro, autenticación o incluso modificar los objetos de solicitud y respuesta antes de que lleguen a tus manejadores de ruta.
