# --description--

Die D3-Bibliothek setzt den Fokus auf einen datengesteuerten Ansatz. Wenn Sie einen Datensatz haben, können Sie D3-Methoden anwenden, um ihn auf der Seite darzustellen. Daten liegen in vielen Formaten vor, aber diese Aufgabe verwendet ein einfaches Array von Zahlen.

Der erste Schritt besteht darin, D3 die Daten bekannt zu machen. Die `data()`-Methode wird auf eine Auswahl von DOM-Elementen angewendet, um die Daten an diese Elemente anzuhängen. Der Datensatz wird als Argument an die Methode übergeben.

Ein gängiges Arbeitsmuster ist es, für jedes Datenelement im Satz ein neues Element im Dokument zu erstellen. D3 bietet dafür die `enter()`-Methode.

Wenn `enter()` mit der `data()`-Methode kombiniert wird, betrachtet sie die ausgewählten Elemente auf der Seite und vergleicht sie mit der Anzahl der Datenobjekte im Satz. Wenn es weniger Elemente als Datenobjekte gibt, werden die fehlenden Elemente erstellt.

Hier ist ein Beispiel, das ein `ul`-Element auswählt und basierend auf der Anzahl der Einträge im Array ein neues Listenelement erstellt:

```html
<body>
  <ul></ul>
  <script>
    const dataset = ['a', 'b', 'c'];
    d3.select('ul')
      .selectAll('li')
      .data(dataset)
      .enter()
      .append('li')
      .text('New item');
  </script>
</body>
```

Es mag verwirrend erscheinen, Elemente auszuwählen, die noch nicht existieren. Dieser Code sagt D3, zuerst das `ul` auf der Seite auszuwählen. Dann werden alle Listenelemente ausgewählt, was eine leere Auswahl zurückgibt. Danach überprüft die `data()`-Methode den Datensatz und führt den folgenden Code dreimal aus, einmal für jedes Element im Array. Die `enter()`-Methode sieht, dass keine `li`-Elemente auf der Seite vorhanden sind, aber 3 benötigt werden (eines für jedes Datenelement in `dataset`). Neue `li`-Elemente werden an das `ul` angehängt und enthalten den Text `New item`.
