# --explanation--

`Not just ... but` es una estructura común usada para mostrar contraste o añadir énfasis. Significa que algo es más que una cosa. La segunda parte añade más información o importancia. Por ejemplo:

`Debugging is not just about finding errors, but about understanding the code better.` - Esto significa que depurar es más que solo corregir errores; también ayuda a comprender el código más a fondo.
