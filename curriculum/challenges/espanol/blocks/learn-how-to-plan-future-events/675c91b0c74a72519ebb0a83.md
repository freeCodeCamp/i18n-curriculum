# --explanation--

`They'll` es la contracción de `they will`. Se usa para hablar de algo esperado o predicho que sucederá en el futuro. 

`Likely` significa que algo es probable o tiene una alta probabilidad de ocurrir. Por ejemplo:

- `They'll likely ask for more information.` - Esto predice una probable solicitud de los clientes.

- `The feature will likely need additional testing.` - Esto indica una alta probabilidad de que se requiera realizar pruebas.
