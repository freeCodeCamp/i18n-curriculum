# --description--

在询问和回答关于数量的问题时，重要的是要一致地使用 `much` 和 `many`。`Much` 用于不可数名词，而 `many` 用于可数名词。在对话中，问题和答案应在使用 `much` 或 `many` 上保持匹配。

示例：

问题：`How much time do you have?` 答案：`Not much.`

问题：`How many books do you own?` 答案：`Not many.`
