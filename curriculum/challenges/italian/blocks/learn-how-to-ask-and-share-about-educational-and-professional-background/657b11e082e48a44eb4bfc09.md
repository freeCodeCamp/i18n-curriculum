# --description--

`Software` indica i programmi o le applicazioni che girano su un computer o dispositivo. È diverso da `hardware`, che sono le parti fisiche di un computer. Per esempio:

`She works in software development.` - Questo significa che lei crea o progetta programmi per computer.

Un `software update team` è un gruppo di persone responsabili di migliorare il software esistente correggendo problemi (bug), aggiungendo nuove funzionalità o rendendolo più sicuro.
