# --interactive--

絕對定位允許你將元素從正常的文件流程中取出，使其行為獨立於其他元素。當元素被絕對定位時，它會被放置在自己的圖層中，與版面中的其他所有內容完全分離。這使得它適用於建立浮動的使用者介面功能，例如模態視窗、工具提示或下拉選單，這些都可以覆蓋頁面上的其他元素。

預設情況下，絕對定位的元素會相對於最近的定位祖先進行定位。如果找不到定位祖先，該元素將相對於初始包含區塊定位，通常是瀏覽器的視窗。你可以使用 `top`、`bottom`、`left` 和 `right` 屬性來移動絕對定位的元素，指定它與包含元素邊緣的距離。

例如，設定 `top: 30px` 和 `left: 30px` 將會使元素距離包含區塊的上邊和左邊邊緣各移動 `30px`。以下是一個如何在 CSS 中為元素套用絕對定位的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="positioned">Absolutely Positioned</div>
```

```css
body {
  background-color: #eeeeee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

當此程式碼被套用時，該元素將從正常文件流程中移除，並距離其包含區塊的頂部和左側 `30px`。

絕對定位對於創建預期浮在其他內容之上的元素特別有幫助，例如覆蓋層或模態視窗。然而，由於該元素已從文件流程中移除，如果沒有適當控制代碼，也可能會產生空隙或導致其他元素塌陷在一起。
