# --explanation--

`Enough`는 충분한 양을 가지고 있다는 뜻입니다. 예를 들어:

`We don't have enough resources to complete the project.` - 이것은 사용 가능한 자원이 부족하다는 뜻입니다.

`Actual`는 단순히 논의하거나 계획하는 것이 아니라 실제적이거나 본질적인 것을 의미합니다. 예를 들어:

`The meeting took longer than expected, so we had little time for the actual work.` - 이것은 논의 때문에 실제 작업이 지연되었다는 뜻입니다.

`To get something done`는 작업을 완료하거나 해야 할 일을 끝내는 것을 의미합니다. 예를 들어:

`We need to get this report done by Friday.` - 이것은 보고서를 마감일 전에 완료해야 한다는 뜻입니다.
