# --description--

考慮邊長為整數且滿足 $a ≤ b ≤ c$ 的三角形。

若 $gcd(a,b,c) = 1$，則整數邊長三角形 $(a,b,c)$ 稱為原始三角形。

周長不超過 $10\,000\,000$ 的原始整數邊三角形有多少個？
