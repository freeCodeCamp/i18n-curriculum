# --interactive--

## Стилізація списків

- **Властивість `line-height`**: Ця властивість використовується для створення простору між рядками тексту. Допустимі значення `line-height` включають ключове слово `normal`, числа, відсотки та одиниці довжини, як-от одиниця `em`.
- **Властивість `list-style-type`**: Ця властивість використовується для вказівки маркера для елемента списку. Допустимі значення можуть включати коло, диск або десяткове число.

Ось приклад використання `list-style-type` для зміни стилю маркера:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
ul {
  list-style-type: square;
}
```

:::

- **Властивість `list-style-position`**: Ця властивість використовується для встановлення позиції маркера списку. Є лише два допустимі значення: inside і outside.

Ось приклад, що показує різницю між `inside` і `outside`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="inside-list">
  <li>Item with inside position</li>
  <li>Item with inside position</li>
</ul>
<ul class="outside-list">
  <li>Item with outside position</li>
  <li>Item with outside position</li>
</ul>
```

```css
.inside-list {
  list-style-position: inside;
}

.outside-list {
  list-style-position: outside;
}
```

:::

- **Властивість `list-style-image`**: Ця властивість використовується для застосування зображення як маркера елемента списку. Загальне застосування — використання функції `url` зі значенням, встановленим на дійсне розташування зображення.

## Відступи між елементами списку за допомогою `margin`

- Окрім `line-height`, у CSS також можна використовувати поля для покращення відступів і читабельності елементів списку.
- Поля створюють простір поза межами кожного елемента `li`, що дозволяє контролювати горизонтальний розрив між елементами списку.
- `margin-bottom` використовується для створення простору під кожним елементом списку. Наприклад, `margin-bottom: 10px;` створить 10-піксельний розрив під кожним елементом списку.

Ось приклад використання `margin-bottom` для відступів між елементами списку:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="list">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
.list li {
  margin-bottom: 20px;
}
```

## Стилізація посилань

- **`pseudo-class`**: Це ключове слово, додане до селектора, яке дозволяє вибирати елементи на основі певного стану. Загальні стани включають `:hover`, `:visited` і `:focus`.
- **`:link pseudo-class`**: Цей псевдоклас використовується для стилізації посилань, які користувач ще не відвідував.

Ось приклад використання псевдокласу `:link`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:link {
  color: red;
}
```

:::

- **`:visited pseudo-class`**: Цей псевдоклас використовується для стилізації посилань, які користувач уже відвідав.
- **`:hover pseudo-class`**: Цей псевдоклас використовується для стилізації елементів, над якими користувач активно навів курсор.

Ось приклад використання псевдокласу `:hover`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Hover over me!</a>
```

```css
a:hover {
  color: green;
  text-decoration: underline;
}
```

:::

- **`:focus pseudo-class`**: Цей псевдоклас використовується для стилізації елемента, коли він отримує фокус. Прикладами можуть бути `input` або `select`, коли користувач клікає або переходить табом на елемент, щоб сфокусувати його.

Ось приклад використання псевдокласу `:focus`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:focus {
  outline: 2px solid orange;
}
```

:::

- **`:active pseudo-class`**: Цей псевдоклас використовується для стилізації елемента, який був активований користувачем. Загальний приклад — коли користувач натискає на кнопку.

Ось приклад використання псевдокласу `:active`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Click me!</a>
```

```css
a:active {
  color: pink;
}
```

:::

## Робота з фоном і кордонами

- **Властивість `background-size`**: Ця властивість використовується для встановлення розміру фону елемента. Деякі загальні значення включають `cover` для того, щоб фонове зображення покривало весь елемент, і `contain` для того, щоб фонове зображення поміщалося всередині елемента.

Ось приклад використання `background-size: cover`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

Ось приклад використання `background-size: contain`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **Властивість `background-repeat`**: Ця властивість визначає, як фонові зображення мають повторюватися по горизонтальній і вертикальній осях. Значення за замовчуванням для `background-repeat` — `repeat`, що означає, що зображення повторюватиметься і по горизонталі, і по вертикалі. Також можна вказати, що повторення не має бути, використовуючи властивість `no-repeat`.

Ось приклад використання `background-repeat: no-repeat`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **Властивість `background-position`**: Ця властивість використовується для вказівки позиції фонового зображення. Її можна встановити як конкретну довжину, відсоток або ключові слова, такі як `top`, `bottom`, `left`, `right` і `center`.

Ось приклад використання `background-position`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

- **Властивість `background-attachment`**: Ця властивість використовується для вказівки, чи має фонове зображення прокручуватися разом із вмістом, чи залишатися зафіксованим на місці. Основні значення — `scroll` (за замовчуванням), коли фонове зображення прокручується разом із вмістом, і `fixed`, коли фонове зображення залишається на одному місці на екрані.
- **Властивість `background-image`**: Ця властивість використовується для встановлення фонового зображення елемента. Можна встановити кілька фонових зображень одночасно та використовувати функції `url`, `radial-gradient` або `linear-gradient` як значення.

Ось приклад використання `background-image`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **Властивість `background`**: Це скорочена властивість для встановлення всіх властивостей фону в одному оголошенні. Ось приклад встановлення фонового зображення та заборони його повторення: `background: no-repeat url("example-url-goes-here");`

Ось приклад використання скороченої властивості `background`:

:::interactive_editor

```html
<style>
  body {
    background: center top no-repeat url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **Добрий контраст для кольорів фону та переднього плану**: Важливо забезпечити добрий коефіцієнт контрастності між кольорами фону та переднього плану, щоб текст був читабельним. Рекомендації Web Content Accessibility Guidelines (WCAG) радять мінімальний коефіцієнт контрастності 4.5:1 для звичайного тексту і 3:1 для великого тексту.

## Кордони

- **Властивість `border-top`**: Ця властивість використовується для встановлення стилів верхнього кордону елемента. `border-top: 3px solid blue;` встановлює 3-піксельний суцільний синій кордон зверху елемента.
- **Властивість `border-right`**: Ця властивість використовується для встановлення стилів правого кордону елемента. `border-right: 2px solid red;` встановлює 2-піксельний суцільний червоний кордон справа елемента.
- **Властивість `border-bottom`**: Ця властивість використовується для встановлення стилів нижнього кордону елемента. `border-bottom: 1px dashed green;` встановлює 1-піксельний пунктирний зелений кордон знизу елемента.
- **Властивість `border-left`**: Ця властивість використовується для встановлення стилів лівого кордону елемента. `border-left: 4px dotted orange;` встановлює 4-піксельний пунктирний помаранчевий кордон зліва елемента.

Ось приклад використання окремих властивостей кордону:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="bordered-box">Box with different borders</div>
```

```css
.bordered-box {
  border-top: 3px solid blue;
  border-right: 2px solid red;
  border-bottom: 1px dashed green;
  border-left: 4px dotted orange;
  padding: 20px;
}
```

:::

- **Властивість `border`**: Це скорочена властивість для встановлення ширини, стилю та кольору кордону елемента. `border: 1px solid black;` встановлює 1-піксельний суцільний чорний кордон.

Ось приклад використання скороченої властивості `border`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

- **Властивість `border-radius`**: Ця властивість використовується для створення заокруглених кутів кордону елемента.

Ось приклад використання `border-radius`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

- **Властивість `border-style`**: Ця властивість використовується для встановлення стилю кордону елемента. Деякі допустимі значення включають `solid`, `dashed`, `dotted` і `double`.

Ось приклад використання різних значень `border-style`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="solid-border">Solid border</div>
<div class="dashed-border">Dashed border</div>
<div class="dotted-border">Dotted border</div>
<div class="double-border">Double border</div>
```

```css
.solid-border {
  border: 3px solid blue;
  margin-bottom: 10px;
  padding: 10px;
}

.dashed-border {
  border: 3px dashed red;
  margin-bottom: 10px;
  padding: 10px;
}

.dotted-border {
  border: 3px dotted green;
  padding: 10px;
}

.double-border {
  border: 3px double blueviolet;
  padding: 10px;
}
```

:::

## Градієнти

- **Функція `linear-gradient()`**: Ця CSS-функція використовується для створення переходу між кількома кольорами вздовж прямої лінії.

Ось приклад використання `linear-gradient()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

- **Функція `radial-gradient()`**: Ця CSS-функція створює зображення, що розходиться від певної точки, як коло або еліпс, і поступово переходить між кількома кольорами.

Ось приклад використання `radial-gradient()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```
