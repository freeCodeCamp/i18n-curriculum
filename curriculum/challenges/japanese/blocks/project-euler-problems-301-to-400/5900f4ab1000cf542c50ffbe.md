# --description--

長さ $n$ のシーケンス $x_1, x_2, \ldots, x_n$ が次の条件を満たすとします：

- $x_1 = 2$
- すべての $1 < i ≤ n$ に対して：$x_{i - 1} < x_i$
- すべての $i$ と $j$ で $1 ≤ i, j ≤ n$ のとき：${(x_i)}^j < {(x_j + 1)}^i$

長さ2のこのようなシーケンスは5つだけで、それぞれ {2,4}、{2,5}、{2,6}、{2,7}、{2,8} です。長さ5のこのようなシーケンスは293個あり、例として {2,5,11,25,55}、{2,6,14,36,88}、{2,8,22,64,181} が挙げられます。

$t(n)$ を長さ $n$ のこのようなシーケンスの個数とします。$t(10) = 86195$、$t(20) = 5227991891$ が与えられています。

$t({10}^{10})$ を求め、その答えを $10^9$ で割った余りで答えてください。
