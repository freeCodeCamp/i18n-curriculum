# --explanation--

`Lightweight` تصف شيئًا لا يتطلب الكثير من الموارد أو الطاقة، مما يجعله سهل التشغيل، سريعًا، وفعالًا. على سبيل المثال:

`The app is lightweight, so it runs smoothly even on older smartphones.` - هذا يعني أن التطبيق يستخدم ذاكرة أو قوة معالجة قليلة جدًا، مما يسمح له بالعمل جيدًا على الأجهزة ذات السعة الأقل.

`Features` هي دوال أو أدوات محددة يقدمها منتج أو برنامج للمستخدمين. على سبيل المثال:

`The software has many useful features like file sharing and cloud storage.` - هذا يعني أن البرنامج يتضمن عدة دوال مفيدة تجعله أكثر قيمة وكفاءة للمستخدمين.
