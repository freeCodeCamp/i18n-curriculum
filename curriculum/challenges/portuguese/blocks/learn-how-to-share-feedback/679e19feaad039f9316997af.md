# --explanation--

A frase `Thanks for` + `-ing` é usada para expressar gratidão por uma ação que aconteceu ou está acontecendo.O verbo que segue `for` assume a forma `-ing` porque atua como um substantivo (gerúndio).Por exemplo:

`Thanks for helping me with the report.` - Isso significa que o falante aprecia a ajuda dada para o relatório.
