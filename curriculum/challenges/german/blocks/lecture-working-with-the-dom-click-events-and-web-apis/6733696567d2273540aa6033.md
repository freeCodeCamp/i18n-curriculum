# --interactive--

Die `Web Animations`-API (WAAPI) ermöglicht es Ihnen, Animationen direkt in JavaScript zu erstellen und zu steuern. Mit WAAPI können Sie Animationen dynamischer handhaben, was die Manipulation erleichtert.

Werfen wir einen genaueren Blick auf WAAPI, damit Sie direkt in Ihrem JavaScript-Code mit Animationen arbeiten können. Im Kern von WAAPI steht der `Animation`-Konstruktor, der mehrere Instanzmethoden und -eigenschaften bereitstellt, mit denen Sie Elemente dynamisch animieren können.

Eine wichtige Methode im `Animation`-Konstruktor ist `animate()`. Damit können Sie eine Animation erstellen, indem Sie Keyframes und Optionen wie Dauer, Richtung, Easing und Wiederholungen angeben.

Hier ist die grundlegende Syntax der `animate()`-Methode:

```js
element.animate(keyframes, options);
```

Schauen wir uns ein Beispiel an.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="square" id="square"></div>
<script src="index.js"></script>
```

```css
body {
  background: #f1f1f1;
  display: flex;
  align-items: center;
  justify-content: center;
  height: 100vh;
}

.square {
  background: #1b1b32;
  width: 10rem;
  aspect-ratio: 1/1;
}
```

```js
const square = document.querySelector("#square");

const animation = square.animate(
  [{ transform: "translateX(0px)" }, { transform: "translateX(100px)" }],
  {
    duration: 2000, // makes the animation last 2 seconds
    iterations: Infinity, // loops indefinitely
    direction: "alternate", // moves back and forth
    easing: "ease-in-out" // smooth easing
  }
);
```

:::

Das Ergebnis im Browser ist ein blaues Quadrat, das sich horizontal hin und her bewegt.

Zu den Instanzmethoden des `Animation`-Konstruktors gehören:

```js
play()
pause()
reverse()
finish()
cancel()
```

Und die Instanz-Eigenschaften umfassen:

```md
playbackRate
currentTime
startTime
effect
timeline
playState
finished
onfinish
oncancel
```

Lassen Sie uns das Beispiel so ändern, dass die Methoden `play()`, `pause()` und die Eigenschaft `onfinish` verwendet werden. Hier ist die neue Version:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="square" id="square"></div>

<button id="playBtn">Play</button>
<button id="pauseBtn">Pause</button>
<script src="index.js"></script>
```

```css
body {
  background: #f1f1f1;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  height: 100vh;
}
.square {
  background: #1b1b32;
  width: 10rem;
  aspect-ratio: 1/1;
  margin-bottom: 20px;
}
button {
  margin: 10px;
  padding: 10px 20px;
  font-size: 16px;
}
```

```js
const square = document.querySelector("#square");
const playBtn = document.querySelector("#playBtn");
const pauseBtn = document.querySelector("#pauseBtn");

const animation = square.animate(
  [{ transform: "translateX(0px)" }, { transform: "translateX(200px)" }],
  {
    duration: 5000, // Animation lasts 5 seconds
    // iterations: Infinity, // Loops indefinitely
    direction: "alternate", // Moves back and forth
    easing: "ease-in-out" // Smooth easing function
  }
);

// Set the onfinish property to log a message when the animation ends
animation.onfinish = () => {
  console.log("Animation finished!");
};

// Play the animation when the "Play" button is clicked
playBtn.addEventListener("click", () => {
  animation.play();
  console.log("You start the animation");
});

// Pause the animation when the "Pause" button is clicked
pauseBtn.addEventListener("click", () => {
  animation.pause();
  console.log("You pause the animation");
});
```

:::

Das Ergebnis im Browser zeigt die blaue Box, die sich von links nach rechts bewegt, wenn der `play`-Button geklickt wird.

WAAPI erweitert die Möglichkeiten von CSS-Animationen, indem es eine dynamischere Steuerung von Animationen direkt in JavaScript ermöglicht.

Bei CSS-Animationen definieren Sie Animationen deklarativ mit Eigenschaften wie `animation-name`, `animation-duration` und `animation-timing-function`. Sie können dasselbe auch mit der `animate()`-Methode von WAAPI tun.

Der Unterschied besteht darin, dass Sie die mit der `animate()`-Methode erstellten Animationen direkter und dynamischer steuern können, während Sie bei CSS-Animationen viel mehr tun müssen, indem Sie benutzerdefinierte Stile definieren und diese in Ihrer JavaScript-Datei auslösen.

CSS-Animationen eignen sich ideal für einfache und deklarative Animationen, die automatisch ablaufen. Dazu gehören Hover-Effekte, Übergänge oder Animationen, die nach dem Auslösen nicht viel Interaktion benötigen. Wenn Ihre Animation auf Benutzerinteraktionen wie Klicks oder Scrollen reagieren soll oder Sie möchten, dass der Benutzer die Animation pausieren, rückwärts abspielen oder die Geschwindigkeit dynamisch ändern kann, ist WAAPI die bessere Wahl.

Sie können auch beide kombinieren, wenn Sie die Einfachheit von CSS für die Grundkonfiguration wünschen, aber die Animationen zur Laufzeit steuern müssen.
