# --description--

Una secuencia de enteros $S = \{s_i\}$ se llama una $n$-secuencia si tiene $n$ elementos y cada elemento $s_i$ cumple $1 ≤ s_i ≤ n$. Por lo tanto, hay $n^n$ $n$-secuencias distintas en total.

Por ejemplo, la secuencia $S = \{1, 5, 5, 10, 7, 7, 7, 2, 3, 7\}$ es una secuencia de 10 elementos.

Para cualquier secuencia $S$, sea $L(S)$ la longitud de la subsecuencia contigua más larga de $S$ con el mismo valor. Por ejemplo, para la secuencia $S$ dada arriba, $L(S) = 3$, debido a los tres 7 consecutivos.

Sea $f(n) = \sum L(S)$ para todas las secuencias $n$ $S$.

Por ejemplo, $f(3) = 45$, $f(7) = 1\,403\,689$ y $f(11) = 481\,496\,895\,121$.

Encuentra $f(7\,500\,000)\bmod 1\,000\,000\,009$.
