# --description--

У попередніх уроках ви вперше познайомилися з властивістю `length`, ця властивість повертає кількість елементів у масиві. Ось коротке нагадування, як це працює:

```js
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits.length); // 3
```

Можна мати масиви з порожніми слотами. Порожні слоти — це слоти, у яких нічого немає. Це відрізняється від масиву зі значенням `undefined`. Такі масиви називають розрідженими масивами. Ось приклад:

```js
const sparseArray = [1, , , 4];
console.log(sparseArray.length); // 4
```

У цьому випадку, хоча у нас визначено лише два елементи, `1` і `4`, довжина дорівнює `4`, тому що найбільший індекс (`3`) плюс `1` дає нам довжину `4`.

Тепер обговоримо, як створити порожній масив фіксованої довжини. Існує кілька способів зробити це в JavaScript, але одним із загальних методів є використання конструктора `Array()` з числовим аргументом. Конструктор `Array()` можна використовувати з ключовим словом `new` для створення нового масиву. Ось приклад:

```js
const emptyArray = new Array(5);
console.log(emptyArray.length); // 5
console.log(emptyArray); // [ , , , , ]
```

У цьому прикладі ми створюємо новий масив за допомогою `Array(5)`. Це створює розріджений масив з довжиною `5`, де всі слоти порожні.

Інший спосіб створити порожній масив фіксованої довжини — використати метод `Array.from()` з аргументом довжини. На відміну від `new Array(n)`, цей метод створює масив заданої довжини, де всі елементи існують і мають значення `undefined`:

```js
const fixedLengthArray = Array.from({ length: 5 });
console.log(fixedLengthArray.length); // 5
console.log(fixedLengthArray); // [undefined, undefined, undefined, undefined, undefined]
```

Якщо ви хочете створити масив певної довжини і заповнити його значенням за замовчуванням, ви можете використати метод `Array.fill()`:

```js
const filledArray = new Array(3).fill(0);
console.log(filledArray); // [0, 0, 0]
```

Це створює масив довжиною три і заповнює всі елементи значенням `0`. Примітка: при заповненні об’єктами всі слоти посилаються на один і той самий об’єкт; якщо потрібні незалежні копії, використовуйте функцію зворотного виклику або `Array.from()`.

Розуміння того, як отримати довжину масиву і створювати масиви фіксованої довжини, важливе для багатьох програмних завдань, особливо коли потрібно ініціалізувати масиви для конкретних алгоритмів або структур даних.
