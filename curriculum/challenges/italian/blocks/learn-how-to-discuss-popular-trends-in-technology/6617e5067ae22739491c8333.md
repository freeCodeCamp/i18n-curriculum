# --description--

Un `chatbot` è un programma per computer progettato per simulare una conversazione con utenti umani. Invece di una persona, è un computer che risponde ai tuoi messaggi. Le aziende usano i chatbot sui loro siti web per aiutare a rispondere alle domande o fornire informazioni. Quindi, quando vedi apparire una piccola finestra di chat mentre fai acquisti online, quello è un chatbot pronto ad assisterti!
