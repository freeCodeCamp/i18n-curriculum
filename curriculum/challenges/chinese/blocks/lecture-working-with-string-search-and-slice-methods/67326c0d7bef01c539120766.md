# --interactive--

在 JavaScript 中处理字符串时，有很多情况下你可能需要检查一个字符串是否包含一个特定的子字符串，即该字符串的一个较小部分。

例如，在执行某些操作前，你可能需要检查用户的输入是否包含特定的单词或字符。 实现这一目标的方式之一是使用 `includes()` 方法。

`includes()` 方法用于检查字符串是否包含特定子字符串。 如果在字符串中找到子字符串，该方法返回 `true`，否则返回 `false`。

这是基本语法：

```js
string.includes(searchValue);
```

就语法而言，`searchValue` 是要在字符串中查找的子字符串。 这是一个示例：

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("awesome");

console.log(result);  // true
```

:::

在本例中，字符串 `JavaScript is awesome!` 中包含单词 `awesome`，所以 `includes()` 方法返回 `true`。

需要注意的是，`includes()` 方法区分大小写。 这意味着要求字符完全匹配，包括大小写。

例如：

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("Awesome");

console.log(result);  // false
```

:::

由于 `Awesome`（`A` 大写）不同于 `awesome`（`a` 小写），结果为 `false`。

你还可以通过为 `includes()` 方法提供第二个参数来查找从字符串中特定索引开始的子字符串：

:::interactive_editor

```js
let text = "Hello, JavaScript world!";
let result = text.includes("JavaScript", 7);

console.log(result);  // true
```

:::

这里，对子字符串 `JavaScript` 的搜索从字符串的第 7 个位置开始，确保跳过该位置之前的任何字符。

`includes()` 方法仅返回 `true` 或 `false` 结果。 它不提供子字符串在字符串中的位置或出现次数的信息。 如果你需要这种详细程度，其他方法（如 `indexOf()` 方法）可能更适合。
