# --description--

У минулому уроці ви дізналися, що сервер обробляє HTTP-запит і надсилає відповідь. Ця відповідь містить різні ресурси, які ви бачите в інтернеті.

Ресурси, що повертаються у відповіді HTTP, включають текстові ресурси, медіаресурси, шрифти тощо. Кожен із цих ресурсів має заголовок **Content-Type**, який повідомляє браузеру, як правильно обробити та відобразити цей ресурс.

Текстові ресурси завжди були фундаментальною частиною вебу. Насправді, веб починався як чистий текст. Сьогодні текстові ресурси включають такі:

* **HTML (Hypertext Markup Language)**: базова структура будь-якої сторінки. Без HTML сторінка не може існувати.
    
* **CSS (Cascading Stylesheet)**: ресурс, який контролює, як виглядає сторінка. За допомогою CSS ви застосовуєте стилі, такі як кольори, шрифти та макет, до HTML.
    
* **JavaScript**: JavaScript додає інтерактивність і динамічний вміст до HTML і CSS.
    
* **JSON і XML**: ці два формати дозволяють передавати структуровані дані між серверами та браузерами.
    
Медіаресурси включають зображення, аудіо та відео. Вони оживляють веб за допомогою візуальних і звукових ефектів.

Багато вебсайтів покладаються на зображення для передачі візуальної інформації. Зображення бувають у різних форматах, таких як `JPG`, `JPEG`, `PNG`, `WEBP` і `GIF`.

Вебсайти на кшталт YouTube і Vimeo надають відеоконтент у форматах `MP4`, `MOV`, `AVI`, `WebM` та інших.

Аудіо є ключовою частиною багатьох платформ, оскільки дозволяє користувачам слухати музику, подкасти та інший звуковий вміст. Загальні аудіоформати включають `MP3`, `AAC`, Ogg і `WAV`. Платформи, як Spotify і YouTube Music, використовують ці формати для забезпечення високоякісного звукового досвіду.

Інші ресурси, що повертаються у відповіді HTTP, — це документи, такі як `PDF`, `DOCX` і `PAGES`, стиснені архіви, як `ZIP` і `RAR`, файли шрифтів, як `WOFF`, `WOFF2` і `TTF`, а також виконувані файли, як `EXE`.

Ось таблиця цих загальних ресурсів і їхніх типів вмісту:

| Тип ресурсу | Заголовок Content-Type |
| --- | --- |
| HTML | text/html |
| CSS | text/css |
| JavaScript | application/javascript |
| JPEG | image/jpeg |
| PNG | image/png |
| GIF | image/gif |
| SVG | image/svg+xml |
| WebP | image/webp |
| WOFF | font/woff |
| WOFF2 | font/woff2 |
| TTF | font/ttf |
| OTF | font/otf |
| MP4 | video/mp4 |
| WebM | video/webm |
| Ogg (відео) | video/ogg |
| MP3 | audio/mpeg |
| AAC | audio/aac |
| WAV | audio/wav |
| Ogg (аудіо) | audio/ogg |
| JSON | application/json |
| XML | application/xml |
| PDF | application/pdf |
| ZIP | application/zip |
