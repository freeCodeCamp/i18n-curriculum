# --explanation--

`First Conditional` використовується для опису реальних або можливих ситуацій у майбутньому та їх ймовірних наслідків. Він має структуру `if` + `Present Simple` час для умови, і `will` або `can` + базове дієслово для результату. Наприклад:

- `If you study hard, you will pass the exam.` — це описує можливу ситуацію та її результат.

- `If it rains tomorrow, we can cancel the picnic.` — це пояснює, що може статися, якщо умова (дощ) відбудеться.
