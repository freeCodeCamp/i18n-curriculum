# --interactive--

Працюючи з CSS, ви часто зустрічаєте три різні типи поведінки відображення елементів: `inline`, `block` та `inline-block`.

Кожна з цих властивостей впливає на те, як елементи розташовуються і як вони взаємодіють з іншими елементами на сторінці.

У цьому уроці ми зосередимося на тому, як працює властивість `inline-block` і чим вона відрізняється від рядкових та блокових елементів.

Блокові елементи поводяться як великі контейнери або «блоки», які займають всю ширину свого батьківського елемента. Вони завжди починаються з нового рядка, а їхня висота і ширина можуть бути налаштовані.

Рядкові елементи займають лише стільки місця, скільки їм потрібно. Вони розташовуються в потоці навколишнього вмісту і не переносяться на новий рядок.

Властивість `inline-block` є гібридом цих двох поведінок. Як і рядкові елементи, `inline-block` елементи залишаються в текстовому потоці, не починаючи нового рядка.

Однак, на відміну від рядкових елементів, ви можете налаштовувати ширину і висоту `inline-block` елемента, так само як і для блокових елементів.

Коротко кажучи, ключова відмінність між `inline` та `inline-block` полягає в тому, що `inline` елементи не можна контролювати за розмірами, тоді як `inline-block` елементи дозволяють повністю керувати розмірами, залишаючись при цьому в рядку з іншим вмістом.

Давайте розглянемо приклад.

:::interactive_editor

```html
<link href="styles.css" rel="stylesheet">

<div class="container">
  <span class="inline-block-element element1">Inline-Block</span>
  <span class="inline-block-element element2">Inline-Block</span>
</div>
```

```css
.inline-block-element {
  display: inline-block;
  width: 150px;
  height: 100px;
}

.element1 {
  background-color: lightblue;
}

.element2 {
  background-color: lightgreen;
}
```

:::

У наведеному прикладі ми маємо `div` з класом `container`. Всередині цього `div` елемента є два `span` елементи.

Кожен із span елементів має встановлену властивість `display:inline-block` та задані ширину і висоту.

Елементи inline-block відображатимуться поруч один з одним, бо поводяться як рядкові елементи, але також мають задані ширину і висоту, що надає їм властивостей блокових елементів.

Але якщо ви видалите властивість `display: inline-block`, то ні `height`, ні `width` не будуть застосовані, навіть якщо ви чітко їх визначили.

Ось оновлений CSS:

:::interactive_editor

```html
<link href="styles.css" rel="stylesheet">

<div class="container">
  <span class="inline-block-element element1">Span element</span>
  <span class="inline-block-element element2">Span element</span>
</div>
```

```css
.inline-block-element {
  width: 150px;
  height: 100px;
}

.element1 {
  background-color: lightblue;
}

.element2 {
  background-color: lightgreen;
}
```

:::

У цьому коді ми видалили властивість `display: inline-block;`, але залишили все інше без змін. Тут `span` елементи повертаються до своєї поведінки за замовчуванням як рядкові елементи.

Внаслідок цього задані ширина і висота ігноруються, і елементи займають лише простір, необхідний для їхнього вмісту.

Розуміння того, як працює `inline-block`, корисне, бо ви можете використовувати його для створення макетів, які потребують і вирівнювання, і контролю розмірів у безперервному текстовому потоці.
