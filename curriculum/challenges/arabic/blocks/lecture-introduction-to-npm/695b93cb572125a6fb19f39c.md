# --description--

في درس سابق، تعلمت ما هي التبعيات وكيف تختار التبعيات المناسبة لمشروعك.

في هذا الدرس، ستتعلم كيفية تثبيت وحذف التبعيات باستخدام npm.

لتثبيت تبعية، استخدم الأمر `npm install` بهذا الشكل:

```bash
npm install dependencyName
```

يمكنك أيضًا اختصار `install` إلى `i` بهذا الشكل:

```bash
npm i dependencyName
```

بعد تثبيت حزمة، يضيف npm قسمًا جديدًا `dependencies` إلى ملف `package.json` الخاص بك (إذا لم يكن موجودًا بالفعل) ويسرد الحزمة التي ثبتها. على سبيل المثال، هذا ما يبدو عليه عند تثبيت حزمة `chalk`:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "chalk": "^5.6.2"
  }
}
```

عند تثبيت تبعية، يظهر عنصران جديدان في مجلد الجذر لمشروعك:

* `node_modules/`: مجلد يحتوي على الشفرة الفعلية للحزم التي ثبتها.
    
* `package-lock.json`: ملف يخزن معلومات مفصلة عن الإصدارات الدقيقة لتلك الحزم، بما في ذلك تبعياتها الخاصة. (ستتعلم المزيد عن هذا الملف في درس لاحق.)
    

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/how-can-you-install-and-remove-dependencies-1.png" alt="">

معظم المشاريع تحتاج أكثر من تبعية واحدة. بدلًا من تثبيتها واحدة تلو الأخرى، يتيح لك npm تثبيت عدة حزم دفعة واحدة بإدخال قائمة أسماء الحزم مفصولة بمسافات:

```bash
npm install express lodash bcrypt
```

تمامًا كما في السابق، عند تثبيت عدة تبعيات دفعة واحدة، يقوم npm بتحديث ملفات `package.json` و `package-lock.json` ومجلد `node_modules` ليشملها. هذا ما سيبدو عليه ملف `package.json`:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "bcrypt": "^6.0.0",
    "chalk": "^5.6.2",
    "express": "^5.1.0",
    "lodash": "^4.17.21"
  }
}
```

لتثبيت `devDependency`، أضف العلامة `--save-dev` إلى أمر `install`. يمكنك أيضًا استخدام النسخة المختصرة `-D`:

```bash
npm i nodemon -D
```

بعد تشغيل هذا الأمر، يقوم npm بتحديث ملف `package.json` ويضيف الحزمة تحت قسم `devDependencies`. هذا مثال على ما يبدو عليه:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "bcrypt": "^6.0.0",
    "chalk": "^5.6.2",
    "express": "^5.1.0",
    "lodash": "^4.17.21"
  },
  "devDependencies": {
    "nodemon": "^3.1.10"
  }
}
```

كلما شغلت أمر `npm install`، يعرض npm رسالة تخبرك ما إذا كانت عملية التثبيت ناجحة أم لا. عادةً ما تبدو رسالة النجاح مثل هذا:

لحذف أو إزالة حزمة، استخدم أمر `npm uninstall`:

```bash
npm uninstall dependencyName
```

على سبيل المثال، لإزالة `chalk` من مشروعك، شغّل هذا الأمر:

```bash
npm uninstall chalk
```

يمكنك أيضًا حذف عدة حزم دفعة واحدة بذكرها مفصولة بمسافات بين أسماء الحزم:

```bash
npm uninstall express lodash bcrypt
```


وسيتم تحديث ملف `package.json` تلقائيًا:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "devDependencies": {
    "nodemon": "^3.1.10"
  }
}
```

لحذف `devDependency`، يمكنك ببساطة تشغيل:

```bash
npm uninstall nodemon
```

لاحظ أنك لا تحتاج إلى العلامة `-D` عند الحذف. سيحذف npm الحزمة تلقائيًا من القسم الصحيح، وهو `devDependencies` في هذه الحالة.

بعد حذف `devDependency`، سيتم تحديث ملف `package.json` مرة أخرى:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC"
}
```
