# --description--

Es gibt mehrere Möglichkeiten, Ihren Beruf auf Spanisch zu nennen, eine davon ist die Verwendung dieser Struktur:

`Soy` + Beruf

Dies ist eine sehr natürliche Art, einen Beruf vorzustellen. Zum Beispiel:

`Soy profesor` – Ich bin Lehrkraft.

`Soy diseñadora.` – Ich bin Designerin.
