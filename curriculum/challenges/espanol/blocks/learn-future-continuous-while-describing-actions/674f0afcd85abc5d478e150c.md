# --explanation--

`Scheduled` significa que algo ha sido planificado u organizado para que ocurra en un momento específico. Por ejemplo:  

`The scheduled meeting will start at 2 PM.` - Esto significa que la reunion ha sido planeada con antelacion para comenzar en el tiempo especificado.

`Maintenance` se refiere a actividades hechas para mantener algo en buen funcionamiento o para solucionar cualquier problema. Por ejemplo:  

`The team performed maintenance on the server to ensure it runs smoothly.` - Esto significa que el equipo trabajó para mantener el server en buenas condiciones al repararlo o actualizarlo.
