# --description--

تُسمى ثلاثية من الأعداد الصحيحة الموجبة ($a$,$b$,$c$) ثلاثية كاردانو إذا حققت الشرط التالي:

$$\sqrt[3]{a + b \sqrt{c}} + \sqrt[3]{a - b \sqrt{c}} = 1$$

على سبيل المثال، (2,1,5) هي ثلاثية كاردانو.

يوجد 149 ثلاثية كاردانو حيث $a + b + c ≤ 1000$.

اوجد عدد ثلاثيات كاردانو التي تحقق $a + b + c ≤ 110\\,000\\,000$.
