## ローカル環境での Python

- **インストール**: Windows、Mac、Linux に Python をインストールする最も良い方法は、公式 Python ウェブサイトからインストーラーをダウンロードすることです (`https://www.python.org/`)。
- **ターミナル**: コマンドを入力してコンピューターとやり取りできるテキストベースのインターフェースです。各オペレーティングシステムにはデフォルトのターミナルアプリがあります。macOS では Terminal アプリを使えます。Windows では Command Prompt または PowerShell を使えます。Linux では、GNOME Terminal や Konsole のように、各デスクトップ環境にデフォルトのターミナルアプリがあります。
- **IDE**: IDE は統合開発環境のことで、コードエディター、テストツール、ターミナルなどの機能を備えています。
- **Python に人気のコードエディターと IDE**: VS Code、PyCharm、Spyder
- **ローカルでコードを実行する**: Python コードを実行するにはいくつかの方法があります。1つ目は、VS Code の右上にある再生ボタンのような実行ボタンをクリックする方法です。これでターミナルが開き、Python スクリプトが実行されます。

もう1つの方法は、ターミナルを開いて手動でプログラムを実行することです。たとえば、`main.py` というプログラムを実行する例は以下の通りです。

```bash
python main.py
```

このコマンドは、`main.py` が保存されているフォルダーから実行する必要があります。たとえば、`main.py` が `python-projects` というフォルダー内にある場合、まずターミナルで次のコマンドを使います。

```bash
cd python-projects
```

その後、次を実行します。

```bash
python main.py
```

## Python インタラクティブシェルの使い方

- **定義**: インタラクティブシェルは、コマンドを1つずつ入力して結果を確認できるプログラムです。ターミナルアプリを開いて `python` と入力し、`Enter` を押してください。これで Python インタラクティブシェルが起動します。`>>>` の記号は、Python がコマンドの入力を待っていることを示します。ターミナルに `"Hello, world!"` を出力してみましょう。

```bash
print("Hello, world!")
```

次のようにテキストが表示されるはずです。

```bash
>>> print("Hello, world!")
Hello, world!
```

テキストが出力された後、Python は次の状態に戻ります。

```bash
>>>
```

これは、さらにコマンドを入力できることを意味します。

- **読み取り、評価、出力、ループサイクル（REPL）**: コマンドを入力すると、インタプリタは入力を読み取り、評価し、結果を出力し、`>>>` を表示してさらにコマンドを入力できるようにループします。
