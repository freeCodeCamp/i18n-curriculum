# --description--

特定のオブジェクトにプロパティが存在するかどうかを確認するには、`.hasOwnProperty()` メソッドを使うことができます。`someObject.hasOwnProperty(someProperty)` は、そのプロパティがオブジェクトに存在するかどうかによって `true` または `false` を返します。

**例**

```js
function checkForProperty(object, property) {
  return object.hasOwnProperty(property);
}

checkForProperty({ top: 'hat', bottom: 'pants' }, 'top'); // true
checkForProperty({ top: 'hat', bottom: 'pants' }, 'middle'); // false
```

最初の `checkForProperty` 関数の呼び出しは `true` を返し、2回目は `false` を返します。

# --instructions--

関数 `checkObj` を修正して、関数のパラメータ `obj` に渡されたオブジェクトが、関数のパラメータ `checkProp` に渡された特定のプロパティを持っているかどうかをテストしてください。`checkProp` に渡されたプロパティが `obj` に見つかった場合、そのプロパティの値を返してください。見つからなければ `Not Found` を返してください。
