# --description--

一排中有 $N$ 个座位。$N$ 个人依次按照以下规则坐满这些座位：

1. 如果没有这样的座位并且有任何座位仅占用一个相邻的座位，则使用这样的座位。
2. 我们可以验证T（10）= 61632和T（1 000）mod 100 000 007 = 47255094.求T（1 000 000）mod 100 000 007。
3. `euler364()`应返回44855254。

设 $T(N)$ 为在给定规则下 $N$ 个座位被 $N$ 个人占据的可能性数量。下图显示了 $T(4) = 8$。

<img alt="eight ways for N seats to be occupied by N people" src="https://cdn.freecodecamp.org/curriculum/project-euler/comfortable-distance.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

我们可以验证 $T(10) = 61\\,632$ 和 $T(1\\,000)\bmod 100\\,000\\,007 = 47\\,255\\,094$。

查找 $T(1\,000\,000)\bmod 100\,000\,007$。
