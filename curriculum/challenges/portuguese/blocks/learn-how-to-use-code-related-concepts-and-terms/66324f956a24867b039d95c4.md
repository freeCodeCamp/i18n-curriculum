# --description--

A palavra `should` é um verbo modal usado para dar conselhos ou sugestões.É seguido pela forma base do verbo.Por exemplo, na frase `You should try that restaurant`, `should` é seguido por `try`, que é a forma base do verbo.

O verbo `pick` significa escolher ou selecionar algo de uma variedade de opções.No contexto da programação, `picking a data type` significa escolher o tipo apropriado de dado para usar, como um inteiro, string e boolean, com base no que é mais adequado para a tarefa em questão.

`Match` significa garantir que duas coisas sejam adequadas ou combinem bem entre si.Por exemplo, `This key matches the lock.` Indica que a chave é adequada para a fechadura.
