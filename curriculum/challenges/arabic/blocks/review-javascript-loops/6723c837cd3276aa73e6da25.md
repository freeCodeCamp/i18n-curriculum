# --interactive--

## العمل مع الحلقات

- **حلقة `for`**: يُستخدم هذا النوع من الحلقات لتكرار كتلة شفرة عددًا معينًا من المرات. تنقسم هذه الحلقة إلى ثلاثة أجزاء: جملة التهيئة، الشرط، وجملة الزيادة/النقصان. تُنفذ جملة التهيئة قبل بدء الحلقة. عادةً ما تُستخدم لتهيئة متغير عداد. يُقيَّم الشرط قبل كل تكرار من الحلقة. التكرار هو مرور واحد عبر الحلقة. إذا كان الشرط `true`، تُنفذ كتلة الشفرة داخل الحلقة. إذا كان الشرط `false`، تتوقف الحلقة وتنتقل إلى كتلة الشفرة التالية. تُنفذ جملة الزيادة/النقصان بعد كل تكرار من الحلقة. عادةً ما تُستخدم لزيادة أو نقصان متغير العداد.

:::interactive_editor

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

:::

- **حلقة `for...of`**: يُستخدم هذا النوع من الحلقات عندما تحتاج إلى التكرار على القيم من قابل للتكرار. أمثلة على القابل للتكرار هي المصفوفات والسلاسل النصية.

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

:::

- **حلقة `for...in`**: يُفضل استخدام هذا النوع من الحلقات عندما تحتاج إلى التكرار على خصائص كائن. ستتكرر هذه الحلقة على جميع الخصائص القابلة للتعداد في الكائن، بما في ذلك الخصائص الموروثة والخصائص غير الرقمية.

:::interactive_editor

```js
const fruit = {
  name: 'apple',
  color: 'red',
  price: 0.99
};

for (const prop in fruit) {
  console.log(fruit[prop]);
}
```

:::

- **حلقة `while`**: ستشغل هذه الحلقة كتلة شفرة طالما كان الشرط `true`.

:::interactive_editor

```js
let i = 5;

while (i > 0) {
  console.log(i);
  i--;
}
```

:::

- **حلقة `do...while`**: ستنفذ هذه الحلقة كتلة الشفرة مرة واحدة على الأقل قبل التحقق من الشرط.

```js
let userInput;

do {
  userInput = prompt("Please enter a number between 1 and 10");
} while (Number(userInput) < 1 || Number(userInput) > 10);

alert("You entered a valid number!");
```

## جملتا `break` و `continue`

- **التعريف**: تُستخدم جملة `break` للخروج المبكر من الحلقة، بينما تُستخدم جملة `continue` لتخطي التكرار الحالي من الحلقة والانتقال إلى التكرار التالي.

:::interactive_editor

```js
// Example of break statement
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, and 4

// Example of continue statement 
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, 4, 6, 7, 8, and 9
```

:::
