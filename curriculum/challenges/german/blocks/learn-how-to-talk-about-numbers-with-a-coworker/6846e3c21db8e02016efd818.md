### --feedback--

Das erste Wort ist eine Ordnungszahl, und das zweite bezieht sich auf eine Phase im Projekt.

# --explanation--

`By when...` wird häufig verwendet, um zu fragen, wann etwas fertiggestellt sein wird. Dies ist eine höfliche Art, nach einer Zeitschätzung oder einem Termin zu fragen. Zum Beispiel:

`By when do you need the report?` — Bis wann möchten Sie den Bericht spätestens?

`Third` ist eine Ordnungszahl, die die Position in einer Folge angibt. `Phase` bedeutet eine Phase eines Projekts, daher bezieht sich `third phase` auf den dritten großen Teil ihres Projektplans.
