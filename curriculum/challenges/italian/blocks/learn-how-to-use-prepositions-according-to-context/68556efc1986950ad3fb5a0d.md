# --explanation--

`Thorough` significa fatto con grande cura e attenzione ai dettagli. Per esempio:

`She gave the code a thorough review.` – Questo significa che l'ha esaminato molto attentamente.

`Coverage` si riferisce a quanto di qualcosa è incluso, testato o gestito. Per esempio:

`The testing coverage for the new feature was excellent.` – Questo significa che molte parti della funzionalità sono state testate.
