# --explanation--

`To depend on` показує, що щось визначається іншим фактором. Наприклад:

- `The success of the project depends on good teamwork.` - Проєкт вдасться лише якщо команда добре працюватиме разом.

- `Whether we finish today depends on how many bugs we find.` Якщо вони знайдуть багато помилок, ви можете не закінчити сьогодні.
