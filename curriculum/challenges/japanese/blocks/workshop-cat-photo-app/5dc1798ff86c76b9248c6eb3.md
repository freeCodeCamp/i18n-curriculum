# --description--

`h1` 要素の下に、次のテキストを持つ `h2` 要素を追加してください：

`Cat Photos`

# --hints--

`h1` 要素には開始タグが必要です。開始タグの構文は次の通りです：`<elementName>`。

```js
assert.exists(document.querySelector('h1'));
```

`h1` 要素には終了タグが必要です。終了タグの構文は次の通りです：`</elementName>`。

```js
assert.match(code, /<\/h1\>/);
```

`h1` 要素は1つだけにしてください。余分なものは削除してください。

```js
assert.lengthOf(document.querySelectorAll('h1'), 1);
```

`h1` 要素のテキストは 'CatPhotoApp' であるべきです。テキストが抜けているか、タイプミスがあります。

```js
assert.equal(document.querySelector('h1')?.innerText?.trim().toLowerCase(), 'catphotoapp');
```

`h2` タグは小文字であるべきです。慣例として、すべてのHTMLタグは小文字で書かれます。

```js
assert.notMatch(code, /<\/?H2>/);
```

`h2` 要素には開始タグが必要です。開始タグの構文は次の通りです：`<elementName>`。

```js
assert.exists(document.querySelector('h2'));
```

`h2` 要素には終了タグが必要です。終了タグは `/` 文字の直後に `<` があります。

```js
assert.match(code, /<\/h2\>/);
```

`h2` 要素のテキストは `Cat Photos` であるべきです。開閉 `Cat Photos` タグの間には `h2` のテキストだけを置いてください。

```js
assert.equal(document.querySelector('h2')?.innerText?.trim().replace(/\s+/g, ' ').toLowerCase(), 'cat photos');
```

`h2` 要素は `h1` 要素の下にあるべきです。`h1` 要素はより重要で、`h2` 要素の上にある必要があります。

```js
const collection = [...document.querySelectorAll('h1,h2')].map(
  (node) => node.nodeName
);
assert.isBelow(collection.indexOf('H1'), collection.indexOf('H2'));
```
