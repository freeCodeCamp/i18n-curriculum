# --explanation--

Ángela 说 `¿Y tu profesión?`，这是一个简短的后续问题。`Y` 将这个问题与前一个话题连接起来。

尽管她没有说 `¿Cuál es...?`，但完整的问题可以从上下文中理解：

`¿Cuál es tu profesión?`

在对话中，当意思明确时，西班牙语经常省略重复的词。
