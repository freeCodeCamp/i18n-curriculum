# --description--

القوة المفرطة أو التتريشن لعدد $a$ بواسطة عدد صحيح موجب $b$، المشار إليها بـ $a↑↑b$ أو ${}^ba$، تُعرّف بشكل تكراري كما يلي:

$a↑↑1 = a$,

$a↑↑(k+1) = a^{(a↑↑k)}$.

لذا لدينا مثلاً $3↑↑2 = 3^3 = 27$، ومن ثم $3↑↑3 = 3^{27} = 7625597484987$ و$3↑↑4$ تقريبًا ${10}^{3.6383346400240996 \times {10}^{12}}$. أوجد آخر 8 أرقام من $1777↑↑1855$.
