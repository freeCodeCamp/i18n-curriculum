# --description--

파이썬 변수로 작업하기 전에 데이터 타입을 이해하는 것이 중요합니다. 데이터 타입은 변수가 어떤 종류의 값을 가지는지 설명합니다. 예를 들어, 숫자, 텍스트 조각, 또는 항목 목록 등이 있습니다. 프로그래밍 언어는 다양한 종류의 정보를 어떻게 저장하고 다룰지 알기 위해 데이터 타입을 사용합니다.

파이썬은 JavaScript처럼 동적 타이핑 언어로, 변수에 대해 타입을 명시적으로 선언할 필요가 없습니다. 변수에 할당하는 값에 따라 언어가 자동으로 데이터 타입을 인식합니다.

다음은 몇 가지 예시입니다:

```python
name = 'John Doe' # Python knows this is a string
age = 25 # Python knows this is an integer
```

이는 C#, Java, C++ 같은 정적 타이핑 언어와 대조적입니다. 이런 언어에서는 변수와 함께 타입을 선언해야 합니다:

```csharp
string name = 'John Doe'
int age = 25
```

파이썬의 동적 타이핑 특성은 코딩을 빠르고 유연하게 하지만, 타입 오류가 프로그램이 실행될 때만 발견되어 예상치 못한 버그가 발생할 수 있습니다.

파이썬은 프로그램이 **실행되는 동안** 데이터 타입을 결정하기 때문에, 타입 관련 실수는 그 순간에만 발견됩니다. 프로그램이 실행되면 파이썬은 코드를 한 줄씩 실행합니다. 특정 객체가 기대한 방식으로 동작하지 못하는 줄에 도달하면 파이썬은 멈추고 오류를 보여줍니다.

반면, 일부 언어는 프로그램을 실행 전에 **컴파일**합니다. 컴파일은 컴퓨터가 코드를 미리 검사하고 실행 준비를 하는 과정입니다. 이 단계에서 타입 오류를 프로그램 실행 전에 잡아낼 수 있습니다.

아직 그런 언어를 배울 필요는 없습니다. 중요한 점은 다음과 같습니다:

- **파이썬에서는 타입 오류가 프로그램이 실제로 실행될 때 드러날 수 있습니다.**

- **컴파일 언어는 프로그램 실행 전에 컴파일 단계에서 타입 오류를 잡아냅니다.**

이 때문에 파이썬에서는 프로그램이 특정 코드 줄에 도달할 때까지 타입 실수를 알지 못할 수 있습니다.

파이썬에서 가장 흔히 사용하는 데이터 타입은 다음과 같습니다:

- 정수: 소수점 없는 전체 숫자, 예를 들어 `10` 또는 `-5`.

```python
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- 부동소수점: 소수점이 있는 숫자, 예를 들어 `4.41` 또는 `-0.4`.

```python
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- 문자열: 작은따옴표나 큰따옴표로 감싼 문자 수열, 예를 들어 `'Hello world!'`.

```python
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- 불리언: 참 또는 거짓 타입, `True` 또는 `False`로 표현.

```python
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- 집합: 순서가 없고 중복 없는 요소들의 모음, 예를 들어 `{0.5, 4, 'apple'}`.

```python
my_set_var = {7, 'hello', 8.5}
print('Set:', my_set_var) # Set: {7, 'hello', 8.5}
```

- 사전: 중괄호로 감싼 키-값 쌍들의 모음, 예를 들어 `{'name': 'John Doe', 'age': 28}`.

```python
my_dictionary_var = {'name': 'Alice', 'age': 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- 튜플: 소괄호로 감싼 변경 불가능한 순서 있는 모음, 예를 들어 `('apple', 4.5, 7)`.

```python
my_tuple_var = (7, 'hello', 8.5)
print('Tuple:', my_tuple_var) # Tuple: (7, 'hello', 8.5)
```

- 범위: 주로 반복문에서 사용하는 숫자 수열, 예를 들어 `range(5)`.

```python
my_range_var = range(5)
print('Range:', my_range_var) # Range: range(0, 5)
```

- 리스트: 다양한 데이터 타입을 지원하는 순서 있는 요소 모음.

```python
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- None: 값이 없음을 나타내는 특별한 값.

```python
my_none_var = None
print('None:', my_none_var) # None: None
```

변수의 데이터 타입을 확인하려면 `type()` 함수를 사용할 수 있습니다:

```python
my_var_1 = 'Hello world'
my_var_2 = 21

print(type(my_var_1)) # <class 'str'>
print(type (my_var_2)) # <class 'int'>
```

다음은 이 수업에서 다룬 모든 데이터 타입과 터미널에서 확인한 타입입니다:

```python
my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 'hello', 8.5}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 'hello', 8.5)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>
```

내장된 `isinstance()` 함수는 변수가 특정 데이터 타입과 일치하는지 확인할 수 있게 해줍니다. 객체와 확인할 타입을 받아 불리언 값을 반환합니다. 다음은 몇 가지 예시입니다:

```python
isinstance('Hello world', str) # True
isinstance(True, bool) # True
isinstance(42, int) # True
isinstance('John Doe', int) # False
```
