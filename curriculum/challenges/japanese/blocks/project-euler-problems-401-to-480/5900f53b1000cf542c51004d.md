# --description--

3スムーズ数とは、素因数が3より大きくない整数のことです。整数 $N$ に対して、$S(N)$ を $N$ 以下の3スムーズ数の集合と定義します。例えば、$S(20) = \\{1, 2, 3, 4, 6, 8, 9, 12, 16, 18\\}$ です。

$F(N)$ を、$S(N)$ の要素がすべての真の約数の後に来るような順列の数と定義します。

$N = 20$ の場合の可能な順列の一例は次の通りです。

-   1, 2, 4, 3, 9, 8, 16, 6, 18, 12.

これは無効な順列です。なぜなら12が約数である6の前に来ているからです。

-   1, 2, 4, 3, 9, 8, 12, 16, 6, 18.

$F(6) = 5$, $F(8) = 9$, $F(20) = 450$, $F(1000) ≈ 8.8521816557e\\,21$ であることが確認できます。

$F({10}^{18})$ を求めてください。答えは、小数点以下10桁で四捨五入した科学的表記の文字列で提出してください。答えを示す際は、仮数部と指数部の区切りに小文字の `e` を使ってください。例えば答えが $112\\,233\\,445\\,566\\,778\\,899$ の場合、答えの形式は `1.1223344557e17` となります。
