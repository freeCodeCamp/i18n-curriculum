# --description--

要宣告一個常數（不變的）變數，請使用 `const` 而不是 `let`：

```javascript
const myBirthday = '16-09-2003';
```

使用 `const` 宣告的變數稱為「常數」。它們無法被重新指定。嘗試這麼做會導致錯誤：

```javascript
const myBirthday = '16-09-2003';

myBirthday = '01-01-2001'; // error, can't reassign the constant!
```

當程式設計師確定一個變數永遠不會改變時，他們可以使用 `const` 宣告它，以保證並向所有人傳達這個事實。
