# --description--

我們說一個有 $d$ 位數的正整數（無前導零）是單子數，如果它的子字串中恰好有一個能被 $d$ 整除。

例如，5671 是一個 4 位數的一子數。在它所有的子字串 5、6、7、1、56、67、71、567、671 和 5671 中，只有 56 可被 4 整除。

同樣地，104 是一個 3 位數的一子數，因為只有 0 可被 3 整除。1132451 是一個 7 位數的一子數，因為只有 245 可被 7 整除。

設 $F(N)$ 為小於 $N$ 的一子數量。我們可以驗證 $F(10) = 9$、$F({10}^3) = 389$ 以及 $F({10}^7) = 277\\,674$。

尋找 $F({10}^{19})$。
