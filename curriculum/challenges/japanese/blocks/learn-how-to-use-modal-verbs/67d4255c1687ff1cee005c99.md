## --text--

どの文が文法的に正しいですか？

## --answers--

`Everyone must to follow the security guidelines.`

### --feedback--

`Must` の後には `to` なしで動詞の原形が続きます。

---

`Everyone must following the security guidelines.`

### --feedback--

`Must` の後には動詞の原形が続くべきで、`-ing` 形ではありません。

---

`Everyone must followed the security guidelines.`

### --feedback--

`Must` の後には過去形ではなく動詞の原形が続きます。

---

`Everyone must follow the security guidelines.`

## --video-solution--

4

# --explanation--

助動詞の後には常に動詞の原形が続きます。助動詞は `to`、`-ing`、または過去形の形を取りません。以下は誤った動詞の形の例です：

- `Must to follow` は誤りです。なぜなら `must` は `to` を取るからです。

- `Must following` は誤りです。なぜなら `must` の後に `-ing` 形が続いているからです。

- `Must followed` は誤りです。なぜなら `followed` が過去形だからです。
