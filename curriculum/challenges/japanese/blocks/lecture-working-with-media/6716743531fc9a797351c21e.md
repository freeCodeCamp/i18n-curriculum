# --interactive--

置き換えられた要素とは、その内容がCSS自体ではなく外部リソースによって決まる要素のことです。CSS（カスケーディングスタイルシート）はウェブページにスタイルを追加するために使われます。置き換えられた要素の一般的な例には、image、iframe、video要素があります。

置き換えられた要素では、要素の位置やレイアウトを制御できます。しかし、CSSでその要素の内容を直接変更することはできません。いくつかの例で説明するとわかりやすいでしょう。例えば、image要素はウェブページに画像を埋め込みます。

```html
<img src="example-img-url" alt="Descriptive text goes here">
```

要素自体は外部オブジェクト、つまり画像に置き換えられます。CSSで画像の位置を制御したり、フィルターを適用したりできますが、画像自体を変更することはできません。より実用的な例としては、`iframe`要素があり、これは外部サイトをウェブページに埋め込みます。

こちらはfreeCodeCampチャンネルの人気YouTube動画を`iframe`要素で埋め込んだ例です。プレビューウィンドウで別の動画を見たい場合は、`src`属性の値をお好みの動画に変更してください。プレビューを見るにはインタラクティブエディターを有効にする必要があります。

**注意**：インタラクティブ例で出てくる`referrerpolicy`や`allowfullscreen`のような追加属性は気にしなくて大丈夫です。`iframe`要素の扱いについては今後のワークショップで学びます。

:::interactive_editor

```html
<iframe width="400" height="200" src="https://www.youtube.com/embed/u43gJJrVa1I?si=BoDW_puFsy8OEr_Z" title="Professional Cloud Architect Certification Course – Pass the Exam! (YouTube video)" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen></iframe>
```

:::

`iframe`要素を使う他の一般的な例としては、ページに地図を埋め込むことがあります。こちらは埋め込み地図の例です。

インタラクティブエディターを有効にして、地図をズームイン・ズームアウトして操作してみてください。

:::interactive_editor

```html
<iframe
  title="Map of the Royal Observatory, Greenwich, London"
  width="300"
  height="200"
  src="https://www.openstreetmap.org/export/embed.html?bbox=-0.004017949104309083%2C51.47612752641776%2C0.00030577182769775396%2C51.478569861898606&amp;layer=mapnik">
</iframe>
```

:::

要素自体は外部オブジェクト、つまりサイトに置き換えられます。CSSで埋め込まれたサイトの位置は変更できますが、サイトの内容を変更することはできません。さらに詳しく言うと、埋め込まれたサイトに`h1`要素があっても、その`h1`要素をCSSでスタイル変更することはできません。サイズやフォントの色などを変えることはできません。

他にも`video`や`embed`のような置き換えられた要素があります。また、特定の条件下で置き換えられた要素として振る舞う要素もあります。例えば、`input`要素に`type`属性が`image`に設定されている例です。

```html
<input type="image" alt="Descriptive text goes here" src="example-img-url">
```

このタイプの`input`は置き換えられた要素とみなされますが、`input`や`text`のような他の`email`タイプは置き換えられた要素ではありません。
