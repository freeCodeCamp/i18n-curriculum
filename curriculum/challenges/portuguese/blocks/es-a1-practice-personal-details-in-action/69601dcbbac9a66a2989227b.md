# --description--

Você aprendeu recentemente o padrão `¿Quieres que lo deletree?` para oferecer soletrar algo. Outra pergunta comum é:

`¿Cómo se deletrea?`

Esta pergunta é usada quando alguém quer saber como uma palavra, nome ou e-mail é soletrado letra por letra.

É muito comum em conversas onde as pessoas estão confirmando detalhes de contato.

# --explanation--

O áudio diz `¿Cómo se deletrea?`. Esta pergunta está perguntando como algo é soletrado. A resposta correta deve, portanto, soletrar a informação letra por letra.

Apenas uma opção faz isso: `j-u-l-i-e-t-a punto c-a-c-e-r-e-s arroba a-n-d-e-s punto com.`

As outras opções são declarações completas (name, nationality, profession) e não fornecem ortografia.
