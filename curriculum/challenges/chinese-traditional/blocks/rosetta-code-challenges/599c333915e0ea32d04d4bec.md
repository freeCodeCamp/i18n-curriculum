# --description--

實作基本的元素對元素矩陣-矩陣和純量-矩陣操作。

**實作：**

<ul>
  <li>addition</li>
  <li>subtraction</li>
  <li>multiplication</li>
  <li>division</li>
  <li>exponentiation</li>
</ul>

第一個參數將是要執行的操作行為，例如，"m_add" 代表矩陣加法，"s_add" 代表純量加法。第二和第三個參數將是要執行操作行為的矩陣。

```js
function operation(op, arr1, arr2) {
  const ops = {
    add: ((a, b) => a + b),
    sub: ((a, b) => a - b),
    mult: ((a, b) => a * b),
    div: ((a, b) => a / b),
    exp: ((a, b) => Math.pow(a, b))
  };
  const ifm = op.startsWith('m');
  const doOp = ops[op.substring(2)];
  for (let i = 0; i < arr1.length; i++) {
    for (let j = 0; j < arr1[0].length; j++) {
      arr1[i][j] = doOp(arr1[i][j], (ifm) ? (arr2[i][j]) : (arr2));
    }
  }
  return arr1;
}
```
