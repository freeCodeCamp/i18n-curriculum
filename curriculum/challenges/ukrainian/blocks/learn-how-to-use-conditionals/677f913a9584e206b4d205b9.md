# --explanation--

`Present Simple` час часто описує дії або ситуації, які є істинними завжди, а не тимчасовими випадками. Наприклад:

- `The system updates automatically every night.` — описує заплановану рутину, що відбувається регулярно.

- `The app saves data to the cloud.` — стверджує загальний факт про функціональність застосунку.

`Securely` означає, що щось закріплене, розташоване або зроблене так, щоб забезпечити безпеку або правильне функціонування.

У цьому реченні `isn't connected securely` описує стан, коли кабель живлення приєднаний неправильно, через що комп’ютер не вмикається.
