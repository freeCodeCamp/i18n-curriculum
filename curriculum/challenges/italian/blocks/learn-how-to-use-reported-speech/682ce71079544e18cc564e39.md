# --explanation--

`To prevent` significa agire per evitare che qualcosa accada. Per esempio:

`They installed new software to prevent viruses.` – Questo significa che il software aiuta a fermare i virus prima che causino problemi.

Un `incident` è un evento inaspettato, spesso negativo o problematico. Per esempio:

`There was a security incident last night that shut down the server.` – Questo significa che è successo qualcosa che ha causato problemi al sistema.
