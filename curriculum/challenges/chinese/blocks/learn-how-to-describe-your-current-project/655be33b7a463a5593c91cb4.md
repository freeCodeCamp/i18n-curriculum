# --description--

当你确定某事时，你说它是 `definite`。 

当你为 `-ly` 添加后缀时，`definite` 变成副词 `definitely`，意思是对某事确定或有信心。 

它通常用于强调说话者认为是真实的或很可能真实的内容。例如：

- `I can definitely see why this is important.`——这强调了重要性应该是显而易见的。

- `She will definitely finish the project on time.`——这显示了对她按时完成项目的信心。
