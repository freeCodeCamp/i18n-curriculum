# --explanation--

Il tempo `Present Continuous` spesso sottolinea una condizione temporanea o un problema che si verifica nel momento in cui si parla. Per esempio:

- `The server isn't running diagnostics.` - Evidenzia che il processo di diagnostica non sta avvenendo in questo momento.

- `Developers aren't debugging the application.` - Mostra che gli sviluppatori non stanno lavorando a quell'attività al momento.

`Properly` significa fare qualcosa nel modo corretto o previsto.

In questa frase, `aren't booting properly` indica che i computer attualmente non si avviano correttamente.
