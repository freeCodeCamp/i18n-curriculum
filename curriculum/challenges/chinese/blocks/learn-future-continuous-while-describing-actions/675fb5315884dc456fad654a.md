# --explanation--

`Every month` 意味着某事每月发生一次，定期发生。例如：

`We back up our system data every month to prevent data loss.`——这句话表明备份系统数据是一个每月进行的常规任务，以确保数据安全。

`每两周一次` 意味着某事每两周发生一次，或者每月发生两次。例如：

`The IT team checks for security updates every two weeks to keep systems up to date.` — 这句话强调了每月两次查看更新以确保系统保持安全和正常运行的重要性。

`Vigilant` 意味着保持警惕和小心，以避免风险或问题。例如：

`Employees must stay vigilant when opening emails to avoid phishing scams.`
