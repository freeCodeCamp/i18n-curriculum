# --description--

在函式（程式）內宣告的變數只在函式（程式）內可見。這稱為變數的範圍，這個特定的範圍稱為區域的範圍。在函式（程式）外宣告的變數稱為全域變數，並且在整個程式中可見。例如，以下程式碼將引發錯誤：

```js
function myFunction() {
  let x = 10;
}

console.log(x);
```

變數 `x` 在函式（程式）`myFunction` 內宣告，且在函式外不可見。程式碼會引發 `ReferenceError`，因為未定義 `x`。

如果你在函式外宣告變數，該變數稱為全域變數，並且在整個程式中可見。 例如，以下程式碼將會運作：

```js
let x = 10;

function myFunction() {
  console.log(x);
}

myFunction();
```

如果在函式（程式）內宣告了同名的變數，則會遮蔽外層的變數。
