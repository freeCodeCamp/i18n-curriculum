# --description--

Der Ausdruck `only if` wird verwendet, um eine Bedingung dafür festzulegen, dass etwas passiert. Es ist wie zu sagen: „Das wird passieren, wenn das passiert“. Es ist eine Art Vereinbarung, bei der eine Handlung von einer anderen abhängt.

Zum Beispiel könnte Ihr Freund sagen, `I'll go out only if it doesn't rain`. Das bedeutet, dass er ausgehen wird, aber das Wetter muss gut sein – kein Regen.
