# --description--

**목표:** 다음과 기능적으로 유사한 앱을 만드세요: <a href="https://choropleth-map.freecodecamp.rocks" target="_blank" rel="noopener noreferrer nofollow">https://choropleth-map.freecodecamp.rocks</a>.

아래 사용자 스토리를 충족하고 모든 테스트를 통과하세요. 필요한 라이브러리나 API를 자유롭게 사용하세요. 자신만의 스타일을 적용해도 좋습니다.

HTML, JavaScript, CSS, 그리고 D3 svg 기반 시각화 라이브러리를 사용할 수 있습니다. 필요한 DOM 요소는 각 테스트 시점에 쿼리됩니다. 만약 Vue 같은 프론트엔드 프레임워크를 사용하면 동적 콘텐츠에 대해 테스트 결과가 정확하지 않을 수 있습니다. 앞으로 지원할 계획이지만 현재 D3 프로젝트에서는 이러한 프레임워크가 지원되지 않습니다.

**사용자 스토리 #1:** 내 코로플레스 지도에는 `id="title"`에 해당하는 제목이 있어야 합니다.

**사용자 스토리 #2:** 내 코로플레스 지도에는 `id="description"`에 해당하는 설명 요소가 있어야 합니다.

**사용자 스토리 #3:** 내 코로플레스 지도에는 데이터를 나타내는 `class="county"`에 해당하는 군(County)들이 있어야 합니다.

**사용자 스토리 #4:** 군들에 최소 4가지 이상의 서로 다른 채우기 색상이 사용되어야 합니다.

**사용자 스토리 #5:** 내 군 각각에는 해당 `data-fips`와 `data-education` 값을 포함하는 `fips`와 `education` 속성이 있어야 합니다.

**사용자 스토리 #6:** 내 코로플레스 지도에는 제공된 데이터 포인트마다 군이 있어야 합니다.

**사용자 스토리 #7:** 군들은 샘플 데이터와 일치하는 `data-fips`와 `data-education` 값을 가져야 합니다.

**사용자 스토리 #8:** 내 코로플레스 지도에는 `id="legend"`에 해당하는 범례가 있어야 합니다.

**사용자 스토리 #9:** 범례에는 최소 4가지 이상의 서로 다른 채우기 색상이 사용되어야 합니다.

**사용자 스토리 #10:** 마우스를 영역 위에 올리면 해당 영역에 대한 추가 정보를 표시하는 `id="tooltip"`에 해당하는 툴팁이 보여야 합니다.

**사용자 스토리 #11:** 내 툴팁에는 활성 영역의 `data-education`에 해당하는 `data-education` 속성이 있어야 합니다.

이 프로젝트를 완료하는 데 필요한 데이터셋은 다음과 같습니다:

- **미국 교육 데이터:**`https://cdn.freecodecamp.org/testable-projects-fcc/data/choropleth_map/for_user_education.json`
- **미국 군 데이터:**`https://cdn.freecodecamp.org/testable-projects-fcc/data/choropleth_map/counties.json`

<a href='https://codepen.io/pen?template=MJjpwO' target="_blank" rel="noopener noreferrer nofollow">이 CodePen 템플릿</a>을 사용해 프로젝트를 만들고 `Save`를 클릭해 자신만의 펜을 생성할 수 있습니다. 또는 이 CDN 링크를 사용해 원하는 환경에서 테스트를 실행할 수 있습니다: `https://cdn.freecodecamp.org/testable-projects-fcc/v1/bundle.js`

완료하면 모든 테스트를 통과하는 작동하는 프로젝트 URL을 제출하세요.
