# --description--

2つの正の整数 $a$ と $b$ に対して、ウラム数列 $U(a,b)$ は ${U{(a,b)}\_1} = a$、${U{(a,b)}\_2} = b$ と定義され、$k > 2$ の場合、${U{(a,b)}\_k}$ は ${U{(a,b)}\_{(k-1)}}$ より大きい最小の整数で、$U(a,b)$ の異なる2つの以前のメンバーの和としてちょうど1通りの方法で表せるものです。

例えば、数列 $U(1,2)$ は次のように始まります。

$$1, 2, 3 = 1 + 2, 4 = 1 + 3, 6 = 2 + 4, 8 = 2 + 6, 11 = 3 + 8$$

5は含まれません。なぜなら $5 = 1 + 4 = 2 + 3$ と、以前のメンバー2つの和として2通りの表し方があるからです。同様に $7 = 1 + 6 = 3 + 4$ も含まれません。

$2 ≤ n ≤ 10$ のとき、$k = {10}^{11}$ に対して $\sum {U(2, 2n + 1)_k}$ を求めてください。
