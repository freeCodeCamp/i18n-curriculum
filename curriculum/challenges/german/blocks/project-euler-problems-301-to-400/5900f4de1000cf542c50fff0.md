# --description--

In einem Standardkartenspiel mit 52 Karten ist eine Menge von 4 Karten ein Badugi, wenn sie 4 Karten enthält, ohne Paare und ohne zwei Karten derselben Farbe.

Sei $f(n)$ die Anzahl der Möglichkeiten, $n$ Karten auszuwählen, die eine 4-Karten-Teilmenge enthalten, die ein Badugi ist. Zum Beispiel gibt es $2\\,598\\,960$ Möglichkeiten, fünf Karten aus einem Standardkartenspiel mit 52 Karten auszuwählen, von denen $514\\,800$ eine 4-Karten-Teilmenge enthalten, die ein Badugi ist, also ist $f(5) = 514800$.

Berechnen Sie $\sum f(n)$ für $4 ≤ n ≤ 13$.
