# --explanation--

`Buenas noches` es un saludo usado por la noche. Se traduce como "Good night".

Puedes usar `Buenas noches` tanto para saludar a alguien por la noche **como** para saludar o despedirte de alguien durante la noche. Por ejemplo, la siguiente oración funciona tanto para saludar como para despedirse durante la noche:

`Buenas noches, señora López.` – Buenas noches, señora López.
