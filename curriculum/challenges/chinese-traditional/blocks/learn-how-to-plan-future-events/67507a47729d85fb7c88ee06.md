# --explanation--

`Is planned` 使用 `Present Simple` 時態來描述已經排定的未來事件。它顯示對訓練何時發生的確定性，而不是表示訓練已經發生。 例如：

`The meeting is planned for next Monday.`－這表示會議已排定在未來。
