# --description--

Websites wären ziemlich langweilig, wenn sie nur Text anzeigen könnten. Glücklicherweise bietet HTML eine große Vielfalt an Elementen, um alle möglichen Medien darzustellen. Das am häufigsten verwendete davon ist das Bildelement.

Um ein Bild in HTML anzuzeigen, verwenden Sie das `<img>`-Element. Im Gegensatz zu den anderen Elementen, die Sie kennengelernt haben, ist das `<img>` ein Void-Element. Leere, Void-HTML-Elemente benötigen kein schließendes Tag.

Anstatt Inhalte mit einem öffnenden und schließenden Tag einzuschließen, bettet es ein Bild in die Seite ein, indem es ein `src`-Attribut verwendet, das dem Browser mitteilt, wo sich die Bilddatei befindet. Das `src`-Attribut funktioniert ähnlich wie das `href`-Attribut für Anker-Tags. Es kann ein Bild sowohl mit absoluten als auch relativen Pfaden einbetten.

Zum Beispiel können Sie mit einem absoluten Pfad ein Bild anzeigen, das auf der Website von The Odin Project liegt:

Um Bilder zu verwenden, die Sie auf Ihren eigenen Websites haben, können Sie einen relativen Pfad verwenden.

- Erstellen Sie ein neues Verzeichnis namens `images` innerhalb des `odin-links-and-images`-Projekts.

- Laden Sie als Nächstes <a href="https://unsplash.com/photos/Mv9hjnEUHR4/download?force=true&w=640" target="_blank">dieses Bild</a> herunter und verschieben Sie es in das gerade erstellte `images`-Verzeichnis.

- Benennen Sie das Bild in `dog.jpg` um.

Fügen Sie schließlich das Bild in die `index.html`-Datei ein:

```html
<body>
  <h1>Homepage</h1>
	<a href="https://www.theodinproject.com/about">click me</a>

	<a href="pages/about.html">About</a>

	<img src="images/dog.jpg">
</body>
```

Speichern Sie die `index.html`-Datei und öffnen Sie sie in einem Browser, um Charles in voller Pracht zu sehen.


## Übergeordnete Verzeichnisse

Was ist, wenn Sie das Hundebild auf der `about`-Seite verwenden möchten? Sie müssten zuerst eine Ebene nach oben aus dem pages-Verzeichnis in dessen übergeordnetes Verzeichnis wechseln, um dann auf das images-Verzeichnis zugreifen zu können.

Um in das übergeordnete Verzeichnis zu wechseln, müssen Sie im relativen Dateipfad zwei Punkte wie folgt verwenden: `../`. Sehen wir uns das in der Praxis an: Fügen Sie im `body` der `about.html`-Datei das folgende Bild unterhalb der zuvor hinzugefügten Überschrift ein:

```html
<img src="../images/dog.jpg">
```

Zur Erklärung:

- Zuerst wechseln Sie in das übergeordnete Verzeichnis des pages-Verzeichnisses, also `odin-links-and-images`.

- Dann können Sie vom übergeordneten Verzeichnis aus in das `images`-Verzeichnis wechseln.

- Schließlich können Sie auf die `dog.jpg`-Datei zugreifen.

Wenn wir die zuvor verwendete Metapher nutzen, ist die Verwendung von `../` in einem Dateipfad so, als würden Sie den Raum, in dem Sie sich gerade befinden, verlassen und in den Hauptflur treten, um in einen anderen Raum zu gelangen.

## `Alt`-Attribut

Neben dem `src`-Attribut sollte jedes Bildelement auch ein `alt` (alternativer Text) Attribut haben.

Das `alt`-Attribut wird verwendet, um ein Bild zu beschreiben. Es wird anstelle des Bildes angezeigt, wenn dieses nicht geladen werden kann. Außerdem wird es mit Screenreadern verwendet, um sehbehinderten Nutzenden zu beschreiben, was das Bild zeigt.
