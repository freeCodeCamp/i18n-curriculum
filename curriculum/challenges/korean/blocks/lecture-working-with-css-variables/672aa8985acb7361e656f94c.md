# --interactive--

CSS 커스텀 속성, 즉 CSS 변수는 CSS 작성자가 문서 전체에서 재사용할 특정 값을 담기 위해 정의하는 항목입니다. 이는 더 효율적이고 유지 관리가 쉽고 유연한 스타일시트를 만들 수 있게 해주는 강력한 기능입니다.

커스텀 속성을 선언하는 구문은 간단합니다. 두 개의 대시(`--`)로 시작하고 그 뒤에 속성 이름이 옵니다:

```css
:root {
  --main-color: #3498db;
}
```

이 예제에서는 `--main-color`이라는 이름의 커스텀 속성을 `#3498db` 값으로 선언하고 있습니다. `:root` 가상 클래스는 DOM 트리에서 가장 상위 부모를 나타내므로 전역 커스텀 속성을 선언할 때 자주 사용됩니다.

커스텀 속성을 사용하려면 `var()` 함수를 사용합니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
:root {
  --main-color: #3498db;
}

.button {
  background-color: var(--main-color);
}
```

:::

이 코드는 `button` 클래스를 가진 요소들의 배경색을 `--main-color`에 저장된 값으로 설정합니다.

커스텀 속성의 주요 기능 중 하나는 CSS 캐스케이드를 따른다는 점입니다. 즉, 특정 요소나 상황에 맞게 다시 정의할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="alert">This is an alert message.</div>
```

```css
.alert {
  --main-color: #e74c3c;
  background-color: var(--main-color);
}
```

:::

이 경우 `alert` 클래스를 가진 요소들은 전역 정의를 덮어쓰고 다른 `--main-color` 값을 사용합니다.

커스텀 속성은 대체 수단도 지원합니다. 커스텀 속성이 정의되지 않았거나 유효하지 않을 때 대체 값을 제공할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="text">This is some text.</div>
```

```css
:root {
  --text-color: green;
}

.text {
  color: var(--text-color, green);
}
```

:::

여기서 `--text-color`가 정의되지 않으면 `color`은 기본값으로 `green`을 사용합니다.

커스텀 속성은 테마 가능한 디자인을 만들 때 특히 유용합니다. 서로 다른 테마에 대해 속성 집합을 정의할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="light-theme">
  <p>This is some light themed text.</p>
</div>
```

```css
:root {
  --bg-color: black;
}

body {
  background-color: var(--bg-color);
}

.light-theme {
  --bg-color: white;
  --text-color: black;
  background-color: var(--bg-color);
  color: var(--text-color);
  padding: 10px;
}

```

:::

테마를 전환하는 것은 `body` 요소에 클래스를 추가하거나 제거하는 것만큼 간단합니다.

커스텀 속성은 미디어 쿼리와 함께 사용하여 반응형 디자인을 만들 수도 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <h2>Responsive Design</h2>
  <p>Resize the window to see the card adapt!</p>
</div>

```

```css
:root {
  --card-width: 90%;
  --card-bg: #f0f0f0;
  --card-padding: 1rem;
  --text-color: #333;
}

/* Tablet screens and up */
@media (min-width: 600px) {
  :root {
    --card-width: 70%;
    --card-bg: #e8f5e9;
    --card-padding: 1.5rem;
  }
}

/* Desktop screens and up */
@media (min-width: 1024px) {
  :root {
    --card-width: 50%;
    --card-bg: #d0f0ff;
    --card-padding: 2rem;
  }
}

body {
  font-family: system-ui, sans-serif;
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
  background-color: #fafafa;
}

.card {
  width: var(--card-width);
  background-color: var(--card-bg);
  padding: var(--card-padding);
  color: var(--text-color);
  border-radius: 8px;
  box-shadow: 0 2px 6px rgba(0,0,0,0.1);
  text-align: center;
  transition: all 0.3s ease;
}

```

:::

이 방법은 스타일시트 전체에서 값을 바꾸는 대신 한 곳에서 값을 변경할 수 있어 유지 관리가 더 쉬운 반응형 레이아웃을 만듭니다. 특히 자주 사용되는 복잡한 값의 반복을 크게 줄일 수 있습니다.

커스텀 속성은 다른 커스텀 속성을 참조할 수도 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
:root {
  --primary-color: #3498db;
  --button-bg: var(--primary-color);
}

button {
  background-color: var(--button-bg);
}

```

:::

이렇게 하면 스타일의 다양한 측면 간 관계를 만들 수 있어 사이트 전반에 걸쳐 일관된 테마를 유지하기가 더 쉬워집니다.

커스텀 속성은 많은 장점을 제공하지만, 브라우저 지원 상황을 인지하는 것이 중요합니다. 최신 브라우저에서는 잘 지원되지만, 구형 브라우저에서는 인식하지 못할 수 있습니다. 최신 기능을 사용할 때는 항상 대체 수단을 준비하세요.

결론적으로, CSS 커스텀 속성은 더 동적이고 유연하며 유지 관리가 쉬운 스타일시트를 만드는 강력한 방법을 제공합니다. 테마 가능한 디자인을 만들고, 반응형 레이아웃을 단순화하며, 런타임에 스타일을 조작할 수 있게 해줍니다.

웹 개발이 계속 발전함에 따라 커스텀 속성은 개발자의 도구 상자에서 점점 더 중요한 도구가 되어 적응 가능하고 효율적인 CSS를 만드는 새로운 가능성을 열어줍니다.
