# --description--

`To skim` означає швидко прочитати або переглянути щось, щоб зрозуміти основні ідеї без фокусування на деталях. Наприклад, `She skimmed the article to understand the general topic.`

`To skim through` схоже за значенням, означає швидко пройтися по тексту або документу. Наприклад, `He skimmed through the manual to find the troubleshooting section.`
