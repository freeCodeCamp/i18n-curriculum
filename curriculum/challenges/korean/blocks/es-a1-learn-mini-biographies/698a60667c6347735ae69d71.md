# --description--

`Le`는 "그에게", "그녀에게", 또는 "당신에게"를 나타내는 대명사입니다.

`Le gusta`는 남성인 Diego를 가리킬 때 "그는 좋아한다"는 뜻입니다. 여성일 때도 같습니다. 예를 들어:

- `Le gusta el fútbol.` - 그는 축구를 좋아합니다 (남성을 가리킬 때).

- `Le gusta el fútbol.` - 그녀는 축구를 좋아합니다 (여성을 가리킬 때).
