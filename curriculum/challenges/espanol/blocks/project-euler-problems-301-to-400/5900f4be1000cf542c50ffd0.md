# --description--

Sea $\{a_1, a_2, \ldots, a_n\}$ una secuencia de enteros de longitud $n$ tal que:

- $a_1 = 6$
- for all $1 ≤ i &lt; n$ : $φ(a_i) &lt; φ(a_{i + 1}) &lt; a_i &lt; a_{i + 1}$

$φ$ denota la función totiente de Euler.

Sea $S(N)$ el número de tales secuencias con $a_n ≤ N$.

Por ejemplo, $S(10) = 4$: {6}, {6, 8}, {6, 8, 9} y {6, 10}.

Podemos verificar que $S(100) = 482\,073\,668$ y $S(10\,000)\bmod {10}^8 = 73\,808\,307$.

Encuentra $S(20\,000\,000)\bmod {10}^8$.
