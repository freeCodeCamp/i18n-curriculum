## Numberデータ型の扱い方

- **定義**: JavaScriptの`Number`型には整数、浮動小数点数、`Infinity`および`NaN`が含まれます。浮動小数点数は小数点を含む数値です。正の`Infinity`は他のどの数値よりも大きい数を表し、`-Infinity`は他のどの数値よりも小さい数を表します。`NaN`（`Not a Number`）は文字列`"Jessica"`のような無効な数値を表します。

## 一般的な算術演算

- **加算演算子**: この演算子（`+`）は2つ以上の数値の合計を計算するために使います。 
- **減算演算子**: この演算子（`-`）は2つの数値の差を計算するために使います。
- **乗算演算子**: この演算子（`*`）は2つ以上の数値の積を計算するために使います。 
- **除算演算子**: この演算子（`/`）は2つの数値の商を計算するために使います。
- **ゼロ除算**: もしゼロで割ろうとすると、JavaScriptは`Infinity`を返します。
- **余り演算子**: この演算子（`%`）は割り算の余りを返します。 
- **べき乗演算子**: この演算子（`**`）はある数値を別の数値のべき乗にします。

## 数値と文字列の計算

- **説明**: 数値と文字列に`+`演算子を使うと、JavaScriptは数値を文字列に型変換して2つの値を連結します。文字列と数値に`-`、`*`、または`/`演算子を使うと、JavaScriptは文字列を数値に型変換し、結果は数値になります。`null`と`undefined`の場合、JavaScriptは数学的演算で`null`を0として扱い、undefinedを`NaN`として扱います。

:::interactive_editor

```js
const result = 5 + '10';

console.log(result); // "510"
console.log(typeof result); // string

const subtractionResult = '10' - 5;
console.log(subtractionResult); // 5
console.log(typeof subtractionResult); // number

const multiplicationResult = '10' * 2;
console.log(multiplicationResult); // 20
console.log(typeof multiplicationResult); // number

const divisionResult = '20' / 2;
console.log(divisionResult); // 10
console.log(typeof divisionResult); // number

const result1 = null + 5;
console.log(result1); // 5
console.log(typeof result1); // number

const result2 = undefined + 5;
console.log(result2); // NaN
console.log(typeof result2); // number
```

:::

## 演算子の優先順位

- **定義**: 演算子の優先順位は式の中でどの順番で演算が評価されるかを決めます。優先順位が高い演算子は低いものより先に評価されます。括弧内の値は最初に評価され、乗算・除算は加算・減算より優先されます。もし演算子の優先順位が同じ場合は、JavaScriptは結合規則を使います。

:::interactive_editor

```js
const result = (2 + 3) * 4;

console.log(result); // 20

const result2 = 10 - 2 + 3;

console.log(result2); // 11

const result3 = 2 ** 3 ** 2;

console.log(result3); // 512
```

:::

- **定義**: 結合規則は同じ種類の演算子が複数ある場合に式がどの方向に評価されるかを示します。左から右（`left-associative`）か右から左（`right-associative`）かを定義します。例えば、べき乗演算子は右から左の結合規則です。

:::interactive_editor

```js
const result4 = 5 ** 4 ** 1; 

console.log(result4); // 625
```

:::

## インクリメント演算子とデクリメント演算子

- **インクリメント演算子**: この演算子は値を1増やすために使います。接頭辞表記の`++num`は変数の値を先に増やしてから新しい値を返します。後置表記の`num++`は変数の現在の値を先に返してから増やします。

:::interactive_editor

```js
let x = 5;

console.log(++x); // 6
console.log(x); // 6


let y = 5;

console.log(y++); // 5
console.log(y); // 6
```

:::

- **デクリメント演算子**: この演算子は値を1減らすために使います。接頭辞表記と後置表記はインクリメント演算子と同じ動作をします。

:::interactive_editor

```js
let num = 5;

console.log(--num); // 4
console.log(num--); // 4
console.log(num); // 3
```

:::

## 複合代入演算子

- **加算代入（`+=`）演算子**: この演算子は値の加算を行い、その結果を変数に代入します。
- **減算代入（`-=`）演算子**: この演算子は値の減算を行い、その結果を変数に代入します。
- **乗算代入（`*=`）演算子**: この演算子は値の乗算を行い、その結果を変数に代入します。
- **除算代入（`/=`）演算子**: この演算子は値の除算を行い、その結果を変数に代入します。
- **余り代入（`%=`）演算子**: この演算子は変数を指定した数で割り、その余りを変数に代入します。
- **べき乗代入（`**=`）演算子**: この演算子は変数を指定した数のべき乗にし、その結果を変数に再代入します。

## ブール値と等価性

- **ブール値の定義**: ブール値は`true`か`false`の2つの値だけを持つデータ型です。 
- **等価（`==`）演算子**: この演算子は値が等しいかをチェックする前に型変換を行います。

:::interactive_editor

```js
console.log(5 == '5'); // true
```

:::

- **厳密等価（`===`）演算子**: この演算子は型変換を行わず、型と値の両方が等しいかをチェックします。

:::interactive_editor

```js
console.log(5 === '5'); // false
```

:::

- **不等価（`!=`）演算子**: この演算子は値が等しくないかをチェックする前に型変換を行います。
- **厳密不等価（`!==`）演算子**: この演算子は型変換を行わず、型と値の両方が等しくないかをチェックします。

## 比較演算子

- **より大きい（`>`）演算子**: この演算子は左の値が右の値より大きいかをチェックします。
- **より大きいか等しい（`>=`）演算子**: この演算子は左の値が右の値以上かをチェックします。
- **より小さい（`<`）演算子**: この演算子は左の値が右の値より小さいかをチェックします。
- **より小さいか等しい（`<=`）演算子**: この演算子は左の値が右の値以下かをチェックします。

## 単項演算子

- **単項プラス演算子**: この演算子はオペランドを数値に変換します。もしオペランドがすでに数値なら変わりません。

:::interactive_editor

```js
const str = '42';
const num = +str;

console.log(num); // 42
console.log(typeof num); // number
```

:::

- **単項マイナス（`-`）演算子**: この演算子はオペランドの符号を反転させます。

:::interactive_editor

```js
const num = 4;
console.log(-num); // -4
```

:::

- **論理NOT（`!`）演算子**: この演算子はオペランドのブール値を反転させます。つまり、オペランドが`true`なら`false`になり、`false`なら`true`になります。

## ビット演算子

- **ビットAND（`&`）演算子**: この演算子は両方のオペランドの対応するビットが1の場合に1を返します。 
- **ビットAND代入（`&=`）演算子**: この演算子は指定した数値と`bitwise AND`演算を行い、その結果を変数に再代入します。
- **ビットOR（`|`）演算子**: この演算子はどちらかまたは両方のオペランドの対応するビットが1の場合に1を返します。 
- **ビットOR代入（`|=`）演算子**: この演算子は指定した数値と`bitwise OR`演算を行い、その結果を変数に再代入します。
- **ビットXOR（`^`）演算子**: この演算子はどちらか一方のオペランドの対応するビットが1の場合に1を返します（両方が1の場合は0）。
- **ビットNOT（`~`）演算子**: この演算子は数値の2進数表現を反転させます。
- **左シフト（`<<`）演算子**: この演算子はすべてのビットを指定した数だけ左にシフトします。 
- **右シフト（`>>`）演算子**: この演算子はすべてのビットを右にシフトします。

## 条件文、真値、偽値、三項演算子

- **`if/else if/else`**: `if`文は条件を取り、その条件が`truthy`の場合にコードブロックを実行します。条件が`false`の場合は`else if`ブロックに移ります。どの条件も`true`でなければ、`else`節を実行します。`Truthy`値は`true`文のようなブール文脈で評価したときに`if`になる値です。`Falsy`値はブール文脈で`false`になる値です。

:::interactive_editor

```js
const score = 87;

if (score >= 90) {
 console.log('You got an A'); 
} else if (score >= 80) {
 console.log('You got a B'); // You got a B
} else if (score >= 70) {
 console.log('You got a C');
} else {
 console.log('You failed! You need to study more!');
}
```

:::

- **三項演算子**: この演算子は`if else`文を短く書くためによく使われます。

:::interactive_editor

```js
const temperature = 30;
const weather = temperature > 25 ? 'sunny' : 'cool';

console.log(`It's a ${weather} day!`); // It's a sunny day!
```

:::

## 二項論理演算子

- **論理AND（`&&`）演算子**: この演算子は両方のオペランドが`true`かをチェックします。両方が`true`なら2番目の値を返します。どちらかのオペランドが`falsy`なら`falsy`値を返します。両方が`falsy`なら最初の`falsy`値を返します。

:::interactive_editor

```js
const result = true && 'hello';

console.log(result); // hello
```

:::

- **論理OR（`||`）演算子**: この演算子は少なくとも1つのオペランドが真値かをチェックします。 
- **Nullish合体演算子（`??`）**: この演算子は最初の値が`null`または`undefined`の場合のみ値を返します。

:::interactive_editor

```js
const userSettings = {
 theme: null,
 volume: 0,
 notifications: false,
};

let theme = userSettings.theme ?? 'light';
console.log(theme); // light
```

:::

## `Math`オブジェクト

- **`Math.random()`メソッド**: このメソッドは0（含む）から1（含まない）までのランダムな浮動小数点数を生成します。つまり、出力は0になる場合がありますが、1には決して達しません。
- **`Math.max()`メソッド**: このメソッドは複数の数値を受け取り、最大値を返します。
- **`Math.min()`メソッド**: このメソッドは複数の数値を受け取り、最小値を返します。
- **`Math.ceil()`メソッド**: このメソッドは値を切り上げて最も近い整数にします。 
- **`Math.floor()`メソッド**: このメソッドは値を切り捨てて最も近い整数にします。 
- **`Math.round()`メソッド**: このメソッドは値を四捨五入して最も近い整数にします。

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

- **`Math.trunc()`メソッド**: このメソッドは数値の小数部分を取り除き、丸めずに整数部分だけを返します。
- **`Math.sqrt()`メソッド**: このメソッドは数値の平方根を返します。
- **`Math.cbrt()`メソッド**: このメソッドは数値の立方根を返します。
- **`Math.abs()`メソッド**: このメソッドは数値の絶対値を返します。
- **`Math.pow()`メソッド**: このメソッドは2つの数値を受け取り、最初の数値を2番目の数値のべき乗にします。

## よく使う数値メソッド

- **`isNaN()`**: `NaN`は「Not-a-Number」の略です。これは表現できないか未定義の数値結果を表す特別な値です。`isNaN()`関数プロパティは値が`NaN`かどうかを判定するために使われます。`Number.isNaN()`は特に型変換によって予期しない結果になる場合に、グローバル`NaN`関数よりも信頼性の高い`isNaN()`値のチェック方法を提供します。

:::interactive_editor

```js
console.log(isNaN(NaN));       // true
console.log(isNaN(undefined)); // true
console.log(isNaN({}));        // true

console.log(isNaN(true));      // false
console.log(isNaN(null));      // false
console.log(isNaN(37));        // false


console.log(Number.isNaN(NaN));        // true
console.log(Number.isNaN(Number.NaN)); // true
console.log(Number.isNaN(0 / 0));      // true

console.log(Number.isNaN("NaN"));      // false
console.log(Number.isNaN(undefined));  // false
```

:::

- **`parseFloat()`メソッド**: このメソッドは文字列の引数をパースして浮動小数点数を返します。文字列の先頭から数値を抽出するよう設計されており、後ろに非数値文字があっても問題ありません。
- **`parseInt()`メソッド**: このメソッドは文字列の引数をパースして整数を返します。`parseInt()`は最初に非数字を見つけたところでパースを停止します。浮動小数点数の場合は整数部分だけを返します。文字列の先頭に有効な整数がなければ`NaN`を返します。
- **`toFixed()`メソッド**: このメソッドは数値に対して呼び出され、1つのオプション引数（小数点以下の桁数）を取ります。指定された小数点以下の桁数で数値の文字列表現を返します。
