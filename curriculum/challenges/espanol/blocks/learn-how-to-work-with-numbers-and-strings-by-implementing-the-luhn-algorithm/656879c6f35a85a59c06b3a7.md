# --description--

Has accedido a elementos (caracteres) de una cadena antes, usando el operador de índice `[]`. También puedes usar el operador de índice para acceder a un rango de caracteres en una cadena con `string[start:stop:step]`:

```python
my_string = 'camperbot'
my_string[0:6] == 'camper' # True
my_string[0:6:3] == 'cp' # True
```

Donde `start` es el índice de inicio (inclusive), `stop` es el último índice (exclusivo), y `step` es la cantidad de caracteres a saltar. Si no se especifica, el valor predeterminado de `step` es 1.

Crea una variable llamada `card_number_reversed` y asígnale el valor de los primeros 4 caracteres de `card_number`.
