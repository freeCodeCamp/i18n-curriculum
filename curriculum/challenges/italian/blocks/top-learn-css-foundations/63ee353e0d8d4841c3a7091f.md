# --description--

Ok, finora hai visto parecchio. L’unica cosa che resta per ora è spiegare come aggiungere tutto questo CSS al tuo HTML. Ci sono tre metodi per farlo.

Il CSS esterno è il metodo più comune che incontrerai, e consiste nel creare un file separato per il CSS e collegarlo all’interno dei tag di apertura e chiusura `<head>` di un HTML con un elemento `<link>` (che è un elemento void e non richiede un tag di chiusura):

Per prima cosa, aggiungi un elemento void `<link>` all’interno dei tag di apertura e chiusura `<head>` del file HTML. L’attributo `href` indica la posizione del file CSS, che può essere un URL assoluto oppure, come farai tu, un URL relativo alla posizione del file HTML. Nell’esempio sopra, si assume che entrambi i file si trovino nella stessa directory. L’attributo `rel` è obbligatorio e specifica la relazione tra il file HTML e il file collegato.

Poi, all’interno del file `styles.css` appena creato, trovi il selettore (il `div` e `p`), seguito da una coppia di parentesi graffe di apertura e chiusura, che creano un “blocco di dichiarazioni”. Infine, inserisci tutte le dichiarazioni all’interno del blocco di dichiarazioni. `color: white; ` è una dichiarazione, con `color` che è la proprietà e `white` che è il valore, e `background-color: black;` è un’altra dichiarazione.

Una nota sui nomi dei file: `styles.css` è semplicemente il nome che hai scelto per il file qui. Puoi chiamare il file come vuoi purché il tipo di file sia `.css`, anche se “style” o “styles” è il nome più comune.

Alcuni vantaggi di questo metodo sono:

1. Mantiene separati HTML e CSS, il che rende il file HTML più piccolo e tutto più ordinato.
2. Devi modificare il CSS in un solo posto, cosa particolarmente comoda per siti con molte pagine che condividono stili simili.
