# --description--

マテオは25歳で、ルナは28歳です。

空欄を埋めるために、`veinti` と数字の単語の正しい組み合わせを使ってください。

参照用：

| 数字 | スペイン語の単語 |
| :---: | :---: |
| 1 | `uno` |
| 2 | `dos` |
| 3 | `tres` |
| 4 | `cuatro` |
| 5 | `cinco` |
| 6 | `seis` |
| 7 | `siete` |
| 8 | `ocho` |
| 9 | `nueve` | 

# --fillInTheBlank--

## --sentence--

`Luna tiene BLANK años.`

`Mateo tiene BLANK años.`

## --blanks--

`veintiocho`

### --feedback--

`veinte`（20）と`ocho`（8）をどう組み合わせて一つの単語にするか考えてみてください。

---

`veinticinco`

### --feedback--

`veinte`（20）と`cinco`（5）を組み合わせるとき、`veinte`の最後の「e」が消えることを覚えておいてください。

# --explanation--

正しい答えは`veintiocho`と`veinticinco`です。

スペイン語では、21から29までの数字は一つの単語で書きます。例えば：

- `Luna tiene veintiocho años` - ルナは28歳です。

- `Mateo tiene veinticinco años` - マテオは25歳です。
