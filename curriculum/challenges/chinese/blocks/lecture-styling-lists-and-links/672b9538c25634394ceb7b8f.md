# --interactive--

链接有不同的状态，包括 `link`、`visited`、`hover`、`focus` 和 `active`。这些状态对于帮助用户识别链接并在交互后提供清晰的反馈非常重要，从而提升了可用性和无障碍性。

为这些不同的 `link` 状态设置样式对于可用性和无障碍至关重要，因为它提供了关于 `link` 当前状态的视觉提示。这有助于用户了解他们访问过哪些 `link`、正在与哪个 `link` 交互以及点击时会发生什么。

对于有视觉障碍或认知障碍的用户，这些不同的样式可以使导航更加轻松和直观。

此外，清晰的链接状态通过提供对用户交互的即时反馈，提升整体用户体验，减少混淆并改善网站的可导航性。

这些状态可以使用 CSS 中称为 `pseudo-classes` 的东西来设置样式。

伪类是添加到选择器中的关键字，用于指定所选元素的特殊状态。

例如，` :hover` 可以在用户的指针悬停时改变按钮的颜色，而 `:visited` 可以改变已访问链接的颜色。

伪类允许你根据元素的状态或用户与它们的交互来设置样式，而无需在你的 `HTML` 中添加额外的标记。

`pseudo-class` 的语法看起来像这样，其中 `A` 是选择器，`:B` 是 `pseudo-class`：

```css
A:B {
  property: value;
}
```

为了更好地理解如何为不同的 `link` 状态设置样式，让我们来看一些示例。

`:link` 伪类样式用于未访问的链接，表示它们是可点击的。

下面是定位锚元素并使用 `:link` 伪类的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Normal state (unvisited link) */
a:link { 
  color: red;
}
```

:::

上面的示例将在链接未访问时将其默认蓝色更改为 `red`。

`:visited` 样式用于已经访问或点击过的链接，帮助用户跟踪他们之前点击过的链接。以下是 `:visited` 伪类的示例用法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="https://www.freecodecamp.org/learn/" target="_blank">freeCodeCamp</a>
```

```css
/* Visited link */
a:visited {
  color: green;
}
```

:::

当点击时，这段代码会将链接颜色设置为 `green`。

`:hover` 在用户悬停时更改链接的样式，提供链接可交互的视觉提示。以下是 `:hover` 伪类的示例用法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Hover state */
a:hover {
  color: green;
}
```

:::

当鼠标悬停在链接上时，此代码会将链接颜色设置为 `green`。

`:focus` 在链接获得焦点时添加样式，例如使用键盘导航时，或增强无障碍功能时。

这是一个示例，使用 `outline` 属性在 `link` 获得焦点时应用实心橙色轮廓。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Focus state */
a:focus {
  outline: 2px solid orange;
}
```

:::

`:active` 会在链接被点击时更改链接的样式，立即向用户提供他们的操作已被寄存器的反馈。以下是 `:active` 伪类的示例用法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Active state */
a:active {
  color: pink;
}
```

:::

此代码示例将在点击链接时立即将链接颜色变为 `pink`。
