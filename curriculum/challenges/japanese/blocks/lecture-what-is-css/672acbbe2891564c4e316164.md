# --interactive--

CSSはウェブページのスタイルを担当しています。これらのスタイルはすべて、さまざまなCSSルールで構成されています。

CSSルールは主に2つの部分でできています：**セレクター**と**宣言**ブロックです。

基本的な構文を見てみましょう：

```css
selector {
  property: value;
}
```

セレクターは、CSSで特定のHTML要素を識別し、スタイルを適用するためのパターンです。

セレクターの例には、型セレクター、クラスセレクター、IDセレクターがあります。

基本構文で使われている波括弧（中括弧）は宣言ブロックと呼ばれます。宣言ブロックは、指定されたセレクターまたはセレクター群に対して一連のスタイルを適用します。

宣言ブロックの中には複数の宣言が並びます。各宣言はプロパティと値で構成されています。

プロパティは、どの機能にスタイルを適用するかを指定するCSSの識別子です。例えば、`background-color` プロパティが挙げられます。

値は、そのプロパティに適用される具体的な設定です。例えば、プロパティが`background-color`の場合、値は`purple`で、背景色を紫に設定します。

各プロパティ名の後にはコロンを置き、各値の後にはセミコロンを付ける必要があります。

CSSルールの構文がわかったところで、例を見てみましょう。インタラクティブエディターを有効にして、`styles.css` タブをクリックするとCSSコードが表示されます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Learning about CSS</h1>

<p>Example paragraph element</p>
<p>Another example paragraph element</p>
```

```css
p {
  color: blue;
}
```

:::

このCSSルールでは、型セレクターがページ内のすべての段落要素を対象にしています。

宣言ブロックの中には1つの宣言があります。

その宣言は`color` プロパティで、値は`blue` に設定されています。これにより、すべての段落要素のテキスト色が`blue` に変わります。

複数のセレクターに同じスタイルを適用したい場合は、各セレクターをカンマで区切ったセレクターリストを作成できます。

複数のセレクターをスタイル付けする例はこちらです（例とやり取りするにはインタラクティブエディターを有効にしてください）：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1 id="title">Example heading</h1>
<h2 class="subheading">Example subheading</h2>
<p>This paragraph is not affected by the selector.</p>
```

```css
#title,
.subheading {
  color: navy;
}
```

:::

このセレクターリストには、`id` セレクターがあり、`id` プロパティの値が`title` のHTML要素を対象にしています。すべての`id` セレクターはハッシュ記号`#`で始まる必要があります。

次にカンマがあり、`class` セレクターが続きます。これは`class` プロパティの値が`subheading` のすべてのHTML要素を対象にしています。すべてのクラスセレクターはドット`.`で始まります。

このセレクターリスト全体に同じスタイルが適用され、テキスト色は`navy` に設定されます。
