## HTML 基礎知識

- **HTML 的角色**：HTML（超文字標記語言）是網頁結構的基礎，定義了網頁的元素。
- **HTML 元素**：用來表述頁面上的內容。大多數由一個開啟標記和一個關閉標記組成（例如，`<h1></h1>`、`<p></p>`）。
- **HTML 結構**：HTML 由 `head` 和 `body` 組成，其中詮釋資料、樣式和內容被結構化。
- **常見的 HTML 元素**：標題（`<h1>` - `<h6>`）、段落（`<p>`）和 div 容器（`<div>`）。
- **`div` 元素**：`div` 元素是一個泛型的 HTML 元素，沒有任何語意意義。它用作泛型容器來容納其他 HTML 元素。
- **Void Elements**：沒有結束標籤（例如 `<img>`）。
- **屬性**：為元素添加詮釋資料和行為。

## 識別字與群組

- **IDs**：唯一元素識別符。
- **類別**：用於樣式和行為的元素群組。

## 特殊字元和聯結

- **HTML 實體**：使用像 `&amp;` 和 `&lt;` 這樣的特殊字元。
- **`link` 元素**：連結到外部樣式表。
- **`script` 元素**：嵌入外部 JavaScript 檔案。

## 樣板程式碼和編碼

- **HTML boilerplate**：網頁的基本結構，包括 `DOCTYPE`、`html`、`head` 和 `body` 元素。它應該作為 HTML 文件的起點使用。
- **UTF-8 字元編碼**：確保通用字元顯示。

## SEO 和社群分享

- **Meta 頁籤 (`description`)**：為網頁提供簡短描述並影響 SEO。
- **Open Graph 頁籤**：強化社群媒體分享。

## 媒體元素與優化

- **取代的元素**：嵌入內容（例如，影像、`iframe`）。
- **優化媒體**：提升媒體效率、性能的技術。
- **影像格式和授權**：了解使用權利和型別。
- **SVGs**：可縮放的向量圖形，用於清晰的視覺效果。

## 多媒體整合

- **HTML 音訊和影片、視訊元素**：嵌入多媒體。
- **使用 `<iframe>` 嵌入**：整合外部影片內容。

## 路徑和聯結行為

- **標的屬性型別**：控制聯結行為。
- **絕對路徑與相對路徑**：導覽目錄。
- **路徑語法**：理解 `/`、`./`、`../` 用於檔案導覽。
- **聯結狀態**：管理不同的聯結互動（暫留、作用中的）。
- **內部聯結**：透過在 `a` 元素上使用 `href="#id"` 並給予目標元素相同的 `id`，將頁面連結到另一個區段。

## 語意化 HTML 的重要性

- **標題元素的結構階層體系**：使用正確的標題元素以維護內容的結構階層體系非常重要。`h1` 元素是最高層級的標題，而 `h6` 元素是最低層級的標題。
- **呈現型 HTML 元素**：定義內容外觀的元素，例如已棄用的 `center`、`big` 和 `font` 元素。
- **語意 HTML 元素**：這些元素為內容的結構提供意義。範例包括：
  - `<header>`：表述介紹內容。
  - `<nav>`：包含導覽聯結。
  - `<article>`：表述獨立內容。
  - <aside>：用於側邊欄或相關內容。
  - <section>：在文件中將相關內容分組。
  - `<footer>`：定義區段或文件的頁腳。

## 語意化 HTML 元素

- **強調 (`em`) 元素**：標示具有強調語氣的文字。
- **慣用語文字（`i`）元素**：用於凸顯標示替代語氣或情緒、來自其他語言的慣用語、技術術語和想法。
- **強調重要性（`strong`）元素**：標示具有強烈重要性的文字。
- **將注意力帶到 (`b`) 元素**：用於將注意力帶到對內容意義不重要的文字。
- **描述列表（`dl`）元素**：用於表述一組術語與描述的列表。
- **說明詞條（`dt`）元素**：用於表述被定義的詞條。
- **Description Details (`dd`) 元素**：用於表述該術語的描述。
- **區塊引用（`blockquote`）元素**：用於表述從其他原始碼引用的區段。
- **行內引用 (`q`) 元素**：用於表述短的行內引用。
- **縮寫（`abbr`）元素**：用於表述縮寫或首字母縮略詞。
- **聯絡位址 (`address`) 元素**：用於表述聯絡資訊。
- **(Date) Time (`time`) element**：用於表述日期和／或時間。
- **上標（`sup`）元素**：用於表述上標文字。
- **下標（`sub`）元素**：用於表述下標文字。
- **行內程式碼（`code`）元素**：用於表述一段電腦程式碼。
- **未明確標註的註解（`u`）元素**：用於表述一段行內文字，該文字應以顯示其具有非文字註解的方式呈現。
- **Ruby Annotation (`ruby`) 元素**：用於表述 ruby 註解的文字。
- **刪除線（`s`）元素**：用於表述不再準確或相關的內容。

## HTML 表單元素和屬性

### 表單

- **`form` 元素**：用於創建供使用者輸入的 HTML 表單。
- **常見輸入型別**：
  - `text`、`email`、`password`、`radio`、`checkbox`、`number`、`date`。
- **`action` 屬性**：用來指定表單資料應該送出的 URL。
- **`method` 屬性**：用來指定在傳送表單資料時要使用的 HTTP 方法。最常見的方法是 `GET` 和 `POST`。

```html
<form method="value-goes-here" action="url-goes-here">
  <!-- inputs go inside here -->
</form>
```

- **`input` 元素**：用於創建用戶輸入的輸入欄。
- **`type` 屬性**：用於指定輸入欄的型別，例如 `text`、`email`、`number`、`radio`、`checkbox`。
- **`placeholder` 屬性**：用來向使用者顯示提示，告訴他們在輸入欄中應輸入什麼內容。
- **`value` 屬性**：用於指定輸入的值。如果輸入的 `type` 是 `button`，`value` 屬性可以用來設定按鈕文字。
- **`name` 屬性**：用於為輸入欄指定名稱，該名稱在提交表單資料時作為鍵。對於圓鈕，將它們指定相同的 `name` 可將它們分組，因此群組中一次只能選擇一個選項。
- **`size` 屬性**：用來定義使用者在輸入時應該可見的字元數量。
- **`min` 屬性**：可用於像是 `number` 這類輸入型別，以指定輸入欄位允許的最小值。
- **`max` 屬性**：可用於像是 `number` 的輸入型別，以指定輸入欄位允許的最大值。
- **`minlength` 屬性**：用於指定輸入欄位中所需的最少字元數。
- **`maxlength` 屬性**：用於指定輸入欄位中允許的最大字元數。
- **`required` 屬性**：用於指定在提交表單之前必須填寫輸入欄。
- **`disabled` 屬性**：用於指定輸入欄應該被停用。
- **`readonly` 屬性**：用於指定輸入欄為唯讀。

:::interactive_editor

```html
<!-- Text input -->
<input 
  type="text"
  id="name"
  name="name"
  placeholder="e.g. Quincy Larson"
  size="20"
  minlength="5"
  maxlength="30"
  required
/>

<!-- Number input -->
<input 
  type="number"
  id="quantity"
  name="quantity"
  min="2"
  max="10"
  disabled
/>

<!-- Button -->
<input type="button" value="Show Alert" />
```

:::

- **`label` 元素**：用於為輸入欄位創建標籤。
- **`for` 屬性**：用來指定標籤所對應的輸入欄。
- **隱式的表單相關**：輸入可以透過將輸入欄位包裹在 `label` 元素內來與標籤相關聯。

:::interactive_editor

```html
<form action="">
  <label>
    Full Name:
    <input type="text" />
  </label>
</form>
```

:::

- **明確的表單相關**：輸入欄位可以透過在 `label` 元素上使用 `for` 屬性來與標籤相關聯。

:::interactive_editor

```html
<form action="">
  <label for="email">Email Address: </label>
  <input type="email" id="email" />
</form>
```

:::

- **`button` 元素**：用於創建可點擊的按鈕。按鈕也可以有一個 `type` 屬性，該屬性控制按鈕被作用時的行為，例如 `submit`、`reset`、`button`。

:::interactive_editor

```html
<button type="button">Show Form</button>
<button type="submit">Submit Form</button>
<button type="reset">Reset Form</button>
```

:::

- **`fieldset` 元素**：用於將相關輸入欄位群組在一起。
- **`legend` 元素**：用於為輸入群組添加說明標題。

:::interactive_editor

```html
<!-- Radio group -->
<fieldset>
  <legend>Was this your first time at our hotel?</legend>

  <input id="yes-option" type="radio" name="hotel-stay" value="yes" />
  <label for="yes-option">Yes</label>

  <input id="no-option" type="radio" name="hotel-stay" value="no" />
  <label for="no-option">No</label>
</fieldset>

<!-- Checkbox group -->
<fieldset>
  <legend>
    Why did you choose to stay at our hotel? (Check all that apply)
  </legend>

  <input type="checkbox" id="location" name="location" value="location" />
  <label for="location">Location</label>

  <input type="checkbox" id="price" name="price" value="price" />
  <label for="price">Price</label>
</fieldset>
```

:::

- **聚焦狀態**：這是使用者選取輸入欄位時的狀態。

## 操作 HTML 表格元素和屬性

- **Table element**：用於創建 HTML 表格。
- **表格標頭（`thead`）元素**：用於將 HTML 表格中的標頭內容分組。
- **表格列 (`tr`) 元素**：用於在 HTML 表格中創建一列。
- **表格標題（`th`）元素**：用於在 HTML 表格中創建標題儲存格。
- **表格主體（`tbody`）元素**：用於將 HTML 表格中的主體內容分組。
- **表格資料儲存格（`td`）元素**：用於在 HTML 表格中創建資料儲存格。
- **Table Foot (`tfoot`) 元素**：用於將 HTML 表格中的頁腳內容分組。
- **`caption` 元素**：用於為 HTML 表格添加標題。
- **`colspan` 屬性**：用來指定表格儲存格應該跨越的行數。

:::interactive_editor

```html
<table>
  <caption>Exam Grades</caption>

  <thead>
    <tr>
      <th>Last Name</th>
      <th>First Name</th>
      <th>Grade</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>Davis</td>
      <td>Alex</td>
      <td>54</td>
    </tr>

    <tr>
      <td>Doe</td>
      <td>Samantha</td>
      <td>92</td>
    </tr>

    <tr>
      <td>Rodriguez</td>
      <td>Marcus</td>
      <td>88</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <td colspan="2">Average Grade</td>
      <td>78</td>
    </tr>
  </tfoot>
</table>
```

:::

## HTML 工具

- **HTML validator**：一個檢查 HTML 程式碼語法以確保其有效的工具。
- **DOM inspector**：一個允許你檢查和修改網頁 HTML 結構的工具。
- **Devtools**：一組直接內建於瀏覽器中的網頁開發工具，幫助你偵錯、評測及分析網頁。

## 無障礙介紹

- **網頁內容無障礙指引**：網頁內容無障礙指引（WCAG）是一組讓網頁內容對身心障礙者更易取用的指引。WCAG 的四大原則是 POUR，分別代表可感知（Perceivable）、可操作（Operable）、可理解（Understandable）和強固（Robust）。

## 無障礙輔助技術

- **螢幕閱讀器**：讀出電腦螢幕內容的軟體程式。它們被盲人或視障者用來存取網路。
- **大型文字或點字鍵盤**：供視障者存取網路時使用。
- **螢幕放大鏡**：將電腦螢幕內容放大的軟體程式。它們供視力低下的人士用來存取網路。
- **替代指點裝置**：供行動不便者存取網路時使用。這包括搖桿、滾球和觸控板等裝置。
- **語音辨識**：供行動不便的人士存取網路。它允許使用者使用聲音來控制電腦。

## 無障礙性稽核工具

- **常見的無障礙工具**：Google Lighthouse、Wave、IBM Equal Accessibility Checker 和 axe DevTools 是用來稽核網站無障礙性的常見無障礙工具。

## 無障礙最佳實踐

- **正確的標題層級結構**：你應該使用正確的標題層級來創建內容的邏輯結構。這有助於輔助技術理解你網站的內容。
- **無障礙性與表格**：使用表格時，你應該使用 `th` 元素來定義標題儲存格，並使用 `td` 元素來定義資料儲存格。這有助於輔助技術理解表格的結構。
- **輸入欄位擁有相關 `label` 的重要性**：你應該使用 `label` 元素將標籤與表單輸入欄位相關聯。這有助於輔助技術理解輸入欄位的用途。
- **良好 `alt` 文字 的重要性**：你應該使用 `alt` 屬性為影像提供文字替代。這有助於輔助技術理解影像的內容。
- **良好連結文字的重要性**：你應該使用具描述性的連結文字來幫助使用者了解連結的目的。
- **讓音訊和影片內容可存取的最佳做法**：你應該為音訊和影片內容提供字幕和文字稿，以便讓聽力障礙者能夠存取。你也應該為影片內容提供音訊描述，以便讓視力障礙者能夠存取。
- **`tabindex` 屬性**：用於使元素可聚焦並定義使用鍵盤導航時的相對順序。切勿將 `tabindex` 屬性的值設為大於 0。你應該使用 0 或 -1 作為值。
- **`accesskey` 屬性**：用於定義元素的鍵盤快速鍵。這可以幫助有行動不便的使用者更輕鬆地瀏覽網站。

## WAI-ARIA、角色和屬性

- **WAI-ARIA**：代表網頁無障礙倡議－可存取的豐富網際網路應用程式。它是一組可以添加到 HTML 元素的屬性，用以提升無障礙性。它為輔助技術提供關於內容目的和結構的額外資訊。
- **ARIA roles**：一組可添加到 HTML 元素以定義其用途的預設角色。這有助於輔助技術理解網站內容。範例包括 `role="tab"`、`role="menu"` 和 `role="alert"`。
- **`aria-label` 和 `aria-labelledby` 屬性**：這些屬性用於為元素提供程式化（或可存取的）名稱，有助於輔助技術（例如螢幕閱讀器）理解該元素的用途。當元素的視覺標籤是影像或符號而非文字時，這些屬性經常被使用。`aria-label` 允許你直接在屬性中定義名稱，而 `aria-labelledby` 則允許你取用頁面上現有的文字。
- **`aria-hidden` 屬性**：用於將元素從輔助技術（例如螢幕閱讀器）中隱藏。舉例來說，這可用於隱藏不提供任何有意義內容的裝飾性影像。
- **`aria-describedby` 屬性**：用來透過將元素與包含資訊的另一個元素關聯，提供該元素的額外資訊。這有助於輔助技術理解該元素的用途。
