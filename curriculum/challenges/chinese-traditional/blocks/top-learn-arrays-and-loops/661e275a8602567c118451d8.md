# --description--

另一個有用的陣列方法是 `filter()` 方法。`filter()` 方法會創建一個新陣列，包含所有通過所提供函式實作的測試的元素。`filter()` 方法的語法如下：

```javascript
const numbers = [2, 5, 6, 1, 9, -1]

const newNumbers = numbers.filter((number) => {
  return number > 2;
});

console.log(newNumbers); // Output: [5, 6, 9]
```

`filter()` 方法會創建一個包含所有大於 2 元素的新陣列。`filter()` 方法不會改變原始陣列。
