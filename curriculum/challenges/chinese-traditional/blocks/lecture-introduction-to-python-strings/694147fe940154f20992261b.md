# --description--

在先前的課程中，你學會了如何透過索引（從零開始）識別字串中的每個字元，並使用括號表示法存取它：

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
print(my_str[-1]) # d
```

**字串切片** 讓你擷取字串的一部分或只處理其中的特定零件。以下是基本語法：

```python
string[start:stop]
```

如果你想從某個索引擷取字元到另一個索引，只要用冒號分隔 `start` 和 `stop` 索引即可：

```python
my_str = 'Hello world'
print(my_str[1:4]) # ell
```

請注意，`stop` 索引是不包含在內的，所以 `[1:4]` 只擷取了從索引 `1` 開始，到但不包含索引 `4` 的字元。

你也可以省略 `start` 和 `stop` 索引，Python 將分別預設為 `0` 或字串的結尾。 例如，如果你省略 `start` 索引，會發生以下情況：

```python
my_str = 'Hello world'
print(my_str[:7])  # Hello w
```

這會擷取從索引 `0` 開始直到（但不包含）索引 `7` 的字元。若你省略 `stop` 索引，會發生以下情況：

```python
my_str = 'Hello world'
print(my_str[8:])  # rld
```

這會從索引為 `8` 的字元開始擷取直到字串結尾的所有內容。

請注意，切割字串不會修改原始字串：

```py
my_str = 'Hello world'
print(my_str[8:])  # rld
print(my_str)  # Hello world
```

你也可以省略 `start` 和 `stop` 兩個索引，這樣會擷取整個字串：

```python
my_str = 'Hello world'
print(my_str[:])  # Hello world
```

除了 `start` 和 `stop` 索引外，還有一個選用的 `step` 參數，用於指定切片中每個索引之間的增量。

以下是該語法：

```python
string[start:stop:step]
```

在以下範例中，切片從索引 `0` 開始，在 `11` 之前停止，並擷取每隔一個字元：

```python
my_str = 'Hello world'
print(my_str[0:11:2])  # Hlowrd
```

你可以對 `step` 參數做的一個有用技巧是，將 `step` 設為 `-1`，並將 `start` 和 `stop` 留空，以反轉字串：

```python
my_str = 'Hello world'
print(my_str[::-1]) # dlrow olleH
```
