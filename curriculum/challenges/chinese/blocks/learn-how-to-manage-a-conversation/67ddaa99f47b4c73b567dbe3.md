# --explanation--

`Sure thing.` 是一种非正式且友好的表达方式，表示 `Of course` 或 `I'll do it`。它表明说话者毫不犹豫地同意该请求。例如：

- 人物 1：`Can you send me the file this afternoon?` - 这个人正在请求一个文件。

- 人物 2：`Sure thing.`——这意味着说话者同意并将按请求发送文件。
