# --explanation--

`Past Perfect` 时态（`had` + 动词的 `past participle` 形式）用于描述发生在另一个过去事件之前的动作。它显示哪个动作先发生。例如：

`The server had crashed before the backup started.` – 崩溃先发生，然后备份开始。

`Third Conditional` 结构（`if` + 动词的 `past perfect` 形式作为假设原因，情态动词 + `have` + 动词的 `past participle` 形式作为假设结果）表达了一个不真实的过去条件及其可能但不真实的结果。它讲述了如果采取了其他行动，情况可能会有所不同。例如：

`If they had updated the software, they could have avoided the error.` – 这意味着更新没有发生，且误差没有被避免。
