# --description--

缓存是将文件副本存储在临时存储位置的过程，以便可以更快地访问。 

当你访问一个网站时，你的浏览器可以将某些文件（例如图像、CSS 和 JavaScript）储存在本地。这意味着下次你访问同一网站时，它可以从你的设备装载这些文件，而不是再次从服务器获取，从而使网站加载更快。

服务工作者是一个在后台运行的 `Script`，与您的网页分离。它可以拦截 `network` 请求、访问 `cache`，并启用网页应用离线工作。它是渐进式网页应用的关键 `component`。

那么，`cache` 和服务工作线程如何在渐进式网页应用中协同工作？

PWA 是可以提供类似应用体验的网页应用。它们可以离线工作、发送推送通知，甚至可以安装到移动设备或计算机的主屏幕上。

当用户首次访问 PWA 时，service worker 可以缓存重要的文件。 

用户可以继续离线使用该应用，当他们重新在线时，任何即将发生的更改都可以与服务器同步。

缓存和 service workers 的结合使得 web 应用即使在网络状况不佳时也能提供快速且可靠的体验。
