# --description--

3차원 공간에서 두 점 ($x_1$, $y_1$, $z_1$)과 ($x_2$, $y_2$, $z_2$)가 주어졌을 때, 이 두 점 사이의 맨해튼 거리는 $|x_1 - x_2| + |y_1 - y_2| + |z_1 - z_2|$로 정의됩니다.

$C(r)$를 반지름이 $r$이고 원점 $O(0, 0, 0)$을 중심으로 하는 구라고 합시다.

$I(r)$를 $C(r)$의 표면 위에 있는 모든 정수 좌표 점들의 집합이라고 합시다.

$S(r)$를 $I(r)$의 모든 원소가 원점 $O$까지 가지는 맨해튼 거리의 합이라고 합시다.

예를 들어, $S(45)=34518$입니다.

$S({10}^{10})$을 구하세요.
