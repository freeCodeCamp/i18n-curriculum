# --explanation--

`Supposed to be` significa que algo era esperado ou planejado para acontecer.Por exemplo:

`The new feature was supposed to be ready by last week.` — Isso significa que estava planejado para estar pronto na semana passada.

`Coming later` significa que algo está planejado para acontecer no futuro.Por exemplo:

`The update is coming later this month.` — Isso significa que a atualização será lançada em um momento futuro, não agora.
