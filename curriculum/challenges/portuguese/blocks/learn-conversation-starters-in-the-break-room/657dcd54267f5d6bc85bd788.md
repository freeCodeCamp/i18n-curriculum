# --description--

Se você precisa pegar algum dinheiro e está procurando uma máquina que distribua dinheiro, você pode perguntar, `Is there an ATM nearby?`

Um `ATM`, que significa `Automated Teller Machine`, é uma máquina onde você pode usar um cartão bancário para sacar dinheiro.

A palavra `nearby` significa que não está longe de você, talvez a uma curta caminhada.É uma palavra útil para usar quando você precisa encontrar algo rapidamente em um lugar novo.
