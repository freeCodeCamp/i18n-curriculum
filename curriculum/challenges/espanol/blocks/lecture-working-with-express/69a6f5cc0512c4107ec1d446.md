# --description--

Express.js o simplemente Express, es un framework web minimalista y flexible construido **sobre** Node.js. Simplifica el proceso de crear servidores web y APIs.

En lugar de escribir toneladas de código repetitivo con el módulo nativo `http` de Node, Express te permite crear aplicaciones web robustas con solo unas pocas líneas.

Puedes pensar en él como la capa rápida y sin opiniones que hace que Node.js sea más fácil y potente.

Aquí tienes tres razones clave por las que a los desarrolladores les encanta Express:

1. **Minimalista y rápido** – Express proporciona solo lo esencial, dándote control total y permitiéndote agregar funciones según necesites mediante middleware y plugins.

2. **Ecosistema enorme** – Cuenta con una gran colección de middleware y plugins para registro, autenticación, seguridad, sesiones y más.

3. **Ideal para APIs** – Su modelo simple de enrutamiento y manejo de solicitudes hace que Express sea especialmente adecuado para construir APIs RESTful.

Con Express puedes crear:

* Aplicaciones web

* APIs RESTful

* Backends para aplicaciones de una sola página (SPA)

* Servicios con mucho middleware

* Cualquier cosa del lado servidor con HTTP


## Cómo instalar Express

Primero, asegúrate de tener Node.js y npm instalados. Así es como puedes verificar si node y npm están instalados:

```bash
node -v
npm -v
```

Si ves un número de versión, significa que están instalados. Si no, tendrás que instalarlos. Puedes encontrar instrucciones de instalación en una lección anterior o en el sitio oficial de Node.js.

Luego, en tu terminal, escribe los siguientes comandos:

```bash
mkdir my-app
cd my-app
npm init -y
npm install express
```

Aprenderás más sobre middleware y plugins en futuras lecciones, pero en resumen, el middleware es código que se ejecuta entre recibir una solicitud y enviar una respuesta. Se usa comúnmente para manejo de errores, validación de entradas, modificar datos en la solicitud o respuesta y más. Puedes encadenar middleware para construir lógica flexible y reutilizable para manejar solicitudes.

Los plugins suelen ser módulos externos de terceros para Node.js que extienden la funcionalidad de Express. Puedes instalarlos por separado con npm, y la mayoría se implementan como middleware que puedes usar en tu app con solo un poco de configuración. En lugar de escribir funciones comunes desde cero, puedes usar un plugin para manejar cosas como registro, autenticación o buenas prácticas de seguridad.

Express es un framework popular para desarrollo del lado servidor en Node.js. Te da las bases para levantar un servidor rápidamente, manejar rutas, gestionar middleware y construir APIs, todo con código limpio y legible.
