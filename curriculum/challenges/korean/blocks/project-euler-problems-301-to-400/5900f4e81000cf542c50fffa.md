# --description--

$f(n)$을 $x$와 $y$가 양의 정수이고, $x ≤ y$이며 $x$와 $y$의 최소공배수가 $n$인 쌍 ($x$, $y$)의 개수라고 하겠습니다.

$g$는 $f$의 누적합 함수로, 즉 $g(n) = \sum f(i)$이며 $1 ≤ i ≤ n$입니다.

$g({10}^6) = 37\,429\,395$임이 주어졌습니다.

$g({10}^{12})$를 구하세요.
