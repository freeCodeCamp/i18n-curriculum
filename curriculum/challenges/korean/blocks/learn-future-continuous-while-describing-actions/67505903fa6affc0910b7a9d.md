# --explanation--

`Prevent`는 무언가가 일어나는 것을 막거나 발생하지 않도록 하는 것을 의미합니다. 예를 들어:  

`The new software update will prevent unauthorized access to the system.` - 이 문장은 업데이트가 무단 사용자의 시스템 접근을 막을 것임을 의미합니다.

`DoS attacks`는 서비스 거부 공격(Denial of Service attack)을 가리키며, 컴퓨터나 네트워크 서비스를 트래픽으로 과부하 시켜 사용할 수 없게 만드는 시도입니다. 예를 들어:  

`The website was temporarily offline due to a large-scale DoS attack.` - 이 문장은 웹사이트가 유해한 트래픽으로 과부하되어 사용할 수 없게 되었음을 의미합니다.
