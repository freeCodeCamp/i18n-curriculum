# --description--

文件根据其内容和结构进行分类。文件类型决定计算机程序如何打开和处理文件。

我们根据文件扩展名来识别文件类型。文件扩展名是文件名末尾的三或四个字母后缀。你会在网页应用中发现的三种主要文件类型是 HTML、CSS 和 JavaScript。

HTML 文件包含网页应用的结构和内容。它们具有 `.html` 文件扩展名。

CSS 文件定义样式。它们具有 `.css` 文件扩展名。

JavaScript 文件添加更多高级的功能性和交互性。它们具有 `.js` 文件扩展名。

在你开发网页应用时，你还需要包含图像。这些是一些最广泛使用的图像文件类型。

JPEG，全称为 "Joint Photographic Experts Group"，以其高效的压缩而闻名。它非常适合照片和图像。然而，压缩是有损的，这意味着为了减小文件大小，会丢失部分图像质量。

PNG，代表“Portable Network Graphics”，即使在压缩后也能保持图像质量。这导致文件更大。PNG 支持透明度，非常适合具有犀利边缘的图像，如标志和图标。

GIF 图像格式支持动画和透明度，但颜色调色板有限。GIF 代表 `"Graphics Interchange Format"`。

最后，我们有 `SVG`，代表“可缩放向量图形”。它是一种向量图像格式。这些图像可以根据需要任意缩放而不会丢失质量。

很好。既然你已经了解了图像格式，现在让我们看看一些视频和音频格式。

MP3 是一种以高效压缩著称的音频格式。它是一种有损格式，因此在压缩过程中会丢失部分音频数据，以使这些文件更小。

MP4 是最常见的视频格式之一。它提供良好的压缩，并支持多种音频和视频编解码器、字幕和元数据。

MOV，是苹果开发的 QuickTime 多媒体文件格式，主要与 QuickTime Player 关联。

其他流行的音频和视频文件类型包括 WAV，这是一种无损音频格式，能够保持音频的原始质量，以及 WebM，一种高质量的开源视频格式。

如果你需要自定义你的网页应用的字体，你还将使用字体格式。这是三种最广泛使用的格式。

TTF，代表 `"TrueType Font"`，是一种兼容不同操作系统的字体形式。

WOFF 是一种现代字体格式，专门为网页开发目的设计。它代表“Web Open Font Format”。这些文件更小，因为它们具有更好的压缩。它们还可以保存元数据，包括许可信息。

而 WOFF2 是 WOFF 的继任者，具有更高效的压缩和性能。

如果你需要将多个文件夹和文件分组，也可以创建归档文件。最广泛使用的归档格式是 ZIP。ZIP 提供无损压缩，因此在此过程中不会丢失任何数据。它被各种操作系统和软件广泛支持。

最后，让我们谈谈文档。在网页应用中，你通常会找到一个名为 `README` 的文件，其中包含关于该应用的信息，比如如何使用它、如何安装它、它的许可证以及如何贡献。

它们通常以一种称为 Markdown 的文件格式编写。使用 Markdown，你可以创建带有标题、副标题、链接、图像、列表等结构化文档。Markdown 文件具有 `.md` 或 `.markdown` 扩展名。

在这里你可以找到 freeCodeCamp GitHub 仓库的 `README` 文件：https://github.com/freeCodeCamp/freeCodeCamp/blob/main/README.md

你可以使用 Markdown 创建此详细的结构和形式。

在你的职业生涯中，你肯定会读取和写入许多 `README` 文件。了解这些常见的文件类型对于网页开发至关重要。 

从 HTML、CSS 和 JavaScript 的核心创建块，到图像、视频 和 音频 格式，每种文件类型在创建交互式网页应用中都有重要作用。
