# --description--

La parola `so` è un connettore usato per mostrare causa ed effetto o per spiegare il motivo di qualcosa. In tecnologia e programmazione, `so` collega spesso il problema o la situazione con la soluzione o il risultato.

# --questions--

## --text--

Scegli la frase in cui `so` è usato correttamente per collegare una causa e un effetto in un contesto tecnologico.

## --answers--

`The program crashed, so we must reboot the computer.`

---

`We wrote the code, so but it didn't compile.`

### --feedback--

La parola `so` è usata in modo errato qui. `But` non dovrebbe essere usato dopo `so`. Dovrebbe essere o `The code didn't compile, so we rewrote it` o `We wrote the code, but it didn't compile.`

---

`The server is online, so therefore we can't connect.`

### --feedback--

L'uso di `so` e `therefore` insieme è ridondante. Dovrebbe essere o `The server is online, so we can connect` o `The server is online; therefore, we can connect.`

---

`We need more data, so we can ignore the results.`

### --feedback--

Questa frase non collega logicamente causa ed effetto. Un uso migliore di `so` sarebbe `We need more data, so we will perform more tests.`
