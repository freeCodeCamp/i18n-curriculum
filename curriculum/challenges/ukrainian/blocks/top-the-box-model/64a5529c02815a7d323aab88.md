# --description--

Вміння інспектувати та налагоджувати свій HTML і CSS є критично важливим для фронтенд-розробки. Цей урок проведе вас через Chrome Dev Tools, які дозволяють бачити детальну інформацію про ваші елементи та CSS-правила, а також допомагають знаходити й виправляти проблеми у вашому коді.

Щоб відкрити інспектор, ви можете натиснути правою кнопкою миші на будь-який елемент сторінки та вибрати «Inspect» або натиснути F12. Зробіть це зараз, щоб побачити HTML і CSS, які використовуються на цій сторінці.

Не лякайтеся всіх інструментів, які ви зараз бачите! Для цього уроку ми хочемо зосередитися на панелях Elements і Styles.
