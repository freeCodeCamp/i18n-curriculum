# --description--

Betrachten Sie Gleichungen der Form: $a^2 + b^2 = N$, $0 ≤ a ≤ b$, wobei $a$, $b$ und $N$ ganze Zahlen sind.

Für $N = 65$ gibt es zwei Lösungen:

$a = 1, b = 8$ und $a = 4, b = 7$.

Wir nennen $S(N)$ die Summe der Werte von $a$ aller Lösungen von $a^2 + b^2 = N$, $0 ≤ a ≤ b$, wobei $a$, $b$ und $N$ ganze Zahlen sind.

Somit ist $S(65) = 1 + 4 = 5$.

Finden Sie $\sum S(N)$ für alle quadratfreien $N$, die nur durch Primzahlen der Form $4k + 1$ teilbar sind, mit $4k + 1 < 150$.
