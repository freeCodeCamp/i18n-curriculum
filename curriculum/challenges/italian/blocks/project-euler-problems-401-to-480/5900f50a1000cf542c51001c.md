# --description--

Diciamo che un numero positivo di $d$ cifre (senza zeri iniziali) è un numero con uno solo figlio se esattamente una delle sue sotto-stringhe è divisibile per $d$.

Per esempio, 5671 è un numero con uno solo figlio di 4 cifre. Tra tutte le sue sotto-stringhe 5, 6, 7, 1, 56, 67, 71, 567, 671 e 5671, solo 56 è divisibile per 4.

Allo stesso modo, 104 è un numero con uno solo figlio di 3 cifre perché solo 0 è divisibile per 3. 1132451 è un numero con uno solo figlio di 7 cifre perché solo 245 è divisibile per 7.

Sia $F(N)$ il numero di numeri con uno solo figlio minori di $N$. Possiamo verificare che $F(10) = 9$, $F({10}^3) = 389$ e $F({10}^7) = 277\\,674$.

Trova $F({10}^{19})$.
