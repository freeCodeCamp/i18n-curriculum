# --description--

Estabas usando la función `rgb` para establecer colores. Para agregar un canal alfa a un color `rgb`, utiliza la función `rgba` en su lugar.

La función `rgba` funciona igual que la función `rgb`, pero toma un número más del `0` al `1.0` para el canal alfa:

```css
rgba(redValue, greenValue, blueValue, alphaValue);
```

También puedes usar un canal alfa con los colores `hsl` y `hex`. Pronto verás cómo hacer eso.

En la regla `.sleeve`, utiliza la función `rgba` para establecer la propiedad `background-color` a blanco puro con 50% de opacidad.

# --hints--

La regla CSS `.sleeve` debe tener una propiedad `background-color` establecida en `rgba(255, 255, 255, 0.5)`.

```js
assert.strictEqual(new __helpers.CSSHelp(document).getStyle('.sleeve')?.backgroundColor, 'rgba(255, 255, 255, 0.5)');
```
