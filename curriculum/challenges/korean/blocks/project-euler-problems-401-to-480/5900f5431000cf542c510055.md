# --description--

Ramvok 게임 한 판을 생각해 보세요:

$t$가 게임이 지속되는 최대 턴 수를 나타낸다고 합시다. 만약 $t = 0$이면, 게임은 즉시 종료됩니다. 그렇지 않으면, 각 턴 $i$마다 플레이어가 주사위를 굴립니다. 굴린 후, 만약 $i &lt; t$라면 플레이어는 게임을 멈추고 현재 굴린 값만큼의 상금을 받거나, 굴린 값을 버리고 다음 턴에 다시 시도할 수 있습니다. 만약 $i = t$라면, 굴린 값을 버릴 수 없고 반드시 상금을 받아야 합니다. 게임이 시작되기 전에, $t$는 플레이어가 선택하며, 그 후 상수 $c$에 대해 선불 비용 $ct$를 지불해야 합니다. $c = 0$일 경우, $t$는 무한대로 선택할 수 있습니다(선불 비용은 0). $R(d, c)$를 공정한 $d$면체 주사위와 비용 상수 $c$가 주어졌을 때 최적 플레이로 한 판의 Ramvok 게임에서 플레이어가 받는 기대 이익(즉, 순이익)이라고 합시다. 예를 들어, $R(4, 0.2) = 2.65$입니다. 플레이어는 모든 선불 비용을 지불할 충분한 자금이 있다고 가정합니다.

이제 슈퍼 Ramvok 게임을 생각해 보세요:

슈퍼 Ramvok에서는 Ramvok 게임을 반복해서 진행하지만 약간의 변형이 있습니다. 각 게임 후에 주사위가 변경됩니다. 변경 과정은 다음과 같습니다: 주사위를 한 번 굴리고, 나온 면에 점이 보인다면 그 면은 빈 면으로 바뀝니다. 만약 그 면이 이미 빈 면이라면 원래 값으로 다시 바뀝니다. 변경이 이루어진 후, 또 다른 Ramvok 게임이 시작될 수 있습니다(이 게임 동안 각 턴마다 주사위를 굴려 값이 있는 면이 나올 때까지 굴립니다). 플레이어는 어떤 면이 빈 면이고 어떤 면이 아닌지 항상 알고 있습니다. 슈퍼 Ramvok 게임은 주사위의 모든 면이 빈 면이 되면 종료됩니다.

$S(d, c)$를 공정한 $d$면체 주사위(모든 면이 보이는 상태)와 비용 상수 $c$가 주어졌을 때 최적 플레이로 진행한 슈퍼 Ramvok 게임에서 플레이어가 받는 기대 이익이라고 합시다. 예를 들어, $S(6, 1) = 208.3$입니다.

$F(n) = \sum_{4 ≤ d ≤ n} \sum_{0 ≤ c ≤ n} S(d, c)$를 계산하세요.

$F(20)$을 가장 가까운 정수로 반올림하여 구하세요.
