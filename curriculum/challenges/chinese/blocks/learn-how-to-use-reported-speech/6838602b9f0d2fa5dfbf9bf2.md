# --explanation--

`Unusually` 是一个副词，用来描述某事发生的频率比正常或预期的更多或更少。另一方面，`unusual` 是一个形容词，用来描述某事不常见或不正常。例如：

- `The server` 今天经历了异常高的装载。– 在这个句子中，`unusually` 与装载的高度有关（远远超出正常范围）。

- `It was unusual for the server to slow down like that.`——在这个句子中，`unusual` 指的是情况。它不是修饰动词或形容词。

`query` 是发送到数据库以获取信息或执行操作的问题或请求。例如：

`The application sent a query to the database to fetch user details.` – 这意味着应用向数据库请求了特定的信息。
