# --instructions--

このプロジェクトでは、折れ線グラフ、棒グラフ、箱ひげ図を使って時系列データを可視化します。Pandas、Matplotlib、Seabornを使い、2016-05-09から2019-12-03までのfreeCodeCamp.orgフォーラムの1日あたりのページビュー数を含むデータセットを可視化します。これらのデータの可視化は、訪問のパターンを理解し、年間および月間の成長を特定するのに役立ちます。

以下のタスクをデータを使って完了してください：

- Pandasを使って「fcc-forum-pageviews.csv」からデータをインポートし、インデックスを`date`列に設定してください。
- ページビュー数がデータセットの上位2.5％または下位2.5％に入る日のデータを除外してデータをクリーンアップしてください。
- Matplotlibを使って「examples/Figure_1.png」に似た折れ線グラフを描く`draw_line_plot`関数を作成してください。タイトルは`Daily freeCodeCamp Forum Page Views 5/2016-12/2019`にし、x軸のラベルは`Date`、y軸のラベルは`Page Views`にしてください。
- 「examples/Figure_2.png」に似た棒グラフを描く`draw_bar_plot`関数を作成してください。これは年ごとにグループ化された各月の平均日別ページビュー数を示します。凡例は月のラベルを表示し、タイトルは`Months`にしてください。チャート上のx軸のラベルは`Years`、y軸のラベルは`Average Page Views`にしてください。
- Seabornを使って「examples/Figure_3.png」に似た2つの隣接する箱ひげ図を描く`draw_box_plot`関数を作成してください。これらの箱ひげ図は、特定の年または月内での値の分布と時間経過による比較を示します。最初のチャートのタイトルは`Year-wise Box Plot (Trend)`、2つ目のチャートのタイトルは`Month-wise Box Plot (Seasonality)`にしてください。月のラベルは`Jan`から始まり、x軸とy軸のラベルが正しく設定されていることを確認してください。ボイラープレートにはデータ準備のコマンドが含まれています。

各チャートには、データフレームのコピーを使ってください。

ボイラープレートには画像を保存して返すコマンドも含まれています。
