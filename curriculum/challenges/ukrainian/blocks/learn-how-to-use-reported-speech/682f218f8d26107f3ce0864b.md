# --explanation--

`Of course` — це ввічливий спосіб показати згоду або сказати `yes` без вагань. Це означає, що мовник радий або готовий щось зробити. Наприклад:

`Of course, I'll help you with the report.` – Це означає, що мовник радий допомогти.

`Will` часто використовують, щоб давати обіцянки або висловлювати рішення в момент мовлення. Наприклад:

`I'll email you the file this afternoon.` – Це показує обіцянку зробити щось найближчим часом.
