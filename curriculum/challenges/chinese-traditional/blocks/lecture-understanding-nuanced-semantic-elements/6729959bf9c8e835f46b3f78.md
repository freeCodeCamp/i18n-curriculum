# --interactive--

這些元素與呈現型和語意型 HTML 的概念非常密切相關。慣用的文字元素 `i` 最初是用於呈現目的，以斜體顯示文字。但現在，它經常用於凸顯標示替代語音或語氣、另一種語言的慣用語、技術術語和想法。

這裡是一個來自官方 HTML 規範的範例，使用 `i` 元素來顯示法語的慣用語句。

:::interactive_editor

```html
<p>There is a certain <i lang="fr">je ne sais quoi</i> in the air.</p>
```

:::

開啟的 `<i>` 標籤內的 `lang` 屬性用來指定內容的語言。在這個例子中，語言會是法語。`i` 元素不會顯示文字是否重要，它只顯示該文字與周圍文字有所不同。

如果你確實需要強調文字的重要性，你可以使用一個類似的語意元素，稱為強調元素 `em`。如果你需要提供更多背景關係，通常建議使用這個元素。你應該將這個元素用於相較於週遭文字需要特別強調的文字部分。通常只限於幾個字，因為它可能會改變句子的意義。

這是在段落中使用強調元素的範例。 

:::interactive_editor

```html
<p>
  Never give up on <em>your</em> dreams.
</p>
```

:::

你可以看到句子 `Never give up on your dreams`。注意字詞 `your` 會被強調，因為它在這個元素內。在瀏覽器中你會看到字詞 `your` 被斜體顯示，以告訴讀者這是句子中重要的字詞。

即使當文字位於慣用語文字元素內時看起來相同，語意強調元素仍在幕後傳達其意義和重要性。

重要的是要知道這些元素不應僅用於呈現目的。如果你需要以斜體顯示文字，但該文字在段落中沒有特殊的用途、樣式或意義，你應該改用 CSS。
