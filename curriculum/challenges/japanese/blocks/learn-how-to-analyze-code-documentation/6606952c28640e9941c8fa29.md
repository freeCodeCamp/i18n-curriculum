# --description--

通常、`with` は接続や使用を示すために使われます。例えば、`I write with a pen.` です。しかし、いくつかの文では、`with` が使われる単語の後に来ることがあります。これは、特に技術や仕事に関する質問や発言で、誰かが使っているものや扱っているものを強調したい場合に起こります。

例えば、技術の分野では、`This program I work with is challenging.` のように言うことがあります。ここで、`with` は扱いにくいプログラムを強調するために文末に使われています。別の例として、`What else is he working with?` のような質問があり、`with` が文末に来て他のツールや技術が使われているかを尋ねています。
