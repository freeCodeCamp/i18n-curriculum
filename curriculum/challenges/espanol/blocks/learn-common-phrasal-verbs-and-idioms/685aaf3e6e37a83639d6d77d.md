# --explanation--

Las oraciones con `before` se usan para mostrar el orden de las acciones. La acción después de `before` ocurre más tarde. Por ejemplo:

`I'll check my calendar before I confirm the meeting.` – Esto significa que la persona primero revisará el calendario y luego confirmará.
