# --description--

Фраза `Got it` часто використовується в розмові, щоб показати, що хтось щось зрозумів. Це невимушений і швидкий спосіб підтвердити, що ви отримали і усвідомили інформацію. Наприклад:

Lucy: `You need to finish the report by tomorrow.`
Bob: `Got it!` 

`Got it` означає, що Боб розуміє інструкцію.
