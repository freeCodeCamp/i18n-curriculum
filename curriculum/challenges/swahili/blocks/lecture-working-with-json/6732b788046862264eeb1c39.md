# --description--

JSON ni nini na unapataje thamani kwa kutumia uandishi wa mabano na nukta?

JSON ni kifupi cha JavaScript Object Notation. Ni muundo wa data mwepesi, wa maandishi unaotumika sana kubadilishana data kati ya seva na programu ya mtandao.

Moja ya sababu kwanini JSON ni maarufu katika ukuzaji wa mtandao ni kwa sababu ni rahisi kwa mashine kusoma na pia ni rahisi kwa binadamu kuelewa.

Kwa kuwa JSON haitegemei lugha yoyote, unaweza kwa urahisi kutuma data za JSON kutoka programu ya Java kwenda programu ya Python, au kutoka programu ya JavaScript kwenda programu ya C#.

JSON inaunga mkono aina nyingi za data ikiwa ni pamoja na vitu, safu za data, mfuatano wa herufi, thamani za kweli au siyo kweli, null, na nambari.

Hapa kuna mfano wa kitu cha JSON:

```js
{
  "name": "Alice",
  "age": 30,
  "isStudent": false,
  "list of courses": ["Mathematics", "Physics", "Computer Science"]
}
```

Kama unavyoona, JSON hutumia jozi za ufunguo-thamani kuhifadhi taarifa na kila jozi hutenganishwa kwa koma. Kila ufunguo lazima ufunwe na alama za nukuu maradufu, vinginevyo utapata kosa.

Ili kupata data kutoka kwa kitu cha JSON, unaweza kutumia uandishi wa nukta au uandishi wa mabano. Katika mfano huu, tunatumia uandishi wa nukta kupata `age` kutoka kwa kitu cha JSON:

```js
import data from "./example.json" with { type: "json" };

console.log(data.age);
```

Mfano huu maalum unatumia kile kinachojulikana kama tamko la `import`, ambalo linaleta kitu cha JSON ndani ya faili hii ili tuwe na upatikanaji wa hicho. Utajifunza zaidi kuhusu tamko la `import` katika funzo lijalo.

Pia unaweza kutumia uandishi wa mabano kupata taarifa kutoka kwa vitu vya JSON. Hapa kuna mfano wa kupata safu ya `list of courses`:

```js
import data from "./example.json" with { type: "json" };

console.log(data["list of courses"]);
```

Kutumia uandishi wa mabano ni muhimu hasa hapa kwa sababu ufunguo una maneno mengi yaliyozuiwa na nafasi. Ikiwa tungejaribu kutumia uandishi wa nukta, ingesababisha kosa.

Kwa muhtasari, JSON ni muundo unaobadilika unaoweza kuhifadhi aina nyingi za data, ikiwa ni pamoja na safu za data na vitu vilivyopangwa ndani. Kwa kutumia uandishi wa nukta au uandishi wa mabano, unaweza kwa urahisi kupata thamani zilizohifadhiwa ndani ya kitu cha JSON.
