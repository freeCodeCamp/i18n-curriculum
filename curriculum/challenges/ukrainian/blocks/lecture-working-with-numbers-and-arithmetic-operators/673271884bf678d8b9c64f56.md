# --interactive--

JavaScript надає інструменти для виконання базових арифметичних дій над числами, таких як додавання, віднімання, множення та ділення. JavaScript також включає оператори для більш складних арифметичних дій, таких як остача та піднесення до степеня.

Усі ці інструменти називаються арифметичними операторами. Давайте детальніше розглянемо ці оператори, як їх використовувати і як їх комбінувати.

Оператор додавання — це знак плюс (`+`). Оператор додавання дозволяє знайти суму двох або більше чисел. У додаванні порядок чисел не має значення:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 + num2;
const result2 = num2 + num1;
const result3 = num2 + num1 + num3;

console.log(result1); // 15
console.log(result2); // 15
console.log(result3); // 30
```

:::

Оператор віднімання — це знак мінус (`-`). Він дозволяє знайти різницю між двома числами. Використовуйте знак мінус, коли хочете відняти одне число від іншого, зазвичай менше від більшого:

:::interactive_editor

```js
const difference = 10 - 5;
console.log(difference); // 5
```

:::

Якщо менше число стоїть першим, ви отримаєте від’ємний результат:

:::interactive_editor

```js
const difference = 5 - 10;
console.log(difference); // -5
```

:::

Ви також можете призначити числа змінним і виконувати віднімання за допомогою імен змінних:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const result = num1 - num2;

console.log(result); // 5
```

:::

В JavaScript оператор множення позначається зірочкою (`*`) і використовується для знаходження добутку двох або більше чисел. Порядок чисел, які ви множите, не має значення:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 * num2;
const result2 = num2 * num1;
const result3 = num2 * num1 * num3;

console.log(result1); // 50
console.log(result2); // 50
console.log(result3); // 750
```

:::

В JavaScript оператор ділення — це коса риска (`/`), яка відрізняється від символу ділення, що використовується в традиційній математиці (`÷`). Ви виконуєте операції ділення за допомогою оператора ділення. У цьому випадку порядок чисел, які ви ділимо, має значення:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 / num2;
const result2 = num2 / num1;
const result3 = num2 / num1 / num3;

console.log(result1); // 2
console.log(result2); // 0.5
console.log(result3); // 0.03333333333333333
```

:::

Важливо зазначити, що якщо ви спробуєте поділити на нуль, JavaScript поверне `Infinity`:

:::interactive_editor

```js
const result = 10 / 0; 

console.log(result); // Infinity
```

:::

Уникайте таких обчислень, щоб не отримати несподівані результати у своєму коді.

Оператор остачі, позначений знаком відсотка (`%`), повертає остачу від ділення. Остача в математиці — це залишкове значення після виконання ділення:

:::interactive_editor

```js
const num1 = 10;
const num2 = 3;
const remainder = num1 % num2;

console.log(remainder); // 1
```

:::

Оператор піднесення до степеня, позначений подвійною зірочкою (`**`), підносить одне число до степеня іншого:

:::interactive_editor

```js
const num1 = 2;
const num2 = 3;

const exponent = num1 ** num2;
console.log(exponent); // 8
```

:::

Можна комбінувати оператори в одній операції або виразі:

:::interactive_editor

```js
const result = 10 + 5 * 2 - 8 / 4;
console.log(result); // 18
```

:::

Коли ви змішуєте різні оператори в одному виразі, движок JavaScript дотримується системи, яка називається пріоритетом операторів, щоб визначити порядок виконання операцій. Пріоритет операторів визначає порядок, у якому виконуються операції у виразах. Ви дізнаєтеся більше про пріоритет операторів у майбутніх уроках.
