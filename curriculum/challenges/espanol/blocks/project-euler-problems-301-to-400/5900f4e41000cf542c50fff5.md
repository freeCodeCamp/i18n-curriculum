# --description--

Sea $S_n$ una secuencia de enteros producida con el siguiente generador de números pseudoaleatorios:

$$\begin{align}
        S_0 & = 290\\,797 \\\\
  S_{n + 1} & = {S_n}^2\bmod 50\\,515\\,093
\end{align}$$

Sea $A(i, j)$ el mínimo de los números $S_i, S_{i + 1}, \ldots, S_j$ para $i ≤ j$. Sea $M(N) = \sum A(i, j)$ para $1 ≤ i ≤ j ≤ N$.

Podemos verificar que $M(10) = 432\,256\,955$ y $M(10\,000) = 3\,264\,567\,774\,119$.

Encuentra $M(2\,000\,000\,000)$.
