# --description--

HTMLページ上のほとんどすべての要素は、開始タグと終了タグで囲まれたコンテンツの断片です。

開始タグはブラウザーにHTML要素の始まりを伝えます。これは山括弧で囲まれたキーワードで構成されています `<>`。例えば、段落の開始タグはこのようになります `<p>`。

終了タグは要素の終わりをブラウザーに伝えます。開始タグとほとんど同じですが、キーワードの前にスラッシュが付いている点が異なります。例えば、段落の終了タグはこのようになります `</p>`。
