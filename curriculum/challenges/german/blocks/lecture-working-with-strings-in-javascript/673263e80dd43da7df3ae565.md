# --interactive--

Wenn Sie mit Strings in JavaScript arbeiten, gibt es Situationen, in denen Sie die Position eines bestimmten Substrings innerhalb eines größeren Strings finden müssen.

Ein Substring ist eine Folge von Zeichen, die innerhalb eines größeren Strings erscheint. Zum Beispiel sind in dem String `hello world` die `hello` und `world` Substrings.

Um die Position eines Substrings innerhalb eines Strings zu finden, können Sie die `indexOf()`-Methode verwenden.
Die `indexOf()`-Methode in JavaScript ermöglicht es Ihnen, nach einem Substring innerhalb eines Strings zu suchen.

Wenn der Substring gefunden wird, gibt `indexOf()` den Index (oder die Position) des ersten Vorkommens dieses Substrings zurück. Wenn der Substring nicht gefunden wird, gibt `indexOf()` `-1` zurück, was anzeigt, dass die Suche erfolglos war.

Die `indexOf()`-Methode nimmt zwei Argumente entgegen: Das erste ist der Substring, den Sie innerhalb des größeren Strings finden möchten, und das zweite ist eine optionale Startposition für die Suche. Wenn Sie keine Startposition angeben, beginnt die Suche am Anfang des Strings.

In diesem Zusammenhang ist ein **Argument** ein Wert, den Sie einer Funktion oder Methode beim Aufrufen übergeben, damit diese Funktion oder Methode ihre Aufgabe mit den von Ihnen bereitgestellten spezifischen Informationen ausführen kann. Sie werden in zukünftigen Lektionen mehr über Argumente lernen.

Hier ist ein Beispiel für die Verwendung der `indexOf()`-Methode, um die Position für den String `awesome` zu finden:

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("awesome!");
console.log(position); // 14
```

:::

In diesem Beispiel beginnt das Wort `awesome` bei Index `14` im String `JavaScript is awesome!`, daher gibt die `indexOf()`-Methode `14` zurück.

Sehen wir uns nun an, was passiert, wenn der Substring nicht gefunden wird:

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("fantastic");
console.log(position); // -1
```

:::

Da das Wort `fantastic` im String nicht vorkommt, gibt die Methode `-1` zurück.

Sie können auch angeben, wo innerhalb des Strings die Suche beginnen soll, indem Sie der `indexOf()` ein zweites Argument übergeben. Hier ist ein Beispiel:

:::interactive_editor

```js
let sentence = "JavaScript is awesome, and JavaScript is powerful!";
let position = sentence.indexOf("JavaScript", 10);
console.log(position); // 27
```

:::

In diesem Fall beginnt die Suche nach `JavaScript` nach dem 10. Zeichen, und so wird das zweite Vorkommen von `JavaScript` bei Index `27` gefunden.

Es ist wichtig zu beachten, dass die `indexOf()`-Methode Groß- und Kleinschreibung unterscheidet.

In diesem Beispiel würde Folgendes `-1` zurückgeben, weil der Großbuchstabe `F` im String `freeCodeCamp` nicht gefunden wird.

:::interactive_editor

```js
console.log("freeCodeCamp".indexOf("F")) // -1
```

:::

Die Verwendung von `indexOf()` kann sehr nützlich sein, wenn Sie überprüfen müssen, ob ein Substring in einem String vorhanden ist, und um dessen Position für weitere Operationen zu bestimmen.
