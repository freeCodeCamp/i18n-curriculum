# --description--

既然你已经知道了 30 多的数字，现在让我们进入 40 多。

在西班牙语中，数字 40 被称为 `cuarenta`。

41 到 49 的数字是通过使用十位词 `cuarenta`（40），后跟连词 `y`（“和”），然后是个位数字（1-9）组成的。

| Number | Spanish |
| :--- | :--- |
| 40 | `Cuarenta` |
| 41 | `Cuarenta y uno` |
| 42 | `Cuarenta y dos` |
| 43 | `Cuarenta y tres` |
| 44 | `Cuarenta y cuatro` |
| 45 | `Cuarenta y cinco` |
| 46 | `Cuarenta y seis` |
| 47 | `Cuarenta y siete` |
| 48 | `Cuarenta y ocho` |
| 49 | `Cuarenta y nueve` |

数字 `cuarenta y uno`（41）在此范围内是独特的，因为它在阳性名词前变为 `cuarenta y un`。例如：

`Cuarenta y un años` - 四十一年。
