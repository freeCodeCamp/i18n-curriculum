# --description--

Un selettore di tipo (o selettore di elemento) seleziona tutti gli elementi del tipo di elemento indicato, e la sintassi è semplicemente il nome dell'elemento:

```html
<!-- index.html -->

<div>Hello, World!</div>
<div>Hello again!</div>
<p>Hi...</p>
<div>Okay, bye.</div>
```

```css
/* styles.css */

div {
  color: white;
}
```

Qui, tutti e tre gli elementi `<div>` verrebbero selezionati, mentre l'elemento `<p>` no.

# --questions--    

## --text--

Quale delle seguenti descrive meglio il codice CSS mostrato sopra?

## --answers--

Il codice applica un colore `white` a tutti gli elementi nel file HTML.

---

Il codice applica un colore `white` a tutti gli elementi `div` nel file HTML.

---

Il codice applica un colore `white` a tutti gli elementi `p` nel file HTML.
