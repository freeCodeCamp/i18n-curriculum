# --description--

**Objetivo:** Cumplir con las historias de usuario a continuación y pasar todas las pruebas para completar el laboratorio.

**Historia de usuario:**

1. Debes tener un `div` con una clase de `form-container` para contener tu formulario.

2. Debes tener un elemento `form` dentro del elemento `.form-container`.

3. Dentro del elemento `form`, debes tener los siguientes elementos y campos de entrada:

   - Un elemento `h2` con algo de texto.
   - Un campo de entrada de texto para el nombre con el `type` establecido en `text` y los atributos `id`, `name`, `required`.
   - Un campo de entrada de correo electrónico con el `type` establecido en `email` y los atributos `id`, `name`, `required`.
   - Un `textarea` para el mensaje con los atributos `id`, `name` y `required`.
   - Un elemento `button` con el texto `"Submit"` y un atributo `type` de `submit`.

4. Cada elemento `input` y `textarea` debe tener su correspondiente elemento `label` con un atributo `for` configurado al `id` del elemento.

5. El elemento `.form-container` debe tener un color de fondo, y tener valores para `border-radius`, `padding` y `width`.

6. Los elementos `label` deben tener un margen y un color de fuente.

7. Los elementos `input` y `textarea` deben tener valores para:

   - `width`.
   - `padding`.
   - Margen en la parte inferior.

8. El elemento `button` debe tener un:

   - Color de fondo
   - Tamaño de fuente diferente al predeterminado.

9.  El botón debe tener un efecto hover que cambie el color de fondo.

**Nota:** Asegúrese de vincular su hoja de estilos en su HTML y aplicar su CSS.
