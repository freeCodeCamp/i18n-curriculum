# --description--

Un altro metodo utile per gli array è il metodo `filter()`. Il metodo `filter()` crea un nuovo array con tutti gli elementi che superano il test implementato dalla funzione fornita. La sintassi del metodo `filter()` è la seguente:

```javascript
const numbers = [2, 5, 6, 1, 9, -1]

const newNumbers = numbers.filter((number) => {
  return number > 2;
});

console.log(newNumbers); // Output: [5, 6, 9]
```

Il metodo `filter()` crea un nuovo array con tutti gli elementi maggiori di 2. Il metodo `filter()` non modifica l'array originale.
