# --explanation--

**間接話法**では、`to suggest` は誰かが与えた助言やアイデアを伝えるために使われます。例えば：

- 直接話法：`I think we should update our firewall.`

- 間接話法：`Jake suggested updating the firewall.`

これは、誰かが正確な言葉を使わずに助言や可能な解決策を伝えたことを示しています。
