# --description--

Il problema di Josephus è un rompicapo matematico con una descrizione cupa: $n$ prigionieri sono disposti in cerchio, numerati in sequenza da $0$ a $n-1$.

Un boia cammina lungo il cerchio, partendo dal prigioniero $0$, eliminando ogni $k$-esimo prigioniero e uccidendolo.

Man mano che il processo procede, il cerchio si restringe sempre di più, fino a quando rimane un solo prigioniero, che viene poi liberato.

Per esempio, se ci sono $n=5$ prigionieri e $k=2$, l'ordine in cui i prigionieri vengono uccisi (chiamiamolo la "sequenza di uccisione") sarà 1, 3, 0 e 4, e il sopravvissuto sarà il #2.

Dato qualsiasi $n, k > 0$, trova quale prigioniero sarà l'ultimo sopravvissuto.

In un caso simile, c'erano 41 prigionieri e veniva ucciso ogni 3<sup>o</sup> prigioniero ($k=3$).

Tra loro c'era un tipo furbo di nome Josephus che risolse il problema, si posizionò nella posizione del sopravvissuto e visse per raccontare la storia.

Quale numero aveva?

# --instructions--

Scrivi una funzione che prende come parametri il numero iniziale di prigionieri e `k` e restituisce il numero del prigioniero che sopravvive.

```js
function josephus(init, kill) {
  const arr = Array.from(Array(init).keys());
  let curr = -1
  while (arr.length > 1) {
    curr = (curr + kill) % arr.length;
    arr.splice(curr, 1);
    curr--;
  }
  return arr[0];
}
```
