# --explanation--

`Code reuse`는 프로그램의 새로운 부분에서 기존 코드를 사용하는 관행을 말합니다. 예를 들어:

`The "Login" class was used in multiple applications to handle user access, which is a good example of code reuse.` - 모든 앱에 로그인 로직을 새로 작성하는 대신 같은 코드를 재사용하여 효율성을 높입니다.

`Extending`는 이미 존재하는 것에 새로운 기능을 추가하는 것을 의미합니다. 예를 들어:

`The "AdminUser" class extends the basic "User" class by adding an "approveUser()" method.` - `AdminUser`는 `User`의 모든 것을 유지하면서 새로운 기능을 덧붙입니다.
