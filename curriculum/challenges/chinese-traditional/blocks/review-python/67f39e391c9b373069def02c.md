## 什麼是 Python？

- **介紹**：Python 是一種通用程式設計語言，以簡潔和易用著稱。Python 被廣泛應用於資料科學與機器學習、網頁開發、腳本與自動化、嵌入式系統與物聯網等多個領域。
- **常見應用場景**：Python 用於資料科學、機器學習、網頁開發、資安、自動化以及像 Raspberry Pi 和支援 MicroPython 的微型電腦。

## 在你的本機環境使用 Python

- **安裝**：在 Windows、Mac 和 Linux 上安裝 Python 的最佳方式是從官方 Python 網站下載安裝程式 (`https://www.python.org/`)。
- **終端機**：一種以文字為基礎的介面，讓你透過輸入指令與電腦互動。每個作業系統都附帶預設的終端機應用程式。在 macOS 上，你可以使用 Terminal 應用程式；在 Windows 上，可以使用命令提示字元或 PowerShell；在 Linux 上，每個桌面環境都有自己的預設終端機應用程式，如 GNOME Terminal 或 Konsole。
- **整合開發環境（IDE）**：IDE 是整合開發環境，具備程式碼編輯器、測試工具、終端機等功能。
- **Python 常用的程式碼編輯器和 IDE**：VS Code、PyCharm 和 Spyder
- **本機執行程式碼**：執行 Python 程式碼有幾種選擇。第一種是點擊 VS Code 右上角像播放按鈕的執行按鈕，這會開啟終端機並執行你的 Python 腳本。

另一種選擇是開啟終端機並手動執行程式。以下是執行名為 `main.py` 的程式範例：

```bash
python main.py
```

你必須在 `main.py` 所在的資料夾執行此指令。例如，如果 `main.py` 位於名為 `python-projects` 的資料夾中，先在終端機輸入：

```bash
cd python-projects
```

接著執行：

```bash
python main.py
```

## 使用 Python 互動式 Shell

- **定義**：互動式 shell 是一個程式，讓你一次輸入一個指令並立即看到結果。打開終端機應用程式，輸入 `python` 並按下 `Enter`。這會啟動 Python 互動式 shell。`>>>` 符號表示 Python 正在等待你輸入指令。試著在終端機輸入列印 `"Hello, world!"`：

```bash
print("Hello, world!")
```

你應該會看到文字如下顯示：

```bash
>>> print("Hello, world!")
Hello, world!
```

文字列印完後，Python 會回到以下狀態：

```bash
>>>
```

這表示你可以繼續輸入下一個指令。

- **讀取、求值、列印、迴圈循環（REPL）**：每當你輸入指令時，直譯器會讀取輸入、求值、列印結果，然後回到 `>>>`，讓你繼續輸入更多指令。

## 變數

- **宣告變數**：宣告變數時，先寫變數名稱，接著是指定運算子 (`=`)，然後是值。值可以是數字、字串、布林值等。以下是一些範例：

```py
name = 'John Doe'
age = 25
```

- **變數命名規則**：變數命名應遵守以下規則：

- 變數名稱只能以字母或底線（_）開頭，不能以數字開頭。
  - 變數名稱只能包含字母、數字和底線（_）。
  - 變數名稱區分大小寫 — `age`、`Age` 和 `AGE` 都被視為不同名稱。
  - 變數名稱不能是 Python 保留關鍵字，如 `if`、`class` 或 `def`。
  - 多字變數名稱用底線分隔，例如 `snake_case`。


## 註解

- **單行註解**：用於在程式碼中留下簡短的備註。

```py
# This is a single line comment
```

- **多行字串**：可用來留下較長的備註或註解掉程式碼區段。

```py
"""
This is a multi-line string.
Here is some code commented out.

name = 'John Doe'
age = 25
"""
```

- **`print()` 函式**：要在主控台列印資料，可以使用 `print()` 函式，如下：

```py
print('Hello world!') # Hello world!
```

## Python 常見資料型別

- **介紹**：Python 是動態型別語言，像 JavaScript 一樣，你不需要明確宣告變數型別。語言會根據你指定給變數的值自動判斷型別。
- **整數**：沒有小數點的整數：

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **浮點數**：帶有小數點的數字：

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **字串**：用引號包裹的字元序列：

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **布林值**：代表 `True` 或 `False` 的值：

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **集合**：無序且元素唯一的群集：

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **字典**：由鍵值對組成的群集，用大括號包住：

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **元組**：不可變的有序群集，用小括號包住：

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **範圍**：數字序列，常用於迴圈：

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **列表**：有序且可包含不同資料型別的群集：

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**：代表沒有值的特殊值：

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## 不可變與可變型別

- **不可變型別**：宣告後無法改變，但可以將變數指向新的值，稱為重新指派。包括整數、浮點數、複數、布林值、字串、元組、範圍和 `None`。
- **可變型別**：宣告後可以改變，可以新增、刪除或更新項目。包括列表、集合和字典等群集型別。
- **`type()` 函式**：用於檢查變數類型的 `type()` 函數如下所示：

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **`isinstance()` 函式**：用來檢查變數是否符合特定資料型別：

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## 字串操作

- **定義**：如你從 JavaScript 所知，字串是不可變的，代表建立後無法更改。在 Python 中，你可以使用單引號或雙引號。建議選擇一種規則並持續使用：

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **存取字串字元**：可用中括號表示法存取字串字元，如下：

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **字串跳脫**：如果字串中包含引號，可以用反斜線 (`\`) 跳脫，如下：

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **字串串接**：可用 `+` 運算子串接字串，如下：

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

另一種串接字串方式是用 `+=` 運算子，這會同時執行串接和指派，如下：

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**：格式化字串字面值的簡寫，允許你用簡潔易讀的語法做插值和串接：

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **字串切片**：可擷取字串部分內容，基本語法如下：

```py
str[start:stop:step]
```

起始位置是擷取開始的索引，結束位置是擷取結束的索引（不包含該位置），步進表示切片的間隔。以下是範例：

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **取得字串長度**：用 `len()` 函式回傳字串字元數：

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## 使用 `in` 運算子

- **`in` 運算子**：回傳布林值，表示字串中是否存在指定字元或字串：

```py
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

## 常用字串方法

- **`upper()`**：將字串所有字元轉成大寫並回傳新字串：

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`lower()`**：將字串所有字元轉成小寫並回傳新字串：

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`strip()`**：回傳去除指定前後字元的字串副本（若無參數則去除前後空白）：

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**：回傳將字串中所有指定字串替換成新字串的新字串：

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**：用指定分隔符將字串拆分成列表。分隔符是指定拆分位置的字串：

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**：用分隔符將可迭代物件的元素串接成字串。可迭代物件是可被迴圈遍歷的集合，如列表、字串或元組：

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`startswith(prefix)`**：判斷字串是否以指定前綴開頭，回傳布林值：

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`endswith(suffix)`**：判斷字串是否以指定後綴結尾，回傳布林值：

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`find()`**：回傳子字串首次出現的索引，找不到則回傳 `-1`：

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`count(substring)`**：計算子字串在字串中出現的次數：

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`capitalize()`**：回傳首字母大寫、其他字母小寫的新字串：

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`isupper()`**：若字串所有字母皆為大寫，回傳 `True`，否則回傳 `False`：

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`islower()`**：若字串所有字母皆為小寫，回傳 `True`，否則回傳 `False`：

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`title()`**：回傳每個單字首字母皆大寫的新字串：

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`maketrans()`**：用來建立字元對應表，常與 `translate()` 方法搭配使用，將字串轉換成對應結果：

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## 整數與浮點數常用運算

- **基本數學運算**：Python 可對整數和浮點數做加、減、乘、除等基本運算：

```py
int_1 = 56
int_2 = 12
float_1 = 5.4
float_2 = 12.0

# Addition

print('Integer Addition:', int_1 + int_2) # Integer Addition: 68
print('Float Addition:', float_1 + float_2) # Float Addition: 17.4

# Subtraction

print('Int Subtraction:', int_1 - int_2) # Int Subtraction: 44
print('Float Subtraction:',  float_2 - float_1) # Float Subtraction: 6.6

# Multiplication

print('Int Multiplication:', int_1 * int_2) # Int Multiplication: 672
print('Float Multiplication:', float_2 * float_1) # Float Multiplication: 64.80000000000001

# Division

print('Division:', int_1 / int_2) # Division: 4.666666666666667
print('Float Division:', float_2 / float_1) # Float Division: 2.222222222222222
```

當加法運算中有浮點數和整數時，結果會轉成浮點數，如下：

```py
int_1 = 56
float_1 = 5.4

print(int_1 + float_1) # 61.4
```

- **取餘數運算子（`%`）**：回傳除法的餘數：

```py
int_1 = 56
int_2 = 12

print(int_1 % int_2) # 8
```

- **向下取整除法（`//`）**：將兩數相除後向下取整到最接近的整數：

```py
int_1 = 56
int_2 = 12

print(int_1 // int_2) # 4
```

- **次方運算子（`**`）**：將一數字提升到另一數字的次方：

```py
int_1 = 4
int_2 = 2

print(int_1 ** int_2) # 16
```

- **`float()` 函式**：用來將整數轉成浮點數：

```py
num = 4

print(float(num)) # 4.0
```

- **`int()` 函式**：用來將浮點數轉成整數：

```py
num = 4.0

print(int(num)) # 4
```

- **`round()` 函式**：用來將數字四捨五入到最接近的整數：

```py
num_1 = 3.4
num_2 = 7.7

print(round(num_1)) # 3
print(round(num_2)) # 8
```

- **`abs()` 函式**：用來回傳數字的絕對值：

```py
num = -13

print(abs(num)) # 13
```

- **`pow()` 函式**：用來將數字提升到另一數字的次方：

```py
result = pow(2, 3) 
print(result)  # 8
```

## 使用函式

- **定義**：函式是可重複使用的程式碼區塊，接受輸入（引數）並回傳輸出。呼叫函式時，需寫函式名稱後接一對小括號：

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call
print(result) # 7
```

若函式沒有明確回傳值，預設回傳值為 `None`：

```py
def greet():
    print('hello') 

result = greet() # hello
print(result) # None
```

你也可以為參數指定預設值，如下：

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

若呼叫函式時引數數量不正確，會得到 `TypeError`：

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

## Python 的作用域

- **區域作用域**：在函式或類別內宣告的變數只能在該函式或類別內存取。

```py
def my_func():
    num = 10
    print(num)
```

- **封閉作用域**：巢狀函式可以存取其外層函式的變數。

```py
def outer_func():
    msg = 'Hello there!'
    
    def inner_func():
        print(msg)
    inner_func()

print(outer_func()) # Hello there!
```

- **全域作用域**：在任何函式或類別外宣告的變數，可在程式任何地方存取。

```py
tax = 0.70 

def get_total(subtotal):
    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **內建作用域**：Python 保留名稱，用於預定義函式、模組、關鍵字和物件。

```py
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```

## 使用 `if`、`elif` 和 `else` 語句

- **`if` 語句**：用來判斷條件是否為真。若條件評估為 `True`，則執行該區塊程式碼。

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **`elif` 子句**：跟在 `if` 語句後的條件。若 `elif` 條件評估為 `True`，則執行該區塊程式碼。

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **`else` 子句**：當其他條件皆不為 `True` 時執行。

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

你也可以使用巢狀 `if` 語句，如下：

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## 真值與假值

- **定義**：在 Python 中，每個值都有內建的布林值，表示在邏輯判斷中應視為 `True` 或 `False`。許多值被視為真值，意即在邏輯判斷中評估為 `True`。其他值則為假值，評估為 `False`。以下是假值範例：

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

其他值如非零數字和非空字串則為真值。

## 使用 `bool()` 函式

- **定義**：若想檢查一個值是真值或假值，可以使用內建 `bool()` 函式。它會將值明確轉成布林值，真值回傳 `True`，假值回傳 `False`。以下是幾個範例：

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## 布林運算子與短路運算

- **定義**：這些特殊運算子讓你結合多個表達式，建立更複雜的決策邏輯。Python 有三個布林運算子：`and`、`or` 和 `not`。
- **`and` 運算子**：接受兩個運算元，若第一個為假值，回傳第一個，否則回傳第二個。兩個運算元都必須為真值，表達式才為真。

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

你也可以在條件判斷中使用 `and` 運算子，如下：

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **`or` 運算子**：若第一個運算元為真值，回傳第一個，否則回傳第二個。只要有一個運算元為真，`or`表達式即為真。範例如下：

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

就像 `and` 運算子一樣，你可以在條件判斷中使用 `or` 運算子：

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **短路運算**：`and` 和 `or` 運算子稱為短路運算子。短路運算表示 Python 從左到右檢查值，當確定結果時即停止。
- **`not` 運算子**：接受單一運算元，反轉其布林值。將真值轉為 `False`，假值轉為 `True`。與前述運算子不同，`not` 總是回傳 `True` 或 `False`。以下是範例：

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

以下是 `not` 運算子在條件判斷中的範例：

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```


## Python 列表

- **介紹**：Python 中的列表是有序元素序列，可以包含字串、數字甚至其他列表。列表是可變且以 0 為起始索引。

```python
cities = ['Los Angeles', 'London', 'Tokyo']
```

- **存取列表元素**：要存取 `cities` 列表中的元素，可以用索引號碼：

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # Los Angeles
```

- **使用負索引存取元素**：要存取列表最後一個元素，可以用 `-1` 作為索引：

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # Tokyo
```

- 負索引是從列表尾端開始存取元素，而非從起始索引 `0` 開始。

- **使用 `list()` 建構式建立列表**：列表也可以用 `list()` 建構式建立。`list()` 建構式用來將可迭代物件轉成列表：

```python
developer = 'Jessica'

print(list(developer)) 
# Result: ['J', 'e', 's', 's', 'i', 'c', 'a']
```

- **取得列表長度**：用 `len()` 函式取得列表長度：

```python
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

- **列表的可變性**：列表是可變的，只要索引有效，就能更新列表中任一元素。更新列表元素可將新值指定給該索引：

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

- **索引超出範圍錯誤**：若索引（正或負）超出列表範圍，會得到 `IndexError`：

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
"""
```

- **從列表移除元素**：可用 `del` 關鍵字移除列表元素：

```python
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

- **檢查元素是否存在列表中**：可用 `in` 關鍵字檢查元素是否存在列表：

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **列表巢狀**：列表可以巢狀在其他列表中：

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

- 要存取巢狀列表，需用索引 `2`，因為列表是從 0 開始索引。

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

- 若要進一步存取該巢狀列表中的第二個語言，需用索引 `1`：

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # Rust
```

- **從列表解包值**：解包是將列表中的值指派給新變數的技巧。以下範例將 `developer` 列表解包到名為 `name`、`age` 和 `job` 的變數：

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer
```

- 若指派運算子左側變數數量與列表項目數不符，會得到 `ValueError`。

- **收集列表剩餘項目**：可用星號（`*`）運算子收集列表剩餘元素，如下：

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer
```

- **列表切片**：切片是用切片運算子 `:` 存取列表部分元素。要切片從索引 `1` 開始到索引 `3`（不包含）的元素，可用以下語法：

```python
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie']
desserts[1:3] # ['Cookies', 'Ice Cream']
```

- **步進間隔**：也可以指定步進間隔，決定索引間隔。以下範例示範如何用切片擷取偶數列表：

```python
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

```py
numbers = [1, 2, 3, 4, 5]
```
```py
even_numbers = [6, 8, 10]
```

- **insert()**：用來在列表特定索引插入項目。以下範例示範 `insert()` 方法：

```py
numbers = [1, 2, 3, 4, 5]
```

- **sort()**：`sort()` 方法用來就地排序列表元素。以下範例示範就地排序隨機 `numbers` 列表：

- **reverse()**：用來反轉列表元素順序：

```py
numbers = [6, 5, 4, 3, 2, 1]
```

- **index()**：用來尋找元素在列表中首次出現的索引：

```python
programming_languages = ['Rust', 'Java', 'Python', 'C++']
```

- 若用 `index()` 方法找不到元素，結果會是 `ValueError`。

## Python 元組

- **定義**：元組是 Python 資料型別，用來建立有序值序列。元組可包含混合型別資料：

```py
developer = ('Alice', 34, 'Rust Developer')
```

- 元組是不可變的，建立後無法更改元素。若嘗試更新元組中的項目，會得到 `TypeError`：

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **存取元組元素**：用中括號表示法和索引號碼存取元組元素：

```py
developer = ('Alice', 34, 'Rust Developer')

```

- 可用負索引從元組尾端存取元素：

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- 若索引號碼超過或等於元組長度，會得到 `IndexError`：

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- 元組也可用 `tuple()` 建構式建立。建構式中可傳入字串、列表甚至其他元組等可迭代物件。

```python
developer = 'Jessica'

print(tuple(developer))
```

- **檢查元組中項目**：用 `in` 關鍵字檢查項目是否在元組中，如下：

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
```

- **解包元組**：可用以下方式解包元組項目：

```py
developer = ('Alice', 34, 'Rust Developer')
```

- 若要收集元組剩餘元素，可用星號（`*`）運算子，如下：

```py
developer = ('Alice', 34, 'Rust Developer')

```

- **切片元組**：切片可擷取元組部分元素。例如，將 `pie` 和 `cookies` 項目切片成新元組：

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
```

- **移除元組項目**：移除元組項目會引發 `TypeError`，因為元組是不可變的：

```py
developer = ('Jane Doe', 23, 'Python Developer')

TypeError: "tuple" object doesn't support item deletion

```

- **何時使用元組或列表？**：若需要動態集合，可新增、刪除和更新元素，應使用列表。若資料是固定且不可變，則應使用元組。

## 常用的 Tuple 方法

- **`count()`**：用來判斷某個項目在 tuple 中出現了多少次。例如，你可以檢查語言 `'Rust'` 在 tuple 中出現了幾次：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

- 如果 `count()` 方法中指定的項目在 tuple 中完全不存在，則回傳值會是 `0`：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

- 如果沒有傳入任何引數給 `count()` 方法，Python 會回傳 `TypeError`。

- **index()**：用來尋找特定項目在 tuple 中出現的索引。以下是使用 `index()` 方法尋找語言 `'Java'` 索引的範例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

- 如果找不到指定的項目，Python 會回傳 `ValueError`。

- 你可以傳入選用的起始索引給 `index()` 方法，指定從 tuple 中哪裡開始搜尋該項目：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

- 你也可以傳入選用的結束索引給 `index()` 方法，指定在哪裡停止搜尋該項目：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

- **`sorted()`**：用來對任何可迭代物件的元素進行排序，並回傳一個新的排序後列表。以下是使用 `sorted()` 函式建立一個新的數字列表的範例：

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

- **修改排序行為**：你可以使用選用的 `reverse` 和 `key` 引數來自訂可迭代物件的排序行為。以下是使用 `key` 引數依照長度排序 tuple 項目的範例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

- 你可以使用 `reverse` 引數來建立一個反向排序的新列表，如下所示：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)

"""
Result 

Rust
Java
Python
C++
"""
```

- 以下是使用 `for` 迴圈遍歷字串 `code` 並列印每個字元的範例：

```py
for char in 'code':
    print(char)

"""
Result 

c
o
d
e
"""
```

- `for` 迴圈可以巢狀使用。以下是使用巢狀 `for` 迴圈的範例：

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)

Result

Fruit Apple

Vegetable Banana
"""

```

- **`while` 迴圈**：重複執行一段程式碼直到條件為 `False`。以下是使用 `while` 迴圈進行猜數字遊戲的範例：

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')

"""
Result

Wrong! Try again.

```

- **`break` 和 `continue` 陳述式**：用於迴圈中修改迴圈的執行流程。

- `break` 陳述式用來在特定條件成立時立即離開迴圈。以下是使用 `break` 陳述式處理 `developer_names` 列表的範例：

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

- `continue` 陳述式用來跳過當前迭代並繼續下一次迭代。以下是使用 `continue` 陳述式取代 `break` 陳述式的範例：

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

- `for` 和 `while` 迴圈都可以搭配 `else` 子句使用，該子句只會在迴圈沒有被 `break` 終止時執行：

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:

            print(f"'{word}' contains the vowel '{letter}'")

    else:
        print(f"'{word}' has no vowels")
```

## 範圍（Ranges）及其在迴圈中的使用

- **`range()` 函式**：用來產生一連串整數。

```py
range(start, stop, step)
```

- 必填的 `stop` 引數是一個整數（不包含該值），代表產生數字序列的終點。以下是使用 `range()` 函式的範例：

```py
for num in range(3):
    print(num)
```

- 如果沒有指定 `start` 引數，預設值會是 `0`。預設情況下，整數序列會以 `1` 遞增。你可以使用選用的 `step` 引數來更改預設的遞增值。以下是產生從 2 到不包含 11（即包含 10）的偶數序列的範例：

```py
for num in range(2, 11, 2):
    print(num)
```

- 如果你不傳入任何引數給 `range()` 函式，則會得到 `TypeError`。

- `range()` 函式只接受整數作為引數，不接受浮點數。使用浮點數會導致 `TypeError`：

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- 你可以對 `step` 引數使用負整數，來產生遞減的整數序列：

```py
for num in range(40, 0, -10):
    print(num)
```

- `range()` 函式也可以搭配 `list` 建構式使用，來建立整數列表。`list` 建構式用來將可迭代物件轉換成列表。以下是產生包含 2 到 10（含）的偶數列表的範例：

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

## Python 中的 `enumerate()` 和 `zip()` 函式

- **`enumerate()`**：用來對序列進行迭代，並追蹤序列中每個項目的索引。`enumerate()` 函式接受一個可迭代物件作為引數，並回傳一個 `enumerate` 物件，該物件包含可迭代物件中每個項目的索引和值。

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')

# Result
# Index 0 and language Spanish
# Index 1 and language English
# Index 2 and language Russian
# Index 3 and language Chinese

```

- `enumerate()` 函式也可以在 `for` 迴圈外使用：

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

print(list(enumerate(languages)))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

- `enumerate()` 函式接受一個選用的 `start` 引數，指定計數的起始值。如果省略此引數，計數將從 `0` 開始。

- **`zip()`**：用來平行迭代多個可迭代物件。以下是使用 `zip()` 函式同時迭代 `developers` 和 `ids` 的範例：

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, dev_id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {dev_id}')


"""
Result

Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
```

## Python 中的列表推導式

- **定義**：列表推導式允許你在一行內結合迴圈和條件，直接在中括號內建立新的列表。這讓程式碼更簡潔且通常更易讀。

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

## 可迭代物件函式

- **`filter()`**：用來根據條件過濾可迭代物件中的元素。它回傳一個只包含符合條件元素的迭代器。以下是建立一個只包含字元數超過四個字的單字新列表的範例：

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**：用來對可迭代物件中的每個項目套用函式，並回傳一個包含結果的新可迭代物件。以下是使用 `map()` 函式將攝氏溫度列表轉換為華氏溫度的範例：

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**：用來取得可迭代物件（如列表或 tuple）的總和。以下是使用 `sum()` 函式的範例：

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- 你也可以傳入選用的 `start` 引數，設定加總的初始值。以下是使用 `start` 引數作為位置引數的更新範例：

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- 你也可以選擇使用 `start` 引數作為關鍵字引數，如下所示：

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## Lambda 函式

- **定義**：Python 中的 lambda 函式是一種簡潔的方式來建立匿名函式（沒有名稱的函式）。
- Lambda 函式常用作為其他函式的引數。以下是 lambda 函式的範例：

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

- 使用 lambda 函式的最佳實踐包括不將它們指定給變數，保持簡潔易讀，並用於短小且一次性的函式。

## 字典（Dictionaries）

- **字典**：字典是內建的資料結構，用來儲存鍵值對集合。鍵必須是不可變的資料型別。以下是 Python 字典的一般語法：

```python
dictionary = {
    key1: value1,
    key2: value2
}
```

- **`dict()` 建構式**：`dict()` 建構式是建立字典的另一種方式。你可以將一個包含鍵值對 tuple 的列表作為引數傳入 `dict()` 建構式。這些 tuple 的第一個元素是鍵，第二個元素是值。

```python
pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
```

- **中括號表示法**：要存取鍵值對的值，可以使用稱為中括號表示法的語法。

```python
dictionary[key]
```

## 常用字典方法

- **`get()` 方法**：`get()` 方法用來取得與鍵相關聯的值。它類似中括號表示法，但允許你設定預設值，避免鍵不存在時發生錯誤。

```python
dictionary.get(key, default)
```

- **`keys()` 和 `values()` 方法**：`keys()` 和 `values()` 方法分別回傳字典中所有鍵和值的檢視物件。檢視物件是一種查看字典內容而不建立資料複本的方式。

```python
pizza = {
    'name': 'Margherita Pizza',
    'price': 8.9,
    'calories_per_slice': 250
}

pizza.keys()
# dict_keys(['name', 'price', 'calories_per_slice'])

pizza.values()
# dict_values(['Margherita Pizza', 8.9, 250])
```

- **`items()` 方法**：`items()` 方法回傳包含字典所有鍵值對的檢視物件，包含鍵和值。

```python
pizza.items()
# dict_items([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250)])
```

- **`clear()` 方法**：`clear()` 方法會移除字典中所有的鍵值對。

```python
pizza.clear()
```

- **`pop()` 方法**：`pop()` 方法會移除指定為第一個引數的鍵值對，並回傳其值。如果鍵不存在，則回傳第二個引數指定的預設值。如果鍵不存在且未指定預設值，則會引發 `KeyError`。

```python
pizza.pop('price', 10)
pizza.pop('total_price') # KeyError
```

- **`popitem()` 方法**：在 Python 3.7 及以上版本，`popitem()` 方法會移除最後插入的項目。

```python
pizza.popitem()
```

- **`update()` 方法**：`update()` 方法會用另一個字典的鍵值對更新字典。如果有相同的鍵，值會被覆寫。新的鍵會被加入字典。

```python
pizza.update({ 'price': 15, 'total_time': 25 })
```

## 遍歷字典

- **遍歷值**：如果你需要遍歷字典中的值，可以寫一個 `for` 迴圈搭配 `values()` 來取得字典的所有值。

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,
    'Headphones': 70,
}

for price in products.values():
    print(price)
```

輸出：

```md
990
600
250
70
```

- **遍歷鍵**：如果你需要遍歷上述 `products` 字典的鍵，可以直接寫 `products.keys()` 或 `products`。

```python
for product in products.keys():
    print(product)
    
# Or

for product in products:
    print(product)
```

輸出：

```md
Laptop
Smartphone
Tablet
Headphones
```

- **遍歷鍵值對**：如果你需要同時遍歷鍵和對應的值，可以遍歷 `products.items()`。你會得到包含鍵和值的個別 tuple。

```python
for product in products.items():
    print(product)
```

輸出：

```md
('Laptop', 990)
('Smartphone', 600)
('Tablet', 250)
('Headphones', 70)
```

要將鍵和值分別存放在不同的迴圈變數中，需要用逗號分隔。第一個變數存放鍵，第二個存放值。

```python
for product, price in products.items():
    print(product, price)
```

輸出：

```md
Laptop 990
Smartphone 600
Tablet 250
Headphones 70
```

- **`enumerate()` 函式**：如果你需要在遍歷字典時同時追蹤計數器，可以呼叫 `enumerate()` 函式。該函式回傳一個 `enumerate` 物件，會為每個項目分配一個整數計數器。你可以從任意數字開始計數，但預設從 0 開始。

將索引和值分別指定給不同的迴圈變數是使用 `enumerate()` 的常見方式。例如，使用 `products.items()`，你可以取得索引以及整個鍵值對：

```python
for index, product in enumerate(products.items()):
    print(index, product)
```

輸出：

```md
0 ('Laptop', 990)
1 ('Smartphone', 600)
2 ('Tablet', 250)
3 ('Headphones', 70)
```

要自訂計數的起始值，可以傳入第二個引數給 `enumerate()`。以下範例從 1 開始計數。

```python
for index, product in enumerate(products.items(), 1):
    print(index, product)
```

輸出：

```md
1 ('Laptop', 990)
2 ('Smartphone', 600)
3 ('Tablet', 250)
4 ('Headphones', 70)
```

## 集合（Sets）

- **集合**：集合是 Python 內建的資料結構，不允許重複值。集合是可變且無序的，這表示元素沒有特定順序，因此無法用索引或鍵來存取。集合只能包含不可變資料型別的值，例如數字、字串和 tuple。

- **定義集合**：要定義集合，需要將元素寫在大括號內並用逗號分隔。

```python
my_set = {1, 2, 3, 4, 5}
```

- **定義空集合**：如果你需要定義空集合，必須使用 `set()` 函式。只寫空的大括號會自動建立字典。

```python
set() # Set
{}    # Dictionary
```

## 常用集合方法

- **`add()` 方法**：你可以使用 `add()` 方法將元素加入集合，並將新元素作為引數傳入。

```python
my_set.add(6)
```

- **`remove()` 和 `discard()` 方法**：要從集合中移除元素，可以使用 `remove()` 方法或 `discard()` 方法，並將要移除的元素作為引數傳入。`remove()` 方法在找不到元素時會引發 `KeyError`，而 `discard()` 方法則不會。

```python
my_set.remove(4)
my_set.discard(4)
```

- **`clear()` 方法**：`clear()` 方法會移除集合中所有元素。

```python
my_set.clear()
```

## 數學集合運算

- **`issubset()` 和 `issuperset()` 方法**：`issubset()` 和 `issuperset()` 方法分別用來檢查一個集合是否為另一個集合的子集合或超集合。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 5}

print(your_set.issubset(my_set)) # True
print(my_set.issuperset(your_set)) # True
```

- **`isdisjoint()` 方法**：`isdisjoint()` 方法用來檢查兩個集合是否不相交，即沒有共同元素。

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

print(my_set.isdisjoint(your_set)) # True
```

- **聯集運算子 (`|`)**：聯集運算子 `|` 回傳一個包含兩個集合所有元素的新集合。

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

my_set | your_set # {1, 2, 3, 4, 5, 6}
```

- **交集運算子 (`&`)**：交集運算子 `&` 回傳一個只包含兩個集合共有元素的新集合。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set & your_set # {2, 3, 4}
```

- **差集運算子 (`-`)**：差集運算子 `-` 回傳一個包含第一個集合中不在第二個集合的元素的新集合。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set - your_set # {1, 5}
```

- **對稱差集運算子 (`^`)**：對稱差集運算子 `^` 回傳一個包含只存在於第一或第二集合，但不在兩者共有的元素的新集合。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set ^ your_set # {1, 5, 6}
```

- **`in` 運算子**：你可以使用 `in` 運算子檢查元素是否存在於集合中。

```python
print(5 in my_set)
```

## Python 標準程式庫

- **Python 標準程式庫**：程式庫提供你預先撰寫且可重複使用的程式碼，例如函式、類別和資料結構，你可以在專案中重複使用。Python 擁有豐富的標準程式庫，內建模組實作了許多問題和任務的標準解決方案。常見的內建模組範例有 `math`、`random`、`re`（正規表達式的縮寫）和 `datetime`。

## 匯入陳述式

- **匯入陳述式**：要存取內建模組中定義的元素，你使用匯入陳述式。匯入陳述式通常寫在檔案頂端。匯入陳述式對函式、類別、常數、變數以及模組中定義的其他元素都適用。

- **基本匯入陳述式**：你可以使用 `import` 關鍵字，後面接模組名稱：

```python
import module_name
```

接著，如果你需要呼叫該模組的函式，可以使用點記法，模組名稱後接函式名稱。

```python
module_name.function_name()
```

例如，你可以在程式碼中寫下以下內容，匯入 `math` 模組並取得 36 的平方根：

```python
import math

math.sqrt(36)
```

- **以不同名稱匯入模組**：如果你需要用不同名稱（別名）匯入模組，可以在匯入陳述式末尾使用 `as`，後面接別名。這通常用於模組名稱過長或避免命名衝突。

```python
import module_name as module_alias
```

例如，若要在程式碼中將 `math` 模組稱為 `m`，可以這樣指定別名：

```python
import math as m
```

然後，你可以使用別名存取模組的元素：

```python
m.sqrt(36)
```

- **匯入特定元素**：如果你不需要整個模組，可以使用 `from` 匯入特定元素。此時匯入陳述式以 `from` 開頭，接著是模組名稱，再用 `import` 關鍵字，最後列出要匯入的元素名稱。

```python
from module_name import name1, name2
```

接著，你可以在 Python 腳本中直接使用這些名稱，而不需模組前綴。例如：

```python
from math import radians, sin, cos

angle_degrees = 40
angle_radians = radians(angle_degrees)

sine_value = sin(angle_radians)
cos_value = cos(angle_radians)

print(sine_value) # 0.6427876096865393
print(cos_value)  # 0.766044443118978
```
 
這很方便，但如果你已有同名的函式或變數，可能會導致命名衝突。選擇匯入方式時要注意。

如果你需要為這些名稱指定別名，也可以使用 `as` 關鍵字，後面接別名。

```python
from module_name import name1 as alias1, name2 as alias2
```

- **使用星號（`*`）的匯入陳述式**：星號告訴 Python 你想匯入該模組的所有內容，但不想使用模組名稱作為前綴。

```python
from module_name import *
```

例如，使用此方式匯入 `math` 模組後，你可以直接呼叫該模組中定義的任何函式，而不需指定模組名稱。

```python
from math import *
print(sqrt(36))  # 6.0
```

不過，這通常不建議使用，因為可能導致命名空間衝突，且難以判斷名稱來源。

## `if __name__ == '__main__'`

- **`__name__` 變數**：`__name__` 是 Python 中一個特殊的內建變數。當 Python 檔案被直接執行時，Python 會將此變數的值設為字串 `"__main__"`。但如果該 Python 檔案被匯入為另一個 Python 腳本的模組，`__name__` 變數的值會設為該模組的名稱。

這就是為什麼你常在 Python 腳本中看到這個條件判斷。它包含了你只想在 Python 腳本作為主程式執行時才執行的程式碼。

```python
if __name__ == '__main__': 
    # Code
```

## Python 常見錯誤

- **SyntaxError**：當你的程式碼不符合語法規則時，Python 會引發此錯誤。例如，程式碼 `print("Hello there"` 會導致語法錯誤，錯誤訊息為 `SyntaxError: '(' was never closed`，因為程式碼缺少結尾的括號。
- **NameError**：當你嘗試存取未定義的變數或函式時，Python 會引發 `NameError`。例如，如果你的程式碼中有 `print(username)`，但沒有先定義 `username` 變數，會得到錯誤訊息 `NameError: name 'username' is not defined`。
- **TypeError**：當你對兩個或多個不相容的資料型別執行操作時，Python 會丟出此錯誤。例如，嘗試將字串與數字相加，會得到錯誤 `TypeError: can only concatenate str (not "int") to str`。
- **IndexError**：如果你存取列表或其他序列（如 tuple 和字串）中不存在的索引，會得到 `IndexError`。例如，在 `Hello world` 字串中，最後一個字元的索引是 `10`。如果你嘗試用 `greet = "hello world"; print(greet[11])` 方式存取字元，會得到錯誤訊息 `IndexError: string index out of range`。
- **AttributeError**：當你嘗試在該型別的物件上使用不存在的方法或屬性時，Python 會引發此錯誤。例如，對字串 `"hello".append("!")` 呼叫 `.append()`，會導致錯誤訊息 `AttributeError: 'str' object has no attribute 'append'`。

- **使用 Python 內建除錯器（`pdb`）**：Python 提供 `pdb` 模組用於除錯。它是 Python 標準程式庫的一部分，隨時可用。使用 `pdb`，你可以用 `set_trace()` 函式設定追蹤，開始逐步執行程式碼並以互動方式檢查變數。

## 例外處理

- **`try...except`**：用來執行可能引發例外的程式區塊。`try` 區塊是你預期可能發生錯誤的地方，而 `except` 區塊會捕捉指定的例外並執行對應程式。範例如下：

  ```py
  try:
    print(22 / 0)
  except ZeroDivisionError:
    print('You can\'t divide by zero!')
    # You can't divide by zero!
  ```

你也可以串接多個 `except` 區塊，以處理更多種類的例外：

  ```py
  try:
    number = int(input('Enter a number: '))
    print(22 / number)
  except ZeroDivisionError:
    print('You cannot divide by zero!')
    # You cannot divide by zero! prints when you enter 0
  except ValueError:
    print('Please enter a valid number!')
    # Please enter a valid number! prints when you don't enter an integer
  ```

- **`else` 和 `finally`**：這些區塊是 `try...except` 的延伸。如果沒有發生例外，會執行 `else` 區塊。`finally` 區塊無論是否有錯誤都會執行。

  ```py
  try:
    result = 100 / 4
  except ZeroDivisionError:
    print('You cannot divide by zero!') # This will not run
  else:
    print(f'Result is {result}') # Result is 25.0
  finally:
    print('Execution complete!') # Execution complete!
  ```

- **例外物件**：讓你能存取例外本身，方便除錯和直接列印錯誤訊息。要存取例外物件，需要使用 `as` 關鍵字。範例如下：

  ```py
  try:
      value = int('This will raise an error')
  except ValueError as e:
      print(f'Caught an error: {e}')
      # Caught an error: invalid literal for int() with base 10: 'This will raise an error'
  ```

- **`raise` 陳述式**：允許你手動引發例外。你可以在特定條件成立時丟出例外。範例如下：

  ```py
  def divide(a, b):
      if b == 0:
          raise ZeroDivisionError('You cannot divide by zero')
      return a / b
  ```
  
## 例外訊號
  
- `raise` 陳述式在你建立自訂例外時也很有用，因為你可以用它丟出帶有自訂訊息的例外。以下是範例：

  ```py
  class InvalidCredentialsError(Exception):
      def __init__(self, message="Invalid username or password"):
          self.message = message
          super().__init__(self.message)

  def login(username, password):
      stored_username = "admin"
      stored_password = "password123"
      
      if username != stored_username or password != stored_password:
          raise InvalidCredentialsError()
      
      return f"Welcome, {username}!"
  ```  

以下示範如何搭配 `login` 函式使用 `InvalidCredentialsError` 例外：

  ```py
  # failed login attempt
  try:
      message = login("user", "wrongpassword")
  except InvalidCredentialsError as e:
      print(f"Login failed: {e}")
  else:
      print(message)

  # successful login attempt
  try:
      message = login("admin", "password123")
  except InvalidCredentialsError as e:
      # This block is not executed because the login was successful
      print(f"Login failed: {e}")
  else:
      # The else block runs if the 'try' block completes without an exception
      print(message)
  ```

`raise` 陳述式也可以搭配 `from` 關鍵字使用，串接例外，顯示不同錯誤間的關聯：

  ```py
  def parse_config(filename):
    try:
        with open(filename, 'r') as file:
            data = file.read()
            return int(data)
    except FileNotFoundError:
        raise ValueError('Configuration file is missing') from None
    except ValueError as e:
        raise ValueError('Invalid configuration format') from e

  config = parse_config('config.txt')
  ```

## Python 類別與物件

- **類別定義**：類別是用來創建物件的藍圖。它透過屬性和方法定義物件的行為。以下是 Python 中類別定義的基本範例：

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f'{self.name.upper()} says woof woof!')
```

- **創建物件**：物件是類別的實體。它們是透過呼叫類別並傳入必要的引數來創建的。

```python
dog1 = Dog('Jack', 3)
dog2 = Dog('Thatcher', 5)

dog1.bark()  # JACK says woof woof!
dog2.bark()  # THATCHER says woof woof!
```

- **呼叫物件的方法**：你可以對物件呼叫方法來執行動作或取得資訊。

```python
object_name1.method_name()
object_name2.method_name()
```

- **類別與物件的差異**：類別是可重複使用的模板，而物件是該類別的具體實體，擁有實際的資料。

## 屬性

- **實體屬性**：在 `__init__()` 中使用 `self` 定義，且每個物件獨有。
- **類別屬性**：直接在類別內定義，所有實體共享。

```python
class Dog:
    species = 'French Bulldog'  # Class attribute

    def __init__(self, name):
        self.name = name  # Instance attribute

print(Dog.species) # French Bulldog

jack = Dog('Jack')
print(jack.name)     # Jack
print(jack.species)  # French Bulldog
```

```python
class Car:

my_car_1 = Car('red', 'Tesla Model S')

```

- **存取方法**：使用點記法對物件呼叫方法。以下是對兩個不同車輛物件呼叫 `describe` 方法的範例：

```python
class Car:
    def __init__(self, color, model):
        self.color = color
```

## 雙底線（魔術）方法

- **定義**：特殊方法，名稱前後各有兩個底線（例如 `__init__`、`__len__`、`__str__`、`__eq__`）。Python 內部使用它們來執行內建操作。

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **間接呼叫雙底線方法**：你不需要直接呼叫雙底線方法。當特定操作發生時，Python 會自動呼叫它們。這些操作包括：

- **算術運算如加法、減法、乘法、除法等**。加法會呼叫 `__add__()`，減法呼叫 `__sub__()`，乘法呼叫 `__mul__()`，除法呼叫 `__truediv__()`。
        
- **字串運算如串接、重複、格式化和轉換為文字**。串接呼叫 `__add__()`，重複呼叫 `__mul__()`，格式化呼叫 `__format__()`，文字轉換呼叫 `__str__()` 和 `__repr__()`，等等。
        
- **比較運算如相等、小於、大於等**。相等檢查呼叫 `__eq__()`，小於呼叫 `__lt__()`，大於呼叫 `__gt__()`，等等。
        
- **迭代運算如使物件可迭代並取得下一項目**。呼叫 `__iter__()` 回傳迭代器，呼叫 `__next__()` 取得下一項目。

## 實際範例：購物車

- **帶有雙底線方法的購物車類別**：允許新增、移除、迭代和檢查內容，具備內建行為。

```python
class Cart:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

    def remove(self, item):
        if item in self.items:
            self.items.remove(item)
        else:
            print(f'{item} is not in cart')

    def list_items(self):
        return self.items

    def __len__(self):
        return len(self.items)

    def __getitem__(self, index):
        return self.items[index]

    def __contains__(self, item):
        return item in self.items

    def __iter__(self):
        return iter(self.items)

cart = Cart()
cart.add('Laptop')
print(len(cart))        # 1
print('Laptop' in cart) # True
```

## 什麼是物件導向程式設計？

- **物件導向程式設計**：一種程式設計風格，開發者將程式中的所有事物視為真實世界的物件。通常稱為 OOP。四個關鍵原則幫助你有效組織和管理程式碼，分別是 **封裝**、**繼承**、**多型** 和 **抽象性**。
- **類別**：用來創建物件的藍圖。每個由類別創建的物件都有定義資料的屬性和決定物件行為的方法。

## 什麼是封裝？

- **封裝**：將物件的屬性和方法打包成單一單元。它讓你將物件的內部狀態隱藏在一組簡單的公開方法和屬性後面，這些公開介面就像門一樣。門後是私有的屬性和方法，控制資料如何變更以及誰能看到它。
- **封裝範例**：如果你想追蹤錢包餘額，你會允許存款和提款，但不希望任何人直接竄改錢包餘額：

```py
class Wallet:
   def __init__(self, balance):
       self.__balance = balance # Private attribute

   def deposit(self, amount):
       if amount > 0:
           self.__balance += amount # Add to the balance safely

   def withdraw(self, amount):
       if 0 < amount <= self.__balance:
           self.__balance -= amount # Remove from the balance safely

account = Wallet(500)
print(account.__balance) # AttributeError: 'Wallet' object has no attribute '__balance'
```

- **單底線與雙底線前綴屬性的差異**：屬性和方法前加單底線表示僅供內部使用，這是慣例，並不會阻止外部存取。前加雙底線則有效防止外部存取該屬性和方法。

## 什麼是 Getter 和 Setter？

- **Getter 和 Setter**：讓你控制類別屬性如何被存取和修改的方法。使用 getter 取得值，使用 setter 設定值。
- **屬性（Properties）**：連結 getter 和 setter，允許存取資料。當你取得、設定或刪除值時，會在背後執行額外邏輯。
- **為什麼用屬性而非方法**：屬性比方法更易讀且程式碼更乾淨。它們讓你像存取一般屬性一樣使用點記法存取值，無需括號。
- **建立 Getter**：使用 `@property` 裝飾器建立 getter。以下是取得圓半徑的 getter 範例：

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self): # A getter to get the radius
        return self._radius
  
    @property
    def area(self):  # A getter to calculate area
        return 3.14 * (self._radius ** 2)

my_circle = Circle(3)

print(my_circle.radius) # 3
print(my_circle.area) # 28.26
```

在 `__init__()` 中使用 self.radius 確保在物件建立時會執行 setter，立即捕捉無效的半徑值。

- **建立 Setter**：建立設定半徑的 setter，需要定義同名方法並在上方使用 `@<property_name>.setter`：

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self):  # A getter to get the radius
        return self._radius

    @radius.setter
    def radius(self, value):  # A setter to set the radius
        if value <= 0:
            raise ValueError('Radius must be positive')
        self._radius = value

my_circle = Circle(3)
print('Initial radius:', my_circle.radius) # Initial radius: 3

my_circle.radius = 8
print('After modifying the radius:', my_circle.radius) # After modifying the radius: 8
```

- **Python 如何處理 Getter 和 Setter**：定義 getter 和 setter 後，當你使用一般屬性語法時，Python 會自動在背後呼叫它們：

```py
my_circle.radius # This will call the getter
my_circle.radius = 4 # This will call the setter
```

在 setter 內不應該直接指定給屬性名稱本身，否則會造成 `RecursionError`。

- **刪除器（Deleter）**：在設定和取得值後，你可以用 `deleter` 控制如何刪除屬性。當你對屬性使用 `del` 陳述時，deleter 會執行自訂邏輯。建立 deleter 使用 `@<property_name>.deleter` 裝飾器。

```py
  # Deleter
    @radius.deleter
    def radius(self):
        print("Deleting radius...")
        del self._radius
```

## 什麼是繼承？它如何促進程式碼重用？

- **繼承**：子類別使用父類別的屬性和方法的過程。繼承促進程式碼重用，提供清晰的階層結構，並能自訂行為而無需重寫所有程式碼。實作繼承時，子類別會帶入父類別名稱：

```py
class Parent:
    # Parent attributes and methods

class Child(Parent):
    # Child inherits, extends, and/or overrides where necessary
```

- **單繼承與多繼承**：子類別從單一父類別繼承屬性和方法稱為 **單繼承**，從多個父類別繼承則稱為 **多繼承**。語法如下：

```py
class Parent:
    # Attributes and methods for Parent

class Child:
    # Attributes and methods for Child

class GrandChild(Parent, Child):
    # GrandChild inherits from both Parent and Child
    # GrandChild can combine or override behavior from each
```

- **`super()` 函式**：讓你在類別有不同實作或擴充方法時，呼叫父類別的方法，避免重複程式碼。

## 什麼是多型？它如何促進程式碼重用？

- **多型**：物件導向原則，允許不同類別使用相同的方法名稱，但每個類別在呼叫時有不同的實作。語法如下：

```py
class A:
   def action(self): ...

class B:
   def action(self): ...

class C:
   def action(self): ...

Class().method()  # Works for A, B, or C
```

- **基於繼承的多型**：父類別定義方法，子類別各自調整以符合需求。

## 什麼是名稱改寫（Name Mangling）？它如何運作？

- **名稱改寫**：Python 內部將雙底線前綴的屬性重新命名，透過在前面加底線和類別名稱，將 `__attribute` 變成 `_ClassName__attribute`。
- **名稱改寫的目的**：主要用來防止在使用繼承時意外覆寫屬性和方法。以下程式碼讓你更容易理解：

```py
class Parent:
    def __init__(self):
        self.__data = 'Parent data'

class Child(Parent):
    def __init__(self):
        super().__init__()
        self.__data = 'Child data'

c = Child()
print(c.__dict__) # {'_Parent__data': 'Parent data', '_Child__data': 'Child data'}
```

## 什麼是抽象性？它如何幫助維持複雜系統的組織？

- **抽象性**：程式設計概念，將物件或系統的複雜實作細節隱藏，只顯示必要的特性。在 Python 和其他程式語言中，抽象性透過提高重用性來簡化複雜系統。
- **抽象性的範例**：日常生活中，汽車讓你只用方向盤、踏板和排檔桿，而不需知道引擎或煞車如何運作。
- **Python 如何實作抽象性**：Python 透過 `abc` 模組實作抽象性。該模組提供 `ABC` 類別（抽象基底類別）和 `@abstractmethod` 裝飾器。抽象基底類別定義子類別必須實作的共通方法和屬性，且無法被實體化。
- **抽象方法的定義**：使用 `@abstractmethod` 定義抽象方法，子類別必須覆寫它，即使有預設實作。抽象性的基本語法如下：

```py
from abc import ABC, abstractmethod

# Define an abstract base class
class AbstractClass(ABC):
    @abstractmethod
    def abstract_method(self):
        pass

# Concrete subclass that implements the abstract method
class ConcreteClassOne(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassOne')

# Another concrete subclass
class ConcreteClassTwo(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassTwo')
```

## 演算法與大 O 符號

- **演算法**：解決問題或執行任務的一組明確指令。演算法必須在有限步驟內完成，每一步都必須精確且明確。

- **大 O 符號**：描述演算法在輸入規模增加時的最壞情況效能或成長率。它關注資源使用如何隨輸入大小成長，忽略常數因子和低階項。

### 常見時間複雜度

- **O(1) - 常數時間**：演算法執行時間不受輸入大小影響。

```python
def check_even_or_odd(number):
    if number % 2 == 0:
        return 'Even'
    else:
        return 'Odd'
```

- **O(log n) - 對數時間**：隨輸入增長時間緩慢增加。常見於反覆將問題規模縮小的演算法（如二分搜尋）。

- **O(n) - 線性時間**：執行時間與輸入大小成正比。

```python
for grade in grades:
    print(grade)
```

- **O(n log n) - 線性對數時間**：高效排序演算法如合併排序和快速排序的常見時間複雜度。

- **O(n²) - 平方時間**：執行時間以平方速度增加。常見於巢狀迴圈。

```python
for i in range(n):
    for j in range(n):
        print("Hello, World!")
```

### 空間複雜度

- **O(1) - 常數空間**：演算法使用的記憶體不隨輸入大小改變。
- **O(n) - 線性空間**：記憶體使用與輸入大小成正比。
- **O(n²) - 平方空間**：記憶體使用以平方速度增加。

## 問題解決技巧

- **理解問題**：多次閱讀問題描述。辨識輸入、預期輸出，以及如何將輸入轉換為輸出。

- **偽程式碼**：用高階語言描述演算法邏輯，與程式語言無關。使用常見書寫語言混合程式結構如 `IF`、`ELSE`、`FOR`、`WHILE`。

```md
GET original_string
SET reversed_string = ""
FOR EACH character IN original_string:
  ADD character TO THE BEGINNING OF reversed_string
DISPLAY reversed_string
```

- **邊界案例**：在演算法應處理範圍邊界出現的特定有效輸入。務必考慮並測試邊界案例。

## 陣列

- **靜態陣列**：初始化時大小固定。元素存放於相鄰記憶體位置。執行期間大小不可變。

- **動態陣列**：執行期間可自動增減大小。透過自動複製到更大陣列來調整大小。

### Python 列表（動態陣列）

```python
numbers = [3, 4, 5, 6]

# Access elements
numbers[0]  # 3

# Update elements
numbers[2] = 16

# Add elements
numbers.append(7)
numbers.insert(3, 15)  # Insert at specific index

# Remove elements
numbers.pop(2)  # Remove at specific index
numbers.pop()   # Remove last element
```

### 動態陣列的時間複雜度

- **存取**：O(1)
- **尾端插入**：平均 O(1)，調整大小時 O(n)
- **中間插入**：O(n)
- **刪除**：中間 O(n)，尾端 O(1)

## 堆疊

- **堆疊**：後進先出（LIFO）資料結構。元素只能從頂端新增和移除。

- **推入操作**：將元素加入堆疊頂端。時間複雜度：O(1)。

- **彈出操作**：從堆疊頂端移除元素。時間複雜度：O(1)。

```python
# Using Python list as stack
stack = []

# Push operations
stack.append(1)
stack.append(2)
stack.append(3)

# Pop operations
top_element = stack.pop()  # Returns 3
```

## 佇列

- **佇列**：先進先出（FIFO）資料結構。元素從尾端加入，從前端移除。

- **入列操作**：將元素加入佇列尾端。時間複雜度：O(1)。

- **出列操作**：從佇列前端移除元素。時間複雜度：O(1)。

```python
from collections import deque

# Using deque for efficient queue operations
queue = deque()

# Enqueue operations
queue.append(1)
queue.append(2)
queue.append(3)

# Dequeue operations
first_element = queue.popleft()  # Returns 1
```

## 鏈結串列

- **鏈結串列**：線性資料結構，每個節點包含資料和指向下一節點的參考。節點像串鏈般連接。

### 單向鏈結串列

- **結構**：每個節點有資料和一個指向下一節點的參考。
- **巡訪**：只能從頭到尾向前移動。
- **頭節點**：串列的第一個節點，通常是唯一可直接存取的節點。
- **尾節點**：串列的最後一個節點，指向 `None`。

### 操作與時間複雜度

- **開頭插入**：O(1)
- **尾端插入**：O(n) - 需巡訪至尾端
- **中間插入**：O(n) - 需巡訪至指定位置
- **開頭刪除**：O(1)
- **尾端刪除**：O(n) - 需巡訪尋找前一節點
- **中間刪除**：O(n) - 需巡訪尋找節點

### 雙向鏈結串列

- **結構**：每個節點有資料和兩個參考：下一節點與前一節點。
- **巡訪**：可雙向移動。
- **記憶體**：因多一個參考，需比單向鏈結串列更多記憶體。

## 雜湊映射與集合

### 映射與雜湊映射

- **映射（抽象資料型態）**：管理鍵值對集合。每個鍵必須唯一，但值可重複。

- **雜湊映射**：映射 ADT 的具體實作，使用雜湊技術。透過雜湊函式產生鍵的雜湊值，決定在底層陣列的存放位置。

### Python 字典（雜湊映射）

```python
# Creating dictionaries
my_dictionary = {
    "A": 1,
    "B": 2, 
    "C": 3
}

# Alternative creation
my_dictionary = dict(A=1, B=2, C=3)

# Access and modify
value = my_dictionary["A"]  # 1
my_dictionary["A"] = 4      # Update value
del my_dictionary["A"]      # Remove key-value pair

# Check membership
"C" in my_dictionary

# Get keys, values, items
my_dictionary.keys()
my_dictionary.values()
my_dictionary.items()
```

### 雜湊映射的時間複雜度

- **平均情況**：插入、取得、刪除皆為 O(1)
- **最壞情況**：大量雜湊碰撞時為 O(n)

### 集合

- **集合**：無序且元素唯一的集合。不允許重複，且不維持特定順序。

- **僅限不可變元素**：集合只能包含不可變資料型態（數字、字串、元組），因為雜湊值必須保持不變。

```python
# Creating sets
numbers = {1, 2, 3, 4}
empty_set = set()  # Must use set(), not {}

# Add and remove elements
numbers.add(5)
numbers.remove(4)      # Raises KeyError if not found
numbers.discard(4)     # No error if not found

# Set operations
set_a = {1, 2, 3, 4}
set_b = {2, 3, 4, 5, 6}

# Union, intersection, difference, symmetric difference
set_a.union(set_b)                    # or set_a | set_b
set_a.intersection(set_b)             # or set_a & set_b
set_a.difference(set_b)               # or set_a - set_b
set_a.symmetric_difference(set_b)     # or set_a ^ set_b

# Subset and superset checks
set_a.issubset(set_b)
set_a.issuperset(set_b)
set_a.isdisjoint(set_b)

# Membership testing
5 in numbers
```

### 集合的時間複雜度

- **平均情況**：新增、移除、成員測試皆為 O(1)
- **最壞情況**：因雜湊碰撞為 O(n)

## 雜湊碰撞

- **雜湊碰撞**：兩個不同鍵產生相同雜湊值。

- **碰撞解決策略**：
  - **串鏈法**：每個陣列索引指向一個鏈結串列，存放所有相同雜湊值的元素。
  - **開放定址法**：使用預定序列尋找下一個可用索引。

## 何時使用各種資料結構

- **列表**：需要有序且可索引存取，且大小不確定時。
- **堆疊**：用於 LIFO 操作（復原功能、表達式求值、回溯）。
- **佇列**：用於 FIFO 操作（任務排程、廣度優先搜尋）。
- **鏈結串列**：頻繁在開頭插入/刪除、大小不確定且不需隨機存取時。
- **雜湊映射**：快速鍵值查找、計數、快取。
- **集合**：檢查唯一性、數學集合運算、移除重複。

## 搜尋演算法

搜尋演算法讓你在特定的項目列表中搜尋標的。

在計算機科學中，有兩種搜尋演算法你應該了解。它們是**線性搜尋**和**二分搜尋法**演算法。了解這兩種演算法的差異以及何時使用各自非常重要。

### 線性搜尋

- 線性搜尋會從列表的開頭開始，逐一迭代每個項目，直到找到標的項目為止。
- 如果找到標的項目，則傳回該項目在列表中的索引。
- 如果找不到標的，則傳回 `-1`，這在大多數程式語言中表示**無效索引**。
- 因為線性搜尋會檢查每一個項目直到找到標的，所以對於大量項目的列表來說並不高效。
- 線性搜尋的時間複雜度是 `O(n)`，因為搜尋整個列表所需的時間會隨著列表大小線性增加。
- 線性搜尋的空間複雜度是 `O(1)`，因為它不需要任何額外空間來搜尋整個列表。

### 二元搜尋

- 二分搜尋透過將列表對半分割，檢查目標是否在中間。
- 二分搜尋的前提是列表必須是遞增排序。
- 二分搜尋對大型列表更有效率，因為它將列表分半，忽略不包含目標的半邊。
- 找到目標時，回傳目標在列表中的索引。
- 若未找到，演算法會判斷目標在左半邊還是右半邊。
- 持續將剩餘列表分半，直到找到目標。
- 若最終找不到目標，回傳 `-1`。
- 二分搜尋的時間複雜度是 `O(log n)`，因為搜尋時間隨列表大小以對數方式增加。
- 二分搜尋的空間複雜度是 `O(1)`，因為不需額外空間。

### 線性搜尋與二元搜尋的差異

- 與線性搜尋相比，二分搜尋法更適合用於大型列表的項目。
- 線性搜尋的時間複雜度是 `O(n)`，因為搜尋整個列表所需的時間會隨著列表大小線性增加。
- 二分搜尋法的時間複雜度是 `O(log n)`，因為搜尋列表所需的時間會隨著列表大小以對數方式成長。

## 排序演算法與分治法

在計算機科學中，分而治之是一種用來將問題拆解成較小子問題的技術，使其更容易解決。遞迴是分而治之中常用的技術，而分而治之是一種強大的策略，用於實作許多高效的排序演算法，如合併排序。

### 合併排序

- 合併排序是一種遵循分而治之方法的排序演算法。
- 它的運作方式是透過遞迴地將一個列表分割成更小的子列表，直到每個子列表只包含一個元素。
- 然後它會反覆將子列表以排序的順序合併回去。
- 合併排序的時間複雜度是 `O(n log n)`，因為列表會持續被對半切分 `(log n)`，然後再合併在一起 `(O(n))`。
- 合併排序的空間複雜度是 `O(n)`，因為它不是就地排序演算法。

## 圖形概述

圖形是一組由邊（連接）連結的節點（頂點）。每個節點可連接多個其他節點，形成網路。圖形類型包括：

- 有向圖：邊有方向（從一節點到另一節點），通常以直線和箭頭表示。
- 無向圖：邊無方向，以簡單線條表示。
- 頂點：每個節點有標籤或識別符。
- 有環圖：包含環路（起點和終點相同的路徑）。
- 無環圖（DAG）：不包含環路。
- 邊標籤圖：每條邊有標籤，通常畫在對應邊旁。
- 加權圖：邊有權重（數值），可用於算術運算。
- 不連通圖：包含兩個或以上未被邊連接的節點。

圖形用於地圖、網路、推薦系統、相依性解析等多種應用。

## 圖形巡訪

涉及拜訪圖中所有節點。兩種主要演算法：

- **廣度優先搜尋（BFS）**
  - 使用佇列。
  - 按層級探索。
  - 找出無權重圖的最短路徑。

- **深度優先搜尋（DFS）**
  - 使用堆疊（或遞迴）。
  - 完全探索一條分支後回溯。
  - 用於環路偵測和路徑尋找。

## 圖形表示法

圖形主要有兩種表示方式：

- **鄰接列表**
  - 每個節點有鄰居列表。
  - 對稀疏圖空間效率高。
  - 易於巡訪鄰居。

- **鄰接矩陣**
  - 二維陣列，行列代表節點。
  - 對大型圖空間需求大。
  - 快速檢查兩節點間是否有邊。

## 樹

樹是特殊的圖，無環且連通。主要特性：

- 無迴圈或環路（起點和終點相同的路徑）。
- 必須連通（每個節點可從其他節點抵達）。

### 常見樹型

最常見的樹型有：

- 二元樹
  - 每個節點最多有兩個子節點，分別為左子節點和右子節點。

- 二元搜尋樹（BST）
  - 一種二元樹，左子節點值小於父節點，右子節點值大於父節點。


## 字首樹（Trie）

又稱前綴樹，用於儲存字串集合，每個節點代表一個字元。

共用的字首只儲存一次，對自動完成和拼字檢查等任務效率高。

搜尋和插入操作的時間複雜度為 O(L)，L 為字串長度。

## 優先佇列

優先佇列是一種抽象資料型態，每個元素有優先權。

佇列和堆疊只考慮插入順序，優先佇列則依元素優先權服務。

標準佇列遵循 FIFO（先進先出），堆疊遵循 LIFO（後進先出）。優先佇列中，優先權高的元素先被處理，無視插入順序。

## 堆積（Heap）

特殊的樹狀資料結構，具備堆積性質。

堆積性質決定父節點與子節點的關係。有兩種堆積：

- 最大堆積（Max-Heap）
  - 每個父節點的值大於或等於子節點值。
  - 最大元素位於根節點。

- 最小堆積（Min-Heap）
  - 每個父節點的值小於或等於子節點值。
  - 最小元素位於根節點。

### Python `heapq` 模組範例

```py
import heapq

# Create empty heap
my_heap = []

# Insert elements
heapq.heappush(my_heap, 9)
heapq.heappush(my_heap, 3)
heapq.heappush(my_heap, 5)
print(my_heap) # [3, 9, 5]

# Remove smallest element
print(heapq.heappop(my_heap))  # 3
print(my_heap) # [5, 9]

# Push + Pop in one step
print(heapq.heappushpop(my_heap, 7)) # 5
print(my_heap) # [7, 9]

# Transform list into heap
nums = [5, 7, 3, 1]
heapq.heapify(nums)
```

### 使用優先權

```py
my_heap = []
heapq.heappush(my_heap, (3, "A"))
heapq.heappush(my_heap, (2, "B"))
heapq.heappush(my_heap, (1, "C"))

# Removes lowest number = highest priority
print(heapq.heappop(my_heap))  # (1, "C")
```

## 動態規劃簡介

- **定義**：動態規劃是一種演算法技巧，透過將複雜問題拆解成較簡單子問題並儲存結果，避免重複計算。
- **重疊子問題**：解決較大問題時，會多次遇到相同的較小問題。透過儲存解答避免重複計算。
- **最優子結構**：問題的最優解包含其子問題的最優解。表示可透過組合子問題的最佳解來構建整體最佳解。

## 動態規劃解法

- **備忘錄法（自頂向下）**：備忘錄法儲存昂貴函式呼叫的結果，當相同輸入再次出現時回傳快取結果。

```py
def climb_stairs_memo(n, memo={}):
    """Dynamic programming with memoization"""
    # Check if we've already calculated this value
    if n in memo:
        return memo[n]  # Return cached result - O(1) lookup!
    
    # Base cases
    if n <= 2:
        return n
    
    # Calculate once and store in memo for future use
    memo[n] = climb_stairs_memo(n-1, memo) + climb_stairs_memo(n-2, memo)
    return memo[n]
```

- **表格法（自底向上）**：表格法從底層開始建立解答，填滿子問題解答的表格。

```py
def climb_stairs_tabulation(n):
    """Dynamic programming with tabulation"""
    if n <= 2:
        return n
    
    # Create array to store results for all steps from 0 to n
    dp = [0] * (n + 1)
    dp[1] = 1  # 1 way to reach step 1
    dp[2] = 2  # 2 ways to reach step 2
    
    # Build up the solution iteratively
    for i in range(3, n + 1):
        # Ways to reach step i = ways to reach (i-1) + ways to reach (i-2)
        dp[i] = dp[i-1] + dp[i-2]
    
    return dp[n]
```

## 使用動態規劃的實際應用

- **路徑最佳化**：GPS 系統使用動態規劃演算法來尋找地點之間的最短路徑。
- **文字處理**：拼字檢查和自動完成功能經常依賴動態規劃來計算單字之間的編輯距離。
- **財務建模**：投資策略和投資組合最佳化常常運用動態規劃技術。
- **資源分配**：背包問題及其變體出現在排程、預算和資源管理中。

## 何時使用動態規劃

你應該在以下情況考慮使用動態規劃：

- 問題可以拆解成重疊的子問題。
- 問題具有最佳子結構。
- 簡單的遞迴解法會涉及重複計算。
- 你需要以空間複雜度為代價來優化時間複雜度。
