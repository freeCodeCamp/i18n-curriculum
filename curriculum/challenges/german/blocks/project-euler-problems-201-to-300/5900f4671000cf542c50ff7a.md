# --description--

Ein Tripel aus positiven ganzen Zahlen ($a$,$b$,$c$) wird Cardano-Triplet genannt, wenn es die Bedingung erfüllt:

$$\sqrt[3]{a + b \sqrt{c}} + \sqrt[3]{a - b \sqrt{c}} = 1$$

Zum Beispiel ist (2,1,5) ein Cardano-Triplet.

Es gibt 149 Cardano-Triplets, für die $a + b + c ≤ 1000$ gilt.

Finden Sie heraus, wie viele Cardano-Triplets existieren, sodass $a + b + c ≤ 110\\,000\\,000$ gilt.
