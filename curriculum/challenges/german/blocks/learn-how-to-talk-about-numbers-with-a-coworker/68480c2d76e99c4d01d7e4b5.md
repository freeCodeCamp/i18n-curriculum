# --explanation--

`User experience` (oder `UX`) beschreibt, wie sich ein Benutzer bei der Nutzung eines Produkts fühlt – ob es einfach, angenehm und reibungslos ist. Zum Beispiel:

`Improving page load time boosts user experience.` — Schnellere Ladezeiten der Seiten sorgen dafür, dass Nutzer die App mehr genießen.

`Enhancements` sind Änderungen, die vorgenommen werden, um ein Produkt oder einen Prozess zu verbessern. Zum Beispiel:

`The latest update includes performance enhancements.` — Die neue Version lässt die App besser und schneller funktionieren.

In Alices Satz wird die Hälfte des Design-Budgets dafür ausgegeben, das Produkt einfacher und besser nutzbar zu machen.
