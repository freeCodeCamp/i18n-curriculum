# --interactive--

Працюючи з JavaScript, ви часто зустрічатимете складні структури даних, які містять вкладені об’єкти та масиви всередині об’єктів. Ці структури можуть представляти багаті ієрархічні дані, але для роботи з ними потрібно чітко розуміти, як отримувати доступ і маніпулювати даними всередині них. Давайте розглянемо, як ефективно орієнтуватися у цих вкладених структурах.

Доступ до властивостей вкладених об’єктів здійснюється за допомогою точкової або дужкової нотації, так само як і для простих об’єктів. Однак вам потрібно буде ланцюжком викликати ці доступи, щоб заглибитися у вкладену структуру.

Наприклад, розглянемо вкладений об’єкт, що представляє людину з контактною інформацією:

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};
```

Щоб отримати робочий номер телефону `Alice`, ви ланцюжком викликаєте доступ до властивостей так:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person.contact.phone.work); // "098-765-4321"
```

:::

Також можна використовувати дужкову нотацію, що особливо корисно, коли імена властивостей містять пробіли або спеціальні символи, або коли ви використовуєте змінні для доступу до властивостей:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person['contact']['phone']['work']); // "098-765-4321"
```

:::

Тепер подивімося, як отримати дані, коли одна з властивостей об’єкта має значення масиву. Ось змінений об’єкт `person`, який містить масив адрес:

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};
```

Ось приклад, як отримати місто робочої адреси `Alice`:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};

console.log(person.addresses[1].city); // "Workville"
```

:::

У цьому прикладі `person.addresses` посилається на масив адрес. Щоб отримати другу адресу в цьому масиві, ми використовуємо дужкову нотацію з індексом `1`. Потім за допомогою точкової нотації отримуємо `city` з цього об’єкта адреси.

Розуміння того, як отримувати доступ до властивостей у вкладених об’єктах і масивах, є необхідним при роботі зі складними структурами даних. У майбутніх практичних заняттях і лабораторних роботах ви матимете можливість попрактикуватися з такими структурами даних.
