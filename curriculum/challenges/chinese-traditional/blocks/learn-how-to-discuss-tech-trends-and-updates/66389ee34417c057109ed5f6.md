# --description--

`automate` 意味著讓一個處理程序自行運作，而不需要人員協助。在程式設計中，它可能涉及使用軟體來撰寫程式碼、測試應用程式，或尋找並修正錯誤。 例如，早上自動開啟並煮咖啡的咖啡機就是 `automated`。

字 `certain` 用來談論特定的事物或人，但不是所有的事物或人。例如，如果你說 `certain books are on sale`，表示只有部分書籍在特價，而不是全部。
