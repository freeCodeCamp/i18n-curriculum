# --explanation--

`Scheduled` は、何かが特定の時間に予定または手配されていることを意味します。例えば：

`The scheduled meeting will start at 2 PM.` - これは、会議が指定された時間に始まるようにあらかじめ計画されていることを意味します。

`Maintenance` は、物事を正常に機能させたり、問題を修正したりするために行われる活動を指します。例えば：

`The team performed maintenance on the server to ensure it runs smoothly.` - これは、チームがサーバーを良好な状態に保つために修正や更新を行ったことを意味します。
