# --description--

`Data` 是指信息，特别是为审查和考虑而收集的事实或数字，并用于帮助决策。 

有不同类型的 数据，比如 `quantitative data`，它关于数字（比如有多少人访问一个 store），还有 `qualitative data`，它关于描述（比如人们对一个产品的看法）。 

`Customer data` 是关于客户的特定信息，例如他们的购买习惯或偏好。 

例如，一个保存可能会收集关于客户最常访问的时间段或最受欢迎产品的 `data`。
