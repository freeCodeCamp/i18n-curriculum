# --description--

`update` 是关于某事的新信息或最新消息。当你 `update` 某人时，你是在向他们提供最新的信息。

短语 `updates on project progress` 意味着提供有关当前项目的新信息。 

介词 `on` 在此上下文中将更新与它们所针对的具体主题——项目进展——关联起来。

当谈论通信主题时，使用 `on`，例如 `presentation on science` 或 `update on work`。它显示了通信与主题之间的直接关系。
