# --description--

在專案管理中，使用特定短語來描述任務如何被控制代碼。 

`break down` 意指將某物劃分為較小且較易管理的零件。 例如，`When I work on a large project, I often break down the work into smaller sections`。

`set deadlines` 是指決定任務需要完成的時間。 例如，`Setting deadlines for each task ensures that the project stays on schedule`。

理解這些片語有助於掌握專案如何組織。
