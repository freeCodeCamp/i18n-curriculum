# --explanation--

`for loop` корисний, коли ви знаєте, скільки разів цикл має виконатися. Наприклад:

- `A for loop is useful for going through a list.` — ви можете використовувати його, щоб перевірити кожен елемент у списку.

- `The for loop stops when it reaches the last item.` — цикл завершується, коли елементів більше немає.

`while loop` краще, коли кількість повторень залежить від умови. Наприклад:

- `A while loop runs until a condition is false.` — цикл триває, доки правило є істинним.

- `Be careful, a while loop can run forever!` — якщо його не зупинити, цикл не завершиться.
