# --explanation--

`Automation` — це використання технологій для автоматичного виконання завдань без участі людини. Наприклад:

`The company uses automation to schedule social media posts every day.` — це означає використання програмних інструментів для планування публікацій без ручного введення щоразу.

`Data-driven` означає прийняття рішень на основі аналізу даних. Наприклад:

`The team made a data-driven decision to improve website performance based on user activity data.` — це вказує, що рішення було прийняте на основі реальних даних про поведінку користувачів, а не припущень чи здогадок.
