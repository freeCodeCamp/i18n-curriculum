# --description--

最初のオブジェクトチャレンジでは、変数の評価を使ってプロパティの値にアクセスする方法としてブラケット記法の使用を紹介しました。例えば、`foods` オブジェクトがスーパーマーケットのレジ用プログラムで使われていると想像してください。`selectedFood` を設定する関数があり、その食品が`foods` オブジェクトに存在するかどうかを確認したい場合、次のようになります。

```js
let selectedFood = getCurrentFood(scannedItem);
let inventory = foods[selectedFood];
```

このコードは`selectedFood` 変数に格納された値を評価し、`foods` オブジェクトのそのキーの値を返します。もし存在しなければ`undefined` を返します。ブラケット記法は、オブジェクトのプロパティが実行時までわからなかったり、より動的にアクセスする必要がある場合に非常に便利です。

# --instructions--

`checkInventory` という関数を定義しました。この関数はスキャンされたアイテムを引数として受け取ります。`scannedItem` オブジェクトの`foods` キーの現在の値を返してください。`checkInventory` に渡される引数は有効なキーのみであると仮定してかまいません。

# --hints--

`checkInventory` は関数であるべきです。

```js
assert.strictEqual(typeof checkInventory, 'function');
```

`foods` オブジェクトは次のキーと値のペアのみを持つべきです：`apples: 25`、`oranges: 32`、`plums: 28`、`bananas: 13`、`grapes: 35`、`strawberries: 27`。

```js
assert.deepEqual(foods, {
  apples: 25,
  oranges: 32,
  plums: 28,
  bananas: 13,
  grapes: 35,
  strawberries: 27
});
```

`checkInventory("apples")` は`25` を返すべきです。

```js
assert.strictEqual(checkInventory('apples'), 25);
```

`checkInventory("bananas")` は`13` を返すべきです。

```js
assert.strictEqual(checkInventory('bananas'), 13);
```

`checkInventory("strawberries")` は`27` を返すべきです。

```js
assert.strictEqual(checkInventory('strawberries'), 27);
```
