# --explanation--

`It` 用于替代已经提到的名词，帮助避免句子中的重复。例如：

- 我买了一台新建的笔记本电脑。它运行得非常好。— 这里，`it` 代替了 `laptop` 以避免重复使用该词。

在 `The Live Preview isn't showing in Visual Studio Code. Isn't it supposed to support that feature?` 中，`it` 指的是 `Visual Studio Code`，因为 Tom 正在谈论该程序及其支持某个特色的能力。 

`It` 替代 `Visual Studio Code` 以避免再次提及该程序的名称。 

通过确定预期支持该特色的内容，你可以判断 `it` 指的是程序本身。
