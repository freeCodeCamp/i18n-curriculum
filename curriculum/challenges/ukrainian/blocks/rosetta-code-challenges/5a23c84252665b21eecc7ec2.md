# --description--

Відстань Джаро — це міра схожості між двома рядками. Чим вища відстань Джаро для двох рядків, тим вони більш схожі. Оцінка нормалізована так, що `0` означає відсутність схожості, а `1` — точну відповідність.

**Визначення**

Відстань Джаро \\( d_j \\) для двох заданих рядків \\(s_1\\) та \\(s_2\\) визначається як

\\begin{align}d_j = \\begin{cases}0& & \\text{якщо }m=0 \\\\\\\\{\\frac {1}{3}}\\left({\\frac {m}{|s\_{1}|}}+{\\frac {m}{|s\_{2}|}}+{\\frac {m-t}{m}}\\right)& & \\text{в іншому випадку}\\end{cases}\\end{align}

Де:

<ul>
  <li>\(m\) — кількість <i>співпадаючих символів</i>;</li>
  <li> \(t\) — половина кількості <i>транспозицій</i>.</li>
</ul>

Два символи з \\(s_1\\) та \\(s_2\\) відповідно вважаються *співпадаючими* лише якщо вони однакові і знаходяться не далі ніж \\(\\left\\lfloor\\frac{\\max(|s_1|,|s_2|)}{2}\\right\\rfloor-1\\).

Кожен символ \\(s_1\\) порівнюється з усіма його співпадаючими символами в \\(s_2\\). Кількість співпадаючих символів, які мають різний порядок у послідовності, поділена на 2, визначає кількість *транспозицій*.

**Приклад**

Для рядків \\(s_1\\) *DWAYNE* та \\(s_2\\) *DUANE* маємо:

<ul>
  <li>\(m = 4\)</li>
  <li>\(|s_1| = 6\)</li>
  <li>\(|s_2| = 5\)</li>
  <li>\(t = 0\)</li>
</ul>

Отримуємо оцінку Джаро: \\(d_j = \\frac{1}{3}\\left(\\frac{4}{6} + \\frac{4}{5} + \\frac{4-0}{4}\\right) = 0.822\\).

```js
function jaro(s, t) {
  var s_len = s.length;
  var t_len = t.length;

  if (s_len == 0 && t_len == 0) return 1;

  var match_distance = Math.max(s_len, t_len) / 2 - 1;

  var s_matches = new Array(s_len);
  var t_matches = new Array(t_len);

  var matches = 0;
  var transpositions = 0;

  for (var i = 0; i < s_len; i++) {
    var start = Math.max(0, i - match_distance);
    var end = Math.min(i + match_distance + 1, t_len);

    for (var j = start; j < end; j++) {
      if (t_matches[j]) continue;
      if (s.charAt(i) != t.charAt(j)) continue;
      s_matches[i] = true;
      t_matches[j] = true;
      matches++;
      break;
    }
  }

  if (matches == 0) return 0;

  var k = 0;
  for (var i = 0; i < s_len; i++) {
    if (!s_matches[i]) continue;
    while (!t_matches[k]) k++;
    if (s.charAt(i) != t.charAt(k)) transpositions++;
    k++;
  }

  return ((matches / s_len) +
    (matches / t_len) +
    ((matches - transpositions / 2.0) / matches)) / 3.0;
}
```
