# --explanation--

كلمة `some` تعني `a part of` أو `not all`. على سبيل المثال:

- `Some of the team members are working remotely.` - ليس كل أعضاء الفريق، بل جزء منهم فقط.

- `She bought some new books to prepare for the exam.` - بعض الكتب، وليس كل الكتب المتاحة.

استخدام `some` يوضح أنه رغم أهمية TechCorp للشركات الصغيرة، فهي لا تمول كل مشاريعها، بل جزءًا منها فقط. ولهذا السبب فإن فقدان TechCorp سيظل له تأثير كبير.
