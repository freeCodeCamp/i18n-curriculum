# --interactive--

Katika JavaScript, njia ya `indexOf()` ni muhimu kwa kupata kielezo cha kwanza cha kipengele maalum ndani ya safu ya data. Ikiwa kipengele hakipatikani, basi itarudisha `-1`. Hii ni sintaksia ya msingi:

```js
array.indexOf(element, fromIndex)
```

`element` inawakilisha thamani unayotaka kutafuta ndani ya safu ya data, na kigezo cha `fromIndex` ni nafasi ambayo utaanza kutafuta. Kigezo cha `fromIndex` ni hiari. Ikiwa `fromIndex` haitatolewa, utaftaji unaanza kutoka mwanzo wa safu ya data. Tuchukulie mfano:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "banana"];
let index = fruits.indexOf("banana");
console.log(index); // 1
```

:::

Katika mfano huu, tuna safu ya data `fruits` yenye majina mbalimbali ya matunda. Tunatumia njia ya `indexOf()` kupata kielezo cha mfuatano wa herufi `banana` ndani ya safu ya data `fruits`. Kwa kuwa `banana` ipo katika kielezo cha `1`, njia hiyo inarudisha `1`, ambayo huhifadhiwa katika tofauti ya `index` na kuandikwa kwenye konsoli.

Ikiwa kipengele unachotafuta hakipatikani katika safu ya data, `indexOf()` inarudisha `-1`. Kwa mfano:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let index = fruits.indexOf("grape");
console.log(index); // -1
```

:::

Hapa, tunatafuta mfuatano wa herufi `grape` katika safu ya matunda kwa kutumia `indexOf()`. Kwa kuwa `grape` haipo katika safu ya data, njia hiyo inarudisha `-1`, ambayo huhifadhiwa katika tofauti ya `index` na kuandikwa kwenye konsoli.

Ikiwa unataka kuanza kutafuta kipengele baada ya nambari maalum ya kielezo, basi unaweza kupitisha hoja ya pili kama katika mfano huu:

:::interactive_editor

```js
let colors = ["red", "green", "blue", "yellow", "green"];
let index = colors.indexOf("green", 3);
console.log(index); // 4
```

:::

Katika mfano huu, utaftaji hauanza mwanzo wa safu ya data, badala yake unaanza kutoka kielezo nambari `3` ambacho ni `yellow` na hupata matokeo ya `4`.
