# --interactive--

L'elemento `div` viene usato come contenitore per raggruppare altri elementi.

Ecco un esempio di elemento `div`. Aggiungi un altro elemento paragrafo all'interno dell'elemento `div` e osserva le modifiche nella finestra di anteprima. Per vedere le anteprime, devi abilitare l'editor interattivo.

:::interactive_editor

```html
<div>
  <p>Example paragraph element.</p>
</div>
```

:::

Userai principalmente l'elemento `div` quando vuoi raggruppare elementi HTML che condivideranno un insieme di stili CSS. Imparerai di più su CSS nelle lezioni e workshop futuri.

Anche se l'elemento `div` è comunemente usato nei codici reali, devi fare attenzione a non abusarne. Ci sono momenti in cui un altro elemento sarebbe più appropriato.

Per esempio, se vuoi dividere il tuo contenuto in sezioni, allora l'elemento `section` sarebbe più adatto di un elemento `div`.

Aggiungi un altro elemento `section` sotto il primo. Poi, all'interno dell'elemento `section`, aggiungi un elemento `h2` e un elemento `p`. Puoi usare qualsiasi testo ti piaccia e vedrai le modifiche nella finestra di anteprima. Per interagire con l'esempio, devi abilitare l'editor interattivo.

:::interactive_editor

```html
<section>
  <h2>Mammals</h2>
  <p>
    Mammals are warm-blooded animals with fur or hair. Most give birth to live
    young.
  </p>
  <ul>
    <li>Lion</li>
    <li>Elephant</li>
    <li>Dolphin</li>
  </ul>
</section>
```

:::

L'elemento `section` ha un significato semantico rispetto all'elemento `div` che non è semantico. La semantica è il significato delle parole o delle frasi in una lingua. In HTML, che è una lingua, anche gli elementi hanno il loro significato semantico. Quindi, questo significa che se usi un elemento `section`, il browser ne capirà il significato semantico e saprà trattarlo come una sezione — su desktop, dispositivi mobili, ovunque.
