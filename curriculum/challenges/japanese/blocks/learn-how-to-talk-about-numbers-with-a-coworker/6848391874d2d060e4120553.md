# --explanation--

`Initial setup` はプロジェクトの最初のステップ、つまり実際の作業が始まる前にシステムやツール、アクセスの準備をする段階を意味します。例えば：

`The initial setup of the system took one day.` — システムのインストールの最初の部分に1日かかりました。

`Early development` は構築プロセスの初期段階を指し、多くの場合プロトタイプや基本機能を含みます。例えば：

`In early development, we focused on design.` — 最初は主に見た目の部分に取り組みました。
