# --explanation--

`Review` は、何かを注意深くチェックまたは調べることを意味します。例えば：

`Let's review the schedule before the meeting.` – これは、それが正しいかどうかを確認するために見直すことを意味します。

`Budget` は、タスクや期間のために計画された金額を指します。例えば：

`We set a budget for travel expenses.` – これは、チームがその目的のために資金を割り当てたことを意味します。

`Resources` は、仕事をするために必要な人々、道具、または材料です。例えば：

`The team lacks the resources to finish on time.` – これは、十分な支援や材料がないことを意味します。
