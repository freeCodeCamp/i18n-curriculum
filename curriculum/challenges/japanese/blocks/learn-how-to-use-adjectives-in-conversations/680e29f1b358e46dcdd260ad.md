# --explanation--

`Send it over to` は、ファイル、書類、またはメッセージを他の人に届けることを意味し、しばしばメールやオンラインで行われます。例えば：

`I'll send it over to the design team after making the changes.` – これは書類を相手に転送することを意味します。

`By tomorrow` は、翌日までにまたはその日の終わりまでに行動を完了することを意味します。例えば：

`We need to finalize the software update by tomorrow.` – これは明確な締め切りを示しています。
