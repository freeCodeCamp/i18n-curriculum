# --description--

`block`와 `inline` 요소에 대해 이야기하려면 `div`와 `span`도 함께 알아야 합니다. 지금까지 접한 다른 모든 HTML 요소는 그 내용에 의미를 부여합니다. 예를 들어, 문단 요소는 브라우저에 포함된 텍스트를 문단으로 표시하라고 알려줍니다. Strong 요소는 어떤 텍스트가 중요한지 브라우저에 알려주고요. 하지만 `div`와 `span`는 그 내용에 특별한 의미를 부여하지 않습니다. 단지 무엇이든 담을 수 있는 일반적인 상자일 뿐입니다.

# --questions--    

## --text--

`span`와 `div` 요소가 다른 요소와 가장 큰 차이점은 무엇인가요?

## --answers--

고유한 의미를 지닌 특정 HTML 요소와 달리, `span`와 `div` 요소는 다용도로 사용되며 주로 스타일링과 레이아웃 목적으로 사용됩니다.

---

`span`와 `div` 요소는 문단이나 제목 같은 다른 HTML 요소와 달리 포함된 내용에 특정 의미론적 의미를 부여하지 않습니다.

---

`span`와 `div` 요소는 개발자가 특정 의미론적 중요성 없이 CSS 스타일을 적용하고 콘텐츠를 구조화할 수 있게 하는 일반적인 컨테이너입니다.
