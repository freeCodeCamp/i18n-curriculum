# --interactive--

有时你需要在你的 JavaScript 文件中操作超文本标记语言元素的样式。一个例子是当你需要在用户点击按钮时隐藏或显示一个菜单。

在本课中，你将学习如何使用 `Element.style` 和 `Element.classList` 属性操作样式。

`Element.style` 属性是一个只读属性，表现元素的内联样式。你可以使用此属性来获取或设置元素的样式。

这是一个 ID 为 `para` 的段落元素示例：

```html
<p id="para">This is a paragraph</p>
```

你可以使用 `style` 属性将段落元素的颜色更改为红色：

:::interactive_editor

```html
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```js
const paraEl = document.getElementById("para");
paraEl.style.color = "red";
```

:::

`style` 属性可用于设置许多 CSS 属性，例如 `color`、`background-color`、`font-size`、`font-weight` 等。

操作样式的另一种方法是使用 `Element.classList` 属性。`classList` 属性是一个只读属性，可用于为元素添加、移除或切换类。让我们来看几个示例。

在此示例中，你将为 ID 为 `para` 的段落元素添加一个名为 `highlight` 的类。在 CSS 中，我们将设置 `highlight` 类以将段落元素的背景颜色更改为黄色。在 JavaScript 中，我们将使用 `classList.add()` 方法为段落元素添加 `highlight` 类。 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```css
.highlight {
  background-color: yellow;
}
```

```js
const paraEl = document.getElementById("para");
paraEl.classList.add("highlight");
```

:::

你也可以通过将多个类作为参数传递给 `classList.add()` 方法，为一个元素添加多个类：

```js
classList.add("class1", "class2", "class3");
```

如果你需要从元素中移除一个类，可以像这样使用 `classList.remove()` 方法：

```js
classList.remove("highlight");
```

要切换元素上的类，你可以使用 `classList.toggle()` 方法。在此示例中，我们有一个 ID 为 `toggle-btn` 的 `button` 元素和一个 ID 为 `menu` 的 `nav` 元素。在 CSS 中，我们为菜单添加了一些样式以及一个名为 `show` 的类，该类将 `.menu` 的 `display` 属性更改为 `block`。在 JavaScript 中，我们使用 `getElementById()` 方法访问菜单元素和 `button` 元素。然后，我们为 `button` 元素添加了一个事件监听器，当按钮被点击时，将切换 `.menu` 元素上的 `show` 类：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button id="toggle-btn">Toggle Menu</button>

<nav id="menu" class="menu">
  <ul>
    <li>Home</li>
    <li>About</li>
    <li>Products</li>
  </ul>
</nav>
<script src="index.js"></script>
```

```css
.menu {
  display: none;
  background-color: lightgray;
  width: 50%;
  padding: 10px;
}

.menu.show {
  display: block;
}
```

```js
const menu = document.getElementById("menu");
const toggleBtn = document.getElementById("toggle-btn");

toggleBtn.addEventListener("click", () => menu.classList.toggle("show"));
```

:::

当用户点击按钮时，`show` 类将被添加到 `.menu` 元素，菜单将显示。如果用户再次点击按钮，`show` 类将从 `.menu` 元素中移除，菜单将隐藏。

通过使用诸如 `style`、`add()`、`remove()` 和 `toggle()` 等方法和属性，你可以轻松地用 JavaScript 操作元素的样式，创建动态且交互式的网页。
