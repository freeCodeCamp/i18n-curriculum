# --interactive--

CSS는 웹 페이지의 스타일을 담당합니다. 이러한 스타일은 모두 다양한 CSS 규칙으로 구성됩니다.

CSS 규칙은 두 가지 주요 부분으로 이루어집니다: **선택자**와 **선언** 블록입니다.

기본 구문을 살펴보겠습니다:

```css
selector {
  property: value;
}
```

선택자는 CSS에서 특정 HTML 요소를 식별하고 스타일을 적용하기 위해 사용하는 패턴입니다.

선택자의 예로는 타입 선택자, 클래스 선택자, ID 선택자가 있습니다.

기본 구문에 제공된 중괄호는 선언 블록으로 알려져 있습니다. 선언 블록은 주어진 선택자 또는 선택자들에 대해 스타일 집합을 적용합니다.

선언 블록 안에는 여러 선언이 있습니다. 각 선언은 속성과 값으로 구성됩니다.

속성은 어떤 기능에 스타일을 적용할지 지정하는 CSS 식별자입니다. 예를 들어 `background-color` 속성이 있습니다.

값은 해당 속성에 적용되는 구체적인 설정입니다. 예를 들어 속성이 `background-color`이라면, 값은 `purple`일 수 있으며, 이는 배경색을 보라색으로 설정합니다.

각 속성 이름 뒤에는 콜론을, 각 값 뒤에는 세미콜론을 붙여야 합니다.

이제 CSS 규칙의 구문을 알았으니, 예제를 살펴보겠습니다. 인터랙티브 편집기를 활성화하고 `styles.css` 탭을 클릭하여 CSS 코드를 확인하세요.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Learning about CSS</h1>

<p>Example paragraph element</p>
<p>Another example paragraph element</p>
```

```css
p {
  color: blue;
}
```

:::

이 CSS 규칙에서는 타입 선택자가 페이지의 모든 단락 요소를 대상으로 합니다.

선언 블록 안에는 하나의 선언이 있습니다.

선언은 `color` 속성과 `blue` 값으로 구성되어 있습니다. 이 선언은 모든 단락 요소의 텍스트 색상을 `blue`로 변경합니다.

여러 선택자에 동일한 스타일 집합을 적용하려면, 각 선택자를 쉼표로 구분하여 선택자 목록을 만들 수 있습니다.

다음은 여러 선택자를 스타일링하는 예입니다 (예제와 상호작용하려면 인터랙티브 편집기를 활성화하세요):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1 id="title">Example heading</h1>
<h2 class="subheading">Example subheading</h2>
<p>This paragraph is not affected by the selector.</p>
```

```css
#title,
.subheading {
  color: navy;
}
```

:::

이 선택자 목록에는 `id` 선택자가 있으며, `id` 값이 `title`인 HTML 요소를 대상으로 합니다. 모든 `id` 선택자는 해시 기호 `#`로 시작해야 합니다.

그 다음 쉼표가 있고 `class` 선택자가 있으며, `class` 값이 `subheading`인 모든 HTML 요소를 대상으로 합니다. 모든 클래스 선택자는 점 `.`으로 시작해야 합니다.

전체 선택자 목록에 동일한 스타일이 적용되며, 텍스트 색상은 `navy`로 설정됩니다.
