# --description--

當你說某事 `is likely to`，表示該事預期或可能會發生。它顯示某個特定動作在未來發生的機率很高。 例如，如果某種程式語言容易使用，你可能會說它 `is likely to` 保持受歡迎，意思是它未來很可能會持續受歡迎。
 
在英文中，你可以在動詞前使用字 `to` 來談論不同的動作。 例如，`to eat`、`to run` 或 `to learn`。 當你談論需要發生或可能在未來發生的事情時，你會使用這個結構。 例如，`I want to eat`、`She likes to run` 或 `They need to learn`。
 
所以，`to + infinitive` 是你表達可能或應該發生的動作的一種方式，通常表示計劃、願望或可能性。

`Python is likely to remain popular` 是什麼意思？

## --answers--

Python 很快就會消失。

### --feedback--

`Likely to remain` 表示對未來的預測，而非消失。
