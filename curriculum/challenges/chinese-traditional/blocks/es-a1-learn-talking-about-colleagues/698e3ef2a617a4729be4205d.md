# --explanation--

Mateo 問 `¿Dónde vive Valeria?`，意思是「Valeria 住在哪裡？」

`¿Dónde...?` 的意思是「在哪裡…？」

記得在句首要加上開頭問號 `¿`，並且在 `ó` 這個字的字母上加上重音符號。

`vive` 這個字的意思是「住」，是動詞 `vivir` 的第三人稱單數形式。
