# --description--

Збільшення або зменшення числа на одиницю — одна з найпоширеніших числових операцій.

Для цього існують спеціальні оператори:

- Інкремент `++` збільшує змінну на 1:

```js
let counter = 2;

// works the same as counter = counter + 1, but is shorter
counter++;      

console.log(counter); // 3
```

- Декремент `--` зменшує змінну на 1:

```js
let counter = 2;

// works the same as counter = counter - 1, but is shorter
counter--;

console.log(counter); // 1
```

Оператори `++` і `--` можна розміщувати як перед, так і після змінної.

- Коли оператор стоїть після змінної, це «постфіксна форма»: `counter++`.

- «Префіксна форма» — це коли оператор стоїть перед змінною: `++counter`.

Обидві ці інструкції роблять те саме: збільшують `counter` на `1`.

Чи є між ними різниця? Так, але її можна побачити лише якщо використовувати повернене значення `++/--`.

Пояснимо. Як ви знаєте, усі оператори повертають значення. Інкремент/декремент не виняток. Префіксна форма повертає нове значення, а постфіксна — старе (до збільшення/зменшення).

Щоб побачити різницю, ось приклад:

```js
let counter = 1;
let a = ++counter; // (*)

console.log(a); // 2
```

Підсумуємо:

- Якщо результат інкременту/декременту не використовується, різниці у виборі форми немає:

```js
let counter = 0;
counter++;
++counter;

console.log( counter ); // 2, the lines above did the same
```

- Якщо хочете збільшити значення і одразу використати результат оператора, потрібна префіксна форма:

```js
let counter = 0;
console.log( ++counter ); // 1
```

- Якщо хочете збільшити значення, але використати його попереднє значення, потрібна постфіксна форма:

```js 
let counter = 0;
console.log( counter++ ); // 0
```

# --questions--

## --text--

Які вихідні дані покажуть два `console.log` оператори в наведеному нижче коді JavaScript?

```js
let counter = 1;
console.log(2 * ++counter); // Statement A
```

```js
let counter = 1;
console.log(2 * counter++); // Statement B
```

## --answers--

Різниці немає; обидва `console.log` оператори покажуть однаковий результат.

---

Інструкція A виведе `4`, тому що `++counter` збільшує значення перед множенням. Інструкція B виведе `2`, тому що `counter++` збільшує значення після множення.

---

Інструкція A виведе `2`, тому що `++counter` є синтаксичною помилкою. Інструкція B виведе `4`, тому що `counter++` — правильний спосіб збільшення лічильника.

---

Обидві інструкції виведуть `3`, тому що лічильник збільшується в обох випадках перед `console.log`.
