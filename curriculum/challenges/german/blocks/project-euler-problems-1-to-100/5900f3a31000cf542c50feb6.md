# --description--

Wenn wir 47 nehmen, umkehren und addieren, ergibt 47 + 74 = 121, was palindromisch ist.

Nicht alle Zahlen erzeugen so schnell Palindrome. Zum Beispiel,

<div style="margin-left: 4em;">
  349 + 943 = 1292,<br>
  1292 + 2921 = 4213<br>
  4213 + 3124 = 7337<br>
</div>

Das heißt, 349 benötigte drei Iterationen, um ein Palindrom zu erreichen.

Obwohl es noch niemand bewiesen hat, wird angenommen, dass einige Zahlen, wie 196, niemals ein Palindrom erzeugen. Eine Zahl, die durch den Prozess des Umkehrens und Addierens niemals ein Palindrom bildet, nennt man eine Lychrel-Zahl. Aufgrund der theoretischen Natur dieser Zahlen nehmen wir für diese Aufgabe an, dass eine Zahl eine Lychrel-Zahl ist, bis das Gegenteil bewiesen ist. Außerdem wird Ihnen gesagt, dass für jede Zahl unter zehntausend gilt, dass sie entweder (i) in weniger als fünfzig Iterationen ein Palindrom wird oder (ii) bisher niemand mit der gesamten verfügbaren Rechenleistung es geschafft hat, sie in ein Palindrom zu überführen. Tatsächlich ist 10677 die erste Zahl, bei der gezeigt wurde, dass sie mehr als fünfzig Iterationen benötigt, um ein Palindrom zu erzeugen: 4668731596684224866951378664 (53 Iterationen, 28 Stellen).

Überraschenderweise gibt es palindromische Zahlen, die selbst Lychrel-Zahlen sind; das erste Beispiel ist 4994.

Wie viele Lychrel-Zahlen gibt es unter `num`?

**Hinweis:** Die Formulierung wurde am 24. April 2007 leicht geändert, um die theoretische Natur der Lychrel-Zahlen zu betonen.

```js
const countLychrelNumbers = (size) => {
  const numReverse = (num) => {
    return Number(num.toString().split('').reverse().join(''));
  };
  const isPalin = (num) => {
    if (numReverse(num) === num) {
      return true;
    }
    return false;
  };
  let total = 0;
  for (let i = 1; i < size; i++) {
    let loopCount = 1;
    let sum = i;
    while (loopCount < 50) {
      sum = sum + numReverse(sum);
      if (isPalin(sum)) {
        break;
      } else {
        loopCount++;
      }
    }
    if (loopCount === 50) {
      total++;
    }
  }
  return total;
}
```
