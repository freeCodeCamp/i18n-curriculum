# --description--

コンピューターサイエンスにおける分割統治法は、問題を再帰的により小さな部分問題に分解していく手法です。この手法の重要なポイントの一つは再帰で、関数が基本ケースに到達するまで自分自身を繰り返し呼び出すことを指します。このレッスンでは、マージソートアルゴリズムを見て、分割統治法がどのように機能するかを理解していきます。

次の数字のリストがあったとしましょう：

```md
42 37 53 17
```

このリストをマージソートアルゴリズムを使って、小さい順に並べ替えることが目標です。最初のステップは、そのリストを半分に分割することです：

```md
42 37 | 53 17
```

次にリストの左側を見てみます：

```md
42 37
```

そのサブリストをさらに半分に分割し、各サブリストが1つの要素だけになるまで繰り返します：

```md
42 | 37
```

1つの要素だけのリストはデフォルトではソート済みです。次に、それぞれの1要素のサブリストをマージしてソートされたリストにします：

```md
37 42
```

同じ手順を元のリストの右側にも適用します：

```javascript
// right side of original list
53 17

// divide the list in half
53 | 17

// merge the lists in sorted order
17 53
```

元のリストの両方の半分がソートされたので、それら2つの半分をマージして要素をソートします：

```markdown
17 37 42 53
```

アルゴリズムのコードは次のようになります：

```javascript
function mergeSort(arr) {
    if (arr.length <= 1) {
        return arr;
    }

    const mid = Math.floor(arr.length / 2);
    const left = mergeSort(arr.slice(0, mid));
    const right = mergeSort(arr.slice(mid));

    const sorted = [];
    let i = 0;
    let j = 0;

    while (i < left.length && j < right.length) {
        if (left[i] <= right[j]) {
            sorted.push(left[i]);
            i += 1;
        } else {
            sorted.push(right[j]);
            j += 1;
        }
    }

    return sorted.concat(left.slice(i)).concat(right.slice(j));
}
```

マージソートの時間計算量は`O(n log n)`です。なぜならリストが継続的に半分に分割され（`log n`）、その後マージされるからです（`O(n)`）。バブルソートのような他のソートアルゴリズムとは異なり、マージソートはインプレースでソートされず、空間計算量は`O(n)`です。
