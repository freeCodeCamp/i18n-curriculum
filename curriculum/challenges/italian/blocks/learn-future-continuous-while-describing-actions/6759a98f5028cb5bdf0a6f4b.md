# --explanation--

Le domande tag sono negative se la frase principale è positiva, e positive se la frase principale è negativa.

La frase `are they?` è una domanda tag aggiunta alla fine di una frase negativa per confermare o verificare un'informazione. Per esempio:

`The updates aren't applied automatically, are they?` - Qui, `are they?` verifica la conferma che gli aggiornamenti non vengano applicati automaticamente. La frase principale è negativa, quindi la domanda tag è positiva.
