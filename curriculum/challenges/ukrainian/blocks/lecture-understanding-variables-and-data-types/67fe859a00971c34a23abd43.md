# --description--

Перед тим як працювати зі змінними в Python, важливо розуміти типи даних. Тип даних описує, який саме тип значення зберігає змінна. Наприклад, число, текст або список елементів. Мови програмування використовують типи даних, щоб знати, як зберігати і працювати з різними видами інформації.

Python — це мова з динамічною типізацією, як і JavaScript, тобто вам не потрібно явно оголошувати типи змінних. Мова визначає тип змінної на основі того, яке значення ви їй призначаєте.

Ось кілька прикладів:

```python
name = 'John Doe' # Python knows this is a string
age = 25 # Python knows this is an integer
```

Це відрізняється від деяких мов зі статичною типізацією, як-от C#, Java та C++, де потрібно оголошувати типи змінних, наприклад так:

```csharp
string name = 'John Doe'
int age = 25
```

Динамічна типізація в Python робить програмування дуже швидким і гнучким, але може призводити до несподіваних помилок, бо помилки типів виявляються лише під час виконання програми, а не під час компіляції.

Оскільки Python визначає типи даних під час **виконання** програми, помилки, пов’язані з типами, виявляються лише в цей момент. Коли програма запускається, Python виконує код по рядках. Якщо він доходить до рядка, де очікується, що об’єкт поводитиметься певним чином, а він не може, Python зупиняється і показує помилку.

Натомість деякі мови **компілюють** програму перед запуском. Компіляція означає, що комп’ютер перевіряє код заздалегідь і готує його до запуску. Під час цього кроку такі мови можуть виявити помилки типів ще до початку виконання.

Вам поки не потрібно знати ці мови. Головна ідея така:

- **У Python помилки типів можуть проявлятися під час виконання**, коли програма фактично працює і використовує ваш код.

- **У компільованих мовах помилки типів виявляються під час компіляції**, до того, як програму дозволять запустити.

Через це ви можете дізнатися про помилку типу в Python лише тоді, коли програма дійде до конкретного рядка коду під час виконання.

Ось найпоширеніші типи даних, які ви будете використовувати в Python:

- Ціле число: ціле число без десяткових знаків, наприклад, `10` або `-5`.

```python
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- Подвійний (float): число з десятковою крапкою, як `4.41` або `-0.4`.

```python
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- Рядок: послідовність символів, укладена в одинарні або подвійні лапки, як `'Hello world!'`.

```python
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- Булевий: тип, що приймає значення істина або хиба, записується як `True` або `False`.

```python
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- Множина: невпорядкована колекція унікальних елементів, як `{0.5, 4, 'apple'}`.

```python
my_set_var = {7, 'hello', 8.5}
print('Set:', my_set_var) # Set: {7, 'hello', 8.5}
```

- Словник: колекція пар ключ-значення, укладена у фігурні дужки, як `{'name': 'John Doe', 'age': 28}`.

```python
my_dictionary_var = {'name': 'Alice', 'age': 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- Кортеж: незмінна впорядкована колекція, укладена в круглі дужки, як `('apple', 4.5, 7)`.

```python
my_tuple_var = (7, 'hello', 8.5)
print('Tuple:', my_tuple_var) # Tuple: (7, 'hello', 8.5)
```

- Діапазон: послідовність чисел, часто використовується в циклах, наприклад, `range(5)`.

```python
my_range_var = range(5)
print('Range:', my_range_var) # Range: range(0, 5)
```

- Список: впорядкована колекція елементів, що підтримує різні типи даних.

```python
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- None: спеціальне значення, яке означає відсутність значення.

```python
my_none_var = None
print('None:', my_none_var) # None: None
```

Щоб дізнатися тип змінної, можна використати функцію `type()`:

```python
my_var_1 = 'Hello world'
my_var_2 = 21

print(type(my_var_1)) # <class 'str'>
print(type (my_var_2)) # <class 'int'>
```

Ось усі типи даних, розглянуті в цьому уроці, разом із їхніми типами у терміналі:

```python
my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 'hello', 8.5}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 'hello', 8.5)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>
```

Вбудована функція `isinstance()` дозволяє перевірити, чи відповідає змінна певному типу. Вона приймає об’єкт і тип, який потрібно перевірити, і повертає булеве значення. Ось кілька прикладів:

```python
isinstance('Hello world', str) # True
isinstance(True, bool) # True
isinstance(42, int) # True
isinstance('John Doe', int) # False
```
