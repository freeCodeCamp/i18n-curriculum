# --description--

FRACTRAN è un linguaggio di programmazione esoterico completo di Turing inventato dal matematico John Horton Conway.

Un programma FRACTRAN è una lista ordinata di frazioni positive $P = (f_1, f_2, \\ldots, f_m)$, insieme a un input iniziale positivo intero $n$.

Il programma viene eseguito aggiornando l'intero $n$ nel modo seguente:

<ul>
  <li>per la prima frazione, $f_i$, nella lista per cui $nf_i$ è un intero, sostituisci $n$ con $nf_i$ ;</li>
  <li>ripeti questa regola finché nessuna frazione nella lista produce un intero moltiplicata per $n$, quindi termina.</li>
</ul>

Conway ha fornito un programma per i numeri primi in FRACTRAN:

$\\dfrac{17}{91}$, $\\dfrac{78}{85}$, $\\dfrac{19}{51}$, $\\dfrac{23}{38}$, $\\dfrac{29}{33}$, $\\dfrac{77}{29}$, $\\dfrac{95}{23}$, $\\dfrac{77}{19}$, $\\dfrac{1}{17}$, $\\dfrac{11}{13}$, $\\dfrac{13}{11}$, $\\dfrac{15}{14}$, $\\dfrac{15}{2}$, $\\dfrac{55}{1}$

Partendo da $n=2$, questo programma FRACTRAN cambierà $n$ in $15=2\\times (\\frac{15}{2})$, poi $825=15\\times (\\frac{55}{1})$, generando la seguente sequenza di numeri interi:

$2$, $15$, $825$, $725$, $1925$, $2275$, $425$, $390$, $330$, $290$, $770$, $\\ldots$

Dopo 2, questa sequenza contiene le seguenti potenze di 2:

$2^2=4$, $2^3=8$, $2^5=32$, $2^7=128$, $2^{11}=2048$, $2^{13}=8192$, $2^{17}=131072$, $2^{19}=524288$, $\\ldots$

che sono le potenze prime di 2.

function fractran(progStr){
  var num = new Array();
  var den = new Array();
  var val ;
  var out="";
  function compile(prog){
    var regex = /\s*(\d*)\s*\/\s*(\d*)\s*(.*)/m;
    while(regex.test(prog)){
      num.push(regex.exec(prog)[1]);
      den.push(regex.exec(prog)[2]);
      prog = regex.exec(prog)[3];
    }
  }

  function step(val){
    var i=0;
    while(i<den.length && val%den[i] != 0) i++;
    return num[i]*val/den[i];
  }

  var seq=[]

  function exec(val){
    var i = 0;
    while(val && i<limit){
      seq.push(val)
      val = step(val);
      i ++;
    }
  }

  // Main
  compile(progStr);
  var limit = 10;
  exec(2);
  return seq;
}
