# --interactive--

يمكن تطبيق CSS على صفحة ويب بثلاث طرق رئيسية: مضمَّنة، داخلية، أو خارجية.

لكل طريقة حالة استخدام ومزايا وقيود خاصة بها، ومعرفة متى تستخدم كل واحدة ضروري لكتابة شفرة نظيفة وفعالة وسهلة الصيانة.

دعنا نشرح الأنواع الثلاثة من CSS ومتى يجب استخدامها.

تُكتب CSS المضمَّنة مباشرة داخل عنصر HTML باستخدام الخاصية `style`. تطبق الأنماط على عنصر محدد.

إليك مثالًا يستخدم CSS المضمَّنة:

:::interactive_editor

```html
<p style="color: green;">This is an inline-styled paragraph.</p>
```

:::

في هذا المثال، نستخدم الخاصية `style` لتعيين نص الفقرة إلى اللون `green`.

تُستخدم CSS المضمَّنة عادةً لأنماط سريعة ومؤقتة أو لتجاوز أنماط أخرى لعنصر معين.

ومع ذلك، يجب تجنبها في معظم الحالات لأنها تزدحم HTML وتجعل الشفرة أصعب في الصيانة.

في معظم الأحيان، من الأفضل استخدام CSS الداخلية أو الخارجية للحفاظ على تنظيم أنماطك وسهولة صيانتها.

تُكتب CSS الداخلية داخل وسم `style` في قسم `head` من مستند HTML. تطبق الأنماط على الصفحة بأكملها وتكون مفيدة عند الحاجة لتنسيق مستند واحد.

إليك مثالًا على CSS الداخلية:

:::interactive_editor

```html
<head>
  <style>
    p {
      color: blue;
    }
  </style>
</head>
<body>
  <p>This paragraph is styled using internal CSS.</p>
</body>
```

:::

في المثال أعلاه، تطبق CSS الداخلية نصًا أزرق على جميع عناصر `p` في الصفحة.

تُستخدم CSS الداخلية بشكل أفضل عندما تحتاج لتطبيق أنماط على صفحة محددة بدلاً من عدة صفحات. هي مفيدة لمواقع الصفحة الواحدة أو عندما لا تحتاج الأنماط لإعادة الاستخدام في أماكن أخرى.

لكن هناك بعض العيوب، مثل عدم تعزيز إعادة الاستخدام عبر صفحات متعددة. بالإضافة إلى ذلك، مثل CSS المضمَّنة، تخلط بين HTML وCSS، مما يصعب صيانة الشفرة في المشاريع الكبيرة.

تُكتب CSS الخارجية في ملف `.css` منفصل ويرتبط بمستند HTML باستخدام عنصر `link` في قسم `head`.

تسمح لك بتنسيق صفحات متعددة بشكل متسق، وهي الطريقة المفضلة في تطوير الويب الاحترافي.

إليك مثالًا على تنسيق جميع عناصر الفقرة:

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <p>This paragraph is styled using external CSS.</p>
</body>
```

```css
p {
  color: red;
}
```

:::

في درس سابق، تعلمت أن عنصر `link` يحتوي على خاصية `rel` التي تحدد العلاقة بين المستند الحالي والموارد المرتبطة، مثل الربط بورقة أنماط أو مورد خارجي.

من ناحية أخرى، تحدد الخاصية `href` عنوان URL للمورد المرتبط، مما يشير إلى مكان استرجاع المورد منه.

تُعد CSS الخارجية مثالية للمشاريع الكبيرة حيث تريد الحفاظ على نمط متسق عبر صفحات متعددة.

تعزز فصل الاهتمامات بجعل HTML مسؤولًا عن الهيكل وCSS مسؤولًا عن التنسيق، مما يجعل الشفرة أكثر قابلية للصيانة والتوسع.

فهم متى تستخدم كل نوع من CSS ضروري لتطوير ويب فعال وكفء.

في معظم الحالات، يجب أن تكون CSS الخارجية هي الخيار الأساسي، خاصةً للمشاريع الأكبر والأكثر تعقيدًا.
