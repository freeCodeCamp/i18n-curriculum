# --description--

In diesem Labor erstellen Sie eine `rangeOfNumbers`-Funktion, die ein Array von Zahlen innerhalb eines angegebenen Bereichs generiert.

Beispiele:

- `rangeOfNumbers(3, 9)` gibt `[3, 4, 5, 6, 7, 8, 9]` zurück
- `rangeOfNumbers(5, 5)` gibt `[5]` zurück

Anforderungen:

- Verwenden Sie Rekursion (die Funktion muss sich selbst aufrufen)
- Keine Schleifen oder Array-Methoden (`for`, `while`, `Array.from`, `forEach`, `map`, `filter`, `reduce`)

**Ziel:** Erfüllen Sie die untenstehenden User Stories und lassen Sie alle Tests erfolgreich durchlaufen, um das Labor abzuschließen.

**User Stories:**

1. Sie sollen eine Funktion namens `rangeOfNumbers` erstellen, die zwei Parameter entgegennimmt: `startNum` und `endNum`.
1. Die Funktion soll ein Array von ganzen Zahlen zurückgeben, das mit der Zahl beginnt, die durch den `startNum`-Parameter dargestellt wird, und mit der Zahl endet, die durch den `endNum`-Parameter dargestellt wird (einschließlich).
1. Der `startNum` ist immer kleiner oder gleich dem `endNum`.
1. Ihre Funktion muss Rekursion verwenden, indem sie sich selbst aufruft. Sie darf keine Schleifen-Syntax verwenden (`for`, `while`, `Array.from()` oder höherwertige Funktionen wie `forEach`, `map`, `filter` oder `reduce`).
1. Die Funktion soll den Basisfall behandeln, bei dem `startNum` gleich `endNum` ist, indem sie ein Array zurückgibt, das nur diese einzelne Zahl enthält.
1. Für den rekursiven Fall soll die Funktion sich selbst mit veränderten Parametern aufrufen, um das Array aufzubauen, und dann die aktuelle Zahl zum Ergebnis hinzufügen.
1. Die Funktion soll sich nicht auf globale Variablen verlassen, um das Array zwischenzuspeichern oder aufzubauen.

# --solutions--

```js
function rangeOfNumbers(startNum, endNum) {
  if (endNum - startNum === 0) {
    return [startNum];
  } else {
    const numbers = rangeOfNumbers(startNum, endNum - 1);
    numbers.push(endNum);
    return numbers;
  }
}
```
