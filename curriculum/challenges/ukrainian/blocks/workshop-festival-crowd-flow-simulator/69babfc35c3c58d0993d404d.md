# --description--

Щоб завершити вашу функцію `processGateFlow`, поверніть об’єкт, що містить дві властивості:

- `processed`: кількість відвідувачів, яких ворота змогли обробити під час цього кроку.
- `overflow`: кількість відвідувачів, що залишилися в черзі після обробки (це `currentTickQueue` після циклу).

Ви повинні безпосередньо повернути об’єкт у такому вигляді:

```js
return {
  firstKey: firstValue,
  secondKey: secondValue
}
```
