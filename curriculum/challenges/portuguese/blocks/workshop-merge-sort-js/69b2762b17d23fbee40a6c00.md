# --description--

O algoritmo de merge sort realiza três ações:

- Dividir uma sequência não ordenada de itens em subpartes
- Ordenar os itens nas subpartes
- Mesclar as subpartes ordenadas

O que foi descrito acontece de forma recursiva até que as subpartes sejam mescladas na sequência completa ordenada. Vamos começar dividindo a sequência.

Primeiro, dentro da função `mergeSort`, declare uma variável `middlePoint` usando `const` e atribua a ela `Math.floor(array.length / 2)` para que você possa dividir a sequência ao meio antes de chamar a recursão.
