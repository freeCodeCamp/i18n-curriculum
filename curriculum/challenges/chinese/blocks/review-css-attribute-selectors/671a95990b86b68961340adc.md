## 使用不同的属性选择器和链接

- **定义**: 通过 `attribute` 选择器，你可以根据 HTML 元素的属性来定位这些元素，如 `href` 或 `title` 属性。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Link with href</a>
<a>No href</a>
```

```css
a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

- **`title` 属性**：该属性提供有关元素的附加信息。 以下是使用 `title` 属性定位链接的方法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Tooltip">Link with title</a>
<a href="#">Normal link</a>
```

```css
a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

- **组合选择器以匹配同时具有 `href` 和 `title` 属性的链接**：组合多个属性选择器。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Info">Href + Title</a>
<a href="#">Only href</a>
```

```css
a[href][title] {
  color: green;
}
```

:::

- **匹配以空格分隔的列表中的单个单词**：目标是具有特定类单词的链接。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a class="link primary">Primary link</a>
<a class="link secondary">Secondary link</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

- **匹配以特定前缀开头的值**：目标以 `https://` 开头的链接。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">HTTPS link</a>
<a href="http://www.freecodecamp.org">HTTP link</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

- **匹配以特定后缀结尾的值**：目标链接以 `.jpg` 结尾。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="photo.jpg">Image link</a>
<a href="index.html">HTML link</a>
```

```css
a[href$=".jpg"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

- **匹配包含子字串的值（任意位置）**：目标为值中任意位置包含 `https` 的链接。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Secure link</a>
<a href="page.html">Local link</a>
```

```css
a[href*="https"] {
  color: teal;
}
```

:::

- **总结**：这些模式使得基于链接的属性和值轻松实现一致的样式。

## 使用 `lang` 和 `data-lang` 属性定位元素

- **`lang` 属性**：该属性在 HTML 中用于指定元素内容的语言。 特别是在多语言网站上，你可能需要根据元素所使用的语言为其设计不同的样式。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">English text</p>
<p lang="fr">French text</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

- **`data-lang` 属性**：自定义数据属性如 `data-lang` 通常用于在元素中存储附加信息，例如指定特定文本部分所使用的语言。 以下是如何根据 `data-lang` 属性来样式化元素：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div data-lang="fr">French content</div>
<div data-lang="en">English content</div>
```

```css
div[data-lang="fr"] {
  color: blue;
}
```

:::

## 使用属性选择器、有序列表元素和 `type` 属性

- **`type` Attribute**：在 HTML 中处理有序列表时，`type` 属性允许你指定所使用的编号样式，如数字、字母或罗马数字。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Alpha</li>
  <li>Beta</li>
</ol>

<ol type="i">
  <li>One</li>
  <li>Two</li>
</ol>
```

```css
/*Example targeting uppercase alphabetical numbering*/
ol[type="A"] {
  color: purple;
  font-weight: bold;
}

/*Example targeting lowercase Roman numerals*/
ol[type="i"] {
  color: green;
}
```
