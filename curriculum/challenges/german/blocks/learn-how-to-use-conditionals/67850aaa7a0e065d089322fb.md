# --explanation--

Der Satz verwendet die `Second Conditional`, die folgendermaßen aufgebaut ist:

`If` + `Past Simple` oder `Past Continuous` Zeitform, `might / would / could` + Grundform des Verbs. In diesem Beispiel:

`If they closed` ist die Bedingung (Vorstellung einer Schließung).

`It would affect` ist das Ergebnis (was als Folge passieren würde).

Diese Art von Bedingungssatz wird verwendet, um unreale oder hypothetische Situationen zu beschreiben.
