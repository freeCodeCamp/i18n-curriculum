# --description--

Kuna njia kumi tu za kuchagua tatu kutoka tano, 12345:

<div style='text-align: center;'>123, 124, 125, 134, 135, 145, 234, 235, 245, na 345</div>

Katika mchanganyiko, tunatumia alama, $\\displaystyle \\binom 5 3 = 10$

Kwa ujumla, $\\displaystyle \\binom n r = \\dfrac{n!}{r!(n-r)!}$, ambapo $r \\le n$, $n! = n \\times (n-1) \\times ... \\times 3 \\times 2 \\times 1$, na $0! = 1$.

Haitokei hadi $n = 23$, thamani inapozidi milioni moja: $\\displaystyle \\binom {23} {10} = 1144066$.

Ni ngapi, si lazima ziwe tofauti, thamani za $\\displaystyle \\binom n r$ kwa $1 \\le n \\le 100$, ambazo ni zaidi ya milioni moja?

```js
function combinatoricSelections(limit) {
    const factorial = n =>
        Array.apply(null, { length: n })
            .map((_, i) => i + 1)
            .reduce((p, c) => p * c, 1);

    let result = 0;
    const nMax = 100;

    for (let n = 1; n <= nMax; n++) {
        for (let r = 0; r <= n; r++) {
            if (factorial(n) / (factorial(r) * factorial(n - r)) >= limit)
                result++;
        }
    }

    return result;
}
```
