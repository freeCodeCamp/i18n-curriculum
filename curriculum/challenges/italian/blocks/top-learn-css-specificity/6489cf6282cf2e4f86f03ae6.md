# --description--

```css
/* rule 1 */
#subsection .list {
  background-color: yellow;
  color: blue;
}

/* rule 2 */
#subsection .main .list {
  color: red;
}
```

In questo esempio finale, entrambe le regole usano selettori ID e classe, quindi nessuna delle due regole usa un selettore più specifico dell’altra. La cascata quindi verifica la quantità di ogni tipo di selettore. Entrambe le regole hanno solo uno selettore ID, ma la regola 2 ha più selettori di classe, quindi la regola 2 ha una specificità più alta!

Mentre la dichiarazione `color: red` avrebbe la precedenza, la dichiarazione `background-color: yellow` verrebbe comunque applicata dato che non c’è nessuna dichiarazione in conflitto per essa.
