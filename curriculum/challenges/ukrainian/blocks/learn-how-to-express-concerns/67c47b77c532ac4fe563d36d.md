# --explanation--

`A tight squeeze` буквально означає невеликий простір, де щось ледве поміщається. У цьому контексті це метафора, що означає, що щось важко виконати за короткий проміжок часу. Наприклад:

`Finishing this report by noon is going to be a tight squeeze.` — це означає, що буде складно завершити звіт до полудня.
