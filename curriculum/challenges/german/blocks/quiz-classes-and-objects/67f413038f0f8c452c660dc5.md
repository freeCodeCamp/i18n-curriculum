Welche der folgenden Möglichkeiten ist der korrekte Weg, eine Klasse zu definieren?

#### --distractors--

```py
def className(class):
    pass
```

---

```py
self className:
    pass
```

---

```py
class className
    pass
```

#### --answer--

```py
class className:
    pass
```

Was ist der Zweck der speziellen Methode `__init__`?

#### --distractors--

Sie legt eine Abkürzung fest, um eine Instanz einer Klasse zu erstellen.

---

Sie verhindert, dass ein Objekt doppelte Attribute hat.

---

Sie setzt den Wert von `self` auf das aktuelle Objekt.

#### --answer--

Sie initialisiert die Attribute von Objekten bei der Instanziierung.

### --question--

#### --text--

Was ist der Unterschied zwischen einem Instanzattribut und einem Klassenattribut?

#### --distractors--

Instanzattribute definieren Methoden, auf die von einem Objekt zugegriffen werden kann; Klassenattribute definieren Methoden, auf die von einer Klasse zugegriffen werden kann.

---

Instanzattribute werden innerhalb der `__init__`-Methode gesetzt; Klassenattribute werden innerhalb der `__class__`-Methode gesetzt.

---

Instanzattribute können direkt von der Klasse aus aufgerufen werden; Klassenattribute können nur von einer tatsächlichen Instanz der Klasse aufgerufen werden.

#### --answer--

Instanzattribute gehören zu einem bestimmten Objekt; Klassenattribute gehören zu einer Klasse.
