## `date()` 객체와 자주 쓰이는 메서드

- **정의**: `date()` 객체는 JavaScript에서 날짜와 시간을 생성, 조작, 형식화하는 데 사용됩니다. 다음 예제에서는 `new` 키워드를 사용해 `Date` 객체의 새 인스턴스를 만들고, `Date` 객체를 변수 `now`에 할당합니다. `now` 값을 콘솔에 출력하면, 코드를 실행하는 컴퓨터의 시스템 시계를 기준으로 현재 날짜와 시간이 표시됩니다.

:::interactive_editor

```js
const now = new Date();
console.log(now); // prints the current date and time
```

:::

- **`Date.now()` 메서드**: 이 메서드는 현재 날짜와 시간을 가져오는 데 사용됩니다. `Date.now()`는 1970년 1월 1일 00:00:00 UTC 이후 경과한 밀리초 수를 반환합니다. 이를 유닉스 에포크 시간이라고 합니다. 유닉스 에포크 시간은 정수로 쉽게 저장하고 조작할 수 있어 컴퓨터 시스템에서 날짜와 시간을 나타내는 일반적인 방식입니다. UTC는 세계가 시계와 시간을 조절하는 기본 시간 표준인 협정 세계시를 뜻합니다.
- **`getDate()` 메서드**: 이 메서드는 현재 날짜를 기준으로 월의 일을 가져오는 데 사용됩니다. `getDate()`는 월의 일에 따라 1부터 31 사이의 정수 값을 반환합니다. 날짜가 유효하지 않으면 `NaN` (숫자가 아님)을 반환합니다.

:::interactive_editor

```js
const now = new Date("2014-10-15");
const date = now.getDate();
console.log(date); // 15
```

:::

- **`getMonth()` 메서드**: 이 메서드는 월을 가져오는 데 사용됩니다. 월은 0부터 시작하므로 1월은 0, 2월은 1 등입니다. 이 예제에서 출력값은 9로, 10월에 해당합니다. 월이 유효하지 않으면 `NaN`을 반환합니다.

:::interactive_editor

```js
const now = new Date("2014-10-15");
const month = now.getMonth();
console.log(month); // 9
```

:::

- **`getFullYear()` 메서드**: 이 메서드는 연도를 가져오는 데 사용됩니다. 연도가 유효하지 않으면 `NaN`를 반환합니다.

:::interactive_editor

```js
const now = new Date("2014-10-15");
const year = now.getFullYear();
console.log(year); // 2014
```

:::

## 날짜를 형식화하는 다양한 방법

- **`toISOString()` 메서드**: 이 메서드는 날짜를 확장된 `ISO` (ISO 8601) 형식으로 형식화하는 데 사용됩니다. ISO 8601은 날짜와 시간을 나타내는 국제 표준입니다. 형식은 `YYYY-MM-DDTHH:mm:ss.sssZ`입니다.

:::interactive_editor

```js
const date = new Date("2014-10-15");
console.log(date.toISOString()); // "2014-10-15T00:00:00.000Z"
```

:::

- **`toLocaleDateString()` 메서드**: 이 메서드는 사용자의 로케일을 기준으로 날짜를 형식화하는 데 사용됩니다.

:::interactive_editor

```js
const date = new Date("2014-10-15");
console.log(date.toLocaleDateString("en-US"));  // "10/15/2014"
```

:::

`toLocaleDateString()` 메서드는 두 개의 선택적 매개변수인 locales와 options를 받습니다.

locales 매개변수는 사용할 로케일을 나타내는 문자열입니다. 예를 들어, 영어(미국)는 `"en-US"`, 프랑스어(프랑스)는 `"fr-FR"`를 전달할 수 있습니다. locales 매개변수를 전달하지 않으면 기본 로케일이 사용됩니다. 두 번째 선택적 매개변수는 options 매개변수입니다. 이 매개변수는 날짜 문자열의 형식을 지정할 수 있는 객체입니다.

:::interactive_editor

```js
const date = new Date("2014-10-15");
const options = {
  weekday: "long",
  year: "numeric",
  month: "long",
  day: "numeric",
};
console.log(date.toLocaleDateString("en-GB", options)); // "Wednesday, 15 October 2014"
```

:::
