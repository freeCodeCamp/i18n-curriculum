# --description--

在之前的课程中，你学习了如何使用 `for` 循环，它用于重复执行一段代码块指定的次数。下面是一个使用 `for` 循环将 `languages` 列表中的每种语言打印到控制台的示例：

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for language in languages:
    print(language)
```

但是如果你想跟踪每个元素的索引呢？一种选项是创建一个 `index` 变量，并在循环的每次迭代中将其递增 `1`，如下所示：

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

index = 0

for language in languages:
    print(f'Index {index} and language {language}')
    index += 1
```

虽然那样可行，但更简单的方法是使用 `enumerate()` 函数。`enumerate()` 函数跟踪可迭代对象的索引并返回一个枚举对象。

如果我们将 `languages` 列表传递给 `enumerate()` 函数，并使用 `list()` 函数将其返回值转换为列表，它看起来像这样：

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

list(enumerate(languages))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

枚举对象（现在是一个列表）中的每个条目都是一个元组，包含一个计数，后跟传递给 `enumerate()` 函数的可迭代对象中的一个值。

现在，让我们重构之前的示例以使用 `enumerate()` 函数：

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')
```

我们将枚举对象中每个元组的计数和值分别解包到名为 `index` 和 `language` 的变量中。最后，这两个变量都被用在一个 f-字串中，该字串在循环的每次迭代中打印到控制台。

```md
Index 0 and language Spanish
Index 1 and language English
Index 2 and language Russian
Index 3 and language Chinese
```

这消除了手动创建和更新 `index` 变量的需求。

`enumerate()` 函数还接受一个可选的 `start` 参数，用于指定计数的起始值。如果省略此参数，则计数将从 `0` 开始。以下是使用可选 `start` 参数的示例：

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages, 1):
    print(f'Index {index} and language {language}')
```

下面是在控制台中结果的显示效果：

```md
Index 1 and language Spanish
Index 2 and language English
Index 3 and language Russian
Index 4 and language Chinese
```

到目前为止，我们只迭代了一个列表。但如果你需要并行迭代多个可迭代对象怎么办？你可以使用 `zip()` 函数，它将列表组合成元素的点对并返回元组的迭代器。

如果我们将 `developers` 和 `ids` 的列表传递给 `zip()` 函数，并使用 `list()` 函数将其返回值转换为列表，结果如下：

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

list(zip(developers, ids))
# [('Naomi', 1), ('Dario', 2), ('Jessica', 3), ('Tom', 4)]
```

下面是使用 `zip()` 函数和 `for` 循环迭代 `developers` 和 `ids` 的示例：

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')
```

在这个例子中，`zip()` 将两个列表组合成元素的点对并返回一个元组的迭代器。然后，`for` 循环将每个元组解包到 `name` 和 `id` 中。最后，对于每个打印语句，我们分别打印来自 `ids` 和 `developers` 列表的每个 `name` 和 `id`。以下是在控制台中显示的结果：

```md
Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
```

`enumerate()` 和 `zip()` 函数非常强大，当与循环结合使用时，可以使你的代码更加简洁。
