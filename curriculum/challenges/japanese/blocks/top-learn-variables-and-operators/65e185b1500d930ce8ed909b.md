# --description--
最も簡単な始め方は、JavaScriptのコードを含むHTMLファイルを作成することです。パソコンのどこかに基本的なHTMLの骨組みを入力してください。

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>
  <script>
    // Your JavaScript goes here!
    console.log("Hello, World!")
  </script>
</body>
</html>
```

このファイルを保存してウェブブラウザで開きます（Visual Studio Codeの<a href="https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer" target="_blank" >"Live Server"</a>を使うと便利です）。次に、空白のウェブページ上で右クリックして「検証」または「要素を検証」を選び、ブラウザのコンソールを開いてください。‘開発者ツール’のペインで‘コンソール’タブを選ぶと、`console.log`文の出力が見えます。

> `console.log()`はブラウザの開発者コンソールに何かを出力するコマンドです。この記事や演習のどの例でも結果をコンソールに出力するために使えます。すべての例を実際にコードを書きながら学ぶことをお勧めします。

JavaScriptをウェブページに含める別の方法は、外部スクリプトを使うことです。これは外部のCSSファイルをウェブサイトにリンクするのと非常に似ています。

```html
  <script src="javascript.js"></script>
```

JavaScriptファイルの拡張子は`.js`で、スタイルシートの`.css`に似ています。外部JavaScriptファイルはより複雑なスクリプトに使われます。

# --questions--

## --text--

HTMLドキュメントにJavaScriptを含める方法について正しく説明している文はどれですか？

## --answers--

JavaScriptはHTMLの本文内の`<script>`タグの中にのみ内部で含めることができます。

---

JavaScriptコードは正しく実行されるためにHTMLドキュメントの`<head>`セクションに書かなければなりません。

---

JavaScriptはHTMLファイル内で`<script>`タグを使って直接含めるか、`<script src="javascript.js">`タグを使って外部からリンクできます。

---

外部JavaScriptファイルはウェブブラウザに認識されるために`<script>`タグに特別な属性が必要です。
