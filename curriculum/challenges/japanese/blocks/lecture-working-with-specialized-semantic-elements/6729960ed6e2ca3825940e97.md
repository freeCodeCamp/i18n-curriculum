# --interactive--

インラインコード要素は、テキスト内の短いコードスニペットを表すために使われます。code要素の一般的な使用例は、技術記事やドキュメントページです。

以下は、`code`要素を使ってCSSのコードスニペットを表示する例です。

:::interactive_editor

```html
<p>
  To set the text color to blue in CSS, use the following code:
  <code>color: blue;</code>
</p>
```

:::

この例では、CSSの`color`プロパティを使ってテキストの色を`blue`に設定しています。このコードスニペットを`<code>`タグで囲むことで、ブラウザに対してテキストがインラインコードであることを伝えています。

ブラウザは`code`要素内のコンテンツに対してデフォルトのスタイルを適用します。デフォルトのスタイルは等幅フォントです。

`code`要素は1行のコードを表すためのものです。複数行のコードを表したい場合は、`code`要素を整形済みテキスト要素の中に配置する必要があります。

整形済みテキスト要素は整形済みテキストを表すために使われます。以下は整形済みテキスト要素を使ってCSSの宣言を表示する例です。

:::interactive_editor

```html
<pre>
  <code>
    body {
      color: red;
    }
  </code>
</pre>
```

:::

`pre`要素を使う場合は、スペースの扱いに注意してください。HTMLドキュメント内に書かれた通りに表示されるためです。

ブラウザではコードが右に数スペース分インデントされているのがわかります。コード例のインデントを変更すると、画面上のインデントも変わります。

HTMLドキュメント内にコード例を含める場合、短いインライン例には`code`要素を使うことをお勧めします。

より長いコードスニペットを表示する必要がある場合は、`pre`と`code`要素を使う必要があります。
