# --description--

在上一課中，你學到了什麼是 Python 以及你可以用它做什麼。現在，讓我們來看看如何在你的區域的機器上設定 Python。

在 Windows 和 Mac 上安裝 Python 最簡單的方法是從官方 Python 網站下載安裝程式。我們稍後也會在本課程中介紹如何在 Linux 上執行 Python。

前往 `https://www.python.org/` 並暫留在「Downloads」上。將會出現一個視窗，顯示你作業系統（OS）目前的 Python 版本。

我們將先介紹如何在執行 macOS 的電腦上安裝 Python：

- 點擊顯示先前視窗中目前 Python 版本的按鈕，系統將自動開始下載 `.pkg` 安裝檔案。
- 當 `.pkg` 安裝程式下載完成後，打開它，然後在開啟的視窗中點擊「繼續」。
- 持續點擊「Continue」按鈕直到你到達「Installation Type」區段。在那裡，點擊「Install」按鈕。
- 如有必要，請輸入你的密碼，然後開始安裝。
- 之後，你應該會收到一則恭喜訊息，表示 Python 已成功安裝。
- 點擊「關閉」按鈕，即完成！

你可以透過開啟你的終端機並執行 `python --version` 或 `python3 --version` 來驗證安裝。

你也可以在終端機中執行 `python` 或 `python3` 來開啟 Python 直譯器。

<dfn>終端機</dfn> 是一種基於文字的介面，讓你透過輸入指令與你的電腦互動。每個作業系統都附帶一個預設的終端機應用程式。在 macOS 上，你可以使用 Terminal 應用程式。在 Windows 上，你可以使用命令提示字元或 PowerShell。在 Linux 上，每個桌面環境都有自己的預設終端機應用程式，例如 GNOME Terminal 或 Konsole。

請注意，在某些較舊的 macOS 和 Linux 系統上，`python` 可能保留給 Python 2，而 `python3` 則專門用於 Python 3。如果你執行 `python --version` 並看到像 `Python 2.7.18` 這樣的 Python 2 版本，那麼你的作業系統可能依賴於某些使用舊版 Python 撰寫的軟體。如果是這種情況，你應該使用 `python3` 來執行你的 Python 程式碼。Python 2 已經終止支援，不應用於任何新的開發。

要在 Windows 上安裝 Python，請遵循以下步驟：

- 前往 `https://www.python.org/`，並暫留在“Downloads”上。你應該會看到一個顯示“Download for Windows”的模式視窗，以及一個帶有目前 Python 版本的下載按鈕。
- 點擊版本號，系統會自動開始下載 Windows 可執行檔（`.exe`）檔案。
- 當你完成下載適用於 Windows 的 Python 安裝程式後，請雙擊它，並依照指示操作。
- 當你看到選項 `Add python.exe to Path` 時，請勾選該選項，然後點擊 `Install Now`。這樣做會讓你之後的操作更簡單。


你可以透過開啟像 PowerShell 這樣的命令列殼層並執行 `python --version` 來驗證安裝。你也可以透過執行 `python` 來開啟 Python 直譯器。

對於 Linux 上的 Python，大多數主要發行版如 Ubuntu、Debian 和 Fedora 都附帶 Python。

只要打開終端機並執行 `python --version`，或是 `python3 --version`：

如果任一指令未顯示 Python 版本，你可以在 `https://www.python.org` 為你的 Linux 版本搜尋安裝軟體包，或線上搜尋為你的發行版安裝 Python 的推薦方式。
