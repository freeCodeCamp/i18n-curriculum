# --instructions--

Per questa sfida, devi creare una calcolatrice finanziaria usando Python che prenda input dall’utente e faccia quanto segue:

- Calcolare un’annualità con crescita mensile o continua
- Calcolare la rata mensile del mutuo
- Stimare il saldo di un investimento per la pensione
- Determinare quanto tempo ci vuole perché un importo raddoppi, dato il tasso
- Risolvere equazioni logaritmiche
- Convertire in notazione scientifica (e viceversa)

Se hai difficoltà, puoi seguire la video guida per questo progetto.

Quando hai finito, invia l’URL del notebook Colab pubblico sul tuo Google drive.
