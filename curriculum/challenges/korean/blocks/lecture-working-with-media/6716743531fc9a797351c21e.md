# --interactive--

교체된 요소는 내용이 CSS 자체가 아니라 외부 리소스에 의해 결정되는 요소입니다. CSS, 즉 캐스케이딩 스타일시트는 웹 페이지에 스타일을 추가하는 데 사용됩니다. 교체된 요소의 일반적인 예로는 image, iframe, video 요소가 있습니다.

교체된 요소를 사용하면 요소의 위치나 레이아웃을 제어할 수 있습니다. 하지만 CSS로 그 요소의 내용을 직접 수정할 수는 없습니다. 몇 가지 예시로 설명하는 것이 더 쉬울 수 있습니다. 웹 페이지에 이미지를 임베드하는 image 요소를 생각해 보세요:

```html
<img src="example-img-url" alt="Descriptive text goes here">
```

요소 자체는 외부 객체인 이미지로 교체됩니다. CSS는 이미지의 위치를 제어하거나 필터를 적용할 수 있지만, 이미지 자체를 수정할 수는 없습니다. 좀 더 구체적인 예는 `iframe` 요소일 수 있는데, 이 요소는 외부 사이트를 웹 페이지에 임베드합니다.

다음은 freeCodeCamp 채널의 인기 YouTube 동영상을 `iframe` 요소로 임베드한 예시입니다. 미리보기 창에서 다른 동영상을 보고 싶으면 `src` 속성 값을 원하는 동영상으로 변경하세요. 미리보기를 보려면 인터랙티브 편집기를 활성화해야 합니다.

**참고**: 인터랙티브 예시에서 언급된 `referrerpolicy`와 `allowfullscreen` 같은 추가 속성은 걱정하지 마세요. `iframe` 요소 작업에 대해서는 앞으로 워크숍에서 더 배우게 됩니다.

:::interactive_editor

```html
<iframe width="400" height="200" src="https://www.youtube.com/embed/u43gJJrVa1I?si=BoDW_puFsy8OEr_Z" title="Professional Cloud Architect Certification Course – Pass the Exam! (YouTube video)" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen></iframe>
```

:::

`iframe` 요소를 사용하는 또 다른 일반적인 예는 페이지에 지도를 임베드하는 것입니다. 다음은 임베드된 지도 예시입니다.

인터랙티브 편집기를 활성화하고 지도를 확대/축소하며 직접 조작해 보세요.

:::interactive_editor

```html
<iframe
  title="Map of the Royal Observatory, Greenwich, London"
  width="300"
  height="200"
  src="https://www.openstreetmap.org/export/embed.html?bbox=-0.004017949104309083%2C51.47612752641776%2C0.00030577182769775396%2C51.478569861898606&amp;layer=mapnik">
</iframe>
```

:::

요소 자체는 외부 객체인 사이트로 교체됩니다. CSS로 임베드된 사이트의 위치를 변경할 수 있지만, 사이트 내용은 수정할 수 없습니다. 좀 더 자세히 들어가면, 임베드된 사이트에 `h1` 요소가 있다면 CSS로 그 `h1` 요소를 스타일링할 수 없습니다. 크기, 폰트 색상 등을 변경할 수 없습니다.

다른 교체된 요소로는 `video`, `embed` 등이 있습니다. 그리고 특정 상황에서 교체된 요소처럼 동작하는 요소도 있습니다. 다음은 `input` 속성이 `type`으로 설정된 `image` 요소 예시입니다:

```html
<input type="image" alt="Descriptive text goes here" src="example-img-url">
```

이 유형의 `input`는 교체된 요소로 간주되지만, `input`이나 `text` 같은 다른 `email` 유형은 교체된 요소가 아닙니다.
