# --interactive--

반응형 웹 디자인은 데스크톱 컴퓨터부터 모바일폰까지 다양한 기기에서 최적의 시청 및 상호작용 경험을 제공하는 웹사이트를 만드는 웹 개발 접근법입니다.

반응형 디자인의 핵심 원칙은 적응성입니다 – 웹사이트가 보고 있는 기기의 화면 크기와 기능에 따라 레이아웃과 콘텐츠를 조정하는 능력입니다.

반응형 디자인은 일반적으로 세 가지 주요 컴포넌트에 의존합니다 - 유동 그리드, 유연한 이미지, 그리고 미디어 쿼리:

- 유동 그리드는 픽셀 같은 고정 단위 대신 백분율 같은 상대 단위를 사용하여 화면 크기에 따라 콘텐츠가 크기 조정되고 재배치되도록 합니다.

- 유연한 이미지는 포함하는 요소 내에서 크기 조정되도록 설정되어 작은 화면에서 컨테이너를 넘치지 않게 합니다.

- 그리고 미디어 쿼리는 주로 뷰포트 너비 같은 기기 특성에 따라 개발자가 다른 스타일을 적용할 수 있게 합니다. 미디어 쿼리에 대해서는 앞으로 수업에서 더 배우게 됩니다.

반응형 웹 디자인과 CSS Grid, Flexbox 같은 도구의 관계는 상호 보완적입니다. 반응형 디자인은 개념이나 접근법인 반면, CSS Grid와 Flexbox는 반응형 디자인 구현을 훨씬 쉽고 효율적으로 만드는 실용적인 도구입니다.

이전 수업에서 Flexbox를 다뤘고 앞으로 수업에서 CSS Grid를 다루게 됩니다. 지금은 CSS Grid에 대한 간단한 소개를 드리겠습니다.

CSS Grid는 더 복잡한 배열을 가능하게 하는 2차원 레이아웃 시스템입니다. 전체 페이지 레이아웃뿐 아니라 작은 컴포넌트 레이아웃을 만드는 데도 훌륭합니다.

다음은 CSS Grid가 반응형으로 사용되는 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr;
  gap: 20px;
}

@media (min-width: 768px) {
  .grid-container {
    grid-template-columns: 1fr 1fr;
  }
}

@media (min-width: 1024px) {
  .grid-container {
    grid-template-columns: 1fr 1fr 1fr;
  }
}
```

:::

이 예제에서 그리드는 작은 화면에서는 한 열로 시작하고, 중간 크기 화면에서는 두 열로, 큰 화면에서는 세 열로 전환됩니다. 이는 Grid가 다양한 화면 크기에 맞춰 적응하는 반응형 레이아웃을 만들 수 있음을 보여줍니다.

Flexbox와 Grid는 모두 플로트나 표 기반 레이아웃 같은 이전 레이아웃 방식보다 큰 장점을 제공합니다. 더 많은 유연성과 제어를 제공하고, 코드가 적으며, 반응형을 염두에 두고 설계되었습니다. 개발자가 복잡하고 반응형인 레이아웃을 상대적으로 쉽게 만들 수 있게 해주어 반응형 웹 디자인 구현에 없어서는 안 될 도구입니다.

Flexbox와 Grid는 반응형 디자인에서 함께 자주 사용된다는 점도 주목할 만합니다. Flexbox는 주로 컴포넌트와 1차원 레이아웃에 사용되고, Grid는 전체 페이지 구조와 2차원 레이아웃에 사용됩니다. 둘 중 선택은 디자인의 특정 레이아웃 요구에 따라 달라집니다.

Flexbox와 Grid 외에도 다른 CSS 기능들도 반응형 디자인에서 중요한 역할을 합니다.

예를 들어 `calc()` 함수는 단위를 혼합하고 계산을 수행할 수 있어 유연한 레이아웃을 만드는 데 매우 유용합니다.

반응형 이미지는 반응형 웹 디자인의 또 다른 중요한 측면입니다. HTML5의 `srcset` 속성과 `picture` 요소는 기기 기능에 따라 다른 이미지 파일을 제공할 수 있어, 작은 화면이나 낮은 해상도의 기기에서 불필요하게 큰 이미지 파일을 다운로드하지 않도록 합니다.

결론적으로, 반응형 웹 디자인은 어떤 기기에서도 잘 작동하는 웹사이트를 만드는 접근법입니다.

특정 기술이라기보다 디자인 철학이지만, 미디어 쿼리 같은 CSS 기능에 크게 의존하며, Flexbox와 Grid 같은 최신 레이아웃 도구 덕분에 훨씬 수월해졌습니다. 이 도구들은 진정한 반응형 디자인을 만드는 데 필요한 유연성과 제어를 제공하여, 웹에 접근하는 기기의 다양성이 계속 증가하는 상황에서 웹사이트가 원활하게 적응하도록 합니다.
