# --description--

Bis jetzt haben Sie Regex verwendet, um Strings exakt zu matchen. Manchmal möchten Sie aber auch Groß- und Kleinschreibung ignorieren.

Groß- und Kleinschreibung (oder manchmal Buchstabenschreibung) bezeichnet den Unterschied zwischen Großbuchstaben und Kleinbuchstaben. Beispiele für Großbuchstaben sind `A`, `B` und `C`. Beispiele für Kleinbuchstaben sind `a`, `b` und `c`.

Sie können beide Schreibweisen mit einem sogenannten Flag matchen. Es gibt weitere Flags, aber hier konzentrieren Sie sich auf das Flag, das die Groß- und Kleinschreibung ignoriert – das `i`-Flag. Sie können es verwenden, indem Sie es an den Regex anhängen. Ein Beispiel für die Verwendung dieses Flags ist `/ignorecase/i`. Dieser Regex kann die Strings `ignorecase`, `igNoreCase` und `IgnoreCase` matchen.

# --instructions--

Schreiben Sie einen Regex `fccRegex`, der `freeCodeCamp` matcht, unabhängig von der Groß- oder Kleinschreibung. Ihr Regex sollte keine Abkürzungen oder Varianten mit Leerzeichen matchen.
