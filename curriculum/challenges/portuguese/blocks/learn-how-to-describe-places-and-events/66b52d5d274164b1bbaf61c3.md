# --explanation--

O tempo `Present Continuous` é usado para descrever ações que estão acontecendo no momento da fala.É formado usando o verbo `to be` (`am, is, are`) + a forma base do verbo + `-ing`.Por exemplo:

- `She is coding a new feature right now.` - Ela está no processo de codificar uma nova funcionalidade neste momento.

- `They are testing the software.` - Eles estão no meio de testar o software agora mesmo.
