# --explanation--

`To address`는 문제나 상황을 처리하기 위해 조치를 취하는 것을 의미합니다. 예를 들어:

`We need to address the login error before users report it.` – 이 표현은 문제를 해결하기 위해 조치가 필요하다는 뜻입니다.

`Quickly`는 무언가가 얼마나 빠르게 이루어졌는지를 나타냅니다. 반면 `Quick`는 단순히 무언가가 빠르다는 것을 묘사합니다. 예를 들어:

- `She responded quickly to the email.` – 이메일에 그녀가 얼마나 빠르게 답변했는지를 설명할 때 `quickly`를 사용합니다.

- `It was a quick meeting.` – 여기서는 회의 자체가 짧고 적절한 시간 내에 끝났다는 것을 설명합니다.
