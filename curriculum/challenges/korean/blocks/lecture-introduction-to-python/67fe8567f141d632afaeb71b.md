# --description--

지난 수업에서 Python이 무엇이고 무엇을 할 수 있는지 배웠습니다. 이제 로컬 컴퓨터에 Python을 어떻게 설정하는지 살펴보겠습니다.

Windows와 Mac에서 Python을 설치하는 가장 쉬운 방법은 공식 Python 웹사이트에서 설치 프로그램을 다운로드하는 것입니다. 이 수업 후반에는 Linux에서 Python을 실행하는 방법도 다룹니다.

`https://www.python.org/`에 접속해 "Downloads" 위에 마우스를 올려보세요. 운영 체제(OS)에 맞는 현재 Python 버전이 표시된 모달이 나타납니다.

먼저 macOS에서 Python을 설치하는 방법을 알아보겠습니다:

- 현재 Python 버전이 표시된 버튼(이전 모달에서)을 클릭하면 `.pkg` 설치 파일이 자동으로 다운로드됩니다.
- `.pkg` 설치 프로그램 다운로드가 완료되면 열고, 열린 창에서 "Continue"를 클릭하세요.
- "Continue" 버튼을 계속 클릭해 "Installation Type" 섹션까지 진행하세요. 거기서 "Install" 버튼을 클릭합니다.
- 필요하면 비밀번호를 입력한 후 설치를 시작하세요.
- 설치가 완료되면 Python이 성공적으로 설치되었다는 축하 메시지가 표시됩니다.
- "Close" 버튼을 클릭하면 완료입니다!

터미널을 열고 `python --version` 또는 `python3 --version`를 실행해 설치를 확인할 수 있습니다.

또한 터미널에서 `python` 또는 `python3`를 실행해 Python 인터프리터를 열 수 있습니다.

<dfn>터미널</dfn>은 명령어를 입력해 컴퓨터와 상호작용할 수 있는 텍스트 기반 인터페이스입니다. 각 운영 체제는 기본 터미널 앱을 제공합니다. macOS에서는 Terminal 앱을, Windows에서는 Command Prompt 또는 PowerShell을, Linux에서는 GNOME Terminal이나 Konsole 같은 데스크톱 환경별 기본 터미널 앱을 사용할 수 있습니다.

일부 구형 macOS 및 Linux 시스템에서는 `python`가 Python 2용으로 예약되어 있고 `python3`가 Python 3용일 수 있습니다. `python --version`를 실행했을 때 `Python 2.7.18` 같은 Python 2 버전이 표시된다면, OS가 구버전 Python으로 작성된 소프트웨어에 의존할 가능성이 있습니다. 이 경우 앞으로 Python 코드를 실행할 때 `python3`를 사용하세요. Python 2는 지원이 종료되어 새로운 개발에 사용해서는 안 됩니다.

Windows에서 Python을 설치하려면 다음 단계를 따르세요:

- `https://www.python.org/`에 접속해 “Downloads” 위에 마우스를 올려보세요. "Download for Windows"라는 모달과 현재 Python 버전이 적힌 다운로드 버튼이 보입니다.
- 버전 번호를 클릭하면 Windows 실행 파일(`.exe`)이 자동으로 다운로드됩니다.
- Windows용 Python 설치 프로그램 다운로드가 완료되면 더블클릭해 지시에 따라 설치하세요.
- `Add python.exe to Path` 옵션이 보이면 선택한 후 `Install Now`를 클릭하세요. 이렇게 하면 나중에 더 편리합니다.

PowerShell 같은 명령줄 셸을 열고 `python --version`를 실행해 설치를 확인할 수 있습니다. Python 인터프리터는 `python`를 실행해 열 수 있습니다.

Linux에서는 Ubuntu, Debian, Fedora 같은 주요 배포판에 Python이 기본으로 포함되어 있습니다.

터미널을 열고 `python --version` 또는 `python3 --version`를 실행하세요:

두 명령어 중 어느 것도 Python 버전을 표시하지 않으면 `https://www.python.org`에서 Linux 배포판에 맞는 설치 패키지를 검색하거나 온라인에서 권장 설치 방법을 찾아보세요.
