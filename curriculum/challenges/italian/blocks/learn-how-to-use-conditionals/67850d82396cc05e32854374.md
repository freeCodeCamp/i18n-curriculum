# --explanation--

Una **domanda a tag** è una breve domanda aggiunta alla fine di una frase, usata per confermare un'informazione o invitare all'accordo. Si forma:

- Prendendo il verbo ausiliare (ad esempio `would`, `is`, `has`, `will`) dalla frase principale.

- Usando la forma opposta: positiva se la frase è negativa, negativa se la frase è positiva.

- Aggiungendo il pronome appropriato.

- La tag è `wouldn't it?` (forma negativa di `would`).
