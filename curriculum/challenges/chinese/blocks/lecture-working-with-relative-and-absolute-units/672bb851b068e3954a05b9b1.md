# --interactive--

使用 `calc()` 函数，你可以直接在样式表中执行计算，以动态确定属性值。这意味着你可以通过基于视口大小或其他元素计算维度，创建灵活且响应式的用户界面。

`calc()` 是一个 CSS 函数。你将在开始学习 JavaScript 时了解更多关于函数的内容，但在本课中，你将学习理解 `calc()` 工作原理所需的基础知识。

函数是执行特定任务的代码块。某些函数已经在 CSS 中定义，因此你可以直接使用它们，并传递任何必要的值以自定义它们执行任务的方式。

在编程世界中，当我们运行由函数执行的任务时，我们称之为“调用”函数。传递给函数的值称为参数。 

正如你在下面的代码中看到的，要调用一个函数，你需要写出它的名称，后跟括号内用逗号分隔的参数。函数名称和左括号之间不应有空格：

```css
function(argument1, argument2, argument3)
```

一个函数可能只需要一个值来确定该做什么。在这种情况下，它只会接受一个参数。`calc()` 函数就是这种情况。它接受一个参数，因为它需要知道计算什么。

为此，你传递一个称为表达式的参数。表达式是产生结果的值和操作符的组合。

这就是你如何调用 `calc()` 函数。你写名称 calc，后跟括号，在括号内，你写表达式：

```css
calc(expression)
```

表达式会被计算以得出最终结果。“计算”只是意味着值和操作符在幕后被转换成单一的值。结果会被赋值到进行计算的 CSS 属性。

你可以对表现长度、角度、时间、百分比、数字和颜色的值执行计算。你也可以组合不同的单位，比如像素、百分比和 em。

对于数字，表达式中的所有值，也称为操作对象，必须具有相应的单位，如 `px`、`em` 和百分比（`%`）。根据操作符的不同，不同的操作对象可能具有不同的单位。

你可以在表达式中使用加法（`+`）、减法（`-`）、乘法（`*`）和除法（`/`）操作符。

如果有多个操作对象和操作符，`calc()` 将遵循标准的操作符优先级规则。如果需要，你也可以添加括号来确定操作的顺序。

在下面的示例中，你可以看到一个带有文本 `Hello, World!` 的 `div`。

使用 CSS 类型选择器选择 `div`，你可以将其样式设置为白色文本和深蓝色背景： 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>Hello, World!</div>
```

```css
div {
  color: white;
  background-color: #1b1b32;
  width: calc(50% - 20px);
}
```

:::

这里的新内容是宽度是动态计算的。注意我们如何调用 `calc()` 函数并传递一个表达式作为参数。该表达式有两个不同单位的操作对象和一个操作符，即减法操作符。

百分比是一个相对单位。该值（`50%`）将由父容器的宽度决定。然后，从该值中减去 `20px`。该表达式的结果将决定 `div` 的宽度。

`div` 的宽度大约是其容器总宽度的一半，如果你调整父容器的大小，宽度将自动重新计算。

这就是使用 `calc()` 的关键优势。你可以根据应用或视口的不同方面动态确定 CSS 属性的值。

表达式也可以包含 CSS 函数和变量，如果你需要在计算中使用它们。你将在接下来的课程中学习更多关于 CSS 变量的内容。

太好了。既然你已经了解了 `calc()` 函数 的基础知识，接下来让我们介绍一些最佳实践。

首先，你必须在加法操作符（`+`）和减法操作符（`-`）两侧添加空白。

例如，下面的表达式不是有效的表达式，因为减法操作符紧挨着第二个操作对象。

```css
calc(100% -30px)
```

减法操作符（`-`）两边必须有空白字符，就像这样。添加空白字符将创建一个有效的表达式。

```css
calc(100% - 30px)
```

这对于乘法和除法操作符不是必须的，但强烈推荐。

如果你需要执行计算并在其他计算中使用这些结果，也可以嵌套 `calc()` 函数调用。

此外，如果你使用值零来表现传递给 `calc()` 函数的表达式中的长度，必须包含单位。例如，以下表达式无效：

```css
calc(100% - 0)
```

你需要添加单位，比如 px。

```css
calc(100% - 0px)
```

你还应该知道，目前如果你使用乘法或除法操作符，其中一个操作对象必须是无单位的。对于除法操作符，特别是右操作对象必须是无单位的。这个表达式无效，因为两个操作对象都有单位（像素）。其中一个操作对象，5 或 50，必须是无单位的：

```css
calc(5px * 50px)
```

你需要在其中一个中省略单位。这两个备选方案都是有效的：

```css
calc(5 * 50px)
calc(5px * 50)
```

这是一个带有除法操作符的示例。由于两者都有单位，这将不是一个有效的表达式：

```css
calc(50% / 5%)
```

当你使用除法操作符时，应从右侧操作对象中移除单位：

```css
calc(50% / 5)
```

作为网页开发者，`calc()` 函数对你非常有帮助。通过这个函数，你可以动态设置属性值，从而创建灵活且响应式的设计。
