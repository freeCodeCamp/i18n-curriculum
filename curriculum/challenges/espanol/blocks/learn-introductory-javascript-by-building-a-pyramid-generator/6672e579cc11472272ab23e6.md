# --description--


JavaScript es el lenguaje de programación que impulsa la web. A diferencia de HTML y CSS que has aprendido anteriormente, JavaScript se usa más comúnmente para escribir lógica en lugar de etiquetado.

En este proyecto aprenderás los conceptos básicos de JavaScript y aplicarás esos conceptos a la construcción de un generador de pirámides.

Un generador de pirámides es un programa donde se puede establecer el tipo de carácter, el contador para la pirámide, y la dirección de la pirámide. El programa generará entonces una pirámide basada en esas entradas.

Haz clic sobre el botón "Check your code" para proceder al siguiente paso e iniciar el proyecto.
