# --explanation--

`By itself` означає, що щось відбувається автоматично, без потреби, щоб хтось це робив. Наприклад:  

`The layout updates by itself when the window size changes.` – Воно змінюється автоматично.

`Nicer` — це ступінь порівняння `nice`. Означає, що щось краще, комфортніше або приємніше за щось інше. Наприклад:  

`The new interface is nicer to use.` – Це більш приємно або зручніше для користувача.
