# --description--

Sarahs Antwort an Bob enthält einen bestimmten Ausdruck, der zeigt, dass sie das Problem, mit dem er konfrontiert ist, versteht.

# --questions--

## --text--

Welcher Teil von Sarahs Satz zeigt, dass sie das Problem versteht?

## --answers--

`I see`

---

`Let's open an issue`

### --feedback--

Während dieser Teil eine Lösung nahelegt, ist dies nicht der Teil, der direkt Verständnis zeigt.

---

`Then`

### --feedback--

Das Wort `then` gehört dazu, eine Lösung vorzuschlagen, zeigt aber nicht direkt Verständnis.

---

`An issue`

### --feedback--

Der Begriff `an issue` bezieht sich auf die Lösung, nicht auf das Ausdrücken des Verständnisses des Problems.
