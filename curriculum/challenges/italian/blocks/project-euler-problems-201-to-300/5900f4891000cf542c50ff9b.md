# --description--

Il numero di 3 cifre 376 nel sistema di numerazione decimale è un esempio di numeri con la proprietà speciale che il suo quadrato termina con le stesse cifre: ${376}^2 = 141376$. Chiamiamo quadrato stabile un numero con questa proprietà.

I quadrati stabili si possono osservare anche in altri sistemi di numerazione. Nel sistema di numerazione in base 14, il numero di 3 cifre $c37$ è anch’esso un quadrato stabile: $c37^2 = aa0c37$, e la somma delle sue cifre è $c+3+7=18$ nello stesso sistema di numerazione. Le lettere $a$, $b$, $c$ e $d$ sono usate rispettivamente per le cifre 10, 11, 12 e 13, in modo simile al sistema esadecimale.

Per $1 ≤ n ≤ 9$, la somma delle cifre di tutti i quadrati stabili di $n$ cifre nel sistema di numerazione in base 14 è $2d8$ (582 decimale). I quadrati stabili con zeri iniziali non sono ammessi.

Trova la somma delle cifre di tutti i quadrati stabili di $n$ cifre nel sistema di numerazione in base 14 per $1 ≤ n ≤ 10000$ (decimale) e fornisci la tua risposta come stringa nel sistema in base 14 usando lettere minuscole dove necessario.
