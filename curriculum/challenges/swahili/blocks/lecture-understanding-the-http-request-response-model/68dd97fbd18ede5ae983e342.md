# --description--

Katika funzo la mwisho, ulijifunza kwamba seva inashughulikia ombi la HTTP na kurudisha jibu. Jibu hilo linajumuisha mali mbalimbali unazoziona kwenye mtandao.

Mali zinazorejeshwa na jibu la HTTP ni pamoja na mali za maandishi, mali za vyombo vya habari, fonti, na kadhalika. Kila moja ya mali hizi huambatana na kichwa cha **Content-Type**, ambacho hueleza kivinjari jinsi ya kushughulikia na kuonyesha mali hiyo kwa usahihi.

Mali za maandishi zimekuwa sehemu muhimu ya mtandao tangu mwanzo. Kwa kweli, mtandao ulianza kama maandishi safi. Siku hizi, mali za maandishi ni pamoja na:

* **HTML (Hypertext Markup Language)**: Muundo wa msingi wa ukurasa wowote wa wavuti. Bila HTML, ukurasa wa wavuti hauwezi kuwepo.
    
* **CSS (Cascading Stylesheet)**: Mali inayodhibiti muonekano wa ukurasa wa wavuti. Kwa CSS, unaweka mitindo kama rangi, fonti, na mpangilio kwa HTML.
    
* **JavaScript**: JavaScript huongeza mwingiliano na maudhui yanayobadilika kwa HTML na CSS.
    
* **JSON na XML**: Miundo hii miwili huruhusu kuhamisha data iliyopangwa kati ya seva na vivinjari.
    
Mali za vyombo vya habari ni pamoja na picha, sauti, na video. Zinachangia kuleta mtandao kuwa hai kwa picha na sauti.

Tovuti nyingi hutegemea picha kuwasilisha taarifa za kuona. Picha huja katika miundo tofauti kama `JPG`, `JPEG`, `PNG`, `WEBP`, na `GIF`.

Tovuti kama YouTube na Vimeo hutumia maudhui ya video katika miundo kama `MP4`, `MOV`, `AVI`, `WebM`, na mingine.

Sauti ni sehemu muhimu ya majukwaa mengi kwani huruhusu watumiaji kusikiliza muziki, podikasti, na maudhui mengine ya sauti. Miundo ya kawaida ya sauti ni `MP3`, `AAC`, Ogg, na `WAV`. Majukwaa kama Spotify na YouTube Music hutegemea miundo hii kutoa uzoefu wa sauti bora.

Mali nyingine zinazorejeshwa katika jibu la HTTP ni nyaraka kama `PDF`, `DOCX`, na `PAGES`, mafaili yaliyoshinikwa kama `ZIP` na `RAR`, mafaili ya fonti kama `WOFF`, `WOFF2`, na `TTF`, na mafaili yanayotekelezwa kama `EXE`.

Hapa kuna jedwali la mali hizi za kawaida na aina zao za maudhui:

| Aina ya mali | Kichwa cha Content-Type |
| --- | --- |
| HTML | text/html |
| CSS | text/css |
| JavaScript | application/javascript |
| JPEG | image/jpeg |
| PNG | image/png |
| GIF | image/gif |
| SVG | image/svg+xml |
| WebP | image/webp |
| WOFF | font/woff |
| WOFF2 | font/woff2 |
| TTF | font/ttf |
| OTF | font/otf |
| MP4 | video/mp4 |
| WebM | video/webm |
| Ogg (Video) | video/ogg |
| MP3 | audio/mpeg |
| AAC | audio/aac |
| WAV | audio/wav |
| Ogg (Audio) | audio/ogg |
| JSON | application/json |
| XML | application/xml |
| PDF | application/pdf |
| ZIP | application/zip |
