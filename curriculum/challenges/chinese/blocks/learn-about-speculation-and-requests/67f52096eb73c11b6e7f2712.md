### --feedback--

# --explanation--

`Physical access` 是指进入某个地方，例如办公室或服务器室。

当有人未经许可进入安全区域时，就会发生 `physical access breach`。例如：

- `To avoid a physical access breach, we should require ID checks at the entrance.` - 公司可以通过在门口查看身份证来防止未经授权的进入。

- `A physical access breach can happen if someone enters the server room without permission.` - 如果有人未经授权进入服务器机房，可能会发生物理安全性问题。
