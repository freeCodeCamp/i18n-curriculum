# --interactive--

الهامش وline-height ضروريان لتباعد عناصر القائمة لتحسين قابلية القراءة والمظهر البصري.

أولًا، لنبدأ بتباعد عناصر القائمة باستخدام الهوامش!

يمكن استخدام الهوامش لإنشاء مساحة بين عناصر القائمة عبر تطبيق خاصيات الهامش على عناصر `li`. تتيح لك هذه الطريقة التحكم في التباعد خارج كل عنصر قائمة، مما يزيد أو يقلل الفجوة بينها بفعالية.

لنلقِ نظرة على مثال لقائمة غير مرتبة تحتوي على ثلاثة عناصر.

:::interactive_editor

```html
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

افتراضيًا، لن يطبق HTML هذا القدر الكبير من التباعد بين عناصر القائمة.

لتطبيق بعض التباعد أسفل كل عنصر قائمة، يمكنك استخدام الخاصية `margin-bottom` بهذا الشكل:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  margin-bottom: 40px;
}
```

:::

في هذا المثال، سيتم تطبيق هامش `40px` أسفل كل عنصر قائمة داخل القائمة غير المرتبة.

طريقة أخرى لتباعد عناصر القائمة هي استخدام خاصية `line-height`.

تضبط خاصية `line-height` التباعد الرأسي بين أسطر النص داخل عنصر قائمة واحد.

بينما تؤثر أساسًا على التباعد بين أسطر النص داخل كل عنصر، يمكنها أيضًا التأثير بشكل غير مباشر على التباعد الكلي بين عناصر القائمة إذا كانت العناصر تحتوي على سطر نص واحد فقط.

إذا كانت عناصر القائمة تحتوي على عدة أسطر نص، فإن `line-height` تؤثر على التباعد بين تلك الأسطر، لكنها لا تضبط التباعد مباشرة بين عناصر القائمة المنفصلة نفسها.

للتحكم في التباعد بين عناصر القائمة الفردية، تستخدم خاصيات `margin` أو `padding` بدلاً من ذلك.

باستخدام نفس القائمة غير المرتبة من قبل، إليك مثالًا لتطبيق `line-height` على عناصر القائمة:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  line-height: 2; 
}
```

:::

في هذا المثال، تحدد `line-height: 2;` ارتفاع السطر ليكون ضعف حجم الخط، مما يخلق مساحة رأسية أكبر داخل كل عنصر قائمة.
