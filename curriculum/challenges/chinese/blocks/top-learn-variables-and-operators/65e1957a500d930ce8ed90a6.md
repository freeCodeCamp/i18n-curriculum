# --description--

要声明一个常量（不变的）变量，使用 `const` 替代 `let`：

```javascript
const myBirthday = '16-09-2003';
```

使用 `const` 声明的变量称为“常量”。它们不能被重新赋值。尝试这样做会导致误差：

```javascript
const myBirthday = '16-09-2003';

myBirthday = '01-01-2001'; // error, can't reassign the constant!
```

当程序员确定一个变量永远不会改变时，他们可以用 `const` 来声明它，以保证并向所有人传达这一事实。
