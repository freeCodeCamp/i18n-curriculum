# --description--

<dfn>Opacidad</dfn> describe cuán opaco, o no transparente, es algo. Por ejemplo, una pared sólida es opaca y no puede pasar luz a través de ella. Pero un vaso de vidrio es mucho más transparente, y puedes ver a través del vidrio hacia el otro lado.

Con la propiedad de CSS `opacity`, puedes controlar cuán opaco o transparente es un elemento. Con el valor `0`, o 0%, el elemento será completamente transparente, y a `1.0`, o 100%, el elemento será completamente opaco como lo es por defecto.

En la regla CSS `.sleeve`, establece la propiedad `opacity` a `0.5`.

# --hints--

La regla CSS `.sleeve` debe tener la propiedad `opacity` establecida en `0.5`.

```js
assert.strictEqual(new __helpers.CSSHelp(document).getStyle('.sleeve')?.opacity, '0.5');
```
