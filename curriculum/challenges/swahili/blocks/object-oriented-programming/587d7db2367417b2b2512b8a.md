# --description--

Katika changamoto iliyopita, `bird` alikuwa na sifa ya umma `name`. Inachukuliwa kuwa ya umma kwa sababu inaweza kufikiliwa na kubadilishwa kutoka nje ya ufafanuzi wa `bird`.

```js
bird.name = "Duffy";
```

Kwa hiyo, sehemu yoyote ya msimbo wako inaweza kubadilisha kwa urahisi jina la `bird` kuwa thamani yoyote. Fikiria mambo kama nywila na akaunti za benki kubadilishwa kwa urahisi na sehemu yoyote ya msimbo wako. Hilo linaweza kusababisha matatizo mengi.

Njia rahisi ya kufanya sifa hii ya umma kuwa ya faragha ni kwa kuunda kigezo ndani ya kitendakazi cha kijenzi. Hii hubadilisha wigo wa kigezo hicho kuwa ndani ya kitendakazi cha kijenzi badala ya kupatikana kwa kiwango cha ulimwengu. Kwa njia hii, kigezo kinaweza kufikiliwa na kubadilishwa tu na njia pia zilizo ndani ya kitendakazi cha kijenzi.

```js
function Bird() {
  let hatchedEgg = 10;

  this.getHatchedEggCount = function() { 
    return hatchedEgg;
  };
}
let ducky = new Bird();
ducky.getHatchedEggCount();
```

Hapa `getHatchedEggCount` ni njia ya kipekee, kwa sababu ina upatikanaji wa kigezo cha faragha `hatchedEgg`. Hii inawezekana kwa sababu `hatchedEgg` imetangazwa katika muktadha uleule na `getHatchedEggCount`. Katika JavaScript, kitendakazi kila mara kina upatikanaji wa muktadha ambamo kiliumbwa. Hii huitwa `closure`.
