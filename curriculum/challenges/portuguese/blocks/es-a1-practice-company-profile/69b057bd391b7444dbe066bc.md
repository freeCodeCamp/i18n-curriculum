# --explanation--

Esta tarefa reforça várias estruturas importantes:

`¿Dónde...?` é usado para localização. Por exemplo:

- `¿Dónde vives?` - Onde você mora?
- `¿Dónde está la oficina?` - Onde fica o escritório?

`¿Cómo...?` é usado para perguntar o nome ou identificação. Por exemplo:

- `¿Cómo te llamas?` - Qual é o seu nome?
- `¿Cómo se llama el proyecto?` - Qual é o nome do projeto?

`Trabajas` é segunda pessoa (`tú`), enquanto `trabajo` é primeira pessoa (`yo`).

`Tiene` é terceira pessoa do singular porque se refere a `la empresa`. Por exemplo:

- `La empresa tiene diez empleados`. - A empresa tem dez funcionários.
- `El equipo tiene un nuevo proyecto`. - A equipe tem um novo projeto.
