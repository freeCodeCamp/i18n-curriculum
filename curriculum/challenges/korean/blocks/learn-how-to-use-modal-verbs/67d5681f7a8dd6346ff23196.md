# --explanation--

기억하세요:

`Must`는 강한 필요성이나 의무를 표현합니다. 예를 들어:

`You must wear a seatbelt.`

`Have to`도 의무를 표현하지만 must보다 덜 엄격할 때가 있습니다. 예를 들어:

`You have to submit the report by Friday.`

`Should`는 권고나 조언에 사용됩니다. 예를 들어:

`You should review your notes before the test.`
