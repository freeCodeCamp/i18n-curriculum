# --description--

Acha $N(i)$ iwe nambari ndogo kabisa $n$ ambayo $n!$ inagawanyika na $(i!)^{1234567890}$

Acha $S(u) = \sum N(i)$ kwa $10 ≤ i ≤ u$.

$S(1000)=614\\,538\\,266\\,565\\,663$.

Tafuta $S(1\\,000\\,000)\bmod {10}^{18}$.
