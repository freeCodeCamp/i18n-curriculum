# --description--

Sowohl Chrome als auch Firefox verfügen über ausgezeichnete JavaScript-Konsolen, auch bekannt als Entwicklerwerkzeuge, zum Debuggen Ihres JavaScripts.

Sie finden die Entwicklerwerkzeuge im Menü von Chrome oder die Web-Konsole im Menü von Firefox. Wenn Sie einen anderen Browser oder ein Mobiltelefon verwenden, empfehlen wir dringend, auf die Desktop-Version von Firefox oder Chrome umzusteigen.

Die `console.log()`-Methode, die die Ausgabe dessen, was sich in ihren runden Klammern befindet, in der Konsole „ausgibt“, ist wahrscheinlich das hilfreichste Debugging-Werkzeug. Wenn Sie sie an strategischen Stellen in Ihrem Code platzieren, können Sie die Zwischenwerte von Variablen sehen. Es ist gute Praxis, eine Vorstellung davon zu haben, wie die Ausgabe aussehen sollte, bevor Sie sich ansehen, wie sie tatsächlich ist. Kontrollpunkte, um den Status Ihrer Berechnungen im gesamten Code zu überprüfen, helfen dabei, den Fehler einzugrenzen.

Hier ist ein Beispiel, um den String `Hello world!` in der Konsole auszugeben:

```js
console.log('Hello world!');
```

# --instructions--

Verwenden Sie die `console.log()`-Methode, um den Wert der Variablen `a` an der im Code angegebenen Stelle auszugeben.
