# --interactive--

ウェブページでスムーズなアニメーションを作るのは、タイミングの扱い方がわからないと難しいことがあります。嬉しいことに、`requestAnimationFrame()` APIを使えば簡単になります。

`requestAnimationFrame()` は、次の画面の再描画前にアニメーションの次のステップをスケジュールできるメソッドで、滑らかで見た目にも美しい体験を実現します。

次の画面の再描画とは、ブラウザがウェブページの表示を更新する瞬間のことです。これは1秒間に複数回、通常はほとんどのディスプレイで60回（1秒あたり60フレーム）ほど行われます。

`requestAnimationFrame()` メソッドを使うには、呼び出してコールバック関数を渡すだけです。

```js
requestAnimationFrame(callback);
```

`requestAnimationFrame()` の呼び出しは、アニメーションを扱う関数の中、例えば`animate()`の中で行い、アニメーションを更新する関数（通常は`update()`と呼ばれます）も用意します。

```js
function animate() {
 // Update the animation...
 // for example, move an element, change a style, and more.
 update();
 // Request the next frame
 requestAnimationFrame(animate);
}
```

`update()` 関数の中で魔法が起こります。ここでアニメーションしたいものを自由に変更できます。例えば、スタイルを更新したり、要素の位置を変えたりします。

```js
function update() {
 element.style.transform = `translateX(${position}px)`;
 position += 2;
}
```

アニメーションを最終的に開始するのは、`requestAnimationFrame()` を呼び出して`animate` 関数を渡すことです。これは`animate` 関数の外側で行います。

```js
requestAnimationFrame(animate);
```

このループは停止するまで続きます。

では、別の例を見てみましょう。この例のHTMLは`div` 要素で、テキストは`freeCodeCamp is Awesome`です。CSSは`div` を長方形にし、左または右のビューポート外に出たものを隠します。JavaScriptコードは、`2px` の呼び出しごとに`requestAnimationFrame(animate)` の長方形を右に動かし、`animate` 関数をコールバックとして使っています。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div id="rect" class="rect">freeCodeCamp is Awesome</div>
<script src="index.js"></script>
```

```css
body {
  overflow-x: hidden;
}

.rect {
  display: flex;
  align-items: center;
  justify-content: center;
  width: 400px;
  height: 250px;
  border-radius: 5px;
  background-color: #1b1b32;
  color: #f5f6f7;
  font-size: 2rem;
  position: absolute;
}
```

```js
const rect = document.getElementById("rect");

let position = 0;

function update() {
  // Move the rectangle 2px to the right
  rect.style.left = position + "px";
  position += 2;

  if (position > window.innerWidth) {
    // Move the rectangle just outside the left side of the screen
    position = -rect.offsetWidth;
  }
}

function animate() {
  update();

  //request the next frame
  requestAnimationFrame(animate);
}

// Start the animation
requestAnimationFrame(animate);
```

:::

ブラウザでの結果は、画面を横切って浮かぶアニメーションタイトルカードになります。
