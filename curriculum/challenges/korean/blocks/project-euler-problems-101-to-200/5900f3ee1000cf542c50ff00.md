# --description--

모두 1로만 이루어진 수를 반복 단위수라고 합니다. $R(k)$를 길이가 $k$인 반복 단위수로 정의하겠습니다; 예를 들어, $R(6) = 111111$입니다.

$n$이 양의 정수이고 $GCD(n, 10) = 1$일 때, 항상 $R(k)$가 $n$으로 나누어지는 $k$ 값이 존재함을 보일 수 있으며, $A(n)$을 그러한 $k$ 중 가장 작은 값으로 정의합니다; 예를 들어, $A(7) = 6$이고 $A(41) = 5$입니다.

모든 소수 $p > 5$에 대해 $p − 1$이 $A(p)$로 나누어짐이 주어졌습니다. 예를 들어, $p = 41$일 때, $A(41) = 5$이고 40은 5로 나누어집니다.

하지만 드물게 이 조건을 만족하는 합성수도 존재하는데, 처음 다섯 예는 91, 259, 451, 481, 그리고 703입니다.

$GCD(n, 10) = 1$이고 $n − 1$이 $A(n)$으로 나누어지는 합성수 $n$ 중 처음 스물다섯 개의 합을 구하세요.
