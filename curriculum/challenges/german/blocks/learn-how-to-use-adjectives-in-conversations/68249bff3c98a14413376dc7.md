# --explanation--

`Put ... into ...` ist eine gebräuchliche Wendung, die beschreibt, etwas in eine bestimmte Form, an einen bestimmten Ort oder in einen Container zu bringen. `Into` zeigt die Richtung oder Platzierung innerhalb von etwas an. Zum Beispiel:

`She put the data into a chart.` – Das bedeutet, sie hat die Informationen in Diagrammform organisiert.

`Should` ist ein Modalverb, das verwendet wird, um eine Empfehlung, Möglichkeit oder Erwartung auszudrücken. Zum Beispiel:

`We should test the app before launching it.` – Es ist eine gute Idee oder eine erwartete Handlung.
