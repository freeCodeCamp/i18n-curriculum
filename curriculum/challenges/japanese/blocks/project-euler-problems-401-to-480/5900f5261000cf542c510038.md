# --description--

整数 $M$ に対して、$R(M)$ を次の条件をすべて満たす整数の組 $(p, q)$ に対して $\frac{1}{p·q}$ を足し合わせたものと定義します。

- $1 ≤ p < q ≤ M$
- $p + q ≥ M$
- $p$ と $q$ は互いに素である。

また、$S(N)$ を $2 ≤ i ≤ N$ の範囲で $R(i)$ を足し合わせたものと定義します。

$S(2) = R(2) = \frac{1}{2}$、$S(10) ≈ 6.9147$、$S(100) ≈ 58.2962$ であることが確認できます。

$S({10}^7)$ を求めてください。答えは小数点以下4桁に丸めてください。
