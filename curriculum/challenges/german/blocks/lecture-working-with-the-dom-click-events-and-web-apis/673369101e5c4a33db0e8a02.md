# --interactive--

In der vorherigen Lektion haben Sie gelernt, wie Sie mit `"click"` und `"input"` Ereignissen arbeiten, indem Sie die `addEventListener()` Methode verwenden. In dieser Lektion lernen Sie, wie Sie Ereignis-Listener mit der `removeEventListener()` Methode entfernen.

Die `removeEventListener()` Methode wird verwendet, um einen Ereignis-Listener zu entfernen, der zuvor mit der `addEventListener()` Methode zu einem Element hinzugefügt wurde. Das ist nützlich, wenn Sie aufhören möchten, auf ein bestimmtes Ereignis an einem Element zu hören.

Hier ist die grundlegende Syntax für die `removeEventListener()` Methode:

```js
element.removeEventListener("event", listener);
```

Genau wie die `addEventListener()` Methode nimmt die `removeEventListener()` Methode zwei Argumente entgegen: das Ereignis, das Sie entfernen möchten, und die Listener-Funktion, die zuvor hinzugefügt wurde.

Es gibt auch ein zusätzliches optionales drittes Argument, das an die `removeEventListener()` Methode übergeben werden kann. Dieses Argument kann entweder das `options` oder `useCapture` sein.

Das `options` Argument ist ein Objekt, das die Optionen für den Ereignis-Listener angibt, zum Beispiel, ob der Listener passiv oder nur einmalig sein soll.

Das `useCapture` Argument ist ein Boolean-Wert, der angibt, ob das Ereignis während der Ereignis-Ausbreitungsphase erfasst werden soll.

Meistens müssen Sie nur die Argumente für das Ereignis und den Listener an die `removeEventListener()` Methode übergeben.

Wenn wir die Hintergrundfarbe zwischen Grau und Blau wechseln möchten, können wir dafür einen Ereignis-Listener verwenden.

Zuerst müssen wir auf das `button` Element sowie auf das `body` Element zugreifen:

```js
const bodyEl = document.querySelector("body");
const btn = document.getElementById("btn");
```

Dann müssen wir die Funktion erstellen, die für das Umschalten zwischen den grauen und blauen Farben verantwortlich ist:

```js
let isBgColorGrey = true;

function toggleBgColor() {
  bodyEl.style.backgroundColor = isBgColorGrey ? "blue" : "grey";
  isBgColorGrey = !isBgColorGrey;
}
```

Wir verwenden eine Boolean-Variable `isBgColorGrey`, um die aktuelle Hintergrundfarbe zu verfolgen. Wenn die Hintergrundfarbe grau ist, ändern wir sie zu blau und umgekehrt.

Dann müssen wir dem `button` Element einen Ereignis-Listener hinzufügen, der die `toggleBgColor` Funktion aufruft, wenn der Button geklickt wird:

```js
btn.addEventListener("click", toggleBgColor);
```

Jedes Mal, wenn der Button geklickt wird, ändert sich die Hintergrundfarbe des Bodys zwischen Grau und Blau.

Wir können das HTML aktualisieren, um ein Absatz-Element einzufügen, das verwendet wird, um den Ereignis-Listener zu entfernen, wenn man mit der Maus darüber fährt:

```html
<link rel="stylesheet" href="styles.css" />
<h1>Event Listener examples</h1>
<p id="para">MouseOver this text to remove the event listener</p>
<button id="btn">Change background color</button>
<script src="index.js"></script>
```

Um den Ereignis-Listener zu entfernen, wenn der Absatz mit der Maus berührt wird, können wir dem Absatz-Element einen Ereignis-Listener hinzufügen. Wir verwenden das `mouseover` Ereignis und übergeben eine Funktion, die den Ereignis-Listener vom `button` Element entfernt:

```js
const para = document.getElementById("para");
const btn = document.getElementById("btn");

para.addEventListener("mouseover", () => {
  btn.removeEventListener("click", toggleBgColor);
});
```

Wenn der Absatz mit der Maus berührt wird, wird der Ereignis-Listener für das Button-Klick-Ereignis entfernt, und die Hintergrundfarbe ändert sich nicht mehr, wenn der Button geklickt wird.
