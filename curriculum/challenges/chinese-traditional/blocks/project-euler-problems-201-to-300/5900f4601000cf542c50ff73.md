# --description--

分子小於分母的正分數稱為真分數。

對於任何分母 `d`，將會有 `d−1` 個真分數；例如，當 `d = 12` 時：

$$\frac{1}{12}、\frac{2}{12}、\frac{3}{12}、\frac{4}{12}、\frac{5}{12}、\frac{6}{12}、\frac{7}{12}、\frac{8}{12}、\frac{9}{12}、\frac{10}{12}、\frac{11}{12}$$

我們將稱無法約分的分數為「彈性分數」。

此外，我們將定義分母的韌性 $R(d)$ 為其韌性真分數的比例；例如，$R(12) = \frac{4}{11}$。

事實上，$d = 12$ 是最小的分母，其韌性 $R(d) &lt; \frac{4}{10}$。

找到最小的分母 $d$，其韌性 $R(d) &lt; \frac{15\\,499}{94\\,744}$。
