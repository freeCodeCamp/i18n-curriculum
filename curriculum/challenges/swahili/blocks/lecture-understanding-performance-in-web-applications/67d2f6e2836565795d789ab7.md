# --description--

Je, umewahi kujiuliza nini hutokea nyuma ya pazia unapoingiza URL na kubonyeza enter? Kivinjari hubadilisha vipi msimbo kuwa kurasa za mtandao zinazoshirikisha tunazoziona?

Kuelewa jinsi kivinjari kinaonyesha ukurasa wa mtandao ni muhimu kwa wasanidi programu wanaolenga kuboresha utendaji na kuongeza uzoefu wa mtumizi.

Katika funzo hili, tutaelezea mchakato wa kuonyesha ukurasa, tukigawanya katika hatua wazi na rahisi kuelewa.

Kwanza kivinjari huchanganua HTML na kujenga DOM.

Mchakato huanza kwa kivinjari kupata maudhui ya HTML ya ukurasa. Kivinjari huchanganua HTML hii kujenga Mfano wa Kitu cha Hati, au DOM — muundo wa mti unaowakilisha maudhui ya ukurasa.

Fikiria kipande hiki rahisi cha HTML:

```html
<html>
  <body>
    <h1>Hello, World!</h1>
    <p>Welcome to our website.</p>
  </body>
</html>
```

Kivinjari huchanganua hii kuunda mti wa DOM wenye `html` kama mzizi, unaojumuisha `body`, ambayo kwa upande wake ina vipengele vya `h1` na `p`.

Ifuatayo, kivinjari huchakata CSS, kujenga Mfano wa Kitu cha CSS, au CSSOM. Huu ni muundo mwingine wa mti unaoelekeza jinsi vipengele vinavyopaswa kupambwa.

Kwa CSS hii:

```css
h1 {
  color: blue;
}
p {
  font-size: 16px;
}
```

Mfano wa Kitu cha CSS unaeleza kuwa vipengele vya `h1` ni buluu na vipengele vya `p` vina ukubwa wa fonti wa pikseli 16.

Kivinjari kisha huunganisha DOM na Mfano wa Kitu cha CSS kuunda mti wa kuonyesha. Mti huu unajumuisha nodi tu zinazohitajika kuonyesha ukurasa, kila moja ikiwa na mitindo yake inayolingana.

Kwa mfano wetu, mti wa kuonyesha unajumuisha vipengele vya `h1` na `p`, vilivyopambwa ipasavyo.

Kwa mti wa kuonyesha mkononi, kivinjari huhesabu nafasi halisi na ukubwa wa kila kipengele katika mchakato unaoitwa upangaji au reflow. Hii inahakikisha vipengele vinaonekana mahali sahihi na kwa vipimo sahihi kwenye skrini.

Mwishowe, kivinjari huchoraza pikseli kwenye skrini, kuonyesha kila kipengele kulingana na mitindo na upangaji uliohesabiwa. Katika kurasa ngumu, hii inaweza kuhusisha tabaka nyingi zinazochanganywa pamoja kuunda matokeo ya mwisho ya kuona.

Kwa kumalizia, kuanzia kuchanganua HTML na CSS hadi kuchora pikseli kwenye skrini, mchakato wa kuonyesha wa kivinjari ni mfululizo wa hatua tata.
