# --description--

مجموع مربعات أول عشرة أعداد طبيعية هو،

<div style='text-align: center;'>1<sup>2</sup> + 2<sup>2</sup> + ... + 10<sup>2</sup> = 385</div>

مربع مجموع أول عشرة أعداد طبيعية هو،

<div style='text-align: center;'>(1 + 2 + ... + 10)<sup>2</sup> = 55<sup>2</sup> = 3025</div>

لذا فإن الفرق بين مجموع مربعات أول عشرة أعداد طبيعية ومربع المجموع هو 3025 − 385 = 2640.

احسب الفرق بين مجموع مربعات أول `n` عدد طبيعي ومربع المجموع.
