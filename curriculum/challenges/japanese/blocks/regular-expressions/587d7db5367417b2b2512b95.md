# --description--

リテラルパターン（`/literal/`）とワイルドカード文字（`/./`）の一致方法を学びました。これらは正規表現の両極端であり、一方は完全一致を見つけ、もう一方はすべてに一致します。その中間のバランスを取るオプションもあります。

<dfn>文字クラス</dfn>を使うと、リテラルパターンをある程度柔軟に検索できます。文字クラスは、マッチさせたい文字のグループを角括弧（`[` と `]`）の中に入れて定義します。

例えば、`bag`、`big`、`bug`にはマッチさせたいが、`bog`にはマッチさせたくない場合、正規表現 `/b[aiu]g/` を作成できます。`[aiu]` は、`a`、`i`、または `u` の文字だけにマッチする文字クラスです。

```js
let bigStr = "big";
let bagStr = "bag";
let bugStr = "bug";
let bogStr = "bog";
let bgRegex = /b[aiu]g/;
bigStr.match(bgRegex);
bagStr.match(bgRegex);
bugStr.match(bgRegex);
bogStr.match(bgRegex);
```

順に、4つの `match` の呼び出しは `["big"]`、`["bag"]`、`["bug"]`、`null` の値を返します。

# --instructions--

正規表現 `a` に母音（`e`、`i`、`o`、`u`、`vowelRegex`）を含む文字クラスを使って、文字列 `quoteSample` の中のすべての母音を見つけてください。

**注意:** 大文字と小文字の両方の母音にマッチさせることを忘れないでください。
