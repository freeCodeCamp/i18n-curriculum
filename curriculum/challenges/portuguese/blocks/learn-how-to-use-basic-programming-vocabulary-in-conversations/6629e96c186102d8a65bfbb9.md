# --description--

`There is` e `there are` são usados para indicar a existência ou presença de algo. `There is` é usado com substantivos no singular ou incontáveis, enquanto `there are` é usado com substantivos no plural.

Exemplos:

- `There is a book on the table.` (Substantivo singular)

- `There is some water in the glass.` (substantivo incontável)

- `There are three apples in the basket.` (Substantivo no plural)
