# --description--

Бібліотека D3 орієнтована на підхід на основі даних. Коли у вас є множина даних, ви можете застосувати методи D3, щоб відобразити їх на сторінці. Дані бувають у різних форматах, але в цьому завданні використовується простий масив чисел.

Перший крок — зробити так, щоб D3 «побачив» дані. Метод `data()` застосовується до вибраних DOM-вузлів, щоб приєднати до них дані. Множина даних передається як аргумент цьому методу.

Загальний шаблон потоку роботи — створити новий елемент у документі для кожного елемента даних у множині. Для цього в D3 є метод `enter()`.

Коли `enter()` поєднується з методом `data()`, він порівнює вибрані елементи на сторінці з кількістю елементів даних у множині. Якщо елементів менше, ніж даних, він створює відсутні елементи.

Ось приклад, який вибирає елемент `ul` і створює новий елемент списку на основі кількості записів у масиві:

```html
<body>
  <ul></ul>
  <script>
    const dataset = ['a', 'b', 'c'];
    d3.select('ul')
      .selectAll('li')
      .data(dataset)
      .enter()
      .append('li')
      .text('New item');
  </script>
</body>
```

Може здатися дивним вибирати елементи, яких ще немає. Цей код каже D3 спочатку вибрати `ul` на сторінці. Потім вибрати всі елементи списку, що повертає порожній вибір. Далі метод `data()` переглядає множину даних і виконує наведений код тричі, по одному разу для кожного елемента масиву. Метод `enter()` бачить, що на сторінці немає елементів `li`, але потрібно 3 (по одному для кожного елемента в `dataset`). Нові елементи `li` додаються до `ul` і мають текст `New item`.
