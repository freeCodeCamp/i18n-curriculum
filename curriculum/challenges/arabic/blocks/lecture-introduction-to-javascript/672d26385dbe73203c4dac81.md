# --interactive--

JavaScript هي لغة برمجة قوية تضيف التفاعل والسلوك الديناميكي إلى مواقع الويب.

بينما تُستخدم HTML و CSS لبناء هيكل المحتوى وتنسيق عناصر الصفحة، يتجاوز JavaScript ذلك بتمكين وظائف أكثر تعقيدًا، مثل التعامل مع مُدخَلات المستخدم، وتحريك العناصر، وحتى بناء تطبيقات ويب كاملة.

على سبيل المثال، عندما تنقر زرًا، أو ترسل نموذجًا، أو تمرر المؤشر فوق قائمة، يحدد JavaScript كيف تتصرف الصفحة.

إليك مثالًا على كيفية عمل هذه الثلاثة معًا:

:::interactive_editor

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: green;
    }
  </style>
</head>
<body>
  <h1>Hello, World!</h1>
  <button onclick="alert('Button clicked!')">Click me</button>
</body>
</html>
```

:::

في هذا المثال، تُستخدم HTML لتعريف المحتوى: عنوان (عنصر `h1`) وزر (عنصر `button`).

تُستخدم CSS لتطبيق أنماط على العنوان، مثل جعل النص أخضر. تُستخدم JavaScript لعرض رسالة تنبيه عند نقر الزر.

لتلخيص الأمر، توفر HTML الهيكل، وتضيف CSS التنسيق، وتمكّن JavaScript السلوك، مما يخلق تجربة ويب كاملة وتفاعلية.
