**目的:** 次のアプリと機能的に似たアプリを作成してください：<a href="https://codepen.io/freeCodeCamp/full/bELRjV" target="_blank" rel="noopener noreferrer nofollow">https://codepen.io/freeCodeCamp/full/bELRjV</a>。

**ルール #1:** 例のプロジェクトのコードは見ないでください。自分で考えて作成してください。

**ルール #2:** 以下のユーザーストーリーを満たし、すべてのテストに合格してください。必要なライブラリやAPIは自由に使ってください。自分の個性を出して構いません。

**ユーザーストーリー:** 現在の場所の天気が見られます。

**ユーザーストーリー:** 天気に応じて異なるアイコンや背景画像（例：雪山、暑い砂漠）が表示されます。

**ユーザーストーリー:** 華氏と摂氏を切り替えるボタンを押せます。

**注意:** 多くのインターネットブラウザは、HTML5 Geolocationを使ってユーザーの位置情報を取得する際にHTTP Secure (`https://`) 接続を必要とします。このため、HTML5 Geolocationでユーザーの位置を取得し、その後にHTTP Secure接続を使うfreeCodeCampのWeather API <a href="https://weather-proxy.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://weather-proxy.freecodecamp.rocks/</a> を使うことをお勧めします。また、<a href="https://codepen.io" target="_blank" rel="noopener noreferrer nofollow">CodePen.io</a> には`https://` 経由で接続してください。
