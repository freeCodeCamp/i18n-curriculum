# --description--

`push()` と `unshift()` には、それぞれほぼ反対の動作をするメソッドがあります：`pop()` と `shift()` です。もうお分かりのように、追加する代わりに、`pop()` は配列の末尾から要素を*削除*し、`shift()` は配列の先頭から要素を削除します。`pop()` と `shift()`、およびそれらの対応する `push()` と `unshift()` の主な違いは、どちらのメソッドもパラメータを取らず、一度に配列を1つの要素だけ変更できることです。

見てみましょう：

```js
let greetings = ['whats up?', 'hello', 'see ya!'];

greetings.pop();
```

`greetings` は `['whats up?', 'hello']` の値になります。

```js
greetings.shift();
```

`greetings` は `['hello']` の値になります。

また、どちらのメソッドでも削除された要素の値を次のように返すことができます：

```js
let popped = greetings.pop();
```

`greetings` は `[]` の値になり、`popped` は `hello` の値になります。

# --instructions--

`popShift` という関数が定義されており、配列を引数に取り、新しい配列を返します。`pop()` と `shift()` を使って、引数の配列の最初と最後の要素を削除し、削除した要素を対応する変数に代入するように関数を修正してください。返される配列にはそれらの値が含まれるようにしてください。
