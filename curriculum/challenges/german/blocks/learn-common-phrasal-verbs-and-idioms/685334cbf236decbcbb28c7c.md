# --explanation--

Ein `draft` ist eine frühe Version eines Dokuments oder Plans, die bearbeitet oder überprüft werden kann. Zum Beispiel:

`I wrote a draft of the email before sending the final version.` – Das bedeutet, die Person hat eine frühe Version der E-Mail geschrieben.

`To run something by someone` wird verwendet, um jemandem etwas zur Rückmeldung oder Genehmigung zu zeigen oder zu erklären. Zum Beispiel:

`Can I run this idea by you before we present it?` – Das bedeutet, Sie bitten jemanden um seine Meinung dazu.

`To roll something out` bedeutet, etwas offiziell einer größeren Gruppe vorzustellen oder einzuführen. Zum Beispiel:

`They will roll out the new software update next week.` – Das bedeutet, die neue Version der Software wird nächste Woche veröffentlicht.

Das Suffix `-wide` bedeutet „über das Ganze“ von etwas. In `company-wide` bedeutet es etwas, das für das gesamte Unternehmen gilt. Ein weiteres Beispiel:

`There was a system-wide outage yesterday.` – Das bedeutet, das Problem betraf das gesamte System.
