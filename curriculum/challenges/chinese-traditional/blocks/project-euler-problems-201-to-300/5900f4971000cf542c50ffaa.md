# --description--

選取了四個具有整數座標的點：

$A(a, 0)$、$B(b, 0)$、$C(0, c)$ 和 $D(0, d)$，其中 $0 < a < b$ 且 $0 < c < d$。

點 $P$，同樣具有整數座標，被選在直線 $AC$ 上，使得三個三角形 $ABP$、$CDP$ 和 $BDP$ 都相似。

<img alt="points A, B, C, D and P creating three triangles: ABP, CDP, and BDP" src="https://cdn.freecodecamp.org/curriculum/project-euler/three-similar-triangles.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

很容易證明三角形只有在 $a = c$ 時才會相似。

所以，鑑於 $a = c$，我們正在尋找三元組 ($a$、$b$、$d$)，使得至少存在一個點 $P$（具有整數座標）位於 $AC$ 上，令三個三角形 $ABP$、$CDP$ 和 $BDP$ 全部相似。

例如，如果 $(a, b, d) = (2, 3, 4)$，可以輕易驗證點 $P(1, 1)$ 滿足上述條件。請注意，三元組 (2,3,4) 和 (2,4,3) 被視為不同，儘管點 $P(1, 1)$ 對兩者皆相同。

如果 $b + d &lt; 100$，則存在 92 組不同的三元組 ($a$, $b$, $d$)，使得點 $P$ 存在。

如果 $b + d &lt; 100\\,000$，則存在 320471 個不同的三元組 ($a$, $b$, $d$)，使得點 $P$ 存在。

如果 $b + d &lt; 100\\,000\\,000$，有多少個不同的三元組 ($a$, $b$, $d$) 使得點 $P$ 存在？
