# --explanation--

`¿Cuál es tu nacionalidad?` bedeutet „Was ist Ihre Nationalität?“ Wenn Sie im Feld `Nacionalidad` nachsehen, sehen Sie, dass dort `chilena` steht. Die richtige Antwort ist also `Soy chilena`.

Beachten Sie, dass Nationalitäten im Spanischen in Kleinbuchstaben geschrieben werden. Zum Beispiel:

- `Soy mexicana.` – Ich bin Mexikanerin. (weiblich)
- `Soy español.` – Ich bin Spanier. (männlich)
