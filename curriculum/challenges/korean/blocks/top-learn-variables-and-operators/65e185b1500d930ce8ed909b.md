# --description--
가장 간단하게 시작하는 방법은 JavaScript 코드를 포함한 HTML 파일을 만드는 것입니다. 컴퓨터의 적당한 위치에 기본 HTML 뼈대를 입력하세요:

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>
  <script>
    // Your JavaScript goes here!
    console.log("Hello, World!")
  </script>
</body>
</html>
```

이 파일을 저장하고 웹 브라우저에서 열어보세요(Visual Studio Code에서 <a href="https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer" target="_blank" >"Live Server"</a>를 사용해도 됩니다!). 그런 다음 빈 웹페이지에서 마우스 오른쪽 버튼을 눌러 "검사" 또는 "요소 검사"를 선택해 브라우저의 콘솔을 열어보세요. ‘개발자 도구’ 창에서 ‘콘솔’ 탭을 찾아 선택하면 `console.log` 문장의 출력 결과를 볼 수 있습니다.

> `console.log()`은 브라우저의 개발자 콘솔에 무언가를 출력하는 명령어입니다. 이 명령어를 사용해 이후의 모든 글과 연습 문제에서 결과를 콘솔에 출력할 수 있습니다. 이 수업과 앞으로의 수업 예제들을 따라 코딩해 보시길 권장합니다.

웹페이지에 JavaScript를 포함하는 또 다른 방법은 외부 스크립트를 사용하는 것입니다. 이는 외부 CSS 문서를 웹사이트에 연결하는 것과 매우 비슷합니다.

```html
  <script src="javascript.js"></script>
```

JavaScript 파일은 스타일시트의 `.js`와 비슷하게 `.css` 확장자를 가집니다. 외부 JavaScript 파일은 더 복잡한 스크립트에 사용됩니다.

# --questions--

## --text--

HTML 문서에 JavaScript를 포함하는 방법을 정확히 설명한 문장은 무엇인가요?

## --answers--

JavaScript는 HTML 본문의 `<script>` 태그 안에만 내부적으로 포함될 수 있습니다.

---

JavaScript 코드는 제대로 실행되려면 HTML 문서의 `<head>` 섹션에 작성되어야 합니다.

---

JavaScript는 `<script>` 태그를 사용해 HTML 파일에 직접 포함하거나 `<script src="javascript.js">` 태그를 통해 외부에서 연결할 수 있습니다.

---

외부 JavaScript 파일은 웹 브라우저가 인식하도록 `<script>` 태그에 특별한 속성이 필요합니다.
