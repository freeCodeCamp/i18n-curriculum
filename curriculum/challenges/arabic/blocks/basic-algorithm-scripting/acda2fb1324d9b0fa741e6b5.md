# --description--

تحقق مما إذا كانت سلسلة نصية (المعلمة الأولى، `str`) تنتهي بالسلسلة النصية الهدف المعطاة (المعلمة الثانية، `target`).

يمكن حل هذا التحدٍّ باستخدام الطريقة `.endsWith()`، التي أُدخلت في ES2015. لكن لغرض هذا التحدٍّ، نود منك استخدام إحدى طرق السلاسل النصية الفرعية في JavaScript بدلاً من ذلك.

function confirmEnding(str, target) {
  return str.substring(str.length - target.length) === target;
}
