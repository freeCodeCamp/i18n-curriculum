# --description--

يأتي Bootstrap مع عدة ألوان معرفة مسبقًا للأزرار. تُستخدم الصنف `btn-info` للفت الانتباه إلى الإجراءات الاختيارية التي يمكن للمستخدم اتخاذها.

أنشئ زر Bootstrap جديد على مستوى الكتلة أسفل زر `Like` الخاص بك مع النص `Info`، وأضف إليه الصنف `btn-info` الخاص بـ Bootstrap.

لاحظ أن هذه الأزرار لا تزال بحاجة إلى أصناف `btn` و `btn-block`.

# --hints--

يجب أن تنشئ عنصر `button` جديدًا مع النص `Info`.

```js
const infoButton = document.querySelectorAll('button')?.[1]; 
assert.match(infoButton.textContent,/info/gi);
```

يجب أن يحتوي كلا زري Bootstrap على الأصناف `btn` و `btn-block`.

```js
assert.lengthOf(document.querySelectorAll('button.btn-block.btn'),2);
```

يجب أن يحتوي الزر الجديد على الصنف `btn-info`.

```js
const infoButton = [...document.querySelectorAll('button')].at(1);
assert.isTrue(infoButton?.classList?.contains('btn-info'));
```

يجب أن تحتوي جميع عناصر `button` الخاصة بك على وسم الإغلاق.

```js
assert.match(code,/<\/button>/g);
assert.match(code,/<button/g);

assert.equal(code.match(/<\/button>/g)?.length,code.match(/<button/g)?.length);
```
