# --description--

3-гладке число — це ціле число, яке не має простих множників більших за 3. Для цілого числа $N$ ми визначаємо $S(N)$ як множину 3-гладких чисел, менших або рівних $N$. Наприклад, $S(20) = \\{1, 2, 3, 4, 6, 8, 9, 12, 16, 18\\}$.

Ми визначаємо $F(N)$ як кількість перестановок множини $S(N)$, у яких кожен елемент йде після всіх своїх власних дільників.

Ось одна з можливих перестановок для $N = 20$.

-   1, 2, 4, 3, 9, 8, 16, 6, 18, 12.

Це не є дійсною перестановкою, бо 12 йде перед своїм дільником 6.

-   1, 2, 4, 3, 9, 8, 12, 16, 6, 18.

Ми можемо перевірити, що $F(6) = 5$, $F(8) = 9$, $F(20) = 450$ і $F(1000) ≈ 8.8521816557e\\,21$.

Знайдіть $F({10}^{18})$. Відповідь подайте у вигляді рядка в науковій нотації, округленої до десяти знаків після десяткової крапки. При поданні відповіді використовуйте нижній регістр `e` для розділення мантиси та експоненти. Наприклад, якщо відповідь $112\\,233\\,445\\,566\\,778\\,899$, то формат відповіді буде `1.1223344557e17`.
