# --description--

编程中的 `assignment operator` 用于为变量赋值。在许多编程语言中，包括 Python，`assignment operator` 由符号 `=` 表现。它告诉计算机将 `=` 右侧的值保存到左侧命名的变量中。例如，`number = 10` 将值 `10` 赋值给变量 `number`。
