# --explanation--

`The day before` ist eine adverbiale Phrase, die sich auf den vorherigen Tag bezieht. Sie gibt einen Zeitbezug an und zeigt, dass etwas einen Tag vor dem aktuellen Tag passiert ist.

Beachten Sie, dass es nicht ausreicht, einfach zu sagen, `yesterday` habe mit dem Tag vor `today` zu tun. Sie können `the day before` verwenden, um über einen Tag vor einem beliebigen Tag zu sprechen. Zum Beispiel:

`We discussed the new feature the day before.` – Das bedeutet, die Diskussion fand am Tag vor dem besprochenen Tag statt.

Im Gegensatz dazu bedeutet `yesterday`, wie in `We discussed the new feature yesterday.`, speziell, dass es am Tag vor heute passiert ist.
