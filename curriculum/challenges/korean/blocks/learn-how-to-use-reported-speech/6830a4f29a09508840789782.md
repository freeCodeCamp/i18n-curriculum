# --explanation--

`The additional contact options were highlighted...`는 **수동태** 구조입니다. 누군가 연락 옵션을 훌륭한 개선점으로 지적했지만, 누가 그렇게 했는지는 말하지 않습니다(사용자와 그들의 피드백을 이미 말하고 있으므로). 초점은 동작과 기능 자체에 있습니다. 또 다른 예:

`The security updates were highlighted in the weekly report.` – 보고서가 그것들을 지적했지만, 누가 작성했는지에 초점을 맞추지 않습니다.
