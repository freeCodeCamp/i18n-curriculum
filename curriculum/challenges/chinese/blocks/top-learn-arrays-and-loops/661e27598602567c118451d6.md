# --description--

`for` 循环是另一种用于多次执行一段代码块的循环类型。当已知迭代次数时使用 `for` 循环。`for` 循环的语法如下：

```javascript
for (initialization; condition; increment/decrement) {
  // code block to be executed
}
```

例如，以下代码截图打印从 1 到 5 的数字：

```javascript
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
```

要使用 `for` 循环迭代一个数组，你可以使用数组的长度属性。例如，以下代码截图打印数组的元素：

```javascript
const arr = [1, 2, 3, 4, 5];
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}
```
