# --explanation--

`Lastly` si usa per introdurre l'ultimo punto in una lista o discussione. Per esempio:

`Lastly, let's review the budget changes.` – Questo significa che è l'ultimo elemento di cui parlare.

`Any thoughts on...` è un modo informale per chiedere le idee o opinioni di qualcuno. Spesso omette la frase completa `Do you have any thoughts on...` all'inizio. Per esempio:

`Any thoughts on how we can improve this layout?` – Questo significa che chi parla sta chiedendo idee in modo amichevole e informale su come migliorare un layout.
