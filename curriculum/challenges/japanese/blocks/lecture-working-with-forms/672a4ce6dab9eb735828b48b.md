# --interactive--

ユーザーがあなたのウェブサイトのフォームに入力するとき、必要な情報を正しい形式で入力することが重要です。入力のようなHTMLのフォームコントロールには、多くの組み込みバリデーションがあり、不正なデータをチェックするために使えます。これにより、情報がサーバーに提出され処理される前に、ユーザーがこれらの間違いを修正することができます。

「クライアントサイド」という用語は、ユーザーのコンピューターやデバイス上で起こるすべてのことを指し、ウェブサイトやアプリの直接操作する部分を含みます。これにはレイアウト、デザイン、そしてインタラクティブな機能が含まれます。

「サーバーサイド」という用語は、ウェブサイトやアプリをホストしているサーバー、コンピューター、またはシステム上で起こるすべてのことを指します。これにはデータの処理、アプリケーションの実行、ユーザーのデバイスからのリクエストの処理が含まれます。

クライアントサイドのバリデーションは重要ですが、追加のセキュリティのためにサーバーサイドのバリデーションも必要です。悪意のあるユーザーはクライアントサイドのチェックを回避できるため、堅牢なサーバーサイドの対策が不可欠です。これについては後のモジュールで詳しく学びます。今はクライアントサイドのフォームバリデーションの例を見てみましょう。

組み込みのフォームバリデーションの一般的な例は、入力に`required`属性を使うことです。`required`属性は、ユーザーがフォームのその部分を提出前に必ず入力する必要があることを指定します。以下はメール入力で`required`属性を使った例です。

メールアドレスを入力せずに`Submit Form`ボタンをクリックすると、そのフィールドを入力するように促すメッセージが表示されます。

:::interactive_editor

```html
<form action="">
  <label for="email">Email Address (Required field):</label>
  <input required type="email" name="email" id="email" />
  <button type="submit">Submit Form</button>
</form>
```

:::

各ブラウザはこの警告メッセージを表示するための独自のスタイルを持っています。

メール入力を使うもう一つの利点は、メール入力には正しい形式のメールアドレスかどうかを確認する基本的なバリデーションがあることです。例えば、ランダムな単語を入力して送信ボタンをクリックすると、ブラウザは`@`記号が欠けているという警告を表示します。

メールフィールドに`abc`を入力して送信ボタンをクリックしてください。無効なメールアドレスなのでメッセージが表示されるはずです。

:::interactive_editor

```html
<form action="">
  <label for="email">Email Address (Required field):</label>
  <input required type="email" name="email" id="email" />
  <button type="submit">Submit Form</button>
</form>
```

:::

ブラウザは標準的なメールアドレスの基本的なバリデーションしか行わないことに注意してください。追加のバリデーションはあなたが後のモジュールで学ぶ必要があります。

メール入力の他のバリデーションとして、`minlength`属性と`maxlength`属性を使う方法があります。以下は追加のバリデーションを使った例です。

フィールドに`b@m`を入力して送信ボタンをクリックしてください。最小値の長さを満たしていないため、メッセージが表示されます。

:::interactive_editor

```html
<form action="">
  <label for="email">Email Address (Required field):</label>
  <input
    required
    type="email"
    name="email"
    id="email"
    minlength="4"
    maxlength="64"
  />
  <button type="submit">Submit Form</button>
</form>
```

:::

`minlength`属性と`maxlength`属性は、メール入力の文字数の最小値と最大値を設定するために使われます。最小値を含めなかったり、最大文字数を超えたりすると、ブラウザは警告メッセージを表示します。
