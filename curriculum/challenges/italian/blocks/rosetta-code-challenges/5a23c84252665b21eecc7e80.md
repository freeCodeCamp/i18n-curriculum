# --description--

Il codice Gray è una forma di codifica binaria in cui le transizioni tra numeri consecutivi differiscono per un solo bit.

Questa codifica è utile per ridurre i problemi hardware con dati che cambiano rapidamente e/o si collegano a hardware più lento come input.

È anche utile per generare input per le mappe di Karnaugh in ordine da sinistra a destra o dall’alto verso il basso.

# --instructions--

Crea una funzione per codificare un numero in codice Gray e decodificare un numero dal codice Gray. La funzione deve avere 2 parametri.

Il primo sarà un booleano. La funzione deve codificare se è true e decodificare se è false. Il secondo parametro sarà il numero da codificare/decodificare.

Mostra le rappresentazioni binarie normali, le rappresentazioni in codice Gray e i valori decodificati dal codice Gray per tutti i numeri binari a 5 bit (da 0 a 31 inclusi, non è necessario mostrare gli zeri iniziali).

Esistono molti codici Gray possibili. Quello seguente codifica il cosiddetto "codice Gray riflesso binario".

Codifica (MSB è il bit 0, b è binario, g è codice Gray):

<pre>if b[i-1] = 1
  g[i] = not b[i]
else
  g[i] = b[i]
</pre>

Oppure:

<pre>g = b xor (b logical right shifted 1 time)
</pre>

Decodifica (MSB è il bit 0, b è binario, g è codice Gray):

<pre>b[0] = g[0]<br>
per gli altri bit:
b[i] = g[i] xor b[i-1]
</pre>

function gray(enc, number){
  if(enc){
      return number ^ (number >> 1);
  }else{
      let n = number;

      while (number >>= 1) {
          n ^= number;
      }
