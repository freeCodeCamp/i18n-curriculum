# --interactive--

عند العمل مع صور الخلفية في CSS، لديك عدة خصائص للتحكم في كيفية عرض هذه الصور.

الخصائص الرئيسية التي سنركز عليها هي `background-size`، و`background-repeat`، و`background-position`، و`background-attachment`.

لنلقِ نظرة أولاً على خاصية `background-image`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

تُعيّن شفرة CSS أعلاه صورة خلفية لقطة لعنصر `body`.

إذا أردت تحديد حجم صورة الخلفية، يمكنك استخدام خاصية `background-size`.

يمكنك استخدام القيمة `contain` لتكبير الصورة إلى أكبر حجم ممكن دون قص أو تمديد.

إليك مثالًا مع `background-size: contain`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    min-height: 100px;
  }
</style>
```

:::

نحدد `min-height` إلى `100px` حتى تكون صورة الخلفية مرئية ويحافظ التخطيط على ارتفاع أساسي، مما يضمن أن التصميم يبدو متوازنًا وجذابًا حتى مع محتوى قليل.

إذا غيرنا خاصية `background-size` لاستخدام القيمة `cover`، فإن صورة الخلفية ستتكبر لتغطي كامل عنصر `body` مع الحفاظ على نسبة العرض إلى الارتفاع.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    min-height: 100px;
  }
</style>
```

:::

في الأمثلة السابقة، ربما لاحظت أن صورة الخلفية كانت تتكرر باستمرار.

افتراضيًا، تتكرر صور الخلفية أفقيًا وعموديًا لملء العنصر بالكامل. لكن يمكنك التحكم في هذا السلوك.

يمكنك استخدام خاصية `background-repeat` مع القيمة `no-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

مع تعيين `background-size` إلى `contain` و`background-repeat` إلى `no-repeat`، لن تتكرر الصورة على الشاشة.

إذا أردت تكرار صورة الخلفية أفقيًا، يمكنك استخدام القيمة `repeat-x` لخاصية `background-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-x;
    min-height: 100px;
  }
</style>
```

:::

إذا أردت تكرار صورة الخلفية عموديًا، يمكنك استخدام القيمة `repeat-y` لخاصية `background-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-y;
    min-height: 100px;
  }
</style>
```

:::

لوضع صورة خلفية على الشاشة، يمكنك استخدام خاصية `background-position`.

تسمح خاصية `background-position` بتحديد مكان ظهور صورة الخلفية داخل العنصر. يمكنك استخدام كلمات مفتاحية مثل `top`، `bottom`، `left`، `right`، و`center`، أو قيم بكسل أو نسب مئوية محددة.

إليك مثالًا يستخدم `center` و`top` لـ `background-position`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

تضع شفرة CSS هذه صورة الخلفية في مركز العنصر أفقيًا وفي الأعلى عموديًا.

أخيرًا، تحدد خاصية `background-attachment` ما إذا كانت صورة الخلفية تتحرك مع المحتوى عند التمرير أو تبقى ثابتة عند التمرير.

القيم الرئيسية هي `scroll` (افتراضيًا)، حيث تتحرك صورة الخلفية مع المحتوى، و`fixed`، حيث تبقى صورة الخلفية في نفس الموضع على الشاشة.

إليك مثالًا يستخدم القيمة `fixed` لخاصية `background-attachment`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-position: center top;
    background-attachment: fixed;
  }
</style>
```

:::

تجعل شفرة CSS هذه صورة الخلفية تبقى ثابتة في مكانها حتى عند تمرير الصفحة.

إذا أردت دمج بعض الخصائص في سطر واحد، يمكنك فعل ذلك باستخدام خاصية الاختصار `background`.

إليك مثالًا:

:::interactive_editor

```html
<style>
  body {
    background: center top fixed
      url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

يجمع المثال أعلاه بين `background-image` و`background-position` و`background-attachment` في سطر واحد.

بإتقان هذه الخصائص، ستحصل على تحكم ممتاز في كيفية عرض صور الخلفية في تصاميم الويب الخاصة بك، مما يسمح بتصاميم أكثر جاذبية ووظيفية.
