# --description--

Вираз `a lot of` можна використовувати як з іменниками, що можна порахувати, так і з незлічуваними іменниками, щоб позначити велику кількість або обсяг.

Для іменників, які можна порахувати, таких як `cars`, ви можете сказати `a lot of cars`. Для незлічуваних іменників, як `water`, ви скажете `a lot of water`.
