# --description--

Das Balken- und Streudiagramm haben beide Daten direkt auf das SVG geplottet. Wenn jedoch die Höhe eines Balkens oder eines Datenpunkts größer als die Höhe oder Breite des SVG wäre, würde es außerhalb des SVG-Bereichs liegen.

In D3 gibt es Skalen, die beim Plotten von Daten helfen. `scales` sind Funktionen, die dem Programm sagen, wie eine Menge von Rohdatenpunkten auf die Pixel des SVG abgebildet werden.

Zum Beispiel haben Sie ein 100x500 großes SVG und möchten das Bruttoinlandsprodukt (BIP) für mehrere Länder darstellen. Die Zahlen liegen im Milliarden- oder Billionenbereich. Sie geben D3 eine Art Skala vor, die angibt, wie die großen BIP-Werte in diesen 100x500-Bereich eingeordnet werden.

Es ist unwahrscheinlich, dass Sie Rohdaten unverändert plotten. Bevor Sie sie plotten, legen Sie die Skala für Ihren gesamten Datensatz fest, sodass die `x` und `y` Werte in Ihre SVG-Breite und -Höhe passen.

D3 bietet verschiedene Skalentypen. Für eine lineare Skala (normalerweise bei quantitativen Daten verwendet) gibt es die D3-Methode `scaleLinear()`:

```js
const scale = d3.scaleLinear();
```

Standardmäßig verwendet eine Skala die Identitätsbeziehung. Der Wert der Eingabe ist derselbe wie der Wert der Ausgabe. Eine separate Aufgabe behandelt, wie man dies ändert.
