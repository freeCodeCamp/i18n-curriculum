# --interactive--

型セレクターは、要素セレクターとも呼ばれ、タグ名に基づいて要素を指定します。

これらのセレクターはCSSの基本であり、特定のHTML要素のすべてのインスタンスにスタイルを適用できます。

型セレクターは使い方が簡単で、スタイルを適用したい要素のタグ名をそのまま書くだけです。

以下は、ページ上のすべての段落要素を対象にした型セレクターの例です。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  color: blue;
}
```

:::

この例では、すべての`p`要素のテキスト色が`blue`に設定されます。

型セレクターは他のセレクターに比べて特異性が比較的低く、型セレクターの特異性の値は`(0, 0, 0, 1)`です。

つまり、型セレクターはクラスセレクター、IDセレクター、インラインスタイルによってオーバーライドされますが、それらの高い特異性のルールがない限りスタイルを適用できます。

次に、クラスセレクターが型セレクターのスタイルをオーバーライドする例を見てみましょう。

以下は2つの段落要素の例です。

:::interactive_editor

```html
<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

:::

両方の段落要素には`para`というクラスがあります。

CSSファイル内では、型セレクターが段落を対象にし、クラスセレクターが`para`クラスを持つ要素を対象にしています。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

```css
p {
  color: blue;
}

.para {
  color: red;
}
```

:::

ページ上の`para`クラスを持つすべての段落は、型セレクターの`red`ではなく、クラスセレクターの`blue`にテキスト色が設定されます。これはクラスセレクターの特異性が型セレクターより高いためです。
