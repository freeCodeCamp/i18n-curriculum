# --description--

你最近学会了模式 `¿Quieres que lo deletree?` 来提供拼写某物。另一个常见的问题是：

`¿Cómo se deletrea?`

当有人想知道一个单词、名字或电子邮件的字母拼写时，会使用这个问题。

在人们确认联系方式的对话中，这种情况非常常见。

# --explanation--

音频说 `¿Cómo se deletrea?`。这个问题是在询问某物如何拼写。因此，正确答案必须逐字母拼写该信息。

只有一个选项是这样：`j-u-l-i-e-t-a punto c-a-c-e-r-e-s arroba a-n-d-e-s punto com.`

其他选项是完整的语句（name、nationality、profession），并且不提供拼写。
