# --description--

У попередніх уроках ви ознайомилися з візуалізацією даних — процесом створення чітких, наочних форматів для представлення складних даних. У цьому уроці ви дізнаєтеся про бібліотеку D3.

Бібліотека D3 — це безкоштовна та з відкритим кодом бібліотека, яку ви можете використовувати для створення привабливої та ефективної візуалізації даних на JavaScript.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-1.png" alt="A graph with four nodes labeled A, B, C, and D. Node A connects to B, C, and D. Node B connects to A and D. Node C connects to A. Node D connects to A and B. It's the same image as before.">


Назва D3 є скороченням від data-driven documents, що означає, як вона динамічно зв’язує дані з елементами в Document Object Model (DOM). Ця бібліотека гнучка і природно працює з вебом. Замість того, щоб вивчати цілком нове графічне представлення, ви можете будувати візуалізації даних за допомогою `svg` та `canvas` елементів.

Наприклад, якщо ви хочете створити гістограму за допомогою D3, ви можете зробити це все в одному JS файлі:

* Оголосити змінні, що представляють розміри діаграми та поля
    
* Оголосити шкалу горизонтального положення
    
* Оголосити шкалу вертикального положення
    
* Створити SVG контейнер з стовпчиками та мітками
    
<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-2.png" alt="A graph with four nodes labeled A, B, C, and D. Node A connects to B, C, and D. Node B connects to A and D. Node C connects to A. Node D connects to A and B. It's the same image as before.">

Отже, які ще є причини вивчати D3 замість інших інструментів візуалізації, як-от Chart.js, Plotly або Tableau?

На відміну від інших інструментів, D3 надає вам багато низькорівневого контролю, де ви можете створювати кастомні та інтерактивні візуалізації, використовуючи лише звичайний HTML, CSS і JavaScript. До того ж, бібліотека написана на JavaScript, тож вам не потрібно вивчати зовсім нову мову.

У майбутніх практичних заняттях і лабораторних роботах ви вивчите основи бібліотеки D3, такі як вибірки, форми та шкали. Ви опануєте ці ключові концепції, створюючи точкові діаграми, гістограми, застосунки для панелей керування та інше.
