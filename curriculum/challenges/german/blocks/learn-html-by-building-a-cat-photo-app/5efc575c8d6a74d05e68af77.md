# --description--

Fügen Sie nach dem vorherigen ein abschließendes Kontrollkästchen mit einem `id`-Attributwert von `energetic` hinzu. Das `name`-Attribut sollte dasselbe sein wie beim vorherigen Kontrollkästchen.

Fügen Sie außerdem rechts neben dem neuen Kontrollkästchen ein `label`-Element mit dem Text `Energetic` hinzu. Stellen Sie sicher, dass das `label`-Element mit dem neuen Kontrollkästchen verknüpft ist.

# --hints--

Sie müssen ein neues Kontrollkästchen hinzufügen.

```js
assert.lengthOf(document.querySelectorAll('input[type="checkbox"]'),3);
```

Ihr neues Kontrollkästchen sollte ein `id`-Attribut mit dem Wert `energetic` und ein `name`-Attribut mit dem Wert `personality` haben. Achten Sie darauf, dass nach dem Namen des öffnenden Tags ein Leerzeichen steht und/oder vor allen Attributnamen Leerzeichen sind.

```js
const checkboxes = [...document.querySelectorAll('input[type="checkbox"]')];
assert(
  checkboxes.some(
    (checkbox) =>
      checkbox.id === 'energetic' &&
      checkbox.getAttribute('name') === 'personality'
  )
);
```

Ihr neues Kontrollkästchen sollte nach dem zweiten stehen. Sie haben sie in der falschen Reihenfolge.

```js
const checkboxes = [...document.querySelectorAll('input[type="checkbox"]')].map(
  (checkbox) => checkbox.id
);
assert(checkboxes.indexOf('lazy') < checkboxes.indexOf('energetic'));
```

Rechts neben Ihrem neuen Kontrollkästchen sollte ein `label`-Element mit dem Text `Energetic` stehen.

```js
const nextElementSibling = document.querySelectorAll('input[type="checkbox"]')?.[2]?.nextElementSibling;
assert.isOk(
  nextElementSibling?.nodeName === 'LABEL' &&
  nextElementSibling?.innerText?.trim().match(/^Energetic$/i)
);
```

Das neue `label` sollte ein `for`-Attribut mit demselben Wert haben wie das `id`-Attribut des neuen Kontrollkästchens. Sie haben entweder den Wert weggelassen oder einen Tippfehler.

```js
assert.strictEqual(
  document.querySelectorAll('input[type="checkbox"]')?.[2]?.nextElementSibling?.getAttribute('for'),
  'energetic'
);
```
