# --description--

Quando si fornisce un nome o un indirizzo email, può essere necessario dire ogni lettera separatamente per evitare confusione. Questa azione si chiama `deletrear`. Il verbo `deletrear` si usa quando qualcuno compone una parola lettera per lettera. Per esempio:

- `Deletreo mi apellido.`  - Compongo il mio cognome.

- `¿Quieres que deletree tu nombre?` - Vuoi che componga il tuo nome?
