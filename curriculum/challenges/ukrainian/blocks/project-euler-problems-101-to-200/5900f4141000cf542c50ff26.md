# --description--

Для двох натуральних чисел $a$ та $b$ послідовність Улама $U(a,b)$ визначається як ${U{(a,b)}\_1} = a$, ${U{(a,b)}\_2} = b$ і для $k > 2$ ${U{(a,b)}\_k}$ — це найменше ціле число, більше за ${U{(a,b)}\_{(k-1)}}$, яке можна записати рівно одним способом як суму двох різних попередніх членів $U(a,b)$.

Наприклад, послідовність $U(1,2)$ починається з

$$1, 2, 3 = 1 + 2, 4 = 1 + 3, 6 = 2 + 4, 8 = 2 + 6, 11 = 3 + 8$$

Число 5 не належить до неї, бо $5 = 1 + 4 = 2 + 3$ має два представлення як сума двох попередніх членів, так само як і $7 = 1 + 6 = 3 + 4$.

Знайдіть $\sum {U(2, 2n + 1)_k}$ для $2 ≤ n ≤ 10$, де $k = {10}^{11}$.
