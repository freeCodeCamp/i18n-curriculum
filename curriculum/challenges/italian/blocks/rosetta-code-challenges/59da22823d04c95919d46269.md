# --description--

Cinque marinai naufragano su un'isola e raccolgono durante il giorno una grande pila di noci di cocco.

Quella notte il primo marinaio si sveglia e decide di prendere subito la sua prima parte, quindi prova a dividere la pila di noci di cocco in cinque pile uguali ma scopre che avanza una noce di cocco, così la lancia a una scimmia e poi nasconde "la sua" di cinque pile di noci di cocco di uguale dimensione, spinge insieme le altre quattro pile per formare di nuovo una singola pila visibile di noci di cocco e va a dormire.

Per farla breve, ciascuno dei marinai a turno si alza una volta durante la notte e compie le stesse azioni: divide la pila di noci di cocco in cinque, trova che avanza una noce di cocco e dà quella singola noce di cocco rimanente alla scimmia, poi spinge insieme le altre quattro pile per formare una singola pila.

Al mattino (dopo l'azione furtiva e separata di ciascuno dei cinque marinai durante la notte), le noci di cocco rimanenti vengono divise in cinque pile uguali per ciascuno dei marinai, e si scopre che la pila di noci di cocco si divide equamente tra i marinai senza resto. (Niente per la scimmia al mattino.)

# --instructions--

Crea una funzione che restituisca la dimensione minima possibile della pila iniziale di noci di cocco raccolte durante il giorno per `N` marinai. **Nota:** Ovviamente il racconto è ambientato in un mondo dove la raccolta di qualsiasi numero di noci di cocco in un giorno e le divisioni multiple della pila, ecc., possono avvenire in un tempo coerente con la storia, senza influenzare la matematica.

```js
function splitCoconuts(intSailors) {
  let intNuts = intSailors;
  let result = splitCoconutsHelper(intNuts, intSailors);
  while (!result) {
    intNuts += 1;
    result = splitCoconutsHelper(intNuts, intSailors);
  }

  return intNuts;
}

function splitCoconutsHelper(intNuts, intSailors, intDepth) {
  const nDepth = intDepth !== undefined ? intDepth : intSailors;
  const portion = Math.floor(intNuts / intSailors);
  const remain = intNuts % intSailors;

  if (portion <= 0 || remain !== (nDepth ? 1 : 0)) {
    return null;
  }

  if (nDepth) {
    return splitCoconutsHelper(
      intNuts - portion - remain, intSailors, nDepth - 1
    );
  }

  return intNuts;
}
```
