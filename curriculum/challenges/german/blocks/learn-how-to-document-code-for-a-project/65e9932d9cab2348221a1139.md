# --description--

`Changes` bedeutet in diesem Zusammenhang Änderungen oder Bearbeitungen im Code.

`Descriptive` bedeutet, Details darüber zu geben, wie etwas aussieht, sich anfühlt, klingt, riecht oder schmeckt. Es ist ein Adjektiv, das verwendet wird, um mehr Informationen über ein Substantiv zu liefern und die Beschreibung lebendiger und klarer zu machen. Zum Beispiel ist im Satz `she wears a beautiful dress` das Wort `beautiful` `descriptive`, weil es den Zuhörenden mehr über das Kleid erzählt.

Ein `message` ist eine Information oder Mitteilung, die jemand an eine andere Person sendet. Sie kann schriftlich, mündlich oder sogar per E-Mail oder Textnachricht übermittelt werden.
