# --description--

`Mostly` is used to indicate that something is true to a large extent but not completely. Sugiere que, aunque puedan estar involucrados otros factores o métodos, el aspecto principal es lo que se menciona. For example:

`The app is mostly used by students.` - This means students are the main users, but other people also use it.
