# --explanation--

`Another`は数字の前に使うと追加の数量を意味します。例えば：

`We ordered another five laptops for the new team.` — 新しいチームのために5台のノートパソコンをさらに手に入れました。

トムはすでに1か所に10台のタブレットがあると言っていて、今別の場所にさらに5台あると言っています。
