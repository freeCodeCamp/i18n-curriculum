# --interactive--

當建置多語言網站或處理訂製資料屬性時，你經常需要根據元素所包含的語言或特定資料值來設定樣式。

`lang` 和 `data-lang` 屬性通常用於這些目的，而 CSS 中的屬性選擇器允許你根據這些屬性有效地套用樣式。

`lang` 屬性用於 HTML 中指定元素內內容的語言。你可能會想根據內容所使用的語言來不同地設計元素的樣式，特別是在多語言網站上。

以下是如何使用屬性選擇器來標的具有特定 `lang` 屬性的元素的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">This is an English paragraph.</p>
<p lang="fr">Ceci est un paragraphe en français.</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

這條 CSS 規則會對任何 `lang` 屬性設定為英文（`en`）的段落元素套用斜體樣式。這在包含多種語言的文件中強調英文文字時可能很有用。

同樣地，你可以使用屬性選擇器來標的具有 `data-lang` 屬性的元素。 

訂製資料屬性如 `data-lang` 通常用於在元素中儲存額外的資訊，例如指定特定文字區段中使用的語言。 

以下是你如何使用 CSS 根據 `data-lang` 屬性標的並設定元素樣式的方法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p data-lang="fr">Ceci est un paragraphe en français.</p>
<p data-lang="en">This is a paragraph in English.</p>
```

```css
p[data-lang="fr"] {
  color: blue;
}
```

:::

在這種情況下，任何具有 `data-lang` 屬性且設定為法文（`fr`）的 `p` 元素，其文字顏色將會變成藍色。這讓你能夠根據儲存在 `data-lang` 屬性中的語言資訊，快速識別並設定內容區段的樣式。

像這樣的屬性選擇器提供了一種強大的方式，根據嵌入在你的 HTML 中的詮釋資料套用條件式樣式，使你的網頁更具動態性和背景關係感知能力。
