# --description--

任意の正の整数 $n$ に対して、関数 $\text{next_prime}(n)$ は $p > n$ を満たす最小の素数 $p$ を返します。

シーケンス $a(n)$ は次のように定義されます：$a(1) = \text{next_prime}({10}^{14})$ かつ $n > 1$ のとき $a(n) = \text{next_prime}(a(n - 1))$。

フィボナッチシーケンス $f(n)$ は次のように定義されます：$f(0) = 0$、$f(1) = 1$、および $n > 1$ のとき $f(n) = f(n - 1) + f(n - 2)$。

シーケンス $b(n)$ は $f(a(n))$ と定義されます。

$1≤n≤100\,000$ の範囲で $\sum b(n)$ を求めてください。答えは $\bmod 1\,234\,567\,891\,011$ で出してください。
