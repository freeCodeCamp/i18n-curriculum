# --explanation--

Слово `Guatemalteco` — це чоловіча форма національності людини з Гватемали.

Національності повинні узгоджуватися за родом із людиною, яку вони описують.

| Рід | Національність | Приклад |
| :--- | :--- | :--- |
| **Жіночий** | `Guatemalteca` | `Ella es guatemalteca.` ("Вона гватемалка.") |
| **Чоловічий** | `Guatemalteco` | `Él es guatemalteco.` ("Він гватемалець.") |
