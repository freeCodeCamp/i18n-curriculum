## 你本地环境中的 Python

- **安装**：在 Windows、Mac 和 Linux 上安装 Python 的最佳方式是从官方 Python 网站下载安装程序 (`https://www.python.org/`)。
- **终端**：一种基于文本的接口，允许你通过输入命令与计算机交互。每个操作系统都自带一个默认的终端应用程序。在 macOS 上，你可以使用 Terminal 应用。在 Windows 上，你可以使用命令提示符或 PowerShell。在 Linux 上，每个桌面环境都有自己的默认终端应用，比如 GNOME Terminal 或 Konsole。
- **集成开发环境（IDE）**：IDE 是集成开发环境的缩写，具有代码编辑器、测试工具、终端等功能。
- **Python 的流行代码编辑器和 IDE**：VS Code、PyCharm 和 Spyder
- **本地运行代码**：运行 Python 代码有几种选择。第一种选择是在 VS Code 中点击位于右上角的运行按钮（看起来像播放按钮）。这会打开一个终端并在那里运行你的 Python 脚本。

另一种选择是打开终端并手动运行程序。下面是运行名为 `main.py` 的程序的示例：

```bash
python main.py
```

你必须在保存 `main.py` 的文件夹中运行此命令。例如，如果 `main.py` 在名为 `python-projects` 的文件夹内，先在终端中使用以下命令：

```bash
cd python-projects
```

然后运行：

```bash
python main.py
```

## 使用 Python 交互式 Shell

- **定义**：交互式 shell 是一个程序，允许你一次输入一条命令并查看结果。打开终端应用，输入 `python` 并按下 `Enter`。这将启动 Python 交互式 shell。`>>>` 符号表示 Python 正在等待你输入命令。试着向终端打印 `"Hello, world!"`：

```bash
print("Hello, world!")
```

你应该看到文本像这样显示：

```bash
>>> print("Hello, world!")
Hello, world!
```

文本打印后，Python 会返回以下提示：

```bash
>>>
```

这意味着你可以输入另一条命令。

- **读取、求值、打印、循环周期（REPL）**：每当你输入命令时，解释器会读取输入、求值、打印结果，然后循环回显示 `>>>`，以便你输入更多命令。
