# --explanation--

`Server` يشير إلى جهاز كمبيوتر قوي أو نظام يوفّر خدمات أو بيانات أو موارد لأجهزة أخرى، غالبًا عبر شبكة. على سبيل المثال:

`The server hosts the company's website.` - هذا يعني أن الخادم مسؤول عن تخزين وتسليم محتوى الموقع للمستخدمين الذين يصلون إليه.

`Load` يشير إلى كمية العمل أو الطلب أو الضغط المفروض على نظام أو مورد. على سبيل المثال:

`The system crashed because it couldn't handle the high load during peak hours.` - هذا يعني أن النظام تعرّض للإرهاق بسبب الكم الكبير من الطلب المفروض عليه.
