# --description--

لتكن $S_n$ متتالية أعداد صحيحة تُنتج باستخدام مولد أعداد شبه عشوائي كما يلي:

$$\begin{align}
        S_0 & = 290\\,797 \\\\
  S_{n + 1} & = {S_n}^2\bmod 50\\,515\\,093
\end{align}$$

لتكن $A(i, j)$ الحد الأدنى للأعداد $S_i, S_{i + 1}, \ldots, S_j$ حيث $i ≤ j$. ولتكن $M(N) = \sum A(i, j)$ حيث $1 ≤ i ≤ j ≤ N$.

يمكننا التحقق من أن $M(10) = 432\\,256\\,955$ و $M(10\\,000) = 3\\,264\\,567\\,774\\,119$.

احسب $M(2\\,000\\,000\\,000)$.
