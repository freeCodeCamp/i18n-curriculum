# --explanation--

عند تكوين سؤال باستخدام الأفعال الناقصة، يكون التركيب: الفعل الناقص + الفاعل + الفعل في صورته الأساسية + (المفعول به)؟ على سبيل المثال:

- `Can we complete the project by Friday?` - سؤال عن إمكانية إكمال المشروع بحلول يوم الجمعة.

- `Should we update the software to the latest version?` - سؤال لطلب نصيحة أو اقتراح حول تحديث البرنامج إلى الإصدار / النسخة الأحدث.

في الإجابة الصحيحة، يأتي `can` قبل الفاعل (`you`)، ويكون الفعل (`explain`) في صورته الأساسية.
