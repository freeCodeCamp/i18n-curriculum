# --description--

辺の長さが6、8、10の三角形を考えます。この三角形の周の長さと面積はどちらも24です。

したがって、$\frac{\text{area}}{\text{perimeter}}$ の比は1となります。

また、辺の長さが13、14、15の三角形を考えます。この三角形の周の長さは42で、面積は84です。

この三角形の場合、$\frac{\text{area}}{\text{perimeter}}$ の比は2となります。

面積／周の比が1000を超えない正の整数となる、すべての整数辺の三角形の周の長さの合計を求めてください。
