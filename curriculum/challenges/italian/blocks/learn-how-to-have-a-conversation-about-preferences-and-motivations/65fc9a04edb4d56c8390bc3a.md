# --description--

Un `glitch` è un piccolo problema che impedisce a qualcosa di funzionare correttamente. Come quando un gioco sul tuo telefono smette improvvisamente di funzionare, quello è un glitch. Per parlare di più di uno, diciamo `glitches`, come `My computer had a few glitches`.
