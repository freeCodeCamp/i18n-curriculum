# --description--

Der Übersichtlichkeit halber versehen wir beide Wells mit ihren IDs.

Fügen Sie über Ihrem linken Well, innerhalb seines `col-xs-6` `div`-Elements, ein `h4`-Element mit dem Text `#left-well` hinzu.

Fügen Sie über Ihrem rechten Well, innerhalb seines `col-xs-6` `div`-Elements, ein `h4`-Element mit dem Text `#right-well` hinzu.

# --hints--

Sie sollten jedem Ihrer `h4`-Elemente ein `<div class="col-xs-6">`-Element hinzufügen.

```js
const columnSixes = document.querySelectorAll('.col-xs-6');
const columnSixOneChildren = columnSixes?.[0]?.querySelectorAll(`:scope ${'h4'}`);
assert.lengthOf(columnSixOneChildren,1); 

const columnSixTwoChildren = columnSixes?.[1]?.querySelectorAll(`:scope ${'h4'}`);
assert.lengthOf(columnSixTwoChildren,1); 
```

Ein `h4`-Element sollte den Text `#left-well` enthalten.

```js
const firstH4 = document.querySelectorAll('h4')?.[0]; 
assert.match(firstH4?.textContent,/#left-well/gi); 
```

Ein `h4`-Element sollte den Text `#right-well` enthalten.

```js
const secondH4 = document.querySelectorAll('h4')?.[1]; 
assert.match(secondH4?.textContent,/#right-well/gi); 
```

Alle Ihre `h4`-Elemente sollten schließende Tags haben.

```js
assert.match(code,/<\/h4>/g);
assert.match(code,/<h4/g);
assert.equal(code.match(/<\/h4>/g).length , code.match(/<h4/g).length);
```
