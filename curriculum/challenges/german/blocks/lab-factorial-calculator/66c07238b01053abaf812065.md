# --description--

Eine Fakultät ist das Produkt einer ganzen Zahl und aller ganzen Zahlen darunter. Zum Beispiel ist die Fakultät von `5` `5 * 4 * 3 * 2 * 1 = 120`. In diesem Labor erstellen Sie einen Fakultätsrechner, der eine Zahl vom Benutzer entgegennimmt und die Fakultät dieser Zahl berechnet.

**Ziel:** Erfüllen Sie die untenstehenden User Stories und bestehen Sie alle Tests, um das Labor abzuschließen.

**User Stories:**

1. Sie sollten eine Variable `num` deklarieren und ihr eine Zahl Ihrer Wahl zuweisen. Die zugewiesene Zahl sollte inklusive zwischen `1` und `20` liegen.
1. Erstellen Sie eine Funktion namens `factorialCalculator`, die eine Zahl als Argument entgegennimmt und die Fakultät dieser Zahl zurückgibt.
1. Deklarieren Sie innerhalb der Funktion eine `result`-Variable und weisen Sie ihr den Wert `1` zu. Verwenden Sie eine Schleife, um alle Zahlen von `1` bis zur Eingabezahl (inklusive) zu durchlaufen, und multiplizieren Sie für jede Zahl die `result`-Variable mit der aktuellen Zahl und weisen Sie das Ergebnis der `result`-Variable zu. Sie können hier entweder eine `for`-Schleife, `while`-Schleife oder `do...while`-Schleife verwenden.
1. Sie sollten die `factorialCalculator`-Funktion mit `num` als Argument aufrufen und das Ergebnis der Variable `factorial` zuweisen.
1. Sie sollten die endgültige Ausgabe im Format `Factorial of [num] is [factorial]` speichern und der Variable `resultMsg` zuweisen.
1. Sie sollten den Wert von `resultMsg` in der Konsole ausgeben.

# --solutions--

```js
const num = 5;

function factorialCalculator(n) {
  let result = 1;

  for (let i = 2; i <= n; i++) {
    result *= i;
  }

  return result;
}

const factorial = factorialCalculator(num);
const resultMsg = `Factorial of ${num} is ${factorial}`;
console.log(resultMsg);
```
