# --description--

문자열과 숫자가 기본 구성 요소일 수 있지만, 스크립트가 복잡해질수록 많은 양을 다룰 방법이 필요합니다. 다행히 JavaScript에는 이를 위해 사용되는 몇 가지 데이터 유형이 있습니다. 배열은 항목(문자열, 숫자 또는 기타 것들)의 순서 있는 모음입니다.

배열은 여러 값을 하나의 변수에 저장하는 방법입니다. 배열은 길이 속성과 일련의 번호가 매겨진 속성을 가진 특별한 유형의 객체입니다. 각 번호가 매겨진 속성을 요소라고 하며, 각 요소는 어떤 유형의 값도 저장할 수 있습니다.

배열의 예는 다음과 같습니다:

```javascript
const fruits = ['apple', 'banana', 'orange'];
```
