# --description--

引数 `list` にはクエリセレクターの結果を渡そうと思います。クエリセレクターは `NodeList` を返します。 `NodeList` は要素のリストで、配列と似ています。 これにはクエリセレクターに一致した要素が含まれます。 このリスト内の要素をループで処理する必要があります。

以前のステップで、`for` ループを使って配列をループする方法を学びました。 配列や `NodeList` をループするには <dfn>for...of</dfn> ループを使うこともできます。

`for...of` ループは、配列のような反復可能オブジェクト内の要素を反復処理するために使用されます。 ループ構文で宣言される変数は、反復処理の各回における現在の要素を表します。

```js
for (const element of elementArray) {
  console.log(element);
}
```

Create a `for...of` loop that loops through the `list`. ループの変数名として、`const` を使用して `item` という変数を宣言してください。
