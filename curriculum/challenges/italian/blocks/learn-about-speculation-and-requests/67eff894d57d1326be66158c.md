# --explanation--

`Could it have been...?` è un modo per chiedere se qualcosa potrebbe essere successo.

`Could` esprime possibilità.

`Have been` si usa per parlare di qualcosa che potrebbe essere successo nel passato.

Jessica sta ipotizzando o suggerendo una possibilità. Un altro esempio:

`Could it have been the update that broke the app?` - Qualcuno sta chiedendo se l'aggiornamento ha fatto smettere di funzionare l'app.
