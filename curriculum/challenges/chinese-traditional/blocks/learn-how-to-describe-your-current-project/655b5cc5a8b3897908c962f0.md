# --description--

`Cybersecurity` 是關於保護電腦和線上資訊安全。它阻止壞人竊取或破壞電腦和網際網路上的內容。 

例如，當你使用密碼來保護你的電子郵件，或當網站保護你的信用卡資訊安全時，那就是 `cybersecurity`。這很重要，因為人們經常使用電腦和網際網路。
