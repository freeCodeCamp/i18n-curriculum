# --description--

매개변수 $\{ (x_0,y_0,z_0), (dx,dy,dz) \}$로 지정된 축에 정렬된 직육면체는 $x_0 ≤ X ≤ x_0 + dx$, $y_0 ≤ Y ≤ y_0 + dy$, $z_0 ≤ Z ≤ z_0 + dz$를 만족하는 모든 점 ($X$,$Y$,$Z$)으로 구성됩니다. 직육면체의 부피는 곱 $dx × dy × dz$입니다. 여러 직육면체의 결합 부피는 이들의 합집합 부피이며, 직육면체들이 겹치는 부분이 있으면 개별 부피의 합보다 작습니다.

$C_1, \ldots, C_{50000}$을 50000개의 축에 정렬된 직육면체 모음이라 할 때, $C_n$의 매개변수는 다음과 같습니다.

$$\begin{align}
  & x_0 = S_{6n - 5} \\; \text{나머지 연산} \\; 10000    \\\\
  & y_0 = S_{6n - 4} \\; \text{나머지 연산} \\; 10000    \\\\
  & z_0 = S_{6n - 3} \\; \text{나머지 연산} \\; 10000    \\\\
  & dx = 1 + (S_{6n - 2} \\; \text{나머지 연산} \\; 399) \\\\
  & dy = 1 + (S_{6n - 1} \\; \text{나머지 연산} \\; 399) \\\\
  & dz = 1 + (S_{6n} \\; \text{나머지 연산} \\; 399)     \\\\
\end{align}$$

여기서 $S_1, \ldots, S_{300000}$은 "지연 피보나치 생성기"에서 나옵니다:

$1 ≤ k ≤ 55$일 때, $S_k = [100003 - 200003k + 300007k^3] \\; (나머지 연산 \\; 1000000)$

$56 ≤ k$일 때, $S_k = [S_{k - 24} + S_{k - 55}] \\; (나머지 연산 \\; 1000000)$

따라서 $C_1$의 매개변수는 $\{(7,53,183), (94,369,56)\}$이고, $C_2$는 $\{(2383,3563,5079), (42,212,344)\}$이며, 계속 이어집니다.

처음 100개의 직육면체 $C_1, \ldots, C_{100}$의 결합 부피는 723581599입니다.

모든 50000개의 직육면체 $C_1, \ldots, C_{50000}$의 결합 부피는 얼마인가요?
