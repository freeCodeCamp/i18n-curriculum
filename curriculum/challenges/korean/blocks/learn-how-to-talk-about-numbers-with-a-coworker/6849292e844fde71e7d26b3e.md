# --explanation--

`Go through`는 무언가를 단계별로 검토하는 것을 의미합니다. 예를 들어:

`She will go through the design with the client tomorrow.` — 그녀는 클라이언트와 디자인을 검토하고 논의할 것입니다.

Sophie가 말하는 것은 소프트웨어 도구 목록을 검토하고, 여전히 사용 중이거나 업데이트가 필요한지 확인해야 한다는 뜻입니다.
