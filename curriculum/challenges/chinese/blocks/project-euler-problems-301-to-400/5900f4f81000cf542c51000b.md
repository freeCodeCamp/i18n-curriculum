# --description--

对于任意正整数 $n$，第 $n$ 个弱 Goodstein 序列 $\\{g1, g2, g3, \ldots\\}$ 被定义为：

- $g_1 = n$
- 对于 $k > 1$，$g_k$ 是通过将 $g_{k - 1}$ 写成以 $k$ 为底的数，解释为以 $k + 1$ 为底的数，然后减去 1 得到的。

当 $g_k$ 变为 0 时，序列终止。

例如，第 6 个弱 Goodstein 序列是 $\\{6, 11, 17, 25, \ldots\\}$：

- $g_1 = 6$。
- $g_2 = 11$ 因为 $6 = 110_2$，$110_3 = 12$，且 $12 - 1 = 11$。
- $g_3 = 17$，因为 $11 = 102_3$，$102_4 = 18$，且 $18 - 1 = 17$。
- $g_4 = 25$，因为 $17 = 101_4$，$101_5 = 26$，且 $26 - 1 = 25$。

等等。

找到ΣG（n）的最后9位数，1≤n&lt;16。

令 $G(n)$ 为第 $n$ 个弱 Goodstein 序列中非零元素的数量。

可以验证 $G(2) = 3$、$G(4) = 21$ 和 $G(6) = 381$。

也可以验证当 $1 ≤ n &lt; 8$ 时，$\sum G(n) = 2517$。

查找 $\sum G(n)$ 在 $1 ≤ n &lt; 16$ 时的最后 9 位数字。
