# --description--

El elemento `em` hace el texto en cursiva. También coloca semánticamente énfasis en el texto, lo cual nuevamente puede afectar cosas como los lectores de pantalla. Para definir un elemento enfatizado, envuelve el contenido del texto en una etiqueta `<em>`.
