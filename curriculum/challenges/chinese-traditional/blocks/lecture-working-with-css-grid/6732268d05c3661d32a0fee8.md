# --interactive--

在之前的課程中，我們一直在使用 `grid-template-columns` 屬性並將值設為幾個分數單位。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

雖然以下的程式碼完全有效，但有更簡單的方法可以重複你音軌列表中的部分或全部內容。

`repeat()` 函式用於重複某個區段或所有行或列的軌道。此函式接受一個重複次數以及你希望重複的軌道。

這裡是使用 `repeat` 函式（程式）的先前範例的修訂版本：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

瀏覽器中的樣式不會改變，但這是為 `columns` 重複值撰寫更簡潔的方式。

`repeat()` 函式（程式）會接受你可以用於列或行的任何有效樣式。

這裡有一個範例，使用 `repeat()` 函式（程式）將第一和第三行設定為 20 像素，第二和第四行設定為一個分數單位。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(2, 20px 1fr);
  column-gap: 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

有時候，你可能會選擇寫出每個單獨的值，而不是使用 `repeat()` 函式（程式）。但當你想要為曲目列表重複特定樣式時，這個函式（程式）特別有用。
