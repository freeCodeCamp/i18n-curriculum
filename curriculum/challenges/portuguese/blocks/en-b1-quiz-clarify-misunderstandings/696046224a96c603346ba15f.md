# --description--

Este quiz verifica seu entendimento sobre a linguagem usada para esclarecer ideias, evitar confusões, verificar informações e explicar conceitos técnicos.

Para passar no quiz, você deve responder corretamente pelo menos 18 das 20 perguntas abaixo.

Leia cada pergunta e escolha a resposta correta. Há apenas uma resposta correta para cada pergunta.

Para acessibilidade, transcrições estão disponíveis para alunos com deficiência auditiva; recomendamos usá-las apenas se necessário.
