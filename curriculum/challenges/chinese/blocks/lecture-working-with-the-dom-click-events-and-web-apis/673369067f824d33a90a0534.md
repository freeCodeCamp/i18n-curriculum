# --interactive--

当用户点击按钮或表单发生变化时，这称为事件。在你的程序中，你需要有一种方法来监听这些事件并对其作出响应。

`addEventListener()` 方法用于监听事件。它接受两个参数：你想监听的事件和事件发生时将被调用的函数。

这是基本语法：

```js
element.addEventListener("event", listener);
```

元素是要监视事件的超文本标记语言元素，event 指定要监听的事件类型，例如 `"click"`。

监听器是一个对象，当事件发生时它将接收通知。大多数情况下，这将是你定义的用于处理该事件的函数。

这是一个示例：

```js
element.addEventListener("click", () => {
  // code to run when the click event occurs
});
```

你也可以选择传入一个函数引用，像这样：

```js
function handleClick() {
  // code to run when the click event occurs
}

element.addEventListener("click", handleClick);
```

有时你可能想创建一个单独的函数来处理该事件。这可以使你的代码更易于读取和维护。

监听器参数也可以是 `null`，或者可以是使用 `EventListener` 接口的对象。`EventListener` 接口定义了一个名为 `handleEvent()` 的单一方法。每当你监听的事件发生时，该方法会被自动调用。使用此接口允许同一个对象在需要时处理多个事件。

大多数情况下，你不需要使用此接口，而是可以传入一个在事件发生时会被调用的函数。

现在，让我们看一个示例，以更好地理解它是如何工作的：

在此示例中，我们有一个 id 为 `btn` 的 `button` 元素：

```html
<button id="btn">Show alert</button>
```

我们想监听此按钮上的 `"click"` 事件，并在按钮被点击时显示警告。

我们首先需要在我们的 JavaScript 代码中访问那个 `button` 元素。然后我们需要为点击事件添加一个事件监听器，当按钮被点击时显示一个警告：

:::interactive_editor

```html
<button id="btn">Show alert</button>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("btn");

btn.addEventListener("click", () => alert("You clicked the button"));
```

:::

现在每次点击按钮时，用户将在屏幕上看到如下显示的警告消息：

```md
You clicked the button
```

你可以监听的另一种事件类型是 `"input"` 事件。当输入元素的值发生变化时，会触发此事件。

这是一个示例：

```html
<input type="text" id="input" placeholder="Type something" />
```

我们想监听此 `input` 元素上的 `input` 事件，并在用户每次输入内容时将输入的值记录到控制台。

我们首先需要在我们的 JavaScript 代码中访问那个 `input` 元素。然后我们需要添加一个事件监听器来监听 `input` 事件，并在每次用户输入内容时将输入的值记录到控制台：

:::interactive_editor

```html
<input type="text" id="input" placeholder="Type something" />
<script src="index.js"></script>
```

```js
const input = document.getElementById("input");

input.addEventListener("input", () => {
  console.log(input.value);
});
```

:::

你可以使用 `addEventListener()` 方法监听更多的事件。常见的事件包括 `mouseover`、`mouseout`、`keydown`、`keyup` 和 `submit`。

在以后的课程中，我们将介绍更多事件以及如何使用 `addEventListener()` 方法来监听它们。
