# --explanation--

`Could it have been...?` es una forma de preguntar si algo podría haber ocurrido.

`Could` expresa posibilidad.

`Have been` se usa para hablar de algo que posiblemente ocurrió en el pasado.

Jessica está adivinando o sugiriendo una posibilidad. Otro ejemplo:

`Could it have been the update that broke the app?` - Alguien está preguntando si la actualización causó que la app dejara de funcionar.
