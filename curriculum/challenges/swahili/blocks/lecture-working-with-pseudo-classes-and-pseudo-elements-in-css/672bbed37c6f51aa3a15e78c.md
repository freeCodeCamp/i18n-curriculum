# --interactive--

Moja ya mambo ya kuvutia zaidi kuhusu CSS ni matumizi ya pseudo-element. Katika muktadha huu, "pseudo" inamaanisha "si halisi", kwa hivyo pseudo-element ni vipengele vya mtandao au bandia ambavyo havilingani moja kwa moja na vipengele halisi vya HTML. Huvutia mtindo kwa sehemu maalum za kipengele au kuingiza maudhui bila kuongeza HTML zaidi.

Ili kutumia pseudo-element, iambatisha kwenye kichaguzi cha kipengele cha asili kwa kutumia nukta mbili (`::`). Kumbuka kwamba kichaguzi kinaweza kuwa aina yoyote, kama vile kichaguzi cha darasa la vitu au ID. Hii ndiyo sintaksia ya msingi ya pseudo-element:

```css
selector::pseudo-element {
  property: value;
}
```

Nukta mbili hizi ndizo zinazotofautisha pseudo-element na darasa la bandia, ambalo hutumia nukta moja.

Pseudo-element hukuwezesha kupamba sehemu maalum za maudhui ya kipengele au kuingiza maudhui kabla au baada yake, lakini haiwezi kuwepo peke yake. Kipengele ambacho pseudo-element imeambatishwa nalo huitwa kipengele cha asili.

Tuanze kwa kuangalia mifano ya pseudo-element za `::before` na `::after`. Kama majina yao yanavyosema, `::before` hukuwezesha kuingiza maudhui kabla kabisa ya maudhui ya kipengele wakati `::after` hukuwezesha kuingiza maudhui baada yake.

Hapa kuna mfano wa kipengele cha kitufe. Katika CSS, tutatumia upangaji wa absolute na pseudo-element ya `::before` kuongeza nyota kabla ya maandishi ya `Learn More` ya kitufe. Utajifunza zaidi kuhusu upangaji wa absolute katika funzo zijazo.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: lightseagreen;
  color: white;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
  position: relative;
}

.cta-button::before {
  content: "⭐";
  position: absolute;
  left: 3px;
  top: 8px;
  font-size: 0.75rem;
}
```

:::

Sifa ya `content` hutumika kuwakilisha maudhui unayotaka kuongeza kabla ya maandishi ya kitufe. Katika mfano huu, tunaongeza nyota. Utaona kwamba si tu kuingiza maudhui bali pia unaweza kuyapamba. Hapa kuna mfano wa pseudo-element ya `::after` na kitufe kile kile:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: '➡️';
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}
```

:::

Katika pseudo-element ya `::after`, sifa ya `transition` hutumika kuendesha mabadiliko kwa muda wa sekunde 0.3 kwa athari ya laini, ikitengeneza mabadiliko ya taratibu badala ya ghafla. Utajifunza zaidi kuhusu sifa ya `transition` katika funzo zijazo.

Pia unaweza kuambatisha darasa la bandia kwa maudhui unayoingiza katika maudhui mengine kwa pseudo-element za `::before` na `::after`. Kwa mfano, hali ya hover kwa maudhui. Hapa kuna mfano:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: "➡️";
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}

.cta-button:hover::after {
  transform: translateX(2px);
}
```

:::

Kwa `transform: translateX(2px)` katika hali ya hover, maudhui yanasukumwa kwenda kulia kwa `2px` kila mtumizi anapobofya kitufe. Sifa ya transition katika `::after` yenyewe huhakikisha mchakato unachukua `0.3s`.

Hilo ndilo sifa ya `transform` – hukuwezesha kuzungusha, kupinda, kupanua, au kutafsiri kipengele katika mwelekeo fulani. Utajifunza zaidi kuhusu hilo katika funzo zijazo.

Katika mfano unaofuata, tutaangalia pseudo-element ya `::first-letter`. Pseudo-element ya `::first-letter` inalenga herufi ya kwanza ya maudhui ya kipengele, ikikuruhusu kuipamba. Hapa kuna mfano wa maandishi ya aya. Ikiwa tunataka kupamba herufi ya kwanza, tunaweza kutumia pseudo-element ya `::first-letter` kama ifuatavyo:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>freeCodeCamp lets you learn to code without having to pay.</p>
```

```css
p::first-letter {
  font-size: 4rem;
}
```

:::

Katika mfano wa mwisho, tutaangalia pseudo-element ya `::marker`, ambayo hukuwezesha kuchagua alama, risasi au nambari za vitu vya orodha kwa ajili ya kupamba. Pseudo-element ya `::marker` hutoa njia ya kuboresha utambulisho wa chapa ya tovuti yako kwa kubinafsisha alama za orodha ili ziendane na mpangilio wa rangi zako.

Hapa kuna mfano wa orodha isiyopangwa na orodha iliyopangwa. Ili kubadilisha rangi na ukubwa wa alama za vitu vya orodha, unaweza kutumia pseudo-element ya `::marker` kama ifuatavyo:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<ul>
  <li>Unordered list item 1</li>
  <li>Unordered list item 2</li>
  <li>Unordered list item 3</li>
  <li>Unordered list item 4</li>
</ul>

<ol>
  <li>Ordered list item 1</li>
  <li>Ordered list item 2</li>
  <li>Ordered list item 3</li>
  <li>Ordered list item 4</li>
</ol>
```

```css
li::marker {
  color: crimson;
  font-size: 1.5em;
  font-weight: bold;
}
```

:::

Katika funzo hili, tumegusia pseudo-element chache tu. Lakini kuna nyingi zaidi kama `::placeholder`, `::spelling-error` na `::selection` ambazo nakuhimiza uzichunguze mwenyewe.
