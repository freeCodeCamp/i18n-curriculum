# --description--

`log in` 意味着输入你的用户名和密码以访问计算机或网站。例如：

`Every morning, I log in to my computer to start work.`——这意味着每天早上你输入用户名和密码打开你的计算机并开始工作。

`check emails` 意味着查看并读取你邮箱账户中的新消息。例如：

`After logging in, I check my emails to see if I have any new messages.`——这意味着在你打开你的计算机或电子邮件账户后，你查看是否有人发送了新消息给你。
