# --explanation--

`User experience` (또는 `UX`)는 사용자가 제품을 사용할 때 얼마나 쉽고 즐겁고 원활한지에 관한 것입니다. 예를 들어:

`Improving page load time boosts user experience.` — 페이지 로딩 속도를 빠르게 하면 사용자가 앱을 더 즐길 수 있습니다.

`Enhancements`는 제품이나 과정을 개선하기 위해 이루어진 변경 사항입니다. 예를 들어:

`The latest update includes performance enhancements.` — 새 버전은 앱이 더 잘 작동하고 빨라지게 만듭니다.

Alice의 문장에서는 디자인 예산의 절반이 제품을 더 쉽고 좋게 만드는 데 쓰입니다.
