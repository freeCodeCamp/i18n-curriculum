# --interactive--

在之前的课程中，我们稍微讲了一下如何在网格项之间创建空间。但在本课中，我们将更详细地探讨如何在网格版型中使用 `row-gap`、`column-gap` 和 `gap` 属性。

但我们首先需要复习 什么是 CSS 网格中的轨道。 轨道是两个相邻网格线之间的空间。 当你使用 CSS 网格时，这些线会自动创建。 在这种情况下，轨道通常是指构成网格布局的行和列。

要在 CSS 网格中创建列之间的间隙，可以使用 `column-gap` 属性。 该属性可接受的值包括像素、`em` 单位、百分比或 `normal` 关键字。

如果你对 `column-gap` 属性使用 `normal` 值，那么结果将是，网格布局时其属性值是 `0`，多列布局是其属性值是 `1em`。

这是一个四列网格布局的示例：

```html
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

对于 CSS，我们将显示设置为 grid，并将 `column-gap` 属性设置为 10 像素：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  height: 100px;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.grid-container div {
  background-color: darkblue;
}
```

:::

如果我们想这个例子变为蓝色的两行，并在行之间创造更多的空间，我们可以使用 `row-gap` 属性：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  column-gap: 10px;
  row-gap: 30px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

在这个修改后的示例中，我们将 `row-gap` 属性设置为 30 像素，并将 `grid-template-columns` 更改为仅使用两个 `1fr` 单元而不是四个来创建两行。

就像 `column-gap` 属性一样，`row-gap` 属性的可接受值可以包括百分比、`em` 和像素。

如果希望使用一种简写方式来创建行与列之间的空间，可以使用 `gap` 属性。 这是基本语法：

```css
gap: row-value optional-column-value;
```

如果为 `gap` 属性指定一个值，那么该值将应用于行和列。 如果指定了两个值，那么第一个值将应用于行，第二个值将应用于列：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 30px 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

`gap` 简写属性的可接受值包括百分比、像素、`em` 甚至 `calc()` 值。 但你不能在这里使用单位 `fr`。

`row-gap`、`column-gap` 和 `gap` 属性为控制 CSS 网格布局中项目之间的间距提供了灵活的方法。 通过使用这些属性，你可以轻松创建具有行与列间一致且可调节间隙的视觉吸引力的网格布局。
