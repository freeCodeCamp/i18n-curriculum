## --text--

`Cursor` 인스턴스가 가진 메서드는 무엇이며, 무엇을 할 수 있나요?

## --answers--

`Cursor` 인스턴스는 SQL 쿼리를 실행할 수 있는 `.run()` 메서드를 가지고 있습니다.

---

`Cursor` 인스턴스는 레코드를 선택할 수 있는 `.select()` 메서드를 가지고 있습니다.

---

`Cursor` 인스턴스는 데이터베이스에 대해 실행할 SQL 매개변수를 받을 `.execute()` 메서드를 가지고 있습니다.
