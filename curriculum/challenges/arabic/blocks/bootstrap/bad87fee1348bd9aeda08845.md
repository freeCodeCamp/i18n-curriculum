# --description--

بما أن أصناف `col-xs-*` في Bootstrap تنطبق على جميع عناصر `form`، يمكنك استخدامها على مربعات الاختيار أيضًا! بهذه الطريقة، ستتوزع مربعات الاختيار بالتساوي عبر الصفحة، بغض النظر عن مدى عرض دقة الوضوح للشاشة.

# --instructions--

قم بتسكين جميع مربعات الاختيار الثلاثة داخل عنصر `<div class="row">`. ثم قم بتسكين كل منها داخل عنصر `<div class="col-xs-4">`.

# --hints--

يجب أن تكون جميع مربعات الاختيار مسكنة داخل `div` واحد يحمل الصنف `row`.

```js
const checkboxes = document.querySelectorAll(`div.row input[type="checkbox"]`); 
assert.lengthOf(checkboxes,3);
```

يجب أن يكون كل مربع اختيار مسكنًا داخل `div` خاص به يحمل الصنف `col-xs-4`.

```js
const checkboxes = document.querySelectorAll(`div.col-xs-4 input[type="checkbox"]`); 
assert.lengthOf(checkboxes,3);
```

يجب أن تحتوي جميع عناصر `div` على وسم الإغلاق.

```js
assert.match(code,/<\/div>/g);
assert.match(code,/<div/g);
assert.equal(code.match(/<\/div>/g).length ,code.match(/<div/g).length);
```
