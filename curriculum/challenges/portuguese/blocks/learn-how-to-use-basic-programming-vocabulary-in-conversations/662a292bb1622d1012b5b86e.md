# --description--

Aprenda a usar `how much` e `how many` para perguntar sobre quantidades. `How much` é usado com substantivos incontáveis para perguntar sobre o volume ou a extensão de algo, como `time`, `water` ou `money`.Por exemplo, `How much money do you have?` pergunta sobre a quantidade de dinheiro, que você não pode contar individualmente.

`How many` é usado com substantivos contáveis quando você pode contar os itens, como `books`, `apples` ou `people`.Por exemplo, `How many books do you own?` pergunta por um número contável de livros.
