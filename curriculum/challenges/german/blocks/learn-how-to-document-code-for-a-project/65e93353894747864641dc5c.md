# --description--

Die Präposition `on` wird häufig verwendet, um das Thema oder den Fokus von etwas anzugeben, also worauf man `on` in einem Projekt kommentieren soll. Zum Beispiel `The report focused on the project's progress.`

`Any` wird verwendet, um zu fragen, ob es ein oder mehrere Elemente gibt, ohne anzugeben, welche genau. So wie die Frage, ob es `any` Fragen gibt.
