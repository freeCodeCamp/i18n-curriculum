# --explanation--

`Soy uruguaya` означає "Я уругвайка".

Слово `uruguaya` — це жіночий варіант для людини з Уругваю.

В іспанській мові національності мають родові форми, тому, оскільки Хулієта — жінка, вона використовує `uruguaya` (а не `uruguayo`).
