# --interactive--

هناك طريقتان قويتان في JavaScript للتعامل مع بيانات JSON: `JSON.parse()` و `JSON.stringify()`. تُستخدم هاتان الطريقتان عادةً للتحويل بين سلاسل JSON وكائنات JavaScript.

تُستخدم `JSON.stringify()` لتحويل كائن JavaScript إلى سلسلة JSON. هذا مفيد عندما تريد تخزين البيانات أو نقلها بصيغة يمكن مشاركتها أو نقلها بسهولة بين الأنظمة.

إليك كيفية استخدام طريقة `JSON.stringify()`:

:::interactive_editor

```js
const user = {
  name: "John",
  age: 30,
  isAdmin: true
};

const jsonString = JSON.stringify(user);
console.log(jsonString);
```

:::

تقبل طريقة `JSON.stringify()` أيضًا معلمة اختيارية تُسمى replacer، والتي يمكن أن تكون دالة أو مصفوفة. إليك مثالًا على استخدام مصفوفة كمعلمة replacer اختيارية:

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

// result: {"firstName":"Jessica","country":"USA"}
console.log(JSON.stringify(developerObj, ["firstName", "country"]));
```

:::

في هذا المثال، لدينا `developerObj` بأربع خصائص. عند استخدام طريقة `JSON.stringify()`، يمكننا تمرير مصفوفة كمعلمة ثانية وتحديد الخصائص التي نريد تحويلها إلى سلسلة. ستكون النتيجة كائنًا محولًا إلى سلسلة يحتوي فقط على خاصيتي `firstName` و `country`.

معلمة اختيارية أخرى لطريقة `JSON.stringify()` هي معلمة spacer. تتيح لك هذه التحكم في المسافات البادئة للنتيجة المحولة إلى سلسلة:

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

console.log(JSON.stringify(developerObj, null, 2));

/* result
{
  "firstName": "Jessica",
  "isAwesome": true,
  "isMusician": true,
  "country": "USA"
}
*/
```

:::

في معظم الأحيان لن تستخدم أيًا من هاتين المعلمتين الاختياريتين لطريقة `JSON.stringify()`، لكن من المفيد أن تكون على علم بهما.

طريقة أخرى ستستخدمها كثيرًا في برمجتك هي طريقة `JSON.parse()`. تقوم `JSON.parse()` بتحويل سلسلة JSON مرة أخرى إلى كائن JavaScript. هذا مفيد عندما تسترجع بيانات JSON من خادم ويب أو من `localStorage` وتحتاج إلى معالجة البيانات في تطبيقك. ستتعلم المزيد عن `localStorage` في درس لاحق.

إليك مثالًا على كيفية العمل مع طريقة `JSON.parse()`:

:::interactive_editor

```js
const jsonString = '{"name":"John","age":30,"isAdmin":true}';
const userObject = JSON.parse(jsonString);
console.log(userObject);

// Result:
// { name: 'John', age: 30, isAdmin: true }
```

:::

هذا يسمح لك بالتعامل مع البيانات في برنامجك ككائن JavaScript عادي، مما يسهل التلاعب بها واستخدامها.

في الوحدات القادمة، سنواصل تعلم المزيد عن كيفية العمل مع JSON وطريقتي `JSON.parse()` و `JSON.stringify()`.
