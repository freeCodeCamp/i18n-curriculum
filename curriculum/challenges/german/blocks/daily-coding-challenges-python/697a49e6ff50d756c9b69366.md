# --description--

Gegeben sind der Höhenunterschied, die horizontale Distanz und der Typ eines Skihangs. Bestimmen Sie die Schwierigkeitsbewertung des Hangs.

Zur Bestimmung der Bewertung:

- Berechnen Sie die Steilheit des Hangs, indem Sie den Höhenunterschied durch die Distanz teilen.
- Berechnen Sie dann die angepasste Steilheit basierend auf dem Hangtyp:
  - `"Downhill"`: multiplizieren Sie die Steilheit mit 1,2
  - `"Slalom"`: multiplizieren Sie die Steilheit mit 0,9
  - `"Giant Slalom"`: multiplizieren Sie die Steilheit mit 1,0

Return:

- `"Green"`, wenn die angepasste Steilheit kleiner oder gleich 0,1 ist
- `"Blue"`, wenn die angepasste Steilheit größer als 0,1 und kleiner oder gleich 0,25 ist
- `"Black"`, wenn die angepasste Steilheit größer als 0,25 ist

```py
def get_hill_rating(drop, distance, hill_type):
    steepness = drop / distance

    if hill_type == "Downhill":
        steepness *= 1.2
    elif hill_type == "Slalom":
        steepness *= 0.9
    elif hill_type == "Giant Slalom":
        steepness *= 1.0

    if steepness <= 0.1:
        return "Green"
    elif steepness <= 0.25:
        return "Blue"
    else:
        return "Black"
```
