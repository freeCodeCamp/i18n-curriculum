# --explanation--

`you've done a great job` 中的 `Present Perfect` 時態是使用的，因為 Jessica 的工作從過去開始，並且在現在仍有相關性。這個時態將過去的動作與其當前的影響連結起來。另一個例子：

`She has improved the software significantly.`－這表示她的改進從過去開始，並且現在仍然重要。
