# --description--

Graphen können auf verschiedene Arten dargestellt werden. Hier beschreiben wir eine Methode, die als <dfn>Adjazenzliste</dfn> bezeichnet wird. Eine Adjazenzliste ist im Wesentlichen eine Aufzählungsliste, bei der auf der linken Seite der Knoten steht und auf der rechten Seite alle anderen Knoten aufgelistet sind, mit denen er verbunden ist. Unten sehen Sie eine Darstellung einer Adjazenzliste.

<blockquote>Knoten1: Knoten2, Knoten3<br>Knoten2: Knoten1<br>Knoten3: Knoten1</blockquote>

Oben ist ein ungerichteter Graph, weil `Node1` mit `Node2` und `Node3` verbunden ist, und diese Information mit den Verbindungen übereinstimmt, die `Node2` und `Node3` zeigen. Eine Adjazenzliste für einen gerichteten Graphen würde bedeuten, dass jede Zeile der Liste eine Richtung zeigt. Wenn das Obige gerichtet wäre, würde `Node2: Node1` bedeuten, dass die gerichtete Kante von `Node2` zu `Node1` zeigt. Wir können den oben gezeigten ungerichteten Graphen als Adjazenzliste darstellen, indem wir ihn in einem JavaScript-Objekt unterbringen.

```js
var undirectedG = {
  Node1: ["Node2", "Node3"],
  Node2: ["Node1"],
  Node3: ["Node1"]
};
```

Dies kann auch einfacher als Array dargestellt werden, bei dem die Knoten nur Zahlen anstelle von String-Bezeichnungen haben.

```js
var undirectedGArr = [
  [1, 2], // Node1
  [0],    // Node2
  [0]     // Node3
];
```

# --instructions--

Erstellen Sie ein soziales Netzwerk als ungerichteten Graphen mit 4 Knoten/Personen namens `James`, `Jill`, `Jenny` und `Jeff`. Es gibt Kanten/Beziehungen zwischen James und Jeff, Jill und Jenny sowie Jeff und Jenny.

# --hints--

```js
assert(Object.keys(undirectedAdjList).length === 4);
```

```js
assert(
  undirectedAdjList.James.indexOf('Jeff') !== -1 &&
    undirectedAdjList.Jeff.indexOf('James') !== -1
);
```

);
