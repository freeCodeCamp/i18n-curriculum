# --interactive--

讓我們看看你如何使用 `innerHTML` 和 `createElement()` 創建節點。

我們將從 `innerHTML` 開始。

`innerHTML` 是 `Element` 物件的一個屬性，你可以用它來設定它們的 HTML 標記。使用 `innerHTML`，你可以用字串設定現有元素的 HTML 結構，建立所有必要的節點。

這是一個範例。我們有一個空的 `div`，並且將使用 `innerHTML` 屬性在其中添加一些元素：

```html
<div id="container">
  <!-- Add new elements here -->
</div>
```

首先，我們需要透過其 ID 選取元素：

```js
const container = document.getElementById("container");
```

然後，我們將容器的 `innerHTML` 屬性設定為一個字串。這個字串必須包含所有用來表現你想要創建的節點和結構的標記。你可以將它視為在字串中撰寫 HTML。

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
container.innerHTML = "<ul><li>Cheese</li><li>Tomato</li></ul>";
```

:::

我們將把此元素的 `innerHTML` 設定為披薩配料（起司和番茄）的無序列表。

如果能讓標記更容易理解，你也可以將字串寫在多行。重要的是標記必須準確表述你想要創建的 HTML 結構。

在執行這段程式碼後，如果你檢查你的標記，將會看到以下的 HTML 結構。 

```html
<div id="container">
  <ul>
    <li>Cheese</li>
    <li>Tomato</li>
  </ul>
</div>
```

新的節點在字串被解析後動態地創建並添加到 DOM。

`innerHTML` 在某些情境下非常有用。然而，你應該熟悉它存在的一些安全風險。如果你無法完全控制該字串，就不應該使用它。 

例如，如果字串將由使用者輸入，你不應該使用 `innerHTML`，因為使用者可能會將惡意內容插入你的網站。因此，通常建議改用 `textContent` 來插入純文字。

另一種創建新節點的方法是使用 `createElement()` 行為。透過這個新行為，你可以透過指定其標記名稱來創建新元素。 

例如，如果你想創建一個影像元素，當呼叫這個方法時，你會將 `img` 頁籤作為字串傳入：

```js
document.createElement("img");
```

而且你可以將這個新物件指定給一個變數：

```js
const img = document.createElement("img");
```

如果 `document` 是 `HTMLDocument`，則 `createElement()` 方法會傳回新的 `HTMLElement` 物件。否則，會傳回 `Element` 物件。

當你準備好這個新元素後，你可以使用 `appendChild()` 方法將它作為另一個現有元素的子元素添加到 DOM，或者你可以使用其他方法將它插入到特定位置。選擇最符合你需求的方法。

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
const img = document.createElement("img");
img.src = "https://cdn.freecodecamp.org/curriculum/cat-photo-app/lasagna.jpg";
img.alt = "A slice of lasagna on a plate.";
container.appendChild(img);
```

:::

使用 `innerHTML` 和 `createElement()` 創建節點可讓你動態操作網站的結構和內容。透過結合這些技術，你可以建置互動式網頁應用程式。
