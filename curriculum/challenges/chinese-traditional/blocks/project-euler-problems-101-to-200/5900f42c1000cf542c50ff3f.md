# --description--

設 $x$ 為實數。

對於分母界限 `d`，對 $x$ 的最佳近似值是一個約分後的有理數 $\frac{r}{s}$，其中 $s ≤ d$，使得任何比 $\frac{r}{s}$ 更接近 $x$ 的有理數，其分母都大於 `d`：

$$|\frac{p}{q} - x| &lt; |\frac{r}{s} - x| ⇒ q > d$$

例如，分母界限為 20 時，對 $\sqrt{13}$ 最佳的近似值是 $\frac{18}{5}$，分母界限為 30 時，對 $\sqrt{13}$ 最佳的近似值是 $\frac{101}{28}$。

找出分母上限為 ${10}^{12}$ 時，對 $\sqrt{n}$ 的最佳近似分母總和，其中 $n$ 不是完全平方數且 $1 < n ≤ 100000$。
