# --description--

La palabra `super` se usa a menudo informalmente antes de un adjetivo para mejorar su significado, similar a `very` o `extremely`, para mejorar el significado del adjetivo que sigue.

Por ejemplo, decir `this book is super interesting` significa que el libro es extremadamente interesante, más que simplemente interesante.

`Helpful` describe a alguien o algo que proporciona asistencia o apoyo valioso. Juntos, `super helpful` significa proporcionar una asistencia excepcional o ser extremadamente útil.
