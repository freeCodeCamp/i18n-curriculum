# --interactive--

У JavaScript шаблонні літерали — це потужний і гнучкий спосіб працювати з рядками. На відміну від звичайних рядків, які використовують одинарні (`'`) або подвійні (`"`) лапки, шаблонні літерали визначаються за допомогою поділок (`` ` ``).

Вони дозволяють легше маніпулювати рядками, зокрема вставляти змінні безпосередньо всередину рядка — це функція, відома як інтерполяція рядків.

Шаблонні літерали полегшують створення рядків, які розтягуються на кілька ліній або містять вирази (наприклад, змінні чи навіть код JavaScript) безпосередньо в рядку.

Ось приклад шаблонного літерала:

:::interactive_editor

```js
const name = "Alice";
const greeting = `Hello, ${name}!`;

console.log(greeting);
```

:::

Зверніть увагу на використання поділок замість одинарних або подвійних лапок. Синтаксис `${name}` є прикладом інтерполяції рядків, де значення змінної `name` безпосередньо вставляється в рядок.

Інтерполяція рядків дозволяє вставляти змінні та вирази всередину рядка. Це суттєве покращення порівняно зі старим методом, коли ви об’єднували рядки та змінні за допомогою оператора `+`.

Ось приклад із конкатенацією рядків і оператором плюс (`+`):

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = "My name is " + name + " and I am " + age + " years old.";
console.log(message); 
```

:::

А ось приклад із використанням шаблонних літералів та інтерполяції рядків:

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = `My name is ${name} and I am ${age} years old.`;
console.log(message); 
```

:::

Як бачите, інтерполяція рядків із шаблонними літералами значно чистіша і легша для читання, особливо коли ви працюєте з кількома змінними.

Ще одна чудова функція шаблонних літералів — підтримка багаторядкових рядків. У звичайних рядках потрібно використовувати символи екранування (`\n`), щоб створити нові лінії. У шаблонних літералах ви просто пишете рядок на кількох лініях, і форматування зберігається:

:::interactive_editor

```js
let poem = `Roses are red,
Violets are blue,
JavaScript is fun,
And so are you.`;

console.log(poem);
```

:::

Ще одна функція шаблонних літералів — можливість вставляти вирази JavaScript безпосередньо в рядок, як у цьому прикладі:

:::interactive_editor

```js
const song = "Bohemian Rhapsody";
const score = 9.5;
const highestScore = 10;
const output = `One of my favorite songs is "${song}". I rated it ${
  (score / highestScore) * 100
}%.`;
console.log(output); 
```

:::

Шаблонні літерали особливо корисні, коли потрібно включити змінні або вирази в рядки, форматувати складні рядки або працювати з багаторядковим текстом. Вони більш лаконічні та читабельні порівняно з традиційною конкатенацією рядків.
