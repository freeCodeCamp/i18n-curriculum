# --interactive--

CSS에서는 이미지 주변에 테두리를 추가하는 여러 가지 방법이 있으며, 각각은 다양한 스타일 옵션과 제어 수준을 제공합니다.

가장 일반적이고 다용도인 몇 가지 방법을 살펴보겠습니다.

이미지에 테두리를 추가하는 가장 간단한 방법은 `border` 속성을 사용하는 것입니다. 이 속성은 테두리의 너비, 스타일, 색상을 한 번에 설정할 수 있는 축약형입니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

이 CSS 규칙은 모든 `img` 요소 주위에 2픽셀 너비의 실선 빨간색 테두리를 추가합니다. 너비, 스타일(`dashed`, `dotted`, 또는 `double` 등), 색상을 디자인에 맞게 조절할 수 있습니다.

테두리의 각 면을 개별적으로 더 세밀하게 제어해야 한다면, 각 면에 해당하는 특정 테두리 속성을 사용할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border-top: 10px solid red;
  border-right: 10px dashed green;
  border-bottom: 10px dotted blue;
  border-left: 10px double purple;
}
```

:::

이렇게 하면 이미지 각 면에 독특한 테두리 스타일을 만들 수 있습니다.

또 다른 테두리 효과를 만드는 방법은 `outline` 속성을 사용하는 것입니다. border와 비슷하지만 outline은 요소의 크기나 레이아웃에 영향을 주지 않습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  outline: 3px solid gold;
}
```

:::

이것은 이미지 주위에 크기를 변경하거나 위치를 바꾸지 않고 금색 윤곽선을 만듭니다.

테두리에 둥근 모서리를 만들고 싶다면, border와 함께 `border-radius` 속성을 사용할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

이 기법들은 조합하여 독특한 테두리 효과를 만들 수 있습니다. 어떤 방법을 선택할지는 특정 디자인 요구사항과 필요한 복잡성 수준에 따라 달라집니다.
