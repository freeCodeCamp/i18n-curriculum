# --description--

Eine weitere nützliche Array-Methode ist die `filter()`-Methode. Die `filter()`-Methode erstellt ein neues Array mit allen Elementen, die den Test bestehen, der durch die bereitgestellte Funktion implementiert wird. Die Syntax der `filter()`-Methode ist wie folgt:

```javascript
const numbers = [2, 5, 6, 1, 9, -1]

const newNumbers = numbers.filter((number) => {
  return number > 2;
});

console.log(newNumbers); // Output: [5, 6, 9]
```

Die `filter()`-Methode erstellt ein neues Array mit allen Elementen, die größer als 2 sind. Die `filter()`-Methode ändert das ursprüngliche Array nicht.
