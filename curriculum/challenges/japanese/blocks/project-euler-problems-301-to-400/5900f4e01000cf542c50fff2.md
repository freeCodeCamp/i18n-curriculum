# --description--

オレゴン州のライセンスプレートは、3文字のアルファベットの後に3桁の数字が続きます（各数字は[0...9]の範囲です）。

セスは通勤中に次のゲームをしています。

彼の旅の途中で見た2つのライセンスプレートの数字の合計が1000になると勝ちです。

例えば、`MIC-012` と `HAN-988` は勝ちで、`RYU-500` と `SET-500` も同様です（同じ旅の中で見た場合に限ります）。

勝つために必要なライセンスプレートの期待値を求めてください。答えは小数点以下8桁まで四捨五入してください。

**注意:** 見るライセンスプレートの数字は、どの3桁の数字も同じ確率で現れると仮定します。
