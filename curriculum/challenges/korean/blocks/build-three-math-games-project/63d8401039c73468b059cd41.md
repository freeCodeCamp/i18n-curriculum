이 도전 과제에서는 Python을 사용해 다음 세 가지 수학 게임을 만들어야 합니다:

- 산점도 게임
  - 그래프에 무작위로 점을 생성하고 플레이어가 (x,y) 좌표를 입력해야 합니다
  - 난이도를 높이려면 그래프 크기를 키우세요
- 대수 연습 게임
  - 무작위 정수 값으로 한 단계 및 두 단계 문제를 생성하고 플레이어가 답을 입력해야 합니다
  - 양수와 음수를 사용하세요. 난이도를 높이려면 숫자를 더 크게 만드세요
- 투사체 게임
  - 무작위 높이와 위치의 "벽"을 표시합니다. 플레이어가 슬라이더를 움직여 포물선 경로를 조정해 벽을 넘겨야 합니다
  - 난이도를 높이려면 슬라이더 없이 a, b, c 값을 직접 입력하는 두 번째 단계를 만드세요
