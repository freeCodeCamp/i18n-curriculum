# --description--

Sei d(`n`) definiert als die Summe der echten Teiler von `n` (Zahlen kleiner als `n`, die `n` ohne Rest teilen).

Wenn d(`a`) = `b` und d(`b`) = `a` gilt, wobei `a` ≠ `b` ist, dann sind `a` und `b` ein freundschaftliches Paar, und sowohl `a` als auch `b` werden freundschaftliche Zahlen genannt.

Zum Beispiel sind die echten Teiler von 220 1, 2, 4, 5, 10, 11, 20, 22, 44, 55 und 110; daher ist d(220) = 284. Die echten Teiler von 284 sind 1, 2, 4, 71 und 142; also ist d(284) = 220.

Berechnen Sie die Summe aller freundschaftlichen Zahlen unter `n`.

```js
const sumAmicableNum = (n) => {
  const fsum = (n) => {
    let sum = 1;
    for (let i = 2; i <= Math.floor(Math.sqrt(n)); i++)
      if (Math.floor(n % i) === 0)
        sum += i + Math.floor(n / i);
    return sum;
  };
  let d = [];
  let amicableSum = 0;
  for (let i=2; i<n; i++) d[i] = fsum(i);
  for (let i=2; i<n; i++) {
    let dsum = d[i];
    if (d[dsum]===i && i!==dsum) amicableSum += i+dsum;
  }
  return amicableSum/2;
};
```
