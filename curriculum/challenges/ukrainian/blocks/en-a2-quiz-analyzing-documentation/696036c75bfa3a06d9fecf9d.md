# --description--

Цей тест перевіряє ваше розуміння загальних виразів, які використовуються для аналізу документації коду.

Щоб пройти тест, ви повинні правильно відповісти щонайменше на 9 із 10 питань нижче.

Прочитайте кожне запитання та виберіть правильну відповідь. Для кожного запитання є лише одна правильна відповідь.

Для доступності транскрипти доступні для учнів із порушеннями слуху; рекомендуємо використовувати їх лише за потреби.
