# --description--

你的酒店反馈表还应允许用户对食物进行评分。

首先添加一个 `label` 元素，其文本为 `How was the food?`。 该 `label` 元素应将 `for` 属性设置为 `"food"`。

在你的 `label` 元素下方，添加一个 `select` 元素，`id` 和 `name` 设置为 `"food"`。

# --hints--

你应将 `label` 元素的 `for` 属性设置为 `"food"`。

```js
assert.exists(document.querySelector('label[for="food"]'));
```

你的 `label` 应包含文本 `How was the food?`。

```js
assert.strictEqual(document.querySelector('label[for="food"]')?.textContent.trim(), 'How was the food?');
```

你应将 `label` 下方的 `select` 元素的 `id` 设置为 `"food"`。

```js
assert.exists(document.querySelector('label + select#food'));
```

你的 `select` 元素应将 `name` 属性设置为 `"food"`。

```js
assert.exists(document.querySelector('label + select[name="food"]'));
```
