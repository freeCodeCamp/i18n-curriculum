# --description--

你的迭代語句也會有所不同。 你不需要在每次循環時將 `1` 添加到 `i`，而是需要減去 `1`。

就像你之前對 `i = i + 1` 所做的那樣，更新迭代語句，賦予 `i` 從其自身中減去 `1` 的值。

# --hints--

你的 `for` 循環應使用 `i = i - 1` 作爲迭代。

