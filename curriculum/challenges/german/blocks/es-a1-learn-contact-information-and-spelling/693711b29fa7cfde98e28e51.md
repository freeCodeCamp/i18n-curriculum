# --description--

Wenn Sie persönliche Informationen wie eine Telefonnummer angeben, können Sie diese Struktur verwenden:

`Mi` + Substantiv + `es ...`

`Mi` zeigt Besitz an und bezieht sich auf „mein“. Zum Beispiel:

- `Mi nombre es Ana.`

- `Mi correo es ana@ejemplo.com.`

- `Mi número es 5551234.`

In dieser Aufgabe hören Sie eine Telefonnummer, die laut ausgesprochen wird.
