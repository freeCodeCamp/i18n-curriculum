# --description--

3 x 2 x 1 크기의 직육면체에서 모든 보이는 면을 덮기 위한 최소 큐브 개수는 스물두 개입니다.

<img alt="3x2x1 직육면체를 스물두 개의 1x1x1 큐브로 덮음" src="https://cdn.freecodecamp.org/curriculum/project-euler/cuboid-layers.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

이 입체에 두 번째 층을 추가하면 모든 보이는 면을 덮기 위해 마흔여섯 개의 큐브가 필요하고, 세 번째 층은 일흔여덟 개, 네 번째 층은 백열여덟 개의 큐브가 필요합니다.

하지만 5 x 1 x 1 크기의 직육면체 첫 번째 층도 스물두 개의 큐브가 필요합니다. 마찬가지로 5 x 3 x 1, 7 x 2 x 1, 11 x 1 x 1 크기의 직육면체 첫 번째 층은 모두 마흔여섯 개의 큐브를 포함합니다.

$C(n)$을 한 층에 $n$개의 큐브를 포함하는 직육면체의 개수를 나타내는 함수로 정의합니다. 따라서 $C(22) = 2$, $C(46) = 4$, $C(78) = 5$, $C(118) = 8$입니다.

154는 $C(n) = 10$이 되는 가장 작은 $n$ 값입니다.

$C(n) = 1000$이 되는 가장 작은 $n$ 값을 찾으세요.
