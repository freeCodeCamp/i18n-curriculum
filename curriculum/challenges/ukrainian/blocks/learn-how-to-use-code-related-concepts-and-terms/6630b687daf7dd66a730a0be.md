# --description--

Слово `fundamental` означає щось базове або суттєве, подібно до фундаменту будинку.

Наприклад, `Understanding grammar is fundamental to learning a new language.` Тут `fundamental` використовується, щоб підкреслити, що граматика є основною або базовою частиною вивчення будь-якої нової мови.

Фраза `what kind of` використовується, щоб запитати про тип або природу чогось. Це загальний спосіб чіткіше визначити або класифікувати щось.

Наприклад, коли хтось запитує `what kind of music do you like?`, він цікавиться жанром музики.
