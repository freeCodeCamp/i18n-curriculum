# --explanation--

`The day before` は前日を指す副詞句です。時間の感覚を与え、ある出来事が現在の日の1日前に起きたことを示します。

単に `yesterday` が `today` の前日であると言うだけでなく、`the day before` を使って任意の日の前日について話すことができます。例えば：

`We discussed the new feature the day before.` – これは話されている日の前日に議論があったことを意味します。

対照的に、`yesterday`、例えば `We discussed the new feature yesterday.` のように言うと、特に今日の前日であったことを意味します。
