# --description--

我們將定義畢氏多邊形為具有以下屬性的凸多邊形：

- 至少有三個頂點，
- 沒有三個頂點共線，
- 每個頂點都有整數座標，
- 每條邊都有整數長度。

對於給定的整數 $n$，定義 $P(n)$ 為周長 $≤ n$ 的不同畢氏多邊形數量。

只要沒有任何一個是另一個的平移，畢氏多邊形就應被視為不同的。

已知 $P(4) = 1$、$P(30) = 3655$ 和 $P(60) = 891045$。

尋找 $P(120)$。
