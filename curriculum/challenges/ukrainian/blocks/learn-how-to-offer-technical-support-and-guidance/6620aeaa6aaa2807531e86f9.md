# --description--

Дієслово `encounter` означає зустрітися або натрапити на щось, особливо на щось несподіване. У програмуванні `encountering` на проблему або помилку означає зіткнутися з проблемою під час виконання коду.

Наприклад, `You might encounter an error if the user inputs the wrong type of data.` Це речення означає, що ви можете несподівано зіткнутися або мати справу з помилкою.
