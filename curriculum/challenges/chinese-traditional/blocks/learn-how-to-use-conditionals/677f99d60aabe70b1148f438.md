# --explanation--

`Present Simple` 時態經常用來描述通常為真的狀態、事實或條件。當電腦嘗試開機時，因為 BIOS 損壞被視為一種穩定或不變的狀態，所以會使用此時態。

`Is corrupted` 意味著損壞或以導致其無法正常運作的方式被更改。 例如：

`The file is corrupted and won't open.` - 該檔案已損壞且無法被存取。
