# --description--

Como você pode ver na saída, `.items()` cria uma estrutura de dados que armazena cada par chave-valor em uma tupla distinta.Para iterar sobre os elementos nesses tuplas você pode adicionar uma segunda variável de loop:

```py
for i, j in dict.items():
    print(i, j)
```

Modifique seu loop `for` para usar duas variáveis de loop e imprimir ambas dentro do corpo do loop.
