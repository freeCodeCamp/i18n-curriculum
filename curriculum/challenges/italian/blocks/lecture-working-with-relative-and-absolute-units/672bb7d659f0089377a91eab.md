# --interactive--

Le percentuali in CSS sono unità relative che ti permettono di definire dimensioni, misure e altre proprietà come una proporzione rispetto all'elemento genitore. Quando usi un valore percentuale, stai praticamente dicendo: "Rendi questo elemento il X% del suo contenitore."

Per esempio, se imposti `width: 50%` su un elemento, occuperà metà della larghezza del suo contenitore genitore. Questo rende le percentuali estremamente utili per creare design responsivi che si adattano a diverse dimensioni dello schermo.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <div class="box"></div>
</div>
```

```css
.container {
  width: 400px;
  height: 200px;
  background-color: lightgray;
}

.box {
  width: 50%;
  height: 100%;
  background-color: red;
}
```

:::

Le percentuali sono ideali per creare layout fluidi che si adattano a varie dimensioni dello schermo. Per esempio, impostare un contenitore a `width: 80%` garantisce che occupi l'80% della larghezza del suo genitore, indipendentemente dal dispositivo.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="parent">
  <div class="child"></div>
</div>
```

```css
.parent {
  width: 100%;
  height: 300px;
  background-color: lightblue;
}

.child {
  width: 80%;
  height: 100%;
  background-color: red;
}
```

:::

Usare le percentuali per immagini flessibili è un'altra pratica comune. Applicando `max-width: 100%` alle immagini, permetti loro di ridimensionarsi su schermi più piccoli mantenendo il rapporto d'aspetto.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<img src="https://placehold.co/150x150" alt="Example Product Image" />
```

```css
img {
  max-width: 100%;
  height: auto;
}
```

:::

Anche se meno comune, le percentuali possono essere usate per le dimensioni del carattere tipografico per creare una tipografia scalabile. Per esempio, `font-size: 120%` renderebbe il testo il 20% più grande rispetto alla dimensione del carattere del genitore.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="text-container">
  Parent text.
  <p class="text">This is some example text.</p>
</div>
```

```css
.text-container {
  font-size: 16px;
}

.text {
  font-size: 120%;
}
```

:::

Le percentuali possono essere particolarmente utili per il centraggio verticale. Ecco un esempio di come potresti usare le percentuali con la proprietà `transform` per centrare un elemento verticalmente.

Questo esempio posiziona l'elemento al 50% dall'alto del suo contenitore, poi usa `transform` per spostarlo verso l'alto di metà della sua altezza, centrando così verticalmente l'elemento.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="centered"></div>
```

```css
.centered {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
  width: 300px;
  height: 300px;
  background-color: red;
}
```

:::

Imparerai di più su come funzionano il posizionamento assoluto e le proprietà transform in lezioni future.

Ricorda, le percentuali sono sempre relative a qualcosa. Per proprietà orizzontali come `width`, sono relative alla larghezza del genitore. Per proprietà verticali come `height`, di solito sono relative all'altezza del genitore (se specificata).

Tuttavia, fai attenzione quando annidi elementi con dimensioni basate su percentuali, perché questo può portare a risultati inattesi. Inoltre, tieni presente che le altezze basate su percentuali possono essere complicate se il genitore non ha un'altezza definita.

In sintesi, le percentuali in CSS sono strumenti potenti per creare design flessibili e responsivi. Usale quando vuoi che gli elementi si ridimensionino proporzionalmente ai loro contenitori o quando hai bisogno di layout che si adattino senza problemi a diverse dimensioni dello schermo.
