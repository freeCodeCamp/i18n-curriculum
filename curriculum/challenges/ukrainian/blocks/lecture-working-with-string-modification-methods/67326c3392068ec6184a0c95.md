# --interactive--

Працюючи з JavaScript, ви можете стикнутися з ситуаціями, коли потрібно повторити рядок певну кількість разів.

Чи то для створення повторюваних шаблонів, чи просто для дублювання тексту, метод `repeat()` надає простий і ефективний спосіб це зробити.

Метод `repeat()` — це вбудована функція в JavaScript, яка дозволяє повторити рядок задану кількість разів. Ось базовий синтаксис:

```js
string.repeat(count);
```

`string` — це рядок, який ви хочете повторити, а `count` — кількість повторів рядка. Ось приклад:

:::interactive_editor

```js
let word = "Hello!";
let repeatedWord = word.repeat(3);
console.log(repeatedWord);  // "Hello!Hello!Hello!"
```

:::

У цьому випадку рядок `Hello!` повторюється тричі, в результаті чого виходить `Hello!Hello!Hello!`.

Хоча метод `repeat()` корисний, є кілька винятків і обмежень, які варто враховувати.

Параметр `count` має бути невід’ємним числом. Якщо передати від’ємне число, JavaScript викине `RangeError`.

```js
let word = "Test";
console.log(word.repeat(-1));  // Throws RangeError: Invalid count value
```

Параметр `count` має бути скінченним числом. Якщо ви спробуєте повторити рядок нескінченну кількість разів або використати `Infinity` як кількість, ви також отримаєте `RangeError`.

У JavaScript `Infinity` — це спеціальне значення, яке представляє нескінченну кількість. Його використовують для позначення чисел, більших за будь-яке скінченне число.

```js
let word = "Test";
console.log(word.repeat(Infinity));  // Throws RangeError: Invalid count value
```

Якщо кількість не є цілим числом (наприклад, десятковим, як `2.5`), метод `repeat()` округлить її вниз до найближчого цілого.

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(2.5));  // "TestTest"
```

:::

Якщо передати `0` як кількість, метод `repeat()` поверне порожній рядок.

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(0));  // ""
```

:::

Метод `repeat()` може спростити завдання, пов’язані з дублюванням рядків, роблячи ваш код більш лаконічним і читабельним.

Чи то для створення повторюваних текстових шаблонів, чи для заповнення простору символами, `repeat()` може заощадити вам час, позбавляючи від необхідності писати цикли або складніший код.

Ви не обмежені передачею числа безпосередньо в метод `repeat()`. Ви також можете передати змінну, яка зберігає числове значення.

:::interactive_editor

```js
let count = 4;
let word = "Test";
let repeatedWord = word.repeat(count);
console.log(repeatedWord); // TestTestTestTest
```

:::

У цьому прикладі змінна `count` зберігає кількість повторів. Це може бути корисно, коли кількість повторів залежить від вхідних даних користувача або інших динамічних значень у вашій програмі.
