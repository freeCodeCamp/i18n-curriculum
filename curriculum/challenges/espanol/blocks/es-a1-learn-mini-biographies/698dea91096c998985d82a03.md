# --description--

`Le gusta` significa "a ella le gusta" o "a él le gusta" en español. Para decir lo que a alguien le gusta, puedes usar esta estructura:

**`Le gusta` + lo que le gusta a la persona.**

Por ejemplo:

`Le gusta comer pizza` - A ella le gusta comer pizza.
