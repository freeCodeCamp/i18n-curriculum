# --description--

`Low-code development` 是一種以最少手寫程式碼來建立軟體應用程式的方法。開發者不是撰寫大量程式碼，而是使用具有拖放功能的圖形使用者介面來組裝和組態應用程式。這種方法允許更快速的開發，對於可能沒有廣泛程式設計技能的使用者尤其有利。
