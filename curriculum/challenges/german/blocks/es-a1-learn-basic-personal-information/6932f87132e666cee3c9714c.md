# --description--

Ángela stellt eine Frage mit der Struktur `¿Cuál es tu` + Substantiv.

Sie könnte dort aufhören. Stattdessen fügt sie mit dem Bindewort `y` und einem zweiten Substantiv eine weitere Information hinzu.

`y` ist eine Konjunktion, die zwei Substantive verbindet. Hier ist die Struktur, die sie verwendet:

`¿Cuál es tu` + Substantiv + `y` + Substantiv`?`

Damit kann die sprechende Person in derselben Frage nach mehr als einem Detail fragen.

# --explanation--

Im Satz `¿Cuál es tu nombre y apellido?` werden zwei Substantive mit `y` verbunden:

- `Nombre`

- `Apellido`

Das Wort `y` verbindet zwei Informationsstücke.

Ángela fragt zuerst nach dem `nombre` der Person und fügt dann `apellido` hinzu.

Diese Struktur ermöglicht es der sprechenden Person, in einer Frage mehrere Details zu erfragen:

`¿Cuál es tu correo y número?`

Hier ist das zweite Informationsstück, nach dem Ángela fragt, `apellido`.
