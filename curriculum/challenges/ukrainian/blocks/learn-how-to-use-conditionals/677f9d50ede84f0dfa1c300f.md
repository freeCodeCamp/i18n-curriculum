# --explanation--

`Fail` означає, що щось не працює так, як має. Це стосується проблеми, яка впливає на нормальну роботу. Наприклад:

- `The server is failing because of high traffic.` - Сервер падає через надто велику кількість користувачів одночасно.

- `The app failed during the beta testing phase.` - Застосунок не працював правильно під час тестування користувачами.
