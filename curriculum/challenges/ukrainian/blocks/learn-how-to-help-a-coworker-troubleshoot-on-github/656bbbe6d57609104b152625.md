# --description--

Інструкція Боба використовує минулий безперервний час `I was checking` для опису дії, яка тривала в минулому.

Ви можете створювати речення в минулому безперервному часі, поєднуючи дієслово to be у минулому `was` або `were` + `verb + ing`.

Він також використовує модальне дієслово `should` для пропозиції певного курсу дій.
