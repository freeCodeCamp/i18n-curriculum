# --explanation--

Dieser Satz verwendet das **Passiv**, das den Fokus auf das Objekt legt, das die Handlung empfängt, anstatt darauf, wer sie ausführt. Zum Beispiel:  

`The navigation menu was appreciated.` – Das bedeutet, dass es den Leuten gefallen hat, aber es wird nicht gesagt, wer es mochte.  

Im **Aktiv** würde man sagen `The users appreciated the navigation menu.` – Das zeigt klar, wer die Handlung ausgeführt hat.

Das **Passiv** wird oft verwendet, wenn die handelnde Person unbekannt oder unwichtig ist. Ein weiteres Beispiel:

`The report was completed on time.` – Sie wissen, dass der Bericht fertig ist, aber nicht, wer ihn erstellt hat oder dass Sie es sagen müssten.
