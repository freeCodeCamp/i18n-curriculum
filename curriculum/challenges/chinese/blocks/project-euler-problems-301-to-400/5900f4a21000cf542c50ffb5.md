# --description--

爱丽丝和鲍勃玩游戏Nim Square。

Nim Square就像普通的三堆普通游戏Nim一样，但是玩家只能从堆中取出一平方的石头。

三个堆中石头的数量由有序三元组（$a$、$b$、$c$）表现。

如果 $0 ≤ a ≤ b ≤ c ≤ 29$，那么下一位玩家的失败位置数量是 1160。

查找下一个玩家的失败位置数量，条件是 $0 ≤ a ≤ b ≤ c ≤ 100\,000$。
