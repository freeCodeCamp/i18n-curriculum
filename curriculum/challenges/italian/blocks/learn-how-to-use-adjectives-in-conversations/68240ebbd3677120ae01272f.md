`By itself` indica che qualcosa succede automaticamente, senza che nessuno debba farlo. Per esempio:  

`The layout updates by itself when the window size changes.` – Cambia automaticamente.

`Nicer` è la forma comparativa di `nice`. Significa che qualcosa è migliore, più comodo o più piacevole di qualcos'altro. Per esempio:  

`The new interface is nicer to use.` – È più piacevole o facile da usare.
