# --description--

`only if` という式は、何かが起こるための条件を設定するために使われます。これは「これが起こるのは、あれが起こった場合」という意味です。ある行動が別の行動に依存する約束をする方法です。

例えば、友達が `I'll go out only if it doesn't rain` と言うかもしれません。これは、天気が良ければ外出するという意味で、雨が降らないことが条件です。
