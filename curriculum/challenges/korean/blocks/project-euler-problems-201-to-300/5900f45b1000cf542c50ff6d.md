# --description--

"Blum Blum Shub" 의사난수 생성기를 사용해 수열을 만드세요:

$$
s_0 = 14025256 \\\\
s_{n + 1} = {s_n}^2 \\; mod \\; 20\\,300\\,713
$$

이 숫자들 $s_0s_1s_2\ldots$ 을 이어 붙여 무한 길이의 문자열 $w$ 를 만드세요. 예를 들어, $w = 14025256741014958470038053646\ldots$ 입니다.

양의 정수 $k$ 에 대해, $w$ 의 부분 문자열 중 자릿수 합이 $k$ 인 것이 없으면 $p(k)$ 는 0으로 정의합니다. 만약 자릿수 합이 $k$ 인 부분 문자열이 하나 이상 있으면, 가장 이른 위치에서 시작하는 부분 문자열의 시작 위치를 $p(k) = z$ 로 정의합니다.

예를 들어:

부분 문자열 1, 14, 1402, … 는 각각 자릿수 합이 1, 5, 7, … 이고 위치 1에서 시작하므로 $p(1) = p(5) = p(7) = \ldots = 1$ 입니다.

부분 문자열 4, 402, 4025, … 는 각각 자릿수 합이 4, 6, 11, … 이고 위치 2에서 시작하므로 $p(4) = p(6) = p(11) = \ldots = 2$ 입니다.

부분 문자열 02, 0252, … 는 각각 자릿수 합이 2, 9, … 이고 위치 3에서 시작하므로 $p(2) = p(9) = \ldots = 3$ 입니다.

부분 문자열 025 는 위치 3에서 시작하며 자릿수 합이 7이지만, 자릿수 합이 7인 더 이른 위치의 부분 문자열(위치 1에서 시작)이 있으므로 $p(7) = 1$ 이고 3이 아닙니다.

$0 < k \leq {10}^3$ 에 대해 $\sum p(k) = 4742$ 임을 확인할 수 있습니다.

$0 < k \leq 2 \times {10}^{15}$ 에 대해 $\sum p(k)$ 를 구하세요.
