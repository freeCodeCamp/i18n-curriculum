# --description--

모든 정수 $n > 1$에 대해, 함수족 $f_{n, a, b}$는 다음과 같이 정의됩니다:

$f_{n, a, b}(x) ≡ ax + b\bmod n$ 여기서 $a, b, x$는 정수이고 $0 \lt a \lt n$, $0 \le b \lt n$, $0 \le x \lt n$입니다.

모든 $0 \le x \lt n$에 대해 $f_{n, a, b}(f_{n, a, b}(x)) \equiv f_{n, a, b}(x)\bmod n$를 만족하면 $f_{n, a, b}$를 축소 함수라고 부릅니다.

$R(n)$을 $n$에 대한 축소 함수의 개수라고 합시다.

다음이 주어졌습니다:

$$\sum_{k = 1}^{99\\,999} R(\displaystyle\binom{100\\,000}{k}) \equiv 628\\,701\\,600\bmod 1\\,000\\,000\\,007$$

다음 값을 구하세요:

$$\sum_{k = 1}^{9\\,999\\,999} R(\displaystyle\binom{10\\,000\\,000}{k})$$

답은 $1\\,000\\,000\\,007$로 나눈 나머지로 제출하세요.
