# --description--

`Loop` — це послідовність інструкцій, яка повторюється безперервно, доки не буде досягнута певна умова. `Loops` використовують, щоб ефективно запускати один і той самий код кілька разів. Загальні типи `loop` включають цикли `for` і цикли `while`.

Наприклад, `for loop`: `for (int i = 0; i < 5; i++) { /* code to be executed */ }`

У цьому прикладі код всередині дужок `{ }` виконується повторно, доки умова залишається істинною.
