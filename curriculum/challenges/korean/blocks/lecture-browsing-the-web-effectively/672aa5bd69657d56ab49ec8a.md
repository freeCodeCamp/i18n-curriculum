# --description--

웹 브라우저는 인터넷에서 웹 페이지를 방문할 때 사용하는 애플리케이션입니다. freeCodeCamp 커리큘럼을 진행 중이라면 지금 웹 브라우저를 사용하고 있을 가능성이 매우 높습니다!

2024년 기준으로 가장 인기 있는 브라우저는 Microsoft Edge, Firefox, Google Chrome, Safari입니다. 다른 옵션도 꽤 많고 시장 점유율은 계속 변하지만, 이 네 가지가 현재 가장 흔한 브라우저입니다.

운영체제에는 기본적으로 브라우저가 내장되어 있을 가능성이 큽니다. Windows에는 Microsoft Edge가, macOS에는 Safari가, Linux에는 보통 Firefox가 내장되어 있습니다.

그런데 다른 브라우저를 사용하고 싶다면 어떻게 해야 할까요?

좋은 소식은 이미 기본 브라우저가 있다는 점입니다. 원하는 브라우저의 웹 페이지로 이동해 운영체제에 맞는 설치 프로그램을 다운로드하면 됩니다.

브라우저를 선택할 때는 속도, 보안 기능, 자주 방문하는 웹사이트와의 호환성 같은 요소를 고려하세요.

일부 브라우저는 독특한 기능을 제공합니다. 예를 들어 Opera는 내장 VPN이 있고, Brave는 개인정보 보호와 광고 차단에 중점을 둡니다.

설치는 보통 간단합니다. 설치 프로그램을 다운로드한 후 실행하고 화면 안내를 따르면 됩니다.

대부분의 브라우저는 현재 기본 브라우저에서 북마크와 설정을 가져오는 기능을 제공해 전환을 쉽게 합니다.

또는 macOS에서 Homebrew 같은 `brew install --cask google-chrome`를 사용하거나 Arch Linux에서 `yay -S brave-bin` 같은 운영체제의 패키지 관리자를 통해 브라우저를 설치할 수도 있습니다.

사용 가능한 패키지 관리자와 패키지 이름은 운영체제에 따라 다릅니다.

한 가지 브라우저만 써야 하는 것은 아닙니다. 많은 사람이 용도에 따라 여러 브라우저를 사용합니다—예를 들어 업무용과 개인용으로 각각 다른 브라우저를 쓰기도 합니다.

여러 브라우저를 시도해 보면서 자신에게 가장 잘 맞는 것을 찾아보세요.
