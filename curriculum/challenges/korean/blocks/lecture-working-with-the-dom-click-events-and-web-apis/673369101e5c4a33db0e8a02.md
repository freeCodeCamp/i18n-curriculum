# --interactive--

이전 수업에서는 `"click"`와 `"input"` 이벤트를 `addEventListener()` 메서드를 사용해 다루는 방법을 배웠습니다. 이번 수업에서는 `removeEventListener()` 메서드를 사용해 이벤트 리스너를 제거하는 방법을 배웁니다.

`removeEventListener()` 메서드는 이전에 `addEventListener()` 메서드로 요소에 추가한 이벤트 리스너를 제거하는 데 사용됩니다. 특정 요소에서 특정 이벤트를 더 이상 듣지 않으려 할 때 유용합니다.

`removeEventListener()` 메서드의 기본 구문은 다음과 같습니다:

```js
element.removeEventListener("event", listener);
```

`addEventListener()` 메서드와 마찬가지로 `removeEventListener()` 메서드는 두 개의 인수를 받습니다: 제거하려는 이벤트와 이전에 추가한 리스너 함수입니다.

또한 `removeEventListener()` 메서드에 전달할 수 있는 선택적 세 번째 인수가 있습니다. 이 인수는 `options` 또는 `useCapture`일 수 있습니다.

`options` 인수는 이벤트 리스너의 옵션을 지정하는 객체로, 이벤트 리스너가 수동인지 또는 한 번만 실행되는지 등을 설정합니다.

`useCapture` 인수는 이벤트 전파 단계에서 이벤트를 캡처할지 여부를 지정하는 불리언 값입니다.

대부분의 경우 `removeEventListener()` 메서드에는 이벤트와 리스너 인수만 전달하면 충분합니다.

`removeEventListener()` 메서드가 어떻게 작동하는지 더 잘 이해하기 위해 예제를 살펴보겠습니다:

배경색을 회색과 파란색 사이에서 토글하려면 이벤트 리스너를 사용할 수 있습니다.

먼저 `button` 요소와 `body` 요소에 접근해야 합니다:

```js
const bodyEl = document.querySelector("body");
const btn = document.getElementById("btn");
```

그다음 회색과 파란색 사이를 토글하는 함수를 만들어야 합니다:

```js
let isBgColorGrey = true;

function toggleBgColor() {
  bodyEl.style.backgroundColor = isBgColorGrey ? "blue" : "grey";
  isBgColorGrey = !isBgColorGrey;
}
```

현재 배경색을 추적하기 위해 불리언 변수 `isBgColorGrey`를 사용합니다. 배경색이 회색이면 파란색으로, 반대면 회색으로 변경합니다.

그다음 `button` 요소에 이벤트 리스너를 추가해 버튼이 클릭될 때 `toggleBgColor` 함수를 호출하도록 합니다:

```js
btn.addEventListener("click", toggleBgColor);
```

이제 마우스를 올리면 이벤트 리스너를 제거하는 데 사용할 단락 요소를 포함하도록 HTML을 업데이트할 수 있습니다:

```html
<link rel="stylesheet" href="styles.css" />
<h1>Event Listener examples</h1>
<p id="para">MouseOver this text to remove the event listener</p>
<button id="btn">Change background color</button>
<script src="index.js"></script>
```

단락에 마우스를 올렸을 때 이벤트 리스너를 제거하려면 단락 요소에 이벤트 리스너를 추가합니다. `mouseover` 이벤트를 사용하고 `button` 요소에서 이벤트 리스너를 제거하는 함수를 전달합니다:

```js
const para = document.getElementById("para");
const btn = document.getElementById("btn");

para.addEventListener("mouseover", () => {
  btn.removeEventListener("click", toggleBgColor);
});
```

단락에 마우스를 올리면 버튼 클릭 이벤트에 대한 이벤트 리스너가 제거되어 버튼을 클릭해도 배경색이 더 이상 바뀌지 않습니다.
