# --interactive--

이전 수업에서는 픽셀과 같은 절대 길이 단위에 대해 배웠습니다. 절대 단위는 특정 상황에서 유용할 수 있지만, 상대 단위를 사용해야 할 때도 있습니다.

이번 수업에서는 두 가지 상대 단위인 `em`s와 `rem`s에 대해 배웁니다.

`em` 단위는 요소의 폰트 크기를 기준으로 합니다. 만약 `font-size` 속성에 em 단위를 사용한다면, 텍스트 크기는 부모 요소의 폰트 크기에 상대적입니다.

이 작동 방식을 더 잘 이해하기 위해 예제를 살펴보겠습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p class="para">I am a paragraph element</p>

<div class="blue-box"></div>
```

```css
.para {
  font-size: 20px;
  margin-bottom: 1.5em;
  border: 2px solid red;
}

.blue-box {
  background-color: blue;
  color: white;
  padding: 10px;
  width: 100px;
  height: 100px;
}
```

:::

HTML에는 단락과 `div` 요소가 있습니다. 단락 요소에는 `para`라는 클래스가 있고, `div` 요소에는 `blue-box`라는 클래스가 있습니다.

`para` 클래스에서는 `font-size`를 `20px`으로, `margin-bottom`을 `1.5em`로 설정했습니다. 이는 여백이 단락 요소의 폰트 크기의 1.5배가 된다는 뜻입니다. `1.5em`은 단락 하단에 30픽셀 여백을 만듭니다. 또한 여백을 더 잘 볼 수 있도록 `border`를 `2px solid red`로 설정했습니다.

`blue-box` 클래스에서는 배경색을 `blue`로, 텍스트 색상을 `white`로, 그리고 네 면 모두에 `padding`를 `10px`로 설정했습니다.

예제에서 단락 요소 하단과 파란 상자 사이에 명확한 공간이 생깁니다.

그렇다면 `font-size` 클래스에서 `para` 속성을 제거하면 어떻게 될까요?

하단 여백은 부모 요소의 폰트 크기에 상대적이 됩니다. 이 경우 부모 요소는 기본 폰트 크기가 `16px`인 body 요소입니다.

`em` 단위는 버튼이나 카드 같은 모듈형 컴포넌트를 작업할 때 좋은 사용 사례입니다. `em` 단위를 사용하면 패딩, 여백, 테두리 같은 컴포넌트의 모든 부분이 폰트 크기에 비례해 크기가 조절되어 일관된 비율을 유지할 수 있습니다.

지금까지는 요소의 폰트 크기를 픽셀로 설정해왔습니다. 하지만 이것은 사용자에게 몇 가지 문제를 일으킬 수 있습니다.

브라우저 설정에서 기본 폰트 크기를 조절할 수 있습니다.

시각 장애가 있는 사용자는 읽기 쉽게 폰트 크기를 키울 수 있습니다. 하지만 웹 디자인에서 폰트 크기를 픽셀로 설정하면 텍스트가 다른 콘텐츠와 비례해 크기가 조절되지 않습니다.

이 문제를 해결하는 한 가지 방법은 타이포그래피에 `rem` 단위를 사용하는 것입니다. `rem` 단위는 루트 요소인 `html` 요소의 폰트 크기를 기준으로 합니다.

기본적으로 `html` 요소의 폰트 크기는 `16px`입니다. 사용자가 브라우저 설정에서 폰트 크기를 키우면 `html` 요소의 폰트 크기도 커지고, 모든 rem 단위가 비례해 조절됩니다.

픽셀 대신 `rem` 단위를 폰트 크기에 사용하는 예는 다음과 같습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is regular text.</p>
<p class="para">This text is slightly larger.</p>
```

```css
.para {
  font-size: 1.2rem;
  margin-bottom: 1.5em;
  border: 2px solid red;
}
```

:::

폰트 크기를 `1.2rem`로 설정하면 단락 요소의 폰트 크기는 루트 요소 폰트 크기의 1.2배가 됩니다. 사용자가 기본 폰트 크기를 변경하지 않았다면 단락 요소의 폰트 크기는 `19.2px`가 되는데, 이는 `16px`의 1.2배이기 때문입니다.

그렇다면 언제 `rem` 단위를 사용해야 할까요? `rem` 단위는 픽셀보다 타이포그래피에 더 적합한데, 사용자의 브라우저 설정에 따라 비례해 크기가 조절되기 때문입니다. 이는 시각 장애가 있는 사용자에게 콘텐츠 접근성을 높여줍니다.

`rem` 단위는 다양한 요소 간 일관된 간격과 레이아웃을 유지하는 데도 도움이 됩니다.
