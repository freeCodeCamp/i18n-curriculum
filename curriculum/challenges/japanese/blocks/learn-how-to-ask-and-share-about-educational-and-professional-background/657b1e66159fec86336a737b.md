# --description--

通常、`Past Simple` 時制は規則動詞に `-ed` を付け加えて作られます。例えば、動詞 `play` は `played` になります。

`play around` とは、何かをカジュアルに、または試験的に試してみることを意味し、多くの場合、結果を見てみるために行います。通常は `with` が続きます。例えば：

`He played around with the new software to see how it works.` - これは彼が真剣な計画なしにソフトウェアを試してみたことを意味します。
