# --explanation--

`Short-term` bedeutet, dass etwas für eine begrenzte Zeit geplant ist und nicht dauerhaft ist. Zum Beispiel:

`We hired a short-term contractor to help this week.` – Das bedeutet, die Unterstützung ist nur vorübergehend.

`During` zeigt, dass etwas in einem bestimmten Zeitbereich stattfindet. Zum Beispiel:

`We worked remotely during the summer.` – Das bedeutet während dieser Saison.

`Period` bezeichnet eine festgelegte Zeitspanne mit Anfang und Ende. Zum Beispiel:

`The trial period lasts two weeks.` – Das bedeutet den zweiwöchigen Zeitraum zum Testen von etwas.
