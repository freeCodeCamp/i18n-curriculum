# --description--

在 Python 中，类和对象协同工作以组织和管理数据。你创建一个类来定义共享行为，然后创建使用这些行为的对象。

换句话说，`class` 就像你用来创建对象的蓝图或模板。

让我们看看什么是类，以及如何使用它们来创建对象。

要创建一个类，你使用 `class` 关键字，后跟类名和冒号。然后在类中，你可以添加一个初始化方法，以及任何属性和方法。

属性就像类中的变量，用于保存数据。方法是在类中定义的函数，是用类创建的对象可以执行的操作。

这是一个类的基本语法：

```python
class ClassName:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def sample_method(self):               
        print(self.name.upper())
```

* `class ClassName` 由用于创建类的 `class` 关键字组成，后跟类的名称，这里称为 `ClassName`。在 Python 中，命名类时通常使用 **PascalCase** 约定。
    
* `def __init__(self, name, age)` 是在新建对象时自动调用的特殊方法。它初始化将由类创建的对象的属性。除此之外，`__init__` 的第一个参数总是对正在创建或使用的特定对象的引用。按照惯例，这个参数命名为 `self`，但从技术上讲，你可以使用任何名称。`self` 让你访问对象自身的属性和方法。
    
* `self.name = name` 和 `self.age = age` 是对象将拥有的属性。
    
* `def sample_method(self):` 是每个创建的对象都可以调用的方法。
    
* `print(self.name.upper())` 是 `sample_method` 方法将要执行的操作，在这个分支中，它打印大写的名字。
    

如果这些听起来很多，不用担心。让我们来看一个类似的 `Dog` 类示例，以及你如何从中创建对象：

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name.upper()} says woof woof!")
```

使用这个 `Dog` 类，你可以创建一个对象。以下是从类创建对象的基本语法：

```python
object_1 = ClassName(attribute_1, attribute_2)
object_2 = ClassName(attribute_1, attribute_2)
```

你也可以从每个对象调用类中定义的任何方法：

```python
object_1.method_name()
object_2.method_name()
```

现在让我们使用 `Dog` 类作为蓝图创建两只狗：

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name.upper()} says woof woof! I'm {self.age} years old!")

dog_1 = Dog("Jack", 3)
dog_2 = Dog("Thatcher", 5)

# Call the bark method
dog_1.bark()  # JACK says woof woof! I'm 3 years old!
dog_2.bark()  # THATCHER says woof woof! I'm 5 years old!
```

如你所见，我们使用 `Dog` 类创建了两个狗对象。在初始化 `dog_1` 时，传入了字串 `Jack` 和数字 `3`，这为该实例设置了 `name` 和 `age` 属性。而 `dog_2` 则分别以字串 `Thatcher` 和数字 `5` 作为其 `name` 和 `age` 进行初始化。

然后当你对 `dog_1` 和 `dog_2` 调用 `.bark()` 方法时，你可以看到两个输出的不同，并使用你在创建每个对象时传入的唯一 `name` 和 `age` 属性。

总之，`class` 和 `object` 之间的区别在于，`class` 是模板或蓝图，而 `object` 是使用该模板创建的对象。

此外，类定义了对象应具有的数据和行为，而对象则保存实际数据并使用这些行为。你只需定义一次类，就可以从中创建多个对象，每个对象拥有不同的数据。
