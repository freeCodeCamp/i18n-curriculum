# --description--

La hiperexponenciación o tetración de un número $a$ por un entero positivo $b$, denotada por $a↑↑b$ o ${}^ba$, se define recursivamente como:

$a↑↑1 = a$,

$a↑↑(k+1) = a^{(a↑↑k)}$.

Así tenemos por ejemplo $3↑↑2 = 3^3 = 27$, por lo tanto $3↑↑3 = 3^{27} = 7625597484987$ y $3↑↑4$ es aproximadamente ${10}^{3.6383346400240996 \times {10}^{12}}$. Encuentra los últimos 8 dígitos de $1777↑↑1855$.
