# --description--

Iniciar sesión significa ingresar tu nombre de usuario y contraseña para acceder a una computadora o sitio web. Por ejemplo:

`Every morning, I log in to my computer to start work.` - Esto significa que cada mañana ingresas tu nombre de usuario y contraseña para abrir tu computadora y comenzar a trabajar.

Revisar `check emails` significa mirar y leer mensajes nuevos en tu cuenta de correo electrónico. Por ejemplo:

`After logging in, I check my emails to see if I have any new messages.` - Esto significa que después de encender tu computadora o abrir tu cuenta de correo, revisas si alguien te ha enviado mensajes nuevos.
