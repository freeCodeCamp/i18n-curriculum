# --description--

`Actually` se usa para mostrar el hecho real sobre algo. Lo usas cuando quieres corregir a alguien o reforzar tu punto. Es como decir: "Esta es la verdad". Por ejemplo:

`He thought the project was finished, but actually, there are still some tasks left.` - Esto corrige la idea. El proyecto aún no está terminado.

# --questions--

## --text--
