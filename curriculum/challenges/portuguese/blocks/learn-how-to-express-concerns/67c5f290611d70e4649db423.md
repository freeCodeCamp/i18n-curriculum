# --explanation--

`To draft` significa criar uma versão inicial de algo, como um documento, plano ou design.Por exemplo:

`She drafted an email to send to the client before finalizing it.` - Isso significa que ela escreveu uma versão inicial do email.  

Um `schedule` refere-se a um plano que descreve quando tarefas ou eventos ocorrerão.Por exemplo:

`The project schedule shows all the deadlines for the team.` - Isso significa que o cronograma lista quando diferentes tarefas precisam ser concluídas.
