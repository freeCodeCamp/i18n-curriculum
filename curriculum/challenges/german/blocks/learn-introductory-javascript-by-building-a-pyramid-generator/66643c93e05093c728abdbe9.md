# --description--

Bevor Sie weitermachen, nehmen Sie sich einen Moment Zeit, um zu überprüfen, wie Funktionen funktionieren.

Deklarieren Sie eine Funktion namens `addTwoNumbers`. Diese Funktion soll zwei Argumente entgegennehmen und die Summe dieser beiden Argumente zurückgeben.

Ihre Funktion sollte keine fest codierten Werte verwenden. Ein Beispiel für eine fest codierte Funktion könnte sein:

```js
function sayName(firstName, lastName) {
  return "John Doe";
}

sayName("Camper", "Cat");
```

Diese Funktion würde `"John Doe"` zurückgeben, unabhängig von den an die Parameter `firstName` und `lastName` übergebenen Argumenten, daher gilt `"John Doe"` als fest codierter Wert.

Deklarieren Sie eine `sum`-Variable und weisen Sie ihr den Wert zu, der durch den Aufruf Ihrer `addTwoNumbers`-Funktion mit `5` und `10` als Argumente entsteht. Geben Sie die `sum`-Variable in der Konsole aus.
