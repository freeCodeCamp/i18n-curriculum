`frustrating`

### --feedback--

Прикметник, який описує почуття роздратування або засмучення через те, що щось не працює так, як очікувалося. Закінчується на `-ing`.

# --explanation--

`This is` — це простий спосіб почати речення, описуючи щось.

`Frustrating` — прикметник, який описує почуття роздратування або засмучення через те, що щось не працює так, як очікувалося. Наприклад:

- `This is confusing.` — ситуація важка для розуміння.

- `This is exciting.` — ситуація викликає у вас захоплення.
