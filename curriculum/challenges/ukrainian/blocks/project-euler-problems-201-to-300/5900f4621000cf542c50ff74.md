# --description--

Ми назвемо дріб, який не можна скоротити, коресилієнтним дробом.

Крім того, визначимо коресилієнтність знаменника, $R(d)$, як відношення його правильних дробів, які є коресилієнтними; наприклад, $R(12) = \frac{4}{11}$.

Коресилієнтність числа $d > 1$ тоді дорівнює $\frac{φ(d)}{d − 1}$, де $φ$ — функція Ейлера.

Далі визначимо коресилієнтність числа $n > 1$ як $C(n) = \frac{n − φ(n)}{n − 1}$.

Коресилієнтність простого числа $p$ дорівнює $C(p) = \frac{1}{p − 1}$.

Знайдіть суму всіх складених цілих чисел $1 < n ≤ 2 × {10}^{11}$, для яких $C(n)$ є одиничним дробом.
