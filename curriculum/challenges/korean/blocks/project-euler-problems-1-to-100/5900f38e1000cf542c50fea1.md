# --description--

145는 흥미로운 숫자인데, 1! + 4! + 5! = 1 + 24 + 120 = 145이기 때문입니다.

각 자리 숫자의 팩토리얼 합과 같은 숫자들과 그 합을 찾으세요.

**참고:** 1! = 1과 2! = 2는 합이 아니므로 포함하지 않습니다.
