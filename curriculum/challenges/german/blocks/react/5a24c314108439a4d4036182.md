# --description--

Sie haben vielleicht in der letzten Aufgabe bemerkt, dass es neben dem `style`-Attribut, das auf ein JavaScript-Objekt gesetzt wird, noch weitere Syntaxunterschiede zu HTML-inline-Stilen gibt. Zuerst verwenden die Namen bestimmter CSS-Stileigenschaften camel case. Zum Beispiel wurde in der letzten Aufgabe die Schriftgröße mit `fontSize` statt `font-size` gesetzt. Wörter mit Bindestrich wie `font-size` sind für JavaScript-Objekteigenschaften ungültige Syntax, daher verwendet React camel case. Als Regel gilt: Alle mit Bindestrich geschriebenen Stileigenschaften werden in JSX in camel case geschrieben.

Alle Längeneinheiten von Eigenschaftswerten (wie `height`, `width` und `fontSize`) werden standardmäßig in `px` angenommen, sofern nicht anders angegeben. Wenn Sie zum Beispiel `em` verwenden möchten, setzen Sie den Wert und die Einheit in Anführungszeichen, wie `{fontSize: "4em"}`. Abgesehen von den Längenwerten, die standardmäßig `px` sind, sollten alle anderen Eigenschaftswerte in Anführungszeichen gesetzt werden.

# --instructions--

Wenn Sie eine große Anzahl von Stilen haben, können Sie ein Stil-`object` einer Konstanten zuweisen, um Ihren Code übersichtlich zu halten. Deklarieren Sie Ihre Stilkonstante als globale Variable am Anfang der Datei. Initialisieren Sie eine `styles`-Konstante und weisen Sie ihr ein `object` mit drei Stil-Eigenschaften und deren Werten zu. Geben Sie dem `div` eine Farbe von `purple`, eine Schriftgröße von `40` und einen Rahmen von `2px solid purple`. Setzen Sie dann das `style`-Attribut gleich der `styles`-Konstanten.

# --hints--

Die `styles`-Variable sollte ein `object` mit drei Eigenschaften sein.

```js
assert(Object.keys(styles).length === 3);
```

Die `styles`-Variable sollte eine `color`-Eigenschaft mit dem Wert `purple` haben.

```js
assert(styles.color === 'purple');
```

Die `styles`-Variable sollte eine `fontSize`-Eigenschaft mit dem Wert `40` haben.

```js
assert(styles.fontSize == 40);
```

Die `styles`-Variable sollte eine `border`-Eigenschaft mit dem Wert `2px solid purple` haben.

```js
assert(styles.border === '2px solid purple');
```

Die Komponente sollte ein `div`-Element rendern.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.shallow(React.createElement(Colorful));
    return mockedComponent.type() === 'div';
  })()
);
```

Das `div`-Element sollte seine Stile durch das `styles`-Objekt definiert haben.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.shallow(React.createElement(Colorful));
    return (
      mockedComponent.props().style.color === 'purple' &&
      mockedComponent.props().style.fontSize == 40 &&
      mockedComponent.props().style.border === '2px solid purple'
    );
  })()
);
```
