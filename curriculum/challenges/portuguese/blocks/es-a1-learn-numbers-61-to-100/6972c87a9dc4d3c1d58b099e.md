# --description--

Agora vamos mergulhar nos números de 90 a 99.

Em espanhol, o número 90 é chamado de `noventa`.

Esses números são formados usando a palavra das dezenas `noventa` (90), seguida da conjunção `y` ("e") e depois o dígito das unidades (1-9).

| Number | Spanish |
| :--- | :--- |
| 90 | `Noventa` |
| 91 | `Noventa y uno` |
| 92 | `Noventa y dos` |
| 93 | `Noventa y tres` |
| 94 | `Noventa y cuatro` |
| 95 | `Noventa y cinco` |
| 96 | `Noventa y seis` |
| 97 | `Noventa y siete` |
| 98 | `Noventa y ocho` |
| 99 | `Noventa y nueve` |

O número `noventa y uno` (91) é único nesta faixa porque se torna `noventa y un` antes de um substantivo masculino. Por exemplo:

`Noventa y un años` - Noventa e um anos.
