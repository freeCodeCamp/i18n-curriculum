# --description--

دعنا نتعلم عن عنصر `link`، وكيفية استخدامه للربط بملفات الأنماط الخارجية.

يُستخدم عنصر `link` للربط بالموارد الخارجية مثل ملفات الأنماط وأيقونات الموقع. فيما يلي الصيغة النحوية الأساسية لاستخدام عنصر `link` لملف CSS خارجي:

```html
<link rel="stylesheet" href="./styles.css" />
```

تُستخدم الخاصية `rel` لتحديد العلاقة بين المورد المرتبط ومستند HTML. في هذه الحالة، نحتاج إلى تحديد أن المورد المرتبط هو `stylesheet`.

تُعتبر أفضل الممارسات فصل HTML وCSS في ملفات مختلفة. يستخدم المطوّرون عنصر `link` لملف CSS الخارجي بدلاً من كتابة كل شيء داخل مستند HTML.

تُستخدم الخاصية `href` لتحديد موقع عنوان URL للمورد الخارجي.

النقطة `.` متبوعة بشرطة مائلة للأمام في المثال تخبر الكمبيوتر بالبحث في المجلد الحالي، أو الدليل، عن ملف `styles.css`.

يجب وضع عنصر `link` داخل عنصر `head` كما هو موضح في المثال التالي:

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Examples of the link element</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

غالبًا ما ترى عدة عناصر `link` داخل شفرة احترافية تربط بأنماط خطوط وأيقونات مختلفة. هذا مثال على استخدام عنصر `link` للربط بخط Google خارجي يسمى *Playwrite Cuba*:

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Playwrite+CU:wght@100..400&display=swap"
  rel="stylesheet"
/>
```

خطوط Google هي مجموعة من الخطوط المخصصة المجانية ومفتوحة المصدر التي يمكنك استخدامها داخل أي مشروع. يمكنك اختيار الخطوط التي تريد استخدامها وستوفر Google الشفرة اللازمة لـ HTML وCSS. في هذا المثال، القيمة `preconnect` للخاصية `rel` تخبر المتصفح بإنشاء اتصال مبكر مع القيمة المحددة في الخاصية `href`. يُجرى ذلك لتسريع أوقات تحميل هذه الموارد الخارجية.

حالة استخدام شائعة أخرى لعنصر `link` هي الربط بالأيقونات. إليك مثالًا على الربط بأيقونة مفضلة (favicon):

```html
<link rel="icon" href="favicon.ico" />
```

الأيقونة المفضلة، وهي اختصار لـ favorite icon، هي أيقونة صغيرة تُعرض عادةً في تبويب المتصفح بجانب عنوان الموقع. تستخدم العديد من المواقع الأيقونة المفضلة لعرض شعار علامتها التجارية.
