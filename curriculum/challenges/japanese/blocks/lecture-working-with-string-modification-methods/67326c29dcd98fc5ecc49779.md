# --interactive--

JavaScriptでは、文字列の一部を別の文字列に置き換える必要がある場面が多くあります。

例えば、URL内のユーザー情報を更新したり、日付のフォーマットを変更したり、ユーザーが作成したコンテンツの誤りを修正したりする場合です。

JavaScriptの`replace()`メソッドを使うと、文字列の中から指定した値（単語や文字など）を見つけて、別の値に置き換えることができます。このメソッドは置き換えた新しい文字列を返し、元の文字列は変更されません。なぜならJavaScriptの文字列は不変（immutable）だからです。

基本的な構文は次の通りです。

```js
string.replace(searchValue, newValue);
```

`searchValue`は文字列の中で検索したい値です。これは文字列か正規表現（regex）で指定できます。正規表現はテキストのパターンを表現するもので、柔軟かつ強力に文字列を検索・操作できます。正規表現については今後のレッスンで詳しく学びます。

`newValue`は`searchValue`を置き換える値です。簡単な例を見てみましょう。

:::interactive_editor

```js
let text = "I love JavaScript!";
console.log(text); // "I love JavaScript!"
let newText = text.replace("JavaScript", "coding");
console.log(newText);  // "I love coding!"
```

:::

この例では、文字列の中にある`JavaScript`という単語が見つかり、`coding`に置き換えられています。

`replace()`メソッドは大文字・小文字を区別するため、`searchValue`と完全に一致するものだけを見つけます。例えば：

:::interactive_editor

```js
let sentence = "I enjoy working with JavaScript.";
console.log(sentence);  // "I enjoy working with JavaScript."
let updatedSentence = sentence.replace("javascript", "coding");
console.log(updatedSentence);  // "I enjoy working with JavaScript."
```

:::

ここでは、`javascript`（小文字の`j`）が`JavaScript`（大文字の`J`）と一致しないため、置き換えは行われません。

デフォルトでは、`replace()`メソッドは`searchValue`の最初の出現だけを置き換えます。文字列に複数回出てきても、最初の1つだけが置き換えられます。

:::interactive_editor

```js
let phrase = "Hello, world! Welcome to the world of coding.";
console.log(phrase);  // "Hello, world! Welcome to the world of coding."
let updatedPhrase = phrase.replace("world", "universe");
console.log(updatedPhrase);  // "Hello, universe! Welcome to the world of coding."
```

:::

`world`の最初の出現だけが`universe`に置き換えられていることに注目してください。

JavaScriptの`replace()`メソッドは文字列操作において強力で柔軟なツールです。

個々の文字や単語、あるいは正規表現を使った複雑なパターンまで、文字列の特定の部分を置き換えることができます。

単純な置き換えに最適ですが、大文字・小文字の区別やデフォルトの動作（最初の1つだけ置き換える）を理解すると、より効果的に使えます。
