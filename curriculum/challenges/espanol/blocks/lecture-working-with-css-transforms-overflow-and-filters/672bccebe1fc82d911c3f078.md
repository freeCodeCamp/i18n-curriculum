# --interactive--

La propiedad CSS `filter` es una herramienta poderosa que te permite aplicar efectos gráficos a elementos en una página web. Es especialmente útil para ajustar la presentación visual de imágenes, fondos e incluso texto sin modificar el recurso original.

La propiedad `filter` se puede usar para crear varios efectos, como desenfoque, cambio de color y ajustes de contraste. Vamos a discutir cómo funciona la propiedad de filtro y explorar algunos ejemplos comunes. La sintaxis básica para una propiedad `filter` es sencilla:

```css
selector {
  filter: function(amount);
}
```

Aquí, `function` representa el efecto de filtro específico que quieres aplicar, y `amount` es típicamente un valor que determina la intensidad del efecto. Ahora veamos algunas funciones de filtro comunes y sus usos.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: blur(2px);
}
```

:::

La función `blur` aplica un desenfoque gaussiano al elemento, la cantidad se especifica en píxeles y representa el radio del desenfoque. Esta regla CSS aplicará un desenfoque de 2 píxeles a todas las imágenes de la página. El efecto `blur` puede ser útil para crear profundidad en tu diseño o para ocultar partes de una imagen. 

La función `brightness` ajusta el brillo del elemento. Un valor de `0%` hará que el elemento sea completamente negro, mientras que valores por encima de `100%` aumentarán el brillo. 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: brightness(150%);
}
```

:::

Esta regla de CSS aumenta el brillo del elemento de imagen en `50%`. Los ajustes de brillo se pueden usar para hacer que las imágenes resalten o crear un efecto deslavado.

La función `grayscale` convierte el elemento a escala de grises. La cantidad se define como un porcentaje, donde `100%` es completamente en escala de grises y `0%` deja la imagen sin cambios.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: grayscale(100%);
}
```

:::

Esta regla convertirá el elemento de imagen a escala de grises completa. `grayscale` puede usarse para crear un aspecto vintage o para desactivar la atención en ciertos elementos de una página.

La función `sepia` aplica un tono sepia al elemento. Al igual que la escala de grises, utiliza un valor porcentual: 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: sepia(80%);
}
```

:::

Esta regla aplica un efecto sepia del `80%` al elemento de imagen. El efecto sepia es ideal para crear un aspecto vintage o antiguo.

La función `hue-rotate` aplica una rotación de matiz al elemento. El valor se define en grados y representa una rotación alrededor del círculo de color. 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: hue-rotate(90deg);
}
```

:::

Esta regla rota el tono del elemento de imagen en `90` grados. La rotación del tono se puede usar para crear efectos psicodélicos o para ajustar el esquema de color general de una imagen.

Uno de los aspectos más potentes de la propiedad `filter` es la capacidad para combinar múltiples efectos. Puedes aplicar varios filtros al mismo elemento separándolos con espacios: 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {  
  filter: contrast(150%) brightness(110%) sepia(30%);  
}
```

:::

Esta regla aplica un contraste aumentado, un brillo ligeramente incrementado y un sutil efecto sepia al elemento de imagen. 

Al combinar filtros, puedes crear efectos visuales complejos y únicos adaptados a las necesidades de tu diseño. La propiedad CSS `filter` es una herramienta versátil que permite la manipulación visual creativa de elementos web. 

Aunque hemos cubierto algunas de las funciones de filtro más comunes, hay otras disponibles, como `contrast`, `invert` y `saturate`. Como con cualquier función poderosa, es importante tener cuidado con cómo usas los filtros para mejorar tu diseño sin abrumar a tus usuarios ni comprometer la accesibilidad.
