# --description--

In den vorherigen Lektionen wurden Sie in die Datenvisualisierung eingeführt, also den Prozess, komplexe Daten in klaren, visuellen Formaten darzustellen. In dieser Lektion lernen Sie die D3-Bibliothek kennen.

Die D3-Bibliothek ist eine kostenlose und Open-Source-Bibliothek, mit der Sie überzeugende und effektive Datenvisualisierungen in JavaScript erstellen können.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-1.png" alt="Eine grüne Choroplethenkarte der Vereinigten Staaten, die Daten auf Landkreisebene zeigt, mit einer Farbskala von Hellgrün (3 %) bis Dunkelgrün (60 %+)">


Der Name D3 ist eine Abkürzung für data-driven documents und bezieht sich darauf, wie sie Daten dynamisch an Elemente im Document Object Model (DOM) bindet. Diese Bibliothek ist flexibel und arbeitet natürlich mit dem Web zusammen. Anstatt eine ganz neue grafische Darstellung lernen zu müssen, können Sie Datenvisualisierungen mit `svg` und `canvas` Elementen erstellen.

Wenn Sie zum Beispiel ein Balkendiagramm mit D3 erstellen möchten, können Sie Folgendes alles in einer JS-Datei tun:

* Variablen deklarieren, die die Diagrammabmessungen und Außenabstände repräsentieren
    
* Die horizontale Positionsskala deklarieren
    
* Die vertikale Positionsskala deklarieren
    
* Den SVG-Container mit den Balken und Bezeichnungen erstellen
    
<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-is-the-d3-library-used-for-2.png" alt="Ein Balkendiagramm mit dem Titel United States GDP, das das Bruttoinlandsprodukt von 1947 bis 2015 zeigt, mit Werten, die von fast null auf etwa 18.000 steigen">

Was sind also weitere Gründe, D3 gegenüber anderen Visualisierungstools wie Chart.js, Plotly oder Tableau zu lernen?

Im Gegensatz zu anderen Tools bietet D3 Ihnen viel Kontrolle auf niedriger Ebene, sodass Sie anpassbare und interaktive Visualisierungen nur mit einfachem HTML, CSS und JavaScript erstellen können. Außerdem ist die Bibliothek in JavaScript geschrieben, sodass Sie keine neue Sprache lernen müssen.

In kommenden Workshops und Laboren lernen Sie die Grundlagen der D3-Bibliothek kennen, wie Selektionen, Formen und Skalen. Diese Schlüsselkonzepte lernen Sie, indem Sie Streudiagramme, Balkendiagramme, Dashboard-Apps und mehr erstellen.
