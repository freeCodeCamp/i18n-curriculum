# --description--

Usar as preposições corretas no contexto das rotinas diárias é essencial para uma comunicação clara.

`At` é usado para horários específicos. Por exemplo, `I start work at 9:00`.

`After` é usado para sequências de ações. Por exemplo, `I have lunch after the meeting`.

`To` é usado para indicar direção ou destino. Por exemplo, `I go to the office by bus`.
