# --description--

向量 $\mathbf{a}$ 的长度或范数通常表示为 $\\| \mathbf{a} \\|$。 它可以计算为其各分量平方和的平方根： \\[ \\| \mathbf{a} \\| = \sqrt{a_1^2 + a_2^2 + \ldots + a_n^2} \\]

计算向量范数并从 `norm` 方法返回结果。
