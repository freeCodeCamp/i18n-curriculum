# --description--

`Present Perfect` は、現在と過去のつながりを示すために使われる時制です。行動の時点は現在より前ですが特定されておらず、使う人は通常、行動自体よりもその結果に関心があります。例えば：

- `I played this game yesterday.` - これは `past simple` 時制です。ゲームをした行動が昨日という特定の過去の時点で起こったことを示しています。

- `I have played this game before.` - これは `present perfect` 時制です。ゲームをした行動が特定されていない過去の時点で起こったことを示しています。

- `I've played this game before.` - この文も `present perfect` 時制で、`I've` の短縮形 `I have` を使っています。前の例と同様に、ゲームをした行動が特定されていない過去の時点で起こったことを示しています。
