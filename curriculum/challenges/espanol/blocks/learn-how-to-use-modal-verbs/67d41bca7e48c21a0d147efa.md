`apply to`

### --feedback--

Estas dos palabras significan que algo es relevante o afecta a un grupo particular.

# --explanation--

`Does that apply to...` se usa para preguntar si una regla o requisito es relevante para un grupo específico. Por ejemplo:

- `Does that apply to remote workers too?` - ¿Es esto lo mismo para las personas que trabajan desde casa?

- `Does that apply to projects outside of our department?` - ¿Esta regla también cubre proyectos de otros departamentos?
