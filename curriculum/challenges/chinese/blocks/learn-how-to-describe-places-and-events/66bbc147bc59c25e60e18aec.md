# --explanation--

`Present Continuous` 时态用于描述正在发生的动作。要表达某事**没有**发生，使用否定形式。 

在句子 `The live preview isn't showing` 中，`isn't` 是 `is not` 的缩写，使句子是否定的。 

这表明实时预览未按预期工作。因此，Tom 正在报告实时预览当前在 Visual Studio Code 中不可见。
