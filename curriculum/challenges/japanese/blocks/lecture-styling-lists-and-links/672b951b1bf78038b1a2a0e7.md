# --interactive--

CSSでは、`list-style` プロパティを使ってウェブページ上のリストの見た目を制御します。

番号付きリスト（`ol`）や番号なしリスト（`ul`）のどちらを扱う場合でも、`list-style` プロパティを使うことでリスト項目の表示方法をカスタマイズできます。

`list-style` プロパティは実際には3つの他のプロパティの省略形プロパティです：

- `list-style-type`
- `list-style-position`
- `list-style-image`

それぞれがリストの見た目を定義する異なる役割を持っています。

`list-style-type` プロパティは、リストで使われる箇条書きの種類や番号のタイプを指定できます。

番号なしリストの場合、ディスク、サークル、スクエアなどのいくつかの箇条書きスタイルから選べます。

番号付きリストの場合は、10進数、ローマ数字、アルファベットなどの異なる番号付けシステムを使えます。

`list-style-type` を使った例を見てみましょう：

:::interactive_editor

```html
<ul style="list-style-type: square;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

この例では、番号なしリストの箇条書きがスクエアに変更されています。

`list-style-type` プロパティは3つの中で最もよく使われるもので、リストの箇条書きや番号のスタイルの見た目に直接影響します。

`list-style-position` プロパティは、箇条書きや番号がリスト項目の内容に対してどの位置に表示されるかを制御します。使える値は`inside` と `outside` の2つです。

`outside` の値を使うと、箇条書きや番号は内容の外側に表示され、これはデフォルトの動作です。

一方、`inside` の値を使うと、箇条書きや番号は内容の内側に表示され、テキストが折り返されて箇条書きや番号と揃うことがあります。

`list-style-position` を使った例を見てみましょう：

:::interactive_editor

```html
<ul style="list-style-position: inside;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
<ul style="list-style-position: outside;">
  <li>Item 4</li>
  <li>Item 5</li>
  <li>Item 6</li>
</ul>
```

:::

この例では、2つの異なる番号なしリストタグに対して`inside` と `outside` の値が指定されています。

`list-style-position` プロパティは、特に1つのリスト項目内に複数行のテキストがある場合に、リスト内容の整列を制御したいときに役立ちます。

`list-style-image` プロパティは、リスト項目の箇条書きに画像を使うことを可能にします。リストに独自のビジュアルスタイルを加えたいときに便利です。

`list-style-image` を使った例を見てみましょう：

:::interactive_editor

```html
<head>
  <style>
    ul {
      list-style-image: url('https://cdn.freecodecamp.org/platform/universal/freecodecamp-org-gravatar.jpeg');
      list-style-position: inside;
    }
  </style>
</head>
<body>
  <ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
  </ul>
</body>
```

:::

この例では、箇条書きがカスタムの freeCodeCamp ロゴに置き換えられ、リストに個性的な雰囲気が加わっています。

`list-style-image` を使うときは、選ぶ画像が小さくてウェブページのデザインに適していることを確認してください。画像が大きすぎたり複雑すぎると、リストの読みやすさが損なわれる場合があります。

3つのプロパティ、`list-style-type`、`list-style-position`、`list-style-image` は1つの`list-style` 省略形プロパティにまとめて指定できます。

省略形での値の順序は重要ではありませんが、3つすべてを一緒に指定できます。

省略形プロパティを使った例を見てみましょう：

:::interactive_editor

```html
<ul style="list-style: square inside url('https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg');">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

この例では、リスト項目はスクエアの箇条書きを使い、内容の内側に配置され、カスタム画像が箇条書きとして使われています。

ただし、画像が利用できないかレンダリングされない場合は、スクエアの箇条書きがフォールバックとして使われます。
