# --description--

Definir a tradução não traduz a string por si só.O método `translate` deve ser chamado na string a ser traduzida com a tabela de tradução como argumento:

```py
my_string = "tamperlot"
translation_table = str.maketrans({'t': 'c', 'l': 'b'})
translated_string = my_string.translate(translation_table)
```

Crie uma variável chamada `translated_card_number` e atribua a ela o resultado da chamada do método `translate` em `card_number` com `card_translation` como argumento.
