# --description--

`Object-oriented programming`（OOP）是一种基于`objects`概念的编程范式，`objects`可以包含数据和代码：数据以字段（通常称为属性或属性）的形式存在，代码以过程（通常称为方法）的形式存在。

`OOP` 的一个常见例子是在像 Java 或 Python 这样的编程语言中创建一个类。这个类定义了对象的结构，包括它们的属性和方法。例如，`Car` 类可能包含像 `color` 和 `model` 这样的属性，以及像 `drive()` 和 `brake()` 这样的方法。

术语 `oriented` 在此上下文中表示对齐或指向特定目的或原则。

例如，freeCodeCamp 的响应式网页设计课程被描述为 `project-oriented`，因为它围绕完成项目进行结构化，这意味着教学方法侧重于实际应用，而不仅仅是理论知识。
