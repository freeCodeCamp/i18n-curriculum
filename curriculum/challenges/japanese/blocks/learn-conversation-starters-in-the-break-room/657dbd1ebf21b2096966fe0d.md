# --description--

`do` と `does` を使って `present simple` 時制の質問をします。

`do` は `I`、`you`、`we`、`they` と一緒に使います。例えば、`Do you like pizza?`（好みについて尋ねる場合）です。

`does` は `he`、`she`、`it` と一緒に使います。例えば、`Does he work in an office?`（彼の仕事について尋ねる場合）です。

`do` または `does` の後は、動詞は基本形のままで（`-s` は使いません）ということを覚えておいてください。
