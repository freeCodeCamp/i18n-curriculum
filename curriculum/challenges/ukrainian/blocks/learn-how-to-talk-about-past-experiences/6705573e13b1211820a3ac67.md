# --explanation--

Заперечна форма дієслова `to be` у часі `Simple Past` — це `was not` або `wasn't` для однини, і `were not` або `weren't` для множини. Вона використовується, щоб описати те, що не відбулося або не було правдою. Наприклад:  

- Однина: `The code wasn't running as expected.` — Виникала проблема з кодом.

- Множина: `The changes weren't implemented correctly.` — Помилки в реалізації спричинили баги.

У реченні Джеймса він використовує `wasn't`, щоб пояснити, що система не змогла підключитися до бази даних. Заперечні речення з `wasn't` або `weren't` вказують на те, що не відбулося або не працювало так, як очікувалося.
