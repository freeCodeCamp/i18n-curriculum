# --explanation--

`Doable` bedeutet, dass etwas möglich oder erreichbar ist. Es wird oft in informellen Gesprächen verwendet, um auszudrücken, dass eine Aufgabe oder ein Ziel erledigt werden kann. Zum Beispiel:

- `Finishing this project by Friday is doable.` – Das bedeutet, dass es möglich ist, das Projekt bis Freitag abzuschließen.

- `Is it doable within our budget?` – Das fragt, ob etwas mit dem verfügbaren Geld erreicht werden kann.
