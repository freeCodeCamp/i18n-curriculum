# --description--

表達式 `That/it sounds` 用來描述某事看起來或顯得如何，通常是根據聽到的資訊或所給予的印象。 例如，`That sounds like a great idea!` 這顯示該想法根據所聽到的內容看起來不錯。

同樣地，`That/it looks` 可用來根據外觀描述某物的樣子，例如在 `It looks beautiful.`。

字 `convenient` 是一個形容詞，用來描述適合某人需求、易於使用或有助於更輕鬆快速完成事情的事物。它通常指使任務變得不那麼困難或節省時間的東西。

例如，`Online shopping is convenient because you can do it from home.`
