# --description--

`Object-oriented programming` (OOP) ist ein Programmierparadigma, das auf dem Konzept von `objects` basiert, welche Daten und Code enthalten können: Daten in Form von Feldern (oft bekannt als Attribute oder Props) und Code in Form von Prozeduren (oft bekannt als Methoden).

Ein häufiges Beispiel für `OOP` ist das Erstellen einer Klasse in einer Programmiersprache wie Java oder Python. Diese Klasse definiert die Struktur von Objekten, einschließlich ihrer Attribute und Methoden. Zum Beispiel könnte eine `Car`-Klasse Attribute wie `color` und `model` sowie Methoden wie `drive()` und `brake()` enthalten.

Der Begriff `oriented` bedeutet in diesem Zusammenhang ausgerichtet oder auf einen bestimmten Zweck oder ein Prinzip fokussiert.

Zum Beispiel wird der Responsive-Webdesign-Lehrplan von freeCodeCamp als `project-oriented` beschrieben, weil er um das Abschließen von Projekten strukturiert ist, was bedeutet, dass die Lehrmethode auf praktische Anwendung und nicht nur auf theoretisches Wissen ausgerichtet ist.
