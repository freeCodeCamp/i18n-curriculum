# --description--

Triplet la Pythagorean ni seti ya nambari tatu asilia, `a` &lt; `b` &lt; `c`, kwa ambayo,

<div style='text-align: center;'><var>a</var><sup>2</sup> + <var>b</var><sup>2</sup> = <var>c</var><sup>2</sup></div>

Kwa mfano, 3<sup>2</sup> + 4<sup>2</sup> = 9 + 16 = 25 = 5<sup>2</sup>.

Kuna triplet moja tu la Pythagorean ambalo `a` + `b` + `c` = 1000. Tafuta bidhaa `abc` ili `a` + `b` + `c` = `n`.

function specialPythagoreanTriplet(n) {
  for (let a = 1; a <= n / 2; a++) {
    for (let b = a + 1; b <= n / 2; b++) {
      const c = Math.sqrt(a * a + b * b);

      if ((a + b + c) % n == 0) {
        const factor = n / (a + b + c);

        return (a * b * c) * (factor ** 3);
      }
    }
  }
}
