# --interactive--

JavaScriptを学び始めると、数学的な操作を行うことがよくあるとすぐに気づくでしょう。基本的な算術演算子で簡単な計算はできますが、JavaScriptにはより複雑な数学的課題に対応するための組み込み`Math`オブジェクトがあります。

この便利なツールは、高度な計算を簡単に行い、数値を操作するためのさまざまなメソッドを提供しています。これらのメソッドを見て、コーディングをどのように簡単にできるかを探ってみましょう。

`Math.random()`メソッドは、`0`（含む）から`1`（含まない）までの範囲でランダムな浮動小数点数を生成します。つまり、出力される可能性があるのは`0`ですが、実際に`1`に達することはありません。以下は`Math.random()`メソッドを使った例です：

:::interactive_editor

```js
const randomNum = Math.random();

console.log(randomNum);
// any number between 0 and 1 – 0 inclusive and 1 exclusive
```

:::

`Math.min()`と`Math.max()`はどちらも数値の集合を受け取り、それぞれ最小値と最大値を返します。これらのメソッドを使った例はこちらです：

:::interactive_editor

```js
const smallest = Math.min(1, 5, 3, 9);
console.log(smallest); // 1

const largest = Math.max(1, 5, 3, 9);
console.log(largest); // 9
```

:::

最初の`console.log()`は、`1`がその数値リストの中で最も小さいため、`1`をログに出力します。そして2つ目の`console.log()`は、`9`がそのリストの中で最大の数値なので、`9`をログに出力します。

数値を最も近い整数に切り上げたり切り下げたりしたい場合は、`Math.ceil()`と`Math.floor()`メソッドを使うことができます。以下は`Math.ceil()`を使った例です：

:::interactive_editor

```js
console.log(Math.ceil(4.3)); // 5
```

:::

`Math.ceil()`は`4.3`を最も近い上の整数に切り上げます。この場合は`5`です。次に数値を切り下げる例を見てみましょう：

:::interactive_editor

```js
console.log(Math.floor(4.7)); // 4
```

:::

`Math.floor()`は`4.7`を最も近い下の整数に切り下げます。この場合は`4`です。`Math.round()`は`Math.ceil()`と`Math.floor()`のハイブリッドです。小数点以下を考慮して数値を最も近い整数に丸めます：

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

小数点以下が`5`未満なら切り下げられ、小数点以下が`5`以上なら切り上げられます。`Math.floor()`と`Math.random()`の実用的な使い方は、2つの整数の間でランダムな数値を生成することです。構文は以下の通りです：

:::interactive_editor

```js
const max = 10;
const min = 5;
const randomNum = Math.floor(Math.random() * (max - min + 1)) + min;
console.log(randomNum);
```

:::

`20`から`1`の間でランダムな数値を生成する例は次のようになります：

:::interactive_editor

```js
const randomNumBtw1And20 = Math.floor(Math.random() * 20) + 1;
console.log(randomNumBtw1And20);
```

:::

もう一つ便利な`Math`メソッドは`Math.trunc()`メソッドです。`Math.trunc()`は数値の小数部分を取り除き、丸めずに整数部分だけを返します：

:::interactive_editor

```js
console.log(Math.trunc(2.9)); // 2
console.log(Math.trunc(9.1)); // 9
```

:::

数値の平方根や立方根を求めたい場合は、それぞれ`Math.sqrt()`と`Math.cbrt()`メソッドを使うことができます：

:::interactive_editor

```js
console.log(Math.sqrt(81)); // 9
console.log(Math.cbrt(27)); // 3
```

:::

最初のログ文は、`9`の平方根が`81`なので`9`をログに出力します。2つ目のログ文は、`3`の立方根が`27`なので`3`をログに出力します。数値の絶対値を求めたい場合は、`Math.abs()`メソッドを使います：

:::interactive_editor

```js
console.log(Math.abs(-5)); // 5
console.log(Math.abs(5)); // 5
```

:::

`Math.abs()`は数値の絶対値を返し、負の数を正の数に変換します。最後に見ていくメソッドは`Math.pow()`メソッドです：

:::interactive_editor

```js
console.log(Math.pow(2, 3)); // 8
console.log(Math.pow(8, 2)); // 64
```

:::

`Math.pow()`は2つの数値を受け取り、最初の数値を2つ目の数値のべき乗にします。`Math`オブジェクトには他にも多くのメソッドがあり、自分で探求することができます。しかし、これらはJavaScriptのコードベースでよく使われるメソッドのほんの一部です。
