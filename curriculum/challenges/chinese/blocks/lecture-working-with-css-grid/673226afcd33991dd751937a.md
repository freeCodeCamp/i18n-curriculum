# --interactive--

`grid-column` 和 `grid-row` 属性允许你指定网格布局中网格项的水平和垂直位置。

换句话说，它们都允许你通过引用网格线来控制网格项的开始和结束位置。 这些网格线是分隔已经使用 `grid-template-rows` 和 `grid-template-columns` 属性定义的行和列的边界。

让我们看一下 `grid-row` 和 `grid-column` 的语法，并给出一个实际示例。

这是 `grid-row` 属性的语法：

```css
grid-row: <start-line> / <end-line>;
```

这是 `grid-column` 的：

```css
grid-column: <start-line> / <end-line>;
```

`<start-line>` 是项开始的网格线，`<end-line>` 是项结束的网格线。两者均为从 1 开始计数，即从 1 而非 0 开始计数。

记住，行的网格线是根据 `grid-template-rows` 属性中指定的行数生成的。 这同样适用于具有 `grid-template-columns` 属性的列。

对于实际示例，让我们从浏览器同时使用 `grid-template-rows` 和 `grid-template-columns` 属性生成的行开始。

这是一个包含 4 列和 3 行的示例网格：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}
```

:::

观察网格容器（类 `grid`）会发现，每一行和每一列都有两条线——行或列开头的起始线和行或列末尾的结束线。

你可以针对这些行，开始使用 `grid-row` 和 `grid-column` 属性来确定应该放置项目的位置。

这是如何使第一个网格项占据前两列的方法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
}
```

:::

这样，你的意思就是第一个网格项应该从第 1 列开始并在第 3 列之前结束。

第一个项目现在占据两列，第四个项目被推到第二行。 酷！

如果你还想让第一个项目占用两行，可以指定 `grid-row` 为 `1 / 3`：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
  grid-row: 1 / 3;
}
```

:::

你还可以使用 `span` 关键字来告诉网格项跨哪一行和哪一列。 例如，`1 / 3` 与 `1 / span 2` 是相同的：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / span 2;
  grid-row: 1 / span 2;
}
```

:::

你可以继续将此技术应用到网格上的任何项，并将它们放置在你想要的任何位置。 

继续这个过程，这就是我们如何用这些项目实现瀑布流布局的。

```html
  <div class="item5">5</div>
```

```css
  display: grid;

  grid-template-rows: repeat(4, 100px); /* 4 equal rows */

  grid-column: 1 / span 2;
}

.item4 {
  grid-column: 1 / span 3;
}

.item6 {
  grid-column: 1 / span 2;
}

.item7 {
  grid-column: 3 / span 2;
}
```
