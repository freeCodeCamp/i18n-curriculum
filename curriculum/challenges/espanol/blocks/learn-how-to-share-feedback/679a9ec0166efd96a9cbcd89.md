# --explanation--

La estructura `it's + adjective + to` se usa para describir la naturaleza de una acción. Por ejemplo:

`It's necessary to prepare before an important presentation.` - Esto significa que preparar es una acción esencial antes de presentar.

En esta oración, `it's healthy to have` significa que tener desacuerdo es beneficioso.
