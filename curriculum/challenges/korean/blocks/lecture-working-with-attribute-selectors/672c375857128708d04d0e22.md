# --interactive--

HTML에서 순서 있는 목록을 다룰 때, `type` 속성은 숫자, 알파벳, 로마 숫자 등 사용되는 번호 매기기 스타일을 지정할 수 있게 해줍니다.

`type` 속성은 `ol` (순서 있는 목록) 요소에 설정하여 번호 매기기 형식을 제어할 수 있습니다:

- `1`는 숫자 목록(예: 1, 2, 3, ...)용입니다,

- `A`는 대문자 알파벳 목록(예: A, B, C, ...)용입니다,

- `a`는 소문자 알파벳 목록(예: a, b, c, ...)용입니다,

- `I`는 대문자 로마 숫자 목록(예: I, II, III, ...)용입니다,

- `i`는 소문자 로마 숫자 목록(예: i, ii, iii, ...)용입니다.

다음은 특정 `type` 속성을 가진 순서 있는 목록을 선택하기 위해 속성 선택자를 사용하는 방법입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="A"] {
  color: purple;
  font-weight: bold;
}
```

:::

이 예제에서 CSS 규칙은 `type="A"`로 지정된 대문자 알파벳 번호 매기기를 사용하는 모든 순서 있는 목록에 보라색 텍스트와 굵은 글꼴 두께를 적용합니다.

마찬가지로, `type="i"`를 가진 소문자 로마 숫자 번호 매기기를 사용하는 순서 있는 목록을 선택할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="i">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="i"] {
  color: green;
}
```

:::

이 규칙은 소문자 로마 숫자 번호 매기기를 사용하는 모든 순서 있는 목록의 텍스트 색상을 녹색으로 변경합니다.

`type` 속성에 따라 순서 있는 목록의 스타일을 지정할 수 있는 기능은 문서 내 목록의 모양을 더 세밀하게 제어할 수 있게 하여, 콘텐츠 구조에 따라 맞춤 설정할 수 있게 해줍니다.
