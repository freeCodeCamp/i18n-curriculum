# --description--

Il pig latin è un modo di modificare le parole inglesi seguendo regole di trasformazione specifiche.

**Obiettivo:** Completa le storie utente qui sotto e fai passare tutti i test per completare il laboratorio.

**Storie utente:**

1. Dovresti creare una funzione `translatePigLatin` che accetta una stringa come argomento.
1. Se la stringa inizia con una consonante, la funzione dovrebbe prendere la prima consonante o il gruppo di consonanti, spostarlo alla fine della parola, aggiungere `ay` e restituire il risultato.
1. Se la stringa inizia con una vocale, la funzione dovrebbe aggiungere `way` alla fine e restituire il risultato.
1. La funzione dovrebbe gestire stringhe in cui la prima vocale si trova nel mezzo della parola e restituire la stringa trasformata correttamente.
1. Se la stringa non contiene vocali, la funzione dovrebbe aggiungere `ay` alla fine e restituire il risultato.

Nota: Per il contesto di questo laboratorio, le vocali sono `a`, `e`, `i`, `o` e `u`. La lettera `y` non è considerata una vocale.

```js
function translatePigLatin(str) {
  if (isVowel(str.charAt(0))) return str + "way";
  let front = [];
  str = str.split('');
  while (str.length && !isVowel(str[0])) {
    front.push(str.shift());
  }
  return [].concat(str, front).join('') + 'ay';
}

function isVowel(c) {
  return ['a', 'e', 'i', 'o', 'u'].indexOf(c.toLowerCase()) !== -1;
}
```
