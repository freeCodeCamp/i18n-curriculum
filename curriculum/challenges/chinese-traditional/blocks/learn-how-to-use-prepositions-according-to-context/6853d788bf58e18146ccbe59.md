# --explanation--

`Ten developers` 是一種報告目前有多少人在一個專案上工作的方式。使用數字有助於清楚顯示可用的資源。例如：

`We have five designers and ten developers for this release.`－這表示團隊現在包含這些人。

`On the team` 意味著成為一個在專案中共同合作的群組的一部分。例如：

`She is on the team responsible for testing the app.`－這表示她屬於負責該工作的群組。
