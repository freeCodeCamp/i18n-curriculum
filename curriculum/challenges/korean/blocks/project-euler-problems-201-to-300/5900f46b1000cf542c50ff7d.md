# --description--

$f(n)$을 $n$의 각 자리 숫자의 팩토리얼 합으로 정의하세요. 예를 들어, $f(342) = 3! + 4! + 2! = 32$입니다.

$sf(n)$을 $f(n)$의 각 자리 숫자의 합으로 정의하세요. 따라서 $sf(342) = 3 + 2 = 5$입니다.

$g(i)$를 $sf(n) = i$를 만족하는 가장 작은 양의 정수 $n$으로 정의하세요. $sf(342)$가 5이지만, $sf(25)$도 5이고, $g(5)$가 25임을 확인할 수 있습니다.

$sg(i)$를 $g(i)$의 각 자리 숫자의 합으로 정의하세요. 따라서 $sg(5) = 2 + 5 = 7$입니다.

또한, $g(20)$이 267이고 $1 ≤ i ≤ 20$에 대해 $\sum sg(i)$가 156임을 확인할 수 있습니다.

$1 ≤ i ≤ 150$에 대해 $\sum sg(i)$는 얼마인가요?
