# --explanation--

`Might` 用來表示某事有可能發生，但不確定。Jessica 表示這個錯誤 *可能* 會造成問題，但尚未確認。與使用 `Simple Present` 比較，例如 `we've discovered a critical bug that affects the user interface`，表示這個錯誤確實會造成問題。其他範例：

- `The delay might cause problems.`－這是可能的，但不一定會發生。

- `The delay causes problems.`－它絕對會產生問題。
