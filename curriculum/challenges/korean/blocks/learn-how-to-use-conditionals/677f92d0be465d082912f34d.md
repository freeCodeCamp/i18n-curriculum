# --explanation--

질문 `Did you check that?`은 누군가가 무언가를 확인하거나 점검했는지 묻고 있습니다.

이 상황에서 Mark는 Sarah가 전원 케이블 연결이 안전한지 확인했는지 알고 싶어 합니다. 다른 예시는 다음과 같습니다:

- `Did you check that the windows are closed?` - 누군가 창문이 단단히 닫혔는지 확인했는지 묻는 의미입니다.

- `Did you check that the light is off?` - 누군가 불이 꺼졌는지 확인했는지 묻는 의미입니다.
