# --description--

Eine Folge ist definiert als:

- $g_k = 1$, für $0 ≤ k ≤ 1999$
- $g_k = g_{k - 2000} + g_{k - 1999}$, für $k ≥ 2000$.

Finden Sie $g_k$ mod 20092010 für $k = {10}^{18}$.
