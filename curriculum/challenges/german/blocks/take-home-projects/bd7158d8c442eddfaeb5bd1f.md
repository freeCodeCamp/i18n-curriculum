**Ziel:** Erstellen Sie eine App, die funktional ähnlich ist wie diese: <a href="https://codepen.io/freeCodeCamp/full/Myvqmo/" target="_blank" rel="noopener noreferrer nofollow">https://codepen.io/freeCodeCamp/full/Myvqmo/</a>.

Die Twitch API ist eine RESTful API, die Entwickler:innen ermöglicht, kreative Integrationen für die größere Twitch-Community zu erstellen.

Erfüllen Sie die untenstehenden User Stories und bringen Sie alle Tests zum Bestehen. Verwenden Sie beliebige Bibliotheken oder APIs, die Sie benötigen. Verleihen Sie der App Ihren persönlichen Stil.

**User Story:** Ich kann sehen, ob freeCodeCamp gerade auf Twitch.tv streamt.

**User Story:** Ich kann auf die Statusanzeige klicken und werde direkt zum Twitch.tv-Kanal von freeCodeCamp weitergeleitet.

**User Story:** Wenn ein Twitch-Nutzer gerade streamt, kann ich zusätzliche Details darüber sehen, was er streamt.

**Hinweis:** Hier ist ein Array mit den Twitch.tv-Benutzernamen von Personen, die regelmäßig streamen: `["ESL_SC2", "OgamingSC2", "cretetion", "freecodecamp", "storbeck", "habathcx", "RobotCaleb", "noobs2ninjas"]`

**AKTUALISIERUNG:** Aufgrund einer Änderung der API-Nutzungsbedingungen verlangt Twitch.tv jetzt einen API-Schlüssel. Wir haben jedoch eine Lösung gebaut. Verwenden Sie <a href="https://twitch-proxy.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://twitch-proxy.freecodecamp.rocks/</a> anstelle der Basis-URL der Twitch-API, und Sie können weiterhin Kontoinformationen abrufen, ohne sich für einen API-Schlüssel anmelden zu müssen.
