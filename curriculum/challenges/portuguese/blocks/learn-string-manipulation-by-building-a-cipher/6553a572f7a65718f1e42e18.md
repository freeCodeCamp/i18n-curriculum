# --description--

Uma função é essencialmente um bloco reutilizável de código.Você já conheceu algumas funções internas, como `print()`, `find()` e `len()`.Mas você também pode definir funções personalizadas assim:

```py
def function_name():
    <code>
```

Uma declaração de função começa com a palavra-chave `def` seguida pelo nome da função — um nome de variável válido — e um par de parênteses.A declaração termina com dois pontos.

Logo após a sua variável `shift`, declare uma função chamada `caesar` e indente todas as linhas seguintes para dar um corpo à sua nova função.
