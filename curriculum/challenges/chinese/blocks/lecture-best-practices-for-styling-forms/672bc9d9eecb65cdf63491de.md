# --interactive--

让我们学习一下何时使用 `appearance: none` 来解决搜索输入、复选框和单选按钮的样式问题。

浏览器会对很多元素应用默认样式。 对于输入元素，使用 CSS 对其进行样式设置的能力可能会受到很大限制。 因此，你可能需要使用 `appearance: none` 来隐藏元素的某些默认样式，然后创建自己的样式。 例如，这将隐藏复选框输入的默认复选框，允许你使用绿色勾和红色叉等自定义标识来显示状态。 对于搜索输入，基于 WebKit 的浏览器会显示一个默认搜索图标和一个取消按钮。 隐藏这些可让你创建自己的标识，并显示在所有浏览器上。

.checkbox {

WebKit 是一个帮助网络浏览器显示网站的软件引擎。 Safari 等浏览器使用 WebKit 来确保网页正常显示和运行。 `appearance: none` CSS 属性可让你完全控制样式，但也有一些需要注意的地方。 输入元素的默认交互组件包括焦点和错误标识等功能，你需要确保这些功能不会丢失。

创建一致的跨平台样式是使用此属性的一个重要原因。 你还可以使用它来确保移动设备上的点击目标足够大，或者复选框的颜色有足够的对比度。
