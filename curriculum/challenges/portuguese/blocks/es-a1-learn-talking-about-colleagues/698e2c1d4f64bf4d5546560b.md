# --description--

Em espanhol, existem duas formas comuns de falar sobre origem.

A primeira é usando **`Es de` + país**. Por exemplo:

`Es de Venezuela` - Ele é da Venezuela.

A segunda é usando o adjetivo de nacionalidade. Por exemplo:

`Es venezolano` - Ele é venezuelano.

Nacionalidades **não** são capitalizadas.

# --explanation--

Esteban diz `Sí, es brasileño`, que significa "Sim, ele é brasileiro" na forma masculina.

É muito importante incluir o acento na letra `i` da palavra `Sí`. Com o acento, essa palavra significa "sim". Sem o acento, pelo contexto, essa palavra pode significar "se", como condicional.

Lembre-se que nacionalidades não são capitalizadas em espanhol.
