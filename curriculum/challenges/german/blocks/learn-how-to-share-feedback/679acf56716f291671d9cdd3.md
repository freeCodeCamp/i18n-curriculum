# --explanation--

`Too much + noun` wird verwendet, um eine übermäßige Menge von etwas anzuzeigen, was oft negativ bewertet wird. Zum Beispiel:

`Too much noise in the office makes it hard to concentrate.` – Das bedeutet, dass eine übermäßige Geräuschmenge die Konzentration negativ beeinflusst.

In diesem Dialog bedeutet `too much disagreement` ein übermäßiges Maß an Meinungsverschiedenheiten, das den Fortschritt verlangsamen könnte.
