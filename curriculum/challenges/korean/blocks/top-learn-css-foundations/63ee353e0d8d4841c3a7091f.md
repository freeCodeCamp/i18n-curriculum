# --description--

좋아요, 지금까지 꽤 많은 내용을 다뤘습니다. 이제 남은 것은 이 모든 CSS를 HTML에 추가하는 방법을 살펴보는 것입니다. 이를 위한 세 가지 방법이 있습니다.

외부 CSS는 가장 흔히 접하게 될 방법으로, CSS를 별도의 파일로 만들고 HTML의 여는 `<head>` 태그와 닫는 `<head>` 태그 사이에 `<link>` 요소를 링크하는 방식입니다(이 요소는 닫는 태그가 필요 없는 void 요소입니다).

먼저, HTML 파일의 여는 `<head>` 태그와 닫는 `<head>` 태그 사이에 void 요소인 `<link>` 태그를 추가합니다. `href` 속성은 CSS 파일의 위치를 나타내며, 절대 URL이 될 수도 있고, 여기서 사용할 것처럼 HTML 파일 위치를 기준으로 한 상대 URL일 수도 있습니다. 위 예제에서는 두 파일이 같은 디렉터리에 있다고 가정합니다. `rel` 속성은 필수이며, HTML 파일과 링크된 파일 간의 관계를 지정합니다.

그다음 새로 만든 `styles.css` 파일 안에는 선택자(`div`와 `p`)가 있고, 그 뒤에 여는 중괄호와 닫는 중괄호 쌍이 있어 “선언 블록”을 만듭니다. 마지막으로 선언 블록 안에 선언들을 넣습니다. `color: white; `는 하나의 선언이며, `color`은 속성, `white`은 값이고, `background-color: black;`는 또 다른 선언입니다.

파일 이름에 관한 참고 사항: `styles.css`는 여기서 파일 이름으로 사용한 예시일 뿐입니다. 파일 형식이 `.css`인 한 원하는 이름으로 파일을 지을 수 있지만, 보통 “style”이나 “styles”가 가장 많이 사용됩니다.

이 방법의 장점 몇 가지는 다음과 같습니다:

1. HTML과 CSS를 분리해서 HTML 파일 크기가 작아지고 코드가 더 깔끔해 보입니다.
2. CSS를 한 곳에서만 수정하면 되므로, 비슷한 스타일을 공유하는 여러 페이지가 있는 웹사이트에 특히 편리합니다.
