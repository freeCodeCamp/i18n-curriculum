# --description--

In this task, you'll learn about phrases like `Did you get it?`, `Got it`, and `What else?` These phrases are common in English conversations. `Did you get it?` es una pregunta que pregunta si alguien entendió algo. `Got it` es una forma corta de decir `I understood.` `What else?` se usa para pedir información o elementos adicionales.

Por ejemplo, en una reunión, alguien podría decir, `Did you get it?` y puedes responder, `Got it, what else do we need to discuss?`
