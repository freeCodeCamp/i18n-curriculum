# --description--

$p_n$을 $n$번째 소수라고 하자: 2, 3, 5, 7, 11, ..., 그리고 $r$을 ${(p_n−1)}^n + {(p_n+1)}^n$을 ${p_n}^2$로 나눈 나머지라고 하자.

예를 들어, $n = 3$, $p_3 = 5$일 때, $4^3 + 6^3 = 280 ≡ 5\\ mod\\ 25$입니다.

나머지가 처음으로 $10^9$를 초과하는 가장 작은 $n$의 값은 7037입니다.

나머지가 처음으로 $10^{10}$을 초과하는 가장 작은 $n$의 값을 구하세요.
