# --description--

Wenn von links nach rechts keine Ziffer von der Ziffer links von ihr übertroffen wird, nennt man die Zahl eine steigende Zahl; zum Beispiel 134468.

Ebenso nennt man eine Zahl eine fallende Zahl, wenn keine Ziffer von der Ziffer rechts von ihr übertroffen wird; zum Beispiel 66420.

Wir nennen eine positive ganze Zahl, die weder steigend noch fallend ist, eine „bouncy“-Zahl; zum Beispiel 155349.

Mit zunehmendem n steigt der Anteil der bouncy-Zahlen unterhalb von n so stark an, dass es nur 12.951 Zahlen unter einer Million gibt, die nicht bouncy sind, und nur 277.032 nicht-bouncy Zahlen unter ${10}^{10}$.

Wie viele Zahlen unter einem Googol (${10}^{100}$) sind nicht bouncy?
