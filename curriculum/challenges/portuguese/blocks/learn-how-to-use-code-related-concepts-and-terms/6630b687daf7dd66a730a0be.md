# --description--

A palavra `fundamental` implica que algo é básico ou essencial, assim como a fundação de uma casa.

Por exemplo, `Understanding grammar is fundamental to learning a new language.` Aqui `fundamental` é usado para enfatizar que a gramática é uma parte essencial ou básica do aprendizado de qualquer novo idioma.

A frase `what kind of` é usada para perguntar sobre o tipo ou a natureza de algo.É uma maneira comum de especificar ou classificar algo de forma mais clara.

Por exemplo, quando alguém pergunta `what kind of music do you like?` eles estão perguntando sobre o gênero de música.
