# --explanation--

`Couple` usualment significa dos, pero a veces puede significar un número pequeño o `some`. Por ejemplo:

- `A couple of days` - Puede significar dos días o unos pocos días.

- `A couple of friends` - Puede significar dos amigos o algunos amigos.
