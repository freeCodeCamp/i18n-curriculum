# --description--

الطريقة هي دالة تنتمي إلى كائن أو صنف معين. ستتعلم المزيد عن الأصناف والكائنات والدوال والطرق لاحقًا. في الوقت الحالي، اعلم أن الطرق يجب استدعاؤها على الكائن الذي تنتمي إليه بكتابة اسم الكائن متبوعًا بنقطة ثم استدعاء الطريقة. يوفر Python عددًا من الطرق المدمجة التي يمكنك استخدامها لمعالجة السلاسل النصية. تشمل هذه الطرق، ولكن لا تقتصر على، ما يلي:

- `upper()`: تُعيد سلسلة نصية جديدة مع تحويل جميع الأحرف إلى أحرف كبيرة.
    

```python
my_str = 'hello world'

uppercase_my_str = my_str.upper()
print(uppercase_my_str)  # HELLO WORLD
```

- `lower()`: تُعيد سلسلة نصية جديدة مع تحويل جميع الأحرف إلى أحرف صغيرة.
    

```python
my_str = 'Hello World'

lowercase_my_str = my_str.lower()
print(lowercase_my_str)  # hello world
```

- `strip()`: تُعيد سلسلة نصية جديدة مع إزالة الأحرف المحددة من البداية والنهاية. إذا لم تُمرر معلمة، تزيل الفراغات من البداية والنهاية.
    

```python
my_str = '  hello world  '

trimmed_my_str = my_str.strip()
print(trimmed_my_str)  # "hello world"
```

- `replace(old, new)`: تُعيد سلسلة نصية جديدة مع استبدال كل ظهور لـ `old` بـ `new`.
    

```python
my_str = 'hello world'

replaced_my_str = my_str.replace('hello', 'hi')
print(replaced_my_str)  # hi world
```

- `split(separator)`: تقسم السلسلة النصية عند فاصل محدد إلى قائمة من السلاسل النصية. إذا لم يُحدد فاصل، تقسم عند الفراغات.
    

```python
my_str = 'hello world'

split_words = my_str.split()
print(split_words)  # ['hello', 'world']
```

- `join(iterable)`: تجمع عناصر قابل للتكرار في سلسلة نصية باستخدام فاصل.
    

```python
my_list = ['hello', 'world']

joined_my_str = ' '.join(my_list)
print(joined_my_str)  # hello world
```

- `startswith(prefix)`: تُعيد قيمة منطقية تشير إلى ما إذا كانت السلسلة النصية تبدأ بالبادئة المحددة.
    

```python
my_str = 'hello world'

starts_with_hello = my_str.startswith('hello')
print(starts_with_hello)  # True
```

- `endswith(suffix)`: تُعيد قيمة منطقية تشير إلى ما إذا كانت السلسلة النصية تنتهي باللاحقة المحددة.
    

```python
my_str = 'hello world'

ends_with_world = my_str.endswith('world')
print(ends_with_world)  # True
```

- `find(substring)`: تُعيد فهرس أول ظهور لـ `substring`، أو `-1` إذا لم تجده.
    

```python
my_str = 'hello world'

world_index = my_str.find('world')
print(world_index)  # 6
```

- `count(substring)`: تُعيد عدد مرات ظهور سلسلة فرعية في السلسلة النصية.
    

```python
my_str = 'hello world'

o_count = my_str.count('o')
print(o_count)  # 2
```

- `capitalize()`: تُعيد سلسلة نصية جديدة مع تحويل الحرف الأول إلى كبير وباقي الأحرف إلى صغيرة.
    

```python
my_str = 'hello world'

capitalized_my_str = my_str.capitalize()
print(capitalized_my_str)  # Hello world
```

- `isupper()`: تُعيد `True` إذا كانت كل الأحرف في السلسلة كبيرة و`False` إذا لم تكن كذلك.
    

```python
my_str = 'hello world'

is_all_upper = my_str.isupper()
print(is_all_upper)  # False
```

- `islower()`: تُعيد `True` إذا كانت كل الأحرف في السلسلة صغيرة و`False` إذا لم تكن كذلك.
    

```python
my_str = 'hello world'

is_all_lower = my_str.islower()
print(is_all_lower)  # True
```

- `title()`: تُعيد سلسلة نصية جديدة مع تحويل الحرف الأول من كل كلمة إلى كبير.
    

```python
my_str = 'hello world'

title_case_my_str = my_str.title()
print(title_case_my_str)  # Hello World
```
