# --interactive--

Die `reduce`-Methode ist eine Funktion in JavaScript, mit der Sie ein Array verarbeiten und zu einem einzelnen Wert verdichten können. Dieser einzelne Wert kann eine Zahl, ein String, ein Objekt oder sogar ein weiteres Array sein.

Sie heißt `reduce`, weil sie ein Array auf eine einzige Ausgabe reduziert. Obwohl es anfangs kompliziert erscheinen mag, kann das Verständnis von `reduce` Ihren Code in vielen Situationen erheblich vereinfachen.

Im Kern funktioniert `reduce` dadurch, dass eine Funktion auf jedes Element im Array der Reihe nach angewendet wird und das Ergebnis jeder Berechnung an das nächste Element weitergegeben wird. Diese Funktion wird oft als Reducer-Funktion bezeichnet.

Die Reducer-Funktion nimmt zwei Hauptparameter: einen Akkumulator und den aktuellen Wert. Der Akkumulator ist der Ort, an dem Sie das laufende Ergebnis Ihrer Operationen speichern, und der aktuelle Wert ist das gerade verarbeitete Array-Element.

Schauen wir uns ein Beispiel an, um zu veranschaulichen, wie `reduce` funktioniert:

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];
const sum = numbers.reduce(
  (accumulator, currentValue) => accumulator + currentValue,
  0
);

console.log(sum); // 15
```

:::

In diesem Beispiel verwenden wir `reduce`, um die Summe aller Zahlen im Array zu erhalten.

Die Reducer-Funktion nimmt den Akkumulator (der bei `0` startet, wie durch das zweite Argument von `reduce` angegeben) und addiert jede Zahl dazu.

Das Ergebnis jeder Addition wird zum neuen Akkumulator für die nächste Iteration.

Die `reduce`-Methode kann auch einen Anfangswert als zweiten Parameter annehmen. Dies ist der Wert, mit dem der Akkumulator startet. Im obigen Beispiel haben wir ihn auf `0` gesetzt.

Wenn Sie keinen Anfangswert angeben, verwendet `reduce` das erste Element des Arrays als Anfangsakkumulator und startet den Prozess ab dem zweiten Element.

Eines der großartigen Dinge an `reduce` ist seine Flexibilität. Da Sie die Reducer-Funktion definieren, haben Sie die volle Kontrolle darüber, wie das Array verarbeitet wird und welches Ergebnis Sie erzeugen möchten. Das macht `reduce` extrem mächtig, aber es kann anfangs auch etwas herausfordernd sein, es zu verstehen.

Mit Übung werden Sie den Umgang mit der `reduce`-Methode meistern.
