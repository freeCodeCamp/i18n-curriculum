# --description--

フレーズ `to keep in mind` は、何かを覚えておく、または考慮することを意味します。これは、後で使うために覚えておくべきアドバイスや重要な情報を誰かが伝えるときによく使われます。

たとえば、誰かが「ハイキングの前に天気を確認するのを忘れないで」と言った場合、あなたは `I'll keep that in mind` と答えて、このアドバイスを考慮するつもりであることを示すかもしれません。
