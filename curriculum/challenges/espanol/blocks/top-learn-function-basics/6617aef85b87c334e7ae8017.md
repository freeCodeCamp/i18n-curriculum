# --description--

Las funciones flecha son una forma más moderna de escribir funciones en JavaScript. Proporcionan una sintaxis más compacta en comparación con las expresiones de función tradicionales. Las funciones flecha se definen usando la sintaxis `=>`. 

```js
const add = (a, b) => {
  return a + b;
}
```

En el ejemplo anterior, la función flecha `add` toma dos parámetros `a` y `b` y devuelve su suma. La palabra clave `return` se usa para devolver el resultado de la operación de suma.

Si la función flecha tiene solo una expresión, las llaves `{}` y la palabra clave `return` pueden omitirse. La expresión se devolverá implícitamente.

```js
const add = (a, b) => a + b;
```

En el ejemplo anterior, la función flecha `add` toma dos parámetros `a` y `b` y devuelve su suma. La palabra clave `return` y las llaves `{}` se omiten, y el resultado de la operación de suma se devuelve implícitamente.
