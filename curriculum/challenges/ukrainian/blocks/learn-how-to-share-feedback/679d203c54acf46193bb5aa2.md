# --explanation--

`These` використовується як іменник, коли йдеться про кілька речей, які були згадані раніше, наприклад, проблеми, ідеї чи завдання. Якщо ж ви використовуєте `this`, то маєте на увазі один раніше згаданий елемент. Наприклад:

- `I know we added a lot of new features, but these need to be tested before release.` — тут `these` посилається на `features`, згаданий раніше.

- `This needs more discussion before we proceed.` — тут знову згадується певна тема, яка була раніше, за допомогою `this`.

`Customer satisfaction` позначає, наскільки задоволені або приємно вражені клієнти продуктом чи послугою. Наприклад:

`Improving response time in customer support helps increase customer satisfaction.` — це означає, що швидкі відповіді призводять до більш задоволених клієнтів.
