# --description--

`e-commerce platform` — це застосунок, який дозволяє бізнесам керувати своїми операціями з онлайн-продажів. Зазвичай він включає функції для відображення товарів, обробки замовлень і ведення транзакцій з клієнтами.

Цей термін є важливим у світі онлайн-бізнесу, оскільки він дає змогу компаніям продавати товари та послуги через інтернет.

# --questions--

## --text--

Що таке `e-commerce platform` у контексті онлайн-бізнесу?

## --answers--

Платформа для маркетингу в соціальних мережах.

### --feedback--

Хоча `e-commerce platforms` може використовувати соціальні мережі для маркетингу, вони в першу чергу призначені для операцій з онлайн-продажів.

---

Застосунок для керування онлайн-продажами.

---

Інструмент для дизайну та розробки вебсайтів.

### --feedback--

`e-commerce platform` призначений саме для керування онлайн-продажами, а не лише для дизайну вебсайтів.

---

Система для електронного спілкування з клієнтами.
