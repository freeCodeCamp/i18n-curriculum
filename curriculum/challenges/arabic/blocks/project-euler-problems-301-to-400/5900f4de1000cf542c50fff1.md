# --description--

لنعرّف المثلث الهندسي بأنه مثلث ذو أضلاع صحيحة بحيث تكون الأضلاع $a ≤ b ≤ c$ وتشكل متتالية هندسية، أي أن $b^2 = a \times c$.

مثال على مثلث هندسي هو المثلث الذي أضلاعه $a = 144$، $b = 156$ و $c = 169$.

يوجد $861\\,805$ مثلثًا هندسيًا بمحيط $\text{perimeter} ≤ {10}^6$.

كم عدد المثلثات الهندسية التي توجد بمحيط $\text{perimeter} ≤ 2.5 \times {10}^{13}$؟
