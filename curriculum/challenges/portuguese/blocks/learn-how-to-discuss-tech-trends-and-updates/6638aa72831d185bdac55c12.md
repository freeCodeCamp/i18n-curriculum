# --description--

A frase `give it a go` significa tentar algo, frequentemente pela primeira vez, para ver se será bem-sucedido ou agradável.Sugere uma disposição para tentar algo novo ou diferente.Esta frase é informal e comumente usada no inglês coloquial.

Por exemplo, `You've never played tennis?`Você deveria tentar!
