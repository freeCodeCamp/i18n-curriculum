# --description--

Katika funzo lililopita, ulijifunza jinsi ya kufanya kazi na aina ya data ya tuple. Katika funzo hili, utajifunza kuhusu baadhi ya njia za kawaida utakazotumia unapo fanya kazi na tuples.

Njia ya kwanza tutakayojifunza ni `count()`. Njia hii hutumika kubaini ni mara ngapi kipengee kinaonekana katika tuple. Hapa kuna mfano wa kuangalia ni mara ngapi mfuatano wa herufi `Rust` unaonekana katika tuple iitwayo `programming_languages`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

Kwa kuwa `Rust` inaonekana mara mbili katika tuple, njia ya `count()` inarudisha nambari `2`. Ikiwa kipengee kilichotajwa katika kitendakazi cha `count()` hakipo kabisa katika tuple, basi thamani ya return ni `0`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

Kama hakuna hoja zitakazopitishwa katika kitendakazi cha `count()`, basi Python itatoa `TypeError`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count()

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: tuple.count() takes exactly one argument (0 given)
"""
```

Njia inayofuata tutakayochunguza ni njia ya `index()`. Njia hii hutumika kupata kielezo ambacho kipengee fulani kiko katika tuple. Hapa kuna mfano wa kutumia njia ya `index()` kupata kielezo cha mfuatano wa herufi `Java`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

Ikiwa kipengee kilichotajwa hakiwezi kupatikana, basi Python itatoa `ValueError`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: tuple.index(x): x not in tuple
"""
```

Jambo lingine unaloweza kufanya na njia ya `index()` ni kupitisha hoja za hiari za kielezo cha kuanzia na kielezo cha kuacha. Hapa kuna mfano wa kupitisha kielezo cha kuanzia cha hiari:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

Katika mfano huu, tunabainisha mahali pa kuanza kutafuta mfuatano wa herufi `Python`. Kwa kupitisha nambari `3` kama hoja ya pili kwa kitendakazi cha `index()`, tunabainisha kuanza kutafuta kwenye kielezo `3`. Kwa kuwa `Python` inaonekana mara mbili katika tuple, kitendakazi cha `index()` kitarudisha kielezo `5` badala ya kielezo `2` kwa sababu ya matumizi ya hoja ya hiari ya kielezo cha kuanzia.

Pia unaweza kupitisha kielezo cha hiari cha kuacha. Hapa kuna mfano uliobadilishwa wa kubainisha kielezo cha kuanzia na kuacha:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

Sasa matokeo ni kielezo `2` kwa sababu tunaanza kutafuta kwenye kielezo `2`, na kutafuta hadi, lakini si pamoja na, kielezo `5`.

Kitendakazi kingine kinachotumika sana na tuples ni kitendakazi cha `sorted()`. Katika funzo lililopita ulijifunza kuhusu njia ya `sort()` kwa orodha. Vizuri, kitendakazi cha `sorted()` kinaweza kutumika kwa iterable yoyote ikiwa ni pamoja na tuples.

Hapa kuna mfano wa kuunda orodha mpya ya nambari kwa kutumia kitendakazi cha `sorted()`:

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

Kitendakazi cha `sorted()` kila mara kitaunda orodha mpya ya thamani zilizopangwa. Hii ni tofauti na njia ya `sort()` ambayo hupanga vipengee vya orodha mahali pale na hairudishi orodha mpya.

Ikiwa unahitaji kubinafsisha tabia ya kupanga kwa iterable, unaweza kutumia hoja za hiari za `reverse` na `key`. Hapa kuna mfano wa kutumia hoja ya `key` kupanga vipengee katika tuple kwa urefu:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

Ikiwa unataka kuunda orodha mpya ya thamani kwa mpangilio wa kinyume, basi unaweza kutumia hoja ya `reverse` kama ifuatavyo:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

Tuples ni aina ya data inayotumika sana katika Python. Kuelewa jinsi ya kufanya kazi nazo, pamoja na baadhi ya njia na vitendakazi vinavyosaidia, kutakusaidia kuandika msimbo wenye ufanisi zaidi.
