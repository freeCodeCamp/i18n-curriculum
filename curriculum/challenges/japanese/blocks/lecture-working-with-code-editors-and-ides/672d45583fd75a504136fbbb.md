# --description--

VS Code をインストールするには、まずご自身のオペレーティングシステムに合ったインストーラーを入手するために公式ウェブサイトを訪問してください。Visual Studio のウェブサイトではなく、必ず VS Code のウェブサイトにアクセスしてください。

Windows では、EXE インストーラーをダウンロードできます。ダウンロード後、そのファイルを実行するとセットアップウィザードがインストール手順を案内します。

Mac では、実行可能な `.app` を含む zip アーカイブをダウンロードできます。また、Homebrew を使って VS Code をインストールすることも可能です。

Mac の Terminal アプリを開き、次のコマンドを実行してください。

```sh
brew install --cask visual-studio-code
```

Linux では、`.deb` または `.rpm` ファイルを直接ダウンロードできます。インストール後は、適切な実行ファイルや CLI コマンド（通常は `code`）を使ってアプリケーションを起動できます。これでコーディングの準備が整いました。
