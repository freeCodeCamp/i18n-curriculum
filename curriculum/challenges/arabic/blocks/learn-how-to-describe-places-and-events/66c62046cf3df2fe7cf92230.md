# --explanation--

للعثور على الإجابة، انتبه إلى كلمة `but` التي تشير إلى التباين أو الاستثناء.

بعد `but`، تذكر سارة `they're not visible`، تليها الحالة `if the project isn't linked to Git.`

هذا يخبرك أن الأدوات مخفية ما لم يكن المشروع متصلًا بـ Git.

عبارة `if the project isn't` هي علامة رئيسية تقدم الشرط الذي بموجبه لن تظهر أدوات Git.

في جملة سارة، تستخدم `they're not` لتوضح أن أدوات Git غير مرئية، و`isn't` لتقول إن المشروع غير مرتبط بـ Git.
