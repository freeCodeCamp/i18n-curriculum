# --description--

Os artigos `a` e `an` são usados para indicar um substantivo singular e não específico.A escolha entre eles depende do som que segue.Se a próxima palavra começar com um som consonantal, você usa `a`.Se começar com um som de vogal, você usa `an`.Por exemplo:

- `Quincy is a teacher at freeCodeCamp.` - A palavra que segue o artigo é `teacher`. Como começa com o som de uma consoante, o artigo `a` é a escolha apropriada.

- `Jake is an information security analyst.` - A palavra que segue o artigo é `information`. Como começa com o som de uma vogal, o artigo `an` é a escolha apropriada.
