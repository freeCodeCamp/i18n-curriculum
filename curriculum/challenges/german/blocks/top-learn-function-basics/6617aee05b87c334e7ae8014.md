# --description--

Eine Variable, die innerhalb einer Funktion deklariert wird, ist nur innerhalb dieser Funktion sichtbar. Dies nennt man den Gültigkeitsbereich der Variable, dieser spezielle Gültigkeitsbereich heißt lokaler Gültigkeitsbereich. Variablen, die außerhalb einer Funktion deklariert werden, nennt man globale Variablen, und sie sind im gesamten Programm sichtbar. Zum Beispiel führt der folgende Code zu einem Fehler:

```js
function myFunction() {
  let x = 10;
}

console.log(x);
```

Die Variable `x` wird innerhalb der Funktion `myFunction` deklariert und ist außerhalb der Funktion nicht sichtbar. Der Code löst einen `ReferenceError` aus, weil `x` nicht definiert ist.

Wenn Sie eine Variable außerhalb einer Funktion deklarieren, nennt man sie globale Variable, und sie ist im gesamten Programm sichtbar. Zum Beispiel funktioniert der folgende Code:

```js
let x = 10;

function myFunction() {
  console.log(x);
}

myFunction();
```

Wenn innerhalb der Funktion eine Variable mit demselben Namen deklariert wird, überschattet sie die äußere Variable.
