# --description--

특정한 것에 대해 묻고 싶을 때는 질문 단어 `what`를 사용할 수 있습니다. `What`는 사물에 대해 이야기할 때 흔히 사용됩니다.

질문에서는 보통 `what`를 문장 맨 앞에 둡니다. 그다음에 `to be` 동사(예: `is` 또는 `are`)나 다른 동사를 쓰고, 그다음에 주어(예: `you` 또는 `the team`)를 둡니다. 예를 들어:

- `What is your favorite color?` – 누군가가 당신이 가장 좋아하는 색깔을 알고 싶어 합니다.

- `What does this code do?` – 누군가가 코드의 역할이나 결과를 알고 싶어 합니다.
