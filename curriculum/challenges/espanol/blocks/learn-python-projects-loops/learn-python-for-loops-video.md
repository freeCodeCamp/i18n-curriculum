## --text--

¿Cuál de las siguientes es la forma correcta de iterar sobre cada letra en la cadena `"Giraffe Academy"`?

## --answers--

```python
for (letter) in "Giraffe Academy":
    print(letter)
```

---

```python
for key, letter in "Giraffe Academy":
    print(letter)
```

---

```python
for letter in "Giraffe Academy":
    print(letter)
```

---

```python
for letter, value in "Giraffe Academy":
    print(letter)
```
