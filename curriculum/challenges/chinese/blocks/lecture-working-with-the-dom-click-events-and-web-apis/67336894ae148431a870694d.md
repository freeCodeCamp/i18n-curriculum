# --interactive--

让我们了解一下 DOM 以及它为何对网页开发如此重要。DOM 代表文档对象模型。它是一个编程接口，允许我们与超文本标记语言文档进行交互。

通过 DOM，你可以添加、修改或删除网页上的元素。你甚至可以通过让元素监听并响应事件，使你的网站具有交互性。

在 DOM 中，HTML 文档表现为一个节点树。每个节点代表 HTML 文档中的一个 HTML 元素：

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>DOM Example</title>
  </head>
  <body>
    <h1>What is the DOM?</h1>
    <h2>Let's learn about the DOM</h2>
  </body>
</html>
```

这是一个表现我们示例基本 DOM 结构的图示：

```md
Document
========

HTML
-----

- Head
  - Title
- Body
  - H1
  - H2   
```

当然，这可以根据网页的 `HTML` 标记结构变得更加详细和复杂。

HTML 文档是 DOM 层次结构中的根节点。它有一个子节点，即 `html` 元素。由于所有其他节点都从它派生，因此它是 HTML 文档的根元素。

`html` 元素有两个子元素：`head` 和 `body`。

`head` 元素包含关于文档的元数据。它提供有关网页的基本信息。

`body` 元素包含网页的可见内容。

你可以使用 JavaScript 访问这些元素，在你的代码中操作它们，甚至动态修改它们。这就是 DOM 和网页 API 的强大之处。

要在 JavaScript 中访问这些元素，你可以使用 `getElementById()` 和 `querySelector()` 方法。这些方法是 Web API，因为它们提供了使用 JavaScript 访问 DOM 的标准化方式。

使用 `getElementById()`，你可以获取一个表现具有指定 `id` 的超文本标记语言元素的对象。请记住，`id` 在每个超文本标记语言文档中必须是唯一的，因此此方法只会返回一个 `Element` 对象。这里有一个示例：

```js
const container = document.getElementById("container");
```

这行 JavaScript 代码获取 `id` 值为 `container` 的元素，并将该对象赋值给一个 JavaScript 常量。你必须将 `id` 作为参数放在引号内传递。如果你将该对象记录到控制台，你将在输出中看到它：

:::interactive_editor

```html
<div id="container">
  <h1>Hello, World!</h1>
  <p>Welcome to learning about the DOM.</p>
</div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
console.log(container); 
```

:::

`querySelector()` 比 `getElementById()` 更广泛。使用 `querySelector()`，你可以获取 HTML 文档中第一个匹配作为参数传入的 CSS 选择器的元素。 

在此示例中，你将获得匹配 `section` 选择器的元素：

:::interactive_editor

```html
<section>
  <h2>Section Title</h2>
  <p>This is a section of the webpage.</p>
</section>
<script src="index.js"></script>  
```

```js
const sectionEl = document.querySelector("section");
console.log(sectionEl);
```

:::

如果你想使用 `querySelector()` 通过类名选择一个元素，你需要在类名前加上点号（`.`）。下面是一个示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="highlight">
  <p>This content is highlighted.</p>
</div>
<script src="index.js"></script>
```

```css
body {
  background-color: lightgray;
}

.highlight {
  background-color: yellow;
}
```

```js
const highlightEl = document.querySelector(".highlight");
console.log(highlightEl);
```

:::

你还有其他匹配多个元素的方法，比如 `getElementsByClassName()` 和 `querySelectorAll()`。你将在后续课程中学习更多关于它们的内容。

通过理解如何使用 web APIs 高效地操作 DOM，你可以创建强大且交互性强的网页应用。
