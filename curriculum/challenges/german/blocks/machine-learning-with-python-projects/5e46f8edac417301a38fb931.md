# --instructions--

In dieser Aufgabe müssen Sie ein Machine-Learning-Modell erstellen, das SMS-Nachrichten entweder als „ham“ oder „spam“ klassifiziert. Eine „ham“-Nachricht ist eine normale Nachricht, die von einem Freund gesendet wird. Eine „spam“-Nachricht ist eine Werbung oder eine Nachricht, die von einem Unternehmen gesendet wird.

Sie sollen eine Funktion namens `predict_message` erstellen, die einen Nachrichten-String als Argument entgegennimmt und eine Liste zurückgibt. Das erste Element der Liste soll eine Zahl zwischen null und eins sein, die die Wahrscheinlichkeit für „ham“ (0) oder „spam“ (1) angibt. Das zweite Element der Liste soll das Wort „ham“ oder „spam“ sein, je nachdem, was am wahrscheinlichsten ist.

Für diese Aufgabe verwenden Sie den <a href="http://www.dt.fee.unicamp.br/~tiago/smsspamcollection/" target="_blank" rel="noopener noreferrer nofollow">SMS Spam Collection</a>-Datensatz. Der Datensatz wurde bereits in Trainingsdaten und Testdaten aufgeteilt.

Die ersten beiden Zellen importieren die Bibliotheken und Daten. Die letzte Zelle testet Ihr Modell und Ihre Funktion. Fügen Sie Ihren Code zwischen diesen Zellen ein.
