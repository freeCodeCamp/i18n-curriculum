## --text--

Яке з наведених тверджень є правильним щодо відображення/видалення фігурних дужок для блоків коду, повʼязаних з інструкцією `if`?

## --answers--

Фігурні дужки не можна видаляти з блоку коду для інструкцій `else if` та `else`.

---

Якщо фігурні дужки видалити з блоків коду інструкції `if-elseif-else`, також потрібно видалити порожній простір.
