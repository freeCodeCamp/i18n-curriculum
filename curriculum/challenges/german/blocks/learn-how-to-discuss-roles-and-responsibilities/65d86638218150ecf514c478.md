# --description--

Der Genitiv (`'s`) wird verwendet, um Eigentümerschaft oder eine Beziehung zwischen zwei Dingen anzuzeigen.

Bei Singularnomen fügen Sie `'s` an das Ende des Nomens an, um Besitz anzuzeigen. Zum Beispiel bedeutet `the cat's toy` das Spielzeug, das der Katze gehört.

Bei Pluralnomen, die bereits auf `-s` enden, fügen Sie einfach ein `'` am Ende hinzu. Zum Beispiel bedeutet `the teachers' lounge` den Aufenthaltsraum für Lehrkräfte.

Ein `facility` ist ein Ort, an dem eine bestimmte Tätigkeit stattfindet, wie eine Fabrik, ein Büro oder eine Schule.

# --fillInTheBlank--
