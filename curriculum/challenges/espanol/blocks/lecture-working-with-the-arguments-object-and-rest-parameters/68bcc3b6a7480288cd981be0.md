# --interactive--

Como recuerdas de lecciones y talleres anteriores, puedes crear funciones con un número de parámetros y llamar a esa función con argumentos.

Aquí tienes un recordatorio:

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call
getSum(3, 4); // 7
```

¿Pero qué pasa si tienes una función que se llama con más argumentos de los que fue definida para aceptar?

:::interactive_editor

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call with extra argument
console.log(getSum(3, 4, 5)); // 7
```

:::

JavaScript no generará un error en este caso. En su lugar, ignorará el argumento extra y simplemente sumará los números `3` y `4`. Las funciones que aceptan un número variable de argumentos se conocen como <dfn>funciones variádicas</dfn>.

Si estás trabajando con funciones variádicas, entonces puedes utilizar el objeto `arguments`. Este objeto similar a un arreglo contiene los valores de los argumentos pasados a la función.

Aquí hay un ejemplo:

:::interactive_editor

```js
function logArgs() {
  for (const arg of arguments) {
    console.log(arg);
  }
}

logArgs(1, 2, 3);
// result:
// 1
// 2
// 3

logArgs("example"); // "example"
```

:::

Dado que el objeto `arguments` es similar a un arreglo, puedes acceder a un argumento en un índice específico así:

:::interactive_editor

```js
function getArg() {
  return arguments[1];
}

console.log(getArg(2, 4, 6)); // 4
```

:::

También puedes usar la propiedad `length` así para obtener el número de argumentos con los que se llamó a la función:

:::interactive_editor

```js
function getArgs() {
  return arguments.length;
}

console.log(getArgs("Example")); // 1
console.log(getArgs("Another", "Example")); // 2
```

:::

Aunque el objeto `arguments` parece comportarse como un arreglo real, no tiene métodos integrados de `Array` como `includes` o `push`. Para tener acceso a esos métodos, primero necesitarías convertir el objeto `arguments` en un arreglo real usando algo como `slice`, `Array.from()` o el operador spread:

:::interactive_editor

```js
function hasCat() {
  return [...arguments].includes("cat");
}

console.log(hasCat("dog", "chicken", "cat")); // true
console.log(hasCat("dog", "chicken", "horse")); // false
```

:::

Aunque es posible trabajar con el objeto `arguments` para funciones variádicas, las aplicaciones modernas de JavaScript normalmente usarán la sintaxis de parámetros rest. Aprenderás más sobre eso en una lección futura.
