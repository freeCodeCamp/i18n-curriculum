# --interactive--

上标元素用于将一段文字显示为上标。 上标是印在正常文本行上方的符号或字母。

下面是使用上标元素来说明指数的示例（要查看预览，请启用交互式编辑器）：

:::interactive_editor

```html
<p>2<sup>2</sup> (2 squared) is 4.</p>
```

:::

在此示例中，数字 2 被包裹在 `sup` 标签中，以表现段落内的上标。在预览窗口中，你将看到第二个数字 2 比第一个数字 2 更小且稍微偏高。

上标元素的常见用例包括指数、上标字母和序数词。以下是使用上标元素表示上标字母的示例（要查看预览，请启用交互式编辑器）： 

:::interactive_editor

```html
<p>
  Monseigneur is often written as <strong>M<sup>gr</sup></strong>.
</p>
```

:::

上位字母是指用上标书写的字母，通常用来表示缩写。 本例中，字母 `g` 和 `r` 被包裹在上标标记内，表示缩写。

重要的是要注意，上标元素应仅用于排版原因。如果你想对一段文本应用上升基线样式，则应使用 CSS 而不是上标元素。

要在 HTML 中表示化学式，可以使用下标元素。 下标会显示为比正常文本低的基线，并且字体较小。

这是使用下标元素来显示二氧化碳化学表现的示例（要查看预览，请启用交互式编辑器）：

:::interactive_editor

```html
<p>CO<sub>2</sub></p>
```

:::

数字 2 包裹在 `sub` 标记内，表示该字符为下标。

下标元素的常见用例包括化学式、脚注和变量下标。
