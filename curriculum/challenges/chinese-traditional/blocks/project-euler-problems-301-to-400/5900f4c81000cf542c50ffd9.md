# --description--

小於或等於 100 且僅能被質數 2 和 3 整除的最大整數是 96，因為 $96 = 32 \times 3 = 2^5 \times 3$。

對於兩個不同的質數 `p` 和 `q`，令 `M(p, q, N)` 為小於等於 `N` 且僅可被 `p` 和 `q` 同時整除的最大正整數，若不存在此類正整數，則 `M(p, q, N)=0`。

例如 $M(2, 3, 100) = 96$。

$M(3, 5, 100) = 75$ 而不是 90，因為 90 可被 2、3 和 5 整除。且 $M(2, 73, 100) = 0$，因為不存在小於等於 100 且同時可被 2 和 73 整除的正整數。

令 $S(N)$ 為所有不同的 $M(p, q, N)$ 的總和。$S(100)=2262$。

尋找 $S(10\,000\,000)$。
