# --explanation--

`Code reuse` يشير إلى ممارسة استخدام شفرة موجودة في أجزاء جديدة من البرنامج. على سبيل المثال:

`The "Login" class was used in multiple applications to handle user access, which is a good example of code reuse.` - بدلاً من كتابة منطق تسجيل الدخول لكل تطبيق، تُعاد استخدام نفس الشفرة، مما يحسن الكفاءة.

`Extending` يعني إضافة ميزات جديدة إلى شيء موجود بالفعل. على سبيل المثال:

`The "AdminUser" class extends the basic "User" class by adding an "approveUser()" method.` - يحتفظ `AdminUser` بكل شيء من `User` ويبني عليه بقدرات جديدة.
