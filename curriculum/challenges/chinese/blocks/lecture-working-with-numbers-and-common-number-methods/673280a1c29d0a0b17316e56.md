# --interactive--

`.toFixed()`方法是一个内置的 JavaScript 函数，用定点表示法格式化数字。 当你需要控制数字的小数位数时，尤其是在显示货币值或处理精确测量时，它尤其有用。 

`.toFixed()` 方法是在一个数字上调用的，它包含一个可选参数，即小数点后的位数。 它返回数字的字符串形式，并带有指定的小数位数。 下面是一个关于 `.toFixed()` 如何工作的基本示例：

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed(2)); // "3.14"
```

:::

在本例中，我们将小数位数限制为两位。 所以，`3.14159` 变成了 `3.14`。 需要注意的是，`.toFixed()` 返回的是字符串，而不是数字。 这是因为该方法主要用于格式化数字以便显示，而不是进一步计算。

`.toFixed()` 方法会将数字四舍五入到能用指定的小数位数表示的最接近的数值。 理解这种四舍五入行为非常重要：

:::interactive_editor

```js
console.log((3.14159).toFixed(3));  // "3.142"
console.log((3.14449).toFixed(3));  // "3.144"
console.log((3.14550).toFixed(3));  // "3.146"
```

:::

如你所见，`.toFixed()` 在下一位数字大于等于 `5` 时向上舍入，否则向下舍入。 如果调用 `.toFixed()` 而不带参数，默认为 `0` 位小数：

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed()); // "3"
```

:::

在进行财务计算或显示价格时，`.toFixed()` 方法尤其有用：

:::interactive_editor

```js
let price = 19.99;
let taxRate = 0.08;
let total = price + (price * taxRate);

console.log("Total: $" + total.toFixed(2)); // "Total: $21.59"
```

:::

在此示例中，`.toFixed(2)` 可确保显示的总数始终保留两位小数，这是许多国家的货币标准。

总之，`.toFixed()` 方法是 JavaScript 中格式化数字的强大工具，尤其是需要控制小数位的显示时。 虽然它主要用于格式化输出，但请记住它的行为，尤其是需要精确计算时。
