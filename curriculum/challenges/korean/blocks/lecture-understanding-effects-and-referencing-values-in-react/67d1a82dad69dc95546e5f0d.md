# --description--

React에서는 DOM 요소에 직접 접근해야 하는 상황이 있을 수 있습니다. 이럴 때 "refs"가 유용합니다. refs는 변경 가능한 값을 저장할 수도 있지만, 그런 경우에는 state가 더 적합합니다.

순수 JavaScript에서는 `getElementById()`와 `querySelector()` 메서드를 사용해 DOM 요소에 접근했습니다. 하지만 React에서는 refs를 사용해 DOM 요소에 접근합니다.

주요 차이점 중 하나는 refs를 사용할 때 ID나 클래스 같은 식별자가 필요 없다는 점입니다.

그렇다면 refs를 어떻게 만들고 사용할 수 있을까요? React는 바로 그 기능을 제공하는 `useRef()` 훅을 제공합니다.

첫 번째 단계는 React에서 훅을 가져오는 것입니다:

```js
import { useRef } from "react";
```

다음으로, `useRef` 훅 안에 초기값을 가진 ref를 보관할 변수를 만들어야 합니다. 예를 들어 `sectionRef`를 `null`로 초기화합니다:

```js
const sectionRef = useRef(null);
```

마지막으로, JSX 요소에 `ref` 속성을 사용해 ref 변수를 연결하세요:

```jsx
<section ref={sectionRef}>
  {/* Section content */}
</section>
```

ref를 콘솔에 출력하면, 현재 값이 들어 있는 객체임을 알 수 있습니다. 이 경우 `null`입니다:

```js
console.log(sectionRef); // { current: null }
```

또한 `current` 속성을 사용해 현재 값을 직접 콘솔에 출력할 수도 있습니다:

```js
console.log(sectionRef.current); // null
```

ref의 이후 값은 컴포넌트 생명주기에 따라 달라집니다.

예를 들어, `sectionRef`의 초기값은 항상 `null`입니다. 이는 초기화한 값이기 때문입니다. 컴포넌트가 마운트된 후에는 ref가 연결된 `section` 요소가 값이 됩니다.

컴포넌트가 언마운트되면 ref의 값은 다시 초기값인 `null`로 돌아갑니다.

ref를 보여주는 대표적인 예는 렌더링 시 또는 버튼 클릭으로 input 요소에 포커스를 주는 것입니다.

버튼을 클릭했을 때 이렇게 할 수 있습니다:

```jsx
import { useRef } from "react";

const Focus = () => {
  const inputRef = useRef(null);

  const handleFocus = () => {
    if (inputRef.current) {
      inputRef.current.focus();
    }
  };

  return (
    <div>
      <input ref={inputRef} type="text" placeholder="Enter text" />
      <button onClick={handleFocus}>Focus Input</button>
    </div>
  );
};

export default Focus;
```

위 코드에서 `inputRef`가 생성되어 `input` 요소에 연결되어 있습니다. 또한 `onClick` 이벤트가 있는 버튼이 있고, `handleFocus` 함수를 호출합니다.

`handleFocus` 함수는 `focus()` 요소에서 `input` 메서드를 호출하는 역할만 합니다. `input`는 React에 내장된 컴포넌트이므로 실제 `input` DOM 요소는 ref의 `current` 속성에 설정됩니다. 따라서 `focus()`와 함께 `input.current.focus()` 메서드를 호출합니다.

refs를 사용할 때 알아야 할 모범 사례는 다음과 같습니다:

- refs는 주로 DOM과 상호작용할 때 사용하세요. 변경 가능한 데이터를 위해서도 사용할 수 있지만, state가 더 적합합니다.

- 기본적인 상태 관리를 위해 refs를 사용하지 마세요. 그 용도는 `useState`입니다.

- `ref.current`가 존재하는지 확인한 후에 속성에 접근하세요. 다시 한 번 방법은 다음과 같습니다:

```js
const handleFocus = () => {
  if (inputRef.current) {
    inputRef.current.focus();
  }
};
```

이렇게 하면 ref가 DOM에 연결되기 전이나 제거된 후에 접근할 때 발생할 수 있는 오류를 방지할 수 있습니다.
