# --description--

整数の$n$-タプル$t = (a_1, \ldots, a_n)$に対して、多項式方程式 $x^n + a_1x^{n - 1} + a_2x^{n - 2} + \ldots + a_{n - 1}x + a_n = 0$ の解を$(x_1, \ldots, x_n)$とします。

次の2つの条件を考えます：

- $x_1, \ldots, x_n$ はすべて実数である。
- $x_1, ..., x_n$ を昇順に並べたとき、$1 ≤ i ≤ n$ に対して $⌊x_i⌋ = i$ となる。($⌊·⌋:$ 床関数)

$n = 4$ の場合、両方の条件を満たす整数の$n$-タプルは12個あります。

$t$の整数の絶対値の和を$S(t)$と定義します。

$n = 4$の場合、両方の条件を満たすすべての$n$-タプル$t$について、$\sum S(t) = 2087$であることが確認できます。

$n = 7$の場合の$\sum S(t)$を求めてください。
