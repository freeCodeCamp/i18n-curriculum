# --interactive--

Цикли в програмуванні використовують для повторення блоку коду кілька разів.

Прикладом циклу може бути ситуація, коли ви створюєте програму, яка має вивести список елементів. Ви можете використати цикл, щоб вивести кожен елемент зі списку.

Іншим прикладом є створення гри, де потрібно переміщувати персонажа по екрану. Ви можете використати цикл, щоб переміщувати персонажа на певну кількість пікселів щоразу, коли цикл виконується.

У JavaScript існує кілька типів циклів, які ви можете використовувати. У цьому уроці ми розглянемо цикл `for`. Ось базовий синтаксис циклу `for`:

```js
for (initialization; condition; increment or decrement) {
  // code block to be executed
}
```

Ініціалізація виконується перед початком циклу. Зазвичай її використовують для ініціалізації змінної-лічильника. Змінна-лічильник — це змінна, яка відстежує, скільки разів цикл уже виконався.

Умова перевіряється перед кожною ітерацією циклу. Ітерація — це один прохід через цикл.

Якщо умова істинна, виконується блок коду всередині циклу. Якщо умова хибна, цикл зупиняється, і ви переходите до наступного блоку коду.

Остання частина циклу — це інструкція збільшення/зменшення. Вона виконується після кожної ітерації циклу. Зазвичай її використовують для збільшення або зменшення змінної-лічильника.

:::interactive_editor

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

:::

У першій частині прикладу вище ми ініціалізуємо змінну-лічильник `i` значенням `0`. Згідно з загальною конвенцією, у циклі `i` змінна-лічильник часто позначається як `for`.

Наступний крок — перевірка умови. У цьому випадку умова перевіряє, чи менше `i` за `5`. Оскільки `i` дорівнює `0`, умова істинна, і виконується блок коду всередині циклу.

Блок коду всередині циклу виводить значення `i` у консоль. Значення `i` дорівнює `0`, тому в консолі буде показано значення `0`.

Потім виконується інструкція збільшення. У цьому випадку ми збільшуємо `i` на `1`. Тож `i` тепер дорівнює `1`.

Потім знову перевіряємо умову — чи менше `i` за `5`. Оскільки `i` тепер дорівнює `1`, умова все ще істинна, і блок коду всередині циклу виконується знову.

Ми повторюємо цей процес, доки умова не стане хибною. У цьому випадку, коли `i` дорівнює `5`, умова хибна, і цикл зупиняється.

Працюючи з циклами, будьте обережні, щоб не створити умову, яка завжди істинна. Якщо це станеться, цикл буде виконуватися вічно, і ваша програма зависне. Це називається нескінченним циклом.

Можна створювати вкладені цикли `for`. Вкладений цикл — це коли один цикл розміщений всередині іншого. Ми побачимо приклади, коли це може знадобитися, пізніше.

Цикли корисні в програмуванні, коли потрібно повторити блок коду певну кількість разів. Хоча спочатку робота з циклами `for` може бути складною, з практикою ви швидко освоїтеся.
