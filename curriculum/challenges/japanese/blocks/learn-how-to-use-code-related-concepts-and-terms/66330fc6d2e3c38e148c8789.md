# --description--

`this`、`that`、`these`、`those` のような指示代名詞についてはすでに学びました。これらは時間や空間の近さを考慮して、特定のものや人を指すときに使います。

所有格は、アポストロフィーと `s`（例えば `function's`）で示されることが多く、人や物の所有や所属を表します。何かが他のものに属している、または一部であることを示します。

`purpose` は、何かが行われる理由や使われる理由、または存在する理由を指します。例えば `The purpose of a clock is to tell time.`
