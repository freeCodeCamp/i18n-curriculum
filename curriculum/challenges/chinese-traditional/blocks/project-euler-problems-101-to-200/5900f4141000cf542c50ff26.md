# --description--

對於兩個正整數 $a$ 和 $b$，Ulam 數列 `U(a,b)` 定義為 ${U{(a,b)}\_1} = a$、${U{(a,b)}\_2} = b`，且對於 $k > 2$，${U{(a,b)}\_k}$ 是大於 ${U{(a,b)}\_{(k-1)}}$ 的最小整數，該整數可以且只能以唯一一種方式表示為 `U(a,b)` 之前兩個不同成員的和。

例如，序列 $U(1,2)$ 以開始

$$1、2、3 = 1 + 2、4 = 1 + 3、6 = 2 + 4、8 = 2 + 6、11 = 3 + 8$$

5 不屬於它，因為 $5 = 1 + 4 = 2 + 3$ 有兩種作為兩個先前成員和的表述，類似地 $7 = 1 + 6 = 3 + 4$。

求 $\sum {U(2, 2n + 1)_k}$，其中 $2 ≤ n ≤ 10$，且 $k = {10}^{11}$。
