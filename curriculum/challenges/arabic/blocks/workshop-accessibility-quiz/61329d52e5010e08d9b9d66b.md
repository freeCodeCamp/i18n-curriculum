# --description--

عنصر `meta` مهم آخر لـ إمكانية الوصول وتحسين محركات البحث هو تعريف `description`. تُستخدم قيمة الخاصية `content` من قبل محركات البحث لتوفير وصف لصفحتك.

أضف عنصر `meta` مع تعيين الخاصية `name` إلى `description`، وأعطه خاصية `content` مفيدة.

# --hints--

```js
assert.equal(document.querySelectorAll('meta').length, 3);
```

يجب إعطاء عنصر `meta` خاصية `name` بقيمة `description`.

```js

```

يجب إعطاء عنصر `meta` خاصية `content`.
