# --description--

Є $N$ місць у ряду. $N$ людей приходять один за одним, щоб зайняти місця за такими правилами:

1. Якщо є місце, сусідні з яким місця не зайняті, займіть таке місце.
2. Якщо такого місця немає, але є місце, сусіднє лише з одним зайнятим місцем, займіть таке місце.
3. Інакше займіть одне з решти доступних місць.

Нехай $T(N)$ — кількість варіантів, у яких $N$ місць зайняті $N$ людьми за цими правилами. На наступному малюнку показано, що $T(4) = 8$.

<img alt="eight ways for N seats to be occupied by N people" src="https://cdn.freecodecamp.org/curriculum/project-euler/comfortable-distance.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Ми можемо перевірити, що $T(10) = 61\\,632$ і $T(1\\,000)\bmod 100\\,000\\,007 = 47\\,255\\,094$.

Знайдіть $T(1\\,000\\,000)\bmod 100\\,000\\,007$.
