# --explanation--

Reservar significa guardar ou economizar algo, especialmente dinheiro ou tempo, para um propósito futuro específico.Por exemplo:

É inteligente reservar dinheiro para despesas imprevistas. — É bom guardar um dinheiro extra para surpresas.

Uma `contingency` é um plano de backup ou emergência, especialmente para situações que podem não acontecer mas poderiam.Por exemplo:

`It's important to have contingency measures during testing.` — Você precisa de etapas ou planos extras caso as coisas não saiam como esperado durante os testes.

Alice está dizendo que eles reservaram 15% do orçamento para casos de situações não planejadas.
