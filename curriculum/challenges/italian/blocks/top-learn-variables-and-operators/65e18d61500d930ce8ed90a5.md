# --description--

Puoi pensare alle variabili come a dei "contenitori" per i dati nel tuo codice.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/variables-and-operators/top-learn-variables-and-operators.png" alt='il concetto di variabili nella programmazione come riquadri etichettati "birthday" che archiviano e permettono di consultare dati, nello specifico una data "16-09-2003"'>

Una variabile è un "archivio nominato" per i dati. Puoi usare le variabili per archiviare oggetti, visitatori e altri dati.

Per creare una variabile in JavaScript, usa la parola chiave `let`.

L’istruzione qui sotto crea (in altre parole: dichiara) una variabile con il nome "message":

```js
let message;
```

Ora puoi inserire dei dati al suo interno usando l’operatore di assegnazione `=`:

```js
let message;

// store the string 'Hello' in the variable named message
message = 'Hello'; 
```

Per essere più concisi, puoi combinare la dichiarazione della variabile e l’assegnazione in un’unica riga:

```js
let message = 'Hello';
```
