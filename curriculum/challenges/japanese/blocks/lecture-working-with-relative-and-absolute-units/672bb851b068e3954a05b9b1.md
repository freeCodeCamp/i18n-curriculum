# --interactive--

`calc()` 関数を使うと、スタイルシート内で直接計算を行い、プロパティの値を動的に決定できます。これにより、ビューポートのサイズや他の要素に基づいて寸法を計算し、柔軟でレスポンシブなユーザーインターフェースを作成できます。

`calc()` は CSS の関数です。JavaScript の学習を始めると関数についてもっと詳しく学びますが、このレッスンでは `calc()` の仕組みを理解するために必要な基本を学びます。

関数とは、特定のタスクを実行するコードのブロックです。CSS にはすでに定義されている関数があり、それらを直接使って、必要な値を渡すことでタスクの実行方法をカスタマイズできます。

プログラミングの世界では、関数が実行するタスクを動かすことを「関数を呼び出す」と言います。関数に渡す値は引数と呼ばれます。

下のコードのように、関数を呼び出すには、関数名の後に丸括弧で囲んだ引数をカンマで区切って書きます。関数名と開き丸括弧の間にスペースは入れません。

```css
function(argument1, argument2, argument3)
```

関数は、何をするか知るために1つの値だけが必要な場合があります。その場合、引数は1つだけです。`calc()` 関数がそうです。計算する内容を知るために1つの引数を取ります。

この引数には式と呼ばれるものを渡します。式とは、値と演算子の組み合わせで結果を生み出すものです。

`calc()` 関数の呼び出し方はこうです。calc という名前を書き、丸括弧で囲み、その中に式を書きます。

```css
calc(expression)
```

式は評価されて最終結果が計算されます。「評価される」とは、値と演算子が裏で単一の値に変換されることを意味します。結果は計算が行われている CSS プロパティに代入されます。

長さ、角度、時間、パーセンテージ、数値、色を表す値で計算ができます。ピクセル、パーセンテージ、em など異なる単位を組み合わせることも可能です。

数値の場合、式内のすべての値（オペランドとも呼ばれます）は対応する単位を持つ必要があります。例えば `px`、`em`、パーセンテージ（`%`）などです。演算子によっては異なる単位のオペランドが混在する場合があります。

式の中では、加算（`+`）、減算（`-`）、乗算（`*`）、除算（`/`）の演算子を使えます。

複数のオペランドと演算子がある場合、`calc()` は標準の演算子優先順位ルールに従います。必要に応じて丸括弧を使って演算の順序を指定することもできます。

下の例では、`div` に `Hello, World!` というテキストがあります。

`div` を選択するために CSS の型セレクターを使い、白い文字と濃い青の背景でスタイルを設定しています。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>Hello, World!</div>
```

```css
div {
  color: white;
  background-color: #1b1b32;
  width: calc(50% - 20px);
}
```

:::

ここで新しいのは、幅が動的に計算されていることです。`calc()` 関数を呼び出し、式を引数として渡しているのに注目してください。式には異なる単位の2つのオペランドと1つの演算子（減算演算子）があります。

パーセンテージは相対単位です。値（`50%`）は親コンテナの幅によって決まります。そこから `20px` が引かれます。この式の結果が `div` の幅を決定します。

`div` の幅は親コンテナの幅の約半分で、親コンテナのサイズを変更すると幅が自動的に再計算されます。

これが `calc()` を使う最大の利点です。アプリケーションやビューポートのさまざまな要素に基づいて CSS プロパティの値を動的に決定できます。

式には計算に使いたい場合、CSS 関数や変数も含められます。CSS 変数については次のレッスンで詳しく学びます。

では、`calc()` 関数の基本がわかったところで、いくつかのベストプラクティスを紹介します。

まず、加算（`+`）と減算（`-`）の演算子の前後には空白を入れる必要があります。

例えば、下の式は減算演算子の直前にスペースがないため有効な式ではありません。

```css
calc(100% -30px)
```

減算（`-`）演算子の前後には空白を入れてください。空白を入れることで有効な式になります。

```css
calc(100% - 30px)
```

乗算と除算の演算子では必須ではありませんが、空白を入れることを強くお勧めします。

計算結果を他の計算に使いたい場合は、`calc()` 関数の呼び出しをネストすることもできます。

また、`calc()` 関数に渡す式で長さを表す値としてゼロを使う場合は、単位を必ず付けてください。例えば、次の式は有効ではありません。

```css
calc(100% - 0)
```

単位（px など）を付ける必要があります。

```css
calc(100% - 0px)
```

さらに、現在の仕様では乗算や除算の演算子を使う場合、オペランドのうち1つは単位なしでなければなりません。特に除算演算子では右側のオペランドが単位なしである必要があります。次の式は両方のオペランドに単位（ピクセル）があるため有効ではありません。5 または 50 のどちらかは単位なしでなければなりません。

```css
calc(5px * 50px)
```

どちらかの単位を省略する必要があります。次のどちらの例も有効です。

```css
calc(5 * 50px)
calc(5px * 50)
```

こちらは除算演算子の例です。両方に単位があるため有効ではありません。

```css
calc(50% / 5%)
```

除算演算子の場合は右側のオペランドから単位を外してください。

```css
calc(50% / 5)
```

`calc()` 関数はウェブ開発者にとって非常に役立ちます。この関数を使うと、プロパティの値を動的に設定して柔軟でレスポンシブなデザインを作成できます。
