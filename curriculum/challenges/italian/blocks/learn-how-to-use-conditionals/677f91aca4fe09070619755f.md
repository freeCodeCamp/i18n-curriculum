# --explanation--

`Power cable` indica il filo che fornisce elettricità al computer:

`Power` significa energia o elettricità, e `cable` è il filo o cavo usato per trasferirla. Per esempio:

- `The server shut down because the power cable was disconnected.` - Il server si è spento quando il cavo di alimentazione è stato scollegato.

- `Check if the power cable is properly connected before troubleshooting.` - Assicurati che il cavo di alimentazione sia collegato prima di risolvere altri problemi.
