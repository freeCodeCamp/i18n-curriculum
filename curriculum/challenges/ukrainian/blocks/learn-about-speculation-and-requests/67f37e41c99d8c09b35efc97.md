# --explanation--

Джейк каже, `to see if there's any unusual activity or failed login attempts`. Це означає, що він хоче шукати ознаки підозрілої поведінки в системі.

- `Unusual activity` може означати дивні або несподівані дії, наприклад, доступ до файлів у незвичний час.

- `Failed login attempts` — це ознаки того, що хтось міг спробувати отримати доступ до системи без правильного пароля.

Журнали аудиту допомагають відстежувати такі речі і корисні для розслідування потенційних проблем із безпекою.
