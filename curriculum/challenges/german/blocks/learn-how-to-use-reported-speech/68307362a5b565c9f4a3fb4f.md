# --explanation--

`It was mentioned...` ist auch eine Struktur im **Passiv**, bei der der Fokus auf der geteilten Idee liegt, nicht darauf, wer sie gesagt hat. Zum Beispiel:

`It was mentioned that the update improved security.` – Dies vermittelt die Idee, ohne den Sprecher zu nennen.

Die Struktur `it's` + Komparativ + `to` + Verb wird verwendet, um zu zeigen, wie etwas im Vergleich zu etwas anderem gemacht wird. Zum Beispiel:

`It's faster to search with the new tool.` – Das bedeutet, dass das neue Tool beim Suchen im Vergleich zum alten Tool Zeit spart.
