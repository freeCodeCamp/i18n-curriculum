# --description--

단순 사각형은 네 개의 서로 다른 꼭짓점을 가지며, 직각이 없고 자기 교차하지 않는 다각형입니다.

$Q(m, n)$을 꼭짓점이 좌표 ($x$, $y$)인 격자점이고 $0 ≤ x ≤ m$ 및 $0 ≤ y ≤ n$를 만족하는 단순 사각형의 개수라고 합시다.

예를 들어, 아래에서 볼 수 있듯이 $Q(2, 2) = 94$입니다:

또한 $Q(3, 7) = 39\\,590$, $Q(12, 3) = 309\\,000$, 그리고 $Q(123, 45) = 70\\,542\\,215\\,894\\,646$임을 확인할 수 있습니다.

$Q(12\\,345, 6\\,789)\bmod 135\\,707\\,531$ 값을 구하세요.
