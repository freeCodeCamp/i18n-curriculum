# --description--

 Quando você precisa expressar algo que é proibido ou uma forte sugestão contra fazer algo, você pode usar `mustn't` (contração de `must not`) para formar uma frase negativa.

Por exemplo, `You mustn't touch the wet paint` significa que é muito importante não tocar na tinta porque ela está molhada.

`Anyone` é usado quando você está falando sobre todas as pessoas de forma geral e geralmente em perguntas ou frases negativas:

`Don't tell anyone` significa não contar para nenhuma pessoa.

`Someone` é usado quando você está falando sobre uma pessoa não especificada em uma frase afirmativa:

`I need someone to help me` significa que você precisa de uma pessoa não especificada para ajudar.
