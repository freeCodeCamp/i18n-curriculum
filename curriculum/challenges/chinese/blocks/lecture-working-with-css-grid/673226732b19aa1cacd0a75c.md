# --interactive--

在上一课中，你了解了 CSS grid，它可以用来为你的网页创建复杂且流畅的版型。在本课中，我们将探讨如何使用 `fr` 单位创建灵活的网格版型。

让我们从这个 HTML 标记开始，它将代表我们的网格容器：

```html
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

在 CSS 中，我们将容器的 display 属性设置为 grid。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 25% 25% 25% 25%;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

`grid-template-columns` 用于设置每个列的大小。 在本例中，每个列的大小将是容器的 `25%`。 然后使用 gap 属性在每个列周围创建空间。

到目前为止，我们一直使用百分比来表示列的大小，但是我们也可以使用单位 `fr`。

单位 `fr` 是一个分数单位，它表示网格容器空间的一个等份。

下面是代码被重构为使用单位 `fr` 而不是百分比时的样子。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

每一列将占用可用空间的一个等份。 由于有四列，因此每列在网格容器中都有相同的可用空间份额。

当你开始构建你的网格布局时，你会发现自己想要更频繁地使用单位 `fr`，因为它们提供了一种弹性的、按比例分配空间的方式，允许你创建适应不同屏幕尺寸的响应式布局，而无需手动调整像素值。
