# --explanation--

`Work on`は、ある作業に時間や努力を注ぐことを意味します。後には名詞か`-ing`形（動名詞）が続きます。例えば：

- `She is working on a new project at the moment.` - 今、彼女は新しいプロジェクトのための作業をしています。

- `We need to work on improving our website.` - ウェブサイトをより良くするべきです。
