# --description--

Цілі числа та числа з плаваючою комою — це основні числові типи даних у Python. За їх допомогою ви можете зберігати числові дані та виконувати математичні операції.

Давайте розглянемо, що таке цілі числа та числа з плаваючою комою, як виконувати арифметичні обчислення з ними, а також кілька методів, які Python надає для роботи з обома типами.

Цілі числа — це цілі числа без десяткових крапок, які можуть бути додатними або від’ємними:

```python
my_int_1 = 56
my_int_2 = -4

print(type(my_int_1)) # <class 'int'>
print(type(my_int_2)) # <class 'int'>
```

Ось як виконати операцію додавання з цілими числами:

```python
my_int_1 = 56
my_int_2 = 12

sum_ints = my_int_1 + my_int_2
print('Integer Addition:', sum_ints) # Integer Addition: 68
```

Ось як виконати операцію віднімання з цілими числами:

```python
my_int_1 = 56
my_int_2 = 12

# Subtraction
diff_ints = my_int_1 - my_int_2
print('Integer Subtraction:', diff_ints) # Integer Subtraction: 44
```

Ось як виконати операцію множення з цілими числами:

```python
my_int_1 = 12
my_int_2 = 4

# Multiplication
product_ints = my_int_1 * my_int_2
print('Integer Multiplication:', product_ints) # Integer Multiplication: 48
```

А ось як виконати операцію ділення з цілими числами:

```python
my_int_1 = 56
my_int_2 = 12

# Division
div_ints = my_int_1 / my_int_2
print('Integer Division:', div_ints) # Integer Division: 4.666666666666667
```

Числа з плаваючою комою — це додатні або від’ємні числа з десятковими крапками, наприклад `3.14`, `-0.5` або `0.0`.

```python
my_float_1 = -12.0
my_float_2 = 4.9

print(type(my_float_1)) # <class 'float'>
print(type(my_float_2)) # <class 'float'>
```

Ось операція додавання з числами з плаваючою комою:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_addition = my_float_1 + my_float_2
print('Float Addition:', float_addition) # Float Addition: 17.4
```

Ось операція віднімання з числами з плаваючою комою:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_subtraction = my_float_2 - my_float_1
print('Float Subtraction:', float_subtraction) # Float Subtraction: 6.6
```

Ось операція множення з числами з плаваючою комою:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_multiplication = my_float_2 * my_float_1
print('Float Multiplication:', float_multiplication) # Float Multiplication: 64.80000000000001
```

А ось операція ділення з числами з плаваючою комою:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_division = my_float_2 / my_float_1
print('Float Division:', float_division) # Float Division: 2.222222222222222
```

Якщо додати ціле число і число з плаваючою комою, результат автоматично перетворюється на число з плаваючою комою:

```python
my_int = 56
my_float = 5.4

sum_int_and_float = my_int + my_float

print(sum_int_and_float) # 61.4
print(type(sum_int_and_float)) # <class 'float'>
```

Це також справедливо для інших базових арифметичних операцій, таких як віднімання, множення та ділення. Якщо змішувати цілі числа та числа з плаваючою комою, Python поверне число з плаваючою комою як результат.

Ви також можете виконувати більш складні арифметичні обчислення, наприклад отримувати остачу від двох чисел за допомогою оператора за модулем, цілочисельне ділення та піднесення до степеня з цілими числами та числами з плаваючою комою.

Оператор за модулем (`%`) повертає остачу від ділення значення зліва на значення справа:

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

mod_ints = my_int_1 % my_int_2
mod_floats = my_float_2 % my_float_1

print('Integer Modulo:', mod_ints) # Integer Modulo: 8
print('Float Modulo:', mod_floats) # Float Modulo: 1.1999999999999993
```

Цілочисельне ділення ділить два числа і повертає найбільше ціле число, яке не перевищує результат. Це робиться за допомогою подвійного слеша (`//`):

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

floor_div_ints = my_int_1 // my_int_2
floor_div_floats = my_float_2 // my_float_1

print('Integer Floor Division:', floor_div_ints) # Integer Floor Division: 4
print('Float Floor Division:', floor_div_floats) # Float Floor Division: 2.0
```

Піднесення до степеня підносить число до степеня іншого числа і виконується за допомогою подвійної зірочки (`**`):

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

exp_ints = my_int_1 ** my_int_2
exp_floats = my_float_1 ** my_float_2

print('Integer Exponentiation:', exp_ints) # Integer Exponentiation: 951166013805414055936
print('Float Exponentiation:',  exp_floats) # Float Exponentiation: 614787626.1765089
```

Іноді ви можете помітити, що результат операції з числами з плаваючою комою має більше десяткових знаків, ніж очікувалося. Наприклад, сума `0.1 + 0.2` дорівнює `0.30000000000000004` замість `0.3`.

Це відбувається тому, що числа зберігаються у двійковому форматі, і деякі дроби не можуть бути точно представлені у двійковій системі. В результаті вони зберігаються як скінченні наближення, так само як дріб `1/3` не може бути представлений скінченною кількістю знаків у десятковій системі і обрізається після певної кількості своїх нескінченних знаків (`0.33333...`).

Це призводить до невеликих помилок округлення.

Python також надає вбудовані функції для перетворення числових даних або рядків у цілі числа або числа з плаваючою комою.

Функція `float()` повертає число з плаваючою комою, створене з заданого числа:

```python
my_int_1 = 56
my_float_1 = float(my_int_1)

print(my_float_1)  # 56.0
print(type(my_float_1))  # <class 'float'>
```

Функція `int()` повертає ціле число, створене з заданого числа:

```python
my_float = 12.92563
my_int = int(my_float)

print(my_int)  # 12
print(type(my_int))  # <class 'int'>
```

Також ви можете використовувати ті ж вбудовані функції для перетворення рядка у число з плаваючою комою або ціле число:

```python
my_str_int = '45'
my_str_float = '7.8'

converted_int = int(my_str_int)
converted_float = float(my_str_float)

print(converted_int, type(converted_int))  # 45 <class 'int'>
print(converted_float, type(converted_float))  # 7.8 <class 'float'>
```

Ось ще кілька методів, які Python надає для роботи з цілими числами та числами з плаваючою комою.

- `round()`: округлює число до вказаної кількості десяткових знаків. За замовчуванням ця функція округлює до найближчого цілого числа і повертає ціле число без десяткових знаків:
    

```python
my_int_1 = 4.798
my_int_2 = 4.253

rounded_int_1 = round(my_int_1)
rounded_int_2 = round(my_int_2, 1)

print(rounded_int_1) # 5
print(rounded_int_2) # 4.3
```

- `abs()`: повертає абсолютне значення числа,
    

```python
num = -15

absolute_value = abs(num)
print(absolute_value) # 15
```

- `pow()`: підносить число до степеня іншого або виконує модульне піднесення до степеня.
    

```python
result_1 = pow(2, 3)  # Equivalent to 2 ** 3
print(result_1)  # 8

result_2 = pow(2, 3, 5)  # (2 ** 3) % 5
print(result_2)  # 3
```
