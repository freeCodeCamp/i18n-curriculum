# --description--

Ви можете запускати один і той самий код кілька разів, використовуючи цикл.

Найпоширенішим типом циклу в JavaScript є `for` цикл, бо він виконується певну кількість разів.

Цикли for оголошуються з трьома необов’язковими виразами, розділеними крапками з комою:

`for (a; b; c)`, де `a` — це інструкція ініціалізації, `b` — інструкція умови, а `c` — фінальний вираз.

Інструкція ініціалізації виконується лише один раз перед початком циклу. Зазвичай її використовують, щоб оголосити та налаштувати змінну циклу.

Інструкція умови оцінюється на початку кожної ітерації циклу і цикл продовжуватиме виконання, доки умова оцінюється як `true`. Коли умова на початку ітерації стає `false`, цикл припиняє виконання. Це означає, що якщо умова спочатку хибна, цикл ніколи не виконається.

Фінальний вираз виконується в кінці кожної ітерації циклу, перед наступною перевіркою умови, і зазвичай використовується для збільшення або зменшення лічильника циклу.

У наведеному прикладі ми ініціалізуємо `i = 0` і ітеруємо, доки наша умова `i < 5` є істинною. Ми збільшуємо `i` на `1` у кожній ітерації циклу, використовуючи `i++` як фінальний вираз.

```js
const ourArray = [];

for (let i = 0; i < 5; i++) {
  ourArray.push(i);
}
```

`ourArray` тепер матиме значення `[0, 1, 2, 3, 4]`.
