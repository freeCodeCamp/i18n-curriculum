# --description--

Agrega un elemento anchor con el texto `Read our terms and conditions` antes del `label` recién agregado. Establece el `href` a:

```md
https://www.freecodecamp.org/news/terms-of-service/
```

Luego agrega el texto `I accept the terms and conditions` inmediatamente después del elemento input en la etiqueta.

# --hints--

Debes tener un elemento `a` justo antes del `label` con el texto `Read our terms and conditions`.

```js
assert.match(document.querySelector('fieldset:nth-child(3) + a')?.innerText.trim().replaceAll(/\s+/g, ' '), /Read our terms and conditions\.?/i);
```

El elemento `a` debe tener un `href` de `https://www.freecodecamp.org/news/terms-of-service/`.

```js
assert.match(document.querySelector('fieldset:nth-child(3) + a')?.href, /https:\/\/www\.freecodecamp\.org\/news\/terms-of-service\/?/);
```

Debes tener un elemento `input` dentro del `label`.

```js
assert.exists(document.querySelector('fieldset:nth-child(3) + a + label > input'));
```

Dentro del `label`, inmediatamente después del elemento `input` debes tener el texto `I accept the terms and conditions`.

```js
assert.include(document.querySelector('fieldset:nth-child(3) + a + label')?.textContent.trim().replaceAll(/\s+/g, ' '), 'I accept the terms and conditions');
```
