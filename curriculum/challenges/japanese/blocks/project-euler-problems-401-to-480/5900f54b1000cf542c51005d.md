# --description--

$a_k$、$b_k$、$c_k$ は、式 $\frac{1}{x} = {\left(\frac{k}{x} \right)}^2 (k + x^2) - kx$ の3つの解（実数または複素数）を表します。

例えば、$k = 5$ の場合、$\\{a_5, b_5, c_5\\}$ はおよそ $\\{5.727244, -0.363622 + 2.057397i, -0.363622 - 2.057397i\\}$ となります。

$S(n) = \displaystyle\sum_{p = 1}^n \sum_{k = 1}^n {(a_k + b_k)}^p {(b_k + c_k)}^p {(c_k + a_k)}^p$ と定義します。ここで、$p$ と $k$ はすべての整数で $1 ≤ p, k ≤ n$ を満たします。

興味深いことに、$S(n)$ は常に整数になります。例えば、$S(4) = 51\\,160$ です。

$S({10}^6) \text{ を } 1\\,000\\,000\\,007 \text{ で割った余りを求めてください}$。
