# --description--

Vamos praticar escrever alguns números entre 50 e 59. Para completar esta tarefa, lembre-se de que em chinês, o número 50 é chamado `五十 (wǔ shí)`.

Os números de 0 a 9 são:

| Número | Chinês |
| :--- | :--- |
| 0 | `零 (líng)` |
| 1 | `一 (yī)` |
| 2 | `二 (èr)` |
| 3 | `三 (sān)` |
| 4 | `四 (sì)` |
| 5 | `五 (wǔ)` |
| 6 | `六 (liù)` |
| 7 | `七 (qī)` |
| 8 | `八 (bā)` |
| 9 | `九 (jiǔ)` |

# --instructions--

Esta tarefa não tem áudio. Complete os espaços em branco abaixo com os números correspondentes em chinês.
