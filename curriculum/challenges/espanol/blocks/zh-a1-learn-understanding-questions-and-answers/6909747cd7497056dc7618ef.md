# --explanation--

`我叫 (wǒ jiào)` significa "Me llamo..." o "Mi nombre es...". `叫 (jiào)` significa "llamarse". Aquí se usa para mostrar identidad. Por ejemplo:

`我叫王华 (wǒ jiào wáng huá)。` — Me llamo Wang Hua, o, mi nombre es Wang Hua.
