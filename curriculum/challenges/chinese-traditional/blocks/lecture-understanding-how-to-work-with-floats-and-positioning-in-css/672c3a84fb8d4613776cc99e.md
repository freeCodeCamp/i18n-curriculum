# --interactive--

在 CSS 中，定位允許我們控制元素在頁面上的佈局方式。兩種常見的定位類型是靜態定位和相對定位。元素預設是靜態定位。這表示它們遵循文件的正常流程，依序從上到下、從左到右排列。 

靜態定位是所有元素的預設值，且在 CSS 中不需要任何特殊的宣告。使用靜態定位時你不會注意到任何不同，因為它只是讓元素保持在文件中自然出現的位置。以下是一個使用預設靜態定位的段落範例：

:::interactive_editor

```html
<p>This paragraph is statically positioned.</p>
```

:::

當在瀏覽器中呈現時，這段落會出現在其自然位置，遵循正常的文件流程。沒有套用特殊樣式來移動它，並且它會保持在版面配置中自然出現的位置。

相對定位則允許元素從其正常位置移動，而不會干擾文件的流程。可以將其視為透過給予元素新的座標，將其從預設的靜態位置移動。以下是你可能如何應用相對定位的方法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="relative">This paragraph is positioned relatively.</p>
```

```css
body {
  border: solid 1px black;
}

.relative {
  position: relative;
  top: 30px;
  left: 30px;
}
```

:::

在此範例中，段落將會從其原始位置向下 `30px` 並向右 `30px`。然而，它在正常流程中所佔據的空間仍然被保留。相對定位在你想稍微移動元素而不影響其餘版面配置時非常有用。
