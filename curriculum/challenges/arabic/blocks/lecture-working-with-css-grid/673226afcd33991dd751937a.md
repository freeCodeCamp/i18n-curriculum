# --interactive--

تسمح خاصيتا `grid-column` و `grid-row` بتحديد الموضع الأفقي والعمودي لعناصر الشبكة داخل تخطيط الشبكة.

بمعنى آخر، تتيح لك كلتاهما التحكم في مكان بدء وانتهاء عنصر الشبكة من خلال الإشارة إلى خطوط الشبكة. هذه الخطوط هي الحدود التي تفصل بين الصفوف والأعمدة التي عرّفتها مسبقًا باستخدام خاصيتي `grid-template-rows` و `grid-template-columns`.

لنلقِ نظرة على صيغ كل من `grid-row` و `grid-column`، بالإضافة إلى مثال عملي.

إليك صيغة خاصية `grid-row`:

```css
grid-row: <start-line> / <end-line>;
```

وصيغة `grid-column`:

```css
grid-column: <start-line> / <end-line>;
```

`<start-line>` هو خط الشبكة الذي يبدأ عنده العنصر و `<end-line>` هو خط الشبكة الذي ينتهي عنده العنصر. كلاهما يبدأ العد من 1، أي تبدأ العد من 1 وليس من 0.

تذكر أن خطوط الشبكة للصفوف تُولد بناءً على عدد الصفوف المحددة في خاصية `grid-template-rows`. وينطبق نفس الأمر على الأعمدة مع خاصية `grid-template-columns`.

للمثال العملي، لنبدأ بالخطوط التي يولدها المتصفح عند استخدام كل من خاصيتي `grid-template-rows` و `grid-template-columns`.

إليك مثالًا لشبكة بها 4 أعمدة و3 صفوف:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}
```

:::

عند فحص حاوية الشبكة (الصنف `grid`) يظهر أن كل صف وعمود محاط بخطين – خط بداية في بداية الصف أو العمود، وخط نهاية في نهاية الصف أو العمود.

يمكنك استهداف هذه الخطوط لتبدأ باستخدام خاصيتي `grid-row` و `grid-column` لتحديد مكان وضع العنصر.

إليك كيف تجعل العنصر الشبكي الأول يحتل أول عمودين:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
}
```

:::

بهذا، تقول إن العنصر الشبكي الأول يجب أن يبدأ من العمود 1 وينتهي قبل العمود 3 مباشرة.

الآن سيشغل العنصر الأول عمودين، وسيُدفع العنصر الرابع إلى الصف الثاني. رائع!

إذا أردت أيضًا أن يشغل العنصر الأول صفين، يمكنك تحديد `grid-row` بقيمة `1 / 3`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
  grid-row: 1 / 3;
}
```

:::

يمكنك أيضًا استخدام الكلمة المفتاحية `span` لتخبر العنصر الشبكي بعدد الصفوف والأعمدة التي يجب أن يمتد عبرها. على سبيل المثال، `1 / 3` تعادل `1 / span 2`:

:::interactive_editor

```html

<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / span 2;
  grid-row: 1 / span 2;
}
```

:::

يمكنك الاستمرار في تطبيق هذه التقنية على أي عنصر في الشبكة ووضعه حيثما تريد.

مواصلةً للعملية، إليك كيف صنعنا تخطيطًا على شكل ماسوني مع العناصر.

هذا مثال جديد:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(4, 100px); /* 4 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / span 2;
}

.item4 {
  grid-column: 1 / span 3;
}

.item6 {
  grid-column: 1 / span 2;
}

.item7 {
  grid-column: 3 / span 2;
}
```
