# --description--

考虑数字 6。6 的除数是：1、2、3 和 6。

从 1 到 6（包括 6）中的每个数字都可以写成 6 的不同除数之和：

$1 = 1$，$2 = 2$，$3 = 1 + 2$，$4 = 1 + 3$，$5 = 2 + 3$，$6 = 6$。

如果一个数字 $n$ 满足从 1 到包括 $n$ 的每个数字都可以表示为 $n$ 的不同除数之和，则称该数字为实用数。

一对差异为6的连续素数被称为性感对（因为“性”是拉丁语中的“六”）。 第一个性感的对是 (23, 29)。

我们有时可能会查找到一个三重点对，这意味着三个连续的性感素数点对，其中每个点对的第二个成员是下一个点对的第一个成员。

我们称一个数字 $n$ 满足：

- ($n - 9$)、($n - 3$)、($n - 3$)、($n + 3$)、($n + 3$)、($n + 9$) 形成一个三点对，且
- 数字 $n - 8$、$n - 4$、$n$、$n + 4$ 和 $n + 8$ 都是实用数，

工程师的天堂。

查找前 4 个工程师的天堂的总和。
