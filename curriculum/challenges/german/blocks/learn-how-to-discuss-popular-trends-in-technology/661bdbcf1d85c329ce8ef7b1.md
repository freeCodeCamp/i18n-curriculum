# --description--

`Identity verification` ist wie der Nachweis, wer Sie sind. Es ist eine Methode, um zu bestätigen, dass die Person, die ein System oder einen Dienst nutzt, wirklich diejenige ist, die sie vorgibt zu sein. Das kann das Vorzeigen eines Ausweises, die Verwendung eines Passworts oder biometrische Verfahren wie Gesichtserkennung oder Fingerabdrücke umfassen.
