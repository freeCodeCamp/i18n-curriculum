# --description--

對於正整數 $n$，令 $σ(n)$ 為 $n$ 的所有因數之和，例如 $σ(6) = 1 + 2 + 3 + 6 = 12$。

完美數，正如你可能知道的，是一個滿足 $σ(n) = 2n$ 的數字。

讓我們定義正整數的完美商為 $p(n) = \frac{σ(n)}{n}$。

找出所有正整數 $n ≤ {10}^{18}$，使得 $p(n)$ 具有形式 $k + \frac{1}{2}$（其中 $k$ 是整數）的總和。
