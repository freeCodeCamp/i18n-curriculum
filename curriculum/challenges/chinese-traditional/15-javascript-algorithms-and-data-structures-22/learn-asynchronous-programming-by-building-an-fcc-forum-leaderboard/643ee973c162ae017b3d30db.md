# --description--

要在 `Activity` 欄中顯示數據，你需要使用每個主題的 `bumped_at` 屬性，這是一個 ISO 8601 格式的時間戳。 你需要先處理這些數據，然後才能顯示某個主題有任何活動後過去了多長時間。

創建一個帶有 `time` 參數的 `timeAgo` 函數。

在 `timeAgo` 函數中，創建兩個名爲 `currentTime` 和 `lastPost` 的變量，並將它們分別設置爲 `new Date()` 和 `new Date(time)`。

`lastPost` 將是一個主題上次活動的日期，`currentTime` 表示當前日期和時間。
