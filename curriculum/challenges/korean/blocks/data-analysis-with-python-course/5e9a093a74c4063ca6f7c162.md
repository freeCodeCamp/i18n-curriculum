## --text--

Pandas 모듈을 사용해 CSV 파일 `data.csv`을 어떻게 가져와 데이터프레임에 저장하나요?

## --answers--

```python
import pandas as pd
df = pd.csv("data.csv")
```

---

```python
import pandas as pd
df = pd.read_csv("data.csv")
```

---

```python
import pandas as pd
pd.read_csv("data.csv")
```

---

```python
import pandas as pd
df = pd.csv_reader("data.csv")
```
