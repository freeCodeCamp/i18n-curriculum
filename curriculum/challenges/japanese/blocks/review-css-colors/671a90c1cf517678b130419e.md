## 色彩理論

- **色彩理論の定義**：色が互いにどのように作用し、私たちの知覚にどのような影響を与えるかを研究する学問です。色の関係性、色の調和、色の心理的影響を扱います。
- **原色**：黄色、青、赤のこれらの色は、すべての他の色の基本となる色相です。
- **二次色**：二つの原色を等量混ぜてできる色です。緑、オレンジ、紫が二次色の例です。
- **三次色**：原色と隣接する二次色を組み合わせてできる色です。イエローグリーン、ブルーグリーン、ブルーバイオレットが三次色の例です。
- **暖色**：赤、オレンジ、黄色を含むこれらの色は、快適さ、暖かさ、居心地の良さを感じさせます。
- **寒色**：青、緑、紫を含むこれらの色は、落ち着き、静けさ、プロフェッショナルな印象を与えます。
- **カラーホイール**：色同士の関係を示す円形の図です。デザイナーにとって重要なツールで、色の組み合わせを選ぶのに役立ちます。
- **類似色配色**：カラーホイール上で隣接する類似色を使い、一体感があり落ち着いた印象を作り出します。
- **補色配色**：カラーホイールの正反対に位置する色を使い、高いコントラストと視覚的なインパクトを生み出します。
- **三色配色**：ほぼ等間隔に配置された三色を使い、鮮やかな色彩を作り出します。これらを結ぶとカラーホイール上で正三角形になります。
- **単色配色**：同じ基色から明度、暗度、彩度を調整して作られる配色で、統一感と調和を感じさせつつコントラストも生み出します。

## CSSでの色の扱い方

- **名前付き色**：ブラウザが認識するあらかじめ定義された色の名前です。例として `blue`、`darkred`、`lightgreen` があります。
- **`rgb()` 関数**：RGBは赤、緑、青の光の三原色を表します。これら三色を異なる強度で組み合わせて幅広い色を作り出します。`rgb()` 関数はRGBカラーモデルを使って色を定義できます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>RGB Color</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

- **`rgba()`関数**：この関数は4番目の値であるアルファを追加し、色の透明度を制御します。指定しない場合、アルファ値は`1`がデフォルトです。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>RGBA Background</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

- **`hsl()` 関数**：HSLは色相（Hue）、彩度（Saturation）、明度（Lightness）の三つの主要な要素で色を定義します。最新の `hsl()` 構文では、`/` 区切りでオプションのアルファ値も指定できます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>HSL Color</p>
<p class="alpha">HSL Color with Alpha</p>
```

```css
p {
  color: hsl(120 100% 50%);
}

.alpha {
  color: hsl(120 100% 50% / 0.8);
}
```

:::

- **`hsla()` 関数**：`hsla()` 関数はHSL色に透明度を加えるレガシーな方法です。最新のCSSでは、アルファ値を指定する場合は `/` 区切りの `hsl()` を使うことが推奨されています。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>HSLA Background</div>
```


```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

- **16進数表記**：16進数コード（hexコード）はRGBカラーモデルの色を表すために使われます。"hex" は16進数のことで、数字の `0` から `9` と文字の `A` から `F` を使います。6文字のコード（例：`#F498A2`）か、各ペアの文字が同じ場合に使える3文字の短縮形（例：`#2FB` は `#22FFBB` と同じ）があります。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1 class="hex-text">Hex Text</h1>
<p class="hex-bg">Hex Background</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}

p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

## `box-shadow`プロパティ

- **定義**: `box-shadow`プロパティは要素の周囲に1つ以上の影を適用します。
- **オフセット値**: 水平方向（`offset-x`）と垂直方向（`offset-y`）の値を指定します。正の`offset-x`は影を右に、負の値は左に移動させます。正の`offset-y`は影を下に、負の値は上に移動させます。値が`0`の場合、単位は不要です。
- **ぼかし半径**: 影のぼかし具合を制御する任意の値です。指定しない場合は`0`で、シャープなエッジになります。値が大きいほど影は柔らかくなります。
- **広がり半径**: 影の広がりや縮小を制御する任意の値です。指定しない場合は`0`です。
- **影の色**: 名前付き色、16進数、`rgb()`、`rgba()`、`hsl()`、`hsla()`関数で指定できます。
- **`inset` キーワード**: `inset`キーワードを追加すると、影が要素の内側に配置されます。
- **複数のボックスシャドウの適用**: 複数の影はカンマで区切って適用できます。影は前から後ろへ重なります。

### 構文

```css
box-shadow: offset-x offset-y blur-radius spread-radius color;
```

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="shadow-box">Shadow Color Example</div>
```

```css
.shadow-box {
  width: 200px;
  padding: 20px;
  background-color: lightblue;
  box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}
```

:::

## 線形および放射状グラデーション

- **線形グラデーション**：これらのグラデーションは直線に沿って色が徐々に混ざり合います。`to top`、`to right`、`to bottom right`のようなキーワードや`45deg`のような角度で方向を制御できます。任意の有効なCSSカラーと好きな数のストップを使えます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient">Linear Gradient</div>
```

```css
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

:::

- **放射状グラデーション**：これらのグラデーションは中心点から放射状に広がる円形または楕円形のグラデーションを作ります。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient">Radial Gradient</div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::
