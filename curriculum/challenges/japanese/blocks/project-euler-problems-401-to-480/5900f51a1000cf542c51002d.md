# --description--

$N$ 枚のディスクが一列に並べられており、左から右へ 1 から $N$ までのインデックスが付けられています。

各ディスクは黒い面と白い面を持ちます。最初はすべてのディスクが白い面を上にしています。

各ターンで、1 から $N$ までの整数 $A$ と $B$（同じ値でもかまいません）が一様にランダムに選ばれます。インデックスが $A$ から $B$ までのすべてのディスクが反転されます。

以下の例は $N = 8$ の場合を示しています。最初のターンでは $A = 5$、$B = 2$、2 回目のターンでは $A = 4$、$B = 6$ です。

<img alt="example for N = 8, with first turn A = 5 and B = 2, and second turn A = 4 and B = 6" src="https://cdn.freecodecamp.org/curriculum/project-euler/range-flips.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$E(N, M)$ を $M$ ターン後に白い面を上にしているディスクの期待値とします。$E(3, 1) = \frac{10}{9}$、$E(3, 2) = \frac{5}{3}$、$E(10, 4) ≈ 5.157$、$E(100, 10) ≈ 51.893$ であることが確認できます。

$E({10}^{10}, 4000)$ を求めてください。小数点以下2桁までで四捨五入してください。
