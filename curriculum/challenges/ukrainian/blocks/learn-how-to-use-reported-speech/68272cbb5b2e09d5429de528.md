# --explanation--

`Several` означає більше, ніж кілька, але не всі. Наприклад:

`Several users reported issues with the new software.` – Це означає, що більше двох користувачів мали проблеми, але не всі.

`To target` означає спрямовувати щось на когось або щось. Наприклад:

`The hackers targeted customers by sending fake emails.` – Це означає, що вони намагалися обдурити саме клієнтів.
