# --interactive--

Абсолютне позиціювання дозволяє вивести елемент із нормального потоку документа, роблячи його незалежним від інших елементів. Коли елемент позиціюється абсолютно, він розміщується на власному шарі, повністю відокремленому від усього іншого в макеті. Це корисно для створення плаваючих функцій інтерфейсу, таких як модальні вікна, підказки або випадаючі меню, які можуть накладатися на інші елементи сторінки.

За замовчуванням абсолютно позиційовані елементи розміщуються відносно найближчого позиційованого предка. Якщо позиційованого предка не знайдено, елемент буде розміщений відносно ініціального контейнера, яким зазвичай є вікно перегляду браузера. Ви можете переміщувати абсолютно позиційований елемент, використовуючи властивості `top`, `bottom`, `left` і `right`, щоб вказати, на яку відстань він має бути від країв свого контейнера.

Наприклад, встановлення `top: 30px` і `left: 30px` перемістить елемент `30px` від верхнього та лівого країв контейнера. Ось приклад, як застосувати абсолютне позиціювання до елемента в CSS:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="positioned">Absolutely Positioned</div>
```

```css
body {
  background-color: #eeeeee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

Коли цей код застосовано, елемент буде виведений із нормального потоку документа і розміщений `30px` від верхнього та лівого країв свого контейнера.

Абсолютне позиціювання особливо корисне для створення елементів, які мають плавати над рештою вмісту, наприклад, накладок або модальних вікон. Однак, оскільки елемент виводиться з потоку документа, це може створювати горизонтальні розриви або призводити до змикання інших елементів, якщо не обробляти це правильно.
