# --explanation--

`To be within budget` o `to stay within budget` significa gastar solo la cantidad de dinero que se planeó o permitió y no más. Esto muestra que el proyecto o la actividad está bien gestionado financieramente. Por ejemplo:

`The team stayed within budget, so we didn't need extra funding.` – Esto significa que no gastaron más de lo que se había planeado originalmente.
