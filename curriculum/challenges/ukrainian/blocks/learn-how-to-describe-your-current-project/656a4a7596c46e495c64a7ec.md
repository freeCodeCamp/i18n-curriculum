# --description--

To `work on` означає витрачати час і зусилля на виконання завдання або покращення чогось. Наприклад:

`She is working on a new app.` - Це означає, що вона розробляє або вдосконалює застосунок.

To `take time off` означає тимчасово припинити роботу, зазвичай для відпочинку або з особистих причин. Наприклад:

`I'm taking time off tomorrow because I have a doctor's appointment.` - Це означає не працювати протягом дня.
