# --explanation--

`Make sure` означає вжити дії, щоб підтвердити, що щось є правильним або завершеним. Наприклад:  

`Please make sure the report is submitted before the deadline.` – Це означає перевірку і впевненість, що звіт зданий вчасно.  

`Reliable` означає, що чомусь або комусь можна довіряти, що він працюватиме добре або буде послідовним з часом. Наприклад:

`The server is reliable; it has been running without any issues for months.` - Це означає, що сервер працює послідовно і не викликає проблем.

`More reliable` означає, що щось стало ще більш надійним або стабільним. Наприклад:

`The new software update makes the app more reliable.` - Це означає, що застосунок став більш надійним або стабільним після оновлення.
