# --description--

يسمح العرض الشرطي في React بإنشاء واجهات مستخدم ديناميكية. يُستخدم لعرض محتوى مختلف بناءً على شروط أو حالات معينة داخل التطبيق.

تشمل الطرق الأكثر شيوعًا لاستخدام العرض الشرطي استخدام جمل `if`، والمعامل الثلاثي (`?:`)، ومعامل AND المنطقي (`&&`).

أبسط أشكال العرض الشرطي يستخدم جملة `if`. إليك مثالًا:

```jsx
function Greeting({ isLoggedIn }) {
  if (isLoggedIn) {
    return <h1>Welcome back!</h1>;
  }
  return <h1>Please sign in</h1>;
}
```

في هذا المثال، يتحقق المكوِّن `Greeting` من الخاصية `isLoggedIn`. إذا كانت `true`، يعرض رسالة ترحيب، وإلا يحث المستخدم على تسجيل الدخول.

إليك مثالًا يستخدم المكوِّن `Greeting` داخل المكوِّن الأب `App`:

```jsx
function App() {
  return (
    <div className="App">
      <Greeting isLoggedIn={false} />
    </div>
  );
}
```

للحالات الأبسط، يُستخدم المعامل الثلاثي (`?:`) غالبًا مباشرة داخل JSX. يسمح بالعرض الشرطي المضمَّن، مما يجعل الشفرة أكثر إيجازًا:

```jsx
function Greeting({ isLoggedIn }) {
  return <h1>{isLoggedIn ? "Welcome back!" : "Please sign in."}</h1>;
}
```

تحقق هذه الشفرة نفس النتيجة مثل المثال السابق ولكن بشكل أكثر اختصارًا. يفحص المعامل الثلاثي `isLoggedIn` ويعرض الرسالة المناسبة.

نمط شائع آخر للعرض الشرطي هو استخدام معامل AND المنطقي (`&&`). هذا مفيد بشكل خاص عندما تريد عرض شيء ما، أو لا تعرض شيئًا، بناءً على شرط:

```jsx
function Notification({ message }) {
  return (
    <div>
      {message && <p>{message}</p>}
    </div>
  );
}
```

في هذا المثال، يتم عرض عنصر الفقرة الذي يحتوي على الرسالة فقط إذا كانت الخاصية `message` صحيحة. إذا كانت `message` قيمة خاطئة - مثل سلسلة نصية فارغة، `null`، أو `undefined`، فلا يتم عرض أي شيء على الشاشة.

بإتقان هذه التقنيات للعرض الشرطي، يمكنك بناء تطبيقات أكثر تفاعلية وسهلة الاستخدام تتكيف مع تغير البيانات وتفاعلات المستخدم.
