# --description--

„Die Verfolgung“ ist ein Spiel, das mit zwei Würfeln und einer geraden Anzahl von Spielern gespielt wird.

Die Spieler sitzen um eine Tabelle; das Spiel beginnt, indem zwei gegenüberliegende Spieler jeweils einen Würfel haben. In jedem Zug würfeln die beiden Spieler mit einem Würfel.

Wenn der Spieler eine 1 würfelt, gibt er den Würfel an seinen linken Nachbarn weiter.

Wenn der Spieler eine 6 würfelt, gibt er den Würfel an seinen rechten Nachbarn weiter.

Andernfalls behält er den Würfel für den nächsten Zug.

Das Spiel endet, wenn ein Spieler nach dem Würfeln und Weitergeben beide Würfel hat; dieser Spieler hat dann verloren.

In einem Spiel mit 100 Spielern, wie viele Züge dauert das Spiel im Durchschnitt? Geben Sie Ihre Antwort auf zehn signifikante Stellen gerundet an.
