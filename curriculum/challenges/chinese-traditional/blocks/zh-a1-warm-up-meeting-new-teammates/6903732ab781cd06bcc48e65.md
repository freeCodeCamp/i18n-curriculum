# --explanation--

`UI` stands for "user interface", which refers to the visual elements of a product that users interact with. `設計師 (shè jì shī)` means "designer". Together, `UI 設計師 (shè jì shī)` means a profession that focuses on designing user interfaces. For example:

`我是 UI 設計師。(wǒ shì UI shè jì shī.)` – I am a UI designer.
