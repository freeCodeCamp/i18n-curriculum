# --explanation--

`Is that right?` se usa para confirmar información que has escuchado. Es una forma común de pedir confirmación. Por ejemplo:

- Si alguien dice, `I'm starting a new job next week`, puedes responder, `Is that right?` para confirmar lo que dijo.
