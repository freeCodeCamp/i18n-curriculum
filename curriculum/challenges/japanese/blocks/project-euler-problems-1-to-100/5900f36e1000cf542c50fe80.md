# --description--

10未満の自然数のうち、3または5の倍数をすべて挙げると、3、5、6、9になります。これらの倍数の合計は23です。

与えられたパラメータ値 `number` 未満の3または5の倍数の合計を求めてください。

# --solutions--

```js
const multiplesOf3Or5 = (number) => {
  var total = 0;

  for(var i = 0; i < number; i++) {
    if(i % 3 == 0 || i % 5 == 0) {
      total += i;
    }
  }
  return total;
};
```
