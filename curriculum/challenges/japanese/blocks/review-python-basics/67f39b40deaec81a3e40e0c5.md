## Pythonとは何ですか？

- **概要**: Pythonはシンプルで使いやすいことで知られる汎用プログラミング言語です。Pythonはデータサイエンスや機械学習、ウェブ開発、スクリプトや自動化、組み込みシステムやIoTなど多くの分野で使われています。
- **よくある利用例**: Pythonはデータサイエンス、機械学習、ウェブ開発、サイバーセキュリティ、自動化、Raspberry PiやMicroPython対応ボードのようなマイクロコンピュータで使われています。

## ローカル環境でのPython

- **インストール**: Windows、Mac、LinuxでPythonをインストールする最良の方法は、公式Pythonウェブサイトからインストーラーをダウンロードすることです（`https://www.python.org/`）。

## 変数

- **変数の宣言**: 変数を宣言するには、変数名の後に代入演算子（`=`）を置き、その後に値を指定します。値は数字、文字列、ブール値などが使えます。以下は例です：

```py
name = 'John Doe'
age = 25
```

- **変数名の慣例**: 変数名には以下の慣例を守ってください：

  - 変数名は文字かアンダースコア（_）で始めることができ、数字では始められません。
  - 変数名には英数字（a-z、A-Z、0-9）とアンダースコア（_）のみ使えます。
  - 変数名は大文字小文字を区別します — `age`、`Age`、`AGE`はすべて別の名前として扱われます。
  - 変数名はPythonの予約キーワード（`if`、`class`、`def`など）と同じにできません。
  - 複数単語の変数名はアンダースコアで区切ります。例：`snake_case`。

## コメント

- **一行コメント**: これはコード内に短いメモを残すために使います。

```py
# This is a single line comment
```

- **複数行文字列**: これらは大きなメモを残したり、コードの一部をコメントアウトするのに使えます。

```py
"""
This is a multi-line string.
Here is some code commented out.

name = 'John Doe'
age = 25
"""
```

- **`print()`関数**: コンソールにデータを出力するには、`print()`関数を次のように使います：

```py
print('Hello world!') # Hello world!
```

## Pythonの一般的なデータ型

- **概要**: PythonはJavaScriptのような動的型付け言語で、変数の型を明示的に宣言する必要はありません。変数に代入された値によって型が決まります。
- **整数**: 小数点のない整数：

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **浮動小数点数**: 小数点を含む数値：

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **文字列**: 引用符で囲まれた文字の並び：

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **ブール値**: `True`か`False`のどちらかを表す値：

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **セット**: 順序のない一意の要素の集合：

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **辞書**: 波括弧で囲まれたキーと値のペアの集合：

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **タプル**: 丸括弧で囲まれた変更不可の順序付き集合：

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **レンジ**: ループでよく使われる数値のシーケンス：

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **リスト**: 複数の異なるデータ型を含められる順序付き集合：

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**: 値が存在しないことを表す特別な値：

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## 不変型と可変型

- **不変型**: 宣言後に変更できませんが、変数に新しい値を代入することはできます（再代入と呼ばれます）。整数、浮動小数点数、複素数、ブール値、文字列、タプル、レンジ、そして`None`が含まれます。
- **可変型**: 宣言後に変更可能で、要素の追加、削除、更新ができます。リスト、セット、辞書などのコレクション型が含まれます。
- **`type()`関数**: 変数の型を確認するには、次のように`type()`関数を使います：

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **`isinstance()`関数**: 変数が特定のデータ型かどうかをチェックするのに使います：

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## 文字列の操作

- **定義**: JavaScriptで覚えたように、文字列は不変で、一度作成したら変更できません。Pythonではシングルクォートかダブルクォートのどちらでも使えます。どちらかのルールを決めて統一することをお勧めします：

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **文字列の文字へのアクセス**: ブラケット記法を使って文字列の文字にアクセスできます：

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **文字列のエスケープ**: 文字列に引用符が含まれる場合はバックスラッシュ（`\`）を使います：

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **文字列の連結**: 文字列を連結するには`+`演算子を使います：

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

もう一つの連結方法は`+=`演算子を使うことです。これは連結と代入を同時に行います：

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**: これはフォーマット済み文字列リテラルの略称で、補間や連結をコンパクトで読みやすい構文で扱えます：

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **文字列のスライス**: 文字列の一部を抽出する操作です。基本構文は以下の通りです：

```py
str[start:stop:step]
```

開始位置は抽出を始めるインデックスを表します。終了位置はスライスの終わりで、この位置は含まれません。ステップはスライスの間隔を表します。例をいくつか示します：

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **文字列の長さを取得**: `len()`関数は文字列の文字数を返します：

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## `in`演算子の操作

- **`in`演算子**: 文字列に指定した文字が存在するかどうかを示すブール値を返します：

```py
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

## よく使う文字列メソッド

- **`str.upper()`**: すべての文字を大文字に変換した新しい文字列を返します：

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`str.lower()`**: すべての文字を小文字に変換した新しい文字列を返します：

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`str.strip()`**: 指定した先頭と末尾の文字を削除した文字列のコピーを返します（引数なしの場合は空白文字を削除）。

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**: 古い文字列のすべての出現箇所を新しい文字列に置き換えた新しい文字列を返します。

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**: 指定した区切り文字で文字列を分割し、リストを返します。区切り文字は分割位置を指定する文字列です。

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**: イテラブルの要素を区切り文字で連結して文字列を返します。イテラブルとはリスト、文字列、タプルのようにループ処理できるコレクションです。

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`str.startswith(prefix)`**: 指定した接頭辞で文字列が始まるかどうかを示すブール値を返します：

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`str.endswith(suffix)`**: 指定した接尾辞で文字列が終わるかどうかを示すブール値を返します：

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`str.find()`**: 部分文字列が最初に現れるインデックスを返します。見つからなければ`-1`を返します：

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`str.count(substring)`**: 部分文字列が文字列内に何回現れるかを数えます：

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`str.capitalize()`**: 最初の文字を大文字にし、他の文字を小文字にした新しい文字列を返します：

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`str.isupper()`**: 文字列のすべての文字が大文字なら`True`を返し、そうでなければ`False`を返します：

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`str.islower()`**: 文字列のすべての文字が小文字なら`True`を返し、そうでなければ`False`を返します：

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`str.title()`**: 各単語の最初の文字を大文字にした新しい文字列を返します：

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`str.maketrans()`**: 1対1の文字変換テーブルを作成するメソッドです。`translate()`メソッドと組み合わせて使い、そのテーブルを文字列に適用して変換結果を返します。

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## 整数と浮動小数点数でよく使う操作

- **基本的な算術演算**: Pythonでは整数と浮動小数点数で加算、減算、乗算、除算ができます：

```py
int_1 = 56
int_2 = 12
float_1 = 5.4
float_2 = 12.0

# Addition

print('Integer Addition:', int_1 + int_2) # Integer Addition: 68
print('Float Addition:', float_1 + float_2) # Float Addition: 17.4

# Subtraction

print('Int Subtraction:', int_1 - int_2) # Int Subtraction: 44
print('Float Subtraction:',  float_2 - float_1) # Float Subtraction: 6.6

# Multiplication

print('Int Multiplication:', int_1 * int_2) # Int Multiplication: 672
print('Float Multiplication:', float_2 * float_1) # Float Multiplication: 64.80000000000001

# Division

print('Int Division:', int_1 / int_2) # Int Division: 4.666666666666667
print('Float Division:', float_2 / float_1) # Float Division: 2.222222222222222
```

浮動小数点数と整数を加算すると、結果は浮動小数点数に変換されます：

```py
int_1 = 56
float_1 = 5.4

print(int_1 + float_1) # 61.4
```

- **剰余演算子（`%`）**: 数字を別の数字で割った余りを返します：

```py
int_1 = 56
int_2 = 12

print(int_1 % int_2) # 8
```

- **切り捨て除算（`//`）**: 2つの数値を割り、結果を最も近い小さい整数に切り捨てます：

```py
int_1 = 56
int_2 = 12

print(int_1 // int_2) # 4
```

- **べき乗演算子（`**`）**: 数値を別の数値のべき乗にします：

```py
int_1 = 4
int_2 = 2

print(int_1 ** int_2) # 16
```

- **`float()`関数**: 整数を浮動小数点数に変換するのに使います。

```py
num = 4

print(float(num)) # 4.0
```

- **`int()`関数**: 浮動小数点数を整数に変換するのに使います。

```py
num = 4.0

print(int(num)) # 4
```

- **`round()`関数**: 数値を最も近い整数に丸めるのに使います：

```py
num_1 = 3.4
num_2 = 7.7

print(round(num_1)) # 3
print(round(num_2)) # 8
```

- **`abs()`関数**: 数値の絶対値を返します：

```py
num = -13

print(abs(num)) # 13
```

- **`bin()`関数**: 整数を2進数表現の文字列に変換します：

```py
num = 56

print(bin(num))  # 0b111000
```

- **`oct()`関数**: 整数を8進数表現の文字列に変換します：

```py
num = 56

print(oct(num))  # 0o70
```

- **`hex()`関数**: 整数を16進数表現の文字列に変換します：

```py
num = 56

print(hex(num))  # 0x38
```

- **`pow()`関数**: 数値を別の数値のべき乗にします：

```py
result = pow(2, 3) 
print(result)  # 8
```

## 複合代入

- **定義**: 複合代入は二項演算と代入を1ステップで行います。変数に対して演算を適用し、その結果を同じ変数に代入します。

```py
# Addition assignment 
my_var = 10
my_var += 5

print(my_var) # 15

# Subtraction assignment
count = 14
count -= 3

print(count) # 11

# Multiplication assignment 
product = 65
product *= 7

print(product) # 455

# Division assignment 
price = 100
price /= 4

print(price) # 25.0

# Floor Division assignment 
total_pages = 23
total_pages //= 5

print(total_pages) # 4

# Modulo assignment 
bits = 35
bits %= 2

print(bits) # 1

# Exponentiation assignment 
power = 2
power **= 3

print(power) # 8
```

ビット演算子用の複合代入演算子もあります。`&=`、`^=`、`>>=`、`<<=`が含まれます。

## 関数の操作

- **定義**: 関数は入力（引数）を受け取り、出力を返す再利用可能なコードの塊です。関数を呼び出すには、関数名の後に丸括弧を付けます：

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call
print(result) # 7
```

関数が明示的に値を返さない場合、戻り値は`None`になります：

```py
def greet():
    print('hello') 

result = greet() # hello
print(result) # None
```

パラメータにデフォルト値を設定することもできます：

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

引数の数が正しくない場合、`TypeError`が発生します：

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

- **`input()`関数**: ユーザーからの入力を促すのに使います：

```py
name = input('What is your name?') # User types 'Kolade' and presses Enter  
print('Hello', name) # Hello Kolade
```

- **`int()`関数**: 数値、ブール値、または数値文字列を整数に変換します：

```py
print(int(3.14)) # 3

print(int(True)) # 1

```

## Pythonのスコープ

- **ローカルスコープ**: 関数やクラス内で宣言された変数は、その関数やクラス内でのみアクセスできます。

```py
def my_func():
    num = 10
    print(num)
```

- **囲むスコープ**: 関数の中にネストされた関数は、外側の関数の変数にアクセスできます。

```py
def outer_func():
    msg = 'Hello there!'
    
    def inner_func():
        print(msg)
    inner_func()

print(outer_func()) # Hello there!
```

- **グローバルスコープ**: 関数やクラスの外で宣言された変数は、プログラムのどこからでもアクセスできます。

```py
tax = 0.70 

def get_total(subtotal):
    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **組み込みスコープ**: Pythonの予約名で、組み込み関数、モジュール、キーワード、オブジェクトが含まれます。

```py
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```

## 比較演算子

- **等しい（`==`）**: 2つの値が等しいかチェックします：

```py
print(3 == 4) # False
```

- **等しくない（`!=`）**: 2つの値が等しくないかチェックします：

```py
print(3 != 4) # True
```

- **厳密に大きい（`>`）**: 1つの値がもう1つより大きいかチェックします：

```py
print(3 > 4) # False
```

- **厳密に小さい（`<`）**: 1つの値がもう1つより小さいかチェックします：

```py
print(3 < 4) # True
```

- **以上（`>=`）**: 1つの値がもう1つ以上かチェックします：

```py
print(3 >= 4) # False
```

- **以下（`<=`）**: 1つの値がもう1つ以下かチェックします：

```py
print(3 <= 4) # True
```

## `if`、`elif`、`else`文の操作

- **`if`文**: 何かが真かどうかを判定する条件です。条件が`True`と評価されると、そのコードブロックが実行されます。

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **`elif`節**: `if`文の後に続く条件です。`elif`節は、すべての前の条件が`False`と評価され、自身の条件が`True`と評価された場合にのみ実行されます。

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **`else`節**: 他の条件がすべて`True`と評価された場合に実行されます。

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

ネストされた`if`文も次のように使えます：

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## 真値と偽値

- **定義**: Pythonではすべての値に固有のブール値があり、論理的文脈で`True`か`False`として扱われます。多くの値は真値で、論理的文脈で`True`と評価されます。偽値は`False`と評価されます。以下は偽値の例です：

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

非ゼロの数値や空でない文字列などは真値です。

## `bool()`関数の操作

- **定義**: 値が真値か偽値かをチェックしたい場合、組み込みの`bool()`関数を使えます。これは値を明示的にブール値に変換し、真値なら`True`、偽値なら`False`を返します。例をいくつか示します：

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## ブール演算子とショートサーキット

- **定義**: これらは複数の式を組み合わせて複雑な条件判定を作る特別な演算子です。Pythonには3つのブール演算子があります：`and`、`or`、`not`。
- **`and`演算子**: 2つのオペランドを取り、最初のオペランドが偽値ならそれを返し、そうでなければ2番目のオペランドを返します。両方のオペランドが真値の場合にのみ式は真値になります。

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

条件文で`and`演算子を次のように使うこともできます：

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **`or`演算子**: 2つのオペランドを取り、最初のオペランドが真値ならそれを返し、そうでなければ2番目のオペランドを返します。`or`式は少なくとも1つのオペランドが真値なら真値になります。例を示します：

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

`and`演算子と同様に、条件文で`or`演算子を次のように使えます：

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **ショートサーキット**: `and`と`or`演算子はショートサーキット演算子として知られています。ショートサーキットとは、Pythonが左から右に値をチェックし、最終結果が決まった時点で処理を停止することです。
- **`not`演算子**: 1つのオペランドを取り、そのブール値を反転させます。真値を`False`に、偽値を`True`に変換します。これまでの演算子とは異なり、`not`は常に`True`か`False`を返します。例を示します：

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

条件文での`not`演算子の例：

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```
