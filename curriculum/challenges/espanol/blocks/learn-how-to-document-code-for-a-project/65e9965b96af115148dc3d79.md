# --description--

`Straight` significa en una línea o camino directo. For example, `Walk straight until you see the store.` 

`Forward` significa hacia el frente o en la dirección a la que uno se enfrenta. Like `He stepped forward to volunteer.` 

`Straightforward` combina estas ideas para significar algo simple y fácil de entender, sin complejidad. For instance, `The instructions were straightforward and easy to follow.`

`Keep` significa continuar teniendo o manteniendo. An example is `to keep the door closed`.
