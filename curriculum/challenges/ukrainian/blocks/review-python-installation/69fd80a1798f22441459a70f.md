## Python у вашому локальному середовищі

- **Встановлення**: Найкращий спосіб встановити Python на Windows, Mac і Linux — скачати інсталятор з офіційного сайту Python (`https://www.python.org/`).
- **Термінал**: Текстовий інтерфейс, який дозволяє взаємодіяти з комп’ютером, вводячи команди. Кожна операційна система має застосунок термінала за замовчуванням. На macOS можна використовувати застосунок Terminal. На Windows — Command Prompt або PowerShell. На Linux кожне робоче середовище має свій термінал за замовчуванням, наприклад GNOME Terminal або Konsole.
- **IDE**: IDE означає інтегроване середовище розробки, яке має функції, включно з редактором коду, інструментами тестування, терміналом і не тільки.
- **Популярні редактори коду та IDE для Python**: VS Code, PyCharm і Spyder
- **Запуск коду локально**: Щоб запустити код Python, у вас є кілька варіантів. Перший — натиснути на кнопку запуску у VS Code, яка виглядає як кнопка відтворення у верхньому правому куті. Це відкриє термінал і запустить ваш скрипт Python там.

Інший варіант — відкрити термінал і вручну запустити програму. Ось приклад запуску програми з назвою `main.py`:

```bash
python main.py
```

Цю команду потрібно запускати з папки, де збережено `main.py`. Наприклад, якщо `main.py` знаходиться у папці з назвою `python-projects`, спочатку використайте у терміналі таке:

```bash
cd python-projects
```

Потім запустіть:

```bash
python main.py
```

## Використання інтерактивної оболонки Python

- **Визначення**: Інтерактивна оболонка — це програма, яка дозволяє вводити команди по одній і бачити результати. Відкрийте застосунок термінала і введіть `python`, потім натисніть `Enter`. Це запустить інтерактивну оболонку Python. Символ `>>>` означає, що Python чекає на введення команди. Спробуйте вивести `"Hello, world!"` у термінал:

```bash
print("Hello, world!")
```

Ви повинні побачити текст, який з’явиться так:

```bash
>>> print("Hello, world!")
Hello, world!
```

Після виведення тексту Python повертається до такого вигляду:

```bash
>>>
```

Це означає, що ви можете ввести ще одну команду.

- **Цикл читання, оцінки, виведення, повторення (REPL)**: Коли ви вводите команди, інтерпретатор читає вхідні дані, оцінює їх, виводить результат і повертається, щоб показати `>>>`, щоб ви могли вводити нові команди.
