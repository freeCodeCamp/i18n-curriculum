# --interactive--

Arrays in JavaScript sind dynamisch, was bedeutet, dass Sie ganz einfach Elemente hinzufügen oder entfernen können. Es gibt vier Hauptmethoden, um Elemente am Anfang und Ende eines Arrays hinzuzufügen und zu entfernen: `push()`, `pop()`, `shift()` und `unshift()`. Lassen Sie uns jede dieser Methoden im Detail ansehen.

Die `push()`-Methode wird verwendet, um ein oder mehrere Elemente am Ende eines Arrays hinzuzufügen. Der Rückgabewert der `push()`-Methode ist die neue Länge des Arrays. Hier ist ein Beispiel, wie Sie eine neue Frucht zum bestehenden `fruits`-Array hinzufügen:

:::interactive_editor

```js
const fruits = ["apple", "banana"];
const newLength = fruits.push("orange");
console.log(newLength); // 3
console.log(fruits); // ["apple", "banana", "orange"]
```

:::

In diesem Beispiel beginnen wir mit einem Array namens `fruits`, das zwei Elemente enthält. Dann verwenden wir die `push()`-Methode, um den String `orange` am Ende des Arrays hinzuzufügen.

Vielleicht ist Ihnen aufgefallen, dass wir bei der Deklaration des `const`-Arrays `fruits` verwenden. Aber warum ist es möglich, diesem `fruits`-Array weitere Elemente hinzuzufügen, obwohl `fruits` eine Konstante ist? Das ist möglich, weil die Deklaration eines Arrays mit dem Schlüsselwort `const` eine Referenz auf das Array erstellt. Während das Array selbst veränderbar ist und modifiziert werden kann, können Sie der `fruits`-Konstanten keinen neuen Wert zuweisen, wie hier:

```js
const fruits = ["apple", "banana"];
fruits = ["This", "will", "not", "work"];
console.log(fruits); // Uncaught TypeError: Assignment to constant variable. 
```

Die nächste Methode, die wir uns ansehen, ist die `pop()`-Methode. Die `pop()`-Methode entfernt das letzte Element aus einem Array und gibt dieses Element zurück. Sie verändert auch das ursprüngliche Array. So funktioniert das:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let lastFruit = fruits.pop();
console.log(fruits); // ["apple", "banana"]
console.log(lastFruit); // "orange"
```

:::

In diesem Beispiel starten wir mit einem Array aus drei Früchten. Die `pop()`-Methode entfernt das letzte Element (`orange`) aus dem Array und gibt es zurück. Das ursprüngliche `fruits`-Array wird verändert und enthält nur noch zwei Elemente.

Die `unshift()`-Methode fügt ein oder mehrere Elemente am Anfang eines Arrays hinzu und gibt dessen neue Länge zurück. Sie funktioniert ähnlich wie `push()`, verändert aber den Anfang des Arrays statt das Ende. Hier ein Beispiel:

:::interactive_editor

```js
let numbers = [2, 3];
let newLength = numbers.unshift(1);
console.log(numbers); // [1, 2, 3]
console.log(newLength); // 3
```

:::

In diesem Beispiel verwenden wir `unshift()`, um die Zahl `1` am Anfang des `numbers`-Arrays hinzuzufügen. Die Methode gibt die neue Länge des Arrays zurück, die `3` beträgt.

Schließlich entfernt die `shift()`-Methode das erste Element aus einem Array und gibt dieses Element zurück. Sie ist ähnlich wie `pop()`, arbeitet aber am Anfang des Arrays statt am Ende. So funktioniert das:

:::interactive_editor

```js
let colors = ["red", "green", "blue"];
let firstColor = colors.shift();
console.log(colors); // ["green", "blue"]
console.log(firstColor); // "red"
```

:::

In diesem Beispiel starten wir mit einem Array aus drei Farben. Die `shift()`-Methode entfernt das erste Element (`red`) aus dem Array und gibt es zurück. Das ursprüngliche `colors`-Array wird verändert und enthält nur noch zwei Elemente.

Beachten Sie, dass `push()` und `unshift()` mehrere Elemente auf einmal hinzufügen können, während `pop()` und `shift()` jeweils nur ein Element entfernen.
