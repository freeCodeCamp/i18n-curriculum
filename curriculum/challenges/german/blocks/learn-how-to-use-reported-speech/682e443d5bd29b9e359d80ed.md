# --explanation--

`To conduct` bedeutet, eine organisierte Aktivität wie ein Meeting, einen Test oder eine Inspektion durchzuführen oder zu leiten. Zum Beispiel:

`The manager will conduct interviews for the new position tomorrow.` – Das bedeutet, der Manager wird die Interviews leiten oder durchführen.

Eine `audit` ist eine detaillierte Kontrolle oder Überprüfung von etwas, meist um sicherzustellen, dass es Standards oder Regeln entspricht. Zum Beispiel:

`The IT team performed a system audit to check for security risks.` – Das bedeutet, sie haben das System überprüft, um mögliche Probleme zu finden.
