# --description--

`That sounds great` è un'espressione usata per manifestare entusiasmo o approvazione riguardo a una proposta, un piano o un invito. Per esempio:

- Persona 1: `We could try pair programming tomorrow.`

- Persona 2: `That sounds great!` - Sei entusiasta della proposta.
