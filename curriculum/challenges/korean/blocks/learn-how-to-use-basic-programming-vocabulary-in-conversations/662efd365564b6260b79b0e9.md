# --description--

`loop`는 특정 조건에 도달할 때까지 계속 반복되는 명령어 수열입니다. `Loops`는 동일한 코드를 여러 번 효율적으로 실행하는 데 사용됩니다. 일반적인 `loops` 유형에는 `for` 루프와 `while` 루프가 포함됩니다.

예를 들어, `for loop`: `for (int i = 0; i < 5; i++) { /* code to be executed */ }`

예제에서 중괄호 `{ }` 안의 코드는 조건이 더 이상 참이 아닐 때까지 반복해서 실행됩니다.
