# --description--

JSONとは何か、そしてブラケット記法とドット記法で値にどうアクセスするかを学びましょう。

JSONはJavaScript Object Notationの略です。これは軽量でテキストベースのデータ形式で、サーバーとウェブアプリケーション間でデータをやり取りする際によく使われます。

JSONがウェブ開発で非常に人気がある理由の一つは、機械が解析可能でありながら人間にも読みやすいからです。

JSONは言語に依存しないため、JavaのアプリケーションからPythonのアプリケーションへ、あるいはJavaScriptのアプリケーションからC#のアプリケーションへ簡単にJSONデータを送ることができます。

JSONはオブジェクト、配列、文字列、ブール値、null、数値など多くのデータ型をサポートしています。

以下はJSONオブジェクトの例です：

```js
{
  "name": "Alice",
  "age": 30,
  "isStudent": false,
  "list of courses": ["Mathematics", "Physics", "Computer Science"]
}
```

ご覧の通り、JSONはキーと値のペアで情報を保存し、各ペアはカンマで区切られています。各キーは二重引用符で囲む必要があり、そうしないとエラーになります。

JSONオブジェクトからデータにアクセスするには、ドット記法かブラケット記法のどちらかを使えます。この例ではドット記法を使ってJSONオブジェクトの`age`にアクセスしています：

```js
import data from "./example.json" with { type: "json" };

console.log(data.age);
```

この例は`import`文と呼ばれるもので、JSONオブジェクトをこのファイルにインポートしてアクセスできるようにしています。`import`文については今後のレッスンで詳しく学びます。

また、ブラケット記法を使ってJSONオブジェクトから情報にアクセスすることもできます。以下は`list of courses`配列にアクセスする例です：

```js
import data from "./example.json" with { type: "json" };

console.log(data["list of courses"]);
```

ブラケット記法は、キーに複数の単語がスペースで区切られている場合に特に便利です。ドット記法を使うとエラーになります。

まとめると、JSONは配列やネストされたオブジェクトを含む多くのデータ型を保存できる多用途な形式です。ドット記法やブラケット記法を使えば、JSONオブジェクト内の値に簡単にアクセスできます。
