# --description--

정수와 부동 소수점 숫자는 Python에서 기본적인 숫자 데이터 유형입니다. 이를 사용해 숫자 데이터를 저장하고 수학 연산을 수행할 수 있습니다.

정수와 부동 소수점 숫자가 무엇인지, 산술 연산을 어떻게 수행하는지, 그리고 Python이 두 가지 모두를 다루기 위해 제공하는 여러 메서드를 살펴보겠습니다.

정수는 소수점이 없는 양수 또는 음수의 온전한 숫자입니다:

```python
my_int_1 = 56
my_int_2 = -4

print(type(my_int_1)) # <class 'int'>
print(type(my_int_2)) # <class 'int'>
```

정수로 덧셈 연산을 수행하는 방법은 다음과 같습니다:

```python
my_int_1 = 56
my_int_2 = 12

sum_ints = my_int_1 + my_int_2
print('Integer Addition:', sum_ints) # Integer Addition: 68
```

정수로 뺄셈을 수행하는 방법은 다음과 같습니다:

```python
my_int_1 = 56
my_int_2 = 12

# Subtraction
diff_ints = my_int_1 - my_int_2
print('Integer Subtraction:', diff_ints) # Integer Subtraction: 44
```

정수로 곱셈 연산을 수행하는 방법은 다음과 같습니다:

```python
my_int_1 = 12
my_int_2 = 4

# Multiplication
product_ints = my_int_1 * my_int_2
print('Integer Multiplication:', product_ints) # Integer Multiplication: 48
```

정수로 나눗셈 연산을 수행하는 방법은 다음과 같습니다:

```python
my_int_1 = 56
my_int_2 = 12

# Division
div_ints = my_int_1 / my_int_2
print('Integer Division:', div_ints) # Integer Division: 4.666666666666667
```

부동 소수점 숫자는 `3.14`, `-0.5`, 또는 `0.0`처럼 소수점이 있는 양수 또는 음수입니다.

```python
my_float_1 = -12.0
my_float_2 = 4.9

print(type(my_float_1)) # <class 'float'>
print(type(my_float_2)) # <class 'float'>
```

부동 소수점 숫자로 덧셈 연산을 수행하는 방법은 다음과 같습니다:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_addition = my_float_1 + my_float_2
print('Float Addition:', float_addition) # Float Addition: 17.4
```

부동 소수점 숫자로 뺄셈 연산을 수행하는 방법은 다음과 같습니다:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_subtraction = my_float_2 - my_float_1
print('Float Subtraction:', float_subtraction) # Float Subtraction: 6.6
```

부동 소수점 숫자로 곱셈 연산을 수행하는 방법은 다음과 같습니다:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_multiplication = my_float_2 * my_float_1
print('Float Multiplication:', float_multiplication) # Float Multiplication: 64.80000000000001
```

부동 소수점 숫자로 나눗셈 연산을 수행하는 방법은 다음과 같습니다:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_division = my_float_2 / my_float_1
print('Float Division:', float_division) # Float Division: 2.222222222222222
```

정수와 부동 소수점 숫자를 더하면 결과가 자동으로 부동 소수점 숫자로 변환됩니다:

```python
my_int = 56
my_float = 5.4

sum_int_and_float = my_int + my_float

print(sum_int_and_float) # 61.4
print(type(sum_int_and_float)) # <class 'float'>
```

뺄셈, 곱셈, 나눗셈 같은 다른 기본 산술 연산도 마찬가지입니다. 정수와 부동 소수점 숫자를 섞으면 Python은 결과를 부동 소수점 숫자로 반환합니다.

또한, 나머지 연산자, 몫 연산, 거듭제곱과 같은 더 복잡한 산술 계산도 정수와 부동 소수점 숫자 모두에 대해 수행할 수 있습니다.

나머지 연산자(`%`)는 왼쪽 값이 오른쪽 값으로 나누어질 때 나머지를 반환합니다:

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

mod_ints = my_int_1 % my_int_2
mod_floats = my_float_2 % my_float_1

print('Integer Modulo:', mod_ints) # Integer Modulo: 8
print('Float Modulo:', mod_floats) # Float Modulo: 1.1999999999999993
```

몫 연산은 두 숫자를 나누고 결과보다 작거나 같은 가장 큰 정수를 반환합니다. 이는 이중 슬래시 연산자(`//`)로 수행합니다:

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

floor_div_ints = my_int_1 // my_int_2
floor_div_floats = my_float_2 // my_float_1

print('Integer Floor Division:', floor_div_ints) # Integer Floor Division: 4
print('Float Floor Division:', floor_div_floats) # Float Floor Division: 2.0
```

거듭제곱은 한 숫자를 다른 숫자의 거듭제곱으로 올리는 연산이며, 이중 별표 연산자(`**`)로 수행합니다:

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

exp_ints = my_int_1 ** my_int_2
exp_floats = my_float_1 ** my_float_2

print('Integer Exponentiation:', exp_ints) # Integer Exponentiation: 951166013805414055936
print('Float Exponentiation:',  exp_floats) # Float Exponentiation: 614787626.1765089
```

때때로 부동 소수점 숫자가 포함된 연산 결과가 예상보다 더 많은 소수 자릿수를 가질 수 있습니다. 예를 들어, `0.1 + 0.2`의 합은 `0.30000000000000004` 대신 `0.3`입니다.

이것은 숫자가 이진 형식으로 저장되고 일부 분수가 이진으로 정확히 나타낼 수 없기 때문에 발생합니다. 결과적으로, 무한한 자릿수를 가진 `1/3` 분수가 10진수에서 유한한 자릿수로 나타낼 수 없고 `0.33333...` 이후로 잘리는 것과 같은 방식으로 유한한 근사값으로 저장됩니다.

이로 인해 작은 반올림 오차가 발생합니다.

Python은 숫자 데이터나 문자열을 정수 또는 부동 소수점 숫자로 변환하는 내장 함수를 제공합니다.

`float()` 함수는 주어진 숫자로부터 부동 소수점 숫자를 반환합니다:

```python
my_int_1 = 56
my_float_1 = float(my_int_1)

print(my_float_1)  # 56.0
print(type(my_float_1))  # <class 'float'>
```

`int()` 함수는 주어진 숫자로부터 정수를 반환합니다:

```python
my_float = 12.92563
my_int = int(my_float)

print(my_int)  # 12
print(type(my_int))  # <class 'int'>
```

또한, 같은 내장 함수를 사용해 문자열을 부동 소수점 숫자나 정수로 변환할 수 있습니다:

```python
my_str_int = '45'
my_str_float = '7.8'

converted_int = int(my_str_int)
converted_float = float(my_str_float)

print(converted_int, type(converted_int))  # 45 <class 'int'>
print(converted_float, type(converted_float))  # 7.8 <class 'float'>
```

다음은 Python이 정수와 부동 소수점 숫자를 다루기 위해 제공하는 다른 메서드입니다.

- `round()`: 숫자를 지정한 소수점 자리수로 반올림합니다. 기본적으로 이 함수는 가장 가까운 정수로 반올림하며, 소수점 자릿수가 없는 정수를 반환합니다:
    

```python
my_int_1 = 4.798
my_int_2 = 4.253

rounded_int_1 = round(my_int_1)
rounded_int_2 = round(my_int_2, 1)

print(rounded_int_1) # 5
print(rounded_int_2) # 4.3
```

- `abs()`: 숫자의 절댓값을 반환합니다,
    

```python
num = -15

absolute_value = abs(num)
print(absolute_value) # 15
```

- `pow()`: 숫자를 거듭제곱하거나 모듈로 거듭제곱을 수행합니다.
    

```python
result_1 = pow(2, 3)  # Equivalent to 2 ** 3
print(result_1)  # 8

result_2 = pow(2, 3, 5)  # (2 ** 3) % 5
print(result_2)  # 3
```
