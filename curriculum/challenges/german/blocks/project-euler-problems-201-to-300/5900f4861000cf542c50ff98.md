# --description--

Sie erhalten eine Pizza (perfekter Kreis), die in $m·n$ gleich große Stücke geschnitten wurde, und Sie möchten genau einen Belag auf jedem Stück haben.

Sei $f(m,n)$ die Anzahl der Möglichkeiten, wie Sie die Pizza mit $m$ verschiedenen Belägen ($m ≥ 2$) belegen können, wobei jeder Belag genau auf $n$ Stücken verwendet wird ($n ≥ 1$). Spiegelungen werden als unterschiedlich betrachtet, Drehungen nicht.

Zum Beispiel gilt $f(2,1) = 1$, $f(2,2) = f(3,1) = 2$ und $f(3,2) = 16$. $f(3,2)$ ist unten dargestellt:

<img alt="Animation mit 16 Möglichkeiten, 3 verschiedene Beläge jeweils auf 2 Stücken zu verteilen" src="https://cdn.freecodecamp.org/curriculum/project-euler/pizza-toppings.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Finden Sie die Summe aller $f(m,n)$, für die $f(m,n) ≤ {10}^{15}$ gilt.
