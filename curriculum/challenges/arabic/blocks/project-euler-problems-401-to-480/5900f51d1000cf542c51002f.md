# --description--

ليكن $E(x_0, y_0)$ عدد الخطوات اللازمة لتحديد القاسم المشترك الأكبر لـ $x_0$ و $y_0$ باستخدام خوارزمية إقليدس. بشكل أكثر رسمية:

$$\begin{align}
  & x_1 = y_0, y_1 = x_0\bmod y_0 \\\\
  & x_n = y_{n - 1}, y_n = x_{n - 1}\bmod y_{n - 1}
\end{align}$$

$E(x_0, y_0)$ هو أصغر $n$ بحيث يكون $y_n = 0$.

لدينا $E(1, 1) = 1$، $E(10, 6) = 3$ و $E(6, 10) = 4$.

عرف $S(N)$ كمجموع $E(x, y)$ لكل $1 ≤ x$، $y ≤ N$.

لدينا $S(1) = 1$، $S(10) = 221$ و $S(100) = 39\\,826$.

احسب $S(5 \times {10}^6)$.
