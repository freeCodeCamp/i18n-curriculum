How would you access the value of the `name` key in the following JSON object using dot notation?

```js
{
  "name": "Alice",
  "age": 30
}
```

## --answers--

`data("name")`

### --feedback--

Dot notation involves using a period (`.`) followed by the key name.

`data["name"]`
