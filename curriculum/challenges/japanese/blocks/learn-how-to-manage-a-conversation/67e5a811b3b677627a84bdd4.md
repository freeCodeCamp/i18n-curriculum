# --explanation--

`Might` は何かが可能であることを示すために使われますが、確実ではありません。ジェシカはバグが問題を引き起こす「かもしれない」と言っていますが、まだ確認されていません。これに対して`Simple Present`や`we've discovered a critical bug that affects the user interface`は、バグが確実に問題を引き起こすことを意味します。その他の例：

- `The delay might cause problems.` – 可能性はあるが確実ではない。

- `The delay causes problems.` – 確実に問題を引き起こす。
