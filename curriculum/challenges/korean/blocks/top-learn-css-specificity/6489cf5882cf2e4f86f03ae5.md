# --description--

이제, 조금 바꿔보겠습니다:

```html
<!-- index.html -->

<div class="main">
  <div class="list" id="subsection"></div>
</div>
```

```css
/* rule 1 */
#subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

위 예제에서, 규칙 2가 ID 선택자보다 클래스 선택자가 더 많음에도 불구하고, ID가 클래스보다 우선하기 때문에 규칙 1이 더 구체적입니다.
