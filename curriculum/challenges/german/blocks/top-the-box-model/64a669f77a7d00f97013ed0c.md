# --description--

Jetzt, da Sie die grundlegende Syntax von HTML und CSS verstehen, wird es ernst. Die wichtigsten Fähigkeiten, die Sie mit CSS beherrschen müssen, sind Positionierung und Layout. Schriftarten und Farben zu ändern ist eine wichtige Fähigkeit, aber die Fähigkeit, Elemente genau dort auf einer Webseite zu platzieren, wo Sie sie haben möchten, ist noch entscheidender. Schließlich, wie viele Webseiten finden Sie, bei denen wirklich jedes Element einfach nur übereinander gestapelt ist?

Das Positionieren von Elementen auf einer Webseite zu lernen ist nicht so schwierig, sobald Sie nur ein paar Schlüsselkonzepte verstanden haben. Leider eilen viele Lernende durch das Erlernen von HTML und CSS, um zu JavaScript zu kommen, und verpassen dabei diese grundlegenden Konzepte. Das führt zu Frustration, Ärger (und lustigen Gifs), denn alle JavaScript-Fähigkeiten der Welt sind bedeutungslos, wenn Sie Ihre Elemente nicht an der Stelle auf der Seite platzieren können, an der sie sein müssen. Also, mit diesem Gedanken im Hinterkopf, legen wir los.
