# --interactive--

## Listen gestalten

- **`line-height` Eigenschaft**: Diese Eigenschaft wird verwendet, um Abstand zwischen Textzeilen zu schaffen. Akzeptierte `line-height` Werte umfassen das Schlüsselwort `normal`, Zahlen, Prozentsätze und Längeneinheiten wie die `em` Einheit.
- **`list-style-type` Eigenschaft**: Diese Eigenschaft wird verwendet, um das Markierungssymbol für ein Listenelement festzulegen. Zulässige Werte können ein Kreis, eine Scheibe oder eine Dezimalzahl sein.

Hier ist ein Beispiel, das `list-style-type` verwendet, um den Aufzählungsstil zu ändern:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
ul {
  list-style-type: square;
}
```

:::

- **`list-style-position` Eigenschaft**: Diese Eigenschaft wird verwendet, um die Position des Listensymbols festzulegen. Die einzigen zwei zulässigen Werte sind inside und outside.

Hier ist ein Beispiel, das den Unterschied zwischen `inside` und `outside` zeigt:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="inside-list">
  <li>Item with inside position</li>
  <li>Item with inside position</li>
</ul>
<ul class="outside-list">
  <li>Item with outside position</li>
  <li>Item with outside position</li>
</ul>
```

```css
.inside-list {
  list-style-position: inside;
}

.outside-list {
  list-style-position: outside;
}
```

:::

- **`list-style-image` Eigenschaft**: Diese Eigenschaft wird verwendet, um ein Bild als Listensymbol zu verwenden. Ein häufiger Anwendungsfall ist die Verwendung der `url` Funktion mit einem Wert, der auf einen gültigen Bildpfad gesetzt ist.

## Abstand zwischen Listenelementen mit `margin`

- Neben `line-height` können auch Außenabstände in CSS verwendet werden, um den Abstand und die Lesbarkeit von Listenelementen zu verbessern.
- Außenabstände schaffen Raum außerhalb jedes `li` Elements und ermöglichen so die Steuerung des Abstands zwischen Listenelementen.
- `margin-bottom` wird verwendet, um unter jedem Listenelement Abstand zu schaffen. Zum Beispiel erzeugt `margin-bottom: 10px;` einen 10-Pixel-Abstand unter jedem Listenelement.

Hier ist ein Beispiel, das `margin-bottom` verwendet, um Listenelemente zu trennen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="list">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
.list li {
  margin-bottom: 20px;
}
```

:::

## Links gestalten

- **`pseudo-class`**: Dies ist ein Schlüsselwort, das einem Selektor hinzugefügt wird und es Ihnen ermöglicht, Elemente basierend auf einem bestimmten Zustand auszuwählen. Häufige Zustände sind `:hover`, `:visited` und `:focus`.
- **`:link pseudo-class`**: Diese Pseudoklasse wird verwendet, um Links zu gestalten, die vom Benutzer noch nicht besucht wurden.

Hier ist ein Beispiel, das die `:link` Pseudoklasse verwendet:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:link {
  color: red;
}
```

:::

- **`:visited pseudo-class`**: Diese Pseudoklasse wird verwendet, um Links zu gestalten, die der Benutzer bereits besucht hat.
- **`:hover pseudo-class`**: Diese Pseudoklasse wird verwendet, um Elemente zu gestalten, über die der Benutzer gerade mit der Maus fährt.

Hier ist ein Beispiel, das die `:hover` Pseudoklasse verwendet:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Hover over me!</a>
```

```css
a:hover {
  color: green;
  text-decoration: underline;
}
```

:::

- **`:focus pseudo-class`**: Diese Pseudoklasse wird verwendet, um ein Element zu gestalten, wenn es den Fokus erhält. Beispiele sind `input` oder `select` Elemente, auf die geklickt oder mit der Tabulatortaste fokussiert wird.

Hier ist ein Beispiel, das die `:focus` Pseudoklasse verwendet:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:focus {
  outline: 2px solid orange;
}
```

:::

- **`:active pseudo-class`**: Diese Pseudoklasse wird verwendet, um ein Element zu gestalten, das vom Benutzer aktiviert wurde. Ein häufiges Beispiel ist, wenn der Benutzer auf eine Schaltfläche klickt.

Hier ist ein Beispiel, das die `:active` Pseudoklasse verwendet:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Click me!</a>
```

```css
a:active {
  color: pink;
}
```

:::

## Arbeiten mit Hintergründen und Rahmen

- **`background-size` Eigenschaft**: Diese Eigenschaft wird verwendet, um die Hintergrundgröße eines Elements festzulegen. Einige gängige Werte sind `cover`, damit das Hintergrundbild das gesamte Element abdeckt, und `contain`, damit das Hintergrundbild innerhalb des Elements passt.

Hier ist ein Beispiel mit `background-size: cover`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

Hier ist ein Beispiel mit `background-size: contain`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **`background-repeat` Eigenschaft**: Diese Eigenschaft bestimmt, wie Hintergrundbilder entlang der horizontalen und vertikalen Achse wiederholt werden sollen. Der Standardwert für `background-repeat` ist `repeat`, was bedeutet, dass das Bild sowohl horizontal als auch vertikal wiederholt wird. Sie können auch angeben, dass keine Wiederholung erfolgen soll, indem Sie die `no-repeat` Eigenschaft verwenden.

Hier ist ein Beispiel mit `background-repeat: no-repeat`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **`background-position` Eigenschaft**: Diese Eigenschaft wird verwendet, um die Position des Hintergrundbilds festzulegen. Sie kann auf eine bestimmte Länge, einen Prozentsatz oder Schlüsselwörter wie `top`, `bottom`, `left`, `right` und `center` gesetzt werden.

Hier ist ein Beispiel mit `background-position`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

- **`background-attachment` Eigenschaft**: Diese Eigenschaft legt fest, ob das Hintergrundbild mit dem Inhalt mitscrollen oder an einer festen Position bleiben soll. Die Hauptwerte sind `scroll` (Standard), bei dem das Hintergrundbild mit dem Inhalt mitscrollt, und `fixed`, bei dem das Hintergrundbild an der gleichen Position auf dem Bildschirm bleibt.
- **`background-image` Eigenschaft**: Diese Eigenschaft wird verwendet, um das Hintergrundbild eines Elements festzulegen. Sie können mehrere Hintergrundbilder gleichzeitig setzen und entweder die `url`, `radial-gradient` oder `linear-gradient` Funktionen als Werte verwenden.

Hier ist ein Beispiel mit `background-image`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **`background` Eigenschaft**: Dies ist die Kurzschreibweise-Eigenschaft, um alle Hintergrund-Eigenschaften in einer Deklaration zu setzen. Hier ist ein Beispiel, wie man das Hintergrundbild setzt und es so einstellt, dass es sich nicht wiederholt: `background: no-repeat url("example-url-goes-here");`

Hier ist ein Beispiel mit der `background` Kurzschreibweise-Eigenschaft:

:::interactive_editor

```html
<style>
  body {
    background: center top no-repeat url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **Guter Kontrast für Hintergrund- und Vordergrundfarben**: Es ist wichtig sicherzustellen, dass Hintergrund- und Vordergrundfarben einen guten Kontrast haben, damit der Text lesbar ist. Die Web Content Accessibility Guidelines (WCAG) empfehlen ein Mindestkontrastverhältnis von 4,5:1 für normalen Text und 3:1 für großen Text.

## Rahmen

- **`border-top` Eigenschaft**: Diese Eigenschaft wird verwendet, um die Stile für den oberen Rahmen eines Elements festzulegen. `border-top: 3px solid blue;` setzt einen 3 Pixel breiten, durchgezogenen blauen Rahmen auf der Oberseite des Elements.
- **`border-right` Eigenschaft**: Diese Eigenschaft wird verwendet, um die Stile für den rechten Rahmen eines Elements festzulegen. `border-right: 2px solid red;` setzt einen 2 Pixel breiten, durchgezogenen roten Rahmen auf der rechten Seite des Elements.
- **`border-bottom` Eigenschaft**: Diese Eigenschaft wird verwendet, um die Stile für den unteren Rahmen eines Elements festzulegen. `border-bottom: 1px dashed green;` setzt einen 1 Pixel breiten, gestrichelten grünen Rahmen auf der Unterseite des Elements.
- **`border-left` Eigenschaft**: Diese Eigenschaft wird verwendet, um die Stile für den linken Rahmen eines Elements festzulegen. `border-left: 4px dotted orange;` setzt einen 4 Pixel breiten, gepunkteten orangefarbenen Rahmen auf der linken Seite des Elements.

Hier ist ein Beispiel, das einzelne Rahmen-Eigenschaften verwendet:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="bordered-box">Box with different borders</div>
```

```css
.bordered-box {
  border-top: 3px solid blue;
  border-right: 2px solid red;
  border-bottom: 1px dashed green;
  border-left: 4px dotted orange;
  padding: 20px;
}
```

:::

- **`border` Eigenschaft**: Dies ist die Kurzschreibweise-Eigenschaft, um Breite, Stil und Farbe eines Rahmens eines Elements festzulegen. `border: 1px solid black;` setzt einen 1 Pixel breiten, durchgezogenen schwarzen Rahmen.

Hier ist ein Beispiel, das die `border` Kurzschreibweise-Eigenschaft verwendet:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

- **`border-radius` Eigenschaft**: Diese Eigenschaft wird verwendet, um abgerundete Ecken für den Rahmen eines Elements zu erstellen.

Hier ist ein Beispiel mit `border-radius`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

- **`border-style` Eigenschaft**: Diese Eigenschaft wird verwendet, um den Stil eines Rahmens eines Elements festzulegen. Einige akzeptierte Werte sind `solid`, `dashed`, `dotted` und `double`.

Hier ist ein Beispiel mit verschiedenen `border-style` Werten:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="solid-border">Solid border</div>
<div class="dashed-border">Dashed border</div>
<div class="dotted-border">Dotted border</div>
<div class="double-border">Double border</div>
```

```css
.solid-border {
  border: 3px solid blue;
  margin-bottom: 10px;
  padding: 10px;
}

.dashed-border {
  border: 3px dashed red;
  margin-bottom: 10px;
  padding: 10px;
}

.dotted-border {
  border: 3px dotted green;
  padding: 10px;
}

.double-border {
  border: 3px double blueviolet;
  padding: 10px;
}
```

:::

## Verläufe

- **`linear-gradient()` Funktion**: Diese CSS-Funktion wird verwendet, um einen Übergang zwischen mehreren Farben entlang einer geraden Linie zu erzeugen.

Hier ist ein Beispiel mit `linear-gradient()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

- **`radial-gradient()` Funktion**: Diese CSS-Funktion erzeugt ein Bild, das von einem bestimmten Punkt ausstrahlt, wie ein Kreis oder eine Ellipse, und allmählich zwischen mehreren Farben übergeht.

Hier ist ein Beispiel mit `radial-gradient()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```
