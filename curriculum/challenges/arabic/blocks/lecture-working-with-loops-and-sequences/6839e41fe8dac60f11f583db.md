# --description--

تُستخدم الدالة `range()` لتوليد متتالية من الأعداد الصحيحة. فيما يلي الصيغة النحوية الأساسية للدالة `range()`:

```py
range(start, stop, step)
```

المعلمة المطلوبة `stop` هي عدد صحيح يمثل نقطة النهاية للمتتالية التي يتم توليدها. إليك مثالًا على استخدام الدالة `range()`:

```py
for num in range(3):
    print(num)
```

الشيفرة التالية تولد متتالية أعداد بين `0` و `2`. العدد الصحيح `3` غير مدرج لأن المعلمة `stop` غير شاملة للنهاية.

إذا لم تُحدد المعلمة `start`، فالقيمة الافتراضية هي `0`. وإلا، يمكنك استخدام المعلمة الاختيارية `start` لبدء المتتالية من عدد صحيح غير `0`. إليك مثالًا على توليد متتالية أعداد صحيحة بين `1` و `4`:

```py
for num in range(1, 5):
    print(num)
```

افتراضيًا، ستزداد المتتالية بمقدار `1`. لكن إذا أردت تغيير هذا الافتراض، يمكنك استخدام المعلمة الاختيارية `step`. إليك مثالًا على توليد متتالية أعداد صحيحة زوجية بين `2` و `10`:

```py
for num in range(2, 11, 2):
    print(num)
```

كما ذُكر سابقًا، هناك معلمة واحدة مطلوبة فقط للدالة `range()`. إذا لم تقدم أي معلمات لـ `range()`، فستحصل على خطأ `TypeError`:

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: range expected at least 1 argument, got 0
```

من المهم ملاحظة أن الدالة `range()` تقبل فقط الأعداد الصحيحة كمعلمات، وليس الأعداد العشرية. تذكر أن الأعداد العشرية هي أرقام تحتوي على فاصلة عشرية مثل `3.4`. إذا حاولت تمرير أعداد عشرية كمعلمات، فستحصل على خطأ `TypeError`:

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

إذا أردت توليد متتالية أعداد صحيحة تناقصية، يمكنك استخدام عدد صحيح سالب للمعلمة `step`، كما يلي:

```py
for num in range(40, 0, -10):
    print(num)
```

الشيفرة التالية تطبع الأعداد `40`، `30`، `20`، و `10` بهذا الترتيب في وحدة التحكم.

شيء آخر يمكنك فعله مع الدالة `range()` هو إنشاء قائمة من الأعداد الصحيحة باستخدامها مع دالة البناء `list`. تُستخدم دالة البناء `list` لتحويل قابل للتكرار إلى قائمة. إليك مثالًا على توليد قائمة من الأعداد الصحيحة الزوجية بين `2` و `10`:

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

الدالة `range()` طريقة مفيدة جدًا لتوليد متتالية من الأعداد الصحيحة في بايثون. بمجرد أن تتقنها، ستجد نفسك تستخدمها كثيرًا في برامج بايثون الخاصة بك.
