# --interactive--

هناك عدة طرق لحذف الخصائص من كائن، ويُعتبر عامل `delete` أبسطها والأكثر استخدامًا.

عند استخدام `delete`، فإنه يحذف الخاصية المحددة من الكائن. إليك مثالًا على كيفية استخدام عامل `delete`:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  job: "Engineer"
};

delete person.job;

console.log(person.job); // undefined
```

:::

في هذا المثال، نبدأ بكائن `person` يحتوي على ثلاث خصائص: `name` و `age` و `job`. ثم نستخدم عامل `delete` لحذف الخاصية `job`. بعد الحذف، لم يعد كائن `person` يحتوي على الخاصية `job`.

طريقة أخرى لحذف الخصائص هي باستخدام تعيين بالتفكيك مع معلمات الباقي. هذه الطريقة لا تحذف الخاصية فعليًا، لكنها تنشئ كائنًا جديدًا بدون الخصائص المحددة:

:::interactive_editor

```js
const person = {
  name: "Bob",
  age: 25,
  job: "Designer",
  city: "New York"
};

const { job, city, ...remainingProperties } = person;

// { name: "Bob", age: 25 }
console.log(remainingProperties);
```

:::

في هذا المثال، نستخدم التفكيك لاستخراج `job` و `city` من كائن `person`، ونجمع الخصائص المتبقية في كائن جديد يسمى `remainingProperties`. هذا ينشئ كائنًا جديدًا بدون خصائص `job` و `city`.

فهم كيفية حذف الخصائص من الكائنات مهارة مهمة في برمجة JavaScript. تتيح لك التلاعب بالكائنات ديناميكيًا وتنظيف البيانات غير الضرورية.
