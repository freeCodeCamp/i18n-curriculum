`might`

### --feedback--

這個字表示可能性——某事不確定但可能發生或需要。

# --explanation--

`Any` 用來指涉一個或多個某物，且不具體說明。 例如：

如果你有任何問題，請告訴我。

`Might` 表示某事是可能的，但不確定。 例如：

`We might need more time to finish.`－這表示可能需要額外的時間。
