# --interactive--

У CSS позиціонування дозволяє контролювати, як елементи розташовуються на сторінці. Два загальні типи позиціонування — це статичне позиціонування та відносне позиціонування. За замовчуванням елементи мають статичне позиціонування. Це означає, що вони слідують нормальному потоку документа, один за одним, зверху вниз, зліва направо.

Статичне позиціонування є за замовчуванням для всіх елементів і не потребує спеціального оголошення в CSS. Ви не помітите нічого особливого при використанні статичного позиціонування, бо воно просто залишає елементи там, де вони природно розташовані в документі. Ось приклад абзацу з використанням статичного позиціонування за замовчуванням:

:::interactive_editor

```html
<p>This paragraph is statically positioned.</p>
```

:::

Коли цей абзац відтворюється у браузері, він з’явиться у своєму природному положенні, слідуючи нормальному потоку документа. Ніяких спеціальних стилів для його переміщення не застосовано, і він залишається там, де природно має бути в макеті.

Відносне позиціонування, натомість, дозволяє зсунути елемент зі свого нормального положення, не порушуючи потік документа. Уявіть, що ви переміщуєте елемент від його позиції за замовчуванням, задаючи йому нові координати. Ось як можна застосувати відносне позиціонування:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="relative">This paragraph is positioned relatively.</p>
```

```css
body {
  border: solid 1px black;
}

.relative {
  position: relative;
  top: 30px;
  left: 30px;
}
```

:::

У цьому прикладі абзац з’явиться `30px` вниз і `30px` вправо від свого початкового положення. Проте простір, який він би займав у нормальному потоці, залишається збереженим. Відносне позиціонування дуже корисне, коли потрібно трохи зсунути елемент, не впливаючи на решту макету.
