# --description--

次の方程式において、x、y、n は正の整数です。

$$\frac{1}{x} + \frac{1}{y} = \frac{1}{n}$$

`n` = 1260 のとき、113 個の異なる解が存在し、これは異なる解の総数が100を超える最小の `n` の値であることが確認できます。

異なる解の数が400万を超える最小の `n` の値はいくつですか？

**注意:** この問題は問題108のはるかに難しいバージョンであり、単純な総当たりアプローチの限界を超えているため、巧妙な実装が必要です。
