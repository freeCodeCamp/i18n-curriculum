# --description--

La frase `give that a go` significa provare qualcosa, di solito per la prima volta, per vedere se ti piace o se riesci a farcela.

Per esempio, `I've never tried sushi before, but I'll give it a go.` Lo speaker usa questa frase per mostrare apertura a provare il sushi.
