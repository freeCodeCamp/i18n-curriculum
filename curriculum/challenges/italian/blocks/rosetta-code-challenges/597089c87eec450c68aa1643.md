# --description--

Definisci un tipo che si comporta come un intero ma ha un valore valido più basso di 1 e un valore valido più alto di 10.

Gestione degli errori:

<ul>
  <li>Se provi a istanziare un <code>Num</code> con un valore esterno all'intervallo 1 - 10, deve lanciare un <code>TypeError</code> con un messaggio di errore <code>'Out of range'</code>.</li>
  <li>Se provi a istanziare un <code>Num</code> con un valore che non è un numero, deve lanciare un <code>TypeError</code> con un messaggio di errore <code>'Not a Number'</code>.</li>
</ul>

# --solutions--

```js
function Num(n) {
  if (isNaN(n)) {
    throw new TypeError('Not a Number');
  }
  if (n < 1 || n > 10) {
    throw new TypeError('Out of range');
  }

  this._value = +n;
}
Num.prototype.valueOf = function() { return this._value; };
Num.prototype.toString = function () { return this._value.toString(); };
```
