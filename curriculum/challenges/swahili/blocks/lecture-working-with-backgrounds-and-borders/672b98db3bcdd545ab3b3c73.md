# --interactive--

Katika CSS, kuna njia kadhaa za kuongeza mipaka kando ya picha, kila moja ikitoa chaguzi tofauti za mtindo na viwango vya udhibiti.

Tuchunguze baadhi ya njia maarufu na zinazobadilika zaidi.

Njia rahisi zaidi ya kuongeza mpaka kwa picha ni kwa kutumia sifa ya `border`. Sifa hii ni kifupi inayokuwezesha kuweka upana, mtindo, na rangi ya mpaka kwa wakati mmoja.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

Sheria hii ya CSS inaongeza mpaka mwekundu imara wenye upana wa pikseli 2 kando ya vipengele vyote vya `img`. Unaweza kubadilisha upana, mtindo (kama `dashed`, `dotted`, au `double`), na rangi ili kufaa mahitaji ya muundo wako.

Ikiwa unahitaji udhibiti zaidi juu ya pande binafsi za mpaka, unaweza kutumia sifa maalum za mpaka kwa kila upande:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border-top: 10px solid red;
  border-right: 10px dashed green;
  border-bottom: 10px dotted blue;
  border-left: 10px double purple;
}
```

:::

Hii inakuwezesha kuunda mitindo ya kipekee ya mpaka kwa kila upande wa picha.

Njia nyingine ya kuunda athari ya mpaka ni kwa kutumia sifa ya `outline`. Ingawa ni sawa na border, outline haibadilishi vipimo au mpangilio wa kipengele:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  outline: 3px solid gold;
}
```

:::

Hii huunda mstari wa dhahabu kando ya picha bila kubadilisha ukubwa au nafasi yake.

Ikiwa unataka kuunda pembe zilizozungushwa kwenye mpaka wako, unaweza kutumia sifa ya `border-radius` pamoja na border:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

Kumbuka, mbinu hizi zinaweza kuchanganywa na kubinafsishwa kuunda athari za kipekee za mpaka. Uchaguzi wa njia unategemea mahitaji maalum ya muundo wako na kiwango cha ugumu unachohitaji.
