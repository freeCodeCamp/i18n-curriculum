# --description--

Un altro modo per usare i selettori è concatenarli come una lista senza alcuna separazione. Supponiamo di avere il seguente HTML:

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection preview">This is where a preview for a post might go.</p>
</div>
```

Hai due elementi con la classe `subsection` che hanno uno stile unico, ma cosa succede se vuoi applicare una regola separata solo all'elemento che ha anche `header` come seconda classe? Bene, potresti concatenare entrambi i selettori `class` nel tuo CSS in questo modo:

```css
.subsection.header {
  color: red;
}
```

Quello che fa `.subsection.header` è selezionare qualsiasi elemento che abbia entrambe le classi `subsection` e `header`. Nota come non ci sia alcuno spazio tra i selettori `.subsection` e `.header` `class`. Questa sintassi funziona praticamente per concatenare qualsiasi combinazione di selettori, tranne concatenare più di un selettore di tipo.

Questo può essere usato anche per concatenare una classe e un id, come mostrato qui sotto:

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection" id="preview">This is where a preview for a post might go.</p>
</div>
```

Puoi prendere i due elementi sopra e combinarli con il seguente:

```css
.subsection.header {
  color: red;
}

.subsection#preview {
  color: blue;
}
```

In generale, non puoi concatenare più di un selettore di tipo perché un elemento non può essere due tipi diversi contemporaneamente. Per esempio, concatenare due selettori di tipo come `div` e `p` ci darebbe il selettore `divp`, che non funzionerebbe perché il selettore cercherebbe un elemento letterale `<divp>`, che non esiste.
