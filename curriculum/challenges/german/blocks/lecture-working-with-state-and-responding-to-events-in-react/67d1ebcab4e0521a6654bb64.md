# --description--

In React ist das Aktualisieren von Arrays im State recht einfach, aber es ist leicht, einen Fehler zu machen, besonders wenn Sie von Vanilla JavaScript kommen, wo Sie Arrays direkt verändern können.

In React wird der State als unveränderlich behandelt, damit Änderungen erkannt und die Benutzeroberfläche korrekt aktualisiert werden kann.

Schauen wir uns an, wie Sie Arrays, die im State gehalten werden, in React aktualisieren können.

Einer der häufigsten Fehler beim Aktualisieren von Arrays im React-State ist, das Array direkt mit Methoden wie `push()`, `pop()` oder `splice()` zu verändern. Diese Methoden verändern das ursprüngliche Array, und React erlaubt das nicht.

React verlässt sich auf eine neue Array-Referenz, um Änderungen zu erkennen. Das direkte Verändern des Arrays kann verhindern, dass die Komponente wie erwartet neu gerendert wird.

Hier ist ein Beispiel für die Verwendung der `push()`-Methode, um ein Array im State zu erweitern, was nicht funktioniert:

```jsx
import { useState } from "react";

function ItemsList() {
  const [items, setItems] = useState([
    { id: 0, name: "Item 1" },
    { id: 1, name: "Item 2" },
    { id: 2, name: "Item 3" },
  ]);

  const addItem = () => {
    const newItem = { id: items.length + 1, name: `Item ${items.length + 1}` };
    items.push(newItem); // This modifies the original array
    setItems(items); // React will not detect this change
  };

  return (
    <div>
      <button onClick={addItem}>Add Item</button>
      <ul>
        {items.map((item) => (
          <li key={item.id}>{item.name}</li>
        ))}
      </ul>
    </div>
  );
}

export default ItemsList;
```

Wenn Sie auf den `Add Item`-Button klicken, passiert in der Benutzeroberfläche nichts.

Es könnte auch verlockend sein, Elemente mit der `pop()`-Methode aus dem Array zu entfernen:

```js
const removeItem = () => {
  items.pop(); // Modifies the original array
  setItems(items); // React will not detect this change, either
};
```

Um ein Array im State zu aktualisieren, ist der Schlüssel, ein neues Array zu erstellen, Ihre Operationen darauf auszuführen und dieses an React zu übergeben, anstatt das bestehende Array zu verändern.

Da es sich um ein neues Array handelt, weiß React, dass sich der State geändert hat, und löst ein erneutes Rendern aus.

So können Sie mit dem Spread-Operator Elemente zum `items`-Array hinzufügen:

```js
const addItem = () => {
  const newItem = {
    id: items.length + 1,
    name: `Item ${items.length + 1}`,
  };

  // Creates a new array
  setItems((prevItems) => [...prevItems, newItem]);
};
```

`[...prevItems, newItem]` erstellt ein neues Array, indem alle Elemente des bestehenden `items`-Arrays im State kopiert werden, und fügt dann `newItem` am Ende hinzu, was `id` und die Elementnummer erhöht.

Wenn Sie etwas aus dem Array entfernen möchten, können Sie die `filter()`-Methode verwenden, die ein neues Array zurückgibt, nachdem das Gewünschte herausgefiltert wurde:

```js
const removeItem = (id) => {
  setItems((prevItems) => prevItems.filter((item) => item.id !== id));
};
```

Hier ist der vollständige Code:

```jsx
import { useState } from "react";

function ItemsList() {
  const [items, setItems] = useState([
    { id: 0, name: "Item 1" },
    { id: 1, name: "Item 2" },
    { id: 2, name: "Item 3" },
  ]);

  const addItem = () => {
    const newItem = { id: items.length + 1, name: `Item ${items.length + 1}` };
    setItems((prevItems) => [...prevItems, newItem]); // Creates a new array
  };

  const removeItem = (id) => {
    setItems((prevItems) => prevItems.filter((item) => item.id !== id)); // Creates a new array
  };

  return (
    <div>
      <button onClick={addItem}>Add Item</button>
      <ul>
        {items.map((item) => (
          <li key={item.id}>
            {item.name}{" "}
            <button onClick={() => removeItem(item.id)}>Remove</button>
          </li>
        ))}
      </ul>
    </div>
  );
}

export default ItemsList;
```

Das sind einige gängige Methoden, um ein Array im State zu aktualisieren.
