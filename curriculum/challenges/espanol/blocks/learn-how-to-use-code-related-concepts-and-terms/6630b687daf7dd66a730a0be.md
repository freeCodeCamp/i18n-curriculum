# --description--

La palabra `fundamental` implica que algo es básico o esencial, similar a la base de una casa.

Por ejemplo, `Understanding grammar is fundamental to learning a new language.` Aquí `fundamental` se usa para enfatizar que la gramática es una parte esencial o básica de aprender cualquier nuevo idioma.

La frase `what kind of` se usa para preguntar sobre el tipo o la naturaleza de algo. Es una forma común de especificar o clasificar algo con más claridad.

Por ejemplo, cuando alguien pregunta `what kind of music do you like?` está preguntando sobre el género de música.
