# --explanation--

Mark menciona `incompatible drivers` y `outdated software` como causas de un fallo del software.

Los `Incompatible drivers` son controladores que no funcionan correctamente con el hardware, mientras que el `outdated software` significa que el software no ha sido actualizado a su última versión. Ambos pueden causar fallos. Por ejemplo:

- `The graphics card isn't working because of incompatible drivers.` - El controlador incorrecto está causando que la tarjeta gráfica funcione mal.

- `Outdated software can lead to security vulnerabilities.` - Los programas antiguos podrían no ser seguros de usar debido a la falta de actualizaciones.
