# --interactive--

Tukio la `DOMContentLoaded` hufanyika wakati kila kitu katika hati ya HTML kimepakuliwa na kuchambuliwa. Ikiwa una majalada ya mitindo ya nje, au picha, tukio la `DOMContentLoaded` halitasubiri hayo kupakuliwa. Litangojea tu HTML kupakuliwa.

Hii ni tofauti na tukio la `load`, ambalo hangojea kila kitu kupakuliwa, ikiwa ni pamoja na majalada ya mitindo ya nje, picha, na kadhalika.

Hapa kuna sintaksia ya mfano ya kutumia tukio la `DOMContentLoaded`:

```js
document.addEventListener("DOMContentLoaded", function () {
  console.log("DOM is loaded.");
});
```

Mara DOM inapopakuliwa, kitendakazi kitaendeshwa na ujumbe wa `DOM is loaded.` utaandikwa kwenye konsoli.

Sasa, tuchunguze mfano mwingine tunapotumia tukio la `DOMContentLoaded`. Katika mfano huu, tuna picha ndani ya HTML. Ili kusasisha picha, tunaweza kuunda kitendakazi kinachobadilisha sifa za `src` na `alt` za picha:

:::interactive_editor

```html
<h1>Image Change on DOM Loaded</h1>
<img
  id="example-img"
  src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg"
  alt="Cat relaxing"
/>

<script src="index.js"></script>
```

```js
function changeImg() {
  const img = document.getElementById("example-img");
  img.src =
    "https://cdn.freecodecamp.org/curriculum/responsive-web-design-principles/FCCStickers-CamperBot200x200.jpg";
  img.alt = "CamperBot sticker";
  console.log("image was just changed");
}

changeImg();
```

:::

Kisha tunaweza kuangalia kama DOM bado inapakia na kuongeza msikilizaji wa tukio la `DOMContentLoaded`. Ikiwa tukio la `DOMContentLoaded` tayari limefanyika, tunaweza kuita kitendakazi cha `changeImg` moja kwa moja:

```js
if (document.readyState === "loading") {
  document.addEventListener("DOMContentLoaded", changeImg);
} else {
  console.log("DOMContentLoaded has already fired");
  changeImg();
}
```

Ikiwa utaendesha msimbo huu, na kuonyesha upya ukurasa, utaona picha kubadilika kuwa picha mpya baada ya DOM kupakuliwa. Hii hutokea haraka sana, lakini utaona mwangaza mdogo wakati picha inabadilika.

Tukio la `DOMContentLoaded` ni muhimu unapotaka kuendesha msimbo wa JavaScript mara tu DOM inapopakuliwa, lakini kabla ya rasilimali zote za nje kama picha na majalada ya mitindo kupakuliwa.
