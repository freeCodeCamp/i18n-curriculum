# --interactive--

Unapofanya kazi na mfuatano wa herufi katika JavaScript, mara nyingi unahitaji kutoa sehemu au mfuatano mdogo kutoka kwa mfuatano mkubwa.

Kwa mfano, unaweza kutaka kutoa sehemu ya neno, mfululizo maalum wa herufi, au hata kipande cha sentensi.

JavaScript hutoa njia kadhaa za zoezi hili, mojawapo ya njia zinazotumika sana ni njia ya `slice()`.

Njia ya `slice()` inakuwezesha kutoa sehemu ya mfuatano wa herufi na kurudisha mfuatano mpya, bila kubadilisha mfuatano wa asili. Inachukua vigezo viwili: kielezo cha kuanzia na kielezo cha mwisho kinachoweza kutolewa.

Hapa ni sintaksia ya msingi:

```js
string.slice(startIndex, endIndex);
```

`startIndex` ni nafasi ambapo utoaji huanza. `endIndex` ni mahali ambapo utoaji unakamilika. Ikiwa haijatolewa, `slice()` hutoa hadi mwisho wa mfuatano wa herufi.

Tuchunguze mfano rahisi wa kutoa sehemu ya mfuatano wa herufi:

:::interactive_editor

```js
let message = "Hello, world!";
let greeting = message.slice(0, 5);

console.log(greeting);  // Hello
```

:::

Katika mfano huu, `slice(0, 5)` hutoka kuanzia kielezo `0` hadi lakini si pamoja na kielezo `5`. Matokeo yake, neno `Hello` linatolewa.

Ikiwa utapuuza kigezo cha pili, `slice()` itatoa kila kitu kuanzia kielezo cha kuanzia hadi mwisho wa mfuatano wa herufi:

:::interactive_editor

```js
let message = "Hello, world!";
let world = message.slice(7);

console.log(world);  // world!
```

:::

Hapa, `slice(7)` hutoka mfuatano wa herufi kuanzia kielezo `7` hadi mwisho wa mfuatano, na matokeo ni `world!`.

Unaweza pia kutumia nambari hasi kama vielezo. Unapotumia nambari hasi, inahesabu nyuma kutoka mwisho wa mfuatano wa herufi:

:::interactive_editor

```js
let message = "JavaScript is fun!";
let lastWord = message.slice(-4);

console.log(lastWord);  // fun!
```

:::

Katika kesi hii, `slice(-4)` hutoka herufi nne za mwisho kutoka kwa mfuatano, na kutupa `fun!`.

Tuseme unataka kutoa sehemu kutoka katikati ya mfuatano wa herufi. Unaweza kutoa vielezo vya kuanzia na kumalizia ili kudhibiti kwa usahihi sehemu gani ya mfuatano unayotaka:

:::interactive_editor

```js
let message = "I love JavaScript!";
let language = message.slice(7, 17);

console.log(language);  // JavaScript
```

:::

Hapa, `slice(7, 17)` hutoka mfuatano mdogo kuanzia kielezo 7 na kumalizika kabla ya kielezo `17`, ambacho ni neno `JavaScript`.

Njia ya `slice()` ni chombo chenye nguvu cha kutoa sehemu za mfuatano wa herufi katika JavaScript.

Unaeleza vielezo vya kuanzia na kumalizia, na njia hiyo hurudisha mfuatano mpya unaoonyesha sehemu iliyotolewa.

Kwa chaguzi za vielezo chanya, hasi, na visivyotolewa, unaweza kuibadilisha kwa hali mbalimbali bila kubadilisha mfuatano wa asili.
