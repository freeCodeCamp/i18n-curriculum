### --question--

```js

console.log(hello);
```

#### --distractors--

`" world"`를 변수 `hello`에 재할당하고 콘솔에 출력합니다.

---

`Hello world`의 문자 수를 콘솔에 출력합니다.

---

#### --answer--

현재 `hello` 값에 문자열 `" world"`을 결합해 `hello`에 재할당하고 콘솔에 출력합니다.
