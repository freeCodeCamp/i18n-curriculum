## Lavorare con i loop

- **`for` Loop**: Questo tipo di loop si usa per ripetere un blocco di codice un certo numero di volte. Questo loop è suddiviso in tre parti: l’istruzione di inizializzazione, la condizione e l’istruzione di incremento/decremento. L’istruzione di inizializzazione viene eseguita prima che il loop inizi. Di solito serve per inizializzare una variabile contatore. La condizione viene valutata prima di ogni iterazione del loop. Un’iterazione è un singolo passaggio attraverso il loop. Se la condizione è `true`, il blocco di codice all’interno del loop viene eseguito. Se la condizione è `false`, il loop si ferma e si passa al blocco di codice successivo. L’istruzione di incremento/decremento viene eseguita dopo ogni iterazione del loop. Di solito serve per incrementare o decrementare la variabile contatore.

:::interactive_editor

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

:::

- **`for...of` Loop**: Questo tipo di loop si usa quando devi iterare su valori provenienti da un iterabile. Esempi di iterabili sono array e stringhe.

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

:::

- **`for...in` Loop**: Questo tipo di loop è più adatto quando devi iterare sulle proprietà di un oggetto. Questo loop itera su tutte le proprietà enumerabili di un oggetto, incluse quelle ereditate e quelle non numeriche.

:::interactive_editor

```js
const fruit = {
  name: 'apple',
  color: 'red',
  price: 0.99
};

for (const prop in fruit) {
  console.log(fruit[prop]);
}
```

:::

- **`while` Loop**: Questo tipo di loop esegue un blocco di codice finché la condizione è `true`.

:::interactive_editor

```js
let i = 5;

while (i > 0) {
  console.log(i);
  i--;
}
```

:::

- **`do...while` Loop**: Questo tipo di loop esegue il blocco di codice almeno una volta prima di verificare la condizione.

```js
let userInput;

do {
  userInput = prompt("Please enter a number between 1 and 10");
} while (Number(userInput) < 1 || Number(userInput) > 10);

alert("You entered a valid number!");
```

## Istruzioni `break` e `continue`

- **Definizione**: Un’istruzione `break` serve per uscire anticipatamente da un loop, mentre un’istruzione `continue` serve per saltare l’iterazione corrente di un loop e passare a quella successiva.

:::interactive_editor

```js
// Example of break statement
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, and 4

// Example of continue statement 
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, 4, 6, 7, 8, and 9
```
