# --interactive--

CSS 的 `filter` 属性是一个强大的工具，允许你为网页上的元素应用图形效果。它对于调整图像、背景甚至文本的视觉呈现特别有用，而无需修改原始资源。

`filter` 属性可用于创建各种效果，如模糊、色彩偏移和对比度调整。 让我们来讨论一下滤镜属性是如何工作的，并探讨一些常见的例子。 `filter` 属性的基本语法非常简单：

```css
selector {
  filter: function(amount);
}
```

这里，`function` 表现你想要应用的特定过滤器效果，`amount` 通常是决定效果强度的数值。现在让我们来看一些常见的过滤器函数及其用法。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: blur(2px);
}
```

:::

`blur` 函数为元素应用高斯模糊，模糊量以像素为单位，表现为模糊的半径。此 CSS 规则将为页面上的所有图像应用 2 像素模糊。`blur` 效果对于在你的设计中创建深度或模糊图像的部分内容非常有用。 

`brightness` 函数用于调整元素的亮度。 `0%` 的值将使元素完全变黑，而 `100%` 以上的值会增加亮度。 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: brightness(150%);
}
```

:::

此 CSS 规则将图像元素的亮度提高 `50%`。亮度调整可用于使图像更突出或创建洗白效果。

`grayscale` 函数将元素转换为灰度。 数量定义为百分比，`100%` 表示完全灰度，`0%` 则使图像保持不变。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: grayscale(100%);
}
```

:::

此规则将把图像元素转换为全灰度。`grayscale` 可用于创建复古效果或弱化页面上的某些元素。

`sepia` 函数对元素应用棕褐色调。 与灰度一样，它也使用百分比值： 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: sepia(80%);
}
```

:::

此规则对图像元素应用 `80%` 的棕褐色效果。棕褐色效果非常适合营造复古或怀旧的外观。

`hue-rotate` 函数对元素应用色相旋转。该值以度为单位定义，表现为围绕颜色圆的旋转。 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: hue-rotate(90deg);
}
```

:::

此规则将图像元素的色相旋转 `90` 度。色相旋转可用于创建迷幻效果或调整图像的整体配色方案。

`filter` 属性最强大的功能之一是可以组合多种效果。 你可以对同一元素应用多个滤镜，方法是用空格分隔： 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {  
  filter: contrast(150%) brightness(110%) sepia(30%);  
}
```

:::

此规则对图像元素应用增强的对比度、略微增加的亮度和细微的棕褐色效果。 

通过组合过滤器，你可以创建复杂且独特的视觉效果，以满足你的设计需求。CSS `filter` 属性是一个多功能工具，允许对网页元素进行创造性的视觉操作。 

虽然我们已经介绍了一些最常见的过滤器函数，但还有其他可用的，如 `contrast`、`invert` 和 `saturate`。正如任何强大的特色一样，谨慎使用过滤器以增强你的设计，同时不让用户感到不适或影响无障碍非常重要。
