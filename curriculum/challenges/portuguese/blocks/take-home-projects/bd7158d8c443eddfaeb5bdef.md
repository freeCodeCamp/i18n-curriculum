# --description--

Construa um aplicativo JavaScript full stack que seja funcionalmente semelhante a este: <a href="https://voting-app.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://voting-app.freecodecamp.rocks/</a>.Use um criador de sites de sua escolha para completar o projeto.

Aqui estão as histórias de usuário que você deve implementar para este projeto:

**História do Usuário:** Como um usuário autenticado, você pode salvar minhas enquetes e voltar depois para acessá-las.

**História do Usuário:** Como um usuário autenticado, você pode compartilhar minhas enquetes com meus amigos.

**História do Usuário:** Como um usuário autenticado, você pode ver os resultados agregados das minhas enquetes.

**História do Usuário:** Como um usuário autenticado, você pode deletar enquetes que eu decida que não quero mais.

**História do Usuário:** Como um usuário autenticado, você pode criar uma enquete com qualquer número de itens possíveis.

**História do Usuário:** Como um usuário não autenticado ou autenticado, você pode ver e votar nas enquetes de todos.

**História do Usuário:** Como um usuário não autenticado ou autenticado, você pode ver os resultados das enquetes em forma de gráfico. (Isso pode ser implementado usando `Chart.js` ou `Google Charts`.)

**História do Usuário:** Como um usuário autenticado, se você não gostar das opções em uma enquete, você pode criar uma nova opção.

Quando terminar, certifique-se de que uma demonstração funcional do seu projeto está hospedada em algum lugar público.Em seguida, envie o URL para a solução no campo Solution Link.Como opção, envie também um link para o código-fonte do projeto no campo GitHub Link.
