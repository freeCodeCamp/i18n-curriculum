# --description--

Злодій грабує м’ясний магазин, де він може вибирати з кількох елементів.

Злодій знає вагу та ціну кожного елемента. Оскільки у нього є рюкзак з обмеженням на максимальну вагу, яку він може нести, він хоче вибрати елементи так, щоб максимізувати свій прибуток. Він може розрізати елементи; у елемента після розрізання ціна скорочена пропорційно до оригінальної ціни за співвідношенням мас. Це означає: половина елемента має половину ціни оригіналу.

# --instructions--

Напишіть функцію, яка приймає масив об’єктів, що представляють елементи, доступні в магазині. Кожен об’єкт має 3 атрибути: name, weight і value. Функція також приймає максимальну вагу як параметр. Функція повинна повернути максимальне можливе значення, а загальна вага вибраних елементів не повинна перевищувати максимальну вагу.

```js
function knapContinuous(items, maxweight) {
  function item_cmp(a, b) {
    const ua = a.unitVal,
      ub = b.unitVal;
    return ua < ub ? 1 : ua > ub ? -1 : 0;
  }
  items = items.map(({ value, weight }) => ({
    unitVal: value / weight,
    weight
  }));
  items.sort(item_cmp);

  let val = 0;
  let wt = 0;
  for (let { unitVal, weight } of items) {
    var portion = Math.min(maxweight - wt, weight);
    wt += portion;
    var addVal = portion * unitVal;
    val += addVal;
    if (wt >= maxweight) {
      break;
    }
  }
  return val;
}
```
