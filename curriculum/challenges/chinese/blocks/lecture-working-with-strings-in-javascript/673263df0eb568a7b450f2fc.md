# --interactive--

在 JavaScript 中处理字符串时，有时需要插入特殊字符，以避免 JavaScript 引擎错误解析它们。

两个常见操作涉及在字符串中创建换行符和转义字符（如引号），以确保它们正确显示。

在包括 JavaScript 在内的许多编程语言中，都可以使用一种称为转义序列的特殊字符在字符串中创建换行。 换行符最常用的转义序列是 `\n` 。

例如，如果你想将一个字符串分成多行，可以在希望新行开始的地方使用 `\n`：

:::interactive_editor

```js
let poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

`\n` 转义序列会告诉 JavaScript 在该处插入换行符，从而使字符串跨行显示。

处理字符串时的另一个重要概念是转义字符。 有时，你需要在字符串中包含 JavaScript 通常用于其他用途的字符，例如引号。

如果只在字符串内使用引号而不转义，可能会导致错误，因为 JavaScript 会认为你试图结束字符串。

例如，这会导致错误：

```js
let statement = "She said, "Hello!"";
```

JavaScript 会感到困惑，因为它认为字符串在 `"said,"` 之后结束，但是，你希望 `"Hello!"` 周围的引号是字符串的一部分。

要解决这个问题，可以在内部引号前加一个反斜杠（`\`）来转义内部引号：

:::interactive_editor

```js
let statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

反斜杠告诉 JavaScript 将引号视为字面字符，因此它们会正确地显示在输出中。

你还可以转义其他特殊字符，例如反斜杠本身（`\\`），或单引号包围的字符串内的单引号（`\'`）。

下面是另一个使用单引号的例子：

:::interactive_editor

```js
let quote = 'It\'s a beautiful day!';
console.log(quote); // It's a beautiful day!
```

:::

通过用 `\'` 转义单引号，JavaScript 知道要将其作为字符串的一部分，而不是提前结束字符串。

在格式化输出或处理字符串中的特殊字符时，换行和创建换行符是必不可少的。 这些技巧可以帮助你防止错误，确保你的文本完全按照原意显示。
