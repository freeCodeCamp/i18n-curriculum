# --explanation--

`True` — це прикметник, який означає, що щось є правильним, справжнім або точним. Його використовують, щоб погодитися з твердженням або фактом. Це відрізняється від `the truth`, що є іменником і позначає фактичні дані або реальність про щось, часто в ширшому або більш абстрактному сенсі. Наприклад:

- `It's true that the server crashed yesterday.` – Це означає, що твердження про аварію є правильним.

- `The truth is that we didn't have a backup.` – Це означає, що фактично запасного не існувало.
