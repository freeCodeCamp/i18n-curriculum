# --interactive--

Unapoanza kujifunza JavaScript, utagundua haraka kwamba kufanya hesabu za kihesabu ni zoezi la kawaida. Ingawa vibonye vya hesabu vya msingi vinaweza kushughulikia mahesabu rahisi, JavaScript inatoa kitu cha `Math` kilichojengwa ndani kushughulikia changamoto za hesabu ngumu zaidi.

Chombo hiki cha msaada kinatoa njia mbalimbali zinazorahisisha kufanya mahesabu ya hali ya juu na kudhibiti nambari. Hebu tuchunguze njia hizi na kuona jinsi zinavyoweza kurahisisha uzoefu wako wa kuandika msimbo.

Njia ya `Math.random()` hutengeneza nambari ya kubeba isiyo na mpangilio kati ya `0` (ikiwa ni pamoja) na `1` (isipokuwa). Hii inamaanisha matokeo yanayowezekana yanaweza kuwa `0`, lakini hayatafikia kabisa `1`. Hapa kuna mfano wa kutumia njia ya `Math.random()`:

:::interactive_editor

```js
const randomNum = Math.random();

console.log(randomNum);
// any number between 0 and 1 – 0 inclusive and 1 exclusive
```

:::

`Math.min()` na `Math.max()` zote huchukua seti ya nambari na kurudisha thamani ya kiwango cha chini na cha juu, mtawalia. Hapa kuna mfano wa kutumia njia zote mbili:

:::interactive_editor

```js
const smallest = Math.min(1, 5, 3, 9);
console.log(smallest); // 1

const largest = Math.max(1, 5, 3, 9);
console.log(largest); // 9
```

:::

`console.log()` ya kwanza itaandika nambari `1`, kwa sababu `1` ndilo ndogo zaidi katika orodha hiyo ya nambari. Na `console.log()` ya pili itaandika nambari `9`, kwa sababu `9` ndilo kubwa zaidi katika orodha hiyo ya nambari.

Ikiwa ungependa kuzungusha nambari juu au chini hadi nambari kamili iliyo karibu zaidi, unaweza kutumia njia za `Math.ceil()` na `Math.floor()`. Hapa kuna mfano wa kutumia `Math.ceil()`:

:::interactive_editor

```js
console.log(Math.ceil(4.3)); // 5
```

:::

`Math.ceil()` itazungusha `4.3` juu hadi nambari kamili iliyo karibu zaidi, ambayo ni `5` katika kesi hii. Sasa, tuchunguze kuzungusha nambari chini:

:::interactive_editor

```js
console.log(Math.floor(4.7)); // 4
```

:::

`Math.floor()` itazungusha `4.7` chini hadi nambari kamili iliyo karibu zaidi, ambayo ni `4` katika kesi hii. `Math.round()` ni mchanganyiko wa `Math.ceil()` na `Math.floor()`. Inazungusha nambari hadi nambari kamili iliyo karibu, ikizingatia sehemu ya desimali:

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

Hivyo, ikiwa sehemu ya desimali ni chini ya `5`, nambari huzungushwa chini. Na ikiwa sehemu ya desimali ni `5` au zaidi, nambari huzungushwa juu. Matumizi ya vitendo ya `Math.floor()` na `Math.random()` ni kutengeneza nambari ya kubeba isiyo na mpangilio kati ya nambari mbili kamili. Hii ndiyo sintaksia yake:

:::interactive_editor

```js
const max = 10;
const min = 5;
const randomNum = Math.floor(Math.random() * (max - min + 1)) + min;
console.log(randomNum);
```

:::

Kutengeneza nambari ya kubeba isiyo na mpangilio kati ya `20` na `1` itakuwa kama hii:

:::interactive_editor

```js
const randomNumBtw1And20 = Math.floor(Math.random() * 20) + 1;
console.log(randomNumBtw1And20);
```

:::

Njia nyingine ya msaada ya `Math` ni njia ya `Math.trunc()`. `Math.trunc()` huondoa sehemu ya desimali ya nambari, ikirudisha sehemu ya nambari kamili tu, bila kuzungusha:

:::interactive_editor

```js
console.log(Math.trunc(2.9)); // 2
console.log(Math.trunc(9.1)); // 9
```

:::

Ikiwa unahitaji kupata mzizi wa mraba au mzizi wa duara wa nambari, unaweza kutumia njia za `Math.sqrt()` na `Math.cbrt()`, mtawalia:

:::interactive_editor

```js
console.log(Math.sqrt(81)); // 9
console.log(Math.cbrt(27)); // 3
```

:::

Kauli ya kwanza ya kuandika, itaandika `9` kwa sababu mzizi wa mraba wa `81` ni `9`, wakati kauli ya pili ya kuandika itaandika `3` kwa sababu mzizi wa duara wa `27` ni `3`. Ikiwa unahitaji kupata thamani kamili ya nambari, unaweza kutumia njia ya `Math.abs()`:

:::interactive_editor

```js
console.log(Math.abs(-5)); // 5
console.log(Math.abs(5)); // 5
```

:::

`Math.abs()` hurudisha thamani kamili ya nambari, ikibadilisha hasi kuwa chanya. Njia ya mwisho tutakayochunguza ni njia ya `Math.pow()`:

:::interactive_editor

```js
console.log(Math.pow(2, 3)); // 8
console.log(Math.pow(8, 2)); // 64
```

:::

`Math.pow()` huchukua nambari mbili na kuinua ya kwanza kwa nguvu ya ya pili. Kuna njia nyingi zaidi zinazomilikiwa na kitu cha `Math`, ambazo unaweza kuchunguza mwenyewe. Hata hivyo, hizi ni chache tu kati ya njia zinazotumika zaidi katika misimbo ya JavaScript.
