# --description--

$n$のラジカル、$rad(n)$は、$n$の異なる素因数の積です。例えば、$504 = 2^3 × 3^2 × 7$なので、$rad(504) = 2 × 3 × 7 = 42$です。

正の整数の三つ組$(a, b, c)$をabcヒットと定義します。条件は以下の通りです：

1. $GCD(a, b) = GCD(a, c) = GCD(b, c) = 1$
2. $a &lt; b$
3. $a + b = c$
4. $rad(abc) &lt; c$

例えば、$(5, 27, 32)$はabcヒットです。なぜなら：

1. $GCD(5, 27) = GCD(5, 32) = GCD(27, 32) = 1$
2. $5 &lt; 27$
3. $5 + 27 = 32$
4. $rad(4320) = 30 &lt; 32$

abcヒットは非常にまれで、$c &lt; 1000$の場合、abcヒットは31個だけ存在し、$\sum{c} = 12523$です。

$c &lt; 120000$の場合の$\sum{c}$を求めてください。
