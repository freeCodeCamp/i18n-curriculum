# --description--

組件是 React 應用程式的建造區塊，允許開發者將複雜的使用者介面拆解成較小且易於管理的部分，使得開發和維護大型應用程式更加容易。

React 中的兩種類型組件是函式組件和類別組件。在現代 React 中，開發者會使用函式組件，而我們今天看到的所有範例都將是函式組件。

在較高層次上，你可以將組件視為傳回描述 UI 元素的 JavaScript 函式。

此使用者介面是使用 JSX 描述的，JSX 是 JavaScript 的一種語法擴充套件，看起來類似 HTML，但允許你以更宣告式的方式撰寫使用者介面元素。

讓我們來看一個 React 組件的範例：

```jsx
function Greeting() {
  const name = "John"
  {/* The result will be Hello John*/}
  return <h1 className="title">Hello {name}</h1>;
}
```

在這個範例中，我們定義了一個名為 `Greeting` 的組件。`h1` 頁籤內的花括號 `{}` 讓我們能夠嵌入 JavaScript 表達式，從而在 `h1` 元素中存取 `name` 變數。

我們也將一個名為 `title` 的 `className` 套用到 `h1` 元素。

但是為什麼我們使用 `className` 而不是像一般 HTML 元素那樣使用 `class`？

這是因為在 JavaScript 中，`class` 是保留名稱。所以，我們需要改用 `className`。

我們也在 JSX 中使用註解來顯示結果會是什麼。重要的是要注意，你可以使用像這樣的常規註解語法，但必須用大括號包起來才能使其生效：

```jsx
{/* Block Comments */}
```

你可能還注意到另一件事是我們在組件名稱的開頭使用了大寫字母。但為什麼我們不能全部使用小寫字母？

這是因為 React 將大寫字母開頭的組件視為訂製組件，而小寫字母開頭的元素則被視為內建的 HTML 元素。

當 React 遇到小寫的頁籤，例如 `<div>` 或 `<span>`，它會假設這是標準的 HTML 元素。然而，如果組件名稱以大寫字母開頭，React 會將其視為使用者定義的組件並相應地呈現。這種區分有助於 React 區分原生的 HTML 頁籤和你創建的組件。

要在我們的應用程式中使用這個 `Greeting` 組件，我們會這樣寫：

```jsx
<Greeting />
```

這會將一個帶有文字 `Hello John` 的 `h1` 元素呈現到頁面上。但請仔細看看這裡的語法。當我們使用這個組件時，它以斜線結尾，然後是大於符號。

在使用 JSX 時，所有頁籤和組件的使用都需要明確地關閉。因此，如果組件或頁籤沒有任何子元素，則你需要像這裡所示那樣明確地關閉它：

```jsx
<Greeting /> {/* /> is required */}
```

到目前為止，我們只看過如何呈現單一的 `h1` 元素。但你實際上可以呈現多個元素。

讓我們來看看以下的範例程式碼：

```jsx
function Greeting() {
  const name = "John";
  {/* This will throw an error */}
  return <h1>Hello {name}</h1> 
  <p>Nice to meet you.</p>
}
```

我們正嘗試為 `Nice to meet you` 添加另一個句子，但它未能正確地在頁面上呈現。

似乎出現了錯誤訊息。錯誤訊息顯示 `Adjacent JSX elements must be wrapped in an enclosing tag.`

你收到該錯誤訊息的原因是因為多個同層元素需要包裹在一個父元素中。雖然你可以用簡單的 `div` 包裹 `h1` 和 `p` 元素，但還有另一種方法可以消除該錯誤。

React 片段用於將元素群組在一起。以下是修訂後的範例外觀：

```jsx
function Greeting() {
  const name = "John";
  return (
    <Fragment>
      <h1>Hello {name}</h1>
      <p>Nice to meet you.</p>
    </Fragment>
  );
}
```

你也可以選擇使用空的 JSX 頁籤，這可以作為片段的簡寫：

```jsx
function Greeting() {
  const name = "John";
  return (
    <>
      <h1>Hello {name}</h1>
      <p>Nice to meet you.</p>
    </>
  );
}
```
