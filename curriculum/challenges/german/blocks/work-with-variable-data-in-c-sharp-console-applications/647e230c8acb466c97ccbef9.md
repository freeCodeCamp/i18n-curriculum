## --text--

Warum ist es wichtig, die `String.Trim()`-Methode auf jedes Element eines String-Arrays anzuwenden, bevor Sie das Array sortieren?

## --answers--

Nur String-Arrays mit Elementen, die mit `String.Trim()` getrimmt wurden, können sortiert werden.

---

`String.Trim()` entfernt führende Leerzeichen, die vor Zahlen und Buchstaben sortiert würden.

---

Das Sortieren des Arrays ist deutlich schneller, wenn `String.Trim()` auf jedes Element angewendet wird.
