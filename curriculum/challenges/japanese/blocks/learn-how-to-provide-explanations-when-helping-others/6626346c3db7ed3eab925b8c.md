# --description--

日常生活では、`right` と `left` は方向や側面を指します。たとえば、`right` を回すと、`right` 側に動きます。

コンピュータのマウスでは、`right click` と `left click` は通常異なる操作のために `right` または `left` ボタンを押すことを指します。`right click` は追加のオプションやメニューを開くことが多く、`left click` は項目の選択やナビゲーションに使われます。

# --questions--

## --text--

マウスの `right click` と `left click` の違いは何ですか？

## --answers--

`Right click` はナビゲーション用で、`left click` はより多くのオプション用です。
