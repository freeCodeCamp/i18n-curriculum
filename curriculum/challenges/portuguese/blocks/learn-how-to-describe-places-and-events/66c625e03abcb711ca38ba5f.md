# --explanation--

`This is` é uma maneira simples de começar uma frase ao descrever algo.

`Frustrating` é um adjetivo que descreve uma sensação de estar chateado ou irritado porque algo não está funcionando como esperado.Por exemplo:

- `This is confusing.` - A situação é difícil de entender.

- `This is exciting.` - A situação faz você se sentir animado.
