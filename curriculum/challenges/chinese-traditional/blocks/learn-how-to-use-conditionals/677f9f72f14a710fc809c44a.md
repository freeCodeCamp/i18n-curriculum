# --explanation--

Mark 提到 `incompatible drivers` 和 `outdated software` 是軟體當機的原因。

`Incompatible drivers` 是指與硬體不相容且無法正常運作的驅動程式，而 `outdated software` 則表示軟體尚未更新到最新版本。兩者都可能導致當機。<dfn>例如</dfn>：

- `The graphics card isn't working because of incompatible drivers.` - 錯誤的驅動程式導致顯示卡故障。

- `Outdated software can lead to security vulnerabilities.`－舊的程式可能因缺乏更新而不安全。
