# --description--

類型選擇器（或元素選擇器）將選擇給定元素類型的所有元素，其語法是：元素名 { css 代碼 }：

```html
<!-- index.html -->

<div>Hello, World!</div>
<div>Hello again!</div>
<p>Hi...</p>
<div>Okay, bye.</div>
```

```css
/* styles.css */

div {
  color: white;
}
```

Here, all three `<div>` elements would be selected, while the `<p>` element wouldn’t be.

# 問題

## --text--

以下哪一項最能描述上面給出的CSS代碼？

## --answers--

該代碼`white`將白色應用於HTML文件中的所有元素。
