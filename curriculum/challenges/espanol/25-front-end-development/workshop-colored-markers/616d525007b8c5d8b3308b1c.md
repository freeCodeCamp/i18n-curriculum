# --description--

Tu marcador se vería mejor si estuviera centrado en la página. Una manera fácil de hacer esto es utilizando la `propiedad abreviada` `margin`.

Puedes establecer el área de margen de los elementos por separado con propiedades como `margin-top` y `margin-left`, la `propiedad abreviada` `margin` facilita establecer múltiples áreas de margen al mismo tiempo.

Para centrar tu marcador en la página, establece su `propiedad` `margin` en `auto`. Esto establece `margin-top`, `margin-right`, `margin-bottom`, y `margin-left` todos en `auto`.

# --hints--

Tu regla CSS `.marker` debe tener una `propiedad` `margin` configurada en `auto`.

```js
assert.strictEqual(new __helpers.CSSHelp(document).getStyle('.marker')?.margin, 'auto');
```
