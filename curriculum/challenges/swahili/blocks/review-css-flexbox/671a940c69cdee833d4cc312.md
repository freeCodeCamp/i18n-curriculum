## Utangulizi wa CSS Flexbox na Mfano wa Flex

- **Ufafanuzi**: CSS flexbox ni mfano wa mpangilio wa kipengele wa mwelekeo mmoja unaoruhusu kupanga vipengele katika safu na nguzo ndani ya kontena.
- **Mfano wa Flex**: Mfano huu unaelezea jinsi vipande vya flex vinavyopangwa ndani ya chombo cha flex. Kila chombo cha flex kina mihimili miwili: mhimili mkuu na mhimili wa msalaba.

## Sifa ya `flex-direction`

- **Ufafanuzi**: Sifa hii inaweka mwelekeo wa mhimili mkuu. Thamani ya chaguo-msingi ya `flex-direction` ni `row`, ambayo huweka vipande vyote vya flex kwenye safu moja, kwa mwelekeo wa lugha ya kivinjari chako (kutoka kushoto kwenda kulia au kulia kwenda kushoto).
- **`flex-direction: row-reverse;`**: Hii hubadilisha mpangilio wa vipande kwenye safu.
- **`flex-direction: column;`**: Hii itaweka vipande vya flex wakiwa wamesimama wima badala ya usawa.
- **`flex-direction: column-reverse;`**: Hii hubadilisha mpangilio wa vipande vya flex wima.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: row;
  gap: 10px;
}

.box {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}
```

:::

## Sifa ya `flex-wrap`

- **Ufafanuzi**: Sifa hii inaamua jinsi vipande vya flex vinavyobadilika ndani ya chombo cha flex ili kufaa nafasi inayopatikana. `flex-wrap` inaweza kuchukua thamani tatu: `nowrap`, `wrap`, na `wrap-reverse`.
- **`flex-wrap: nowrap;`**: Hii ni thamani ya chaguo-msingi. Vipande vya flex havitazungushwa kwenye mstari mpya, hata kama upana wao unazidi upana wa kontena.
- **`flex-wrap: wrap;`**: Sifa hii itazungusha vipande wakati vinapozidi upana wa kontena lao.
- **`flex-wrap: wrap-reverse;`**: Sifa hii itazungusha vipande vya flex kwa mpangilio wa nyuma.
- **Sifa ya `flex-flow`**: Sifa hii ni ufupisho wa `flex-direction` na `flex-wrap`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
  <div class="box">4</div>
  <div class="box">5</div>
</div>
```

```css
.container {
  display: flex;
  flex-wrap: wrap;
  width: 150px;
  background: #f0f0f0;
}

.box {
  width: 60px;
  padding: 10px;
  margin: 5px;
  background: skyblue;
  text-align: center;
}
```

:::

## Sifa ya `justify-content`

- **Ufafanuzi**: Sifa hii huoanisha vipengele vidogo kando ya mhimili mkuu wa chombo cha kubadilika.
- **`justify-content: flex-start;`**: Katika kesi hii, vipande vya kubadilika vitaoanishwa mwanzoni mwa mhimili mkuu. Hii inaweza kuwa usawa au wima.
- **`justify-content: flex-end;`**: Katika kesi hii, vipande vya kubadilika vinaoanishwa mwishoni mwa mhimili mkuu, usawa au wima.
- **`justify-content: center;`**: Hii huweka vipande vya kubadilika katikati kando ya mhimili mkuu.
- **`justify-content: space-between;`**: Hii itasambaza vipengele kwa usawa kando ya mhimili mkuu.
- **`justify-content: space-around;`**: Hii itasambaza vipande vya kubadilika kwa usawa ndani ya mhimili mkuu, ikiongeza nafasi kabla ya kipande cha kwanza na baada ya kipande cha mwisho.
- **`justify-content: space-evenly;`**: Hii itasambaza vipande kwa usawa kando ya mhimili mkuu.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box">A</div>
  <div class="box">B</div>
  <div class="box">C</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
  background: #eee;
}

.box {
  padding: 20px;
  background: salmon;
}
```

:::
  
## Sifa ya `align-items`

- **Ufafanuzi**: Sifa hii hutumika kusambaza vipengele kwenye mhimili wa msalaba. Kumbuka mhimili wa msalaba ni wima kwa mhimili mkuu.
- **`align-items: center;`**: Hii hutumika kuweka vipande katikati ya mhimili wa msalaba.
- **`align-items: flex-start;`**: Hii inaweka vipande kuanzia mwanzo wa mhimili wa msalaba.
- **`align-items: stretch;`**: Hii hutumika kunyoosha vipande vya flex kwenye mhimili wa msalaba.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box tall">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  align-items: center;
  height: 150px;
  background: #ddd;
}

.box {
  background: lightgreen;
  padding: 10px;
}

.tall {
  height: 100px;
}
```
