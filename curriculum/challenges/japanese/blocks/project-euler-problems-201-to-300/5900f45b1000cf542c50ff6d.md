# --description--

「Blum Blum Shub」疑似乱数生成器を使って数のシーケンスを作成してください：

$$
s_0 = 14025256 \\\\
s_{n + 1} = {s_n}^2 \\; mod \\; 20\\,300\\,713
$$

これらの数 $s_0s_1s_2\ldots$ を連結して、無限長の文字列 $w$ を作成します。つまり、$w = 14025256741014958470038053646\ldots$ となります。

正の整数 $k$ に対して、$w$ の部分文字列で桁の和が $k$ となるものが存在しない場合、$p(k)$ はゼロと定義します。もし $w$ の部分文字列で桁の和が $k$ となるものが少なくとも1つ存在する場合、$p(k) = z$ と定義し、$z$ はそのような部分文字列の中で最も早く始まる位置です。

例えば：

部分文字列 1、14、1402、… はそれぞれ桁の和が 1、5、7、… で、位置1から始まるため、$p(1) = p(5) = p(7) = \ldots = 1$ となります。

部分文字列 4、402、4025、… はそれぞれ桁の和が 4、6、11、… で、位置2から始まるため、$p(4) = p(6) = p(11) = \ldots = 2$ となります。

部分文字列 02、0252、… はそれぞれ桁の和が 2、9、… で、位置3から始まるため、$p(2) = p(9) = \ldots = 3$ となります。

位置3から始まる部分文字列 025 の桁の和は7ですが、位置1から始まる桁の和が7の部分文字列の方が早いため、$p(7) = 1$ であり、3ではありません。

$0 < k \leq {10}^3$ の場合、$\sum p(k) = 4742$ であることが確認できます。

$0 < k \leq 2 \times {10}^{15}$ のときの $\sum p(k)$ を求めてください。
