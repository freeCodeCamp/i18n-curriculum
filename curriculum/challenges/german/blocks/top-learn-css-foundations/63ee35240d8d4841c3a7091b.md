# --description--

Klassen-Selektoren wählen alle Elemente mit dem angegebenen `class` aus, das einfach ein Attribut ist, das Sie an einem HTML-Element anbringen. So fügen Sie einem HTML-Tag eine Klasse hinzu und wählen sie in CSS aus:

```html
<!-- index.html -->

<div class="alert-text">
  Please agree to our terms of service.
</div>
```

```css
/* styles.css */

.alert-text {
  color: red;
}
```

Beachten Sie die Syntax für `class`-Selektoren: ein Punkt, unmittelbar gefolgt vom case-sensitiven Wert des Klassen-Attributs. Klassen müssen nicht einzigartig sein, daher können Sie denselben `class` an so vielen Elementen verwenden, wie Sie möchten.

Eine weitere Möglichkeit mit dem `class`-Attribut ist, mehreren Klassen zu einem einzelnen Element als durch Leerzeichen getrennte Liste hinzuzufügen, wie `class="alert-text severe-alert"`. Da Leerzeichen zur Trennung von `class`-Namen verwendet werden, sollten Sie niemals Leerzeichen für mehrteilige Namen verwenden, sondern stattdessen einen Bindestrich.

## ID-Selektoren
ID-Selektoren sind ähnlich wie `class`-Selektoren. Sie wählen ein Element mit dem angegebenen `id` aus, das ein weiteres Attribut ist, das Sie an einem HTML-Element anbringen:

```html
<!-- index.html -->

<div id="title">My Awesome 90's Page</div>
```

```css
/* styles.css */

#title {
  background-color: red;
}
```

Anstelle eines Punkts verwenden Sie ein Hashtag, unmittelbar gefolgt vom case-sensitiven Wert des `id`-Attributs. Ein häufiger Fehler ist, dass Leute das `id`-Attribut übermäßig verwenden, obwohl Klassen ausreichen würden. Es gibt zwar Fälle, in denen die Verwendung einer `id` sinnvoll oder notwendig ist, zum Beispiel um von der Spezifität zu profitieren oder Links auf einen Abschnitt der aktuellen Seite umzuleiten, aber Sie sollten `id`s sparsam (wenn überhaupt) verwenden.

Der wesentliche Unterschied zwischen Klassen und IDs ist, dass ein Element nur eine `id` haben kann. Eine `id` darf auf einer einzelnen Seite nicht mehrfach vorkommen, und das `id`-Attribut darf keinerlei Leerzeichen enthalten.

Wie lautet die Syntax für Klassen- und ID-Selektoren?

## --answers--

Um eine `class` auszuwählen, verwenden Sie `$`, und um eine `id` auszuwählen, verwenden Sie `#`

---

Um eine `class` auszuwählen, verwenden Sie `.`, und um eine `id` auszuwählen, verwenden Sie `*`
