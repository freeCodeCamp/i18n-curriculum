# --description--

`which` という言葉は、既知の選択肢の中から特定の情報を尋ねるときに使われます。たとえば、誰かが `Which book did you read?` と尋ねる場合、複数の可能な本の中から特定の本を知りたいということです。

技術的な文脈では、`Which software are you using?` と尋ねることは、既知の選択肢の中から特定のソフトウェア名を尋ねることを意味します。
