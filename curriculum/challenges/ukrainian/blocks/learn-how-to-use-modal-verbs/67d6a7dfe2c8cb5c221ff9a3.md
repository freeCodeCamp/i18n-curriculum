# --explanation--

`High traffic` означає велику кількість користувачів, які одночасно отримують доступ до системи. Наприклад:

- `We need a stronger server to handle high traffic.` - Кращий сервер допоможе, коли багато користувачів відвідують сайт.

- `During sales, online stores experience high traffic.` - Багато людей заходять на сайти магазинів під час знижок.

Софі питає, чи можуть сервери не впоратися, коли надто багато користувачів спробують отримати доступ до системи після релізу.
