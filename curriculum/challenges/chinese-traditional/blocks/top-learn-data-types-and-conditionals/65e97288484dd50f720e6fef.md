# --description--

較複雜的條件述語可能包含多個條件。這就是為什麼使用邏輯運算子來結合多個條件。邏輯運算子是 `&&`、`||` 和 `!`，分別用來表述 `and`、`or` 和 `not`。

邏輯運算子 `||` 用於結合兩個布林值條件。若至少有一個條件為 `true`，它會傳回 `true`。否則，它會傳回 `false`：

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a > b || a > c) {
  console.log("At least one of the conditions is true");
} else {
  console.log("Both of the conditions are false");
}
```

邏輯運算子 `&&` 用於結合兩個布林值條件。只有當兩個條件皆為 `true` 時，它才會傳回 `true`。否則，它會傳回 `false`：

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a < b && a < c) {
  console.log("Both of the conditions are true");
} else {
  console.log("At least one of the conditions is false");
}
```

邏輯運算子 `!` 用於否定布林值條件。若條件為 `false`，它會傳回 `true`。否則，它會傳回 `false`：

```javascript
let a = 5;
let b = 10;

if (!(a > b)) {
  console.log("The condition is false");
} else {
  console.log("The condition is true");
}
```

在上述範例中，額外的小括弧用於讓程式碼更易讀。使用它們並非必要。

# --questions--

## --text--

你的任務是撰寫一個 `if` 述語，用於檢查網頁應用程式中的以下條件，以向使用者顯示 `Welcome!`：

1. 使用者必須擁有高級帳戶（`isPremium`）或成為成員超過 12 個月（`membershipDuration` > 12 個月）。

1. 使用者目前不得被服務阻擋（`!isBlocked`）。

哪個 if 述語正確檢查這些條件？

## --answers--

```javascript
if (isPremium && membershipDuration > 12 && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (isPremium || (membershipDuration > 12 && !isBlocked)) {
  console.log("Welcome!");
}
```

---

```javascript
if ((isPremium || membershipDuration > 12) && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (!isPremium || membershipDuration <= 12 || isBlocked) {
  console.log("Welcome!");
}
```
