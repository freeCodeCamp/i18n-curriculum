# --interactive--

Ключове слово `!important` у CSS використовується, щоб надати правилу стилю найвищий пріоритет, дозволяючи йому заміщати будь-які інші оголошення для властивості.

Коли його застосовують, це змушує браузер застосувати вказаний стиль, незалежно від специфічності інших селекторів.

Припустімо, у вас є HTML-елемент абзацу з рядковими стилями, як ось тут:

:::interactive_editor

```html
<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

:::

У цьому прикладі елемент абзацу має встановлений колір фону `lightblue` і колір тексту `black`.

CSS-файл застосовує такі стилі до елемента абзацу:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black;
  color: white;
}
```

:::

Оскільки рядкові стилі мають вищий пріоритет, ніж селектори класу, ідентифікатора та типу, чорний колір фону та білий колір тексту не будуть застосовані до цього елемента абзацу.

Щоб замістити ці рядкові стилі, ви можете використати ключове слово `!important` ось так:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black !important;
  color: white !important;
}
```

:::

Ключове слово `!important` використовується після значення CSS і перед крапкою з комою.

Тепер елемент абзацу матиме застосовані чорний і білий кольори, тому що рядкові стилі були заміщені за допомогою ключового слова `!important`.

Ключове слово `!important` у CSS використовується, щоб надати правилу стилю найвищий пріоритет, ефективно заміщаючи інші оголошення, включно з тими, що мають вищу специфічність і рядкові стилі.

Однак ключове слово `!important` не змінює специфічність самого селектора CSS. Воно просто гарантує, що правило з `!important` буде застосоване, навіть якщо існують інші конфліктні правила з вищою специфічністю.

Ще одним доречним випадком використання ключового слова `!important` є заміщення стилів із сторонніх бібліотек або фреймворків, коли ви не маєте контролю над оригінальним CSS.

Проте надмірне використання ключового слова `!important` може ускладнити підтримку та налагодження вашого CSS, оскільки порушує природний каскад стилів і може призвести до небажаних наслідків.

Тож найкраще використовувати ключове слово `!important` помірковано.
