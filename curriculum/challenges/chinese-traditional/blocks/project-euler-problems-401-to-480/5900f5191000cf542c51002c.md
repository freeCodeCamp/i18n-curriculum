# --description--

一個數字 $n$ 的單位因數 $d$ 是 $n$ 的一個因數，且具有屬性 $gcd(d, \frac{n}{d}) = 1$。

$4! = 24$ 的單位因數是 1、3、8 和 24。

它們平方的和是 $12 + 32 + 82 + 242 = 650$。

令 $S(n)$ 表述 $n$ 的單位因數平方和。因此 $S(4!) = 650$。

尋找 $S(100\,000\,000!)$ 模除 $1\,000\,000\,009$ 的結果。
