# --interactive--

Іноді вам потрібно буде маніпулювати стилями HTML-елемента у вашому JavaScript-файлі. Прикладом цього може бути приховування або показ меню, коли користувач натискає на кнопку.

У цьому уроці ви навчитеся маніпулювати стилями за допомогою властивостей `Element.style` та `Element.classList`.

Властивість `Element.style` є властивістю лише для читання, яка представляє рядковий стиль елемента. Ви можете використовувати цю властивість, щоб отримати або встановити стиль елемента.

Ось приклад абзацу з ідентифікатором `para`:

```html
<p id="para">This is a paragraph</p>
```

Ви можете використати властивість `style`, щоб змінити колір тексту абзацу на червоний:

:::interactive_editor

```html
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```js
const paraEl = document.getElementById("para");
paraEl.style.color = "red";
```

:::

Властивість `style` можна використовувати для встановлення багатьох CSS-властивостей, таких як `color`, `background-color`, `font-size`, `font-weight` тощо.

Інший спосіб маніпулювати стилями — це використання властивості `Element.classList`. Властивість `classList` є властивістю лише для читання, яка дозволяє додавати, видаляти або переключати класи на елементі. Розглянемо кілька прикладів.

У цьому прикладі ви додасте клас `highlight` до абзацу з ідентифікатором `para`. У CSS ми встановимо клас `highlight`, який змінить колір фону абзацу на жовтий. У JavaScript ми додамо клас `highlight` до абзацу за допомогою методу `classList.add()`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```css
.highlight {
  background-color: yellow;
}
```

```js
const paraEl = document.getElementById("para");
paraEl.classList.add("highlight");
```

:::

Ви також можете додати кілька класів до елемента, передаючи їх як аргументи методу `classList.add()`:

```js
classList.add("class1", "class2", "class3");
```

Якщо потрібно видалити клас з елемента, використовуйте метод `classList.remove()` так:

```js
classList.remove("highlight");
```

Щоб переключити клас на елементі, використовуйте метод `classList.toggle()`. У цьому прикладі у нас є елемент `button` з ідентифікатором `toggle-btn` і елемент `nav` з ідентифікатором `menu`. У CSS ми додали стилі для меню та клас `show`, який змінить властивість `display` елемента `.menu` на `block`. У JavaScript ми отримуємо доступ до елемента меню та елемента `button` за допомогою методу `getElementById()`. Потім додаємо слухача подій до елемента `button`, який переключатиме клас `show` на елементі `.menu` при натисканні кнопки:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button id="toggle-btn">Toggle Menu</button>

<nav id="menu" class="menu">
  <ul>
    <li>Home</li>
    <li>About</li>
    <li>Products</li>
  </ul>
</nav>
<script src="index.js"></script>
```

```css
.menu {
  display: none;
  background-color: lightgray;
  width: 50%;
  padding: 10px;
}

.menu.show {
  display: block;
}
```

```js
const menu = document.getElementById("menu");
const toggleBtn = document.getElementById("toggle-btn");

toggleBtn.addEventListener("click", () => menu.classList.toggle("show"));
```

:::

Коли користувач натискає на кнопку, клас `show` буде додано до елемента `.menu`, і меню відобразиться. Якщо користувач натисне кнопку знову, клас `show` буде видалено з елемента `.menu`, і меню сховається.

Використовуючи методи та властивості, такі як `style`, `add()`, `remove()` і `toggle()`, ви легко можете маніпулювати стилями елемента за допомогою JavaScript, створюючи динамічні та інтерактивні вебсторінки.
