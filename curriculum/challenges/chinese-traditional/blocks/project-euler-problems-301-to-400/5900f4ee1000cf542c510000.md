# --description--

對於平面上的任意三角形 $T$，可以證明存在唯一一個面積最大的簡略號，且該簡略號完全位於 $T$ 內。

<img alt="ellipse completely inside a triangle" src="https://cdn.freecodecamp.org/curriculum/project-euler/ellipses-inside-triangles.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

對於給定的 $n$，考慮三角形 $T$，使得：

-   $T$ 的頂點具有絕對值 ≤ n 的整數座標，且
-   最大面積簡略號內 $T$ 的焦點<sup>1</sup>為 $(\sqrt{13}, 0)$ 和 $(-\sqrt{13}, 0)$。

設 $A(n)$ 為所有此類三角形面積的總和。

例如，如果 $n = 8$，有兩個這樣的三角形。它們的頂點分別是 (-4,-3)、(-4,3)、(8,0) 和 (4,3)、(4,-3)、(-8,0)，且每個三角形的面積都是 36。因此 $A(8) = 36 + 36 = 72$。

可以驗證 $A(10) = 252$、$A(100) = 34\\,632$ 和 $A(1000) = 3\\,529\\,008$。

求 $A(1\,000\,000\,000)$。

<sup>1</sup>簡略號的焦點（focus 的複數）是兩個點 $A$ 和 $B$，使得對簡略號邊界上的每個點 $P$，$AP + PB$ 是常數。
