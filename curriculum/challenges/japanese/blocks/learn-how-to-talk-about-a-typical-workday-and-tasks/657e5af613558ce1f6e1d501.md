# --description--

`security incident` は、会社や組織の安全やセキュリティに害を及ぼす可能性があるイベントです。例えば、`A security incident like a stolen password needs to be reported immediately`。

`harm` と `harmful` に関する注意：

`Harm` は、誰かや何かに対する損害や傷害を意味します。これは、悪影響を与えたり、誰かや何かを危険にさらすことを指します。例えば：

`Touching hot surfaces can cause harm.` - これは、熱い表面に触れると傷害や損害につながる可能性があることを意味します。

`Harmful` は、害を及ぼす可能性があるものを表す形容詞です。損害や傷害につながる危険やリスクがあることを意味します。例えば：

`Smoking is harmful to health.` - これは、喫煙が健康問題や損害を引き起こす可能性があることを示しています。

これらの用語は、特に健康、安全、環境の文脈で、安全性やリスクについて話す際によく使われます。
