# --interactive--

عند بناء مواقع ويب متعددة اللغات أو التعامل مع خصائص بيانات مخصصة، تحتاج غالبًا إلى تنسيق العناصر بناءً على اللغة التي تحتويها أو قيم بيانات محددة.

تُستخدم الخاصيتان `lang` و `data-lang` عادةً لهذه الأغراض، ويسمح محدِّد الخاصية في CSS بتطبيق التنسيقات بناءً على هذه الخصائص بفعالية.

تُستخدم الخاصية `lang` في HTML لتحديد لغة المحتوى داخل العنصر. قد ترغب في تنسيق العناصر بشكل مختلف بناءً على اللغة التي كُتبت بها، خاصةً في موقع ويب متعدد اللغات.

إليك مثالًا على كيفية استخدام محدِّد الخاصية لاستهداف العناصر التي تحتوي على خاصية `lang` معينة:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">This is an English paragraph.</p>
<p lang="fr">Ceci est un paragraphe en français.</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

تطبق قاعدة CSS هذه تنسيق مائل على أي عنصر فقرة حيث تكون الخاصية `lang` مضبوطة على الإنجليزية (`en`). قد يكون هذا مفيدًا لتسليط الضوء على النص الإنجليزي في مستند يحتوي على عدة لغات.

بنفس الطريقة، يمكنك استخدام محدِّد الخاصية لاستهداف العناصر التي تحتوي على خاصية `data-lang`.

تُستخدم خصائص البيانات المخصصة مثل `data-lang` عادةً لتخزين معلومات إضافية في العناصر، مثل تحديد اللغة المستخدمة داخل قسم معين من النص.

إليك كيفية استخدام CSS لاستهداف وتنسيق العناصر بناءً على الخاصية `data-lang`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p data-lang="fr">Ceci est un paragraphe en français.</p>
<p data-lang="en">This is a paragraph in English.</p>
```

```css
p[data-lang="fr"] {
  color: blue;
}
```

:::

في هذه الحالة، أي عنصر `p` يحتوي على خاصية `data-lang` مضبوطة على الفرنسية (`fr`) سيتغير لون نصه إلى الأزرق. هذا يسمح لك بالتعرف بسرعة على أقسام المحتوى وتنسيقها بناءً على معلومات اللغة المخزنة في الخاصية `data-lang`.

تقدم محددات الخصائص مثل هذه طريقة قوية لتطبيق التنسيقات الشرطية بناءً على البيانات الوصفية المضمنة داخل HTML، مما يجعل صفحات الويب أكثر ديناميكية ووعيًا بالسياق.
