# --description--

HTML-сервер зазвичай має одну або кілька папок, доступних користувачу. Ви можете розмістити там статичні ресурси, необхідні вашому застосунку (таблиці стилів, скрипти, зображення).

В Express цю функціональність можна реалізувати за допомогою підпрограмного забезпечення `express.static(path)`, де параметр `path` — це абсолютний шлях до папки з ресурсами.

Якщо ви не знаєте, що таке підпрограмне забезпечення... не хвилюйтеся, ми детально розглянемо це пізніше. По суті, підпрограмне забезпечення — це функції, які перехоплюють обробники маршрутів, додаючи певну інформацію. Підпрограмне забезпечення потрібно підключати за допомогою методу `app.use(path, middlewareFunction)`. Перший аргумент `path` є необов’язковим. Якщо ви його не передасте, підпрограмне забезпечення виконуватиметься для всіх запитів.

# --instructions--

Підключіть підпрограмне забезпечення `express.static()` до шляху `/public` з `app.use()`. Абсолютний шлях до папки з ресурсами — `__dirname + /public`.

Тепер ваш застосунок має вміти обслуговувати таблицю стилів CSS. Зверніть увагу, що файл `/public/style.css` посилається у `/views/index.html` у шаблонному коді проєкту. Ваша головна сторінка тепер має виглядати трохи краще!

# --hints--

Ваш застосунок має обслуговувати файли ресурсів із папки `/public` за шляхом `/public`

```js
  const response = await fetch(code + '/public/style.css');
  if (!response.ok) {
    throw new Error(await response.text());
  }
  const data = await response.text();
  assert.match(
    data,
    /body\s*\{[^\}]*\}/,
    'Your app does not serve static assets'
  );
```

Ваш застосунок не повинен обслуговувати файли з інших папок, окрім папки `/public`

```js
  const response = await fetch(code + '/server.js');
  if (response.ok) {
    const data = await response.text();
    assert.equal(
      response.status + '',
      404 + '',
      'Your app must serve files only from "public" directory'
    );
  } else {
    assert.equal(
      response.status + '',
      404 + '',
      'Your app must serve files only from "public" directory'
    );
  }
```
