# --description--

En la gestión de proyectos, se utilizan frases específicas para describir cómo se manejan las tareas. 

Descomponer significa dividir algo en partes más pequeñas y manejables. Por ejemplo, `When I work on a large project, I often break down the work into smaller sections`.

Establecer `set deadlines` significa decidir el tiempo en el que las tareas deben completarse. Por ejemplo, `Setting deadlines for each task ensures that the project stays on schedule`.

Comprender estas frases ayuda a entender cómo se organizan los proyectos.
