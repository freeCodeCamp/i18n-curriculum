# --interactive--

링크에는 `link`, `visited`, `hover`, `focus`, `active`와 같은 다양한 상태가 있습니다. 이러한 상태는 사용자가 링크를 인식하고 상호작용 후 명확한 피드백을 제공하는 데 중요하며, 이는 사용성 및 접근성 향상에 도움이 됩니다.

이러한 다양한 링크 상태를 스타일링하는 것은 사용성과 접근성에 매우 중요합니다. 현재 링크 상태에 대한 시각적 신호를 제공하여 사용자가 방문한 링크, 상호작용 중인 링크, 클릭 시 어떤 일이 일어날지 이해하는 데 도움을 줍니다.

시각 장애나 인지 장애가 있는 사용자에게는 이러한 구분된 스타일이 탐색을 훨씬 쉽고 직관적으로 만듭니다.

또한 명확한 링크 상태는 사용자 상호작용에 즉각적인 피드백을 제공하여 혼란을 줄이고 사이트 탐색성을 향상시켜 전반적인 사용자 경험을 높입니다.

이 상태들은 CSS에서 `pseudo-classes`라는 것을 사용해 스타일링할 수 있습니다.

가상 클래스는 선택자에 추가되어 선택된 요소의 특별한 상태를 지정하는 키워드입니다.

예를 들어, `:hover`는 사용자의 포인터가 버튼 위에 있을 때 버튼 색상을 변경할 수 있고, `:visited`는 이미 방문한 링크의 색상을 변경할 수 있습니다.

가상 클래스는 HTML에 추가 마크업 없이 요소의 상태나 사용자 상호작용에 따라 스타일을 지정할 수 있게 해줍니다.

`pseudo-class`의 구문은 대략 다음과 같으며, 여기서 `A`는 선택자이고 `:B`은 `pseudo-class`입니다:

```css
A:B {
  property: value;
}
```

링크의 다양한 상태를 스타일링하는 방법을 더 잘 이해하기 위해 몇 가지 예제를 살펴보겠습니다.

`:link` 가상 클래스는 방문하지 않은 링크를 스타일링하여 클릭 가능함을 나타냅니다.

다음은 앵커 요소를 대상으로 `:link` 가상 클래스를 사용하는 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Normal state (unvisited link) */
a:link { 
  color: red;
}
```

:::

위 예제는 방문하지 않은 링크의 기본 파란색을 `red`로 변경합니다.

`:visited`는 이미 방문하거나 클릭한 링크를 스타일링하여 사용자가 이전에 클릭한 링크를 추적할 수 있게 도와줍니다. 다음은 `:visited` 가상 클래스 사용 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="https://www.freecodecamp.org/learn/" target="_blank">freeCodeCamp</a>
```

```css
/* Visited link */
a:visited {
  color: green;
}
```

:::

이 코드는 클릭된 링크의 색상을 `green`로 변경합니다.

`:hover`는 사용자가 링크 위에 마우스를 올렸을 때 링크 스타일을 변경하여 링크가 상호작용 가능함을 시각적으로 알려줍니다. 다음은 `:hover` 가상 클래스 사용 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Hover state */
a:hover {
  color: green;
}
```

:::

이 코드는 마우스가 링크 위에 있을 때 링크 색상을 `green`로 변경합니다.

`:focus`는 키보드로 탐색할 때처럼 링크에 포커스가 있을 때 링크 주변에 스타일을 추가하여 접근성을 향상시킵니다.

다음은 링크에 포커스가 있을 때 `outline` 속성을 사용해 주황색 실선 윤곽선을 적용하는 예입니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Focus state */
a:focus {
  outline: 2px solid orange;
}
```

:::

`:active`는 링크를 클릭하는 동안 링크 스타일을 변경하여 사용자의 동작이 인식되고 있음을 즉시 알려줍니다. 다음은 `:active` 가상 클래스 사용 예입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Active state */
a:active {
  color: pink;
}
```

:::

이 코드 예제는 링크를 클릭할 때 즉시 `pink` 색상으로 변경합니다.
