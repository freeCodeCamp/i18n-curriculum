# --description--

Per chiedere a qualcuno se vuole qualcosa, lo spagnolo usa il verbo `querer`.

La forma verbale `quieres` si usa con `tú` (tu) per chiedere i desideri o le preferenze di un'altra persona. È comune nella conversazione quotidiana. Per esempio:

- `¿Quieres ayuda?` - Vuoi aiuto?

- `¿Quieres más tiempo?` - Vuoi più tempo?
