# --description--

`convention` は、特に仕事の場で、多くの人が物事を行う方法やルールのようなものです。例えば、コーディングでは、コードを特定のスタイルで書くという共通のルールがあります。

`To follow` は、これらのルールに従う、または守ることのような意味です。つまり、コーディングルールに従うとは、多くの人が最良と認める方法でコードを書くことを意味します。
