# --description--

スペイン語のカリキュラムをコンピューターで進める場合、アクセントや`ñ`のような特殊文字を入力できるように設定しておく必要があります。

WindowsでもMacでも、設定は簡単で数分で終わります。

## Windows 11での設定

1. スタートメニューから**設定**を開いてください。

2. **時刻と言語**に進み、次に**言語と地域**をクリックします。

3. **優先する言語**に進み、**言語を追加する**を選んでください。

4. **スペイン語**を検索し、`Español (Latin America)`などのバリアントを選びます。

5. **次へ**をクリックし、指示に従ってください。

言語がインストールされたら：  

- **優先する言語**リストで見つけます。

- 省略記号をクリックして追加オプションを表示し、**言語オプション**を選択します。

- **キーボード**に進み、**キーボードを追加**をクリックして好みのスペイン語レイアウトを選びます。

キーボードを切り替えるには：  

- **Windowsキー + スペースバー**を押します。

- またはタスクバーの言語アイコン（通常は**ENG**と表示）をクリックしてスペイン語を選びます。

## Macでの設定

1. **アップルメニュー**を開き、**システム設定**を選択します。

2. **キーボード**に進みます。

3. **テキスト入力** > **入力ソース**に進みます。

4. **編集...**をクリックします。

5. 下部の**プラス（+）**ボタンをクリックし、リストから**スペイン語**を選びます。

6. **追加**をクリックします。

これでスペイン語のカリキュラムを進める準備が整い、アクセントや特殊文字を使う課題も問題なくこなせます。
