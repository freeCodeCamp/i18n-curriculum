# --interactive--

ستحتاج أحيانًا إلى التحكم في أنماط عنصر HTML داخل ملف JavaScript الخاص بك. مثال على ذلك هو عندما تحتاج إلى إخفاء أو عرض قائمة عند نقر المستخدم على زر.

في هذا الدرس، ستتعلم كيفية التحكم في الأنماط باستخدام خاصيتي `Element.style` و `Element.classList`.

خاصية `Element.style` هي خاصية للقراءة فقط تمثل النمط المضمّن لعنصر. يمكنك استخدام هذه الخاصية للحصول على النمط أو تعيينه لعنصر.

إليك مثال على عنصر فقرة له معرف `para`:

```html
<p id="para">This is a paragraph</p>
```

يمكنك استخدام خاصية `style` لتغيير لون عنصر الفقرة إلى الأحمر:

:::interactive_editor

```html
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```js
const paraEl = document.getElementById("para");
paraEl.style.color = "red";
```

:::

يمكن استخدام خاصية `style` لتعيين العديد من خصائص CSS، مثل `color` و `background-color` و `font-size` و `font-weight`، وهكذا.

طريقة أخرى للتحكم في الأنماط هي باستخدام خاصية `Element.classList`. خاصية `classList` هي خاصية للقراءة فقط يمكن استخدامها لإضافة أو إزالة أو تبديل الأصناف على عنصر. دعنا نلقي نظرة على بعض الأمثلة.

في هذا المثال، ستضيف صنفًا يسمى `highlight` إلى عنصر فقرة له معرف `para`. في CSS، سنحدد أن صنف `highlight` يغير لون خلفية عنصر الفقرة إلى الأصفر. في JavaScript، سنضيف صنف `highlight` إلى عنصر الفقرة باستخدام الطريقة `classList.add()`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```css
.highlight {
  background-color: yellow;
}
```

```js
const paraEl = document.getElementById("para");
paraEl.classList.add("highlight");
```

:::

يمكنك أيضًا إضافة عدة أصناف إلى عنصر بتمريرها كمعلمات إلى الطريقة `classList.add()`:

```js
classList.add("class1", "class2", "class3");
```

إذا احتجت إلى إزالة صنف من عنصر، يمكنك استخدام الطريقة `classList.remove()` بهذا الشكل:

```js
classList.remove("highlight");
```

لتبديل صنف على عنصر، يمكنك استخدام الطريقة `classList.toggle()`. في هذا المثال، لدينا عنصر `button` بمعرف `toggle-btn` وعنصر `nav` بمعرف `menu`. في CSS، أضفنا بعض الأنماط للقائمة وصنفًا يسمى `show` يغير خاصية `display` للعنصر `.menu` إلى `block`. في JavaScript، نصل إلى عنصر القائمة وعنصر الـ `button` باستخدام الطريقة `getElementById()`. ثم نضيف مستمع حدث إلى عنصر الـ `button` يقوم بتبديل الصنف `show` على العنصر `.menu` عند نقر الزر:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button id="toggle-btn">Toggle Menu</button>

<nav id="menu" class="menu">
  <ul>
    <li>Home</li>
    <li>About</li>
    <li>Products</li>
  </ul>
</nav>
<script src="index.js"></script>
```

```css
.menu {
  display: none;
  background-color: lightgray;
  width: 50%;
  padding: 10px;
}

.menu.show {
  display: block;
}
```

```js
const menu = document.getElementById("menu");
const toggleBtn = document.getElementById("toggle-btn");

toggleBtn.addEventListener("click", () => menu.classList.toggle("show"));
```

:::

عندما ينقر المستخدم على الزر، سيُضاف صنف `show` إلى عنصر `.menu`، وستُعرض القائمة. إذا نقر المستخدم الزر مرة أخرى، سيُزال صنف `show` من عنصر `.menu`، وستُخفى القائمة.

باستخدام طرق وخصائص مثل `style` و `add()` و `remove()` و `toggle()`، يمكنك التحكم بسهولة في أنماط العناصر باستخدام JavaScript، مما يتيح إنشاء صفحات ويب ديناميكية وتفاعلية.
