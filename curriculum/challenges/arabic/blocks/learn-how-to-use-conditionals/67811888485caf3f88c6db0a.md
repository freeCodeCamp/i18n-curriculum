# --explanation--

تقول ماريا، `at the end of each day`، مما يشير بوضوح إلى أن النسخ الاحتياطية مجدولة يوميًا. هذا يعني أن المهمة مخططة لتحدث مرة واحدة كل يوم. على سبيل المثال:

- `At the end of each day, the system runs a data backup.` - يحفظ النظام جميع البيانات كل مساء لتجنب الفقدان.

- `We ensure the servers are updated at the end of each day.` - تُطبَّق التحديثات على الخوادم ليلاً لتجنب الانقطاعات أثناء ساعات العمل.
