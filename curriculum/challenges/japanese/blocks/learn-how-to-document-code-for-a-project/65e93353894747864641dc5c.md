# --description--

前置詞 `on` は、プロジェクトで何についてコメントするかのように、何かの主題や話題を示すためによく使われます。例えば、`The report focused `on` the project's progress.`

`Any` は、どれか特定のものを指定せずに、1つ以上の項目があるかどうかを尋ねるときに使われます。例えば、`any` 質問があるかどうかを尋ねる場合です。
