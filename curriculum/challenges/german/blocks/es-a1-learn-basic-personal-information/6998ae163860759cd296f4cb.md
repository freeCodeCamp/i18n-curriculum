# --explanation--

Ángela sagt `¿Navarro con dos erres?` Dies ist eine Bestätigungsfrage.

`Con` fügt zusätzliche Informationen zur Rechtschreibung hinzu.

`Dos` ist die Zahl zwei.

`Erres` ist die Pluralform von `erre`, dem Namen des Buchstabens `r`.

Ángela bestätigt, dass der Nachname Navarro mit zwei Buchstaben `r` geschrieben wird.
