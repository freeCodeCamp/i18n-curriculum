# --description--

Gegeben ist ein String, der eine Zahl darstellt, und eine ganze Zahl als Basis von 2 bis 36. Bestimmen Sie, ob die Zahl in dieser Basis gültig ist.

- Der String kann ganze Zahlen sowie Groß- oder Kleinbuchstaben enthalten.
- Die Prüfung soll nicht zwischen Groß- und Kleinbuchstaben unterscheiden.
- Die Basis kann eine beliebige Zahl von 2 bis 36 sein.
- Eine Zahl ist gültig, wenn jedes Zeichen eine gültige Ziffer in der angegebenen Basis ist.
- Beispiele für gültige Ziffern in verschiedenen Basen:
  - Basis 2: 0-1
  - Basis 8: 0-7
  - Basis 10: 0-9
  - Basis 16: 0-9 und A-F
  - Basis 36: 0-9 und A-Z

# --solutions--

```py
def is_valid_number(n, base):
    allChars = "0123456789abcdefghijklmnopqrstuvwxyz"
    newN = n.lower()

    availableChars = allChars[0:base]

    for char in newN:
        if char not in availableChars:
            return False

    return True
```
