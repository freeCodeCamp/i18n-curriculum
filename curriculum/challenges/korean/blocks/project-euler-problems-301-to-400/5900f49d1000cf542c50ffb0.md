# --description--

$S$를 10진법으로 쓴 연속된 양의 정수(1부터 시작)를 이어 붙여 만든 (무한) 문자열이라고 하겠습니다.

따라서, $S = 1234567891011121314151617181920212223242\ldots$입니다.

어떤 숫자든 $S$에서 무한히 여러 번 나타난다는 것을 쉽게 알 수 있습니다.

$f(n)$을 $S$에서 $n$이 $n$번째로 나타나는 시작 위치라고 하겠습니다. 예를 들어, $f(1) = 1$, $f(5) = 81$, $f(12) = 271$, $f(7780) = 111\\,111\\,365$입니다.

$1 \le k \le 13$일 때 $\sum f(3^k)$를 구하세요.
