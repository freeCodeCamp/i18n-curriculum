# --interactive--

JavaScriptの`splice()`メソッドは、配列を変更する強力な方法です。このメソッドを使うと、配列のどの位置からでも要素を追加または削除できます。`splice()`メソッドの戻り値は、配列から削除されたアイテムの配列になります。何も削除されなかった場合は、空の配列が返されます。

このメソッドは元の配列を変更し、新しい配列を作成するのではなく、その場で配列を変更することに注意してください。これは`splice()`を扱う際に覚えておくべき重要なポイントです。基本的な構文は以下の通りです。

```js
array.splice(startIndex, itemsToRemove, item1, item2)
```

`startIndex`は配列の変更を開始するインデックスを指定し、`itemsToRemove`は削除する要素数を示すオプションのパラメータです。`itemsToRemove`が省略されると、`splice()`は`startIndex`から配列の最後までのすべての要素を削除します。続くパラメータ（`item1`、`item2`など）は、開始インデックスから配列に追加される要素です。

まず、配列の中央から要素を削除する例を見てみましょう。

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango", "kiwi"];
let removed = fruits.splice(2, 2);

console.log(fruits);  // ["apple", "banana", "kiwi"]
console.log(removed); // ["orange", "mango"]
```

:::

この例では、`splice(2, 2)`がインデックス`2`から始まり、`2`個の要素を削除します。変更後の配列は`apple`、`banana`、`kiwi`のみになります。次に、配列の中央に要素を追加する方法を見てみましょう。

:::interactive_editor

```js
let colors = ["red", "green", "blue"];
colors.splice(1, 0, "yellow", "purple");

console.log(colors); // ["red", "yellow", "purple", "green", "blue"]
```

:::

ここでは、`splice(1, 0, "yellow", "purple")`がインデックス`1`から始まり、`0`個の要素を削除し、`yellow`と`purple`を挿入します。2番目のパラメータ（`0`）は挿入前に要素を削除しないことを意味します。`splice()`を使って、要素の削除と追加を同時に行うこともできます。

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];
numbers.splice(1, 2, 6, 7, 8);

console.log(numbers); // [1, 6, 7, 8, 4, 5]
```

:::

この場合、`splice(1, 2, 6, 7, 8)`がインデックス`1`から始まり、`2`個の要素（`2`と`3`）を削除し、`6`、`7`、`8`を挿入します。元の配列を変更せずに保持したい場合は、`splice()`を使う前にコピーを作成してください。

:::interactive_editor

```js
let original = [1, 2, 3, 4, 5];
let copy = [...original];
copy.splice(2, 1, 6);

console.log(original); // [1, 2, 3, 4, 5]
console.log(copy);     // [1, 2, 6, 4, 5]
```

:::

この例では、`original`配列を変更せずにコピーを作成するためにスプレッド演算子（`...`）を使っています。スプレッド演算子は`original`配列の要素を浅くコピーして新しい配列を作成します。これは今後のレッスンで詳しく学びます。

`copy.splice(2, 1, 6)`を使うと、コピー配列のインデックス`2`（`3`）の要素を削除し、その位置に新しい要素`6`を挿入します。

`splice()`の一般的な使い方の1つは、インデックスがわかっている要素を配列から1つだけ削除することです。

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango"];
let indexToRemove = fruits.indexOf("orange");
if (indexToRemove !== -1) {
    fruits.splice(indexToRemove, 1);
}

console.log(fruits); // ["apple", "banana", "mango"]
```

:::

この例では、まず`indexOf()`メソッドを使って`orange`配列の要素`fruits`のインデックスを見つけます。`indexOf()`メソッドは指定した要素の最初の出現位置のインデックスを返し、要素が配列に存在しない場合は`-1`を返します。

次に、`indexToRemove`と`-1`を比較して、要素が配列に存在することを確認してから削除を試みます。`indexToRemove`が`-1`と等しくない場合（つまり要素が見つかった場合）、`splice()`を使って`indexToRemove`の位置から1つの要素を削除します。

`splice()`を使って配列のすべての要素を削除し、配列をクリアすることもできます。

:::interactive_editor

```js
let array = [1, 2, 3, 4, 5];
array.splice(0);

console.log(array); // []
```

:::

`splice()`は強力ですが、非常に大きな配列の場合、特に配列の先頭を変更するときは他の方法より効率が悪くなることがあります。これは`splice()`が後続のすべての要素をシフトする必要があるためです。そのような場合、配列の末尾に要素を追加または削除するだけなら、`push()`、`pop()`、`unshift()`、`shift()`のようなメソッドの方が適しています。

まとめると、`splice()`メソッドはJavaScriptで配列を変更するための多用途な方法です。配列の任意の位置から要素を追加および削除する正確な制御が可能です。`splice()`を効果的に使いこなすことで、JavaScriptコードでの配列操作能力が大きく向上します。
