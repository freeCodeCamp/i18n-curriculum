# --description--

排版是選擇合適字型和格式，使文字在視覺上吸引人且易於閱讀的藝術。「型別」指的是個別字元的設計和排列方式。透過為你的專案選擇合適的字型，你可以喚起情感、建立階層體系，並強化你的品牌識別。

我們將從討論字型和字體開始。字型是一組字元、數字和符號的整體設計與風格。它就像是一個字體家族的藍圖。字體是字型的特定變體，具有特定的特徵，例如大小、粗細、樣式和寬度。

兩個非常重要的字型範例是 Serif 和 Sans Serif。Serif 字型具有經典風格，字元末端帶有小線條。Serif 字型通常用於印刷材料，如書籍。一些範例有 Times New Roman、Georgia 和 Garamond。 

相較之下，無襯線字型具有較現代的外觀，字元末端沒有小線條。無襯線字型常用於數位設計，因為它們在螢幕上易於閱讀。一些範例包括 Helvetica、Arial 和 Roboto。還有其他字型分類，例如 Script、Blackletter、Monospaced 和 Decorative。

我們提到字型就像字體的藍圖，對吧？字體如果有相似的設計，也可以被群組。不同的粗細和樣式可以被組合。<dfn>例如</dfn>，`Times New Roman` 包含像是 `Times New Roman Bold`、`Times New Roman Italic` 和 `Times New Roman Bold Italic` 等變體。這些同一字體家族內的變體通常包含：

- 字重：字元的粗細，包括細體、常規、粗體和黑體。
- 樣式：字元的傾斜和方向，例如斜體和傾斜。
- 寬度：字元所佔用的水平空間，例如壓縮和擴展。

太好了。現在讓我們來探索排版的基本元素：

- 基線是大多數字元所依附的虛擬線。
- 大寫字母高度是從基線到頂部的大寫字母高度。
- x-height 是不包含升部和降部的小寫字母的平均高度。
- 升部是小寫字母中超出 x-height 的零件，例如字母 `h`、`b`、`d` 和 `f` 的頂部。
- 降部是小寫字母中延伸到基線以下的零件，例如 `y`、`g`、`p` 和 `q` 的尾部。

還有更多進階概念，例如字距調整、字距追蹤和行距。

字距調整是如何調整特定字元配對之間的間距，以提升其可讀性和美觀度。例如，減少字母 `A` 和 `V` 之間的間距。字距追蹤是如何調整一個區段中所有字元之間的間距。它本質上是字元之間的距離。它會影響文字的密集度和開放度。行距是文字行與行之間的垂直間距。它是從一行的基線測量到下一行的基線。

熟悉這些概念對於為你的專案選擇正確的字型至關重要。 

列長度也會對可讀性產生重大影響。如果文字過窄，讀者將需要過於頻繁地重新聚焦。如果文字過寬，讀者可能會感到疲勞。你應該找到適合舒適閱讀的最佳列長度。

透過字型，你也可以創建視覺階層體系，幫助讀者了解文字或使用者介面中哪些部分最重要。較大的字型通常顯示文字較為重要。將文字設為粗體也有助於強調它。

使用底線或斜體呈現 `text` 有助於凸顯標示關鍵點。提供準確的間距於 `text` 周圍可以標明其重要性。而色彩對比則能吸引讀者注意 `text` 的特定零件。

具備基本的排版知識對於設計視覺吸引人的應用程式至關重要。透過了解字型、間距和階層體系，你可以創建增強內容可讀性並強化品牌識別的視覺體驗。
