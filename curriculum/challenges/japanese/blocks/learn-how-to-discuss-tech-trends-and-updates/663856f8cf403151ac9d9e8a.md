# --description--

`AI` は人工知能を意味します。これは、人間のように動作し反応する知的な機械の創造を扱うコンピューターサイエンスの一分野です。`AI` は学習、問題解決、知覚、言語理解などのタスクに関わることがあります。

プログラミングでは、`AI` を使って、人間の知能を必要とする複雑なタスクを実行するシステムを開発できます。例としては、音声アシスタント、チャットボット、推薦システムなどがあります。
