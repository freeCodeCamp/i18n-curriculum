# --explanation--

ジェイクは、怪しいメールがフィッシング攻撃の一部である可能性があると思っているので、`It's possible` と言うことができます。

`Impossible` は、接頭辞 `im-`（「～でない」または「～の反対」を意味する）を基本語 `possible` に付けて作られています。例えば：

- `It's possible.` - 何かが起こる可能性がある。

- `It's impossible.` - 起こる可能性がない。
