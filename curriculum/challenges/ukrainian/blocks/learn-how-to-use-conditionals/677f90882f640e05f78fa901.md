# --explanation--

Час `Present Continuous` часто підкреслює тимчасовий стан або проблему, що відбувається в момент мовлення. Наприклад:

- `The server isn't running diagnostics.` — підкреслює, що процес діагностики зараз не відбувається.

- `Developers aren't debugging the application.` — показує, що розробники зараз не працюють над цією діяльністю.

`Properly` означає робити щось правильно або очікуваним способом.

У цьому реченні `aren't booting properly` стосується комп’ютерів, які наразі не запускаються правильно.
