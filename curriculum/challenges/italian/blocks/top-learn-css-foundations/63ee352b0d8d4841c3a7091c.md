# --description--

Cosa succede se hai due gruppi di elementi che condividono alcune delle loro dichiarazioni di stile?

```css
.read {
  color: white;
  background-color: black;
  /* several unique declarations */
}

.unread {
  color: white;
  background-color: black;
  /* several unique declarations */
}
```

Sia i nostri selettori `.read` che `.unread` condividono le dichiarazioni `color: white;` e `background-color: black;`, ma per il resto hanno diverse dichiarazioni uniche. Per ridurre la ripetizione, puoi raggruppare questi due selettori insieme come una lista separata da virgole:

```css
.read,
.unread {
  color: white;
  background-color: black;
}

.read {
  /* several unique declarations */
}

.unread {
  /* several unique declarations */
}
```

Entrambi gli esempi sopra (con e senza raggruppamento) avranno lo stesso risultato, ma il secondo esempio riduce la ripetizione delle dichiarazioni e rende più facile modificare sia `color` che `background-color` per entrambe le classi contemporaneamente.

## --text--

Come applicheresti una singola regola a due selettori diversi, `.red-box` e `.yellow-box`?

## --answers--

```css
.red-box,
.yellow-box {
  width: 25px;
  height: 25px;
}
```
