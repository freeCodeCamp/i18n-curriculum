# --description--

Katika Python, wigo unaamua mahali ambapo unaweza kufikilia kigezo. Ndiyo inayodhibiti muda wa maisha ya kigezo na jinsi kinavyotatuliwa katika sehemu tofauti za msimbo.

Ili kubaini wigo kwa usahihi, Python inafuata kanuni ya **LEGB**, ambayo inasimama kwa yafuatayo:

- **Wigo wa ndani (L)**: Vigezo vilivyotangazwa ndani ya vitendakazi au madarasa la vitu.
    
- **Wigo wa kuzunguka (E)**: Vigezo vilivyotangazwa katika vitendakazi vilivyopangwa ndani au vilivyopangwa ndani zaidi.
    
- **Wigo wa kimataifa (G):** Vigezo vilivyotangazwa katika ngazi ya juu ya moduli au faili.
    
- **Wigo uliyojengwa ndani (B)**: Majina yaliyohifadhiwa katika Python kwa vitendakazi vilivyotangazwa awali, moduli, maneno muhimu, na vitu.

Python inatumia kanuni ya LEGB kutatua wigo wa vigezo katika programu yako. Tutachambua kila moja ya kanuni hizi ili upate uelewa bora wa mchakato.

**Wigo wa ndani** maana yake ni kwamba kigezo kilichotangazwa ndani ya kitendakazi au darasa la vitu kinaweza kufikiliwa tu ndani ya kitendakazi au darasa hilo.

Hapa kuna mfano:

```python
def my_func():
    my_var = 10
    print(my_var)
```

Katika kesi hii, kitendakazi cha `my_func` kina wigo wake ambao hauwezi kufikiliwa kutoka nje ya kitendakazi. Kufanya wito kwa `my_func` kutaonyesha `10`, lakini kuchapisha `my_var` nje ya kitendakazi kutaelekea `NameError`:

```python
def my_func():
    my_var = 10 # Locally scoped to my_func
    print(my_var)

my_func() # 10

print(my_var) # NameError: name 'my_var' is not defined
```

**Wigo wa kuzunguka** maana yake ni kwamba kitendakazi kilichopangwa ndani ya kitendakazi kingine kinaweza kufikilia vigezo vya kitendakazi ambacho kimepangwa ndani yake.

Kwa mfano:

```python
def outer_func():
    msg = 'Hello there!'

    def inner_func():
        print(msg)

    inner_func()

outer_func() # Hello there!
```

Katika mfano huu, kitendakazi cha ndani, `inner_func`, kinaweza kufikilia kwa uhuru kigezo cha `msg` kilichotangazwa katika kitendakazi cha nje, `outer_func`. Hata hivyo, kumbuka kwamba vitendakazi vya nje haviwezi kufikilia vigezo vilivyotangazwa ndani ya vitendakazi vilivyopangwa ndani:

```python
def outer_func():
    msg = 'Hello there!'
    print(res)

    def inner_func():
        res = 'How are you?'
        print(msg)

    inner_func()

outer_func() # NameError: name 'res' is not defined
```

Hii ni kwa sababu `res` ni wigo wa ndani kwa `inner_func`. Pia, angalia kwamba `outer_func` inajaribu kuchapisha `res` kabla ya `inner_func` kuitwa.

Suluhisho moja ni kuanzisha `res` kama mfuatano wa herufi tupu katika wigo wa kuzunguka, ambao uko ndani ya `outer_func`. Kisha ndani ya `inner_func`, fanya `res` kuwa kigezo kisicho cha ndani na neno muhimu la `nonlocal`:

```python
def outer_func():
    msg = 'Hello there!'
    res = ""  # Declare res in the enclosing scope

    def inner_func():
        nonlocal res  # Allow modification of an enclosing variable
        res = 'How are you?'
        print(msg)  # Accessing msg from outer_func()

    inner_func()
    print(res)  # Now res is accessible and modified

outer_func()

# Output:
# Hello there!
# How are you?
```

**Wigo wa kimataifa** unahusu vigezo vilivyotangazwa nje ya vitendakazi au madarasa la vitu ambavyo vinaweza kufikiliwa kutoka sehemu yoyote katika programu. Hapa, `my_var` inaweza kufikiliwa mahali popote, hata ndani ya kitendakazi ambacho hakikitangazi:

```python
my_var = 100

def show_var():
    print(my_var)

show_var() # 100
print(my_var) # 100
```

Na ikiwa unataka kufanya kigezo kilichotangazwa ndani ya kitendakazi kiwe na upatikanaji wa kimataifa, unaweza kutumia neno muhimu la `global`:

```python
my_var_1 = 7

def show_vars():
    global my_var_2
    my_var_2 = 10
    print(my_var_1)
    print(my_var_2)

show_vars() # 7 10

# my_var_2 is now a global variable and can be accessed anywhere in the program
print(my_var_2) # 10
```

Unaweza pia kutumia neno muhimu la `global` kubadilisha kigezo cha kimataifa:

```python
my_var = 10  # A global variable

def change_var():
    global my_var  # Allows modification of a global variable
    my_var = 20

change_var()

print(my_var)  # my_var is now modified globally to 20
```

Mwishowe, **wigo uliyojengwa ndani** unahusu vitendakazi vyote vilivyotangazwa awali vya Python, moduli, na maneno muhimu, na vinapatikana mahali popote katika programu yako:

```python
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```
