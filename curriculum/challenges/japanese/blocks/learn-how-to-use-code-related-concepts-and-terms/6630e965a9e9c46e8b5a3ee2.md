# --description--

プログラミングにおいて、`int` は「整数」を表します。これは分数や小数部分を持たない整数値をストアするためのデータ型です。整数は正の数、負の数、またはゼロでありえます。

`int` データ型は、カウント、配列のインデックス、または分数が不要なあらゆる状況で、整数を扱う際に重要です。

例：

アイテムの数を数える：`int itemCount = 5;`

年齢をストアする：`int age = 30;`
