# --interactive--

In JavaScript ist ein String eine Folge von Zeichen, die verwendet wird, um Textdaten darzustellen. Strings sind einer der primitiven Datentypen in der Sprache, zusammen mit Zahlen, Booleans, `null` und `undefined`.

Um in JavaScript einen String zu erstellen, können Sie einfache Anführungszeichen (`'`) oder doppelte Anführungszeichen (`"`) verwenden.

Hier ist ein Beispiel, wie Sie zwei Variablen erstellen, die String-Werte enthalten:

:::interactive_editor

```js
let singleQuotes = 'This is a string';
console.log(singleQuotes);
let doubleQuotes = "This is also a string";
console.log(doubleQuotes);
```

:::

Obwohl Sie einfache oder doppelte Anführungszeichen verwenden können, um Strings zu erstellen, ist es wichtig, konsequent zu sein. Wenn ein String mit einem einfachen Anführungszeichen beginnt, muss er auch mit einem einfachen Anführungszeichen enden.

Dasselbe gilt für doppelte Anführungszeichen. Das folgende Beispiel wirft einen Fehler, weil es mit einem doppelten Anführungszeichen beginnt und mit einem einfachen endet:

```js
const improperStr = "Do not do this';
```

Ein weiterer wichtiger Punkt ist, dass Strings unveränderlich sind. In der Programmierung bedeutet Unveränderlichkeit, dass etwas, sobald es erstellt wurde, nicht mehr geändert werden kann. Wenn Sie also einen String erstellen, können Sie seine Zeichen nicht direkt ändern. Stattdessen würden Sie einen neuen String erstellen, wenn Sie Änderungen vornehmen möchten.

Hier ist ein Beispiel, wie Sie einer `developer`-Variablen einen neuen String zuweisen:

:::interactive_editor

```js
let developer = "Jessica";
console.log(developer);
developer = "Quincy";
console.log(developer);
```

:::

Da Strings unveränderlich sind, können wir den ersten String nicht direkt aktualisieren. Deshalb weisen wir der `developer`-Variablen einen neuen String zu.

Strings sind ein wichtiger Bestandteil der Programmierung, und in zukünftigen Lektionen lernen Sie fortgeschrittene Techniken, um sie zu manipulieren und ihr volles Potenzial zu nutzen, um dynamische und interaktive Anwendungen zu erstellen.
