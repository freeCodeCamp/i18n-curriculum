# --description--

Acha d(`n`) ifafanuliwe kama jumla ya vigawanyo halisi vya `n` (nambari zilizo chini ya `n` ambazo hugawanya `n` kwa usawa).

Kama d(`a`) = `b` na d(`b`) = `a`, ambapo `a` ≠ `b`, basi `a` na `b` ni jozi za kirafiki na kila moja kati ya `a` na `b` huitwa nambari za kirafiki.

Kwa mfano, vigawanyo halisi vya 220 ni 1, 2, 4, 5, 10, 11, 20, 22, 44, 55 na 110; kwa hiyo d(220) = 284. Vigawanyo halisi vya 284 ni 1, 2, 4, 71 na 142; hivyo d(284) = 220.

Tathmini jumla ya nambari zote za kirafiki chini ya `n`.

```js
const sumAmicableNum = (n) => {
  const fsum = (n) => {
    let sum = 1;
    for (let i = 2; i <= Math.floor(Math.sqrt(n)); i++)
      if (Math.floor(n % i) === 0)
        sum += i + Math.floor(n / i);
    return sum;
  };
  let d = [];
  let amicableSum = 0;
  for (let i=2; i<n; i++) d[i] = fsum(i);
  for (let i=2; i<n; i++) {
    let dsum = d[i];
    if (d[dsum]===i && i!==dsum) amicableSum += i+dsum;
  }
  return amicableSum/2;
};
```
