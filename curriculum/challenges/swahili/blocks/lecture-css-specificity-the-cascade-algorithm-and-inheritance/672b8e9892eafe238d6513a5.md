# --interactive--

Vichaguzi vya darasa la vitu ni sehemu muhimu ya CSS, vinavyowezesha wasanidi programu kulenga vipengele vingi vyenye sifa ya kipengele cha darasa la vitu sawa na kutumia mtindo unaoendana.

Hii inawafanya kuwa na matumizi mengi na ufanisi kwa kutumia mitindo katika mtandao mzima.

Vichaguzi vya darasa la vitu huainishwa kwa alama ya nukta (`.`) ikifuatiwa na jina la darasa la vitu. Vinaweza kutumika kwa kipengele chochote katika hati ya HTML.

Hapa kuna mfano wa kutumia kichaguzi cha darasa la vitu:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="highlight">Example paragraph</p>
```

```css
.highlight {
  color: green;
}
```

:::

Katika mfano huu, kipengele chochote chenye darasa la vitu `highlight` kitakuwa na rangi ya maandishi `green`.

Vichaguzi vya darasa la vitu vina sifa ya kipengele kubwa zaidi kuliko vichaguzi vya aina lakini ndogo kuliko vichaguzi vya ID na mitindo ndani ya mstari.

Thamani ya sifa ya kipengele kwa kichaguzi cha darasa la vitu ni `(0, 0, 1, 0)`. Hii ina maana kwamba vichaguzi vya darasa la vitu vinaweza ghairi vichaguzi vya aina, lakini vinaweza kughairiwa na vichaguzi vya ID na mitindo ndani ya mstari.

Vichaguzi vya darasa la vitu vinaweza kuunganishwa na vichaguzi vingine kuunda sheria maalum zaidi.

Hapa kuna mfano wa kuunganisha kichaguzi cha aina ya aya na kichaguzi cha darasa la vitu:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="bold-text">Example paragraph</p>
<p class="bold-text">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p.bold-text {
  font-weight: bold;
}
```

:::

Sheria hii inatumika tu kwa vipengele vya `p` ambavyo pia vina darasa la vitu la `bold-text`, na kufanya maandishi yao kuwa herufi nzito.
