- `First off, we need to address the project's scope.` – Це означає, що спочатку буде розглянута область видимості проєкту.

- `First off, let's review the current goals before we start planning.` – Це означає, що спочатку відбудеться повторення поточних цілей, перш ніж перейти до планування.
