# --interactive--

Деструктуризація масивів — це функція в JavaScript, яка дозволяє витягувати значення з масивів і призначати їх змінним більш лаконічним і зрозумілим способом. Вона надає зручний синтаксис для розпакування елементів масиву в окремі змінні.

Ця техніка особливо корисна при роботі з масивами та функціями, які повертають кілька значень. Ось приклад використання деструктуризації масивів:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];

let [first, second, third] = fruits;

console.log(first);  // "apple"
console.log(second); // "banana"
console.log(third);  // "orange"
```

:::

У цьому прикладі у нас є масив під назвою `fruits` з трьома елементами. Використовуючи деструктуризацію масивів, ми призначаємо перший елемент змінній `first`, другий — `second`, а третій — `third`. Це дозволяє легко отримувати доступ до окремих елементів масиву без використання індексної нотації.

Ось як це виглядало б, якби ви зверталися до кожного з цих елементів за їхнім індексом замість використання деструктуризації масивів:

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange"];

const first = fruits[0];
const second = fruits[1];
const third = fruits[2];

console.log(first); // "apple"
console.log(second); // "banana"
console.log(third); // "orange"
```

:::

Деструктуризація масивів також дозволяє пропускати елементи, які вас не цікавлять, використовуючи коми. Наприклад:

:::interactive_editor

```js
let colors = ["red", "green", "blue", "yellow"];
let [firstColor, , thirdColor] = colors;

console.log(firstColor); // "red"
console.log(thirdColor); // "blue"
```

:::

У цьому прикладі ми пропускаємо другий елемент масиву `colors`, використовуючи додаткову кому. Це призначає `red` змінній `firstColor` і `blue` змінній `thirdColor`, фактично ігноруючи `green`.

Ще одна потужна функція деструктуризації масивів — можливість використовувати значення за замовчуванням. Якщо в масиві менше елементів, ніж змінних, яким ви намагаєтеся призначити значення, ви можете надати значення за замовчуванням:

:::interactive_editor

```js
let numbers = [1, 2];
let [a, b, c = 3] = numbers;

console.log(a); // 1
console.log(b); // 2
console.log(c); // 3
```

:::

Тут ми призначаємо значення за замовчуванням `3` змінній `c`, тому що масив `numbers` не має третього елемента.

Тепер поговоримо про синтаксис rest, позначений трьома крапками (`...`). Він дозволяє захопити залишок елементів масиву, які не були деструктуровані, у новий масив. Ось як це працює:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango", "kiwi"];
let [first, second, ...rest] = fruits;

console.log(first);  // "apple"
console.log(second); // "banana"
console.log(rest);   // ["orange", "mango", "kiwi"]
```

:::

У цьому прикладі `first` і `second` захоплюють перші два елементи масиву `fruits`, а `rest` захоплює всі решту елементів як новий масив. Синтаксис rest має бути останнім елементом у шаблоні деструктуризації.

Деструктуризація масивів — це потужна функція, яка може зробити ваш код більш лаконічним і легшим для читання. Вона особливо корисна при роботі з масивами та коли потрібно витягти конкретні елементи з масиву.
