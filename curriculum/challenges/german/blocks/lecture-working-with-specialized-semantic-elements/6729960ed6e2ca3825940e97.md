# --interactive--

Das inline-Code-Element wird verwendet, um kurze Code-Ausschnitte innerhalb von Text darzustellen. Häufige Anwendungsfälle für das code-Element sind technische Artikel und Dokumentationsseiten.

Hier ist ein Beispiel für die Verwendung des `code`-Elements, um einen CSS-Code-Ausschnitt anzuzeigen:

:::interactive_editor

```html
<p>
  To set the text color to blue in CSS, use the following code:
  <code>color: blue;</code>
</p>
```

:::

In diesem Beispiel wird die CSS-Eigenschaft `color` verwendet, um die Textfarbe auf `blue` zu setzen. Indem Sie diesen Code-Ausschnitt in `<code>`-Tags einschließen, signalisieren Sie dem Browser, dass der Text ein Stück inline-Code ist.

Der Browser wendet Standardstile für Inhalte innerhalb des `code`-Elements an. Die Standardformatierung ist eine Schriftart mit fester Zeichenbreite.

Das `code`-Element soll eine einzelne Codezeile darstellen. Wenn Sie mehrere Codezeilen darstellen möchten, müssen Sie ein `code`-Element innerhalb eines vorformatierten Textelements platzieren.

Das vorformatierte Textelement wird verwendet, um vorformatierten Text darzustellen. Hier ist ein Beispiel für die Verwendung des vorformatierten Textelements, um eine CSS-Deklaration anzuzeigen:

:::interactive_editor

```html
<pre>
  <code>
    body {
      color: red;
    }
  </code>
</pre>
```

:::

Beim Verwenden des `pre`-Elements müssen Sie auf den Abstand achten, da es genau so angezeigt wird, wie es im HTML-Dokument geschrieben steht.

Im Browser sehen Sie, dass der Code mehrere Leerzeichen nach rechts eingerückt ist. Wenn Sie die Einrückung im Codebeispiel ändern, sehen Sie auch eine Änderung der Einrückung auf dem Bildschirm.

Wenn Sie Codebeispiele in Ihr HTML-Dokument einfügen, sollten Sie das `code`-Element für kurze inline-Beispiele verwenden.

Wenn Sie längere Code-Ausschnitte anzeigen müssen, sollten Sie die `pre`- und `code`-Elemente verwenden.
