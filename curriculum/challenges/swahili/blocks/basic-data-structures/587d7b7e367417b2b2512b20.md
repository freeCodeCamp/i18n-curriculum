# --description--

Hapo chini ni mfano wa utekelezaji rahisi kabisa wa muundo wa data wa safu ya data. Hii inajulikana kama <dfn>safu ya data ya kipimo kimoja</dfn>, ikimaanisha ina kiwango kimoja tu, au kwamba haina safu nyingine zilizopangwa ndani yake. Angalia ina <dfn>thamani za kweli au siyo kweli</dfn>, <dfn>mfuatano wa herufi</dfn>, na <dfn>nambari</dfn>, pamoja na aina nyingine halali za data za JavaScript:

```js
let simpleArray = ['one', 2, 'three', true, false, undefined, null];
console.log(simpleArray.length);
```

Wito la `console.log` linaonyesha `7`.

Safu zote za data zina sifa ya urefu, ambayo kama ilivyoonyeshwa hapo juu, inaweza kufikiliwa kwa urahisi sana kwa sintaksia `Array.length`. Utekelezaji mgumu zaidi wa safu ya data unaweza kuonekana hapa chini. Hii inajulikana kama <dfn>safu ya data zilizopangwa ndani</dfn>, au safu inayohifadhi safu nyingine ndani yake. Angalia kwamba safu hii pia ina vitu vya JavaScript vya <dfn>vitu</dfn>, ambavyo tutachunguza kwa makini katika sehemu yetu inayofuata, lakini kwa sasa, unahitaji tu kujua kwamba safu za data pia zina uwezo wa kuhifadhi vitu tata.

```js
let complexArray = [
  [
    {
      one: 1,
      two: 2
    },
    {
      three: 3,
      four: 4
    }
  ],
  [
    {
      a: "a",
      b: "b"
    },
    {
      c: "c",
      d: "d"
    }
  ]
];
```

# --instructions--

Tumeweka tofauti iitwayo `yourArray`. Kamilisha tamko kwa kuweka thamani ya safu ya data yenye vipengele vya angalau 5 kwa tofauti ya `yourArray`. Safu yako ya data inapaswa kuwa na angalau mfuatano wa herufi mmoja, nambari moja, na thamani ya kweli au siyo kweli moja.

# --hints--

`yourArray` inapaswa kuwa safu ya data.

```js
assert.strictEqual(Array.isArray(yourArray), true);
```

`yourArray` inapaswa kuwa na vipengele vya angalau 5.

```js
assert.isAtLeast(yourArray.length, 5);
```

`yourArray` inapaswa kuwa na angalau `boolean` moja.

```js
assert(yourArray.filter((el) => typeof el === 'boolean').length >= 1);
```

`yourArray` inapaswa kuwa na angalau `number` moja.

```js
assert(yourArray.filter((el) => typeof el === 'number').length >= 1);
```

`yourArray` inapaswa kuwa na angalau `string` moja.

```js
assert(yourArray.filter((el) => typeof el === 'string').length >= 1);
```
