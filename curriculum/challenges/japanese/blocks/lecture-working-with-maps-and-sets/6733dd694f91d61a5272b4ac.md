# --description--

JavaScriptにおいて、`Map` はキーと値のペアを格納する組み込みオブジェクトで、オブジェクトに似ています。しかし、標準のJavaScriptオブジェクトとは異なり、キーにオブジェクトや関数を含む任意の型を使用できます。

`WeakMap` はキーと値のペアのコレクションで、`Map` に似ていますが、キーに対して弱い参照を使用します。キーはオブジェクトでなければならず、値は任意の型を取れます。

`WeakMap` と `Map` の関係は、前のレッスンで学んだように、`WeakSet` と `Set` の関係に似ていると考えてください。

`Map` を作成するには、`Map()` コンストラクタを `new` キーワードとともに使います：

```js
const myFirstMap = new Map();
```

Mapを値で初期化することもできます：

```js
const myTreesMap = new Map([
 [{ type: 'deciduous' }, 'Maple tree'],
 [['forest', 'grove'], 'Pine tree'],
 [42, 'Oak tree'],
 [true, 'Birch tree'],
 [function() { return 'I am a function key'; }, 'Willow tree'],
]);
```

もし`Map` を値で初期化しなかった場合は、`set()` メソッドを使って追加できます：

```js
const myTreesMap = new Map();
myTreesMap.set({ type: 'deciduous' }, 'Maple tree');
myTreesMap.set([1, 2], 'Pine tree');
myTreesMap.set(42, 'Oak tree');
myTreesMap.set(true, 'Birch tree');
myTreesMap.set(function() { return "I'm a function key"; }, 'Willow tree');

console.log(myTreesMap);
```


`Map` を操作するために使える他のメソッドは以下の通りです：

- 指定した`get(key)`に関連付けられた値を取得するための`key`。
- `has(key)`に`key`が存在するか確認するための`Map`。
- `delete(key)`からキーと値のペアを削除するための`Map`。
- すべてのキーと値のペアを削除するための`clear()`。
- `entries()`のエントリーを確認するための`Map`（`MapIterator`でエントリーを返します）。
- `forEach()`のエントリーをループ処理するための`Map`。
- `size`に含まれるキーと値のペアの数を示す`Map`。
    

`WeakMap()` コンストラクタを使って`WeakMap`を作成することもできます：

```js
const myFirstWeakMap = new WeakMap();
```

`set()`、`get()`、`has()`、および`delete()` メソッドはすべて`WeakMap`でも使用可能です。例えば、`Map` メソッドで`set()`にアイテムを代入する方法は次の通りです：

```js
const myTreeWeakMap = new WeakMap();

myTreeWeakMap.set({ id: 1 }, 'Maple tree');
myTreeWeakMap.set({ id: 2 }, 'Pine tree');
myTreeWeakMap.set({ id: 3 }, 'Oak tree');
myTreeWeakMap.set({ id: 4 }, 'Birch tree');
myTreeWeakMap.set({ id: 5 }, 'Willow tree');

console.log(myTreeWeakMap);
```

コンソールでの`WeakMap` の見た目は次の通りです：

```js
/*
WeakMap {{…} => 'Willow tree', {…} => 'Maple tree', {…} => 'Pine tree', {…} => 'Oak tree'}
  [[Entries]]
    No properties
  [[Prototype]]: WeakMap
*/
```

`Map` と `WeakMap` の違いは以下の通りです：

| **機能** | **Map** | **WeakMap** |
| --- | --- | --- |
| **キーの型** | 文字列、数値、オブジェクト、関数など任意のデータ型のキーが使えます。 | キーはオブジェクトでなければなりません。 |
| **使用例** | 任意の型のキーにデータを関連付けたい場合に`Map`を使います。 | オブジェクトにのみデータを関連付けたい場合に`WeakMap`を使います。 |
| **イテレーション** | `Map` は`forEach()`、`keys()`、`values()`、または`entries()`を使ってループ処理できます。 | `WeakMap` はイテラブルではありません。 |
| **サイズプロパティ** | `Map` はキーと値のペアの数を取得する`size`プロパティがあります。 | `WeakMap` は`size`プロパティを持ちません。 |

上記の表では、キーの型、使用例、イテレーション、サイズプロパティを含む違いがまとめられています。これらの違いを理解するために、ぜひ内容を読んでみてください。
