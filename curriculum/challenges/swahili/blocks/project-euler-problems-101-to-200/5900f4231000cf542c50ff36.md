# --description--

Acha $N$ iwe nambari kamili chanya na acha $N$ igawanywe katika sehemu $k$ sawa, $r = \frac{N}{k}$, kwa hivyo $N = r + r + \cdots + r$.

Acha $P$ iwe kizidishi cha sehemu hizi, $P = r × r × \cdots × r = r^k$.

Kwa mfano, ikiwa 11 itagawanywa katika sehemu tano sawa, 11 = 2.2 + 2.2 + 2.2 + 2.2 + 2.2, basi $P = {2.2}^5 = 51.53632$.

Acha $M(N) = P_{max}$ kwa thamani fulani ya $N$.

Inajitokeza kuwa kiwango kikubwa kwa $N = 11$ hupatikana kwa kugawanya kumi na moja katika sehemu nne sawa ambayo husababisha $P_{max} = {(\frac{11}{4})}^4$; yaani, $M(11) = \frac{14641}{256} = 57.19140625$, ambayo ni desimali inayokamilika.

Hata hivyo, kwa $N = 8$ kiwango kikubwa hupatikana kwa kugawanya katika sehemu tatu sawa, hivyo $M(8) = \frac{512}{27}$, ambayo ni desimali isiyoisha.

Acha $D(N) = N$ ikiwa $M(N)$ ni desimali isiyoisha na $D(N) = -N$ ikiwa $M(N)$ ni desimali inayokamilika.

Kwa mfano, $\sum D(N)$ kwa $5 ≤ N ≤ 100$ ni 2438.

Tafuta $\sum D(N)$ kwa $5 ≤ N ≤ 10000$.
