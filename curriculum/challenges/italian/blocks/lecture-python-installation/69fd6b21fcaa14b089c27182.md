# --description--

Nella lezione precedente, hai imparato come installare Python sul tuo dispositivo locale. In questa lezione, imparerai come eseguire codice Python in locale.

Prima di poter iniziare a creare i tuoi script Python in locale, dovrai installare un editor di codice o un IDE. IDE sta per ambiente di sviluppo integrato e include funzionalità come un editor di codice, strumenti di test, un terminale e altro.

Un editor di codice molto usato da molti sviluppatori è VS Code. Visita `https://code.visualstudio.com/download` per scaricare VS Code, che supporta ambienti Mac, Windows e Linux. Questo è l’editor che useremo in questa lezione.

Altre scelte comuni per lo sviluppo Python includono:

- PyCharm
- Spyder

Una volta scaricato l’editor, dovrai aprirlo. Quando è aperto, devi aprire una cartella. Puoi cliccare sull’opzione di menu `File` nell’angolo in alto a sinistra e cliccare su `Open Folder`. Da lì puoi scegliere o creare una cartella per il tuo progetto. Per esercitarti anche al di fuori della lezione, potresti voler creare una cartella chiamata `python-projects`.

Quando hai configurato la cartella, puoi aprire l’explorer che è la barra laterale sinistra in VS Code. Clicca sull’icona `New File` che sembra un foglio con un’icona più sopra. Poi crea un file chiamato `main.py` e premi invio. L’estensione `.py` indica che questo è un file Python.

Apri il file `main.py` e digita il seguente codice:

```py
print("Hello, world!")
```

Per eseguire questo codice, hai alcune opzioni. La prima è cliccare sul pulsante di esecuzione che sembra un pulsante play situato nell’angolo in alto a destra. Questo aprirà un terminale ed eseguirà il tuo script Python lì. Dovresti vedere il testo `"Hello, world!"`.

Un’altra opzione è aprire un terminale ed eseguire manualmente il programma usando:

```bash
python main.py
```

Devi eseguire questo comando dalla cartella dove è salvato `main.py`. Per esempio, se `main.py` si trova dentro una cartella chiamata `python-projects`, prima usa questo comando nel terminale:

```bash
cd python-projects
```

Poi esegui:

```bash
python main.py
```

In alcuni ambienti, specialmente su sistemi macOS e Linux dove sono installate sia Python 2 che Python 3, potresti dover usare `python3` invece di `python`. Questo è comune quando il comando `python` non esiste o punta a una versione più vecchia di Python.

```bash
python3 main.py
```
