# --description--

Припустімо, у вас є два рядки, які ви хочете об’єднати, щоб створити одне повідомлення, як показано нижче:

```js
const str1 = "Hello";
const str2 = "World!";
const message = str1.push(str2);
```

<img src="https://cdn.statically.io/gh/TheOdinProject/curriculum/4ed59981b4ce2c60b5b83bf7415d3127b61821f5/foundations/javascript_basics/understanding_errors/imgs/03.png" style="width:100%" alt="приклад типу помилки в консолі розробника Chrome">

Тут ви отримаєте `TypeError` з повідомленням, що `str1.push is not a function`. Це загальне повідомлення про помилку, яке плутає учнів, бо ви можете знати, що `.push()` безумовно є функцією (наприклад, якщо ви раніше використовували її, щоб додавати елементи до масивів).

Але в цьому і суть — `.push()` не є методом рядка, це метод масиву. Тому це «не функція», яку можна знайти як метод рядка. Якщо ви зміните `.push()` на `.concat()`, правильний метод рядка, наш код працюватиме як задумано!

Добре пам’ятати, коли ви стикаєтеся з `TypeError`, що варто розглянути тип даних, до якого ви намагаєтеся застосувати метод або операцію. Ви, ймовірно, виявите, що це не те, що ви думаєте, або операція чи метод не сумісні з цим типом.

# --questions-- 

## --text--

Чому наведений нижче код JavaScript призводить до `TypeError`?

```js
const str1 = "Hello";
const str2 = "World!";
const message = str1.push(str2);
```

## --answers--

Метод `.push()` можна використовувати лише для об’єктів, а не для рядків.

---

Метод `.push()` недоступний для рядків, бо це метод масиву.

---

Метод `.push()` написаний з помилкою і має бути .pusch().

---

`TypeError` є результатом синтаксичної помилки в коді JavaScript.
