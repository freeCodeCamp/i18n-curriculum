# --description--

長條圖和散佈圖都直接將資料繪製到 SVG 上。然而，如果長條的高度或其中一個資料點大於 SVG 的高度或寬度值，則會超出 SVG 區域。

在 D3 中，有用於繪製資料的比例尺。`scales` 是告訴程式如何將一組未經處理的資料點映射到 SVG 像素上的函式。

例如，假設你有一個 100x500 大小的 SVG，並且你想繪製多個國家的國內生產毛額（GDP）。這組數字會在數十億或數兆美元的區間。你提供給 D3 一種 `scale` 型別，告訴它如何將龐大的 GDP 數值放置到那個 100x500 大小的區域中。

你不太可能直接繪製未經處理的資料。在繪製之前，你會為整個資料集設定比例，讓 `x` 和 `y` 值符合你的 SVG 寬度和高度。

D3 有幾種比例尺型別。對於線性比例尺（通常用於量化資料），有 D3 方法 `scaleLinear()`：

```js
const scale = d3.scaleLinear();
```

預設情況下，比例尺使用恆等關係。輸入的值與輸出的值相同。另一個挑戰涵蓋如何更改此設定。

# --instructions--

將 `scale` 變數更改為創建線性比例尺。然後將 `output` 變數設定為以引數 `50` 呼叫的比例尺。
