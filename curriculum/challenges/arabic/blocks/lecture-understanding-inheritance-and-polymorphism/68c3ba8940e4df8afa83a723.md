# --description--

الوراثة هي المفهوم الرئيسي التالي في البرمجة الكائنية التوجه (OOP) الذي سنغطيه.

لنلق نظرة أعمق على هذا المفهوم وكيف يسمح لك بكتابة شفرة قابلة لإعادة الاستخدام.

مع الوراثة، يمكن للصنف الفرعي (أو الصنف الابن) استخدام الخصائص والطرق الخاصة بالصنف الأساسي (أو الصنف الأصل). هذا يسمح لك بإعادة استخدام الشفرة، وإنشاء تسلسلات هرمية واضحة للصنف، وتخصيص السلوك دون إعادة كتابة كل شيء. يمكنك التخصيص بتمديد الطرق الموجودة أو تجاوزها في الصنف الفرعي.

إليك الصيغة النحوية الأساسية للوراثة:

```py
class Parent:
    # Parent attributes and methods

class Child(Parent):
    # Child inherits, extends, and/or overrides where necessary
```

لكي يرث الصنف `Child` من الصنف `Parent`، يجب تمرير Parent إلى Child.

يُسمى هذا النمط بالوراثة الأحادية، إذ يرث الصنف الفرعي من صنف أصل واحد فقط.

إليك مثالاً:

```py
class Animal:
    def __init__(self, name):
        self.name = name

    def sound(self):
        return f'{self.name} makes a sound'

class Dog(Animal):
    bark = 'woof! woof!! woof!!!'

jack = Dog('Jack')
print(jack.sound())  # Jack makes a sound
print(jack.bark)  # woof! woof!! woof!!!
```

يمكنك أن ترى أننا قادرون على إعادة استخدام الخاصية `self.name` والطريقة `sound()` من الصنف الأصل `Animal` في الصنف الفرعي `Dog`.

دعنا نتجاوز الطريقة `sound()` من الصنف الأصل `Animal` في الصنف الفرعي `Dog` لكي تستخدم `sound()` المتغير الصنف `bark`:

```py
class Animal:
    def __init__(self, name):
        self.name = name

    def sound(self):
        return f'{self.name} makes a sound.'

class Dog(Animal):
    bark = 'woof! woof!! woof!!!'

    # Override sound() to use bark class variable
    def sound(self):
        return f'{self.name} barks {self.bark}'

jack = Dog('Jack')
print(jack.sound())  # Jack barks woof! woof!! woof!!!
```

إذا أردت الاحتفاظ بقيمة الإرجاع من `sound()` وإضافة متغير الصنف bark لاحقًا، يمكنك تمديد `sound()` باستخدام الدالة `super()`:

```py
class Animal:
    def __init__(self, name):
        self.name = name

    def sound(self):
        return f'{self.name} makes a sound'

class Dog(Animal):
    bark = 'woof! woof!! woof!!!'

    # Call Animal.sound(), then append bark
    def sound(self):
        base = super().sound()
        return f'{base}, then {self.name} barks {self.bark}'

jack = Dog('Jack')
print(jack.sound())  # Jack makes a sound, then Jack barks woof! woof!! woof!!!
```

في هذا المثال، `base` هو نتيجة استدعاء الطريقة `sound()` من الصنف `Animal`، ثم نضيف الصوت الخاص بصنف `Dog` إليها. بهذه الطريقة، يمكنك تمديد وظيفة الصنف الأصل `Animal` مع الحفاظ على سلوكه الأصلي.

هناك أيضًا الوراثة المتعددة، حيث يمكن للصنف الفرعي أن يرث من أكثر من صنف أصل واحد.

إليك الصيغة النحوية الأساسية للوراثة المتعددة:

```py
class Parent:
    # Attributes and methods for Parent

class Child:
    # Attributes and methods for Child

class GrandChild(Parent, Child):
    # GrandChild inherits from both Parent and Child
    # GrandChild can combine or override behavior from each
```

طريقة بسيطة لشرح الوراثة المتعددة هي باستخدام الضفدع، الذي يمكنه المشي على الأرض والسباحة في الماء:

```py
class Walker:
    def walk(self):
        return 'I can walk on land'

class Swimmer:
    def swim(self):
        return 'I can swim in water'

# Amphibian inherits from both Walker and Swimmer
class Amphibian(Walker, Swimmer):
    def __init__(self, name):
        self.name = name

    def introduce(self):
        return f"I'm {self.name} the frog. {self.walk()} and {self.swim()}."

frog = Amphibian('Freddy')
print(frog.introduce())
# Output: I'm Freddy the frog. I can walk on land and I can swim in water.
```
