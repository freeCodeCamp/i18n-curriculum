# --description--

Für eine positive ganze Zahl $n$ definieren Sie $f(n)$ als das kleinste positive Vielfache von $n$, das in Basis 10 geschrieben nur Ziffern $≤ 2$ verwendet.

Somit gilt $f(2) = 2$, $f(3) = 12$, $f(7) = 21$, $f(42) = 210$, $f(89) = 1\\,121\\,222$.

Außerdem ist $\displaystyle\sum_{n = 1}^{100} \frac{f(n)}{n} = 11\\,363\\,107$.

Berechnen Sie $\displaystyle\sum_{n = 1}^{10\\,000} \frac{f(n)}{n}$.
