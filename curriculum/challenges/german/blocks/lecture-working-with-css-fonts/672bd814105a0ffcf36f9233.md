# --interactive--

Eine Schriftfamilie ist eine Gruppe von Schriftarten, die ein gemeinsames Design teilen. Alle Schriftarten, die zur gleichen Familie gehören, basieren auf derselben Grundschrift, haben aber auch Variationen in Stil, Gewicht und Breite. Sie können sich das vorstellen wie Geschwister, die ähnliche Merkmale haben, aber auch einige Unterschiede. Zum Beispiel ist Arial eine Schriftfamilie, die Variationen wie Arial Bold und Arial Italic umfasst.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-are-font-families-and-how-do-they-work-1.png" alt="Diagram showing 'Arial' at the top with two blue arrows pointing down to 'Arial Bold' on the left and 'Arial Italic' on the right, illustrating font family variations" />

In CSS können Sie die Schriftfamilie mit der `font-family` Eigenschaft festlegen. Hier ist ein Beispiel, wie Sie die Schriftart für den Absatz auf `Arial` setzen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="arial-font">Example paragraph using Arial font.</p>
<p>Paragraph not using Arial font.</p>
```

```css
#arial-font {
  font-family: Arial;
}
```

:::

Aber was passiert, wenn die Schriftfamilie nicht gefunden wird? Sie können mehrere Schriftfamilien in der Reihenfolge ihrer Priorität von höchster zu niedrigster angeben, indem Sie sie mit Kommas trennen. Diese alternativen Schriftarten fungieren als Fallback-Optionen. In diesem Beispiel ist Arial die primäre Schriftart und Lato eine alternative Schriftart:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="specified-font">Example paragraph using specified fonts.</p>
<p>Paragraph not using specified fonts.</p>
```

```css
#specified-font {
  font-family: Arial, Lato;
}
```

:::

Der Browser rendert die erste Schriftart (Arial), wenn sie auf dem Gerät gefunden wird. Falls nicht, wird die zweite Schriftart (Lato) gerendert.

Sie sollten auch wissen, wie dieser Prozess hinter den Kulissen funktioniert. Der Auswahlprozess endet nicht, wenn die erste Schriftart verfügbar ist. Die Schriftfamilie wird Zeichen für Zeichen ausgewählt, sodass der Browser, falls eine Schriftart ein bestimmtes Zeichen nicht enthält, dieses in den Schriftarten mit niedrigerer Priorität sucht. Interessant, oder?

Im Kontext der Webentwicklung finden Sie auch generische Schriftfamilien. Diese sind sehr wichtig. Eine generische Schriftfamilie ist eine Standardschriftart, die Webbrowser verwenden, wenn die Schriftarten mit höherer Priorität nicht verfügbar sind. Um sicherzustellen, dass der Inhalt weiterhin lesbar ist, ersetzt der Browser die ursprüngliche Schriftart durch die passendste Schriftart, die auf der angegebenen generischen Schriftfamilie basiert. Einige der am häufigsten verwendeten sind:

- serif
- sans-serif
- monospace
- kursiv
- fantasy

Es gibt aber noch mehr Optionen zur Auswahl. Im folgenden Code haben wir drei alternative Schriftfamilien:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="specified-font">Example paragraph using specified fonts.</p>
<p>Paragraph not using specified fonts.</p>
```

```css
#specified-font {
  font-family: Arial, Lato, sans-serif;
}
```

:::

Arial hat die höchste Priorität. Wenn Arial nicht gefunden wird, versucht der Browser, Lato zu rendern. Wenn keine der beiden gefunden wird, verwendet der Browser die generische Schriftfamilie sans-serif und wählt eine Schriftart mit diesen Eigenschaften aus den auf dem System des Nutzers installierten Schriftarten aus.

Sie sollten immer eine generische Schriftfamilie am Ende der font-family-Liste angeben. Bedenken Sie, dass die generische Schriftart anders aussehen kann als die Schriftart, die Sie ursprünglich in Ihrem Design vorgesehen haben. Es ist daher immer hilfreich, zu überprüfen, wie diese Fallback-Schriftarten in verschiedenen Browsern aussehen.

Um die Benutzererfahrung so konsistent wie möglich zu gestalten, wird außerdem empfohlen, web-sichere Schriftarten zu verwenden. Diese Schriftfamilien sind normalerweise auf den meisten Geräten installiert, sodass sie sehr wahrscheinlich gefunden und korrekt gerendert werden. Mehr darüber lernen Sie in den nächsten Lektionen.

Schriftfamilien sind im Bereich Webdesign unverzichtbar. Mit der Eigenschaft font-family können Sie das Aussehen und Gefühl Ihres Textes sofort verändern.
