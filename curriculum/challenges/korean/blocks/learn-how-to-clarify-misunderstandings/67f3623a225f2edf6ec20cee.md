# --explanation--

`Supposed to be`는 어떤 일이 기대되거나 계획되었음을 의미합니다. 예를 들어:

`The new feature was supposed to be ready by last week.` — 이것은 지난주에 준비될 계획이었음을 뜻합니다.

`Coming later`는 어떤 일이 미래에 일어날 계획임을 의미합니다. 예를 들어:

`The update is coming later this month.` — 이것은 업데이트가 지금이 아니라 미래에 출시될 것임을 뜻합니다.
