# --description--

现在，在马克笔和它们所在的 `container` 元素的边缘之间添加更多的垂直空间。

在 `.container` CSS 规则中，使用简写 `padding` 属性添加上下内边距，各为 `10px`，并将左右内边距设置为 `0`。 这类似于你之前使用的简写 `margin` 属性。

# --hints--

你不应从 `.container` CSS 规则中删除 `background-color` 属性及其值。

```js
assert.strictEqual(new __helpers.CSSHelp(document).getStyle('.container')?.backgroundColor, 'rgb(0, 0, 0)');
```

你的 `.container` CSS 规则应该有一个 `padding` 属性，设置为 `10px 0`。

```js
assert.strictEqual(new __helpers.CSSHelp(document).getStyle('.container')?.padding, '10px 0px');
```
