# --explanation--

`must`와 `have to`는 모두 의무나 필요성을 나타내지만, 사용에 약간 차이가 있습니다.

`Must`는 강한 내부 의무나 공식적인 규칙에 사용됩니다. 보통 더 공식적이거나 권위 있게 들립니다. 예를 들어:

`All sensitive data must be encrypted.` - 엄격한 규칙이나 요구사항.

`Have to`는 외부 의무에 사용되며, 개인 권한보다는 규칙이나 상황에 기반합니다. 예를 들어:

`You have to ensure that the encryption keys are stored securely.` - 보안 정책에 따른 책임.
