# --description--

Eine `hard drive` ist eine Komponente in Computern, die verwendet wird, um Daten dauerhaft zu speichern. Man kann sie sich wie einen großen Aktenschrank vorstellen, in dem alle digitalen Dateien aufbewahrt werden. Zum Beispiel, wenn Sie Dokumente, Fotos oder Musik auf Ihrem Computer speichern, werden diese auf der `hard drive` abgelegt.

Sie unterscheidet sich vom temporären Speicher, der wie ein Schreibtisch ist, auf dem Dinge nur für kurze Zeit während der Nutzung aufbewahrt werden.
