# --description--

Python에서 변수는 다양한 유형의 데이터를 저장하고 참조하는 레이블이 붙은 상자와 같습니다. Python에서 변수를 선언하려면 대입 연산자(`=`)를 사용해 식별자에 값을 할당하세요. JavaScript의 `let` 또는 `const`, C#의 `char` 같은 특별한 키워드를 사용할 필요가 없습니다.

Python에서는 변수 이름을 왼쪽에 쓰고, 대입 연산자를 쓴 다음, 변수에 할당할 값을 오른쪽에 씁니다. 다음은 `name`와 `age` 변수를 선언하는 예입니다:

```python
name = 'John Doe'
age = 25
```

위 예에서 변수 `name`는 값 `'John Doe'`을 저장합니다. 이 값은 문자열로, 텍스트를 나타내는 문자들의 연속입니다. 문자열은 작은따옴표나 큰따옴표로 작성하며, 예를 들어 `'Hello'` 또는 `"Hello"`처럼 씁니다. 앞으로 수업에서 Python에서 문자열을 다루는 방법을 더 배우게 됩니다.

Python에서 변수 이름을 지을 때 기억해야 할 중요한 규칙들이 있습니다:

- 변수 이름은 숫자가 아닌 문자나 밑줄(`_`)로 시작해야 합니다.
- 변수 이름에는 영숫자(`a-z`, `A-Z`, `0-9`)와 밑줄(`_`)만 포함할 수 있습니다.
- 변수 이름은 대소문자를 구분합니다 — `age`, `Age`, `AGE`는 모두 서로 다른 이름으로 간주됩니다.
- 변수 이름은 `if`, `class`, `def` 같은 Python 예약어일 수 없습니다.

이 규칙 중 하나라도 어기면 Python 프로그램에서 `SyntaxError`가 발생합니다:

```bash
    5variable_name = 5
     ^
SyntaxError: invalid syntax
```

이제 Python에서 변수 이름을 짓는 일반적인 관례를 살펴보겠습니다.

먼저, 변수 이름은 소문자로 작성하고 단어 사이를 밑줄로 구분해야 합니다. 이것을 스네이크 케이스라고 합니다:

```python
my_variable_name = 'freeCodeCamp'
```

다음으로, 변수 이름은 설명적인 이름을 사용하세요. 예를 들어 사용자의 나이를 변수로 저장하려면 `user_age`가 `age`이나 `ua` 같은 약어보다 낫습니다:

```python
user_age = 30
```

이렇게 하면 큰 코드베이스에서 다른 팀원(또는 미래의 자신)에게 변수의 목적을 쉽게 전달할 수 있습니다.

또 다른 관례는 한 글자 변수 이름을 피하는 것입니다. Python에서 매우 흔하지만, 한 글자 변수 이름은 목적이나 의미를 전달하지 못하므로 피해야 합니다:

```python
x = 56 # What do you mean by x?
```

파운드 기호(`#`)와 그 뒤에 오는 텍스트는 위 예에서 주석입니다. 주석에 익숙할 수도 있으니 간단히 설명하고 어떻게 작동하는지 알아보겠습니다.

Python에서 주석은 파운드 기호(`#`)로 시작하며, 그 줄에서 `#` 기호 뒤의 모든 내용은 무시됩니다:

```python
# This is a single-line comment
```

여러 줄 주석은 연속된 한 줄 주석을 사용해 만들 수 있습니다:

```python
# This is a
# multi-line
# comment
```

주석은 코드를 설명하거나 자신을 위한 메모를 남기거나 특정 코드 줄이 왜 있는지 명확히 할 때 유용합니다. 특히 학습하거나 팀으로 작업할 때 도움이 됩니다.

하지만 변수 이름이 의미하는 바를 설명하기 위해 주석을 사용해서는 안 됩니다. 대신, 변수 이름 자체가 설명적이어야 하며, 앞서 언급한 다른 이름 규칙을 따라 구문 오류를 방지해야 합니다.
