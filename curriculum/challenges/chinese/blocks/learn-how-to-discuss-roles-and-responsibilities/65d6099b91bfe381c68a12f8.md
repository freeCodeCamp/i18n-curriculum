# --description--

单词 `seriously` 是一个副词，表示以大量思考和注意力去做某事，表明这对你很重要。它是形容词 `serious` 的副词形式。

例如，当 `someone is serious about a project` 时，他们认为这是非常重要的。如果 `they work seriously`，意味着他们工作时付出了很多努力和关心。

要将形容词变成副词，通常在形容词末尾添加 `-ly`。当形容词以 `y` 结尾时，需要将 `y` 改为 `i` 并添加 `-ly`。例如，`happy` 变成 `happily`。
