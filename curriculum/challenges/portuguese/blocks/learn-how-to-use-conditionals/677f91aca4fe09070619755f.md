# --explanation--

`Power cable` refere-se ao fio que fornece eletricidade para o computador:

`Power` significa energia ou eletricidade e `cable` é o fio ou cabo usado para transferi-la.Por exemplo:

- `The server shut down because the power cable was disconnected.` - O servidor desligou quando o cabo de energia foi desconectado.

- `Check if the power cable is properly connected before troubleshooting.` - Certifique-se de que o cabo de alimentação está conectado antes de resolver outros problemas.
