Estas dos palabras se refieren a conectar las partes internas de un sistema o software. La primera es una palabra compuesta, recuerda usar `-`.

# --explanation--

`Handle` significa hacerse responsable o gestionar algo. Por ejemplo:

`She will handle all client emails today.` – Esto significa que ella se encargará de esa tarea.

La `Back-end integration` se refiere a conectar y coordinar los sistemas internos de una aplicación para que funcionen juntos. Por ejemplo:

`We need back-end integration to link the database with the server.` – Esto significa conectar las partes internas de la tecnología.
