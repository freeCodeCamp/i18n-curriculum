# --description--

`constant` у програмуванні схожий на змінну, але після того, як йому призначено значення, це значення не можна змінити. `Constants` використовують, коли у вас є значення, які, як ви знаєте, не потрібно буде змінювати протягом програми. Їх часто записують великими літерами, щоб відрізнити від змінних.

Наприклад, `const PI = 3.14;` У багатьох мовах програмування `const` використовується для оголошення константи. Тут `PI` — це `constant` із призначеним значенням `3.14`.
