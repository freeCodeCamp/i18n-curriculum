# --description--

En programación, el término `persistent` a menudo se refiere a algo que continúa existiendo o ocurriendo, especialmente un problema o un error que es difícil de resolver.

`Persistence` es la forma sustantiva, que se refiere al estado de ser persistente.

Por ejemplo, un `persistent error` en el código es un error que sigue ocurriendo y es difícil de arreglar.
