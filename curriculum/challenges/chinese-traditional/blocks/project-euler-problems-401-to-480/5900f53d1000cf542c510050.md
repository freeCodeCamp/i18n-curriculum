# --description--

多邊形的（作業系統）核心定義為從該點集可以看到整個多邊形邊界的所有點集合。我們將極性多邊形定義為原點嚴格包含在其（作業系統）核心內的多邊形。

對於此問題，多邊形可以有共線的連續頂點。然而，多邊形仍然不能有自我交叉且不能有零面積。

例如，以下只有第一個是極性多邊形（第二、第三和第四個的 `kernels` 並未嚴格包含原點，而第五個根本沒有 `kernel`）：

<img alt="five example polygons" src="https://cdn.freecodecamp.org/curriculum/project-euler/polar-polygons.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

請注意第一個多邊形有三個連續共線的頂點。

設 $P(n)$ 為極多邊形的數量，使得頂點 $(x, y)$ 具有絕對值不大於 $n$ 的整數座標。

請注意，如果多邊形的邊集合不同，即使它們包圍相同的區域，也應視為不同的多邊形。 例如，頂點為 [(0,0), (0,3), (1,1), (3,0)] 的多邊形與頂點為 [(0,0), (0,3), (1,1), (3,0), (1,0)] 的多邊形是不同的。

例如，$P(1) = 131$、$P(2) = 1\\,648\\,531$、$P(3) = 1\\,099\\,461\\,296\\,175$ 和 $P(343)\bmod 1\\,000\\,000\\,007 = 937\\,293\\,740$。

尋找 $P(7^{13})\bmod 1\\,000\\,000\\,007$。
