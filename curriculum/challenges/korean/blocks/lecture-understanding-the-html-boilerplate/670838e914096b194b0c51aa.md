# --description--

HTML 보일러플레이트에 대해 배워봅시다.

HTML 보일러플레이트가 무엇인지 궁금하신가요? 웹페이지를 위한 준비된 템플릿과 같습니다. 집의 기초라고 생각하세요. 보일러플레이트는 모든 HTML 문서에 필요한 기본 구조와 필수 요소를 포함합니다. 시간을 절약해주고 페이지가 제대로 설정되도록 도와줍니다. 예시는 다음과 같습니다:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>freeCodeCamp</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
  </body>
</html>
```

이 보일러플레이트의 주요 부분을 살펴봅시다. 먼저 `DOCTYPE` 선언이 있습니다:

```html
<!DOCTYPE html>
```

이 선언은 브라우저에 어떤 HTML 버전을 사용하는지 알려줍니다. 다음은 `html` 태그입니다:

```html
<!DOCTYPE html>
<html lang="en">
  <!--All other elements go inside here-->
</html>
```

이 태그는 모든 콘텐츠를 감싸며, 페이지의 언어를 지정할 수 있습니다. `html` 태그 안에는 두 개의 주요 섹션이 있습니다 - `head`와 `body`:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <!--Important metadata goes here-->
  </head>
  <body>
    <!--Headings, paragraphs, images, etc. go inside here-->
  </body>
</html>
```

`head` 섹션에는 중요한 백그라운드 정보가 들어 있습니다:

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Document Title Goes Here</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

사이트의 메타데이터인 `meta` 요소에는 문자 인코딩, 그리고 Twitter 같은 웹사이트가 페이지 링크를 어떻게 미리보기할지에 관한 정보가 포함되어 있습니다. 사이트 제목은 `title` 요소에 있으며, 브라우저 탭이나 창에 표시되는 텍스트를 결정합니다. 마지막으로, `head` 섹션에서는 `link` 요소를 사용해 외부 스타일시트를 연결하는 것이 일반적입니다.

`body` 섹션은 모든 콘텐츠가 들어가는 곳입니다:

```html
<body>
  <h1>I am a main title</h1>
  <p>Example paragraph text</p>
</body>
```

그렇다면 보일러플레이트가 왜 중요할까요? 페이지가 올바르게 구조화되고 다양한 브라우저에서 잘 작동하도록 보장해줍니다. 보일러플레이트를 사용하면 흔히 발생하는 실수를 피하고 모범 사례를 따를 수 있습니다. 모든 웹 프로젝트의 훌륭한 출발점입니다. 필요에 맞게 자신만의 보일러플레이트를 맞춤 설정할 수도 있습니다. 경험이 쌓이면 선호하는 요소나 `meta` 태그를 추가할 수도 있습니다. 개인 보일러플레이트를 계속 개선하면 새 프로젝트를 시작할 때 시간을 절약할 수 있습니다.

다음에 새 HTML 파일을 시작할 때는 보일러플레이트를 사용하는 것을 고려하세요. 확실히 견고한 기반을 제공할 것입니다.
