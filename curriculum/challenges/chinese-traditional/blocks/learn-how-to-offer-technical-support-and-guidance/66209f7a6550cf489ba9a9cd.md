# --description--

`To handle` 某事物意指處理或管理它。在程式設計中，`handling` 涉及管理資料或操作行為。<dfn>例如</dfn>，`to handle user input means to manage what happens when a user enters data`。

`Error handling` 是處理程式執行時發生的錯誤。它與撰寫能夠管理這些錯誤而不會當機的程式碼有關。<dfn>例如</dfn>，`using try-except blocks in Python to handle errors that might occur when a program runs`。
