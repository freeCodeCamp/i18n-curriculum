# --description--

In Python sind Dictionaries eingebaute Datenstrukturen, die Sammlungen von Schlüssel-Wert-Paaren speichern. Sie funktionieren sehr ähnlich wie echte Wörterbücher, bei denen Sie nach einem Wort suchen, um seine entsprechende Bedeutung zu finden.

Mit Python-Dictionaries verwenden Sie einen Schlüssel, um den dazugehörigen Wert zu finden. Sie sollten Dictionaries verwenden, wenn Sie Werte mit eindeutigen Schlüsseln verknüpfen müssen. Das ist hilfreich, wenn Sie einen Wert schnell anhand des Schlüssels finden müssen und wenn Sie strukturierte Daten darstellen wollen.

Dies ist die allgemeine Syntax eines Python-Dictionaries:

```python
dictionary = {
    key1: value1,
    key2: value2
}
```

Zuerst finden wir die Variable, die das Dictionary enthält. Es ist nicht zwingend erforderlich, das Dictionary einer Variable zuzuweisen, aber es ist sehr üblich, dies zu tun, um es im Speicher zu behalten und später im Code zu verwenden.

Darauf folgen geschweifte Klammern, die manchmal auch geschweifte Brackets genannt werden. Innerhalb der geschweiften Klammern befinden sich Schlüssel-Wert-Paare.

Jeder Schlüssel ist mit einem Wert verknüpft, sodass Sie den Schlüssel verwenden können, um auf diesen Wert zuzugreifen. Nach jedem Wert, außer dem letzten, steht ein Komma, um die verschiedenen Schlüssel-Wert-Paare zu trennen. Schlüssel müssen im Dictionary eindeutig sein und einen unveränderlichen Datentyp haben. Die Werte können jedoch wiederholt werden und können jeden Datentyp haben.

Hier haben wir ein Beispiel für ein Dictionary, das Informationen über ein Margherita-Pizza-Rezept speichert:

```python
pizza = {
    'name': 'Margherita Pizza',
    'price': 8.9,
    'calories_per_slice': 250,
    'toppings': ['mozzarella', 'basil']
}
```

Das Dictionary ist der `pizza`-Variable zugewiesen. Es enthält vier Schlüssel-Wert-Paare: `name`, `price`, `calories_per_slice` und `toppings`.

Eine weitere Alternative wäre die Verwendung des `dict()`-Konstruktors, der das Dictionary aus einer Folge von Schlüssel-Wert-Paaren erstellt.

Dies wäre die äquivalente Syntax für unser Pizza-Beispiel. Wir übergeben eine Liste von Tuples als Argument an den `dict()`-Konstruktor. Diese Tuples enthalten den Schlüssel als erstes Element und den Wert als zweites Element.

```python
pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
```

Um auf den Wert eines Schlüssel-Wert-Paares zuzugreifen, können Sie diese Syntax verwenden, die als Klammernotation bekannt ist. Es ist der Name der Variable, die das Dictionary enthält, gefolgt von eckigen Klammern, und dem Schlüssel, auf den Sie innerhalb der eckigen Klammern zugreifen möchten:

```python
dictionary[key]
```

In unserem Pizza-Beispiel, wenn Sie auf den Wert von `name` zugreifen möchten, schreiben Sie den Namen der Variable, `pizza`, gefolgt von eckigen Klammern, und den Schlüssel `name` in Anführungszeichen:

```python
pizza['name']
```

Das ergibt:

```python
'Margherita Pizza'
```

Um einen Wert zu aktualisieren, müssen Sie nur den Zuweisungsoperator hinzufügen, gefolgt vom neuen Wert.

Wenn der Schlüssel im Dictionary nicht existiert, wird ein neues Schlüssel-Wert-Paar erstellt. In neueren Python-Versionen behalten Dictionaries die Einfügereihenfolge bei. Das ist hilfreich, wenn Sie das Dictionary durchlaufen müssen:

```python
pizza['name'] = 'Margherita'
```

Jetzt ist der Wert des Schlüssels `name` `'Margherita'`:

```python
print(pizza['name']) # 'Margherita'
```

Dictionaries haben auch hilfreiche Methoden, um häufige Operationen durchzuführen.

Die `.get()`-Methode ruft den Wert ab, der mit einem Schlüssel verknüpft ist. Sie ist ähnlich wie die gerade verwendete Klammernotation, aber ihr Vorteil ist, dass Sie einen Standardwert festlegen können, sodass kein Fehler auftritt, wenn der Schlüssel nicht existiert:

```python
dictionary.get(key, default)
```

In diesem Beispiel, wenn der Schlüssel `toppings` nicht existiert, wird eine leere Liste zurückgegeben, die der Standardwert ist, den wir hier als zweiten Parameter übergeben. Wenn `toppings` jedoch existiert, wird dieser Wert zurückgegeben:

```python
pizza.get('toppings', []) # ['mozzarella', 'basil']
```

Die `.keys()`- und `.values()`-Methoden geben jeweils ein View-Objekt mit allen Schlüsseln und Werten im Dictionary zurück:

```python
pizza.keys()
# dict_keys(['name', 'price', 'calories_per_slice'])

pizza.values()
# dict_values(['Margherita Pizza', 8.9, 250])
```

Ein View-Objekt ist einfach eine Möglichkeit, den Inhalt eines Dictionaries zu sehen, ohne eine separate Kopie der Daten zu erstellen.

Die `.items()`-Methode gibt ein View-Objekt mit allen Schlüssel-Wert-Paaren im Dictionary zurück, einschließlich der Schlüssel und Werte:

```python
pizza.items()
# dict_items([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250)])
```

Die `.clear()`-Methode entfernt alle Schlüssel-Wert-Paare aus dem Dictionary:

```python
pizza.clear()
```

Die `.pop()`-Methode entfernt das Schlüssel-Wert-Paar mit dem Schlüssel, den Sie als erstes Argument angeben, und gibt dessen Wert zurück. Wenn der Schlüssel nicht existiert, wird der Standardwert zurückgegeben, den Sie als zweiten Parameter angeben. Wenn der Schlüssel nicht existiert und Sie keinen Standardwert angeben, wird ein `KeyError` ausgelöst:

```python
pizza.pop('price', 10)
pizza.pop('total_price') # KeyError
```

In Python 3.7 und neueren Versionen entfernt die `.popitem()`-Methode das zuletzt eingefügte Element:

```python
pizza.popitem()
```

Und schließlich aktualisiert die `.update()`-Methode die Schlüssel-Wert-Paare mit den Schlüssel-Wert-Paaren eines anderen Dictionaries. Wenn sie gemeinsame Schlüssel haben, werden deren Werte überschrieben.

In diesem Beispiel aktualisieren wir das `pizza`-Dictionary. Der Schlüssel `price` existiert in beiden, daher wird sein Wert durch `15` ersetzt.

Aber `total_time` ist neu, daher wird es als neues Schlüssel-Wert-Paar zum `pizza`-Dictionary hinzugefügt:

```python
pizza.update({ 'price': 15, 'total_time': 25 })
```

Dies ist das neue Dictionary mit dem aktualisierten `price` und dem neuen `total_time`. Beachten Sie, wie der Preis jetzt `15` ist und `total_time` ein neues Schlüssel-Wert-Paar ist:

```python
{
    'name': 'Margherita Pizza', 
    'price': 15, 
    'calories_per_slice': 250, 
    'toppings': ['mozzarella', 'basil'], 
    'total_time': 25
}
```

Dies sind einige der am häufigsten verwendeten Dictionary-Methoden, aber es gibt noch viele weitere. Die richtige Wahl kann hilfreich sein, um komplexe Operationen effizient durchzuführen.
