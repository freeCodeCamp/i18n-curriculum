# --description--

**目的:** 以下のストーリーを満たし、すべてのテストに合格してラボを完了してください。

**ストーリー:**

1. `pyramid` という名前の関数を作成し、3つの引数を受け取る必要があります。
1. 最初の引数は、ピラミッドで繰り返すパターン文字を表す文字列である必要があります。
1. 2番目の引数は、ピラミッドの行数を表す整数である必要があります。
1. 3番目の引数はブール値である必要があります。
1. `pyramid` 関数は、パターン文字が繰り返されて配置され、3番目の引数が `false` の場合に頂点が上向きのピラミッドを形成する文字列を返す必要があります。
1. 3番目の引数が `true` の場合、ピラミッドは頂点が下向きになります。
1. 頂点の行はパターン文字が1つだけで、他の各行は前の行より2つ多いパターン文字を持つ必要があります。
1. 各行は、各行の中央の文字が同じ列に来るように十分な数のスペースで始まり、行末にスペースがあってはいけません。
1. ピラミッドは改行文字で始まり、改行文字で終わる必要があります。

例えば、`pyramid("o", 4, false)` を呼び出すと次の出力が得られます:

```js

   o
  ooo
 ooooo
ooooooo

```

function pyramid(char, count, isInverted) {
    const rows = []
    for (let i = 1; i <= count; i++) {
        if (isInverted) {
            rows.unshift(" ".repeat(count - i) + char.repeat(2 * i - 1))
        } else {
            rows.push(" ".repeat(count - i) + char.repeat(2 * i - 1))
        }        
    }
    return "\n" + rows.join("\n") + "\n";
}

console.log(pyramid("#", 10, false))
