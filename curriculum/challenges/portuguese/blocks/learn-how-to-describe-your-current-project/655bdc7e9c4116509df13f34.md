# --description--

Usar `use` significa colocar algo para um propósito específico.Por exemplo:

`She uses a notebook to take notes in class.` - Isso mostra como ela usa o notebook para escrever.

Uma `tool` é algo que ajuda você a fazer um trabalho.No trabalho, as ferramentas geralmente são software ou coisas no computador.Por exemplo:

`Zoom is a tool for online meetings.` - Neste caso, a ferramenta é um software que ajuda as pessoas a se conectarem e conversarem.
