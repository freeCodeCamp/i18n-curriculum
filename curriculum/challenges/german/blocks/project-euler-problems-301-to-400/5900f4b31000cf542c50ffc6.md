# --description--

Eine Reihe von drei Räumen ist durch automatische Türen miteinander verbunden.

<img alt="Reihe von drei Räumen, die durch automatische Türen miteinander verbunden sind" src="https://cdn.freecodecamp.org/curriculum/project-euler/rooms-of-doom.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Jede Tür wird mit einer Sicherheitskarte bedient. Sobald Sie einen Raum betreten, schließt sich die Tür automatisch, und diese Sicherheitskarte kann nicht erneut verwendet werden. Eine Maschine gibt zu Beginn eine unbegrenzte Anzahl von Karten aus, aber jeder Raum (einschließlich des Startzimmers) enthält Scanner. Wenn sie feststellen, dass Sie mehr als drei Sicherheitskarten bei sich tragen oder wenn sie eine unbeaufsichtigte Sicherheitskarte auf dem Boden entdecken, werden alle Türen dauerhaft verriegelt. Allerdings enthält jeder Raum eine Box, in der Sie eine beliebige Anzahl von Sicherheitskarten sicher für eine spätere Verwendung aufbewahren können.

Wenn Sie einfach versuchen würden, die Räume nacheinander zu durchqueren, hätten Sie beim Betreten von Raum 3 alle drei Karten verbraucht und wären für immer in diesem Raum gefangen!

Wenn Sie jedoch die Aufbewahrungsboxen nutzen, ist eine Flucht möglich. Zum Beispiel könnten Sie Raum 1 mit Ihrer ersten Karte betreten, eine Karte in der Aufbewahrungsbox ablegen und Ihre dritte Karte verwenden, um den Raum wieder zu verlassen und zum Start zurückzukehren. Nachdem Sie dann drei weitere Karten von der Ausgabemaschine gesammelt haben, könnten Sie eine verwenden, um Raum 1 erneut zu betreten und die Karte abzuholen, die Sie vorhin in der Box abgelegt haben. Sie haben jetzt wieder drei Karten und können durch die verbleibenden drei Türen reisen. Mit dieser Methode können Sie alle drei Räume mit insgesamt sechs Sicherheitskarten durchqueren.

Es ist möglich, sechs Räume mit insgesamt 123 Sicherheitskarten zu durchqueren, während Sie maximal 3 Karten bei sich tragen.

Sei $C$ die maximale Anzahl von Karten, die zu jedem Zeitpunkt getragen werden können.

Sei $R$ die Anzahl der zu durchquerenden Räume.

Sei $M(C, R)$ die minimale Anzahl von Karten, die von der Ausgabemaschine benötigt werden, um $R$ Räume zu durchqueren, wobei maximal $C$ Karten zu jedem Zeitpunkt getragen werden.

Zum Beispiel gilt $M(3, 6) = 123$ und $M(4, 6) = 23$.

Und, $\sum M(C, 6) = 146$ für $3 ≤ C ≤ 4$.

Es ist gegeben, dass $\sum M(C, 10) = 10382$ für $3 ≤ C ≤ 10$.

Finden Sie $\sum M(C, 30)$ für $3 ≤ C ≤ 40$.
