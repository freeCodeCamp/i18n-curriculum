# --description--

في البرمجة، `int` تعني "عدد صحيح". هو نوع بيانات يُستخدم لتخزين الأعداد الصحيحة، وهي الأعداد التي لا تحتوي على أجزاء كسرية أو أرقام عشرية. يمكن أن تكون الأعداد الصحيحة موجبة أو سالبة أو صفرًا.

اختيار نوع البيانات `int` ضروري عند التعامل مع الأعداد الصحيحة، مثل العد، الفهارس في المصفوفات، أو أي حالة لا تحتاج إلى أعداد كسرية.

أمثلة:

عد العناصر: `int itemCount = 5;`

تخزين العمر: `int age = 30;`
