# --description--

양수 $n$에 대해, $S(n)$을 $1 < x < n$ 이고 $x^3 \equiv 1\bmod n$인 정수 $x$들의 합으로 정의하세요.

$n = 91$일 때, 가능한 $x$ 값은 8개로 9, 16, 22, 29, 53, 74, 79, 81입니다. 따라서 $S(91) = 9 + 16 + 22 + 29 + 53 + 74 + 79 + 81 = 363$입니다.

$S(13\\,082\\,761\\,331\\,670\\,030)$을 구하세요.
