# --interactive--

Funktionen sind wiederverwendbare Codeabschnitte, die eine bestimmte Aufgabe ausführen oder einen Wert berechnen. Stellen Sie sich Funktionen wie eine Maschine vor, die eine Eingabe erhält, einige Operationen darauf ausführt und dann eine Ausgabe erzeugt. Hier ist ein Beispiel für das Deklarieren einer Funktion:

```js
function greet() {
  console.log("Hello, Jessica!");
}
```

In diesem Beispiel haben wir eine Funktion namens `greet` deklariert. Innerhalb dieser Funktion haben wir eine `console.log`, die die Nachricht `Hello, Jessica!` in der Konsole ausgibt. Wenn wir versuchen würden, diesen Code auszuführen, würden wir die Nachricht nicht in der Konsole sehen. Das liegt daran, dass wir die Funktion aufrufen müssen.

Ein Funktionsaufruf oder eine Invocation ist der Moment, in dem wir die Funktion tatsächlich verwenden oder ausführen. Um eine Funktion aufzurufen, müssen Sie den Funktionsnamen gefolgt von einem Satz runder Klammern angeben:

:::interactive_editor

```js
function greet() {
  console.log("Hello, Jessica!");
}

greet(); // "Hello, Jessica!"
```

:::

Jetzt wird die Nachricht `Hello, Jessica!` in der Konsole ausgegeben. Aber was, wenn wir möchten, dass die Nachricht `Hello, Nick!` oder `Hello, Anna!` sagt? Wir wollen nicht jedes Mal eine neue Funktion schreiben, wenn wir einen anderen Benutzer begrüßen. Stattdessen können wir eine wiederverwendbare Funktion erstellen, die Funktionsparameter und Argumente verwendet.

Parameter fungieren als Platzhalter für die Werte, die an die Funktion übergeben werden, wenn sie aufgerufen wird. Sie ermöglichen es Funktionen, Eingaben zu akzeptieren und mit diesen Eingaben zu arbeiten. Argumente sind die tatsächlichen Werte, die an die Funktion übergeben werden, wenn sie aufgerufen wird. Hier ist eine aktualisierte Version der `greet`-Funktion, die Parameter und Argumente verwendet:

:::interactive_editor

```js
function greet(name) {
  console.log("Hello, " + name + "!");
}

greet("Alice"); // Hello, Alice!
greet("Nick"); // Hello, Nick!
```

:::

Der `name` dient als Parameter, während die Strings `Alice` und `Nick` als Argumente dienen. Jetzt haben wir eine wiederverwendbare Funktion, die im Code mit verschiedenen Argumenten dutzende Male verwendet werden kann.

Wenn eine Funktion ihre Ausführung beendet, gibt sie immer einen Wert zurück. Standardmäßig ist der Rückgabewert `undefined`. Hier ist ein Beispiel:

:::interactive_editor

```js
function doSomething() {
  console.log("Doing something...");
}

let result = doSomething();
console.log(result); // undefined
```

:::

Wenn Ihre Funktion einen bestimmten Wert zurückgeben soll, müssen Sie die `return`-Anweisung verwenden. Hier ist ein Beispiel für die Verwendung einer `return`-Anweisung, um die Summe von zwei Werten zurückzugeben:

:::interactive_editor

```js
function calculateSum(num1, num2) {
  return num1 + num2;
}

console.log(calculateSum(3, 4)); // 7
```

:::

Oft werden Sie die `return`-Anweisung verwenden, weil Sie den Wert, der von der Funktion ausgegeben wurde, später im Code verwenden können.

Bisher haben wir mit benannten Funktionen gearbeitet, aber Sie können auch eine sogenannte anonyme Funktion erstellen. Eine anonyme Funktion ist eine Funktion ohne Namen, die einer Variablen zugewiesen werden kann, wie hier:

:::interactive_editor

```js
const sum = function (num1, num2) {
  return num1 + num2;
};

console.log(sum(3, 4)); // 7
```

:::

In diesem Beispiel haben wir eine `const`-Variable namens `sum` und weisen ihr eine anonyme Funktion zu, die die Summe von `num1` und `num2` zurückgibt. Dann können wir `sum` aufrufen und die Zahlen `3` und `4` übergeben, um das Ergebnis `7` zu erhalten.

Funktionen unterstützen Standardparameter, mit denen Sie Standardwerte für Parameter festlegen können. Diese Standardwerte werden verwendet, wenn die Funktion ohne Argument für diesen Parameter aufgerufen wird. Hier ist ein Beispiel:

:::interactive_editor

```js
function greetings(name = "Guest") {
  console.log("Hello, " + name + "!");
}

greetings(); // Hello, Guest!
greetings("Anna"); // Hello, Anna!
```

:::

In diesem Beispiel wird, wenn kein Argument für `name` angegeben wird, standardmäßig `Guest` verwendet.

Zusammenfassend ermöglichen Funktionen das Schreiben von wiederverwendbarem und organisiertem Code. Sie können Eingaben (Parameter) entgegennehmen, Aktionen ausführen und Ausgaben zurückgeben.
