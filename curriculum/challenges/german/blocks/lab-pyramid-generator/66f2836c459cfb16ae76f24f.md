# --description--

**Ziel:** Erfüllen Sie die untenstehenden User Stories und bestehen Sie alle Tests, um das Labor abzuschließen.

**User Stories:**

1. Sie sollten eine Funktion namens `pyramid` haben, die drei Argumente entgegennimmt.
1. Das erste Argument sollte ein String sein, der das Musterzeichen darstellt, das in Ihrer Pyramide wiederholt wird.
1. Das zweite Argument sollte eine ganze Zahl sein, die die Anzahl der Reihen in der Pyramide angibt.
1. Das dritte Argument sollte ein Boolean-Wert sein.
1. Die Funktion `pyramid` sollte einen String zurückgeben, in dem das Musterzeichen wiederholt und so angeordnet wird, dass eine Pyramide mit der Spitze nach oben entsteht, wenn das dritte Argument `false` ist.
1. Wenn das dritte Argument `true` ist, sollte die Pyramide die Spitze nach unten zeigen.
1. Die Spitzenreihe sollte ein einzelnes Musterzeichen haben, und jede weitere Reihe sollte zwei Musterzeichen mehr als die vorherige enthalten.
1. Jede Reihe sollte mit einer Anzahl von Leerzeichen beginnen, die ausreicht, um das mittlere Zeichen jeder Reihe in derselben Spalte auszurichten, und am Ende jeder Reihe sollten keine Leerzeichen stehen.
1. Die Pyramide sollte mit einem Zeilenumbruch beginnen und enden.

Zum Beispiel sollte der Aufruf von `pyramid("o", 4, false)` folgende Ausgabe erzeugen:

```js

   o
  ooo
 ooooo
ooooooo

```

function pyramid(char, count, isInverted) {
    const rows = []
    for (let i = 1; i <= count; i++) {
        if (isInverted) {
            rows.unshift(" ".repeat(count - i) + char.repeat(2 * i - 1))
        } else {
            rows.push(" ".repeat(count - i) + char.repeat(2 * i - 1))
        }        
    }
    return "\n" + rows.join("\n") + "\n";
}

console.log(pyramid("#", 10, false))
