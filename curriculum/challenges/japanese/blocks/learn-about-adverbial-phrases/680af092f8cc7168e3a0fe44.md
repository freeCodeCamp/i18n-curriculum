# --explanation--

`To cost` は支払いを必要とする、または価格があることを意味します。`costs more than expected` と言うと、予定より多くのお金を支払わなければならないことを意味します。例えば：

`The event cost more than we budgeted for because of extra equipment.` – これは最終的な費用が当初の計画より高かったことを意味します。
