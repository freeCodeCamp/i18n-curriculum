# --interactive--

JavaScriptは、加算、減算、乗算、除算などの基本的な算術演算を数値に対して行うためのツールを提供しています。JavaScriptには、余りやべき乗など、より複雑な算術演算のための演算子も含まれています。

これらすべてのツールは算術演算子と呼ばれます。これらの演算子を詳しく見ていき、使い方や組み合わせ方を学びましょう。

加算演算子はプラス記号（`+`）です。加算演算子を使うと、2つ以上の数の合計を求められます。加算では、数の順序は関係ありません。

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 + num2;
const result2 = num2 + num1;
const result3 = num2 + num1 + num3;

console.log(result1); // 15
console.log(result2); // 15
console.log(result3); // 30
```

:::

減算演算子はマイナス記号（`-`）です。2つの数の差を求めるために使います。通常、大きい数から小さい数を引く場合にマイナス記号を使います。

:::interactive_editor

```js
const difference = 10 - 5;
console.log(difference); // 5
```

:::

もし小さい数が先に来ると、結果は負の値になります。

:::interactive_editor

```js
const difference = 5 - 10;
console.log(difference); // -5
```

:::

数値を変数に代入して、変数名を使って減算することもできます。

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const result = num1 - num2;

console.log(result); // 5
```

:::

JavaScriptでは、乗算演算子はアスタリスク（`*`）で表され、2つ以上の数の積を求めるために使います。乗算する数の順序は関係ありません。

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 * num2;
const result2 = num2 * num1;
const result3 = num2 * num1 * num3;

console.log(result1); // 50
console.log(result2); // 50
console.log(result3); // 750
```

:::

JavaScriptの除算演算子はスラッシュ（`/`）で、従来の数学で使われる除算記号（`÷`）とは異なります。除算演算子を使って除算を行います。この場合、割る数と割られる数の順序は重要です。

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 / num2;
const result2 = num2 / num1;
const result3 = num2 / num1 / num3;

console.log(result1); // 2
console.log(result2); // 0.5
console.log(result3); // 0.03333333333333333
```

:::

ゼロで割ろうとすると、JavaScriptは`Infinity`を返すことに注意してください。

:::interactive_editor

```js
const result = 10 / 0; 

console.log(result); // Infinity
```

:::

コードで予期しない結果にならないように、このような計算は避けてください。

余り演算子はパーセント記号（`%`）で表され、除算の余りを返します。数学での余りとは、除算を行った後に残る値のことです。

:::interactive_editor

```js
const num1 = 10;
const num2 = 3;
const remainder = num1 % num2;

console.log(remainder); // 1
```

:::

べき乗演算子はダブルアスタリスク（`**`）で表され、ある数を別の数のべき乗にします。

:::interactive_editor

```js
const num1 = 2;
const num2 = 3;

const exponent = num1 ** num2;
console.log(exponent); // 8
```

:::

1つの演算や式の中で演算子を混ぜることも可能です。

:::interactive_editor

```js
const result = 10 + 5 * 2 - 8 / 4;
console.log(result); // 18
```

:::

異なる演算子を1つの式で混ぜる場合、JavaScriptエンジンは演算子の優先順位という仕組みに従って演算の順序を決定します。演算子の優先順位は、式の中でどの演算が先に実行されるかを決めます。演算子の優先順位については今後のレッスンで詳しく学びます。
