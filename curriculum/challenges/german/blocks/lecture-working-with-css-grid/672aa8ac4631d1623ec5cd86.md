# --interactive--

CSS Grid ist ein leistungsstarkes Layout-System, das Webentwicklern ermöglicht, komplexe und responsive Webseiten-Layouts mühelos zu erstellen.

Stellen Sie sich vor, Sie richten Möbel in einem Raum ein – CSS Grid ist wie ein unsichtbares Raster auf Ihrem Boden, das Ihnen hilft, alles genau dort zu platzieren, wo Sie es möchten.

Beim Erstellen von Webseiten müssen wir oft verschiedene Elemente auf der Seite anordnen.

Vor CSS Grid war das manchmal schwierig, besonders bei komplexen Layouts. CSS Grid vereinfacht diesen Prozess, indem es Ihre Webseite in Reihen und Spalten unterteilt und so eine rasterähnliche Struktur schafft.

Stellen Sie sich vor, Sie arbeiten mit einem Container `div`, in dem mehrere Elemente wie folgt verschachtelt sind:

:::interactive_editor

```html
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

:::

Wenn Sie diese Elemente im Rasterformat gestalten möchten, können Sie den `display` auf `grid` setzen und Spalten wie folgt anwenden:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 20px;
}

.item {
  background-color: lightgray;
  padding: 20px;
  text-align: center;
  border: 1px solid #ccc;
}
```

:::

In diesem Code sagen wir dem Browser, er soll ein Raster mit drei gleich breiten Spalten erstellen, das ist es, was `1fr 1fr 1fr` bedeutet, und wir fügen einen Abstand von 20 Pixeln zwischen jedem Grid-Element hinzu.

Jetzt fragen Sie sich vielleicht: „Wie sieht es mit Flexbox aus? Wird das nicht auch für Layouts verwendet?“

Das stimmt! Flexbox ist ein weiteres CSS-Layout-Modell und ebenfalls sehr nützlich. Aber es gibt einige wichtige Unterschiede.

Flexbox ist eindimensional, während Grid zweidimensional ist. Das bedeutet, Flexbox eignet sich hervorragend, um Dinge in einer einzigen Reihe oder Spalte anzuordnen, während Grid darin glänzt, Layouts mit Reihen und Spalten zu erstellen.

Flexbox ist inhaltsorientiert, das heißt, es passt das Layout basierend auf dem Inhalt an. Grid hingegen ist layoutorientiert, sodass Sie das Layout zuerst erstellen und dann Elemente darin platzieren können. Grid gibt Ihnen eine präzisere Kontrolle über die Platzierung. Sie können einem Element genau sagen, welche Reihe und Spalte es einnehmen soll.

Hier ist ein Flexbox-Beispiel zum Vergleich:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
}
```

:::

Dies erstellt einen Flex-Container, in dem die Elemente entlang der Hauptachse gleichmäßig verteilt sind.

Sowohl Grid als auch Flexbox haben ihre Stärken, und oft verwenden die besten Layouts eine Kombination aus beiden. Sie könnten Grid für das gesamte Seitenlayout verwenden und dann Flexbox, um Elemente innerhalb jedes Rasterbereichs auszurichten.

Zusammenfassend ist CSS Grid ein mächtiges Werkzeug, das präzise, zweidimensionale Layouts ermöglicht. Auch wenn es anfangs komplex erscheint, wird es mit Übung zu einem unverzichtbaren Werkzeug für die Erstellung responsiver und komplexer Weblayouts.
