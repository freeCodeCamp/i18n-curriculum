# --explanation--

`Build tools` ni programu zinazosaidia kuendesha na kusimamia kazi zinazojirudia, kama vile kukusanya, kujaribu, na kufunga miradi ya programu. Kwa mfano:

`Maven is a popular build tool for Java projects, helping automate the compilation process.` - Hii ina maana Maven hurahisisha mchakato wa kujenga programu za Java, kwa kuendesha kazi kama kukusanya msimbo na kutengeneza vifurushi.

`Built-in` inahusu vipengele vilivyo tayari sehemu ya programu na havihitaji usakinishaji au usanidi wa ziada. Kwa mfano:

`Visual Studio has built-in tools for debugging code.` - Hii ina maana zana za kurekebisha makosa tayari zipo ndani ya Visual Studio, na watumiaji hawahitaji kusakinisha zana zozote za ziada kuanza kurekebisha makosa.

`Version control` ni mfumo unaofuatilia na kusimamia mabadiliko ya msimbo, kuruhusu timu kufanya kazi kwenye mradi mmoja bila kupoteza kazi au kusababisha migongano. Kwa mfano:

`Git is a widely-used version control system for tracking code changes and collaborating with teammates.` - Hii ina maana Git husaidia wasanidi programu kusimamia matoleo tofauti ya msimbo na kushirikiana kwenye mradi mmoja.
