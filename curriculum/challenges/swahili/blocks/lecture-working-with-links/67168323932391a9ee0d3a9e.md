# --interactive--

Huenda umewahi kuona kiungo kwenye ukurasa wa mtandao kikageuka rangi ya zambarau baada ya kubofya. Hii ni kwa sababu hali ya kiungo imebadilika, hivyo mtindo tofauti unatumika. Kuna hali tano tofauti ambazo kiungo kinaweza kuwa nazo.

Ya kwanza ni hali ya chaguo-msingi, ambayo ni `:link`. Hali hii inawakilisha kiungo ambacho mtumizi hajatembelea, kubofya, au kuingiliana nacho bado. Unaweza kufikiria hali hii kama kutoa mitindo ya msingi kwa viungo vyote kwenye ukurasa wako. Hali zingine hujengwa juu yake.

Washia kiolesura cha kuingiliana na angalia kiungo kwenye dirisha la onyesho. Kinapaswa kuwa na rangi ya bluu ya chaguo-msingi. Ukibofya, kitatokea rangi ya zambarau.

:::interactive_editor

```html
<a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
```

:::

Hali ya pili ni `:visited`, ambayo hutumika wakati mtumizi tayari amewahi kutembelea ukurasa unaounganishwa. Kwa chaguo-msingi, hii hubadilisha kiungo kuwa rangi ya zambarau - lakini unaweza tumia CSS kutoa dalili tofauti za kuona kwa mtumizi. Hii ni msaada kumjulisha mtu kwamba tayari amesoma sehemu ya nyaraka zako. Au, kwamba tovuti ni ya kuaminika kwa sababu wameitumia hapo awali.

Bofya kiungo kwenye dirisha la onyesho na utaona kiungo kilichotembelewa kinabadilika kuwa rangi ya kahawia. Ili kuona onyesho, utahitaji kuwasha kiolesura cha kuingiliana.

**KUMBUKA**: Baadhi ya CSS imetolewa kwa mfano huu wa kuingiliana ili uone mabadiliko katika mitindo ya viungo. Usijali kuelewa msimbo wa CSS kwa sasa kwa sababu utajifunza maana yake katika moduli zijazo.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:visited {
  color: brown;
}
```

:::

Hali ya tatu ni `:hover`. Hali hii hutumika wakati mtumizi anapobeba kidole cha panya juu ya kiungo. Hali hii ni msaada kutoa makini zaidi kwa kiungo, kuhakikisha mtumizi anakusudia kubofya kweli.

Washia kiolesura cha kuingiliana na jaribu kubeba panya juu ya kiungo. Utaona rangi ikibadilika kuwa nyekundu.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:hover {
  color: red;
}
```

:::

Kisha tuna `:focus`. Hali hii hutumika tunapokuwa makini na kiungo.

Bofya sehemu yoyote ya nafasi tupu kwenye dirisha la onyesho kisha bonyeza `tab` kwenye kibodi yako. Utapona kuona kiungo kikibadilika kuwa kijani. Ili kuona onyesho, utahitaji kuwasha kiolesura cha kuingiliana.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:focus {
  color: green;
}
```

:::

Na hatimaye, tuna `:active`. Hali hii hutumika kwa viungo vinavyotekelezwa na mtumizi. Hii kwa kawaida inamaanisha kubofya kiungo kwa kitufe kikuu cha panya kwa kubofya kushoto, katika hali nyingi. Hali hii ni msaada kuonyesha mtumizi kuwa kipengele alichobofya ni cha kuingiliana.

Washia kiolesura cha kuingiliana na bofya kiungo. Utapona kuona kiungo kikibadilika kuwa nyeusi. Hii hutokea haraka sana hivyo huenda ukahitaji kubofya mara kadhaa kuona mabadiliko ya rangi.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:active {
  color: black;
}
```

:::

Unapotumia hali hizi kuunda mitindo ya viungo vyako, kuna mpangilio maalum wa kuandika CSS yako: `link`, `visited`, `hover`, `focus`, kisha `active`.

Sasa unapaswa kujua jinsi ya kutoa mtindo wako binafsi kwa viungo kwenye ukurasa wako, huku ukitoa taarifa muhimu mtumizi anazohitaji kuhusu hali ya kila kiungo.
