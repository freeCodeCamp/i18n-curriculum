# --description--

선택자를 사용하는 또 다른 방법은 구분 없이 리스트처럼 체이닝하는 것입니다. 다음과 같은 HTML이 있다고 가정해 보겠습니다.

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection preview">This is where a preview for a post might go.</p>
</div>
```

`subsection` 클래스를 가진 두 개의 요소가 각각 고유한 스타일을 가지고 있는데, 만약 두 번째 클래스가 `header`인 요소에만 별도의 규칙을 적용하고 싶다면 어떻게 할까요? CSS에서 두 `class` 선택자를 다음과 같이 체이닝할 수 있습니다.

```css
.subsection.header {
  color: red;
}
```

`.subsection.header`가 하는 일은 `subsection`와 `header` 클래스를 모두 가진 요소를 선택하는 것입니다. `.subsection`와 `.header` `class` 선택자 사이에 공백이 없다는 점을 주목하세요. 이 구문은 타입 선택자를 두 개 이상 체이닝하는 경우를 제외하고는 어떤 조합의 선택자도 체이닝하는 데 기본적으로 작동합니다.

아래와 같이 클래스와 ID를 체이닝하는 데에도 사용할 수 있습니다.

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection" id="preview">This is where a preview for a post might go.</p>
</div>
```

위의 두 요소를 다음과 같이 결합할 수 있습니다.

```css
.subsection.header {
  color: red;
}

.subsection#preview {
  color: blue;
}
```

일반적으로 요소가 한 번에 두 가지 다른 타입일 수 없기 때문에 타입 선택자를 두 개 이상 체이닝할 수 없습니다. 예를 들어, `div`와 `p` 같은 두 타입 선택자를 체이닝하면 `divp` 선택자가 되는데, 이는 실제로 존재하지 않는 `<divp>` 요소를 찾으려 하기 때문에 작동하지 않습니다.
