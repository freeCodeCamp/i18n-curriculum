# --explanation--

`That's a good suggestion, but...` — це спосіб визнати ідею, одночасно висловлюючи сумнів. Ця фраза дозволяє вести поважну дискусію, залишаючи відкритими питання для подальшого обговорення. Наприклад:

- **Розробник:** `How about we use this new software to improve productivity?`

- **Керівник команди:** `That's a good suggestion, but we should check if it's compatible with our current systems.` — ця особа визнає ідею, але підкреслює потенційну проблему сумісності.
