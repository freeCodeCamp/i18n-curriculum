# --explanation--

`To conduct` significa svolgere o condurre un'attività organizzata come una riunione, un esame o un'ispezione. Per esempio:

`The manager will conduct interviews for the new position tomorrow.` – Significa che il responsabile condurrà o gestirà i colloqui.

Un `audit` è un controllo o una revisione dettagliata di qualcosa, di solito per assicurarsi che rispetti standard o regole. Per esempio:

`The IT team performed a system audit to check for security risks.` – Significa che hanno revisionato il sistema per trovare possibili problemi.
