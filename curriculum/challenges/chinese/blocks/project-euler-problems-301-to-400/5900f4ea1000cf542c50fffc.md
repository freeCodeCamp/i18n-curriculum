# --description--

对于素数 $p$，令 $S(p) = (\sum (p - k)!)\bmod (p)$，其中 $1 ≤ k ≤ 5$。

例如，如果 $p = 7$，

$$(7 - 1)! + (7 - 2)! + (7 - 3)! + (7 - 4)! + (7 - 5)! = 6! + 5! + 4! + 3! + 2! = 720 + 120 + 24 + 6 + 2 = 872$$

由于 $872\bmod (7) = 4$，所以 $S(7) = 4$。

可以验证，对于 $5 ≤ p &lt; 100$，$\sum S(p) = 480$。

查找 $\sum S(p)$，其中 $5 ≤ p &lt; {10}^8$。
