# --explanation--

`That's true` se usa a menudo para mostrar acuerdo. Por ejemplo:

`That's true, we can proceed with the update.` - Esto significa que estás de acuerdo con algo que se dijo antes.

`Back-end stuff` se usa de manera informal para referirse a los sistemas y servicios que soportan la parte visible de un sitio web o aplicación, pero que no son vistos por los usuarios. 

`Stuff` por sí solo es una palabra general que se usa para referirse a cosas, tareas o actividades. Por ejemplo:

`We need to fix the back-end stuff before the site can go live.` - Esto significa que el trabajo técnico relacionado con el back end debe completarse primero.
