# --description--

`go as expected` とは、物事が計画や予想通りに進むことを意味します。これは、予期せぬ問題が起こらず、状況が意図した通りに展開することを望んだり必要としたりする場合によく使われます。例えば：

`We hope the event goes as expected, without any disruptions.` - 彼らはイベントが計画通りに進み、何も問題が起こらないことを望んでいます。
