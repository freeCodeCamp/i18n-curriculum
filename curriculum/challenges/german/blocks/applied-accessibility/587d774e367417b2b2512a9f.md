# --description--

HTML5 hat mehrere neue Elemente eingeführt, die Entwickler:innen mehr Möglichkeiten bieten und gleichzeitig Funktionen zur Barrierefreiheit integrieren. Zu diesen Tags gehören `main`, `header`, `footer`, `nav`, `article` und `section` sowie weitere.

Standardmäßig rendert ein Browser diese Elemente ähnlich wie das einfache `div`. Die Verwendung an passenden Stellen verleiht Ihrer Auszeichnung jedoch zusätzliche Bedeutung. Der Tag-Name allein kann den Typ der enthaltenen Informationen anzeigen, was der Seite eine semantische Bedeutung verleiht. Hilfstechnologien können auf diese Informationen zugreifen, um bessere Seitenübersichten oder Navigationsoptionen für ihre Nutzer:innen bereitzustellen.

Das `main`-Element wird verwendet, um (Sie haben es erraten) den Hauptinhalt einzuschließen, und es sollte nur eines pro Seite geben. Es soll die Informationen um das zentrale Thema Ihrer Seite herum umfassen. Wiederkehrende Elemente wie Navigationslinks oder Banner sollen nicht darin enthalten sein.

Der `main`-Tag verfügt außerdem über eine eingebaute Landmarken-Funktion, die Hilfstechnologien nutzen können, um schnell zum Hauptinhalt zu navigieren. Wenn Sie schon einmal einen Link „Zum Hauptinhalt springen“ oben auf einer Seite gesehen haben, bietet die Verwendung des `main`-Tags diese Funktionalität automatisch für Hilfsgeräte.

```js

```

```js
assert.match(code,/<\/header>\s*?<main>\s*?<\/main>/gi);
```
