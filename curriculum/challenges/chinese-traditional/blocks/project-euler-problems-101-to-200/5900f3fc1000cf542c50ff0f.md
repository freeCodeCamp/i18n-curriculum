# --description--

在雷射物理學中，「white cell」是一種鏡面系統，作為雷射光束的延遲線。光束進入該 cell，在鏡面間反射，最終再從中射出。

我們將考慮的特定白色儲存格是一個方程式為 $4{x}^2 + y^2 = 100$ 的簡略號。

頂部對應於 $−0.01 ≤ x ≤ +0.01$ 的區段遺失，允許光線通過孔洞進入和離開。

此問題中的光束從位於白色格子外的點 (0.0, 10.1) 開始，光束首次撞擊鏡面的座標為 (1.4, -9.6)。

每次雷射光束擊中簡略號的表面時，它都遵循通常的反射定律「入射角與反射角相等」。也就是說，入射光束和反射光束與入射點的法線所成的角度相同。

在左圖中，紅線顯示雷射光束與白色腔體牆壁的前兩個接觸點；藍線顯示第一個反彈點處與簡略號相切的線。

給定簡略號上任意點 (x, y) 處切線的斜率 m 為：$m = −4 × \frac{x}{y}$

法線線在入射點與此切線垂直。

右側的動畫顯示光束的前 10 次反射。

光束在離開前擊中白色腔體內部表面多少次？
