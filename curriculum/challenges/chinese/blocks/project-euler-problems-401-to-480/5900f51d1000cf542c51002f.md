# --description--

设 $E(x_0, y_0)$ 为使用欧几里得算法确定 $x_0$ 和 $y_0$ 的最大公因数所需的步骤数。更正式地说：

$$\begin{align}
  & x_1 = y_0, y_1 = x_0\bmod y_0 \\\\
  & x_n = y_{n - 1}, y_n = x_{n - 1}\bmod y_{n - 1}
\end{align}$$

$E(x_0, y_0)$ 是满足 $y_n = 0$ 的最小 $n$。

我们有 $E(1, 1) = 1$，$E(10, 6) = 3$ 和 $E(6, 10) = 4$。

将 $S(N)$ 定义为 $1 ≤ x$、$y ≤ N$ 时 $E(x, y)$ 的和。

我们有 $S(1) = 1$，$S(10) = 221$ 和 $S(100) = 39\\,826$。

查找 $S(5 \times {10}^6)$。
