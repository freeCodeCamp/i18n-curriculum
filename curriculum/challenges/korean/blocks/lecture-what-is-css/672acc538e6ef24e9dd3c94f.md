# --interactive--

여백과 패딩은 CSS에서 잘 구조화되고 읽기 쉽고 시각적으로 매력적인 웹 페이지를 만드는 데 필수적인 속성입니다.

여백은 요소 바깥의 공간을 제어하여 다른 요소와 구분하고 레이아웃 구조를 정의하는 데 도움을 주며, 패딩은 요소 안쪽의 공간을 제어하여 콘텐츠의 가독성과 미적 매력을 높입니다.

`margin`와 `padding`의 차이를 더 잘 이해하려면 몇 가지 예제를 살펴보겠습니다.

다음은 페이지에 있는 세 개의 단락 요소에 대한 HTML 예제입니다:

:::interactive_editor

```html
<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

:::

각 단락 요소의 위쪽에 간격을 적용하려면 `margin-top` 속성을 이렇게 사용할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  margin-top: 20px;
  border: 2px solid black;
}
```

:::

이 예제에서는 각 단락 요소의 위쪽에만 `20px`의 여백을 적용하고 있습니다.

또한 여백을 더 잘 볼 수 있도록 네 면 모두에 검은색 실선 테두리를 추가했습니다.

네 가지 다른 `margin` 속성은 `margin-top`, `margin-right`, `margin-bottom` 그리고 `margin-left`입니다.

다음은 네 가지 속성을 모두 사용하는 예제입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 30px;
  margin-left: 40px;
  border: 2px solid black;
}
```

:::

이 예제에서는 단락 요소의 네 면 모두에 간격 값을 할당하고 있습니다. 여백을 더 잘 볼 수 있도록 검은색 실선 테두리도 추가했습니다.

`margin` 속성을 사용하는 또 다른 방법은 단축 표기법입니다. 한 번에 하나, 둘, 셋 또는 넷의 값을 지정할 수 있습니다. 함께 살펴보겠습니다.

`margin` 단축 속성에 단일 값을 사용하면 그 값이 대상 요소의 네 면 모두에 동일하게 적용됩니다.

다음은 `margin` 단축 속성에 단일 값을 사용하는 예제입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px;
}
```

:::

이 코드 예제는 단락 요소의 네 면 모두에 `10px`의 `margin`을 동일하게 적용합니다.

두 값을 사용할 때는 첫 번째 값이 `top`와 `bottom`에 적용되고, 두 번째 값이 `left`와 `right`에 적용됩니다.

다음은 `margin` 단축 속성에 두 값을 사용하는 예제입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px;
}
```

:::

이 코드는 단락 요소의 `top`와 `bottom` 여백을 `10px`로, `20px`를 `left`와 `right` 여백에 설정합니다.

세 값을 제공하면 첫 번째 값이 `top` 여백에, 두 번째 값이 `left`와 `right` 여백에, 세 번째 값이 `bottom` 여백에 적용됩니다.

더 잘 이해할 수 있도록 예제를 보겠습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px;
}
```

:::

이 코드는 `10px`에 `top`, `20px`와 `left`에 `right`, `30px`에 `bottom` 여백을 설정합니다.

네 값을 사용할 때는 대상 요소의 각 면에 대해 여백 값을 독립적으로 지정할 수 있어 더 세밀한 제어가 가능합니다.

첫 번째 값은 `top`에, 두 번째 값은 `right`에, 세 번째 값은 `bottom`에, 네 번째 값은 `left`에 적용됩니다.

다음은 네 값을 사용하는 여백 단축 속성 예제입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px 40px;
}
```

:::

이 코드는 `10px`에 `top`, `20px`에 `right`, `30px`에 `bottom`, `40px`에 `left` 여백을 설정합니다.

`padding` 속성은 요소 내부, 콘텐츠와 테두리 사이에 공간을 적용하는 데 사용됩니다.

`margin` 속성과 마찬가지로 네 가지 `padding` 속성은 `padding-top`, `padding-right`, `padding-bottom` 그리고 `padding-left`입니다.

다음은 단락 요소에 `padding`를 설정하는 예제입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 30px;
  padding-left: 40px;
  border: 2px solid black;
}
```

:::

이 코드는 `10px`에 `top`, `20px`에 `right`, `30px`에 `bottom`, `40px`에 `left` 패딩을 설정합니다.

보시다시피 `padding`는 테두리 안쪽의 콘텐츠에 적용되며, `margin`는 테두리 바깥쪽에 적용됩니다.

`margin` 속성과 마찬가지로 `padding` 속성에도 단축 표기법을 사용할 수 있습니다.

`padding` 단축 속성에도 한 개, 두 개, 세 개 또는 네 개의 값을 지정할 수 있습니다.

다음은 앞서 본 단락 요소에 `padding` 단축 속성을 사용하는 예제입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding: 10px 20px 30px 40px;
  border: 2px solid black;
}
```

:::

이 예제에서 단축 표기를 사용하면 단락 요소의 `10px`에 `top`, `20px`에 `right`, `30px`에 `bottom`, `40px`에 `left` 패딩이 설정됩니다.
