# --description--

직접 질문은 특정 정보를 명확하게 묻는 간단한 문의입니다. 보통 주어와 동사의 위치를 바꾸어 표현하며, `Are you sure?`와 같습니다. 이 형태는 빠르고 명확한 답변이 필요할 때 일상 대화에서 자주 사용됩니다.

반면 간접 질문이나 진술은 더 공손하고 덜 직접적입니다. 주어와 동사의 위치를 바꾸지 않으며, `I was wondering` 또는 `Could you tell me` 같은 구문으로 시작하는 경우가 많습니다. 예를 들어, `Where do you live?`라고 묻는 대신 `Could you tell me where you live?`라고 말할 수 있습니다. 이렇게 하면 질문이 덜 갑작스럽고 더 공손하게 들려, 공식적인 상황이나 잘 모르는 사람과 대화할 때 특히 유용합니다.

예시:

직접 질문: `What time is it?` (동사 `is`와 주어 `it`의 위치 바꿈)

간접 진술: `Please tell me what time it is.` (위치 바꾸지 않음)
