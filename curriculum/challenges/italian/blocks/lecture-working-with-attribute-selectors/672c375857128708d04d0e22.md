# --interactive--

Quando lavori con le liste ordinate in HTML, l'attributo `type` ti permette di specificare lo stile di numerazione usato, come numerico, alfabetico o numeri romani.

L'attributo `type` può essere impostato su un elemento `ol` (lista ordinata) per controllare il formato della numerazione:

- `1` per liste numeriche (come 1, 2, 3, ...),

- `A` per liste alfabetiche maiuscole (come A, B, C, ...),

- `a` per liste alfabetiche minuscole (come a, b, c, ...),

- `I` per numeri romani maiuscoli (come I, II, III, ...),

- `i` per numeri romani minuscoli (come i, ii, iii, ...).

Ecco come puoi usare il selettore attributo per definire lo stile di una lista ordinata con un attributo `type` specifico:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="A"] {
  color: purple;
  font-weight: bold;
}
```

:::

In questo esempio, la regola CSS applicherà testo viola e carattere in grassetto a qualsiasi lista ordinata che usa la numerazione alfabetica maiuscola, specificata da `type="A"`.

Allo stesso modo, puoi definire lo stile delle liste ordinate che usano numeri romani minuscoli selezionando quelle con `type="i"`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="i">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="i"] {
  color: green;
}
```

:::

Questa regola cambierà il colore del testo in verde per qualsiasi lista ordinata che usa numeri romani minuscoli.

La possibilità di definire lo stile delle liste ordinate in base al loro attributo `type` ti dà un controllo maggiore sull'aspetto delle liste nel tuo documento, permettendo personalizzazioni in base a come è strutturato il contenuto.
