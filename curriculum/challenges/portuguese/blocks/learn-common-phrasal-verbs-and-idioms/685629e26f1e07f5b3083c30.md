# --explanation--

`Real-time alerts` são mensagens ou avisos que aparecem imediatamente assim que algo acontece.Por exemplo:

`The app sends real-time alerts if someone logs in from a new location.` – Isso significa que toda vez que alguém faz login de um local diferente, o usuário recebe uma mensagem de aviso imediata.

`Graphics` são imagens ou elementos visuais usados para comunicar informações.Por exemplo:

`The dashboard uses graphics to show project progress.` – Isso significa que ferramentas visuais como gráficos ou ícones são usadas para facilitar a compreensão dos dados.
