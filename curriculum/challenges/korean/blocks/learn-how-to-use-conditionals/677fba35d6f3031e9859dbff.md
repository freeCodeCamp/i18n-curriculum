# --explanation--

문장 끝의 `Right`는 질문 태그입니다. 동의를 확인하거나 듣는 사람이 이해했는지 확인할 때 사용합니다. 예를 들어:

- `It's going to rain today, right?` - 화자(말하는 사람)가 날씨에 대해 확인을 요청하는 경우입니다.

- `You finished the project, right?` - 화자가 듣는 사람이 과제를 완료했는지 확인하는 경우입니다.

이 대화에서 Bob은 Maria에게 자신의 말에 동의하거나 확인해 달라고 요청하고 있습니다.
