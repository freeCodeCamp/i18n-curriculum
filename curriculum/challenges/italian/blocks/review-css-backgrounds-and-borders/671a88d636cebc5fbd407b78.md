## Definire lo stile delle liste

- **Proprietà `line-height`**: Questa proprietà serve a creare spazio tra le righe di testo. I valori `line-height` accettati includono la parola chiave `normal`, numeri, percentuali e unità di lunghezza come l'unità `em`.
- **Proprietà `list-style-type`**: Questa proprietà serve a specificare il marcatore per un elemento della lista. I valori accettabili possono includere un cerchio, un disco o un numero decimale.

Ecco un esempio che usa `list-style-type` per cambiare lo stile del punto elenco:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
ul {
  list-style-type: square;
}
```

:::

- **Proprietà `list-style-position`**: Questa proprietà serve a impostare la posizione del marcatore della lista. Gli unici due valori accettabili sono inside e outside.

Ecco un esempio che mostra la differenza tra `inside` e `outside`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="inside-list">
  <li>Item with inside position</li>
  <li>Item with inside position</li>
</ul>
<ul class="outside-list">
  <li>Item with outside position</li>
  <li>Item with outside position</li>
</ul>
```

```css
.inside-list {
  list-style-position: inside;
}

.outside-list {
  list-style-position: outside;
}
```

:::

- **Proprietà `list-style-image`**: Questa proprietà serve a usare un'immagine come marcatore per l'elemento della lista. Un caso d'uso comune è usare la funzione `url` con un valore impostato su una posizione valida dell'immagine.

## Spaziare gli elementi della lista usando `margin`

- Oltre a `line-height`, in CSS si possono usare anche i margini per migliorare lo spazio e la leggibilità degli elementi della lista.
- I margini creano spazio all'esterno di ogni elemento `li`, permettendo di controllare lo spazio tra gli elementi della lista.
- `margin-bottom` serve a creare spazio sotto ogni elemento della lista. Per esempio, `margin-bottom: 10px;` creerà uno spazio di 10 pixel sotto ogni elemento della lista.

Ecco un esempio che usa `margin-bottom` per spaziare gli elementi della lista:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="list">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
.list li {
  margin-bottom: 20px;
}
```

## Definire lo stile dei collegamenti

- **`pseudo-class`**: Questa è una parola chiave aggiunta a un selettore che permette di selezionare elementi in base a uno stato particolare. Gli stati comuni includono `:hover`, `:visited` e `:focus`.
- **`:link pseudo-class`**: Questa pseudo-classe serve a definire lo stile dei collegamenti che non sono stati ancora visitati dall'utente.

Ecco un esempio che usa la pseudo-classe `:link`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:link {
  color: red;
}
```

:::

- **`:visited pseudo-class`**: Questa pseudo-classe serve a definire lo stile dei collegamenti già visitati dall'utente.
- **`:hover pseudo-class`**: Questa pseudo-classe serve a definire lo stile degli elementi su cui l'utente sta attivamente passando con il cursore.

Ecco un esempio che usa la pseudo-classe `:hover`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Hover over me!</a>
```

```css
a:hover {
  color: green;
  text-decoration: underline;
}
```

:::

- **`:focus pseudo-class`**: Questa pseudo-classe serve a definire lo stile di un elemento quando riceve il focus. Esempi includono elementi `input` o `select` su cui si clicca o si tabula per metterli a fuoco.

Ecco un esempio che usa la pseudo-classe `:focus`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:focus {
  outline: 2px solid orange;
}
```

:::

- **`:active pseudo-class`**: Questa pseudo-classe serve a definire lo stile di un elemento che è stato attivato dall'utente. Un esempio comune è quando l'utente clicca su un pulsante.

Ecco un esempio che usa la pseudo-classe `:active`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Click me!</a>
```

```css
a:active {
  color: pink;
}
```

## Lavorare con sfondi e bordi

- **Proprietà `background-size`**: Questa proprietà serve a impostare la dimensione dello sfondo per un elemento. Alcuni valori comuni includono `cover` per far coprire l'intero elemento dall'immagine di sfondo e `contain` per far adattare l'immagine di sfondo all'interno dell'elemento.

Ecco un esempio che usa `background-size: cover`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

Ecco un esempio che usa `background-size: contain`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **Proprietà `background-repeat`**: Questa proprietà serve a determinare come le immagini di sfondo devono ripetersi lungo gli assi orizzontale e verticale. Il valore predefinito per `background-repeat` è `repeat`, il che significa che l'immagine si ripeterà sia orizzontalmente sia verticalmente. Puoi anche specificare che non ci sia ripetizione usando la proprietà `no-repeat`.

Ecco un esempio che usa `background-repeat: no-repeat`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **Proprietà `background-position`**: Questa proprietà serve a specificare la posizione dell'immagine di sfondo. Può essere impostata a una lunghezza specifica, percentuale o valori parola chiave come `top`, `bottom`, `left`, `right` e `center`.

Ecco un esempio che usa `background-position`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

- **Proprietà `background-attachment`**: Questa proprietà serve a specificare se l'immagine di sfondo deve scorrere con il contenuto o rimanere fissa in posizione. I valori principali sono `scroll` (predefinito), dove l'immagine di sfondo scorre con il contenuto, e `fixed`, dove l'immagine di sfondo resta nella stessa posizione sullo schermo.
- **Proprietà `background-image`**: Questa proprietà serve a impostare l'immagine di sfondo di un elemento. Puoi impostare più immagini di sfondo contemporaneamente e usare le funzioni `url`, `radial-gradient` o `linear-gradient` come valori.

Ecco un esempio che usa `background-image`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **Proprietà `background`**: Questa è la proprietà abbreviata per impostare tutte le proprietà di sfondo in un'unica dichiarazione. Ecco un esempio di impostazione dell'immagine di sfondo e di non ripetizione: `background: no-repeat url("example-url-goes-here");`

Ecco un esempio che usa la proprietà abbreviata `background`:

:::interactive_editor

```html
<style>
  body {
    background: center top no-repeat url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **Buon contrasto per colori di sfondo e primo piano**: È importante assicurarsi che i colori di sfondo e primo piano abbiano un buon contrasto per rendere il testo leggibile. Le Web Content Accessibility Guidelines (WCAG) raccomandano un rapporto di contrasto minimo di 4.5:1 per il testo normale e 3:1 per il testo grande.

## Bordi

- **Proprietà `border-top`**: Questa proprietà serve a impostare gli stili per il bordo superiore di un elemento. `border-top: 3px solid blue;` imposta un bordo solido blu di 3 pixel di larghezza sul lato superiore dell'elemento.
- **Proprietà `border-right`**: Questa proprietà serve a impostare gli stili per il bordo destro di un elemento. `border-right: 2px solid red;` imposta un bordo solido rosso di 2 pixel di larghezza sul lato destro dell'elemento.
- **Proprietà `border-bottom`**: Questa proprietà serve a impostare gli stili per il bordo inferiore di un elemento. `border-bottom: 1px dashed green;` imposta un bordo tratteggiato verde di 1 pixel di larghezza sul lato inferiore dell'elemento.
- **Proprietà `border-left`**: Questa proprietà serve a impostare gli stili per il bordo sinistro di un elemento. `border-left: 4px dotted orange;` imposta un bordo puntinato arancione di 4 pixel di larghezza sul lato sinistro dell'elemento.

Ecco un esempio che usa le proprietà individuali dei bordi:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="bordered-box">Box with different borders</div>
```

```css
.bordered-box {
  border-top: 3px solid blue;
  border-right: 2px solid red;
  border-bottom: 1px dashed green;
  border-left: 4px dotted orange;
  padding: 20px;
}
```

:::

- **Proprietà `border`**: Questa è la proprietà abbreviata per impostare la larghezza, lo stile e il colore del bordo di un elemento. `border: 1px solid black;` imposta un bordo solido nero di 1 pixel di larghezza.

Ecco un esempio che usa la proprietà abbreviata `border`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

- **Proprietà `border-radius`**: Questa proprietà serve a creare angoli arrotondati per il bordo di un elemento.

Ecco un esempio che usa `border-radius`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

- **Proprietà `border-style`**: Questa proprietà serve a impostare lo stile del bordo di un elemento. Alcuni valori accettati includono `solid`, `dashed`, `dotted` e `double`.

Ecco un esempio che usa valori `border-style` diversi:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="solid-border">Solid border</div>
<div class="dashed-border">Dashed border</div>
<div class="dotted-border">Dotted border</div>
<div class="double-border">Double border</div>
```

```css
.solid-border {
  border: 3px solid blue;
  margin-bottom: 10px;
  padding: 10px;
}

.dashed-border {
  border: 3px dashed red;
  margin-bottom: 10px;
  padding: 10px;
}

.dotted-border {
  border: 3px dotted green;
  padding: 10px;
}

.double-border {
  border: 3px double blueviolet;
  padding: 10px;
}
```

:::

## Gradienti

- **Funzione `linear-gradient()`**: Questa funzione CSS serve a creare una transizione tra più colori lungo una linea retta.

Ecco un esempio che usa `linear-gradient()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

- **Funzione `radial-gradient()`**: Questa funzione CSS crea un'immagine che irradia da un punto particolare, come un cerchio o un'ellisse, e passa gradualmente tra più colori.

Ecco un esempio che usa `radial-gradient()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```
