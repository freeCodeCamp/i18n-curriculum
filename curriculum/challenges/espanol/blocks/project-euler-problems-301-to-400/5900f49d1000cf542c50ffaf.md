# --description--

Para cualquier entero positivo $n$, la función $\text{next_prime}(n)$ devuelve el menor primo $p$ tal que $p > n$.

La secuencia $a(n)$ se define por: $a(1) = \text{next_prime}({10}^{14})$ y $a(n) = \text{next_prime}(a(n - 1))$ para $n > 1$.

La secuencia de Fibonacci $f(n)$ se define como: $f(0) = 0$, $f(1) = 1$ y $f(n) = f(n - 1) + f(n - 2)$ para $n > 1$.

La secuencia $b(n)$ se define como $f(a(n))$.

Encuentra $\sum b(n)$ para $1≤n≤100\\,000$. Da tu respuesta $\bmod 1\\,234\\,567\\,891\\,011$.
