# --interactive--

입력 필드에 붙는 레이블은 사소한 부분처럼 보일 수 있지만, 모든 사람이 폼을 더 쉽게 사용할 수 있도록 하는 데 중요한 역할을 합니다.

레이블은 장애가 있는 사람들을 돕고, 모든 사용자에게 폼을 더 친숙하게 만들며, 궁극적으로 폼 전환율을 높입니다.

입력 요소에 레이블을 추가하는 것이 접근성과 사용성에 왜 중요한지 자세히 알아보겠습니다.

화면 읽기 프로그램은 종종 입력 필드의 목적을 설명하기 위해 레이블에 의존합니다. 이를 제대로 작동하게 하려면 레이블이 `input`와 프로그래밍적으로 연결되어야 합니다. 이를 수행하는 방법은 여러 가지가 있지만, 가장 일반적인 방법은 HTML `label` 요소를 사용하는 것입니다.

`label`를 `input`와 연결하는 방법은 다음과 같습니다:

:::interactive_editor

```html
<form>
   <label for="name">Your Name</label>
   <input type="text" id="name" />
</form>
```

:::

이 예제에서 `for` 요소의 `label` 속성은 `id` 요소의 `input`와 연결되어 있습니다. 이 연결 덕분에 화면 읽기 프로그램은 `label`가 포커스 상태일 때 `input`를 알릴 수 있어, 화면 읽기 프로그램 사용자가 `input`의 목적을 이해할 수 있습니다.

모든 입력 필드에 레이블을 연결하는 것은 SEO에도 도움이 됩니다. 적절한 레이블이 있으면 검색 엔진이 페이지 내용을 더 잘 이해하여 페이지 검색 순위에 긍정적인 영향을 줍니다.

하지만 중요한 것은 검색 엔진이 아니라 사람을 위한 콘텐츠를 만드는 것입니다. 따라서 레이블이 가능한 한 명확하고 설명적이어야 합니다.
