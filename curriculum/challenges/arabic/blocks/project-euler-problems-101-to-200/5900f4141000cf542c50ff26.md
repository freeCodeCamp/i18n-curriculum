# --description--

لمجموعتين من الأعداد الصحيحة الموجبة $a$ و $b$، تُعرّف متتالية Ulam $U(a,b)$ بحيث ${U{(a,b)}\_1} = a$، و${U{(a,b)}\_2} = b$، ولـ $k > 2$، يكون ${U{(a,b)}\_k}$ هو أصغر عدد صحيح أكبر من ${U{(a,b)}\_{(k-1)}}$ يمكن كتابته بطريقة واحدة فقط كمجموع عددين مختلفين من الأعضاء السابقين في $U(a,b)$.

على سبيل المثال، تبدأ المتتالية $U(1,2)$ بـ

$$1, 2, 3 = 1 + 2, 4 = 1 + 3, 6 = 2 + 4, 8 = 2 + 6, 11 = 3 + 8$$

العدد 5 لا ينتمي إليها لأن $5 = 1 + 4 = 2 + 3$ له تمثيلان كمجموع عددين سابقين، وكذلك الحال مع $7 = 1 + 6 = 3 + 4$.

احسب $\sum {U(2, 2n + 1)_k}$ لـ $2 ≤ n ≤ 10$، حيث $k = {10}^{11}$.
