# --interactive--

클래스 선택자는 CSS의 핵심 요소로, 개발자가 같은 클래스 속성을 가진 여러 요소를 대상으로 일관된 스타일을 적용할 수 있게 합니다.

이로 인해 웹사이트 전반에 스타일을 적용할 때 매우 유연하고 효율적입니다.

클래스 선택자는 점(`.`) 다음에 클래스 이름이 옵니다. HTML 문서 내 어떤 요소에도 적용할 수 있습니다.

다음은 클래스 선택자를 사용한 예시입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="highlight">Example paragraph</p>
```

```css
.highlight {
  color: green;
}
```

:::

이 예시에서 `highlight` 클래스를 가진 모든 요소는 텍스트 색상이 `green`로 설정됩니다.

클래스 선택자는 타입 선택자보다 특이성이 높지만 ID 선택자와 인라인 스타일보다는 낮습니다.

클래스 선택자의 특이성 값은 `(0, 0, 1, 0)`입니다. 즉, 클래스 선택자는 타입 선택자를 재정의할 수 있지만 ID 선택자와 인라인 스타일에는 재정의됩니다.

클래스 선택자는 다른 선택자와 결합하여 더 구체적인 규칙을 만들 수 있습니다.

다음은 단락 타입 선택자와 클래스 선택자를 결합한 예시입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="bold-text">Example paragraph</p>
<p class="bold-text">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p.bold-text {
  font-weight: bold;
}
```

:::

이 규칙은 `p` 요소 중 `bold-text` 클래스를 가진 요소에만 적용되어 텍스트를 굵게 만듭니다.
