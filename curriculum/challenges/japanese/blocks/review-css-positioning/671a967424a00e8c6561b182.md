## フロートの扱い方

- **定義**: フロートは、要素をページの通常のフローから外して、そのコンテナの左側または右側に配置するために使います。こうすると、テキストがそのフロートした要素の周りに回り込みます。

```css
float: left;
float: right;
```

- **フロートのクリア**: `clear` プロパティは、要素をフロートしたコンテンツの下に移動させる必要があるかどうかを判断するために使います。複数のフロート要素が隣り合って積み重なっている場合、レイアウトで重なりや崩れが起きることがあります。そこで、この問題を解決するために `clearfix` ハックが作られました。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="clearfix">
  <div class="box left">Left</div>
  <div class="box right">Right</div>
</div>
```

```css
.box {
  width: 100px;
  height: 60px;
  color: white;
  text-align: center;
  line-height: 60px;
}

.left {
  float: left;
  background: teal;
}

.right {
  float: right;
  background: purple;
}

.clearfix::after {
  content: "";
  display: block;
  clear: both;
}
```

:::

## スタティック、相対、絶対位置指定

- **スタティック位置指定**: これはドキュメントの通常のフローです。要素は上から下へ、左から右へ順番に配置されます。
- **相対位置指定**: これにより、`top`、`left`、`right`、`bottom` プロパティを使って、要素を通常のドキュメントフロー内で位置指定できます。また、相対位置指定を使うと、要素をページ上の他の要素と重ねることもできます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="relative">Relative Element</div>
```

```css
.relative {
    position: relative;
    top: 30px;
    left: 30px;
}
```

:::

- **絶対位置指定**: これにより、要素を通常のドキュメントフローから外し、他の要素とは独立して動作させることができます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  Parent
  <div class="positioned">Absolute</div>
</div>
```

```css
.parent {
  position: relative;
  height: 120px;
  background: #eee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

## 固定位置指定とスティッキー位置指定

- **固定位置指定**: `position: fixed` で位置指定された要素は通常のドキュメントフローから外され、ビューポートに対して相対的に配置されます。つまり、ユーザーがスクロールしても同じ位置に固定されます。これは、ヘッダーやナビゲーションバーのように常に表示しておきたい要素に使われます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="navbar">Fixed Navbar</div>
<div style="height: 600px;"></div>
```

```css
.navbar {
  position: fixed; 
  top: 0; 
  width: 100%;
  background: #0077ff;
  color: white;
  padding: 8px; 
}
```

:::

- **スティッキー位置指定**: この位置指定は相対位置指定と固定位置指定のハイブリッドのように動作します。最初は相対的に配置され、ドキュメントのフロー内に留まります。しかし、ユーザーがスクロールして要素が特定の位置を超えると、その要素はビューポート（通常は上部）に「くっつき」、固定されたかのように振る舞います。これは、ユーザーがスクロールしたときにだけ固定されるスティッキーナビゲーションバーの作成に最適です。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Scroll down</p>
<div class="positioned">Sticky Element</div>
<div style="height: 500px;"></div>
```

```css
.positioned {
  position: sticky;
  top: 30px;
  left: 30px;
  background: #4caf50;
  color: white;
  padding: 6px;
}
```

:::

## `z-index` プロパティの扱い方

- **定義**: CSSの `z-index` プロパティは、ページ上で重なり合う位置指定要素の垂直方向の重なり順序を制御するために使います。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box box1">1</div>
  <div class="box box2">2</div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

.box {
  position: absolute;
  width: 100px;
  height: 100px;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

.box1 {
  background: lightcoral;
  top: 20px;
  left: 20px;
  z-index: 1;
}

.box2 {
  background: steelblue;
  top: 40px;
  left: 40px;
  z-index: 2;
}
```
