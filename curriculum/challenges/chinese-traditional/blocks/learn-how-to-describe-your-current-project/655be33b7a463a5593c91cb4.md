# --description--

當你確定某件事時，你會說它是 `definite`。 

當你為字尾添加 `-ly` 時，`definite` 會變成副詞 `definitely`，意思是對某事有把握或有信心。 

它經常用來強調說話者相信是真實的或可能是真實的。 例如：

- `I can definitely see why this is important.`－這強調了這個重要性應該是明顯的。

- `She will definitely finish the project on time。` - 這顯示她有信心準時完成專案。
