# --description--

使用 freeCodeCamp 的訂製輸入，你可以直接在任務中輸入拼音和中文字元。

如果你在你的電腦上使用中文鍵盤，你將能夠輸入拼音並從建議中選擇中文字元。以下是設定方法。

## 在 Windows 11 上設定

1. 從開始選單開啟**設定**。

2. 前往 **時間與語言**，然後點擊 **語言與地區**。

3. 前往 **Preferred languages** 並選擇 **Add a language**。

4. 搜尋 **Chinese** 並選擇 **Chinese (Simplified)**。

5. 點擊 **Next**，然後依照步驟操作。

語言安裝完成後：  

- 在你的**偏好語言**列表中找到它。

- 點擊省略符號以顯示其他選項，可選方案，並選擇 **Language options**。

- 前往 **Keyboards**，點擊 **Add a keyboard** 並選擇 Chinese (Simplified) 版面配置。

要在鍵盤之間切換：  

- 按下 **Windows key + Spacebar**。

- 或者點擊工作列上的語言圖示（通常標示為 **ENG**）並選擇中文。

## 在 Mac 上設定

**注意：** 根據你正在執行的作業系統，步驟可能會有所不同。

1. 打開 **Apple 選單** 並選取 **系統偏好設定**。

2. 前往 **Keyboard**。

3. 點擊 **Input Sources** 分頁。

4. 選取 **plus (+)** 按鈕，點擊 **Chinese, Simplified**，並從列表中選擇 **Pinyin - Simplified**。

要在鍵盤之間切換：

- 按下 **Control + Spacebar**。

- 或者點擊選單列上的輸入圖示並選擇 Pinyin 方法。

現在你已經準備好了。
