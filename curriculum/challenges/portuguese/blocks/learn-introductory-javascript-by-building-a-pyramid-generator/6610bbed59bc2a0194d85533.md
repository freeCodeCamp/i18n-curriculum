# --description--

Perceba como o valor dentro do seu array `rows` foi alterado diretamente?Isto é chamado <dfn>mutation</dfn>.À medida que você aprende mais sobre arrays, você aprenderá quando deve mutar um array e quando não deve.

Antes de continuar, esta é uma ótima oportunidade para aprender um uso comum de array.Atualmente, seu código acessa o último elemento no array com `rows[2]`.Mas você pode não saber quantos elementos há em um array quando quiser o último.

Você pode usar a propriedade `.length` de um array - isso retorna o número de elementos no array.Para obter o último elemento de qualquer array, você pode usar a seguinte sintaxe:

```js
array[array.length - 1]
```

`array.length` retorna o número de elementos no array.Subtraindo `1`, você obtém o índice do último elemento no array.Você pode aplicar esse mesmo conceito ao seu array `rows`.

Atualize seu `rows[2]` para acessar dinamicamente o último elemento no array `rows`.Consulte o exemplo acima para ajudá-lo.

Você não deve ver nenhuma alteração no seu console.
