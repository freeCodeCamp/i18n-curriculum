# --explanation--

`Regularly` es un adverbio que se usa para describir una acción realizada en intervalos consistentes o repetidos. A menudo se asocia con hábitos o rutinas. Por ejemplo:

- `I exercise regularly.` - Esto significa hacer ejercicio de manera constante, quizás todos los días o varias veces a la semana.

- `She checks her email regularly.` - Esto significa que revisa su correo electrónico con frecuencia, como diariamente.

Hacer algo regularmente asegura buenos hábitos y previene problemas, como hacer copias de seguridad de los datos para evitar pérdidas.
