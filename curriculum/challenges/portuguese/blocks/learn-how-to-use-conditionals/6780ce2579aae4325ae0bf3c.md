# --explanation--

`Critical` significa algo muito importante ou essencial e `critical data` refere-se a informações importantes nas quais a equipe depende para trabalhar de forma eficaz.Perder isso causaria atrasos ou problemas.Por exemplo:

- `The backup includes all critical data in case of a failure.` - A cópia salva contém informações essenciais para recuperar de um problema.

- `You must encrypt critical data to ensure it's secure.` - Proteger informações importantes com criptografia é necessário para a segurança.
