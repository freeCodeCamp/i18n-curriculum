# --interactive--

JavaScript에는 JSON 데이터를 처리하는 데 강력한 두 가지 메서드가 있습니다: `JSON.parse()`와 `JSON.stringify()`. 이 메서드들은 JSON 문자열과 JavaScript 객체 간 변환에 자주 사용됩니다.

`JSON.stringify()`는 JavaScript 객체를 JSON 문자열로 변환하는 데 사용됩니다. 이는 데이터를 쉽게 공유하거나 시스템 간 전송할 수 있는 형식으로 저장하거나 전송할 때 유용합니다.

다음은 `JSON.stringify()` 메서드를 사용하는 방법입니다:

:::interactive_editor

```js
const user = {
  name: "John",
  age: 30,
  isAdmin: true
};

const jsonString = JSON.stringify(user);
console.log(jsonString);
```

:::

`JSON.stringify()` 메서드는 함수나 배열일 수 있는 선택적 매개변수인 replacer도 받습니다. 다음은 선택적 replacer 매개변수로 배열을 사용하는 예입니다:

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

// result: {"firstName":"Jessica","country":"USA"}
console.log(JSON.stringify(developerObj, ["firstName", "country"]));
```

:::

이 예에서는 네 개의 속성을 가진 `developerObj`가 있습니다. `JSON.stringify()` 메서드를 사용할 때 두 번째 매개변수로 배열을 전달하여 문자열화할 속성을 지정할 수 있습니다. 결과는 `firstName`와 `country` 속성만 포함하는 문자열화된 객체가 됩니다.

`JSON.stringify()` 메서드의 또 다른 선택적 매개변수는 spacer 매개변수입니다. 이것은 문자열화된 결과의 들여쓰기를 조절할 수 있게 해줍니다:

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

console.log(JSON.stringify(developerObj, null, 2));

/* result
{
  "firstName": "Jessica",
  "isAwesome": true,
  "isMusician": true,
  "country": "USA"
}
*/
```

:::

대부분의 경우 `JSON.stringify()` 메서드에서 이 선택적 매개변수들을 사용하지 않지만, 알고 있으면 도움이 됩니다.

프로그래밍에서 자주 사용할 또 다른 메서드는 `JSON.parse()` 메서드입니다. `JSON.parse()`는 JSON 문자열을 다시 JavaScript 객체로 변환합니다. 이는 웹 서버나 `localStorage`에서 JSON 데이터를 가져와 애플리케이션에서 데이터를 조작해야 할 때 유용합니다. `localStorage`에 대해서는 앞으로 수업에서 더 배우게 됩니다.

다음은 `JSON.parse()` 메서드를 사용하는 예입니다:

:::interactive_editor

```js
const jsonString = '{"name":"John","age":30,"isAdmin":true}';
const userObject = JSON.parse(jsonString);
console.log(userObject);

// Result:
// { name: 'John', age: 30, isAdmin: true }
```

:::

이렇게 하면 프로그램에서 데이터를 일반 JavaScript 객체처럼 다룰 수 있어 조작하고 사용하기가 더 쉬워집니다.

앞으로의 모듈에서 JSON과 `JSON.parse()`, `JSON.stringify()` 메서드를 다루는 방법을 계속 배울 것입니다.
