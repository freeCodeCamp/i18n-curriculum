# --description--

正の整数 $n$ を考えます。整数の三つ組 ($a$, $b$, $c$) が $n$ の因数分解トリプルと呼ばれるのは、次の条件を満たす場合です：

- $1 ≤ a ≤ b ≤ c$
- $a \times b \times c = n$。

$f(n)$ を、$\frac{c}{a}$ を最小化する $n$ の因数分解トリプル ($a$, $b$, $c$) に対する $a + b + c$ と定義します。この三つ組は一意であることが示せます。

例えば、$f(165) = 19$、$f(100\\,100) = 142$、$f(20!) = 4\\,034\\,872$ です。

$f(43!)$ を求めてください。
