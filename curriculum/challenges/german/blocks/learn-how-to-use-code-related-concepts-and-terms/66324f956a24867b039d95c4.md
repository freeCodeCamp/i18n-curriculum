# --description--

Das Wort `should` ist ein Modalverb, das verwendet wird, um Ratschläge oder Vorschläge zu geben. Es wird vom Infinitiv des Verbs ohne „to“ gefolgt. Zum Beispiel wird im Satz `You should try that restaurant` `should` vom Infinitiv `try` gefolgt.

Das Verb `pick` bedeutet, etwas aus einer Reihe von Optionen auszuwählen oder zu wählen. Im Programmierkontext bedeutet `picking a data type` die passende Art von Daten auszuwählen, wie eine Ganzzahl, einen String oder einen Boolean, je nachdem, was für die jeweilige Aufgabe am besten geeignet ist.

`Match` bedeutet sicherzustellen, dass zwei Dinge zueinander passen oder gut zusammenpassen. Zum Beispiel `This key matches the lock.` Es zeigt an, dass der Schlüssel für das Schloss geeignet ist.
