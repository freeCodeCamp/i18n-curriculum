# --description--

Barbara 是一名数学家和篮球运动员。她发现从距离 `x` 投篮得分的概率正好是 ($1 - \frac{x}{q}$)，其中 `q` 是一个大于 50 的实数常量。

在每次练习 `Run` 中，她从距离 $x = 1, x = 2, \ldots, x = 50$ 处投篮，根据她的 `record`，她恰好有 2 % 的几率得到总共正好 20 分。

查找 $q$ 并将你的答案四舍五入到 10 位小数。
