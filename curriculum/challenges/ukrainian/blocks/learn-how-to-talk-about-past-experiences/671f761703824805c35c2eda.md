# --explanation--

Фраза, утворена за допомогою `get` з прикметником, описує зміну стану або ставання якимось. Наприклад:

- `They always get excited when a new Python update is released.` - `Get excited` означає стати захопленим або у захваті. Це означає, що вони завжди наповнені ентузіазмом, коли з’являється нове оновлення Python.

- `He got tired after working on the project all night.` - Це означає, що раніше він не був втомленим, а тепер він втомився після цілонічної роботи над проєктом.

У реченні Марії вона запитує, чи пам’ятає Браян те захоплення, яке вони відчували раніше щодо нових технологій, працюючи разом.
