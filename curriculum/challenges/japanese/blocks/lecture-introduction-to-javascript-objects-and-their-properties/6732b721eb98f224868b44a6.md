# --interactive--

オブジェクトからプロパティを削除する方法はいくつかありますが、`delete`演算子が最も簡単で一般的に使われる方法です。

`delete`を使うと、選択したプロパティがオブジェクトから削除されます。`delete`演算子の使い方の例を見てみましょう。

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  job: "Engineer"
};

delete person.job;

console.log(person.job); // undefined
```

:::

この例では、`person`オブジェクトが`name`、`age`、`job`という3つのプロパティを持っています。次に、`delete`演算子を使って`job`プロパティを削除します。削除後、`person`オブジェクトには`job`プロパティがなくなっています。

別の方法として、分割代入とレストパラメータを使ってプロパティを削除する方法があります。この方法は実際にプロパティを削除するわけではありませんが、指定したプロパティを除いた新しいオブジェクトを作成します。

:::interactive_editor

```js
const person = {
  name: "Bob",
  age: 25,
  job: "Designer",
  city: "New York"
};

const { job, city, ...remainingProperties } = person;

// { name: "Bob", age: 25 }
console.log(remainingProperties);
```

:::

この例では、`job`と`city`を`person`オブジェクトから分割代入で取り出し、残りのプロパティを`remainingProperties`という新しいオブジェクトにまとめています。これにより、`job`と`city`プロパティを持たない新しいオブジェクトが作成されます。

オブジェクトからプロパティを削除する方法を理解することは、JavaScriptプログラミングで重要なスキルです。これにより、オブジェクトを動的に操作し、不要なデータを整理できます。
