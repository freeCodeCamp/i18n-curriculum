# --interactive--

In JavaScript ist das Arbeiten mit Text ein wesentlicher Teil des Codens, und oft müssen Sie Textstücke miteinander verbinden oder zusammenfügen. Dieser Vorgang wird String-Konkatenation genannt.

In dieser Lektion konzentrieren wir uns darauf, wie String-Konkatenation funktioniert, insbesondere mit dem `+`-Operator, dem `+=`-Operator und der `concat()`-Methode.

Der `+`-Operator ist eine der einfachsten und am häufigsten verwendeten Methoden, um Strings zu verketten. Er ermöglicht es Ihnen, mehrere Strings zu verbinden oder Strings mit Variablen zu kombinieren, die Text enthalten.

Hier ist ein Beispiel:

:::interactive_editor

```js
let firstName = "John";
let lastName = "Doe";

let fullName = firstName + " " + lastName; 
console.log(fullName); // John Doe
```

:::

In diesem Beispiel haben wir den `+`-Operator verwendet, um die `firstName`- und `lastName`-Variablen zusammen mit einem Leerzeichen (`" "`) zu verketten und so den vollständigen Namen zu erstellen.

Ein Nachteil bei der Verwendung des `+`-Operators zur String-Konkatenation ist, dass es zu Abstandsproblemen kommen kann, wenn Sie den Abstand zwischen den verketteten Strings nicht sorgfältig handhaben.

Hier ist ein Beispiel, bei dem ein Leerzeichen fehlt:

:::interactive_editor

```js
let firstName = "John";
let lastName = "Doe";

let fullName = firstName + lastName; 
console.log(fullName); // "JohnDoe"
```

:::

Wann immer Sie den `+`-Operator verwenden, um Strings zu verketten, ist es wichtig, mögliche Abstandsprobleme sorgfältig zu überprüfen.

Wenn Sie einen bestehenden String erweitern oder anhängen möchten, können Sie den `+=`-Operator verwenden. Das ist hilfreich, wenn Sie einen String nach und nach mit mehr Text aufbauen wollen.

Hier ist ein Beispiel, wie man mit dem `+=`-Operator einen String an einen anderen anhängt:

:::interactive_editor

```js
let greeting = 'Hello';
greeting += ', John!';

console.log(greeting); // "Hello, John!"
```

:::

Es ist wichtig zu wissen, dass Strings unveränderlich sind, was bedeutet, dass ein einmal erstellter String nicht mehr verändert werden kann.

In diesem Fall wird der ursprüngliche String von `Hello` nicht verändert. Stattdessen verweist greeting jetzt auf den neuen String von `Hello, John!`.

Eine weitere Möglichkeit, Strings zu verketten, ist die Verwendung der `concat()`-Methode.

Bevor wir mit der `concat()`-Methode beginnen, ist es wichtig, zunächst auf höherer Ebene zu verstehen, was eine Methode und eine Funktion sind.

In der Programmierung ist eine Funktion ein wiederverwendbarer Codeblock, der eine bestimmte Aufgabe ausführt und mit verschiedenen Eingaben aufgerufen werden kann. Eine Methode hingegen ist eine Art Funktion, die mit einem Objekt verbunden ist, das heißt, sie arbeitet mit den Daten, die in diesem Objekt enthalten sind.

In zukünftigen Lektionen werden wir viel tiefer darauf eingehen, wie Funktionen, Objekte und Methoden in JavaScript funktionieren. Aber für den Moment ist es wichtig zu verstehen, dass JavaScript Dutzende von Methoden hat, die Sie verwenden können, wie die `concat()`-Methode.

Hier ist ein Beispiel, wie man die `concat()`-Methode verwendet, um zwei Strings zusammenzufügen:

:::interactive_editor

```js
let str1 = 'Hello';
let str2 = 'World';

let result = str1.concat(' ', str2); 
console.log(result); // Hello World
```

:::

In diesem Beispiel verwenden wir die `concat()`-Methode, um `str1`, ein Leerzeichen (`' '`) und `str2` zu einem einzigen String zusammenzufügen.

Zusammenfassend ist der `+`-Operator am besten für einfache Verkettungen geeignet, besonders wenn Sie nur wenige Strings oder Variablen kombinieren müssen.

Der `+=`-Operator ist nützlich, wenn Sie einen String Schritt für Schritt aufbauen oder neuen Inhalt an eine bestehende String-Variable anhängen möchten.

Schließlich ist die `concat()`-Methode vorteilhaft, wenn Sie mehrere Strings miteinander verketten müssen.
