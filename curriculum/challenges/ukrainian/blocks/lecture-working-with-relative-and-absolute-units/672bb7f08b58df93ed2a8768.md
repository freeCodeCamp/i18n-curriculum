# --interactive--

У попередньому уроці ми вивчали абсолютні одиниці довжини, такі як пікселі. Хоча абсолютні одиниці можуть бути корисними в певних ситуаціях, будуть випадки, коли ви захочете використовувати відносні одиниці.

У цьому уроці ми дізнаємося про дві відносні одиниці: `em`s і `rem`s.

Одиниці `em` відносні до розміру шрифту елемента. Якщо ви використовуєте em для властивості `font-size`, розмір тексту буде відносним до розміру шрифту батьківського елемента.

Щоб краще зрозуміти, як це працює, давайте розглянемо приклад:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p class="para">I am a paragraph element</p>

<div class="blue-box"></div>
```

```css
.para {
  font-size: 20px;
  margin-bottom: 1.5em;
  border: 2px solid red;
}

.blue-box {
  background-color: blue;
  color: white;
  padding: 10px;
  width: 100px;
  height: 100px;
}
```

:::

Для HTML у нас є абзац і елемент `div`. Елемент абзацу має клас `para`, а елемент `div` має клас `blue-box`.

Для класу `para` ми встановили поле `font-size` у `20px` і відступ `margin-bottom` у `1.5em`. Це означає, що поле буде в 1,5 раза більшим за розмір шрифту елемента абзацу. `1.5em` дає 30 пікселів поля внизу абзацу. Ми також встановили кордон `border` у `2px solid red`, щоб краще бачити поля.

Для класу `blue-box` ми встановили колір фону `blue`, колір тексту `white` і відступ `padding` у `10px` з усіх чотирьох сторін.

З прикладу видно, що між низом елемента абзацу і синім блоком буде чіткий простір.

Отже, що станеться, якщо ми видалимо властивість `font-size` з класу `para`?

Нижнє поле буде відносним до розміру шрифту батьківського елемента. У цьому випадку батьківським елементом є елемент body, який має розмір шрифту за замовчуванням `16px`.

Добре використовувати одиниці `em`, коли ви працюєте з модульними компонентами, такими як кнопки або картки. Використовуючи одиниці `em`, ви можете забезпечити, що всі аспекти компонента (наприклад, відступи, поля і кордони) масштабуються пропорційно до розміру шрифту, зберігаючи сталі пропорції.

Отже, до цього моменту ми встановлювали розмір шрифту для елемента за допомогою пікселів. Але це створює певні проблеми для користувачів.

У налаштуваннях браузера ви можете контролювати розмір шрифту за замовчуванням.

Для людей з порушеннями зору вони можуть збільшувати розмір шрифту, щоб легше було читати. Але якщо ви встановлюєте розмір шрифту в пікселях у своїх вебдизайнах, текст не буде масштабуватися пропорційно до решти вмісту.

Один зі способів вирішення цієї проблеми — використовувати одиниці `rem` для типографіки. Одиниця `rem` відносна до розміру шрифту кореневого елемента, яким є елемент `html`.

За замовчуванням розмір шрифту елемента `html` становить `16px`. Якщо користувач збільшує розмір шрифту в налаштуваннях браузера, розмір шрифту елемента `html` також збільшується, і всі одиниці rem масштабуються пропорційно.

Ось приклад використання одиниці `rem` для розміру шрифту замість пікселів:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is regular text.</p>
<p class="para">This text is slightly larger.</p>
```

```css
.para {
  font-size: 1.2rem;
  margin-bottom: 1.5em;
  border: 2px solid red;
}
```

:::

Встановивши розмір шрифту в `1.2rem`, розмір шрифту елемента абзацу буде в 1,2 раза більшим за розмір шрифту кореневого елемента. Якщо користувач не змінював розмір шрифту за замовчуванням, розмір шрифту елемента абзацу буде `19.2px`, оскільки це 1,2 від `16px`.

Отже, коли слід використовувати одиниці `rem`? Одиниці `rem` переважні над пікселями для типографіки, тому що вони масштабуються пропорційно до налаштувань браузера користувача. Це робить ваш вміст більш доступним для користувачів з порушеннями зору.

Одиниці `rem` також допомагають підтримувати сталі відступи та макет між різними елементами.
