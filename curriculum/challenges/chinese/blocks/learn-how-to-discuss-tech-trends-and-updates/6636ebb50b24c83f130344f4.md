# --description--

单词 `without` 后跟动词的 `-ing` 形式，用于表示某个动作或状态的缺失。这是一种常见结构，用来描述在做某事的同时不做另一件事。

例如，`She left without saying goodbye.` 这意味着她离开了且没有说再见。
