# --description--

短语 `crazy part` 用来表达某事令人惊讶或出乎意料。它就像是在说 `the surprising thing is` 或 `the unexpected thing is`。

例如，`The crazy part is that he finished all the work in one day.` 这意味着他能这么快完成工作令人惊讶。
