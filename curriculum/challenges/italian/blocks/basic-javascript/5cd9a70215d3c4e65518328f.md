# --description--

In una <a href="/learn/javascript-algorithms-and-data-structures/basic-javascript/replace-loops-using-recursion" target="_blank" rel="noopener noreferrer nofollow">sfida precedente</a>, hai imparato come usare la ricorsione per sostituire un loop `for`. Ora, vediamo una funzione più complessa che restituisce un array di numeri interi consecutivi a partire da `1` fino al numero passato alla funzione.

Come menzionato nella sfida precedente, ci sarà un <dfn>caso base</dfn>. Il caso base indica alla funzione ricorsiva quando non deve più chiamare se stessa. È un caso semplice in cui il valore di ritorno è già noto. Ci sarà anche una <dfn>chiamata ricorsiva</dfn> che esegue la funzione originale con argomenti diversi. Se la funzione è scritta correttamente, alla fine si raggiungerà il caso base.

Per esempio, supponiamo di voler scrivere una funzione ricorsiva che restituisce un array contenente i numeri `1` fino a `n`. Questa funzione dovrà accettare un argomento, `n`, che rappresenta il numero finale. Poi dovrà chiamare se stessa con valori progressivamente più piccoli di `n` fino a raggiungere `1`. Potresti scrivere la funzione come segue:

```javascript
function countup(n) {
  if (n < 1) {
    return [];
  } else {
    const countArray = countup(n - 1);
    countArray.push(n);
    return countArray;
  }
}
console.log(countup(5));
```

Il valore `[1, 2, 3, 4, 5]` verrà mostrato nella console.

All’inizio, questo sembra controintuitivo dato che il valore di `n` *diminuisce*, ma i valori nell’array finale sono *in crescita*. Questo accade perché il push avviene per ultimo, dopo che la chiamata ricorsiva è tornata. Nel momento in cui `n` viene inserito nell’array, `countup(n - 1)` è già stato valutato e ha restituito `[1, 2, ..., n - 1]`.

# --instructions--

Abbiamo definito una funzione chiamata `countdown` con un parametro (`n`). La funzione dovrebbe usare la ricorsione per restituire un array contenente i numeri interi `n` fino a `1` basandosi sul parametro `n`. Se la funzione viene chiamata con un numero minore di 1, dovrebbe restituire un array vuoto. Per esempio, chiamare questa funzione con `n = 5` dovrebbe restituire l’array `[5, 4, 3, 2, 1]`. La tua funzione deve usare la ricorsione chiamando se stessa e non deve usare loop di alcun tipo.
