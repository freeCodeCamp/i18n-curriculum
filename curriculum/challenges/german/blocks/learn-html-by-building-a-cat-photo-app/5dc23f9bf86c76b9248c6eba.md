# --description--

Sie können Bilder zu Ihrer Website hinzufügen, indem Sie das `img`-Element verwenden. `img`-Elemente haben ein öffnendes Tag ohne schließendes Tag. Ein Element ohne schließendes Tag wird als <dfn>void element</dfn> bezeichnet.

Fügen Sie ein `img`-Element unter dem `p`-Element hinzu. Zu diesem Zeitpunkt wird im Browser kein Bild angezeigt.

# --hints--

Ihr `img`-Element sollte ein öffnendes Tag haben. Öffnende Tags haben diese Syntax: `<elementName>`.

```js
assert(document.querySelector('img'));
```

Ihr `img`-Element sollte kein schließendes Tag `</img>` haben.

```js
assert(!code.match(/<\/img\>/));
```

Sie sollten nur ein `img`-Element haben. Entfernen Sie alle zusätzlichen.

```js
assert(document.querySelectorAll('img').length === 1);
```

Ihr `img`-Element sollte unter dem `p`-Element stehen. Sie haben sie in der falschen Reihenfolge.

```js
const collection = [...document.querySelectorAll('p,img')].map(
  (node) => node.nodeName
);
assert(collection.indexOf('P') < collection.indexOf('IMG'));
```
