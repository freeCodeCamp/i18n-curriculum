# --explanation--

短縮形 `We'll` は `We will` の略で、将来の行動や計画を示しています。例えば：

- `I'll call you tomorrow.` - これは呼び出す行動が将来行われることを意味します。

- `We'll review the report later.` - これはレビューが計画されているが、まだ行われていないことを示しています。
