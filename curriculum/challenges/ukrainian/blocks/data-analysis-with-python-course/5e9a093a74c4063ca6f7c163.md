## --text--

Який метод має екземпляр `Cursor` і що він дозволяє робити?

## --answers--

Екземпляр `Cursor` має метод `.run()`, який дозволяє запускати SQL-запити.

---

Екземпляр `Cursor` має метод `.select()`, який дозволяє вибирати записи.

---

Екземпляр `Cursor` має метод `.execute()`, який прийматиме SQL-параметри для виконання над базою даних.
