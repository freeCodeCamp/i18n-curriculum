# --description--

Dieses Quiz überprüft Ihr Verständnis von Begrüßungen, Verabschiedungen und einfachen Vorstellungen auf Spanisch.

Lesen Sie jede Frage und wählen Sie die richtige Antwort aus. Es gibt nur eine richtige Antwort pro Frage.
