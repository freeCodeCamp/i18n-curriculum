# --description--

يمكنك الربط بصفحة أخرى باستخدام عنصر وسم الرابط (`a`).

إليك مثالًا يربط بـ `https://www.freecodecamp.org`:

```html
<a href="https://www.freecodecamp.org"></a>
```

أضف عنصر وسم الرابط بعد الفقرة يربط بـ `https://freecatphotoapp.com`. في هذه المرحلة، لن يظهر الرابط في المعاينة.

# --hints--

يجب أن يحتوي عنصر وسم الرابط (`a`) على وسم بداية. وسم البداية له هذه الصيغة النحوية: `<elementName>`.

```js
assert(document.querySelector('a'));
```

يجب أن يحتوي عنصر وسم الرابط (`a`) على وسم إغلاق. وسم الإغلاق يحتوي على `/` مباشرة بعد حرف `<`.

```js
assert(code.match(/<\/a\>/));
```

يجب أن يكون عنصر وسم الرابط (`a`) أسفل عنصر `p`. لقد وضعتهم بترتيب خاطئ.

```js
const collection = [...document.querySelectorAll('a, p')].map(
  (node) => node.nodeName
);
assert(collection.indexOf('P') < collection.indexOf('A'));
```

عنصر وسم الرابط (`a`) لا يحتوي على خاصية `href`. تحقق من وجود فراغ بعد اسم وسم البداية و/أو وجود فراغات قبل أسماء كل الخصائص.

```js
assert(document.querySelector('a').hasAttribute('href'));
```

يجب أن يربط عنصر وسم الرابط (`a`) بـ `https://freecatphotoapp.com`. إما أنك حذفت عنوان URL أو هناك خطأ مطبعي.

```js
assert(
  document.querySelector('a').getAttribute('href') ===
    'https://freecatphotoapp.com'
);
```

على الرغم من أنك ضبطت خاصية `href` لعنصر وسم الرابط (`a`) على الرابط الصحيح، يُنصح دائمًا بوضع قيمة الخاصية بين علامات اقتباس.

```js

  !/\<a\s+href\s*=\s*https:\/\/www.freecodecamp.org\/cat-photos/.test(code)
);
```
