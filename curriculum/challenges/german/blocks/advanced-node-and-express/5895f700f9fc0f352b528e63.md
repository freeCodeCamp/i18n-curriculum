# --description--

Bei diesen Aufgaben schreiben Sie Ihren Code mit einer der folgenden Methoden:

- Klonen Sie <a href="https://github.com/freeCodeCamp/boilerplate-advancednode/" target="_blank" rel="noopener noreferrer nofollow">dieses GitHub-Repository</a> und bearbeiten Sie die Aufgaben lokal.
- Verwenden Sie einen Website-Builder Ihrer Wahl, um das Projekt abzuschließen. Stellen Sie sicher, dass Sie alle Dateien aus unserem GitHub-Repository einbinden.

Eine Template-Engine ermöglicht es Ihnen, statische Template-Dateien (wie solche, die in *Pug* geschrieben sind) in Ihrer Anwendung zu verwenden. Zur Laufzeit ersetzt die Template-Engine Variablen in einer Template-Datei durch tatsächliche Werte, die Ihr Server bereitstellen kann. Anschließend wandelt sie das Template in eine statische HTML-Datei um, die an den Client gesendet wird. Dieser Ansatz erleichtert das Design einer HTML-Seite und erlaubt es, Variablen auf der Seite anzuzeigen, ohne dass der Client eine API-Anfrage machen muss.

`pug@~3.0.0` wurde bereits installiert und ist als Abhängigkeit in Ihrer `package.json`-Datei aufgeführt.

Express muss wissen, welche Template-Engine Sie verwenden. Verwenden Sie die `set`-Methode, um `pug` als Wert der `view engine`-Eigenschaft zuzuweisen:

```javascript
app.set('view engine', 'pug');
```

Fügen Sie danach eine weitere `set`-Methode hinzu, die die `views`-Eigenschaft Ihrer `app` auf das `./views/pug`-Verzeichnis zeigt. Damit teilen Sie Express mit, alle Views relativ zu diesem Verzeichnis zu rendern.

Verwenden Sie schließlich `res.render()` in der Route für Ihre Startseite und übergeben Sie `index` als erstes Argument. Dadurch wird das `pug`-Template gerendert.

Wenn alles wie geplant funktioniert hat, ist Ihre Startseite nicht mehr leer. Stattdessen zeigt sie eine Nachricht an, die bestätigt, dass Sie das Pug-Template erfolgreich gerendert haben!

Reichen Sie Ihre Seite ein, wenn Sie glauben, dass alles stimmt. Wenn Sie auf Fehler stoßen, können Sie <a href="https://forum.freecodecamp.org/t/advanced-node-and-express/567135#set-up-a-template-engine-1" target="_blank" rel="noopener noreferrer nofollow">das bis zu diesem Punkt abgeschlossene Projekt ansehen</a>.
