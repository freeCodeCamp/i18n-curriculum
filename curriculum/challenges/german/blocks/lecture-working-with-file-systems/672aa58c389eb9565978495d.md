# --description--

Eine Datei-Management-Anwendung ist eine Möglichkeit, Ihre Dateien auf dem Computer einfach zu speichern, zu organisieren und zu bearbeiten. Egal, ob Sie einen Mac oder Windows-Computer verwenden, es gibt eingebaute Anwendungen, mit denen Sie Ihre Dateien organisieren können.

Wenn Sie einen Windows-Computer haben, ist der Standard-Dateimanager der Datei-Explorer. Dieser wird verwendet, um Dateien und Ordner zu durchsuchen, zu suchen und zu verwalten sowie Dateioperationen wie Kopieren, Verschieben und Löschen durchzuführen.

Um den Datei-Explorer zu finden, können Sie zum Startmenü gehen oder die Windows-Logo-Taste auf Ihrer Tastatur drücken.

Um einen Ordner anzuheften, können Sie mit der rechten Maustaste klicken und „Anheften“ auswählen. Um eine Datei oder einen Ordner zu verschieben, müssen Sie ihn zuerst auswählen, dann „Ausschneiden“ wählen und anschließend zum neuen Speicherort navigieren, um ihn einzufügen.

Wenn Sie einen Mac-Computer haben, ist der Standard-Dateimanager der Finder. Dieser bietet Zugriff auf lokale und entfernte Dateien, unterstützt Dateivorschauen und bietet verschiedene Organisationsfunktionen wie Tags und intelligente Ordner.

Um auf dem Mac auf den Finder zuzugreifen, gehen Sie zum Dock unten und klicken auf das Finder-Symbol.

Sie werden eine Seitenleiste auf der linken Seite bemerken, in der Ihre Favoritenordner angezeigt werden.

Eine Möglichkeit, Ihre Dateien zu organisieren, ist die Verwendung von Tags. Um eine Datei oder einen Ordner zu taggen, können Sie das Element im Finder auswählen und dann mit der rechten Maustaste klicken. Im Dropdown-Menü sehen Sie die Option „Tags...“ und können ein benutzerdefiniertes farbiges Tag anwenden.

Um nach getaggten Elementen zu suchen, können Sie zum unteren Rand der Seitenleiste scrollen und auf das entsprechende farbige Tag klicken oder im Suchfeld nach dem Farbnamen des Tags suchen.

Eine weitere Möglichkeit, Ihre Dateien zu organisieren, ist die Verwendung von intelligenten Ordnern. Intelligente Ordner sind Sammlungen von Dateien, die automatisch basierend auf von Ihnen festgelegten Kriterien wie Dateityp, Datum oder Schlüsselwörtern organisiert werden.

Egal, ob Sie den Datei-Explorer unter Windows oder den Finder auf dem Mac verwenden, diese Anwendungen bieten Flexibilität, um Ihre Dateien mit Funktionen wie Tags, Suche und intelligenten Ordnern zu verwalten.
