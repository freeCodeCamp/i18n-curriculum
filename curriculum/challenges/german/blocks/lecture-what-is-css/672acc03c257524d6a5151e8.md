# --interactive--

In CSS werden die `width`- und `height`-Eigenschaften verwendet, um die Abmessungen von Elementen auf einer Webseite zu steuern.

Diese Eigenschaften können in verschiedenen Einheiten definiert werden, wie Pixel (`px`), Prozent (`%`), Viewport-Einheiten (`vw`, `vh`) und mehr.

Die `width`-Eigenschaft legt die Breite eines Elements fest. Wenn Sie keine Breite angeben, ist standardmäßig `auto` eingestellt. Dadurch bestimmt der Browser die Breite des Elements basierend auf seinem Inhalt, dem übergeordneten Element und dem Anzeige-Typ. Für ein `div`-Element bewirkt `width: auto`, dass es sich auf die volle Breite seines übergeordneten Containers ausdehnt.

Die `height`-Eigenschaft legt die Höhe eines Elements fest. Ebenso ist die Höhe standardmäßig `auto`, was bedeutet, dass sie sich an den Inhalt anpasst.

Hier ist ein Beispiel mit den `width`- und `height`-Eigenschaften:

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 100px;
      height: 100px;
      background-color: skyblue
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

In diesem Beispiel haben wir ein `div`-Element mit der Klasse `box`. Dieses Element wird `100px` in Höhe und Breite einnehmen, während die Hintergrundfarbe `skyblue` sein wird.

Pixel sind eine feste Maßeinheit in CSS, die eine präzise Kontrolle über die Abmessungen ermöglicht.

Die `min-width`-Eigenschaft gibt die minimale Breite an, die ein Element haben kann. Selbst wenn der Inhalt kleiner ist, wird das Element nicht unter diesen Wert schrumpfen.

Die `min-height` gibt die minimale Höhe an, die ein Element haben kann. Sie stellt sicher, dass das Element nicht kleiner als der festgelegte Wert wird.

Hier ist ein Beispiel:

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 50px;
      min-width: 100px;
      height: 50px;
      min-height: 100px;
      background-color: lightcoral;
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

Das obige Beispiel zeigt, wie `min-width` und `min-height` funktionieren.

Obwohl die Box ihre `width` und `height` auf 50px gesetzt hat, wird sie tatsächlich `100px` mal `100px` groß sein. Das liegt daran, dass `min-width` und `min-height` auf `100px` gesetzt sind, welche größer sind als die angegebenen `width` und `height`.

Denken Sie daran, wenn `min-width` oder `min-height` größer sind als `width` oder `height`, überschreiben sie die kleineren Werte. Das stellt sicher, dass Elemente nicht zu klein werden, was wichtig ist, um ein konsistentes und benutzbares Design zu erhalten.

Die `max-width` gibt die maximale Breite an, die ein Element erreichen kann, selbst wenn genügend Platz für eine größere Breite vorhanden ist.

Die `max-height` gibt die maximale Höhe an, die ein Element erreichen kann, unabhängig von der Größe des Inhalts.

Hier ist ein Beispiel:

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 200px;
      max-width: 150px;
      height: 200px;
      max-height: 150px;
      background-color: lightgreen;
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

Das obige Beispiel zeigt, wie `max-width` und `max-height` `width` und `height` überschreiben. Obwohl die Box auf `200px` mal `200px` gesetzt ist, wird sie tatsächlich `150px` mal `150px` groß sein. Das liegt daran, dass `max-width` und `max-height` auf `150px` gesetzt sind, was kleiner ist.

Denken Sie daran, wenn `max-width` oder `max-height` kleiner sind als `width` oder `height`, haben sie Vorrang. Das ist wichtig, um die maximale Größe von Elementen in Ihren Layouts zu steuern.

CSS priorisiert `min-width` und `min-height` gegenüber `width` und `height`. `max-width` und `max-height` begrenzen die Abmessungen, wenn Werte ihre Grenzen überschreiten.

Zum Beispiel, wenn Sie `width` auf `600px` und `max-width` auf `500px` setzen, wird das Element auf `500px` Breite begrenzt. Die `max-width` überschreibt die größere Einstellung und hält das Element innerhalb der angegebenen maximalen Größe.

Das stellt sicher, dass Elemente innerhalb der gewünschten Größenbereiche bleiben, unabhängig von den Standardwerten für Breite und Höhe.
