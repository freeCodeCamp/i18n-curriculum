# --explanation--

`Was there`는 `There was`의 의문문 형태로, 과거에 어떤 일이 있었는지 또는 발생했는지 묻는 데 사용됩니다. 주어와 동사의 위치를 바꿔 질문을 만듭니다. 예를 들어:

- `There was a problem with the layout.` (평서문 - 문제가 발생했다고 말하는 경우)  

- `Was there a problem with the layout?` (의문문 - 문제가 발생했는지 묻는 경우)

이는 과거에 어떤 일이 있었는지 또는 발생했는지 확인하는 데 도움이 됩니다.
