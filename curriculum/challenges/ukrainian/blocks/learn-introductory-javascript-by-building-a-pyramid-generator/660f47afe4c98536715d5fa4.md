# --description--

Оператор <dfn>строгої нерівності</dfn> `!==` дозволяє перевірити, чи два значення не рівні або не мають однакового типу. Синтаксис схожий на оператор рівності: `value !== 4`.

Наразі цикл `while` виконується лише якщо `continueLoop` є true. Оновіть умову циклу `while`, щоб перевірити, чи `done` не дорівнює `count`.
