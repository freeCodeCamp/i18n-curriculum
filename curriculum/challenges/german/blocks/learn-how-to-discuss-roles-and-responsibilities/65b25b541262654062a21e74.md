# --description--

Das Verb `recognize` bedeutet, jemanden oder etwas zu kennen, weil Sie ihn oder es schon einmal gesehen, gehört, getroffen oder erlebt haben. Zum Beispiel, wenn Sie einen Freund auf einer Party sehen, `recognize` Sie ihn, weil Sie wissen, wie er aussieht.

Die Phrase `right away` bedeutet schnell oder sehr bald. Zum Beispiel `If you find a bug in the code, tell me and I'll fix it right away.` Das bedeutet, wenn Sie ein Problem im Computerprogramm entdecken, sagen Sie mir Bescheid, und ich werde es sofort korrigieren.
