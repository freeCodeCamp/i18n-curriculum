# --explanation--

`Something` 可用于提问中，当说话者假设某物存在时。

在 `Is it something serious?` 中，James 假设存在问题——因为 Maria 告诉他客户抱怨了这个问题——并询问这是否严重。相比之下，`anything` 用于提问时，表示说话者甚至不知道某事是否存在。例如：

- `Is there something wrong with the software?` - 这里，说话者假设存在问题并询问具体是什么。

- `Is there anything wrong with the software?`——这里，说话者只是好奇是否有任何问题。他们并不是在寻求确认。
