# --explanation--

`Could` і `might` — це модальні дієслова, які використовують для опису можливостей.

`Could` вказує, що щось можливо, але не певно. Наприклад:

`If we don't fix this bug, it could cause a system crash.` — Існує ймовірність, що система перестане працювати.

`Might` вказує, що щось можливо, але менш певно, ніж `could`. Наприклад:

`We might need to rewrite some parts of the code.` — Можливо, знадобляться зміни.
