# --interactive--

DOM과 웹 개발에서 왜 중요한지 배워봅시다. DOM은 Document Object Model의 약자입니다. HTML 문서와 상호작용할 수 있게 해주는 프로그래밍 인터페이스입니다.

DOM을 사용하면 웹페이지의 요소를 추가, 수정, 삭제할 수 있습니다. 요소가 이벤트를 듣고 반응하게 만들어 웹사이트를 인터랙티브하게 만들 수도 있습니다.

DOM에서 HTML 문서는 노드 트리로 나타납니다. 각 노드는 HTML 문서의 HTML 요소를 나타냅니다:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>DOM Example</title>
  </head>
  <body>
    <h1>What is the DOM?</h1>
    <h2>Let's learn about the DOM</h2>
  </body>
</html>
```

다음은 예제의 기본 DOM 구조를 나타내는 도표입니다:

```md
Document
========

HTML
-----

- Head
  - Title
- Body
  - H1
  - H2   
```

물론, 웹페이지의 HTML 마크업 구조에 따라 더 상세하고 복잡할 수 있습니다.

HTML 문서는 DOM 계층 구조에서 루트 노드입니다. 하나의 자식 노드인 `html` 요소를 가집니다. 이 요소는 HTML 문서의 루트 요소로, 모든 다른 노드가 여기서 파생됩니다.

`html` 요소는 두 개의 자식 노드 `head`와 `body`를 가집니다.

`head` 요소는 문서에 대한 메타데이터를 포함합니다. 웹페이지에 필수 정보를 제공합니다.

`body` 요소는 웹페이지의 보이는 콘텐츠를 포함합니다.

이 요소들은 JavaScript로 접근할 수 있고, 코드에서 다루거나 동적으로 수정할 수도 있습니다. 이것이 DOM과 웹 API의 힘입니다.

JavaScript에서 이 요소들에 접근하려면 `getElementById()`와 `querySelector()` 메서드를 사용할 수 있습니다. 이 메서드들은 JavaScript로 DOM과 상호작용하는 표준화된 방법을 제공하기 때문에 웹 API입니다.

`getElementById()`를 사용하면 지정한 `id` 값을 가진 HTML 요소를 나타내는 객체를 얻을 수 있습니다. `id`는 모든 HTML 문서에서 고유해야 하므로 이 메서드는 하나의 `Element` 객체만 반환합니다. 예를 보세요:

```js
const container = document.getElementById("container");
```

이 JavaScript 코드 한 줄은 `id` 값이 `container`인 요소를 가져와 그 객체를 JavaScript 상수에 할당합니다. `id`는 인수로 따옴표 안에 전달해야 합니다. 이 객체를 콘솔에 출력하면 결과를 볼 수 있습니다:

:::interactive_editor

```html
<div id="container">
  <h1>Hello, World!</h1>
  <p>Welcome to learning about the DOM.</p>
</div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
console.log(container); 
```

:::

`querySelector()`는 `getElementById()`보다 더 포괄적입니다. `querySelector()`를 사용하면 인수로 전달된 CSS 선택자와 일치하는 HTML 문서 내 첫 번째 요소를 얻을 수 있습니다.

이 예제에서는 `section` 선택자와 일치하는 요소를 가져옵니다:

:::interactive_editor

```html
<section>
  <h2>Section Title</h2>
  <p>This is a section of the webpage.</p>
</section>
<script src="index.js"></script>  
```

```js
const sectionEl = document.querySelector("section");
console.log(sectionEl);
```

:::

클래스 이름으로 요소를 선택하려면 `querySelector()`를 사용할 때 클래스 이름 앞에 점(`.`)을 붙여야 합니다. 예를 보세요:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="highlight">
  <p>This content is highlighted.</p>
</div>
<script src="index.js"></script>
```

```css
body {
  background-color: lightgray;
}

.highlight {
  background-color: yellow;
}
```

```js
const highlightEl = document.querySelector(".highlight");
console.log(highlightEl);
```

:::

여러 요소와 일치시키는 다른 메서드도 있습니다. 예를 들어 `getElementsByClassName()`와 `querySelectorAll()`가 있습니다. 이들은 앞으로 수업에서 더 배우게 됩니다.

웹 API를 사용해 DOM을 효율적으로 조작하는 방법을 이해하면 강력하고 인터랙티브한 웹 애플리케이션을 만들 수 있습니다.
