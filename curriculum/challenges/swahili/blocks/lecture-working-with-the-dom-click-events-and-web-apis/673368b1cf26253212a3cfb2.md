# --description--

Tujifunze kuhusu njia ya `querySelectorAll()`. Unaweza kutumia njia hii kupata orodha ya vipengele vyote vya DOM vinavyolingana na kichaguzi maalum cha CSS.

Hivi ndivyo unavyoweza kuita njia ya `querySelectorAll()`:

```js
document.querySelectorAll(selectors);
```

Iite kwenye kitu cha `document` na pita mfuatano wa herufi wenye vichaguzi vya CSS kama hoja. Hoja lazima iwe mfuatano halali wa kichaguzi cha CSS. Vinginevyo, hitilafu ya `SyntaxError` itatupwa.

`querySelectorAll()` inarudisha kitu cha `NodeList`, mkusanyiko wa nodi zinazolingana na kichaguzi cha CSS kilichotajwa.

Orodha itakuwa na kitu kimoja cha `Element` kwa kila kipengele kinacholingana na vichaguzi vya CSS. Ikiwa hakuna vinavyolingana, orodha itakuwa tupu.

Vipengele vitakuwa katika mpangilio walivyoonekana katika hati ya HTML. Kwa mfano, unaweza kulinganisha vipengele vyote vya aina maalum:

```js
document.querySelectorAll("div");
```

Vipengele vyote vyenye darasa maalum:

```js
document.querySelectorAll(".rounded");
```

Vipengele vyote vyenye kitambulisho maalum:

```js
document.querySelectorAll("#logo");
```

Au vipengele vyote vyenye sifa maalum, kama viungo vyote vinavyowapeleka watumizi kwenye URL maalum:

```js
document.querySelectorAll("a[href='https://www.freecodecamp.org/']");
```

Unaweza pia kutumia vichaguzi vya CSS tata zaidi, kama hiki, kinacholingana na vitu vyote vya orodha isiyopangwa ndani ya kipengele cha orodha isiyopangwa chenye darasa `ingredients`:

```js
document.querySelectorAll("ul.ingredients li");
```

Kwa kuwa njia hii itarudisha mkusanyiko wa nodi, unaweza kuweka mkusanyiko huu katika kigezo na kuvitumia katika programu zako kufanya kazi na vipengele binafsi, kama unavyoona katika mfano huu wenye sintaksia ya jumla:

```js
const matches = document.querySelectorAll(selectors); 
```

Kwa mfano, ikiwa una orodha isiyopangwa yenye vitu vitatu vya orodha:

```html
<ul class="ingredients">
  <li>Flour</li>
  <li>Cheese</li>
  <li>Water</li>
</ul>
```

Unaweza kuita njia ya `querySelectorAll()` kulinganisha vitu vyote vya orodha ndani ya orodha isiyopangwa yenye darasa `ingredients` na kuweka thamani ya return kwenye kigezo cha `matches`:

```js
const matches = document.querySelectorAll("ul.ingredients li");
```

Hii itarudisha mkusanyiko wa nodi tatu. Kila nodi inawakilisha kipengele cha kitu cha orodha:

```js
// NodeList(3)
{
  0: `<li>Flour</li>`,
  1: `<li>Cheese</li>`,
  2: `<li>Water</li>`,
  length: 3,
}
```

Unaweza kufanya kazi na mkusanyiko huu kama unavyofanya na safu yoyote ya JavaScript.

Kwa mfano, unaweza kuionyesha kwenye konsoli kwa `console.log()`:

```js
console.log(matches);
```

Unaweza pia kuangalia urefu wa mkusanyiko kwa sifa ya `length`:

```js
console.log(matches.length);
```

Matokeo yatakuwa nambari kamili inayowakilisha idadi ya nodi katika mkusanyiko:

```md
3
```

Unaweza pia kupata vipengele binafsi vya mkusanyiko kwa kielezo chao kinacholingana. Kipengele cha kwanza kitakuwa kwenye kielezo cha `0`:

```js
console.log(matches[0]);
console.log(matches[1]);
console.log(matches[2]);
```

Huu ndio matokeo:

```html
<li>Flour</li>
<li>Cheese</li>
<li>Water</li>
```

Vipengele vinachapishwa moja moja kwenye konsoli.

Unaweza pia rudia juu ya vipengele kwa kutumia mzunguko wa `for`:

```js
for (let i = 0; i < matches.length; i++) {
  console.log(matches[i]);    
}
```

Matokeo yatakuwa sawa kabisa katika kesi hii. Lakini hii inaweza kuwa na nguvu sana kwa kufanya kazi na vipengele vyote vya mkusanyiko.

Njia ya `querySelectorAll()` katika JavaScript ni chombo chenye nguvu cha kuchagua vipengele vingi kulingana na vichaguzi vyao vya CSS. Kwa njia hii, unaweza kuchagua vipengele maalum ndani ya kurasa zako za mtandao na kuvitumia kama inavyohitajika.
