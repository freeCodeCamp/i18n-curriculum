# --explanation--

`Past Continuous`時制（`was/were`＋動詞の`-ing`形）は、過去のある期間にわたって起こっていた行動や考えを表すために使われ、しばしば別の出来事と同時に起こっていることを示します。これにより、誰かがすでに同じ考えを持っていたことを示せます。例えば：

`I was reading the report when you called.` – これは電話がかかってきたときに読書の行動が続いていたことを意味します。

この場合、`I was thinking the same thing`はマークの考えがリサのコメントの最中またはそれ以前にあったことを意味し、賛成を示しています。
