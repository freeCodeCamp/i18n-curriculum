# --description--

このタスクでは、`Did you get it?`、`Got it`、`What else?` のようなフレーズについて学びます。これらのフレーズは英会話でよく使われます。`Did you get it?` は、誰かが何かを理解したかどうかを尋ねる質問です。`Got it` は `I understood.` を短く言ったものです。`What else?` は追加の情報や項目を求めるときに使います。

例えば、会議で誰かが `Did you get it?` と言い、あなたは `Got it, what else do we need to discuss?` と返答できます。
