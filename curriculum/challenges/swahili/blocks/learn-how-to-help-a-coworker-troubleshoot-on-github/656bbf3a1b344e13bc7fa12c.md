# --description--

`Strange` inamaanisha jambo lisilo la kawaida au lisilotarajiwa. Ni kama kupata kitu tofauti na kile unachokiona kawaida. Kwa mfano: `A strange sound in a car might mean a problem.`

`error` ni kosa, hasa lile linalosababisha kitu kisifanye kazi ipasavyo. Kwa mfano: `An error in a code means there is a mistake that needs fixing.`
