# --description--

在 Python 中，例外處理是撰寫強固且容錯程式的核心零件。它允許你以結構化的方式預期、捕捉並回應錯誤。

例外處理是捕捉和管理程式執行期間發生錯誤的處理程序，讓你的程式碼不會意外當機。

Python 提供 `try`、`except`、`else` 和 `finally` 區塊，以優雅地處理錯誤。以下是一個基本範例：

```py
try:
    x = 10 / 0
except ZeroDivisionError:
    print("You can't divide by zero!")
```

- `try`：你預期可能會發生錯誤的程式碼區塊。

- `except`：如果在 try 內引發指定型別的錯誤，則執行此區塊。

- 在這種情況下，除以零會引發 `ZeroDivisionError`，然後該錯誤會被捕捉並控制代碼。

以下是一個範例，也展示如何使用 `else` 和 `finally` 區塊：

```py
try:
    x = 10 / 2
except ZeroDivisionError:
    print("You can't divide by zero!")
else:
    print('Division successful:', x)
finally:
    print('This block always runs.')
```

- `else`：如果在 try 區塊中沒有引發例外，則執行。

- `finally`：無論是否發生例外，都會執行。適用於關閉檔案或釋放資源等清理工作。

你也可以使用多個獨立的 `except` 區塊來捕捉多個例外：

```py
try:
    number = int('abc')
    result = 10 / number
except ValueError:
    print('That was not a valid number.')
except ZeroDivisionError:
    print("Can't divide by zero.")
```

透過使用獨立的 `except` 語句，你可以讓你的錯誤回應更具體且更有用。

你也可以使用例外物件，通常會使用 `as` 關鍵字將其別名為另一個名稱。這裡我們使用 `e` 作為錯誤物件的別名：

```py
try:
    x = 1 / 0
except ZeroDivisionError as e:
    print(f'Error occurred: {e}')
```

使用 `e` 讓你能存取實際的錯誤訊息或物件，以便記錄或偵錯。

你也可以透過將例外指定為一個元組，在單一 `except` 語句中捕捉多個例外：

```py
try:
    number = int(input('Enter a number: '))
    result = 10 / number
except (ValueError, ZeroDivisionError) as e:
    print(f'Error occurred: {e}')
```

例外處理允許你的程式優雅地從錯誤中復原。透過使用 `try`、`except`、`else` 和 `finally`，你可以預見潛在問題並建置更具韌性的應用程式。
