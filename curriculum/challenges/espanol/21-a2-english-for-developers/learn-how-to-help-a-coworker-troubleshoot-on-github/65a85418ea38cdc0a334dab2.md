# --description--

`To roll back` significa volver a un estado o versión anterior de algo. Por ejemplo:

`If the new app is not working, try rolling back to the old version.`

Una `versión previa` es una forma anterior de algo, como una versión antigua de un software. Vamos a usarlo en la misma oración que arriba:

`If the new app is not working, try rolling back to the previous version.`
