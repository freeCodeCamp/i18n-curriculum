# --description--

Eine `function` ist eine Reihe von Anweisungen, die eine bestimmte Aufgabe ausführen. `Functions` können den Code organisierter, wiederverwendbarer und leichter lesbar machen. Zum Beispiel könnte eine `function` die Summe von zwei Zahlen berechnen oder eine Nachricht an den Benutzer anzeigen.

Stellen Sie sich eine `function` wie eine kleine Maschine in einer Fabrik vor; Sie geben ihr eine Eingabe, sie verarbeitet diese Eingabe und gibt Ihnen dann eine Ausgabe zurück.
