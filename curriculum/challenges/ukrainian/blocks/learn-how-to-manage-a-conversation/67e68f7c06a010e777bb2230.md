# --explanation--

`It seems to be` + дієслово у формі `-ing` використовується, щоб описати те, що, здається, відбувається, хоча це може бути не підтверджено. Наприклад:

`The app seems to be running slower today.` – Це означає, що здається, застосунок працює повільніше, ніж зазвичай.

`To cause something to` + дієслово означає спричинити щось або викликати дію. Наприклад:

`Low memory can cause the program to crash.` – Це означає, що низька пам’ять є причиною, чому програма припиняє працювати.
