# --description--

整数 $1 < a_1 < a_2 < \ldots < a_n$ の値が与えられたとき、整数値 $q_k ≥ 0$ のみを使った線形結合 $q_1a_1 + q_2a_2 + \ldots + q_na_n = b$ を考えます。

与えられた $a_k$ の集合に対して、すべての $b$ の値が可能とは限らないことに注意してください。例えば、$a_1 = 5$、$a_2 = 7$ の場合、$q_1 ≥ 0$ および $q_2 ≥ 0$ で $b$ が 1、2、3、4、6、8、9、11、13、16、18、または 23 になるような組み合わせは存在しません。

実際、$a_1 = 5$ と $a_2 = 7$ に対して、23 は不可能な最大の $b$ の値です。したがって、$f(5, 7) = 23$ と呼びます。同様に、$f(6, 10, 15) = 29$ および $f(14, 22, 77) = 195$ であることが示せます。

$p$、$q$、$r$ が素数で $p < q < r < 5000$ のとき、$\sum f(pq, pr, qr)$ を求めてください。
