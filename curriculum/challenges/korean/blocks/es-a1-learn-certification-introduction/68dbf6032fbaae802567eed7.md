# --description--

스페인어 커리큘럼을 컴퓨터로 진행할 예정이라면, `ñ` 같은 악센트와 특수 문자를 입력할 수 있도록 컴퓨터가 준비되어 있는지 확인하세요.

Windows든 Mac이든 설정하는 데 몇 분밖에 걸리지 않아 빠르게 완료할 수 있습니다.

## Windows 11에서 설정하기

1. 시작 메뉴에서 **설정**을 여세요.

2. **시간 및 언어**로 이동한 다음 **언어 및 지역**을 클릭하세요.

3. **선호하는 언어**에서 **언어 추가**를 선택하세요.

4. **스페인어**를 검색하고 `Español (Latin America)` 같은 변형을 선택하세요.

5. **다음**을 클릭하고 안내에 따라 단계를 진행하세요.

언어가 설치되면:  

- **선호하는 언어** 목록에서 해당 언어를 찾으세요.

- 줄임표를 클릭해 추가 옵션을 표시한 후 **언어 옵션**을 선택하세요.

- **키보드**로 이동해 **키보드 추가**를 클릭하고 원하는 스페인어 키보드 레이아웃을 선택하세요.

키보드 간 전환 방법:  

- **Windows 키 + 스페이스바**를 누르세요.

- 또는 작업 표시줄의 언어 아이콘(보통 **ENG**로 표시됨)을 클릭하고 스페인어를 선택하세요.

## Mac에서 설정하기

1. **애플 메뉴**를 열고 **시스템 설정**을 선택하세요.

2. **키보드**로 이동하세요.

3. **텍스트 입력** > **입력 소스**로 가세요.

4. **편집...**을 클릭하세요.

5. 하단의 **플러스 (+)** 버튼을 클릭하고 목록에서 **스페인어**를 선택하세요.

6. **추가**를 클릭하세요.

이제 스페인어 커리큘럼을 진행하며 악센트 표시와 특수 문자가 필요한 작업도 문제없이 완료할 수 있습니다.
