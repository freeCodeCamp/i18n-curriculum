# --description--

JavaScriptには、厳密等値演算子`===`を使って型と値の両方を比較する機能もあります。この演算子は、2つの値が等しくかつ同じ型であるかどうかをチェックします。例えば：

```javascript
let x = 5;
let y = "5";

let result = x === y;

console.log(result); // false
```

この例では、`result`変数には`false`が代入されます。なぜなら`x`は数値で、`y`は文字列だからです。しかし、`==`演算子を使うと、JavaScriptが文字列を数値に変換して値を比較するため、結果は`true`になります。

この演算子は値と型の両方の等しさをチェックするため、厳密等値演算子と呼ばれます。値を比較するときに予期しない結果を避けるため、`===`演算子の使用がよく推奨されます。

また、2つの値が等しくなくかつ同じ型であるかをチェックする厳密不等値演算子`!==`もあります。

厳密等値演算子の使い方は多岐にわたり、その動作を理解することが重要です。
