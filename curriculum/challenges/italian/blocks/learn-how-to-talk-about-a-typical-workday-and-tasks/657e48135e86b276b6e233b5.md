# --description--

`To revoke` significa annullare ufficialmente o ritirare una decisione, un diritto o un permesso. Per esempio, `The administrator may revoke access if rules are broken`.

`To issue` in questo contesto significa fornire o distribuire qualcosa ufficialmente, come badge o permessi. Per esempio, `The office issues ID cards to all new employees`.

`Access control` si riferisce alla pratica di limitare e gestire l’ingresso a un luogo o a una risorsa. Per esempio, `Access control systems are used to secure the building`.
