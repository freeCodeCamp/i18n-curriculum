# --description--

在拋物線 $y = \frac{x^2}{k}$ 上，選擇三個點 $A(a, \frac{a^2}{k})$、$B(b, \frac{b^2}{k})$ 和 $C(c, \frac{c^2}{k})$。

令 $F(K, X)$ 為整數四元組 $(k, a, b, c)$ 的數量，使得三角形 $ABC$ 至少有一個角為 45°，且滿足 $1 ≤ k ≤ K$ 及 $-X ≤ a &lt; b &lt; c ≤ X$。

例如，$F(1, 10) = 41$ 和 $F(10, 100) = 12\\,492$。

尋找 $F({10}^6, {10}^9)$。
