## The `date()` Object and Common Methods

- **定義**: `date()` オブジェクトは、JavaScriptで日付や時刻を作成、操作、フォーマットするために使われます。次の例では、`new` キーワードを使って`Date` オブジェクトの新しいインスタンスを作成し、その`Date` オブジェクトを変数`now` に代入しています。もし`now` の値をコンソールに出力すると、コードを実行しているコンピューターのシステムクロックに基づいた現在の日付と時刻が表示されます。

:::interactive_editor

```js
const now = new Date();
console.log(now); // prints the current date and time
```

:::

- **`Date.now()` メソッド**: このメソッドは現在の日付と時刻を取得するために使われます。`Date.now()` は1970年1月1日00:00:00 UTCからのミリ秒数を返します。これはUnixエポック時間として知られています。Unixエポック時間は整数で表されるため、コンピューターシステムで日付や時刻を表す一般的な方法です。UTCは協定世界時（Universal Time Coordinated）の略で、世界が時計や時間を調整するための基準となる時間です。
- **`getDate()` メソッド**: このメソッドは現在の日付に基づいて月の日を取得します。`getDate()` は1から31までの整数値を返します。日付が無効な場合は`NaN`（Not a Number）を返します。

:::interactive_editor

```js
const now = new Date("2014-10-15");
const date = now.getDate();
console.log(date); // 15
```

:::

- **`getMonth()` メソッド**: このメソッドは月を取得するために使われます。月はゼロベースなので、1月は0、2月は1、というように続きます。この例では出力が9で、これは10月に対応します。月が無効な場合は`NaN`を返します。

:::interactive_editor

```js
const now = new Date("2014-10-15");
const month = now.getMonth();
console.log(month); // 9
```

:::

- **`getFullYear()` メソッド**: このメソッドは西暦年を取得するために使われます。年が無効な場合は`NaN`を返します。

:::interactive_editor

```js
const now = new Date("2014-10-15");
const year = now.getFullYear();
console.log(year); // 2014
```

:::

## 日付のフォーマット方法の違い

- **`toISOString()` メソッド**: このメソッドは拡張`ISO`（ISO 8601）形式で日付をフォーマットするために使われます。ISO 8601は日付と時刻を表す国際標準です。フォーマットは`YYYY-MM-DDTHH:mm:ss.sssZ` です。

:::interactive_editor

```js
const date = new Date("2014-10-15");
console.log(date.toISOString()); // "2014-10-15T00:00:00.000Z"
```

:::

- **`toLocaleDateString()` メソッド**: このメソッドはユーザーのロケールに基づいて日付をフォーマットするために使われます。

:::interactive_editor

```js
const date = new Date("2014-10-15");
console.log(date.toLocaleDateString("en-US"));  // "10/15/2014"
```

:::

`toLocaleDateString()` メソッドは2つのオプションのパラメータを受け取ります：locales と options です。

locales パラメータは使用するロケールを表す文字列です。例えば、英語（アメリカ合衆国）には`"en-US"`、フランス語（フランス）には`"fr-FR"` を渡せます。locales パラメータを渡さない場合はデフォルトのロケールが使われます。2つ目のオプションパラメータは options です。このパラメータは日付文字列のフォーマットを指定できるオブジェクトです。

:::interactive_editor

```js
const date = new Date("2014-10-15");
const options = {
  weekday: "long",
  year: "numeric",
  month: "long",
  day: "numeric",
};
console.log(date.toLocaleDateString("en-GB", options)); // "Wednesday, 15 October 2014"
```

:::
