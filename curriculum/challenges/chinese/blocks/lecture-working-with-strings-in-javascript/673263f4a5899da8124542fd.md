# --interactive--

`prompt()` 方法是 JavaScript 与用户交互的重要组成部分。 这是通过弹出式小对话框获取用户输入信息的最简单方法之一。

在网页需要用户提供信息的情况下你会经常看到使用它，如姓名或其他形式的文本输入。

那么，`prompt()` 方法具体是做什么的呢？ 它打开一个对话框，要求用户输入一些内容，然后以字符串形式返回用户输入的文本。

`prompt()` 方法需要两个参数：第一个参数是将出现在对话框中的信息，通常是提示用户输入信息。 第二个是默认值，是可选的，将作为填充输入字段的初始值。

```js
prompt(message, default);
```

这是它如何工作的示例。

**注意**：此示例包含你尚未学习的代码。不要担心尝试理解代码中的所有内容。这只是为了说明 `prompt()` 方法的工作原理，并确保提示符不会在页面装载时立即出现，因为这可能被视为侵入式。如果你启用了交互式编辑器，可以自己尝试。

:::interactive_editor

```html
<button id="prompt-btn">Show Prompt</button>
<p id="output"></p>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("prompt-btn");
const output = document.getElementById("output");
btn.addEventListener("click", () => {
  const userName = prompt("What is your name?", "Guest");
  output.textContent = "Hello, " + userName + "!";
});
```

:::

在此示例中，当用户点击按钮时，`prompt()` 方法会显示一个带有消息 `What is your name?` 和初始包含值为 `Guest` 的输入字段的对话框。

如果用户输入他们的名字并按下 "OK"，`userName` 变量将保存输入的值。如果用户按下 "Cancel"，`userName` 变量将被设置为 `null`。`null` 表示用户未提供任何输入。输出段落随后将使用提供的名字或如果用户取消则使用 `null` 显示问候消息。你将在后续课程中学习避免在用户取消提示符时显示 `null` 的技巧。

请记住，`prompt()` 方法将停止脚本的执行，直到用户与对话框进行交互。

这意味着在用户提供输入并点击“OK”或取消提示之前，JavaScript 代码的其余部分不会运行。

还需要考虑的一点是，虽然 `prompt()` 对于快速测试或小型应用程序很有用，但在现代复杂的网络应用程序中，由于其破坏性和在不同浏览器中的不一致行为，通常会避免使用它。

通过了解 `prompt()` 方法，你可以获得一种与用户交互并直接通过浏览器获取信息的简单方法，尽管这种方法在现代 Web 应用程序中可能并不广泛使用。
