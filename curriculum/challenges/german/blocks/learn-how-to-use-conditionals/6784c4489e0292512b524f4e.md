# --explanation--

Lisa verwendet den `Second Conditional`, um über eine hypothetische Situation zu sprechen.

Der `Second Conditional` wird verwendet, um über irreale oder unwahrscheinliche Situationen in der Gegenwart oder Zukunft zu sprechen. Er folgt der Struktur `if` + `Present Simple` oder `Past Continuous` (für die Bedingung) und `might / would / could` + Grundform des Verbs (für die Folge). Zum Beispiel:

- `If I had more time, I would learn a new programming language.` – Dies beschreibt eine hypothetische Situation, in der mehr Zeit zum Lernen führen würde.

- `If the server were faster, it could handle more users.` – Der Server ist jetzt nicht schnell, aber wenn er es wäre, könnte er mehr Benutzer unterstützen.
