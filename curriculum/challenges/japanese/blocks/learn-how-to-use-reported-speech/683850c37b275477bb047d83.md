# --explanation--

`log`は、コンピューターシステムやソフトウェアによって作成されるイベント、操作、またはメッセージの記録です。例えば：

`The system logs showed several errors before the crash.` – これは記録に問題が発生した情報が含まれていたことを意味します。

`crash`は、コンピューターシステムやプログラムが突然正常に動作しなくなることです。例えば：

`The application crash caused all unsaved data to be lost.` – これはプログラムが予期せず失敗し、データが保存されなかったことを意味します。
