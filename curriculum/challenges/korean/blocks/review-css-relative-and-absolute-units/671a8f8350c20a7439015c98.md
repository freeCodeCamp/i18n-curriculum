## 절대 단위

- **`px` (픽셀)**: 이 절대 단위는 CSS에서 고정 크기 단위입니다. 가장 일반적인 절대 단위이며 치수를 정확하게 제어할 수 있습니다. `1px`은 항상 1/96인치와 같습니다.
- **`in` (인치)**: 이 절대 단위는 96px과 같습니다.
- **`cm` (센티미터)**: 이 절대 단위는 25.2/64인치와 같습니다.
- **`mm` (밀리미터)**: 이 절대 단위는 1/10센티미터와 같습니다.
- **`q` (쿼터 밀리미터)**: 이 절대 단위는 1/40센티미터와 같습니다.
- **`pc` (파이카)**: 이 절대 단위는 1/6인치와 같습니다.
- **`pt` (포인트)**: 이 절대 단위는 1/72인치와 같습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="units">
  <div class="unit px">px</div>
  <div class="unit inch">in</div>
  <div class="unit cm">cm</div>
  <div class="unit mm">mm</div>
  <div class="unit q">q</div>
  <div class="unit pc">pc</div>
  <div class="unit pt">pt</div>
</div>
```

```css
.units {
  display: flex;
  gap: 12px;
  flex-wrap: wrap;
  align-items: flex-end;
}

.unit {
  background: steelblue;
  color: white;
  text-align: center;
  padding: 4px;
}

.px {
  width: 40px;
  height: 40px;
}

.inch {
  width: 0.5in;
  height: 0.5in;
}

.cm {
  width: 1cm;
  height: 1cm;
}

.mm {
  width: 10mm;
  height: 10mm;
}

.q {
  width: 40q;
  height: 40q;
}

.pc {
  width: 6pc;
  height: 6pc;
}

.pt {
  width: 36pt;
  height: 36pt;
}
```

## 상대 단위

- **백분율**: 이 상대 단위는 크기, 치수 및 기타 속성을 부모 요소의 비율로 정의할 수 있게 합니다. 예를 들어, 요소에 `width: 50%;`을 설정하면 부모 컨테이너 너비의 절반을 차지합니다.
- **`em` 단위**: 이 단위는 요소의 폰트 크기를 기준으로 합니다. `ems`을 `font-size` 속성에 사용하면 텍스트 크기가 부모 요소의 폰트 크기에 상대적입니다.
- **`rem` 단위**: 이 단위는 루트 요소인 `html` 요소의 폰트 크기를 기준으로 합니다.
- **`vh` 단위**: `vh`는 `"viewport height"`를 의미하며 `1vh`은 뷰포트 높이의 1%와 같습니다.
- **`vw` 단위**: `vw`는 `"viewport width"`를 의미하며 `1vw`은 뷰포트 너비의 1%와 같습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  <div class="box percent">50%</div>
  <div class="box em">2em</div>
  <div class="box rem">2rem</div>
  <div class="box vh">10vh</div>
  <div class="box vw">10vw</div>
</div>
```

```css
html {
  font-size: 16px;
}

.parent {
  width: 200px;
  font-size: 20px;
  border: 2px dashed #555;
  padding: 10px;
  display: flex;
  flex-direction: column;
  gap: 12px;
}

.box {
  background: seagreen;
  color: white;
  text-align: center;
  padding: 6px;
}

.percent {
  width: 50%;
}

.em {
  font-size: 2em;
}

.rem {
  font-size: 2rem;
}

.vh {
  height: 10vh;
}

.vw {
  width: 10vw;
}
```

## `calc` 함수

- **`calc()` 함수**: `calc()` 함수를 사용하면 스타일시트 내에서 직접 계산을 수행해 속성 값을 동적으로 결정할 수 있습니다. 이를 통해 뷰포트 크기나 다른 요소를 기준으로 치수를 계산하여 유연하고 반응형 사용자 인터페이스를 만들 수 있습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="calc-box">calc()</div>
```

```css
.calc-box {
  width: calc(100% - 40px);
  padding: 10px;
  background: steelblue;
  color: white;
  text-align: center;
  border: 2px solid black;
}
```
