# --description--

Taarifa za if ngumu zaidi zinaweza kujumuisha masharti mengi. Ndiyo sababu opereta za kimantiki hutumika kuunganisha masharti mengi. Opereta za kimantiki ni `&&`, `||` na `!` ambazo hutumika kuwakilisha `and`, `or` na `not` mtawalia.

Opereta wa kimantiki `||` hutumika kuunganisha masharti mawili ya thamani ya kweli au siyo kweli. Hurejesha `true` ikiwa angalau mojawapo ya masharti ni `true`. Vinginevyo, hurejesha `false`:

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a > b || a > c) {
  console.log("At least one of the conditions is true");
} else {
  console.log("Both of the conditions are false");
}
```

Opereta wa kimantiki `&&` hutumika kuunganisha masharti mawili ya thamani ya kweli au siyo kweli. Hurejesha `true` tu ikiwa masharti yote mawili ni `true`. Vinginevyo, hurejesha `false`:

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a < b && a < c) {
  console.log("Both of the conditions are true");
} else {
  console.log("At least one of the conditions is false");
}
```

Opereta wa kimantiki `!` hutumika kupinga hali ya thamani ya kweli au siyo kweli. Hurejesha `true` ikiwa hali ni `false`. Vinginevyo, hurejesha `false`:

```javascript
let a = 5;
let b = 10;

if (!(a > b)) {
  console.log("The condition is false");
} else {
  console.log("The condition is true");
}
```

Katika mfano hapo juu, jozi ya ziada ya mabano ya kawaida imetumika kufanya msimbo uwe rahisi kusomeka. Sio lazima kuyatumia.

# --questions--

## --text--

Umepewa jukumu la kuandika taarifa ya `if` inayokagua masharti yafuatayo katika programu ya mtandao ili kuonyesha `Welcome!` kwa mtumizi:

1. Mtumizi lazima awe na akaunti ya premium (`isPremium`) au awe mwanachama kwa zaidi ya mwaka mmoja (`membershipDuration` > miezi 12).

1. Mtumizi lazima asizuiliwe sasa hivi kutoka kwa huduma (`!isBlocked`).

Ni taarifa gani ya if inayokagua masharti haya kwa usahihi?

## --answers--

```javascript
if (isPremium && membershipDuration > 12 && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (isPremium || (membershipDuration > 12 && !isBlocked)) {
  console.log("Welcome!");
}
```

---

```javascript
if ((isPremium || membershipDuration > 12) && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (!isPremium || membershipDuration <= 12 || isBlocked) {
  console.log("Welcome!");
}
```
