# --interactive--

CSS Grid ni mfumo wenye nguvu wa upangaji unaowawezesha wasanidi programu wa mtandao kuunda mipangilio tata na inayojibadilisha kulingana na kifaa kwa urahisi.

Fikiria unapopanga samani katika chumba – CSS Grid ni kama kuwa na gridi isiyoonekana sakafuni mwako inayokusaidia kuweka kila kitu mahali pasipo makosa.

Tunapojenga tovuti, mara nyingi tunahitaji kupanga vipengele tofauti kwenye ukurasa.

Kabla ya CSS Grid, hili lilikuwa changamoto mara nyingine, hasa kwa mipangilio tata. CSS Grid inarahisisha mchakato huu kwa kugawanya ukurasa wako wa mtandao katika safu na nguzo, ikitengeneza muundo wa gridi.

Fikiria unafanya kazi na kontena `div` yenye vipengele vingi vilivyopangwa ndani kama hivi:

:::interactive_editor

```html
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

:::

Ikiwa ungependa kupamba vipengele hivyo kwa muundo wa gridi, unaweza kuweka `display` kuwa `grid` na kutumia nguzo kama hivi:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-gap: 20px;
}

.item {
  background-color: lightgray;
  padding: 20px;
  text-align: center;
  border: 1px solid #ccc;
}
```

:::

Katika msimbo huu, tunamwambia kivinjari kuunda gridi yenye nguzo tatu zenye upana sawa, hiyo ndiyo maana ya `1fr 1fr 1fr`, na tunaongeza pengo la pikseli 20 kati ya kila kipande cha gridi.

Sasa, unaweza kuwa unajiuliza: "Je, Flexbox je? Je, siyo pia hutumika kwa mipangilio?"

Uko sahihi! Flexbox ni mfano mwingine wa upangaji wa CSS, na pia ni mzuri sana. Lakini kuna tofauti muhimu.

Flexbox ni wa mwelekeo mmoja, wakati Grid ni wa mwelekeo miwili. Hii ina maana Flexbox inafaa kwa kupanga vitu katika safu moja au nguzo moja, wakati Grid ni bora kwa kuunda mipangilio yenye safu na nguzo zote mbili.

Flexbox huanza na maudhui, ikibadilisha muundo kulingana na maudhui. Grid, kwa upande mwingine, huanza na muundo, ikikuruhusu kuunda muundo kisha kuweka vitu ndani yake. Grid inakupa udhibiti sahihi zaidi wa mahali pa vitu. Unaweza kusema kipande hasa ni safu gani na nguzo gani kiwe.

Hapa kuna mfano wa Flexbox kwa kulinganisha:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
}
```

:::

Hii huunda chombo cha kubadilika ambapo vipengele vimepangwa kwa usawa kando ya mhimili mkuu.

Zote Grid na Flexbox zina nguvu zao, na mara nyingi, mipangilio bora hutumia mchanganyiko wa zote mbili. Unaweza kutumia Grid kwa muundo wa jumla wa ukurasa, kisha kutumia Flexbox kwa kupanga vitu ndani ya kila eneo la gridi.

Kwa muhtasari, CSS Grid ni chombo chenye nguvu kinachoruhusu mipangilio sahihi ya mwelekeo miwili. Ingawa inaweza kuonekana ngumu mwanzoni, kwa mazoezi, inakuwa chombo muhimu kwa kuunda mipangilio tata na inayojibadilisha kulingana na kifaa.
