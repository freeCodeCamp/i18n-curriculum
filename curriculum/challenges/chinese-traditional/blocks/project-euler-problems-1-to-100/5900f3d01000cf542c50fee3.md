# --description--

如果一個盒子裡有二十一個彩色圓盤，由十五個藍色圓盤和六個紅色圓盤組成，且隨機取出兩個圓盤，可以看出取出兩個藍色圓盤的機率。

$${P(BB)} = \frac{15}{21}×\frac{14}{20} = \frac{1}{2}$$

下一個此類排列，隨機抽取兩個藍色圓盤正好有 50％ 機率的，是一個包含八十五個藍色圓盤和三十五個紅色圓盤的盒子。

透過尋找第一個總共包含超過 `limit` 個圓盤的排列，來判定盒子中會包含多少個藍色圓盤。

```js
function arrangedProbability(limit) {
  // Based on https://www.mathblog.dk/project-euler-100-blue-discs-two-blue/
  let blue = 15;
  let discs = 21;

  while (discs < limit) {
    const nextBlue = 3 * blue + 2 * discs - 2;
    const nextDiscs = 4 * blue + 3 * discs - 3;

    blue = nextBlue;
    discs = nextDiscs;
  }
  return blue;
}
```
