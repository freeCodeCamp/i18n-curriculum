# --explanation--

`Really` 和 `too` 都是**强调词**，但它们的含义不同。`Really` 加强它所修饰的形容词，而不表示过度。另一方面，`too` 表示过度，意味着超过了可接受或可处理的范围。例如：

- `The project is really important.` - 这意味着该项目非常重要，但不一定过于重要。
 
- `This task is too complicated.`——这意味着它比应该的更复杂。
