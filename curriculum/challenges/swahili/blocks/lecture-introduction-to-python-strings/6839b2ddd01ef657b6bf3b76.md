# --description--

Njia ni kitendakazi kinachomilikiwa na kitu maalum au darasa la vitu. Utajifunza zaidi kuhusu madarasa ya vitu, vitu, vitendakazi, na njia baadaye. Kwa sasa, fahamu kwamba njia lazima zitumike kwa kitu zinachomilikiwa kwa kuandika kitu kifuatiwe na nukta na wito wa njia hiyo. Python hutoa njia kadhaa zilizojengwa ndani ambazo unaweza kutumia kudhibiti mfuatano wa herufi. Zinajumuisha, lakini siyo tu, zifuatazo:

- `upper()`: Hurejesha mfuatano mpya wa herufi ambapo herufi zote zimebadilishwa kuwa herufi kubwa.
    

```python
my_str = 'hello world'

uppercase_my_str = my_str.upper()
print(uppercase_my_str)  # HELLO WORLD
```

- `lower()`: Hurejesha mfuatano mpya wa herufi ambapo herufi zote zimebadilishwa kuwa herufi ndogo.
    

```python
my_str = 'Hello World'

lowercase_my_str = my_str.lower()
print(lowercase_my_str)  # hello world
```

- `strip()`: Hurejesha mfuatano mpya wa herufi ambapo herufi za mwanzo na mwisho zilizobainishwa zimeondolewa. Ikiwa hakuna hoja iliyotolewa, huondoa nafasi za mwanzo na mwisho.
    

```python
my_str = '  hello world  '

trimmed_my_str = my_str.strip()
print(trimmed_my_str)  # "hello world"
```

- `replace(old, new)`: Hurejesha mfuatano mpya wa herufi ambapo matukio yote ya `old` yamebadilishwa na `new`.
    

```python
my_str = 'hello world'

replaced_my_str = my_str.replace('hello', 'hi')
print(replaced_my_str)  # hi world
```

- `split(separator)`: Hugawanya mfuatano wa herufi kwa separator iliyobainishwa na kuunda orodha ya mfuatano wa herufi. Ikiwa hakuna separator iliyobainishwa, hugawanya kwa nafasi.
    

```python
my_str = 'hello world'

split_words = my_str.split()
print(split_words)  # ['hello', 'world']
```

- `join(iterable)`: Huunganisha vipengele vya iterable kuwa mfuatano wa herufi kwa separator.
    

```python
my_list = ['hello', 'world']

joined_my_str = ' '.join(my_list)
print(joined_my_str)  # hello world
```

- `startswith(prefix)`: Hurejesha thamani ya kweli au siyo kweli ikionyesha kama mfuatano wa herufi unaanza na kiambishi awali kilichobainishwa.
    

```python
my_str = 'hello world'

starts_with_hello = my_str.startswith('hello')
print(starts_with_hello)  # True
```

- `endswith(suffix)`: Hurejesha thamani ya kweli au siyo kweli ikionyesha kama mfuatano wa herufi unaisha na kiambishi cha mwisho kilichobainishwa.
    

```python
my_str = 'hello world'

ends_with_world = my_str.endswith('world')
print(ends_with_world)  # True
```

- `find(substring)`: Hurejesha kielezo cha tukio la kwanza la `substring`, au `-1` ikiwa halipatikani.
    

```python
my_str = 'hello world'

world_index = my_str.find('world')
print(world_index)  # 6
```

- `count(substring)`: Hurejesha idadi ya mara mfuatano mdogo unaonekana ndani ya mfuatano.
    

```python
my_str = 'hello world'

o_count = my_str.count('o')
print(o_count)  # 2
```

- `capitalize()`: Hurejesha mfuatano mpya wa herufi ambapo herufi ya kwanza imewekwa kuwa kubwa na herufi zingine ziko ndogo.
    

```python
my_str = 'hello world'

capitalized_my_str = my_str.capitalize()
print(capitalized_my_str)  # Hello world
```

- `isupper()`: Hurejesha `True` ikiwa herufi zote katika mfuatano ni kubwa na `False` ikiwa siyo.
    

```python
my_str = 'hello world'

is_all_upper = my_str.isupper()
print(is_all_upper)  # False
```

- `islower()`: Hurejesha `True` ikiwa herufi zote katika mfuatano ni ndogo na `False` ikiwa siyo.
    

```python
my_str = 'hello world'

is_all_lower = my_str.islower()
print(is_all_lower)  # True
```

- `title()`: Hurejesha mfuatano mpya wa herufi ambapo herufi ya kwanza ya kila neno imewekwa kuwa kubwa.
    

```python
my_str = 'hello world'

title_case_my_str = my_str.title()
print(title_case_my_str)  # Hello World
```
