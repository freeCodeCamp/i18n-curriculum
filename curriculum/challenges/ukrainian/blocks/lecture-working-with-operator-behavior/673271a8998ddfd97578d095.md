# --interactive--

Якщо ви працюєте з числами і потрібно збільшити або зменшити значення на одиницю, оператори інкременту та декременту значно полегшують це завдання. Давайте розберемося простою мовою.

Оператори інкременту та декременту представлені `++` та `--` відповідно. Вони обидва дозволяють вам змінити значення змінної на `1`.

Замість того, щоб писати щось на кшталт `x = x + 1` або `x = x - 1`, ви можете просто використати `x++`, щоб додати `1`, або `x--`, щоб відняти `1`. Це швидше, чистіше і легше для читання.

Є одна особливість у роботі операторів інкременту та декременту: вони бувають двох форм — префіксна та постфіксна, і різниця полягає в тому, коли саме оновлюється значення. Для оператора інкременту префікс — це `++x`, а постфікс — `x++`.

Префікс (`++x`) спочатку збільшує значення змінної, а потім повертає нове значення. Постфікс (`x++`) спочатку повертає поточне значення змінної, а потім збільшує його:

:::interactive_editor

```js
let x = 5;

console.log(++x); // 6
console.log(x); // 6
```

:::

У наведеному коді `++x` означає «спочатку збільшити `x`, а потім використати його». Тож коли ви виводите `++x`, ви одразу отримуєте збільшене значення, яке дорівнює `6`.

Тепер подивімося на приклад із постфіксом:

:::interactive_editor

```js
let y = 5;

console.log(y++); // 5
console.log(y); // 6
```

:::

У цьому прикладі `y++` означає «спочатку використати `y`, а потім збільшити його». Коли ви виводите `y++`, ви отримуєте `5`, але `y` після цього рядка коду стає `6`.

Оператор декременту працює так само, як і інкремент, тільки зменшує значення на `1`. Знову ж таки, є дві форми: префікс (`--x`) спочатку зменшує значення змінної, а потім повертає нове значення. А постфікс (`x--`) спочатку повертає поточне значення, а потім зменшує його:

:::interactive_editor

```js
let x = 5;
console.log(--x); // 4
console.log(x); // 4

let y = 5;
console.log(y--); // 5
console.log(y); // 4
```

:::

Отже, яку форму слід використовувати: префікс чи постфікс? У багатьох випадках не має значення, яку саме ви використаєте. Обидві виконують свою роботу. Однак якщо ви одразу використовуєте значення у виразі, різниця стає важливою. Подивімося на цей приклад:

:::interactive_editor

```js
let a = 5;
let b = ++a;
console.log(b); // 6 (a was incremented before assignment)

let c = 5;
let d = c++;
console.log(d); // 5 (c was incremented after assignment)
```

:::

Отже, якщо вам потрібне оновлене значення негайно, використовуйте префікс. Якщо ж спочатку потрібне поточне значення і вам байдуже до інкременту до пізніше, обирайте постфікс.
