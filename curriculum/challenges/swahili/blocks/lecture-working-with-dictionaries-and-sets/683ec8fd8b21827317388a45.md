# --description--

Sets ni mojawapo ya miundo ya data iliyojengwa ndani ya Python. Moja ya sifa kuu za sets ni kwamba hazihifadhi thamani zinazojirudia. Ukijaribu kuongeza thamani inayojirudia kwenye set, moja tu kati ya hizo itahifadhiwa.

Sets ni zinazobadilika na hazina mpangilio maalum, ambayo ina maana vipengele vyake havihifadhiwi kwa mpangilio wowote maalum, hivyo huwezi kutumia vielezo au funguo kuzipata. Sets zinaweza kuwa na thamani za aina za data zisizobadilika kama nambari, mfuatano wa herufi, na tuple. Pia zinaunga mkono operesheni za kihesabu za set, ikiwa ni pamoja na muungano, mkusanyiko, tofauti, na tofauti ya mlinganyo.

Ili kufafanua set, unahitaji tu kuandika vipengele vyake ndani ya mabano ya mviringo na kuvitenganisha kwa koma. Hii ni mfano wa set ya nambari:

```python
my_set = {1, 2, 3, 4, 5} 
```

Tabia moja ya ajabu ya kufanya kazi na sets ni kwamba, ukihitaji kufafanua set tupu, lazima utumie kitendakazi cha `set()`. Ukichora mabano ya mviringo tupu tu, kama `{}`, Python itaunda moja kwa moja kamusi.

```python
set() # Set
{}    # Dictionary
```

Unaweza kuongeza kipengele kwenye set kwa kutumia njia ya `.add()`, na kupitisha kipengele kipya kama hoja:

```python
my_set.add(6)
```

Katika mfano wetu, set mpya itakuwa:

```python
{1, 2, 3, 4, 5, 6}
```

Ukijaribu kuongeza kipengele kilichopo tayari kwenye set, moja tu itahifadhiwa. Katika kesi hii, tayari tuna nambari 5 kwenye set:

```python
my_set.add(5)
```

Hivyo set haitabadilika:

```python
{1, 2, 3, 4, 5, 6}
```

Ili kuondoa kipengele kutoka kwenye set, una chaguzi mbili. Unaweza kutumia njia ya `.remove()` au njia ya `.discard()`, na kupitisha kipengele unachotaka kuondoa kama hoja.

Njia ya `.remove()` itatoa `KeyError` ikiwa kipengele hakipatikani, wakati njia ya `.discard()` haitatoa:

```python
my_set.remove(4)
my_set.discard(4)
```

Njia ya `.clear()` huondoa vipengele vyote kutoka kwenye set:

```python
my_set.clear()
```

Sets za Python pia zina njia zenye nguvu zinazofanya operesheni za kawaida za kihesabu za set.

Njia za `.issubset()` na `.issuperset()` zinakagua kama set ni seti ndogo au seti kubwa ya seti nyingine, mtawalia.

Hapa, tunakagua kama `your_set` ni seti ndogo ya `my_set`, ambayo ni `False` kwa sababu si vipengele vyote vya `your_set` vipo katika `my_set`.

Pia tunakagua kama `my_set` ni seti kubwa ya `your_set`. Hii pia ni `False` kwa sababu `my_set` haina vipengele vyote vya `your_set`:

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

print(your_set.issubset(my_set)) # False
print(my_set.issuperset(your_set)) # False
```

Njia ya `.isdisjoint()` inakagua kama sets mbili hazina kipengele chochote sawa, ambayo ina maana hazina vipengele vinavyofanana. Katika kesi hii, ni `False` kwa sababu `my_set` na `your_set` zina vipengele vinavyofanana – 2, 3, na 4:

```python
print(my_set.isdisjoint(your_set)) # False
```

Opereta ya muungano `|` inarudisha seti mpya yenye vipengele vyote kutoka kwenye sets zote mbili:

```python
my_set | your_set # {1, 2, 3, 4, 5, 6}
```

Opereta ya mkusanyiko `&` inarudisha seti mpya yenye vipengele tu ambazo sets zote zina kwa pamoja:

```python
my_set & your_set # {2, 3, 4}
```

Opereta ya tofauti `-` inarudisha seti mpya yenye vipengele za seti ya kwanza ambazo hazipo katika seti nyingine. Katika mfano huu, nambari 1 na 5 zipo katika `my_set` lakini HAZIPO katika `your_set`:

```python
my_set - your_set # {1, 5}
```

Opereta ya tofauti ya mlinganyo `^` inarudisha seti mpya yenye vipengele vilivyoko katika seti ya kwanza au ya pili, lakini si zote mbili. Katika kesi hii, 1 na 5 zipo katika `my_set` lakini si katika `your_set`, hivyo zinajumuishwa. Na nambari 6 ipo katika `your_set` lakini si katika `my_set`, hivyo pia inajumuishwa:

```python
my_set ^ your_set # {1, 5, 6}
```

Kila moja ya opereta hizi pia ina opereta yake ya mgawo mchanganyiko ikiwa utaongeza ishara ya usawa kando yake. Opereta hizi huweka moja kwa moja seti inayotokana na matokeo kwenye seti ya kwanza katika usemi wa kihesabu:

```python
|= &= -= ^=
```

Kwa mfano, opereta ya `-=` hupata tofauti kati ya sets na kusahihisha seti ya kwanza kwa matokeo hayo:

```python
my_set -= your_set
```

Baada ya hili, `my_set` itasasishwa kuwa `{1, 5}`:

```python
print(my_set) # {1, 5}
```

Unaweza kukagua kama kipengele kiko katika seti au hakiko kwa kutumia opereta ya `in`. Hapa, tunakagua kama 5 iko katika `my_set`. Matokeo yatakuwa thamani ya kweli au siyo kweli `True` au `False`:

```python
print(5 in my_set)
```

Na hayo ni msingi wa sets. Ni msaada sana wakati hutaki kuhifadhi thamani kwa mpangilio maalum, na wakati unahitaji kuhifadhi thamani za kipekee tu.
