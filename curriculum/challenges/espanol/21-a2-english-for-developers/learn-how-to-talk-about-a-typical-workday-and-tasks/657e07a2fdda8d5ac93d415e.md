# --description--

En inglés, puedes usar palabras como `then` para hablar sobre el orden en el que ocurren las cosas.

`Then` se utiliza a menudo para indicar el próximo paso o acción después de que se haya hecho algo más.

Por ejemplo, `First, I check my emails, then I start coding.`

Esto ayuda a las personas a entender cómo una serie de acciones o eventos están conectados en el tiempo.
