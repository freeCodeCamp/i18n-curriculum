# --description--

O verbo modal `can` é usado para indicar possibilidade ou habilidade.Por exemplo, na frase `You can see the stars at night`, `can` mostra que é possível ver as estrelas durante a noite.

`Change` significa fazer ou tornar-se diferente.No dia a dia, você pode `change` suas roupas depois do trabalho, indicando que você troca de roupas de trabalho para roupas casuais.

`Stay the same` significa permanecer inalterado ou constante.Por exemplo, `The rules of the game stay the same every year.` Isso significa que as regras não mudam de um ano para o outro.
