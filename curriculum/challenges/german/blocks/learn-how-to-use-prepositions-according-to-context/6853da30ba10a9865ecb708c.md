`five more`

### --feedback--

Diese zwei Wörter geben die zusätzliche Anzahl der benötigten Personen an.

# --explanation--

`By next month` bedeutet vor Beginn des kommenden Monats. Es legt eine klare Frist fest. Zum Beispiel:  

`The new version should be ready by next month.` – Das bedeutet, es sollte bis dahin fertig sein.

`Five more` bedeutet fünf zusätzliche Entwickler im Kontext. Es wird verwendet, um eine Mengensteigerung anzuzeigen. Ein weiteres Beispiel:  

`We need five more testers before we launch.` – Das bedeutet, es werden fünf zusätzliche Tester benötigt.
