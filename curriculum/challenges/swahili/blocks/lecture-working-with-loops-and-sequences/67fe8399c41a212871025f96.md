# --description--

Katika mafundisho yajayo kadhaa tutajifunza kuhusu orodha, tuple, na ranges, ambazo ni aina tatu za msingi za mfululizo zinazotumika katika Python.

Aina ya data ya orodha ni mfululizo uliopangwa wa vipengele ambavyo vinaweza kuwa mfuatano wa herufi, nambari, au hata orodha nyingine. Orodha ni zinazobadilika na hutumia kielezo kuanzia sufuri, ikimaanisha kuwa kipengele cha kwanza cha orodha kiko kwenye kielezo sufuri.

Hapa kuna sintaksia ya msingi ya orodha:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
```

Ili kupata kipengele kutoka kwenye orodha ya `cities`, unaweza kurejea nambari yake ya kielezo katika mfululizo. Hapa kuna mfano wa kupata kipengele cha kwanza cha orodha ya `cities`:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # 'Los Angeles'
```

Kielezo hasi hutumika kupata vipengele kuanzia mwisho wa orodha badala ya mwanzo kwenye kielezo `0`. Ili kupata kipengele cha mwisho cha orodha yoyote, unaweza kutumia `-1` kama ifuatavyo:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # 'Tokyo'
```

Njia nyingine ya kuunda orodha ni kutumia kijenzi cha `list()`. Kijenzi cha `list()` hutumika kubadilisha kitu kinachoweza kurudiwa kuwa orodha kama ifuatavyo:

```py
developer = 'Jessica'
list(developer) # ['J', 'e', 's', 's', 'i', 'c', 'a']
```

Kitu kinachoweza kurudiwa ni aina maalum ya kitu ambacho kinaweza kurudiwa kipengele kimoja kwa wakati mmoja. Utajifunza zaidi kuhusu mizunguko katika Python baadaye.

Ili kupata jumla ya vipengele katika orodha, unaweza kutumia kitendakazi cha `len()` kama ifuatavyo:

```py
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

Ikiwa ungependa kusahihisha thamani katika kielezo fulani, unaweza kufanya kitu kama hiki:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

Kwa kuwa orodha ni zinazobadilika, unaweza kusahihisha kipengele chochote ndani ya orodha mradi tu ulete nambari halali ya kielezo. Ikiwa ulete kielezo (chanya au hasi) ambacho kiko nje ya mipaka ya orodha, basi utapokea `IndexError`:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list assignment index out of range
"""
```

Ikiwa unataka kuondoa kipengele kutoka kwenye orodha unaweza kutumia neno kuu la `del` kama ifuatavyo:

```py
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

Wakati mwingine ni msaada kuangalia kama kipengele kiko ndani ya orodha. Ili kufanya hivyo, unaweza kutumia neno kuu la `in` kama ifuatavyo:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

Wakati mwingine ni kawaida kuwa na orodha zilizopangwa ndani ya orodha nyingine kama ifuatavyo:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

Katika mfano huu, tuna orodha moja iliyopangwa ndani yenye lugha tatu maarufu za programu. Ili kupata orodha iliyopangwa ndani, utahitaji kuipata kwa kutumia kielezo `2` kwa kuwa orodha hutumia kielezo kuanzia sufuri:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

Kisha ili kupata lugha ya pili kutoka orodha hiyo iliyopangwa ndani, utahitaji kuipata kwa kutumia kielezo `1` kama ifuatavyo:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # 'Rust'
```

Mbinu nyingine ya kawaida inayotumika na orodha ni kufungua thamani.

Kufungua thamani kutoka kwenye orodha ni mbinu inayotumika kuweka thamani kutoka kwenye orodha kwa vigezo vipya. Hapa kuna mfano wa kufungua orodha ya `developer` kwa vigezo vipya vinavyoitwa `name`, `age` na `job`.

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

Hapa, `name` ana thamani ya `'Alice'`, `age` ana thamani ya `34`, na `job` ana thamani ya `'Rust Developer'`.

Ikiwa unahitaji kukusanya vipengele vyote vilivyobaki kutoka kwenye orodha, unaweza kutumia opereta ya nyota (`*`) kama ifuatavyo:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

Katika mfano huu, `name` bado atakuwa na thamani ya `'Alice'`, na `rest` ni orodha ya vitu viwili: nambari `34` na mfuatano wa herufi `'Rust Developer'`.

Ikiwa idadi ya vigezo upande wa kushoto wa opereta ya mgawo haitalingana na jumla ya vitu katika orodha, basi utapokea `ValueError`:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job, city = developer

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: not enough values to unpack (expected 4, got 3)
"""
```

Dhana ya mwisho tutakayozungumzia ni opereta ya slice (`:`). Kama ilivyo kwa mfuatano wa herufi, unaweza kupata sehemu za orodha kwa kutumia opereta ya slice kama ifuatavyo:

```py
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie', 'Brownies']
desserts[1:4] # ['Cookies', 'Ice Cream', 'Pie']
```

Katika mfano huu, kielezo cha kuanzia ni `1` kwa kuwa hicho kinahusu kipengele cha pili katika orodha. Kisha tunatumia opereta ya slice ikifuatiwa na kielezo cha mwisho cha `4`, ambacho kinajumuisha kila kitu hadi (lakini si pamoja na) kipengele kilichopo kwenye kielezo hicho.

Jambo lingine unaloweza kufanya na opereta ya slice `:` ni kubainisha muda wa hatua ambayo huamua ni kwa kiasi gani kielezo kinapandishwa. Tuseme una orodha ya nambari kama ifuatavyo:

```py
numbers = [1, 2, 3, 4, 5, 6]
```

Ikiwa ungependa kutoa orodha ya nambari chanya tu, unaweza kutumia opereta ya slice kama ifuatavyo:

```py
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

Nambari ya kwanza chanya iko kwenye kielezo `1`, hivyo hicho kitakuwa kielezo cha kuanzia. Kwa kuwa tunataka kupitia hadi mwisho wa orodha, basi tunapuuza kielezo cha mwisho. Mwisho, tunabainisha `2` kwa muda wa hatua ya hiari ili ipandishe kwa `2` badala ya `1` ya kawaida.

Orodha ni muundo wa data wenye manufaa na unaobadilika ambao utatumia sana katika programu zako za Python. Katika funzo lijalo, utajifunza kuhusu njia za kawaida unazoweza kutumia na orodha.
