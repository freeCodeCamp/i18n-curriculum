# --description--

Das Open-Graph-Protokoll ermöglicht es Ihnen, zu steuern, wie die Inhalte Ihrer Webseite auf verschiedenen sozialen Medienplattformen wie Facebook, LinkedIn und vielen mehr dargestellt werden. Durch das Festlegen dieser Open-Graph-Eigenschaften können Sie Nutzer dazu verleiten, auf Ihre Inhalte zu klicken und sich damit zu beschäftigen. Sie können diese Eigenschaften über eine Sammlung von `meta`-Elementen im HTML-`head`-Abschnitt festlegen.

Die erste wichtige OG-Eigenschaft, die Sie einfügen sollten, ist das `title`. Hier ein Beispiel für das Festlegen des OG-`title` für die freeCodeCamp-Startseite:

```html
<meta content="freeCodeCamp.org" property="og:title" />
```

Für das `property`-Attribut müssen Sie angeben, dass es `og:title` ist. Das `content`-Attribut ist der Ort, an dem Sie den Titel schreiben, der auf den sozialen Medienseiten angezeigt werden soll.

Die nächste wichtige OG-Eigenschaft ist das `type`. Hier ein Beispiel für die Verwendung des OG-`type` für die freeCodeCamp-Startseite:

```html
<meta property="og:type" content="website" />
```

Die `type`-Eigenschaft wird verwendet, um den Typ des Inhalts darzustellen, der in sozialen Medien geteilt wird. Beispiele für diesen Inhalt sind Artikel, Webseiten, Videos oder Musik.

Die dritte wichtige OG-Eigenschaft ist das `image`. Hier ein Beispiel für das Festlegen des OG-`image` für die freeCodeCamp-Startseite:

```html
<meta
  content="https://cdn.freecodecamp.org/platform/universal/fcc_meta_1920X1080-indigo.png"
  property="og:image"
/>
```

In diesem Beispiel verweist das Open-Graph-Bild auf das freeCodeCamp-Logo. Alle diese Bilder sollten von hoher Qualität mit guten Abmessungen und Verhältnissen sein. Die meisten sozialen Medienplattformen geben Kriterien für Bildanforderungen vor, um Ihnen zu helfen, sicherzustellen, dass Ihre Inhalte auf deren Seite gut dargestellt werden. Zum Beispiel heißt es auf der developers.facebook.com-Dokumentationsseite:

„Verwenden Sie Bilder, die mindestens 1200 mal 630 Pixel groß sind, für die beste Darstellung auf Geräten mit hoher Auflösung. Mindestens sollten Sie Bilder verwenden, die 600 mal 315 Pixel groß sind, um Linkseitenbeiträge mit größeren Bildern anzuzeigen.“

Die vierte wichtige OG-Eigenschaft ist das `url`. Hier ein Beispiel für das Festlegen des OG-`url` für die freeCodeCamp-Startseite:

```html
<meta property="og:url" content="https://www.freecodecamp.org" />
```

Es gibt viele weitere OG-Eigenschaften, die Sie festlegen können, wie `description`, `audio`, `video` und `locale`. Allerdings sind die Open-Graph-`url`, `image`, `type` und `title` die wichtigsten, die Sie einfügen sollten.

Wie beeinflussen diese Open-Graph-Eigenschaften also die Suchmaschinenoptimierung? Wenn Ihre Inhalte in sozialen Medien geteilt werden, können gut gestaltete OG-Eigenschaften das Erscheinungsbild Ihrer Inhalte in den Feeds der Nutzer verbessern. Dies kann zu höheren Klickraten führen, was Suchmaschinen signalisieren könnte, dass Ihre Inhalte relevant und ansprechend sind.
