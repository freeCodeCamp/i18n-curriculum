# --description--

Тепер трохи змінимо ситуацію:

```html
<!-- index.html -->

<div class="main">
  <div class="list" id="subsection"></div>
</div>
```

```css
/* rule 1 */
#subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

У наведеному вище прикладі, незважаючи на те, що правило 2 має більше селекторів класу, ніж селекторів ідентифікатора, правило 1 є більш специфічним, бо ідентифікатор має вищий пріоритет за клас.
