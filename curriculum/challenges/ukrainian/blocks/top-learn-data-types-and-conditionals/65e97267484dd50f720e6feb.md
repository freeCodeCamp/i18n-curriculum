# --description--

Ви знаєте, що в JavaScript існує різноманіття типів даних, включно з рядками, числами та булевими значеннями. Ви можете використовувати оператори порівняння, щоб порівнювати ці типи даних і приймати рішення на основі результатів.

У JavaScript доступні такі оператори порівняння:

| Оператор | Опис                                   | Приклад     |
|----------|----------------------------------------|-------------|
| ==       | Рівність (порівняння значень)          | x == y      |
| !=       | Не рівність                            | x != y      |
| >        | Більше                                | x > y       |
| <        | Менше                                 | x < y       |
| >=       | Більше або дорівнює                    | x >= y      |
| <=       | Менше або дорівнює                     | x <= y      |

Щоб використати порівняння у присвоєнні змінній, ви можете застосувати оператор `==`. Наприклад:

```javascript
let x = 5;
let y = 5;

let result = x == y;

console.log(result); // true
```
