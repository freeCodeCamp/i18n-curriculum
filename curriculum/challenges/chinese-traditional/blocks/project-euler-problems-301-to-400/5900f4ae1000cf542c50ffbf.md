# --description--

令 $N(i)$ 為最小的整數 $n$，使得 $n!$ 可被 $(i!)^{1234567890}$ 整除。

令 $S(u) = \sum N(i)$，其中 $10 ≤ i ≤ u$。

$S(1000)=614\\,538\\,266\\,565\\,663$。

尋找 $S(1\,000\,000)\bmod {10}^{18}$。
