# --description--

React는 사용자 인터페이스와 웹 애플리케이션을 구축하는 데 가장 인기 있는 JavaScript 라이브러리 중 하나입니다. 원래 Facebook에서 개발 및 유지 관리했으며, 효율성, 유연성, 그리고 다양한 기능 덕분에 웹 개발에서 큰 인기를 얻고 있습니다.

React의 기본 개념 중 하나는 재사용 가능한 UI 컴포넌트를 만드는 것입니다. 버튼, 카드, 아바타 컴포넌트 같은 이러한 컴포넌트는 애플리케이션 전반에 걸쳐 쉽게 재사용할 수 있습니다. 더 복잡하고 동적인 인터페이스를 만들기 위해 이러한 컴포넌트를 서로 중첩할 수 있습니다.

React의 주요 장점 중 하나는 이러한 커스텀 컴포넌트가 데이터가 변경될 때 독립적으로 업데이트되고 렌더링될 수 있다는 점입니다.

전통적인 JavaScript와 달리 DOM(문서 객체 모델)을 직접 조작할 필요 없이, React는 가상 DOM을 사용하여 성능과 효율성을 향상시킵니다. 가상 DOM과 그 작동 방식에 대해서는 앞으로의 수업에서 더 자세히 배우게 됩니다.

재사용 가능한 컴포넌트 외에도 React는 애플리케이션 상태를 관리하는 강력한 방법을 제공합니다. 상태는 컴포넌트가 어떻게 렌더링되고 동작하는지를 결정하는 데이터를 의미합니다.

React를 사용하면 컴포넌트의 상태를 쉽게 추적하고 업데이트하여 UI가 가장 최신 데이터를 반영하도록 할 수 있습니다. 상태 작업에 대해서는 앞으로의 수업에서 더 배우게 됩니다.

JavaScript 생태계에는 많은 라이브러리가 있지만, freeCodeCamp는 산업계에서 널리 사용되고 수요가 많은 React 교육에 주로 포커스를 맞출 것입니다.

다음 수업들에서는 커스텀 컴포넌트 구축, 상태 관리 등에 대해 더 깊이 다룰 예정입니다.
