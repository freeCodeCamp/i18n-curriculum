# --explanation--

Apuntar (`aim`) significa planear para alcanzar una meta u objetivo. Por ejemplo:

`We aim to release the app next month.` — Esperan o planean hacerlo.

`Beta testing` es cuando usuarios reales prueban un producto casi terminado para encontrar cualquier error. Por ejemplo:

`The app is now in beta testing.` — La gente está probando la app para encontrar problemas.

Los `Critical issues` son problemas muy graves que deben ser corregidos antes del lanzamiento. Por ejemplo:

`The app crashing on login is a critical issue.` — Es un problema serio que bloquea el uso.
