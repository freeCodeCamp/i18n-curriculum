# --description--

طريقة أخرى لتمثيل رسم بياني هي وضعه في <dfn>المصفوفة المجاورة</dfn>. <dfn>المصفوفة المجاورة</dfn> هي مصفوفة ثنائية الأبعاد حيث تحتوي كل مصفوفة مسكنة على نفس عدد العناصر مثل المصفوفة الخارجية. بعبارة أخرى، هي مصفوفة أو شبكة من الأرقام، حيث تمثل الأرقام الحواف.

**ملاحظة**: الأرقام في الأعلى وعلى يسار المصفوفة هي مجرد مسميات للعقد. داخل المصفوفة، الرقم واحد يعني وجود حافة بين الرؤوس (العقد) التي تمثل الصف والعمود. وأخيرًا، الرقم صفر يعني عدم وجود حافة أو علاقة.

<pre>
    1 2 3
  \------
1 | 0 1 1
2 | 1 0 0
3 | 1 0 0
</pre>

أعلاه رسم بياني بسيط جدًا غير موجَّه يحتوي على ثلاث عقد، حيث العقدة الأولى متصلة بالعقدة الثانية والثالثة. أدناه تنفيذ جافاسكريبت لنفس الشيء.

```js
var adjMat = [
  [0, 1, 1],
  [1, 0, 0],
  [1, 0, 0]
];
```

على عكس قائمة المجاورة، يجب أن يحتوي كل "صف" في المصفوفة على نفس عدد العناصر مثل عدد العقد في الرسم البياني. هنا لدينا مصفوفة 3×3، مما يعني أن لدينا ثلاث عقد في الرسم البياني. الرسم البياني الموجَّه سيكون مشابهًا. أدناه رسم بياني حيث العقدة الأولى لها حافة تشير إلى العقدة الثانية، ثم العقدة الثانية لها حافة تشير إلى العقدة الثالثة.

```js
var adjMatDirected = [
  [0, 1, 0],
  [0, 0, 1],
  [0, 0, 0]
];
```

يمكن أن تحتوي الرسوم البيانية أيضًا على <dfn>أوزان</dfn> على حوافها. حتى الآن، لدينا حواف <dfn>غير موزونة</dfn> حيث وجود الحافة أو عدمها ثنائي (`0` أو `1`). يمكنك استخدام أوزان مختلفة حسب تطبيقك.

# --instructions--

أنشئ مصفوفة مجاورة لرسم بياني غير موجَّه يحتوي على خمس عقد. يجب أن تكون هذه المصفوفة في مصفوفة متعددة الأبعاد. هذه العقد الخمس لها علاقات بين العقدة الأولى والرابعة، والعقدة الأولى والثالثة، والعقدة الثالثة والخامسة، والعقدة الرابعة والخامسة. جميع أوزان الحواف تساوي واحدًا.
