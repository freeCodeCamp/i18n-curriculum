# --description--

Los arrays son una estructura de datos fundamental en la informática. Todos los arrays almacenan colecciones ordenadas de datos, pero dependiendo de su tipo, pueden funcionar de manera diferente detrás de escena.

El comportamiento subyacente puede tener un efecto importante en la eficiencia del programa, así que aprendamos sobre arreglos dinámicos y estáticos y sus diferencias, para que puedas elegir el más eficiente para tu programa.

Comenzaremos con arreglos estáticos.

**Los arrays estáticos** tienen un tamaño fijo. Almacenan elementos en ubicaciones de memoria adyacentes.

El tamaño de un arreglo estático se determina cuando el arreglo se inicializa. Una vez que ese bloque específico de memoria está asignado, es fijo y no puede cambiarse mientras el programa se ejecuta. Esta es una característica clave de los arreglos estáticos.

Almacenar elementos en ubicaciones de memoria adyacentes hace que el proceso de recuperación de datos sea más eficiente porque el programa puede almacenar la ubicación del primer elemento y luego usar índices para hacer cálculos simples y encontrar los otros elementos en la memoria.

Gracias a esto, acceder a los valores de un arreglo estático toma un tiempo constante `O(1)`, lo cual es muy eficiente.

Puedes usar un arreglo estático cuando sabes de antemano el número de elementos que se almacenarán. También es útil cuando los valores se accederán con mucha frecuencia, ya que la operación de acceso es muy eficiente.

Sin embargo, esta estructura de datos no puede crecer ni reducirse, así que si la cantidad de elementos que se almacenarán puede variar, deberías usar un arreglo dinámico en su lugar.

Intentar aumentar el tamaño de un arreglo estático implicaría crear un nuevo arreglo y copiar todos los elementos del arreglo antiguo a uno nuevo, lo cual es ineficiente. En ese caso, un arreglo dinámico sería mucho mejor porque maneja este proceso automáticamente.

Python no incluye arrays estáticos tradicionales como estructuras de datos integradas.

Pero otros lenguajes de programación, como Java, sí los soportan. Este es un ejemplo de un arreglo estático en Java que puede almacenar tres enteros:

```java
int[] numbers = new int[3]; 
```

Los arrays en Python son dinámicos, así que echemos un vistazo a ellos.

**Los arreglos dinámicos** son más flexibles porque pueden crecer o reducirse automáticamente mientras el programa se está ejecutando.

Funcionan mediante un mecanismo de redimensionamiento automático que copia los elementos en un nuevo array cuando el array original está lleno. El proceso se realiza de manera eficiente porque el tamaño del nuevo array se elige de una forma eficiente que hace que estas operaciones computacionalmente costosas sean menos frecuentes.

Acceder a los elementos de un arreglo dinámico toma un tiempo constante `O(1)`, por lo que esta operación es muy eficiente.

Insertar un elemento en medio del arreglo toma tiempo lineal `O(n)` porque los elementos después de él necesitan ser reubicados.

Insertar un elemento al final del arreglo toma un tiempo constante `O(1)` si todavía hay espacio disponible en el arreglo dinámico, pero si el arreglo está lleno y necesita redimensionarse, esta operación tiene una complejidad de `O(n)`.

Debes usar arreglos dinámicos cuando no sepas de antemano la cantidad de valores que necesitarás almacenar en el arreglo. También son útiles cuando insertarás y eliminarás elementos con frecuencia.

La estructura de datos `list` incorporada de Python funciona como un arreglo dinámico. Puedes crear una lista escribiendo los elementos dentro de corchetes, separados por comas.

```python
numbers = [3, 4, 5, 6]
```

Puedes acceder a un elemento escribiendo el nombre de la variable que contiene la lista, seguido de corchetes, y dentro de los corchetes, el índice correspondiente.

Los índices comienzan desde 0 para el primer elemento y se incrementan en 1 para cada elemento subsecuente:

```python
numbers[0]  # 3
numbers[1]  # 4
numbers[2]  # 5
numbers[3]  # 6
```

Para actualizar un valor, solo necesitas reasignarlo:

```python
numbers[2] = 16
```

Puedes agregar elementos a la lista con el método `.append()`:

```python
numbers.append(7)
```

Puedes insertar elementos en un índice específico con el método `.insert()`, pasando el índice como primer argumento y el elemento como segundo argumento.

```python
numbers.insert(3, 15)
```

Puedes eliminar un elemento en un índice específico con el método `.pop()`:

```python
numbers.pop(2)
```

Si no especificas el índice, `.pop()` eliminará el último elemento.

Hay otros métodos integrados de listas que puedes consultar en la documentación para agregar y eliminar elementos con bastante facilidad.

Esa es la potencia de los arreglos dinámicos, o listas en este caso.

En general, debes usar arreglos estáticos cuando sabes el número de elementos de antemano y necesitas acceder a ellos con frecuencia, y usar arreglos dinámicos cuando el número de elementos es desconocido o variable con el tiempo.

Siempre debes considerar la compensación entre la simplicidad de los arreglos estáticos y la flexibilidad de los arreglos dinámicos. Ambos son útiles para casos de uso y escenarios específicos. Poder elegir el mejor para un problema dado es parte de las habilidades para resolver problemas que desarrollarás gradualmente con la práctica.
