# --description--

`Step by step` означає робити щось поетапно. Наприклад, `debugging step by step` означає розв’язувати проблему маленькими, керованими частинами.

`Input` у програмуванні часто означає інформацію або пораду, надану для проєкту.

`Work together` означає співпрацювати, а `pair up` схоже, означає двох людей, які тісно працюють разом.

Нарешті, `to solve a problem` означає намагатися знайти рішення. У цьому випадку Том і Сара намагаються знайти рішення для можливої помилки в коді.
