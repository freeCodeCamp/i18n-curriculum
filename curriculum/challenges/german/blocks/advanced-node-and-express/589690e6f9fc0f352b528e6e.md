# --description--

Im Moment befindet sich alles, was Sie haben, in Ihrer `server.js`-Datei. Das kann zu schwer zu verwaltendem Code führen, der nicht sehr erweiterbar ist. Erstellen Sie 2 neue Dateien: `routes.js` und `auth.js`

Beide sollten mit folgendem Code beginnen:

```js
module.exports = function (app, myDataBase) {

}
```

Fordern Sie nun ganz oben in Ihrer Server-Datei diese Dateien wie folgt an: `const routes = require('./routes.js');` Direkt nachdem Sie eine erfolgreiche Verbindung mit der Datenbank hergestellt haben, instanziieren Sie jede von ihnen so: `routes(app, myDataBase)`

Kopieren Sie schließlich alle Routen aus Ihrem Server in Ihre neuen Dateien und entfernen Sie sie aus Ihrer Server-Datei. Nehmen Sie auch die `ensureAuthenticated`-Funktion mit, da sie speziell für das Routing erstellt wurde. Nun müssen Sie die verwendeten Abhängigkeiten korrekt hinzufügen, wie zum Beispiel `const passport = require('passport');`, ganz oben über der Exportzeile in Ihrer `routes.js`-Datei.

Fügen Sie sie so lange hinzu, bis keine Fehler mehr auftreten und Ihre Server-Datei keine Routen mehr enthält (**außer der Route im Catch-Block**)!

Machen Sie dasselbe in Ihrer `auth.js`-Datei mit allen Dingen, die mit der Authentifizierung zu tun haben, wie der Serialisierung und dem Einrichten der lokalen Strategie, und löschen Sie sie aus Ihrer Server-Datei. Achten Sie darauf, die Abhängigkeiten hinzuzufügen und `auth(app, myDataBase)` an derselben Stelle im Server aufzurufen.

Senden Sie Ihre Seite ab, wenn Sie denken, dass alles richtig ist. Wenn Sie auf Fehler stoßen, können Sie <a href="https://forum.freecodecamp.org/t/advanced-node-and-express/567135#clean-up-your-project-with-modules-2" target="_blank" rel="noopener noreferrer nofollow">ein Beispiel des fertigen Projekts ansehen</a>.
