# --explanation--

当被问及为什么选择 Python 作为你的第一门语言时，最好强调 Python 的具体优势。

Python 以其对初学者友好而闻名，具有简单且易于读取和学习的语法。此外，Python 拥有丰富的库和框架集合，这使其成为初学者的实用选择。

另一方面，说 `I've always wanted to be a developer` 并不能解释你为什么选择 Python。它侧重于成为开发者的愿望，但缺乏任何针对 Python 的具体理由。
