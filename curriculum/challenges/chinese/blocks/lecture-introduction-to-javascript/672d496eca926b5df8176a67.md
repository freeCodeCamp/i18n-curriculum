# --interactive--

在 JavaScript 中，数据类型是你保存的值的种类，比如数字或一段文本。

变量是一个命名的容器，用来存储特定数据类型的值，允许你在整个代码中引用和操作它。 

你可能还记得在数学课上这样使用变量：

```md
x = 2
y = 4

x + y
```

你能够创建像 `x` 和 `y` 这样的变量，然后在你的程序中引用它们并执行诸如加法的数学操作。这在编程中是一个类似的概念。你可以创建你自己的变量名并为它们赋值。这些值将是不同的数据类型。

数据类型可以帮助程序理解所处理的数据的类型，无论是数值、文本还是其他数据。

JavaScript 有几种基本数据类型，你会在程序中用到。  We'll explore each data type in greater detail in future lessons. 现在，我们将简要介绍 JavaScript 中的不同数据类型。

我们要了解的第一个数据类型是 `Number` 类型。

`Number` 表示整数和浮点数。 整数的例子包括 `7`、`19` 和 `90`。 

**注意**：`console.log()` 是一个将信息输出到控制台的函数，控制台是你网页浏览器中用于调试代码的部分。你将在后续课程中学习更多关于 `console.log()` 的内容。此外，`//` 符号用于在你的代码中添加评论。评论是给你自己或其他程序员的注释，代码运行时会被忽略。

启用交互式编辑器并尝试更改一些整数以查看它在控制台中的更新。

:::interactive_editor

```js
// Examples of integers
console.log(3);
console.log(5);
console.log(-67);
```

:::

浮点数是带有小数点的数值。 浮点数的例子包括 `3.14` 和 `5.2`。 

:::interactive_editor

```js
// Examples of floating point numbers
console.log(3.14);
console.log(7.2);
console.log(-14.5);
```

:::

下一个数据类型是 `String` 。

`String` 是由字符或文本组成的序列，包含在引用中。以下是使用双引号的示例字串： 

:::interactive_editor

```js
console.log("I love to code!");
```

:::

这是一个使用单引号的示例：

:::interactive_editor

```js
console.log('I love to code!');
```

:::

你经常会使用字串来表现名称、标签、消息等。

JavaScript 中使用的另一种数据类型是 `Boolean` 类型。

一个 `Boolean` 表现为两个值之一：`true` 或 `false`。例如，一个程序可能会查看用户是否已登录（`true`）或未登录（`false`），并根据此更改页面。如果用户已登录，你可能想向他们显示仪表盘页面。否则，你将想向他们显示登录页面。

JavaScript 中使用的接下来的两种数据类型是 `undefined` 和 `null`。

`undefined` 意味着变量已被声明但尚未赋值。你将在下一课中学习更多内容。

`null` 意味着变量已被故意设置为“无”，且不包含任何值。我们将在后续课程中进一步探讨其工作原理。

最后三种数据类型本质上更复杂。这些是对象、符号和 `BigInt`。 

一个 `Object` 是一个键值点对的集合。 

```js
{
  name: "Alice",
  age: 30
};
```

对象非常适合将相关信息分组在一起。你将在未来的模块中学习更多关于如何使用对象的内容。

`Symbol` 是 JavaScript 中一种特殊类型的值，始终唯一且不可更改。 它通常用于为属性创建独特的标签或标识符：

```js
Symbol('mySymbol');
```

`BigInt` 用于超过 `Number` 类型限制的超大数字：

```js
1234567890123456789012345678901234567890n;
```

在这个示例中，我们通过在一个非常大的数值末尾添加 `n` 来创建一个 `BigInt`。

`Symbol` 和 `BigInt` 是两种较少使用的类型，但了解它们仍然很重要。

了解这些数据类型有助于你在程序中处理和使用各种数据，因为每种数据类型都有自己的特性和行为。
