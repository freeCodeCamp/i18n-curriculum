# --explanation--

`Present Continuous` 시제는 지금 일어나고 있는 동작을 설명할 때 사용합니다. 무언가가 **일어나지 않고 있다**는 것을 표현하려면 부정형을 사용합니다.

문장 `The live preview isn't showing`에서 `isn't`는 `is not`의 축약형으로, 문장을 부정문으로 만듭니다.

이는 라이브 미리보기가 예상대로 작동하지 않고 있음을 나타냅니다. 따라서 Tom은 Visual Studio Code에서 라이브 미리보기가 현재 보이지 않는다고 보고하고 있습니다.
