# --description--

Express.js, ou simplesmente Express, é um framework web minimalista e flexível construído **sobre** o Node.js. Ele simplifica o processo de criar servidores web e APIs.

Em vez de escrever uma tonelada de boilerplate com o módulo nativo `http` do Node, o Express permite criar aplicações web robustas com apenas algumas linhas.

Você pode pensar nele como a camada rápida e sem opiniões que torna o Node.js mais fácil e poderoso.

Aqui estão três motivos principais pelos quais os desenvolvedores adoram o Express:

1. **Minimalista e rápido** – O Express fornece apenas o essencial, dando controle total enquanto permite adicionar recursos conforme necessário por meio de middleware e plugins.

2. **Ecossistema enorme** – Possui uma grande coleção de middleware e plugins para logging, autenticação, segurança, sessões e muito mais.

3. **Ótimo para APIs** – Seu modelo simples de roteamento e tratamento de requisições torna o Express especialmente adequado para construir APIs RESTful.

Com o Express, você pode criar:

* Aplicações web

* APIs RESTful

* Backends para Single Page Applications (SPA)

* Serviços com muito middleware

* Qualquer coisa do lado do servidor com HTTP


## Como instalar o Express

Primeiro, certifique-se de que você tem Node.js e npm instalados. Veja como verificar se node e npm estão instalados:

```bash
node -v
npm -v
```

Se você vir um número de versão, significa que está instalado. Caso contrário, será necessário instalá-los. Você pode encontrar instruções de instalação em uma lição anterior ou no site oficial do Node.js.

Em seguida, no seu terminal, digite os seguintes comandos:

```bash
mkdir my-app
cd my-app
npm init -y
npm install express
```

Você vai aprender mais sobre middleware e plugins em lições futuras, mas, resumidamente, middleware é código que roda entre o recebimento de uma requisição e o envio de uma resposta. É comumente usado para tratamento de erros, validação de entrada, modificação de dados na requisição ou resposta, e mais. Você pode encadear middleware para construir lógica flexível e reutilizável para tratar requisições.

Plugins são normalmente módulos externos de terceiros para Node.js que estendem a funcionalidade do Express. Você pode instalá-los separadamente com npm, e a maioria é implementada como middleware que pode ser usado no seu app com pouca configuração. Em vez de escrever recursos comuns do zero, você pode usar um plugin para lidar com coisas como logging, autenticação ou melhores práticas de segurança.

Express é um framework popular para desenvolvimento do lado do servidor em Node.js. Ele oferece os blocos de construção para rapidamente levantar um servidor, gerenciar rotas, middleware e construir APIs — tudo com código limpo e legível.
