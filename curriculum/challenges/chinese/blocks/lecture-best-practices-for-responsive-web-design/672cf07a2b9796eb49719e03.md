# --interactive--

媒体断点是网站设计中的特定点，在这些点上，布局和内容会进行调整，以适应不同的屏幕尺寸。 这些断点在响应式网页设计中至关重要，可让开发人员创建的网站在各种设备上，从手机到大型台式机显示器，都具有良好的外观和功能。

在 CSS 中，媒体断点是通过媒体查询来实现的。 通过这些查询，你可以根据设备的特征，最常见的是视口宽度，应用不同的样式。 例如，你可以将断点设置为 768 像素，以区分手机和平板电脑布局。

下面是在 CSS 中的一个带有断点的媒体查询的简单示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>
  <h1>Responsive Design</h1>
  <p>This is a simple example of responsive design using media queries.</p>
</div>
```

```css
/* Styles for screens wider than 768px */
@media screen and (min-width: 768px) {
  body {
    font-size: 1.125rem;
  }
}
```

:::

当屏幕宽度为 768 像素或更大时，字体大小将增加到 `1.125rem` 即 18 像素。 这演示了如何使用断点来调整设计，以适应不同的屏幕尺寸。

在选择断点时，没有放之四海而皆准的解决方案。 适合你网站的断点取决于你的具体设计和内容。 不过，在现代网页设计中，有一些常见的断点被许多设计师用作设计起点。

一组常用的断点与常见的设备类别相对应：

- 小型设备（智能手机）：最大 640px。

- 中型设备（平板电脑）： 641px 至 1024px。

- 大型设备（台式机）： 1025px 或更大。

有些设计人员喜欢更细的粒度，如使用如下断点：

- 超小设备：最大 576px。

- 小型设备： 577px 至 768px。

- 中型设备： 769px 至 992px。

- 大型设备： 993px 至 1200px。

- 超大设备： 1201px 或更大。

需要注意的是，这些并不是严格的规定，而是常见的做法。

现代响应式设计的趋势是采用更流畅的方法，使设计在各种屏幕尺寸下都能流畅适应，而不是在设定的断点上做出巨大的改变。

下面是一些其他常见的断点示例：

- 超小设备：576px 以下。

- 小型设备：大于或等于 576px。

- 中型设备：大于或等于 768px。

- 大型设备：大于或等于 992px。

- 超大设备：大于或等于 1200px。

- 超超大设备：大于或等于 1400px。

这些断点应用广泛，可以作为许多项目的良好起点。 不过，重要的是要记住，项目的最佳断点应由内容和设计决定，而不是由任意的数字或设备尺寸决定。
