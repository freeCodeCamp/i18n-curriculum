# --interactive--

前のレッスンでは、`grid-template-columns` プロパティを使い、その値をいくつかの分数単位に設定してきました。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

以下のコードは完全に有効ですが、トラックリストの一部またはすべてを繰り返すもっと簡単な方法があります。

`repeat()` 関数は、列や行のトラックの一部またはすべてを繰り返すために使われます。この関数は繰り返し回数と繰り返したいトラックを受け取ります。

こちらは、先ほどの例を repeat 関数を使って書き直したものです。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

ブラウザ上のスタイルに変化はありませんが、列の繰り返し値をより簡潔に書く方法です。

`repeat()` 関数は、行や列に使える任意の有効なパターンを受け入れます。

こちらは、`repeat()` 関数を使って、1列目と3列目を20ピクセル、2列目と4列目を1分数単位に設定した例です。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(2, 20px 1fr);
  column-gap: 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

時には、`repeat()` 関数を使わずに各値を個別に書くこともあります。しかし、特定のパターンをトラックリストで繰り返したい場合、この関数はとても便利です。
