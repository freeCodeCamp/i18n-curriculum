# --explanation--

`Several` significa mais do que alguns, mas não todos.Por exemplo:

`Several users reported issues with the new software.` – Isso significa que mais de dois usuários tiveram problemas, mas não todos.

`To target` significa mirar ou direcionar algo para alguém ou algo.Por exemplo:

`The hackers targeted customers by sending fake emails.` – Isso significa que eles tentaram enganar os clientes especificamente.
