# --description--

在本次研讨会中，你将学习如何创建一个卡路里计数器表单，该表单允许用户输入他们的每日卡路里预算和各种餐食的卡路里数。然后，表单将计算并显示用户是处于卡路里赤字还是盈余。

已为你提供样板 CSS 和 HTML。 然而，你需要建立自己的卡路里计数器表格。

请随意探索 HTML 和 CSS，然后添加 `form` 元素并将 `id` 设置为 `calorie-counter`。
