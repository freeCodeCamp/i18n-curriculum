# --description--

`Strange` 意味著不尋常或出乎意料。這就像發現與你平常看到的不同的東西。例如：`A strange sound in a car might mean a problem.`

`error` 是一種錯誤，特別是會導致某些功能無法正常運作的錯誤。例如：`An error in a code means there is a mistake that needs fixing.`
