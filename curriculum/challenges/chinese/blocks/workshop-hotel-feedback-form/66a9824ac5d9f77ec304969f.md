# --description--

如果你希望用户有更多空间来撰写评论，可以使用 `textarea` 元素。

`textarea` 元素是一个多行文本输入控件，允许用户输入比单行更长的文本。 它可用于创建评论框、消息输入或其他需要多行的文本输入。

下面是一个 `textarea` 元素的示例：

```html
<textarea id="comments" name="comments" rows="4" cols="50"></textarea>
```

`rows` 属性用于指定 `textarea` 的可见高度，`cols` 属性用于制定 `textarea` 的可见宽度。

在你的 `label` 元素下方，添加一个 `textarea` 元素。 下一步，你将添加必要的属性。

# --hints--

你的标签下方应该有一个 `textarea` 元素。

```js
assert.exists(document.querySelector('label + textarea'));
```
