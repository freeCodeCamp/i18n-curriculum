# --explanation--

`Lastly` se usa para introducir el punto final en una lista o discusión. Por ejemplo:

`Lastly, let's review the budget changes.` – Esto significa que es el último tema del que hablar.

`Any thoughts on...` es una forma informal de pedir las ideas u opiniones de alguien. A menudo omite la frase completa `Do you have any thoughts on...` al inicio. Por ejemplo:

`Any thoughts on how we can improve this layout?` – Esto significa que el hablante está pidiendo ideas de manera amigable e informal sobre cómo mejorar un diseño.
