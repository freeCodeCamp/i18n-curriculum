# --interactive--

**注意**: いくつかのインタラクティブな例では、まだ学んでいないCSSやJavaScriptが使われている場合があります。すべてのコードを理解しようと心配しないでください。これらの例の目的は、デザインコンセプトのプレビューを見せて、仕組みをよりよく理解してもらうことです。プレビューを見るには、インタラクティブエディターを有効にしてください。

無限スクロールは、ユーザーがページをスクロールダウンするにつれてコンテンツを追加で読み込むデザインパターンです。多くの場合、Twitterのようなソーシャルメディアサイトで使われています。例えば、ログインしていてもっとツイートを見たい場合、スクロールダウンするとさらにツイートが読み込まれます。これが無限スクロールの例です。

```js
// Simulate loading more content as the user scrolls
window.addEventListener('scroll', () => {
  if (window.innerHeight + window.scrollY >= document.body.offsetHeight) {
    loadMorePosts();
  }
});

function loadMorePosts() {
  const container = document.querySelector('.infinite-scroll');
  for (let i = 0; i < 3; i++) {
    const post = document.createElement('div');
    post.className = 'post';
    post.textContent = `Post ${container.children.length + 1}`;
    container.appendChild(post);
  }
}
```

無限スクロールはページネーションの代わりとしても使われます。ページネーションはコンテンツを複数のページに分割するデザインパターンで、表示するコンテンツが多い場合によく使われます。例えば、Googleで検索すると複数ページにわたって検索結果が表示されます。ページネーションでは次のページに行くためにボタンをクリックする必要がありますが、無限スクロールではスクロールを続けるだけでさらにコンテンツが読み込まれます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="pagination">
  <div class="post">Post 1</div>
  <div class="post">Post 2</div>
  <div class="post">Post 3</div>
  <!-- More posts will be on the next pages -->
</div>
<div class="pagination-controls">
  <button class="prev" disabled>Previous</button>
  <button class="next">Next</button>
</div>
<script src="index.js"></script>
```

```css
.pagination {
  display: flex;
  flex-direction: column;
  gap: 12px;
}

.post {
  padding: 12px;
  border: 1px solid #ccc;
  border-radius: 4px;
}

.pagination-controls {
  margin-top: 12px;
}

.pagination-controls button {
  padding: 8px 16px;
  margin-right: 8px;
}
```

```js
let currentPage = 1;
const postsPerPage = 3;
const totalPosts = 9; 
const totalPages = Math.ceil(totalPosts / postsPerPage);
const container = document.querySelector('.pagination');
const prevButton = document.querySelector('.prev');
const nextButton = document.querySelector('.next'); 

function renderPosts() {
  container.innerHTML = '';
  const start = (currentPage - 1) * postsPerPage;
  const end = start + postsPerPage;
  for (let i = start; i < end && i < totalPosts; i++) {
    const post = document.createElement('div');
    post.className = 'post';
    post.textContent = `Post ${i + 1}`;
    container.appendChild(post);
  }
  prevButton.disabled = currentPage === 1;
  nextButton.disabled = currentPage === totalPages;
}

prevButton.addEventListener('click', () => {
  if (currentPage > 1) {
    currentPage--;
    renderPosts();
  }
});

nextButton.addEventListener('click', () => {
  if (currentPage < totalPages) {
    currentPage++;
    renderPosts();
  }
});

renderPosts();
```

無限スクロールをデザインに取り入れる際には、いくつかのベストプラクティスを覚えておくと良いです。最初のポイントは、「Load More」ボタンを用意して、ユーザーがクリックしたときに次の結果セットを読み込むようにすることです。これはユーザーがいつもっとコンテンツを見たいかをコントロールできる良い方法です。

```html
  <div class="post">Post 2</div>
  <div class="post">Post 3</div>
  <!-- More posts will load when the user clicks "Load More" -->
</div>
<button class="load-more">Load More</button>
<script src="index.js"></script>
```

```css

  padding: 12px;
  border: 1px solid #ccc;
  border-radius: 4px;
```

```js
const loadMoreButton = document.querySelector('.load-more');
```

:::

もう一つのポイントは「Back」ボタンを追加することです。これによりユーザーはスクロールで上まで戻らなくても前のページに戻ることができ、より良いユーザー体験となり、ブラウジングのコントロールが増えます。

時には「Back to the top」ボタンがあり、ユーザーを結果ページのトップに戻します。さらに、読み込み中のインジケーターを用意することも重要です。ユーザーはコンテンツが読み込まれていることをはっきりと認識できなければ、ページが壊れていると思うかもしれません。

:::interactive_editor

```js
const container = document.querySelector('.infinite-scroll');
const loadingIndicator = document.querySelector('.loading-indicator');
const backToTopBtn = document.getElementById('back-to-top');

window.addEventListener('scroll', () => {
  if (window.scrollY > 400) {
    backToTopBtn.style.display = 'block';
  } else {
    backToTopBtn.style.display = 'none';
  }

  if (window.innerHeight + window.scrollY >= document.body.offsetHeight) {
    loadMorePosts();
  }
});

backToTopBtn.addEventListener('click', () => {
  window.scrollTo({ top: 0, behavior: 'smooth' });
});

function loadMorePosts() {
  loadingIndicator.style.display = 'block';
  setTimeout(() => {
    for (let i = 0; i < 3; i++) {
      const post = document.createElement('div');
      post.className = 'post';
      post.textContent = `Post ${container.children.length + 1}`;
      container.appendChild(post);
    }
    loadingIndicator.style.display = 'none';
  }, 1000);
}
``` 

:::

最後のポイントの一つは、フッターを常にユーザーがアクセスできるようにしておくことです。フッターに重要な情報が含まれている場合は、常にアクセス可能であるべきです。

まとめると、無限スクロールはウェブサイトでコンテンツを表示する素晴らしい方法です。ただし、無限スクロールをデザインする際にはベストプラクティスを守り、最高のユーザー体験を提供できるようにしてください。
