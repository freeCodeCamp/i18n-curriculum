# --interactive--

في الدرس السابق، تعلمت كيف تتعامل مع كائن `arguments` الذي يشبه المصفوفة ويحتوي على قيم المعلمات الممررة إلى الدالة.

:::interactive_editor

```js
function logArgs() {
  for (const arg of arguments) {
    console.log(arg);
  }
}

logArgs(1, 2, 3);
// result:
// 1
// 2
// 3
```

:::

بينما هذه طريقة صحيحة للوصول إلى مجموعة متغيرة من المعلمات والعمل معها في دالة، تستخدم تطبيقات JavaScript الحديثة صيغة معلمات الباقي بدلاً من ذلك.

إليك مثالًا محدثًا يستخدم صيغة معلمات الباقي بدلاً من كائن `arguments`:

:::interactive_editor

```js
function logArgs(...args) {
  for (const arg of args) {
    console.log(arg);
  }
}

logArgs(1, 2, 3);
// result:
// 1
// 2
// 3
```

:::

هذا المثال المحدث لم يعد يشير مباشرة إلى كائن `arguments`. بدلاً من ذلك، تحتوي المعلمة الأخيرة في تعريف الدالة على ثلاث نقاط أمامها. هذا يجعل معلمة الباقي تُوضع داخل كائن `Array`. يمكنك تسمية معلمة الباقي بأي اسم تريده، فقط تأكد من أنها المعلمة الأخيرة في تعريف الدالة كما يلي:

```js
function exampleFunction(a, b, ...restOfArgs) {
  // some code here
}

function anotherFunction(x, y, ...theArgs) {
  // some code here
}
```

هناك بعض القيود الإضافية عند التعامل مع صيغة معلمات الباقي. أحد القيود هو أن تعريفات الدوال يمكن أن تحتوي على معلمة باقي واحدة فقط. لذا المثال التالي يعتبر غير صالح.

```js
// Won't work

function badFunction(...args, ...moreArgs) {
  // some code here
}
```

القيود التالية هي أنه لا يُسمح بفواصل زائدة بعد معلمة الباقي: `function exampleFunction(a, b, ...restOfArgs, )`.

قيد آخر هو أن معلمة الباقي لا يمكن أن تحتوي على قيمة افتراضية. وإلا سيتم رمي `SyntaxError`.

```js
function badFunction(...args = [1,2]){
  // some code here
}
```

فما هي بعض الاختلافات بين كائن `arguments` ومعلمات `rest`؟

فرق رئيسي هو أن كائن `arguments` ليس مصفوفة حقيقية لذا لا يدعم طرق مثل `includes` و `pop` و `push`. لكن معلمة الباقي هي نسخة من `Array`. لذلك يمكنك استخدام طرق المصفوفة المدمجة الصحيحة دون الحاجة لتحويلها إلى مصفوفة حقيقية أولاً.

:::interactive_editor

```js
function hasCat(...args) {
  return args.includes("cat");
}

console.log(hasCat("dog", "chicken", "cat")); // true
console.log(hasCat("dog", "chicken", "horse")); // false
```

:::

ما هو الفرق الرئيسي بين كائن `arguments` ومعلمات الباقي؟
