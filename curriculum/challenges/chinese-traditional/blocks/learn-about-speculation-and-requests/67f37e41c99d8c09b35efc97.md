# --explanation--

Jake 說，`to see if there's any unusual activity or failed login attempts`。這表示他想要尋找系統中可疑行為的跡象。

- `Unusual activity` 可以指奇怪或意外的行為，例如在非正常時間存取檔案。

- `Failed login attempts` 是有人可能嘗試在沒有正確密碼的情況下存取系統的跡象。

稽核日誌有助於追蹤這些事項，並且對調查潛在的安全問題很有幫助。
