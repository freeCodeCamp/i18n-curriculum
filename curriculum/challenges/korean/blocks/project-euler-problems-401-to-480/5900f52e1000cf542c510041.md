# --description--

하이포사이클로이드는 큰 원 안에서 작은 원이 구를 때 작은 원 위의 한 점이 그리는 곡선입니다. 원점에 중심을 두고 오른쪽 끝 점에서 시작하는 하이포사이클로이드의 매개변수 방정식은 다음과 같습니다:

$$x(t) = (R - r) \cos(t) + r \cos(\frac{R - r}{r}t)$$

$$y(t) = (R - r) \sin(t) - r \sin(\frac{R - r}{r} t)$$

여기서 $R$은 큰 원의 반지름이고 $r$은 작은 원의 반지름입니다.

$C(R, r)$를 반지름이 $R$과 $r$인 하이포사이클로이드 위에 있고, $\sin(t)$와 $\cos(t)$가 유리수인 $t$ 값에 대응하는 정수 좌표를 갖는 서로 다른 점들의 집합이라고 합시다.

$S(R, r) = \sum\_{(x,y) \in C(R, r)} |x| + |y|$는 $C(R, r)$에 속하는 점들의 $x$와 $y$ 좌표 절댓값의 합입니다.

$T(N) = \sum_{R = 3}^N \sum_{r=1}^{\left\lfloor \frac{R - 1}{2} \right\rfloor} S(R, r)$는 $R$과 $r$이 양의 정수이고 $R \leq N$, $2r < R$인 조건에서 $S(R, r)$의 합입니다.

다음이 주어졌습니다:

$$\begin{align}
  C(3, 1) = & \\{(3, 0), (-1, 2), (-1,0), (-1,-2)\\} \\\\
  C(2500, 1000) = & \\{(2500, 0), (772, 2376), (772, -2376), (516, 1792), (516, -1792), (500, 0), (68, 504), \\\\
                  &(68, -504),(-1356, 1088), (-1356, -1088), (-1500, 1000), (-1500, -1000)\\}
\end{align}$$

**참고:** (-625, 0)은 $C(2500, 1000)$의 원소가 아닙니다. 해당 $t$ 값에 대해 $\sin(t)$가 유리수가 아니기 때문입니다.

$S(3, 1) = (|3| + |0|) + (|-1| + |2|) + (|-1| + |0|) + (|-1| + |-2|) = 10$

$T(3) = 10$; $T(10) = 524$; $T(100) = 580\\,442$; $T({10}^3) = 583\\,108\\,600$.

$T({10}^6)$를 구하세요.
