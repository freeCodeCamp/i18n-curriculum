# --interactive--

有幾種方法可以從物件中移除屬性，其中使用 `delete` 運算子是最直接且常用的方法。

當你使用 `delete`，它會從物件中移除所選的屬性。以下是如何使用 `delete` 運算子的範例：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  job: "Engineer"
};

delete person.job;

console.log(person.job); // undefined
```

:::

在這個範例中，我們從一個具有三個屬性：`name`、`age` 和 `job` 的 `person` 物件開始。然後，我們使用 `delete` 運算子來移除 `job` 屬性。刪除後，`person` 物件不再有 `job` 屬性。

另一種移除屬性的方法是使用帶有剩餘參數的解構指派。這種方法並不會真正刪除屬性，而是創建一個不包含指定屬性的全新物件：

:::interactive_editor

```js
const person = {
  name: "Bob",
  age: 25,
  job: "Designer",
  city: "New York"
};

const { job, city, ...remainingProperties } = person;

// { name: "Bob", age: 25 }
console.log(remainingProperties);
```

:::

在這個範例中，我們使用解構賦值從 `person` 物件中提取 `job` 和 `city`，並將剩餘的屬性收集到一個名為 `remainingProperties` 的新物件中。這會創建一個不包含 `job` 和 `city` 屬性的新物件。

了解如何從物件中移除屬性是 JavaScript 程式設計中的一項重要技能。它讓你能夠動態操作物件，並清理不必要的資料。
