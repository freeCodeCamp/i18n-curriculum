# --explanation--

`Pushback` означає опір або незгоду з планом чи пропозицією. Наприклад:

- `We got some pushback from the client about the new design.` - Клієнт не повністю погодився або мав зауваження до дизайну.

- `Expect pushback if we change the deadline again.` - Люди можуть чинити опір або скаржитися, якщо змінити дату.
