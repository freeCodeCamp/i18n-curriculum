# --explanation--

`Should have` + `Past Participle`는 과거에 대한 후회나 조언을 표현할 때 사용합니다. 예를 들어:

- `You should have run the program on a different machine to check for compatibility.` - 다른 컴퓨터에서 테스트했으면 도움이 되었겠지만 하지 않았습니다.

- `I should have run the tests before deploying the app.` - 테스트를 실행하는 것이 옳았지만 당시에는 하지 않았습니다.

이 경우 Sophie는 Mark에게 테스트 스위트를 실행했어야 한다고 말하는데, 이는 중요했지만 하지 않았다는 뜻입니다.
