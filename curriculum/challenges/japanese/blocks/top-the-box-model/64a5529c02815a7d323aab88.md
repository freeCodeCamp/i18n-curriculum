# --description--

HTMLとCSSを検査してデバッグできることは、フロントエンド開発において非常に重要です。このレッスンでは、Chrome Dev Toolsを使って、要素やCSSルールの詳細情報を確認し、コードの問題を見つけて修正する方法を学びます。

インスペクターを開くには、ウェブページの任意の要素を右クリックして「Inspect」をクリックするか、F12キーを押してください。今すぐ試して、このページで使われているHTMLとCSSを見てみましょう。

今見ているたくさんのツールに圧倒されないでください！このレッスンでは、ElementsとStylesのペインにフォーカスします。
