# --description--

$n$을 양의 정수라고 하겠습니다. 정수 삼중항 ($a$, $b$, $c$)가 다음 조건을 만족하면 $n$의 인수분해 삼중항이라고 합니다:

- $1 ≤ a ≤ b ≤ c$
- $a \times b \times c = n$.

$f(n)$을 $n$의 인수분해 삼중항 ($a$, $b$, $c$) 중에서 $\frac{c}{a}$를 최소화하는 삼중항의 $a + b + c$로 정의합니다. 이 삼중항은 유일함을 증명할 수 있습니다.

예를 들어, $f(165) = 19$, $f(100\\,100) = 142$, 그리고 $f(20!) = 4\\,034\\,872$입니다.

$f(43!)$를 구하세요.
