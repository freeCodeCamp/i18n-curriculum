# --interactive--

## Робота з різними селекторами атрибутів і посиланнями

- **Визначення**: селектор `attribute` дозволяє цілитися в HTML-елементи на основі їхніх атрибутів, таких як атрибути `href` або `title`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Link with href</a>
<a>No href</a>
```

```css
a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

- **Атрибут `title`**: цей атрибут надає додаткову інформацію про елемент. Ось як можна цілитися в посилання з атрибутом `title`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Tooltip">Link with title</a>
<a href="#">Normal link</a>
```

```css
a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

- **Комбінування селекторів для цільового вибору посилань, які мають одночасно атрибути `href` і `title`**: поєднує кілька селекторів атрибутів.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Info">Href + Title</a>
<a href="#">Only href</a>
```

```css
a[href][title] {
  color: green;
}
```

:::

- **Вибір одного слова в списку, розділеному пробілами**: цілиться в посилання, які мають конкретне слово класу.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a class="link primary">Primary link</a>
<a class="link secondary">Secondary link</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

- **Вибір значень, що починаються з певного префікса**: цілиться в посилання, які починаються з `https://`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">HTTPS link</a>
<a href="http://www.freecodecamp.org">HTTP link</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

- **Вибір значень, що закінчуються на певний суфікс**: цілиться в посилання, які закінчуються на `.jpg`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="photo.jpg">Image link</a>
<a href="index.html">HTML link</a>
```

```css
a[href$=".jpg"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

- **Вибір значень, що містять підрядок у будь-якому місці**: цілиться в посилання, які містять `https` у значенні.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Secure link</a>
<a href="page.html">Local link</a>
```

```css
a[href*="https"] {
  color: teal;
}
```

:::

- **Підсумок**: ці шаблони полегшують послідовне стилізування посилань на основі їхніх атрибутів і значень.

## Цільові елементи з атрибутами `lang` і `data-lang`

- **Атрибут `lang`**: цей атрибут використовується в HTML для вказівки мови вмісту всередині елемента. Ви можете захотіти стилізувати елементи по-різному залежно від мови, якою вони написані, особливо на багатомовному сайті.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">English text</p>
<p lang="fr">French text</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

- **Атрибут `data-lang`**: кастомні атрибути даних, як-от атрибут `data-lang`, часто використовуються для зберігання додаткової інформації в елементах, наприклад, для вказівки мови, що використовується в певному розділі тексту. Ось як можна стилізувати елементи на основі атрибута `data-lang`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div data-lang="fr">French content</div>
<div data-lang="en">English content</div>
```

```css
div[data-lang="fr"] {
  color: blue;
}
```

:::

## Робота з селектором атрибутів, елементами впорядкованого списку та атрибутом `type`

- **Атрибут `type`**: при роботі з впорядкованими списками в HTML атрибут `type` дозволяє вказати стиль нумерації, наприклад числовий, алфавітний або римські цифри.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Alpha</li>
  <li>Beta</li>
</ol>

<ol type="i">
  <li>One</li>
  <li>Two</li>
</ol>
```

```css
/*Example targeting uppercase alphabetical numbering*/
ol[type="A"] {
  color: purple;
  font-weight: bold;
}

/*Example targeting lowercase Roman numerals*/
ol[type="i"] {
  color: green;
}
```
