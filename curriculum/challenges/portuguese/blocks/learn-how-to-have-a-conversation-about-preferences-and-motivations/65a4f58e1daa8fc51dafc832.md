# --description--

`Which` é usado ao perguntar sobre um ou mais itens de um conjunto conhecido de opções.Por exemplo, `Which computer do you prefer?` pede uma escolha entre computadores conhecidos. 

`What` é mais aberto, usado quando qualquer resposta é possível, como `What do you like to do?`

`Absolute` significa completo ou total.É usado para enfatizar que algo é tanto quanto pode ser.Por exemplo, `My absolute favorite game is chess,` significa que xadrez é o favorito absoluto, sem dúvida.
