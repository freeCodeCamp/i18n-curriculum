# --description--

令 $s_k$ 為從 0 到 $k$ 以二進位表示時 1 的數量。

例如，將 0 到 5 寫成二進位，我們有 0、1、10、11、100、101。有七個 1，所以 $s_5 = 7$。

數列 $S = \\{s_k : k ≥ 0\\}$ 從 $\\{0, 1, 2, 4, 5, 7, 9, 12, \ldots\\}$ 開始。

一個遊戲由兩個玩家進行。在遊戲開始前，選擇一個數字 $n$。計數器 $c$ 從 0 開始。在每回合中，玩家從 1 到 $n$（含）中選擇一個數字，並將 $c$ 增加該數字。$c$ 的結果值必須是集合 $S$ 的成員。如果沒有更多有效的移動，玩家即輸掉遊戲。

例如，當 $n = 5$ 且起始為 $c = 0$：

- 玩家 1 選擇 4，因此 $c$ 變成 $0 + 4 = 4$。
- 玩家 2 選擇 5，因此 $c$ 變成 $4 + 5 = 9$。
- 玩家 1 選擇 3，所以 $c$ 變成 $9 + 3 = 12$。
- etc.

請注意，$c$ 必須始終屬於 $S$，且每個玩家最多可以將 $c$ 增加 $n$。

設 $M(n)$ 為第一位玩家在她的第一回合中可以選擇的最高數字以確保獲勝，若沒有此類移動則 $M(n) = 0$。例如，$M(2) = 2$、$M(7) = 1$ 和 $M(20) = 4$。

可以驗證 $\sum M{(n)}^3 = 8150$ 對於 $1 ≤ n ≤ 20$。

求 $\sum M{(n)}^3$，其中 $1 ≤ n ≤ 1000$。
