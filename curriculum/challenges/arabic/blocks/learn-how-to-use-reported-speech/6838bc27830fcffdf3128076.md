# --explanation--

`Exactly` ظرف يُستخدم لإظهار الموافقة الكاملة أو الصحة مع ما قاله شخص آخر. أما `Exact` فهو صفة تعني دقيق أو صحيح في التفاصيل. على سبيل المثال:

- `You are exactly right about the cause of the problem.` – هذا يُظهر موافقة كاملة. يعبر عن مدى صحة (صفة) الشخص.

- `We need the exact measurements to build the new server rack.` – هذا يعني قياسات دقيقة جدًا. يحدد اسمًا بدلًا من فعل أو صفة.
