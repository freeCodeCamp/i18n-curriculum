# --description--

La SEO, o ottimizzazione per i motori di ricerca, è una pratica che ottimizza le pagine web affinché diventino più visibili e si posizionino più in alto nei motori di ricerca. Un modo per migliorare la SEO del tuo sito è fornire una breve descrizione per la pagina web usando l'elemento `meta`. Ecco un esempio di utilizzo dell'elemento meta per impostare la descrizione di una pagina per un sito di giardinaggio:

```html
<meta
  name="description"
  content="Discover expert tips and techniques for gardening in small spaces, choosing the right plants, and maintaining a thriving garden."
/>
```

Impostando l'attributo `name` su `description`, si assicura che browser, motori di ricerca e altri strumenti web interpretino correttamente questi metadati. L'attributo `content` è dove inserirai la tua descrizione. Si consiglia di mantenere le descrizioni brevi e concise, perché i motori di ricerca spesso troncano la descrizione in base al layout della pagina dei risultati.

Simile ad altri tipi di elementi `meta`, la descrizione `meta` non sarà visibile direttamente sulla pagina web. Un luogo dove si può trovare la descrizione della pagina è nello snippet della pagina dei risultati del motore di ricerca. Ecco alcuni esempi di snippet dei risultati per il subreddit e i repository GitHub di freeCodeCamp:

```sh
r\FreeCodeCamp: This is the official subreddit for the freeCodeCamp.org community. Learn to
code for free together with millions of other people...
```

```sh
Our full-stack web development and machine learning curriculum is completely free and self-
paced. We have thousands of interactive coding challenges to help you...
```

Negli esempi, ciascuna descrizione della pagina si trova appena sotto i collegamenti del sito. In pochi secondi, gli utenti possono farsi un’idea generale di cosa tratta la pagina e decidere di cliccare sui collegamenti per avere più informazioni.

Anche se le descrizioni `meta` non influenzano direttamente il posizionamento di un sito nei motori di ricerca, avere una descrizione efficace potrebbe portare più traffico al tuo sito web.
