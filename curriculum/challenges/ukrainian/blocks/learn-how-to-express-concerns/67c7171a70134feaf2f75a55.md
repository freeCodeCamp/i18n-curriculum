# --explanation--

У цьому реченні Марія використовує `they` для позначення **членів команди**. Раніше в розмові вона згадувала `the team` і вживала дієслово в однині, коли сказала `I need to understand how the team feels about the new project deadline`. Тепер вона говорить про людей індивідуально, тому переходить на `they`. Ще один приклад:

- `The committee decides on the rules.` — Зверніть увагу на використання дієслова в однині для позначення групи в цілому.

- `They discuss the rules before making a decision.` — Відбувається перехід, щоб підкреслити окремих осіб у групі (членів комітету), тому вживається множина.
