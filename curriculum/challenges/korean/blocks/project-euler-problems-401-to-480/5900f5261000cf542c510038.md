# --description--

정수 $M$에 대해, $R(M)$을 다음 조건을 모두 만족하는 정수 쌍 $p$와 $q$에 대해 $\frac{1}{p·q}$의 합으로 정의합니다:

- $1 ≤ p &lt; q ≤ M$
- $p + q ≥ M$
- $p$와 $q$는 서로소입니다.

또한 $S(N)$을 $2 ≤ i ≤ N$인 모든 $i$에 대해 $R(i)$의 합으로 정의합니다.

$S(2) = R(2) = \frac{1}{2}$, $S(10) ≈ 6.9147$, $S(100) ≈ 58.2962$임을 확인할 수 있습니다.

$S({10}^7)$을 구하세요. 소수점 넷째 자리까지 반올림하여 답을 제출하세요.
