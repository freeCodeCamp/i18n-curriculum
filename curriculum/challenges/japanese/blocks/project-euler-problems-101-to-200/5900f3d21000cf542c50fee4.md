# --description--

シーケンスの最初の k 項が与えられた場合、次の項の値を確実に言い当てることは不可能です。なぜなら、そのシーケンスをモデル化できる多項式関数は無限に存在するからです。

例として、立方数のシーケンスを考えましょう。これは母関数 $u_n = n^3$ によって定義され、1, 8, 27, 64, 125, 216, \ldots となります。

もしこのシーケンスの最初の2項だけが与えられたとしたら、「単純なものが最良」という原則に基づき、線形の関係を仮定して次の項を15（公差7）と予測するべきです。最初の3項が与えられた場合でも、同じ単純さの原則により、2次関係を仮定することになります。

ここで、$OP(k, n)$ をシーケンスの最初の k 項に対する最適多項式母関数の $n$ 番目の項と定義します。$OP(k, n)$ は $n ≤ k$ の範囲でシーケンスの項を正確に生成し、最初の誤った項（FIT）は $OP(k, k+1)$ になる可能性があります。この場合、それを悪い OP（BOP）と呼びます。

基準として、もしシーケンスの最初の1項だけが与えられた場合、定数関数を仮定するのが最も合理的です。つまり、$n ≥ 2$ のとき $OP(1, n) = u_1$ となります。

したがって、立方数シーケンスに対する以下の OP が得られます：

$$\begin{array}{ll}
  OP(1, n) = 1          & 1, {\color{red}1}, 1, 1, \ldots     \\\\
  OP(2, n) = 7n−6       & 1, 8, {\color{red}{15}}, \ldots     \\\\
  OP(3, n) = 6n^2−11n+6 & 1, 8, 27, {\color{red}{58}}, \ldots \\\\
  OP(4, n) = n^3        & 1, 8, 27, 64, 125, \ldots
\end{array}$$

明らかに、$k ≥ 4$ では BOP は存在しません。BOP によって生成される FIT の合計（上記の $\color{red}{赤}$ で示されたもの）を考えると、1 + 15 + 58 = 74 となります。次の10次多項式母関数を考えてください：

$$u_n = 1 − n + n^2 − n^3 + n^4 − n^5 + n^6 − n^7 + n^8 − n^9 + n^{10}$$

BOP の FIT の合計を求めてください。
