# --description--

幾乎所有 HTML 頁面上的元素都是包裹在開啟和關閉 HTML 頁籤、標記中的內容片段。

開啟標記告訴瀏覽器這是 HTML 元素的開始。它們由夾在尖括號 `<>` 中的關鍵字組成。例如，開啟段落標記看起來像這樣：`<p>`。

結束標記告訴瀏覽器元素在哪裡結束。它們幾乎和開始標記一樣；唯一的差別是在關鍵字前面有一個斜線。例如，結束段落標記看起來像這樣：`</p>`。
