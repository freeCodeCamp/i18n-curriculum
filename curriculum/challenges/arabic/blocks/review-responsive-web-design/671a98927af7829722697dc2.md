# --interactive--

## تصميم الويب المتجاوب

- **التعريف**: المبدأ الأساسي في التصميم المتجاوب هو القدرة على التكيف – قدرة الموقع على تعديل تخطيطه ومحتواه بناءً على حجم الشاشة وإمكانيات الجهاز الذي يُعرض عليه.
- **الشبكات السائلة**: تستخدم وحدات نسبية مثل النسب المئوية بدلًا من الوحدات الثابتة مثل البكسل، مما يسمح للمحتوى بإعادة الحجم وإعادة التدفق حسب حجم الشاشة.
- **الصور المرنة**: تُضبط لتُعاد تحجيمها داخل العناصر الحاوية لها، مما يضمن عدم تجاوزها حدود الحاويات على الشاشات الصغيرة.

## استعلامات الوسائط

- **التعريف**: تتيح للمطوّرين تطبيق أنماط مختلفة بناءً على خصائص الجهاز، وخصوصًا عرض إطار العرض.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="mq-example">The background of this paragraph will change when the viewport is at least 768px wide.</p>
```

```css
.mq-example {
  padding: 10px;
  border: 1px solid #ccc;
  background-color: lightyellow;
}

@media screen and (min-width: 768px) {
  /* Styles for screens at least 768px wide */
  .mq-example {
    background-color: lightblue;
  }
}
```

:::

- **نوع الوسائط `all`**: مناسب لجميع الأجهزة. هذا هو الافتراضي إذا لم يُحدد نوع الوسائط.
- **أنواع الوسائط `print`**: مخصصة للمواد المطبوعة والوثائق المعروضة على الشاشة في وضع معاينة الطباعة.
- **أنواع الوسائط `screen`**: مخصصة أساسًا للشاشات.
- **`aspect-ratio`**: يصف النسبة بين عرض وارتفاع إطار العرض.

```css
@media screen and (aspect-ratio: 16/9) {
  /* Styles for screens with a 16:9 aspect ratio */
}
```

- **`orientation`**: يُستخدم لتحديد ما إذا كان الجهاز في وضع العرض الأفقي أو الرأسي.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="orientation-example">This paragraph's background will change in landscape orientation.</p>
```

```css
.orientation-example {
  padding: 10px;
  border: 1px solid #ccc;
  background-color: lightyellow;
}

@media screen and (orientation: landscape) {
  /* Styles for landscape orientation */
  .orientation-example {
    background-color: lightgreen;
  }
}
```

:::

- **`resolution`**: يُستخدم لوصف دقة الوضوح لجهاز الإخراج بوحدات النقاط لكل إنش (dpi) أو النقاط لكل سنتيمتر (dpcm).

```css
@media screen and (min-resolution: 300dpi) {
  /* Styles for high-resolution screens */
}
```

- **`hover`**: يُستخدم لاختبار ما إذا كانت آلية الإدخال الأساسية يمكنها التحويم فوق العناصر.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button>Hover over me</button>
```

```css
button {
  padding: 10px 20px;
  font-size: 1rem;
  background-color: lightgray;
  border: 1px solid #999;
  cursor: pointer;
}

@media (hover: hover) {
  /* Styles for devices that support hover */
  button:hover {
    background-color: yellow;
  }
}
```

:::

- **`prefers-color-scheme`**: يُستخدم لاكتشاف ما إذا طلب المستخدم سمة ألوان فاتحة أو داكنة.
- **استعلامات الوسائط والمعاملات المنطقية**: يُستخدم المعامل `and` لدمج ميزات وسائط متعددة، بينما يمكن استخدام `not` و`only` لنفي أو عزل استعلامات الوسائط.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="logical-example">This paragraph's background will change when the screen is at least 768px wide AND in landscape orientation.</p>
```

```css
.logical-example {
  padding: 10px;
  border: 1px solid #ccc;
  background-color: lightyellow;
}

@media screen and (min-width: 768px) and (orientation: landscape) {
  /* Styles for landscape screens at least 768px wide */
  .logical-example {
    background-color: lightcoral;
  }
}
```

:::

## نقاط توقف الوسائط الشائعة

- **التعريف**: نقاط توقف الوسائط هي نقاط محددة في تصميم الموقع حيث يتغير التخطيط والمحتوى ليتناسب مع أحجام الشاشات المختلفة. هناك نقاط توقف عامة يمكنك استخدامها لاستهداف الهواتف والأجهزة اللوحية وشاشات سطح المكتب. لكن من غير الحكمة محاولة استهداف كل حجم شاشة ممكن لأجهزة مختلفة.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="breakpoint-example">This text will change size when the viewport width is at least 768px.</p>
```

```css
.breakpoint-example {
  font-size: 1rem;
  padding: 10px;
  border: 1px solid #ccc;
}

/* Styles for screens wider than 768px */
@media screen and (min-width: 768px) {
  .breakpoint-example {
    font-size: 1.125rem;
    background-color: lightblue;
  }
}
```

:::

- **الأجهزة الصغيرة (الهواتف الذكية)**: حتى 640 بكسل
- **الأجهزة المتوسطة (الأجهزة اللوحية)**: من 641 إلى 1024 بكسل
- **الأجهزة الكبيرة (أجهزة سطح المكتب)**: 1025 بكسل فما فوق

## نهج الجوال أولًا

- **التعريف**: نهج الجوال أولًا هو فلسفة تصميم واستراتيجية تطوير في تصميم الويب المتجاوب تعطي الأولوية لإنشاء مواقع للأجهزة المحمولة قبل تصميمها للشاشات الأكبر.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <p>The width of this container changes based on the viewport size.</p>
</div>
```

```css
/* Base styles for mobile */
.container {
  width: 100%;
  padding: 10px;
  background-color: lightblue;
}

/* Styles for larger screens */
@media screen and (min-width: 768px) {
  .container {
    width: 750px;
    margin: 0 auto;
    padding: 20px;
  }
}

@media screen and (min-width: 1024px) {
  .container {
    width: 960px;
  }
}
```
