# --description--

Nachdem Sie den doctype deklariert haben, müssen Sie ein `<html>`-Element bereitstellen. Dies ist das sogenannte Wurzelelement des Dokuments, was bedeutet, dass jedes andere Element im Dokument ein Nachfahre davon sein wird.

Das wird später noch wichtiger, wenn Sie lernen, wie man HTML mit JavaScript manipuliert. Für den Moment sollten Sie nur wissen, dass das `html`-Element in jedem HTML-Dokument enthalten sein sollte.

Kehren wir zur `index.html`-Datei zurück und fügen das `<html>`-Element hinzu, indem wir seine öffnenden und schließenden Tags wie folgt eintippen:

```html
<!DOCTYPE html>
<html lang="en">
</html>
```

## Was ist das Attribut lang?
`lang` gibt die Sprache des Textinhalts in diesem Element an. Dieses Attribut wird hauptsächlich verwendet, um die Barrierefreiheit der Webseite zu verbessern. Es ermöglicht unterstützenden Technologien, zum Beispiel Screenreadern, sich an die Sprache anzupassen und die korrekte Aussprache aufzurufen.
