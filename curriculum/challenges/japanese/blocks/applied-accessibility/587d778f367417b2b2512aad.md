# --description--

色覚障害にはさまざまな種類があります。特定の波長の光に対する感度が低下するものから、色をまったく認識できないものまであります。最も一般的なタイプは、緑色を識別する感度が低下するものです。

たとえば、似たような緑色がコンテンツの前景色と背景色に使われている場合、色覚障害のあるユーザーはそれらを区別できないかもしれません。近い色はカラーホイール上で隣り合う色と考えられ、重要な情報を伝える際にはそのような組み合わせは避けるべきです。

**注意：** 一部のオンラインカラーピッカーには、色覚障害の種類ごとに色がどのように見えるかを視覚的にシミュレーションする機能があります。これらはオンラインのコントラスト比チェック計算機と合わせて使うと非常に役立ちます。

# --instructions--

Camper Catは重要なボタンのさまざまなスタイルをテストしていますが、黄色の (`#FFFF33`) `background-color` と緑色の (`#33FF33`) テキスト `color` はカラーホイール上で隣り合う色相であり、色覚障害のある一部のユーザーにはほとんど区別できません。（明度も似ているためコントラスト比のチェックにも合格しません。）テキスト `color` を濃い青色の (`#003366`) に変更して、両方の問題を解決してください。

# --hints--

コードは `color` のテキスト `button` を濃い青色に変更する必要があります。

```js
const button = document.querySelector('button');
const buttonColor = window.getComputedStyle(button).color; 
assert.equal(buttonColor, 'rgb(0, 51, 102)');
```
