# --interactive--

Le proprietà `grid-column` e `grid-row` ti permettono di specificare la posizione orizzontale e verticale degli elementi griglia all’interno di un layout a griglia.

In altre parole, entrambe ti consentono di controllare dove un elemento griglia inizia e finisce facendo riferimento alle linee della griglia. Queste linee della griglia sono i confini che separano le righe e le colonne che hai già definito usando le proprietà `grid-template-rows` e `grid-template-columns`.

Diamo un’occhiata alle sintassi di `grid-row` e `grid-column`, e anche a un esempio pratico.

Ecco la sintassi della proprietà `grid-row`:

```css
grid-row: <start-line> / <end-line>;
```

E di `grid-column`:

```css
grid-column: <start-line> / <end-line>;
```

`<start-line>` è la linea della griglia dove l’elemento inizia e `<end-line>` è la linea della griglia dove l’elemento finisce. Entrambe sono indicizzate a partire da 1, cioè inizi a contarle da 1, non da 0.

Ricorda che le linee della griglia per le righe si generano in base al numero di righe specificato nella proprietà `grid-template-rows`. Lo stesso vale per le colonne con la proprietà `grid-template-columns`.

Per l’esempio pratico, iniziamo con le linee generate dal browser quando usi sia la proprietà `grid-template-rows` che `grid-template-columns`.

Ecco una griglia di esempio con 4 colonne e 3 righe:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}
```

:::

Ispezionando il contenitore griglia (la classe `grid`) si vede che ogni riga e colonna è delimitata da due linee – una linea di inizio all’inizio della riga o colonna, e una linea di fine alla fine della riga o colonna.

Puoi prendere di mira queste linee per iniziare a usare le proprietà `grid-row` e `grid-column` per determinare dove posizionare un elemento.

Ecco come puoi fare in modo che il primo elemento griglia occupi le prime due colonne:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
}
```

:::

Con questo, stai dicendo che il primo elemento griglia dovrebbe iniziare alla colonna 1 e finire appena prima della colonna 3.

Il primo elemento ora occuperà due colonne, e il quarto elemento viene spostato alla seconda riga. Fantastico!


Puoi anche usare la parola chiave `span` per dire all’elemento griglia su quali righe e colonne estendersi. Per esempio, `1 / 3` è uguale a `1 / span 2`:

:::interactive_editor

```html

  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / span 2;
  grid-row: 1 / span 2;
}
```

:::

Puoi continuare ad applicare questa tecnica a qualsiasi elemento della griglia e posizionarli dove vuoi.
