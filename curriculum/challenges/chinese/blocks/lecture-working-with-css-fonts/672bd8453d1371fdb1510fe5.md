# --interactive--

外部字体是指没有直接包含在项目文件中的字体文件。 它们通常托管在单独的服务器上。 服务器是通过网络向其他计算机提供数据或服务的计算机。  You will learn more about servers in future lessons. 外部字体为你提供了更大的灵活性，因为你可以使用用户设备上可能没有安装的自定义字体。

Google Fonts 和 Font Squirrel 是查找和使用免费字体进行网页开发的常用在线资源。 你可以非常容易地将它们融入到项目中。

让我们从 Google 字体开始。 这是一项 Google 服务，提供一系列字体，其中许多字体是专门为网页开发而设计的。 在 Google Fonts 的用户界面上，你可以看到许多不同的元素。 让我们逐一介绍。 在左侧边栏，你可以找到：

- 用于查看、查找和过滤字体的区域。
- 特别介绍 Noto 字体的区域，这是一套高质量字体，具有不同的字重、宽度和样式，支持超过 1,000 多种语言和 150 多种书写系统进行的交流。
- 可以为你的网络项目查找和下载图标的区域。
- 可以了解更多有关字体及其最佳实践以及常见问题的区域。

要在网站上添加 Google Font，应进入第一部分（字体）。  You can customize the preview text on the left sidebar, where it says "Preview." Just write the text that you would like to see and it will be updated automatically. 这有助于你根据网站上已有的文本预览字体。 你还可以调整字体大小，并根据字体的特性筛选字体。 

要查看字体的更多信息，只需点击该字体，就能看到字体的重要信息，如设计者、预览文本和不同样式，如细体、常规、中等、粗体和黑体。 如果点击顶部的选项，就可以进入字体测试器，查看单个字形，了解有关字体及其许可证的更多信息。

你可以使用输入预览文本字段旁边的蓝色滑块自定义预览文本的字体大小。 准备好将字体添加到项目后，点击顶部的蓝色“获取字体”按钮。 你将进入一个摘要页面，在这里可以看到你当前选择的字体。 你可以同时选择多种字体。

接下来，你必须选择是将字体文件下载为局部文件并添加到你的项目中，还是作为外部字体使用，并在用户进入你的网站时从 Google 的服务器下载它们。如果你想下载字体文件，请点击 "Download all"；如果你更愿意将它们作为 Google 服务器上的外部字体使用，请点击 "Get embed code"。如果你点击 "Get embed code"，你将看到添加这些外部字体到你的项目中应遵循的说明。

对于网页开发项目，你有两种选择。 你可以使用 `link` 元素或 `@import`。 如果你选择 `link` 元素选项，则可以复制并粘贴 HTML 代码段和 CSS 规则，将其添加到项目中。 你应将代码嵌入 HTML 文件的 `head` 元素中，并添加适合你需要的 CSS 规则。

每种字体样式都有一条 CSS 规则。 每条规则都会为自定义字体分配备用字体，以防自定义字体无法正常加载。

如果你选择 `@import` 选项，则需要在 CSS 文件中添加该规则。 这是一个示例：

```css
@import url('https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap');
```

CSS 规则完全相同。

如果你只想包含特定的字体样式，也可以选择从这些代码段中移除不需要的。 HTML 和 CSS 代码段将自动更新。 

这就是 Google Fonts。 它是为网页开发项目查找免费和开源字体的最常用选项之一。

另一个不错的选择是 Font Squirrel，你可以在这里找到并下载你为设计所设想的定制字体。 如果你搜索字体并点击搜索结果，你将看到有关该字体的更多详细信息，包括字体样本、测试工具、字形和 Webfont 工具包。 你还可以看到字体的不同样式和变体，如样本，如极细、细体、中等、粗体和黑体。

确定要在项目中使用的字体后，请转到 "Webfont Kit" 选项卡。 在这里，你可以检查字体的许可协议是否允许你在 `@font-face` CSS 嵌入中使用该字体。 你还可以选择子集和格式。

点击“下载 @font-face 工具包”按钮将下载一个压缩（zip）文件。 解压缩后，你会发现一个“网页字体”文件夹、一个文本文件形式的许可协议，以及一个包含如何使用网络字体的说明的 HTML 文件。 网页字体文件夹将包含项目所需的所有字体文件。 你会发现它们被归类到不同字体样式的特定文件夹中。

如果在浏览器上打开 HTML 文件，就会看到如何在项目中添加字体、如何在样式表中使用 `@font-face` 声明包含字体以及如何在 CSS 规则中使用字体的说明。

这就是如何在项目中添加自定义 Font Squirrel 字体的方法。 你也可以将它们托管在单独的服务器上，作为外部字体使用。 外部字体非常有助于定制网站的整体外观和设计。 它们可以让你的网站真正独一无二。

不过，你也应该记住，使用多种外部字体会增加网站的加载时间，影响网站的整体性能。 你应始终努力在风格和效率之间找到平衡。
