# --description--

Debido a que el elemento `input type="number"` permite caracteres especiales como `.`, `+` y `e`, los usuarios pueden ingresar números flotantes como `2.2`, ecuaciones como `2e+3` o incluso solo `e`, lo cual no deseas permitir.

Una buena forma de verificar y normalizar números en JavaScript es usar la función integrada `parseInt()`, que convierte una cadena en un número entero. `parseInt()` toma al menos un argumento, una cadena que se va a convertir en un número entero, y devuelve ya sea un entero o `NaN`, que significa `Not a Number`.  Por ejemplo:

```js
parseInt(2.2); // 2
parseInt("2e+3"); // 2
parseInt("e") // NaN
```

Agrega un operador OR lógico (`||`) después de la primera condición en tu sentencia `if`. Luego, pasa el valor de `numberInput` a la función `parseInt()` como la segunda condición de tu sentencia `if`.
