# --description--

让我们聊聊科技行业中的职业。

`UX (User Experience)` 是关于使用网站或应用程序等东西的易用性和愉悦感。它侧重于确保一切运行良好且使用起来感觉舒适。  
`UI (User Interface)` 与网站或应用程序上的内容外观及其查找位置有关。它包括你点击的按钮、颜色和版型。它旨在确保一切看起来美观且易于查找。

如果你是 `UX designer`、`UI designer` 或 `UX/UI designer`，你就是负责让网站和应用易于使用且美观的人。
