# --interactive--

JavaScript에서 문자열을 다룰 때, 더 큰 문자열에서 일부 또는 부분 문자열을 추출해야 할 때가 많습니다.

예를 들어, 단어의 일부, 특정 문자 수열, 또는 문장의 일부 조각을 추출하고 싶을 수 있습니다.

JavaScript는 이 작업을 위한 여러 메서드를 제공하며, 그중 가장 자주 사용되는 것 중 하나가 `slice()` 메서드입니다.

`slice()` 메서드는 문자열의 일부를 추출하여 새로운 문자열을 반환하며, 원본 문자열은 변경하지 않습니다. 이 메서드는 시작 인덱스와 선택적인 종료 인덱스 두 개의 매개변수를 받습니다.

기본 구문은 다음과 같습니다:

```js
string.slice(startIndex, endIndex);
```

`startIndex`는 추출이 시작되는 위치입니다. `endIndex`는 추출이 끝나는 위치입니다. 만약 제공하지 않으면, `slice()`는 문자열 끝까지 추출합니다.

문자열의 일부를 추출하는 간단한 예제를 살펴보겠습니다:

:::interactive_editor

```js
let message = "Hello, world!";
let greeting = message.slice(0, 5);

console.log(greeting);  // Hello
```

:::

이 예제에서 `slice(0, 5)`는 인덱스 `0`부터 인덱스 `5` 바로 전까지의 문자를 추출합니다. 결과적으로 단어 `Hello`가 추출됩니다.

두 번째 매개변수를 생략하면, `slice()`는 시작 인덱스부터 문자열 끝까지 모두 추출합니다:

:::interactive_editor

```js
let message = "Hello, world!";
let world = message.slice(7);

console.log(world);  // world!
```

:::

여기서 `slice(7)`는 인덱스 `7`부터 문자열 끝까지 추출하여 `world!`를 결과로 얻습니다.

음수 인덱스도 사용할 수 있습니다. 음수를 사용하면 문자열 끝에서부터 거꾸로 셉니다:

:::interactive_editor

```js
let message = "JavaScript is fun!";
let lastWord = message.slice(-4);

console.log(lastWord);  // fun!
```

:::

이 경우, `slice(-4)`는 문자열의 마지막 네 문자를 추출하여 `fun!`를 얻습니다.

문자열 중간에서 섹션을 추출하고 싶다면, 시작과 종료 인덱스 둘 다 제공하여 정확히 원하는 부분을 제어할 수 있습니다:

:::interactive_editor

```js
let message = "I love JavaScript!";
let language = message.slice(7, 17);

console.log(language);  // JavaScript
```

:::

여기서 `slice(7, 17)`는 인덱스 7에서 시작하여 인덱스 `17` 바로 전까지 부분 문자열을 추출하며, 결과는 단어 `JavaScript`입니다.

`slice()` 메서드는 JavaScript에서 문자열의 일부를 추출하는 강력한 도구입니다.

시작과 끝 인덱스를 지정하면, 메서드는 추출된 부분을 포함하는 새 문자열을 반환합니다.

양수, 음수, 생략된 인덱스 옵션을 통해 원본 문자열을 변경하지 않고 다양한 상황에 맞게 활용할 수 있습니다.
