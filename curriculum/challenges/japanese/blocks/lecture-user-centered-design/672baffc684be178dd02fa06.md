# --interactive--

**注意**: いくつかのインタラクティブな例では、まだ学習していないCSSが使われている場合があります。コードのすべてを理解しようと心配しないでください。これらの例の目的は、デザインコンセプトのプレビューを見せて、仕組みをよりよく理解してもらうことです。プレビューを見るには、インタラクティブエディターを有効にしてください。

カードコンポーネントは、eコマース、ソーシャルメディア、ニュースサイトで非常によく使われます。情報を構造的に表示するのに役立ちます。カードをデザインするときは、ユーザーが伝えたい情報を簡単に理解できるように、ベストプラクティスを理解することが重要です。

カードデザインで最初に考慮すべきはシンプルさです。カードが視覚的にごちゃごちゃしていたり、情報が多すぎたりしてはいけません。例えば、カードデザインが視覚的にごちゃごちゃしていると、ユーザーが一度に処理する情報が多すぎてしまいます。

こちらはごちゃごちゃしたカードデザインの例です：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2><strong><em>Product Name: Ultimate Edition</em></strong></h2>
  <p>
    This is a great product that you will love! It has many features and benefits that will make your life easier and more enjoyable. Whether you're at home or on the go, this product is perfect for all situations. Don’t miss out on this incredible offer, buy now and experience the difference!
  </p>
  <button>Buy Now</button>
  <button>Learn More</button>
  <button>Add to Wishlist</button>
  <p><strong>Only 3 left in stock!</strong></p>
  <p><em>Rated 4.8 stars by 2,391 customers</em></p>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}
.card p {
  font-size: 1em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

カード上の情報が少なく、アイテム間のスペースが十分にあると、ユーザーが情報を処理しやすくなり、ページ上に複数のカードを配置できます。

こちらはシンプルなカードデザインの例です：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2>Product Name</h2>
  <button>Buy Now</button>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 200px;
  text-align: center;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

もう一つ考慮すべきは、ユーザーがカードのどこをクリックできるかです。カードデザインによってはボタンが1つだけあり、ユーザーがどこをクリックできるかが明確です。別のカードデザインではカード全体がクリック可能です。ユーザーがカードのどの部分にマウスを乗せても、カードの色が変わったり影がついたりしてクリック可能であることを示します。どのデザインを選んでも、サイト全体で一貫していて、ユーザーにわかりやすい必要があります。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <a href="#">
    <img src="https://placehold.co/150x150" alt="Example Product Image">
    <h2>Product Name</h2>
    <button>Buy Now</button>
  </a>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
  cursor: pointer;
}

.card:hover {
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
}

.card a {
  text-decoration: none;
  color: inherit;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

もう一つの考慮点は、カードで使うメディアの選択です。高品質のメディアを選ぶことでユーザー体験が大きく向上します。例えば、商品カードに画像や動画を使う場合、品質が高いほどユーザーはその商品に興味を持ちます。しかし、品質が低いメディアを使うと、ユーザーは商品の品質やサービスを信用しないかもしれません。

最後に考慮すべきは色の階層の使い方です。カード上で最も重要な情報が目立つようにする必要があります。コールトゥアクションボタンのような重要な要素には明るい色を使い、カード上のあまり重要でない項目には淡い色を使うことができます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2>Product Name</h2>
  <button>Buy Now</button>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
  text-align: center;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #28a745; 
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

ウェブアプリケーションの開発を続ける中で、カードデザインのさまざまなベストプラクティスを覚えておいてください。これにより、ユーザーにとってより良い体験を作り出せます。
