# --description--

テンプレートエンジンを使う最大の機能の1つは、サーバーからテンプレートファイルに変数を渡して、それをHTMLにレンダリングできることです。

Pugファイルでは、要素のテキスト内で変数名を`#{variable_name}`のように参照したり、等号を使って`p=variable_name`のようにスペースなしで要素に変数の値を代入したりして変数を使えます。

Pugはインデントやタブでネストされた要素を表現し、美しいサイトを作るためのコード量を減らすことに特化しています。

例えば、次のPugコードを見てください：

```pug
head
  script(type='text/javascript').
    if (foo) bar(1 + 5);
body
  if youAreUsingPug
      p You are amazing
    else
      p Get on it!
```

上記は次のHTMLを生成します：

```html
<head>
  <script type="text/javascript">
    if (foo) bar(1 + 5);
  </script>
</head>
<body>
  <p>You are amazing</p>
</body>
```

あなたのプロジェクトに含まれる`index.pug`ファイルは、変数`title`と`message`を使っています。

サーバーからこれらの変数とその値をオブジェクトとして`res.render`の呼び出しの第2引数に渡してください。`title`には`Hello`を、`message`には`Please log in`を代入してください。

次のようになります：

```javascript
res.render('index', { title: 'Hello', message: 'Please log in' });
```

ページを更新すると、`index.pug`ファイルに記述された通り、正しい場所に値がレンダリングされているはずです！
