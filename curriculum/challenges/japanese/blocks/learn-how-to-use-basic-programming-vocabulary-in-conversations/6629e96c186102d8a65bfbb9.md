# --description--

`There is` と `there are` は、何かの存在や有無を示すために使われます。`There is` は単数形または不可算名詞と一緒に使われ、`there are` は複数形の名詞と一緒に使われます。

例：

- `There is a book on the table.`（単数名詞）

- `There is some water in the glass.`（不可算名詞）

- `There are three apples in the basket.`（複数名詞）
