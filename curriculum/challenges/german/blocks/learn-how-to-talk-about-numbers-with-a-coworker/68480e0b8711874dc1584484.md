# --explanation--

`User Interface (UI)` ist das, was der Benutzer sieht (Bildschirme, Buttons, Farben).

`User Experience (UX)` ist, wie sich der Benutzer fühlt (Flüssigkeit, Klarheit, Benutzerfreundlichkeit).

Gute Produkte legen Wert auf beides: klare visuelle Gestaltung und ein zufriedenstellendes Erlebnis.
