# --description--

$S(A)$ をサイズ n の集合 A の要素の和と表します。任意の2つの空でない互いに素な部分集合 B と C に対して、次の性質が成り立つ場合、その集合を特別な和集合と呼びます。

1. $S(B) ≠ S(C)$；つまり、部分集合の和は等しくなりません。
2. B の要素数が C より多い場合、$S(B) > S(C)$ となります。

この問題では、与えられた集合が n 個の厳密に増加する要素を含み、すでに2番目のルールを満たしていると仮定します。

驚くべきことに、n = 4 の集合から得られる25通りの部分集合ペアのうち、和の等しさ（1番目のルール）をテストする必要があるのはわずか1組だけです。同様に、n = 7 の場合は966通りのうち70組だけがテスト対象となります。

n = 12 の場合、得られる261625通りの部分集合ペアのうち、いくつが和の等しさをテストする必要があるでしょうか？

**注意：** この問題は問題103および問題105に関連しています。
