# --interactive--

JavaScriptで文字列を扱うとき、見出しのためにすべての文字を大文字に変換したり、統一感を出すために小文字に変換したりと、文字の大文字・小文字を調整する必要がある場面が多くあります。

幸いなことに、JavaScriptには組み込みの2つのメソッド、`toUpperCase()` と `toLowerCase()` があり、これを簡単に行えます。

`toUpperCase()` メソッドはすべての文字を大文字に変換し、大文字の文字列を新しく返します。これはテキストを強調したり、文字列の形式を統一したりするときに便利です。

例を見てみましょう：

:::interactive_editor

```js
let greeting = "Hello, World!";
let uppercaseGreeting = greeting.toUpperCase();
console.log(uppercaseGreeting);  // "HELLO, WORLD!"
```

:::

このコードでは、`toUpperCase()` メソッドが文字列全体を大文字に変換しています。

元の文字列は変更されません。なぜなら `toUpperCase()` は元の文字列を変更せず、新しい文字列を返すからです。

一方、`toLowerCase()` メソッドは文字列内のすべての文字を小文字に変換します。これはユーザーの入力を標準化したり、大文字・小文字を区別しない比較を行うときに役立ちます。

例を見てみましょう：

:::interactive_editor

```js
let shout = "I AM LEARNING JAVASCRIPT!";
let lowercaseShout = shout.toLowerCase();
console.log(lowercaseShout);  // "i am learning javascript!"
```

:::

`toLowerCase()` メソッドはすべての文字を小文字に変換し、文字列を柔らかくしつつ、元の文字列は変更しません。

まとめると、JavaScriptの `toUpperCase()` と `toLowerCase()` メソッドは、文字列をすべて大文字または小文字に変換する強力なツールです。

これらのメソッドは、テキスト入力の標準化、大文字・小文字を区別しない比較、デザインの一貫性を保つのに特に役立ちます。

これらのシンプルで効果的なメソッドを使えば、テキスト操作をより制御しやすく、予測可能に行えます。
