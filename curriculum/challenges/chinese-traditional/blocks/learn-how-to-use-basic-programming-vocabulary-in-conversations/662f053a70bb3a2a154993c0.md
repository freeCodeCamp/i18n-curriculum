# --description--

`extension` 是一種軟體組件，能為較大型的軟體應用程式添加特定的能力。`Extensions` 可以提供新的功能或強化現有的特性。

例如，瀏覽器 `extension` 可以為你的網頁瀏覽器添加特殊功能，例如封鎖廣告或管理密碼。
