## --text--

Quale frase è grammaticalmente corretta?

## --answers--

`If we lose critical data, it do set us back weeks.`

### --feedback--

La parola `do` è usata in modo errato al posto di `could` in questa frase.

---

`If we lose critical data, it could set us back weeks.`

---

`If we lose critical data, it could sets us back weeks.`

### --feedback--

Il verbo `sets` non è nella forma corretta per questa frase.

---

`If we losing critical data, it could set us back weeks.`

### --feedback--

La parola `losing` è usata in modo errato al posto di `lose` in questa frase.

# --explanation--

La frase corretta usa una struttura `First Conditional`, che si forma con:

`if` + `Present Simple`: `If we lose critical data`.

Verbo modale (`could`) + verbo base: `it could set us back weeks`.


La `First Conditional` si usa per descrivere un possibile evento futuro e il suo risultato.
