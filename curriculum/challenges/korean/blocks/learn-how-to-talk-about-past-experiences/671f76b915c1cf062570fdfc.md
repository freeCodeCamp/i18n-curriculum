# --explanation--

AI는 `Artificial Intelligence`의 약자로, 학습, 추론, 의사결정과 같이 일반적으로 인간의 지능을 필요로 하는 작업을 수행할 수 있는 시스템을 만드는 데 중점을 둔 컴퓨터 과학 분야입니다. 예를 들어:

`AI is being used to improve natural language processing in chatbots.` - 이는 인공지능이 챗봇이 사람과 더 효율적으로 소통하도록 돕는 데 사용된다는 뜻입니다.

`AI workshop`는 참가자가 AI 개념, 도구 또는 애플리케이션에 대해 배우는 세션이나 이벤트입니다. 이러한 워크숍은 참석자가 AI 기술을 이해하고 구현할 수 있도록 실습 활동을 포함하는 경우가 많습니다. 예를 들어:

`The AI workshop covered topics like machine learning and neural networks.` - 이는 AI 워크숍에서 머신러닝과 신경망을 가르쳤다는 뜻입니다.

이 대화에서 마리아는 어제 `AI workshop`에 참석했다고 확인합니다.
