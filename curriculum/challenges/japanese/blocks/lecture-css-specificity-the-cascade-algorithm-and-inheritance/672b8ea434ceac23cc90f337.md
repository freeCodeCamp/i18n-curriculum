# --interactive--

IDセレクターはCSSの中でも最も強力なセレクターの一つで、開発者がユニークな識別子を持つ特定の要素にスタイルを適用できます。

これにより、個別のスタイリングが必要な要素を効果的にターゲットにできます。

IDセレクターはハッシュ記号（`#`）に続けてID名を指定して定義します。HTMLドキュメント内で一意である必要があり、同じIDを持つ要素が複数存在してはいけません。

以下はIDセレクターを使った例です：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="unique">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#unique {
  color: purple;
}
```

:::

この例では、IDが`unique`の要素のテキスト色が`purple`に設定されます。

IDセレクターは非常に高い特異性を持ち、型セレクターやクラスセレクターよりも高く、インラインスタイルよりは低いです。IDセレクターの特異性の値は`(0, 1, 0, 0)`です。

つまり、IDセレクターはクラスセレクターや型セレクターをオーバーライドできますが、インラインスタイルにはオーバーライドされます。
