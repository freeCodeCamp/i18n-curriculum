# --interactive--

خاصية `filter` في CSS هي أداة قوية تتيح لك تطبيق تأثيرات رسومية على العناصر في صفحة الويب. تفيد بشكل خاص في تعديل العرض البصري للصور، والخلفيات، وحتى النصوص دون تعديل الأصل.

يمكن استخدام خاصية `filter` لإنشاء تأثيرات متنوعة مثل التمويه، وتحويل الألوان، وتعديل التباين. دعنا نناقش كيف تعمل خاصية الفلتر ونستعرض بعض الأمثلة الشائعة. الصيغة الأساسية لخاصية `filter` بسيطة:

```css
selector {
  filter: function(amount);
}
```

هنا، تمثل `function` التأثير الفلتر المحدد الذي تريد تطبيقه، و`amount` عادةً قيمة تحدد شدة التأثير. الآن دعنا نلقي نظرة على بعض دوال الفلتر الشائعة واستخداماتها.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: blur(2px);
}
```

:::

تطبق دالة `blur` تمويه غاوسي على العنصر، وتُحدد القيمة بالبكسل وتمثل نصف قطر التمويه. ستطبق هذه القاعدة في CSS تمويهًا بمقدار 2 بكسل على جميع الصور في الصفحة. يمكن أن يكون تأثير `blur` مفيدًا لإنشاء عمق في التصميم أو لإخفاء أجزاء من الصورة.

تُعدل دالة `brightness` سطوع العنصر. قيمة `0%` تجعل العنصر أسود تمامًا، بينما القيم التي تزيد عن `100%` تزيد السطوع.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: brightness(150%);
}
```

:::

تزيد هذه القاعدة في CSS سطوع عنصر الصورة بنسبة `50%`. يمكن استخدام تعديلات السطوع لجعل الصور تبرز أو لإنشاء تأثير باهت.

تحول دالة `grayscale` العنصر إلى تدرج رمادي. تُحدد القيمة كنسبة مئوية، حيث `100%` تعني تدرج رمادي كامل و`0%` تترك الصورة دون تغيير.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: grayscale(100%);
}
```

:::

ستحول هذه القاعدة عنصر الصورة إلى تدرج رمادي كامل. يمكن استخدام `grayscale` لإنشاء مظهر قديم أو لتقليل التركيز على عناصر معينة في الصفحة.

تطبق دالة `sepia` نغمة سيبيا على العنصر. مثل التدرج الرمادي، تستخدم قيمة مئوية:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: sepia(80%);
}
```

:::

تطبق هذه القاعدة تأثير سيبيا بنسبة `80%` على عنصر الصورة. تأثير السيبيا رائع لإنشاء مظهر قديم أو كلاسيكي.

تطبق دالة `hue-rotate` تدويرًا للون على العنصر. تُحدد القيمة بالدرجات وتمثل دورانًا حول دائرة الألوان.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: hue-rotate(90deg);
}
```

:::

تدور هذه القاعدة لون عنصر الصورة بمقدار `90` درجة. يمكن استخدام تدوير اللون لإنشاء تأثيرات نفسية أو لتعديل نظام الألوان العام للصورة.

واحدة من أقوى ميزات خاصية `filter` هي القدرة على دمج تأثيرات متعددة. يمكنك تطبيق عدة فلاتر على نفس العنصر بفصلها بمسافات:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {  
  filter: contrast(150%) brightness(110%) sepia(30%);  
}
```

:::

تطبق هذه القاعدة زيادة في التباين، وزيادة طفيفة في السطوع، وتأثير سيبيا خفيف على عنصر الصورة.

بدمج الفلاتر، يمكنك إنشاء تأثيرات بصرية معقدة وفريدة تناسب احتياجات تصميمك. خاصية فلتر CSS أداة متعددة الاستخدامات تتيح التلاعب البصري الإبداعي لعناصر الويب.

بينما غطينا بعض دوال الفلتر الشائعة، هناك أخرى متاحة مثل `contrast` و`invert` و`saturate`. كما هو الحال مع أي ميزة قوية، من المهم الحذر في استخدام الفلاتر لتعزيز تصميمك دون إرباك المستخدمين أو التأثير على إمكانية الوصول.
