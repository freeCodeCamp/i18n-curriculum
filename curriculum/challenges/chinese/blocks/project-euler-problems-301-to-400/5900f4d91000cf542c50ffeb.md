# --description--

三次贝塞尔曲线由四个点定义：$P_0$、$P_1$、$P_2$ 和 $P_3$。

曲线构造如下：

<img alt="construction of Bézier curve" src="https://cdn.freecodecamp.org/curriculum/project-euler/bzier-curves.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

在线段 $P_0P_1$、$P_1P_2$ 和 $P_2P_3$ 上绘制点 $Q_0$、$Q_1$ 和 $Q_2$，使得 $\frac{P_0Q_0}{P_0P_1} = \frac{P_1Q_1}{P_1P_2} = \frac{P_2Q_2}{P_2P_3} = t$，其中 $t$ 在 [0,1] 范围内。

在线段 $Q_0Q_1$ 和 $Q_1Q_2$ 上绘制点 $R_0$ 和 $R_1$，使得 $\frac{Q_0R_0}{Q_0Q_1} = \frac{Q_1R_1}{Q_1Q_2} = t$，其中 $t$ 的值相同。

在线段 $R_0R_1$ 上绘制点 $B$，使得 $\frac{R_0B}{R_0R_1} = t$，其中 $t$ 的值相同。

由点 $P_0$、$P_1$、$P_2$、$P_3$ 定义的贝塞尔曲线是当 $Q_0$ 在线段 $P_0P_1$ 上取所有可能位置时 $B$ 的轨迹。（请注意，所有点的 $t$ 值相同。）

从构造中可以看出，Bézier 曲线将在 $P_0$ 处与线段 $P_0P_1$ 相切，在 $P_3$ 处与线段 $P_2P_3$ 相切。

一个三次贝塞尔曲线，$P_0 = (1, 0)$，$P_1 = (1, v)$，$P_2 = (v, 1)$ 和 $P_3 = (0, 1)$，用于近似一个四分之一圆。选择值 $v > 0$，使得由直线 $OP_0$、$OP_3$ 和曲线围成的面积与 $\frac{π}{4}$（四分之一圆的面积）相等。

曲线长度与四分之一圆的长度有多少百分比？  That is, if $L$ is the length of the curve, calculate $100 × \displaystyle\frac{L − \frac{π}{2}}{\frac{π}{2}}$. 将您的答案四舍五入到小数点后面的10位数。
