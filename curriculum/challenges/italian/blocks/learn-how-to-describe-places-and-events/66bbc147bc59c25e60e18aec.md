# --explanation--

Il tempo `Present Continuous` si usa per descrivere azioni che stanno accadendo proprio ora. Per esprimere che qualcosa **non sta** accadendo, si usa la forma negativa.

Nella frase `The live preview isn't showing`, `isn't` è una contrazione di `is not`, rendendo la frase negativa.

Questo indica che la live preview non funziona come previsto. Quindi, Tom segnala che la live preview attualmente non è visibile in Visual Studio Code.
