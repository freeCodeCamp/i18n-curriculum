# --description--

花火玉が地面から100mの高さで爆発します。非常に多くの小さな破片に分かれ、それらはあらゆる方向に動きます。すべての破片は同じ初速度20 $\frac{\text{m}}{\text{s}}$ を持っています。

破片は空気抵抗を受けず、重力加速度 $g=9.81 \frac{\text{m}}{\text{s}^2}$ の一様な重力場の中で動くと仮定します。

破片が地面に到達するまでに動く領域の体積（$\text{m}^3$）を求めてください。答えは小数点以下4桁に丸めてください。
