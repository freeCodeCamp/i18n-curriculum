# --explanation--

`Clarifying` significa explicar algo de uma forma que elimina a confusão.Ajuda alguém a entender melhor um conceito ou situação.Por exemplo:

`Thanks for clarifying how the API works.` - O orador está grato porque a explicação tornou a API mais fácil de entender.

`Abstraction` refere-se a simplificar algo mostrando apenas as características essenciais e ocultando os funcionamentos internos complexos.Por exemplo:

`Using a login function is an example of abstraction - you don't need to know how the password is checked, just that it works.` - Isso mostra como os usuários interagem com interfaces simples enquanto a lógica técnica permanece oculta.
