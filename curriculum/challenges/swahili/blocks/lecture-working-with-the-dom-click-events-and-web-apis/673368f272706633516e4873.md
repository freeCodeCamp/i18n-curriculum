# --interactive--

Huenda umeshawahi kufanya kazi na sifa za vipengele katika miradi yako ya HTML na CSS. Lakini je, ulijua kuwa unaweza kuongeza sifa za kipengele kwa vipengele vyako vya HTML katika faili lako la JavaScript?

Katika funzo hili, utafundishwa jinsi ya kufanya kazi na njia ya `setAttribute()` kuongeza sifa za kipengele kwa vipengele vyako vya HTML.

Hapa kuna sintaksia ya msingi:

```js
setAttribute(attribute, value);
```

Tuchunguze mifano michache ili kuelewa vizuri jinsi ya kutumia njia ya `setAttribute()`.

Katika mfano huu wa kwanza, tuna kipengele cha `p` ndani ya HTML:

```html
<p id="para">I am a paragraph</p>
```

Ili kuongeza sifa ya `class` kwanza tunahitaji kupata upatikanaji wa kipengele hicho cha `p` kwa kutumia njia ya `getElementById()`. Kisha tunaweza kutumia njia ya `setAttribute()` kuongeza sifa ya `class` na kuweka thamani kuwa `my-class`.

Tunatumia `outerHTML` kuandika kipengele kizima cha HTML pamoja na sifa zake.

:::interactive_editor

```html
<p id="para">I am a paragraph</p>
<script src="index.js"></script>
```

```js
const para = document.getElementById("para");
para.setAttribute("class", "my-class");

console.log(`${para.outerHTML}`);
```

:::

Kama una sifa ya kipengele iliyopo kwenye kipengele cha HTML, unaweza kusahihisha thamani yake kwa kutumia njia ya `setAttribute()`.

Katika mfano huu tuna kipengele cha `div` chenye sifa ya `class` iliyowekwa kuwa `my-class`:

```html
<div class="my-class"></div>
```

Ili kusahihisha thamani ya sifa ya `class` kuwa example, tunaweza kutumia tena njia ya `setAttribute()`:

:::interactive_editor

```html
<div class="my-class"></div>
<script src="index.js"></script>
```

```js
const divEl = document.querySelector(".my-class");
divEl.setAttribute("class", "example");

console.log(`${divEl.outerHTML}`);
```

:::

Basi, ni mifano gani halisi ya wakati wa kutumia njia ya `setAttribute()`?

Kama unajenga maktaba ya picha inayobadilika, unaweza kuhitaji kusahihisha sifa ya `src` ya kipengele cha picha wakati mtumizi anapobofya kwenye picha ndogo.

Mfano mwingine ni kama unashughulika na uthibitisho wa fomu na unahitaji kuongeza sifa fulani kama `required` au `minlength` kwa kipengele cha `input`.
