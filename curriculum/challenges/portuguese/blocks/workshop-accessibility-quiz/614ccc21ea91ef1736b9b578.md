# --description--

Para este workshop, você recebeu um boilerplate básico.

Inicie esta jornada de acessibilidade fornecendo um atributo `lang` ao seu elemento `html`.Isto ajudará os leitores de tela a identificar o idioma da página.

Além disso, lembre-se de vincular sua folha de estilos à página.

# --hints--

You should give the `html` element a `lang` attribute. _Hint: You can use the value `en` for English._

```js
assert.match(code, /<html\s+lang=('|")[\w\-]+?\1\s*>/i);
// TODO: This should/could be fixed in the builder.js
// assert.notThrow(Intl.getCanonicalLocales(document.querySelector('html').lang));
```

Você deve ter um elemento `link` no seu elemento `head`.

```js
assert.exists(document.querySelector("head>link"));
```

Seu elemento `link` deve ter `rel="stylesheet"` e `href="styles.css"`.

```js
const link = document.querySelector("head>link");
assert.equal(link?.getAttribute("rel"), "stylesheet");
const href = link?.getAttribute("data-href");
assert.oneOf(href, ["./styles.css", "styles.css"]);
```
