# --description--

Wenn Sie über jemand anderen sprechen, können Sie diese Frage verwenden, um nach dessen Namen zu fragen:

`¿Cómo se llama?` – Wie heißt er/sie?

Das Wort `cómo` bedeutet „wie“. Es ist sehr wichtig, das Akzentzeichen auf dem Buchstaben `o` zu setzen, wenn dieses Wort eine Frage einleitet.

Das Wort `se` zeigt an, dass Sie über eine dritte Person sprechen.

Elena spricht mit Esteban über seinen neuen Kollegen Diego.
