# --explanation--

`To work hard` significa esforzarse mucho. Observa que el adverbio para `hard` en este caso también es `hard`, no `hardly`. `To hardly work` significa hacer casi nada de trabajo, hacer poco o ningún esfuerzo, lo cual es lo opuesto a lo que María está diciendo sobre el equipo. Por ejemplo:

- `She hardly trained for the marathon.` - Esto significa que casi no hizo ningún esfuerzo para estar preparada para el maratón.

- `She trained hard for the marathon.` - Esto significa que ella se esforzó mucho para prepararse.
