# --description--

Eine Zahl, die ausschließlich aus Einsen besteht, wird Repunit genannt. Wir definieren $R(k)$ als einen Repunit der Länge $k$; zum Beispiel ist $R(6) = 111111$.

Gegeben sei eine positive ganze Zahl $n$ mit $GCD(n, 10) = 1$. Es lässt sich zeigen, dass es immer einen Wert $k$ gibt, für den $R(k)$ durch $n$ teilbar ist. Wir bezeichnen mit $A(n)$ den kleinsten solchen Wert von $k$; zum Beispiel ist $A(7) = 6$ und $A(41) = 5$.

Der kleinste Wert von $n$, für den $A(n)$ erstmals zehn übersteigt, ist 17.

Finden Sie den kleinsten Wert von $n$, für den $A(n)$ erstmals eine Million übersteigt.
