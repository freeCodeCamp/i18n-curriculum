# --interactive--

在 JavaScript 中處理字串時，有許多情況你可能需要調整文字的大小寫，例如將所有字母轉換為大寫以用於標題，或將文字轉換為小寫以保持一致性。

幸運的是，JavaScript 透過兩個內建的方法：`toUpperCase()` 和 `toLowerCase()` 讓這件事變得簡單。

`toUpperCase()` 方法將所有字元轉換為大寫字母，並傳回一個包含所有大寫字元的新字串。當你想強調文字或在字串格式上保持一致性時，這非常有用。

讓我們來看一個範例：

:::interactive_editor

```js
let greeting = "Hello, World!";
let uppercaseGreeting = greeting.toUpperCase();
console.log(uppercaseGreeting);  // "HELLO, WORLD!"
```

:::

在這段程式碼中，`toUpperCase()` 方法將整個字串轉換成大寫字母。

原始字串保持不變，因為 `toUpperCase()` 傳回一個新的字串，而不是修改原始字串。

另一方面，`toLowerCase()` 方法會將字串中的所有字元轉換為小寫。當你需要標準化輸入時，例如比較使用者提供的文字或進行不區分大小寫的檢查，這非常有用。

讓我們來看一個範例：

:::interactive_editor

```js
let shout = "I AM LEARNING JAVASCRIPT!";
let lowercaseShout = shout.toLowerCase();
console.log(lowercaseShout);  // "i am learning javascript!"
```

:::

`toLowerCase()` 方法將所有字元轉換為小寫，使字串不那麼強烈，同時保持原始字串不變。

總結來說，JavaScript 中的 `toUpperCase()` 和 `toLowerCase()` 方法是將字串轉換成全大寫或全小寫字母的強大工具。

這些方法特別適用於標準化文字輸入、進行不區分大小寫的比較，以及確保設計一致性。

透過這些簡單卻有效的行為，你可以更有控制性且可預測地處理文字操作。
