# --description--

Indem Sie verschiedene Selektoren und Eigenschaften manipulieren, können Sie interessante Formen erstellen. Eine der einfacheren Formen zum Ausprobieren ist eine Mondsichel. Für diese Aufgabe müssen Sie mit der `box-shadow`-Eigenschaft arbeiten, die den Schatten eines Elements festlegt, zusammen mit der `border-radius`-Eigenschaft, die die Rundung der Ecken des Elements steuert.

Sie werden ein rundes, transparentes Objekt mit einem scharfen Schatten erstellen, der leicht seitlich versetzt ist – der Schatten wird tatsächlich die Mondform sein, die Sie sehen.

Um ein rundes Objekt zu erzeugen, sollte die `border-radius`-Eigenschaft auf einen Wert von 50 % gesetzt werden.

Sie erinnern sich vielleicht aus einer früheren Aufgabe, dass die `box-shadow`-Eigenschaft Werte für `offset-x`, `offset-y`, `blur-radius`, `spread-radius` und einen `color`-Wert in dieser Reihenfolge annimmt. Die `blur-radius`- und `spread-radius`-Werte sind optional.

# --instructions--

Manipulieren Sie das quadratische Element im Editor, um die Mondform zu erstellen. Ändern Sie zuerst die `background-color` in `transparent`, dann setzen Sie die `border-radius`-Eigenschaft auf 50 %, um die kreisförmige Form zu erzeugen. Ändern Sie schließlich die `box-shadow`-Eigenschaft, um die `offset-x` auf 25px, die `offset-y` auf 10px, `blur-radius` auf 0, `spread-radius` auf 0 und `color` auf `blue` zu setzen.

# --hints--

Der Wert der `background-color`-Eigenschaft sollte auf `transparent` gesetzt werden.

```js
assert.match(code,/background-color:\s*?transparent;/gi);
```

Der Wert der `border-radius`-Eigenschaft sollte auf `50%` gesetzt werden.

```js
assert.match(code,/border-radius:\s*?50%;/gi);
```

Der Wert der `box-shadow`-Eigenschaft sollte für `offset-x` 25px, für `offset-y` 10px, für `blur-radius` 0, für `spread-radius` 0 und schließlich für `blue` den Wert `color` haben.

```js
assert.match(code,/box-shadow:\s*?25px\s+?10px\s+?0(px)?\s+?0(px)?\s+?blue\s*?;/gi);
```
