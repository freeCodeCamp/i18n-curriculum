# --description--

Debajo de tu elemento `div` existente, agrega un nuevo elemento `div` con un `class` establecido en `table-wrap`. Este será el contenedor para tus tablas.

Dentro de eso, agregue tres elementos `table`. Usará CSS para dar estilo a estas en una sola tabla, pero usar tablas separadas ayudará a los lectores de pantalla a entender el flujo del documento.

# --hints--

Debes crear un nuevo elemento `div`.

```js
assert(document.querySelectorAll('div')?.length === 2);
```

Su nuevo elemento `div` debe tener la `class` establecida en `table-wrap`.

```js
assert(document.querySelector('.table-wrap')?.localName === 'div');
```

Su elemento `.table-wrap` debe venir después de su `div` existente.

```js
assert(document.querySelectorAll('div')?.[1]?.classList?.contains('table-wrap'));
```

Su elemento `.table-wrap` debe tener tres elementos `table`.

```js
const children = [...(document.querySelector('.table-wrap')?.children ?? [])];
assert(children?.length === 3);
children.forEach(child => assert(child?.localName === 'table'));
```
