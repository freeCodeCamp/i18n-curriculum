# --explanation--

Nel **discorso indiretto**, `to explain` si usa quando qualcuno dà una ragione o chiarisce qualcosa. Per esempio:

- Discorso diretto: `The server went down because of a power issue.`

- Discorso indiretto: `She explained that the server went down due to a power issue.`

Un `system component` è una parte o componente di un sistema più grande, di solito in computer o reti. Per esempio:

`The login module is a key system component for user access.` – Questo significa che è uno dei componenti essenziali del sistema.
