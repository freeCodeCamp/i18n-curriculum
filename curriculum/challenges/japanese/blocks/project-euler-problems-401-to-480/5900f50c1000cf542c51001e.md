# --description--

格子点の集合 $S$ は、$S$ のちょうど2点を通る直線が存在する場合に titanic set と呼ばれます。

titanic set の例として、$S = \\{(0, 0), (0, 1), (0, 2), (1, 1), (2, 0), (1, 0)\\}$ があり、(0, 1) と (2, 0) を通る直線は $S$ の他の点を通りません。

一方、集合 {(0, 0), (1, 1), (2, 2), (4, 4)} は titanic set ではありません。なぜなら、この集合の任意の2点を通る直線は他の2点も通るからです。

任意の正の整数 $N$ に対して、$0 ≤ x$, $y ≤ N$ を満たすすべての点 ($x$, $y$) を含む titanic set $S$ の個数を $T(N)$ とします。$T(1) = 11$, $T(2) = 494$, $T(4) = 33\\,554\\,178$, $T(111)\bmod {10}^8 = 13\\,500\\,401$, $T({10}^5)\bmod {10}^8 = 63\\,259\\,062$ であることが確認できます。

$T({10}^{11})\bmod {10}^8$ を求めてください。
