# --explanation--

`Could`와 `might`는 가능성을 말할 때 쓰는 조동사입니다.

`Could`는 어떤 일이 가능하지만 확실하지 않다는 뜻입니다. 예를 들어:

`This update could make the app run faster.` - 업데이트가 앱 속도를 향상시킬 가능성이 있습니다.

`Might`도 가능성을 나타내지만 `could`보다 가능성이 낮을 때 자주 쓰입니다. 예를 들어:

`The new feature might improve the user experience.` - 이 기능이 경험을 더 좋게 만들 가능성이 있습니다.
