# --description--

`revoke`とは、決定、権利、許可を正式に取り消したり取り戻したりすることを意味します。例えば、`The administrator may revoke access if rules are broken`。

この文脈で`issue`とは、バッジや許可証のようなものを正式に提供または配布することを意味します。例えば、`The office issues ID cards to all new employees`。

`Access control`は、場所やリソースへの出入りを制限し管理する行為を指します。例えば、`Access control systems are used to secure the building`。
