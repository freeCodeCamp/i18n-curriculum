# --interactive--

CSS ist verantwortlich für die Gestaltung einer Webseite. All diese Gestaltungen bestehen aus verschiedenen CSS-Regeln.

Eine CSS-Regel besteht aus zwei Hauptteilen: einem **Selektor** und einem **Deklarationsblock**.

Werfen wir einen Blick auf die grundlegende Syntax:

```css
selector {
  property: value;
}
```

Ein Selektor ist ein Muster, das in CSS verwendet wird, um bestimmte HTML-Elemente für die Gestaltung zu identifizieren und anzusprechen.

Beispiele für Selektoren sind Typselektoren, Klassenselektoren und ID-Selektoren.

Die geschweiften Klammern, die in der Grundsyntax verwendet werden, nennt man Deklarationsblock. Ein Deklarationsblock wendet eine Reihe von Stilen auf einen bestimmten Selektor oder mehrere Selektoren an.

Innerhalb des Deklarationsblocks finden Sie eine Reihe von Deklarationen. Jede Deklaration besteht aus einer Eigenschaft und einem Wert.

Die Eigenschaft ist der CSS-Bezeichner, der angibt, welche Funktion gestaltet wird. Ein Beispiel für eine Eigenschaft wäre die `background-color` Eigenschaft.

Der Wert ist die spezifische Einstellung, die auf diese Eigenschaft angewendet wird. Zum Beispiel, wenn die Eigenschaft `background-color` ist, könnte ein Wert `purple` sein, der die Hintergrundfarbe auf Lila setzt.

Nach jedem Eigenschaftsnamen müssen Sie einen Doppelpunkt setzen, und nach jedem Wert sollte ein Semikolon folgen.

Jetzt, da Sie die Syntax für eine CSS-Regel kennen, sehen wir uns ein Beispiel an. Aktivieren Sie den interaktiven Editor und klicken Sie auf den `styles.css` Tab, um den CSS-Code zu sehen.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Learning about CSS</h1>

<p>Example paragraph element</p>
<p>Another example paragraph element</p>
```

```css
p {
  color: blue;
}
```

:::

In dieser CSS-Regel spricht ein Typselektor alle Absatz-Elemente auf der Seite an.

Innerhalb des Deklarationsblocks gibt es eine Deklaration.

Die Deklaration besteht aus der `color` Eigenschaft mit einem Wert, der auf `blue` gesetzt ist. Dies ändert die Textfarbe aller Absatz-Elemente auf `blue`.

Wenn Sie denselben Satz von Stilen auf mehrere Selektoren anwenden möchten, können Sie eine Selektorenliste erstellen, bei der jeder Selektor durch ein Komma getrennt ist.

Hier ist ein Beispiel für das Gestalten mehrerer Selektoren (um mit dem Beispiel zu interagieren, aktivieren Sie den interaktiven Editor):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1 id="title">Example heading</h1>
<h2 class="subheading">Example subheading</h2>
<p>This paragraph is not affected by the selector.</p>
```

```css
#title,
.subheading {
  color: navy;
}
```

:::

In dieser Selektorenliste gibt es einen `id` Selektor, der das HTML-Element mit dem `id` Wert `title` anspricht. Alle `id` Selektoren müssen mit einem Hash `#` Symbol beginnen.

Dann folgt ein Komma, gefolgt von einem `class` Selektor, der alle HTML-Elemente mit dem `class` Wert `subheading` anspricht. Alle Klassenselektoren müssen mit einem Punkt `.` beginnen.

Die gesamte Selektorenliste erhält dieselbe Gestaltung, wobei die Textfarbe auf `navy` gesetzt wird.
