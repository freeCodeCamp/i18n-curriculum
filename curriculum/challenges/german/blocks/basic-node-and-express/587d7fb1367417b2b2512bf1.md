# --description--

Während ein HTML-Server HTML bereitstellt, liefert eine API Daten. Eine <dfn>REST</dfn> (Representational State Transfer) API ermöglicht den Datenaustausch auf einfache Weise, ohne dass Clients Details über den Server kennen müssen. Der Client muss nur wissen, wo sich die Ressource befindet (die URL) und welche Aktion er darauf ausführen möchte (das Verb). Das Verb GET wird verwendet, wenn Sie Informationen abrufen, ohne etwas zu verändern. Heutzutage ist JSON das bevorzugte Datenformat, um Informationen im Web zu übertragen. Einfach gesagt ist JSON eine praktische Möglichkeit, ein JavaScript-Objekt als String darzustellen, damit es leicht übertragen werden kann.

Lassen Sie uns eine einfache API erstellen, indem wir eine Route anlegen, die unter dem Pfad `/json` mit JSON antwortet. Sie können dies wie gewohnt mit der `app.get()`-Methode tun. Innerhalb des Route-Handlers verwenden Sie die Methode `res.json()` und übergeben ein Objekt als Argument. Diese Methode schließt den Anfrage-Antwort-Zyklus ab, indem sie die Daten zurückgibt. Im Hintergrund wandelt sie ein gültiges JavaScript-Objekt in einen String um, setzt dann die passenden Kopfzeilen, um Ihrem Browser mitzuteilen, dass Sie JSON bereitstellen, und sendet die Daten zurück. Ein gültiges Objekt hat die übliche Struktur `{key: data}`. `data` kann eine Zahl, ein String, ein verschachteltes Objekt oder ein Array sein. `data` kann auch eine Variable oder das Ergebnis eines Funktionsaufrufs sein, in diesem Fall wird es ausgewertet, bevor es in einen String umgewandelt wird.

# --instructions--

Geben Sie das Objekt `{"message": "Hello json"}` als Antwort im JSON-Format auf GET-Anfragen an die Route `/json` zurück. Rufen Sie dann in Ihrem Browser `your-app-url/json` auf, Sie sollten die Nachricht auf dem Bildschirm sehen.

# --hints--

Der Endpunkt `/json` sollte das JSON-Objekt `{"message": "Hello json"}` bereitstellen

```js
  const response = await fetch(code + '/json');
  if (!response.ok) {
    throw new Error(await response.text());
  }
  const data = await response.json();
  assert.equal(
    data.message,
    'Hello json',
    "The '/json' endpoint does not serve the right data"
  );
```
