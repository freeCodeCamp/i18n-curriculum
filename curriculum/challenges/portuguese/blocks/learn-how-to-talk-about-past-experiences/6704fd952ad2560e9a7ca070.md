# --explanation--

A expressão `stuck on` é usada quando alguém não consegue avançar em uma tarefa ou resolver um problema. Sugere estar preso ou incapaz de progredir. Por exemplo:

- `We were stuck on this code error for hours.` - Não conseguimos prosseguir até que esse erro no código fosse encontrado.

- `She got stuck on the final question of the exam.` - Ela passou muito tempo na última questão.

No diálogo, Lisa usa `stuck on` para descrever como eles não conseguiram avançar na resolução de um bug no trabalho deles.
