# --description--

La palabra `actually` se utiliza a menudo para expresar un hecho o para corregir una proposición o suposición anterior. Hace hincapié en lo que se está diciendo y puede introducir información nueva, a menudo sorprendente.

Por ejemplo, `She isn't just my colleague; she's actually my sister.`

La frase `think of` se utiliza para expresar la idea de considerar o planear hacer algo. Al usar `think of`, es común seguirlo con un verbo en su forma `-ing` (gerund). Esta estructura se utiliza para hablar de posibles acciones o intenciones.

Por ejemplo, `'m thinking of starting a new project.` Significa considerar iniciar algo nuevo.
