Cos’è JSON e come si consultano i valori usando la notazione a parentesi e a punto?

JSON sta per JavaScript Object Notation. È un formato di dati leggero basato su testo, comunemente usato per scambiare dati tra un server e un’applicazione web.

Uno dei motivi per cui JSON è così popolare nello sviluppo web è che è sia facilmente analizzabile dalle macchine sia leggibile dall’essere umano.

Poiché JSON è indipendente dal linguaggio, puoi facilmente inviare dati JSON da un’applicazione Java a un’applicazione Python, o da un’applicazione JavaScript a un’applicazione C#.

JSON supporta molti tipi di dati, inclusi oggetti, array, stringhe, booleani, null e numeri.

Ecco un esempio di oggetto JSON:

```js
{
  "name": "Alice",
  "age": 30,
  "isStudent": false,
  "list of courses": ["Mathematics", "Physics", "Computer Science"]
}
```

Come puoi vedere, JSON usa coppie chiave-valore per archiviare informazioni e ogni coppia è separata da una virgola. Ogni chiave deve essere racchiusa tra virgolette doppie, altrimenti otterrai un errore.

Per consultare i dati da un oggetto JSON, puoi usare la notazione a punto o la notazione a parentesi. In questo esempio, usiamo la notazione a punto per consultare il `age` dall’oggetto JSON:

```js
import data from "./example.json" with { type: "json" };

console.log(data.age);
```

Questo esempio particolare usa quella che si chiama un’istruzione `import`, che importa l’oggetto JSON in questo file così possiamo consultarlo. Imparerai di più sull’istruzione `import` in una futura lezione.

Puoi anche usare la notazione a parentesi per consultare informazioni dagli oggetti JSON. Ecco un esempio di consultazione dell’array `list of courses`:

```js
import data from "./example.json" with { type: "json" };

console.log(data["list of courses"]);
```

Usare la notazione a parentesi è particolarmente utile qui perché la chiave contiene più parole separate da spazi. Se provassimo a usare la notazione a punto, otterremmo un errore.

In sintesi, JSON è un formato versatile che può archiviare molti tipi di dati, inclusi array e oggetti annidati. Usando la notazione a punto o a parentesi, puoi facilmente consultare i valori archiviati all’interno di un oggetto JSON.
