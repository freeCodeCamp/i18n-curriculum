# --description--

محدِّد النوع (أو محدِّد العنصر) يختار كل العناصر من نوع العنصر المعطى، والصيغة النحوية هي فقط اسم العنصر:

```html
<!-- index.html -->

<div>Hello, World!</div>
<div>Hello again!</div>
<p>Hi...</p>
<div>Okay, bye.</div>
```

```css
/* styles.css */

div {
  color: white;
}
```

هنا، سيتم اختيار كل العناصر الثلاثة `<div>`، بينما العنصر `<p>` لن يتم اختياره.

## --text--

أي مما يلي يصف شفرة CSS المعطاة أعلاه بشكل أفضل؟

## --answers--

تطبق الشفرة لون `white` على كل عناصر `div` في ملف HTML.

---

تطبق الشفرة لون `white` على كل عناصر `p` في ملف HTML.
