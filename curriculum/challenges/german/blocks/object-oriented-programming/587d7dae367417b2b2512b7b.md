# --description--

Im folgenden Beispiel definiert der `Bird` Konstruktor zwei Eigenschaften: `name` und `numLegs`:

```js
function Bird(name) {
  this.name = name;
  this.numLegs = 2;
}

let duck = new Bird("Donald");
let canary = new Bird("Tweety");
```

`name` und `numLegs` werden <dfn>eigene Eigenschaften</dfn> genannt, weil sie direkt auf dem Instanzobjekt definiert sind. Das bedeutet, dass `duck` und `canary` jeweils eine eigene separate Kopie dieser Eigenschaften besitzen. Tatsächlich hat jede Instanz von `Bird` ihre eigene Kopie dieser Eigenschaften. Der folgende Code fügt alle eigenen Eigenschaften von `duck` zum Array `ownProps` hinzu:

```js
let ownProps = [];

for (let property in duck) {
  if(duck.hasOwnProperty(property)) {
    ownProps.push(property);
  }
}

console.log(ownProps);
```

Die Konsole würde den Wert `["name", "numLegs"]` anzeigen.
