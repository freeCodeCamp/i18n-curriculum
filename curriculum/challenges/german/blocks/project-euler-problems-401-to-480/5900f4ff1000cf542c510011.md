# --description--

Es lässt sich zeigen, dass das Polynom $n^4 + 4n^3 + 2n^2 + 5n$ für jede ganze Zahl $n$ ein Vielfaches von 6 ist. Ebenso lässt sich zeigen, dass 6 die größte ganze Zahl ist, die diese Eigenschaft erfüllt.

Definieren Sie $M(a, b, c)$ als das maximale $m$, so dass $n^4 + an^3 + bn^2 + cn$ für alle ganzen Zahlen $n$ ein Vielfaches von $m$ ist. Zum Beispiel gilt $M(4, 2, 5) = 6$.

Definieren Sie außerdem $S(N)$ als die Summe von $M(a, b, c)$ für alle $0 < a, b, c ≤ N$.

Wir können überprüfen, dass $S(10) = 1\\,972$ und $S(10\\,000) = 2\\,024\\,258\\,331\\,114$ gilt.

Sei $F_k$ die Fibonacci-Folge:

- $F_0 = 0$, $F_1 = 1$ und
- $F_k = F_{k - 1} + F_{k - 2}$ für $k ≥ 2$.

Finden Sie die letzten 9 Ziffern von $\sum S(F_k)$ für $2 ≤ k ≤ 1\\,234\\,567\\,890\\,123$.
