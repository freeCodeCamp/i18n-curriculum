# --description--

放物線 $y = \frac{x^2}{k}$ 上に、3つの点 $A(a, \frac{a^2}{k})$、$B(b, \frac{b^2}{k})$、$C(c, \frac{c^2}{k})$ が選ばれています。

$F(K, X)$ を、三角形 $ABC$ の少なくとも1つの角が45°であるような整数の4つ組 $(k, a, b, c)$ の個数とします。ただし、$1 ≤ k ≤ K$ かつ $-X ≤ a < b < c ≤ X$ とします。

例えば、$F(1, 10) = 41$、$F(10, 100) = 12\\,492$ です。

$F({10}^6, {10}^9)$ を求めてください。
