# --description--

Розгляньте всі вузли ґратки ($a$, $b$, $c$) з $0 ≤ a, b, c ≤ N$.

Від початку координат $O(0, 0, 0)$ проведено всі лінії до інших вузлів ґратки. Нехай $D(N)$ — кількість різних таких ліній.

Відомо, що $D(1\\,000\\,000) = 831\\,909\\,254\\,469\\,114\\,121$.

Знайдіть $D({10}^{10})$. Відповіддю будуть перші дев’ять цифр, за якими йдуть останні дев’ять цифр.
