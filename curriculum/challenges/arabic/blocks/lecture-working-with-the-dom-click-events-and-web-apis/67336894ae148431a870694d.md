# --interactive--

دعنا نتعلم عن DOM ولماذا هو مهم جدًا لتطوير الويب. DOM تعني نموذج كائن المستند. إنها واجهة برمجة تتيح لنا التفاعل مع مستندات HTML.

مع DOM، يمكنك إضافة أو تعديل أو حذف عناصر على صفحة الويب. يمكنك حتى جعل موقعك تفاعليًا بجعل العناصر تستمع إلى الأحداث وتستجيب لها.

في DOM، يُمثَّل مستند HTML كشجرة من العقد. كل عقدة عنصر تمثل عنصر HTML من مستند HTML:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>DOM Example</title>
  </head>
  <body>
    <h1>What is the DOM?</h1>
    <h2>Let's learn about the DOM</h2>
  </body>
</html>
```

هذا رسم بياني يمثل الهيكل الأساسي لـ DOM في مثالنا:

```md
Document
========

HTML
-----

- Head
  - Title
- Body
  - H1
  - H2   
```

بالطبع، يمكن أن يكون هذا أكثر تفصيلاً وتعقيدًا بناءً على هيكل ترميز HTML لصفحة الويب.

مستند HTML هو العقدة الجذرية في تسلسل DOM. له عقدة فرعية واحدة، وهي عنصر `html`. هذا هو العنصر الجذري لمستند HTML، إذ تنحدر منه جميع العقد الأخرى.

عنصر `html` له طفلان: `head` و `body`.

عنصر `head` يحتوي على بيانات وصفية عن المستند. يوفر معلومات أساسية عن صفحة الويب.

عنصر `body` يحتوي على المحتوى المرئي لصفحة الويب.

يمكنك الوصول إلى هذه العناصر باستخدام JavaScript، والعمل معها في شفرتك، وحتى تعديلها ديناميكيًا. هذه هي قوة DOM وواجهات برمجة التطبيقات للويب.

للوصول إلى هذه العناصر في JavaScript، يمكنك استخدام طريقتي `getElementById()` و `querySelector()`. هذه الطرق هي واجهات برمجة تطبيقات ويب لأنها توفر طرقًا موحدة للتفاعل مع DOM باستخدام JavaScript.

مع `getElementById()`، يمكنك الحصول على كائن يمثل عنصر HTML بالـ `id` المحدد. تذكر أن الـ `id` يجب أن يكون فريدًا في كل مستند HTML، لذا سترجع هذه الطريقة كائن `Element` واحد فقط. هنا مثال:

```js
const container = document.getElementById("container");
```

سطر شفرة JavaScript هذا يحصل على عنصر بقيمة `id` تساوي `container` ويُعيّن هذا الكائن إلى ثابت في JavaScript. يجب تمرير الـ `id` بين علامات اقتباس كمعلمة. إذا سجلت هذا الكائن في وحدة التحكم، سترى النتيجة في الإخراج:

:::interactive_editor

```html
<div id="container">
  <h1>Hello, World!</h1>
  <p>Welcome to learning about the DOM.</p>
</div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
console.log(container); 
```

:::

`querySelector()` أوسع من `getElementById()`. مع `querySelector()`، يمكنك الحصول على أول عنصر في مستند HTML يطابق محدد CSS الممرر كمعلمة.

في هذا المثال، ستحصل على العنصر الذي يطابق المحدد `section`:

:::interactive_editor

```html
<section>
  <h2>Section Title</h2>
  <p>This is a section of the webpage.</p>
</section>
<script src="index.js"></script>  
```

```js
const sectionEl = document.querySelector("section");
console.log(sectionEl);
```

:::

إذا أردت اختيار عنصر حسب اسم الصنف باستخدام `querySelector()`، يجب أن تسبق اسم الصنف بنقطة (`.`). إليك مثالًا:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="highlight">
  <p>This content is highlighted.</p>
</div>
<script src="index.js"></script>
```

```css
body {
  background-color: lightgray;
}

.highlight {
  background-color: yellow;
}
```

```js
const highlightEl = document.querySelector(".highlight");
console.log(highlightEl);
```

:::

لديك أيضًا طرق أخرى لمطابقة عدة عناصر، مثل `getElementsByClassName()` و `querySelectorAll()`. ستتعلم المزيد عنها في دروس لاحقة.

بفهم كيفية استخدام واجهات برمجة التطبيقات للويب للتعامل مع DOM بكفاءة، يمكنك إنشاء تطبيقات ويب قوية وتفاعلية.
