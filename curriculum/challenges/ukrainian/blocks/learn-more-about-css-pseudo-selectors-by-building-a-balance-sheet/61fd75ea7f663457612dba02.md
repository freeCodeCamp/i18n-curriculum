# --description--

Усередині кожного з ваших нових елементів `th` вкладіть елемент `span` з атрибутом `class`, встановленим у `sr-only year`. Дайте їм такий текст (у порядку): `2019`, `2020` і `2021`.

Дайте вашому третьому елементу `th` атрибут `class`, встановлений у `current`.

Залиште елемент `td` порожнім. Цей елемент існує лише для того, щоб забезпечити чотириколонковий макет таблиці та асоціювати заголовки з правильними колонками.
