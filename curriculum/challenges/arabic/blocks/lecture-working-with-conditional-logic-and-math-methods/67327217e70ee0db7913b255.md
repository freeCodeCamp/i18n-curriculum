# --interactive--

عند الغوص في JavaScript، ستكتشف بسرعة أن إجراء العمليات الرياضية مهمة شائعة. بينما يمكن للمشغلات الحسابية الأساسية التعامل مع العمليات البسيطة، يوفر JavaScript كائن `Math` مدمجًا للتعامل مع تحديات رياضية أكثر تعقيدًا.

يوفر هذا الأداة المفيدة مجموعة متنوعة من الطرق التي تسهل إجراء الحسابات المتقدمة والتعامل مع الأرقام. دعنا نستكشف هذه الطرق ونرى كيف يمكنها تبسيط تجربة البرمجة الخاصة بك.

تُولد الطريقة `Math.random()` رقمًا عشريًا عشوائيًا بين `0` (شامل) و`1` (حصري). هذا يعني أن الناتج قد يكون `0`، لكنه لن يصل أبدًا إلى `1`. إليك مثالًا على العمل مع الطريقة `Math.random()`:

:::interactive_editor

```js
const randomNum = Math.random();

console.log(randomNum);
// any number between 0 and 1 – 0 inclusive and 1 exclusive
```

:::

تأخذ كل من `Math.min()` و`Math.max()` مجموعة من الأرقام وتُرجع القيمة الأدنى والأعلى على التوالي. إليك مثالًا على العمل مع كلتا الطريقتين:

:::interactive_editor

```js
const smallest = Math.min(1, 5, 3, 9);
console.log(smallest); // 1

const largest = Math.max(1, 5, 3, 9);
console.log(largest); // 9
```

:::

ستسجل أول دالة `console.log()` الرقم `1`، لأن `1` هو الأصغر في تلك القائمة من الأرقام. وستسجل دالة `console.log()` الثانية الرقم `9`، لأن `9` هو الأكبر في تلك القائمة.

إذا أردت تقريب الأرقام لأعلى أو لأسفل إلى أقرب عدد صحيح، يمكنك استخدام الطريقتين `Math.ceil()` و`Math.floor()`. إليك مثالًا على العمل مع `Math.ceil()`:

:::interactive_editor

```js
console.log(Math.ceil(4.3)); // 5
```

:::

ستقرب `Math.ceil()` الرقم `4.3` لأعلى إلى أقرب عدد صحيح، وهو `5` في هذه الحالة. الآن، لنلق نظرة على تقريب الرقم لأسفل:

:::interactive_editor

```js
console.log(Math.floor(4.7)); // 4
```

:::

ستقرب `Math.floor()` الرقم `4.7` لأسفل إلى أقرب عدد صحيح، وهو `4` في هذه الحالة. أما `Math.round()` فهي مزيج بين `Math.ceil()` و`Math.floor()`. تقرب الرقم إلى أقرب عدد صحيح مع الأخذ في الاعتبار العلامة العشرية:

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

إذا كانت العلامة العشرية أقل من `5`، يُقرب الرقم لأسفل. وإذا كانت العلامة العشرية `5` أو أكثر، يُقرب الرقم لأعلى. تطبيق عملي لـ `Math.floor()` و`Math.random()` هو توليد رقم عشوائي بين عددين صحيحين. إليك الصيغة لذلك:

:::interactive_editor

```js
const max = 10;
const min = 5;
const randomNum = Math.floor(Math.random() * (max - min + 1)) + min;
console.log(randomNum);
```

:::

توليد رقم عشوائي بين `20` و`1` سيكون كالتالي:

:::interactive_editor

```js
const randomNumBtw1And20 = Math.floor(Math.random() * 20) + 1;
console.log(randomNumBtw1And20);
```

:::

طريقة أخرى مفيدة في كائن `Math` هي `Math.trunc()`. تزيل `Math.trunc()` الجزء العشري من الرقم، وتُرجع الجزء الصحيح فقط، دون تقريب:

:::interactive_editor

```js
console.log(Math.trunc(2.9)); // 2
console.log(Math.trunc(9.1)); // 9
```

:::

إذا كنت بحاجة إلى حساب الجذر التربيعي أو الجذر التكعيبي لرقم، يمكنك استخدام الطريقتين `Math.sqrt()` و`Math.cbrt()` على التوالي:

:::interactive_editor

```js
console.log(Math.sqrt(81)); // 9
console.log(Math.cbrt(27)); // 3
```

:::

ستسجل العبارة الأولى في السجل الرقم `9` لأن الجذر التربيعي لـ `81` هو `9`، بينما ستسجل العبارة الثانية الرقم `3` لأن الجذر التكعيبي لـ `27` هو `3`. إذا كنت بحاجة إلى الحصول على القيمة المطلقة لرقم، يمكنك استخدام الطريقة `Math.abs()`:

:::interactive_editor

```js
console.log(Math.abs(-5)); // 5
console.log(Math.abs(5)); // 5
```

:::

تعيد `Math.abs()` القيمة المطلقة للرقم، محولة السالب إلى موجب. آخر طريقة سننظر إليها هي `Math.pow()`:

:::interactive_editor

```js
console.log(Math.pow(2, 3)); // 8
console.log(Math.pow(8, 2)); // 64
```

:::

تأخذ `Math.pow()` رقمين وترفع الأول إلى قوة الثاني. هناك العديد من الطرق الأخرى التي تنتمي إلى كائن `Math`، يمكنك استكشافها بنفسك. مع ذلك، هذه بعض الطرق الأكثر استخدامًا في قواعد شفرة JavaScript.
