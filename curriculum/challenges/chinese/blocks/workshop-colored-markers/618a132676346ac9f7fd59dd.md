# --description--

注意到你的视线自然而然地被中间的红色所吸引了吗？ 在设计网站时，你可以用这种效果来吸引人们对重要标题、按钮或链接的注意。

除了互补色，还有其他几种重要的颜色组合，稍后你会了解到。

现在，使用 `.two` CSS 规则中的 `rgb` 函数将 `background-color` 设置为黑色。

# --hints--

你的 `.two` CSS 规则应该有一个 `background-color` 属性，设置为 `rgb(0, 0, 0)`。

```js
assert.strictEqual(new __helpers.CSSHelp(document).getStyle('.two')?.backgroundColor, 'rgb(0, 0, 0)');
```
