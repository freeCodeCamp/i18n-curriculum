# --description--

在你宣告 doctype 之後，你需要提供一個 `<html>` 元素。這就是所謂的文件根元素，表示文件中的每個其他元素都將是它的後代。

當你學習使用 JavaScript 操作 HTML 時，這點會變得更加重要。現在，只要知道每個 HTML 文件都應該包含 `html` 元素即可。

回到 `index.html` 檔案，讓我們透過輸入開啟和關閉標籤來添加 `<html>` 元素，如下所示：

```html
<!DOCTYPE html>
<html lang="en">
</html>
```

## 什麼是 lang 屬性？
`lang` 指定該元素中文字內容的語言。此屬性主要用於提升網頁的可存取性。它允許輔助技術，例如螢幕閱讀器，根據語言進行調整並喚起正確的發音。
