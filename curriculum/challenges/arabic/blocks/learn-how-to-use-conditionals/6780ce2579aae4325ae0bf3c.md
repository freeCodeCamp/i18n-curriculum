# --explanation--

`Critical` تعني شيئًا مهمًا جدًا أو أساسيًا، و`critical data` تشير إلى معلومات مهمة يعتمد عليها الفريق للعمل بفعالية. فقدانها يسبب تأخيرات أو مشاكل. على سبيل المثال:

- `The backup includes all critical data in case of a failure.` - النسخة المحفوظة تحتوي على معلومات أساسية للتعافي من مشكلة.

- `You must encrypt critical data to ensure it's secure.` - حماية المعلومات المهمة بالتشفير ضرورية للأمان.
