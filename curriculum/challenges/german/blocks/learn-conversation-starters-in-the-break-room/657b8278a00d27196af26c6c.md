# --description--

Wenn Sie Fragen zu einer dritten Person Singular (`he`, `she` oder `it`) formulieren, müssen Sie möglicherweise das Hilfsverb `does` verwenden.

Zum Beispiel: Wenn Alice neugierig auf Sophies Hobbys ist und Tom fragen möchte, kann sie sagen, `Does Sophie like hiking?` Das Wort `does` wird hier verwendet, weil das Subjekt `Sophie` ist, eine dritte Person Singular.

Denken Sie daran, dass `does` verwendet wird, um Fragen im `present simple`-Tempus für `he`, `she` oder `it` zu bilden.
