# --description--

HTML і CSS — це дві мови, які працюють разом, щоб створити все, що ви бачите, коли дивитесь на щось в інтернеті. HTML — це сирі дані, з яких побудована сторінка. Увесь текст, посилання, картки, списки та кнопки створюються в HTML. CSS додає стиль цим простим елементам. HTML розміщує інформацію на сторінці, а CSS позиціонує цю інформацію, задає колір, змінює шрифт і робить її привабливою!

Які основні обов’язки HTML і CSS у створенні сторінки?

---

CSS керує функціональністю, а HTML контролює макет.
