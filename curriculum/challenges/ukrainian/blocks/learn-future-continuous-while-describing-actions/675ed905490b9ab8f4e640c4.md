# --explanation--

`Public` означає щось, що доступне для всіх або спільно використовується спільнотою. Наприклад:

`The repository is public so anyone can access it.` — показує, що репозиторій відкритий для всіх і не є приватним.

`Wi-Fi` (скорочення від `Wireless Fidelity`) позначає бездротове інтернет-з’єднання, яке дозволяє пристроям підключатися до інтернету. Наприклад:

`I connected my phone to the café's Wi-Fi to check my emails.` — показує, що ви отримали доступ до інтернету в кафе без використання кабелю.
