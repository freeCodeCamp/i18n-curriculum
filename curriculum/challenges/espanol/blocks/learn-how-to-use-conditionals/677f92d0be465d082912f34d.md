# --explanation--

La pregunta `Did you check that?` está preguntando si alguien verificó o confirmó algo. 

En este contexto, Mark quiere saber si Sarah revisó la conexión del cable de alimentación para asegurarse de que esté segura. Otros ejemplos:

- ¿Revisaste que las ventanas estén cerradas? - Esto significa preguntar si alguien se aseguró de que las ventanas estén bien cerradas.

- ¿Revisaste que la luz esté apagada? - Esto significa preguntar si alguien confirmó que la luz está apagada.
