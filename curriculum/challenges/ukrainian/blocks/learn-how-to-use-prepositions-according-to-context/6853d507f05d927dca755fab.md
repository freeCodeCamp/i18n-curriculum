# --explanation--

`Currently` означає «на даний момент часу». Наприклад:

`We currently have three open positions.` – Це означає, що зараз є три вакансії.

`Working on` означає бути зайнятим конкретним завданням або проєктом. Наприклад:

`She is working on the design for the new homepage.` – Це означає, що вона активно його проєктує.
