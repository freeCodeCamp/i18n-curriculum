# --description--

Нехай d(`n`) визначається як сума власних дільників `n` (чисел менших за `n`, які рівно діляться на `n`).

Якщо d(`a`) = `b` і d(`b`) = `a`, де `a` ≠ `b`, тоді `a` і `b` утворюють пару дружніх чисел, а кожне з `a` і `b` називають дружнім числом.

Наприклад, власні дільники числа 220 — це 1, 2, 4, 5, 10, 11, 20, 22, 44, 55 і 110; отже d(220) = 284. Власні дільники числа 284 — це 1, 2, 4, 71 і 142; тому d(284) = 220.

Обчисліть суму всіх дружніх чисел, менших за `n`.

```js
const sumAmicableNum = (n) => {
  const fsum = (n) => {
    let sum = 1;
    for (let i = 2; i <= Math.floor(Math.sqrt(n)); i++)
      if (Math.floor(n % i) === 0)
        sum += i + Math.floor(n / i);
    return sum;
  };
  let d = [];
  let amicableSum = 0;
  for (let i=2; i<n; i++) d[i] = fsum(i);
  for (let i=2; i<n; i++) {
    let dsum = d[i];
    if (d[dsum]===i && i!==dsum) amicableSum += i+dsum;
  }
  return amicableSum/2;
};
```
