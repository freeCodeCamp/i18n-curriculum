# --explanation--

`To be within budget` または `to stay within budget` は、計画または許可された金額だけを使い、それ以上は使わなかったことを意味します。これはプロジェクトや活動が財政的にうまく管理されていることを示します。例えば：

`The team stayed within budget, so we didn't need extra funding.` – これは、元々計画されていた以上の支出をしなかったことを意味します。
