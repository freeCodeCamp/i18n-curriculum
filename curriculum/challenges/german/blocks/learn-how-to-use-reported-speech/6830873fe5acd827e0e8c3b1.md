# --explanation--

`To seem` wird verwendet, wenn etwas basierend auf dem, was wir wissen, als wahr erscheint, wir uns aber nicht ganz sicher sind. Zum Beispiel:

`It seems the users liked the new menu.` – Das bedeutet, Sie denken, dass die Nutzer es mochten, basierend auf dem Feedback oder Verhalten, sind sich aber nicht sicher.

`To be found to be` ist eine **Passivkonstruktion**, die verwendet wird, wenn jemand etwas findet, entdeckt oder feststellt, wobei der Fokus auf dem Ergebnis liegt, nicht darauf, wer das Urteil gefällt hat. Zum Beispiel:

`The layout was found to be confusing by some testers.` – Das bedeutet, dass Tester es verwirrend fanden, aber der Satz betont die Feststellung, nicht die Personen (in diesem Fall die Tester).
