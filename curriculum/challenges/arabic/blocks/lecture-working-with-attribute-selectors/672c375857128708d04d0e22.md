# --interactive--

عند العمل مع القوائم المرتبة في HTML، تتيح لك الخاصية `type` تحديد نمط الترقيم المستخدم، مثل الأرقام، الأحرف الأبجدية، أو الأرقام الرومانية.

يمكن تعيين الخاصية `type` على عنصر `ol` (قائمة مرتبة) للتحكم في تنسيق الترقيم:

- `1` للقوائم الرقمية (مثل 1، 2، 3، ...)،

- `A` للقوائم الأبجدية بأحرف كبيرة (مثل A، B، C، ...)،

- `a` للقوائم الأبجدية بأحرف صغيرة (مثل a، b، c، ...)،

- `I` للأرقام الرومانية بأحرف كبيرة (مثل I، II، III، ...)،

- `i` للأرقام الرومانية بأحرف صغيرة (مثل i، ii، iii، ...).

إليك كيفية استخدام محدِّد الخاصية لاستهداف قائمة مرتبة ذات خاصية `type` محددة:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="A"] {
  color: purple;
  font-weight: bold;
}
```

:::

في هذا المثال، ستطبق قاعدة CSS نصًا باللون البنفسجي ووزن خط عريض على أي قائمة مرتبة تستخدم الترقيم الأبجدي بأحرف كبيرة، والمحددة بـ `type="A"`.

بنفس الطريقة، يمكنك استهداف القوائم المرتبة التي تستخدم الأرقام الرومانية بأحرف صغيرة عن طريق اختيار تلك التي تحتوي على `type="i"`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="i">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="i"] {
  color: green;
}
```

:::

ستغير هذه القاعدة لون النص إلى الأخضر لأي قائمة مرتبة تستخدم الأرقام الرومانية بأحرف صغيرة.

تمكنك القدرة على تنسيق القوائم المرتبة بناءً على خاصية `type` من التحكم بشكل أكبر في مظهر القوائم في مستندك، مما يسمح بالتخصيص حسب كيفية تنظيم المحتوى.
