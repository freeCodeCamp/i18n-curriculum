# --description--

如果我们计算 $0 ≤ a ≤ 5$ 时的 $a^2\bmod 6$，结果是：0、1、4、3、4、1。

满足 $a^2 ≡ a\bmod 6$ 的最大值 $a$ 是 $4$。

我们称 $M(n)$ 为满足 $a^2 ≡ a (\text{mod} n)$ 且 $a &lt; n$ 的最大值。所以 $M(6) = 4$。

查找 $\sum M(n)$，其中 $1 ≤ n ≤ {10}^7$。
