# --description--

在最後一個範例中，你有兩個選擇 `if` 或 `else`。但是如果你想要有超過兩個選擇呢？你可以使用 `else if` 述語為你的程式碼添加更多條件。`else if` 述語允許我們檢查多個條件，並在第一個條件為真時執行一個區塊的程式碼。如果第一個條件為假，它會檢查下一個條件，依此類推。如果沒有任何條件為真，則會執行 `else` 區塊。

這裡是一個 `else if` 述語的範例：

```javascript
let x = 5;

if (x > 10) {
  console.log("x is greater than 10");
} else if (x > 5) {
  console.log("x is greater than 5");
} else {
  console.log("x is less than or equal to 5");
}
```

在這個範例中，`x` 變數使用 `>` 運算子與數字 `10` 進行比較。如果 `x` 大於 `10`，第一個程式碼區塊將會被執行。否則，`else if` 述語將會檢查下一個條件。如果 `x` 大於 `5`，第二個程式碼區塊將會被執行。否則，`else` 區塊將會被執行。
