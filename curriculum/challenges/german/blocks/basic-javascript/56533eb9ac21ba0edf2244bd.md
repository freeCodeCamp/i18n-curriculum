# --description--

<dfn>Parameter</dfn> sind Variablen, die als Platzhalter für die Werte dienen, die bei einem Aufruf an eine Funktion übergeben werden sollen. Wenn eine Funktion definiert wird, wird sie typischerweise zusammen mit einem oder mehreren Parametern definiert. Die tatsächlichen Werte, die bei einem Aufruf an eine Funktion übergeben (oder <dfn>„passed“</dfn>) werden, nennt man <dfn>Argumente</dfn>.

Hier ist eine Funktion mit zwei Parametern, `param1` und `param2`:

```js
function testFun(param1, param2) {
  console.log(param1, param2);
}
```

Dann können wir `testFun` so aufrufen: `testFun("Hello", "World");`. Wir haben zwei String-Argumente übergeben, `Hello` und `World`. Innerhalb der Funktion wird `param1` den String `Hello` und `param2` den String `World` enthalten. Beachten Sie, dass Sie `testFun` erneut mit anderen Argumenten aufrufen können und die Parameter dann die Werte der neuen Argumente annehmen.
