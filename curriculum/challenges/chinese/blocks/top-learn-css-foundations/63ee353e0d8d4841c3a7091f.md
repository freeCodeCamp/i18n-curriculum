# --description--

好的，到目前为止你已经讲解了很多。现在唯一剩下的就是讲解如何将所有这些 CSS 添加到你的 `HTML`。有三种方法可以做到这一点。

外部 CSS 是你最常遇到的方法，它涉及为 CSS 创建一个单独的文件，并在超文本标记语言的开头和闭合 `<head>` 标签内使用 `<link>` 元素（这是一个空元素，不需要闭合标签）进行链接：

首先，你在 HTML 文件的开头和结尾的 `<head>` 标签内添加一个空元素 `<link>` 标签。`href` 属性是 CSS 文件的位置，可以是绝对 URL，或者你将使用的相对于 HTML 文件位置的 URL。在上面的示例中，你假设两个文件位于同一目录。`rel` 属性是必需的，它指定 HTML 文件与链接文件之间的关系。

然后在新创建的 `styles.css` 文件中，你有选择器（`div` 和 `p`），后面跟着一对开括弧和闭括弧，这些括弧状的花括弧创建了一个“声明块”。最后，你将任何声明放在声明块内。`color: white;` 是一个声明，其中 `color` 是属性，`white` 是值，而 `background-color: black;` 是另一个声明。

关于文件名的说明：`styles.css` 只是你在这里选择的文件名。只要文件类型是 `.css`，你可以将文件命名为任何你想要的名字，尽管“style”或“styles”是最常用的。

这个方法的几个优点是：

1. 它保持你的 HTML 和 CSS 分离，这使得 HTML 文件更小且看起来更整洁。
2. 你只需在一个地方编辑 CSS，这对于拥有许多页面且都共享相似样式的网站尤其方便。
