# --explanation--

這句話使用了**被動語態**，重點放在接收動作的物件，而不是執行動作的人。例如：  

`The navigation menu was appreciated.`－這表示人們喜歡它，但沒有說明是誰喜歡它。  

在**作用中的語態**中，你會說 `The users appreciated the navigation menu.`－這清楚地顯示了誰執行了動作。

**被動語態** 通常在執行動作的人不明或不重要時使用。另一個例子：

`The report was completed on time.` – 你知道報告已完成，但你不知道或不需要說明是誰完成的。
