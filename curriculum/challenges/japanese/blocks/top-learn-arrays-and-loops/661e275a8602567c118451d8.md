# --description--

もう一つ便利な配列のメソッドは`filter()`メソッドです。`filter()`メソッドは、指定した関数によって実装されたテストに合格したすべての要素を含む新しい配列を作成します。`filter()`メソッドの構文は次の通りです。

```javascript
const numbers = [2, 5, 6, 1, 9, -1]

const newNumbers = numbers.filter((number) => {
  return number > 2;
});

console.log(newNumbers); // Output: [5, 6, 9]
```

`filter()`メソッドは、2より大きいすべての要素を含む新しい配列を作成します。`filter()`メソッドは元の配列を変更しません。
