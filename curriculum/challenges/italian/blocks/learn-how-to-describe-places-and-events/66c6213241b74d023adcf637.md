# --explanation--

Quando si formano domande con il verbo `to be`, l'ordine abituale tra soggetto e verbo si inverte. Questo significa che il verbo precede il soggetto. Per esempio:

- `Your project is connected to Git.` (Affermazione)

- `Is your project connected to Git?` (Domanda)

Nella frase di Sarah, usa questa inversione per chiedere a Tom se il suo progetto è collegato a Git in Eclipse.
