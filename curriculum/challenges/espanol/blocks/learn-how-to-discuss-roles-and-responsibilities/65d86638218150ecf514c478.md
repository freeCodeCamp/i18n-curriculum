# --description--

El caso genitivo (`'s`) se utiliza para mostrar propiedad o una relación entre dos cosas.

Para los sustantivos singulares, agrega `'s` al final del sustantivo para mostrar posesión. Por ejemplo, `the cat's toy` significa el juguete que pertenece al gato.

Para los sustantivos plurales que ya terminan en `-s`, solo agrega un `'` al final. Por ejemplo, `the teachers' lounge` significa el salón para profesores.

Una `instalación` es un lugar donde ocurre una actividad particular, como una fábrica, oficina o escuela.
