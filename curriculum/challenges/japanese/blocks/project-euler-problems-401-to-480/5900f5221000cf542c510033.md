# --description--

ジュリーは妹のルイーズに次の賭けを提案します。

彼女は、誰が皿洗いをするかを決めるために、運試しのゲームをしようと提案します。

このゲームでは、0から1までの一様分布に従う独立した乱数生成器を使います。

ゲームは $S = 0$ から始まります。

最初のプレイヤーであるルイーズは、$S > 1$ になるまで生成器から異なる乱数を $S$ に加え、最後の乱数 '$x$' を記録します。

次のプレイヤーであるジュリーは、$S > 2$ になるまで生成器から異なる乱数を $S$ に加え、最後の乱数 '$y$' を記録します。

より大きい数を出したプレイヤーが勝ち、負けた方が皿洗いをします。つまり、$y > x$ ならば2番目のプレイヤーが勝ちます。

例えば、最初のプレイヤーが 0.62 と 0.44 を引いた場合、$0.62 + 0.44 > 1$ なので最初のプレイヤーのターンは終了し、$x = 0.44$ となります。2番目のプレイヤーが 0.1、0.27、0.91 を引いた場合、$0.62 + 0.44 + 0.1 + 0.27 + 0.91 > 2$ なので2番目のプレイヤーのターンは終了し、$y = 0.91$ となります。$y > x$ なので2番目のプレイヤーが勝ちます。

ルイーズは少し考えて、「それは不公平だ」と反論します。

2番目のプレイヤーが勝つ確率はいくらでしょうか？答えは小数点以下10桁まで、0.abcdefghij の形で答えてください。
