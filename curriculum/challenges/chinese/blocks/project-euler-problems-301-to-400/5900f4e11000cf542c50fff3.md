# --description--

设 $R(M, N)$ 为满足 $M \lt x \le N$、$M \lt y \le N$ 且 $\left\lfloor\frac{y^2}{x^2}\right\rfloor$ 为奇数的格点 ($x$, $y$) 的数量。

我们可以验证 $R(0, 100) = 3\,019$ 和 $R(100, 10\,000) = 29\,750\,422$。

查找 $R(2 \times {10}^6, {10}^9)$。

**注意：** $\lfloor x\rfloor$ 表现为下取整函数。
