# --description--

`Soy BLANK`

## --blanks--

`peruano`

### --feedback--

Diese Nationalität wird für einen Mann aus Perú verwendet. Sie endet auf `o`.

# --explanation--

`Soy` stammt vom Verb `ser`. Es wird verwendet, um dauerhafte Eigenschaften wie Nationalität, Identität und Beruf zu beschreiben. Zum Beispiel:

- `Soy de Perú.` – Ich komme aus Peru.

- `Soy peruana.` – Ich bin Peruanerin. (weiblich)

- `Soy peruano.` – Ich bin Peruaner. (männlich)
