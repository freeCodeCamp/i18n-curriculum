# --description--

이미지는 지적 재산으로 간주되며, 이는 저작권 규정에 의해 보호되고 대부분 창작자에게 속한다는 뜻입니다. 기본적으로 이미지는 모든 권리가 보호되는 상태로 공개됩니다. 창작자나 게시자가 이미지에 대한 모든 저작권을 보유합니다.

즉, 저작권자의 서면 허가를 받거나, 저작권자로부터 라이선스를 구매하거나, 공정 사용에 해당하는 방식으로 이미지를 포함하지 않는 한 웹 페이지에서 사용할 수 없습니다.

세 번째 경우가 다소 까다롭습니다. 공정 사용은 이미지 사용이 제한적이고 변형적이어야 한다는 조건이 있습니다. 예를 들어, 예술 작품에 대해 논평하거나 검토하거나 이미지를 패러디하는 경우가 공정 사용에 해당합니다.

일부 이미지는 Creative Commons 라이선스나 freeCodeCamp가 사용하는 BSD 라이선스 같은 관대한 라이선스 하에 공개될 수 있습니다. 이러한 이미지는 웹사이트에서 사용할 수 있지만, 라이선스를 읽어 사용 시 따라야 할 규칙을 이해해야 합니다. 예를 들어, 웹사이트를 오픈 소스로 만들어야 하거나 이미지를 어떤 방식으로든 수정할 수 없을 수도 있습니다.

마지막으로, 일부 이미지는 퍼블릭 도메인으로 공개될 수 있습니다. 퍼블릭 도메인에 속한 이미지는 저작권이 없으며 아무런 제한 없이 자유롭게 사용할 수 있습니다. Creative Commons 0 라이선스 하에 명시적으로 공개된 이미지는 퍼블릭 도메인으로 간주됩니다.

대부분의 검색 엔진은 라이선스별로 이미지 결과를 필터링할 수 있게 합니다. Pixabay나 Unsplash 같은 사이트도 무료로 사용할 수 있는 이미지를 제공합니다. 웹사이트에서 이미지를 사용할 때는 항상 저작권과 라이선스를 주의하세요.
