# --description--

Sie sagen, etwas ist `definite`, wenn Sie sich dessen sicher sind.

Wenn Sie die Endung `-ly` hinzufügen, wird `definite` zu einem Adverb, `definitely`, das bedeutet, sich einer Sache sicher zu sein oder Vertrauen darin zu haben.

Es wird oft verwendet, um zu betonen, was der Sprecher für wahr hält oder wahrscheinlich für wahr hält. Zum Beispiel:

- `I can definitely see why this is important.` – Dies betont, dass die Wichtigkeit voraussichtlich klar ist.

- `She will definitely finish the project on time.` – Dies zeigt Vertrauen darin, dass sie das Projekt abschließt.
