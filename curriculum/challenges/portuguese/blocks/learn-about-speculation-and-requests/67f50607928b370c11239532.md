# --explanation--

`Evidence of` significa sinais ou prova de que algo aconteceu.Por exemplo:

`The report showed evidence of user error.` - O relatório mostrou que uma pessoa cometeu um erro.

Um `brute force attack` é quando alguém tenta muitas senhas diferentes até que uma funcione.

Uma `SQL injection` é quando alguém tenta invadir um sistema inserindo código malicioso em uma caixa de busca ou formulário.

Jake está falando sobre verificar sinais desses tipos de ataques.
