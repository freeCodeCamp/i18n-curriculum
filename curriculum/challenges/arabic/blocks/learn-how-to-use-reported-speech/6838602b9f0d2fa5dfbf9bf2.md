# --explanation--

`Unusually` ظرف يُستخدم لوصف شيء يحدث أكثر أو أقل مما هو طبيعي أو متوقع. من ناحية أخرى، `unusual` صفة تصف شيئًا غير شائع أو غير طبيعي. على سبيل المثال:

- `The server experienced an unusually high load today.` – في هذه الجملة، `unusually` تتعلق بمدى ارتفاع الحمل (أبعد من الطبيعي).

- `It was unusual for the server to slow down like that.` – في هذه الجملة، `unusual` تشير إلى الحالة. لا تعدل فعلًا أو صفة.

الـ`query` هو سؤال أو طلب يُرسل إلى قاعدة بيانات للحصول على معلومات أو تنفيذ إجراء. على سبيل المثال:

`The application sent a query to the database to fetch user details.` – هذا يعني أن التطبيق طلب من قاعدة البيانات معلومات محددة.
