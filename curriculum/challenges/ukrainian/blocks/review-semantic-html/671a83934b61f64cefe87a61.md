# --interactive--

## Важливість семантичного HTML

- **Структурна ієрархія для заголовків**: Важливо використовувати правильний елемент заголовка, щоб зберегти структурну ієрархію вмісту. Елемент `h1` є найвищим рівнем заголовка, а елемент `h6` — найнижчим рівнем заголовка.
- **Презентаційні HTML-елементи**: Елементи, які визначають зовнішній вигляд вмісту. Наприклад, застарілі елементи `center`, `big` і `font`.
- **Семантичні HTML-елементи**: Елементи, які несуть значення та структуру. Наприклад, `header`, `nav`, `figure`.

## Семантичні HTML-елементи

- **Елемент header**: використовується для визначення заголовка документа або розділу.

:::interactive_editor

  ```html
  <header>
    <h1>CatPhotoApp</h1>
    <p>Welcome to our cat gallery.</p>
  </header>
  ```

:::

- **Елемент main**: використовується для розміщення основного вмісту вебсторінки.

:::interactive_editor

  ```html
  <main>
    <section>
      <h2>Cat Photos</h2>
      <p>Browse adorable cat pictures.</p>
    </section>
  </main>
  ```

:::

- **Елемент section**: використовується для поділу вмісту на менші розділи.

:::interactive_editor

  ```html
  <section>
    <h2>About Me</h2>
    <p>Hi, I am Jane Doe and I am a web developer.</p>
  </section>
  ```

:::

- **Елемент навігаційного розділу (`nav`)**: представляє розділ із навігаційними посиланнями.

:::interactive_editor

  ```html
  <nav>
    <ul>
      <li><a href="#photos">Photos</a></li>
      <li><a href="#videos">Videos</a></li>
    </ul>
  </nav>
  ```

:::

- **Елемент figure**: використовується для розміщення ілюстрацій та діаграм.

:::interactive_editor

  ```html
  <figure>
    <img
      src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"
      alt="Two tabby kittens sleeping together on a couch."
    />
    <figcaption>Cats <strong>hate</strong> other cats.</figcaption>
  </figure>
  ```

:::

- **Елемент наголосу (`em`)**: позначає текст із наголосом.

:::interactive_editor

  ```html
  <p>
    Never give up on <em>your</em> dreams.
  </p>
  ```

:::

- **Елемент ідіоматичного тексту (`i`)**: використовується для виділення альтернативного голосу або настрою, ідіоматичних виразів іншою мовою, технічних термінів і думок.

:::interactive_editor

  ```html
  <p>
    There is a certain <i lang="fr">je ne sais quoi</i> in the air.
  </p>
  ```

:::

Атрибут `lang` всередині відкриваючого тегу `i` використовується для вказівки мови вмісту. У цьому випадку мовою буде французька. Елемент `i` не вказує, чи є текст важливим, він лише показує, що він якимось чином відрізняється від навколишнього тексту.

- **Елемент сильної важливості (`strong`)**: позначає текст із сильною важливістю.

:::interactive_editor

  ```html
  <p>
    <strong>Warning:</strong> This product may cause allergic reactions.
  </p>
  ```

:::

- **Елемент привернення уваги (`b`)**: використовується для привернення уваги до тексту, який не є важливим для значення вмісту. Зазвичай використовується для виділення ключових слів у резюме або назв продуктів у відгуках.

:::interactive_editor

  ```html
  <p>
    We tested several products, including the <b>SuperSound 3000</b> for audio quality, the <b>QuickCharge Pro</b> for fast charging, and the <b>Ecoclean Vacuum</b> for cleaning. The first two performed well, but the <b>Ecoclean Vacuum</b> did not meet expectations.
  </p>
  ```

:::

- **Елемент списку описів (`dl`)**: використовується для представлення списку груп термін-опис.
- **Елемент терміна опису (`dt`)**: використовується для представлення терміна, що визначається.
- **Елемент деталей опису (`dd`)**: використовується для представлення опису терміна.

:::interactive_editor

  ```html
  <dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
  </dl>
  ```

:::

- **Елемент блочного цитування (`blockquote`)**: використовується для представлення розділу, процитованого з іншого джерела. Цей елемент має атрибут `cite`. Значення атрибута `cite` — URL джерела.

:::interactive_editor

  ```html
  <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
    "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
  </blockquote>
  ```

:::

- **Елемент цитування (`cite`)**: використовується для візуального позначення джерела посилання. Позначає заголовок референсу.

:::interactive_editor

  ```html
  <div>
    <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
      "Can you imagine what it would be like to be a successful developer? To have built software systems that people rely upon?"
    </blockquote>
    <p>
      -Quincy Larson, <cite>How to learn to Code and Get a Developer Job [Full Book].</cite>
    </p>
  </div>
  ```

:::

- **Елемент рядкового цитування (`q`)**: використовується для представлення короткої рядкової цитати.

:::interactive_editor

  ```html
  <p>
    As Quincy Larson said,
    <q cite="https://www.freecodecamp.org/news/learn-to-code-book/">
      Momentum is everything.
    </q>
  </p>
  ```

:::

- **Елемент абревіатури (`abbr`)**: використовується для представлення абревіатури або акроніма. Щоб допомогти користувачам зрозуміти, що означає абревіатура або акронім, можна показати її повну форму, опис, зрозумілий людині, за допомогою атрибута `title`.

:::interactive_editor

  ```html
  <p>
    <abbr title="HyperText Markup Language">HTML</abbr> is the foundation of the web.
  </p>
  ```

:::

- **Елемент контактної адреси (`address`)**: використовується для представлення контактної інформації.
- **Елемент дати та/або часу (`time`)**: використовується для представлення дати і/або часу. Атрибут `datetime` використовується для перекладу дат і часу у формат, зрозумілий машині.

:::interactive_editor

  ```html
  <p>
    The reservations are for the <time datetime="20:00">20:00 </time>
  </p>
  ```

:::

- **Атрибут дати ISO 8601 (`datetime`)**: використовується для представлення дат і часу у форматі, зрозумілому машині. Стандартний формат — `YYYY-MM-DDThh:mm:ss`, де велика `T` є роздільником між датою і часом.
- **Елемент верхнього індексу (`sup`)**: використовується для представлення тексту у верхньому індексі. Загальні випадки використання елемента `sup` включають степені, верхні літери та порядкові числівники.

:::interactive_editor

  ```html
  <p>
    2<sup>2</sup> (2 squared) is 4.
  </p>
  ```

:::

- **Елемент нижнього індексу (`sub`)**: використовується для представлення тексту у нижньому індексі. Загальні випадки використання елемента нижнього індексу включають хімічні формули, сноски та індекси змінних.

:::interactive_editor

  ```html
  <p>
    CO<sub>2</sub>
  </p>
  ```

:::

- **Елемент рядкового коду (`code`)**: використовується для представлення фрагмента комп’ютерного коду.
- **Елемент попередньо форматованого тексту (`pre`)**: представляє попередньо форматований текст.

:::interactive_editor

  ```html
  <pre>
    <code>
      body {
        color: red;
      }
    </code>
  </pre>
  ```

:::

- **Елемент неартикульованої анотації (`u`)**: використовується для представлення ділянки рядкового тексту, який слід відтворити так, щоб показати, що він має нетекстову анотацію.

:::interactive_editor

  ```html
  <p>
    You can use the unarticulated annotation element to highlight
    <u>inccccort</u> <u>spling</u> <u>issses</u>.
  </p>
  ```

:::

- **Елемент ruby-анотації (`ruby`)**: використовується для анотування тексту з поясненнями вимови або значення. Приклад використання — східноазіатська типографіка.
- **Елемент запасного ruby-окреслення (`rp`)**: використовується як запасний варіант для браузерів, які не підтримують відображення ruby-анотацій.
- **Елемент ruby-тексту (`rt`)**: використовується для позначення тексту ruby-анотації. Зазвичай використовується для вимови або деталей перекладу у східноазіатській типографіці.

:::interactive_editor

  ```html
  <ruby>
    明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp>
  </ruby>
  ```

:::

- **Елемент закреслення (`s`)**: використовується для позначення вмісту, який більше не є точним або актуальним.

:::interactive_editor

  ```html
  <p>
    <s>Tomorrow's hike will be meeting at noon.</s>
  </p>
  <p>
    Due to unforeseen weather conditions, the hike has been canceled.
  </p>
  ```

:::

- **Внутрішні посилання**: використовуються для посилання на інший розділ сторінки за допомогою `href="#id"` на елементі `a` і надання цільовому елементу того самого `id`. Це часто використовується для пропускних посилань, змісту або довгих сторінок із кількома розділами.

:::interactive_editor

```html
<a href="#about-section">Go to About Section</a>

<section id="about-section">
  <h2>About</h2>
  <p>This is the about section of the page.</p>
</section>
```

:::
