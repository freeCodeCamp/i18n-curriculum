# --explanation--

当你知道循环应运行多少次时，`for loop` 非常有用。例如：

- `A for loop is useful for going through a list.` - 你可以用它来查看列表中的每一项。

- `The for loop stops when it reaches the last item.` - 当没有更多项时，循环结束。

当重复次数取决于条件时，`while loop` 更好。例如：

- `A while loop runs until a condition is false。` - 只要规则为真，循环就会继续。

- `Be careful, a while loop can run forever!` - 如果你不停止它，循环将不会结束。
