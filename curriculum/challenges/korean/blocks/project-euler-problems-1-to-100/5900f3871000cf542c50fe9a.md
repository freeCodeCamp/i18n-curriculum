# --description--

오일러는 놀라운 이차 공식 하나를 발견했습니다:

<div style='margin-left: 4em;'>$n^2 + n + 41$</div>

이 공식은 연속된 정수 값 $0 \\le n \\le 39$에 대해 40개의 소수를 만들어냅니다. 하지만 $n = 40$일 때, $40^2 + 40 + 41 = 40(40 + 1) + 41$은 41로 나누어지고, $n = 41$일 때는 $41^2 + 41 + 41$이 명백히 41로 나누어집니다.

또한, $n^2 - 79n + 1601$이라는 놀라운 공식이 발견되었는데, 이 공식은 연속된 값 $0 \\le n \\le 79$에 대해 80개의 소수를 만들어냅니다. 계수 −79와 1601의 곱은 −126479입니다.

다음과 같은 형태의 이차식을 고려하세요:

<div style='margin-left: 4em;'>
  $n^2 + an + b$, 여기서 $|a| < range$이고 $|b| \le range$<br>
  여기서 $|n|$은 $n$의 나머지/절댓값입니다<br>
  예: $|11| = 11$ 그리고 $|-4| = 4$<br>
</div>

$n = 0$부터 시작하여 연속된 $n$ 값에 대해 가장 많은 소수를 만들어내는 이차식의 계수 $a$와 $b$의 곱을 찾으세요.
