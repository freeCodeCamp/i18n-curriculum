# --interactive--

Quando lavori con le stringhe in JavaScript, ci sono molti casi in cui potresti aver bisogno di verificare se una stringa contiene una sottostringa specifica, cioè una parte più piccola di quella stringa.

Ad esempio, potresti voler controllare se l'input di un utente include una parola o un carattere specifico prima di eseguire qualche azione. Un modo per farlo è usare il metodo `includes()`.

Il metodo `includes()` serve a verificare se una stringa contiene una sottostringa specifica. Se la sottostringa viene trovata all'interno della stringa, il metodo restituisce `true` altrimenti restituisce `false`.

Ecco la sintassi di base:

```js
string.includes(searchValue);
```

Nella sintassi, `searchValue` è la sottostringa che vuoi cercare all'interno della stringa. Ecco un esempio:

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("awesome");

console.log(result);  // true
```

:::

In questo esempio, la parola `awesome` è presente all'interno della stringa `JavaScript is awesome!`, quindi il metodo `includes()` restituisce `true`.

È importante notare che il metodo `includes()` fa distinzione tra maiuscole e minuscole. Questo significa che è richiesta una corrispondenza esatta dei caratteri, compreso il loro caso.

Ad esempio:

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("Awesome");

console.log(result);  // false
```

:::

Poiché `Awesome` (con una `A` maiuscola) non corrisponde a `awesome` (con una `a` minuscola), il risultato è `false`.

Puoi anche usare il metodo `includes()` per cercare una sottostringa a partire da un indice specifico nella stringa fornendo un secondo parametro:

:::interactive_editor

```js
let text = "Hello, JavaScript world!";
let result = text.includes("JavaScript", 7);

console.log(result);  // true
```

:::

Qui, la ricerca della sottostringa `JavaScript` inizia dalla posizione 7 della stringa, assicurandosi di saltare tutti i caratteri prima di questa posizione.

Il metodo `includes()` restituisce solo un risultato `true` o `false`. Non fornisce informazioni su dove si trova la sottostringa nella stringa o quante volte si presenta. Se ti serve quel livello di dettaglio, altri metodi, come il metodo `indexOf()`, potrebbero essere più adatti.
