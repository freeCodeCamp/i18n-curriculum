### --feedback--

Queste due parole indicano che vorresti qualcosa di più, mostrando una scelta o priorità cortese. La prima parola è la contrazione di `we would`. La prima lettera della prima parola è maiuscola.

`since`

### --feedback--

Questa parola si usa per dare una ragione o una spiegazione di qualcosa.

# --explanation--

`We'd prefer` è un modo cortese per dire cosa vorresti di più. Per esempio:

`We'd prefer faster processors for the new computers.` – Questo mostra un interesse più forte per opzioni migliori.

`Since` si usa per spiegare la ragione dietro una decisione o situazione. Per esempio:

`We upgraded the servers since the old ones were too slow.` – Questo mostra la causa per fare l'aggiornamento.
