# --description--

短语 **`Soy` + [nationality]** 是用西班牙语表达你的国籍的最常见方式。它相当于 "I am + [nationality]"。

这个短语用作介绍时对问题 `¿Cuál es tu nacionalidad?` 的直接回答，其中 `nacionalidad` 意思是“nationality”。

在西班牙语中，国籍名称除非出现在句首，否则不大写。它们还必须与说话者的性别一致。例如：

| Country | Nationality (Feminine) | Nationality (Masculine) |
| :---: | :---: | :---: |
| **`México`** | `mexicana` | `mexicano` |
| **`Argentina`** | `argentina` | `argentino` |
| **`Perú`** | `peruana` | `peruano` |
| **`Uruguay`** | `uruguaya` | `uruguayo` |
| **`El Salvador`** | `salvadoreña` | `salvadoreño` |
