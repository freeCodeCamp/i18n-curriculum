# --explanation--

`Layout` si riferisce al design o alla struttura di uno spazio. Per esempio:  

`He shared the layout of the new office.` – Indica il piano di dove vanno le cose.

`Draft` è una versione iniziale di un documento. Per esempio:  

`She wrote the first draft of the proposal.` – Può ancora essere modificato.

`Send it over` significa consegnare qualcosa. Per esempio:  

`I'll send it over once it's ready.` – Significa condividere un file o un messaggio.

`To` indica per chi è l’azione. Per esempio:  

`She gave the file to her manager.` – Mostra chi lo riceve.
