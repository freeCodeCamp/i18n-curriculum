# --description--

You say something is `definite` when you are certain about it.

Cuando agregas el sufijo `-ly`, `definite` se convierte en un adverbio, `definitely`, que significa estar seguro sobre algo o estar confiado al respecto.

A menudo se utiliza para enfatizar lo que el hablante cree que es verdadero o probablemente sea verdadero. Por ejemplo:

- `I can definitely see why this is important.` - This emphasizes that the importance is expected to be clear.

- `She will definitely finish the project on time.` - This shows confidence in her completing the project.
