# --explanation--

`To review` означає подивитися або проаналізувати щось, щоб краще зрозуміти або перевірити на помилки.

Сара має на увазі, що команда переглядає свій прогрес і будь-які проблеми, щоб зрозуміти поточну ситуацію і, можливо, зробити покращення. Наприклад:

`We need to review the report before submitting it.` – Це означає, що команда перевірить звіт на помилки або покращить його перед відправленням.
