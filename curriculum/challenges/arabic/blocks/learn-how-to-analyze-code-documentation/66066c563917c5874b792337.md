# --description--

`API` (واجهة برمجة التطبيقات - Application Programming Interface) هو مجموعة من القواعد والأدوات لبناء التطبيقات البرمجية. يشبه الجسر الذي يتيح لبرامج البرمجيات المختلفة التواصل مع بعضها البعض. على سبيل المثال، `The weather app uses an API to get data from the weather service.`

`approach` تعني طريقة أو أسلوب في التفكير أو القيام بشيء ما. مثل، `Her approach to solving the problem was very creative.`
