# --interactive--

숫자를 다루면서 값을 1씩 증가시키거나 감소시켜야 할 때, 증가 연산자와 감소 연산자가 작업을 더 쉽게 만들어줍니다. 간단하게 살펴보겠습니다.

증가 연산자와 감소 연산자는 각각 `++`와 `--`로 나타냅니다. 둘 다 변수의 값을 `1`만큼 조정할 수 있게 해줍니다.

`x = x + 1`나 `x = x - 1`처럼 쓰는 대신, `x++`를 사용해 `1`를 더하거나, `x--`를 사용해 `1`를 빼면 됩니다. 더 빠르고, 깔끔하며, 읽기 쉽습니다.

증가 연산자와 감소 연산자가 작동하는 방식에는 반전이 있습니다: 접두사와 접미사 두 가지 형태가 있으며, 차이는 값이 언제 업데이트되는지에 있습니다. 증가 연산자의 경우, 접두사는 `++x`이고 접미사는 `x++`입니다.

접두사(`++x`)는 변수의 값을 먼저 증가시키고, 그 다음 새 값을 반환합니다. 접미사(`x++`)는 변수의 현재 값을 먼저 반환하고, 그 다음 값을 증가시킵니다:

:::interactive_editor

```js
let x = 5;

console.log(++x); // 6
console.log(x); // 6
```

:::

위 코드에서 `++x`는 "먼저 `x`를 증가시키고, 그다음 사용한다"는 뜻입니다. 그래서 `++x`를 출력하면 즉시 증가된 값인 `6`를 얻습니다.

이제 접미사를 사용한 예를 살펴보겠습니다:

:::interactive_editor

```js
let y = 5;

console.log(y++); // 5
console.log(y); // 6
```

:::

이 예에서 `y++`는 "먼저 `y`를 사용하고, 그다음 증가시킨다"는 뜻입니다. `y++`를 출력하면 `5`가 나오지만, 그 코드 줄 이후 `y`는 `6`가 됩니다.

감소 연산자는 증가 연산자와 같은 방식으로 작동하지만, 값이 `1`만큼 감소합니다. 마찬가지로 두 가지 형태가 있습니다: 접두사(`--x`)는 변수의 값을 먼저 감소시키고 새 값을 반환합니다. 접미사(`x--`)는 현재 값을 먼저 반환하고 그다음 감소시킵니다:

:::interactive_editor

```js
let x = 5;
console.log(--x); // 4
console.log(x); // 4

let y = 5;
console.log(y--); // 5
console.log(y); // 4
```

:::

그렇다면, 접두사와 접미사 중 어느 것을 사용해야 할까요? 대부분 경우에는 어느 쪽을 써도 상관없습니다. 둘 다 작업을 수행합니다. 하지만 값을 즉시 표현식에서 사용해야 한다면 차이가 중요해집니다. 다음 예제를 보세요:

:::interactive_editor

```js
let a = 5;
let b = ++a;
console.log(b); // 6 (a was incremented before assignment)

let c = 5;
let d = c++;
console.log(d); // 5 (c was incremented after assignment)
```

:::

즉, 업데이트된 값이 바로 필요하면 접두사를 사용하세요. 현재 값이 먼저 필요하고 증가가 나중에 중요하다면 접미사를 사용하세요.
