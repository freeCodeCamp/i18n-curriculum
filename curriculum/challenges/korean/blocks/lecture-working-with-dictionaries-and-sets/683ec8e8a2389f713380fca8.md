# --description--

사전의 키-값 쌍에 접근하여 처리해야 할 때 사전을 순회할 수 있습니다. 이는 값들을 업데이트하거나 어떤 로직을 적용할 때 유용합니다.

사용할 수 있는 몇 가지 기법을 살펴보겠습니다.

모든 제품과 그 가격을 연결하는 `products` 사전이 있다고 가정해 봅시다:

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,
    'Headphones': 70,
}
```

모든 제품에 20% 할인을 제공하려면 모든 키-값 쌍을 순회하며 가격을 수정할 수 있습니다.

이 기법에 필수적인 `.values()`, `.keys()`, `.items()` 메서드를 이전 수업에서 간단히 다뤘습니다.

이들은 사전의 값, 키, 키-값 쌍을 가진 뷰 객체를 반환합니다. 이 뷰 객체들을 `for` 반복문에서 요소를 순회하는 데 사용할 수 있습니다.

예를 들어, 사전의 모든 값을 다음과 같이 순회할 수 있습니다.

`for`, 반복 변수(`price`인 경우), `products.values()`를 작성하여 `products` 사전의 모든 값을 가져오고, 콜론을 쓴 뒤 반복문의 본문에서 값에 어떤 로직을 적용할 수 있습니다. 여기서는 값을 출력하고 있습니다.

반복 변수는 각 값을 하나씩 차례로 받습니다:

```python
for price in products.values():
    print(price)
```

출력 결과는 다음과 같습니다. 각 값이 콘솔에 하나씩 출력됩니다:

```md
990
600
250
70
```

키를 순회해야 할 때도 `.keys()`에 대해 똑같이 작동합니다. `products.keys()` 또는 `products`을 직접 순회하고 반복 변수에 설명적인 이름을 할당하면 됩니다:

```python
for product in products.keys():
    print(product)

# Or

for product in products:
    print(product)
```

출력 결과입니다. 각 키가 콘솔에 하나씩 출력됩니다:

```md
Laptop
Smartphone
Tablet
Headphones
```

키와 해당 값을 동시에 순회해야 할 때도 똑같이 작동합니다. `products.items()`를 순회하면 됩니다:

```python
for product in products.items():
    print(product)
```

출력 결과입니다. 키와 해당 값을 가진 개별 튜플을 받습니다:

```md
('Laptop', 990)
('Smartphone', 600)
('Tablet', 250)
('Headphones', 70)
```

키와 값을 별도의 반복 변수에 저장하려면, 변수를 정의하고 쉼표로 구분하면 됩니다. 그런 다음 반복문 본문에서 사용할 수 있습니다.

여기서는 `product` 반복 변수와 `price` 반복 변수를 정의했습니다. 각각 해당 값을 담습니다. 순서가 중요하며, 키가 먼저이고 값이 그 다음입니다:

```python
for product, price in products.items():
    print(product, price)
```

출력 결과입니다. 나란히 출력했지만, 코드에서 필요에 따라 이 값을 사용할 수 있습니다.

```md
Laptop 990
Smartphone 600
Tablet 250
Headphones 70
```

이제 이 내용을 알았으니 처음 예제로 돌아가 봅시다. 20% 할인을 제공하려면 각 가격에 `0.8`를 곱하고 그 값을 해당 제품 키의 값으로 다시 할당하면 됩니다.

정수로 작업하려면 결과를 내림하여 반올림할 수도 있습니다:

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,
    'Headphones': 70,
}

for product, price in products.items():
    products[product] = round(price * 0.8)

print(products)
```

그런 다음 사전을 출력하면 할인된 가격이 적용된 키-값 쌍을 얻습니다:

```python
{
    'Laptop': 792, 
    'Smartphone': 480, 
    'Tablet': 200, 
    'Headphones': 56
}
```

마지막으로, 키-값 쌍을 순회하면서 카운터를 추적해야 한다면 `enumerate()` 함수를 호출할 수 있습니다. 이 카운터는 반복문 내에서 해당 요소의 "인덱스" 또는 "개수" 역할을 합니다.

이 함수는 `enumerate` 객체를 반환하며, 각 키-값 쌍에 정수를 할당하는 카운터 역할을 합니다. 카운터는 어떤 숫자부터 시작할 수 있지만 기본적으로 0부터 시작합니다.

여기서는 `products` 사전의 키를 순회하고 있습니다:

```python
for product in enumerate(products):
    print(product)
```

하지만 `enumerate()` 함수도 각 키에 정수를 할당하므로, 정수와 키가 포함된 튜플을 얻습니다.

출력 결과는 다음과 같습니다:

```md
(0, 'Laptop')
(1, 'Smartphone')
(2, 'Tablet')
(3, 'Headphones')
```

필요하다면 이 값을 별도의 반복 변수에 할당할 수 있습니다. 여기서는 두 개의 반복 변수(`index`과 `product`)를 사용했습니다. `enumerate()`를 사용할 때 흔히 볼 수 있는 방식입니다:

```python
for index, product in enumerate(products):
    print(index, product)
```

값을 순회해야 한다면 `products`을 `products.values()`로 바꾸면 됩니다:

```python
for price in enumerate(products.values()):
    print(price)
```

출력 결과는 각 튜플에 인덱스와 가격이 포함됩니다:

```md
(0, 990)
(1, 600)
(2, 250)
(3, 70)
```

별도의 반복 변수에 할당할 수도 있습니다:

```python
for index, price in enumerate(products.values()):
    print(index, price)
```

출력 결과입니다. 코드에서 필요에 따라 사용할 수 있습니다:

```md
0 990
1 600
2 250
3 70
```

`products.items()`를 사용하면 "인덱스" 또는 "카운터"뿐 아니라 전체 키-값 쌍도 얻을 수 있습니다:

```python
for index, product in enumerate(products.items()):
    print(index, product)
```

이 예에서는 인덱스 다음에 해당 키-값 쌍의 키와 값을 포함하는 튜플이 옵니다:

```md
0 ('Laptop', 990)
1 ('Smartphone', 600)
2 ('Tablet', 250)
3 ('Headphones', 70)
```

카운트의 초기 값을 사용자 지정하려면 `enumerate()`에 두 번째 인수를 전달하면 됩니다. 예를 들어, 여기서는 카운트를 1부터 시작합니다:

```python
for index, product in enumerate(products.items(), 1):
    print(index, product)
```

출력에서 이 변화를 확인할 수 있습니다. 이제 첫 번째 정수가 0이 아니라 1입니다:

```md
1 ('Laptop', 990)
2 ('Smartphone', 600)
3 ('Tablet', 250)
4 ('Headphones', 70)
```

지금까지 본 어떤 변형에도 이 방법이 작동합니다. 초기 숫자를 두 번째 인수로 전달하기만 하면 됩니다.

사전을 순회하는 기법은 다양합니다. 여기 소개한 방법들은 흔히 쓰이는 것들이며, 프로젝트에 가장 적합한 방법을 선택하면 됩니다.
