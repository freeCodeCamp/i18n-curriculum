# --explanation--

表达式 `stuck on` 用于描述某人无法继续转发一个任务或解决一个问题。它暗示被困住或无法进展。例如：

- `We were stuck on this code error for hours.` - 我们无法继续，直到找到代码中的那个误差。

- `She got stuck on the final question of the exam.` - 她在最后一道题上花了很长时间。

在对话框中，Lisa 使用 `stuck on` 来描述他们在解决工作中的错误时无法取得进展。
