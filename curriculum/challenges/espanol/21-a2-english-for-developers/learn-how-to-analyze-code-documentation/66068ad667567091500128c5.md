# --description--

Una `library` es como una gran estantería con mucho código que los programadores pueden usar. Piénsalo como un libro de cocina con muchas recetas (funciones) que puedes usar para cocinar (programar) diferentes platos (proyectos).

Por ejemplo, una `library` en programación podría tener una función para hacer gráficos. En lugar de escribir código desde cero, puedes usar esta función, como seguir una receta para hacer un pastel.
