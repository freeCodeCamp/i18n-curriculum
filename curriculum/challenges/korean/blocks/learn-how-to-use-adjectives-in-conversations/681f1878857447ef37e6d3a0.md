# --explanation--

`Not only ... but also ...`는 두 관련된 진술이 모두 사실임을 강조하는 일반적인 구조로, 두 번째 진술이 더 놀랍거나 중요할 때 자주 쓰입니다. 예를 들어:

`She is not only skilled but also very creative.` – 그녀가 두 가지 강점을 가지고 있으며, 둘 다 중요하다는 뜻입니다.

`Adaptable`는 상황에 따라 쉽게 변하거나 조정될 수 있음을 의미합니다. 예를 들어:

`This program is adaptable to different types of teams.` – 다양한 환경에서 잘 작동한다는 뜻입니다.
