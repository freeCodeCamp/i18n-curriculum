# --description--

`Uncertain` significa não ter certeza sobre algo, como sentir-se inseguro sobre uma decisão.O oposto é `certain`, que significa estar certo.Por exemplo, `I am certain this is the right way` significa estar certo sobre a direção.

`Achieve` significa alcançar com sucesso um objetivo após esforço, como `achieve a good result in a test`.Em programação, `achieve` frequentemente se refere aos objetivos que um programador quer alcançar com seu código.
