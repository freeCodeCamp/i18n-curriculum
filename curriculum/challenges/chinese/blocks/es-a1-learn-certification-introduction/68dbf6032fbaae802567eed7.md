# --description--

如果你将使用计算机来学习西班牙语课程，你需要确保它能够输入重音符号和特殊字符，比如 `ñ`。 

无论你使用 Windows 还是 Mac，设置这个都很快。只需几分钟。

## 在 Windows 11 上设置

1. 从开始菜单打开 **设置**。

2. 进入 **时间和语言**，然后点击 **语言和区域**。

3. 转到 **首选语言** 并选择 **添加语言**。

4. 查找 **Spanish** 并选择一个变体，例如 `Español (Latin America)`。

5. 点击 **Next** ，然后按照步骤操作。

语言安装完成后：  

- 在你的**首选语言**列表中查找。

- 点击省略号以显示更多选项并选择**语言选项**。

- 进入 **Keyboards**，点击 **Add a keyboard** 并选择你喜欢的西班牙语版型。

在键盘之间切换：  

- 按 **Windows 密钥 + 空格键**。

- 或者点击任务栏上的语言图标（通常标记为 **ENG**）并选择西班牙语。

## 在 Mac 上设置

1. 打开 **Apple 菜单** 并选择 **System Settings**。

2. 转到 **Keyboard**。

3. 转到 **文本输入** > **输入源**。

4. 点击 **Edit...**

5. 点击底部的 **plus (+)** 按钮并从 `list` 中选择 **Spanish**。

6. 点击 **添加**。

现在你已准备好完成西班牙语课程和所有任务，包括那些需要重音符号和特殊字符的任务。
