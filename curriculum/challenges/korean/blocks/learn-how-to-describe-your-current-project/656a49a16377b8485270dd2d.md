# --description--

`What are you up to?`는 누군가의 현재 활동이나 계획에 대해 알고 싶을 때 사람들이 사용하는 편안하고 친근한 질문입니다. 또한 누군가가 지금 무엇을 하고 있을지 궁금해하는 의미도 담고 있습니다. 예를 들어:

`What are you up to this weekend?` — 이 문장은 누군가의 주말 계획에 대해 묻습니다.

마찬가지로, `up to`라는 것은 무언가를 하고 있다는 뜻입니다. 예를 들어:

`He is up to something.` — 이것은 그가 어떤 활동을 하고 있다는 의미로, 종종 호기심이나 의심의 뉘앙스를 포함합니다.
