# --interactive--

Під час створення своїх програм ви часто захочете, щоб щось відбулося з затримкою або повторювалося через певні інтервали. Саме тут стають у пригоді методи `setTimeout()` і `setInterval()`. Обидва методи приймають два параметри: функцію та затримку.

Давайте розглянемо кожен із методів детальніше, щоб ви могли почати їх використовувати у своїх вебпроєктах.

`setTimeout()` дозволяє відкласти виконання дії на певний час. Ось базовий синтаксис методу `setTimeout()`:

```js
setTimeout(functionToRun, delay);
```

`functionToRun` — це код, який ви хочете виконати після вказаної затримки, а `delay` — час (у мілісекундах), який потрібно почекати перед запуском функції. Ось приклад:

:::interactive_editor

```js
setTimeout(function () {
 console.log("This runs after 3 seconds");
}, 3000);
```

:::

Ви також можете використати стрілкову функцію для функції зворотного виклику, ось так:

:::interactive_editor

```js
setTimeout(() => {
 console.log("This runs after 3 seconds"); 
}, 3000);
```

:::

`setInterval()` повторює виконання функції через задані інтервали. Це як будильник, який спрацьовує кожні кілька секунд. `setInterval()` ідеально підходить, коли ви хочете, щоб щось відбувалося знову і знову, наприклад, оновлення даних або створення анімації, що рухається безперервно.

Як і `setTimeout()`, він приймає функцію та затримку:

```js
setInterval(functionToRun, delay);
```

`functionToRun` — це код, який ви хочете запускати повторно, а `delay` — час у мілісекундах між кожним виконанням цієї функції. Ось приклад:

:::interactive_editor

```js
setInterval(() => {
 console.log("This runs every 2 seconds");
}, 2000);

// Result:
// This runs every 2 seconds
// This runs every 2 seconds
// This runs every 2 seconds
// ...
```

:::

Оскільки `setInterval()` продовжує виконувати надану функцію через задані інтервали, можливо, ви захочете його зупинити. Для цього потрібно використати метод `clearInterval()`.

Для роботи `clearInterval()` приймає ідентифікатор `setInterval()`, який ви хочете зупинити. Це може бути змінна, якій ви призначили інтервал. Один зі способів зупинити інтервал — зробити це всередині `setTimeout()`, адже це зупинить інтервал після певного часу:

:::interactive_editor

```js
const intervalID = setInterval(() => {
 console.log("This will stop after 5 seconds");
}, 1000);

setTimeout(() => {
 clearInterval(intervalID);
}, 5000);
```

:::

Також можна зупинити таймаут за допомогою методу `clearTimeout()`:

:::interactive_editor

```js
let timeoutID = setTimeout(() => {
 console.log("This will not run");
}, 5000);

clearTimeout(timeoutID);
```

:::

З цим кодом `setTimeout()` взагалі не виконається. Тож більш елегантний спосіб — через DOM, додавши слухача події click на кнопку, щоб зупинити таймаут:

:::interactive_editor

```html
<h1>Cancel Timeout Example</h1>
<button id="cancelButton">Cancel Timeout</button>
<script src="index.js"></script>
```

```js
let timeoutID = setTimeout(() => {
  console.log("This will run if not canceled");
}, 5000);

document
.querySelector("#cancelButton")
.addEventListener("click", () => {
  clearTimeout(timeoutID);
  console.log("Timeout canceled!");
});
```

:::
