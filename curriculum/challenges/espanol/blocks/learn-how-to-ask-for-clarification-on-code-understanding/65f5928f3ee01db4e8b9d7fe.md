# --description--

En esta tarea, aprenderás sobre la frase `to be willing`.

Cuando alguien dice que está `willing` a hacer algo, significa que está listo y preparado para hacerlo, a menudo por amabilidad o cooperación. Por ejemplo, `Brian is willing to help Sophie` significa que Brian está listo y feliz de asistir a Sophie.

Esta frase se usa a menudo en contextos profesionales y personales para mostrar una actitud positiva hacia hacer algo o ayudar a alguien.
