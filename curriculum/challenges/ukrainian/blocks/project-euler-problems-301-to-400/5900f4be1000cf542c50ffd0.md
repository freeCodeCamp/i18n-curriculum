# --description--

Нехай $\\{a_1, a_2, \ldots, a_n\\}$ — послідовність цілих чисел довжини $n$, така що:

- $a_1 = 6$
- для всіх $1 ≤ i &lt; n$ : $φ(a_i) &lt; φ(a_{i + 1}) &lt; a_i &lt; a_{i + 1}$

$φ$ позначає функцію Ейлера.

Нехай $S(N)$ — кількість таких послідовностей, для яких $a_n ≤ N$.

Наприклад, $S(10) = 4$: {6}, {6, 8}, {6, 8, 9} та {6, 10}.

Ми можемо перевірити, що $S(100) = 482\\,073\\,668$ і $S(10\\,000)\bmod {10}^8 = 73\\,808\\,307$.

Знайдіть $S(20\\,000\\,000)\bmod {10}^8$.
