# --description--

تلعب أليس وبوب لعبة Nim Square.

لعبة Nim Square تشبه تمامًا لعبة Nim العادية ذات الثلاث حزم، لكن اللاعبين يمكنهم فقط إزالة عدد مربع من الحجارة من حزمة واحدة.

عدد الحجارة في الحزم الثلاثة يُمثّل بالثلاثي المرتب ($a$, $b$, $c$).

إذا كان $0 ≤ a ≤ b ≤ c ≤ 29$ فإن عدد المواقع الخاسرة للاعب التالي هو 1160.

احسب عدد المواقع الخاسرة للاعب التالي إذا كان $0 ≤ a ≤ b ≤ c ≤ 100\\,000$.
