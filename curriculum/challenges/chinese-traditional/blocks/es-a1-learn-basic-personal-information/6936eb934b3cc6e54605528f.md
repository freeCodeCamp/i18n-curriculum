# --description--

在此任務中，你將聽到 Camila 向 Mateo 詢問一項特定的個人資訊。

注意結構 `¿Cuál es tu` + 名詞？這個結構用來詢問特定的個人資訊。這個結構是西班牙語中詢問個人資訊最標準的方式之一。  

# --instructions--

聆聽音訊並回答以下問題。

# --explanation--

Camila 說 `¿Cuál es tu correo electrónico?` 結構遵循這個樣式：

`¿Cuál es tu` + 名詞？

此結構用於要求特定的個人資訊。在此情況下，`correo electrónico` 指的是用於線上傳送和接收訊息的電子郵件位址。
