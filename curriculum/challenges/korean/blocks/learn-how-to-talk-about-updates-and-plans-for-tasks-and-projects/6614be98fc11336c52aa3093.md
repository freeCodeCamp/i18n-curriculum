# --description--

`slide deck`는 보통 PowerPoint나 Google Slides 같은 소프트웨어로 만든 발표용 슬라이드 모음입니다. 발표 내용 일부를 담은 카드 더미의 디지털 버전과 같습니다.

예를 들어, `We prepared a slide deck for the meeting to showcase our project's progress.`

`standard`라는 용어는 일반적으로 사용되거나 표준으로 인정되는 것을 의미합니다. 여기서 `standard format`는 널리 사용되고 인정받는 형식을 뜻합니다. 예를 들어, `Using bullet points for key points is a standard practice in presentations.`
