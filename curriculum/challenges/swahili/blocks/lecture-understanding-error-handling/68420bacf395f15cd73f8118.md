# --description--

Unapoandika msimbo wa Python, ni kawaida kukutana na hitilafu. Kuelewa hitilafu hizi ni muhimu kwa kurekebisha makosa ya msimbo wako kwa haraka na kwa ufanisi. Ujumbe huu unakuambia hasa kilichotokea kosa, ikiwa unajua jinsi ya kuusoma.

Hitilafu za kawaida za Python ni `SyntaxError`, `NameError`, `TypeError`, `IndexError`, na `AttributeError`. Hizi hutokea wakati Python haielewi msimbo wako, au wakati mantiki yako hailingani na data unayotumia.

Hapa kuna mfano wa `SyntaxError`:

```py
print("Hello, world!"
# SyntaxError: unexpected EOF while parsing
```

Mstari huu unakosa mkato wa kufunga mabano. Python huibua `SyntaxError` kwa sababu msimbo haufuati sheria sahihi za sintaksia.

Hapa kuna mfano wa `NameError`:

```py
print(name)
# NameError: name 'name' is not defined
```

Unajaribu kuchapisha kigezo ambacho hakijafafanuliwa bado. Python huibua `NameError` wakati haipati kigezo kwa jina hilo.

Hapa kuna mfano wa `TypeError`:

```py
5 + "5"
# TypeError: unsupported operand type(s) for +: 'int' and 'str'
```

Huwezi kuongeza nambari kamili na mfuatano wa herufi pamoja. Python huibua `TypeError` unapo jaribu kufanya operesheni kwenye aina za data zisizolingana.

Hapa kuna mfano wa `IndexError`:

```py
my_list = [1, 2, 3]
print(my_list[5])
# IndexError: list index out of range
```

Unajaribu kupata kielezo ambacho hakipo kwenye orodha. Python huibua `IndexError` unapoenda nje ya mipaka.

Hapa kuna mfano wa `AttributeError`:

```py
num = 42
num.append(5)
# AttributeError: 'int' object has no attribute 'append'
```

Kielezo cha `int` hakina njia ya `append()`. Python huibua `AttributeError` unapo jaribu kutumia njia au sifa ambayo haipo kwa aina hiyo ya data.

Kutambua ujumbe wa kawaida wa hitilafu za Python kunakusaidia kutatua matatizo kwa haraka. Badala ya kubahatisha, soma ujumbe wa hitilafu kwa makini, mara nyingi unakuambia hasa kilichotokea kosa na wapi pa kuangalia.
