# --explanation--

`Several`는 몇 명 이상이지만 모두는 아니라는 뜻입니다. 예를 들어:

`Several users reported issues with the new software.` – 두 명 이상의 사용자가 문제를 겪었지만 모두는 아니라는 의미입니다.

`To target`는 누군가나 무언가를 목표로 하거나 향한다는 뜻입니다. 예를 들어:

`The hackers targeted customers by sending fake emails.` – 고객을 특정해서 속이려 했다는 의미입니다.
