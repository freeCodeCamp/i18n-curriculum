# --description--

Dentro de su elemento `head`, agregue un elemento `link` con el atributo `rel` configurado en `stylesheet` y el atributo `href` configurado en `https://fonts.googleapis.com/css?family=Open+Sans:400,700,800`.

Esto importará la familia de fuentes `Open Sans`, con los valores de peso de fuente `400`, `700` y `800`.

También agregue un elemento `link` para vincular su archivo `styles.css`.

# --hints--

Su código debe tener dos elementos `link`.

```js
assert.strictEqual(document.querySelectorAll('link').length, 2);
```

Sus elementos `link` deben ser elementos vacíos, no deben tener una etiqueta de cierre `</link>`.

```js
assert.notMatch(code, /<\/link>/);
```

Sus dos elementos `link` deben estar dentro del elemento `head`.

```js
const headContentRegex = /(?<=<head\s*>)[\S|\s]*(?=<\/head\s*>)/;
const headElementContent = code.match(headContentRegex);

const headElement = document.createElement("head");
headElement.innerHTML = headElementContent;
assert.strictEqual(headElement.querySelectorAll('link').length, 2);
```

Ambos elementos `link` deben tener el atributo `rel` configurado en `stylesheet`.

```js
const links = [...document.querySelectorAll('link')];
assert.isTrue(links.every(link => link.getAttribute('rel') === 'stylesheet'));
```

Uno de sus elementos `link` debe tener un atributo `href` configurado en `styles.css`.

```js
const styleElement = document.querySelector('[data-href]');
assert.isNotNull(styleElement);
```

Uno de sus elementos `link` debe tener un atributo `href` configurado en `https://fonts.googleapis.com/css?family=Open+Sans:400,700,800`.

```js
const links = [...document.querySelectorAll('link')];
assert.exists(links.find(link => link?.getAttribute('href') === 'https://fonts.googleapis.com/css?family=Open+Sans:400,700,800'));
```
