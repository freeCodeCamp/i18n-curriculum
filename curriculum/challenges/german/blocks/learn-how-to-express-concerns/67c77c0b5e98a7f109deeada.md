# --explanation--

`I don't think it's an option` hat die gleiche Bedeutung wie `I think it's not an option`. Beide zeigen an, dass Sie etwas nicht für eine mögliche Wahl halten. Zum Beispiel:

`I think delaying the project is not an option.` – Das bedeutet, dass eine Verzögerung des Projekts wahrscheinlich nicht eintreten wird.
