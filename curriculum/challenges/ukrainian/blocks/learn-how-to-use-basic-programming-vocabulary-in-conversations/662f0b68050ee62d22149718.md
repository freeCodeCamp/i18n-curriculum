# --description--

Фраза `to keep in mind` означає пам’ятати або враховувати щось. Її часто використовують, коли хтось дає пораду або важливу інформацію, яку слід запам’ятати для подальшого використання.

Наприклад, якщо хтось каже: «Remember to check the weather before hiking.» Ви можете відповісти `I'll keep that in mind`, що означає, що ви пам’ятатимете про цю пораду.
