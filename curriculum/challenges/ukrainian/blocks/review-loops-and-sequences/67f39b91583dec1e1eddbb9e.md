## Списки в Python

- **Вступ**: У Python тип даних список — це впорядкована послідовність елементів, які можуть складатися з рядків, чисел або навіть інших списків. Списки змінні та мають індексацію, починаючи з нуля.

```python
cities = ['Los Angeles', 'London', 'Tokyo']
```

- **Доступ до елементів списку**: Щоб отримати доступ до елемента списку `cities`, ви можете звернутися до його індексу в послідовності:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # Los Angeles
```

- **Доступ до елементів за допомогою від’ємної індексації**: Щоб отримати доступ до останнього елемента будь-якого списку, можна використати `-1` як номер індексу:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # Tokyo
```

- Від’ємна індексація використовується для доступу до елементів, починаючи з кінця списку, а не з початку за індексом `0`.

- **Створення списків за допомогою конструктора `list()`**: Списки також можна створювати за допомогою конструктора `list()`. Конструктор `list()` використовується для перетворення ітерованого об’єкта на список:

```python
developer = 'Jessica'

print(list(developer)) 
# Result: ['J', 'e', 's', 's', 'i', 'c', 'a']
```

- **Знаходження довжини списку**: Ви можете використати функцію `len()`, щоб отримати довжину списку:

```python
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

- **Змінність списків**: Списки є змінними, тобто ви можете оновити будь-який елемент у списку, якщо передати дійсний номер індексу. Щоб оновити список за певним індексом, можна призначити нове значення цьому індексу:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

- **Помилка виходу за межі індексу**: Якщо ви передасте індекс (позитивний або від’ємний), який виходить за межі списку, ви отримаєте `IndexError`:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
"""
```

- **Видалення елементів зі списку**: Елементи можна видалити зі списку за допомогою ключового слова `del`:

```python
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

- **Перевірка наявності елемента у списку**: Ключове слово `in` можна використати, щоб перевірити, чи існує елемент у списку:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **Вкладені списки**: Списки можуть бути вкладені всередину інших списків:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

- Щоб отримати доступ до вкладеного списку, потрібно звернутися до нього за індексом `2`, оскільки списки мають індексацію, починаючи з нуля.

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

- Щоб далі отримати доступ до другої мови з цього вкладеного списку, потрібно звернутися до неї за індексом `1`:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # Rust
```

- **Розпакування значень зі списку:** Розпакування значень зі списку — це техніка, яка використовується для призначення значень зі списку новим змінним. Ось приклад розпакування списку `developer` у нові змінні `name`, `age` і `job` таким чином:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer
```

- Якщо кількість змінних зліва від оператора присвоєння не співпадає з загальною кількістю елементів у списку, ви отримаєте `ValueError`.

- **Збір залишкових елементів зі списку**: Щоб зібрати будь-які залишкові елементи зі списку, можна використати оператор зірочки (`*`) ось так:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer
```

- **Зрізи списків**: Зріз — це концепція доступу до частини списку за допомогою оператора зрізу `:`. Щоб зробити зріз списку, який починається з індексу `1` і закінчується перед індексом `3`, використовуйте такий синтаксис:

```python
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie']
desserts[1:3] # ['Cookies', 'Ice Cream']
```

- **Кроки інтервалу**: Також можна вказати крок інтервалу, який визначає, на скільки збільшувати індекси. Ось приклад, якщо ви хочете отримати список лише парних чисел за допомогою зрізу:

```python
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

## Методи списків

- **append()**: Використовується, щоб додати елемент у кінець списку. Ось приклад використання методу `append()` для додавання числа `6` до списку `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **Додавання списків**: Метод `append()` також можна використати, щоб додати один список у кінець іншого:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

- **extend()**: Використовується для додавання кількох елементів у кінець списку. Ось приклад додавання чисел `6`, `8` і `10` у кінець списку `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

- **insert()**: Використовується для вставлення елемента на певний індекс у списку. Ось приклад використання методу `insert()`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

- **remove():** Використовується для видалення елемента зі списку. Метод `remove()` видалить лише перше входження елемента у списку:

```py
numbers = [1, 2, 3, 4, 5, 5, 5]
numbers.remove(5)

print(numbers) # [1, 2, 3, 4, 5, 5]

```

- **pop()**: Використовується для видалення конкретного елемента зі списку з одночасним його поверненням:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

- Якщо не вказати елемент для методу `pop`, буде видалено останній елемент.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

- **clear()**: Використовується для видалення всіх елементів зі списку:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

- **sort()**: Метод `sort()` використовується для сортування елементів на місці. Ось приклад сортування випадкового списку `numbers` на місці:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

- **sorted()**: Вбудована функція, яка повертає новий відсортований список замість зміни оригінального списку:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
print(numbers) # [19, 2, 35, 1, 67, 41]
```

- **reverse()**: Використовується для обертання порядку елементів у списку:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **index()**: Використовується, щоб знайти перший індекс, де можна знайти елемент у списку:

```python
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

- Якщо елемент не можна знайти за допомогою методу `index()`, результатом буде `ValueError`.

## Кортежі в Python

- **Визначення**: Кортеж — це тип даних Python, який використовується для створення впорядкованої послідовності значень. Кортежі можуть містити змішаний набір типів даних:

```py
developer = ('Alice', 34, 'Rust Developer')
```

- Кортежі незмінні, тобто елементи в кортежі не можна змінити після створення. Якщо ви спробуєте оновити один із елементів кортежу, отримаєте `TypeError`:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **Доступ до елементів кортежу**: Щоб отримати доступ до елемента кортежу, використовуйте дужкову нотацію та номер індексу:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

- Від’ємна індексація може бути використана для доступу до елементів, починаючи з кінця кортежу:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- Якщо ви спробуєте передати індекс, який перевищує або дорівнює довжині кортежу, отримаєте `IndexError`:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- Кортеж також можна створити за допомогою конструктора `tuple()`. У конструктор можна передати різні ітеровані об’єкти, такі як рядки, списки і навіть інші кортежі.

```python
developer = 'Jessica'

print(tuple(developer)) 
# Result: ('J', 'e', 's', 's', 'i', 'c', 'a')
```

- **Перевірка елементів у кортежі**: Щоб перевірити, чи є елемент у кортежі, можна використати ключове слово `in` ось так:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **Розпакування кортежів**: Елементи можна розпакувати з кортежу таким чином:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

- Якщо потрібно зібрати залишкові елементи з кортежу, можна використати оператор зірочки (`*`) ось так:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer
```

- **Зрізи кортежів**: Зрізи можна використати, щоб витягти частину кортежу. Наприклад, елементи `pie` і `cookies` можна вирізати в окремий кортеж:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

- **Видалення елементів з кортежів**: Видалення елемента з кортежу викличе `TypeError`, оскільки кортежі незмінні:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

- **Коли використовувати кортеж замість списку?**: Якщо вам потрібна динамічна колекція елементів, де можна додавати, видаляти та оновлювати елементи, слід використовувати список. Якщо ви знаєте, що працюєте з фіксованою та незмінною колекцією даних, слід використовувати кортеж.

## Загальні методи кортежів

- **`count()`**: Використовується, щоб визначити, скільки разів елемент зустрічається в кортежі. Наприклад, можна перевірити, скільки разів мова `'Rust'` зустрічається в кортежі:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

- Якщо вказаний елемент у методі `count()` зовсім відсутній у кортежі, повернене значення буде `0`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

- Якщо до методу `count()` не передано аргументів, Python поверне `TypeError`.

- **index()**: Використовується, щоб знайти індекс, де знаходиться певний елемент у кортежі. Ось приклад використання методу `index()` для пошуку індексу мови `'Java'`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

- Якщо вказаний елемент не знайдено, Python поверне `ValueError`.

- Ви можете передати необов’язковий початковий індекс у метод `index()`, щоб вказати, з якого місця почати пошук елемента в кортежі:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

- Ви також можете передати необов’язковий кінцевий індекс у метод `index()`, щоб вказати, де припинити пошук елемента в кортежі:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

- **`sorted()`**: Використовується для сортування елементів будь-якого ітерованого об’єкта та повертає новий відсортований список. Ось приклад створення нового списку чисел за допомогою функції `sorted()`:

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

- **Зміна поведінки сортування**: Ви можете налаштувати поведінку сортування для ітерованого об’єкта, використовуючи необов’язкові аргументи `reverse` і `key`. Ось приклад використання аргументу `key` для сортування елементів кортежу за довжиною:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

- Ви можете створити новий список значень у зворотному порядку, використовуючи аргумент `reverse` ось так:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

## Цикли в Python

- **Визначення**: Цикли використовуються для повторення блоку коду певну кількість разів.

- **Цикл `for`**: Використовується для ітерування над послідовністю (наприклад, списком, кортежем або рядком) і виконання блоку коду для кожного елемента цієї послідовності. Ось приклад використання циклу `for` для ітерування списком і виведення кожної мови в консоль:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)

"""
Result 

Rust
Java
Python
C++
"""
```

- Ось приклад використання циклу `for` для проходження рядком `code` і виведення кожного символу:

```py
for char in 'code':
    print(char)

"""
Result 

c
o
d
e
"""
```

- Цикли `for` можна вкладати. Ось приклад використання вкладеного циклу `for`:

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)

"""
Result

Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
"""

```

- **Цикл `while`**: Повторює блок коду, доки умова `False`. Ось приклад використання циклу `while` для гри в угадайку:

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')

"""
Result

Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
"""

```

- **Інструкції `break` і `continue`**: Використовуються в циклах для зміни виконання циклу.

- Інструкція `break` використовується, щоб негайно вийти з циклу, коли виконується певна умова. Ось приклад використання інструкції `break` для списку `developer_names`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

- Інструкція `continue` використовується, щоб пропустити поточну ітерацію і перейти до наступної. Ось приклад використання інструкції `continue` замість інструкції `break`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

- Цикли `for` і `while` можна поєднувати з умовою `else`, яка виконується лише тоді, коли цикл не був завершений інструкцією `break`:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

## Функція range і її використання в циклах

- **Функція `range()`**: Використовується для генерації послідовності цілих чисел.

```py
range(start, stop, step)
```

- Обов’язковий аргумент `stop` — це ціле число (не включно), яке представляє кінцеву точку послідовності чисел, що генерується. Ось приклад використання функції `range()`:

```py
for num in range(3):
    print(num)
```

- Якщо аргумент `start` не вказано, за замовчуванням він буде `0`. За замовчуванням послідовність цілих чисел збільшується на `1`. Ви можете використати необов’язковий аргумент `step`, щоб змінити значення кроку за замовчуванням. Ось приклад генерації послідовності парних чисел від 2 до 10 включно (не включно 11):

```py
for num in range(2, 11, 2):
    print(num)
```

- Якщо не передати жодних аргументів функції `range()`, ви отримаєте `TypeError`.

- Функція `range()` приймає лише цілі числа як аргументи, а не числа з плаваючою комою. Використання чисел з плаваючою комою також призведе до `TypeError`:

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- Ви можете використати від’ємне ціле число для аргументу `step`, щоб згенерувати послідовність цілих чисел у спадному порядку:

```py
for num in range(40, 0, -10):
    print(num)
```

- Функцію `range()` також можна використати для створення списку цілих чисел, застосувавши її разом із конструктором `list`. Конструктор `list` використовується для перетворення ітерованого об’єкта на список. Ось приклад генерації списку парних чисел від 2 до 10 включно:

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

## Функції `enumerate()` і `zip()` у Python

- **`enumerate()`**: використовується для ітерування послідовністю з одночасним відстеженням індексу кожного елемента. Функція `enumerate()` приймає ітерований об’єкт як аргумент і повертає об’єкт `enumerate`, який складається з індексу та значення кожного елемента ітерованого об’єкта.

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')

# Result
# Index 0 and language Spanish
# Index 1 and language English
# Index 2 and language Russian
# Index 3 and language Chinese

```

- Функцію `enumerate()` також можна використати поза циклом `for`:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

print(list(enumerate(languages)))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

- Функція `enumerate()` також приймає необов’язковий аргумент `start`, який задає початкове значення для лічильника. Якщо цей аргумент опущено, лічильник починається з `0`.

-  **`zip()`** : Використовується для ітерування кількома ітерованими об’єктами паралельно. Ось приклад використання функції `zip()` для ітерування над `developers` і `ids`:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')


"""
Result

Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
"""

```

## Генератори списків у Python

- **Визначення**: Генератор списку дозволяє створити новий список в одному рядку, поєднуючи цикл і умову безпосередньо в квадратних дужках. Це робить код коротшим і часто легшим для читання.

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

## Методи ітерованих об’єктів

- **`filter()`**: Використовується для фільтрації елементів ітерованого об’єкта на основі умови. Повертає ітератор, який містить лише елементи, що задовольняють умову. Ось приклад створення нового списку лише зі словами довжиною більше чотирьох символів:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**: Використовується для застосування функції до кожного елемента ітерованого об’єкта і повертає новий ітерований об’єкт з результатами. Ось приклад використання функції `map()` для перетворення списку температур у Цельсіях у Фаренгейти:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**: Використовується для отримання суми з ітерованого об’єкта, наприклад списку або кортежу. Ось приклад використання функції `sum()`:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- Ви також можете передати необов’язковий аргумент `start`, який задає початкове значення для суми. Ось оновлений приклад із використанням аргументу `start` як позиційного аргументу:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- Ви також можете використати аргумент `start` як ключовий аргумент ось так:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## Lambda-функції

- **Визначення**: Lambda-функція в Python — це короткий спосіб створити функцію без імені (анонімну функцію). 
- Lambda-функції часто використовуються як аргумент для іншої функції. Ось приклад lambda-функції:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

- Найкращі практики використання lambda-функцій включають не призначати їх змінним, тримати їх простими та читабельними, а також використовувати для коротких одноразових функцій.
