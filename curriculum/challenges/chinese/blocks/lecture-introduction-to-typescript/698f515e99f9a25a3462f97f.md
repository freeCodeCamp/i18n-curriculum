# --description--

在本课中，你将学习如何使用函数类型。让我们从参数类型注解开始。

这是一个基本 `function` 的示例：

```ts
function circleArea(radius) {
  return Math.PI * radius * radius;
}
```

现在，TypeScript 会显示一条 `Parameter 'radius' implicitly has an 'any' type` 的消息。`any` 类型表示一个值可以具有任何类型。你将在以后的课程中学习更多关于此类型的内容。

为了消除误差并显式地为参数指定类型，你可以这样指定：

```ts
function circleArea(radius: number) {
  return Math.PI * radius * radius;
}
```

如果你尝试用非数字的不同 `type` 调用 `circleArea` 函数，TypeScript 会显示如下误差：

```ts
function circleArea(radius: number) {
  return Math.PI * radius * radius;
}

// Argument of type 'string' is not assignable to parameter of type 'number'
circleArea("3");
```

如果你尝试提供比函数预期更多的参数，TypeScript 会显示一个误差：

```ts
function circleArea(radius: number) {
  return Math.PI * radius * radius;
}

// Expected 1 arguments, but got 2.
circleArea(3, 4);
```

如果你提供的参数过少，那么 TypeScript 也会显示一个误差：

```ts
function circleArea(radius: number) {
  return Math.PI * radius * radius;
}

// Expected 1 arguments, but got 0.
// An argument for 'radius' was not provided.
circleArea();
```

你可以通过在参数附近使用 `?` 符号使参数变为可选的，像这样：

```ts
function area(radius: number, height?: number): number {
  const baseArea = Math.PI * radius * radius;

  if (height !== undefined) {
    const sideArea = 2 * Math.PI * radius * height;
    return baseArea + sideArea;
  }

  return baseArea;
}
```

函数内部的条件式用于查看是否未提供 `height` 参数。 

如果你想为你的返回值添加类型注解，可以像这样添加返回类型注解：

```ts
function circleArea(radius: number): number {
  return Math.PI * radius * radius;
}
```

返回类型注解位于参数列表之后。类似于变量类型注解，大多数情况下 TypeScript 会智能地推导返回类型。因此，当你想让函数的意图更清晰或防止对返回值的意外更改时，请使用显式的返回类型注解。

下一个示例将处理异步的函数和 promises。如果你的函数将返回一个 promise，你可以像这样使用 `Promise` 类型：

```ts
async function getCityTemperature(city: string): Promise<number> {
  const response = await fetch(`https://api.example.com/weather?city=${city}`);
  const data = await response.json(); 
  return data.temperature;
}
```

在这种情况下，如果你尝试移除返回 `type`，TypeScript 可能会推导返回为 `Promise<any>`。所以这将是一个为返回值显式指定 `type` 的好情况。

到目前为止，我们只看过具名函数的示例。但类型如何与匿名函数一起工作呢？

下面是一个示例，展示如何在传递给 `forEach` 的回调中使用匿名函数：

```ts
const languages = ["JavaScript", "Python", "TypeScript"];

languages.forEach((lang) => {
  console.log(`${lang.toUpperCase()} has ${lang.length} characters`);
});
```

`lang` 参数没有类型注解。相反，TypeScript 使用上下文类型指派，这意味着它从出现的上下文中推导类型。
