# --explanation--

`To prevent` означає вжити заходів, щоб щось не сталося. Наприклад:

`They installed new software to prevent viruses.` – Це означає, що програмне забезпечення допомагає зупинити віруси до того, як вони спричинять проблеми.

`incident` — це несподівана подія, часто від’ємна або проблемна. Наприклад:

`There was a security incident last night that shut down the server.` – Це означає, що сталася подія, яка спричинила проблеми з системою.
