# --interactive--

CSS flexbox 是一種一維佈局模型，允許你在容器內以列和行排列元素。你也可以控制它們的順序和方向。網頁開發者使用它來創建響應式網站和網頁應用程式，以適應不同的螢幕尺寸和方向。我們稱 flexbox 為一維佈局模型，因為它專注於一次沿單一軸線排列元素。該軸線可以是水平或垂直方向。 

在你開始使用 flexbox 之前，有兩個你應該了解的關鍵概念：flex 容器和 flex 項目。

flex 容器是一個具有 flex 佈局的 HTML 元素。你可以在 flex 容器內以多種方式排列和對齊元素。若要將 HTML 元素設為 flex 容器，你需要為它的 CSS 樣式添加 `display: flex`。

Flex 項目是 flex 容器的直接子元素。這些元素可以根據 flex 容器的屬性進行排列和對齊。它們也可以縮小或擴展以適應可用空間。

這是一個包含 `main` 容器和三個子 `div` 元素的範例：

```html
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

如果你只在 CSS 樣式表中設定這些 `div` 元素的 `width`、`height` 和 `background-color`，每個子元素都會被放置在自己的列上，因為容器預設不是 flex。 

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

但是如果你將 `display: flex` 添加到 `main` 容器，`div` 元素將會重新排列以適應同一列，並且在必要時會縮小（要查看預覽，請啟用互動式編輯器）：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

預設情況下，`flex container` 將會是區塊層級元素，因此該容器本身會相對於其他元素和容器位於自己的列上。

既然你已經更了解 flex 容器和 flex 項目，你也應該知道 flex 屬性。這些屬性決定了 flex 項目在 flex 容器中如何排列、調整大小和分配。其中一些最常用的是 `flex-direction`、`justify-content`、`align-items` 和 `flex-wrap`。

很好。現在讓我們稍微談談 flex 模型。此模型定義了 flex 項目如何在 flex 容器中排列。每個 flex 容器都有兩個軸：

- 主軸。
- 交叉軸。

這些軸的方向決定了不同屬性如何影響彈性項目的佈局和分佈。預設情況下，彈性容器的主軸是水平的，交叉軸是垂直的。彈性項目會沿著主軸方向排列。交叉軸與主軸垂直。

`flex-direction` 屬性設定主軸的方向。`flex-direction` 的預設值是 `row`，會將所有的彈性項目放在同一列，方向依照你的瀏覽器預設語言（從左到右或從右到左）：

```css
flex-direction: row; /* Default */
```

要反轉該列中的項目，你可以使用 `flex-direction: row-reverse`：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: row-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

這將反轉 flex 項目的順序。如果你想要改為垂直對齊 flex 項目，只需要在 flex 容器中將 `flex-direction` 設定為 `column`。這將改變主軸的方向：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

現在 `div` 元素將會垂直對齊，因為主軸將會是垂直方向，而交叉軸將會是水平方向。 

你也可以使用 `flex-direction: column-reverse` 垂直反轉彈性項目的順序：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

CSS flexbox 是一種強大的版面配置模型，提供在容器內排列元素的靈活且高效的方法。透過理解 flex 容器、flex 項目及各種 flex 屬性的關鍵概念，你可以創建動態且響應式的網站，適應不同螢幕尺寸和方向。
