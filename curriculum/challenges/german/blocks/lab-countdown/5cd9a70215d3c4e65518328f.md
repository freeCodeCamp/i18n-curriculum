# --description--

In diesem Labor bauen Sie eine `countdown`-Funktion, die eine Startzahl entgegennimmt und eine Countdown-Liste erzeugt.

Wenn Sie `countdown(n)` aufrufen, sollte sie ein Array zurückgeben, das bei `n` beginnt und in Schritten von `1` herunterzählt, bis es `1` erreicht.

Zum Beispiel:

- `countdown(5)` sollte `[5, 4, 3, 2, 1]` zurückgeben
- `countdown(1)` sollte `[1]` zurückgeben

Wenn die Startzahl kleiner als `1` ist, gibt es nichts herunterzuzählen, also geben Sie ein leeres Array zurück:

- `countdown(0)` sollte `[]` zurückgeben
- `countdown(-3)` sollte `[]` zurückgeben

Um das Labor abzuschließen, müssen Sie das Ergebnis mit Rekursion erstellen und dürfen keine Schleifen (`for`, `while`), `Array.from()` oder Array-Iteration-Methoden (`forEach`, `map`, `filter`, `reduce`) verwenden. Jeder Aufruf sollte sein eigenes Ergebnis-Array erzeugen (verwenden Sie keine globalen Variablen, um den Countdown zu speichern).

**Ziel:** Erfüllen Sie die untenstehenden User Stories und bestehen Sie alle Tests, um das Labor abzuschließen.

**User Stories:**

1. Sie sollten eine Funktion namens `countdown` erstellen.
2. Die `countdown`-Funktion sollte ein einzelnes Argument `n` (eine Zahl) entgegennehmen.
3. Wenn `n` kleiner als `1` ist, sollte `countdown` ein leeres Array zurückgeben.
4. Andernfalls sollte `countdown` ein Array mit den ganzen Zahlen von `n` abwärts bis `1` in absteigender Reihenfolge zurückgeben.
5. Sie müssen Rekursion verwenden (die Funktion sollte sich selbst aufrufen) und dürfen keine Schleifen jeglicher Art verwenden (`for`, `while`, `Array.from()` oder höherwertige Methoden wie `forEach`, `map`, `filter`, `reduce`).
6. Jeder rekursive Aufruf sollte einen kleineren Wert als der vorherige verwenden (zum Beispiel `n - 1`), damit die Funktion ihren Basisfall erreicht.
7. Die Lösung darf keine globalen Variablen verwenden, um das Ergebnis-Array zu speichern oder zwischenzuspeichern.
8. Mehrmaliges Aufrufen von `countdown` mit unterschiedlichen Eingaben sollte immer das korrekte, unabhängige Ergebnis zurückgeben.
