# --explanation--

`Works fine` significa que algo está funcionando corretamente, sem problemas.Por exemplo:

`The program works fine.` – Isso significa que o programa está funcionando sem nenhum problema.

`Weird` descreve algo que é estranho ou não normal. `Acts weird` significa que o comportamento é incomum ou não o que se espera.Por exemplo:

`The software acts weird when I try to open a file.` – Isso significa que o software se comporta de forma estranha quando você tenta realizar essa ação.
