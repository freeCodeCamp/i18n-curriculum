# --description--

다음 의사난수 생성기로 만들어진 정수 수열 $S_n$을 생각하세요:

$$\begin{align}
        S_0 & = 290\\,797 \\\\
  S_{n + 1} & = {S_n}^2\bmod 50\\,515\\,093
\end{align}$$

$i ≤ j$일 때, $A(i, j)$는 $S_i, S_{i + 1}, \ldots, S_j$ 중 최솟값입니다. $M(N) = \sum A(i, j)$이며, 여기서 $1 ≤ i ≤ j ≤ N$입니다.

$M(10) = 432\\,256\\,955$와 $M(10\\,000) = 3\\,264\\,567\\,774\\,119$임을 확인할 수 있습니다.

$M(2\\,000\\,000\\,000)$을 구하세요.
