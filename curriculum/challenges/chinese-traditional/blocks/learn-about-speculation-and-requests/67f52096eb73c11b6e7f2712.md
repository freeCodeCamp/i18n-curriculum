### --feedback--

這個兩字詞指的是某人實際進入受限區域，而非遠端駭入系統。

# --explanation--

`Physical access` 是指進入某個地方，例如辦公室或伺服器室。

當有人未經許可進入安全地點時，會發生 `physical access breach`。例如：

- 為避免實體的存取違規，我們應該在入口處要求身分證明檢查。－ 公司可以透過在門口檢查身分證來防止未經授權的進入。

- `A physical access breach can happen if someone enters the server room without permission.` - 如果有人未經授權進入伺服器室，可能會發生實體的安全問題。
