# --description--

箱に青いディスクが15枚、赤いディスクが6枚、合計21枚のカラーディスクが入っているとします。この中からランダムに2枚取り出したとき、2枚とも青いディスクである確率は次のようになります。

$${P(BB)} = \frac{15}{21}×\frac{14}{20} = \frac{1}{2}$$

2枚とも青いディスクをランダムに取り出す確率がちょうど50％になる次のような配列は、青いディスクが85枚、赤いディスクが35枚入った箱です。

合計が`limit`枚を超える最初の配列を見つけ、その箱に入っている青いディスクの枚数を求めてください。

```js
function arrangedProbability(limit) {
  // Based on https://www.mathblog.dk/project-euler-100-blue-discs-two-blue/
  let blue = 15;
  let discs = 21;

  while (discs < limit) {
    const nextBlue = 3 * blue + 2 * discs - 2;
    const nextDiscs = 4 * blue + 3 * discs - 3;

    blue = nextBlue;
    discs = nextDiscs;
  }
  return blue;
}
```
