# --explanation--

`hard drive`는 문서, 소프트웨어, 시스템 파일 같은 데이터를 저장하는 컴퓨터의 저장소 장치입니다. 컴퓨터가 작동하고 정보를 영구적으로 저장하는 데 필수적입니다. 예를 들어:

- `I saved the project files on my hard drive.` - 프로젝트 파일을 컴퓨터 저장소에 저장했습니다.

- `The hard drive crashed, and we lost some data.` - 저장소 장치가 고장 나서 팀이 정보를 잃었습니다.
