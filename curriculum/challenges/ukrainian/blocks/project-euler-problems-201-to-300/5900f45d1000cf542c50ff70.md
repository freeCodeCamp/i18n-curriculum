# --description--

Для натурального числа $n$ позначимо $σ(n)$ суму всіх дільників числа $n$, наприклад, $σ(6) = 1 + 2 + 3 + 6 = 12$.

Досконале число, як ви, мабуть, знаєте, — це число, для якого $σ(n) = 2n$.

Визначимо частку досконалості для натурального числа як $p(n) = \frac{σ(n)}{n}$.

Знайдіть суму всіх натуральних чисел $n ≤ {10}^{18}$, для яких $p(n)$ має форму $k + \frac{1}{2}$, де $k$ — ціле число.
