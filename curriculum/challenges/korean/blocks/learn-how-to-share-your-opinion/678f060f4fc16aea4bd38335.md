# --explanation--

`Thinking of`는 아이디어, 행동, 계획을 고려할 때 사용합니다. 예를 들어:

`I'm thinking of buying a new laptop.` - 이것은 새 노트북을 사는 옵션을 고려하고 있지만 아직 결정하지 않았다는 뜻입니다.

`Cloud`는 저장소와 같은 컴퓨팅 자원을 장치에 로컬로 저장하는 대신 인터넷을 통해 제공하는 시스템을 말합니다. 예를 들어:

`We use a cloud system to back up our files online.` - 이것은 파일이 물리적 하드 드라이브나 서버가 아닌 온라인 서비스에 저장된다는 뜻입니다.
