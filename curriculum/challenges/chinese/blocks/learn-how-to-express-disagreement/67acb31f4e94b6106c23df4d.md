# --explanation--

`True, but…` 允许你在同意某人观点的同时添加不同的观点。这个短语在讨论中非常有用，当你想认可一个有效观点的同时提出你自己的参数时。例如：

- 人物 1：`AI tools can speed up coding.`

- 第 2 个人：`True, but they still make mistakes that need human review.`——此人同意 AI 有助于编码，但指出它并不完美，仍然需要人工监督。
