# --explanation--

Слово `Salvadoreña` — це жіноча форма національності людини зі Сальвадору.

Національності мають узгоджуватися за родом із людиною, яку вони описують. Наприклад:

| Рід | Національність | Приклад |
| :--- | :--- | :--- |
| **Жіночий** | `Salvadoreña` | `Ella es salvadoreña.` ("Вона сальвадорка".) |
| **Чоловічий** | `Salvadoreño` | `Él es salvadoreño.` ("Він сальвадорець".) |
