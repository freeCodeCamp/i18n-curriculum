# --description--

모바일 기기로 스페인어 커리큘럼을 진행할 때는 `á`, `é`, `ñ` 같은 악센트 부호와 특수 문자를 입력할 준비가 되어 있는지 확인하세요.

## 빠른 누르고 있기 방법  

악센트가 있는 글자를 입력하는 가장 간단한 방법은 필요한 글자를 누르고 있는 것입니다.

1. 키보드가 표시되면, 예를 들어 `a` 또는 `n` 글자 키를 누르고 계세요.

2. `á` 또는 `ñ` 같은 악센트가 있는 글자들이 팝업 메뉴로 나타납니다.

3. 원하는 글자 위로 손가락을 슬라이드한 후 떼어 선택하세요.

가끔 악센트가 필요할 때 아주 편리합니다.

## 언어 키보드 추가하기  

또 다른 방법은 스페인어 키보드 레이아웃을 추가하는 것입니다. 이렇게 하면 악센트와 다른 문자에 더 빠르게 접근할 수 있어 유용합니다.

### 안드로이드에서  

**참고:** 안드로이드 설정 경로는 기기에 따라 다를 수 있습니다.

1. 설정을 엽니다.

2. 시스템(또는 기기에 따라 일반 관리)으로 이동합니다.

3. 키보드 설정(또는 언어 및 입력)을 탭하세요.

4. 언어 및 유형(또는 키보드 추가)을 탭하세요.

5. Spanish (US)를 선택하거나 입력 언어 관리에서 Spanish (US)를 선택하세요.

6. 저장하거나 요청 시 완료를 탭하세요.

### 아이폰에서

1. 설정을 엽니다.

2. 일반 > 키보드로 이동합니다.

3. 키보드를 탭하세요.

4. 새 키보드 추가를 탭하세요.

5. Spanish (Latin America)를 선택하세요.

6. 키보드의 지구본 아이콘을 사용해 언어를 전환하세요.

## 서드파티 키보드  

스마트 제안, 다국어 지원, 악센트 입력을 더 빠르게 하는 기능 등 편리한 기능을 제공하는 다양한 앱을 설치해 스페인어 키보드를 쉽게 사용할 수도 있습니다.

이제 스페인어 커리큘럼을 진행하며 특히 악센트가 필요한 과제도 문제없이 완료할 준비가 되었습니다.
