# --description--

另一种使用选择器的方法是将它们作为一个没有任何分隔的列表链式连接。假设你有以下 `HTML`：

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection preview">This is where a preview for a post might go.</p>
</div>
```

你有两个带有 `subsection` 类的元素，它们有某种独特的样式，但如果你只想为同时具有 `header` 作为第二个类的元素应用单独的规则怎么办？那么，你可以像这样在你的 CSS 中将两个 `class` 选择器链接起来：

```css
.subsection.header {
  color: red;
}
```

`.subsection.header` 所做的是选择同时具有 `subsection` 和 `header` 类的任何元素。注意 `.subsection` 和 `.header` `class` 选择器之间没有任何空格。此语法基本上适用于链接任何组合的选择器，但不适用于链接多个类型选择器。

这也可以用于链接一个类和一个 ID，如下所示：

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection" id="preview">This is where a preview for a post might go.</p>
</div>
```

你可以将上面的两个元素与以下内容结合起来：

```css
.subsection.header {
  color: red;
}

.subsection#preview {
  color: blue;
}
```

通用情况下，你不能链接多个类型选择器，因为一个元素不能同时是两种不同的类型。例如，链接两个类型选择器如 `div` 和 `p` 会得到选择器 `divp`，这不会起作用，因为该选择器会尝试查找字面 `<divp>` 元素，而该元素并不存在。

给定一个 `id` 为 `title` 且 `class` 为 `primary` 的元素，如何为单个规则同时使用这两个属性？

## --answers--

```css
.title.primary {
  ...
}
```

---

```css
.title > primary {
  ...
}
```

---

```css
#title.primary { 
  ...
}
```
