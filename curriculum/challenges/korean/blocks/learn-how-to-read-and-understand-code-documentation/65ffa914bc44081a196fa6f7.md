# --description--

`To skim`는 세부 사항에 포커스를 맞추지 않고 주요 아이디어를 빠르게 읽거나 훑어보는 것을 의미합니다. 예를 들어, `She skimmed the article to understand the general topic.`

`To skim through`도 비슷한 의미로, 텍스트나 문서를 빠르게 훑어보는 것을 뜻합니다. 예를 들어, `He skimmed through the manual to find the troubleshooting section.`
