# --explanation--

Прийменник `on` може мати багато значень, але в цьому завданні його слід використовувати, щоб показати предмет або тему, яку обговорюють чи над якою працюють.

`Working on the new software update` означає, що команда зосереджує свої зусилля та увагу на конкретному проєкті або темі.

Інші речення використовують `on` для позначення часу (`on the weekend`, `on time`) або місця/пристрою (`on his computer`), що є іншими значеннями.
