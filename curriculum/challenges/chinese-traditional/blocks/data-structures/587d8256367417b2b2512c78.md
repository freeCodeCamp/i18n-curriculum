# --description--

另一種表述圖形的方法是將它放入 <dfn>鄰接矩陣</dfn>。<dfn>鄰接矩陣</dfn> 是一個二維 (2D) 陣列，其中每個巢狀陣列的元素數量與外層陣列相同。換句話說，它是一個數字的矩陣或網格，這些數字表述邊。

**注意**：矩陣上方和左側的數字只是節點的標籤。矩陣內的 1 表示表述列和行所代表的頂點（節點）之間存在邊。最後，0 表示沒有邊或關係。

<pre>
    1 2 3
  \------
1 | 0 1 1
2 | 1 0 0
3 | 1 0 0
</pre>

上方是一個非常簡單的無向圖，裡面有三個節點，其中第一個節點連接到第二和第三個節點。下方是相同內容的 JavaScript 實作。

```js
var adjMat = [
  [0, 1, 1],
  [1, 0, 0],
  [1, 0, 0]
];
```

與鄰接列表不同，矩陣的每一「列」必須具有與圖中節點數相同的元素數量。這裡我們有一個三乘三的矩陣，表示我們的圖中有三個節點。有向圖看起來會類似。以下是一個圖，其中第一個節點有一條邊指向第二個節點，然後第二個節點有一條邊指向第三個節點。

```js
var adjMatDirected = [
  [0, 1, 0],
  [0, 0, 1],
  [0, 0, 0]
];
```

圖形的邊也可以有<dfn>權重</dfn>。到目前為止，我們有<dfn>無權重</dfn>的邊，邊的存在與否是二進位的（`0` 或 `1`）。你可以根據你的應用有不同的權重。

# --instructions--

創建一個具有五個節點的無向圖的鄰接矩陣。此矩陣應該存在於多維 `array` 中。這五個節點之間的關係為第一節點與第四節點、第一節點與第三節點、第三節點與第五節點，以及第四節點與第五節點。所有邊的權重均為一。
