# --description--

Para un entero positivo $n$, sea $σ(n)$ la suma de todos los divisores de $n$, por ejemplo, $σ(6) = 1 + 2 + 3 + 6 = 12$.

Un número perfecto, como probablemente sabes, es un número con $σ(n) = 2n$.

Definamos el cociente de perfección de un número entero positivo como $p(n) = \frac{σ(n)}{n}$.

Encuentra la suma de todos los enteros positivos $n ≤ {10}^{18}$ para los cuales $p(n)$ tiene la forma $k + \frac{1}{2}$, donde $k$ es un entero.
