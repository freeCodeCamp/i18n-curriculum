# --description--

任意の正の整数 $n$ に対して、第 $n$ 弱いグッドシュタイン列 $\\{g1, g2, g3, \ldots\\}$ は次のように定義されます。

- $g_1 = n$
- $k > 1$ のとき、$g_k$ は $g_{k - 1}$ を基数 $k$ で表し、それを基数 $k + 1$ の数として解釈し、1を引いたものです。

このシーケンスは $g_k$ が0になると終了します。

例えば、第 $6$ 弱いグッドシュタイン列は $\\{6, 11, 17, 25, \ldots\\}$ です。

- $g_1 = 6$。
- $g_2 = 11$ は $6 = 110_2$、$110_3 = 12$、そして $12 - 1 = 11$ だからです。
- $g_3 = 17$ は $11 = 102_3$、$102_4 = 18$、そして $18 - 1 = 17$ だからです。
- $g_4 = 25$ は $17 = 101_4$、$101_5 = 26$、そして $26 - 1 = 25$ だからです。

以下同様です。

すべての弱いグッドシュタイン列は終了することが示されています。

$G(n)$ を第 $n$ 弱いグッドシュタイン列の非ゼロ要素の数とします。

$G(2) = 3$、$G(4) = 21$、$G(6) = 381$ であることが確認できます。

また、$1 ≤ n < 8$ のとき $\sum G(n) = 2517$ であることも確認できます。

$1 ≤ n < 16$ のときの $\sum G(n)$ の下9桁を求めてください。
