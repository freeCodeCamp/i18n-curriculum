# --description--

Die Möglichkeit, Ihr HTML und CSS zu inspizieren und zu debuggen, ist entscheidend für die Front-End-Entwicklung. Diese Lektion führt Sie durch die Chrome Dev Tools, mit denen Sie detaillierte Informationen über Ihre Elemente und CSS-Regeln sehen sowie Probleme in Ihrem Code finden und beheben können.

Um den Inspektor zu öffnen, können Sie mit der rechten Maustaste auf ein beliebiges Element einer Webseite klicken und „Untersuchen“ auswählen oder F12 drücken. Machen Sie das jetzt, um das HTML und CSS dieser Seite zu sehen.

Lassen Sie sich nicht von all den Werkzeugen überwältigen, die Sie jetzt sehen! Für diese Lektion wollen wir uns auf die Bereiche Elemente und Stile konzentrieren.
