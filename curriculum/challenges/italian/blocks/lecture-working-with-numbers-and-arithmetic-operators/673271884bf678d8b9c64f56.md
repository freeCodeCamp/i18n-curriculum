# --interactive--

JavaScript offre strumenti per eseguire operazioni aritmetiche di base sui numeri, come addizione, sottrazione, moltiplicazione e divisione. JavaScript include anche operatori per operazioni aritmetiche più complesse, come il resto e l'elevamento a potenza.

Tutti questi strumenti si chiamano operatori aritmetici. Vediamo nel dettaglio questi operatori, come usarli e come combinarli.

L'operatore di addizione è il segno più (`+`). L'operatore di addizione ti permette di trovare il totale di due o più numeri. Nelle operazioni di addizione, l'ordine dei numeri non conta:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 + num2;
const result2 = num2 + num1;
const result3 = num2 + num1 + num3;

console.log(result1); // 15
console.log(result2); // 15
console.log(result3); // 30
```

:::

L'operatore di sottrazione è il segno meno (`-`). Ti permette di trovare la differenza tra due numeri. Usa il segno meno quando vuoi sottrarre un numero da un altro, di solito un numero più piccolo da uno più grande:

:::interactive_editor

```js
const difference = 10 - 5;
console.log(difference); // 5
```

:::

Se un numero più piccolo viene prima, otterrai un risultato negativo:

:::interactive_editor

```js
const difference = 5 - 10;
console.log(difference); // -5
```

:::

Puoi anche assegnare i numeri a variabili e fare la sottrazione usando i nomi delle variabili:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const result = num1 - num2;

console.log(result); // 5
```

:::

In JavaScript, l'operatore di moltiplicazione è rappresentato da un asterisco (`*`) ed è usato per trovare il prodotto di due o più numeri. L'ordine dei numeri che moltiplichi non conta:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 * num2;
const result2 = num2 * num1;
const result3 = num2 * num1 * num3;

console.log(result1); // 50
console.log(result2); // 50
console.log(result3); // 750
```

:::

In JavaScript, l'operatore di divisione è una barra (`/`), che differisce dal simbolo di divisione usato nella matematica tradizionale (`÷`). Esegui operazioni di divisione con l'operatore di divisione. In questo caso, l'ordine dei numeri che dividi conta:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 / num2;
const result2 = num2 / num1;
const result3 = num2 / num1 / num3;

console.log(result1); // 2
console.log(result2); // 0.5
console.log(result3); // 0.03333333333333333
```

:::

È importante notare che se provi a dividere per zero, JavaScript restituirà `Infinity`:

:::interactive_editor

```js
const result = 10 / 0; 

console.log(result); // Infinity
```

:::

Assicurati di evitare questo tipo di calcoli per non ottenere risultati inattesi nel tuo codice.

L'operatore resto, rappresentato da un segno di percentuale (`%`), restituisce il resto di una divisione. Il resto in matematica è il valore avanzato dopo aver eseguito la divisione:

:::interactive_editor

```js
const num1 = 10;
const num2 = 3;
const remainder = num1 % num2;

console.log(remainder); // 1
```

:::

L'operatore di elevamento a potenza, rappresentato da un doppio asterisco (`**`), eleva un numero alla potenza di un altro:

:::interactive_editor

```js
const num1 = 2;
const num2 = 3;

const exponent = num1 ** num2;
console.log(exponent); // 8
```

:::

È possibile mescolare operatori in una singola operazione o espressione:

:::interactive_editor

```js
const result = 10 + 5 * 2 - 8 / 4;
console.log(result); // 18
```

:::

Quando mescoli operatori diversi in una singola espressione, il motore JavaScript segue un sistema chiamato precedenza degli operatori per determinare l'ordine delle operazioni. La precedenza degli operatori determina l'ordine in cui le operazioni vengono eseguite nelle espressioni. Imparerai di più sulla precedenza degli operatori nelle lezioni future.
