# --explanation--

O tempo `Present Continuous` frequentemente enfatiza uma condição temporária ou um problema ocorrendo no momento da fala.Por exemplo:

- `The server isn't running diagnostics.` - Destaca que o processo de diagnóstico não está acontecendo agora.

- `Developers aren't debugging the application.` - Mostra que os desenvolvedores não estão trabalhando nessa atividade no momento.

`Properly` significa fazer algo da maneira correta ou esperada.

Nesta frase, `aren't booting properly` refere-se aos computadores que atualmente não estão iniciando corretamente.
