# --description--

Eine weitere Möglichkeit, bedingte Anweisungen zu vergleichen, ist der ternäre Operator. Der ternäre Operator ist eine Kurzschreibweise für eine `if-else`-Anweisung. Er wird verwendet, um eine Bedingung zu prüfen und basierend auf dem Ergebnis der Bedingung einen Wert zurückzugeben. Die Syntax des ternären Operators ist wie folgt:

```javascript
let result = condition ? value1 : value2;
```

Im obigen Beispiel gilt: Wenn die Bedingung wahr ist, hat der Wert von `result` den Wert `value1`. Andernfalls hat der Wert von `result` den Wert `value2`.

Ternäre Operatoren werden häufig verwendet, um einer Variablen basierend auf einer Bedingung einen Wert zuzuweisen. Sie werden auch verwendet, um basierend auf einer Bedingung einen Wert zurückzugeben.
