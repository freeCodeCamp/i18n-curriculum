# --description--

对于整数 $n ≥ 4$，我们定义 $n$ 的下素数平方根，记为 $lps(n)$，为 $\text{小于等于 \sqrt{n} 的最大素数}$，以及 $n$ 的上素数平方根，$ups(n)$，为 $\text{大于等于 \sqrt{n} 的最小素数}$。

所以，例如，$lps(4) = 2 = ups(4)$，$lps(1000) = 31$，$ups(1000) = 37$。

如果一个整数 $n ≥ 4$ 满足 $lps(n)$ 和 $ups(n)$ 中的一个能整除 $n$，但不是两个都能整除，则称其为半可除数。

不超过 15 的半可除数之和是 30，这些数字是 8、10 和 12。15 不是半可除数，因为它是 $lps(15) = 3$ 和 $ups(15) = 5$ 的倍数。作为进一步的例子，不超过 1000 的 92 个半可除数之和是 34825。

不超过 999966663333 的所有半可除数的和是多少？
