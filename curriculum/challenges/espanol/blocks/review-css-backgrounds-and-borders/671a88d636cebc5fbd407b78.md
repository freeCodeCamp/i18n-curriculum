## Estilizado de Listas

- **Propiedad `line-height`**: Esta propiedad se usa para crear espacio entre líneas de texto. Los valores aceptados para `line-height` incluyen la palabra clave `normal`, números, porcentajes y unidades de longitud como la unidad `em`.
- **Propiedad `list-style-type`**: Esta propiedad se usa para especificar el marcador de un elemento de lista. Los valores aceptables pueden incluir un `circle`, `disc` o `decimal`.

Aquí tienes un ejemplo usando `list-style-type` para cambiar el estilo de viñeta:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
ul {
  list-style-type: square;
}
```

:::

- **Propiedad `list-style-position`**: Esta propiedad se usa para establecer la posición del marcador de la lista. Los únicos dos valores aceptables son `inside` y `outside`.

Aquí tienes un ejemplo que muestra la diferencia entre `inside` y `outside`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="inside-list">
  <li>Item with inside position</li>
  <li>Item with inside position</li>
</ul>
<ul class="outside-list">
  <li>Item with outside position</li>
  <li>Item with outside position</li>
</ul>
```

```css
.inside-list {
  list-style-position: inside;
}

.outside-list {
  list-style-position: outside;
}
```

:::

- **Propiedad `list-style-image`**: Esta propiedad se usa para usar una imagen como marcador de un elemento de lista. Un caso común es usar la función `url` con un valor que apunte a una ubicación válida de imagen.

## Espaciado de elementos de lista usando `margin`

- Además de `line-height`, los márgenes también se pueden usar en CSS para mejorar el espacio y la legibilidad de los elementos de lista.
- Los márgenes crean espacio fuera de cada elemento `li`, permitiendo controlar la separación entre los elementos de la lista.
- `margin-bottom` se usa para crear espacio debajo de cada elemento de lista. Por ejemplo, `margin-bottom: 10px;` creará una separación de 10 píxeles debajo de cada elemento de lista.

Aquí tienes un ejemplo usando `margin-bottom` para espaciar los elementos de la lista:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="list">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
.list li {
  margin-bottom: 20px;
}
```

## Estilizado de Enlaces

- **`pseudo-class`**: Esta es una palabra clave que se agrega a un selector y permite seleccionar elementos según un estado particular. Los estados comunes incluyen los estados `:hover`, `:visited` y `:focus`.
- **`:link pseudo-class`**: Esta pseudoclase se usa para estilizar enlaces que el usuario no ha visitado.

Aquí tienes un ejemplo usando la pseudo-clase `:link`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:link {
  color: red;
}
```

:::

- **`:visited pseudo-class`**: Esta pseudoclase se usa para dar estilo a los enlaces que un usuario ya ha visitado.
- **`:hover pseudo-class`**: Esta pseudoclase se usa para dar estilo a los elementos sobre los que un usuario está pasando activamente el cursor.

Aquí tienes un ejemplo usando la pseudo-clase `:hover`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Hover over me!</a>
```

```css
a:hover {
  color: green;
  text-decoration: underline;
}
```

:::

- **`:focus pseudo-class`**: Esta pseudoclase se usa para dar estilo a un elemento cuando recibe enfoque. Ejemplos incluyen elementos `input` o `select` donde el usuario hace clic o usa la tecla tab para enfocarlo.

Aquí tienes un ejemplo usando la pseudo-clase `:focus`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:focus {
  outline: 2px solid orange;
}
```

:::

- **`:active pseudo-class`**: Esta pseudoclase se usa para dar estilo a un elemento que fue activado por el usuario. Un ejemplo común es cuando el usuario hace clic en un botón.

Aquí tienes un ejemplo usando la pseudo-clase `:active`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Click me!</a>
```

```css
a:active {
  color: pink;
}
```

## Trabajar con Fondos y Bordes

- **Propiedad `background-size`**: Esta propiedad se usa para establecer el tamaño del fondo de un elemento. Algunos valores comunes incluyen `cover` para que la imagen de fondo cubra todo el elemento y `contain` para que la imagen de fondo quepa dentro del elemento.

Aquí tienes un ejemplo usando `background-size: cover`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

Aquí tienes un ejemplo usando `background-size: contain`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **Propiedad `background-repeat`**: Esta propiedad se usa para determinar cómo se deben repetir las imágenes de fondo a lo largo de los ejes horizontal y vertical. El valor predeterminado para `background-repeat` es `repeat`, lo que significa que la imagen se repetirá tanto horizontal como verticalmente. También puedes especificar que no haya repetición usando el valor `no-repeat`.

Aquí tienes un ejemplo usando `background-repeat: no-repeat`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **Propiedad `background-position`**: Esta propiedad se usa para especificar la posición de la imagen de fondo. Puede establecerse con una longitud específica, porcentaje o valores de palabra clave como `top`, `bottom`, `left`, `right` y `center`.

Aquí tienes un ejemplo usando `background-position`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

- **Propiedad `background-attachment`**: Esta propiedad se usa para especificar si la imagen de fondo debe desplazarse con el contenido o permanecer fija en su lugar. Los valores principales son `scroll` (predeterminado), donde la imagen de fondo se desplaza con el contenido, y `fixed`, donde la imagen de fondo permanece en la misma posición en la pantalla.
- **Propiedad `background-image`**: Esta propiedad se usa para establecer la imagen de fondo de un elemento. Puedes establecer múltiples imágenes de fondo al mismo tiempo y usar las funciones `url`, `radial-gradient` o `linear-gradient` como valores.

Aquí tienes un ejemplo usando `background-image`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **Propiedad `background`**: Esta es la propiedad abreviada para establecer todas las propiedades de fondo en una sola declaración. Aquí hay un ejemplo de establecer la imagen de fondo y configurarla para que no se repita: `background: no-repeat url("example-url-goes-here");`

Aquí tienes un ejemplo usando la propiedad abreviada `background`:

:::interactive_editor

```html
<style>
  body {
    background: center top no-repeat url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **Buen contraste para los colores de fondo y primer plano**: Es importante asegurarse de que los colores de fondo y primer plano tengan un buen contraste para que el texto sea legible. Las Pautas de Accesibilidad del Contenido Web (WCAG, por sus siglas en inglés) recomiendan una relación de contraste mínima de 4.5:1 para texto normal y 3:1 para texto grande.

## Bordes

- **Propiedad `border-top`**: Esta propiedad se usa para establecer los estilos del borde superior de un elemento. `border-top: 3px solid blue;` establece un borde sólido azul de 3 píxeles de ancho en el lado superior del elemento.
- **Propiedad `border-right`**: Esta propiedad se usa para establecer los estilos del borde derecho de un elemento. `border-right: 2px solid red;` establece un borde rojo sólido de 2 píxeles de ancho en el lado derecho del elemento.
- **Propiedad `border-bottom`**: Esta propiedad se usa para establecer los estilos del borde inferior de un elemento. `border-bottom: 1px dashed green;` establece un borde verde punteado de 1 píxel de ancho en el lado inferior del elemento.
- **Propiedad `border-left`**: Esta propiedad se usa para establecer los estilos del borde izquierdo de un elemento. `border-left: 4px dotted orange;` establece un borde naranja punteado de 4 píxeles de ancho en el lado izquierdo del elemento.

Aquí tienes un ejemplo usando propiedades individuales de borde:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="bordered-box">Box with different borders</div>
```

```css
.bordered-box {
  border-top: 3px solid blue;
  border-right: 2px solid red;
  border-bottom: 1px dashed green;
  border-left: 4px dotted orange;
  padding: 20px;
}
```

:::

- **Propiedad `border`**: Esta es la propiedad abreviada para establecer el ancho, estilo y color del borde de un elemento. `border: 1px solid black;` establece un borde negro sólido de 1 píxel de ancho.

Aquí tienes un ejemplo usando la propiedad abreviada `border`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

- **Propiedad `border-radius`**: Esta propiedad se usa para crear esquinas redondeadas en el borde de un elemento.

Aquí tienes un ejemplo usando `border-radius`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

- **Propiedad `border-style`**: Esta propiedad se usa para establecer el estilo del borde de un elemento. Algunos valores aceptados incluyen `solid`, `dashed`, `dotted` y `double`.

Aquí tienes un ejemplo usando diferentes valores de `border-style`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="solid-border">Solid border</div>
<div class="dashed-border">Dashed border</div>
<div class="dotted-border">Dotted border</div>
<div class="double-border">Double border</div>
```

```css
.solid-border {
  border: 3px solid blue;
  margin-bottom: 10px;
  padding: 10px;
}

.dashed-border {
  border: 3px dashed red;
  margin-bottom: 10px;
  padding: 10px;
}

.dotted-border {
  border: 3px dotted green;
  padding: 10px;
}

.double-border {
  border: 3px double blueviolet;
  padding: 10px;
}
```

## Gradientes

- **Función `linear-gradient()`**: Esta función CSS se usa para crear una transición entre varios colores a lo largo de una línea recta.

Aquí tienes un ejemplo usando `linear-gradient()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

- **Función `radial-gradient()`**: Esta función CSS crea una imagen que irradia desde un punto particular, como un círculo o una elipse, y transiciona gradualmente entre varios colores.

Aquí tienes un ejemplo usando `radial-gradient()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```
