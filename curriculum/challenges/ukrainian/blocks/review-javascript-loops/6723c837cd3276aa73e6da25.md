# --interactive--

## Робота з циклами

- **Цикл `for`**: Цей тип циклу використовується, щоб повторити блок коду певну кількість разів. Цикл складається з трьох частин: інструкції ініціалізації, умови та інструкції збільшення/зменшення. Інструкція ініціалізації виконується перед початком циклу. Зазвичай її використовують для ініціалізації змінної-фішки. Умова оцінюється перед кожною ітерацією циклу. Ітерація — це один прохід циклом. Якщо умова `true`, виконується блок коду всередині циклу. Якщо умова `false`, цикл зупиняється, і ви переходите до наступного блоку коду. Інструкція збільшення/зменшення виконується після кожної ітерації циклу. Зазвичай її використовують, щоб збільшити або зменшити змінну-фішку.

:::interactive_editor

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

:::

- **Цикл `for...of`**: Цей тип циклу використовується, коли потрібно ітерувати над значеннями ітерованого об’єкта. Прикладами ітерованих об’єктів є масиви та рядки.

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

:::

- **Цикл `for...in`**: Цей тип циклу найкраще підходить, коли потрібно ітерувати над пропами об’єкта. Цикл пройде всі перелічувані пропи об’єкта, включно з успадкованими та непронумерованими пропами.

:::interactive_editor

```js
const fruit = {
  name: 'apple',
  color: 'red',
  price: 0.99
};

for (const prop in fruit) {
  console.log(fruit[prop]);
}
```

:::

- **Цикл `while`**: Цей тип циклу виконуватиме блок коду доти, доки умова `true`.

:::interactive_editor

```js
let i = 5;

while (i > 0) {
  console.log(i);
  i--;
}
```

:::

- **Цикл `do...while`**: Цей тип циклу виконає блок коду принаймні один раз перед перевіркою умови.

```js
let userInput;

do {
  userInput = prompt("Please enter a number between 1 and 10");
} while (Number(userInput) < 1 || Number(userInput) > 10);

alert("You entered a valid number!");
```

## Інструкції `break` та `continue`

- **Визначення**: Інструкція `break` використовується, щоб достроково вийти з циклу, тоді як інструкція `continue` використовується, щоб пропустити поточну ітерацію циклу і перейти до наступної.

:::interactive_editor

```js
// Example of break statement
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, and 4

// Example of continue statement 
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, 4, 6, 7, 8, and 9
```

:::
