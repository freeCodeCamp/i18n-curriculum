# --description--

`loop` 是一系列指令，持续重复直到达到某个条件。`loops` 用于高效地多次运行相同的代码。常见的 `loops` 类型包括 `for` 循环和 `while` 循环。 

例如，`for loop`：`for (int i = 0; i < 5; i++) { /* code to be executed */ }`

在示例中，花括弧 `{ }` 内的代码会被重复执行，直到条件不再为真。
