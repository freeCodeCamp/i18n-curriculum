# --description--

Der Ausdruck **`Soy` + [Nationalität]** ist die gebräuchlichste Art, Ihre Nationalität auf Spanisch anzugeben. Er entspricht dem deutschen „Ich bin + [Nationalität]“.

Dieser Ausdruck wird bei Vorstellungen als direkte Antwort auf die Frage `¿Cuál es tu nacionalidad?` verwendet, wobei `nacionalidad` „Nationalität“ bedeutet.

Im Spanischen werden Nationalitäten nur dann großgeschrieben, wenn sie am Satzanfang stehen. Außerdem müssen sie im Geschlecht mit der sprechenden Person übereinstimmen. Zum Beispiel:

| Land | Nationalität (weiblich) | Nationalität (männlich) |
| :---: | :---: | :---: |
| **`México`** | `mexicana` | `mexicano` |
| **`Argentina`** | `argentina` | `argentino` |
| **`Perú`** | `peruana` | `peruano` |
| **`Uruguay`** | `uruguaya` | `uruguayo` |
| **`El Salvador`** | `salvadoreña` | `salvadoreño` |
