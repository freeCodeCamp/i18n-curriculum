# --description--

箭頭函式是用來撰寫 JavaScript 函式（程式）的一種較現代的方式。它們提供比傳統函式（程式）表達式更簡潔的語法。箭頭函式是使用 `=>` 語法定義的。 

```js
const add = (a, b) => {
  return a + b;
}
```

在上述範例中，箭頭函式 `add` 接受兩個參數 `a` 和 `b`，並傳回它們的總和。`return` 關鍵字用於傳回加法操作的結果。

如果箭頭函式只有一個表達式，則可以省略大括號 `{}` 和 `return` 關鍵字。該表達式將會被隱式傳回。

```js
const add = (a, b) => a + b;
```

在上述範例中，箭頭函式 `add` 接受兩個參數 `a` 和 `b`，並傳回它們的和。`return` 關鍵字和大括號 `{}` 被省略，加法操作的結果是隱式傳回的。
