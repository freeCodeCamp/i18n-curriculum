# --interactive--

偽類別是特殊的 CSS 關鍵字，允許你根據元素的特定狀態或位置來選取該元素。元素的狀態或位置可能包括：

- 當它是作用中的。
- 當被滑鼠暫留時。
- 當它是父元素的第一個子元素時。
- 當它是父元素的最後一個子元素時。
- 當 `link` 已被造訪。
- 當它被停用時。

要使用偽類別，你必須在選擇器後面加上冒號（`:`），然後接上偽類別的名稱： 

```css
selector:pseudo-class {
  /* CSS properties */
}
```

讓我們看看你如何使用偽類別來表述我們已經提到的每個狀態和位置。

`:active` 偽類讓你選取元素的作用中狀態，例如點擊按鈕時：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example Button</button>
```

```css
button:active {
  background: greenyellow;
}
```

:::

`:hover` 偽類別定義元素的暫留狀態。範例是暫留在按鈕或聯結上：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="#">Hover over me!</a>  
```

```css
a:hover {
  text-decoration: none;
  color: white;
  background: crimson;
}
```

:::

`:first-child` 偽類別會選取作為其父元素第一個子元素的元素。以下是針對 `div` 容器中第一個段落元素的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:first-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

該 `div` 中的第一個段落元素將會收到 `lightcoral` 背景色以及四邊 `0.4rem` 的 `padding`。 

`:last-child` 偽類別標的其父元素的最後一個子元素。以下是標的容器 `div` 元素中最後一個段落元素的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:last-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

`:visited` 偽類讓你為使用者已經造訪過的聯結設定樣式：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="https://www.example.com" target="_blank">Visit Example.com</a>
```

```css
a:visited {
  color: purple;
}
```

:::

`:disabled` 偽類讓你為作用中的元素設定停用模式的樣式：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button disabled>Disabled Button</button>
```

```css
button:disabled {
  background-color: lightgray;  
}
```

:::

如你所見，偽類別讓你可以根據使用者互動（例如暫留或造訪 `link`）來設定元素的樣式。這使你的網站感覺更具互動性。

除了已經提到的偽類別之外，還有其他像是：

- `:focus`
- `:first-of-type`
- `:last-of-type`
- `:nth-of-type`
- `:modal`
- `:enabled`
- `:checked`
- `:required`，以及更多。
