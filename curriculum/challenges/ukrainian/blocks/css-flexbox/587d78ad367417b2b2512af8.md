# --description--

Властивість `align-items` схожа на `justify-content`. Нагадаємо, що властивість `justify-content` вирівнювала гнучкі елементи вздовж головної осі. Для рядків головна вісь — це горизонтальна лінія, а для стовпців — вертикальна.

Гнучкі контейнери також мають **поперечну вісь**, яка є протилежною до головної осі. Для рядків поперечна вісь вертикальна, а для стовпців — горизонтальна.

CSS пропонує властивість `align-items` для вирівнювання гнучких елементів вздовж поперечної осі. Для рядка вона вказує CSS, як підняти або опустити елементи всього рядка всередині контейнера. А для стовпця — як зсунути всі елементи вліво або вправо всередині контейнера.

Різні значення, доступні для `align-items`, включають:

<ul><li><code>flex-start</code>: вирівнює елементи на початку гнучкого контейнера. Для рядків це вирівнювання елементів до верху контейнера. Для стовпців — вирівнювання елементів до лівого краю контейнера.</li><li><code>flex-end</code>: вирівнює елементи в кінці гнучкого контейнера. Для рядків це вирівнювання елементів до низу контейнера. Для стовпців — вирівнювання елементів до правого краю контейнера.</li><li><code>center</code>: вирівнює елементи по центру. Для рядків це вертикальне вирівнювання елементів (рівний простір зверху і знизу елементів). Для стовпців — горизонтальне вирівнювання (рівний простір зліва і справа від елементів).</li><li><code>stretch</code>: розтягує елементи, щоб заповнити гнучкий контейнер. Наприклад, елементи рядка розтягуються, щоб заповнити контейнер зверху до низу. Це значення за замовчуванням, якщо не вказано значення <code>align-items</code>.</li><li><code>baseline</code>: вирівнює елементи за їхніми базовими лініями. Базова лінія — це поняття з тексту, уявіть її як лінію, на якій стоять літери.</li></ul>

# --hints--

Елемент `#box-container` має мати властивість `align-items` зі значенням `center`.

```js
const boxContainer = document.querySelector('#box-container');
const alignment = window.getComputedStyle(boxContainer)['align-items'];
assert.strictEqual(alignment, 'center');
```
