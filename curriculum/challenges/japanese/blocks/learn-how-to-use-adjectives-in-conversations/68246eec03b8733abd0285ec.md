`at once`

### --feedback--

この2語のフレーズは「同時に」または「一緒にすべて」を意味します。

# --explanation--

`Support` は、何かを適切に機能させるために扱ったり許可したりすることを意味します。例えば：

`The app can support 1,000 users at the same time.` – それはクラッシュせずにその数を扱うことができます。

`At once` は、複数のことを同時に行うことを意味します。例えば：

`The software can open many files at once.` – すべてのファイルが一つずつではなく、一緒に開きます。
