# --interactive--

HTML，全名為 Hypertext Markup Language，是用來建立網頁的標記語言。當你造訪網站並看到段落、標題、聯結、影像和影片等內容時；那就是 HTML。 

這裡有一個使用 HTML 元素的小範例。試著編輯編輯器中的部分文字，並查看預覽視窗中的變更更新。

:::interactive_editor

```html
<h1>Main heading element</h1>

<p>I am a paragraph element.</p>
```

:::

HTML 透過元素來表述網頁的內容和結構。大多數元素會有一個開啟標記和一個關閉標記。有時這些標記會被取用為起始標記和結束標記。在這兩個標記之間，你會有內容。這些內容可以是文字或其他 HTML 元素。 

這是另一個段落元素的範例。在編輯器中將文字改為 `I love coding!`，並在預覽視窗中查看結果。 

:::interactive_editor

```html
<p>I am a paragraph element.</p>
```

:::

開啟和關閉的標記皆以左尖括號（`<`）開始，以右尖括號（`>`）結束，標記名稱置於這些尖括號之間。雖然 HTML 標記名稱不區分大小寫，但普遍接受的慣例和最佳做法是以小寫字母書寫。 

這裡是對開啟和關閉段落 `tag` 的更詳細說明：

```html
<p>
```

```html
</p>
```

開啟標記與關閉標記的區別在於關閉標記中緊接在左尖括號後的正斜線（`/`）。有些 HTML 元素沒有關閉標記，這些稱為空元素。

這是一個影像元素的範例，它是一個空元素：

```html
<img>
```

請注意，此影像元素沒有結束標籤且沒有任何內容。空元素不能有任何內容，且只有起始標籤。

有時你會看到 void 元素在 `>` 前使用 `/`，像這樣：

```html
<img />
```

雖然許多程式碼格式化工具如 Prettier 會選擇在 void 元素中包含 `/`，但 HTML 規範指出 `/` 的存在「並不表示起始標記是自我封閉的，而是多餘的且沒有任何效果」。

在實際開發中，你會看到兩種表單，因此熟悉兩者非常重要。

如果你想顯示一張影像，你需要在你的影像元素中包含幾個屬性。屬性是用來調整 HTML 元素行為的特殊值。 

這是一個帶有 `src` 屬性的影像元素範例。將 `src` 屬性的值更新為 `"https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"`，你將會看到影像變成兩隻安靜睡覺的貓。

:::interactive_editor

```html
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/running-cats.jpg" />
```

:::

`src` 屬性用於指定該影像的位置。對於影像元素，最好包含另一個稱為 `alt` 屬性的屬性。`alt` 屬性用於為影像提供簡短且具描述性的文字。

這是一個具有 `src` 和 `alt` 屬性的影像元素範例。嘗試將 `src` 值更新為 `"https://.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"` 來破壞影像。你會看到影像消失，僅顯示 `alt` 文字。  

:::interactive_editor

```html
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" alt="Two tabby kittens sleeping together on a couch." />
```

:::

所以，你可能會想知道單靠 HTML 是否足以建置一個網站。答案是：視情況而定。如果你正在建置一個只顯示文字和圖片的小型練習專案，單用 HTML 可能就足夠了。然而，如果你正在建立一個現代專業網站，你將需要同時使用 HTML、CSS 和 JavaScript。

HTML 用於內容和結構。CSS 用於樣式設計。JavaScript 用於為你的網頁添加互動性。一個很好的比喻是將 HTML、CSS 和 JavaScript 比較成一棟完整的建築。 

HTML 表述組成牆壁的區塊、具體的材料和鐵件。它是使建築堅固的基礎。CSS 表述使房子看起來美觀的內部和外部設計。JavaScript 表述確保水電不中斷存取的電力和水系統。
