# --explanation--

`Zero Conditional`는 조건이 충족될 때 항상 참인 사실에 사용됩니다.

구조는 `if` + `Present Simple` (조건 부분) 그리고 `Present Simple` (결과 부분)을 따릅니다. 예를 들어:

- `If you press the power button, the computer starts.` - 전원 버튼을 누르면 항상 컴퓨터가 켜집니다.

- `If the internet is down, the website doesn't load.` - 인터넷 연결이 없으면 웹사이트가 열리지 않습니다.

이 대화에서 `if the BIOS is corrupted`는 조건이고 `it prevents the computer from starting`는 결과입니다. 이것은 손상된 BIOS가 컴퓨터에 미치는 일반적인 진리입니다.
