# --description--

单词 `so` 是一个连接词，用于表示因果关系或解释某事的原因。在技术和编程中，`so` 通常连接问题或情况与方案或结果。

# --questions--

## --text--

选择在技术上下文中正确使用 `so` 连接原因和效果的句子。

## --answers--

`The program crashed, so we must reboot the computer.`

---

`We wrote the code, so but it didn't compile.`

### --feedback--

单词 `so` 在这里使用不正确。`so` 后面不应使用 `but`。应该是 `The code didn't compile, so we rewrote it`，或者 `We wrote the code, but it didn't compile.`

---

`The server is online, so therefore we can't connect.`

### --feedback--

同时使用 `so` 和 `therefore` 是多余的。应该是 `The server is online, so we can connect` 或者 `The server is online; therefore, we can connect.`。

---

`We need more data, so we can ignore the results.`

### --feedback--

这句话在逻辑上没有连接因果关系。更好的 `so` 用法是，`We need more data, so we will perform more tests.`
