# --description--

Ángela 使用結構體 `¿Cuál es tu` + 名詞來提問。

她可以停在那裡。但她改為使用連接詞 `y` 和第二個名詞添加另一則資訊。

`y` 是用來連接兩個名詞的連接詞。以下是她使用的結構：

`¿Cuál es tu` + 名詞 + `y` + 名詞`？`

這允許說話者在同一個問題中請求多於一個細節。

# --explanation--

在句子 `¿Cuál es tu nombre y apellido?` 中，兩個名詞由 `y` 連接：

- `nombre`

- `apellido`

字 `y` 連接兩個資訊。

Ángela 首先要求該人的 `nombre`，然後添加 `apellido`。 

此結構體允許說話者在一個問題中請求多個細節：

`¿Cuál es tu correo y número?`

這裡，Ángela 要求的第二個資訊是 `apellido`。
