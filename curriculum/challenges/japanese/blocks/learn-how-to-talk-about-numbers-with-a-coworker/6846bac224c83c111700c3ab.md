# --explanation--

**基数**は「いくつあるか」を示します。例えば`one`、`two`、`three`のようなものです。

文`We need three new developers for the next sprint.`の中の`three`は**基数**です。

他のすべての文は、順序や位置を表す間違った形を使っています。
