# --explanation--

`Simplifying complex systems` significa tornar algo grande ou difícil mais fácil de usar ou de entender.

`Hiding unnecessary details` significa manter partes internas (como o funcionamento de algo) ocultas do usuário, para que ele veja apenas o que precisa para interagir.

A pergunta de Mark mostra que ele entende a abstração como uma forma de gerenciar a complexidade — não apagando partes, mas organizando sistemas para que os usuários lidem apenas com o que é relevante.
