# --explanation--

`UI` 代表 `user interface`，指的是使用者與產品互動的視覺元素。`设计师 (shè jì shī)` 意思是「設計師」。

總體而言，`UI 设计师 (shè jì shī)` 是指專注於設計產品外觀和佈局，使其易於且愉快使用的職業。這是在科技公司中非常常見的術語，人們直接使用 `UI` 而不將其翻譯成中文。例如：

`我是 (wǒ shì) UI 设计师 (shè jì shī)。` – 我是 UI 設計師。
