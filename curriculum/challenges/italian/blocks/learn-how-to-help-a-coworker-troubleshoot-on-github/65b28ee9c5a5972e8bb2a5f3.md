# --description--

`Debugging` è un termine comune nella programmazione. Significa trovare e correggere problemi o `bugs` nel software. Per esempio:

`The programmer spent the morning debugging the new app.` 

# --questions--

## --text--

Cosa significa `debugging` nel contesto della programmazione?

## --answers--

Far funzionare un programma più velocemente

### --feedback--

Il debugging riguarda la correzione dei problemi, non necessariamente il miglioramento della velocità di esecuzione di un programma.
