# --interactive--

Wenn Sie mit Strings in JavaScript arbeiten, gibt es Situationen, in denen Sie Sonderzeichen einfügen müssen, die die JavaScript-Engine sonst möglicherweise falsch interpretiert.

Zwei häufige Aufgaben sind das Erstellen eines Zeilenumbruchs innerhalb eines Strings und das Escapen bestimmter Zeichen (wie Anführungszeichen), damit sie korrekt angezeigt werden.

In vielen Programmiersprachen, einschließlich JavaScript, können Sie einen Zeilenumbruch in einem String mit einem Sonderzeichen erzeugen, das Escape-Sequenz genannt wird. Die gebräuchlichste Escape-Sequenz für Zeilenumbrüche ist `\n`.

Wenn Sie beispielsweise einen String in mehrere Zeilen aufteilen möchten, verwenden Sie `\n` an der Stelle, an der die neue Zeile beginnen soll:

:::interactive_editor

```js
let poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

Die `\n`-Escape-Sequenz weist JavaScript an, an dieser Stelle einen Zeilenumbruch einzufügen, sodass der String über mehrere Zeilen angezeigt wird.

Ein weiterer wichtiger Begriff beim Arbeiten mit Strings ist das Escapen von Zeichen. Manchmal müssen Sie Zeichen in Ihren String einfügen, die JavaScript normalerweise für etwas anderes verwendet, wie zum Beispiel Anführungszeichen.

Wenn Sie Anführungszeichen einfach innerhalb eines Strings verwenden, ohne sie zu escapen, kann das einen Fehler verursachen, weil JavaScript denkt, Sie wollen den String beenden.

Zum Beispiel führt dies zu einem Fehler:

```js
let statement = "She said, "Hello!"";
```

JavaScript ist verwirrt, weil es denkt, der String endet nach dem Wort `"said,"`, aber Sie möchten, dass die Anführungszeichen um `"Hello!"` Teil des Strings sind.

Um das zu beheben, können Sie die inneren Anführungszeichen escapen, indem Sie einen Backslash (`\`) davor setzen:

:::interactive_editor

```js
let statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

Der Backslash sagt JavaScript, die Anführungszeichen als wörtliche Zeichen zu behandeln, sodass sie korrekt ausgegeben werden.

Sie können auch andere Sonderzeichen escapen, wie den Backslash selbst (`\\`) oder einfache Anführungszeichen innerhalb eines Strings, der von einfachen Anführungszeichen umgeben ist (`\'`).

Hier ist ein weiteres Beispiel mit einfachen Anführungszeichen:

:::interactive_editor

```js
let quote = 'It\'s a beautiful day!';
console.log(quote); // It's a beautiful day!
```

:::

Indem Sie das einfache Anführungszeichen mit `\'` escapen, weiß JavaScript, dass es als Teil des Strings enthalten sein soll und nicht den String vorzeitig beendet.

Das Escapen und Erstellen von Zeilenumbrüchen ist essenziell, wenn Sie Ausgaben formatieren oder Sonderzeichen in Strings verarbeiten. Diese Techniken helfen Ihnen, Fehler zu vermeiden und sicherzustellen, dass Ihr Text genau so erscheint, wie Sie es möchten.
