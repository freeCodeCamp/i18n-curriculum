# --description--

Das Verb `vivir` bedeutet „leben“. 

Wenn man sich auf eine andere Person („er“/„sie“) bezieht, wird das Verb als `vive` konjugiert. Es folgt typischerweise die Präposition `en`, um anzugeben, wo die Person lebt, mit folgender Struktur:

**`Vive en` + Ort**

Julieta antwortet auf Mateos Frage und sagt ihm, wo Valeria lebt.
