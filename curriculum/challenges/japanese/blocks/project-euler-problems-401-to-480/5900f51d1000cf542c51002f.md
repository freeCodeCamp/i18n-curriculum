# --description--

$E(x_0, y_0)$ を、ユークリッドのアルゴリズムで $x_0$ と $y_0$ の最大公約数を求めるのにかかるステップ数とします。より正式には：

$$\begin{align}
  & x_1 = y_0, y_1 = x_0\bmod y_0 \\\\
  & x_n = y_{n - 1}, y_n = x_{n - 1}\bmod y_{n - 1}
\end{align}$$

$E(x_0, y_0)$ は $y_n = 0$ となる最小の $n$ です。

$E(1, 1) = 1$、$E(10, 6) = 3$、$E(6, 10) = 4$ となります。

$S(N)$ を $1 ≤ x, y ≤ N$ の範囲での $E(x, y)$ の合計と定義します。

$S(1) = 1$、$S(10) = 221$、$S(100) = 39\,826$ です。

$S(5 \times {10}^6)$ を求めてください。
