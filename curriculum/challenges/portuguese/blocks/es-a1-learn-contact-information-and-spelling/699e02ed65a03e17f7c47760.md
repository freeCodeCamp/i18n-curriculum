# --description--

Como você aprendeu, `repetir` significa dizer algo novamente. Quando você adiciona o pronome `lo`, está se referindo a algo já mencionado. O pronome substitui a coisa específica (uma palavra, um número, um e-mail, etc.). Então:

- `Repetir` = repetir.

- `Repetirlo` = repetir isso.

`Repetirlo` é usado quando "a coisa" já é conhecida pelo contexto. O mesmo padrão funciona com outros verbos:

- `¿Puedes escribirlo?` - Você pode escrever?

- `¿Puedes decirlo otra vez?` - Você pode dizer isso novamente?

# --explanation--

`Lo` é um pronome objeto direto usado para substituir algo já conhecido na conversa. Quando anexado a um infinitivo, forma uma palavra só:

- `repetir` + `lo` → `repetirlo`  - Repita-o.

- `escribir` + `lo` → `escribirlo` - Escreva-o.

- `decir` + `lo` → `decirlo` - Diga isso.

Essa estrutura é comum quando o ouvinte já entende a que "it" se refere.
