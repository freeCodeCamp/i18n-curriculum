# --description--

У попередньому уроці ви ознайомилися з типом даних список і навчилися отримувати доступ до елементів списку, а також робити зрізи списку. У цьому уроці ви продовжите вивчати списки та деякі загальні методи, пов’язані з ними, такі як `append()`, `pop()` і `sort()`.

Перший метод, який ми розглянемо, — це метод `append()`. Він використовується, щоб додати елемент у кінець списку. Ось приклад використання методу `append()` для додавання числа `6` до списку чисел:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

Якщо ви хочете додати один список у кінець іншого, ви також можете використати метод `append()` ось так:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

Зверніть увагу, що весь список `even_numbers` вкладений у список `numbers`.

Але якщо ви хочете додати всі окремі числа зі списку `even_numbers` у кінець списку `numbers`, тоді можна використати метод `extend()`.

Метод `extend()` схожий на метод `append()`, але з `extend()` ви можете додати кілька елементів з одного списку до іншого. Ось приклад додавання чисел `6`, `8` і `10` з одного списку в кінець списку `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

Як бачите, вкладений список зник, і залишився просто список чисел.

Щоб вставити елемент у певний індекс списку, можна використати метод `insert()`. Цей метод приймає два аргументи: індекс, куди ви хочете вставити новий елемент, і сам елемент, який потрібно вставити.

Ось приклад використання методу `insert()`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

Наступний код вставить число `2.5` на індекс `2` у списку `numbers`.

Якщо ви хочете видалити елемент зі списку, можна використати метод `remove()`. Метод `remove()` приймає значення елемента, який потрібно видалити, як аргумент:

```py
numbers = [10, 20, 30, 40, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50]
```

Важливо зазначити, що цей метод видалить лише перше входження елемента. Не всі:

```py
numbers = [10, 20, 30, 40, 50, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50, 50]
```

Щоб видалити елемент за певним індексом у списку, можна використати метод `pop()` ось так:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

Якщо ви не вкажете елемент для методу `pop`, тоді буде видалено останній елемент.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

Якщо потрібно очистити список, можна використати метод `clear()` ось так:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

Наступний метод, який ми розглянемо, — це метод `sort()`. Цей метод використовується для сортування елементів на місці. Ось приклад сортування випадкового списку чисел на місці:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

На відміну від методу `sort()`, існує функція `sorted()`, яка працює з будь-яким ітерованим об’єктом і повертає новий відсортований список замість зміни оригінального списку. Наприклад:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(numbers) # [19, 2, 35, 1, 67, 41]
print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
```

Нагадаємо, що ітерований об’єкт — це спеціальний тип об’єкта, по якому можна пройтися циклом, отримуючи доступ до кожного елемента по черзі. Ви дізнаєтеся більше про роботу циклів у Python у майбутньому уроці.

І метод `sort()`, і функція `sorted()` приймають необов’язкові параметри `key` і `reverse`. Ви дізнаєтеся більше про ці необов’язкові параметри у майбутньому уроці, коли вивчатимете кортежі. Також ви дізнаєтеся більше про інші вбудовані функції, як `sorted()`, у майбутніх уроках.

Наступний метод, який ми розглянемо, — це метод `reverse()`. Цей метод переверне список елементів на місці ось так:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

Останній метод, який ми розглянемо, — це метод `index`. Він використовується, щоб знайти перший індекс, де можна знайти елемент у списку. Ось приклад використання методу `index`, щоб знайти мову `'Java'` у списку `programming_languages`:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

Якщо елемент не знайдено, Python викликає `ValueError`:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: 'JavaScript' is not in list
"""
```

Існує ще кілька методів для списків Python, але цей початковий набір методів — гарна точка для початку.
