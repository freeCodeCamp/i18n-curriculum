# --description--

讓我們來了解 Bash 及其一些基本指令。

Bash 代表 Bourne Again SHell，可以說是在類 Unix 環境中你最常遇到的 shell。因此，了解如何操作這個 shell 並使用常見指令會非常有幫助。

第一個指令是 `pwd`，它會將目前的工作目錄列印到終端機。所謂「工作目錄」是指終端機目前指向的目錄，這對許多其他指令來說是非常重要的資訊。

你的終端機很可能指向錯誤的目錄。`cd` 指令允許你更改目錄。你可以指定以斜線（`/`）為前置式的絕對路徑、沒有前置式的相對路徑，並使用雙點（`..`）語法移動到父目錄。

下一個指令是 `ls`，它會列出你目前工作目錄的內容。這有助於查看你的目錄中是否存在某個檔案或資料夾。`ls` 指令也接受旗標來顯示隱藏檔案（`-a` 旗標）、檔案權限（`-l` 旗標）以及其他特性。你將在下一課中學習更多關於指令旗標的內容。你也可以使用 `less` 或 `cat` 來查看檔案的內容。 

如果目錄中不包含你需要的檔案或資料夾怎麼辦？有一些稱為 `mkdir` 和 `touch` 的指令可以讓你創建它們。`mkdir` 創建一個新的目錄或資料夾，而 `touch` 創建一個新的檔案。這兩個指令都接受你想要創建的檔案或資料夾名稱。例如，`touch readme.md` 創建一個名為 `readme.md` 的新的 Markdown 檔案。

也許你的新檔案不應該命名為 `readme.md`，而應該是 `readthis.md`。幸運的是，`mv` 指令允許你移動（或重新命名）檔案。它接受舊檔案名稱和新檔案名稱。

`rm` 指令允許你移除（或刪除）一個檔案，並且使用 `-r` 旗標會刪除一個目錄。有時候檔案可能受到保護，你需要包含 `-f` 旗標。

`cp` 指令可用來將檔案或目錄複製到新位置。與 `mv` 指令不同，`cp` 指令不會移除原始檔案－而且若要複製目錄，你需要傳入 `-r` 旗標。

`echo` 指令可以被視為 Bash 中等同於 `console.log()` 或 `print()` 函式（程式）。`echo` 接受一個用引號包住的字串引數，並將其列印到終端機。這看起來在終端機執行只是為了在終端機中取得文字似乎很愚蠢，但你其實可以將 `echo` 串鏈到控制符號。`>` 符號允許你指定一個檔案名稱來創建或覆寫該檔案，並寫入新的字串，例如 `echo "Naomi was here." > readme.md`，而 `>>` 符號則會將內容附加到該檔案。

還有許多 Bash 指令可供你使用，例如 `head`、`tail`、`chown` 和 `chmod`。在本課程中涵蓋所有指令是不可能的。但你可以使用 `man` 指令來查看幾乎任何指令的手冊或說明頁。

你很可能會記住你最常使用的指令。至於其他指令，你隨時都可以查看說明頁。
