# --description--

約分できない分数をレジリエント分数と呼びます。

さらに、分母のレジリエンス $R(d)$ を、その分母の真分数のうちレジリエントなものの割合と定義します。例えば、$R(12) = \frac{4}{11}$ です。

$1$ より大きい数 $d$ のレジリエンスは、オイラーのトーシェント関数 $φ$ を用いて $\frac{φ(d)}{d − 1}$ となります。

また、$1$ より大きい数 $n$ のコアレジリエンスを $C(n) = \frac{n − φ(n)}{n − 1}$ と定義します。

素数 $p$ のコアレジリエンスは $C(p) = \frac{1}{p − 1}$ です。

$C(n)$ が単位分数となる合成整数 $1 < n ≤ 2 × {10}^{11}$ のすべての合計を求めてください。
