# --description--

Gli array sono una struttura dati fondamentale nell'informatica. Tutti gli array archiviano collezioni ordinate di dati, ma a seconda del loro tipo possono funzionare in modo diverso dietro le quinte.

Il loro comportamento sottostante può avere un effetto importante sull'efficienza del programma, quindi impariamo a conoscere gli array dinamici e statici e le loro differenze, così potrai scegliere quello più efficiente per il tuo programma.

Inizieremo con gli array statici.

Gli **array statici** hanno una dimensione fissa. Archiviano gli elementi in posizioni di memoria adiacenti.

La dimensione di un array statico viene determinata quando l'array viene inizializzato. Una volta allocato quel blocco specifico di memoria, è fisso e non può essere modificato mentre il programma è in esecuzione. Questa è una caratteristica chiave degli array statici.

Archiviare gli elementi in posizioni di memoria adiacenti rende il processo di recupero dei dati più efficiente perché il programma può memorizzare la posizione del primo elemento e poi usare gli indici per fare semplici calcoli e trovare gli altri elementi in memoria.

Grazie a questo, consultare i valori di un array statico richiede un tempo costante `O(1)`, che è molto efficiente.

Puoi usare un array statico quando conosci in anticipo il numero di elementi che verranno archiviati. È anche utile quando i valori verranno consultati molto frequentemente, dato che l'operazione di consultazione è molto efficiente.

Tuttavia, questa struttura dati non può crescere o ridursi, quindi se il numero di elementi da archiviare può variare, dovresti usare invece un array dinamico.

Cercare di aumentare la dimensione di un array statico comporterebbe la creazione di un nuovo array e la copia di tutti gli elementi dal vecchio array a quello nuovo, il che è inefficiente. In quel caso, un array dinamico sarebbe molto meglio perché gestisce questo processo automaticamente.

JavaScript non ha array statici tradizionali a basso livello come C o Java, ma puoi simulare un array a dimensione fissa usando l'oggetto integrato `Array`:

```js
let numbers = new Array(3); // fixed size of 3
numbers[0] = 10;
numbers[1] = 20;
numbers[2] = 30;
```

Anche se JavaScript permette di ridimensionare, concettualmente questo rappresenta un array a lunghezza fissa.

Gli array in JavaScript sono dinamici per impostazione predefinita, quindi diamo un'occhiata a quelli.

Gli **array dinamici** sono più flessibili perché possono crescere o ridursi automaticamente mentre il programma è in esecuzione.

Funzionano tramite un meccanismo di ridimensionamento automatico che copia gli elementi in un nuovo array quando l'array originale è pieno. Il processo è fatto in modo efficiente perché la dimensione del nuovo array viene scelta in modo da rendere queste operazioni computazionalmente costose meno frequenti.

Consultare gli elementi di un array dinamico richiede un tempo costante `O(1)`, quindi questa operazione è molto efficiente.

Inserire un elemento nel mezzo dell'array richiede un tempo lineare `O(n)` perché gli elementi successivi devono essere riallocati.

Inserire un elemento alla fine dell'array richiede un tempo costante `O(1)` se c'è ancora spazio disponibile nell'array dinamico, ma se l'array è pieno e necessita di ridimensionamento, questa operazione ha una complessità `O(n)`.

Dovresti usare gli array dinamici quando non conosci in anticipo il numero di valori che dovrai archiviare nell'array. Sono anche utili quando inserirai e eliminerai elementi frequentemente.

Per creare un array in JavaScript, scrivi gli elementi all'interno di parentesi quadre, separati da virgole:

```js
let numbers = [3, 4, 5, 6];
```

Questo crea un array dinamico con quattro elementi.

Puoi consultare un elemento nell'array scrivendo il nome della variabile seguito da parentesi quadre contenenti l'indice.

Gli indici partono da 0 per il primo elemento e aumentano di 1 per ogni elemento successivo.

```js
numbers[0]; // 3
numbers[1]; // 4
numbers[2]; // 5
numbers[3]; // 6
```

Per aggiornare un valore, devi solo riassegnarlo:

```js
numbers[2] = 16
```

Ora, l'array diventa `[3, 4, 16, 6]`.

Per aggiungere un elemento alla fine dell'array, usa il metodo `.push()`:

```js
numbers.push(7)
```

Ora l'array diventa `[3, 4, 16, 6, 7]`:

Per inserire elementi a un indice specifico, JavaScript non ha `.insert()` come Python, ma puoi usare il metodo `.splice()`. La sintassi è questa:

```js
array.splice(index, deleteCount, newElement)
```

Ecco un esempio:

```js
numbers.splice(3, 0, 15);
```

Questo esempio significa:

- Inizia dall'indice 3
- Rimuovi 0 elementi
- Inserisci 15

Se l'array era `[3, 4, 16, 6, 7]`, ora diventa `[3, 4, 16, 15, 6, 7]`.

Puoi rimuovere un elemento a un indice specifico con lo stesso metodo `.splice`:

```js
numbers.splice(2, 1);
```

Questo significa iniziare dall'indice 2 e rimuovere 1 elemento.

Per rimuovere l'ultimo elemento, usa il metodo `.pop()`:

```js
numbers.pop();
```

Ci sono tantissimi altri metodi integrati per gli array che puoi consultare nella documentazione per lavorare con gli array in JavaScript.

Questa è la potenza degli array dinamici.

In generale, dovresti usare gli array statici quando conosci in anticipo il numero di elementi e hai bisogno di consultarli frequentemente, e usare gli array dinamici quando il numero di elementi è sconosciuto o variabile nel tempo.

Dovresti sempre considerare il compromesso tra la semplicità degli array statici e la flessibilità degli array dinamici. Entrambi sono utili per casi d'uso e scenari specifici. Saper scegliere il migliore per un dato problema fa parte delle abilità di problem solving che svilupperai gradualmente con la pratica.
