# --description--

Einige Symbole sind im Arbeitsalltag unverzichtbar geworden.

Dies sind vier der am häufigsten verwendeten Symbole im Spanischen:

| Symbol | Spanisch | Bedeutung |
| :---: | :--- | :--- |
| `.` | `Punto` | Punkt |
| `@` | `Arroba` | At-Zeichen |
| `-` | `Guión` | Bindestrich / Gedankenstrich |
| `_` | `Guión bajo` | Unterstrich |

Beachten Sie, dass das Wort `Guión` einen Akzent auf dem Buchstaben `o` trägt.

Camila wird Ihnen helfen, diese Symbole zu üben. Wiederholen Sie deren Laute, bis Sie sich sicherer fühlen, und ordnen Sie dann jedes Symbol seinem spanischen Namen zu.
