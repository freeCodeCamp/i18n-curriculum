# --description--

Дано множину {1,2,..., $n$}, визначимо $f(n, k)$ як число її підмножин з $k$ елементів, сума яких непарна. Наприклад, $f(5,3) = 4$, оскільки множина {1,2,3,4,5} має чотири підмножини з 3 елементів, сума яких непарна, а саме: {1,2,4}, {1,3,5}, {2,3,4} та {2,4,5}.

Коли всі три числа $n$, $k$ та $f(n, k)$ непарні, кажуть, що вони утворюють непарну трійку $[n, k, f(n, k)]$.

Існує рівно п’ять непарних трійок з $n ≤ 10$, а саме: $[1, 1, f(1, 1) = 1]$, $[5, 1, f(5, 1) = 3]$, $[5, 5, f(5, 5) = 1]$, $[9, 1, f(9, 1) = 5]$ та $[9, 9, f(9, 9) = 1]$.

Скільки існує непарних трійок з $n ≤ {10}^{12}$?
