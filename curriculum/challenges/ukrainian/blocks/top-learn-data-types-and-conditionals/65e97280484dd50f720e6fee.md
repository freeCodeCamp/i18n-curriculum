# --description--

У останньому прикладі у вас було два варіанти `if` або `else`. Але що, якщо ви хочете мати більше ніж два варіанти? Ви можете використати інструкцію `else if`, щоб додати більше умов у наш код. Інструкція `else if` дозволяє перевірити кілька умов і виконати блок коду, коли перша умова є істинною. Якщо перша умова хибна, вона перевірить наступну умову і так далі. Якщо жодна з умов не є істинною, буде виконано блок `else`.

Ось приклад інструкції `else if`:

```javascript
let x = 5;

if (x > 10) {
  console.log("x is greater than 10");
} else if (x > 5) {
  console.log("x is greater than 5");
} else {
  console.log("x is less than or equal to 5");
}
```

У цьому прикладі змінна `x` порівнюється з числом `10` за допомогою оператора `>`. Якщо `x` більше за `10`, буде виконано перший блок коду. Інакше інструкція `else if` перевірить наступну умову. Якщо `x` більше за `5`, буде виконано другий блок коду. Інакше буде виконано блок `else`.
