# --explanation--

在這句話中，Maria 使用 `they` 來指涉 **團隊的成員**。在對話的早些時候，她提到 `the team`，並在說 `I need to understand how the team feels about the new project deadline` 時使用了單數動詞。現在，她個別指涉這些人，因此她改用 `they`。另一個例子：

- `The committee decides on the rules.`－注意動詞使用單數形式以指涉整個群組。

- `They discuss the rules before making a decision.`－這裡有一個切換，強調群組中的個人（委員會的成員），所以使用複數形式。
