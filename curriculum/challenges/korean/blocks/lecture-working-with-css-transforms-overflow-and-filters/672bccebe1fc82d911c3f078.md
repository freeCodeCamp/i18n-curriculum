# --interactive--

CSS `filter` 속성은 웹 페이지 요소에 그래픽 효과를 적용할 수 있는 강력한 도구입니다. 원본 자산을 수정하지 않고도 이미지, 배경, 심지어 텍스트의 시각적 표현을 조정할 때 특히 유용합니다.

`filter` 속성은 흐림, 색상 변화, 대비 조정 등 다양한 효과를 만드는 데 사용할 수 있습니다. 필터 속성이 어떻게 작동하는지 살펴보고 몇 가지 일반적인 예시를 알아보겠습니다. `filter` 속성의 기본 구문은 간단합니다:

```css
selector {
  filter: function(amount);
}
```

여기서 `function`은 적용하려는 특정 필터 효과를 나타내고, `amount`는 일반적으로 효과의 강도를 결정하는 값입니다. 이제 몇 가지 일반적인 필터 함수와 그 용도를 살펴보겠습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: blur(2px);
}
```

:::

`blur` 함수는 요소에 가우시안 블러를 적용하며, 양은 픽셀 단위로 지정되고 블러 반경을 나타냅니다. 이 CSS 규칙은 페이지의 모든 이미지에 2픽셀 블러를 적용합니다. `blur` 효과는 디자인에 깊이를 주거나 이미지의 일부를 흐리게 할 때 유용합니다.

`brightness` 함수는 요소의 밝기를 조절합니다. `0%` 값은 요소를 완전히 검게 만들고, `100%` 이상의 값은 밝기를 증가시킵니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: brightness(150%);
}
```

:::

이 CSS 규칙은 이미지 요소의 밝기를 `50%`만큼 증가시킵니다. 밝기 조정은 이미지를 돋보이게 하거나 희미한 효과를 만들 때 사용할 수 있습니다.

`grayscale` 함수는 요소를 그레이스케일로 변환합니다. 양은 백분율로 정의되며, `100%`는 완전한 그레이스케일, `0%`은 이미지를 변경하지 않습니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: grayscale(100%);
}
```

:::

이 규칙은 이미지 요소를 완전한 그레이스케일로 변환합니다. `grayscale`는 빈티지 느낌을 내거나 페이지의 특정 요소를 덜 강조할 때 사용할 수 있습니다.

`sepia` 함수는 요소에 세피아 톤을 적용합니다. 그레이스케일과 마찬가지로 백분율 값을 사용합니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: sepia(80%);
}
```

:::

이 규칙은 이미지 요소에 `80%` 세피아 효과를 적용합니다. 세피아 효과는 빈티지하거나 오래된 느낌을 줄 때 좋습니다.

`hue-rotate` 함수는 요소에 색상 회전을 적용합니다. 값은 도 단위로 정의되며, 색상 원을 기준으로 회전하는 것을 나타냅니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: hue-rotate(90deg);
}
```

:::

이 규칙은 이미지 요소의 색상을 `90` 도만큼 회전시킵니다. 색상 회전은 환각적인 효과를 만들거나 이미지의 전체 색상 구성을 조정할 때 사용할 수 있습니다.

`filter` 속성의 가장 강력한 측면 중 하나는 여러 효과를 결합할 수 있다는 점입니다. 여러 필터를 같은 요소에 공백으로 구분하여 적용할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {  
  filter: contrast(150%) brightness(110%) sepia(30%);  
}
```

:::

이 규칙은 이미지 요소에 대비를 높이고, 밝기를 약간 증가시키며, 은은한 세피아 효과를 적용합니다.

필터를 결합하면 디자인 요구에 맞게 복잡하고 독특한 시각 효과를 만들 수 있습니다. CSS 필터 속성은 웹 요소의 시각적 조작을 창의적으로 할 수 있는 다재다능한 도구입니다.

가장 일반적인 필터 함수 몇 가지를 다뤘지만, `contrast`, `invert`, `saturate` 같은 다른 함수도 있습니다. 강력한 기능인 만큼 필터를 사용할 때는 사용자에게 부담을 주거나 접근성을 해치지 않도록 주의하세요.
