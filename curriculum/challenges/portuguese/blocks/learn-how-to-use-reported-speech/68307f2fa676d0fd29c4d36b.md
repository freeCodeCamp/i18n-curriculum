# --explanation--

`Was there` é a forma interrogativa (pergunta) de `There was`, que é usada para perguntar se algo existiu ou aconteceu no passado.Ele inverte o sujeito e o verbo para formar uma pergunta.Por exemplo:

- Houve um problema com o layout.  

- `Was there a problem with the layout?`

Isso ajuda a perguntar sobre a presença ou ocorrência de algo no passado.
