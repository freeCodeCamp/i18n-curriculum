# --description--

一个整数序列 $S = \\{s_i\\}$ 被称为 $n$-序列，如果它有 $n$ 个元素且每个元素 $s_i$ 满足 $1 ≤ s_i ≤ n$。因此，总共有 $n^n$ 个不同的 $n$-序列。

例如，序列 $S = \\{1, 5, 5, 10, 7, 7, 7, 2, 3, 7\\}$ 是一个 10 序列。

对于任意序列 $S$，令 $L(S)$ 为 $S$ 中具有相同值的最长连续子序列的长度。例如，对于上述给定的序列 $S$，$L(S) = 3$，因为有三个连续的 7。

令 $f(n) = \sum L(S)$，其中 $S$ 是所有长度为 n 的序列。

例如，$f(3) = 45$，$f(7) = 1\\,403\\,689$ 和 $f(11) = 481\\,496\\,895\\,121$。

查找 $f(7\,500\,000)\bmod 1\,000\,000\,009$。
