# --description--

Express ofrece una variedad de funciones de middleware integradas que ayudan a realizar tareas comunes, como servir archivos estáticos, interpretar cuerpos de solicitudes JSON y más. También puedes usar middleware de terceros para agregar funcionalidad.

**Aquí tienes algunos ejemplos de middleware integrado**:

1. `express.json()`: Un middleware integrado que interpreta cargas útiles JSON entrantes.

```js
app.use(express.json())
```

1. `express.static()`: Un middleware para servir archivos estáticos como imágenes, CSS y archivos JavaScript.

```js
app.use(express.static('public'))
```

1. `express.urlencoded()`: Un middleware que interpreta solicitudes entrantes con cargas útiles codificadas en URL, el formato que típicamente envían los formularios HTML. Hace que los datos interpretados estén disponibles en `req.body`.

```js
app.use(express.urlencoded({ extended: true }))
```

**Ahora, aquí tienes algunos ejemplos de middleware de terceros:**

1. `cors`: Un middleware de terceros popular para habilitar el intercambio de recursos de origen cruzado.

```bash
npm install cors
```

```js
const cors = require('cors')
app.use(cors())
```

1. `morgan`: Un middleware de registro para solicitudes HTTP.

```bash
npm install morgan
```

```js
const morgan = require('morgan')
app.use(morgan('tiny'))
```

Ahora, veamos un ejemplo de uso tanto de middleware integrado como de terceros:

```js
const express = require('express')
const cors = require('cors')
const morgan = require('morgan')
const app = express()

app.use(express.json()) // Built-in middleware to parse JSON
app.use(cors())       // Third-party middleware for CORS
app.use(morgan('tiny')) // Third-party middleware for logging

app.get('/', (req, res) => {
  res.send('Hello, World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* El middleware integrado `express.json()` interpreta cargas útiles JSON entrantes.
* El middleware de terceros `cors` permite solicitudes de origen cruzado, y `morgan` registra las solicitudes HTTP entrantes.

Es importante notar que las funciones de middleware se agregan en el orden en que se llaman con `app.use()`.

Express proporciona middleware integrado esencial para manejar tareas comunes como la interpretación de JSON y el servicio de archivos estáticos. También puedes extender la funcionalidad con middleware de terceros para capacidades adicionales.
