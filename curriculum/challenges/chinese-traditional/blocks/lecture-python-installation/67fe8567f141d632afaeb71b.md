# --description--

在所有的工作坊和實驗中，你都會使用 freeCodeCamp 的 Python 編輯器。但學會如何在你的區域電腦上設定 Python 也很重要。

在 Windows 和 Mac 上安裝 Python 最簡單的方法是從官方 Python 網站下載安裝程式。我們稍後也會介紹如何在 Linux 上執行 Python。

前往 `https://www.python.org/` 並將滑鼠暫留在「Downloads」上。會跳出一個視窗，顯示你作業系統（OS）目前的 Python 版本。

我們先介紹如何在 macOS 電腦上安裝 Python：

- 點擊剛剛視窗中顯示的 Python 版本按鈕，會自動開始下載 `.pkg` 安裝檔。
- 下載完成後，打開 `.pkg` 安裝程式，然後在開啟的視窗中點擊「Continue」。
- 持續點擊「Continue」直到看到「Installation Type」部分，然後點擊「Install」按鈕。
- 如有需要，輸入你的密碼，開始安裝。
- 安裝完成後，會看到一則恭喜訊息，表示 Python 已成功安裝。
- 點擊「Close」按鈕，完成安裝！

你可以打開終端機，執行 `python --version` 或 `python3 --version` 來驗證安裝。

你也可以在終端機執行 `python` 或 `python3` 來開啟 Python 直譯器。Python 直譯器是讀取 Python 程式碼、將其轉換成電腦能理解的指令並執行的程式。

<dfn>終端機</dfn>是一種以文字為基礎的介面，讓你透過輸入指令與電腦互動。每個作業系統都會附帶預設的終端機應用程式。macOS 可以使用 Terminal 應用程式，Windows 可以使用命令提示字元或 PowerShell，Linux 則依桌面環境不同，有 GNOME Terminal 或 Konsole 等預設終端機。

請注意，在某些較舊的 macOS 和 Linux 系統中，`python` 可能是保留給 Python 2 使用，而 `python3` 則是專門給 Python 3。如果你執行 `python --version`，看到像 `Python 2.7.18` 這樣的 Python 2 版本，那可能是你的作業系統依賴某些用舊版 Python 撰寫的軟體。如果是這樣，你應該改用 `python3` 來執行 Python 程式。Python 2 已經終止支援，不應用於任何新開發。

在 Windows 上安裝 Python，請依照以下步驟：

- 前往 `https://www.python.org/`，將滑鼠暫留在「Downloads」上。會看到一個視窗寫著「Download for Windows」和一個帶有目前 Python 版本的下載按鈕。
- 點擊版本號，會自動開始下載 Windows 可執行檔（`.exe`）。
- 下載完成後，雙擊 Python 安裝程式，並依照指示操作。
- 當看到 `Add python.exe to Path` 選項時，勾選該選項，然後點擊 `Install Now`。這樣以後會更方便。

你可以打開命令列殼層（例如 PowerShell），執行 `python --version` 來驗證安裝。你也可以執行 `python` 來開啟 Python 直譯器。

在 Linux 上，大多數主要發行版如 Ubuntu、Debian 和 Fedora 都預載 Python。

只要打開終端機，執行 `python --version` 或 `python3 --version`：

如果這兩個指令都沒有顯示 Python 版本，你可以在 `https://www.python.org` 搜尋適合你 Linux 發行版的安裝套件，或在線上搜尋該發行版推薦的 Python 安裝方式。
