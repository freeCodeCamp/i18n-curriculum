# --description--

Der Ausdruck `crazy part` wird verwendet, um auszudrücken, dass etwas überraschend oder unerwartet ist. Es ist so, als würde man `the surprising thing is` oder `the unexpected thing is` sagen.

Zum Beispiel `The crazy part is that he finished all the work in one day.` Das bedeutet, es ist überraschend, dass er es so schnell schaffen konnte.
