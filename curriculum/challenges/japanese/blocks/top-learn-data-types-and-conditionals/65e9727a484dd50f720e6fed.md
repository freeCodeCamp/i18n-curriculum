# --description--

どのプログラミング言語でも、値を比較して判断を下す必要があります。たとえば、天気が晴れなら外出し、そうでなければ家にいるといった場合です。別の例としては、ユーザーがゲームでレベルアップするのに十分なポイントを持っているかどうかを確認することがあります。

`if-else` 条件文は、JavaScriptで判断を下すために使われます。比較演算子と一緒に使われることが多く、値を比較して結果に基づいて判断を行います。

if文の例は次の通りです。

```javascript
let x = 5;

if (x > 3) {
  console.log("x is greater than 3");
} else {
  console.log("x is less than or equal to 3");
}
```

この例では、`x` 変数が `3` という数値と `>` 演算子で比較されています。もし `x` が `3` より大きければ、最初のコードブロックが実行されます。そうでなければ、2番目のコードブロックが実行されます。
