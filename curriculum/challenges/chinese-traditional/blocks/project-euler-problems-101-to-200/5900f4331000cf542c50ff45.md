# --description--

對於分母界限 `d`，對實數 $x$ 最佳的近似值是一個分母 $s ≤ d$ 的約分後有理數 $\frac{r}{s}$，使得任何比 $\frac{r}{s}$ 更接近 $x$ 的有理數 $\frac{p}{q}$ 都有 $q > d$。

通常對所有分母界限來說，實數的最佳近似值是唯一決定的。然而，也有一些例外，例如 $\frac{9}{40}$ 在分母界限 $6$ 時有兩個最佳近似值 $\frac{1}{4}$ 和 $\frac{1}{5}$。如果存在至少一個分母界限，使得實數 $x$ 擁有兩個最佳近似值，我們稱該實數 $x$ 為模糊數。顯然，模糊數必定是有理數。

在 0 ＜ x ＜ `\frac{1}{100}` 範圍內，有多少個模糊數 $x = \frac{p}{q}$，其分母 $q$ 不超過 ${10}^8$？
