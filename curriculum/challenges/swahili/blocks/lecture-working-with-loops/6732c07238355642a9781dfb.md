# --interactive--

Kauli ya `break` hutumika kutoka katika mzunguko mapema, wakati kauli ya `continue` hutumika kuruka mzunguko wa sasa na kuendelea na ule unaofuata.

Hapa kuna mfano wa kutumia kauli ya `break` katika mzunguko wa `for`:

:::interactive_editor

```js
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}
```

:::

Katika mfano hapo juu, mzunguko unaanza kuhesabu kutoka `0` na wakati `i` iko chini ya `10`, mzunguko utaendelea kuendeshwa.

Ndani ya mzunguko, tunakagua kama `i` ni sawa na `5`. Ikiwa ni sawa, tunatumia kauli ya `break` kutoka katika mzunguko mapema. Ikiwa si sawa, tunachapisha thamani ya `i` kwenye konsoli. Hivyo matokeo ya msimbo yatachapisha nambari `0`, `1`, `2`, `3`, na `4`.

Kauli ya `break` ni muhimu unapotaka kutoka katika mzunguko mapema kulingana na hali fulani. Kwa mfano, ikiwa unatafuta thamani maalum katika safu ya data, unaweza kutumia kauli ya `break` kutoka katika mzunguko mara tu unapopata thamani hiyo.

Wakati mwingine unaweza kutaka kuruka mzunguko fulani wa mzunguko bila kutoka kabisa. Hapa ndipo kauli ya `continue` inapotumika. Hapa kuna mfano wa kutumia kauli ya `continue` katika mzunguko wa `for`:

:::interactive_editor

```js
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}
```

:::

Kama awali, tumeanzisha `i` kuwa `0` na tuna hali itakayowezesha mzunguko kuendelea mradi `i` iko chini ya `10`.

Ndani ya mzunguko, wakati `i` ni sawa na `5`, tunatumia kauli ya `continue` kuruka mzunguko wa sasa na kuendelea na ule unaofuata.

Matokeo ya msimbo huu yatachapisha nambari `0`, `1`, `2`, `3`, `4`, `6`, `7`, `8`, na `9`. Nambari `5` imerukwa kwa sababu ya kauli ya `continue`.

Jambo jingine unaloweza kufanya kwa kauli za `break` na `continue` ni kutumia lebo kuonyesha mzunguko gani unataka kutoka au kuendelea nao.

Hii ni muhimu wakati una mizunguko zilizopangwa ndani na unataka kudhibiti mtiririko wa mzunguko wa nje kutoka ndani ya mzunguko wa ndani.

Hapa kuna mfano wa kutumia lebo na kauli ya `break`:

:::interactive_editor

```js
outerLoop: for (let i = 0; i < 3; i++) {
  innerLoop: for (let j = 0; j < 3; j++) {
    if (i === 1 && j === 1) {
      break outerLoop;
    }
    console.log(`i: ${i}, j: ${j}`);
  }
}
```

:::

Katika mfano huu, tuna mzunguko wa nje wa `for` ulio na lebo `outerLoop` na mzunguko wa ndani wa `for` ulio na lebo `innerLoop`.

Wakati `i` ni sawa na `1` na `j` ni sawa na `1`, tunatumia kauli ya `break` na lebo ya `outerLoop` kutoka katika mzunguko wa nje mapema. Hii itatoka katika mizunguko yote ya ndani na nje.

Matokeo ya msimbo huu yatachapishwa kwenye konsoli kama ifuatavyo:

```md
"i: 0, j: 0"
"i: 0, j: 1"
"i: 0, j: 2"
"i: 1, j: 0"
```

Mara nyingi hutahitaji kutumia lebo na kauli za `break` na `continue`, lakini ni vizuri kujua kuwa una chaguo hilo ikiwa utalihitaji.
