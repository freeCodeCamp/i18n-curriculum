## Aspectos Destacados de la Gramática

### Hablando sobre Origin

Para decir de dónde eres, debes usar: **`Soy` + `de` + [país]**. Por ejemplo:

- `Soy de El Salvador.`

- `Soy de Guatemala.`

Para saber de dónde es alguien, pregunta `¿De dónde eres?`.

<br>

### Hablando sobre la nacionalidad

Para decir tu nacionalidad, debes usar: **`Soy` + [nacionalidad]**.

- `Soy salvadoreña.`

- `Soy guatemalteco.`

Estas son algunas de las nacionalidades que has aprendido hasta ahora:

| País | Forma masculina | Forma femenina |
| :--- | :--- | :--- |
| **`El Salvador`** | `salvadoreño` | `salvadoreña` |
| **`Guatemala`** | `guatemalteco` | `guatemalteca` |
| **`Colombia`** | `colombiano` | `colombiana` |
| **`Argentina`** | `argentino` | `argentina` |
| **`Uruguay`** | `uruguayo` | `uruguaya` |
| **`Puerto Rico`** | `puertorriqueño` | `puertorriqueña` |
| **`Chile`** | `chileno` | `chilena` |
| **`Perú`** | `peruano` | `peruana` |

En español, las nacionalidades típicamente cambian con el género. Si una nacionalidad termina con `-o`, esta última letra usualmente se reemplaza por una `-a` para crear la forma femenina.

Sin embargo, las nacionalidades que terminan con una consonante a menudo añaden un `-a` para crear la forma femenina. Por ejemplo, la forma masculina de alguien de `España` ("Spain") es `español`, mientras que la forma femenina es `española`.

Las nacionalidades que terminan en `-e` o `-í` suelen ser iguales para ambos géneros. Por ejemplo, la forma masculina de alguien de `Estados Unidos` ("United States") es `estadounidense`, mientras que la forma femenina también es `estadounidense`.

<br>

### Hablando Sobre la Profesión

Para decir tu profesión, debes usar: **`Soy` + [profesión]**. Por ejemplo:

- `Soy representante de recursos humanos.`

- `Soy analista junior.`

Usa esta pregunta para preguntar a alguien sobre su profesión: `¿A qué te dedicas?`.

<br>

### Hablando Sobre la Edad

Para decir tu edad en español, debes usar: **`Tengo` + [número] + `años`**. Por ejemplo:

- `Tengo veintiún años.`

- `Tengo treinta y cinco años.`

Puedes usar esta pregunta para preguntar la edad de alguien: `¿Cuántos años tienes?`.

<br>

### Ser Cortés Durante las Presentaciones

Hay tres expresiones de cortesía importantes en español.

**`Mucho gusto`** significa "Nice to meet you". Es una forma educada de decir que estás contento de conocer a alguien. Esta frase es neutral en cuanto al género.

**`Encantada`** también significa "Mucho gusto", pero es una forma más expresiva de decir que estás encantado de conocer a alguien. Debe concordar con el género del hablante. Su forma femenina es `Encantada`, mientras que su forma masculina es `Encantado`.

**`Igualmente`** significa "Igualmente". Se usa para devolver cortésmente lo que la otra persona expresó, sin repetirlo.
