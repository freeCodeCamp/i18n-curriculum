# --description--

Kwa siku ya mwisho ya Wiki ya Anga, utapewa uzito kwa kilo (kg) wa mzigo unaotaka kutuma angani. Tambua kiasi cha mafuta kinachohitajika kutuma mzigo wako angani kwa kutumia sheria zifuatazo:

- Mizinga ya roketi inahitaji kilo 1 ya mafuta kwa kila kilo 5 za uzito wa kuinua.
- Mafuta yenyewe yana uzito. Hivyo unapoongeza mafuta, uzito wa kuinua unaongezeka, jambo linalohitaji mafuta zaidi, ambayo huongeza uzito, na kadhalika.
- Ili kuhesabu jumla ya mafuta yanayohitajika: anza na uzito wa mzigo, hesabu mafuta yanayohitajika kwa huo uzito, ongeza mafuta hayo kwenye jumla ya uzito, na hesabu tena. Rudia mchakato huu hadi mafuta ya ziada yanayohitajika yapungue kilo 1, kisha simama.
- Puuza uzito wa roketi yenyewe. Hesabu tu mafuta yanayohitajika kuinua mzigo na mafuta yake mwenyewe.

Kwa mfano, ukipata mzigo wa kilo 50, utahitaji kilo 10 za mafuta kuinua mzigo huo (mzigo / 5), jambo linaloongeza jumla ya uzito kuwa kilo 60, ambayo inahitaji kilo 12 kuinua (kilo 2 za ziada), jambo linaloongeza jumla ya uzito kuwa kilo 62, ambayo inahitaji kilo 12.4 kuinua - kilo 0.4 za ziada - ambazo ni chini ya kilo 1 za ziada, hivyo tunasimama hapa. Jumla ya uzito wa kuinua ni kilo 62.4, ambapo 50 ni mzigo wa awali na 12.4 ni mafuta.

- Rudisha kiasi cha mafuta kinachohitajika kilichozungushwa hadi sehemu moja ya desimali.

```js
function launchFuel(payload) {
  let totalMass = payload;
  let additionalFuel = totalMass / 5;
  let totalFuel = additionalFuel;

  while (additionalFuel >= 1) {
    totalMass += additionalFuel;
    additionalFuel = (totalMass / 5) - totalFuel;
    totalFuel += additionalFuel;
  }

  return Number(totalFuel.toFixed(1));
}
```
