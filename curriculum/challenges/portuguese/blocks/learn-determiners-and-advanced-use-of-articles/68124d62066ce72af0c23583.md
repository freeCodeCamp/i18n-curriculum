# --explanation--

`Every two weeks` significa que algo acontece uma vez em cada período de duas semanas - isso é frequentemente chamado de `bi-weekly`.

`Any problems` usa `any` para se referir a quaisquer problemas que possam acontecer, mesmo que você ainda não saiba quais são.

**Nota:** `Bi-weekly` às vezes pode significar "duas vezes por semana" ou "a cada duas semanas", mas aqui significa "a cada duas semanas" com base no contexto.
