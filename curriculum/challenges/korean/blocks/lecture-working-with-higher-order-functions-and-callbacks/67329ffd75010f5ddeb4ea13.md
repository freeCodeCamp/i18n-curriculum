# --interactive--

JavaScript에서 콜백 함수는 다른 함수에 인수로 전달되어, 외부 함수가 특정 시점에 이 함수를 호출할 수 있게 하는 함수입니다.

이 개념은 JavaScript의 여러 측면을 이해하는 데 기본이 되며, `forEach` 메서드가 어떻게 작동하는지 이해하는 데도 중요합니다.

간단한 상황에서 콜백 함수가 무엇인지부터 시작해 보겠습니다.

함수가 있고 그 안에서 어떤 작업을 여러 번 하고 싶다고 상상해 보세요. 모든 코드를 한꺼번에 큰 함수에 작성하는 대신, 특정 작업이 필요할 때마다 실행할 두 번째 함수(콜백)를 전달할 수 있습니다. 이렇게 하면 더 유연하고 모듈화된 코드를 작성할 수 있습니다.

이제 `forEach` 메서드를 소개하겠습니다. `forEach`는 JavaScript 배열에 내장된 메서드입니다. 배열의 각 요소를 순회하며 각 요소에 대해 작업을 수행할 수 있게 해줍니다. 수행할 작업은 `forEach`에 제공하는 콜백 함수 안에 정의합니다.

다음은 `forEach`가 콜백 함수와 함께 작동하는 예시입니다:

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];

numbers.forEach(function(number) {
  console.log(number * 2);
});
```

:::

이 예시에서는 숫자 배열이 있습니다. 이 배열에 `forEach` 메서드를 사용하고, `forEach`에 콜백 함수를 인수로 전달합니다.

이 콜백 함수는 배열의 각 요소를 나타내는 하나의 매개변수를 받습니다. `forEach` 메서드는 배열의 각 요소마다 이 콜백 함수를 한 번씩 호출합니다.

이 경우 콜백 함수는 각 숫자에 `2`를 곱하고 결과를 출력합니다. 따라서 이 코드를 실행하면 다음과 같은 결과가 나옵니다:

```md
2
4
6
8
10
```

콜백 함수는 배열의 각 요소마다 순서대로 한 번씩 호출된다는 점을 이해하는 것이 중요합니다. `forEach`가 반복을 대신 처리하므로, 직접 `for` 반복문을 작성할 필요가 없습니다.

콜백으로 화살표 함수를 사용할 수도 있는데, 이렇게 하면 코드가 더 간결해집니다:

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];
numbers.forEach(number => console.log(number * 2));
```

:::

이 코드는 이전 예시와 정확히 같은 일을 하지만 코드가 더 짧습니다.

`forEach`의 콜백 함수는 실제로 최대 세 개의 인수를 받을 수 있습니다: 현재 요소, 현재 요소의 인덱스, 그리고 `forEach`가 호출된 배열입니다.

세 가지 인수를 모두 사용하는 예시는 다음과 같습니다:

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];
numbers.forEach((number, index, array) => {
  console.log(`Element ${number} is at index ${index} in array ${array}`);
});
```

:::

이 코드는 각 요소, 그 인덱스, 그리고 원래 배열에 관한 정보를 출력합니다.

콜백 함수와 `forEach` 같은 메서드를 이해하는 것은 JavaScript를 배우는 데 중요합니다. 이들은 특히 앞으로 배우게 될 비동기 프로그래밍에서 많은 고급 개념의 기초가 됩니다.
