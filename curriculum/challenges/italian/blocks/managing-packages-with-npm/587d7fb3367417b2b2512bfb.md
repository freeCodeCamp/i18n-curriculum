Il file `package.json` è il centro di ogni progetto Node.js o pacchetto npm. Archivia informazioni sul tuo progetto. È costituito da un singolo oggetto JSON dove le informazioni sono archiviate in coppie chiave-valore. Ci sono solo due campi obbligatori; `name` e `version`, ma è buona pratica fornire informazioni aggiuntive.

Puoi creare il file `package.json` dal terminale usando il comando `npm init`. Questo avvierà una configurazione guidata. Usare `npm init` con la flag `-y` genererà il file senza farti alcuna domanda, `npm init -y`.

Se guardi l’albero dei file del tuo progetto, troverai il file `package.json` al livello più alto dell’albero. Questo è il file che migliorerai nelle prossime sfide.

Una delle informazioni più comuni in questo file è il campo `author`. Specifica chi ha creato il progetto e può essere una stringa o un oggetto con dettagli di contatto o altre informazioni. Si consiglia un oggetto per progetti più grandi, ma una semplice stringa come nell’esempio seguente va bene per questo progetto.

```json
"author": "Jane Doe",
```

# --instructions--

Aggiungi il tuo nome come `author` del progetto nel file `package.json`.

**Nota:** Ricorda che stai scrivendo JSON, quindi tutti i nomi dei campi devono usare le virgolette doppie (") e essere separati da una virgola (,).
