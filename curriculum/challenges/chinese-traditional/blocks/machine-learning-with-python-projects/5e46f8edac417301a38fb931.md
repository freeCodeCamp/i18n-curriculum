# --instructions--

在這個挑戰中，你需要創建一個機器學習模型，將 SMS 訊息分類為「ham」或「spam」。所謂「ham」訊息是朋友傳送的正常訊息。「spam」訊息則是廣告或公司傳送的訊息。

你應該創建一個名為 `predict_message` 的函式（程式），該函式接受一個訊息字串作為引數並傳回一個列表。列表中的第一個元素應該是一個介於零和一之間的數字，用以顯示「ham」（0）或「spam」（1）的可能性。列表中的第二個元素應該是字「ham」或「spam」，取決於哪一個最有可能。

在這個挑戰中，你將使用 <a href="http://www.dt.fee.unicamp.br/~tiago/smsspamcollection/" target="_blank" rel="noopener noreferrer nofollow">SMS Spam Collection</a> 資料集。該資料集已經被分成訓練資料和測試資料群組。
