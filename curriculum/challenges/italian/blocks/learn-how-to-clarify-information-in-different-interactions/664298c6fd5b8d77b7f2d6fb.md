# --description--

Controllare `make sure` significa verificare qualcosa con attenzione per essere sicuri che sia corretto o che qualcosa sia stato fatto. Per esempio, puoi dire `I wanted to make sure the door is locked` quando controlli la serratura per confermare che sia sicura.

`Choices` sono le diverse opzioni tra cui puoi scegliere. Per esempio, se stai scegliendo un gusto di gelato, le `choices` potrebbero essere cioccolato, vaniglia o fragola. Significa che hai più di una possibilità tra cui selezionare.
