# --description--

设 $C$ 为半径为 $r$ 的圆，$x^2 + y^2 = r^2$。我们选择两点 $P(a, b)$ 和 $Q(-a, c)$，使得通过 $P$ 和 $Q$ 的直线与 $C$ 相切。

例如，四元组 $(r, a, b, c) = (2, 6, 2, -7)$ 满足此属性。

设 $F(R, X)$ 为满足该属性且满足 $0 &lt; r ≤ R$ 和 $0 &lt; a ≤ X$ 的整数四元组 $(r, a, b, c)$ 的数量。

我们可以验证 $F(1, 5) = 10$、$F(2, 10) = 52$ 和 $F(10, 100) = 3384$。

查找 $F({10}^8, {10}^9) + F({10}^9, {10}^8)$。
