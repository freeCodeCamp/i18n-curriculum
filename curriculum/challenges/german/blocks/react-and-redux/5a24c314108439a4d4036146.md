# --description--

Die `mapDispatchToProps()`-Funktion wird verwendet, um bestimmten Action Creators Ihre React-Komponenten bereitzustellen, damit diese Aktionen gegen das Redux speichern auslösen können. Sie ähnelt in der Struktur der `mapStateToProps()`-Funktion, die Sie in der letzten Aufgabe geschrieben haben. Sie gibt ein Objekt zurück, das Dispatch-Aktionen auf Eigenschaftsnamen abbildet, die zu Komponenten- `props` werden. Anstatt jedoch ein Stück `state` zurückzugeben, gibt jede Eigenschaft eine Funktion zurück, die `dispatch` mit einem Action Creator und allen relevanten Aktionsdaten aufruft. Sie haben Zugriff auf diese `dispatch`, weil sie als Parameter an `mapDispatchToProps()` übergeben wird, wenn Sie die Funktion definieren, genau wie Sie `state` an `mapStateToProps()` übergeben haben. Im Hintergrund verwendet React Redux das `store.dispatch()` von Redux, um diese Dispatches mit `mapDispatchToProps()` durchzuführen. Dies ähnelt der Verwendung von `store.subscribe()` für Komponenten, die auf `state` abgebildet sind.

Zum Beispiel haben Sie einen `loginUser()`-Action Creator, der einen `username` als Aktionsnutzlast entgegennimmt. Das von `mapDispatchToProps()` für diesen Action Creator zurückgegebene Objekt würde ungefähr so aussehen:

```jsx
{
  submitLoginUser: function(username) {
    dispatch(loginUser(username));
  }
}
```

# --instructions--

Der Code-Editor stellt einen Action Creator namens `addMessage()` bereit. Schreiben Sie die Funktion `mapDispatchToProps()`, die `dispatch` als Argument entgegennimmt und dann ein Objekt zurückgibt. Das Objekt sollte eine Eigenschaft `submitNewMessage` haben, die auf die Dispatch-Funktion gesetzt ist, welche einen Parameter für die neue Nachricht entgegennimmt, die hinzugefügt wird, wenn sie `addMessage()` auslöst.

# --hints--

`addMessage` sollte ein Objekt mit den Schlüsseln `type` und `message` zurückgeben.

```js
assert(
  (function () {
    const addMessageTest = addMessage();
    return (
      addMessageTest.hasOwnProperty('type') &&
      addMessageTest.hasOwnProperty('message')
    );
  })()
);
```

`mapDispatchToProps` sollte eine Funktion sein.

```js
assert(typeof mapDispatchToProps === 'function');
```

`mapDispatchToProps` sollte ein Objekt zurückgeben.

```js
assert(typeof mapDispatchToProps() === 'object');
```

Das Auslösen von `addMessage` mit `submitNewMessage` von `mapDispatchToProps` sollte eine Nachricht an die Dispatch-Funktion zurückgeben.

```js
assert(
  (function () {
    let testAction;
    const dispatch = (fn) => {
      testAction = fn;
    };
    let dispatchFn = mapDispatchToProps(dispatch);
    dispatchFn.submitNewMessage('__TEST__MESSAGE__');
    return (
      testAction.type === 'ADD' && testAction.message === '__TEST__MESSAGE__'
    );
  })()
);
```
