# --interactive--

Wenn Sie mit Hintergrundbildern in CSS arbeiten, stehen Ihnen mehrere Eigenschaften zur Verfügung, um zu steuern, wie diese Bilder angezeigt werden.

Die Haupt-Eigenschaften, auf die wir uns konzentrieren, sind `background-size`, `background-repeat`, `background-position` und `background-attachment`.

Schauen wir uns zuerst die `background-image`-Eigenschaft an:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

Das obige CSS setzt ein Katzen-Hintergrundbild für das `body`-Element.

Wenn Sie die Größe des Hintergrundbilds festlegen möchten, können Sie die `background-size`-Eigenschaft verwenden.

Sie können `contain` verwenden, um das Bild so groß wie möglich zu skalieren, ohne es zuzuschneiden oder zu verzerren.

Hier ist ein Beispiel mit `background-size: contain`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    min-height: 100px;
  }
</style>
```

:::

Wir setzen das `min-height` auf `100px`, damit das Hintergrundbild sichtbar ist und das Layout eine Grundhöhe beibehält, sodass das Design auch bei minimalem Inhalt ausgewogen und ansprechend wirkt.

Wenn wir die `background-size`-Eigenschaft auf den Wert `cover` ändern, skaliert das Hintergrundbild, um das gesamte `body`-Element abzudecken und dabei das Seitenverhältnis beizubehalten.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    min-height: 100px;
  }
</style>
```

:::

In den vorherigen Beispielen haben Sie wahrscheinlich bemerkt, dass sich das Hintergrundbild kontinuierlich wiederholt.

Standardmäßig wiederholen sich Hintergrundbilder sowohl horizontal als auch vertikal, um das gesamte Element zu füllen. Sie können dieses Verhalten jedoch steuern.

Sie können die `background-repeat`-Eigenschaft mit dem Wert `no-repeat` verwenden.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

Wenn `background-size` auf `contain` und `background-repeat` auf `no-repeat` gesetzt ist, wird das Bild auf dem Bildschirm nicht mehr wiederholt.

Wenn Sie das Hintergrundbild horizontal wiederholen möchten, können Sie den Wert `repeat-x` für die `background-repeat`-Eigenschaft verwenden.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-x;
    min-height: 100px;
  }
</style>
```

:::

Wenn Sie das Hintergrundbild vertikal wiederholen möchten, können Sie den Wert `repeat-y` für die `background-repeat`-Eigenschaft verwenden.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-y;
    min-height: 100px;
  }
</style>
```

:::

Um ein Hintergrundbild auf dem Bildschirm zu positionieren, können Sie die `background-position`-Eigenschaft verwenden.

Die `background-position`-Eigenschaft ermöglicht es Ihnen, festzulegen, wo im Element das Hintergrundbild erscheint. Sie können Schlüsselwörter wie `top`, `bottom`, `left`, `right` und `center` oder spezifische Pixel- oder Prozentwerte verwenden.

Hier ist ein Beispiel, das `center` und `top` für das `background-position` verwendet:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

Dieser CSS-Code positioniert das Hintergrundbild horizontal in der Mitte und vertikal oben im Element.

Zuletzt bestimmt `background-attachment`, ob das Hintergrundbild mit dem Inhalt mitscrollt oder beim Scrollen der Seite fixiert bleibt.

Die Hauptwerte sind `scroll` (standardmäßig), bei dem das Hintergrundbild mit dem Inhalt mitscrollt, und `fixed`, bei dem das Hintergrundbild an der gleichen Position auf dem Bildschirm bleibt.

Hier ist ein Beispiel, das den Wert `fixed` für die `background-attachment`-Eigenschaft verwendet:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-position: center top;
    background-attachment: fixed;
  }
</style>
```

:::

Dieser CSS-Code sorgt dafür, dass das Hintergrundbild an Ort und Stelle bleibt, selbst wenn die Seite gescrollt wird.

Wenn Sie einige der Eigenschaften in einer Zeile kombinieren möchten, können Sie das mit der Kurzschreibweise `background` tun.

Hier ist ein Beispiel:

:::interactive_editor

```html
<style>
  body {
    background: center top fixed
      url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

Das obige Beispiel kombiniert `background-image`, `background-position` und `background-attachment` in einer einzigen Zeile.

Wenn Sie diese Eigenschaften beherrschen, haben Sie eine großartige Kontrolle darüber, wie Hintergrundbilder in Ihren Webdesigns angezeigt werden, was zu ansprechenderen und funktionaleren Layouts führt.
