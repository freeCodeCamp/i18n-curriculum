# --description--

Un `comentario` es una nota o explicación agregada al código por el programador. Los comentarios no se ejecutan como parte del programa; en cambio, ayudan a cualquiera que lea el código a comprender su propósito, funcionalidad o cualquier detalle específico. Son esenciales para hacer que el código sea legible y mantenible.
