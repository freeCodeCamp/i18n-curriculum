# --description--

`flex-grow` erwartet als Wert eine einzelne Zahl, und diese Zahl wird als „Wachstumsfaktor“ des Flex-Elements verwendet. Wenn Sie `flex: 1` auf jedes div innerhalb unseres Containers angewendet haben, haben Sie jedem div gesagt, dass es im gleichen Maße wachsen soll. Das Ergebnis ist, dass jedes div genau gleich groß wird. Wenn Sie stattdessen `flex: 2` nur einem der divs hinzufügen, wächst dieses div auf die doppelte Größe der anderen.

## --text--

Was bedeutet es, `flex: 2` auf ein bestimmtes div innerhalb eines Flex-Containers zu setzen, während andere divs `flex: 1` haben?

## --answers--

Der Container wird schrumpfen, um das größere div aufzunehmen.

---

Alle divs behalten aufgrund widersprüchlicher `flex`-Werte die gleiche Größe.

---

Das bestimmte div wird auf die doppelte Größe der anderen wachsen.

---

Alle divs schrumpfen gleichermaßen, um sich an die neuen `flex`-Werte anzupassen.
