# --description--

색상 이론은 색상이 서로 어떻게 상호작용하는지, 그리고 우리의 인식에 어떤 영향을 미치는지 연구하는 학문입니다. 색상 관계, 색상 조화, 그리고 색상이 심리적으로 미치는 영향 등을 다룹니다. 이 세계로 함께 들어가 봅시다. 색상은 기본색, 2차색, 3차색으로 분류할 수 있습니다.

기본색인 노랑, 파랑, 빨강은 모든 다른 색상이 파생되는 근본적인 색조입니다.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/lecture-what-is-color-theory-in-design-1.png" alt="Color classification diagram" />

2차색은 두 기본색을 같은 비율로 섞어서 만듭니다. 초록, 주황, 보라가 2차색의 예입니다.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/lecture-what-is-color-theory-in-design-2.png" alt="Primary colors diagram" />

예를 들어, 초록은 노랑과 파랑을 섞은 결과입니다.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/lecture-what-is-color-theory-in-design-3.png" alt="Secondary colors diagram" />

3차색은 기본색과 인접한 2차색을 섞어서 만듭니다. 노랑-초록, 파랑-초록, 파랑-보라가 3차색의 예입니다.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/lecture-what-is-color-theory-in-design-4.png" alt="Green color mixing example" />

이것은 색상 이론 세계에서 기본적인 분류법이지만, 색상을 분류하는 다른 방법도 있습니다. 색상은 온도에 따라 따뜻한 색과 차가운 색으로 분류할 수 있습니다.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/lecture-what-is-color-theory-in-design-5.png" alt="Warm and cool colors diagram" />

빨강, 주황, 노랑 같은 따뜻한 색은 편안함, 따뜻함, 아늑함을 불러일으킵니다.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/lecture-what-is-color-theory-in-design-6.png" alt="Warm colors examples" />

파랑, 초록, 보라 같은 차가운 색은 차분함, 평온함, 전문성을 느끼게 합니다.

색상은 색상 모델을 통해서도 나타낼 수 있습니다. 색상을 표준 방식으로 설명하고 재현하는 데 필수적입니다. 자주 사용되는 색상 모델로는 RGB 모델, HSV 모델, HSL 모델이 있습니다. 이들은 서로 다른 속성을 기준으로 색상을 나타냅니다. 앞으로 수업에서 더 자세히 배우게 될 것입니다.

좋습니다. 이제 이 내용을 좀 더 알게 되었으니, 디자이너들이 색상과 그 관계를 나타내기 위해 사용하는 기본 도구에 대해 이야기해 봅시다.

색상환은 색상이 서로 어떻게 관련되는지 보여주는 원형 다이어그램입니다. 디자이너에게 필수 도구인데, 색상 조합을 선택하는 데 도움을 주기 때문입니다. 색상 팔레트와 색상 스킴을 만드는 데 매우 유용합니다.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/lecture-what-is-color-theory-in-design-7.png" alt="Color wheel diagram" />

색상 스킴은 특정 디자인이나 프로젝트에 선택된 색상 집합입니다. 보통 색상 이론의 원칙을 기반으로 합니다. 색상환에서 색상 간 관계를 이해하면 다양한 유형의 색상 스킴을 개발할 수 있습니다. 몇 가지를 살펴봅시다.

유사 색상 스킴은 일관되고 편안한 경험을 만듭니다. 색상환에서 서로 인접한 유사한 색상으로 구성됩니다.

<!-- <img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/lecture-what-is-color-theory-in-design-8.png" alt="Analogous color scheme example" /> -->

보색 스킴은 강한 대비와 시각적 임팩트를 만듭니다. 색상환에서 서로 반대편에 위치한 색상들입니다.

색상 대비는 웹 접근성에 필수적입니다. 텍스트와 다른 중요한 요소가 배경과 명확히 구분되도록 보장합니다. 특히 시각 장애가 있는 사람들에게 중요합니다.

RGB 색상환에서 보색은 색상환의 반대편에 위치합니다. 예를 들어, 마젠타는 초록의 보색이고, 파랑은 노랑의 보색입니다.

<!-- <img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/lecture-what-is-color-theory-in-design-9.png" alt="RGB color wheel with complementary colors" /> -->

삼원색 스킴은 생동감 있는 색상으로 구성됩니다. 서로 대략 같은 거리에 있는 색상으로 만들어지며, 연결하면 색상환에서 정삼각형을 이룹니다.

<!-- <img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/lecture-what-is-color-theory-in-design-10.png" alt="Triadic color scheme example" /> -->

마지막으로 단색 스킴이 있습니다. 이 스킴은 같은 기본색에서 밝기, 어둠, 채도를 조절해 파생된 모든 색상으로 구성됩니다. 통일감과 조화를 느끼게 하면서도 대비를 만듭니다.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/lecture-what-is-color-theory-in-design-11.png" alt="Monochromatic color scheme example" />

마지막으로, 웹 개발에서 색상을 효과적으로 사용하는 몇 가지 팁입니다:

- 웹사이트의 브랜드를 정의하는 색상 스킴을 만드세요.
- 목표에 맞는 감정과 인식을 불러일으키는 색상을 사용하세요.
- 모든 사람이 접근할 수 있도록 충분한 대비가 있는 색상을 선택하세요.
- 버튼 같은 중요한 요소를 강조하는 데 색상을 사용하세요.
- 색상을 일관되게 사용하고 시각적 계층 구조를 만드는 데 활용하세요.

색상 이론은 디자인의 핵심 요소입니다. 색상 심리, 조화, 접근성을 이해하면 효과적인 디자인을 만들고 감정을 불러일으키며 사용자 경험을 향상할 수 있습니다.
