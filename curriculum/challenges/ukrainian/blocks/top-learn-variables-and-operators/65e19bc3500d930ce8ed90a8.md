# --description--

Числа — це будівельні блоки програмної логіки! Насправді важко уявити корисне програмне завдання, яке не включало б хоча б трохи базової математики… тому знати, як працюють числа, очевидно, дуже важливо. На щастя, це також досить просто.

| Оператор | Опис                         | Приклад            |
|----------|------------------------------|--------------------|
| +        | Додавання                    | 5 + 3 = 8          |
| -        | Віднімання                   | 5 - 3 = 2          |
| *        | Множення                    | 5 * 3 = 15         |
| **       | Піднесення до степеня        | 5 ** 3 = 125       |
| /        | Ділення                      | 6 / 3 = 2          |
| %        | Модуль                       | 5 % 3 = 2          |
