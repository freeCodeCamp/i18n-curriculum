# --explanation--

`Sit in on` significa asistir a una reunión o sesión como observador.

Mark dice `thanks for letting me sit in on the code review`. Esto muestra que le dieron permiso para estar allí y observar. Él no está dirigiendo la sesión ni haciendo ningún código — solo asiste para ver cómo funciona.
