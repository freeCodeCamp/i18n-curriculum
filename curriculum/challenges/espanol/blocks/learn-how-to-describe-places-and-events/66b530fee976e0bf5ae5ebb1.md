# --explanation--

`Extra` significa adicional o más de lo que es habitual o necesario. Por ejemplo:

- `We need extra chairs for the meeting.` - Necesitamos más sillas de lo habitual.

`Because` se usa para dar una razón de algo. Por ejemplo:

- `We canceled the meeting because of bad weather.` - El mal tiempo es la razón para cancelar la reunión.

`Recent` se refiere a algo que ocurrió hace poco tiempo. Por ejemplo:

- `The recent news was surprising.` - Las noticias salieron hace poco.

`Threat` significa un peligro o algo que puede causar daño. Por ejemplo:

- `The virus is a threat to our system.` - Esto significa que el virus podría causar daño al sistema.
