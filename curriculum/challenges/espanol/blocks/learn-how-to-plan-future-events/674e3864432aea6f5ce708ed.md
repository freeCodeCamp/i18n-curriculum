# --explanation--

`Entire` significa el grupo completo o total, enfatizando que cada miembro del equipo de soporte técnico está incluido. 

`On duty` se refiere a estar oficialmente asignado para trabajar o disponible para realizar tareas durante un tiempo específico. Por ejemplo:

- `The entire staff attended the meeting.` - Esto significa que todos los miembros del personal estuvieron presentes.

- `The security guard is on duty until midnight.` - Esto significa que el guardia está trabajando y disponible hasta la hora especificada.
