# --description--

Beim Arbeiten mit Strings ist das Zusammenfügen verschiedener Textstücke eine häufige Operation, mit der Sie oft zu tun haben werden.

In Python können Sie mehrere Strings mit dem Plusoperator (`+`) zusammenfügen. Dieser Vorgang wird **String-Konkatenation** genannt. So können Sie zwei Strings mit dem Plusoperator verketten:

```python
my_str_1 = 'Hello'
my_str_2 = "World"

str_plus_str = my_str_1 + ' ' + my_str_2
print(str_plus_str) # Hello World
```

Beachten Sie jedoch, dass dies nur mit Strings funktioniert. Wenn Sie versuchen, einen String mit einer Zahl zu verketten, erhalten Sie einen `TypeError`:

```python
name = 'John Doe'
age = 26

name_and_age = name + age
print(name_and_age) # TypeError: can only concatenate str (not "int") to str
```

Das passiert, weil Python andere Datentypen wie ganze Zahlen nicht automatisch in Strings umwandelt, wenn Sie sie verketten. Python verlangt, dass alle Elemente Strings sind, bevor es sie verketten kann. Um das zu beheben, können Sie die Zahl mit der eingebauten `str()`-Funktion in einen String umwandeln, die die String-Darstellung des übergebenen Objekts zurückgibt, ohne das Originalobjekt zu verändern:

```python
name = 'John Doe'
age = 26

name_and_age = name + str(age)
print(name_and_age) # John Doe26
```

Sie können auch den erweiterten Zuweisungsoperator für die Konkatenation verwenden. Dieser wird durch ein Plus- und Gleichheitszeichen (`+=`) dargestellt und führt sowohl Konkatenation als auch Zuweisung in einem Schritt aus. So sieht das in der Praxis aus:

```py
name = 'John Doe'
age = 26

name_and_age = name  # Start with the name
name_and_age += str(age)  # Append the age as string

print(name_and_age)  # John Doe26
```

Der Vorgang, Variablen und Ausdrücke in einen String einzufügen, heißt **String-Interpolation**. Python hat eine Kategorie von Strings namens **f-strings** (kurz für formatierte String-Literale), die es Ihnen ermöglicht, Interpolation mit einer kompakten und gut lesbaren Syntax zu handhaben.

F-strings beginnen mit `f` (entweder Klein- oder Großbuchstaben) vor den Anführungszeichen und erlauben es Ihnen, Variablen oder Ausdrücke innerhalb von Platzhaltern anzugeben, die durch geschweifte Klammern (`{}`) markiert sind. Hier ein Beispiel:

```python
name = 'John Doe'
age = 26
name_and_age = f'My name is {name} and I am {age} years old'
print(name_and_age) # My name is John Doe and I am 26 years old

num1 = 5
num2 = 10
print(f'The sum of {num1} and {num2} is {num1 + num2}') # The sum of 5 and 10 is 15
```

Beachten Sie, dass Sie nicht nicht-String-Typen mit der `str()`-Funktion umwandeln müssen. Im obigen Beispiel wird der Wert der Variablen `age`, `num1` und `num2` während des Interpolationsprozesses automatisch in einen String umgewandelt.
