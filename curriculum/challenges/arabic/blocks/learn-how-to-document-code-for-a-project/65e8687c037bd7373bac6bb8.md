# --description--

`Make sure` تعني التحقق أو التأكد من أن شيئًا ما تم بشكل صحيح أو أن شرطًا ما تحقق. يتطلب ذلك اتخاذ الخطوات اللازمة لضمان الدقة أو الإكمال. على سبيل المثال: `Please make sure you turn off the lights before you leave.`

`Follow instructions` تعني الالتزام بالتوجيهات أو الإرشادات المقدمة لإتمام مهمة ما. يتطلب ذلك تنفيذ كل خطوة بالترتيب والطريقة المحددين. مثال على ذلك: `follow the instructions on the box to assemble the table`.
