# --explanation--

让我们来分析一下：

马克说了什么？他说 BIOS `prevents the computer from starting.` 这意味着他在谈论 BIOS **能做什么** 当它被损坏时。 

如果 Sarah 同意，她会谈论 BIOS **是什么** 还是 BIOS **能做什么**？是后者，对吧？这就是为什么 `Yeah, it can` 是正确的选择。

为什么 `yeah, it is` 是错误的？因为 `it is` 讲的是 BIOS 的状态，而不是它能做什么。这里有一个例子：

**正确**：`Yeah, it can.` – 这与 Mark 关于 BIOS 能做什么的观点一致。

**错误**：`Yeah, it is.` – 这不合适，因为 Mark 不是在描述 BIOS 是什么，他是在谈论它的效果。
