# --description--

正の整数 $n$ に対して、$σ(n)$ を $n$ のすべての約数の和とします。例えば、$σ(6) = 1 + 2 + 3 + 6 = 12$ です。

ご存じの通り、完全数とは $σ(n) = 2n$ を満たす数のことです。

正の整数の完全商を $p(n) = \frac{σ(n)}{n}$ と定義します。

$p(n)$ が整数 $k$ と $\frac{1}{2}$ の和の形、すなわち $k + \frac{1}{2}$ となるような $n ≤ {10}^{18}$ までのすべての正の整数 $n$ の和を求めてください。
