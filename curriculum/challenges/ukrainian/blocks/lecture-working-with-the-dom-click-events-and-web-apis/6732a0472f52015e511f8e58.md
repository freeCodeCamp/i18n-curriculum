# --description--

Давайте дізнаємося про API і чому вони важливі для веб-розробки. API означає Application Programming Interface (інтерфейс програмування застосунків). API встановлюють набір правил і протоколів, які дозволяють програмним застосункам ефективно спілкуватися між собою та обмінюватися даними.

Їх можна уявити як конструкції, що дозволяють розробникам створювати більш складний функціонал на основі простіших будівельних блоків, які вже реалізовані.

Існують різні типи API. Веб-API спеціально розроблені для веб-застосунків. Існують різні типи веб-API для розробки на клієнтському JavaScript. Вони не є частиною самого JavaScript.

Ці типи API часто поділяють на дві основні категорії: браузерні API та сторонні API.

Браузерні API відкривають доступ до даних браузера. Як веб-розробник, ви можете отримувати доступ і маніпулювати цими даними за допомогою JavaScript.

Вони також надають доступ до різних функціональностей, таких як маніпулювання структурою сайту, обробка подій, робота зі сховищем і комунікація з мережею.

Деякі приклади загальновживаних браузерних API включають:

DOM API, який ви можете використовувати для маніпулювання HTML-елементами, їхніми стилями та атрибутами. Ви дізнаєтеся набагато більше про DOM у наступних уроках. Це ключова концепція у веб-розробці.

Storage API, щоб зберігати дані локально на пристрої користувача.

І багато іншого.

Існують різні браузерні API, які ви можете використовувати для обробки взаємодії користувача та отримання додаткової інформації з браузера, наприклад, поточного місцезнаходження користувача.

Ви також можете працювати з іншими типами елементів, такими як відео та аудіо.

Браузерні API можуть бути корисними інструментами для створення потужних веб-застосунків. Вони вже вбудовані у браузер.

Але ви також можете використовувати сторонні API, які за замовчуванням не вбудовані у браузер. Щоб їх використовувати, потрібно отримати їхній код якимось чином. Зазвичай вони супроводжуються детальною документацією, що пояснює, як користуватися їхніми сервісами.

Прикладом є Google Maps API, який ви можете використовувати для відображення інтерактивних карт на своєму сайті.

Існують різні типи сторонніх веб-API, включно з API погоди, API соціальних мереж, платіжними API, API даних, API перекладу та іншими. Вам просто потрібно знайти API, який відповідає вашим потребам.

Веб-API — це важливі інструменти для створення сучасних веб-застосунків, які можуть взаємодіяти з різними сервісами та джерелами даних. Вивчаючи, як використовувати їхню потужність, ви розширите свій набір інструментів для створення динамічних і привабливих сайтів.
