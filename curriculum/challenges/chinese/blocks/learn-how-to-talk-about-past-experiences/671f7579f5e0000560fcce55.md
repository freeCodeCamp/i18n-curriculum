# --explanation--

结构 `what` + 不定冠词 用于强调对某事的强烈感情或反应。它通常表达惊讶、钦佩或情感。例如： 

`What a beautiful day it is!`——这里，结构突出了说话者对这一天的赞美。 

同样，在 Maria 的句子中，`What a surprise` 强调了她在 PyCon 看到 Brian 时的惊讶。  

`PyCon` 是一个著名的 Python 编程爱好者、开发者和专业人士的会议。它为学习、网络交流和分享与 Python 相关的知识提供了一个平台。例如： 

`I attended PyCon last year and learned about the latest Python frameworks。` - 这意味着此人在去年参加了 PyCon 并学到了很多。
