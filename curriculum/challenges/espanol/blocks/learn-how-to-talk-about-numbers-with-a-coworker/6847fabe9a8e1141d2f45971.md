# --explanation--

`Eighty-three percent` es la forma correcta de leer 83%.

Siempre usas `percent` (no `percentages`) después de un número exacto. ¿Cuál es la diferencia?

Usa `percent` después de un número:

- El 83% es `eighty-three percent`.

- El 20% es `twenty percent`.

Usa `percentage` o `percentages` en general, no después de un número:

- `Eighty-three percentages` es incorrecto.

- `We saw an increase in percentages across departments.` es correcto (usado sin un número específico).

Esto es importante para dar actualizaciones claras, especialmente al hablar de estadísticas o informes.
