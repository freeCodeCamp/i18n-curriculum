# --description--

设 $N$ 为一个正整数，并将 $N$ 分成 $k$ 个相等的部分，$r = \frac{N}{k}$，使得 $N = r + r + \cdots + r$。

设 $P$ 为这些部分的乘积，$P = r × r × \cdots × r = r^k$。

例如，如果 11 被分成五个相等的部分，11 = 2.2 + 2.2 + 2.2 + 2.2 + 2.2，那么 $P = {2.2}^5 = 51.53632$。

令 $M(N) = P_{max}$，其中 $N$ 为给定值。

事实证明，对于 $N = 11$，最大值是通过将十一分成四个相等的部分得到的，这导致 $P_{max} = {(\frac{11}{4})}^4$；也就是说，$M(11) = \frac{14641}{256} = 57.19140625$，这是一个有限小数。

但是，对于 $N = 8$，最大值通过将其分成三个相等部分来实现，因此 $M(8) = \frac{512}{27}$，这是一个无限循环小数。

令 $D(N) = N$，如果 $M(N)$ 是无限小数；如果 $M(N)$ 是有限小数，则 $D(N) = -N$。

例如，$\sum D(N)$ 对于 $5 ≤ N ≤ 100$ 是 2438。

查找 $\sum D(N)$，其中 $5 ≤ N ≤ 10000$。
