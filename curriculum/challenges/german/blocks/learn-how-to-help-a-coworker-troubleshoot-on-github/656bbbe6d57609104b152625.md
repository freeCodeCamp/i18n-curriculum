# --description--

Bobs Aussage verwendet die past continuous tense `I was checking`, um eine Handlung zu beschreiben, die in der Vergangenheit andauerte.

Sie können Sätze im past continuous bilden, indem Sie das Verb to be in der Vergangenheit `was` oder `were` + `verb + ing` kombinieren.

Er verwendet außerdem das Modalverb `should`, um einen Handlungsverlauf vorzuschlagen.
