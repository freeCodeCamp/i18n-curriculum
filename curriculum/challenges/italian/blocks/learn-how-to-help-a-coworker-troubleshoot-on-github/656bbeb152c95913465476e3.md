# --description--

`Compile` in programmazione significa trasformare il codice scritto dalle persone in una forma che i computer possono comprendere e usare.

È come tradurre una ricetta scritta in una lingua in un'altra lingua affinché qualcun altro possa cucinare il piatto.

Per esempio, `When we compile Java code, it changes into a form that the computer can run.`
