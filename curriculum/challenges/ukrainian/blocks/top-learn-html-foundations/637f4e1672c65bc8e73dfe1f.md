# --description--

Ви можете уявити елементи як контейнери для вмісту. Відкриваючий і кінцевий теги повідомляють браузеру, який вміст містить елемент. Потім браузер може використати цю інформацію, щоб визначити, як інтерпретувати та форматувати вміст.

Існують деякі HTML-елементи, які не мають кінцевого тегу. Ці елементи часто виглядають так: `<br />` або `<img/>`, але деякі також можна використовувати без закриваючого косого слеша, наприклад `<br>` або `<img>`. Їх називають порожніми елементами або void-елементами, бо вони не охоплюють жодного вмісту. Ви зустрінете кілька таких у наступних уроках, але здебільшого елементи матимуть і відкриваючий, і кінцевий теги.

HTML має великий перелік заздалегідь визначених тегів, які ви можете використати для створення різноманітних елементів. Важливо використовувати правильні теги для вмісту. Використання правильних тегів може суттєво вплинути на два аспекти ваших сайтів: як вони ранжуються в пошукових системах і наскільки вони доступні для користувачів, які покладаються на допоміжні технології, такі як читачі екрана, щоб користуватися інтернетом.

Використання правильних елементів для вмісту називається семантичним HTML. Ви детальніше вивчите це пізніше в навчальній програмі.
