## Conceptos básicos de CSS Grid

- **Definición**: CSS Grid es un sistema de diseño bidimensional utilizado para crear diseños complejos en páginas web. Las cuadrículas tendrán filas y columnas con espacios entre ellas. Para definir un diseño de cuadrícula, debes establecer la propiedad `display` en `grid`.

```css
.container {
  display: grid;
}
```

- **La unidad `fr` (Fraccional)**: Esta unidad representa una fracción del espacio dentro del contenedor de la cuadrícula. Puedes usar la unidad `fr` para crear cuadrículas flexibles.
- **Creación de espacios entre las pistas**: Hay tres formas de crear espacios entre pistas en CSS grid. Puedes usar la propiedad `column-gap` para crear espacios entre columnas. Puedes usar la propiedad `row-gap` para crear espacios entre filas. O puedes usar la propiedad abreviada `gap` para crear espacios entre filas y columnas.
- **`grid-template-columns`**: Se utiliza para establecer los nombres de las líneas y el tamaño para las columnas de la pista de la cuadrícula.

```css
.container {
  display: grid;
  width: 100%;
  grid-template-columns: 30px 1fr;
}
```

- **`grid-template-rows`**: Se utiliza para establecer los nombres de las líneas y el tamaño para las filas de la pista de la cuadrícula.
- **`grid-auto-flow`**: Determina cómo los elementos colocados automáticamente se ajustan en la cuadrícula.

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-flow: column;
}
```

- **`grid-auto-columns`**: Se utiliza para establecer el tamaño de las columnas creadas implícitamente.

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-columns: auto;
}
```

- **`place-items`**: Se utiliza para alinear elementos tanto en direcciones de bloque como en línea.
- **`align-items`**: Se utiliza para establecer la alineación de los elementos en un contenedor de cuadrícula.
- **Función `repeat()`**: Esta función se utiliza para repetir secciones en la lista de pistas. En lugar de escribir `grid-template-columns: 1fr 1fr 1fr;` puedes usar la función `repeat()`.

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}
```

- **Cuadrícula explícita**: Puedes especificar el número de líneas o pistas usando las propiedades `grid-template-columns` o `grid-template-rows`.
- **Cuadrícula implícita**: Cuando los elementos se colocan fuera de la cuadrícula, se crean automáticamente filas y columnas para esos elementos externos.
- **Función `minmax()`**: Esta función se utiliza para establecer los tamaños mínimo y máximo para una pista.

```css
.container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-auto-rows: minmax(150px, auto);
}
```

- **Colocación basada en líneas**: Todas las cuadrículas tienen líneas. Para especificar donde comienza el elemento en una línea, puedes usar las propiedades `grid-column-start` y `grid-row-start`. Para especificar donde termina el elemento en la línea, puedes usar las propiedades `grid-column-end` y `grid-row-end`. También puedes elegir usar las propiedades abreviadas `grid-column` o `grid-row`.

Aquí tienes un ejemplo de cómo usar la propiedad `grid-column` para hacer que un elemento se extienda a lo largo de todas las columnas.

```css
.element {
  grid-column: 1 / -1;
}
```

- **`grid-template-areas`**: La propiedad se usa para proporcionar un nombre a los elementos que vas a posicionar en la cuadrícula.

```html
<div class="container">
  <div class="header">Header</div>
  <div class="sidebar">Sidebar</div>
  <div class="main">Main Content</div>
  <div class="footer">Footer</div>
</div>

```

```css
.container {
  display: grid;
  grid-template-columns: 200px 1fr; 
  grid-template-rows: auto 1fr auto; 
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer"; 
  gap: 20px; 
}

.header {
  grid-area: header; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}

.sidebar {
  grid-area: sidebar;
  background-color: #f4f4f4;
  padding: 10px;
}

.main {
  grid-area: main; 
  background-color: #e0e0e0;
  padding: 10px;
}

.footer {
  grid-area: footer; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}
```

## Depuración de CSS

- **DevTools (Herramientas de desarrollo)**: DevTools te permite inspeccionar y modificar tu CSS en tiempo real. El panel de estilos muestra todas las reglas CSS aplicadas al elemento seleccionado, incluidas las de estilos heredados. Puedes alternar propiedades individuales encendidas y apagadas, editar valores e incluso agregar nuevas reglas directamente en el navegador. Este feedback inmediato es increíblemente útil para experimentar con diferentes estilos sin modificar tu código fuente.
- **Validadores de CSS**: Se utilizan para verificar tu CSS contra las especificaciones oficiales y reportar cualquier error o advertencia. Un validador popular que puedes usar es el W3C CSS Validator.
- **Depuración de Diseños Responsivos**: Las DevTools tienen una opción que te permite simular cómo se ve tu sitio en varios tamaños de pantalla y dispositivos. Esto puede ayudarte a identificar problemas de punto de interrupción o estilos que no se escalan bien a través de diferentes tamaños de vista.
