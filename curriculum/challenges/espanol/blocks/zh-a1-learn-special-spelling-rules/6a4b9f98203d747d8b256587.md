# --explanation--

La primera sílaba es `yè`, y lleva el cuarto tono. La segunda sílaba es `wù`, y también lleva el cuarto tono.

`业务 (yè wù)` se refiere a las actividades profesionales, operaciones y tareas en las que una organización, departamento o individuo participa para lograr sus objetivos principales.
