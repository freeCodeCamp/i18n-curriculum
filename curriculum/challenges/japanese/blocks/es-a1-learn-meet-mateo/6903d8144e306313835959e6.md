# --description--

マテオ は、動詞 `tener`（「持つ」）を使って、スペイン語で自分の年齢を次の構造で伝えています：

`Tengo` + 数字の単語 + `años`。

スペイン語の 21 から 29 までの数字は `veinti-` で始まり、その後に対応する数字が続くことを覚えておいてください。例えば：

25 は `veinticinco` です。これは `e` を取り除き、数字 `i` の前に `cinco` を加えた結果です。

スペイン語の 1 から 9 までの数字は以下の通りです：

| 数字 | スペイン語の単語 |
| :---: | :---: |
| 1 | `uno` |
| 2 | `dos` |
| 3 | `tres` |
| 4 | `cuatro` |
| 5 | `cinco` |
| 6 | `seis` |
| 7 | `siete` |
| 8 | `ocho` |
| 9 | `nueve` |
