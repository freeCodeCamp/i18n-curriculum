# --description--

加算代入は、二項演算と代入を1つのステップで組み合わせたものです。変数に対して、別の値と演算を行い、その結果を同じ変数に代入します。

JavaScriptのような言語に慣れているなら、加算代入演算子（`+=`）や減算代入（`-=`）などを聞いたことがあるでしょう。Pythonにも同様のものがあり、これらは**加算代入**と呼ばれています。

加算代入の基本的な構文は次のようになります：

```python
variable <operator>= value
```

これは次のように書くより効率的な方法です：

```python
variable = variable <operator> value
```

例えば、既存の変数に`5`を加算代入で足す例は以下の通りです：

```python
my_var = 10
my_var += 5

print(my_var) # 15
```

同じことを加算代入を使わずに書くとこうなります：

```python
my_var = 10
my_var = my_var + 5

print(my_var) # 15
```

加算代入の利点は、変数名を繰り返さずに変数の値を簡潔かつ読みやすく更新できることです。これにより冗長さが減り、タイプミスなどによるエラーの可能性も減ります。

すべての演算子で加算代入が使えます。加算代入演算子（`+=`）を見てきましたが、他のものも見てみましょう。

- 減算代入演算子（`-=`）は、左の変数から右の値を引き、その差を左の変数に代入します：

```python
count = 14
count -= 3

print(count) # 11
```

- 乗算代入演算子（`*=`）は、左の変数に右の値を掛け、その積を左の変数に代入します：
    

```python
product = 65
product *= 7

print(product) # 455
```

- 除算代入演算子（`/=`）は、左の変数を右の値で割り、その結果を左の変数に代入します：
    

```python
price = 100
price /= 4

print(price) # 25.0
```

- 切り捨て除算演算子（`//=`）は、左の変数を右の値で切り捨て除算し、その結果を左の変数に代入します：
    

```python
total_pages = 23
total_pages //= 5

print(total_pages) # 4
```

- モジュラス代入演算子（`%=`）は、左の変数を右の値で割った余りを計算し、それを左の変数に代入します：
    

```python
bits = 35
bits %= 2

print(bits) # 1
```

- べき乗代入演算子（`**=`）は、左の変数を右の値のべき乗にし、その結果を左の変数に代入します：
    

```python
power = 2
power **= 3

print(power) # 8
```

文字列に対しても一部の加算代入演算子が使えます。例えば、加算代入演算子は文字列の連結を簡単にします：

```python
greet = 'Hello'
greet += ' World'

print(greet) # Hello World
```

また、乗算代入演算子は文字列の繰り返しに使えます：

```python
greet = 'Hello'
greet *= 3

print(greet) # HelloHelloHello
```

他の加算代入演算子を文字列に使うと`TypeError`がスローされます：

```python
greet = 'Hello'
greet -= ' World'

print(greet) # TypeError: unsupported operand type(s) for -=: 'str' and 'str'


greet = 'Hello'
greet /= 'World'

print(greet) # TypeError: unsupported operand type(s) for /=: 'str' and 'str' 
```

Pythonでインクリメント演算子（`++`）やデクリメント演算子（`--`）が使えるか気になるかもしれませんが、使えません。Pythonは言語を明確かつ明示的に保つために、Cスタイルのインクリメント・デクリメントのショートカットを意図的に避けています。

`x++`の代わりに、単に`x += 1`と書けば、`x`の値を`1`だけ増やしていることが明確になります。

Pythonで`++x`と書くと、単に単項プラスが2回適用されるだけで、値は増えません：

```python
my_var = 5

print(+my_var)   # 5
print(++my_var)  # 5
print(+++my_var) # 5

my_var += 1

print(my_var) # 6
```
