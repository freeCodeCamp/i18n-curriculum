# --explanation--

`Updated`는 최신 정보를 반영하도록 수정되거나 변경되었음을 의미합니다. 예를 들어:

`The document has been updated with new data.` — 이 문서는 이제 가장 최신 정보를 포함하고 있음을 뜻합니다.

`Timeline`는 작업이나 이벤트가 언제 일어날지 개요를 보여주는 계획을 의미합니다. 예를 들어:

`The project timeline shows when each phase will be completed.` — 이것은 프로젝트 완료 일정을 뜻합니다.
