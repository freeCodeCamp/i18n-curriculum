# --explanation--

`Maybe` означає `possibly` або `there's a chance`. Його використовують, щоб висловити невпевненість або відсутність повної впевненості.

`Hard to know` означає, що важко визначити або зробити висновок через нестачу інформації чи ясності.

`Maybe, but it's hard to know` поєднує ці ідеї, щоб показати, що вирішити фінансові проблеми можливо, але результат невизначений.
