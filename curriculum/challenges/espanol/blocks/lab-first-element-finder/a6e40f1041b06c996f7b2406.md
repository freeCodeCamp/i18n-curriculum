# --description--

En este laboratorio crearás una función que mire a través de una matriz y devuelva el primer elemento que pase una función de prueba (también conocida como "prueba de la verdad").

La función iteraría a través de la matriz y evaluaría cada elemento usando la función de prueba proporcionada. Al final, devolvería el primer elemento donde la función de prueba devuelva `true`.

Ejemplo:

```js
findElement([1, 3, 5, 8], num => num % 2 === 0) // returns 8
findElement([1, 3, 5], num => num % 2 === 0)    // returns undefined
```

**Objetivo:** Cumplir con los siguientes instrucciones y lograr que pasen todas las pruebas para completar el laboratorio.

**Instrucciones:**

1. Debes tener una función llamada `findElement` que acepte un array y una función como argumentos.
2. La función debe devolver el primer elemento de la matriz que pase una prueba de la verdad. Esto significa que, llamando a la función `func` pasada, dado un elemento `x`, la prueba de la verdad se pasa si `func(x)` es `true`.
3. Si ningún elemento pasa la prueba. la función debería devolver `undefined`.
