# --description--

Si estás usando tu dispositivo móvil para navegar por el currículo de chino, puedes escribir Pinyin y caracteres chinos directamente usando las entradas personalizadas de freeCodeCamp. Sin embargo, si quieres usar el teclado chino en tu dispositivo, aquí te explicamos cómo configurarlo.

## Agregar teclados de idioma

### En Android

**NOTA:** La ruta de Android no es universal y puede variar según el dispositivo.

1. Abre Settings.

2. Ve a Sistema (o Administración General, dependiendo del dispositivo).

3. Configuración del teclado táctil (o Idiomas y entrada).

4. Toca Languages and types (o Add keyboard).
 
5. Activa chino (simplificado o CN) o descárgalo si es necesario.

6. Guarda o toca Done, si se te solicita.

### En iPhone

1. Abre Settings.

2. Ve a General > Keyboard.

3. Teclados táctiles.

4. Toca Agregar nuevo teclado.

5. Selecciona chino (simplificado).

6. Usa el ícono del globo en el teclado para cambiar de idioma.

## Probando teclados de terceros

También puedes instalar aplicaciones de teclado de terceros en tu teléfono o tableta para facilitar la escritura en chino. Estos teclados suelen incluir funciones útiles como predicciones inteligentes de caracteres y soporte para múltiples métodos de entrada, como Pinyin o escritura a mano.
