# --description--

O espanhol frequentemente usa as palavras `cuál` e `cómo` para formar perguntas comuns. 

`¿Cuál ... ?` é usado para pedir **informações específicas**, especialmente antes do verbo `ser`. Por exemplo:

- `¿Cuál es tu nombre?` – Qual é o seu nome?

- `¿Cuál es tu número de teléfono?` – Qual é o seu número de telefone?

`¿Cómo ... ?` é usado para perguntar **como** algo é feito, soletrado ou pronunciado. Por exemplo:

- `¿Cómo te llamas?` – Qual é o seu nome?

- `¿Cómo se escribe?` – Como se soletra?
