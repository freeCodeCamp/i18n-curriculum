# --description--

浏览器通常会对特定元素应用默认的外边距和内边距值。 要确保钢琴布局显示正确，需要重置盒子模型。

将 `html` 规则选择器添加到 CSS 文件，并将 `box-sizing` 属性设置为 `border-box`。

# --hints--

你应该有一个 `html` 选择器。

```js
assert.exists(new __helpers.CSSHelp(document).getStyle('html'));
```

你的 `html` 选择器应该将 `box-sizing` 属性设置为 `border-box`。

```js
assert.equal(new __helpers.CSSHelp(document).getStyle('html')?.boxSizing, 'border-box');
```
