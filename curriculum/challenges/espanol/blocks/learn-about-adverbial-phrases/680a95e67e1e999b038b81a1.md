# --explanation--

Un `final report` es un documento completo y terminado que resume los resultados, el progreso o los hallazgos de un proyecto, evento o actividad. Generalmente se crea al final de una tarea para revisar lo que se hizo y lo que se aprendió. Por ejemplo:

`The final report showed that the training program improved employee performance.` – Esto significa que el informe dio un resumen de los resultados después de que terminó el programa de entrenamiento.
