# --interactive--

CSS 특이성은 여러 규칙이 적용될 수 있을 때 어떤 스타일이 요소에 적용될지 결정하는 기본 개념입니다.

특이성을 이해하면 개발자가 서로 다른 CSS 규칙 간의 충돌을 해결하고 원하는 스타일이 일관되게 적용되도록 할 수 있습니다.

CSS 특이성은 사용된 선택자 유형에 따라 계산됩니다.

가장 높은 특이성은 `style` 속성을 통해 요소에 직접 적용되는 인라인 스타일에 부여됩니다.

이 예제에서 첫 번째 문단은 빨간색이고 다른 `p` 요소들은 파란색이 됩니다. 이는 인라인 스타일이 `p` 파일에 표시된 `styles.css` 타입 선택자보다 특이성이 더 높기 때문입니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p style="color: red;">Red paragraph</p>
<p>Other paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p {
  color: blue;
}
```

:::

다음으로, ID 선택자는 높은 특이성을 가집니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Red paragraph</p>
<p>Other paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

p {
  color: blue;
}
```

:::

그다음으로 클래스 선택자, 속성 선택자, 가상 클래스가 등장합니다.

예를 들어 `.container`와 `.button` 같은 클래스 선택자, `[type="text"]` 같은 속성 선택자, `:hover` 같은 가상 클래스가 있습니다. 이들은 중간 정도의 특이성을 가집니다.

**참고**: 가상 클래스에 대해서는 앞으로 수업에서 더 배우게 됩니다.

이 예제에서 첫 번째 문단은 `id` 선택자가 클래스와 타입 선택자보다 특이성이 높기 때문에 빨간색이 됩니다. `.example-para` 요소들은 녹색이고 나머지 문단 요소들은 파란색이 됩니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p class="example-para">Other paragraph</p>
<p class="example-para">Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

.example-para {
  color: green;
}

p {
  color: blue;
}
```

:::

타입 선택자, 예를 들어 `div`와 `h1`, 그리고 가상 요소인 `::before`와 `::after`는 이전 그룹에 비해 특이성이 낮습니다.

**참고**: 가상 요소에 대해서는 앞으로 수업에서 더 배우게 됩니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p>Other paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

p {
  color: blue;
}
```

:::

마지막으로, 별표 `*`로 나타내는 범용 선택자는 모든 선택자 중 가장 낮은 특이성을 가집니다.

다음은 `*` 선택자를 사용해 모든 요소의 색상을 빨간색으로 설정하는 예입니다. 하지만 ID와 타입 선택자가 이 스타일을 재정의하기 때문에 빨간색 요소는 보이지 않습니다. 이는 `*` 선택자의 낮은 특이성을 보여줍니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p>Other paragraph</p>
<p>Yet another paragraph</p>
```

```css
* {
  color: red;
}

#para {
  color: green; 
}

p {
  color: blue;
}
```

:::

특이성 값은 네 부분으로 구성된 값 `(a, b, c, d)`으로 계산됩니다:

- `a`: 인라인 스타일 (`1` 또는 `0`).
- `b`: ID 선택자의 개수.
- `c`: 클래스 선택자, 속성 선택자, 가상 클래스의 개수.
- `d`: 타입 선택자, 가상 요소, 범용 선택자의 개수.

특이성 값의 각 부분은 서로 다른 가중치를 가집니다:

- **인라인 스타일** (`a`)은 가장 높은 가중치를 가지며, 특이성 값의 첫 번째 부분에 `1` 값을 더합니다.
    
- **ID 선택자** (`b`)는 다음으로 높은 가중치를 가지며, 각 ID는 특이성 값의 두 번째 부분에 `1` 값을 더합니다.
    
- **클래스 선택자, 속성 선택자, 가상 클래스** (`c`)는 중간 가중치를 가지며, 각 선택자는 특이성 값의 세 번째 부분에 `1` 값을 더합니다.
    
- **타입 선택자와 가상 요소** (`d`)는 가장 낮은 가중치를 가지며, 각 선택자는 특이성 값의 네 번째 부분에 `1` 값을 더합니다.
    
- **범용 선택자** (`*`): 범용 선택자는 특이성 계산에 `0` 값을 더하며 특이성에 영향을 주지 않습니다. 선택자에 포함되어도 특이성 값은 변하지 않습니다.

인라인 CSS는 요소에 직접 적용되기 때문에 가장 높은 특이성을 가집니다. 내부 또는 외부 CSS를 재정의합니다. 인라인 스타일의 특이성 값은 `(1, 0, 0, 0)`입니다.

다음은 인라인 스타일의 또 다른 예입니다:

:::interactive_editor

```html
<p style="color: red;">This text is red.</p>
```

:::

내부 CSS는 HTML 문서의 `style` 섹션 내 `head` 요소 안에 정의됩니다. 인라인 스타일보다 특이성이 낮습니다. 내부와 외부 CSS는 같은 특이성 수준을 가지며, 특이성은 CSS가 정의된 위치가 아니라 사용된 선택자에 의해 결정됩니다. 두 규칙의 특이성이 같으면 문서에서 나중에 나오는 규칙이 우선합니다. 따라서 외부 스타일시트가 내부 `style` 블록 뒤에 연결되면 외부 규칙이 우선합니다. 앞에 연결되면 내부 규칙이 우선합니다.

내부 스타일의 특이성 값은 사용된 선택자에 따라 결정됩니다. 예를 들어 내부 CSS의 ID 선택자는 특이성 값이 `(0, 1, 0, 0)`입니다.

:::interactive_editor

```html
<head>
  <style>
    #text {
      color: blue;
    }
  </style>
</head>
<body>
  <div id="text">This text is blue.</div>
</body>
```

:::

이 예제에서는 인라인 스타일이나 더 구체적인 선택자가 적용되지 않는 한 텍스트가 파란색입니다.

외부 CSS는 `link` 섹션의 `head` 요소를 통해 연결되며 별도의 `.css` 파일에 작성됩니다. 내부 CSS와 마찬가지로 외부 스타일의 특이성은 사용된 선택자에 의해 결정됩니다. 외부 CSS는 큰 프로젝트에서 유지 관리에 가장 적합합니다.

외부 스타일의 특이성 값도 사용된 선택자에 의해 결정됩니다. 예를 들어 외부 CSS의 클래스 선택자는 특이성 값이 `(0, 0, 1, 0)`입니다.

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <div class="text">This text's color is defined in an external CSS file.</div>
</body>
```

```css
.text {
  color: purple;
}
```

:::

이 예제에서는 `styles.css` 파일에서 텍스트 색상이 정의되며, 더 구체적인 선택자나 인라인 스타일에 의해 재정의되지 않는 한 적용됩니다.
