# --interactive--

JavaScriptでは、文字列とはテキストデータを表すために使われる文字のシーケンスです。文字列は、数値、ブール値、`null`、および`undefined`と並ぶ言語のプリミティブデータ型の一つです。

JavaScriptで文字列を作成するには、シングルクォート（`'`）または二重引用符（`"`）を使うことができます。

以下は、文字列の値を保持する2つの変数を作成する例です：

:::interactive_editor

```js
let singleQuotes = 'This is a string';
console.log(singleQuotes);
let doubleQuotes = "This is also a string";
console.log(doubleQuotes);
```

:::

文字列を作成する際にシングルクォートか二重引用符のどちらかを使えますが、一貫性を保つことが重要です。文字列がシングルクォートで始まる場合は、必ずシングルクォートで終わらなければなりません。

二重引用符の場合も同様です。次の例は、二重引用符で始まりシングルクォートで終わっているため、エラーが発生します：

```js
const improperStr = "Do not do this';
```

もう一つ注意すべき点は、文字列は不変であるということです。プログラミングにおける不変性とは、一度作成されたものは変更できないことを意味します。つまり、文字列を作成した後、その文字を直接変更することはできません。変更したい場合は、新しい文字列を作成する必要があります。

以下は、`developer`変数に新しい文字列を代入する例です：

:::interactive_editor

```js
let developer = "Jessica";
console.log(developer);
developer = "Quincy";
console.log(developer);
```

:::

文字列は不変なので、最初の文字列を直接更新することはできません。だからこそ、`developer`変数に新しい文字列を代入しています。

文字列はプログラミングにおいて重要な役割を果たしており、今後のレッスンで文字列を操作する高度な手法を学び、動的でインタラクティブなアプリケーションを作成するための可能性を最大限に活用できるようになります。
