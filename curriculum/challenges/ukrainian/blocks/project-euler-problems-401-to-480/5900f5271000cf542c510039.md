# --description--

Ціле число називається без одинадцяток, якщо його десятковий запис не містить жодного підрядка, який представляє ступінь числа 11, окрім 1.

Наприклад, 2404 і 13431 є без одинадцяток, а 911 і 4121331 — ні.

Нехай $E(n)$ — це $n$-те додатне ціле число без одинадцяток. Наприклад, $E(3) = 3$, $E(200) = 213$ і $E(500\\,000) = 531\\,563$.

Знайдіть $E({10}^{18})$.
