# --explanation--

`Typically` se usa para describir algo que ocurre de manera habitual o esperada. Es similar a `usually` y se usa a menudo para hablar de lo que normalmente sucede. Por ejemplo:

- `Typically, we have meetings on Mondays.` - Esto significa que las reuniones usualmente ocurren los lunes.

- `She typically arrives at work by 9 AM.` - Esto significa que ella normalmente llega al trabajo antes de las 9 AM.
