# --description--

職場で、動詞 `to document` は将来のために情報を記録することを意味します。これは詳細やステップ、重要なポイントを書き留めることを含みます。

`Future reference` は、この記録された情報を後で使うことを指し、問題を解決したり手順を思い出したりするために使われることが多いです。例えば：

`They are documenting their studies for future reference.`
