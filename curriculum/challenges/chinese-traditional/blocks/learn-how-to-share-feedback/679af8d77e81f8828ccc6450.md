# --explanation--

`Perhaps` 用來顯示不確定性或建議某事而不確定。 例如：

`Perhaps we should wait before making a decision.`－這表示說話者並不完全確定，但認為等待可能是個好主意。

`To make sure` 是指確保某件事發生或被正確完成。 例如：

`Please make sure the documents are sent before the deadline.`－這表示說話者正在請求某人確保文件在截止時間前寄出。
