# --explanation--

在此背景關係中，`Would` 表示過去的習慣性動作或重複行為。它常用來描述某人經常執行的動作，類似於 `used to`。 

然而，`would` 更專注於動作本身，而 `used to` 則可以描述狀態或習慣。例如：

- `She would always read a book before going to bed.`－睡覺前閱讀書籍是她的常見習慣。

`She used to live in New York before moving here.`－這裡使用 `used to` 是因為住在一個地方表示一種狀態，而不是一個動作或你習慣性做的事情。

在這句話中，Maria 使用 `would` 來描述 Brian 一再熬夜微調程式碼的習慣，強調他的投入。
