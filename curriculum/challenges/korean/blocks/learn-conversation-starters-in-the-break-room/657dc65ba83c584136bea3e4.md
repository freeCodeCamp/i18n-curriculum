# --description--

`assign tasks`는 특정 작업이나 임무를 누군가에게 주는 것을 의미합니다. 예를 들어:

- `The manager assigns tasks to the team every morning.` - 매니저가 각 팀원에게 작업을 할당합니다.

- `They assign tasks based on skills.` - 각 사람이 잘하는 것에 따라 작업을 할당합니다.
