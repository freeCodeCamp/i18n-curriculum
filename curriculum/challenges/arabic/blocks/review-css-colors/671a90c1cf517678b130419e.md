## نظرية الألوان

- **تعريف نظرية الألوان**: هي دراسة كيفية تفاعل الألوان مع بعضها وكيف تؤثر على إدراكنا. تشمل علاقات الألوان، تناغم الألوان، والتأثير النفسي للألوان.
- **الألوان الأساسية**: هذه الألوان هي الأصفر، الأزرق، والأحمر، وهي الألوان الأساسية التي تُشتق منها جميع الألوان الأخرى.
- **الألوان الثانوية**: تنتج هذه الألوان من خلط كميات متساوية من لونين أساسيين. الأخضر، البرتقالي، والبنفسجي أمثلة على الألوان الثانوية.
- **الألوان الثلاثية**: تنتج هذه الألوان من دمج لون أساسي مع لون ثانوي مجاور له. الأصفر-الأخضر، الأزرق-الأخضر، والأزرق-البنفسجي أمثلة على الألوان الثلاثية.
- **الألوان الدافئة**: تشمل هذه الألوان الأحمر، البرتقالي، والأصفر، وتثير مشاعر الراحة والدفء والحميمية.
- **الألوان الباردة**: تشمل هذه الألوان الأزرق، الأخضر، والبنفسجي، وتثير مشاعر الهدوء والسكينة والاحترافية.
- **عجلة الألوان**: هي مخطط دائري يوضح كيف ترتبط الألوان ببعضها. أداة أساسية للمصممين لأنها تساعدهم على اختيار تركيبات الألوان.
- **مخططات الألوان المتجاورة**: تخلق هذه المخططات تجارب متماسكة ومهدئة. تحتوي على ألوان متجاورة تقع بجانب بعضها في عجلة الألوان.
- **مخططات الألوان التكاملية**: تخلق هذه المخططات تباينًا عاليًا وتأثيرًا بصريًا قويًا. تقع ألوانها في نهايات متقابلة من عجلة الألوان.
- **مخطط الألوان الثلاثي**: يحتوي هذا المخطط على ألوان زاهية. تتكون من ألوان متباعدة تقريبًا بمسافات متساوية. إذا رُبطت، تشكل مثلثًا متساوي الأضلاع على عجلة الألوان.
- **مخطط الألوان أحادي اللون**: في هذا المخطط، تُشتق كل الألوان من نفس اللون الأساسي عن طريق تعديل فاتح اللون، والداكن، والتشبع. يثير شعورًا بالوحدة والتناغم مع خلق تباين.

## طرق مختلفة للعمل مع الألوان في CSS

- **الألوان المسماة**: هي أسماء ألوان معرفة مسبقًا ومعترف بها من المتصفحات. أمثلة تشمل `blue`، `darkred`، `lightgreen`.
- **دالة `rgb()`**: تمثل RGB الأحمر، الأخضر، والأزرق — الألوان الأساسية للضوء. تُدمج هذه الألوان الثلاثة بكثافات مختلفة لإنشاء مجموعة واسعة من الألوان. تتيح لك دالة `rgb()` تعريف الألوان باستخدام نموذج ألوان RGB.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>RGB Color</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

- **دالة `rgba()`**: تضيف هذه الدالة قيمة رابعة، ألفا، تتحكم في شفافية اللون. إذا لم تُحدد، تكون قيمة ألفا الافتراضية `1`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>RGBA Background</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

- **دالة `hsl()`**: تمثل HSL اللون، التشبع، وفاتح اللون — ثلاثة مكونات رئيسية تحدد اللون.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>HSL Color</p>
```

```css
p {
  color: hsl(120, 100%, 50%);
}
```

:::

- **دالة `hsla()`**: تضيف هذه الدالة قيمة رابعة، ألفا، تتحكم في عتامة اللون.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>HSLA Background</div>
```


```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

- **التمثيل الست عشري**: رمز الست عشري (اختصارًا للكود الست عشري) هو سلسلة مكونة من ستة أحرف تُستخدم لتمثيل الألوان في نموذج ألوان RGB. يشير "hex" إلى نظام الترقيم الأساسي 16، الذي يستخدم الأرقام من 0 إلى 9 والحروف من A إلى F.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1 class="hex-text">Hex Text</h1>
<p class="hex-bg">Hex Background</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}

p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

## خاصية `box-shadow`

- **التعريف**: تطبق خاصية `box-shadow` ظلًا واحدًا أو أكثر حول عنصر.
- **قيم الإزاحة**: يجب تحديد القيم الأفقية (`offset-x`) والعمودية (`offset-y`). تحرك القيمة الموجبة لـ `offset-x` الظل إلى اليمين، والقيم السالبة تحركه إلى اليسار. تحرك القيمة الموجبة لـ `offset-y` الظل إلى الأسفل، والقيم السالبة تحركه إلى الأعلى. إذا كانت القيمة `0`، لا تحتاج إلى تضمين وحدة قياس.
- **نصف قطر التمويه**: هذه القيمة الاختيارية تتحكم في مدى ضبابية الظل. إذا لم تُدرج، تكون القيمة الافتراضية `0`، مما يخلق حوافًا حادة. كلما زادت القيمة، أصبح الظل أكثر نعومة.
- **نصف قطر الانتشار**: هذه القيمة الاختيارية تتحكم في مدى توسع الظل أو تقلصه. إذا لم تُدرج، تكون القيمة الافتراضية `0`.
- **لون الظل**: يمكنك تحديد اللون باستخدام الألوان المسماة، القيم الست عشرية، دوال `rgb()`، `rgba()`، `hsl()`، أو `hsla()`.
- **الكلمة المفتاحية `inset`**: إضافة الكلمة المفتاحية `inset` تضع الظل داخل العنصر بدلًا من خارجه.
- **تطبيق عدة ظلال**: يمكنك تطبيق عدة ظلال بفصلها بفواصل. تُرتب الظلال من الأمام إلى الخلف.

### الصيغة النحوية

```css
box-shadow: offset-x offset-y blur-radius spread-radius color;
```

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="shadow-box">Shadow Color Example</div>
```

```css
.shadow-box {
  width: 200px;
  padding: 20px;
  background-color: lightblue;
  box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}
```

## التدرجات الخطية والشعاعية

- **التدرجات الخطية**: تخلق هذه التدرجات دمجًا تدريجيًا بين الألوان على طول خط مستقيم. يمكنك التحكم في اتجاه هذا الخط باستخدام كلمات مفتاحية مثل `to top`، `to right`، `to bottom right`، أو زوايا مثل `45deg`. يمكنك استخدام أي لون CSS صالح وعدد غير محدود من توقفات اللون.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient">Linear Gradient</div>
```

```css
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

:::

- **التدرجات الشعاعية**: تخلق هذه التدرجات تدرجات دائرية أو بيضاوية تشع من نقطة مركزية.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient">Radial Gradient</div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::
