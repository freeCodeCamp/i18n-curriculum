# --description--

Menschen kommunizieren auf unterschiedliche Weise, je nach Situation.

Sie können das `teléfono` verwenden, um direkt mit jemandem zu sprechen, oder eine `mensaje de texto`, eine kurze schriftliche Nachricht, die per Telefon gesendet wird.

Sie können auch die `correo` verwenden, die sich auf traditionelle Post bezieht, wie Briefe, die an eine Wohnadresse geschickt werden. Heute werden viele Nachrichten online versendet. In diesen Fällen nennt man das `correo electrónico`.

`Electrónico` beschreibt etwas, das Technologie oder elektronische Geräte verwendet.
