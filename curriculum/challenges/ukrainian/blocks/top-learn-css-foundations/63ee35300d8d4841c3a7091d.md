# --description--

Ще один спосіб використання селекторів — це ланцюжок їх у вигляді списку без будь-яких роздільників. Припустімо, у вас є такий HTML:

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection preview">This is where a preview for a post might go.</p>
</div>
```

У вас є два елементи з класом `subsection`, які мають унікальні стилі, але що, якщо ви хочете застосувати окреме правило лише до елемента, який також має `header` як другий клас? Ви можете об’єднати обидва селектори `class` у вашому CSS так:

```css
.subsection.header {
  color: red;
}
```

Що робить `.subsection.header`, так це вибирає будь-який елемент, який має одночасно класи `subsection` і `header`. Зверніть увагу, що між селекторами `.subsection` і `.header` `class` немає пробілу. Цей синтаксис працює для ланцюжка будь-якої комбінації селекторів, окрім ланцюжка більше ніж одного селектора типу.

Це також можна використати для ланцюжка класу і ідентифікатора, як показано нижче:

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection" id="preview">This is where a preview for a post might go.</p>
</div>
```

Ви можете взяти два наведені вище елементи і об’єднати їх так:

```css
.subsection.header {
  color: red;
}

.subsection#preview {
  color: blue;
}
```

Загалом, не можна ланцюжити більше одного селектора типу, оскільки елемент не може бути одночасно двома різними типами. Наприклад, ланцюжок двох селекторів типу, як `div` і `p`, дав би нам селектор `divp`, який не працюватиме, бо селектор намагатиметься знайти буквальний елемент `<divp>`, якого не існує.
