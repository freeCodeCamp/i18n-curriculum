# --instructions--

Для цього завдання потрібно створити три математичні ігри на Python, які виконують наступне:

- Гра з точковим графіком
  - Випадковим чином генеруйте точки на графіку, і гравець має ввести координати (x,y)
  - Для більшої складності зробіть графік більшим
- Гра для практики алгебри
  - Генеруйте задачі з одним і двома кроками з випадковими цілими числами, і гравець має ввести відповідь
  - Використовуйте додатні та від’ємні значення. Для більшої складності зробіть числа більшими
- Гра з рухом снаряда
  - Відобразьте «стіни» з випадковою висотою та розташуванням. Гравець має рухати повзунки, щоб налаштувати параболічний шлях і обійти стіну
  - Для більшої складності зробіть другий рівень, де гравці вводять a, b і c без повзунків

Після завершення відправте URL публічного записника Colab на своєму Google диску.
