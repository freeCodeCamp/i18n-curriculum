# --description--

このタスクでは、比喩的な用語の使い方を通じてプログラミングの概念を理解します。比喩は、日常の物や行動に例えることで複雑なアイデアを視覚化し説明する強力な方法です。

`Container` は変数を、さまざまなデータを保持しストアできるものとして比喩的に表現しています。これは `container` が異なるアイテムを保持できるのと似ています。

`fill and empty` は変数にデータを追加したり取り除いたりする動作を表しており、水筒に水を `fill` したり水を注ぎ出したりするのに似ています。

`Locked boxes` は定数を安全で変更できない `containers` として表現し、一度設定すると変更できないことを強調しています。

`keep the same contents` は定数の中のデータが変わらないことを示しており、鍵のかかった箱が常に同じものを保持しているようなイメージです。
