# --description--

أعطِ سلسلة نصية من Markdown تحتوي على كتلة أو أكثر من كتل الشفرة المضمَّنة، أرجع سلسلة HTML المكافئة.

تستخدم كتل الشفرة المضمَّنة في Markdown علامة backtick واحدة (``` ` ```) في بداية ونهاية نص كتلة الشفرة.

أرجع السلسلة المعطاة مع تحويل كل كتل الشفرة إلى وسوم HTML من نوع `code`.

على سبيل المثال، إذا أعطيت السلسلة ```"Use `let` to declare the variable."```، أعد `"Use <code>let</code> to declare the variable."`.

ملاحظة: قد لا تعرض وحدة التحكم وسوم HTML في السلاسل النصية عند تسجيل الرسائل. تحقق من وحدة تحكم المتصفح لرؤية السجلات مع الوسوم مضمنة.

function parseInlineCode(markdown) {

}
