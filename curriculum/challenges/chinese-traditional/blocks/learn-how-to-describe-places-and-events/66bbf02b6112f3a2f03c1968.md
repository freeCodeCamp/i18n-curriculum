# --explanation--

`It` 用來取代已經提過的名詞，有助於避免句子中的重複。例如：

- `I bought a new laptop. It works really well.`－這裡，`it` 取代 `laptop` 以避免重複該字。

在 `The Live Preview isn't showing in Visual Studio Code. Isn't it supposed to support that feature?` 中，`it` 指的是 `Visual Studio Code`，因為 Tom 正在談論該程式及其支援某個特性的能力。 

`It` 取代 `Visual Studio Code`，以避免再次提及該程式的名稱。 

透過識別預期支援該特性，你可以判斷 `it` 指的是程式本身。
