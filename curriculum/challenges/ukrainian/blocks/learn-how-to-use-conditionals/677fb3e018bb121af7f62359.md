# --explanation--

`First Conditional` використовується для опису реальних і можливих ситуацій у майбутньому. Він утворюється за допомогою `if` + `Present Simple` (для умови) та `will` + початкової форми дієслова (для результату). Наприклад:

`If you debug the code, the app will run smoothly.` - Виправлення коду призведе до того, що застосунок працюватиме правильно.

У правильному реченні умова — це `If we don't back up regularly`, а результат — `we could lose important information`. Це свідчить про реальну можливість.
