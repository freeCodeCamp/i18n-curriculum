# --description--

Lassen Sie uns Parameter und Argumente im Kontext der folgenden Beispiel-Funktion besprechen:

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

In JavaScript sind Parameter die Elemente, die zwischen den runden Klammern `()` in der Funktionsdeklaration aufgelistet sind. Funktionsargumente sind die tatsächlichen Werte, die Sie an die Funktion übergeben.

Im obigen Beispiel ist die Funktionsdefinition in der ersten Zeile geschrieben: `function favoriteAnimal(animal)`. Der Parameter `animal` befindet sich innerhalb der runden Klammern. Sie könnten `animal` genauso gut durch `pet`, `x` oder `blah` ersetzen. Aber in diesem Fall gibt die Benennung des Parameters als `animal` jemandem, der Ihren Code liest, etwas Kontext, sodass er nicht raten muss, was `animal` schließlich enthalten könnte.
