## أُطُر CSS

- **أُطُر CSS**: تُسرّع أُطُر CSS سير عملك، وتُنشئ نمطًا بصريًا موحدًا عبر الموقع، وتجعل تصميمك متناسقًا عبر متصفحات متعددة، وتحافظ على تنظيم شفرة CSS الخاصة بك.
- **أُطُر CSS الشهيرة**: من أشهر أُطُر CSS هي Tailwind CSS وBootstrap وMaterialize وFoundation.
- **العيوب المحتملة**:
  - قد تتعارض CSS المقدمة من الإطار مع CSS المخصصة الخاصة بك.
  - قد يبدو موقعك مشابهًا لمواقع أخرى تستخدم نفس الإطار.
  - قد تسبب الأُطُر الكبيرة مشاكل في الأداء.

## نوعان من أُطُر CSS

- **أُطُر CSS ذات الأولوية للأدوات**: تحتوي هذه الأُطُر على أصناف صغيرة لأغراض محددة، مثل تعيين الهامش، الحشو، أو لون الخلفية. يمكنك تعيين هذه الأصناف الصغيرة مباشرة لعناصر HTML حسب الحاجة. يُصنف Tailwind CSS كإطار CSS ذو أولوية للأدوات.

إليك مثالًا على استخدام Tailwind CSS لتنسيق زر.

```html
<button class="bg-blue-500 text-white font-bold py-2 px-4 rounded-full hover:bg-blue-700">
  Button
</button>
```

- **أُطُر CSS المعتمدة على المكونات**: تحتوي هذه الأُطُر على مكونات مُعدة مسبقًا بأنماط معرفة مسبقًا يمكنك إضافتها إلى موقعك. تتوفر المكونات في الوثائق الرسمية لإطار CSS، ويمكنك نسخها ولصقها في مشروعك. يُصنف Bootstrap كإطار CSS معتمد على المكونات.

إليك مثالًا على استخدام Bootstrap لإنشاء مجموعة قائمة. بدلًا من تطبيق أصناف صغيرة على عناصر HTML، ستضيف المكوِّن كاملًا، بما في ذلك الهيكلية HTML.

```html
<div class="card" style="width: 25rem;">
  <ul class="list-group list-group-flush">
    <li class="list-group-item">HTML</li>
    <li class="list-group-item">CSS</li>
    <li class="list-group-item">JavaScript</li>
  </ul>
</div>
```

## Tailwind CSS

Tailwind هو إطار CSS ذو أولوية للأدوات. بدلًا من كتابة قواعد CSS مخصصة، تبني تصميماتك بدمج أصناف أدوات صغيرة مباشرة في HTML.

### أدوات التصميم المتجاوب

يستخدم Tailwind بادئات مثل `sm:`, `md:`, و`lg:` لتطبيق الأنماط عند أحجام شاشة مختلفة.

```html
<div class="w-full md:w-1/2 lg:flex-row">Responsive layout</div>
```

### أدوات Flexbox

أصناف مثل `flex`, `flex-col`, `justify-around`, و`items-center` تسهل إنشاء تخطيطات مرنة.

```html
<div class="flex flex-col md:flex-row justify-around items-center">
  <p>Column on small screens</p>
  <p>Row on medium and larger screens</p>
</div>
```

### أدوات الشبكة (Grid)

يشمل Tailwind أدوات لـ CSS Grid، مثل `grid`, `grid-cols-1`, و`md:grid-cols-3`.

```html
<div class="grid grid-cols-1 md:grid-cols-3 gap-8">
  <div class="bg-gray-100 p-4">Column 1</div>
  <div class="bg-gray-100 p-4">Column 2</div>
  <div class="bg-gray-100 p-4">Column 3</div>
</div>
```

### أدوات التباعد

أدوات مثل `mt-8`, `mx-auto`, `p-4`, و`gap-4` تساعد على إنشاء تباعد متناسق بدون كتابة CSS.

```html
<div class="mt-8 p-4 bg-indigo-600 text-white">Spaced content</div>
```

### أدوات الطباعة

أدوات مثل `uppercase`, `font-bold`, `font-semibold`, و`text-4xl` تتحكم في مظهر النص.

يمكنك تعيين أحجام الخطوط التي تتغير عند نقاط التوقف، مثل `text-3xl` و`md:text-5xl`.

```html
<h1 class="text-3xl md:text-5xl font-semibold text-center">Responsive Heading
</h1>
```

### الألوان وحالات التمرير

يوفر Tailwind لوحة ألوان واسعة، مثل `text-red-700`, `bg-indigo-600`, و`bg-gray-100`.

أصناف مثل `hover:bg-pink-600` تجعل التأثيرات التفاعلية بسيطة.

```html
<a href="#" class="bg-pink-500 hover:bg-pink-600 text-white px-4 py-2 rounded-md">
  Hover Me
</a>
```

### الحدود، الحلقات، والتأثيرات

- **الحدود**: `border-2 border-red-300` تضيف حدودًا بسماكة وألوان محددة.
- **الحلقات**: `ring-1 ring-gray-300` تخلق تأثيرات تشبه الإطار تُستخدم غالبًا للتركيز أو البطاقات.
- **الزوايا المستديرة والتكبير**: أصناف مثل `rounded-md`, `rounded-xl`, و`scale-105` تضيف لمسة نهائية.

```html
<div class="p-6 rounded-xl ring-2 ring-fuchsia-500 scale-105">
  Highlighted card
</div>
```

### التدرجات اللونية

يدعم Tailwind أدوات التدرجات مثل `bg-gradient-to-r from-fuchsia-500 to-indigo-600`.

```html
<div class="p-4 text-white bg-gradient-to-r from-fuchsia-500 to-indigo-600">
  Gradient background
</div>
```

## معالجات CSS المسبقة

- **معالج CSS المسبق**: هو أداة توسع CSS القياسي. يُحوّل الشفرة ذات الصيغة الموسعة إلى ملف CSS أصلي. يساعد في كتابة CSS أنظف، قابلة لإعادة الاستخدام، أقل تكرارًا، وقابلة للتوسع للمشاريع المعقدة.
- **الميزات**: بعض الميزات التي توفرها معالجات CSS المسبقة هي المتغيرات، الخليط، التسكين، ووراثة المحدد.
- **معالجات CSS المسبقة الشهيرة**: من أشهرها Sass وLess وStylus.
- **العيوب المحتملة**:
  - قد يسبب تحويل قواعد CSS إلى CSS قياسي عبئًا إضافيًا.
  - قد يصعب معالجة الأخطاء في الشفرة المحولة.

## Sass

- **Sass**: هو أحد أشهر معالجات CSS المسبقة. Sass تعني "أوراق الأنماط الرائعة نحويًا".
- **الميزات المدعومة في Sass**: يدعم Sass ميزات مثل المتغيرات، قواعد CSS المسكنة، الوحدات، الخليط، الوراثة، والعمليات الحسابية الأساسية.

## نوعان من الصيغ المدعومة في Sass

- **صيغة SCSS**: توسع SCSS (Sassy CSS) الصيغة الأساسية لـ CSS. هي الصيغة الأكثر استخدامًا في Sass. ملفات SCSS تحمل الامتداد `.scss`.

إليك مثالًا على تعريف واستخدام متغير في SCSS.

```scss
$primary-color: #3498eb;

header {
  background-color: $primary-color;
}
```

- **الصيغة المسندة**: كانت الصيغة الأصلية لـ Sass وتعرف أيضًا باسم "صيغة Sass".

إليك مثالًا على تعريف واستخدام متغير في الصيغة المسندة.

```sass
$primary-color: #3498eb

header
  background-color: $primary-color
```

### الخليط (Mixins)

- **الخليط**: يسمح لك بتجميع عدة خصائص CSS وقيمها تحت اسم واحد وإعادة استخدام ذلك المقطع من الشفرة في كامل ملف الأنماط.

إليك مثالًا على تعريف خليط في صيغة SCSS. في هذه الحالة، الخليط يسمى `center-flex`. يحتوي على ثلاث خصائص CSS لمركزة العناصر باستخدام flexbox.

```scss
@mixin center-flex {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

إليك مثالًا على استخدام الخليط الذي عرّفته.

```scss
section {
  @include center-flex;
  height: 500px;
  background-color: #3289a8;
}
```
