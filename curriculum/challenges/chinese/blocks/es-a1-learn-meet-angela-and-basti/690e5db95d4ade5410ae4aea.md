# --description--

`Mi nombre es` 意思是 "My name is"。

**`Mi nombre es` + [name]** 是用西班牙语介绍自己时常用且礼貌的表达方式。

该短语使用动词 `ser`（“是”）的第三人称单数形式，即 `es`（“是”）。
