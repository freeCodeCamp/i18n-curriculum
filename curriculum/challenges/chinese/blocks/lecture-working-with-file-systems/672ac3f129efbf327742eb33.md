# --description--

让我们学习如何在 Windows 和 macOS 上查找文件和文件夹。

我们将从 Windows 开始。你可以从任务栏或文件资源管理器中查找。

要从任务栏查找，请输入你要查找的文件或文件夹名称。你也可以查找关键字。你将看到来自你电脑和 OneDrive 的结果。

如果你只想查看文档，可以点击 `"Documents"` 标签（页）。你还有用于应用和网页结果的标签（页）。

另一个备选是使用 `Search File Explorer`。你会在 `Explorer window` 的右上角找到它，在那里你可以看到 `"Search Documents"`。

你有三个选项来定义查找的作用域：

- 你可以从 Home 进行查找，快速查找来自你的 PC 和云端的文件。

- 你可以从一个文件夹中查找存储在其中的文件。

- 你可以从 This PC 进行查找，速度较慢但更深入。

这就是你如何在 Windows 上查找文件和文件夹。现在让我们看看你如何在 macOS 上执行此操作。

首先，你可以使用 Spotlight，这是一款帮助你查找 Mac 上文件的工具。要打开 Spotlight，请前往右上角的菜单栏并查找放大镜图标。如果你在菜单栏上看不到此图标，可以在控制中心设置中将其添加。

进入你的设置，按 "Control Center" 过滤，点击此选项，然后向下滚动直到找到 "Spotlight" 选项。你也可以使用快捷键打开 Spotlight：`Command + Spacebar`。

当你打开 Spotlight 时，你会看到 Spotlight 查找，在这里你可以通过类型指派名称来查找任何文件或文件夹。当你开始类型指派时，你会看到按类别分组的结果，包括建议、文件夹、演示文稿、照片、文档和相关查找。

你也可以点击 "Show More" 来展开特定类别。

另一种查找你最近打开的 `file` 的方法是通过点击 Dock 中的 Finder `icon` 打开 Finder。然后，转到 Finder 侧边栏中的“最近使用”。在那里，你会找到所有你最近查看过的 `file`。

这就是你如何在 macOS 上查找文件和文件夹。通过这些查找工具，你可以优化你的工作流程，并非常快速地找到你需要的文件和文件夹。
