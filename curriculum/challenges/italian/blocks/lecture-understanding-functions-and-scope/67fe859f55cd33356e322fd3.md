# --description--

Le funzioni sono porzioni di codice riutilizzabili che vengono eseguite quando le chiami. Molti linguaggi di programmazione includono funzioni integrate che facilitano l'inizio. Python non fa eccezione, e abbiamo già visto alcune funzioni integrate come `print()` nelle lezioni precedenti.

Un'altra funzione integrata utile è `input()`, che ti permette di chiedere un input all'utente:

```python
name = input('What is your name?') # User types "Kolade" and presses Enter  
print('Hello', name) # Output: Hello Kolade
```

D'altra parte, `int()` converte un numero, un booleano e una stringa numerica in un intero:

```python
print(int(3.14)) # 3
print(int('42')) # 42
print(int(True)) # 1
print(int(False)) # 0 
```

Puoi anche scrivere le tue funzioni personalizzate. Per farlo, usi la parola chiave `def`, seguita dal nome che vuoi dare alla funzione, una coppia di parentesi tonde e due punti. Poi, su una nuova riga, scrivi il codice che la funzione deve eseguire. Il codice che la funzione esegue è anche chiamato corpo della funzione.

Ecco un esempio di una funzione personalizzata chiamata `hello` che stampa la stringa `Hello World` nel terminale:

```python
def hello():
    print('Hello World')
```

Per eseguire la funzione, devi chiamarla con il suo nome seguito da una coppia di parentesi tonde:

```python
hello() # Hello World
```

Nota l'indentazione prima di `print('Hello World')`. Come ricorderai dalle lezioni precedenti, Python si basa sull'indentazione per determinare quali gruppi di istruzioni appartengono insieme. Questi gruppi di istruzioni si chiamano blocchi di codice.

Ecco un'altra funzione semplice che stampa la somma di due numeri nel terminale:

```python
def calculate_sum(a, b):
    print(a + b)
```

Puoi vedere che la nostra funzione, `calculate_sum`, ha `a` e `b` nelle sue parentesi tonde, separate da una virgola. Questi si chiamano parametri. Pensa ai parametri come a variabili segnaposto che fungono da "slot" per i valori che passi alle funzioni quando le chiami.

Per usare i parametri, devi passare degli "argomenti". Gli argomenti sono i valori che passi a una funzione quando la chiami.

Ecco come chiamare la funzione `calculate_sum` per sommare i numeri `3` e `1`:

```python
calculate_sum(3, 1) # 4
```

Se chiami la funzione senza il numero corretto di argomenti, otterrai un `TypeError`:

```python
calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

Le funzioni usano anche una parola chiave speciale `return` per uscire dalla funzione e restituire un valore. Se non usi esplicitamente `return`, Python restituirà `None` per impostazione predefinita.

Ecco un esempio:

```python
def calculate_sum(a, b):
    print(a + b)

my_sum = calculate_sum(3, 1) # 4
print(my_sum) # None
```

Puoi vedere che la funzione `calculate_sum` stampa la somma di `a` e `b`, ma non restituisce nulla esplicitamente. Quindi, quando assegni il suo risultato a `my_sum`, il valore è in realtà `None`. Per risolvere questo, puoi usare la parola chiave `return` per restituire il risultato:

```python
def calculate_sum(a, b):
    return a + b

my_sum = calculate_sum(3, 1)
print(my_sum) # 4
```

Ora, `calculate_sum` restituisce la somma di `a` e `b`, che viene archiviata in `my_sum`.
