# --description--

O array de títulos de livros é bom, mas seria interessante exibir o `title` e o `author` para cada livro assim:

```js
[ 
  "Your Next Five Moves: Master the Art of Business Strategy by Patrick Bet-David and Greg Dinkin",
  "Atomic Habits by James Clear",
  "Choose Your Enemies Wisely: Business Planning for the Audacious Few by Patrick Bet-David",
  ...
]
```

Atualize seu uso do método `map()` para retornar um array de strings neste formato: `TITLE by AUTHOR`. 

Você precisará usar ou template literals ou concatenação de strings com o operador `+` para alcançar esse resultado. 

# --hints--

