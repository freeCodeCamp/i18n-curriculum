# --questions--

## --text--

哪一句使用 `in` 的方式與這句話相同：`The server room is located in the building's basement`？

## --answers--

`The developer worked in the morning to fix the bug.`

### --feedback--

這裡使用的 `in` 是指時間，而非像範例句子中指位置。

---

`She found an error in the code after testing.`

### --feedback--

這裡，`in` 顯示在某個抽象的（程式碼）內的位置，但它不是像範例中那樣的實體位置。

---

`The printer is in the office next to the window.`

---

`He is interested in learning new programming languages.`

### --feedback--

這顯示的是與利息相關的 `in`，而非實體的位置。

## --video-solution--

3

# --explanation--

在範例句中，`in` 表示實體的位置——某物在一個地方的內部。

正確答案也使用 `in` 來表示實體的位置：印表機在辦公室內。

其他句子使用 `in` 表示時間、抽象的位置或興趣，這些用法不同。
