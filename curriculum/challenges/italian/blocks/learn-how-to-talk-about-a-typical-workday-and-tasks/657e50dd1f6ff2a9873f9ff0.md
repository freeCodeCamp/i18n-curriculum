# --description--

`Compliance` significa rispettare le regole e fare ciò che è richiesto in un lavoro. Si tratta di assicurarsi di fare le cose nel modo giusto. Per esempio:

`Compliance with safety rules keeps everyone safe.` - Questo significa che seguire le regole di sicurezza aiuta a proteggere tutte le persone.
