# --explanation--

`Should have been` は形容詞の前に続き、過去に起こるべきだったけれど起こらなかったことを表します。例えば：

- `I should have been more careful.` - 注意深くなかったけれど、注意深くしていればよかった。

- `He should have been more prepared for the meeting.` - 十分に準備ができていなかった。

この場合、マークはコメントをもっと詳しくすべきだったと言っています。
