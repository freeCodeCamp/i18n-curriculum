# --description--

De manera similar, los usuarios deberían poder hacer clic en la dirección de correo electrónico y enviar un correo desde su cliente de correo electrónico predeterminado.

Aquí está cómo puedes hacer que las direcciones de correo electrónico sean clicables:

```html
<a href="mailto:contact@company.com">contact@company.com</a>
```

Para este paso final, envuelve el texto `fake@email.com` en un elemento de anclaje y usa `mailto:` para convertirlo en una dirección de correo electrónico clicable.

Y con esos cambios, tu página de blog está ahora completa.

# --hints--

Deberías envolver el texto `fake@email.com` en un elemento `a`.

```js
const secondPElement = document.querySelector('footer #contact address p:nth-of-type(2)');
const link = secondPElement.querySelector('a');
assert.exists(link);
assert.strictEqual(link.textContent.trim(), "fake@email.com");
```

Tu elemento `a` debería tener un atributo `href`.

```js
const secondPElement = document.querySelector('footer #contact address p:nth-of-type(2)');
const link = secondPElement.querySelector('a');
assert.exists(link.getAttribute('href'));
```

El atributo `href` para tu nuevo elemento `a` debería establecerse en `mailto:fake@email.com`.

```js
const secondPElement = document.querySelector('footer #contact address p:nth-of-type(2)');
const link = secondPElement.querySelector('a');
assert.strictEqual(link.getAttribute('href'), "mailto:fake@email.com");
```
