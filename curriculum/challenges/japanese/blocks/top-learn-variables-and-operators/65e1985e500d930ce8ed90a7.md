# --description--

JavaScriptの変数名には2つの制限があります：

1. 名前には文字、数字、または記号`$`と`_`のみを含める必要があります。
1. 最初の文字は数字であってはいけません。

有効な名前の例：

```js
let userName;
let test123;
```

興味深いことに、ドル記号`'$'`とアンダースコア`'_'`も名前に使えます。これらは文字と同じように特別な意味を持たない通常の記号です。

これらの名前は有効です：

```js
let $ = 1; // declared a variable with the name "$"
let _ = 2; // and now a variable with the name "_"

console.log($ + _); // 3
```

無効な変数名の例：

```js
let 1a; // cannot start with a digit

let my-name; // hyphens '-' aren't allowed in the name
```
