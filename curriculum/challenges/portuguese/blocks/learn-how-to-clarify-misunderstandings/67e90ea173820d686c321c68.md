# --explanation--

Um exemplo é `the "Student" class inherits properties and methods from the "Person" class`.

- `Student` e `Person` são ambas `classes`. Uma `class` define um modelo para criar objetos com dados e comportamentos específicos.

- `Student` herda funcionalidades de `Person`, o que significa que não precisa redefinir características compartilhadas — ele as reutiliza.

- `Properties` são as variáveis ou dados dentro da classe, como `name` ou `age`, que descrevem o objeto.

- `Methods` são as funções definidas na classe, como `enroll()` ou `introduce()`, que descrevem o que o objeto pode fazer.
