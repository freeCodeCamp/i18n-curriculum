# --explanation--

이 문맥에서 `point`는 대화 중 누군가가 공유하는 생각, 주석, 또는 의견을 의미합니다. `That's an important point`라고 말하는 것은 동의하며 그 생각이 가치 있다고 여긴다는 뜻입니다. 또 다른 예:

`She made a good point about improving password strength.` – 이것은 그녀의 제안이나 생각이 유용하고 타당했다는 의미입니다.
