# --explanation--

`Set up right` significa configurare qualcosa correttamente affinché funzioni come previsto. Se qualcosa non è `set up right`, potrebbe non funzionare correttamente. Per esempio:

- `The printer isn't working because it's not set up right.` - La stampante non è configurata correttamente.

- `Make sure the software is set up right before you start using it.` - Devi configurare correttamente il software per assicurarti che funzioni.
