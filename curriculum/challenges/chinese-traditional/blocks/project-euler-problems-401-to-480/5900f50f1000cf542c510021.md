# --description--

設 $n$ 為正整數。若整數三元組 ($a$、$b$、$c$) 為 $n$ 的因式分解三元組，則：

- $1 ≤ a ≤ b ≤ c$
- $a \times b \times c = n$。

定義 $f(n)$ 為因數分解三元組 ($a$、$b$、$c$) 的 $a + b + c$，該三元組使得 $\frac{c}{a}$ 最小化。可以證明此三元組是唯一的。

例如，$f(165) = 19$、$f(100\\,100) = 142$ 和 $f(20!) = 4\\,034\\,872$。

求 $f(43!)$。
