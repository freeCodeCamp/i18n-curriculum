# --explanation--

`Typically` は、通常または予想される方法で何かが起こることを表すために使われます。`usually` に似ていて、通常何が起こるかを話すときによく使われます。例えば：

- `Typically, we have meetings on Mondays.` - これは会議が通常月曜日に行われることを意味します。

- `She typically arrives at work by 9 AM.` - これは彼女が通常午前9時までに仕事に到着することを意味します。
