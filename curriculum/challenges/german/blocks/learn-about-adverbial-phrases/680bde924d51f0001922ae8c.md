# --explanation--

`To be based on something` bedeutet, dass eine Idee, ein Plan oder eine Entscheidung auf dieser Sache basiert oder von ihr ausgeht. Es zeigt, was die endgültige Wahl beeinflusst oder prägt. Zum Beispiel:

- `Their new pricing model is based on customer feedback.` – Das bedeutet, sie haben die Preise anhand der Meinungen und Vorschläge der Kunden erstellt.

- `The movie is based on a true story.` – Das bedeutet, die wahre Geschichte hat die im Film gezeigten Ereignisse inspiriert.
