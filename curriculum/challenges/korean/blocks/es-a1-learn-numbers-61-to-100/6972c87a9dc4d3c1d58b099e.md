# --description--

이제 90부터 99까지 숫자를 살펴보겠습니다.

스페인어에서 90은 `noventa`라고 합니다.

이 숫자들은 10의 자리 단어 `noventa` (90)에 접속사 `y` ("그리고")를 붙이고, 그 다음에 1부터 9까지의 일의 자리 숫자를 덧붙여 만듭니다.

| 숫자 | 스페인어 |
| :--- | :--- |
| 90 | `Noventa` |
| 91 | `Noventa y uno` |
| 92 | `Noventa y dos` |
| 93 | `Noventa y tres` |
| 94 | `Noventa y cuatro` |
| 95 | `Noventa y cinco` |
| 96 | `Noventa y seis` |
| 97 | `Noventa y siete` |
| 98 | `Noventa y ocho` |
| 99 | `Noventa y nueve` |

이 범위에서 숫자 `noventa y uno` (91)은 독특한데, 남성 명사 앞에서는 `noventa y un`로 변합니다. 예를 들어:

`Noventa y un años` - 91년.
