# --description--

A palavra `everyone` significa toda pessoa em um grupo. `Every-` é usado quando se fala sobre todas as coisas ou pessoas em um grupo, uma por uma.

É importante notar que `everyone` soa como se estivesse falando sobre muitas pessoas, mas você o usa com um verbo no singular.Isto é semelhante a outras palavras que começam com `every-` como `everything` e `everybody`.

Por exemplo, você diz `Everyone goes to the party`, `Everything is right` ou `Everybody works hard`.

`Make sure` significa verificar ou garantir que algo está correto antes de continuar.Por exemplo, `You make sure the code runs` significa que você verifica o código para ter certeza de que não há erros e que ele pode ser executado corretamente.

`Follow the rules` significa fazer o que as regras dizem.Se houver uma regra que diz `use camelCase for variable names`, para `follow the rules` significa que você nomeia suas variáveis como `myVariable` em vez de `myBLANKvariable`.
