# --explanation--

La oración completa es `Hola, Camila. ¿Cuál es tu número de teléfono?`

Esta pregunta sigue una estructura que ya has visto anteriormente:

`¿Cuál es tu` + noun?

Aquí, el sintagma nominal es `número de teléfono`.

Tanto `número` como `teléfono` tienen tildes, las cuales son importantes para la ortografía y pronunciación correctas.

La frase se usa para pedir el número de teléfono de alguien de manera educada y directa.
