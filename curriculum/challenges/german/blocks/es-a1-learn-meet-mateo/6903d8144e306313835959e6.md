# --description--

Mateo sagt auf Spanisch, wie alt er ist, und verwendet dabei die folgende Struktur mit dem Verb `tener` („haben“):

`Tengo` + Zahl in Worten + `años`.

Denken Sie daran, dass Zahlen im Spanischen von 21 bis 29 mit `veinti-` beginnen, gefolgt von der entsprechenden Zahl. Zum Beispiel:

25 ist `veinticinco`. Dies entsteht, indem man `e` entfernt und `i` vor der Zahl `cinco` hinzufügt.

Zahlen von 1 bis 9 im Spanischen sind:

| Zahl | Spanisches Wort |
| :---: | :---: |
| 1 | `uno` |
| 2 | `dos` |
| 3 | `tres` |
| 4 | `cuatro` |
| 5 | `cinco` |
| 6 | `seis` |
| 7 | `siete` |
| 8 | `ocho` |
| 9 | `nueve` |
