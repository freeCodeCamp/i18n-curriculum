# --explanation--

Ángela dice `¿Navarro con dos erres?` Esta es una pregunta de confirmación.

`Con` introduce información adicional sobre la ortografía.

`Dos` es el número dos.

`Erres` es la forma plural de `erre`, el nombre de la letra `r`.

Ángela está confirmando que el apellido Navarro se escribe con dos letras `r`.
