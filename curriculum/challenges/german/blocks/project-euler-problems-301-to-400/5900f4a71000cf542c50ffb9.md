# --description--

Der Mond wurde erschlossen, und Land kann kostenlos erworben werden, aber es gibt einen Haken. Sie müssen eine Mauer um das Land bauen, das Sie abstecken, und das Bauen einer Mauer auf dem Mond ist teuer. Jedem Land wurde ein 500 m mal 500 m großes Quadrat zugeteilt, aber sie besitzen nur das Gebiet, das sie mit einer Mauer umschließen. 251001 Pfosten wurden in einem rechteckigen Raster mit 1 Meter Abstand aufgestellt. Die Mauer muss eine geschlossene Reihe von geraden Linien sein, wobei jede Linie von Pfosten zu Pfosten verläuft.

Die größeren Länder haben natürlich eine 2000 m lange Mauer gebaut, die das gesamte 250.000 $\text{m}^2$ große Gebiet umschließt. Das Herzogtum Grand Fenwick hat ein engeres Budget und hat Sie (ihren königlichen Programmierer) gebeten zu berechnen, welche Form das beste maximale Verhältnis von $\frac{\text{eingeschlossener Bereich}}{\text{Mauerlänge}}$ erzielt.

Sie haben einige Vorberechnungen auf einem Blatt Papier gemacht. Für eine 2000 Meter lange Mauer, die das 250.000 $\text{m}^2$ große Gebiet umschließt, beträgt das Verhältnis $\frac{\text{eingeschlossener Bereich}}{\text{Mauerlänge}}$ 125.

Obwohl nicht erlaubt, aber um eine Vorstellung zu bekommen, ob das besser ist: Wenn Sie einen Kreis innerhalb des quadratischen Gebiets platzieren, der die vier Seiten berührt, ist die Fläche gleich $π \times {250}^2 \text{m}^2$ und der Umfang $π \times 500 \text{m}$, sodass das Verhältnis $\frac{\text{eingeschlossener Bereich}}{\text{Mauerlänge}}$ ebenfalls 125 beträgt.

Wenn Sie jedoch vom Quadrat vier Dreiecke mit den Seiten 75 m, 75 m und $75\sqrt{2}$ m abschneiden, wird die Gesamtfläche 238.750 $\text{m}^2$ und der Umfang $1400 + 300\sqrt{2}$ m. Das ergibt ein Verhältnis von $\frac{\text{eingeschlossener Bereich}}{\text{Mauerlänge}}$ von 130,87, was deutlich besser ist.

Finden Sie das maximale Verhältnis von $\frac{\text{eingeschlossener Bereich}}{\text{Mauerlänge}}$. Geben Sie Ihre Antwort gerundet auf 8 Nachkommastellen in der Form abc.defghijk an.
