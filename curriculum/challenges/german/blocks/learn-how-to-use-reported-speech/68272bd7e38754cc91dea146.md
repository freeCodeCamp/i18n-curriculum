# --explanation--

Ein `phishing attack` ist, wenn eine gefälschte Nachricht (wie eine E-Mail oder eine Nachricht auf einer Website) versucht, Menschen dazu zu verleiten, private Informationen wie Passwörter oder Kreditkartennummern zu teilen. Zum Beispiel:

`The team received a phishing email that looked like it was from IT support asking for login details.` – Das bedeutet, dass jemand versucht hat, Informationen zu stehlen, indem er vorgab, jemand anderes zu sein.
