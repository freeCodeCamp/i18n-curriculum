# --description--

Гіпоциклоїда — це крива, яку описує точка на малому колі, що котиться всередині більшого кола. Параметричні рівняння гіпоциклоїди, центрованої в початку координат і що починається з правої крайньої точки, задаються так:

$$x(t) = (R - r) \cos(t) + r \cos(\frac{R - r}{r}t)$$

$$y(t) = (R - r) \sin(t) - r \sin(\frac{R - r}{r} t)$$

Де $R$ — радіус великого кола, а $r$ — радіус малого кола.

Нехай $C(R, r)$ — множина різних точок з цілими координатами на гіпоциклоїді з радіусами $R$ та $r$, для яких існує відповідне значення $t$, таке що $\sin(t)$ та $\cos(t)$ є раціональними числами.

Нехай $S(R, r) = \sum\_{(x,y) \in C(R, r)} |x| + |y|$ — сума абсолютних значень координат $x$ та $y$ точок із множини $C(R, r)$.

Нехай $T(N) = \sum_{R = 3}^N \sum_{r=1}^{\left\lfloor \frac{R - 1}{2} \right\rfloor} S(R, r)$ — сума $S(R, r)$ для натуральних чисел $R$ та $r$, де $R \leq N$ та $2r < R$.

Вам задано:

$$\begin{align}
  C(3, 1) = & \\{(3, 0), (-1, 2), (-1,0), (-1,-2)\\} \\\\
  C(2500, 1000) = & \\{(2500, 0), (772, 2376), (772, -2376), (516, 1792), (516, -1792), (500, 0), (68, 504), \\\\
                  &(68, -504),(-1356, 1088), (-1356, -1088), (-1500, 1000), (-1500, -1000)\\}
\end{align}$$

**Примітка:** (-625, 0) не є елементом множини $C(2500, 1000)$, бо $\sin(t)$ не є раціональним числом для відповідних значень $t$.

$S(3, 1) = (|3| + |0|) + (|-1| + |2|) + (|-1| + |0|) + (|-1| + |-2|) = 10$

$T(3) = 10$; $T(10) = 524$; $T(100) = 580\\,442$; $T({10}^3) = 583\\,108\\,600$.

Знайдіть $T({10}^6)$.
