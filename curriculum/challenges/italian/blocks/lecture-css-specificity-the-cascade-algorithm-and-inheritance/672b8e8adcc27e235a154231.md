# --interactive--

I selettori di tipo, noti anche come selettori di elemento, mirano agli elementi in base al loro nome di tag.

Questi selettori sono fondamentali in CSS e ti permettono di applicare stili a tutte le istanze di un elemento HTML specifico.

I selettori di tipo sono semplici da usare e si scrivono semplicemente come il nome del tag dell'elemento che vuoi definire lo stile.

Ecco un esempio di selettore di tipo che mira a tutti gli elementi paragrafo nella pagina:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  color: blue;
}
```

:::

In questo esempio, tutti gli elementi `p` avranno il colore del testo impostato su `blue`.

I selettori di tipo hanno una specificità relativamente bassa rispetto ad altri selettori. Il valore di specificità per un selettore di tipo è `(0, 0, 0, 1)`.

Questo significa che i selettori di tipo verranno sovrascritti dai selettori di classe, dai selettori ID e dagli stili in linea, ma possono comunque applicare stili a meno che non siano presenti regole con specificità più alta.

Diamo un'occhiata a un esempio in cui i selettori di classe sovrascrivono gli stili del selettore di tipo.

Ecco un esempio con due elementi paragrafo:

:::interactive_editor

```html
<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

:::

Entrambi gli elementi paragrafo hanno una classe chiamata `para`.

Nel file CSS, il selettore di tipo mira ai paragrafi, e il selettore di classe mira agli elementi con la classe `para`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

```css
p {
  color: blue;
}

.para {
  color: red;
}
```

:::

Tutti i paragrafi nella pagina con la classe `para` avranno il colore del testo impostato su `red` invece di `blue` perché i selettori di classe hanno una specificità più alta rispetto ai selettori di tipo.
