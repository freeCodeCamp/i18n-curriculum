# --description--

A differenza di lavorare con progetti più piccoli in vanilla HTML, CSS e JavaScript, iniziare un nuovo progetto React richiede qualche passaggio in più per assicurarsi che tutto funzioni correttamente. Invece di cercare di configurare tutto da solo, ci sono strumenti che lo fanno per te.

Uno degli strumenti più popolari per impostare progetti è Vite. Vite, che in francese significa "veloce", è uno strumento di build che punta a offrire un’esperienza di sviluppo più rapida per i progetti web moderni. Vite può essere usato con React, così come con altre librerie e framework come Vue o Svelte. Puoi anche usarlo con progetti vanilla JavaScript.

Per creare un nuovo progetto con Vite, devi usare la riga di comando. Se usi un computer Windows, puoi usare il Prompt dei comandi o Windows PowerShell. Se usi un Mac, puoi usare l’app Terminale.

Se non hai Node.js installato, che serve perché npm funzioni, ecco un collegamento alla pagina ufficiale di Node.js: https://nodejs.org. Segui le istruzioni per l’installazione e poi torna qui per continuare con l’installazione di React.

Una volta aperta la riga di comando, puoi usare il seguente comando:

```bash
npm create vite@latest my-react-app -- --template react
```

Questo comando crea un nuovo progetto React chiamato `my-react-app` usando il template React di Vite.

Poi puoi aprire il nuovo progetto e vedere il codice boilerplate React che Vite ha preparato per te.

La cosa fantastica di Vite è che fornisce solo i file assolutamente necessari per iniziare con il tuo progetto React.

Per far partire il progetto così com’è, devi installare le dipendenze usando i seguenti comandi nella riga di comando:

```bash
cd my-react-app
npm install
```

Il comando `cd my-react-app` serve per spostarti nella directory corretta dove si trova il tuo progetto.

Il comando `npm install` serve per installare le dipendenze necessarie a far funzionare correttamente il tuo progetto React. Le dipendenze sono librerie o pacchetti che il tuo progetto React richiede per funzionare, come React stesso, ReactDOM o altri pacchetti di terze parti che aggiungono funzionalità.

Eseguendo `npm install` verrà letto il file `package.json` nella directory del progetto e verranno installate tutte le dipendenze elencate, permettendoti di costruire e far girare la tua app React senza mancare nessun componente necessario.

Il file `package.json` è un file di configurazione chiave nei progetti che contiene metadati sul progetto, inclusi nome, versione e dipendenze. Definisce anche script, informazioni sulla licenza e altre impostazioni che aiutano a gestire il progetto e le sue dipendenze.

Una volta installate le dipendenze, dovresti notare una nuova cartella nel progetto chiamata `node_modules`.

La cartella `node_modules` è dove sono archiviati tutti i pacchetti e le librerie richiesti dal progetto. Questa cartella contiene il codice effettivo delle dipendenze elencate nel file `package.json`, incluse sia le dipendenze dirette del progetto sia quelle delle dipendenze stesse.

Per far partire il progetto, esegui il comando `npm run dev` e apri una nuova scheda del browser all’indirizzo `http://localhost:5173/`.

Dovresti vedere il template iniziale che Vite ha preparato per te.

Se devi fermare il server locale, premi `CTRL + C` nella riga di comando.

Per vedere il codice di questo template iniziale, puoi entrare nella cartella `src` dentro il progetto e dovresti vedere il file `App.jsx`.

Da lì puoi iniziare a modificare il file, salvare le modifiche e vedere i nuovi cambiamenti apparire nel browser.

E con questo, sei pronto per iniziare a costruire la tua applicazione React!
