Jessica 正在告訴 Mark 執行 `regular security audits` 以確保安全。

`security audit` 是檢查系統漏洞的處理程序。例如：

`After the security audit, the team made improvements to the system。` - 在審查後進行了變更以提升安全性。

`Regular` 是指隨時間持續發生的。 例如：

`Regular testing helps find problems before they get worse.`－經常測試有助於及早發現問題。
