# --interactive--

在为网站设计样式时，使用 HSL 颜色模型是最灵活且直观的颜色处理方式之一。HSL 代表色相、饱和度和亮度——定义颜色的三个关键组成部分。

HSL 颜色模型以更符合人类感知颜色的方式表现颜色。在本课中，我们将探讨什么是 HSL 颜色模型，以及 `hsl()` 函数在 CSS 中的用法。

色相是颜色的类型，用色轮上的角度表示，范围从 `0` 到 `360` 度。例如，`0` 度代表红色，`120` 度是绿色，`240` 度是蓝色。通过调整色相，你可以在色谱上切换不同的颜色。

饱和度指颜色的强度或纯度。它以百分比表示，从 `0%`（完全去饱和的灰度色）到 `100%`（该颜色最鲜艳的形式）。饱和度为 `100%` 的颜色是完全鲜艳的，而饱和度为 `0%` 的颜色看起来像灰色调。

亮度决定颜色的明暗程度，同样以百分比表示。亮度值为 `0%` 会产生黑色，`50%` 会给出色相的正常色调，`100%` 会得到白色。

在 CSS 中，`hsl()` 函数用于使用 HSL 颜色模型定义颜色。现代语法使用空格分隔的数值：

```css
element {
  color: hsl(hue saturation lightness);
}
```

你也可能在现有代码中遇到旧的逗号分隔语法，浏览器仍然支持：

```css
element {
  color: hsl(hue, saturation, lightness);
}
```

让我们通过一个例子来解析：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
body {
  background-color: hsl(0 0% 1% / 1);
}

p {
  color: hsl(120 100% 50%);
}
```

:::

在这个例子中，色相是 `120` 度，对应绿色。饱和度是 `100%`，所以绿色是完全鲜艳的。亮度是 `50%`，处于正常色调——既不太暗也不太亮。因此，段落的文本颜色将是明亮的绿色。

HSL 颜色模型的主要优势之一是它的直观性。通过调整饱和度和亮度值，你可以轻松改变颜色的鲜艳度或明暗，而无需改变核心颜色（色相）。

例如，如果你想创建同一颜色的不同色调或色彩，可以简单地调整亮度值。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="light">This is a light blue div.</div>
<div class="dark">This is a dark blue div.</div>
```

```css
div.light {
  background-color: hsl(240 100% 80%);
}

div.dark {
  background-color: hsl(240 100% 20%);
  color: hsl(0 0% 100%);
}
```

:::

这里，两个 `div` 元素使用相同的色相（`240` 度，即蓝色），但一个亮度为 `80%`（较浅的蓝色），另一个亮度为 `20%`（较深的蓝色）。

就像 RGB 模型一样，`hsl()` 函数也支持可选的 alpha 值来控制透明度。你可以在 `/` 分隔符后添加第四个参数。基本语法如下：

```css
element {
  background-color: hsl(hue saturation lightness / alpha);
}
```

我们来看一个例子：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: hsl(0 100% 50% / 0.5);
}
```

:::

这段代码会给 `div` 一个半透明的红色背景，其中色相设置为 `0` 度（红色），饱和度为 `100%`，亮度为 `50%`，alpha 为 `0.5`（50% 不透明度）。你也可以用百分比指定 alpha——例如，`hsl(0 100% 50% / 50%)` 等同于 `hsl(0 100% 50% / 0.5)`。

你还会在现有 CSS 中常见 `hsla()` 函数。这是最初在 HSL 颜色中包含透明度的方式，使用逗号分隔的数值：

```css
element {
  background-color: hsla(hue, saturation, lightness, alpha);
}
```

例如，`hsla(0, 100%, 50%, 0.5)` 是 `hsl(0 100% 50% / 0.5)` 的旧版等价写法。两者都被浏览器支持，但现在更推荐使用带 `hsl()` 分隔符的现代 `/` 语法。

HSL 颜色模型在需要创建配色方案并轻松调整色调或色彩时特别有用。

因为 HSL 将色相与饱和度和亮度分开，你可以保持一致的颜色主题，同时调整亮度来制作阴影或高光。这种灵活性使 HSL 成为设计师和开发者的最爱。

此外，与十六进制代码或 RGB 等其他方法相比，HSL 使颜色值更易读、更易理解。你可以更清楚地看到颜色背后的意图，并据此调整，尤其是在尝试不同设计时。

使用 HSL 颜色模型，你可以更灵活地控制颜色的色调、饱和度和亮度，使其成为在 CSS 中创建丰富且动态设计的强大工具。
