# --description--

Дано трикутник $ABC$ з цілими сторонами $a ≤ b ≤ c$ ($AB = c$, $BC = a$ та $AC = b$).

Кутові бісектриси трикутника перетинають сторони у точках $E$, $F$ та $G$ (див. зображення нижче).

<img alt="трикутник ABC, з кутовими бісектрисами, що перетинають сторони у точках E, F та G" src="https://cdn.freecodecamp.org/curriculum/project-euler/angular-bisectors.gif" style="колір фону: white; відступ: 10px; display: block; поле-right: auto; поле-left: auto; поле-bottom: 1.2rem;">

Відрізки $EF$, $EG$ та $FG$ ділять трикутник $ABC$ на чотири менші трикутники: $AEG$, $BFE$, $CGF$ та $EFG$. Можна довести, що для кожного з цих чотирьох трикутників відношення $\frac{\text{область}(ABC)}{\text{область}(\text{підтрикутник})}$ є раціональним числом. Однак існують трикутники, для яких деякі або всі ці відношення є цілими числами.

Скільки існує трикутників $ABC$ з периметром $≤ 100\\,000\\,000$, для яких відношення $\frac{\text{область}(ABC)}{\text{область}(AEG)}$ є цілим числом?
