数字の1から始めて、時計回りに右方向へ動くと、5×5のらせんが次のように形成されます。

同じ方法で形成された `n` × `n` のらせんの対角線上の数字の和はいくつですか？

# --solutions--

```js
const spiralDiagonals = (n) => {
  const Sn2 = (n) => {
    return n*(n+1)*(2*n+1)/6;
  };
  const Sn = (n) => {
    return n*(n+1)/2;
  };
  let sum = (Sn2(n-1) + Sn(n-1) + n-1) + (Math.floor(n/2) + Sn2(n));
  return sum;
};
```
