# --description--

次のいくつかのレッスンでは、Pythonで使われる基本的なシーケンス型であるリスト、タプル、レンジについて学びます。

リストのデータ型は、文字列、数値、あるいは他のリストで構成されることもある順序付きの要素のシーケンスです。リストはミュータブルでゼロベースのインデックスを使います。つまり、リストの最初の要素はインデックスゼロにあります。

リストの基本的な構文は次の通りです。

```py
cities = ['Los Angeles', 'London', 'Tokyo']
```

`cities` リストから要素をアクセスするには、そのシーケンス内のインデックス番号を参照します。以下は `cities` リストの最初の要素にアクセスする例です。

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # 'Los Angeles'
```

負のインデックスは、リストの先頭ではなく末尾から要素にアクセスするために使われます。リストの最後の要素にアクセスするには、インデックス `0` の代わりに `-1` を使います。

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # 'Tokyo'
```

リストを作成する別の方法は、`list()` コンストラクタを使うことです。`list()` コンストラクタはイテラブルをリストに変換するために使います。

```py
developer = 'Jessica'
list(developer) # ['J', 'e', 's', 's', 'i', 'c', 'a']
```

イテラブルとは、一度に1つずつループ処理できる特別なタイプのオブジェクトです。Pythonのループについては後で詳しく学びます。

リストの要素数を取得するには、`len()` 関数を次のように使います。

```py
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

特定のインデックスの値を更新したい場合は、次のようにします。

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

リストはミュータブルなので、有効なインデックス番号を渡せばどの要素でも更新できます。もしリストの範囲外のインデックス（正または負）を渡すと、`IndexError` が発生します。

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list assignment index out of range
"""
```

リストから要素を削除したい場合は、`del` キーワードを次のように使います。

```py
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

リストに要素が含まれているかどうかを確認したい場合は、`in` キーワードを次のように使います。

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

リストの中にリストがネストされていることもよくあります。

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

この例では、3つの人気プログラミング言語を含むネストされたリストが1つあります。ネストされたリストにアクセスするには、リストがゼロベースのインデックスなので `2` のインデックスを使います。

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

次に、そのネストされたリストから2番目の言語にアクセスするには、`1` のインデックスを使います。

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # 'Rust'
```

リストでよく使われるもう一つの手法は値のアンパックです。

リストから値をアンパックするとは、リストの値を新しい変数に代入する手法です。以下は `developer` リストを `name`、`age`、`job` という新しい変数にアンパックする例です。

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

ここで、`name` は `'Alice'` の値を持ち、`age` は `34` の値を持ち、`job` は `'Rust Developer'` の値を持ちます。

リストの残りの要素をまとめて取得したい場合は、アスタリスク（`*`）演算子を次のように使います。

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

この例では、`name` は `'Alice'` の値を持ち、`rest` は2つの要素、数値 `34` と文字列 `'Rust Developer'` のリストになります。

代入演算子の左側の変数の数がリストの要素数と一致しない場合、`ValueError` が発生します。

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job, city = developer

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: not enough values to unpack (expected 4, got 3)
"""
```

最後に見ていくのはスライス演算子（`:`）です。文字列と同様に、スライス演算子を使ってリストの一部にアクセスできます。

```py
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie', 'Brownies']
desserts[1:4] # ['Cookies', 'Ice Cream', 'Pie']
```

この例では、開始インデックスは `1` で、リストの2番目の要素を指します。次にスライス演算子の後に終了インデックス `4` を指定し、そのインデックスの要素は含まずにそれまでのすべてを取得します。

スライス演算子 `:` では、インデックス間の増分を決めるステップ間隔を指定することもできます。例えば次のような数値のリストがあったとします。

```py
numbers = [1, 2, 3, 4, 5, 6]
```

偶数だけのリストを抽出したい場合は、スライス演算子を次のように使います。

```py
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

最初の偶数はインデックス `1` にあるので、そこを開始インデックスにします。リストの最後まで取得したいので終了インデックスは省略します。最後にオプションのステップ間隔として `2` を指定し、デフォルトの `2` ではなく `1` ずつ増分します。
