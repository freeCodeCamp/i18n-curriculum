# --explanation--

`Logs` são registros do sistema que acompanham a atividade em uma rede. Por exemplo:

`The logs recorded a failed login attempt.` - Isso significa que os registros de atividade do sistema mostram que alguém tentou fazer login mas não teve sucesso.  

`IP address` é um identificador único para um dispositivo em uma rede. Por exemplo:

`The attacker used a different IP address each time.` - O atacante usou um identificador de rede único para cada tentativa.
