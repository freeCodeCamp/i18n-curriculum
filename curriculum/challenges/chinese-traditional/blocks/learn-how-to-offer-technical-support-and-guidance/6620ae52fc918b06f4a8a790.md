# --description--

`Error handling` 是程式設計中一個重要的概念。它指的是撰寫能夠管理並回應執行期間發生錯誤的程式碼的過程。這可以防止程式崩潰，並提供更順暢的使用者體驗。`error handling` 的技術在不同程式語言中有所不同。例如，在 Python 中，使用 try-except 區塊是一種常見的方法。
