# --description--

모든 정수 $n > 1$에 대해, 함수족 $f_{n, a, b}$는 다음과 같이 정의됩니다:

$f_{n, a, b}(x) ≡ ax + b\bmod n$ 여기서 $a, b, x$는 정수이며 $0 \lt a \lt n$, $0 \le b \lt n$, $0 \le x \lt n$입니다.

모든 $0 \le x \lt n$에 대해 $f_{n, a, b}(f_{n, a, b}(x)) \equiv f_{n, a, b}(x)\bmod n$를 만족하면 $f_{n, a, b}$를 축소 함수라고 부릅니다.

$n$에 대한 축소 함수의 개수를 $R(n)$이라 합시다.

$F(N) = \displaystyle\sum_{n = 2}^N R(n)$입니다.

$F({10}^7) ≡ 638\\,042\\,271\bmod 1\\,000\\,000\\,007$입니다.

$F({10}^{14})$를 구하세요. 답은 $1\\,000\\,000\\,007$로 나눈 나머지로 제출하세요.
