# --explanation--

當未經授權的人員存取敏感資訊時，就會發生 `data breach`。例如：

- 如果駭客取得私密資訊，就可能發生 `A data breach can happen if hackers get access to private information.`。如果網路犯罪者竊取資訊，這稱為資料外洩。

- `Companies must act quickly if there is a data breach.`－當發生資料外洩時，商務需要快速回應以修復問題。

Jessica 說加密資料和保護加密金鑰是防止此風險的關鍵。
