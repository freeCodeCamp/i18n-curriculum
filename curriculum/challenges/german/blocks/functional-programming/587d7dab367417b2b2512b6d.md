# --description--

Die letzten Aufgaben behandelten eine Reihe nützlicher Array- und String-Methoden, die den Grundsätzen der funktionalen Programmierung folgen. Wir haben auch `reduce` kennengelernt, eine mächtige Methode, mit der Probleme auf einfachere Formen reduziert werden können. Von der Berechnung von Durchschnitten bis zum Sortieren kann jede Array-Operation durch deren Anwendung erreicht werden. Denken Sie daran, dass `map` und `filter` Spezialfälle von `reduce` sind.

Kombinieren wir unser Wissen, um ein praktisches Problem zu lösen.

Viele Content-Management-Systeme (CMS) fügen die Titel eines Beitrags als Teil der URL hinzu, um das Lesezeichenmachen zu erleichtern. Wenn Sie zum Beispiel einen Medium-Beitrag mit dem Titel `Stop Using Reduce` schreiben, enthält die URL wahrscheinlich eine Form des Titel-Strings (`.../stop-using-reduce`). Das haben Sie vielleicht schon auf der freeCodeCamp-Seite bemerkt.

# --instructions--

Füllen Sie die `urlSlug`-Funktion so aus, dass sie einen String `title` umwandelt und die mit Bindestrich verbundene Version für die URL zurückgibt. Sie können jede der in diesem Abschnitt behandelten Methoden verwenden, aber nutzen Sie nicht `replace`. Hier sind die Anforderungen:

Die Eingabe ist ein String mit Leerzeichen und großgeschriebenen Wörtern

Die Ausgabe ist ein String, bei dem die Leerzeichen zwischen den Wörtern durch einen Bindestrich (`-`) ersetzt sind

Die Ausgabe sollte ausschließlich aus Kleinbuchstaben bestehen

Die Ausgabe darf keine Leerzeichen enthalten
