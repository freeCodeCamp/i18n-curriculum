# --description--

`Near` significa cerca en distancia o tiempo, mientras que `nearly` se usa para indicar que algo está casi, pero no completamente terminado.

Por ejemplo, si estás `near` de terminar un proyecto, estás cerca de completarlo. Si has `nearly` terminado, estás casi pero no del todo acabado.

La expresión `for nearly a decade now` combina `nearly` con `a decade` (diez años) para describir un largo período de tiempo, casi pero no del todo alcanzando diez años. Se usa para hablar sobre algo que ha estado sucediendo o continuando durante casi diez años.
