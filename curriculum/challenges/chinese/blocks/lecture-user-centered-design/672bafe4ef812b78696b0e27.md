# --interactive--

**注意**：某些交互示例可能使用你尚未学习的 CSS。不要担心试图理解所有的代码。示例的目的是向你展示这些设计概念的预览，以便你更好地理解其工作原理。

暗黑模式是网页应用中的一种特殊特色，你可以将默认的浅色配色方案更改为暗色配色方案。这有助于减少眼睛疲劳并提高在弱光条件下的可读性。在设计你的暗黑模式特色时，了解最佳实践非常重要，以确保你的暗黑模式特色有效且用户友好。

启用交互式编辑器，然后点击下面示例中的 `Toggle Dark Mode` 按钮以查看暗模式的工作原理。

:::interactive_editor

```html

<button id="theme-toggle">Toggle Dark Mode</button>

<div class="content">
  <p>This is a simple dark mode example.</p>
</div>

<script src="index.js"></script>
```

```css
body {

}

  margin: 1rem;

  document.body.classList.toggle('dark');

```

第一个考虑因素是在暗模式下避免使用饱和色。饱和色是指明亮且强烈的颜色。例如，在深灰色背景下，明亮的品红色按钮可能过于强烈，导致眼睛疲劳。相反，你应该在暗模式下使用去饱和色。去饱和色是指强度较低、饱和度较低且在暗模式下更舒适的颜色。要查看预览，你需要启用交互式编辑器。

暗黑模式的另一个考虑是使用纯黑色背景配白色文本。虽然这种高对比度可能有效，但也可能对眼睛过于刺激。相反，可以考虑使用深灰色背景配浅灰色文本，以获得更柔和的对比度。在暗黑模式下，文本会更护眼，阅读起来更舒适。要查看预览，你需要启用交互式编辑器。

另一个需要考虑的是网站品牌同一性与暗模式的使用。品牌同一性是一组表现品牌的视觉元素，例如标志、颜色和排版。在实现暗模式时，你应考虑暗模式特色如何与你品牌的颜色和风格保持一致。品牌图标和按钮保持全饱和度是可以的，而周围的元素则可以降低饱和度。

通用的来说，在设计时，你总是要注意用户体验和对比度。暗黑模式也不例外，通过遵循这些最佳实践，你可以创建一个有效且用户友好的暗黑模式特色。
