# --explanation--

字 `Salvadoreña` 是來自薩爾瓦多的女性國籍形式。

國籍必須與所描述的人稱性別一致。例如：

| Gender | Nationality | Example |
| :--- | :--- | :--- |
| **Feminine** | `Salvadoreña` | `Ella es salvadoreña.` ("She is Salvadoran".) |
| **Masculine** | `Salvadoreño` | `Él es salvadoreño.` ("He is Salvadoran".) |
