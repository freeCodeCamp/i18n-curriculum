# --description--

Ein `security breach` ist, wenn jemand unbefugt auf ein System oder Daten zugreift. Das kann an Arbeitsplätzen ein großes Problem sein, besonders wenn es private Informationen betrifft. Zum Beispiel:

`A security breach occurred when an outsider accessed confidential files.` – Das bedeutet, es gab ein Sicherheitsproblem, weil jemand von außen auf private Dateien zugegriffen hat.

# --questions--

## --answers--
