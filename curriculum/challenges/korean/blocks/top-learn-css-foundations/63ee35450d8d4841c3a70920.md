# --description--

내부 CSS(또는 임베드된 CSS)는 완전히 별도의 파일을 만들지 않고 CSS를 HTML 파일 내에 직접 추가하는 방법입니다. 내부 방법에서는 모든 규칙을 `<style>` 태그 쌍 안에 넣고, 이 태그 쌍을 HTML 파일의 `<head>` 태그 쌍 안에 배치합니다. 스타일이 `<head>` 태그 안에 직접 들어가므로 외부 방법에서 필요한 `<link>` 요소가 더 이상 필요하지 않습니다.

이 차이점 외에는 구문이 외부 방법과 정확히 같습니다(선택자, 중괄호, 선언):

```html
<head>
  <style>
    div {
      color: white;
      background-color: black;
    }

    p {
      color: red;
    }
  </style>
</head>
<body>...</body>
```

이 방법은 웹사이트의 단일 페이지에 고유한 스타일을 추가할 때 유용할 수 있지만, 외부 방법처럼 분리하지 않으므로 규칙과 선언이 많아지면 HTML 파일 크기가 꽤 커질 수 있습니다.

# --questions--    

## --text--

내부 CSS 방법과 외부 CSS 방법의 차이점은 무엇인가요?

## --answers--

외부 방법은 CSS 규칙을 별도의 파일에 두는 반면, 내부 방법은 CSS 규칙을 HTML 파일 내에 직접 둡니다.

---

내부 방법은 CSS를 HTML과 분리하지만, 외부 방법은 CSS를 HTML에 직접 임베드합니다.

---

내부 방법은 CSS를 HTML에 연결하기 위해 `<link>` 요소를 사용하지만, 외부 방법은 CSS를 HTML에 직접 임베드합니다.
