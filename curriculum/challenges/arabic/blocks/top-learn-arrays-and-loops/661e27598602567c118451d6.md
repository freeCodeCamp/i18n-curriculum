# --description--

حلقة `for` هي نوع آخر من الحلقات تُستخدم لتنفيذ كتلة من الشفرة عدة مرات. تُستخدم حلقة `for` عندما يكون عدد التكرارات معروفًا. صيغة الحلقة `for` النحوية هي كما يلي:

```javascript
for (initialization; condition; increment/decrement) {
  // code block to be executed
}
```

على سبيل المثال، يطبع مقتطف الشفرة التالي الأرقام من 1 إلى 5:

```javascript
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
```

للتكرار على مصفوفة باستخدام حلقة `for`، يمكنك استخدام خاصية الطول للمصفوفة. على سبيل المثال، يطبع مقتطف الشفرة التالي عناصر المصفوفة:

```javascript
const arr = [1, 2, 3, 4, 5];
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}
```
