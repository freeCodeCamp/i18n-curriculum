# --description--

Un `concept` generalmente se refiere a una idea o principio que se usa para planificar o definir acciones y operaciones. En varios temas, incluyendo la programación, los `concepts` son las ideas o teorías fundamentales.

Por ejemplo, `Object-oriented programming is a concept in computer programming.`
