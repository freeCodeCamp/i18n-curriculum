# --description--

`flex` 宣言は、実はフレックスアイテムに設定できる3つのプロパティの省略形プロパティです。これらのプロパティは、フレックスアイテムがコンテナ内でどのようにサイズを決めるかに影響します。省略形プロパティは以前にも見たことがありますが、正式に定義したことはまだありません。

> 省略形プロパティとは、複数の他のCSSプロパティの値を同時に設定できるCSSプロパティのことです。省略形プロパティを使うと、より簡潔で（そして多くの場合読みやすい）スタイルシートを書けるため、時間と労力を節約できます。

この場合、`flex` は実際には `flex-grow`、`flex-shrink`、`flex-basis` の省略形プロパティです。

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/flex-box/flexbox-04.png" alt="CSS code setting the flex property to 1 for a div element.">

上のスクリーンショットでは、`flex: 1` は `flex-grow: 1`、`flex-shrink: 1`、`flex-basis: 0` に相当します。

非常に多くの場合、flexの省略形プロパティは1つの値だけで定義されます。その場合、その値は `flex-grow` に適用されます。ですから、divに `flex: 1` を設定したときは、実際には `flex: 1 1 0` の省略形プロパティを指定していたことになります。
