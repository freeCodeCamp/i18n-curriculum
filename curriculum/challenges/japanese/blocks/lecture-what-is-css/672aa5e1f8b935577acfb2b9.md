# --interactive--

CSS（Cascading Style Sheetsの略）は、現代のウェブ開発において重要なコンポーネントです。HTMLにスタイルを適用するためのマークアップ言語です。簡単に言うと、HTMLがウェブページの構造であるなら、CSSはその見た目を良くするものです。

ここにナビゲーションバーをスタイル付けする例があります。

**注意**：まだCSSコードを理解しようと心配しないでください。これらがどのように機能するかは、今後のレッスンやワークショップで学びます。これはCSSで何ができるかのイメージを持ってもらうためのものです。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<nav class="navbar">
    <ul class="nav-links">
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Services</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>
```

```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: 'Segoe UI', sans-serif;
  background-color: #f4f4f4;
}

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  background-color: #333;
  padding: 1rem 2rem;
  color: white;
}

.navbar .logo {
  font-size: 1.5rem;
  font-weight: bold;
}

.nav-links {
  list-style: none;
  display: flex;
  gap: 1.5rem;
}

.nav-links a {
  color: white;
  text-decoration: none;
  font-weight: 500;
  transition: color 0.3s ease;
}

.nav-links a:hover {
  color: #ff9800;
}
```

:::

CSSの主な役割は、ウェブページの内容と視覚的な見た目を分離することです。この分離により、開発者はより柔軟で保守しやすいウェブサイトを作成できます。

CSSを使うと、HTMLの構造を変えずにレイアウト、色、フォント、全体の見た目をコントロールできます。

簡単な例えを考えてみましょう。ウェブサイトを家と考えると、HTMLは基礎と骨組みで、CSSは家を見た目良くユニークにするためのペンキや壁紙、装飾にあたります。

CSSはHTML要素を選択してスタイルを適用することで機能します。

ここに、スタイルがない`button`要素とスタイル付きの例があります。この例を操作するにはインタラクティブエディターを有効にしてください。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<button class="regular-btn">Unstyled Button</button>
<button class="round-btn">Round Button</button>
```

```css

.round-btn {
  font-size: 1rem;
  font-family: 'Segoe UI', sans-serif;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease, transform 0.2s ease;
  background-color: #2ecc71;
  color: white;
  border-radius: 50px;
  padding: 0.6rem 1.6rem;
}

.round-btn:hover {
  background-color: #27ae60;
  transform: translateY(-2px);
}
```

:::

これらのスタイルには、色やフォントサイズなどのプロパティが含まれます。これらのプロパティを変えることで、内容を変えずにウェブページの見た目を大きく変えることができます。

CSSの最も強力な特徴の一つは、レスポンシブデザインを作れることです。

つまり、CSSを使うと、デスクトップパソコン、タブレット、スマートフォンなど、どんなデバイスでもウェブサイトを見栄え良くできます。

CSSは、ウェブサイトを見ているデバイスの画面サイズに応じて、レイアウトやフォントサイズ、その他の見た目の要素を調整できます。

インタラクティブエディターを有効にして、プレビューウィンドウのサイズを変えてみて、レイアウトが画面サイズにどう適応するかを見てみましょう。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<header class="header">
    <h1>Welcome to My Responsive Site</h1>
    <p>This layout adapts to your screen size!</p>
</header>

<main class="content">
    <section class="box">Box 1</section>
    <section class="box">Box 2</section>
    <section class="box">Box 3</section>
</main>
```

```css
body {
  margin: 0;
  font-family: 'Segoe UI', sans-serif;
  background-color: #f9f9f9;
  color: #333;
}

.header {
  background-color: #4CAF50;
  color: white;
  padding: 2rem;
  text-align: center;
}

.content {
  display: flex;
  gap: 1rem;
  padding: 1rem;
  justify-content: center;
}

.box {
  flex: 1;
  min-width: 200px;
  background-color: #ddd;
  padding: 2rem;
  text-align: center;
  font-size: 1.2rem;
  border-radius: 8px;
  transition: background-color 0.3s;
}

.box:hover {
  background-color: #ccc;
}

/* Responsive: for tablets and below (≤ 768px) */
@media (max-width: 768px) {
  .content {
    flex-direction: column;
    align-items: center;
  }

  .box {
    width: 90%;
    font-size: 1.1rem;
  }
}

/* Responsive: for phones (≤ 480px) */
@media (max-width: 480px) {
  .header {
    padding: 1.5rem 1rem;
  }

  .header h1 {
    font-size: 1.5rem;
  }

  .header p {
    font-size: 1rem;
  }

  .box {
    font-size: 1rem;
    padding: 1.5rem;
  }
}
```

:::

CSSのもう一つ重要な機能は、その「カスケーディング（階層的）」な性質です。これが名前の由来でもあります。

これは、スタイルが継承されたり上書きされたりできることを意味し、階層的なスタイリング構造を可能にします。この仕組みについては今後のレッスンで詳しく学びます。

CSSは外部スタイルシートの使用もサポートしています。

つまり、すべてのスタイルルールを別のファイルにまとめておき、それを複数のHTMLページにリンクできます。

この機能は特に大規模なウェブサイトの保守性を大きく向上させます。個々のページごとにスタイルを変える代わりに、1つのCSSファイルを変えるだけでリンクされたすべてのページに影響を与えられます。

ウェブ開発の世界で、CSSは視覚的に魅力的でレスポンシブ、かつユーザーフレンドリーなウェブサイトを作る上で欠かせない役割を果たしています。

開発者はCSSを使って、単純なHTML文書をユーザーの注意を引きつけ、ウェブコンテンツとのやり取りを向上させる魅力的なウェブ体験に変えることができます。
