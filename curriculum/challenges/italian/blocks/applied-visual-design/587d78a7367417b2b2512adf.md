# --description--

Per animare un elemento, devi conoscere le proprietà di animazione e la regola `@keyframes`. Le proprietà di animazione controllano come deve comportarsi l'animazione e la regola `@keyframes` controlla cosa succede durante quell'animazione. Ci sono otto proprietà di animazione in totale. Questa sfida sarà semplice e coprirà prima le due più importanti:

`animation-name` imposta il nome dell'animazione, che viene poi usato da `@keyframes` per dire a CSS quali regole corrispondono a quali animazioni.

`animation-duration` imposta la durata dell'animazione.

`@keyframes` serve a specificare esattamente cosa succede all'interno dell'animazione durante la sua durata. Questo si fa assegnando proprietà CSS a specifici "fotogrammi" durante l'animazione, con percentuali che vanno da 0% a 100%. Se lo paragoni a un film, le proprietà CSS per 0% indicano come l'elemento appare nella scena iniziale. Le proprietà CSS per 100% indicano come l'elemento appare alla fine, proprio prima dei titoli di coda. Poi CSS applica la magia per far transitare l'elemento durante la durata data per rappresentare la scena. Ecco un esempio per illustrare l'utilizzo di `@keyframes` e delle proprietà di animazione:

```css
#anim {
  animation-name: colorful;
  animation-duration: 3s;
}

@keyframes colorful {
  0% {
    background-color: blue;
  }
  100% {
    background-color: yellow;
  }
}
```

Per l'elemento con id `anim`, il frammento di codice sopra imposta la `animation-name` su `colorful` e imposta la `animation-duration` a 3 secondi. Poi la regola `@keyframes` collega le proprietà di animazione con il nome `colorful`. Imposta il colore su blu all'inizio dell'animazione (0%) che poi transiterà al giallo alla fine dell'animazione (100%). Non sei limitato solo alle transizioni dall'inizio alla fine, puoi impostare proprietà per l'elemento a qualsiasi percentuale tra 0% e 100%.
