# --description--

Tom verwendet `I'll` (Kurzform von `I will`), um darüber zu sprechen, was er später vorhat.

`Will` wird oft verwendet, wenn jemand entscheidet, in der Zukunft etwas zu tun. Zum Beispiel bedeutet `I'll call you later`, dass die Person plant, irgendwann nach jetzt anzurufen.
