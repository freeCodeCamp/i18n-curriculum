# --description--

Ao fornecer um nome ou um endereço de e-mail, uma pessoa pode precisar dizer cada letra separadamente para evitar confusão. Essa ação é chamada de `deletrear`. O verbo `deletrear` é usado quando alguém soletra uma palavra letra por letra. Por exemplo:

- `Deletreo meu sobrenome.`

- `¿Quieres que deletree tu nombre?` - Você quer que eu soletre seu nome?
