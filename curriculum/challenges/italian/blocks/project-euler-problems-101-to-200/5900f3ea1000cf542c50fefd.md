# --description--

Il numero minimo di cubi per coprire ogni faccia visibile di un cuboide di dimensioni 3 x 2 x 1 è ventidue.

<img alt="cuboide 3x2x1 coperto da ventidue cubi 1x1x1" src="https://cdn.freecodecamp.org/curriculum/project-euler/cuboid-layers.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Se aggiungiamo un secondo strato a questo solido, servirebbero quarantasei cubi per coprire ogni faccia visibile, il terzo strato richiederebbe settantotto cubi, e il quarto strato centodiciotto cubi per coprire ogni faccia visibile.

Tuttavia, il primo strato di un cuboide di dimensioni 5 x 1 x 1 richiede anch’esso ventidue cubi; similmente, il primo strato di cuboidi di dimensioni 5 x 3 x 1, 7 x 2 x 1 e 11 x 1 x 1 contiene tutti quarantasei cubi.

Definiamo $C(n)$ come il numero di cuboidi che contengono $n$ cubi in uno dei loro strati. Quindi $C(22) = 2$, $C(46) = 4$, $C(78) = 5$ e $C(118) = 8$.

Si scopre che 154 è il valore minimo di $n$ per cui $C(n) = 10$.

Trova il valore minimo di $n$ per cui $C(n) = 1000$.
