# --description--

JavaScript також має можливість порівнювати типи та значення за допомогою оператора строгої рівності `===`. Цей оператор перевіряє, чи два значення рівні і належать до одного типу. Наприклад:

```javascript
let x = 5;
let y = "5";

let result = x === y;

console.log(result); // false
```

У цьому прикладі змінна `result` міститиме `false`, тому що `x` — це число, а `y` — рядок. Але якщо ви використаєте оператор `==`, результат буде `true`, тому що JavaScript перетворить рядок на число і порівняє значення.

Цей оператор називається оператором строгої рівності, бо він перевіряє рівність і за значенням, і за типом. Часто рекомендують використовувати оператор `===`, щоб уникнути несподіваних результатів при порівнянні значень.

Існує також оператор строгої нерівності `!==`, який перевіряє, чи два значення не рівні і належать до одного типу.

Існує багато способів використання оператора строгої рівності, і важливо розуміти, як він працює.
