# --interactive--

연락처 주소 요소는 웹 페이지의 섹션에 대한 연락처 정보를 나타내는 데 사용됩니다. `address` 요소는 다용도로, 비즈니스 페이지, 저자 페이지, 개인 사이트 등 다양한 용도로 사용할 수 있습니다.

웹사이트의 연락처 섹션을 구성할 때는 `address` 같은 일반 요소보다 의미론적 `div` 요소를 사용하는 것이 좋습니다.

다음은 회사 연락처 페이지에 `address` 요소를 사용하는 예시입니다:

:::interactive_editor

```html
<address>
  <h2>Company Name</h2>
  <p>
    1234 Elm Street<br />
    Springfield, IL 62701<br />
    United States
  </p>
  <p>Phone: <a href="tel:+15555555555">+1 (555) 555-5555</a></p>
  <p>Email: <a href="mailto:contact@company.com">contact@company.com</a></p>
</address>
```

:::

이 예시에는 회사 이름, 실제 주소, 전화번호, 이메일 정보가 포함되어 있습니다. 실제 주소의 경우 줄 바꿈 요소인 `br`를 사용해 거리 이름, 도시, 국가 사이를 구분합니다.

전화번호에는 `href` 값이 설정된 앵커 요소가 있습니다. `tel:+` 속성 안의 `href` 값은 특정 장치에서 전화를 걸 수 있도록 클릭 가능한 링크를 만듭니다.

이메일 주소에는 또 다른 앵커 요소가 사용되며 `href` 값이 `mailto` 링크로 설정되어 있습니다. `mailto` 링크는 HTML 문서에서 사용자가 선호하는 이메일 클라이언트로 새 이메일을 열 수 있게 합니다.

`mailto` 링크를 사용할 때 단점 중 하나는 사용자가 이를 스팸으로 인식하는 경우가 많다는 점입니다. 안타깝게도 많은 스패머가 이 옵션을 이용해 사용자에게 이메일을 보냅니다. 그러니 사용할 때 이 점을 염두에 두세요.
