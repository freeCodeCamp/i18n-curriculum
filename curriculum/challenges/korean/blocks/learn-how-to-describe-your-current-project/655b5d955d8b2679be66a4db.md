# --description--

`What`는 특정 정보를 묻는 데 자주 사용됩니다. 이것은 누군가가 알고 싶어 하는 사물, 행동 또는 세부 사항에 대해 질문할 수 있습니다. 예를 들어:

- `What time is the meeting?` - 회의의 정확한 시간을 묻습니다.

- `What do you usually have for lunch?` - 누군가의 일상에 관한 세부 사항을 묻습니다.

그래서 `what`는 구체적인 무언가를 알고 싶을 때 명확한 답변을 얻는 데 도움이 됩니다.
