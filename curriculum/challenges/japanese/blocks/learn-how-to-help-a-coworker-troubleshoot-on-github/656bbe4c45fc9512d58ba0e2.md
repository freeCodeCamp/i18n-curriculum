# --description--

単語`earlier`は、現在の瞬間や話している時間より前の時間について話すときに使います。これは`before now`や`some time ago`と言うのと同じようなものです。

例えば、`If you finished your work earlier, you can relax now`は、もし少し前に仕事を終えたなら、今はリラックスする時間があるという意味です。
