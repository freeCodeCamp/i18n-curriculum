# --description--

このタスクでは、一人称単数の正しい動詞の形を見分ける方法を引き続き学びます。

動詞 `trabajar` は話者によって変わります。例えば：

- `Trabajo` - 私は働きます。一人称（`yo`）
- `Trabaja` - 彼/彼女/それは働きます。三人称（`él` / `ella` / `la empresa`）
- `Trabajas` - あなたは働きます。二人称（`tú`）

# --explanation--

正しい形は `trabajo` です。

語尾の **`-o`** は一人称単数（`yo`）を示します。

もし三人称なら、動詞は `trabaja` になります。
