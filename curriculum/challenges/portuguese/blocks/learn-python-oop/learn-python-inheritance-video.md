O que é herança em programação orientada a objetos?

## --answers--

Isso é quando uma classe reutiliza código de uma função.

Isso acontece quando uma classe cria uma cópia de outra classe.

---

Isso acontece quando uma classe herda propriedades e comportamentos de outra classe.

---

É quando uma classe oculta seus métodos e variáveis de outras classes.
