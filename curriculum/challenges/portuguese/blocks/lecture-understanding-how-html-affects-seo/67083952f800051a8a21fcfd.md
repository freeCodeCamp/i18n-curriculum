# --description--

SEO, ou otimização para mecanismos de busca, é uma prática que otimiza páginas web para que fiquem mais visíveis e tenham melhor posicionamento nos mecanismos de busca. Uma forma de melhorar o SEO do seu site é fornecer uma descrição curta para a página web usando o elemento `meta`. Aqui está um exemplo de como usar o elemento meta para definir a descrição de uma página de jardinagem:

```html
<meta
  name="description"
  content="Discover expert tips and techniques for gardening in small spaces, choosing the right plants, and maintaining a thriving garden."
/>
```

Definir o atributo `name` como `description` garante que navegadores, mecanismos de busca e outras ferramentas web interpretem corretamente esses metadados. O atributo `content` é onde você colocará sua descrição. Recomenda-se que suas descrições sejam curtas e concisas. Isso porque os mecanismos de busca frequentemente cortam a descrição com base no layout da página de resultados.

Semelhante a outros tipos de elementos `meta`, a descrição `meta` não será visível na própria página web. Um lugar onde a descrição da página pode ser encontrada é no trecho da página de resultados do motor de busca. Aqui estão alguns exemplos de trechos de resultados de página para o subreddit do freeCodeCamp e repositórios do GitHub:

```md
r/freeCodeCamp: This is the official subreddit for the freeCodeCamp.org community. Learn to
code for free together with millions of other people...
```

```md
Our full-stack web development and machine learning curriculum is completely free and self-
paced. We have thousands of interactive coding challenges to help you...
```

Nos exemplos, cada uma das descrições das páginas está localizada logo abaixo dos links do site. Em poucos segundos, os usuários conseguem ter uma noção geral do que a página trata e decidem clicar nos links para obter mais informações.

Embora as descrições `meta` não afetem diretamente o posicionamento de um site nos mecanismos de busca, ter uma descrição forte pode resultar em mais tráfego para seu site.
