## Nozioni di base su CSS Grid

- **Definizione**: CSS Grid è un sistema di layout bidimensionale usato per creare layout complessi nelle pagine web. Le griglie hanno righe e colonne con spazi tra di esse. Per definire un layout a griglia, imposteresti la proprietà `display` su `grid`.

```css
.container {
  display: grid;
}
```

- **L’unità `fr` (frazionaria)**: Questa unità rappresenta una frazione dello spazio all’interno del contenitore griglia. Puoi usare l’unità `fr` per creare griglie flessibili.
- **Creare spazi tra le tracce**: Ci sono tre modi per creare spazi tra le tracce in CSS grid. Puoi usare la proprietà `column-gap` per creare spazi tra le colonne. Puoi usare la proprietà `row-gap` per creare spazi tra le righe. Oppure puoi usare la proprietà abbreviata `gap` per creare spazi sia tra righe sia tra colonne.
- **`grid-template-columns`**: Questa proprietà serve per impostare i nomi delle linee e le dimensioni delle colonne della griglia.

```css
.container {
  display: grid;
  width: 100%;
  grid-template-columns: 30px 1fr;
}
```

- **`grid-template-rows`**: Questa proprietà serve per impostare i nomi delle linee e le dimensioni delle righe della griglia.
- **`grid-auto-flow`**: Questa proprietà determina come gli elementi posizionati automaticamente si adattano alla griglia.

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-flow: column;
}
```

- **`grid-auto-columns`**: Questa proprietà serve per impostare la dimensione delle colonne create implicitamente.

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-columns: auto;
}
```

- **`place-items`**: Questa proprietà serve per allineare gli elementi sia nella direzione blocco sia in quella in linea.
- **`align-items`**: Questa proprietà serve per impostare l’allineamento degli elementi all’interno di un contenitore griglia.
- **Funzione `repeat()`**: Questa funzione serve per ripetere sezioni nella lista delle tracce. Invece di scrivere `grid-template-columns: 1fr 1fr 1fr;` puoi usare la funzione `repeat()`.

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}
```

- **Griglia esplicita**: Puoi specificare il numero di linee o tracce usando le proprietà `grid-template-columns` o `grid-template-rows`.
- **Griglia implicita**: Quando gli elementi sono posizionati fuori dalla griglia, vengono create automaticamente righe e colonne per quegli elementi esterni.
- **Funzione `minmax()`**: Questa funzione serve per impostare le dimensioni minime e massime di una traccia.

```css
.container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-auto-rows: minmax(150px, auto);
}
```

- **Posizionamento basato sulle linee**: Tutte le griglie hanno linee. Per specificare dove inizia un elemento su una linea, puoi usare le proprietà `grid-column-start` e `grid-row-start`. Per specificare dove finisce l’elemento sulla linea, puoi usare le proprietà `grid-column-end` e `grid-row-end`. Puoi anche scegliere di usare le proprietà abbreviate `grid-column` o `grid-row`.

Ecco un esempio di come usare la proprietà `grid-column` per far estendere un elemento su tutte le colonne.

```css
.element {
  grid-column: 1 / -1;
}
```

- **`grid-template-areas`**: Questa proprietà serve per assegnare un nome agli elementi che vuoi posizionare sulla griglia.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="header">Header</div>
  <div class="sidebar">Sidebar</div>
  <div class="main">Main Content</div>
  <div class="footer">Footer</div>
</div>

```

```css
.container {
  display: grid;
  grid-template-columns: 200px 1fr; 
  grid-template-rows: auto 1fr auto; 
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer"; 
  gap: 20px; 
}

.header {
  grid-area: header; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}

.sidebar {
  grid-area: sidebar;
  background-color: #f4f4f4;
  padding: 10px;
}

.main {
  grid-area: main; 
  background-color: #e0e0e0;
  padding: 10px;
}

.footer {
  grid-area: footer; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}
```

:::

## Eseguire il debug del CSS

- **DevTools (strumenti per sviluppatori)**: I DevTools ti permettono di ispezionare e modificare il tuo CSS in tempo reale. Il pannello Stili mostra tutte le regole CSS applicate all’elemento selezionato, inclusi gli stili ereditati. Puoi attivare o disattivare singole proprietà, modificare i valori e persino aggiungere nuove regole direttamente nel browser. Questo feedback immediato è molto utile per sperimentare con stili diversi senza modificare il codice sorgente.
- **Validator CSS**: I validator servono a controllare il tuo CSS rispetto alle specifiche ufficiali e segnalano eventuali errori o avvisi. Un validator popolare che puoi usare è il W3C CSS Validator.
- **Debugging di design responsivi**: I DevTools hanno un’opzione che ti permette di simulare come appare il tuo sito su varie dimensioni di schermo e dispositivi. Questo può aiutarti a identificare problemi con i breakpoint o stili che non si adattano bene a diverse dimensioni di viewport.
