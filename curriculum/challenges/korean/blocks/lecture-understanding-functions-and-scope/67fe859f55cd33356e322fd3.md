# --description--

함수는 호출할 때 실행되는 재사용 가능한 코드 조각입니다. 많은 프로그래밍 언어는 시작하기 쉽게 해주는 내장 함수를 제공합니다. 파이썬도 예외가 아니며, 이전 수업에서 `print()` 같은 내장 함수를 이미 다뤘습니다.

또 다른 유용한 내장 함수는 `input()`로, 사용자에게 입력을 요청할 수 있게 해줍니다:

```python
name = input('What is your name?') # User types "Kolade" and presses Enter  
print('Hello', name) # Output: Hello Kolade
```

반면에 `int()`는 숫자, 불리언, 그리고 숫자 문자열을 정수로 변환합니다:

```python
print(int(3.14)) # 3
print(int('42')) # 42
print(int(True)) # 1
print(int(False)) # 0 
```

자신만의 사용자 정의 함수도 작성할 수 있습니다. 그렇게 하려면 `def` 키워드 다음에 함수 이름, 소괄호 한 쌍, 그리고 콜론을 씁니다. 그리고 새 줄에 함수가 실행할 코드를 작성하세요. 함수가 실행하는 코드를 함수의 본문이라고도 합니다.

다음은 `hello`라는 이름의 사용자 정의 함수 예제로, 터미널에 `Hello World` 문자열을 출력합니다:

```python
def hello():
    print('Hello World')
```

함수를 실행하려면 이름 뒤에 소괄호 한 쌍을 붙여 호출해야 합니다:

```python
hello() # Hello World
```

`print('Hello World')` 앞의 들여쓰기에 주목하세요. 이전 수업에서 기억하듯이, 파이썬은 어떤 문장 그룹이 함께 속하는지 판단할 때 들여쓰기를 사용합니다. 이런 문장 그룹을 코드 블록이라고 합니다.

다음은 두 숫자의 합을 터미널에 출력하는 또 다른 간단한 함수입니다:

```python
def calculate_sum(a, b):
    print(a + b)
```

함수 `calculate_sum`의 소괄호 안에 `a`과 `b`가 쉼표로 구분되어 있는 것을 볼 수 있습니다. 이것을 매개변수라고 합니다. 매개변수는 함수 호출 시 전달하는 값을 위한 "자리" 역할을 하는 자리 표시자 변수라고 생각하세요.

매개변수를 사용하려면 "인수"를 전달해야 합니다. 인수는 함수를 호출할 때 전달하는 값입니다.

다음은 `calculate_sum` 함수를 호출해 `3`과 `1` 숫자를 더하는 방법입니다:

```python
calculate_sum(3, 1) # 4
```

함수를 올바른 인수 개수 없이 호출하면 `TypeError` 오류가 발생합니다:

```python
calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

함수는 또한 함수 실행을 종료하고 값을 반환하기 위해 특별한 `return` 키워드를 사용합니다. 명시적으로 `return`를 사용하지 않으면, 파이썬은 기본적으로 `None`를 반환합니다.

예를 들어:

```python
def calculate_sum(a, b):
    print(a + b)

my_sum = calculate_sum(3, 1) # 4
print(my_sum) # None
```

`calculate_sum` 함수가 `a`과 `b`의 합을 출력하지만 명시적으로 아무것도 반환하지 않는 것을 볼 수 있습니다. 그래서 그 결과를 `my_sum`에 할당하면 실제 값은 `None`입니다. 이를 고치려면 `return` 키워드를 사용해 결과를 반환할 수 있습니다:

```python
def calculate_sum(a, b):
    return a + b

my_sum = calculate_sum(3, 1)
print(my_sum) # 4
```

이제 `calculate_sum`은 `a`와 `b`의 합을 반환하며, 그 값이 `my_sum`에 저장됩니다.
