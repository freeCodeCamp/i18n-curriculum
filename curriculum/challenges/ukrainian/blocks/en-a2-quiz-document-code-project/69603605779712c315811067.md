# --description--

Цей тест перевіряє ваше розуміння написання повідомлень, пояснення ідей і дотримання конвенцій.

Щоб пройти тест, потрібно правильно відповісти щонайменше на 9 із 10 питань нижче.

Прочитайте кожне питання і виберіть правильну відповідь. Для кожного питання є лише одна правильна відповідь.

Для доступності доступні транскрипти для учнів з порушеннями слуху; рекомендуємо використовувати їх лише за потреби.
