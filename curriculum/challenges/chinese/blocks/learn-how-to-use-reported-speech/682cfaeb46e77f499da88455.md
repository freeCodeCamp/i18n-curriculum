# --explanation--

在**转述语气**中，`to suggest` 用于传达某人给出的建议或想法。例如：  

- 直接引语：`I think we should update our firewall.`

- 转述：`Jake suggested updating the firewall.`

这表明有人给出了建议或可能的方案，但没有使用他们的确切用词。
