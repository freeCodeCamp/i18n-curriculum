# --description--

`Would you like...?` si usa spesso quando si propone o suggerisce un'attività. Per esempio:

`Would you like to test the new app with me?` - È un invito educato a collaborare nel testare.

Quando Linda dice `Would you like to go cycling with me one of these days?`, è simile a dire "Ehi, andiamo a fare un giro in bici insieme."
