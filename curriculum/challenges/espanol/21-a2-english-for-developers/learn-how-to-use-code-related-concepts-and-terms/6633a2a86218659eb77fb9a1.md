# --description--

El verbo modal `can` se usa para indicar posibilidad o habilidad. Por ejemplo, en la oración `You can see the stars at night`, `can` muestra que es posible ver las estrellas durante la noche.

`Change` significa hacer o volverse diferente. En la vida cotidiana, podrías `change` tu ropa después del trabajo, indicando que cambias de ropa de trabajo a ropa casual.

`Stay the same` significa permanecer sin cambios o constante. Por ejemplo, `The rules of the game stay the same every year.` Esto significa que las reglas no cambian de un año a otro.
