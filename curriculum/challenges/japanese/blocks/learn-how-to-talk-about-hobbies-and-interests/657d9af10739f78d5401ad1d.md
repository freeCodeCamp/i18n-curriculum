# --description--

`As` は、何かの役割、機能、または様式を説明するためによく使われます。例えば：

`She works as a web developer.` - ここで、`as` は彼女の役割を説明しています。

活動の文脈では、`as` は何かが特定の方法やスタイルで行われていること、またはプロとしてではなく楽しみのために行われていることを示す場合があります。ボブの文、`It started as a hobby` のようにです。
