# --description--

每个三角形都有一个穿过三个顶点的外接圆。 设S（n）是半径不超过n的所有这些三角形的外接圆的半径之和。

设 $S(n)$ 为所有外接圆半径不超过 $n$ 的此类三角形的外接圆半径之和。

$S(100) = 4\,950$ 和 $S(1\,200) = 1\,653\,605$。

查找 $S({10}^7)$。
