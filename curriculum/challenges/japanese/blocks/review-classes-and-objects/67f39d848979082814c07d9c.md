## Pythonのクラスとオブジェクト

- **クラス定義**: クラスはオブジェクトを作成するための設計図です。属性やメソッドを通じてオブジェクトの振る舞いを定義します。以下はPythonでの基本的なクラス定義の例です。

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f'{self.name.upper()} says woof woof!')
```

- **オブジェクトの作成**: オブジェクトはクラスのインスタンスです。必要な引数を指定してクラスを呼び出すことで作成されます。

```python
dog1 = Dog('Jack', 3)
dog2 = Dog('Thatcher', 5)

dog1.bark()  # JACK says woof woof!
dog2.bark()  # THATCHER says woof woof!
```

- **オブジェクトのメソッド呼び出し**: オブジェクトのメソッドを呼び出して、動作を実行したり情報を取得したりできます。

```python
object_name1.method_name()
object_name2.method_name()
```

- **クラスとオブジェクトの違い**: クラスは再利用可能なテンプレートであり、オブジェクトはそのクラスの具体的なデータを持つインスタンスです。

## 属性

- **インスタンス属性**: `__init__()`内で`self`を使って定義され、各オブジェクトごとに固有です。
- **クラス属性**: クラス内に直接定義され、すべてのインスタンスで共有されます。

```python
class Dog:
    species = 'French Bulldog'  # Class attribute

    def __init__(self, name):
        self.name = name  # Instance attribute

print(Dog.species) # French Bulldog

jack = Dog('Jack')
print(jack.name)     # Jack
print(jack.species)  # French Bulldog
```

## メソッド

- **メソッド**: クラス内で定義され、オブジェクトの属性に対して操作を行う関数です。

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
print(my_car_1.describe())  # This car is a red Tesla Model S
```

- **メソッドのアクセス**: ドット記法を使ってオブジェクトのメソッドを呼び出します。以下は2つの異なるcarオブジェクトで`describe`メソッドを呼び出す例です。

```python
class Car:
    def __init__(self, color, model):
        self.color = color  
        self.model = model  

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
my_car_2 = Car('green', 'Lamborghini Revuelto')

print(my_car_1.describe()) # Calling method using the dot notation

print(my_car_2.describe()) # Calling method using the dot notation
```

## ダンダー（マジック）メソッド

- **定義**: 両端がダブルアンダースコアで囲まれた特別なメソッド（例：`__init__`、`__len__`、`__str__`、`__eq__`）です。Pythonは組み込みの操作で内部的にこれらを使います。

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **ダンダーメソッドの間接的な呼び出し**: ダンダーメソッドを直接呼び出す必要はありません。特定の操作が行われるとPythonが自動的に呼び出します。これらの操作には以下が含まれます：

  - **加算、減算、乗算、除算などの算術演算**。加算には`__add__()`、減算には`__sub__()`、乗算には`__mul__()`、除算には`__truediv__()`が呼び出されます。
        
  - **連結、繰り返し、フォーマット、文字列変換などの文字列操作**。連結には`__add__()`、繰り返しには`__mul__()`、フォーマットには`__format__()`、文字列変換には`__str__()`や`__repr__()`などが呼び出されます。
        
  - **等価、不等、小なり、大なりなどの比較操作**。等価チェックには`__eq__()`、小なりには`__lt__()`、大なりには`__gt__()`などが呼び出されます。
        
  - **イテレーション操作として、オブジェクトをイテラブルにし、アイテムを順に取得する操作**。イテレータを返すには`__iter__()`、次のアイテムを取得するには`__next__()`が呼び出されます。

## 実例：ショッピングカート

- **ダンダーメソッドを持つCartクラス**：追加、削除、イテレーション、内容の確認を組み込みの振る舞いで実現します。

```python
class Cart:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

    def remove(self, item):
        if item in self.items:
            self.items.remove(item)
        else:
            print(f'{item} is not in cart')

    def list_items(self):
        return self.items

    def __len__(self):
        return len(self.items)

    def __getitem__(self, index):
        return self.items[index]

    def __contains__(self, item):
        return item in self.items

    def __iter__(self):
        return iter(self.items)

cart = Cart()
cart.add('Laptop')
print(len(cart))        # 1
print('Laptop' in cart) # True
```
