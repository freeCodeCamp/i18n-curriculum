# --questions--

## --text--

¿Qué opción describe correctamente `front end` o `back end`?

## --answers--

El diseño y la estructura de un sitio web son front end.

---

El almacenamiento de datos y las operaciones del lado del servidor son front end.

### --feedback--

El front end se encarga de lo que los usuarios ven e interactúan, no del almacenamiento de datos ni de las tareas del lado del servidor.

---

Las funciones orientadas al usuario como botones, formularios y menús de navegación están en el back end.

### --feedback--

El back end maneja datos y procesos del servidor, no funciones visibles para el usuario como botones y formularios.

---

Bases de datos, APIs y mantenimiento del servidor son front end.

### --feedback--

El front end está enfocado en los aspectos visuales e interactivos de un sitio web, no en bases de datos ni en el mantenimiento del servidor.

# --explanation--

El `front end` es todo con lo que los usuarios interactúan en un sitio web, como el diseño, la estructura y los botones. El `back end` gestiona tareas del lado del servidor como el almacenamiento de datos, las APIs y el mantenimiento del servidor.
