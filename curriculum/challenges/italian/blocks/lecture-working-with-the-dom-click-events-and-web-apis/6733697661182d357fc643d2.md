# --interactive--

L’API `Canvas` è uno strumento potente che ti permette di manipolare la grafica direttamente nel tuo file JavaScript. Tutto inizia con un elemento `canvas` in HTML. Questo elemento funge da superficie di disegno che puoi manipolare usando i metodi e le proprietà dell’istanza dell’API `Canvas`.

L’API `Canvas` fornisce tutto ciò che ti serve per creare immagini sorprendenti, incluse forme, testo, animazioni e persino giochi complessi. Ha interfacce come `HTMLCanvasElement`, `CanvasRenderingContext2D`, `CanvasGradient`, `CanvasPattern`, `TextMetrics` che offrono metodi e proprietà che puoi usare per creare grafica nel tuo file JavaScript.

Vediamo come funziona l’API `Canvas` così puoi iniziare a creare grafica con JavaScript.

Per prima cosa, devi creare un elemento `canvas` nel tuo file HTML:

```html
<canvas id="my-canvas"></canvas>
```

L’elemento `canvas` è rappresentato dall’interfaccia `HTMLCanvasElement`, che fornisce metodi e proprietà per manipolarlo. Inoltre, puoi usare metodi e proprietà di altre interfacce nell’API `Canvas`.

Puoi dare al tuo `canvas` un `width` e un `height` all’interno dell’HTML:

```html
<canvas id="my-canvas" width="400" height="400"></canvas>
```

Oppure puoi usare le proprietà `width` e `height` dell’interfaccia `HTMLCanvasElement`:

```js
const canvas = document.getElementById("my-canvas");
canvas.width = 400;
canvas.height = 400;
```

Per ora non vedi ancora nulla sullo schermo. Dopo aver creato il tuo elemento `canvas`, il passo successivo è consultare il contesto di disegno del canvas con il metodo `getContext()` dell’interfaccia `HTMLCanvasElement`.

Il contesto più comune è `2d`, che ti permette di disegnare in due dimensioni:

```js
const canvas = document.getElementById("my-canvas");
const ctx = canvas.getContext('2d');
```

Se stampi la variabile `ctx` nella console, vedrai i metodi e le proprietà di `CanvasRenderingContext2D` che puoi usare per creare forme, colori, linee e altro, insieme ai loro valori predefiniti:

```js
console.log(ctx);
```

Una volta ottenuto il contesto 2D, puoi iniziare a disegnare sul canvas.

L’API `Canvas` offre diversi metodi e proprietà per disegnare forme, linee e testo. Uno di questi è la proprietà `fillStyle`, che puoi combinare con il metodo `fillRect()` per disegnare un rettangolo o un quadrato:

:::interactive_editor

```html
<html>
  <head>
  </head>
  <body>
    <canvas id="my-canvas" width="400" height="400"></canvas>
    <script src="index.js"></script>
  </body>
</html>
```

```js
const canvas = document.getElementById("my-canvas");

const ctx = canvas.getContext("2d");

// Set the background color
ctx.fillStyle = "crimson";

// Draw a rectangle
ctx.fillRect(1, 1, 150, 100);
```

:::

`fillRect` prende 4 valori numerici che rappresentano rispettivamente l’asse x, l’asse y, la larghezza e l’altezza.

Ora c’è qualcosa sullo schermo. Puoi anche disegnare testo o persino creare un’animazione. Ecco un canvas per rappresentare del testo:

```html
<canvas id="my-text-canvas" width="300" height="70"></canvas>
```

Per disegnare infine il testo, passa il testo al metodo `fillText()` come primo argomento, seguito dai valori per l’asse x e l’asse y:

:::interactive_editor

```html
<html>
  <head>
  </head>
  <body>
    <canvas id="my-text-canvas" width="300" height="70"></canvas>
    <script src="index.js"></script>
  </body>
</html>
```

```js
const textCanvas = document.getElementById("my-text-canvas");

const textCanvasCtx = textCanvas.getContext("2d");

// Set font family and size
textCanvasCtx.font = "30px Arial";

// Set text color
textCanvasCtx.fillStyle = "crimson";

// Draw the text
textCanvasCtx.fillText("Hello HTML Canvas!", 1, 50);
```

:::

Il risultato nel browser sarà il testo rosso `Hello HTML Canvas!`.

C’è molto altro che puoi fare con l’API `Canvas`. Per esempio, puoi combinarla con `requestAnimationFrame` per creare animazioni personalizzate, visualizzazioni, giochi e altro.
