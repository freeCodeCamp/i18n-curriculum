# --interactive--

ID 选择器是 CSS 中最强大的选择器之一，允许开发者将样式应用到具有唯一标识符的特定元素。

这使它们在针对需要独特样式的单个元素时非常有效。

ID 选择器由一个井号符号（`#`）后跟 ID 名称定义。它们在一个超文本标记语言文档中应是唯一的，意味着不应有两个元素共享相同的 ID。

这是一个使用 ID 选择器的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="unique">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#unique {
  color: purple;
}
```

:::

在此示例中，ID 为 `unique` 的元素的文本颜色将设置为 `purple`。

ID 选择器具有非常高的特异性，高于类型选择器和类选择器，但低于内联样式。ID 选择器的特异性值为 `(0, 1, 0, 0)`。

这意味着 ID 选择器可以覆写类选择器和类型选择器，但可以被内联样式覆写。
