# --description--

编程中的 `constant` 类似于变量，但一旦赋值，该值就不能更改。`Constants` 用于当你知道某些值在整个程序中不需要修改时。它们通常用大写字母书写，以区别于变量。

例如，`const PI = 3.14;` 在许多编程语言中，`const` 用于声明一个常量。这里，`PI` 是一个赋值为 `3.14` 的 `constant`。
