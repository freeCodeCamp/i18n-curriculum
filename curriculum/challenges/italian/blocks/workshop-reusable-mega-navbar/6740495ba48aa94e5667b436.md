# --description--

In questo workshop, eserciterai l'uso dei componenti funzionali React costruendo una barra di navigazione riutilizzabile.

Tutto il CSS è già fornito, così puoi concentrarti sulla creazione della barra di navigazione.

Inizia definendo un componente funzionale `Navbar`. Nel passaggio successivo, inizierai a costruire il componente usando JSX.

Assicurati di usare un export nominato quando esporti il componente, perché verrà importato nel file HTML.

Ecco un esempio di export nominato:

```js
export function multiply(a, b) {
  return a * b;
}
```

Inoltre, i componenti funzionali React devono iniziare con una lettera maiuscola perché è un modo per distinguerli dagli elementi HTML normali.

# --hints--

assert.match(code, /export\s+(const|function)\s+Navbar\s*(=\s*)?\(\)\s*(=>\s*)?\{\s*/);
