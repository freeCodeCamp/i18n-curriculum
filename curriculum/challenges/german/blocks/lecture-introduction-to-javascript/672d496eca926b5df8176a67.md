# --interactive--

In JavaScript ist ein Datentyp die Art von Wert, die Sie speichern, wie eine Zahl oder ein Textstück.

Eine Variable ist ein benannter Container, der einen Wert eines bestimmten Datentyps speichert, sodass Sie ihn im gesamten Code referenzieren und bearbeiten können.

Vielleicht erinnern Sie sich aus dem Mathematikunterricht an Variablen wie diese:

```md
x = 2
y = 4

x + y
```

Sie konnten Variablen wie `x` und `y` erstellen und diese dann im Programm verwenden und mathematische Operationen wie Addition durchführen. Das ist ein ähnliches Konzept in der Programmierung. Sie können eigene Variablennamen erstellen und ihnen Werte zuweisen. Diese Werte haben unterschiedliche Datentypen.

Datentypen helfen dem Programm zu verstehen, mit welcher Art von Daten es arbeitet, ob es sich um eine Zahl, Text oder etwas anderes handelt.

JavaScript hat mehrere grundlegende Datentypen, die Sie in Ihren Programmen verwenden werden. Wir werden jeden Datentyp in zukünftigen Lektionen ausführlicher betrachten. Für den Moment hier eine kurze Einführung in die verschiedenen Datentypen in JavaScript.

Der erste Datentyp, den wir uns ansehen, ist der `Number`-Typ.

Ein `Number` stellt sowohl ganze Zahlen als auch Fließkommazahlen dar. Beispiele für ganze Zahlen sind `7`, `19` und `90`.

**HINWEIS**: `console.log()` ist eine Funktion, die Informationen in der Konsole ausgibt, einem Teil Ihres Webbrowsers, der zum Debuggen von Code verwendet wird. Mehr über `console.log()` lernen Sie in zukünftigen Lektionen. Außerdem werden die Symbole `//` verwendet, um Kommentare in Ihrem Code hinzuzufügen. Kommentare sind Notizen für Sie selbst oder andere Programmierende, die beim Ausführen des Codes ignoriert werden.

Aktivieren Sie den interaktiven Editor und versuchen Sie, einige der ganzen Zahlen zu ändern, um zu sehen, wie sich die Ausgabe in der Konsole aktualisiert.

:::interactive_editor

```js
// Examples of integers
console.log(3);
console.log(5);
console.log(-67);
```

:::

Eine Fließkommazahl ist eine Zahl mit Dezimalpunkt. Beispiele für Fließkommazahlen sind `3.14` und `5.2`.

:::interactive_editor

```js
// Examples of floating point numbers
console.log(3.14);
console.log(7.2);
console.log(-14.5);
```

:::

Der nächste Datentyp ist ein `String`.

Ein `String` ist eine Folge von Zeichen oder Text, der in Anführungszeichen eingeschlossen ist. Hier ist ein Beispiel für einen String mit doppelten Anführungszeichen:

:::interactive_editor

```js
console.log("I love to code!");
```

:::

Hier ist ein Beispiel mit einfachen Anführungszeichen:

:::interactive_editor

```js
console.log('I love to code!');
```

:::

Oft verwenden Sie Strings, um Namen, Bezeichnungen, Warnmeldungen usw. darzustellen.

Ein weiterer in JavaScript verwendeter Datentyp ist der `Boolean`-Typ.

Ein `Boolean` stellt einen von zwei Werten dar: `true` oder `false`. Zum Beispiel könnte ein Programm prüfen, ob ein Benutzer angemeldet ist (`true`) oder nicht (`false`) und die Seite entsprechend ändern. Wenn der Benutzer angemeldet ist, möchten Sie ihm wahrscheinlich die Dashboard-Seite zeigen. Andernfalls zeigen Sie ihm die Anmeldeseite.

Die nächsten beiden in JavaScript verwendeten Datentypen sind `undefined` und `null`.

`undefined` bedeutet, dass eine Variable deklariert wurde, aber noch kein Wert zugewiesen wurde. Mehr dazu lernen Sie in der nächsten Lektion.

`null` bedeutet, dass die Variable absichtlich auf „nichts“ gesetzt wurde und keinen Wert enthält. Wir werden in zukünftigen Lektionen genauer darauf eingehen, wie das funktioniert.

Die letzten drei Datentypen sind komplexer in ihrer Natur. Dabei handelt es sich um `Object`, `Symbol` und `BigInt`.

Ein `Object` ist eine Sammlung von Schlüssel-Wert-Paaren.

```js
{
  name: "Alice",
  age: 30
};
```

Objekte sind ideal, um zusammengehörige Informationen zu gruppieren. Sie werden in einem zukünftigen Modul mehr darüber lernen, wie man mit Objekten arbeitet.

Ein `Symbol` ist ein spezieller Werttyp in JavaScript, der immer einzigartig ist und nicht verändert werden kann. Er wird oft verwendet, um eindeutige Bezeichnungen oder Identifikatoren für Eigenschaften zu erstellen:

```js
Symbol('mySymbol');
```

`BigInt` wird für sehr große Zahlen verwendet, die das Limit des `Number`-Typs überschreiten:

```js
1234567890123456789012345678901234567890n;
```

In diesem Beispiel erstellen wir einen `BigInt`, indem wir `n` am Ende einer sehr großen Zahl hinzufügen.

`Symbol` und `BigInt` sind zwei weniger häufig verwendete Typen, aber es ist trotzdem wichtig, sie zu kennen.

Das Verständnis dieser Datentypen hilft Ihnen, verschiedene Arten von Daten in Ihren Programmen zu handhaben und zu verarbeiten, da jeder Typ seine eigenen Eigenschaften und Verhaltensweisen hat.
