# --explanation--

`Could` na `might` ni vitenzi vya modal vinavyotumika kuzungumzia uwezekano.

`Could` ina maana kwamba jambo linawezekana lakini halijathibitishwa. Kwa mfano:

`This update could make the app run faster.` - Sasisho lina uwezo wa kuboresha kasi ya programu.

`Might` pia linaonyesha uwezekano lakini mara nyingi linaashiria nafasi ndogo zaidi kuliko `could`. Kwa mfano:

`The new feature might improve the user experience.` - Inawezekana kwamba kipengele kitaboresha uzoefu.
