# --description--

Esta tarefa ajuda você a praticar a escolha da forma verbal correta para um departamento singular.

# --instructions--

# --questions--

## --text--

Qual frase usa gramaticalmente o verbo correto para completar esta frase?

## --answers--

`El departamento de ingeniería desarrollan el software`.

### --feedback--

`Desarrollan` é a forma para `ellos/ellas`, mas `el departamento` é singular.

---

`El departamento de ingeniería desarrollar el software`.

### --feedback--

`Desarrollar` é a forma infinitiva, então não completa a frase corretamente aqui.

---

`El departamento de ingeniería desarrolla el software`.

---

`El departamento de ingeniería desarrollas el software`.

### --feedback--

`Desarrollas` é a forma para `tú`, não para `el departamento`.

## --video-solution--

3

# --explanation--

A resposta correta é `El departamento de ingeniería desarrolla el software.` 

O sujeito `el departamento` é singular, então precisa da forma verbal singular `desarrolla`.
