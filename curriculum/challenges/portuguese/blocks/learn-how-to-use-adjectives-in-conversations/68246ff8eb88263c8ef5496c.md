# --explanation--

`Made sure` significa tomar uma ação para confirmar que algo é verdadeiro ou está funcionando corretamente.Por exemplo:  

`They made sure the software was ready before launch.` – Eles verificaram se tudo funcionava corretamente.

`As` é usado para descrever algo acontecendo ao mesmo tempo que outra ação.Por exemplo:  

O sistema adiciona mais recursos conforme a demanda aumenta. – Ele se ajusta enquanto mais pessoas começam a usá-lo.
