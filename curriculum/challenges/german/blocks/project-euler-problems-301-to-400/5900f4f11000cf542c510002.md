# --description--

Betrachten Sie alle Gitterpunkte ($a$, $b$, $c$) mit $0 ≤ a, b, c ≤ N$.

Vom Ursprung $O(0, 0, 0)$ werden alle Geraden zu den anderen Gitterpunkten gezogen. Sei $D(N)$ die Anzahl der unterschiedlichen solchen Geraden.

Es ist gegeben, dass $D(1\\,000\\,000) = 831\\,909\\,254\\,469\\,114\\,121$.

Finden Sie $D({10}^{10})$. Geben Sie als Antwort die ersten neun Ziffern gefolgt von den letzten neun Ziffern an.
