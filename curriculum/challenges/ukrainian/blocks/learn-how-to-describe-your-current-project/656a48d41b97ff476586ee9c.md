# --description--

У цьому контексті бути `out` означає бути доступним або опублікованим. Наприклад:

`The new book is out next week.` — це означає, що книга буде доступна тоді.

`Looking forward to` + дієслово, що закінчується на `-ing`, — це загальний спосіб виразити захоплення майбутньою подією. Наприклад:

- `I'm looking forward to meeting you.` — ви раді зустрічі з кимось.

- `She's looking forward to traveling next month.` — вона радіє подорожі.
