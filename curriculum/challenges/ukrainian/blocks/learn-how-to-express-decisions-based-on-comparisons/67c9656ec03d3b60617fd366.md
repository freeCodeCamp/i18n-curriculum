# --explanation--

`Performance` означає, наскільки добре щось функціонує, особливо з точки зору швидкості та ефективності. Наприклад:  

`This computer has better performance because it processes data quickly.` - Тут `performance` означає, наскільки ефективно працює комп’ютер.  

`Faster` — це порівняльна форма `fast`, що означає, що щось рухається або працює швидше за інше. Наприклад:  

`This new software runs faster than the old version.` - Тут `faster` означає, що нове програмне забезпечення працює швидше.
