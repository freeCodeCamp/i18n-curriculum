### --feedback--

これら3つの単語は、過去にやっておけばよかったのに実行しなかったことについて話すときに使います。3つ目の単語は`Past Participle`の`run`です。

# --explanation--

`Should have` + `Past Participle`は、過去の後悔や助言を表すときに使います。例えば：

- `You should have run the program on a different machine to check for compatibility.` - 別のマシンでテストするのは役に立ったはずですが、実行されませんでした。

- `I should have run the tests before deploying the app.` - テストを実行するのが正しいことでしたが、その時は行われませんでした。

この場合、ソフィーはマークにテストスイートを実行すべきだったと言っています。つまり、それは重要でしたが、彼は実行しませんでした。
