# --description--

Vichaguzi vya darasa la vitu vitachagua vipengele vyote vyenye `class`, ambayo ni sifa ya kipengele unayoweka kwenye kipengele cha HTML. Hapa ni jinsi unavyoongeza darasa la vitu kwenye lebo ya HTML na kulichagua katika CSS:

```html
<!-- index.html -->

<div class="alert-text">
  Please agree to our terms of service.
</div>
```

```css
/* styles.css */

.alert-text {
  color: red;
}
```

Kumbuka sintaksia ya vichaguzi vya `class`: nukta ikifuatiwa mara moja na thamani ya sifa ya darasa la vitu inayotambuliwa kwa herufi kubwa na ndogo. Madarasa hayawezi kuwa ya kipekee, kwa hivyo unaweza kutumia `class` ile ile kwenye vipengele vingi kama unavyotaka.

Jambo lingine unaloweza kufanya na sifa ya `class` ni kuongeza madarasa mengi kwenye kipengele kimoja kama orodha iliyotenganishwa kwa nafasi, kama `class="alert-text severe-alert"`. Kwa kuwa nafasi hutumika kutenganisha majina ya `class` kama haya, haipaswi kutumia nafasi kwa majina yenye maneno mengi na badala yake tumia alama ya mshale (-).

## ID Selectors
Vichaguzi vya ID ni sawa na vichaguzi vya `class`. Vinachagua kipengele chenye `id`, ambayo ni sifa nyingine unayoweka kwenye kipengele cha HTML:

```html
<!-- index.html -->

<div id="title">My Awesome 90's Page</div>
```

```css
/* styles.css */

#title {
  background-color: red;
}
```

Badala ya nukta, unatumia alama ya hashtag ikifuatiwa mara moja na thamani ya sifa ya `id` inayotambuliwa kwa herufi kubwa na ndogo. Tatizo la kawaida ni watu kutumia sifa ya `id` kupita kiasi wakati hawahitaji, na wakati madarasa ya vitu yatatosha. Ingawa kuna matukio ambapo kutumia `id` ni sahihi au inahitajika, kama vile kutumia faida ya maalum au kuwa na viungo vinavyoelekeza upya kwenye sehemu ya ukurasa wa sasa, unapaswa kutumia `id` kwa kiasi kidogo (ikiwa kabisa).

Tofauti kuu kati ya madarasa na ID ni kwamba kipengele kinaweza kuwa na `id` moja tu. `id` haiwezi kurudiwa kwenye ukurasa mmoja, na sifa ya `id` haipaswi kuwa na nafasi yoyote kabisa.
