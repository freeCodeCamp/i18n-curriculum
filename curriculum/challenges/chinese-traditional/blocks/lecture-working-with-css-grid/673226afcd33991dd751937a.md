# --interactive--

`grid-column` 和 `grid-row` 屬性讓你指定網格項目在網格佈局中的水平和垂直位置。

換句話說，它們都允許你透過取用格線來控制格子項目開始和結束的位置。這些格線是你已使用 `grid-template-rows` 和 `grid-template-columns` 屬性定義的列和行之間的邊界。

讓我們來看看 `grid-row` 和 `grid-column` 的語法，以及一個實際範例。

以下是 `grid-row` 屬性的語法：

```css
grid-row: <start-line> / <end-line>;
```

以及 `grid-column`：

```css
grid-column: <start-line> / <end-line>;
```

`<start-line>` 是項目開始的網格線，`<end-line>` 是項目結束的網格線。兩者皆為從 1 開始計數，也就是說，你從 1 而非 0 開始計數。

請記住，列的網格線是根據 `grid-template-rows` 屬性中指定的列數來產生的。對於行，則同樣適用於 `grid-template-columns` 屬性。

在實際範例中，讓我們從瀏覽器在你同時使用 `grid-template-rows` 和 `grid-template-columns` 屬性時產生的行開始。

這是一個有 4 行和 3 列的範例網格：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}
```

:::

檢查網格容器（類別 `grid`）會發現每個列和行都被兩條線界定——列或行的開始線位於起始處，結束線位於結尾處。

你可以標的這些列，開始使用 `grid-row` 和 `grid-column` 屬性來決定你應該將項目放置在哪裡。

以下是讓第一個網格項目佔據前兩個行的方法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
}
```

:::

這表示你希望第一個網格項目從第 1 行開始，並在第 3 行之前結束。

第一個項目現在將佔據兩個行，第四個項目則被推到第二列。很酷！

如果你也想讓第一個項目佔用兩列，你可以指定 `grid-row` 為 `1 / 3`：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
  grid-row: 1 / 3;
}
```

:::

你也可以使用 `span` 關鍵字來告訴網格項目跨越哪些列和行。例如，`1 / 3` 與 `1 / span 2` 是相同的：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / span 2;
  grid-row: 1 / span 2;
}
```

:::

你可以繼續將此技術應用於網格上的任何項目，並將它們放置在你想要的位置。
