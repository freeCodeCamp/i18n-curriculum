# --description--

$N$개의 원판이 한 줄로 놓여 있으며, 왼쪽부터 오른쪽까지 1부터 $N$까지 인덱스가 매겨져 있습니다.

각 원판은 검은 면과 흰 면이 있습니다. 처음에는 모든 원판이 흰 면을 보입니다.

매 턴마다 1부터 $N$까지(포함) 두 개의 정수 $A$와 $B$가 균등하게 무작위로 선택됩니다. $A$부터 $B$까지(포함) 인덱스를 가진 모든 원판이 뒤집힙니다.

다음 예시는 $N = 8$인 경우입니다. 첫 번째 턴에는 $A = 5$, $B = 2$이고, 두 번째 턴에는 $A = 4$, $B = 6$입니다.

<img alt="example for N = 8, with first turn A = 5 and B = 2, and second turn A = 4 and B = 6" src="https://cdn.freecodecamp.org/curriculum/project-euler/range-flips.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$E(N, M)$을 $M$번의 턴 후 흰 면이 보이는 원판의 기대값이라고 합시다. $E(3, 1) = \frac{10}{9}$, $E(3, 2) = \frac{5}{3}$, $E(10, 4) ≈ 5.157$, $E(100, 10) ≈ 51.893$임을 확인할 수 있습니다.

$E({10}^{10}, 4000)$을 구하세요. 소수점 아래 둘째 자리까지 반올림하여 답을 제출하세요.
