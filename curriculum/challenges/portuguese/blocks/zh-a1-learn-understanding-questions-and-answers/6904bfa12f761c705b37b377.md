# --description--

Wang Hua faz uma pergunta para alguém. Ela usa uma palavra para se referir à outra pessoa. Você já viu essa palavra em uma saudação comum.

# --instructions--

# --fillInTheBlank--

## --sentence--

`你好 (nǐ hǎo)，我是王华 (wǒ shì wáng huá)，请问 (qǐng wèn) BLANK 叫什么名字 (jiào shén me míng zi)？`

## --blanks--

`你 (nǐ)`

### --feedback--

Esta palavra significa "você" e se refere a alguém com quem você está falando.

# --explanation--

`你 (nǐ)` significa "você". É usado para falar diretamente com outra pessoa. Por exemplo:

`你是刘明吗 (nǐ shì liú míng ma)？` – Você é Liu Ming?

Você aprendeu a usar `我 (wǒ)` para se referir a si mesmo. Tanto `我 (wǒ)` quanto `你 (nǐ)` são **pronomes pessoais**, o que significa que são usados para se referir a pessoas.
