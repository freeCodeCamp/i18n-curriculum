# --interactive--

在 JavaScript 中处理字符串时，有很多情况下可能需要调整文本的大小写，例如将标题中的所有字母转换为大写，或将文本转换为小写以保持统一。

幸运的是，JavaScript 提供了两种内置方法，使这变得简单：`toUpperCase()` 和 `toLowerCase()`。

`toUpperCase()` 方法将所有字符转换为大写字母，并返回一个包含所有大写字母的新字符串。 当你想强调文本或使字符串格式保持一致时，这种方法非常有用。

让我们看一个示例：

:::interactive_editor

```js
let greeting = "Hello, World!";
let uppercaseGreeting = greeting.toUpperCase();
console.log(uppercaseGreeting);  // "HELLO, WORLD!"
```

:::

在这段代码中，`toUpperCase()` 方法将整个字符串转换为大写字母。

原始字符串保持不变，因为 `toUpperCase()` 返回的是一个新字符串，而不是修改原始字符串。

反过来，`toLowerCase()` 方法会将字符串中的所有字符转换为小写。 当需要对输入进行标准化时，例如比较用户提供的文本或进行大小写不敏感检查时，这很有帮助。

来看一个示例：

:::interactive_editor

```js
let shout = "I AM LEARNING JAVASCRIPT!";
let lowercaseShout = shout.toLowerCase();
console.log(lowercaseShout);  // "i am learning javascript!"
```

:::

`toLowerCase()` 方法将所有字符转换为小写，使字符串不那么扎眼，同时保留原始字符串不变。

总之，JavaScript 中的 `toUpperCase()` 和 `toLowerCase()` 方法是将字符串转换为大写或小写字母的强大工具。

这些方法对于规范文本输入、进行不区分大小写的比较以及确保设计一致性特别有用。

有了这些简单而有效的方法，你就能以更可控、更可预测的方式处理文本操作。
