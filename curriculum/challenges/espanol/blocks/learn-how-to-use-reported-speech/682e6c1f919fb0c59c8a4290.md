# --explanation--

En el **discurso indirecto**, la frase `said that` se usa para compartir lo que alguien dijo de una manera diferente, sin usar sus palabras exactas. La palabra `that` es opcional y a menudo se omite en el habla cotidiana. Por ejemplo:

- Discurso directo: `We need to update the software.`

- Discurso reportado: `He said that we need to update the software.` o `He said we need to update the software.`

Ambas versiones son correctas y significan lo mismo.
