# --explanation--

`Would` en este contexto expresa una acción habitual o un comportamiento repetido en el pasado. A menudo se usa para describir acciones que alguien realizaba regularmente, similar a `used to`. 

Sin embargo, `would` se enfoca más en la acción en sí, mientras que `used to` puede describir estados o hábitos. Por ejemplo:

- `She would always read a book before going to bed.` - Leer un libro antes de acostarse era una acción común para ella.

`She used to live in New York before moving here.` - aquí `used to` se usa porque vivir en un lugar expresa un estado y no una acción o algo que harías habitualmente.

En esta oración, Maria usa `would` para describir el hábito repetido de Brian de quedarse hasta tarde para afinar el código, lo que enfatiza su dedicación.
