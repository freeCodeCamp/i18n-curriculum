# --description--

Betrachten Sie die Zahl 45656.

Man erkennt, dass jede Paarung aufeinanderfolgender Ziffern von 45656 eine Differenz von eins hat.

Eine Zahl, bei der jedes Paar aufeinanderfolgender Ziffern eine Differenz von eins aufweist, nennt man eine Schrittzahl.

Eine pandigitale Zahl enthält jede Dezimalziffer von 0 bis 9 mindestens einmal.

Wie viele pandigitale Schrittzahlen kleiner als ${10}^{40}$ gibt es?
