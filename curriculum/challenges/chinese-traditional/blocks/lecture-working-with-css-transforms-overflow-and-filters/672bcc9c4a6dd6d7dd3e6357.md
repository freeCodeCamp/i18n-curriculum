# --description--

CSS 盒模型是網頁開發的基本概念。它定義了 HTML 元素的結構和定位方式。如果你了解這個模型，你將能夠控制網站上元素的大小、間距和外觀。

在 CSS 盒模型中，每個元素都被一個盒子包圍。這個盒子由四個部分組成：內容區域、內距、邊框和外距。 

內容區域是盒子最內層的零件。它是包含元素實際內容（例如文字或影像）的空間。

內距是緊接在內容區域之後的區域。它是內容區域與元素邊框之間的空間。使用內距你可以在內容周圍添加空間以提升可讀性。你可以使用 `padding` 屬性為上、右、下、左內距設定不同的數值。  

這是一個使用 `padding` 簡寫屬性的範例，我們將上方內距設為十五 像素、右方內距設為五 像素、下方內距設為二 像素，以及左方內距設為八 像素：  

```css
padding: 15px 5px 2px 8px;
```

邊框是 CSS 盒模型中元素的外緣或輪廓。它是元素的視覺邊界。你可以使用 `border` 屬性自訂邊框的樣式、寬度、顏色及其他屬性。以下是一個範例，我們將邊框設定為寬度為 5 像素、樣式為實線、顏色為藍色： 

```css
border: 5px solid blue;
```

如果你省略值，將會使用該值的預設屬性。寬度的預設值是 `medium`，樣式的預設值是 `none`，顏色的預設值是目前的顏色。 

如果你想讓所有邊都完全相同，可以直接在簡寫的 `border` 屬性中設定這三個屬性。但如果你想為每個邊指定不同的樣式，可以使用 `border-width`、`border-style` 和 `border-color` 屬性。 

```css
border-width: 2px 4px 7px 12px;
border-style: dashed solid solid dashed;
border-color: blue red green black;
```

你可以為這些屬性中的每一個寫入最多 4 個值。它們將從上方開始，依順時針方向依序套用。如果你只寫入一個值，該值將套用到所有四個邊。

最後，margin 是元素邊框外的空間。它決定元素與周圍其他元素之間的距離。你可以使用 `margin` 屬性為元素的上、右、下和左側設定不同的 margin 值。  

所以在這個範例中，上邊距是三 像素，右邊距是十二 像素，下邊距是九 像素，左邊距是七 像素：  

```css
margin: 3px 12px 9px 7px;
```

這四個組件是計算元素總寬度和高度的必要條件。

在接下來的幾堂課中，你將學習更多關於瀏覽器如何控制代碼以及你如何自訂它。CSS 盒模型是網頁開發的基本概念。  

了解這些組件如何互動並影響元素的尺寸，對於實作網頁設計至關重要。
