# --explanation--

Давайте розберемо, як Том порівнює JavaScript і Python:  

`I'm not a fan of Python's syntax—it feels a bit rigid.` — Тут Том висловлює, що йому не подобається синтаксис Python, бо він здається занадто суворим або негнучким. Це показує його незадоволення Python.  

`I find JavaScript more flexible.` — Том протиставляє Python JavaScript, описуючи JavaScript як `more flexible`, тобто легший у роботі або адаптації в різних ситуаціях.  

`And it's the go-to language for front-end development.` — Том підкреслює основне застосування JavaScript: це головна мова для створення фронтенд-застосунків, що демонструє його перевагу у цій області.  

Порівнюючи обидві, Том вважає JavaScript більш універсальним і кращим для фронтенд-розробки, тоді як Python здається йому менш гнучким через свій суворий синтаксис.
