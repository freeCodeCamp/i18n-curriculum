# --explanation--

`Unusually`는 평소나 예상보다 더 많거나 적게 일어나는 일을 설명할 때 쓰는 부사입니다. 반면 `unusual`는 흔하지 않거나 정상적이지 않은 것을 설명하는 형용사입니다. 예를 들어:

- `The server experienced an unusually high load today.` – 이 문장에서 `unusually`는 부하가 얼마나 높았는지를 나타냅니다(정상 범위를 훨씬 넘음).

- `It was unusual for the server to slow down like that.` – 이 문장에서 `unusual`는 상황을 나타내며 동사나 형용사를 수식하지 않습니다.

`query`는 데이터베이스에 정보를 얻거나 작업을 수행하도록 보내는 질문이나 요청입니다. 예를 들어:

`The application sent a query to the database to fetch user details.` – 이 뜻은 앱이 데이터베이스에 특정 정보를 요청했다는 의미입니다.
