# --description--

Un `function` nella programmazione è un insieme di istruzioni all'interno di un programma. È progettato per svolgere una particolare attività. Definisci una `function` una volta, specificando cosa deve fare, e poi puoi usarla tutte le volte che ti serve chiamandone il nome.

Se qualcosa è `missing` nel codice, significa che manca una parte necessaria o è stata trascurata.

Quando il codice è `working as expected`, funziona esattamente come dovrebbe, senza errori o problemi.
