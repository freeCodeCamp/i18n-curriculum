# --explanation--

`Reducing complexity`는 시스템을 더 이해하기 쉽고 다루기 쉽게 만드는 것을 의미합니다.
  
`Showing only what's necessary`는 사용자가 상호작용해야 하는 부분만 보게 된다는 뜻입니다.
  
`Keeping the details hidden`는 사용자가 필요로 하지 않는 내부 코드나 논리를 숨기는 것을 말합니다.

이 모두를 합치면 추상화는 복잡한 내부 작동을 숨기면서 명확하고 단순한 인터페이스를 만드는 것입니다.
