# --description--

ジョセフス問題は、次のような残酷な設定の数学パズルです：$n$ 人の囚人が円形に並び、0 から $n-1$ まで順番に番号が付けられています。

処刑人が円の周りを歩き、囚人 $0$ から始めて、$k$ 番目の囚人を順に取り除いて処刑します。

この処理が続くにつれて円はどんどん小さくなり、最後に1人だけ囚人が残ります。その囚人は解放されます。

例えば、囚人が $n=5$ 人で $k=2$ の場合、処刑される順番（「処刑シーケンス」と呼びます）は 1、3、0、4 となり、生き残るのは #2 です。

任意の $n, k > 0$ に対して、最終的に生き残る囚人の番号を求めてください。

ある事件では、41人の囚人がいて、3番目ごとに囚人が処刑されていました（$k=3$）。

その中にジョセフスという賢い男がいて、この問題を解き、生き残る位置に立ち、物語を語り継ぎました。

彼の番号は何番でしょうか？

囚人の初期人数と `k` をパラメータとして受け取り、生き残る囚人の番号を返す関数を書いてください。

function josephus(init, kill) {
  const arr = Array.from(Array(init).keys());
  let curr = -1
  while (arr.length > 1) {
    curr = (curr + kill) % arr.length;
    arr.splice(curr, 1);
    curr--;
  }
  return arr[0];
}
