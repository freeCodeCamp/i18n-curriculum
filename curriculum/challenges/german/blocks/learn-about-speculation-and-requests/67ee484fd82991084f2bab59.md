# --explanation--

Eine `code submission` ist die Sammlung von Code-Dateien, die ein Programmierer zur Überprüfung einreicht. Zum Beispiel:

- `Make sure your code submission follows the guidelines.` – Der Code muss die Regeln erfüllen, bevor Sie ihn absenden.

- `The professor reviewed my code submission and gave feedback.` – Die Lehrkraft hat Ihren Code überprüft und Kommentare geteilt.

Eine `loop structure` führt einen Codeblock wiederholt aus, bis eine bestimmte Bedingung erfüllt ist, und spart Zeit, indem sie sich wiederholende Aufgaben automatisiert, wie zum Beispiel `for`-Schleifen oder `while`-Schleifen. Zum Beispiel:

- `The loop structure in my code runs five times.` – Ihre Schleife wird im Programm fünfmal wiederholt.

- `A loop structure allows a program to repeat actions.` – Eine Schleife hilft einem Programm, dieselbe Aufgabe mehrfach auszuführen.
