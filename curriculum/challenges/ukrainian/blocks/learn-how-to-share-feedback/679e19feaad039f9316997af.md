# --explanation--

Фраза `Thanks for` + `-ing` використовується, щоб висловити подяку за дію, яка відбулася або відбувається. Дієслово після `for` має форму `-ing`, оскільки воно виступає як іменник (герундій). Наприклад:

`Thanks for helping me with the report.` — це означає, що мовник цінує допомогу, надану для звіту.
