# --description--

في هذه الورشة، ستبني صفحة مكتبة كتب بإنشاء بطاقات كتب تعرض معلومات عن كتب مختلفة. ستتدرب على تنظيم المحتوى باستخدام عناصر `div`، الأصناف، والمعرفات.

ابدأ صفحة المكتبة بإنشاء قالب جاهز لـ HTML.

أضف الإعلان `<!DOCTYPE html>` وعنصري `html` و`head`.

أضف خاصية `lang` إلى عنصر `html` واضبطها على `"en"`.

# --hints--

```js

```

يجب أن تتضمن شفرتك وسمَي `<html>` افتتاحي وختامي.

```

```

```js
assert.match(code,/<head>[\s\S]*<\/head>/i);
```

```js

```
