# --description--

Je, ungemfanyaje ikiwa una makundi mawili ya vitu vinavyoshiriki baadhi ya tamko zao za mtindo?

```css
.read {
  color: white;
  background-color: black;
  /* several unique declarations */
}

.unread {
  color: white;
  background-color: black;
  /* several unique declarations */
}
```

Kichaguzi chetu cha `.read` na `.unread` vinashiriki tamko za `color: white;` na `background-color: black;`, lakini vinginevyo vina tamko zao za kipekee. Ili kupunguza kurudia, unaweza kuunganisha vichaguzi hivi viwili pamoja kama orodha iliyotenganishwa kwa koma:

```css
.read,
.unread {
  color: white;
  background-color: black;
}

.read {
  /* several unique declarations */
}

.unread {
  /* several unique declarations */
}
```

Mifano yote miwili hapo juu (ikiwa na kuunganishwa au bila) itatoa matokeo sawa, lakini mfano wa pili unapunguza kurudia tamko na kufanya iwe rahisi kuhariri `color` au `background-color` kwa madarasa yote mawili kwa wakati mmoja.
