# --description--

片語 `to keep in mind` 意指記住或考慮某事。當有人給出建議或重要資訊，且應該記住以便日後使用時，常會使用此片語。

例如，如果有人說：「記得在健行前查看天氣。」你可能會回應 `I'll keep that in mind`，意思是你會記得考慮這個建議。
