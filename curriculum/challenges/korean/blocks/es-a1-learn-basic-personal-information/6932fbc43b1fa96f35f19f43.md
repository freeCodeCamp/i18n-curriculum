# --explanation--

Esteban은 이렇게 말합니다: `Me llamo Esteban Navarro Huamán.`

`Me llamo + name` 구조는 자신의 이름을 말할 때 사용합니다. 여기서 Esteban은 자신의 `nombre`와 `apellido`를 모두 제공하며, 자신의 전체 신원을 묻는 질문에 답하고 있습니다. 이 대답은 다음 질문과 일치합니다:

`¿Cuál es tu nombre y apellido?`

사람의 이름을 묻는 또 다른 흔한 질문은 다음과 같습니다:

`¿Cómo te llamas?`

두 질문 모두 사람의 이름을 묻는 데 사용되지만, 구조가 다릅니다.
