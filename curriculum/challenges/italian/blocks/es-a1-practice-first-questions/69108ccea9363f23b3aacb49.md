# --explanation--

Questo dialogo tra Mateo e Camila ha come focus le professioni.

`¿A qué te dedicas?` significa "Qual è la tua professione?" o "Che lavoro fai?".

**`Soy` + [professione]** significa "Sono un/una [professione]".

In spagnolo, la maggior parte delle professioni cambia in base al genere. Tuttavia, alcune professioni che finiscono in `-ista` o `-nte` spesso rimangono esattamente uguali per entrambi i generi:

| Professione | Maschile | Femminile |
| :---: | :---: | :---: |
| Analista | `analista` | `analista` |
| Giornalista | `periodista` | `periodista` |
| Receptionist | `recepcionista` | `recepcionista` |
| Cantante | `cantante` | `cantante` |
