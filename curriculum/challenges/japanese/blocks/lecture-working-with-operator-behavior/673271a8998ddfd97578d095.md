# --interactive--

数値を扱っていて、値を1増やしたり減らしたりしたい場合、インクリメント演算子とデクリメント演算子を使うと簡単にできます。わかりやすく説明しましょう。

インクリメント演算子とデクリメント演算子はそれぞれ`++`と`--`で表されます。どちらも変数の値を`1`だけ調整できます。

`x = x + 1`や`x = x - 1`のように書く代わりに、`x++`を使って`1`を足したり、`x--`を使って`1`を引いたりできます。より速く、すっきりしていて読みやすいです。

インクリメント演算子とデクリメント演算子にはひとつ注意点があります。接頭辞（prefix）と後置（postfix）の2つの形があり、値が更新されるタイミングが違います。インクリメント演算子の場合、接頭辞は`++x`、後置は`x++`です。

接頭辞（`++x`）は変数の値を先に増やしてから新しい値を返します。後置（`x++`）は変数の現在の値を先に返してから増やします：

:::interactive_editor

```js
let x = 5;

console.log(++x); // 6
console.log(x); // 6
```

:::

上のコードでは、`++x`は「`x`を先にインクリメントしてから使う」という意味です。だから`++x`をログに出すと、すぐにインクリメントされた値、つまり`6`が表示されます。

次に後置を使った例を見てみましょう：

:::interactive_editor

```js
let y = 5;

console.log(y++); // 5
console.log(y); // 6
```

:::

この例では、`y++`は「`y`を先に使ってからインクリメントする」という意味です。`y++`をログに出すと`5`が表示されますが、その後`y`は`6`になります。

デクリメント演算子はインクリメントと同じですが、値を`1`だけ減らします。こちらも2つの形があります。接頭辞（`--x`）は変数の値を先に減らしてから新しい値を返します。後置（`x--`）は現在の値を先に返してから減らします：

:::interactive_editor

```js
let x = 5;
console.log(--x); // 4
console.log(x); // 4

let y = 5;
console.log(y--); // 5
console.log(y); // 4
```

:::

では、どちらを使うべきでしょうか？接頭辞か後置か？多くの場合、どちらを使っても問題ありません。どちらも目的を果たします。ただし、値をすぐに式の中で使いたい場合は違いが重要になります。次の例を見てみましょう：

:::interactive_editor

```js
let a = 5;
let b = ++a;
console.log(b); // 6 (a was incremented before assignment)

let c = 5;
let d = c++;
console.log(d); // 5 (c was incremented after assignment)
```

:::

すぐに更新された値が必要なら接頭辞を使い、現在の値を先に使って後でインクリメントしたいなら後置を使ってください。
