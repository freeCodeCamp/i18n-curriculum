# --interactive--

HTML 엔티티 또는 문자 참조는 HTML에서 예약된 문자를 나타내기 위해 사용하는 문자 집합입니다.

화면에 `This is an <img/> element` 텍스트를 표시하고 싶다고 가정해 보겠습니다. 현재 편집기에 있는 코드를 사용하면 원하는 결과가 표시되지 않습니다. 예제에 `src`와 `alt` 속성을 추가해도 문단 중간에 이미지가 나타납니다. 원하는 결과가 아닙니다. 예제와 상호작용하려면 인터랙티브 편집기를 활성화해야 합니다.


:::interactive_editor

```html
<p>This is an <img /> element</p>
```

:::

HTML 파서가 less than (`<`) 기호 다음에 HTML 태그 이름을 보면 이를 HTML 요소로 해석합니다. 그래서 화면에 `This is an <img/> element`가 원하는 대로 표시되지 않는 것입니다.

이 문제를 해결하려면 HTML 엔티티를 사용할 수 있습니다. 다음은 less than (`<`)과 greater than (`>`) 기호에 올바른 HTML 엔티티를 사용한 업데이트된 예제입니다. 이제 화면에 `This is an <img/> element`가 표시될 것입니다.

인터랙티브 편집기를 활성화하고 문단 요소 아래에 `&lt;p&gt;learning is fun&lt;/p&gt;`를 추가해 보세요. 화면에 `<p>learning is fun</p>`가 표시될 것입니다.

:::interactive_editor

```html
<p>This is an &lt;img /&gt; element</p>

```

:::

이러한 유형의 문자 참조를 명명된 문자 참조라고 합니다. 명명된 참조는 앰퍼샌드 기호(`&`)로 시작하고 세미콜론(`;`)으로 끝납니다. 명명된 문자 참조를 사용하면 HTML 파서가 이를 실제 HTML 요소로 혼동하지 않습니다.

또 다른 문자 참조 유형은 10진수 숫자 참조입니다. 이 문자 참조는 앰퍼샌드 기호와 해시 기호(`#`)로 시작하고, 하나 이상의 10진수 숫자가 뒤따르며, 세미콜론으로 끝납니다.

다음은 less than 기호에 10진수 숫자 참조를 사용하는 예입니다.

인터랙티브 편집기를 활성화하고 코드를 변경하여 다른 기호를 확인해 보세요. 저작권 기호에는 `&#169;`를, 등록 상표 기호에는 `&#174;`를 사용할 수 있습니다.

:::interactive_editor

```html
&#60;
```

:::

마지막 문자 참조 유형은 16진수 숫자 참조입니다. 이 문자 참조는 앰퍼샌드 기호, 해시 기호, 그리고 `x` 문자로 시작합니다. 그다음 하나 이상의 ASCII 16진수 숫자가 나오고 세미콜론으로 끝납니다.

다음은 less than 기호에 16진수 숫자 참조를 사용하는 예입니다.

인터랙티브 편집기를 활성화하고 코드를 변경하여 다른 기호를 확인해 보세요. 유로 기호에는 `&#x20AC;`를, 그리스 대문자 오메가 기호에는 `&#x03A9;`를 사용할 수 있습니다.

:::interactive_editor

```html
&#x3C;
```

:::
