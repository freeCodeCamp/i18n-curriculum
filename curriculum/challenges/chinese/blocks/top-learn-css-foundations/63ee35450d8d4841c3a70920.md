# --description--

内部 CSS（或嵌入式 CSS）涉及将 CSS 添加到 HTML 文件本身，而不是创建一个完全独立的文件。使用内部方法，你将所有规则放在一对开闭的 `<style>` 标签内，然后将其放置在 HTML 文件的开闭 `<head>` 标签内。由于样式直接放在 `<head>` 标签内，因此不再需要外部方法所需的 `<link>` 元素。

除了这些差异，语法与外部方法（选择器、括弧状的花括弧、声明）完全相同：

```html
<head>
  <style>
    div {
      color: white;
      background-color: black;
    }

    p {
      color: red;
    }
  </style>
</head>
<body>...</body>
```

此方法对于为网站的单个页面添加独特样式非常有用，但它不像外部方法那样保持内容分离，并且根据规则和声明的数量，可能会导致 `HTML` 文件变得相当大。

# --questions--    

## --text--

以下哪项是内部和外部 CSS 方法之间的区别？

## --answers--

外部方法将 CSS 规则放在单独的文件中，而内部方法则将 CSS 规则放在超文本标记语言文件内部。

---

内部方法将 CSS 与超文本标记语言分开，而外部方法则将 CSS 直接嵌入超文本标记语言中。

---

内部方法使用 `<link>` 元素将 CSS 链接到超文本标记语言，而外部方法则将 CSS 直接嵌入超文本标记语言。
