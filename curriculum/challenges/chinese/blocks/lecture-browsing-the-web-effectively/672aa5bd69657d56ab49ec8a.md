# --description--

网页浏览器是你用来访问互联网页面的应用。如果你正在学习 freeCodeCamp 课程，很有可能你现在正在使用网页浏览器！

截至 2024 年，最流行的浏览器是 Microsoft Edge、Firefox、Google Chrome 和 Safari。还有许多其他选项，市场份额也在不断变化，但这四个是目前最常见的。

你的操作系统很可能默认安装了浏览器。对于 Windows，你会有 Microsoft Edge。对于 macOS，你会有 Safari。对于 Linux，你可能会有 Firefox。

但是如果你想使用不同的浏览器怎么办？

好消息是你已经有一个默认浏览器。所以你可以导航到你想要的浏览器的网页，并下载适合你操作系统的安装程序。

选择浏览器时，请考虑速度、安全性特色以及与你经常访问的网站的兼容性等因素。

一些浏览器提供独特的特色。例如，Opera 有内置的 VPN，而 Brave 则专注于隐私和广告拦截。

安装通常很简单。下载安装程序后，只需运行它并按照屏幕上的说明操作。

大多数浏览器会提供将你的书签和 `Settings` 从你当前的默认浏览器导入的选项，使过渡更容易。

或者，你可以通过操作系统的包管理器安装浏览器，例如在 macOS 上使用 Homebrew 执行 `brew install --cask google-chrome`，或在 Arch Linux 上执行 `yay -S brave-bin`。

可用的包管理器和包名称将根据你的操作系统而有所不同。

请记住，你不限于只使用一个浏览器。许多人根据不同的用途使用不同的浏览器——也许一个用于工作，另一个用于个人浏览。

随意尝试不同的浏览器以查找最适合你的。
