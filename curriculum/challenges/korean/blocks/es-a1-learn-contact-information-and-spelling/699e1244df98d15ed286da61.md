# --description--

`Otro` (남성형)과 `otra` (여성형)은 추가적이거나 다른 것을 말할 때 사용합니다. 예를 들어:

- `Quiero otro café.` – 나는 다른 커피를 원해요. (`Café`는 남성형이므로 여기서는 `otro`가 사용됩니다).

- `Necesito otra copia.` – 나는 다른 복사본이 필요해요. (`Copia`는 여성형이므로 여기서는 `otra`가 사용됩니다).

무언가를 확인할 때는 `otra`와 `vez`가 자주 함께 들립니다.

`Vez`은 어떤 일이 일어나는 경우나 시간을 의미합니다. 여성 명사입니다. `vez`가 여성형이기 때문에 표현은 항상 여성형을 사용합니다:

`Puedes repetirlo otra vez?` - 다시 한 번 말해줄 수 있나요?

이 흔한 표현은 무언가를 한 번 더 하거나 다시 하는 것을 말할 때 사용합니다.

# --explanation--

`Otra vez`는 다음을 결합합니다:

- `Otra` (“다른”의 여성형).

- `Vez` (시간 / 경우).

`vez`가 여성형이므로 항상 `otra vez`를 사용하세요.

이 전체 표현은 무언가가 다시 일어나야 한다고 요청하거나 나타낼 때 흔히 사용됩니다.
