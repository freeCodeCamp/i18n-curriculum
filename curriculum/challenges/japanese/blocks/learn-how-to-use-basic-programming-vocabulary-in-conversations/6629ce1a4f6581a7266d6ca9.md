# --description--

**現在完了進行形**は、過去に始まり今も続いている動作や状況について話すときに使います。活動の継続時間や進行中のプロセスであることを強調したいときにとても役立ちます。

**現在完了進行形**の作り方は次の通りです：主語 + `has/have been` + 動詞の`-ing`形。この時制は、`for`や`since`のような時間の表現とよく使われます。`For`は「2年間」や「6か月」のような期間を表すときに使い、`since`は「2018年」や「今朝」のような特定の時点を表すときに使います。

例：

`I have been studying English for two years.` これは、2年前に英語の勉強を始めて、今も続けていることを意味します。

`She has been working here since 2018.` これは、2018年にここで働き始めて、今も働いていることを意味します。

ブライアンの話を聞いて、空欄を埋めてください。
