# --explanation--

`To notice` 是指透過看見、聽見或感覺到某事而意識到它——尤其是當某事已經改變或顯眼時。例如：

`Did you notice the new button on the homepage?`－這等同於問某人是否看到了變更。

Tom 的問題，`Were the changes noticed?`，是**被動語態**。它著重於變更（物件），而非注意到變更的人（主詞）。另一個**被動語態**問題的例子：

`Was the mistake corrected before the launch?`－這著重於修正，而非誰進行了修正。
