# --explanation--

`Staging server`는 시스템이나 애플리케이션의 변경 사항을 라이브 운영 서버에 적용하기 전에 테스트하는 환경을 가리킵니다. 예를 들어:

`We uploaded the new website features to the staging server for testing.` - 이 문장에서 `staging server`는 새 웹사이트 기능이 라이브 전에 제대로 작동하는지 확인하는 테스트 공간입니다.

`Can't`는 이 문맥에서 태그 질문의 일부입니다. 긍정문 뒤에 와서 정중하거나 대화체로 동의를 확인하거나 확인하는 역할을 합니다. 예를 들어:

`We can finish the task today, can't we?` - 이 문장에서 긍정문(`We can finish the task today`) 뒤에 부정 태그 질문(`can't we?`)이 따라와 동의나 확인을 요청합니다.
