# --description--

화살표 함수는 JavaScript에서 함수를 작성하는 보다 현대적인 방법입니다. 전통적인 함수 표현식에 비해 더 간결한 구문을 제공합니다. 화살표 함수는 `=>` 구문을 사용해 정의합니다.

```js
const add = (a, b) => {
  return a + b;
}
```

위 예제에서 화살표 함수 `add`는 두 개의 매개변수 `a`와 `b`를 받아 그 합을 반환합니다. 덧셈 연산 결과를 반환하는 데 `return` 키워드를 사용합니다.

화살표 함수에 표현식이 하나만 있다면 중괄호 `{}`와 `return` 키워드를 생략할 수 있습니다. 표현식은 암묵적으로 반환됩니다.

```js
const add = (a, b) => a + b;
```

위 예제에서 화살표 함수 `add`는 두 개의 매개변수 `a`와 `b`를 받아 그 합을 반환합니다. `return` 키워드와 중괄호 `{}`가 생략되었으며, 덧셈 연산 결과가 암묵적으로 반환됩니다.
