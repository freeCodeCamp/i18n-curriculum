# --explanation--

`Sometimes` は文の最初に置くと、その状況が時々起こるが常にではないことを強調します。主動詞の前に置くこともできますが、文頭に置くと自然な話し方になります。例えば：

`Sometimes we forget how important breaks are.` - これは時々起こることを意味します。

`Too` は強調語として使われ、過剰または望ましい以上を意味します。`very` や `so` は程度を強調するだけで過剰を示しません。例えば：

- `It's too late.` - これは許容できるより遅いことを意味します。

- `This exercise is too difficult.` - これは運動が必要以上に難しいことを意味します。
