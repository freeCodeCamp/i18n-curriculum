# --explanation--

Camila agora está usando a mesma estrutura que usou para dizer seu nome. Mas desta vez, para dizer sua profissão. A estrutura é assim:

`Soy` + profissão

Novamente, você não precisa dizer o pronome `yo` porque o verbo `soy` já indica quem está falando.

`Soy analista` significa "Eu sou um analista".

`Analista` é um substantivo neutro, então é o mesmo para falantes masculinos e femininos. Por exemplo:  

`Soy analista de datos.` – Eu sou um analista de dados.

Esta frase de exemplo funciona tanto para masculino e feminino.
