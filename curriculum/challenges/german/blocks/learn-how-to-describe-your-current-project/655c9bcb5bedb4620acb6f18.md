# --description--

Wenn Sie etwas `make sure` möchten, tun Sie dies mit besonderer Sorgfalt, um das Ergebnis am Ende zu bestätigen. Zum Beispiel:

- `We have to make sure the code is running.` – Das bedeutet, dass Sie beim Code besonders vorsichtig sind, um sicherzustellen, dass er gut läuft.

- `Please make sure that you order the correct color.` – Hier werden Sie gebeten, die Farbe, die Sie möchten, vor der Bestellung zu bestätigen.
