# --explanation--

`To seem` 用於當某事根據我們所知看起來是真的，但我們並不完全確定。 例如：

`It seems the users liked the new menu。`－這表示你認為使用者喜歡它，根據回授或行為，但你不確定。

`To be found to be` 是一個用於表示某人發現、察覺或得出結論的 **被動語態** 結構，但重點在於結果，而非誰做出判斷。 例如：

`The layout was found to be confusing by some testers。`－這表示測試者認為它令人困惑，但句子強調的是發現，而不是人（在此情況下為測試者）。
