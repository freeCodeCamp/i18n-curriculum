# --description--

Alice 和 Bob 玩遊戲 Nim Square。

Nim Square 就像常規的三堆正常玩法 Nim，但玩家只能從堆積中移除平方數量的石頭。

三個堆積中石頭的數量由有序三元組（$a$、$b$、$c$）表述。

如果 $0 ≤ a ≤ b ≤ c ≤ 29$，則下一位玩家的輸局位置數量為 1160。

找出下一位玩家的輸敗位置數量，條件為 $0 ≤ a ≤ b ≤ c ≤ 100\,000$。
