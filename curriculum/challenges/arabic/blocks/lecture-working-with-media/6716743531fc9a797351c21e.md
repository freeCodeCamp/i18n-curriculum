# --interactive--

العنصر المُستبدَل هو عنصر يُحدَّد محتواه بواسطة مورد خارجي بدلاً من CSS نفسه. تُستخدم CSS، أو أوراق الأنماط المتتالية، لإضافة أنماط إلى صفحة الويب. من الأمثلة الشائعة للعناصر المُستبدَلة عناصر الصورة، وiframe، والفيديو.

مع العناصر المُستبدَلة، يمكنك التحكم في موضع العنصر أو تخطيطه. لكن لا يمكن لـ CSS تعديل محتوى ذلك العنصر مباشرة. قد يكون من الأسهل شرح ذلك ببعض الأمثلة. خذ عنصر الصورة كمثال، الذي يضمّن صورة في صفحة الويب الخاصة بك:

```html
<img src="example-img-url" alt="Descriptive text goes here">
```

يُستبدَل العنصر نفسه بالكائن الخارجي: الصورة. يمكن لـ CSS التحكم في أشياء مثل موضع الصورة أو تطبيق فلتر عليها، لكن لا يمكنك تعديل الصورة نفسها فعليًا. مثال أكثر شمولًا قد يكون عنصر `iframe`، الذي يضمّن موقعًا خارجيًا في صفحة الويب الخاصة بك.

إليك مثالًا على استخدام عنصر `iframe` لفيديو شهير على قناة freeCodeCamp في YouTube. إذا أردت مشاهدة فيديوهات مختلفة في نافذة المعاينة، غيّر قيمة الخاصية `src` إلى فيديو تختاره. لعرض المعاينات، يجب تفعيل المحرر التفاعلي.

**ملاحظة**: لا تقلق بشأن الخصائص الإضافية المذكورة في المثال التفاعلي مثل `referrerpolicy` و`allowfullscreen`. ستتعلم المزيد عن التعامل مع عناصر `iframe` في ورشة عمل مستقبلية.

:::interactive_editor

```html
<iframe width="400" height="200" src="https://www.youtube.com/embed/u43gJJrVa1I?si=BoDW_puFsy8OEr_Z" title="Professional Cloud Architect Certification Course – Pass the Exam! (YouTube video)" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen></iframe>
```

:::

أمثلة شائعة أخرى لاستخدام عنصر `iframe` هي تضمين الخرائط في الصفحة. إليك مثالًا لخريطة مضمّنة.

فعّل المحرر التفاعلي وجرب التفاعل مع الخريطة نفسها عن طريق التكبير والتصغير.

:::interactive_editor

```html
<iframe
  title="Map of the Royal Observatory, Greenwich, London"
  width="300"
  height="200"
  src="https://www.openstreetmap.org/export/embed.html?bbox=-0.004017949104309083%2C51.47612752641776%2C0.00030577182769775396%2C51.478569861898606&amp;layer=mapnik">
</iframe>
```

:::

يُستبدَل العنصر نفسه بالكائن الخارجي: الموقع. يمكن لـ CSS تغيير موضع الموقع المضمّن، لكن لا يمكنك تعديل محتويات الموقع. للتعمق أكثر، إذا كان للموقع المضمّن عنصر `h1`، فلن يستطيع CSS تنسيق ذلك العنصر `h1`. لا يمكنك تغيير الحجم أو لون الخط، وما إلى ذلك.

هناك بعض العناصر المُستبدَلة الأخرى، مثل `video` و`embed`. وبعض العناصر تتصرف كعناصر مُستبدَلة في ظروف معينة. إليك مثالًا لعنصر `input` مع الخاصية `type` مضبوطة على `image`:

```html
<input type="image" alt="Descriptive text goes here" src="example-img-url">
```

يُعتبر هذا النوع من `input` عنصرًا مُستبدَلًا، لكن أنواع `input` الأخرى مثل `text` أو `email` ليست عناصر مُستبدَلة.
