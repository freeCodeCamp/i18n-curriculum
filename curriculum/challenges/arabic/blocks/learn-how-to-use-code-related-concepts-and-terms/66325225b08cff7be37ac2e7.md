# --description--

تُستخدم الضمائر الوصلية مثل `who`، و`whom`، و`which`، و`that` لربط البنود التابعة باسم أو ضمير.

يشير `Who` و`whom` إلى الأشخاص. الفرق هو أن `who` يكون فاعل الفعل، بينما يُستخدم `whom` كمفعول به لفعل أو حرف جر.

يُستخدم `Which` للأشياء أو الحيوانات المحددة، ويمكن استخدام `that` للأشخاص أو الأشياء أو الحيوانات في البنود التعريفية.

أمثلة:

`The programmer who wrote this code is brilliant.` (`who` يشير إلى المبرمج)

`The artist whom we met at the gallery is internationally recognized.` (`whom` يشير إلى الفنان)

`The tool that you lent me was useful.` (`that` يشير إلى الأداة)

`The book, which I bought yesterday, is already a bestseller.` (`which` يشير إلى الكتاب)

`The teacher which teaches us math is very kind.`

### --feedback--

عادةً ما يُستخدم `which` للأشياء وليس للأشخاص. الضمير الصحيح هنا يجب أن يكون `who`.

`The car, that won the race, is very fast.`

في هذه الجملة، استخدام `that` صحيح نحويًا، لكن الفواصل غير ضرورية. يجب أن تكون: `The car that won the race is very fast.`
