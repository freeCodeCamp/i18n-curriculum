# --interactive--

Es wird Situationen geben, in denen Sie Knoten zum DOM hinzufügen oder daraus entfernen müssen, und es gibt einige Web-APIs, die Sie dafür verwenden können.

In dieser Lektion behandeln wir die Methoden `appendChild()` und `removeChild()`.

Die Methode `appendChild()` wird verwendet, um einen Knoten ans Ende der Liste der Kindknoten eines bestimmten Elternknotens hinzuzufügen.

Hier ist die grundlegende Syntax für die Methode `appendChild()`:

```js
parentNode.appendChild(newNode);
```

Schauen wir uns ein Beispiel an, um besser zu verstehen, wie man die Methode `appendChild()` verwendet.

In diesem Beispiel haben wir ein ungeordnetes Listenelement mit der ID `desserts` im HTML. Diese Liste enthält zwei Listenelemente mit `Cake` und `Pie`:

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
```

Um auf diese Liste in der JavaScript-Datei zuzugreifen, können wir die Methode `getElementById()` verwenden:

```js
const dessertsList = document.getElementById("desserts");
```

Wir haben eine Variable namens `dessertsList`, die die Referenz auf das Element `ul` mit der ID `desserts` speichert.

Dann müssen wir ein neues Listenelement mit der Methode `createElement()` erstellen:

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");
```

Der folgende Code erstellt ein neues Listenelement und speichert es in einer Variablen namens `listItem`.

Um diesen Knoten ans Ende der Liste der Kindknoten des Elements `ul` hinzuzufügen, können wir die Methode `appendChild()` verwenden:

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

dessertsList.appendChild(listItem);
```

Wenn Sie diesen Code ausführen, sehen Sie, dass ein neues Listenelement ans Ende der Liste der Kindknoten des Elements `ul` hinzugefügt wird.

:::interactive_editor

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
<script src="index.js"></script>
```

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

dessertsList.appendChild(listItem);
```

:::

Das Problem ist, dass das neue Listenelement leer ist. Um dem neuen Listenelement Textinhalt hinzuzufügen, können Sie die Eigenschaft `textContent` verwenden:

:::interactive_editor

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
<script src="index.js"></script>
```

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

listItem.textContent = "Cookies";
dessertsList.appendChild(listItem);
```

:::

Jetzt zeigt die Liste `Cake`, `Pie` und `Cookies` an.

Um einen Knoten aus dem DOM zu entfernen, können Sie die Methode `removeChild()` verwenden.

Hier ist die grundlegende Syntax für die Methode `removeChild()`:

```js
parentNode.removeChild(childNode);
```

Schauen wir uns ein Beispiel an, bei dem wir das letzte Absatz-Element aus diesem `section` Element entfernen möchten:

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
```

Wir können auf das Element `section` in der JavaScript-Datei mit der Methode `getElementById()` zugreifen:

```js
const sectionEl = document.getElementById("example-section");
```

Sobald wir die Referenz auf das Element `section` haben, können wir mit der Methode `querySelector()` auf den letzten Absatz zugreifen:

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");
```

Wir verwenden die Pseudoklasse `:last-of-type`, um das letzte Absatz-Element innerhalb des Elements `section` auszuwählen.

Jetzt, da wir den Eltern- und Kindknoten haben, können wir das letzte Absatz-Element aus dem Element `section` mit der Methode `removeChild()` entfernen:

:::interactive_editor

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
<script src="index.js"></script>
```

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");

sectionEl.removeChild(lastParagraph);
```

:::

Wann möchten Sie also Knoten zum DOM hinzufügen oder daraus entfernen?

Wenn Sie mit dynamischem Inhalt arbeiten, müssen Sie möglicherweise Knoten zum DOM hinzufügen oder daraus entfernen. Beispiele aus der Praxis sind Einkaufswagen, To-do-Listen und Social-Media-Feeds.
