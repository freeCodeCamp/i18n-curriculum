# --interactive--

二元逻辑运算符可帮助你计算两个表达式，并根据它们的真假返回结果。 让我们来看看三个最常见的二进制逻辑运算符：逻辑与、逻辑或和空值合并运算符。

The logical AND operator is represented by a double ampersand (`&&`). 它检查两个操作数是否都为真，并返回一个结果。 如果两个操作数都为真，则返回第二个值，即右边的值：

:::interactive_editor

```js
const result = true && 'hello';

console.log(result); // hello
```

:::

在上面的示例中，文本 `hello` 将被记录到控制台，因为两个操作数都是 `true` 。 如果任一操作数是假，则返回该假值：

:::interactive_editor

```js
const result = 0 && 3;

console.log(result); // 0
```

:::

因为 `0` 是一个假值，数字 `0` 将被记录到控制台。 如果两个操作数都是假值，则返回第一个假值：

:::interactive_editor

```js
const result = false && 0;

console.log(result); // false
```

:::

由于 `false` 是一个假值，则 `false` 会被记录到控制台。 逻辑与运算符在检查多个条件并确保所有条件均为真时非常有用。 这是一个示例：

:::interactive_editor

```js
if (2 < 3 && 3 < 4) {
 console.log('The if block runs'); 
} else {
 console.log('The else block runs');
} 
```

:::

在这个条件中，由于 `2` 小于 `3` 且 `3` 小于 `4`，那么 `The if block runs` 这句话将被记录到控制台。

逻辑 OR 操作符检查是否至少有一个操作对象为真值。如果第一个操作对象为真值，它将返回该值：

:::interactive_editor

```js
const result = 'This is truthy' || false;

console.log(result); // This is truthy
```

:::

如果第一个操作数是假，但第二个操作数是真，那么第二个值将被记录到控制台：

:::interactive_editor

```js
const result = 0 || 'This is truthy';

console.log(result); // This is truthy
```

:::

在这样的 `if/else` 语句中使用逻辑或运算符是很常见的：

:::interactive_editor

```js
let userInput;

if (userInput || 'Guest') {
 console.log('A user is present');
} else {
 console.log('No user detected');
}
```

:::

因为我们没有为 `userInput` 变量赋值，它目前是 `undefined`。 `if` 语句中的条件会检查变量 `userInput` 或字符串 `Guest` 是否为真。 由于字符串 `Guest` 在布尔上下文中为真，因此 `A user is present` 字符串将被记录到控制台。

空值合并操作符比逻辑或和逻辑与更复杂。用双问号（`??`）表示，它帮助在你只想在第一个值为 `null` 或 `undefined` 时返回某个值的场景中使用。以下是使用空值合并操作符的示例：

:::interactive_editor

```ts
const result = null ?? 'default';

console.log(result); // default
```

:::

Since `null` is a nullish value, the string `default` would be logged to the console. 在 `null` 或 `undefined` 是唯一应触发回退或默认值的值的情况下，空值合并符非常有用。 下面是一个处理用户偏好设置的示例：


:::interactive_editor

```ts
const userSettings = {
 theme: null,
 volume: 0,
 notifications: false,
};

let theme = userSettings.theme ?? 'light';
console.log(theme); // light
```

:::

在上面的示例中，有一个名为 `userSettings` 的对象，包含 `theme`、`volume` 和 `notifications` 属性。 我们使用点符号访问 `theme`，即 `userSettings.theme`。  You will learn more about how to work with objects in a future lesson. 由于用户的 `theme` 当前设置为 `null`，则字符串 `light` 将被记录到控制台。
