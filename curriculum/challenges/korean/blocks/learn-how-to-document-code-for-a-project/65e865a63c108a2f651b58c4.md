# --description--

이번 작업에서는 `specific`와 `specify`의 차이를 배우세요. `Specific`는 명확하게 정의되거나 자세히 설명된 것을 나타내는 형용사이고, `specify`는 어떤 내용을 자세히 말하는 동사입니다.

예를 들어, `specific` 지침은 `use port 8080`일 수 있고, `specify`는 그 세부 사항을 말하는 것을 의미할 수 있습니다: `specify the port number.`

`Setup`는 무언가를 정리하거나 준비하는 것을 가리키며, 소프트웨어나 장비를 준비하는 상황에서 자주 사용됩니다. 예를 들어 `setup instructions for the software.`
