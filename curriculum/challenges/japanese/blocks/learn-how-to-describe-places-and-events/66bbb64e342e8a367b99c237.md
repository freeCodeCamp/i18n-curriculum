# --explanation--

誰かが「`struggling with`している」と言うと、それは何かに困っている、苦労していることを意味します。例えば：

- `The team is struggling with the project deadline.` - チームは締め切りに間に合わず苦労しています。

`IDEs`は`Integrated Development Environments`の略です。これは開発者がコードを書き、テストし、デバッグするために使うソフトウェアツールです。代表的なIDEにはVisual Studio Code、IntelliJ IDEA、Eclipseなどがあります。

この会話では、トムがIDEの使い方に苦労していて、サラに相談したいと思っています。
