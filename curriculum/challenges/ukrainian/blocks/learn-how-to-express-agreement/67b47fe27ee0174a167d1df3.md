# --explanation--

`Tend to` — це фраза, яка використовується для опису того, що відбувається регулярно або ймовірно станеться. Наприклад:

`She tends to be early for meetings.` — це означає, що приходити раніше на зустрічі для неї є звичкою або регулярною подією.

`Lag` означає затримку або повільну реакцію. Наприклад:

`The video lags when there is a poor internet connection.` — це означає, що відео затримується або повільне через з’єднання.

`Switch` означає змінюватися від одного до іншого. Наприклад:

`Users can switch between different tabs in the browser.` — це означає, що користувачі можуть переходити з однієї вкладки на іншу під час перегляду.
