# --explanation--

`Could` 和 `might` 是用来描述可能性的情态动词。

`Could` 表示某事可能发生，但不确定。例如：

`If we don't fix this bug, it could cause a system crash。` - 系统有可能会停止工作。

`Might` 表示某事有可能，但比 `could` 的确定性更低。例如：

`We might need to rewrite some parts of the code。` - 可能需要进行更改。
