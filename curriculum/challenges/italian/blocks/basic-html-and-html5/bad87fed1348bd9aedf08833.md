# --description--

Il nostro telefono non ha molto spazio verticale.

Eliminiamo gli elementi non necessari così possiamo iniziare a costruire la nostra CatPhotoApp.

# --instructions--

Elimina il tuo elemento `h1` così possiamo semplificare la nostra vista.

# --hints--

Il tuo elemento `h1` dovrebbe essere eliminato.

```js
assert.notMatch(code,/<h1>/gi);
assert.notMatch(code,/<\/h1>/gi);
```

Il tuo elemento `h2` dovrebbe essere sulla pagina.

```js
assert.match(code,/<h2>[\w\W]*<\/h2>/gi);
```

Il tuo elemento `p` dovrebbe essere sulla pagina.

```js
assert.match(code,/<p>[\w\W]*<\/p>/gi);
```
