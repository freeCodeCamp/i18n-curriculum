# --explanation--

網路釣魚攻擊經常誘騙使用者點擊 `malicious links`，導致對帳戶或系統的 `unauthorized access`。

`Unauthorized access` 意味著有人未經許可進入系統。 例如：

`The system blocked unauthorized access.` - 系統防止未經批准的使用者取得存取權。

`Malicious link` 指的是可能竊取資訊或感染系統的有害連結。例如：

`The email had a malicious link inside.` - 電子郵件包含一個用來欺騙你的惡意聯結。
