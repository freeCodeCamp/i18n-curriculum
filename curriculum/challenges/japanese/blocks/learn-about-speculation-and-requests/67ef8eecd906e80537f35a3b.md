# --explanation--

`Should have` は過去のことに対する後悔や果たせなかった責任を表すために使います。マークはテストを実行するのが正しいことだったのに、それをしなかったという意味です。

`But` は彼が行動しなかった理由を示します：`I was in a hurry`。別の例：

`We should have fixed the bug, but we didn't notice it.` - バグを修正すればよかったのに、見逃してしまったという後悔です。
