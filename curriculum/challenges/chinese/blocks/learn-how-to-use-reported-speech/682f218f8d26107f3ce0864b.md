# --explanation--

`Of course` 是一种礼貌的表达方式，用来表示同意或毫不犹豫地说 `yes`。它意味着说话者乐意或愿意做某事。例如：

`Of course, I'll help you with the report.`——这意味着说话者很乐意提供帮助。

`Will` 通常用于做出承诺或表达说话时刻的决定。例如：

`I'll email you the file this afternoon.` – 这表示很快会做某事的承诺。
