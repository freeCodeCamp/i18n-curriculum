# --explanation--

Los ataques de phishing a menudo engañan a los usuarios para que hagan clic en `malicious links`, lo que conduce a `unauthorized access` a cuentas o sistemas.

`Unauthorized access` significa que alguien entra a un sistema sin permiso. Por ejemplo:

`The system blocked unauthorized access.` - El sistema evitó que usuarios no autorizados obtuvieran acceso.

`Malicious link` se refiere a un enlace dañino que puede robar información o infectar un sistema. Por ejemplo:

`The email had a malicious link inside.` - El correo electrónico incluía un enlace malicioso diseñado para engañarte.
