# --description--

Nella lezione precedente, hai imparato i tipi di dati comuni in Python, inclusi i tipi stringa, numero intero e float. Mentre costruisci i tuoi programmi, dovrai imparare a vedere il tipo di una variabile.

Ecco un esempio di variabile:

```py
developer = 'Devin'
```

Per vedere di che tipo è `developer`, puoi usare la funzione `type()` in questo modo:

```py
developer = 'Devin'

print(type(developer)) # <class 'str'>
```

L’output di `<class 'str'>` significa che `developer` è di tipo stringa.

Se non fornisci argomenti alla funzione `type()`, riceverai il seguente messaggio di errore:

```md
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: type() takes 1 or 3 arguments
```

Ecco tutti i tipi di dati che hai imparato finora insieme ai loro tipi nel terminale:

```python
my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 'hello', 8.5}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 'hello', 8.5)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>
```

Ci saranno momenti nel tuo programma in cui dovrai verificare che una variabile specifica sia di un certo tipo prima di eseguire operazioni su di essa. Qui entra in gioco la funzione `isinstance()`.

Ecco un esempio di variabile a cui è assegnata una stringa:

```py
account_balance = '12'
```

Se provi a fare espressioni matematiche come la divisione usando la variabile `account_balance`, riceverai un messaggio di errore.

```py
account_balance = '12'

account_balance / 2

# Traceback (most recent call last):
#   File "<stdin>", line 1, in <module>
# TypeError: unsupported operand type(s) for /: 'str' and 'int'
```

Per vedere se `account_balance` è un numero intero, puoi controllare usando la funzione `isinstance()` in questo modo:

```py
account_balance = '12'

isinstance(account_balance, int) # False
```

La funzione integrata `isinstance()` ti permette di verificare se una variabile corrisponde a un tipo di dato specifico. Prende in input un oggetto e il tipo contro cui vuoi verificarlo, poi restituisce un booleano. In questo caso, dato che `account_balance` è una stringa, restituirà `False`.

La funzione `isinstance()` ti permette anche di controllare più tipi contemporaneamente.

Ecco un esempio che verifica se `account_balance` è un `int` o un `float`:

```py
account_balance = 12
isinstance(account_balance, (int, float)) # True
```

In questo esempio, `account_balance` è un numero intero quindi `isinstance()` restituisce `True`. Se `account_balance` fosse `12.0`, `isinstance()` restituirebbe comunque `True` perché stai controllando se è un numero intero o un float.

Nei futuri workshop e laboratori, userai le funzioni `type()` e `isinstance()` per assicurarti che le tue variabili contengano i tipi di dati corretti prima di eseguire operazioni su di esse.
