# --interactive--

CSS에서 `width`와 `height` 속성은 웹페이지 요소의 크기를 제어하는 데 사용됩니다.

이 속성들은 픽셀(`px`), 백분율(`%`), 뷰포트 단위(`vw`, `vh`) 등 다양한 단위로 정의할 수 있습니다.

`width` 속성은 요소의 너비를 지정합니다. 너비를 지정하지 않으면 기본적으로 `auto`로 설정됩니다. 이는 브라우저가 요소의 내용, 부모, 그리고 디스플레이 유형에 따라 너비를 결정하도록 합니다. `div` 요소의 경우, `width: auto`는 부모 컨테이너의 전체 너비를 채우도록 확장하게 만듭니다.

`height` 속성은 요소의 높이를 지정합니다. 마찬가지로 높이는 기본적으로 `auto`로 설정되어 내부 내용에 맞게 조절됩니다.

다음은 `width`와 `height` 속성을 사용한 예시입니다:

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 100px;
      height: 100px;
      background-color: skyblue
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

이 예시에서는 `div` 요소에 `box`라는 클래스를 지정했습니다. 이 요소는 높이와 너비가 `100px`를 차지하며, 배경색은 `skyblue`입니다.

픽셀은 CSS에서 고정 크기 단위로, 크기를 정확하게 제어할 수 있습니다.

`min-width` 속성은 요소가 가질 수 있는 최솟값 너비를 지정합니다. 내부 내용이 더 작더라도 요소는 이 값보다 작아지지 않습니다.

`min-height`는 요소가 가질 수 있는 최솟값 높이를 지정합니다. 요소가 설정된 값보다 더 짧아지지 않도록 보장합니다.

다음은 예시입니다:

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 50px;
      min-width: 100px;
      height: 50px;
      min-height: 100px;
      background-color: lightcoral;
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

위 예시는 `min-width`와 `min-height`가 어떻게 작동하는지 보여줍니다.

박스의 `width`와 `height`가 50px로 설정되어 있지만, 실제 크기는 `100px` 곱하기 `100px`입니다. 이는 `min-width`와 `min-height`가 `100px`로 설정되어 있어서, 지정된 `width`와 `height`보다 크기 때문입니다.

기억하세요, `min-width`나 `min-height`가 `width`나 `height`보다 크면 더 작은 값을 재정의합니다. 이는 요소가 너무 작아지지 않도록 하여 일관되고 사용하기 편한 디자인을 유지하는 데 중요합니다.

`max-width`는 요소가 커질 수 있는 최대 너비를 지정합니다. 충분한 공간이 있어도 이 값을 넘지 않습니다.

`max-height`는 요소가 커질 수 있는 최대 높이를 지정합니다. 내용 크기와 상관없이 이 값을 넘지 않습니다.

다음은 예시입니다:

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 200px;
      max-width: 150px;
      height: 200px;
      max-height: 150px;
      background-color: lightgreen;
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

위 예시는 `max-width`와 `max-height`가 `width`와 `height`를 재정의하는 방식을 보여줍니다. 박스가 `200px` 곱하기 `200px`로 설정되어 있지만, 실제 크기는 `150px` 곱하기 `150px`입니다. 이는 `max-width`와 `max-height`가 `150px`로 설정되어 더 작기 때문입니다.

기억하세요, `max-width`나 `max-height`가 `width`나 `height`보다 작으면 우선합니다. 이는 레이아웃에서 요소의 최대 크기를 제어하는 데 중요합니다.

CSS는 `min-width`와 `min-height`를 `width`와 `height`보다 우선시합니다. `max-width`와 `max-height`는 값이 한계를 넘을 때 크기를 제한합니다.

예를 들어, `width`를 `600px`로, `max-width`를 `500px`로 설정하면 요소 너비가 `500px`로 제한됩니다. `max-width`가 더 넓게 설정된 값을 재정의하여 요소가 지정된 최대 크기 내에 머무르게 합니다.

이렇게 하면 표준 너비와 높이 값과 상관없이 요소가 원하는 크기 범위 내에 유지됩니다.
