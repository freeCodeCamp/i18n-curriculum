# --description--

Para un entero positivo $n$, define $f(n)$ como el múltiplo positivo más pequeño de $n$ que, escrito en base 10, usa solo dígitos $≤ 2$.

Así $f(2) = 2$, $f(3) = 12$, $f(7) = 21$, $f(42) = 210$, $f(89) = 1\\,121\\,222$.

Además, $\displaystyle\sum_{n = 1}^{100} \frac{f(n)}{n} = 11\\,363\\,107$.

Encuentra $\displaystyle\sum_{n = 1}^{10\\,000} \frac{f(n)}{n}$.
