# --interactive--

웹 페이지에서 링크를 클릭한 후 보라색으로 변하는 것을 본 적이 있을 거예요. 이는 링크의 상태가 바뀌어 다른 스타일이 적용되기 때문입니다. 링크가 가질 수 있는 상태는 다섯 가지가 있습니다.

첫 번째는 기본 상태로, `:link`입니다. 이 상태는 사용자가 아직 방문하거나 클릭하거나 상호작용하지 않은 링크를 나타냅니다. 이 상태는 페이지의 모든 링크에 기본 스타일을 제공한다고 생각하면 됩니다. 나머지 상태들은 이 위에 덧붙여집니다.

인터랙티브 편집기를 활성화하고 미리보기 창의 링크를 확인하세요. 기본 상태를 나타내는 파란색 기본 색상이어야 합니다. 클릭하면 보라색으로 변할 거예요.

:::interactive_editor

```html
<a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
```

:::

두 번째 상태는 `:visited`으로, 사용자가 이미 링크된 페이지를 방문한 경우에 적용됩니다. 기본적으로 이 상태는 링크를 보라색으로 바꾸지만, CSS를 활용해 사용자에게 다른 시각적 표시를 제공할 수 있습니다. 이는 사용자가 문서의 일부를 이미 읽었음을 알리거나, 이전에 사용해 신뢰할 수 있는 사이트임을 알려주는 데 도움이 됩니다.

미리보기 창에서 링크를 클릭하면 방문한 링크가 갈색으로 바뀌는 것을 볼 수 있습니다. 미리보기를 보려면 인터랙티브 편집기를 활성화해야 합니다.

**참고**: 이 인터랙티브 예제에서는 링크 스타일 변화를 볼 수 있도록 일부 CSS가 제공되었습니다. CSS 코드를 이해하려고 걱정하지 마세요. 앞으로의 모듈에서 모두 배우게 됩니다.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:visited {
  color: brown;
}
```

:::

세 번째 상태는 `:hover`입니다. 이 상태는 사용자가 링크 위에 마우스 커서를 올려놓았을 때 적용됩니다. 이 상태는 사용자가 실제로 클릭하려는 링크에 더 주의를 기울이도록 돕습니다.

인터랙티브 편집기를 활성화하고 링크 위에 마우스를 올려보세요. 색상이 빨간색으로 바뀌는 것을 볼 수 있습니다.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:hover {
  color: red;
}
```

:::

다음은 `:focus`입니다. 이 상태는 링크에 포커스가 있을 때 적용됩니다.

미리보기 창의 공백 부분 아무 곳이나 클릭한 후 키보드에서 `tab`를 누르세요. 링크가 초록색으로 바뀌는 것을 볼 수 있습니다. 미리보기를 보려면 인터랙티브 편집기를 활성화해야 합니다.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:focus {
  color: green;
}
```

:::

마지막으로 `:active`가 있습니다. 이 상태는 사용자가 링크를 활성화할 때 적용됩니다. 일반적으로는 마우스 왼쪽 버튼으로 링크를 클릭하는 경우입니다. 이 상태는 사용자가 클릭한 요소가 상호작용 가능하다는 것을 보여주는 데 도움이 됩니다.

인터랙티브 편집기를 활성화하고 링크를 클릭하세요. 링크가 검은색으로 변하는 것을 볼 수 있습니다. 이 변화는 매우 빠르게 일어나므로 색상 변화를 보려면 몇 번 클릭해야 할 수도 있습니다.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:active {
  color: black;
}
```

:::

이 상태들을 사용해 링크 스타일을 지정할 때는 CSS를 작성하는 특정 순서가 있습니다: `link`, `visited`, `hover`, `focus`, 그리고 `active` 순서입니다.

이제 페이지의 링크에 개성을 부여하면서도 각 링크 상태에 대해 사용자가 알아야 할 중요한 정보를 제공하는 방법을 알게 되었습니다.
