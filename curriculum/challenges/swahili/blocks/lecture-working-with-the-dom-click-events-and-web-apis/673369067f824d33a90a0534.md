# --interactive--

Wakati mtumizi anabofya kitufe au kuna mabadiliko katika fomu, hii inajulikana kama tukio. Katika programu zako, utahitaji kuwa na njia ya kusikiliza matukio haya na kuyajibu.

Njia ya `addEventListener()` hutumika kusikiliza matukio. Inachukua hoja mbili: tukio unalotaka kusikiliza na kitendakazi ambacho kitaitwa wakati tukio litakapotokea.

Hapa kuna sintaksia ya msingi:

```js
element.addEventListener("event", listener);
```

Kipengele ni kipengele cha HTML kinachotazamwa kwa matukio, na tukio linaelezea aina ya tukio la kusikiliza, kama `"click"`.

Msikilizaji ni kitu ambacho kitapokea taarifa wakati tukio litakapotokea. Mara nyingi, hii itakuwa kitendakazi unachokieleza kushughulikia tukio.

Hapa kuna mfano:

```js
element.addEventListener("click", () => {
  // code to run when the click event occurs
});
```

Unaweza pia kuchagua kupitisha rejea ya kitendakazi kama hii:

```js
function handleClick() {
  // code to run when the click event occurs
}

element.addEventListener("click", handleClick);
```

Wakati mwingine unaweza kutaka kuunda kitendakazi tofauti kushughulikia tukio. Hii inaweza kufanya msimbo wako kuwa rahisi kusoma na kudumisha.

Hoja ya msikilizaji pia inaweza kuwa `null`, au inaweza kuwa kitu kinachotumia kiolesura cha `EventListener`. Kiolesura cha `EventListener` kinaelezea njia moja inayoitwa `handleEvent()`. Njia hii huitwa moja kwa moja kila tukio unalosikiliza linapotokea. Kutumia kiolesura hiki kunaruhusu kitu kimoja kushughulikia matukio mengi ikiwa inahitajika.

Mara nyingi, hutahitaji kutumia kiolesura hiki, na badala yake unaweza kupitisha kitendakazi kitakachoitwa wakati tukio litakapotokea.

Sasa, tuchukulie mfano ili kuelewa vizuri jinsi hii inavyofanya kazi:

Katika mfano huu, tuna kipengele cha `button` chenye kitambulisho cha `btn`:

```html
<button id="btn">Show alert</button>
```

Tunataka kusikiliza tukio la `"click"` kwenye kitufe hiki na kuonyesha onyo wakati kitufe kinapobofwa.

Kwanza tunahitaji kupata upatikanaji wa kipengele cha `button` katika msimbo wetu wa JavaScript. Kisha tunahitaji kuongeza msikilizaji wa tukio kusikiliza tukio la bofya na kuonyesha onyo wakati kitufe kinapobofwa:

:::interactive_editor

```html
<button id="btn">Show alert</button>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("btn");

btn.addEventListener("click", () => alert("You clicked the button"));
```

:::

Sasa kila wakati kitufe kinapobofwa, mtumizi ataona ujumbe wa onyo unaoonyeshwa kwenye skrini kama hii:

```md
You clicked the button
```

Aina nyingine ya tukio unaloweza kusikiliza ni tukio la `"input"`. Tukio hili huchochewa wakati thamani ya kipengele cha ingizo inabadilika.

Hapa kuna mfano:

```html
<input type="text" id="input" placeholder="Type something" />
```

Tunataka kusikiliza tukio la ingizo kwenye kipengele hiki cha ingizo na kurekodi thamani ya ingizo kwenye konsoli kila mtumizi anapochapa kitu.

Kwanza tunahitaji kupata upatikanaji wa kipengele cha `input` katika msimbo wetu wa JavaScript. Kisha tunahitaji kuongeza msikilizaji wa tukio kusikiliza tukio la ingizo na kurekodi thamani ya ingizo kwenye konsoli kila mtumizi anapochapa kitu:

:::interactive_editor

```html
<input type="text" id="input" placeholder="Type something" />
<script src="index.js"></script>
```

```js
const input = document.getElementById("input");

input.addEventListener("input", () => {
  console.log(input.value);
});
```

:::

Kuna matukio mengi zaidi unayoweza kusikiliza kwa kutumia njia ya `addEventListener()`. Baadhi ya matukio ya kawaida ni `mouseover`, `mouseout`, `keydown`, `keyup`, na `submit`.

Katika mafundisho yajayo, tutashughulikia matukio zaidi na jinsi ya kutumia njia ya `addEventListener()` kusikiliza matukio hayo.
