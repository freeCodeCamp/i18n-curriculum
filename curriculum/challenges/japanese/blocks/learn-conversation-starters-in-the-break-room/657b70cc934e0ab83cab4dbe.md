# --description--

これまでに、`do` / `does` を使った質問の仕方と、`don't` / `doesn't` を使った否定文の作り方を練習しました。ここで復習しましょう。覚えておいてください：

- 肯定文：`He does work on weekends.`（その人が週末に働くことを意味します）

- 否定文：`He doesn't work on weekends.`（その人が週末に働かないことを意味します）

- 疑問文：`Does he work on weekends?`（その人が週末に働くかどうかを尋ねます）
