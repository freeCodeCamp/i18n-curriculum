# --description--

Unaweza kufikiria vigezo kama "makontena ya kuhifadhi" data katika msimbo wako.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/variables-and-operators/top-learn-variables-and-operators.png" alt='the concept of variables in coding as boxes labeled "birthday" that store and provide access to data, specifically a date "16-09-2003"'>

Kigezo ni "hifadhi yenye jina" ya data. Unaweza kutumia vigezo kuhifadhi vitu vizuri, wageni, na data nyingine.

Ili kuunda kigezo katika JavaScript, tumia neno kuu `let`.

Tamko hapa chini huunda (kwa maneno mengine: hutangaza) kigezo chenye jina "message":

```js
let message;
```

Sasa, unaweza kuweka data ndani yake kwa kutumia opereta ya mgawo `=`:

```js
let message;

// store the string 'Hello' in the variable named message
message = 'Hello'; 
```

Ili kuwa kifupi, unaweza kuunganisha tamko la kigezo na mgawo wa thamani katika mstari mmoja:

```js
let message = 'Hello';
```
