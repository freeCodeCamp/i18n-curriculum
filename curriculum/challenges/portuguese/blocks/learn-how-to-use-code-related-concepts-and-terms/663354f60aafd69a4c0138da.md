# --description--

O `equals symbol` (`=`) é comumente usado para atribuição.É usado para atribuir um valor a uma variável.O nome da variável é escrito à esquerda do símbolo de igual e o valor que você deseja armazenar na variável está à direita.

Por exemplo, `name = "Sophie";` atribui a string `"Sophie"` à variável `name`.

A `assign a value` em programação significa dar um valor específico a uma variável.O `equals symbol` (`=`) é comumente usado para esse propósito.Quando você atribui um valor a uma variável, você está dizendo ao programa para lembrar esse valor para uso posterior.

Por exemplo, `age = 25` atribui o valor `25` à variável `age`.
