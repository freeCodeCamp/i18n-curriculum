# --description--

元组是 Python 的一种数据类型，用于创建有序的值序列。元组可以包含像这样混合的数据类型：

```py
developer = ('Alice', 34, 'Rust Developer')
```

元组类似于列表，但列表是可变的数据类型，而元组是不可变的。这意味着元组中的元素在创建后不能被更改。

如果你尝试更新元组中的某一项，你将得到一个 `TypeError`：

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: 'tuple' object does not support item assignment
"""
```

要访问元组中的元素，你可以使用方括弧表示法和索引号：

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

如果你需要从元组的末尾访问元素，那么你可以使用负索引。下面是使用负索引访问元组中倒数第二个元素的示例：

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

如果你尝试传入一个大于或相等于元组长度的索引号，那么你将收到如下的 `IndexError`：

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list index out of range
"""
```

另一种创建元组的方法是使用 `tuple()` 构造函数，如下所示：

```py
developer = 'Jessica'
tuple(developer) # ('J', 'e', 's', 's', 'i', 'c', 'a')
```

对于 `tuple()` 构造函数，你可以传入不同的可迭代对象，比如字串、列表甚至其他元组。 

要查看一项是否在元组中，你可以像这样使用 `in` 关键字：

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

你也可以像处理列表一样从元组中解包项：

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

在此示例中，`name` 的值为 `'Alice'`，`age` 的值为 `34`，`job` 的值为 `'Rust Developer'`。

如果你需要收集元组中剩余的元素，可以像这样使用星号（`*`）操作符：

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

这里，`name` 的值是 `'Alice'`，而 `rest` 是一个由数字 `34` 和字串 `'Rust Developer'` 组成的列表。

就像对列表一样，你可以对元组使用切片操作符来提取其中的一部分。下面是将项 `'pie'` 和 `'cookies'` 提取到一个单独元组中的示例：

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

请记住，第一个数字表现提取的起始索引，而第二个数字表现结束索引。但请注意，结束索引处的项不包含在提取的元组中。

如果你需要从一个元组中移除一项，这是不可能的，因为元组是不可变的。所以这个例子会产生一个误差：

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

那么什么时候你会使用 `tuple` 而不是 `list`？

如果你需要一个可以添加、删除和更新元素的动态集合，那么你应该使用列表。如果你知道你正在处理一个固定且不可变的数据集合，那么你应该使用元组。

在下一课中，我们将查看你在处理元组时会使用的一些常见方法。
