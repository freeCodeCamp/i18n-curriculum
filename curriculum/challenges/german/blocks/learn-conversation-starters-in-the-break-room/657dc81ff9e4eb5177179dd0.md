# --description--

`Occasionally` bedeutet, dass etwas ab und zu passiert. Es ist nicht die ganze Zeit, und auch nicht nie, sondern irgendwo dazwischen. Zum Beispiel:

`I occasionally get a bug in my code.` – Manchmal gibt es einen Fehler in Ihrem Code, aber nicht sehr oft.

Sehen Sie sich diese Adverbien und die Häufigkeit an, die sie normalerweise darstellen.

| Adverb       | Häufigkeit | Adverb    | Häufigkeit |
| ------------ | ---------- | --------- | ---------- |
| Never        | 0%         | Often     | 70%        |
| Rarely       | 5%         | Generally | 80%        |
| Seldom       | 15%        | Usually   | 90%        |
| Occasionally | 30%        | Always    | 100%       |
| Sometimes    | 50%        |           |            |
