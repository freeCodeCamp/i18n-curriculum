# --explanation--

누군가 자신이 `struggling with` 하고 있다고 말하면, 그것은 그 일이 어렵거나 문제가 있다는 뜻입니다. 예를 들어:

- `The team is struggling with the project deadline.` - 팀이 마감일을 맞추는 데 어려움을 겪고 있습니다.

`IDEs`는 `Integrated Development Environments`를 의미합니다. 이것들은 개발자가 코드를 작성, 테스트, 디버그하는 데 사용하는 소프트웨어 도구입니다. 대표적인 IDE로는 Visual Studio Code, IntelliJ IDEA, Eclipse가 있습니다.

대화에서 Tom은 자신의 IDE 사용에 어려움을 겪고 있으며 Sarah와 그 문제에 대해 이야기하려고 합니다.
