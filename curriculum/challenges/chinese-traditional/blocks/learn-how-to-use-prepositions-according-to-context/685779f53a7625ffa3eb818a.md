# --explanation--

`Trying to` 意味著努力去做某事。 例如：

`She's trying to finish the report before noon.`－這表示她正在努力完成它。

`Monitor` 是一種顯示影像的裝置，並且常與電腦一起使用。例如：

`He connected a second monitor to his laptop.`－這表示他為他的筆記型電腦添加了另一個螢幕。

`They've` 是 `they have` 的縮寫。這是現在完成式，表示在現在之前完成的動作或從過去開始並持續到現在的情況。​ 例如：

`They've moved the files to a new folder。`－這表示檔案是由他們重新定位的。
