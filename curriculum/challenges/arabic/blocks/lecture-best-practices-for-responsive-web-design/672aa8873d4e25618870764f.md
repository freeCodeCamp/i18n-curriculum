# --interactive--

تصميم الويب المتجاوب هو نهج في تطوير الويب يهدف إلى إنشاء مواقع توفر تجربة مشاهدة وتفاعل مثالية عبر مجموعة واسعة من الأجهزة، من أجهزة الكمبيوتر المكتبية إلى الهواتف المحمولة.

المبدأ الأساسي في التصميم المتجاوب هو القدرة على التكيف: قدرة الموقع على تعديل تخطيطه ومحتواه بناءً على حجم الشاشة وإمكانيات الجهاز الذي يُعرض عليه.

عادةً ما يعتمد التصميم المتجاوب على ثلاثة مكونات رئيسية:

- تستخدم الشبكات السائلة وحدات نسبية مثل النسب المئوية بدلاً من الوحدات الثابتة مثل البكسل، مما يسمح للمحتوى بإعادة الحجم وإعادة التدفق بناءً على حجم الشاشة.

- تُضبط الصور المرنة لتغيير حجمها داخل العناصر الحاوية لها، مما يضمن عدم تجاوزها حدود الحاويات على الشاشات الصغيرة.

- وتسمح استعلامات الوسائط للمطورين بتطبيق أنماط مختلفة بناءً على خصائص الجهاز، وخاصة عرض إطار العرض. ستتعلم المزيد عن استعلامات الوسائط في دروس لاحقة.

العلاقة بين تصميم الويب المتجاوب وأدوات مثل CSS Grid وFlexbox علاقة تكاملية. بينما التصميم المتجاوب هو مفهوم أو نهج، فإن CSS Grid وFlexbox هما أداتان عمليتان تسهلان تنفيذ التصاميم المتجاوبة بشكل كبير وأكثر كفاءة.

في الدروس السابقة تعلمت كيفية العمل مع Flexbox وفي الدروس القادمة ستتعلم كيفية العمل مع CSS Grid. لكن الآن، إليك مقدمة موجزة عن CSS Grid.

CSS Grid هو نظام تخطيط ثنائي الأبعاد يسمح بترتيبات أكثر تعقيدًا. إنه ممتاز لإنشاء تخطيطات صفحات كاملة وكذلك تخطيطات مكونات أصغر.

إليك مثالًا على كيفية استخدام CSS Grid بشكل متجاوب:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr;
  gap: 20px;
}

@media (min-width: 768px) {
  .grid-container {
    grid-template-columns: 1fr 1fr;
  }
}

@media (min-width: 1024px) {
  .grid-container {
    grid-template-columns: 1fr 1fr 1fr;
  }
}
```

:::

في هذا المثال، تبدأ الشبكة بعمود واحد على الشاشات الصغيرة، ثم تتحول إلى عمودين على الشاشات متوسطة الحجم، ثم إلى ثلاثة أعمدة على الشاشات الأكبر. هذا يوضح كيف يمكن لـ Grid إنشاء تخطيطات متجاوبة تتكيف مع أحجام الشاشات المختلفة.

يوفر كل من Flexbox وGrid مزايا كبيرة مقارنة بأساليب التخطيط القديمة مثل العناصر العائمة أو تخطيطات الجداول. يقدمان مرونة وتحكمًا أكبر، ويتطلبان شفرة أقل، وصُمما مع وضع الاستجابة في الاعتبار. يسمحان للمطورين بإنشاء تخطيطات معقدة ومتجاوبة بسهولة نسبية، مما يجعلهما أدوات لا تقدر بثمن في تنفيذ تصميم الويب المتجاوب.

يجدر بالذكر أن Flexbox وGrid غالبًا ما يُستخدمان معًا في التصاميم المتجاوبة. يُستخدم Flexbox عادةً للمكونات والتخطيطات أحادية البعد، بينما يُستخدم Grid لبنية الصفحة العامة والتخطيطات ثنائية البعد. يعتمد الاختيار بينهما غالبًا على احتياجات التخطيط المحددة للتصميم.

بالإضافة إلى Flexbox وGrid، تلعب ميزات CSS الأخرى أدوارًا مهمة في التصميم المتجاوب.

على سبيل المثال، تسمح الدالة `calc()` بخلط الوحدات وإجراء الحسابات، مما يمكن أن يكون مفيدًا جدًا في إنشاء تخطيطات مرنة.

الصور المتجاوبة هي جانب حاسم آخر من تصميم الويب المتجاوب. تسمح الخاصية `srcset` وعنصر `picture` في HTML5 بتقديم ملفات صور مختلفة بناءً على إمكانيات الجهاز، مما يضمن عدم تحميل المستخدمين لملفات صور كبيرة غير ضرورية على الأجهزة ذات الشاشات الصغيرة أو الدقة المنخفضة.

في الختام، تصميم الويب المتجاوب هو نهج يهدف إلى إنشاء مواقع تعمل بشكل جيد على أي جهاز.

بينما هو فلسفة تصميم أكثر من كونه تقنية محددة، فإنه يعتمد بشكل كبير على ميزات CSS مثل استعلامات الوسائط، ويسهل بشكل كبير باستخدام أدوات التخطيط الحديثة مثل Flexbox وGrid. توفر هذه الأدوات المرونة والتحكم اللازمين لإنشاء تصاميم متجاوبة حقيقية، مما يسمح للمواقع بالتكيف بسلاسة مع التنوع المتزايد للأجهزة المستخدمة للوصول إلى الويب.
