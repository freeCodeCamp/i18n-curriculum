# --description--

소수 2와 3으로만 나누어지는 $100$ 이하의 가장 큰 정수는 96입니다. 왜냐하면 $96 = 32 \times 3 = 2^5 \times 3$이기 때문입니다.

서로 다른 두 소수 $p$와 $q$에 대해, $M(p, q, N)$을 $N$ 이하의 양의 정수 중 오직 $p$와 $q$로만 나누어지는 가장 큰 수로 정의하며, 그런 양의 정수가 존재하지 않으면 $M(p, q, N)=0$입니다.

예를 들어 $M(2, 3, 100) = 96$입니다.

$M(3, 5, 100) = 75$인데, 90이 아닌 이유는 90이 2, 3, 5로 나누어지기 때문입니다. 또한 $M(2, 73, 100) = 0$인데, 이는 2와 73으로 나누어지는 $100$ 이하의 양의 정수가 존재하지 않기 때문입니다.

$S(N)$을 모든 서로 다른 $M(p, q, N)$의 합으로 정의합니다. $S(100)=2262$입니다.

$S(10\,000\,000)$을 구하세요.
