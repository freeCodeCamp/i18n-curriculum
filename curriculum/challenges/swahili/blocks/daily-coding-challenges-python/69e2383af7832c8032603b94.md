# --description--

Kutolewa mfuatano wa herufi wa oda ya kahawa, tambua vitu vyovyote vya menyu na rudisha oda iliyopangwa vizuri.

Tumia vitu vya menyu na bei zifuatazo:

| Item | Price |
| - | - |
| `"cold brew"` | $4.50 |
| `"oat latte"` | $5.00 |
| `"cappuccino"` | $4.75 |
| `"espresso"` | $3.00 |
| `"vanilla syrup"` | $0.75 |
| `"caramel drizzle"` | $0.60 |
| `"extra shot"` | $0.50 |
| `"oat milk"` | $0.75 |
| `"cream"` | $0.75 |

Rudisha mfuatano wa herufi wenye vitu vilivyolingana vikiwa vimeunganishwa na `" + "`, ikifuatiwa na nukta mbili na nafasi (`": "`), na jumla ya bei.

Kwa mfano, ukipokea `"I'd like an oat latte with vanilla syrup and an extra shot please."`, rudisha `"oat latte + vanilla syrup + extra shot: $6.25"`

Vitu vinapaswa kuonekana kwa mpangilio walivyo kwenye menyu na jumla ya bei inapaswa kuwa na sehemu mbili za desimali kila wakati.

```py
def format_coffee_order(order):
    menu = [
        ("cold brew", 450),
        ("oat latte", 500),
        ("cappuccino", 475),
        ("espresso", 300),
        ("vanilla syrup", 75),
        ("caramel drizzle", 60),
        ("extra shot", 50),
        ("oat milk", 75),
        ("cream", 75),
    ]
    found = [(name, price) for name, price in menu if name in order.lower()]
    total = sum(price for _, price in found)
    items = " + ".join(name for name, _ in found)
    return f"{items}: ${total / 100:.2f}"
```
