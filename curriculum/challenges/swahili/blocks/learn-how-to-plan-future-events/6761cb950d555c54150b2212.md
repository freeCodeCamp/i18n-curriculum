# --questions--

## --text--

Huu ni ujumbe ambao Jake alitumia kwa wateja baada ya shambulio la ulaghai wa mtandao (phishing):

`We're currently addressing the issue and taking all necessary steps to secure the system.`  

`We've run a full system scan to identify hidden malware, and the IT support team is actively monitoring the situation.`  

`I'll provide you with regular updates to ensure transparency.`  

`Please let us know if you have further concerns.`  

Wateja walijibu:  

`We're very concerned about the security of our data. Can you confirm when this issue will be fully resolved?`

Hatua inayofuata ambayo Jake anarejelea katika ujumbe wake ni ipi?

# --explanation--

Ili kupata hatua inayofuata ambayo Jake anarejelea, angalia hatua maalum zilizotajwa katika ujumbe wake:  

`We've run a full system scan` inaonyesha hatua hii tayari imefanyika.  

`The IT support team is actively monitoring the situation` ni hatua inayofuata inayofanyika sasa ili kulinda mfumo.  

Jake bado hajataja kutatua tatizo au kutoa maelezo zaidi kuhusu shambulio la ulaghai wa mtandao.
