# --description--

لأعداد صحيحة $a$ و $b$، نُعرّف $D(a, b)$ كالمجال المحصور بين القطع المكافئ $y = x^2$ والخط $y = ax + b: D(a, b) = \\{ (x, y) | x^2 ≤ y ≤ ax + b \\}$.

يُعرّف $L(a, b)$ كعدد نقاط الشبكة المحتواة في $D(a, b)$. على سبيل المثال، $L(1, 2) = 8$ و $L(2, -1) = 1$.

نُعرّف أيضًا $S(N)$ كمجموع $L(a, b)$ لكل الأزواج ($a$, $b$) بحيث تكون مساحة $D(a, b)$ عددًا نسبيًا و $|a|,|b| ≤ N$.

يمكننا التحقق من أن $S(5) = 344$ و $S(100) = 26\\,709\\,528$.

احسب $S({10}^{12})$. قدّم إجابتك $\bmod {10}^8$.
