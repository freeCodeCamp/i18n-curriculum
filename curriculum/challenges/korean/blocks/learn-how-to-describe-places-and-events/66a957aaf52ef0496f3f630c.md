# --explanation--

비즈니스에서 `branch`는 더 큰 조직의 지역 사무소나 지점을 의미합니다. 예를 들어:

- `The company has a branch in Seattle.` - 회사가 본사와 별도로 시애틀에 작은 사무소나 지점을 가지고 있습니다.

`Headquarters`는 조직의 본사나 중앙 허브를 의미합니다. 예를 들어:

- `The company's headquarters is in New York.` - 이는 최고 경영진이 일하는 중앙 사무소가 뉴욕에 있음을 뜻합니다.

이 과제에서는 Lisa가 최고 경영진과 만나는 장소를 답하는 데 도움이 되도록 `branch`와 `headquarters`의 차이에 주목하세요.
