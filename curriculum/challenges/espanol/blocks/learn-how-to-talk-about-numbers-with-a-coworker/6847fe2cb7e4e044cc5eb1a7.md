# --explanation--

La forma correcta de escribir ⅗ en palabras es `three-fifths`. Usa este formato:

Numerador (en forma cardinal) + denominador (en plural si el numerador > 1). Por ejemplo:

- ⅖ = `two-fifths`

- ⅘ = `four-fifths`

Esto es útil cuando explicas datos o compartes detalles financieros en presentaciones orales.
