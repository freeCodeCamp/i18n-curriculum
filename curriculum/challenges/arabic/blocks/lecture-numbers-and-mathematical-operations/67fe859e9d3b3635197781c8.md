# --description--

الأعداد الصحيحة والأعداد العشرية هي أنواع البيانات الرقمية الأساسية في Python. يمكنك معها تخزين البيانات الرقمية وأداء العمليات الحسابية.

لننظر إلى ماهية الأعداد الصحيحة والأعداد العشرية، وكيفية إجراء العمليات الحسابية باستخدامها، وإلى عدة طرق يوفرها Python للعمل مع كلا النوعين.

الأعداد الصحيحة هي أعداد كاملة بدون نقاط عشرية، إما موجبة أو سالبة:

```python
my_int_1 = 56
my_int_2 = -4

print(type(my_int_1)) # <class 'int'>
print(type(my_int_2)) # <class 'int'>
```

إليك كيفية إجراء عملية جمع باستخدام الأعداد الصحيحة:

```python
my_int_1 = 56
my_int_2 = 12

sum_ints = my_int_1 + my_int_2
print('Integer Addition:', sum_ints) # Integer Addition: 68
```

إليك كيفية إجراء عملية طرح باستخدام الأعداد الصحيحة:

```python
my_int_1 = 56
my_int_2 = 12

# Subtraction
diff_ints = my_int_1 - my_int_2
print('Integer Subtraction:', diff_ints) # Integer Subtraction: 44
```

إليك كيفية إجراء عملية ضرب باستخدام الأعداد الصحيحة:

```python
my_int_1 = 12
my_int_2 = 4

# Multiplication
product_ints = my_int_1 * my_int_2
print('Integer Multiplication:', product_ints) # Integer Multiplication: 48
```

وهنا كيفية إجراء عملية قسمة باستخدام الأعداد الصحيحة:

```python
my_int_1 = 56
my_int_2 = 12

# Division
div_ints = my_int_1 / my_int_2
print('Division:', div_ints) # Division: 4.666666666666667
```

الأعداد العشرية هي أعداد موجبة أو سالبة تحتوي على نقاط عشرية، مثل `3.14`، `-0.5`، أو `0.0`.

```python
my_float_1 = -12.0
my_float_2 = 4.9

print(type(my_float_1)) # <class 'float'>
print(type(my_float_2)) # <class 'float'>
```

إليك عملية جمع باستخدام الأعداد العشرية:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_addition = my_float_1 + my_float_2
print('Float Addition:', float_addition) # Float Addition: 17.4
```

إليك عملية طرح باستخدام الأعداد العشرية:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_subtraction = my_float_2 - my_float_1
print('Float Subtraction:', float_subtraction) # Float Subtraction: 6.6
```

إليك عملية ضرب باستخدام الأعداد العشرية:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_multiplication = my_float_2 * my_float_1
print('Float Multiplication:', float_multiplication) # Float Multiplication: 64.80000000000001
```

وهنا عملية قسمة باستخدام الأعداد العشرية:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_division = my_float_2 / my_float_1
print('Float Division:', float_division) # Float Division: 2.222222222222222
```

إذا جمعت عددًا صحيحًا مع عدد عشري، يتحول الناتج افتراضيًا إلى عدد عشري:

```python
my_int = 56
my_float = 5.4

sum_int_and_float = my_int + my_float

print(sum_int_and_float) # 61.4
print(type(sum_int_and_float)) # <class 'float'>
```

هذا ينطبق أيضًا على العمليات الحسابية الأساسية الأخرى مثل الطرح والضرب والقسمة. إذا خلطت بين الأعداد الصحيحة والعشرية، يعيد Python عددًا عشريًا كنتيجة.

يمكنك أيضًا إجراء عمليات حسابية أكثر تعقيدًا مثل الحصول على باقي القسمة بين رقمين باستخدام عامل باقي القسمة، والقسمة الأرضية، والأسّية مع كل من الأعداد الصحيحة والعشرية.

عامل باقي القسمة (`%`) يعيد الباقي عند قسمة القيمة على اليسار على القيمة على اليمين:

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

mod_ints = my_int_1 % my_int_2
mod_floats = my_float_2 % my_float_1

print('Integer Modulo:', mod_ints) # Integer Modulo: 8
print('Float Modulo:', mod_floats) # Float Modulo: 1.1999999999999993
```

القسمة الأرضية تقسم رقمين وتعيد أكبر عدد صحيح أقل من أو يساوي الناتج. يتم ذلك باستخدام عامل الشرطتين الأماميتين (`//`):

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

floor_div_ints = my_int_1 // my_int_2
floor_div_floats = my_float_2 // my_float_1

print('Integer Floor Division:', floor_div_ints) # Integer Floor Division: 4
print('Float Floor Division:', floor_div_floats) # Float Floor Division: 2.0
```

الأسّية ترفع رقمًا إلى قوة رقم آخر، ويتم ذلك باستخدام عامل النجمتين (`**`):

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

exp_ints = my_int_1 ** my_int_2
exp_floats = my_float_1 ** my_float_2

print('Integer Exponentiation:', exp_ints) # Integer Exponentiation: 951166013805414055936
print('Float Exponentiation:',  exp_floats) # Float Exponentiation: 614787626.1765089
```

أحيانًا قد تلاحظ أن ناتج عملية تشمل أعدادًا عشرية يحتوي على أرقام عشرية أكثر مما توقعت. على سبيل المثال، مجموع `0.1 + 0.2` يساوي `0.30000000000000004` بدلًا من `0.3`.

يحدث هذا لأن الأرقام تُخزن بصيغة ثنائية، وبعض الكسور لا يمكن تمثيلها بدقة في النظام الثنائي. نتيجة لذلك، تُخزن كتقريبات محدودة، بنفس الطريقة التي لا يمكن بها تمثيل الكسر `1/3` بعدد محدود من الأرقام العشرية ويتم اقتطاعه بعد عدد معين من أرقامه اللانهائية (`0.33333...`).

هذا يؤدي إلى أخطاء تقريب صغيرة.

يوفر Python أيضًا دوال مدمجة لتحويل البيانات الرقمية أو السلاسل النصية إلى أعداد صحيحة أو عشرية.

الدالة `float()` تعيد عددًا عشريًا مُنشأً من الرقم المعطى:

```python
my_int_1 = 56
my_float_1 = float(my_int_1)

print(my_float_1)  # 56.0
print(type(my_float_1))  # <class 'float'>
```

الدالة `int()` تعيد عددًا صحيحًا مُنشأً من الرقم المعطى:

```python
my_float = 12.92563
my_int = int(my_float)

print(my_int)  # 12
print(type(my_int))  # <class 'int'>
```

يمكنك أيضًا استخدام نفس الدوال المدمجة لتحويل سلسلة نصية إلى عدد عشري أو صحيح:

```python
my_str_int = '45'
my_str_float = '7.8'

converted_int = int(my_str_int)
converted_float = float(my_str_float)

print(converted_int, type(converted_int))  # 45 <class 'int'>
print(converted_float, type(converted_float))  # 7.8 <class 'float'>
```

إليك بعض الطرق الأخرى التي يوفرها Python للعمل مع الأعداد الصحيحة والعشرية.

- `round()`: تقرب رقمًا إلى عدد معين من الأرقام العشرية. افتراضيًا، تقرب هذه الدالة إلى أقرب عدد صحيح، وتعيد رقمًا كاملاً بدون أرقام عشرية:
    

```python
my_int_1 = 4.798
my_int_2 = 4.253

rounded_int_1 = round(my_int_1)
rounded_int_2 = round(my_int_2, 1)

print(rounded_int_1) # 5
print(rounded_int_2) # 4.3
```

- `abs()`: تعيد القيمة المطلقة لرقم،
    

```python
num = -15

absolute_value = abs(num)
print(absolute_value) # 15
```

- `pow()`: ترفع رقمًا إلى قوة رقم آخر أو تنفذ الأسّية المودولية.
    

```python
result_1 = pow(2, 3)  # Equivalent to 2 ** 3
print(result_1)  # 8

result_2 = pow(2, 3, 5)  # (2 ** 3) % 5
print(result_2)  # 3
```
