# --interactive--

## Sets katika JavaScript

- `Set` ni chaguo iliyo ndani kwa ajili ya kusimamia mkusanyiko wa data.
- Sets huhakikisha kwamba kila thamani ndani yake inaonekana mara moja tu, na hii inafanya iwe muhimu kwa kuondoa rudufu kutoka kwenye safu ya data au kushughulikia makusanyo ya thamani tofauti.
- Unaweza kuunda `Set` kwa kutumia kijenzi cha `Set()`:

:::interactive_editor

```js
const set = new Set([1, 2, 3, 4, 5]);
console.log(set); // Set { 1, 2, 3, 4, 5 }
```

:::

- Sets zinaweza kudhibitiwa kwa kutumia njia hizi:

  - `add()`: Inaongeza kipengele kipya kwenye `Set`.
  - `delete()`: Hutoa kipengele kutoka kwenye `Set`.
  - `has()`: Hukagua kama kipengele kipo ndani ya `Set`.
  - `clear()`: Hutoa vipengele vyote kutoka kwenye `Set`.
  - `keys()` na `values()`: Zote hurudisha `SetIterator` inayoshikilia thamani za `Set`. Ni sawa kwa sababu `keys()` ni jina lingine la `values()`.
  - `forEach()`: kwa kuzunguka thamani za `Set`.

## Weaksets katika JavaScript

- `WeakSet` ni mkusanyiko wa vitu unaokuruhusu kuhifadhi vitu vilivyo dhaifu.

## Sets dhidi ya WeakSets

- Tofauti na Sets, `WeakSet` haitegemei aina za msingi kama nambari au mfuatano wa herufi.
- `WeakSet` huhifadhi vitu tu, na rejea kwa vitu hivyo ni "dhaifu," ikimaanisha kwamba ikiwa kitu hakitumiki mahali pengine katika msimbo wako, hutolewa moja kwa moja ili kuachilia kumbukumbu.

## Maps katika JavaScript

- `Map` ni kitu kilichojengwa ndani kinachoshikilia jozi za ufunguo-thamani kama vile kitu cha kawaida.
- Maps zinatofautiana na vitu vya kawaida vya JavaScript kwa uwezo wao wa kuruhusu funguo za aina yoyote, ikiwa ni pamoja na vitu na kitendakazi.
- `Map` hutoa utendaji bora zaidi kuliko kitu cha kawaida linapokuja suala la kuongeza na kuondoa mara kwa mara jozi za ufunguo-thamani.
- Unaweza kuunda `Map` kwa kutumia kijenzi cha `Map()`:

:::interactive_editor

```js
const map = new Map([
  ['flower', 'rose'],
  ['fruit', 'apple'],
  ['vegetable', 'carrot']
]);
console.log(map); // Map(3) { 'flower' => 'rose', 'fruit' => 'apple', 'vegetable' => 'carrot' }
```

:::

- Maps zinaweza kudhibitiwa kwa kutumia njia hizi:

  - `set()`: Inaongeza jozi mpya ya ufunguo-thamani kwenye `Map`.
  - `get()`: Inapata thamani ya ufunguo kutoka kwenye `Map`.
  - `delete()`: Hutoa jozi ya ufunguo-thamani kutoka kwenye `Map`.
  - `has()`: Hukagua kama ufunguo upo ndani ya `Map`.
  - `clear()`: Hutoa jozi zote za ufunguo-thamani kutoka kwenye `Map`.

Kumbuka kwamba zote Maps na Sets zina sifa ya `size` inayorudisha idadi ya vipengele vya kipekee ndani yao.

## WeakMaps katika JavaScript

- `WeakMap` ni mkusanyiko wa jozi za ufunguo-thamani kama `Map`, lakini na rejea dhaifu kwa funguo. Funguo lazima ziwe vitu na thamani zinaweza kuwa chochote unachotaka.

## Maps dhidi ya WeakMaps

- WeakMaps ni sawa na WeakSets kwa kuwa huhifadhi vitu tu na rejea kwa vitu hivyo ni "dhaifu".
