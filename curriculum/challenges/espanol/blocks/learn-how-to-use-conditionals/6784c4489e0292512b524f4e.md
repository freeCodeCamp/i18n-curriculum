# --explanation--

Lisa está usando el `Second Conditional` para hablar de una situación hipotética.

El `Second Conditional` se usa para hablar de situaciones irreales o poco probables en el presente o futuro. Sigue la estructura `if` + `Present Simple` o `Past Continuous` (para la condición) y `might / would / could` + verbo base (para el resultado). Por ejemplo:

- `If I had more time, I would learn a new programming language.` - Esto describe una situación hipotética donde tener más tiempo llevaría a aprender.

- `If the server were faster, it could handle more users.` - El servidor no es rápido ahora, pero si lo fuera, podría soportar más usuarios.
