# --description--

`Security sweeps` は、通常、安全とセキュリティの目的でエリアや建物内で行われる詳細な検査や捜索です。潜在的なリスク、不正なアクセス、危険な物品がないかを確認します。例えば：

`Security officers perform security sweeps to find any hidden dangers.` - これは、警備員が見えにくい危険なものがないかエリアを注意深くチェックすることを意味します。
