# --description--

Sotto l'elemento `p`, crea un elemento `div` con la classe `btn-container`.

All'interno dell'elemento `div`, crea un elemento `button`.

Questo `button` dovrebbe avere un `id` di `happy-btn`, una classe `emoji-btn` e un `aria-label` di `Happy face emoji`.

# --hints--

Crea un nuovo elemento `div` sotto l'elemento `p`.

```js
assert.exists(document.querySelector("p + div"));
```

Il nuovo elemento `div` dovrebbe avere una classe `btn-container`.

```js
assert.equal(document.querySelector("p + div")?.className, "btn-container");
```

Dovrebbe esserci un elemento `button` all'interno dell'elemento `div`.

```js
assert.exists(document.querySelector("div > button"));
```

L'elemento `button` dovrebbe avere un `id` di `happy-btn`.

```js
assert.equal(document.querySelector("div > button")?.id, "happy-btn");
```

L'elemento `button` dovrebbe avere un `class` di `emoji-btn`.

```js
assert.equal(document.querySelector("div > button")?.className, "emoji-btn");
```

L'elemento `button` dovrebbe avere un `aria-label` di `Happy face emoji`.

```js
assert.equal(document.querySelector("div > button")?.getAttribute("aria-label"), "Happy face emoji");
```
