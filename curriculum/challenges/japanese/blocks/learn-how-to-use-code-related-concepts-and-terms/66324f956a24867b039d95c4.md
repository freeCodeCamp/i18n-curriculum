# --description--

`should` は助言や提案を表すモーダル動詞です。動詞の原形が続きます。例えば、文中の `You should try that restaurant` では、`should` の後に動詞の原形である `try` が続いています。

動詞 `pick` は、選択肢の中から何かを選ぶことを意味します。プログラミングの文脈では、`picking a data type` は、整数、文字列、ブール値など、その時の作業に最も適したデータ型を選ぶことを指します。

`Match` は、2つのものが適合していることを確認することを意味します。例えば、`This key matches the lock.` は鍵が錠に適していることを示しています。
