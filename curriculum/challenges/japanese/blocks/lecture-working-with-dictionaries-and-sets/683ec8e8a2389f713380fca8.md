# --description--

辞書のキーと値のペアにアクセスして処理する必要がある場合、辞書をループ処理できます。これは値を更新したり、何らかのロジックを適用したりするのに役立ちます。

いくつかの使える手法を見てみましょう。

すべての製品とその価格を関連付ける`products`辞書があるとします。

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,
    'Headphones': 70,
}
```

すべての製品に20％の割引を提供したい場合、すべてのキーと値のペアをループ処理して価格を変更できます。

`.values()`、`.keys()`、`.items()`メソッドはこれらの手法に不可欠です。前のレッスンで簡単に説明しました。

これらは辞書の値、キー、キーと値のペアのビューオブジェクトを返します。これらのビューオブジェクトは`for`ループで要素を反復するために使えます。

たとえば、辞書のすべての値を次のように繰り返せます。

`for`、ループ変数（この場合は`price`）、`products.values()`を書き、`products`辞書のすべての値を取得し、コロンを書いてループ本体に進みます。ここで値に任意のロジックを適用できます。この例では値を出力しています。

ループ変数は各値を1回ずつ受け取ります。

```python
for price in products.values():
    print(price)
```

出力は次の通りです。値が1つずつコンソールに出力されているのがわかります。

```md
990
600
250
70
```

これは`.keys()`でもまったく同じです。辞書のキーを繰り返す必要がある場合は、`products.keys()`または`products`を直接繰り返し、ループ変数に説明的な名前を付けるだけです。

```python
for product in products.keys():
    print(product)

# Or

for product in products:
    print(product)
```

出力は次の通りです。各キーが1つずつコンソールに出力されています。

```md
Laptop
Smartphone
Tablet
Headphones
```

キーと対応する値の両方を同時に繰り返す必要がある場合もまったく同じです。`products.items()`を繰り返すだけです。

```python
for product in products.items():
    print(product)
```

出力は次の通りです。キーと対応する値を含む個別のタプルが得られます。

```md
('Laptop', 990)
('Smartphone', 600)
('Tablet', 250)
('Headphones', 70)
```

キーと値を別々のループ変数に格納したい場合は、それらを定義してカンマで区切るだけです。ループ本体でそれらを使えます。

ここでは`product`ループ変数と`price`ループ変数を定義しています。それぞれ対応する値を保持します。順序は重要で、最初にキー、次に値を定義してください。

```python
for product, price in products.items():
    print(product, price)
```

出力は次の通りです。横並びで出力していますが、コード内で必要に応じてこれらの値を使えます。

```md
Laptop 990
Smartphone 600
Tablet 250
Headphones 70
```

これらを理解したところで、最初の例に戻りましょう。20％の割引を提供したい場合、各価格に`0.8`を乗算して、その製品キーの値として再代入します。

整数で扱いたい場合は結果を切り捨てることもできます。

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,
    'Headphones': 70,
}

for product, price in products.items():
    products[product] = round(price * 0.8)

print(products)
```

辞書を出力すると、割引価格のキーと値のペアが得られます。

```python
{
    'Laptop': 792, 
    'Smartphone': 480, 
    'Tablet': 200, 
    'Headphones': 56
}
```

最後に、カウンターを追跡しながらキーと値のペアを繰り返す必要がある場合は、`enumerate()`関数を呼び出せます。このカウンターはループ内の要素の「インデックス」や「カウント」として機能します。

この関数は`enumerate`オブジェクトを返し、各キーと値のペアに整数を割り当てます。カウンターは任意の数から開始できますが、デフォルトでは0から始まります。

ここでは`products`辞書のキーを繰り返しています。

```python
for product in enumerate(products):
    print(product)
```

しかし`enumerate()`関数は各キーに整数を割り当てるため、整数とキーのタプルが得られます。

出力は次の通りです。

```md
(0, 'Laptop')
(1, 'Smartphone')
(2, 'Tablet')
(3, 'Headphones')
```

必要に応じて、これらの値を別々のループ変数に代入できます。ここでは2つのループ変数（`index`と`product`）があります。`enumerate()`を扱う際によく見る使い方です。

```python
for index, product in enumerate(products):
    print(index, product)
```

値を繰り返す場合は、`products`を`products.values()`に置き換えられます。

```python
for price in enumerate(products.values()):
    print(price)
```

出力はインデックスと価格がそれぞれのタプルに含まれます。

```md
(0, 990)
(1, 600)
(2, 250)
(3, 70)
```

これらも別々のループ変数に代入できます。

```python
for index, price in enumerate(products.values()):
    print(index, price)
```

出力は次の通りです。コード内で必要に応じて使えます。

```md
0 990
1 600
2 250
3 70
```

`products.items()`を使うと、「インデックス」や「カウンター」に加えてキーと値のペア全体を取得できます。

```python
for index, product in enumerate(products.items()):
    print(index, product)
```

この例では、インデックスの後に対応するキーと値のタプルが続きます。

```md
0 ('Laptop', 990)
1 ('Smartphone', 600)
2 ('Tablet', 250)
3 ('Headphones', 70)
```

カウントの初期値をカスタマイズしたい場合は、`enumerate()`に2番目の引数を渡せます。たとえば、ここではカウントを1から開始しています。

```python
for index, product in enumerate(products.items(), 1):
    print(index, product)
```

出力でこの変更が確認できます。最初の整数が0ではなく1になっています。

```md
1 ('Laptop', 990)
2 ('Smartphone', 600)
3 ('Tablet', 250)
4 ('Headphones', 70)
```

これまで見たどのバリエーションでも動作します。初期値を2番目の引数として渡すだけです。

辞書をループ処理する手法はたくさんあります。これらは一般的な方法の一部で、プロジェクトに最適なものを選ぶ必要があります。
