# --description--

给定一个数组，表现你流媒体服务购物车中的电影，以及一个表示你的订阅等级的字串，返回电影的总费用。

购物车中的每一项是一个对象，包含一个 `"format"`（`"HD"` 或 `"4K"`）和一个 `"type"`（`"rent"` 或 `"buy"`）。它们的费用如下：

| | `"rent"` | `"buy"` |
| - | - | - |
| `"HD"` | $3.99 | $12.99 |
| `"4K"` | $5.99 | $19.99 |

应用以下订阅等级折扣：

- `"none"`：全价
- `"basic"`：九折
- `"premium"`：七五折

返回总费用，四舍五入到小数点后两位，格式为 `"$D.CC"`。

```js
function getStreamingBill(cart, subscription) {
  const pricing = {
    HD: { rent: 3.99, buy: 12.99 },
    "4K": { rent: 5.99, buy: 19.99 },
  };

  const discounts = {
    none: 1,
    basic: 0.9,
    premium: 0.75,
  };

  const multiplier = discounts[subscription];

  const total = cart.reduce((sum, item) => {
    const price = pricing[item.format][item.type];
    return sum + price * multiplier;
  }, 0);

  return `$${total.toFixed(2)}`;
}
```
