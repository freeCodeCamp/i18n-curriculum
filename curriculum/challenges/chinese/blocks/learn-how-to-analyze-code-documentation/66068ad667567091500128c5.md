# --description--

`library` 就像一个装满代码的大书架，程序员可以使用。把它想象成一本包含许多食谱（函数）的烹饪书，你可以用来烹饪（编写）不同的菜肴（项目）。

例如，编程中的一个 `library` 中可能有一个函数能绘制图表。 这样你就不用从头开始写代码，只需使用该函数，就像照着配方做蛋糕一样。
