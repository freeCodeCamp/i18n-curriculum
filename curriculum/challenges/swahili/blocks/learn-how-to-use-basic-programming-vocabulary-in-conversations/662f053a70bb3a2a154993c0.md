# --description--

`extension` ni sehemu ya programu inayoongeza uwezo maalum kwa programu kubwa zaidi ya kompyuta. `Extensions` inaweza kutoa kitendakazi kipya au kuboresha vipengele vilivyopo.

Kwa mfano, kivinjari `extension` kinaweza kuongeza kitendakazi maalum kwa kivinjari chako cha mtandao, kama kuzuia matangazo au kusimamia nywila.
