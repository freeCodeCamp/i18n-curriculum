# --explanation--

`Going to` 用來談論對未來的計畫行動或意圖。例如：

- `We are going to upgrade the system next month.`－這表示升級已經計劃好並且將在不久後進行。
  
- `She is going to learn SQL next year.`－這表示她打算在未來學習 SQL。

在這句話中：

`We're` 是 `we are` 的縮寫。

`Going to` 表示該動作計劃在不久的將來進行。

`Need` 是表示某事必要或必須的基本動詞。
