# --description--

두 양의 정수 $a$와 $b$에 대해, 울람 수열 $U(a,b)$는 ${U{(a,b)}\_1} = a$, ${U{(a,b)}\_2} = b$로 정의되고, $k > 2$일 때 ${U{(a,b)}\_k}$는 ${U{(a,b)}\_{(k-1)}}$보다 큰 정수 중에서 $U(a,b)$의 서로 다른 이전 항 두 개의 합으로 정확히 한 가지 방법으로만 나타낼 수 있는 가장 작은 정수입니다.

예를 들어, 수열 $U(1,2)$는 다음과 같이 시작합니다.

$$1, 2, 3 = 1 + 2, 4 = 1 + 3, 6 = 2 + 4, 8 = 2 + 6, 11 = 3 + 8$$

5는 $5 = 1 + 4 = 2 + 3$로 이전 항 두 개의 합으로 두 가지 방법으로 나타낼 수 있으므로 포함되지 않고, 마찬가지로 $7 = 1 + 6 = 3 + 4$도 포함되지 않습니다.

$k = {10}^{11}$일 때, $2 ≤ n ≤ 10$에 대해 $\sum {U(2, 2n + 1)_k}$를 구하세요.
