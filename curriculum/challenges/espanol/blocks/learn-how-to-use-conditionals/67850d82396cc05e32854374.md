# --explanation--

Una **tag question** es una pregunta corta añadida al final de una oración, usada para confirmar información o invitar a la conformidad. Se forma por:

- Tomando el verbo auxiliar (por ejemplo, `would`, `is`, `has`, `will`) de la oración principal.

- Usando la forma opuesta: de positivo a negativo o de negativo a positivo.

- Agregando el pronombre apropiado.

En la oración de Lisa:

- La cláusula principal es `It would affect a lot of tech companies` (positivo).

- La etiqueta es `wouldn't it?` (forma negativa de `would`).

Esto confirma la afirmación e invita a estar de acuerdo o a dar retroalimentación.
