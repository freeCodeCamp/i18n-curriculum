# --description--

字 `earlier` 用來表示在目前時刻或正在討論的時間之前的時間。這就像是在說 `before now` 或 `some time ago`。

例如，`If you finished your work earlier, you can relax now` 意思是如果你在某個時間點之前完成了你的工作，你現在有時間放鬆。
