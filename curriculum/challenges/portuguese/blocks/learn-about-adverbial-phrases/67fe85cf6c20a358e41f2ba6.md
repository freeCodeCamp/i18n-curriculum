# --explanation--

`Switch` significa mudar ou alterar de um estado, sistema ou método para outro.Por exemplo:

`We should switch to using email for all important communications.` – Isso significa mudar o método de comunicação para email.

Neste contexto, Bob está sugerindo alterar a frequência das atualizações. 

`Unless` seria o mesmo que `except if`.É usado para introduzir uma exceção ou condição que deve ser verdadeira para que a ação principal não aconteça.Por exemplo:

`I will attend the meeting unless I have a conflicting appointment.` – Isso significa que você irá à reunião a menos que haja algo que o impeça, como outro compromisso.

Você poderia dizer o mesmo usando `if` e uma declaração negativa com `I will attend the meeting if I don't have a conflicting appointment.`
