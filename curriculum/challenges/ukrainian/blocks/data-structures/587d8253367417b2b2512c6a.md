# --description--

Масиви — це об’єкти JavaScript, які можуть містити багато різних елементів.

```js
var complexArr = [1, 5, "2", "Word", {"name": "James"}];
```

В основному, що відбувається «під капотом», так це те, що ваш браузер автоматично виділяє потрібний обсяг пам’яті для цього масиву. Він також змінюється за потреби, якщо ви додаєте або видаляєте дані.

Однак у світі високої продуктивності та різних типів елементів іноді потрібно точніше визначати, скільки пам’яті виділяти масиву.

<dfn>Типізовані масиви</dfn> — це відповідь на цю проблему. Тепер ви можете вказати, скільки пам’яті хочете виділити масиву. Нижче наведено базовий огляд різних типів масивів і розмір у байтах для кожного елемента в цьому масиві.

<table><tbody><tr><th>Тип</th><th>Розмір кожного елемента в байтах</th></tr><tr><td><code>Int8Array</code></td><td>1</td></tr><tr><td><code>Uint8Array</code></td><td>1</td></tr><tr><td><code>Uint8ClampedArray</code></td><td>1</td></tr><tr><td><code>Int16Array</code></td><td>2</td></tr><tr><td><code>Uint16Array</code></td><td>2</td></tr><tr><td><code>Int32Array</code></td><td>4</td></tr><tr><td><code>Uint32Array</code></td><td>4</td></tr><tr><td><code>Float32Array</code></td><td>4</td></tr><tr><td><code>Float64Array</code></td><td>8</td></tr></tbody></table>

Існує два способи створення таких масивів. Один із способів — створити його безпосередньо. Нижче показано, як створити масив довжиною 3 `Int16Array`.

```js
var i8 = new Int16Array(3);
console.log(i8);
// Returns [0, 0, 0]
```

Ви також можете створити <dfn>буфер</dfn>, щоб призначити, скільки даних (у байтах) ви хочете, щоб масив займав. **Примітка**  
Щоб створювати типізовані масиви за допомогою буферів, потрібно призначити кількість байтів, кратну байтам, наведеним вище.

```js
// Create same Int16Array array differently
var byteSize = 6; // Needs to be multiple of 2
var buffer = new ArrayBuffer(byteSize);
var i8View = new Int16Array(buffer);
buffer.byteLength; // Returns 6
i8View.byteLength; // Returns 6
console.log(i8View); // Returns [0, 0, 0]
```

<dfn>Буфери</dfn> — це об’єкти загального призначення, які просто містять дані. Ви не можете отримати до них доступ звичайним способом. Щоб отримати доступ, спочатку потрібно створити <dfn>вигляд</dfn>.

```js
i8View[0] = 42;
console.log(i8View); // Returns [42, 0, 0]
```

**Примітка**  
Типізовані масиви не мають деяких методів, які є у традиційних масивів, таких як `.pop()` або `.push()`. Типізовані масиви також не проходять `Array.isArray()`, який перевіряє, чи є щось масивом. Хоча вони простіші, це може бути перевагою для менш складних JavaScript-двигунів при їх реалізації.

# --instructions--

Спочатку створіть `buffer` розміром 64 байти. Потім створіть типізований масив `Int32Array` з виглядом на нього під назвою `i32View`.

# --hints--

Ваш `buffer` має бути розміром 64 байти.

```js
assert(buffer.byteLength === 64);
```

Ваш вигляд `i32View` на ваш буфер має бути розміром 64 байти.

```js
assert(i32View.byteLength === 64);
```

Ваш вигляд `i32View` на ваш буфер має бути довжиною 16 елементів.

```js
assert(i32View.length === 16);
```
