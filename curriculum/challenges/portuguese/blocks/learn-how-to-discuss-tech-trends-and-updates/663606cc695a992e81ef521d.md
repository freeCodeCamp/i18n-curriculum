# --description--

A expressão `That/it sounds` é usada para descrever como algo parece ou se apresenta, frequentemente com base na informação ouvida ou na impressão que causa.Por exemplo, `That sounds like a great idea!` Isso indica que a ideia parece boa com base no que foi ouvido.

Da mesma forma, `That/it looks` pode ser usado para descrever como algo aparece com base na aparência, como em `It looks beautiful.`

A palavra `convenient` é um adjetivo que descreve algo que é adequado para as necessidades de alguém, fácil de usar ou que ajuda a fazer as coisas de forma mais fácil e rápida.Frequentemente se refere a algo que torna uma tarefa menos difícil ou economiza tempo.

Por exemplo, `Online shopping is convenient because you can do it from home.`
