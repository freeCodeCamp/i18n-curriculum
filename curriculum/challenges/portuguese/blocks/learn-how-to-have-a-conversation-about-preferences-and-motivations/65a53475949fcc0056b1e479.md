# --description--

Projetos `Open-source` são quando o código-fonte é disponibilizado gratuitamente e pode ser modificado ou compartilhado. Por exemplo, o navegador web Firefox é um projeto de código aberto, o que significa que qualquer pessoa pode ver como ele foi feito e pode ajudar a melhorá-lo.

`To work on` algo, especialmente com a preposição `on`, indica colocar esforço em uma tarefa ou projeto ao longo do tempo. Como quando você diz, `I'm working on a new app`, significa que você está no processo de desenvolvê-lo.
