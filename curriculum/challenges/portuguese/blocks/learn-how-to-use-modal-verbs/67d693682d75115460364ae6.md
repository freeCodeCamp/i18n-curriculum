# --explanation--

Um `critical bug` é um erro grave no software que afeta a funcionalidade e deve ser corrigido imediatamente.Por exemplo:

- `We need to fix the critical bugs before launching the app.` - Os problemas mais graves devem ser resolvidos antes do lançamento.

- `A critical bug caused the website to crash.` - Um erro sério fez o website parar de funcionar.
