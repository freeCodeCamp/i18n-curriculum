# --explanation--

`To struggle` significa ter dificuldade com algo.Por exemplo:

- `The program is struggling to load.` - Isso significa que o programa está tendo dificuldade para carregar no momento.

O tempo `Present Continuous` é usado para descrever ações que estão **acontecendo agora mesmo**.

No exemplo acima, observe que ao formar o tempo `Present Continuous` com o verbo `struggle`, o `e` final é removido, resultando em `struggling`.Esta é uma regra comum para verbos que terminam em `e`.

Na frase de Tom, ele usa o tempo `Present Continuous` para descrever sua dificuldade contínua com IDEs (Integrated Development Environments).

Você deve adicionar `-ing` a `struggle` porque a ação está acontecendo agora.
