# --explanation--

`To be better` 表示某些不同的東西可能是一種改進。例如：

`Wouldn't it be better to start earlier and avoid last-minute stress?`－這表示提早開始可能會是一種改進。

`Differently` 意味著以不同的方式。它描述 **如何** 做某事。這與用來描述事物、人或情況的狀態的 `different` 形成對比。其他範例：

- `We need a different plan.` - 這裡，`different` 描述該計劃。

- `We need to plan differently.` - 這裡的描述是關於如何以不同的方式來規劃某件事。
