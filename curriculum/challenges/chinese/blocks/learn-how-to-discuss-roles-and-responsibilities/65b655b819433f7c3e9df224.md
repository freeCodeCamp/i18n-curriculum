# --description--

短语 `word is` 用于分享你从别人那里听到的信息或谣言。它类似于说 `people say` 或 `I've heard`。

例如，如果有人说，`Word is that the new update will make the program faster`，他们的意思是他们听说新更新将提升程序的速度。
