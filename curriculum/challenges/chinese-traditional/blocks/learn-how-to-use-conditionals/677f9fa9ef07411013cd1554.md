# --explanation--

這句話使用 `First Conditional` 來描述因果關係。

片語 `if there's a software crash` 描述了條件。結果 `it can be because of incompatible drivers or outdated software` 表示當機的可能原因。

條件語句中的主語句通常使用情態動詞，例如 `can`、`could`、`may`、`should` 等。<dfn>例如</dfn>：

- `If the system overheats, it can shut down automatically.`－過熱總是有可能導致系統關閉。

- `If users don't follow instructions, errors can occur.` - 不遵守指引總是會產生錯誤的可能性。
