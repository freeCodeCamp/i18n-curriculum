要理解正确答案，请听索菲说：`It would have caught the error early on and saved you some time.`

她使用了短语 `would have caught`，这表明 Mark 本可以更早发现该误差，但这 **没有** 发生。这里有另一个例子： 

`If you had run the test suite, it would have found the missing bracket.` —— 这意味着运行测试套件可以帮助你及早查找问题，避免以后浪费时间。
