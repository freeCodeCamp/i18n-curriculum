# --description--

정수 $a$와 $b$에 대해, $D(a, b)$를 포물선 $y = x^2$와 직선 $y = ax + b$로 둘러싸인 영역으로 정의합니다: $D(a, b) = \\{ (x, y) | x^2 ≤ y ≤ ax + b \\}$.

$L(a, b)$는 $D(a, b)$에 포함된 격자점의 개수로 정의합니다. 예를 들어, $L(1, 2) = 8$이고 $L(2, -1) = 1$입니다.

또한 $S(N)$을 $D(a, b)$의 영역이 유리수이고 $|a|,|b| ≤ N$인 모든 쌍 ($a$, $b$)에 대해 $L(a, b)$의 합으로 정의합니다.

$S(5) = 344$이고 $S(100) = 26\\,709\\,528$임을 확인할 수 있습니다.

$S({10}^{12})$를 구하세요. 답은 $\bmod {10}^8$로 제출하세요.
