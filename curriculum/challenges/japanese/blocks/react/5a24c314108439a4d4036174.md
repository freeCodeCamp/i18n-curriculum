# --description--

`state` を設定および更新することに加えて、コンポーネントクラスのメソッドを定義することもできます。クラスメソッドは通常、メソッドのスコープ内でクラスのプロパティ（`state` や `props` など）にアクセスできるように、`this` キーワードを使う必要があります。クラスメソッドが `this` にアクセスできるようにする方法はいくつかあります。

一般的な方法の一つは、コンストラクタ内で `this` を明示的にバインドすることで、コンポーネントが初期化されるときに `this` がクラスメソッドにバインドされるようにすることです。前のチャレンジでは、コンストラクタ内で `this.handleClick = this.handleClick.bind(this)` を `handleClick` メソッドに使っていたことに気づいたかもしれません。そして、クラスメソッド内で `this.setState()` のような関数を呼び出すと、`this` はクラスを指し、`undefined` にはなりません。

**注意:** `this` キーワードは JavaScript の中でも最も混乱しやすい部分の一つですが、React では重要な役割を果たします。ここでの動作は完全に通常のものですが、これらのレッスンは `this` の詳細なレビューを行う場所ではないため、上記がわかりにくい場合は他のレッスンを参照してください。

# --instructions--

コードエディタにはテキストを管理する `state` を持つコンポーネントがあります。また、テキストを `You clicked!` に設定できるメソッドもあります。しかし、そのメソッドは未定義の `this` キーワードを使っているため動作しません。コンポーネントのコンストラクタ内で `this` を `handleClick()` メソッドに明示的にバインドして修正してください。

次に、レンダリングメソッド内の `button` 要素にクリックハンドラを追加してください。ボタンがクリックイベントを受け取ったときに `handleClick()` メソッドを起動するようにします。`onClick` ハンドラに渡すメソッドは、JavaScript として直接解釈される必要があるため、波括弧（中括弧）が必要なことを覚えておいてください。

上記のステップを完了すると、ボタンをクリックして `You clicked!` を見ることができるはずです。

# --hints--

`MyComponent` は、ボタンと `h1` 要素の2つの要素を順にラップする `div` 要素を返す必要があります。

```js
assert(
  Enzyme.mount(React.createElement(MyComponent)).find('div').length === 1 &&
    Enzyme.mount(React.createElement(MyComponent))
      .find('div')
      .childAt(0)
      .type() === 'button' &&
    Enzyme.mount(React.createElement(MyComponent))
      .find('div')
      .childAt(1)
      .type() === 'h1'
);
```

`MyComponent` の state は、キーと値のペア `{ text: "Hello" }` で初期化する必要があります。

```js
assert(
  Enzyme.mount(React.createElement(MyComponent)).state('text') === 'Hello'
);
```

`button` 要素をクリックすると、`handleClick` メソッドが実行され、state の `text` が `You clicked!` に設定されます。

```js
  const waitForIt = (fn) =>
    new Promise((resolve, reject) => setTimeout(() => resolve(fn()), 250));
  const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
  const first = () => {
    mockedComponent.setState({ text: 'Hello' });
    return waitForIt(() => mockedComponent.state('text'));
  };
  const second = () => {
    mockedComponent.find('button').simulate('click');
    return waitForIt(() => mockedComponent.state('text'));
  };
  const firstValue = await first();
  const secondValue = await second();
  assert(firstValue === 'Hello' && secondValue === 'You clicked!');
```

# --solutions--

```jsx
class MyComponent extends React.Component {
  constructor(props) {
    super(props);
    this.state = {
      text: "Hello"
    };
    this.handleClick = this.handleClick.bind(this);
  }
  handleClick() {
    this.setState({
      text: "You clicked!"
    });
  }
  render() {
    return (
      <div>
        <button onClick = {this.handleClick}>Click Me</button>
        <h1>{this.state.text}</h1>
      </div>
    );
  }
};
```
