# --description--

Strings, die mit Backticks definiert werden, heißen Template-Literale. Sie sind eine neue Möglichkeit, Strings in JavaScript zu definieren. Sie erlauben es Ihnen, Ausdrücke innerhalb des Strings einzubetten. Dies geschieht, indem Sie den Ausdruck in `${}` einschließen.

Zum Beispiel führt der folgende Code:

```javascript
let name = "John";
let age = 25;
let greeting = `Hello, my name is ${name} and I am ${age} years old.`;
```

dazu, dass die Variable `greeting` den String „Hello, my name is John and I am 25 years old.“ enthält.

# --questions--

## --text--

Welche der folgenden Möglichkeiten ist der korrekte Weg, ein Template-Literal in JavaScript zu definieren?

## --answers--

``` `Hello, my name is ${name} and I am ${age} years old.` ```

---

``` `Hello, my name is `name` and I am `age` years old.` ```

---

``` `Hello, my name is {name} and I am {age} years old.` ```

---

``` `Hello, my name is $name and I am $age years old.` ```
