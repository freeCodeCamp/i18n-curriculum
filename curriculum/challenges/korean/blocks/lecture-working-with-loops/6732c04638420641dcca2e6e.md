# --interactive--

`for...of` 루프는 반복 가능한 값들을 순회할 때 사용합니다. 반복 가능한 예로는 배열과 문자열이 있습니다.

다음은 `for...of` 루프의 기본 구문입니다:

```js
for (variable of iterable) {
  // code block to be executed
}
```

예제의 변수는 순회 중인 반복 가능한 현재 값을 나타냅니다.

숫자 배열이 있다면, 변수는 배열의 현재 숫자가 됩니다. 문자열이 있다면, 변수는 문자열의 현재 문자가 됩니다.

`for...of` 루프가 어떻게 작동하는지 더 잘 이해할 수 있도록 몇 가지 예제를 살펴보겠습니다.

첫 번째 예제에서는 숫자 배열이 있고, 각 숫자를 순회하며 콘솔에 출력하려고 합니다.

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

:::

현재 배열의 숫자를 나타내는 `num`라는 변수를 만들었습니다. 첫 번째 반복에서 `num`는 `1`가 되고, 두 번째 반복에서 `num`는 `2`가 되는 식입니다.

루프 내부에서는 현재 숫자를 콘솔에 출력하고 있습니다.

다음 예제는 문자열이 있고, 각 문자를 순회하며 콘솔에 출력하려고 합니다.

:::interactive_editor

```js
const str = 'freeCodeCamp';

for (let char of str) {
  console.log(char);
}
```

:::

이 예제에서는 문자열의 현재 문자를 나타내는 `char`라는 변수를 만들었습니다.

각 반복마다 루프는 현재 문자를 콘솔에 출력합니다.

`let` 루프에서 변수를 선언할 때 `const` 또는 `for...of`를 사용할 수 있다는 점을 기억하세요.

하지만 `const`를 사용할 경우, 루프 내부에서 변수의 값이 변경되지 않아야 합니다. 만약 변경하면 오류가 발생합니다.

다음은 `const`를 사용했을 때 오류가 발생하는 예제입니다:

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
  num = num + 1; // This will cause an error
}
```

이 예제에서는 루프 내부에서 `num`의 값을 변경하려고 합니다. `num`를 `const`로 선언했기 때문에 오류가 발생합니다. 따라서 루프 내부에서 변수 값을 변경해야 한다면 `let`를 사용하세요.

마지막으로 객체 배열을 다루는 예제를 살펴보겠습니다.

:::interactive_editor

```js
const people = [
  { name: 'John', age: 30 },
  { name: 'Jane', age: 25 },
  { name: 'Jim', age: 40 }
];

for (const person of people) {
  console.log(`${person.name} is ${person.age} years old`);
}
```

:::

이 예제에서는 `people`라는 객체 배열이 있습니다. 각 객체는 `name`와 `age` 속성을 가지고 있습니다.

배열을 순회할 때 `person`라는 변수를 만들어 현재 배열의 객체를 나타냅니다.

루프 내부에서는 콘솔에 메시지를 출력합니다.

첫 번째 메시지는 `John is 30 years old`, 두 번째 메시지는 `Jane is 25 years old`, 세 번째 메시지는 `Jim is 40 years old`가 됩니다.

`for...of` 루프는 배열이나 문자열 같은 반복 가능한 값들을 순회할 때 매우 유용합니다. 또한 읽기 쉽고 코드를 더 간결하게 만들 수 있습니다.
