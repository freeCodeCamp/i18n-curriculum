# --description--

Одними з більш складних методів, які використовують із масивами, є методи `splice()` та `slice()`. Метод `splice()` змінює вміст масиву, видаляючи або замінюючи елемент у масиві. Метод `slice()` повертає поверхневу копію частини масиву у новий об’єкт масиву, вибраний від `begin` до `end` (`end` не включено). Початковий масив не буде змінено.

Наприклад, щоб видалити другий елемент із масиву `characters`, ви можете використати наступний код:

```javascript
const characters = ['Harry', 'Ron', 'Hermione'];
characters.splice(1, 1);
console.log(characters); // Output: ['Harry', 'Hermione']
```

Вищенаведений елемент видаляє другий елемент із масиву `characters`. Метод `splice()` приймає два аргументи: індекс елемента для видалення та кількість елементів для видалення.


Щоб створити новий масив із другим елементом із масиву `character`, ви можете використати наступний код:

```javascript
const characters = ['Harry', 'Ron', 'Hermione'];
const newCharacters = characters.slice(1, 2);
console.log(newCharacters); // Output: ['Ron']
```

Вищенаведений код створює новий масив `newCharacters` із другим елементом із масиву `characters`. Метод `slice()` приймає два аргументи: індекс елемента, з якого починається зріз, та індекс елемента, на якому зріз закінчується (не включно).
