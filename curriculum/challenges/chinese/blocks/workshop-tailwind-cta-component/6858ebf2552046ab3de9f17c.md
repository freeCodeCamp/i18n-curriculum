# --description--

此时，你会注意到 `div` 中内容右侧的空间太多。 你可以通过增加宽度来最小化这种情况。

将类 `md:w-1/2` 添加到 `div` 以将中型和大型屏幕上的宽度减小到 50%。 也就是说，屏幕分辨率为 768 像素及以上。

要查看效果，请展开预览选项卡。

# --hints--

你的外部 `div` 元素应该具有实用类 `md:w-1/2`。

```js
const firstDiv = document.querySelectorAll("div")[0];
assert.isTrue(firstDiv.classList.contains("md:w-1/2"));
```
