# --interactive--

Un array in JavaScript è una raccolta ordinata di valori, ognuno identificato da un indice numerico. I valori in un array JavaScript possono essere di tipi di dati diversi, inclusi numeri, stringhe, booleani, oggetti e persino altri array.

Per creare un array in JavaScript, puoi usare le parentesi quadre, `[]`, e separare i valori con le virgole. Ecco un esempio:

```js
let fruits = ["apple", "banana", "orange"];
```

In questo esempio, dichiariamo una variabile `fruits` e le assegniamo un array contenente tre valori stringa: `apple`, `banana` e `orange`.

Una delle caratteristiche principali degli array è che sono indicizzati a partire da zero, il che significa che il primo elemento di un array ha indice `0`, il secondo elemento ha indice `1` e così via. Puoi consultare singoli elementi in un array usando il loro indice. Per esempio:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits[0]); // "apple"
console.log(fruits[2]); // "orange"
```

:::

In questo esempio, usiamo l’indice `0` per consultare il primo elemento (`apple`) e l’indice `2` per consultare il terzo elemento (`orange`).

Gli array in JavaScript hanno una proprietà speciale `length` che restituisce il numero di elementi nell’array. Puoi consultare questa proprietà usando la parola chiave `length`. Per esempio:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits.length); // 3
```

:::

Un’altra caratteristica importante degli array in JavaScript è che sono dinamici, il che significa che la loro dimensione può cambiare dopo la creazione. Puoi aggiungere o rimuovere elementi da un array usando vari metodi degli array, come `push()`, `pop()`, `shift()`, `unshift()`, `splice()` e altri. Questi metodi saranno spiegati nelle prossime lezioni.

Gli array JavaScript sono versatili e utili per l’archivio dei dati all’interno dei tuoi programmi. In questo modulo, vedrai direttamente come lavorare con gli array ti aiuterà a gestire e manipolare in modo efficiente raccolte di dati.
