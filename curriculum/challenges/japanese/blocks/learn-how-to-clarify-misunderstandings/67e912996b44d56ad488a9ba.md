# --explanation--

`Code reuse`は、プログラムの新しい部分で既存のコードを使う方法を指します。例えば：

`The "Login" class was used in multiple applications to handle user access, which is a good example of code reuse.` - すべてのアプリでログインのロジックを書く代わりに、同じコードを再利用して効率を上げます。

`Extending`は、すでに存在するものに新しい機能を追加することを意味します。例えば：

`The "AdminUser" class extends the basic "User" class by adding an "approveUser()" method.` - `AdminUser`は`User`のすべてを保持し、新しい機能を加えて拡張します。
