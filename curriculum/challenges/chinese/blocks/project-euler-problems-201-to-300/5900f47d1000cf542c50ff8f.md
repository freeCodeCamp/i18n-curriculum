# --description--

对于正数 $n$，定义 $C(n)$ 为满足 $1 < x < n$ 且 $x^3 \equiv 1\bmod n$ 的整数 $x$ 的个数。

当 $n = 91$ 时，$x$ 有 8 个可能的值，分别是：9、16、22、29、53、74、79、81。因此，$C(91) = 8$。

查找满足 $C(n)=242$ 的正数 $n ≤ {10}^{11}$ 的和。
