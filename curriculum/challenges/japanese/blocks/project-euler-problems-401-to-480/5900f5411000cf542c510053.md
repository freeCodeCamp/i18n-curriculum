# --description--

部屋に円形のテーブルを囲むように$N$脚の椅子が置かれています。

騎士たちは一人ずつ部屋に入り、空いている椅子の中からランダムに選びます。

肘を置くスペースを確保するため、騎士たちは常にお互いの間に少なくとも1脚の空席を残します。

適切な椅子がもう残っていないとき、空席の割合$C$が決まります。また、$E(N)$を$C$の期待値と定義します。

$E(4) = \frac{1}{2}$および$E(6) = \frac{5}{9}$であることが確認できます。

$E({10}^{18})$を求めてください。答えは小数点以下14桁まで、0.abcdefghijklmnの形式で四捨五入してください。
