# --description--

Jetzt können Sie beginnen, Ihr JavaScript zu schreiben. Starten Sie damit, ein `script`-Element zu erstellen. Dieses Element wird verwendet, um JavaScript in Ihre HTML-Datei zu laden.

```html
<script>
  // JavaScript code goes here
</script>
```

# --hints--

Sie sollten ein `script`-Element haben.

```js
assert.isAtLeast(document.querySelectorAll('script').length, 1);
```

Ihr `script`-Element sollte ein öffnendes Tag haben.

```js
assert.match(code, /<script\s*>/i);
```

Ihr `script`-Element sollte ein schließendes Tag haben.

```js
assert.match(code, /<\/script\s*>/i);
```
