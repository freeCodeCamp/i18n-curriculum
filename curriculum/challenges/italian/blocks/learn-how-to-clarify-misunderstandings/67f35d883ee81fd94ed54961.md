# --explanation--

`On schedule` indica che le cose stanno procedendo secondo la tempistica pianificata. Per esempio:

`The project is still on schedule for delivery next week.` — Significa che sarà finito in tempo.

`Miscommunication` si riferisce a una situazione in cui le informazioni non sono state condivise o comprese correttamente. Per esempio:

`There was a miscommunication about the meeting location.` — Le persone non hanno capito il posto giusto.

`Right away` significa immediatamente o il prima possibile. Per esempio:

`I'll send the update right away.` — Lo invierai senza ritardo.
