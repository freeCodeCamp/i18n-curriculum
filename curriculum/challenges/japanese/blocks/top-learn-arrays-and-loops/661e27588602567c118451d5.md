# --description--

配列の基本が理解できたので、次はループについて説明します。ループはコードのブロックを複数回実行するために使います。そのループの一つが`while`ループです。`while`ループは条件が真である限りコードのブロックを実行します。`while`ループの構文は以下の通りです。

```javascript
while (condition) {
  // code block to be executed
}
```

例えば、次のコードスニペットは1から5までの数字を出力します。

```javascript
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
```

上記のコードスニペットでは、変数`i`を値`1`で初期化しています。`while`ループは`i`の値が`5`以下である限りコードのブロックを実行します。`i`の値は各繰り返しで`1`ずつ増加します。
