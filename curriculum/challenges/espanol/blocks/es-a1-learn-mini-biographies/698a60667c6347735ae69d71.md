# --description--

`Le` es un pronombre que representa "a él", "a ella" o "a ti".

`Le gusta` significa "le gusta" cuando se refiere a Diego, que es un hombre. Esto es igual cuando se refiere a una mujer. Por ejemplo:

- `Le gusta el fútbol.` - A él le gusta el fútbol (refiriéndose a un hombre).

- `Le gusta el fútbol.` - A ella le gusta el fútbol (refiriéndose a una mujer).
