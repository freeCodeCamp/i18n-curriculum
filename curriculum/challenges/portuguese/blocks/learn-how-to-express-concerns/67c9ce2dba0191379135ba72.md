# --explanation--

`Really` e `too` são ambos **intensificadores**, mas têm significados diferentes. `Really` reforça o adjetivo que modifica sem indicar excesso. `Too`, por outro lado, indica excesso, significando mais do que o aceitável ou gerenciável. Por exemplo:

- `The project is really important.` - Isso significa que o projeto é muito importante, mas não necessariamente demais.
 
- `This task is too complicated.` - Isso significa que é mais complicado do que deveria ser.
