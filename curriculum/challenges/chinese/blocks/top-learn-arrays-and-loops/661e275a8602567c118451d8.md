# --description--

另一个有用的数组方法是 `filter()` 方法。`filter()` 方法创建一个包含所有通过所提供函数实现的测试的元素的新数组。`filter()` 方法的语法如下：

```javascript
const numbers = [2, 5, 6, 1, 9, -1]

const newNumbers = numbers.filter((number) => {
  return number > 2;
});

console.log(newNumbers); // Output: [5, 6, 9]
```

`filter()` 方法创建一个包含所有大于 2 的元素的新数组。`filter()` 方法不会更改原始数组。
