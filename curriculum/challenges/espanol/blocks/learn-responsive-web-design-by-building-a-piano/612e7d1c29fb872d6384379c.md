# --hints--

```js

```

```js
const div = document.querySelector('div');
assert(div?.children?.length === 1);
assert(div?.children?.[0]?.localName === 'div');
```

```js
const div = document.querySelector('div');
assert(div?.children?.[0]?.className === 'keys');
```
