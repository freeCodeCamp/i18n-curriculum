# --questions--

## --text--

Quale frase usa correttamente la forma comparativa?

## --answers--

`This task is more easy than the previous one.`  

### --feedback--

La parola `more` è superflua perché `easy` cambia in `easier` nella forma comparativa.

---

`This code runs more faster than before.` 

### --feedback--

`More faster` è errato perché `faster` è già una forma comparativa.

---

`This algorithm is more efficient than the old one.` 

---

`The new update is gooder than the last one.`

### --feedback--

La forma comparativa corretta di `good` è `better`.

## --video-solution--

3

# --explanation--

Per formare il grado comparativo, aggiungi `-er` alla fine degli aggettivi o avverbi con una sola sillaba, mentre per quelli più lunghi aggiungi `more` all'inizio.

`More efficient` è corretto perché `efficient` è un aggettivo più lungo.

Potresti notare che ogni opzione contiene la parola magica `than`. È una congiunzione usata per i confronti. Tipicamente segue aggettivi o avverbi nella forma comparativa e introduce il secondo elemento in un confronto, mostrando una differenza di grado, quantità o preferenza.
