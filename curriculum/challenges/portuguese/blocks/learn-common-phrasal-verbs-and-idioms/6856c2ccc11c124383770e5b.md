# --explanation--

`Lastly` é usado para introduzir o ponto final em uma lista ou discussão.Por exemplo:

`Lastly, let's review the budget changes.` – Isso significa que é o último item para discutir.

`Any thoughts on...` é uma forma informal de pedir as ideias ou opiniões de alguém.Frequentemente omite a frase completa `Do you have any thoughts on...` no início.Por exemplo:

`Any thoughts on how we can improve this layout?` – Isso significa que o falante está pedindo ideias de forma amigável e casual sobre como melhorar um layout.
