# --description--

Express hutoa aina mbalimbali za vitendakazi vya programu ya kati iliyojengwa ndani vinavyosaidia kutekeleza kazi za kawaida, kama vile kuhudumia mafaili ya static, changanuzi ya miili ya ombi la JSON, na zaidi. Pia unaweza kutumia programu ya kati ya wahusika wa tatu kuongeza uwezo.

**Hapa kuna baadhi ya mifano ya programu ya kati iliyojengwa ndani**:

1. `express.json()`: Programu ya kati iliyojengwa ndani inayochanganuzi mzigo wa data wa JSON unaoingia.

```js
app.use(express.json())
```

1. `express.static()`: Programu ya kati ya kuhudumia mafaili ya static kama picha, CSS, na mafaili ya JavaScript.

```js
app.use(express.static('public'))
```

1. `express.urlencoded()`: Programu ya kati inayochanganuzi maombi yanayoingia yenye mzigo wa data wa URL-encoded, muundo unaotumwa kawaida na fomu za HTML. Hufanya data iliyochanganuliwa ipatikane kwenye `req.body`.

```js
app.use(express.urlencoded({ extended: true }))
```

**Sasa, hapa kuna baadhi ya mifano ya programu ya kati ya wahusika wa tatu:**

1. `cors`: Programu maarufu ya kati ya wahusika wa tatu kuwezesha Kushirikiana kwa Asili za Mbalimbali (Cross-Origin Resource Sharing).

```bash
npm install cors
```

```js
const cors = require('cors')
app.use(cors())
```

1. `morgan`: Programu ya kati ya kurekodi maombi ya HTTP.

```bash
npm install morgan
```

```js
const morgan = require('morgan')
app.use(morgan('tiny'))
```

Sasa, tuchunguze mfano wa kutumia programu ya kati iliyojengwa ndani na ya wahusika wa tatu pamoja:

```js
const express = require('express')
const cors = require('cors')
const morgan = require('morgan')
const app = express()

app.use(express.json()) // Built-in middleware to parse JSON
app.use(cors())       // Third-party middleware for CORS
app.use(morgan('tiny')) // Third-party middleware for logging

app.get('/', (req, res) => {
  res.send('Hello, World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* Programu ya kati iliyojengwa ndani `express.json()` inachanganuzi mzigo wa data wa JSON unaoingia.
* Programu ya kati ya wahusika wa tatu `cors` inaruhusu maombi ya asili za mbali, na `morgan` inarekodi maombi ya HTTP yanayoingia.

Ni muhimu kutambua kuwa vitendakazi vya programu ya kati huongezwa kwa mpangilio wa wito zao kwa `app.use()`.

Express hutoa programu ya kati muhimu iliyojengwa ndani kwa kushughulikia kazi za kawaida kama changanuzi ya JSON na kuhudumia mafaili ya static. Pia unaweza kuongeza uwezo kwa kutumia programu ya kati ya wahusika wa tatu kwa vipengele zaidi.
