# --description--

Uno dei modi più comuni per aggiungere un nuovo elemento a un array è usare il metodo `push()`. Il metodo `push()` aggiunge uno o più elementi alla fine di un array e restituisce la nuova lunghezza dell'array.

Per esempio, per aggiungere un nuovo elemento all'array `pet`, puoi usare il seguente codice:

```javascript
const pet = ['cat', 'dog', 'bunny'];
pet.push('parrot');
console.log(pet); // Output: ['cat', 'dog', 'bunny', 'parrot']
```

Per rimuovere l'ultimo elemento di un array, puoi usare il metodo `pop()`. Il metodo `pop()` rimuove l'ultimo elemento da un array e restituisce quell'elemento.

Per esempio, per rimuovere l'ultimo elemento dall'array `pet`, puoi usare il seguente codice:

```javascript
const pet = ['cat', 'dog', 'tiger'];
pet.pop();
console.log(pet); // Output: ['cat', 'dog']
```
