# --description--

正の整数 $n$ は、$n$ のすべての素因数 $p$ に対して $p^2$ が $n$ の約数であれば強力数です。

正の整数 $n$ は、別の正の整数のべき乗として表せる場合、完全べき乗数です。

正の整数 $n$ は、強力数でありながら完全べき乗数でない場合、アキレス数と呼ばれます。例えば、864 と 1800 はアキレス数です：$864 = 2^5 \times 3^3$ および $1800 = 2^3 \times 3^2 \times 5^2$。

正の整数 $S$ が強いアキレス数であるとは、$S$ と $φ(S)$ の両方がアキレス数であることを意味します。ここで $φ$ はオイラーのトーシェント関数を表します。

例えば、864 は強いアキレス数です：$φ(864) = 288 = 2^5 \times 3^2$。しかし、1800 は強いアキレス数ではありません。なぜなら：$φ(1800) = 480 = 2^5 \times 3^1 \times 5^1$ だからです。

${10}^4$ 未満には 7 個の強いアキレス数があり、${10}^8$ 未満には 656 個あります。

${10}^{18}$ 未満にはいくつの強いアキレス数があるでしょうか？
