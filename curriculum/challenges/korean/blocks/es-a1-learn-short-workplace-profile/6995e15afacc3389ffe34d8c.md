# --explanation--

`¿Cuál es...?`는 "무엇입니까...?"를 의미합니다. `a` 글자에 있는 강세 표시를 꼭 포함하세요.

이 문맥에서 `su`는 "그의" 또는 "그녀의"를 의미합니다.

`Correo electrónico`는 "이메일"을 의미합니다. 일상 대화에서는 `correo`로 줄여 쓰는 경우가 매우 흔합니다. 전문적인 상황에서는 전체 형태가 일반적으로 사용됩니다.

단어 `correo`는 "우편"을 의미하고 단어 `electrónico`는 "전자"를 의미합니다. `o`의 첫 글자인 `electrónico`에 있는 강세 표시를 꼭 포함하세요.
