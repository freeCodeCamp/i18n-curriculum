# --explanation--

`Exactly` 是一個副詞，用來表示完全同意或正確符合別人所說的內容。另一方面，`Exact` 是一個形容詞，表示精確或細節上的正確。 例如：

- `You are exactly right about the cause of the problem.`－這表示完全同意。它表達了這個人是多麼正確（形容詞）。

- `We need the exact measurements to build the new server rack.`－這表示非常精確的測量值。它修飾名詞而非動詞或形容詞。
