# --explanation--

`Lastly` introduce el último elemento o punto. Por ejemplo:

`Lastly, please remember to lock the door.` – Esto muestra que es la instrucción final.

Los `Ethernet cables` son cables usados para conectar computadoras y otros dispositivos a una red. Por ejemplo:

`We need more Ethernet cables for the new desks.` – Esto significa que se necesitan más cables de red.

`Ended up` significa dónde alguien o algo terminó finalmente. Por ejemplo:

`The documents ended up on my desk.` – Esto significa que ahí fue donde finalmente se encontraron.
