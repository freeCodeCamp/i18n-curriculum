# --explanation--

`Critical` 意味着非常重要或必不可少，`critical data` 指团队依赖以有效工作的重要信息。丢失它会导致延误或问题。例如：

- `The backup includes all critical data in case of a failure.` - 储存的拷贝包含从问题中恢复所需的关键信息。

- `You must encrypt critical data to ensure it's secure。`－使用加密保护重要信息对于安全性是必要的。
