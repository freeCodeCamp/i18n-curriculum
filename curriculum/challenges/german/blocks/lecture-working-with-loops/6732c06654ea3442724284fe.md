# --interactive--

In vorherigen Lektionen haben Sie gelernt, wie man `for`-Schleifen, `for...in`-Schleifen und `for...of`-Schleifen verwendet. In dieser Lektion lernen Sie die `while`-Schleife und die `do...while`-Schleife kennen.

Eine `while`-Schleife führt einen Codeblock so lange aus, wie die Bedingung wahr ist. Hier ist die grundlegende Syntax für eine `while`-Schleife:

```js
while (condition) {
  // code block to be executed
}
```

Die Bedingung wird überprüft, bevor der Codeblock ausgeführt wird. Wenn die Bedingung falsch ist, wird der Codeblock nicht ausgeführt.

`while`-Schleifen sind nützlich, wenn Sie nicht wissen, wie oft Sie den Codeblock ausführen müssen. Hier ist ein Beispiel für die Verwendung einer `while`-Schleife:

:::interactive_editor

```js
let counter = 0;
while(counter < 5) {
  console.log(counter);
  counter++;
}
```

:::

In diesem Beispiel haben wir eine Variable namens `counter`, die mit `0` initialisiert wird. Die `while`-Schleife läuft weiter, solange der Wert von `counter` kleiner als `5` ist. Innerhalb der Schleife geben wir den Wert von `counter` in der Konsole aus und erhöhen dann `counter` um `1`.

Eine andere Schleife, die der `while`-Schleife ähnlich ist, wäre die `do...while`-Schleife. Hier ist die grundlegende Syntax:

```js
do {
  // code block to be executed
} while (condition);
```

Ein wesentlicher Unterschied zwischen einer `do...while`-Schleife und einer `while`-Schleife ist, dass die `do...while`-Schleife den Codeblock mindestens einmal ausführt, bevor die Bedingung überprüft wird.

Wenn die Bedingung wahr ist, wird der Codeblock weiterhin ausgeführt. Wenn die Bedingung falsch ist, wird die Ausführung des Codeblocks beendet.

Hier ist ein Beispiel für die Verwendung einer `do...while`-Schleife:

:::interactive_editor

```js
let counter = 0;
do {
  console.log(counter);
  counter++;
} while (counter < 5);  
```

:::

In diesem Beispiel haben wir eine Variable namens `counter`, die mit `0` initialisiert wird. Die `do...while`-Schleife gibt den Wert von `counter` in der Konsole aus und erhöht dann `counter` um `1`. Nachdem der Codeblock ausgeführt wurde, überprüft sie, ob der Wert von `counter` kleiner als `5` ist. Wenn ja, läuft die Schleife weiter. Wenn nicht, wird die Schleife beendet.

In den meisten Fällen werden Sie wahrscheinlich die `while`-Schleife häufiger verwenden als die `do...while`-Schleife. Es ist jedoch gut, beide Schleifentypen zu kennen und zu wissen, wann man sie einsetzt.
