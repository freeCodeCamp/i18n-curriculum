# --description--

`UX` significa Experiencia de Usuario. Se refiere a cómo se siente una persona cuando está utilizando un producto, sistema o servicio. Esto incluye sitios web, aplicaciones y dispositivos electrónicos. Un `diseñador UX` hace que las cosas sean fáciles y agradables de usar en la computadora o el teléfono.

Por ejemplo: `El UX de este sitio web es excelente porque es fácil de navegar y encontrar la información que necesitas.`
