# --description--

在這一步中，需要添加博客作者的電話號碼和電子郵件地址。

在 `address` 元素中，添加一個段落元素，其文本爲 `Phone: 555-555-5555`。

在該段落元素下方，添加另一個段落元素，其文本爲 `Email: fake@email.com`。

# --hints--

你的 `address` 元素中應該有兩個 `p` 元素。

```js
const addressElement = document.querySelector('footer #contact address');
const pElements = addressElement?.querySelectorAll('p');

assert.lengthOf(pElements, 2);
```

第一個 `p` 元素的文本應爲 `Phone: 555-555-5555`。

```js
const pElement = document.querySelector('#contact p:first-child');
assert.strictEqual(pElement?.innerText.trim(), 'Phone: 555-555-5555');
```

第二個 `p` 元素的文本應爲 `Email: fake@email.com`。

```js
const pElement = document.querySelector('#contact p:last-child');
assert.strictEqual(pElement?.innerText.trim(), 'Email: fake@email.com');
```
