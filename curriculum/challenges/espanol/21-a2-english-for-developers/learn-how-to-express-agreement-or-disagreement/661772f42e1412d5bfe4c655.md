# --description--

Puedes usar diferentes frases o palabras para hablar sobre razonamiento, tales como:

`Justifying` significa dar razones o explicaciones para tus opiniones o acciones. Un ejemplo es, `I chose to study at the library instead of at home. Justificando mi decisión, encuentro que es más silencioso y menos distractor, lo que me ayuda a enfocarme mejor en mi trabajo.`

`Because` se usa para explicar la razón de algo. Por ejemplo, `We're updating the website because it's outdated.`

`That is why` es otra manera de dar una razón. Es como decir, `The server is down. Por eso el sitio web no está accesible.`

`Given` se usa para introducir un hecho que es la base para la decisión. Por ejemplo, `Given the high demand, we need more people on the team.` o `We need more people on the team given the high demand.`
