# --description--

整数の集合 {1, 2, ..., $n$} から選ばれた隠された数を質問によって見つけようとしています。質問する各数には<u>質問した数と同じコストがかかり</u>、次の3つのうちいずれかの答えが返ってきます：

- 「あなたの推測は隠された数より小さい」、
- 「はい、それです！」、
- 「あなたの推測は隠された数より大きい」。

$n$ の値が与えられたとき、最適な戦略は<u>最悪の場合の合計コスト（つまり質問した数の合計）を最小化します</u>。例えば、

$n = 3$ の場合、最善の方法は明らかに「<strong>2</strong>」と質問することです。答えによってすぐに隠された数がわかり（合計コスト = 2）、これが最適です。

$n = 8$ の場合、「二分探索」タイプの戦略を使うことが考えられます。最初の質問は「<strong>4</strong>」で、隠された数が4より大きければ追加で1回か2回質問が必要です。2回目の質問は「<strong>6</strong>」とし、隠された数がまだ6より大きければ、7と8のどちらかを判別するために3回目の質問「<strong>7</strong>」が必要です。したがって、この最悪の場合の合計コストは $4 + 6 + 7 = \mathbf{\color{red}{17}}$ となります。

しかし、$n = 8$ の最悪コストは、最初の質問を「<strong>5</strong>」にすることで大幅に改善できます。隠された数が5より大きいと言われたら、2回目の質問は「<strong>7</strong>」で、これで隠された数が確定します（合計コストは $5 + 7 = \mathbf{\color{blue}{12}}$）。隠された数が5より小さいと言われたら、2回目の質問は「<strong>3</strong>」で、さらに隠された数が3より小さい場合は3回目の質問「<strong>1</strong>」をします。合計コストは $5 + 3 + 1 = \mathbf{\color{blue}{9}}$ です。$\mathbf{\color{blue}{12 > 9}}$ なので、この戦略の最悪コストは<strong><span style="color: red;">12</span></strong>となります。これは先ほどの「二分探索」戦略より良く、他のどの戦略よりも良いか同等です。つまり、$n = 8$ に対する最適戦略を説明したことになります。

$C(n)$ を、上記のように $n$ に対して最適戦略が達成する最悪コストとします。したがって $C(1) = 0$、$C(2) = 1$、$C(3) = 2$、$C(8) = 12$ です。

同様に、$C(100) = 400$ であり、$\displaystyle\sum_{n = 1}^{100} C(n) = 17575$ です。

$\displaystyle\sum_{n = 1}^{200\,000} C(n)$ を求めてください。
