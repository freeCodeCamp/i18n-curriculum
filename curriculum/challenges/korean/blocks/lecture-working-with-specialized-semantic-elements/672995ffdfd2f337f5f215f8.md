# --interactive--

superscript 요소는 텍스트를 위첨자로 표시할 때 사용합니다. 위첨자는 일반 텍스트 줄 위에 인쇄된 기호나 문자입니다.

다음은 superscript 요소를 사용해 지수를 나타낸 예시입니다(미리보기를 보려면 인터랙티브 편집기를 활성화하세요):

:::interactive_editor

```html
<p>2<sup>2</sup> (2 squared) is 4.</p>
```

:::

이 예시에서 숫자 2는 문단 내에서 위첨자를 나타내기 위해 `sup` 태그로 감싸져 있습니다. 미리보기 창에서는 두 번째 숫자 2가 첫 번째 숫자 2보다 작고 약간 위에 위치한 것을 볼 수 있습니다.

superscript 요소의 일반적인 사용 사례는 지수, 위첨자 문자, 서수입니다. 다음은 위첨자 문자를 나타내기 위해 superscript 요소를 사용한 예시입니다(미리보기를 보려면 인터랙티브 편집기를 활성화하세요):

:::interactive_editor

```html
<p>
  Monseigneur is often written as <strong>M<sup>gr</sup></strong>.
</p>
```

:::

위첨자 문자는 보통 약어를 나타내기 위해 위첨자로 쓰인 문자를 말합니다. 이 예시에서는 `g`와 `r` 문자가 위첨자 태그 안에 감싸져 약어를 나타냅니다.

중요한 점은 superscript 요소는 오직 타이포그래피 목적으로만 사용해야 한다는 것입니다. 텍스트를 시각적으로 올려서 스타일링하려면 superscript 요소 대신 CSS를 사용하세요.

HTML에서 화학식을 나타내려면 subscript 요소를 사용합니다. 이 요소는 아래첨자를 사용해 기준선보다 낮게, 더 작은 텍스트로 표시합니다.

다음은 subscript 요소를 사용해 이산화탄소의 화학식을 나타낸 예시입니다(미리보기를 보려면 인터랙티브 편집기를 활성화하세요):

:::interactive_editor

```html
<p>CO<sub>2</sub></p>
```

:::

숫자 2는 아래첨자임을 나타내기 위해 `sub` 태그로 감싸져 있습니다.

subscript 요소의 일반적인 사용 사례는 화학식, 각주, 변수 아래첨자입니다.
