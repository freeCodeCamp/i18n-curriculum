# --interactive--

Існують різні стани посилання, зокрема `link`, `visited`, `hover`, `focus` та `active`. Ці стани важливі для того, щоб допомогти користувачам розпізнавати посилання та надавати чіткий зворотний зв’язок після взаємодії, що покращує як зручність використання, так і доступність.

Стилізація цих різних станів посилань є ключовою для зручності використання та доступності, оскільки вона надає візуальні підказки про поточний стан посилання. Це допомагає користувачам зрозуміти, які посилання вони вже відвідали, з яким посиланням вони взаємодіють і що станеться, коли вони натиснуть на нього.

Для користувачів із порушеннями зору або когнітивними особливостями ці відмінні стилі можуть значно полегшити навігацію та зробити її більш інтуїтивною.

Крім того, чіткі стани посилань покращують загальний досвід користувача, надаючи миттєвий зворотний зв’язок про взаємодію, зменшуючи плутанину та покращуючи навігацію сайтом.

Ці стани можна стилізувати за допомогою чогось, що називається `pseudo-classes` у CSS.

Псевдоклас — це ключове слово, додане до селектора, яке вказує на особливий стан вибраного елемента.

Наприклад, `:hover` може змінювати колір кнопки, коли курсор користувача наводиться на неї, а `:visited` може змінювати колір посилання, яке вже було відвідане.

Псевдокласи дозволяють стилізувати елементи залежно від їхнього стану або взаємодії користувача з ними без необхідності додаткової розмітки в HTML.

Синтаксис `pseudo-class` виглядає приблизно так, де `A` — це селектор, а `:B` — `pseudo-class`:

```css
A:B {
  property: value;
}
```

Щоб краще зрозуміти, як стилізувати різні стани посилань, розглянемо кілька прикладів.

Псевдоклас `:link` стилізує невідвідані посилання, вказуючи, що вони активні.

Ось приклад націлювання на елемент посилання з використанням псевдокласу `:link`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Normal state (unvisited link) */
a:link { 
  color: red;
}
```

:::

У наведеному прикладі колір посилання за замовчуванням зміниться з синього на `red`, коли посилання не було відвідане.

`:visited` стилізує посилання, які вже були відвідані або натиснуті, допомагаючи користувачам відстежувати, які посилання вони вже відкривали. Ось приклад використання псевдокласу `:visited`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="https://www.freecodecamp.org/learn/" target="_blank">freeCodeCamp</a>
```

```css
/* Visited link */
a:visited {
  color: green;
}
```

:::

Цей код змінить колір посилання на `green`, коли воно було натиснуте.

`:hover` змінює стиль посилання, коли користувач наводить на нього курсор, надаючи візуальну підказку, що посилання активне. Ось приклад використання псевдокласу `:hover`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Hover state */
a:hover {
  color: green;
}
```

:::

Цей код змінить колір посилань на `green` при наведенні миші.

`:focus` додає стилі навколо посилання, коли воно у фокусі, наприклад, під час навігації за допомогою клавіатури або для покращення доступності.

Ось приклад використання властивості `outline` для застосування суцільних помаранчевих обрисів, коли посилання у фокусі.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Focus state */
a:focus {
  outline: 2px solid orange;
}
```

:::

`:active` змінює стиль посилання під час його натискання, надаючи миттєвий зворотний зв’язок користувачу про те, що його дія зафіксована. Ось приклад використання псевдокласу `:active`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Active state */
a:active {
  color: pink;
}
```

:::

Цей приклад коду миттєво змінить колір посилання на `pink` під час натискання.
