# --description--

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

將 `animal` 放入 `favoriteAnimal()` 函式的小括弧內，表示你告訴 JavaScript 你將會傳送某個值給你的 `favoriteAnimal` 函式。這表示 animal 只是某個未來值的佔位符。但你傳送的是什麼值？

最後一行，`favoriteAnimal('Goat')`，是你在呼叫你的 `favoriteAnimal` 函式（程式），並將值 `'Goat'` 傳遞到該函式中。這裡，`'Goat'` 是你的引數。你告訴 `favoriteAnimal` 函式，「請將 `'Goat'` 傳送到 `favoriteAnimal` 函式，並在任何 `'animal'` 佔位符處使用 `'Goat'`。」因為使用參數所提供的彈性，你可以宣告任何動物為你最喜愛的。

這裡有一個圖表可以幫助你視覺化參數如何傳遞給函式，以及值如何從中傳回。

請注意，透過在 `console.log()` 裡呼叫 `favoriteAnimal()` 並傳入引數 `'Goat'`，你會得到函式的返回值，也就是字串 `"Goat is my favorite animal!"`，並列印到主控台。你是在另一個函式呼叫 `log()` 中，將函式呼叫 `favoriteAnimal('Goat')` 作為引數傳入。

## --text--

如果你將 `favoriteAnimal('Goat')` 函式（程式）呼叫中的引數從 `'Goat'` 改為 `'Elephant'`，`console.log(favoriteAnimal('Elephant'))` 會在主控台列印出什麼？

## --answers--

`"Goat is my favorite animal!"`

---

該函式（程式）將傳回錯誤，因為 `"Elephant"` 不是可識別的引數。
