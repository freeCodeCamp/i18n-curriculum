# --description--

Quando lavori con le stringhe, combinare diversi pezzi di testo insieme è un'operazione comune con cui ti troverai spesso a fare i conti.

In Python, puoi unire più stringhe insieme con l'operatore più (`+`). Questo processo si chiama **concatenazione di stringhe**. Ecco come concatenare due stringhe con l'operatore più:

```python
my_str_1 = 'Hello'
my_str_2 = "World"

str_plus_str = my_str_1 + ' ' + my_str_2
print(str_plus_str) # Hello World
```

Ma nota che questo funziona solo con le stringhe. Se provi a concatenare una stringa con un numero, otterrai un `TypeError`:

```python
name = 'John Doe'
age = 26

name_and_age = name + age
print(name_and_age) # TypeError: can only concatenate str (not "int") to str
```

Questo succede perché Python non converte automaticamente altri tipi di dati come i numeri interi in stringhe quando li concateni. Python richiede che tutti gli elementi siano stringhe prima di poterli concatenare. Per risolvere questo, puoi convertire il numero in una stringa con la funzione integrata `str()`, che restituisce la rappresentazione in stringa dell'oggetto dato senza modificare l'oggetto originale:

```python
name = 'John Doe'
age = 26

name_and_age = name + str(age)
print(name_and_age) # John Doe26
```

Puoi anche usare l'operatore di assegnazione aumentata per la concatenazione. Questo è rappresentato da un più seguito da un uguale (`+=`) e esegue sia la concatenazione sia l'assegnazione in un solo passaggio. Ecco come funziona:

```py
name = 'John Doe'
age = 26

name_and_age = name  # Start with the name
name_and_age += str(age)  # Append the age as string

print(name_and_age)  # John Doe26
```

Il processo di inserire variabili ed espressioni in una stringa si chiama **interpolazione di stringhe**. Python ha una categoria di stringhe chiamate **f-string** (abbreviazione di formatted string literals), che ti permette di gestire l'interpolazione con una sintassi compatta e leggibile.

Le f-string iniziano con `f` (minuscolo o maiuscolo) prima delle virgolette e ti permettono di incorporare variabili o espressioni all'interno di campi di sostituzione indicati da parentesi graffe (`{}`). Ecco un esempio:

```python
name = 'John Doe'
age = 26
name_and_age = f'My name is {name} and I am {age} years old'
print(name_and_age) # My name is John Doe and I am 26 years old

num1 = 5
num2 = 10
print(f'The sum of {num1} and {num2} is {num1 + num2}') # The sum of 5 and 10 is 15
```

Nota come non sia necessario convertire i tipi non stringa con la funzione `str()`. Nell'esempio sopra, il valore delle variabili `age`, `num1` e `num2` viene convertito automaticamente in stringa durante il processo di interpolazione.
