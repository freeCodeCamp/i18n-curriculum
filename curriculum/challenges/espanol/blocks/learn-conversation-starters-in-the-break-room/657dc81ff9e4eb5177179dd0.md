# --description--

`Occasionally` significa que algo ocurre de vez en cuando. No es todo el tiempo, y no es nunca, sino algo intermedio. Por ejemplo:

`I occasionally get a bug in my code.` - A veces hay un error en tu código, pero no muy seguido.

Mira estos adverbios, y la frecuencia que usualmente representan.

| Adverb       | Frequency | Adverb    | Frequency |
| ------------ | --------- | --------- | --------- |
| Never        | 0%        | Often     | 70%       |
| Rarely       | 5%        | Generally | 80%       |
| Seldom       | 15%       | Usually   | 90%       |
| Occasionally | 30%       | Always    | 100%      |
| Sometimes    | 50%       |           |           |
