# --explanation--

`Walkthrough` は2つの部分から成り立っています：

- `Walk` は通常徒歩で、一歩一歩進むことを指します。

- `Through` は一方からもう一方へ、または始めから終わりまで移動することを意味します。

これらの言葉が `walkthrough` として組み合わさると、何かのやり方を理解するのに役立つ段階的な説明や案内を意味します。例えば：

- `The software includes a walkthrough to help new users get started.` - ソフトウェアの使い方を各ステップで説明する案内があります。

- `Let me give you a walkthrough of the process.` - プロセスの各ステップを詳しく説明します。
