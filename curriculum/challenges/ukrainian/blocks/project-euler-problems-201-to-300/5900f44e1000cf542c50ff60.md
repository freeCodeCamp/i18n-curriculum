# --description--

Послідовність 1, 1, 1, 3, 5, 9, 17, 31, 57, 105, 193, 355, 653, 1201 ...

визначається як $T_1 = T_2 = T_3 = 1$ та $T_n = T_{n - 1} + T_{n - 2} + T_{n - 3}$.

Доведено, що 27 не ділить жоден член цієї послідовності. Насправді, 27 — це перше непарне число з такою властивістю.

Знайдіть ${124}^{\text{те}}$ непарне число, яке не ділить жоден член наведеного вище ряду.
