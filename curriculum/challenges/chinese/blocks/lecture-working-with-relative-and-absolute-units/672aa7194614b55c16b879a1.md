# --interactive--

在设计你的页面时，你将使用不同的属性，如宽度、高度、内边距、外边距等。当你定义这些属性时，你需要指定你想使用的长度单位。

有两种类型的单位可用于定义这些属性：相对单位和绝对单位。在本课中，我们将只关注绝对单位。

绝对长度单位是固定长度，不相对于任何其他内容。相对长度意味着长度相对于其他内容，比如屏幕大小或父元素的大小。我们将在接下来的几节课中详细讲解相对单位。

The most common absolute unit is the pixel (`px`). 像素是 CSS 中的固定大小测量单位，提供对维度的精确控件。 这意味着 1px 始终等于 1/96 英寸。

需要注意的是，虽然 1px 在 CSS 版型中被标准化为 1/96 英寸，但像素的实际物理大小可能因显示器而异。

让我们来看一个使用像素设置盒子宽度和高度的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="box"></div>
```

```css
.box {
  width: 100px;
  height: 100px;
  background-color: red;
}
```

:::

在此示例中，我们有一个宽度和高度设置为 `100px` 的红色盒子。该盒子的维度将保持不变，无论屏幕尺寸如何变化。

所以，如果你使用的是小屏幕设备，盒子仍然是 `100px` 宽。如果你使用的是大屏幕桌面设备，盒子仍然是 `100px` 宽。

那么，什么时候你应该使用像素这样的绝对单位呢？这取决于具体情况。但通常当你需要对元素维度、间距和版型进行精确控制时，会使用像素。有时你可能会为边距、内边距和边框使用像素。

下面是使用像素设置两个盒子边距的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="box"></div>
<div class="box"></div>
```

```css
.box {
  width: 100px;
  height: 100px;
  background-color: red;
  margin: 10px;
}
```

:::

请记住，margin 是盒子外部的空间。因此，在此示例中，盒子的四周将有 `10px` 的 margin。

其他类型的绝对单位包括以下内容：

- `in`（英寸）单位，相当于 96px
- `cm`（厘米）单位，其长度相当于 25.2/64 英寸
- `mm`（毫米）单位，相当于 1/10 厘米
- `q`（四分之一毫米）单位，相当于 1/40 厘米
- `pc`（派卡）单位，相当于 1/6 英寸
- `pt`（点）单位，相当于 1/72 英寸

这些单位大多数将用于打印，而非屏幕。 

虽然你最常用的单位是像素，但了解其他绝对单位的存在也很重要。
