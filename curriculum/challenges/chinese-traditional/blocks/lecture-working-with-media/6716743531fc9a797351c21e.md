# --interactive--

被取代的元素是其內容由外部資源決定，而非由 CSS 本身決定的元素。CSS，或稱串接樣式表，用於為網頁添加樣式。常見的被取代元素範例包括 `image`、`iframe` 和 `video` 元素。

使用取代元素，你可以控制元素的位置或版面配置。但你的 CSS 無法直接修改該元素的內容。這可能用一些範例來說明會比較容易。考慮 `image` 控制元件，它會在你的網頁中嵌入一個影像：

```html
<img src="example-img-url" alt="Descriptive text goes here">
```

該元素本身會被外部物件取代：影像。你的 CSS 可以控制影像的位置或套用濾鏡，但你無法實際修改影像本身。一個更強固的範例可能是 `iframe` 元素，它會在你的網頁中嵌入外部網站。

這裡是一個使用 `iframe` 元素來播放 freeCodeCamp 頻道熱門 YouTube 影片的範例。如果你想在預覽視窗中觀看不同的影片，請將 `src` 屬性的值更改為你選擇的影片。要看到預覽，你需要啟用互動式編輯器。

**注意**：不用擔心互動範例中提到的額外屬性，例如 `referrerpolicy` 和 `allowfullscreen`。你將在未來的工作坊中學習更多關於操作 `iframe` 元素的內容。 

:::interactive_editor

```html
<iframe width="400" height="200" src="https://www.youtube.com/embed/u43gJJrVa1I?si=BoDW_puFsy8OEr_Z" title="Professional Cloud Architect Certification Course – Pass the Exam! (YouTube video)" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen></iframe>
```

:::

其他使用 `iframe` 元素的常見範例是將地圖嵌入頁面。以下是一個嵌入地圖的範例。

啟用互動式編輯器，並嘗試透過放大/縮小來操作地圖本身。 

:::interactive_editor

```html
<iframe
  title="Map of the Royal Observatory, Greenwich, London"
  width="300"
  height="200"
  src="https://www.openstreetmap.org/export/embed.html?bbox=-0.004017949104309083%2C51.47612752641776%2C0.00030577182769775396%2C51.478569861898606&amp;layer=mapnik">
</iframe>
```

:::

該元素本身會被外部物件取代：該網站。你的 CSS 可以改變嵌入網站的位置，但你無法修改該網站的內容。更深入一點，如果嵌入的網站有一個 `h1` 元素，你的 CSS 將無法為該 `h1` 元素設定樣式。你無法改變大小、字型顏色等等。

還有一些其他的取代元素，例如 `video` 和 `embed`。而且某些元素在特定情況下會表現為取代元素。以下是一個 `input` 元素，其 `type` 屬性設定為 `image`：

```html
<input type="image" alt="Descriptive text goes here" src="example-img-url">
```

這種類型的 `input` 被視為取代元素，但其他 `input` 型別如 `text` 或 `email` 並非取代元素。
