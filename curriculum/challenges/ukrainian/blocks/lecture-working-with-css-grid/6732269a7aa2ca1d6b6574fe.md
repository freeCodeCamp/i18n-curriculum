# --interactive--

Неявна сітка — це рядки та стовпці, які браузер створює автоматично при розміщенні елементів у макеті сітки — ті, що не визначені явно за допомогою `grid-template-rows` або `grid-template-columns`.

Властивості, які контролюють стовпці та рядки, створені неявно браузером, — це `grid-auto-columns` та `grid-auto-rows`.

Неявна сітка також означає додаткові рядки та стовпці, які браузер автоматично генерує, коли ви розміщуєте елемент поза межами явно визначених рядків і стовпців.

Наприклад, припустимо, ви визначили лише два явних стовпці в макеті сітки таким чином:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
  <div class="grid-item">Item 4</div>
  <div class="grid-item">Item 5</div>
  <div class="grid-item">Item 6</div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 100px 100px; /* Only 2 explicit columns */
}

.grid-item {
  background-color: burlywood;
  border: 1px solid orangered;
  padding: 0.5rem;
  margin: 0.5rem;
}
```

:::

Два елементи заповнили перший рядок, використовуючи два явних стовпці — `Item 1` у першому стовпці та `Item 2` у другому стовпці.

Наступні елементи починають новий рядок: `Item 3` розміщується у першому стовпці другого рядка, а `Item 4` — у другому стовпці другого рядка, і так далі.

Як ви вже бачили, явні сітки — це області сітки, які ви свідомо налаштовуєте. Тобто рядки та стовпці, які ви явно визначаєте для макету сітки за допомогою властивостей `grid-template-rows` та `grid-template-columns`.

Ось таблиця, що підсумовує відмінності між неявними та явними сітками:

| **Функція**          | **Явна сітка**                                                                  | **Неявна сітка**                                                                          |
|----------------------|---------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| **Контроль розміру**  | Повністю налаштовується за допомогою ` grid-template-rows ` та ` grid-template-columns `.                  | Контролюється ` grid-auto-rows ` та ` grid-auto-columns `, або за замовчуванням ` auto `.                  |
| **Поведінка за замовчуванням** | Не змінюється, якщо не визначено явно.                                         | Автоматично адаптується до елементів, розміщених поза межами явної сітки.                  |
| **Складність**       | Вимагає більше планування структури макету.                                    | Легше реалізувати для неструктурованого або змінного вмісту.                              |
| **Гнучкість**        | Жорстка структура з визначеними рядками та стовпцями.                         | Гнучка і адаптується до динамічно розміщеного вмісту.                                    |
| **Продуктивність**   | Може бути більш продуктивною, оскільки макет заданий заздалегідь.              | Може вимагати додаткових обчислень браузера для генерації неявної сітки.                  |
| **Використання**     | Корисна, коли структура сітки передбачувана і визначена заздалегідь.           | Корисна для динамічних макетів, де вміст невідомий або часто змінюється.                  |
