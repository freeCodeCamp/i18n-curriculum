# --description--

La declaración `flex` es en realidad una estructura abreviada para 3 propiedades que puedes configurar en un ítem flex. Estas propiedades afectan cómo los ítems flex se dimensionan dentro de su contenedor. Ya has visto algunas propiedades abreviadas antes, pero aún no las has definido oficialmente.

> Las propiedades abreviadas son propiedades CSS que te permiten establecer los valores de múltiples otras propiedades CSS simultáneamente. Usando una propiedad abreviada, puedes escribir hojas de estilo más concisas (y a menudo más legibles), ahorrando tiempo y esfuerzo.

En este caso, `flex` es en realidad una estructura abreviada para `flex-grow`, `flex-shrink` y `flex-basis`.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/flex-box/flexbox-04.png" alt="CSS code setting the flex property to 1 for a div element.">

En la captura de pantalla anterior, `flex: 1` equivale a: `flex-grow: 1`, `flex-shrink: 1`, `flex-basis: 0`.

Muy a menudo, ves la estructura flex definida con solo un valor. En ese caso, ese valor se aplica a `flex-grow`. Así que cuando pones `flex: 1` en nuestros divs, en realidad estabas especificando una estructura abreviada de `flex: 1 1 0`.
