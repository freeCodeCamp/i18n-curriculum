# --description--

`.reduce()` 메서드는 누산기의 초기값으로 사용되는 두 번째 인수를 받습니다. 두 번째 인수가 없으면 `.reduce()` 메서드는 배열의 첫 번째 요소를 누산기로 사용하여 예상치 못한 결과가 발생할 수 있습니다.

안전하게 하려면 초기값을 설정하는 것이 가장 좋습니다. 초기값을 빈 문자열로 설정하는 예는 다음과 같습니다.

```js
array.reduce((acc, el) => acc + el.toLowerCase(), "");
```

누산기의 초기값을 `0`로 설정하세요.
