# --explanation--

`If only` é usado no início de uma frase para expressar um desejo ou arrependimento sobre algo que não aconteceu no passado. Mostra que o falante gostaria que a situação tivesse sido diferente. Por exemplo:

`If only we had checked the system earlier, we could have avoided the crash.` – Isso significa que eles não verificaram cedo o suficiente e agora se arrependem.
