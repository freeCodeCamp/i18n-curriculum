# --description--

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

Kwa kuweka `animal` ndani ya mabano ya kawaida ya kitendakazi cha `favoriteAnimal()`, unamwambia JavaScript kwamba utatuma thamani fulani kwa kitendakazi chako cha `favoriteAnimal`. Hii ina maana kwamba animal ni kiwakilishi tu kwa thamani fulani ya baadaye. Lakini ni thamani gani unayotuma?

Mstari wa mwisho, `favoriteAnimal('Goat')`, ndio unapo wito kitendakazi chako cha `favoriteAnimal` na kupitisha thamani `'Goat'` ndani ya kitendakazi hicho. Hapa, `'Goat'` ni hoja yako. Unamwambia kitendakazi cha `favoriteAnimal`, "Tafadhali tuma `'Goat'` kwa kitendakazi cha `favoriteAnimal` na tumia `'Goat'` popote ambapo kiwakilishi cha `'animal'` kiko." Kwa sababu ya unyumbufu unaotolewa na kutumia kigezo, unaweza kutangaza animal yoyote kuwa kipenzi chako.

Hapa kuna mchoro wa kusaidia kuona jinsi vigezo vinavyopitishwa kwa kitendakazi, na jinsi thamani zinavyorejeshwa kutoka kwake.

Fahamu kwamba kwa kuita `favoriteAnimal()` ndani ya `console.log()` na hoja `'Goat'`, unapata thamani ya return ya kitendakazi, mfuatano wa herufi wa `"Goat is my favorite animal!"`, ukachapishwa kwenye konsoli. Unapitisha wito wa kitendakazi `favoriteAnimal('Goat')` kama hoja katika wito mwingine wa kitendakazi - `log()`.
