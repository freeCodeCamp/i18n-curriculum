# --description--

Llamemos a un triángulo con lados enteros y lados $a ≤ b ≤ c$ apenas agudo si los lados cumplen $a^2 + b^2 = c^2 + 1$.

¿Cuántos triángulos apenas agudos hay con perímetro $≤ 25\,000\,000$?
