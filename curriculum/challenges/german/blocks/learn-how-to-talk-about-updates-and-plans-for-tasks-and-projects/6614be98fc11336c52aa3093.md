# --description--

Ein `slide deck` ist eine Sammlung von Folien, die für eine Präsentation zusammengestellt wurden, oft mit Software wie PowerPoint oder Google Slides erstellt. Es ist wie eine digitale Version eines Kartenstapels, wobei jede Folie einen Teil des Inhalts der Präsentation enthält.

Zum Beispiel `We prepared a slide deck for the meeting to showcase our project's progress.`

Der Begriff `standard` bezieht sich auf etwas, das allgemein verwendet oder als Norm akzeptiert wird. In diesem Zusammenhang bedeutet `standard format` ein Format, das weit verbreitet und anerkannt ist. Zum Beispiel `Using bullet points for key points is a standard practice in presentations.`

# --questions--
