# --description--

다음 형태의 방정식을 고려하세요: $a^2 + b^2 = N$, $0 ≤ a ≤ b$, $a$, $b$ 그리고 $N$은 정수입니다.

$N = 65$일 때 두 가지 해가 있습니다:

$a = 1, b = 8$ 그리고 $a = 4, b = 7$.

$S(N)$을 $a^2 + b^2 = N$, $0 ≤ a ≤ b$, $a$, $b$ 그리고 $N$이 정수인 모든 해의 $a$ 값들의 합이라고 부릅니다.

따라서 $S(65) = 1 + 4 = 5$입니다.

$4k + 1$ 형태의 소수로만 나누어지는 제곱 인수 없는 모든 $N$에 대해 $\sum S(N)$을 구하세요. 단, $4k + 1 < 150$입니다.
