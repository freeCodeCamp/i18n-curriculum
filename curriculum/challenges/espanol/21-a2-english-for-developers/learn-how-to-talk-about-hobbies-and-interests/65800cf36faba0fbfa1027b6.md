# --description--

Las preguntas negativas son una forma de pedir a alguien que haga algo o sugerir una acción de manera cortés o alentadora. A menudo comienzan con `why` seguido de un formulario negativo como `don't`. Por ejemplo, `Why don't we go for a walk?` sugiere ir a dar un paseo como una buena idea.
