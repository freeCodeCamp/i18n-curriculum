# --description--

```css
/* rule 1 */
#subsection .list {
  background-color: yellow;
  color: blue;
}

/* rule 2 */
#subsection .main .list {
  color: red;
}
```

У цьому фінальному прикладі обидва правила використовують селектори ідентифікатора та класу, тож жодне з правил не має більш специфічного селектора за інше. Каскад тоді перевіряє кількість кожного типу селектора. Обидва правила мають по одному селектору ідентифікатора, але правило 2 має більше селекторів класу, тому правило 2 має вищу специфічність!

Хоча оголошення `color: red` матиме пріоритет, оголошення `background-color: yellow` все одно буде застосоване, оскільки для нього немає конфліктного оголошення.
