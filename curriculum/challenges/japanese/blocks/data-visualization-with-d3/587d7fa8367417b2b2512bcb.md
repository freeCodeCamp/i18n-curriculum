# --description--

<dfn>SVG</dfn>は<dfn>Scalable Vector Graphics</dfn>の略です。

ここで「スケーラブル」とは、オブジェクトを拡大・縮小してもピクセル化されず、表示システムに合わせてサイズが変わることを意味します。小さなモバイル画面でも大きなテレビモニターでも同様にスケールします。

SVGは一般的な幾何学的形状を作成するために使われます。D3はデータを視覚的に表現するため、可視化の形状を作るのにSVGを使用します。ウェブページのSVG形状はHTMLの`svg`タグ内に配置する必要があります。

CSSは相対単位（`vh`、`vw`、パーセンテージなど）を使うとスケーラブルになりますが、データの可視化を作るにはSVGのほうがより柔軟です。

# --instructions--

`svg`に`body`を使って`append()`ノードを追加してください。`width`属性には提供された`w`定数を、`height`属性には提供された`h`定数を、それぞれ`attr()`または`style()`メソッドを使って設定してください。`background-color`タグでピンクの`style`が適用されているので、出力で確認できます。

**注意:** `attr()`のwidthとheight属性には単位を付けません。これはスケーリングの基本で、ズームレベルに関係なく常に幅と高さの比率が5:1になります。

# --hints--

ドキュメントには`svg`要素が1つあるはずです。

```js
assert.lengthOf(document.querySelectorAll('svg'), 1);
```

`svg`要素には`width`属性が`500`に設定されているか、幅が`500px`になるようにスタイル付きにしてください。

```js
const svg = document.querySelector('svg');
assert.exists(svg); 
const width = window.getComputedStyle(svg)['width'];
assert.isTrue(svg?.getAttribute('width') === '500' || width === '500px');
```

`svg`要素には`height`属性が`100`に設定されているか、高さが`100px`になるようにスタイル付きにしてください。

```js
const svg = document.querySelector('svg');
assert.exists(svg); 
const height = window.getComputedStyle(svg)['height'];
assert.isTrue(svg?.getAttribute('height') === '100' || height === '100px');
```
