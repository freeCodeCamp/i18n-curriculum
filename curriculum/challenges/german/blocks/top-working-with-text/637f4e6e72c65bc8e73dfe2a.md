# --description--

Das `em`-Element macht Text kursiv. Es betont den Text auch semantisch, was sich beispielsweise auf Screenreader auswirken kann. Um ein betontes Element zu definieren, umschließen Sie den Textinhalt mit einem `<em>`-Tag.
