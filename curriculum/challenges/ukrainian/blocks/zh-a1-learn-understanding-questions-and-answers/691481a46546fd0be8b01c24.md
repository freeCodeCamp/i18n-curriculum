# --explanation--

`UI` означає `user interface`, що стосується візуальних елементів продукту, з якими взаємодіють користувачі. `设计师 (shè jì shī)` означає "дизайнер".

Разом `UI 设计师 (shè jì shī)` — це професія, яка зосереджена на проєктуванні зовнішнього вигляду та макету продуктів, щоб зробити їх зручними та приємними у використанні. Це дуже загальний термін у технологічних компаніях, і люди безпосередньо використовують `UI`, не перекладаючи його на китайську. Наприклад:

`我是 (wǒ shì) UI 设计师 (shè jì shī)。` – Я UI дизайнер.
