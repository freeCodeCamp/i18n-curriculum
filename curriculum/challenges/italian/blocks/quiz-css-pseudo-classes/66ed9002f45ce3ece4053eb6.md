## --quiz--

### --question--

#### --text--

Cosa sono le pseudo-classi?

#### --distractors--

Sono regole CSS che creano contenuto aggiuntivo all'interno di un elemento quando necessario.

---

Sono proprietà che regolano la dimensione o il layout di un elemento nella pagina.

---

Sono selettori usati per aggiungere effetti di movimento a un elemento durante le interazioni.

#### --answer--

Sono parole chiave aggiunte a un selettore che definiscono lo stile di un elemento in base al suo stato.
