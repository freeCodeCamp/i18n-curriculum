# --explanation--

`Sounds good` 是一種隨意的說法，用來表示 `I agree` 或 `That works for me`。

`Keep me posted` 意思是 `Let me know if there are any updates`。這是一種禮貌的方式，請求持續提供關於某情況的資訊。例如：

- `If you find any issues with the system, keep me posted.` - 你正在請求某人如果發現系統有任何問題，請告訴你。

- `Keep me posted if you get any feedback from the client.` - 你是在請某人如果從客戶那裡聽到任何新消息，請告訴你。
