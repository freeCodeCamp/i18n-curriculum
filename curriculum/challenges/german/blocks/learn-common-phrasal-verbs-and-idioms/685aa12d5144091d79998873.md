# --explanation--

`To get something done` bedeutet, dafür zu sorgen, dass etwas erledigt wird, entweder selbst oder durch jemand anderen. Zum Beispiel:

`I'll get the report done by Friday.` – Das bedeutet, die Person wird sicherstellen, dass der Bericht bis zu diesem Zeitpunkt fertig ist, entweder indem sie ihn selbst fertigstellt oder jemand anderen damit beauftragt.
