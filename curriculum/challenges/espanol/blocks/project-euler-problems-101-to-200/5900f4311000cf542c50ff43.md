# --description--

Llamemos triángulo de 60° a un triángulo con lados enteros que tiene exactamente un ángulo de 60°.

Sea $r$ el radio del círculo inscrito de un triángulo de 60°.

Hay 1234 triángulos de 60° para los cuales $r ≤ 100$.

Sea $T(n)$ el número de triángulos de 60° para los cuales $r ≤ n$, así que $T(100) = 1234$, $T(1000) = 22767$ y $T(10000) = 359912$.

Encuentra $T(1053779)$.
