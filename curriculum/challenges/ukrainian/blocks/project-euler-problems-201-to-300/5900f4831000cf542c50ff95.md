# --description--

Дано значення цілих чисел $1 < a_1 < a_2 < \ldots < a_n$, розглянемо лінійну комбінацію $q_1a_1 + q_2a_2 + \ldots + q_na_n = b$, використовуючи лише цілі значення $q_k ≥ 0$.

Зверніть увагу, що для заданої множини $a_k$ не всі значення $b$ можуть бути можливими. Наприклад, якщо $a_1 = 5$ і $a_2 = 7$, не існує таких $q_1 ≥ 0$ і $q_2 ≥ 0$, щоб $b$ могло бути 1, 2, 3, 4, 6, 8, 9, 11, 13, 16, 18 або 23.

Насправді, 23 — це найбільше неможливе значення $b$ для $a_1 = 5$ і $a_2 = 7$. Тому ми позначаємо $f(5, 7) = 23$. Аналогічно можна показати, що $f(6, 10, 15)=29$ і $f(14, 22, 77) = 195$.

Знайдіть $\sum f(pq,pr,qr)$, де $p$, $q$ і $r$ — прості числа, і $p < q < r < 5000$.
