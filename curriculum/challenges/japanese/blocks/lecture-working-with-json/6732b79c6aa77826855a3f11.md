# --interactive--

JavaScriptにはJSONデータを扱うための強力なメソッドが2つあります：`JSON.parse()` と `JSON.stringify()`。これらのメソッドは、JSON文字列とJavaScriptオブジェクトの間で変換する際によく使われます。

`JSON.stringify()` はJavaScriptオブジェクトをJSON文字列に変換するために使います。これは、データを簡単に共有したりシステム間で転送したりできる形式で保存や送信したい場合に便利です。

`JSON.stringify()` メソッドの使い方は次の通りです：

:::interactive_editor

```js
const user = {
  name: "John",
  age: 30,
  isAdmin: true
};

const jsonString = JSON.stringify(user);
console.log(jsonString);
```

:::

`JSON.stringify()` メソッドは、replacerと呼ばれる関数または配列のオプションパラメータも受け取れます。以下は、オプションのreplacerパラメータに配列を使った例です：

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

// result: {"firstName":"Jessica","country":"USA"}
console.log(JSON.stringify(developerObj, ["firstName", "country"]));
```

:::

この例では、4つのプロパティを持つ`developerObj`があります。`JSON.stringify()` メソッドを使うと、第2パラメータに配列を渡して、文字列化したいプロパティを指定できます。結果は`firstName` と `country` のプロパティだけを含む文字列化されたオブジェクトになります。

`JSON.stringify()` メソッドのもう1つのオプションパラメータはspacerパラメータです。これは文字列化された結果のスペースを調整できます：

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

console.log(JSON.stringify(developerObj, null, 2));

/* result
{
  "firstName": "Jessica",
  "isAwesome": true,
  "isMusician": true,
  "country": "USA"
}
*/
```

:::

ほとんどの場合、`JSON.stringify()` メソッドのこれらのオプションパラメータは使わないかもしれませんが、知っておくと役立ちます。

プログラミングでよく使うもう1つのメソッドは`JSON.parse()` メソッドです。`JSON.parse()` はJSON文字列をJavaScriptオブジェクトに戻します。これはウェブサーバーや`localStorage` からJSONデータを取得し、アプリケーション内でデータを操作したい場合に便利です。`localStorage` については今後のレッスンで詳しく学びます。

`JSON.parse()` メソッドの使い方の例はこちらです：

:::interactive_editor

```js
const jsonString = '{"name":"John","age":30,"isAdmin":true}';
const userObject = JSON.parse(jsonString);
console.log(userObject);

// Result:
// { name: 'John', age: 30, isAdmin: true }
```

:::

これにより、プログラム内で通常のJavaScriptオブジェクトとしてデータを扱えるため、操作や利用が簡単になります。

今後のモジュールでは、JSONや`JSON.parse()` と `JSON.stringify()` メソッドの使い方についてさらに学んでいきます。
