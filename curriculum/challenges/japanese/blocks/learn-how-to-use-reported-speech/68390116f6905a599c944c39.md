# --explanation--

`If only` は過去に起こらなかったことに対する願望や後悔を表すために文の冒頭で使われます。話者が状況が違っていてほしかったことを示しています。例えば：

`If only we had checked the system earlier, we could have avoided the crash.` – これは彼らが早くチェックしなかったことを意味し、今それを後悔していることを表します。
