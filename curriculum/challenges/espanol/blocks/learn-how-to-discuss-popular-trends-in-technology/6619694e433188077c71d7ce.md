# --description--

Normalmente, lo que diferencia un adjetivo de un adverbio es el sufijo `-ly` al final, agregado a un adjetivo para crear un adverbio. Ejemplos: `extreme` (adjetivo) x `extremely` (adverbio) / `quick` (adjetivo) x `quickly` (adverbio).

También, sabe que es posible usar un adverbio como `extremely` para intensificar un adjetivo o incluso otro adverbio. Si dices que algo sucede `quickly`, es una acción rápida. Si es `extremely quickly`, la acción es súper rápida.

Rellena los espacios en blanco de acuerdo con lo que dice Sarah en el diálogo.
