# --explanation--

`Could` 和 `might` 是用来谈论可能性的情态动词。

`Could` 表示某事是可能的，但不确定。例如：

`This update could make the app run faster.` - 该更新有可能提升应用的运行速度。

`Might` 也表示可能性，但通常暗示的可能性比 `could` 更低。例如：

`The new feature might improve the user experience.` - 该特色可能会改善用户体验。
