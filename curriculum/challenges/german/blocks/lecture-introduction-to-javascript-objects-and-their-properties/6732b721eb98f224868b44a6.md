# --interactive--

Es gibt mehrere Möglichkeiten, Eigenschaften aus einem Objekt zu entfernen, wobei der `delete`-Operator die einfachste und am häufigsten verwendete Methode ist.

Wenn Sie `delete` verwenden, wird die ausgewählte Eigenschaft aus dem Objekt entfernt. Hier ist ein Beispiel, wie Sie den `delete`-Operator verwenden:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  job: "Engineer"
};

delete person.job;

console.log(person.job); // undefined
```

:::

In diesem Beispiel beginnen wir mit einem `person`-Objekt, das drei Eigenschaften hat: `name`, `age` und `job`. Dann verwenden wir den `delete`-Operator, um die Eigenschaft `job` zu entfernen. Nach dem Löschen hat das `person`-Objekt die Eigenschaft `job` nicht mehr.

Eine andere Möglichkeit, Eigenschaften zu entfernen, ist die Verwendung von Destrukturierungszuweisung mit Rest-Parametern. Dieser Ansatz löscht die Eigenschaft nicht wirklich, sondern erstellt ein neues Objekt ohne die angegebenen Eigenschaften:

:::interactive_editor

```js
const person = {
  name: "Bob",
  age: 25,
  job: "Designer",
  city: "New York"
};

const { job, city, ...remainingProperties } = person;

// { name: "Bob", age: 25 }
console.log(remainingProperties);
```

:::

In diesem Beispiel verwenden wir Destrukturierung, um `job` und `city` aus dem `person`-Objekt zu extrahieren und die verbleibenden Eigenschaften in einem neuen Objekt namens `remainingProperties` zu sammeln. So entsteht ein neues Objekt ohne die Eigenschaften `job` und `city`.

Zu verstehen, wie man Eigenschaften aus Objekten entfernt, ist eine wichtige Fähigkeit in der JavaScript-Programmierung. Es ermöglicht Ihnen, Objekte dynamisch zu manipulieren und unnötige Daten zu bereinigen.
