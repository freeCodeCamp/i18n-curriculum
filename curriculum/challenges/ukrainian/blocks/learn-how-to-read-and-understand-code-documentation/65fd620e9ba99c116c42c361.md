# --description--

Розробники часто використовують `once` для позначення часу виконання завдань у своєму потоці роботи. Це означає `as soon as` або `when`.

Наприклад, `Once you receive the client's feedback, update the project documentation.` Це означає, що оновлення документації має початися одразу після отримання відгуку, показуючи послідовність дій у життєвому циклі проєкту.
