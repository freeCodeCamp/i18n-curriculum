# --description--

في جميع ورش العمل والمختبرات، ستستخدم محرر Python الخاص بـ freeCodeCamp. لكن من المهم أن تتعلم كيفية إعداد Python على جهازك المحلي.

أسهل طريقة لتثبيت Python على ويندوز وماك هي تنزيل المثبت من الموقع الرسمي لـ Python. سنشرح أيضًا كيفية تشغيل Python على لينكس لاحقًا في هذا الدرس.

اذهب إلى `https://www.python.org/` وحرك المؤشر فوق "التنزيلات". ستظهر نافذة منبثقة تعرض الإصدار الحالي من Python لنظام التشغيل الخاص بك.

سنبدأ بشرح كيفية تثبيت Python على جهاز يعمل بنظام macOS أولاً:

- انقر على الزر الذي يعرض الإصدار الحالي من Python (من النافذة السابقة)، وسيبدأ تنزيل ملف تثبيت `.pkg` تلقائيًا.
- بعد انتهاء تنزيل مثبت `.pkg`، افتحه ثم انقر على "متابعة" في النافذة التي تظهر.
- استمر في النقر على زر "متابعة" حتى تصل إلى قسم "نوع التثبيت". هناك، انقر على زر "تثبيت".
- أدخل كلمة المرور إذا لزم الأمر، ثم ابدأ التثبيت.
- بعد ذلك، يجب أن تظهر رسالة تهنئة تفيد بأن Python تم تثبيته بنجاح.
- انقر على زر "إغلاق"، وبهذا تكون انتهيت!

يمكنك التحقق من التثبيت بفتح الطرفية وتشغيل الأمر `python --version` أو `python3 --version`.

يمكنك أيضًا فتح مفسر Python بتشغيل `python` أو `python3` في الطرفية. مفسر Python هو البرنامج الذي يقرأ شفرة Python، يترجمها إلى تعليمات يفهمها الكمبيوتر وينفذها.

<dfn>الطرفية</dfn> هي واجهة نصية تتيح لك التفاعل مع جهاز الكمبيوتر بكتابة الأوامر. كل نظام تشغيل يأتي مع تطبيق طرفية افتراضي. على macOS، يمكنك استخدام تطبيق Terminal. على ويندوز، يمكنك استخدام Command Prompt أو PowerShell. على لينكس، لكل بيئة سطح مكتب تطبيق طرفية افتراضي خاص بها، مثل GNOME Terminal أو Konsole.

لاحظ أنه في بعض أنظمة macOS ولينكس القديمة، قد يكون الأمر `python` مخصصًا لـ Python 2، بينما `python3` مخصص لـ Python 3 تحديدًا. إذا شغلت `python --version` ورأيت إصدار Python 2 مثل `Python 2.7.18`، فمن المحتمل أن نظام التشغيل يعتمد على بعض البرامج المكتوبة بإصدار Python القديم. في هذه الحالة، يجب عليك استخدام `python3` لتشغيل شفرتك المستقبلية. Python 2 انتهى دعمه ولا يجب استخدامه في أي تطوير جديد.

لتثبيت Python على ويندوز، اتبع الخطوات التالية:

- اذهب إلى `https://www.python.org/`، وحرك المؤشر فوق "التنزيلات". سترى نافذة تقول "تنزيل لويندوز" وزر تنزيل مع الإصدار الحالي من Python.
- انقر على رقم الإصدار، وسيبدأ تنزيل ملف تنفيذي لويندوز (`.exe`) تلقائيًا.
- بعد الانتهاء من تنزيل مثبت Python لويندوز، انقر عليه مرتين، واتبع التعليمات.
- عندما ترى خيار `Add python.exe to Path`، فعّل هذا الخيار، ثم انقر على `Install Now`. هذا سيسهل عليك الأمور لاحقًا.


يمكنك التحقق من التثبيت بفتح واجهة سطر الأوامر مثل PowerShell وتشغيل `python --version`. يمكنك أيضًا فتح مفسر Python بتشغيل `python`.

بالنسبة لـ Python على لينكس، معظم التوزيعات الرئيسية مثل أوبونتو، ديبيان، وفيدورا تأتي مع Python مثبتة.
