# --explanation--

`Lastly` はリストや議論の最後のポイントを紹介するために使われます。例えば：

`Lastly, let's review the budget changes.` – これは話すべき最後の項目であることを意味します。

`Any thoughts on...` は誰かのアイデアや意見を尋ねる非公式な表現です。多くの場合、最初のフレーズ `Do you have any thoughts on...` を省略します。例えば：

`Any thoughts on how we can improve this layout?` – これは話者がレイアウトをより良くする方法について、親しみやすくカジュアルにアイデアを求めていることを意味します。
