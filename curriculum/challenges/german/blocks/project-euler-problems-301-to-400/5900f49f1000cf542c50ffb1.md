# --description--

Das folgende Spiel ist ein klassisches Beispiel der Kombinatorischen Spieltheorie:

Zwei Spieler starten mit einem Streifen aus $n$ weißen Quadraten und wechseln sich ab. In jedem Zug wählt ein Spieler zwei benachbarte weiße Quadrate aus und färbt sie schwarz. Der erste Spieler, der keinen Zug mehr machen kann, verliert.

- $n = 1$: Keine gültigen Züge, daher verliert der erste Spieler automatisch.
- $n = 2$: Nur ein gültiger Zug, nach dem der zweite Spieler verliert.
- $n = 3$: Zwei gültige Züge, aber beide führen zu einer Situation, in der der zweite Spieler verliert.
- $n = 4$: Es gibt drei gültige Züge für den ersten Spieler; dieser kann das Spiel gewinnen, indem er die beiden mittleren Quadrate schwarz färbt.
- $n = 5$: Vier gültige Züge für den ersten Spieler (unten rot markiert); aber egal, was der Spieler macht, gewinnt der zweite Spieler (blau).

<img alt="gültige Startzüge für Streifen mit 5 Quadraten" src="https://cdn.freecodecamp.org/curriculum/project-euler/paper-strip-game.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Für $1 ≤ n ≤ 5$ gibt es also 3 Werte von $n$, für die der erste Spieler einen Gewinn erzwingen kann.

Ebenso gibt es für $1 ≤ n ≤ 50$ 40 Werte von $n$, für die der erste Spieler einen Gewinn erzwingen kann.

Für $1 ≤ n ≤ 1\\,000\\,000$: Wie viele Werte von $n$ gibt es, für die der erste Spieler einen Gewinn erzwingen kann?
