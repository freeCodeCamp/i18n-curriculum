# --description--

考虑边长为 $\sqrt{5}$、$\sqrt{65}$ 和 $\sqrt{68}$ 的三角形。可以证明该三角形的面积为 9。

$S(n)$ 是所有边长为 $\sqrt{1 + b^2}$、$\sqrt{1 + c^2}$ 和 $\sqrt{b^2 + c^2}$（其中 $b$ 和 $c$ 为正整数）且面积为不超过 $n$ 的整数的三角形面积之和。

示例三角形的 $b = 2$ 和 $c = 8$。

$S({10}^6) = 18\\,018\\,206$。

查找 $S({10}^{10})$。
