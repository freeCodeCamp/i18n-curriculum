# --description--

화면 확대기는 저시력 및 기타 시각 장애가 있는 사람들이 디지털 콘텐츠와 웹에 더 잘 접근할 수 있도록 돕는 도구입니다.

이 도구들이 무엇인지, 그리고 디지털 콘텐츠 접근성에서 어떤 역할을 하는지 더 자세히 살펴보겠습니다.

화면 확대기는 컴퓨터나 모바일 기기 화면의 텍스트, 그래픽 및 기타 요소를 확대하여 작동합니다. 많은 화면 확대기는 사용자가 화면을 200% 이상 확대할 수 있게 합니다. 사용자는 포인팅 장치나 키보드를 사용해 페이지를 탐색할 수 있습니다. 또한 대부분의 확대기는 설정에서 확대 비율과 기타 기능을 사용자 맞춤 설정할 수 있습니다.

화면 확대기는 주로 저시력 사용자가 문서나 애플리케이션의 작은 폰트를 읽는 데 도움을 줍니다. 텍스트를 확대함으로써 이메일, 기사 및 기타 콘텐츠를 눈의 피로 없이 읽을 수 있습니다. 화면 확대기는 웹 브라우징에도 도움을 줍니다. 버튼, 링크 및 기타 상호작용 요소를 찾고 클릭하는 데 어려움이 있을 수 있는데, 확대기를 사용하면 가시성이 향상되어 웹사이트를 탐색하고, 양식을 작성하며, 온라인 활동에 문제없이 참여할 수 있습니다.

따라서 소프트웨어 개발자는 저시력 사용자가 디지털 제품에 접근할 수 있도록 해야 합니다. 고려할 점은 다음과 같습니다:

- 사용자가 페이지 크기를 조절해도 레이아웃이 깨지지 않도록 확장 가능한 폰트를 사용하세요.
- 반응형 디자인을 통해 사용자 인터페이스가 다양한 화면 크기에 적응하도록 하세요.
- 고대비 색상 구성과 사용자 맞춤 색상을 사용하세요.
- 확대기를 사용할 때도 콘텐츠를 볼 수 있도록 고정되지 않고 작은 내비게이션 바를 구현하세요.
- 텍스트 이미지를 사용하지 말고 일반 HTML 텍스트를 사용하세요.
- 피드백은 해당 요소 바로 옆에 제공하세요, 그 외에도 여러 가지가 있습니다.

주요 운영체제들은 제조사에서 기본적으로 하나 이상의 확대기를 내장하고 있습니다:

- macOS와 iOS에는 모두 Zoom이 있습니다. macOS에서는 설정에서 접근성을 필터링한 후 "Zoom"을 클릭해 켤 수 있습니다. "키보드 단축키로 확대 사용" 옵션을 켜세요.
  - iPhone에서는 _설정 > 접근성 > Zoom_에서 켤 수 있습니다.
- Android 기기에는 Magnification이 있습니다. 켜려면 _설정 > 특수 기능 > 접근성 > Magnification_으로 이동하세요. 기기마다 다를 수 있으니 설정 홈에서 "Magnification"을 검색해 접근할 수 있습니다.
- Windows에는 Magnifier가 있습니다. _설정 > 접근성 > Magnifier_에서 사용할 수 있습니다.
- Linux 운영체제용 확대기는 다양하며 Zoom 또는 Magnifier 중 하나입니다.

운영체제에 내장된 것 외에도 유용한 서드파티 화면 확대기는 다음과 같습니다:

- Windows용 ZoomText.
- macOS와 Windows용 ClaroView.
- Windows용 iZoom.
- macOS용 Zoomify - Screen Magnifier.
- Windows용 LunarPluse.
- macOS용 Loupe.
