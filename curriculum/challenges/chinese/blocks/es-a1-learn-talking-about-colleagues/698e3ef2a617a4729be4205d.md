# --explanation--

Mateo 问 `¿Dónde vive Valeria?`，意思是“Valeria 住在哪里？”

`¿Dónde...?` 的意思是“哪里...？”

记得在句首加上开头的问号 `¿`，并且在 `ó` 字母上加上重音符号。

`vive` 这个词的意思是“住”，是动词 `vivir` 的第三人称单数形式。
