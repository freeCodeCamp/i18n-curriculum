# --description--

في البرمجة، خاصة عند استخدام أنظمة التحكم في الإصدارات مثل Git، تعني `push` رفع التغييرات من مستودع محلي إلى مستودع بعيد.

الزمن الماضي `pushed` يشير إلى أن هذا الإجراء قد تم بالفعل.

على سبيل المثال، تعني العبارة `He pushed his code to the GitHub repository` أنه رفع تغييرات الشفرة الأخيرة إلى GitHub.
