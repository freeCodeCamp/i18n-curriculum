# --interactive--

CSSでは、画像の周りにボーダーを追加する方法がいくつかあり、それぞれ異なるスタイルオプションや制御レベルを提供します。

ここでは、最も一般的で多用途な手法をいくつか見ていきましょう。

画像にボーダーを追加する最も簡単な方法は、`border` プロパティを使うことです。このプロパティは幅、スタイル、色を一度に設定できるショートハンドです。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

このCSSルールはすべての`img`要素の周りに2ピクセル幅の実線赤ボーダーを追加します。幅やスタイル（`dashed`、`dotted`、`double`など）、色はデザインに合わせて調整できます。

ボーダーの各辺を個別に制御したい場合は、各辺専用のボーダープロパティを使うことができます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border-top: 10px solid red;
  border-right: 10px dashed green;
  border-bottom: 10px dotted blue;
  border-left: 10px double purple;
}
```

:::

これにより、画像の各辺に独自のボーダースタイルを作成できます。

別のボーダー効果を作る方法として、`outline` プロパティを使う手法があります。borderに似ていますが、outlineは要素のサイズやレイアウトに影響を与えません。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  outline: 3px solid gold;
}
```

:::

これにより、画像の周りにサイズや位置を変えずに金色のアウトラインが作成されます。

ボーダーに丸みをつけたい場合は、borderと一緒に`border-radius` プロパティを使うことができます。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

これらの手法は組み合わせてカスタマイズし、独自のボーダー効果を作ることができます。どの方法を使うかは、デザインの要件や必要な複雑さによります。
