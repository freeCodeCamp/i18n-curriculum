# --description--

짝수 양의 정수 $N$이 2의 거듭제곱이거나 서로 다른 소인수가 연속된 소수일 때, $N$을 허용 가능한 수라고 합니다.

처음 열두 개의 허용 가능한 수는 2, 4, 6, 8, 12, 16, 18, 24, 30, 32, 36, 48입니다.

$N$이 허용 가능한 수라면, $N + M$이 소수가 되는 가장 작은 정수 $M > 1$을 $N$의 의사-포춘 수라고 합니다.

예를 들어, $N = 630$은 짝수이고 서로 다른 소인수가 연속된 소수 2, 3, 5, 7이므로 허용 가능합니다. 631 다음의 소수는 641이므로, 630의 의사-포춘 수는 $M = 11$입니다. 또한 16의 의사-포춘 수가 3임을 알 수 있습니다.

${10}^9$보다 작은 허용 가능한 수 $N$에 대한 모든 서로 다른 의사-포춘 수의 합을 구하세요.
