# --description--

Um die Skalierung eines Elements zu ändern, bietet CSS die Eigenschaft `transform` zusammen mit ihrer Funktion `scale()`. Das folgende Codebeispiel verdoppelt die Größe aller Absatz-Elemente auf der Seite:

```css
p {
  transform: scale(2);
}
```

# --instructions--

Vergrößern Sie die Größe des Elements mit der ID `ball2` auf das 1,5-fache seiner ursprünglichen Größe.
