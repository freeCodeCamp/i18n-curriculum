# --description--

Immagina di avere due stringhe che vuoi combinare per creare un unico messaggio, come nell'esempio seguente:

```js
const str1 = "Hello";
const str2 = "World!";
const message = str1.push(str2);
```

<img src="https://cdn.statically.io/gh/TheOdinProject/curriculum/4ed59981b4ce2c60b5b83bf7415d3127b61821f5/foundations/javascript_basics/understanding_errors/imgs/03.png" style="width:100%" alt="un esempio di errore di tipo nella console sviluppatore di Chrome">

Qui, otterrai un `TypeError` con un messaggio che dice che `str1.push is not a function`. Questo è un messaggio di errore comune che confonde chi sta imparando perché potresti sapere che `.push()` è certamente una funzione (per esempio, se l'hai usata per aggiungere elementi agli array in precedenza).

Ma questo è il punto chiave: `.push()` non è un metodo delle stringhe, è un metodo degli array. Perciò, non è “una funzione” che puoi trovare come metodo delle stringhe. Se cambi `.push()` in `.concat()`, un metodo corretto delle stringhe, il nostro codice funziona come previsto!

Una buona nota da tenere a mente quando ti trovi davanti a un `TypeError` è considerare il tipo di dato su cui stai cercando di eseguire un metodo o un'operazione. Probabilmente scoprirai che non è quello che pensi, oppure che l'operazione o il metodo non è compatibile con quel tipo.

# --questions-- 

## --text--

Perché il seguente codice JavaScript genera un `TypeError`?

```js
const str1 = "Hello";
const str2 = "World!";
const message = str1.push(str2);
```

## --answers--

Il metodo `.push()` può essere usato solo sugli oggetti, non sulle stringhe.

---

Il metodo `.push()` non è disponibile per le stringhe perché è un metodo degli array.

---

Il metodo `.push()` è scritto in modo errato e dovrebbe essere .pusch().

---

Il `TypeError` è il risultato di un errore di sintassi nel codice JavaScript.
