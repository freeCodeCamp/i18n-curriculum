# --explanation--

Уявіть, що ви пишете код і хочете швидко знайти потрібну функцію або змінну, не вводячи все повністю.

`Auto-completion` допомагає, пропонуючи те, що ви, ймовірно, намагаєтеся написати, на основі того, що вже введено. Це робить кодування швидшим і зменшує кількість помилок.
