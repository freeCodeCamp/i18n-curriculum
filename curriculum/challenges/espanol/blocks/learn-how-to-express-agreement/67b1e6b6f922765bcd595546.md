# --explanation--

`Make sure` significa tomar acción para confirmar que algo es correcto o está completado. Por ejemplo:  

`Please make sure the report is submitted before the deadline.` – Esto significa revisar y asegurarte de que el informe se entregue a tiempo.  

`Reliable` significa que algo o alguien puede ser confiable para funcionar bien o para ser consistente a lo largo del tiempo. Por ejemplo:

`The server is reliable; it has been running without any issues for months.` - Esto significa que el servidor funciona de manera constante y no causa problemas.

`More reliable` significa que algo se ha vuelto aún más confiable o seguro. Por ejemplo:

`The new software update makes the app more reliable.` - Esto significa que la app se ha vuelto más confiable o estable después de la actualización.
