# --description--

シーケンスは次のように定義されます：

- $g_k = 1$、ただし $0 ≤ k ≤ 1999$
- $g_k = g_{k - 2000} + g_{k - 1999}$、ただし $k ≥ 2000$。

$k = {10}^{18}$ のときの $g_k$ を 20092010 で割った余りを求めてください。
