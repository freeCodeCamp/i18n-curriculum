Повне речення: `Mi correo es mateo.delgado@sanjuan.pr.`

Тут `correo` означає адресу е-пошти.

Структура `Mi` + іменник + `es...` використовується для надання особистої інформації. Наприклад:

- `Mi nombre es Ana.` - Мене звати Ана.

- `Mi número es 555-1234.` - Мій номер 555-1234.
