# --description--

`Several` é uma palavra usada para indicar uma quantidade não especificada que é mais que dois mas não muitos.É frequentemente usado quando o número exato não é importante ou não é conhecido. 

# --questions--

## --text--

O que `several` implica na frase de Bob?

## --answers--
