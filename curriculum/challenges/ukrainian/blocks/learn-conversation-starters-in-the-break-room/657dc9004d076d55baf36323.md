# --description--

Пам’ятайте, `occasionally` означає, що щось відбувається іноді, але не регулярно чи часто. Це трапляється рідше, ніж `sometimes`, і частіше, ніж `rarely`.

Ось діаграма частоти прислівників із останнього завдання для вашої зручності.

| Прислівник   | Частота  | Прислівник | Частота  |
| ------------ | -------- | ---------- | -------- |
| Never        | 0%       | Often      | 70%      |
| Rarely       | 5%       | Generally  | 80%      |
| Seldom       | 15%      | Usually    | 90%      |
| Occasionally | 30%      | Always     | 100%     |
| Sometimes    | 50%      |            |          |
