# --description--

令 $S(A)$ 表示大小為 n 的集合 A 中元素的總和。若對於任意兩個非空且不相交的子集合 B 和 C，以下性質皆成立，則我們稱之為特殊總和集合：

1. $S(B) ≠ S(C)$；也就是說，子集合的總和不能相等。
2. 如果 B 包含比 C 更多的元素，則 $S(B) > S(C)$。

對於這個問題，我們假設給定的集合包含 n 個嚴格遞增的元素，且它已經符合第二條規則。

令人驚訝的是，對於 n = 4 的集合，可取得的 25 個可能子集配對中，只有 1 個配對需要測試相等性（第一條規則）。同樣地，當 n = 7 時，966 個子集配對中只有 70 個需要測試。

對於 n = 12，必須測試多少個可取得的 261625 個子集合配對的相等性？

**注意：** 此問題與問題 103 和問題 105 有關。
