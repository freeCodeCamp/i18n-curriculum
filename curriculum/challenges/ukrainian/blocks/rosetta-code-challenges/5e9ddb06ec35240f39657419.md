# --description--

Напишіть програму, яка генерує масив цілих чисел від 1 до 100 (включно). Але:

<ul>
    <li>для кратних 3 додайте <code>"Fizz"</code> до масиву замість числа</li>
    <li>для кратних 5 додайте <code>"Buzz"</code> до масиву замість числа</li>
    <li>для кратних 3 і 5 додайте <code>"FizzBuzz"</code> до масиву замість числа</li>
</ul>
