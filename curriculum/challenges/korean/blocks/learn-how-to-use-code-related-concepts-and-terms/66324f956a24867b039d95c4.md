# --description--

단어 `should`는 조언이나 제안을 할 때 쓰이는 조동사입니다. 뒤에는 동사의 기본형이 옵니다. 예를 들어, 문장 `You should try that restaurant`에서 `should`는 동사의 기본형인 `try` 뒤에 옵니다.

동사 `pick`는 여러 선택지 중에서 고르거나 선택하는 것을 의미합니다. 프로그래밍 맥락에서는 `picking a data type`가 작업에 가장 적합한 정수, 문자열, 불리언 같은 데이터 유형을 선택하는 것을 뜻합니다.

`Match`는 두 가지가 서로 잘 맞거나 적합한지 확인하는 것을 의미합니다. 예를 들어, `This key matches the lock.`는 열쇠가 자물쇠에 적합하다는 것을 나타냅니다.
