# --interactive--

固定定位和粘性定位是两种重要的 CSS 定位策略，与绝对定位相比，它们都具有不同的行为。 当使用 `position: fixed` 定位元素时，它会从正常文档流中移除并相对于视口放置，这意味着即使用户滚动，它也会保持在相同的位置。 这通常用于标题或导航栏等需要始终保持可见的元素。 例如，如果希望标题固定在页面顶部，可以使用以下代码：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Fixed Header</h1>

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer nec odio. Praesent libero. Sed cursus ante dapibus diam.</p>
<p>Sed nisi. Nulla quis sem at nibh elementum imperdiet. Duis sagittis ipsum. Praesent mauris.</p>
<p>Fusce nec tellus sed augue semper porta. Mauris massa. Vestibulum lacinia arcu eget nulla.</p>
<p>Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos.</p>
<p>Curabitur sodales ligula in libero. Sed dignissim lacinia nunc. Curabitur tortor.</p>
<p>Pellentesque nibh. Aenean quam. In scelerisque sem at dolor. Maecenas mattis.</p>
<p>Sed convallis tristique sem. Proin ut ligula vel nunc egestas porttitor. Morbi lectus risus.</p>
<p>Donec congue lacinia dui, a porttitor lectus condimentum laoreet. Nunc eu ullamcorper orci.</p>
<p>Quisque eget odio ac lectus vestibulum faucibus eget in metus. In pellentesque faucibus vestibulum.</p>
<p>Nulla at nulla justo, eget luctus tortor. Nulla facilisi. Duis aliquet egestas purus in blandit.</p>
```

```css
body {
  margin: 0;
  padding-top: 60px;
  font-family: Arial, sans-serif;
  line-height: 1.6;
}

h1 {
  position: fixed;
  top: 0;
  width: 500px;
  background: white;
  padding: 10px;
  border-bottom: 2px solid #ccc;
}

p {
  max-width: 600px;
  margin: 20px auto;
}
```

:::

在此示例中，`h1` 元素将紧贴视口顶部，即使用户滚动，它也将保持原位。 这对于创建粘性标题或始终可见的导航等持久性用户界面元素尤其有用。

`position: sticky` 是相对定位和固定定位的混合体。 起初，元素的行为就像相对定位一样，保持在文档流程中。 但是，一旦用户滚动元素超过某个点，它就会“粘”在视口（通常是顶部）上，并表现得像是固定定位。 这非常适合创建粘性导航栏等元素，只有当用户滚动到某个位置时，导航栏才会固定下来。 这是应用粘性定位的方式：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Sticky Header</h1>

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer nec odio. Praesent libero. Sed cursus ante dapibus diam.</p>
<p>Sed nisi. Nulla quis sem at nibh elementum imperdiet. Duis sagittis ipsum. Praesent mauris.</p>
<p>Fusce nec tellus sed augue semper porta. Mauris massa. Vestibulum lacinia arcu eget nulla.</p>
<p>Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos.</p>
<p>Curabitur sodales ligula in libero. Sed dignissim lacinia nunc. Curabitur tortor.</p>
<p>Pellentesque nibh. Aenean quam. In scelerisque sem at dolor. Maecenas mattis.</p>
<p>Sed convallis tristique sem. Proin ut ligula vel nunc egestas porttitor. Morbi lectus risus.</p>
<p>Donec congue lacinia dui, a porttitor lectus condimentum laoreet. Nunc eu ullamcorper orci.</p>
<p>Quisque eget odio ac lectus vestibulum faucibus eget in metus. In pellentesque faucibus vestibulum.</p>
<p>Nulla at nulla justo, eget luctus tortor. Nulla facilisi. Duis aliquet egestas purus in blandit.</p>
```

```css
h1 {
  position: sticky;
  top: 30px;
  left: 30px;
}
```

:::

在此示例中，`h1` 元素将正常表现，直到用户将其向上滚动 `30px`，此时它将“粘附”在视图顶部。

这是绝对定位、固定定位和粘性定位之间的一些主要区别：

- 绝对定位将元素从文档流中移除，并将其相对于位置最近的已定位祖先定位，如果没有祖先，则相对于初始包含块定位。 无论滚动与否，元素都会保持在该位置。
- 固定定位也是将元素从文档流中移除，但将其固定在相对于视口的位置，这意味着即使页面滚动，它也会保持在同一位置可见。
- 粘性定位最初会让元素保持在正常流程中，但在滚动超过定义的阈值后，它就会粘在原处。

通过这些策略，你可以创建包含浮动元素、粘性标题和更多动态用户界面组件的布局，使重要信息随时都能被访问，从而增强用户体验。
