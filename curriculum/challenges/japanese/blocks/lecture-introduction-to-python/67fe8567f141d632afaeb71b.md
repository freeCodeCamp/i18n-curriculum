# --description--

前のレッスンでは、Pythonとは何か、そしてPythonで何ができるかを学びました。今回は、ローカルマシンにPythonをセットアップする方法を見ていきましょう。

WindowsやMacにPythonをインストールする最も簡単な方法は、公式Pythonウェブサイトからインストーラーをダウンロードすることです。このレッスンの後半ではLinuxでPythonを実行する方法も説明します。

`https://www.python.org/`にアクセスして、「Downloads」にカーソルを合わせてください。お使いのOSに対応した現在のPythonバージョンが表示されるモーダルが現れます。

まずはmacOSでPythonをインストールする方法を説明します：

- 先ほどのモーダルに表示されている現在のPythonバージョンのボタンをクリックすると、`.pkg`のインストールファイルのダウンロードが自動的に始まります。
- `.pkg`インストーラーのダウンロードが完了したら、それを開き、表示されるウィンドウで「Continue」をクリックしてください。
- 「Continue」ボタンをクリックし続けて「Installation Type」セクションまで進み、そこで「Install」ボタンをクリックします。
- 必要に応じてパスワードを入力し、インストールを開始してください。
- インストールが完了すると、Pythonが正常にインストールされたことを知らせるお祝いのメッセージが表示されます。
- 「Close」ボタンをクリックして完了です！

インストールが成功したかどうかは、ターミナルを開いて`python --version`または`python3 --version`を実行して確認できます。

また、ターミナルで`python`または`python3`を実行すればPythonインタープリターを起動できます。

<dfn>ターミナル</dfn>とは、コマンドを入力してコンピューターとやり取りできるテキストベースのインターフェースです。各OSにはデフォルトのターミナルアプリが付属しています。macOSではTerminalアプリ、WindowsではCommand PromptやPowerShell、LinuxではGNOME TerminalやKonsoleなどのデスクトップ環境ごとのターミナルアプリがあります。

古いmacOSやLinuxシステムでは、`python`がPython 2用に予約されている場合があり、`python3`はPython 3専用です。もし`python --version`を実行して`Python 2.7.18`のようなPython 2のバージョンが表示されたら、そのOSは古いPythonバージョンで書かれたソフトウェアに依存している可能性があります。その場合は、今後Pythonコードを実行するときに`python3`を使うべきです。Python 2はサポート終了しており、新しい開発には使わないでください。

WindowsにPythonをインストールするには、以下の手順に従ってください：

- `https://www.python.org/`にアクセスし、「Downloads」にカーソルを合わせます。すると「Download for Windows」と書かれたモーダルと、現在のPythonバージョンのダウンロードボタンが表示されます。
- バージョン番号をクリックすると、Windows用の実行可能ファイル（`.exe`）のダウンロードが自動的に始まります。
- Pythonインストーラーのダウンロードが完了したら、ダブルクリックして指示に従ってください。
- `Add python.exe to Path`というオプションが表示されたら、そのチェックボックスをオンにしてから`Install Now`をクリックしてください。これにより後で作業が楽になります。

インストールが成功したかどうかは、PowerShellなどのコマンドラインシェルを開いて`python --version`を実行して確認できます。Pythonインタープリターは`python`を実行すれば起動します。

Linuxの主要なディストリビューション（Ubuntu、Debian、Fedoraなど）にはほとんどPythonがプリインストールされています。

ターミナルを開いて`python --version`または`python3 --version`を実行してください：

どちらのコマンドもPythonのバージョンを表示しない場合は、`https://www.python.org`でお使いのLinuxディストリビューション向けのインストールパッケージを探すか、オンラインで推奨されるPythonのインストール方法を検索してください。
