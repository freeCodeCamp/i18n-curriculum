# --description--

Al dar un nombre o una dirección de correo electrónico, una persona puede necesitar decir cada letra por separado para evitar confusiones. Esta acción se llama `deletrear`. El verbo `deletrear` se usa cuando alguien deletrea una palabra letra por letra. Por ejemplo:

- `Deletreo mi apellido.`

- `¿Quieres que deletree tu nombre?` - ¿Quieres que deletree tu nombre?
