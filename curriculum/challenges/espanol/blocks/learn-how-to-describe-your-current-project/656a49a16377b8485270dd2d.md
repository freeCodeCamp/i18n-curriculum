# --description--

`What are you up to` es una forma amistosa de preguntar a alguien qué está haciendo. Es como preguntar, "¿Qué estás haciendo estos días?"

Por ejemplo:

- `What are you up to this weekend?` - Significa "¿Cuáles son tus planes para este fin de semana?"

- `He is up to something.` - Significa "Está haciendo algo."
