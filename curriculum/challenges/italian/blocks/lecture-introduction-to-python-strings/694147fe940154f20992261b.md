# --description--

In una lezione precedente, hai imparato come ogni carattere in una stringa può essere identificato dal suo indice (che parte da zero) e consultato usando la notazione a parentesi:

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
print(my_str[-1]) # d
```

Lo **string slicing** ti permette di estrarre una porzione di una stringa o lavorare solo con una parte specifica di essa. Ecco la sintassi base:

```python
string[start:stop]
```

Se vuoi estrarre i caratteri da un certo indice a un altro, devi solo separare gli indici `start` e `stop` con due punti:

```python
my_str = 'Hello world'
print(my_str[1:4]) # ell
```

Nota che l’indice `stop` non è incluso, quindi `[1:4]` ha appena estratto i caratteri dall’indice `1` fino a, ma senza includere, il carattere all’indice `4`.

Puoi anche omettere gli indici `start` e `stop`, e Python userà come predefinito `0` o la fine della stringa, rispettivamente. Per esempio, ecco cosa succede se ometti l’indice `start`:

```python
my_str = 'Hello world'
print(my_str[:7])  # Hello w
```

Questo estrae tutto dall’indice `0` fino a (ma senza includere) il carattere all’indice `7`. Ecco cosa succede se ometti l’indice `stop`:

```python
my_str = 'Hello world'
print(my_str[8:])  # rld
```

Questo estrae tutto dal carattere all’indice `8` fino alla fine della stringa.

Nota che fare slicing su una stringa non modifica la stringa originale:

```py
my_str = 'Hello world'
print(my_str[8:])  # rld
print(my_str)  # Hello world
```

Puoi anche omettere entrambi gli indici `start` e `stop`, così estrarrai l’intera stringa:

```python
my_str = 'Hello world'
print(my_str[:])  # Hello world
```

Oltre agli indici `start` e `stop`, c’è anche un parametro opzionale `step`, che serve a specificare l’incremento tra ogni indice nello slice.

Ecco la sintassi per questo:

```python
string[start:stop:step]
```

Nell’esempio qui sotto, lo slicing inizia all’indice `0`, si ferma prima di `11` ed estrae ogni secondo carattere:

```python
my_str = 'Hello world'
print(my_str[0:11:2])  # Hlowrd
```

Un trucco utile che puoi fare con il parametro `step` è invertire una stringa impostando step a `-1` e lasciando `start` e `stop` vuoti:

```python
my_str = 'Hello world'
print(my_str[::-1]) # dlrow olleH
```
