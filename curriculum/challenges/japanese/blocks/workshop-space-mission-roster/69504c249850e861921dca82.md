# --description--

`EVAChunks` は基本的に配列の中に配列がある、つまり二次元（2D）配列であることに気づいたかもしれません。ネストされた `for` ループを使って、2D 配列のデータをログに出力できます。

```js
for (let i = 0; i < rootArray.length; i++) {
  console.log(`Group ${i + 1}:`);
  for (let j = 0; j < rootArray[i].length; j++) {
    console.log(rootArray[i][j].property);
  }
}
```

上の例では、`rootArray` がメインの配列で、`rootArray[i]` はメイン配列の中のサブ配列、`rootArray[i][j]` はそのサブ配列の中の1つのオブジェクト、そして `.property` はそのオブジェクト内の値にアクセスしています。

ネストされた `for` ループを使って `EVAChunks` を繰り返し処理し、各チャンクのすべての宇宙飛行士の `name` をログに出力してください。必ず以下を守ってください：

- `EVAChunks` をルート配列として使うこと
- チャンク間に `Chunk ${i+1}:` をログに出力すること
