# --explanation--

Wakati wa `Present Continuous` hutumika kuelezea vitendo vinavyotokea wakati wa kuzungumza. Hutengenezwa kwa kutumia kitenzi `to be` (`am, is, are`) + aina ya msingi ya kitenzi + `-ing`. Kwa mfano:

- `She is coding a new feature right now.` - Yeye yuko katika mchakato wa kuandika kipengele kipya wakati huu.

- `They are testing the software.` - Wao wako katikati ya kujaribu programu sasa hivi.
