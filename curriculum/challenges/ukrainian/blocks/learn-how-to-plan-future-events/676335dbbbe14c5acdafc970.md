# --explanation--

Зверніть увагу на дієслово `share`. Воно означає надати або зробити щось доступним для інших. Марія питає, чи буде інформація поширена.

`Other departments` вказує, хто може отримати цю інформацію. Запитання Марії стосується саме цих відділів.

Структура `Are we going to...` показує, що Марія питає про план на майбутнє.

Визначивши ці частини речення, ви зрозумієте, що Марія запитує, чи буде інформація поширена серед інших відділів.
