# --interactive--

У JavaScript усі арифметичні оператори мають форму складеного присвоєння. Складені оператори присвоєння надають коротший запис для операції над змінною з подальшим збереженням результату в цю ж змінну. Вони поєднують операцію і присвоєння у коротшу форму, як `x += y`, що є еквівалентним запису `x = x + y`, але без повторення імені змінної. Замість того, щоб писати щось на кшталт цього:

:::interactive_editor

```js
let num = 5;
num = num + 2;

console.log(num); // 7
```

:::

Ви можете написати так:

:::interactive_editor

```js
let num = 5;
num += 2;

console.log(num); // 7
```

:::

Зверніть увагу, як `num += 2` поєднує кроки додавання та присвоєння в один. Це економить час і зменшує захаращення у вашому коді. Давайте детальніше розглянемо найпоширеніші складені оператори присвоєння в JavaScript.

Як ви вже бачили, оператор `+=` дозволяє додати значення до існуючої змінної. Він відомий як оператор додавання з присвоєнням. Оператор додавання з присвоєнням бере поточне значення змінної, додає до нього вказане число, а потім присвоює результат назад змінній:

:::interactive_editor

```js
let total = 10;
total += 5;

console.log(total); // 15
```

:::

Як ви можете здогадатися, існує оператор віднімання з присвоєнням, позначений `-=`. Оператор віднімання з присвоєнням віднімає вказане значення від поточного значення змінної і присвоює нове значення назад змінній:

:::interactive_editor

```js
let score = 20;
score -= 7;

console.log(score); // 13
```

:::

Якщо б ви не використовували оператор віднімання з присвоєнням, ви б зробили щось на кшталт цього:

:::interactive_editor

```js
let score = 20;
score = score - 7;

console.log(score); // 13
```

:::

Оператор множення з присвоєнням позначається `*=`. Він множить поточне значення змінної на вказане число і знову присвоює його змінній:

:::interactive_editor

```js
let points = 5;
points *= 3;

console.log(points); // 15
```

:::

Нарешті, існує оператор ділення з присвоєнням, позначений `/=`. Як і інші, він дозволяє поділити поточне значення змінної на вказане число, а потім присвоїти результат назад змінній:

:::interactive_editor

```js
let balance = 100;
balance /= 4;

console.log(balance); // 25
```

:::

Пам’ятайте, що для кожного оператора в JavaScript існує складений оператор присвоєння. Тож, окрім чотирьох уже згаданих, ми також маємо:

- Оператор присвоєння остачі (`%=`), який ділить змінну на вказане число і присвоює змінній остачу.

- Оператор присвоєння степеня (`**=`), який підносить змінну до степеня вказаного числа і присвоює результат змінній.

- Оператор побітового AND з присвоєнням (`&=`), який виконує побітову операцію AND з вказаним числом і присвоює результат змінній.

- Оператор побітового OR з присвоєнням (`|=`), який виконує побітову операцію OR з вказаним числом і присвоює результат змінній.
