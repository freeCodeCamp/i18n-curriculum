# --interactive--

ウェブページ上のリンクをクリックした後に紫色に変わるのを見たことがあるかもしれません。これはリンクの状態が変わったためで、異なるスタイルが適用されるからです。リンクには5つの異なる状態があります。

最初はデフォルトの状態で、`:link`です。この状態は、ユーザーがまだ訪問、クリック、または操作していないリンクを表します。この状態はページ上のすべてのリンクの基本スタイルを提供していると考えてください。他の状態はこれに上乗せされます。

インタラクティブエディターを有効にして、プレビューウィンドウのリンクを見てみましょう。デフォルトの青色で表示されているはずで、これはデフォルト状態を表しています。クリックすると紫色に変わります。

:::interactive_editor

```html
<a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
```

:::

2番目の状態は`:visited`で、ユーザーがリンク先のページをすでに訪問した場合に適用されます。デフォルトではリンクが紫色になりますが、CSSを活用してユーザーに異なる視覚的な表示を提供することもできます。これは、ユーザーがドキュメントの一部をすでに読んだことを知らせたり、以前に利用したことがある信頼できるサイトであることを示すのに役立ちます。

プレビューウィンドウのリンクをクリックすると、訪問済みリンクが茶色に変わるのがわかります。プレビューを見るにはインタラクティブエディターを有効にする必要があります。

**注意**：このインタラクティブ例ではリンクスタイルの変化を確認できるように一部CSSが提供されています。CSSコードの意味は今後のモジュールで学ぶので、理解しようと心配しないでください。

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:visited {
  color: brown;
}
```

:::

3番目の状態は`:hover`です。この状態はユーザーがリンクにカーソルをホバーしているときに適用されます。この状態はリンクに注目を集め、ユーザーが実際にクリックする意図があることを確認するのに役立ちます。

インタラクティブエディターを有効にして、リンクにマウスをホバーしてみてください。色が赤に変わるのが見えます。

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:hover {
  color: red;
}
```

:::

次に`:focus`があります。この状態はリンクにフォーカスが当たっているときに適用されます。

プレビューウィンドウの空白部分をクリックしてからキーボードの`tab`を押してください。リンクが緑色に変わるはずです。プレビューを見るにはインタラクティブエディターを有効にしてください。

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:focus {
  color: green;
}
```

:::

最後に`:active`があります。この状態はユーザーがリンクをアクティブにしているときに適用されます。通常は左クリックでリンクをクリックすることを意味します。この状態は、ユーザーがクリックした要素がインタラクティブであることを示すのに役立ちます。

インタラクティブエディターを有効にしてリンクをクリックしてください。リンクが黒に変わるのが見えます。この変化はかなり速いので、色の変化を見るために何度かクリックする必要があるかもしれません。

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:active {
  color: black;
}
```

:::

これらの状態を使ってリンクのスタイルを設定するときは、CSSを書く順番が決まっています：`link`、`visited`、`hover`、`focus`、そして`active`の順です。

これで、ページのリンクに自分らしいスタイルを付けつつ、ユーザーにリンクの状態に関する重要な情報を提供できるようになりました。
