# --description--

Un `operador de asignación` en programación se utiliza para asignar un valor a una variable. En muchos lenguajes de programación, incluido Python, el `operador de asignación` se representa con el símbolo `=`. Indica a la computadora que almacene el valor que está en el lado derecho del `=` en la variable nombrada en el lado izquierdo. Por ejemplo, `number = 10` asigna el valor `10` a la variable `number`.
