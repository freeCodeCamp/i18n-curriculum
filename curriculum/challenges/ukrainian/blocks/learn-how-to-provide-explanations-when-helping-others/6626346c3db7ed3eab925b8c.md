# --description--

У повсякденному житті `right` і `left` позначають напрямки або сторони. Наприклад, коли ви повертаєте `right`, ви рухаєтеся в напрямку `right` сторони.

На комп’ютерній миші `right click` і `left click` означають натискання `right` або `left` кнопки, зазвичай для різних дій. `right click` часто відкриває додаткові опції або меню, тоді як `left click` використовується для вибору елементів або навігації.

# --questions--

## --text--

У чому різниця між `right click` і `left click` на миші?

## --answers--

`Right click` використовується для навігації, `left click` — для додаткових опцій.

### --feedback--

Насправді навпаки: `right click` для опцій, `left click` для навігації.

---

`Right click` відкриває більше опцій, `left click` використовується для вибору або навігації.

---

Обидва натискання виконують однакову функцію.

### --feedback--

`Right and left clicks` зазвичай мають різні функції.

---

`Right click` використовується для вимкнення комп’ютера, `left click` — для перезавантаження.

### --feedback--

Ці натискання призначені для навігації та опцій, а не для вимкнення чи перезавантаження комп’ютера.
