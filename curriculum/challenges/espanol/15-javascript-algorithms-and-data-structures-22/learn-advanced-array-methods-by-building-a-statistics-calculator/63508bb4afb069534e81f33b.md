# --description--

El primer paso para calcular la mediana es asegurarse de que la lista de números esté ordenada de menor a mayor. Una vez más, existe un método de arreglos ideal para esto: el método `.sort()`.

Declara una variable `sorted` y asigna `array.sort()` a la misma.
