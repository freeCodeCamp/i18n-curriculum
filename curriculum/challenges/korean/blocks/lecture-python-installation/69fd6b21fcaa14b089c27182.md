# --description--

이전 수업에서는 로컬 장치에 Python을 설치하는 방법을 배웠습니다. 이번 수업에서는 Python 코드를 로컬에서 실행하는 방법을 배우세요.

로컬에서 Python 스크립트를 작성하기 전에 코드 편집기나 IDE를 설치해야 합니다. IDE는 통합 개발 환경을 뜻하며, 코드 편집기, 테스트 도구, 터미널 등 여러 기능을 포함합니다.

많은 개발자가 사용하는 인기 있는 코드 편집기 중 하나는 VS Code입니다. `https://code.visualstudio.com/download`에 방문해 Mac, Windows, Linux 환경을 지원하는 VS Code를 다운로드하세요. 이번 수업에서는 이 편집기를 사용합니다.

Python 개발에 흔히 사용되는 다른 선택지는 다음과 같습니다:

- PyCharm
- Spyder

편집기를 다운로드한 후에는 열어야 합니다. 열면 폴더를 열어야 합니다. 왼쪽 상단의 `File` 메뉴 옵션을 클릭하고 `Open Folder`를 클릭하세요. 거기서 프로젝트용 폴더를 선택하거나 새로 만드세요. 수업 외 연습용으로는 `python-projects`라는 폴더를 만들어도 좋습니다.

폴더를 설정한 후에는 VS Code 왼쪽 사이드바인 탐색기를 열 수 있습니다. `New File` 아이콘(종이 위에 더하기 아이콘처럼 생긴)을 클릭하세요. 그런 다음 `main.py`라는 파일을 만들고 엔터를 누르세요. `.py`는 이 파일이 Python 파일임을 나타내는 확장자입니다.

`main.py` 파일을 열고 다음 코드를 입력하세요:

```py
print("Hello, world!")
```

이 코드를 실행하는 방법은 여러 가지가 있습니다. 첫 번째 방법은 오른쪽 상단에 있는 재생 버튼처럼 생긴 실행 버튼을 클릭하는 것입니다. 그러면 터미널이 열리고 Python 스크립트가 실행됩니다. `"Hello, world!"`라는 텍스트가 표시되어야 합니다.

또 다른 방법은 터미널을 열고 다음 명령어로 수동으로 프로그램을 실행하는 것입니다:

```bash
python main.py
```

이 명령어는 `main.py`가 저장된 폴더에서 실행해야 합니다. 예를 들어 `main.py`가 `python-projects`라는 폴더 안에 있다면, 먼저 터미널에서 다음을 사용하세요:

```bash
cd python-projects
```

그런 다음 다음을 실행하세요:

```bash
python main.py
```

특히 macOS와 Linux 시스템에서 Python 2와 Python 3가 모두 설치된 환경에서는 `python3` 대신 `python`를 사용해야 할 수 있습니다. 이는 `python` 명령어가 없거나 이전 버전 Python을 가리키는 경우에 흔합니다.

```bash
python3 main.py
```

수업을 진행하면서 수업 예제를 로컬에서 실행해 보고 결과를 확인하세요. 또한 수업에서 배운 내용을 테스트할 자신만의 예제를 만들어 보는 것도 좋습니다.
