# --explanation--

`user interface` (UI) означає ту частину системи або програмного забезпечення, з якою користувачі взаємодіють безпосередньо, наприклад кнопки, меню та екрани. Наприклад:

`The user interface was redesigned to be more intuitive.` — означає, що покращили ту частину програмного забезпечення, з якою взаємодіє користувач.

`User feedback` означає думки, пропозиції або проблеми, які повідомляють користувачі про свій досвід із продуктом або послугою. Наприклад:

`The company improved the app based on user feedback.` — означає, що коментарі та пропозиції від користувачів були прочитані і використані для покращення застосунку.

У цьому контексті Лінда пояснює, що команда покладалася на відгуки користувачів, щоб постійно вдосконалювати інтерфейс користувача, забезпечуючи його відповідність потребам користувачів.
