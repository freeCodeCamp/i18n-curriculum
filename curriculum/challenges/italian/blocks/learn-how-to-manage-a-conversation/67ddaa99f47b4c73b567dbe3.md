# --explanation--

`Sure thing` è un modo informale e amichevole per dire `Of course` o `I'll do it`. Mostra che chi parla accetta la richiesta senza esitazioni. Per esempio:

- **Ashley:** `Can you send me the file this afternoon?` - Questa persona sta chiedendo un file.

- **Sam:** `Sure thing.` - Questo significa che chi parla è d'accordo e invierà il file come richiesto.
