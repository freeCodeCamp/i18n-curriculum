# --explanation--

축약형 `We'll`은 `We will`를 나타내며, 이는 미래의 행동이나 계획을 의미합니다. 예를 들어:

- `I'll call you tomorrow.` - 호출하는 행동이 미래에 일어날 것임을 의미합니다.

- `We'll review the report later.` - 검토가 계획되었지만 아직 이루어지지 않았음을 나타냅니다.
