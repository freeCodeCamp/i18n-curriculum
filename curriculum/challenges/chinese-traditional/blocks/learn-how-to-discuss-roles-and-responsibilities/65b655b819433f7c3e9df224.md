# --description--

當你分享從別人那裡聽來的資訊或謠言時，會使用片語 `word is`。這就像是在說 `people say` 或 `I've heard`。

例如，如果有人說，`Word is that the new update will make the program faster`，他們的意思是他們聽說新更新會提升程式的速度。
