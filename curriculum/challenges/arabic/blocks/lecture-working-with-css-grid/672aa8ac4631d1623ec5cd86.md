# --interactive--

CSS Grid هو نظام تخطيط قوي يسمح لمطوّري الويب بإنشاء تخطيطات صفحات ويب معقدة ومتجاوبة بسهولة.

تخيل أنك ترتب الأثاث في غرفة – CSS Grid يشبه وجود شبكة غير مرئية على الأرض تساعدك في وضع كل شيء بدقة حيث تريد.

عندما نبني مواقع الويب، نحتاج غالبًا إلى ترتيب عناصر مختلفة على الصفحة.

قبل CSS Grid، كان هذا أحيانًا صعبًا، خاصةً للتخطيطات المعقدة. CSS Grid يبسط هذه العملية بتقسيم صفحة الويب إلى صفوف وأعمدة، مكونًا هيكلًا شبيهًا بالشبكة.

تخيل أنك تعمل مع حاوية `div` تحتوي على عدة عناصر مسكنة بداخلها هكذا:

:::interactive_editor

```html
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

:::

إذا أردت تنسيق تلك العناصر بتخطيط شبكي، يمكنك ضبط `display` إلى `grid` وتطبيق الأعمدة هكذا:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 20px;
}

.item {
  background-color: lightgray;
  padding: 20px;
  text-align: center;
  border: 1px solid #ccc;
}
```

:::

في هذا الكود، نخبر المتصفح بإنشاء شبكة بثلاثة أعمدة متساوية العرض، وهذا ما يعنيه `1fr 1fr 1fr`، ونضيف فجوة 20 بكسل بين كل عنصر شبكي.

قد تتساءل الآن: "ماذا عن Flexbox؟ أليس يُستخدم أيضًا للتخطيطات؟"

أنت محق! Flexbox هو نموذج تخطيط CSS آخر، وهو مفيد جدًا أيضًا. لكن هناك بعض الاختلافات الرئيسية.

Flexbox أحادي البعد، بينما Grid ثنائي البعد. هذا يعني أن Flexbox يعمل بشكل رائع لترتيب الأشياء في صف أو عمود واحد، بينما يتفوق Grid في إنشاء تخطيطات تحتوي على صفوف وأعمدة معًا.

Flexbox يعتمد على المحتوى أولًا، بمعنى أنه يضبط التخطيط بناءً على المحتوى. أما Grid، فيعتمد على التخطيط أولًا، مما يسمح لك بإنشاء التخطيط ثم وضع العناصر فيه. Grid يمنحك تحكمًا أدق في التموقع. يمكنك تحديد الصف والعمود الذي يشغله العنصر بدقة.

إليك مثال Flexbox للمقارنة:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
}
```

:::

هذا ينشئ حاوية مرنة حيث توزع العناصر بالتساوي على طول المحور الرئيسي.

لكل من Grid وFlexbox نقاط قوة، وغالبًا ما تستخدم أفضل التخطيطات مزيجًا منهما. قد تستخدم Grid لتخطيط الصفحة بالكامل، ثم تستخدم Flexbox لمحاذاة العناصر داخل كل مساحة شبكية.

باختصار، CSS Grid أداة قوية تسمح بتخطيطات ثنائية الأبعاد دقيقة. قد تبدو معقدة في البداية، لكن مع الممارسة تصبح أداة لا غنى عنها لإنشاء تخطيطات ويب متجاوبة ومعقدة.
