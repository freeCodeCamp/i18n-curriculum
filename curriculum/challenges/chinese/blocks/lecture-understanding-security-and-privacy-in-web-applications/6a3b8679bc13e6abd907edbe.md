# --description--

**安全性保护数据免受未经授权的访问、盗窃和恶意破坏。** 它在你的在线信息周围构建了一道坚固的数字防线。

把数据安全想象成你家门上的一把沉重的锁。这把锁阻止入侵者进入你的房子。在数字世界中，安全性专注于保护数据的机密性、数据完整性和系统可用性。

网络安全工具包括网络防火墙、强密码、数据加密和杀毒软件。如果外部黑客试图窃取你的文件，安全系统会阻止这种尝试。

安全性不分析数据的含义。它只关注保护数据免受恶意行为者的侵害。没有安全性，你在线存储的任何数据都容易被立即盗取。

**隐私控制组织如何收集、保存、使用和共享你的个人数据。** 它赋予你决定谁可以看到你的私人信息的权力。

把数据隐私看作窗帘。即使你的窗户有坚固的锁，如果你把窗帘完全拉开，人们仍然可以看到里面。

隐私确保公司尊重你的个人权利。它决定公司是否可以出售你的浏览历史、追踪你的 GPS 位置，或将你的电子邮件地址分享给第三方广告商。

像欧盟（EU）的 GDPR 这样的全球法律严格规范数据隐私。你将在后续课程中学习这些法律。

现在让我们看看安全性和隐私之间的关系。虽然它们是不同的概念，但彼此密切依赖。

你可以轻松拥有安全性而没有隐私。例如，智能手机制造商可能将你的每日位置数据存储在高度安全的云数据库中。高级黑客无法攻破该服务器，因此数据完全安全，不受恶意行为者的侵害。

然而，如果制造商未经你明确许可将这些数据出售给广告机构，那么你的隐私就被侵犯了。安全性完美运行，但你的数据隐私为零。

另一方面，没有强大的安全性，你永远无法实现真正的隐私。如果一家医疗诊所承诺永不分享你的健康记录，那他们承诺的是隐私。但如果诊所未能保护其本地服务器，恶意黑客可以轻易窃取这些医疗记录。

这种完全缺乏安全性会立即破坏你的数据隐私。因此，强大的安全性是有效隐私的基础。

安全性提供技术防御机制，而隐私管理有关数据使用的法律权利和政策。理解这一差异有助于你安全地导航数字空间。

安全团队构建针对主动网络攻击的技术防御，而隐私团队确保遵守隐私法律并尊重用户选择。

这两个要素对于保护当今在线个人仍然至关重要。安全锁住前门，隐私拉上窗帘。两者共同为你的个人信息创造一个安全、可信赖的环境。
