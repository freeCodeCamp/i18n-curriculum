# --description--

Esta tarefa foca no pronome relativo `that`.É usado para introduzir uma oração que fornece mais detalhes sobre um substantivo.

Uma oração é uma parte de uma frase que contém pelo menos um sujeito e um verbo.Pode expressar uma ideia completa ou adicionar informações extras à frase principal.

O pronome `that` ajuda a especificar ou definir algo com mais clareza, conectando-o a uma descrição ou a um detalhe necessário.

Por exemplo, `She read the book that won the Pulitzer Prize last year.` Nesta frase, `that won the Pulitzer Prize last year` é uma oração relativa introduzida por `that`.A cláusula descreve ou especifica o livro, ajudando a distingui-lo de outros livros.Aqui, `that` atua como um pronome relativo conectando a oração ao substantivo `book` e descreve qual livro em particular ela leu.
