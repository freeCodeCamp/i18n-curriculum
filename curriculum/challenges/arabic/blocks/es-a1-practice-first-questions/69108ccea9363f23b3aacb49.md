# --explanation--

يركز هذا الحوار بين ماتيو وكاميلا على المهن.

`¿A qué te dedicas?` تعني "ما مهنتك؟" أو "ما الذي تعمل به؟"

**`Soy` + [profession]** تعني "أنا [المهنة]".

في اللغة الإسبانية، تتغير معظم أسماء المهن حسب الجنس. لكن بعض المهن التي تنتهي بـ `-ista` أو `-nte` غالبًا ما تبقى كما هي لكلا الجنسين:

| المهنة | مذكر | مؤنث |
| :---: | :---: | :---: |
| محلل | `analista` | `analista` |
| صحفي | `periodista` | `periodista` |
| موظف استقبال | `recepcionista` | `recepcionista` |
| مغنٍ | `cantante` | `cantante` |
