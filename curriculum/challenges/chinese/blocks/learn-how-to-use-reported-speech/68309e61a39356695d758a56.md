# --explanation--

`To notice` 意味着通过看见、听见或感知某物而意识到它——尤其是当某物发生变化或突出时。例如：

`Did you notice the new button on the homepage?` – 这相当于问某人是否看到了变化。

Tom 的问题，`Were the changes noticed?`，是**被动语态**。它关注的是更改（对象），而不是注意到更改的人（主语）。另一个**被动语态**问题的例子：

`Was the mistake corrected before the launch?` – 这关注的是纠正本身，而不是谁纠正的。
