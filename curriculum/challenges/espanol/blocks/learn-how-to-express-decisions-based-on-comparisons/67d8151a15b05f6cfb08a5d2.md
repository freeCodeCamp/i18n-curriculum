# --explanation--  

`That way` se usa para explicar el resultado o beneficio de una decisión o acción. Por ejemplo:

Debemos automatizar las pruebas. De esa manera, podemos detectar errores más rápido. – `That way` indica que la automatización ayudará a encontrar errores más rápidamente.  

`While` se usa para indicar que dos cosas están ocurriendo al mismo tiempo. Por ejemplo:  

`The team is fixing bugs while the designers work on the UI.` – Esto significa que ambas tareas están ocurriendo al mismo tiempo.
