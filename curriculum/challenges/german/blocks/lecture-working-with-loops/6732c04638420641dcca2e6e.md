# --interactive--

Eine `for...of`-Schleife wird verwendet, wenn Sie über Werte aus einem iterierbaren Objekt iterieren müssen. Beispiele für Iterables sind Arrays und Strings.

Hier ist die grundlegende Syntax für eine `for...of`-Schleife:

```js
for (variable of iterable) {
  // code block to be executed
}
```

Die Variable im Beispiel stellt den aktuellen Wert des iterierbaren Objekts dar, über das gerade iteriert wird.

Wenn Sie ein Array von Zahlen haben, wäre die Variable die aktuelle Zahl im Array. Wenn Sie einen String haben, wäre die Variable das aktuelle Zeichen im String.

Schauen wir uns einige Beispiele an, damit Sie besser verstehen, wie die `for...of`-Schleife funktioniert.

Im ersten Beispiel haben wir ein Array von Zahlen und wollen über jede Zahl iterieren und sie in der Konsole ausgeben.

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

:::

Wir haben eine Variable namens `num` erstellt, die die aktuelle Zahl im Array darstellt. Für die erste Iteration wird `num` `1` sein, für die zweite Iteration wird `num` `2` sein, und so weiter.

Innerhalb der Schleife geben wir die aktuelle Zahl in der Konsole aus.

Hier ist ein weiteres Beispiel, bei dem wir einen String haben und über jedes Zeichen iterieren und es in der Konsole ausgeben wollen.

:::interactive_editor

```js
const str = 'freeCodeCamp';

for (let char of str) {
  console.log(char);
}
```

:::

In diesem Beispiel haben wir eine Variable namens `char` erstellt, die das aktuelle Zeichen im String darstellt.

Für jede Iteration gibt die Schleife das aktuelle Zeichen in der Konsole aus.

Es ist wichtig zu beachten, dass Sie `let` oder `const` verwenden können, wenn Sie die Variable in einer `for...of`-Schleife deklarieren.

Wenn Sie jedoch `const` verwenden, stellen Sie sicher, dass sich der Wert der Variable innerhalb der Schleife nicht ändert. Wenn doch, erhalten Sie einen Fehler.

Hier ist ein Beispiel für die Verwendung von `const`, das zu einem Fehler führt:

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
  num = num + 1; // This will cause an error
}
```

In diesem Beispiel versuchen wir, den Wert von `num` innerhalb der Schleife zu ändern. Da wir `num` mit `const` deklariert haben, erhalten wir einen Fehler. Wenn Sie also den Wert der Variable innerhalb der Schleife ändern müssen, verwenden Sie stattdessen `let`.

Schauen wir uns zum Schluss noch ein Beispiel mit einem Array von Objekten an.

:::interactive_editor

```js
const people = [
  { name: 'John', age: 30 },
  { name: 'Jane', age: 25 },
  { name: 'Jim', age: 40 }
];

for (const person of people) {
  console.log(`${person.name} is ${person.age} years old`);
}
```

:::

In diesem Beispiel haben wir ein Array von Objekten namens `people`. Jedes Objekt hat eine `name`- und eine `age`-Eigenschaft.

Wenn wir durch das Array iterieren, erstellen wir eine Variable namens `person`, die das aktuelle Objekt im Array darstellt.

Innerhalb der Schleife geben wir eine Nachricht in der Konsole aus.

Die erste Nachricht wird `John is 30 years old` sein, die zweite Nachricht `Jane is 25 years old` und die dritte Nachricht `Jim is 40 years old`.

`for...of`-Schleifen sind wirklich nützlich, wenn Sie über Werte aus einem iterierbaren Objekt wie einem Array oder einem String iterieren müssen. Sie sind außerdem leicht lesbar und können Ihren Code kompakter machen.
