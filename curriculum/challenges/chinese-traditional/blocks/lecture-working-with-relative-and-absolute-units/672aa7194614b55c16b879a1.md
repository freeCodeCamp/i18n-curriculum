# --interactive--

當你設計你的頁面時，你會使用不同的屬性，例如寬度、高度、內距、外距等等。當你定義這些屬性時，你需要指定你想使用的長度單位。

你可以使用兩種型別的單位來定義這些屬性：相對單位和絕對單位。在本課程中，我們將只專注於絕對單位。

絕對長度單位具有固定長度，且不相對於其他任何事物。相對長度是指長度相對於其他事物，例如螢幕大小或父元素的大小。我們會在接下來的幾課中詳細討論相對單位。

最常見的絕對單位是像素（`px`）。像素是在 CSS 中的固定大小測量單位，提供對尺寸的精確控制。這表示 1px 永遠相當於 1/96 英吋。

重要的是要注意，雖然 1px 在 CSS 佈局中標準化為 1/96 英吋，但像素的實際物理大小可能會因顯示器而異。

讓我們來看一個使用像素設定方塊寬度和高度的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="box"></div>
```

```css
.box {
  width: 100px;
  height: 100px;
  background-color: red;
}
```

:::

在此範例中，我們有一個寬度和高度設為 `100px` 的紅色方塊。無論螢幕尺寸如何變化，此方塊的尺寸都會保持不變。

所以，如果你使用的是小螢幕的裝置，方塊仍然會是 `100px` 寬。如果你使用的是大型桌面螢幕，方塊仍然會是 `100px` 寬。

那麼，什麼時候你應該使用像素這類絕對單位呢？這取決於情況。但通常你會在需要精確控制元素尺寸、間距和版面配置時使用像素。有時你可能會為邊距、內距和邊框使用像素。

這裡是一個使用像素來設定兩個方塊邊距的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="box"></div>
<div class="box"></div>
```

```css
.box {
  width: 100px;
  height: 100px;
  background-color: red;
  margin: 10px;
}
```

:::

請記住，margin 是盒子外部的空間。因此，在此範例中，盒子四邊的 margin 都是 `10px`。

其他型別的絕對單位包括以下內容：

- `in`（英吋）單位，相當於 96px
- `cm`（公分）單位，相當於 25.2/64 英吋
- `mm`（毫米）單位，相等於 1/10 公分
- `q`（四分之一毫米）單位，相當於 1/40 公分
- `pc`（派卡）單位，相當於 1/6 英吋
- `pt`（點）單位，相等於 1/72 英吋

這些單位大多數將用於列印，而非螢幕。 

雖然你最常使用的單位是像素，但了解其他絕對單位的存在也很重要。
