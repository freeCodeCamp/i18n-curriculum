# --description--

配列は、さまざまな要素を多く保持できるJavaScriptのオブジェクトです。

```js
var complexArr = [1, 5, "2", "Word", {"name": "James"}];
```

基本的に、ブラウザはその配列に適切なメモリ容量を自動的に割り当てます。データを追加したり削除したりすると、その容量も必要に応じて変わります。

しかし、高速処理や異なる要素タイプの世界では、配列に割り当てるメモリ容量をより具体的に指定する必要がある場合があります。

<dfn>型付き配列</dfn>はこの問題の解決策です。配列に割り当てたいメモリ容量を指定できるようになりました。以下は利用可能な配列の種類と、それぞれの要素のバイト単位のサイズの基本的な概要です。

<table><tbody><tr><th>種類</th><th>各要素のサイズ（バイト）</th></tr><tr><td><code>Int8Array</code></td><td>1</td></tr><tr><td><code>Uint8Array</code></td><td>1</td></tr><tr><td><code>Uint8ClampedArray</code></td><td>1</td></tr><tr><td><code>Int16Array</code></td><td>2</td></tr><tr><td><code>Uint16Array</code></td><td>2</td></tr><tr><td><code>Int32Array</code></td><td>4</td></tr><tr><td><code>Uint32Array</code></td><td>4</td></tr><tr><td><code>Float32Array</code></td><td>4</td></tr><tr><td><code>Float64Array</code></td><td>8</td></tr></tbody></table>

これらの配列を作成する方法は2つあります。1つは直接作成する方法です。以下は長さ3の`Int16Array`を作成する方法です。

```js
var i8 = new Int16Array(3);
console.log(i8);
// Returns [0, 0, 0]
```

また、<dfn>バッファ</dfn>を作成して、配列が占めるデータ量（バイト単位）を指定することもできます。**注意**  
バッファを使って型付き配列を作成する場合、割り当てるバイト数は上記のバイト数の倍数である必要があります。

```js
// Create same Int16Array array differently
var byteSize = 6; // Needs to be multiple of 2
var buffer = new ArrayBuffer(byteSize);
var i8View = new Int16Array(buffer);
buffer.byteLength; // Returns 6
i8View.byteLength; // Returns 6
console.log(i8View); // Returns [0, 0, 0]
```

<dfn>バッファ</dfn>はデータを保持する汎用オブジェクトです。通常は直接アクセスできません。アクセスするには、まず<dfn>ビュー</dfn>を作成する必要があります。

```js
i8View[0] = 42;
console.log(i8View); // Returns [42, 0, 0]
```

**注意**  
型付き配列は、従来の配列が持つ`.pop()`や`.push()`のようなメソッドを持っていません。また、配列かどうかを判定する`Array.isArray()`も失敗します。シンプルですが、これはあまり高度でないJavaScriptエンジンが型付き配列を実装しやすいという利点にもなります。

# --instructions--

まず64バイトの`buffer`を作成してください。次に、それに対するビューとして`Int32Array`という名前の`i32View`型付き配列を作成してください。

# --hints--

`buffer`は64バイトの大きさである必要があります。

```js
assert(buffer.byteLength === 64);
```

バッファに対する`i32View`ビューは64バイトの大きさである必要があります。

```js
assert(i32View.byteLength === 64);
```

バッファに対する`i32View`ビューは16要素の長さである必要があります。

```js
assert(i32View.length === 16);
```
