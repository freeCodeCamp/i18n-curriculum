# --description--

式 `That/it sounds` は、聞いた情報や与えられた印象に基づいて、何かがどのように見えるかや思われるかを表すために使われます。例えば、`That sounds like a great idea!` は、聞いた内容に基づいてその考えが良さそうだということを示しています。

同様に、`That/it looks` は、`It looks beautiful.` のように見た目に基づいて何かがどのように見えるかを表すために使われます。

単語 `convenient` は、ニーズに合っている、使いやすい、または物事をより簡単かつ迅速に行うのに役立つものを表す形容詞です。これは、作業を楽にしたり時間を節約したりするものを指すことが多いです。

例えば、`Online shopping is convenient because you can do it from home.`
