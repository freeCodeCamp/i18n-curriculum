# --description--

Нехай $R(a, b, c)$ — максимальна площа, яку займають три неперекривні кола всередині трикутника з довжинами сторін $a$, $b$ та $c$.

Нехай $S(n)$ — середнє значення $R(a, b, c)$ для всіх цілих трійок $(a, b, c)$, таких що $1 ≤ a ≤ b ≤ c &lt; a + b ≤ n$.

Вам задано $S(2) = R(1, 1, 1) ≈ 0.31998$, $S(5) ≈ 1.25899$.

Знайдіть $S(1803)$, округлене до 5 знаків після десяткової крапки.
