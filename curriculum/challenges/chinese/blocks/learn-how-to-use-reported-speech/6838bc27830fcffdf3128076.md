# --explanation--

`Exactly` 是一个副词，用于表示完全同意或正确地符合别人所说的内容。另一方面，`Exact` 是一个形容词，表示精确或细节上的正确。例如：

- `You are exactly right about the cause of the problem.` – 这表示完全同意。它表达了这个人是多么正确（形容词）。

- `We need the exact measurements to build the new server rack.` – 这意味着非常精确的测量。它修饰名词，而不是动词或形容词。
