## 컴퓨터, 인터넷 및 개발자 도구 기초 이해하기

- **마더보드**: 컴퓨터를 작동하는 데 필요한 모든 메모리, 커넥터, 하드 드라이브를 담고 있습니다. 컴퓨터의 주요 회로 기판 역할을 합니다.
- **중앙 처리 장치(CPU)**: 명령을 실행하고 계산을 수행하는 프로세서입니다.
- **랜덤 액세스 메모리(RAM)**: 컴퓨터의 CPU를 위한 임시 저장 공간입니다.
- **하드 디스크 드라이브(HDD)**: 컴퓨터가 꺼져 있어도 데이터를 저장하는 영구 저장 공간입니다.
- **솔리드 스테이트 드라이브(SSD)**: 비휘발성 플래시 메모리로 하드 드라이브 대신 사용할 수 있습니다.
- **전원 공급 장치(PSU)**: 벽면 콘센트의 전기를 컴퓨터가 사용할 수 있는 형태로 변환하는 역할을 합니다.
- **그래픽 처리 장치(GPU)**: 컴퓨터 화면에 시각적 요소를 렌더링하는 역할을 합니다.
- **인터넷 서비스 제공자(ISP)의 다양한 유형**: ISP는 인터넷 접근을 제공하는 회사입니다. 다이얼업, DSL, 케이블, 광섬유, 위성 등 다양한 유형이 있습니다.
- **컴퓨터에 안전하게 로그인하는 방법**: 강력한 비밀번호 사용, 이중 인증 활성화, 비밀번호 관리자 사용 등이 안전한 로그인 방법의 예입니다.
- **통합 개발 환경(IDE)**: 개발자가 코드를 효율적으로 작성, 테스트, 디버그할 수 있도록 돕는 도구입니다.
- **코드 에디터**: 개발자가 코드를 작성하고 디버그하는 데 사용하는 도구입니다.
- **Git**: 개발자가 코드 변경 사항을 추적하고 협업할 수 있게 하는 인기 있는 버전 관리 시스템입니다.
- **저장소를 위한 클라우드 기반 호스팅 서비스**: 저장소는 프로젝트 파일과 버전 기록을 저장하는 장소입니다. GitHub, GitLab, Bitbucket 등이 인기 있는 클라우드 기반 호스팅 서비스입니다.
- **패키지 관리자**: 개발자가 라이브러리와 프로젝트 의존성을 추가, 업데이트, 삭제하는 과정을 간소화하는 도구입니다. npm, pip, Maven 등이 예입니다.
- **테스트 라이브러리와 프레임워크**: 소프트웨어가 예상대로 작동하는지 확인하기 위해 테스트를 수행합니다. Jest, PHPUnit, JUnit 등이 테스트 라이브러리와 프레임워크의 예입니다.

## 파일, 파일 시스템 및 미디어 포맷 다루기

- **파일 이름 지정 모범 사례**: 파일 이름은 이해하기 쉽고 관리하기 편하게 지정하세요. 예를 들어, `about-us.html`가 `page1.html`보다 더 설명적인 이름입니다.
- **루트 디렉터리**: 파일 시스템에서 최상위 디렉터리입니다. 디렉터리는 폴더의 다른 이름입니다.
- **마크다운**: 문서화와 `README` 파일에 흔히 사용되는 마크업 언어입니다. `README` 파일은 설치 및 사용 방법 같은 프로젝트 정보를 담은 파일입니다.
- **`index.html`**: 사용자가 웹사이트를 방문할 때 기본으로 표시되는 페이지를 나타냅니다.
- **Explorer/Finder를 사용해 파일과 폴더 생성, 이동, 삭제하기**: Explorer는 Windows의 파일 관리자이고, Finder는 macOS의 파일 관리자입니다. 이 도구들을 사용해 파일과 폴더를 생성, 이동, 삭제할 수 있습니다.
- **파일과 폴더 검색하기**: Explorer나 Finder의 검색 기능을 사용해 컴퓨터 내 파일과 폴더를 찾을 수 있습니다.
- **HTML, CSS, JS 파일 유형**: `.html` 확장자는 HTML 파일에, `.css`는 CSS 파일에, `.js`는 JavaScript 파일에 사용됩니다.
- **일반적인 이미지 및 그래픽 포맷**: `JPEG`와 `PNG`는 흔한 이미지 파일 포맷입니다. `GIF`는 애니메이션을 지원하는 또 다른 이미지 파일 포맷입니다. `SVG`는 벡터 그래픽용 파일 포맷입니다.
- **일반적인 오디오 및 비디오 포맷**: `MP3` 포맷은 오디오 파일에 흔히 사용됩니다. `MP4` 포맷은 비디오 파일에 흔히 사용됩니다. `MOV` 포맷은 Apple에서 개발했으며 비디오 파일에 흔히 사용됩니다.
- **일반적인 폰트 포맷**: `TTF` 포맷은 TrueType 폰트에 흔히 사용됩니다. `WOFF` 포맷은 웹 폰트에 흔히 사용됩니다. WOFF의 후속 포맷인 `WOFF2`는 더 나은 압축을 제공합니다.
- **ZIP**: 파일과 폴더를 압축하는 데 사용되는 파일 포맷입니다.

## 웹 효과적으로 탐색하기

- **웹 브라우저란?**: 사용자가 인터넷에서 웹사이트에 접근하고 볼 수 있게 하는 소프트웨어 애플리케이션입니다.
- **검색 엔진이란?**: 사용자가 인터넷에서 정보를 검색할 수 있게 하는 도구입니다. Google, Bing, Yahoo 등이 예입니다.
- **일반적인 웹 브라우저**: Google Chrome, Mozilla Firefox, Microsoft Edge 등이 일반적인 웹 브라우저의 예입니다.
- **일반적인 검색 전략**: `site:` 다음에 웹사이트 URL을 입력해 해당 사이트 내 콘텐츠를 검색할 수 있습니다. `filetype:` 다음에 파일 확장자를 입력해 해당 유형의 파일을 검색할 수 있습니다. 검색어 앞에 마이너스 기호를 붙여 해당 단어가 포함된 결과를 제외할 수 있습니다. 검색어 앞에 플러스 기호를 붙여 해당 단어가 포함된 결과만 포함할 수 있습니다.
