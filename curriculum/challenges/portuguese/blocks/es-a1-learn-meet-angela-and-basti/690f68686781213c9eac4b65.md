# --description--

`¿Cuántos años tienes?` é a forma mais comum de perguntar a idade de alguém em espanhol. 

É equivalente a "How old are you?"

Esta frase tem uma conjugação do verbo `tener` ("to have") porque o espanhol expressa idade usando posse.
