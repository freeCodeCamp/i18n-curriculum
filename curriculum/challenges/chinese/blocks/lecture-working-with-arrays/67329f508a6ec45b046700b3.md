# --interactive--

JavaScript 中的数组是数值的有序集合，每个数值由一个数字索引标识。 JavaScript 数组中的值可以是不同的数据类型，包括数字、字符串、布尔值、对象甚至其他数组。

要在 JavaScript 中创建数组，可以使用方括号 `[]` 并用逗号分隔数值。 这是一个示例：

```js
let fruits = ["apple", "banana", "orange"];
```

在这个例子中，我们声明了一个变量 `fruits`，并给它赋值为一个包含三个字符串值的数组：`apple`、`banana` 和 `orange`。

数组的一个关键特性是它们是 从零开始索引 的，这意味着数组中的第一个元素索引是 `0`，第二个元素索引是 `1`，依此类推。 你可以使用索引访问数组中的单个元素。 例如：

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits[0]); // "apple"
console.log(fruits[2]); // "orange"
```

:::

在这个例子中，我们使用索引 `0` 来访问第一个元素（`apple`），使用索引 `2` 来访问第三个元素（`orange`）。

JavaScript 中的数组有一个特殊的 `length` 属性，用于返回数组中元素的个数。 你可以使用 `length` 关键字访问该属性。 例如：

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits.length); // 3
```

:::

JavaScript 中数组的另一个关键特性是动态性，这意味着数组的大小可以在创建后发生变化。 你可以使用各种数组方法，如 `push()`、`pop()`、`shift()`、`unshift()`、`splice()` 等，来添加或删除数组中的元素。  These methods will be taught in upcoming lessons.

JavaScript 数组在程序中存储数据时用途广泛，非常有用。 在本模块中，你将实际看到，使用数组如何帮助你高效地管理和操作数据集合。
