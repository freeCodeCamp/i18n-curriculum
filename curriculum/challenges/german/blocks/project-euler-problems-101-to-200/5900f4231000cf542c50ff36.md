# --description--

Sei $N$ eine positive ganze Zahl, und $N$ wird in $k$ gleiche Teile aufgeteilt, $r = \frac{N}{k}$, so dass $N = r + r + \cdots + r$ gilt.

Sei $P$ das Produkt dieser Teile, $P = r × r × \cdots × r = r^k$.

Zum Beispiel, wenn 11 in fünf gleiche Teile aufgeteilt wird, 11 = 2,2 + 2,2 + 2,2 + 2,2 + 2,2, dann ist $P = {2,2}^5 = 51,53632$.

Sei $M(N) = P_{max}$ für einen gegebenen Wert von $N$.

Es stellt sich heraus, dass das Maximum für $N = 11$ gefunden wird, indem elf in vier gleiche Teile aufgeteilt wird, was zu $P_{max} = {(\frac{11}{4})}^4$ führt; das heißt, $M(11) = \frac{14641}{256} = 57,19140625$, was eine endliche Dezimalzahl ist.

Für $N = 8$ wird das Maximum jedoch erreicht, indem es in drei gleiche Teile aufgeteilt wird, also ist $M(8) = \frac{512}{27}$, was eine unendliche Dezimalzahl ist.

Sei $D(N) = N$, wenn $M(N)$ eine unendliche Dezimalzahl ist, und $D(N) = -N$, wenn $M(N)$ eine endliche Dezimalzahl ist.

Zum Beispiel ist $\sum D(N)$ für $5 ≤ N ≤ 100$ gleich 2438.

Finden Sie $\sum D(N)$ für $5 ≤ N ≤ 10000$.
