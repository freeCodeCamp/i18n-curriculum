# --explanation--

`To seem`는 우리가 알고 있는 바에 근거해 어떤 것이 사실인 것 같지만 완전히 확신하지는 못할 때 사용합니다. 예를 들어:

`It seems the users liked the new menu.` – 이는 사용자가 피드백이나 행동을 바탕으로 좋아했을 것이라고 생각하지만 확실하지는 않다는 뜻입니다.

`To be found to be`는 누군가가 무언가를 발견하거나 결론 내렸을 때 사용하는 **수동태** 구조로, 판단한 사람보다는 결과에 초점이 맞춰져 있습니다. 예를 들어:

`The layout was found to be confusing by some testers.` – 이는 시험자가 혼란스럽다고 생각했지만 문장은 그 발견에 초점을 맞추고 사람들(이 경우 시험자)에게는 초점을 두지 않는다는 뜻입니다.
