# --interactive--

يُستخدم عنصر الشفرة المضمَّنة لتمثيل مقاطع شفرة قصيرة داخل النص. من الاستخدامات الشائعة لعنصر الشفرة في المقالات التقنية وصفحات التوثيق.

إليك مثالًا على استخدام عنصر `code` لعرض مقتطف برمجي من CSS:

:::interactive_editor

```html
<p>
  To set the text color to blue in CSS, use the following code:
  <code>color: blue;</code>
</p>
```

:::

في هذا المثال، تُستخدم خاصية CSS `color` لتعيين لون النص إلى `blue`. بلف هذا المقتطف البرمجي داخل وسم `<code>`، يُخبر المتصفح أن النص هو قطعة شفرة مضمَّنة.

يطبق المتصفح أنماطًا افتراضية على المحتوى داخل عنصر `code`. النمط الافتراضي هو خط أحادي المسافة.

يُقصد من عنصر `code` تمثيل سطر واحد من الشفرة. إذا أردت تمثيل عدة أسطر، يجب وضع عنصر `code` داخل عنصر نص منسق مسبقًا.

يُستخدم عنصر النص المنسق مسبقًا لتمثيل نص منسق مسبقًا. إليك مثالًا على استخدام هذا العنصر لعرض إعلان CSS:

:::interactive_editor

```html
<pre>
  <code>
    body {
      color: red;
    }
  </code>
</pre>
```

:::

عند استخدام عنصر `pre`، يجب الانتباه إلى المسافات لأنه سيعرض النص تمامًا كما هو مكتوب في مستند HTML.

في المتصفح، سترى أن الشفرة متداخلة بعدة مسافات إلى اليمين. إذا غيرت التداخل في المثال البرمجي، سترى فرقًا في التداخل على الشاشة.

عندما يتعلق الأمر بإدراج أمثلة شفرة داخل مستند HTML، يجب استخدام عنصر `code` للأمثلة المضمَّنة القصيرة.

إذا كنت بحاجة إلى عرض مقتطفات شفرة أطول، فستحتاج إلى استخدام عنصري `pre` و `code` معًا.
