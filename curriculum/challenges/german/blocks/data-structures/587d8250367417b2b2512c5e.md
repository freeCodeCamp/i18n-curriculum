# --description--

Sie kennen wahrscheinlich einen Stapel Bücher auf Ihrem Tisch. Wahrscheinlich haben Sie schon die Rückgängig-Funktion eines Texteditors verwendet. Sie sind auch vermutlich daran gewöhnt, auf Ihrem Handy die Zurück-Taste zu drücken, um zur vorherigen Ansicht in Ihrer App zurückzukehren.

Wissen Sie, was sie alle gemeinsam haben? Sie speichern die Daten so, dass Sie rückwärts durch sie hindurchgehen können.

Das oberste Buch im Stapel ist das, das zuletzt daraufgelegt wurde. Wenn Sie dieses Buch vom oberen Ende Ihres Stapels entfernen, wird das Buch sichtbar, das vor dem letzten Buch daraufgelegt wurde, und so weiter.

Wenn Sie darüber nachdenken, erhalten Sie in all den oben genannten Beispielen einen Dienst vom Typ <dfn>Last-In-First-Out</dfn>. Wir werden versuchen, dies mit unserem Code nachzubilden.

Dieses Daten-Speicherschema heißt <dfn>Stack</dfn>. Insbesondere müssten wir die `push()`-Methode implementieren, die JavaScript-Objekte oben auf den Stack legt; und die `pop()`-Methode, die das JavaScript-Objekt entfernt, das sich gerade oben auf dem Stack befindet.
