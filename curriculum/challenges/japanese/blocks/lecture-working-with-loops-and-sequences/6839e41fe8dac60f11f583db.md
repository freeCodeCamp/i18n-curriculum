# --description--

`range()` 関数は整数のシーケンスを生成するために使われます。`range()` 関数の基本的な構文は以下の通りです。

```py
range(start, stop, step)
```

必須の`stop` 引数は、生成される数列の終点を表す整数です。`range()` 関数を使った例を示します。

```py
for num in range(3):
    print(num)
```

次のコードは`0` から `2` までの数列を生成します。`3` は含まれません。なぜなら`stop` 引数が非包含だからです。

`start` 引数が指定されていない場合、デフォルトは`0` です。そうでなければ、オプションの`start` 引数を使って、整数のシーケンスを`0` 以外の整数から開始できます。`1` から `4` までの整数のシーケンスを生成する例を示します。

```py
for num in range(1, 5):
    print(num)
```

デフォルトでは整数のシーケンスは`1` ずつ増加します。しかし、そのデフォルトを変更したい場合は、オプションの`step` 引数を使えます。`2` から `10` までの偶数の整数のシーケンスを生成する例を示します。

```py
for num in range(2, 11, 2):
    print(num)
```

前述の通り、`range()` 関数には必須の引数が1つだけあります。`range()` に引数を何も渡さなければ、`TypeError` が得られます。

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: range expected at least 1 argument, got 0
```

重要なのは、`range()` 関数は引数として整数のみを受け付け、浮動小数点数は受け付けないことです。浮動小数点数とは`3.4` のような小数点を含む数のことです。もし浮動小数点数を引数として渡そうとすると、`TypeError` が返されます。

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

整数のシーケンスを減少順に生成したい場合は、`step` 引数に負の整数を使えます。例は以下の通りです。

```py
for num in range(40, 0, -10):
    print(num)
```

次のコードは`40`、`30`、`20`、`10` の順にコンソールに出力します。

`range()` 関数でできるもう一つのことは、`list` コンストラクタと組み合わせて整数のリストを作成することです。`list` コンストラクタはイテラブルをリストに変換するために使われます。`2` から `10` までの偶数の整数のリストを生成する例を示します。

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

`range()` 関数はPythonで整数のシーケンスを生成する非常に便利な方法です。慣れてくると、Pythonプログラムで頻繁に使うようになるでしょう。
