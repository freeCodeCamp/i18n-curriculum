# --description--

Para crear una lista desordenada de elementos, puedes usar el elemento `ul`.

Después del elemento `h3` con el texto `Things cats love:`, añade un elemento de lista desordenada (`ul`). Ten en cuenta que no se mostrará nada en este punto.

# --hints--

Tu elemento `ul` debe tener una etiqueta de apertura. Las etiquetas de apertura tienen esta sintaxis: `<elementName>`.

```js
assert.exists(document.querySelector('ul'));
```

Tu elemento `ul` debe tener una etiqueta de cierre. Las etiquetas de cierre tienen un `/` justo después del carácter `<`.

```js
assert.match(code, /<\/ul>/);
```

El elemento `ul` debe estar por encima de la etiqueta de cierre del segundo elemento `section`.

```js
const secondSectionLastElemNode =
  document.querySelectorAll('main > section')[1]?.lastElementChild;
assert.equal(secondSectionLastElemNode?.nodeName, 'UL');
```
