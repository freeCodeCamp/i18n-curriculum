# --description--

設 $n$ 為正整數。考慮以下的 nim 位置：

- 有 $n$ 個非空堆疊。
- 每個堆疊大小都小於 $2^n$。
- 沒有兩個堆疊的大小相同。

令 $W(n)$ 為符合上述條件的獲勝 nim 位置數量（若第一位玩家有獲勝策略，該位置即為獲勝位置）。

例如，$W(1) = 1$、$W(2) = 6$、$W(3) = 168$、$W(5) = 19\\,764\\,360$ 以及 $W(100)\bmod 1\\,000\\,000\\,007 = 384\\,777\\,056$。

尋找 $W(10\,000\,000)\bmod 1\,000\,000\,007$。
