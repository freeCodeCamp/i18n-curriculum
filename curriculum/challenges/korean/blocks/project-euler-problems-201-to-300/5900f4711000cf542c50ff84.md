# --description--

양의 정수 $k$를 제곱 중심(square-pivot)이라고 부르겠습니다. 만약 $m > 0$이고 $n ≥ k$인 정수 쌍 $(m, n)$이 존재하여, $k$까지의 연속된 $(m + 1)$개의 제곱수의 합이 $(n + 1)$부터 시작하는 연속된 $m$개의 제곱수의 합과 같다면,

$${(k - m)}^2 + \ldots + k^2 = {(n + 1)}^2 + \ldots + {(n + m)}^2$$

몇 가지 작은 제곱 중심은 다음과 같습니다.

$$\begin{align}
  & \mathbf{4}: 3^2 + \mathbf{4}^2 = 5^2 \\\\
  & \mathbf{21}: {20}^2 + \mathbf{21}^2 = {29}^2 \\\\
  & \mathbf{24}: {21}^2 + {22}^2 + {23}^2 + \mathbf{24}^2 = {25}^2 + {26}^2 + {27}^2 \\\\
  & \mathbf{110}: {108}^2 + {109}^2 + \mathbf{110}^2 = {133}^2 + {134}^2 \\\\
\end{align}$$

${10}^{10}$ 이하의 모든 서로 다른 제곱 중심의 합을 구하세요.
