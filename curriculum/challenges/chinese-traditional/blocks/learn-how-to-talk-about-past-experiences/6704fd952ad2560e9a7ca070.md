# --explanation--

表達式 `stuck on` 用於描述某人無法繼續執行任務或解決問題。它表示被困住或無法前進。 例如：

- `We were stuck on this code error for hours.` - 我們無法繼續，直到找到程式碼中的那個錯誤。

- `She got stuck on the final question of the exam.`－她花了很長時間在最後一題。

在對話窗中，Lisa 使用 `stuck on` 來描述他們無法在工作中解決錯誤的進展情況。
