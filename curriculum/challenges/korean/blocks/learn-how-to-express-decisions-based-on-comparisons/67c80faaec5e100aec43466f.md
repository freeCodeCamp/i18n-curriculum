# --explanation--

`More robust`는 다른 것과 비교했을 때 더 강력하거나 더 신뢰할 수 있다는 뜻입니다. 예를 들어:

`This software is more robust than the previous version because it can handle larger data sets.` - 여기서 `more robust`는 소프트웨어가 더 강력하고 능력이 뛰어나다는 의미입니다.

`When it comes to`는 특정 주제를 소개할 때 쓰이며, "관하여" 또는 "에 관해"라는 뜻입니다. 예를 들어:

`When it comes to customer support, this company is very responsive.` - 여기서 `when it comes to`는 고객 지원이라는 주제를 소개합니다.
