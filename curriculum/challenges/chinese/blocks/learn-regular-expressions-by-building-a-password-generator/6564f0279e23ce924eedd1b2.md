# --description--

由于下划线字符是变量名的有效字符，因此它被包含在 `\w` 字符类中（相当于 `[a-zA-Z0-9_]`），可以方便地用于匹配变量名。

因此，`\W` 字符类等同于带有不匹配下划线字符的 `[^a-zA-Z0-9_]`。 所以，你不能使用它来匹配所有特殊字符。
