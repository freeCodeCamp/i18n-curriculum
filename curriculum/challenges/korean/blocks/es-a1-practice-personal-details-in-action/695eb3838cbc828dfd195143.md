# --explanation--

질문 `¿Cuántos años tienes?`는 누군가의 나이를 묻고 있습니다.

답을 찾으려면 양식의 `Edad` 필드를 확인해야 합니다.

양식에는 `Edad`: 25가 표시되어 있습니다.

따라서 올바른 답은 **25**입니다.

이 질문은 스페인어에서 일반적으로 다음 구조로 답합니다:

`Tengo` + 숫자 + `años`

글자 **`ñ`**는 단순히 `n`에 `virgulilla` (`˜`)가 붙은 것이 아니라, 스페인어 알파벳에서 완전히 별개의 글자이며, 단어의 발음과 의미 모두를 바꿉니다.
