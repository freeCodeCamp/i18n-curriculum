# --description--

애플리케이션 수준 미들웨어는 Express 앱 인스턴스에 바인딩된 미들웨어입니다. 주로 여러 라우트에 걸쳐 처리가 필요한 요청을 다루는 데 사용합니다. 전역적으로 또는 특정 라우트에 적용할 수 있습니다.

이 미들웨어는 `app.use()` 또는 `app.get()`, `app.post()` 같은 특정 라우트 메서드를 사용해 등록합니다. 특정 라우트에 제한하지 않는 한 모든 들어오는 요청에 대해 실행됩니다.

```js
const express = require('express')
const app = express()

// Application-level middleware that logs all incoming requests
app.use((req, res, next) => {
  console.log('Request received at:', new Date())
  next(); // Passes control to the next middleware
})

app.get('/', (req, res) => {
  res.send('Hello World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* `app.use()`: 이것은 애플리케이션 수준 미들웨어입니다. 라우트와 상관없이 서버에 들어오는 모든 요청을 기록합니다.
* 이 미들웨어는 모든 들어오는 요청에 대해 실행되어 현재 타임스탬프 같은 세부 정보를 서버에 제공합니다.

애플리케이션 수준 미들웨어는 앱 내 많은 라우트 또는 모든 라우트에 적용해야 하는 작업에 유용합니다. 여기에는 로깅, 인증, 오류 처리 등이 포함될 수 있습니다.
