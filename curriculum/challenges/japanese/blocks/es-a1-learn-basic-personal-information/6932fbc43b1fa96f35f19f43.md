# --explanation--

エステバンはこう言っています：`Me llamo Esteban Navarro Huamán.`

`Me llamo + name`の構造は自分の名前を述べるときに使われます。ここでエステバンは自分の`nombre`と`apellido`の両方を伝えており、フルネームについての質問に答えています。この答えは次の質問に合っています：

`¿Cuál es tu nombre y apellido?`

もう一つよく使われる名前を尋ねる質問は：

`¿Cómo te llamas?`

どちらの質問も名前を尋ねるために使われますが、構造が異なります。
