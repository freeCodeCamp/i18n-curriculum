# --description--

偶数の正の整数 $N$ は、2のべき乗であるか、または異なる素因数が連続した素数であれば、許容されると呼ばれます。

最初の12個の許容される数は 2、4、6、8、12、16、18、24、30、32、36、48 です。

$N$ が許容される場合、$N + M$ が素数となる最小の整数 $M > 1$ は、$N$ の疑似フォーチュン数と呼ばれます。

例えば、$N = 630$ は偶数であり、その異なる素因数が連続した素数 2、3、5、7 であるため許容されます。631の次の素数は641なので、630の疑似フォーチュン数は $M = 11$ です。また、16の疑似フォーチュン数は3であることもわかります。

${10}^9$ 未満の許容される数 $N$ に対するすべての異なる疑似フォーチュン数の合計を求めてください。
