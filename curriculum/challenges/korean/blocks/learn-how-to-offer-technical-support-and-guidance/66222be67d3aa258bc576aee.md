# --description--

표현식 `first things first`는 가장 중요한 일을 먼저 해야 한다는 점을 강조할 때 사용합니다. 보통 작업 시작 시 가장 중요한 단계를 먼저 집중하라는 의미로 말합니다. 예를 들어 `First things first, let's review the project plan`는 다른 작업에 들어가기 전에 계획을 검토하는 것부터 시작하라는 뜻입니다.
