# --description--

列車は4両の車両を $ABCD$ の順番で輸送するために使われます。しかし、列車が車両を回収しに来たときに、車両が正しい順番になっていないことがあります。

車両を並べ替えるために、すべての車両は大きな回転式ターンテーブルに移されます。車両が特定の地点で切り離された後、列車はまだ連結されている車両を引いてターンテーブルから離れます。残った車両は180°回転されます。すべての車両が再び連結され、このプロセスはターンテーブルの使用回数を最小にするために必要なだけ繰り返されます。

$ADCB$ のような配列は簡単に解決できます。車両は $A$ と $D$ の間で分けられ、$DCB$ を回転させると正しい順番になります。

しかし、列車の運転手であるシンプル・サイモンは効率的とは言えず、常に最初に車両 $A$ を正しい位置に置き、次に $B$、その次に…という順で問題を解決します。

4両の車両を使った場合、サイモンにとって最悪の配列（maximix 配列と呼びます）は $DACB$ と $DBAC$ で、それぞれ5回の回転が必要です（もっと効率的な方法なら3回の回転で解決できます）。$DACB$ に対する彼の手順は以下の通りです。

<img alt="five rotations for maximix arrangement DACB" src="https://cdn.freecodecamp.org/curriculum/project-euler/maximix-arrangements.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

6両の車両の場合、24の maximix 配列があることが確認されており、そのうち辞書順で10番目の maximix 配列は $DFAECB$ です。

11両の車両に対して、辞書順で ${2011}^{\text{th}}$ の maximix 配列を見つけてください。
