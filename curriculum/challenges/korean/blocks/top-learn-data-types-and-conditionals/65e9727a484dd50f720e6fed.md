# --description--

모든 프로그래밍 언어에서 값을 비교해 결정을 내려야 합니다. 예를 들어, 날씨가 맑으면 외출하고, 그렇지 않으면 집에 머무릅니다. 또 다른 예로는 사용자가 게임에서 레벨업할 만큼 충분한 점수를 가지고 있는지 확인하는 경우가 있습니다.

JavaScript에서 `if-else` 조건문은 결정을 내릴 때 사용합니다. 보통 비교 연산자와 함께 값을 비교하고 결과에 따라 결정을 내리는 데 사용합니다.

if 문의 예시는 다음과 같습니다:

```javascript
let x = 5;

if (x > 3) {
  console.log("x is greater than 3");
} else {
  console.log("x is less than or equal to 3");
}
```

이 예시에서 `x` 변수는 `3` 연산자를 사용해 숫자 `>`와 비교됩니다. 만약 `x`가 `3`보다 크면 첫 번째 코드 블록이 실행됩니다. 그렇지 않으면 두 번째 코드 블록이 실행됩니다.
