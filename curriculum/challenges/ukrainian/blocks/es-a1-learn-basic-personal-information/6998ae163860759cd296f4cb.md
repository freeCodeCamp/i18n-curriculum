# --explanation--

Ángela каже `¿Navarro con dos erres?` Це питання для підтвердження.

`Con` додає додаткову інформацію про написання.

`Dos` — це число два.

`Erres` — множина від `erre`, назви літери `r`.

Ángela підтверджує, що прізвище Navarro написане з двома літерами `r`.
