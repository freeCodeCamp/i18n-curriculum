# --description--

当你谈论某人希望将来做什么时，你可以使用 `want to` 来表示愿望或计划。例如：

- `I want to visit Paris next summer.` - 这意味着你希望将来去那里。

- `He wants to learn guitar.` - 这意味着他计划或希望有一天弹吉他。
