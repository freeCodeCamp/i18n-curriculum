# --description--

Agora, é hora de gerar um número aleatório usando o método `Math.random()`.

Crie uma variável chamada `randomNum` e atribua a ela o valor do resultado da chamada do método `Math.random()`.

Então, registre a variável `randomNum` no console.

Tente adicionar um espaço no código para reexecutar o bot e ver diferentes números aleatórios que são gerados.

```js
assert.isNotNull(randomNum);
```
