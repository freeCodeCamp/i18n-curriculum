# --instructions--

Für diese Aufgabe müssen Sie einen grafikfähigen Taschenrechner mit Python erstellen, der Eingaben annimmt und Folgendes kann:

- Eine oder mehrere Funktionen grafisch darstellen
- Eine Tabelle mit (x,y)-Werten erstellen
- Über oder unter der Linie schattieren
- Ein Gleichungssystem lösen und grafisch darstellen
- In eine Grafik hinein- oder herauszoomen
- Quadratische Gleichungen lösen
