# --description--

Eine weitere häufige Datenstruktur, auf die Sie in der Informatik stoßen, ist die <dfn>verkettete Liste</dfn>. Eine verkettete Liste ist eine lineare Sammlung von Datenelementen, die „Knoten“ genannt werden, von denen jeder auf den nächsten zeigt. Jeder <dfn>Knoten</dfn> in einer verketteten Liste enthält zwei wichtige Informationen: den `element` selbst und eine Referenz auf den nächsten `node`.

Stellen Sie sich vor, Sie stehen in einer Polonaise. Sie haben Ihre Hände auf der nächsten Person in der Reihe, und die Person hinter Ihnen hat ihre Hände auf Ihnen. Sie sehen die Person direkt vor sich, aber sie versperrt die Sicht auf die anderen Personen weiter vorne in der Reihe. Ein Knoten ist wie eine Person in einer Polonaise: Er weiß, wer er ist, und kann nur die nächste Person in der Reihe sehen, aber er ist sich der anderen Personen vor oder hinter ihm nicht bewusst.

```js
assert(Puppy.next.element === 'Cat');
```

