# --description--

Umejifunza kuhusu API za `HTMLMediaElement` na MediaStreams, lakini kuna API chache zaidi ambazo zinaweza kusaidia sana katika kazi za video na sauti.

Ya kwanza ni Screen Capture API. Kama jina linavyoweza kuonyesha, API hii inakuwezesha kurekodi skrini ya mtumizi. API hii inafunguliwa kwa kuita njia ya `getDisplayMedia()` ya kitu cha `mediaDevices` na kutumia mtiririko wa media unaorejeshwa.

Kisha kuna MediaStream Recording API. API hii hufanya kazi pamoja na API za MediaStreams, ikikuruhusu kurekodi MediaStream (au hata `HTMLMediaElement` moja kwa moja). Kisha hutoa matukio ya `dataavailable` yenye mzigo wa Blob ambao unaweza kuandika kwenye hifadhi ya faili ya eneo la mtumiaji.

Kificho cha teknolojia hii yote ni Media Source Extensions API. Media Source Extensions API ndiyo inayokuwezesha kupitisha moja kwa moja picha ya kamera ya mtumizi kwa kipengele cha video kwa kutumia sifa ya `srcObject`, kwa mfano.

Na hatimaye, Web Audio API ambayo inaendesha kila kitu kinachosikika kwenye mtandao. API hii inajumuisha vitu muhimu kama `AudioBuffer` (inayowakilisha Kihifadhi cha muda kinachoshikilia data ya sauti) au `AudioContext`.

Na hapo tunamaliza mafundisho yetu kuhusu sauti na video!
