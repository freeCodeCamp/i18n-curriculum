# --explanation--

在疑問句中，情態動詞如 `can`、`could` 和 `should` 位於主詞之前。接著的動詞必須使用原形動詞。因此你會說 `Could you help?`，而不是 `Do you can help?` 或 `Can helping?`

例如，在 `Should we report this issue?` 和 `Might he know the password?` 中，情態動詞位於前面，主要動詞保持原形。

在情態動詞後使用 `do` 或 `does` 是錯誤的。你不會說 `Do you can reset the system?`，而是說 `Can you reset the system?`

英文疑問句中，字序很重要。說 `You can help me?` 聽起來不自然，因為主詞和情態動詞的順序不正確。正確的形式是 `Can you help me?`
