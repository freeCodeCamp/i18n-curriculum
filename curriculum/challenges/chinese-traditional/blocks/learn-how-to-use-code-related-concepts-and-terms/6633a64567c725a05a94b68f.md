# --description--

在這個任務中，你將探索術語的隱喻用法以理解程式設計概念。隱喻是一種強大的方式，透過將複雜的想法與日常物件或動作聯繫起來，來視覺化和解釋它們。

`Container` 用作比喻來描述變數，表示它們可以像 `container` 一樣容納和儲存不同的資料。

動作 `fill and empty` 描述你如何為變數添加或移除資料，類似於你如何 `fill` 一個水瓶或將水倒出。

「`Locked boxes`」描述常數為安全、不可變更的`容器`，強調一旦設定後就無法更改。

片語 `keep the same contents` 凸顯標示常數中的資料不會改變，就像一個被鎖住且內部永遠裝著相同東西的盒子。
