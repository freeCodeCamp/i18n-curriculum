# --explanation--

`Minor bugs` — це невеликі помилки або проблеми в системі чи програмі. Наприклад:

`There are some minor bugs in the app, but it still runs.` — це означає, що проблеми несерйозні, але їх потрібно виправити.

`User interface` (часто називають `UI`) — це візуальна частина програмного забезпечення, з якою люди взаємодіють. Наприклад:

`The user interface was updated to include a dark mode.` — це стосується змін, які користувачі можуть бачити і безпосередньо використовувати.
