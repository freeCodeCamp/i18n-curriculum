# --description--

Для будь-якого додатного цілого числа $n$ функція $\text{next_prime}(n)$ повертає найменше просте число $p$, таке що $p > n$.

Послідовність $a(n)$ визначається так: $a(1) = \text{next_prime}({10}^{14})$ та $a(n) = \text{next_prime}(a(n - 1))$ для $n > 1$.

Послідовність Фібоначчі $f(n)$ визначається так: $f(0) = 0$, $f(1) = 1$ та $f(n) = f(n - 1) + f(n - 2)$ для $n > 1$.

Послідовність $b(n)$ визначається як $f(a(n))$.

Знайдіть $\sum b(n)$ для $1≤n≤100\\,000$. Відповідь подайте за модулем $1\\,234\\,567\\,891\\,011$.
