# --interactive--

`reduce` 方法是 JavaScript 中的一個函式，允許你處理陣列並將其縮減為單一值。這個單一值可以是數字、字串、物件，甚至是另一個陣列。

它被稱為 `reduce`，因為它將陣列縮減為單一輸出。雖然一開始看起來可能很複雜，但理解 `reduce` 可以在許多情況下大大簡化你的程式碼。

在核心上，`reduce` 是透過依序對陣列中的每個元素套用一個函式，並將每次計算的結果傳遞給下一個元素來運作。這個函式通常稱為 reducer 函式。

reducer 函式接受兩個主要參數：累加器和當前值。累加器是你用來儲存操作結果的地方，而當前值是正在處理的陣列元素。

讓我們看一個範例來說明 `reduce` 如何運作：

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];
const sum = numbers.reduce(
  (accumulator, currentValue) => accumulator + currentValue,
  0
);

console.log(sum); // 15
```

:::

在這個範例中，我們使用 `reduce` 來取得陣列中所有數字的總和。

reducer 函式（程式）會取得累加器（從 `0` 開始，依據傳給 `reduce` 的第二個引數指定）並將每個數字加到累加器中。

每次相加的結果會成為下一次迭代的新累加器。

`reduce` 方法也可以將初始值作為第二個引數。這是累加器開始時的值。在上述範例中，我們將它設為 `0`。

如果你沒有提供初始值，`reduce` 將會使用陣列的第一個元素作為初始累加器，並從第二個元素開始處理程序。

`reduce` 的一大優點是它的彈性。因為你定義了 reducer 函式（程式），所以你可以完全控制陣列的處理程序以及你想產生的結果類型。這使得 `reduce` 非常強大，但一開始也可能讓人有點難以理解。

經過練習，你將會熟悉使用 `reduce` 方法。
