# --explanation--

La oración usa `will` para indicar el tiempo `Future Simple`, mostrando que se espera que la acción ocurra. 

La estructura `will + verb` se usa comúnmente para hablar de eventos futuros, planes o predicciones. Por ejemplo:

- `They will be at the meeting tomorrow.` - Esto significa que su presencia está planeada para el futuro.

- `We will start the project next week.` - Esto significa que el proyecto está programado para comenzar en el futuro.

Usar `will` es una forma sencilla de expresar intenciones futuras, promesas o horarios.
