# --fillInTheBlank--

## --sentence--

`BLANK BLANK organizar productos y equipos.`

## --blanks--

`Le`

### --feedback--

Este é o pronome de objeto indireto que se refere a "ela". Lembre-se de colocar a primeira letra em maiúscula.

---

`gusta`

### --feedback--

Esta palavra significa "gosta".

# --explanation--

Julieta diz isso sobre Valeria, `le gusta organizar productos y equipos`. Isso significa que "ela gosta de organizar produtos e equipes".

No contexto de uma pessoa do sexo feminino como Valeria, `le gusta` significa "Ela gosta de..."

`Organizar` significa "organizar".

`Productos` significa "produtos".

`Equipos` significa "equipes".
