# --explanation--

A frase usa `will` para indicar o tempo `Future Simple`, mostrando que a ação é esperada para acontecer. 

A estrutura `will + verb` é comumente usada para falar sobre eventos futuros, planos ou previsões. Por exemplo:

- `They will be at the meeting tomorrow.` - Isso significa que a presença deles está planejada para o futuro.

- `We will start the project next week.` - Isso significa que o projeto está programado para começar no futuro.

Usar `will` é uma maneira simples de expressar intenções futuras, promessas ou horários.
