# --explanation--

`When` тут використовується, щоб описати конкретну умову або ситуацію, яка спричиняє щось.

У реченні Сари `when` вводить умову, за якої IDE стають складними: `if they're not set up properly`. Наприклад:

- `The program crashes when it runs out of memory.` — умова вичерпання пам’яті спричиняє збій програми.

- `The printer only works when it's plugged in.` — умова підключення до мережі дозволяє принтеру працювати.

В обох випадках `when` використовується, щоб пояснити ситуацію, у якій щось відбувається.
