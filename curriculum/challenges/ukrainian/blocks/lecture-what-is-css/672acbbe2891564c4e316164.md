# --interactive--

CSS відповідає за стилі вебсторінки. Усі ці стилі складаються з різних правил CSS.

Правило CSS складається з двох основних частин: **селектора** та блоку **оголошення**.

Давайте розглянемо базовий синтаксис:

```css
selector {
  property: value;
}
```

Селектор — це шаблон, який використовується в CSS для ідентифікації та цільового застосування стилів до конкретних HTML-елементів.

Прикладами селекторів є селектори типу, селектори класу та селектори ідентифікатора.

Фігурні дужки, наведені в базовому синтаксисі, називаються блоком оголошення. Блок оголошення застосовує множину стилів для заданого селектора або селекторів.

Всередині блоку оголошення знаходиться ряд оголошень. Кожне оголошення складається з властивості та значення.

Властивість — це CSS-ідентифікатор, який вказує, яка функція стилізується. Прикладом властивості є властивість `background-color`.

Значення — це конкретне налаштування, застосоване до цієї властивості. Наприклад, якщо властивість — `background-color`, значенням може бути `purple`, яке встановлює колір фону фіолетовим.

Після кожної назви властивості потрібно поставити двокрапку, а після кожного значення — крапку з комою.

Тепер, коли ви знаєте синтаксис правила CSS, давайте розглянемо приклад. Увімкніть інтерактивний редактор і натисніть на вкладку `styles.css`, щоб побачити код CSS.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Learning about CSS</h1>

<p>Example paragraph element</p>
<p>Another example paragraph element</p>
```

```css
p {
  color: blue;
}
```

:::

У цьому правилі CSS селектор типу цілиться у всі елементи абзацу на сторінці.

Всередині блоку оголошення є одне оголошення.

Оголошення складається з властивості `color` зі значенням `blue`. Це змінить колір тексту для всіх елементів абзацу на `blue`.

Якщо ви хочете застосувати один і той самий набір стилів до кількох селекторів, ви можете створити список селекторів, розділених комою.

Ось приклад стилізації кількох селекторів (щоб взаємодіяти з прикладом, увімкніть інтерактивний редактор):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1 id="title">Example heading</h1>
<h2 class="subheading">Example subheading</h2>
<p>This paragraph is not affected by the selector.</p>
```

```css
#title,
.subheading {
  color: navy;
}
```

:::

У цьому списку селекторів є селектор `id`, який цілиться в HTML-елемент з `id` значенням `title`. Усі селектори `id` повинні починатися з символу хеш `#`.

Потім іде кома, після якої йде селектор `class`, що цілиться у всі HTML-елементи з `class` значенням `subheading`. Усі селектори класу повинні починатися з крапки `.`.

Весь список селекторів отримає однакове стилізування, з кольором тексту, встановленим у `navy`.
