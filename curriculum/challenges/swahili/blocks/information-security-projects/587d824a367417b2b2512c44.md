Unaweza kutuma ombi la `GET` kwa `/api/stock-prices`, ukipitisha alama ya hisa ya NASDAQ kama kigezo cha hoja ya utafutaji `stock`. Kifaa kilichorejeshwa kitakuwa na sifa iitwayo `stockData`.

```js
  const data = await fetch(
    code + '/api/stock-prices?stock=GOOG'
  );
  const parsed = await data.json();
  assert.property(parsed, 'stockData');
```

Sifa ya `stockData` inajumuisha alama ya `stock` kama mfuatano wa herufi, `price` kama nambari, na `likes` kama nambari.

```js
  const data = await fetch(
    code + '/api/stock-prices?stock=GOOG'
  );
  const parsed = await data.json();
  const ticker = parsed.stockData;
  assert.typeOf(ticker.price, 'number');
  assert.typeOf(ticker.likes, 'number');
  assert.typeOf(ticker.stock, 'string');
```
