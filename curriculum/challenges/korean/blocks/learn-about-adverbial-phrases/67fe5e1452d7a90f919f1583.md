# --explanation--

`To be honest`는 누군가가 보통 말하지 않을 수도 있는 것을 진실되거나 직접적으로 말하고 싶을 때 사용합니다. 이는 화자가 자신의 의견에 대해 솔직하거나 진심임을 나타냅니다. 예를 들어:

`To be honest, I think we need more time to finish the project.` – 여기서는 진심 어린 의견을 말하고 있습니다.

`To end up doing something`는 일련의 사건이나 행동의 결과로, 때로는 예상치 못하게 결국 어떤 일을 하게 된다는 뜻입니다. 이는 결과가 계획된 것이 아님을 암시합니다. 예를 들어:

`We planned to visit the museum, but we ended up going to the park instead.` – 이 문장은 팀이 원래 박물관에 가려고 했지만 어떤 이유나 계획 변경으로 결국 공원에 갔다는 뜻입니다.
