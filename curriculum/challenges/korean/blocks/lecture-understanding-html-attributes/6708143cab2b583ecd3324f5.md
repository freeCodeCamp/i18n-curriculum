# --interactive--

속성은 HTML 요소의 여는 태그 안에 위치하는 값입니다. 속성은 요소에 대한 추가 정보를 제공하거나 요소가 어떻게 동작해야 하는지를 지정합니다. 속성의 기본 구문은 다음과 같습니다:

```html
<element attribute="value"></element>
```

속성 이름 다음에는 등호(`=`)가 오고, 따옴표 안에 값이 옵니다. 값은 속성에 따라 문자열이나 숫자가 될 수 있습니다.

첫 번째 예제에서는 `href`와 `target` 속성을 사용합니다. `href` 속성은 링크의 URL을 지정하고 `target` 속성은 링크를 어디에서 열지 지정합니다.

**참고:** `a` 요소는 앵커 요소라고도 하며 하이퍼링크를 만드는 데 사용됩니다. 여는 `a` 태그와 닫는 태그 사이의 텍스트가 사용자가 선택하여 이동할 수 있는 클릭 가능한 부분입니다.

인터랙티브 편집기를 활성화하고 `href="https://www.freecodecamp.org/news/"`을 `href="https://www.freecodecamp.org"`로 변경하세요. 이제 인터랙티브 편집기에서 링크를 클릭하면 새 브라우저 탭에서 freeCodeCamp 홈페이지가 열립니다.

:::interactive_editor

```html
<a href="https://www.freecodecamp.org/news/" target="_blank">Visit freeCodeCamp</a>
```

:::

`href` 속성이 없으면 링크가 작동하지 않습니다. 목적지 URL이 없기 때문입니다. 따라서 링크를 작동시키려면 이 `href` 속성을 반드시 포함해야 합니다. `target="_blank"` 속성은 링크가 새 브라우저 탭에서 열리도록 합니다. `target` 속성에 대해서는 앞으로 수업에서 더 배우게 됩니다.

다른 일반적인 속성으로는 `src`와 `alt`, 즉 이미지의 출처를 지정하는 속성과 이미지에 대한 대체 설명 텍스트를 제공하는 대체 속성이 있습니다.

인터랙티브 편집기를 활성화하고 `src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"`를 `src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/running-cats.jpg"`로 변경하세요. 그런 다음 `alt="Two tabby kittens sleeping together on a couch."`를 `alt="Two cats running in the dirt."`로 변경하세요.

:::interactive_editor

```html
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" alt="Two tabby kittens sleeping together on a couch." />
```

:::

`href` 속성과 마찬가지로 `src` 속성은 표시할 이미지 파일을 지정하기 때문에 필수입니다. `alt` 속성은 필수는 아니지만 접근성을 위해 권장됩니다. 접근성은 장애가 있는 사람을 포함해 모든 사용자가 웹사이트, 앱, 물리적 공간 등을 사용할 수 있고 이해할 수 있도록 하는 것을 의미합니다. 앞으로 수업에서 접근성에 대해 더 배우게 됩니다.

일부 속성은 `checked` 속성처럼 구문이 조금 독특합니다.

인터랙티브 편집기를 활성화하고 미리보기 창에서 체크박스를 클릭해 체크된 상태와 체크되지 않은 상태가 번갈아 나타나는 것을 확인하세요.

:::interactive_editor

```html
<input type="checkbox" checked />
```

:::

다음 예제에는 `input` 요소가 있고 `type` 속성이 `checkbox`로 설정되어 있습니다. 입력 요소는 사용자로부터 데이터를 수집하는 데 사용되며 `type` 속성은 입력 유형을 지정합니다. 이 경우 입력 유형은 체크박스입니다. 입력 요소가 어떻게 작동하는지에 대해서는 앞으로 수업에서 더 배우게 됩니다.

`checked` 속성은 체크박스가 기본적으로 선택되도록 지정하는 데 사용됩니다. `checked` 속성은 값이 필요 없습니다. 이 속성이 있으면 체크박스가 기본적으로 선택되고, 없으면 선택되지 않은 상태입니다. 이것을 불리언 속성이라고 합니다. 불리언에 대해서는 JavaScript 섹션에서 더 배우게 됩니다.

인터랙티브 편집기를 활성화하고 `checked` 속성을 `input`에서 제거해 보세요. 그러면 체크박스가 더 이상 기본적으로 선택되지 않은 것을 볼 수 있습니다.

:::interactive_editor

```html
<input type="checkbox" checked />
```

:::

HTML에서 자주 접하는 불리언 속성으로는 `disabled`, `readonly`, `required` 등이 있습니다. 이 속성들은 요소가 비활성화되었는지, 읽기 전용인지, 필수인지 등의 상태를 지정하는 데 사용됩니다.

다음은 기본적으로 비활성화된 텍스트 `input` 요소의 예입니다. 인터랙티브 편집기를 활성화하고 미리보기 창에서 `input` 요소를 클릭해 보세요. 이제 `disabled` 속성을 `input` 요소에서 제거하면 `input`가 더 이상 기본적으로 비활성화되지 않은 것을 볼 수 있습니다. 이제 클릭하고 필드 안에 입력할 수 있습니다.

:::interactive_editor

```html
<input type="text" disabled>
```

:::

HTML에는 웹페이지의 요소 동작과 외관을 맞춤 설정하는 데 사용할 수 있는 많은 속성이 있습니다. 속성을 어떻게 사용하는지 이해하는 것은 상호작용 가능하고 접근성 높은 웹 콘텐츠를 만드는 데 필수적입니다. 앞으로 몇 수업에서 더 많은 HTML 속성과 웹 개발 프로젝트에서 효과적으로 사용하는 방법을 배우게 됩니다.
