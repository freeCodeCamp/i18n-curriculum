# --explanation--

`Right` наприкінці речення — це питальний тег. Його використовують, щоб підтвердити згоду або перевірити, чи слухач розуміє. Наприклад:

- `It's going to rain today, right?` — мовник просить підтвердження щодо погоди.

- `You finished the project, right?` — мовник перевіряє, чи слухач виконав завдання.

У цьому діалозі Боб просить Марію підтвердити або погодитися з його твердженням.
