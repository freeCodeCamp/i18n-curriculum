# --description--

게임 숫자 마음은 잘 알려진 게임 마스터 마인드의 변형입니다.

색깔 핀 대신, 비밀 수열의 숫자를 맞혀야 합니다. 각 추측 후에는 몇 자리에서 정확한 숫자를 맞혔는지만 알려줍니다. 예를 들어, 수열이 1234이고 2036을 추측했다면, 정확한 숫자가 한 자리 있다고 알려주지만, 잘못된 위치에 다른 숫자가 있다는 것은 알려주지 않습니다.

예를 들어, 5자리 비밀 수열에 대해 다음과 같은 추측이 주어졌을 때,

  & 34109 ;1\\;\text{correct}\\\\
  & 51545 ;2\\;\text{correct}\\\\
  & 12531 ;1\\;\text{correct}
\end{align}$$
