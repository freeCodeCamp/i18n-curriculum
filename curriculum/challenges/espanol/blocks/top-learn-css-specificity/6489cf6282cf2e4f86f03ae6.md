# --description--

```css
/* rule 1 */
#subsection .list {
  background-color: yellow;
  color: blue;
}

/* rule 2 */
#subsection .main .list {
  color: red;
}
```

En este ejemplo final, ambas reglas están usando selectores de ID y clase, por lo que ninguna regla está usando un selector más específico que la otra. La cascada entonces verifica la cantidad de cada tipo de selector. Ambas reglas solo tienen un selector de ID, pero la regla 2 tiene más selectores de clase, ¡por lo que la regla 2 tiene una especificidad mayor!

Aunque la declaración `color: red` tendría prioridad, la declaración `background-color: yellow` aún se aplicaría porque no hay una declaración conflictiva para ella.
