# --description--

```css
/* rule 1 */
.class.second-class {
  font-size: 12px;
}

/* rule 2 */
.class .second-class {
  font-size: 24px;
}
```

ここではルール1とルール2が同じ特異性を持っています。ルール1はチェーンセレクター（スペースなし）を使い、ルール2は子孫結合子（空白）を使っています。しかし両方のルールは2つのクラスを持ち、結合子の記号自体は特異性に影響しません。

```css
/* rule 1 */
.class.second-class {
  font-size: 12px;
}
/* rule 2 */
.class > .second-class {
  font-size: 24px;
}
```

この例も同じことを示しています。ルール2が子結合子（`>`）を使っていても、特異性の値は変わりません。両方のルールは2つのクラスを持つため、同じ特異性の値です。

*注意:* すべてが特異性に加算されるわけではありません

セレクターを比較するとき、ユニバーサルセレクター（`*`）や結合子（`+`、`~`、`>`、および空白）などの特殊な記号に出会うことがあります。これらの記号はそれ自体では特異性に加算されません。
