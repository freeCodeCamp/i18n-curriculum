# --explanation--

**Frações** são usadas para descrever partes de um todo.

`One-fourth` (¼) é uma parte de quatro partes iguais.Outros exemplos:

- ½ = `one-half`

- ⅓ = `one-third`

- ¾ = `three-fourths`

No caso da Alice, um quarto do orçamento de desenvolvimento é gasto em desenvolvimento back-end, o que mostra como o dinheiro é dividido dentro de uma categoria.
