# --interactive--

函式是可重複使用的程式碼片段，用來執行特定任務或計算一個值。將函式想像成一台機器，接收一些輸入，對其進行一些操作，然後產生輸出。以下是一個宣告函式的範例：

```js
function greet() {
  console.log("Hello, Jessica!");
}
```

在這個範例中，我們宣告了一個名為 `greet` 的函式。在該函式內，我們有一個 `console.log`，它會記錄訊息 `Hello, Jessica!`。如果我們嘗試執行這段程式碼，我們不會在主控台看到該訊息出現。這是因為我們需要呼叫該函式。

函式（程式）的呼叫，或稱為呼叫，是指我們實際使用或執行該函式。要呼叫函式，你需要取用函式名稱，後面接上一組小括弧：

:::interactive_editor

```js
function greet() {
  console.log("Hello, Jessica!");
}

greet(); // "Hello, Jessica!"
```

:::

現在 `Hello, Jessica!` 的訊息將會被記錄到主控台。但如果我們想要訊息顯示為 `Hello, Nick!` 或 `Hello, Anna!` 呢？我們不想每次向不同使用者問候時都寫一個新的函式。相反地，我們可以創建一個可重複使用的函式，該函式使用函式參數和引數。 

參數作為佔位符，用來接收在呼叫函式時傳遞給函式的值。它們允許函式接受輸入並處理該輸入。引數是在呼叫函式時實際傳遞給函式的值。以下是使用參數和引數的 `greet` 函式更新版本：

:::interactive_editor

```js
function greet(name) {
  console.log("Hello, " + name + "!");
}

greet("Alice"); // Hello, Alice!
greet("Nick"); // Hello, Nick!
```

:::

`name` 作為參數，而字串 `Alice` 和 `Nick` 作為引數。現在我們有一個可重複使用的函式（程式），可以在我們的程式碼中用不同的引數使用數十次。 

當一個函式（程式）完成執行時，它總是會傳回一個值。預設情況下，返回值會是 `undefined`。以下是一個範例： 

:::interactive_editor

```js
function doSomething() {
  console.log("Doing something...");
}

let result = doSomething();
console.log(result); // undefined
```

:::

如果你需要你的函式（程式）傳回特定的值，那麼你將需要使用 `return` 述語。以下是一個使用 `return` 述語來傳回兩個值總和的範例：

:::interactive_editor

```js
function calculateSum(num1, num2) {
  return num1 + num2;
}

console.log(calculateSum(3, 4)); // 7
```

:::

你經常會使用 `return` 述語，因為你可以在程式碼中稍後使用從函式輸出的該值。

到目前為止，我們一直在使用具名函式（function），但你也可以創建所謂的匿名函式。匿名函式是沒有名稱的函式，可以像這樣指定給變數：

:::interactive_editor

```js
const sum = function (num1, num2) {
  return num1 + num2;
};

console.log(sum(3, 4)); // 7
```

:::

在這個範例中，我們有一個名為 `sum` 的 `const` 變數，並且我們將一個傳回 `num1` 和 `num2` 總和的匿名函式指定給它。接著，我們能夠呼叫 `sum` 並傳入數字 `3` 和 `4`，以取得結果 `7`。 

函式（程式）支援預設參數，允許你為參數設定預設值。如果函式（程式）在呼叫時沒有提供該參數的引數，則會使用這些預設值。以下是一個範例：

:::interactive_editor

```js
function greetings(name = "Guest") {
  console.log("Hello, " + name + "!");
}

greetings(); // Hello, Guest!
greetings("Anna"); // Hello, Anna!
```

:::

在此範例中，如果沒有為 `name` 提供引數，則會使用 `Guest` 作為預設值。

總結來說，函式（程式）允許你撰寫可重複使用且有組織的程式碼。它們可以接受輸入（參數）、執行動作，並傳回輸出。
