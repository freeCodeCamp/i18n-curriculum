# --interactive--

CSS Grid 是一个强大的布局系统，允许 web 开发人员轻松创建复杂的响应式网页布局。

想象一下，你正在房间里安排家具——CSS Grid 就像在地板上有一个看不见的网格，可以帮助你精确地将所有东西放置在你想要的位置。

当我们建立网站时，我们经常需要在页面上安排不同的元素。

在 CSS Grid 出现之前，这有时很棘手，特别是对于复杂的布局。 CSS Grid 通过将你的网页分成行和列并创建一个网格状的结构来简化这个过程。

让我们假设你正在使用一个容器 `div`，其中嵌套了几个项目，如下所示：

:::interactive_editor

```html
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

:::

如果你想给这些元素设置网格格式的样式，你可以将 `display` 设置为 `grid`，并像这样应用列：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-gap: 20px;
}

.item {
  background-color: lightgray;
  padding: 20px;
  text-align: center;
  border: 1px solid #ccc;
}
```

:::

在这段代码中，我们告诉浏览器创建一个具有三个等宽列的网格，这就是 `1fr 1fr 1fr` 的含义，我们在每个网格项之间添加了一个 20 像素的间隙。

现在，你可能会想：“Flexbox 呢？ 不是也用于布局的吗？”

你说的对！ Flexbox 是另一个 CSS 布局模型，它也非常有用。 但是有一些关键的区别。

Flexbox 是一维的，而 Grid 是二维的。 这意味着 Flexbox 在单行或单列布局方面很适合，而 Grid 擅长同时创建行和列布局。

Flexbox 是内容优先的，这意味着它会根据内容调整布局。 而 Grid 是布局优先的，允许你先创建布局，然后将项目放入其中。 Grid 使你可以更精确地控制位置。 你可以准确地告诉一个项目占用哪一行和哪一列。

这是一个用于对比的 Flexbox 的例子：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
}
```

:::

这将创建一个弹性容器，其中项目沿主轴均匀间隔分布。

Grid 和 Flexbox 都有各自的优势，通常，最好的布局是使用两者的组合。 你可以使用 Grid 来进行整个页面布局，然后使用 Flexbox 来对齐每个网格区域内的项目。

总之，CSS Grid 是一个强大的工具，可以实现精确的二维布局。 虽然一开始看起来很复杂，但经过练习，它会成为创建响应性和复杂网页布局的宝贵工具。
