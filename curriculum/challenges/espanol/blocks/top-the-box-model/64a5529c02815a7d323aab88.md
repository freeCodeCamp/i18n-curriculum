# --description--

Ser capaz de inspeccionar y depurar tu HTML y CSS es critico para el desarrollo frontend. Esta lección te llevara atraves de las herramientas de desarrollo de Chrome, cual te permitira ver información detallada sobre tus elementos y reglas de CSS, como tambien te asistira en encontrar y arreglar los problemas en tu código.

Para abri el inspector, tu puedes hacer click derecho sobre cualquier elemento de una pagina web y hacer click en "Inspeccionar" o presionar F12. Ve hazlo ahora mismo para ver el uso del HTML y CSS en la pagina.

¡No te abrumas con todas las herramientas que ahora estas viendo! Para esta lección, Nosotros queremos enfocarnos sobre los paneles de Elementos y Estilos.
