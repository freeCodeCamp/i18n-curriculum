# --explanation--

Джессіка каже `you must update your software tools`, що означає, що це необхідно.

`Must` виражає сильну необхідність або обов’язок. Наприклад:

- `You must wear a helmet when riding a bike.` — обов’язково для безпеки.

- `Employees must complete training before starting work.` — правило, якого працівники зобов’язані дотримуватися.
