# --description--

Натуральні числа $x$, $y$ та $z$ є послідовними членами арифметичної прогресії. Якщо $n$ — натуральне число, то рівняння $x^2 − y^2 − z^2 = n$ має рівно одне розв’язання, коли $n = 20$:

$$13^2 − 10^2 − 7^2 = 20$$

Насправді існує двадцять п’ять значень $n$ менше за сто, для яких рівняння має унікальне розв’язання.

Скільки значень $n$ менше п’ятдесяти мільйонів мають рівно одне розв’язання?
