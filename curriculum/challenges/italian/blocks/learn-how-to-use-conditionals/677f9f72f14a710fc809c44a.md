# --explanation--

Mark menziona `incompatible drivers` e `outdated software` come cause di un crash del software.

`Incompatible drivers` sono driver che non funzionano correttamente con l'hardware, mentre `outdated software` indica che il software non è stato aggiornato all'ultima versione. Entrambi possono causare crash. Per esempio:

- `The graphics card isn't working because of incompatible drivers.` - Il driver sbagliato sta causando malfunzionamenti nella scheda grafica.

- `Outdated software can lead to security vulnerabilities.` - I programmi vecchi potrebbero non essere sicuri da usare a causa della mancanza di aggiornamenti.
