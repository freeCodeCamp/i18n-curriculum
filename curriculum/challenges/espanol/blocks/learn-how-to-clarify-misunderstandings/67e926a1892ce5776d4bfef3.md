# --explanation--

`Reducing complexity` significa hacer que un sistema sea más fácil de entender y manejar.
  
`Showing only what's necessary` significa que el usuario solo ve las partes con las que necesita interactuar.
  
`Keeping the details hidden` se refiere a ocultar el código interno o la lógica que no es necesaria para el usuario.

En conjunto, esto significa que la abstracción crea una interfaz clara y sencilla mientras oculta los mecanismos internos más complejos detrás de escena.
