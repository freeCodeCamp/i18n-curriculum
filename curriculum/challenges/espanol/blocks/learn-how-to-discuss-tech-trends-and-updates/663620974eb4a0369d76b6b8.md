# --description--

La frase `some of these` se utiliza para referirse a una parte específica de un grupo o un subconjunto de elementos de un conjunto mayor. A menudo se usa cuando quieres mencionar algunas cosas de un grupo sin enumerar cada una individualmente.

Por ejemplo, `Let's try some of these cookies.` El hablante sugiere probar algunas galletas de un lote más grande.

### --feedback--

Esta palabra indica una porción de algo, no el todo, refiriéndose a una cantidad no especificada de los elementos discutidos.

---

### --feedback--

Esta preposición se usa para enlazar la cantidad general con los elementos específicos que se están discutiendo, formando una expresión partitiva.

---

`these`

### --feedback--

Esta palabra especifica de qué grupo de elementos se está hablando, indicando un grupo específico, presente.
