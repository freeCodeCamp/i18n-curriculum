# --description--

Змінена послідовність Коллатца цілих чисел отримується зі стартового значення $a_1$ таким чином:

$a_{n + 1} = \frac{a_n}{3}$, якщо $a_n$ ділиться на 3 без остачі. Позначимо це як великий крок вниз, "D".

$a_{n + 1} = \frac{4a_n + 2}{3}$, якщо при діленні $a_n$ на 3 отримуємо остачу 1. Позначимо це як крок вгору, "U".

$a_{n + 1} = \frac{2a_n - 1}{3}$, якщо при діленні $a_n$ на 3 отримуємо остачу 2. Позначимо це як малий крок вниз, "d".

Послідовність завершується, коли деяке $a_n = 1$.

Для будь-якого цілого числа ми можемо записати послідовність кроків. Наприклад, якщо $a_1 = 231$, тоді послідовність $\\{a_n\\} = \\{231, 77, 51, 17, 11, 7, 10, 14, 9, 3, 1\\}$ відповідає крокам "DdDddUUdDD".

Звісно, існують інші послідовності, які починаються з тієї самої послідовності "DdDddUUdDD...".

Наприклад, якщо $a_1 = 1004064$, тоді послідовність буде DdDddUUdDDDdUDUUUdDdUUDDDUdDD.

Насправді, 1004064 — це найменше можливе $a_1 > {10}^6$, що починається з послідовності DdDddUUdDD.

Яке найменше $a_1 > {10}^{15}$, що починається з послідовності "UDDDUdddDDUDDddDdDddDDUDDdUUDd"?
