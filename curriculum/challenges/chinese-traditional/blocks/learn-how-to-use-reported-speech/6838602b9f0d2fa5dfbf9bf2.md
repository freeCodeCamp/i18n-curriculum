# --explanation--

`Unusually` 是一個副詞，用來描述某件事發生的頻率比正常或預期的多或少。另一方面，`unusual` 是一個形容詞，用來描述某件事不常見或不正常。 例如：

- `The server experienced an unusually high load today。` – 在這句話中，`unusually` 是指載入的程度非常高（遠超過正常）。

- `It was unusual for the server to slow down like that.`－在這句話中，`unusual` 是指該情況。它並未修飾動詞或形容詞。

`query` 是發送到資料庫以取得資料或執行動作的問題或請求。例如：

`The application sent a query to the database to fetch user details.`－這表示應用程式向資料庫請求特定的資訊。
