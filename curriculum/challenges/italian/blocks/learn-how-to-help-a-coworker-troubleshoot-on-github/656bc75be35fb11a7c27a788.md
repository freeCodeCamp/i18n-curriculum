# --description--

La frase `Got it` viene spesso usata in conversazioni per indicare che qualcuno ha capito qualcosa. È un modo informale e rapido per riconoscere di aver ricevuto e compreso l'informazione. Per esempio:

Lucy: `You need to finish the report by tomorrow.`
Bob: `Got it!` 

`Got it` significa che Bob ha capito l'istruzione.
