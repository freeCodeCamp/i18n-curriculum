# --description--

ピタゴラス多角形を、次の条件を満たす凸多角形と定義します。

- 頂点が少なくとも3つあること、
- 3つの頂点が一直線上に並ばないこと、
- 各頂点の座標が整数であること、
- 各辺の長さが整数であること。

整数 $n$ が与えられたとき、周囲の長さが $≤ n$ である異なるピタゴラス多角形の数を $P(n)$ と定義します。

ピタゴラス多角形は、どれも他の多角形の平行移動で表せない限り、異なるものとみなします。

$P(4) = 1$、$P(30) = 3655$、$P(60) = 891045$ であることが与えられています。

$P(120)$ を求めてください。
