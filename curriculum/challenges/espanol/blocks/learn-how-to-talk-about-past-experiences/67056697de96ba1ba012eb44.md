# --explanation--

Los conectores secuenciales como `first`, `then`, `after that` y `finally` se usan para describir el orden de las acciones o eventos.

- `First` se usa para describir la acción inicial.

- `Then` se refiere al siguiente paso inmediato.

- `After that` se usa para introducir un paso que sigue a otra acción con cierta distinción entre ellas.

- `Finally` marca el último paso en una secuencia.
