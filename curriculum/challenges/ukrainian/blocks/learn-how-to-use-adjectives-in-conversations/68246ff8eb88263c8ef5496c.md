# --explanation--

`Made sure` означає вжити дії, щоб підтвердити, що щось є правдивим або працює правильно. Наприклад:  

`They made sure the software was ready before launch.` – Вони перевірили, що все працює правильно.

`As` використовується, щоб описати щось, що відбувається одночасно з іншою дією. Наприклад:  

`The system adds more resources as demand increases.` – Воно налаштовується, поки більше людей починають користуватися ним.
