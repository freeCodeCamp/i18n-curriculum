# --description--

$n$을 양의 정수라고 하겠습니다. 다음 조건을 만족하는 님 위치를 고려하세요:

- $n$개의 비어 있지 않은 더미가 있습니다.
- 각 더미의 크기는 $2^n$보다 작습니다.
- 두 더미가 같은 크기를 가지지 않습니다.

$W(n)$을 위 조건을 만족하는 승리 님 위치의 개수라고 하겠습니다(첫 번째 플레이어가 승리 전략을 가진 위치를 승리 위치라고 합니다).

예를 들어, $W(1) = 1$, $W(2) = 6$, $W(3) = 168$, $W(5) = 19\\,764\\,360$이고 $W(100)\bmod 1\\,000\\,000\\,007 = 384\\,777\\,056$입니다.

$W(10\\,000\\,000)\bmod 1\\,000\\,000\\,007$을 구하세요.
