# --description--

次の反復シーケンスは正の整数の集合に対して定義されています：

<div style='padding-left: 4em;'><var>n</var> → <var>n</var>/2（<var>n</var>が偶数の場合）</div>

<div style='padding-left: 4em;'><var>n</var> → 3<var>n</var> + 1（<var>n</var>が奇数の場合）</div>

上記のルールを使い、13から始めると次のシーケンスが生成されます：

<div style='text-align: center;'>13 → 40 → 20 → 10 → 5 → 16 → 8 → 4 → 2 → 1</div>

このシーケンス（13から始まり1で終わる）は10項を含むことがわかります。まだ証明されていませんが（コラッツ問題）、すべての開始数は1で終わると考えられています。

与えられた`limit`の範囲内で、どの開始数が最も長いチェーンを生成しますか？

**注意：** チェーンが始まると、項は`limit`を超えてもかまいません。

```js
function longestCollatzSequence(limit) {
  let longest = 1;
  let maxLength = 1;
  for (let i = Math.floor(limit / 2); i < limit; i++) {
    let len = colLen(i);
    if (len > maxLength) {
      longest = i;
      maxLength = len;
    }
  }
  return longest;
}

const knownSequence = { '1': 1 };

function colLen(n) {
  if (knownSequence[n]) {
    return knownSequence[n];
  } else {
    const len = n % 2 === 0 ? colLen(n / 2) + 1 : colLen((3 * n + 1) / 2) + 2;
    knownSequence[n] = len;
    return len;
  }
}
```
