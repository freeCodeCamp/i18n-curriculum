# --description--

Дієслово `to contain` означає тримати або мати щось всередині. У контексті документації або README файлу `to contain` стосується інформації, яка міститься всередині документа.

Наприклад, пляшка може `contain` воду, так само як README файл `contains` інформацію про проєкт.

# --questions--

## --text--

У контексті документації, що означає `to contain`?

## --answers--

Виключити

### --feedback--

Це означає тримати щось поза межами, що є протилежним до містити. `To contain` означає включати або мати щось всередині.

---

Видалити

### --feedback--

Це означає прибрати або стерти щось, що відрізняється від `containing`. `Containing` означає тримати або включати, а не видаляти.

---

Копіювати

### --feedback--

Цей вираз означає зробити копію чогось. `Containing`, однак, означає мати щось всередині, а не дублювати це.
