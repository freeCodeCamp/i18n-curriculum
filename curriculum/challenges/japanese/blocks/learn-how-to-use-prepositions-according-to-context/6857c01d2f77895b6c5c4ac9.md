# --explanation--

`Opposite` は「～の真向かいに」または「～に面している」という意味です。例えば：

`The meeting room is opposite the elevator.` – これは両者が向かい合っていることを意味します。

`Mail room` はオフィス内で郵便物を扱う場所です。例えば：

`Check the mail room for the delivery.` – これは荷物がそこにあるかもしれないことを意味します。
