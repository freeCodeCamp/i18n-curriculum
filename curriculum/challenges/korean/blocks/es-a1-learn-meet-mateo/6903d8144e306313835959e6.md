# --description--

Mateo가 `tener` 동사("가지다")를 사용하여 스페인어로 나이를 말하는 다음 구조를 설명합니다:

`Tengo` + 숫자 단어 + `años`.

스페인어에서 21부터 29까지 숫자는 `veinti-`로 시작하며, 그 뒤에 해당 숫자가 옵니다. 예를 들어:

25는 `veinticinco`입니다. 이는 `e`를 제거하고 `i`를 숫자 `cinco` 앞에 추가한 결과입니다.

스페인어에서 1부터 9까지 숫자는 다음과 같습니다:

| 숫자 | 스페인어 단어 |
| :---: | :---: |
| 1 | `uno` |
| 2 | `dos` |
| 3 | `tres` |
| 4 | `cuatro` |
| 5 | `cinco` |
| 6 | `seis` |
| 7 | `siete` |
| 8 | `ocho` |
| 9 | `nueve` |
