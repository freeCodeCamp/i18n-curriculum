# --description--

Wörter, die auf `-y` enden, beschreiben oft einen Zustand oder eine Eigenschaft.

Zum Beispiel, wenn Sie `-y` an das Substantiv `sleep` anhängen, entsteht das Adjektiv `sleepy`, das bedeutet, dass man das Bedürfnis hat zu schlafen oder müde ist.

Die Endung `-y` verwandelt das Substantiv `sleep` in ein Adjektiv, das einen Zustand beschreibt. Ähnlich:

- `snow` + `y` = `snowy` (viel Schnee haben)

- `rain` + `y` = `rainy` (viel Regen haben)
