# --explanation--

예시는 `the "Student" class inherits properties and methods from the "Person" class`입니다.

- `Student`와 `Person`는 모두 `classes`입니다. `class`은 특정 데이터와 동작을 가진 객체를 생성하기 위한 청사진을 정의합니다.

- `Student`는 `Person`로부터 기능을 물려받아 공유된 특성을 다시 정의할 필요 없이 재사용합니다.

- `Properties`는 클래스 내부의 변수나 데이터로, `name`나 `age`처럼 객체를 설명합니다.

- `Methods`는 클래스 내에 정의된 함수로, `enroll()`나 `introduce()`처럼 객체가 할 수 있는 일을 설명합니다.
