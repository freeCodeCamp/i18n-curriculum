# --interactive--

響應式網頁設計是一種網頁開發方法，旨在創建能在從桌上型電腦到行動電話的廣泛裝置上，提供最佳瀏覽和互動體驗的網站。

響應式設計的核心原則是適應性－網站能根據所瀏覽裝置的螢幕大小和功能調整其版面配置和內容的能力。

響應式設計通常依賴三個主要組件－流體網格、彈性影像和媒體查詢：

- 流體網格使用相對單位如百分比，而非固定單位如像素，允許內容根據螢幕大小調整大小和重新排列。

- 彈性影像設定為在其包含元素內調整大小，確保它們在較小的螢幕上不會超出其容器的範圍。

- 而且，媒體查詢允許開發者根據裝置的特性，主要是視窗寬度，套用不同的樣式。你將在未來的課程中學習更多關於媒體查詢的內容。

響應式網頁設計與像 `CSS Grid` 和 `Flexbox` 這類工具之間的關係是共生的。雖然響應式設計是一種概念或方法，但 `CSS Grid` 和 `Flexbox` 是實作響應式設計時使過程更加簡單且高效的實用工具。

在之前的課程中你學會了如何使用 Flexbox，在未來的課程中你將學會如何使用 CSS Grid。但現在，這裡有一個關於 CSS Grid 的簡要介紹。

CSS Grid 是一個二維佈局系統，允許更複雜的排列。它非常適合用於建立整體頁面佈局以及較小的組件佈局。

以下是 CSS Grid 如何響應式使用的範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr;
  gap: 20px;
}

@media (min-width: 768px) {
  .grid-container {
    grid-template-columns: 1fr 1fr;
  }
}

@media (min-width: 1024px) {
  .grid-container {
    grid-template-columns: 1fr 1fr 1fr;
  }
}
```

:::

在此範例中，網格在小螢幕上以單一行開始，於中等大小螢幕切換為兩行，然後在較大螢幕切換為三行。這展示了 Grid 如何生成可響應的版面配置，以適應不同的螢幕尺寸。

Flexbox 和 Grid 相較於舊有的版面配置方法如浮動或基於表格的版面配置，提供了顯著的優勢。它們提供更多的彈性和控制元件，所需的程式碼更少，且設計時考量了響應式設計。它們讓開發者能夠相對輕鬆地創建複雜且響應式的版面配置，成為實作響應式網頁設計中不可或缺的工具。

值得注意的是，Flexbox 和 Grid 通常會在響應式設計中一起使用。Flexbox 通常用於組件和一維佈局，而 Grid 用於整體頁面結構和二維佈局。兩者之間的選擇通常取決於設計的具體佈局需求。

除了 Flexbox 和 Grid，其他 CSS 特性在響應式設計中也扮演重要角色。

例如，`calc()` 函式允許混合單位並執行計算，這在建立彈性版面配置時非常有用。

響應式影像是響應式網頁設計的另一個關鍵面向。HTML5 中的 `srcset` 屬性和 `picture` 元素允許根據裝置能力提供不同的影像檔案，確保使用者在較小螢幕或較低決議程序的裝置上不會下載不必要的大型影像檔案。

總結來說，響應式網頁設計是一種旨在為任何裝置創建運作良好網站的方法。

雖然這是一種設計理念，而非特定技術，但它大量依賴像媒體查詢這類的 CSS 特性，並且現代版面配置工具如 Flexbox 和 Grid 大大促進了這種設計。這些工具提供了創建真正響應式設計所需的彈性和控制元件，使網站能夠無縫適應日益多樣化的用來存取網路的裝置。
