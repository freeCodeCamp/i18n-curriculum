# --explanation--

`stuck on`という表現は、誰かがタスクを進められなかったり問題を解決できなかったりするときに使われます。行き詰まったり進展できないことを示唆します。例えば：

- `We were stuck on this code error for hours.` - コードのそのエラーが見つかるまで進めませんでした。

- `She got stuck on the final question of the exam.` - 彼女は最後の問題に長い時間を費やしました。

この会話では、リサが`stuck on`を使って、仕事のバグを解決するのに進展できなかったことを説明しています。
