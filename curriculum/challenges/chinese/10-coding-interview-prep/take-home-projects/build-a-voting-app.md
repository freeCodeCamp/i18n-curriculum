# --description--

Build a full stack JavaScript app that is functionally similar to this: <a href="https://voting-app.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://voting-app.freecodecamp.rocks/</a>. Use a site builder of your choice to complete the project.

以下是你应该为此项目实现的用户故事：

**用户故事：** 作为经过身份验证的用户，你可以保留我的调查，稍后再回来访问它们。

**用户故事：** 作为经过身份验证的用户，我可以与朋友分享我的调查。

**用户故事：** 作为经过身份验证的用户，我可以看到我的调查的汇总结果。

**用户故事：** 作为经过身份验证的用户，我可以删除我认为不再需要的调查。

**用户故事：** 作为经过身份验证的用户，我可以创建包含任意数量项目的投票。

**用户故事：** 作为未经身份验证或经过身份验证的用户，我可以查看并对每个人的民意调查进行投票。

**用户故事：** 作为未经身份验证或经过身份验证的用户，我可以以图表形式查看调查的结果。 （可以使用 Chart.js 或 Google Charts 来实现。）

**用户故事：** 作为认证用户，如果你不喜欢调查上的选项，你可以创建一个新选项。

完成本项目后，请将一个正常运行的 demo（项目演示）托管在可以公开访问的平台。 然后将 URL 提交到 Solution Link 字段中。 也可以将项目的源码链接提交到 GitHub Link 字段中。
