# --interactive--

Con la funzione `calc()` puoi eseguire calcoli direttamente all’interno dei tuoi fogli di stile per determinare dinamicamente i valori delle proprietà. Questo significa che puoi creare interfacce utente flessibili e responsivi calcolando le dimensioni in base alla dimensione della viewport o ad altri elementi.

`calc()` è una funzione CSS. Imparerai di più sulle funzioni quando inizierai a studiare JavaScript, ma in questa lezione imparerai le basi necessarie per capire come funziona `calc()`.

Una funzione è un blocco di codice che esegue un’attività specifica. Alcune funzioni sono già definite in CSS, quindi puoi usarle direttamente e passare loro i valori necessari per personalizzare come verranno eseguite le loro attività.

Nel mondo della programmazione, quando eseguiamo l’attività svolta da una funzione, diciamo che "chiamiamo" la funzione. I valori che passiamo alla funzione sono chiamati argomenti.

Come puoi vedere nel codice qui sotto, per chiamare una funzione scrivi il suo nome seguito dagli argomenti tra parentesi tonde, separati da virgole. Non ci deve essere spazio tra il nome della funzione e la parentesi di apertura:

```css
function(argument1, argument2, argument3)
```

Una funzione potrebbe aver bisogno di un solo valore per sapere cosa fare. In quel caso prende un solo argomento. È quello che succede con la funzione `calc()`. Prende un argomento perché deve sapere cosa calcolare.

Per questo, passi qualcosa chiamato espressione come argomento. Un’espressione è una combinazione di valori e operatori che produce un risultato.

Ecco come puoi chiamare la funzione `calc()`. Scrivi il nome calc, seguito dalle parentesi tonde, e all’interno delle parentesi scrivi l’espressione:

```css
calc(expression)
```

L’espressione viene valutata per calcolare il risultato finale. "Valutata" significa semplicemente che i valori e gli operatori vengono convertiti in un singolo valore dietro le quinte. Il risultato viene assegnato alla proprietà CSS dove viene effettuato il calcolo.

Puoi eseguire calcoli su valori che rappresentano lunghezza, angolo, tempo, percentuali, numeri e colori. Puoi anche combinare unità diverse come pixel, percentuali ed em.

Con i numeri, tutti i valori nell’espressione, chiamati anche operandi, devono avere le loro unità corrispondenti, come `px`, `em` e percentuale (`%`). A seconda dell’operatore, operandi diversi possono avere unità diverse.

Puoi usare gli operatori di addizione (`+`), sottrazione (`-`), moltiplicazione (`*`) e divisione (`/`) nell’espressione.

Se ci sono più operandi e operatori, `calc()` seguirà la regola standard della precedenza degli operatori. Puoi anche aggiungere parentesi per stabilire l’ordine delle operazioni se necessario.

Nell’esempio qui sotto, puoi vedere un `div` con il testo `Hello, World!`.

Usando il selettore di tipo CSS per selezionare il `div`, puoi definirne lo stile con testo bianco e sfondo blu scuro:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>Hello, World!</div>
```

```css
div {
  color: white;
  background-color: #1b1b32;
  width: calc(50% - 20px);
}
```

:::

La novità qui è che la larghezza viene calcolata dinamicamente. Nota come stiamo chiamando la funzione `calc()` e passando un’espressione come argomento. L’espressione ha due operandi con unità diverse e un operatore, l’operatore di sottrazione.

La percentuale è un’unità relativa. Il valore (`50%`) sarà determinato dalla larghezza del contenitore genitore. Poi `20px` viene sottratto dal valore. Il risultato di questa espressione determinerà la larghezza del `div`.

La larghezza del `div` è circa la metà della larghezza totale del suo contenitore, e se ridimensioni il contenitore genitore, la larghezza verrà ricalcolata automaticamente.

Questo è il vantaggio principale di usare `calc()`. Puoi determinare dinamicamente il valore di una proprietà CSS basandoti su diversi aspetti dell’applicazione o della viewport.

L’espressione può anche contenere funzioni CSS e variabili se hai bisogno di usarle nei tuoi calcoli. Imparerai di più sulle variabili CSS nelle prossime lezioni.

Ottimo. Ora che conosci le basi della funzione `calc()`, vediamo alcune delle sue buone pratiche.

Prima di tutto, devi circondare gli operatori di addizione (`+`) e sottrazione (`-`) con spazi bianchi.

Per esempio, l’espressione qui sotto non sarebbe valida perché l’operatore di sottrazione è immediatamente prima del secondo operando.

```css
calc(100% -30px)
```

L’operatore di sottrazione (`-`) deve essere circondato da spazi bianchi, così. Aggiungere gli spazi bianchi creerà un’espressione valida.

```css
calc(100% - 30px)
```

Questo non è necessario per gli operatori di moltiplicazione e divisione ma è altamente raccomandato.

Puoi anche annidare chiamate alla funzione `calc()` se devi eseguire calcoli e usare quei risultati in altri calcoli.

Inoltre, se usi il valore zero per rappresentare una lunghezza nell’espressione che passi alla funzione `calc()`, devi includere le unità. Per esempio, questa espressione non sarebbe valida:

```css
calc(100% - 0)
```

Dovresti aggiungere le unità, come px.

```css
calc(100% - 0px)
```

Devi anche sapere che attualmente, se usi gli operatori di moltiplicazione o divisione, uno degli operandi deve essere senza unità. Per l’operatore di divisione, in particolare l’operando a destra deve essere senza unità. Questa non sarebbe un’espressione valida perché entrambi gli operandi hanno unità (pixel). Uno degli operandi, 5 o 50, deve essere senza unità:

```css
calc(5px * 50px)
```

Dovresti omettere le unità in uno dei due. Entrambe queste alternative sarebbero valide:

```css
calc(5 * 50px)
calc(5px * 50)
```

E questo è un esempio con l’operatore di divisione. Questa non sarebbe un’espressione valida poiché entrambi hanno unità:

```css
calc(50% / 5%)
```

Dovresti rimuovere l’unità dall’operando a destra quando usi l’operatore di divisione:

```css
calc(50% / 5)
```

La funzione `calc()` può essere molto utile per te come sviluppatore web. Con questa funzione puoi impostare dinamicamente i valori delle proprietà per creare design flessibili e responsivi.
