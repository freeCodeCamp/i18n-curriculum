# --explanation--

`Is planned` は、すでに予定されている未来の出来事を表すために `Present Simple` 時制を使っています。これはトレーニングがいつ行われるか確実に示しており、すでに起こったことを示しているわけではありません。例えば：

`The meeting is planned for next Monday.` - これは会議が将来に予定されていることを示しています。
