# --interactive--

У попередньому уроці ви дізналися, як почати працювати з лінійними та радіальними градієнтами. У цьому уроці ми заглибимося у те, як працювати з CSS-властивостями, щоб створювати більш привабливі та складні дизайни.

Градієнти в CSS дозволяють створювати плавні переходи між двома або більше заданими кольорами. Їх часто використовують, щоб додати глибини, текстури та візуальної привабливості вебдизайнам без потреби у файлах зображень.

CSS підтримує два основні типи градієнтів: лінійні градієнти та радіальні градієнти.

Лінійні градієнти створюють поступове змішування кольорів уздовж прямої лінії. Ви можете контролювати напрямок цієї лінії та кольори, які використовуються. Ось базовий приклад лінійного градієнта:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

Це створює градієнт, який переходить від червоного зліва до синього справа. Параметр `to right` задає напрямок градієнта. Ви можете використовувати інші напрямки, як `to left`, `to top`, `to bottom`, або навіть вказувати кути, наприклад `45deg`.

Також можна використовувати кілька зупинок кольору в лінійному градієнті:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="multi-color-gradient"></div>
```

```css
.multi-color-gradient {
  background: linear-gradient(to right, red, yellow, blue);
  height: 40vh;
}
```

:::

Це створює градієнт, який переходить від червоного до жовтого, а потім до синього, рівномірно розподілених по елементу.

Радіальні градієнти, навпаки, створюють кругові або еліптичні градієнти, що розходяться від центральної точки. Ось приклад:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::

Це створює круговий градієнт, який починається з червоного в центрі і переходить до синього на краях. Ви також можете вказати форму та розмір радіальних градієнтів:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="custom-radial-gradient"></div>
```

```css
.custom-radial-gradient {
  background: radial-gradient(ellipse at top left, red, blue);
  height: 40vh;
}
```

:::

Це створює еліптичний градієнт, що починається з верхнього лівого кута елемента. І лінійні, і радіальні градієнти дозволяють задавати зупинки кольору у точних позиціях. Наприклад:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="precise-gradient"></div>
```

```css
.precise-gradient {
  background: linear-gradient(to right, red 20%, yellow 40%, blue 80%);
  height: 40vh;
}
```

:::

Це створює градієнт, де червоний займає перші `20%`, потім переходить у жовтий на `40%`, і нарешті в синій на `80%`.

Градієнти можна використовувати там, де ви б використали зображення в CSS. Це робить їх універсальними для створення фонів, кнопок або навіть текстових ефектів у поєднанні з іншими CSS-властивостями.

Пам’ятайте, що хоча градієнти можуть покращити візуальну привабливість вашого сайту, важливо забезпечити достатній контраст для доступності, особливо коли градієнти використовуються за текстом.
