# --explanation--

`Present Continuous` 时态用于描述说话时刻正在发生的动作。它由动词 `to be`（`am, is, are`）+ 动词原形 + `-ing` 组成。例如：

- `She is coding a new feature right now.` - 她此刻正在编码一个新建特色的进程。

- `They are testing the software.` - 他们正在进行软件测试。
