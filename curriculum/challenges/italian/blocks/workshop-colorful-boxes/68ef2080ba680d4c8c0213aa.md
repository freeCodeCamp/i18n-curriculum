# --description--

L'ultimo valore della proprietà `flex` imposta la proprietà `flex-basis`. Questa proprietà imposta la dimensione iniziale di un elemento flex prima che cresca o si riduca.

Infine, cambia l'ultimo valore in `flex` da `100px` a `150px` all'interno del tuo selettore `.box`.

# --hints--

Il tuo selettore `.box` dovrebbe avere la proprietà `flex` con un valore di `0 0 150px`.

```js
assert.equal(new __helpers.CSSHelp(document).getStyle('.box')?.getPropVal('flex'), '0 0 150px');
```
