# --explanation--

`Figure out` ist ein Phrasal Verb, das bedeutet, etwas Schwieriges zu lösen oder zu verstehen. Zum Beispiel:

`We need to figure out why the website isn't loading.` – Sie müssen den Grund verstehen, warum die Website nicht funktioniert.

Eine `firewall` ist eine Sicherheitsfunktion, die Computernetzwerke vor Hackern schützt. Sie funktioniert wie eine digitale Barriere, die sichere Daten hereinlässt und gefährlichen Datenverkehr blockiert. Zum Beispiel:

`Make sure the firewall is active before you connect to the internet.` – Stellen Sie sicher, dass das Sicherheitssystem eingeschaltet ist, bevor Sie das Internet nutzen.
