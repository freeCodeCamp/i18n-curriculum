# --description--

Ein `endpoint` in einer API ist ein bestimmter Ort, an dem Sie bestimmte Daten oder Dienste anfragen können. Zum Beispiel `The API endpoint for weather data retrieves the latest forecasts.`

Zu `offer` bedeutet bereitstellen oder verfügbar machen. Zum Beispiel `This service offers real-time updates.`

Eine `route` ist ein in der API definierter Pfad, der bestimmt, wie Anfragen verarbeitet werden. Zum Beispiel `The login route verifies user credentials.`
