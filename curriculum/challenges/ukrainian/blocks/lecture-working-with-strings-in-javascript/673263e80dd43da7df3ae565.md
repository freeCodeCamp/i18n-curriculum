# --interactive--

Працюючи з рядками в JavaScript, іноді потрібно знайти позицію певного підрядка всередині більшого рядка.

Підрядок — це послідовність символів, яка зустрічається всередині більшого рядка. Наприклад, у рядку `hello world` `hello` і `world` є підрядками.

Щоб знайти позицію підрядка всередині рядка, можна використати метод `indexOf()`.
Метод `indexOf()` у JavaScript дозволяє виконати пошук підрядка в рядку.

Якщо підрядок знайдено, `indexOf()` повертає індекс (або позицію) першого входження цього підрядка. Якщо підрядок не знайдено, `indexOf()` повертає `-1`, що означає, що пошук не увінчався успіхом.

Метод `indexOf()` приймає два аргументи: перший — це підрядок, який ви хочете знайти в більшому рядку, а другий — необов’язкова початкова позиція для пошуку. Якщо початкова позиція не вказана, пошук починається з початку рядка.

У цьому контексті **аргумент** — це значення, яке ви передаєте функції або методу під час виклику, щоб ця функція або метод могли виконати свою задачу з використанням конкретної інформації, яку ви надали. Ви дізнаєтеся більше про аргументи в майбутніх уроках.

Ось приклад використання методу `indexOf()` для пошуку позиції рядка `awesome`:

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("awesome!");
console.log(position); // 14
```

:::

У цьому прикладі слово `awesome` починається з індексу `14` у рядку `JavaScript is awesome!`, тому метод `indexOf()` повертає `14`.

Тепер подивимося, що станеться, якщо підрядок не буде знайдено:

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("fantastic");
console.log(position); // -1
```

:::

Оскільки слово `fantastic` не зустрічається в рядку, метод повертає `-1`.

Ви також можете вказати, з якої позиції починати пошук у рядку, передавши другий аргумент методу `indexOf()`. Ось приклад:

:::interactive_editor

```js
let sentence = "JavaScript is awesome, and JavaScript is powerful!";
let position = sentence.indexOf("JavaScript", 10);
console.log(position); // 27
```

:::

У цьому випадку пошук `JavaScript` починається після 10-го символу, і тому друге входження `JavaScript` знайдено на індексі `27`.

Важливо зазначити, що метод `indexOf()` чутливий до регістру.

У цьому прикладі наступне поверне `-1`, тому що велика літера `F` не знайдена в рядку `freeCodeCamp`.

:::interactive_editor

```js
console.log("freeCodeCamp".indexOf("F")) // -1
```

:::

Використання `indexOf()` може бути дуже корисним, коли потрібно перевірити, чи присутній підрядок у рядку, і визначити його позицію для подальших операцій.
