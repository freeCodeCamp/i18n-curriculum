## 出身地について話す

出身地を言うには、**`Soy` + `de` + [国名]** を使います。例えば：

- `Soy de El Salvador.`

- `Soy de Guatemala.`

誰かの出身地を尋ねるには、`¿De dónde eres?` と聞いてください。

<br>

## 国籍について話す

国籍を言うには、**`Soy` + [国籍]** を使います。

- `Soy salvadoreña.`

- `Soy guatemalteco.`

これまでに学んだ国籍の一部は以下の通りです：

| 国名 | 男性形 | 女性形 |
| :--- | :--- | :--- |
| **`El Salvador`** | `salvadoreño` | `salvadoreña` |
| **`Guatemala`** | `guatemalteco` | `guatemalteca` |
| **`Colombia`** | `colombiano` | `colombiana` |
| **`Argentina`** | `argentino` | `argentina` |
| **`Uruguay`** | `uruguayo` | `uruguaya` |
| **`Puerto Rico`** | `puertorriqueño` | `puertorriqueña` |
| **`Chile`** | `chileno` | `chilena` |
| **`Perú`** | `peruano` | `peruana` |

スペイン語では、国籍は通常、性別によって変わります。国籍が`-o`で終わる場合、女性形を作るにはこの最後の文字が通常`-a`に置き換えられます。

しかし、子音で終わる国籍は女性形を作るために`-a`を付け加えることが多いです。例えば、`España`（「スペイン」）出身の男性形は`español`ですが、女性形は`española`です。

`-e`や`-í`で終わる国籍は、男女で同じ形になることが多いです。例えば、`Estados Unidos`（「アメリカ合衆国」）出身の男性形は`estadounidense`で、女性形も`estadounidense`です。

<br>

## 職業について話す

職業を言うには、**`Soy` + [職業]** を使います。例えば：

- `Soy representante de recursos humanos.`

- `Soy analista junior.`

誰かの職業を尋ねるには、`¿A qué te dedicas?` と聞いてください。

<br>

## 年齢について話す

スペイン語で年齢を言うには、**`Tengo` + [数字] + `años`** を使います。例えば：

- `Tengo veintiún años.`

- `Tengo treinta y cinco años.`

誰かの年齢を尋ねるには、`¿Cuántos años tienes?` と聞いてください。

<br>

## 紹介時の丁寧な表現

スペイン語には重要な丁寧な表現が3つあります。

**`Mucho gusto`** は「はじめまして」を意味します。誰かに会えてうれしいという丁寧な言い方で、このフレーズは性別に関係ありません。

**`Encantada`** も「はじめまして」を意味しますが、より感情を込めて誰かに会えてうれしいことを表現します。話者の性別に合わせる必要があります。女性形は`Encantada`、男性形は`Encantado`です。

**`Igualmente`** は「こちらこそ」を意味します。相手の表現を繰り返さずに丁寧に返すときに使います。
