# --description--

Nell'ultimo esempio avevi due scelte `if` o `else`. Ma cosa succede se vuoi avere più di due scelte? Puoi usare l'istruzione `else if` per aggiungere più condizioni al nostro codice. L'istruzione `else if` ci permette di verificare più condizioni ed eseguire un blocco di codice quando la prima condizione è vera. Se la prima condizione è falsa, controllerà la condizione successiva e così via. Se nessuna delle condizioni è vera, verrà eseguito il blocco `else`.

Ecco un esempio dell'istruzione `else if`:

```javascript
let x = 5;

if (x > 10) {
  console.log("x is greater than 10");
} else if (x > 5) {
  console.log("x is greater than 5");
} else {
  console.log("x is less than or equal to 5");
}
```

In questo esempio, la variabile `x` viene confrontata con il numero `10` usando l'operatore `>`. Se `x` è maggiore di `10`, verrà eseguito il primo blocco di codice. Altrimenti, l'istruzione `else if` controllerà la condizione successiva. Se `x` è maggiore di `5`, verrà eseguito il secondo blocco di codice. Altrimenti, verrà eseguito il blocco `else`.
