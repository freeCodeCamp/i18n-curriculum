# --description--

Um `assignment operator` em programação é usado para atribuir um valor a uma variável.Em muitas linguagens de programação, incluindo Python, o `assignment operator` é representado pelo símbolo `=`.Ele instrui o computador a armazenar o valor do lado direito do `=` na variável nomeada do lado esquerdo.Por exemplo, `number = 10` atribui o valor `10` à variável `number`.
