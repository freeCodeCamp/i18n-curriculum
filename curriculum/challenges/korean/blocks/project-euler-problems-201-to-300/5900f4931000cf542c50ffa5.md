# --description--

양의 정수 $k$에 대해, $d(k)$를 $k$의 일반적인 십진법 표기에서 자릿수의 합으로 정의합니다. 따라서 $d(42) = 4 + 2 = 6$입니다.

양의 정수 $n$에 대해, $S(n)$을 다음 성질을 만족하는 $10^n$보다 작은 양의 정수 $k$의 개수로 정의합니다:

- $k$는 23으로 나누어떨어지고,
- $d(k) = 23$이다.

$S(9) = 263\,626$이고 $S(42) = 6\,377\,168\,878\,570\,056$임이 주어졌습니다.

$S(11^{12})$를 구하고, 답을 $\bmod 10^9$로 제출하세요.
