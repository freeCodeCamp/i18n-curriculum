# --description--

정수 $m$, $n$ ($0 ≤ n &lt; m$)에 대해, $L(m, n)$은 오른쪽 위의 $n×n$ 격자가 제거된 $m×m$ 격자입니다.

예를 들어, $L(5, 3)$은 다음과 같습니다:

<img alt="5x5 격자, 오른쪽 위에서 3x3 격자가 제거됨" src="https://cdn.freecodecamp.org/curriculum/project-euler/gnomon-numbering-1.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$L(m, n)$의 각 칸에 연속된 정수 1, 2, 3, ...을 다음 조건을 만족하도록 번호를 매기려 합니다: 각 칸의 숫자는 그 아래 칸과 왼쪽 칸의 숫자보다 작아야 합니다.

예를 들어, 다음은 $L(5, 3)$의 두 가지 유효한 번호 매기기 예입니다:

<img alt="L(5, 3)의 두 가지 유효한 번호 매기기" src="https://cdn.freecodecamp.org/curriculum/project-euler/gnomon-numbering-2.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$LC(m, n)$을 $L(m, n)$의 유효한 번호 매기기 개수라고 합시다. $LC(3, 0) = 42$, $LC(5, 3) = 250\,250$, $LC(6, 3) = 406\,029\,023\,400$, 그리고 $LC(10, 5)\bmod 76\,543\,217 = 61\,251\,715$임을 확인할 수 있습니다.

$LC(10\,000, 5\,000)\bmod 76\,543\,217$ 값을 구하세요.
