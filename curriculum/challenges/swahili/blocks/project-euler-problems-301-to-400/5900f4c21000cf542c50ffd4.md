# --description--

Kwa nambari kamili zilizo thabiti $a$, $b$, $c$, fafanua kitendakazi kichaa $F(n)$ kama ifuatavyo:

$$\begin{align}
  & F(n) = n - c \\;\text{ kwa wote } n > b \\\\
  & F(n) = F(a + F(a + F(a + F(a + n)))) \\;\text{ kwa wote } n ≤ b.
\end{align}$$

Pia, fafanua $S(a, b, c) = \displaystyle\sum_{n = 0}^b F(n)$.

Kwa mfano, ikiwa $a = 50$, $b = 2000$ na $c = 40$, basi $F(0) = 3240$ na $F(2000) = 2040$. Pia, $S(50, 2000, 40) = 5\\,204\\,240$.

Tafuta tarakimu 9 za mwisho za $S({21}^7, 7^{21}, {12}^7)$.
