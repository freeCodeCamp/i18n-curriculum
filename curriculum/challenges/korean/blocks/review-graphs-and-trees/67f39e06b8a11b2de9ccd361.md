## 그래프 개요

그래프는 엣지(연결선)로 연결된 노드(정점)들의 집합입니다. 각 노드는 여러 다른 노드와 연결되어 네트워크를 형성할 수 있습니다. 그래프의 다양한 유형은 다음과 같습니다:

- 방향 그래프: 엣지에 방향이 있으며(한 노드에서 다른 노드로), 보통 직선과 화살표로 나타냅니다.
- 무방향 그래프: 엣지에 방향이 없으며, 단순한 선으로 나타냅니다.
- 정점 그래프: 각 노드가 레이블이나 식별자와 연결됩니다.
- 순환 그래프: 사이클(시작 노드와 끝 노드가 같은 경로)을 포함합니다.
- 비순환 그래프 (DAG): 사이클을 포함하지 않습니다.
- 엣지 레이블 그래프: 각 엣지에 보통 해당 엣지 옆에 그려진 레이블이 있습니다.
- 가중치 그래프: 엣지에 산술 연산에 사용할 수 있는 가중치(값)가 할당되어 있습니다.
- 비연결 그래프: 두 개 이상의 노드가 어떤 엣지로도 연결되어 있지 않습니다.

그래프는 지도, 네트워크, 추천 시스템, 의존성 해결 등 다양한 애플리케이션에서 사용됩니다.

## 그래프 순회

그래프의 모든 노드를 방문하는 작업입니다. 주요 알고리즘 두 가지는 다음과 같습니다:

- **너비 우선 탐색 (BFS)**
  - 큐를 사용합니다.
  - 레벨별로 탐색합니다.
  - 가중치 없는 그래프에서 최단 경로를 찾습니다.

- **깊이 우선 탐색 (DFS)**
  - 스택(또는 재귀)을 사용합니다.
  - 한 갈래를 완전히 탐색한 후 되돌아갑니다.
  - 사이클 탐지와 경로 찾기에 유용합니다.

## 그래프 표현 방법

그래프는 두 가지 주요 방식으로 나타낼 수 있습니다:

- **인접 리스트**
  - 각 노드가 이웃 노드들의 리스트를 가집니다.
  - 희소 그래프에 공간 효율적입니다.
  - 이웃 노드를 순회하기 쉽습니다.

- **인접 행렬**
  - 행과 열이 노드를 나타내는 2차원 배열입니다.
  - 큰 그래프에서는 공간을 많이 차지합니다.
  - 두 노드 사이에 엣지가 있는지 빠르게 확인할 수 있습니다.

## 트리

트리는 비순환이고 연결된 특별한 유형의 그래프입니다. 주요 속성은 다음과 같습니다:

- 루프나 사이클(시작 노드와 끝 노드가 같은 경로)이 없습니다.
- 연결되어 있어 모든 노드가 다른 모든 노드에서 도달 가능합니다.

### 일반적인 트리 유형

가장 일반적인 트리 유형은 다음과 같습니다:

- 이진 트리
  - 각 노드는 최대 두 개의 자식 노드, 왼쪽 자식과 오른쪽 자식을 가집니다.

- 이진 탐색 트리 (BST)
  - 모든 왼쪽 자식 노드는 부모 노드보다 작고, 모든 오른쪽 자식 노드는 부모 노드보다 큽니다.

## 트라이

접두사 트리라고도 하며, 각 노드가 문자를 나타내는 문자열 집합을 저장하는 데 사용됩니다.

공유하는 접두사는 한 번만 저장되어 자동 완성이나 맞춤법 검사 같은 작업에 효율적입니다.

검색과 삽입 연산의 시간 복잡도는 문자열 길이 L에 대해 O(L)입니다.

## 우선순위 큐

우선순위 큐는 각 요소가 우선순위를 가지는 추상 자료형입니다.

큐와 스택은 삽입 순서만 고려하지만, 우선순위 큐는 요소의 우선순위를 고려합니다.

표준 큐는 FIFO(선입선출), 스택은 LIFO(후입선출)를 따릅니다. 그러나 우선순위 큐에서는 우선순위가 높은 요소가 삽입 순서와 상관없이 먼저 처리됩니다.

## 힙

힙은 힙 속성이라는 매우 특정한 속성을 가진 특수한 트리 기반 자료 구조입니다.

힙 속성은 부모 노드와 자식 노드 간의 관계를 결정합니다. 힙에는 두 가지 유형이 있습니다:

- 최대 힙
  - 각 부모 노드의 값이 자식 노드 값보다 크거나 같습니다.
  - 가장 큰 요소가 루트에 있습니다.

- 최소 힙
  - 각 부모 노드의 값이 자식 노드 값보다 작거나 같습니다.
  - 가장 작은 요소가 루트에 있습니다.

### Python `heapq` 모듈 예제

```py
import heapq

# Create empty heap
my_heap = []

# Insert elements
heapq.heappush(my_heap, 9)
heapq.heappush(my_heap, 3)
heapq.heappush(my_heap, 5)
print(my_heap) # [3, 9, 5]

# Remove smallest element
print(heapq.heappop(my_heap))  # 3
print(my_heap) # [5, 9]

# Push + Pop in one step
print(heapq.heappushpop(my_heap, 7)) # 5
print(my_heap) # [7, 9]

# Transform list into heap
nums = [5, 7, 3, 1]
heapq.heapify(nums)
```

### 우선순위 사용하기

```py
my_heap = []
heapq.heappush(my_heap, (3, "A"))
heapq.heappush(my_heap, (2, "B"))
heapq.heappush(my_heap, (1, "C"))

# Removes lowest number = highest priority
print(heapq.heappop(my_heap))  # (1, "C")
```
