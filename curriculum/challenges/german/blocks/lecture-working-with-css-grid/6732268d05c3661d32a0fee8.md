# --interactive--

In den vorherigen Lektionen haben wir mit der `grid-template-columns`-Eigenschaft gearbeitet und den Wert auf einige Bruchteileinheiten gesetzt.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

Obwohl der folgende Code völlig gültig ist, gibt es eine einfachere Möglichkeit, einen Abschnitt oder alle Ihre Track-Listings zu wiederholen.

Die `repeat()`-Funktion wird verwendet, um einen Abschnitt oder alle Tracks für Spalten oder Zeilen zu wiederholen. Diese Funktion nimmt eine Wiederholungsanzahl und die Tracks, die Sie wiederholen möchten, entgegen.

Hier ist eine überarbeitete Version des früheren Beispiels mit der repeat-Funktion:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

Im Browser ändert sich an den Styles nichts, aber dies ist eine prägnantere Art, wiederholte Werte für die Spalten zu schreiben.

Die `repeat()`-Funktion akzeptiert jedes gültige Muster, das Sie für Zeilen oder Spalten verwenden können.

Hier ist ein Beispiel, das die `repeat()`-Funktion verwendet, um die erste und dritte Spalte auf 20 Pixel und die zweite und vierte Spalte auf eine Bruchteileinheit zu setzen.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(2, 20px 1fr);
  column-gap: 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

Manchmal entscheiden Sie sich vielleicht dafür, jeden einzelnen Wert auszuschreiben, anstatt die `repeat()`-Funktion zu verwenden. Aber es gibt Zeiten, in denen diese Funktion sehr nützlich ist, besonders wenn Sie ein bestimmtes Muster für ein Track-Listing wiederholen möchten.
