# --explanation--

`Resolve` означає розв’язати або виправити проблему чи ситуацію. Це передбачає вжиття заходів для вирішення та врегулювання проблеми. Наприклад:

`The team worked hard to resolve the customer complaints.` — це вказує, що команда зробила кроки для розв’язання проблем, які підняли клієнти.

`Don't` у питальних тегах використовується після позитивного головного речення, щоб ввічливо або розмовно отримати підтвердження або перевірити згоду. Наприклад:

`We need to leave now, don't we?` — головне речення позитивне, а питальний тег (`don't we?`) використовується для підтвердження або перевірки згоди щодо терміновості виходу.
