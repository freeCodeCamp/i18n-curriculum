# --explanation--

`Might` 用來表達可能性。它表示某事是可能的，但不確定。例如：

- `I might go to the party if I finish my work。`－我可能會去派對，但我不確定。

- `She might call you later.`－她可能稍後會呼叫你，但不一定。
