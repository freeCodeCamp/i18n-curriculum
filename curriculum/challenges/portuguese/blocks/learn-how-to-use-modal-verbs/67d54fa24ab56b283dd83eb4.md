# --explanation--

Tanto `must` quanto `have to` expressam obrigação ou necessidade, mas há uma pequena diferença no uso deles.

`Must` é usado para obrigação interna forte ou regras oficiais. Soa frequentemente mais formal ou autoritário. Por exemplo:

`All sensitive data must be encrypted.` - Uma regra ou requisito rigoroso.

`Have to` é usado para obrigação externa, frequentemente baseada em regras ou circunstâncias em vez de autoridade pessoal. Por exemplo:

`You have to ensure that the encryption keys are stored securely.` - Uma responsabilidade devido às políticas de segurança.
