# --description--

如你之前所見，函式（程式）可以使用 `return` 關鍵字傳回一個值。`return` 關鍵字用於從函式（程式）傳回一個值。當使用 `return` 關鍵字時，函式（程式）將停止執行並傳回 `return` 關鍵字後指定的值。

```js
function add(a, b) {
  return a + b
}

console.log(add(2, 3)); // Output: 5
```

但是如果在函式結束之前使用了 `return` 關鍵字會發生什麼事？  
為了回答這個問題，請考慮以下範例：

```js
function add(a, b) {
  if(a > 2){
    return b;
  }

  return a + b;
}

console.log(add(3, 7)); // Output: 7
``` 

在上述範例中，函式 `add` 有一個條件述語，用來檢查 `a` 的值是否大於 `2`。如果條件成立，函式將傳回 `b` 的值並停止執行。如果條件不成立，函式將傳回 `a` 和 `b` 的總和。
