# --description--

In Python, classi e oggetti lavorano insieme per organizzare e gestire i dati. Costruisci una classe per definire un comportamento condiviso, poi crei oggetti che usano quei comportamenti.

In altre parole, una classe è come un modello o uno schema che usi per creare oggetti.

Vediamo cosa sono le classi e come usarle per creare oggetti.

Per creare una classe, usi la parola chiave `class` seguita dal nome della classe e due punti. Poi all’interno della classe puoi aggiungere un inizializzatore, insieme a eventuali attributi e metodi.

Gli attributi sono come variabili all’interno di una classe e servono per archiviare dati. I metodi sono funzioni definite all’interno di una classe e sono le azioni che gli oggetti creati con una classe possono eseguire.

Ecco la sintassi base di una classe:

```python
class ClassName:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def sample_method(self):               
        print(self.name.upper())
```

* `class ClassName` è composta dalla parola chiave `class` per creare una classe, seguita dal nome della classe, qui chiamata `ClassName`. In Python è comune usare la convenzione **PascalCase** per nominare le classi.
    
* `def __init__(self, name, age)` è il metodo speciale chiamato automaticamente quando viene creato un nuovo oggetto. Inizializza gli attributi degli oggetti che saranno creati con la classe.
    
    Inoltre, il primo parametro di `__init__` è sempre un riferimento all’oggetto specifico che viene creato o usato. Per convenzione, questo parametro si chiama `self`, ma tecnicamente puoi usare qualsiasi nome. `self` ti permette di consultare gli attributi e i metodi dell’oggetto stesso.
    
* `self.name = name` e `self.age = age` sono gli attributi che gli oggetti avranno.
    
* `def sample_method(self):` è il metodo che ogni oggetto creato può chiamare.
    
* `print(self.name.upper())` è ciò che farà il metodo `sample_method`, in questo caso stampa il nome in maiuscolo.
    

Se ti sembra tanto, non preoccuparti. Diamo un’occhiata a un esempio simile di una classe `Dog` e a come puoi creare oggetti da essa:

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name.upper()} says woof woof!")
```

Con questa classe `Dog` puoi creare un oggetto. Ecco la sintassi base per creare oggetti da una classe:

```python
object_1 = ClassName(attribute_1, attribute_2)
object_2 = ClassName(attribute_1, attribute_2)
```

Puoi anche chiamare uno qualsiasi dei metodi definiti nella classe da ogni oggetto:

```python
object_1.method_name()
object_2.method_name()
```

Ora creiamo due cani usando la classe `Dog` come modello:

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name.upper()} says woof woof! I'm {self.age} years old!")

dog_1 = Dog("Jack", 3)
dog_2 = Dog("Thatcher", 5)

# Call the bark method
dog_1.bark()  # JACK says woof woof! I'm 3 years old!
dog_2.bark()  # THATCHER says woof woof! I'm 5 years old!
```

Come vedi, creiamo due oggetti cane usando la classe `Dog`. Quando inizializzi `dog_1`, vengono passati la stringa `Jack` e il numero `3`, che impostano rispettivamente gli attributi `name` e `age` per quell’istanza. E `dog_2` viene inizializzato con la stringa `Thatcher` e il numero `5` come rispettivi `name` e `age`.

Poi, quando chiami il metodo `.bark()` su `dog_1` e `dog_2`, puoi vedere come i due output differiscono e usano gli attributi unici `name` e `age` che hai passato creando ogni oggetto.

In sintesi, la differenza tra una classe e un oggetto è che una classe è il modello o lo schema, e un oggetto è ciò che viene creato usando quel modello.

Inoltre, una classe definisce quali dati e comportamenti l’oggetto dovrebbe avere, e un oggetto contiene i dati reali e usa quel comportamento. Scrivi una classe una volta sola e puoi creare molti oggetti da essa, ognuno con dati diversi.
