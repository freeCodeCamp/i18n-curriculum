# --interactive--

CSS 플렉스박스는 컨테이너 안에서 요소들을 행과 열로 배치할 수 있는 1차원 레이아웃 모델입니다. 요소들의 순서와 방향도 제어할 수 있습니다. 웹 개발자들은 다양한 화면 크기와 방향에 맞게 반응형 웹사이트와 웹 애플리케이션을 만들 때 플렉스박스를 사용합니다. 플렉스박스를 1차원 레이아웃 모델이라고 부르는 이유는 한 번에 하나의 축을 따라 요소들을 배치하는 데 집중하기 때문입니다. 이 축은 가로 또는 세로가 될 수 있습니다.

플렉스박스를 다루기 전에 알아야 할 두 가지 핵심 개념이 있습니다: 플렉스 컨테이너와 플렉스 항목입니다.

플렉스 컨테이너는 플렉스 레이아웃을 가진 HTML 요소입니다. 플렉스 컨테이너 안에서 요소들을 다양한 방식으로 배치하고 정렬할 수 있습니다. HTML 요소를 플렉스 컨테이너로 만들려면 CSS 스타일에 `display: flex`를 추가해야 합니다.

플렉스 항목은 플렉스 컨테이너의 직계 자식 요소입니다. 이 요소들은 플렉스 컨테이너의 속성에 따라 배치되고 정렬됩니다. 또한 사용 가능한 공간에 맞게 축소하거나 확장할 수 있습니다.

다음은 `main` 컨테이너와 세 개의 자식 `div` 요소가 있는 예시입니다:

```html
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

CSS 스타일시트에서 이 `width` 요소들의 `height`, `background-color`, `div`만 설정하면, 컨테이너가 기본적으로 플렉스가 아니기 때문에 각 자식 요소가 각자의 행에 배치됩니다.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

하지만 `display: flex` 컨테이너에 `main`를 추가하면 `div` 요소들이 같은 행에 맞게 재배치되고 필요하면 축소됩니다(미리보기를 보려면 인터랙티브 에디터를 활성화하세요):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

기본적으로 플렉스 컨테이너는 블록 레벨 요소이므로, 컨테이너 자체는 다른 요소 및 컨테이너와 비교해 각자의 행에 위치합니다.

플렉스 컨테이너와 플렉스 항목에 대해 더 알게 되었으니, 플렉스 속성도 알아야 합니다. 이 속성들은 플렉스 항목이 플렉스 컨테이너 내에서 어떻게 배치되고 크기가 조정되며 분배되는지를 결정합니다. 가장 자주 쓰이는 속성으로는 `flex-direction`, `justify-content`, `align-items`, `flex-wrap`가 있습니다.

좋습니다. 이제 플렉스 모델에 대해 조금 이야기해 보겠습니다. 이 모델은 플렉스 항목들이 플렉스 컨테이너 내에서 어떻게 배치되는지를 정의합니다. 모든 플렉스 컨테이너에는 두 개의 축이 있습니다:

- 주축
- 교차 축

이 축들의 방향이 플렉스 항목의 레이아웃과 분배에 영향을 주는 다양한 속성들의 작용 방식을 결정합니다. 기본적으로 플렉스 컨테이너의 주축은 가로 방향이고 교차 축은 세로 방향입니다. 플렉스 항목들은 주축 방향으로 배치됩니다. 교차 축은 주축에 수직입니다.

`flex-direction` 속성은 주축의 방향을 설정합니다. `flex-direction`의 기본값은 `row`로, 모든 플렉스 항목을 브라우저 기본 언어 방향(왼쪽에서 오른쪽 또는 오른쪽에서 왼쪽)으로 같은 행에 배치합니다:

```css
flex-direction: row; /* Default */
```

행의 항목 순서를 반대로 하려면 `flex-direction: row-reverse`를 사용할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: row-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

이렇게 하면 플렉스 항목의 순서가 반대로 바뀝니다. 플렉스 항목을 세로로 정렬하려면 플렉스 컨테이너에 `flex-direction`를 `column`로 설정하면 됩니다. 이렇게 하면 주축 방향이 바뀝니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

이제 `div` 요소들이 주축이 세로이고 교차 축이 가로이기 때문에 세로로 정렬됩니다.

`flex-direction: column-reverse`를 사용하면 플렉스 항목의 세로 순서도 반대로 할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

CSS 플렉스박스는 컨테이너 내에서 요소들을 유연하고 효율적으로 배치할 수 있는 강력한 레이아웃 모델입니다. 플렉스 컨테이너, 플렉스 항목, 그리고 다양한 플렉스 속성의 핵심 개념을 이해하면, 다양한 화면 크기와 방향에 맞게 동적이고 반응형인 웹사이트를 만들 수 있습니다.
