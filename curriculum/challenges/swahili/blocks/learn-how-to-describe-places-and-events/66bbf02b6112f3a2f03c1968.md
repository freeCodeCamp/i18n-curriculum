# --explanation--

`It` hutumika kubadilisha nomino ambayo tayari imetajwa, kusaidia kuepuka kurudia katika sentensi. Kwa mfano:

- `I bought a new laptop. It works really well.` - Hapa, `it` hubadilisha `laptop` ili kuepuka kurudia neno hilo.

Katika `The Live Preview isn't showing in Visual Studio Code. Isn't it supposed to support that feature?`, `it` inarejelea `Visual Studio Code` kwa sababu Tom anazungumzia programu na uwezo wake wa kuunga mkono kipengele.

`It` hubadilisha `Visual Studio Code` ili kuepuka kusema tena jina la programu.

Kwa kutambua kile kinachotarajiwa kuunga mkono kipengele, unaweza kubaini kuwa `it` inarejelea programu yenyewe.
