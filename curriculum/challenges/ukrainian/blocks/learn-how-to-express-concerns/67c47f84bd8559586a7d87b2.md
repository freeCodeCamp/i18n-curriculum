# --explanation--

`To work hard` означає докладати багато зусиль. Зверніть увагу, що прислівник для `hard` у цьому випадку також `hard`, а не `hardly`. `To hardly work` означає майже не працювати, докладати мало або зовсім не докладати зусиль, що є протилежним тому, що каже Марія про команду. Наприклад:

- `She hardly trained for the marathon.` - Це означає, що вона не доклала жодних зусиль, щоб підготуватися до марафону.

- `She trained hard for the marathon.` - Це означає, що вона доклала багато зусиль для підготовки.
