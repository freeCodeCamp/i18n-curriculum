# --description--

Рядок — це послідовність символів, оточена або одинарними, або подвійними лапками. У деяких мовах програмування символи в одинарних лапках обробляються інакше, ніж символи в подвійних лапках, але в Python вони обробляються однаково. Тож ви можете використовувати будь-які з них при роботі з рядками. Ось кілька прикладів рядків:

```python
my_str_1 = 'Hello'
my_str_2 = "World"
```

Якщо вам потрібен багаторядковий рядок, ви можете використати потрійні подвійні лапки або потрійні одинарні лапки:

```python
my_str_3 = """Multiline
string"""
my_str_4 = '''Another
multiline
string'''
```

Якщо ваш рядок містить або одинарні, або подвійні лапки, у вас є два варіанти:

- Використати протилежний тип лапок. Тобто, якщо ваш рядок містить одинарні лапки, обгорніть рядок подвійними лапками, і навпаки:
    

```python
msg = "It's a sunny day"
quote = 'She said, "Hello World!"'
```

- Екранувати одинарну або подвійні лапки в рядку за допомогою зворотної скісної риски (`\`). Цим способом ви можете використовувати або одинарні, або подвійні лапки для обгортання самого рядка:
    

```python
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

Іноді потрібно перевірити, чи містить рядок один або кілька символів. Для цього Python надає оператор `in`, який повертає булеве значення, що вказує, чи існує символ або символи в рядку.

Ось кілька прикладів:

```python
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

Тепер подивимося, як отримати довжину рядка і працювати з окремими символами в рядку, процес, який називається **індексацією**. Щоб отримати довжину рядка, ви можете використати вбудовану функцію `len()`. Ось приклад:

```python
my_str = 'Hello world'
print(len(my_str))  # 11
```

Кожен символ у рядку має позицію, яку називають індексом. Індекс починається з нуля, тобто індекс першого символу рядка — `0`, індекс другого символу — `1`, і так далі. Щоб отримати доступ до символу за його індексом, використовуйте квадратні дужки (`[]`) з індексом символу всередині. Ось кілька прикладів:

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
```

Дозволена також від’ємна індексація, тож ви можете отримати останній символ будь-якого рядка за допомогою `-1`, передостанній символ — `-2`, і так далі:

```python
my_str = 'Hello world'
print(my_str[-1])  # d
print(my_str[-2]) # l
```

Багато інших мов програмування поділяють типи даних на примітивні та референсні. Примітивні типи прості й незмінні, тобто їх не можна змінити після оголошення. Референсні типи можуть містити кілька значень і бути або змінними, або незмінними. Але Python не проводить чітку межу між цими двома групами. Замість цього всі дані розглядаються як об’єкти, і деякі об’єкти незмінні, а інші — змінні.

Незмінні типи даних не можна змінювати або модифікувати після оголошення. Ви можете призначити їхнім змінним нове значення, що називається переназначенням, але не можете змінити сам об’єкт, додаючи, видаляючи або замінюючи будь-які його елементи.

Рядки — це незмінні типи даних у Python. Це означає, що ви можете переназначити змінній інший рядок:

```python
greeting = 'hi'
greeting = 'hello'
print(greeting) # hello
```

Але пряме модифікування рядка не дозволяється:

```python
greeting = 'hi'
greeting[0] = 'H' # TypeError: 'str' object does not support item assignment
```

Прикладами інших незмінних типів даних у Python є ціле число, число з плаваючою крапкою, булевий, кортеж і діапазон. Ви познайомитеся з кожним із цих типів у наступних уроках.
