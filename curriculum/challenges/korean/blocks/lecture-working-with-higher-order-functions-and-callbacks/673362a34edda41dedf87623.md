# --interactive--

`map` 메서드는 JavaScript에서 배열을 대상으로 작동하는 강력하고 널리 사용되는 함수입니다. 이 메서드는 주어진 함수를 원래 배열의 각 요소에 적용하여 새로운 배열을 만드는 데 설계되었습니다.

이 메서드는 원래 배열을 수정하지 않고, 대신 각 요소에 적용된 함수의 결과를 담은 새 배열을 반환합니다.

다음은 숫자 배열에 `map` 메서드를 사용하는 예시입니다:

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map((num) => num * 2);
console.log(numbers); // [1, 2, 3, 4, 5]
console.log(doubled); // [2, 4, 6, 8, 10]
```

:::

각 숫자를 두 배로 만드는 새 배열을 만들기 위해 `map` 메서드를 사용하고 있습니다. `map` 메서드는 콜백 함수를 매개변수로 받아 배열의 모든 요소에 대해 해당 함수가 호출됩니다.

이 경우 배열의 각 숫자는 `2` 곱해집니다. 결과는 `2,4,6,8,10` 숫자로 이루어진 새 배열이 됩니다.

콜백 함수는 최대 세 개의 인수를 받을 수 있습니다.

첫 번째 인수는 현재 처리 중인 요소입니다.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element) => {
  console.log("Element:", element);
  return element * 2;
});
```

:::

두 번째 인수는 현재 처리 중인 요소의 인덱스입니다.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index) => {
  console.log("Element:", element);
  console.log("Index:", index);
  return element * 2;
});
```

:::

세 번째 인수는 `map` 메서드가 호출되는 배열입니다.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index, array) => {
  console.log("Element:", element);
  console.log("Index:", index);
  console.log("Array:", array);
  return element * 2;
});
```

:::

`map` 메서드를 이해하고 효과적으로 사용하면 JavaScript에서 배열을 다루는 능력이 크게 향상됩니다. 앞으로의 수업에서 `map`의 더 고급 활용법을 깊이 탐구하며, 동적이고 효율적인 프로그램을 만드는 데 강력한 도구가 될 수 있음을 살펴보겠습니다.
