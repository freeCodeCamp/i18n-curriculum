# --description--

Die `align-items`-Eigenschaft ist ähnlich wie `justify-content`. Erinnern Sie sich, dass die `justify-content`-Eigenschaft Flex-Elemente entlang der Hauptachse ausrichtete. Bei Reihen ist die Hauptachse eine horizontale Linie, bei Spalten eine vertikale Linie.

Flex-Container haben außerdem eine **Querachse**, die der Hauptachse entgegengesetzt ist. Bei Reihen ist die Querachse vertikal, bei Spalten horizontal.

CSS bietet die `align-items`-Eigenschaft, um Flex-Elemente entlang der Querachse auszurichten. Bei einer Reihe gibt sie an, wie CSS die Elemente in der gesamten Reihe innerhalb des Containers nach oben oder unten schiebt. Bei einer Spalte gibt sie an, wie alle Elemente innerhalb des Containers nach links oder rechts geschoben werden.

Die verschiedenen Werte, die für `align-items` verfügbar sind, umfassen:

<ul><li><code>flex-start</code>: richtet die Elemente am Anfang des Flex-Containers aus. Bei Reihen richtet dies die Elemente oben im Container aus. Bei Spalten richtet es die Elemente links im Container aus.</li><li><code>flex-end</code>: richtet die Elemente am Ende des Flex-Containers aus. Bei Reihen richtet dies die Elemente unten im Container aus. Bei Spalten richtet es die Elemente rechts im Container aus.</li><li><code>center</code>: richtet die Elemente zentriert aus. Bei Reihen richtet dies die Elemente vertikal aus (gleicher Abstand ober- und unterhalb der Elemente). Bei Spalten richtet es sie horizontal aus (gleicher Abstand links und rechts der Elemente).</li><li><code>stretch</code>: streckt die Elemente, um den Flex-Container auszufüllen. Zum Beispiel werden Reihen-Elemente gestreckt, um den Flex-Container von oben bis unten auszufüllen. Dies ist der Standardwert, wenn kein Wert für <code>align-items</code> angegeben ist.</li><li><code>baseline</code>: richtet die Elemente an ihren Grundlinien aus. Die Grundlinie ist ein Textkonzept, denken Sie daran als die Linie, auf der die Buchstaben sitzen.</li></ul>

# --instructions--

Ein Beispiel zeigt diese Eigenschaft in Aktion. Fügen Sie die CSS-Eigenschaft `align-items` zum `#box-container`-Element hinzu und geben Sie ihr den Wert `center`.

**Bonus**  
Probieren Sie die anderen Optionen für die `align-items`-Eigenschaft im Code-Editor aus, um deren Unterschiede zu sehen. Beachten Sie jedoch, dass nur der Wert `center` diese Aufgabe besteht.

# --hints--

Das `#box-container`-Element sollte eine `align-items`-Eigenschaft mit dem Wert `center` haben.

```js
const boxContainer = document.querySelector('#box-container');
const alignment = window.getComputedStyle(boxContainer)['align-items'];
assert.strictEqual(alignment, 'center');
```
