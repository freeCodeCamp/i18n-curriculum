# --explanation--

Ángela dice `¿Y tu profesión?`, una breve pregunta de seguimiento. `Y` conecta esta pregunta con el tema anterior.

Aunque ella no dice `¿Cuál es...?`, la pregunta completa se entiende por el contexto:

`¿Cuál es tu profesión?`

En conversación, el español a menudo omite palabras repetidas cuando el significado es claro.
