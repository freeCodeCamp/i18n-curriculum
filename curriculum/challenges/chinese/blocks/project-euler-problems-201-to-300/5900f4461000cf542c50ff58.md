# --description--

一个具有 $k$ 个（十进制）数字的正整数称为平衡数，如果它的前 $⌈\frac{k}{2}⌉$ 个数字的和与它的后 $⌈\frac{k}{2}⌉$ 个数字的和相等，其中 $⌈x⌉$，读作 $x$ 的上取整，是大于或等于 $x$ 的最小整数，因此 $⌈π⌉ = 4$ 且 $⌈5⌉ = 5$。

例如，所有回文都是平衡的，13722 也是如此。

令 $T(n)$ 为所有小于 $10^n$ 的平衡数之和。

因此：$T(1) = 45$，$T(2) = 540$ 和 $T(5) = 334\\,795\\,890$。

查找 $T(47)\\,mod\\,3^{15}$
