# --description--

`Loops` nella programmazione indicano istruzioni che si ripetono finché non viene soddisfatta una certa condizione. Vengono usati per eseguire lo stesso codice più volte.

`To approach` significa affrontare qualcosa o iniziare a considerare un problema in un modo particolare. Nella risoluzione di problemi, riguarda il metodo o la strategia usata per affrontare una sfida.
