# --description--

Ukizingatia safu mbili za data, tengeneza kitu cha hash ambapo vipengele kutoka kwenye safu moja (funguo) vinaunganishwa na vipengele vya nyingine (thamani).

Mfano:

```js
const firstArr = [1, 2, 3];
const secondArr = ["a", "b", "c"];
const hashObj = arrToObj(firstArr, secondArr);
```

Kitu cha hash kilichotengenezwa kutoka kwa safu zilizo hapo juu kitakuwa `{1: "a", 2: "b", 3: "c"}`.

# --solutions--

```js
function arrToObj (keys, vals) {
  return keys.reduce((map, key, index) => {
    map[key] = vals[index];
    return map;
  }, {});
}
```
