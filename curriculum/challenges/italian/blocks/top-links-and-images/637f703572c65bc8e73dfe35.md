Per fare un po’ di pratica con link e immagini durante questa lezione ti serve un progetto HTML su cui lavorare.

- Crea una nuova directory chiamata `odin-links-and-images`.

- All’interno di quella directory, crea un nuovo file chiamato `index.html`.

- Compila il solito codice boilerplate HTML.

- Infine, aggiungi il seguente `h1` al `body`: `<h1>Homepage</h1>`

## Elementi àncora
Per creare un collegamento in HTML, usi l’elemento àncora. Un elemento àncora si definisce racchiudendo il testo o un altro elemento HTML che vuoi trasformare in collegamento con un tag `<a>`.

Aggiungi quanto segue al `body` della pagina `index.html` che hai creato e aprila nel browser:

```html
<a>click me</a>
```

Avrai notato che cliccando questo collegamento non succede nulla. Questo perché un tag àncora da solo non sa dove vuoi collegarti. Devi indicargli una destinazione. Lo fai usando un attributo HTML.

Un attributo HTML fornisce informazioni aggiuntive a un elemento HTML e va sempre nel tag di apertura dell’elemento. Un attributo è solitamente composto da due parti: un nome e un valore; tuttavia, non tutti gli attributi richiedono un valore. Nel tuo caso, devi aggiungere un attributo `href` (riferimento ipertestuale) al tag di apertura dell’àncora. Il valore dell’attributo `href` è la destinazione a cui vuoi che il collegamento punti.

Aggiungi il seguente attributo `href` all’elemento àncora che hai creato prima e prova a cliccarlo di nuovo, non dimenticare di aggiornare il browser così si applicano le nuove modifiche.

```html
<a href="https://www.theodinproject.com/about">click me</a>
```

Per impostazione predefinita, qualsiasi testo racchiuso in un tag àncora senza un attributo `href` apparirà come testo normale. Se l’attributo `href` è presente, il browser darà al testo un colore blu e lo sottolineerà per indicare che è un collegamento.

Vale la pena notare che puoi usare i tag àncora per collegarti a qualsiasi tipo di risorsa su internet, non solo ad altri documenti HTML. Puoi collegarti a video, file pdf, immagini e così via, ma per la maggior parte collegherai ad altri documenti HTML.
