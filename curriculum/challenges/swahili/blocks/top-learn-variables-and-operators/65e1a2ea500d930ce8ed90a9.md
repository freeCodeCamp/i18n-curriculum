# --description--

Tukutane na vipengele vya waendeshaji wa JavaScript vinavyotoka zaidi ya hesabu za shule.

Kawaida, waendeshaji wa kuongeza `+` hujumlisha nambari.

Lakini, ikiwa waendeshaji wa binary `+` utatumika kwa mfuatano wa herufi, huunganisha (kuunganisha) mfuatano huo:

```js
let s = "my" + "string";
console.log(s); // mystring
```

Kumbuka kwamba ikiwa mojawapo ya wapokeaji ni mfuatano wa herufi, basi mwingine hubadilishwa kuwa mfuatano wa herufi pia.

Kwa mfano:

```js
console.log(1 + '2'); // '12'
console.log('1' + 2); // '12'
```

Tazama, haijalishi kama mpokeaji wa kwanza ni mfuatano wa herufi au wa pili.

Hapa kuna mfano mgumu zaidi:

```js
console.log(2 + 2 + '1') // "41" and not "221"
```

Hapa, waendeshaji hufanya kazi mmoja baada ya mwingine. `+` ya kwanza hujumlisha nambari mbili, hivyo hurudisha `4`, kisha `+` inayofuata inaongeza mfuatano wa herufi `1` kwake, hivyo ni kama `4 + '1' = '41'`.

```js
console.log('1' + 2 + 2) // "122" and not "14"
```

Hapa, `+` ya kwanza huunganisha mfuatano wa herufi na nambari, hivyo hurudisha `'12'`, kisha `+` inayofuata inaongeza `2` kwake, hivyo ni kama `'12' + 2 = '122'`.

Waendeshaji wa binary `+` ni waendeshaji pekee wanaounga mkono mfuatano wa herufi kwa njia hii. Waendeshaji wengine wa hesabu hufanya kazi tu na nambari na kila mara hubadilisha wapokeaji wao kuwa nambari.

Hapa kuna maonyesho ya kutoa na kugawanya:

```js
console.log( 6 - '2' ); // 4, converts '2' to a number
console.log( '6' / '2' ); // 3, converts both operands to numbers
```
