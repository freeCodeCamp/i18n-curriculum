# --description--

لـ CSS خاصية تسمى `width` تتحكم في عرض العنصر. تمامًا كما هو الحال مع الخطوط، سنستخدم `px` (بكسل) لتحديد عرض الصورة.

على سبيل المثال، إذا أردنا إنشاء صنف CSS يسمى `larger-image` يمنح عناصر HTML عرضًا قدره 500 بكسل، سنستخدم:

```html
<style>
  .larger-image {
    width: 500px;
  }
</style>
```

# --instructions--

أنشئ صنفًا يسمى `smaller-image` واستخدمه لتغيير حجم الصورة بحيث يكون عرضها 100 بكسل فقط.

# --hints--

يجب أن يحتوي عنصر `img` على الصنف `smaller-image`.

```js
const relaxingCatImage = document.querySelector("img[src='https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg']");
const catImageClass = relaxingCatImage.getAttribute('class').trim().split(/\s+/g);
assert.include(catImageClass, 'smaller-image');
```

يجب أن يكون عرض صورتك 100 بكسل.

```js
const image = document.querySelector('img');
const width = image.getBoundingClientRect().width;
assert.isBelow(width, 200); 
assert.match(__helpers.removeCssComments(code), /\.smaller-image\s*{\s*width\s*:\s*100px\s*(;\s*}|})/i);
```
