# --description--

La funzione `range()` viene usata per generare una sequenza di numeri interi. Ecco la sintassi base per la funzione `range()`:

```py
range(start, stop, step)
```

L'argomento `stop` richiesto è un numero intero che rappresenta il punto finale della sequenza di numeri generata. Ecco un esempio di utilizzo della funzione `range()`:

```py
for num in range(3):
    print(num)
```

Il codice seguente genera una sequenza di numeri tra `0` e `2`. Il numero intero `3` non è incluso perché l'argomento `stop` è esclusivo.

Se non viene specificato un argomento `start`, allora il valore predefinito è `0`. In alternativa, puoi usare l'argomento opzionale `start` per iniziare la sequenza di numeri interi da un numero diverso da `0`. Ecco un esempio di generazione di una sequenza di numeri interi tra `1` e `4`:

```py
for num in range(1, 5):
    print(num)
```

Per impostazione predefinita la sequenza di numeri interi incrementa di `1`. Ma se vuoi cambiare questo valore predefinito, puoi usare l'argomento opzionale `step`. Ecco un esempio di generazione di una sequenza di numeri interi pari tra `2` e `10`:

```py
for num in range(2, 11, 2):
    print(num)
```

Come detto prima, la funzione `range()` ha un solo argomento obbligatorio. Se non fornisci argomenti a `range()`, otterrai un `TypeError`:

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: range expected at least 1 argument, got 0
```

È importante notare che la funzione `range()` accetta solo numeri interi come argomenti, non numeri decimali. Ricorda che i numeri decimali sono numeri con la virgola come `3.4`. Se provi a passare numeri decimali come argomenti, otterrai un `TypeError`:

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

Se vuoi generare una sequenza di numeri interi in ordine decrescente, puoi usare un numero intero negativo per l'argomento `step`, così:

```py
for num in range(40, 0, -10):
    print(num)
```

Il codice seguente stampa i numeri `40`, `30`, `20` e `10` in quest’ordine sulla console.

Un’altra cosa che puoi fare con la funzione `range()` è creare una lista di numeri interi usandola con il costruttore `list`. Il costruttore `list` serve a convertire un iterabile in una lista. Ecco un esempio di generazione di una lista di numeri interi pari tra `2` e `10`:

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

La funzione `range()` è un modo molto utile per generare una sequenza di numeri interi in Python. Una volta che la prendi confidenza, probabilmente la userai spesso nei tuoi programmi Python.
