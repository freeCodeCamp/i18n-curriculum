# --explanation--

`Looks better` означає, що щось виглядає привабливіше або більш естетично. Наприклад:

`The updated login screen looks better with the new icons.` – Дизайн покращений і більш привабливий.

`Works better` означає, що щось працює ефективніше або легше у використанні. Наприклад:

`The search feature works better after the update.` – Працює швидше або дає точніші результати.

Структура `not only ... but also ...` використовується, щоб показати два пов’язані позитивні зміни. У цьому контексті нова версія покращується як у зовнішньому вигляді, так і у функціональності.
