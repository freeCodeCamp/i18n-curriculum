# --description--

Тепер, коли ви знаєте про найпоширеніші способи ітерувати над значеннями та масивами, існують ще кілька способів ітерувати над масивами. Масиви мають вбудований метод під назвою `map()`, який використовується для створення нового масиву шляхом застосування функції до кожного елемента оригінального масиву. Метод `map()` не змінює оригінальний масив. Синтаксис методу `map()` виглядає так:

```javascript
const array = [1, 2, 3, 4, 5];

const newArray = array.map((arrayValue) => {
  return arrayValue * 2;
});

console.log(newArray); // Output: [2, 4, 6, 8, 10]
```

Метод `map()` створює новий масив, застосовуючи функцію `(arrayValue) => { return arrayValue * 2; }` до кожного елемента оригінального масиву. Це особливо корисно, коли ви хочете трансформувати елементи масиву, не змінюючи оригінальний масив.
