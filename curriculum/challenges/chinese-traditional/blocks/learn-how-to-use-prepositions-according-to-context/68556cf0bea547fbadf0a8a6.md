# --explanation--

`Around` 意味著大約或接近時間或地點。例如：

`We plan to deploy the update around Friday.`－這表示在那天附近，而非確切時間。

`Mid-quarter` 指的是三個月商務期間的中點。例如：

`The review will happen in mid-quarter。`－這表示它將在本季度中間進行。
