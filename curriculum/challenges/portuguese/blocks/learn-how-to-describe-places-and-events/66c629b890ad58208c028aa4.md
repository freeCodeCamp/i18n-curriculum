### --feedback--

# --explanation--

`Walkthrough` é composto por duas partes: 

- `Walk` refere-se a mover-se passo a passo, geralmente a pé.

- `Through` significa mover-se de um lado para o outro ou do início ao fim.

Quando essas palavras são combinadas em `walkthrough`, significa uma explicação ou guia passo a passo para ajudar alguém a entender como fazer algo. Por exemplo:

- `The software includes a walkthrough to help new users get started.` - Há um guia que explica cada passo para usar o software.

- `Let me give you a walkthrough of the process.` - Você explicará cada etapa do processo em detalhes.
