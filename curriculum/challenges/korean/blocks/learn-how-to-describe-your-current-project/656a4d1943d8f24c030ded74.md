# --description--

누군가가 부정문 끝에 `either`를 사용하면, 그 상황이 자신에게도 해당된다는 뜻입니다. 예를 들어:

`I don't like carrots. My sister doesn't like them either.` - 여기서 `either`는 화자와 그의 여동생 모두 당근을 좋아하지 않는다는 의미입니다.
