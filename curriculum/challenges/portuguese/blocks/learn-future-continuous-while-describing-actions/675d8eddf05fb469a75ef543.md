# --explanation--

`Solution` significa uma resposta ou forma de resolver um problema.Por exemplo:

`The engineers worked hard to find a solution to the network issue.` - Isso significa que os engenheiros trabalharam duro para resolver o problema da rede.

`Frustrated` descreve uma sensação de estar chateado ou irritado quando algo não funciona ou demora muito.Por exemplo:

`The customers were frustrated by the long wait times.` - Isso descreve a irritação e insatisfação que os clientes sentiram por causa dos atrasos.

`Do` em perguntas com tag é usado após uma frase principal negativa para confirmar ou verificar concordância.Por exemplo:

`We don't need to leave now, do we?` - A frase principal é negativa e a tag (`do we?`) é positiva, pedindo confirmação de forma conversacional.
