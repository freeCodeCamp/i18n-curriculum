# --description--

`Invent` significa crear algo nuevo que nadie ha hecho antes. Puede ser una nueva herramienta, máquina o forma de hacer algo. Las personas inventan cosas para resolver problemas o hacer la vida más fácil. Ejemplo: `Alexander Graham Bell inventó el teléfono.`

`Medicine` es algo que tomas para ayudarte a sentirte mejor cuando estás enfermo. Puede ser una pastilla, un líquido o una crema. Los médicos te dicen qué medicina usar para mejorar. Ejemplo: `Tomo medicina para mis dolores de cabeza`.
