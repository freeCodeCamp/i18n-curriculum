# --description--

Використайте універсальний селектор, щоб додати `box-sizing: border-box;` до свого CSS. Це гарантує, що елементи враховують відступ і кордон у своїй заданій ширині та висоті.

# --hints--

У вас має бути універсальний селектор (`*`).

```js
// assert.match(code, /\*\s*{\s*.*\s*}/)
assert(new __helpers.CSSHelp(document).getStyle('*'))
```

У вашому універсальному селекторі має бути властивість `box-sizing`, встановлена в `border-box`.

```js
// assert.match(code, /\*\s*{\s*box-sizing:\s*border-box;?\s*}/)
assert(new __helpers.CSSHelp(document).getStyle('*')?.boxSizing === 'border-box')
```
