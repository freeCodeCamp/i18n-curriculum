# --description--

Eine Spinne, S, sitzt in einer Ecke eines Quaderraums mit den Maßen 6 mal 5 mal 3, und eine Fliege, F, sitzt in der gegenüberliegenden Ecke. Wenn man sich auf den Oberflächen des Raums bewegt, beträgt die kürzeste „gerade“ Entfernung von S zu F 10, und der Weg ist im Diagramm dargestellt.

<img alt="ein Diagramm des Wegs einer Spinne und einer Fliege von einer Ecke eines Quaderraums zur gegenüberliegenden Ecke" src="https://cdn-media-1.freecodecamp.org/project-euler/cuboid-route.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Es gibt jedoch bis zu drei „kürzeste“ Wegkandidaten für jeden gegebenen Quader, und die kürzeste Route hat nicht immer eine ganzzahlige Länge.

Es lässt sich zeigen, dass es genau `2060` verschiedene Quader gibt, Rotationen ignoriert, mit ganzzahligen Abmessungen bis zu einer maximalen Größe von M mal M mal M, für die die kürzeste Route eine ganzzahlige Länge hat, wenn M = 100 ist. Dies ist der kleinste Wert von M, für den die Anzahl der Lösungen erstmals über zweitausend liegt; die Anzahl der Lösungen bei M = 99 ist `1975`.

Finden Sie den kleinsten Wert von M, so dass die Anzahl der Lösungen erstmals `n` übersteigt.

```js
function cuboidRoute(n) {
  // Based on https://www.mathblog.dk/project-euler-86-shortest-path-cuboid/
  function getLength(a, b) {
    return Math.sqrt(a ** 2 + b ** 2);
  }

  let M = 2;
  let counter = 0;

  while (counter < n) {
    M++;
    for (let baseHeightWidth = 3; baseHeightWidth <= 2 * M; baseHeightWidth++) {
      const pathLength = getLength(M, baseHeightWidth);
      if (Number.isInteger(pathLength)) {
        if (baseHeightWidth <= M) {
          counter += Math.floor(baseHeightWidth / 2);
        } else {
          counter += 1 + M - Math.floor((baseHeightWidth + 1) / 2);
        }
      }
    }
  }

  return M;
}
```
