# --description--

$d$ 桁の正の数（先頭にゼロなし）が、ちょうど1つの部分文字列だけが $d$ で割り切れる場合、その数をワンチャイルド数と呼びます。

例えば、5671 は4桁のワンチャイルド数です。部分文字列 5、6、7、1、56、67、71、567、671、5671 の中で、4で割り切れるのは 56 のみです。

同様に、104 は3桁のワンチャイルド数で、3で割り切れる部分文字列は 0 のみです。1132451 は7桁のワンチャイルド数で、7で割り切れる部分文字列は 245 のみです。

$F(N)$ を $N$ 未満のワンチャイルド数の個数とします。$F(10) = 9$、$F({10}^3) = 389$、$F({10}^7) = 277\,674$ であることが確認できます。

$F({10}^{19})$ を求めてください。
