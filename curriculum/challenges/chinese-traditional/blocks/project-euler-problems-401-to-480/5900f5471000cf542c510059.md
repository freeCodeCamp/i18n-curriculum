# --description--

對於正整數 $n$ 和數字 $d$，我們定義 $F(n, d)$ 為 $n$ 的除數中最後數字與 $d$ 相等的數量。

例如，$F(84, 4) = 3$。在 84 的因數中（1、2、3、4、6、7、12、14、21、28、42、84），有三個（4、14、84）的最後一位數字是 4。

我們也可以驗證 $F(12!, 12) = 11$ 和 $F(50!, 123) = 17\,888$。

尋找 $F({10}^6!, 65\\,432) \text{ 取模 } ({10}^{16} + 61)$。
