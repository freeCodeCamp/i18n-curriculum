# --explanation--

`Sure thing.` é uma forma informal e amigável de dizer `Of course` ou `I'll do it`.Mostra que o orador concorda com o pedido sem hesitação.Por exemplo:

- Pessoa 1: `Can you send me the file this afternoon?` - Esta pessoa está pedindo um arquivo. 

- Pessoa 2: `Sure thing.` - Isso significa que o falante concorda e enviará o arquivo conforme solicitado.
