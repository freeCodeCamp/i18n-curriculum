# --description--

最后的因素，最后的手段，平局中的平局。假设在考虑了所有其他因素之后，仍然有多个冲突的规则针对一个元素。级联如何确定应用哪个规则？

实际上非常简单。最后被定义的规则获胜。

```css
/* styles.css */

.alert {
  color: red;
}

.warning {
  color: yellow;
}
```

对于同时具有 `alert` 和 `warning` 类的元素，级联将遍历所有其他因素，包括继承（此处无）和规格（两个规则的规格都不高于对方）。由于 `.warning` 规则是最后定义的，并且没有其他因素能够决定应用哪个规则，因此它将被应用到该元素。
