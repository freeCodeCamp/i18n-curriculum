# --explanation--

`Exactly` は、相手の発言に完全に同意したり正しいと示すための副詞です。一方、`Exact` は、詳細に正確であることを意味する形容詞です。例えば：

- `You are exactly right about the cause of the problem.` – これは完全な同意を示します。人がどれほど正しかったか（形容詞）を表現しています。

- `We need the exact measurements to build the new server rack.` – これは非常に正確な測定を意味します。動詞や形容詞ではなく名詞を修飾しています。
