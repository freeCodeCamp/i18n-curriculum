# --description--

在太空週的第三天，你會得到一個陣列，該陣列表述你自己、衛星和你的母星之間通訊路徑中的距離（以公里為單位）。請根據以下條件判斷透過該路徑傳送的訊息到達目的星球所需的時間：

- 陣列中的第一個值是你的位置到第一顆衛星的距離。
- 每個後續值（最後一個除外）都是到下一顆衛星的距離。
- 陣列中的最後一個值是從前一顆衛星到你的母星的距離。
- 該訊息以 300,000 公里／秒 的速度傳遞。
- 每個衛星訊息**通過**時都會增加 0.5 秒的傳輸延遲。
- 傳回一個四捨五入到 4 位小數且移除尾隨零的數字。

```py
def send_message(route):
    total_distance = sum(route)
    delay = (len(route) - 1) * 0.5
    time = total_distance / 300_000
    total = time + delay
    return round(total, 4)
```
