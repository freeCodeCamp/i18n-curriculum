# --explanation--

`To be within budget` 또는 `to stay within budget`은 계획하거나 허용된 금액만 지출하고 그 이상은 지출하지 않았다는 뜻입니다. 이는 프로젝트나 활동이 재정적으로 잘 관리되고 있음을 보여줍니다. 예를 들어:

`The team stayed within budget, so we didn't need extra funding.` – 이는 원래 계획한 것보다 더 많이 지출하지 않았다는 의미입니다.
