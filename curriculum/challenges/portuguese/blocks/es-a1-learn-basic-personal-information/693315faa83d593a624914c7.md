# --description--

Ángela faz uma pergunta rápida de acompanhamento. A pergunta usa a palavra `¿cómo?`, que é uma forma comum de perguntar **como** algo é feito ou expresso. Por exemplo:

`¿Cómo te llamas?` – Qual é o seu nome?

Nesta tarefa, preste atenção nestas palavras:

- `y` – usado para conectar ideias ou adicionar informações.

- `cómo` – significa como e é usado para perguntar sobre a forma como algo acontece.

- `escribe` – vem do verbo `escribir`, que significa escrever.
