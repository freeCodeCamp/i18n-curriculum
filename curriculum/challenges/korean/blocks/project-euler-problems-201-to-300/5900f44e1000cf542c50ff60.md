# --description--

수열 1, 1, 1, 3, 5, 9, 17, 31, 57, 105, 193, 355, 653, 1201 ...

은 $T_1 = T_2 = T_3 = 1$이고 $T_n = T_{n - 1} + T_{n - 2} + T_{n - 3}$로 정의됩니다.

27이 이 수열의 어떤 항도 나누지 않는다는 것을 증명할 수 있습니다. 사실, 27은 이런 성질을 가진 첫 번째 홀수입니다.

위 수열의 어떤 항도 나누지 않는 ${124}^{\text{번째}}$ 홀수를 찾으세요.
