# --interactive--

Властивість CSS `filter` — це потужний інструмент, який дозволяє застосовувати графічні ефекти до елементів на вебсторінці. Вона особливо корисна для коригування візуального подання зображень, фонових зображень і навіть тексту без зміни оригінального ресурсу.

Властивість `filter` можна використовувати для створення різних ефектів, таких як розмиття, зміна кольору та корекція контрасту. Давайте розглянемо, як працює властивість filter, і дослідимо кілька загальних прикладів. Основний синтаксис для властивості `filter` досить простий:

```css
selector {
  filter: function(amount);
}
```

Тут `function` представляє конкретний ефект фільтра, який ви хочете застосувати, а `amount` зазвичай є значенням, що визначає інтенсивність ефекту. Тепер розглянемо деякі загальні функції фільтра та їхнє використання.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: blur(2px);
}
```

:::

Функція `blur` застосовує гаусове розмиття до елемента, кількість задається в пікселях і представляє радіус розмиття. Це правило CSS застосує розмиття в 2 пікселі до всіх зображень на сторінці. Ефект `blur` може бути корисним для створення глибини у вашому дизайні або для затемнення частин зображення.

Функція `brightness` регулює яскравість елемента. Значення `0%` зробить елемент повністю чорним, тоді як значення понад `100%` збільшать яскравість.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: brightness(150%);
}
```

:::

Це правило CSS збільшує яскравість елемента зображення на `50%`. Корекція яскравості може використовуватися, щоб зробити зображення більш виразним або створити ефект вигорілості.

Функція `grayscale` перетворює елемент у відтінки сірого. Кількість задається у відсотках, де `100%` означає повністю сірий, а `0%` залишає зображення без змін.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: grayscale(100%);
}
```

:::

Це правило перетворить елемент зображення у повний відтінок сірого. `grayscale` можна використовувати для створення вінтажного вигляду або щоб зменшити акцент на певних елементах сторінки.

Функція `sepia` застосовує сепію до елемента. Як і у випадку з відтінками сірого, використовується відсоткове значення:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: sepia(80%);
}
```

:::

Це правило застосовує `80%` ефект сепії до елемента зображення. Ефект сепії чудово підходить для створення вінтажного або старовинного вигляду.

Функція `hue-rotate` застосовує обертання відтінку до елемента. Значення задається в градусах і представляє оберт навколо кола кольорів.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: hue-rotate(90deg);
}
```

:::

Це правило обертає відтінок елемента зображення на `90` градусів. Обертання відтінку можна використовувати для створення психоделічних ефектів або для корекції загальної кольорової гами зображення.

Одна з найпотужніших особливостей властивості `filter` — це можливість комбінувати кілька ефектів. Ви можете застосувати кілька фільтрів до одного елемента, розділяючи їх пробілами:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {  
  filter: contrast(150%) brightness(110%) sepia(30%);  
}
```

:::

Це правило застосовує підвищений контраст, трохи збільшену яскравість і легкий ефект сепії до елемента зображення.

Комбінуючи фільтри, ви можете створювати складні та унікальні візуальні ефекти, пристосовані до ваших дизайнерських потреб. Властивість CSS filter — це універсальний інструмент, який дозволяє творчо маніпулювати візуальним виглядом вебелементів.

Хоча ми розглянули деякі з найпоширеніших функцій фільтрів, існують і інші, такі як `contrast`, `invert` та `saturate`. Як і з будь-якою потужною функцією, важливо обережно використовувати фільтри, щоб покращити дизайн, не перевантажуючи користувачів і не порушуючи доступність.
