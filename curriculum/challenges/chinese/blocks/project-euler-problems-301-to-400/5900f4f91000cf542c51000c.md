# --description--

在抛物线 $y = \frac{x^2}{k}$ 上，选择了三个点 $A(a, \frac{a^2}{k})$、$B(b, \frac{b^2}{k})$ 和 $C(c, \frac{c^2}{k})$。

设 $F(K, X)$ 为满足三角形 $ABC$ 至少有一个角为 45° 的整数四元组 $(k, a, b, c)$ 的数量，其中 $1 ≤ k ≤ K$ 且 $-X ≤ a < b < c ≤ X$。

例如，$F(1, 10) = 41$ 和 $F(10, 100) = 12\\,492$。

查找 $F({10}^6, {10}^9)$。
