# --description--

Wenn Sie Ihr Mobilgerät verwenden, um durch den chinesischen Lehrplan zu navigieren, können Sie Pinyin und chinesische Schriftzeichen direkt mit den benutzerdefinierten Eingaben von freeCodeCamp eingeben. Wenn Sie jedoch die chinesische Tastatur auf Ihrem Gerät verwenden möchten, erfahren Sie hier, wie Sie sie einrichten.

## Hinzufügen von Sprach-Tastaturen  

### Auf Android

**HINWEIS:** Der Pfad auf Android ist nicht universell und kann je nach Gerät variieren.

1. Öffnen Sie die Einstellungen.

2. Gehen Sie zu System (oder Allgemeine Verwaltung, je nach Gerät).

3. Tippen Sie auf Tastatureinstellungen (oder Sprachen & Eingabe).  

4. Tippen Sie auf Sprachen und Typen (oder Tastatur hinzufügen). 
 
5. Aktivieren Sie Chinesisch (vereinfacht oder CN) oder laden Sie es bei Bedarf herunter.  

6. Speichern Sie oder tippen Sie auf Fertig, falls angezeigt.

### Auf dem iPhone

1. Öffnen Sie die Einstellungen.

2. Gehen Sie zu Allgemein > Tastatur.

3. Tippen Sie auf Tastaturen.

4. Tippen Sie auf Neue Tastatur hinzufügen.  

5. Wählen Sie Chinesisch (vereinfacht).  

6. Verwenden Sie das Globus-Symbol auf der Tastatur, um die Sprache zu wechseln.

## Verwendung von Tastaturen von Drittanbietern  

Sie können auch Tastatur-Apps von Drittanbietern auf Ihrem Telefon oder Tablet installieren, um das Tippen auf Chinesisch zu erleichtern. Diese Tastaturen bieten oft nützliche Funktionen wie intelligente Zeichen-Vorhersagen und Unterstützung für mehrere Eingabemethoden, wie Pinyin oder Handschrift.
