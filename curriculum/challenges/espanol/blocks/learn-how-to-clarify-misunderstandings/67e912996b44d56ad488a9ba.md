# --explanation--

`Code reuse` se refiere a la práctica de usar código existente en nuevas partes de un programa. Por ejemplo:

La clase `Login` se usó en múltiples aplicaciones para manejar el acceso de usuarios, lo cual es un buen ejemplo de reutilización de código. En lugar de escribir la lógica de inicio de sesión para cada aplicación, se reutiliza el mismo código, mejorando la eficiencia.

`Extending` significa agregar nuevas características a algo que ya existe. Por ejemplo:

La clase `AdminUser` extiende la clase básica `User` agregando un método `approveUser()`. - El `AdminUser` mantiene todo de `User` y estructura sobre ello con nuevas capacidades.
