# --description--

Ein String ist eine Folge von Zeichen, die entweder von einfachen oder doppelten Anführungszeichen umgeben sind. In einigen Programmiersprachen werden Zeichen, die von einfachen Anführungszeichen umgeben sind, anders behandelt als Zeichen, die von doppelten Anführungszeichen umgeben sind, aber in Python werden sie gleich behandelt. Sie können also beide verwenden, wenn Sie mit Strings arbeiten. Hier sind einige Beispiele für Strings:

```python
my_str_1 = 'Hello'
my_str_2 = "World"
```

Wenn Sie einen mehrzeiligen String benötigen, können Sie dreifache doppelte oder einfache Anführungszeichen verwenden:

```python
my_str_3 = """Multiline
string"""
my_str_4 = '''Another
multiline
string'''
```

Wenn Ihr String entweder einfache oder doppelte Anführungszeichen enthält, haben Sie zwei Möglichkeiten:

- Verwenden Sie die entgegengesetzte Art von Anführungszeichen. Das heißt, wenn Ihr String einfache Anführungszeichen enthält, verwenden Sie doppelte Anführungszeichen, um den String einzuschließen, und umgekehrt:
    

```python
msg = "It's a sunny day"
quote = 'She said, "Hello World!"'
```

- Maskieren Sie das einfache oder doppelte Anführungszeichen im String mit einem Backslash (`\`). Mit dieser Methode können Sie entweder einfache oder doppelte Anführungszeichen verwenden, um den String selbst einzuschließen:
    

```python
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

Manchmal müssen Sie überprüfen, ob ein String ein oder mehrere Zeichen enthält. Dafür stellt Python den `in`-Operator bereit, der einen Boolean zurückgibt, der angibt, ob das oder die Zeichen im String vorhanden sind oder nicht.

Hier sind einige Beispiele:

```python
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

Schauen wir uns nun an, wie Sie die Länge eines Strings ermitteln und mit den einzelnen Zeichen in einem String arbeiten können, ein Vorgang, der **Indexierung** genannt wird. Um die Länge eines Strings zu erhalten, können Sie die eingebaute `len()`-Funktion verwenden. Hier ist ein Beispiel:

```python
my_str = 'Hello world'
print(len(my_str))  # 11
```

Jedes Zeichen in einem String hat eine Position, die Index genannt wird. Der Index ist nullbasiert, das heißt, der Index des ersten Zeichens eines Strings ist `0`, der Index des zweiten Zeichens ist `1` und so weiter. Um auf ein Zeichen anhand seines Index zuzugreifen, verwenden Sie eckige Klammern (`[]`) mit dem Index des Zeichens, auf das Sie zugreifen möchten, darin. Hier sind einige Beispiele:

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
```

Negative Indizes sind ebenfalls erlaubt, sodass Sie das letzte Zeichen eines Strings mit `-1`, das vorletzte Zeichen mit `-2` und so weiter erhalten können:

```python
my_str = 'Hello world'
print(my_str[-1])  # d
print(my_str[-2]) # l
```

Viele andere Programmiersprachen gruppieren Datentypen grob in primitive oder Referenztypen. Primitive Typen sind einfach und unveränderlich, das heißt, sie können nach der Deklaration nicht mehr geändert werden. Referenztypen können mehrere Werte enthalten und sind entweder veränderlich oder unveränderlich. Python zieht jedoch keine klare Grenze zwischen diesen beiden Gruppen. Stattdessen wird jede Daten als Objekt behandelt, und einige Objekte sind unveränderlich, während andere veränderlich sind.

Unveränderliche Datentypen können nach der Deklaration nicht modifiziert oder verändert werden. Sie können ihre Variablen auf etwas Neues zeigen lassen, was als Neu-Zuweisung bezeichnet wird, aber Sie können das ursprüngliche Objekt selbst nicht ändern, indem Sie Elemente hinzufügen, entfernen oder ersetzen.

Strings sind unveränderliche Datentypen in Python. Das bedeutet, dass Sie einer Variablen einen anderen String neu zuweisen können:

```python
greeting = 'hi'
greeting = 'hello'
print(greeting) # hello
```

Aber eine direkte Änderung eines Strings ist nicht erlaubt:

```python
greeting = 'hi'
greeting[0] = 'H' # TypeError: 'str' object does not support item assignment
```

Beispiele für andere unveränderliche Datentypen in Python sind Integer, Float, Boolean, Tuple und Range. Sie werden jeden dieser Typen in den kommenden Lektionen kennenlernen.

## --text--

Was bedeutet es, dass ein String unveränderlich ist?

## --answers--

Eine Variable kann nicht auf einen anderen String neu zugewiesen werden.

### --feedback--

Unveränderlich bedeutet, dass sich etwas nicht ändert.

---

Die Neu-Zuweisung einer Variable ändert den ursprünglichen String direkt.

### --feedback--

Unveränderlich bedeutet, dass sich etwas nicht ändert.

---

Ein String kann nicht direkt geändert werden, indem seine einzelnen Zeichen verändert werden.

---

String-Zeichen können nur unter bestimmten Bedingungen direkt geändert werden.
