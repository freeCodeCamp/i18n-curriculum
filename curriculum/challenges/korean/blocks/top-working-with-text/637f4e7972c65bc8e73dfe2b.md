# --description--

이 수업의 모든 예제에서 다른 요소 안에 있는 요소를 들여쓰기한 것을 눈치채셨을 겁니다. 이것을 요소를 중첩하는 것이라고 합니다.

요소를 다른 요소 안에 중첩하면 부모와 자식 관계가 만들어집니다. 중첩된 요소는 자식이고, 그 안에 중첩된 요소는 부모입니다.

다음 예제에서 `body` 요소가 부모이고 `p` 요소가 자식입니다:

```html
<html>
  <head>
  </head>
  <body>
    <p>Lorem ipsum dolor sit amet.</p>
  </body>
 </html>
```

사람 관계와 마찬가지로 HTML 부모 요소는 여러 자식을 가질 수 있습니다. 같은 중첩 수준에 있는 요소들은 형제 요소로 간주됩니다.

예를 들어, 다음 코드에서 두 `p` 요소는 모두 `body` 태그의 자식이고 서로 같은 중첩 수준에 있으므로 형제입니다:

```html
<html>
  <head>
  </head>
  <body>
    <p>Lorem ipsum dolor sit amet.</p>
    <p>Ut enim ad minim veniam.</p>
  </body>
 </html>
```

들여쓰기는 중첩 수준을 명확하고 읽기 쉽게 만들어, 앞으로 여러분과 다른 개발자가 HTML 작업을 할 때 도움이 됩니다. 자식 요소는 두 칸 들여쓰는 것을 권장합니다.

요소 간의 부모, 자식, 형제 관계는 나중에 CSS로 HTML 스타일을 지정하고 JavaScript로 동작을 추가할 때 훨씬 중요해집니다. 지금은 요소들이 어떻게 관계되는지와 그 관계를 설명하는 용어를 아는 것이 중요합니다.
