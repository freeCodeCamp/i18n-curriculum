# --description--

考慮數字 6。6 的因數是：1、2、3 和 6。

從 1 到 6（包含 6）中的每個數字都可以寫成 6 的不同因數的和：

$1 = 1$、$2 = 2$、$3 = 1 + 2$、$4 = 1 + 3$、$5 = 2 + 3$、$6 = 6$。

當一個數字 $n$ 滿足從 1 到包含 $n$ 的每個數字都可以表示為 $n$ 的不同因數的和時，該數字 $n$ 稱為實用數。

一對相差六的連續質數稱為 sexy pair（因為「sex」是拉丁文中「六」的意思）。第一個 sexy pair 是 (23, 29)。

我們有時會發現三重配對，意指三組連續的性感質數配對，使得每組配對的第二個成員是下一組配對的第一個成員。

我們將呼叫一個數字 $n$，使得：

- ($n - 9$)、($n - 3$)、($n - 3$)、($n + 3$)、($n + 3$)、($n + 9$) 形成一個三重配對，且
- 數字 $n - 8$、$n - 4$、$n$、$n + 4$ 和 $n + 8$ 都是實用的，

工程師的天堂。

找出前 4 位工程師天堂的總和。
