# --description--

لنسم مثلثًا أضلاعه أعداد صحيحة وله زاوية واحدة بالضبط مقدارها 60° مثلث 60°.

ليكن $r$ نصف قطر الدائرة المحيطة لمثلث 60° هذا.

يوجد 1234 مثلثًا 60° حيث $r ≤ 100$.

ليكن $T(n)$ عدد مثلثات 60° حيث $r ≤ n$، إذًا $T(100) = 1234$، $T(1000) = 22767$، و $T(10000) = 359912$.

احسب $T(1053779)$.
