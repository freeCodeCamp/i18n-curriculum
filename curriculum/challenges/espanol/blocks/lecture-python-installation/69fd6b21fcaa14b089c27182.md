# --description--

En la lección anterior, aprendiste cómo instalar Python en tu dispositivo local. En esta lección, aprenderás cómo ejecutar código Python localmente.

Antes de que puedas empezar a crear tus scripts de Python localmente, necesitarás instalar un editor de código o un IDE. IDE significa entorno de desarrollo integrado y tiene funcionalidades que incluyen un editor de código, herramientas de prueba, una terminal y más.

Un editor de código popular que muchos desarrolladores usan es VS Code. Visita `https://code.visualstudio.com/download` para descargar VS Code, que es compatible con entornos Mac, Windows y Linux. Este es el editor que se usará en esta lección.

Otras opciones comunes para el desarrollo en Python incluyen las siguientes:

- PyCharm
- Spyder

Una vez que hayas descargado tu editor, necesitarás abrirlo. Cuando esté abierto, deberás abrir una carpeta. Puedes hacer clic en la opción de menú `File` en la esquina superior izquierda y hacer clic en `Open Folder`. Desde ahí, puedes elegir o crear una carpeta para tu proyecto. Para practicar fuera de la lección, podrías querer una carpeta llamada `python-projects`.

Una vez que tengas tu carpeta configurada, puedes abrir el explorador que es la barra lateral izquierda en VS Code. Haz clic en el ícono `New File` que parece una hoja de papel con un ícono de más encima. Luego crea un archivo llamado `main.py` y presiona enter. La `.py` es una extensión de archivo que indica que este es un archivo Python.

Abre tu archivo `main.py` y escribe el siguiente código:

```py
print("Hello, world!")
```

Para ejecutar este código, tienes varias opciones. La primera opción es hacer clic en el botón de ejecutar que parece un botón de reproducir ubicado en la esquina superior derecha. Eso abrirá una terminal y ejecutará tu script de Python allí. Deberías ver el texto `"Hello, world!"`.

Otra opción es abrir una terminal y ejecutar el programa manualmente usando:

```bash
python main.py
```

Debes ejecutar este comando desde la carpeta donde está guardado `main.py`. Por ejemplo, si `main.py` está dentro de una carpeta llamada `python-projects`, primero usa lo siguiente en la terminal:

```bash
cd python-projects
```

Luego ejecuta:

```bash
python main.py
```

En algunos entornos, especialmente en sistemas macOS y Linux donde están instalados Python 2 y Python 3, puede que necesites usar `python3` en lugar de `python`. Esto es común cuando el comando `python` no existe o apunta a una versión antigua de Python.

```bash
python3 main.py
```

A medida que avances en el curso, deberías probar los ejemplos de las lecciones localmente y ver los resultados. También deberías intentar crear tus propios ejemplos para poner a prueba lo que has aprendido en las lecciones.
