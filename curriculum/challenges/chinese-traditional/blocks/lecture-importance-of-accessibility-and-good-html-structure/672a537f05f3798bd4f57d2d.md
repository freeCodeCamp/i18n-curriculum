# --description--

無障礙性是數位內容中一個關鍵但常被忽略的面向。當你使你的數位內容具備無障礙性時，確保它符合無障礙標準是很重要的。

無障礙審核工具是一種應用程式，透過報告可透過自動化測試輕易發現的無障礙問題，幫助你改善數位內容的無障礙性。這些內容包括網站、網頁應用程式和行動應用程式。

重要的是要注意，雖然自動化無障礙工具在改善無障礙性方面扮演一定角色，但它們通常只能找到大約三分之一的所有可能無障礙問題。因此，不應完全依賴它們來評估你的內容的無障礙性。手動測試，最好由有障礙的人士進行，始終是確保你的內容盡可能無障礙所必需的。

讓我們來看看一些可以幫助你提升數位內容可存取性的免費工具。

Google Lighthouse 是一個流行的網頁指標檢查工具，你可以直接在 Chrome DevTools 或線上使用。這表示你不僅可以檢查線上網站，也可以檢查區域開發的網站。

你可以檢查的指標包括存取、SEO、最佳實務和效率。

要使用 Lighthouse，請按下 `F12` 開啟你的 DevTools，並切換到 Lighthouse 分頁。選擇你想檢查的指標，選擇你想測試的裝置，然後點擊「Analyze page load」按鈕。

在檢查完成後，將會顯示一個無障礙分數，以及需要修正的問題列表。

如果你想要更可靠的指標，請考慮使用網頁版。缺點是它不支援測試區域的網站。你可以在 `pagespeed.web.dev` 存取網頁版。

WAVE 是另一個你可以作為 Chrome 擴充套件或在網路上使用的可靠存取檢查工具。你只需要輸入你網站的 URL，系統就會為你產生一份完整的存取報告。這份報告包含已實作的存取功能、ARIA 和對比度。

IBM Equal Accessibility Checker 是另一個強固的工具，用於提升數位內容的可存取性。使用它，你可以掃描你的網站以找出可存取性問題，並產生詳細報告。

你可以將它用作 Chrome 擴充套件或 Firefox 增益集。

要將 IBM Accessibility Checker 作為 Chrome 擴充套件使用，請從 Chrome 線上應用程式商店下載。按下 `F12` 開啟你的 Devtools，並在 Elements 面板中選擇「Accessibility Checker」標籤。點擊掃描按鈕開始檢查，系統會為你產生報告。你可以點擊「Export XLS」按鈕，將報告匯出為試算表和 HTML 檔案。

請記住，雖然這些自動化工具能幫助你讓內容更易於存取，但從任何一個工具獲得完美分數並不代表你的內容完全可存取。這些工具測試的問題範圍有限，且始終需要手動測試以確保每個人都有更易於存取的體驗。
