# --description--

어떤 방향을 사용하든 플렉스 컨테이너는 주축과 교차 축, 두 개의 축이 있다고 생각해야 합니다. 이 축들의 방향은 `flex-direction`가 변경될 때 바뀝니다. 대부분의 경우 `flex-direction: row`는 주축을 수평(왼쪽에서 오른쪽)으로 설정하고, `column`는 주축을 수직(위에서 아래)으로 설정합니다.

다시 말해, 첫 번째 예제에서는 `display: flex`를 div에 적용해 자식 요소들을 수평으로 배치했습니다. 이것은 기본 설정인 `flex-direction: row`의 예시입니다. 다음 예제도 매우 비슷합니다. `flex-direction: column`라는 줄의 주석을 해제하면 div들이 수직으로 쌓이게 됩니다.
