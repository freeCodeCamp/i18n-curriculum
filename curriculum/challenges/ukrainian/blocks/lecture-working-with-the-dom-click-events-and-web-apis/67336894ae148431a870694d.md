# --interactive--

Давайте дізнаємося про DOM і чому він такий важливий для веброзробки. DOM означає Document Object Model. Це програмний інтерфейс, який дозволяє нам взаємодіяти з HTML-документами.

За допомогою DOM ви можете додавати, змінювати або видаляти елементи на вебсторінці. Ви навіть можете зробити свій сайт інтерактивним, змусивши елементи слухати та реагувати на події.

У DOM HTML-документ представлений у вигляді дерева вузлів. Кожен вузол представляє HTML-елемент із HTML-документа:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>DOM Example</title>
  </head>
  <body>
    <h1>What is the DOM?</h1>
    <h2>Let's learn about the DOM</h2>
  </body>
</html>
```

Це діаграма, що представляє базову структуру DOM у нашому прикладі:

```md
Document
========

HTML
-----

- Head
  - Title
- Body
  - H1
  - H2   
```

Звісно, вона може бути більш детальною та складною залежно від структури розмітки HTML вебсторінки.

HTML-документ є кореневим вузлом у ієрархії DOM. Він має один дочірній вузол — елемент `html`. Це кореневий елемент HTML-документа, оскільки всі інші вузли походять від нього.

Елемент `html` має двох дочірніх: `head` і `body`.

Елемент `head` містить метадані про документ. Він надає важливу інформацію про вебсторінку.

Елемент `body` містить видимий вміст вебсторінки.

Ви можете отримати доступ до цих елементів за допомогою JavaScript, працювати з ними у своєму коді та навіть динамічно їх змінювати. Ось сила DOM і веб API.

Щоб отримати доступ до цих елементів у JavaScript, ви можете використовувати методи `getElementById()` і `querySelector()`. Ці методи є веб API, бо вони надають стандартизовані способи взаємодії з DOM за допомогою JavaScript.

За допомогою `getElementById()` ви можете отримати об’єкт, який представляє HTML-елемент із заданим `id`. Пам’ятайте, що `id` має бути унікальним у кожному HTML-документі, тому цей метод поверне лише один об’єкт `Element`. Ось приклад:

```js
const container = document.getElementById("container");
```

Цей рядок JavaScript-коду отримує елемент із значенням `id` `container` і призначає цей об’єкт константі JavaScript. Ви повинні передати `id` у лапках як аргумент. Якщо ви виведете цей об’єкт у консоль, побачите його у вихідних даних:

:::interactive_editor

```html
<div id="container">
  <h1>Hello, World!</h1>
  <p>Welcome to learning about the DOM.</p>
</div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
console.log(container); 
```

:::

`querySelector()` ширший за `getElementById()`. За допомогою `querySelector()` ви можете отримати перший елемент у HTML-документі, який відповідає CSS-селектору, переданому як аргумент.

У цьому прикладі ви отримаєте елемент, що відповідає селектору `section`:

:::interactive_editor

```html
<section>
  <h2>Section Title</h2>
  <p>This is a section of the webpage.</p>
</section>
<script src="index.js"></script>  
```

```js
const sectionEl = document.querySelector("section");
console.log(sectionEl);
```

:::

Якщо ви хочете вибрати елемент за його ім’ям класу, використовуючи `querySelector()`, потрібно додати префікс крапки (`.`) перед ім’ям класу. Ось приклад:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="highlight">
  <p>This content is highlighted.</p>
</div>
<script src="index.js"></script>
```

```css
body {
  background-color: lightgray;
}

.highlight {
  background-color: yellow;
}
```

```js
const highlightEl = document.querySelector(".highlight");
console.log(highlightEl);
```

:::

Також є інші методи для вибору кількох елементів, як `getElementsByClassName()` і `querySelectorAll()`. Ви дізнаєтеся про них більше в майбутніх уроках.

Розуміючи, як ефективно використовувати веб API для маніпуляції DOM, ви зможете створювати потужні та інтерактивні вебзастосунки.
