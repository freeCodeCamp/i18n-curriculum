# --explanation--

A estrutura `help` + `to` + verbo é usada para mostrar que alguém apoia ou auxilia a fazer algo. Também é correto dizer `help` + verbo sem `to` (por exemplo, `help make`). Outro exemplo: 

`He helped to fix the code.` ou `He helped fix the code.` – Ambos significam que ele deu suporte para resolver o problema.
