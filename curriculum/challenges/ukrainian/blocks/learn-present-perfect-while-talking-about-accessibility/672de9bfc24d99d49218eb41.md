# --explanation--

`Specific` означає щось конкретне або відмінне. Наприклад:

`She had specific questions about the project requirements.` — це означає, що у неї були конкретні питання щодо певних деталей проєкту, а не просто загальні питання.

`As expected` означає, що щось відбулося так, як люди думали. Наприклад:

`The software update improved performance, as expected, because the team fixed key bugs.` — це означає, що продуктивність покращилася саме так, як команда передбачала, завдяки виправленню помилок в оновленні.
