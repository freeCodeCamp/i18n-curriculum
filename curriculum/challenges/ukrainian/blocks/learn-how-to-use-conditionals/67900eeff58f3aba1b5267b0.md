# --explanation--

Давайте розберемося:

Що сказав Марк? Він сказав, що BIOS `prevents the computer from starting.` Це означає, що він говорить про те, що BIOS **може робити**, коли він пошкоджений.

Якщо Сара погоджується, чи буде вона говорити про те, що BIOS **є**, чи про те, що BIOS **може робити**? Друге, правда? Ось чому `Yeah, it can` — правильний вибір.

Чому `yeah, it is` неправильний? Тому що `it is` говорить про стан BIOS, а не про те, що він може робити. Ось приклад:

**Правильно**: `Yeah, it can.` – Це погоджується з думкою Марка про те, що BIOS може робити.

**Неправильно**: `Yeah, it is.` – Це не підходить, бо Марк не описує, що BIOS є, він говорить про його впливи.
