# --description--

令 $M(n, k, m)$ 表示二项式系数 $\displaystyle\binom{n}{k}$ 模 $m$。

计算 $\sum M({10}^{18}, {10}^9, p \times q \times r)$，其中 $1000 < p < q < r < 5000$ 且 $p$、$q$、$r$ 为素数。
