# --description--

辺の長さが $\sqrt{5}$、$\sqrt{65}$、$\sqrt{68}$ の三角形を考えます。この三角形の面積は 9 であることが示せます。

$S(n)$ は、辺の長さが $\sqrt{1 + b^2}$、$\sqrt{1 + c^2}$、$\sqrt{b^2 + c^2}$（正の整数 $b$ と $c$ に対して）の三角形のうち、面積が整数で $n$ を超えないものすべての面積の合計です。

例の三角形は $b = 2$、$c = 8$ です。

$S({10}^6) = 18\,018\,206$ です。

$S({10}^{10})$ を求めてください。
