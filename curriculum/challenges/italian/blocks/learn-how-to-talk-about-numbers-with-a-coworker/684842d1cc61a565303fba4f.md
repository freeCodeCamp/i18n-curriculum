# --explanation--

`Make sure to` si usa per esprimere responsabilità o promessa, cioè che chi parla farà sicuramente qualcosa. Per esempio:

`Make sure to save your work before closing.` — Non dimenticare di salvare prima di chiudere il programma.

`As needed` significa fare qualcosa solo quando diventa necessario. Per esempio:

`We can add more team members as needed.` — Aggiungeranno persone solo se diventerà necessario.

Nella frase di Alice, dice che informerà regolarmente David e modificherà il piano di budget se il progresso del progetto lo richiede.
