# --explanation--

`Present Continuous` 시제는 지금 일어나고 있는 행동에 대해 말할 때 사용합니다. `I'm thinking` 구문은 누군가가 현재 고려하거나 계획 중인 것을 나타냅니다.

Brian이 교육과 관련해 현재 무엇을 생각하는지 이해하려면, 그가 자신의 계획에 대해 말하는 오디오 부분을 주의 깊게 들으세요.

`I'm thinking of using...`와 `especially for...` 바로 뒤에 Brian이 하는 말을 집중해서 들으면 교육에 대한 그의 구체적인 생각을 파악하는 데 도움이 됩니다.
