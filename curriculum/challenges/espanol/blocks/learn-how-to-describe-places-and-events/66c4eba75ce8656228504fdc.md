# --explanation--

`Should` se usa para expresar que se espera que algo ocurra o sea cierto. Cuando Sarah dice, `It should`, quiere decir que se espera que la vista previa en vivo funcione en Visual Studio Code. Por ejemplo:

- `The app should start when you press the button.` - Esto significa que esperas que la app se inicie al presionar el botón.
