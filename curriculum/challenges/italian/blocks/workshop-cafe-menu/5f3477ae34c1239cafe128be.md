# --description--

Ora hai tre selettori di tipo con lo stesso stile. Puoi aggiungere lo stesso gruppo di stili a molti elementi creando una lista di selettori. Ogni selettore è separato da virgole in questo modo:

```css
selector1, selector2 {
  property: value;
}
```

Elimina i tre selettori di tipo esistenti e sostituiscili con una lista di selettori che centra il testo per gli elementi `h1`, `h2` e `p`.

# --hints--

Dovresti usare un singolo selettore di tipo per tutti e tre gli elementi, `h1, h2, p`. Assicurati di usare quell’ordine.

```js
const hasSelector = new __helpers.CSSHelp(document).getStyle('h1, h2, p');
assert.exists(hasSelector);
```

Dovresti avere solo un selettore nel tuo elemento `style`.

```js
const selectors = new __helpers.CSSHelp(document).getCSSRules().filter(x => x.selectorText)
assert.equal(selectors.length, 1);
```
