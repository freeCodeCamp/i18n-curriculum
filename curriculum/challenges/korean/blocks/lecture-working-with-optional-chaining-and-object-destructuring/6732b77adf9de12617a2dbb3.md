# --interactive--

객체 구조 분해는 JavaScript에서 객체의 값을 추출해 더 간결하고 읽기 쉽게 변수에 할당할 수 있게 해주는 강력한 기능입니다. 

이는 ES6(ECMAScript 2015) 명세의 일부이며 많은 JavaScript 개발자에게 필수 도구가 되었습니다. 

구조 분해는 특히 복잡한 객체를 다루거나 여러 값을 한 번에 추출해야 할 때 코드를 단순화할 수 있습니다.

본질적으로 객체 구조 분해는 객체에서 값을 꺼내 별도의 변수에 할당하는 것입니다. 객체 속성에 하나씩 접근하는 대신 한 문장으로 여러 속성을 추출할 수 있어 코드를 더 깔끔하고 효율적으로 만듭니다.

객체 구조 분해가 어떻게 작동하는지 예제로 시작해 보겠습니다:

:::interactive_editor

```js
const person = { name: "Alice", age: 30, city: "New York" };

const { name, age } = person;

console.log(name); // Alice
console.log(age);  // 30
```

:::

이 예제에서는 `name` 객체에서 `age`와 `person` 속성을 추출해 같은 이름의 변수에 할당하고 있습니다. 

객체 구조 분해의 강력한 점 중 하나는 추출한 값을 다른 이름의 변수에 할당할 수 있다는 것입니다. 이는 속성 이름이 기존 변수와 충돌할 수 있거나 다른 이름을 사용하고 싶을 때 특히 유용합니다:

:::interactive_editor

```js
let person = { name: "Alice", age: 30, city: "New York" };

let { name: personName, age: personAge } = person;

console.log(personName); // Alice
console.log(personAge); //  30
```

:::

이 경우 `name` 속성을 `personName`라는 변수에 할당하고, `age`와 `personAge`도 마찬가지로 처리합니다.

객체 구조 분해는 기본값을 설정할 수도 있습니다. 구조 분해하는 객체에 속성이 없으면 대체 값을 지정할 수 있습니다:

:::interactive_editor

```js
let person = { name: "Alice", age: 30, city: "New York" };
let { name, age, country = "Unknown" } = person;

console.log(country); // Unknown
```

:::

여기서는 `country` 객체에 `person`가 없으므로 기본값 `Unknown`가 할당됩니다.

또 다른 흔한 경우는 중첩 객체 구조 분해입니다. 중첩된 객체 안의 속성을 또 다른 중괄호를 사용해 구조 분해할 수 있습니다:

:::interactive_editor

```js
const recipe = {
  name: "Chocolate Cake",
  ingredients: {
    flour: "2 cups",
    sugar: "1 cup"
  }
};

// Extract `flour` from `ingredients`
const { ingredients: { flour } } = recipe;

console.log(flour); // "2 cups"
```

:::

이는 속성에 직접 접근하는 것과 같습니다:

```js
const flour = recipe.ingredients.flour;
console.log(flour); // "2 cups"
```

이제 객체 구조 분해의 축약 구문에 대해 이야기해 보겠습니다. 특히 속성 이름과 변수 이름이 같을 때 객체를 만들 때 축약 구문을 사용할 수 있습니다:

:::interactive_editor

```js
let name = "Bob";
let age = 25;

let person = { name, age };

console.log(person); // { name: "Bob", age: 25 }
```

:::

위 코드는 변수와 같은 이름의 속성에 그 변수 값을 할당합니다.

이 축약 구문은 함수에서 객체를 반환하거나 여러 속성을 가진 객체를 만들 때 특히 유용합니다:

:::interactive_editor

```js
function createPerson(name, age) {
  return { name, age };
}

let person = createPerson("Charlie", 35);
console.log(person); // { name: "Charlie", age: 35 }
```

:::

객체 구조 분해와 축약 객체 구문은 코드를 더 간결하고 읽기 쉽게 만드는 강력한 기능입니다. 

복잡한 자료 구조를 다루거나 함수에 여러 매개변수를 전달할 때 특히 유용합니다. 

JavaScript를 계속 사용하면서 이러한 기법들이 코드를 단순화하고 더 표현력 있게 만드는 많은 상황을 만나게 될 것입니다.
