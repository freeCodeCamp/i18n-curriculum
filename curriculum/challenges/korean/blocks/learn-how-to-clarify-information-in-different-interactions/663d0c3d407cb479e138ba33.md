# --description--

`Next`는 현재 것 바로 다음에 오는 것을 의미합니다. 예를 들어, `next month`는 이것 바로 다음에 오는 `month`를 뜻합니다.

`The following`는 다른 언급된 것 바로 다음에 오는 것을 의미합니다. 예를 들어, 2주 후에 어떤 일이 일어난다고 말할 때 `the following week` after `next week`라고 할 수 있습니다. 이는 방금 이야기한 다른 것 다음에 온다는 뜻입니다.

대화를 듣고 빈칸을 채우세요.
