# --description--

Nesta tarefa, você aprenderá a diferença entre `specific` e `specify`. `Specific` é um adjetivo que descreve algo claramente definido ou detalhado, enquanto `specify` é um verbo que significa declarar algo em detalhes.

Por exemplo, uma instrução `specific` pode ser `use port 8080`, enquanto `specify` pode significar declarar esse detalhe: `specify the port number.`

`Setup` refere-se ao arranjo ou preparação de algo, frequentemente usado no contexto de preparar software ou equipamento, como `setup instructions for the software.`
