# --description--

Ángela faz uma pergunta usando a estrutura `¿Cuál es tu` + substantivo.

Ela poderia parar por aí. Em vez disso, ela adiciona outra informação usando o conector `y` e um segundo substantivo.

`y` é uma conjunção usada para conectar dois substantivos. Aqui está a estrutura que ela usa:

`Qual é o seu` + noun + `e` + noun`?`

Isso permite que o orador faça uma `request` de mais de um detalhe na mesma pergunta.

# --explanation--

Na frase `¿Cuál es tu nombre y apellido?`, dois substantivos estão conectados com `y`:

- `nombre`

- `apellido`

A palavra `y` conecta duas informações.

Ángela primeiro pergunta o `nombre` da pessoa e depois adiciona o `apellido`. 

Essa estrutura permite que o locutor faça uma `request` de vários detalhes em uma única pergunta:

`¿Cuál es tu correo y número?`

Aqui, a segunda informação que Ángela pede é `apellido`.
