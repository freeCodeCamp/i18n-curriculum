# --explanation--

En el **discurso indirecto**, se usa `to explain` cuando alguien da una razón o aclara algo. Por ejemplo:

- Discurso directo: `The server went down because of a power issue.`

- Discurso reportado: `She explained that the server went down due to a power issue.`

Un `system component` es una parte o pieza de un sistema más grande, usualmente en computadoras o redes. Por ejemplo:

`The login module is a key system component for user access.` – Esto significa que es una de las partes esenciales del sistema.
