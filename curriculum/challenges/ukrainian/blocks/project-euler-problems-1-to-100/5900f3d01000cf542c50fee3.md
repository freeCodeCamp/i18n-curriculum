# --description--

Якщо в блоці є двадцять один кольоровий диск, що складається з п’ятнадцяти синіх дисків і шести червоних дисків, і випадковим чином взяли два диски, можна побачити, що ймовірність взяти два сині диски дорівнює

$${P(BB)} = \frac{15}{21}×\frac{14}{20} = \frac{1}{2}$$

Наступне таке розташування, для якого існує точно 50% шанс взяти два сині диски випадковим чином, — це блок, що містить вісімдесят п’ять синіх дисків і тридцять п’ять червоних дисків.

Знайдіть перше розташування, в якому загальна кількість дисків перевищує `limit`, і визначте, скільки синіх дисків міститиме цей блок.

```js
function arrangedProbability(limit) {
  // Based on https://www.mathblog.dk/project-euler-100-blue-discs-two-blue/
  let blue = 15;
  let discs = 21;

  while (discs < limit) {
    const nextBlue = 3 * blue + 2 * discs - 2;
    const nextDiscs = 4 * blue + 3 * discs - 3;

    blue = nextBlue;
    discs = nextDiscs;
  }
  return blue;
}
```
