# --description--

Quando vuoi chiedere se esistono luoghi o cose specifiche nella tua posizione attuale, puoi usare la struttura della domanda `Are there any ... around here?`. Per esempio:

`Are there any Wi-Fi spots around here?` - Stai chiedendo se puoi trovare il Wi-Fi vicino a questo posto.

La frase `around here` si riferisce all’area vicina.
