# --explanation--

`Same`는 변하지 않았거나 동일함을 의미합니다. 예를 들어:

- `We have the same task to complete this week.` - 이는 작업이 모두에게 동일함을 의미합니다.

- `They have the same opinion about the new software update.` - 이는 두 사람이 업데이트에 동의함을 의미합니다.

Bob은 `we still have the same issue`라고 말했는데, 이는 문제가 변하지 않았다는 뜻입니다.
