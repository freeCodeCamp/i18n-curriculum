# --explanation--

`Will` wird verwendet, um auf eine Anfrage zu reagieren, indem man Bereitschaft oder Absicht ausdrückt, etwas zu tun. Zum Beispiel:

- **Mary:** `Can you send me the report?`

- **Rosie:** `Sure, I'll send it right away.` – Hier zeigt `I'll send it`, dass der Sprecher der Anfrage zustimmt.

In diesem Dialog verwendet James `I'll check it out`, um zu bestätigen, dass er das Problem untersuchen wird.
