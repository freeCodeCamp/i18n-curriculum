# --interactive--

Деструктуризація об’єктів — це потужна функція в JavaScript, яка дозволяє витягувати значення з об’єктів і призначати їх змінним більш лаконічним і зрозумілим способом.

Вона є частиною специфікації ES6 (ECMAScript 2015) і стала незамінним інструментом для багатьох розробників JavaScript.

Деструктуризація може спростити ваш код, особливо коли ви працюєте зі складними об’єктами або коли потрібно одночасно витягти кілька значень.

У своїй основі деструктуризація об’єктів полягає у розпакуванні значень з об’єктів у окремі змінні. Замість того, щоб отримувати доступ до властивостей об’єкта по одній, ви можете витягнути кілька властивостей в одному рядку. Це робить ваш код чистішим і ефективнішим.

Почнемо з прикладу, щоб показати, як працює деструктуризація об’єктів:

:::interactive_editor

```js
const person = { name: "Alice", age: 30, city: "New York" };

const { name, age } = person;

console.log(name); // Alice
console.log(age);  // 30
```

:::

У цьому прикладі ми витягуємо властивості `name` і `age` з об’єкта `person` і призначаємо їх змінним з такими ж іменами.

Одна з потужних особливостей деструктуризації об’єктів — це можливість призначати витягнуті значення змінним з іншими іменами. Це особливо корисно, коли ви працюєте з об’єктами, у яких імена властивостей можуть конфліктувати з існуючими змінними, або коли хочете використати інше ім’я:

:::interactive_editor

```js
let person = { name: "Alice", age: 30, city: "New York" };

let { name: personName, age: personAge } = person;

console.log(personName); // Alice
console.log(personAge); //  30
```

:::

У цьому випадку ми витягуємо властивість `name` і призначаємо її змінній з ім’ям `personName`, а також робимо те саме з `age` і `personAge`.

Деструктуризація об’єктів також дозволяє встановлювати значення за замовчуванням. Якщо властивість не існує в об’єкті, який ви деструктуруєте, ви можете вказати запасний варіант:

:::interactive_editor

```js
let person = { name: "Alice", age: 30, city: "New York" };
let { name, age, country = "Unknown" } = person;

console.log(country); // Unknown
```

:::

Тут, оскільки `country` не існує в нашому об’єкті `person`, їй присвоюється значення за замовчуванням `Unknown`.

Ще один поширений випадок — вкладена деструктуризація об’єктів. Ви можете деструктурувати властивості, вкладені всередині інших об’єктів, використовуючи ще один набір фігурних дужок:

:::interactive_editor

```js
const recipe = {
  name: "Chocolate Cake",
  ingredients: {
    flour: "2 cups",
    sugar: "1 cup"
  }
};

// Extract `flour` from `ingredients`
const { ingredients: { flour } } = recipe;

console.log(flour); // "2 cups"
```

:::

Це еквівалентно прямому доступу до властивості:

```js
const flour = recipe.ingredients.flour;
console.log(flour); // "2 cups"
```

Тепер поговоримо про скорочений запис у деструктуризації об’єктів. Коли ви створюєте об’єкти, особливо якщо імена властивостей збігаються з іменами змінних, можна використовувати скорочений синтаксис:

:::interactive_editor

```js
let name = "Bob";
let age = 25;

let person = { name, age };

console.log(person); // { name: "Bob", age: 25 }
```

:::

Код вище бере властивості з іменами, що збігаються з нашими змінними, і призначає їм значення цих змінних.

Цей скорочений запис особливо корисний, коли ви повертаєте об’єкти з функцій або створюєте об’єкти з кількома властивостями:

:::interactive_editor

```js
function createPerson(name, age) {
  return { name, age };
}

let person = createPerson("Charlie", 35);
console.log(person); // { name: "Charlie", age: 35 }
```

:::

Деструктуризація об’єктів і скорочений запис об’єктів — це потужні функції, які можуть зробити ваш код більш лаконічним і легшим для читання.

Вони особливо корисні при роботі зі складними структурами даних або коли потрібно передати кілька параметрів у функції.

Продовжуючи працювати з JavaScript, ви знайдете багато випадків, де ці техніки спростять ваш код і зроблять його більш виразним.
