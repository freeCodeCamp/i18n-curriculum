# --interactive--

In der vorherigen Lektion wurden Sie mit CSS Grid vertraut gemacht, mit dem Sie komplexe und fließende Layouts auf Ihren Webseiten erstellen können. In dieser Lektion werden wir untersuchen, wie Sie flexible Grid-Layouts mit der `fr`-Einheit erstellen.

Beginnen wir mit diesem HTML-Auszeichnung, die unseren Grid-Container darstellt:

```html
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

Im CSS setzen wir die Eigenschaft display für den Container auf grid.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 25% 25% 25% 25%;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

`grid-template-columns` wird verwendet, um die Größe jeder Spalte festzulegen. In diesem Fall beträgt die Größe jeder Spalte `25%` des Containers. Dann wird die Eigenschaft gap verwendet, um Abstand um jede Spalte zu schaffen.

Bisher haben wir Prozentsätze für die Spaltengröße verwendet, aber wir können auch die `fr`-Einheit nutzen.

Die `fr`-Einheit ist eine Bruch-Einheit, die einen Bruchteil des Raums für den Grid-Container darstellt.

So sieht der Code aus, wenn er umgeschrieben wird, um `fr`-Einheiten anstelle von Prozentsätzen zu verwenden.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

Jede Spalte nimmt einen Bruchteil des verfügbaren Raums ein. Da es vier Spalten gibt, hat jede Spalte einen gleichen Anteil am verfügbaren Raum im Grid-Container.

Wenn Sie anfangen, Ihre Grid-Layouts zu erstellen, werden Sie feststellen, dass Sie `fr`-Einheiten häufiger verwenden möchten, weil sie eine flexible, proportionale Möglichkeit bieten, Raum zu verteilen. So können Sie responsive Layouts erstellen, die sich an unterschiedliche Bildschirmgrößen anpassen, ohne Pixelwerte manuell anpassen zu müssen.
