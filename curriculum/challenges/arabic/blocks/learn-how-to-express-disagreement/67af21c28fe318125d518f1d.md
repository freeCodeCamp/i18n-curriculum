# --explanation--

تُستخدم عبارة `I'm not sure` للتعبير عن عدم اليقين أو الشك. غالبًا ما تُستخدم هذه العبارة عندما يريد شخص التعبير عن تردد أو اختلاف بسيط بطريقة مهذبة. على سبيل المثال:

- **المتدرب:** `Should we use Python or JavaScript for this automation?`

- **المدرب:** `I'm not sure. It depends on the project requirements.` - هذا الشخص لا يعرف الخيار الأفضل بدون مزيد من التفاصيل.
