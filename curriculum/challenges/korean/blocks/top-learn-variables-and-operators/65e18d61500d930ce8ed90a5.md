# --description--

변수는 코드에서 데이터를 저장하는 "저장 컨테이너"라고 생각할 수 있습니다.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/variables-and-operators/top-learn-variables-and-operators.png" alt='the concept of variables in coding as boxes labeled "birthday" that store and provide access to data, specifically a date "16-09-2003"'>

변수는 데이터를 위한 "이름이 붙은 저장소"입니다. 변수는 간식, 방문자, 그리고 다른 데이터를 저장하는 데 사용할 수 있습니다.

JavaScript에서 변수를 만들려면 `let` 키워드를 사용하세요.

아래 문장은 "message"라는 이름의 변수를 생성(즉, 선언)합니다:

```js
let message;
```

이제 대입 연산자 `=`를 사용해 데이터를 넣을 수 있습니다:

```js
let message;

// store the string 'Hello' in the variable named message
message = 'Hello'; 
```

간단히 말해, 변수 선언과 할당을 한 줄로 합칠 수 있습니다:

```js
let message = 'Hello';
```
