# --description--

Werfen wir einen Blick auf einige schnelle Beispiele, um zu veranschaulichen, wie Spezifität funktioniert. Betrachten Sie den folgenden HTML- und CSS-Code:

```html
<!-- index.html -->

<div class="main">
  <div class="list subsection"></div>
</div>
```

```css
/* rule 1 */
.subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

Im obigen Beispiel verwenden beide Regeln nur Klassenselektoren, aber Regel 2 ist spezifischer, weil sie mehr Klassenselektoren verwendet.
