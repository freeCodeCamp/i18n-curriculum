# --description--

Para mejorar la experiencia del usuario, querrás mejorar el número de teléfono para que los usuarios hagan clic en él e inicien una llamada.

Aquí se muestra cómo puedes hacer clic en números de teléfono:

```html
<a href="tel:2345678912">234-567-8912</a>
```

Envuelve el texto `555-555-5555` en un elemento anchor y usa `tel:` para que sea un número de teléfono clickable.

# --hints--

Deberías envolver el texto `555-555-5555` en un elemento `a`.

```js
const firstPElement = document.querySelector('footer #contact address p:nth-of-type(1)');
const link = firstPElement.querySelector('a');
assert.exists(link);
assert.strictEqual(link.textContent, "555-555-5555");
```

Su elemento `a` debe tener un atributo `href`.

```js
const firstPElement = document.querySelector('footer #contact address p:nth-of-type(1)');
const link = firstPElement.querySelector('a');
assert.exists(link.getAttribute('href'));
```

El atributo `href` de su nuevo elemento `a` debe ser `tel:5555555555`.

```js
const firstPElement = document.querySelector('footer #contact address p:nth-of-type(1)');
const link = firstPElement.querySelector('a');

```
