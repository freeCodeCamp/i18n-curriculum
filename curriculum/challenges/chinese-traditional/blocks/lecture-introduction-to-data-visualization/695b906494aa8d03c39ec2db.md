# --description--

每個產業，從音樂到醫療保健，都處理龐大且常常複雜的資料集。這些資料集可能包含病患醫療記錄、隨時間變化的播放列表趨勢，或社群媒體互動指標。

為了理解這些資訊，商務會使用各種工具來幫助他們的團隊分析資料，並對他們的產品和服務做出明智的決策。

理解和傳達這些資料最強大的工具之一是**資料視覺化**。

資料視覺化將複雜的資料轉換成清晰且具視覺效果的格式。這些視覺格式的範例包括圓餅圖、散佈圖、長條圖等。良好的視覺化有助於商務節省時間，辨識資料中的關鍵資訊、趨勢和關聯。

在商務中使用許多流行的資料視覺化工具，包括 Microsoft Excel、Power BI、Google Charts 和 Tableau。

在接下來的課程和工作坊中，你將學習如何使用 `D3.js` JavaScript 函式庫創建互動式視覺化。

這個模組的目標是為你介紹資料視覺化的基本原理，並引導你使用 JavaScript 建置你自己的視覺化。
