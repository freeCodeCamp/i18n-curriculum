## Python 리스트

- **소개**: Python에서 리스트 데이터 타입은 문자열, 숫자 또는 다른 리스트로 구성될 수 있는 순서가 있는 수열입니다. 리스트는 변경 가능하며 0부터 시작하는 인덱스를 가집니다.

```python
cities = ['Los Angeles', 'London', 'Tokyo']
```

- **리스트 요소 접근**: `cities` 리스트에서 요소에 접근하려면 수열 내 인덱스 번호를 참조하면 됩니다:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # Los Angeles
```

- **음수 인덱싱을 사용한 요소 접근**: 리스트의 마지막 요소에 접근하려면 `-1`를 인덱스 번호로 사용할 수 있습니다:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # Tokyo
```

- 음수 인덱싱은 리스트의 시작 인덱스 `0` 대신 끝에서부터 요소에 접근할 때 사용합니다.

- **`list()` 생성자를 사용한 리스트 생성**: 리스트는 `list()` 생성자를 사용해 생성할 수도 있습니다. `list()` 생성자는 반복 가능한 객체를 리스트로 변환하는 데 사용됩니다:

```python
developer = 'Jessica'

print(list(developer)) 
# Result: ['J', 'e', 's', 's', 'i', 'c', 'a']
```

- **리스트 길이 구하기**: `len()` 함수를 사용해 리스트의 길이를 구할 수 있습니다:

```python
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

- **리스트 변경 가능성**: 리스트는 변경 가능하므로 유효한 인덱스 번호를 전달하면 리스트 내 어떤 요소든 업데이트할 수 있습니다. 특정 인덱스의 리스트를 업데이트하려면 해당 인덱스에 새 값을 할당하면 됩니다:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

- **인덱스 범위 초과 오류**: 리스트에 대해 범위를 벗어난 인덱스(양수 또는 음수)를 전달하면 `IndexError`가 발생합니다:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
"""
```

- **리스트에서 요소 제거**: `del` 키워드를 사용해 리스트에서 요소를 제거할 수 있습니다:

```python
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

- **리스트에 요소 존재 여부 확인**: `in` 키워드를 사용해 리스트에 특정 요소가 있는지 확인할 수 있습니다:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **리스트 중첩**: 리스트는 다른 리스트 안에 중첩될 수 있습니다:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

- 중첩된 리스트에 접근하려면 리스트가 0부터 시작하는 인덱스를 가지므로 `2` 인덱스를 사용해야 합니다.

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

- 중첩된 리스트에서 두 번째 언어에 더 접근하려면 `1` 인덱스를 사용해야 합니다:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # Rust
```

- **리스트 값 언패킹**: 리스트에서 값을 언패킹하는 기법은 리스트의 값을 새 변수에 할당하는 데 사용됩니다. 다음은 `developer` 리스트를 `name`, `age`, `job`라는 새 변수로 언패킹하는 예입니다:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer
```

- 대입 연산자 왼쪽의 변수 개수가 리스트 내 항목 수와 일치하지 않으면 `ValueError`가 발생합니다.

- **리스트에서 남은 항목 수집**: 리스트에서 남은 요소를 수집하려면 별표(`*`) 연산자를 이렇게 사용할 수 있습니다:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer
```

- **리스트 슬라이싱**: 슬라이싱은 슬라이스 연산자 `:`를 사용해 리스트의 일부를 접근하는 개념입니다. 인덱스 `1`부터 시작해 `3` 이전까지 슬라이싱하려면 다음 구문을 사용하세요:

```python
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie']
desserts[1:3] # ['Cookies', 'Ice Cream']
```

- **단계 간격**: 인덱스 간 증가량을 지정하는 단계 간격도 설정할 수 있습니다. 다음은 슬라이싱을 사용해 짝수만 추출하는 예입니다:

```python
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

## 리스트 메서드

- **append()**: 리스트 끝에 항목을 추가하는 데 사용합니다. 다음은 `append()` 메서드를 사용해 `6` 리스트에 숫자 `numbers`를 추가하는 예입니다:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **리스트 추가**: `append()` 메서드는 한 리스트를 다른 리스트 끝에 추가하는 데도 사용할 수 있습니다:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

- **extend()**: 여러 항목을 리스트 끝에 추가하는 데 사용합니다. 다음은 `6` 리스트 끝에 숫자 `8`, `10`, `numbers`를 추가하는 예입니다:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

- **insert()**: 리스트 내 특정 인덱스에 항목을 삽입하는 데 사용합니다. 다음은 `insert()` 메서드를 사용하는 예입니다:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

- **remove()**: 리스트에서 항목을 제거하는 데 사용합니다. `remove()` 메서드는 리스트에서 첫 번째로 나타나는 항목만 제거합니다:

```py
numbers = [1, 2, 3, 4, 5, 5, 5]
numbers.remove(5)

print(numbers) # [1, 2, 3, 4, 5, 5]

```

- **pop()**: 리스트에서 특정 항목을 제거하고 반환하는 데 사용합니다:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

- `pop` 메서드에 요소를 지정하지 않으면 마지막 요소가 제거됩니다.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

- **clear()**: 리스트의 모든 항목을 제거하는 데 사용합니다:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

- **sort()**: `sort()` 메서드는 리스트 내 요소를 제자리에서 정렬하는 데 사용합니다. 다음은 `numbers`의 임의 리스트를 제자리에서 정렬하는 예입니다:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

- **sorted()**: 원본 리스트를 수정하지 않고 새 정렬된 리스트를 반환하는 내장 함수입니다:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
print(numbers) # [19, 2, 35, 1, 67, 41]
```

- **reverse()**: 리스트 내 요소 순서를 반대로 뒤집는 데 사용합니다:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **index()**: 리스트에서 특정 요소가 처음 나타나는 인덱스를 찾는 데 사용합니다:

```python
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

- `index()` 메서드로 요소를 찾을 수 없으면 `ValueError`가 결과로 나옵니다.

## Python 튜플

- **정의**: 튜플은 Python 데이터 타입으로, 순서가 있는 값의 수열을 생성하는 데 사용됩니다. 튜플은 다양한 데이터 타입을 혼합해 포함할 수 있습니다:

```py
developer = ('Alice', 34, 'Rust Developer')
```

- 튜플은 변경 불가능하므로 생성 후 요소를 변경할 수 없습니다. 튜플 내 항목을 업데이트하려 하면 `TypeError`가 발생합니다:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **튜플 요소 접근**: 튜플에서 요소에 접근하려면 대괄호 표기법과 인덱스 번호를 사용하세요:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

- 음수 인덱싱을 사용해 튜플 끝에서부터 요소에 접근할 수 있습니다:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- 튜플 길이 이상이거나 같은 인덱스 번호를 전달하면 `IndexError`가 발생합니다:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- 튜플은 `tuple()` 생성자를 사용해 생성할 수도 있습니다. 생성자 내에 문자열, 리스트, 다른 튜플 같은 다양한 반복 가능한 객체를 전달할 수 있습니다.

```python
developer = 'Jessica'

print(tuple(developer)) 
# Result: ('J', 'e', 's', 's', 'i', 'c', 'a')
```

- **튜플 내 항목 확인**: 튜플에 항목이 있는지 확인하려면 `in` 키워드를 이렇게 사용하세요:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **튜플 언패킹**: 튜플에서 항목을 언패킹하는 방법은 다음과 같습니다:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

- 튜플에서 남은 요소를 수집하려면 별표(`*`) 연산자를 이렇게 사용할 수 있습니다:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer
```

- **튜플 슬라이싱**: 슬라이싱을 사용해 튜플 일부를 추출할 수 있습니다. 예를 들어 `pie`와 `cookies` 항목을 별도의 튜플로 슬라이싱할 수 있습니다:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

- **튜플에서 항목 제거**: 튜플은 변경 불가능하므로 항목을 제거하려 하면 `TypeError`가 발생합니다:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

- **튜플과 리스트 중 언제 사용해야 할까?**: 요소를 추가, 제거, 업데이트할 수 있는 동적 컬렉션이 필요하면 리스트를 사용하세요. 고정되고 변경 불가능한 데이터 컬렉션을 다룰 때는 튜플을 사용하세요.

## 튜플의 일반 메서드

- **`count()`**: 튜플 내 특정 항목이 몇 번 나타나는지 확인하는 데 사용합니다. 예를 들어 튜플에서 언어 `'Rust'`가 몇 번 나타나는지 확인할 수 있습니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

- `count()` 메서드에 지정한 항목이 튜플에 전혀 없으면 반환값은 `0`입니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

- `count()` 메서드에 인수를 전달하지 않으면 Python은 `TypeError`를 반환합니다.

- **index()**: 튜플 내 특정 항목이 있는 인덱스를 찾는 데 사용합니다. 다음은 `index()` 메서드를 사용해 언어 `'Java'`의 인덱스를 찾는 예입니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

- 지정한 항목을 찾을 수 없으면 Python은 `ValueError`를 반환합니다.

- `index()` 메서드에 선택적 시작 인덱스를 전달해 튜플 내 검색 시작 위치를 지정할 수 있습니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

- `index()` 메서드에 선택적 종료 인덱스를 전달해 검색 종료 위치를 지정할 수도 있습니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

- **`sorted()`**: 반복 가능한 객체 내 요소를 정렬해 새 정렬된 리스트를 반환하는 데 사용합니다. 다음은 `sorted()` 함수를 사용해 새 숫자 리스트를 만드는 예입니다:

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

- **정렬 동작 수정**: 선택적 `reverse`와 `key` 인수를 사용해 반복 가능한 객체의 정렬 동작을 사용자 지정할 수 있습니다. 다음은 `key` 인수를 사용해 튜플 내 항목을 길이별로 정렬하는 예입니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

- `reverse` 인수를 사용해 역순으로 정렬된 새 리스트를 만들 수도 있습니다:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

## Python 반복문

- **정의**: 반복문은 코드 블록을 정해진 횟수만큼 반복 실행하는 데 사용합니다.

- **`for` 반복문**: 수열(리스트, 튜플, 문자열 등)을 순회하며 각 항목마다 코드 블록을 실행합니다. 다음은 `for` 반복문을 사용해 리스트를 순회하며 각 언어를 콘솔에 출력하는 예입니다:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)

"""
Result 

Rust
Java
Python
C++
"""
```

- 다음은 `for` 반복문을 사용해 문자열 `code`를 순차적으로 반복하며 각 문자를 출력하는 예입니다:

```py
for char in 'code':
    print(char)

"""
Result 

c
o
d
e
"""
```

- `for` 반복문은 중첩할 수 있습니다. 다음은 중첩된 `for` 반복문을 사용하는 예입니다:

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)

"""
Result

Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
"""

```

- **`while` 반복문**: 조건이 `False`일 때까지 코드 블록을 반복 실행합니다. 다음은 `while` 반복문을 사용한 추측 게임 예입니다:

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')

"""
Result

Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
"""

```

- **`break`와 `continue` 문**: 반복문 내 실행 흐름을 수정하는 데 사용합니다.

- `break` 문은 특정 조건이 충족되면 즉시 반복문을 종료하는 데 사용합니다. 다음은 `break` 문을 사용해 `developer_names` 리스트를 처리하는 예입니다:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

- `continue` 문은 현재 반복을 건너뛰고 다음 반복으로 넘어가는 데 사용합니다. 다음은 `continue` 문 대신 `break` 문을 사용하는 예입니다:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

- `for`와 `while` 반복문은 `else` 절과 함께 사용할 수 있으며, 이 절은 `break`에 의해 반복문이 종료되지 않았을 때만 실행됩니다:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

## 반복문에서의 범위와 사용법

- **`range()` 함수**: 정수 수열을 생성하는 데 사용합니다.

```py
range(start, stop, step)
```

- 필수 `stop` 인수는 생성할 수열의 끝점을 나타내는 정수(비포함)입니다. 다음은 `range()` 함수를 사용하는 예입니다:

```py
for num in range(3):
    print(num)
```

- `start` 인수를 지정하지 않으면 기본값은 `0`입니다. 기본적으로 정수 수열은 `1`씩 증가합니다. 선택적 `step` 인수를 사용해 기본 증가값을 변경할 수 있습니다. 다음은 2부터 11 미만(즉, 10 포함)까지 짝수 정수 수열을 생성하는 예입니다:

```py
for num in range(2, 11, 2):
    print(num)
```

- `range()` 함수에 인수를 전달하지 않으면 `TypeError`가 반환됩니다.

- `range()` 함수는 인수로 정수만 허용하며 실수를 사용하면 `TypeError`가 발생합니다:

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- `step` 인수에 음수를 사용해 감소하는 정수 수열을 생성할 수 있습니다:

```py
for num in range(40, 0, -10):
    print(num)
```

- `range()` 함수는 `list` 생성자와 함께 사용해 정수 리스트를 생성할 수도 있습니다. `list` 생성자는 반복 가능한 객체를 리스트로 변환하는 데 사용됩니다. 다음은 2부터 10까지(포함) 짝수 정수 리스트를 생성하는 예입니다:

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

## Python의 `enumerate()`와 `zip()` 함수

- **`enumerate()`**: 수열을 순회하며 각 항목의 인덱스를 추적하는 데 사용합니다. `enumerate()` 함수는 반복 가능한 객체를 인수로 받아 인덱스와 값으로 구성된 `enumerate` 객체를 반환합니다.

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')

# Result
# Index 0 and language Spanish
# Index 1 and language English
# Index 2 and language Russian
# Index 3 and language Chinese

```

- `enumerate()` 함수는 `for` 반복문 밖에서도 사용할 수 있습니다:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

print(list(enumerate(languages)))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

- `enumerate()` 함수는 선택적 `start` 인수를 받아 카운트 시작 값을 지정합니다. 이 인수를 생략하면 카운트는 `0`부터 시작합니다.

- **`zip()`**: 여러 반복 가능한 객체를 병렬로 순회하는 데 사용합니다. 다음은 `zip()` 함수를 사용해 `developers`와 `ids`를 순회하는 예입니다:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')


"""
Result

Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
"""

```

## Python 리스트 내포

- **정의**: 리스트 내포는 대괄호 안에 반복문과 조건을 직접 결합해 한 줄로 새 리스트를 만드는 방법입니다. 코드를 더 짧고 읽기 쉽게 만듭니다.

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

## 반복 가능한 객체 메서드

- **`filter()`**: 조건에 따라 반복 가능한 객체에서 요소를 필터링하는 데 사용합니다. 조건을 만족하는 요소만 포함하는 이터레이터를 반환합니다. 다음은 네 글자 이상인 단어만 새 리스트로 만드는 예입니다:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**: 반복 가능한 객체의 각 항목에 함수를 적용해 결과가 담긴 새 반복 가능한 객체를 반환합니다. 다음은 `map()` 함수를 사용해 섭씨 온도를 화씨로 변환하는 예입니다:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**: 리스트나 튜플 같은 반복 가능한 객체의 합계를 구하는 데 사용합니다. 다음은 `sum()` 함수를 사용하는 예입니다:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- 선택적 `start` 인수를 전달해 합계의 초기값을 설정할 수도 있습니다. 다음은 `start` 인수를 위치 인수로 사용한 예입니다:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- `start` 인수를 키워드 인수로 사용하는 방법도 있습니다:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## 람다 함수

- **정의**: Python의 람다 함수는 이름 없는 간결한 함수(익명 함수)를 만드는 방법입니다. 
- 람다 함수는 종종 다른 함수의 인수로 사용됩니다. 다음은 람다 함수의 예입니다:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

- 람다 함수 사용 모범 사례는 변수에 할당하지 않고, 간단하고 읽기 쉽게 유지하며, 짧고 일회성 함수에 사용하는 것입니다.
