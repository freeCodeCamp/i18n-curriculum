# --explanation--

A frase completa é `Hola, Camila. ¿Cuál es tu número de teléfono?`

Esta pergunta segue uma estrutura que você já viu anteriormente:

`Qual é o seu` + noun?

Aqui, a frase nominal é `número de teléfono`.

Tanto `número` quanto `teléfono` têm acentos, que são importantes para a ortografia e pronúncia corretas.

A frase é usada para solicitar o número de telefone de alguém de forma educada e direta.
