# --explanation--

El tiempo `Past Perfect` (`had` + verbo en la forma de `past participle`) se usa para hablar de una acción que ocurrió antes de otro evento pasado. Muestra qué acción ocurrió primero. Por ejemplo:

`The server had crashed before the backup started.` – El fallo ocurrió primero, luego comenzó la copia de seguridad.

La estructura `Third Conditional` (`if` + verbo en el `past perfect` como la causa hipotética, verbo modal + `have` + verbo en la forma de `past participle` como la consecuencia hipotética) expresa una condición pasada irreal y su posible, pero irreal, resultado. Habla sobre cómo las cosas podrían ser diferentes si se hubieran tomado otros cursos de acción. Por ejemplo:

`If they had updated the software, they could have avoided the error.` – Esto significa que la actualización no ocurrió y que el error no se evitó.
