# --description--

对于正整数 $k$，定义 $d(k)$ 为 $k$ 在其通常十进制表现中的数字之和。因此 $d(42) = 4 + 2 = 6$。

对于正整数 $n$，定义 $S(n)$ 为满足以下性质的正整数 $k &lt; {10}^n$ 的数量：

- $k$ 可被 23 整除，且，
- $d(k) = 23$。

已知 $S(9) = 263\\,626$ 和 $S(42) = 6\\,377\\,168\\,878\\,570\\,056$。

查找 $S({11}^{12})$ 并给出你的答案 $\bmod {10}^9$。
