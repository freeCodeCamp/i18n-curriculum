# --description--

의미론은 언어에서 단어나 구의 의미를 뜻합니다. HTML도 하나의 언어로서, 각 요소는 고유한 의미론적 의미를 가집니다. 사실, HTML 문서를 텍스트 문서처럼 생각할 수 있습니다. 텍스트 문서처럼 제목, 이미지, 굵게 표시된 텍스트, 그리고 다른 서식이 있을 수 있습니다.

요소의 의미론적 의미는 그 요소가 전달하는 특별한 정보를 말합니다. 예를 들어 `p` 요소의 의미론적 의미는 텍스트 단락입니다:

```html
<p>
  Let me tell you about my fantastic holiday in Paris.
  I saw the impressive Eiffel Tower up close!
</p>
```

대부분의 요소는 의미론적 의미를 가집니다. `div` 요소는 의미론적 의미가 없는 아주 몇 안 되는 요소 중 하나입니다. 그런데 왜 이것이 중요할까요?

무엇보다도, 올바른 의미론적 HTML을 사용하면 화면 읽기 프로그램 같은 보조 기술을 사용하는 사용자에게 최상의 경험을 보장할 수 있습니다. 또한 의미론적 HTML은 검색 순위를 향상시킬 수 있습니다. 이를 검색 엔진 최적화, 즉 SEO라고 합니다.

마지막으로, 올바른 의미론적 요소를 사용하면 개발 경험도 향상됩니다. 네비게이션 바를 찾기 위해 많은 개발 코드를 뒤질 필요 없이 `nav` 요소를 직접 편집하며 무엇을 변경하는지 알 수 있습니다. 이 섹션 전체에서 이러한 주제, 의미론적 요소 사용법, 그리고 의미론적 HTML이 왜 중요한지 더 배우게 될 것입니다.
