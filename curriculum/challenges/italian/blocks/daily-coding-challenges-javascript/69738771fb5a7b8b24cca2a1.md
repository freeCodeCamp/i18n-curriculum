# --description--

Dato un array dei tuoi log di accesso, determina se hai raggiunto il tuo obiettivo di disintossicazione digitale.

Ogni log è una stringa nel formato `"YYYY-MM-DD HH:mm:ss"`.

Hai raggiunto il tuo obiettivo di disintossicazione digitale se entrambe le seguenti istruzioni sono vere:

- Hai effettuato l'accesso non più di una volta in un periodo di quattro ore.
- Hai effettuato l'accesso non più di 2 volte in un singolo giorno.

```js
function digitalDetox(logs) {
  if (logs.length === 0) return true;

  const times = logs
    .map(log => new Date(log.replace(" ", "T") + "Z").getTime())
    .sort((a, b) => a - b);

  const ONE_DAY = 24 * 60 * 60 * 1000;
  const FOUR_HOURS = 4 * 60 * 60 * 1000;

  for (let i = 1; i < times.length; i++) {
    if (times[i] - times[i - 1] < FOUR_HOURS) {
      return false;
    }
  }

  const dailyCounts = {};
  for (const time of times) {
    const day = new Date(time).toISOString().slice(0, 10);
    dailyCounts[day] = (dailyCounts[day] || 0) + 1;
    if (dailyCounts[day] > 2) {
      return false;
    }
  }

  return true;
}
```
