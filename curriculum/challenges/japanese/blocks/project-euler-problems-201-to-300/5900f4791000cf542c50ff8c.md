# --description--

多項式 $P(x)$ の根または零点とは、方程式 $P(x) = 0$ の解のことです。

$n$ の各桁の数字を係数とする多項式を $P_n$ と定義します。

例えば、$P_{5703}(x) = 5x^3 + 7x^2 + 3$ です。

次のことがわかります：

- $P_n(0)$ は $n$ の一の位の数字、
- $P_n(1)$ は $n$ の各桁の数字の和、
- $P_n(10)$ は $n$ 自身。

$Z(k)$ を、多項式 $P_n$ が少なくとも1つの整数根を持つような、$k$ 以下の正の整数 $n$ の個数と定義します。

$Z(100\,000)$ は 14696 であることが確認できます。

では、$Z({10}^{16})$ はいくつでしょうか？
