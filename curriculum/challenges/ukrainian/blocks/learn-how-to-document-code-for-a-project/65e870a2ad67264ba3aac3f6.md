# --description--

У цьому завданні ви дізнаєтеся різницю між `simple` і `simply`, а також зрозумієте, що означають `double` і `slashes`.

`Simple` — це прикметник, що означає легкий або нескладний, як `a simple task`. `Simply` — це прислівник, що означає просто або легко, як `doing something simply`.

`Double` означає два чогось. А `slashes` — це `/` символи, які використовуються у письмі. Ви додаєте `es`, щоб зробити `slash` у множині, тому що слова, що закінчуються на `sh`, утворюють множину додаванням `es`, як у `dishes` або `brushes`.
