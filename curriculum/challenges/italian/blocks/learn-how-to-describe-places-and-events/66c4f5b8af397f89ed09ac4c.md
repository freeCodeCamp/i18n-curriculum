# --explanation--

Immagina di scrivere codice e di voler trovare rapidamente la funzione o la variabile giusta senza dover digitare tutto.

`Auto-completion` aiuta suggerendo ciò che probabilmente stai cercando di scrivere in base a quello che hai già digitato. Questo rende la scrittura del codice più veloce e riduce gli errori.
