# --description--

- 再帰は、基本ケースに到達するまで関数を繰り返し呼び出すことができるプログラミングの概念です。

以下は、数の階乗を計算する再帰関数の例です。

```js
function findFactorial(n) {
  if (n === 0) {
    return 1;
  }
  return n * findFactorial(n - 1);
}
```

上の例では、`findFactorial` 関数が `n` が `0` に達するまで再帰的に呼び出されます。`n` が `0` のとき、基本ケースに到達し関数は `1` を返します。その後、関数は `n` と `findFactorial(n - 1)` への再帰呼び出しの結果の積を返します。

- 再帰を使うと、深くネストされたオブジェクトや配列、またはファイルツリーのような未知の深さのものを扱うことができます。
- 再帰関数の呼び出しを追跡するためにコールスタックが使われます。関数が呼び出されるたびにコールスタックに追加され、基本ケースに到達すると関数呼び出しはスタックから削除されます。
- 基本ケースは慎重に定義する必要があります。無限に呼び出すとコードがクラッシュする可能性があります。これは再帰がどんどん関数呼び出しを積み重ね、システムのメモリが不足するためです。
- 再帰は階乗やフィボナッチのような数学的問題の解決、ツリーやグラフの走査、順列や組み合わせの生成などに使われます。
