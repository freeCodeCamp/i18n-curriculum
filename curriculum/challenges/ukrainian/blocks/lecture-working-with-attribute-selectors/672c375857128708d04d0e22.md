# --interactive--

Працюючи з впорядкованими списками в HTML, атрибут `type` дозволяє вказати стиль нумерації, наприклад числовий, алфавітний або римські цифри.

Атрибут `type` можна встановити на елементі `ol` (впорядкований список), щоб керувати форматом нумерації:

- `1` для числових списків (наприклад, 1, 2, 3, ...),

- `A` для алфавітних списків у верхньому регістрі (наприклад, A, B, C, ...),

- `a` для алфавітних списків у нижньому регістрі (наприклад, a, b, c, ...),

- `I` для римських цифр у верхньому регістрі (наприклад, I, II, III, ...),

- `i` для римських цифр у нижньому регістрі (наприклад, i, ii, iii, ...).

Ось як ви можете використати селектор атрибутів, щоб цілитися у впорядкований список з певним значенням атрибута `type`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="A"] {
  color: purple;
  font-weight: bold;
}
```

:::

У цьому прикладі правило CSS застосує фіолетовий колір тексту та жирний шрифт до будь-якого впорядкованого списку, який використовує алфавітну нумерацію у верхньому регістрі, що вказано `type="A"`.

Аналогічно, ви можете цілитися у впорядковані списки, які використовують римські цифри у нижньому регістрі, вибираючи ті, що мають `type="i"`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="i">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="i"] {
  color: green;
}
```

:::

Це правило змінить колір тексту на зелений для будь-якого впорядкованого списку, який використовує римські цифри у нижньому регістрі.

Можливість стилізувати впорядковані списки на основі їх атрибута `type` дає вам більший контроль над виглядом списків у вашому документі, дозволяючи кастомізувати їх залежно від структури вмісту.
