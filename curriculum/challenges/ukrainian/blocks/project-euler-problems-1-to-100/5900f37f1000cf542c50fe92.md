# --description--

Вам надано таку інформацію, але ви можете захотіти провести власне дослідження.

<ul>
  <li>1 січня 1900 року був понеділок.</li>
  <li>Вересень має тридцять днів,<br>квітень, червень і листопад.<br>Всі інші мають тридцять один,<br>окрім лютого,<br>який має двадцять вісім, незалежно від погоди.<br>А в високосні роки — двадцять дев’ять.</li>
  <li>Високосний рік трапляється кожного року, що ділиться на 4 без остачі, але не в століття, якщо тільки воно не ділиться на 400.</li>
</ul>

Скільки неділь припало на перше число місяця протягом двадцятого століття (з 1 січня 1901 до 31 грудня 2000)?

```js
function countingSundays(firstYear, lastYear) {
  let sundays = 0;

  for (let year = firstYear; year <= lastYear; year++) {
    for (let month = 0; month <= 11; month++) {
      const thisDate = new Date(year, month, 1);
      if (thisDate.getDay() === 0) {
        sundays++;
      }
    }
  }
  return sundays;
}
```
