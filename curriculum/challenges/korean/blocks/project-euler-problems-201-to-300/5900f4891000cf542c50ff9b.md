# --description--

10진법에서 3자리 숫자 376은 제곱했을 때 끝자리가 자기 자신과 같은 특별한 속성을 가진 숫자의 예입니다: ${376}^2 = 141376$. 이런 속성을 가진 숫자를 고정 제곱수라고 부릅니다.

고정 제곱수는 다른 진법에서도 관찰할 수 있습니다. 14진법에서 3자리 숫자 $c37$ 역시 고정 제곱수입니다: $c37^2 = aa0c37$, 그리고 그 자리수 합은 같은 진법에서 $c+3+7=18$입니다. 여기서 문자 $a$, $b$, $c$, $d$는 각각 10, 11, 12, 13을 나타내며, 16진법과 비슷한 방식으로 사용됩니다.

$1 ≤ n ≤ 9$에 대해, 14진법에서 모든 $n$자리 고정 제곱수의 자리수 합은 $2d8$ (10진수로 582)입니다. 앞자리가 0인 고정 제곱수는 허용하지 않습니다.

$1 ≤ n ≤ 10000$ (10진수) 범위에서 14진법의 모든 $n$자리 고정 제곱수의 자리수 합을 구하고, 필요한 경우 소문자 문자를 사용하여 14진법 문자열로 답을 제출하세요.
