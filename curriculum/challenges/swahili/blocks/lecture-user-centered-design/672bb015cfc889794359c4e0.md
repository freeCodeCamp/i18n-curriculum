# --interactive--

**KUMBUKA**: Baadhi ya mifano ya mwingiliano inaweza kutumia CSS na JavaScript ambazo bado hujajifunza. Usijali kuhusu kuelewa msimbo wote. Lengo la mifano ni kukuonyesha mapitio ya dhana hizi za kubuni ili uelewe vyema jinsi vitu vinavyofanya kazi. Ili kuona mapitio, utahitaji kuwezesha mhariri wa mwingiliano.

Modal ni nini? Ni aina ya dirisha la pop-up ambalo tovuti inaweza kukuonyesha juu ya yaliyomo yake. HTML ina kipengele cha `dialog` ambacho unaweza kutumia kuunda modals.

Yaliyomo nyuma ya modal kawaida huwa na mwangaza mdogo. Hii husaidia mtumizi kuwa makini kwa kuona eneo unalotaka wafanye kazi nalo – katika kesi hii, modal.

Daima ni wazo zuri kuruhusu mtumizi kubofya nje ya modal kufunga dirisha hilo.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button id="open-modal">Open Modal</button>
<dialog>
  <h2>Subscribe to our Newsletter!</h2>
  <p>Get the latest updates and offers.</p>
  <button>Subscribe</button>
  <button>Close</button>
</dialog>

<script src="index.js"></script>
```

```css
dialog {
  border: none;
  border-radius: 8px;
  padding: 20px;
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
}

dialog::backdrop {
  background: rgba(0, 0, 0, 0.5);
}
```

```js
const dialog = document.querySelector('dialog');
const closeButton = dialog.querySelector('button:last-of-type');
const openModalButton = document.getElementById('open-modal');

closeButton.addEventListener('click', () => {
  dialog.close();
});

openModalButton.addEventListener('click', () => {
  dialog.showModal();
});

// Close the modal when clicking outside of it
dialog.addEventListener('click', (event) => {
  const rect = dialog.getBoundingClientRect();
  const isInDialog = (
    event.clientX >= rect.left &&
    event.clientX <= rect.right &&
    event.clientY >= rect.top &&
    event.clientY <= rect.bottom
  );
  if (!isInDialog) {
    dialog.close();
  }
}); 

```

:::

Mara nyingi utaona vifungo vinavyoonekana sana kwenye modals. Hivi huitwa CTAs, au call-to-action. Unataka vifahamike kwa urahisi kwa sababu kusitisha mtiririko wa mtumizi kwa modal ni kumshawishi kuchukua hatua fulani.

Modals pia zinapaswa kuwa na kitufe cha kufunga. Ingawa unaweza kweli kutaka mtumizi kubofya CTAs zako, ni muhimu kuwapa chaguo la kutoka kwenye modal na kuendelea na kile walichokuwa wakifanya awali.
