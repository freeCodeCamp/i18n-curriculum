# --explanation--

`Haven't` — це скорочення від `have not`, яке використовується у `Present Perfect` часі, щоб показати, що дія не відбулася.

`Negative` означає щось несприятливе або критичне. Наприклад:

`The user left a negative review about the app's performance.` — тут `negative` описує несприятливий характер відгуку.

`Comment` — це думка або відгук, який хтось надає. Його множина — `comments`. Наприклад:

`I received a comment suggesting more features in the app.` — у цьому контексті `comment` означає відгук, який користувач залишив про застосунок.
