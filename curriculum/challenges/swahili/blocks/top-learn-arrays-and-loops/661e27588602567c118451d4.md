# --description--

Moja ya njia ngumu zaidi zinazotumika na safu za data ni njia za `splice()` na `slice()`. Njia ya `splice()` hubadilisha yaliyomo katika safu ya data kwa kuondoa au kubadilisha kipengele katika safu hiyo. Njia ya `slice()` hurudisha nakala ya sehemu ya safu ya data katika kitu kipya cha safu ya data kilichochaguliwa kuanzia `begin` hadi `end` (`end` haijumuishwi). Safu ya asili haitabadilishwa.

Kwa mfano, kuondoa kipengele cha pili kutoka katika safu ya `characters`, unaweza kutumia msimbo ufuatao:

```javascript
const characters = ['Harry', 'Ron', 'Hermione'];
characters.splice(1, 1);
console.log(characters); // Output: ['Harry', 'Hermione']
```

Kipengele kilicho hapo juu kinaondoa kipengele cha pili kutoka katika safu ya `characters`. Njia ya `splice()` inachukua hoja mbili: kielezo cha kipengele cha kuondoa na idadi ya vipengele vya kuondoa.


Ili kuunda safu mpya yenye kipengele cha pili kutoka katika safu ya `character`, unaweza kutumia msimbo ufuatao:

```javascript
const characters = ['Harry', 'Ron', 'Hermione'];
const newCharacters = characters.slice(1, 2);
console.log(newCharacters); // Output: ['Ron']
```

Msimbo huo hapo juu huunda safu mpya `newCharacters` yenye kipengele cha pili kutoka katika safu ya `characters`. Njia ya `slice()` inachukua hoja mbili: kielezo cha kipengele cha kuanza kukata na kielezo cha kipengele cha kuishia kukata (haijumuishwi).
