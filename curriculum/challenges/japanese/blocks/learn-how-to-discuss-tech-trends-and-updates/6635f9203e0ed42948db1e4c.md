# --description--

`Low-code development` は、最小限の手書きコードでソフトウェアアプリケーションを作成する方法です。大量のコードを書く代わりに、開発者はドラッグアンドドロップ機能を備えたグラフィカルユーザーインターフェースを使ってアプリケーションを組み立て、設定します。この方法により開発が速くなり、プログラミングスキルがあまりないユーザーにも特に役立ちます。
