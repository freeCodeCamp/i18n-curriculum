# --interactive--

CSS에서 `list-style` 속성은 웹페이지에서 목록의 모양을 제어하는 데 사용됩니다.

순서 있는 목록(`ol`)이든 순서 없는 목록(`ul`)이든, `list-style` 속성으로 목록 항목이 어떻게 표시되는지 사용자 지정할 수 있습니다.

`list-style` 속성은 실제로 세 가지 다른 속성의 단축 속성입니다:

- `list-style-type`
- `list-style-position`
- `list-style-image`

각각은 목록 모양을 정의하는 데 서로 다른 역할을 합니다.

`list-style-type` 속성은 목록에서 사용되는 글머리 기호나 숫자의 유형을 정의할 수 있게 해줍니다.

순서 없는 목록에서는 원형, 동그라미, 사각형 등 여러 글머리 기호 스타일 중에서 선택할 수 있습니다.

순서 있는 목록에서는 십진수, 로마 숫자, 또는 알파벳 문자 같은 다양한 번호 매기기 방식을 사용할 수 있습니다.

다음은 `list-style-type`를 사용하는 예입니다:

:::interactive_editor

```html
<ul style="list-style-type: square;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

이 예제에서는 순서 없는 목록의 글머리 기호가 사각형으로 변경되었습니다.

`list-style-type` 속성은 세 가지 중 가장 자주 사용되며, 목록의 글머리 기호나 번호 매기기 스타일의 모양에 직접 영향을 줍니다.

`list-style-position` 속성은 글머리 기호나 번호가 목록 항목 내용과 어떻게 위치하는지 제어합니다. 사용할 수 있는 값은 `inside`와 `outside` 두 가지입니다.

값 `outside`를 사용하면 글머리 기호나 번호가 내용 바깥에 나타나며, 이는 기본 동작입니다.

값 `inside`를 사용하면 글머리 기호나 번호가 내용 안에 나타나 텍스트가 줄 바꿈되고 글머리 기호나 번호와 정렬될 수 있습니다.

다음은 `list-style-position`를 사용하는 예입니다:

:::interactive_editor

```html
<ul style="list-style-position: inside;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
<ul style="list-style-position: outside;">
  <li>Item 4</li>
  <li>Item 5</li>
  <li>Item 6</li>
</ul>
```

:::

이 예제에서는 두 개의 서로 다른 순서 없는 목록 태그에 대해 `inside`와 `outside` 값이 제공되었습니다.

`list-style-position` 속성은 특히 하나의 목록 항목에 여러 줄 텍스트가 있을 때 목록 내용의 정렬을 제어하는 데 유용할 수 있습니다.

`list-style-image` 속성은 목록 항목의 글머리 기호로 이미지를 사용할 수 있게 해줍니다. 이는 목록에 독특한 시각적 스타일을 추가할 때 유용합니다.

다음은 `list-style-image`를 사용하는 예입니다:

:::interactive_editor

```html
<head>
  <style>
    ul {
      list-style-image: url('https://cdn.freecodecamp.org/platform/universal/freecodecamp-org-gravatar.jpeg');
      list-style-position: inside;
    }
  </style>
</head>
<body>
  <ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
  </ul>
</body>
```

:::

이 예제에서는 글머리 기호가 사용자 지정 freeCodeCamp 로고로 교체되어 목록에 개성을 더했습니다.

`list-style-image`를 사용할 때는 선택한 이미지가 작고 웹페이지 디자인에 적합한지 확인하세요. 이미지가 너무 크거나 복잡하면 목록을 읽기 어렵게 만들 수 있습니다.

세 가지 속성인 `list-style-type`, `list-style-position`, `list-style-image`를 하나의 `list-style` 단축 속성으로 결합할 수 있습니다.

단축 속성에서 값의 순서는 중요하지 않지만 세 가지 모두 함께 지정할 수 있습니다.

다음은 단축 속성을 사용하는 예입니다:

:::interactive_editor

```html
<ul style="list-style: square inside url('https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg');">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

이 예제에서는 목록 항목이 사각형 글머리 기호를 사용하고, 내용 안에 위치하며, 사용자 지정 이미지가 글머리 기호로 사용됩니다.

하지만 이미지가 없거나 렌더링되지 않으면 사각형 글머리 기호가 대체 수단으로 사용됩니다.
