# --interactive--

`grid-column` と `grid-row` プロパティは、グリッドレイアウト内でグリッドアイテムの水平および垂直の配置を指定できます。

つまり、どちらもグリッドラインを参照してグリッドアイテムの開始位置と終了位置を制御できます。このグリッドラインは、すでに `grid-template-rows` と `grid-template-columns` プロパティで定義した行と列を区切る境界線です。

`grid-row` と `grid-column` の構文と実際の例を見てみましょう。

まずは `grid-row` プロパティの構文です：

```css
grid-row: <start-line> / <end-line>;
```

次に `grid-column` の構文です：

```css
grid-column: <start-line> / <end-line>;
```

`<start-line>` はアイテムが始まるグリッドラインで、`<end-line>` はアイテムが終わるグリッドラインです。どちらも1から数え始める1始まりのインデックスです。

行のグリッドラインは `grid-template-rows` プロパティで指定した行数に基づいて生成されることを覚えておいてください。列についても `grid-template-columns` プロパティで同様です。

実例として、`grid-template-rows` と `grid-template-columns` の両方を使ったときにブラウザが生成するラインから始めましょう。

4列3行のグリッドの例です：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}
```

:::

グリッドコンテナ（クラス `grid`）を調べると、各行と列は2本のラインで区切られていることがわかります。行や列の始まりにある開始ラインと、終わりにある終了ラインです。

これらのラインをターゲットにして、`grid-row` と `grid-column` プロパティを使い、アイテムをどこに配置するか決められます。

最初のグリッドアイテムを最初の2列にまたがらせる方法は次の通りです：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
}
```

:::

これで最初のグリッドアイテムは列1から始まり、列3の直前で終わるように指定したことになります。

最初のアイテムは2列分を占め、4番目のアイテムは2行目に押し出されます。すごいですね！

もし最初のアイテムを2行分にまたがらせたい場合は、`grid-row` に `1 / 3` を指定してください：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
  grid-row: 1 / 3;
}
```

:::

また、`span` キーワードを使ってグリッドアイテムがまたがる行数や列数を指定することもできます。例えば、`1 / 3` は `1 / span 2` と同じ意味です：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / span 2;
  grid-row: 1 / span 2;
}
```

:::

この手法はグリッド上のどのアイテムにも適用でき、好きな場所に配置できます。

続けて、この方法でアイテムを使ったメイソンリーレイアウトを作った例です。
