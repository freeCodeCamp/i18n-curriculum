# --description--

ルナは次の構文を使ってスペイン語で自分の職業を述べています：

`Soy` + 職業。

動詞`ser`はスペイン語で最も重要な動詞の一つです。**身元**、**職業**、および/または**国籍**を表現するために使われます。例えば：

- `Soy amable.` - 私は親切です。

- `Soy mexicana.` - 私はメキシコ人です。

- `Soy profesor.` - 私は教師です。

# --explanation--
 
`Soy`は動詞`ser`の一人称単数形で、身元、職業、または国籍について話すときに使われます。

`Soy científica de datos`はその人がデータサイエンティストとして働いていることを意味します。ほかの例をいくつか紹介します：

- `Soy ingeniero de software.` - 私はソフトウェアエンジニアです。（男性形）

- `Soy uruguaya.` - 私はウルグアイ人です。（女性形）
