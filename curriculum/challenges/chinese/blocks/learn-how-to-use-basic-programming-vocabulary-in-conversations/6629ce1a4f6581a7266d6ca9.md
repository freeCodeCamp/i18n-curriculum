# --description--

**现在完成进行时** 用于描述从过去开始并且现在仍在发生的动作或情况。当你想强调一个活动的持续时间或它是一个正在进行的进程时，它非常有用。

以下是构成**现在完成进行时**的方式：主语 + `has/have been` + 动词的 `-ing` 形式。该时态通常与表示时间的表达式如 `for` 和 `since` 一起使用。`For` 用于表示一段时间，如两年或六个月，而 `since` 用于表示具体时间点，如 2018 年或今天早上。

示例：

`I have been studying English for two years.` 这意味着我在两年前开始学习英语，并且现在仍在学习。

`She has been working here since 2018.` 意味着她从 2018 年开始在这里工作，并且仍然在这里工作。
