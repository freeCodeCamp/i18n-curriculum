# --interactive--

JavaScript에서 텍스트를 다루는 것은 코딩의 필수적인 부분이며, 종종 여러 텍스트 조각을 합치거나 연결해야 합니다. 이 과정을 문자열 연결이라고 합니다.

이번 수업에서는 문자열 연결이 어떻게 작동하는지, 특히 `+` 연산자, `+=` 연산자, 그리고 `concat()` 메서드를 사용하는 방법에 집중하겠습니다.

`+` 연산자는 문자열을 연결하는 가장 간단하고 자주 사용되는 방법 중 하나입니다. 여러 문자열을 합치거나 텍스트를 담고 있는 변수와 문자열을 결합할 수 있습니다.

예를 들어 보겠습니다:

:::interactive_editor

```js
let firstName = "John";
let lastName = "Doe";

let fullName = firstName + " " + lastName; 
console.log(fullName); // John Doe
```

:::

이 예제에서는 `+` 연산자를 사용해 `firstName`와 `lastName` 변수를 공백(`" "`)과 함께 연결하여 전체 이름을 만들었습니다.

`+` 연산자를 사용해 문자열을 연결할 때 한 가지 단점은 연결된 문자열 사이의 공백을 신경 써서 관리하지 않으면 공백 문제가 발생할 수 있다는 점입니다.

공백이 빠진 예는 다음과 같습니다:

:::interactive_editor

```js
let firstName = "John";
let lastName = "Doe";

let fullName = firstName + lastName; 
console.log(fullName); // "JohnDoe"
```

:::

문자열을 연결할 때 `+` 연산자를 사용할 경우, 공백 문제 여부를 반드시 다시 확인하세요.

기존 문자열에 추가하거나 덧붙여야 할 때는 `+=` 연산자를 사용할 수 있습니다. 시간이 지나면서 문자열에 더 많은 텍스트를 추가해 나갈 때 유용합니다.

`+=` 연산자를 사용해 한 문자열을 다른 문자열에 덧붙이는 예는 다음과 같습니다:

:::interactive_editor

```js
let greeting = 'Hello';
greeting += ', John!';

console.log(greeting); // "Hello, John!"
```

:::

문자열은 불변이라는 점을 기억하는 것이 중요합니다. 즉, 문자열이 한 번 생성되면 변경할 수 없습니다.

이 경우 원래 문자열 `Hello`는 수정되지 않고, greeting은 새 문자열 `Hello, John!`을 참조하게 됩니다.

문자열을 연결하는 또 다른 방법은 `concat()` 메서드를 사용하는 것입니다.

`concat()` 메서드에 대해 배우기 전에, 메서드와 함수가 무엇인지 더 높은 수준에서 이해하는 것이 중요합니다.

프로그래밍에서 함수는 특정 작업을 수행하고 다양한 입력으로 호출할 수 있는 재사용 가능한 코드 블록입니다. 반면 메서드는 객체와 연관된 함수의 한 종류로, 해당 객체 내 데이터를 대상으로 작동합니다.

앞으로 수업에서 JavaScript의 함수, 객체, 메서드가 어떻게 작동하는지 더 깊이 다룰 예정입니다. 지금은 JavaScript에 `concat()` 메서드처럼 사용할 수 있는 수십 가지 메서드가 있다는 점을 이해하는 것이 중요합니다.

두 문자열을 연결하기 위해 `concat()` 메서드를 사용하는 예는 다음과 같습니다:

:::interactive_editor

```js
let str1 = 'Hello';
let str2 = 'World';

let result = str1.concat(' ', str2); 
console.log(result); // Hello World
```

:::

이 예제에서는 `concat()` 메서드를 사용해 `str1`, 공백(`' '`), 그리고 `str2`을 하나의 문자열로 합쳤습니다.

요약하자면, `+` 연산자는 몇 개의 문자열이나 변수를 결합할 때 간단한 연결에 가장 적합합니다.

`+=` 연산자는 문자열을 단계별로 만들거나 기존 문자열 변수에 새 내용을 덧붙일 때 유용합니다.

마지막으로 `concat()` 메서드는 여러 문자열을 함께 연결해야 할 때 유용합니다.
