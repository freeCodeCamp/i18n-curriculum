# --description--

이 작업은 능력을 표현하는 데 사용하는 `can`와 과거 진행형 시제의 동사 `try` 사용을 검토하는 데 포커스를 맞춥니다.

`Can`는 `I can solve this problem.`처럼 누군가가 무언가를 할 수 있는 능력에 대해 말할 때 사용합니다.

과거 진행형(`Try`/`was` + `were` 형태의 동사)인 `-ing`는 `I was trying to fix the issue.`처럼 과거에 진행 중이던 동작을 나타냅니다.
