# --explanation--

`Built-in` 指的是作為軟體程式或裝置一部分，從一開始就包含的特性或工具，無需另外添加或安裝。 

當某些東西是 `built-in` 時，它已經存在且可立即使用。 例如：

- `This laptop has a built-in camera。`－相機是筆記型電腦的零件。

- `The software comes with built-in security features.`－安全特性已經包含在軟體中。
