# --description--

Fast alle Elemente auf einer HTML-Seite sind einfach Inhaltsstücke, die in öffnende und schließende HTML-Tags eingeschlossen sind.

Öffnende Tags sagen dem Browser, dass hier ein HTML-Element beginnt. Sie bestehen aus einem Schlüsselwort, das in spitze Klammern `<>` eingeschlossen ist. Zum Beispiel sieht ein öffnendes Absatz-Tag so aus: `<p>`.

Schließende Tags sagen dem Browser, wo ein Element endet. Sie sind fast identisch mit den öffnenden Tags; der einzige Unterschied ist, dass sie einen Schrägstrich vor dem Schlüsselwort haben. Zum Beispiel sieht ein schließendes Absatz-Tag so aus: `</p>`.
