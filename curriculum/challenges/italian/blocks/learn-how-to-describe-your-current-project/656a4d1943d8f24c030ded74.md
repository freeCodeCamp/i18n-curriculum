# --description--

Quando qualcuno usa `either` alla fine di una frase negativa, significa che la stessa situazione vale anche per lui. Per esempio:

`I don't like carrots. My sister doesn't like them either.` - Qui, `either` indica che sia il parlante sia sua sorella non apprezzano le carote.
