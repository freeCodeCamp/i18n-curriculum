# --explanation--

`First off` si usa per introdurre la prima cosa di cui si parlerà o che si farà. Indica l'inizio di una lista o di un ordine di argomenti su cui concentrarsi. Per esempio:

- `First off, we need to address the project's scope.` – Significa che affrontare l'ambito del progetto è la prima cosa che verrà fatta.

- `First off, let's review the current goals before we start planning.` – Significa che rivedere gli obiettivi attuali è la prima cosa che succederà prima di passare alla pianificazione.
