# --explanation--

`It` використовується для заміни іменника, який уже згадувався, щоб уникнути повторення в реченні. Наприклад:

- `I bought a new laptop. It works really well.` - тут `it` замінює `laptop`, щоб не повторювати слово.

У `The Live Preview isn't showing in Visual Studio Code. Isn't it supposed to support that feature?` `it` позначає `Visual Studio Code`, тому що Том говорить про програму та її здатність підтримувати функцію.

`It` замінює `Visual Studio Code`, щоб не повторювати назву програми.

Визначивши, що очікується підтримка функції, ви можете зрозуміти, що `it` позначає саму програму.
