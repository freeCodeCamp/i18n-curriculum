# --description--

In diesem Teil des Dialogs schlägt Sarah vor, `Gartic` beim nächsten Spieleabend zu spielen. Sie verwendet `maybe` und `we can`, um die Möglichkeit auszudrücken, dass dies passieren könnte.

`Maybe` ist ein Wort, das verwendet wird, wenn Sie eine Möglichkeit vorschlagen, aber nicht sicher sind, ob diese Möglichkeit eintreten wird oder nicht. Es zeigt, dass Sie keinen festen Plan machen, sondern offen für die Idee sind. Zum Beispiel: Wenn Sie gefragt werden, ob Sie ins Kino gehen möchten, können Sie `Maybe` sagen, wenn Sie sich nicht sicher sind, ob Sie gehen wollen oder verfügbar sind.

`We can` wird verwendet, um eine Idee vorzuschlagen oder etwas höflich anzubieten. Es zeigt, dass die Personen im Gespräch die Fähigkeit oder Option haben, etwas zu tun. Zum Beispiel: Wenn Sie vorschlagen möchten, ein Spiel zu spielen, können Sie `We can play a game` sagen, um die Idee anzubieten.

Der Unterschied zwischen `maybe` und `we can` besteht darin, dass `maybe` eine Möglichkeit vorschlägt, ohne einen festen Plan zu machen, während `we can` eine Idee oder einen Vorschlag auf höfliche Weise ausdrückt.
