### --question--

#### --text--

Welches der folgenden Beispiele ist ein Beispiel für Rekursion?

#### --distractors--

```js
function factorial(n) {
  let result = 1;

  while (n > 0) {
    result = result * n;
    n--;
  }

  return result;
}
```

---

```js
function factorial(n) {
  const arr = Array(n).fill().map((_, i) => i + 1);
  return arr.reduce((acc, curr) => acc * curr, 1);
}
```

---

```js
function factorial(n) {
  let result = 1;

  for (let i = n; i > 0; i--) {
    result *= i;
  }

  return result;
}
```

#### --answer--

```js
function factorial(n) {
  if (n === 0 || n === 1) {
    return 1;
  }
  return n * factorial(n - 1);
}
```

### --question--

#### --text--

Warum muss Rekursion einen Basisfall haben?

#### --distractors--

Um sicherzustellen, dass die Funktion immer einen Rückgabewert hat.

---

Um die Anzahl der Funktionsaufrufe zu reduzieren.

---

Um sicherzustellen, dass alle rekursiven Aufrufe in der richtigen Reihenfolge ausgeführt werden und kein `RecursionError` geworfen wird.

#### --answer--

Um der Funktion eine Möglichkeit zu geben, aus ihren rekursiven Aufrufen auszubrechen und unendliche Schleifen zu verhindern.
