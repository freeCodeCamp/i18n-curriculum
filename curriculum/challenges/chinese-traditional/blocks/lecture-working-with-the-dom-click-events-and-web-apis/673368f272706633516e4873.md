# --interactive--

你可能熟悉在你的 HTML 和 CSS 專案中使用屬性。但你知道你可以在你的 JavaScript 檔案中為你的 HTML 元素添加屬性嗎？

在本課程中，你將學習如何使用 `setAttribute()` 方法為你的 HTML 元素添加屬性。

這是基本語法：

```js
setAttribute(attribute, value);
```

讓我們來看幾個範例，以更好地理解如何使用 `setAttribute()` 方法。

在這個第一個範例中，我們在 HTML 裡有一個 `p` 元素：

```html
<p id="para">I am a paragraph</p>
```

要為 `class` 屬性添加，我們首先需要使用 `getElementById()` 方法存取該 `p` 元素。然後我們可以使用 `setAttribute()` 方法添加 `class` 屬性並將值設為 `my-class`。

我們使用 `outerHTML` 來記錄整個 HTML 元素及其屬性。

:::interactive_editor

```html
<p id="para">I am a paragraph</p>
<script src="index.js"></script>
```

```js
const para = document.getElementById("para");
para.setAttribute("class", "my-class");

console.log(`${para.outerHTML}`);
```

:::

如果你的 HTML 元素上已有現有的屬性，你可以使用 `setAttribute()` 方法更新其值。

在此範例中，我們有一個 `div` 元素，其 `class` 屬性設定為 `my-class`：

```html
<div class="my-class"></div>
```

要將 `class` 屬性值更新為 example，我們可以再次使用 `setAttribute()` 方法：

:::interactive_editor

```html
<div class="my-class"></div>
<script src="index.js"></script>
```

```js
const divEl = document.querySelector(".my-class");
divEl.setAttribute("class", "example");

console.log(`${divEl.outerHTML}`);
```

:::

那麼，什麼是真實世界中使用 `setAttribute()` 方法的範例呢？

如果你正在建置一個動態影像畫廊，當使用者點擊縮圖時，你可能需要更新影像元素的 `src` 屬性。

另一個例子是如果你正在處理表單驗證，並且需要為 `input` 元素添加像 `required` 或 `minlength` 這樣的某些屬性。
