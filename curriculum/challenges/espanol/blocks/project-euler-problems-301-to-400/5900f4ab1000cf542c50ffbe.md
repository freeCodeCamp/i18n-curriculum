# --description--

Sea $x_1, x_2, \ldots, x_n$ una secuencia de longitud $n$ tal que:

- $x_1 = 2$
- para todo $1 < i ≤ n : x_{i - 1} < x_i$
- para todos $i$ y $j$ con $1 ≤ i, j ≤ n : {(x_i)}^j &lt; {(x_j + 1)}^i$

Solo hay cinco secuencias de este tipo de longitud 2, a saber: {2,4}, {2,5}, {2,6}, {2,7} y {2,8}. Hay 293 secuencias de este tipo de longitud 5; se dan tres ejemplos a continuación: {2,5,11,25,55}, {2,6,14,36,88}, {2,8,22,64,181}.

Sea $t(n)$ el número de tales secuencias de longitud $n$. Se te da que $t(10) = 86195$ y $t(20) = 5227991891$.

Encuentra $t({10}^{10})$ y da tu respuesta módulo $10^9$.
