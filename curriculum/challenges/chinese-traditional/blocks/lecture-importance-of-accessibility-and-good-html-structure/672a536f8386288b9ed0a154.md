# --description--

語音辨識軟體幫助身心障礙者與電腦及其他數位裝置互動。讓我們討論什麼是語音辨識軟體以及它在數位包容中扮演的角色。

在無障礙的背景關係中，語音辨識工具讓身心障礙者使用他們的聲音來傳達指令以執行各種任務，而非使用傳統的輸入裝置如鍵盤和滑鼠。這包括撰寫電子郵件和其他文件、瀏覽網路，以及控制智慧家庭裝置。

因為語音辨識軟體工具消除了對實體互動的需求，它們賦予身心障礙者對環境的重大獨立性和控制元件。 

以下是可能會發現語音辨識軟體特別有幫助的特定人士：

- 視覺障礙者，包括低視力或失明者。
- 行動不便者，例如手部和手臂使用受限，或患有關節炎和腕隧道症候群等狀況。
- 那些正在從手部或手臂受傷中恢復的人。
- 有認知障礙的人，例如記憶體問題或注意力缺陷障礙。
- 可能會覺得使用語音指令較為容易的長者。

請注意，使用語音辨識技術的不僅僅是身心障礙者。執法機關、遊戲玩家、駕駛員和忙碌的專業人士也會使用語音辨識工具。

一些允許人們與他們的電腦互動的語音辨識軟體範例包括 macOS/iOS 的 Voice Control、Android 的 Voice Access，以及 Windows 的 Windows Speech Recognition（在最新版本的 Windows 中稱為 Voice Access）。Nuance 的 Dragon 是 Windows 上受歡迎的第三方語音辨識軟體。
