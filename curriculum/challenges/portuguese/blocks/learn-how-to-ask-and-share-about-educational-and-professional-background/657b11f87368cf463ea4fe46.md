# --description--

Questions in the `Past Simple` tense with the verb `to be` are very similar to how you do it in the `Present Simple` tense. A única diferença é que você precisa lembrar de conjugar o verbo para `were` ou `was`, seguido pelo sujeito e depois o verbo principal. Por exemplo:
 
- `Were you at the meeting yesterday?` - Note como `were` vem antes de `you` para formar uma pergunta. 

- `What was my assignment?` - Para uma pergunta específica, observe como `was` vem depois de `what` e antes de `my`. 

# --questions--

## --answers--

`Where there any specific projects you'd like to mention?`

### --feedback--

`Where` é usado para locais. A forma correta para perguntar sobre eventos passados é `were`.

---

`Were there any specific projects you'd like to mention?`

---

`Was there any specific projects you'd like to mention?`

### --feedback--

`Was` é usado com substantivos no singular. Como `projects` está no plural, `were` é a escolha correta.

---

`Are there any specific projects you'd like to mention?`

### --feedback--

`Are` está no tempo presente. A questão é sobre projetos passados, então `were` é necessário.
