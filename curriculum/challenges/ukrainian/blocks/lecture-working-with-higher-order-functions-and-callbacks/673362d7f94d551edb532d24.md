# --interactive--

Існує багато різних способів сортування даних у програмуванні. У цьому уроці ми зосередимося на вбудованому методі `sort` у JavaScript.

Метод `sort` використовується для впорядкування елементів масиву і повертає референс на відсортований масив. Копія не створюється, оскільки елементи сортуються на місці.

Ось базовий синтаксис методу `sort`:

```js
array.sort(compareFunction);
```

Параметр `compareFunction` є необов’язковим і задає функцію, яка визначає порядок сортування. Пізніше ми розглянемо, як використовувати функцію порівняння при сортуванні чисел.

У цьому першому прикладі у нас є масив рядків у випадковому порядку.

```js
const fruits = ["Banana", "Orange", "Apple", "Mango"];
```

Наша мета — відсортувати масив в алфавітному порядку. Це можна зробити, викликавши метод `sort` для масиву `fruits`.

:::interactive_editor

```js
const fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.sort();

console.log(fruits); // ["Apple", "Banana", "Mango", "Orange"]
```

:::

Результатом буде відсортований масив фруктів в алфавітному порядку, починаючи з фрукта `Apple`.

У наступному прикладі ми хочемо відсортувати наступний масив чисел:

```js
const numbers = [414, 200, 5, 10, 3];
```

Якщо ми спробуємо використати метод `sort` для цього масиву `numbers`, отримаємо несподівані результати.

:::interactive_editor

```js
const numbers = [414, 200, 5, 10, 3];
numbers.sort();

console.log(numbers); // [10, 200, 3, 414, 5]
```

:::

Ми очікували побачити результат `[3, 5, 10, 200, 414]`, але натомість отримали `[10, 200, 3, 414, 5]`.

Це тому, що метод `sort` перетворює елементи на рядки, а потім порівнює їх послідовності значень кодових одиниць UTF-16.

Кодові одиниці UTF-16 — це числові значення, які представляють символи в рядку. Прикладами кодових одиниць UTF-16 є числа `65`, `66` і `67`, які представляють символи `A`, `B` і `C` відповідно.

Отже, число `200` з’являється перед числом `3` у масиві, тому що рядок `200` йде перед рядком `3` при порівнянні їх кодових одиниць UTF-16.

Рішення цієї проблеми — надати функцію порівняння методу `sort`.

Ось приклад, як відсортувати масив `numbers` за допомогою функції порівняння:

:::interactive_editor

```js
const numbers = [414, 200, 5, 10, 3];

numbers.sort((a, b) => a - b);

console.log(numbers); // [3, 5, 10, 200, 414]
```

:::

Параметри `a` і `b` — це два елементи, які порівнюються. Функція порівняння повинна повертати від’ємне значення, якщо `a` має йти перед `b`, додатне значення, якщо `a` має йти після `b`, і нуль, якщо `a` і `b` рівні.

Перше порівняння — між числами `414` і `200`. Результат `414 - 200` — `214`, що є додатним значенням. Це означає, що `414` має йти після `200` у відсортованому масиві.

Наступне порівняння — між числами `200` і `5`. Результат `200 - 5` — `195`, що є додатним значенням. Це означає, що `200` має йти після `5` у відсортованому масиві.

Ми повторюємо цей процес для всіх елементів масиву, і результатом є відсортований масив чисел.

Хоча існує багато інших способів сортування даних у програмуванні, метод `sort` у JavaScript може бути корисним і ефективним у багатьох випадках, коли потрібно відсортувати масив елементів.
