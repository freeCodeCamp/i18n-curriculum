# --description--

تعرف أعداد فيبوناتشي $\{f_n, n ≥ 0\}$ بشكل تكراري كالتالي: $f_n = f_{n - 1} + f_{n - 2}$ مع الحالات الأساسية $f_0 = 0$ و $f_1 = 1$.

عرف كثيرات الحدود $\{F_n, n ≥ 0\}$ كالتالي: $F_n(x) = \displaystyle\sum_{i = 0}^n f_ix^i$.

على سبيل المثال، $F_7(x) = x + x^2 + 2x^3 + 3x^4 + 5x^5 + 8x^6 + 13x^7$، و $F_7(11) = 268\,357\,683$.

لتكن $n = {10}^{15}$. احسب مجموع $\displaystyle\sum_{x = 0}^{100} F_n(x)$ وقدم إجابتك باقي القسمة على $1\,307\,674\,368\,000 \, (= 15!)$.
