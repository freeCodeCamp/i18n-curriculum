# --description--

`str.maketrans()` メソッドは、同じ長さの2つの文字列を受け取り、最初の文字列の各文字を2番目の文字列の対応する文字にマッピングする変換テーブルを返します。変換テーブルの各文字は、文字を一意に識別する番号であるUnicodeの序数として格納されます。

```py
lower_chars = 'abc'
upper_chars = 'ABC'

table = str.maketrans(lower_chars, upper_chars)
print(table) # {97: 65, 98: 66, 99: 67}
```

カリキュラムの後半で`str.maketrans()`が返す構造の種類について詳しく学びます。今は、`alphabet`の文字を`shifted_alphabet`の文字にマッピングする変換テーブルを作成し、`translation_table`という名前の変数に代入してください。
