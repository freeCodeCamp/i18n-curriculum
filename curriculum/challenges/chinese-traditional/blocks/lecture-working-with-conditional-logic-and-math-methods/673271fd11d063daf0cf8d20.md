# --interactive--

條件述語讓你能在你的 JavaScript 程式碼中做出決策。它們允許你的程式根據特定條件以特定方式流程。讓我們來看看 `if`、`else if`、`else` 和三元運算子如何運作，讓你控制你的程式碼流程。

`if` 述語會接受一個條件，並在該條件為真值時執行一個程式碼區塊。真值是指在布林值背景關係中（例如 `if` 述語）評估後結果為 `true` 的任何值。以下是真值的範例：

- 非空字串，例如 `hello`

- 任何非 `0` 和 `-0` 的數字，例如 `4`、`-5` 和其他。

- 陣列

- 物件

- 布林值 `true`

另一方面，偽值是在布林值背景關係中會被評估為 `false` 的值。JavaScript 有少數偽值，使它們容易記住。以下是一些偽值：

- 布林值 `false`

- `0`（零）

- `""` (空字串)

- `null`

- `undefined`

- `NaN`（非數字）

現在，我們對 truthy 和 falsy 值有了基本的了解，讓我們來看看它如何與 `if` 述語一起運作。在這個第一個範例中，我們使用了幾個 `if` 述語來檢查 truthy 和 falsy 值：

:::interactive_editor

```js
if (null) {
  console.log("This will not run.");
}

if ("freeCodeCamp") {
  console.log("This will run.");
}
```

:::

由於 `null` 是一個假值，區塊內的訊息永遠不會被記錄到主控台。但對於第二個 `if` 述語，字串 `freeCodeCamp` 是一個真值，並且在這個 `if` 述語的布林值背景關係中會被視為 `true`。因此，訊息 `This will run.` 將會被記錄到主控台。

讓我們來看看更多關於 `if` 述語如何與不同比較運算子一起運作的範例。以下是一個使用 `if` 述語來檢查使用者是否有資格投票的範例： 

:::interactive_editor

```js
const age = 22;

if (age >= 18) {
 console.log("You're eligible to vote"); // You're eligible to vote
}
```

:::

在這個範例中，因為 `age` 目前是 `22`，這表示條件會評估為 `true`，因為 `22` 大於或相等於 `18`。所以 `You're eligible to vote` 這則訊息會被記錄到主控台。如果我們將範例改成 `age` 現在是 `15`，那麼條件會評估為 `false`，且該訊息不會被記錄到主控台：

:::interactive_editor

```js
const age = 15;

if (age >= 18) {
 console.log("You're eligible to vote"); // Code not running because age is less than 18
}
```

:::

當條件為 `false` 時，你可以使用 `else` 語句：

:::interactive_editor

```js
const age = 15;

if (age >= 18) {
 console.log("You're eligible to vote");
} else {
 console.log("You're not eligible to vote"); // You're not eligible to vote
}
```

:::

在這個範例中，`15` 不大於或相等於 `18`，所以條件會是 `false`。在這種情況下，`else` 區塊內的程式碼會被執行。

如果你想檢查多個條件，你可以使用 `else if` 區塊。這讓你的程式能在多於兩條路徑之間做選擇。

:::interactive_editor

```js
const score = 87;

if (score >= 90) {
 console.log('You got an A'); 
} else if (score >= 80) {
 console.log('You got a B'); // You got a B
} else if (score >= 70) {
 console.log('You got a C');
} else {
 console.log('You failed! You need to study more!');
}
```

:::

由於 `score` 目前是 `87`，因此 `You got a B` 的訊息會被記錄到主控台。

三元運算子是一種簡潔的方式來撰寫簡單的 `if/else` 述語。它有三個零件：一個條件、條件為真時的結果，以及條件為假時的結果。以下是基本語法：

```js
condition ? expressionIfTrue : expressionIfFalse;
```

這裡有一個處理攝氏溫度的範例： 

:::interactive_editor

```js
const temperature = 30;
const weather = temperature > 25 ? 'sunny' : 'cool';

console.log(`It's a ${weather} day!`);
```

:::

如果 `temperature` 大於 `25`，上述程式碼會記錄 `It's a sunny day!`。如果 `temperature` 小於或相等於 `25`，它會記錄 `It's a cool day!`。

那麼，你應該在 `if` 述語和三元運算子之間選擇哪一個呢？當處理單一條件或單一表達式，或當你想要簡潔的語法來實現簡單邏輯時，使用三元運算子。當你處理複雜條件和多個述語時，使用 `if/else` 述語，因為如果你巢狀使用三元運算子，程式碼會變得難以閱讀。
