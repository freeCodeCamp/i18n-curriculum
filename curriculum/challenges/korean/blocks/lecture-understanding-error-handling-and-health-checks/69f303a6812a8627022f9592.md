# --description--

웹사이트를 방문하거나 API를 사용할 때, 브라우저와 서버는 끊임없이 서로 대화하고 있습니다.

그 대화에서 가장 중요한 부분 중 하나가 바로 HTTP 응답 상태 코드입니다.

이들은 200, 404, 500 같은 짧은 숫자 메시지로, 클라이언트(노트북이나 휴대폰 같은 최종 사용자 기기)에게 요청 후 어떤 일이 일어났는지 알려줍니다.

이미 몇 가지를 배웠지만, 이번에는 카테고리별로 나누어 정확히 무슨 뜻이고 언제 사용하는지 기억하기 쉽게 정리해 보겠습니다.

## 1xx – 정보 응답

1로 시작하는 코드는 일상적인 개발에서는 잘 보이지 않지만, 요청이 수신되어 처리 중임을 나타냅니다. 예를 들어 파일 업로드 중일 때 그렇습니다.

일반적인 예:

* 100 Continue – 서버가 클라이언트에게 "좋아요, 나머지 요청을 보내도 됩니다"라고 알립니다.
* 101 Switching Protocols – 클라이언트가 `Upgrade` 헤더를 보내서 서버가 프로토콜을 전환합니다. 주로 Websockets를 사용할 때 볼 수 있습니다.

## 2xx – 성공 응답

요청이 성공적으로 수신, 이해, 처리되었음을 의미합니다.

* 200 OK – 요청이 성공했습니다. 가장 흔한 성공 응답입니다.
* 201 Created – 새 리소스가 성공적으로 생성되었습니다. 예를 들어 새 사용자를 등록한 후입니다.
* 204 No Content – 요청은 성공했지만, 응답 본문을 보낼 내용이 없습니다. 주로 `DELETE` 요청에 사용됩니다.

## 3xx – 리다이렉션 메시지

클라이언트에게 "찾는 리소스가 다른 곳에 있습니다"라고 알려줍니다.

* 301 Moved Permanently – 리소스가 새 영구 URL로 이동했습니다.
* 302 Found – 리소스가 임시로 다른 URL에 있습니다.
* 304 Not Modified – 캐싱에 사용됩니다. 클라이언트에게 변경된 내용이 없으니 캐시된 버전을 사용하라고 알립니다.

## 4xx – 클라이언트 오류

클라이언트가 잘못된 요청을 보냈을 때 발생합니다.

* 400 Bad Request – 요청이 잘못되었거나 유효하지 않습니다.
* 401 Unauthorized – 인증이 필요합니다(로그인이 필요함).
* 403 Forbidden – 인증은 성공했지만 권한이 없습니다.
* 404 Not Found – 요청한 리소스가 존재하지 않습니다. 네, 웹에서 가장 상징적인 오류입니다.

## 5xx – 서버 오류

서버 측에서 문제가 발생했음을 의미합니다.

* 500 Internal Server Error – 일반적인 "문제가 발생했습니다" 메시지입니다.
* 502 Bad Gateway – 프록시 역할을 하는 서버가 다른 서버로부터 잘못된 응답을 받았습니다.
* 503 Service Unavailable – 서버가 일시적으로 과부하 상태이거나 유지 관리 중입니다.

이제 가장 흔한 HTTP 상태 코드를 익혔으니, Express에서 실제 예제를 살펴보겠습니다.

API를 만든다고 가정해 보겠습니다. 사용자가 잘못된 데이터를 보내면 400 상태 코드를 응답할 수 있습니다. 이는 "잘못된 요청"을 의미합니다. 클라이언트가 400 응답 코드가 무엇을 뜻하는지 명확히 알 수 있도록 "Bad Request" 문자열도 함께 보낼 수 있습니다:

```js
res.status(400).send("Bad Request");
```

모든 것이 잘 처리되면 200 응답을 보낼 수 있습니다. 원한다면 메시지도 포함할 수 있습니다. 여기서는 간단한 "Success" 메시지를 담은 JSON을 보내고 있습니다:

```js
res.status(200).json({ message: "Success" });
```

서버에서 문제가 발생하면 500 상태 코드를 보내서 내부 서버 오류가 발생했음을 알리고, 오류에 대한 명확한 메시지도 함께 보낼 수 있습니다:

```js
res.status(500).send("Internal Server Error");
```

이 상태 코드는 단지 기계용이 아닙니다. 개발자가 문제를 디버그하는 데 도움을 주고, 사용자(및 다른 개발자)가 무슨 일이 일어나고 있는지 이해하도록 돕습니다.
