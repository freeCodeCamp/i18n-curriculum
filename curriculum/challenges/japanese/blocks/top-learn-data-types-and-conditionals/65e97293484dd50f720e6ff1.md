# --description--

条件文を比較するもう一つの方法は三項演算子です。三項演算子は`if-else`文を短く書く方法です。条件を評価し、その結果に基づいて値を返すために使われます。三項演算子の構文は次の通りです。

```javascript
let result = condition ? value1 : value2;
```

上の例では、条件が真の場合、`result`の値は`value1`になります。そうでなければ、`result`の値は`value2`になります。

三項演算子は条件に基づいて変数に値を代入するためによく使われます。また、条件に基づいて値を返すためにも使われます。
