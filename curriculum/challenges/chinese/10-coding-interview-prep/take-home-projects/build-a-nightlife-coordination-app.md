# --description--

Build a full stack JavaScript app that is functionally similar to this: <a href="https://yoyo44.herokuapp.com/" target="_blank" rel="noopener noreferrer nofollow">https://yoyo44.herokuapp.com/</a>. Use a site builder of your choice to complete the project.

以下是你应该为此项目实现的用户故事：

**用户故事：** 作为一个未经验证的用户，你可以与浏览我所在区域的所有酒吧。

**用户故事：** 作为经过身份验证的用户，可以将自己添加到酒吧中，表示你今晚会去那里。

**用户故事：** 作为一名经过身份验证的用户，如果我不想再去那里，可以将自己从酒吧中删除。

**用户故事：** 作为未经身份验证的用户，当我登录时，我不需要再次搜索。

**提示：** 尝试使用 Yelp API 在用户搜索的城市中找到相关地点。 如果使用 Yelp 的 API，请务必在应用中提及。

当您完成后，请确保您项目的可用演示是在某个地方公开的。 然后将 URL 提交到 Solution Link 字段中。 也可以将项目的源码链接提交到 GitHub Link 字段中。
