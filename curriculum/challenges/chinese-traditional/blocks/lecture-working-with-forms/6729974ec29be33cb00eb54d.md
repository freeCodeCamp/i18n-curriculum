# --interactive--

HTML 中的 `form` 元素用於收集使用者資訊，例如姓名和電子郵件位址。以下是一個 `form` 元素的範例：

```html
<form action="url-goes-here">
  <!-- input elements go here -->
</form>
```

`action` 屬性指定表單資料在提交後將被送往的位置。若要收集特定的資訊，例如姓名和電子郵件位址，你會使用 `input` 元素。以下是一個使用 `input` 元素的範例。

啟用互動式編輯器，並在預覽視窗中透過在欄位輸入你的名字與 `input` 元素互動。

:::interactive_editor

```html
<form action="">
  <input type="text" />
</form>
```

:::

`input` 元素是空元素，沒有結束標籤。`type` 屬性定義了預期從使用者取得的資料型別。在這個例子中，資料將是純文字。若要為 `input` 添加標籤，你會使用 `label` 元素。以下是使用帶有 `Full Name:` 文字的 `label` 元素的範例。

在預覽視窗中點擊文字 `Full Name:`，你將會看到輸入框獲得焦點。若要與範例互動，你需要啟用互動式編輯器。

:::interactive_editor

```html
<form action="">
  <label>
    Full Name:
    <input type="text" />
  </label>
</form>
```

:::

透過將 `input` 巢狀放入 `label` 元素中，你會在 `label` 和 `input` 欄位之間建立隱式關聯。「隱式」一詞指的是不需要明確說明或使用額外屬性或元素定義，即可理解或推斷的事物。若要明確地將 `label` 與 `input` 關聯，你可以使用 `for` 屬性。以下是使用 `for` 屬性為電子郵件位址標籤的範例。

在預覽視窗中透過輸入假電子郵件位址如 `jane@example.com` 與 `input` 元素互動。若要與範例互動，你需要啟用互動式編輯器。

:::interactive_editor

```html
<form action="">
  <label for="email"> Email Address: </label>
  <input type="email" id="email" />
</form>
```

:::

當使用明確的關聯時，`for` 屬性和 `id` 的值需要相同。在此情況下，兩者的值都設定為 `email`。輸入中的 `email` 型別提供了對正確格式化電子郵件位址的基本驗證。如果你想向使用者顯示有關預期輸入的額外提示，可以使用 `placeholder` 屬性。以下是使用 `placeholder` 屬性於電子郵件輸入中的範例。

啟用互動式編輯器，點擊電子郵件輸入欄並開始輸入電子郵件，你將會看到佔位文字消失。 

:::interactive_editor

```html
<form action="">
  <label for="email"> Email Address: </label>
  <input type="email" id="email" placeholder="example@email.com" />
</form>
```

:::

對於佔位符文字，你想提供有用的簡短文字來顯示你期望使用者輸入的資料格式和型別。在此情況下，佔位符文字 `example@email.com` 告訴使用者他們必須輸入格式正確的電子郵件位址。
