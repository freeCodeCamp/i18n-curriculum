# --interactive--

تستهدف محدِّدات النوع، المعروفة أيضًا بمحدِّدات العناصر، العناصر بناءً على اسم الوسم الخاص بها.

تُعد هذه المحدِّدات أساسية في CSS وتسمح لك بتطبيق الأنماط على جميع حالات عنصر HTML معين.

محدِّدات النوع سهلة الاستخدام وتُكتب ببساطة كاسم الوسم للعنصر الذي تريد تنسيقه.

إليك مثالًا على محدِّد نوع يستهدف جميع عناصر الفقرة في الصفحة:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  color: blue;
}
```

:::

في هذا المثال، ستُضبط لون نص جميع عناصر `p` إلى `blue`.

تمتلك محدِّدات النوع خصوصية منخفضة نسبيًا مقارنة بالمحدِّدات الأخرى. قيمة الخصوصية لمحدِّد النوع هي `(0, 0, 0, 1)`.

هذا يعني أن محدِّدات النوع ستتجاوزها محدِّدات الأصناف، ومحدِّدات المعرفات، والأنماط المضمَّنة، لكنها لا تزال تطبق الأنماط ما لم تكن هناك قواعد ذات خصوصية أعلى.

لنلقِ نظرة على مثال حيث تتجاوز محدِّدات الأصناف أنماط محدِّد النوع.

إليك مثالًا يحتوي على عنصرين فقرتين:

:::interactive_editor

```html
<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

:::

كلا العنصرين من نوع الفقرة يحملان صنفًا يسمى `para`.

داخل ملف CSS، يستهدف محدِّد النوع الفقرات، ويستهدف محدِّد الصنف العناصر التي تحمل صنف `para`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

```css
p {
  color: blue;
}

.para {
  color: red;
}
```

:::

جميع الفقرات في الصفحة التي تحمل صنف `para` سيكون لون نصها `red` بدلًا من `blue` لأن محدِّدات الأصناف لها خصوصية أعلى من محدِّدات النوع.
