# --description--

通常，`with` 用于表示连接或使用。例如，`I write with a pen.` 但是，在某些句子中，`with` 会出现在它所搭配的词之后。这通常发生在你想强调某人使用或处理的对象或事物时，尤其是在关于技术或工作的疑问句或陈述句中。

例如，在技术领域，你可能会说，`This program I work with is challenging.` 这里，`with` 用在句尾是为了强调这个难以使用的程序。另一个例子可能是一个问题，`What else is he working with?`，其中句尾的 `with` 是在询问正在使用的其他工具或技术。
