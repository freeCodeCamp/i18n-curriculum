# --description--

アメリカ合衆国の一般的な8時間労働日は、通常`9 AM`（午前9時、または単に`9`）から`5 PM`（午後5時、または単に`5`）までです。毎日9時に始まり5時に終わる勤務時間を持つことを表す際に、`9-to-5 job`という言い方がよく使われます。
