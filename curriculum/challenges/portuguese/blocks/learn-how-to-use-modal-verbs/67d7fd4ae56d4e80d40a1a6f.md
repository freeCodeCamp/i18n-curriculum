# --explanation--

`Figure out` é um verbo frasal que significa resolver ou entender algo que é difícil. Por exemplo:

`We need to figure out why the website isn't loading.` - Eles precisam entender o motivo pelo qual o site não está funcionando.

Um `firewall` é um recurso de segurança que protege redes de computadores contra hackers. Funciona como uma barreira digital, permitindo a entrada segura de dados e bloqueando o tráfego perigoso. Por exemplo:

`Make sure the firewall is active before you connect to the internet.` - Certifique-se de que o sistema de segurança esteja ligado antes de usar a internet.
