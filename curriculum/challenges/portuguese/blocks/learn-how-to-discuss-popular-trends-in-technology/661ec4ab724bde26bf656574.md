# --description--

`Palm vein recognition` é uma tecnologia que identifica indivíduos com base nos padrões únicos de veias nas suas palmas.Envolve capturar e analisar os padrões das veias, que são distintos para cada pessoa, fornecendo um método seguro de autenticação.

`Behavioral biometrics` significa estudar e usar comportamentos únicos como uma forma de identificar e verificar indivíduos.Inclui coisas como como você digita, como você se move ou outras ações distintas que são específicas para você.
