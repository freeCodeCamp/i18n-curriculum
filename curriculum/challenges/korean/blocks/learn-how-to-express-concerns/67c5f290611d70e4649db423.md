# --explanation--

`To draft`는 문서, 계획, 디자인 같은 무언가의 초기 버전을 만드는 것을 의미합니다. 예를 들어:

`She drafted an email to send to the client before finalizing it.` - 이는 그녀가 이메일의 초안을 작성했다는 뜻입니다.  

`schedule`는 작업이나 이벤트가 언제 진행될지 개요를 보여 주는 계획을 가리킵니다. 예를 들어:

`The project schedule shows all the deadlines for the team.` - 이는 일정표가 언제 각 작업을 완료해야 하는지 나열한다는 뜻입니다.
