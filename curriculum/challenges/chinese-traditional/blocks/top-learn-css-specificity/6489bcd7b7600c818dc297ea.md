# --description--

有時候我們可能會有彼此衝突的規則，結果導致一些意想不到的結果。「但我想要這些段落是藍色的，為什麼它們像其他段落一樣是紅色的？！」雖然這可能令人沮喪，但理解 CSS 並不會違背你的意願去做事是很重要的。CSS 只會做我們告訴它要做的事。有一個例外是瀏覽器提供的預設樣式。這些預設樣式因瀏覽器而異，這也是為什麼某些元素會在自己和其他元素之間產生很大的「間隙」，或為什麼按鈕看起來是那樣，儘管我們沒有寫任何 CSS 規則來設定它們的樣式。

所以如果你遇到像這樣的意外行為，要麼是因為這些預設樣式，要麼是不理解某個屬性如何運作，或者是不理解這個叫做 cascade 的小東西。

層疊決定了哪些規則實際套用到你的 HTML。層疊會使用不同的因素來判斷這一點。我們將檢視其中三個因素，希望能幫助你避免那些令人沮喪的「我討厭 CSS」時刻。
