# --description--

Kichaguzi cha aina (au kichaguzi cha kipengele) kitachagua vipengele vyote vya aina ya kipengele kilichotolewa, na sintaksia ni jina tu la kipengele:

```html
<!-- index.html -->

<div>Hello, World!</div>
<div>Hello again!</div>
<p>Hi...</p>
<div>Okay, bye.</div>
```

```css
/* styles.css */

div {
  color: white;
}
```

Hapa, vipengele vyote vitatu vya `<div>` vitachaguliwa, wakati kipengele cha `<p>` hakitachaguliwa.
