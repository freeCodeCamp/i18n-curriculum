# --description--

피터는 면에 1, 2, 3, 4가 적힌 4면체(피라미드형) 주사위 9개를 가지고 있습니다.

콜린은 면에 1, 2, 3, 4, 5, 6이 적힌 6면체(정육면체) 주사위 6개를 가지고 있습니다.

피터와 콜린은 주사위를 굴려 합계를 비교합니다: 합계가 더 높은 쪽이 이깁니다. 합계가 같으면 무승부입니다.

피라미드형 피터가 정육면체 콜린을 이길 확률은 얼마일까요? 답은 소수점 아래 7자리까지 반올림하여 0.abcdefg 형식으로 제출하세요.
