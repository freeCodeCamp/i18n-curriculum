# --interactive--

في JavaScript، لكل معاملات الحساب شكل تعيين مركب. معاملات التعيين المركب توفر اختصارًا موجزًا لعملية على متغير تليها تخزين النتيجة في نفس المتغير. تجمع بين العملية والتعيين في شكل أقصر مثل `x += y`، وهو ما يعادل كتابة `x = x + y` لكن بدون تكرار اسم المتغير. بدلًا من كتابة شيء مثل هذا:

:::interactive_editor

```js
let num = 5;
num = num + 2;

console.log(num); // 7
```

:::

يمكنك كتابة شيء مثل هذا:

:::interactive_editor

```js
let num = 5;
num += 2;

console.log(num); // 7
```

:::

لاحظ كيف يجمع `num += 2` بين خطوتي الجمع والتعيين في خطوة واحدة. هذا يوفر الوقت ويقلل الفوضى في شفرتك. دعنا نتعمق أكثر في أشهر معاملات التعيين المركب في JavaScript.

كما رأيت بالفعل، يسمح لك المعامل `+=` بإضافة قيمة إلى متغير موجود. يُعرف بمعامل التعيين بالإضافة. يأخذ معامل التعيين بالإضافة القيمة الحالية للمتغير، ويضيف إليها الرقم المحدد، ثم يعيد تعيين النتيجة إلى المتغير:

:::interactive_editor

```js
let total = 10;
total += 5;

console.log(total); // 15
```

:::

كما قد تتوقع، هناك معامل تعيين للطرح يُرمز له بـ `-=`. يقوم معامل التعيين للطرح بطرح القيمة المحددة من القيمة الحالية للمتغير ويُعيد تعيين القيمة الجديدة إلى المتغير:

:::interactive_editor

```js
let score = 20;
score -= 7;

console.log(score); // 13
```

:::

لو لم تستخدم تعيين الطرح، لكنت كتبت شيئًا مثل هذا:

:::interactive_editor

```js
let score = 20;
score = score - 7;

console.log(score); // 13
```

:::

يمثل معامل التعيين للضرب بـ `*=`. يضرب القيمة الحالية للمتغير بالرقم المحدد ويُعيد تعيين النتيجة إلى المتغير:

:::interactive_editor

```js
let points = 5;
points *= 3;

console.log(points); // 15
```

:::

أخيرًا، هناك معامل تعيين للقسمة يُرمز له بـ `/=`. مثل الآخرين، يسمح لك بقسمة القيمة الحالية لمتغير على رقم تحدده، ثم يعيد تعيين النتيجة إلى المتغير:

:::interactive_editor

```js
let balance = 100;
balance /= 4;

console.log(balance); // 25
```

:::

تذكر أن هناك معامل تعيين مركب لكل معامل في JavaScript. إذًا، بالإضافة إلى الأربعة المذكورة، لدينا أيضًا:

- معامل التعيين للباقي (`%=`)، الذي يقسم متغيرًا على الرقم المحدد ويُعيّن الباقي إلى المتغير.

- معامل التعيين للأس (الأسية) (`**=`)، الذي يرفع متغيرًا إلى قوة الرقم المحدد ويُعيد تعيين النتيجة إلى المتغير.

- معامل التعيين بالعملية AND على مستوى البتات (`&=`)، الذي ينفذ عملية AND على مستوى البتات مع الرقم المحدد ويُعيد تعيين النتيجة إلى المتغير.

- معامل التعيين بالعملية OR على مستوى البتات (`|=`)، الذي ينفذ عملية OR على مستوى البتات مع الرقم المحدد ويُعيد تعيين النتيجة إلى المتغير.
