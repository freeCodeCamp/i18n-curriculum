# --explanation--

Sophie dice que ejecutar pruebas es una buena práctica porque `it helps avoid mistakes`. Esto significa que ayuda a los desarrolladores a detectar errores temprano antes de que el código sea liberado.

Ejecutar pruebas puede mostrar si algo está mal en el código, incluso si todo parece estar bien al principio. Es una parte importante de escribir software de alta calidad.
