# --description--

所有格の`'s`は、`Brian's book`のように、何かが誰かまたは何かに属していることを示します。これは、その本がブライアンに属していることを意味します。

`Dealing`は、何かを管理したり扱ったりすることを意味します。例えば、`When dealing with homework, it's important to be focused.`

プログラミングにおける`function's signature`は、関数の身分証明書のようなものです。関数の名前、必要なもの（パラメータ）、そして返すもの（`return type`）を教えてくれます。

関数の`return type`は、実行後に関数が返すもので、ちょうど自動販売機にお金を入れるとソーダが出てくるのと同じです。
