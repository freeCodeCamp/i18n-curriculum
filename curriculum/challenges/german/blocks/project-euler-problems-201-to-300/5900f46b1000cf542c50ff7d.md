# --description--

Definieren Sie $f(n)$ als die Summe der Fakultäten der Ziffern von $n$. Zum Beispiel ist $f(342) = 3! + 4! + 2! = 32$.

Definieren Sie $sf(n)$ als die Summe der Ziffern von $f(n)$. Also ist $sf(342) = 3 + 2 = 5$.

Definieren Sie $g(i)$ als die kleinste positive ganze Zahl $n$, so dass $sf(n) = i$ gilt. Obwohl $sf(342)$ 5 ist, ist auch $sf(25)$ 5, und es kann bestätigt werden, dass $g(5)$ 25 ist.

Definieren Sie $sg(i)$ als die Summe der Ziffern von $g(i)$. Also ist $sg(5) = 2 + 5 = 7$.

Außerdem kann bestätigt werden, dass $g(20)$ 267 ist und $\sum sg(i)$ für $1 ≤ i ≤ 20$ gleich 156 ist.

Wie groß ist $\sum sg(i)$ für $1 ≤ i ≤ 150$?
