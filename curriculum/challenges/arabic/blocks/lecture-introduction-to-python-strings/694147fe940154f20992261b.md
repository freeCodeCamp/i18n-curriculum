# --description--

في درس سابق، تعلمت كيف يمكن تحديد كل حرف في السلسلة النصية بواسطة فهرسه (يبدأ من الصفر)، والوصول إليه باستخدام ترميز الأقواس:

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
print(my_str[-1]) # d
```

**تقطيع السلسلة النصية** يتيح لك استخراج جزء من السلسلة أو العمل على جزء محدد منها فقط. إليك الصيغة النحوية الأساسية:

```python
string[start:stop]
```

إذا أردت استخراج أحرف من فهرس معين إلى آخر، فقط افصل بين فهارسي `start` و `stop` بنقطتين:

```python
my_str = 'Hello world'
print(my_str[1:4]) # ell
```

لاحظ أن فهرس `stop` غير شامل، لذا فإن `[1:4]` يستخرج الأحرف من الفهرس `1` وحتى، لكن دون تضمين، الحرف عند الفهرس `4`.

يمكنك أيضًا حذف فهارسي `start` و `stop`، وسيستخدم Python القيمة الافتراضية `0` أو نهاية السلسلة النصية على التوالي. على سبيل المثال، هذا ما يحدث إذا حذفت فهرس `start`:

```python
my_str = 'Hello world'
print(my_str[:7])  # Hello w
```

هذا يستخرج كل شيء من الفهرس `0` وحتى (دون تضمين) الحرف عند الفهرس `7`. وهذا ما يحدث إذا حذفت فهرس `stop`:

```python
my_str = 'Hello world'
print(my_str[8:])  # rld
```

هذا يستخرج كل شيء من الحرف عند الفهرس `8` حتى نهاية السلسلة النصية.

لاحظ أن تقطيع السلسلة النصية لا يغير السلسلة الأصلية:

```py
my_str = 'Hello world'
print(my_str[8:])  # rld
print(my_str)  # Hello world
```

يمكنك أيضًا حذف فهارسي `start` و `stop` معًا، مما يستخرج السلسلة النصية كاملة:

```python
my_str = 'Hello world'
print(my_str[:])  # Hello world
```

بجانب فهارسي `start` و `stop`، هناك معلمة اختيارية `step` تُستخدم لتحديد الزيادة بين كل فهرس في التقطيع.

إليك الصيغة النحوية لذلك:

```python
string[start:stop:step]
```

في المثال أدناه، يبدأ التقطيع من الفهرس `0`، ويتوقف قبل `11`، ويستخرج كل حرف ثاني:

```python
my_str = 'Hello world'
print(my_str[0:11:2])  # Hlowrd
```

خدعة مفيدة يمكنك تنفيذها باستخدام معلمة `step` هي عكس سلسلة نصية من خلال تعيين step إلى `-1` وترك `start` و`stop` فارغتين:

```python
my_str = 'Hello world'
print(my_str[::-1]) # dlrow olleH
```
