# --description--

這是你獲取認證所需完成的專案之一。對於這個專案，你將創建一個 Bash 腳本，將世界盃遊戲的資訊輸入到 PostgreSQL，然後查詢資料庫以取得有用的統計資料。

# --instructions--

**重要：** 在你通過所有專案測試後，將你的資料庫匯出成 `worldcup.sql` 檔案，並且將你的 `insert_data.sh` 和 `queries.sh` 檔案一併存檔，以便你完成步驟 2。虛擬機器內會有相關指示說明如何操作。

# --notes--

必要的檔案：`worldcup.sql`、`insert_data.sh`、`queries.sh`
