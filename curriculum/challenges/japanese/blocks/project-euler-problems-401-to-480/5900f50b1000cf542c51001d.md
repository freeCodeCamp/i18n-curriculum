# --description--

6174は特別な数です。桁を昇順に並べた数を、桁を降順に並べた数から引くと、$7641 - 1467 = 6174$となります。

さらに驚くべきことに、任意の4桁の数からこの並べ替えと引き算の操作を繰り返すと、最終的に6174にたどり着くか、すべての桁が同じ場合はすぐに0になります。

また、4桁未満の数でも、先頭に0を補って4桁にすれば同じことが成り立ちます。

例として、数値0837から始めると：

$$\begin{align}
  & 8730 - 0378 = 8352 \\\\
  & 8532 - 2358 = 6174
\end{align}$$

6174はカプレカ定数と呼ばれます。並べ替えと引き算を繰り返し、0かカプレカ定数に到達するまで続ける操作はカプレカルーチンと呼ばれます。

このカプレカルーチンは他の進数や桁数でも考えられます。ただし、すべての場合にカプレカ定数が存在するとは限りません。入力によってはルーチンが循環に陥ったり、異なる入力で異なる定数に到達したりする場合があります。しかし、5桁かつ基数$b = 6t + 3 ≠ 9$の場合にはカプレカ定数が存在することが示されています。

例：
基数15: ${(10, 4, 14, 9, 5)}\_{15}$
基数21: $(14, 6, 20, 13, 7)\_{21}$

5桁の基数$b$におけるカプレカ定数を$C_b$と定義します。関数$sb(i)$は以下のように定義します：

- $i = C_b$の場合、または$i$を基数$b$で表したときに5桁すべてが同じ数字の場合は0
- それ以外の場合は、基数$b$のカプレカルーチンで$C_b$に到達するまでの繰り返し回数

なお、$sb(i)$はすべての整数$i < b^5$に対して定義できます。$i$を基数$b$で表したときに5桁未満なら、先頭に0を補って5桁にしてからカプレカルーチンを適用します。

$S(b)$を$0 < i < b^5$の範囲での$sb(i)$の総和と定義します。例：$S(15) = 5\,274\,369$、$S(111) = 400\,668\,930\,299$

$2 ≤ k ≤ 300$の範囲で$S(6k + 3)$の総和を求めてください。答えは下18桁を提出してください。
