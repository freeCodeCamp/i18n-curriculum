# --description--

`function` ni seti ya maagizo ambayo hufanya zoezi maalum. `Functions` inaweza kufanya msimbo kuwa uliopangwa vizuri zaidi, unaoweza kutumika tena, na rahisi kusoma. Kwa mfano, `function` inaweza kuhesabu jumla ya nambari mbili au kuonyesha ujumbe kwa mtumizi.

Fikiria `function` kama mashine ndogo katika kiwanda; unampa ingizo fulani, hufanya kitu na ingizo hilo, kisha inakupa matokeo.
