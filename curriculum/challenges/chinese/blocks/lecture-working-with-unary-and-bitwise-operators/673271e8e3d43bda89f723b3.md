# --interactive--

一元运算符作用于单个操作数，执行类型转换、数值操作或检查某些条件等操作。 让我们来看一些常见的一元运算符及其工作原理。

一元加号运算符将操作数转换为数字。 如果操作数已经是数字，则保持不变。

:::interactive_editor

```js
const str = '42';
const strToNum = +str;

console.log(strToNum); // 42
console.log(typeof str); // string
console.log(typeof strToNum); // number
```

:::

当你想确保处理的是数值时，一元加号非常方便。 正如你可能猜到的，也有一元负号运算符。 它会对操作数的值取负。 它的工作原理与一元加号类似，只是符号会翻转。

:::interactive_editor

```js
const str = '42';
const strToNegativeNum = -str;

console.log(strToNegativeNum); // -42
console.log(typeof str); // string
console.log(typeof strToNegativeNum); // number
```

:::

逻辑非运算符由感叹号（`!`）表示，是另一种一元运算符。 它会翻转操作数的布尔值。 因此，如果操作数是 `true`，会变成 `false`，如果是 `false`，会变成 `true`。 

:::interactive_editor

```js
let isOnline = true;
console.log(!isOnline); // false

let isOffline = false;
console.log(!isOffline); // true
```

:::

按位非运算符是一种不太常用的一元运算符。 它由一个波浪号（`~`）表示，会对一个数的二进制表示进行取反操作。 计算机以二进制格式（1 和 0）存储数字。 `~` 运算符会翻转每一位，这意味着它会将所有 1 变为 0，将所有 0 变为 1。  You will learn more about binary and bits in a future lesson.

:::interactive_editor

```js
const num = 5; // The binary for 5 is 00000101

console.log(~num); // -6
```

:::

在这个例子中，`5` 被取反后变成了 `-6`，这是因为 `~``5` 的结果是 `- (5 + 1)`，等于 `-6`。 这是因为计算机使用补码来表示负数。 如果你不需要直接操作位，比如底层编程，通常很少用到按位非运算符。

`void` 关键字是一个一元运算符，用于评估表达式并返回 `undefined` 结果。

:::interactive_editor

```js
const result = void (2 + 2);

console.log(result); // undefined
```

:::

`void` 也常用于超链接，防止点击后页面跳转：

```js
<a href="javascript:void(0);">Click Me</a>
```

最后，还有你在之前课程中学过的 `typeof` 操作符。它返回其操作对象的类型，类型以字串形式表示。 

:::interactive_editor

```js
const value = 'Hello world';

console.log(typeof value); // string
```

:::
