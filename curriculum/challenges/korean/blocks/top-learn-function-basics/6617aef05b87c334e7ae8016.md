# --description--

앞서 본 것처럼, 함수는 `return` 키워드를 사용해 값을 반환할 수 있습니다. `return` 키워드는 함수에서 값을 반환할 때 사용합니다. `return` 키워드가 사용되면, 함수는 실행을 멈추고 `return` 키워드 뒤에 지정된 값을 반환합니다.

```js
function add(a, b) {
  return a + b
}

console.log(add(2, 3)); // Output: 5
```

그런데 함수 끝나기 전에 `return` 키워드가 사용되면 어떻게 될까요?
이 질문에 답하려면 다음 예제를 살펴보세요:

```js
function add(a, b) {
  if(a > 2){
    return b;
  }

  return a + b;
}

console.log(add(3, 7)); // Output: 7
``` 

위 예제에서 함수 `add`는 `a`의 값이 `2`보다 큰지 확인하는 조건문이 있습니다. 조건이 참이면 함수는 `b` 값을 반환하고 실행을 멈춥니다. 조건이 거짓이면 함수는 `a`와 `b`의 합을 반환합니다.
