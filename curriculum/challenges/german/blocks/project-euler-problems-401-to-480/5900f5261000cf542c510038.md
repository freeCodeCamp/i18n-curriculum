# --description--

Für eine ganze Zahl $M$ definieren wir $R(M)$ als die Summe von $\frac{1}{p·q}$ für alle ganzzahligen Paare $p$ und $q$, die alle folgenden Bedingungen erfüllen:

- $1 ≤ p &lt; q ≤ M$
- $p + q ≥ M$
- $p$ und $q$ sind teilerfremd.

Wir definieren außerdem $S(N)$ als die Summe von $R(i)$ für $2 ≤ i ≤ N$.

Wir können überprüfen, dass $S(2) = R(2) = \frac{1}{2}$, $S(10) ≈ 6.9147$ und $S(100) ≈ 58.2962$ gilt.

Finden Sie $S({10}^7)$. Geben Sie Ihre Antwort auf vier Dezimalstellen gerundet an.
