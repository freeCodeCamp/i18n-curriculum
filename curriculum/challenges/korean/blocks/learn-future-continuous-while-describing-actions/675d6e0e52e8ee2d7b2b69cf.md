# --explanation--

`Unusual`는 흔하지 않거나 정상적이지 않거나 예상되지 않는 것을 의미합니다. 평범하지 않은 것을 설명할 때 씁니다. 예를 들어:

`It's unusual for him to be late for a meeting.` - 사람이 늦게 도착하는 것이 정상적이거나 예상되지 않는다는 뜻입니다. 그 행동이 평범하지 않다는 것을 암시합니다.

`Isn't`는 태그 질문에서 정보를 확인하거나 점검할 때 사용합니다. `is`를 사용한 긍정문 뒤에 씁니다.
