# --interactive--

Katika mafundisho yaliyopita, tulizungumzia kidogo kuhusu jinsi ya kuunda nafasi kati ya vipande vya gridi. Lakini katika funzo hili, tutaingia kwa undani zaidi kuhusu jinsi ya kutumia sifa za `row-gap`, `column-gap` na `gap` katika mpangilio wa gridi.

Lakini kwanza tunahitaji kukagua ni nini maana ya mstari katika gridi ya CSS. Mstari ni nafasi kati ya mistari miwili ya gridi inayopakana. Mistari hii huundwa moja kwa moja unapotumia CSS Grid. Katika muktadha huu, mistari kwa kawaida inahusu safu na nguzo zinazounda mpangilio wa gridi.

Ili kuunda mapengo kati ya nguzo katika CSS Grid, unaweza kutumia sifa ya `column-gap`. Thamani zinazokubalika kwa sifa hii ni pamoja na pikseli, kitengo cha `em`, asilimia, au neno kuu la `normal`.

Ikiwa utatumia thamani ya `normal` kwa sifa ya `column-gap`, basi matokeo yatakuwa `0` kwa mipangilio ya gridi na `1em` kwa mipangilio ya safu nyingi.

Hapa kuna mfano wa alama za uandishi kwa mpangilio wa gridi yenye nguzo nne:

```html
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

Kwa CSS, tumeweka display kuwa grid na sifa ya `column-gap` kuwa pikseli 10:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  height: 100px;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.grid-container div {
  background-color: darkblue;
}
```

:::

Ikiwa tungetaka kubadilisha mfano huu kuwa na safu mbili za masanduku ya buluu na kuunda nafasi zaidi kati ya safu hizo, tunaweza kutumia sifa ya `row-gap`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  column-gap: 10px;
  row-gap: 30px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

Katika mfano huu uliobadilishwa, tunaweka sifa ya `row-gap` kuwa pikseli 30 na kubadilisha `grid-template-columns` kutumia vitengo viwili vya `1fr` badala ya vinne ili kuunda safu mbili za masanduku.

Kama sifa ya `column-gap`, thamani zinazokubalika kwa sifa ya `row-gap` zinaweza kujumuisha asilimia, `em`, na pikseli.

Ikiwa unataka kutumia sifa ya ufupisho kuunda mapengo kati ya safu na nguzo, unaweza kutumia sifa ya `gap`. Hii ni sintaksia ya msingi:

```css
gap: row-value optional-column-value;
```

Ikiwa utaeleza thamani moja kwa sifa ya `gap`, basi thamani hiyo itatumika kwa safu na nguzo zote mbili. Ikiwa utaeleza thamani mbili, basi thamani ya kwanza itatumika kwa safu na ya pili itatumika kwa nguzo:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 30px 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

Thamani zinazokubalika kwa sifa ya ufupisho ya `gap` ni pamoja na asilimia, pikseli, `em` na hata thamani za `calc()`. Lakini huwezi kutumia vitengo vya `fr` hapa.

Sifa za `row-gap`, `column-gap`, na `gap` hutoa njia zinazobadilika za kudhibiti nafasi kati ya vipande katika mpangilio wa CSS Grid. Kwa kutumia sifa hizi, unaweza kwa urahisi kuunda gridi zenye mvuto wa kuona na mapengo ya kawaida na yanayoweza kubadilishwa kati ya safu na nguzo.
