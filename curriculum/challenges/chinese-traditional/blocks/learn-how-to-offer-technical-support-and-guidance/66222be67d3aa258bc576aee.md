# --description--

表達式 `first things first` 用來強調重要的事情應該在其他任何事情之前完成。這通常在任務開始時說，以先專注於最重要的步驟。 例如，`First things first, let's review the project plan` 意味著在進行其他任務之前，先從檢視專案計畫開始。

在技術背景關係中，`environment` 通常指系統運作的一組軟體和硬體條件。 例如，`development environment` 是程式設計師撰寫和測試他們程式碼的地方。
