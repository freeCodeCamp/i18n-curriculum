# --description--

Il 2 novembre 1988 è stato rilasciato il primo grande worm su internet, che ha infettato circa il 10% dei computer connessi a internet dopo solo un giorno.

In questa sfida, ti viene dato un numero di giorni trascorsi dal rilascio di un worm su internet, e devi determinare quanti computer sono infettati seguendo queste regole:

- Al giorno 0, il primo computer è infettato.
- Ogni giorno successivo, il numero di computer infettati raddoppia.
- Ogni 3° giorno, dopo che il virus si è diffuso, viene applicata una patch che riduce del 20% il numero di computer infettati. Arrotonda il numero di computer riparati al numero intero più vicino superiore.

Per esempio, al:

- Giorno 0: 1 computer totale è infettato.
- Giorno 1: 2 computer totali sono infettati.
- Giorno 2: 4 computer totali sono infettati.
- Giorno 3: 8 computer totali sono infettati. Poi, applica la patch: 8 infettati * 20% = 1,6 riparati. Arrotonda 1,6 a 2. 8 computer infettati - 2 riparati = 6 computer totali infettati dopo il giorno 3.

Restituisci il numero totale di computer infettati dopo che è trascorso il numero di giorni indicato.

# --solutions--

```js
function infected(days) {
  let infected = 1;

  for (let day = 1; day <= days; day++) {
    infected *= 2;

    if (day % 3 === 0) {
      let patched = Math.ceil(infected * 0.2);
      infected -= patched;
    }
  }

  return infected;
}
```
