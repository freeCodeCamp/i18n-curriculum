# --explanation--

El `Zero Conditional` se usa para cosas que siempre son verdaderas cuando se cumple una condición. 

Sigue la estructura `if` + `Present Simple` (para la condición) y `Present Simple` (para el resultado). Por ejemplo:

- `If you press the power button, the computer starts.` - Presionar el botón de encendido siempre enciende la computadora.

- `If the internet is down, the website doesn't load.` - El sitio web no se abrirá si no hay conexión a internet.

En este diálogo, `if the BIOS is corrupted` es la condición, y `it prevents the computer from starting` es el resultado. Esta es una verdad general sobre cómo el BIOS corrupto afecta la computadora.
