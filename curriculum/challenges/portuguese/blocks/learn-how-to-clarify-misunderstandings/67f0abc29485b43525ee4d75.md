# --explanation--

`Not that I know of` é usado para dizer que, pelo que você sabe, algo não é verdade ou não aconteceu.É frequentemente usado para dizer educadamente `no` deixando espaço para a possibilidade de que o falante possa estar errado.Por exemplo:

- Pessoa 1: `Has the update been installed?`

- Pessoa 2：`Not that I know of.` - Isso significa que você não acredita que a atualização foi instalada, mas não tem certeza absoluta.
