# --explanation--

`Database security` означає захист баз даних від несанкціонованого доступу або атак. Наприклад:

`The team is improving database security to prevent hackers from accessing confidential data.` - Це означає, що вони роблять базу даних безпечнішою, щоб запобігти будь-якому незаконному доступу.

Джейк каже, що він `looking into` їхню `database security`, тобто він досліджує або перевіряє її безпеку.
