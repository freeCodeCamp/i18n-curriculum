# --description--

تتضمن CSS الداخلية (أو CSS المضمّنة) إضافة CSS داخل ملف HTML نفسه بدلاً من إنشاء ملف منفصل تمامًا. باستخدام الطريقة الداخلية، تضع كل القواعد داخل زوج من وسمَي `<style>` المفتوح والمغلق، ثم توضع هذه الوسوم داخل وسمَي `<head>` المفتوح والمغلق في ملف HTML الخاص بك. بما أن الأنماط تُوضع مباشرة داخل وسمَي `<head>`، فلا تحتاج بعد الآن إلى عنصر `<link>` الذي تتطلبه الطريقة الخارجية.

بجانب هذه الاختلافات، الصيغة النحوية هي نفسها تمامًا كما في الطريقة الخارجية (محدِّد، أقواس متعرجة، إعلانات):

```html
<head>
  <style>
    div {
      color: white;
      background-color: black;
    }

    p {
      color: red;
    }
  </style>
</head>
<body>...</body>
```

يمكن أن تكون هذه الطريقة مفيدة لإضافة أنماط فريدة لصفحة واحدة من موقع الويب، لكنها لا تفصل الأمور كما تفعل الطريقة الخارجية، وبناءً على عدد القواعد والإعلانات قد يجعل ملف HTML كبيرًا جدًا.
