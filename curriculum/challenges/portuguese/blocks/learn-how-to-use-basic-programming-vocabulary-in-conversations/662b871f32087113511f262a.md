# --description--

O padrão `be` seguido da forma `-ing` de um verbo é usado para falar sobre ações acontecendo no momento.

A frase `have trouble with` é usada para expressar dificuldade em lidar com um problema ou objeto específico.Por exemplo, `She is having trouble with her computer` significa que ela está enfrentando dificuldades ao usar o computador dela.
