# --explanation--

`Could` zeigt eine Möglichkeit an. Das bedeutet, er ist sich nicht sicher, schlägt aber eine mögliche Ursache vor.

`Unpatched` bedeutet, dass etwas nicht behoben oder aktualisiert wurde. Zum Beispiel:

`The system was attacked because of an unpatched vulnerability.` – Das bedeutet, dass ein bekanntes Problem im System nicht korrigiert wurde und dadurch der Angriff möglich war.
