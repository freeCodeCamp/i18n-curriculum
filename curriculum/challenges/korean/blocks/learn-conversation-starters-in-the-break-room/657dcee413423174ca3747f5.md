# --description--

`there is`를 사용해 질문을 자세히 표현하려면 `to be` 동사의 순서만 바꾸면 됩니다. 예를 들어:

| 긍정문 | 의문문 |
| --- | --- |
| `There is an ATM nearby.` | `Is there an ATM nearby?` |

이 형태는 그 영역에 장소나 물건이 있는지 확실하지 않을 때 그것에 대해 묻고 싶을 때 유용합니다.

# --instructions--

아래 문장을 완성하세요.

# --fillInTheBlank--

## --sentence--

`Hi, Maria. I'm still getting to know the area. BLANK an ATM nearby?`

## --blanks--

`Is there`
