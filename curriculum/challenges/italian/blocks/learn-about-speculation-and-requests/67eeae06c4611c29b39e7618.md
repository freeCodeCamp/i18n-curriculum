### --feedback--

Queste tre parole si usano per parlare di qualcosa che sarebbe stata una buona idea in passato ma non è stata fatta. La terza parola è il `Past Participle` di `run`.

# --explanation--

`Should have` + `Past Participle` si usa per esprimere rimpianto o un consiglio riguardo al passato. Per esempio:

- `You should have run the program on a different machine to check for compatibility.` - Testare su una macchina diversa sarebbe stato utile, ma non è stato fatto.

- `I should have run the tests before deploying the app.` - Eseguire i test era la cosa giusta da fare, ma non è stato fatto al momento.

In questo caso, Sophie sta dicendo a Mark che avrebbe dovuto eseguire la suite di test, cioè era importante, ma non l'ha fatto.
