# --explanation--

`Logs` sind Systemaufzeichnungen, die Aktivitäten in einem Netzwerk verfolgen. Zum Beispiel:

`The logs recorded a failed login attempt.` – Das bedeutet, dass die Systemaufzeichnungen zeigen, dass jemand versucht hat, sich anzumelden, aber erfolglos war.  

`IP address` ist ein eindeutiger Bezeichner für ein Gerät in einem Netzwerk. Zum Beispiel:

`The attacker used a different IP address each time.` – Der Angreifer verwendete für jeden Versuch eine eindeutige Netzwerkkennung.
