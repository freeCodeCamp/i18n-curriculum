# --interactive--

Das Superscript-Element wird verwendet, um einen Text als Hochstellung anzuzeigen. Eine Hochstellung ist ein Symbol oder Buchstabe, der oberhalb der normalen Textzeile gedruckt wird.

Hier ist ein Beispiel, das das Superscript-Element zur Darstellung von Exponenten verwendet (um die Vorschau zu sehen, aktivieren Sie den interaktiven Editor):

:::interactive_editor

```html
<p>2<sup>2</sup> (2 squared) is 4.</p>
```

:::

In diesem Beispiel ist die Zahl 2 in `sup`-Tags eingeschlossen, um die Hochstellung im Absatz darzustellen. Im Vorschaufenster sehen Sie, dass die zweite Zahl 2 kleiner und etwas höher als die erste Zahl 2 angezeigt wird.

Typische Anwendungsfälle für das Superscript-Element sind Exponenten, hochgestellte Buchstaben und Ordnungszahlen. Hier ist ein Beispiel, das das Superscript-Element für hochgestellte Buchstaben verwendet (um die Vorschau zu sehen, aktivieren Sie den interaktiven Editor):

:::interactive_editor

```html
<p>
  Monseigneur is often written as <strong>M<sup>gr</sup></strong>.
</p>
```

:::

Hochgestellte Buchstaben sind Buchstaben, die als Hochstellung geschrieben werden, meist um Abkürzungen anzuzeigen. Die Buchstaben `g` und `r` sind in diesem Beispiel in Superscript-Tags eingeschlossen, um die Abkürzung zu veranschaulichen.

Es ist wichtig zu beachten, dass das Superscript-Element nur aus typografischen Gründen verwendet werden sollte. Wenn Sie einen Text mit einer angehobenen Grundlinie gestalten möchten, sollten Sie stattdessen CSS verwenden.

Um chemische Formeln in HTML darzustellen, verwenden Sie das Subscript-Element. Dieses Element nutzt einen Tiefgestellt-Text, der eine abgesenkte Grundlinie mit kleinerer Schrift anzeigt.

Hier ist ein Beispiel für die Verwendung des Subscript-Elements zur Darstellung der chemischen Formel von Kohlendioxid (um die Vorschau zu sehen, aktivieren Sie den interaktiven Editor):

:::interactive_editor

```html
<p>CO<sub>2</sub></p>
```

:::

Die Zahl zwei ist in `sub`-Tags eingeschlossen, um zu zeigen, dass das Zeichen tiefgestellt sein soll.

Typische Anwendungsfälle für das Subscript-Element sind chemische Formeln, Fußnoten und Variablen-Indizes.
