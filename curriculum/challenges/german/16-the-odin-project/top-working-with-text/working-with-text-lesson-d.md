# --description--

Das `em`-Element macht den Text kursiv. Außerdem wird der Text semantisch hervorgehoben, was wiederum Auswirkungen auf Bildschirmlesegeräte haben kann. To define an emphasized element you wrap text content in an `<em>` tag.
