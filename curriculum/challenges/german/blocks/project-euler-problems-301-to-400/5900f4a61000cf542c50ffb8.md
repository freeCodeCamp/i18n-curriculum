# --description--

In einem Schiebespiel kann eine Spielfigur horizontal oder vertikal in einen leeren Raum verschoben werden. Das Ziel des Spiels ist es, die rote Spielfigur von der oberen linken Ecke eines Gitters in die untere rechte Ecke zu bewegen; der freie Raum beginnt immer in der unteren rechten Ecke. Zum Beispiel zeigt die folgende Folge von Bildern, wie das Spiel in fünf Zügen auf einem 2x2-Gitter abgeschlossen werden kann.

<img alt="completing game in five moves on grid 2x2" src="https://cdn.freecodecamp.org/curriculum/project-euler/sliding-game-1.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Sei $S(m, n)$ die minimale Anzahl von Zügen, um das Spiel auf einem $m$ mal $n$ Gitter zu beenden. Zum Beispiel kann verifiziert werden, dass $S(5, 4) = 25$ gilt.

<img alt="initial grid state and final grid state for game on grid 5x4" src="https://cdn.freecodecamp.org/curriculum/project-euler/sliding-game-2.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Es gibt genau 5482 Gitter, für die $S(m, n) = p^2$ gilt, wobei $p < 100$ eine Primzahl ist.

Für wie viele Gitter gilt $S(m, n) = p^2$, wobei $p < {10}^6$ eine Primzahl ist?
