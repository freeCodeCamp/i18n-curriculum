# --explanation--

`Critical` significa algo muy importante o esencial, y `critical data` se refiere a información importante en la que el equipo confía para trabajar de manera efectiva. Perderla causaría retrasos o problemas. Por ejemplo:

- `The backup includes all critical data in case of a failure.` - La copia guardada contiene información esencial para recuperarse de un problema.

- `You must encrypt critical data to ensure it's secure.` - Proteger información importante con cifrado es necesario para la seguridad.
