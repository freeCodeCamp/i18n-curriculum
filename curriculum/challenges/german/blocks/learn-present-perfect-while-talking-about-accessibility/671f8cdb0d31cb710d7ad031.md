# --explanation--

Die Verneinung der `Present Perfect Continuous` Zeitform (`hasn't/haven't + been + present participle`) wird verwendet, um eine andauernde Handlung zu beschreiben, die über einen Zeitraum hinweg nicht wie erwartet stattgefunden hat. In diesem Kontext zeigt sie an, dass das Problem nur einige wenige Videos betrifft, nicht alle.
