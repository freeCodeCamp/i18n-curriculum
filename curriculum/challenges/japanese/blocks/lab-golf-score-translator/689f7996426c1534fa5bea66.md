# --description--

ゴルフのゲームでは、各ホールに`par`があり、これはゴルファーがボールをホールに入れてプレイを完了するために期待される平均の`strokes`の数を意味します。`par`に対してあなたの`strokes`がどれだけ上回っているか下回っているかによって、異なるニックネームがあります。

このラボでは、`par`と`strokes`をニックネームに変換する関数を書きます。

**目的:** 以下のユーザーストーリーを満たし、すべてのテストに合格してラボを完了してください。

**ユーザーストーリー:**

1. `golfScore`という名前の関数を作成してください。
1. `golfScore`はコースのパーと打数の2つの数値の引数を取る必要があります。
1. `golfScore`は文字列を返す必要があります。
1. `golfScore`は`"Hole-in-one!"`が`strokes`の場合に`1`を返す必要があります。
1. `golfScore`は`"Eagle"`が`strokes`から`par`を引いた値以下の場合に`2`を返す必要があります。
1. `golfScore`は`"Birdie"`が`strokes`から`par`を引いた値と等しい場合に`1`を返す必要があります。
1. `golfScore`は`"Par"`が`strokes`と等しい場合に`par`を返す必要があります。
1. `golfScore`は`"Bogey"`が`strokes`に`par`を足した値と等しい場合に`1`を返す必要があります。
1. `golfScore`は`"Double Bogey"`が`strokes`に`par`を足した値と等しい場合に`2`を返す必要があります。
1. `golfScore`は`"Go Home!"`が`strokes`に`par`を足した値以上の場合に`3`を返す必要があります。

# --solutions--

```js
function golfScore(par, strokes) {
  if (strokes === 1) {
    return "Hole-in-one!";
  }

  if (strokes <= par - 2) {
    return "Eagle";
  }

  if (strokes === par - 1) {
    return "Birdie";
  }

  if (strokes === par) {
    return "Par";
  }

  if (strokes === par + 1) {
    return "Bogey";
  }

  if(strokes === par + 2) {
    return "Double Bogey";
  }

  return "Go Home!";
}
```
