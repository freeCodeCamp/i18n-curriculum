# --description--

Hablemos un poco sobre las ocupaciones en tecnología.

`UX (User Experience)` se refiere a lo fácil y agradable que es usar algo como un sitio web o una app. Se enfoca en asegurarse de que todo funcione bien y se sienta agradable de usar. `UI (User Interface)` tiene que ver con cómo se ven las cosas en un sitio web o app y dónde las encuentras. Incluye los botones en los que haces clic, los colores y el diseño. Se trata de asegurarse de que todo se vea bien y sea fácil de encontrar.

Si eres un `UX designer`, un `UI designer`, o un `UX/UI designer`, eres la persona responsable de hacer que los sitios web y las apps sean fáciles de usar y se vean bien.
