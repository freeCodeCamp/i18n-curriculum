# --interactive--

Nella lezione precedente, hai imparato a lavorare con l'oggetto `Date` in JavaScript. Ma ci sono diversi modi per formattare le date in JavaScript.

In questa lezione, vedremo come usare i metodi `toISOString()`, `toString()` e `toLocaleDateString()` per formattare le date in JavaScript.

Prima di esaminare i diversi metodi, rivediamo com'è fatto l'oggetto `Date`:

:::interactive_editor

```js
const date = new Date();
console.log(date);
```

:::

Quando stampi l'oggetto `date` nella console, vedrai la data e l'ora correnti basate sulle impostazioni di sistema dell'utente. Ecco un esempio dell'output che potresti vedere:

```js
Sun Sep 29 2024 19:45:37 GMT-0700 (Pacific Daylight Time)
```

Se usassi il metodo `toString()` sull'oggetto `date`, vedresti lo stesso output mostrato sopra.

Per formattare la data in un formato ISO esteso (ISO 8601), puoi usare il metodo `toISOString()` così:

:::interactive_editor

```js
const date = new Date();
console.log(date.toISOString());
```

:::

ISO 8601 è uno standard internazionale per rappresentare date e orari. Il formato è `YYYY-MM-DDTHH:mm:ss.sssZ`.

Ecco un esempio dell'output che potresti vedere:

```js
2024-09-30T02:47:20.292Z
```

Un altro modo per formattare la data è usare il metodo `toLocaleDateString()`. Questo metodo ti permette di formattare la data in base alla localizzazione dell'utente. Ecco la sintassi base per usare il metodo `toLocaleDateString()`:

:::interactive_editor

```js
const date = new Date();
console.log(date.toLocaleDateString());
```

:::

Ecco un esempio dell'output che potresti vedere:

```js
9/29/2024
```

Il metodo `toLocaleDateString()` accetta due parametri opzionali: `locales` e `options`.

Il parametro `locales` è una stringa che rappresenta la localizzazione da usare. Per esempio, puoi passare `en-US` per l'inglese (Stati Uniti) o `fr-FR` per il francese (Francia). Se non passi un parametro `locales`, viene usata la localizzazione predefinita. Ecco un esempio di come usare il parametro `locales`:

:::interactive_editor

```js
const date = new Date();
console.log(date.toLocaleDateString("fr-FR"));
```

:::

Il secondo parametro opzionale è il parametro `options`. Questo parametro è un oggetto che ti permette di specificare il formato della stringa della data. Ecco un esempio di come usare il parametro `options`:

:::interactive_editor

```js
const date = new Date();
const options = {
  weekday: "long",
  year: "numeric",
  month: "long",
  day: "numeric",
};
console.log(date.toLocaleDateString("en-GB", options));
```

:::

Nell'esempio sopra, abbiamo specificato le opzioni per l'inglese (Gran Bretagna) e per includere il giorno della settimana completo, l'anno, il mese e il giorno.

Ecco un esempio dell'output che potresti vedere:

```js
Sunday, September 29, 2024
```

In questa lezione abbiamo coperto solo alcuni modi per formattare le date in JavaScript. Ci sono molti altri metodi e librerie disponibili per aiutarti a formattare le date in JavaScript. Ma i metodi `toISOString()` e `toLocaleDateString()` sono un buon punto di partenza per formattare le date in JavaScript.
