Cos'è la ricorsione nella programmazione?

Un metodo per ordinare gli array.

---

Un loop che non finisce mai.

---

#### --answer--

Un processo in cui una funzione chiama se stessa.

### --question--

#### --text--

Quale dei seguenti è un esempio di ricorsione?

#### --distractors--

```js
function factorial(n) {
  let result = 1;

  while (n > 0) {
    result = result * n;
    n--;
  }

  return result;
}
```

---

```js
function factorial(n) {
  const arr = Array(n).fill().map((_, i) => i + 1);
  return arr.reduce((acc, curr) => acc * curr, 1);
}
```

---

```js
function factorial(n) {
  let result = 1;

  for (let i = n; i > 0; i--) {
    result *= i;
  }

  return result;
}
```

#### --answer--

```js
function factorial(n) {
  if (n === 0 || n === 1) {
    return 1;
  }
  return n * factorial(n - 1);
}
```

### --question--

#### --text--

Perché la ricorsione deve avere un caso base?

#### --distractors--

Per assicurare che la funzione abbia sempre un valore di ritorno.

---

Per ridurre il numero di chiamate di funzione.

---

Per assicurare che tutte le chiamate ricorsive vengano eseguite nell'ordine corretto e che non venga lanciato un `RecursionError`.

#### --answer--

Per fornire un modo alla funzione di uscire dalle sue chiamate ricorsive ed evitare loop infiniti.
