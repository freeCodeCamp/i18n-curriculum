# --interactive--

Il metodo `map` è una funzione potente e molto usata in JavaScript che opera sugli array. È progettato per creare un nuovo array applicando una funzione data a ogni elemento dell’array originale.

Questo metodo non modifica l’array originale ma restituisce invece un nuovo array contenente i risultati della funzione applicata a ogni elemento.

Ecco un esempio di utilizzo del metodo `map` su un array di numeri:

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map((num) => num * 2);
console.log(numbers); // [1, 2, 3, 4, 5]
console.log(doubled); // [2, 4, 6, 8, 10]
```

:::

Per creare un nuovo array in cui ogni numero è raddoppiato, usiamo il metodo `map`. Il metodo `map` accetta una funzione di callback in cui la funzione viene chiamata su ogni singolo elemento dell’array.

In questo caso, ogni numero nell’array sarà moltiplicato per `2`. Il risultato sarà un nuovo array dei numeri `2,4,6,8,10`.

La funzione di callback può accettare fino a tre argomenti.

Il primo argomento è l’elemento corrente che viene elaborato.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element) => {
  console.log("Element:", element);
  return element * 2;
});
```

:::

Il secondo argomento è l’indice dell’elemento corrente che viene elaborato.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index) => {
  console.log("Element:", element);
  console.log("Index:", index);
  return element * 2;
});
```

:::

Il terzo argomento è l’array su cui `map` viene chiamato.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index, array) => {
  console.log("Element:", element);
  console.log("Index:", index);
  console.log("Array:", array);
  return element * 2;
});
```

:::

Capire e usare efficacemente il metodo `map` può migliorare significativamente la tua capacità di lavorare con gli array in JavaScript. Nelle lezioni future approfondiremo usi più avanzati di `map` ed esploreremo come può essere uno strumento potente per costruire programmi dinamici ed efficienti.
