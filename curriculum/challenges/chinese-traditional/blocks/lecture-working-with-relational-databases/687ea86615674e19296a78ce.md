# --description--

關聯式資料庫有廣泛的應用，包括網頁開發、庫存管理系統、醫療系統、電子商務應用程式等。

一些最常見的關聯式資料庫有 MySQL、PostgreSQL、SQLite 和 Microsoft SQL Server。讓我們來了解一下它們各自的特點。

MySQL 是一個開源的關聯式資料庫管理系統。它因為易於使用、可靠且高效於建造實際應用程式，而在網頁開發中非常受歡迎。它也有一個作用中的開發者社群，支援並維護它。

PostgreSQL 官方描述為「全球最進階的開放原始碼關聯式資料庫」。它是一個免費、開放原始碼的物件關聯式資料庫系統，以其可靠性、資料完整性和可擴充性著稱。<dfn>例如</dfn>，你可以定義你自己的資料型別、訂製函式，並且在不同的程式設計語言中撰寫 `code`，而無需重新編譯你的資料庫。

SQLite 是一個免費、開源且輕量級的基於檔案的 SQL 資料庫引擎。它是世界上使用最廣泛的資料庫引擎之一。SQLite 是自包含且無伺服器的。它是一個零設定的資料庫，這表示它不需要初始設定程序，非常適合快速入門。

這三個資料庫是免費且開放原始碼的。還有其他受歡迎的專有選項，可包括 Microsoft SQL Server 和 Oracle 資料庫。

Microsoft SQL server 是由 Microsoft 開發並用於各種應用程式的熱門關聯式資料庫管理系統。

Oracle 資料庫是一個可擴充的關聯式資料庫管理系統，常用於需要高效率的企業應用程式。

太好了。既然你已經更了解可用的不同選項，可選方案，讓我們來看看你如何安裝 PostgreSQL。

首先，你需要前往官方網站：<a href="https://www.postgresql.org/" target="_blank">https://www.postgresql.org/</a>。

一旦你到達那裡，點擊「下載」按鈕。

這會帶你到一個頁面，在那裡你可以選擇你的作業系統。如果你點擊其中一個，你將會被帶到該作業系統的說明頁面。

我們將說明 Windows 作業系統的安裝流程，但此流程在各作業系統間相似。

從此頁面，你可以點擊「Download the installer」聯結來下載安裝程式。

這會帶你到一個頁面，在那裡你可以選擇你想安裝的 PostgreSQL 版本。如果你點擊對應的圖示，下載應該會開始。

然後，在你的 Downloads 資料夾或你預設儲存下載檔案的資料夾中，你應該會看到安裝程式。

如果你雙擊它，將會開始設定處理程序。點擊下一步。

選擇 PostgreSQL 將要安裝的目錄。你會看到一個預設值，但你可以自訂它。

然後，選擇你想安裝的組件。有四種可能的組件：

- PostgreSQL 伺服端，資料庫引擎本身。
- pgAdmin 4，一個用於資料庫管理的使用者友善工具。
- Stack Builder，一個用於下載和安裝額外 PostgreSQL 相關軟體和擴充套件的工具
- 以及命令列工具，讓你能直接從終端機與 PostgreSQL 伺服器互動。這包括 psql，一個用於執行 SQL 查詢的互動式終端機。

當你準備好時，點擊 Next。

然後，選擇一個目錄來儲存你的資料。你會看到一個預設路徑，但你可以自訂它。準備好後，點擊 Next。

現在你需要為資料庫超級使用者輸入密碼，該使用者擁有對所有資料庫和物件的無限制存取權限。輸入你的密碼，重新輸入一次，然後點擊 下一步。

然後，選擇伺服端應監聽的埠號。預設值是 5432。然後，點擊 下一步。

你需要為新的資料庫叢集選擇一個地區設定。這對資料庫解讀地區敏感的資料型別非常重要。然後，點擊 Next。

最後，你會看到一個預先安裝摘要。花點時間檢查這個摘要，然後點擊下一步。

你需要確認你想要開始安裝處理程序。如果你點擊下一步，處理程序將會開始。這應該只需要幾分鐘。

當處理程序完成後，根據你安裝的組件，你可能必須選擇是否要啟動 Stack Builder 來下載並安裝額外的工具、驅動程式和應用程式。

如果你決定啟動 Stack Builder，你會看到一個可以開始安裝的螢幕。或者，你可以點擊 Cancel 以便稍後再進行。

恭喜！現在你已經在你的電腦上安裝了 PostgreSQL。對其他作業系統來說，這個過程應該也類似。完成後，你就可以開始使用資料庫。

前往你的應用程式並搜尋 pgAdmin，這是一個開放原始碼的 PostgreSQL 管理與開發平台，如果你在組件中選擇了它，應該已經安裝完成。點擊 pgAdmin 4 以開啟它。

當應用程式啟動時，你會看到初始載入螢幕。接著，你應該會看到一個儀表板，在那裡你可以啟動新的伺服器、檢查你的資料庫、組態 pgAdmin，還有更多功能。你也會找到有用的文件聯結。

如果你點擊左上角的「Servers」，你將需要輸入安裝期間所選擇的密碼。

在輸入你的密碼後，你應該會看到一個螢幕，在那裡你將能存取你的資料庫，包括它們的資料表。

你還會發現另一個非常有用的工具，psql，一個用於 SQL 查詢的互動式終端機。要開啟它，請在你的系統中搜尋 psql。點擊它。

它會開啟 SQL Shell，在那裡你可以開始執行指令。

這就是你如何在 Windows 上安裝 PostgreSQL。macOS 和 Linux 的處理程序非常相似。

或者，如果你正在 macOS 上安裝它，你可以使用 Homebrew，一個軟體包管理工具。當你安裝好 Homebrew 後，執行此命令來安裝 PostgreSQL：

```bash
brew install postgresql@<version>
```

你可以在 at (`@`) 符號後面寫上數字來指定你想安裝的版本。
