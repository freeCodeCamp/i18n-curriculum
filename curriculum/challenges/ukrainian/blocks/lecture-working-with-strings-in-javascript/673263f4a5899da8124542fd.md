# --interactive--

Метод `prompt()` є важливою частиною взаємодії JavaScript з користувачем. Це один із найпростіших способів отримати вхідні дані від користувача через невелике спливаюче діалогове вікно.

Ви часто побачите його використання у випадках, коли вебсторінці потрібна якась інформація від користувача, наприклад ім’я або інший текстовий ввід.

Отже, що саме робить метод `prompt()`? Він відкриває діалогове вікно, яке запитує у користувача вхідні дані, а потім повертає введений користувачем текст у вигляді рядка.

Метод `prompt()` приймає два аргументи: перший — це повідомлення, яке з’явиться всередині діалогового вікна, зазвичай з проханням ввести інформацію. Другий — це значення за замовчуванням, яке є необов’язковим і спочатку заповнює поле введення.

```js
prompt(message, default);
```

Ось приклад того, як це працює.

**ПРИМІТКА**: цей приклад містить код, який ви ще не вивчали. Не хвилюйтеся, якщо не розумієте все в коді. Це лише ілюстрація роботи методу `prompt()` і гарантія того, що prompt не з’явиться одразу при завантаженні сторінки, що може сприйматися як нав’язливе. Якщо у вас увімкнений інтерактивний редактор, ви можете спробувати це самі.

:::interactive_editor

```html
<button id="prompt-btn">Show Prompt</button>
<p id="output"></p>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("prompt-btn");
const output = document.getElementById("output");
btn.addEventListener("click", () => {
  const userName = prompt("What is your name?", "Guest");
  output.textContent = "Hello, " + userName + "!";
});
```

:::

У цьому прикладі, коли користувач натискає на кнопку, метод `prompt()` показує діалогове вікно з повідомленням `What is your name?` і полем введення, яке спочатку містить значення `Guest`.

Якщо користувач введе своє ім’я і натисне "OK", змінна `userName` збереже введене значення. Якщо користувач натисне "Cancel", змінна `userName` отримає значення `null`. `null` означає, що користувач не надав жодних вхідних даних. Абзац з вихідними даними потім покаже привітальне повідомлення з використанням наданого імені або `null`, якщо користувач скасував. Ви навчитеся технікам, які допоможуть уникнути показу `null` при скасуванні prompt у майбутніх уроках.

Майте на увазі, що метод `prompt()` зупиняє виконання скрипта, доки користувач не взаємодіє з діалоговим вікном.

Це означає, що решта вашого JavaScript-коду не виконуватиметься, доки користувач або не введе дані і не натисне "OK", або не скасує prompt.

Ще один момент: хоча `prompt()` корисний для швидкого тестування або невеликих застосунків, у сучасних складних вебзастосунках його зазвичай уникають через нав’язливість і непослідовну поведінку в різних браузерах.

Розуміючи метод `prompt()`, ви отримуєте простий спосіб взаємодії з користувачами і отримання інформації безпосередньо через браузер, навіть якщо він не дуже поширений у сучасних вебзастосунках.
