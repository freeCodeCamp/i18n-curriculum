# --explanation--

`Tend to` é uma frase usada para descrever algo que acontece regularmente ou que provavelmente acontecerá.Por exemplo:

`She tends to be early for meetings.` - Isso significa que chegar cedo para reuniões é um hábito ou ocorrência regular para ela.

`Lag` refere-se a um atraso ou tempo de resposta lento.Por exemplo:

`The video lags when there is a poor internet connection.` - Isso significa que o vídeo está atrasado ou lento por causa da conexão.

`Switch` significa mudar de uma coisa para outra.Por exemplo:

`Users can switch between different tabs in the browser.` - Isso significa que os usuários podem mudar de uma aba para outra enquanto navegam.
