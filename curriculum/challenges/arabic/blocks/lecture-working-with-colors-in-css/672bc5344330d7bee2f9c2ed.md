# --interactive--

عند العمل مع CSS لتنسيق صفحة ويب، تُعد واحدة من أكثر الطرق شيوعًا لتعريف الألوان هي استخدام قيم ألوان الهيكس. قيم ألوان الهيكس، أو رموز الهيكس، هي طريقة مختصرة لتمثيل الألوان في نموذج ألوان RGB. في هذا الدرس، سنستكشف ما هي رموز الهيكس، كيف تعمل في CSS، ومتى تستخدمها.

رمز الهيكس (اختصار لرمز سداسي عشري) هو سلسلة مكونة من ستة أحرف تُستخدم لتمثيل الألوان في نموذج ألوان RGB. يشير "الهيكس" إلى نظام الترقيم بالقاعدة 16، الذي يستخدم الأرقام `0` إلى `9` والحروف `A` إلى `F`.

في سياق الألوان، تحدد رموز الهيكس كميات الأحمر والأخضر والأزرق (RGB) التي تشكل لونًا معينًا. يبدأ كل رمز هيكس برمز الهاش (`#`)، يليه ستة أحرف تحدد شدة الأحمر والأخضر والأزرق في اللون.

على سبيل المثال، رمز الهيكس للون الأسود هو `#000000`، ورمز الهيكس للون الأبيض هو `#FFFFFF`. يمثل أول حرفين قيمة الأحمر، والزوج التالي يمثل الأخضر، والزوج الأخير يمثل الأزرق.

في CSS، يمكنك تطبيق الألوان على العناصر باستخدام رموز الهيكس. إليك الصيغة النحوية الأساسية:

```css
element {
  color: #RRGGBB;
}
```

هنا، `#RRGGBB` هو عنصر نائب لرمز الهيكس الفعلي. تمثل أزواج `RR` و`GG` و`BB` شدة الأحمر والأخضر والأزرق، على التوالي.

يمكن أن تتراوح هذه الأزواج من `00` (أدنى شدة) إلى `FF` (أعلى شدة). كلما زاد الرقم، زادت كمية ذلك اللون في المزيج النهائي. على سبيل المثال، دعنا نطبق بعض ألوان الهيكس على عدة عناصر:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h1>This is a heading.</h1>
<p>This is a paragraph.</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}
p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

في الحالة الأولى، `#FF5733` هو لون برتقالي مائل إلى الأحمر. إليك التفاصيل:

- `FF` هو أعلى قيمة للأحمر (مشبع بالكامل).
- `57` كمية معتدلة من الأخضر.
- `33` كمية صغيرة من الأزرق.

في الحالة الثانية، `#4CAF50` هو ظل من اللون الأخضر:

- `4C` قيمة منخفضة للأحمر.
- `AF` قيمة قوية للأخضر.
- `50` قيمة معتدلة للأزرق.

تُعد رموز الهيكس شائعة لأنها دقيقة وسهلة الاستخدام. تسمح لك بضبط ألوانك بدقة، مما يجعلها مثالية للتصاميم التي تتطلب اتساقًا.

توفر معظم برامج التصميم، مثل Adobe Photoshop أو Figma، رموز الهيكس للألوان التي تختارها، مما يسهل نسخ هذه القيم ولصقها مباشرة في ملفات CSS الخاصة بك.

في تطوير الويب، تُستخدم رموز الهيكس غالبًا لتعريف ألوان النصوص، والخلفيات، والحدود، وغيرها من العناصر المرئية. على سبيل المثال:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example</button>
```

```css
body {
  background-color: #F0F0F0; /* Light gray background */
}

button {
  background-color: #007BFF; /* A shade of blue */
  color: #FFFFFF; /* White text */
}
```

:::

هذا يضمن بقاء ألوانك متسقة عبر موقعك الإلكتروني أو تطبيقك بأكمله.

في بعض الحالات، قد تلاحظ كتابة رموز الهيكس بصيغة مختصرة، باستخدام ثلاثة أحرف فقط بدلًا من ستة. يكون ذلك ممكنًا عندما يكون كلا الحرفين في كل زوج لوني متماثلين. على سبيل المثال، لا يمكن اختصار `#FF5733`، لكن يمكن كتابة `#FFFFFF` (الأبيض) كـ `#FFF`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h2>This is a heading.</h2>
```

```css
body {
  background-color: #000;
}

h2 {
  color: #FFF; /* White in shorthand */
}
```

:::

الصيغة المختصرة `#FFF` تعادل `#FFFFFF`. وبالمثل، `#000` تمثل الأسود الذي هو `#000000` في الصيغة الكاملة.

تُعد رموز الهيكس شائعة لدقتها وتوافقها، مما يسمح بأكثر من 16 مليون لون عبر جميع المتصفحات. هي مثالية عندما تحتاج إلى ألوان علامة تجارية محددة، أو تريد التحكم في درجات دقيقة لا يمكن تعريفها بسهولة بالألوان المسماة أو HSL، أو عند نسخ الألوان من برامج التصميم التي توفر رموز الهيكس افتراضيًا.
