# --description--

الآن بعد أن فهمت الصيغة النحوية الأساسية لـ HTML و CSS، سنبدأ بجدية. أهم المهارات التي يجب أن تتقنها مع CSS هي تحديد المواقع والتخطيط. تغيير الخطوط والألوان مهارة مهمة، لكن القدرة على وضع الأشياء بالضبط حيث تريدها على صفحة الويب أهم بكثير. فكم عدد صفحات الويب التي تجد فيها كل عنصر مكدس فوق الآخر فقط؟

تعلم تحديد مواقع العناصر على صفحة الويب ليس صعبًا بمجرد فهم بعض المفاهيم الأساسية. للأسف، يسرع كثير من المتعلمين في تعلم HTML و CSS للوصول إلى JavaScript وينتهي بهم الأمر بتفويت هذه المفاهيم الأساسية. هذا يؤدي إلى الإحباط والألم (وصور GIF مضحكة) لأن كل مهارات JavaScript في العالم لا معنى لها إذا لم تستطع تثبيت عناصر الصفحة حيث تحتاجها. لذا، لنبدأ مع هذا في الاعتبار.
