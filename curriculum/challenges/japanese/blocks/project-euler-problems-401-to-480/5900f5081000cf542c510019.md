# --description--

正の整数 $n$ を考えます。座標 $(x, y) = (2^i\bmod n, 3^i\bmod n)$ に駅があり、$0 ≤ i ≤ 2n$ とします。同じ座標の駅は同じ駅とみなします。

$(0, 0)$ から $(n, n)$ まで、$x$ 座標と $y$ 座標が減少しない経路を作りたいと思います。

$S(n)$ を、そのような経路が通過できる最大の駅数とします。

例えば、$n = 22$ の場合、11個の異なる駅があり、有効な経路は最大で5つの駅を通過できます。したがって、$S(22) = 5$ です。以下に最適な経路の例を示します。

<img alt="n = 22 の場合、11個の異なる駅を通過し、5つの駅を通過する有効な経路" src="https://cdn.freecodecamp.org/curriculum/project-euler/uphill-paths.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

また、$S(123) = 14$ および $S(10\,000) = 48$ であることも確認できます。

$1 ≤ k ≤ 30$ について、$\sum S(k^5)$ を求めてください。
