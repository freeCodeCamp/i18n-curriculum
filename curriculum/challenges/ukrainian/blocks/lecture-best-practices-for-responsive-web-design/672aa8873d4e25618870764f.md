# --interactive--

Адаптивний вебдизайн — це підхід до веброзробки, який має на меті створення сайтів, що забезпечують оптимальний перегляд і взаємодію на широкому спектрі пристроїв — від настільних комп’ютерів до мобільних телефонів.

Основний принцип адаптивного дизайну — це здатність сайту підлаштовувати свій макет і вміст залежно від розміру екрана та можливостей пристрою, на якому його переглядають.

Адаптивний дизайн зазвичай базується на трьох основних компонентах — гнучких сітках, гнучких зображеннях і медіазапитах:

- Гнучкі сітки використовують відносні одиниці, як-от відсотки, замість фіксованих одиниць, як пікселі, що дозволяє вмісту змінювати розмір і перетікати залежно від розміру екрана.

- Гнучкі зображення налаштовані так, щоб змінювати розмір у межах своїх контейнерів, забезпечуючи, що вони не виходять за межі контейнерів на менших екранах.

- А медіазапити дозволяють розробникам застосовувати різні стилі залежно від характеристик пристрою, головним чином ширини вікна перегляду. Ви дізнаєтеся більше про медіазапити в майбутніх уроках.

Взаємозв’язок між адаптивним вебдизайном і такими інструментами, як CSS Grid і Flexbox, є симбіотичним. Адаптивний дизайн — це концепція або підхід, а CSS Grid і Flexbox — це практичні інструменти, які значно полегшують і підвищують ефективність реалізації адаптивних дизайнів.

У попередніх уроках ви вивчали, як працювати з Flexbox, а в майбутніх уроках дізнаєтеся, як працювати з CSS Grid. А поки що ось коротке введення в CSS Grid.

CSS Grid — це двовимірна система макету, яка дозволяє створювати більш складні розташування. Вона чудово підходить як для загальних макетів сторінок, так і для менших макетів компонентів.

Ось приклад того, як CSS Grid можна використовувати адаптивно:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr;
  gap: 20px;
}

@media (min-width: 768px) {
  .grid-container {
    grid-template-columns: 1fr 1fr;
  }
}

@media (min-width: 1024px) {
  .grid-container {
    grid-template-columns: 1fr 1fr 1fr;
  }
}
```

:::

У цьому прикладі сітка починається з одного стовпця на малих екранах, переходить до двох стовпців на середніх екранах, а потім до трьох стовпців на більших екранах. Це демонструє, як Grid може створювати адаптивні макети, що підлаштовуються під різні розміри екрана.

І Flexbox, і Grid мають значні переваги порівняно зі старішими методами макету, як-от плаваючі елементи або макети на основі таблиць. Вони забезпечують більшу гнучкість і контроль, вимагають менше коду і розроблені з урахуванням адаптивності. Вони дозволяють розробникам створювати складні, адаптивні макети з відносною легкістю, що робить їх незамінними інструментами для реалізації адаптивного вебдизайну.

Варто зазначити, що Flexbox і Grid часто використовують разом в адаптивних дизайнах. Flexbox зазвичай застосовують для компонентів і одновимірних макетів, тоді як Grid — для загальної структури сторінки і двовимірних макетів. Вибір між ними часто залежить від конкретних потреб макету.

Окрім Flexbox і Grid, інші функції CSS відіграють важливу роль в адаптивному дизайні.

Функція `calc()`, наприклад, дозволяє змішувати одиниці виміру і виконувати обчислення, що може бути дуже корисним при створенні гнучких макетів.

Адаптивні зображення — ще один ключовий аспект адаптивного вебдизайну. Атрибут `srcset` і елемент `picture` у HTML5 дозволяють подавати різні файли зображень залежно від можливостей пристрою, забезпечуючи, що користувачі не завантажують надто великі файли на пристроях із меншими екранами або нижчою роздільністю.

Підсумовуючи, адаптивний вебдизайн — це підхід, який має на меті створення сайтів, що добре працюють на будь-якому пристрої.

Хоч це і філософія дизайну, а не конкретна технологія, він значною мірою спирається на функції CSS, як-от медіазапити, і значно полегшується сучасними інструментами макету, як Flexbox і Grid. Ці інструменти забезпечують гнучкість і контроль, необхідні для створення справді адаптивних дизайнів, дозволяючи сайтам бездоганно підлаштовуватися під все зростаючу різноманітність пристроїв для доступу до вебу.
