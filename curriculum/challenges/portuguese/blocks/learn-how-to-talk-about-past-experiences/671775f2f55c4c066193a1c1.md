# --explanation--

`Exactly` é usado para confirmar ou concordar fortemente com algo que outra pessoa disse.Indica que a afirmação está totalmente correta.Por exemplo: 

Pessoa 1: `Did you mean we should start testing today?`

Pessoa 2: `Exactly.` - significa que você tem certeza de que os testes devem começar hoje.

Neste contexto, Jake poderia dizer `Exactly` para confirmar que ele realmente estava corrigindo um problema de segurança naquela época como Linda mencionou.
