# --explanation--

O `front end` é a parte do software voltada para o usuário — tudo o que o usuário vê e interage (como botões, menus, layout). No diálogo, atua como um adjetivo composto e usa `-`. Outro exemplo:

`The front-end team built the new homepage interface.` — A equipe que trabalha no que os usuários veem fez o novo design para a página principal.

`Design` refere-se a como o software parece e funciona — cor, layout, ícones, tipografia e usabilidade.Por exemplo:

`Good design improves how users experience the app.` — Se o app tem uma boa aparência e é fácil de usar, as pessoas gostam mais de usá-lo.
