# --interactive--

تتيح لك الحوارات عرض معلومات أو إجراءات مهمة للمستخدمين. باستخدام عنصر `dialog` المدمج في HTML، يمكنك بسهولة إنشاء هذه الحوارات (سواء الحوارات النموذجية أو غير النموذجية) في تطبيقات الويب الخاصة بك.

الحوار النموذجي هو نوع من الحوارات يجبر المستخدم على التفاعل معه قبل أن يتمكن من الوصول إلى بقية التطبيق أو صفحة الويب. فهو يمنع التفاعل مع المحتوى الآخر حتى يكمل المستخدم إجراءً ما، مثل إغلاق الحوار أو إرسال نموذج.

في المقابل، يسمح الحوار غير النموذجي للمستخدم بالاستمرار في التفاعل مع أجزاء أخرى من الصفحة أو التطبيق حتى عندما يكون الحوار مفتوحًا. فهو لا يمنع الوصول إلى بقية المحتوى.

في هذا الدرس، ستتعلم كيفية فتح وإغلاق الحوارات النموذجية باستخدام الطريقتين `showModal()` و `close()`.

عندما تريد التأكد من تركيز المستخدم على إجراء أو رسالة محددة في الحوار النموذجي، يمكنك فتح الحوار باستخدام الطريقة `showModal()`. ستضيف هذه الطريقة خلفية مظللة للعناصر الأخرى في الصفحة وتعطلها. هذا مثالي للحوارات التي تعرض نماذج، تأكيدات، ومعلومات حرجة تتطلب إجراء من المستخدم.

إليك شفرة HTML للحوار النموذجي:

:::interactive_editor

```html
<dialog id="my-modal">
  <p>This is a modal dialog.</p>
</dialog>
```

:::

في الوقت الحالي، لا يمكنك رؤية أي شيء على الصفحة لأن الحوار مغلق عند العرض الأولي. يمكنك فتح الحوار تلقائيًا باستخدام الطريقة `showModal()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
dialog.showModal();
```

:::

النتيجة في المتصفح ستظهر حوارًا يحتوي على النص `This is a modal dialog.`

من الأفضل أن تمنح التحكم للمستخدم. لتحقيق ذلك، يمكنك إضافة مستمع حدث نقر إلى زر واستخدام الطريقة `showModal()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.showModal();
});
```

:::

إذا كنت بحاجة إلى عرض حوار مع السماح بالتفاعل مع المحتوى خارج الحوار، فيمكنك استخدام الطريقة `show()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});
```

:::

لإغلاق الحوار النموذجي، يمكنك إضافة زر داخل عنصر `dialog` واستخدام الطريقة `close()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
  <button id="close-modal-btn">Close Modal</button>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");
const closeButton = document.getElementById("close-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});

closeButton.addEventListener("click", () => {
  dialog.close();
});
```

:::
