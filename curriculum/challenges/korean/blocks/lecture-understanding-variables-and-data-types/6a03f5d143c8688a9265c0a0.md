# --description--

이전 수업에서 문자열, 정수, 부동소수점 타입을 포함한 Python의 일반적인 데이터 타입에 대해 배웠습니다. 프로그램을 작성하면서 변수의 타입을 확인하는 방법을 배워야 합니다.

다음은 예시 변수입니다:

```py
developer = 'Devin'
```

`developer`의 타입을 확인하려면 `type()` 함수를 이렇게 사용할 수 있습니다:

```py
developer = 'Devin'

print(type(developer)) # <class 'str'>
```

`<class 'str'>`의 출력은 `developer`가 문자열 타입임을 의미합니다.

`type()` 함수에 인수를 제공하지 않으면 다음과 같은 오류 메시지가 나타납니다:

```md
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: type() takes 1 or 3 arguments
```

지금까지 배운 모든 데이터 타입과 터미널에서의 타입은 다음과 같습니다:

```python
my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 'hello', 8.5}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 'hello', 8.5)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>
```

프로그램에서 특정 변수의 타입이 맞는지 확인한 후에 작업을 수행해야 할 때가 있습니다. 이럴 때 `isinstance()` 함수가 유용합니다.

다음은 문자열이 할당된 예시 변수입니다:

```py
account_balance = '12'
```

`account_balance` 변수를 사용해 나눗셈 같은 수학 연산을 시도하면 오류 메시지가 나타납니다.

```py
account_balance = '12'

account_balance / 2

# Traceback (most recent call last):
#   File "<stdin>", line 1, in <module>
# TypeError: unsupported operand type(s) for /: 'str' and 'int'
```

`account_balance`가 정수인지 확인하려면 `isinstance()` 함수를 이렇게 사용할 수 있습니다:

```py
account_balance = '12'

isinstance(account_balance, int) # False
```

내장 `isinstance()` 함수는 변수가 특정 데이터 타입과 일치하는지 확인할 수 있게 해줍니다. 객체와 확인하려는 타입을 인수로 받아 불리언 값을 반환합니다. 이 경우 `account_balance`가 문자열이므로 `False`를 반환합니다.

`isinstance()` 함수는 여러 타입을 한 번에 확인할 수도 있습니다.

다음은 `account_balance`가 `int` 또는 `float`인지 확인하는 예시입니다:

```py
account_balance = 12
isinstance(account_balance, (int, float)) # True
```

이 예시에서 `account_balance`는 정수이므로 `isinstance()`는 `True`를 반환합니다. 만약 `account_balance`가 `12.0`였다면, 정수 또는 부동소수점 타입을 확인하기 때문에 `isinstance()`는 여전히 `True`를 반환했을 것입니다.

앞으로 워크숍과 실습에서 `type()`와 `isinstance()` 함수를 사용해 변수에 올바른 데이터 타입이 들어있는지 확인한 후 작업을 수행할 것입니다.
