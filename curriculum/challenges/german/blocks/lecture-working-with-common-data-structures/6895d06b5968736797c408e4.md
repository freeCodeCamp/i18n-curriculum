# --description--

Arrays sind eine grundlegende Datenstruktur in der Informatik. Alle Arrays speichern geordnete Datensammlungen, aber je nach Typ können sie hinter den Kulissen unterschiedlich funktionieren.

Ihr zugrundeliegendes Verhalten kann einen wichtigen Einfluss auf die Effizienz des Programms haben. Lernen wir also dynamische und statische Arrays und ihre Unterschiede kennen, damit Sie das effizienteste für Ihr Programm auswählen können.

Wir beginnen mit statischen Arrays.

**Statische Arrays** haben eine feste Größe. Sie speichern Elemente an benachbarten Speicherstellen.

Die Größe eines statischen Arrays wird bestimmt, wenn das Array initialisiert wird. Sobald dieser bestimmte Speicherblock zugewiesen ist, ist er fest und kann während der Programmausführung nicht geändert werden. Dies ist ein zentrales Merkmal statischer Arrays.

Das Speichern von Elementen an benachbarten Speicherstellen macht den Datenabruf effizienter, weil das Programm die Position des ersten Elements speichern und dann Indizes verwenden kann, um einfache Berechnungen durchzuführen und die anderen Elemente im Speicher zu finden.

Dank dessen erfolgt der Zugriff auf die Werte eines statischen Arrays in konstanter Zeit `O(1)`, was sehr effizient ist.

Sie können ein statisches Array verwenden, wenn Sie die Anzahl der zu speichernden Elemente im Voraus kennen. Es ist auch hilfreich, wenn die Werte sehr häufig abgerufen werden, da der Zugriff sehr effizient ist.

Diese Datenstruktur kann jedoch weder wachsen noch schrumpfen. Wenn sich die Anzahl der zu speichernden Elemente ändern kann, sollten Sie stattdessen ein dynamisches Array verwenden.

Der Versuch, die Größe eines statischen Arrays zu vergrößern, würde bedeuten, ein neues Array zu erstellen und alle Elemente vom alten Array in das neue zu kopieren, was ineffizient ist. In diesem Fall wäre ein dynamisches Array viel besser, da es diesen Prozess automatisch handhabt.

Python enthält keine traditionellen statischen Arrays als eingebaute Datenstrukturen.

Andere Programmiersprachen wie Java unterstützen sie jedoch. Hier ist ein Beispiel für ein statisches Array in Java, das drei ganze Zahlen speichern kann:

```java
int[] numbers = new int[3]; 
```

Arrays in Python sind dynamisch, schauen wir uns also diese an.

**Dynamische Arrays** sind flexibler, weil sie während der Programmausführung automatisch wachsen oder schrumpfen können.

Sie funktionieren über einen automatischen Größenanpassungsmechanismus, der die Elemente in ein neues Array kopiert, wenn das ursprüngliche Array voll ist. Der Prozess wird effizient durchgeführt, weil die Größe des neuen Arrays so gewählt wird, dass diese rechenintensiven Operationen seltener auftreten.

Der Zugriff auf die Elemente eines dynamischen Arrays erfolgt in konstanter Zeit `O(1)`, daher ist diese Operation sehr effizient.

Das Einfügen eines Elements in die Mitte des Arrays benötigt lineare Zeit `O(n)`, weil die Elemente danach neu positioniert werden müssen.

Das Einfügen eines Elements am Ende des Arrays benötigt konstante Zeit `O(1)`, wenn im dynamischen Array noch Platz ist. Ist das Array jedoch voll und muss vergrößert werden, hat diese Operation eine `O(n)`-Komplexität.

Sie sollten dynamische Arrays verwenden, wenn Sie die Anzahl der Werte, die Sie speichern müssen, nicht im Voraus kennen. Sie sind auch hilfreich, wenn Sie häufig Elemente einfügen und löschen.

Die eingebaute `list`-Datenstruktur von Python funktioniert als dynamisches Array. Sie können eine Liste erstellen, indem Sie die Elemente innerhalb von eckigen Klammern schreiben, getrennt durch Kommas.

```python
numbers = [3, 4, 5, 6]
```

Sie können auf ein Element zugreifen, indem Sie den Namen der Variablen, die die Liste hält, gefolgt von eckigen Klammern schreiben, und innerhalb der eckigen Klammern den entsprechenden Index angeben.

Die Indizes beginnen bei 0 für das erste Element und erhöhen sich um 1 für jedes folgende Element:

```python
numbers[0]  # 3
numbers[1]  # 4
numbers[2]  # 5
numbers[3]  # 6
```

Um einen Wert zu aktualisieren, müssen Sie ihn einfach neu zuweisen:

```python
numbers[2] = 16
```

Sie können Elemente mit der `.append()`-Methode an die Liste anhängen:

```python
numbers.append(7)
```

Sie können Elemente mit der `.insert()`-Methode an einem bestimmten Index einfügen, indem Sie den Index als erstes Argument und das Element selbst als zweites Argument übergeben.

```python
numbers.insert(3, 15)
```

Sie können ein Element an einem bestimmten Index mit der `.pop()`-Methode entfernen:

```python
numbers.pop(2)
```

Wenn Sie den Index nicht angeben, entfernt `.pop()` das letzte Element.

Es gibt weitere eingebaute Listenmethoden, die Sie in der Dokumentation nachschlagen können, um Elemente ganz einfach hinzuzufügen und zu entfernen.

Das ist die Stärke dynamischer Arrays, oder in diesem Fall von Listen.

Im Allgemeinen sollten Sie statische Arrays verwenden, wenn Sie die Anzahl der Elemente im Voraus kennen und häufig darauf zugreifen müssen, und dynamische Arrays, wenn die Anzahl der Elemente unbekannt oder variabel ist.

Sie sollten immer den Kompromiss zwischen der Einfachheit statischer Arrays und der Flexibilität dynamischer Arrays bedenken. Beide sind für bestimmte Anwendungsfälle und Szenarien hilfreich. Die Fähigkeit, das beste für ein gegebenes Problem auszuwählen, gehört zu den Problemlösungsfähigkeiten, die Sie mit der Zeit durch Übung entwickeln werden.
