# --interactive--

في JavaScript، الكائن هو هيكل بيانات أساسي يسمح لك بتخزين وتنظيم البيانات والوظائف المرتبطة بها.

يمكنك التفكير في الكائن كحاوية تحتوي على قطع مختلفة من المعلومات، تمامًا مثل خزانة الملفات التي تحتوي على مجلدات ومستندات مختلفة.

تُسمى هذه القطع من المعلومات خصائص، وتتكون من اسم (أو مفتاح) وقيمة.

```js
const exampleObject = {
  propertyName: value,
}
```

الكائنات متعددة الاستخدامات للغاية وتشكل العمود الفقري لـ JavaScript. في الواقع، كل شيء تقريبًا في JavaScript هو كائن أو يمكن التعامل معه ككائن. يشمل ذلك المصفوفات، والدوال، وحتى أنواع البيانات الأولية مثل السلاسل النصية والأرقام عند استخدامها بطرق معينة.

هذه الطبيعة المرتكزة على الكائنات في JavaScript هي أحد أسباب كونها لغة مرنة وقوية. لنلقِ نظرة على كيفية إنشاء كائن:

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};
```

في هذا المثال، أنشأنا كائنًا يسمى `person` بثلاث خصائص: `name`، و`age`، و`city`. كل خاصية لها اسم وقيمة مفصولان بنقطتين.

الآن، دعنا نستكشف كيف يمكنك الوصول إلى هذه الخصائص. هناك طريقتان رئيسيتان للوصول إلى خصائص الكائن في JavaScript: ترميز النقطة وترميز الأقواس.

ترميز النقطة هو الطريقة الأكثر شيوعًا وبساطة للوصول إلى خصائص الكائن. هذه هي الصيغة النحوية الأساسية لترميز النقطة:

```js
objectName.propertyName
```

إليك كيف تستخدم ترميز النقطة مع كائن `person` الخاص بنا:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

console.log(person.name);  // Alice
console.log(person.age);   // 30
```

:::

ترميز النقطة مختصر وسهل القراءة، مما يجعله الخيار المفضل عندما تعرف الاسم الدقيق للخاصية التي تريد الوصول إليها ويكون هذا الاسم معرفًا صالحًا في JavaScript (أي لا يبدأ برقم ولا يحتوي على أحرف خاصة أو مسافات).

من ناحية أخرى، يسمح لك ترميز الأقواس بالوصول إلى خصائص الكائن باستخدام سلسلة نصية داخل أقواس مربعة. إليك كيف تستخدم ترميز الأقواس:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

console.log(person["name"]); // Alice
console.log(person["age"]); //  30
```

:::

ترميز الأقواس أكثر مرونة من ترميز النقطة لأنه يسمح لك باستخدام أسماء خصائص ليست معرفات صالحة في JavaScript. على سبيل المثال، إذا كان لديك اسم خاصية يحتوي على مسافات أو يبدأ برقم، فستحتاج إلى استخدام ترميز الأقواس:

:::interactive_editor

```js
const oddObject = {
  "1stProperty": "Hello",
  "property with spaces": "World"
};

console.log(oddObject["1stProperty"]);  // Hello
console.log(oddObject["property with spaces"]);  // World
```

:::

ميزة أخرى لترميز الأقواس هي أنه يسمح لك باستخدام متغيرات للوصول إلى الخصائص بشكل ديناميكي:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "Wonderland"
};

let propertyName = "city";
console.log(person[propertyName]); // Wonderland
```

:::

تجعل هذه المرونة ترميز الأقواس مفيدًا بشكل خاص عندما لا تعرف الاسم الدقيق للخاصية أثناء كتابة الشفرة، أو عندما تعمل مع أسماء خصائص تأتي من مُدخلات المستخدم أو مصدر ديناميكي آخر.

من الجدير بالذكر أن الكائنات في JavaScript قوية ومتعددة الاستخدامات للغاية. يمكنها أن تحتوي ليس فقط على قيم بسيطة مثل السلاسل النصية والأرقام، بل أيضًا على مصفوفات أو كائنات أخرى.

فهم الكائنات وكيفية العمل معها أمر حاسم في JavaScript لأنها تُستخدم على نطاق واسع في جميع أنحاء اللغة وفي العديد من مكتبات وأُطُر عمل JavaScript.

كلما واصلت التعلم والعمل مع JavaScript، ستجد أن إتقان الكائنات يفتح أمامك عالمًا من الإمكانيات لإنشاء تطبيقات معقدة وقوية.
