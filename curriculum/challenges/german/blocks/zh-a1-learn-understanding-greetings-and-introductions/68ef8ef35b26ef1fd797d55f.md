# --explanation--

`我是中国人 (wǒ shì zhōng guó rén)` verwendet die Struktur `我是 (wǒ shì)` + Nationalität, um anzugeben, woher jemand kommt.

`我是 (wǒ shì)` bedeutet „Ich bin...“. Eine Nationalität wird gebildet, indem man `人 (rén)`, was „Person“ bedeutet, an den Ländernamen anhängt.

`中国 (zhōng guó)` bedeutet „China“, daher bedeutet `中国人 (zhōng guó rén)` eine Person aus China.

Deshalb sagt Wang Hua, dass sie Chinesin ist.
