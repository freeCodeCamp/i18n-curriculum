# --description--

Якщо `previous` є `null`, це означає, що елемент, який потрібно видалити, знаходиться на початку списку, тому слід оновити `head`, щоб він вказував на наступний вузол, щоб видалити перший елемент.

Для цього додайте умову `else`. Усередині неї встановіть `list.head` у значення `current.next`.
