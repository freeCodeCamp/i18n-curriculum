# --description--

一个数 $n$ 的幺除数 $d$ 是 $n$ 的一个除数，且满足属性 $gcd(d, \frac{n}{d}) = 1$。

$4! = 24$ 的幺除数是 1、3、8 和 24。

它们的平方和是 $12 + 32 + 82 + 242 = 650$。

令 $S(n)$ 表现为 $n$ 的幺除数的平方和。因此 $S(4!) = 650$。

查找 $S(100\,000\,000!)$ 模 $1\,000\,000\,009$。
