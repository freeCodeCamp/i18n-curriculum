# --description--

無論你使用哪個方向，你都需要將你的彈性容器視為具有兩個軸：主軸和交叉軸。當 `flex-direction` 改變時，這些軸的方向也會改變。在大多數情況下，`flex-direction: row` 將主軸設為水平（從左到右），而 `column` 則將主軸設為垂直（從上到下）。

換句話說，在我們的第一個範例中，你在一個 div 上設定了 `display: flex`，它會將其子元素水平排列。這是 `flex-direction: row` 的示範，這是預設值。以下範例非常相似。如果你取消註解寫著 `flex-direction: column` 的那一行，這些 div 就會垂直堆疊。
