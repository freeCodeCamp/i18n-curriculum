# --description--

Sei $x$ eine reelle Zahl.

Eine beste Approximation von $x$ für die Nennergrenze $d$ ist eine rationale Zahl $\frac{r}{s}$ in gekürzter Form, mit $s ≤ d$, sodass jede rationale Zahl, die näher an $x$ liegt als $\frac{r}{s}$, einen Nenner größer als $d$ hat:

$$|\frac{p}{q} - x| &lt; |\frac{r}{s} - x| ⇒ q > d$$

Zum Beispiel ist die beste Approximation von $\sqrt{13}$ für die Nennergrenze $20$ $\frac{18}{5}$ und die beste Approximation von $\sqrt{13}$ für die Nennergrenze $30$ $\frac{101}{28}$.

Finden Sie die Summe aller Nenner der besten Approximationen von $\sqrt{n}$ für die Nennergrenze ${10}^{12}$, wobei $n$ keine perfekte Quadratzahl ist und $1 &lt; n ≤ 100000$ gilt.
