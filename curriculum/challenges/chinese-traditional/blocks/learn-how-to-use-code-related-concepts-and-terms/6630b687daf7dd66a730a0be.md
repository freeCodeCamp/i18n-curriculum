# --description--

字 `fundamental` 意味著某事是基本或必要的，就像房子的基礎一樣。

例如，`Understanding grammar is fundamental to learning a new language.` 這裡的 `fundamental` 用來強調文法是學習任何新語言的基本零件。

片語 `what kind of` 用來詢問某物的型別或性質。這是更清楚指定或分類某物的常用方式。

例如，當有人問 `what kind of music do you like?` 時，他們是在詢問音樂的類型。
