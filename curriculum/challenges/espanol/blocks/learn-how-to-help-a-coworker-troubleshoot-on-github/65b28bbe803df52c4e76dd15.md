# --description--

Los verbos modales como `might` y `can` se utilizan para expresar posibilidad y habilidad. 

`Might` se usa para algo que podría suceder o ser cierto, como decir: `It might rain today`. 

`Can` se utiliza para hablar de la capacidad de alguien de hacer algo, como, `She can speak three languages`. Practicemos usando estas palabras en oraciones.
