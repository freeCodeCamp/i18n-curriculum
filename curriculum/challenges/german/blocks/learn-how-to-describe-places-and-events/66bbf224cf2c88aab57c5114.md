`Supposed to` wird verwendet, um etwas auszudrücken, das erwartet oder angenommen wird, basierend auf einem Plan, einer Regel oder einer Erwartung. Zum Beispiel:

- `The project is supposed to be finished by Friday.` – Das bedeutet, dass erwartet wird, dass es bis zu diesem Zeitpunkt abgeschlossen ist.

In Toms Satz verwendet er `supposed to`, um über eine erwartete Funktion in Visual Studio Code zu sprechen, und deutet damit an, dass diese Funktion unterstützt werden sollte.
