# --explanation--

Ángela dice `¿Navarro con dos erres?` Questa è una domanda di conferma.

`Con` introduce un'informazione aggiuntiva sull'ortografia.

`Dos` è il numero due.

`Erres` è la forma plurale di `erre`, il nome della lettera `r`.

Ángela sta confermando che il cognome Navarro si scrive con due lettere `r`.
