# --description--

양의 정수 $n$의 반올림 제곱근을 $n$의 제곱근을 가장 가까운 정수로 반올림한 값으로 정의합니다.

다음 절차(본질적으로 정수 산술에 맞게 조정된 헤론 방법)는 $n$의 반올림 제곱근을 찾습니다:

$n$의 자릿수를 $d$라고 합시다.

$d$가 홀수이면, $x_0 = 2 × {10}^{\frac{d - 1}{2}}$로 설정합니다.

$d$가 짝수이면, $x_0 = 7 × {10}^{\frac{d - 2}{2}}$로 설정합니다.

다음 과정을 반복합니다:

$$x_{k + 1} = \left\lfloor\frac{x_k + \left\lceil\frac{n}{x_k}\right\rceil}{2}\right\rfloor$$

단, $x_{k + 1} = x_k$가 될 때까지 반복합니다.

예를 들어, $n = 4321$의 반올림 제곱근을 찾아봅시다.

$n$은 4자리 수이므로, $x_0 = 7 × {10}^{\frac{4-2}{2}} = 70$입니다.

$$x_1 = \left\lfloor\frac{70 + \left\lceil\frac{4321}{70}\right\rceil}{2}\right\rfloor = 66 \\\\
x_2 = \left\lfloor\frac{66 + \left\lceil\frac{4321}{66}\right\rceil}{2}\right\rfloor = 66$$

$x_2 = x_1$이므로 여기서 멈춥니다. 따라서 단 두 번의 반복으로 4321의 반올림 제곱근이 66임을 찾았습니다 (실제 제곱근은 65.7343137…입니다).

이 방법을 사용할 때 필요한 반복 횟수는 놀랍도록 적습니다. 예를 들어, 5자리 정수($10\,000 ≤ n ≤ 99\,999$)의 반올림 제곱근을 찾는 데 평균 3.2102888889회의 반복이 필요합니다 (평균값은 소수점 10자리까지 반올림됨).

위 절차를 사용하여 14자리 수(${10}^{13} ≤ n < {10}^{14}$)의 반올림 제곱근을 찾는 데 필요한 평균 반복 횟수는 얼마입니까? 답은 소수점 10자리까지 반올림하여 제출하세요.

**참고:** 기호 $⌊x⌋$와 $⌈x⌉$는 각각 바닥 함수와 천장 함수를 나타냅니다.
