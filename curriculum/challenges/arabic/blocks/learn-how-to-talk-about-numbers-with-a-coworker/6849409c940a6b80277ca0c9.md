# --explanation--

التركيب الصحيح لإعطاء عدد هو:

`There are` + رقم عددي + اسم في صيغة الجمع + `in` + مكان. — `There are ten tablets in the storage room.`

هذا تركيب نحوي صحيح ويجيب عن السؤال بوضوح.
