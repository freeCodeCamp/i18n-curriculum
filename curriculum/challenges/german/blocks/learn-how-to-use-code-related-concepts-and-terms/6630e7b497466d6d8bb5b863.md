# --description--

In der Programmierung ist ein `string` ein Datentyp, der zum Speichern von Text verwendet wird. Ein `string` kann Buchstaben, Zahlen und Symbole enthalten. Er wird oft dargestellt, indem der Text in Anführungszeichen eingeschlossen wird.

Zum Beispiel ist `"Hello, world!"` ein `string`.
