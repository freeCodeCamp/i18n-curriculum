# --description--

`¿A qué te dedicas?` é uma forma comum de perguntar sobre o trabalho ou profissão de uma pessoa.

É equivalente a "Qual é a sua profissão?" ou "O que você faz para viver?"

A resposta comum para esta pergunta tem a estrutura **`Soy` + [profession]**. Por exemplo:

`Soy ingeniero` - Eu sou um engenheiro.
