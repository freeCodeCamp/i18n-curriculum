# --explanation--

このマテオとカミラの会話は職業にフォーカスしています。

`¿A qué te dedicas?`は「あなたの職業は何ですか？」または「何の仕事をしていますか？」という意味です。

**`Soy` + [職業]** は「私は[職業]です」という意味になります。

スペイン語では、ほとんどの職業名は性別によって変わります。しかし、`-ista`または`-nte`で終わる職業名の中には、男女でまったく同じ形のものもよくあります：

| 職業 | 男性形 | 女性形 |
| :---: | :---: | :---: |
| アナリスト | `analista` | `analista` |
| ジャーナリスト | `periodista` | `periodista` |
| 受付係 | `recepcionista` | `recepcionista` |
| 歌手 | `cantante` | `cantante` |
