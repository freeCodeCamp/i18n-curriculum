# --explanation--

`As well` significa lo mismo que `also` o `too`. Se usa para mostrar que algo aplica a una persona o cosa adicional. Por ejemplo:

`She's joining the project as well.` – Esto significa que la otra persona también se está uniendo al proyecto.
