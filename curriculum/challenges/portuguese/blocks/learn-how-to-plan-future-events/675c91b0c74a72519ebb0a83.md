# --explanation--

`They'll` é a contração de `they will`. É usado para falar sobre algo esperado ou previsto para acontecer no futuro. 

`Likely` significa que algo é provável ou tem uma grande chance de acontecer. Por exemplo:

- `They'll likely ask for more information.` - Isso prevê uma provável solicitação dos clientes.  

- `The feature will likely need additional testing.` - Isso indica uma alta probabilidade de que testes serão necessários.
