# --explanation--

`Server` 指的是一台強大的電腦或系統，透過網路向其他裝置提供服務、資料或資源。例如：  

`The server hosts the company's website。`－這表示該伺端負責儲存並傳送網站內容給存取它的使用者。

`Load` 指系統或資源所承受的工作量、需求或壓力。例如：  

`The system crashed because it couldn't handle the high load during peak hours.`－這表示系統因為在尖峰時段承受大量需求而不堪負荷。
