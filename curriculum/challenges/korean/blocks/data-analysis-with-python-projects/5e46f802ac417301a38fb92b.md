# --instructions--

이 프로젝트에서는 선 차트, 막대 그래프, 박스 플롯을 사용해 시계열 데이터를 시각화할 것입니다. Pandas, Matplotlib, Seaborn을 사용해 2016-05-09부터 2019-12-03까지 freeCodeCamp.org 포럼의 일별 페이지 조회수 데이터셋을 시각화합니다. 데이터 시각화는 방문 패턴을 이해하고 연간 및 월간 성장 추이를 파악하는 데 도움이 됩니다.

다음 작업을 데이터로 완료하세요:

- Pandas를 사용해 "fcc-forum-pageviews.csv"에서 데이터를 가져오세요. 인덱스는 `date` 열로 설정하세요.
- 페이지 조회수가 데이터셋 상위 2.5% 또는 하위 2.5%인 날을 필터링해 데이터를 정리하세요.
- Matplotlib을 사용해 "examples/Figure_1.png"와 유사한 선 차트를 그리는 `draw_line_plot` 함수를 만드세요. 제목은 `Daily freeCodeCamp Forum Page Views 5/2016-12/2019`이어야 합니다. x축 레이블은 `Date`, y축 레이블은 `Page Views`이어야 합니다.
- "examples/Figure_2.png"와 유사한 막대 그래프를 그리는 `draw_bar_plot` 함수를 만드세요. 연도별로 그룹화된 각 월의 일일 평균 페이지 조회수를 보여야 합니다. 범례에는 월 레이블이 표시되고 제목은 `Months`이어야 합니다. 차트에서 x축 레이블은 `Years`, y축 레이블은 `Average Page Views`이어야 합니다.
- Seaborn을 사용해 "examples/Figure_3.png"와 유사한 두 개의 인접한 박스 플롯을 그리는 `draw_box_plot` 함수를 만드세요. 이 박스 플롯들은 주어진 연도 또는 월 내 값의 분포와 시간에 따른 비교를 보여야 합니다. 첫 번째 차트 제목은 `Year-wise Box Plot (Trend)`, 두 번째 차트 제목은 `Month-wise Box Plot (Seasonality)`이어야 합니다. 아래쪽 월 레이블은 `Jan`부터 시작해야 하며 x축과 y축 레이블이 올바르게 지정되어야 합니다. 보일러플레이트에는 데이터를 준비하는 명령어가 포함되어 있습니다.

각 차트마다 데이터 프레임 복사본을 사용하세요.

보일러플레이트에는 이미지 저장 및 반환 명령도 포함되어 있습니다.
