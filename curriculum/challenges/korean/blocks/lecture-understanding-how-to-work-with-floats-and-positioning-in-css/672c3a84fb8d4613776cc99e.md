# --interactive--

CSS에서 위치 지정은 페이지에서 요소가 어떻게 배치되는지 제어할 수 있게 해줍니다. 흔히 쓰이는 위치 지정 방식 두 가지는 정적 위치 지정과 상대 위치 지정입니다. 기본적으로 요소들은 정적으로 위치 지정됩니다. 이는 문서의 정상 흐름을 따라 위에서 아래, 왼쪽에서 오른쪽으로 차례대로 배치된다는 뜻입니다.

정적 위치 지정은 모든 요소의 기본값이며 CSS에서 특별한 선언이 필요하지 않습니다. 정적 위치 지정을 사용할 때는 요소가 문서 내에서 자연스럽게 위치하는 곳에 그대로 있기 때문에 달라진 점을 느끼기 어렵습니다. 다음은 기본 정적 위치 지정을 사용한 문단 예시입니다:

:::interactive_editor

```html
<p>This paragraph is statically positioned.</p>
```

:::

브라우저에서 렌더링하면 이 문단은 문서의 정상 흐름에 따라 자연스러운 위치에 나타납니다. 특별한 스타일이 적용되어 이동하지 않고 레이아웃에서 자연스럽게 위치합니다.

반면 상대 위치 지정은 요소를 문서의 정상 흐름을 방해하지 않고 원래 위치에서 이동시킬 수 있게 해줍니다. 기본 정적 위치에서 새로운 좌표를 주어 요소를 옮기는 것으로 생각하세요. 상대 위치 지정을 적용하는 방법은 다음과 같습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="relative">This paragraph is positioned relatively.</p>
```

```css
body {
  border: solid 1px black;
}

.relative {
  position: relative;
  top: 30px;
  left: 30px;
}
```

:::

이 예시에서 문단은 원래 위치에서 `30px` 아래로, `30px` 오른쪽으로 이동합니다. 하지만 정상 흐름에서 차지하던 공간은 그대로 유지됩니다. 상대 위치 지정은 레이아웃에 영향을 주지 않고 요소를 약간 이동시키고 싶을 때 매우 유용합니다.
