# --description--

要素をアニメーションさせるには、animationプロパティと`@keyframes`ルールについて知る必要があります。animationプロパティはアニメーションの動作を制御し、`@keyframes`ルールはそのアニメーション中に何が起こるかを制御します。animationプロパティは全部で8つありますが、このチャレンジではシンプルにして、まず最も重要な2つを説明します。

`animation-name`はアニメーションの名前を設定し、後で`@keyframes`がどのルールがどのアニメーションに対応するかをCSSに伝えるために使います。

`animation-duration`はアニメーションの長さ（時間）を設定します。

`@keyframes`はアニメーションの期間中に具体的に何が起こるかを指定する方法です。これはアニメーションの特定の「フレーム」に対してCSSプロパティをパーセンテージ（0%から100%）で指定することで行います。これを映画に例えると、0%のCSSプロパティはオープニングシーンでの要素の表示方法、100%のCSSプロパティはエンドクレジット直前の要素の見え方です。CSSは指定された期間内に要素を変化させてシーンを演じます。以下は`@keyframes`とanimationプロパティの使い方を示す例です。

```css
#anim {
  animation-name: colorful;
  animation-duration: 3s;
}

@keyframes colorful {
  0% {
    background-color: blue;
  }
  100% {
    background-color: yellow;
  }
}
```

`anim`というidの要素に対して、上のコードスニペットは`animation-name`を`colorful`に設定し、`animation-duration`を3秒に設定しています。次に`@keyframes`ルールが`colorful`という名前のanimationプロパティにリンクしています。アニメーションの開始時（0%）に色を青に設定し、アニメーションの終了時（100%）に黄色に変化します。開始と終了だけでなく、0%から100%の間の任意のパーセンテージに対しても要素のプロパティを設定できます。

# --instructions--

idが`rect`の要素に対して、`animation-name`を`rainbow`に、`animation-duration`を4秒に設定してアニメーションを作成してください。次に`@keyframes`ルールを宣言し、アニメーションの開始時（`background-color`）に`0%`を青に、アニメーションの中間（`50%`）に緑に、アニメーションの終了時（`100%`）に黄色に設定してください。

```js
const rectElement = document.querySelector('#rect');
```
