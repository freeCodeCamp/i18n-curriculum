# --description--

Die nächste Schleifenart, die Sie lernen werden, heißt `do...while`-Schleife. Sie heißt `do...while`-Schleife, weil sie zuerst immer eine Ausführung des Codes innerhalb der Schleife `do` wird, egal was passiert, und dann die Schleife `while` weiterlaufen lässt, solange die angegebene Bedingung `true` ergibt.

```js
const ourArray = [];
let i = 0;

do {
  ourArray.push(i);
  i++;
} while (i < 5);
```

Das obige Beispiel verhält sich ähnlich wie andere Schleifenarten, und das resultierende Array wird wie `[0, 1, 2, 3, 4]` aussehen. Was die `do...while` jedoch von anderen Schleifen unterscheidet, ist ihr Verhalten, wenn die Bedingung beim ersten Prüfen fehlschlägt. Sehen wir uns das in der Praxis an. Hier ist eine normale `while`-Schleife, die den Code in der Schleife so lange ausführt, wie `i < 5` gilt:

```js
const ourArray = []; 
let i = 5;

while (i < 5) {
  ourArray.push(i);
  i++;
}
```

In diesem Beispiel initialisieren wir den Wert von `ourArray` als leeres Array und den Wert von `i` mit 5. Wenn wir die `while`-Schleife ausführen, ergibt die Bedingung `false`, weil `i` nicht kleiner als 5 ist, also führen wir den Code innerhalb der Schleife nicht aus. Das Ergebnis ist, dass `ourArray` keine Werte hinzugefügt werden, und es wird nach Abschluss des gesamten Codes im obigen Beispiel immer noch wie `[]` aussehen. Schauen Sie sich nun eine `do...while`-Schleife an:

```js
const ourArray = []; 
let i = 5;

do {
  ourArray.push(i);
  i++;
} while (i < 5);
```

In diesem Fall initialisieren wir den Wert von `i` mit 5, genau wie bei der `while`-Schleife. Wenn wir zur nächsten Zeile kommen, gibt es keine Bedingung zu prüfen, also gehen wir in den Code innerhalb der geschweiften Klammern und führen ihn aus. Wir fügen dem Array ein einzelnes Element hinzu und erhöhen dann `i`, bevor wir zur Bedingungsprüfung kommen. Wenn wir schließlich die Bedingung `i < 5` in der letzten Zeile auswerten, sehen wir, dass `i` jetzt 6 ist, was die Bedingungsprüfung nicht besteht, also verlassen wir die Schleife und sind fertig. Am Ende des obigen Beispiels ist der Wert von `ourArray` `[5]`. Im Wesentlichen stellt eine `do...while`-Schleife sicher, dass der Code innerhalb der Schleife mindestens einmal ausgeführt wird. Versuchen wir, eine `do...while`-Schleife zum Laufen zu bringen, indem wir Werte in ein Array einfügen.
