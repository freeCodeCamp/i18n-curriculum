# --description--

- 3의 배수는 `"Fizz"`로 바꾸세요.
- 5의 배수는 `"Buzz"`로 바꾸세요.
- 3과 5의 배수는 `"FizzBuzz"`로 바꾸세요.

def fizz_buzz(n):
    result = []
    for i in range(1, n+1):
        if i % 15 == 0:
            result.append("FizzBuzz")
        elif i % 3 == 0:
            result.append("Fizz")
        elif i % 5 == 0:
            result.append("Buzz")
        else:
            result.append(i)
    return result
