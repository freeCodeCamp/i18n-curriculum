# --description--

用户隐私关乎你控制个人数据如何被收集、使用和在线共享的权利。现代隐私框架依赖几个核心创建模块来保护你的数字足迹。

隐私保护的核心是**个人身份信息（PII）**。PII 是任何可以直接或间接确定你身份的数据。

直接标识符包括你的全名、家庭地址和国家身份证号码，而间接标识符包括你的 IP 地址、设备序列号或精确的位置信息历史。

为了保护这些数据安全，公司必须维护**机密性**。机密性通过使用强加密和访问控件，确保你的私人数据免受未经授权的访问。

如果未经授权的第三方读取你的记录或泄露你的 PII，你的机密性就被破坏了。

隐私框架通过**数据最小化**和**目的限制**来执行这一点。数据最小化规定公司只应收集完成特定任务所需的最少量 PII。

目的限制规定，一旦企业为特定原因收集了你的数据，就不能合法地将其用于其他用途，比如将你的联系信息出售给广告网络。

这些原则直接对抗侵入性的数字追踪。**追踪**是对你在互联网中的在线行为的持续监控。

广告网络使用脚本、设备指纹和浏览器历史日志来追踪你的行为，并在你不知情的情况下编制详细的用户画像。

为了防止这种追踪，用户同意是个人控制的基础。真正的隐私需要知情且显式的同意，这意味着你必须主动选择加入数据追踪，而不是被迫接受。

最后，**隐私法律保障被遗忘权，**也称为数据删除权。该概念赋予你法律权利，在关闭账户后请求组织永久删除其数据库中所有你的个人记录和追踪历史。

你将在下一讲中了解这些隐私法律及其负责机构。
