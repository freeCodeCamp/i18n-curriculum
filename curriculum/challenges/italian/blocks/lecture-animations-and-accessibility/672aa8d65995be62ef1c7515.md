# --interactive--

Le animazioni CSS ti permettono di creare effetti dinamici e visivamente coinvolgenti sulle pagine web senza bisogno di JavaScript o programmazione complessa. Offrono un modo per passare in modo fluido tra stili diversi su un elemento in un intervallo di tempo specificato.

Alla base, un'animazione CSS è composta da due componenti principali: la regola `@keyframes` e la proprietà animation.

La regola `@keyframes` definisce le fasi e gli stili dell'animazione. Specifica quali stili l'elemento deve avere in vari momenti durante l'animazione.

Ecco un esempio:

```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(0);
  }
}
```

Questa regola `@keyframes`, chiamata `slide-in`, definisce un'animazione che sposta un elemento da sinistra a destra. Le percentuali rappresentano il progresso dell'animazione, con `0%` che indica l'inizio e `100%` la fine.

La funzione `translateX` nella tua animazione `@keyframes` controlla la posizione orizzontale di un elemento mentre si anima entrando in vista.

Per applicare questa animazione a un elemento, usi la proprietà `animation`. Questo esempio ripete l'animazione all'infinito così puoi vederla in azione (dovrai abilitare l'editor interattivo per vedere le anteprime):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="sliding-element">Hello, I slide in!</div>
```

```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(0);
  }
}

.sliding-element {
  animation: slide-in 2s ease-out infinite;
}
```

:::

Questo applica l'animazione `slide-in` all'elemento con una durata di 2 secondi e una funzione di temporizzazione `ease-out`.

La proprietà `animation` è in realtà una scorciatoia per diverse proprietà individuali:

`animation-name` che specifica la regola `@keyframes` da usare.

`animation-duration` che imposta quanto tempo deve durare l'animazione.

`animation-timing-function` che definisce come l'animazione procede nel tempo — come `ease`, `linear`, `ease-in-out`.

`animation-delay` che specifica un ritardo prima che l'animazione inizi.

`animation-iteration-count` che imposta quante volte l'animazione deve ripetersi.

`animation-direction` che determina se l'animazione deve andare avanti, indietro o alternarsi.

`animation-fill-mode` che specifica come l'elemento deve essere stilizzato prima e dopo l'animazione.

`animation-play-state` che ti permette di mettere in pausa e riprendere l'animazione.

Puoi usare queste proprietà singolarmente per un controllo più preciso:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="complex-animation">Watch my colors change!</div>
```

```css
.complex-animation {
  animation-name: color-change;
  animation-duration: 3s;
  animation-timing-function: linear;
  animation-delay: 1s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}

@keyframes color-change {
  0% {
    background-color: red;
  }
  50% {
    background-color: blue;
  }
  100% {
    background-color: green;
  }
}
```

:::

Questo crea un'animazione che cambia continuamente il colore di sfondo di un elemento tra rosso, blu e verde.

Le animazioni CSS possono essere attivate da vari eventi, come passare il mouse sopra un elemento:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Hover over me!</button>
```

```css
.button {
  background-color: blue;
  transition: background-color 0.3s;
}

.button:hover {
  background-color: red;
}
```

:::

Anche se questo esempio usa la proprietà `transition`, che è più semplice per effetti base, dimostra come il CSS possa creare elementi animati e interattivi.

È importante notare che, sebbene le animazioni CSS siano potenti, dovrebbero essere usate con moderazione. L'uso eccessivo di animazioni può causare prestazioni scadenti e risultare distraente o problematico per utenti con particolari esigenze di accessibilità. Considera sempre di offrire opzioni per ridurre o disabilitare le animazioni per chi preferisce meno movimento.

Le animazioni CSS offrono un modo per creare esperienze web coinvolgenti e interattive senza dipendere da JavaScript. Capendo i principi delle regole `@keyframes` e delle proprietà `animation`, puoi dare vita ai tuoi progetti web in modo performante e accessibile.
