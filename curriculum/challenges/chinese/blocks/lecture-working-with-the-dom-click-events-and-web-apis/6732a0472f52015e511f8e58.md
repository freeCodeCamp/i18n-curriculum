# --description--

让我们了解一下 API 以及它们为何对网页开发至关重要。API 代表应用编程接口。API 建立了一套规则和协议，使软件应用能够相互通信并高效交换数据。

你可以将它们视为允许开发者基于已实现的更简单的创建块来创建更复杂功能性的构造。

API 有多种类型。Web API 专门为网页应用设计。客户端 JavaScript 开发有不同类型的 Web API。它们不是 JavaScript 本身的一部分。

这些类型的 API 通常分为两大类：浏览器 API 和第三方 API。

浏览器 API 公开浏览器中的数据。作为一名网页开发者，你可以使用 JavaScript 访问和操作这些数据。

它们还提供对各种功能的访问，例如操作网站结构、处理事件、使用存储以及与网络通信。

一些常用的浏览器 API 示例包括：

DOM API，你可以用它来操作 HTML 元素、它们的样式和属性。你将在接下来的课程中学到更多关于 DOM 的内容。它是网页开发中的一个核心概念。

Storage API，用于在用户设备上局部地保存数据。

等等。 

有各种浏览器 API 可供你使用，以处理用户交互并从浏览器获取更多信息，比如用户的当前位置。 

你也可以处理其他类型的元素，例如视频和音频。

浏览器 API 是构建强大网页应用的有用工具。它们已经内置于浏览器中。

但你也可以使用第三方 API，这些 API 并非浏览器默认内置。要使用它们，你需要以某种方式获取它们的代码。通常，它们随附详细的文档，说明如何使用它们的服务。

一个例子是 Google Maps API，你可以用它在你的网站上显示交互式地图。

有各种类型的第三方网页 API，包括天气 API、社交媒体 API、支付 API、数据 API、翻译 API 等。你只需要查找符合你需求的 API。

Web APIs 是构建现代 Web 应用的关键工具，能够与各种服务和数据源交互。通过学习如何杠杆它们的功能，你将扩展为你的创建动态且引人入胜的网站的工具集。
