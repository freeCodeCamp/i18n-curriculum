# --description--

다음 수업들에서는 Python에서 사용하는 세 가지 기본 수열 유형인 리스트, 튜플, 그리고 range에 대해 배울 것입니다.

리스트 데이터 타입은 문자열, 숫자, 또는 다른 리스트로 구성될 수 있는 순서가 있는 요소들의 수열입니다. 리스트는 변경 가능하며 0부터 시작하는 인덱스를 사용하므로 리스트의 첫 번째 요소는 인덱스 0에 있습니다.

리스트의 기본 구문은 다음과 같습니다:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
```

`cities` 리스트에서 요소에 접근하려면 수열 내 인덱스 번호를 참조하면 됩니다. 다음은 `cities` 리스트의 첫 번째 요소에 접근하는 예입니다:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # 'Los Angeles'
```

음수 인덱싱은 리스트의 시작이 아닌 끝에서부터 요소에 접근할 때 사용합니다. 리스트의 마지막 요소에 접근하려면 `0` 대신 `-1`을 이렇게 사용할 수 있습니다:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # 'Tokyo'
```

리스트를 만드는 또 다른 방법은 `list()` 생성자를 사용하는 것입니다. `list()` 생성자는 반복 가능한 객체를 리스트로 변환할 때 이렇게 사용합니다:

```py
developer = 'Jessica'
list(developer) # ['J', 'e', 's', 's', 'i', 'c', 'a']
```

반복 가능한 객체는 한 번에 하나씩 반복할 수 있는 특별한 유형의 객체입니다. Python에서 반복문에 대해서는 나중에 더 배우게 됩니다.

리스트 내 요소의 총 개수를 얻으려면 `len()` 함수를 이렇게 사용할 수 있습니다:

```py
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

특정 인덱스의 값을 업데이트하려면 다음과 같이 할 수 있습니다:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

리스트는 변경 가능하므로 유효한 인덱스 번호만 전달하면 리스트 내 어떤 요소든 업데이트할 수 있습니다. 만약 리스트 범위를 벗어난 인덱스(양수 또는 음수)를 전달하면 `IndexError`가 발생합니다:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list assignment index out of range
"""
```

리스트에서 요소를 제거하려면 `del` 키워드를 이렇게 사용할 수 있습니다:

```py
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

요소가 리스트 안에 있는지 확인하는 것이 유용할 때가 있습니다. 이럴 때는 `in` 키워드를 이렇게 사용할 수 있습니다:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

때때로 리스트 안에 리스트가 중첩되어 있는 경우가 있습니다:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

이 예제에서는 세 가지 인기 있는 프로그래밍 언어를 포함하는 중첩 리스트가 하나 있습니다. 중첩 리스트에 접근하려면 리스트가 0부터 인덱스가 시작하므로 `2` 인덱스를 사용해야 합니다:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

그다음 중첩 리스트에서 두 번째 언어에 접근하려면 `1` 인덱스를 이렇게 사용해야 합니다:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # 'Rust'
```

리스트와 함께 자주 사용하는 또 다른 기법은 값 풀기입니다.

리스트에서 값을 풀어내는 것은 리스트의 값을 새로운 변수에 할당하는 기법입니다. 다음은 `developer` 리스트를 `name`, `age`, `job`라는 새 변수들에 풀어내는 예입니다.

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

여기서 `name`는 `'Alice'` 값을, `age`는 `34` 값을, `job`는 `'Rust Developer'` 값을 가집니다.

리스트에서 남은 요소들을 모두 모아야 할 때는 별표(`*`) 연산자를 이렇게 사용할 수 있습니다:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

이 예제에서 `name`은 여전히 `'Alice'` 값을 가지며, `rest`는 숫자 `34`과 문자열 `'Rust Developer'` 두 개의 요소를 가진 리스트입니다.

할당 연산자 왼쪽에 있는 변수 수가 리스트 내 아이템 총 수와 일치하지 않으면 `ValueError`가 발생합니다:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job, city = developer

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: not enough values to unpack (expected 4, got 3)
"""
```

마지막으로 살펴볼 개념은 슬라이스 연산자(`:`)입니다. 문자열과 마찬가지로 슬라이스 연산자를 사용해 리스트의 일부를 이렇게 접근할 수 있습니다:

```py
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie', 'Brownies']
desserts[1:4] # ['Cookies', 'Ice Cream', 'Pie']
```

이 예제에서 시작 인덱스는 `1`인데, 이는 리스트의 두 번째 항목을 가리킵니다. 그다음 슬라이스 연산자 뒤에 `4`라는 끝 인덱스를 사용했는데, 이 인덱스는 포함하지 않고 그 전까지의 모든 항목을 포함합니다.

슬라이스 연산자 `:`를 사용하면 인덱스 사이의 증가 간격을 지정하는 단계 간격도 설정할 수 있습니다. 예를 들어 다음과 같은 숫자 리스트가 있다고 합시다:

```py
numbers = [1, 2, 3, 4, 5, 6]
```

짝수만 추출하려면 슬라이스 연산자를 이렇게 사용할 수 있습니다:

```py
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

첫 번째 짝수는 인덱스 `1`에 있으므로 시작 인덱스가 됩니다. 리스트 끝까지 가려면 끝 인덱스는 생략합니다. 마지막으로 선택적 단계 간격으로 `2`를 지정해 기본값 `2` 대신 `1`만큼만 증가하도록 합니다.

리스트는 Python 프로그램에서 자주 사용하게 될 유용하고 유연한 자료 구조입니다. 다음 수업에서는 리스트와 함께 사용할 수 있는 일반적인 메서드에 대해 배울 것입니다.
