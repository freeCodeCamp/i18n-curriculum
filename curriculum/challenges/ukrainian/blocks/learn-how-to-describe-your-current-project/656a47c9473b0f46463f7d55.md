# --description--

`infographic` — це візуальне зображення, як-от діаграма або схема, яке використовується для представлення інформації. Наприклад:

`The infographic shows the survey results clearly.` — тут `infographic` означає картинку, яка пояснює дані у легкий для сприйняття спосіб.

`interactive infographic` — це тип інфографіки, що дозволяє глядачеві натискати, наводити курсор або прокручувати, щоб дослідити інформацію. Наприклад:

`The company made an interactive infographic where readers can click to see details about each product.` — тут `interactive infographic` означає візуалізацію, яка не лише показує інформацію, а й дає змогу читачеві взаємодіяти з нею.
