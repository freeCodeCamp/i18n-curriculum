# --explanation--

`Stable` significa que algo é estável e confiável, sem mudanças inesperadas.Por exemplo:

`A stable app doesn't freeze or close by itself.` – Ela continua funcionando corretamente.

`Work without` é uma frase comum que significa que algo funciona enquanto evita um problema ou necessidade.Por exemplo:

`The tool works without an internet connection.` – Ele ainda funciona mesmo quando está offline.

`Crashing` é quando um software ou sistema para de funcionar repentinamente.Por exemplo:

`The browser keeps crashing when I open too many tabs.` – Ele fecha inesperadamente.
