# --description--

Safu za data ni vitu vya JavaScript vinavyoweza kushikilia vipengele vingi tofauti.

```js
var complexArr = [1, 5, "2", "Word", {"name": "James"}];
```

Kwa msingi, kinachotokea nyuma ni kwamba kivinjari chako hutoa nafasi sahihi ya kumbukumbu kwa ajili ya safu hiyo moja kwa moja. Pia nafasi hiyo hubadilika inapohitajika ikiwa utaongeza au kuondoa data.

Hata hivyo, katika ulimwengu wa utendaji wa juu na aina tofauti za vipengele, wakati mwingine unahitaji kuwa maalum zaidi kuhusu ni kiasi gani cha kumbukumbu kinatolewa kwa safu.

<dfn>Safu yenye aina maalum</dfn> ni suluhisho la tatizo hili. Sasa unaweza kusema ni kiasi gani cha kumbukumbu unachotaka kutoa kwa safu. Hapa chini ni muhtasari wa aina tofauti za safu zinazopatikana na ukubwa wa kila kipengele kwa bait.

Kuna njia mbili za kuunda aina hizi za safu. Njia moja ni kuunda moja kwa moja. Hapa chini ni jinsi ya kuunda `Int16Array` yenye urefu wa 3.

```js
var i8 = new Int16Array(3);
console.log(i8);
// Returns [0, 0, 0]
```

Unaweza pia kuunda <dfn>kihifadhi cha muda</dfn> ili kuweka kiasi cha data (kwa bait) unachotaka safu ichukue. **Kumbuka**  
Ili kuunda safu yenye aina maalum kwa kutumia vihifadhi cha muda, unahitaji kuweka idadi ya bait kuwa mara ya bait zilizoorodheshwa hapo juu.

```js
// Create same Int16Array array differently
var byteSize = 6; // Needs to be multiple of 2
var buffer = new ArrayBuffer(byteSize);
var i8View = new Int16Array(buffer);
buffer.byteLength; // Returns 6
i8View.byteLength; // Returns 6
console.log(i8View); // Returns [0, 0, 0]
```

<dfn>Vihifadhi cha muda</dfn> ni vitu vya matumizi ya jumla vinavyobeba data tu. Huwezi kuzipata moja kwa moja. Ili kuzifikia, unahitaji kwanza kuunda <dfn>mtazamo</dfn>.

```js
i8View[0] = 42;
console.log(i8View); // Returns [42, 0, 0]
```

**Kumbuka**  
Safu yenye aina maalum haina baadhi ya njia ambazo safu za kawaida zina kama `.pop()` au `.push()`. Safu hizi pia haziwezi kupitisha `Array.isArray()` inayokagua kama kitu ni safu ya data. Ingawa ni rahisi zaidi, hii inaweza kuwa faida kwa injini za JavaScript zisizo na uwezo mkubwa kutekeleza safu hizi.

# --instructions--

Kwanza unda `buffer` yenye ukubwa wa bait 64. Kisha unda safu yenye aina maalum `Int32Array` yenye mtazamo wa hiyo iitwayo `i32View`.

# --hints--

`buffer` yako inapaswa kuwa na ukubwa wa bait 64.

```js
assert(buffer.byteLength === 64);
```

Mtazamo wako `i32View` wa kihifadhi cha muda unapaswa kuwa na ukubwa wa bait 64.

```js
assert(i32View.byteLength === 64);
```

Mtazamo wako `i32View` wa kihifadhi cha muda unapaswa kuwa na vipengele 16 kwa urefu.

```js
assert(i32View.length === 16);
```
