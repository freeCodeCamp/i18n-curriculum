## CSS-Eigenschaft overflow

- **Definition**: Overflow beschreibt, wie Elemente mit Inhalten umgehen, die die Größe des umgebenden Elements überschreiten oder „überlaufen“. Overflow ist zweidimensional.
- **`overflow-x`**: Die x-Achse bestimmt den horizontalen Überlauf.
- **`overflow-y`**: Die y-Achse bestimmt den vertikalen Überlauf.
- **`overflow`**: Kurzschreibweise-Eigenschaft für `overflow-x` und `overflow-y`. Wenn ein Wert angegeben wird, verwenden beide Überläufe diesen Wert. Bei zwei Werten verwendet der `overflow-x` den ersten und der `overflow-y` den zweiten.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="overflow-box">
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
</div>
```

```css
.overflow-box {
  width: 140px;
  height: 70px;
  border: 2px solid black;
  overflow-x: auto;
  overflow-y: auto;
  white-space:nowrap;
}
```

## CSS-Eigenschaft transform

- **Definition**: Mit dieser Eigenschaft können Sie verschiedene Transformationen auf Elemente anwenden, wie Drehen, Skalieren, Schrägstellen oder Verschieben (Translation) im 2D- oder 3D-Raum.
- **`translate()`-Funktion**: Diese Funktion wird verwendet, um ein Element von seiner aktuellen Position zu verschieben.
- **`scale()`-Funktion**: Diese Funktion ermöglicht es, die Größe eines Elements zu ändern.
- **`rotate()`-Funktion**: Diese Funktion erlaubt es, ein Element zu drehen.
- **`skew()`-Funktion**: Diese Funktion erlaubt es, ein Element schräg zu stellen.
- **Transformationen und Barrierefreiheit**: Wenn Sie transform verwenden, um Inhalte zu verbergen oder anzuzeigen, stellen Sie sicher, dass die Inhalte weiterhin für Screenreader und Tastaturnavigation zugänglich sind. Versteckte Inhalte sollten wirklich verborgen sein, z. B. durch Verwendung von `display: none` oder `visibility: hidden`, und nicht nur visuell außerhalb des Bildschirms verschoben werden.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="transform-box translate">Translate</div>
  <div class="transform-box scale">Scale</div>
  <div class="transform-box rotate">Rotate</div>
  <div class="transform-box skew">Skew</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: column;
  align-items: center;
  gap: 16px;
  flex-wrap: wrap;
}

.transform-box {
  width: 100px;
  height: 100px;
  background: coral;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

/* translate() */
.translate {
  transform: translateX(20px);
}

/* scale() */
.scale {
  transform: scale(1.2);
}

/* rotate() */
.rotate {
  transform: rotate(15deg);
}

/* skew() */
.skew {
  transform: skew(10deg, 5deg);
}
```

## Das Box-Modell

- **Definition**: Im CSS-Box-Modell ist jedes Element von einer Box umgeben. Diese Box besteht aus vier Komponenten: dem Inhaltsbereich, `padding`, `border`, `margin`.
- **Inhaltsbereich**: Der Inhaltsbereich ist der innerste Teil der Box. Er ist der Raum, der den eigentlichen Inhalt eines Elements enthält, wie Text oder Bilder.
- **`padding`**: Der Innenabstand ist der Bereich direkt nach dem Inhaltsbereich. Er ist der Raum zwischen dem Inhaltsbereich und dem Rahmen eines Elements.
- **`border`**: Der Rahmen ist der äußere Rand oder die Umrandung eines Elements im CSS-Box-Modell. Er ist die visuelle Begrenzung des Elements.
- **`margin`**: Der Außenabstand ist der Raum außerhalb des Rahmens eines Elements. Er bestimmt den Abstand zwischen einem Element und anderen Elementen in seiner Umgebung.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="box-model">Content</div>
```

```css
.box-model {
  width: 120px;
  background: lightblue;
  padding: 16px;
  border: 4px solid teal;
  margin: 20px;
  text-align: center;
}
```

:::

## Margin Collapse

- **Definition**: Dieses Verhalten tritt auf, wenn sich die vertikalen Außenabstände benachbarter Elemente überlappen und zu einem einzigen Außenabstand werden, der dem größeren der beiden entspricht. Dieses Verhalten gilt nur für vertikale Außenabstände (oben und unten), nicht für horizontale (links und rechts).

## Die `content-box`- und `border-box`-Eigenschaftswerte

- **`box-sizing`-Eigenschaft**: Diese Eigenschaft wird verwendet, um zu bestimmen, wie die endgültigen `width` und `height` für ein HTML-Element berechnet werden.
- **`content-box`-Wert**: Im `content-box`-Modell bestimmen die `width` und `height`, die Sie für ein Element festlegen, die Abmessungen des Inhaltsbereichs, schließen jedoch nicht den `padding`, `border` oder `margin` ein.
- **`border-box`-Wert**: Mit `border-box` umfassen die `width` und `height` eines Elements den Inhaltsbereich, den `padding` und den `border`, schließen jedoch den `margin` nicht ein.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="sizing-box content-box">Content-box</div>
<div class="sizing-box border-box">Border-box</div>
```

```css
.sizing-box {
  width: 150px;
  padding: 10px;
  border: 4px solid teal;
  margin-bottom: 10px;
  background: peachpuff;
  text-align: center;
}

.content-box {
  box-sizing: content-box;  
}

.border-box {
  box-sizing: border-box;
  background: lightgreen;
}
```

## CSS-Reset

- **Definition**: Ein CSS-Reset ist ein Stylesheet, das alle oder einige der Standardformatierungen entfernt, die Webbrowser auf HTML-Elemente anwenden. Drittanbieter-Optionen für CSS-Resets umfassen `sanitize.css` und `normalize.css`.

## CSS-Eigenschaft filter

- **Definition**: Diese Eigenschaft kann verwendet werden, um verschiedene Effekte zu erzeugen, wie Unschärfe, Farbverschiebung und Kontrastanpassung.
- **`blur()`-Funktion**: Diese Funktion wendet eine Gaußsche Unschärfe auf das Element an. Die Menge wird in Pixeln definiert und stellt den Radius der Unschärfe dar.
- **`brightness()`-Funktion**: Diese Funktion passt die Helligkeit des Elements an. Ein Wert von 0 % macht das Element komplett schwarz, während Werte über 100 % die Helligkeit erhöhen.
- **`contrast()`-Funktion**: Diese Funktion passt den Kontrast des Elements an. Ein Wert von 0 % macht das Element komplett grau, 100 % zeigt das Element normal an, und Werte über 100 % erhöhen den Kontrast.
- **`grayscale()`-Funktion**: Diese Funktion wandelt das Element in Graustufen um. Die Menge wird als Prozentsatz definiert, wobei 100 % komplett graustufig und 0 % das Bild unverändert lässt.
- **`sepia()`-Funktion**: Diese Funktion wendet einen Sepia-Ton auf das Element an. Wie bei Graustufen wird ein Prozentwert verwendet.
- **`hue-rotate()`-Funktion**: Diese Funktion wendet eine Farbtonrotation auf das Element an. Der Wert wird in Grad angegeben und stellt eine Drehung im Farbkreis dar.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <img class="filter blur" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Blur" />
  <img class="filter brightness" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Brightness" />
  <img class="filter contrast" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Contrast" />
  <img class="filter grayscale" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Grayscale" />
  <img class="filter sepia" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Sepia" />
  <img class="filter hue" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Hue Rotate" />
</div>
```

```css
.container {
  display: flex;
  gap: 12px;
  flex-wrap: wrap;
}

.filter {
  width: 120px;
}

/* blur() */
.blur {
  filter: blur(3px);
}

/* brightness() */
.brightness {
  filter: brightness(130%);
}

/* contrast() */
.contrast {
  filter: contrast(150%);
}

/* grayscale() */
.grayscale {
  filter: grayscale(100%);
}

/* sepia() */
.sepia {
  filter: sepia(100%);
}

/* hue-rotate() */
.hue {
  filter: hue-rotate(90deg);
}
```
