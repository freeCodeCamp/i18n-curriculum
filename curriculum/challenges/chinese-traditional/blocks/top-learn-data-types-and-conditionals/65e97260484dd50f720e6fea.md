# --description--

使用反引號定義的字串稱為模板字面值。它們是在 JavaScript 中定義字串的一種新方式。它們允許你在字串中嵌入表達式。這是透過將表達式包裹在 `${}` 中完成的。

例如，以下程式碼：

```javascript
let name = "John";
let age = 25;
let greeting = `Hello, my name is ${name} and I am ${age} years old.`;
```

將導致 `greeting` 變數包含字串 "Hello, my name is John and I am 25 years old."。
