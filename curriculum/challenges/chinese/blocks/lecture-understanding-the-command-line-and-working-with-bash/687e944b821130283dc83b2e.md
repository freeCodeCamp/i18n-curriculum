# --description--

让我们了解一下 Bash 及其一些基本命令。

Bash 代表 Bourne Again SHell，可以说是你在类 Unix 环境中最常遇到的 shell。因此，了解如何导航这个 shell 并使用常见命令非常有益。

第一个命令是 `pwd`，它将当前工作目录打印到终端。 “工作目录”指的是终端当前指向的目录，这对于许多其他命令来说是一个重要的信息。

你的终端很可能指向了错误的目录。`cd` 命令允许你更改目录。你可以指定以斜线（`/`）开头的绝对路径，没有前缀的相对路径，并使用双点（`..`）语法移动到父目录。

下一个命令是 `ls`，它列出你当前工作目录的内容。这有助于查看你的目录中是否存在某个文件或文件夹。`ls` 命令还可以使用标记来显示隐藏文件（`-a` 标记）、文件权限（`-l` 标记）以及其他特色。你将在下一课中学习更多关于命令标记的内容。你也可以使用 `less` 或 `cat` 来查看文件的内容。 

如果目录中不包含你需要的文件或文件夹怎么办？有一些名为 `mkdir` 和 `touch` 的命令允许你创建它们。`mkdir` 创建一个新目录或文件夹，`touch` 创建一个新文件。这两个命令都接受你想创建的文件或文件夹的名称。例如，`touch readme.md` 会创建一个名为 `readme.md` 的新 Markdown 文件。

也许你的新建文件不应该命名为 `readme.md`，而应该命名为 `readthis.md`。幸运的是，`mv` 命令允许你移动（或重命名）文件。它接受旧文件名和新文件名。

`rm` 命令允许你删除（或移除）一个文件，使用 `-r` 标记将删除一个目录。有时文件可能受到保护，你需要包含 `-f` 标记。

`cp` 命令可用于将文件或目录拷贝到新位置。与 `mv` 命令不同，`cp` 命令不会删除原始文件——并且为了拷贝目录，你需要传递 `-r` 标记。

`echo` 命令可以被视为 Bash 中等同于 `console.log()` 或 `print()` 函数的功能。`echo` 接受一个用引号包裹的字串参数，并将其打印到终端。虽然在终端运行它只是为了在终端中得到文本看起来有些无聊，但你实际上可以将 `echo` 链接到控件符号。`>` 符号允许你指定一个文件名，用新字串创建或覆盖该文件，比如 `echo "Naomi was here." > readme.md`，而 `>>` 符号则会追加到该文件。

还有许多 Bash 命令可供你使用，例如 `head`、`tail`、`chown` 和 `chmod`。在本课中不可能涵盖所有命令。但你可以使用 `man` 命令查看几乎任何命令的手册或帮助页面。

你很可能会记住你最常用的命令。对于其他命令，你总是可以查看手册页。
