# --explanation--

Un `full scan` è un controllo di sicurezza completo che esamina un intero sistema o rete alla ricerca di minacce. Per esempio:

- `You should run a full scan to check for viruses.` - È una buona idea scansionare l'intero sistema per eventuali file dannosi.

- `The IT team performs a full scan of the network every month.` - Il team tecnico controlla tutti i sistemi di rete per problemi di sicurezza una volta al mese.
