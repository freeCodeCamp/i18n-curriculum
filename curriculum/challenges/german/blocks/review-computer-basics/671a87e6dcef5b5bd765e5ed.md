## Grundlagen von Computer, Internet und Entwickler-Tools verstehen

- **Motherboard**: enthält den gesamten Speicher, die Anschlüsse und Festplatten, die zum Betrieb des Computers benötigt werden. Es dient als Hauptplatine des Computers.
- **Central Processing Unit (CPU)**: ein Prozessor, der für die Ausführung von Anweisungen und Berechnungen verantwortlich ist.
- **Random Access Memory (RAM)**: ein temporärer Speicherort für die CPU des Computers.
- **Hard Disk Drive (HDD)**: ein dauerhafter Speicherort, der Daten auch dann speichert, wenn der Computer ausgeschaltet ist.
- **Solid State Drive (SSD)**: nichtflüchtiger Flash-Speicher, der anstelle einer Festplatte verwendet werden kann.
- **Power Supply Unit (PSU)**: verantwortlich für die Umwandlung des Stroms aus der Steckdose in eine Form, die der Computer nutzen kann.
- **Graphics Processing Unit (GPU)**: verantwortlich für das Rendern von Bildern auf dem Computerbildschirm.
- **Verschiedene Arten von Internetdienstanbietern**: Ein Internetdienstanbieter (ISP) ist ein Unternehmen, das Zugang zum Internet bereitstellt. Es gibt verschiedene Arten von ISPs, darunter Einwahl, DSL, Kabel, Glasfaser und Satellit.
- **Sichere Möglichkeiten, sich an Ihrem Computer anzumelden**: Beispiele für sichere Anmeldemethoden sind die Verwendung eines starken Passworts, das Aktivieren der Zwei-Faktor-Authentifizierung und die Nutzung eines Passwortmanagers.
- **Integrierte Entwicklungsumgebung (IDE)**: ein Werkzeug, das Entwickler dabei unterstützt, Code effizient zu schreiben, zu testen und zu debuggen.
- **Code-Editor**: ein Werkzeug, das Entwickler zum Schreiben und Debuggen von Code verwenden.
- **Git**: ein beliebtes Versionskontrollsystem, das Entwicklern ermöglicht, Änderungen im Code nachzuverfolgen und mit anderen zusammenzuarbeiten.
- **Cloud-basierte Hosting-Dienste für Repositories**: Ein Repository ist ein Speicherort für Projektdateien und Versionsverlauf. Beliebte cloud-basierte Hosting-Dienste für Repositories sind GitHub, GitLab und Bitbucket.
- **Paketmanager**: Werkzeuge, die Entwicklern helfen, den Prozess des Hinzufügens, Aktualisierens und Entfernens von Bibliotheken und Projektabhängigkeiten zu vereinfachen. Beispiele sind npm, pip und Maven.
- **Testbibliotheken und Frameworks**: Tests werden in der Softwareentwicklung durchgeführt, um sicherzustellen, dass der Code wie erwartet funktioniert. Beispiele für Testbibliotheken und Frameworks sind Jest, PHPUnit und JUnit.

## Arbeiten mit Dateien, Dateisystemen und Medienformaten

- **Bewährte Vorgehensweisen für die Benennung von Dateien**: Sie sollten Ihre Dateien so benennen, dass sie leicht verständlich und wartbar sind. Zum Beispiel ist `about-us.html` ein aussagekräftigerer Name als `page1.html`.
- **Root-Verzeichnis**: oberstes Verzeichnis in einem Dateisystem. Verzeichnis ist ein anderer Begriff für Ordner.
- **Markdown**: eine Auszeichnungssprache, die häufig für Dokumentationen und `README`-Dateien verwendet wird. Eine `README`-Datei ist eine Datei, die Informationen über ein Projekt enthält, wie z. B. wie man es installiert und verwendet.
- **`index.html`**: stellt die Standardseite dar, die angezeigt wird, wenn ein Benutzer eine Website besucht.
- **Erstellen, Verschieben und Löschen von Dateien und Ordnern mit Explorer/Finder**: Explorer ist der Dateimanager in Windows, und Finder ist der Dateimanager in macOS. Sie können diese Werkzeuge verwenden, um Dateien und Ordner zu erstellen, zu verschieben und zu löschen.
- **Suche nach Dateien und Ordnern**: Sie können die Suchfunktion in Explorer oder Finder verwenden, um Dateien und Ordner auf Ihrem Computer zu finden.
- **HTML-, CSS- und JS-Dateitypen**: Die Dateiendung `.html` wird für HTML-Dateien verwendet, `.css` für CSS-Dateien und `.js` für JavaScript-Dateien.
- **Gängige Bild- und Grafikformate**: `JPEG` und `PNG` sind gängige Bilddateiformate. `GIF` ist ein weiteres gängiges Bilddateiformat, das Animationen unterstützt. `SVG` ist ein Dateiformat für Vektorgrafiken.
- **Gängige Audio- und Videoformate**: Das Format `MP3` wird häufig für Audiodateien verwendet. Das Format `MP4` wird häufig für Videodateien verwendet. Das Format `MOV` wurde von Apple entwickelt und wird häufig für Videodateien verwendet.
- **Gängige Schriftartformate**: Das Format `TTF` wird häufig für TrueType-Schriftarten verwendet. Das Format `WOFF` wird häufig für Web-Schriftarten verwendet. Der Nachfolger von WOFF ist `WOFF2`, der eine bessere Kompression bietet.
- **ZIP**: ein Dateiformat, das zum Komprimieren von Dateien und Ordnern verwendet wird.

## Effektives Surfen im Web

- **Was ist ein Webbrowser?**: eine Softwareanwendung, die es Benutzern ermöglicht, Websites im Internet zu besuchen und anzuzeigen.
- **Was ist eine Suchmaschine?**: ein Werkzeug, das es Benutzern ermöglicht, Informationen im Internet zu suchen. Beispiele sind Google, Bing und Yahoo.
- **Gängige Webbrowser**: Einige Beispiele für gängige Webbrowser sind Google Chrome, Mozilla Firefox und Microsoft Edge.
- **Gängige Suchstrategien**: Sie können `site:` gefolgt von der URL einer Website verwenden, um Inhalte auf dieser Website zu suchen. Sie können `filetype:` gefolgt von einer Dateiendung verwenden, um Dateien dieses Typs zu suchen. Sie können einen Suchbegriff mit einem Minuszeichen voranstellen, um Ergebnisse auszuschließen, die diesen Begriff enthalten. Sie können einen Suchbegriff mit einem Pluszeichen voranstellen, um Ergebnisse einzuschließen, die diesen Begriff enthalten.
