**목표:** 다음과 기능적으로 유사한 앱을 만드세요: <a href="https://codepen.io/freeCodeCamp/full/Myvqmo/" target="_blank" rel="noopener noreferrer nofollow">https://codepen.io/freeCodeCamp/full/Myvqmo/</a>.

Twitch API는 개발자가 더 넓은 Twitch 커뮤니티를 위한 창의적인 통합을 구축할 수 있도록 하는 RESTful API입니다.

아래 사용자 스토리를 충족하고 모든 테스트를 통과하세요. 필요한 라이브러리나 API를 자유롭게 사용하세요. 자신만의 스타일을 담으세요.

**사용자 스토리:** freeCodeCamp가 현재 Twitch.tv에서 스트리밍 중인지 볼 수 있습니다.

**사용자 스토리:** 상태 출력 부분을 클릭하면 freeCodeCamp의 Twitch.tv 채널로 바로 이동할 수 있습니다.

**사용자 스토리:** Twitch 사용자가 현재 스트리밍 중이라면, 그들이 스트리밍하는 내용에 대한 추가 정보를 볼 수 있습니다.

**힌트:** 정기적으로 스트리밍하는 Twitch.tv 사용자 이름들의 배열이 있습니다: `["ESL_SC2", "OgamingSC2", "cretetion", "freecodecamp", "storbeck", "habathcx", "RobotCaleb", "noobs2ninjas"]`

**업데이트:** API 사용 조건 변경으로 Twitch.tv는 API 키를 요구하지만, 우회 방법을 마련했습니다. Twitch의 API 기본 URL 대신 <a href="https://twitch-proxy.freecodecamp.rocks/" target="_blank" rel="noopener noreferrer nofollow">https://twitch-proxy.freecodecamp.rocks/</a>를 사용하면 API 키 등록 없이도 계정 정보를 얻을 수 있습니다.

완료하면 CodePen에 프로젝트 링크를 포함하고 "이 도전 과제를 완료했습니다" 버튼을 클릭하세요.

프로젝트에 대한 피드백은 <a href="https://forum.freecodecamp.org/c/project-feedback/409" target="_blank" rel="noopener noreferrer nofollow">freeCodeCamp 포럼</a>에서 받을 수 있습니다.
