# --description--

Si quieres tener una lista de elementos donde el orden no importa, como una lista de compras de elementos que se pueden comprar en cualquier orden, entonces puedes usar una lista desordenada.

Las listas desordenadas se crean utilizando el elemento `<ul>`, y cada elemento dentro de la lista se crea utilizando el elemento de lista `<li>`.

Cada elemento de lista en una lista desordenada comienza con un punto:
