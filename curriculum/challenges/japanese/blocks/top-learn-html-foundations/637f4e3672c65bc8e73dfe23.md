# --description--

HTMLボイラープレートを完成させるために必要な最後の要素は`<body>`要素です。ここにはユーザーに表示されるすべてのコンテンツ、つまりテキスト、画像、リスト、リンクなどが入ります。

ボイラープレートを完成させるには、`body`ファイルに`index.html`要素を追加してください。`body`要素も`html`要素の中にあり、常に`head`要素の下に配置されます。具体的には次のようになります。

`index.html`ファイルの内容を削除して、ボイラープレートを記憶だけで書き出す練習をして筋肉記憶をつけましょう。最初は詰まったらレッスン内容を見ても構いません。何度か記憶だけで書けるようになるまで続けてください。

この[HTML validator](https://www.freeformatter.com/html-validator.html)でボイラープレートを検証してください。バリデーターはマークアップが正しいかを確認し、終了タグの抜けやHTML内の余分なスペースなど、気づかずに繰り返している構文エラーについてフィードバックをくれる優れた学習ツールです。
