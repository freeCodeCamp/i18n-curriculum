# --description--

赤と青のマーカーは同じように見えますが、緑のマーカーは以前よりかなり薄くなっています。これは、`green` カラーキーワードが実際にはより暗い色合いで、黒と緑の最大値のほぼ中間の色だからです。

`.two` の CSS ルールで、`rgb` 関数の緑の値を `127` に設定して、その強度を下げてください。
