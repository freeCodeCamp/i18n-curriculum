# --explanation--

`Going to` — це фраза, яка описує дії, що відбудуться в майбутньому, або наміри людини. Наприклад:  

`She is going to start her new job next week.` — це означає, що вона планує почати свою нову роботу в майбутньому.

`Online` означає бути підключеним до інтернету або користуватися інструментами, що базуються на інтернеті. Наприклад:  

`The developer worked online to collaborate with the team using a shared coding platform.` — це означає, що розробник використовував інтернет для підключення та роботи з іншими на цифровій платформі для завдань з кодування.
