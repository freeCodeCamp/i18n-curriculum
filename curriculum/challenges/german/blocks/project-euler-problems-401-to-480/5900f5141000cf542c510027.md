# --description--

Sei $n$ eine positive ganze Zahl.

Ein sechsseitiger Würfel wird $n$-mal geworfen. Sei $c$ die Anzahl der Paare von aufeinanderfolgenden Würfen, die denselben Wert zeigen.

Zum Beispiel, wenn $n = 7$ ist und die Werte der Würfe (1, 1, 5, 6, 6, 6, 3) sind, dann ergeben die folgenden Paare von aufeinanderfolgenden Würfen denselben Wert:

$$\begin{align}
  & (\underline{1}, \underline{1}, 5, 6, 6, 6, 3) \\\\
  & (1, 1, 5, \underline{6}, \underline{6}, 6, 3) \\\\
  & (1, 1, 5, 6, \underline{6}, \underline{6}, 3)
\end{align}$$

Daher ist $c = 3$ für (1, 1, 5, 6, 6, 6, 3).

Definieren Sie $C(n)$ als die Anzahl der Ergebnisse beim $n$-maligen Werfen eines sechsseitigen Würfels, sodass $c$ den Wert $π(n)$ nicht überschreitet.<sup>1</sup>

Zum Beispiel gilt $C(3) = 216$, $C(4) = 1290$, $C(11) = 361\\,912\\,500$ und $C(24) = 4\\,727\\,547\\,363\\,281\\,250\\,000$.

Definieren Sie $S(L)$ als $\sum C(n)$ für $1 ≤ n ≤ L$.

Zum Beispiel ist $S(50)\bmod 1\\,000\\,000\\,007 = 832\\,833\\,871$.

Finden Sie $S(50\\,000\\,000)\bmod 1\\,000\\,000\\,007$.

<sup>1</sup> $π$ bezeichnet die Primzahler-Funktion, das heißt $π(n)$ ist die Anzahl der Primzahlen $≤ n$.
