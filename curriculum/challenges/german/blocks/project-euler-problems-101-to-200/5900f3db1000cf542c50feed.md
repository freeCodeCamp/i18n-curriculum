# --description--

In der folgenden Gleichung sind x, y und n positive ganze Zahlen.

$$\frac{1}{x} + \frac{1}{y} = \frac{1}{n}$$

Es kann überprüft werden, dass es für `n` = 1260 genau 113 verschiedene Lösungen gibt und dies der kleinste Wert von `n` ist, für den die Gesamtzahl der verschiedenen Lösungen über hundert liegt.

Was ist der kleinste Wert von `n`, für den die Anzahl der verschiedenen Lösungen vier Millionen übersteigt?

**Hinweis:** Diese Aufgabe ist eine deutlich schwierigere Version von Aufgabe 108 und da sie weit über die Grenzen eines Brute-Force-Ansatzes hinausgeht, erfordert sie eine clevere Implementierung.
