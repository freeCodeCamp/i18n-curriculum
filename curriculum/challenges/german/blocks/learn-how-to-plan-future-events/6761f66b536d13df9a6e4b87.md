### --feedback--

Dieses Wort beschreibt Abfragen, die vordefinierte Werte verwenden, um die Sicherheit zu erhöhen. Es ist ein langes Wort, das oft im Datenbankkontext verwendet wird.

# --explanation--

`Parameterized` stammt vom Grundwort `parameter`, das einen Wert oder Platzhalter bezeichnet, der steuert, wie etwas funktioniert.

Wenn man sagt, eine Abfrage sei `parameterized`, bedeutet das, dass sie vordefinierte Werte (Parameter) verwendet, anstatt Benutzereingaben direkt einzufügen.

Das hilft, die Sicherheit der Datenbank zu verbessern, indem bestimmte Angriffe wie SQL-Injection verhindert werden. Zum Beispiel:

- `The developer used parameterized queries to secure the database.` – Das bedeutet, dass Platzhalter verwendet wurden, um sicherzustellen, dass kein schädlicher Code eingefügt werden kann.
