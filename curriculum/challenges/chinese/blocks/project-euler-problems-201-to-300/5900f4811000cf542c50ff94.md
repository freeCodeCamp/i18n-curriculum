# --description--

从起始值 $a_1$ 以以下方式获得一个修改后的整数 Collatz 序列：

如果 `a_n` 可被 3 整除，则 $a_{n + 1} = \frac{a_n}{3}$。我们将其表示为一个大的向下步骤，`"D"`。

当 $a_n$ 除以 3 余数为 1 时，$a_{n + 1} = \frac{4a_n + 2}{3}$。我们将其表示为一个向上步骤，"U"。

$a_{n + 1} = \frac{2a_n - 1}{3}$，如果 $a_n$ 除以 3 的余数为 2。我们将其表示为一个小的向下步骤，"d"。

当某个 $a_n = 1$ 时，序列终止。

给定任意整数，我们可以列出步骤序列。例如，如果 `a_1 = 231`，那么序列 `\{a_n\} = \{231, 77, 51, 17, 11, 7, 10, 14, 9, 3, 1\}` 对应的步骤是 `"DdDddUUdDD"`。

当然，还有其他序列以相同的序列 `DdDddUUdDD...` 开头。

例如，如果 $a_1 = 1004064$，那么序列是 DdDddUUdDDDdUDUUUdDdUUDDDUdDD。

事实上，1004064 是以序列 DdDddUUdDD 开头且大于 {10}^6 的最小可能的 `a_1`。

大于 {10}^{15} 并以序列 "UDDDUdddDDUDDddDdDddDDUDDdUUDd" 开头的最小 `a_1` 是多少？
