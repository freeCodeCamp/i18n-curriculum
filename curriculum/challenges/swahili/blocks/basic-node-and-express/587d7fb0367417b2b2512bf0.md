# --description--

Seva ya HTML kawaida huwa na saraka moja au zaidi zinazopatikana kwa mtumizi. Unaweza kuweka hapo mali zisizobadilika zinazohitajika na programu yako (majalada ya mitindo, maandishi ya programu, picha).

Katika Express, unaweza kuweka utendaji huu kwa kutumia programu ya kati `express.static(path)`, ambapo kigezo `path` ni njia kamili ya saraka inayohifadhi mali hizo.

Kama hujui middleware ni nini... usijali, tutajadili kwa kina baadaye. Kwa msingi, middleware ni vitendakazi vinavyokamata mshughulikiaji wa njia ya data, na kuongeza aina fulani ya taarifa. Middleware inahitaji kuwekwa kwa kutumia njia `app.use(path, middlewareFunction)`. Hoja ya kwanza `path` ni hiari. Ukikosa kuipatia, middleware itatekelezwa kwa maombi yote.

# --instructions--

Weka middleware `express.static()` kwenye njia ya data `/public` kwa `app.use()`. Njia kamili ya saraka ya mali ni `__dirname + /public`.

Sasa programu yako inapaswa kuweza kuhudumia faili la mtindo wa CSS. Kumbuka kuwa faili `/public/style.css` linatajwa katika `/views/index.html` katika muundo wa msingi wa mradi. Ukurasa wako wa mbele unapaswa kuonekana bora kidogo sasa!

# --hints--

Programu yako inapaswa kuhudumia mafaili ya mali kutoka saraka ya `/public` kwa njia ya data `/public`

```js
  const response = await fetch(code + '/public/style.css');
  if (!response.ok) {
    throw new Error(await response.text());
  }
  const data = await response.text();
  assert.match(
    data,
    /body\s*\{[^\}]*\}/,
    'Your app does not serve static assets'
  );
```

Programu yako haipaswi kuhudumia mafaili kutoka saraka nyingine yoyote isipokuwa saraka ya `/public`

```js
  const response = await fetch(code + '/server.js');
  if (response.ok) {
    const data = await response.text();
    assert.equal(
      response.status + '',
      404 + '',
      'Your app must serve files only from "public" directory'
    );
  } else {
    assert.equal(
      response.status + '',
      404 + '',
      'Your app must serve files only from "public" directory'
    );
  }
```
