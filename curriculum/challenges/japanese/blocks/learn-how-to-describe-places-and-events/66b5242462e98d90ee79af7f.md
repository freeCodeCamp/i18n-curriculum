# --explanation--

`Might` は可能性や不確実性を表すために使われます。何かが可能であることを示しますが、確実ではありません。例えば：

- `The system might crash if too many users log in at the same time.` - システムがクラッシュする可能性はありますが、確実ではありません。

- `He might finish the coding task by the end of the day.` - 彼はタスクを終えられるかもしれませんが、確定ではありません。
