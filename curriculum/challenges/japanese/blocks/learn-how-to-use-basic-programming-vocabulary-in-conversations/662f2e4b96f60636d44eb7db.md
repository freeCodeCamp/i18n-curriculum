# --description--

`class`は特に`object-oriented programming`で基本的な概念です。これはオブジェクトを作成するための設計図のようなものです。`class`は、`methods`から作成されたオブジェクトが持つことができる属性（プロパティ）や動作（`class`）を定義します。

例えば、`Person`クラスは`name`や`age`のようなプロパティを持ち、`talk()`や`walk()`のような動作を持つかもしれません。

`method`はオブジェクトに関連付けられた関数です。`Methods`は`class`から作成されたオブジェクトが実行できる動作やアクションを定義します。これはオブジェクトに何をすべきか指示する命令のようなものです。

例えば、`Calculator class`では、`method`は`add(number1, number2)`かもしれません。これは2つの数字を足し合わせます。
