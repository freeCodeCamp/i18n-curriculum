# --interactive--

У попередньому уроці ви вперше познайомилися з елементом `iframe`. У цьому уроці ви дізнаєтеся більше про те, як працювати з елементом `iframe`. Цей елемент означає рядковий фрейм. Це рядковий елемент, який використовується для вставлення іншого HTML-вмісту безпосередньо в HTML-сторінку. Цей HTML-вміст може бути відео, картою, іншим HTML-елементом або навіть іншими вебсторінками.

Ось приклад вставлення популярного курсу freeCodeCamp з YouTube. Щоб побачити інше відео, увімкніть інтерактивний редактор і змініть значення `src` на відео на ваш вибір.

:::interactive_editor

```html
<iframe
  width="400"
  height="400"
  src="https://www.youtube.com/embed/PkZNo7MFNFg?si=-UBVIUNM3csdeiWF"
  title="Learn JavaScript - Full Course for Beginners (YouTube video)"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
  referrerpolicy="strict-origin-when-cross-origin"
  allowfullscreen
></iframe>
```

:::

Атрибут `src` вказує URL сторінки, яку ви хочете вставити. Атрибут `width` задає ширину `iframe`. Атрибут `height` задає висоту `iframe`. Атрибут `allowfullscreen` дозволяє користувачу відображати `iframe` у повноекранному режимі. Також корисно вказати атрибут `title` для `iframe`, оскільки це важливо для доступності.

Атрибут `allow`, з іншого боку, дозволяє визначити, що iframe може або не може робити. Це називається allowlist (дозволений список). У наведеному вище прикладі додавання `clipboard-write` дозволяє вставленій сторінці записувати елементи у ваш буфер обміну. Елементи в allowlist можна розділяти крапкою з комою або пробілами, і обидва варіанти можна використовувати разом.

Зверніть увагу, що відео може бути звідки завгодно. Воно не обов’язково має бути з відеосервісів, як YouTube чи Vimeo.

Не забувайте, що ви також можете вставити карту, іншу вебсторінку або прямий HTML всередину елемента `iframe`. Ось приклад вставленої карти.

Спробуйте взаємодіяти з картою, збільшуючи та зменшуючи масштаб.

:::interactive_editor

```html
<h1>A Map from Openstreetmap.org Embedded with the iframe Element</h1>

<iframe
  width="425"
  height="350"
  src="https://www.openstreetmap.org/export/embed.html?bbox=3.006134033203125%2C6.150112578753815%2C3.6357879638671875%2C6.749850810550778&amp;layer=mapnik"
  title="Map of Lagos area, Nigeria"
  style="border: 1px solid black"
>
</iframe>
<br />
<small>
  <a href="https://www.openstreetmap.org/#map=11/6.4501/3.3210">
    View Larger Map
  </a>
</small>
```

:::

Якщо ви хочете вставити прямий HTML всередину елемента `iframe`, вам потрібно використовувати атрибут `srcdoc` замість `src`.
