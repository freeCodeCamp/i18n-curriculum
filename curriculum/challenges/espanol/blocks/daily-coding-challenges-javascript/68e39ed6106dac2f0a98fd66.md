# --description--

El 2 de noviembre de 1988, se lanzó el primer gusano importante de internet, infectando alrededor del 10% de las computadoras conectadas a internet después de solo un día.

En este desafío, se te da un número de días que han pasado desde que se liberó un gusano de internet, y necesitas determinar cuántas computadoras están infectadas usando las siguientes reglas:

- En el día 0, la primera computadora está infectada.
- Cada día siguiente, el número de computadoras infectadas se duplica.
- Cada 3 días, se aplica un parche después de que el virus se propaga y reduce el número de computadoras infectadas en un 20%. Redondea el número de computadoras parcheadas hacia arriba al número entero más cercano.

Por ejemplo, en:

- Día 0: 1 computadora en total está infectada.
- Día 1: 2 computadoras en total están infectadas.
- Día 2: 4 computadoras en total están infectadas.
- Día 3: 8 computadoras en total están infectadas. Luego, aplica el parche: 8 infectadas * 20% = 1.6 parcheadas. Redondea 1.6 hacia arriba a 2. 8 computadoras infectadas - 2 parcheadas = 6 computadoras en total infectadas después del día 3.

Devuelve el número total de computadoras infectadas después de que haya pasado la cantidad de días indicada.

```js
function infected(days) {
  let infected = 1;

  for (let day = 1; day <= days; day++) {
    infected *= 2;

    if (day % 3 === 0) {
      let patched = Math.ceil(infected * 0.2);
      infected -= patched;
    }
  }

  return infected;
}
```
