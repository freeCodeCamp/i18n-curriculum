# --description--

El problema de Josephus es un acertijo matemático con una descripción sombría: $n$ prisioneros están parados en un círculo, numerados secuencialmente del $0$ al $n-1$.

Un verdugo camina alrededor del círculo, comenzando desde el prisionero $0$, eliminando a cada $k$-ésimo prisionero y matándolo.

A medida que el proceso avanza, el círculo se vuelve cada vez más pequeño, hasta que solo queda un prisionero, quien es liberado.

Por ejemplo, si hay $n=5$ prisioneros y $k=2$, el orden en que los prisioneros son asesinados (llamémoslo la "secuencia de asesinatos") será 1, 3, 0 y 4, y el sobreviviente será el #2.

Dado cualquier $n, k > 0$, descubre qué prisionero será el último sobreviviente.

En un incidente así, había 41 prisioneros y cada 3<sup>rd</sup> prisionero estaba siendo asesinado ($k=3$).

Entre ellos había un tipo inteligente llamado Josephus que resolvió el problema, se situó en la posición sobreviviente y vivió para contar la historia.

¿Cuál número fue él?

# --instructions--

Escribe una función que tome el número inicial de prisioneros y `k` como parámetros y devuelva el número del prisionero que sobrevive.

```js
function josephus(init, kill) {
  const arr = Array.from(Array(init).keys());
  let curr = -1
  while (arr.length > 1) {
    curr = (curr + kill) % arr.length;
    arr.splice(curr, 1);
    curr--;
  }
  return arr[0];
}
```
