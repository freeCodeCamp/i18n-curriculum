# --description--

O possessivo `'s` mostra que algo pertence a alguém ou algo, como em `Brian's book`, ou seja, o livro pertence a Brian.

`Dealing` significa gerenciar ou lidar com algo.Por exemplo, `When dealing with homework, it's important to be focused.`

`Function's signature` na programação é como uma carteira de identidade para uma função.Ele diz o nome da função, o que ela precisa (parâmetros) e o que ela retorna (`return type`).

O `return type` de uma função é o que a função retorna depois que é executada, como uma máquina de venda automática que retorna um refrigerante depois de colocar o dinheiro.
