# --description--

تقريبًا كل العناصر في صفحة HTML هي مجرد قطع محتوى ملفوفة داخل وسوم البداية ووسوم الإغلاق الخاصة بـ HTML.

تُخبر وسوم البداية المتصفح أن هذا هو بداية عنصر HTML. تتكون من كلمة مفتاحية محاطة بأقواس زاوية `<>`. على سبيل المثال، وسم بداية الفقرة يبدو هكذا: `<p>`.

تُخبر وسوم الإغلاق المتصفح بمكان انتهاء العنصر. هي مشابهة جدًا لوسوم البداية؛ الفرق الوحيد هو وجود شرطة مائلة قبل الكلمة المفتاحية. على سبيل المثال، وسم إغلاق الفقرة يبدو هكذا: `</p>`.
