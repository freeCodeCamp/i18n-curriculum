# --description--

`Object-oriented programming`（OOP）は、`objects`の概念に基づくプログラミングパラダイムで、データとコードを含むことができます。データはフィールド（属性やプロップとして知られることが多い）という形で、コードは手続き（メソッドとして知られることが多い）という形で表されます。

`OOP`の一般的な例は、JavaやPythonのようなプログラミング言語でクラスを作成することです。このクラスはオブジェクトの構造を定義し、その属性やメソッドを含みます。例えば、`Car`クラスには`color`や`model`のような属性、`drive()`や`brake()`のようなメソッドが含まれるかもしれません。

この文脈での`oriented`という用語は、特定の目的や原則に沿っていることを意味します。

例えば、freeCodeCampのレスポンシブウェブデザインカリキュラムは、プロジェクトを完成させることを中心に構成されているため、`project-oriented`と表現されます。つまり、教え方が単なる理論知識ではなく実践的な応用にフォーカスしているということです。
