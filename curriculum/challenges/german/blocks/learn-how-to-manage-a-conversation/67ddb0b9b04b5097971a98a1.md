# --explanation--

In der gesprochenen Alltagssprache lassen Sie oft Wörter weg, wenn die Bedeutung trotzdem klar bleibt. In diesem Fall ist `Anything else we should discuss?` eine verkürzte Form von `Is there anything else we should discuss?`. Solche Auslassungen sind im Gespräch üblich. Ein weiteres Beispiel:

`Need any help?` ist eine kürzere Art zu sagen `Do you need any help?` – Das Subjekt und das Hilfsverb werden weggelassen, aber die Bedeutung bleibt gleich.
