# --explanation--

Die `Present Continuous`-Zeitform wird verwendet, um Handlungen zu beschreiben, die gerade jetzt passieren. Um auszudrücken, dass etwas **nicht** passiert, wird die Verneinungsform verwendet.

Im Satz `The live preview isn't showing` ist `isn't` eine Kontraktion von `is not`, wodurch der Satz verneint wird.

Das zeigt an, dass die Live-Vorschau nicht wie erwartet funktioniert. Tom berichtet also, dass die Live-Vorschau derzeit in Visual Studio Code nicht sichtbar ist.
