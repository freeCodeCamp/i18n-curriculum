# --description--

Una stringa è una sequenza di caratteri racchiusa da virgolette singole o doppie. In alcuni linguaggi di programmazione, i caratteri racchiusi da virgolette singole vengono trattati in modo diverso rispetto a quelli racchiusi da virgolette doppie, ma in Python vengono trattati allo stesso modo. Quindi, puoi usare entrambe quando lavori con le stringhe. Ecco alcuni esempi di stringhe:

```python
my_str_1 = 'Hello'
my_str_2 = "World"
```

Se ti serve una stringa su più righe, puoi usare le virgolette triple doppie o singole:

```python
my_str_3 = """Multiline
string"""
my_str_4 = '''Another
multiline
string'''
```

Se la tua stringa contiene virgolette singole o doppie, hai due opzioni:

- Usa il tipo di virgolette opposto. Cioè, se la tua stringa contiene virgolette singole, usa le virgolette doppie per racchiuderla, e viceversa:
    

```python
msg = "It's a sunny day"
quote = 'She said, "Hello World!"'
```

- Fai l’escape della virgoletta singola o doppia nella stringa con una barra rovesciata (`\`). Con questo metodo, puoi usare sia virgolette singole sia doppie per racchiudere la stringa stessa:
    

```python
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

A volte, potresti dover verificare se una stringa contiene uno o più caratteri. Per questo, Python fornisce l’operatore `in`, che restituisce un booleano che indica se il carattere o i caratteri esistono o meno nella stringa.

Ecco alcuni esempi:

```python
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

Ora vediamo come ottenere la lunghezza di una stringa e lavorare con i singoli caratteri in una stringa, un processo chiamato **indicizzazione**. Per ottenere la lunghezza di una stringa, puoi usare la funzione integrata `len()`. Ecco un esempio:

```python
my_str = 'Hello world'
print(len(my_str))  # 11
```

Ogni carattere in una stringa ha una posizione chiamata indice. L’indice è a base zero, il che significa che l’indice del primo carattere di una stringa è `0`, l’indice del secondo carattere è `1` e così via. Per consultare un carattere tramite il suo indice, usi le parentesi quadre (`[]`) con l’indice del carattere che vuoi consultare all’interno. Ecco alcuni esempi:

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
```

È consentita anche l’indicizzazione negativa, quindi puoi ottenere l’ultimo carattere di qualsiasi stringa con `-1`, il penultimo carattere con `-2` e così via:

```python
my_str = 'Hello world'
print(my_str[-1])  # d
print(my_str[-2]) # l
```

Molti altri linguaggi di programmazione raggruppano i tipi di dati in modo ampio come primitivi o di riferimento. I tipi primitivi sono semplici e immutabili, il che significa che non possono essere modificati una volta dichiarati. I tipi di riferimento possono contenere più valori e sono mutabili o immutabili. Ma Python non traccia una linea netta tra questi due gruppi. Invece, tutti i dati vengono trattati come oggetti, e alcuni oggetti sono immutabili mentre altri sono mutabili.

I tipi di dati immutabili non possono essere modificati o alterati una volta dichiarati. Puoi assegnare le loro variabili a qualcosa di nuovo, cosa chiamata riassegnazione, ma non puoi cambiare l’oggetto originale aggiungendo, rimuovendo o sostituendo uno qualsiasi dei suoi elementi.

Le stringhe sono tipi di dati immutabili in Python. Questo significa che puoi riassegnare una stringa diversa a una variabile:

```python
greeting = 'hi'
greeting = 'hello'
print(greeting) # hello
```

Ma la modifica diretta di una stringa non è consentita:

```python
greeting = 'hi'
greeting[0] = 'H' # TypeError: 'str' object does not support item assignment
```

Esempi di altri tipi di dati immutabili in Python sono integer, float, boolean, tuple e range. Imparerai a conoscere ciascuno di questi tipi nelle prossime lezioni.
