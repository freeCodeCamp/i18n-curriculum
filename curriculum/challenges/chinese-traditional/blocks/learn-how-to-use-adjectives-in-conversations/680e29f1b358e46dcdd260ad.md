# --explanation--

`Send it over to` 是指將檔案、文件或訊息傳送給另一個人，通常是透過電子郵件或線上。 例如：

`I'll send it over to the design team after making the changes.`－這表示將文件傳送給他們。

`By tomorrow` 意味著在隔天結束前或當天完成該動作。例如：

`We need to finalize the software update by tomorrow。` – 這顯示了一個明確的截止日期。
