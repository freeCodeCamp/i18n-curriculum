# --description--


一般的に、作成するリンクには2種類あります：

- インターネット上の他のウェブサイトのページへのリンク

- 自分のウェブサイト内のページへのリンク


## 絶対リンク
インターネット上の他のウェブサイトのページへのリンクは絶対リンクと呼ばれます。典型的な絶対リンクは次の部分で構成されます：`protocol://domain/path`。絶対リンクには常に宛先のプロトコルとドメインが含まれます。

すでに絶対リンクを実際に見たことがあります。以前に作成したThe Odin ProjectのAboutページへのリンクは、プロトコルとドメインが含まれているため絶対リンクでした。

`https://www.theodinproject.com/about`

## 相対リンク
自分のウェブサイト内の他のページへのリンクは相対リンクと呼ばれます。相対リンクにはドメイン名が含まれません。同じサイト内の別のページなので、リンクを作成したページと同じドメイン名であると想定されます。

相対リンクには、リンクを作成しているページから見た他のページへのファイルパスだけが含まれます。これは少し抽象的なので、例を使って実際に見てみましょう。

`odin-links-and-images` ディレクトリ内に、`about.html` という名前の別のHTMLファイルを作成し、次のコードを貼り付けてください：

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Odin Links and Images</title>
  </head>

  <body>
    <h1>About Page</h1>
  </body>
</html>
```

`index` ページに戻り、次のアンカー要素を追加して`about` ページへのリンクを作成してください：

```html
<body>
  <h1>Homepage</h1>
	<a href="https://www.theodinproject.com/about">click me</a>

	<a href="about.html">About</a>
</body>
```

`index.html` ファイルをブラウザで開き、aboutリンクをクリックしてすべてが正しくつながっているか確認してください。リンクをクリックすると、先ほど作成した`about` ページに移動するはずです。

これは`index` と`about` ページが同じディレクトリにあるためです。つまり、リンクの`about.html` 値として単にその名前（`href`）を使うことができます。

しかし、通常はウェブサイトのディレクトリをもう少し整理したいでしょう。通常はルートディレクトリに`index.html` だけを置き、他のHTMLファイルはそれぞれのディレクトリに入れます。

`pages` ディレクトリ内に`odin-links-and-images` という名前のディレクトリを作成し、`about.html` ファイルをこの新しいディレクトリに移動してください。

ブラウザで`index` ページを更新し、`about` リンクをクリックしてください。リンクは壊れているはずです。これは`about` ページファイルの場所が変わったためです。

これを修正するには、`about` リンクの`href` 値を更新して、`pages/` ディレクトリを含めるだけでいいです。これは`about.html` ファイルから見た`index.html` ファイルの新しい場所だからです。

```html
<body>
  <h1>Homepage</h1>
  <a href="pages/about.html">About</a>
</body>
```

ブラウザで`index` ページを更新し、もう一度`about` リンクをクリックしてみてください。今度は正常に動作するはずです。

多くの場合、これで問題ありませんが、この方法では予期しない問題が起こることもあります。リンクの前に`./` を付けることで、ほとんどの場合そうした問題を防げます。`./` を追加すると、コードに対して**現在の**ディレクトリからファイルやディレクトリを探し始めるよう指定することになります。

```html
<body>
  <h1>Homepage</h1>
  <a href="./pages/about.html">About</a>
</body>
```

絶対リンクと相対リンクの違いは何ですか？

## --answers--

絶対リンクは現在のウェブサイトの別のページへのリンクです。相対リンクは別のウェブサイトへのリンクです。
