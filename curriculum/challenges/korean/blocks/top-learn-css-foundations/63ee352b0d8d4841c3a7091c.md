# --description--

두 그룹의 요소가 일부 스타일 선언을 공유한다면 어떻게 할까요?

```css
.read {
  color: white;
  background-color: black;
  /* several unique declarations */
}

.unread {
  color: white;
  background-color: black;
  /* several unique declarations */
}
```

`.read`와 `.unread` 선택자 모두 `color: white;`와 `background-color: black;` 선언을 공유하지만, 각각 고유한 선언도 여러 개 있습니다. 중복을 줄이기 위해 이 두 선택자를 쉼표로 구분된 목록으로 묶을 수 있습니다.

```css
.read,
.unread {
  color: white;
  background-color: black;
}

.read {
  /* several unique declarations */
}

.unread {
  /* several unique declarations */
}
```

위 두 예제(그룹화한 경우와 하지 않은 경우)는 결과가 같지만, 두 번째 예제는 선언의 중복을 줄이고 두 클래스 모두에 대해 `color` 또는 `background-color`를 한 번에 더 쉽게 수정할 수 있습니다.

# --questions--    

## --text--

두 개의 다른 선택자 `.red-box`와 `.yellow-box`에 단일 규칙을 적용하려면 어떻게 해야 할까요?

## --answers--

```css
.red-box,
.yellow-box {
  width: 25px;
  height: 25px;
}
```

---

```css
.red-box {
  width: 25px;
  height: 25px;
}

.yellow-box {
  width: 25px;
  height: 25px;
}
```

---

```css
.red-box { 
  width: 25px;
  .yellow-box {
    height: 25px;
  }
}
```
