# --interactive--

プログラミングにおけるループは、コードのブロックを複数回繰り返すために使われます。

ループの例としては、リストのアイテムを順番に出力するプログラムを設計するときが挙げられます。リストの各アイテムを出力するためにループを使うことができます。

別の例としては、ゲームを設計していてキャラクターを画面上で動かしたい場合です。ループを使って、ループが実行されるたびにキャラクターを一定のピクセル数だけ移動させることができます。

JavaScriptにはいくつかの種類のループがあります。このレッスンでは`for`ループについて説明します。`for`ループの基本的な構文は以下の通りです。

```js
for (initialization; condition; increment or decrement) {
  // code block to be executed
}
```

初期化文はループが始まる前に実行されます。通常はカウンター変数を初期化するために使われます。カウンター変数とは、ループが何回実行されたかを追跡するための変数です。

条件文はループの各反復の前に評価されます。反復とはループを一回通過することです。

条件が真であれば、ループ内のコードブロックが実行されます。条件が偽であれば、ループはストップし、次のコードブロックに進みます。

ループの最後の部分はインクリメント／デクリメント文です。この文はループの各反復の後に実行されます。通常はカウンター変数を増減させるために使われます。

:::interactive_editor

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

:::

上記の例の最初の部分では、カウンター変数`i`を`0`に初期化しています。`i`ループでは、カウンター変数に`for`を使うのが慣例です。

次に条件をチェックします。この場合、条件は`i`が`5`より小さいかどうかを確認しています。`i`が`0`なので、条件は真となり、ループ内のコードブロックが実行されます。

ループ内のコードブロックは`i`の値をコンソールに出力することです。`i`の値は`0`なので、コンソールには`0`の値が表示されます。

次にインクリメント文が実行されます。この場合、`i`を`1`だけ増やしています。したがって`i`は今や`1`です。

そして再び条件をチェックします。`i`が`5`より小さいかどうかを確認します。`i`は今や`1`なので、条件はまだ真であり、ループ内のコードブロックが再度実行されます。

このプロセスを条件が偽になるまで繰り返します。この場合、`i`が`5`になると条件は偽となり、ループはストップします。

ループを使うときは、常に真となる条件を作らないように注意してください。もしそうすると、ループは永遠に実行され続け、プログラムがクラッシュします。これを無限ループと呼びます。

ネストされた`for`ループを作ることも可能です。ネストされたループとは、ループの中に別のループを入れることです。後ほどそのような例を見ていきます。

ループは、コードのブロックを一定回数繰り返す必要がある場合にプログラミングで役立ちます。`for`ループの扱いは最初は難しいかもしれませんが、練習すれば慣れてきます。
