## --text--

Warum ist es wichtig, eine Testsuite auszuführen?

## --answers--

Es verlangsamt den Programmierprozess.

### --feedback--

Das Ausführen von Tests kann etwas Zeit in Anspruch nehmen, verhindert aber größere Probleme später.

---

Es hilft, Fehler zu finden, bevor sie größere Probleme verursachen.

---

Es macht den Code kürzer.

### --feedback--

Eine Testsuite ändert nicht die Länge des Codes.

---

Es entfernt alle Bugs aus dem Code.

### --feedback--

Tests helfen, Bugs zu finden, garantieren aber nicht, dass ein Programm perfekt ist.

# --explanation--

Um die richtige Antwort zu verstehen, hören Sie, was Sophie sagt: `It would have caught the error early on and saved you some time.`

Sie verwendet den Ausdruck `would have caught`, der zeigt, dass Mark den Fehler früher hätte finden können, dies aber **nicht** passiert ist. Hier ist ein weiteres Beispiel:

`If you had run the test suite, it would have found the missing bracket.` – Das bedeutet, dass das Ausführen einer Testsuite Ihnen hilft, Probleme frühzeitig zu erkennen und später Zeitverschwendung zu vermeiden.
