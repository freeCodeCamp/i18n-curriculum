# --description--

令 $S_n$ 為使用以下偽隨機數產生器產生的整數序列：

$$\begin{align}
        S_0 & = 290\\,797 \\\\
  S_{n + 1} & = {S_n}^2\bmod 50\\,515\\,093
\end{align}$$

令 $A(i, j)$ 為數字 $S_i, S_{i + 1}, \ldots, S_j$ 中的最小值，條件為 $i ≤ j$。令 $M(N) = \sum A(i, j)$，條件為 $1 ≤ i ≤ j ≤ N$。

我們可以驗證 $M(10) = 432\,256\,955$ 和 $M(10\,000) = 3\,264\,567\,774\,119$。

尋找 $M(2\,000\,000\,000)$。
