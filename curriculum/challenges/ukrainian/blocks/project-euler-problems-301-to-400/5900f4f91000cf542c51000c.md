# --description--

На параболі $y = \frac{x^2}{k}$ вибрано три точки $A(a, \frac{a^2}{k})$, $B(b, \frac{b^2}{k})$ та $C(c, \frac{c^2}{k})$.

Нехай $F(K, X)$ — це кількість цілих четвірок $(k, a, b, c)$ таких, що принаймні один кут трикутника $ABC$ дорівнює 45°, де $1 ≤ k ≤ K$ та $-X ≤ a &lt; b &lt; c ≤ X$.

Наприклад, $F(1, 10) = 41$ та $F(10, 100) = 12\\,492$.

Знайдіть $F({10}^6, {10}^9)$.
