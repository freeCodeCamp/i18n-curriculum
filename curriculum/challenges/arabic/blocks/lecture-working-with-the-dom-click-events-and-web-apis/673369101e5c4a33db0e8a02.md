# --interactive--

في الدرس السابق، تعلمت كيف تتعامل مع أحداث `"click"` و`"input"` باستخدام طريقة `addEventListener()`. في هذا الدرس، ستتعلم كيف تزيل مستمعي الأحداث باستخدام طريقة `removeEventListener()`.

تُستخدم طريقة `removeEventListener()` لإزالة مستمع حدث تمت إضافته سابقًا إلى عنصر باستخدام طريقة `addEventListener()`. هذا مفيد عندما تريد التوقف عن الاستماع لحدث معين على عنصر.

إليك الصيغة الأساسية لطريقة `removeEventListener()`:

```js
element.removeEventListener("event", listener);
```

تمامًا مثل طريقة `addEventListener()`، تأخذ طريقة `removeEventListener()` معلمتين: الحدث الذي تريد إزالته ودالة المستمع التي أُضيفت سابقًا.

هناك أيضًا معلمة ثالثة اختيارية يمكن تمريرها إلى طريقة `removeEventListener()`. يمكن أن تكون هذه المعلمة إما `options` أو `useCapture`.

معلمة `options` هي كائن يحدد الخيارات لمستمع الحدث، مثل ما إذا كان مستمع الحدث يجب أن يكون سلبيًا أو يُنفذ مرة واحدة.

معلمة `useCapture` هي قيمة منطقية تحدد ما إذا كان يجب التقاط الحدث خلال مرحلة انتشار الحدث.

في معظم الأحيان، ستحتاج فقط إلى تمرير الحدث ودالة المستمع إلى طريقة `removeEventListener()`.

إذا أردنا تبديل لون الخلفية بين الرمادي والأزرق، يمكننا استخدام مستمع حدث لذلك.

أولًا، نحتاج إلى الوصول إلى عنصر `button` مع عنصر `body`:

```js
const bodyEl = document.querySelector("body");
const btn = document.getElementById("btn");
```

ثم نحتاج إلى إنشاء الدالة المسؤولة عن التبديل بين اللونين الرمادي والأزرق:

```js
let isBgColorGrey = true;

function toggleBgColor() {
  bodyEl.style.backgroundColor = isBgColorGrey ? "blue" : "grey";
  isBgColorGrey = !isBgColorGrey;
}
```

نستخدم متغيرًا منطقيًا `isBgColorGrey` لتتبع لون الخلفية الحالي. إذا كان لون الخلفية رماديًا، نغيره إلى أزرق، والعكس صحيح.

بعدها نضيف مستمع حدث إلى عنصر `button` لاستدعاء دالة `toggleBgColor` عند نقر الزر:

```js
btn.addEventListener("click", toggleBgColor);
```

في كل مرة يُنقر فيها الزر، يتغير لون خلفية الجسم بين الرمادي والأزرق.

يمكننا تحديث HTML ليشمل عنصر فقرة يُستخدم لإزالة مستمع الحدث عند التمرير فوقه:

```html
<link rel="stylesheet" href="styles.css" />
<h1>Event Listener examples</h1>
<p id="para">MouseOver this text to remove the event listener</p>
<button id="btn">Change background color</button>
<script src="index.js"></script>
```

لإزالة مستمع الحدث عند التمرير فوق الفقرة، نضيف مستمع حدث إلى عنصر الفقرة. نستخدم حدث `mouseover` ونمرر دالة تزيل مستمع الحدث من عنصر `button`:

```js
const para = document.getElementById("para");
const btn = document.getElementById("btn");

para.addEventListener("mouseover", () => {
  btn.removeEventListener("click", toggleBgColor);
});
```

عند التمرير فوق الفقرة، يُزال مستمع حدث نقر الزر، ولن يتغير لون الخلفية عند نقر الزر بعد ذلك.
