# --explanation--

`Database` はデータが保存・整理されるシステムを指します。

`Activity` はデータベース内で起こっている動作や処理を指します。

- `The database contains customer information and order details.` - これはデータベースが重要なビジネスデータを保存していることを意味します。

- `There was unusual activity in the database last night.` - これはデータベースで予期しないことが起こったことを意味します。

合わせて、`database activity` はデータベースシステム内で起こっているすべての動作、変更、処理を意味します。
