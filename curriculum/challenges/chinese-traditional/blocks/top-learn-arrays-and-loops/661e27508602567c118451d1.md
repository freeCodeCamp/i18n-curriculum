# --description--

字串和數字可能是我們的建造區塊，但隨著你的腳本變得更複雜，你將需要一種方法來處理大量的它們。幸運的是，JavaScript 有幾種資料型別正是用來做這件事的。陣列是一個有序的項目群集（字串、數字或其他東西）。

陣列是一種在單一變數中儲存多個值的方法。它們是一種特殊型別的物件，具有一個 length 屬性和一系列編號的屬性。每個編號的屬性稱為元素，每個元素可以儲存任何型別的值。

陣列的一個範例是：

```javascript
const fruits = ['apple', 'banana', 'orange'];
```
