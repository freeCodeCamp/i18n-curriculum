# --description--

프로그래밍에서 `string`는 텍스트를 저장하는 데 사용되는 데이터 유형입니다. `string`는 문자, 숫자, 기호를 포함할 수 있습니다. 보통 텍스트를 따옴표로 감싸서 나타냅니다.

예를 들어, `"Hello, world!"`는 `string`입니다.

### --feedback--
