# --explanation--

`Last time` стосується найостаннішої події. Наприклад:

`Last time we updated the server, we had some downtime.` – Це вказує на попереднє оновлення.

`Order` означає офіційно зробити запит на товари або обладнання. Його минула форма — `ordered`. Наприклад:

`The IT team ordered new laptops for the developers.` – Це означає, що вони зробили запит на ноутбуки.

`Double that` означає подвійний попередній обсяг. Наприклад:

`Last time we ordered 10 routers; this time we need double that, so 20 routers.` – Це показує, що кількість збільшилася вдвічі.
