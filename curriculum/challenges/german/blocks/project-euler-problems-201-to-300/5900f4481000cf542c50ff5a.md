# --description--

Seien $A$ und $B$ Bitstrings (Folgen von 0 und 1).

Wenn $A$ gleich den <u>linkesten</u> length($A$) Bits von $B$ ist, dann sagt man, dass $A$ ein Präfix von $B$ ist.

Zum Beispiel ist 00110 ein Präfix von <u>00110</u>1001, aber nicht von 00111 oder 100110.

Ein präfixfreier Code der Größe $n$ ist eine Sammlung von $n$ verschiedenen Bitstrings, so dass kein String ein Präfix eines anderen ist. Zum Beispiel ist dies ein präfixfreier Code der Größe 6:

$$0000, 0001, 001, 01, 10, 11$$

Angenommen, es kostet einen Penny, ein '0'-Bit zu übertragen, aber vier Pence, ein '1'-Bit zu übertragen. Dann betragen die Gesamtkosten des oben gezeigten präfixfreien Codes 35 Pence, was zufällig die günstigste Möglichkeit für das hier betrachtete verzerrte Preisschema ist. Kurz gesagt, wir schreiben $Cost(6) = 35$.

Wie groß ist $Cost(10^9)$?
