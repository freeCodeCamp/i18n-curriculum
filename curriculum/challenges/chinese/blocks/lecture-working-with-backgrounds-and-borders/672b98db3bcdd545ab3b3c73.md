# --interactive--

在 CSS 中，有多种方法可以为图像添加边框，每种方法提供不同的样式选项和控件级别。

让我们探索一些最常见且多功能的 `methods`。

为图像添加边框最直接的方法是使用 `border` 属性。该属性是一个简写，允许你一次性设置边框的宽度、样式和颜色。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

此 CSS 规则为所有 `img` 元素添加一个宽度为 2 像素、实线的红色边框。你可以调整宽度、样式（例如 `dashed`、`dotted` 或 `double`）和颜色以适应你的设计需求。

如果你需要对边框的各个单独边进行更多控制，可以使用每个边的特定边框属性：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border-top: 10px solid red;
  border-right: 10px dashed green;
  border-bottom: 10px dotted blue;
  border-left: 10px double purple;
}
```

:::

这允许你为图像的每一侧创建独特的边框样式。

另一种创建边框效果的方法是使用 `outline` 属性。虽然与边框类似，但 outline 不会影响元素的维度或版型：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  outline: 3px solid gold;
}
```

:::

这会在图像周围创建一个金色轮廓，而不会改变其大小或位置。

如果你想为你的边框创建圆角，可以将 `border-radius` 属性与边框一起使用：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

请记住，这些技术可以组合并自定义，以创建独特的边框效果。方法的选择取决于你的具体设计需求和所需的复杂度水平。
