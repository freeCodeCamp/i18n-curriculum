# --instructions--

Tengeneza skana ya porti kwa kutumia Python.

Katika faili la `port_scanner.py`, tengeneza kitendakazi kinachoitwa `get_open_ports` kinachopokea hoja ya `target` na hoja ya `port_range`. `target` inaweza kuwa URL au anwani ya IP. `port_range` ni orodha ya nambari mbili zinazoonyesha nambari ya kwanza na ya mwisho ya safu ya porti za kuchunguza.

Hapa kuna mifano ya jinsi kitendakazi kinaweza kuitwa:

```py
get_open_ports("209.216.230.240", [440, 445])
get_open_ports("www.stackoverflow.com", [79, 82])
```

Kitendakazi kinapaswa kurudisha orodha ya porti zilizo wazi katika safu iliyotolewa.

Kitendakazi cha `get_open_ports` kinapaswa pia kupokea hoja ya tatu ya hiari ya `True` kuonyesha hali ya "Verbose". Ikiwa hii itakuwa kweli, kitendakazi kinapaswa kurudisha mfuatano wa herufi wa maelezo badala ya orodha ya porti.

Huu ni muundo wa mfuatano wa herufi unaotakiwa kurudishwa katika hali ya verbose (maandishi ndani ya `{}` yanaonyesha taarifa zinazotakiwa kuonekana):

```bash
Open ports for {URL} ({IP address})
PORT     SERVICE
{port}   {service name}
{port}   {service name}
```

Unaweza kutumia kamusi iliyoko katika `common_ports.py` kupata jina sahihi la huduma kwa kila porti.

Kwa mfano, ikiwa kitendakazi kitaitwa hivi:

```py
port_scanner.get_open_ports("scanme.nmap.org", [20, 80], True)
```

Kinapaswa kurudisha yafuatayo:

```bash
Open ports for scanme.nmap.org (45.33.32.156)
PORT     SERVICE
22       ssh
80       http
```

Hakikisha unajumuisha nafasi sahihi na herufi za mstari mpya.

Ikiwa URL iliyotumwa kwenye kitendakazi cha `get_open_ports` si halali, kitendakazi kinapaswa kurudisha mfuatano wa herufi: "Error: Invalid hostname".

Ikiwa anwani ya IP iliyotumwa kwenye kitendakazi cha `get_open_ports` si halali, kitendakazi kinapaswa kurudisha mfuatano wa herufi: "Error:  Invalid IP address".
