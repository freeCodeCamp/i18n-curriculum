# --description--

$E(x_0, y_0)$를 유클리드 알고리즘으로 $x_0$와 $y_0$의 최대공약수를 구하는 데 걸리는 단계 수라고 하겠습니다. 더 공식적으로는:

$$\begin{align}
  & x_1 = y_0, y_1 = x_0\bmod y_0 \\\\
  & x_n = y_{n - 1}, y_n = x_{n - 1}\bmod y_{n - 1}
\end{align}$$

$E(x_0, y_0)$는 $y_n = 0$이 되는 가장 작은 $n$입니다.

$E(1, 1) = 1$, $E(10, 6) = 3$, $E(6, 10) = 4$임을 알 수 있습니다.

$S(N)$을 $1 ≤ x$, $y ≤ N$인 모든 쌍에 대해 $E(x, y)$를 더한 값으로 정의하세요.

$S(1) = 1$, $S(10) = 221$, $S(100) = 39\,826$입니다.

$S(5 \times {10}^6)$를 구하세요.
