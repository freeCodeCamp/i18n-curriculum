# --description--

另一种比较条件式的方法是三元操作符。三元操作符是编写 `if-else` 语句的简写方式。它用于评估一个条件式，并根据条件式的结果返回一个值。三元操作符的语法如下：

```javascript
let result = condition ? value1 : value2;
```

在上述示例中，如果条件为真，`result` 的值将是 `value1`。否则，`result` 的值将是 `value2`。

三元操作符通常用于根据条件为变量赋值。它们也用于根据条件返回值。
