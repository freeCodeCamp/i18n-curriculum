# --description--

学习使用 `how much` 和 `how many` 来询问数量。`How much` 用于不可数名词，询问某物的体积或范围，例如 `time`、`water` 或 `money`。例如，`How much money do you have?` 是在询问你拥有的钱的数量，这些钱无法逐个计数。

`How many` 用于可数名词，当你可以数出项时，比如 `books`、`apples` 或 `people`。例如，`How many books do you own?` 询问的是书的可数数量。
