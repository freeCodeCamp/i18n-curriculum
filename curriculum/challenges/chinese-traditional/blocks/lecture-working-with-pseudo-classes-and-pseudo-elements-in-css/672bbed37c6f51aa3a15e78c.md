# --interactive--

CSS 最有趣的方面之一是使用偽元素。在這個背景關係中，「偽」意指「不真實」，因此偽元素是虛擬或合成的元素，不會直接對應任何實際的 HTML 元素。它們允許你為元素的特定零件設計樣式，或在不新增額外 HTML 的情況下插入內容。

要套用偽元素，請使用雙冒號（`::`）將其附加到原始元素的選擇器。請注意，選擇器可以是任何類型，例如類別或 ID 選擇器。以下是偽元素的基本語法：

```css
selector::pseudo-element {
  property: value;
}
```

這個雙冒號是用來區分偽元素與使用單冒號的偽類。

偽元素允許你為元素內容的特定零件或部分設計樣式，或在其前後插入內容，但它們無法獨立存在。附加偽元素的元素稱為其起始元素。

讓我們先來看看 `::before` 和 `::after` 偽元素的範例。正如它們的名稱所示，`::before` 讓你在元素內容之前插入內容，而 `::after` 則讓你在元素內容之後插入內容。

這裡是一個按鈕元素的範例。在 CSS 中，我們將使用絕對定位和 `::before` 偽元素，為按鈕的 `Learn More` 文字前添加一顆星星。你將在未來的課程中學習更多關於絕對定位的內容。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: lightseagreen;
  color: white;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
  position: relative;
}

.cta-button::before {
  content: "⭐";
  position: absolute;
  left: 3px;
  top: 8px;
  font-size: 0.75rem;
}
```

:::

`content` 屬性用於表述你希望在按鈕文字前添加的內容。在此範例中，我們正在添加一顆星星。你會注意到你不僅可以插入內容，還可以為其設計樣式。以下是使用相同按鈕的 `::after` 偽元素範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: '➡️';
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}
```

:::

在 `::after` 偽元素中，`transition` 屬性用於在 0.3 秒內以緩動效果動畫化變化，創造平滑且漸進的轉換，而非突然的變化。你將在未來的課程中學習更多關於 `transition` 屬性的內容。 

你也可以將偽類別附加到你使用 `::before` 和 `::after` 偽元素插入到其他內容中的內容。<dfn>例如</dfn>，內容的暫留狀態。以下是一個範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: "➡️";
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}

.cta-button:hover::after {
  transform: translateX(2px);
}
```

:::

在暫留狀態下使用 `transform: translateX(2px)`，每當使用者暫留在按鈕上時，內容會被推向右方 `2px`。`::after` 本身的 transition 屬性確保該處理程序持續 `0.3s`。

這就是 `transform` 屬性所做的－它允許你旋轉、傾斜、縮放或將元素沿特定方向平移。你將在未來的課程中學到更多相關內容。

在下一個範例中，我們將查看 `::first-letter` 偽元素。`::first-letter` 偽元素會標的元素內容的第一個字母，讓你可以為它設定樣式。這裡有一個段落文字的範例。如果我們想要為第一個字母設定樣式，我們可以像這樣使用 `::first-letter` 偽元素：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>freeCodeCamp lets you learn to code without having to pay.</p>
```

```css
p::first-letter {
  font-size: 4rem;
}
```

:::

在最後一個範例中，我們將查看 `::marker` 偽元素，它讓你選取列表項目的標記、項目符號或編號以進行樣式設定。`::marker` 偽元素提供一種方法，讓你透過訂製列表標記以符合你的色彩結構綱目，來強化你網站的品牌識別。

這裡是一個無序列表和有序列表的範例。若要更改列表項目的標記顏色和大小，你可以像這樣使用 `::marker` 偽元素：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<ul>
  <li>Unordered list item 1</li>
  <li>Unordered list item 2</li>
  <li>Unordered list item 3</li>
  <li>Unordered list item 4</li>
</ul>

<ol>
  <li>Ordered list item 1</li>
  <li>Ordered list item 2</li>
  <li>Ordered list item 3</li>
  <li>Ordered list item 4</li>
</ol>
```

```css
li::marker {
  color: crimson;
  font-size: 1.5em;
  font-weight: bold;
}
```

:::

在本課程中，我們只涵蓋了少數偽元素。但還有許多其他的偽元素，例如 `::placeholder`、`::spelling-error` 和 `::selection`，我鼓勵你自行探索。
