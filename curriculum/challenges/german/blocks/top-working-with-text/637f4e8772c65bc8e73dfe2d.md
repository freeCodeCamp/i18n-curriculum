# --description--

HTML-Kommentare sind für den Browser nicht sichtbar; sie ermöglichen es uns, Ihren Code zu kommentieren, damit andere Entwickler oder Sie selbst in der Zukunft sie lesen und Kontext zu etwas erhalten können, das im Code vielleicht nicht klar ist.

Einen HTML-Kommentar zu schreiben ist einfach: Sie schließen den Kommentar einfach mit `<!--` und `-->`-Tags ein. Zum Beispiel:

```html
<h1> View the html to see the hidden comments </h1>

<!-- I am a html comment -->

<p>Some paragraph text</p>

<!-- I am another html comment -->
```

# --assignment--

Um etwas Übung im Umgang mit Text in HTML zu bekommen, erstellen Sie eine einfache Blog-Artikel-Seite, die verschiedene Überschriften verwendet, Absätze nutzt und in den Absätzen einige Texte fett und kursiv formatiert hat. Sie können [Lorem Ipsum](https://loremipsum.io) verwenden, um Dummy-Text zu generieren, anstelle von echtem Text, während Sie Ihre Seiten erstellen.
