# --description--

直接疑問文は、特定の情報を明確に尋ねるシンプルな質問です。通常、主語と動詞が倒置されており、`Are you sure?` のような形になります。この形式は日常会話でよく使われ、特に迅速かつ明確な回答が必要な場合に適しています。

一方、間接疑問文や間接表現は、より丁寧で直接的ではありません。主語と動詞の倒置はなく、`I was wondering` や `Could you tell me` のようなフレーズで始まることが多いです。例えば、`Where do you live?` と尋ねる代わりに、`Could you tell me where you live?` と言うことがあります。これにより質問が唐突に聞こえず、より丁寧な印象になり、特にフォーマルな場面やあまり親しくない相手と話すときに役立ちます。

例：

直接疑問文：`What time is it?`（動詞 `is` と主語 `it` の倒置）

間接表現：`Please tell me what time it is.`（倒置なし）
