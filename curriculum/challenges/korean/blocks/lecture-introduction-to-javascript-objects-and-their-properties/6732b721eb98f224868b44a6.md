# --interactive--

객체에서 속성을 삭제하는 방법은 여러 가지가 있으며, `delete` 연산자가 가장 간단하고 일반적으로 사용되는 방법입니다.

`delete`을 사용하면 선택한 속성이 객체에서 삭제됩니다. `delete` 연산자를 사용하는 방법은 다음 예제를 참고하세요:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  job: "Engineer"
};

delete person.job;

console.log(person.job); // undefined
```

:::

이 예제에서는 `person` 객체가 `name`, `age`, `job` 세 가지 속성을 가지고 시작합니다. 그런 다음 `delete` 연산자를 사용해 `job` 속성을 삭제합니다. 삭제 후 `person` 객체에는 더 이상 `job` 속성이 없습니다.

또 다른 속성 삭제 방법은 rest 매개변수를 사용한 구조 분해 할당입니다. 이 방법은 실제로 속성을 삭제하지는 않지만, 지정한 속성이 없는 새 객체를 만듭니다:

:::interactive_editor

```js
const person = {
  name: "Bob",
  age: 25,
  job: "Designer",
  city: "New York"
};

const { job, city, ...remainingProperties } = person;

// { name: "Bob", age: 25 }
console.log(remainingProperties);
```

:::

이 예제에서는 구조 분해를 사용해 `job`과 `city`를 `person` 객체에서 추출하고, 나머지 속성들을 `remainingProperties`라는 새 객체에 모읍니다. 이렇게 하면 `job`와 `city` 속성이 없는 새 객체가 만들어집니다.

객체에서 속성을 삭제하는 방법을 이해하는 것은 JavaScript 프로그래밍에서 중요한 기술입니다. 이를 통해 객체를 동적으로 조작하고 불필요한 데이터를 정리할 수 있습니다.
