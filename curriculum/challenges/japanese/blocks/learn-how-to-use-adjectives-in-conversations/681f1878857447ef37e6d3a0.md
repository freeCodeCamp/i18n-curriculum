# --explanation--

`Not only ... but also ...` は、2つの関連する文がどちらも真実であることを強調するためによく使われる構文で、2つ目の文がより驚きや重要性を持つ場合が多いです。例えば：

`She is not only skilled but also very creative.` – これは彼女に2つの強い特性があり、どちらも重要であることを意味します。

`Adaptable` は、状況に応じて簡単に変化や調整ができることを意味します。例えば：

`This program is adaptable to different types of teams.` – これは多様な環境でうまく機能することを意味します。
