# --interactive--

Ufumbuzi wa muundo wa vitu ni kipengele chenye nguvu katika JavaScript kinachokuwezesha kutoa thamani kutoka kwa vitu na kuziweka katika vigezo kwa njia fupi na inayosomeka vizuri zaidi.

Ni sehemu ya sifa ya ES6 (ECMAScript 2015) na imekuwa chombo muhimu kwa msanidi programu wengi wa JavaScript.

Ufumbuzi wa muundo unaweza kurahisisha msimbo wako, hasa unapotumia vitu tata au unapotakiwa kutoa thamani nyingi kwa wakati mmoja.

Kwa msingi wake, ufumbuzi wa muundo wa vitu ni kuhusu kufungua thamani kutoka kwa vitu na kuziweka katika vigezo tofauti. Badala ya kupata sifa za kitu moja kwa moja, unaweza kutoa sifa nyingi kwa tamko moja. Hii inaweza kufanya msimbo wako kuwa safi na wenye ufanisi zaidi.

Tuanze na mfano kuonyesha jinsi ufumbuzi wa muundo wa vitu unavyofanya kazi:

:::interactive_editor

```js
const person = { name: "Alice", age: 30, city: "New York" };

const { name, age } = person;

console.log(name); // Alice
console.log(age);  // 30
```

:::

Katika mfano huu, tunatoa sifa za `name` na `age` kutoka kwa kitu cha `person` na kuziweka katika vigezo vyenye majina sawa.

Moja ya mambo yenye nguvu ya ufumbuzi wa muundo wa vitu ni kwamba unaweza kuweka thamani zilizotolewa katika vigezo vyenye majina tofauti. Hii ni muhimu hasa unapotumia vitu vyenye majina ya sifa yanayoweza kuleta mgongano na vigezo vilivyopo au unapotaka kutumia jina tofauti:

:::interactive_editor

```js
let person = { name: "Alice", age: 30, city: "New York" };

let { name: personName, age: personAge } = person;

console.log(personName); // Alice
console.log(personAge); //  30
```

:::

Katika kesi hii, tunatoa sifa ya `name` na kuziweka katika kigezo kinachoitwa `personName`, na kufanya vivyo hivyo na `age` na `personAge`.

Ufumbuzi wa muundo wa vitu pia unakuwezesha kuweka thamani za chaguo-msingi. Ikiwa sifa haipo katika kitu unachofumbua, unaweza kuweka thamani mbadala:

:::interactive_editor

```js
let person = { name: "Alice", age: 30, city: "New York" };
let { name, age, country = "Unknown" } = person;

console.log(country); // Unknown
```

:::

Hapa, kwa kuwa `country` haipo katika kitu chetu cha `person`, inapata thamani ya chaguo-msingi `Unknown`.

Kesi nyingine ya kawaida ni ufumbuzi wa muundo wa vitu zilizopangwa ndani. Unaweza kufumbua sifa zilizo ndani ya vitu vingine kwa kutumia mabano mengine:

:::interactive_editor

```js
const recipe = {
  name: "Chocolate Cake",
  ingredients: {
    flour: "2 cups",
    sugar: "1 cup"
  }
};

// Extract `flour` from `ingredients`
const { ingredients: { flour } } = recipe;

console.log(flour); // "2 cups"
```

:::

Hii ni sawa na kupata sifa moja kwa moja:

```js
const flour = recipe.ingredients.flour;
console.log(flour); // "2 cups"
```

Sasa, tuzungumzie kuhusu sintaksia fupi katika ufumbuzi wa muundo wa vitu. Unapotengeneza vitu, hasa wakati majina ya sifa yanalingana na majina ya vigezo, unaweza kutumia sintaksia fupi:

:::interactive_editor

```js
let name = "Bob";
let age = 25;

let person = { name, age };

console.log(person); // { name: "Bob", age: 25 }
```

:::

Msimbo hapo juu unachukua sifa zilizo na majina sawa na vigezo vyetu na kuziweka thamani za vigezo hivyo.

Sintaksia hii fupi ni muhimu hasa unaporudisha vitu kutoka kwa kitendakazi au unapotengeneza vitu vyenye sifa nyingi:

:::interactive_editor

```js
function createPerson(name, age) {
  return { name, age };
}

let person = createPerson("Charlie", 35);
console.log(person); // { name: "Charlie", age: 35 }
```

:::

Ufumbuzi wa muundo wa vitu na sintaksia fupi ya vitu ni vipengele vyenye nguvu vinavyoweza kufanya msimbo wako kuwa mfupi na rahisi kusoma.

Ni muhimu hasa unapotumia miundo tata ya data, au unapotakiwa kupitisha vigezo vingi kwa vitendakazi.

Unapoendelea kutumia JavaScript, utagundua hali nyingi ambapo mbinu hizi zinaweza kurahisisha msimbo wako na kuufanya udhihirike zaidi.
