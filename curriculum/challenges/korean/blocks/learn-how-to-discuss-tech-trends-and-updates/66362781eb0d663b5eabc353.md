# --description--

`Serverless computing`는 클라우드 제공자가 서버를 운영하고 기계 자원 할당을 동적으로 관리하는 `cloud computing` 실행 모델입니다. 가격은 애플리케이션이 실제로 소비한 자원량을 기준으로 책정되며, 미리 구매한 용량 단위에 기반하지 않습니다. 개발자가 서버를 직접 다루지 않고 코드를 업로드하면 클라우드 제공자가 나머지를 처리하기 때문에 `serverless`라고 부릅니다.

예를 들어, `serverless computing`에서는 함수를 클라우드 서비스에 업로드하고 필요할 때만 실행할 수 있습니다.

---
