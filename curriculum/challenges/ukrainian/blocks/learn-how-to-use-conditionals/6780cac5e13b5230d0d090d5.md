`If we losing critical data, it could set us back weeks.`

### --feedback--

2

# --explanation--

Правильне речення використовує структуру `First Conditional`, яка утворюється за допомогою:

`if` + `Present Simple`: `If we lose critical data`.

Модальне дієслово (`could`) + базове дієслово: `it could set us back weeks`.

`First Conditional` використовується для опису можливої майбутньої події та її наслідку.
