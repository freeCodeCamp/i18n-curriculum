# --description--

파일 관리 애플리케이션은 컴퓨터에서 파일을 쉽게 저장, 정리, 편집할 수 있는 방법입니다. Mac이나 Windows 컴퓨터를 사용하든, 파일을 정리하는 데 사용할 수 있는 내장 애플리케이션이 있습니다.

Windows 컴퓨터를 사용한다면 기본 파일 관리자는 File Explorer입니다. 이 애플리케이션은 파일과 폴더를 탐색, 검색, 관리하며 복사, 이동, 삭제 같은 파일 작업을 수행하는 데 사용됩니다.

File Explorer를 찾으려면 시작 메뉴로 가거나 키보드에서 Windows 로고 키를 누르세요.

폴더를 고정하려면 마우스 오른쪽 버튼을 클릭하고 "Pin"을 선택하세요. 파일이나 폴더를 이동하려면 먼저 선택한 다음 "Cut"을 선택하고 새 위치로 이동해 붙여넣기 하세요.

Mac 컴퓨터를 사용한다면 기본 파일 관리자는 Finder입니다. Finder는 로컬 및 원격 파일에 접근할 수 있고, 파일 미리보기를 지원하며 태그와 스마트 폴더 같은 다양한 정리 기능을 제공합니다.

Mac에서 Finder에 접근하려면 화면 하단의 Dock으로 가서 Finder 아이콘을 클릭하세요.

왼쪽에 즐겨찾기 폴더가 있는 사이드바가 보일 것입니다.

파일을 정리하는 한 가지 방법은 태그를 사용하는 것입니다. 파일이나 폴더에 태그를 달려면 Finder에서 항목을 선택한 후 마우스 오른쪽 버튼을 클릭하세요. 드롭다운 메뉴에서 "Tags..." 옵션이 보이고 원하는 색상의 태그를 적용할 수 있습니다.

태그가 달린 항목을 검색하려면 사이드바 맨 아래로 스크롤해 해당 색상의 태그를 클릭하거나 검색 필드에 색상 이름으로 태그를 검색하세요.

파일을 정리하는 또 다른 방법은 스마트 폴더를 사용하는 것입니다. 스마트 폴더는 파일 유형, 날짜, 키워드 같은 설정한 기준에 따라 자동으로 정리되는 파일 모음입니다.

Windows의 File Explorer든 Mac의 Finder든, 이 애플리케이션들은 태그, 검색, 스마트 폴더 같은 기능으로 파일을 유연하게 관리할 수 있게 해줍니다.
