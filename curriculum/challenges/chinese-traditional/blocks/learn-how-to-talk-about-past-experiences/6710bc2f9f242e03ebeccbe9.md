# --explanation--

當你同意某人的想法時，你可以簡短地對對方說 `Sure`。例如：

- `Would you like to visit another country some day? Sure.`－這表示你想要這麼做。

- `How about going to the movies this week? Sure.`－這表示你接受邀請。
