## Python 类和对象

- **类定义**：类是创建对象的蓝图。它通过属性和方法定义对象将具有的行为。以下是 Python 中类定义的基本示例：

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f'{self.name.upper()} says woof woof!')
```

- **创建对象**：对象是类的实例。通过调用类并传入必要的参数来创建对象。

```python
dog1 = Dog('Jack', 3)
dog2 = Dog('Thatcher', 5)

dog1.bark()  # JACK says woof woof!
dog2.bark()  # THATCHER says woof woof!
```

- **调用对象的方法**：你可以调用对象的方法来执行操作或获取信息。

```python
object_name1.method_name()
object_name2.method_name()
```

- **类和对象的区别**：类是可重用的模板，而对象是该类的具体实例，包含实际数据。

## 属性

- **实例属性**：在 `__init__()` 中使用 `self` 定义，每个对象独有。
- **类属性**：直接在类内部定义，所有实例共享。

```python
class Dog:
    species = 'French Bulldog'  # Class attribute

    def __init__(self, name):
        self.name = name  # Instance attribute

print(Dog.species) # French Bulldog

jack = Dog('Jack')
print(jack.name)     # Jack
print(jack.species)  # French Bulldog
```

## 方法

- **方法**：定义在类内部的函数，用于操作对象的属性。

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
print(my_car_1.describe())  # This car is a red Tesla Model S
```

- **访问方法**：使用点符号调用对象的方法。以下是对两个不同汽车对象调用 `describe` 方法的示例：

```python
class Car:
    def __init__(self, color, model):
        self.color = color  
        self.model = model  

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
my_car_2 = Car('green', 'Lamborghini Revuelto')

print(my_car_1.describe()) # Calling method using the dot notation

print(my_car_2.describe()) # Calling method using the dot notation
```

## 双下划线（魔法）方法

- **定义**：以双下划线开头和结尾的特殊方法（例如 `__init__`、`__len__`、`__str__`、`__eq__`）。Python 内部使用它们来实现内置操作。

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **间接调用双下划线方法**：你不需要直接调用双下划线方法。Python 会在特定操作发生时自动调用它们。这些操作包括：

- **算术操作，如加法、减法、乘法、除法等**。加法调用 `__add__()`，减法调用 `__sub__()`，乘法调用 `__mul__()`，除法调用 `__truediv__()`。
        
- **字符串操作，如串接、重复、格式化和转换为文本**。串接调用 `__add__()`，重复调用 `__mul__()`，格式化调用 `__format__()`，文本转换调用 `__str__()` 和 `__repr__()`，等等。
        
- **比较操作，如相等、小于、大于等**。相等检查调用 `__eq__()`，小于调用 `__lt__()`，大于调用 `__gt__()`，等等。
        
- **迭代操作，比如使对象可迭代并遍历各项**。调用 `__iter__()` 返回一个迭代器，调用 `__next__()` 获取下一项。

## 现实示例：购物车

- **带双下划线方法的购物车类**：允许添加、移除、迭代和检查内容，具有内置行为。

```python
class Cart:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

    def remove(self, item):
        if item in self.items:
            self.items.remove(item)
        else:
            print(f'{item} is not in cart')

    def list_items(self):
        return self.items

    def __len__(self):
        return len(self.items)

    def __getitem__(self, index):
        return self.items[index]

    def __contains__(self, item):
        return item in self.items

    def __iter__(self):
        return iter(self.items)

cart = Cart()
cart.add('Laptop')
print(len(cart))        # 1
print('Laptop' in cart) # True
```
