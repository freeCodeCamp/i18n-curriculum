# --description--

어떤 양의 정수 $n$은 합 [ $n + reverse(n)$ ]이 모두 홀수(십진수) 숫자로만 이루어지는 성질을 가집니다. 예를 들어, $36 + 63 = 99$ 그리고 $409 + 904 = 1313$입니다. 이런 수를 가역수라고 부르겠습니다; 따라서 36, 63, 409, 그리고 904는 가역수입니다. $n$이나 $reverse(n)$ 모두에서 선행 0은 허용되지 않습니다.

1000 미만에는 120개의 가역수가 있습니다.

10억 미만(${10}^9$)에는 몇 개의 가역수가 있나요?
