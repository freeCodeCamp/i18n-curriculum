# --description--

$n$ 的根數 $rad(n)$ 是 $n$ 的不同質數因數的乘積。 例如，$504 = 2^3 × 3^2 × 7$，所以 $rad(504) = 2 × 3 × 7 = 42$。

我們將定義正整數三元組 (a, b, c) 為 abc-hit，條件是：

1. $GCD(a, b) = GCD(a, c) = GCD(b, c) = 1$
2. $a &lt; b$
3. $a + b = c$
4. $rad(abc) &lt; c$

例如，(5, 27, 32) 是一個 abc-hit，因為：

1. $GCD(5, 27) = GCD(5, 32) = GCD(27, 32) = 1$
2. $5 &lt; 27$
3. $5 + 27 = 32$
4. $rad(4320) = 30 &lt; 32$

事實證明 abc-hits 相當罕見，且在 $c &lt; 1000$ 時只有三十一個 abc-hits，且 $\sum{c} = 12523$。

找出所有 $c < 120000$ 的 $\sum{c}$。
