# --description--

Nambari kamili na nambari za desimali ni aina kuu za data za nambari katika Python. Kwa kutumia hizi, unaweza kuhifadhi data za nambari na kufanya operesheni za kihesabu.

Tuchunguze ni nini nambari kamili na nambari za desimali, jinsi ya kufanya mahesabu ya kihesabu kwa kutumia hizi, na njia kadhaa Python inazotoa za kufanya kazi na zote mbili.

Nambari kamili ni nambari kamili zisizo na alama za desimali, iwe chanya au hasi:

```python
my_int_1 = 56
my_int_2 = -4

print(type(my_int_1)) # <class 'int'>
print(type(my_int_2)) # <class 'int'>
```

Hivi ndivyo unavyofanya operesheni ya kuongeza kwa nambari kamili:

```python
my_int_1 = 56
my_int_2 = 12

sum_ints = my_int_1 + my_int_2
print('Integer Addition:', sum_ints) # Integer Addition: 68
```

Hivi ndivyo unavyofanya operesheni ya kutoa kwa nambari kamili:

```python
my_int_1 = 56
my_int_2 = 12

# Subtraction
diff_ints = my_int_1 - my_int_2
print('Integer Subtraction:', diff_ints) # Integer Subtraction: 44
```

Hivi ndivyo unavyofanya operesheni ya kuzidisha kwa nambari kamili:

```python
my_int_1 = 12
my_int_2 = 4

# Multiplication
product_ints = my_int_1 * my_int_2
print('Integer Multiplication:', product_ints) # Integer Multiplication: 48
```

Na hivi ndivyo unavyofanya operesheni ya kugawanya kwa nambari kamili:

```python
my_int_1 = 56
my_int_2 = 12

# Division
div_ints = my_int_1 / my_int_2
print('Division:', div_ints) # Division: 4.666666666666667
```

Nambari za desimali ni nambari chanya au hasi zenye alama za desimali, kama `3.14`, `-0.5`, au `0.0`.

```python
my_float_1 = -12.0
my_float_2 = 4.9

print(type(my_float_1)) # <class 'float'>
print(type(my_float_2)) # <class 'float'>
```

Hivi ndivyo unavyofanya operesheni ya kuongeza kwa nambari za desimali:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_addition = my_float_1 + my_float_2
print('Float Addition:', float_addition) # Float Addition: 17.4
```

Hivi ndivyo unavyofanya operesheni ya kutoa kwa nambari za desimali:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_subtraction = my_float_2 - my_float_1
print('Float Subtraction:', float_subtraction) # Float Subtraction: 6.6
```

Hivi ndivyo unavyofanya operesheni ya kuzidisha kwa nambari za desimali:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_multiplication = my_float_2 * my_float_1
print('Float Multiplication:', float_multiplication) # Float Multiplication: 64.80000000000001
```

Na hivi ndivyo unavyofanya operesheni ya kugawanya kwa nambari za desimali:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_division = my_float_2 / my_float_1
print('Float Division:', float_division) # Float Division: 2.222222222222222
```

Ikiwa utaongeza nambari kamili na nambari ya desimali, matokeo hubadilishwa moja kwa moja kuwa nambari ya desimali:

```python
my_int = 56
my_float = 5.4

sum_int_and_float = my_int + my_float

print(sum_int_and_float) # 61.4
print(type(sum_int_and_float)) # <class 'float'>
```

Hii ni kweli pia kwa operesheni nyingine za msingi za kihesabu, kama kutoa, kuzidisha, na kugawanya. Ikiwa unachanganya nambari kamili na nambari za desimali, Python itarudisha nambari ya desimali kama matokeo.

Pia unaweza kufanya mahesabu magumu zaidi ya kihesabu kama kupata mabaki ya nambari mbili kwa kutumia operator ya modulo, mgawanyo wa sakafu, na nguvu kwa nambari zote mbili, nambari kamili na nambari za desimali.

Operator ya modulo (`%`) hurudisha mabaki wakati thamani upande wa kushoto inagawanywa na thamani upande wa kulia:

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

mod_ints = my_int_1 % my_int_2
mod_floats = my_float_2 % my_float_1

print('Integer Modulo:', mod_ints) # Integer Modulo: 8
print('Float Modulo:', mod_floats) # Float Modulo: 1.1999999999999993
```

Mgawanyo wa sakafu hugawanya nambari mbili na hurudisha nambari kamili kubwa zaidi isiyozidi matokeo. Hii hufanywa kwa kutumia operator ya mstari wa mbele mara mbili (`//`):

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

floor_div_ints = my_int_1 // my_int_2
floor_div_floats = my_float_2 // my_float_1

print('Integer Floor Division:', floor_div_ints) # Integer Floor Division: 4
print('Float Floor Division:', floor_div_floats) # Float Floor Division: 2.0
```

Nguvu huinua nambari hadi nguvu ya nyingine, na hufanywa kwa kutumia operator ya nyota mbili (`**`):

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

exp_ints = my_int_1 ** my_int_2
exp_floats = my_float_1 ** my_float_2

print('Integer Exponentiation:', exp_ints) # Integer Exponentiation: 951166013805414055936
print('Float Exponentiation:',  exp_floats) # Float Exponentiation: 614787626.1765089
```

Wakati mwingine, unaweza kugundua kuwa matokeo ya operesheni inayohusisha nambari za desimali yana tarakimu za desimali zaidi kuliko ulivyotarajia. Kwa mfano, jumla `0.1 + 0.2` ni sawa na `0.30000000000000004` badala ya `0.3`.

Hii hutokea kwa sababu nambari huhifadhiwa katika muundo wa binary, na baadhi ya sehemu haziwezi wakilishwa kwa usahihi katika binary. Matokeo yake, huhifadhiwa kama makadirio ya mwisho, kwa njia ile ile sehemu `1/3` haiwezi wakilishwa kwa idadi ya tarakimu za desimali na hupunguzwa baada ya idadi fulani ya tarakimu zake zisizo na mwisho (`0.33333...`).

Hii husababisha makosa madogo ya kuzungusha.

Python pia hutoa kitendakazi kilichojengwa ndani cha kubadilisha data za nambari au mfuatano wa herufi kuwa nambari kamili au nambari za desimali.

Kitendakazi cha `float()` hurudisha nambari ya desimali iliyojengwa kutoka kwa nambari iliyotolewa:

```python
my_int_1 = 56
my_float_1 = float(my_int_1)

print(my_float_1)  # 56.0
print(type(my_float_1))  # <class 'float'>
```

Kitendakazi cha `int()` hurudisha nambari kamili iliyojengwa kutoka kwa nambari iliyotolewa:

```python
my_float = 12.92563
my_int = int(my_float)

print(my_int)  # 12
print(type(my_int))  # <class 'int'>
```

Pia, unaweza kutumia vitendakazi vilivyojengwa ndani kubadilisha mfuatano wa herufi kuwa nambari ya desimali au nambari kamili:

```python
my_str_int = '45'
my_str_float = '7.8'

converted_int = int(my_str_int)
converted_float = float(my_str_float)

print(converted_int, type(converted_int))  # 45 <class 'int'>
print(converted_float, type(converted_float))  # 7.8 <class 'float'>
```

Hapa kuna njia nyingine Python inazotoa za kufanya kazi na nambari kamili na nambari za desimali.

- `round()`: Huzungusha nambari hadi idadi ya tarakimu za desimali zilizobainishwa. Kwa chaguo-msingi kitendakazi hiki huzungusha hadi nambari kamili iliyo karibu zaidi, na hurudisha nambari kamili isiyo na sehemu za desimali:
    

```python
my_int_1 = 4.798
my_int_2 = 4.253

rounded_int_1 = round(my_int_1)
rounded_int_2 = round(my_int_2, 1)

print(rounded_int_1) # 5
print(rounded_int_2) # 4.3
```

- `abs()`: hurudisha thamani kamili ya nambari,
    

```python
num = -15

absolute_value = abs(num)
print(absolute_value) # 15
```

- `pow()`: huinua nambari hadi nguvu ya nyingine au hufanya nguvu za modulo.
    

```python
result_1 = pow(2, 3)  # Equivalent to 2 ** 3
print(result_1)  # 8

result_2 = pow(2, 3, 5)  # (2 ** 3) % 5
print(result_2)  # 3
```
