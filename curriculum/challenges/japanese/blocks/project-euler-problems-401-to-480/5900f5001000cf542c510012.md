# --description--

$E_a$ は方程式 $x^2 + 4y^2 = 4a^2$ の形をした楕円です。

$E_a'$ は原点 $O(0, 0)$ を中心に反時計回りに $θ$ 度回転させた $E_a$ の像で、$0° < θ < 90°$ です。

<img alt="ellipse E_a and ellipse rotated by θ degrees E_a'" src="https://cdn.freecodecamp.org/curriculum/project-euler/crisscross-ellipses.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$b$ は原点に最も近い2つの交点までの距離で、$c$ は他の2つの交点までの距離です。

順序付けられた三つ組 ($a$, $b$, $c$) が、$a$、$b$、$c$ がすべて正の整数である場合、これを標準楕円三つ組と呼びます。

例えば、(209, 247, 286) は標準楕円三つ組です。

$C(N)$ を $a ≤ N$ の範囲での異なる標準楕円三つ組 ($a$, $b$, $c$) の数とします。

$C({10}^3) = 7$、$C({10}^4) = 106$、$C({10}^6) = 11\,845$ であることが確認されています。

$C({10}^{17})$ を求めてください。
