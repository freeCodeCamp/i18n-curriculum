# --interactive--

Potresti già conoscere il lavoro con gli attributi nei tuoi progetti HTML e CSS. Ma sapevi che puoi aggiungere attributi ai tuoi elementi HTML nel file JavaScript?

In questa lezione imparerai come lavorare con il metodo `setAttribute()` per aggiungere attributi ai tuoi elementi HTML.

Ecco la sintassi di base:

```js
setAttribute(attribute, value);
```

Diamo un’occhiata a qualche esempio per capire meglio come usare il metodo `setAttribute()`.

In questo primo esempio, abbiamo un elemento `p` all’interno dell’HTML:

```html
<p id="para">I am a paragraph</p>
```

Per aggiungere un attributo `class` dobbiamo prima consultare quell’elemento `p` usando il metodo `getElementById()`. Poi possiamo usare il metodo `setAttribute()` per aggiungere l’attributo `class` e impostare il valore su `my-class`.

Stiamo usando `outerHTML` per mostrare l’intero elemento HTML compresi i suoi attributi.

:::interactive_editor

```html
<p id="para">I am a paragraph</p>
<script src="index.js"></script>
```

```js
const para = document.getElementById("para");
para.setAttribute("class", "my-class");

console.log(`${para.outerHTML}`);
```

:::

Se un elemento HTML ha già un attributo esistente, puoi aggiornare il suo valore usando il metodo `setAttribute()`.

In questo esempio abbiamo un elemento `div` con un attributo `class` impostato su `my-class`:

```html
<div class="my-class"></div>
```

Per aggiornare il valore dell’attributo `class` a example, possiamo usare di nuovo il metodo `setAttribute()`:

:::interactive_editor

```html
<div class="my-class"></div>
<script src="index.js"></script>
```

```js
const divEl = document.querySelector(".my-class");
divEl.setAttribute("class", "example");

console.log(`${divEl.outerHTML}`);
```

:::

Allora, quali sono alcuni esempi pratici di quando usare il metodo `setAttribute()`?

Se stessi costruendo una galleria di immagini dinamica, potresti dover aggiornare l’attributo `src` di un elemento immagine quando un utente clicca su una miniatura.

Un altro esempio sarebbe se stai gestendo la validazione di un modulo e devi aggiungere certi attributi come `required` o `minlength` a un elemento `input`.
