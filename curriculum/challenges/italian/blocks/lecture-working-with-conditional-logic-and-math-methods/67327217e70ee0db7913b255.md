# --interactive--

Quando inizi a studiare JavaScript, scoprirai presto che eseguire operazioni matematiche è un’attività comune. Mentre gli operatori aritmetici di base possono gestire calcoli semplici, JavaScript offre un oggetto `Math` integrato per affrontare sfide matematiche più complesse.

Questo strumento utile fornisce una varietà di metodi che rendono più facile eseguire calcoli avanzati e manipolare i numeri. Esploriamo questi metodi e vediamo come possono semplificare la tua esperienza di programmazione.

Il metodo `Math.random()` genera un numero a virgola mobile casuale compreso tra `0` (incluso) e `1` (escluso). Questo significa che l’output possibile può essere `0`, ma non raggiungerà mai `1`. Ecco un esempio che usa il metodo `Math.random()`:

:::interactive_editor

```js
const randomNum = Math.random();

console.log(randomNum);
// any number between 0 and 1 – 0 inclusive and 1 exclusive
```

:::

`Math.min()` e `Math.max()` prendono entrambi un insieme di numeri e restituiscono rispettivamente il valore minimo e massimo. Ecco un esempio che usa entrambi questi metodi:

:::interactive_editor

```js
const smallest = Math.min(1, 5, 3, 9);
console.log(smallest); // 1

const largest = Math.max(1, 5, 3, 9);
console.log(largest); // 9
```

:::

La prima `console.log()` stamperà il numero `1`, dato che `1` è il più piccolo in quella lista di numeri. E la seconda `console.log()` stamperà il numero `9`, dato che `9` è il numero più grande in quella lista.

Se vuoi arrotondare i numeri verso l’alto o verso il basso all’intero più vicino, puoi usare i metodi `Math.ceil()` e `Math.floor()`. Ecco un esempio che usa `Math.ceil()`:

:::interactive_editor

```js
console.log(Math.ceil(4.3)); // 5
```

:::

`Math.ceil()` arrotonderà `4.3` verso l’alto all’intero più vicino, che in questo caso è `5`. Ora, vediamo come arrotondare un numero verso il basso:

:::interactive_editor

```js
console.log(Math.floor(4.7)); // 4
```

:::

`Math.floor()` arrotonderà `4.7` verso il basso all’intero più vicino, che in questo caso è `4`. `Math.round()` è l’ibrido tra `Math.ceil()` e `Math.floor()`. Arrotonda un numero al suo intero più vicino, tenendo conto della parte decimale:

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

Quindi, se la parte decimale è inferiore a `5`, il numero viene arrotondato verso il basso. E se la parte decimale è `5` o superiore, il numero viene arrotondato verso l’alto. Un’applicazione pratica di `Math.floor()` e `Math.random()` è generare un numero casuale tra due numeri interi. Ecco la sintassi per farlo:

:::interactive_editor

```js
const max = 10;
const min = 5;
const randomNum = Math.floor(Math.random() * (max - min + 1)) + min;
console.log(randomNum);
```

:::

Generare un numero casuale tra `20` e `1` sarebbe così:

:::interactive_editor

```js
const randomNumBtw1And20 = Math.floor(Math.random() * 20) + 1;
console.log(randomNumBtw1And20);
```

:::

Un altro metodo `Math` utile è il metodo `Math.trunc()`. `Math.trunc()` rimuove la parte decimale di un numero, restituendo solo la parte intera, senza arrotondare:

:::interactive_editor

```js
console.log(Math.trunc(2.9)); // 2
console.log(Math.trunc(9.1)); // 9
```

:::

Se hai bisogno di ottenere la radice quadrata o la radice cubica di un numero, puoi usare rispettivamente i metodi `Math.sqrt()` e `Math.cbrt()`:

:::interactive_editor

```js
console.log(Math.sqrt(81)); // 9
console.log(Math.cbrt(27)); // 3
```

:::

La prima istruzione di log stamperà `9` perché la radice quadrata di `81` è `9`, mentre la seconda stamperà `3` perché la radice cubica di `27` è `3`. Se hai bisogno di ottenere il valore assoluto di un numero, puoi usare il metodo `Math.abs()`:

:::interactive_editor

```js
console.log(Math.abs(-5)); // 5
console.log(Math.abs(5)); // 5
```

:::

`Math.abs()` restituisce il valore assoluto di un numero, trasformando i negativi in positivi. L’ultimo metodo che vedremo sarà il metodo `Math.pow()`:

:::interactive_editor

```js
console.log(Math.pow(2, 3)); // 8
console.log(Math.pow(8, 2)); // 64
```

:::

`Math.pow()` prende due numeri ed eleva il primo alla potenza del secondo. Ci sono molti altri metodi che appartengono all’oggetto `Math` che puoi esplorare da solo. Tuttavia, questi sono solo alcuni dei più comunemente usati nei codici JavaScript.
