# --description--

Стилі для елемента `div` починають складатися. Але було б добре, якби навколо тексту був пробіл.

Щоб застосувати `padding` до елемента за допомогою Tailwind CSS, ви можете використати такий шаблон: `p-number`, де `number` — це кількість пікселів, яку ви хочете застосувати з усіх чотирьох сторін.

Ось приклад застосування `padding` розміром `8px` до елемента:

```html
<p class="p-8">This is padding.</p>
```

Всередині відкриваючого тегу `div` використайте правильний клас утиліти, щоб застосувати `padding` розміром `4px`.

# --hints--

Ваш відкриваючий тег `div` має містити клас утиліти `p-4`.

```js
const divEl = document.querySelector("div");
assert.isTrue(divEl.classList.contains("p-4"));
```
