# --description--

현재 위치에 특정 장소나 물건이 있는지 물어보고 싶을 때는 질문 구조 `Are there any ... around here?`를 사용할 수 있습니다. 예를 들어:

`Are there any Wi-Fi spots around here?` - 이 장소 근처에 Wi-Fi가 있는지 묻는 것입니다.

`around here`라는 표현은 근처 영역을 가리킵니다.
