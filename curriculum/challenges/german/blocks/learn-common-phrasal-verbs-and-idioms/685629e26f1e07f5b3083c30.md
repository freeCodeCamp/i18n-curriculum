# --explanation--

`Real-time alerts` sind Meldungen oder Warnungen, die sofort erscheinen, sobald etwas passiert. Zum Beispiel:

`The app sends real-time alerts if someone logs in from a new location.` – Das bedeutet, dass jedes Mal, wenn sich jemand von einem anderen Ort anmeldet, der Benutzer sofort eine Warnmeldung erhält.

`Graphics` sind Bilder oder visuelle Elemente, die verwendet werden, um Informationen zu vermitteln. Zum Beispiel:

`The dashboard uses graphics to show project progress.` – Das bedeutet, dass visuelle Hilfsmittel wie Diagramme oder Symbole verwendet werden, um Daten leichter verständlich zu machen.
