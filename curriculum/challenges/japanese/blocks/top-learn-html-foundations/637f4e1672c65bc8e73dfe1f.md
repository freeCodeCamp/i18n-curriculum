# --description--

要素はコンテンツのコンテナのように考えられます。開始タグと終了タグがブラウザーに要素がどのコンテンツを含んでいるかを伝えます。ブラウザーはその情報を使って、コンテンツをどのように解釈し、フォーマットするかを決定します。

終了タグを持たないHTML要素もあります。これらの要素はしばしば`<br />`や`<img/>`のように見えますが、`<br>`や`<img>`のように終了のスラッシュなしで使われる場合もあります。これらは空要素またはvoid要素と呼ばれ、コンテンツを囲まないためです。これらは後のレッスンでいくつか出てきますが、ほとんどの場合、要素は開始タグと終了タグの両方を持ちます。

HTMLにはさまざまな要素を作成するために使える膨大な数の定義済みタグがあります。コンテンツに正しいタグを使うことが重要です。正しいタグを使うことは、サイトの検索エンジンでのランキングや、スクリーンリーダーのような支援技術に頼るユーザーのアクセスのしやすさに大きな影響を与えます。

コンテンツに適切な要素を使うことをセマンティックHTMLと呼びます。これはカリキュラムの後半でより詳しく学びます。
