# --explanation--

`No one mentioned anything` bedeutet, dass niemand etwas Ungewöhnliches gemeldet hat.

`Ask around` bedeutet, mit verschiedenen Personen zu sprechen, um Informationen zu sammeln. Zum Beispiel:

- `I'll ask around to see if anyone changed the settings.` – Sie werden mit Leuten sprechen, um herauszufinden, wer die Einstellungen geändert hat.

- `He asked around about the new software tool.` – Er hat mit anderen gesprochen, um mehr Informationen über das Tool zu erhalten.
