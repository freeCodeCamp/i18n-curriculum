# --description--

Числа у формі $n^{15} + 1$ є складеними для кожного цілого числа $n > 1$.

Для натуральних чисел $n$ та $m$ нехай $s(n, m)$ визначається як сума різних простих множників числа $n^{15} + 1$, що не перевищують $m$.

Наприклад, $2^{15} + 1 = 3 × 3 × 11 × 331$.

Отже, $s(2, 10) = 3$ та $s(2, 1000) = 3 + 11 + 331 = 345$.

Також ${10}^{15} + 1 = 7 × 11 × 13 × 211 × 241 × 2161 × 9091$.

Отже, $s(10, 100) = 31$ та $s(10, 1000) = 483$.

Знайдіть $\sum s(n, {10}^8)$ для $1 ≤ n ≤ {10}^{11}$.
