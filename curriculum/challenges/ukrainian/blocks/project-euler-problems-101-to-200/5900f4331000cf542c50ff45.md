# --description--

Найкращим наближенням до дійсного числа $x$ для обмеження знаменника $d$ є раціональне число $\frac{r}{s}$ (у скороченій формі) з $s ≤ d$, таке, що будь-яке раціональне число $\frac{p}{q}$, яке ближче до $x$, ніж $\frac{r}{s}$, має $q > d$.

Зазвичай найкраще наближення до дійсного числа є унікальним для всіх обмежень знаменника. Однак існують винятки, наприклад, $\frac{9}{40}$ має два найкращі наближення $\frac{1}{4}$ та $\frac{1}{5}$ для обмеження знаменника $6$. Ми назвемо дійсне число $x$ неоднозначним, якщо існує принаймні одне обмеження знаменника, для якого $x$ має два найкращі наближення. Очевидно, що неоднозначне число обов’язково є раціональним.

Скільки існує неоднозначних чисел $x = \frac{p}{q}$, $0 < x < \frac{1}{100}$, знаменник $q$ яких не перевищує ${10}^8$?
