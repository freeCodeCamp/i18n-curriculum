# --description--

Alice und Bob spielen das Spiel Nim Square.

Nim Square ist genau wie das gewöhnliche Nim mit drei Heaps im Normalspiel, aber die Spieler dürfen nur eine Quadratzahl von Steinen aus einem Heap entfernen.

Die Anzahl der Steine in den drei Heaps wird durch das geordnete Tripel ($a$, $b$, $c$) dargestellt.

Wenn $0 ≤ a ≤ b ≤ c ≤ 29$ gilt, dann beträgt die Anzahl der Verlustpositionen für den nächsten Spieler 1160.

Finden Sie die Anzahl der Verlustpositionen für den nächsten Spieler, wenn $0 ≤ a ≤ b ≤ c ≤ 100\\,000$ gilt.
