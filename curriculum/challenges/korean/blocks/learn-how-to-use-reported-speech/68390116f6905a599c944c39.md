# --explanation--

`If only`는 과거에 일어나지 않은 일에 대해 바람이나 후회를 표현할 때 문장 맨 앞에 사용합니다. 이는 화자가 상황이 달랐으면 좋겠다는 뜻을 나타냅니다. 예를 들어:

`If only we had checked the system earlier, we could have avoided the crash.` – 이는 그들이 충분히 일찍 확인하지 않았고 지금 후회하고 있다는 의미입니다.
