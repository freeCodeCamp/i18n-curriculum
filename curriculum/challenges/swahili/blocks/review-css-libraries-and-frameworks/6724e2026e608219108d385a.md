## Mifumo ya kazi ya CSS

- **Mifumo ya kazi ya CSS**: Mifumo ya kazi ya CSS inaweza kuharakisha mtiririko wako wa kazi, kuunda mtindo wa kuona unaolingana katika tovuti nzima, kufanya muundo wako uonekane sawa katika vivinjari vingi, na kuweka msimbo wako wa CSS kuwa na mpangilio mzuri zaidi.
- **Mifumo maarufu ya kazi ya CSS**: Baadhi ya mifumo maarufu ya kazi ya CSS ni Tailwind CSS, Bootstrap, Materialize, na Foundation.
- **Hasara zinazoweza kutokea**:
  - CSS inayotolewa na mfumo wa kazi inaweza kuleta mgongano na CSS yako ya kawaida.
  - Tovuti yako inaweza kuonekana kama tovuti nyingine zinazotumia mfumo huo huo.
  - Mifumo mikubwa inaweza kusababisha matatizo ya utendaji.

## Aina mbili za mifumo ya kazi ya CSS

- **Mifumo ya kazi ya CSS inayotegemea matumizi ya darasa ndogo**: Mifumo hii ina madarasa madogo yenye madhumuni maalum, kama kuweka ukingo, nafasi ya ndani, au rangi ya nyuma. Unaweza kuweka madarasa haya madogo moja kwa moja kwenye vipengele vya HTML kama inavyohitajika. Tailwind CSS inahesabiwa kama mfumo wa kazi wa CSS unaotegemea matumizi ya darasa ndogo.

Hapa kuna mfano wa kutumia Tailwind CSS kupamba kitufe.

```html
<button class="bg-blue-500 text-white font-bold py-2 px-4 rounded-full hover:bg-blue-700">
  Button
</button>
```

- **Mifumo ya kazi ya CSS inayotegemea sehemu**: Mifumo hii ina sehemu zilizojengwa tayari zenye mitindo iliyowekwa kabla ambazo unaweza kuongeza kwenye tovuti yako. Sehemu hizi zinapatikana katika nyaraka rasmi za mfumo wa kazi wa CSS, na unaweza kuzirekebisha na kuzibandika kwenye mradi wako. Bootstrap inahesabiwa kama mfumo wa kazi wa CSS unaotegemea sehemu.

Hapa kuna mfano wa kutumia Bootstrap kuunda kundi la orodha. Badala ya kutumia madarasa madogo kwenye vipengele vya HTML, utaongeza sehemu nzima, ikiwa ni pamoja na muundo wa HTML.

```html
<div class="card" style="width: 25rem;">
  <ul class="list-group list-group-flush">
    <li class="list-group-item">HTML</li>
    <li class="list-group-item">CSS</li>
    <li class="list-group-item">JavaScript</li>
  </ul>
</div>
```

## Tailwind CSS

Tailwind ni mfumo wa kazi wa CSS unaotegemea matumizi ya darasa ndogo. Badala ya kuandika sheria za CSS za kawaida, unajenga miundo yako kwa kuchanganya madarasa madogo ya matumizi moja kwa moja katika HTML yako.

### Zana za muundo unaobadilika

Tailwind hutumia kiambishi awali kama `sm:`, `md:`, na `lg:` kuweka mitindo kwa ukubwa tofauti wa skrini.

```html
<div class="w-full md:w-1/2 lg:flex-row">Responsive layout</div>
```

### Zana za Flexbox

Madarasa kama `flex`, `flex-col`, `justify-around`, na `items-center` hufanya iwe rahisi kuunda mipangilio inayobadilika.

```html
<div class="flex flex-col md:flex-row justify-around items-center">
  <p>Column on small screens</p>
  <p>Row on medium and larger screens</p>
</div>
```

### Zana za Grid

Tailwind inajumuisha zana za CSS Grid, kama `grid`, `grid-cols-1`, na `md:grid-cols-3`.

```html
<div class="grid grid-cols-1 md:grid-cols-3 gap-8">
  <div class="bg-gray-100 p-4">Column 1</div>
  <div class="bg-gray-100 p-4">Column 2</div>
  <div class="bg-gray-100 p-4">Column 3</div>
</div>
```

### Zana za nafasi

Zana kama `mt-8`, `mx-auto`, `p-4`, na `gap-4` husaidia kuunda nafasi zinazolingana bila kuandika CSS.

```html
<div class="mt-8 p-4 bg-indigo-600 text-white">Spaced content</div>
```

### Zana za Typography

Zana kama `uppercase`, `font-bold`, `font-semibold`, na `text-4xl` hudhibiti muonekano wa maandishi.

Unaweza kuweka ukubwa wa fonti unaobadilika kwenye sehemu za kuvunja, kama `text-3xl` `md:text-5xl`.

```html
<h1 class="text-3xl md:text-5xl font-semibold text-center">Responsive Heading
</h1>
```

### Rangi na hali za hover

Tailwind hutoa rangi nyingi, kama `text-red-700`, `bg-indigo-600`, na `bg-gray-100`.

Madarasa kama `hover:bg-pink-600` hufanya athari za mwingiliano kuwa rahisi.

```html
<a href="#" class="bg-pink-500 hover:bg-pink-600 text-white px-4 py-2 rounded-md">
  Hover Me
</a>
```

### Mipaka, pete, na athari

- **Mipaka**: `border-2 border-red-300` huongeza mipaka yenye unene na rangi zilizobainishwa.
- **Pete**: `ring-1 ring-gray-300` huunda athari za mduara zinazotumika mara nyingi kwa makini au kadi.
- **Pembe zilizozungushwa na upanuzi**: Madarasa kama `rounded-md`, `rounded-xl`, na `scale-105` huongeza urembo.

```html
<div class="p-6 rounded-xl ring-2 ring-fuchsia-500 scale-105">
  Highlighted card
</div>
```

### Mwinuko wa rangi

Tailwind inaunga mkono zana za mwinuko wa rangi kama `bg-gradient-to-r from-fuchsia-500 to-indigo-600`.

```html
<div class="p-4 text-white bg-gradient-to-r from-fuchsia-500 to-indigo-600">
  Gradient background
</div>
```

## Wasindikaji wa awali wa CSS

- **Msindikaji wa awali wa CSS**: Msindikaji wa awali wa CSS ni chombo kinachoongeza uwezo wa CSS ya kawaida. Huchambua msimbo wenye sintaksia iliyopanuliwa na kuutafsiri kuwa faili halisi la CSS. Inaweza kusaidia kuandika CSS safi, inayoweza kutumika tena, isiyo na kurudiarudia, na inayoweza kupanuka kwa miradi tata.
- **Vipengele**: Baadhi ya vipengele vinavyotolewa na wasindikaji wa awali wa CSS ni vigezo, mixin, upangaji ndani, na urithi wa kichaguzi.
- **Wasindikaji maarufu wa awali wa CSS**: Baadhi ya wasindikaji maarufu wa awali wa CSS ni Sass, Less, na Stylus.  
- **Hasara zinazoweza kutokea**:
  - Kusindika sheria za CSS kuwa CSS ya kawaida kunaweza kusababisha mzigo wa ziada.
  - Msimbo uliosindikwa unaweza kuwa mgumu kutafuta makosa.

## Sass

- **Sass**: Ni mojawapo ya wasindikaji maarufu wa awali wa CSS. Sass inasimama kwa "Syntactically Awesome Style Sheets."
- **Vipengele vinavyoungwa mkono na Sass**: Sass inaunga mkono vipengele kama vigezo, sheria za CSS zilizopangwa ndani, moduli, mixin, urithi, na waendeshaji wa hesabu za msingi.

## Sintaksia mbili zinazoungwa mkono na Sass

- **Sintaksia ya SCSS**: SCSS (Sassy CSS) inaongeza sintaksia ya msingi ya CSS. Ni sintaksia inayotumika zaidi kwa Sass. Faili za SCSS zina kiambishi awali cha `.scss`.

Hapa kuna mfano wa kufafanua na kutumia kigezo katika SCSS.

```scss
$primary-color: #3498eb;

header {
  background-color: $primary-color;
}
```

- **Sintaksia iliyopangwa kwa nafasi**: Sintaksia iliyopangwa kwa nafasi ilikuwa sintaksia ya awali ya Sass na pia inajulikana kama "Sintaksia ya Sass."

Hapa kuna mfano wa kufafanua na kutumia kigezo katika sintaksia iliyopangwa kwa nafasi.

```sass
$primary-color: #3498eb

header
  background-color: $primary-color
```

### Mixin

- **Mixin**: Mixin hukuruhusu kuunganisha mali nyingi za CSS na thamani zao chini ya jina na kutumia tena sehemu hiyo ya msimbo wa CSS katika karatasi yako ya mitindo.

Hapa kuna mfano wa kufafanua mixin katika sintaksia ya SCSS. Katika kesi hii, mixin inaitwa `center-flex`. Ina mali tatu za CSS za kuweka vitu katikati kwa kutumia flexbox.

```scss
@mixin center-flex {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

Hapa kuna mfano wa kutumia mixin uliyofafanua.

```scss
section {
  @include center-flex;
  height: 500px;
  background-color: #3289a8;
}
```
