# --description--

Змінна, оголошена всередині функції, видима лише всередині цієї функції. Це називається областю видимості змінної, а ця конкретна область видимості називається локальною областю видимості. Змінні, оголошені поза межами функції, називаються глобальними змінними і видимі по всій програмі. Наприклад, наступний код викличе помилку:

```js
function myFunction() {
  let x = 10;
}

console.log(x);
```

Змінна `x` оголошена всередині функції `myFunction` і не видима поза межами функції. Код викличе `ReferenceError`, тому що `x` не визначена.

Якщо оголосити змінну поза межами функції, вона називається глобальною змінною і видима по всій програмі. Наприклад, наступний код працюватиме:

```js
let x = 10;

function myFunction() {
  console.log(x);
}

myFunction();
```

Якщо всередині функції оголосити змінну з таким самим іменем, вона затінює зовнішню.
