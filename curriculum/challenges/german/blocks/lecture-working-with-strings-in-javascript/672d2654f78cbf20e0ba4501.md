# --interactive--

In JavaScript werden Strings als Folgen von Zeichen behandelt, und auf jedes Zeichen in einem String kann mit Klammernotation zugegriffen werden. Damit können Sie ein bestimmtes Zeichen aus einem String basierend auf seiner Position abrufen, die als Index bezeichnet wird.

Ein Index ist die Position eines Zeichens innerhalb eines Strings und ist nullbasiert. Das bedeutet, dass das erste Zeichen eines Strings den Index `0` hat, das zweite Zeichen den Index `1` und so weiter.

Zum Beispiel befindet sich im String `hello` das Zeichen `h` am Index `0`, `e` am Index `1`, `l` am Index `2` und so weiter.

Die Klammernotation verwendet eckige Klammern (`[]`) und den Index des Zeichens, auf das Sie zugreifen möchten. Schauen wir uns ein Beispiel an:

:::interactive_editor

```js
let greeting = "hello";
console.log(greeting[1]); // "e"
```

:::

In diesem Beispiel können wir auf das Zeichen am Index `1` zugreifen, das `e` ist.

Um das letzte Zeichen eines Strings zu erhalten, können Sie die Länge des Strings minus eins verwenden.
Die `length`-Eigenschaft eines Strings sagt Ihnen, wie viele Zeichen er enthält. Um also auf das letzte Zeichen zuzugreifen, ziehen Sie eins von der Länge ab:

:::interactive_editor

```js
let greeting = "hello";
console.log(greeting[greeting.length - 1]); // "o"
```

:::

In diesem Fall ist die `length` von `hello` `5`, und das letzte Zeichen (`o`) befindet sich am Index `4`, der `5 - 1` ist.

Wenn Sie mehrere Zeichen erhalten möchten, können Sie die Klammernotation so verwenden:

:::interactive_editor

```js
let greeting = "hello";
let firstTwo = greeting[0] + greeting[1]; // "he"
console.log(firstTwo);
```

:::

In diesem Beispiel verketten wir das erste und zweite Zeichen mit Klammernotation, um den String `he` zu bilden.

Die Klammernotation ist nützlich, wenn Sie auf bestimmte Zeichen in einem String zugreifen müssen, zum Beispiel um Initialen aus einem Namen zu extrahieren oder einen bestimmten Buchstaben für die Validierung zu überprüfen.
