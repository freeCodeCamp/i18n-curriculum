# --description--

考慮一場 Ramvok 的單一遊戲：

令 $t$ 表述遊戲持續的最大回合數。若 $t = 0$，則遊戲立即結束。否則，在每個回合 $i$，玩家擲骰子。擲完後，若 $i &lt; t$，玩家可以選擇停止遊戲並獲得等於當前擲骰點數的獎勵，或捨棄該擲骰結果並在下一回合再試一次。若 $i = t$，則該擲骰結果不可捨棄，必須接受該獎勵。遊戲開始前，玩家選擇 $t$，並必須支付一筆固定成本 $ct$，其中 $c$ 為常數。當 $c = 0$ 時，$t$ 可選擇為無限大（且前期成本為 0）。令 $R(d, c)$ 為玩家在使用公平的 $d$ 面骰子及成本常數 $c$，以最佳策略玩 Ramvok 單場遊戲所獲得的期望利潤（即淨收益）。例如，$R(4, 0.2) = 2.65$。假設玩家有足夠資金支付任何／所有前期成本。

現在考慮一個 Super Ramvok 遊戲：

在 Super Ramvok 中，Ramvok 遊戲會反覆進行，但有些微的修改。每場遊戲結束後，骰子會被改變。改變的過程如下：骰子擲一次，如果擲出的面上有點數可見，該面就會被改成空白。如果該面已經是空白，則會改回原本的點數。改變完成後，可以開始另一場 Ramvok 遊戲（在這樣的遊戲中，每回合會持續擲骰子，直到出現有點數的面）。玩家隨時知道哪些面是空白的，哪些不是。當骰子的所有面都變成空白時，Super Ramvok 遊戲結束。

設 $S(d, c)$ 為玩家在以公平的 $d$ 面骰子開始（所有面皆可見）且成本常數為 $c$ 的最佳策略下，從 Super Ramvok 遊戲中獲得的期望利潤。 例如，$S(6, 1) = 208.3$。

令 $F(n) = \sum_{4 ≤ d ≤ n} \sum_{0 ≤ c ≤ n} S(d, c)$。

計算 $F(20)$，並四捨五入到最接近的整數。
