# --description--

对于每个整数 $n > 1$，函数族 $f_{n, a, b}$ 被定义为：

$f_{n, a, b}(x) ≡ ax + b\bmod n$，其中 $a, b, x$ 为整数且 $0 \lt a \lt n$，$0 \le b \lt n$，$0 \le x \lt n$。

如果对于每个 $0 \le x \lt n$，都有 $f_{n, a, b}(f_{n, a, b}(x)) \equiv f_{n, a, b}(x)\bmod n$，我们将称 $f_{n, a, b}$ 为一个收缩映射。

令 $R(n)$ 为 $n$ 的收缩数。

你被告知

$$\sum_{k = 1}^{99\\,999} R(\displaystyle\binom{100\\,000}{k}) \equiv 628\\,701\\,600\bmod 1\\,000\\,000\\,007$$

查找 $$\sum_{k = 1}^{9\,999\,999} R(\displaystyle\binom{10\,000\,000}{k})$$ 给出你的答案对 $1\,000\,000\,007$ 取模。
