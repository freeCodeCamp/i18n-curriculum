# --explanation--

Sebastián dice `¿Cómo se llama el ingeniero de software senior?`. Esto significa "¿Cuál es el nombre del ingeniero de software senior?"

Cuando la palabra `Cómo` inicia una pregunta, debe llevar tilde en la primera letra `o`.

Además, recuerda que las preguntas en español comienzan con un signo de interrogación de apertura, `¿`.
