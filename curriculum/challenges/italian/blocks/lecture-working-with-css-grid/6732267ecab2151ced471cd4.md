# --interactive--

Nelle lezioni precedenti, abbiamo parlato un po’ di come creare spazio tra gli elementi della griglia. Ma in questa lezione entreremo più nel dettaglio su come usare le proprietà `row-gap`, `column-gap` e `gap` in un layout a griglia.

Prima però dobbiamo rivedere cos’è una traccia in CSS grid. Una traccia è lo spazio tra due linee della griglia adiacenti. Queste linee vengono create automaticamente quando usi CSS Grid. In questo contesto, le tracce si riferiscono generalmente alle righe e alle colonne che compongono il layout della griglia.

Per creare spazi tra le colonne in una CSS Grid, puoi usare la proprietà `column-gap`. I valori accettabili per questa proprietà includono pixel, l’unità `em`, percentuali o la parola chiave `normal`.

Se usi il valore `normal` per la proprietà `column-gap`, il risultato sarà `0` per i layout a griglia e `1em` per i layout a più colonne.

Ecco un esempio del markup per un layout a griglia con quattro colonne:

```html
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

Per il CSS, impostiamo display su grid e la proprietà `column-gap` a 10 pixel:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  height: 100px;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.grid-container div {
  background-color: darkblue;
}
```

:::

Se volessimo modificare l’esempio per avere due righe di riquadri blu e creare più spazio tra le righe, possiamo usare la proprietà `row-gap`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  column-gap: 10px;
  row-gap: 30px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

In questo esempio rivisto, impostiamo la proprietà `row-gap` a 30 pixel e cambiamo la `grid-template-columns` per usare solo due unità `1fr` invece di quattro, così da creare due righe di riquadri.

Proprio come la proprietà `column-gap`, i valori accettabili per la proprietà `row-gap` possono includere percentuali, `em` e pixel.

Se vuoi usare una proprietà abbreviata per creare spazi tra righe e colonne, puoi usare la proprietà `gap`. Ecco la sintassi base:

```css
gap: row-value optional-column-value;
```

Se specifichi un solo valore per la proprietà `gap`, quel valore verrà applicato sia alle righe sia alle colonne. Se specifichi due valori, il primo andrà alla riga e il secondo alla colonna:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 30px 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

I valori accettabili per la proprietà abbreviata `gap` includono percentuali, pixel, `em` e persino valori `calc()`. Ma non puoi usare unità `fr` qui.

Le proprietà `row-gap`, `column-gap` e `gap` offrono modi flessibili per controllare lo spazio tra gli elementi in un layout CSS Grid. Usando queste proprietà, puoi facilmente creare griglie visivamente gradevoli con spazi coerenti e regolabili tra righe e colonne.
