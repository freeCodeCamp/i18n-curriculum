# --description--

Нехай $p_n$ — це $n$-те просте число: 2, 3, 5, 7, 11, ..., а $r$ — остача від ділення ${(p_n−1)}^n + {(p_n+1)}^n$ на ${p_n}^2$.

Наприклад, коли $n = 3$, $p_3 = 5$, і $4^3 + 6^3 = 280 ≡ 5\\ mod\\ 25$.

Найменше значення $n$, для якого остача вперше перевищує $10^9$, дорівнює 7037.

Знайдіть найменше значення $n$, для якого остача вперше перевищує $10^{10}$.
