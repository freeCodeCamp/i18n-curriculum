# --description--

Der Begriff `reusable` bezeichnet etwas, das mehrmals verwendet werden kann, insbesondere in verschiedenen Situationen oder für unterschiedliche Zwecke. In der Programmierung beschreibt `reusable` häufig Code, der so gestaltet ist, dass er mehrfach verwendet werden kann, entweder innerhalb desselben Programms oder in verschiedenen Programmen, ohne neu geschrieben werden zu müssen.
