# --description--

Für eine positive ganze Zahl $n$ und Ziffern $d$ definieren wir $F(n, d)$ als die Anzahl der Teiler von $n$, deren letzte Ziffern gleich $d$ sind.

Zum Beispiel gilt $F(84, 4) = 3$. Unter den Teilern von 84 (1, 2, 3, 4, 6, 7, 12, 14, 21, 28, 42, 84) haben drei von ihnen (4, 14, 84) die letzte Ziffer 4.

Wir können auch überprüfen, dass $F(12!, 12) = 11$ und $F(50!, 123) = 17\\,888$ gilt.

Finden Sie $F({10}^6!, 65\\,432) \text{ modulo } ({10}^{16} + 61)$.
