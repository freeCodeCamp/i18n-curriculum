# --description--

La preposizione `on` viene spesso usata per indicare il soggetto o l'argomento di qualcosa, come ciò su cui commentare `on` in un progetto. Per esempio, `The report focused on the project's progress.`

`Any` si usa per chiedere se ci sono uno o più elementi, senza specificare quali. Come chiedere se ci sono `any` domande.
