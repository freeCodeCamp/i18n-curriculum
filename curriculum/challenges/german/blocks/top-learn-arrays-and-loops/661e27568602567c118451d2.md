# --description--

Um auf die Elemente eines Arrays zuzugreifen, können Sie die Indexnummer verwenden. Die Indexnummer beginnt bei 0, daher befindet sich das erste Element eines Arrays am Index 0, das zweite Element am Index 1 und so weiter.

Zum Beispiel können Sie, um auf das erste Element des `fruits` Arrays zuzugreifen, den folgenden Code verwenden:

```javascript
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits[0]); // Output: apple
console.log(fruits[2]); // Output: orange
```

Wenn der Index außerhalb des Bereichs liegt, gibt JavaScript `undefined` zurück. Das bedeutet, wenn Sie versuchen, in dem obigen Beispiel auf `fruits[3]` zuzugreifen, wird `undefined` zurückgegeben.
