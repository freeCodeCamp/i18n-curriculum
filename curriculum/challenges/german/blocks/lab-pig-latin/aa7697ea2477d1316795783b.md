# --description--

Pig Latin ist eine Methode, englische Wörter durch bestimmte Umwandlungsregeln zu verändern.

**Ziel:** Erfüllen Sie die untenstehenden User Stories und bestehen Sie alle Tests, um das Lab abzuschließen.

**User Stories:**

1. Sie sollten eine `translatePigLatin`-Funktion erstellen, die einen String als Argument akzeptiert.
1. Beginnt das String-Argument mit einem Konsonanten, soll Ihre Funktion den ersten Konsonanten oder die Konsonantenfolge nehmen, ans Ende des Wortes verschieben, `ay` hinzufügen und das Ergebnis zurückgeben.
1. Beginnt das String-Argument mit einem Vokal, soll Ihre Funktion `way` ans Ende anhängen und das Ergebnis zurückgeben.
1. Ihre Funktion soll String-Argumente verarbeiten, bei denen der erste Vokal in der Wortmitte steht, und den entsprechend umgewandelten String zurückgeben.
1. Hat das String-Argument keine Vokale, soll Ihre Funktion `ay` ans Ende anhängen und das Ergebnis zurückgeben.

Hinweis: Für den Kontext dieses Labs sind die Vokale `a`, `e`, `i`, `o` und `u`. Der Buchstabe `y` wird nicht als Vokal betrachtet.

```js
function translatePigLatin(str) {
  if (isVowel(str.charAt(0))) return str + "way";
  let front = [];
  str = str.split('');
  while (str.length && !isVowel(str[0])) {
    front.push(str.shift());
  }
  return [].concat(str, front).join('') + 'ay';
}

function isVowel(c) {
  return ['a', 'e', 'i', 'o', 'u'].indexOf(c.toLowerCase()) !== -1;
}
```
