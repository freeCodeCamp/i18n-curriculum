# --description--

Neno `issue` linaweza kumaanisha tatizo au jambo ambalo linahitaji kushughulikiwa.

Katika uandishi wa programu, `issue` mara nyingi linahusiana na hitilafu au kosa katika msimbo unaohitaji kurekebishwa. Kwa mfano, `We found an issue in the latest software update.`

# --questions--
