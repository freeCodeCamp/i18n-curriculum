# --explanation--

`我是设计师 (wǒ shì shè jì shī)` usa la estructura `我是 (wǒ shì)` + profesión para presentar el trabajo de uno.

`我是 (wǒ shì)` significa "yo soy…", y `设计师 (shè jì shī)` significa "diseñador". Así que Liu Ming está diciendo que es diseñador.
