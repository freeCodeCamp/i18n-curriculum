# --description--

`Data` は、特に意思決定を助けるために収集され、検討される事実や数字などの情報を意味します。

`quantitative data` のように数字に関するもの（例えば、何人が店舗を訪れるか）や、`qualitative data` のように説明に関するもの（例えば、製品について人々がどう思っているか）など、さまざまな種類のデータがあります。

`Customer data` は、顧客の購買習慣や好みなど、顧客に関する具体的な情報です。

例えば、店舗が顧客が最もよく訪れる時間帯や、どの製品が最も人気があるかについて `data` を収集する場合があります。
