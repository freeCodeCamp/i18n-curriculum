### --feedback--

Essas duas palavras funcionam juntas para descrever como os elementos são posicionados incorretamente em um dispositivo específico.Uma palavra lida com o problema de posicionamento e a outra especifica a localização.

# --explanation--

`Overlap` é usado quando uma coisa cobre ou se estende sobre parte de outra.Nesse contexto, significa que os elementos na tela estão posicionados incorretamente, cobrindo partes uns dos outros.

`On` é usado para indicar o dispositivo onde ocorre a sobreposição.Ele mostra a relação entre os elementos e os dispositivos.Por exemplo:

`The images overlap` na página da web, dificultando ver todo o conteúdo claramente. - Aqui, `overlap` descreve o problema de posicionamento e `on` especifica onde ele ocorre.
