# --description--

`Otro` (Masculino) y `otra` (Femenino) se usan para hablar de una cosa adicional o diferente. Por ejemplo:

- `Quiero otro café.` – Quiero otro café. (`Café` es masculino, por eso se usa `otro` aquí.)

- `Necesito otra copia.` – Necesito otra copia. (`Copia` es femenino por eso se usa `otra` aquí.)

Al confirmar cosas, a menudo escucharás `otra` combinado con `vez`.

`Vez` significa una ocasión o el momento en que algo sucede. Es un sustantivo femenino. Debido a que `vez` es femenino, la expresión siempre usa la forma femenina:

`Puedes repetirlo otra vez?` - ¿Puedes repetirlo otra vez?

Esta expresión común se usa para hablar de hacer algo una vez más o de nuevo.
