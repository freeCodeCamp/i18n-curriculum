# --description--

`What` 經常用來詢問特定的資訊。它可以詢問某人想知道的事物、動作或細節。 例如：

- `What time is the meeting?`－這是詢問會議的確切時間。

- `What do you usually have for lunch?`－這是詢問某人常式的詳細資料。

所以，`what` 幫助你在想知道特定事情時獲得明確的答案。
