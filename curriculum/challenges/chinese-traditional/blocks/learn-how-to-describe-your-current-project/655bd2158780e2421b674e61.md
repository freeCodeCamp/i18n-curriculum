# --description--

`Data` 是指資料，特別是為了檢視和考慮而收集的事實或數字，並用來協助決策。 

有不同型別的資料，例如 `quantitative data`，它是關於數字的（例如有多少人造訪商店），以及 `qualitative data`，它是關於描述的（例如人們對產品的看法）。 

`Customer data` 是關於客戶的特定資訊，例如他們的購買習慣或偏好。 

例如，商店可能會收集關於顧客最常在什麼時間造訪，或哪些產品最受歡迎的 `data`。
