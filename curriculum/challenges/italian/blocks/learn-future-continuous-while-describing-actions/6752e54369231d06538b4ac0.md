# --explanation--

`Frustrated` significa sentirsi infastiditi, contrariati o delusi a causa di difficoltà o ostacoli. Per esempio:  

`She was frustrated because the software kept crashing.` - Questo significa che lei si è sentita infastidita e contrariata perché il software non funzionava correttamente.

`Access` significa la capacità di usare o collegarsi a qualcosa, specialmente tecnologia o informazioni. Per esempio:  

`The team couldn't access the server due to a network issue.` - Questo significa che il team non è riuscito a collegarsi al server a causa di un problema con la rete.
