# --explanation--

詳しく見てみましょう：

マークは何と言いましたか？BIOSが`prevents the computer from starting.`と言っています。つまり、BIOSが破損したときに**できること**について話しています。

サラが同意するなら、BIOSが**何であるか**について話すでしょうか、それともBIOSが**できること**について話すでしょうか？後者ですよね？だから`Yeah, it can`が正しい選択です。

なぜ`yeah, it is`が間違いなのか？`it is`はBIOSの状態について話していて、できることについてではないからです。例を挙げると：

**正しい例**：`Yeah, it can.` – これはマークのBIOSができることに関する指摘に合っています。

**間違いの例**：`Yeah, it is.` – マークはBIOSが何であるかを説明しているのではなく、その影響について話しているので合いません。
