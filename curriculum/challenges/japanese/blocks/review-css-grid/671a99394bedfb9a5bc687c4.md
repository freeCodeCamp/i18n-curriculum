## CSS Gridの基本

- **定義**: CSS Gridは、ウェブページで複雑なレイアウトを作成するための二次元レイアウトシステムです。グリッドは行と列があり、それらの間にギャップがあります。グリッドレイアウトを定義するには、`display`プロパティを`grid`に設定します。

```css
.container {
  display: grid;
}
```

- **`fr`（分数）単位**: この単位はグリッドコンテナ内のスペースの分数を表します。柔軟なグリッドを作成するために`fr`単位を使うことができます。
- **トラック間のギャップの作成**: CSS Gridでトラック間にギャップを作る方法は3つあります。`column-gap`プロパティを使って列間のギャップを作成できます。`row-gap`プロパティを使って行間のギャップを作成できます。または`gap`省略形プロパティを使って行と列の両方のギャップを作成できます。
- **`grid-template-columns`**: これはグリッドトラックの列のライン名とサイズを設定するために使います。

```css
.container {
  display: grid;
  width: 100%;
  grid-template-columns: 30px 1fr;
}
```

- **`grid-template-rows`**: これはグリッドトラックの行のライン名とサイズを設定するために使います。
- **`grid-auto-flow`**: これは自動配置されたアイテムがグリッド内でどのように収まるかを決定します。

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-flow: column;
}
```

- **`grid-auto-columns`**: これは暗黙的に作成された列のサイズを設定するために使います。

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-columns: auto;
}
```

- **`place-items`**: これはブロック方向とインライン方向の両方でアイテムを整列させるために使います。
- **`align-items`**: これはグリッドコンテナ内のアイテムの整列を設定するために使います。
- **`repeat()`関数**: この関数はトラックリスト内のセクションを繰り返すために使います。`grid-template-columns: 1fr 1fr 1fr;`と書く代わりに`repeat()`関数を使うことができます。

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}
```

- **明示的グリッド**: `grid-template-columns`または`grid-template-rows`プロパティを使ってラインやトラックの数を指定できます。
- **暗黙的グリッド**: アイテムがグリッドの外に配置されると、その外側の要素のために行や列が自動的に作成されます。
- **`minmax()`関数**: この関数はトラックの最小値と最大値のサイズを設定するために使います。

```css
.container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-auto-rows: minmax(150px, auto);
}
```

- **ラインベースの配置**: すべてのグリッドにはラインがあります。アイテムがどのラインから始まるかを指定するには、`grid-column-start`と`grid-row-start`プロパティを使います。アイテムがどのラインで終わるかを指定するには、`grid-column-end`と`grid-row-end`プロパティを使います。代わりに`grid-column`または`grid-row`省略形プロパティを使うこともできます。

以下は、`grid-column`プロパティを使って要素をすべての列にまたがらせる例です。

```css
.element {
  grid-column: 1 / -1;
}
```

- **`grid-template-areas`**: このプロパティはグリッド上で配置するアイテムに名前を付けるために使います。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="header">Header</div>
  <div class="sidebar">Sidebar</div>
  <div class="main">Main Content</div>
  <div class="footer">Footer</div>
</div>

```

```css
.container {
  display: grid;
  grid-template-columns: 200px 1fr; 
  grid-template-rows: auto 1fr auto; 
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer"; 
  gap: 20px; 
}

.header {
  grid-area: header; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}

.sidebar {
  grid-area: sidebar;
  background-color: #f4f4f4;
  padding: 10px;
}

.main {
  grid-area: main; 
  background-color: #e0e0e0;
  padding: 10px;
}

.footer {
  grid-area: footer; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}
```

:::

## CSSのデバッグ

- **開発者ツール（DevTools）**: 開発者ツールを使うと、CSSをリアルタイムで検査・変更できます。スタイルペインには選択した要素に適用されているすべてのCSSルールが表示され、継承されたスタイルも含まれます。個別のプロパティをオン・オフしたり、値を編集したり、新しいルールをブラウザ内で直接追加したりできます。この即時フィードバックは、ソースコードを変更せずにさまざまなスタイルを試すのに非常に役立ちます。
- **CSSバリデーター**: バリデーターは公式仕様に対してCSSをチェックし、エラーや警告を報告します。よく使われるバリデーターにはW3C CSSバリデーターがあります。
- **レスポンシブデザインのデバッグ**: 開発者ツールには、さまざまな画面サイズやデバイスでサイトの見え方をシミュレートするオプションがあります。これにより、ブレークポイントの問題や異なるビューポートサイズでうまくスケールしないスタイルを特定できます。
