#### --distractors--

O loop `for...in` itera sobre os valores das propriedades, enquanto o loop `for...of` itera sobre os nomes das propriedades (chaves).

---

Loops `for...in` criam novas propriedades, enquanto loops `for...of` modificam propriedades existentes.

---

Os loops `for...in` só podem ser usados com strings, enquanto os loops `for...of` podem ser usados com strings e números.

#### --answer--

O loop `for...in` itera sobre nomes de propriedades (chaves), enquanto o loop `for...of` itera sobre valores de propriedades.
