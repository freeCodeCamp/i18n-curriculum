# --interactive--

## تنسيق القوائم

- **خاصية `line-height`**: تُستخدم هذه الخاصية لإنشاء مساحة بين أسطر النص. القيم المقبولة لـ `line-height` تشمل الكلمة المفتاحية `normal`، الأرقام، النسب المئوية، ووحدات الطول مثل وحدة `em`.
- **خاصية `list-style-type`**: تُستخدم هذه الخاصية لتحديد علامة العنصر في القائمة. القيم المقبولة يمكن أن تشمل `circle`، `disc`، أو `decimal`.

إليك مثالًا يستخدم `list-style-type` لتغيير نمط الرصاصة:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
ul {
  list-style-type: square;
}
```

:::

- **خاصية `list-style-position`**: تُستخدم هذه الخاصية لتعيين موضع علامة القائمة. القيم المقبولة الوحيدة هي `inside` و `outside`.

إليك مثالًا يوضح الفرق بين `inside` و `outside`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="inside-list">
  <li>Item with inside position</li>
  <li>Item with inside position</li>
</ul>
<ul class="outside-list">
  <li>Item with outside position</li>
  <li>Item with outside position</li>
</ul>
```

```css
.inside-list {
  list-style-position: inside;
}

.outside-list {
  list-style-position: outside;
}
```

:::

- **خاصية `list-style-image`**: تُستخدم هذه الخاصية لاستخدام صورة كعلامة عنصر القائمة. حالة الاستخدام الشائعة هي استخدام دالة `url` مع قيمة تشير إلى موقع صورة صالح.

## تباعد عناصر القائمة باستخدام `margin`

- بالإضافة إلى `line-height`، يمكن استخدام الهوامش في CSS لتحسين التباعد وقابلية قراءة عناصر القائمة.
- تخلق الهوامش مساحة خارج كل عنصر `li`، مما يسمح بالتحكم في الفجوة بين عناصر القائمة.
- تُستخدم `margin-bottom` لإنشاء مساحة أسفل كل عنصر قائمة. على سبيل المثال، `margin-bottom: 10px;` ستُنشئ فجوة بعرض 10 بكسل أسفل كل عنصر قائمة.

إليك مثالًا يستخدم `margin-bottom` لتباعد عناصر القائمة:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="list">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
.list li {
  margin-bottom: 20px;
}
```

:::

## تنسيق الروابط

- **`pseudo-class`**: هذه كلمة مفتاحية تُضاف إلى المحدد لتسمح لك باختيار العناصر بناءً على حالة معينة. الحالات الشائعة تشمل حالات `:hover`، و`:visited`، و`:focus`.
- **`:link pseudo-class`**: تُستخدم هذه الفئة المزيفة لتنسيق الروابط التي لم يزرها المستخدم بعد.

إليك مثالًا يستخدم فئة مزيفة `:link`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:link {
  color: red;
}
```

:::

- **`:visited pseudo-class`**: تُستخدم هذه الفئة المزيفة لتنسيق الروابط التي زارها المستخدم بالفعل.
- **`:hover pseudo-class`**: تُستخدم هذه الفئة المزيفة لتنسيق العناصر التي يحوم المستخدم فوقها بنشاط.

إليك مثالًا يستخدم فئة مزيفة `:hover`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Hover over me!</a>
```

```css
a:hover {
  color: green;
  text-decoration: underline;
}
```

:::

- **`:focus pseudo-class`**: تُستخدم هذه الفئة المزيفة لتنسيق عنصر عندما يحصل على التركيز. من الأمثلة على ذلك عناصر `input` أو `select` حيث ينقر المستخدم أو ينتقل عبر التبويب إلى العنصر ليحصل على التركيز.

إليك مثالًا يستخدم فئة مزيفة `:focus`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:focus {
  outline: 2px solid orange;
}
```

:::

- **`:active pseudo-class`**: تُستخدم هذه الفئة المزيفة لتنسيق عنصر تم تفعيله بواسطة المستخدم. مثال شائع هو عندما ينقر المستخدم على زر.

إليك مثالًا يستخدم فئة مزيفة `:active`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Click me!</a>
```

```css
a:active {
  color: pink;
}
```

:::

## العمل مع الخلفيات والحدود

- **خاصية `background-size`**: تُستخدم هذه الخاصية لتعيين حجم الخلفية لعنصر. بعض القيم الشائعة تشمل `cover` لجعل صورة الخلفية تغطي العنصر بالكامل و `contain` لجعل صورة الخلفية تناسب داخل العنصر.

إليك مثالًا يستخدم `background-size: cover`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

إليك مثالًا يستخدم `background-size: contain`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **خاصية `background-repeat`**: تُستخدم هذه الخاصية لتحديد كيفية تكرار صور الخلفية على المحورين الأفقي والعمودي. القيمة الافتراضية لـ `background-repeat` هي `repeat`، مما يعني أن الصورة ستتكرر أفقيًا وعموديًا. يمكنك أيضًا تحديد عدم التكرار باستخدام القيمة `no-repeat`.

إليك مثالًا يستخدم `background-repeat: no-repeat`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **خاصية `background-position`**: تُستخدم هذه الخاصية لتحديد موضع صورة الخلفية. يمكن تعيينها لطول محدد، نسبة مئوية، أو قيم كلمات مفتاحية مثل `top`، `bottom`، `left`، `right`، و `center`.

إليك مثالًا يستخدم `background-position`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

- **خاصية `background-attachment`**: تُستخدم هذه الخاصية لتحديد ما إذا كانت صورة الخلفية يجب أن تتحرك مع المحتوى أو تبقى ثابتة في مكانها. القيم الرئيسية هي `scroll` (الافتراضية)، حيث تتحرك صورة الخلفية مع المحتوى، و `fixed`، حيث تبقى صورة الخلفية في نفس الموضع على الشاشة.
- **خاصية `background-image`**: تُستخدم هذه الخاصية لتعيين صورة الخلفية لعنصر. يمكنك تعيين صور خلفية متعددة في نفس الوقت واستخدام دوال `url`، `radial-gradient`، أو `linear-gradient` كقيم.

إليك مثالًا يستخدم `background-image`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **خاصية `background`**: هذه خاصية مختصرة لتعيين جميع خصائص الخلفية في إعلان واحد. إليك مثالًا لتعيين صورة الخلفية وجعلها لا تتكرر: `background: no-repeat url("example-url-goes-here");`

إليك مثالًا يستخدم الخاصية المختصرة `background`:

:::interactive_editor

```html
<style>
  body {
    background: center top no-repeat url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **تباين جيد لألوان الخلفية والمقدمة**: من المهم التأكد من أن ألوان الخلفية والمقدمة تتمتع بنسبة تباين جيدة لجعل النص قابلًا للقراءة. توصي إرشادات إمكانية الوصول لمحتوى الويب (WCAG) بنسبة تباين لا تقل عن 4.5:1 للنص العادي و3:1 للنص الكبير.

## الحدود

- **خاصية `border-top`**: تُستخدم هذه الخاصية لتعيين أنماط الحد العلوي لعنصر. `border-top: 3px solid blue;` يحدد حدًا أزرق صلب بعرض 3 بكسل على الجانب العلوي للعنصر.
- **خاصية `border-right`**: تُستخدم هذه الخاصية لتعيين أنماط الحد الأيمن لعنصر. `border-right: 2px solid red;` يحدد حدًا أحمر صلب بعرض 2 بكسل على الجانب الأيمن للعنصر.
- **خاصية `border-bottom`**: تُستخدم هذه الخاصية لتعيين أنماط الحد السفلي لعنصر. `border-bottom: 1px dashed green;` يحدد حدًا أخضر متقطع بعرض 1 بكسل على الجانب السفلي للعنصر.
- **خاصية `border-left`**: تُستخدم هذه الخاصية لتعيين أنماط الحد الأيسر لعنصر. `border-left: 4px dotted orange;` يحدد حدًا برتقالي منقط بعرض 4 بكسل على الجانب الأيسر للعنصر.

إليك مثالًا يستخدم خصائص الحدود الفردية:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="bordered-box">Box with different borders</div>
```

```css
.bordered-box {
  border-top: 3px solid blue;
  border-right: 2px solid red;
  border-bottom: 1px dashed green;
  border-left: 4px dotted orange;
  padding: 20px;
}
```

:::

- **خاصية `border`**: هذه خاصية مختصرة لتعيين عرض، نمط، ولون حد العنصر. `border: 1px solid black;` يحدد حدًا أسود صلب بعرض 1 بكسل.

إليك مثالًا يستخدم الخاصية المختصرة `border`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

- **خاصية `border-radius`**: تُستخدم هذه الخاصية لإنشاء زوايا مستديرة لحد العنصر.

إليك مثالًا يستخدم `border-radius`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

- **خاصية `border-style`**: تُستخدم هذه الخاصية لتعيين نمط حد العنصر. بعض القيم المقبولة تشمل `solid`، `dashed`، `dotted`، و `double`.

إليك مثالًا يستخدم قيمًا مختلفة لـ `border-style`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="solid-border">Solid border</div>
<div class="dashed-border">Dashed border</div>
<div class="dotted-border">Dotted border</div>
<div class="double-border">Double border</div>
```

```css
.solid-border {
  border: 3px solid blue;
  margin-bottom: 10px;
  padding: 10px;
}

.dashed-border {
  border: 3px dashed red;
  margin-bottom: 10px;
  padding: 10px;
}

.dotted-border {
  border: 3px dotted green;
  padding: 10px;
}

.double-border {
  border: 3px double blueviolet;
  padding: 10px;
}
```

:::

## التدرجات اللونية

- **دالة `linear-gradient()`**: هذه الدالة في CSS تُستخدم لإنشاء انتقال بين ألوان متعددة على خط مستقيم.

إليك مثالًا يستخدم `linear-gradient()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

- **دالة `radial-gradient()`**: هذه الدالة في CSS تُنشئ صورة تشع من نقطة معينة، مثل دائرة أو بيضاوي، وتنتقل تدريجيًا بين ألوان متعددة.

إليك مثالًا يستخدم `radial-gradient()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```
