# --description--

You can use the `hr` tag to add a horizontal line across the width of its containing element. This can be used to define a change in topic or to visually separate groups of content.

# --instructions--

Aggiungi un tag `hr` sotto il tag `h4` che contiene il titolo della card.

**Nota:** In HTML, `hr` Ã¨ un tag a chiusura automatica, e quindi non ha bisogno di un tag di chiusura separato.

# --hints--

Il tuo codice dovrebbe aggiungere un tag `hr` al markup.

```js
assert.lengthOf(document.querySelectorAll('hr'),1);
```

Il tag `hr` dovrebbe stare tra il titolo e il paragrafo.

```js
assert.match(code,/<\/h4>\s*?<hr(>|\s*?\/>)\s*?<p>/gi);
```
