# --description--

Продовжуючи з елементами `meta`, визначення `viewport` повідомляє браузеру, як відтворити сторінку. Додавання такого визначення покращує візуальну доступність на мобільних пристроях і підвищує _SEO_ (оптимізацію для пошукових систем).

Додайте визначення `viewport` з атрибутом `content`, який описує `width` та `initial-scale` сторінки.

# --hints--

Ви маєте створити ще один елемент `meta` у `head`.

```js
assert.equal(document.querySelectorAll('head > meta')?.length, 2);
```

Ви маєте надати `meta` атрибут `name` зі значенням `viewport`.

```js
assert.equal(document.querySelectorAll('head > meta[name="viewport"]')?.length, 1);
```

Ви маєте надати `meta` атрибут `content` зі значенням `width=device-width, initial-scale=1`.

```js
assert.equal(document.querySelectorAll('head > meta[content="width=device-width, initial-scale=1.0"]')?.length || document.querySelectorAll('head > meta[content="width=device-width, initial-scale=1"]')?.length, 1);
```
