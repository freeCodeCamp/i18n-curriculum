# --interactive--

## Arbeiten mit verschiedenen Attributselektoren und Links

- **Definition**: Der `attribute`-Selektor ermöglicht es Ihnen, HTML-Elemente basierend auf ihren Attributen wie den `href`- oder `title`-Attributen anzusprechen.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Link with href</a>
<a>No href</a>
```

```css
a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

- **`title`-Attribut**: Dieses Attribut liefert zusätzliche Informationen über ein Element. So können Sie Links mit dem `title`-Attribut ansprechen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Tooltip">Link with title</a>
<a href="#">Normal link</a>
```

```css
a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

- **Kombinieren Sie Selektoren, um Links zu finden, die sowohl `href`- als auch `title`-Attribute besitzen**: Kombiniert mehrere Attributselektoren.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Info">Href + Title</a>
<a href="#">Only href</a>
```

```css
a[href][title] {
  color: green;
}
```

:::

- **Ein einzelnes Wort innerhalb einer durch Leerzeichen getrennten Liste ansprechen**: Richtet sich an Links, die ein bestimmtes Klassenwort besitzen.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a class="link primary">Primary link</a>
<a class="link secondary">Secondary link</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

- **Werte ansprechen, die mit einem bestimmten Präfix beginnen**: Richtet sich an Links, die mit `https://` beginnen.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">HTTPS link</a>
<a href="http://www.freecodecamp.org">HTTP link</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

- **Werte ansprechen, die mit einem bestimmten Suffix enden**: Richtet sich an Links, die mit `.jpg` enden.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="photo.jpg">Image link</a>
<a href="index.html">HTML link</a>
```

```css
a[href$=".jpg"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

- **Werte ansprechen, die eine Teilzeichenfolge irgendwo enthalten**: Richtet sich an Links, die `https` irgendwo im Wert enthalten.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Secure link</a>
<a href="page.html">Local link</a>
```

```css
a[href*="https"] {
  color: teal;
}
```

:::

- **Zusammenfassung**: Diese Muster erleichtern es, Links konsistent basierend auf ihren Attributen und Werten zu gestalten.

## Elemente mit den `lang` und `data-lang` Attributen ansprechen

- **`lang`-Attribut**: Dieses Attribut wird in HTML verwendet, um die Sprache des Inhalts innerhalb eines Elements anzugeben. Sie möchten Elemente möglicherweise unterschiedlich gestalten, je nachdem, in welcher Sprache sie geschrieben sind, besonders auf einer mehrsprachigen Website.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">English text</p>
<p lang="fr">French text</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

- **`data-lang`-Attribut**: Benutzerdefinierte Datenattribute wie das `data-lang`-Attribut werden häufig verwendet, um zusätzliche Informationen in Elementen zu speichern, etwa um die Sprache anzugeben, die innerhalb eines bestimmten Textabschnitts verwendet wird. So können Sie Elemente basierend auf dem `data-lang`-Attribut gestalten:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div data-lang="fr">French content</div>
<div data-lang="en">English content</div>
```

```css
div[data-lang="fr"] {
  color: blue;
}
```

:::

## Arbeiten mit Attributselektoren, geordneten Listen und dem `type` Attribut

- **`type` Attribut**: Beim Arbeiten mit geordneten Listen in HTML ermöglicht das `type` Attribut, den Stil der Nummerierung festzulegen, z. B. numerisch, alphabetisch oder römisch.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Alpha</li>
  <li>Beta</li>
</ol>

<ol type="i">
  <li>One</li>
  <li>Two</li>
</ol>
```

```css
/* Example targeting uppercase alphabetical numbering */
ol[type="A"] {
  color: purple;
  font-weight: bold;
}

/* Example targeting lowercase Roman numerals */
ol[type="i"] {
  color: green;
}
```
