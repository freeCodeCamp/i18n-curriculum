# --description--

Los conjuntos son una de las estructuras de datos integradas de Python. Una de las características principales de los conjuntos es que no almacenan valores duplicados. Si intentas agregar un valor duplicado a un conjunto, solo se almacenará uno de ellos.

Los conjuntos son mutables y no ordenados, lo que significa que sus elementos no se almacenan en un orden específico, por lo que no puedes usar índices o claves para acceder a ellos. Solo pueden contener valores de tipos de datos inmutables como números, cadenas y tuplas. Además, soportan operaciones matemáticas de conjuntos, incluyendo unión, intersección, diferencia y diferencia simétrica.

Para definir un conjunto, solo necesitas escribir sus elementos dentro de llaves y separarlos con comas. Este es un ejemplo de un conjunto de números:

```python
my_set = {1, 2, 3, 4, 5} 
```

Una particularidad de trabajar con conjuntos es que, si alguna vez necesitas definir un conjunto vacío, debes usar la función `set()`. Si solo escribes llaves vacías, como `{}`, Python creará automáticamente un diccionario.

```python
set() # Set
{}    # Dictionary
```

Puedes agregar un elemento a un conjunto con el método `.add()`, y pasar el nuevo elemento como argumento:

```python
my_set.add(6)
```

En nuestro ejemplo, el nuevo conjunto sería:

```python
{1, 2, 3, 4, 5, 6}
```

Si intentas agregar un elemento que ya está en el conjunto, solo se mantendrá uno. En este caso, ya tenemos el número 5 en el conjunto:

```python
my_set.add(5)
```

Entonces el conjunto no cambiará:

```python
{1, 2, 3, 4, 5, 6}
```

Para eliminar un elemento del conjunto, tienes dos opciones. Puedes usar el método `.remove()` o el método `.discard()`, y pasar como argumento el elemento que quieres eliminar.

El método `.remove()` generará un `KeyError` si el elemento no se encuentra, mientras que el método `.discard()` no lo hará:

```python
my_set.remove(4)
my_set.discard(4)
```

El método `.clear()` elimina todos los elementos del conjunto:

```python
my_set.clear()
```

Los conjuntos de Python también tienen métodos potentes que realizan operaciones matemáticas comunes de conjuntos.

Los métodos `.issubset()` y `.issuperset()` verifican si un conjunto es un subconjunto o un superconjunto de otro conjunto, respectivamente.

Aquí, estamos verificando si `your_set` es un subconjunto de `my_set`, lo cual es `False` porque no todos los elementos de `your_set` están en `my_set`.

También estamos verificando si `my_set` es un superconjunto de `your_set`. Esto también es `False` porque `my_set` no tiene todos los elementos de `your_set`:

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

print(your_set.issubset(my_set)) # False
print(my_set.issuperset(your_set)) # False
```

El método `.isdisjoint()` verifica si dos conjuntos son disjuntos, lo que significa que no tienen elementos en común. En este caso, eso es `False` porque `my_set` y `your_set` sí tienen elementos en común: 2, 3 y 4:

```python
print(my_set.isdisjoint(your_set)) # False
```

El operador de unión `|` devuelve un nuevo conjunto con todos los elementos de ambos conjuntos:

```python
my_set | your_set # {1, 2, 3, 4, 5, 6}
```

El operador de intersección `&` devuelve un nuevo conjunto con solo los elementos que los conjuntos tienen en común:

```python
my_set & your_set # {2, 3, 4}
```

El operador de diferencia `-` devuelve un nuevo conjunto con los elementos del primer conjunto que no están en los otros conjuntos. En este ejemplo, los números 1 y 5 están en `my_set` pero NO en `your_set`:

```python
my_set - your_set # {1, 5}
```

El operador de diferencia simétrica `^` devuelve un nuevo conjunto con los elementos que están en el primer o en el segundo conjunto, pero no en ambos. En este caso, 1 y 5 están en `my_set` pero no en `your_set`, por lo que se incluyen. Y el número 6 está en `your_set` pero no en `my_set`, por lo que también se incluye:

```python
my_set ^ your_set # {1, 5, 6}
```

Cada uno de estos operadores también tiene su operador de asignación compuesta correspondiente si agregas el signo igual junto a él. Estos operadores asignan automáticamente el conjunto resultante al primer conjunto en la expresión:

```python
|= &= -= ^=
```

Por ejemplo, el operador `-=` encuentra la diferencia entre los conjuntos y actualiza el primer conjunto con ese resultado:

```python
my_set -= your_set
```

Después de esto, `my_set` se actualizará a `{1, 5}`:

```python
print(my_set) # {1, 5}
```

Puedes verificar si un elemento está en un conjunto o no con el operador `in`. Aquí, estamos comprobando si 5 está en `my_set`. El resultado será un valor booleano `True` o `False`:

```python
print(5 in my_set)
```

Y esos son los fundamentos de los conjuntos. Son muy útiles cuando no necesitas almacenar los valores en un orden específico, y cuando solo necesitas almacenar valores únicos.
