# --description--

طريقة أخرى لاستخدام المحدِّدات هي تسلسلها كقائمة بدون أي فاصل. لنفترض أن لديك HTML التالي:

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection preview">This is where a preview for a post might go.</p>
</div>
```

لديك عنصران يحملان صنف `subsection` لهما بعض الأنماط الفريدة، لكن ماذا لو أردت تطبيق قاعدة منفصلة فقط على العنصر الذي يحمل أيضًا `header` كصنف ثاني؟ يمكنك تسلسل كلا المحدِّدين `class` معًا في CSS بهذه الطريقة:

```css
.subsection.header {
  color: red;
}
```

ما يفعله `.subsection.header` هو أنه يحدد أي عنصر يحمل صنفي `subsection` و`header` معًا. لاحظ أنه لا توجد أي مسافة بين محدِّدي الصنف `.subsection` و`.header` (`class`). تعمل هذه الصيغة أساسًا لتسلسل أي مجموعة من المحدِّدات، باستثناء تسلسل أكثر من محدِّد نوع واحد.

يمكن أيضًا استخدام هذا لتسلسل صنف وID، كما هو موضح أدناه:

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection" id="preview">This is where a preview for a post might go.</p>
</div>
```

يمكنك أخذ العنصرين أعلاه ودمجهما معًا باستخدام التالي:

```css
.subsection.header {
  color: red;
}

.subsection#preview {
  color: blue;
}
```

بشكل عام، لا يمكنك تسلسل أكثر من محدِّد نوع واحد لأن العنصر لا يمكن أن يكون نوعين مختلفين في نفس الوقت. على سبيل المثال، تسلسل محدِّدين نوع مثل `div` و `p` سيعطينا المحدِّد `divp`، وهذا لن يعمل لأن المحدِّد سيحاول إيجاد عنصر `<divp>` حرفيًا، وهذا غير موجود.
