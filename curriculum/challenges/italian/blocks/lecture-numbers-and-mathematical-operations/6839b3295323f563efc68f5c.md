# --description--

L'assegnazione aumentata combina un'operazione binaria con un'assegnazione in un unico passaggio. Prende una variabile, applica un'operazione su di essa con un altro valore e archivia il risultato nella stessa variabile.

Se conosci un linguaggio come JavaScript, probabilmente hai sentito parlare dell'operatore di assegnazione con addizione (`+=`) o dell'assegnazione con sottrazione (`-=`), e altri ancora. Esistono anche in Python. L'unica differenza è che qui si chiamano **assegnazioni aumentate**.

La sintassi base di un'assegnazione aumentata è questa:

```python
variable <operator>= value
```

Che è un modo più efficiente di fare questo:

```python
variable = variable <operator> value
```

Per esempio, ecco un esempio di uso dell'assegnazione aumentata per aggiungere `5` a una variabile esistente:

```python
my_var = 10
my_var += 5

print(my_var) # 15
```

E qui c'è la stessa cosa, ma senza assegnazione aumentata:

```python
my_var = 10
my_var = my_var + 5

print(my_var) # 15
```

Il vantaggio dell'assegnazione aumentata è che offre un modo conciso e leggibile per aggiornare il valore di una variabile senza ripetere il nome della variabile. Questo riduce la ridondanza e gli errori potenziali che potrebbero derivare da un errore di battitura o simili.

Ogni operatore può usare un'assegnazione aumentata. Abbiamo visto l'operatore di assegnazione con addizione (`+=`), ora vediamo gli altri.

- L'operatore di assegnazione con sottrazione (`-=`) sottrae l'operando a destra dalla variabile a sinistra e archivia la differenza nella variabile a sinistra:

```python
count = 14
count -= 3

print(count) # 11
```

- L'operatore di assegnazione con moltiplicazione (`*=`) moltiplica la variabile a sinistra per l'operando a destra e archivia il prodotto nella variabile a sinistra:
    

```python
product = 65
product *= 7

print(product) # 455
```

- L'operatore di assegnazione con divisione (`/=`) divide la variabile a sinistra per quella a destra e archivia il risultato nella variabile a sinistra:
    

```python
price = 100
price /= 4

print(price) # 25.0
```

- L'operatore di assegnazione con divisione intera (`//=`) esegue la divisione intera della variabile a sinistra per quella a destra e archivia il risultato nella variabile a sinistra:
    

```python
total_pages = 23
total_pages //= 5

print(total_pages) # 4
```

- L'operatore di assegnazione con modulo (`%=`) calcola il resto della variabile a sinistra divisa per quella a destra e lo archivia nella variabile a sinistra:
    

```python
bits = 35
bits %= 2

print(bits) # 1
```

- L'operatore di assegnazione con esponenziazione (`**=`) eleva la variabile a sinistra alla potenza dell'operando a destra e archivia il risultato nella variabile a sinistra:
    

```python
power = 2
power **= 3

print(power) # 8
```

Puoi usare alcuni operatori di assegnazione aumentata anche con le stringhe. Per esempio, l'operatore di assegnazione con addizione rende facile concatenare stringhe:

```python
greet = 'Hello'
greet += ' World'

print(greet) # Hello World
```

E l'operatore di assegnazione con moltiplicazione può essere usato per ripetere una stringa:

```python
greet = 'Hello'
greet *= 3

print(greet) # HelloHelloHello
```

Altre assegnazioni aumentate lanciano un `TypeError` se usate con stringhe:

```python
greet = 'Hello'
greet -= ' World'

print(greet) # TypeError: unsupported operand type(s) for -=: 'str' and 'str'


greet = 'Hello'
greet /= 'World'

print(greet) # TypeError: unsupported operand type(s) for /=: 'str' and 'str' 
```

Se ti stai chiedendo se gli operatori di incremento e decremento (`++` e `--`) funzionano in Python, la risposta è no. Questo perché Python evita volutamente scorciatoie di incremento e decremento in stile C per mantenere il linguaggio chiaro e esplicito.

Invece di `x++`, puoi semplicemente scrivere `x += 1`, che rende ovvio che stai incrementando il valore di `x` di `1`.

Scrivere `++x` in Python applica semplicemente l'operatore unario più due volte, senza incrementare nulla:

```python
my_var = 5

print(+my_var)   # 5
print(++my_var)  # 5
print(+++my_var) # 5

my_var += 1

print(my_var) # 6
```
