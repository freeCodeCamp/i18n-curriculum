# --description--

لنلتقِ بميزات مشغلات JavaScript التي تتجاوز الحسابات المدرسية.

عادةً، المشغل الجمع `+` يجمع الأعداد.

لكن، إذا طُبّق المشغل الثنائي `+` على سلاسل نصية، فإنه يدمجها (يربطها):

```js
let s = "my" + "string";
console.log(s); // mystring
```

لاحظ أنه إذا كان أي من المعاملين سلسلة نصية، فإن الآخر يتحول إلى سلسلة نصية أيضًا.

على سبيل المثال:

```js
console.log(1 + '2'); // '12'
console.log('1' + 2); // '12'
```

انظر، لا يهم إذا كان المعامل الأول سلسلة نصية أو الثاني.

إليك مثالًا أكثر تعقيدًا:

```js
console.log(2 + 2 + '1') // "41" and not "221"
```

هنا، تعمل المشغلات واحدة تلو الأخرى. المشغل `+` الأول يجمع عددين، فيُرجع `4`، ثم المشغل `+` التالي يضيف السلسلة النصية `1` إليه، فيكون الأمر مثل `4 + '1' = '41'`.

```js
console.log('1' + 2 + 2) // "122" and not "14"
```

هنا، المشغل `+` الأول يربط السلسلة النصية والعدد، فيُرجع `'12'`، ثم المشغل `+` التالي يضيف `2` إليه، فيكون الأمر مثل `'12' + 2 = '122'`.

المشغل الثنائي `+` هو المشغل الوحيد الذي يدعم السلاسل النصية بهذه الطريقة. المشغلات الحسابية الأخرى تعمل فقط مع الأعداد وتحول دائمًا معاملاتها إلى أعداد.

إليك العرض التوضيحي للطرح والقسمة:

```js
console.log( 6 - '2' ); // 4, converts '2' to a number
console.log( '6' / '2' ); // 3, converts both operands to numbers
```
