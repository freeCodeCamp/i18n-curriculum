# --explanation--

Reservar significa apartar o guardar algo, especialmente dinero o tiempo, para un propósito futuro específico. Por ejemplo:

Es inteligente reservar dinero para gastos imprevistos. — Es bueno tener algo de dinero extra para sorpresas.

Una `contingency` es un plan de respaldo o de emergencia, especialmente para situaciones que podrían no ocurrir pero que son posibles. Por ejemplo:

`It's important to have contingency measures during testing.` — Necesitas pasos o planes adicionales por si las cosas no salen como se espera durante las pruebas.
