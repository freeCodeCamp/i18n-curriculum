# --description--

莫比烏斯函式（程式）、記作 $μ(n)$，定義為：

- $μ(n) = (-1)^{ω(n)}$，如果 $n$ 是無平方因數的（其中 $ω(n)$ 是 $n$ 的不同質數因數的數量）
- 如果 $n$ 不是平方自由數，則 $μ(n) = 0$。

令 $P(a, b)$ 為區間 $[a, b]$ 中滿足 $μ(n) = 1$ 的整數 $n$ 的數量。

令 $N(a, b)$ 為區間 $[a, b]$ 中滿足 $μ(n) = -1$ 的整數 $n$ 的數量。

例如，$P(2, 10) = 2$ 和 $N(2, 10) = 4$。

令 $C(n)$ 為整數配對 $(a, b)$ 的數量，使得：

- $1 ≤ a ≤ b ≤ n$，
- $99 \times N(a, b) ≤ 100 \times P(a, b)$，且
- $99 \times P(a, b) ≤ 100 \times N(a, b)$。

例如，$C(10) = 13$、$C(500) = 16\\,676$ 和 $C(10\\,000) = 20\\,155\\,319$。

尋找 $C(20\,000\,000)$。
