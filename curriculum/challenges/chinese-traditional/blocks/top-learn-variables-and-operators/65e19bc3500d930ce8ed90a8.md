# --description--

數字是程式設計邏輯的建造區塊！事實上，很難想像有任何有用的程式任務不涉及至少一點基本數學……所以了解數字的運作方式顯然非常重要。幸運的是，這也相當簡單明瞭。

| Operator | Description                  | Example            |
|----------|------------------------------|--------------------|
| +        | Addition                     | 5 + 3 = 8          |
| -        | Subtraction                  | 5 - 3 = 2          |
| *        | Multiplication               | 5 * 3 = 15         |
| **       | Exponentiation               | 5 ** 3 = 125       |
| /        | Division                     | 6 / 3 = 2          |
| %        | Modulus                      | 5 % 3 = 2          |
