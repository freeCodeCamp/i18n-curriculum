# --description--

As palavras `super` e `pretty` são usadas para enfatizar o grau de algo.

`Super` significa muito ou extremamente.Por exemplo, `This book is super interesting` significa que o livro é muito interessante. 

`Pretty` é usado para significar "bastante" ou "razoavelmente".É uma forma de dizer que algo é em grande medida, mas não extremamente.Por exemplo, se você disser `It's pretty cold outside`, você quer dizer que está bastante frio, mas não extremamente frio.
