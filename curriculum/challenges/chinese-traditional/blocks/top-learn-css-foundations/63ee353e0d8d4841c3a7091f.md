# --description--

好的，你到目前為止已經講解了相當多內容。現在唯一剩下的是說明如何將所有這些 CSS 添加到你的 HTML。共有三種方法可以做到這一點。

外部 CSS 是你最常遇到的行為，它涉及為 CSS 建立一個獨立的檔案，並使用 `<link>` 元素（這是一個空元素，不需要閉合標記）將其連結在 HTML 的開啟和關閉 `<head>` 頁籤內：

首先，你在 HTML 檔案的開啟和關閉 `<head>` 標記之間添加一個空元素 `<link>` 標記。`href` 屬性是 CSS 檔案的位置，可以是絕對 URL，或者你將使用的相對於 HTML 檔案位置的 URL。在上述範例中，你假設兩個檔案位於相同的目錄中。`rel` 屬性是必需的，它指定 HTML 檔案與連結檔案之間的關係。

然後在新創建的 `styles.css` 檔案中，你有選擇器（`div` 和 `p`），接著是一對開啟和關閉的大括號，這會生成一個「宣告區塊」。最後，你將任何宣告放在宣告區塊內。`color: white;` 是一個宣告，其中 `color` 是屬性，`white` 是值，而 `background-color: black;` 是另一個宣告。

關於檔案名稱的說明：`styles.css` 只是你在這裡選擇的檔案名稱。只要檔案型別是 `.css`，你可以將檔案命名為任何你想要的名稱，不過最常用的是 “style” 或 “styles”。

這個方法的幾個優點是：

1. 它將你的 HTML 和 CSS 分開，這使得 HTML 檔案更小且看起來更整潔。
2. 你只需要在一個地方編輯 CSS，這對於擁有許多頁面且都共用相似樣式的網站特別方便。
