# --interactive--

JavaScript에서 문자열을 다룰 때, 제목에 모든 글자를 대문자로 바꾸거나 통일성을 위해 소문자로 변환하는 등 텍스트의 대소문자를 조정해야 하는 경우가 많습니다.

다행히 JavaScript는 `toUpperCase()`와 `toLowerCase()`라는 두 가지 내장 메서드로 이를 쉽게 처리할 수 있습니다.

`toUpperCase()` 메서드는 모든 문자를 대문자로 변환하여 대문자만 포함된 새 문자열을 반환합니다. 이는 텍스트를 강조하거나 문자열 형식을 일관되게 만들 때 유용합니다.

예제를 살펴보겠습니다:

:::interactive_editor

```js
let greeting = "Hello, World!";
let uppercaseGreeting = greeting.toUpperCase();
console.log(uppercaseGreeting);  // "HELLO, WORLD!"
```

:::

이 코드에서 `toUpperCase()` 메서드는 문자열 전체를 대문자로 변환합니다.

원본 문자열은 `toUpperCase()`가 새 문자열을 반환하고 원본을 수정하지 않기 때문에 변경되지 않습니다.

반대로 `toLowerCase()` 메서드는 문자열의 모든 문자를 소문자로 변환합니다. 이는 사용자 입력을 표준화하거나 대소문자를 구분하지 않는 비교를 할 때 유용합니다.

예제를 살펴보겠습니다:

:::interactive_editor

```js
let shout = "I AM LEARNING JAVASCRIPT!";
let lowercaseShout = shout.toLowerCase();
console.log(lowercaseShout);  // "i am learning javascript!"
```

:::

`toLowerCase()` 메서드는 모든 문자를 소문자로 변환하여 문자열을 덜 강하게 만들면서도 원본 문자열은 변경하지 않습니다.

요약하자면, JavaScript의 `toUpperCase()`와 `toLowerCase()` 메서드는 문자열을 모두 대문자 또는 소문자로 변환하는 강력한 도구입니다.

이 메서드들은 텍스트 입력을 표준화하고, 대소문자를 구분하지 않는 비교를 하며, 디자인 일관성을 유지하는 데 특히 유용합니다.

이 간단하지만 효과적인 메서드들을 사용하면 텍스트 조작을 보다 통제되고 예측 가능하게 처리할 수 있습니다.
