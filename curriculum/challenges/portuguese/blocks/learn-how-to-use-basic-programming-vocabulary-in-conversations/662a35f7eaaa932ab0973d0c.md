# --description--

Aprenda `variables`, `data types` e `conditional statements` em programação.

Uma `variable` é como uma caixa onde você pode armazenar informações.Estas informações podem ser números, texto ou outros tipos de dados.Você pode mudar o que está na caixa, por isso é chamado de `variable` – porque pode variar.Por exemplo, em `x = 5`, `x` mantém temporariamente o valor 5.

Um `data type` é uma classificação que especifica o tipo de dado que uma `variable` pode armazenar.Tipos comuns de `data types` incluem inteiros (números inteiros), floats (números com decimal), strings (texto) e booleans (verdadeiro ou falso).Por exemplo, se `age = 21`, o `data type` é um inteiro.

Uma `conditional statement` é um comando que diz ao computador para fazer algo somente se uma condição específica for verdadeira.É como tomar uma decisão na vida real com base em uma determinada situação.Instruções `conditional statements` comuns são declarações `if-else`.Por exemplo, `if (temperature > 20) {console.log("It's warm outside!");}` é uma `conditional statement` que executa `console.log("It's warm outside!")` se a temperatura estiver acima de 20 graus.
