# --description--

배운 대로 `repetir`는 무언가를 다시 말하는 것을 의미합니다. `lo` 대명사를 추가하면 이미 언급된 것을 가리키는 것입니다. 이 대명사는 특정한 것(단어, 숫자, 이메일 주소 등)을 대신합니다. 그래서:

- `Repetir` = 반복하다.

- `Repetirlo` = 그것을 반복하다.

`Repetirlo`는 "그것"이 문맥상 이미 알려져 있을 때 사용합니다. 같은 패턴이 다른 동사에도 적용됩니다:

- `¿Puedes escribirlo?` - 그것을 쓸 수 있나요?  

- `¿Puedes decirlo otra vez?` - 그것을 다시 말할 수 있나요?

# --explanation--

`Lo`는 대화에서 이미 알려진 것을 대신하는 직접 목적어 대명사입니다. 부정사에 붙으면 한 단어가 됩니다:

- `repetir` + `lo` → `repetirlo`  - 그것을 반복하다.

- `escribir` + `lo` → `escribirlo` - 그것을 쓰다.

- `decir` + `lo` → `decirlo` - 그것을 말하다.

이 구조는 듣는 사람이 "그것"이 무엇인지 이미 이해하고 있을 때 흔히 사용됩니다.
