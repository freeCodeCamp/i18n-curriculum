# --explanation--

`He shared the layout of the new office.` – これは物の配置計画を意味します。

`Draft` は文書の初期バージョンです。例えば：  

`She wrote the first draft of the proposal.` – まだ編集可能です。

`Send it over` は何かを届けることを意味します。例えば：  

`I'll send it over once it's ready.` – ファイルやメッセージを共有することです。

`To` は動作の対象を示します。例えば：  

`She gave the file to her manager.` – 誰がそれを受け取るかを示します。
