# --instructions--

أنشئ دالة باسم `calculate()` في `mean_var_std.py` تستخدم Numpy لإخراج المتوسط، التباين، الانحراف المعياري، القيم العظمى، القيم الصغرى، والمجموع للصفوف والأعمدة والعناصر في مصفوفة 3 × 3.

يجب أن يكون مُدخل الدالة قائمة تحتوي على 9 أرقام. تُحوّل الدالة القائمة إلى مصفوفة Numpy بحجم 3 × 3، ثم تُرجع قاموسًا يحتوي المتوسط، التباين، الانحراف المعياري، القيم العظمى، القيم الصغرى، والمجموع على كلا المحورين وللمصفوفة المسطحة.

يجب أن يتبع القاموس المرجع هذا التنسيق:

```py
{
  'mean': [axis1, axis2, flattened],
  'variance': [axis1, axis2, flattened],
  'standard deviation': [axis1, axis2, flattened],
  'max': [axis1, axis2, flattened],
  'min': [axis1, axis2, flattened],
  'sum': [axis1, axis2, flattened]
}
```

إذا تم تمرير قائمة تحتوي على أقل من 9 عناصر إلى الدالة، يجب أن ترفع استثناء `ValueError` مع الرسالة: "List must contain nine numbers." يجب أن تكون القيم في القاموس المرجع قوائم وليست مصفوفات Numpy.

على سبيل المثال، يجب أن تُرجع `calculate([0,1,2,3,4,5,6,7,8])`:

```py
{
  'mean': [[3.0, 4.0, 5.0], [1.0, 4.0, 7.0], 4.0],
  'variance': [[6.0, 6.0, 6.0], [0.6666666666666666, 0.6666666666666666, 0.6666666666666666], 6.666666666666667],
  'standard deviation': [[2.449489742783178, 2.449489742783178, 2.449489742783178], [0.816496580927726, 0.816496580927726, 0.816496580927726], 2.581988897471611],
  'max': [[6, 7, 8], [2, 5, 8], 8],
  'min': [[0, 1, 2], [0, 3, 6], 0],
  'sum': [[9, 12, 15], [3, 12, 21], 36]
}
```
