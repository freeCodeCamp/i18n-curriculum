# --interactive--

プログラムを作るとき、遅延の後に何かを実行したり、一定の間隔で繰り返したりしたいことがよくあります。そんなときに役立つのが `setTimeout()` と `setInterval()` メソッドです。どちらのメソッドも、関数と遅延時間の2つのパラメータを受け取ります。

それぞれのメソッドを詳しく見て、ウェブプロジェクトで使い始めましょう。

`setTimeout()` は指定した時間だけ処理を遅らせることができます。`setTimeout()` メソッドの基本的な構文は次の通りです。

```js
setTimeout(functionToRun, delay);
```

`functionToRun` は指定した遅延後に実行したいコードで、`delay` は関数を実行するまでの待ち時間（ミリ秒）です。例を見てみましょう。

:::interactive_editor

```js
setTimeout(function () {
 console.log("This runs after 3 seconds");
}, 3000);
```

:::

コールバックにアロー関数を使うこともできます。例えば次のようにします。

:::interactive_editor

```js
setTimeout(() => {
 console.log("This runs after 3 seconds"); 
}, 3000);
```

:::

`setInterval()` は関数を一定の間隔で繰り返し実行します。数秒ごとに鳴るアラームのようなものです。`setInterval()` はデータの更新や連続して動くアニメーションのように、何度も繰り返したいときに最適です。

`setTimeout()` と同様に、関数と遅延時間を受け取ります。

```js
setInterval(functionToRun, delay);
```

`functionToRun` は繰り返し実行したいコードで、`delay` はその関数を実行する間隔（ミリ秒）です。例を見てみましょう。

:::interactive_editor

```js
setInterval(() => {
 console.log("This runs every 2 seconds");
}, 2000);

// Result:
// This runs every 2 seconds
// This runs every 2 seconds
// This runs every 2 seconds
// ...
```

:::

`setInterval()` は指定した間隔で関数を繰り返し実行し続けるので、停止したい場合があります。そのためには `clearInterval()` メソッドを使います。

`clearInterval()` は停止したい `setInterval()` のIDを受け取ります。これは間隔を代入した変数かもしれません。間隔を停止する一つの方法は `setTimeout()` の中で行うことで、一定時間後に間隔を停止できます。

:::interactive_editor

```js
const intervalID = setInterval(() => {
 console.log("This will stop after 5 seconds");
}, 1000);

setTimeout(() => {
 clearInterval(intervalID);
}, 5000);
```

:::

`clearTimeout()` メソッドでタイムアウトを停止することもできます。

:::interactive_editor

```js
let timeoutID = setTimeout(() => {
 console.log("This will not run");
}, 5000);

clearTimeout(timeoutID);
```

:::

このコードでは `setTimeout()` はまったく実行されません。よりスマートな方法は、DOMを使ってボタンにクリックイベントリスナーを追加し、タイムアウトを停止することです。

:::interactive_editor

```html
<h1>Cancel Timeout Example</h1>
<button id="cancelButton">Cancel Timeout</button>
<script src="index.js"></script>
```

```js
let timeoutID = setTimeout(() => {
  console.log("This will run if not canceled");
}, 5000);

document
.querySelector("#cancelButton")
.addEventListener("click", () => {
  clearTimeout(timeoutID);
  console.log("Timeout canceled!");
});
```

:::
