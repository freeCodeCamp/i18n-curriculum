# --description--

$N$을 양의 정수라고 하고, $N$을 $k$개의 같은 크기 부분으로 나누어 $r = \frac{N}{k}$가 되도록 하여 $N = r + r + \cdots + r$라고 하겠습니다.

이 부분들의 곱을 $P$라고 하면, $P = r × r × \cdots × r = r^k$입니다.

예를 들어, 11을 다섯 개의 같은 크기 부분으로 나누면 11 = 2.2 + 2.2 + 2.2 + 2.2 + 2.2가 되고, 이때 $P = {2.2}^5 = 51.53632$입니다.

$M(N)$을 주어진 $N$에 대해 $P$의 최댓값 $P_{max}$라고 하겠습니다.

$N = 11$일 때 최댓값은 11을 네 개의 같은 크기 부분으로 나누었을 때 찾아지며, 이때 $P_{max} = {(\frac{11}{4})}^4$가 됩니다. 즉, $M(11) = \frac{14641}{256} = 57.19140625$로 유한소수입니다.

하지만 $N = 8$일 때는 세 개의 같은 크기 부분으로 나누었을 때 최댓값이 나오며, $M(8) = \frac{512}{27}$로 무한소수입니다.

$M(N)$이 무한소수이면 $D(N) = N$, 유한소수이면 $D(N) = -N$이라고 하겠습니다.

예를 들어, $5 ≤ N ≤ 100$에 대해 $\sum D(N)$의 값은 2438입니다.

$5 ≤ N ≤ 10000$에 대해 $\sum D(N)$을 구하세요.
