# --explanation--

`It` wird verwendet, um ein bereits erwähntes Substantiv zu ersetzen und so Wiederholungen im Satz zu vermeiden. Zum Beispiel:

- `I bought a new laptop. It works really well.` – Hier ersetzt `it` `laptop`, um das Wort nicht zu wiederholen.

In `The Live Preview isn't showing in Visual Studio Code. Isn't it supposed to support that feature?` bezieht sich `it` auf `Visual Studio Code`, weil Tom über das Programm und dessen Fähigkeit spricht, eine Funktion zu unterstützen.

`It` ersetzt `Visual Studio Code`, um den Namen des Programms nicht erneut zu nennen.

Indem Sie erkennen, was die Funktion unterstützen soll, können Sie feststellen, dass `it` sich auf das Programm selbst bezieht.
