# --interactive--

## Функції JavaScript

- Функції — це багаторазові блоки коду, які виконують певне завдання.
- Функції можна оголосити за допомогою ключового слова `function`, за яким іде ім’я, список параметрів і блок коду, що виконує завдання.

:::interactive_editor

```js
function addNumbers(x, y, z) {
  return x + y + z;
}

console.log(addNumbers(5, 3, 8)); // Output: 16
```

:::

- Аргументи — це значення, які передаються функції під час її виклику.
- Виклик функції — це процес виконання функції в програмі шляхом вказання її імені з круглими дужками, у яких за потреби можуть бути аргументи.
- Коли функція завершує виконання, вона завжди повертає значення.
- За замовчуванням повернене значення функції — це `undefined`.
- Ключове слово `return` використовується, щоб вказати значення, яке має повернути функція, і завершує її виконання.
- Параметри за замовчуванням дозволяють функціям мати попередньо визначені значення, які використовуються, якщо аргумент не передано під час виклику функції. Це робить функції гнучкішими і запобігає помилкам у випадках, коли деякі аргументи можуть бути пропущені.

:::interactive_editor

```js
const calculateTotal = (amount, taxRate = 0.05) => {
  return amount + (amount * taxRate);
};

console.log(calculateTotal(100)); // Output: 105
```

:::

- Анонімні функції — це функції без імені, які можна призначити змінним. Призначивши їх змінним, ви можете повторно використовувати їх у будь-якому місці, де доступна ця змінна.

:::interactive_editor

```js
const multiplyNumbers = function(firstNumber, secondNumber) {
  return firstNumber * secondNumber;
};

console.log(multiplyNumbers(4, 5)); // Output: 20
```

:::

## Стрілкові функції

- Стрілкові функції — це більш лаконічний спосіб написання функцій у JavaScript.

:::interactive_editor

```js
const calculateArea = (length, width) => {
  const area = length * width;
  return `The area of the rectangle is ${area} square units.`;
};

console.log(calculateArea(5, 10)); // Output: "The area of the rectangle is 50 square units."
```

:::

- При оголошенні стрілкової функції ключове слово `function` не потрібне.
- Якщо використовується один параметр, можна опустити круглі дужки навколо списку параметрів.

:::interactive_editor

```js
const cube = x => {
  return x * x * x;
};

console.log(cube(3)); // Output: 27
```

:::

- Якщо тіло функції складається з одного виразу, можна опустити фігурні дужки та ключове слово `return`.

:::interactive_editor

```js
const square = number => number * number;

console.log(square(5)); // Output: 25
```

:::

## Область видимості в програмуванні

- **Глобальна область видимості**: це найзовнішніша область видимості в JavaScript. Змінні, оголошені в глобальній області видимості, доступні з будь-якого місця коду і називаються глобальними змінними.
- **Локальна область видимості**: це змінні, оголошені всередині функції. Ці змінні доступні лише в тій функції, де вони оголошені, і називаються локальними змінними.
- **Блочна область видимості**: блок — це множина інструкцій, укладених у фігурні дужки `{}`, наприклад, у `if` інструкціях або циклах.
- Блочна область видимості з `let` і `const` дає ще точніший контроль над доступністю змінних, допомагаючи уникати помилок і роблячи код більш передбачуваним.
