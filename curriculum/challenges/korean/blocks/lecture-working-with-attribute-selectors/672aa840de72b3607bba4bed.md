# --interactive--

CSS의 속성 선택자는 특정 속성을 가진 HTML 요소를 타겟팅할 수 있는 강력한 도구입니다. 즉, 특정 속성을 가진 요소나 그 속성의 특정 값에 스타일을 적용할 수 있습니다.

이것은 동적으로 요소에 스타일을 적용하거나 클래스 이름만으로는 충분한 구체성을 제공하지 못할 때 특히 유용합니다.

예를 들어, 속성 선택자를 사용해 `href` 속성을 가진 모든 링크를 타겟팅할 수 있습니다. 이는 페이지 내 모든 링크에 일관된 스타일을 적용할 때 매우 편리합니다.

다음은 `href` 속성을 가진 링크를 타겟팅하는 속성 선택자 사용법입니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Example link with an href attribute</a>
<a>Example link without an href attribute</a>
```

```css
a {
  display: block;
}

a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

이 CSS 규칙은 `href` 속성을 포함하는 모든 링크에 파란색 글자색과 밑줄을 적용하여 클릭 가능한 모든 링크가 일관되게 스타일이 적용되도록 합니다.

하지만 속성 선택자는 더 구체적으로 사용할 수도 있습니다. 예를 들어 `title` 속성을 가진 링크만 타겟팅하고 싶을 때가 있습니다. `title` 속성은 링크에 추가 정보를 제공하는 경우가 많아, 이런 링크를 다르게 스타일링해 추가 정보가 있음을 시각적으로 표시할 수 있습니다. 다음과 같이 할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

이 규칙은 `title` 속성을 가진 링크에 굵은 글씨를 적용하고 밑줄을 제거합니다. 이렇게 하면 페이지 내 다른 링크와 시각적으로 구분할 수 있습니다.

속성 선택자를 조합해 더 정밀하게 타겟팅할 수도 있습니다. 예를 들어 `href`와 `title` 속성을 모두 가진 링크를 스타일링하려면 다음과 같이 합니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[href][title] {
  display:block;
  color: green;
}
```

:::

이 경우 `href`와 `title` 속성을 모두 포함하는 링크만 초록색 글씨로 스타일링됩니다. 이런 수준의 제어가 CSS 속성 선택자를 강력하게 만드는 이유입니다.

또 다른 예로, 속성 값이 공백으로 구분된 목록 중 특정 값을 포함하는 경우를 매칭하는 속성 선택자가 있습니다.

다음은 여러 클래스를 가진 앵커 요소 예시입니다:

```html
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

이 특정 앵커 요소를 타겟팅하려면 다음 선택자를 사용할 수 있습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

여기서 `[attr~=value]` 구문은 클래스 속성에 `"primary"` 단어가 포함된 모든 앵커 요소를 타겟팅하는 데 사용됩니다.

속성 값이 특정 값으로 시작하는 요소를 타겟팅해야 한다면 `[attr^=value]` 구문을 사용할 수 있습니다. 예시는 다음과 같습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

이 예제에서 `a[href^="https://"]` 선택자는 `href` 속성 값이 `"https://"`로 시작하는 모든 앵커 요소를 타겟팅합니다.

속성 값이 특정 값으로 끝나는 요소를 타겟팅하려면 `[attr$=value]` 구문을 사용할 수 있습니다. 예시는 다음과 같습니다:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Visit Example Site</a>
```

```css
a[href$=".com"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

이 예제에서 `a[href$=".com"]` 선택자는 `href` 속성 값이 `.com`로 끝나는 모든 앵커 요소를 타겟팅합니다.

속성 선택자를 사용하면 웹페이지 스타일링을 향상시킬 뿐 아니라, 링크 같은 인터랙티브 요소를 속성에 따라 더 구분하기 쉽게 만들어 접근성도 개선됩니다.
