# --interactive--

In den vorherigen Lektionen haben wir kurz darüber gesprochen, wie man Abstand zwischen Grid-Elementen erzeugt. In dieser Lektion gehen wir jedoch genauer darauf ein, wie man die `row-gap`, `column-gap` und `gap` Eigenschaften in einem Grid-Layout verwendet.

Zuerst müssen wir aber noch einmal überprüfen, was ein Track im CSS Grid ist. Ein Track ist der Raum zwischen zwei benachbarten Gitterlinien. Diese Linien werden automatisch erstellt, wenn Sie CSS Grid verwenden. In diesem Zusammenhang beziehen sich Tracks im Allgemeinen auf die Reihen und Spalten, die das Grid-Layout bilden.

Um Lücken zwischen Spalten in einem CSS Grid zu erzeugen, können Sie die `column-gap` Eigenschaft verwenden. Zulässige Werte für diese Eigenschaft sind Pixel, die `em` Einheit, Prozentsätze oder das Schlüsselwort `normal`.

Wenn Sie den Wert `normal` für die `column-gap` Eigenschaft verwenden, ist das Ergebnis `0` für Grid-Layouts und `1em` für mehrspaltige Layouts.

Hier ist ein Beispiel für die Auszeichnung eines vier-spaltigen Grid-Layouts:

```html
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

Für das CSS setzen wir das Display auf grid und die `column-gap` Eigenschaft auf 10 Pixel:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  height: 100px;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.grid-container div {
  background-color: darkblue;
}
```

:::

Wenn wir das Beispiel so ändern wollten, dass es zwei Reihen mit blauen Kästchen gibt und mehr Abstand zwischen den Reihen entsteht, können wir die `row-gap` Eigenschaft verwenden:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  column-gap: 10px;
  row-gap: 30px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

In diesem überarbeiteten Beispiel setzen wir die `row-gap` Eigenschaft auf 30 Pixel und ändern die `grid-template-columns`, sodass nur zwei `1fr` Einheiten anstelle von vier verwendet werden, um zwei Reihen von Kästchen zu erzeugen.

Genau wie bei der `column-gap` Eigenschaft können zulässige Werte für die `row-gap` Eigenschaft Prozentsätze, `em` und Pixel sein.

Wenn Sie eine Kurzschreibweise für das Erzeugen von Lücken zwischen Reihen und Spalten verwenden möchten, können Sie die `gap` Eigenschaft nutzen. Hier ist die grundlegende Syntax:

```css
gap: row-value optional-column-value;
```

Wenn Sie einen Wert für die `gap` Eigenschaft angeben, wird dieser Wert sowohl für Reihen als auch für Spalten angewendet. Wenn Sie zwei Werte angeben, wird der erste Wert für die Reihen und der zweite für die Spalten verwendet:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 30px 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

Zulässige Werte für die `gap` Kurzschreibweise-Eigenschaft sind Prozentsätze, Pixel, `em` und sogar `calc()` Werte. `fr` Einheiten können hier jedoch nicht verwendet werden.

Die `row-gap`, `column-gap` und `gap` Eigenschaften bieten flexible Möglichkeiten, den Abstand zwischen Elementen in einem CSS Grid-Layout zu steuern. Mit diesen Eigenschaften können Sie ganz einfach optisch ansprechende Grids mit gleichmäßigen und anpassbaren Lücken zwischen Reihen und Spalten erstellen.
