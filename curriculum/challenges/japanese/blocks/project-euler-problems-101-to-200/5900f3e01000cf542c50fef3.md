# --description--

5つの黒い正方形タイルの列があり、そのうちいくつかのタイルを赤（長さ2）、緑（長さ3）、青（長さ4）のいずれかの色付き長方形タイルに置き換えます。

赤いタイルを選んだ場合、ちょうど7通りの置き換え方があります。

<img alt="長さ5の列に赤い長方形を置く可能な方法" src="https://cdn.freecodecamp.org/curriculum/project-euler/red-green-or-blue-tiles-1.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

緑のタイルを選んだ場合は3通りあります。

<img alt="長さ5の列に緑の長方形を置く可能な方法" src="https://cdn.freecodecamp.org/curriculum/project-euler/red-green-or-blue-tiles-2.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

青のタイルを選んだ場合は2通りあります。

<img alt="長さ5の列に青い長方形を置く可能な方法" src="https://cdn.freecodecamp.org/curriculum/project-euler/red-green-or-blue-tiles-3.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

色を混ぜることができないと仮定すると、長さ5の列の黒いタイルを置き換える方法は7 + 3 + 2 = 12通りあります。色を混ぜず、かつ少なくとも1つの色付きタイルを使う場合、長さ50の列の黒いタイルを置き換える異なる方法は何通りありますか？

**注意:** これは問題117に関連しています。
