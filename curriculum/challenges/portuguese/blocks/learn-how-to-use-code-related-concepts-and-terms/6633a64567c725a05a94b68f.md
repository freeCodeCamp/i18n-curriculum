# --description--

Nesta tarefa, você explorará o uso metafórico de termos para entender conceitos de programação. Metáforas são uma maneira poderosa de visualizar e explicar ideias complexas relacionando-as a objetos ou ações do cotidiano.

`Container` é usado metaforicamente para descrever variáveis como coisas que podem conter e armazenar diferentes dados, assim como um `container` pode conter diferentes itens.

As ações `fill and empty` descrevem como você pode adicionar ou remover dados de variáveis, semelhante a como você pode `fill` uma garrafa de água ou despejar a água.

`Locked boxes` descreve constantes como `containers` seguros e imutáveis, enfatizando que não podem ser alterados depois de definidos.

A frase `keep the same contents` destaca que os dados em constantes não mudam, como uma caixa que está trancada e sempre contém as mesmas coisas dentro.
