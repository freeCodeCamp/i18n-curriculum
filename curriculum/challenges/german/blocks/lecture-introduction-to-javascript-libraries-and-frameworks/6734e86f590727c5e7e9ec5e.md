# --description--

React ist eine der beliebtesten JavaScript-Bibliotheken zum Erstellen von Benutzeroberflächen und Webanwendungen. Ursprünglich von Facebook entwickelt und gepflegt, hat React aufgrund seiner Effizienz, Flexibilität und Funktionen große Beliebtheit in der Webentwicklung erlangt.

Ein grundlegendes Konzept in React ist die Erstellung wiederverwendbarer UI-Komponenten. Diese Komponenten, wie Buttons, Karten und Avatar-Komponenten, können in Ihrer Anwendung einfach mehrfach verwendet werden. Sie können diese Komponenten ineinander verschachteln, um komplexere, dynamische Oberflächen zu erstellen.

Ein wesentlicher Vorteil von React ist, dass diese benutzerdefinierten Komponenten unabhängig voneinander aktualisiert und gerendert werden können, wenn sich Daten ändern.

Im Gegensatz zu traditionellem JavaScript, das eine direkte Manipulation des DOM (Document Object Model) erfordert, verwendet React ein virtuelles DOM, das Leistung und Effizienz verbessert. Mehr über das virtuelle DOM und seine Funktionsweise lernen Sie in den kommenden Lektionen.

Neben wiederverwendbaren Komponenten bietet React auch eine leistungsstarke Möglichkeit, den Zustand Ihrer Anwendung zu verwalten. Zustand bezieht sich auf die Daten, die bestimmen, wie eine Komponente gerendert wird und sich verhält.

Mit React können Sie den Zustand von Komponenten einfach verfolgen und aktualisieren, sodass die Benutzeroberfläche stets die aktuellsten Daten widerspiegelt. Mehr über die Arbeit mit Zustand lernen Sie in zukünftigen Lektionen.

Obwohl es viele Bibliotheken im JavaScript-Ökosystem gibt, konzentriert sich freeCodeCamp hauptsächlich darauf, React zu lehren, wegen seiner weitverbreiteten Nutzung und Nachfrage in der Branche.

Unsere nächsten Lektionen werden tiefer in den Aufbau benutzerdefinierter Komponenten, die Verwaltung von Zustand und mehr eintauchen.
