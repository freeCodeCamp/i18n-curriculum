# --description--

整数の集合 {1, 2, ..., $n$} から選ばれた隠された数字を質問を通じて見つけようとしています。私たちが尋ねる各数字（質問）に対して、次の3つのいずれかの答えが返ってきます。

- 「あなたの推測は隠された数字より小さい」（この場合、コストは a がかかります）、または
- 「あなたの推測は隠された数字より大きい」（この場合、コストは b がかかります）、または
- 「はい、それです！」（ゲーム終了）

与えられた $n$、$a$、$b$ の値に対して、最適な戦略は<u>最悪の場合の合計コストを最小化する</u>ことです。

例えば、$n = 5$、$a = 2$、$b = 3$ の場合、最初の質問として「<strong>2</strong>」を尋ねることから始めるかもしれません。

もし「2は隠された数字より大きい」と言われた場合（コストは $b = 3$）、隠された数字は「<strong>1</strong>」であることが確定します（合計コストは <strong><span style="color: blue;">3</span></strong>）。

もし「2は隠された数字より小さい」と言われた場合（コストは $a = 2$）、次の質問は「<strong>4</strong>」になります。

もし「4は隠された数字より大きい」と言われた場合（コストは $b = 3$）、隠された数字は「<strong>3</strong>」であることが確定します（合計コストは $2 + 3 = \color{blue}{\mathbf{5}}$）。

もし「4は隠された数字より小さい」と言われた場合（コストは $a = 2$）、隠された数字は「<strong>5</strong>」であることが確定します（合計コストは $2 + 2 = \color{blue}{\mathbf{4}}$）。

したがって、この戦略による最悪の場合のコストは <strong><span style="color: red">5</span></strong> です。また、これが達成可能な最小の最悪コストであることも示せます。つまり、実際には $n$、$a$、$b$ の与えられた値に対する最適な戦略を説明したことになります。

$C(n, a, b)$ を、与えられた $n$、$a$、$b$ に対して最適な戦略が達成する最悪の場合のコストとします。

いくつかの例を示します：

$$\begin{align}
  & C(5, 2, 3) = 5 \\\\
  & C(500, \sqrt{2}, \sqrt{3}) = 13.220\\,731\\,97\ldots \\\\
  & C(20\\,000, 5, 7) = 82 \\\\
  & C(2\\,000\\,000, √5, √7) = 49.637\\,559\\,55\ldots \\\\
\end{align}$$

$F_k$ をフィボナッチ数列とし、$F_k = F_{k - 1} + F_{k - 2}$、初期条件は $F_1 = F_2 = 1$ とします。

$\displaystyle\sum_{k = 1}^{30} C({10}^{12}, \sqrt{k}, \sqrt{F_k})$ を求め、小数点以下8桁で四捨五入してください。
