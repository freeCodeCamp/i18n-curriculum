# --explanation--

在像這樣的句子中，`I think` 後面使用 `that` 是選用的。說 `I think that it's vital we make these updates.` 也是正確的。然而，在日常對話中，為了讓句子更流暢且更快速，通常會省略 `that`。例如： 

`I think that we need more time to finish the project.` – 這個結構是正確的。不過大多數人會說 `I think we need more time…`。
