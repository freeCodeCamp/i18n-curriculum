# --explanation--  

`That way` は、決定や行動の結果や利点を説明するために使われます。例えば：

`We should automate testing. That way, we can catch bugs faster.` – `That way` は、自動化によってバグをより早く見つけられることを示しています。  

`While` は、2つのことが同時に起こっていることを示すために使われます。例えば：  

`The team is fixing bugs while the designers work on the UI.` – これは両方の作業が同時に行われていることを意味します。
