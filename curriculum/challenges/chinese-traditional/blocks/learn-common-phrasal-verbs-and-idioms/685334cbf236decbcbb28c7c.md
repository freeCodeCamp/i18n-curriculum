# --explanation--

`draft` 是可編輯或審閱的文件或計畫的早期版本。例如：

`I wrote a draft of the email before sending the final version.`－這表示此人先寫了電子郵件的初稿版本。

`To run something by someone` 用來向某人展示或解釋某事以取得回授或批准。 例如：

`Can I run this idea by you before we present it?`－這表示你正在徵求某人的意見。

`To roll something out` 意思是正式推出或為更廣泛的群組介紹某事。例如：

`They will roll out the new software update next week。`－這表示新版本的軟體將在下週發布。

後綴 `-wide` 意味著「跨越整個」某物。在 `company-wide` 中，它表示適用於整個公司的某事。另一個例子：

`There was a system-wide outage yesterday。`－這表示問題影響了整個系統。
