# --description--

Relationale Datenbanken haben ein breites Anwendungsspektrum, darunter Webentwicklung, Bestandsverwaltungssysteme, Gesundheitssysteme, E-Commerce-Anwendungen und mehr.

Einige der gängigsten relationalen Datenbanken sind MySQL, PostgreSQL, SQLite und Microsoft SQL Server. Lernen wir ein wenig über jede von ihnen.

MySQL ist ein Open-Source-Relationale-Datenbank-Management-System. Es ist sehr beliebt für die Webentwicklung, weil es einfach zu bedienen, zuverlässig und effizient für den Aufbau realer Anwendungen ist. Außerdem gibt es eine aktive Community von Entwicklern, die es unterstützen und pflegen.

PostgreSQL wird offiziell als „die weltweit fortschrittlichste Open-Source-relationale Datenbank“ beschrieben. Es ist ein kostenloses, Open-Source-objektrelationales Datenbanksystem, das für seine Zuverlässigkeit, Datenintegrität und Erweiterbarkeit bekannt ist. Zum Beispiel können Sie eigene Datentypen, benutzerdefinierte Funktionen definieren und Code in verschiedenen Programmiersprachen schreiben, ohne Ihre Datenbank neu kompilieren zu müssen.

SQLite ist eine kostenlose, Open-Source- und leichtgewichtige dateibasierte SQL-Datenbank-Engine. Sie ist eine der am weitesten verbreiteten Datenbank-Engines weltweit. SQLite ist eigenständig und serverlos. Es ist eine Null-Konfigurations-Datenbank, was bedeutet, dass kein Einrichtungsprozess erforderlich ist, was ideal ist, um schnell loszulegen.

Diese drei Datenbanken sind kostenlos und Open Source. Es gibt auch andere beliebte proprietäre Optionen, darunter Microsoft SQL Server und Oracle-Datenbank.

Microsoft SQL Server ist ein beliebtes relationales Datenbank-Management-System, das von Microsoft entwickelt wurde und in verschiedenen Anwendungen verwendet wird.

Oracle-Datenbank ist ein skalierbares relationales Datenbank-Management-System, das häufig für Unternehmensanwendungen verwendet wird, die hohe Leistung erfordern.

Prima. Jetzt, wo Sie mehr über die verschiedenen verfügbaren Optionen wissen, sehen wir uns an, wie Sie PostgreSQL installieren können.

Zuerst müssen Sie auf die offizielle Website gehen: <a href="https://www.postgresql.org/" target="_blank">https://www.postgresql.org/</a>.

Wenn Sie dort sind, klicken Sie auf die Schaltfläche „Download“.

Dadurch gelangen Sie auf eine Seite, auf der Sie Ihr Betriebssystem auswählen können. Wenn Sie auf eines klicken, werden Sie zu den Anweisungen für dieses Betriebssystem weitergeleitet.

Wir gehen den Installationsprozess für das Windows-Betriebssystem durch, aber der Prozess ist bei den Betriebssystemen ähnlich.

Von dieser Seite aus können Sie den Installer herunterladen, indem Sie auf den Link „Download the installer“ klicken.

Dadurch gelangen Sie auf eine Seite, auf der Sie die Version von PostgreSQL auswählen können, die Sie installieren möchten. Wenn Sie auf das entsprechende Symbol klicken, sollte der Download starten.

Dann sollten Sie im Ordner „Downloads“ oder in dem Ordner, in dem Sie standardmäßig Ihre Downloads speichern, den Installer sehen.

Wenn Sie darauf doppelklicken, starten Sie den Setup-Prozess. Klicken Sie auf Weiter.

Wählen Sie das Verzeichnis, in dem PostgreSQL installiert werden soll. Standardmäßig wird Ihnen eines angezeigt, aber Sie können es anpassen.

Dann wählen Sie die Komponente aus, die Sie installieren möchten. Es gibt vier mögliche Komponenten:

- PostgreSQL Server, die Datenbank-Engine selbst.
- pgAdmin 4, ein benutzerfreundliches Tool zur Datenbankverwaltung.
- Stack Builder, ein Tool zum Herunterladen und Installieren zusätzlicher PostgreSQL-bezogener Software und Erweiterungen
- Und Command Line Tools, um direkt über Ihr Terminal mit dem PostgreSQL-Server zu interagieren. Dazu gehört psql, ein interaktives Terminal zum Ausführen von SQL-Abfragen.

Wenn Sie bereit sind, klicken Sie auf Weiter.

Dann wählen Sie ein Verzeichnis zum Speichern Ihrer Daten aus. Ihnen wird ein Standardpfad angezeigt, aber Sie können diesen anpassen. Wenn Sie bereit sind, klicken Sie auf Weiter.

Jetzt müssen Sie ein Passwort für den Datenbank-Superuser eingeben, den Benutzer mit uneingeschränktem Zugriff auf alle Datenbanken und Objekte. Geben Sie Ihr Passwort ein, wiederholen Sie es und klicken Sie dann auf Weiter.

Dann wählen Sie die Portnummer aus, auf der der Server lauschen soll. Standardmäßig ist es 5432. Klicken Sie dann auf Weiter.

Sie müssen eine Gebietsschema-Einstellung für den neuen Datenbank-Cluster auswählen. Das ist wichtig, damit die Datenbank gebietsschemaabhängige Datentypen richtig interpretiert. Klicken Sie dann auf Weiter.

Schließlich sehen Sie eine Zusammenfassung vor der Installation. Nehmen Sie sich einen Moment Zeit, um diese zu überprüfen, und klicken Sie auf Weiter.

Sie müssen bestätigen, dass Sie den Installationsprozess starten möchten. Wenn Sie auf Weiter klicken, beginnt der Prozess. Er sollte nur wenige Minuten dauern.

Wenn der Prozess abgeschlossen ist, müssen Sie je nach den installierten Komponenten möglicherweise auswählen, ob Sie Stack Builder starten möchten, um zusätzliche Tools, Treiber und Anwendungen herunterzuladen und zu installieren.

Wenn Sie sich entscheiden, Stack Builder zu starten, sehen Sie einen Bildschirm, auf dem Sie die Installation starten können. Alternativ können Sie auf Abbrechen klicken, um dies später zu tun.

Herzlichen Glückwunsch! Jetzt haben Sie PostgreSQL auf Ihrem Computer installiert. Der Prozess sollte für andere Betriebssysteme ähnlich sein. Danach können Sie die Datenbank verwenden.

Gehen Sie zu Ihren Apps und suchen Sie nach pgAdmin, einer Open-Source-Administrations- und Entwicklungsplattform für PostgreSQL, die installiert worden sein sollte, wenn Sie sie bei den Komponenten ausgewählt haben. Klicken Sie auf pgAdmin 4, um es zu öffnen.

Sie sehen einen anfänglichen Ladebildschirm, wenn die Anwendung startet. Dann sollten Sie ein Dashboard sehen, auf dem Sie einen neuen Server starten, Ihre Datenbanken überprüfen, pgAdmin konfigurieren und mehr können. Sie finden dort auch hilfreiche Links zur Dokumentation.

Wenn Sie oben links auf „Servers“ klicken, müssen Sie das Passwort eingeben, das Sie während der Installation gewählt haben.

Nach der Eingabe Ihres Passworts sollten Sie einen Bildschirm sehen, auf dem Sie Zugriff auf Ihre Datenbanken haben, einschließlich deren Tabellen.

Sie finden auch ein weiteres sehr hilfreiches Tool, psql, ein interaktives Terminal für SQL-Abfragen. Um es zu öffnen, suchen Sie auf Ihrem System nach psql. Klicken Sie darauf.

Es öffnet sich die SQL Shell, in der Sie Befehle ausführen können.

Und so können Sie PostgreSQL unter Windows installieren. Der Prozess ist für macOS und Linux sehr ähnlich.

Alternativ können Sie, wenn Sie es auf macOS installieren, Homebrew verwenden, einen Paketmanager. Sobald Sie Homebrew installiert haben, führen Sie diesen Befehl aus, um PostgreSQL zu installieren:

```bash
brew install postgresql@<version>
```

Sie können die Version, die Sie installieren möchten, angeben, indem Sie die Nummer nach dem at (`@`) Symbol schreiben.
