# --interactive--

تُضيف الخاصية `id` معرفًا فريدًا لعنصر HTML.

إليك مثالًا على عنصر `h1` يحمل `id` باسم `title`.

أسفل عنصر `h1`، أضف عنصر `h2` مع تعيين `id` إلى `"subtitle"`. يمكنك كتابة أي نص تريده لعنصر `h2` وسترى التغييرات في نافذة المعاينة. لتتفاعل مع المثال، يجب تفعيل المحرر التفاعلي.

:::interactive_editor

```html
<h1 id="title">Movie Review Page</h1>

```

:::

يمكنك الرجوع إلى اسم `id` الخاص بـ `title` داخل JavaScript أو CSS. إليك مثال CSS يشير إلى `id` الخاص بـ `title` لتغيير `color` النص إلى `red`.

**ملاحظة**: تم توفير بعض CSS لك في هذا المثال التفاعلي. لا تقلق بشأن محاولة فهم شفرة CSS لأنك ستتعلم المزيد عنها في الدروس القادمة. لكن إذا أردت رؤية تغيير لون النص إلى الأزرق، فعّل المحرر التفاعلي، وانقر على تبويب `styles.css` وغيّر `color: red;` إلى `color: blue;`.

:::interactive_editor

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>Review page Example</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
    <h1 id="title">Movie Review Page</h1>
  </body>
</html>
```

```css
#title {
  color: red;
}
```

:::

رمز الهاش (`#`) أمام `title` يخبر الكمبيوتر بأنك تريد استهداف `id` بهذه القيمة. يجب ألا تُستخدم أسماء `id` أكثر من مرة، ويجب أن تكون دائمًا فريدة. شيء آخر يجب ملاحظته عن قيم `id` هو أنها لا يمكن أن تحتوي على فراغات. إليك مثالًا على تطبيق الكلمات `main` و `heading` كقيمة لخاصية `id`:

```html
<h1 id="main heading">Main heading</h1>
```

تعتبر المتصفحات هذه المسافة جزءًا من `id`، مما يؤدي إلى مشاكل غير مرغوبة عند تطبيق التنسيق والبرمجة. يجب أن تحتوي قيم خاصية `id` فقط على حروف، أرقام، شرطات سفلية، وشرطات.

على عكس خاصية `id`، لا تحتاج قيمة خاصية `class` لأن تكون فريدة ويمكن أن تحتوي على مسافات.

إليك مثالًا لتطبيق صنف يسمى `box` على عنصر `div`.

```html
<div class="box"></div>
```

إذا أردت إضافة أسماء أصناف متعددة لعنصر، يمكنك فعل ذلك بفصل الأسماء بمسافة. إليك مثالًا محدثًا يطبق أصنافًا متعددة على عنصر `div`.

```html
<div class="box red-box"></div>
```

إليك مثالًا آخر لتطبيق أصناف متعددة على عدة عناصر `div`.

**ملاحظة**: تم توفير بعض CSS لك في هذا المثال التفاعلي. لا تقلق بشأن محاولة فهم شفرة CSS لأنك ستتعلم المزيد عنها في الدروس القادمة. لكن إذا أردت تغيير لون الصندوقين الأول والثالث، فعّل المحرر التفاعلي وانقر على تبويب `styles.css` وغيّر `background-color: red;` إلى `background-color: black;`.

:::interactive_editor

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>Colored boxes example</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
    <div class="box red-box"></div>
    <div class="box blue-box"></div>
    <div class="box red-box"></div>
    <div class="box blue-box"></div>
  </body>
</html>
```

```css
.box {
  width: 100px;
  height: 100px;
}

.red-box {
  background-color: red;
}

.blue-box {
  background-color: blue;
}
```

:::

للتلخيص، متى يجب استخدام `id` مقابل `class`؟ تُستخدم الأصناف عندما تريد تطبيق مجموعة من التنسيقات على عدة عناصر. إذا أردت استهداف عنصر محدد، من الأفضل استخدام `id` لأن هذه القيم يجب أن تكون فريدة.
