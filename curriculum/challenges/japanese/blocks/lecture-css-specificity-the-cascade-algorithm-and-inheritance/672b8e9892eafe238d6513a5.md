# --interactive--

クラスセレクターはCSSの重要な要素で、開発者が同じクラス属性を持つ複数の要素を対象にして一貫したスタイルを適用できます。

これにより、ウェブサイト全体に効率的かつ柔軟にスタイルを適用できます。

クラスセレクターはピリオド（`.`）に続けてクラス名を指定して定義します。HTMLドキュメント内のどの要素にも適用可能です。

以下はクラスセレクターを使った例です：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="highlight">Example paragraph</p>
```

```css
.highlight {
  color: green;
}
```

:::

この例では、`highlight` クラスを持つすべての要素のテキスト色が `green` に設定されます。

クラスセレクターは型セレクターよりも特異性が高いですが、IDセレクターやインラインスタイルよりは低いです。

クラスセレクターの特異性の値は `(0, 0, 1, 0)` です。つまり、クラスセレクターは型セレクターをオーバーライドできますが、IDセレクターやインラインスタイルにはオーバーライドされます。

クラスセレクターは他のセレクターと組み合わせて、より具体的なルールを作成できます。

以下は段落の型セレクターとクラスセレクターを組み合わせた例です：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="bold-text">Example paragraph</p>
<p class="bold-text">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p.bold-text {
  font-weight: bold;
}
```

:::

このルールは、`p` 要素でかつ `bold-text` クラスを持つものにのみ適用され、テキストを太字にします。
