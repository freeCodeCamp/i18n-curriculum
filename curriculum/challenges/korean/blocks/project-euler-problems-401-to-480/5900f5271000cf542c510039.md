# --description--

정수의 십진수 전개에 1을 제외한 11의 거듭제곱을 나타내는 부분 문자열이 포함되지 않으면 그 정수를 11-프리라고 합니다.

예를 들어, 2404와 13431은 11-프리이지만 911과 4121331은 그렇지 않습니다.

$E(n)$을 $n$번째 양의 11-프리 정수라고 합시다. 예를 들어, $E(3) = 3$, $E(200) = 213$, $E(500\\,000) = 531\\,563$입니다.

$E({10}^{18})$을 구하세요.
