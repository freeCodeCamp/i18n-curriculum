# --explanation--

`Monitoring`는 일반적으로 원활하게 작동하는지 확인하거나 문제를 식별하기 위해 무언가의 성능이나 상태를 관찰하거나 추적하는 것을 말합니다. 예를 들어:  

`The team is monitoring the servers to prevent downtime.` - 이는 팀이 중단 없이 서버가 작동하도록 적극적으로 감시하고 있음을 의미합니다. 

`Future Continuous` 시제는 미래의 특정 시점에 계속 진행될 동작을 설명합니다. `will be` + `-ing`로 끝나는 동사 형태로 구성됩니다. 예를 들어:

`The IT team will be troubleshooting the network issues tomorrow afternoon.` - 이는 IT 팀이 미래의 특정 시간 동안 네트워크 문제를 적극적으로 해결할 것임을 나타냅니다.  

Brian은 시스템이 원활하게 작동하도록 성능을 적극적으로 추적할 것임을 보여주기 위해 `Future Continuous` 시제를 사용합니다.
