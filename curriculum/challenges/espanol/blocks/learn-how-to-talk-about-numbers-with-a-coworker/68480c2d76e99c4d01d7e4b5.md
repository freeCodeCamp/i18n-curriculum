# --explanation--

La `User experience` (o `UX`) trata sobre cómo se siente un usuario al usar un producto — si es fácil, agradable y fluido. Por ejemplo:

`Improving page load time boosts user experience.` — Hacer que las páginas carguen más rápido ayuda a los usuarios a disfrutar más de la app.

`Enhancements` son cambios realizados para mejorar un producto o proceso. Por ejemplo:

`The latest update includes performance enhancements.` — La nueva versión hace que la aplicación funcione mejor y más rápido.

En la oración de Alice, la mitad del presupuesto de diseño se gasta en hacer que el producto sea más fácil y mejor de usar.
