# --description--

لعدد موجب $n$، عرّف $C(n)$ على أنه عدد الأعداد الصحيحة $x$ التي تحقق $1 < x < n$ و$x^3 \equiv 1\bmod n$.

عندما يكون $n = 91$، هناك 8 قيم ممكنة لـ $x$، وهي: 9، 16، 22، 29، 53، 74، 79، 81. إذن، $C(91) = 8$.

احسب مجموع الأعداد الموجبة $n ≤ {10}^{11}$ التي تحقق $C(n)=242$.
