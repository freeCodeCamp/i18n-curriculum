# --description--

لأعداد صحيحة ثابتة $a$ و $b$ و $c$، عرّف الدالة المجنونة $F(n)$ كما يلي:

$$\begin{align}
  & F(n) = n - c \\;\text{ for all } n > b \\\\
  & F(n) = F(a + F(a + F(a + F(a + n)))) \\;\text{ for all } n ≤ b.
\end{align}$$

أيضًا، عرّف $S(a, b, c) = \displaystyle\sum_{n = 0}^b F(n)$.

على سبيل المثال، إذا كان $a = 50$ و $b = 2000$ و $c = 40$، فإن $F(0) = 3240$ و $F(2000) = 2040$. وأيضًا، $S(50, 2000, 40) = 5\\,204\\,240$.

احسب آخر 9 أرقام من $S({21}^7, 7^{21}, {12}^7)$.
