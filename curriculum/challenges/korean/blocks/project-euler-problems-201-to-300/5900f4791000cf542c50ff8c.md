# --description--

다항식 $P(x)$의 근 또는 영점은 방정식 $P(x) = 0$의 해입니다.

$n$의 각 자리수가 계수인 다항식을 $P_n$으로 정의하세요.

예를 들어, $P_{5703}(x) = 5x^3 + 7x^2 + 3$입니다.

다음과 같이 알 수 있습니다:

- $P_n(0)$은 $n$의 마지막 자리수입니다,
- $P_n(1)$은 $n$의 자리수 합입니다,
- $P_n(10)$은 $n$ 자체입니다.

$Z(k)$를 다항식 $P_n$이 적어도 하나의 정수 근을 가지는 양의 정수 $n$의 개수로 정의하세요. 단, $n$은 $k$를 넘지 않습니다.

$Z(100\,000)$이 14696임을 확인할 수 있습니다.

$Z({10}^{16})$의 값은 얼마인가요?
