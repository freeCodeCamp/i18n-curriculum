# --description--

변형된 콜라츠 수열은 시작 값 $a_1$에서 다음과 같이 구합니다:

$a_n$이 3으로 나누어떨어지면 $a_{n + 1} = \frac{a_n}{3}$입니다. 이를 큰 하강 단계인 "D"로 표시합니다.

$a_n$을 3으로 나눈 나머지가 1이면 $a_{n + 1} = \frac{4a_n + 2}{3}$입니다. 이를 상승 단계인 "U"로 표시합니다.

$a_n$을 3으로 나눈 나머지가 2이면 $a_{n + 1} = \frac{2a_n - 1}{3}$입니다. 이를 작은 하강 단계인 "d"로 표시합니다.

수열은 어떤 $a_n = 1$이 될 때 종료합니다.

임의의 정수가 주어지면, 단계 수열을 나열할 수 있습니다. 예를 들어 $a_1 = 231$이면 수열 $\\{a_n\\} = \\{231, 77, 51, 17, 11, 7, 10, 14, 9, 3, 1\\}$은 단계 "DdDddUUdDD"에 해당합니다.

물론, 같은 단계 수열 "DdDddUUdDD..."로 시작하는 다른 수열도 존재합니다.

예를 들어 $a_1 = 1004064$이면 수열은 DdDddUUdDDDdUDUUUdDdUUDDDUdDD입니다.

사실, 1004064는 $10^6$보다 크면서 DdDddUUdDD로 시작하는 가장 작은 $a_1$입니다.

"UDDDUdddDDUDDddDdDddDDUDDdUUDd" 단계 수열로 시작하는 $10^{15}$보다 큰 가장 작은 $a_1$은 무엇인가요?
