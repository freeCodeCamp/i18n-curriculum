# --description--

Flexbox es un diseño CSS unidimensional que puede controlar cómo se distribuyen y alinean los elementos dentro de un contenedor.

Para usarlo, asigna a un elemento una propiedad `display` de `flex`. Esto convertirá el elemento en un <em>contenedor flexible</em>. Cualquier hijo directo de un contenedor flexible se llama <em>elemento flexible</em>.

Crea un selector `.gallery` y conviértelo en un contenedor flexible.



# --hints--

Debe tener un selector `.gallery`.

```js
assert.exists(new __helpers.CSSHelp(document).getStyle('.gallery'));
```

El selector `.gallery` debe tener una propiedad `display` establecida en `flex` como valor.

```js
assert.equal(new __helpers.CSSHelp(document).getStyle('.gallery')?.display, 'flex');
```
