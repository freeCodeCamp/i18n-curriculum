# --description--

Доведено, що поліном $n^4 + 4n^3 + 2n^2 + 5n$ ділиться на 6 для будь-якого цілого числа $n$. Також доведено, що 6 — найбільше ціле число, яке задовольняє цю властивість.

Визначимо $M(a, b, c)$ як максимальне $m$, таке що $n^4 + an^3 + bn^2 + cn$ ділиться на $m$ для всіх цілих $n$. Наприклад, $M(4, 2, 5) = 6$.

Також визначимо $S(N)$ як суму $M(a, b, c)$ для всіх $0 < a, b, c ≤ N$.

Ми можемо перевірити, що $S(10) = 1\,972$ і $S(10\,000) = 2\,024\,258\,331\,114$.

Нехай $F_k$ — послідовність Фібоначчі:

- $F_0 = 0$, $F_1 = 1$ та
- $F_k = F_{k - 1} + F_{k - 2}$ для $k ≥ 2$.

Знайдіть останні 9 цифр суми $\sum S(F_k)$ для $2 ≤ k ≤ 1\,234\,567\,890\,123$.
