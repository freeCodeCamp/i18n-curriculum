# --interactive--

CSSで背景画像を扱うとき、画像の表示方法を制御するためにいくつかのプロパティがあります。

ここで注目する主なプロパティは`background-size`、`background-repeat`、`background-position`、`background-attachment`です。

まずは`background-image`プロパティを見てみましょう。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

上記のCSSは`body`要素に猫の背景画像を設定しています。

背景画像のサイズを設定したい場合は、`background-size`プロパティを使えます。

画像を切り抜いたり伸縮させたりせずにできるだけ大きくスケールするには`contain`を使います。

`background-size: contain`を使った例はこちらです。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    min-height: 100px;
  }
</style>
```

:::

`min-height`を`100px`に設定しているので、背景画像が見え、レイアウトは基準となる高さを保ちます。これにより、コンテンツが少なくてもデザインがバランスよく見えます。

`background-size`プロパティを`cover`に変更すると、背景画像はアスペクト比を維持しながら`body`要素全体を覆うようにスケールします。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    min-height: 100px;
  }
</style>
```

:::

これまでの例では、背景画像が繰り返し表示されていることに気づいたかもしれません。

デフォルトでは、背景画像は要素全体を埋めるために水平方向と垂直方向の両方で繰り返されます。しかし、この動作は制御できます。

`background-repeat`プロパティに`no-repeat`の値を設定して使えます。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

`background-size`を`contain`に、`background-repeat`を`no-repeat`に設定すると、画像は画面上で繰り返されなくなります。

背景画像を水平方向に繰り返したい場合は、`repeat-x`プロパティに`background-repeat`の値を使えます。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-x;
    min-height: 100px;
  }
</style>
```

:::

背景画像を垂直方向に繰り返したい場合は、`repeat-y`プロパティに`background-repeat`の値を使えます。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-y;
    min-height: 100px;
  }
</style>
```

:::

背景画像の位置を画面上で設定したい場合は、`background-position`プロパティを使えます。

`background-position`プロパティでは、背景画像が要素内のどこに表示されるかを設定できます。`top`、`bottom`、`left`、`right`、`center`のようなキーワードや、ピクセルやパーセンテージの具体的な値を使えます。

`center`に対して`top`と`background-position`を使った例はこちらです。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

このCSSコードは背景画像を要素の水平方向中央、垂直方向上部に配置しています。

最後に、`background-attachment`は背景画像がコンテンツと一緒にスクロールするか、ページをスクロールしても固定されるかを決めます。

主な値は`scroll`（デフォルト）で、背景画像はコンテンツと一緒にスクロールし、`fixed`では背景画像が画面上の同じ位置に固定されます。

`fixed`プロパティに`background-attachment`の値を使った例はこちらです。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-position: center top;
    background-attachment: fixed;
  }
</style>
```

:::

このCSSコードはページをスクロールしても背景画像が固定されるように設定しています。

いくつかのプロパティを1行にまとめたい場合は、ショートハンドの`background`プロパティを使えます。

例はこちらです。

:::interactive_editor

```html
<style>
  body {
    background: center top fixed
      url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

上記の例は`background-image`、`background-position`、`background-attachment`を1行にまとめています。

これらのプロパティをマスターすれば、背景画像の表示方法を自在にコントロールでき、より視覚的に魅力的で機能的なレイアウトが作れます。
