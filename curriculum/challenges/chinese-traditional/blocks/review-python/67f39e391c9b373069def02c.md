## 什麼是 Python？

- **介紹**：Python 是一種通用程式語言，以簡單易用著稱。Python 被廣泛應用於資料科學和機器學習、網頁開發、腳本和自動化、嵌入式系統與物聯網等多個領域。
- **常見應用場景**：Python 用於資料科學、機器學習、網頁開發、資安、自動化以及像 Raspberry Pi 和相容 MicroPython 的微型電腦。

## 在你的區域環境中使用 Python

- **安裝**：在 Windows、Mac 和 Linux 上安裝 Python 的最佳方式是從官方 Python 網站下載安裝程式 (`https://www.python.org/`)。

## 變數

- **宣告變數**：要宣告一個變數，你先寫變數名稱，接著是指定運算子 (`=`)，然後是值。這個值可以是數字、字串、布林值等等。以下是一些範例：

```py
name = 'John Doe'
age = 25
```

- **變數命名規則**：以下是你應該遵守的變數命名規則：

  - 變數名稱只能以字母或底線（_）開頭，不能以數字開頭。
  - 變數名稱只能包含英文字母、數字（a-z、A-Z、0-9）和底線（_）。
  - 變數名稱區分大小寫 — `age`、`Age` 和 `AGE` 都被視為不同的名稱。
  - 變數名稱不能是 Python 的保留關鍵字，例如 `if`、`class` 或 `def`。
  - 多字變數名稱用底線分隔。例如 `snake_case`。

## 註解

- **單行註解**：這種類型的註解適合用來在你的程式碼中留下簡短的備註。

```py
# This is a single line comment
```

- **多行字串**：這種類型的字串可以用來留下較長的備註，或是用來註解掉程式碼區塊。

```py
"""
This is a multi-line string.
Here is some code commented out.

"""
```

- **`print()` 函式**：要將資料列印到主控台，你可以像這樣使用 `print()` 函式：

```py
print('Hello world!') # Hello world!
```

## Python 中常見的資料型別

- **介紹**：Python 是一種動態型別語言，和 JavaScript 一樣，這表示你不需要明確宣告變數的型別。語言會根據你指定給變數的值，自動判斷變數的型別。
- **整數**：沒有小數點的整數：

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **浮點數**：帶有小數點的數字：

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **字串**：用引號包住的一連串字元：

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **布林值**：代表 `True` 或 `False` 的值：

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **集合**：一個無序且元素唯一的群集：

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **字典**：由鍵值對組成的群集，用大括號包住：

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **元組**：不可變的有序群集，用小括號包住：

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **區間**：一連串的數字，常用於迴圈：

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **列表**：一個有序的元素群集，支援不同型別的資料：

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**：代表沒有值的特殊值：

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## 不可變與可變型別

- **不可變型別**：這些型別一旦宣告後就無法改變，雖然你可以將它們的變數指向新的東西，這稱為重新指定。它們包括整數、浮點數、複數、布林值、字串、元組、範圍和 `None`。
- **可變型別**：這些型別一旦宣告後可以改變。你可以新增、移除或更新它們的項目。它們包括列表、集合和字典等群集型別。
- **`type()` 函式**：要查看變數的型別，可以像這樣使用 `type()` 函式：

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **`isinstance()` 函式**：用來檢查變數是否符合特定資料型別：

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## 處理字串

- **定義**：如你從 JavaScript 記得的，字串是不可變的，這表示你無法在字串創建後更改它們。在 Python 中，你可以使用單引號或雙引號。建議選擇一種規則並堅持使用：

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **存取字串中的字元**：你可以使用中括號表示法來存取字串中的字元，如下所示：

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **字串跳脫**：如果你的字串包含引號，可以使用反斜線 (`\`) 來跳脫，如下所示：

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **字串串接**：要串接字串，你可以使用 `+` 運算子，如下所示：

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

另一種串接字串的方法是使用 `+=` 運算子。這用於在同一步驟中執行串接和指定，如下所示：

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**：這是格式化字串字面值的縮寫。它允許你處理插值，並且用簡潔且易讀的語法進行串接：

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **字串切片**：這是指你可以擷取字串的一部分。基本語法如下：

```py
str[start:stop:step]
```

起始位置代表擷取開始的索引。結束位置是切片結束的索引，該位置不包含在內。步進位置代表切片時的遞增間隔。以下是一些範例：

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **取得字串長度**：使用 `len()` 函式來傳回字串中字元的數量：

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## 使用 `in` 運算子

- **`in` 運算子**：這會傳回一個布林值，用來判斷該字元或字元群是否存在於字串中： 

```py
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

## 常用字串方法

- **`upper()`**：此字串方法傳回一個新字串，將所有字元轉換為大寫：

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`lower()`**：此字串方法傳回一個新字串，將所有字元轉換為小寫：

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`strip()`**：此字串方法傳回字串的複本，移除指定的前置和後置字元（如果沒有傳入引數，則移除前後的空白字元）。

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**：此方法傳回一個新字串，將所有舊字串的出現處替換為新字串。

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**：此方法用來使用指定的分隔符將字串拆分成列表。分隔符是一個字串，指定拆分的位置。

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**：此方法用來將可迭代物件的元素以分隔符串接成字串。可迭代物件是可以像列表、字串或元組一樣被迴圈處理的群集。

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`startswith(prefix)`**：此字串方法傳回布林值，表示字串是否以指定的前置式開始：

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`endswith(suffix)`**：此字串方法傳回布林值，表示字串是否以指定的後綴結尾：

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`find()`**：此字串方法傳回子字串第一次出現的索引。如果找不到，則傳回 `-1`：

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`count(substring)`**：此字串方法計算子字串在字串中出現的次數：

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`capitalize()`**：此字串方法傳回一個新字串，將第一個字元大寫，其餘字元轉為小寫：

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`isupper()`**：此字串方法如果字串中所有字母都是大寫，傳回 `True`，否則傳回 `False`：

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`islower()`**：此字串方法如果字串中所有字母都是小寫，傳回 `True`，否則傳回 `False`：

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`title()`**：此字串方法傳回一個新字串，將每個字的第一個字母大寫：

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`maketrans()`**：此字串方法用來建立一對一字元對應的轉換表。通常會搭配 `translate()` 方法使用，該方法將轉換表套用到字串並傳回轉換結果。

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## 常用於整數和浮點數的操作

- **基本數學運算**：在 Python 中，你可以對整數和浮點數進行基本的數學運算，包括加法、減法、乘法和除法：

```py
int_1 = 56
int_2 = 12
float_1 = 5.4
float_2 = 12.0

# Addition

print('Integer Addition:', int_1 + int_2) # Integer Addition: 68
print('Float Addition:', float_1 + float_2) # Float Addition: 17.4

print('Int Subtraction:', int_1 - int_2) # Int Subtraction: 44
print('Float Subtraction:',  float_2 - float_1) # Float Subtraction: 6.6

# Multiplication

print('Int Multiplication:', int_1 * int_2) # Int Multiplication: 672
print('Float Multiplication:', float_2 * float_1) # Float Multiplication: 64.80000000000001

# Division
```

當你將浮點數和整數相加時，結果會被轉換成浮點數，如下所示：

```py
print(int_1 + float_1) # 61.4
```

- **取餘數運算子（`%`）**：這會回傳一個數字除以另一個數字後的餘數：

```py
int_1 = 56
int_2 = 12

print(int_1 % int_2) # 8
```

- **整除運算子（`//`）**：這個運算子用來將兩個數字相除，並將結果向下取整到最接近的整數：

```py
int_1 = 56
int_2 = 12

print(int_1 // int_2) # 4
```

- **次方運算子（`**`）**：這個運算子用來將一個數字提升到另一個數字的次方：

```py
int_1 = 4
int_2 = 2

print(int_1 ** int_2) # 16
```

- **`float()` 函式**：你可以使用這個函式將整數轉換成浮點數。

```py
num = 4

print(float(num)) # 4.0
```

- **`int()` 函式**：你可以使用這個函式將浮點數轉換成整數。

```py
num = 4.0

print(int(num)) # 4
```

- **`round()` 函式**：這個函式用來將數字四捨五入到最接近的整數：

```py
num_1 = 3.4
num_2 = 7.7

print(round(num_1)) # 3
print(round(num_2)) # 8
```

- **`abs()` 函式**：這個函式用來回傳數字的絕對值：

```py
num = -13

print(abs(num)) # 13
```

- **`bin()` 函式**：這個函式用來將整數轉換成其二進位表示的字串：

```py
num = 56

print(bin(num))  # 0b111000
```

- **`oct()` 函式**：這個函式用來將整數轉換成其八進位表示的字串：

```py
num = 56

print(oct(num))  # 0o70
```

- **`hex()` 函式**：這個函式用來將整數轉換成其十六進位表示的字串：

```py
num = 56

print(hex(num))  # 0x38
```

- **`pow()` 函式**：這個函式用來將一個數字提升到另一個數字的次方：

```py
result = pow(2, 3) 
print(result)  # 8
```

## 增強型指派

- **定義**：增強型指派將二元操作與指派結合成一步。它會取一個變數，對它與另一個值執行操作，然後將結果存回同一個變數。

```py
# Addition assignment 
my_var = 10

print(my_var) # 15

# Subtraction assignment
count = 14

print(count) # 11

# Multiplication assignment 
product = 65

bits = 35
bits %= 2

print(bits) # 1

# Exponentiation assignment 
power = 2
power **= 3
```

還有其他增強型指派運算子，例如用於以 bit 為單元逐一操作的運算子。它們包括 `&=`、`^=`、`>>=` 和 `<<=`。

## 使用函式

- **定義**：函式是可重複使用的程式碼片段，會接受輸入（引數）並傳回輸出。要呼叫函式，你需要取用函式名稱，後面接一組括號：

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call
print(result) # 7
```

如果函式沒有明確傳回值，則預設的返回值是 `None`：

```py
def greet():
    print('hello') 

result = greet() # hello
print(result) # None
```

你也可以像這樣為參數提供預設值：

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

如果你呼叫函式時引數數量不正確，會得到 `TypeError`：

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

## Python 的作用域

- **區域作用域**：指在函式或類別內宣告的變數只能在該函式或類別內存取。

```py
def my_func():
    num = 10
    print(num)
```

- **包覆作用域**：指巢狀在另一個函式內的函式可以存取其所包覆函式的變數。

```py
def outer_func():
    msg = 'Hello there!'
    
    def inner_func():
        print(msg)
    inner_func()

print(outer_func()) # Hello there!
```

- **全域作用域**：指在任何函式或類別外宣告的變數，可以在程式的任何地方存取。

```py
tax = 0.70 

def get_total(subtotal):
    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **內建作用域**：Python 中保留給預定義函式、模組、關鍵字和物件的名稱。

```py
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```

## 使用 `if`、`elif` 和 `else` 述語

- **`if` 述語**：這些是用來判斷某件事是否為真的條件。如果條件評估結果為 `True`，那麼該區塊的程式碼就會執行。

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **`elif` 語句**：這些條件會跟在 `if` 述語之後。如果 `elif` 條件評估結果為 `True`，那麼該區塊的程式碼就會執行。

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **`else` 語句**：如果沒有其他條件評估為 `True`，這個語句就會執行。

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

你也可以像這樣使用巢狀的 `if` 述語：

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## 真值與假值

- **定義**：在 Python 中，每個值都有其固有的布林值，或是在邏輯背景關係中應被視為 `True` 或 `False` 的內建判斷。許多值被視為真值，也就是在邏輯背景關係中會被計算為 `True`。其他則是假值，意指它們會被計算為 `False`。以下是一些假值的範例：

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

其他像是非零數字和非空字串的值則是屬於真值。

## 使用 `bool()` 函式

- **定義**：如果你想檢查一個值是布林值的真（truthy）還是假（falsy），你可以使用內建的 `bool()` 函式。它會將一個值明確地轉換成其布林值等價，並對真值傳回 `True`，對假值傳回 `False`。以下是幾個範例：

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## 布林運算子與短路求值

- **定義**：這些是特殊的運算子，讓你能結合多個運算式，創建更複雜的決策邏輯。Python 中有三個布林運算子：`and`、`or` 和 `not`。
- **`and` 運算子**：此運算子接受兩個運算元，若第一個運算元為假值，則傳回第一個運算元，否則傳回第二個運算元。只有當兩個運算元皆為真值時，整個運算式才會傳回真值。

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

你也可以在條件判斷中使用 `and` 運算子，如下所示：

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **`or` 運算子**：此運算子若第一個運算元為真值，則傳回第一個運算元，否則傳回第二個運算元。只要至少有一個運算元為真值，`or` 運算式就會傳回真值。範例如下：

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

就像 `and` 運算子一樣，你可以在條件判斷中使用 `or` 運算子，如下所示：

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **短路求值**：`and` 和 `or` 運算子被稱為短路運算子。短路求值的意思是 Python 從左到右檢查值，一旦判定出最終結果就停止運算。
- **`not` 運算子**：此運算子接受單一運算元，並反轉其布林值。它會將真值轉為 `False`，將假值轉為 `True`。與前面介紹的運算子不同，`not` 總是傳回 `True` 或 `False`。以下是一些範例：

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

以下是 `not` 運算子在條件判斷中的範例：

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```

## Python 列表

- **介紹**：在 Python 中，列表資料型別是一個有序的元素序列，元素可以是字串、數字，甚至是其他列表。列表是可變的，且以零為基底的索引。

```python
cities = ['Los Angeles', 'London', 'Tokyo']
```

- **存取列表中的元素**：要從 `cities` 列表中存取元素，可以參考序列中的索引編號：

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # Los Angeles
```

- **使用負索引存取元素**：要存取任何列表的最後一個元素，可以使用 `-1` 作為索引編號：

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # Tokyo
```

- 負索引用於從列表尾端開始存取元素，而非從索引 `0` 的開頭開始。

- **使用 `list()` 建構式創建列表**：列表也可以使用 `list()` 建構式創建。`list()` 建構式用於將可迭代物件轉換成列表：

```python
developer = 'Jessica'

print(list(developer)) 
# Result: ['J', 'e', 's', 's', 'i', 'c', 'a']
```

- **取得列表長度**：你可以使用 `len()` 函式來取得列表的長度：

```python
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

- **列表的可變性**：列表是可變的，表示只要傳入有效的索引編號，就可以更新列表中的任何元素。要更新特定索引的列表元素，可以將新值指定給該索引：

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

- **索引超出範圍錯誤**：如果你傳入的索引（正向或負向）超出列表範圍，將會收到 `IndexError`：

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
"""
```

- **從列表中移除元素**：可以使用 `del` 關鍵字從列表中移除元素：

```python
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

- **檢查元素是否存在於列表中**：`in` 關鍵字可用來檢查元素是否存在於列表中：

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **巢狀列表**：列表可以巢狀在其他列表中：

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

- 要存取巢狀列表，需要使用索引 `2`，因為列表是以零為基底索引。

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

- 若要進一步存取該巢狀列表中的第二個語言，需要使用索引 `1`：

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # Rust
```

- **從列表中解包值**：從列表中解包值是一種將列表中的值指定給新變數的技巧。以下範例將 `developer` 列表解包到名為 `name`、`age` 和 `job` 的新變數中：

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer
```

- 如果指派運算子左側的變數數量與列表中項目總數不符，將會收到 `ValueError`。

- **收集列表中剩餘的項目**：要收集列表中剩餘的元素，可以使用星號（`*`）運算子，如下所示：

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer
```

- **列表切片**：切片是使用切片運算子 `:` 存取列表部分區段的概念。要切片一個從索引 `1` 開始，到索引 `3` 之前結束的列表，可以使用以下語法：

```python
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie']
desserts[1:3] # ['Cookies', 'Ice Cream']
```

- **步進間隔**：也可以指定步進間隔，決定索引間的遞增量。以下範例示範如何使用切片提取僅包含偶數的列表：

```python
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

## 列表方法

- **append()**：用來將一個項目添加到列表的末尾。以下是使用 `append()` 方法將數字 `6` 添加到這個 `numbers` 列表的範例：

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **附加列表**：`append()` 方法也可以用來將一個列表添加到另一個列表的末尾：

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

- **extend()**：用來將多個項目添加到列表的末尾。以下是將數字 `6`、`8` 和 `10` 添加到 `numbers` 列表末尾的範例：

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

- **insert()**：用來在列表的特定索引位置插入一個項目。以下是使用 `insert()` 方法的範例：

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

- **remove()**：用來從列表中移除一個項目。`remove()` 方法只會移除列表中第一個出現的該項目：

```py
numbers = [1, 2, 3, 4, 5, 5, 5]
numbers.remove(5)

print(numbers) # [1, 2, 3, 4, 5, 5]

```

- **pop()**：用來移除列表中特定的項目並傳回它：

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

- 如果你沒有為 `pop` 方法指定元素，則會移除最後一個元素。

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

- **clear()**：用來移除列表中的所有項目：

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

- **sort()**：`sort()` 方法用來就地排序元素。以下是就地排序一個隨機 `numbers` 列表的範例：

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

- **sorted()**：一個內建函式，會傳回一個新的排序後列表，而不會修改原始列表：

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
print(numbers) # [19, 2, 35, 1, 67, 41]
```

- **reverse()**：用來反轉列表中元素的順序：

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **index()**：用來尋找元素在列表中第一次出現的索引：

```python
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

- 如果使用 `index()` 方法找不到該元素，結果將會是 `ValueError`。

## Python 中的 Tuple（元組）

- **定義**：Tuple 是 Python 的一種資料型別，用來創建有序的值序列。Tuple 可以包含混合型別的資料：

```py
developer = ('Alice', 34, 'Rust Developer')
```

- Tuple 是不可變的，表示一旦建立後，元組中的元素無法更改。如果你嘗試更新元組中的某個項目，將會引發 `TypeError`：

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **存取元組中的元素**：要存取元組中的元素，使用中括號表示法和索引編號：

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

- 可以使用負索引從元組的尾端開始存取元素：

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- 如果你嘗試傳入超過或等於元組長度的索引編號，將會收到 `IndexError`：

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- 也可以使用 `tuple()` 建構式來建立元組。在建構式中，你可以傳入不同的可迭代物件，例如字串、列表，甚至其他元組。

```python
developer = 'Jessica'

print(tuple(developer)) 
# Result: ('J', 'e', 's', 's', 'i', 'c', 'a')
```

- **驗證元組中的項目**：要檢查某個項目是否在元組中，可以使用 `in` 關鍵字，如下所示：

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **解包元組**：可以像這樣從元組中解包項目：

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

- 如果你需要收集元組中剩餘的元素，可以使用星號（`*`）運算子，如下所示：

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer
```

- **切片元組**：切片可用來擷取元組的一部分。例如，項目 `pie` 和 `cookies` 可以被切片成一個獨立的元組：

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

- **從元組中移除項目**：嘗試從元組中移除項目會引發 `TypeError`，因為元組是不可變的：

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

- **什麼時候該用元組而不是列表？**：如果你需要一個可以動態新增、移除和更新元素的群集，應該使用列表。如果你知道你正在處理的是固定且不可變的資料群集，則應該使用元組。

## 常用的 Tuple 方法

- **count()**：用來判斷某個項目在 tuple 中出現了多少次。例如，你可以檢查語言 `'Rust'` 在 tuple 中出現了多少次：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

- 如果在 `count()` 方法中指定的項目在 tuple 中完全不存在，則返回值會是 `0`：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

- 如果沒有傳入任何引數給 `count()` 方法，Python 會傳回 `TypeError`。

- **index()**：用來尋找特定項目在 tuple 中出現的索引位置。以下是使用 `index()` 方法尋找語言 `'Java'` 索引的範例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

- 如果找不到指定的項目，Python 會傳回 `ValueError`。

- 你可以傳入選用的起始索引給 `index()` 方法，指定從 tuple 的哪個位置開始搜尋該項目：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

- 你也可以傳入選用的結束索引給 `index()` 方法，指定在 tuple 中搜尋該項目時停止的位置：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

- **sorted()**：用來對任何可迭代物件的元素進行排序，並返回一個新的排序後列表。以下是使用 `sorted()` 函式創建一個新的數字列表的範例：

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

- **修改排序行為**：你可以使用選用的 `reverse` 和 `key` 引數來自訂可迭代物件的排序行為。以下是使用 `key` 引數依照長度對 tuple 中項目排序的範例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

- 你可以使用 `reverse` 引數，像這樣創建一個反向排序的新列表：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

## Python 中的迴圈

- **定義**：迴圈用來重複執行一段程式碼區塊指定的次數。

- **`for` 迴圈**：用來對序列（例如列表、元組或字串）進行迭代，並對序列中的每個項目執行一段程式碼。以下是使用 `for` 迴圈遍歷列表並將每個語言列印到主控台的範例：

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)

"""
Result 

Rust
Java
Python
C++
"""
```

- 以下是使用 `for` 迴圈遍歷字串 `code` 並列印出每個字元的範例：

```py
for char in 'code':
    print(char)

"""
Result 

c
o
d
e
"""
```

- `for` 迴圈可以巢狀使用。以下是使用巢狀 `for` 迴圈的範例：

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)

"""
Result

Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
"""

```

- **`while` 迴圈**：重複執行一段程式碼直到條件為 `False`。以下是使用 `while` 迴圈進行猜數字遊戲的範例：

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')

"""
Result

Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
"""

```

- **`break` 和 `continue` 述語**：用於迴圈中以修改迴圈的執行流程。

- `break` 述語用來在符合特定條件時立即離開迴圈。以下是使用 `break` 述語處理 `developer_names` 列表的範例：

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

- `continue` 述語用來跳過當前的迭代，直接進入下一次迭代。以下是使用 `continue` 述語取代 `break` 述語的範例：

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

- `for` 和 `while` 迴圈都可以搭配 `else` 子句使用，該子句僅在迴圈未被 `break` 終止時執行：

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

## 範圍與它們在迴圈中的使用

- **`range()` 函式**：用來產生一連串整數。

```py
range(start, stop, step)
```

- 必須的 `stop` 引數是一個整數（不包含該值），代表所產生數字序列的終點。以下是使用 `range()` 函式的範例：

```py
for num in range(3):
    print(num)
```

- 如果沒有指定 `start` 引數，預設值將是 `0`。預設情況下，整數序列會以 `1` 為增量遞增。你可以使用選用的 `step` 引數來更改預設的增量值。以下是產生從 2 開始到不包含 11（即包含 10）的偶數序列的範例：

```py
for num in range(2, 11, 2):
    print(num)
```

- 如果你沒有為 `range()` 函式提供任何引數，則會得到一個 `TypeError`。

- `range()` 函式只接受整數作為引數，不接受浮點數。使用浮點數也會導致 `TypeError`：

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- 你可以使用負整數作為 `step` 引數，來產生遞減的整數序列：

```py
for num in range(40, 0, -10):
    print(num)
```

- `range()` 函式也可以搭配 `list` 建構式用來創建整數列表。`list` 建構式用於將可迭代物件轉換成列表。以下是產生包含 2 到 10 之間偶數的列表範例：

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

## Python 中的 `enumerate()` 和 `zip()` 函式

- **`enumerate()`**：用來迭代序列並追蹤該序列中每個項目的索引。`enumerate()` 函式接受一個可迭代物件作為引數，並傳回一個 `enumerate` 物件，該物件包含可迭代物件中每個項目的索引和值。

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')

# Result
# Index 0 and language Spanish
# Index 1 and language English
# Index 2 and language Russian
# Index 3 and language Chinese

```

- `enumerate()` 函式也可以在 `for` 迴圈之外使用：

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

print(list(enumerate(languages)))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

- `enumerate()` 函式還接受一個選用的 `start` 引數，用來指定計數的起始值。如果省略此引數，計數將從 `0` 開始。

- **`zip()`**：用來平行迭代多個可迭代物件。以下是使用 `zip()` 函式平行迭代 `developers` 和 `ids` 的範例：

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

Result

ID: 2
Name: Jessica

ID: 4

```

## Python 中的列表推導式

- **定義**：列表推導式允許你在一行內創建一個新列表，透過將迴圈和條件直接結合在方括號中。這使程式碼更簡潔，且通常更容易閱讀。

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

## 可迭代函式

- **`filter()`**：用來根據條件篩選可迭代物件中的元素。它傳回一個只包含符合條件元素的迭代器。以下是一個建立只包含字元數超過四個字的字的列表範例：

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**：用來對可迭代物件中的每個項目套用一個函式，並傳回一個包含結果的新可迭代物件。以下是一個使用 `map()` 函式將攝氏溫度列表轉換為華氏溫度的範例：

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**：用來取得像列表或元組這類可迭代物件的總和。以下是一個使用 `sum()` 函式的範例：

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- 你也可以傳入一個選用的 `start` 引數，設定總和的初始值。以下是一個使用 `start` 引數作為位置引數的更新範例：

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- 你也可以選擇像這樣使用 `start` 引數作為關鍵字引數：

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## Lambda 函式

- **定義**：Python 中的 lambda 函式是一種簡潔的方式，用來創建沒有名稱的函式（匿名函式）。
- lambda 函式經常用作另一個函式的引數。以下是一個 lambda 函式的範例：

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
```

- 使用 lambda 函式的最佳實踐包括不要將它們指定給變數，保持簡單且易讀，並用於短小且一次性的函式。

## 字典

- **字典**：字典是內建的資料結構，用來儲存鍵值對群集。鍵必須是不可變的資料型別。以下是 Python 字典的一般語法：

```python
dictionary = {
    key1: value1,
    key2: value2
}
```

- **`dict()` 建構式**：`dict()` 建構式是建置字典的另一種方式。你將一個包含多個元組的列表作為引數傳入 `dict()` 建構式。這些元組的第一個元素是鍵，第二個元素是值。

```python
pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
```

- **中括號表示法**：要存取鍵值對的值，可以使用稱為中括號表示法的語法。

```python
dictionary[key]
```

## 常用字典方法

- **`get()` 方法**：`get()` 方法會取得與鍵相關的值。它類似於中括號表示法，但允許你設定預設值，避免當鍵不存在時發生錯誤。

```python
dictionary.get(key, default)
```

- **`keys()` 和 `values()` 方法**：`keys()` 和 `values()` 方法分別傳回包含字典中所有鍵和值的檢視物件。檢視物件是一種查看字典內容的方式，不會建立資料的獨立複本。

```python
pizza = {
    'name': 'Margherita Pizza',
    'price': 8.9,
    'calories_per_slice': 250
}

pizza.keys()
# dict_keys(['name', 'price', 'calories_per_slice'])

pizza.values()
# dict_values(['Margherita Pizza', 8.9, 250])
```

- **`items()` 方法**：`items()` 方法傳回包含字典中所有鍵值對的檢視物件，包含鍵和值。

```python
pizza.items()
# dict_items([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250)])
```

- **`clear()` 方法**：`clear()` 方法會移除字典中所有的鍵值對。

```python
pizza.clear()
```

- **`pop()` 方法**：`pop()` 方法會移除第一個引數指定的鍵值對，並傳回其值。如果鍵不存在，則傳回第二個引數指定的預設值。如果鍵不存在且未指定預設值，則會引發 `KeyError`。

```python
pizza.pop('price', 10)
pizza.pop('total_price') # KeyError
```

- **`popitem()` 方法**：在 Python 3.7 及以上版本中，`popitem()` 方法會移除最後插入的項目。

```python
pizza.popitem()
```

- **`update()` 方法**：`update()` 方法會用另一個字典的鍵值對更新字典。如果有相同的鍵，則其值會被覆寫。新的鍵會被加入字典，成為新的鍵值對。

```python
pizza.update({ 'price': 15, 'total_time': 25 })
```

## 迭代字典

- **迭代值**：如果你需要迭代字典中的值，可以使用 `for` 迴圈搭配 `values()` 來取得字典的所有值。

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,
    'Headphones': 70,
}

for price in products.values():
    print(price)
```

輸出：

```md
990
600
250
70
```

- **迭代鍵**：如果你需要迭代上述 `products` 字典的鍵，可以直接寫 `products.keys()` 或 `products`。

```python
for product in products.keys():
    print(product)
    
# Or

for product in products:
    print(product)
```

輸出：

```md
Laptop
Smartphone
Tablet
Headphones
```

- **迭代鍵值對**：如果你需要同時迭代鍵和對應的值，可以迭代 `products.items()`。你會得到包含鍵和對應值的個別元組。

```python
for product in products.items():
    print(product)
```

輸出：

```md
('Laptop', 990)
('Smartphone', 600)
('Tablet', 250)
('Headphones', 70)
```

要將鍵和值分別存到不同的迴圈變數，必須用逗號分隔。第一個變數存鍵，第二個存值。

```python
for product, price in products.items():
    print(product, price)
```

輸出：

```md
Laptop 990
Smartphone 600
Tablet 250
Headphones 70
```

- **`enumerate()` 函式**：如果你需要在迭代字典時同時追蹤計數器，可以呼叫 `enumerate()` 函式。該函式會傳回一個 `enumerate` 物件，會為每個項目指定一個整數，像是計數器。你可以從任意數字開始計數，但預設是從 0 開始。

將索引和項目指定給不同的迴圈變數是使用 `enumerate()` 的常見方式。例如，使用 `products.items()`，你可以取得索引以外的整個鍵值對：

```python
for index, product in enumerate(products.items()):
    print(index, product)
```

輸出：

```md
0 ('Laptop', 990)
1 ('Smartphone', 600)
2 ('Tablet', 250)
3 ('Headphones', 70)
```

要自訂計數的初始值，可以傳入第二個引數給 `enumerate()`。例如，這裡我們從 1 開始計數。

```python
for index, product in enumerate(products.items(), 1):
    print(index, product)
```

輸出：

```md
1 ('Laptop', 990)
2 ('Smartphone', 600)
3 ('Tablet', 250)
4 ('Headphones', 70)
```

## 集合

- **集合**：集合是 Python 內建的資料結構，不允許重複的值。集合是可變的且無序的，這表示其元素不會以特定順序儲存，因此你無法使用索引或鍵來存取它們。此外，集合只能包含不可變型別的值，例如數字、字串和元組。

- **定義集合**：要定義集合，需要將元素寫在大括號內並以逗號分隔。

```python
my_set = {1, 2, 3, 4, 5}
```

- **定義空集合**：如果你需要定義空集合，必須使用 `set()` 函式。僅寫空的大括號會自動建立字典。

```python
set() # Set
{}    # Dictionary
```

## Common Set Methods

- **`add()` 方法**：你可以使用 `add()` 方法將元素添加到集合中，並將新元素作為引數傳入。

```python
my_set.add(6)
```

- **`remove()` 和 `discard()` 方法**：要從集合中移除元素，你可以使用 `remove()` 方法或 `discard()` 方法，並將你想移除的元素作為引數傳入。`remove()` 方法如果找不到該元素會引發 `KeyError`，而 `discard()` 方法則不會。

```python
my_set.remove(4)
my_set.discard(4)
```

- **`clear()` 方法**：`clear()` 方法會移除集合中的所有元素。

```python
my_set.clear()
```

## 數學集合操作

- **`issubset()` 和 `issuperset()` 方法**：`issubset()` 和 `issuperset()` 方法分別用來檢查一個集合是否為另一個集合的子集合或超集合。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 5}

print(your_set.issubset(my_set)) # True
print(my_set.issuperset(your_set)) # True
```

- **`isdisjoint()` 方法**：`isdisjoint()` 方法用來檢查兩個集合是否互斥，即是否沒有共同的元素。

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

print(my_set.isdisjoint(your_set)) # True
```

- **聯集運算子（`|`）**：聯集運算子 `|` 傳回一個包含兩個集合所有元素的新集合。

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

my_set | your_set # {1, 2, 3, 4, 5, 6}
```

- **交集運算子（`&`）**：交集運算子 `&` 傳回一個只包含兩個集合共有元素的新集合。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set & your_set # {2, 3, 4}
```

- **差集運算子（`-`）**：差集運算子 `-` 傳回一個包含第一個集合中但不在另一個集合中的元素的新集合。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set - your_set # {1, 5}
```

- **對稱差集運算子（`^`）**：對稱差集運算子 `^` 傳回一個包含只在第一個集合或第二個集合中，但不在兩者共有的元素的新集合。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set ^ your_set # {1, 5, 6}
```

- **`in` 運算子**：你可以用 `in` 運算子檢查一個元素是否在集合中。

```python
print(5 in my_set)
```

## Python 標準程式庫

- **Python 標準程式庫**：函式庫提供你預先撰寫且可重複使用的程式碼，例如函式、類別和資料結構，讓你能在專案中重複利用。Python 擁有豐富的標準程式庫，內建模組實作了許多問題和任務的標準化解法。一些常見的內建模組範例有 `math`、`random`、`re`（「regular expressions」的縮寫）和 `datetime`。

## 匯入述語

- **匯入述語**：要存取內建模組中定義的元素，你需要使用匯入述語。匯入述語通常寫在檔案的最上方。匯入述語對函式、類別、常數、變數以及模組中定義的任何其他元素都適用。

- **基本匯入述語**：你可以使用 `import` 關鍵字，後面接模組名稱：

```python
import module_name
```

接著，如果你需要呼叫該模組中的函式，就用點記法，先寫模組名稱，再寫函式名稱。

```python
module_name.function_name()
```

例如，你可以在程式碼中寫下以下內容，匯入 `math` 模組並取得 36 的平方根：

```python
import math

math.sqrt(36)
```

- **以不同名稱匯入模組**：如果你需要用不同名稱（也稱為「別名」）匯入模組，可以在匯入述語的最後使用 `as`，後面接別名。這通常用於模組名稱過長或避免命名衝突。

```python
import module_name as module_alias
```

例如，若要在程式碼中將 `math` 模組稱為 `m`，可以這樣指定別名：

```python
import math as m
```

接著，你就可以用別名來存取該模組的元素：

```python
m.sqrt(36)
```

- **匯入特定元素**：如果你不需要整個模組，可以使用 `from` 匯入特定元素。此時，匯入述語以 `from` 開頭，接著是模組名稱，再用 `import` 關鍵字，最後列出你想匯入的元素名稱。

```python
from module_name import name1, name2
```

然後，你就可以在 Python 腳本中直接使用這些名稱，而不需加上模組前綴。例如：

```python
from math import radians, sin, cos

angle_degrees = 40
angle_radians = radians(angle_degrees)

sine_value = sin(angle_radians)
cos_value = cos(angle_radians)

print(sine_value) # 0.6427876096865393
print(cos_value)  # 0.766044443118978
```

這很方便，但如果你已有同名的函式或變數，可能會造成命名衝突。選擇匯入述語類型時要注意這點。

如果你想為這些名稱指定別名，也可以使用 `as` 關鍵字，後面接別名。

```python
from module_name import name1 as alias1, name2 as alias2
```

- **使用星號的匯入述語（`*`）**：星號告訴 Python 你想匯入該模組的所有內容，但不想使用模組名稱作為前綴。

```python
from module_name import *
```

例如，如果你用這種方式匯入 `math` 模組，就能呼叫該模組中定義的任何函式，而不必指定模組名稱作為前綴。

```python
from math import *
print(sqrt(36))  # 6.0
```

不過，這通常不建議使用，因為它可能導致命名空間衝突，並且讓你更難判斷名稱的來源。

## `if __name__ == '__main__'`

- **`__name__` 變數**：`__name__` 是 Python 中一個特殊的內建變數。當 Python 檔案被直接執行時，Python 會將此變數的值設為字串 `"__main__"`。但如果該 Python 檔案被匯入為另一個 Python 程式的模組，`__name__` 變數的值則會被設為該模組的名稱。

這就是為什麼你常會在 Python 程式中看到這個條件判斷。它包含了你只想在 Python 程式作為主程式執行時**才**執行的程式碼。

```python
if __name__ == '__main__': 
    # Code
```

## Python 常見錯誤

- **SyntaxError**：當你的程式碼不符合 Python 的語法規則時，Python 會引發此錯誤。舉例來說，程式碼 `print("Hello there"` 會導致語法錯誤，錯誤訊息為 `SyntaxError: '(' was never closed`，因為程式碼缺少了結尾的括號。
- **NameError**：當你嘗試存取尚未定義的變數或函式時，Python 會引發 `NameError`。例如，如果你的程式碼中有 `print(username)` 這一行，但事先沒有定義 `username` 變數，就會出現名稱錯誤，錯誤訊息為 `NameError: name 'username' is not defined`。
- **TypeError**：當你對兩個或多個不相容的資料型別執行操作時，Python 會丟出此錯誤。舉例來說，如果你嘗試將字串與數字相加，就會得到錯誤 `TypeError: can only concatenate str (not "int") to str`。
- **IndexError**：如果你存取列表或其他序列（如 tuple 和字串）中不存在的索引，就會得到 `IndexError`。例如，在 `Hello world` 字串中，最後一個字元的索引是 `10`。如果你嘗試用 `greet = "hello world"; print(greet[11])` 這種方式存取字元，就會出現錯誤，錯誤訊息為 `IndexError: string index out of range`。
- **AttributeError**：當你嘗試使用某個物件型別中不存在的方法或屬性時，Python 會引發此錯誤。例如，在字串 `"hello".append("!")` 上呼叫 `.append()`，會導致錯誤，錯誤訊息為 `AttributeError: 'str' object has no attribute 'append'`。

## Python 中良好的除錯技巧

- **使用 `print` 函式**：在除錯時，於程式碼的不同位置插入 `print()` 述語，有助於你查看變數的值以及程式碼的執行流程。
- **使用 Python 內建的除錯器（`pdb`）**：Python 提供了一個 `pdb` 模組用於除錯。它是 Python 標準程式庫的一部分，因此隨時可用。透過 `pdb`，你可以使用 `set_trace()` 函式設定追蹤，開始逐步執行程式碼並以互動方式檢查變數。
- **善用整合開發環境的除錯工具**：許多整合開發環境（IDE）和程式碼編輯器，如 Pycharm 和 VS Code，都提供具備中斷點、逐步執行、變數檢查及其他除錯功能的除錯工具。

## 例外處理

- **`try...except`**：用來執行可能會引發例外的程式區塊。`try` 區塊是你預期可能會發生錯誤的地方，而 `except` 區塊會在指定的錯誤被引發時執行。範例如下：

  ```py
  try:
    print(22 / 0)
  except ZeroDivisionError:
    print('You can\'t divide by zero!')
    # You can't divide by zero!
  ```

  你也可以串接多個 `except` 區塊，這樣就能處理更多種類的例外：

  ```py
  try:
    number = int(input('Enter a number: '))
    print(22 / number)
  except ZeroDivisionError:
    print('You cannot divide by zero!')
    # You cannot divide by zero! prints when you enter 0
  except ValueError:
    print('Please enter a valid number!')
    # Please enter a valid number! prints when you don't enter an integer
  ```

- **`else` 和 `finally`**：這些區塊是 `try...except` 的延伸。如果沒有發生例外，`else` 區塊會執行。`finally` 區塊則無論是否有錯誤都會執行。

  ```py
  try:
    result = 100 / 4
  except ZeroDivisionError:
    print('You cannot divide by zero!') # This will not run
  else:
    print(f'Result is {result}') # Result is 25.0
  finally:
    print('Execution complete!') # Execution complete!
  ```

- **例外物件**：讓你能存取例外本身，以便更好地偵錯和列印直接的錯誤訊息。要存取例外物件，需要使用 `as` 關鍵字。範例如下：

  ```py
  try:
      value = int('This will raise an error')
  except ValueError as e:
      print(f'Caught an error: {e}')
      # Caught an error: invalid literal for int() with base 10: 'This will raise an error'
  ```

- **`raise` 述語**：允許你手動引發例外。當某個條件成立時，可以用它來丟出例外。範例如下：

  ```py
  def divide(a, b):
      if b == 0:
          raise ZeroDivisionError('You cannot divide by zero')
      return a / b
  ```

## 例外訊號傳遞
  
  `raise` 述語在你創建自定義例外時也非常有用，因為你可以用它來丟出帶有自訂訊息的例外。以下是一個範例：

  ```py
  class InvalidCredentialsError(Exception):
      def __init__(self, message="Invalid username or password"):
          self.message = message
          super().__init__(self.message)

  def login(username, password):
      stored_username = "admin"
      stored_password = "password123"
      
      if username != stored_username or password != stored_password:
          raise InvalidCredentialsError()
      
      return f"Welcome, {username}!"
  ```  

  這是你如何使用 `login` 函式搭配 `InvalidCredentialsError` 例外的範例：

  ```py
  # 登入失敗嘗試
  try:
      message = login("user", "wrongpassword")
  except InvalidCredentialsError as e:
      print(f"Login failed: {e}")
  else:
      print(message)

  # 登入成功嘗試
  try:
      message = login("admin", "password123")
  except InvalidCredentialsError as e:
      # 這個區塊不會執行，因為登入成功了
      print(f"Login failed: {e}")
  else:
      # 如果 'try' 區塊沒有發生例外，則執行 else 區塊
      print(message)
  ```

  `raise` 述語也可以搭配 `from` 關鍵字使用，以串鏈例外，顯示不同錯誤之間的關聯：

  ```py
  def parse_config(filename):
    try:
        with open(filename, 'r') as file:
            data = file.read()
            return int(data)
    except FileNotFoundError:
        raise ValueError('Configuration file is missing') from None
  ```

## Python 類別和物件

- **類別定義**：類別是用來創建物件的藍圖。它透過屬性和方法定義物件將擁有的行為。以下是在 Python 中類別定義的基本範例：

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f'{self.name.upper()} says woof woof!')
```

- **創建物件**：物件是類別的實體。它們是透過呼叫類別並傳入必要的引數來創建的。

```python
dog1 = Dog('Jack', 3)
dog2 = Dog('Thatcher', 5)

dog1.bark()  # JACK says woof woof!
dog2.bark()  # THATCHER says woof woof!
```

- **對物件呼叫方法**：你可以對物件呼叫方法來執行動作或取得資訊。

```python
object_name1.method_name()
object_name2.method_name()
```

- **類別和物件的差異**：類別是可重複使用的模板，而物件是該類別具有實際資料的特定實體。

## 屬性

```python
class Dog:
```

## Dunder（魔法）方法

- **定義**：以雙底線開頭和結尾的特殊方法（例如 `__init__`、`__len__`、`__str__`、`__eq__`）。Python 在內部用它們來執行內建操作。

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **間接呼叫 dunder 方法**：你不需要直接呼叫 dunder 方法。當某些操作發生時，Python 會自動呼叫它們。這些操作包括：

  - **算術操作，例如加法、減法、乘法、除法等**。加法會呼叫 `__add__()`，減法呼叫 `__sub__()`，乘法呼叫 `__mul__()`，除法呼叫 `__truediv__()`。
        
  - **字串操作，例如串接、重複、格式化和轉換成文字**。串接會呼叫 `__add__()`，重複呼叫 `__mul__()`，格式化呼叫 `__format__()`，轉換成文字則呼叫 `__str__()` 和 `__repr__()`，等等。
        
  - **比較操作，例如相等、小於、大於等**。相等檢查會呼叫 `__eq__()`，小於呼叫 `__lt__()`，大於呼叫 `__gt__()`，等等。
        
  - **迭代操作，例如讓物件可迭代以及取得下一個項目**。呼叫 `__iter__()` 來傳回迭代器，呼叫 `__next__()` 來取得下一個項目。

## 真實世界範例：購物車

- **帶有 Dunder 方法的購物車類別**：允許使用內建行為來新增、移除、迭代和檢查內容。

```python
class Cart:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

    def remove(self, item):
        if item in self.items:
            self.items.remove(item)
        else:
            print(f'{item} is not in cart')

    def list_items(self):
        return self.items

    def __len__(self):
        return len(self.items)

    def __getitem__(self, index):
        return self.items[index]

    def __contains__(self, item):
        return item in self.items

    def __iter__(self):
        return iter(self.items)

cart = Cart()
cart.add('Laptop')

```

## 什麼是物件導向程式設計？

- **物件導向程式設計**：一種程式設計風格，開發者將程式碼中的所有事物視為現實世界中的物件。它通常被稱為 OOP。幫助你有效組織和管理程式碼的四大關鍵原則是 **封裝**、**繼承**、**多型** 和 **抽象性**
- **類別**：用來創建物件的藍圖。每個從類別創建的物件都有定義資料的屬性和決定物件行為的方法。

## 什麼是封裝？

- **封裝**：將物件的屬性和方法打包成一個單一單元。它讓你能夠將物件的內部狀態隱藏在一組簡單的公開方法和屬性背後，這些公開方法和屬性就像門一樣。在這些門的背後，是控制資料如何變化以及誰能看到的私有屬性和方法。
- **封裝範例**：如果你想追蹤錢包餘額，你會允許存款和提款，但你不希望任何人直接竄改錢包餘額本身：

```py
class Wallet:
   def __init__(self, balance):
       self.__balance = balance # Private attribute

   def deposit(self, amount):
       if amount > 0:
           self.__balance += amount # Add to the balance safely

   def withdraw(self, amount):
       if 0 < amount <= self.__balance:
           self.__balance -= amount # Remove from the balance safely

account = Wallet(500)
print(account.__balance) # AttributeError: 'Wallet' object has no attribute '__balance'
```

- **屬性前置單底線與雙底線的差異**：屬性和方法前置單底線表示它們是供內部使用的慣例，並不會阻止外部存取。屬性和方法前置雙底線則有效防止它們被類別外部存取。

## 什麼是取值函式和設值函式？

- **取值函式和設值函式**：讓你控制類別屬性如何被存取和修改的方法。你用取值函式來取得值，用設值函式來設定值。
- **屬性**：它們連結取值函式和設值函式，並允許存取資料。當你取得、設定或刪除值時，屬性會在背後執行額外的邏輯。
- **為什麼用屬性而不是方法**：屬性用來取代方法，以提升可讀性和讓程式碼更乾淨。它們讓你像存取一般屬性一樣，用點記法存取值，而不需要小括弧。
- **創建取值函式**：要創建取值函式，你使用 `@property` 裝飾器。這是一個取得圓半徑的取值函式範例：

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self): # A getter to get the radius
        return self._radius
  
    @property
    def area(self):  # A getter to calculate area
        return 3.14 * (self._radius ** 2)

my_circle = Circle(3)

print(my_circle.radius) # 3
print(my_circle.area) # 28.26
```

在 `__init__()` 中使用 self.radius 確保在物件創建時會執行設值函式，因此無效的半徑值會立即被捕捉。

- **創建設值函式**：要創建設定半徑的設值函式，你必須定義另一個同名方法，並在它上方使用 `@<property_name>.setter`：

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self):  # A getter to get the radius
        return self._radius

    @radius.setter
    def radius(self, value):  # A setter to set the radius
        if value <= 0:
            raise ValueError('Radius must be positive')
        self._radius = value

my_circle = Circle(3)
print('Initial radius:', my_circle.radius) # Initial radius: 3

my_circle.radius = 8
print('After modifying the radius:', my_circle.radius) # After modifying the radius: 8
```

- **Python 如何處理取值函式和設值函式**：一旦你定義了取值函式和設值函式，Python 會在你用一般屬性語法這樣使用時，自動在背後呼叫它們：

```py
my_circle.radius # This will call the getter
my_circle.radius = 4 # This will call the setter
```

在設值函式內，你不應該對屬性名稱本身指定值，因為那會導致 `RecursionError`。

- **刪除器**：在用設值函式和取值函式設定和取得值之後，你可以用 `deleter` 來控制如何刪除它。刪除器會在你對屬性使用 `del` 述語時執行自訂邏輯。要創建刪除器，你使用 `@<property_name>.deleter` 裝飾器。

```py
  # Deleter
    @radius.deleter
    def radius(self):
        print("Deleting radius...")
        del self._radius
```

## 什麼是繼承以及它如何促進程式碼重複使用？

- **繼承**：子類別使用父類別的屬性和方法的過程。繼承促進程式碼重複使用，提供清晰的階層結構，並且能在不重寫所有內容的情況下自訂行為。要實作繼承，子類別需要帶入父類別的名稱：

```py
class Parent:
    # Parent attributes and methods

class Child(Parent):
    # Child inherits, extends, and/or overrides where necessary
```

- **單繼承與多繼承**：當子類別從單一父類別繼承屬性和方法時，如上所示，這個過程稱為**單繼承**。當子類別從多個父類別繼承屬性和方法時，則稱為**多繼承**。語法如下：

```py
class Parent:
    # Attributes and methods for Parent

class Child:
    # Attributes and methods for Child

class GrandChild(Parent, Child):
    # GrandChild inherits from both Parent and Child
    # GrandChild can combine or override behavior from each
```

- **`super()` 函式**：一個讓你在類別對該方法有不同實作或擴充該方法時，能**呼叫**父類別方法的函式，避免重複撰寫程式碼。

## 什麼是多型以及它如何促進程式碼重複使用？

- **多型**：物件導向程式設計的原則，允許不同類別使用相同的方法名稱，但每個類別在呼叫時會有不同的實作。語法如下：

```py
class A:
   def action(self): ...

class B:
   def action(self): ...

class C:
   def action(self): ...

Class().method()  # Works for A, B, or C
```

- **基於繼承的多型**：父類別設定一個方法，每個子類別根據自己的需求進行調整。

## 什麼是名稱改寫（Name Mangling）以及它如何運作？

- **名稱改寫**：Python 內部將以雙底線開頭的屬性重新命名的過程，透過在前面加上一個底線和類別名稱作為前置式，將 `__attribute` 變成 `_ClassName__attribute`。
- **名稱改寫的目的**：名稱改寫的主要目的是防止在使用繼承時，屬性和方法被意外覆寫。以下程式碼能讓你更容易理解： 

```py
class Parent:
    def __init__(self):
        self.__data = 'Parent data'

class Child(Parent):
    def __init__(self):
        super().__init__()
        self.__data = 'Child data'

c = Child()
print(c.__dict__) # {'_Parent__data': 'Parent data', '_Child__data': 'Child data'}
```

## 什麼是抽象性以及它如何幫助維持複雜系統的組織？

- **抽象性**：一種程式設計概念，將物件或系統的複雜實作細節隱藏，只顯示必要的特性。在 Python 和其他程式語言中，抽象性透過提升可重複使用性來簡化複雜系統。
- **抽象性的範例**：日常生活中抽象性的好例子是汽車，讓你只需使用方向盤、踏板和排檔桿，而不必了解引擎或煞車的運作方式。
- **Python 如何實作抽象性**：Python 透過 `abc` 模組實作抽象性。該模組提供 `ABC` 類別（抽象基礎類別）和 `@abstractmethod` 裝飾器。抽象基礎類別（ABC）定義子類別必須實作的共通方法和屬性，且無法被具現化。
- **抽象方法的定義方式**：抽象方法以 `@abstractmethod` 定義，必須在子類別中覆寫，即使它有預設實作。抽象性的基本語法如下： 

```py
from abc import ABC, abstractmethod

# Define an abstract base class
class AbstractClass(ABC):
    @abstractmethod
    def abstract_method(self):
        pass

# Concrete subclass that implements the abstract method
class ConcreteClassOne(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassOne')

# Another concrete subclass
class ConcreteClassTwo(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassTwo')
```

## 演算法與大 O 符號

- **演算法**：一組明確無誤的指令，用來解決問題或執行任務。演算法必須在有限步驟內完成，且每個步驟都必須精確且明確。

- **大 O 符號**：描述演算法在最壞情況下的效率或成長速率，隨著輸入大小增加。它關注資源使用如何隨輸入大小成長，忽略常數因子和低階項。

### 常見時間複雜度

- **O(1) - 常數時間**：演算法所需時間與輸入大小無關，始終相同。

```python
def check_even_or_odd(number):
    if number % 2 == 0:
        return 'Even'
    else:
        return 'Odd'
```

- **O(log n) - 對數時間**：隨著輸入增長，時間緩慢增加。常見於反覆將問題規模縮小一定比例的演算法（如二分搜尋法）。

- **O(n) - 線性時間**：執行時間與輸入大小成正比增加。

```python
for grade in grades:
    print(grade)
```

- **O(n log n) - 線性對數時間**：高效排序演算法如合併排序和快速排序的常見時間複雜度。

- **O(n²) - 平方時間**：執行時間以平方速度增加。常見於巢狀迴圈。

```python
for i in range(n):
    for j in range(n):
        print("Hello, World!")
```

### 空間複雜度

- **O(1) - 常數空間**：演算法使用的記憶體與輸入大小無關，始終相同。
- **O(n) - 線性空間**：記憶體使用量隨輸入大小成正比增加。
- **O(n²) - 平方空間**：記憶體使用量以平方速度隨輸入大小增加。

- **理解問題**：多次閱讀問題描述。辨識輸入、預期輸出，以及如何將輸入轉換成輸出。

- **偽程式碼**：對演算法邏輯的高階描述，與程式語言無關。使用一般書寫語言混合程式構件，如 `IF`、`ELSE`、`FOR`、`WHILE`。

```md
GET original_string
SET reversed_string = ""
FOR EACH character IN original_string:
  ADD character TO THE BEGINNING OF reversed_string
```

- **邊界情況**：演算法應處理範圍邊界上的特定有效輸入。務必考慮並測試邊界情況。

## 陣列

- **靜態陣列**：在初始化時大小固定。元素存放在相鄰的記憶體位置。程式執行期間大小無法改變。

- **動態陣列**：可以在程式執行期間自動增長或縮小。透過自動複製到更大的陣列來處理調整大小的需求。

### Python 列表（動態陣列）

```python
numbers = [3, 4, 5, 6]

# Access elements
numbers[0]  # 3

# Update elements
numbers[2] = 16

# Add elements
numbers.append(7)
numbers.insert(3, 15)  # Insert at specific index

# Remove elements
numbers.pop(2)  # Remove at specific index
numbers.pop()   # Remove last element
```

### 動態陣列的時間複雜度

- **存取**：O(1)
- **在尾端插入**：平均 O(1)，需要調整大小時為 O(n)
- **在中間插入**：O(n)
- **刪除**：中間為 O(n)，尾端為 O(1)

## 堆疊

- **堆疊**：後進先出（LIFO）的資料結構。元素只能從頂端添加和移除。

- **推入操作**：將元素添加到堆疊頂端。時間複雜度：O(1)。

- **彈出操作**：從堆疊頂端移除元素。時間複雜度：O(1)。

```python
# Using Python list as stack
stack = []

# Push operations
stack.append(1)
stack.append(2)
stack.append(3)

# Pop operations
top_element = stack.pop()  # Returns 3
```

## 佇列

- **佇列**：先進先出（FIFO）的資料結構。元素從尾端加入，從前端移除。

- **入佇列操作**：將元素加入佇列尾端。時間複雜度：O(1)。

- **出佇列操作**：從佇列前端移除元素。時間複雜度：O(1)。

```python
from collections import deque

# Using deque for efficient queue operations
queue = deque()

# Enqueue operations
queue.append(1)
queue.append(2)
queue.append(3)

# Dequeue operations
first_element = queue.popleft()  # Returns 1
```

## 鏈結串列

- **鏈結串列**：線性資料結構，每個節點包含資料和指向下一個節點的引用。節點像串鏈一樣相連。

### 單向鏈結串列

- **結構**：每個節點有資料和一個指向下一個節點的引用。
- **巡訪**：只能從頭節點向尾節點前進。
- **頭節點**：列表中的第一個節點，通常是唯一可直接存取的節點。
- **尾節點**：列表中的最後一個節點，指向 `None`。

### 操作與時間複雜度

- **在開頭插入**：O(1)
- **在尾端插入**：O(n) — 必須巡訪到尾端
- **在中間插入**：O(n) — 必須巡訪到指定位置
- **從開頭刪除**：O(1)
- **從尾端刪除**：O(n) — 必須巡訪找到前一個節點
- **從中間刪除**：O(n) — 必須巡訪找到該節點

### 雙向鏈結串列

- **結構**：每個節點有資料和兩個引用：下一個節點和前一個節點。
- **巡訪**：可以雙向移動。
- **記憶體**：因為多了額外的引用，比單向鏈結串列需要更多記憶體。

## 雜湊映射與集合

### 映射與雜湊映射

- **映射（抽象資料型別）**：管理鍵值對群集。每個鍵必須唯一，但值可以重複。

- **雜湊映射**：使用雜湊技術實作的映射抽象資料型別的具體實作。利用雜湊函式為鍵產生雜湊值，決定底層陣列中的儲存位置。

### 雜湊映射的時間複雜度

- **平均情況**：插入、取得、刪除皆為 O(1)

### 集合

- **集合**：無序且元素唯一的群集。不允許重複元素，且不維持特定順序。

- **僅限不可變元素**：集合只能包含不可變資料型別（數字、字串、元組），因為雜湊值必須保持不變。

```python
# Creating sets
numbers = {1, 2, 3, 4}
empty_set = set()  # Must use set(), not {}

numbers.add(5)
numbers.remove(4)      # Raises KeyError if not found

set_a = {1, 2, 3, 4}
```

### 集合的時間複雜度

- **平均情況**：新增、移除、成員測試皆為 O(1)
- **最壞情況**：因雜湊碰撞而為 O(n)

- **雜湊碰撞**：當兩個不同的鍵產生相同的雜湊值時發生。

- **碰撞解決策略**：
  - **串鏈**：每個陣列索引指向一個鏈結串列，儲存所有具有相同雜湊值的元素
  - **開放定址法**：使用預定義的序列搜尋下一個可用的索引

## 何時使用各種資料結構

- **列表**：當你需要有序且可索引的存取，且事先不知道大小時
- **堆疊**：用於後進先出（LIFO）操作（復原功能、運算式求值、回溯）
- **佇列**：用於先進先出（FIFO）操作（任務排程、廣度優先搜尋）
- **鏈結串列**：當需要頻繁在開頭插入或刪除、大小未知且不需要隨機存取時
- **雜湊映射**：用於快速的鍵值查找、計數出現次數、快取
- **集合**：用於檢查唯一性、數學集合運算、移除重複項目

## 搜尋演算法

搜尋演算法讓你能在特定的項目列表中搜尋標的。

在電腦科學中，有兩種搜尋演算法你應該知道。它們是 **線性搜尋** 和 **二分搜尋法**。了解這兩種演算法的差異以及何時使用各自非常重要。

### 線性搜尋

- 線性搜尋會從列表的開頭開始，逐一檢查每個項目，直到找到標的項目為止。
- 如果找到標的項目，會傳回該項目在列表中的索引。
- 如果找不到標的，會傳回 `-1`，這在大多數程式語言中代表 **無效索引**。
- 因為線性搜尋會檢查每個項目直到找到標的，所以對於大型列表來說效率不高。
- 線性搜尋的時間複雜度是 `O(n)`，因為搜尋列表所需的時間會隨著列表大小線性增加。
- 線性搜尋的空間複雜度是 `O(1)`，因為搜尋過程不需要額外空間。

### 二分搜尋法

- 二分搜尋法透過將列表分成兩半，檢查標的值是否在列表中間來運作。
- 二分搜尋法的前提條件是列表中的項目必須是遞增排序。
- 二分搜尋法對於搜尋大型列表來說是更高效的演算法，因為它將列表分成兩半，並忽略標的不存在的那一半。
- 如果在列表中間找到標的項目，會傳回該項目的索引。
- 如果找不到該項目，演算法會檢查標的項目是在列表的左半部還是右半部。
- 它會持續將剩餘的列表部分分成兩半，直到找到標的項目為止。
- 如果最後仍找不到標的項目，會傳回 `-1`。
- 二分搜尋法的時間複雜度是 `O(log n)`，因為搜尋列表所需的時間會隨著列表大小以對數方式增加。
- 二分搜尋法的空間複雜度是 `O(1)`，因為搜尋過程不需要額外空間。

### 線性搜尋與二分搜尋法的差異

- 相較於線性搜尋，二分搜尋法更適合用於大型列表。
- 線性搜尋的時間複雜度是 `O(n)`，因為搜尋列表所需的時間會隨著列表大小線性增加。
- 二分搜尋法的時間複雜度是 `O(log n)`，因為搜尋列表所需的時間會隨著列表大小以對數方式增加。

## 排序演算法與分治法

在電腦科學中，分治法是一種將問題拆解成較小子問題以便更容易解決的技巧。遞迴通常是分治法中使用的技術，而分治法是一種強大的策略，用來實作許多高效的排序演算法，例如合併排序。

### 合併排序

- 合併排序是一種遵循分治法的排序演算法。
- 它透過遞迴將列表不斷拆分成更小的子列表，直到每個子列表只包含一個元素。
- 接著反覆將子列表合併回來，並保持排序的順序。
- 合併排序的時間複雜度是 `O(n log n)`，因為列表持續被對半拆分 `(log n)`，然後再合併 `(O(n))`。
- 合併排序的空間複雜度是 `O(n)`，因為它不是原地排序演算法。

## 圖形概述

圖形是一組由邊（連接）連結的節點（頂點）。每個節點可以連接到多個其他節點，形成一個網路。不同類型的圖形包括：

- 有向圖：邊具有方向（從一個節點指向另一個節點），通常用直線和箭頭表示。
- 無向圖：邊沒有方向，使用簡單的線條表示。
- 頂點圖：每個節點都與一個標籤或識別符號相關聯。
- 有環圖：包含環路（從同一節點開始並結束的路徑）。
- 無環圖（DAG）：不包含環路。
- 邊標籤圖：每條邊都有一個標籤，通常繪製在相應的邊旁。
- 加權圖：邊具有與之相關的權重（數值），可用於執行算術操作。
- 不連通圖：包含兩個或多個未被任何邊連接的節點。

圖形被用於各種應用，如地圖、網路、推薦系統、相依性決議程序。

## 圖形遍歷

這涉及到拜訪圖形中的所有節點。兩個主要的演算法是：

- **廣度優先搜尋（BFS）**
  - 使用佇列。
  - 按層次逐層探索。
  - 在無權重圖中找到最短路徑。

- **深度優先搜尋（DFS）**
  - 使用堆疊（或遞迴）。
  - 先完整探索一個分支，再回溯。
  - 對於循環檢測和路徑尋找很有用。

## 圖形表述

圖形可以用兩種主要方式表述：

- **鄰接列表**
  - 每個節點都有一個鄰居的列表。
  - 對於稀疏圖來說空間高效。
  - 容易迭代鄰居。

- **鄰接矩陣**
  - 一個二維陣列，列和行代表節點。
  - 對於大型圖來說空間密集。
  - 快速檢查兩個節點之間是否存在邊。

## 樹狀結構

樹狀結構是一種特殊類型的圖形，具有無迴圈且連通的特性。主要特性包括：

- 它們沒有迴圈或循環（起點和終點節點相同的路徑）。
- 它們必須是連通的（每個節點都能從其他節點到達）。

### 常見的樹狀結構類型

最常見的樹狀結構類型有：

- 二元樹
  - 每個節點最多有兩個子節點，分別是左子節點和右子節點。

- 二分搜尋樹（BST）
  - 一種二元樹，其中每個左子節點的值都小於其父節點，而每個右子節點的值都大於其父節點。

## Tries

也稱為前置式樹，用來儲存字串集合，其中每個節點代表一個字元。

共用的前置式只儲存一次，使它們在自動完成和拼字檢查等任務中非常高效。

搜尋和插入操作的時間複雜度為 O(L)，其中 L 是字串的長度。

## 優先權佇列

優先權佇列是一種抽象的資料型別，其中每個元素都有一個優先權。

佇列和堆疊只考慮插入的順序，而優先權佇列則考慮元素的優先權。

標準佇列遵循 FIFO（先進先出），堆疊遵循 LIFO（後進先出）。然而，在優先權佇列中，優先權較高的元素會先被處理，無論它們的插入順序為何。

## 堆積

堆積是一種專門的樹狀資料結構，具有一個非常特定的屬性，稱為堆積屬性。

堆積屬性決定父節點和子節點之間的關係。堆積有兩種類型：

- 最大堆積（Max-Heap）
  - 每個父節點的值都大於或等於其子節點的值。
  - 最大的元素位於根節點。

- 最小堆積（Min-Heap）
  - 每個父節點的值都小於或等於其子節點的值。
  - 最小的元素位於根節點。

### Python `heapq` 模組範例

```py
import heapq

# Create empty heap
my_heap = []

# Insert elements
heapq.heappush(my_heap, 9)
heapq.heappush(my_heap, 3)
heapq.heappush(my_heap, 5)
print(my_heap) # [3, 9, 5]

# Remove smallest element
print(heapq.heappop(my_heap))  # 3
print(my_heap) # [5, 9]

# Push + Pop in one step
print(heapq.heappushpop(my_heap, 7)) # 5
print(my_heap) # [7, 9]

# Transform list into heap
nums = [5, 7, 3, 1]
heapq.heapify(nums)
```

### 使用優先權

```py
my_heap = []
heapq.heappush(my_heap, (3, "A"))
heapq.heappush(my_heap, (2, "B"))
heapq.heappush(my_heap, (1, "C"))

# Removes lowest number = highest priority
print(heapq.heappop(my_heap))  # (1, "C")
```

## 動態規劃介紹

- **定義**：動態規劃是一種演算法技巧，透過將複雜問題拆解成較簡單的子問題並儲存結果，以避免重複計算來解決問題。
- **重疊子問題**：在解決較大問題時，相同的較小子問題會多次出現。我們不會反覆重新計算這些子問題，而是將它們的解答儲存起來。
- **最優子結構**：問題的最優解包含其子問題的最優解。這表示我們可以透過結合較小部分的最佳解來構築整體的最佳解。

## 動態規劃解法

- **備忘錄法（自頂向下方法）**：備忘錄法會儲存耗時函式呼叫的結果，當相同輸入再次出現時，會傳回快取的結果。

```py
def climb_stairs_memo(n, memo={}):
    """Dynamic programming with memoization"""
    # Check if we've already calculated this value
    if n in memo:
        return memo[n]  # Return cached result - O(1) lookup!
    
    # Base cases
    if n <= 2:
        return n
    
    # Calculate once and store in memo for future use
    memo[n] = climb_stairs_memo(n-1, memo) + climb_stairs_memo(n-2, memo)
    return memo[n]
```

- **表格法（自底向上方法）**：表格法從基礎開始建置解法，透過填寫子問題解答的表格來完成。

```py
def climb_stairs_tabulation(n):
    """Dynamic programming with tabulation"""
    if n <= 2:
        return n
    
    # Create array to store results for all steps from 0 to n
    dp = [0] * (n + 1)
    dp[1] = 1  # 1 way to reach step 1
    dp[2] = 2  # 2 ways to reach step 2
    
    # Build up the solution iteratively
    for i in range(3, n + 1):
        # Ways to reach step i = ways to reach (i-1) + ways to reach (i-2)
        dp[i] = dp[i-1] + dp[i-2]
    
    return dp[n]
```

## 使用動態規劃的實際應用

- **路徑優化**：GPS 系統使用動態規劃演算法來尋找地點之間的最短路徑。
- **文字處理**：拼字檢查和自動完成功能常依賴動態規劃來計算單字之間的編輯距離。
- **財務模式化**：投資策略和投資組合優化經常運用動態規劃技術。
- **資源分配**：背包問題及其變體常見於排程、預算和資源管理。

## 何時使用動態規劃

你應該在以下情況考慮使用動態規劃：

- 問題可以拆解成重疊的子問題。
- 問題具有最佳子結構。
- 簡單的遞迴解法會涉及重複計算。
- 你需要以空間複雜度為代價來最佳化時間複雜度。
