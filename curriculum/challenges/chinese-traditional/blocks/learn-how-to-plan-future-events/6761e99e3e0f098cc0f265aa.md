# --explanation--

`SQL injection attacks` 是一種網路攻擊，駭客會將有害的程式碼插入資料庫查詢中，以存取或破壞資料。例如：

`The company added security measures to prevent SQL injection attacks on their website。`－這表示他們採取了措施來阻止駭客輸入有害的程式碼。  

Jake 說需要 `additional measures` 來 `prevent SQL injection attacks`，意思是他們正試圖阻止這種特定型別的威脅。
