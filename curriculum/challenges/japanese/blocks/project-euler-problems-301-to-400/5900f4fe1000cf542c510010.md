# --description--

フィボナッチ木は次のように再帰的に定義される二分木です：

- $T(0)$ は空の木です。
- $T(1)$ はノードが1つだけの二分木です。
- $T(k)$ は根ノードを持ち、その子として $T(k - 1)$ と $T(k - 2)$ を持ちます。

このような木で、2人のプレイヤーが取り除きゲームを行います。各ターンでプレイヤーはノードを1つ選び、そのノードとそのノードを根とするサブツリーを取り除きます。ゲーム全体の根ノードを取らざるを得なくなったプレイヤーが負けです。

以下は $k = 1$ から $k = 6$ までの $T(k)$ における、最初のターンでの先手の勝利手の例です。

<img alt="winning moves of first player, on the first turn for k = 1 to k = 6" src="https://cdn.freecodecamp.org/curriculum/project-euler/fibonacci-tree-game.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$f(k)$ を、このゲームが $T(k)$ 上で行われるときの、最初のターンにおける先手の勝利手の数（つまり後手に勝利戦略がない手）の数とします。

例えば、$f(5) = 1$、$f(10) = 17$ です。

$f(10000)$ を求めてください。答えの下18桁を提出してください。
