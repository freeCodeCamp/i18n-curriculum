# --interactive--

**注意**：有些互動範例可能會使用你還沒學過的 CSS。不要擔心試著理解所有的程式碼。這些範例的目標是為了展示這些設計概念的預覽，讓你更了解運作方式。要看到預覽，你需要啟用互動式編輯器。

漸進式揭露是一種設計樣式，用於根據使用者當前的活動只顯示相關內容並隱藏其餘部分。這樣做是為了減少認知負擔並使使用者體驗更直覺。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="progressive-disclosure">
  <button id="show-details-btn" aria-expanded="false" aria-controls="extra-content">
    Show More Details
  </button>
  <div id="extra-content" class="hidden" tabindex="-1">
    <p>
      Here are additional details that are revealed only when you want to see them.
      This keeps the interface clean and reduces cognitive load.
    </p>
  </div>
</div>
<script src="index.js"></script>

```

```js
const btn = document.getElementById('show-details-btn');
const content = document.getElementById('extra-content');

btn.addEventListener('click', () => {
  const isHidden = content.classList.toggle('hidden');
  btn.setAttribute('aria-expanded', !isHidden);
  if (!isHidden) {
    content.focus();
  }
});

```

例如，當你使用 Google 的搜尋頁面時，大多數的搜尋都會很簡單。因此，擁有一個簡單且選項很少的使用者介面是合理的。但當你需要進階搜尋的少數情況時，Google 提供了一個進階搜尋選項，會顯示更多選項。

你可以在 Google 首頁的設定選單中找到 Google 的進階搜尋選項。或者你可以前往 `google.com/advanced_search`。

另一個漸進式揭露的範例是在電子商務網站上的「更多細節」按鈕。當你造訪像 Amazon 這樣的網站時，你會看到不同分類中產品影像的列表。這讓使用者能輕鬆購物，而不會被過多的資訊淹沒。

如果使用者對特定產品感興趣，他們可以點擊產品影像以顯示更多產品細節。立即顯示所有資訊可能會讓使用者感到不知所措。因此，透過漸進式揭露，使用者可以在準備好時選擇查看更多資訊。

```js
document.querySelectorAll('.details-btn').forEach(button => {
  button.addEventListener('click', () => {
    const product = button.closest('.product');
    const details = product.querySelector('.product-details');
    const isHidden = details.classList.toggle('hidden');
    product.setAttribute('aria-expanded', !isHidden);
    if (!isHidden) {
      details.focus();
      button.textContent = 'Hide details';
    } else {
      button.textContent = 'More details';
    }
  });
});
```

那麼，使用漸進式揭露技術設計時，有哪些最佳實務？第一個考量是要讓所有重要資訊隨時可見。使用者不應該因為關鍵資訊被額外的進階特性隱藏而必須費力尋找。

另一個考量是為使用者提供單一存取點，以存取額外的特性或資訊。這可以是一個在頁面上始終可見的按鈕或連結。添加多個存取點可能會讓使用者感到困惑，並導致不佳的使用者體驗。

在你設計應用程式時，請考慮整體使用者體驗，以及如何使用漸進式揭露來讓使用者更容易找到他們需要的資訊。當正確執行時，它能對整體使用者體驗產生重大影響。
