## Lavorare con il tipo di dato number

- **Definizione**: Il tipo `Number` di JavaScript include numeri interi, numeri a virgola mobile, `Infinity` e `NaN`. I numeri a virgola mobile sono numeri con un punto decimale. `Infinity` positivo è un numero maggiore di qualsiasi altro numero mentre `-Infinity` è un numero più piccolo di qualsiasi altro numero. `NaN` (`Not a Number`) rappresenta un valore numerico non valido come la stringa `"Jessica"`.

## Operazioni aritmetiche comuni

- **Operatore di addizione**: Questo operatore (`+`) si usa per calcolare la somma di due o più numeri. 
- **Operatore di sottrazione**: Questo operatore (`-`) si usa per calcolare la differenza tra due numeri.
- **Operatore di moltiplicazione**: Questo operatore (`*`) si usa per calcolare il prodotto di due o più numeri. 
- **Operatore di divisione**: Questo operatore (`/`) si usa per calcolare il quoziente tra due numeri.
- **Divisione per zero**: Se provi a dividere per zero, JavaScript restituirà `Infinity`.
- **Operatore resto**: Questo operatore (`%`) restituisce il resto di una divisione. 
- **Operatore di esponenziazione**: Questo operatore (`**`) eleva un numero alla potenza di un altro.

## Calcoli con numeri e stringhe

- **Spiegazione**: Quando usi l'operatore `+` con un numero e una stringa, JavaScript converte il numero in una stringa e concatena i due valori. Quando usi gli operatori `-`, `*` o `/` con una stringa e un numero, JavaScript converte la stringa in un numero e il risultato sarà un numero. Per `null` e `undefined`, JavaScript tratta `null` come 0 e undefined come `NaN` nelle operazioni matematiche.

:::interactive_editor

```js
const result = 5 + '10';

console.log(result); // "510"
console.log(typeof result); // string

const subtractionResult = '10' - 5;
console.log(subtractionResult); // 5
console.log(typeof subtractionResult); // number

const multiplicationResult = '10' * 2;
console.log(multiplicationResult); // 20
console.log(typeof multiplicationResult); // number

const divisionResult = '20' / 2;
console.log(divisionResult); // 10
console.log(typeof divisionResult); // number

const result1 = null + 5;
console.log(result1); // 5
console.log(typeof result1); // number

const result2 = undefined + 5;
console.log(result2); // NaN
console.log(typeof result2); // number
```

:::

## Precedenza degli operatori

- **Definizione**: La precedenza degli operatori determina l'ordine in cui le operazioni vengono valutate in un'espressione. Gli operatori con precedenza più alta vengono valutati prima di quelli con precedenza più bassa. I valori dentro le parentesi vengono valutati per primi e moltiplicazione/divisione hanno precedenza più alta di addizione/sottrazione. Se gli operatori hanno la stessa precedenza, allora JavaScript usa l'associatività.

:::interactive_editor

```js
const result = (2 + 3) * 4;

console.log(result); // 20

const result2 = 10 - 2 + 3;

console.log(result2); // 11

const result3 = 2 ** 3 ** 2;

console.log(result3); // 512
```

:::

- **Definizione**: L'associatività ci indica la direzione in cui un'espressione viene valutata quando esistono più operatori dello stesso tipo. Definisce se l'espressione deve essere valutata da sinistra a destra (`left-associative`) o da destra a sinistra (`right-associative`). Per esempio, l'operatore di esponenziazione è associativo da destra a sinistra:

:::interactive_editor

```js
const result4 = 5 ** 4 ** 1; 

console.log(result4); // 625
```

:::

## Operatori di incremento e decremento

- **Operatore di incremento**: Questo operatore si usa per aumentare il valore di uno. La notazione prefissa `++num` aumenta prima il valore della variabile, poi restituisce il nuovo valore. La notazione postfissa `num++` restituisce prima il valore corrente della variabile, poi lo aumenta.

:::interactive_editor

```js
let x = 5;

console.log(++x); // 6
console.log(x); // 6


let y = 5;

console.log(y++); // 5
console.log(y); // 6
```

:::

- **Operatore di decremento**: Questo operatore si usa per diminuire il valore di uno. La notazione prefissa e postfissa funzionano allo stesso modo di prima con l'operatore di incremento.

:::interactive_editor

```js
let num = 5;

console.log(--num); // 4
console.log(num--); // 4
console.log(num); // 3
```

:::

## Operatori di assegnazione composta

- **Operatore di assegnazione di addizione (`+=`)**: Questo operatore esegue l'addizione sui valori e assegna il risultato alla variabile.
- **Operatore di assegnazione di sottrazione (`-=`)**: Questo operatore esegue la sottrazione sui valori e assegna il risultato alla variabile.
- **Operatore di assegnazione di moltiplicazione (`*=`)**: Questo operatore esegue la moltiplicazione sui valori e assegna il risultato alla variabile.
- **Operatore di assegnazione di divisione (`/=`)**: Questo operatore esegue la divisione sui valori e assegna il risultato alla variabile.
- **Operatore di assegnazione del resto (`%=`)**: Questo operatore divide una variabile per il numero specificato e assegna il resto alla variabile.
- **Operatore di assegnazione di esponenziazione (`**=`)**: Questo operatore eleva una variabile alla potenza del numero specificato e riassegna il risultato alla variabile.

## Booleani e uguaglianza

- **Definizione di booleano**: Un booleano è un tipo di dato che può avere solo due valori: `true` o `false`. 
- **Operatore di uguaglianza (`==`)**: Questo operatore usa la conversione implicita prima di verificare se i valori sono uguali.

:::interactive_editor

```js
console.log(5 == '5'); // true
```

:::

- **Operatore di uguaglianza stretta (`===`)**: Questo operatore non esegue conversione implicita e verifica se sia i tipi sia i valori sono uguali.

:::interactive_editor

```js
console.log(5 === '5'); // false
```

:::

- **Operatore di disuguaglianza (`!=`)**: Questo operatore usa la conversione implicita prima di verificare se i valori non sono uguali.
- **Operatore di disuguaglianza stretta (`!==`)**: Questo operatore non esegue conversione implicita e verifica se sia i tipi sia i valori non sono uguali.

## Operatori di confronto

- **Operatore maggiore di (`>`)**: Questo operatore verifica se il valore a sinistra è maggiore di quello a destra.
- **Operatore maggiore o uguale (`>=`)**: Questo operatore verifica se il valore a sinistra è maggiore o uguale a quello a destra.
- **Operatore minore di (`<`)**: Questo operatore verifica se il valore a sinistra è minore di quello a destra.
- **Operatore minore o uguale (`<=`)**: Questo operatore verifica se il valore a sinistra è minore o uguale a quello a destra.

## Operatori unari

- **Operatore unario più**: Questo operatore converte il suo operando in un numero. Se l'operando è già un numero, rimane invariato.

:::interactive_editor

```js
const str = '42';
const num = +str;

console.log(num); // 42
console.log(typeof num); // number
```

:::

- **Operatore unario di negazione (`-`)**: Questo operatore nega l'operando.

:::interactive_editor

```js
const num = 4;
console.log(-num); // -4
```

:::

- **Operatore logico NOT (`!`)**: Questo operatore inverte il valore booleano del suo operando. Quindi, se l'operando è `true`, diventa `false`, e se è `false`, diventa `true`.

## Operatori bit a bit

- **Operatore AND bit a bit (`&`)**: Questo operatore restituisce 1 in ogni posizione di bit per cui i bit corrispondenti di entrambi gli operandi sono 1. 
- **Operatore di assegnazione AND bit a bit (`&=`)**: Questo operatore esegue un'operazione `bitwise AND` con il numero specificato e riassegna il risultato alla variabile.
- **Operatore OR bit a bit (`|`)**: Questo operatore restituisce 1 in ogni posizione di bit per cui i bit corrispondenti di uno o entrambi gli operandi sono 1. 
- **Operatore di assegnazione OR bit a bit (`|=`)**: Questo operatore esegue un'operazione `bitwise OR` con il numero specificato e riassegna il risultato alla variabile.
- **Operatore XOR bit a bit (`^`)**: Questo operatore restituisce 1 in ogni posizione di bit per cui i bit corrispondenti di uno solo degli operandi sono 1. 
- **Operatore NOT bit a bit (`~`)**: Questo operatore inverte la rappresentazione binaria di un numero.
- **Operatore di shift a sinistra (`<<`)**: Questo operatore sposta tutti i bit a sinistra di un numero specificato di posizioni. 
- **Operatore di shift a destra (`>>`)**: Questo operatore sposta tutti i bit a destra.

## Istruzioni condizionali, valori truthy, valori falsy e operatore ternario

- **`if/else if/else`**: Un'istruzione `if` prende una condizione ed esegue un blocco di codice se quella condizione è `truthy`. Se la condizione è `false`, allora passa al blocco `else if`. Se nessuna di queste condizioni è `true`, allora eseguirà la clausola `else`. I valori `Truthy` sono tutti quei valori che risultano in `true` quando valutati in un contesto booleano come un'istruzione `if`. I valori `Falsy` sono valori che valutano a `false` in un contesto booleano.

:::interactive_editor

```js
const score = 87;

if (score >= 90) {
 console.log('You got an A'); 
} else if (score >= 80) {
 console.log('You got a B'); // You got a B
} else if (score >= 70) {
 console.log('You got a C');
} else {
 console.log('You failed! You need to study more!');
}
```

:::

- **Operatore ternario**: Questo operatore è spesso usato come modo più breve per scrivere istruzioni `if else`.

:::interactive_editor

```js
const temperature = 30;
const weather = temperature > 25 ? 'sunny' : 'cool';

console.log(`It's a ${weather} day!`); // It's a sunny day!
```

:::

## Operatori logici binari

- **Operatore logico AND (`&&`)**: Questo operatore verifica se entrambi gli operandi sono `true`. Se entrambi sono `true`, allora restituirà il secondo valore. Se uno degli operandi è `falsy`, allora restituirà il valore `falsy`. Se entrambi gli operandi sono `falsy`, restituirà il primo valore `falsy`.

:::interactive_editor

```js
const result = true && 'hello';

console.log(result); // hello
```

:::

- **Operatore logico OR (`||`)**: Questo operatore verifica se almeno uno degli operandi è truthy. 
- **Operatore di coalescenza nullish (`??`)**: Questo operatore restituirà un valore solo se il primo è `null` o `undefined`.

:::interactive_editor

```js
const userSettings = {
 theme: null,
 volume: 0,
 notifications: false,
};

let theme = userSettings.theme ?? 'light';
console.log(theme); // light
```

:::

## L'oggetto `Math`

- **Il metodo `Math.random()`**: Questo metodo genera un numero a virgola mobile casuale tra 0 (incluso) e 1 (escluso). Questo significa che l'output possibile può essere 0, ma non raggiungerà mai 1.
- **Il metodo `Math.max()`**: Questo metodo prende un insieme di numeri e restituisce il valore massimo.
- **Il metodo `Math.min()`**: Questo metodo prende un insieme di numeri e restituisce il valore minimo.
- **Il metodo `Math.ceil()`**: Questo metodo arrotonda un valore per eccesso all'intero più vicino.
- **Il metodo `Math.floor()`**: Questo metodo arrotonda un valore per difetto all'intero più vicino.
- **Il metodo `Math.round()`**: Questo metodo arrotonda un valore all'intero più vicino.

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

- **Il metodo `Math.trunc()`**: Questo metodo rimuove la parte decimale di un numero, restituendo solo la parte intera, senza arrotondare.
- **Il metodo `Math.sqrt()`**: Questo metodo restituisce la radice quadrata di un numero.
- **Il metodo `Math.cbrt()`**: Questo metodo restituisce la radice cubica di un numero.
- **Il metodo `Math.abs()`**: Questo metodo restituisce il valore assoluto di un numero.
- **Il metodo `Math.pow()`**: Questo metodo prende due numeri e eleva il primo alla potenza del secondo.

## Metodi comuni per i numeri

- **`isNaN()`**: `NaN` sta per "Not-a-Number". È un valore speciale che rappresenta un risultato numerico non rappresentabile o indefinito. La proprietà di funzione `isNaN()` si usa per determinare se un valore è `NaN` o no. `Number.isNaN()` fornisce un modo più affidabile per verificare i valori `NaN`, specialmente nei casi in cui la conversione implicita potrebbe portare a risultati inattesi con la funzione globale `isNaN()`.

:::interactive_editor

```js
console.log(isNaN(NaN));       // true
console.log(isNaN(undefined)); // true
console.log(isNaN({}));        // true

console.log(isNaN(true));      // false
console.log(isNaN(null));      // false
console.log(isNaN(37));        // false


console.log(Number.isNaN(NaN));        // true
console.log(Number.isNaN(Number.NaN)); // true
console.log(Number.isNaN(0 / 0));      // true

console.log(Number.isNaN("NaN"));      // false
console.log(Number.isNaN(undefined));  // false
```

:::

- **Il metodo `parseFloat()`**: Questo metodo analizza un argomento stringa e restituisce un numero a virgola mobile. È progettato per estrarre un numero dall'inizio di una stringa, anche se la stringa contiene caratteri non numerici più avanti.
- **Il metodo `parseInt()`**: Questo metodo analizza un argomento stringa e restituisce un numero intero. `parseInt()` interrompe l'analisi al primo carattere non numerico incontrato. Per i numeri a virgola mobile, restituisce solo la parte intera. Se non trova un intero valido all'inizio della stringa, restituisce `NaN`.
- **Il metodo `toFixed()`**: Questo metodo viene chiamato su un numero e prende un argomento opzionale, che è il numero di cifre da mostrare dopo il punto decimale. Restituisce una rappresentazione in stringa del numero con il numero specificato di decimali.
