# --description--

Ви можете ітерувати словник, якщо потрібно отримати доступ і обробити його пари ключ-значення. Це корисно для оновлення їхніх значень або застосування до них певної логіки.

Давайте розглянемо деякі техніки, які ви можете використати.

Припустимо, у нас є `products` словник, який асоціює кожен продукт із його ціною:

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,
    'Headphones': 70,
}
```

Якщо ми хочемо запропонувати знижку 20% на всі наші продукти, ми можемо ітерувати всі пари ключ-значення та змінювати ціни.

Методи `.values()`, `.keys()` і `.items()` є ключовими для цих технік. Ми коротко розглядали їх у попередньому уроці.

Вони повертають об’єкт-представлення зі значеннями, ключами та парами ключ-значення словника. Ви можете використовувати ці об’єкти-представлення в циклах `for`, щоб ітерувати елементи.

Наприклад, ви можете ітерувати всі значення словника так:

Ви пишете `for`, змінну циклу (у цьому випадку `price`), `products.values()`, щоб отримати всі значення словника `products`, двокрапку, а потім тіло циклу, де можна застосувати будь-яку логіку до значень. У цьому випадку ми їх виводимо.

Змінна циклу прийматиме кожне значення по черзі, по одному за ітерацію:

```python
for price in products.values():
    print(price)
```

Ось вихідні дані. Як бачите, кожне значення виводиться в консоль по одному:

```md
990
600
250
70
```

Це працює так само для `.keys()`, якщо потрібно ітерувати ключі словника. Просто ітеруйте `products.keys()` або `products` напряму і призначте описову назву змінній циклу:

```python
for product in products.keys():
    print(product)

# Or

for product in products:
    print(product)
```

Ось вихідні дані. Кожен ключ виводиться в консоль по одному:

```md
Laptop
Smartphone
Tablet
Headphones
```

Так само це працює для пар ключ-значення, якщо потрібно ітерувати ключі та відповідні значення одночасно. Просто ітеруйте `products.items()`:

```python
for product in products.items():
    print(product)
```

Ось вихідні дані. Тепер ви отримуєте окремі кортежі з ключами та відповідними значеннями:

```md
('Laptop', 990)
('Smartphone', 600)
('Tablet', 250)
('Headphones', 70)
```

Якщо хочете зберегти ключ і значення в окремих змінних циклу, просто визначте їх і розділіть комою. Потім можете використовувати їх у тілі циклу.

Тут ми визначаємо змінну циклу `product` і змінну циклу `price`. Кожна триматиме своє відповідне значення. Важливо визначати їх у правильному порядку – спочатку ключ, потім значення:

```python
for product, price in products.items():
    print(product, price)
```

Ось вихідні дані. Ми виводимо їх поруч, але ви можете використовувати ці значення так, як потрібно у вашому коді.

```md
Laptop 990
Smartphone 600
Tablet 250
Headphones 70
```

Тепер, коли ви знаєте більше про це, повернемося до нашого початкового прикладу. Якщо ми хочемо запропонувати знижку 20%, ми помножимо кожну ціну на `0.8` і знову призначимо це як значення для відповідного ключа продукту.

Ми також можемо округлити результат вниз, якщо хочемо працювати з цілими числами:

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,
    'Headphones': 70,
}

for product, price in products.items():
    products[product] = round(price * 0.8)

print(products)
```

Потім, якщо вивести словник, отримаємо такі пари ключ-значення зі зниженими цінами:

```python
{
    'Laptop': 792, 
    'Smartphone': 480, 
    'Tablet': 200, 
    'Headphones': 56
}
```

І нарешті, якщо потрібно ітерувати пари ключ-значення, відстежуючи лічильник, можна викликати функцію `enumerate()`. Цей лічильник фактично виконує роль «індексу» або «рахунку» для елемента в циклі.

Функція повертає об’єкт `enumerate`, який призначає ціле число кожній парі ключ-значення, як лічильник. Ви можете почати лічильник з будь-якого числа, але за замовчуванням він починається з 0.

Тут ми ітеруємо ключі словника `products`:

```python
for product in enumerate(products):
    print(product)
```

Але функція `enumerate()` також призначає ціле число кожному ключу, тому ми отримуємо кортежі з числом і ключем.

Ось вихідні дані:

```md
(0, 'Laptop')
(1, 'Smartphone')
(2, 'Tablet')
(3, 'Headphones')
```

Якщо потрібно, ви можете призначити ці значення окремим змінним циклу. Тут у нас дві змінні циклу (`index` і `product`). Це те, що ви зазвичай побачите і використаєте, працюючи з `enumerate()`:

```python
for index, product in enumerate(products):
    print(index, product)
```

Якщо потрібно ітерувати значення, замініть `products` на `products.values()`:

```python
for price in enumerate(products.values()):
    print(price)
```

Вихідні дані міститимуть індекс і ціну в кожному кортежі:

```md
(0, 990)
(1, 600)
(2, 250)
(3, 70)
```

Ви також можете призначити їх окремим змінним циклу:

```python
for index, price in enumerate(products.values()):
    print(index, price)
```

Ось вихідні дані. Ви можете використовувати їх так, як потрібно у вашому коді:

```md
0 990
1 600
2 250
3 70
```

А з `products.items()` ви можете отримати всю пару ключ-значення разом з «індексом» або «лічильником»:

```python
for index, product in enumerate(products.items()):
    print(index, product)
```

У цьому прикладі ми отримуємо індекс, а потім кортеж, що містить ключ і значення відповідної пари ключ-значення:

```md
0 ('Laptop', 990)
1 ('Smartphone', 600)
2 ('Tablet', 250)
3 ('Headphones', 70)
```

Щоб налаштувати початкове значення лічильника, можна передати другий аргумент у `enumerate()`. Наприклад, тут ми починаємо рахунок з 1:

```python
for index, product in enumerate(products.items(), 1):
    print(index, product)
```

Цю зміну видно у вихідних даних. Тепер перше ціле число — 1 замість 0:

```md
1 ('Laptop', 990)
2 ('Smartphone', 600)
3 ('Tablet', 250)
4 ('Headphones', 70)
```

Це працює з будь-якою варіацією, яку ми бачили раніше. Просто передайте початкове число як другий аргумент.

Існує багато технік ітерування словника. Це деякі загальні способи, і вам потрібно буде обрати найкращий для свого проєкту.
