# --explanation--

`Hidden` significa algo que está oculto ou não é imediatamente visível, frequentemente exigindo um esforço extra para ser encontrado. 
 
`Malware` refere-se a software malicioso projetado para danificar, interromper ou explorar um sistema de computador.  

Quando combinados, `hidden malware` descreve um software malicioso que é difícil de detectar porque está oculto dentro do sistema.Por exemplo:

`The scan detected hidden malware that had bypassed the antivirus software.` - Isso significa que o malware conseguiu evitar a detecção pelo programa antivírus até que uma varredura o encontrasse.
