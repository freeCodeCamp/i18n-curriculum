# --description--

Kiongezaji `+` kipo katika aina mbili: aina ya binary uliyotumia katika changamoto iliyopita na aina ya unary.

Kiongezaji cha unary, au kwa maneno mengine, kiongezaji `+` kinachotumika kwa thamani moja tu, hakifanyi chochote kwa nambari. Lakini ikiwa operand si nambari, kiongezaji cha unary hubadilisha kuwa nambari.

Kwa mfano:

```js
// No effect on numbers
let x = 1;
console.log( +x ); // 1

let y = -2;
console.log( +y ); // -2

// Converts non-numbers
console.log( +true ); // 1
console.log( +"" );   //
```

Haja ya kubadilisha mfuatano wa herufi kuwa nambari hutokea mara nyingi sana. Kwa mfano, ikiwa unapata thamani kutoka kwa sehemu za fomu za HTML, kawaida ni mfuatano wa herufi. Je, unataka kuziongeza?

Kiongezaji cha binary kingeziongeza kama mfuatano wa herufi:

```js
let apples = "2";
let oranges = "3";

alert( apples + oranges ); // "23", the binary plus concatenates strings
```

Ikiwa unataka kuzitumia kama nambari, unahitaji kubadilisha kisha kuziongeza:

```js
let apples = "2";
let oranges = "3";

// both values converted to numbers before the binary plus
alert( +apples + +oranges ); // 5

// the longer variant
// alert( Number(apples) + Number(oranges) ); // 5
```

Kwa mtazamo wa mwanahesabu, wingi wa viongezaji unaweza kuonekana kuwa wa ajabu. Lakini kwa mtazamo wa mpogramu, hakuna kitu maalum: viongezaji vya unary hutumika kwanza, hubadilisha mfuatano wa herufi kuwa nambari, kisha kiongezaji cha binary kinaongeza thamani hizo.

Kwa nini viongezaji vya unary hutumika kwa thamani kabla ya vya binary? Kama utakavyoona, ni kwa sababu ya kipaumbele chao _cha juu_.

## --text--
Kulingana na maelezo ya viongezaji vya unary na binary `+` katika JavaScript, unaweza kuongeza kwa usahihi vifuatazo vya mfuatano wa herufi `'2'` na `'3'` kama nambari kwa kutumia viongezaji vya unary `+` vipi?

## --answers--

`console.log('2' + '3'); // Outputs: '23'` 

---

`console.log(+2 + +3); // Outputs: 5`

---

`console.log(+'2' + +'3'); // Outputs: 5`

---

`console.log(Number('2') + Number('3')); // Outputs: '23'`
