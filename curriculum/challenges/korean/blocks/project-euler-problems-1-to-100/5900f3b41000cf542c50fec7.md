# --description--

분수 $\frac{n}{d}$를 생각해 보세요. 여기서 `n`와 `d`는 양의 정수입니다. 만약 `n` &lt; `d`이고 최대공약수, ${HCF}(n, d) = 1$이라면, 이를 기약 진분수라고 합니다.

`d` ≤ 8인 기약 진분수 집합을 크기 순서대로 나열하면 다음과 같습니다:

$$\frac{1}{8}, \frac{1}{7}, \frac{1}{6}, \frac{1}{5}, \frac{1}{4}, \frac{2}{7}, \frac{1}{3}, \frac{3}{8}, \frac{2}{5}, \frac{3}{7}, \frac{1}{2}, \frac{4}{7}, \frac{3}{5}, \frac{5}{8}, \frac{2}{3}, \frac{5}{7}, \frac{3}{4}, \frac{4}{5}, \frac{5}{6}, \frac{6}{7}, \frac{7}{8}$$

이 집합에는 `21`개의 원소가 있음을 알 수 있습니다.

`d` ≤ `limit`인 기약 진분수 집합에는 몇 개의 원소가 포함될까요?

```js
function countingFractions(limit) {
  const phi = {};
  let count = 0;

  for (let i = 2; i <= limit; i++) {
    if (!phi[i]) {
      phi[i] = i;
    }
    if (phi[i] === i) {
      for (let j = i; j <= limit; j += i) {
        if (!phi[j]) {
          phi[j] = j;
        }
        phi[j] = (phi[j] / i) * (i - 1);
      }
    }
    count += phi[i];
  }

  return count;
}
```
