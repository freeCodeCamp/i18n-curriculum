# --description--

Підпрограмне забезпечення рівня застосунку — це підпрограмне забезпечення, яке прив’язане до екземпляра застосунку Express. Його зазвичай використовують для обробки запитів, які потребують обробки через кілька маршрутів. Його можна застосовувати глобально або до конкретних маршрутів.

Це підпрограмне забезпечення реєструється за допомогою `app.use()` або конкретних методів маршруту, таких як `app.get()`, `app.post()` тощо. Воно виконується для кожного вхідного запиту, якщо не обмежене конкретними маршрутами.

```js
const express = require('express')
const app = express()

// Application-level middleware that logs all incoming requests
app.use((req, res, next) => {
  console.log('Request received at:', new Date())
  next(); // Passes control to the next middleware
})

app.get('/', (req, res) => {
  res.send('Hello World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* `app.use()`: Це підпрограмне забезпечення рівня застосунку. Воно реєструє кожен запит, зроблений до сервера, незалежно від маршруту.
* Підпрограмне забезпечення виконується для кожного вхідного запиту, надаючи серверу деталі, як-от поточний часовий штамп.

Підпрограмне забезпечення рівня застосунку корисне для завдань, які потрібно застосовувати на багатьох або всіх маршрутах у застосунку. Це може включати логування, автентифікацію та обробку помилок.
