# --description--

هناك قيدان على أسماء المتغيرات في JavaScript:

1. يجب أن يحتوي الاسم على حروف أو أرقام فقط، أو الرمزين `$` و`_`.
1. لا يجب أن يكون الحرف الأول رقمًا.

أمثلة على أسماء صحيحة:

```js
let userName;
let test123;
```

ما هو مثير للاهتمام – يمكن أيضًا استخدام علامة الدولار `'$'` والشرطة السفلية `'_'` في الأسماء. هما رمزان عاديان، تمامًا مثل الحروف، بدون أي معنى خاص.

هذه الأسماء صحيحة:

```js
let $ = 1; // declared a variable with the name "$"
let _ = 2; // and now a variable with the name "_"

console.log($ + _); // 3
```

أمثلة على أسماء متغيرات غير صحيحة:

```js
let 1a; // cannot start with a digit

let my-name; // hyphens '-' aren't allowed in the name
```
