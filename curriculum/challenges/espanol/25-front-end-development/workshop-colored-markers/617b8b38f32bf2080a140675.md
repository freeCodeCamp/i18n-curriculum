# --description--

Como has aprendido en las conferencias, una función es un fragmento de código que puede tomar una entrada y realizar una acción específica. La función de CSS `rgb` acepta valores, o <dfn>argumentos</dfn>, para rojo, verde y azul, y produce un color:

```css
rgb(red, green, blue);
```

Cada valor de rojo, verde y azul es un número del `0` al `255`. `0` significa que hay un 0% de ese color, y es negro. `255` significa que hay un 100% de ese color.

En la regla CSS `.one`, reemplaza la palabra clave de color `red` con la función `rgb`. Para la función `rgb`, establece el valor para rojo a `255`, el valor para verde a `0` y el valor para azul a `0`.
