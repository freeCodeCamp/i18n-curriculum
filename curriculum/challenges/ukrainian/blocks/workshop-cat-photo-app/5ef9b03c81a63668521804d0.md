# --description--

Щоб виділити конкретне слово чи фразу, можна використати елемент `em`.

Виділіть слово `love` в елементі `figcaption`, обгорнувши його елементом `em`.

# --hints--

Елемент виділення `em` повинен мати початковий тег. Початкові теги мають такий синтаксис: `<назваЕлемента>`.

```js
assert.exists(document.querySelector('em'));
```

Елемент виділення `em` повинен мати кінцевий тег. Кінцеві теги мають `/` одразу після символу `<`.

```js
assert.match(code, /<\/em\>/);
```

Ви або видалили елемент `figcaption`, або в ньому відсутній початковий чи кінцевий тег.

```js
assert.exists(document.querySelector('figcaption'));
assert.match(code, /<\/figcaption\>/);
```

Елемент `em` має оточувати текст `love`. Ви або не написали текст, або маєте друкарську помилку.

```js
assert.equal(
  document.querySelector('figcaption > em')?.innerText?.trim().toLowerCase(), 'love'
);
```

Текстом елемента `figcaption` має бути `Cats love lasagna`. Перевірте наявність друкарських помилок та необхідних пробілів навколо тегів елемента `em`.

```js
assert.match(
  document
    .querySelector('figcaption')
    ?.innerText?.trim().replace(/\s+/gi, ' '),
    /^cats love lasagna\.?$/i
);
```
