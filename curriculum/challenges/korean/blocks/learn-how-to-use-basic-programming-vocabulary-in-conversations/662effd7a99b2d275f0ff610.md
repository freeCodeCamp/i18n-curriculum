# --description--

단어 `might`는 가능성이나 잠재력을 나타내는 조동사입니다. 구문 `be able to`은 능력을 나타냅니다.

함께 사용될 때 `might be able to`는 무언가를 성취할 가능성이 있지만 확실하지 않다는 뜻입니다.

예를 들어, `She might be able to finish the work by tomorrow.`는 그녀가 그때까지 일을 마칠 가능성이 있지만 보장되지는 않는다는 의미입니다.
