# --explanation--

`How they look` ist eine Klausel, die mit `how` beginnt, welche ein Thema oder Subjekt einführt. Zum Beispiel:

`He didn't like how the icons looked on the screen.` – Das bedeutet, dass ihm das Aussehen der Symbole nicht gefallen hat.

Klauseln wie `how they look` werden **Nominalklauseln** genannt und können als Objekt eines Satzes fungieren.
