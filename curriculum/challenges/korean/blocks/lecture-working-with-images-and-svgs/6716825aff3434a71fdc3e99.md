# --interactive--

먼저, 이미지가 어떻게 작동하는지 이해해야 합니다. PNG와 JPG 같은 일반적인 이미지 형식은 래스터 형식으로 분류됩니다. 이는 기본적으로 픽셀 기반이며, 각 픽셀의 색상 값을 추적하는 데이터라는 뜻입니다.

래스터 기반 이미지의 큰 단점은 확대할 때 품질이 좋지 않다는 점입니다. PNG를 크게 만들어 본 적이 있다면 픽셀이 보이거나 흐릿해지는 것을 경험했을 겁니다.

SVG는 다른 종류의 이미지입니다. SVG는 확장 가능한 벡터 그래픽(scalable vector graphic)의 약자입니다. 벡터 그래픽은 점, 선, 곡선을 그리기 위해 경로와 방정식을 기반으로 데이터를 추적합니다. 이것은 SVG 같은 벡터 그래픽이 크기를 아무리 키워도 품질에 영향을 주지 않는다는 뜻입니다.

특히 SVG는 XML로 데이터를 저장하는 장점이 있습니다. 즉, `svg` 요소로 원시 HTML 코드에 직접 사용할 수 있습니다. 또한 프로그래밍 방식으로 이미지 색상을 변경할 수도 있습니다.

스마일 얼굴을 빨간색으로 바꾸려면, 인터랙티브 편집기를 활성화하고 `fill="yellow"`을 `fill="red"`로 변경하세요.

:::interactive_editor

```html
<svg width="100" height="100" viewBox="0 0 100 100" xmlns="http://www.w3.org/2000/svg">
  <circle cx="50" cy="50" r="45" stroke="black" stroke-width="4" fill="yellow" />
  <circle cx="35" cy="40" r="5" fill="black" />
  <circle cx="65" cy="40" r="5" fill="black" />
  <path d="M35 65 Q50 80 65 65" stroke="black" stroke-width="4" fill="transparent" />
</svg>
```

:::

예제의 기본 요소는 다음과 같습니다:

- `svg` 요소는 전체 그림의 컨테이너입니다. 모든 도형이 나타나는 공간을 설정합니다. 원, 선, 경로 등 SVG로 그리고 싶은 모든 것은 `svg` 요소 안에 들어갑니다.
- `circle` 요소는 얼굴과 눈을 만드는데 사용됩니다. 큰 원 하나가 노란색 얼굴을 만들고, 두 개의 작은 원이 눈을 만듭니다.
- `path` 요소는 미소를 그리는 데 사용됩니다. 입을 위한 곡선선을 만듭니다.
- 각 SVG 요소는 그림 영역 내에서 모양과 위치를 제어하는 속성을 가집니다.

몇 가지 예제를 더 보여드리겠습니다. 예제 중 어느 하나의 색상을 바꾸려면 `fill` 속성 값을 `red`, `green`, `blue`, `yellow` 같은 이름 있는 색상으로 변경하세요.

:::interactive_editor

```html
<!-- Star Icon -->
<svg width="50" height="50" viewBox="0 0 24 24" fill="gold" xmlns="http://www.w3.org/2000/svg">
  <path d="M12 2L14.9 8.6L22 9.3L17 14.1L18.3 21.2L12 17.8L5.7 21.2L7 14.1L2 9.3L9.1 8.6L12 2Z"/>
</svg>

<!-- Heart Icon -->
<svg width="50" height="50" viewBox="0 0 24 24" fill="crimson" xmlns="http://www.w3.org/2000/svg">
  <path d="M12 21.35L10.55 20.03C5.4 15.36 2 12.28 2 8.5C2 6 4 4 6.5 4C8 4 9.5 4.8 10.5 6.09C11.5 4.8 13 4 14.5 4C17 4 19 6 19 8.5C19 12.28 15.6 15.36 10.45 20.04L12 21.35Z"/>
</svg>

<!-- Checkmark Icon -->
<svg width="50" height="50" viewBox="0 0 24 24" fill="green" xmlns="http://www.w3.org/2000/svg">
  <path d="M20.29 5.71L9 17L3.71 11.71L5.12 10.29L9 14.17L18.88 4.29L20.29 5.71Z"/>
</svg>
```

:::

그렇다면 언제 SVG를 사용해야 할까요? 아이콘에 아주 적합합니다. 맞춤형 글머리 기호를 만들거나 소셜 미디어 플랫폼을 나타내는 아이콘을 링크에 추가하고 싶을 때, SVG를 사용하는 것이 가장 좋은 방법입니다. 가장 인기 있는 아이콘 라이브러리 중 하나인 Font Awesome도 아이콘에 SVG 이미지를 사용합니다. SVG는 웹페이지 로고에도 좋습니다. 완벽하게 크기를 조절할 수 있기 때문입니다. 반응형 디자인에 맞춰 레이아웃을 조정할 수 있게 해줍니다. 다음에 로컬에 SVG 파일이 있다면 텍스트 편집기로 열어 코드를 직접 만져보세요.
