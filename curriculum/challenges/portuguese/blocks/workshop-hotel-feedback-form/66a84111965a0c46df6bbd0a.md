# --description--

Nas lições anteriores, você aprendeu como associar um elemento `label` com um `input` assim:

```html
<label for="name">Name:</label>
<input type="text" id="name" name="name">
```

O atributo `for` no elemento `label` deve corresponder ao atributo `id` no elemento `input`. Isso é conhecido como uma associação explícita.

Abaixo do seu elemento `legend`, adicione um elemento `label` com o texto `Name (required):`. Defina seu atributo `for` com o valor `"full-name"`.

Então, abaixo do seu elemento `label`, adicione um elemento `input` sem atributos. Nos próximos passos, você adicionará os atributos necessários.
