# --interactive--

Unapofanya kazi na JavaScript, mara nyingi utakutana na miundo ya data tata inayojumuisha vitu vilivyopangwa ndani na safu za data ndani ya vitu. Miundo hii inaweza wakilisha data tajiri yenye ngazi nyingi, lakini pia inahitaji uelewa wazi wa jinsi ya kupata na kusimamia data ndani yake. Hebu tuchunguze jinsi ya kuvinjari miundo hii iliyopangwa ndani kwa ufanisi.

Kupata sifa kutoka kwa vitu vilivyopangwa ndani kunahusisha kutumia uandishi wa nukta au uandishi wa mabano, kama vile unavyopata sifa kutoka kwa vitu rahisi. Hata hivyo, utahitaji kuunganisha njia hizi za upatikanaji ili kufikia ndani ya muundo uliopangwa ndani.

Kwa mfano, tuchukulie kitu kilichopangwa ndani kinachowakilisha mtu mwenye taarifa za mawasiliano:

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};
```

Ili kupata nambari ya simu ya kazi ya `Alice`, utahitajika kuunganisha njia za upatikanaji wa sifa kama ifuatavyo:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person.contact.phone.work); // "098-765-4321"
```

:::

Pia unaweza kutumia uandishi wa mabano, ambao ni muhimu hasa wakati majina ya sifa yanajumuisha nafasi au herufi maalum, au unapotumia vigezo kupata sifa:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person['contact']['phone']['work']); // "098-765-4321"
```

:::

Sasa, tuchunguze jinsi tunavyoweza kupata data ambapo moja ya sifa za kitu ina thamani ya safu ya data. Hapa kuna mfano wa `person` uliobadilishwa unaojumuisha safu ya anwani:

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};
```

Hapa kuna mfano wa jinsi ya kupata jiji la anwani ya kazi ya `Alice`:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};

console.log(person.addresses[1].city); // "Workville"
```

:::

Katika mfano huu, `person.addresses` inarejelea safu ya anwani. Ili kupata anwani ya pili katika safu hiyo, tunatumia uandishi wa mabano na kielezo cha `1`. Kisha, tunatumia uandishi wa nukta kupata `city` kutoka kwa kitu cha anwani hiyo.

Kuelewa jinsi ya kupata sifa katika vitu vilivyopangwa ndani na safu za data ni muhimu unapofanya kazi na miundo ya data tata. Katika warsha na maabara zijazo, utapata fursa ya kufanya mazoezi ya kazi na aina hizi za miundo ya data.
