# --explanation--

`Built-in`は、ソフトウェアプログラムやデバイスに最初から含まれていて、別途追加やインストールが不要な機能やツールを指します。

何かが`built-in`である場合、それはすでに存在していてすぐに使える状態です。例えば：

- `This laptop has a built-in camera.` - カメラはノートパソコンに組み込みです。

- `The software comes with built-in security features.` - セキュリティ機能はソフトウェアにすでに組み込まれています。
