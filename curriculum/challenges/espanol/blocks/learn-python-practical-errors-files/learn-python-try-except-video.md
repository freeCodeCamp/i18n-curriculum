## --text--

¿Cuál será el resultado del siguiente código si el usuario proporciona la cadena `"random"`?

```python
try:
    number = int(input("Enter a number: "))
    print(number)
except:
    print("Invalid Input")
```

## --answers--

Nada se mostrará en la consola.

---

La cadena `"random"` se mostrará en la consola.

---

La cadena `"Invalid Input"` se mostrará en la consola.

---

El programa se bloqueará.
