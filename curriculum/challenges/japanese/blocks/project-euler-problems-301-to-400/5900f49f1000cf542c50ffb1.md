# --description--

以下のゲームは組合せゲーム理論の古典的な例です：

2人のプレイヤーが長さ$n$の白いマス目のテープを持ち、交互に手番を行います。各手番で、プレイヤーは連続する2つの白いマスを選んで黒く塗ります。動けなくなったプレイヤーが負けです。

- $n = 1$: 有効な手がないため、先手は自動的に負けます。
- $n = 2$: 有効な手は1つだけで、その後後手が負けます。
- $n = 3$: 有効な手は2つありますが、どちらも後手が負ける状況を残します。
- $n = 4$: 先手には3つの有効な手があり、真ん中の2マスを塗ることで勝利できます。
- $n = 5$: 先手には4つの有効な手（下図の赤色）があり、どの手を選んでも後手（青色）が勝ちます。

<img alt="5マスのテープでの有効な初手" src="https://cdn.freecodecamp.org/curriculum/project-euler/paper-strip-game.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

したがって、$1 ≤ n ≤ 5$ の範囲では、先手が必勝できる$n$の値は3つあります。

同様に、$1 ≤ n ≤ 50$ の範囲では、先手が必勝できる$n$の値は40個あります。

$1 ≤ n ≤ 1\,000\,000$ の範囲で、先手が必勝できる$n$の値はいくつあるでしょうか？
