# --description--

令 $f_5(n)$ 为最大的整数 $x$，使得 $5^x$ 整除 $n$。

例如，$f_5(625\\,000) = 7$。

令 $T_5(n)$ 为满足 $f_5((2 \times i - 1)!) < 2 \times f_5(i!)$ 且 $1 ≤ i ≤ n$ 的整数 $i$ 的数量。

可以验证 $T_5({10}^3) = 68$ 和 $T_5({10}^9) = 2\\,408\\,210$。

查找 $T_5({10}^{18})$。
