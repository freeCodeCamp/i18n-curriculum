# --description--

在上一个例子中，你有两个选择 `if` 或 `else`。但是如果你想有超过两个选择怎么办？你可以使用 `else if` 语句为我们的代码添加更多条件。`else if` 语句允许我们查看多个条件，并在第一个条件为真时执行一块代码。如果第一个条件为假，它将查看下一个条件，依此类推。如果没有任何条件为真，将执行 `else` 块。

下面是 `else if` 语句的示例：

```javascript
let x = 5;

if (x > 10) {
  console.log("x is greater than 10");
} else if (x > 5) {
  console.log("x is greater than 5");
} else {
  console.log("x is less than or equal to 5");
}
```

在此示例中，使用 `>` 操作符将 `x` 变量与数字 `10` 进行比较。如果 `x` 大于 `10`，将执行第一块代码。否则，`else if` 语句将检查下一个条件。如果 `x` 大于 `5`，将执行第二块代码。否则，将执行 `else` 块。
