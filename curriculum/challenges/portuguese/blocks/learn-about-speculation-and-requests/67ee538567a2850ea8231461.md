# --explanation--

`What did I do wrong?` é usado quando alguém quer entender seu erro.É comumente usado em diferentes situações.Por exemplo:

- `I got an error in my code. What did I do wrong?` - O falante quer saber qual erro ele cometeu no código dele.

- `My presentation didn't go well. What did I do wrong?` - O palestrante está procurando maneiras de melhorar.

Esta frase é útil para aprender com os erros e melhorar em diferentes situações.Mark usa isso para entender seu erro.
