# --explanation--

`We'd prefer` 是一種禮貌地表達你想要更多的說法。 例如：

`We'd prefer faster processors for the new computers.` – 這表示對更好選項的強烈興趣。

`Since` 用來說明決定或情況背後的原因。例如：

`We upgraded the servers since the old ones were too slow.`－這說明了進行升級的原因。
