# --explanation--

`It seems to be` + kitenzi katika fomu ya `-ing` hutumika kuelezea jambo ambalo linaonekana kutokea, ingawa halijathibitishwa. Kwa mfano:

`The app seems to be running slower today.` – Hii ina maana inaonekana kama programu ni polepole zaidi kuliko kawaida.

`To cause something to` + kitenzi ina maana ya kusababisha jambo lifanyike au kuanzisha kitendo. Kwa mfano:

`Low memory can cause the program to crash.` – Hii ina maana kumbukumbu ndogo ndiyo sababu programu inakoma kufanya kazi.
