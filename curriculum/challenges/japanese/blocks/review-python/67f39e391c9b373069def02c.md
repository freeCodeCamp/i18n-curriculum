## Pythonとは何ですか？

- **概要**: Pythonはシンプルで使いやすいことで知られる汎用プログラミング言語です。Pythonはデータサイエンスや機械学習、ウェブ開発、スクリプトや自動化、組み込みシステムやIoTなど多くの分野で使われています。
- **一般的な使用例**: Pythonはデータサイエンス、機械学習、ウェブ開発、サイバーセキュリティ、自動化、Raspberry PiやMicroPython対応ボードのようなマイクロコンピュータで使われています。

## ローカル環境でのPython

- **インストール**: Windows、Mac、LinuxでPythonをインストールする最良の方法は、公式Pythonウェブサイトからインストーラーをダウンロードすることです(`https://www.python.org/`)。
- **ターミナル**: コマンドを入力してコンピュータとやり取りできるテキストベースのインターフェースです。各OSにはデフォルトのターミナルアプリがあります。macOSではTerminalアプリ、WindowsではCommand PromptやPowerShell、LinuxではGNOME TerminalやKonsoleなどのデスクトップ環境ごとのターミナルがあります。
- **IDE**: IDEは統合開発環境の略で、コードエディタ、テストツール、ターミナルなどの機能を備えています。
- **Python用の人気コードエディタとIDE**: VS Code、PyCharm、Spyder
- **ローカルでコードを実行する**: Pythonコードを実行するにはいくつかの方法があります。1つ目は、VS Codeの右上にある再生ボタンのような実行ボタンをクリックすることです。これでターミナルが開き、Pythonスクリプトが実行されます。

もう1つの方法はターミナルを開いて手動でプログラムを実行することです。例えば、`main.py`というプログラムを実行する例は以下の通りです。

```bash
python main.py
```

このコマンドは`main.py`が保存されているフォルダから実行する必要があります。例えば、`main.py`が`python-projects`というフォルダ内にある場合、まずターミナルで以下を使います。

```bash
cd python-projects
```

次に以下を実行します。

```bash
python main.py
```

## Pythonインタラクティブシェルの使い方

- **定義**: インタラクティブシェルはコマンドを1つずつ入力して結果を確認できるプログラムです。ターミナルアプリを開いて`python`と入力し、`Enter`を押してください。これでPythonインタラクティブシェルが起動します。`>>>`の記号はPythonがコマンド入力を待っていることを示します。ターミナルに`"Hello, world!"`を出力してみましょう。

```bash
print("Hello, world!")
```

以下のようにテキストが表示されるはずです。

```bash
>>> print("Hello, world!")
Hello, world!
```

テキストが表示された後、Pythonは以下に戻ります。

```bash
>>>
```

これはさらにコマンドを入力できる状態です。

- **読み取り、評価、出力、ループサイクル（REPL）**: コマンドを入力すると、インタプリタは入力を読み取り、評価し、結果を出力し、再び`>>>`を表示して次のコマンド入力を待ちます。

## 変数

- **変数の宣言**: 変数を宣言するには、変数名の後に代入演算子(`=`)を置き、その後に値を代入します。値は数値、文字列、ブール値などが使えます。例をいくつか示します。

```py
name = 'John Doe'
age = 25
```

- **変数名の命名規則**: 変数名には以下の規則があります。

- 変数名は文字かアンダースコア(_)で始める必要があり、数字では始められません。
  - 変数名には英数字(a-z、A-Z、0-9)とアンダースコア(_)のみ使用できます。
  - 変数名は大文字と小文字を区別します。`age`、`Age`、`AGE`はすべて異なるものとみなされます。
  - 変数名はPythonの予約キーワード（`if`、`class`、`def`など）を使えません。
  - 複数単語の変数名はアンダースコアで区切ります。例：`snake_case`。


## コメント

- **単一行コメント**: コード内に短いメモを残すために使います。

```py
# This is a single line comment
```

- **複数行文字列**: 大きなメモを残したり、コードの一部をコメントアウトしたりするのに使えます。

```py
"""
This is a multi-line string.
Here is some code commented out.

name = 'John Doe'
age = 25
"""
```

- **`print()`関数**: コンソールにデータを出力するには、`print()`関数を次のように使います。

```py
print('Hello world!') # Hello world!
```

## Pythonの一般的なデータ型

- **概要**: PythonはJavaScriptのような動的型付け言語で、変数の型を明示的に宣言する必要はありません。変数に代入された値によって型が決まります。
- **整数**: 小数点のない整数です。

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **浮動小数点数**: 小数点を含む数値です。

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **文字列**: 引用符で囲まれた文字の並びです。

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **ブール値**: `True`または`False`を表す値です。

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **セット**: 順序のない一意の要素の集合です。

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **辞書**: 波括弧で囲まれたキーと値のペアの集合です。

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **タプル**: 丸括弧で囲まれた変更不可能な順序付きコレクションです。

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **レンジ**: ループでよく使われる数値のシーケンスです。

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **リスト**: 異なるデータ型を含められる順序付きコレクションです。

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**: 値が存在しないことを表す特別な値です。

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## 不変型と可変型

- **不変型**: 宣言後に変更できませんが、変数に新しい値を代入することはできます（再代入）。整数、浮動小数点数、複素数、ブール値、文字列、タプル、レンジ、そして`None`が含まれます。
- **可変型**: 宣言後に変更可能で、要素の追加、削除、更新ができます。リスト、セット、辞書などのコレクション型が含まれます。
- **`type()`関数**: 変数の型を確認するには、`type()`関数を次のように使います。

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **`isinstance()`関数**: 変数が特定のデータ型かどうかをチェックするのに使います。

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## 文字列の操作

- **定義**: JavaScriptで学んだように、文字列は不変で、一度作成したら変更できません。Pythonではシングルクォートかダブルクォートのどちらでも使えます。ルールを決めて一貫して使うことをお勧めします。

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **文字列の文字にアクセスする**: ブラケット記法を使って文字列の文字にアクセスできます。

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **文字列のエスケープ**: 文字列に引用符が含まれる場合はバックスラッシュ(`\`)を使います。

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **文字列の連結**: 文字列を連結するには`+`演算子を使います。

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

もう1つの連結方法は`+=`演算子を使うことです。これは連結と代入を同時に行います。

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**: これはフォーマット済み文字列リテラルの略称で、補間や連結を簡潔で読みやすい構文で行えます。

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **文字列のスライス**: 文字列の一部を抽出することです。基本構文は以下の通りです。

```py
str[start:stop:step]
```

開始位置は抽出を始めるインデックス、終了位置はスライスの終わり（この位置は含まれません）、ステップはスライスの間隔を表します。例を示します。

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **文字列の長さを取得する**: `len()`関数は文字列の文字数を返します。

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## `in`演算子の操作

- **`in`演算子**: 文字列に指定した文字が存在するかどうかを示すブール値を返します。

```py
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

## よく使う文字列メソッド

- **`upper()`**: すべての文字を大文字に変換した新しい文字列を返します。

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`lower()`**: すべての文字を小文字に変換した新しい文字列を返します。

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`strip()`**: 指定した先頭と末尾の文字を削除した文字列のコピーを返します（引数がない場合は空白を削除します）。

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**: 古い文字列を新しい文字列にすべて置き換えた新しい文字列を返します。

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**: 指定した区切り文字で文字列を分割し、リストにします。区切り文字は分割位置を指定する文字列です。

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**: イテラブルの要素を区切り文字で連結して文字列にします。イテラブルとはリスト、文字列、タプルのようにループできるコレクションです。

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`startswith(prefix)`**: 文字列が指定した接頭辞で始まるかどうかを示すブール値を返します。

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`endswith(suffix)`**: 文字列が指定した接尾辞で終わるかどうかを示すブール値を返します。

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`find()`**: 部分文字列が最初に現れるインデックスを返します。見つからなければ`-1`を返します。

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`count(substring)`**: 部分文字列が文字列内に何回現れるかを数えます。

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`capitalize()`**: 最初の文字を大文字にし、残りを小文字にした新しい文字列を返します。

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`isupper()`**: 文字列のすべての文字が大文字なら`True`を返し、そうでなければ`False`を返します。

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`islower()`**: 文字列のすべての文字が小文字なら`True`を返し、そうでなければ`False`を返します。

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`title()`**: 各単語の最初の文字を大文字にした新しい文字列を返します。

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`maketrans()`**: 1対1の文字変換テーブルを作成します。通常は`translate()`メソッドと組み合わせて使い、文字列に適用して変換結果を返します。

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## 整数と浮動小数点数でよく使う操作

- **基本的な算術演算**: Pythonでは整数と浮動小数点数で加算、減算、乗算、除算ができます。

```py
int_1 = 56
int_2 = 12
float_1 = 5.4
float_2 = 12.0

# Addition

print('Integer Addition:', int_1 + int_2) # Integer Addition: 68
print('Float Addition:', float_1 + float_2) # Float Addition: 17.4

# Subtraction

print('Int Subtraction:', int_1 - int_2) # Int Subtraction: 44
print('Float Subtraction:',  float_2 - float_1) # Float Subtraction: 6.6

# Multiplication

print('Int Multiplication:', int_1 * int_2) # Int Multiplication: 672
print('Float Multiplication:', float_2 * float_1) # Float Multiplication: 64.80000000000001

# Division

print('Division:', int_1 / int_2) # Division: 4.666666666666667
print('Float Division:', float_2 / float_1) # Float Division: 2.222222222222222
```

浮動小数点数と整数を加算すると結果は浮動小数点数になります。

```py
int_1 = 56
float_1 = 5.4

print(int_1 + float_1) # 61.4
```

- **剰余演算子（`%`）**: ある数を別の数で割った余りを返します。

```py
int_1 = 56
int_2 = 12

print(int_1 % int_2) # 8
```

- **切り捨て除算（`//`）**: 2つの数を割り、結果を小数点以下切り捨てで整数にします。

```py
int_1 = 56
int_2 = 12

print(int_1 // int_2) # 4
```

- **べき乗演算子（`**`）**: 数値を別の数のべき乗にします。

```py
int_1 = 4
int_2 = 2

print(int_1 ** int_2) # 16
```

- **`float()`関数**: 整数を浮動小数点数に変換します。

```py
num = 4

print(float(num)) # 4.0
```

- **`int()`関数**: 浮動小数点数を整数に変換します。

```py
num = 4.0

print(int(num)) # 4
```

- **`round()`関数**: 数値を最も近い整数に丸めます。

```py
num_1 = 3.4
num_2 = 7.7

print(round(num_1)) # 3
print(round(num_2)) # 8
```

- **`abs()`関数**: 数値の絶対値を返します。

```py
num = -13

print(abs(num)) # 13
```

- **`pow()`関数**: 数値を別の数のべき乗にします。

```py
result = pow(2, 3) 
print(result)  # 8
```

## 複合代入

- **定義**: 複合代入は二項演算と代入を1つのステップで行います。変数に演算を適用し、その結果を同じ変数に代入します。

```py
# Addition assignment 
my_var = 10
my_var += 5

print(my_var) # 15

# Subtraction assignment
count = 14
count -= 3

print(count) # 11

# Multiplication assignment 
product = 65
product *= 7

print(product) # 455

# Division assignment 
price = 100
price /= 4

print(price) # 25.0

# Floor Division assignment 
total_pages = 23
total_pages //= 5

print(total_pages) # 4

# Modulo assignment 
bits = 35
bits %= 2

print(bits) # 1

# Exponentiation assignment 
power = 2
power **= 3

print(power) # 8
```

ビット演算子用の複合代入演算子もあります。`&=`、`^=`、`>>=`、`<<=`などです。

## 関数の使い方

- **定義**: 関数は入力（引数）を受け取り、出力を返す再利用可能なコードの塊です。関数を呼び出すには、関数名の後に丸括弧を付けます。

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call
print(result) # 7
```

関数が明示的に値を返さない場合、戻り値はデフォルトで`None`です。

```py
def greet():
    print('hello') 

result = greet() # hello
print(result) # None
```

パラメータにデフォルト値を設定することもできます。

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

引数の数が正しくない場合、`TypeError`が発生します。

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

## Pythonのスコープ

- **ローカルスコープ**: 関数やクラス内で宣言された変数は、その関数やクラス内でのみアクセスできます。

```py
def my_func():
    num = 10
    print(num)
```

- **囲むスコープ**: 関数の中にネストされた関数は、外側の関数の変数にアクセスできます。

```py
def outer_func():
    msg = 'Hello there!'
    
    def inner_func():
        print(msg)
    inner_func()

print(outer_func()) # Hello there!
```

- **グローバルスコープ**: 関数やクラスの外で宣言された変数で、プログラムのどこからでもアクセスできます。

```py
tax = 0.70 

def get_total(subtotal):
    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **組み込みスコープ**: Pythonの予約名で、組み込み関数、モジュール、キーワード、オブジェクトが含まれます。

```py
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```

## 比較演算子

- **等しい（`==`）**: 2つの値が等しいかチェックします。

```py
print(3 == 4) # False
```

- **等しくない（`!=`）**: 2つの値が等しくないかチェックします。

```py
print(3 != 4) # True
```

- **より大きい（`>`）**: 1つの値がもう1つより大きいかチェックします。

```py
print(3 > 4) # False
```

- **より小さい（`<`）**: 1つの値がもう1つより小さいかチェックします。

```py
print(3 < 4) # True
```

- **以上（`>=`）**: 1つの値がもう1つ以上かチェックします。

```py
print(3 >= 4) # False
```

- **以下（`<=`）**: 1つの値がもう1つ以下かチェックします。

```py
print(3 <= 4) # True
```

## `if`、`elif`、`else`文の使い方

- **`if`文**: 何かが真かどうかを判断する条件です。条件が`True`と評価されると、そのコードブロックが実行されます。

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **`elif`節**: `if`文の後に続く条件です。`elif`条件が`True`と評価されると、そのコードブロックが実行されます。

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **`else`節**: 他の条件が`True`と評価されなかった場合に実行されます。

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

ネストされた`if`文も使えます。

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## 真値と偽値

- **定義**: Pythonではすべての値に論理的な文脈で真（`True`）か偽（`False`）として扱われる組み込みのブール値があります。多くの値は真値で、論理的に`True`と評価されます。偽値は`False`と評価されます。偽値の例を示します。

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

非ゼロの数値や空でない文字列は真値です。

## `bool()`関数の使い方

- **定義**: 値が真値か偽値かをチェックしたい場合、組み込みの`bool()`関数を使います。値を明示的にブール値に変換し、真値なら`True`、偽値なら`False`を返します。例をいくつか示します。

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## ブール演算子とショートサーキット

- **定義**: 複数の式を組み合わせて複雑な条件判断を作る特別な演算子です。Pythonには3つのブール演算子があります：`and`、`or`、`not`。
- **`and`演算子**: 2つのオペランドを取り、最初のオペランドが偽値ならそれを返し、そうでなければ2番目を返します。両方が真値の場合のみ式は真値になります。

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

条件文で`and`演算子も使えます。

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **`or`演算子**: 最初のオペランドが真値ならそれを返し、そうでなければ2番目を返します。`or` 式は、少なくとも一方のオペランドが真であれば真となります。例を示します。

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

`and`演算子と同様に、条件文で`or`演算子も使えます。

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **ショートサーキット**: `and`と`or`演算子はショートサーキット演算子と呼ばれます。左から右に値をチェックし、結果が確定した時点で評価を停止します。
- **`not`演算子**: 1つのオペランドを取り、そのブール値を反転します。真値を`False`に、偽値を`True`に変換します。前述の演算子とは異なり、`not`は常に`True`か`False`を返します。例を示します。

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

条件文での`not`演算子の例です。

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```


## Pythonのリスト

- **概要**: Pythonのリストは文字列、数値、他のリストなどを含む順序付きの要素のシーケンスです。リストは可変でゼロベースのインデックスです。

```python
cities = ['Los Angeles', 'London', 'Tokyo']
```

- **リストの要素にアクセスする**: `cities`リストの要素にアクセスするには、シーケンスのインデックス番号を参照します。

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # Los Angeles
```

- **負のインデックスで要素にアクセスする**: リストの最後の要素にアクセスするには、インデックス番号として`-1`を使います。

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # Tokyo
```

- 負のインデックスはリストの末尾から要素にアクセスするために使われ、先頭のインデックス`0`とは逆方向です。

- **`list()`コンストラクタでリストを作成する**: リストは`list()`コンストラクタを使って作成することもできます。`list()`コンストラクタはイテラブルをリストに変換します。

```python
developer = 'Jessica'

print(list(developer)) 
# Result: ['J', 'e', 's', 's', 'i', 'c', 'a']
```

- **リストの長さを調べる**: `len()`関数を使ってリストの長さを取得できます。

```python
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

- **リストの可変性**: リストは可変で、有効なインデックス番号を指定すれば要素を更新できます。特定のインデックスのリストを更新するには、そのインデックスに新しい値を代入します。

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

- **インデックス範囲外エラー**: リストの範囲外のインデックス（正または負）を指定すると、`IndexError`が発生します。

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
"""
```

- **リストから要素を削除する**: `del`キーワードを使ってリストから要素を削除できます。

```python
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

- **リストに要素が存在するか確認する**: `in`キーワードを使ってリストに要素があるか確認できます。

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **リストのネスト**: リストは他のリストの中にネストできます。

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

- ネストされたリストにアクセスするには、リストがゼロベースインデックスなのでインデックス`2`を使います。

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

- さらにネストされたリストから2番目の言語にアクセスするには、インデックス`1`を使います。

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # Rust
```

- **リストの値のアンパック**: リストの値を新しい変数に割り当てるテクニックです。`developer`リストを`name`、`age`、`job`という新しい変数にアンパックする例です。

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer
```

- 左辺の変数の数がリストの要素数と一致しない場合、`ValueError`が発生します。

- **リストの残りの要素をまとめる**: 残りの要素をまとめるにはアスタリスク(`*`)演算子を使います。

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer
```

- **リストのスライス**: スライス演算子`:`を使ってリストの一部にアクセスします。インデックス`1`から`3`の手前までスライスする構文は以下の通りです。

```python
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie']
desserts[1:3] # ['Cookies', 'Ice Cream']
```

- **ステップ間隔**: インデックス間の増分を指定することも可能です。偶数だけのリストをスライスで抽出する例です。

```python
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

## リストのメソッド

- **append()**: リストの末尾にアイテムを追加します。`append()`メソッドを使って`6`という数値を`numbers`リストに追加する例です。

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **リストの追加**: `append()`メソッドを使って1つのリストを別のリストの末尾に追加できます。

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

- **extend()**: 複数のアイテムをリストの末尾に追加します。`6`、`8`、`10`という数値を`numbers`リストに追加する例です。

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

- **insert()**: リストの特定のインデックスにアイテムを挿入します。`insert()`メソッドの例です。

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

- **remove()**: リストからアイテムを削除します。`remove()`メソッドはリスト内の最初の該当アイテムのみ削除します。

```py
numbers = [1, 2, 3, 4, 5, 5, 5]
numbers.remove(5)

print(numbers) # [1, 2, 3, 4, 5, 5]

```

- **pop()**: リストから特定のアイテムを削除して返します。

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

- `pop`メソッドで要素を指定しない場合は、最後の要素が削除されます。

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

- **clear()**: リストのすべてのアイテムを削除します。

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

- **sort()**: `sort()`メソッドはリストの要素をその場でソートします。ランダムな`numbers`リストをソートする例です。

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

- **sorted()**: 元のリストを変更せずに新しいソート済みリストを返す組み込み関数です。

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
print(numbers) # [19, 2, 35, 1, 67, 41]
```

- **reverse()**: リストの要素の順序を逆にします。

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **index()**: リスト内で要素が最初に見つかるインデックスを返します。

```python
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

- `index()`メソッドで要素が見つからない場合、結果は`ValueError`になります。

## Pythonのタプル

- **定義**: タプルは順序付きの値のシーケンスを作るPythonのデータ型です。異なるデータ型を混在させることもできます。

```py
developer = ('Alice', 34, 'Rust Developer')
```

- タプルは不変で、一度作成した要素は変更できません。要素を更新しようとすると`TypeError`が発生します。

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **タプルの要素にアクセスする**: ブラケット記法とインデックス番号を使います。

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

- 負のインデックスを使ってタプルの末尾から要素にアクセスできます。

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- タプルの長さを超えるか等しいインデックス番号を指定すると`IndexError`が発生します。

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- タプルは`tuple()`コンストラクタを使って作成することもできます。コンストラクタには文字列、リスト、他のタプルなどのイテラブルを渡せます。

```python
developer = 'Jessica'

print(tuple(developer)) 
# Result: ('J', 'e', 's', 's', 'i', 'c', 'a')
```

- **タプル内のアイテムの確認**: `in`キーワードを使ってタプルにアイテムがあるか確認できます。

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **タプルのアンパック**: タプルからアイテムをアンパックできます。

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

- タプルの残りの要素をまとめるにはアスタリスク(`*`)演算子を使います。

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer
```

- **タプルのスライス**: タプルの一部を抽出できます。例えば、`pie`と`cookies`のアイテムを別のタプルにスライスします。

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

- **タプルからのアイテム削除**: タプルは不変なので、アイテムを削除しようとすると`TypeError`が発生します。

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

- **タプルとリストの使い分け**: 要素の追加、削除、更新が必要な動的なコレクションにはリストを使います。固定で不変のコレクションにはタプルを使います。

## タプルの一般的なメソッド

- **`count()`**: タプル内にアイテムが何回出現するかを調べるために使います。例えば、タプル内に言語 `'Rust'` が何回出てくるかを確認できます。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

- `count()` メソッドで指定したアイテムがタプルに全く存在しない場合、戻り値は `0` になります。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

- `count()` メソッドに引数を渡さない場合、Python は `TypeError` を返します。

- **index()**: タプル内の特定のアイテムが存在するインデックスを見つけるために使います。以下は言語 `'Java'` のインデックスを見つけるために `index()` メソッドを使った例です。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

- 指定したアイテムが見つからない場合、Python は `ValueError` を返します。

- `index()` メソッドにオプションの開始インデックスを渡して、タプル内の検索開始位置を指定できます。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

- `index()` メソッドにオプションの終了インデックスを渡して、タプル内の検索終了位置を指定することもできます。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

- **`sorted()`**: 任意のイテラブルの要素をソートして、新しいソート済みリストを返すために使います。以下は `sorted()` 関数を使って新しい数値リストを作成する例です。

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

- **ソート動作の変更**: オプションの `reverse` と `key` 引数を使ってイテラブルのソート動作をカスタマイズできます。以下は `key` 引数を使ってタプルのアイテムを長さでソートする例です。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

- `reverse` 引数を使って逆順の新しいリストを作成することもできます。

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

## Pythonのループ

- **定義**: ループはコードのブロックを決まった回数繰り返すために使います。

- **`for` ループ**: シーケンス（リスト、タプル、文字列など）を反復処理し、そのシーケンス内の各アイテムに対してコードブロックを実行します。以下は `for` ループを使ってリストを繰り返し処理し、各言語をコンソールに出力する例です。

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)

"""
Result 

Rust
Java
Python
C++
"""
```

- 以下は `for` ループを使って文字列 `code` をループし、各文字を出力する例です。

```py
for char in 'code':
    print(char)

"""
Result 

c
o
d
e
"""
```

- `for` ループはネスト可能です。以下はネストされた `for` ループの例です。

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)

"""
Result

Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
"""

```

- **`while` ループ**: 条件が `False` になるまでコードブロックを繰り返します。以下は `while` ループを使った推測ゲームの例です。

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')

"""
Result

Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
"""

```

- **`break` と `continue` 文**: ループの実行を変更するために使います。

- `break` 文は特定の条件が満たされたときにループを即座に終了します。以下は `break` 文を使った `developer_names` のリストの例です。

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

- `continue` 文は現在の繰り返しをスキップして次の繰り返しに進みます。以下は `continue` 文を `break` 文の代わりに使う例です。

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

- `for` と `while` ループは `else` 節と組み合わせることができ、ループが `break` によって終了しなかった場合にのみ実行されます。

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

## ループでの範囲の利用

- **`range()` 関数**: 整数のシーケンスを生成するために使います。

```py
range(start, stop, step)
```

- 必須の `stop` 引数は整数（非包含）で、生成される数列の終点を表します。以下は `range()` 関数の使用例です。

```py
for num in range(3):
    print(num)
```

- `start` 引数が指定されない場合、デフォルトは `0` になります。デフォルトでは整数のシーケンスは `1` ずつ増加します。オプションの `step` 引数を使ってデフォルトの増分値を変更できます。以下は 2 から 11 未満（つまり 10 を含む）までの偶数のシーケンスを生成する例です。

```py
for num in range(2, 11, 2):
    print(num)
```

- `range()` 関数に引数を渡さない場合、`TypeError` が返されます。

- `range()` 関数は引数に整数のみを受け付け、浮動小数点数は受け付けません。浮動小数点数を使うと `TypeError` が発生します。

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- `step` 引数に負の整数を使うと、減少順の整数シーケンスを生成できます。

```py
for num in range(40, 0, -10):
    print(num)
```

- `range()` 関数は `list` コンストラクタと組み合わせて整数のリストを作成することもできます。`list` コンストラクタはイテラブルをリストに変換するために使います。以下は 2 から 10 までの偶数のリストを生成する例です。

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

## Pythonの `enumerate()` と `zip()` 関数

- **`enumerate()`**: シーケンスを反復処理し、そのシーケンス内の各アイテムのインデックスを追跡するために使います。`enumerate()` 関数はイテラブルを引数に取り、インデックスと値のペアからなる `enumerate` オブジェクトを返します。

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')

# Result
# Index 0 and language Spanish
# Index 1 and language English
# Index 2 and language Russian
# Index 3 and language Chinese

```

- `enumerate()` 関数は `for` ループの外でも使えます。

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

print(list(enumerate(languages)))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

- `enumerate()` 関数はオプションの `start` 引数を受け付け、カウントの開始値を指定します。この引数を省略するとカウントは `0` から始まります。

- **`zip()`**: 複数のイテラブルを並行して反復処理するために使います。以下は `zip()` 関数を使って `developers` と `ids` を反復処理する例です。

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
```

## Pythonのリスト内包表記

- **定義**: リスト内包表記は、ループと条件を大括弧内に直接組み合わせて1行で新しいリストを作成する方法です。コードが短くなり、読みやすくなることが多いです。

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

## イテラブル関数

- **`filter()`**: 条件に基づいてイテラブルから要素をフィルタリングします。条件を満たす要素だけを含むイテレータを返します。以下は4文字より長い単語だけの新しいリストを作成する例です。

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**: イテラブルの各アイテムに関数を適用し、結果の新しいイテラブルを返します。以下は `map()` 関数を使って摂氏温度のリストを華氏に変換する例です。

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**: リストやタプルのようなイテラブルの合計を取得します。以下は `sum()` 関数の使用例です。

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- オプションの `start` 引数を渡して合計の初期値を設定することもできます。以下は `start` 引数を位置引数として使った更新例です。

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- また、`start` 引数をキーワード引数として使うこともできます。

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## ラムダ関数

- **定義**: Pythonのラムダ関数は名前のない（無名の）関数を簡潔に作成する方法です。
- ラムダ関数は他の関数の引数としてよく使われます。以下はラムダ関数の例です。

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

- ラムダ関数のベストプラクティスは、変数に代入しないこと、シンプルで読みやすく保つこと、短く一時的な関数に使うことです。

## 辞書

- **辞書**: 辞書はキーと値のペアを格納する組み込みのデータ構造です。キーは不変のデータ型である必要があります。Python辞書の一般的な構文は以下の通りです。

```python
dictionary = {
    key1: value1,
    key2: value2
}
```

- **`dict()` コンストラクタ**: `dict()` コンストラクタは辞書を作成する別の方法です。キーを最初の要素、値を2番目の要素とするタプルのリストを `dict()` コンストラクタに渡します。

```python
pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
```

- **ブラケット記法**: キーと値のペアの値にアクセスするには、ブラケット記法と呼ばれる構文を使います。

```python
dictionary[key]
```

## 辞書の一般的なメソッド

- **`get()` メソッド**: `get()` メソッドはキーに関連付けられた値を取得します。ブラケット記法に似ていますが、キーが存在しない場合にエラーを防ぐためのデフォルト値を設定できます。

```python
dictionary.get(key, default)
```

- **`keys()` と `values()` メソッド**: `keys()` と `values()` メソッドは辞書内のすべてのキーと値のビューオブジェクトをそれぞれ返します。ビューオブジェクトは辞書の内容を別コピーを作らずに見る方法です。

```python
pizza = {
    'name': 'Margherita Pizza',
    'price': 8.9,
    'calories_per_slice': 250
}

pizza.keys()
# dict_keys(['name', 'price', 'calories_per_slice'])

pizza.values()
# dict_values(['Margherita Pizza', 8.9, 250])
```

- **`items()` メソッド**: `items()` メソッドは辞書内のすべてのキーと値のペアを含むビューオブジェクトを返します。

```python
pizza.items()
# dict_items([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250)])
```

- **`clear()` メソッド**: `clear()` メソッドは辞書内のすべてのキーと値のペアを削除します。

```python
pizza.clear()
```

- **`pop()` メソッド**: `pop()` メソッドは最初の引数で指定したキーのキーと値のペアを削除し、その値を返します。キーが存在しない場合は、2番目の引数で指定したデフォルト値を返します。デフォルト値が指定されておらずキーが存在しない場合は `KeyError` が発生します。

```python
pizza.pop('price', 10)
pizza.pop('total_price') # KeyError
```

- **`popitem()` メソッド**: Python 3.7以降では `popitem()` メソッドは最後に挿入されたアイテムを削除します。

```python
pizza.popitem()
```

- **`update()` メソッド**: `update()` メソッドは別の辞書のキーと値のペアで辞書を更新します。共通のキーがある場合は値が上書きされ、新しいキーは新しいキーと値のペアとして追加されます。

```python
pizza.update({ 'price': 15, 'total_time': 25 })
```

## 辞書のループ処理

- **値の反復処理**: 辞書の値を反復処理したい場合は、`for` ループと `values()` を使って辞書のすべての値を取得できます。

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,
    'Headphones': 70,
}

for price in products.values():
    print(price)
```

出力:

```md
990
600
250
70
```

- **キーの反復処理**: 上記の `products` 辞書のキーを反復処理したい場合は、`products.keys()` または `products` を直接書くことができます。

```python
for product in products.keys():
    print(product)
    
# Or

for product in products:
    print(product)
```

出力:

```md
Laptop
Smartphone
Tablet
Headphones
```

- **キーと値のペアの反復処理**: キーと対応する値を同時に反復処理したい場合は `products.items()` を反復処理します。キーと対応する値の個別のタプルが得られます。

```python
for product in products.items():
    print(product)
```

出力:

```md
('Laptop', 990)
('Smartphone', 600)
('Tablet', 250)
('Headphones', 70)
```

キーと値を別々のループ変数に格納するには、カンマで区切る必要があります。最初の変数がキーを、2番目の変数が値を格納します。

```python
for product, price in products.items():
    print(product, price)
```

出力:

```md
Laptop 990
Smartphone 600
Tablet 250
Headphones 70
```

- **`enumerate()` 関数**: カウンターを追跡しながら辞書を反復処理したい場合は `enumerate()` 関数を呼び出せます。この関数は `enumerate` オブジェクトを返し、各アイテムに整数を割り当てます。カウンターは任意の数から開始できますが、デフォルトは0です。

インデックスとアイテムを別々のループ変数に割り当てるのが `enumerate()` の一般的な使い方です。例えば `products.items()` を使うと、インデックスに加えてキーと値のペア全体を取得できます。

```python
for index, product in enumerate(products.items()):
    print(index, product)
```

出力:

```md
0 ('Laptop', 990)
1 ('Smartphone', 600)
2 ('Tablet', 250)
3 ('Headphones', 70)
```

カウントの初期値をカスタマイズしたい場合は `enumerate()` に2番目の引数を渡せます。以下はカウントを1から開始する例です。

```python
for index, product in enumerate(products.items(), 1):
    print(index, product)
```

出力:

```md
1 ('Laptop', 990)
2 ('Smartphone', 600)
3 ('Tablet', 250)
4 ('Headphones', 70)
```

## セット

- **セット**: セットはPythonの組み込みデータ構造で、重複する値を許しません。セットはミュータブルで順序付けされていないため、要素は特定の順序で格納されず、インデックスやキーでアクセスできません。また、セットには数値、文字列、タプルのような不変データ型の値のみを含められます。

- **セットの定義**: セットを定義するには、波括弧の中に要素を書き、カンマで区切ります。

```python
my_set = {1, 2, 3, 4, 5}
```

- **空のセットの定義**: 空のセットを定義する場合は `set()` 関数を使う必要があります。空の波括弧だけを書くと自動的に辞書が作成されます。

```python
set() # Set
{}    # Dictionary
```

## セットの一般的なメソッド

- **`add()` メソッド**: `add()` メソッドを使ってセットに要素を追加できます。新しい要素を引数として渡します。

```python
my_set.add(6)
```

- **`remove()` と `discard()` メソッド**: セットから要素を削除するには `remove()` メソッドか `discard()` メソッドを使います。削除したい要素を引数として渡します。`remove()` メソッドは要素が見つからない場合 `KeyError` を発生させますが、`discard()` メソッドは発生させません。

```python
my_set.remove(4)
my_set.discard(4)
```

- **`clear()` メソッド**: `clear()` メソッドはセットのすべての要素を削除します。

```python
my_set.clear()
```

## 数学的なセット操作

- **`issubset()` と `issuperset()` メソッド**: `issubset()` と `issuperset()` メソッドはセットが別のセットの部分集合かスーパーセットかをそれぞれチェックします。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 5}

print(your_set.issubset(my_set)) # True
print(my_set.issuperset(your_set)) # True
```

- **`isdisjoint()` メソッド**: `isdisjoint()` メソッドは2つのセットが共通の要素を持たないかどうか（互いに素か）をチェックします。

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

print(my_set.isdisjoint(your_set)) # True
```

- **和集合演算子 (`|`)**: 和集合演算子 `|` は両方のセットのすべての要素を含む新しいセットを返します。

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

my_set | your_set # {1, 2, 3, 4, 5, 6}
```

- **積集合演算子 (`&`)**: 積集合演算子 `&` はセットが共通して持つ要素だけを含む新しいセットを返します。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set & your_set # {2, 3, 4}
```

- **差集合演算子 (`-`)**: 差集合演算子 `-` は最初のセットにあって他のセットにない要素だけを含む新しいセットを返します。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set - your_set # {1, 5}
```

- **対称差演算子 (`^`)**: 対称差演算子 `^` は最初か2番目のセットのどちらか一方にだけある要素を含む新しいセットを返します。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set ^ your_set # {1, 5, 6}
```

- **`in` 演算子**: `in` 演算子を使って要素がセットに含まれているかどうかをチェックできます。

```python
print(5 in my_set)
```

## Python標準ライブラリ

- **Python標準ライブラリ**: ライブラリは関数、クラス、データ構造などの再利用可能なコードを提供します。Pythonには多くの問題やタスクに対する標準化された解決策を実装した組み込みモジュールが豊富にあります。人気のある組み込みモジュールの例には `math`、`random`、`re`（「正規表現」の略）、`datetime` などがあります。

## import文

- **import文**: 組み込みモジュールで定義された要素にアクセスするには import 文を使います。import 文は通常ファイルの先頭に書きます。import 文は関数、クラス、定数、変数、その他モジュール内で定義された要素すべてに同じように機能します。

- **基本的なimport文**: `import` キーワードの後にモジュール名を書きます。

```python
import module_name
```

その後、そのモジュールの関数を呼び出すには、モジュール名の後にドット記法で関数名を書きます。

```python
module_name.function_name()
```

例えば、`math` モジュールをインポートして 36 の平方根を取得するには以下のように書きます。

```python
import math

math.sqrt(36)
```

- **別名でモジュールをインポートする**: モジュールを別名（エイリアス）でインポートしたい場合は、import 文の最後に `as` とエイリアスを書きます。長いモジュール名や名前の衝突を避けるためによく使われます。

```python
import module_name as module_alias
```

例えば、`math` モジュールをコード内で `m` として参照したい場合は以下のようにエイリアスを割り当てます。

```python
import math as m
```

その後、エイリアスを使ってモジュールの要素にアクセスできます。

```python
m.sqrt(36)
```

- **特定の要素だけをインポートする**: モジュールのすべてが必要ない場合は `from` を使って特定の要素だけをインポートできます。この場合、import 文は `from` で始まり、モジュール名、`import` キーワード、インポートしたい要素名が続きます。

```python
from module_name import name1, name2
```

その後、Pythonスクリプト内でこれらの名前をモジュールの接頭辞なしで使えます。例えば：

```python
from math import radians, sin, cos

angle_degrees = 40
angle_radians = radians(angle_degrees)

sine_value = sin(angle_radians)
cos_value = cos(angle_radians)

print(sine_value) # 0.6427876096865393
print(cos_value)  # 0.766044443118978
```
 
これは便利ですが、同じ名前の関数や変数がすでにある場合に名前の衝突が起こる可能性があります。どのタイプのimport文を使うか選ぶ際に注意してください。

これらの名前にエイリアスを割り当てたい場合は、`as` キーワードの後にエイリアスを書いて割り当てることもできます。

```python
from module_name import name1 as alias1, name2 as alias2
```

- **アスタリスクを使ったimport文 (`*`)**: アスタリスクはモジュール内のすべてをインポートしたいことをPythonに伝えますが、モジュール名を接頭辞として使う必要がなくなります。

```python
from module_name import *
```

例えば、これを使って `math` モジュールをインポートすると、そのモジュール内で定義された関数をモジュール名を指定せずに呼び出せます。

```python
from math import *
print(sqrt(36))  # 6.0
```

しかし、これは名前空間の衝突を引き起こしやすく、名前の由来がわかりにくくなるため、一般的には推奨されません。

## `if __name__ == '__main__'`

- **`__name__` 変数**: `__name__` はPythonの特別な組み込み変数です。Pythonファイルが直接実行されると、この変数の値は文字列 `"__main__"` に設定されます。しかし、そのPythonファイルが別のPythonスクリプトにモジュールとしてインポートされると、`__name__` 変数の値はそのモジュールの名前に設定されます。

このため、Pythonスクリプトでこの条件文をよく見かけます。これはPythonスクリプトがメインプログラムとして実行されている場合にのみ実行したいコードを含みます。

```python
if __name__ == '__main__': 
    # Code
```

## Pythonの一般的なエラー

- **SyntaxError**: コードがPythonの構文ルールに従っていない場合に発生するエラーです。例えば、コード `print("Hello there"` は閉じ括弧が欠けているため、メッセージ `SyntaxError: '(' was never closed` の構文エラーになります。
- **NameError**: `NameError`は、定義されていない変数または関数にアクセスしようとしたときに発生します。例えば、コード内に `print(username)` があり、`username` 変数が定義されていない場合、メッセージ `NameError: name 'username' is not defined` の名前エラーが発生します。
- **TypeError**: 互換性のないデータ型同士で操作を行った場合に発生します。例えば、文字列と数値を足そうとすると `TypeError: can only concatenate str (not "int") to str` エラーになります。
- **IndexError**: `IndexError`は、リスト、タプル、文字列などのシーケンス内で存在しないインデックスにアクセスしようとしたときに発生します。例えば、`Hello world` 文字列の最後の文字のインデックスは `10` です。`greet = "hello world"; print(greet[11])` のようにアクセスすると、メッセージ `IndexError: string index out of range` のエラーになります。
- **AttributeError**: その型のオブジェクトに存在しないメソッドやプロパティを使おうとすると発生します。例えば、文字列 `"hello".append("!")` に対して `.append()` を呼び出すと、メッセージ `AttributeError: 'str' object has no attribute 'append'` のエラーになります。

- **Python組み込みデバッガー (`pdb`) の使用**: Pythonは `pdb` モジュールを提供しており、標準ライブラリの一部なので常に使えます。`pdb` を使うと `set_trace()` 関数でトレースを設定し、対話的にコードをステップ実行して変数を調査できます。
- **IDEのデバッグツールの活用**: PycharmやVS Codeなどの統合開発環境やコードエディタは、ブレークポイント、ステップ実行、変数検査などのデバッグ機能を提供しています。

## 例外処理

- **`try...except`**: 例外が発生する可能性のあるコードブロックを実行するために使います。`try` ブロックはエラーが起こるかもしれない部分で、`except` ブロックは指定した例外が発生した場合に実行されます。例：

  ```py
  try:
    print(22 / 0)
  except ZeroDivisionError:
    print('You can\'t divide by zero!')
    # You can't divide by zero!
  ```

複数の `except` ブロックを連結して、より多くの例外タイプを処理できます。

  ```py
  try:
    number = int(input('Enter a number: '))
    print(22 / number)
  except ZeroDivisionError:
    print('You cannot divide by zero!')
    # You cannot divide by zero! prints when you enter 0
  except ValueError:
    print('Please enter a valid number!')
    # Please enter a valid number! prints when you don't enter an integer
  ```

- **`else` と `finally`**: これらのブロックは `try...except` を拡張します。例外が発生しなければ `else` ブロックが実行されます。`finally` ブロックはエラーの有無にかかわらず常に実行されます。

  ```py
  try:
    result = 100 / 4
  except ZeroDivisionError:
    print('You cannot divide by zero!') # This will not run
  else:
    print(f'Result is {result}') # Result is 25.0
  finally:
    print('Execution complete!') # Execution complete!
  ```

- **例外オブジェクト**: 例外そのものにアクセスしてデバッグやエラーメッセージの直接出力に使えます。例外オブジェクトにアクセスするには `as` キーワードを使います。例：

  ```py
  try:
      value = int('This will raise an error')
  except ValueError as e:
      print(f'Caught an error: {e}')
      # Caught an error: invalid literal for int() with base 10: 'This will raise an error'
  ```

- **`raise` 文**: 手動で例外を発生させるために使います。特定の条件が満たされたときに例外をスローできます。例：

  ```py
  def divide(a, b):
      if b == 0:
          raise ZeroDivisionError('You cannot divide by zero')
      return a / b
  ```
  
## 例外の通知
  
`raise` 文はカスタム例外を作成するときにも便利で、カスタムメッセージ付きの例外をスローできます。例：

  ```py
  class InvalidCredentialsError(Exception):
      def __init__(self, message="Invalid username or password"):
          self.message = message
          super().__init__(self.message)

  def login(username, password):
      stored_username = "admin"
      stored_password = "password123"
      
      if username != stored_username or password != stored_password:
          raise InvalidCredentialsError()
      
      return f"Welcome, {username}!"
  ```  

以下は `login` 関数と `InvalidCredentialsError` 例外を使った例です。

  ```py
  # failed login attempt
  try:
      message = login("user", "wrongpassword")
  except InvalidCredentialsError as e:
      print(f"Login failed: {e}")
  else:
      print(message)

  # successful login attempt
  try:
      message = login("admin", "password123")
  except InvalidCredentialsError as e:
      # This block is not executed because the login was successful
      print(f"Login failed: {e}")
  else:
      # The else block runs if the 'try' block completes without an exception
      print(message)
  ```

`raise` 文は `from` キーワードと組み合わせて例外を連鎖させ、異なるエラー間の関係を示すこともできます。

  ```py
  def parse_config(filename):
    try:
        with open(filename, 'r') as file:
            data = file.read()
            return int(data)
    except FileNotFoundError:
        raise ValueError('Configuration file is missing') from None
    except ValueError as e:
        raise ValueError('Invalid configuration format') from e

  config = parse_config('config.txt')
  ```

## Pythonのクラスとオブジェクト

- **クラス定義**: クラスはオブジェクトを作成するための設計図です。属性やメソッドを通じてオブジェクトの振る舞いを定義します。以下はPythonでの基本的なクラス定義の例です。

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f'{self.name.upper()} says woof woof!')
```

- **オブジェクトの作成**: オブジェクトはクラスのインスタンスです。必要な引数を渡してクラスを呼び出すことで作成されます。

```python
dog1 = Dog('Jack', 3)
dog2 = Dog('Thatcher', 5)

dog1.bark()  # JACK says woof woof!
dog2.bark()  # THATCHER says woof woof!
```

- **オブジェクトのメソッド呼び出し**: オブジェクトのメソッドを呼び出して、動作を実行したり情報を取得したりできます。

```python
object_name1.method_name()
object_name2.method_name()
```

- **クラスとオブジェクトの違い**: クラスは再利用可能なテンプレートであり、オブジェクトはそのクラスから作られた具体的なデータを持つインスタンスです。

## 属性

- **インスタンス属性**: `__init__()`内で`self`を使って定義され、各オブジェクトごとに固有です。
- **クラス属性**: クラス内に直接定義され、すべてのインスタンスで共有されます。

```python
class Dog:
    species = 'French Bulldog'  # Class attribute

    def __init__(self, name):
        self.name = name  # Instance attribute

print(Dog.species) # French Bulldog

jack = Dog('Jack')
print(jack.name)     # Jack
print(jack.species)  # French Bulldog
```

## メソッド

- **メソッド**: クラス内で定義され、オブジェクトの属性に対して操作を行う関数です。

```python
class Car:
    def __init__(self, color, model):

        self.model = model

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
print(my_car_1.describe())  # This car is a red Tesla Model S
```

- **メソッドのアクセス**: ドット記法を使ってオブジェクトのメソッドを呼び出します。以下は2つの異なる車のオブジェクトで`describe`メソッドを呼び出す例です。

```python
class Car:
    def __init__(self, color, model):
        self.color = color

print(my_car_1.describe()) # Calling method using the dot notation
```

## ダンダー（マジック）メソッド

- **定義**: ダブルアンダースコアで始まり終わる特別なメソッド（例：`__init__`、`__len__`、`__str__`、`__eq__`）。Pythonは組み込み操作のために内部でこれらを使用します。

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **ダンダーメソッドの間接的な呼び出し**: ダンダーメソッドを直接呼び出す必要はありません。特定の操作が起こるとPythonが自動的に呼び出します。これらの操作には以下が含まれます：

- **加算、減算、乗算、除算などの算術演算**。加算には`__add__()`、減算には`__sub__()`、乗算には`__mul__()`、除算には`__truediv__()`が呼ばれます。
        
- **連結、繰り返し、フォーマット、文字列変換などの文字列操作**。連結には`__add__()`、繰り返しには`__mul__()`、フォーマットには`__format__()`、文字列変換には`__str__()`と`__repr__()`が呼ばれます。
        
- **等価性、より小さい、より大きいなどの比較操作**。等価性には`__eq__()`、より小さいには`__lt__()`、より大きいには`__gt__()`が呼ばれます。
        
- **イテレーション操作（オブジェクトをイテラブルにし、次の要素に進む）**。イテレータを返すには`__iter__()`、次の要素を取得するには`__next__()`が呼ばれます。

## 実例：ショッピングカート

- **ダンダーメソッドを持つカートクラス**: 追加、削除、イテレーション、内容の確認を組み込みの振る舞いで可能にします。

```python
class Cart:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

    def remove(self, item):
        if item in self.items:
            self.items.remove(item)
        else:
            print(f'{item} is not in cart')

    def list_items(self):
        return self.items

    def __len__(self):
        return len(self.items)

    def __getitem__(self, index):
        return self.items[index]

    def __contains__(self, item):
        return item in self.items

    def __iter__(self):
        return iter(self.items)

cart = Cart()
cart.add('Laptop')
print(len(cart))        # 1
print('Laptop' in cart) # True
```

## オブジェクト指向プログラミングとは？

- **オブジェクト指向プログラミング**: 開発者がコード内のすべてを現実世界のオブジェクトのように扱うプログラミングスタイルです。一般にOOPと呼ばれます。コードを効果的に整理・管理するための4つの重要な原則は、**カプセル化**、**継承**、**ポリモーフィズム**、**抽象化**です。
- **クラス**: オブジェクトを作成するための設計図です。クラスから作られたすべてのオブジェクトは、データを定義する属性と振る舞いを決めるメソッドを持ちます。

## カプセル化とは？

- **カプセル化**: オブジェクトの属性とメソッドを一つの単位にまとめることです。オブジェクトの内部状態を、ドアのように振る舞う公開されたメソッドや属性の背後に隠せます。そのドアの向こうには、データの変更方法やアクセス権を制御するプライベートな属性やメソッドがあります。
- **カプセル化の例**: 財布の残高を管理したい場合、入金や出金は許可しますが、残高自体を誰かに勝手に変更されるのは避けたいでしょう。

```py
class Wallet:
   def __init__(self, balance):
       self.__balance = balance # Private attribute

   def deposit(self, amount):
       if amount > 0:
           self.__balance += amount # Add to the balance safely

   def withdraw(self, amount):
       if 0 < amount <= self.__balance:
           self.__balance -= amount # Remove from the balance safely

account = Wallet(500)
print(account.__balance) # AttributeError: 'Wallet' object has no attribute '__balance'
```

- **単一アンダースコアと二重アンダースコアの違い**: 属性やメソッドの名前の前に単一アンダースコアを付けるのは内部利用を示す慣例であり、外部からのアクセスを強制的に防ぐものではありません。二重アンダースコアを付けると、クラスの外部からのアクセスが実質的に防がれます。

## ゲッターとセッターとは？

- **ゲッターとセッター**: クラスの属性へのアクセスや変更を制御するメソッドです。ゲッターで値を取得し、セッターで値を設定します。
- **プロパティ**: ゲッターとセッターをつなぎ、データへのアクセスを可能にします。値の取得、設定、削除時に裏で追加の処理を実行します。
- **なぜメソッドではなくプロパティを使うのか**: プロパティは読みやすくコードがすっきりします。通常の属性のようにドット記法でアクセスでき、丸括弧は不要です。
- **ゲッターの作成**: ゲッターを作るには`@property`デコレータを使います。以下は円の半径を取得するゲッターの例です。

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self): # A getter to get the radius
        return self._radius
  
    @property
    def area(self):  # A getter to calculate area
        return 3.14 * (self._radius ** 2)

my_circle = Circle(3)

print(my_circle.radius) # 3
print(my_circle.area) # 28.26
```

`__init__()`内でself.radiusを使うことで、オブジェクト作成時にセッターが実行され、不正な半径値をすぐに検出できます。

- **セッターの作成**: 半径を設定するセッターを作るには、同じ名前の別のメソッドを定義し、その上に`@<property_name>.setter`を付けます。

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self):  # A getter to get the radius
        return self._radius

    @radius.setter
    def radius(self, value):  # A setter to set the radius
        if value <= 0:
            raise ValueError('Radius must be positive')
        self._radius = value

my_circle = Circle(3)
print('Initial radius:', my_circle.radius) # Initial radius: 3

my_circle.radius = 8
print('After modifying the radius:', my_circle.radius) # After modifying the radius: 8
```

- **Pythonのゲッターとセッターの扱い**: ゲッターとセッターを定義すると、通常の属性アクセス構文を使うたびにPythonが自動的に呼び出します。

```py
my_circle.radius # This will call the getter
my_circle.radius = 4 # This will call the setter
```

セッター内でプロパティ名に代入すると`RecursionError`が発生するため避けてください。

- **デリーター**: セッターとゲッターで値を設定・取得した後、`deleter`で削除方法を制御できます。デリーターはプロパティに対して`del`文を使ったときにカスタム処理を実行します。デリーターを作るには`@<property_name>.deleter`デコレータを使います。

```py
  # Deleter
    @radius.deleter
    def radius(self):
        print("Deleting radius...")
        del self._radius
```

## 継承とは？コードの再利用を促進する仕組み

- **継承**: 子クラスが親クラスの属性やメソッドを利用する仕組みです。継承はコードの再利用を促し、明確な階層構造を提供し、すべてを書き直さずに振る舞いをカスタマイズできます。継承を実装するには、子クラスが親クラスの名前を受け取ります。

```py
class Parent:
    # Parent attributes and methods

class Child(Parent):
    # Child inherits, extends, and/or overrides where necessary
```

- **単一継承と多重継承**: 子クラスが1つの親からプロパティやメソッドを継承する場合は**単一継承**と呼びます。複数の親から継承する場合は**多重継承**です。以下は多重継承の構文例です。

```py
class Parent:
    # Attributes and methods for Parent

class Child:
    # Attributes and methods for Child

class GrandChild(Parent, Child):
    # GrandChild inherits from both Parent and Child
    # GrandChild can combine or override behavior from each
```

- **`super()`関数**: クラスがメソッドを独自に実装または拡張している場合に、親クラスのメソッドを重複せずに**呼び出す**ための関数です。

## ポリモーフィズムとは？コードの再利用を促進する仕組み

- **ポリモーフィズム**: 異なるクラスが同じメソッド名を使い、呼び出されたときにそれぞれ異なる実装を行うOOPの原則です。構文は以下の通りです。

```py
class A:
   def action(self): ...

class B:
   def action(self): ...

class C:
   def action(self): ...

Class().method()  # Works for A, B, or C
```

- **継承に基づくポリモーフィズム**: 親クラスがメソッドを用意し、子クラスがそれを自分用に変えて使います。

## ネームマンリングとは？どのように機能する？

- **ネームマンリング**: Pythonが二重アンダースコアで始まる属性名を内部的にクラス名を接頭辞として付けて名前を変更する処理です。例えば`__attribute`が`_ClassName__attribute`に変わります。
- **ネームマンリングの目的**: 継承時に属性やメソッドの誤った上書きを防ぐことが主な目的です。以下のコードで理解が深まります。

```py
class Parent:
    def __init__(self):
        self.__data = 'Parent data'

class Child(Parent):
    def __init__(self):
        super().__init__()
        self.__data = 'Child data'

c = Child()
print(c.__dict__) # {'_Parent__data': 'Parent data', '_Child__data': 'Child data'}
```

## 抽象化とは？複雑なシステムを整理する助け

- **抽象化**: オブジェクトやシステムの複雑な実装の詳細を隠し、必要な機能だけを見せるプログラミングの概念です。Pythonや他の言語では、抽象化により複雑なシステムを簡素化し再利用性を高めます。
- **抽象化の例**: 車の例では、エンジンやブレーキの仕組みを知らなくても、ハンドルやペダル、シフトレバーだけを使えます。
- **Pythonでの抽象化の実装**: Pythonは`abc`モジュールを通じて抽象化を実装します。このモジュールは`ABC`クラス（抽象基底クラス）と`@abstractmethod`デコレータを提供します。抽象基底クラス（ABC）はサブクラスが実装すべき共通のメソッドやプロパティを定義し、インスタンス化できません。
- **抽象メソッドの定義方法**: `@abstractmethod`を使って定義し、サブクラスで必ずオーバーライドしなければなりません。基本的な抽象化の構文は以下の通りです。

```py
from abc import ABC, abstractmethod

# Define an abstract base class
class AbstractClass(ABC):
    @abstractmethod
    def abstract_method(self):
        pass

# Concrete subclass that implements the abstract method
class ConcreteClassOne(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassOne')

# Another concrete subclass
class ConcreteClassTwo(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassTwo')
```

## アルゴリズムとビッグO記法

- **アルゴリズム**: 問題を解決したりタスクを実行したりするための明確な手順の集合です。アルゴリズムは有限のステップで終了し、各ステップは正確かつ明確でなければなりません。

- **ビッグO記法**: 入力サイズが増加したときのアルゴリズムの最悪ケースの性能や成長率を表します。定数や低次の項は無視し、入力サイズに対するリソース使用量の増え方に注目します。

### よくある時間計算量

- **O(1) - 定数時間**: 入力サイズに関係なく一定の時間で処理が終わります。

```python
def check_even_or_odd(number):
    if number % 2 == 0:
        return 'Even'
    else:
        return 'Odd'
```

- **O(log n) - 対数時間**: 入力が増えても時間の増加がゆっくりです。問題サイズを繰り返し一定割合で減らすアルゴリズム（例：二分探索）に多いです。

- **O(n) - 線形時間**: 実行時間が入力サイズに比例して増えます。

```python
for grade in grades:
    print(grade)
```

- **O(n log n) - 線形対数時間**: マージソートやクイックソートなど効率的なソートアルゴリズムの一般的な時間計算量です。

- **O(n²) - 二次時間**: 実行時間が二次関数的に増えます。ネストしたループでよく見られます。

```python
for i in range(n):
    for j in range(n):
        print("Hello, World!")
```

### 空間計算量

- **O(1) - 定数空間**: 入力サイズに関係なく一定のメモリを使います。
- **O(n) - 線形空間**: メモリ使用量が入力サイズに比例して増えます。
- **O(n²) - 二次空間**: メモリ使用量が二次関数的に増えます。

## 問題解決の手法

- **問題の理解**: 問題文を何度も読み、入力、期待される出力、入力から出力への変換方法を特定します。

- **疑似コード**: 言語に依存しないアルゴリズムの高レベルな説明です。一般的な文章と`IF`、`ELSE`、`FOR`、`WHILE`などのプログラミング構造を混ぜて書きます。

```md
GET original_string
SET reversed_string = ""
FOR EACH character IN original_string:
  ADD character TO THE BEGINNING OF reversed_string
DISPLAY reversed_string
```

- **エッジケース**: アルゴリズムが扱う境界にある特定の有効な入力です。必ず考慮しテストしてください。

## 配列

- **静的配列**: 初期化時にサイズが固定されます。要素は隣接したメモリ領域に格納され、実行中にサイズ変更はできません。

- **動的配列**: 実行中に自動でサイズが増減します。必要に応じてより大きな配列にコピーしてリサイズを処理します。

### Pythonのリスト（動的配列）

```python
numbers = [3, 4, 5, 6]

# Access elements
numbers[0]  # 3

# Update elements
numbers[2] = 16

# Add elements
numbers.append(7)
numbers.insert(3, 15)  # Insert at specific index

# Remove elements
numbers.pop(2)  # Remove at specific index
numbers.pop()   # Remove last element
```

### 動的配列の時間計算量

- **アクセス**: O(1)
- **末尾への挿入**: 平均O(1)、リサイズ時はO(n)
- **途中への挿入**: O(n)
- **削除**: 途中はO(n)、末尾はO(1)

## スタック

- **スタック**: 後入れ先出し（LIFO）のデータ構造。要素は上から追加・削除されます。

- **プッシュ操作**: スタックの上に要素を追加します。時間計算量はO(1)。

- **ポップ操作**: スタックの上から要素を削除します。時間計算量はO(1)。

```python
# Using Python list as stack
stack = []

# Push operations
stack.append(1)
stack.append(2)
stack.append(3)

# Pop operations
top_element = stack.pop()  # Returns 3
```

## キュー

- **キュー**: 先入れ先出し（FIFO）のデータ構造。要素は後ろに追加され、前から削除されます。

- **エンキュー操作**: キューの後ろに要素を追加します。時間計算量はO(1)。

- **デキュー操作**: キューの前から要素を削除します。時間計算量はO(1)。

```python
from collections import deque

# Using deque for efficient queue operations
queue = deque()

# Enqueue operations
queue.append(1)
queue.append(2)
queue.append(3)

# Dequeue operations
first_element = queue.popleft()  # Returns 1
```

## 連結リスト

- **連結リスト**: 各ノードがデータと次のノードへの参照を持つ線形データ構造。ノードは鎖のようにつながっています。

### 単方向連結リスト

- **構造**: 各ノードはデータと次のノードへの参照を1つ持ちます。
- **走査**: 頭から尾へ一方向にしか進めません。
- **頭ノード**: リストの最初のノードで、通常は唯一直接アクセス可能なノードです。
- **尾ノード**: リストの最後のノードで、`None`を指します。

### 操作と時間計算量

- **先頭への挿入**: O(1)
- **末尾への挿入**: O(n) - 末尾まで走査が必要
- **途中への挿入**: O(n) - 挿入位置まで走査が必要
- **先頭からの削除**: O(1)
- **末尾からの削除**: O(n) - 前のノードを探すため走査が必要
- **途中からの削除**: O(n) - 削除ノードを探すため走査が必要

### 双方向連結リスト

- **構造**: 各ノードはデータと2つの参照（次ノードと前ノード）を持ちます。
- **走査**: 双方向に移動可能です。
- **メモリ**: 追加の参照があるため単方向連結リストより多くのメモリを使います。

## ハッシュマップとセット

### マップとハッシュマップ

- **マップ（抽象データ型）**: キーと値のペアの集合を管理します。キーはすべて一意ですが、値は重複可能です。

- **ハッシュマップ**: ハッシュ技術を使ったマップADTの具体的な実装です。キーのハッシュ値を生成し、基盤となる配列の格納場所を決定します。

### Pythonの辞書（ハッシュマップ）

```python
# Creating dictionaries
my_dictionary = {
    "A": 1,
    "B": 2, 
    "C": 3
}

# Alternative creation
my_dictionary = dict(A=1, B=2, C=3)

# Access and modify
value = my_dictionary["A"]  # 1
my_dictionary["A"] = 4      # Update value
del my_dictionary["A"]      # Remove key-value pair

# Check membership
"C" in my_dictionary

# Get keys, values, items
my_dictionary.keys()
my_dictionary.values()
my_dictionary.items()
```

### ハッシュマップの時間計算量

- **平均ケース**: 挿入、取得、削除はO(1)
- **最悪ケース**: 多くのハッシュ衝突が起きるとO(n)

### セット

- **セット**: 重複のない要素の無順序集合。重複は許されず、特定の順序は保持されません。

- **不変要素のみ**: セットには数値、文字列、タプルなどの不変データ型のみが含まれます。ハッシュ値は一定でなければならないためです。

```python
# Creating sets
numbers = {1, 2, 3, 4}
empty_set = set()  # Must use set(), not {}

# Add and remove elements
numbers.add(5)
numbers.remove(4)      # Raises KeyError if not found
numbers.discard(4)     # No error if not found

# Set operations
set_a = {1, 2, 3, 4}
set_b = {2, 3, 4, 5, 6}

# Union, intersection, difference, symmetric difference
set_a.union(set_b)                    # or set_a | set_b
set_a.intersection(set_b)             # or set_a & set_b
set_a.difference(set_b)               # or set_a - set_b
set_a.symmetric_difference(set_b)     # or set_a ^ set_b

# Subset and superset checks
set_a.issubset(set_b)
set_a.issuperset(set_b)
set_a.isdisjoint(set_b)

# Membership testing
5 in numbers
```

### セットの時間計算量

- **平均ケース**: 追加、削除、メンバーシップテストはO(1)
- **最悪ケース**: ハッシュ衝突によりO(n)

## ハッシュ衝突

- **ハッシュ衝突**: 異なる2つのキーが同じハッシュ値を生成することです。

- **衝突解決戦略**:
  - **チェーン**: 各配列インデックスが同じハッシュ値の要素を格納する連結リストを指します。
  - **オープンアドレッシング**: 事前定義されたシーケンスを使って次の空きインデックスを探します。

## それぞれのデータ構造の使いどころ

- **リスト**: 順序付きでインデックスアクセスが必要で、サイズが事前にわからない場合
- **スタック**: LIFO操作（元に戻す機能、式の評価、バックトラッキング）
- **キュー**: FIFO操作（タスクスケジューリング、幅優先探索）
- **連結リスト**: 頻繁な先頭での挿入・削除、サイズ不明、ランダムアクセス不要の場合
- **ハッシュマップ**: 高速なキーと値の検索、出現回数のカウント、キャッシュ
- **セット**: 一意性のチェック、集合演算、重複の除去

## 検索アルゴリズム

検索アルゴリズムは、特定のリスト内でターゲットを検索するためのものです。

コンピューターサイエンスでは、知っておくべき検索アルゴリズムが2つあります。これらは**線形探索**と**二分探索**のアルゴリズムです。両者の違いと、それぞれを使うべき状況を理解することが重要です。

### 線形探索

- 線形探索はリストの最初から順に各項目を反復し、ターゲットの項目が見つかるまでチェックします。
- ターゲットの項目が見つかれば、その項目がリスト内にあるインデックスを返します。
- ターゲットが見つからなければ、`-1`を返します。これは多くのプログラミング言語で**無効なインデックス**を意味します。
- 線形探索はターゲットが見つかるまで全ての項目をチェックするため、大きなリストには効率的ではありません。
- 線形探索の時間計算量は`O(n)`です。これはリストのサイズに比例して検索時間が増えるためです。
- 線形探索の空間計算量は`O(1)`です。検索に追加の空間を必要としないためです。

### 二分探索

- 二分探索はリストを半分に分割し、ターゲットがリストの中央にあるか調べます。
- 二分探索が機能する条件は、リストの要素が昇順に並んでいることです。
- 大きなリストの探索に効率的で、リストを半分に分割し、ターゲットがない半分を無視します。
- ターゲットが中央にあれば、そのインデックスを返します。
- 見つからなければ、ターゲットが左半分か右半分かを調べます。
- 残りの部分をさらに半分に分割し、ターゲットが見つかるまで続けます。
- 最終的に見つからなければ`-1`を返します。
- 二分探索の時間計算量は`O(log n)`で、リストのサイズに対して対数的に増えます。
- 二分探索の空間計算量は`O(1)`で、追加のメモリは不要です。

### 線形探索と二分探索の違い

- 二分探索は線形探索よりも大きなリストに適しています。
- 線形探索の時間計算量は`O(n)`です。リストのサイズに比例して検索時間が増えます。
- 二分探索の時間計算量は`O(log n)`です。リストのサイズに対して対数的に検索時間が増えます。

## ソートアルゴリズムと分割統治法

コンピューターサイエンスでは、分割統治法は問題をより小さな部分問題に分割して解きやすくする手法です。分割統治法では再帰がよく使われ、多くの効率的なソートアルゴリズム（例えばマージソート）を実装する強力な戦略です。

### マージソート

- マージソートは分割統治法に従うソートアルゴリズムです。
- リストを再帰的により小さなサブリストに分割し、各サブリストが1つの要素だけになるまで続けます。
- その後、サブリストを繰り返しマージしてソートされた順序に戻します。
- マージソートの時間計算量は`O(n log n)`です。リストが継続的に半分に分割され`(log n)`、その後マージされるため`(O(n))`です。
- マージソートの空間計算量は`O(n)`です。インプレースソートアルゴリズムではないためです。

## グラフの概要

グラフはノード（頂点）とエッジ（接続）からなる集合です。各ノードは複数の他のノードと接続し、ネットワークを形成します。グラフの種類には以下があります：

- 有向グラフ：エッジに方向があり（あるノードから別のノードへ）、直線と矢印で表されます。
- 無向グラフ：エッジに方向がなく、単純な線で表されます。
- 頂点：各ノードにラベルや識別子が関連付けられています。
- 循環グラフ：サイクル（開始ノードと終了ノードが同じ経路）を含みます。
- 非循環グラフ（DAG）：サイクルを含みません。
- エッジラベル付き：各エッジにラベルがあり、通常はエッジの隣に描かれます。
- 重み付き：エッジに重み（値）があり、算術演算に使えます。
- 非連結グラフ：2つ以上のノードがエッジで接続されていません。

グラフは地図、ネットワーク、推薦システム、依存関係解決などで使われます。

## グラフの走査

グラフのすべてのノードを訪問することです。主なアルゴリズムは：

- **幅優先探索（BFS）**
  - キューを使います。
  - レベルごとに探索します。
  - 重みなしグラフで最短経路を見つけます。

- **深さ優先探索（DFS）**
  - スタック（または再帰）を使います。
  - 1つの枝を完全に探索してから戻ります。
  - サイクル検出や経路探索に役立ちます。

## グラフの表現方法

グラフは主に2つの方法で表現されます：

- **隣接リスト**
  - 各ノードが隣接ノードのリストを持ちます。
  - 疎なグラフに対して空間効率が良いです。
  - 隣接ノードの反復が簡単です。

- **隣接行列**
  - 行と列がノードを表す2次元配列です。
  - 大きなグラフでは空間を多く使います。
  - 2つのノード間にエッジがあるかの判定が高速です。

## 木

木は非循環かつ連結な特別なグラフです。主な特徴は：

- ループやサイクルがありません（開始ノードと終了ノードが同じ経路がない）。
- すべてのノードが互いに到達可能でなければなりません。

### よくある木の種類

代表的な木の種類は：

- 二分木
  - 各ノードは最大2つの子ノード（左と右）を持ちます。

- 二分探索木（BST）
  - 左の子は親より小さく、右の子は親より大きい二分木です。


## トライ木

接頭辞木とも呼ばれ、文字列の集合を格納するために使われます。各ノードは1文字を表します。

共通の接頭辞は1回だけ保存されるため、オートコンプリートやスペルチェックに効率的です。

検索や挿入の時間計算量はO(L)で、Lは文字列の長さです。

## 優先キュー

優先キューは各要素に優先度がある抽象データ型です。

キューやスタックは挿入順序のみを考慮しますが、優先キューは優先度の高い要素を先に処理します。

標準のキューはFIFO、スタックはLIFOですが、優先キューでは優先度が高い順に処理されます。

## ヒープ

ヒープは特定のヒープ特性を持つ木構造のデータ構造です。

ヒープ特性は親ノードと子ノードの関係を決めます。ヒープには2種類あります：

- 最大ヒープ
  - 親ノードの値は子ノードの値以上です。
  - 最大の要素が根にあります。

- 最小ヒープ
  - 親ノードの値は子ノードの値以下です。
  - 最小の要素が根にあります。

### Pythonの`heapq`モジュールの例

```py
import heapq

# Create empty heap
my_heap = []

# Insert elements
heapq.heappush(my_heap, 9)
heapq.heappush(my_heap, 3)
heapq.heappush(my_heap, 5)
print(my_heap) # [3, 9, 5]

# Remove smallest element
print(heapq.heappop(my_heap))  # 3
print(my_heap) # [5, 9]

# Push + Pop in one step
print(heapq.heappushpop(my_heap, 7)) # 5
print(my_heap) # [7, 9]

# Transform list into heap
nums = [5, 7, 3, 1]
heapq.heapify(nums)
```

### 優先度の利用例

```py
my_heap = []
heapq.heappush(my_heap, (3, "A"))
heapq.heappush(my_heap, (2, "B"))
heapq.heappush(my_heap, (1, "C"))

# Removes lowest number = highest priority
print(heapq.heappop(my_heap))  # (1, "C")
```

## 動的計画法の紹介

- **定義**: 動的計画法は複雑な問題をより単純な部分問題に分割し、結果を保存して重複計算を避けるアルゴリズム手法です。
- **重複部分問題**: 大きな問題を解く際に同じ小さな問題が何度も現れます。これらを繰り返し計算する代わりに結果を保存します。
- **最適部分構造**: 問題の最適解は部分問題の最適解から構成されます。つまり、小さな部分の最適解を組み合わせて最適解を作れます。

## 動的計画法の解法

- **メモ化（トップダウンアプローチ）**: 高コストな関数呼び出しの結果を保存し、同じ入力が再度現れたときにキャッシュされた結果を返します。

```py
def climb_stairs_memo(n, memo={}):
    """Dynamic programming with memoization"""
    # Check if we've already calculated this value
    if n in memo:
        return memo[n]  # Return cached result - O(1) lookup!
    
    # Base cases
    if n <= 2:
        return n
    
    # Calculate once and store in memo for future use
    memo[n] = climb_stairs_memo(n-1, memo) + climb_stairs_memo(n-2, memo)
    return memo[n]
```

- **タビュレーション（ボトムアップアプローチ）**: 下から順に解を構築し、部分問題の解をテーブルに埋めていきます。

```py
def climb_stairs_tabulation(n):
    """Dynamic programming with tabulation"""
    if n <= 2:
        return n
    
    # Create array to store results for all steps from 0 to n
    dp = [0] * (n + 1)
    dp[1] = 1  # 1 way to reach step 1
    dp[2] = 2  # 2 ways to reach step 2
    
    # Build up the solution iteratively
    for i in range(3, n + 1):
        # Ways to reach step i = ways to reach (i-1) + ways to reach (i-2)
        dp[i] = dp[i-1] + dp[i-2]
    
    return dp[n]
```

## 動的計画法を使った実世界のアプリケーション

- **ルート最適化**：GPSシステムは動的計画法のアルゴリズムを使って、場所間の最短経路を見つけます。
- **テキスト処理**：スペルチェッカーやオートコンプリート機能は、単語間の編集距離を計算するために動的計画法をよく利用します。
- **金融モデリング**：投資戦略やポートフォリオの最適化には動的計画法の手法が頻繁に使われます。
- **資源配分**：ナップサック問題やその派生問題は、スケジューリング、予算管理、資源管理に現れます。

## 動的計画法を使うべき場合

次のような場合に動的計画法の使用を検討してください：

- 問題が重複する部分問題に分解できる。
- 問題が最適部分構造を持つ。
- 単純な再帰的解法では計算が繰り返される。
- 時間計算量を優先し、空間計算量を犠牲にしてもよい。
