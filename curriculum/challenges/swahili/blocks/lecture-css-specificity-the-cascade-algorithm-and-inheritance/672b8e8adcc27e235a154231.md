# --interactive--

Vichaguzi vya aina, vinavyojulikana pia kama vichaguzi vya vipengele, hualenga vipengele kulingana na jina la lebo yao.

Vichaguzi hivi ni msingi katika CSS na hukuwezesha kutumia mitindo kwa matukio yote ya kipengele maalum cha HTML.

Vichaguzi vya aina ni rahisi kutumia na huandikwa kwa urahisi kama jina la lebo ya kipengele unachotaka kupatia mtindo.

Hapa kuna mfano wa kichaguzi cha aina kinacholenga vipengele vyote vya aya kwenye ukurasa:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  color: blue;
}
```

:::

Katika mfano huu, vipengele vyote vya `p` vitakuwa na rangi ya maandishi iliyowekwa kuwa `blue`.

Vichaguzi vya aina vina umuhimu mdogo ikilinganishwa na vichaguzi vingine. Thamani ya umuhimu kwa kichaguzi cha aina ni `(0, 0, 0, 1)`.

Hii ina maana kwamba vichaguzi vya aina vitaghairiwa na vichaguzi vya darasa la vitu, vichaguzi vya ID, na mitindo ndani ya mstari, lakini bado vinaweza kutumika isipokuwa sheria hizo zenye umuhimu mkubwa zipo.

Tuchunguze mfano ambapo vichaguzi vya darasa la vitu vitaghairi mitindo ya kichaguzi cha aina.

Hapa kuna mfano wenye vipengele viwili vya aya:

:::interactive_editor

```html
<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

:::

Vipengele vyote vya aya vina darasa la vitu linaloitwa `para`.

Ndani ya faili ya CSS, kichaguzi cha aina hualenga aya, na kichaguzi cha darasa la vitu hualenga vipengele vyenye darasa la `para`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

```css
p {
  color: blue;
}

.para {
  color: red;
}
```

:::

Aya zote kwenye ukurasa zenye darasa la `para` zitakuwa na rangi ya maandishi iliyowekwa kuwa `red` badala ya `blue` kwa sababu vichaguzi vya darasa la vitu vina umuhimu mkubwa zaidi kuliko vichaguzi vya aina.
