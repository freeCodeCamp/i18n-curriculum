# --description--

Puoi anche selezionare elementi tramite i loro attributi id.

Per prima cosa seleziona il tuo elemento `button` con l'id `target3` usando il selettore `$("#target3")`.

Nota che, proprio come nelle dichiarazioni CSS, devi digitare un `#` prima del nome dell'id.

Poi usa la funzione `.addClass()` di jQuery per aggiungere le classi `animated` e `fadeOut`.

Ecco come faresti scomparire gradualmente l'elemento `button` con l'id `target6`:

```js
$("#target6").addClass("animated fadeOut");
```

# --hints--

Dovresti selezionare l'elemento `button` con il `id` di `target3` e usare la funzione `addClass()` di jQuery per assegnargli la classe `animated`.

```js
assert($('#target3').hasClass('animated'));
```

Dovresti selezionare l'elemento con l'id `target3` e usare la funzione `addClass()` di jQuery per assegnargli la classe `fadeOut`.

```js
assert(
  ($('#target3').hasClass('fadeOut') || $('#target3').hasClass('fadeout')) &&
    code.match(/\$\(\s*.#target3.\s*\)/g)
);
```

Dovresti usare solo jQuery per aggiungere queste classi all'elemento.

```js
assert(!code.match(/class.*animated/g));
```
