# --description--

في هذا الدرس، ستتعلم كيفية العمل مع الأنواع الأولية الأساسية وكيف تعمل تعليقات النوع للمتغيرات.

بالنسبة لأنواع البيانات الأولية `string`، و`null`، و`undefined`، و`number`، و`boolean`، و`bigint`، يقدم TypeScript الكلمات المفتاحية الخاصة بكل نوع.

في هذا الدرس، سنركز على الأنواع الأولية المستخدمة بشكل شائع ولن نتعمق كثيرًا في `bigint`. نوع البيانات `bigint` هو نوع رقمي في JavaScript لتمثيل الأعداد الأكبر من النوع الصحيح القياسي.

لنلقِ نظرة على كل من هذه الأنواع الأولية بدءًا بـ `string`. إليك كيف يمكنك تعيين نوع سلسلة نصية في TypeScript:

```ts
let firstName: string = "Angie";
```

على عكس JavaScript العادي، المتغير `firstName` يتوقع نوع `string`. لذا إذا حاولت إعادة تعيينه بقيمة عددية، فستحصل على خطأ مثل هذا:

```ts
let firstName: string = "Angie";
firstName = 9 // Type 'number' is not assignable to type 'string'.
```

النوع التالي الذي سننظر إليه هو نوع `number`. إليك مثالًا:

```ts
let age: number = 16;
```

يمكنك تعيين أعداد صحيحة وعشرية إلى نوع `number`. لكن إذا حاولت إعادة تعيين قيمة من نوع `bigint`، فستحصل على خطأ مثل هذا:

```ts
let age: number = 16;

// Type 'bigint' is not assignable to type 'number'.
age = 123n;
```

النوع التالي الذي سننظر إليه هو نوع `boolean`. إليك بعض الأمثلة:

```ts
let isLoggedIn: boolean = true;
let isAdmin: boolean = false;
```

آخر نوعين هما نوعا `null` و`undefined`.

كما تتذكر من دروس JavaScript السابقة، `null` تمثل غيابًا مقصودًا للقيمة. و`undefined` تعني أن المتغير قد تم إعلانه لكنه لم يُعيَّن له قيمة.

سترى أمثلة على كيفية العمل مع هذه الأنواع لاحقًا عندما تتعلم عن أنواع الدوال وأنواع الاتحاد.

الآن بعد أن رأيت كيف تعمل تعليقات النوع مع المتغيرات، يطرح السؤال هل تحتاج دائمًا إلى كتابة نوع المتغيرات صراحةً.

إذا كان لديك تعليق نوع بسيط مثل هذا، فقد لا تحتاج إلى استخدام نوع صريح.

```ts
let firstName: string = "Angie";
```

السبب هو أن TypeScript سيحاول استنتاج الأنواع تلقائيًا بناءً على القيمة. لذا إذا كتبت هذا، سيفهم TypeScript أنه سلسلة نصية.

```js
let firstName = "Angie";
```

يجب أن تكتب نوع متغيراتك صراحةً عندما لا يستطيع TypeScript استنتاج النوع بوضوح أو عندما تريد فرض نوع محدد في حالات مثل معلمات الدوال أو الكائنات المعقدة.
