# --instructions--

Pythonを使ってポートスキャナーを作成してください。

`port_scanner.py` ファイルに、`get_open_ports` 引数と `target` 引数を受け取る `port_range` という名前の関数を作成してください。`target` はURLまたはIPアドレスで構いません。`port_range` は調べるポート範囲の最初と最後の番号を示す2つの数字のリストです。

関数の呼び出し例は以下の通りです：

```py
get_open_ports("209.216.230.240", [440, 445])
get_open_ports("www.stackoverflow.com", [79, 82])
```

関数は指定された範囲内の開いているポートのリストを返す必要があります。

`get_open_ports` 関数はオプションの3番目の引数として `True` を受け取ることもできます。これがtrueに設定されている場合、関数はポートのリストではなく説明的な文字列を返す必要があります。

verboseモードで返す文字列の形式は以下の通りです（`{}` 内のテキストは表示される情報を示しています）：

```bash
Open ports for {URL} ({IP address})
PORT     SERVICE
{port}   {service name}
{port}   {service name}
```

各ポートに対応する正しいサービス名を取得するには、`common_ports.py` の辞書を使うことができます。

例えば、関数が次のように呼び出された場合：

```py
port_scanner.get_open_ports("scanme.nmap.org", [20, 80], True)
```

次のように返す必要があります：

```bash
Open ports for scanme.nmap.org (45.33.32.156)
PORT     SERVICE
22       ssh
80       http
```

適切なスペースと改行文字を含めるようにしてください。

`get_open_ports` 関数に渡されたURLが無効な場合、関数は "Error: Invalid hostname" という文字列を返す必要があります。

`get_open_ports` 関数に渡されたIPアドレスが無効な場合、関数は "Error:  Invalid IP address" という文字列を返す必要があります。
