# --description--

Outro tipo de asserção lookaround é o lookahead.Lookahead positivo e negativo são indicados por `(?=...)` e `(?!...)`, respectivamente.Eles são usados para corresponder a um padrão se seguido por uma certa sequência de caracteres, que não é consumida:

```py
spam = 'black back bat'
re.sub('a(?=t)', 'o', spam) == 'black back bot' # True
re.sub('a(?!t)', 'o', spam) == 'block bock bat' # True
```

No exemplo acima, o padrão `a(?=t)` contém um lookahead positivo, que é usado para corresponder ao caractere `a` somente quando seguido por um `t`.Na última linha do exemplo, o padrão `a(?!t)` contém um lookahead negativo, que é usado para corresponder ao caractere `a` somente se **não** for seguido por um `t`.Novamente, em ambos os casos, o caractere contido no lookahead não é consumido.

Adicione um lookahead positivo ao seu padrão regex para que o caractere `1` seja substituído somente se for seguido pelo caractere `x`.
