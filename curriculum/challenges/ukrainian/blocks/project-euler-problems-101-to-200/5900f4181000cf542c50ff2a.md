# --description--

Для додатного цілого числа $n$ позначимо $f(n)$ суму квадратів цифр (у десятковій системі) числа $n$, наприклад

$$\begin{align}
  & f(3) = 3^2 = 9 \\\\
  & f(25) = 2^2 + 5^2 = 4 + 25 = 29 \\\\
  & f(442) = 4^2 + 4^2 + 2^2 = 16 + 16 + 4 = 36 \\\\
\end{align}$$

Знайдіть останні дев’ять цифр суми всіх $n$, $0 &lt; n &lt; {10}^{20}$, для яких $f(n)$ є повним квадратом.
