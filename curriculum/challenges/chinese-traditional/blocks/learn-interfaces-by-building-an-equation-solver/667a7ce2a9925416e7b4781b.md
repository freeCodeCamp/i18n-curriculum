# --description--

結構樣式匹配讓你能夠驗證主體是否具有特定的結構。此外，它會將樣式中的名稱綁定到主體的元素。 例如：

```py
match my_list:
    case [a]:
        print(a)
    case [a, b]:
        print(a, b)
```

將你的 `match`/`case` 構件修改為匹配 `results`，而非 `len(results)`。然後，修改每個 `case` 以使用具有適當元素數量的列表。當列表只包含一個元素時，使用 `x`，當列表包含兩個元素時，使用 `x1` 和 `x2`。

最後，修改 f-strings 以使用每個 `case` 中使用的變數名稱。
