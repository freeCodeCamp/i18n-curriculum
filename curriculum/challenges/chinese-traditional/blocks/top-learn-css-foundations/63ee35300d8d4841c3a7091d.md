# --description--

另一種使用選擇器的方法是將它們串鏈成一個列表，且不加任何分隔。假設你有以下的 HTML：

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection preview">This is where a preview for a post might go.</p>
</div>
```

你有兩個帶有 `subsection` 類別的元素，它們有某種獨特的樣式，但如果你只想對同時具有 `header` 作為第二個類別的元素套用單獨的規則怎麼辦？那麼，你可以在你的 CSS 中將兩個 `class` 選擇器串鏈起來，如下所示：

```css
.subsection.header {
  color: red;
}
```

`.subsection.header` 的作用是選取同時具有 `subsection` 和 `header` 類別的任何元素。請注意 `.subsection` 和 `.header` `class` 選擇器之間沒有任何空格。這種語法基本上適用於串鏈任何組合的選擇器，但不適用於串鏈多於一個的型別選擇器。

這也可以用來串鏈一個類別和一個 ID，如下所示：

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection" id="preview">This is where a preview for a post might go.</p>
</div>
```

你可以將上述兩個元素與以下內容結合：

```css
.subsection.header {
  color: red;
}

.subsection#preview {
  color: blue;
}
```

一般來說，你無法串鏈多於一個型別選擇器，因為一個元素不可能同時是兩種不同的型別。<dfn>例如</dfn>，串鏈兩個型別選擇器如 `div` 和 `p` 會產生選擇器 `divp`，這無法運作，因為該選擇器會嘗試尋找字面上的 `<divp>` 元素，而該元素並不存在。
