## --text--

如果使用者提供字串 `"random"`，以下程式碼的結果會是什麼？

```python
try:
    number = int(input("Enter a number: "))
    print(number)
except:
    print("Invalid Input")
```

## --answers--

不會有任何輸出到主控台。

---

字串 `"random"` 將會輸出到主控台。

---

字串 `"Invalid Input"` 將輸出到主控台。

---

程式會當機。
