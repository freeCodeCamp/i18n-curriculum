# --description--

Постачальники «A» та «B» надали такі кількості продуктів для ринку розкішних наборів:

|       Продукт      | 'A'  | 'B'  |
|--------------------|------|------|
|    Ікра білуги     | 5248 | 640  |
|    Різдвяний торт  | 1312 | 1888 |
|    Шматок шинковий | 2624 | 3776 |
|    Винтажний портвейн | 5760 | 3776 |
|    Шампанські трюфелі | 3936 | 5664 |

Хоча постачальники дуже стараються доставити свої товари в ідеальному стані, неминуче трапляється псування — тобто продукти псуються.

Постачальники порівнюють свої результати, використовуючи два типи статистики:

- П’ять показників рівня псування для кожного продукту окремо для кожного постачальника — це кількість зіпсованих продуктів, поділена на кількість поставлених продуктів для кожного з п’яти продуктів.
- Загальний рівень псування для кожного постачальника — це загальна кількість зіпсованих продуктів, поділена на загальну кількість поставлених продуктів цим постачальником.

На їхнє здивування, постачальники виявили, що кожен із п’яти показників рівня псування для продуктів був гіршим (вищим) у «B», ніж у «A», у однаковому співвідношенні (відношенні рівнів псування) $m > 1$; і, парадоксально, загальний рівень псування був гіршим у «A», ніж у «B», також у співвідношенні $m$.

Існує тридцять п’ять значень $m > 1$, для яких цей дивовижний результат міг статися, найменше з яких — $\frac{1476}{1475}$.

Яке найбільше можливе значення $m$? Вкажіть відповідь у вигляді рядка з дробом, скороченим до найпростішої форми, у форматі `u/v`.
