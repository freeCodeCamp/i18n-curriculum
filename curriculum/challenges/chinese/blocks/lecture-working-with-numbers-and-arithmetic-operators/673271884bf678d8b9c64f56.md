# --interactive--

JavaScript 提供了对数字进行基本算术运算的工具，如加法、减法、乘法和除法。 JavaScript 还包含用于更复杂算术运算的运算符，如取余和指数运算符。

所有这些工具都被称为算术运算符。 让我们详细了解一下这些运算符，以及如何使用和组合它们。 

加法运算符是一个加号（`+`）。 通过加法运算符，你可以求出两个或两个以上数字的和。 在加法运算中，数字的顺序并不重要：

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 + num2;
const result2 = num2 + num1;
const result3 = num2 + num1 + num3;

console.log(result1); // 15
console.log(result2); // 15
console.log(result3); // 30
```

:::

减法运算符是一个减号（`-`）。 它用于计算两个数字之间的差值。 当你想用一个数减去另一个数，通常是用一个大数减去一个小数时，请使用减号：

:::interactive_editor

```js
const difference = 10 - 5;
console.log(difference); // 5
```

:::

如果较小的数字排在前面，则结果为负数：

:::interactive_editor

```js
const difference = 5 - 10;
console.log(difference); // -5
```

:::

你也可以将数字赋值给变量，然后用变量名进行减法运算：

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const result = num1 - num2;

console.log(result); // 5
```

:::

在 JavaScript 中，乘法运算符用星号（`*`）表示，用于求两个或两个以上数字的乘积。 数字相乘的顺序并不重要：

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 * num2;
const result2 = num2 * num1;
const result3 = num2 * num1 * num3;

console.log(result1); // 50
console.log(result2); // 50
console.log(result3); // 750
```

:::

在 JavaScript 中，除法运算符是斜线（`/`），这与传统数学中使用的除法符号（`÷`）不同。 你可以使用除法运算符执行除法运算。 在这种情况下，被数字的顺序很重要：

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 / num2;
const result2 = num2 / num1;
const result3 = num2 / num1 / num3;

console.log(result1); // 2
console.log(result2); // 0.5
console.log(result3); // 0.03333333333333333
```

:::

值得注意的是，如果你尝试除以 0，JavaScript 将返回 `Infinity` ：

:::interactive_editor

```js
const result = 10 / 0; 

console.log(result); // Infinity
```

:::

确保避免这些类型的计算，以免在代码中出现意外结果。 

取余运算符用百分号（`%`）表示，返回除法运算的余数。 数学中的余数是除法运算后的剩余值：

:::interactive_editor

```js
const num1 = 10;
const num2 = 3;
const remainder = num1 % num2;

console.log(remainder); // 1
```

:::

指数运算符由双星号（`**`）表示，用于将一个数字提升为另一个数字的幂：

:::interactive_editor

```js
const num1 = 2;
const num2 = 3;

const exponent = num1 ** num2;
console.log(exponent); // 8
```

:::

在一个运算或表达式中可以混合使用运算符：

:::interactive_editor

```js
const result = 10 + 5 * 2 - 8 / 4;
console.log(result); // 18
```

:::

当你在一个表达式中混合使用不同的运算符时，JavaScript 引擎会遵循一个称为运算符优先级的系统来决定运算顺序。 运算符优先级决定表达式中执行运算的顺序。  You will learn more about operator precedence in future lessons.
