# --description--

Um `QA (Quality Assurance) tester` é alguém que verifica o software para garantir que ele funcione bem e não tenha problemas.

`Quality` significa quão bom ou ruim algo é e `assurance` significa garantir algo. Um `QA tester` precisa testar o software para assegurar que sua qualidade é boa e que ele funciona como deveria.
