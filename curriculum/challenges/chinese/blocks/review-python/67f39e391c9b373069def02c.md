## 什么是 Python？

- **简介**：Python 是一种通用编程语言，以其简洁和易用著称。Python 被广泛应用于数据科学和机器学习、网页开发、脚本和自动化、嵌入式系统和物联网等多个领域。
- **常见应用场景**：Python 用于数据科学、机器学习、网页开发、网络安全、自动化以及像 Raspberry Pi 和兼容 MicroPython 的微型计算机。

## 本地环境中的 Python

- **安装**：在 Windows、Mac 和 Linux 上安装 Python 的最佳方式是从官方 Python 网站下载安装程序（`https://www.python.org/`）。
- **终端**：一种基于文本的界面，允许你通过输入命令与计算机交互。每个操作系统都自带默认的终端应用程序。在 macOS 上，你可以使用 Terminal 应用。在 Windows 上，可以使用命令提示符或 PowerShell。在 Linux 上，每个桌面环境都有自己的默认终端应用，如 GNOME Terminal 或 Konsole。
- **IDE**：IDE 代表集成开发环境，包含代码编辑器、测试工具、终端等功能。
- **Python 常用代码编辑器和 IDE**：VS Code、PyCharm 和 Spyder
- **本地运行代码**：运行 Python 代码有几种方式。第一种是在 VS Code 中点击右上角类似播放按钮的运行按钮，这会打开终端并运行你的 Python 脚本。

另一种方式是打开终端，手动运行程序。下面是运行名为 `main.py` 的程序的示例：

```bash
python main.py
```

你必须从保存 `main.py` 的文件夹中运行此命令。例如，如果 `main.py` 在名为 `python-projects` 的文件夹内，先在终端中使用以下命令：

```bash
cd python-projects
```

然后运行：

```bash
python main.py
```

## 使用 Python 交互式 Shell

- **定义**：交互式 shell 是一个程序，允许你一次输入一条命令并查看结果。打开终端应用，输入 `python` 并按 `Enter`。这将启动 Python 交互式 shell。`>>>` 符号表示 Python 正在等待你输入命令。试着打印 `"Hello, world!"` 到终端：

```bash
print("Hello, world!")
```

你应该看到如下文本：

```bash
>>> print("Hello, world!")
Hello, world!
```

文本打印后，Python 会返回以下提示：

```bash
>>>
```

这意味着你可以输入另一条命令。

- **读取、求值、打印、循环周期（REPL）**：每当你输入命令时，解释器会读取输入、求值、打印结果，然后循环回显示 `>>>`，以便你继续输入命令。

## 变量

- **声明变量**：声明变量时，先写变量名，后跟赋值操作符（`=`），然后是值。值可以是数字、字串、布尔值等。以下是一些示例：

```py
name = 'John Doe'
age = 25
```

- **变量命名规范**：变量命名应遵循以下规则：

- 变量名只能以字母或下划线（_）开头，不能以数字开头。
  - 变量名只能包含字母数字字符（a-z、A-Z、0-9）和下划线（_）。
  - 变量名区分大小写——`age`、`Age` 和 `AGE` 都被视为不同变量。
  - 变量名不能是 Python 的保留关键字，如 `if`、`class` 或 `def`。
  - 多词变量名用下划线分隔。例如 `snake_case`。


## 注释

- **单行注释**：用于在代码中留下简短的备注。

```py
# This is a single line comment
```

- **多行字符串**：可用于留下较长的备注或注释掉代码块。

```py
"""
This is a multi-line string.
Here is some code commented out.

name = 'John Doe'
age = 25
"""
```

- **`print()` 函数**：要在控制台打印数据，可以使用 `print()` 函数，如下所示：

```py
print('Hello world!') # Hello world!
```

## Python 中常见的数据类型

- **简介**：Python 是动态类型语言，类似 JavaScript，意味着你不需要显式声明变量类型。语言会根据赋值自动判断变量类型。
- **整数**：没有小数的整数：

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **浮点数**：带小数的数字：

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **字串**：用引号括起来的字符序列：

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **布尔值**：表示 `True` 或 `False` 的值：

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **集合**：无序且唯一元素的集合：

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **字典**：由键值对组成的集合，用花括弧括起来：

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **元组**：不可变的有序集合，用括号括起来：

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **范围**：数字序列，常用于循环：

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **列表**：支持不同数据类型的有序元素集合：

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**：表示无值的特殊值：

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## 不可变类型和可变类型

- **不可变类型**：声明后不可更改，但变量可以重新赋值指向新对象。包括整数、浮点数、复数、布尔值、字串、元组、范围和 `None`。
- **可变类型**：声明后可以更改，可以添加、删除或更新元素。包括列表、集合和字典等集合类型。
- **`type()` 函数**：查看变量类型可以使用 `type()` 函数，如下所示：

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **`isinstance()` 函数**：用于检查变量是否匹配特定数据类型：

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## 字串操作

- **定义**：回想 JavaScript，字串是不可变的，意味着创建后不能更改。在 Python 中，可以使用单引号或双引号。建议选择一种规则并坚持使用：

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **访问字串中的字符**：可以用方括弧表示法访问字串中的字符，如下：

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **转义字串**：如果字串中包含引号，可以使用反斜线（`\`）转义，如下：

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **字串串接**：可以使用 `+` 操作符串接字串，如下：

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

另一种串接方式是使用 `+=` 操作符，它在同一步骤中执行串接和赋值，如下：

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**：即格式化字串字面值，允许你用简洁易读的语法进行插值和串接：

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **字串切片**：提取字串的部分内容。基本语法如下：

```py
str[start:stop:step]
```

起始位置表示切片开始的索引，结束位置表示切片结束的索引（不包含该位置），步长表示切片的间隔。示例如下：

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **获取字串长度**：使用 `len()` 函数返回字串中的字符数：

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## 使用 `in` 操作符

- **`in` 操作符**：返回布尔值，表示字符是否存在于字串中：

```py
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

## 常用字串方法

- **`upper()`**：将所有字符转换为大写并返回新字串：

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`lower()`**：将所有字符转换为小写并返回新字串：

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`strip()`**：返回去除指定前后字符的字串副本（无参数时去除前后空白）：

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**：返回替换所有旧字串为新字串后的新字串：

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**：使用指定分隔符将字串拆分为列表。分隔符指定拆分位置：

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**：将可迭代对象的元素用分隔符连接成字串。可迭代对象是可循环的集合，如列表、字串或元组：

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`startswith(prefix)`**：判断字串是否以指定前缀开头，返回布尔值：

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`endswith(suffix)`**：判断字串是否以指定后缀结尾，返回布尔值：

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`find()`**：返回子字串首次出现的索引，找不到则返回 `-1`：

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`count(substring)`**：统计子字串在字串中出现的次数：

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`capitalize()`**：返回首字母大写，其余字母小写的新字串：

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`isupper()`**：如果字串所有字母均为大写，返回 `True`，否则返回 `False`：

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`islower()`**：如果字串所有字母均为小写，返回 `True`，否则返回 `False`：

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`title()`**：返回每个单词首字母大写的新字串：

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`maketrans()`**：用于创建字符映射表，常与 `translate()` 方法配合使用，将映射应用于字串并返回翻译结果：

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## 整数和浮点数的常用操作

- **基本数学运算**：Python 支持整数和浮点数的加、减、乘、除运算：

```py
int_1 = 56
int_2 = 12
float_1 = 5.4
float_2 = 12.0

# Addition

print('Integer Addition:', int_1 + int_2) # Integer Addition: 68
print('Float Addition:', float_1 + float_2) # Float Addition: 17.4

# Subtraction

print('Int Subtraction:', int_1 - int_2) # Int Subtraction: 44
print('Float Subtraction:',  float_2 - float_1) # Float Subtraction: 6.6

# Multiplication

print('Int Multiplication:', int_1 * int_2) # Int Multiplication: 672
print('Float Multiplication:', float_2 * float_1) # Float Multiplication: 64.80000000000001

# Division

print('Division:', int_1 / int_2) # Division: 4.666666666666667
print('Float Division:', float_2 / float_1) # Float Division: 2.222222222222222
```

当加法操作中包含浮点数和整数时，结果会转换为浮点数，如下：

```py
int_1 = 56
float_1 = 5.4

print(int_1 + float_1) # 61.4
```

- **取模运算符（`%`）**：返回两个数相除的余数：

```py
int_1 = 56
int_2 = 12

print(int_1 % int_2) # 8
```

- **向下取整除法（`//`）**：除法后向下取整到最接近的整数：

```py
int_1 = 56
int_2 = 12

print(int_1 // int_2) # 4
```

- **幂运算符（`**`）**：用于计算一个数的幂：

```py
int_1 = 4
int_2 = 2

print(int_1 ** int_2) # 16
```

- **`float()` 函数**：将整数转换为浮点数：

```py
num = 4

print(float(num)) # 4.0
```

- **`int()` 函数**：将浮点数转换为整数：

```py
num = 4.0

print(int(num)) # 4
```

- **`round()` 函数**：将数字四舍五入到最接近的整数：

```py
num_1 = 3.4
num_2 = 7.7

print(round(num_1)) # 3
print(round(num_2)) # 8
```

- **`abs()` 函数**：返回数字的绝对值：

```py
num = -13

print(abs(num)) # 13
```

- **`pow()` 函数**：用于计算一个数的幂：

```py
result = pow(2, 3) 
print(result)  # 8
```

## 函数操作

- **定义**：函数是可重用的代码块，接受输入（参数）并返回输出。调用函数时，引用函数名后跟一对括号：

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call
print(result) # 7
```

如果函数没有显式返回值，默认返回值是 `None`：

```py
def greet():
    print('hello') 

result = greet() # hello
print(result) # None
```

你也可以为参数提供默认值，如下：

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

如果调用函数时参数数量不正确，会抛出 `TypeError`：

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

## Python 中的作用域

- **局部作用域**：函数或类内部声明的变量只能在该函数或类内部访问。

```py
def my_func():
    num = 10
    print(num)
```

- **闭包作用域**：嵌套函数可以访问其外层函数的变量。

```py
def outer_func():
    msg = 'Hello there!'
    
    def inner_func():
        print(msg)
    inner_func()

print(outer_func()) # Hello there!
```

- **全局作用域**：在任何函数或类外声明的变量，可在程序任何地方访问。

```py
tax = 0.70 

def get_total(subtotal):
    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **内置作用域**：Python 预定义函数、模块、关键字和对象的保留名称。

```py
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```

## 使用 `if`、`elif` 和 `else` 语句

- **`if` 语句**：用于判断条件是否为真。如果条件为 `True`，则执行该代码块。

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **`elif` 分支**：跟在 `if` 语句后面的条件。如果 `elif` 条件为 `True`，则执行该代码块。

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **`else` 分支**：当没有其他条件为 `True` 时执行。

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

你也可以使用嵌套的 `if` 语句，如下：

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## 真值和假值

- **定义**：在 Python 中，每个值都有内在的布尔值，表示在逻辑上下文中应被视为 `True` 还是 `False`。许多值被认为是真值，即在逻辑上下文中求值为 `True`。其他值是假值，求值为 `False`。以下是假值示例：

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

其他值如非零数字和非空字串是真值。

## 使用 `bool()` 函数

- **定义**：如果你想检查一个值是真值还是假值，可以使用内置的 `bool()` 函数。它显式将值转换为布尔等价，并对真值返回 `True`，对假值返回 `False`。以下是几个示例：

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## 布尔操作符和短路求值

- **定义**：这些特殊操作符允许你组合多个表达式，创建更复杂的决策逻辑。Python 中有三个布尔操作符：`and`、`or` 和 `not`。
- **`and` 操作符**：接受两个操作数，如果第一个是假值，则返回第一个，否则返回第二个。两个操作数都必须是真值，表达式才为真值。

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

你也可以在条件语句中使用 `and` 操作符，如下：

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **`or` 操作符**：如果第一个操作数是真值，则返回第一个，否则返回第二个。只要至少一个操作数是真值，`or` 表达式就为真值。示例如下：

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

和 `and` 操作符一样，你也可以在条件语句中使用 `or` 操作符：

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **短路求值**：`and` 和 `or` 操作符被称为短路操作符。短路求值意味着 Python 从左到右检查值，一旦确定最终结果就停止。
- **`not` 操作符**：接受单个操作数，取反其布尔值。将真值转换为 `False`，假值转换为 `True`。与之前的操作符不同，`not` 总是返回 `True` 或 `False`。示例如下：

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

下面是 `not` 操作符在条件语句中的示例：

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```


## Python 列表

- **简介**：Python 中的列表数据类型是有序元素序列，可以包含字串、数字甚至其他列表。列表是可变的，且索引从零开始。

```python
cities = ['Los Angeles', 'London', 'Tokyo']
```

- **访问列表元素**：访问 `cities` 列表中的元素，可以通过索引号引用：

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # Los Angeles
```

- **使用负索引访问元素**：访问列表最后一个元素，可以使用 `-1` 作为索引：

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # Tokyo
```

- 负索引用于从列表末尾访问元素，而非从索引 `0` 开始。

- **使用 `list()` 构造函数创建列表**：列表也可以用 `list()` 构造函数创建。`list()` 构造函数用于将可迭代对象转换为列表：

```python
developer = 'Jessica'

print(list(developer)) 
# Result: ['J', 'e', 's', 's', 'i', 'c', 'a']
```

- **获取列表长度**：使用 `len()` 函数获取列表长度：

```python
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

- **列表的可变性**：列表是可变的，只要传入有效索引，就可以更新列表中的元素。更新列表中特定索引的元素，可以给该索引赋新值：

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

- **索引超出范围错误**：如果传入的索引（正或负）超出列表边界，会抛出 `IndexError`：

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
"""
```

- **从列表中移除元素**：可以使用 `del` 关键字移除列表元素：

```python
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

- **检查元素是否存在于列表中**：使用 `in` 关键字检查元素是否存在：

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **嵌套列表**：列表可以嵌套在其他列表中：

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

- 访问嵌套列表时，需要使用索引 `2`，因为列表索引从零开始。

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

- 进一步访问嵌套列表中的第二个语言，需要使用索引 `1`：

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # Rust
```

- **列表解包**：列表解包是将列表中的值赋给新变量的技术。下面示例将 `developer` 列表解包到名为 `name`、`age` 和 `job` 的新变量中：

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer
```

- 如果赋值操作符左侧变量数量与列表项总数不匹配，会抛出 `ValueError`。

- **收集列表剩余元素**：使用星号（`*`）操作符收集列表中剩余元素，如下：

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer
```

- **列表切片**：切片是通过切片操作符 `:` 访问列表部分元素的概念。要切片从索引 `1` 开始，到索引 `3` 之前结束的列表，可以使用以下语法：

```python
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie']
desserts[1:3] # ['Cookies', 'Ice Cream']
```

- **步长间隔**：也可以指定步长，决定索引间隔。以下示例展示如何使用切片提取偶数列表：

```python
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

- **append()**：用于向列表末尾添加项。下面示例使用 `append()` 方法向 `numbers` 列表添加数字 `6`：

- **insert()**：用于在列表指定索引插入项。下面示例使用 `insert()` 方法：

- **pop()**：用于移除并返回列表中特定项：

```py
numbers = [1, 2, 3, 4, 5]
```

- 如果不指定 `pop` 方法的元素，则移除最后一个元素。

```py
numbers = [1, 2, 3, 4, 5]
```

- **clear()**：用于移除列表中所有项：

```py
numbers = [1, 2, 3, 4, 5]
```

- **sort()**：`sort()` 方法用于原地排序元素。下面示例对随机 `numbers` 列表进行原地排序：

```py
numbers = [19, 2, 35, 1, 67, 41]

```

- **sorted()**：内置函数，返回新排序列表，不修改原列表：

```py
numbers = [19, 2, 35, 1, 67, 41]

print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
print(numbers) # [19, 2, 35, 1, 67, 41]
```

- **reverse()**：用于反转列表元素顺序：

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **index()**：用于查找元素在列表中首次出现的索引：

```python
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

- 如果使用 `index()` 方法找不到元素，结果将是 `ValueError`。

## Python 元组

- **定义**：元组是 Python 数据类型，用于创建有序值序列。元组可以包含混合数据类型：

```py
developer = ('Alice', 34, 'Rust Developer')
```

- 元组是不可变的，创建后元素不能更改。尝试更新元组中的项会抛出 `TypeError`：

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **访问元组元素**：使用方括弧表示法和索引号访问元组元素：

```py
developer = ('Alice', 34, 'Rust Developer')

```

- 可以使用负索引从元组末尾访问元素：

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- 如果传入的索引号超出或等于元组长度，会抛出 `IndexError`：

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- 元组也可以用 `tuple()` 构造函数创建。构造函数中可以传入字符串、列表甚至其他元组等可迭代对象。

```python
developer = 'Jessica'

print(tuple(developer)) 
# Result: ('J', 'e', 's', 's', 'i', 'c', 'a')
```

- **验证元组中的项**：使用 `in` 关键字检查项是否在元组中：

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True

```

- **元组解包**：可以像这样解包元组中的项：

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

- 如果需要收集元组中剩余元素，可以使用星号（`*`）操作符：

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer
```

- **元组切片**：切片可用于提取元组部分元素。例如，`pie` 和 `cookies` 项可以切片成一个新元组：

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
```

- **从元组中移除项**：移除元组项会引发 `TypeError`，因为元组是不可变的：

```py
developer = ('Jane Doe', 23, 'Python Developer')

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>

```

- **何时使用元组而非列表？**：如果需要动态集合，可以添加、删除和更新元素，应使用列表。如果数据集合是固定且不可变的，应使用元组。

## 常用元组方法

- **`count()`**：用于确定某个项在元组中出现的次数。例如，你可以检查语言 `'Rust'` 在元组中出现了多少次：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

- 如果 `count()` 方法中指定的项在元组中根本不存在，则返回值为 `0`：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

- 如果 `count()` 方法没有传入任何参数，Python 将返回一个 `TypeError`。

- **index()**：用于查找元组中某个特定项所在的索引。下面是使用 `index()` 方法查找语言 `'Java'` 索引的示例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

- 如果找不到指定的项，Python 会返回一个 `ValueError`。

- 你可以向 `index()` 方法传入一个可选的起始索引，用来指定从元组的哪个位置开始查找该项：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

- 你也可以向 `index()` 方法传入一个可选的结束索引，用来指定在元组中查找该项时停止的位置：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

- **`sorted()`**：用于对任何可迭代对象的元素进行排序，并返回一个新的排序列表。下面是使用 `sorted()` 函数创建一个新数字列表的示例：

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

- **修改排序行为**：你可以使用可选的 `reverse` 和 `key` 参数自定义可迭代对象的排序行为。下面是使用 `key` 参数按长度对元组中的项进行排序的示例：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

- 你可以使用 `reverse` 参数创建一个按相反顺序排列的新列表，如下所示：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

```py
numbers = list(range(2, 11, 2))
```

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']
```

## Python 中的列表推导式

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
```

## 可迭代函数

- **`filter()`**：用于根据条件过滤可迭代对象中的元素。它返回一个只包含满足条件元素的迭代器。下面是创建一个只包含长度超过四个字符单词的新列表的示例：

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**：用于对可迭代对象中的每个项应用一个函数，并返回一个包含结果的新可迭代对象。下面是使用 `map()` 函数将摄氏温度列表转换为华氏温度的示例：

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**：用于计算列表或元组等可迭代对象的总和。下面是使用 `sum()` 函数的示例：

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- 你还可以传入一个可选的 `start` 参数，设置求和的初始值。下面是使用 `start` 参数作为位置参数的更新示例：

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- 你也可以选择使用 `start` 参数作为关键字参数，如下所示：

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
```

## Lambda 函数

- **定义**：Python 中的 lambda 函数是一种简洁的创建匿名函数（无名函数）的方法。
- Lambda 函数通常用作另一个函数的参数。下面是一个 lambda 函数的示例：

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
```

```python
dictionary = {
```

```python
pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
```

```python
dictionary[key]
```

## 常用字典方法

- **`get()` 方法**：`get()` 方法用于获取与键关联的值。它类似于方括弧表示法，但允许你设置默认值，避免键不存在时出现错误。

```python
dictionary.get(key, default)
```

- **`keys()` 和 `values()` 方法**：`keys()` 和 `values()` 方法分别返回包含字典所有键和值的视图对象。视图对象是一种查看字典内容而不创建数据副本的方式。

```python
pizza = {
    'name': 'Margherita Pizza',
    'price': 8.9,
    'calories_per_slice': 250
}

pizza.keys()
# dict_keys(['name', 'price', 'calories_per_slice'])

pizza.values()
# dict_values(['Margherita Pizza', 8.9, 250])
```

- **`items()` 方法**：`items()` 方法返回包含字典所有键值对的视图对象，包括键和值。

```python
pizza.items()
# dict_items([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250)])
```

- **`clear()` 方法**：`clear()` 方法删除字典中的所有键值对。

```python
pizza.clear()
```

- **`pop()` 方法**：`pop()` 方法删除指定键的键值对并返回其值。如果键不存在，则返回作为第二个参数指定的默认值。如果键不存在且未指定默认值，则会引发 `KeyError`。

```python
pizza.pop('price', 10)
```

- **`popitem()` 方法**：在 Python 3.7 及以上版本中，`popitem()` 方法删除最后插入的项。

```python
pizza.popitem()
```

- **`update()` 方法**：`update()` 方法使用另一个字典的键值对更新当前字典。如果有相同的键，其值会被覆盖。新键会作为新的键值对添加到字典中。

```python
pizza.update({ 'price': 15, 'total_time': 25 })
```

## 遍历字典

- **遍历值**：如果你需要遍历字典中的值，可以写一个 `for` 循环，使用 `values()` 获取字典的所有值。

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,

for price in products.values():

```

输出：

```md
990
600
250
70
```

- **遍历键**：如果你需要遍历上面 `products` 字典中的键，可以直接写 `products.keys()` 或 `products`。

```python
for product in products.keys():
    print(product)
    
# Or

for product in products:
    print(product)
```

输出：

```md
Laptop
Smartphone
Tablet
Headphones
```

- **遍历键值对**：如果你需要同时遍历键及其对应的值，可以遍历 `products.items()`。你会得到包含键及其对应值的元组。

```python
for product in products.items():
```

输出：

```md
Smartphone 600
Tablet 250
```

将索引和值分别赋给不同的循环变量是使用 `enumerate()` 的常见方式。例如，使用 `products.items()`，你可以获得索引以及整个键值对：

```python
for index, product in enumerate(products.items(), 1):
    print(index, product)
```

输出：

```md
2 ('Smartphone', 600)
```

## 集合

- **集合**：集合是 Python 内置的数据结构，不允许重复值。集合是可变且无序的，这意味着它们的元素没有特定顺序，因此不能使用索引或键访问它们。此外，集合只能包含不可变数据类型的值，如数字、字符串和元组。

- **定义集合**：要定义集合，需要将元素写在花括弧内，并用逗号分隔。

- **`remove()` 和 `discard()` 方法**：要从集合中移除元素，可以使用 `remove()` 方法或 `discard()` 方法，传入要移除的元素作为参数。`remove()` 方法在元素不存在时会引发 `KeyError`，而 `discard()` 方法不会。

- **`clear()` 方法**：`clear()` 方法移除集合中的所有元素。

## 数学集合操作

- **`issubset()` 和 `issuperset()` 方法**：`issubset()` 和 `issuperset()` 方法分别检查一个集合是否是另一个集合的子集或超集。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 5}

print(your_set.issubset(my_set)) # True
print(my_set.issuperset(your_set)) # True
```

- **`isdisjoint()` 方法**：`isdisjoint()` 方法检查两个集合是否不相交，即没有共同元素。

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

print(my_set.isdisjoint(your_set)) # True
```

- **并集操作符（`|`）**：并集操作符 `|` 返回一个包含两个集合所有元素的新集合。

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

my_set | your_set # {1, 2, 3, 4, 5, 6}
```

- **交集操作符（`&`）**：交集操作符 `&` 返回一个只包含两个集合共有元素的新集合。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set & your_set # {2, 3, 4}
```

- **差集操作符（`-`）**：差集操作符 `-` 返回一个包含第一个集合中不在另一个集合中的元素的新集合。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set - your_set # {1, 5}
```

- **对称差集操作符（`^`）**：对称差集操作符 `^` 返回一个包含只在第一个或第二个集合中出现但不同时出现的元素的新集合。

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set ^ your_set # {1, 5, 6}
```

- **`in` 操作符**：你可以使用 `in` 操作符检查元素是否在集合中。

```python
print(5 in my_set)
```

## 导入语句

- **导入语句**：要访问内置模块中定义的元素，使用导入语句。导入语句通常写在文件顶部。导入语句对函数、类、常量、变量以及模块中定义的其他元素的作用相同。

- **基本导入语句**：你可以使用 `import` 关键字后跟模块名：

```python
import module_name
```

然后，如果你需要调用该模块中的函数，使用点表示法，模块名后跟函数名。

```python
module_name.function_name()
```

例如，下面的代码导入 `math` 模块并计算 36 的平方根：

```python
import math

math.sqrt(36)
```

- **使用别名导入模块**：如果需要用不同名称（别名）导入模块，可以在导入语句末尾使用 `as` 关键字加上别名。这通常用于模块名较长或避免命名冲突。

```python
import module_name as module_alias
```

例如，要在代码中将 `math` 模块称为 `m`，可以这样赋予别名：

```python
import math as m
```

然后，可以使用别名访问模块的元素：

```python
m.sqrt(36)
```

- **导入特定元素**：如果不需要模块中的所有内容，可以使用 `from` 导入特定元素。此时导入语句以 `from` 开头，后跟模块名，再用 `import` 关键字，最后是要导入的元素名。

```python
from module_name import name1, name2
```

然后，在 Python 脚本中可以直接使用这些名称，无需模块前缀。例如：

```python
from math import radians, sin, cos

angle_degrees = 40
angle_radians = radians(angle_degrees)

sine_value = sin(angle_radians)
cos_value = cos(angle_radians)

print(sine_value) # 0.6427876096865393
print(cos_value)  # 0.766044443118978
```
 
这很方便，但如果你已有同名函数或变量，可能会导致命名冲突。选择导入方式时请注意。

如果需要为这些名称指定别名，也可以使用 `as` 关键字加上别名。

```python
from module_name import name1 as alias1, name2 as alias2
```

- **带星号的导入语句（`*`）**：星号告诉 Python 导入模块中的所有内容，但导入后使用时不需要模块名前缀。

```python
from module_name import *
```

例如，使用此方式导入 `math` 模块后，可以直接调用该模块定义的任何函数，无需指定模块名。

```python
from math import *
print(sqrt(36))  # 6.0
```

不过，这通常不推荐，因为可能导致命名空间冲突，且难以判断名称来源。

## Python 中的常见错误

- **SyntaxError**：当代码不符合语法规则时，Python 抛出的错误。例如，代码 `print("Hello there"` 会导致语法错误，错误信息为 `SyntaxError: '(' was never closed`，因为代码缺少闭合括号。
- **NameError**：当你尝试访问未定义的变量或函数时，Python 会抛出 `NameError`。例如，如果代码中有 `print(username)`，但未先定义 `username` 变量，则会出现名称错误，错误信息为 `NameError: name 'username' is not defined`。
- **TypeError**：当你对两个或多个不兼容的数据类型执行操作时，Python 会抛出此错误。例如，尝试将字符串与数字相加时，会出现错误 `TypeError: can only concatenate str (not "int") to str`。
- **IndexError**：如果访问列表或其他序列（如元组和字符串）中不存在的索引，会出现 `IndexError`。例如，在 `Hello world` 字符串中，最后一个字符的索引是 `10`。如果你这样访问字符 `greet = "hello world"; print(greet[11])`，会出现错误，错误信息为 `IndexError: string index out of range`。
- **AttributeError**：当你尝试在某种类型的对象上使用不存在的方法或属性时，Python 会抛出此错误。例如，在字符串 `"hello".append("!")` 上调用 `.append()` 会导致错误，错误信息为 `AttributeError: 'str' object has no attribute 'append'`。

## 异常处理

- **`try...except`**：用于执行可能引发异常的代码块。`try` 块是你预期可能发生错误的代码块，而 `except` 块捕获指定异常并在该错误发生时执行。示例如下：

  ```py
  try:
    print(22 / 0)
  except ZeroDivisionError:
    print('You can\'t divide by zero!')
    # You can't divide by zero!
  ```

你也可以链式使用多个 `except` 块，以处理更多类型的异常：

  ```py
  try:
    number = int(input('Enter a number: '))
    print(22 / number)
  except ZeroDivisionError:
    print('You cannot divide by zero!')
    # You cannot divide by zero! prints when you enter 0
  except ValueError:
    print('Please enter a valid number!')
    # Please enter a valid number! prints when you don't enter an integer
  ```

- **`else` 和 `finally`**：这两个块扩展了 `try...except`。如果没有异常发生，`else` 块会执行。`finally` 块无论是否发生错误都会执行。

  ```py
  try:
    result = 100 / 4
  except ZeroDivisionError:
    print('You cannot divide by zero!') # This will not run
  else:
    print(f'Result is {result}') # Result is 25.0
  finally:
    print('Execution complete!') # Execution complete!
  ```

- **异常对象**：允许你访问异常本身，以便更好地调试和打印直接的错误信息。要访问异常对象，需要使用 `as` 关键字。示例如下：

  ```py
  try:
      value = int('This will raise an error')
  except ValueError as e:
      print(f'Caught an error: {e}')
      # Caught an error: invalid literal for int() with base 10: 'This will raise an error'
  ```

- **`raise` 语句**：允许你手动引发异常。当满足某个条件时，可以使用它抛出异常。示例如下：

  ```py
  def divide(a, b):
      if b == 0:
          raise ZeroDivisionError('You cannot divide by zero')
      return a / b
  ```
  
## 异常信号
  
- `raise` 语句在创建自定义异常时也很有用，因为你可以用它抛出带有自定义消息的异常。示例如下：

  ```py
  class InvalidCredentialsError(Exception):
      def __init__(self, message="Invalid username or password"):
          self.message = message
          super().__init__(self.message)

  def login(username, password):
      stored_username = "admin"
      stored_password = "password123"
      
      if username != stored_username or password != stored_password:
          raise InvalidCredentialsError()
      
      return f"Welcome, {username}!"
  ```  

下面是如何使用 `login` 函数处理 `InvalidCredentialsError` 异常的示例：

  ```py
  # failed login attempt
  try:
      message = login("user", "wrongpassword")
  except InvalidCredentialsError as e:
      print(f"Login failed: {e}")
  else:
      print(message)

  # successful login attempt
  try:
      message = login("admin", "password123")
  except InvalidCredentialsError as e:
      # This block is not executed because the login was successful
      print(f"Login failed: {e}")
  else:
      # The else block runs if the 'try' block completes without an exception
      print(message)
  ```

- `raise` 语句也可以与 `from` 关键字一起使用，用于链式异常，显示不同错误之间的关系：

  ```py
  def parse_config(filename):
    try:
        with open(filename, 'r') as file:
            data = file.read()
            return int(data)
    except FileNotFoundError:
        raise ValueError('Configuration file is missing') from None
    except ValueError as e:
        raise ValueError('Invalid configuration format') from e

  config = parse_config('config.txt')
  ```

## 双下划线（魔法）方法

- **定义**：以双下划线开头和结尾的特殊方法（例如 `__init__`、`__len__`、`__str__`、`__eq__`）。Python 内部使用它们来实现内置操作。

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **间接调用双下划线方法**：你不需要直接调用双下划线方法。Python 会在特定操作发生时自动调用它们。这些操作包括：

- **算术操作，如加法、减法、乘法、除法等**。加法调用 `__add__()`，减法调用 `__sub__()`，乘法调用 `__mul__()`，除法调用 `__truediv__()`。
        
- **字符串操作，如串接、重复、格式化和转换为文本**。串接调用 `__add__()`，重复调用 `__mul__()`，格式化调用 `__format__()`，文本转换调用 `__str__()` 和 `__repr__()`，等等。
        
- **比较操作，如相等、小于、大于等**。相等检查调用 `__eq__()`，小于调用 `__lt__()`，大于调用 `__gt__()`，等等。
        
- **迭代操作，如使对象可迭代和获取下一个元素**。返回迭代器调用 `__iter__()`，获取下一个元素调用 `__next__()`。

## 现实示例：购物车

- **带双下划线方法的购物车类**：允许添加、移除、迭代和检查内容，具有内置行为。

```python
class Cart:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

    def remove(self, item):
        if item in self.items:
            self.items.remove(item)
        else:
            print(f'{item} is not in cart')

    def list_items(self):
        return self.items

    def __len__(self):
        return len(self.items)

    def __getitem__(self, index):
        return self.items[index]

    def __contains__(self, item):
        return item in self.items

    def __iter__(self):
        return iter(self.items)

cart = Cart()
cart.add('Laptop')
print(len(cart))        # 1
print('Laptop' in cart) # True
```

## 什么是面向对象编程？

- **面向对象编程**：一种编程风格，开发者将代码中的所有内容视为现实世界中的对象。通常称为 OOP。帮助你有效组织和管理代码的四个关键原则是 **封装**、**继承**、**多态** 和 **抽象**。
- **类**：创建对象的蓝图。每个由类创建的对象都有定义数据的属性和决定对象行为的方法。

## 什么是封装？

- **封装**：将对象的属性和方法打包成一个单元。它让你通过一组简单的公共方法和属性（像门一样）隐藏对象的内部状态。门后是私有属性和方法，控制数据如何变化以及谁可以访问它。
- **封装示例**：如果你想跟踪钱包余额，你会允许存款和取款，但不希望任何人篡改钱包余额本身：

```py
class Wallet:
   def __init__(self, balance):
       self.__balance = balance # Private attribute

   def deposit(self, amount):
       if amount > 0:
           self.__balance += amount # Add to the balance safely

   def withdraw(self, amount):
       if 0 < amount <= self.__balance:
           self.__balance -= amount # Remove from the balance safely

account = Wallet(500)
print(account.__balance) # AttributeError: 'Wallet' object has no attribute '__balance'
```

- **单下划线和双下划线前缀属性的区别**：属性和方法前加单下划线表示仅供内部使用，这是约定，不强制禁止外部访问。前加双下划线则有效防止外部访问这些属性和方法。

## 什么是 Getter 和 Setter？

- **Getter 和 Setter**：允许你控制类属性的访问和修改的方法。用 getter 获取值，用 setter 设置值。
- **属性（Properties）**：连接 getter 和 setter，允许访问数据。当你获取、设置或删除值时，它们会在幕后运行额外逻辑。
- **为什么用属性而不是方法**：属性比方法更易读，代码更简洁。它们允许你像访问普通属性一样用点符号访问值，无需括号。
- **创建 Getter**：使用 `@property` 装饰器创建 getter。以下是获取圆半径的 getter：

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self): # A getter to get the radius
        return self._radius
  
    @property
    def area(self):  # A getter to calculate area
        return 3.14 * (self._radius ** 2)

my_circle = Circle(3)

print(my_circle.radius) # 3
print(my_circle.area) # 28.26
```

在 `__init__()` 中使用 self.radius 确保在对象创建时调用 setter，从而立即捕获无效的半径值。

- **创建 Setter**：定义同名方法并在其上方使用 `@<property_name>.setter` 装饰器来创建 setter：

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self):  # A getter to get the radius
        return self._radius

    @radius.setter
    def radius(self, value):  # A setter to set the radius
        if value <= 0:
            raise ValueError('Radius must be positive')
        self._radius = value

my_circle = Circle(3)
print('Initial radius:', my_circle.radius) # Initial radius: 3

my_circle.radius = 8
print('After modifying the radius:', my_circle.radius) # After modifying the radius: 8
```

- **Python 如何处理 Getter 和 Setter**：定义 getter 和 setter 后，Python 会在你用普通属性语法访问时自动调用它们：

```py
my_circle.radius # This will call the getter
my_circle.radius = 4 # This will call the setter
```

在 setter 内部，不应给属性名本身赋值，否则会导致 `RecursionError`。

- **删除器（Deleter）**：在设置和获取值后，你可以用 `deleter` 控制如何删除它。删除器在你对属性使用 `del` 语句时运行自定义逻辑。创建删除器时，使用 `@<property_name>.deleter` 装饰器。

```py
  # Deleter
    @radius.deleter
    def radius(self):
        print("Deleting radius...")
        del self._radius
```

## 什么是继承？它如何促进代码复用？

- **继承**：子类使用父类的属性和方法的过程。继承促进代码复用，提供清晰的层次结构，并允许定制行为而无需重写所有内容。实现继承时，子类接收父类名称：

```py
class Parent:
    # Parent attributes and methods

class Child(Parent):
    # Child inherits, extends, and/or overrides where necessary
```

- **单继承和多继承**：子类从单个父类继承属性和方法称为 **单继承**，从多个父类继承称为 **多继承**。语法如下：

```py
class Parent:
    # Attributes and methods for Parent

class Child:
    # Attributes and methods for Child

class GrandChild(Parent, Child):
    # GrandChild inherits from both Parent and Child
    # GrandChild can combine or override behavior from each
```

- **`super()` 函数**：允许你在类对某方法有不同实现或扩展时，调用父类的方法而不重复代码。

## 什么是多态？它如何促进代码复用？

- **多态**：面向对象编程原则，允许不同类使用相同的方法名，但每个类在调用时有不同实现。语法如下：

```py
class A:
   def action(self): ...

class B:
   def action(self): ...

class C:
   def action(self): ...

Class().method()  # Works for A, B, or C
```

- **基于继承的多态**：父类定义方法，子类根据需要修改实现。

## 什么是名称重整？它如何工作？

- **名称重整**：Python 内部通过在双下划线前缀的属性前加下划线和类名来重命名属性，将 `__attribute` 变为 `_ClassName__attribute`。
- **名称重整的目的**：防止继承时意外覆盖属性和方法。以下代码帮助理解：

```py
class Parent:
    def __init__(self):
        self.__data = 'Parent data'

class Child(Parent):
    def __init__(self):
        super().__init__()
        self.__data = 'Child data'

c = Child()
print(c.__dict__) # {'_Parent__data': 'Parent data', '_Child__data': 'Child data'}
```

## 什么是抽象？它如何帮助保持复杂系统的组织？

- **抽象**：编程概念，隐藏对象或系统的复杂实现细节，只显示必要特性。在 Python 和其他语言中，抽象通过提高复用性简化复杂系统。
- **抽象示例**：生活中汽车只让你使用方向盘、踏板和换挡杆，而不需要了解发动机或刹车的工作原理。
- **Python 如何实现抽象**：通过 `abc` 模块实现。该模块提供 `ABC` 类（抽象基类）和 `@abstractmethod` 装饰器。抽象基类定义子类必须实现的公共方法和属性，不能实例化。
- **抽象方法的定义**：用 `@abstractmethod` 定义抽象方法，子类必须重写它，即使有默认实现。抽象的基本语法如下：

```py
from abc import ABC, abstractmethod

# Define an abstract base class
class AbstractClass(ABC):
    @abstractmethod
    def abstract_method(self):
        pass

# Concrete subclass that implements the abstract method
class ConcreteClassOne(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassOne')

# Another concrete subclass
class ConcreteClassTwo(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassTwo')
```

## 算法和大 O 表示法

- **算法**：解决问题或完成任务的一组明确指令。算法必须在有限步骤内完成，每步必须精确且明确。

- **大 O 表示法**：描述算法在输入规模增加时的最坏情况性能或增长率。关注资源使用随输入规模的增长，忽略常数和低阶项。

### 常见时间复杂度

- **O(1) - 常数时间**：算法所需时间与输入规模无关。

```python
def check_even_or_odd(number):
    if number % 2 == 0:
        return 'Even'
    else:
        return 'Odd'
```

- **O(log n) - 对数时间**：时间随输入增长缓慢。常见于通过分割问题规模的算法（如二分查找）。

- **O(n) - 线性时间**：运行时间与输入规模成正比。

```python
for grade in grades:
    print(grade)
```

- **O(n log n) - 线性对数时间**：高效排序算法如归并排序和快速排序的常见时间复杂度。

- **O(n²) - 平方时间**：运行时间呈平方增长。常见于嵌套循环。

```python
for i in range(n):
    for j in range(n):
        print("Hello, World!")
```

### 空间复杂度

- **O(1) - 常数空间**：算法使用的内存与输入规模无关。
- **O(n) - 线性空间**：内存使用随输入规模成正比增长。
- **O(n²) - 平方空间**：内存使用随输入规模呈平方增长。

## 解决问题的技巧

- **理解问题**：多次阅读问题描述。确定输入、预期输出以及如何将输入转换为输出。

- **伪代码**：算法逻辑的高级描述，与语言无关。使用常见书面语言混合编程结构，如 `IF`、`ELSE`、`FOR`、`WHILE`。

```md
GET original_string
SET reversed_string = ""
FOR EACH character IN original_string:
  ADD character TO THE BEGINNING OF reversed_string
DISPLAY reversed_string
```

- **边界情况**：算法应处理的边界上的特定有效输入。始终考虑并测试边界情况。

## 数组

- **静态数组**：初始化时大小固定。元素存储在相邻内存位置。程序执行期间大小不可变。

- **动态数组**：程序执行期间可自动增长或缩小。通过自动复制到更大数组来处理调整大小。

### Python 列表（动态数组）

```python
numbers = [3, 4, 5, 6]

# Access elements
numbers[0]  # 3

# Update elements
numbers[2] = 16

# Add elements
numbers.append(7)
numbers.insert(3, 15)  # Insert at specific index

# Remove elements
numbers.pop(2)  # Remove at specific index
numbers.pop()   # Remove last element
```

### 动态数组的时间复杂度

- **访问**：O(1)
- **末尾插入**：平均 O(1)，调整大小时 O(n)
- **中间插入**：O(n)
- **删除**：中间 O(n)，末尾 O(1)

## 栈

- **栈**：后进先出（LIFO）数据结构。元素仅从顶部添加和移除。

- **入栈操作**：向栈顶添加元素。时间复杂度：O(1)。

- **出栈操作**：从栈顶移除元素。时间复杂度：O(1)。

```python
# Using Python list as stack
stack = []

# Push operations
stack.append(1)
stack.append(2)
stack.append(3)

# Pop operations
top_element = stack.pop()  # Returns 3
```

## 队列

- **队列**：先进先出（FIFO）数据结构。元素从尾部添加，从头部移除。

- **入队操作**：向队尾添加元素。时间复杂度：O(1)。

- **出队操作**：从队头移除元素。时间复杂度：O(1)。

```python
from collections import deque

# Using deque for efficient queue operations
queue = deque()

# Enqueue operations
queue.append(1)
queue.append(2)
queue.append(3)

# Dequeue operations
first_element = queue.popleft()  # Returns 1
```

## 链表

- **链表**：线性数据结构，每个节点包含数据和指向下一个节点的引用。节点像链条一样连接。

### 单链表

- **结构**：每个节点有数据和一个指向下一个节点的引用。
- **遍历**：只能从头到尾向前移动。
- **头节点**：链表的第一个节点，通常是唯一可直接访问的节点。
- **尾节点**：链表的最后一个节点，指向 `None`。

### 操作和时间复杂度

- **开头插入**：O(1)
- **末尾插入**：O(n) - 需遍历到末尾
- **中间插入**：O(n) - 需遍历到指定位置
- **开头删除**：O(1)
- **末尾删除**：O(n) - 需遍历找到前一个节点
- **中间删除**：O(n) - 需遍历找到节点

### 双链表

- **结构**：每个节点有数据和两个引用：指向下一个节点和前一个节点。
- **遍历**：可双向移动。
- **内存**：由于额外引用，内存使用比单链表多。

## 哈希映射和集合

### 映射和哈希映射

- **映射（抽象数据类型）**：管理键值对集合。每个键必须唯一，但值可以重复。

- **哈希映射**：映射抽象数据类型的具体实现，使用哈希技术。通过哈希函数生成键的哈希值，决定在底层数组中的存储位置。

### Python 字典（哈希映射）

```python
# Creating dictionaries
my_dictionary = {
    "A": 1,
    "B": 2, 
    "C": 3
}

# Alternative creation
my_dictionary = dict(A=1, B=2, C=3)

# Access and modify
value = my_dictionary["A"]  # 1
my_dictionary["A"] = 4      # Update value
del my_dictionary["A"]      # Remove key-value pair

# Check membership
"C" in my_dictionary

# Get keys, values, items
my_dictionary.keys()
my_dictionary.values()
my_dictionary.items()
```

### 哈希映射的时间复杂度

- **平均情况**：插入、获取、删除均为 O(1)
- **最坏情况**：大量哈希冲突时为 O(n)

### 集合

- **集合**：无序且唯一元素的集合。不允许重复，不维护特定顺序。

- **仅允许不可变元素**：集合只能包含不可变数据类型（数字、字串、元组），因为哈希值必须保持不变。

```python
# Creating sets
numbers = {1, 2, 3, 4}
empty_set = set()  # Must use set(), not {}

# Add and remove elements
numbers.add(5)
numbers.remove(4)      # Raises KeyError if not found
numbers.discard(4)     # No error if not found

# Set operations
set_a = {1, 2, 3, 4}
set_b = {2, 3, 4, 5, 6}

# Union, intersection, difference, symmetric difference
set_a.union(set_b)                    # or set_a | set_b
set_a.intersection(set_b)             # or set_a & set_b
set_a.difference(set_b)               # or set_a - set_b
set_a.symmetric_difference(set_b)     # or set_a ^ set_b

# Subset and superset checks
set_a.issubset(set_b)
set_a.issuperset(set_b)
set_a.isdisjoint(set_b)

# Membership testing
5 in numbers
```

### 集合的时间复杂度

- **平均情况**：添加、移除、成员测试均为 O(1)
- **最坏情况**：哈希冲突导致为 O(n)

## 哈希冲突

- **哈希冲突**：不同键产生相同哈希值时发生。

- **冲突解决策略**：
  - **链式法**：每个数组索引指向一个链表，存储所有具有相同哈希值的元素。
  - **开放寻址法**：使用预定义序列寻找下一个可用索引。

## 何时使用各数据结构

- **列表**：需要有序、索引访问且大小未知时。
- **栈**：用于后进先出操作（撤销功能、表达式求值、回溯）。
- **队列**：用于先进先出操作（任务调度、广度优先搜索）。
- **链表**：频繁在开头插入/删除、大小未知且不需要随机访问时。
- **哈希映射**：快速键值查找、计数、缓存。
- **集合**：唯一性检查、数学集合操作、去重。

## 查找算法

查找算法让你在某个项列表中查找目标。

在计算机科学中，有两种你应该了解的查找算法。它们是**线性查找**和**binary search**算法。理解这两种算法的区别以及何时使用每一种非常重要。

### 线性查找

- 线性查找迭代一个项列表，从头开始查看每一项，直到找到目标项。
- 如果找到目标项，则返回该项在列表中的索引。
- 如果未找到目标，它将返回 `-1`，这在大多数编程语言中表示**无效索引**。
- 因为线性查找会检查每一项直到找到目标，所以对于大量项的列表来说，它并不高效。
- 线性查找的时间复杂度是 `O(n)`，因为查找列表所需的时间随着列表大小线性增长。
- 线性查找的空间复杂度是 `O(1)`，因为它不需要任何额外空间来查找列表。

### 二分查找

- 二分查找通过将列表分成两半，检查目标是否在中间。
- 二分查找的前提是列表项必须按升序排列。
- 二分查找更高效，因为它每次将列表减半，忽略不包含目标的半边。
- 找到目标时返回其索引。
- 未找到时检查目标在左半边还是右半边。
- 继续将剩余部分分半，直到找到目标。
- 最终未找到时返回 `-1`。
- 二分查找的时间复杂度是 `O(log n)`，因为搜索时间随列表大小对数增长。
- 二分查找的空间复杂度是 `O(1)`，因为不需要额外空间。

### 线性查找与二分查找的区别

- 与线性查找相比，二分查找更适合用于大型项列表。
- 线性查找的时间复杂度是 `O(n)`，因为查找列表所需的时间随着列表大小线性增长。
- 二分查找的时间复杂度是 `O(log n)`，因为查找列表所需的时间随着列表大小呈对数增长。

## 排序算法与分治法

在计算机科学中，分治是一种将问题拆分为更小子问题以便更容易解决的技术。递归是分治中常用的技术，分治是一种强大的策略，用于实现许多高效的排序算法，如归并排序。

### 归并排序

- 归并排序是一种遵循分治法的排序算法。
- 它通过递归地将一个列表划分为更小的子列表，直到每个子列表只包含一个元素。
- 然后它反复将子列表合并回去，按排序顺序。
- 归并排序的时间复杂度是 `O(n log n)`，因为列表不断被对半划分 `(log n)`，然后再合并 `(O(n))`。
- 归并排序的空间复杂度是 `O(n)`，因为它不是原地排序算法。

## 图概述

图是一组由边（连接）连接的节点（顶点）。每个节点可以连接多个其他节点，形成网络。图的不同类型包括：

- 有向图：边有方向（从一个节点到另一个节点），通常用直线和箭头表示。
- 无向图：边无方向，用简单线条表示。
- 顶点：每个节点关联一个标签或标识符。
- 有环图：包含环（起点和终点相同的路径）。
- 无环图（DAG）：不包含环。
- 边带标签：每条边有标签，通常绘制在对应边旁。
- 带权图：边有权值（数值），可用于算术操作。
- 非连通图：包含两个或多个不通过边连接的节点。

图用于地图、网络、推荐系统、依赖解析等多种应用。

## 图遍历

涉及访问图中所有节点。两种主要算法：

- **广度优先搜索（BFS）**
  - 使用队列。
  - 按层级探索。
  - 在无权图中找到最短路径。

- **深度优先搜索（DFS）**
  - 使用栈（或递归）。
  - 先深入探索一个分支，再回溯。
  - 用于检测环和路径查找。

## 图的表示

图主要有两种表示方式：

- **邻接表**
  - 每个节点有一个邻居列表。
  - 对稀疏图空间高效。
  - 易于遍历邻居。

- **邻接矩阵**
  - 二维数组，行列代表节点。
  - 对大图空间消耗大。
  - 快速检查两节点间是否有边。

## 树

树是无环且连通的特殊图。关键属性包括：

- 无环（无起点和终点相同的路径）。
- 连通（每个节点都能从其他节点访问）。

### 常见树类型

最常见的树类型：

- 二叉树
  - 每个节点最多有两个子节点，左子节点和右子节点。

- 二叉搜索树（BST）
  - 每个左子节点小于父节点，每个右子节点大于父节点。


## 字典树（Trie）

也称前缀树，用于存储字符串集合，每个节点代表一个字符。

共享前缀只存储一次，适合自动补全和拼写检查。

查找和插入操作时间复杂度为 O(L)，L 是字符串长度。

## 优先队列

优先队列是一种抽象数据类型，每个元素有优先级。

队列和栈只考虑插入顺序，而优先队列考虑元素优先级。

标准队列遵循先进先出（FIFO），栈遵循后进先出（LIFO）。优先队列中，优先级高的元素先被处理，无视插入顺序。

## 堆

堆是一种特殊的基于树的数据结构，具有堆属性。

堆属性决定父节点和子节点的关系。有两种堆：

- 最大堆
  - 每个父节点的值大于或等于其子节点。
  - 最大元素位于根节点。

- 最小堆
  - 每个父节点的值小于或等于其子节点。
  - 最小元素位于根节点。

### Python `heapq` 模块示例

```py
import heapq

# Create empty heap
my_heap = []

# Insert elements
heapq.heappush(my_heap, 9)
heapq.heappush(my_heap, 3)
heapq.heappush(my_heap, 5)
print(my_heap) # [3, 9, 5]

# Remove smallest element
print(heapq.heappop(my_heap))  # 3
print(my_heap) # [5, 9]

# Push + Pop in one step
print(heapq.heappushpop(my_heap, 7)) # 5
print(my_heap) # [7, 9]

# Transform list into heap
nums = [5, 7, 3, 1]
heapq.heapify(nums)
```

### 使用优先级

```py
my_heap = []
heapq.heappush(my_heap, (3, "A"))
heapq.heappush(my_heap, (2, "B"))
heapq.heappush(my_heap, (1, "C"))

# Removes lowest number = highest priority
print(heapq.heappop(my_heap))  # (1, "C")
```

## 动态规划简介

- **定义**：动态规划是一种算法技术，通过将复杂问题拆分为更简单的子问题并存储结果，避免重复计算。
- **重叠子问题**：解决大问题时，多个子问题重复出现。通过存储解决方案避免重复计算。
- **最优子结构**：问题的最优解包含其子问题的最优解。意味着可以通过组合子问题的最优解构建整体最优解。

## 动态规划方案

- **备忘录法（自顶向下）**：存储昂贵函数调用的结果，当相同输入再次出现时返回缓存结果。

```py
def climb_stairs_memo(n, memo={}):
    """Dynamic programming with memoization"""
    # Check if we've already calculated this value
    if n in memo:
        return memo[n]  # Return cached result - O(1) lookup!
    
    # Base cases
    if n <= 2:
        return n
    
    # Calculate once and store in memo for future use
    memo[n] = climb_stairs_memo(n-1, memo) + climb_stairs_memo(n-2, memo)
    return memo[n]
```

- **表格法（自底向上）**：从底层开始构建解决方案，填充子问题的解表。

```py
def climb_stairs_tabulation(n):
    """Dynamic programming with tabulation"""
    if n <= 2:
        return n
    
    # Create array to store results for all steps from 0 to n
    dp = [0] * (n + 1)
    dp[1] = 1  # 1 way to reach step 1
    dp[2] = 2  # 2 ways to reach step 2
    
    # Build up the solution iteratively
    for i in range(3, n + 1):
        # Ways to reach step i = ways to reach (i-1) + ways to reach (i-2)
        dp[i] = dp[i-1] + dp[i-2]
    
    return dp[n]
```

## 使用动态规划的实际应用

- **路径优化**：GPS 系统使用动态规划算法来查找位置之间的最短路径。
- **文本处理**：拼写检查和自动完成功能通常依赖动态规划来计算单词之间的编辑距离。
- **金融建模**：投资策略和投资组合优化经常采用动态规划技术。
- **资源分配**：背包问题及其变体出现在调度、预算和资源管理中。

## 何时使用动态规划

你应该在以下情况下考虑使用动态规划：

- 问题可以分解为重叠的子问题。
- 问题具有最优子结构。
- 朴素的递归解法会涉及重复计算。
- 你需要以空间复杂度为代价来优化时间复杂度。
