# --description--

$2^N$ 個二進位數字可以排列成一個圓圈，使得所有長度為 $N$ 的順時針子序列皆不相同。

對於 $N = 3$，忽略旋轉，有兩種此類圓形排列是可能的：

<img alt="two circular arrangements for N = 3" src="https://cdn.freecodecamp.org/curriculum/project-euler/binary-circles.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

對於第一種排列，順時針方向的 3 位數子序列依序為：000、001、010、101、011、111、110 和 100。

每個圓形排列都可以透過將二進位數字串接起來編碼為一個數字，起始於所有零的子序列作為最高有效位元，並順時針方向進行。對於 $N = 3$，這兩個排列因此被表述為 23 和 29：

$${00010111}_2 = 23\\\\
{00011101}_2 = 29$$

將 $S(N)$ 定義為唯一數字表現的總和，我們可以看到 $S(3) = 23 + 29 = 52$。

尋找 $S(5)$。
