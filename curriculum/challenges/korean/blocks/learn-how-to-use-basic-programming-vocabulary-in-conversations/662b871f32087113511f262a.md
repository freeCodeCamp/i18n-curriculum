# --description--

동사의 `be` 다음에 `-ing` 형태가 오면 현재 진행 중인 동작에 대해 말할 때 사용합니다.

`have trouble with` 구문은 특정 문제나 대상에 대해 어려움을 표현할 때 씁니다. 예를 들어, `She is having trouble with her computer`는 그녀가 컴퓨터를 사용하는 데 어려움을 겪고 있다는 뜻입니다.
