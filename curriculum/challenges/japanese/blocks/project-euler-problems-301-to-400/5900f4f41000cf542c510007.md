# --description--

直交格子は、格子線間の間隔が等間隔である必要がない直交グリッドです。

そのようなグリッドの例として対数グラフ用紙があります。

次の性質を持つデカルト座標系の直交格子を考えます：

- 格子線はデカルト座標系の軸に平行です。
- 垂直方向に $N + 2$ 本、水平に $N + 2$ 本の格子線があります。したがって、長方形のセルは $(N + 1) \times (N + 1)$ 個あります。
- 外側の2本の垂直格子線の方程式は $x = -1$ と $x = 1$ です。
- 外側の2本の水平格子線の方程式は $y = -1$ と $y = 1$ です。
- 格子セルは単位円と重なる場合は赤色、それ以外は黒色で塗られます。

この問題では、赤色のセルが占める面積を最小化するように、残りの $N$ 本の内側の水平格子線と $N$ 本の内側の垂直格子線の位置を求めてください。

例えば、$N = 10$ の解の図は以下の通りです：

<img alt="solution for N = 10" src="https://cdn.freecodecamp.org/curriculum/project-euler/enmeshed-unit-circle.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$N = 10$ のとき、赤色セルが占める面積は小数点以下10桁で四捨五入して 3.3469640797 です。

$N = 400$ の場合の位置を求めてください。答えとして、赤色セルが占める面積を小数点以下10桁で四捨五入した値を提出してください。
