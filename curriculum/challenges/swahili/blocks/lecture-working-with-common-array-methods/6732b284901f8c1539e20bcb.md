# --interactive--

Njia ya `splice()` katika JavaScript ni njia yenye nguvu ya kubadilisha safu za data. Inakuwezesha kuongeza au kufuta vipengele kutoka sehemu yoyote katika safu ya data, ikiwa ni pamoja na katikati. Thamani ya return ya njia ya `splice()` itakuwa safu ya vipengele vilivyofutwa kutoka kwenye safu ya data. Ikiwa hakuna kilichofutwa, basi safu tupu itarudishwa.

Ni muhimu kuelewa kuwa njia hii itabadilisha safu ya data ya awali, ikibadilisha mahali hapo badala ya kuunda safu mpya. Hii ni jambo la kuzingatia unapotumia `splice()`. Hapa kuna sintaksia ya msingi:

```js
array.splice(startIndex, itemsToRemove, item1, item2)
```

`startIndex` inaelezea kielezo ambacho kuanza kubadilisha safu ya data, wakati `itemsToRemove` ni kigezo cha hiari kinachoonyesha ni vipengele vingapi vifutwe. Ikiwa `itemsToRemove` haitajwi, `splice()` itafuta vipengele vyote kuanzia `startIndex` hadi mwisho wa safu ya data. Vigezo vinavyofuata (`item1`, `item2`, na kadhalika) ni vipengele vya kuongezwa kwenye safu ya data, kuanzia kielezo cha kuanza.

Tuanze na mfano wa kufuta vipengele katikati ya safu ya data:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango", "kiwi"];
let removed = fruits.splice(2, 2);

console.log(fruits);  // ["apple", "banana", "kiwi"]
console.log(removed); // ["orange", "mango"]
```

:::

Katika mfano huu, `splice(2, 2)` inaanza kwenye kielezo cha `2` na inafuta vipengele `2`. Safu ya data iliyobadilishwa sasa itakuwa na `apple`, `banana`, na `kiwi` tu. Sasa, tuchunguze jinsi ya kuongeza vipengele katikati ya safu ya data:

:::interactive_editor

```js
let colors = ["red", "green", "blue"];
colors.splice(1, 0, "yellow", "purple");

console.log(colors); // ["red", "yellow", "purple", "green", "blue"]
```

:::

Hapa, `splice(1, 0, "yellow", "purple")` inaanza kwenye kielezo cha `1`, inafuta vipengele `0`, na inaingiza `yellow` na `purple`. Kigezo cha pili (`0`) kinamaanisha hakuna vipengele vinavyofutwa kabla ya kuingiza. Unaweza pia kutumia `splice()` kufuta na kuongeza vipengele kwa wakati mmoja:

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];
numbers.splice(1, 2, 6, 7, 8);

console.log(numbers); // [1, 6, 7, 8, 4, 5]
```

:::

Katika kesi hii, `splice(1, 2, 6, 7, 8)` inaanza kwenye kielezo cha `1`, inafuta vipengele `2` (`2` na `3`), na inaingiza `6`, `7`, na `8`. Ikiwa unahitaji kuweka safu ya data ya awali isibadilike, unapaswa kuunda nakala kabla ya kutumia `splice()`:

:::interactive_editor

```js
let original = [1, 2, 3, 4, 5];
let copy = [...original];
copy.splice(2, 1, 6);

console.log(original); // [1, 2, 3, 4, 5]
console.log(copy);     // [1, 2, 6, 4, 5]
```

:::

Katika mfano huu, kuunda nakala ya safu ya data ya `original` bila kuibadilisha, tunatumia opereta ya spread (`...`). Opereta ya spread itaunda nakala ya kina ya vipengele vya safu ya data ya `original` katika safu mpya. Utajifunza zaidi kuhusu hili katika mafundisho yajayo.

Tunapotumia `copy.splice(2, 1, 6)`, inabadilisha safu ya nakala kwa kufuta kipengele kwenye kielezo cha `2` (ambacho ni `3`) na kuingiza kipengele kipya `6` mahali hapo.

Matumizi moja ya kawaida ya `splice()` ni kufuta kipengele kimoja kutoka kwenye safu ya data unapoijua kielezo chake:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango"];
let indexToRemove = fruits.indexOf("orange");
if (indexToRemove !== -1) {
    fruits.splice(indexToRemove, 1);
}

console.log(fruits); // ["apple", "banana", "mango"]
```

:::

Katika mfano huu, kwanza tunatumia njia ya `indexOf()` kupata kielezo cha kipengele cha `orange` katika safu ya `fruits`. Njia ya `indexOf()` inarudisha kielezo cha mara ya kwanza kipengele hicho kinapotokea au `-1` ikiwa kipengele hakipatikani katika safu ya data.

Kisha tunalinganisha `indexToRemove` na `-1` kuhakikisha kuwa kipengele kipo katika safu ya data kabla ya kujaribu kufuta. Ikiwa `indexToRemove` si sawa na `-1` (maana kipengele kimepatikana), tunatumia `splice()` kufuta kipengele kimoja kuanzia nafasi ya `indexToRemove`.

Unaweza pia kutumia `splice()` kufuta safu ya data kwa kufuta vipengele vyote:

:::interactive_editor

```js
let array = [1, 2, 3, 4, 5];
array.splice(0);

console.log(array); // []
```

:::

Ingawa `splice()` ni yenye nguvu, ni vyema kutambua kuwa kwa safu kubwa sana, inaweza kuwa na ufanisi mdogo ikilinganishwa na njia nyingine, hasa unapotaka kubadilisha mwanzo wa safu ya data. Hii ni kwa sababu `splice()` inaweza kuhitaji kusogeza vipengele vyote vinavyofuata. Katika hali kama hizo, ikiwa unataka tu kuongeza au kufuta vipengele mwishoni mwa safu ya data, njia kama `push()`, `pop()`, `unshift()`, na `shift()` zinaweza kuwa bora zaidi.

Kwa kumalizia, njia ya `splice()` ni njia yenye matumizi mengi ya kubadilisha safu za data katika JavaScript. Inakuwezesha kudhibiti kwa usahihi kuongeza na kufuta vipengele kutoka sehemu yoyote ya safu ya data. Kuelewa jinsi ya kutumia `splice()` kwa ufanisi kunaweza kuongeza uwezo wako wa kushughulikia safu za data katika msimbo wako wa JavaScript.
