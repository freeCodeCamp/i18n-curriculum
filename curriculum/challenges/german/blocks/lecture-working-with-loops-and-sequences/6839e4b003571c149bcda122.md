# --description--

In vorherigen Lektionen haben Sie gelernt, wie man mit der `for`-Schleife arbeitet, die verwendet wird, um einen Codeblock eine festgelegte Anzahl von Malen zu wiederholen. Hier ist ein Beispiel für die Verwendung einer `for`-Schleife, um jede Sprache aus der `languages`-Liste in der Konsole auszugeben:

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for language in languages:
    print(language)
```

Aber was, wenn Sie den Index für jedes Element verfolgen möchten? Eine Möglichkeit ist, eine `index`-Variable zu erstellen und sie bei jeder Iteration der Schleife um `1` zu erhöhen, so wie hier:

```py
languages = ['Spanish', 'English', 'Russian', 'Chinese']

index = 0

for language in languages:
    print(f'Index {index} and language {language}')
    index += 1
```

Das funktioniert zwar, aber eine einfachere Methode ist die Verwendung der `enumerate()`-Funktion. Die `enumerate()`-Funktion verfolgt den Index für ein iterierbares Objekt und gibt ein enumerate-Objekt zurück.

Wenn wir die `languages`-Liste an die `enumerate()`-Funktion übergeben und deren Rückgabewert mit der `list()`-Funktion in eine Liste umwandeln, sieht das so aus:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

list(enumerate(languages))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

Jeder Eintrag im enumerate-Objekt (jetzt eine Liste) ist ein Tuple, das eine Zählung gefolgt von einem Wert aus dem an die `enumerate()`-Funktion übergebenen iterierbaren Objekt enthält.

Nun wollen wir das vorherige Beispiel umgestalten, um die `enumerate()`-Funktion zu verwenden:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')
```

Wir entpacken die Zählung und den Wert für jedes Tuple im enumerate-Objekt in Variablen mit den Namen `index` und `language`. Schließlich werden beide Variablen in einem f-String verwendet, der in jeder Iteration der Schleife in der Konsole ausgegeben wird.

```md
Index 0 and language Spanish
Index 1 and language English
Index 2 and language Russian
Index 3 and language Chinese
```

Dadurch entfällt die Notwendigkeit, manuell eine `index`-Variable zu erstellen und zu aktualisieren.

Die `enumerate()`-Funktion akzeptiert auch ein optionales `start`-Argument, das den Startwert für die Zählung angibt. Wenn dieses Argument weggelassen wird, beginnt die Zählung bei `0`. Hier ist ein Beispiel für die Verwendung des optionalen `start`-Arguments:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages, 1):
    print(f'Index {index} and language {language}')
```

Und so sieht das Ergebnis in der Konsole aus:

```md
Index 1 and language Spanish
Index 2 and language English
Index 3 and language Russian
Index 4 and language Chinese
```

Bisher haben wir nur eine Liste durchlaufen. Aber was, wenn Sie mehrere iterierbare Objekte parallel durchlaufen müssen? Dafür können Sie die `zip()`-Funktion verwenden, die Listen zu Elementpaaren kombiniert und einen Iterator von Tuples zurückgibt.

Wenn wir eine Liste von `developers` und `ids` an die `zip()`-Funktion übergeben und deren Rückgabewert mit der `list()`-Funktion in eine Liste umwandeln, sieht das so aus:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

list(zip(developers, ids))
# [('Naomi', 1), ('Dario', 2), ('Jessica', 3), ('Tom', 4)]
```

Und hier ist ein Beispiel für die Verwendung der `zip()`-Funktion mit einer `for`-Schleife, um über `developers` und `ids` zu iterieren:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')
```

In diesem Beispiel kombiniert `zip()` die beiden Listen zu Elementpaaren und gibt einen Iterator von Tuples zurück. Die `for`-Schleife entpackt dann jedes Tuple in `name` und `id`. Schließlich geben wir in jeder print-Anweisung jeweils `name` und `id` aus den `ids`- und `developers`-Listen aus. So sieht das Ergebnis in der Konsole aus:

```md
Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
```

Die `enumerate()`- und `zip()`-Funktionen sind sehr mächtig und können in Kombination mit Schleifen Ihren Code deutlich kompakter machen.
