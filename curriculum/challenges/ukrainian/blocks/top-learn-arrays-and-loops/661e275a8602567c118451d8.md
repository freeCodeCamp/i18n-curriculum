# --description--

Ще одним корисним методом масиву є метод `filter()`. Метод `filter()` створює новий масив зі всіма елементами, які проходять перевірку, реалізовану наданою функцією. Синтаксис методу `filter()` виглядає так:

```javascript
const numbers = [2, 5, 6, 1, 9, -1]

const newNumbers = numbers.filter((number) => {
  return number > 2;
});

console.log(newNumbers); // Output: [5, 6, 9]
```

Метод `filter()` створює новий масив зі всіма елементами, які більші за 2. Метод `filter()` не змінює оригінальний масив.
