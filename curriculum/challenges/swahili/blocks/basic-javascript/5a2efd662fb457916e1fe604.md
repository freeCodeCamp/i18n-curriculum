# --description--

Aina inayofuata ya mzunguko utakayojifunza huitwa mzunguko wa `do...while`. Unaitwa mzunguko wa `do...while` kwa sababu kwanza utarudia `do` moja ya msimbo ulioko ndani ya mzunguko bila kujali hali yoyote, kisha utaendelea kuendesha mzunguko `while` hali iliyobainishwa itakapothibitishwa kuwa `true`.

```js
const ourArray = [];
let i = 0;

do {
  ourArray.push(i);
  i++;
} while (i < 5);
```

Mfano uliotolewa hapo juu hufanya kazi kama mizunguko mingine, na safu ya data itakuwa kama `[0, 1, 2, 3, 4]`. Hata hivyo, kinachofanya mzunguko wa `do...while` kuwa tofauti na mingine ni jinsi unavyotenda wakati hali inashindwa kwenye ukaguzi wa kwanza. Tuchunguze hili kwa vitendo. Hapa kuna mzunguko wa kawaida wa `while` ambao utaendesha msimbo ndani ya mzunguko mradi tu `i < 5`:

```js
const ourArray = []; 
let i = 5;

while (i < 5) {
  ourArray.push(i);
  i++;
}
```

Katika mfano huu, tunaanzisha thamani ya `ourArray` kuwa safu ya data tupu na thamani ya `i` kuwa 5. Tunapotekeleza mzunguko wa `while`, hali inathibitishwa kuwa `false` kwa sababu `i` si ndogo kuliko 5, hivyo hatutekelezi msimbo ulioko ndani ya mzunguko. Matokeo yake ni kwamba `ourArray` haitakuwa na thamani yoyote iliyoongezwa, na bado itaonekana kama `[]` baada ya msimbo wote katika mfano huu kukamilika kuendesha. Sasa, tazama mzunguko wa `do...while`:

```js
const ourArray = []; 
let i = 5;

do {
  ourArray.push(i);
  i++;
} while (i < 5);
```

Katika kesi hii, tunaanzisha thamani ya `i` kuwa 5, kama tulivyofanya kwenye mzunguko wa `while`. Tunapofika kwenye mstari unaofuata, hakuna hali ya kuthibitisha, hivyo tunaenda kwenye msimbo ulioko ndani ya mabano ya mviringo na kuutekeleza. Tutaongeza kipengele kimoja kwenye safu ya data kisha kuongeza `i` kabla ya kufikia ukaguzi wa hali. Tunapothibitisha hali ya `i < 5` kwenye mstari wa mwisho, tunaona kuwa `i` sasa ni 6, ambayo inashindwa kwenye ukaguzi wa hali, hivyo tunaondoka kwenye mzunguko na kumaliza. Mwishoni mwa mfano huu, thamani ya `ourArray` ni `[5]`. Kwa msingi huo, mzunguko wa `do...while` unahakikisha kuwa msimbo ulioko ndani ya mzunguko utaendeshwa angalau mara moja. Hebu jaribu kufanya mzunguko wa `do...while` ufanye kazi kwa kusukuma thamani kwenye safu ya data.
