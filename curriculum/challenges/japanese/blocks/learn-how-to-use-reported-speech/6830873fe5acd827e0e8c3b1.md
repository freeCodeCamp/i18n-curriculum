# --explanation--

`To seem` は、私たちが知っていることに基づいて何かが真実のように見えるが、完全には確信が持てない場合に使われます。例えば：

`It seems the users liked the new menu.` – これは、ユーザーがフィードバックや行動に基づいてそれを気に入ったと思うが、確信は持てないことを意味します。

`To be found to be` は、誰かが何かを見つけたり、発見したり、結論づけたりしたときに使われる**受動態**の構文で、判断した人ではなく結果に焦点が当たっています。例えば：

`The layout was found to be confusing by some testers.` – これは、テスターがそれを混乱させると考えたことを意味しますが、文は発見されたことに重点を置いており、人（この場合はテスター）には焦点を当てていません。
