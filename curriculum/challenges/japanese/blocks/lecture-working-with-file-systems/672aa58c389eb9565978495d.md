# --description--

ファイル管理アプリケーションは、コンピューター上のファイルを簡単に保存、整理、編集する方法です。MacでもWindowsでも、ファイルを整理するために使える組み込みのアプリケーションがあります。

Windowsコンピューターの場合、デフォルトのファイルマネージャーはFile Explorerです。これはファイルやフォルダーの閲覧、検索、管理に使われ、コピー、移動、削除などのファイル操作も行えます。

File Explorerを見つけるには、スタートメニューに行くか、キーボードのWindowsロゴキーを押してください。

フォルダーをピン留めするには、マウスで右クリックして「ピン留め」を選択します。ファイルやフォルダーを移動するには、まずそれを選択し、「切り取り」を選んでから、新しい場所に移動して貼り付けます。

Macコンピューターの場合、デフォルトのファイルマネージャーはFinderです。これはローカルおよびリモートのファイルにアクセスでき、ファイルのプレビューをサポートし、タグやスマートフォルダーなどのさまざまな整理機能を提供します。

MacでFinderにアクセスするには、画面下部のDockに行き、Finderアイコンをクリックしてください。

左側のサイドバーにはお気に入りのフォルダーが表示されます。

ファイルを整理する方法の一つはタグを使うことです。ファイルやフォルダーにタグを付けるには、Finderでアイテムを選択し、マウスで右クリックします。ドロップダウンに「タグ...」のオプションが表示され、カスタムカラーのタグを付けられます。

タグ付きのアイテムを検索するには、サイドバーの下部までスクロールして対応する色のタグをクリックするか、検索フィールドで色名を検索してください。

もう一つの整理方法はスマートフォルダーを使うことです。スマートフォルダーは、ファイルの種類、日付、キーワードなど、設定した条件に基づいて自動的に整理されるファイルのコレクションです。

WindowsのFile ExplorerでもMacのFinderでも、これらのアプリケーションはタグ、検索、スマートフォルダーなどの機能でファイル管理の柔軟性を提供します。
