# --description--

次の条件を満たすすべての三角形を考えます：

- すべての頂点が格子点にある。
- 外心が原点 O にある。
- 垂心が点 H(5, 0) にある。

$\text{周囲の長さ} ≤ 50$ のそのような三角形は9つあります。

これらの三角形の周囲の長さの合計は、小数点以下4桁に丸めて291.0089です。

$\text{周囲の長さ} ≤ {10}^5$ のすべてのそのような三角形を見つけてください。答えとして、それらの周囲の長さの合計を小数点以下4桁に丸めた値を入力してください。
