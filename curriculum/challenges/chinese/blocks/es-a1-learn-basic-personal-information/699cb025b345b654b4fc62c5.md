# --description--

当提供姓名或电子邮件 `address` 时，为避免混淆，可能需要逐个字母地说出每个字母。这个动作称为 `deletrear`。当有人逐字母拼写一个单词时，使用动词 `deletrear`。例如：

- `Deletreo mi apellido.`  - 我拼写我的姓氏。

- `¿Quieres que deletree tu nombre？` - 你想让我拼写你的名字吗？
