# --description--

Markdownで太字のテキストが含まれている可能性がある文字列が与えられたとき、対応するHTML文字列を返してください。

- Markdownの太字テキストは、2つのアスタリスク（`**`）または2つのアンダースコア（`__`）で始まり終わるテキストです。
- テキストとアスタリスクまたはアンダースコアの間にスペースは入れられませんが、テキスト内にはスペースがあってもかまいません。
- すべての太字の部分をHTMLの`b`タグに変換して文字列を返してください。

例えば、`"**This is bold**"`が与えられたら、`"<b>This is bold</b>"`を返します。

注意：コンソールは文字列内のHTMLタグをログに表示しない場合があります。タグを含むログを見るにはブラウザのコンソールを確認してください。

# --hints--

`parseBold("**This is bold**")`は`"<b>This is bold</b>"`を返すべきです。

```js
assert.equal(parseBold("**This is bold**"), "<b>This is bold</b>");
```

`parseBold("__This is also bold__")`は`"<b>This is also bold</b>"`を返すべきです。

```js
assert.equal(parseBold("__This is also bold__"), "<b>This is also bold</b>");
```

`parseBold("**This is not bold **")`は`"**This is not bold **"`を返すべきです。

```js
assert.equal(parseBold("**This is not bold **"), "**This is not bold **");
```

`parseBold("__ This is also not bold__")`は`"__ This is also not bold__"`を返すべきです。

```js
assert.equal(parseBold("__ This is also not bold__"), "__ This is also not bold__");
```

`parseBold("The **quick** brown fox __jumps__ over the **lazy** dog.")`は`"The <b>quick</b> brown fox <b>jumps</b> over the <b>lazy</b> dog."`を返すべきです。

```js
assert.equal(parseBold("The **quick** brown fox __jumps__ over the **lazy** dog."), "The <b>quick</b> brown fox <b>jumps</b> over the <b>lazy</b> dog.");
```

# --solutions--

```js
function parseBold(markdown) {
  markdown = markdown.replace(/\*\*(\S(?:.*?\S)?)\*\*/g, "<b>$1</b>");
  markdown = markdown.replace(/__(\S(?:.*?\S)?)__/g, "<b>$1</b>");

  return markdown;
}
```
