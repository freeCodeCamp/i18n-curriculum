# --interactive--

In JavaScript ist die `indexOf()`-Methode nützlich, um den ersten Index eines bestimmten Elements innerhalb eines Arrays zu finden. Wenn das Element nicht gefunden werden kann, gibt sie `-1` zurück. Hier ist die grundlegende Syntax:

```js
array.indexOf(element, fromIndex)
```

`element` steht für den Wert, den Sie im Array suchen möchten, und der `fromIndex`-Parameter ist die Position, ab der die Suche beginnen soll. Der `fromIndex`-Parameter ist optional. Wenn `fromIndex` nicht angegeben wird, beginnt die Suche am Anfang des Arrays. Schauen wir uns ein Beispiel an:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "banana"];
let index = fruits.indexOf("banana");
console.log(index); // 1
```

:::

In diesem Beispiel haben wir ein Array `fruits`, das verschiedene Obstnamen enthält. Wir verwenden die `indexOf()`-Methode, um den Index des Strings `banana` im `fruits`-Array zu finden. Da `banana` an Index `1` vorhanden ist, gibt die Methode `1` zurück, das in der Variablen `index` gespeichert und in der Konsole ausgegeben wird.

Wenn das Element, nach dem Sie suchen, nicht im Array gefunden wird, gibt `indexOf()` `-1` zurück. Zum Beispiel:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let index = fruits.indexOf("grape");
console.log(index); // -1
```

:::

Hier suchen wir mit `grape` nach dem String `indexOf()` im fruits-Array. Da `grape` nicht im Array vorhanden ist, gibt die Methode `-1` zurück, das in der Variablen `index` gespeichert und in der Konsole ausgegeben wird.

Wenn Sie nach einem Element erst ab einer bestimmten Indexnummer suchen möchten, können Sie ein zweites Argument übergeben, wie in diesem Beispiel:

:::interactive_editor

```js
let colors = ["red", "green", "blue", "yellow", "green"];
let index = colors.indexOf("green", 3);
console.log(index); // 4
```

:::

In diesem Beispiel beginnt die Suche nicht am Anfang des Arrays, sondern ab der Indexnummer `3`, die `yellow` ist, und liefert die Ausgabe `4`.
