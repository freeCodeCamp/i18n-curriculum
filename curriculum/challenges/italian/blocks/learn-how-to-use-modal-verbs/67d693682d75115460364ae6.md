# --explanation--

Un `critical bug` è un errore grave nel software che influisce sulla funzionalità e deve essere risolto immediatamente. Per esempio:

- `We need to fix the critical bugs before launching the app.` - I problemi più seri devono essere risolti prima della distribuzione.

- `A critical bug caused the website to crash.` - Un errore grave ha fatto smettere di funzionare il sito web.
