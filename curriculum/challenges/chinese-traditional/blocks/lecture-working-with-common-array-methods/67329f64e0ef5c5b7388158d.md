# --interactive--

在 JavaScript 中，`indexOf()` 方法對於尋找陣列中特定元素的第一個索引非常有用。如果找不到該元素，則會傳回 `-1`。以下是基本語法：

```js
array.indexOf(element, fromIndex)
```

`element` 表述你想在陣列中搜尋的值，而 `fromIndex` 參數是搜尋應該開始的位置。`fromIndex` 參數是選用的。如果沒有提供 `fromIndex`，搜尋將從陣列的開頭開始。讓我們來看一個範例：

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "banana"];
let index = fruits.indexOf("banana");
console.log(index); // 1
```

:::

在此範例中，我們有一個包含各種水果名稱的陣列 `fruits`。我們使用 `indexOf()` 方法來尋找字串 `banana` 在 `fruits` 陣列中的索引。由於 `banana` 出現在索引 1，該方法傳回 `1`，並將其儲存在變數 `index` 中，然後輸出到主控台。

如果你搜尋的元素在陣列中找不到，`indexOf()` 會傳回 `-1`。例如：

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let index = fruits.indexOf("grape");
console.log(index); // -1
```

:::

在這裡，我們使用 `indexOf()` 在水果陣列中搜尋字串 `grape`。由於陣列中沒有 `grape`，該方法傳回 `-1`，並將其儲存在 `index` 變數中，然後輸出到主控台。

如果你想在特定索引號之後開始尋找一個項目，則你可以像這個範例中一樣傳遞第二個引數： 

:::interactive_editor

```js
let colors = ["red", "green", "blue", "yellow", "green"];
let index = colors.indexOf("green", 3);
console.log(index); // 4
```

:::

在此範例中，搜尋並非從陣列的起始位置開始，而是從索引號碼 `3`（即 `yellow`）開始，並取得輸出為 `4`。
