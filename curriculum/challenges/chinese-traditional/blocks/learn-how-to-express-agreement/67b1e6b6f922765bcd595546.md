# --explanation--

`Make sure` 意味著採取行動以確認某事是正確的或已完成的。 例如：  

`Please make sure the report is submitted before the deadline。`－這表示要檢查並確保報告準時繳交。  

`Reliable` 意味著某物或某人可以被信賴在一段時間內表現良好或保持一致。例如：

`The server is reliable; it has been running without any issues for months。`－這表示該伺服器穩定運作且不會造成問題。

「`More reliable`」表示某事變得更加值得信賴或可靠。例如：

`The new software update makes the app more reliable。`－這表示在更新後，該應用程式變得更值得信賴或更穩定。
