# --explanation--

`Guatemalteco` はグアテマラ出身の人の国籍の男性形です。

国籍は、その人の性別に合わせる必要があります。

| 性別 | 国籍 | 例文 |
| :--- | :--- | :--- |
| **女性形** | `Guatemalteca` | `Ella es guatemalteca.`（「彼女はグアテマラ人です。」） |
| **男性形** | `Guatemalteco` | `Él es guatemalteco.`（「彼はグアテマラ人です。」） |
