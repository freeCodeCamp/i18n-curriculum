# --description--

Katika React, kusahihisha safu za data katika state ni rahisi sana, lakini inaweza kuwa rahisi kufanya kosa, hasa ikiwa unatoka kwenye JavaScript ya kawaida ambapo unaweza kubadilisha safu moja kwa moja.

Katika React, state hutendewa kama isiyobadilika ili iweze kutambua mabadiliko na kufanya masasisho sahihi kwenye kiolesura cha mtumiaji.

Tuchunguze jinsi unavyoweza kusahihisha safu za data zilizo katika state katika React.

Moja ya makosa yanayojirudia mara nyingi wakati wa kusahihisha safu za data katika state ya React ni kubadilisha safu moja kwa moja kwa kutumia njia kama `push()`, `pop()`, au `splice()`. Njia hizi hubadilisha safu ya awali, na React haikubali hilo.

React hutegemea rejea mpya ya safu ili kugundua mabadiliko, hivyo kubadilisha safu moja kwa moja kunaweza kuzuia sehemu kuonyesha tena kama inavyotarajiwa.

Hapa kuna mfano wa kutumia njia ya `push()` kuongeza kwenye safu katika state, ambayo haitafanya kazi:

```jsx
import { useState } from "react";

function ItemsList() {
  const [items, setItems] = useState([
    { id: 0, name: "Item 1" },
    { id: 1, name: "Item 2" },
    { id: 2, name: "Item 3" },
  ]);

  const addItem = () => {
    const newItem = { id: items.length + 1, name: `Item ${items.length + 1}` };
    items.push(newItem); // This modifies the original array
    setItems(items); // React will not detect this change
  };

  return (
    <div>
      <button onClick={addItem}>Add Item</button>
      <ul>
        {items.map((item) => (
          <li key={item.id}>{item.name}</li>
        ))}
      </ul>
    </div>
  );
}

export default ItemsList;
```

Ukibofya kitufe cha `Add Item`, hakuna kinachotokea kwenye kiolesura cha mtumiaji.

Pia inaweza kuvutia kuondoa vitu kutoka kwenye safu kwa kutumia njia ya `pop()`:

```js
const removeItem = () => {
  items.pop(); // Modifies the original array
  setItems(items); // React will not detect this change, either
};
```

Ili kusahihisha safu katika state, ufunguo ni kuunda safu mpya, fanya shughuli zako, na upitishe hiyo kwa React, badala ya kubadilisha safu iliyopo.

Kwa kuwa ni safu mpya, React itajua kuwa state imebadilika, na kusababisha sehemu ionyeshwe tena.

Hivi ndivyo unavyoweza kuongeza kwenye safu ya `items` kwa kutumia opereta ya spread:

```js
const addItem = () => {
  const newItem = {
    id: items.length + 1,
    name: `Item ${items.length + 1}`,
  };

  // Creates a new array
  setItems((prevItems) => [...prevItems, newItem]);
};
```

`[...prevItems, newItem]` huunda safu mpya kwa kunakili vitu vyote vilivyo kwenye safu ya `items` iliyoko katika state, kisha huongeza `newItem` mwishoni, ambayo huongeza `id` na nambari ya kipengee.

Ikiwa unataka kuondoa kitu kutoka kwenye safu, unaweza kutumia njia ya `filter()`, ambayo hurudisha safu mpya baada ya kuchuja kile unachotaka kuondoa:

```js
const removeItem = (id) => {
  setItems((prevItems) => prevItems.filter((item) => item.id !== id));
};
```

Hapa kuna msimbo kamili:

```jsx
import { useState } from "react";

function ItemsList() {
  const [items, setItems] = useState([
    { id: 0, name: "Item 1" },
    { id: 1, name: "Item 2" },
    { id: 2, name: "Item 3" },
  ]);

  const addItem = () => {
    const newItem = { id: items.length + 1, name: `Item ${items.length + 1}` };
    setItems((prevItems) => [...prevItems, newItem]); // Creates a new array
  };

  const removeItem = (id) => {
    setItems((prevItems) => prevItems.filter((item) => item.id !== id)); // Creates a new array
  };

  return (
    <div>
      <button onClick={addItem}>Add Item</button>
      <ul>
        {items.map((item) => (
          <li key={item.id}>
            {item.name}{" "}
            <button onClick={() => removeItem(item.id)}>Remove</button>
          </li>
        ))}
      </ul>
    </div>
  );
}

export default ItemsList;
```

Hizo ni baadhi ya njia za kawaida unazoweza kusahihisha safu katika state.
