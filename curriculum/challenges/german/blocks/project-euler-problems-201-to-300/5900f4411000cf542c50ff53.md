# --description--

Ein achsenparalleler Quader, angegeben durch die Parameter $\{ (x_0,y_0,z_0), (dx,dy,dz) \}$, besteht aus allen Punkten ($X$,$Y$,$Z$), für die gilt: $x_0 ≤ X ≤ x_0 + dx$, $y_0 ≤ Y ≤ y_0 + dy$ und $z_0 ≤ Z ≤ z_0 + dz$. Das Volumen des Quaders ist das Produkt $dx × dy × dz$. Das kombinierte Volumen einer Sammlung von Quadern ist das Volumen ihrer Vereinigung und ist kleiner als die Summe der Einzelvolumen, falls sich Quader überlappen.

Seien $C_1, \ldots, C_{50000}$ eine Sammlung von 50000 achsenparallelen Quadern, so dass $C_n$ die Parameter

$$\begin{align}
  & x_0 = S_{6n - 5} \\; \text{Modulo} \\; 10000    \\\\
  & y_0 = S_{6n - 4} \\; \text{Modulo} \\; 10000    \\\\
  & z_0 = S_{6n - 3} \\; \text{Modulo} \\; 10000    \\\\
  & dx = 1 + (S_{6n - 2} \\; \text{Modulo} \\; 399) \\\\
  & dy = 1 + (S_{6n - 1} \\; \text{Modulo} \\; 399) \\\\
  & dz = 1 + (S_{6n} \\; \text{Modulo} \\; 399)     \\\\
\end{align}$$

haben,

wobei $S_1, \ldots, S_{300000}$ aus dem „Lagged Fibonacci Generator“ stammen:

Für $1 ≤ k ≤ 55$ gilt $S_k = [100003 - 200003k + 300007k^3] \\; (Modulo \\; 1000000)$

Für $56 ≤ k$ gilt $S_k = [S_{k - 24} + S_{k - 55}] \\; (Modulo \\; 1000000)$

Somit hat $C_1$ die Parameter $\{(7,53,183), (94,369,56)\}$, $C_2$ die Parameter $\{(2383,3563,5079), (42,212,344)\}$ und so weiter.

Das kombinierte Volumen der ersten 100 Quader, $C_1, \ldots, C_{100}$, beträgt 723581599.

Wie groß ist das kombinierte Volumen aller 50000 Quader, $C_1, \ldots, C_{50000}$?
