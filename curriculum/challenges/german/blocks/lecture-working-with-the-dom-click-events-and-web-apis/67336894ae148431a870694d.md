# --interactive--

Lernen wir das DOM kennen und warum es so wichtig für die Webentwicklung ist. DOM steht für Document Object Model. Es ist eine Programmierschnittstelle, die es uns ermöglicht, mit HTML-Dokumenten zu interagieren.

Mit dem DOM können Sie Elemente auf einer Webseite hinzufügen, ändern oder löschen. Sie können Ihre Webseite sogar interaktiv machen, indem Sie Elemente Ereignisse hören und darauf reagieren lassen.

Im DOM wird ein HTML-Dokument als Baum von Knoten dargestellt. Jeder Elementknoten stellt ein HTML-Element aus dem HTML-Dokument dar:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>DOM Example</title>
  </head>
  <body>
    <h1>What is the DOM?</h1>
    <h2>Let's learn about the DOM</h2>
  </body>
</html>
```

Dies ist ein Diagramm, das die grundlegende DOM-Struktur unseres Beispiels darstellt:

```md
Document
========

HTML
-----

- Head
  - Title
- Body
  - H1
  - H2   
```

Natürlich kann dies je nach Struktur der HTML-Auszeichnung einer Webseite detaillierter und komplexer sein.

Das HTML-Dokument ist der Wurzelknoten in der DOM-Hierarchie. Es hat einen Kindknoten, das `html`-Element. Dies ist das Wurzelelement des HTML-Dokuments, da alle anderen Knoten von ihm abstammen.

Das `html`-Element hat zwei Kinder: `head` und `body`.

Das `head`-Element enthält Metadaten über das Dokument. Es liefert wichtige Informationen über die Webseite.

Das `body`-Element enthält den sichtbaren Inhalt der Webseite.

Sie können mit JavaScript auf diese Elemente zugreifen, mit ihnen in Ihrem Code arbeiten und sie sogar dynamisch ändern. Das ist die Stärke des DOM und der Web-APIs.

Um mit JavaScript auf diese Elemente zuzugreifen, können Sie `getElementById()`- und `querySelector()`-Methoden verwenden. Diese Methoden sind Web-APIs, weil sie standardisierte Möglichkeiten bieten, mit dem DOM über JavaScript zu interagieren.

Mit `getElementById()` können Sie ein Objekt erhalten, das das HTML-Element mit dem angegebenen `id` darstellt. Denken Sie daran, dass `id` in jedem HTML-Dokument einzigartig sein muss, daher gibt diese Methode nur ein `Element`-Objekt zurück. Hier sehen Sie ein Beispiel:

```js
const container = document.getElementById("container");
```

Diese JavaScript-Zeile holt ein Element mit dem `id`-Wert `container` und weist dieses Objekt einer JavaScript-Konstanten zu. Sie müssen das `id` als Argument in Anführungszeichen übergeben. Wenn Sie dieses Objekt in der Konsole ausgeben, sehen Sie es in der Ausgabe:

:::interactive_editor

```html
<div id="container">
  <h1>Hello, World!</h1>
  <p>Welcome to learning about the DOM.</p>
</div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
console.log(container); 
```

:::

`querySelector()` ist umfassender als `getElementById()`. Mit `querySelector()` können Sie das erste Element im HTML-Dokument erhalten, das dem als Argument übergebenen CSS-Selektor entspricht.

In diesem Beispiel erhalten Sie das Element, das dem `section`-Selektor entspricht:

:::interactive_editor

```html
<section>
  <h2>Section Title</h2>
  <p>This is a section of the webpage.</p>
</section>
<script src="index.js"></script>  
```

```js
const sectionEl = document.querySelector("section");
console.log(sectionEl);
```

:::

Wenn Sie ein Element anhand seines Klassennamens mit `querySelector()` auswählen möchten, müssen Sie dem Klassennamen ein Punkt-Präfix (`.`) voranstellen. Hier ist ein Beispiel:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="highlight">
  <p>This content is highlighted.</p>
</div>
<script src="index.js"></script>
```

```css
body {
  background-color: lightgray;
}

.highlight {
  background-color: yellow;
}
```

```js
const highlightEl = document.querySelector(".highlight");
console.log(highlightEl);
```

:::

Sie haben auch andere Methoden, um mehrere Elemente auszuwählen, wie `getElementsByClassName()` und `querySelectorAll()`. Mehr darüber lernen Sie in zukünftigen Lektionen.

Indem Sie verstehen, wie Sie Web-APIs effizient nutzen, um das DOM zu manipulieren, können Sie leistungsstarke und interaktive Webanwendungen erstellen.
