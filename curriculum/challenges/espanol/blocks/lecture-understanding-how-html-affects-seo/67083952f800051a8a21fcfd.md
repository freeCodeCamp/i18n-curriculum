# --description--

SEO, o optimización para motores de búsqueda, es una práctica que optimiza las páginas web para que sean más visibles y tengan un mejor posicionamiento en los motores de búsqueda. Una forma de mejorar el SEO de tu sitio es proporcionar una descripción corta para la página web usando el elemento `meta`. Aquí tienes un ejemplo de cómo usar el elemento meta para establecer una descripción de página para un sitio de jardinería:

```html
<meta
  name="description"
  content="Discover expert tips and techniques for gardening in small spaces, choosing the right plants, and maintaining a thriving garden."
/>
```

Establecer el atributo `name` como `description` asegura que los navegadores, motores de búsqueda y otras herramientas web interpreten correctamente estos metadatos. El atributo `content` es donde colocarás tu descripción. Se recomienda que mantengas tus descripciones cortas y concisas. Esto se debe a que los motores de búsqueda a menudo truncarán la descripción según el diseño de la página de resultados.

Similar a otros tipos de elementos `meta`, la descripción `meta` no será visible directamente en la página web. Un lugar donde se puede encontrar la descripción de la página es en el fragmento de resultados de motores de búsqueda. Aquí hay algunos ejemplos de fragmentos de resultados de páginas para el subreddit de FreeCodeCamp y los repositorios de GitHub:

```md
r/freeCodeCamp: This is the official subreddit for the freeCodeCamp.org community. Learn to
code for free together with millions of other people...
```

```md
Our full-stack web development and machine learning curriculum is completely free and self-
paced. We have thousands of interactive coding challenges to help you...
```

En los ejemplos, cada una de las descripciones de página se encuentra justo debajo de los enlaces del sitio. En un par de segundos, los usuarios pueden tener una idea general de qué trata la página y decidir hacer clic en los enlaces para obtener más información.

Aunque las descripciones `meta` no afectan directamente el posicionamiento de un sitio en los motores de búsqueda, tener una descripción sólida podría resultar en más tráfico hacia tu sitio web.
