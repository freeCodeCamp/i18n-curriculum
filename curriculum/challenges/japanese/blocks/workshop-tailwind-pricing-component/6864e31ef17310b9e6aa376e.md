# --description--

グリッドアイテムがページの左右の端に近すぎるので、間隔を調整しましょう。

`max-w-6xl`に`div`クラスを代入して、幅を`72rem`に固定し、アイテムがページ全体に広がりすぎないようにしてください。

それだけだとすべて左寄せになるので、`mx-auto`に`div`を追加してアイテムを水平方向に中央揃えにしましょう。

# --hints--

`div`要素には`max-w-6xl`クラスを付けてください。

```js
const firstDiv = document.querySelector("div")

assert.isTrue(firstDiv.classList.contains("max-w-6xl"))
```

`div`要素には`mx-auto`クラスを付けてください。

```js
const firstDiv = document.querySelector("div")

assert.isTrue(firstDiv.classList.contains("mx-auto"))
```
