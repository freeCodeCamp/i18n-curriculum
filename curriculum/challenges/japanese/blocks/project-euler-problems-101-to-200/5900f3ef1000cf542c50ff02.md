# --description--

ある素数 $p$ に対して、正の整数 $n$ が存在し、式 $n^3 + n^{2}p$ が完全な立方体になる場合があります。

例えば、$p = 19$ のとき、$8^3 + 8^2 × 19 = {12}^3$ となります。

最も驚くべきことは、この性質を持つ各素数に対して $n$ の値が一意であり、100未満の素数の中でこの性質を持つものは4つだけだということです。

100万未満の素数のうち、この驚くべき性質を持つものはいくつあるでしょうか？
