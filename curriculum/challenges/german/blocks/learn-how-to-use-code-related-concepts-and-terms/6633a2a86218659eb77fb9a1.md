# --description--

Das Modalverb `can` wird verwendet, um Möglichkeit oder Fähigkeit auszudrücken. Zum Beispiel zeigt `You can see the stars at night` im Satz `can`, dass es möglich ist, nachts die Sterne zu sehen.

`Change` bedeutet, etwas zu verändern oder anders zu machen. Im Alltag könnten Sie nach der Arbeit Ihre Kleidung `change`, was bedeutet, dass Sie von Arbeitskleidung zu Freizeitkleidung wechseln.

`Stay the same` bedeutet, unverändert oder konstant zu bleiben. Zum Beispiel `The rules of the game stay the same every year.`. Das bedeutet, dass sich die Regeln von einem Jahr zum nächsten nicht ändern.
