# --explanation--

`Typically`는 보통 또는 예상되는 방식으로 일어나는 일을 설명할 때 사용합니다. `usually`와 비슷하며 보통 일어나는 일을 말할 때 자주 씁니다. 예를 들어:

- `Typically, we have meetings on Mondays.` - 이 문장은 회의가 보통 월요일에 열린다는 뜻입니다.

- `She typically arrives at work by 9 AM.` - 이 문장은 그녀가 보통 오전 9시까지 출근한다는 뜻입니다.
