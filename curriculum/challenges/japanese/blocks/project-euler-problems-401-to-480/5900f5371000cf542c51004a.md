# --description--

反転ゲームは、$N$×$N$ の正方形の盤上で行う2人用ゲームです。

各マスには、白面と黒面を持つディスクが1枚置かれています。

ゲームはすべてのディスクが白面を上にして開始します。

1ターンは、次の条件を満たす長方形内のすべてのディスクを反転させることから成ります：

- 長方形の右上隅に白いディスクがある
- 長方形の幅が完全平方数（1、4、9、16、…）
- 長方形の高さが三角数（1、3、6、10、…）

<img alt="5×5の盤上で4×3の長方形内のすべてのディスクを反転させる様子" src="https://cdn.freecodecamp.org/curriculum/project-euler/flipping-game-1.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

プレイヤーは交互にターンを行います。盤面のすべてのディスクを黒にしたプレイヤーが勝ちです。

$W(N)$ を、すべてのディスクが白の $N$×$N$ の盤で、最初のプレイヤーが完全なプレイをした場合の勝利手数の数とします。

$W(1) = 1$、$W(2) = 0$、$W(5) = 8$、そして $W({10}^2) = 31\,395$ です。

$N = 5$ の場合、最初のプレイヤーの勝利につながる最初の8手は以下の通りです：

<img alt="N = 5 の場合の最初のプレイヤーの勝利につながる8手" src="https://cdn.freecodecamp.org/curriculum/project-euler/flipping-game-2.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

$W({10}^6)$ を求めてください。
