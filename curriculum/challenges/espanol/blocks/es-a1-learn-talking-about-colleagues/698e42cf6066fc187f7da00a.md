# --description--

El verbo `vivir` significa "vivir".

Cuando se refiere a otra persona ("él"/"ella"), el verbo se conjuga como `vive`. Normalmente va seguido de la preposición `en` para indicar dónde vive la persona, con esta estructura:

**`Vive en` + lugar**
