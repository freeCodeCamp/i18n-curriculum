# --explanation--

`Unusually` es un adverbio que se usa para describir algo que ocurre más o menos de lo que es normal o esperado. Por otro lado, `unusual` es un adjetivo que describe algo que no es común o normal. Por ejemplo:

- El servidor experimentó una carga `unusually` alta hoy. – En esta oración, `unusually` se refiere a qué tan alta fue la carga (mucho más allá de lo normal).

- `It was unusual for the server to slow down like that.` – En esta oración, `unusual` se refiere a la situación. No modifica un verbo ni un adjetivo.

Una `query` es una pregunta o solicitud enviada a una base de datos para obtener información o realizar una acción. Por ejemplo:

`The application sent a query to the database to fetch user details.` – Esto significa que la aplicación pidió a la base de datos información específica.
