# --explanation--

Sophie erklärt Vererbung als ein Programmierkonzept, bei dem eine Klasse (eine Kind- oder Subklasse) auf die Eigenschaften und Methoden einer anderen Klasse (der Eltern- oder Superklasse) zugreifen kann.

Dies ermöglicht Entwicklern, Code wiederzuverwenden und Funktionalität zu erweitern, ohne alles neu schreiben zu müssen. Es ist ein zentraler Bestandteil der objektorientierten Programmierung, der Effizienz und ein saubereres Design fördert.
