# --explanation--

`Of course` ist eine höfliche Art, Zustimmung zu zeigen oder `yes` ohne Zögern auszudrücken. Es bedeutet, dass die sprechende Person gerne oder bereit ist, etwas zu tun. Zum Beispiel:

`Of course, I'll help you with the report.` – Das bedeutet, die sprechende Person hilft gerne.

`Will` wird oft verwendet, um Versprechen zu machen oder eine Entscheidung im Moment des Sprechens auszudrücken. Zum Beispiel:

`I'll email you the file this afternoon.` – Das zeigt ein Versprechen, bald etwas zu tun.
