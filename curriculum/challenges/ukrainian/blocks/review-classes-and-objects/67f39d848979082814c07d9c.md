## Класи та об’єкти в Python

- **Визначення класу**: Клас — це шаблон для створення об’єктів. Він визначає поведінку об’єкта через його атрибути та методи. Ось базовий приклад визначення класу в Python:

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f'{self.name.upper()} says woof woof!')
```

- **Створення об’єктів**: Об’єкти — це екземпляри класу. Вони створюються шляхом виклику класу з необхідними аргументами.

```python
dog1 = Dog('Jack', 3)
dog2 = Dog('Thatcher', 5)

dog1.bark()  # JACK says woof woof!
dog2.bark()  # THATCHER says woof woof!
```

- **Виклик методів об’єктів**: Ви можете викликати методи об’єктів, щоб виконувати дії або отримувати інформацію.

```python
object_name1.method_name()
object_name2.method_name()
```

- **Різниця між класом і об’єктом**: Клас — це багаторазовий шаблон, а об’єкт — це конкретний екземпляр цього класу з реальними даними.

## Атрибути

- **Атрибути екземпляра**: Визначаються в `__init__()` за допомогою `self` і є унікальними для кожного об’єкта.
- **Атрибути класу**: Визначаються безпосередньо всередині класу і спільні для всіх екземплярів.

```python
class Dog:
    species = 'French Bulldog'  # Class attribute

    def __init__(self, name):
        self.name = name  # Instance attribute

print(Dog.species) # French Bulldog

jack = Dog('Jack')
print(jack.name)     # Jack
print(jack.species)  # French Bulldog
```

## Методи

- **Методи**: Функції, визначені всередині класу, які працюють з атрибутами об’єкта.

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
print(my_car_1.describe())  # This car is a red Tesla Model S
```

- **Доступ до методів**: Викликайте методи об’єктів за допомогою точкової нотації. Ось приклад виклику методу `describe` на двох різних об’єктах car:

```python
class Car:
    def __init__(self, color, model):
        self.color = color  
        self.model = model  

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
my_car_2 = Car('green', 'Lamborghini Revuelto')

print(my_car_1.describe()) # Calling method using the dot notation

print(my_car_2.describe()) # Calling method using the dot notation
```

## Dunder (магічні) методи

- **Визначення**: Спеціальні методи, які починаються і закінчуються подвійним підкресленням (наприклад, `__init__`, `__len__`, `__str__`, `__eq__`). Python використовує їх внутрішньо для вбудованих операцій.

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **Непрямий виклик dunder-методів**: Вам не потрібно викликати dunder-методи безпосередньо. Замість цього Python автоматично викликає їх, коли відбуваються певні дії. Ці операції включають:

  - **арифметичні дії, такі як додавання, віднімання, множення, ділення та інші**. Зокрема, викликається `__add__()`, `__sub__()` для віднімання, `__mul__()` для множення та `__truediv__()` для ділення.
        
  - **операції з рядками, такі як конкатенація, повторення, форматування та перетворення в текст**. Для конкатенації викликається `__add__()`, для повторення — `__mul__()`, для форматування — `__format__()`, для перетворення в текст — `__str__()` і `__repr__()` тощо.
        
  - **операції порівняння, такі як перевірка на рівність, менше, більше та інші**. Для перевірки рівності викликається `__eq__()`, для менше — `__lt__()`, для більше — `__gt__()` тощо.
        
  - **операції ітерації, такі як зробити об’єкт ітерованим і переходити між елементами**. Викликається `__iter__()` для повернення ітератора і `__next__()` для отримання наступного елемента.

## Приклад із реального життя: кошик для покупок

- **Клас Cart з dunder-методами**: Дозволяє додавати, видаляти, ітеруватися та перевіряти вміст із вбудованою поведінкою.

```python
class Cart:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

    def remove(self, item):
        if item in self.items:
            self.items.remove(item)
        else:
            print(f'{item} is not in cart')

    def list_items(self):
        return self.items

    def __len__(self):
        return len(self.items)

    def __getitem__(self, index):
        return self.items[index]

    def __contains__(self, item):
        return item in self.items

    def __iter__(self):
        return iter(self.items)

cart = Cart()
cart.add('Laptop')
print(len(cart))        # 1
print('Laptop' in cart) # True
```
