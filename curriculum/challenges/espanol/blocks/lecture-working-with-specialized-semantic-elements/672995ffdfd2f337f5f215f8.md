# --interactive--

El elemento de superíndice se usa para mostrar un texto como superíndice. Un superíndice es un símbolo o letra impresa por encima de la línea normal de texto.

Aquí tienes un ejemplo usando el elemento superscript para ilustrar exponentes (para ver las vistas previas, activa el editor interactivo):

:::interactive_editor

```html
<p>2<sup>2</sup> (2 squared) is 4.</p>
```

:::

En este ejemplo, el número 2 está envuelto en etiquetas `sup` para representar el superíndice dentro del párrafo. En la ventana de vista previa, verás que el segundo número 2 es más pequeño y está un poco más alto que el primer número 2.

Los casos de uso comunes para el elemento superscript incluirían exponentes, letras superiores y números ordinales. Aquí tienes un ejemplo usando el elemento superscript para letras superiores (para ver las vistas previas, activa el editor interactivo): 

:::interactive_editor

```html
<p>
  Monseigneur is often written as <strong>M<sup>gr</sup></strong>.
</p>
```

:::

Las letras superiores se refieren a letras escritas en superíndice, generalmente para indicar abreviaturas. Las letras `g` y `r` están envueltas dentro de etiquetas de superíndice para ilustrar la abreviatura en este ejemplo.

Es importante notar que el elemento superscript solo debe usarse por razones tipográficas. Si quieres estilizar un texto con una línea base elevada, entonces debes usar CSS en lugar del elemento superscript.

Para representar ecuaciones químicas dentro de HTML, usarías el elemento subíndice. Este elemento utiliza un subíndice que muestra una línea de base más baja usando texto más pequeño.

Aquí tienes un ejemplo de cómo usar el elemento subíndice para mostrar la representación química del dióxido de carbono (para ver las vistas previas, activa el editor interactivo):

:::interactive_editor

```html
<p>CO<sub>2</sub></p>
```

:::

El número dos está envuelto dentro de etiquetas `sub` para ilustrar que el carácter debe ser un subíndice.

Los casos de uso comunes para el elemento subíndice incluyen fórmulas químicas, notas al pie y subíndices de variables.
