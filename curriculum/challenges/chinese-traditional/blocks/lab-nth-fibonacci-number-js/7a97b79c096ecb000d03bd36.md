# --description--

**目標：** 完成以下使用者故事並通過所有測試以完成實驗。

**User Stories:**

1. 你應該創建一個名為 `fibonacci` 的函式。
2. 你應該在 `sequence` 函式內定義一個名為 `fibonacci` 的陣列，並將其初始化為 `[0, 1]` 的值。
3. `fibonacci` 函式應接受一個參數，該參數為正整數 `n`。
4. 呼叫 `fibonacci(n)` 時，應使用動態規劃方法計算並傳回費氏數列中第 `n` 個數字，每個數字都是前兩個數字的和。
5. 費氏數列中第 `n` 個位置計算出的每個數字，應存放在 `sequence` 陣列的索引 `n - 1`。

```js
function fibonacci(n) {
  const sequence = [0, 1];

  if (n < 2) {
    return sequence[n];
  }

  for (let i = 2; i <= n; i++) {
    sequence.push(sequence[i - 1] + sequence[i - 2]);
  }

  return sequence[n];
}
```
