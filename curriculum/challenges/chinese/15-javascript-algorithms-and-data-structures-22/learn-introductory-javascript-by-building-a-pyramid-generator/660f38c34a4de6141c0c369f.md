# --description--

你的金字塔生成器现在可以按预期运行。 但这是一个进一步探索你所编写的代码的绝佳机会。

加法运算符不是向变量添加值的唯一方法。 <dfn>加法赋值</dfn>运算符可以简写为“取变量的原始值，添加该值，然后将结果赋给变量”。 例如，这两个语句将产生相同的结果：

```js
test = test + 1;
test += 1;
```

更新 `for` 循环中的迭代器语句以使用加法赋值。
