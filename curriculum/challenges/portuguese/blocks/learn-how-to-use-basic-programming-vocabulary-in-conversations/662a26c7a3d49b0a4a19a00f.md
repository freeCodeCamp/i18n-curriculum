# --description--

A frase `out there` é usada para se referir a coisas que existem ou acontecem no mundo, frequentemente enfatizando um grande número ou variedade.É como dizer `existing in the world` ou `available in the general environment`.

Por exemplo, `There are so many kinds of technology out there.` Isso significa que existe uma grande variedade de tecnologia no mundo.
