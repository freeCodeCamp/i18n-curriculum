# --description--

Функції — це повторно використовувані частини коду, які виконуються, коли ви їх викликаєте. Багато мов програмування мають вбудовані функції, які полегшують початок роботи. Python не є винятком, і ми вже розглядали деякі вбудовані функції, як `print()` у попередніх уроках.

Ще одна корисна вбудована функція — `input()`, яка дозволяє запитати користувача про вхідні дані:

```python
name = input('What is your name?') # User types "Kolade" and presses Enter  
print('Hello', name) # Output: Hello Kolade
```

З іншого боку, `int()` перетворює число, булеве значення або числовий рядок у ціле число:

```python
print(int(3.14)) # 3
print(int('42')) # 42
print(int(True)) # 1
print(int(False)) # 0 
```

Ви також можете написати власні кастомні функції. Для цього використовуйте ключове слово `def`, за яким іде ім’я функції, пара круглих дужок і двокрапка. Потім на новому рядку напишіть код, який має виконувати функція. Код, що виконує функція, також називають тілом функції.

Ось приклад кастомної функції з ім’ям `hello`, яка виводить рядок `Hello World` у термінал:

```python
def hello():
    print('Hello World')
```

Щоб запустити функцію, потрібно викликати її за ім’ям із парою круглих дужок:

```python
hello() # Hello World
```

Зверніть увагу на відступ перед `print('Hello World')`. Як ви пам’ятаєте з попередніх уроків, Python покладається на відступи, щоб визначити, які групи інструкцій належать разом. Ці групи інструкцій називають блоками коду.

Ось ще одна проста функція, яка виводить суму двох чисел у термінал:

```python
def calculate_sum(a, b):
    print(a + b)
```

Ви бачите, що наша функція `calculate_sum` має `a` і `b` у круглих дужках, розділені комою. Це називаються параметрами. Уявіть параметри як змінні-заповнювачі, які виступають «слотами» для значень, які ви передаєте у функцію під час виклику.

Щоб використати параметри, потрібно передати «аргументи». Аргументи — це значення, які ви передаєте функції під час її виклику.

Ось як викликати функцію `calculate_sum`, щоб підсумувати числа `3` і `1`:

```python
calculate_sum(3, 1) # 4
```

Якщо викликати функцію без правильної кількості аргументів, ви отримаєте `TypeError`:

```python
calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

Функції також використовують спеціальне ключове слово `return`, щоб вийти з функції і повернути значення. Якщо ви явно не використовуєте `return`, Python за замовчуванням поверне `None`.

Ось приклад:

```python
def calculate_sum(a, b):
    print(a + b)

my_sum = calculate_sum(3, 1) # 4
print(my_sum) # None
```

Ви бачите, що функція `calculate_sum` виводить суму `a` і `b`, але нічого явно не повертає. Тож коли ми присвоюємо її результат `my_sum`, значення фактично є `None`. Щоб це виправити, можна використати ключове слово `return`, щоб повернути результат:

```python
def calculate_sum(a, b):
    return a + b

my_sum = calculate_sum(3, 1)
print(my_sum) # 4
```

Тепер `calculate_sum` повертає суму `a` і `b`, яка зберігається в `my_sum`.
