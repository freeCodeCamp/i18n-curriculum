# --description--

Zur Erinnerung: Dieses Projekt baut auf dem folgenden Starter-Projekt auf, das von <a href="https://github.com/freeCodeCamp/boilerplate-mochachai/" target="_blank" rel="noopener noreferrer nofollow">GitHub</a> geklont wurde.

In den nächsten Aufgaben simulieren Sie menschliche Interaktionen mit einer Seite, indem Sie einen Headless-Browser verwenden.

Headless-Browser sind Webbrowser ohne grafische Benutzeroberfläche. Sie können HTML, CSS und JavaScript genauso rendern und interpretieren wie ein normaler Browser, was sie besonders nützlich für das Testen von Webseiten macht.

Für die folgenden Aufgaben verwenden Sie Zombie.js, einen leichtgewichtigen Headless-Browser, der keine zusätzlichen Binärdateien benötigt. Es gibt jedoch viele andere, leistungsfähigere Headless-Browser-Optionen.

Mocha ermöglicht es Ihnen, vor dem Ausführen der eigentlichen Tests Code auszuführen. Das ist nützlich, um beispielsweise Einträge in einer Datenbank hinzuzufügen, die in den restlichen Tests verwendet werden.

Mit einem Headless-Browser müssen Sie vor dem Ausführen der Tests die Seite, die Sie testen, **besuchen**.

Der `suiteSetup`-Hook wird nur einmal zu Beginn einer Testsuite ausgeführt.

Es gibt mehrere andere Hook-Typen, die Code vor jedem Test, nach jedem Test oder am Ende einer Testsuite ausführen können. Weitere Informationen finden Sie in der Mocha-Dokumentation.

# --instructions--

Fügen Sie innerhalb von `tests/2_functional-tests.js`, direkt nach der `Browser`-Deklaration, Ihre Projekt-URL der `site`-Eigenschaft der Variablen hinzu:

```js
Browser.site = 'http://0.0.0.0:3000'; // Your URL here
```

Instanziieren Sie dann auf der obersten Ebene der `'Functional Tests with Zombie.js'`-Suite eine neue Instanz des `Browser`-Objekts mit folgendem Code:

```js
const browser = new Browser();
```

Und verwenden Sie den `suiteSetup`-Hook, um den `browser` zur `/`-Route mit folgendem Code zu leiten. **Hinweis**: `done` wird als Callback an `browser.visit` übergeben, Sie sollten es nicht selbst aufrufen.

```js
suiteSetup(function(done) {
  return browser.visit('/', done);
});
```
