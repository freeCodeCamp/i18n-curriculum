# --description--

当你想询问某个具体事物时，你可以使用疑问词 `what`。`What` 通常用于谈论事物。 

`what` 一般用在疑问句首. 然后你可以接一个 `be` 动词（像是 `is` 或是 `are`）或是其他动词，然后是句子的主语（比如说 `you` 或是 `the team`）。 例如：

- `What is your favorite color?` – 有人想知道你最喜欢的颜色。

- `What does this code do?` - 有人想知道这段代码的作用或结果。
