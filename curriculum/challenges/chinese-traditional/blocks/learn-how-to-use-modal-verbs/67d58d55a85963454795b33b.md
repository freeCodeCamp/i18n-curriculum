# --explanation--

`Could` 和 `might` 是用來談論可能性的情態動詞。

`Could` 意味著某事是可能的，但不確定。 例如：

`This update could make the app run faster.` - 此更新有可能提升應用程式的速度。

`Might` 也表示可能性，但通常暗示的機率比 `could` 低。 例如：

`The new feature might improve the user experience.`－該特性可能會改善使用者體驗。
