# --description--

Vamos a practicar cómo escribir algunos números entre 70 y 79. Para completar esta tarea, ten en cuenta que en chino, el número 70 se llama `七十 (qī shí)`.

Los números del 0 al 9 son:

| Número | Chino |
| :--- | :--- |
| 0 | `零 (líng)` |
| 1 | `一 (yī)` |
| 2 | `二 (èr)` |
| 3 | `三 (sān)` |
| 4 | `四 (sì)` |
| 5 | `五 (wǔ)` |
| 6 | `六 (liù)` |
| 7 | `七 (qī)` |
| 8 | `八 (bā)` |
| 9 | `九 (jiǔ)` |

# --instructions--

Esta tarea no tiene audio. Completa los espacios en blanco a continuación con los números correspondientes en chino.
