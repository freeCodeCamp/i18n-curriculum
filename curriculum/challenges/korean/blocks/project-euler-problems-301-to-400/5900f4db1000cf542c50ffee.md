# --description--

보조 정렬은 약간 덜 효율적인 보고 정렬과 혼동하지 마세요. 보조 정렬은 입력 수열이 정렬되었는지 확인하고, 정렬되지 않았다면 두 요소를 무작위로 교환하는 과정을 반복합니다. 이 과정을 수열이 정렬될 때까지 계속합니다.

처음 4개의 자연수의 모든 순열을 입력으로 고려하면, 모든 $4!$ 입력 수열에 대해 교환 횟수의 기댓값은 $24.75$입니다.

이미 정렬된 수열은 0단계가 걸립니다.

이번 문제에서는 보조 정렬의 다음 변형을 고려합니다.

수열이 정렬되어 있지 않으면 세 요소를 무작위로 선택하여 이 세 요소를 무작위로 섞습니다.

이 세 요소의 모든 $3! = 6$ 순열이 동일한 확률로 발생합니다.

이미 정렬된 수열은 0단계가 걸립니다.

처음 4개의 자연수의 모든 순열을 입력으로 고려하면, 모든 $4!$ 입력 수열에 대해 셔플 횟수의 기댓값은 $27.5$입니다.

처음 11개의 자연수의 순열을 입력 수열로 고려하세요.

모든 $11!$ 입력 수열에 대해 평균적으로 이 정렬 알고리즘이 수행할 셔플 횟수의 기댓값은 얼마일까요? 답은 가장 가까운 정수로 반올림해서 제출하세요.
