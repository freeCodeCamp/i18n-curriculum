# --explanation--

Quando qualcuno dice che sta `struggling with` qualcosa, significa che ha difficoltà o problemi con quella cosa. Per esempio:

- `The team is struggling with the project deadline.` - Il team ha difficoltà a rispettare la scadenza.

`IDEs` indica `Integrated Development Environments`. Questi sono strumenti software usati dagli sviluppatori per scrivere, testare e eseguire il debug del codice. Esempi di IDE popolari includono Visual Studio Code, IntelliJ IDEA e Eclipse.

Nel dialogo, Tom ha difficoltà con i suoi IDE e vuole parlarne con Sarah.
