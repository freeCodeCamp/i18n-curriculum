# --interactive--

I dialog ti permettono di mostrare informazioni o azioni importanti agli utenti. Con l'elemento `dialog` integrato in HTML, puoi creare facilmente questi dialog (sia modali che non modali) nelle tue app web.

Un dialog modale è un tipo di dialog che obbliga l'utente a interagire con esso prima di poter accedere al resto dell'applicazione o della pagina web. Blocca efficacemente l'interazione con altri contenuti finché l'utente non completa un'azione, come chiudere il dialog o inviare un modulo.

Al contrario, un dialog non modale permette all'utente di continuare a interagire con altre parti della pagina o dell'applicazione anche quando il dialog è aperto. Non impedisce l'accesso al resto del contenuto.

In questa lezione imparerai come aprire e chiudere i modali usando i metodi `showModal()` e `close()`.

Quando vuoi assicurarti che l'utente si concentri su un'azione o un messaggio specifico di un modale, puoi aprire il dialog modale usando il metodo `showModal()`. Questo aggiungerà uno sfondo agli altri elementi della pagina e li disabiliterà. È ideale per modali che mostrano moduli, conferme e informazioni critiche che richiedono un'azione da parte dell'utente.

Ecco l'HTML per il dialog modale:

:::interactive_editor

```html
<dialog id="my-modal">
  <p>This is a modal dialog.</p>
</dialog>
```

:::

Per ora non vedi nulla sulla pagina perché il modale è chiuso al render iniziale. Puoi aprire automaticamente il modale usando il metodo `showModal()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
dialog.showModal();
```

:::

Il risultato nel browser mostrerà un modale con il testo `This is a modal dialog.`

È meglio lasciare il controllo all'utente. Per farlo, puoi aggiungere un listener per l'evento clic su un pulsante e usare il metodo `showModal()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.showModal();
});
```

:::

Se hai bisogno di mostrare un dialog permettendo comunque l'interazione con il contenuto esterno al dialog, allora puoi usare il metodo `show()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});
```

:::

Per chiudere un modale, puoi aggiungere un pulsante al modale dentro l'elemento `dialog` e usare il metodo `close()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
  <button id="close-modal-btn">Close Modal</button>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");
const closeButton = document.getElementById("close-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});

closeButton.addEventListener("click", () => {
  dialog.close();
});
```

:::

## --text--

Qual è la differenza tra un dialog modale e un dialog non modale?

## --answers--

Un dialog modale permette l'interazione con il resto della pagina, mentre un dialog non modale no.
