# --explanation--

`tokens`는 인증 시스템에서 사용자를 확인하거나 식별하는 데 사용되는 정보 조각, 주로 디지털 형태를 의미합니다. 예를 들어:

`The system uses tokens to verify each user's session.` - 이 경우 시스템이 특정 자원에 접근하거나 세션을 계속할 수 있도록 사용자를 안전하게 확인하는 정보 조각에 의존한다는 뜻입니다.

`refreshed`는 계속 유효하거나 기능하도록 업데이트되거나 새로 고쳐졌다는 뜻입니다. 예를 들어:

`The page was refreshed to display the latest updates.` - 이 경우 `refreshed`는 페이지가 새 정보나 업데이트된 내용을 보여주기 위해 다시 로드되어 사용자가 가장 최신 버전의 내용을 볼 수 있게 했다는 뜻입니다.
