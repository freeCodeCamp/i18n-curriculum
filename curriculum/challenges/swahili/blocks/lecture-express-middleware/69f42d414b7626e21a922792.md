# --description--

Middleware ya ngazi ya programu ni programu ya kati inayounganishwa na mfano wa programu ya Express. Kwa kawaida hutumika kushughulikia maombi yanayohitaji usindikaji kupitia njia nyingi za data. Inaweza kutumika kwa ujumla au kwa njia maalum.

Middleware hii huandikishwa kwa kutumia `app.use()` au njia maalum za njia ya data kama `app.get()`, `app.post()`, n.k. Hufanya kazi kwa kila ombi linalokuja isipokuwa ikizuiliwa kwa njia maalum za data.

```js
const express = require('express')
const app = express()

// Application-level middleware that logs all incoming requests
app.use((req, res, next) => {
  console.log('Request received at:', new Date())
  next(); // Passes control to the next middleware
})

app.get('/', (req, res) => {
  res.send('Hello World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* `app.use()`: Hii ni middleware ya ngazi ya programu. Inarekodi kila ombi linalofanywa kwa seva, bila kujali njia ya data.
* Middleware hufanya kazi kwa kila ombi linalokuja, ikitoa seva maelezo kama tarehe na saa ya sasa.

Middleware ya ngazi ya programu ni muhimu kwa kazi zinazohitaji kutekelezwa katika njia nyingi au zote za data katika programu. Hii inaweza kujumuisha kurekodi, uthibitisho wa utambulisho, na kushughulikia makosa.
