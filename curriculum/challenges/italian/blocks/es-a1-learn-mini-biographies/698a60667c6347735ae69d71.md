# --description--

`Le` è un pronome che rappresenta "a lui", "a lei" o "a te".

`Le gusta` significa "gli piace" quando si parla di Diego, che è un uomo. È lo stesso quando si parla di una donna. Per esempio:

- `Le gusta el fútbol.` - Gli piace il calcio (riferito a un uomo).

- `Le gusta el fútbol.` - Le piace il calcio (riferito a una donna).
