# --description--

A expressão `first things first` é usada para enfatizar que coisas importantes devem ser feitas antes de qualquer outra coisa.Costuma-se dizer no início de uma tarefa para focar primeiro nos passos mais importantes.Por exemplo, `First things first, let's review the project plan` significa começar revisando o plano antes de passar para outras tarefas.

Um `environment`, em um contexto tecnológico, frequentemente se refere a um conjunto de condições de software e hardware nas quais um sistema opera.Por exemplo, um `development environment` é onde os programadores escrevem e testam seu código.
