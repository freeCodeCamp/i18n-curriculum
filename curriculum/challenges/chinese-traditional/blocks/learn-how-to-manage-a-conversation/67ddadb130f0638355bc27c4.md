# --explanation--

片語動詞是動詞與介系詞或副詞結合而產生新意義的動詞。在這部分的對話盒中，你會看到 `to send something` 和 `to send something over` 之間的差異。

`To send something` 意味著將某物從一個地方或人傳送到另一個地方或人，而 `to send something over` 則加入了將物品交付給某人或某地的概念，通常是非正式或數位方式。請看以下各一個範例：

- `I'll send the letter tomorrow.`－一個基本的轉移某物的述語。

- `Can you send over the file by email?` - 這強調將某物*傳送給某人*，通常是短距離或數位方式。加入 `over` 使這句話在日常對話中聽起來更自然。
