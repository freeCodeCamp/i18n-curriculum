# --interactive--

ربما رأيت رابطًا في صفحة ويب يتحول إلى اللون البنفسجي بعد نقره. هذا لأن حالة الرابط تغيرت، لذا تُطبق أنماط مختلفة. هناك خمس حالات مختلفة يمكن أن يكون عليها الرابط.

الأولى هي الحالة الافتراضية، وهي `:link`. تمثل هذه الحالة رابطًا لم يزره المستخدم أو ينقر عليه أو يتفاعل معه بعد. يمكنك اعتبار هذه الحالة توفر الأنماط الأساسية لكل الروابط في صفحتك. الحالات الأخرى تبنى فوقها.

فعّل المحرر التفاعلي وانظر إلى الرابط في نافذة المعاينة. يجب أن يكون لونه الافتراضي أزرق، وهو ما يمثل الحالة الافتراضية. إذا نقرت عليه، فسيتحول إلى اللون البنفسجي.

:::محرر_تفاعلي

```html
<a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
```

:::

الحالة الثانية هي `:visited`، التي تنطبق عندما يكون المستخدم قد زار الصفحة المرتبطة بها سابقًا. افتراضيًا، يتحول الرابط إلى اللون البنفسجي - لكن يمكنك الاستفادة من CSS لتوفير مؤشر بصري مختلف للمستخدم. هذا مفيد ليعرف الشخص أنه قد قرأ جزءًا من توثيقك بالفعل، أو أن الموقع موثوق لأنه استخدمه من قبل.

انقر على الرابط في نافذة المعاينة وسترى أن الرابط الذي تمت زيارته يتغير إلى اللون البني. لرؤية المعاينات، ستحتاج إلى تفعيل المحرر التفاعلي.

**ملاحظة**: تم توفير بعض CSS لهذا المثال التفاعلي حتى ترى التغييرات في أنماط الرابط. لا تقلق بشأن محاولة فهم شفرة CSS لأنك ستتعلم معناها في الوحدات القادمة.

:::محرر_تفاعلي

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:visited {
  color: brown;
}
```

:::

الحالة الثالثة هي `:hover`. تنطبق هذه الحالة عندما يحوم المستخدم بمؤشره فوق الرابط. تساعد هذه الحالة في جذب انتباه إضافي للرابط، لضمان أن المستخدم ينوي فعلاً النقر عليه.

فعّل المحرر التفاعلي وجرب تحويم الماوس فوق الرابط. سترى اللون يتغير إلى الأحمر.

:::محرر_تفاعلي

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:hover {
  color: red;
}
```

:::

ثم لدينا `:focus`. تنطبق هذه الحالة عندما نركز على الرابط.

انقر على أي جزء من المساحة الفارغة في نافذة المعاينة ثم اضغط على `tab` في لوحة المفاتيح. يجب أن ترى الرابط يتغير إلى اللون الأخضر. لرؤية المعاينات، ستحتاج إلى تفعيل المحرر التفاعلي.

:::محرر_تفاعلي

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:focus {
  color: green;
}
```

:::

وأخيرًا، لدينا `:active`. تنطبق هذه الحالة على الروابط التي يتم تفعيلها من قبل المستخدم. هذا يعني عادةً النقر على الرابط بزر الماوس الأساسي بالنقر الأيسر في معظم الحالات. تساعد هذه الحالة في إظهار للمستخدم أن العنصر الذي نقر عليه تفاعلي.

فعّل المحرر التفاعلي وانقر على الرابط. يجب أن ترى الرابط يتحول إلى اللون الأسود. يحدث هذا بسرعة كبيرة لذا قد تحتاج إلى النقر عليه عدة مرات لرؤية تغير اللون.

:::محرر_تفاعلي

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:active {
  color: black;
}
```

:::

عندما تستخدم هذه الحالات لتنسيق روابطك، هناك ترتيب محدد يجب كتابة CSS به: `link`، `visited`، `hover`، `focus`، ثم `active`.

الآن يجب أن تعرف كيف تضفي طابعك الشخصي على روابط صفحتك، مع توفير المعلومات المهمة التي يحتاجها المستخدم عن حالة كل رابط.
