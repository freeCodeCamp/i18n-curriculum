# --explanation--

Eine `suspicious email` ist eine E-Mail, die unzuverlässig oder gefährlich aussieht. Sie könnte Teil eines `phishing attack` sein, bei dem jemand versucht, Informationen zu stehlen. Zum Beispiel:

`This looks like a suspicious email. Can you check it?` – Sie vermuten, dass die E-Mail gefährlich sein könnte, und bitten jemanden, sie sich anzusehen.
