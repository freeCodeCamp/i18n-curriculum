# --explanation--

`That's a good suggestion, but` é uma forma de reconhecer uma ideia enquanto se introduz uma preocupação.Esta frase permite uma discussão respeitosa enquanto mantém as preocupações abertas para conversas futuras.Por exemplo:

- Pessoa 1: `How about we use this new software to improve productivity?`

- Pessoa 2: `That's a good suggestion, but we should check if it's compatible with our current systems.` - Esta pessoa reconhece a ideia mas destaca um possível problema de compatibilidade.
