# --description--

`Fingerprint scanning` é uma tecnologia que analisa os padrões únicos dos seus dedos para identificar e verificar quem você é.É como se seus dedos tivessem marcas especiais que são diferentes das de todas as outras pessoas.Quando você usa `fingerprint scanning`, o sistema verifica essas marcas para garantir que é realmente você.Assim como `facial recognition`, é uma técnica importante usada por `biometrics`.
