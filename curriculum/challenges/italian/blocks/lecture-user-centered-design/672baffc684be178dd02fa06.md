# --interactive--

**NOTA**: Alcuni esempi interattivi potrebbero usare CSS che non hai ancora imparato. Non preoccuparti di capire tutto il codice. Lo scopo degli esempi è mostrarti anteprime di questi concetti di design per farti capire meglio come funzionano. Per vedere le anteprime, devi abilitare l’editor interattivo.

I componenti card sono molto comuni negli e-commerce, nei social media e nei siti di notizie. Servono a mostrare informazioni in modo strutturato. Quando progetti le tue card, è importante conoscere le buone pratiche per far sì che gli utenti comprendano facilmente le informazioni che vuoi trasmettere.

La prima cosa da considerare nel design delle card è la semplicità. Non vuoi che le card siano visivamente troppo cariche o mostrino troppe informazioni. Per esempio, se una card è troppo affollata, ci saranno troppe informazioni da elaborare tutte insieme per l’utente.

Ecco un esempio di design di card affollata:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2><strong><em>Product Name: Ultimate Edition</em></strong></h2>
  <p>
    This is a great product that you will love! It has many features and benefits that will make your life easier and more enjoyable. Whether you're at home or on the go, this product is perfect for all situations. Don’t miss out on this incredible offer, buy now and experience the difference!
  </p>
  <button>Buy Now</button>
  <button>Learn More</button>
  <button>Add to Wishlist</button>
  <p><strong>Only 3 left in stock!</strong></p>
  <p><em>Rated 4.8 stars by 2,391 customers</em></p>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}
.card p {
  font-size: 1em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

Avere meno informazioni e un buon spazio tra gli elementi della card rende più facile per l’utente elaborare le informazioni e permette di avere più card sulla pagina.

Ecco un esempio di design di card semplice:

:::interactive_editor


:::

Un altro aspetto da considerare è dove l’utente può cliccare sulla card. Alcuni design di card hanno un solo pulsante, rendendo ovvio dove cliccare. Altri design rendono cliccabile tutta la card. Quando l’utente passa il mouse su qualsiasi parte della card, questa cambia colore o mostra un’ombra per indicare che è cliccabile. Qualunque design scegli, deve essere coerente in tutto il sito e facile da capire per l’utente.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <a href="#">
    <img src="https://placehold.co/150x150" alt="Example Product Image">
    <h2>Product Name</h2>
    <button>Buy Now</button>
  </a>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
  cursor: pointer;
}

.card:hover {
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
}

.card a {
  text-decoration: none;
  color: inherit;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

Un’altra considerazione riguarda l’uso dei media nelle card. Scegliere media di alta qualità può migliorare molto l’esperienza utente. Se usi immagini o video, per esempio in una card prodotto, più alta è la qualità, più l’utente sarà interessato a quel prodotto. Ma se usi media di scarsa qualità, l’utente potrebbe non fidarsi della qualità dei tuoi prodotti e servizi.

Una delle ultime cose da considerare è l’uso della gerarchia dei colori. Devi assicurarti che le informazioni più importanti sulla card siano le più evidenti. Puoi usare colori vivaci per elementi importanti come un pulsante call-to-action e colori chiari per elementi meno importanti.
