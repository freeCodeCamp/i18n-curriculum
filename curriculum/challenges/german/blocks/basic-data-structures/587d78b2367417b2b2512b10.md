# --description--

Okay, wir haben gelernt, wie man Elemente am Anfang und Ende von Arrays mit `shift()` und `pop()` entfernt, aber was ist, wenn wir ein Element irgendwo in der Mitte entfernen wollen? Oder mehr als ein Element auf einmal löschen möchten? Genau hier kommt `splice()` ins Spiel. `splice()` ermöglicht uns genau das: **beliebig viele aufeinanderfolgende Elemente** von irgendwo in einem Array zu entfernen.

`splice()` kann bis zu 3 Parameter annehmen, aber vorerst konzentrieren wir uns nur auf die ersten 2. Die ersten beiden Parameter von `splice()` sind ganze Zahlen, die Indizes oder Positionen von Elementen im Array darstellen, auf dem `splice()` aufgerufen wird. Und denken Sie daran, Arrays sind *nullbasiert indiziert*, also verwenden wir `0`, um das erste Element eines Arrays anzugeben. Der erste Parameter von `splice()` gibt den Index im Array an, ab dem Elemente entfernt werden sollen, während der zweite Parameter angibt, wie viele Elemente gelöscht werden. Zum Beispiel:

```js
let array = ['today', 'was', 'not', 'so', 'great'];

array.splice(2, 2);
```

Hier entfernen wir 2 Elemente, beginnend mit dem dritten Element (Index 2). `array` hätte den Wert `['today', 'was', 'great']`.

`splice()` verändert nicht nur das Array, auf dem es aufgerufen wird, sondern gibt auch ein neues Array zurück, das die Werte der entfernten Elemente enthält:

```js
let array = ['I', 'am', 'feeling', 'really', 'happy'];

let newArray = array.splice(3, 2);
```

`newArray` hat den Wert `['really', 'happy']`.
