# --interactive--

これまでのレッスンやワークショップで学んだように、複数のパラメータを持つ関数を作成し、その関数を引数付きで呼び出すことができます。

ここでおさらいです：

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call
getSum(3, 4); // 7
```

しかし、定義されたパラメータよりも多くの引数で関数が呼び出されたらどうなるでしょうか？

:::interactive_editor

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call with extra argument
console.log(getSum(3, 4, 5)); // 7
```

:::

この場合、JavaScriptはエラーをスローしません。代わりに余分な引数を無視して、`3` と `4` の数値を足し合わせます。可変長引数を受け取る関数は <dfn>可変長関数</dfn> と呼ばれます。

可変長関数を扱う場合、`arguments` オブジェクトを利用できます。この配列のようなオブジェクトには、関数に渡された引数の値が含まれています。

例を示します：

:::interactive_editor

```js
function logArgs() {
  for (const arg of arguments) {
    console.log(arg);
  }
}

logArgs(1, 2, 3);
// result:
// 1
// 2
// 3

logArgs("example"); // "example"
```

:::

`arguments` オブジェクトは配列のようなので、特定のインデックスの引数に次のようにアクセスできます：

:::interactive_editor

```js
function getArg() {
  return arguments[1];
}

console.log(getArg(2, 4, 6)); // 4
```

:::

また、`length` プロパティを使って、関数が呼び出されたときの引数の数を取得できます：

:::interactive_editor

```js
function getArgs() {
  return arguments.length;
}

console.log(getArgs("Example")); // 1
console.log(getArgs("Another", "Example")); // 2
```

:::

`arguments` オブジェクトは本物の配列のように見えますが、`Array` や `includes` のような組み込みの`push` メソッドは持っていません。これらのメソッドにアクセスするには、まず`arguments` オブジェクトを `slice`、`Array.from()`、またはスプレッド演算子を使って本物の配列に変換する必要があります：

:::interactive_editor

```js
function hasCat() {
  return [...arguments].includes("cat");
}

console.log(hasCat("dog", "chicken", "cat")); // true
console.log(hasCat("dog", "chicken", "horse")); // false
```

:::

`arguments` オブジェクトを使って可変長関数を扱うことは可能ですが、現代のJavaScriptアプリケーションでは通常、restパラメータ構文を使います。これは今後のレッスンで詳しく学びます。
