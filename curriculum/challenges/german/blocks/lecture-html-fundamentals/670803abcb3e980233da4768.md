# --interactive--

Das `div`-Element wird als Container verwendet, um andere Elemente zu gruppieren.

Hier ist ein Beispiel für ein `div`-Element. Fügen Sie ein weiteres Absatz-Element innerhalb des `div`-Elements hinzu und sehen Sie die Änderungen im Vorschaufenster. Um die Vorschauen zu sehen, müssen Sie den interaktiven Editor aktivieren.

:::interactive_editor

```html
<div>
  <p>Example paragraph element.</p>
</div>
```

:::

Sie verwenden hauptsächlich das `div`-Element, wenn Sie HTML-Elemente gruppieren möchten, die einen gemeinsamen Satz von CSS-Stilen teilen. Mehr über CSS lernen Sie in zukünftigen Lektionen und Workshops.

Obwohl das `div`-Element in realen Codebasen häufig verwendet wird, sollten Sie darauf achten, es nicht zu übernutzen. Es gibt Situationen, in denen ein anderes Element passender wäre.

Zum Beispiel, wenn Sie Ihren Inhalt in Abschnitte unterteilen möchten, wäre das `section`-Element geeigneter als ein `div`-Element.

Fügen Sie unter dem ersten ein weiteres `section`-Element hinzu. Dann innerhalb des `section`-Elements ein `h2`- und ein `p`-Element. Sie können beliebigen Text verwenden und sehen die Änderungen im Vorschaufenster. Um mit dem Beispiel zu interagieren, müssen Sie den interaktiven Editor aktivieren.

:::interactive_editor

```html
<section>
  <h2>Mammals</h2>
  <p>
    Mammals are warm-blooded animals with fur or hair. Most give birth to live
    young.
  </p>
  <ul>
    <li>Lion</li>
    <li>Elephant</li>
    <li>Dolphin</li>
  </ul>
</section>
```

:::

Das `section`-Element hat eine semantische Bedeutung gegenüber dem `div`-Element, das keine semantische Bedeutung hat. Semantik ist die Bedeutung von Wörtern oder Ausdrücken in einer Sprache. In HTML, das eine Sprache ist, haben auch Elemente ihre eigene semantische Bedeutung. Das bedeutet also, wenn Sie ein `section`-Element verwenden, erkennt der Browser dessen semantische Bedeutung und behandelt es als Abschnitt – auf Desktops, Mobilgeräten und überall sonst.
