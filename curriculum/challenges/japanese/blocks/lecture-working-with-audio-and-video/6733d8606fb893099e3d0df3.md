# --description--

`HTMLMediaElement` と MediaStreams API について学びましたが、ビデオやオーディオを扱う際に役立つ他のAPIもいくつかあります。

最初は Screen Capture API です。名前の通り、このAPIはユーザーの画面を録画することができます。このAPIは `getDisplayMedia()` オブジェクトの `mediaDevices` メソッドを呼び出して公開され、返されたメディアストリームを利用します。

次に MediaStream Recording API があります。このAPIは MediaStreams API と連携して動作し、MediaStream（あるいは直接 `HTMLMediaElement`）を録画できます。そして、ローカルのファイルストレージに書き込める Blob ペイロードを含む `dataavailable` イベントを発火します。

これらの技術の基盤となっているのが Media Source Extensions API です。例えば、このAPIを使うとユーザーのウェブカメラの映像を `srcObject` プロパティを使ってビデオ要素に直接渡すことができます。

最後に、ウェブ上のすべての音声を支える Web Audio API があります。このAPIには、オーディオデータを特に含むバッファを表す `AudioBuffer` や `AudioContext` といった重要なオブジェクトが含まれています。

これでオーディオとビデオに関するレッスンは終了です！
