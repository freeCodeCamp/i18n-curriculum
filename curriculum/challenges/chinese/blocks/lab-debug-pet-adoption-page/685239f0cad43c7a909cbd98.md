# --description--

Sally 是一位宠物领养商店的老板，她搭建了自己的第一个网页，但存在一些问题。

你的任务是修复所有错误，以便 Sally 能够继续构建她的页面。

**目标：**实现以下用户需求并通过所有测试以完成实验。

**用户需求：**

1. Sally 想使用一张猫咪的图像，但图像显示不正确。 你需要在 `img` 元素中修正以下内容：
   - 用图像源的正确属性替换 `href` 属性。
   - 将 `att` 属性替换为代表图像简短描述性文本的正确属性。
   - 移除 `</img>` 结束标签，因为 `img` 元素是空元素，没有结束标签。
2. Sally 希望使用一些链接将用户引导到猫和狗的页面。 但链接无法正常工作。 你需要在 `a` 元素中修正以下内容：
   - 将两个 `src` 属性替换为用于指定 URL 的正确属性。

```js

```

```js

```

```js

assert.isDefined(catLink);
assert.isTrue(catLink.hasAttribute("href"));
```
