# --description--

`Got it`라는 표현은 누군가가 무언가를 이해했음을 나타내기 위해 대화에서 자주 사용됩니다. 이는 정보를 받았고 이해했다는 것을 간단하고 빠르게 인정하는 방법입니다. 예를 들어:

Lucy: `You need to finish the report by tomorrow.`
Bob: `Got it!` 

`Got it`는 Bob이 지시를 이해했다는 뜻입니다.
