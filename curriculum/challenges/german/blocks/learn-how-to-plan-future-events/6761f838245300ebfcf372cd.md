# --explanation--

`Unauthorized` bedeutet **nicht erlaubt** oder **nicht genehmigt**. Es wird gebildet, indem das Präfix `un-` an das Stammwort `authorized` angehängt wird, das erlaubt oder genehmigt bedeutet. Zum Beispiel:  

`Unauthorized users tried to access the system.` – Das bedeutet, dass Benutzer ohne Erlaubnis versucht haben, Zugang zu erhalten.
