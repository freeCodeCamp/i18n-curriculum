# --explanation--

`Sure thing.` es una forma informal y amigable de decir `Of course` o `I'll do it`. Muestra que la persona que habla está de acuerdo con la solicitud sin dudar. Por ejemplo:

- Persona 1: `Can you send me the file this afternoon?` - Esta persona está pidiendo un archivo.

- Persona 2: `Sure thing.` - Esto significa que el hablante está de acuerdo y enviará el archivo como se solicitó.
