# --interactive--

ページをデザインするとき、幅、高さ、パディング、マージンなどのさまざまなプロパティを扱います。これらのプロパティを定義するときには、使用したい長さの単位を指定する必要があります。

これらのプロパティを定義するために使える単位には、相対単位と絶対単位の2種類があります。このレッスンでは絶対単位に焦点を当てます。

絶対長さ単位は固定の長さで、他の何かに対して相対的ではありません。相対的とは、画面のサイズや親要素のサイズなど、他の何かに対して長さが決まることを意味します。相対単位については次のレッスンで詳しく説明します。

最も一般的な絶対単位はピクセル（`px`）です。ピクセルはCSSにおける固定サイズの単位で、寸法を正確に制御できます。つまり、1pxは常に1/96インチに相当します。

ただし、1pxはCSSレイアウトの目的で1/96インチに標準化されていますが、実際の物理的なピクセルの大きさはディスプレイによって異なる場合があることに注意してください。

ピクセルを使ってボックスの幅と高さを設定する例を見てみましょう。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="box"></div>
```

```css
.box {
  width: 100px;
  height: 100px;
  background-color: red;
}
```

:::

この例では、幅と高さが`100px`に設定された赤いボックスがあります。このボックスの寸法は画面サイズの変化に関係なく同じままです。

つまり、小さい画面のデバイスでもボックスの幅は`100px`のままです。大きなデスクトップ画面でもボックスの幅は`100px`のままです。

では、ピクセルのような絶対単位はいつ使うべきでしょうか？状況によりますが、一般的には要素の寸法、間隔、レイアウトを正確に制御したい場合にピクセルを使います。マージン、パディング、ボーダーにピクセルを使うこともあります。

2つのボックスのマージンをピクセルで設定する例を見てみましょう。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="box"></div>
<div class="box"></div>
```

```css
.box {
  width: 100px;
  height: 100px;
  background-color: red;
  margin: 10px;
}
```

:::

マージンはボックスの外側のスペースです。この例では、ボックスのすべての辺に`10px`のマージンがあります。

他の種類の絶対単位には以下のものがあります：

- `in`（インチ）単位は96pxに相当します
- `cm`（センチメートル）単位は1インチの25.2/64に相当します
- `mm`（ミリメートル）単位は1センチメートルの1/10に相当します
- `q`（クォーターミリメートル）単位は1センチメートルの1/40に相当します
- `pc`（パイカ）単位は1インチの1/6に相当します
- `pt`（ポイント）単位は1インチの1/72に相当します

これらの単位の多くは印刷用で、画面用ではありません。

最もよく使う単位はピクセルですが、他の絶対単位が存在することを知っておくことは重要です。
