# --description--

Mzizi wa mraba wa 2 unaweza kuandikwa kama sehemu isiyoisha ya mfululizo.

$\\sqrt{2} = 1 + \\dfrac{1}{2 + \\dfrac{1}{2 + \\dfrac{1}{2 + \\dfrac{1}{2 + ...}}}}$

Sehemu isiyoisha ya mfululizo inaweza kuandikwa, $\\sqrt{2} = \[1; (2)]$ inaonyesha kwamba 2 inarudiwa *bila kikomo*. Kwa njia kama hiyo, $\\sqrt{23} = \[4; (1, 3, 1, 8)]$. Inajitokeza kwamba mfululizo wa thamani za sehemu za mfululizo wa sehemu za mzizi wa mraba hutoa makadirio bora ya nambari za mlinganyo. Tuchukulie mfululizo wa karibu wa $\\sqrt{2}$.

$1 + \\dfrac{1}{2} = \\dfrac{3}{2}\\\\ 1 + \\dfrac{1}{2 + \\dfrac{1}{2}} = \\dfrac{7}{5}\\\\ 1 + \\dfrac{1}{2 + \\dfrac{1}{2 + \\dfrac{1}{2}}} = \\dfrac{17}{12}\\\\ 1 + \\dfrac{1}{2 + \\dfrac{1}{2 + \\dfrac{1}{2 + \\dfrac{1}{2}}}} = \\dfrac{41}{29}$

Hivyo mfululizo wa mfululizo wa karibu wa kwanza kumi wa $\\sqrt{2}$ ni:

$1, \\dfrac{3}{2}, \\dfrac{7}{5}, \\dfrac{17}{12}, \\dfrac{41}{29}, \\dfrac{99}{70}, \\dfrac{239}{169}, \\dfrac{577}{408}, \\dfrac{1393}{985}, \\dfrac{3363}{2378}, ...$

Kinachoshangaza zaidi ni kwamba nambari muhimu ya kihesabu, $e = \[2; 1, 2, 1, 1, 4, 1, 1, 6, 1, ... , 1, 2k, 1, ...]$. Maneno kumi ya kwanza katika mfululizo wa mfululizo wa karibu wa `e` ni:

$2, 3, \\dfrac{8}{3}, \\dfrac{11}{4}, \\dfrac{19}{7}, \\dfrac{87}{32}, \\dfrac{106}{39}, \\dfrac{193}{71}, \\dfrac{1264}{465}, \\dfrac{1457}{536}, ...$

Jumla ya tarakimu katika mnyororo wa nambari wa mfululizo wa 10<sup>th</sup> ni $1 + 4 + 5 + 7 = 17$.

Tafuta jumla ya tarakimu katika mnyororo wa nambari wa `n`<sup>th</sup> mfululizo wa karibu wa sehemu ya mfululizo wa `e`.

```js
function convergentsOfE(n) {
  function sumDigits(num) {
    let sum = 0n;
    while (num > 0) {
      sum += num % 10n;
      num = num / 10n;
    }
    return parseInt(sum);
  }

  // BigInt is needed for high convergents
  let convergents = [
    [2n, 1n],
    [3n, 1n]
  ];
  const multipliers = [1n, 1n, 2n];
  for (let i = 2; i < n; i++) {
    const [secondLastConvergent, lastConvergent] = convergents;
    const [secondLastNumerator, secondLastDenominator] = secondLastConvergent;
    const [lastNumerator, lastDenominator] = lastConvergent;
    const curMultiplier = multipliers[i % 3];

    const numerator = secondLastNumerator + curMultiplier * lastNumerator;
    const denominator = secondLastDenominator + curMultiplier * lastDenominator;

    convergents = [lastConvergent, [numerator, denominator]]
    if (i % 3 === 2) {
      multipliers[2] += 2n;
    }
  }
  return sumDigits(convergents[1][0]);
}
```
