# --explanation--

`Unusual activity`는 정상과 다른 행동을 의미합니다. 예를 들어:

- `We should investigate any unusual activity on the website.` - 사이트에서 이상한 일이 일어나고 있는지 확인해야 합니다.

- `If you notice unusual activity, please report it immediately.` - 화자는 누군가 이상한 것을 보면 즉시 알리라고 말합니다.

이 경우 `unusual activity on the servers`는 네트워크에서 이상한 일이 일어나고 있었음을 의미하며, 이는 보안 침해의 신호일 수 있습니다.
