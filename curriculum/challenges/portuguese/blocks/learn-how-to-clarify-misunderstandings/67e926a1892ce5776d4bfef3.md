# --explanation--

`Reducing complexity` significa tornar um sistema mais fácil de entender e trabalhar.
  
`Showing only what's necessary` significa que o usuário vê apenas as partes com as quais precisa interagir.
  
`Keeping the details hidden` refere-se a ocultar código interno ou lógica que não é necessária para o usuário.

Juntos, isso significa que a abstração cria uma interface clara e simples enquanto oculta os funcionamentos internos mais complexos nos bastidores.
