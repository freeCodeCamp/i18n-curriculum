# --explanation--

`I'll` is the contraction for `I will`.Neste contexto, expressa uma decisão tomada no momento de falar.Isso é comum quando alguém reage a uma situação ou concorda em fazer algo prontamente. 

`Now` é usado para indicar algo acontecendo no momento presente.Por exemplo:

`I'll send the report now.` - Isso mostra uma decisão de agir imediatamente.
