# --description--

Für jede ganze Zahl $n > 1$ ist die Familie der Funktionen $f_{n, a, b}$ definiert durch:

$f_{n, a, b}(x) ≡ ax + b\bmod n$ für $a, b, x$ ganze Zahlen mit $0 \lt a \lt n$, $0 \le b \lt n$, $0 \le x \lt n$.

Wir nennen $f_{n, a, b}$ eine Retraktion, wenn $f_{n, a, b}(f_{n, a, b}(x)) \equiv f_{n, a, b}(x)\bmod n$ für jedes $0 \le x \lt n$ gilt.

Sei $R(n)$ die Anzahl der Retraktionen für $n$.

$F(N) = \displaystyle\sum_{n = 1}^N R(n^4 + 4)$.

$F(1024) = 77\\,532\\,377\\,300\\,600$.

Finden Sie $F({10}^7)$. Geben Sie Ihre Antwort modulo $1\\,000\\,000\\,007$ an.
