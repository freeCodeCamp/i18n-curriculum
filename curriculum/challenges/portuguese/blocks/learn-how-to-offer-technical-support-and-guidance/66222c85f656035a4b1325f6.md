# --description--

`To be all set` significa estar totalmente preparado ou pronto para começar.Por exemplo, `Once you finish your training, you'll be all set to start working` indica que após o treinamento, você estará pronto para começar a trabalhar.

`Things` em `first things first` enfatiza a importância de fazer as tarefas mais importantes antes de qualquer outra coisa. 

`To install` significa configurar software ou equipamento para que esteja pronto para ser usado.Por exemplo, `You need to install the software on your computer.`

`Correctly` significa da maneira certa ou sem erros.Por exemplo, `Make sure you follow the instructions correctly` implica que você deve seguir as instruções com precisão.
