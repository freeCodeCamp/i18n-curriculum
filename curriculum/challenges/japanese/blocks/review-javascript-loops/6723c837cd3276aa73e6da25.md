# --interactive--

## ループの使い方

- **`for` ループ**: このタイプのループは、コードのブロックを決まった回数だけ繰り返すために使います。このループは初期化文、条件文、増減文の3つの部分に分かれています。初期化文はループが始まる前に実行されます。通常はカウンター変数の初期化に使います。条件文はループの各反復の前に評価されます。反復とはループを1回通過することです。条件が`true`の場合、ループ内のコードブロックが実行されます。条件が`false`の場合、ループは終了し次のコードブロックに進みます。増減文は各反復の後に実行されます。通常はカウンター変数を増減させるために使います。

:::interactive_editor

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

:::

- **`for...of` ループ**: このタイプのループは、イテラブルの値を繰り返す必要があるときに使います。イテラブルの例としては配列や文字列があります。

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

:::

- **`for...in` ループ**: このタイプのループは、オブジェクトのプロパティを繰り返す必要があるときに最適です。このループはオブジェクトのすべての列挙可能なプロパティを繰り返します。継承されたプロパティや数値以外のプロパティも含まれます。

:::interactive_editor

```js
const fruit = {
  name: 'apple',
  color: 'red',
  price: 0.99
};

for (const prop in fruit) {
  console.log(fruit[prop]);
}
```

:::

- **`while` ループ**: このタイプのループは、条件が`true`限りコードのブロックを実行し続けます。

:::interactive_editor

```js
let i = 5;

while (i > 0) {
  console.log(i);
  i--;
}
```

:::

- **`do...while` ループ**: このタイプのループは、条件をチェックする前に少なくとも一度はコードのブロックを実行します。

```js
let userInput;

do {
  userInput = prompt("Please enter a number between 1 and 10");
} while (Number(userInput) < 1 || Number(userInput) > 10);

alert("You entered a valid number!");
```

## `break` と `continue` 文

- **定義**: `break` 文はループを早期に終了するために使い、`continue` 文はループの現在の反復をスキップして次の反復に進むために使います。

:::interactive_editor

```js
// Example of break statement
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, and 4

// Example of continue statement 
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, 4, 6, 7, 8, and 9
```

:::
