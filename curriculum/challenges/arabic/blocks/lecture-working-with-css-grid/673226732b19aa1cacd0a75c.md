# --interactive--

في الدرس السابق، تعرّفت على شبكة CSS التي يمكن استخدامها لإنشاء تخطيطات معقدة وسلسة في صفحات الويب. في هذا الدرس، سنستكشف كيفية إنشاء تخطيطات شبكية مرنة باستخدام وحدة `fr`.

لنبدأ بهذا الترميز HTML الذي سيمثل حاوية الشبكة الخاصة بنا:

```html
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

داخل CSS، نُعيّن خاصية العرض إلى grid للحاوية.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 25% 25% 25% 25%;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

تُستخدم خاصية `grid-template-columns` لتحديد حجم كل عمود. في هذه الحالة، سيكون حجم كل عمود `25%` من الحاوية. ثم تُستخدم خاصية gap لإنشاء مساحة حول كل عمود.

حتى الآن كنا نستخدم النسب المئوية لحجم الأعمدة، لكن يمكننا أيضًا استخدام وحدة `fr`.

وحدة `fr` هي وحدة كسرية تمثل جزءًا من مساحة حاوية الشبكة.

إليك كيف ستبدو الشفرة عند إعادة صياغتها لاستخدام وحدات `fr` بدلاً من النسب المئوية.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

سيأخذ كل عمود جزءًا واحدًا من المساحة المتاحة. بما أن هناك أربعة أعمدة، فسيحصل كل عمود على حصة متساوية من المساحة المتاحة في حاوية الشبكة.

عندما تبدأ في بناء تخطيطات الشبكة الخاصة بك، ستجد نفسك ترغب في استخدام وحدات `fr` أكثر لأنها توفر طريقة مرنة ونسبية لتوزيع المساحة، مما يسمح لك بإنشاء تخطيطات متجاوبة تتكيف مع أحجام الشاشات المختلفة دون الحاجة لضبط قيم البكسل يدويًا.
