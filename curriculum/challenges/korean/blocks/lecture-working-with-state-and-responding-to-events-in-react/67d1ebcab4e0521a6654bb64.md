# --description--

React에서 상태에 있는 배열을 업데이트하는 것은 꽤 간단하지만, 특히 배열을 직접 수정할 수 있는 일반 JavaScript에 익숙하다면 실수하기 쉽습니다.

React에서는 상태를 불변으로 취급하여 변경 사항을 인식하고 사용자 인터페이스를 적절히 업데이트할 수 있습니다.

React에서 상태에 저장된 배열을 어떻게 업데이트할 수 있는지 살펴보겠습니다.

React 상태에서 배열을 업데이트할 때 가장 흔한 실수 중 하나는 `push()`, `pop()`, 또는 `splice()` 같은 메서드를 사용해 배열을 직접 수정하는 것입니다. 이 메서드들은 원본 배열을 변경하며, React는 이를 허용하지 않습니다.

React는 변경 사항을 감지하기 위해 새로운 배열 참조를 필요로 하므로, 배열을 직접 수정하면 컴포넌트가 예상대로 다시 렌더링되지 않을 수 있습니다.

다음은 상태에 있는 배열에 항목을 추가하기 위해 `push()` 메서드를 사용한 예시로, 작동하지 않습니다:

```jsx
import { useState } from "react";

function ItemsList() {
  const [items, setItems] = useState([
    { id: 0, name: "Item 1" },
    { id: 1, name: "Item 2" },
    { id: 2, name: "Item 3" },
  ]);

  const addItem = () => {
    const newItem = { id: items.length + 1, name: `Item ${items.length + 1}` };
    items.push(newItem); // This modifies the original array
    setItems(items); // React will not detect this change
  };

  return (
    <div>
      <button onClick={addItem}>Add Item</button>
      <ul>
        {items.map((item) => (
          <li key={item.id}>{item.name}</li>
        ))}
      </ul>
    </div>
  );
}

export default ItemsList;
```

`Add Item` 버튼을 클릭해도 사용자 인터페이스에 아무 변화가 없습니다.

배열에서 항목을 제거하기 위해 `pop()` 메서드를 사용하고 싶을 수도 있습니다:

```js
const removeItem = () => {
  items.pop(); // Modifies the original array
  setItems(items); // React will not detect this change, either
};
```

상태에 있는 배열을 업데이트하려면, 기존 배열을 변경하지 말고 새 배열을 만들어 작업한 후 React에 전달하는 것이 핵심입니다.

새 배열이기 때문에 React는 상태가 변경된 것을 인지하고 다시 렌더링을 트리거합니다.

전개 연산자를 사용해 `items` 배열에 항목을 추가하는 방법은 다음과 같습니다:

```js
const addItem = () => {
  const newItem = {
    id: items.length + 1,
    name: `Item ${items.length + 1}`,
  };

  // Creates a new array
  setItems((prevItems) => [...prevItems, newItem]);
};
```

`[...prevItems, newItem]`은 상태에 저장된 기존 `items` 배열의 모든 항목을 복사해 새 배열을 만들고, `newItem`를 끝에 추가하는데, 이는 `id`과 항목 번호를 증가시킵니다.

배열에서 항목을 제거하려면, 제거할 항목을 필터링한 후 새 배열을 반환하는 `filter()` 메서드를 사용할 수 있습니다:

```js
const removeItem = (id) => {
  setItems((prevItems) => prevItems.filter((item) => item.id !== id));
};
```

전체 코드는 다음과 같습니다:

```jsx
import { useState } from "react";

function ItemsList() {
  const [items, setItems] = useState([
    { id: 0, name: "Item 1" },
    { id: 1, name: "Item 2" },
    { id: 2, name: "Item 3" },
  ]);

  const addItem = () => {
    const newItem = { id: items.length + 1, name: `Item ${items.length + 1}` };
    setItems((prevItems) => [...prevItems, newItem]); // Creates a new array
  };

  const removeItem = (id) => {
    setItems((prevItems) => prevItems.filter((item) => item.id !== id)); // Creates a new array
  };

  return (
    <div>
      <button onClick={addItem}>Add Item</button>
      <ul>
        {items.map((item) => (
          <li key={item.id}>
            {item.name}{" "}
            <button onClick={() => removeItem(item.id)}>Remove</button>
          </li>
        ))}
      </ul>
    </div>
  );
}

export default ItemsList;
```

이것들이 상태에 있는 배열을 업데이트하는 일반적인 방법입니다.
