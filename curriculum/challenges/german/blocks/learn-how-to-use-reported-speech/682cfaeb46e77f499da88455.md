# --explanation--

Im **indirekten Rede** wird `to suggest` verwendet, um Ratschläge oder Ideen weiterzugeben, die jemand gegeben hat. Zum Beispiel:  

- Direkte Rede: `I think we should update our firewall.`

- Indirekte Rede: `Jake suggested updating the firewall.`

Dies zeigt, dass jemand einen Rat oder eine mögliche Lösung gegeben hat, ohne die genauen Worte zu verwenden.
