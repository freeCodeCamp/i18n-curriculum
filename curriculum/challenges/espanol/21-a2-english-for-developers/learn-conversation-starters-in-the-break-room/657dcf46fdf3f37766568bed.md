# --description--

`There are` se usa para hablar de la existencia de múltiples cosas o lugares en un lugar particular.

Por ejemplo: `There are many programming languages in the market`.

# --questions--

## --text--

¿Qué frase usa María para indicar la existencia de múltiples parques cerca de aquí?

## --answers--

`There is a park.`

### --feedback--

`There is` se usa para sustantivos singulares, no plurales.
