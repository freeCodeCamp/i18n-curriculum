# --explanation--

Щоб знайти правильну відповідь, зверніть увагу на ключові слова, які вказують на те, що потребує пріоритету.

`Critical systems` показує, що Джейк наголошує на найважливіших частинах системи, які потребують захисту. Це означає, що команда має зосередитися на їхньому захисті перш за все, а не перевіряти кожен файл чи вимикати все.

Зосередившись спочатку на `critical systems`, вони можуть зменшити ризик подальшої шкоди і захистити основні операції.
