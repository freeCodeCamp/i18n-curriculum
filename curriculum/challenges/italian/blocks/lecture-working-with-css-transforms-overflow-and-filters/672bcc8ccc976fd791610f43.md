# --interactive--

La proprietà CSS `transform` è uno strumento potente che ti permette di modificare la presentazione visiva degli elementi sulla tua pagina web senza influenzare il layout degli altri elementi. Ti consente di applicare varie trasformazioni agli elementi come ruotarli, scalarli, inclinarli o traslarli (spostarli) nello spazio 2D o 3D.

La proprietà `transform` funziona applicando una trasformazione matematica al sistema di coordinate di un elemento. Questo significa che puoi manipolare la forma e la posizione di un elemento mantenendo intatto il suo posto originale e il flusso del documento.

Esploriamo alcune funzioni di trasformazione comuni. Ecco un esempio di un elemento box:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
body {
  border: 2px solid black;
}

.box {
  width: 200px;
  height: 200px;
  background-color: red;
}
```

:::

Abbiamo impostato il `body` per avere un bordo nero solido così puoi vedere l’elemento `.box` annidato dentro l’elemento `body`.

La funzione `translate` sposta un elemento dalla sua posizione attuale. Ecco un esempio aggiornato che usa la funzione `translate`:  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
body {
  border: 2px solid black;
}

.box {
  width: 200px;
  height: 200px;
  background-color: red;
  transform: translate(50px, 100px);
}
```

:::

Questa regola CSS sposterà l’elemento con la classe `box` di 50 pixel a destra e 100 pixel in basso rispetto alla sua posizione originale.

La funzione `rotate` ruota un elemento attorno a un punto fisso e questo è un esempio di utilizzo della funzione `rotate` per l’elemento `.box` visto prima:  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: rotate(45deg);
}
```

:::

Questo ruoterà l’elemento di quarantacinque gradi in senso orario.

La funzione `scale` ti permette di cambiare la dimensione di un elemento. Ecco un esempio:  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: scale(1.5, 2);
}
```

:::

Questo renderà l’elemento una volta e mezza più largo e due volte più alto rispetto alla sua dimensione originale.

Puoi combinare più trasformazioni in una singola dichiarazione:  

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: translate(50px, 50px) rotate(45deg) scale(1.5);
}
```

:::

Questo sposterà l’elemento di 50 pixel a destra e in basso, lo ruoterà di 45 gradi e lo scalerà a una volta e mezza la sua dimensione originale.

Anche se la proprietà `transform` è potente per creare design visivamente accattivanti, è importante considerare l’accessibilità quando la usi. Ecco alcune preoccupazioni importanti sull’accessibilità da tenere a mente.

I lettori di schermo potrebbero non trasmettere accuratamente i contenuti trasformati. Per esempio, se usi `transform` per riorganizzare l’ordine visivo degli elementi, i lettori di schermo leggeranno comunque il contenuto nell’ordine originale del DOM. Questo può creare confusione per chi si affida ai lettori di schermo.

Quando usi `scale` per ridimensionare il testo, fai attenzione a non renderlo troppo piccolo o troppo grande. Un testo estremamente piccolo può essere difficile da leggere mentre un testo troppo grande potrebbe uscire dal suo contenitore e diventare illeggibile. In generale è meglio usare tecniche di stile del carattere tipografico adeguate per il ridimensionamento del testo.

Se usi `transform` per effetti di animazione, tieni conto degli utenti sensibili al movimento. Animazioni eccessive o rapide possono causare disagio o addirittura scatenare crisi per alcune persone. Considera di offrire un modo per ridurre o disattivare le animazioni. Quando usi trasformazioni 3D, ricorda che non tutti gli utenti percepiscono la profondità allo stesso modo. Assicurati che qualsiasi informazione critica trasmessa tramite effetti 3D sia disponibile anche in formato 2D o tramite testo.

Se usi `transform` per nascondere o mostrare contenuti, assicurati che il contenuto sia ancora accessibile ai lettori di schermo e alla navigazione da tastiera. Il contenuto nascosto dovrebbe essere veramente nascosto, ad esempio usando `display: none;` o `visibility: hidden;`, invece di essere solo spostato visivamente fuori schermo.

Quando applichi `transform` a elementi interattivi come pulsanti o collegamenti, assicurati che l’area cliccabile rimanga intuitiva e facilmente raggiungibile. Un pulsante trasformato drasticamente potrebbe risultare visivamente confuso o difficile da cliccare, specialmente per utenti con disabilità motorie.

In conclusione, anche se la proprietà CSS `transform` è uno strumento potente per creare design web visivamente dinamici, è essenziale usarla responsabilmente tenendo conto dell’accessibilità. Testa sempre i tuoi elementi trasformati con varie tecnologie assistive e considera di fornire modi alternativi per accedere a informazioni o funzionalità che potrebbero essere influenzate dalle trasformazioni.
