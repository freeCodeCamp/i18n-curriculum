# --interactive--

讓我們來了解 DOM 以及它為何對網頁開發如此重要。DOM 代表 Document Object Model。它是一個程式設計介面，讓我們能與 HTML 文件互動。

使用 DOM，你可以新增、修改或刪除網頁上的元素。你甚至可以讓你的網站具有互動性，透過讓元素監聽並回應 `event`。

在 DOM 中，HTML 文件表述為節點樹。每個節點表述 HTML 文件中的一個 HTML 元素：

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>DOM Example</title>
  </head>
  <body>
    <h1>What is the DOM?</h1>
    <h2>Let's learn about the DOM</h2>
  </body>
</html>
```

這是一個表述我們範例基本 DOM 結構的圖表：

```md
Document
========

HTML
-----

- Head
  - Title
- Body
  - H1
  - H2   
```

當然，這可以根據網頁的 HTML 標記的結構變得更詳細且複雜。

HTML 文件是 DOM 階層體系中的根節點。它有一個子節點，即 `html` 元素。這是 HTML 文件的根元素，因為所有其他節點都從它衍生。

`html` 元素有兩個子元素：`head` 和 `body`。

`head` 元素包含關於文件的詮釋資料。它提供有關網頁的基本資訊。

`body` 元素包含網頁的可見內容。

你可以使用 JavaScript 存取這些元素，在你的程式碼中操作它們，甚至動態修改它們。這就是 DOM 和網頁 API 的威力。

要在 JavaScript 中存取這些元素，你可以使用 `getElementById()` 和 `querySelector()` 方法。這些方法是網頁 API，因為它們提供了使用 JavaScript 與 DOM 互動的標準化方式。

使用 `getElementById()`，你可以取得表述具有指定 `id` 的 HTML 元素的物件。請記得每個 HTML 文件中的 `id` 必須是唯一的，所以此方法只會傳回一個 `Element` 物件。以下是範例：

```js
const container = document.getElementById("container");
```

這行 JavaScript 程式碼會取得 `id` 值為 `container` 的元素，並將該物件指定給一個 JavaScript 常數。你必須將 `id` 以引號包住的形式作為引數傳入。如果你將此物件記錄到主控台，你會在輸出中看到它：

:::interactive_editor

```html
<div id="container">
  <h1>Hello, World!</h1>
  <p>Welcome to learning about the DOM.</p>
</div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
console.log(container); 
```

:::

`querySelector()` 比 `getElementById()` 更廣泛。使用 `querySelector()`，你可以取得 HTML 文件中符合所傳入 CSS 選擇器引數的第一個元素。 

在此範例中，你將取得符合 `section` 選擇器的元素：

:::interactive_editor

```html
<section>
  <h2>Section Title</h2>
  <p>This is a section of the webpage.</p>
</section>
<script src="index.js"></script>  
```

```js
const sectionEl = document.querySelector("section");
console.log(sectionEl);
```

:::

如果你想使用 `querySelector()` 透過類別名稱選取元素，你需要在類別名稱前加上點號（`.`）。以下是一個範例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="highlight">
  <p>This content is highlighted.</p>
</div>
<script src="index.js"></script>
```

```css
body {
  background-color: lightgray;
}

.highlight {
  background-color: yellow;
}
```

```js
const highlightEl = document.querySelector(".highlight");
console.log(highlightEl);
```

:::

你也有其他方法來匹配多個元素，例如 `getElementsByClassName()` 和 `querySelectorAll()`。你會在未來的課程中學到更多關於它們的內容。

透過了解如何使用網頁 API 高效地操作 DOM，你可以創建強大且互動的網頁應用程式。
