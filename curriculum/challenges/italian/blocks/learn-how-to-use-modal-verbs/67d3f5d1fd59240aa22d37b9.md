## --text--

Quale frase è grammaticalmente corretta?

## --answers--

`You must to complete the security training.`

### --feedback--

`Must` è un verbo modale e non usa `to` prima del verbo base.

---

`You must completed the security training.`

### --feedback--

Il verbo dopo `must` dovrebbe essere nella forma base (`complete`), non al passato (`completed`).

---

`You must complete the security training.`

---

`You must completing the security training.`

### --feedback--

`Must` è seguito dal verbo base, non dalla forma `-ing`.

# --explanation--

`Must` è un verbo modale, quindi segue queste regole:

- È sempre seguito dalla forma base del verbo (senza `to`).

- Non prende `-s` alla terza persona singolare.

- Non usa `do/does` nelle domande o nelle forme negative.
