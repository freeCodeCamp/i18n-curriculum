# --explanation--

Ángela는 `Perfecto. Entonces, Esteban Navarro Huamán.`라고 말합니다.

`Perfecto`는 동의나 만족을 나타내는 데 사용됩니다. 이는 정보가 정확하다는 것을 의미합니다.

`Entonces`는 정보를 들은 후에 사용됩니다. 화자가 생각을 정리하고 정보를 명확히 반복하는 데 도움이 됩니다.

여기서 `entonces`는 확인을 도입합니다. Ángela는 전체 이름을 반복하여 정확한지 확인합니다.

그녀는 `Huamán`라고 말하지 `Guzmán`라고 말하지 않습니다. 따라서 이 문장은 거짓입니다.
