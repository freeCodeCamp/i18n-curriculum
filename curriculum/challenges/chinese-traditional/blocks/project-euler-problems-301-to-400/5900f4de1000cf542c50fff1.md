# --description--

讓我們定義一個幾何三角形為邊長為整數且邊長為 $a ≤ b ≤ c$ 的三角形，使其邊長形成等比數列，即 $b^2 = a \times c$。

這樣的幾何三角形範例是邊長為 $a = 144$、$b = 156$ 和 $c = 169$ 的三角形。

有 $861\,805$ 個幾何三角形，其 $\text{perimeter} ≤ {10}^6$。

存在多少個幾何三角形，其 $\text{perimeter} ≤ 2.5 \times {10}^{13}$？
