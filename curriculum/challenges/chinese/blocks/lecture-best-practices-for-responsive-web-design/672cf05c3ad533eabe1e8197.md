# --interactive--

媒体查询是响应式网页设计的基本组成部分，允许开发人员根据用户设备或浏览器的特性应用不同的样式。 它们提供了一种在不改变内容本身的情况下为各种设备量身定制内容呈现方式的方法。

媒体查询的核心是一个媒体类型和一个或多个用于检查特定条件的表达式。 如果这些条件为真，就会应用相应的样式。 这种机制允许创建适应不同屏幕尺寸、分辨率和设备功能的响应式设计。

CSS 中媒体查询的基本语法如下：

```css
@media mediatype and (feature: value) {
  /* CSS rules go here */
}
```

在此结构中，mediatype 指定了查询适用的媒体类型，而 feature: value 对定义了应用样式必须满足的条件。

媒体类型描述了设备的一般类别。 让我们来谈谈最常用的媒体类型：

- `all` 适用于所有设备。 如果未指定媒体类型，则默认为该类型。

- `print` 适用于分页材料和在打印预览模式下在屏幕上查看的文档。

- `screen` 主要用于屏幕。

过去，有更多的媒体类型，比如 `handheld` 和 `tv`，但大多数已被废弃，转而使用特色来更精确地定位设备。

媒体特性描述了用户代理、输出设备或环境的具体特征。 让我们来谈谈一些最常用的媒体特性。 `width` 和 `height` 指的是视口宽度和高度，通常与 `min-` 或 `max-` 前缀一起用于范围查询。 这是一个示例：

```css
@media screen and (min-width: 768px) {
  /* Styles for screens at least 768px wide */
}
```

`aspect-ratio` 描述视口宽度和高度之间的比例。 这是一个示例：

```css
@media screen and (aspect-ratio: 16/9) {
  /* Styles for screens with a 16:9 aspect ratio */
}
```

`orientation` 特性指示设备是横向还是纵向。 这是一个示例：

```css
@media screen and (orientation: landscape) {
  /* Styles for landscape orientation */
}
```

`resolution` 特性描述了输出设备的分辨率，单位是点/英寸（dpi）或点/厘米（dpcm）。 这是一个示例：

```css
@media screen and (min-resolution: 300dpi) {
  /* Styles for high-resolution screens */
}
```

`hover` 特性可测试主输入机制是否能悬停在元素上。 这是一个示例：

```css
@media (hover: hover) {
  /* Styles for devices that support hover */
}
```

`prefers-color-scheme` 特性可检测用户是否要求使用浅色或深色主题。 这是一个示例：

```css
@media (prefers-color-scheme: dark) {
  /* Styles for dark mode */
}
```

媒体查询还可以使用逻辑运算符组合多个条件。 `and` 操作符用于组合多个媒体特性，而 `not` 和 `only` 可用于否定或隔离媒体查询。 下面是一个结合了多种特性的示例：

```css
@media screen and (min-width: 768px) and (orientation: landscape) {
  /* Styles for landscape screens at least 768px wide */
}
```

还可以在逗号分隔的列表中定位多个查询，其功能类似于“或”运算符：

```css
@media screen and (min-width: 768px), print {
  /* Styles for screens at least 768px wide OR for print */
}
```

使用媒体查询时，必须考虑级联。 媒体查询并不会增加特殊性——它们只是将条件规则组合起来。 CSS 级联的正常规则仍适用于每个媒体查询。

在实际应用中，媒体查询通常用于创建响应式布局。 常见的模式是为移动设备定义基本样式，然后使用媒体查询来增强大屏幕的布局：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer nec odio. Praesent libero. Sed cursus ante dapibus diam.</p>
  <p>Sed nisi. Nulla quis sem at nibh elementum imperdiet. Duis sagittis ipsum. Praesent mauris. Fusce nec tellus sed augue semper porta.</p>
  <p>Mauris massa. Vestibulum lacinia arcu eget nulla. Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos.</p>
</div>
```

```css
/* Base styles for mobile */
.container {
  width: 100%;
  padding: 15px;
}

/* Styles for tablets */
@media screen and (min-width: 768px) {
  .container {
    width: 750px;
    margin: 0 auto;
  }
}

/* Styles for desktops */
@media screen and (min-width: 1024px) {
  .container {
    width: 960px;
  }
}
```

:::

这种方法被称为“移动优先”响应式设计，可确保基本样式适合移动设备，并为大屏幕添加增强功能。

总之，媒体查询是 CSS 中的一个强大工具，可用于创建反应灵敏、适应性强的网页设计。 通过了解如何使用不同的媒体类型和功能，开发人员可以创建能在各种设备和偏好中提供最佳用户体验的网站。 随着网络技术的不断发展，不断更新新的媒体特性有助于创建更加细致入微、用户友好的响应式设计。
