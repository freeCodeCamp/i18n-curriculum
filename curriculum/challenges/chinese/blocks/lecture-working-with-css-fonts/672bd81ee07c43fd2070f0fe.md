# --description--

网页安全字体是极有可能已安装在计算机或设备上的一类字体子集。 它们广泛支持不同操作系统和网页浏览器，因此极可能被一致地渲染和显示。 让我们看看其工作原理。

浏览器负责解释和显示网站上的字体。 当浏览器需要渲染字体时，它会尝试在用户系统中查找字体文件。 但若未找到该字体，通常会回退到默认系统字体。 这确保了即使特定字体缺失，内容仍可读。

然而，浏览器选择的备用字体可能与原本应渲染的字体差异显著。 这可能对整体设计和用户体验产生重大影响。 为避免此问题，应尽可能使用网页安全字体。 你有两种选择。 你可以将其作为主要字体，或者你可以使用自定义字体并将网页安全字体作为备用选项。 这样，即使未找到自定义字体，你也能控制网站的显示效果。

我们看一些网页安全字体的示例。  In a previous lesson, you learned that sans-serif fonts are commonly used for web development because they don't have small "feet" or lines at the end of the characters, so they're easy to read on screen. 常见的网页安全无衬线字体包括：

- Arial
- Verdana
- Trebuchet MS

相比之下，衬线字体因字符末端有“小脚”，常用于传统印刷。 但如果你需要将其用于网页开发，网页安全衬线字体包括：

- Times New Roman
- 乔治亚

使用网页安全字体可确保设计在跨设备和平台上保持一致。 它们还能提升视觉障碍用户的可访问性，因为他们简洁易读。 并减少页面加载时间，因为如果字体已安装则无需下载。

既然你已了解网页安全字体的概念，即可开始使用它们或将其设为备用选项，从而跨浏览器和平台打造可靠的用户体验。
